## 4.1 线性融合实例
```python
import numpy as np

# 定义多个模型的预测结果
model1_pred = np.array([1, 2, 3])
model2_pred = np.array([4, 5, 6])
model3_pred = np.array([7, 8, 9])

# 定义模型权重
weights = np.array([0.2, 0.3, 0.5])

# 线性融合
linear_fusion = np.dot(weights, np.hstack((model1_pred, model2_pred, model3_pred)))

print(linear_fusion)
```
在这个例子中，我们定义了三个模型的预测结果，并且为每个模型分配了权重。然后，我们通过线性组合这些预测结果来得到最终的预测结果。

import numpy as np

# 定义多个模型的预测结果
model1_pred = np.array([1, 2, 3])
model2_pred = np.array([4, 5, 6])
