## 4.1 大脑多任务处理的代码实例
以下是一个简单的Python代码实例，用于模拟大脑多任务处理：

```python
import time
import random

def task1():
    time.sleep(random.uniform(0.5, 1.5))
    print("任务1完成")

def task2():
    time.sleep(random.uniform(0.5, 1.5))
    print("任务2完成")

def main():
    tasks = [task1, task2]
    for task in tasks:
        task()
    print("所有任务完成")

if __name__ == "__main__":
    main()
```

在这个代码实例中，我们定义了两个任务`task1`和`task2`，它们的处理时间是随机的，表示大脑处理多任务时的不确定性。在`main`函数中，我们使用一个循环来执行所有任务，这样可以模拟大脑同时处理多个任务的能力。

## 4.2 计算机多任务处理的代码实例
以下是一个简单的Python代码实例，用于模拟计算机多任务处理：

```python
import time
import threading

def task1():
    time.sleep(1)
    print("任务1完成")

def task2():
    time.sleep(1)
    print("任务2完成")

def main():
    tasks = [task1, task2]
    threads = []
    for task in tasks:
        thread = threading.Thread(target=task)
        thread.start()
        threads.append(thread)
    for thread in threads:
        thread.join()
    print("所有任务完成")

if __name__ == "__main__":
    main()
```

在这个代码实例中，我们使用Python的`threading`模块来实现计算机多任务处理。我们定义了两个任务`task1`和`task2`，然后使用多线程来执行这两个任务。在`main`函数中，我们使用一个循环来执行所有任务，这样可以模拟计算机同时处理多个任务的能力。
