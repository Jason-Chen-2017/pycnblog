# 2.核心概念与联系

## 2.1 信息熵

信息熵是信息论的基本概念之一，它用于衡量信息的不确定性。信息熵的定义为：

$$
H(X) = -\sum_{i=1}^{n} P(x_i) \log_2 P(x_i)
$$

其中，$X$ 是一个随机变量，$x_i$ 是 $X$ 的取值，$P(x_i)$ 是 $x_i$ 的概率。信息熵的单位是比特（bit）。

## 2.2 条件熵

条件熵是信息熵的一种泛化，用于衡量给定某个条件下信息的不确定性。条件熵的定义为：

$$
H(X|Y) = -\sum_{j=1}^{m} P(y_j) \sum_{i=1}^{n} P(x_i|y_j) \log_2 P(x_i|y_j)
$$

其中，$Y$ 是另一个随机变量，$y_j$ 是 $Y$ 的取值，$P(x_i|y_j)$ 是 $x_i$ 给定 $y_j$ 时的概率。

## 2.3 互信息

互信息是信息论的另一个重要概念，用于衡量两个随机变量之间的相关性。互信息的定义为：

$$
I(X;Y) = H(X) - H(X|Y)
$$

## 2.4 联系

信息熵、条件熵和互信息之间的联系如下：

1. 信息熵衡量单个随机变量的不确定性。
2. 条件熵衡量给定某个条件下单个随机变量的不确定性。
3. 互信息衡量两个随机变量之间的相关性。

## 4.1 信息熵计算

以下是一个计算信息熵的Python代码实例：

```python
import math

def entropy(prob):
    return -sum(p * math.log2(p) for p in prob if p > 0)

prob = [0.2, 0.3, 0.1, 0.4]
print("信息熵:", entropy(prob))
```

在这个例子中，我们计算了一个概率分布的信息熵。信息熵的计算公式为：

$$
H(X) = -\sum_{i=1}^{n} P(x_i) \log_2 P(x_i)
$$

通过计算信息熵，我们可以衡量随机变量的不确定性。

## 4.2 条件熵计算

以下是一个计算条件熵的Python代码实例：

```python
def conditional_entropy(prob, condition):
    return entropy([p * q for p, q in zip(prob, condition)])

prob = [0.2, 0.3, 0.1, 0.4]
condition = [0.5, 0.4, 0.1, 0.0]
print("条件熵:", conditional_entropy(prob, condition))
```

在这个例子中，我们计算了一个概率分布给定某个条件下的条件熵。条件熵的计算公式为：

$$
H(X|Y) = -\sum_{j=1}^{m} P(y_j) \sum_{i=1}^{n} P(x_i|y_j) \log_2 P(x_i|y_j)
$$

通过计算条件熵，我们可以衡量给定某个条件下随机变量的不确定性。

## 4.3 互信息计算

以下是一个计算互信息的Python代码实例：

```python
def mutual_information(prob_x, prob_y):
    H(X) = entropy(prob_x)
    H(Y) = entropy(prob_y)
    H(X, Y) = entropy([p * q for p, q in zip(prob_x, prob_y)])
    return H(X) + H(Y) - H(X, Y)

prob_x = [0.2, 0.3, 0.1, 0.4]
prob_y = [0.5, 0.4, 0.1, 0.0]
print("互信息:", mutual_information(prob_x, prob_y))
```

在这个例子中，我们计算了两个概率分布之间的互信息。互信息的计算公式为：

$$
I(X;Y) = H(X) - H(X|Y)
$$

通过计算互信息，我们可以衡量两个随机变量之间的相关性。
