### 3.2 图卷积网络的算法原理

图卷积网络的算法原理是通过将图上的节点表示为低维向量，然后通过卷积层进行信息传递，从而实现图上的特征学习。具体来说，图卷积网络的算法原理包括以下几个步骤：

1. 将图转化为邻接矩阵：将图上的节点表示为向量，然后将图上的关系表示为邻接矩阵。
2. 定义卷积核：定义一个滤波器（卷积核），用于在图上进行卷积操作。
3. 进行卷积操作：将卷积核应用于邻接矩阵上，从而得到新的特征向量。
4. 添加激活函数：将新的特征向量通过激活函数进行非线性变换。
5. 重复上述操作：重复上述操作，从而实现多层卷积网络的构建。
6. 得到最终预测结果：将最终的特征向量通过全连接层得到预测结果。

$$
H^{(l+1)} = \sigma \left( AH^{(l)}W^{(l)} \right)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class GCN(nn.Module):
    def __init__(self, nfeat, nhid, nclass, dropout):
        super(GCN, self).__init__()
        self.lin0 = nn.Linear(nfeat, nhid)
        self.dropout = nn.Dropout(dropout)
        self.lin1 = nn.Linear(nhid, nclass)

    def forward(self, input, adj):
        x = self.lin0(input)
        x = torch.mm(adj, x)
        x = F.relu(x)
        x = self.dropout(x)
        x = self.lin1(x)
        return x
```
