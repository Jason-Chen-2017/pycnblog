### 3.3 数学模型公式详细讲解

在知识创造中，我们通常会使用到一些数学模型来描述数据中的规律和关系。以下是一些常见的数学模型公式：

1. 线性回归：用于描述数据中的线性关系。公式为：$$ y = ax + b $$
2. 多项式回归：用于描述数据中的多项式关系。公式为：$$ y = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0 $$
3. 逻辑回归：用于描述数据中的二分类问题。公式为：$$ P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + \dots + \beta_n x_n)}} $$
4. 支持向量机：用于解决线性可分和非线性可分的分类问题。公式为：$$ f(x) = \text{sgn} \left( \alpha_0 + \alpha_1 x_1 + \dots + \alpha_n x_n + b \right) $$
5. 决策树：用于解决分类和回归问题。公式为：$$ \text{if } x_1 \text{ is } A_1 \text{ then } \dots \text{ else if } x_n \text{ is } A_n \text{ then } y $$
6. 随机森林：用于解决分类和回归问题。公式为：$$ y = \text{majority vote of } f_1(x), f_2(x), \dots, f_n(x) $$
7. 神经网络：用于解决分类和回归问题。公式为：$$ y = \text{softmax} \left( Wx + b \right) $$
