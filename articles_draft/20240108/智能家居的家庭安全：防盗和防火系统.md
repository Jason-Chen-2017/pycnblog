# 2.核心概念与联系

## 2.1 智能家居
智能家居是一种利用互联网、无线技术、人工智能等技术来实现家庭设备的智能化控制和管理的系统。它可以让家庭用户通过手机、平板电脑等设备远程控制家庭设备，如灯泡、空调、门锁等，实现家庭自动化和智能化。

## 2.2 家庭安全
家庭安全是智能家居的核心功能之一。它涉及到家庭用户的生活安全、财产安全以及家庭成员的生命安全。家庭安全包括防盗、防火、防盗防火等多种方面。

## 2.3 防盗系统
防盗系统是家庭安全的重要组成部分，其主要目的是防止家庭被盗窃、侵入。防盗系统包括门锁、门磁感应、窗户传感器、报警器等设备。

## 2.4 防火系统
防火系统是家庭安全的另一个重要组成部分，其主要目的是防止家庭发生火灾、爆炸等灾难。防火系统包括烟感、火警器、防火门等设备。
