## 4.1.1 摄像头
### 代码实例1：摄像头数据获取
```python
import cv2

# 打开摄像头
cap = cv2.VideoCapture(0)

# 读取摄像头帧
ret, frame = cap.read()

# 显示帧
cv2.imshow('Frame', frame)

# 关闭摄像头
cap.release()
cv2.destroyAllWindows()
```
### 详细解释说明
在这个代码实例中，我们使用了OpenCV库来获取摄像头帧。首先，我们使用`cv2.VideoCapture(0)`来打开摄像头。然后，我们使用`cap.read()`来读取摄像头帧。最后，我们使用`cv2.imshow()`来显示帧，并使用`cap.release()`和`cv2.destroyAllWindows()`来关闭摄像头。

## 4.1.2 WIFI定位技术

在这个代码实例中，我们使用了wifi库来获取WIFI定位技术数据。首先，我们使用`wifi.get_location()`来获取WIFI定位技术数据。然后，我们使用`print()`来打印WIFI定位技术数据。
