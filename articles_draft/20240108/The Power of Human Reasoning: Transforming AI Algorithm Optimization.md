$$

# 4.1数学推理优化代码实例
```python
def power(a, n):
    if n == 0:
        return 1
    elif n % 2 == 0:
        return power(a, n // 2) ** 2
    else:
        return a * power(a, n // 2) ** 2
```
这个代码实例是快速幂算法的一个简单实现。通过使用递归关系，我们可以在对数时间内计算指数乘法问题。
