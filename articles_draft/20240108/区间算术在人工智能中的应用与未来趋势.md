## 1.1 区间算术的基本概念

区间算术主要关注的是表示和操作有界连续数值数据的方法。在这种方法中，数值数据被表示为一个区间，该区间包含了一个或多个实数。区间可以表示为一个开区间（即，不包含端点）、一个关闭区间（即，包含端点）或一个半开半关闭区间（即，只包含一个端点）。

在人工智能领域，区间算术主要用于处理不确定的数据和信息。例如，在机器学习中，我们经常需要处理不确定的数据，如概率分布、预测间隔、不确定性范围等。在这种情况下，区间算术可以帮助我们更好地处理和表示这些不确定的信息。

```python

    # 将输入的数值数据转换为区间表示
    return (min(data), max(data))

def interval_operations(interval1, interval2):
    # 对两个区间进行交集、并集和差集运算
    intersection = (max(interval1[0], interval2[0]), min(interval1[1], interval2[1]))
    union = (min(interval1[0], interval2[0]), max(interval1[1], interval2[1]))
    difference = (max(interval1[0], interval2[1]), min(interval1[1], interval2[0]))
    return intersection, union, difference

def to_data(interval):
    # 将结果区间转换为原始数据表示
    return list(range(interval[0], interval[1] + 1))

# 示例数据
data1 = [1, 5]
data2 = [3, 7]

# 将数据转换为区间表示
interval1 = to_interval(data1)
interval2 = to_interval(data2)

# 对两个区间进行运算
intersection, union, difference = interval_operations(interval1, interval2)

# 将结果区间转换为原始数据表示
result1 = to_data(intersection)
result2 = to_data(union)
result3 = to_data(difference)

# 输出结果
print("交集: ", result1)
print("并集: ", result2)
print("差集: ", result3)
```
