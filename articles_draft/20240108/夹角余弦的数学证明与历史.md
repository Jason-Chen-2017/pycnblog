## 2. 核心概念与联系

夹角余弦是一种用于描述两个向量之间的关系的数学概念。它通常用符号cos(θ)表示，其中θ是两个向量之间的夹角。夹角余弦的定义如下：

$$
\cos(\theta) = \frac{\mathbf{a} \cdot \mathbf{b}}{|\mathbf{a}||\mathbf{b}|}
$$

其中，$\mathbf{a}$和$\mathbf{b}$是两个向量，$\mathbf{a} \cdot \mathbf{b}$是它们的内积，$|\mathbf{a}|$和$|\mathbf{b}|$是它们的长度。

夹角余弦的一个重要特点是，它是一个范围在$-1$到$1$之间的实数。当两个向量是正交的时，夹角余弦为$0$；当两个向量是相同的时，夹角余弦为$1$；当两个向量是反方向的时，夹角余弦为$-1$。

def dot_product(a, b):
    return np.dot(a, b)

def magnitude(a):
    return np.linalg.norm(a)

    return dot_product(a, b) / (magnitude(a) * magnitude(b))
