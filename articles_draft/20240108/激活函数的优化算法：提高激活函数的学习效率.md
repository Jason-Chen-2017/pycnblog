## 4.具体代码实例和详细解释说明

### 4.1 使用梯度下降算法优化激活函数

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def sigmoid_derivative(x):
    return x * (1 - x)

def gradient_descent(x, learning_rate):
    x_optimized = x
    for _ in range(1000):
        x_optimized = x_optimized - learning_rate * sigmoid_derivative(x_optimized)
    return x_optimized

x = np.array([1, -1])
learning_rate = 0.1
optimized_x = gradient_descent(x, learning_rate)
print(optimized_x)
```
