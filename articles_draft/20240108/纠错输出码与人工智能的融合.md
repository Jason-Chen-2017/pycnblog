# 2.核心概念与联系

## 2.1 纠错输出码的基本概念

纠错输出码是一种在通信系统中用于纠正数据传输中出现错误的技术。它的核心思想是在数据传输过程中，为数据添加一定的冗余信息，使得在数据接收端可以根据冗余信息来检测和纠正数据传输过程中出现的错误。

常见的纠错输出码技术有：

- 冗余检查码（Parity Check Code）：最简单的纠错输出码技术，通过添加一个检查位来实现简单的错误检测。
- 循环冗余检查码（Cyclic Redundancy Check, CRC）：通过计算数据的CRC值来实现更加强大的错误检测功能。
- 块码（Block Code）：通过添加多个冗余位来实现更加强大的错误纠正功能。常见的块码有Hamming码、Reed-Solomon码等。
