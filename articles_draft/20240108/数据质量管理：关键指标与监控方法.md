### 3.4数学模型公式详细讲解

数据质量管理中的数学模型公式主要用于计算数据质量指标的值。常见的数据质量指标计算公式包括：

- 准确性：$$ Accuracy = \frac{TP + TN}{TP + FP + TN + FN} $$
- 完整性：$$ Completeness = \frac{C}{C + M} $$
- 一致性：$$ Consistency = \frac{C}{C + I} $$
- 时效性：$$ Timeliness = \frac{E}{E + L} $$
- 可用性：$$ Availability = \frac{U}{U + D} $$

其中，TP表示真阳性，TN表示真阴性，FP表示假阳性，FN表示假阴性，C表示完整数据，M表示缺失数据，I表示不一致数据，E表示有效数据，L表示失效数据，U表示可用数据，D表示不可用数据。

### 4.1数据清洗代码实例

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 去除重复数据
data = data.drop_duplicates()

# 填充缺失数据
data['age'].fillna(data['age'].mean(), inplace=True)

# 修复错误数据
data['gender'] = data['gender'].map({'M': '男', 'F': '女', 'U': '未知'})

# 转换数据格式
data['birth_year'] = data['birth_year'].astype(int)
```

### 4.2数据验证代码实例

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 检查数据的准确性
accuracy = data['age'].apply(lambda x: x == data['age'].median()).mean()

# 检查数据的完整性
completeness = data['age'].isnull().sum() / len(data)

# 检查数据的一致性
consistency = data[data['age'] > 120].isnull().sum() / len(data[data['age'] > 120])

# 检查数据的时效性
timeliness = data['age'].apply(lambda x: x > data['birth_year'].mean() - 18).mean()

# 检查数据的可用性
availability = data.shape[0] / data.shape[0] + data.shape[1]

# 打印结果
print('准确性:', accuracy)
print('完整性:', completeness)
print('一致性:', consistency)
print('时效性:', timeliness)
print('可用性:', availability)
```
