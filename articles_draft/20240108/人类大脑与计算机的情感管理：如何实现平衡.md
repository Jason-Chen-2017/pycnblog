# 4.具体代码实例和详细解释说明

## 4.1 情感识别

以下是一个情感识别的Python代码实例：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 情感数据
data = [
    ("I love this product", 1),
    ("This is a great product", 1),
    ("I hate this product", 0),
    ("This is a bad product", 0),
]

# 训练情感向量器
vectorizer = TfidfVectorizer()
vectorizer.fit_transform(data)

# 情感识别
def sentiment_recognition(text):
    vector = vectorizer.transform([text])
    similarity = cosine_similarity(vector, vectorizer.vocabulary_)
    return "positive" if similarity > 0.5 else "negative"

# 测试情感识别
print(sentiment_recognition("I love this product"))  # positive
print(sentiment_recognition("I hate this product"))  # negative
```

在这个代码实例中，我们使用了TF-IDF向量器来将文本数据转换为向量，然后使用余弦相似度来计算文本的情感相似度。如果相似度大于0.5，则认为情感是正面的，否则认为情感是负面的。
