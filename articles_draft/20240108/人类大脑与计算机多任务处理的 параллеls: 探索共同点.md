## 4.1 人类大脑多任务处理的代码实例

我们可以通过以下代码实例来说明人类大脑多任务处理的算法原理：

```python
import time

def attention_split(tasks, time):
    task_count = 0
    for task in tasks:
        start_time = time.time()
        task_count += 1
        task()
        end_time = time.time()
        elapsed_time = end_time - start_time
        if elapsed_time < time:
            task_count -= 1
            break
    return task_count
```

在这个代码实例中，我们定义了一个名为 `attention_split` 的函数，它接受两个参数：`tasks` 和 `time`。`tasks` 是一个包含多个任务的列表，`time` 是总时间。函数的返回值是人类大脑同时关注多个任务的数量。
