## 4.1 XML到JSON的转换实例
我们以XML到JSON的转换为例，提供具体代码实例和详细解释说明。

### 4.1.1 代码实例
```python
import xml.etree.ElementTree as ET
import json

def xml_to_json(xml_str):
    root = ET.fromstring(xml_str)
    json_data = {}
    for child in root:
        if child.text:
            json_data[child.tag] = child.text
        else:
            json_data[child.tag] = []
            for subchild in child:
                json_data[child.tag].append(subchild.text)
    return json.dumps(json_data)
```
### 4.1.2 代码解释
1. 首先，我们导入xml.etree.ElementTree和json库。
2. 定义一个函数xml_to_json，接收一个XML字符串作为参数。
3. 使用ET.fromstring()函数将XML字符串解析为ElementTree对象。
4. 创建一个空字典json_data，用于存储JSON数据。
5. 遍历ElementTree对象的子元素，如果子元素有文本内容，将其添加到json_data字典中。
6. 如果子元素没有文本内容，将其视为列表，并将子元素的文本内容添加到列表中。
7. 使用json.dumps()函数将json_data字典转换为JSON字符串。

## 4.2 数据处理实例
我们以数据清洗为例，提供具体代码实例和详细解释说明。

### 4.2.1 代码实例
```python
import re

def clean_data(data):
    if isinstance(data, dict):
        return {k: clean_data(v) for k, v in data.items()}
    elif isinstance(data, list):
        return [clean_data(v) for v in data]
    else:
        data = str(data)
        data = re.sub(r'[^\w\s]', '', data)
        return data
```
### 4.2.2 代码解释
1. 首先，我们导入re库。
2. 定义一个函数clean_data，接收一个数据作为参数。
3. 判断数据类型，如果是字典，将字典中的键值递归地进行清洗。
4. 如果是列表，将列表中的元素递归地进行清洗。
5. 如果是其他类型，将数据转换为字符串，并使用正则表达式将非字母数字字符过滤掉。
6. 返回清洗后的数据。
