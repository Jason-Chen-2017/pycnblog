## 4.1 GraphQL 查询解析

以下是一个使用 GraphQL 查询解析的代码实例：

```javascript
import { useQuery } from '@apollo/client';
import gql from '@apollo/client';

const GET_POSTS = gql`
  query GetPosts {
    posts {
      id
      title
      content
    }
  }
`;

const Posts = () => {
  const { loading, error, data } = useQuery(GET_POSTS);

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error :(</p>;

  return (
    <div>
      {data.posts.map((post) => (
        <div key={post.id}>
          <h2>{post.title}</h2>
          <p>{post.content}</p>
        </div>
      ))}
    </div>
  );
};

export default Posts;
```

在这个代码实例中，我们使用了 `@apollo/client` 库来实现 GraphQL 查询解析。首先，我们定义了一个 GraphQL 查询 `GET_POSTS`，用于获取所有的博客文章。然后，我们使用 `useQuery` 钩子来执行查询，并根据查询结果渲染博客文章列表。
