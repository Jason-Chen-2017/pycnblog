# 4.具体代码实例和详细解释说明

## 4.1 基于距离的聚类算法的代码实例
```python
import numpy as np
from sklearn.cluster import KMeans
from sklearn.metrics import pairwise_distances

# 创建一个有向图
G = nx.DiGraph()
G.add_edges_from([(0, 1), (1, 2), (2, 3), (3, 4)])

# 计算节点之间的距离
distances = pairwise_distances(G.nodes())

# 使用K均值聚类算法将节点分为不同的类别
kmeans = KMeans(n_clusters=2)
kmeans.fit(distances)

# 对于每个类别，计算类别内节点之间的距离
for cluster in kmeans.cluster_centers_:
    print(f"类别 {cluster} 内节点之间的距离为：{np.mean(distances[kmeans.labels_ == cluster])}")
```
