### 3.3.1 线性规划模型

线性规划模型是一种常用的优化模型，可以用于解决农业物质流量优化问题。其主要公式包括：

1. 目标函数：$$
   min\quad c^Tx
   $$
   其中，$c$是目标函数的系数向量，$x$是决策变量向量。
2. 约束条件：$$
   Ax \leq b
   $$
   其中，$A$是约束矩阵，$b$是约束向量。

### 3.3.2 混合整数规划模型

混合整数规划模型是一种用于解决包含整数决策变量的优化问题的模型。其主要公式包括：

1. 目标函数：$$
   min\quad c^Tx
   $$
   其中，$c$是目标函数的系数向量，$x$是决策变量向量。
2. 约束条件：$$
   Ax \leq b
   $$
   其中，$A$是约束矩阵，$b$是约束向量。
3. 整数约束条件：$$
   x_i \in \{0,1,2,...,M\}
   $$
   其中，$x_i$是整数决策变量，$M$是整数约束的上限。

```python
import numpy as np

# 读取数据
data = pd.read_csv('agriculture_data.csv')

# 数据清洗
data = data.dropna()

# 缺失值填充
data['temperature'].fillna(method='ffill', inplace=True)

# 数据转换
data['humidity'] = data['humidity'].astype(float)
data['temperature'] = data['temperature'].astype(float)
data['light_intensity'] = data['light_intensity'].astype(float)
```
