$$
supp(X \Rightarrow Y) = \frac{P(X \cap Y)}{P(X)}

其中，$supp(X \Rightarrow Y)$ 是规则的支持度，$conf(X \Rightarrow Y)$ 是规则的信息度，$P(X \cap Y)$ 是$X$和$Y$的联合概率，$P(X)$ 是$X$的概率，$P(Y|X)$ 是$Y$给定$X$的概率。

# 4.具体代码实例和详细解释说明

在这里，我们以一个简单的Python代码实例来展示数据清洗、数据处理、数据挖掘和数据可视化的具体操作。

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 数据清洗
data = pd.read_csv('data.csv')
data = data.drop_duplicates()
data = data.fillna(method='ffill')

# 数据处理
data['age'] = (data['age'] - data['age'].min()) / (data['age'].max() - data['age'].min())
data = data[data['age'] > 0]

# 数据挖掘
from sklearn.cluster import KMeans

kmeans = KMeans(n_clusters=2)
data['cluster'] = kmeans.fit_predict(data[['age', 'income']])

# 数据可视化
plt.scatter(data['age'], data['income'], c=data['cluster'])
plt.xlabel('Age')
plt.ylabel('Income')
plt.show()
```

这段代码首先通过Pandas库读取CSV文件，然后进行数据清洗，去除重复数据并填充缺失数据。接着进行数据处理，将年龄数据归一化，并筛选出年龄大于0的数据。接着使用Scikit-learn库进行聚类分析，将数据分为2个聚类，并将聚类结果添加到数据框中。最后使用Matplotlib库进行数据可视化，绘制年龄和收入之间的关系。
