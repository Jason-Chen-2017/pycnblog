## 4.1 代码优化实例

假设我们有一个计算两个数之和的函数：

```python
def sum(a, b):
    result = 0
    for i in range(a, b + 1):
        result += i
    return result
```

我们可以对这个函数进行优化，将循环改为使用内置的sum函数：

```python
def sum(a, b):
    return sum(range(a, b + 1))
```

通过这种优化，我们可以减少循环的次数，提高代码的性能。
