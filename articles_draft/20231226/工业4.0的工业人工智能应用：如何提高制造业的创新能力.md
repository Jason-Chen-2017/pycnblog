## 4.1 数据收集

我们选择了一个制造业的温度监控系统作为代表性的数据收集示例。在这个系统中，我们使用了一些温度传感器来收集制造过程中的温度数据。以下是一个简单的Python代码实例：

```python
import time
import requests

def get_temperature():
    url = 'http://sensors.example.com/temperature'
    response = requests.get(url)
    data = response.json()
    return data['temperature']

temperatures = []
while True:
    temperature = get_temperature()
    temperatures.append(temperature)
    time.sleep(60)
```

在这个代码中，我们首先导入了`time`和`requests`模块。然后我们定义了一个`get_temperature`函数，该函数通过发送HTTP请求来获取温度传感器的温度数据。接着我们创建了一个空列表`temperatures`，用于存储温度数据。最后，我们通过一个无限循环来不断获取温度数据，并将其存储到`temperatures`列表中。

我们将使用Python的`scikit-learn`库来建立一个简单的线性回归模型，用于预测未来的温度。以下是一个简单的代码实例：

```python

print('预测未来的温度为：', future_temperature[0])
```

在这个代码中，我们首先导入了`scikit-learn`库。然后我们将温度数据存储到一个数组中，并将其转换为一个训练集。接着我们使用`LinearRegression`类建立了一个线性回归模型，并使用训练集来训练模型。最后，我们使用训练好的模型来预测未来的温度。
