## 4.具体代码实例和详细解释说明

以下是一个Python代码实例，用于实现LU分解：

```python
import numpy as np

def lu_decomposition(A):
    n = A.shape[0]
    L = np.eye(n)
    U = A.copy()
    for i in range(n):
        for j in range(i):
            U[i, j] -= L[i, j] * U[j, j] / U[j, i]
        L[i, i] -= U[i, i]
        for j in range(i+1, n):
            L[i, j] = U[i, j] / U[i, i]
            U[i, j] = 0
    return L, U

A = np.array([[4, 2, 1],
              [2, 4, 1],
              [1, 1, 4]])

L, U = lu_decomposition(A)
print("L矩阵:\n", L)
print("U矩阵:\n", U)
```
