## 3.1 支持向量机(SVM)

支持向量机是一种常用的分类和回归算法，它的核心思想是找到一个最佳的超平面，将不同类别的数据点分开。在多元线性分类问题中，SVM的目标是找到一个最大的线性分类器。

### 3.1.1 算法原理

支持向量机的核心思想是通过找到一个最佳的超平面来将不同类别的数据点分开。这个超平面的表达式为：

$$
f(x) = w^T x + b
$$

其中，$w$ 是权重向量，$x$ 是输入向量，$b$ 是偏置项。

支持向量机的目标是最大化超平面与训练数据的间隔，同时满足约束条件：

$$
\min_{w,b} \frac{1}{2}w^T w \\
s.t. y_i(w^T x_i + b) \geq 1, i=1,2,...,n
$$

其中，$y_i$ 是数据点的标签，$x_i$ 是数据点的特征向量。

### 3.1.2 具体操作步骤

1. 数据预处理：将数据转换为向量，并标准化。
2. 训练SVM：使用支持向量机算法训练模型。
3. 预测：输入新的数据，并使用训练好的模型进行预测。

## 3.2 随机森林

随机森林是一种集成学习方法，它通过构建多个决策树来进行预测。每个决策树都是在随机选择的特征上构建的，这有助于减少过拟合。

### 3.2.1 算法原理

随机森林的核心思想是通过构建多个决策树来进行预测，每个决策树都是在随机选择的特征上构建的。这有助于减少过拟合，并提高预测准确性。

### 3.2.2 具体操作步骤

1. 数据预处理：将数据转换为向量，并标准化。
2. 训练随机森林：使用随机森林算法训练模型。
3. 预测：输入新的数据，并使用训练好的模型进行预测。

### 3.2.3 数学模型公式详细讲解

随机森林的核心思想是通过构建多个决策树来进行预测，每个决策树都是在随机选择的特征上构建的。这有助于减少过拟合，并提高预测准确性。

随机森林的预测过程如下：

1. 从训练数据中随机选择一个子集，作为当前决策树的训练数据。
2. 对于每个决策树，随机选择一个特征作为分裂特征，并对该特征进行排序。
3. 对于每个决策树，使用排序后的特征进行分裂，直到满足停止条件（如最小叶子节点数）。
4. 对于每个决策树，使用训练数据计算每个叶子节点的平均值。
5. 对于输入的新数据，按照决策树的分裂规则遍历决策树，并在最后一个叶子节点获取平均值作为预测结果。

通过对多个决策树的预测结果进行平均，可得到随机森林的最终预测结果。

# 4.具体代码实例和详细解释说明

## 4.1 支持向量机

### 4.1.1 数据预处理

```python
import numpy as np
from sklearn import datasets
from sklearn.preprocessing import StandardScaler

# 加载数据
iris = datasets.load_iris()
X = iris.data
y = iris.target

# 标准化
sc = StandardScaler()
X = sc.fit_transform(X)
```

### 4.1.2 训练SVM

```python
from sklearn import svm

# 训练SVM
clf = svm.SVC(kernel='linear')
clf.fit(X, y)
```

### 4.1.3 预测

```python
# 预测
y_pred = clf.predict(X)
```
