# 1.简介
  


随着5G技术的发展，消费者的需求也越来越多样化、个性化，带来了巨大的商业价值和经济效益。5G在满足消费者需求的同时，也带来了新的经济机遇。5G网络在中国将会成为世界主要通信技术之一。因此，对5G的全面掌握，对经济发展至关重要。

在5G的应用和推广方面，中国已经取得了很好的进展。近年来，我国连续两年通过了28个5G频段，使得全国的5G覆盖率达到75%。但是，由于缺乏定量数据和规模经济实证分析，难以预测5G的经济效益。

2019年，国家主席习近平提出建立“一带一路”倡议，旨在加强互联互通领域合作，推动“一带一路”沿线国家共同打造更高水平开放型经济体系。为了帮助企业更好地理解、落实、部署5G技术，加快布局5G、推进5G产业整合，以及促进人类共同参与5G建设，中国共产党政府、国务院决定成立“5G-For-Economy（5G经济论坛）”。这是由国内顶尖学者、专家组成的行业组织，为中外学者提供了一个交流平台，探讨5G对经济发展的影响、对推动经济转型升级的作用、以及如何通过制度创新等方式推动5G经济领域的健康发展。

当前，国内有关研究者都在做关于5G经济影响的调研工作，但没有一个可以系统性回答以下几个方面的问题：

第一，对于5G如何影响经济发展，各研究者提出的理论和观点各不相同，有的认为5G是实体经济的必然趋势；有的认为5G只是在更快、更低的成本下替代无线宽带、5G终端设备逐渐集聚成各大城市街道、5G将会出现一种全新的形式——“云网”。有的则认为5G将直接改变某些产业链和生产环节的生产方式，可能会极大地刺激GDP增长。
第二，不同研究者对于5G产业链、经济基础设施、终端设备、应用场景等相关研究还存在很多疑问。比如，在产业链层面上，有的认为5G与传统电信业务融合、并进一步形成“一体化”，将会有利于提升电信服务质量和降低成本，但有的研究则认为5G的应用将完全取代或与传统电信业务分离，例如，可能通过5G虚拟网络实现企业之间、社区之间的远程协作、智慧园区及家庭监控等。
第三，虽然各个研究者对于5G的发展前景进行了比较客观的评估，但仍存在一些关键指标、数字无法准确反映5G在经济发展中的实际作用。比如，针对5G基础设施的支撑、终端设备的推广、以及对人口的影响等，目前尚缺乏系统性的数据来源和有效的分析方法。
第四，为了让各位读者更好地了解5G对经济发展的影响，中国共产党政府、国务院决定举办“5G For Economy”论坛。其目的是通过多种方式（包括学术会议、专家咨询、研究报告等），把国内外知名的5G经济研究者的理论研究成果与实践经验相结合，向大家展示5G经济领域的最新进展、展望未来发展方向，和鼓励企业和政府加强对5G经济的关注。

# 2.背景介绍

## 2.1  5G背景简介

5G（Wireless Broadband Communication Systems）是近几年才被提出的一个新词汇，它是一种使用无线电频率范围为2.4~30GHz的技术，能够突破传统的3G移动通信技术瓶颈，并进行高速、多样化的数据传输。其最大特点就是低时延、高带宽和超高可靠性，这为许多应用场景提供了前所未有的便利，也导致了5G在全球范围内迅猛扩张。

2013年，国家主席Huawei Technologies CEO余承东首次提出，将5G看作是“新的物理层、新的传输层和新的网络层”，与现有的3G/4G技术相比，具有巨大的变革性、颠覆性。2015年，美国联邦通信委员会发布的白皮书表明，美国5G项目的实现目标是：“到2020年，拥有10亿人同时使用5G网络。”

2016年，北京大学、清华大学、中国移动、厦门大学等一批著名院校纷纷宣布，将在2019年开展5G网络试验，目的是“打造世界级的社会、经济和科技架构”。从今年开始，更多的研究机构、企业、政府部门和学术界对5G的影响力、发展前景进行了深入研究。

## 2.2  5G为什么能够带来新的经济机遇？

5G虽然能够提供优异的性能，但它并不是一蹴而就的。首先，新技术往往需要时间去成熟，引入新机制，才能真正产生效益。在这一过程中，5G还面临着技术、基础设施、运营管理等方面的挑战。因此，在经济领域，5G要想真正落地，还需要一些时间。

相对于传统的固定宽带网络，5G网络有着独特的特征。首先，传统固定网络的采购价格较高，基础设施建设周期长，传统网络传输速率一般不能满足快速业务需求；而5G网络的价格却十分便宜，基础设施建设时间短，传送速度可以达到更高的水平。其次，5G网络可以提供更大的容量和数据传输能力，对于带宽需求较大的应用来说尤其重要。再者，5G网络具有良好的可靠性，可以抵御大部分干扰源和恶意攻击。最后，5G网络可以提供更丰富的应用场景，有利于加快信息化和云计算的发展。

总之，5G能够带来新的经济机遇，是因为它综合利用传感网、无线通信和机器学习技术，满足消费者的需求，以及降低成本、缩短网络距离、提升用户满意度。


# 3.核心概念术语说明

## 3.1  5G定义

定义：无线宽带通信系统(Wireless Broadband Communication System)或无线宽带电信系统(Wireless Broadband Communications and Telecommunications System)，是用无线电技术增强的高速、多样化的数据传输。它是在5G奠基前的产物，在2013年，主导者Huawei Technologies提出5G概念，称其“新的物理层、新的传输层和新的网络层”，与现有的3G/4G技术相比具有巨大的变革性、颠覆性。

## 3.2   5G技术架构

下图显示了5G技术架构。它包括物理层、传输层、网络层和应用层，并围绕着终端设备、数据中心和云计算等功能节点展开。


### 3.2.1   物理层

物理层（Physical Layer）负责传输物理信号，即比特流，这是一种0、1序列的传输方式，与其他传输方式不同，不需要编码器。

5G采用了新的无线技术，因此，5G系统中的各个设备必须配备相应的无线收发机，用于接收和发送无线信号。5G系统中的无线收发机通常由专用芯片集成电路设计，具有低功耗、高灵敏度、高带宽、低成本等特点。

5G中的无线传输信道是直径约10米的石墨烯、绝缘子绝缘层（Si-Ox layers）或微波系统的三极管场结构，能够在任何角度、任何时刻、任意位置进行无损、高带宽的双向传输。这种天线结构也可以有效降低空气污染、抑制干扰、减小功耗、优化性能。

### 3.2.2   传输层

传输层（Transport Layer）是实现数据封装、数据加密、控制消息完整性、控制数据的流量、控制流量分配、QoS策略以及流量调度等功能的模块。

#### 数据封装

数据封装（Data Encapsulation）是指将数据转换成适合网络传输的格式，如将文件打包成TCP/IP协议中的数据包，并按照相应的传输规范进行传输。

#### 数据加密

数据加密（Data Encryption）是指在网络上传输的数据，采用安全加密的方法，使得只有授权的通信双方才能读取数据。

5G系统中的所有数据都是加密传输的，这样就可以防止黑客截获数据后窃取私密信息。另外，5G系统还可使用混合加密技术，即结合使用静态加密和动态加密的方式，既保障数据的机密性，又保证数据的可用性。

#### 控制消息完整性

控制消息完整性（Control Message Integrity）是指利用数字签名验证消息的完整性，确保网络数据的可靠性。

5G系统采用安全认证协议，比如TLS协议，加密所有数据，并使用数字签名验证数据完整性。

#### 流量控制

流量控制（Traffic Control）是指控制网络资源的利用，根据网络当前的状态，动态调整用户之间的、主机之间的、服务之间的通信速率。

#### QoS策略

QoS策略（Quality of Service）是指基于网络流量、计算资源等多个因素对网络应用流量进行分类，并设置优先级、优先顺序等规则，以保证服务质量的最佳获得。

5G系统提供各种业务类型，如视频会议、视频直播、音频会议、游戏、视频下载等，这些业务都有自己的QoS要求。

#### 流量调度

流量调度（Traffic Shaping）是指依据网络当前的状况，对各个业务之间、各个主机之间以及网络和主机之间的通信流量进行调度，确保网络资源得到充分的利用。

5G系统可以利用多种手段进行流量调度，如通过网络带宽分配和共享方案对网络的使用权进行调度、采用QoS保证关键业务的实时性、通过流量整形技术对网络流量进行优化，以及利用动态QoS决策进行服务质量的自动调整。

### 3.2.3   网络层

网络层（Network Layer）是指为数据提供路由选择、寻址、处理和传输服务的模块。

5G的网络层使用不同的技术来提高传输效率，如成立单一的物理层、分片与重组技术，以及利用端到端的多跳路径传输数据。此外，5G采用并行多接入（PAA）方案，可以将不同业务类型的数据分别传输给对应业务类型的目的地，提高通信效率和资源利用率。

### 3.2.4   应用层

应用层（Application Layer）是指向最终用户提供应用程序接口，包括应用编程接口API、应用服务器以及支持应用的客户端。

5G网络的应用层有着丰富的业务功能，如视频会议、视频直播、文件传输、聊天、搜索、金融支付、远程医疗等。这些业务功能可以通过统一的标准化接口API来调用，以实现服务的泛用性。

## 3.3  5G产品分类

### 3.3.1   网络设备分类

#### 轻型设备

5G的轻型设备是指能够运行5G网络的传感器、微控制器、嵌入式终端设备、个人笔记本电脑、手机等。轻型设备通常不太占用空间，有着方便、省事的特点。它们使用的频率介于2.4GHz到5.8GHz之间。

#### 中型设备

5G的中型设备是指能够运行5G网络的企业级服务器、大型路由器、核心交换机、WLAN路由器等。它们的大小比轻型设备略大一些，通常安装在办公室、学校、医院等。它们的使用频率约束在2.15~6GHz之间。

#### 重型设备

5G的重型设备是指能够运行5G网络的超级计算机、大型集群、分布式数据库、大型交换机等。它们的安装规模与建筑物相比，规模更大，且安装在复杂的建筑物内。它们的使用频率约束在40GHz以上的极高频率。

### 3.3.2   服务形式分类

5G作为新一代的无线宽带通信系统，其带来的新业务形式更丰富、新应用更灵活，具体如下：

#### 普通通信业务

5G的普通通信业务是指利用5G网络在指定的频段上进行通信、通话、实时数据传输、视频会议、互动娱乐等业务。

#### 终端边缘计算

5G终端设备可以作为边缘计算平台，并将其连接到云端的统一计算中心，实时执行AI任务，实现智能物联网、智能医疗、智能农业等。

#### 综合业务

5G网络还可实现综合业务，如边缘计算、大数据、医疗等，通过软硬件结合的模式，实现数据智能化、精细化管理。

## 3.4  5G应用场景

### 3.4.1   移动应用

5G能够帮助移动应用享受到更优质的应用体验。由于5G的传输速度快、带宽宽，而且还可以使用各种智能手机上的应用程序进行通信，因此，移动应用中的5G通信应用更加普遍。例如，智能手机上的WhatsApp、Facebook Messenger、LINE、Viber、Telegram、YouTube Live等应用均支持5G网络。

### 3.4.2   电视盒子

5G能够帮助电视盒子和智能电视应用程序实现更高的播放速度、画质、视频流畅度、屏幕保护效果等。在实现更高质量的视频内容传输、高清视频显示的同时，还能降低运营商的费用开支。

### 3.4.3   医疗领域

5G能够提供高速医疗服务。医生的远程诊断、手术以及日常监护可以由5G网络实现，也可以在家远程观看患者体检结果。医疗领域的5G应用还有药物开发、远程治疗等。

### 3.4.4   工业领域

5G在工业领域的应用则集中在智能化机器人、智能装备、自动化机器人、机器人辅助诊断等方面。在机器人应用方面，5G有着“人工智能、工业互联网、机器人技术”三位一体的发展趋势。
