## 4.1 情感识别的代码实例

我们将使用Python的scikit-learn库来实现一个简单的情感识别模型。首先，我们需要收集和预处理数据。然后，我们可以使用TF-IDF来提取特征，并使用支持向量机（SVM）来训练模型。最后，我们可以使用测试数据集来评估模型的性能。

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 收集和预处理数据
data = [
    ("I am very happy today", "positive"),
    ("I am very sad today", "negative"),
    ("I am very angry today", "negative"),
    ("I am very excited today", "positive"),
    ("I am very disappointed today", "negative"),
    ("I am very calm today", "neutral"),
    ("I am very relaxed today", "neutral"),
    ("I am very tense today", "negative"),
    ("I am very stressed today", "negative"),
    ("I am very tired today", "neutral")
]

# 提取特征
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform([text for text, label in data])
y = [label for text, label in data]

# 训练模型
clf = SVC(kernel='linear', C=1)
clf.fit(X, y)

# 评估模型
X_test = vectorizer.transform(["I am very happy today", "I am very sad today"])
y_pred = clf.predict(X_test)
print("Accuracy:", accuracy_score(["positive", "negative"], y_pred))
```

我们将使用Python的nltk库来实现一个简单的情感表达生成模型。首先，我们需要收集和预处理数据。然后，我们可以使用RNN来生成情感表达的文本。最后，我们可以使用测试数据集来评估模型的性能。

```python
import numpy as np
import tensorflow as tf
from tensorflow.keras.preprocessing.text import Tokenizer
from tensorflow.keras.preprocessing.sequence import pad_sequences
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Embedding, LSTM, Dense
from tensorflow.keras.datasets import imdb

# 收集和预处理数据
tokenizer = Tokenizer(num_words=10000)
tokenizer.fit_on_texts(data)
X = tokenizer.texts_to_sequences(data)
X = pad_sequences(X)
y = np.array([1 if label == "positive" else 0])

# 训练模型
model = Sequential()
model.add(Embedding(input_dim=10000, output_dim=64, input_length=X.shape[1]))
model.add(LSTM(64))
```
