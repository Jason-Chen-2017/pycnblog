## 4.1 神经元模拟代码实例

以下是一个简单的神经元模拟代码实例：

```python
import numpy as np

class Neuron:
    def __init__(self, weight, bias):
        self.weight = weight
        self.bias = bias

    def forward(self, input):
        return 1 / (1 + np.exp(-np.dot(input, self.weight) - self.bias))

# 初始化神经元
neuron = Neuron([0.1, -0.2], 0.5)

# 输入数据
input_data = np.array([1, -1])

# 计算输出
output = neuron.forward(input_data)
print(output)
```

在这个代码实例中，我们首先定义了一个神经元类`Neuron`，包括输入权重、偏置等属性。然后我们初始化了一个神经元，并给出了输入数据。最后，我们通过调用神经元的`forward`方法，计算输出。
