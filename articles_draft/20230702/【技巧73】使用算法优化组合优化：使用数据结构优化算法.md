4.3.2. 股票组合动态规划算法

实现动态规划算法，根据股票组合中的股票数量，分别计算各种组合的收益率。

```python
def dynamic_program(stock_code, stock_name, stock_price, n, max_stock_number):
    dp = [[] for _ in range(max_stock_number + 1)]
    dp[0] = [float('inf')] * n
    dp[0][0] = 0

    for i in range(1, max_stock_number + 1):
        for j in range(0, n):
            if i == 0 or stock_price[j] == 0:
                dp[i][j] = dp[i - 1][j]
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - stock_price[j]] + stock_price[j])

    return dp[max_stock_number][-1]
```

4.3.3. 股票组合优化

通过调用动态规划算法，计算出最优的股票组合，并输出组合名称和组合收益率。

```python
def best_stock_combination(stock_code, stock_name, stock_price, n, max_stock_number):
    dp = dynamic_program(stock_code, stock_name, stock_price, n, max_stock_number)
    max_stock_number_and_return = max(dp, key=dp.get)
    best_stock_combination = [stock_name]
    for i in range(1, max_stock_number_and_return.index(dp[0]) + 1):
        best_stock_combination.append(str(i))
    return best_stock_combination, max_stock_number_and_return
```
