### 4.1.1 平均融合

```python
import numpy as np

X1 = np.array([[1, 2], [3, 4]])
X2 = np.array([[5, 6], [7, 8]])

Y = (X1 + X2) / 2

print(Y)
```

### 4.1.2 权重融合

```python

X1 = np.array([[1, 2], [3, 4]])
X2 = np.array([[5, 6], [7, 8]])

w1 = np.array([1, 1])
w2 = np.array([1, 1])

Y = (w1 * X1 + w2 * X2) / (w1 + w2)

print(Y)
```

```python

```

## 4.4 模型评估

### 4.4.1 准确率

```python

TP = 50
TN = 50
FP = 10
FN = 10

```

### 4.4.2 召回率

```python
import numpy as np

TP = 50
FN = 10

Recall = TP / (TP + FN)

print(Recall)
```

### 4.4.3 F1分数

```python
import numpy as np

TP = 50
FP = 10

Precision = TP / (TP + FP)

Recall = 50 / (50 + 10)

F1 = 2 * (Precision * Recall) / (Precision + Recall)

print(F1)
```
