# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
# 3.1 度中心性
度中心性是一个节点在社交网络中的重要性指标。度中心性可以用公式表示为：
$$
Degree Centrality = \frac{L}{N}
$$
其中，$L$ 是节点的度（即与其他节点的关系数），$N$ 是节点总数。度中心性越高，节点的位置越接近其他节点，这意味着它与其他节点之间的关系越紧密。

# 3.2 Betweenness Centrality
Betweenness Centrality 是一个节点在社交网络中的重要性指标。Betweenness Centrality 可以用公式表示为：
$$
Betweenness Centrality = \sum_{j \neq i \neq k} \frac{σ_{jk}(i)}{σ_{jk}}
$$
其中，$σ_{jk}$ 是从节点 $j$ 到节点 $k$ 的路径数，$σ_{jk}(i)$ 是经过节点 $i$ 的从节点 $j$ 到节点 $k$ 的路径数。Betweenness Centrality 越高，节点的位置越接近其他节点，这意味着它与其他节点之间的关系越紧密。

# 4.具体代码实例和详细解释说明
# 4.1 计算度中心性
要计算度中心性，我们需要首先构建社交网络。我们可以使用 Python 的 NetworkX 库来构建社交网络。然后，我们可以使用 NetworkX 库的 degree_centrality 函数来计算度中心性。

```python
import networkx as nx

# 构建社交网络
G = nx.Graph()

# 添加节点
G.add_node("A")
G.add_node("B")
G.add_node("C")

# 添加边
G.add_edge("A", "B")
G.add_edge("B", "C")

# 计算度中心性
degree_centrality = nx.degree_centrality(G)
print(degree_centrality)
```
