# 2.核心概念与联系

## 2.1 齐次无序单项式向量空间（Homogeneous Unordered Polynomial Spaces，HUPS）

齐次无序单项式向量空间是一种特殊的向量空间，其中元素是由多项式组成的。具体来说，HUPS中的多项式是由图像特征向量组成的，这些向量可以表示图像的颜色、形状、纹理等特征。HUPS可以看作是传统向量空间的拓展，它可以更好地处理图像之间的关系和结构。

## 2.2 图像特征向量

图像特征向量是用于描述图像特征的向量。常见的图像特征向量包括颜色特征向量、纹理特征向量、形状特征向量等。这些特征向量可以通过各种图像处理和分析方法得到，如主成分分析（PCA）、本心分析（ICA）、卷积神经网络（CNN）等。

## 2.3 多项式空间

多项式空间是一种数学概念，它是由多项式组成的向量空间。多项式空间可以用来表示和处理各种类型的数据，包括图像、声音、文本等。在图像分类任务中，我们可以将图像特征向量看作是多项式空间中的元素，然后通过计算多项式之间的关系和结构，来实现图像分类。

    polynomial_space = np.zeros((n_samples, n_features * (degree + 1)))
