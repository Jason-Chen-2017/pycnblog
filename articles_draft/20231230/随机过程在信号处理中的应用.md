# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明随机过程在信号处理中的应用。我们将使用Python编程语言来实现一个白噪声生成和滤波处理的例子。

```python
import numpy as np
import matplotlib.pyplot as plt

# 白噪声生成
def white_noise(sample_rate, duration, amplitude):
    num_samples = int(sample_rate * duration)
    noise = np.random.normal(0, amplitude, num_samples)
    return noise

# 高通滤波
def high_pass_filter(signal, cutoff_frequency, sample_rate):
    nyquist_frequency = 0.5 * sample_rate
    normal_cutoff = cutoff_frequency / nyquist_frequency
    b, a = signal.butter(2, normal_cutoff, btype='high', analog=False)
    filtered_signal = signal.lfilter(b, a, signal)
    return filtered_signal

# 低通滤波
def low_pass_filter(signal, cutoff_frequency, sample_rate):
    nyquist_frequency = 0.5 * sample_rate
    normal_cutoff = cutoff_frequency / nyquist_frequency
    b, a = signal.butter(2, normal_cutoff, btype='low', analog=False)
    filtered_signal = signal.lfilter(b, a, signal)
    return filtered_signal

# 主程序
if __name__ == '__main__':
    sample_rate = 1000  # 采样率
    duration = 1  # 信号持续时间
    amplitude = 1  # 噪声幅值
    cutoff_frequency = 100  # 滤波器截止频率

    # 生成白噪声
    noise = white_noise(sample_rate, duration, amplitude)

    # 高通滤波
    filtered_noise = high_pass_filter(noise, cutoff_frequency, sample_rate)

    # 低通滤波
    filtered_noise = low_pass_filter(filtered_noise, cutoff_frequency, sample_rate)

    # 绘制信号和滤波后的信号
    plt.figure()
    plt.subplot(2, 1, 1)
    plt.plot(noise)
    plt.title('White Noise')

    plt.subplot(2, 1, 2)
    plt.plot(filtered_noise)
    plt.title('Filtered White Noise')

    plt.show()
```

在上述代码中，我们首先定义了白噪声生成和滤波处理的函数。然后，我们通过主程序来生成白噪声，并使用高通滤波和低通滤波来处理白噪声。最后，我们绘制了信号和滤波后的信号的波形图。
