```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机数据
np.random.seed(0)
x = np.random.rand(100)
y = 3 * x + np.random.randn(100)

# 使用最小二乘法进行拟合
X = np.array([x, np.ones(len(x))]).T
beta = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)

# 预测
x_test = np.linspace(0, 10, 100)
y_test = 3 * x_test + np.random.randn(100)
y_pred = X.dot(beta)

# 绘图
plt.scatter(x, y, label='数据点')
plt.plot(x_test, y_pred, 'r-', label='拟合曲线')
plt.legend()
plt.show()
```

在这个代码实例中，我们使用了最小二乘法进行简单的线性回归拟合。首先，我们生成了一组随机的 $x$ 和 $y$ 数据，其中 $y = 3x + \epsilon$，其中 $\epsilon$ 是随机噪声。然后，我们使用最小二乘法进行拟合，得到了参数向量 $\beta$。最后，我们使用得到的参数向量进行预测，并绘制了数据点和拟合曲线。
