## 2.1 二次方程的基本概念

二次方程是一种数学方程，其通用形式如下：

$$
ax^2 + bx + c = 0
$$

其中，$a, b, c$ 是实数，$a \neq 0$。

根据不等式 $a > 0$ 或 $a < 0$，二次方程可以分为以下两类：

1. 正二次方程：$a > 0$，其解一般为实数。
2. 负二次方程：$a < 0$，其解可能为实数或复数。

## 4.1 Python实现

我们使用Python编程语言来实现二次方程的解算。

```python
import math

def solve_quadratic(a, b, c):
    k = b**2 - 4*a*c
    if k > 0:
        x1 = (-b + math.sqrt(k)) / (2*a)
        x2 = (-b - math.sqrt(k)) / (2*a)
        return (x1, x2)
    elif k == 0:
        x = -b / (2*a)
        return (x,)
    else:
        real_part = -b / (2*a)
        imaginary_part = math.sqrt(-k) / (2*a)
        return (complex(real_part, imaginary_part), complex(real_part, -imaginary_part))
```

## 4.2 Java实现

```java
public class QuadraticEquationSolver {
    public static void main(String[] args) {
        double a = 1;
        double b = -3;
        double c = 2;
        double[] x = solveQuadratic(a, b, c);
        for (double value : x) {
            System.out.println(value);
        }
    }

    public static double[] solveQuadratic(double a, double b, double c) {
        double k = b * b - 4 * a * c;
        if (k > 0) {
            double x1 = (-b + Math.sqrt(k)) / (2 * a);
            double x2 = (-b - Math.sqrt(k)) / (2 * a);
            return new double[]{x1, x2};
        } else if (k == 0) {
            double x = -b / (2 * a);
            return new double[]{x};
        } else {
            double realPart = -b / (2 * a);
            double imaginaryPart = Math.sqrt(-k) / (2 * a);
            return new double[]{new Complex(realPart, imaginaryPart), new Complex(realPart, -imaginaryPart)};
        }
    }
}

class Complex {
    double realPart;
    double imaginaryPart;

    public Complex(double realPart, double imaginaryPart) {
        this.realPart = realPart;
        this.imaginaryPart = imaginaryPart;
    }

    @Override
    public String toString() {
        return realPart + " + " + imaginaryPart + "i";
    }
}
```

上述代码实现了二次方程的解算功能。我们首先计算分式 $k$，然后根据分式的值来判断方程的解的数量和类型，最后根据分式的值计算解的具体值。
