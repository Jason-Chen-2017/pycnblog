## 4.1 Wireshark代码实例

Wireshark的代码实例如下：

```python
import wireshark

# 捕捉网络流量
wireshark.start_capture("eth0")

# 使用显示滤器来过滤我关心的数据包
display_filter = "tcp and port 80"
wireshark.apply_display_filter(display_filter)

# 使用协议解析器来解析数据包的内容
protocol_analyzer = wireshark.ProtocolAnalyzer("http")
wireshark.add_protocol_analyzer(protocol_analyzer)

# 停止捕捉网络流量
wireshark.stop_capture()
```

在这个代码实例中，我们使用Wireshark来捕捉网络流量，使用显示滤器来过滤我关心的数据包，并使用协议解析器来解析数据包的内容。

## 4.2 TShark代码实例

```bash
# 捕捉网络流量

# 使用协议解析器来解析数据包的内容
```
