### 4.1.1 心率变化的数据收集

首先，我们需要收集心率变化的数据。我们可以使用心率监测器来实时收集心率数据。

```python
import time

def collect_heart_rate_data():
    heart_rate_data = []
    while True:
        heart_rate = get_heart_rate()  # 获取心率数据
        heart_rate_data.append(heart_rate)
        time.sleep(1)
    return heart_rate_data
```

### 4.1.2 计算心率变化的标准差

接下来，我们可以使用 NumPy 库来计算心率变化的标准差。

```python
import numpy as np

def calculate_heart_rate_std(heart_rate_data):
    delta_t = 1  # 观察时间段为 1 秒
    t1 = 0
    t2 = delta_t
    sigma = np.std(heart_rate_data[t1:t2])
    return sigma
```

```python
import time

def collect_attention_data():
    attention_data = []
    while True:
        event = get_keyboard_event()  # 获取键盘事件
        if event:
            attention_data.append(1)
        else:
            attention_data.append(0)
        time.sleep(1)
    return attention_data
```

### 4.2.2 计算注意力累积值

接下来，我们可以使用 NumPy 库来计算注意力累积值。

```python
import numpy as np

def calculate_attention_accumulation(attention_data):
    T = 10  # 观察时间段为 10 秒
    t1 = 0
    t2 = T
    A_t = np.trapz(attention_data[t1:t2], np.arange(t1, t2))
```
