# 2.核心概念与联系

核函数是一种用于计算两个样本之间距离的函数，它可以用来计算输入空间中的数据点之间的相似性或距离。核函数的核心概念是“核心”，即它将输入空间中的数据点映射到高维空间中，从而实现非线性问题的解决。核函数的主要特点是：

1. 非线性映射：核函数可以将输入空间中的数据点映射到高维空间，从而实现非线性问题的解决。
2. 内积计算：核函数通过计算输入空间中的数据点之间的内积来实现非线性映射。
3. 高维空间映射：核函数将输入空间中的数据点映射到高维空间，从而实现非线性问题的解决。

在金属材料学中，核函数被广泛应用于预测、优化和分类等任务。例如，在预测金属合成材料的性能时，核函数可以用来计算不同金属合成材料之间的相似性，从而实现预测。在优化金属合成过程时，核函数可以用来计算不同过程参数之间的相关性，从而实现优化。在分类金属材料时，核函数可以用来计算不同材料类型之间的距离，从而实现分类。

### 3.1.1 径向基函数核（Radial Basis Function, RBF）

径向基函数核是一种常用的核函数，它可以用来计算输入空间中的数据点之间的距离。径向基函数核的公式如下：

$$
K(x, y) = \exp(-\gamma \|x - y\|^2)
$$

其中，$x$ 和 $y$ 是输入空间中的两个数据点，$\gamma$ 是核参数，$\|x - y\|^2$ 是两个数据点之间的欧氏距离。

### 3.1.2 多项式核

多项式核是一种用于计算输入空间中的数据点之间相似性的核函数，它的公式如下：

$$
K(x, y) = (1 + \langle x, y \rangle)^d
$$

其中，$x$ 和 $y$ 是输入空间中的两个数据点，$\langle x, y \rangle$ 是它们之间的内积，$d$ 是核参数。

### 3.1.3 高斯核

高斯核是一种常用的径向基函数核，它的公式如下：

$$
K(x, y) = \exp(-\gamma \|x - y\|^2)
$$

其中，$x$ 和 $y$ 是输入空间中的两个数据点，$\gamma$ 是核参数，$\|x - y\|^2$ 是两个数据点之间的欧氏距离。

在这个例子中，我们选择径向基函数核（RBF）作为核函数，并通过交叉验证来调整核参数$\gamma$。

from sklearn.model_selection import GridSearchCV
from sklearn.metrics import mean_squared_error

def rbf_kernel(x, y, gamma):
    return np.exp(-gamma * np.linalg.norm(x - y)**2)
