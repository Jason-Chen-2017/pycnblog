## 3.1 电阻计算公式

$$
R = \rho \frac{l}{A}
$$

其中，$\rho$ 是材料的电导率，$l$ 是电流通过的距离，$A$ 是电流通过的面积。

$$
I = \frac{V}{R}
$$

在拓扑绝缘体的情况下，电导的计算公式仍然适用，但是由于绝缘体的布局和材料的不同，电导的大小会发生变化。

```python
import math

def resistance(rho, l, A):
    return rho * l / A

def conductance(V, R):
    return V / R

def capacitance(epsilon, A, d):
    return epsilon * A / d

def inductance(mu, N, l, A):
    return mu * N * l / A

rho = 1.0e-8  # 电导率
l = 1.0       # 距离
A = 1.0e-6    # 面积

R = resistance(rho, l, A)
print("Resistance:", R)

V = 10.0       # 电压
R = 10.0       # 电阻
I = conductance(V, R)
print("Conductance:", I)

epsilon = 8.85e-12  # 电容性
A = 1.0e-6         # 面积
d = 1.0e-4         # 距离
C = capacitance(epsilon, A, d)
print("Capacitance:", C)

mu = 1.25e-6      # 磁导率
N = 100           # turns 数
l = 1.0           # 距离
A = 1.0e-6        # 面积
L = inductance(mu, N, l, A)
print("Inductance:", L)
```
