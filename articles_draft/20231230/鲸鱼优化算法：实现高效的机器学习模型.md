```python
def init_population(n, lb, ub):
    return lb + (ub - lb) * np.random.rand(n, 1)

n = 10
lb = -10
ub = 10
X = init_population(n, lb, ub)
```

接下来我们实现鲸鱼优化算法的主要步骤，包括：

1. 鲸鱼群的分布式搜索

```python
def search(X, f, lb, ub, max_iter):
    for t in range(max_iter):
        # 计算每个鲸鱼的适应度
        fitness = f(X)
        # 根据适应度对鲸鱼群进行排序
        sorted_idx = np.argsort(fitness)
        # 更新鲸鱼群的位置
        X = X[sorted_idx]
        # 自主决策
        X = X - (X - X.mean(axis=0)) * 0.5
        # 适应性调整

    return X

    X = search(X, f, lb, ub, max_iter)
```
