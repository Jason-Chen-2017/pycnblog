```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 计算数据质量

print('数据质量:', quality)
```

在上述代码中，我们首先使用 pandas 库读取数据，然后计算数据中缺失值的比例，得到数据质量评估结果。
