### 4.1 代码实例

以下是一个简单的家庭智能门铰开锁接口的代码实例：

```python
import requests

def unlock_door(api_key, door_id):
    url = f"https://api.smartlock.com/v1/doors/{door_id}/unlock"
    headers = {
        "Authorization": f"Bearer {api_key}"
    }
    response = requests.post(url, headers=headers)
    if response.status_code == 200:
        print("门铰已解锁")
    else:
        print("解锁失败")
```

这个代码实例主要包括以下几个方面：

- 导入requests库：这个库用于发起HTTP请求。
- 定义unlock_door函数：这个函数用于实现门铰的开锁操作。
- 发起HTTP POST请求：通过requests.post()函数发起HTTP POST请求，实现门铰的开锁操作。
- 判断响应状态码：通过response.status_code判断响应状态码，如果状态码为200，表示门铰已解锁；否则，表示解锁失败。
