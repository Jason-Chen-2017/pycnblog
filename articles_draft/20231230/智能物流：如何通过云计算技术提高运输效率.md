## 4.1 物流运输数据收集与存储的代码实例

```python
import requests

url = 'http://example.com/api/v1/shipments'
headers = {'Content-Type': 'application/json'}
data = {'shipments': [{'id': 1, 'origin': 'Beijing', 'destination': 'Shanghai', 'weight': 1000}]}
response = requests.post(url, json=data, headers=headers)

if response.status_code == 201:
    print('Shipment data collected and stored successfully.')
else:
    print('Failed to collect and store shipment data.')
```

## 4.3 物流运输决策的代码实例

```python
from scipy.optimize import linprog

c = [-1, -1]  # Cost coefficients
A = [[1, 1], [1, 0]]  # Technological coefficients
b = [100, 100]  # Limits

x = linprog(c, A_ub=A, b_ub=b)

print(f'Optimal number of trucks: {x.x[0]:.0f}, Optimal number of drivers: {x.x[1]:.0f}')
```
