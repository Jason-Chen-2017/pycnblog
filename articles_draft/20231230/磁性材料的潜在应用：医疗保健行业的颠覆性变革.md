1. 将病人放入磁共振成像仪器。
2. 通过外部磁场使体内水分子产生旋转运动。
3. 通过附加的衰减磁场使水分子的旋转运动停止，产生信号。

### 1.4.1 磁共振成像（MRI）代码实例
在本例中，我们将通过Python编程语言来实现磁共振成像的基本功能。首先，我们需要导入所需的库：

```python
import numpy as np
import matplotlib.pyplot as plt
```

接下来，我们需要定义磁共振成像的核心参数：

```python
B0 = 3.0  # 外部磁场强度（T）
gamma = 4.26e-3  # 水分子磁化率（T·m/A）
M0 = 1.0e-3  # 水分子磁化moment（A/m）
omega = 6.28  # 旋转频率（rad/s）
dt = 0.001  # 时间步长（s）
```

然后，我们需要定义磁共振成像的核心算法：

```python
def MRI(B0, gamma, M0, omega, dt):
    S = gamma * np.pi * M0 * omega * dt
    M = M0 / np.pi
    B = B0 * M
    return B
```

最后，我们需要使用上述算法来构建磁共振成像的图像：

```python
def plot_MRI(B0, gamma, M0, omega, dt, x, y, z):
    B = MRI(B0, gamma, M0, omega, dt)
    plt.imshow(B, cmap='hot', extent=[x, y, z, 0])
    plt.colorbar()
    plt.show()
```

通过以上代码实例，我们可以看到磁共振成像的基本原理和实现。
