## 2.1 有序单项式向量空间的定义

有序单项式向量空间（Ordered Single-Form Vector Spaces, OSFVS）是一种特殊的向量空间，其中向量的线性组合是有序的。具体来说，一个向量空间（V, +, .）是一个集合V，其中元素称为向量，加法操作符+和数乘操作符.满足以下几个条件：

1. 对于任意两个向量a, b在V中，a + b也在V中。
2. 对于任意向量a在V中和任意数字α在实数域F中，αa也在V中。
3. 对于任意向量a, b, c在V中和数字α在F中，有以下性质成立：
   - (a + b) + c = a + (b + c)
   - α(a + b) = αa + αb
   - (αa) + (βb) = (α + β)(a + b)
   - α(βa) = (αβ)(a)
   - 1a = a

有序单项式向量空间的定义需要在向量空间基础上添加一个有序性质。具体来说，有序单项式向量空间（V, +, ., ≤）是一个向量空间（V, +, .）加上一个偏序关系≤，使得：

1. 对于任意向量a, b在V中，如果a ≤ b，那么a + b = b。
2. 对于任意向量a, b在V中，如果a ≤ b，那么对于任意数字α在F中，αa ≤ αb。
