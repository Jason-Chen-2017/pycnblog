## 4.1 代码实例

```python
import numpy as np

# 构建有序集合
words = ['apple', 'banana', 'cherry']

# 构建多项式
def polynomial(words):
    terms = {}
    for word in words:
        terms[word] = 1
    return terms

# 计算多项式之间的关系和依赖
def compute_relations(polynomials):
    relations = {}
    for polynomial in polynomials:
        for term in polynomial.keys():
            if term not in relations.keys():
                relations[term] = 0
            relations[term] += polynomial[term]
    return relations

# 使用关系和依赖来进行词嵌入
def word_embedding(relations, dimension):
    embedding = np.zeros((len(relations), dimension))
    for i, word in enumerate(relations.keys()):
        embedding[i, :] = relations[word]
    return embedding

# 测试代码
words = ['apple', 'banana', 'cherry']
polynomials = [polynomial(words)]
relations = compute_relations(polynomials)
embedding = word_embedding(relations, 2)
print(embedding)
```

## 4.2 详细解释说明

1. 首先，我们构建了一个有序集合 `words`，其中包含了三个词语 `['apple', 'banana', 'cherry']`。

2. 接下来，我们定义了一个 `polynomial` 函数，用于构建多项式。在这个例子中，我们将每个词语看作一个多项式的项，其系数为 1。

3. 然后，我们定义了一个 `compute_relations` 函数，用于计算多项式之间的关系和依赖。在这个例子中，我们将每个词语的系数相加，得到关系和依赖。

4. 最后，我们定义了一个 `word_embedding` 函数，用于将关系和依赖转换为词嵌入。在这个例子中，我们将关系和依赖转换为一个 NumPy 数组，其中每行对应一个词语，每列对应一个维度。

5. 最后，我们测试了代码，并输出了词嵌入结果。
