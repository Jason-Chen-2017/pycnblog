### 3.1.1 水资源利用效率模型

水资源利用效率模型是用于评估水资源利用过程中的效率的模型。水资源利用效率可以通过以下公式计算：

$$
\eta = \frac{W_{out}}{W_{in} + E_{out}}
$$

其中，$\eta$ 表示水资源利用效率，$W_{out}$ 表示输出水量，$W_{in}$ 表示输入水量，$E_{out}$ 表示外部成本。

### 3.1.2 水资源可持续性评估模型

水资源可持续性评估模型是用于评估水资源可持续性的模型。常见的水资源可持续性评估模型有：

- 水资源可持续性指数（WASI）
- 水资源可持续性评估模型（SWA）

### 4.1.2 水资源可持续性评估模型代码实例

#### 4.1.2.1 水资源可持续性指数（WASI）代码实例

```python
def wasi(W_avail, W_demand, R_avail, R_demand):
    return (W_avail / W_demand) * (R_avail / R_demand)
```

#### 4.1.2.2 水资源可持续性评估模型（SWA）代码实例

```python
def swa(W_avail, W_demand, R_avail, R_demand, W_use, R_use):
    return (W_avail / W_demand) * (R_avail / R_demand) * (W_use / W_avail) * (R_use / R_avail)
```

```python
from scipy.optimize import minimize

def multi_objective_optimization(x, W_avail, W_demand, R_avail, R_demand, W_use, R_use):
    def objective(x):
        W_avail = x[0]
        W_demand = x[1]
        R_avail = x[2]
        R_demand = x[3]
        W_use = x[4]
        R_use = x[5]
        return -(W_avail / W_demand) * (R_avail / R_demand) * (W_use / W_avail) * (R_use / R_avail)
    return minimize(objective, bounds=[(0, 1000), (0, 1000), (0, 1000), (0, 1000), (0, 1000), (0, 1000)])
```
