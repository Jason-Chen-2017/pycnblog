## 3.1 时空变换

时空变换的数学模型公式如下：

$$
x' = \gamma (x - vt) \\
t' = \gamma (t - vx/c^2)
$$

其中，$x$ 和 $t$ 是原始坐标系中的空间和时间坐标，$x'$ 和 $t'$ 是变换后的坐标系中的空间和时间坐标，$v$ 是粒子的速度，$c$ 是光速，$\gamma$ 是Lorentz变换因子，可以表示为：

$$
\gamma = \frac{1}{\sqrt{1 - \frac{v^2}{c^2}}}
$$

时空变换的具体操作步骤如下：

1. 确定原始坐标系中的粒子位置和时间。
2. 计算Lorentz变换因子$\gamma$。
3. 使用时空变换公式计算变换后的粒子位置和时间。

## 4.1 时空变换代码实例
```python
import numpy as np

def lorentz_transformation(x, t, v):
    gamma = 1 / np.sqrt(1 - (v / c)**2)
    x_prime = gamma * (x - v * t)
    t_prime = gamma * (t - (v / c**2) * x)
    return x_prime, t_prime

x = 10
t = 5
v = 0.8 * c
x_prime, t_prime = lorentz_transformation(x, t, v)
print("变换后的粒子位置：", x_prime)
print("变换后的粒子时间：", t_prime)
```
