### 4.1指数分布的代码实例

在Python中，我们可以使用`scipy.stats`模块来计算指数分布的概率。以下是一个计算指数分布概率的代码实例：

```python
import numpy as np
from scipy.stats import exponweib

# 设置参数
alpha = 1
beta = 2
x = 3

# 计算概率
prob = exponweib.cdf(x, alpha, scale=beta)
print("指数分布的概率为:", prob)
```

在这个代码实例中，我们首先导入了`numpy`和`scipy.stats`模块。然后，我们设置了指数分布的参数$\alpha$ 和 $\beta$，以及我们想要计算的取值$x$。最后，我们使用`exponweib.cdf`函数计算指数分布的概率，并输出结果。

### 4.2伽马分布的代码实例

