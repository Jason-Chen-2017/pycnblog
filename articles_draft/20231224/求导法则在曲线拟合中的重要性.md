```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import curve_fit

def f(x, a, b, c):
    return a*x**2 + b*x + c

x = np.linspace(-10, 10, 100)
y = np.sin(x)

popt, pcov = curve_fit(f, x, y)

plt.plot(x, y, label='y = sin(x)')
plt.plot(x, f(x, *popt), label='f(x, a, b, c)', linestyle='--')
plt.legend()
plt.show()
```

### 4.3.2 曲线拟合的详细解释说明
在这个代码实例中，我们定义了一个函数f(x, a, b, c)，它是一个二次方程，用于拟合y = sin(x)。我们使用numpy库来创建一个x和y数组，从-10到10，步长为0.1。我们使用这个x和y数组来调用scipy库中的curve_fit函数，来拟合y数据点。curve_fit函数返回了最佳拟合参数popt和参数估计误差pcov。最后，我们使用matplotlib库来绘制y和拟合曲线的图像。
