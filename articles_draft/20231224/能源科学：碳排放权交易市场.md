# 2.核心概念与联系

碳排放权交易市场的核心概念包括：碳排放权、排放权交易、排放权市场等。下面我们将详细介绍这些概念及其之间的联系。

## 2.1 碳排放权

碳排放权是指一种证券，表示其持有者有权利在一定时间内在指定的碳排放限制范围内进行排放。碳排放权可以被视为一种资产，可以通过交易来交换价值。

碳排放权的核心目的是通过市场化的方式将碳排放限制转化为可交易的资产，从而激励企业积极减排，提高排放效率。

## 2.2 排放权交易

排放权交易是指在碳排放权交易市场中，企业通过购买和出售碳排放权来满足其排放需求的过程。排放权交易可以分为两种类型：

1. 前向交易：企业在未来的某个时间点预购碳排放权，以预先满足其排放需求。
2. 后向交易：企业在某个时间点购买已经发行的碳排放权，以满足当期排放需求。

## 2.3 排放权市场
