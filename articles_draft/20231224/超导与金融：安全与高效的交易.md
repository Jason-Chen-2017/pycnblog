# 2.核心概念与联系

## 2.1 超导材料

超导材料是超导技术的基础，它们在低温下能够进行无电阻的电导。超导材料的发现可以追溯到1911年，当时荷兰物理学家赫尔曼·科普兰（Heike Kamerlingh Onnes）在实验中发现了氢的液体状态。随后，他继续研究其他元素和合成物，最终在1913年发现了第一个超导材料——氢氧化钠（H3O）。

超导材料的特点是在某个称为“超导临界温度”（Tc）的温度下，其电阻为零。这意味着在这个温度下，超导材料可以无限制地传输电力，从而实现高效的电力传输。

$$


```python
def string_to_ascii(string):
    ascii_list = [ord(char) for char in string]
    return ascii_list

def ascii_to_binary(ascii_list):
    binary_list = [bin(ascii_value).replace("0b", "") for ascii_value in ascii_list]
    return binary_list

string = "Hello, World!"
ascii_list = string_to_ascii(string)
binary_list = ascii_to_binary(ascii_list)
```
