## 2.1向量空间模型

向量空间模型（Vector Space Model）是一种用于表示文本信息的方法，它将文本转换为一个由词语组成的向量。在向量空间模型中，每个词语都有一个向量，表示该词语在文本中的重要性。向量空间模型可以用来表示文本之间的关系，如文本相似度、文本分类等。

## 2.2向量内积

向量内积（Dot Product）是一个数学概念，它表示两个向量在同一空间中的投影积。向量内积的公式如下：

$$
\mathbf{a} \cdot \mathbf{b} = \|\mathbf{a}\| \|\mathbf{b}\| \cos \theta
$$

其中，$\mathbf{a}$ 和 $\mathbf{b}$ 是两个向量，$\|\mathbf{a}\|$ 和 $\|\mathbf{b}\|$ 是它们的长度，$\theta$ 是它们之间的夹角。向量内积可以用来计算两个向量之间的相似度，如文本相似度计算、文本分类等。

## 2.3余弦相似度

余弦相似度（Cosine Similarity）是一个用于计算两个向量之间相似度的度量，它的公式如下：

$$
\text{Cosine Similarity} = \frac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\| \|\mathbf{b}\|}
$$

其中，$\mathbf{a}$ 和 $\mathbf{b}$ 是两个向量，$\|\mathbf{a}\|$ 和 $\|\mathbf{b}\|$ 是它们的长度。余弦相似度的范围是 [-1, 1]，其中 1 表示两个向量完全相似，-1 表示两个向量完全不相似，0 表示两个向量完全不相关。

## 2.4欧氏距离

欧氏距离（Euclidean Distance）是一个用于计算两个向量之间距离的度量，它的公式如下：

$$
\text{Euclidean Distance} = \sqrt{(\mathbf{a} - \mathbf{b}) \cdot (\mathbf{a} - \mathbf{b})}
$$

其中，$\mathbf{a}$ 和 $\mathbf{b}$ 是两个向量。欧氏距离是一种度量空间中两点之间的距离的方法，它可以用来计算文本之间的距离，如文本聚类等。

## 4.1文本表示

我们将使用TfidfVectorizer类来实现文本表示。具体代码实例如下：

```python
from sklearn.feature_extraction.text import TfidfVectorizer

documents = [
    'this is the first document',
    'this is the second second document',
    'and the third one'
]

vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(documents)
```

在上面的代码中，我们首先导入TfidfVectorizer类，然后定义一个文本列表，其中包含三个文本。接着，我们使用TfidfVectorizer类来构建词频矩阵，并将其转换为向量。

## 4.2文本相似度计算

我们将使用cosine_similarity函数来计算文本相似度。具体代码实例如下：

```python
from sklearn.metrics.pairwise import cosine_similarity

similarity = cosine_similarity(X, X)
print(similarity)
```

在上面的代码中，我们首先导入cosine_similarity函数，然后将X矩阵与自身进行比较，以计算文本相似度。最后，我们将相似度矩阵打印出来。

## 4.3文本分类

我们将使用LogisticRegression类来实现文本分类。具体代码实例如下：

```python
from sklearn.linear_model import LogisticRegression

X_train = X[:2]
y_train = [0, 1]
X_test = X[2:]
y_test = [1]

classifier = LogisticRegression()
classifier.fit(X_train, y_train)

y_pred = classifier.predict(X_test)
print(y_pred)
```

在上面的代码中，我们首先导入LogisticRegression类，然后将X矩阵划分为训练集和测试集。接着，我们使用LogisticRegression类来构建分类器，并将训练集与对应的标签进行训练。最后，我们将测试集与对应的标签进行预测，并将预测结果打印出来。
