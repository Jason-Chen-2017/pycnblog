4.3. 核心代码实现

下面是一个使用Java和Java  EE实现高性能的并行计算的例子：

```java
import java.util.*;

public class ParallelExample {
    public static void main(String[] args) {
        int[] numbers = new int[1000];
        for (int i = 0; i < numbers.length; i++) {
            numbers[i] = i;
        }

        List<Integer> list = new ArrayList<Integer>();
        list.addAll(numbers);

        ParallelExecutorService executor = Executors.newFixedThreadPool(numbers.length);

        for (Integer number : list) {
            executor.submit(() -> {
                int result = number;
                System.out.println(result);
            });
        }

        executor.shutdown();
    }
}
```

在这个例子中，我们首先创建一个包含1000个整数的数组。然后，我们将数组中的每个元素添加到一个新的列表中。接下来，我们创建了一个并行执行器，用于执行对数组的并行计算。我们使用executor.submit()方法提交一个任务，用于计算数组中的每个元素。最后，我们使用executor.shutdown()方法关闭并行执行器。
