```python

# 数据接收

print(data.decode())

# 数据处理
data = data.decode()
```

3. 控制算法代码实例：

```python
import numpy as np

# PID控制
kp = 1
ki = 1
kd = 1

def pid(error, prev_error, dt):
    return kp * error + ki * prev_error + kd * dt

# LPV控制
def lpv(x, u):
    return x * u
```
