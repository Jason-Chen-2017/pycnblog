## 1.2 核心概念与联系

### 1.2.1 数组的基本概念

数组是一种线性数据结构，用于存储相同类型的数据元素。数组中的元素可以通过下标进行访问和修改。数组的长度是固定的，一旦创建，就不能改变。数组在处理大量数据时具有很高的效率，因为它可以通过直接访问元素的下标来访问数据，而不需要遍历整个数据结构。

### 1.2.2 集合的基本概念

集合是一种非线性数据结构，用于存储不同类型的数据元素。集合中的元素可以通过迭代器进行访问和修改。集合的长度是可变的，可以在运行时动态添加或删除元素。集合在处理不同类型的数据时具有很高的灵活性，因为它可以存储任意类型的数据元素。

### 1.2.3 数组与集合的联系

数组和集合都是Java编程中常用的数据结构，但它们在存储和访问数据方面有所不同。数组用于存储相同类型的数据元素，而集合用于存储不同类型的数据元素。数组在处理大量数据时具有很高的效率，因为它可以通过直接访问元素的下标来访问数据，而不需要遍历整个数据结构。集合在处理不同类型的数据时具有很高的灵活性，因为它可以存储任意类型的数据元素。

## 1.4 具体代码实例和详细解释说明

### 1.4.1 数组的实例

```java
public class ArrayExample {
    public static void main(String[] args) {
        // 创建数组
        int[] array = new int[5];

        // 初始化数组
        array[0] = 1;
        array[1] = 2;
        array[2] = 3;
        array[3] = 4;
        array[4] = 5;

        // 访问数组
        int value = array[2];
        System.out.println(value); // 输出：3

        // 遍历数组
        for (int i = 0; i < array.length; i++) {
            System.out.println(array[i]);
        }
    }
}
```

### 1.4.2 集合的实例

```java
import java.util.HashSet;

public class SetExample {
    public static void main(String[] args) {
        // 创建集合
        HashSet<Integer> set = new HashSet<Integer>();

        // 初始化集合
        set.add(1);
        set.add(2);
        set.add(3);
        set.add(4);
        set.add(5);

        // 访问集合
        for (Integer value : set) {
            System.out.println(value);
        }

        // 遍历集合
        for (Integer value : set) {
            System.out.println(value);
        }
    }
}
```
