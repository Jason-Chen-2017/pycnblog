## 3.1 比对算法

比对算法是用于比较两个基因组序列之间的相似性的算法。比对算法的核心思想是通过比较两个序列之间的相似性，从而找到它们之间的共同区域。比对算法的一个重要应用是基因组比对，即比较两个基因组之间的相似性，以便更好地了解它们之间的关系。

比对算法的一个常见的实现方式是Needleman-Wunsch算法。Needleman-Wunsch算法是一种动态规划算法，它的核心思想是通过比较两个序列之间的每个位置的相似性，从而找到它们之间的最佳匹配。Needleman-Wunsch算法的具体步骤如下：

1. 初始化一个矩阵，矩阵的行数为序列1的长度，列数为序列2的长度。
2. 对于矩阵中的每个位置，计算该位置的相似性分数。
3. 对于矩阵中的每个位置，计算该位置的最佳匹配分数。
4. 对于矩阵中的每个位置，计算该位置的最佳匹配路径。
5. 对于矩阵中的每个位置，计算该位置的最佳匹配值。

需要注意的是，Needleman-Wunsch算法的时间复杂度是O(n^2)，其中n是序列1和序列2的长度之和。因此，当序列长度很大时，Needleman-Wunsch算法可能会消耗较多的计算资源。

```python
def needleman_wunsch(seq1, seq2):
    m, n = len(seq1), len(seq2)
    matrix = [[-float('inf')] * (n + 1) for _ in range(m + 1)]
    matrix[0][0] = 0

    for i in range(m + 1):
        for j in range(n + 1):
            if i == 0 or j == 0:
                matrix[i][j] = 0
            else:
                match = matrix[i - 1][j - 1] + (1 if seq1[i - 1] == seq2[j - 1] else -1)
                insert = matrix[i - 1][j] + 1
                delete = matrix[i][j - 1] + 1
                matrix[i][j] = max(match, insert, delete)

    return matrix[-1][-1]
```
