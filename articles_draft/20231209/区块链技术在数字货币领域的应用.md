### 4.1代码实例

以下是一个简单的区块链技术代码实例：

```python
import hashlib
import json

class Block:
    def __init__(self, index, previous_hash, timestamp, data, hash):
        self.index = index
        self.previous_hash = previous_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

    def calc_hash(self):
        sha = hashlib.sha256()
        block_string = json.dumps(self.__dict__, sort_keys=True).encode('utf-8')
        sha.update(block_string)
        return sha.hexdigest()

    def __str__(self):
        return 'Block #{}'.format(self.index)

def create_genesis_block(data):
    return Block(0, '0', '2021-01-01', data, '0')

def add_block(block, new_data):
    block.data = new_data
    block.hash = block.calc_hash()
    return block

def create_blockchain():
    genesis_block = create_genesis_block('Genesis Block')
    return [genesis_block]

def add_block_to_blockchain(blockchain, new_block):
    blockchain.append(new_block)
    return blockchain
```

这个代码实例主要包括以下几个部分：

1. `Block`类：这个类用于表示区块链中的一个数据块，包括索引、前一个数据块的哈希、时间戳、数据和哈希等属性。

2. `calc_hash`方法：这个方法用于计算数据块的哈希值，通过将数据块的属性转换为JSON字符串，然后使用SHA-256算法计算哈希值。

3. `create_genesis_block`函数：这个函数用于创建区块链的第一个数据块，称为“Genesis Block”，其索引为0、前一个数据块的哈希为'0'、时间戳为'2021-01-01'和数据为'Genesis Block'。

4. `add_block`函数：这个函数用于添加新的数据块到区块链中，包括更新数据块的数据和计算新的哈希值。

5. `create_blockchain`函数：这个函数用于创建一个初始的区块链，包含一个Genesis Block。

6. `add_block_to_blockchain`函数：这个函数用于将新的数据块添加到区块链中。
