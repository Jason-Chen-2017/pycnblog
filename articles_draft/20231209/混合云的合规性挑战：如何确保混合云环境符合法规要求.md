```python
from Crypto.Random import get_random_bytes

# 生成AES密钥
key = get_random_bytes(16)

# 生成AES加密对象
cipher = AES.new(key, AES.MODE_EAX)

# 加密数据
ciphertext, tag = cipher.encrypt_and_digest(data)

# 解密数据
plaintext = cipher.decrypt_and_verify(ciphertext, tag)
```

## 4.3 审计和监控

我们可以使用日志记录和监控库来实现审计和监控。以下是一个使用Python的日志记录和监控库的代码实例：

```python
import logging
from watchdog.observers import Observer
from watchdog.events import FileSystemEventHandler

# 配置日志记录
logging.basicConfig(level=logging.INFO)

# 配置文件系统监控
class FileSystemEventHandler(FileSystemEventHandler):
    def on_modified(self, event):
        logging.info(f'文件{event.src_path}已修改')

handler = FileSystemEventHandler()
observer = Observer()
observer.schedule(handler, path='/path/to/monitor', recursive=True)
observer.start()

try:
    while True:
        pass
except KeyboardInterrupt:
    observer.stop()

observer.join()
```

在上述代码中，我们首先配置了日志记录，然后配置了文件系统监控。当文件系统中的文件发生修改时，会触发`on_modified`方法，并记录相关日志。
