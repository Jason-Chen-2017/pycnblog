### 4.1 自然语言处理（NLP）

我们可以使用spaCy库来进行词性标注、命名实体识别、语义角色标注、依存关系解析等任务。

```python
import spacy

# 加载spacy模型
nlp = spacy.load("en_core_web_sm")

# 定义文本
text = "John bought a car from Mary."

# 进行词性标注
doc = nlp(text)
for token in doc:
    print(token.text, token.pos_)

# 进行命名实体识别
for entity in doc.ents:
    print(entity.text, entity.label_)

# 进行语义角色标注
for sent in doc.sents:
    for token in sent:
        print(token.text, token.dep_)

# 进行依存关系解析
for token in doc:
    print(token.text, token.head.text)
```

### 4.4 多领域知识

我们可以将多个知识图谱进行融合，以实现多领域知识的应用。

```python
import networkx as nx

# 创建知识图谱
G1 = nx.Graph()
G2 = nx.Graph()

# 添加实体
G1.add_node("John")
G1.add_node("Mary")
G2.add_node("Car")
G2.add_node("House")

# 添加关系
G1.add_edge("John", "Mary", relation="bought")
G2.add_edge("Car", "House", relation="located_in")

# 添加属性
G1.nodes["John"]["age"] = 30
G1.nodes["Mary"]["age"] = 25
G2.nodes["Car"]["price"] = 20000
G2.nodes["House"]["price"] = 50000

# 融合知识图谱
G_fusion = nx.compose(G1, G2)

# 查询实体
for node in G_fusion.nodes():
    print(node)

# 查询关系
for edge in G_fusion.edges():
    print(edge)

# 查询属性
for node in G_fusion.nodes():
    print(node, G_fusion.nodes[node])
```
