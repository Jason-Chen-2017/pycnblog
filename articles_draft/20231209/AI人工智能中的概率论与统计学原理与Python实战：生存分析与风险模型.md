## 3.1 概率论
概率论是一种数学方法，用于描述和分析不确定性和随机性。在概率论中，我们通过概率来描述事件发生的可能性。

### 3.1.1 概率的基本概念
1.事件：事件是一个可能发生或不发生的结果。
2.样本空间：样本空间是所有可能发生的事件的集合。
3.事件的概率：事件的概率是事件发生的可能性，通常用P(E)表示，取值范围在0到1之间。

### 3.1.2 概率的基本定理
概率的基本定理是概率论中的一个重要定理，它可以帮助我们计算多个事件发生的概率。

基本定理：对于任意事件A1、A2、…、An，有P(A1∩A2∩…∩An)=P(A1)+P(A2)+…+P(An)-P(A1∪A2∪…∪An)。

### 3.1.3 条件概率
条件概率是一个事件发生的概率，给定另一个事件已经发生。

条件概率的定义：对于任意两个事件A和B，有P(A|B)=P(A∩B)/P(B)。

## 3.2 统计学
统计学是一种数学方法，用于从数据中抽取信息，以便进行分析和预测。

### 3.2.1 均值、方差和标准差
均值：对于一个数据集，均值是所有数据点的平均值。
方差：方差是一个数据集的离散程度的度量，用于衡量数据点与均值之间的差异。
标准差：标准差是方差的平方根，用于衡量数据点与均值之间的差异的程度。

### 3.2.2 相关性和相关系数
相关性是两个变量之间的关系程度。相关性可以用相关系数来表示，相关系数的取值范围在-1到1之间。

### 3.2.3 线性回归
线性回归是一种预测方法，用于根据一个或多个输入变量来预测一个输出变量。

线性回归的数学模型：y=β0+β1x1+β2x2+…+βnxn+ε，其中y是输出变量，x1、x2、…、xn是输入变量，β0、β1、…、βn是参数，ε是误差项。

## 3.3 生存分析
生存分析是一种统计学方法，用于分析和预测事件发生的时间。

### 3.3.1 生存函数
生存函数是一个事件发生的概率，给定事件发生的时间。生存函数的定义：S(t)=P(T>t)。

### 3.3.2 风险函数
风险函数是一个事件发生的概率，给定事件发生的时间。风险函数的定义：h(t)=P(T=t)。

### 3.3.3 生存分析的方法
1.生存曲线：生存曲线是一个事件发生的概率，给定事件发生的时间。
2.生存分析模型：生存分析模型是一种数学模型，用于描述和预测事件发生的时间。

## 4.1 生存分析
### 4.1.1 生存曲线
```python
import matplotlib.pyplot as plt
import numpy as np

# 生成随机数据
np.random.seed(0)
t = np.linspace(0, 10, 100)
cens = np.random.binomial(1, 0.2, 100)
event = np.random.binomial(1, 0.8, 100)

# 计算生存函数
survival = np.exp(-t)

# 计算生存曲线
survival_curve = np.zeros_like(t)
for i in range(len(t)):
    if cens[i] == 1:
        survival_curve[i] = survival[i]
    else:
        survival_curve[i] = 1 - survival[i]

# 绘制生存曲线
plt.plot(t, survival_curve, 'o-')
plt.xlabel('Time')
plt.ylabel('Survival Probability')
plt.title('Survival Curve')
plt.show()
```
