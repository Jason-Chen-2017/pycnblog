## 4.1 Quasar组件的使用

Quasar框架的Quasar组件的使用方法如下：

```javascript
<template>
  <div>
    <q-btn label="Click me" @click="onClick" />
    <q-input v-model="value" />
    <q-select v-model="selected" :options="options" />
    <q-dialog v-model="dialog" persistent>
      <q-card>
        <q-card-section>
          <q-input v-model="newOption" />
        </q-card-section>
        <q-card-actions align="right">
          <q-btn flat label="Cancel" @click="dialog = false" />
          <q-btn flat label="OK" @click="addOption" />
        </q-card-actions>
      </q-card>
    </q-dialog>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value: '',
      selected: '',
      options: ['Option 1', 'Option 2', 'Option 3'],
      dialog: false,
      newOption: ''
    };
  },
  methods: {
    onClick() {
      alert('You clicked the button!');
    },
    addOption() {
      if (this.newOption) {
        this.options.push(this.newOption);
        this.newOption = '';
        this.dialog = false;
      }
    }
  }
};
</script>
```

在上述代码中，我们使用了Quasar框架提供的按钮、输入框、选择框、弹出框等组件来构建前端应用程序。
