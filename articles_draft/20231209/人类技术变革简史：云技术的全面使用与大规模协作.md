```python
import hashlib

class DistributedFileSystem:
    def __init__(self):
        self.nodes = {}

    def add_node(self, node_id, node_ip):
        self.nodes[node_id] = node_ip

    def store_data(self, data, node_id):
        node_ip = self.nodes.get(node_id)
        if not node_ip:
            raise ValueError("Node not found")
        with open(f"{node_ip}/data.txt", "w") as f:
            f.write(data)

    def get_data(self, node_id):
        node_ip = self.nodes.get(node_id)
        if not node_ip:
            raise ValueError("Node not found")
        with open(f"{node_ip}/data.txt", "r") as f:
            return f.read()
```

我们通过一个简单的分布式计算任务来演示分布式计算的核心算法原理。

```python
from concurrent.futures import ThreadPoolExecutor

def calculate_square(x):
    return x ** 2

def calculate_sum(numbers):
    return sum(numbers)

def distributed_calculate(numbers):
    with ThreadPoolExecutor() as executor:
        tasks = [executor.submit(calculate_square, x) for x in numbers]
        results = [task.result() for task in tasks]
        return calculate_sum(results)
```

在这个代码实例中，我们创建了一个简单的分布式计算任务，它将计算任务分布在多个计算机上。我们使用负载均衡算法将计算任务分配给多个计算机，以实现更高的并行度和性能。
