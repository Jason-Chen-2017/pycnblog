## 4. 具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来说明遗传算法的具体实现过程：

### 4.1 问题描述

假设我们需要找到一个最小化以下目标函数的解：

$$
f(x) = x^2 + 5x + 6
$$

### 4.2 代码实现

我们可以使用Python来实现遗传算法：

```python
import random
import numpy as np

# 定义适应度函数
def fitness_function(x):
    return x**2 + 5*x + 6

# 定义选择函数
def selection(population):
    fitness_values = np.array([fitness_function(x) for x in population])
    probabilities = fitness_values / np.sum(fitness_values)
    selected_indices = np.random.choice(len(population), size=len(population), p=probabilities, replace=False)
    return [population[i] for i in selected_indices]

# 定义交叉函数
def crossover(parent1, parent2):
    crossover_point = random.randint(1, len(parent1) - 1)
    child1 = parent1[:crossover_point] + parent2[crossover_point:]
    child2 = parent2[:crossover_point] + parent1[crossover_point:]
    return child1, child2

# 定义变异函数
def mutation(individual, mutation_rate):
    mutation_points = [random.randint(0, len(individual) - 1) for _ in range(int(len(individual) * mutation_rate))]
    for point in mutation_points:
        individual[point] = random.randint(-10, 10)
    return individual

# 初始化种群
population = [random.randint(-10, 10) for _ in range(100)]

# 设置终止条件
max_iterations = 1000

# 主循环
for _ in range(max_iterations):
    # 选择
    population = selection(population)

    # 交叉
    new_population = []
    for i in range(0, len(population), 2):
        parent1 = population[i]
        parent2 = population[i + 1]
        child1, child2 = crossover(parent1, parent2)
        new_population.append(mutation(child1, 0.1))
        new_population.append(mutation(child2, 0.1))
    population = new_population

    # 找到最佳个体
    best_individual = min(population, key=fitness_function)
    print("Best individual: ", best_individual, " Fitness: ", fitness_function(best_individual))

```

在上面的代码中，我们首先定义了适应度函数、选择函数、交叉函数和变异函数。然后我们初始化了种群，设置了终止条件，并进行遗传算法的主循环。在主循环中，我们首先进行选择操作，然后进行交叉和变异操作，最后找到最佳个体并输出。

通过运行上述代码，我们可以看到遗传算法找到了问题的最优解。
