## 4.1 计步数代码实例

```python
import time

class StepCounter:
    def __init__(self):
        self.step_frequency = 0
        self.time = 0

    def start_counting(self):
        self.time = time.time()

    def stop_counting(self):
        self.time = time.time() - self.time
        steps = self.step_frequency * self.time
        return steps

step_counter = StepCounter()
step_counter.step_frequency = 100
step_counter.start_counting()
steps = step_counter.stop_counting()
print(steps)  # 输出步数
```

在这个代码实例中，我们定义了一个 `StepCounter` 类，用于计算步数。该类有一个 `step_frequency` 属性，表示步伐频率，一个 `time` 属性，表示计时时间。我们通过调用 `start_counting` 方法开始计时，然后通过调用 `stop_counting` 方法结束计时，并计算步数。
