# 2.核心概念与联系

在推荐系统中，我们需要关注以下几个核心概念：

1. 用户（User）：表示用户，是推荐系统的主体，用户会对物品进行评价或行为。
2. 物品（Item）：表示物品，是推荐系统的目标，用户可能对物品进行评价或行为。
3. 评价（Rating）：表示用户对物品的评价，通常是一个数值，用于衡量用户对物品的喜好程度。
4. 行为（Behavior）：表示用户对物品的行为，例如购买、浏览、点赞等，用于衡量用户对物品的兴趣程度。
5. 特征（Feature）：表示物品的特征，例如物品的类别、品牌、价格等，用于描述物品的属性。

这些概念之间存在着一定的联系：用户通过评价和行为与物品进行互动，而物品的特征可以帮助推荐系统理解用户的喜好和兴趣。
