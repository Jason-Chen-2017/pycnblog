### 1. Python

在Python中，我们可以使用以下代码来实现这个任务：

```python
import os
import sys
import io

def count_words(file_path):
    word_count = {}
    with io.open(file_path, 'r', encoding='utf-8') as f:
        for line in f:
            words = line.split()
            for word in words:
                word_count[word] = word_count.get(word, 0) + 1
    return word_count

if __name__ == '__main__':
    file_path = sys.argv[1]
    word_count = count_words(file_path)
    for word, count in word_count.items():
        print(word, count)
```

这个代码首先导入了必要的模块，然后定义了一个名为`count_words`的函数，该函数接受一个文件路径作为参数，并返回一个字典，其中键是单词，值是单词出现的次数。

然后，我们调用这个函数，并将文件路径作为命令行参数传递给程序。最后，我们遍历字典，并将每个单词和它的出现次数打印出来。

### 2. Java

在Java中，我们可以使用以下代码来实现这个任务：

```java
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;
import java.util.HashMap;
import java.util.Map;

public class WordCount {
    public static void main(String[] args) {
        String filePath = args[0];
        Map<String, Integer> wordCount = new HashMap<>();
        try (BufferedReader br = new BufferedReader(new FileReader(filePath))) {
            String line;
            while ((line = br.readLine()) != null) {
                String[] words = line.split("\\s+");
                for (String word : words) {
                    wordCount.put(word, wordCount.get(word) == null ? 0 : wordCount.get(word) + 1);
                }
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
        for (Map.Entry<String, Integer> entry : wordCount.entrySet()) {
            System.out.println(entry.getKey() + " " + entry.getValue());
        }
    }
}
```

这个代码首先导入了必要的包，然后定义了一个名为`WordCount`的类，该类包含一个名为`main`的静态方法，该方法接受一个文件路径作为参数，并返回一个字典，其中键是单词，值是单词出现的次数。

然后，我们调用这个方法，并将文件路径作为命令行参数传递给程序。最后，我们遍历字典，并将每个单词和它的出现次数打印出来。

### 3. C++

```cpp
#include <iostream>
#include <fstream>
#include <sstream>
#include <string>
#include <map>

int main(int argc, char* argv[]) {
    if (argc != 2) {
        std::cerr << "Usage: word_count <file_path>" << std::endl;
        return 1;
    }

    std::ifstream file(argv[1]);
    if (!file.is_open()) {
        std::cerr << "Cannot open file: " << argv[1] << std::endl;
        return 1;
    }

    std::map<std::string, int> word_count;
    std::string line;
    while (std::getline(file, line)) {
        std::istringstream iss(line);
        std::string word;
        while (iss >> word) {
            word_count[word]++;
        }
    }

    for (const auto& entry : word_count) {
        std::cout << entry.first << ": " << entry.second << std::endl;
    }

    return 0;
}
```
