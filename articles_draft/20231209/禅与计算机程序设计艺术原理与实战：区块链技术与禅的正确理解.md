# 2.核心概念与联系

## 2.1区块链技术的核心概念

区块链技术的核心概念包括：

1.区块：区块链是由一系列区块组成的，每个区块包含一组交易数据和一个时间戳。

2.链：区块之间通过一个加密链接关系，使得整个区块链具有一定的安全性和完整性。

3.去中心化：区块链不依赖于任何中心化的服务器或者机构，而是通过分布式网络来存储和传播数据。

4.透明度：区块链的所有交易数据是公开可见的，任何人都可以查看和验证。

5.一致性：区块链通过使用共识算法来确保数据的一致性，即使在分布式网络中，也可以确保数据的准确性和完整性。

## 2.2禅的核心概念

禅是一种哲学思想，强调人们的心灵成长和自我实现。禅的核心概念包括：

1.直指人心：禅强调人们的内心世界，通过直接指引人心来实现心灵的成长。

2.无言传心：禅认为，真正的知识和智慧不是通过言语来传递的，而是通过直接体验和实践来获得的。

3.一切皆空：禅认为，人们的思维和情感都是一种流畅的过程，不需要被束缚在任何固定的观念和概念中。

4.直指人心与无言传心的联系：禅认为，直接指引人心和无言传心是两种不同的方法来实现心灵成长，但它们在本质上是相同的，因为它们都强调直接体验和实践。

import hashlib
import time

class Block:
    def __init__(self, index, previous_hash, timestamp, data, hash):
        self.index = index
        self.previous_hash = previous_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

    def calc_hash(self):
        sha = hashlib.sha256()
        sha.update(str(self.index).encode('utf-8'))
        sha.update(self.previous_hash.encode('utf-8'))
        sha.update(str(self.timestamp).encode('utf-8'))
        sha.update(self.data.encode('utf-8'))
        return sha.hexdigest()

class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(0, "0", time.time(), "Genesis Block", self.calc_hash())

    def add_block(self, data):
        index = len(self.chain)
        previous_hash = self.chain[-1].hash
        timestamp = time.time()
        block = Block(index, previous_hash, timestamp, data, self.calc_hash())
        self.chain.append(block)

    def calc_hash(self, block):
        sha = hashlib.sha256()
        sha.update(str(block.index).encode('utf-8'))
        sha.update(block.previous_hash.encode('utf-8'))
        sha.update(str(block.timestamp).encode('utf-8'))
        sha.update(block.data.encode('utf-8'))
        return sha.hexdigest()

# 创建一个区块链网络的实例
blockchain = Blockchain()

# 将区块添加到区块链网络中

# 创建一个新的区块，包含一组新的交易数据和一个新的时间戳
data = "This is a new sample transaction data"
block = Block(2, blockchain.chain[-1].hash, time.time(), data, blockchain.calc_hash())

# 将新的区块添加到区块链网络中
blockchain.add_block(data)

# 重复步骤4和5，直到创建一个完整的区块链
