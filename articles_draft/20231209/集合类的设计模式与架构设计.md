## 2. 核心概念与联系

集合类的核心概念包括：集合、元素、子集、并集、交集、差集等。这些概念在数学中有着明确的定义，在计算机科学中也有着相应的实现。

### 2.1 集合

集合是一组元素的集合，可以用大括号 {} 表示。集合中的元素可以是任何类型的数据，包括基本类型、引用类型、其他集合类型等。集合类的实现可以是数组、链表、树等数据结构。

### 2.2 元素

集合中的元素是集合的基本组成单位，可以是任何类型的数据。集合类的实现可以通过添加、删除、查找等操作来对元素进行操作。

### 2.3 子集

子集是一个集合中所有元素都在另一个集合中的集合。例如，集合 A = {1, 2, 3}，集合 B = {1, 2}，那么集合 B 是集合 A 的子集。

### 2.4 并集

并集是两个集合的元素集合的联合。例如，集合 A = {1, 2, 3}，集合 B = {2, 3, 4}，那么集合 A 和集合 B 的并集是 {1, 2, 3, 4}。

### 2.5 交集

交集是两个集合的元素集合的交集。例如，集合 A = {1, 2, 3}，集合 B = {2, 3, 4}，那么集合 A 和集合 B 的交集是 {2, 3}。

### 2.6 差集

差集是一个集合中所有元素都不在另一个集合中的集合。例如，集合 A = {1, 2, 3}，集合 B = {2, 3, 4}，那么集合 A 和集合 B 的差集是 {1}。

这些概念在计算机科学中有着广泛的应用，包括但不限于数据库、文本处理、图像处理等。

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
