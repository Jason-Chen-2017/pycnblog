### 3.1 技术分析

技术分析是一种研究股票价格变动的方法，它主要基于历史价格数据。通过分析股票价格的历史趋势，技术分析师可以预测未来的价格变动。

#### 3.1.1 移动平均线

移动平均线（Moving Average，MA）是一种常用的技术分析指标，它用于计算股票价格的平均价格。移动平均线可以帮助投资者识别股票价格的趋势。

移动平均线的计算公式如下：

$$
MA_t = \frac{\sum_{i=1}^{n} P_{t-i}}{n}
$$

其中，$P_{t-i}$ 是股票价格在 $t-i$ 时刻的价格，$n$ 是移动平均线的长度。

#### 3.1.2 关贸指标

关贸指标（Relative Strength Index，RSI）是一种用于衡量股票价格波动强度的技术分析指标。关贸指标的值范围为0到100，当关贸指标值接近100时，表示股票价格波动较大，当关贸指标值接近0时，表示股票价格波动较小。

关贸指标的计算公式如下：

$$
RSI_t = \frac{P_t - P_{t-n}}{P_{t-n}} \times 100
$$

其中，$P_t$ 是股票价格在 $t$ 时刻的价格，$P_{t-n}$ 是股票价格在 $t-n$ 时刻的价格，$n$ 是关贸指标的长度。

### 3.2 数学模型

数学模型可以帮助投资者预测股票价格的变动。以下是一些常用的数学模型：

#### 3.2.1 随机走势模型

随机走势模型（Random Walk）是一种假设股票价格变动是随机的模型。根据随机走势模型，股票价格的变动是由许多小的随机变动组成的。

随机走势模型的数学公式如下：

$$
P_t = P_{t-1} + \epsilon_t
$$

其中，$P_t$ 是股票价格在 $t$ 时刻的价格，$P_{t-1}$ 是股票价格在 $t-1$ 时刻的价格，$\epsilon_t$ 是 $t$ 时刻的随机变动。

#### 3.2.2 均值回归模型

均值回归模型（Mean Reversion）是一种假设股票价格会回归到其长期平均水平的模型。根据均值回归模型，股票价格的变动是由一个长期平均水平和一个短期波动组成的。

均值回归模型的数学公式如下：

$$
P_t = \mu + \sigma \epsilon_t
$$

其中，$P_t$ 是股票价格在 $t$ 时刻的价格，$\mu$ 是股票价格的长期平均水平，$\sigma$ 是股票价格的短期波动，$\epsilon_t$ 是 $t$ 时刻的随机变动。

### 4.1 计算移动平均线

以下是一个计算移动平均线的Python代码实例：

```python
import numpy as np

def moving_average(prices, window_size):
    return np.convolve(prices, np.ones(window_size)/window_size, mode='valid')

prices = np.array([100, 105, 110, 115, 120, 125, 130, 135, 140, 145, 150, 155, 160, 165, 170, 175, 180, 185, 190, 195, 200])
window_size = 3

moving_avg = moving_average(prices, window_size)
print(moving_avg)
```

在上述代码中，我们首先导入了NumPy库，然后定义了一个名为`moving_average`的函数。该函数接受两个参数：`prices`（股票价格数组）和`window_size`（移动平均线的长度）。

在函数内部，我们使用NumPy的`convolve`函数来计算移动平均线。`convolve`函数接受三个参数：`prices`（输入数组）、`kernel`（卷积核，即`np.ones(window_size)/window_size`）和`mode`（卷积模式，我们使用`'valid'`模式）。

def relative_strength_index(prices, window_size):
    delta = prices[:-window_size] - prices[window_size:]
    return np.abs(delta) / delta.clip(min=0)

在函数内部，我们首先计算股票价格的差分`delta`。然后，我们使用`np.abs`函数计算`delta`的绝对值，并使用`clip`函数将负值设置为0。最后，我们将`delta`除以其绝对值，得到关贸指标。

最后，我们使用`relative_strength_index`函数计算关贸指标，并将结果打印出来。
