## 4.1 随机游走的实现
在这个例子中，我们将实现一个简单的随机游走，其中状态为一个二维坐标（x，y），状态转移概率为随机的。

```python
import numpy as np

# 初始状态
x = 0
y = 0

# 状态转移概率
p_x = [0.5, 0.5]
p_y = [0.5, 0.5]

# 随机游走的步数
steps = 1000

# 随机游走的结果
result = []

for _ in range(steps):
    # 随机生成下一状态
    x_next = np.random.choice([-1, 1], p=p_x)
    y_next = np.random.choice([-1, 1], p=p_y)

    # 更新状态
    x += x_next
    y += y_next

    # 记录状态
    result.append((x, y))

# 打印结果
print(result)
```

在这个例子中，我们首先定义了初始状态和状态转移概率。然后，我们使用循环来模拟随机游走的过程，每次更新状态并记录下来。最后，我们打印出随机游走的结果。
