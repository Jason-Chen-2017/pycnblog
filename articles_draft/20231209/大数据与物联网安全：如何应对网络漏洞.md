```python
import hashlib

def hash_password(password):
    return hashlib.sha256(password.encode('utf-8')).hexdigest()

def verify_password(password, hashed_password):
    return hashlib.sha256(password.encode('utf-8')).hexdigest() == hashed_password

username = input('请输入用户名：')
password = input('请输入密码：')

hashed_password = hash_password(password)

if verify_password(password, hashed_password):
    print('认证成功')
else:
    print('认证失败')
```

### 4.2 认证算法详细解释说明

- `hash_password` 函数用于生成密码的哈希值。
- `verify_password` 函数用于验证用户输入的密码是否与存储的哈希值相匹配。
- 用户输入用户名和密码，然后调用 `hash_password` 函数生成密码的哈希值。
- 调用 `verify_password` 函数验证用户输入的密码是否与存储的哈希值相匹配。
- 如果验证成功，则输出认证成功；否则，输出认证失败。
