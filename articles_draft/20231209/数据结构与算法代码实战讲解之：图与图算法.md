# 2.核心概念与联系

## 2.1 图的基本概念

图是一种数据结构，由一组节点（vertex）和一组边（edge）组成。节点表示问题的实体，边表示实体之间的关系。图可以用来表示各种各样的关系，例如社交关系、交通网络、电子电路等。图的基本概念包括：

- 节点（vertex）：图的基本元素，表示问题的实体。
- 边（edge）：节点之间的关系，表示问题的关系。
- 图（graph）：由节点和边组成的数据结构。

## 2.2 图的基本操作

图的基本操作包括：

- 添加节点：在图中添加一个新的节点。
- 删除节点：从图中删除一个节点。
- 添加边：在图中添加一个新的边。
- 删除边：从图中删除一个边。
- 查找节点：在图中查找一个节点。
- 查找边：在图中查找一个边。
- 遍历图：对图进行遍历，访问所有的节点和边。
- 搜索图：在图中搜索某个节点或某个路径。
- 排序图：对图进行排序，按照某种规则对节点和边进行排序。

## 2.3 图的基本属性

图的基本属性包括：

- 节点数量：图中节点的数量。
- 边数量：图中边的数量。
- 连通性：图中节点之间是否存在连通性。
- 权重：边上的权重，用于表示边之间的关系。
- 图的类型：图可以分为有向图和无向图，有权图和无权图等。

```python
# 深度优先搜索
def dfs_preorder(graph, root):
    visited = set()
    stack = [root]

    while stack:
        node = stack.pop()
        if node not in visited:
            visited.add(node)
            stack.extend(graph[node] - visited)

    return visited
```

## 4.3 图的搜索方法

我们可以使用`networkx`库来实现图的搜索方法。例如，我们可以使用`dijkstra_path`函数来实现最短路径算法。

```python
# 最短路径算法
def dijkstra_path(graph, start, end):
    visited = set()
    distances = {start: 0}
    previous = {}

    nodes = list(graph.nodes)

    while nodes:
        min_node = min(nodes, key=lambda x: distances.get(x, float('inf')))
        visited.add(min_node)
        nodes.remove(min_node)

        for neighbor in graph[min_node]:
            distance = distances[min_node] + graph[min_node][neighbor]
            if neighbor not in distances or distance < distances[neighbor]:
                distances[neighbor] = distance
                previous[neighbor] = min_node

    path = [end]
    while path[-1] != start:
        path.append(previous[path[-1]])

    return path[::-1]
```

def topological_sort(graph):
    visited = set()
    stack = []

    for node in graph.nodes:
        if node not in visited:
            dfs_topological_sort(graph, node, visited, stack)

def dfs_topological_sort(graph, node, visited, stack):
    visited.add(node)
    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs_topological_sort(graph, neighbor, visited, stack)
