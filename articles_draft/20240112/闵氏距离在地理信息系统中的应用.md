## 1.2 闵氏距离的基本概念

闵氏距离（Minkowski distance）是一种用于计算两个多维向量之间距离的数学方法。它是欧氏距离的推广，可以用于计算不同维度空间中的距离。闵氏距离的定义公式为：

$$
d(x, y) = \left(\sum_{i=1}^{n} |x_i - y_i|^p\right)^{\frac{1}{p}}
$$

其中，$x$ 和 $y$ 是多维向量，$n$ 是向量的维度，$p$ 是闵氏距离的参数。当 $p=2$ 时，闵氏距离就是欧氏距离。

在地理信息系统中，闵氏距离可以用于计算地理空间对象之间的距离，从而实现地理空间数据的查询和分析。

以下是闵氏距离的Python实现代码：

```python
import math

def minkowski_distance(x, y, p):
    distance = 0
    for i in range(len(x)):
        distance += abs(x[i] - y[i]) ** p
    return distance ** (1 / p)

# 示例数据

y = [4, 5, 6]
p = 2

# 计算闵氏距离
```
