## 4.1 滤波

我们可以使用以下代码来实现滤波：

```python
import numpy as np
import cv2

# 读取图像

# 将图像转换为频域
fourier_image = cv2.dft(np.float32(image), flags=cv2.DFT_COMPLEX_OUTPUT)

# 计算滤波器的权重矩阵
filter_weights = np.fft.ifft2(fourier_image[0, 0, :, :])

# 使用矩阵逆来计算滤波器的权重
filter_weights = np.linalg.inv(filter_weights)

# 将滤波器的权重应用于原始图像，从而实现滤波的效果
filtered_image = cv2.idft(np.fft.fft2(image * filter_weights), flags=cv2.DFT_COMPLEX_OUTPUT)

# 显示滤波后的图像
cv2.imshow('Filtered Image', filtered_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
