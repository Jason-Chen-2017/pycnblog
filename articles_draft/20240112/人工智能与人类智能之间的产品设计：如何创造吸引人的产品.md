# 4.具体代码实例和详细解释说明

## 4.1 任务：文本摘要

任务：从一篇长文章中提取关键信息，生成一个简短的文本摘要。

## 4.2 代码实例

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

def text_summary(text, max_words=50):
    # 使用TF-IDF向量化对文本进行向量化
    vectorizer = TfidfVectorizer()
    text_matrix = vectorizer.fit_transform([text])
    word_vector = text_matrix[0].toarray()[0]

    # 计算文本的相似度
    similarity_matrix = cosine_similarity(text_matrix, text_matrix)
    max_similarity = np.max(similarity_matrix[0][0])
    similarity_indices = np.where(similarity_matrix[0][0] == max_similarity)[0][0]

    # 从最相似的文本中提取关键词
    similar_text = text_matrix.get_row(similarity_indices)
    similar_words = vectorizer.get_feature_names_out()
    similar_word_indices = similar_text.argsort()[::-1]

    # 生成文本摘要
    summary_words = [similar_words[index] for index in similar_word_indices[:max_words]]
    summary = ' '.join(summary_words)

    return summary

text = """人工智能与人类智能之间的产品设计是指将人工智能技术与人类智能相结合，以创造具有吸引力和实用性的产品。这种产品设计可以帮助提高工作效率、提高生活质量、降低成本、提高安全性、提高准确性等。"""

print(text_summary(text))
```

## 4.3 详细解释说明

在这个例子中，我们使用了TF-IDF向量化和余弦相似度来实现文本摘要的功能。首先，我们使用TF-IDF向量化对文本进行向量化，以便计算文本之间的相似度。然后，我们计算文本的相似度，并找到最相似的文本。最后，我们从最相似的文本中提取关键词，并生成文本摘要。
