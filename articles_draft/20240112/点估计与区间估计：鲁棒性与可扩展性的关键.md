## 3.1 点估计

点估计是指对一个特定值的估计。在数据处理和分析中，点估计通常用于对某个特定参数进行估计，例如均值、中位数等。

### 3.1.1 均值估计

均值估计是指对一个数据集中所有数值的平均值进行估计。假设我们有一个数据集$D = \{x_1, x_2, \dots, x_n\}$，其中$x_i$表示第$i$个数据点。那么数据集$D$的均值$E[x]$可以通过以下公式计算：

$$
E[x] = \frac{1}{n} \sum_{i=1}^{n} x_i
$$

### 3.1.2 中位数估计

中位数估计是指对一个数据集中中间值进行估计。假设我们有一个数据集$D = \{x_1, x_2, \dots, x_n\}$，其中$x_i$表示第$i$个数据点。那么数据集$D$的中位数$M$可以通过以下公式计算：

$$
M = \left\{
\begin{aligned}
& \frac{x_{(n+1)/2} + x_{n/2}}{2}, & \text{if n is odd} \\
& \frac{x_{n/2} + x_{(n/2+1)}}{2}, & \text{if n is even}
\end{aligned}
\right.
$$

## 3.2 区间估计

区间估计是指对一个范围内的值进行估计。在数据处理和分析中，区间估计通常用于对某个参数的上下限进行估计，例如置信区间、信息下界等。

### 3.2.1 置信区间

置信区间是指在某个置信水平下，数据集中参数的可能值范围。假设我们有一个数据集$D = \{x_1, x_2, \dots, x_n\}$，其中$x_i$表示第$i$个数据点，并假设$x_i$是正态分布的。那么，对于一个参数$\theta$，我们可以通过以下公式计算置信区间：

$$
\theta \in \left[ \hat{\theta} - Z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}, \hat{\theta} + Z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}} \right]
$$

其中，$\hat{\theta}$是参数的估计值，$Z_{\alpha/2}$是正态分布的$\alpha/2$分位数，$\sigma$是参数的标准差，$n$是数据集的大小。

### 3.2.2 信息下界

信息下界是指在某个信息量下界下，数据集中参数的可能值范围。假设我们有一个数据集$D = \{x_1, x_2, \dots, x_n\}$，其中$x_i$表示第$i$个数据点，并假设$x_i$是正态分布的。那么，对于一个参数$\theta$，我们可以通过以下公式计算信息下界：

$$
I(\theta) \geq \frac{1}{2} \log \left( \frac{n}{\sigma^2} \right)
$$

其中，$I(\theta)$是参数$\theta$的信息量，$n$是数据集的大小，$\sigma$是参数的标准差。

## 4.2 中位数估计

```python
import numpy as np

def median_estimate(data):
    n = len(data)
    m = n // 2
    if n % 2 == 0:
        return (data[m - 1] + data[m]) / 2
    else:
        return data[m]

data = np.array([1, 2, 3, 4, 5])
print(median_estimate(data))
```

## 4.3 置信区间

```python
import numpy as np
from scipy.stats import norm

def confidence_interval(data, alpha=0.05):
    n = len(data)
    mean = np.mean(data)
    variance = np.var(data)
    z_score = norm.ppf(1 - alpha / 2)
    return mean - z_score * (variance / np.sqrt(n)), mean + z_score * (variance / np.sqrt(n))

data = np.array([1, 2, 3, 4, 5])
print(confidence_interval(data))
```

## 4.4 信息下界

```python
import numpy as np

def information_lower_bound(data, alpha=0.05):
    n = len(data)
    mean = np.mean(data)
    variance = np.var(data)
    z_score = norm.ppf(1 - alpha / 2)
    return 0.5 * np.log(n / variance)

data = np.array([1, 2, 3, 4, 5])
```
