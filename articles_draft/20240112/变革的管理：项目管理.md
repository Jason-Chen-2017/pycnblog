# 2.核心概念与联系

项目管理的核心概念包括：

## 4.1 项目管理的代码实例

以下是一个简单的项目管理代码实例：

```python
class Project:
    def __init__(self, name, start_date, end_date, budget):
        self.name = name
        self.start_date = start_date
        self.end_date = end_date
        self.budget = budget

    def calculate_cost(self):
        return self.budget

    def calculate_duration(self):
        return self.end_date - self.start_date

    def display_info(self):
        print(f"项目名称：{self.name}")
        print(f"开始日期：{self.start_date}")
        print(f"结束日期：{self.end_date}")
        print(f"预算：{self.budget}")

project = Project("建设新办公室", "2021-01-01", "2021-12-31", 100000)
project.display_info()
```

## 4.2 变革的管理的代码实例

```python
class ChangeManagement:
    def __init__(self, name, start_date, end_date, goals):
        self.name = name
        self.start_date = start_date
        self.end_date = end_date
        self.goals = goals

    def calculate_progress(self):
        return (self.end_date - self.start_date) / self.end_date

    def display_info(self):
        print(f"变革名称：{self.name}")
        print(f"开始日期：{self.start_date}")
        print(f"结束日期：{self.end_date}")
        print(f"目标：{self.goals}")
```
