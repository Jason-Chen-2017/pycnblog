泊松分布是一种概率分布，用于描述连续随机变量取值的概率密度函数。它在数学统计学中具有广泛的应用，特别是在处理计数数据时，如人口普查、电子邮件发送次数、网页访问次数等。泊松分布是一种离散分布，其概率密度函数为：

$$
P(X=k) = \frac{e^{-\lambda}\lambda^k}{k!}
$$

其中，$k$ 是非负整数，$e$ 是基数，$\lambda$ 是分布参数。

在实际应用中，我们经常需要估计泊松分布的参数 $\lambda$。本文将介绍一些常见的泊松分布参数估计方法，包括最大似然估计、方差稳定估计、贝叶斯估计等。

## 1.1 泊松分布的性质

泊松分布具有以下性质：

1. 非负整数值：$X$ 只能取值为 $0, 1, 2, \dots$。
2. 期望：$E(X) = \lambda$。
3. 方差：$Var(X) = \lambda$。
4. 连续性：泊松分布是连续的，但其概率密度函数在整数值处具有锐峰。

## 4.1 最大似然估计（MLE）

```python
import numpy as np

def mle(data):
    n = len(data)
    sum_data = np.sum(data)
    lambda_hat = sum_data / n
    return lambda_hat

data = np.random.poisson(10, 100)
lambda_hat = mle(data)
print("MLE:", lambda_hat)
```

## 4.2 方差稳定估计（VST）
