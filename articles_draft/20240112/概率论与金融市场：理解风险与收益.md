# 2.核心概念与联系

在金融市场中，概率论的核心概念包括：随机变量、概率分布、期望、方差、标准差等。这些概念在金融市场中具有重要的意义。

## 2.1 随机变量

随机变量是一个可以取任意值的变量，其取值的概率可以通过概率函数来描述。在金融市场中，随机变量常常用来描述股票价格、利率、通货膨胀率等等。例如，股票价格可以看作是一个随机变量，其取值可能是正数、负数或零。

## 2.2 概率分布

概率分布是用来描述随机变量取值概率的一种数学模型。在金融市场中，常见的概率分布有泊松分布、正态分布、指数分布等。例如，股票价格的波动可以用正态分布来描述，利率的变化可以用指数分布来描述。

## 2.3 期望

期望是随机变量取值平均值的概念。在金融市场中，期望可以用来衡量投资组合的预期收益。例如，如果一个投资组合的股票价格预期平均值是100元，那么该投资组合的预期收益就是100元。

## 2.4 方差与标准差

方差是随机变量取值离平均值的平方和的概率。在金融市场中，方差可以用来衡量投资组合的风险。例如，如果一个投资组合的股票价格方差是100，那么该投资组合的风险就是100。标准差是方差的平方根，可以用来衡量随机变量的离散程度。

$$

$$

$$

$$

$$

$$

$$

```python
import numpy as np

def N(x):
    return (1 / np.sqrt(2 * np.pi)) * np.exp(-0.5 * x**2)

def call_price(S_0, X, r, T, d_1, d_2):
    return S_0 * N(d_1) - X * np.exp(-r * T) * N(d_2)

S_0 = 100
X = 100
r = 0.05
T = 1

d_1 = (S_0 - X) / np.sqrt(T)
d_2 = d_1 - np.sqrt(T)

call_price(S_0, X, r, T, d_1, d_2)
```
