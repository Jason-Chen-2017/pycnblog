$$

$$
K(x, x') = \exp(-\gamma \|x - x'\|^2)

其中，x和x'是输入数据，γ是径向基函数核的参数。

## 4.1 多项式核实例

### 4.1.1 代码实例

```python
import numpy as np

def polynomial_kernel(x, x_prime, degree=3):
    inner_product = np.dot(x, x_prime)
    return (1 + inner_product)**degree

X = np.array([[1, 2], [2, 3], [3, 4]])
K = np.zeros((len(X), len(X)))

for i in range(len(X)):
    for j in range(len(X)):
        K[i, j] = polynomial_kernel(X[i], X[j], degree=2)

print(K)
```

在这个代码实例中，我们首先定义了多项式核函数`polynomial_kernel`，它接受输入数据x、x_prime以及多项式核的阶数degree作为参数。然后，我们创建了一个输入数据集X，并初始化了一个高阶非线性核矩阵K。接下来，我们使用双重循环计算每对输入数据之间的高阶非线性核值，并将其存储到K矩阵中。最后，我们打印了K矩阵。
