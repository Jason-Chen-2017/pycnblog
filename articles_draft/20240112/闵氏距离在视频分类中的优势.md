# 2.核心概念与联系

## 2.1 闵氏距离的定义

闵氏距离（Mahalanobis Distance）是一种衡量两个样本之间距离的方法，它可以考虑特征之间的相关性，从而更准确地衡量样本之间的距离。闵氏距离的定义公式为：

$$
D^2 = (x - \mu) \Sigma^{-1} (x - \mu)^T
$$

其中，$x$ 是一个样本，$\mu$ 是样本的均值，$\Sigma$ 是样本的协方差矩阵，$D^2$ 是闵氏距离的平方值。

## 2.2 闵氏距离在视频分类中的应用

闵氏距离在视频分类中的应用主要有以下几个方面：

- 视频自动分类：通过计算视频之间的闵氏距离，可以实现视频的自动分类。
- 视频推荐：通过计算用户观看的视频与其他视频之间的闵氏距离，可以实现视频的推荐。
- 视频聚类：通过计算视频之间的闵氏距离，可以实现视频的聚类。

import numpy as np

# 假设有两个视频的特征向量

x2 = np.array([4, 5, 6])

# 假设有两个视频的均值向量

# 假设有两个视频的协方差矩阵

Sigma2 = np.array([[1, 0, 0], [0, 1, 0], [0, 0, 1]])

# 计算闵氏距离
D2 = (x1 - mu1) @ np.linalg.inv(Sigma1) @ (x1 - mu1).T + (x2 - mu2) @ np.linalg.inv(Sigma2) @ (x2 - mu2).T
D = np.sqrt(D2)

print("闵氏距离：", D)
