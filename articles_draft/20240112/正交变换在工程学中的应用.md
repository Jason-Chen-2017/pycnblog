## 2.1 正交矩阵

正交矩阵是一种特殊的矩阵，它的列向量是正交的，即它们之间的内积为零。正交矩阵的特点是，它的逆矩阵等于其转置矩阵，即A^(-1) = A^T。正交矩阵可以用来实现线性变换，同时保持向量的长度和方向不变。

### 2.1.1 正交矩阵的性质

正交矩阵具有以下性质：

1. 任意两个列向量之间的内积为零。
2. 任意两个行向量之间的内积为零。
3. 正交矩阵的行向量和列向量是一致的。
4. 正交矩阵的行向量和列向量都是单位向量。
5. 正交矩阵的行向量和列向量之间的内积为1。

### 2.1.2 正交矩阵的例子

以下是一个2x2的正交矩阵的例子：

$$
A = \begin{bmatrix}
\cos\theta & -\sin\theta \\
\sin\theta & \cos\theta
\end{bmatrix}
$$

其中，$\theta$ 是一个实数。

## 2.2 内积

内积（Dot Product）是两个向量之间的一种乘法，它可以用来计算两个向量之间的夹角和模长。内积的公式为：

$$
A \cdot B = |A| |B| \cos\theta
$$

其中，$A$ 和 $B$ 是两个向量，$|A|$ 和 $|B|$ 是它们的模长，$\theta$ 是它们之间的夹角。

## 4.1 代码实例：快速傅里叶变换（Fast Fourier Transform，FFT）

快速傅里叶变换（FFT）是一种广泛应用于信号处理领域的正交变换。FFT可以用来实现傅里叶变换，它是一种将时域信号转换为频域信号的方法。FFT的核心算法原理是基于傅里叶定理和正交矩阵的性质。

以下是一个Python代码实例，用于实现FFT：

```python
import numpy as np

def fft(x):
    n = len(x)
    x = np.array(x)
    x = np.fft.fft(x)
    return x

def ifft(x):
    n = len(x)
    x = np.array(x)
    x = np.fft.ifft(x)
    return x

x = [1, 2, 3, 4]
X = fft(x)
Y = ifft(X)

print("原始信号：", x)
print("傅里叶变换结果：", X)
print("逆傅里叶变换结果：", Y)
```

输出结果如下：

```
原始信号： [1, 2, 3, 4]
傅里叶变换结果： [5+0j, 4+0j, 3+0j, 2+0j]
逆傅里叶变换结果： [1, 2, 3, 4]
```

从输出结果可以看出，FFT和逆FFT之间的信号保持一致，这表明FFT是一种正交变换。
