3. 使用协同过滤推荐算法，根据用户的历史行为数据，为用户推荐相似用户喜欢的汽车。
4. 使用内容过滤推荐算法，根据汽车的特征信息，为用户推荐与自己喜好相似的汽车。
5. 将协同过滤推荐和内容过滤推荐结合，为用户提供更准确的推荐。

1. 协同过滤推荐：

$$
similarity(u,v) = \frac{\sum_{i \in I} (r_{ui} - \bar{r_u})(r_{vi} - \bar{r_v})}{\sqrt{\sum_{i \in I} (r_{ui} - \bar{r_u})^2} \sqrt{\sum_{i \in I} (r_{vi} - \bar{r_v})^2}}
$$

$$
prediction(u,i) = \bar{r_u} + sim(u,i) \times \sigma(u,i)
$$

其中，$similarity(u,v)$ 表示用户 $u$ 和 $v$ 之间的相似度，$prediction(u,i)$ 表示用户 $u$ 对物品 $i$ 的预测评分。

$$

$$

$$

$$

其中，$similarity(i,j)$ 表示物品 $i$ 和 $j$ 之间的相似度，$prediction(u,i)$ 表示用户 $u$ 对物品 $i$ 的预测评分。

3. 混合推荐：

$$
final\_prediction(u,i) = \alpha \times prediction_{coll}(u,i) + (1-\alpha) \times prediction_{con}(u,i)
$$

其中，$final\_prediction(u,i)$ 表示用户 $u$ 对物品 $i$ 的最终预测评分，$\alpha$ 表示协同过滤推荐的权重。

import numpy as np

# 用户行为数据
user_data = {
    'user1': {'car1': 5, 'car2': 3, 'car3': 4},
    'user2': {'car1': 4, 'car2': 2, 'car3': 3},
    'user3': {'car1': 3, 'car2': 5, 'car3': 2},
}

# 计算相似度
def cosine_similarity(u, v):
    # 计算欧氏距离
    dot_product = np.dot(u, v)
    norm_u = np.linalg.norm(u)
    norm_v = np.linalg.norm(v)
    return dot_product / (norm_u * norm_v)

# 计算预测评分
def collaborative_filtering(user, item, similarity_matrix):
    # 计算与当前用户相似的用户
    similar_users = np.where(similarity_matrix[user] > 0)[0]
    # 计算预测评分
    prediction = np.dot(similarity_matrix[user], similarity_matrix[item]) / np.linalg.norm(similarity_matrix[user])
    return prediction

# 构建相似度矩阵
similarity_matrix = np.zeros((len(user_data), len(user_data)))
for user, data in user_data.items():
    for item, rating in data.items():
        for other_user, other_data in user_data.items():
            if other_user != user and item in other_data:
                similarity_matrix[user][other_user] = cosine_similarity(data[item], other_data[item])

# 推荐汽车
def recommend_car(user, similarity_matrix):
    # 获取当前用户的评分
    user_ratings = list(user_data[user].values())
    # 获取与当前用户相似的用户
    similar_users = np.where(similarity_matrix[user] > 0)[0]
    # 获取与当前用户相似的用户的评分
    similar_ratings = [np.mean([user_data[u][item] for u in similar_users if item in user_data[u]]) for item in user_data[user]]
    # 计算预测评分
    prediction = np.dot(similarity_matrix[user], similar_ratings) / np.linalg.norm(similar_ratings)
    # 推荐评分最高的汽车
    recommended_car = np.argmax(prediction)
    return recommended_car

# 推荐结果
recommended_car = recommend_car('user1', similarity_matrix)
print(f'推荐给 user1 的汽车是：{recommended_car}')
