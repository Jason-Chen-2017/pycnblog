## 3.1 概率分布函数
概率分布函数（PDF）是一种用于描述随机变量可能取值的概率的函数。它可以用来描述随机变量的概率分布。概率分布函数的定义如下：

$$
F(x) = P(X \leq x)
$$

其中，$F(x)$ 是概率分布函数，$X$ 是随机变量，$x$ 是随机变量的可能取值。

## 3.2 累积分布函数
累积分布函数（CDF）是一种用于描述随机变量可能取值的概率的函数。它可以用来描述随机变量的概率分布。累积分布函数的定义如下：

$$
F(x) = P(X \leq x)
$$

其中，$F(x)$ 是累积分布函数，$X$ 是随机变量，$x$ 是随机变量的可能取值。

## 3.3 期望值
期望值是一种用于描述随机变量的数学指标。它可以用来描述随机变量的平均值。期望值的定义如下：

$$
E(X) = \int_{-\infty}^{\infty} x f(x) dx
$$

其中，$E(X)$ 是期望值，$X$ 是随机变量，$f(x)$ 是概率分布函数。

## 3.4 方差
方差是一种用于描述随机变量的数学指标。它可以用来描述随机变量的离散程度。方差的定义如下：

$$
Var(X) = E[(X - E(X))^2]
$$

其中，$Var(X)$ 是方差，$X$ 是随机变量，$E(X)$ 是期望值。

## 3.5 相关性
相关性是一种用于描述两个随机变量之间关系的数学指标。它可以用来描述两个随机变量之间的关系。相关性的定义如下：

$$
Corr(X, Y) = \frac{Cov(X, Y)}{\sqrt{Var(X) Var(Y)}}
$$

其中，$Corr(X, Y)$ 是相关性，$Cov(X, Y)$ 是协方差，$Var(X)$ 是方差，$Var(Y)$ 是方差。

## 4.1 使用Python实现概率分布函数
我们可以使用Python的`scipy`库来实现概率分布函数。以下是一个使用`scipy`库实现概率分布函数的例子：

```python
import numpy as np
from scipy.stats import norm

def pdf(x, mu, sigma):
    return norm.pdf(x, mu, sigma)

x = np.linspace(-4, 4, 100)
y = pdf(x, 0, 1)

import matplotlib.pyplot as plt

plt.plot(x, y)
plt.show()
```

在这个例子中，我们使用`scipy`库的`norm`函数来实现概率分布函数。`norm`函数可以用来描述正态分布的概率分布函数。我们可以通过调整`mu`和`sigma`参数来实现不同的正态分布。
