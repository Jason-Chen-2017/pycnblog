# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个具体的代码实例来说明贝塔分布在多项式模型中的应用。

```python
import numpy as np
from scipy.stats import beta

# 生成一组随机数据
np.random.seed(42)
n = 1000
x = np.random.beta(1, 1, n)

# 估计贝塔分布的参数
alpha_hat = np.mean(x)
beta_hat = 1 - alpha_hat

# 绘制贝塔分布
x_values = np.linspace(0, 1, 100)
y_values = beta.pdf(x_values, alpha_hat, beta_hat)

import matplotlib.pyplot as plt
plt.plot(x_values, y_values)
plt.title('Beta Distribution')
plt.xlabel('x')
plt.ylabel('Probability Density')
plt.show()
```

在这个代码实例中，我们首先生成了一组随机数据，并使用贝塔分布的参数来估计贝塔分布的参数。然后，我们使用Matplotlib库来绘制贝塔分布的概率密度函数。
