## 1.3 高斯核的基本概念
高斯核是一种常用的计算方法，它可以用于计算两个数据点之间的相似度。高斯核的定义如下：

$$
K(x, y) = \exp\left(-\frac{1}{2}\left(\frac{||x - y||^2}{\sigma^2}\right)\right)
$$

其中，$x$ 和 $y$ 是两个数据点，$||x - y||$ 是两个数据点之间的欧氏距离，$\sigma$ 是高斯核的标准差。

高斯核的特点如下：

- **非负：**高斯核的值始终非负，表示数据点之间的相似度。
- **对称：**高斯核是对称的，即 $K(x, y) = K(y, x)$。
- **可微：**高斯核是可微的，可以用于计算梯度和偏导数。

以下是高斯核在Python中的一个简单实现：

```python
import numpy as np

def gaussian_kernel(x, y, sigma=1.0):
    d = np.linalg.norm(x - y)
    return np.exp(-0.5 * (d ** 2) / (sigma ** 2))

# 示例数据
x = np.array([1, 2, 3])
y = np.array([4, 5, 6])
sigma = 1.0

# 计算高斯核值
K = gaussian_kernel(x, y, sigma)
print(K)
```
