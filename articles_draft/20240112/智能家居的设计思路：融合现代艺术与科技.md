# 2. 核心概念与联系

在智能家居中，核心概念包括：

- 智能家居系统：智能家居系统是整个智能家居的核心组成部分，它负责收集、处理和控制家居设备的各种信息和功能。
- 传感器：传感器是智能家居系统的重要组成部分，它可以检测到各种物理、化学和生物信息，如温度、湿度、光照、空气质量等。
- 控制器：控制器是智能家居系统的另一个重要组成部分，它负责根据传感器收集到的信息，对家居设备进行控制和管理。
- 通信设备：通信设备是智能家居系统的一部分，它负责实现家居设备之间的数据传输和通信。

这些概念之间的联系如下：

- 传感器与控制器之间的联系：传感器收集到的信息会被传递给控制器，控制器根据这些信息对家居设备进行控制和管理。
- 控制器与通信设备之间的联系：控制器需要通过通信设备与家居设备进行数据传输和通信，以实现智能化管理和自动化控制。
- 传感器、控制器和通信设备之间的联系：这三个组成部分共同构成了智能家居系统，实现了家居设备的智能化管理和自动化控制。

```python
import time

class Sensor:
    def __init__(self, name, value):
        self.name = name
        self.value = value

class Controller:
    def __init__(self, sensors):
        self.sensors = sensors

    def control(self):
        for sensor in self.sensors:
            if sensor.value > threshold:
                device.control(sensor.name)

class Device:
    def __init__(self, name, value):
        self.name = name
        self.value = value

    def control(self, sensor_name):
        if sensor_name == "temperature":
            self.value = temperature_threshold
        elif sensor_name == "humidity":
            self.value = humidity_threshold
        elif sensor_name == "light":
            self.value = light_threshold
        elif sensor_name == "air_quality":
            self.value = air_quality_threshold

if __name__ == "__main__":
    sensors = [
        Sensor("temperature", 30),
        Sensor("humidity", 60),
        Sensor("light", 1000),
        Sensor("air_quality", 100),
    ]

    controller = Controller(sensors)
    device = Device("heater", 25)

    threshold = 35
    temperature_threshold = 28
    humidity_threshold = 65
    light_threshold = 800
    air_quality_threshold = 105

    while True:
        controller.control()
        time.sleep(60)
```
