## 4.1 人才需求预测
在数字化人才管理中，我们可以使用机器学习算法来预测人才需求。以下是一个简单的人才需求预测代码实例：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 人才需求数据
X = np.array([[2010], [2011], [2012], [2013], [2014], [2015], [2016], [2017], [2018], [2019]])
y = np.array([1000, 1200, 1400, 1600, 1800, 2000, 2200, 2400, 2600, 2800])

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 预测人才需求
year = np.array([2020]).reshape(-1, 1)
predicted_demand = model.predict(year)
print(predicted_demand)
```

## 4.2 人才能力评估
在数字化人才管理中，我们可以使用人工智能算法来评估人才能力。以下是一个简单的人才能力评估代码实例：

```python
import numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.neural_network import MLPClassifier

# 人才能力数据
X = np.array([[70, 80, 90], [60, 70, 80], [50, 60, 70], [40, 50, 60]])
y = np.array([0, 1, 2, 3])

# 数据标准化
scaler = StandardScaler()
X = scaler.fit_transform(X)

# 创建多层感知机模型
model = MLPClassifier(hidden_layer_sizes=(10,), max_iter=1000, alpha=0.0001, solver='sgd', random_state=1)

# 训练模型
model.fit(X, y)

# 评估人才能力
new_candidate = np.array([[65, 75, 85]]).reshape(-1, 1)
new_candidate = scaler.transform(new_candidate)
predicted_ability = model.predict(new_candidate)
print(predicted_ability)
```
