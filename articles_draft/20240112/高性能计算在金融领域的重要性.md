## 4.1 代码实例

```python
import numpy as np
from scipy.linalg import svd

# 生成随机数据
data = np.random.rand(1000, 1000)

# 使用奇异值分解（SVD）进行降维
U, s, Vt = svd(data)

# 选择前k个特征
k = 50
Uk = U[:, :k]
Sk = np.diag(s[:k])
Vtk = Vt[:k, :]

# 降维后的数据
reduced_data = np.dot(Uk, np.dot(Sk, Vtk))
```

## 4.2 详细解释说明

上述代码实例中，我们使用了奇异值分解（SVD）进行降维。奇异值分解是一种常用的降维方法，可以用来减少数据的维度，从而提高计算效率。
