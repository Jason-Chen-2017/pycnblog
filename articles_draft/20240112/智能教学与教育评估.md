# 4. 具体代码实例和详细解释说明

## 4.1 智能教学代码实例

以下是一个基于Python的智能教学代码实例：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据收集
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([1, 2, 3, 4])

# 数据预处理
X = (X - np.mean(X, axis=0)) / np.std(X, axis=0)

# 算法训练
model = LinearRegression()
model.fit(X, y)

# 结果输出
print(model.predict([[5, 6]]))
```
