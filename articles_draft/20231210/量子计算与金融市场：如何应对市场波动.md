# 4 具体代码实例和详细解释说明
## 4.1 量子幂算法实现
```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile, assemble

def quantum_power(x, n):
    qc = QuantumCircuit(1, 1)
    qc.initialize([n], 0)
    qc.h(0)
    qc.x(0) ** n
    qc.measure(0, 0)
    return qc

x = 3
n = 2
qc = quantum_power(x, n)
qasm_sim = Aer.get_backend('qasm_simulator')
job = qasm_sim.run(qc)
result = job.result()
counts = result.get_counts()
print(counts)
```
