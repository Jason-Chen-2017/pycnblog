### 1.4.1 代码实例

以下是一个量子自旋震荡实验的代码实例：

```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile
from qiskit.visualization import plot_histogram

# 创建一个量子电路
qc = QuantumCircuit(2)

# 初始化两个量子比特
qc.h(0)
qc.h(1)

# 将两个量子比特的状态保存到一个列表中
state_vector = qc.draw(output='statevector')

# 将量子电路编译为可执行的量子电路
executable = transpile(qc, backend=Aer.get_backend('qasm_simulator'))

# 执行量子电路
result = executable.run()

# 获取量子电路的结果
counts = result.get_counts()

# 绘制结果
plot_histogram(counts)
```
