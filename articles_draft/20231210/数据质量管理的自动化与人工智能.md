## 4. 数据质量管理的代码实例与解释

以下是一个数据质量管理的Python代码实例：

```python
import pandas as pd

# 数据清洗与预处理
# 去除重复数据
df = df.drop_duplicates()

# 填充缺失值
df = df.fillna(df.mean())

# 数据类型转换
df['age'] = df['age'].astype('int')

# 数据验证与校验
# 数据一致性检查
df = df[df['age'].isin(df['age'].unique())]

# 数据格式检查
df = df[df['age'].apply(lambda x: isinstance(x, int))]

# 数据分析与报告
# 数据分析
mean_age = df['age'].mean()

# 报告生成
report = f"数据的平均年龄为：{mean_age}"
print(report)

# 数据质量监控与预警
# 数据质量监控
df = df[df['age'] >= 0]

# 预警
if df['age'].min() < 0:
    print("数据中存在负数年龄，请检查数据源")
```
