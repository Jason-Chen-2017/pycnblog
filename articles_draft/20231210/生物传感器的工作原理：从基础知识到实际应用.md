## 4.1 信号处理代码实例

以心电图分析为例，我们可以使用Python的NumPy库来实现滤波和去噪操作。以下是一个简单的示例代码：

```python
import numpy as np
import matplotlib.pyplot as plt

# 心电图信号
ecg_signal = np.load('ecg_signal.npy')

# 滤波：使用低通滤波去除低频噪声
filtered_signal = np.lowpass(ecg_signal, fs=1000, fc=25)

# 去噪：使用均值滤波去除噪声
denoised_signal = np.mean(filtered_signal, axis=1)

# 绘制原始信号和处理后信号
plt.figure()
plt.plot(ecg_signal, label='原始信号')
plt.plot(denoised_signal, label='处理后信号')
plt.legend()
plt.show()
```

在这个示例中，我们首先加载心电图信号，然后使用低通滤波去除低频噪声，再使用均值滤波去除噪声。最后，我们绘制原始信号和处理后信号，以观察信号的变化。

# 提取时域特征：均值、标准差、峰值、波峰间距等

# 合并时域特征和频域特征

# 标准化特征
scaler = StandardScaler()

# 绘制特征分布

plt.xlabel('特征值')
plt.ylabel('频数')

在这个示例中，我们首先加载处理后的心电图信号，然后提取时域特征（如均值、标准差、峰值、波峰间距等）和频域特征（通过主成分分析）。最后，我们将时域特征和频域特征合并，并进行标准化处理。最后，我们绘制特征分布，以观察特征的分布情况。
