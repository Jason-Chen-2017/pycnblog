## 3.1 Diffie-Hellman 密钥交换

Diffie-Hellman 密钥交换是一种密钥交换算法，它允许两个通信方在公开的通信通道上交换密钥。这种算法的核心思想是，两个通信方分别选择一个随机数，然后根据这个随机数计算一个公共参数。最后，两个通信方分别使用这个公共参数计算出相同的密钥。

具体操作步骤如下：

1. 通信方A选择一个随机数a，计算公共参数g^a mod p，然后将公共参数发送给通信方B。
2. 通信方B选择一个随机数b，计算公共参数g^b mod p，然后将公共参数发送给通信方A。
3. 通信方A使用公共参数B计算密钥，公共参数B^a mod p。
4. 通信方B使用公共参数A计算密钥，公共参数A^b mod p。

数学模型公式为：

公共参数 = g^a mod p
公共参数 = g^b mod p
密钥 = (公共参数^a) mod p
密钥 = (公共参数^b) mod p

## 3.2 RSA 密钥交换

RSA 密钥交换是一种密钥交换算法，它允许两个通信方在公开的通信通道上交换密钥。这种算法的核心思想是，两个通信方分别选择一个大素数，然后计算一个公共参数。最后，两个通信方分别使用这个公共参数计算出相同的密钥。

具体操作步骤如下：

1. 通信方A选择两个大素数p和q，计算n = p * q，然后计算φ(n) = (p-1) * (q-1)。
2. 通信方A选择一个随机数e，使得1 < e < φ(n)，并且gcd(e, φ(n)) = 1。
3. 通信方A计算公共参数n和e，然后将公共参数发送给通信方B。
4. 通信方B选择一个随机数d，使得1 < d < φ(n)，并且gcd(d, φ(n)) = 1。
5. 通信方B使用公共参数n和e计算密钥，公共参数n^d mod φ(n)。

数学模型公式为：

公共参数 = n, e
密钥 = d
密钥 = (n^d) mod φ(n)

## 4.1 Diffie-Hellman 密钥交换

```python
import random

def pow_mod(a, b, p):
    res = 1
    while b > 0:
        if b & 1:
            res = (res * a) % p
        a = (a * a) % p
        b >>= 1
    return res

def diffie_hellman(p, g, a, b):
    A = pow_mod(g, a, p)
    B = pow_mod(g, b, p)
    K = pow_mod(B, a, p)
    K = pow_mod(A, b, p)
    return K

```

## 4.2 RSA 密钥交换

```python
import random

def pow_mod(a, b, p):
    res = 1
    while b > 0:
        if b & 1:
            res = (res * a) % p
        a = (a * a) % p
        b >>= 1
    return res

def rsa_key_pair(p, q):
    n = p * q
    phi = (p - 1) * (q - 1)
    d, e = 1, 1
    while not gcd(d, phi) == 1:
        e += 1
        d += 1
        if gcd(d, phi) == 1:
            break
    return n, e, d

def rsa_encrypt(n, e, m):
    c = pow_mod(m, e, n)
    return c

def rsa_decrypt(n, d, c):
    m = pow_mod(c, d, n)
    return m

p = 13
q = 11
n, e, d = rsa_key_pair(p, q)
m = 23

c = rsa_encrypt(n, e, m)
print(c)

m = rsa_decrypt(n, d, c)
print(m)
```
