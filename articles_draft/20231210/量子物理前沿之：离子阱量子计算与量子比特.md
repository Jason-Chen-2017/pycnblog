# 2.核心概念与联系

## 2.1 量子比特（Qubit）
量子比特是量子计算中的基本单位，它可以存储两种不同的状态：0和1，以及它们之间的混合状态。量子比特的状态可以用纯态或混合态来描述。纯态的量子比特可以用向量表示，如|0⟩和|1⟩，混合态的量子比特可以用密度矩阵表示。

## 2.2 量子门
量子门是量子计算中的基本操作单元，它可以对量子比特进行操作。量子门的典型例子包括：Hadamard门（H）、Pauli-X门（X）、Pauli-Y门（Y）、Pauli-Z门（Z）、CNOT门（C）等。这些门可以用矩阵来描述，如H门可以用H矩阵表示，CNOT门可以用CNOT矩阵表示。

## 2.3 离子阱
离子阱是离子阱量子计算的核心设备，它是一种能够存储和操作离子的微型容器。离子阱通常由电场和磁场构成，可以将离子吸收到内部，并对离子进行存储和操作。离子阱的主要优点是高可扩展性、低噪声性和高稳定性。

    """
    量子测量

    probabilities = np.abs(qubit.state)**2
