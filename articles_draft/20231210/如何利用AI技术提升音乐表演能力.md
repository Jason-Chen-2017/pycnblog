### 4.1 音乐分析

我们可以使用Python的librosa库来实现音乐分析。以下是一个简单的音乐分析代码实例：

```python
import librosa

# 加载音乐数据
y, sr = librosa.load('music.wav')

# 提取音乐特征
mfcc = librosa.feature.mfcc(y=y, sr=sr)

# 分析音乐结构
chroma = librosa.feature.chroma_stft(y=y, sr=sr)

# 生成分析结果
librosa.display.specshow(chroma, y_axis='chroma', x_axis='time')
```

在这个代码实例中，我们首先加载音乐数据，然后提取音乐特征，例如MFCC和Chroma。最后，我们生成音乐分析的结果，例如音乐结构图。

### 4.2 音乐创作

我们可以使用Python的music21库来实现音乐创作。以下是一个简单的音乐创作代码实例：

```python
import music21

# 加载音乐数据
score = music21.stream.Stream()
score.insert(0, music21.note.Note('C4'))
score.insert(0, music21.note.Note('E4'))
score.insert(0, music21.note.Note('G4'))

# 生成音乐作品
score.write('midi', fp='music.mid')
```

在这个代码实例中，我们首先加载音乐数据，然后生成新的音乐作品，例如和弦。最后，我们将音乐作品保存为MIDI文件。
