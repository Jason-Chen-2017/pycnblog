# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的代码实例来说明区块链技术在教育领域的应用。

```python
import hashlib
import time

class Block:
    def __init__(self, data, timestamp):
        self.data = data
        self.timestamp = timestamp
        self.prev_hash = None
        self.hash = self.calc_hash()

    def calc_hash(self):
        sha = hashlib.sha256()
        sha.update(str(self.data).encode('utf-8'))
        sha.update(str(self.timestamp).encode('utf-8'))
        sha.update(str(self.prev_hash).encode('utf-8'))
        return sha.hexdigest()

    def __str__(self):
        return 'Data: {}, Timestamp: {}, Prev Hash: {}, Hash: {}'.format(
            self.data, self.timestamp, self.prev_hash, self.hash)

class Chain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(data='Genesis Block', timestamp=time.time())

    def get_last_block(self):
        return self.chain[-1]

    def add_block(self, data):
        prev_block = self.get_last_block()
        new_block = Block(data, time.time())
        new_block.prev_hash = prev_block.hash
        self.chain.append(new_block)

# 创建区块链
chain = Chain()

# 添加区块
chain.add_block('学术荣誉证书')
chain.add_block('学生成绩')
chain.add_block('教师资格证书')

# 输出区块链
for block in chain.chain:
    print(block)
```

在上述代码中，我们创建了一个简单的区块链，包含三个区块，分别表示学术荣誉证书、学生成绩和教师资格证书。通过使用哈希函数，我们确保了每个区块的数据完整性和不可篡改性。
