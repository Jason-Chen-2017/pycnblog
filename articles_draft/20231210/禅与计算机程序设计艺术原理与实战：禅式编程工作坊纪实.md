### 4.1 代码实例一：求两个数的最大公约数

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a
```

这段代码实现了求两个数的最大公约数的功能。通过使用while循环，我们可以不断更新a和b的值，直到b为0为止。在这个过程中，我们使用了模运算（a % b）来计算a和b的最大公约数。

class Node:

        self.value = value
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, value):
        node = Node(value)
        if not self.head:
            self.head = node
            return

这段代码实现了一个简单的链表。链表是一种数据结构，它由一系列节点组成，每个节点都包含一个值和一个指向下一个节点的指针。在这个实例中，我们定义了一个Node类来表示链表中的一个节点，并定义了一个LinkedList类来表示链表本身。通过使用append方法，我们可以将新的节点添加到链表的末尾。

### 4.3 代码实例三：实现简单的栈

```python
class Stack:
    def __init__(self):
        self.items = []

    def push(self, value):
        self.items.append(value)

    def pop(self):
        if not self.items:
            return None
        return self.items.pop()

    def peek(self):
        if not self.items:
            return None
        return self.items[-1]

    def is_empty(self):
        return not self.items
```

这段代码实现了一个简单的栈。栈是一种数据结构，它后进先出（LIFO）。在这个实例中，我们定义了一个Stack类来表示栈，并定义了一系列方法来实现栈的基本操作，包括push（添加元素）、pop（移除元素）、peek（查看栈顶元素）和is_empty（判断栈是否为空）。
