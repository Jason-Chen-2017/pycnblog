## 4.2 优化代码实例

以下是一个简单的优化代码实例，用于实现城市能源资源的最优分配和最优调度：

```python
from scipy.optimize import linprog

# 定义目标函数和约束条件
c = [-1, -1]  # 代表能源资源的负价值
A = [[1, 1], [1, 2]]  # 代表能源资源的分配矩阵
b = [100, 200]  # 代表能源资源的总量

# 解决线性规划问题
result = linprog(c, A_ub=A, b_ub=b, method='simplex')

# 输出结果
print('Optimal solution:', result.x)
print('Optimal value:', -result.fun)
```

## 4.3 模拟代码实例

以下是一个简单的模拟代码实例，用于实现城市气候数据的模拟和预测：

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义模拟函数
def simulate(t, sigma, rho, omega):
    x = np.zeros(t)
    x[0] = omega
    for i in range(1, t):
        x[i] = sigma * x[i-1] * np.cos(rho * x[i-1]) + rho * x[i-1] * omega
    return x

# 设置参数
t = 1000
sigma = 0.5
rho = 0.1
omega = np.random.normal(0, 1)

# 生成随机气候数据
np.random.seed(42)
random_data = np.random.normal(size=t)

# 模拟气候数据
simulated_data = simulate(t, sigma, rho, omega)

# 绘制图像
plt.plot(simulated_data)
plt.plot(random_data)
plt.legend(['Simulated', 'Random'])
plt.xlabel('Time')
plt.ylabel('Temperature')
plt.title('Simulated and Random Temperature Data')
plt.show()
```
