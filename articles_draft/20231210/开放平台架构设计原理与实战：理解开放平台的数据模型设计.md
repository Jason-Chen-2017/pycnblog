### 4.3 数据处理算法实例

在这个例子中，我们将实现一个简单的用户分组算法，根据用户的年龄将用户分为不同的年龄段。以下是代码实例：

```python
def group_users_by_age(users):
    age_groups = {
        '0-18': [],
        '19-35': [],
        '36-50': [],
        '51+': []
    }
    for user in users:
        age_group = '0-18' if user.age < 18 else '19-35' if user.age < 35 else '36-50' if user.age < 50 else '51+'
        age_groups[age_group].append(user)
    return age_groups
```

在这个例子中，我们定义了一个`group_users_by_age`函数，它接收一个用户列表作为输入，并将用户根据年龄分为不同的年龄段。我们使用字典来存储年龄段和相应的用户列表，然后遍历用户列表并将用户添加到相应的年龄段中。最后，我们返回一个字典，其中包含了每个年龄段的用户列表。
