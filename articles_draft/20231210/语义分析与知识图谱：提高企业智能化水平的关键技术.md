# 4.具体代码实例和详细解释说明

## 4.1 语义分析的具体代码实例

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.tag import pos_tag

def semantic_analysis(text):
    words = word_tokenize(text)
    tagged_words = pos_tag(words)
    return tagged_words
```

## 4.2 知识图谱的具体代码实例

```python
import networkx as nx
from knowledge_graph import KnowledgeGraph

def knowledge_graph_construction(tagged_words):
    kg = KnowledgeGraph()
    for word, tag in tagged_words:
        if tag == 'NNP':
            kg.add_entity(word)
        elif tag == 'NN':
            kg.add_relation(word)
        elif tag == 'IN':
            kg.add_property(word)
    kg.build()
    return kg
```
