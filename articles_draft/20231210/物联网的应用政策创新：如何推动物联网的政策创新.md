## 4.3 优化算法

优化算法可以通过各种算法来实现，如梯度下降、粒子群优化、遗传算法等。以下是一个简单的Python代码实例，用于实现梯度下降算法：

```python
import numpy as np

# 定义梯度下降函数
def gradient_descent(X, y, theta, alpha, iterations):
    m = len(y)
    for _ in range(iterations):
        h = np.dot(X, theta)
        error = h - y
        gradient = np.dot(X.T, error) / m
        theta = theta - alpha * gradient
    return theta

# 读取数据
data = pd.read_csv('data.csv')

# 划分训练集和测试集
X_train = data.iloc[:int(len(data)*0.8), :-1]
X_test = data.iloc[int(len(data)*0.8):, :-1]
y_train = data.iloc[:int(len(data)*0.8), -1]
y_test = data.iloc[int(len(data)*0.8):, -1]

# 初始化参数
theta = np.zeros(X_train.shape[1])
alpha = 0.01
iterations = 1000

# 训练模型
theta = gradient_descent(X_train, y_train, theta, alpha, iterations)

# 预测结果
y_pred = np.dot(X_test, theta)
```
