# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来说明无监督学习在推荐系统中的应用。

假设我们有一个电影推荐系统，用户可以对电影进行评分。我们可以使用基于用户的协同过滤方法来推荐电影。具体步骤如下：

1. 对用户的历史行为进行分类，将相似的行为归入同一类别。
2. 根据用户的历史行为计算用户之间的相似度。
3. 找到与用户喜好相似的其他用户。
4. 根据这些用户的喜好推荐电影。

具体代码实例如下：

```python
import numpy as np
from scipy.spatial.distance import cosine

# 用户历史行为
user_history = [
    [5, 4, 3, 2, 1],  # 用户1的历史评分
    [4, 5, 3, 2, 1],  # 用户2的历史评分
    [3, 4, 5, 2, 1],  # 用户3的历史评分
    [2, 3, 4, 5, 1],  # 用户4的历史评分
    [1, 2, 3, 4, 5]   # 用户5的历史评分
]

# 计算用户之间的相似度
similarity = cosine(user_history)

# 找到与用户喜好相似的其他用户
similar_users = []
for user in user_history:
    similar_users.append(similarity[user])

# 根据这些用户的喜好推荐电影
recommended_movies = []
for user in user_history:
    recommended_movies.append(user_history[similar_users.index(similar_users.index(user))])

print(recommended_movies)
```
