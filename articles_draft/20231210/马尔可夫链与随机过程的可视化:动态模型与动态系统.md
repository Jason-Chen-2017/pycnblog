## 3.1随机过程的可视化

### 3.1.1Python的Matplotlib库

Python的Matplotlib库是一个用于创建静态、动态和交互式图形的库。Matplotlib库可以用来可视化随机过程，例如随机走势图、随机分布图等。

### 3.1.2Matplotlib库的基本使用

Matplotlib库的基本使用包括以下步骤：

1.导入Matplotlib库
```python
import matplotlib.pyplot as plt
```
2.创建一个新的图形窗口
```python
fig = plt.figure()
```
3.在图形窗口上添加一个子图
```python
ax = fig.add_subplot(111)
```
4.使用子图绘制随机过程的可视化图形
```python
ax.plot(x, y)
```
5.显示图形
```python
plt.show()
```

以下是一个随机走势图的可视化示例：

```python

```
2.创建一个新的网络图
```python
G = nx.DiGraph()
```
3.在网络图上添加节点和边
```python
G.add_node(node)
G.add_edge(node1, node2)
```
4.使用网络图绘制马尔可夫链的可视化图形
```python

```

```python
plt.show()
```

Q: 如何生成随机数列？
A: 可以使用Python的numpy库的random.normal函数生成随机数列。例如，以下代码生成了一个含有100个随机数的列表：
```python
import numpy as np

x = np.random.normal(0, 1, 100)
```

Q: 如何创建一个新的网络图？
A: 可以使用Python的networkx库的DiGraph函数创建一个新的网络图。例如，以下代码创建了一个新的有向图：
```python
import networkx as nx

G = nx.DiGraph()
```

Q: 如何在网络图上添加节点和边？
A: 可以使用Python的networkx库的add_node和add_edge函数在网络图上添加节点和边。例如，以下代码在网络图上添加了一个节点和一个边：
```python
import networkx as nx

G = nx.DiGraph()

G.add_node(1)
G.add_edge(1, 2)
```
