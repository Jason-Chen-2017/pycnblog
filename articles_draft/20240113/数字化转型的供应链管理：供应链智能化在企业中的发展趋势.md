线性规划的基本公式为：

$$

其中，$x$ 是变量向量，$c$ 是目标函数系数向量，$A$ 是约束矩阵，$b$ 是约束向量。

动态规划的基本公式为：

其中，$f(x)$ 是目标函数，$Y(x)$ 是可能的决策集合，$g(x, y)$ 是决策成本函数。

## 4.1 线性规划示例

在这个示例中，我们使用 Python 的 SciPy 库来实现线性规划。

```python
from scipy.optimize import linprog

# 目标函数系数向量
c = [1, 2, 3]

# 约束矩阵
A = [[-1, -1, 0], [-1, 0, -1], [0, -1, -1]]

# 约束向量
b = [-10, -10, -10]

# 解决线性规划问题
x = linprog(c, A_ub=A, b_ub=b, method='highs')

print(x)
```

在这个示例中，我们使用 SciPy 库的 linprog 函数来解决线性规划问题。目标函数系数向量为 `[1, 2, 3]`，约束矩阵为 `[[ -1, -1, 0], [-1, 0, -1], [0, -1, -1]]`，约束向量为 `[-10, -10, -10]`。通过运行这个示例，我们可以得到一个最优解 `[0.0, 10.0, 0.0]`。

## 4.2 动态规划示例

在这个示例中，我们使用 Python 的 NumPy 库来实现动态规划。

```python
import numpy as np

# 目标函数
def f(x):
    return np.sum(x**2)

# 决策成本函数
def g(x, y):
    return np.abs(x - y)

# 初始状态
x = 0

# 决策集合
Y = [1, 2, 3]

# 解决动态规划问题
min_f = float('inf')
for y in Y:
    min_f = min(min_f, f(x) + g(x, y))

print(min_f)
```

在这个示例中，我们使用 NumPy 库来实现动态规划。目标函数为 `f(x) = x^2`，决策成本函数为 `g(x, y) = |x - y|`，初始状态为 `x = 0`，决策集合为 `[1, 2, 3]`。通过运行这个示例，我们可以得到一个最优解 `min_f = 0`。

## 4.3 机器学习示例

在这个示例中，我们使用 Python 的 Scikit-learn 库来实现机器学习。

```python

```

在这个示例中，我们使用 Scikit-learn 库的 LogisticRegression 函数来训练一个逻辑回归模型。训练数据为 `[[1, 2], [2, 3], [3, 4], [4, 5]]`，标签为 `[0, 1, 0, 1]`。通过运行这个示例，我们可以得到一个预测输出 `[1]`。
