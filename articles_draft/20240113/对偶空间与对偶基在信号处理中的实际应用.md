# 4. 具体代码实例和详细解释说明

具体的代码实例和详细解释说明包括：

1. 使用傅里叶变换和逆变换实现信号的压缩和恢复：

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.fft import fft, ifft

# 信号生成
t = np.linspace(0, 1, 1000)
x = np.sin(2 * np.pi * 5 * t) + np.sin(2 * np.pi * 10 * t)

# 傅里叶变换
X = fft(x)

# 压缩
Y = X[:500]

# 逆变换
y = ifft(Y)

# 绘制
plt.figure()
plt.plot(t, x, label='原信号')
plt.plot(t, y, label='恢复信号')
plt.legend()
plt.show()
```

2. 使用波лет变换和逆变换实现信号的滤波和过滤：

import numpy as np
import matplotlib.pyplot as plt
from scipy.signal import fftconvolve

# 信号生成

# 滤波器生成

# 波лет变换
X = fft(x)

# 滤波

# 绘制

plt.plot(t, x, label='原信号')
