## 3.1 对称密码学

对称密码学是密码学的一种，它使用同一个密钥来进行加密和解密。对称密码学的核心算法有以下几个方面：

1. 数据加密标准（DES）：DES是一种对称密码学算法，它使用56位密钥来进行加密和解密。DES的数学模型公式如下：

$$
E_k(P) = P \oplus F_k(P)
$$

其中，$E_k(P)$表示加密后的数据，$P$表示原始数据，$F_k(P)$表示密钥$k$加密后的数据，$\oplus$表示异或运算。

1. 三重数据加密算法（3DES）：3DES是DES的一种改进版本，它使用三个56位密钥来进行加密和解密。3DES的数学模型公式如下：

$$
E_{k_3}(E_{k_2}(E_{k_1}(P))) = P
$$

其中，$E_{k_1}(P)$表示使用密钥$k_1$加密后的数据，$E_{k_2}(E_{k_1}(P))$表示使用密钥$k_2$加密后的数据，$E_{k_3}(E_{k_2}(E_{k_1}(P)))$表示使用密钥$k_3$加密后的数据。

## 3.2 非对称密码学

非对称密码学是密码学的另一种，它使用一对公钥和私钥来进行加密和解密。非对称密码学的核心算法有以下几个方面：

1. Diffie-Hellman密钥交换：Diffie-Hellman密钥交换是一种非对称密码学算法，它允许两个用户在公开通道上安全地交换密钥。Diffie-Hellman密钥交换的数学模型公式如下：

$$
A \rightarrow B: g^a \mod p
$$
$$
B \rightarrow A: g^b \mod p
$$
$$
A \rightarrow B: (g^b)^a \mod p = g^{ab} \mod p
$$
$$
B \rightarrow A: (g^a)^b \mod p = g^{ab} \mod p
$$

其中，$A$和$B$是两个用户，$g$是一个大素数，$p$是一个素数，$a$和$b$是两个随机数。

$$

$$
$$
A \rightarrow B: m^e \mod n
$$
$$
B \rightarrow A: (m^e)^d \mod n = m
$$

其中，$A$和$B$是两个用户，$e$和$d$是两个大素数，$n$是一个素数，$m$是一个明文。

## 4.1 对称密码学实例

对称密码学的一个简单实例是AES加密。以下是AES加密和解密的Python代码实例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad

# 生成一个128位密钥
key = get_random_bytes(16)

# 创建一个AES加密器
cipher = AES.new(key, AES.MODE_CBC)

# 加密数据
data = b"Hello, World!"
ciphertext = cipher.encrypt(pad(data, AES.block_size))

# 解密数据
plaintext = unpad(cipher.decrypt(ciphertext), AES.block_size)

print(plaintext)
```

在上述代码中，我们使用了PyCrypto库来实现AES加密和解密。首先，我们生成了一个128位密钥，然后创建了一个AES加密器。接下来，我们使用AES加密器加密了一个字符串“Hello, World!”，并将加密后的数据存储在变量`ciphertext`中。最后，我们使用AES加密器解密了加密后的数据，并将解密后的数据存储在变量`plaintext`中。

## 4.2 非对称密码学实例

非对称密码学的一个简单实例是RSA加密。以下是RSA加密和解密的Python代码实例：

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# 生成一个RSA密钥对
key = RSA.generate(2048)

# 获取公钥和私钥
public_key = key.publickey()
private_key = key

# 使用公钥加密数据
data = b"Hello, World!"
cipher = PKCS1_OAEP.new(public_key)
ciphertext = cipher.encrypt(data)

# 使用私钥解密数据
cipher = PKCS1_OAEP.new(private_key)
plaintext = cipher.decrypt(ciphertext)

print(plaintext)
```

在上述代码中，我们使用了PyCrypto库来实现RSA加密和解密。首先，我们生成了一个2048位RSA密钥对，并获取了公钥和私钥。接下来，我们使用公钥加密了一个字符串“Hello, World!”，并将加密后的数据存储在变量`ciphertext`中。最后，我们使用私钥解密了加密后的数据，并将解密后的数据存储在变量`plaintext`中。
