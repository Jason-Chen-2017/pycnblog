```python

from sklearn.linear_model import LinearRegression

# 数据
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([2, 3, 4, 5])

# 训练
```

5. 神经网络模型：
```python
import numpy as np
import tensorflow as tf

# 数据
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([2, 3, 4, 5])

# 模型
model = tf.keras.Sequential([

y_pred = model.predict(X_new)
print(y_pred)
```
