## 1.1 计算几何的基本概念

计算几何主要研究的是几何问题，它涉及到点、线段、线、曲线等几何对象之间的关系。这些对象可以用二元函数来描述，例如：

- 点：可以用二元函数表示为(x, y)，其中x和y是实数。
- 线段：可以用二元函数表示为y = ax + b，其中a和b是实数。
- 线：可以用二元函数表示为y = ax + b，其中a和b是实数。
- 曲线：可以用二元函数表示为y = ax^2 + bx + c，其中a、b和c是实数。

```python
import math

def distance(x1, y1, x2, y2):
    return math.sqrt((x2 - x1) ** 2 + (y2 - y1) ** 2)

# 例如，计算两个点(1, 2)和(3, 4)之间的距离

```

## 4.2 线段的交叉和重合

```python
def intersect(a1, b1, c1, a2, b2, c2):
    x = (b2 - b1) * c1 - (c2 - c1) * b1
    y = (c2 - c1) * a1 - (a2 - a1) * c1
    return (x, y)
```
