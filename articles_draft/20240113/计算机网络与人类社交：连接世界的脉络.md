# 2.核心概念与联系
# 2.1 计算机网络
计算机网络是一种连接多个计算机和设备的系统，使它们可以相互通信和共享资源。计算机网络可以分为两类：局域网（LAN）和广域网（WAN）。局域网是一种局限在一个特定区域的网络，如家庭或办公室，而广域网则是跨越更大的区域的网络，如国家或全球。

计算机网络的主要组成部分包括：

- 计算机和设备：这些是网络中的基本组成部分，用于存储、处理和传输数据。
- 网络设备：这些设备负责将数据从一个设备传输到另一个设备，例如交换机、路由器和网桥。
- 网络协议：这些是计算机网络中的规则，用于确保数据可以正确地传输和接收。例如，TCP/IP、HTTP、FTP等。

import matplotlib.pyplot as plt

# 创建一个有向图

# 添加节点
G.add_node("Alice")
G.add_node("Bob")
G.add_node("Charlie")

# 添加边

在这个示例中，我们使用了`networkx`库来创建一个有向图，并使用`matplotlib`库来绘制图。然后，我们添加了三个节点（Alice、Bob、Charlie）和三个边（Alice -> Bob、Bob -> Charlie、Charlie -> Alice）。最后，我们使用`nx.draw()`函数来绘制图，并使用`plt.show()`函数来显示图。

# 4.2 推荐算法
在实际应用中，我们可以使用Python编程语言来实现推荐算法的算法。以下是一个简单的示例：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 文本数据
texts = ["I love machine learning", "I hate machine learning", "I love deep learning"]

# 创建TF-IDF向量化器
vectorizer = TfidfVectorizer()

# 创建TF-IDF矩阵
tfidf_matrix = vectorizer.fit_transform(texts)

# 计算相似度
cosine_similarities = cosine_similarity(tfidf_matrix[0:1], tfidf_matrix[1:2])

# 打印结果
print(cosine_similarities)
```

在这个示例中，我们使用了`sklearn`库来创建一个TF-IDF向量化器，并使用`cosine_similarity`函数来计算文本之间的相似度。然后，我们使用`print()`函数来打印结果。
