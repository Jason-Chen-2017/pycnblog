# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 欧几里得算法

欧几里得算法的核心思想是：将两个整数分解为其他两个整数的乘积，然后递归地求解这两个整数的最大公约数。欧几里得算法的具体操作步骤如下：

1. 如果 b 为 0，则 a 是两个整数的最大公约数，返回 a。
2. 如果 a 模 b 不等于 0，交换 a 和 b 的值。
3. 使用辗转相除定理，将 a 除以 b 得到一个商 q 和一个余数 r，即 a = b \* q + r。
4. 将 a 替换为 b，将 b 替换为 r，然后重复步骤 2 和 3，直到 r 为 0。
5. 当 r 为 0 时，a 和 b 的最大公约数为 b。

数学模型公式：

$$
gcd(a, b) = gcd(b, a \bmod b)
$$

## 3.3 欧拉函数

欧拉函数的核心思想是：计算一个数的所有小于它的正整数的个数。欧拉函数的具体操作步骤如下：

1. 如果 n 是一个素数，则欧拉函数为 phi(n) = n - 1。
2. 如果 n 是一个合数，则欧拉函数为 phi(n) = n \* (1 - 1/p1) \* (1 - 1/p2) \* ... \* (1 - 1/pk)，其中 p1, p2, ..., pk 是 n 的所有素因数。

数学模型公式：

$$
\phi(n) = n \times (1 - \frac{1}{p_1}) \times (1 - \frac{1}{p_2}) \times ... \times (1 - \frac{1}{p_k})
$$

## 3.4 费马小定理

费马小定理的核心思想是：一个数 mod 一个素数取值的方式等于该数的 p-1 次方取模的结果。费马小定理的具体操作步骤如下：

1. 如果 a 和 p 是互质的，则 a^(p-1) 模 p 等于 1。
2. 如果 a 和 p 不是互质的，则 a^(p-1) 模 p 可能不等于 1。

数学模型公式：

$$
a^(p-1) \equiv 1 \pmod{p}
$$
