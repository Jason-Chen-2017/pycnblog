## 3.1 排序算法

排序算法是计算机科学中的一个基本概念，用于对数据进行排序。常见的排序算法有选择排序、插入排序、冒泡排序等。

### 3.1.1 选择排序

选择排序是一种简单的排序算法，它的基本思想是在每次迭代中选择最小或最大的元素，并将其放入正确的位置。选择排序的时间复杂度为O(n^2)。

选择排序的具体操作步骤如下：

1. 从未排序的元素中选择最小的元素，并将其放入有序序列的末尾。
2. 重复第一步，直到所有元素都被排序。

### 3.1.2 插入排序

插入排序是一种简单的排序算法，它的基本思想是将元素插入到已排序的序列中的正确位置。插入排序的时间复杂度为O(n^2)。

插入排序的具体操作步骤如下：

1. 从未排序的元素中选择一个元素，并将其插入到已排序序列中的正确位置。
2. 重复第一步，直到所有元素都被排序。

### 3.1.3 冒泡排序

冒泡排序是一种简单的排序算法，它的基本思想是将最大（或最小）的元素逐渐向上（或向下）移动，直到所有元素都被排序。冒泡排序的时间复杂度为O(n^2)。

冒泡排序的具体操作步骤如下：

1. 从未排序的元素中选择两个元素，并将较大的元素放在较小的元素的后面。
2. 重复第一步，直到所有元素都被排序。

## 3.2 搜索算法

搜索算法是计算机科学中的一个基本概念，用于在数据结构中查找特定的元素。常见的搜索算法有线性搜索、二分搜索等。

### 3.2.1 线性搜索

线性搜索是一种简单的搜索算法，它的基本思想是从第一个元素开始，逐个比较每个元素，直到找到目标元素或遍历完整个序列。线性搜索的时间复杂度为O(n)。

线性搜索的具体操作步骤如下：

1. 从第一个元素开始，逐个比较每个元素，直到找到目标元素或遍历完整个序列。
2. 返回目标元素的索引。

### 3.2.2 二分搜索

二分搜索是一种高效的搜索算法，它的基本思想是将搜索范围一直缩小到目标元素所在的区间内。二分搜索的时间复杂度为O(log n)。

二分搜索的具体操作步骤如下：

1. 将搜索范围划分为两个部分，中间的元素作为拆分点。
2. 如果目标元素在拆分点的左侧，则将搜索范围设置为左侧部分；如果目标元素在拆分点的右侧，则将搜索范围设置为右侧部分；如果目标元素等于拆分点的元素，则返回拆分点的索引。
3. 重复第一步和第二步，直到搜索范围为空或找到目标元素。

## 4.1 选择排序实现

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

选择排序的原理是在每次迭代中选择最小的元素，并将其放入正确的位置。在上面的代码中，我们首先定义了一个函数`selection_sort`，接收一个数组作为参数。然后，我们使用两个嵌套循环来遍历数组，找到最小的元素并将其放入正确的位置。最后，我们返回排序后的数组。

## 4.2 插入排序实现

```python
def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

插入排序的原理是将元素插入到已排序的序列中的正确位置。在上面的代码中，我们首先定义了一个函数`insertion_sort`，接收一个数组作为参数。然后，我们使用一个循环来遍历数组，从第二个元素开始。在每次迭代中，我们将当前元素与已排序序列中的元素进行比较，并将较小的元素移动到正确的位置。最后，我们返回排序后的数组。

## 4.3 冒泡排序实现

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

冒泡排序的原理是将最大（或最小）的元素逐渐向上（或向下）移动，直到所有元素都被排序。在上面的代码中，我们首先定义了一个函数`bubble_sort`，接收一个数组作为参数。然后，我们使用两个嵌套循环来遍历数组，将较大的元素移动到正确的位置。最后，我们返回排序后的数组。
