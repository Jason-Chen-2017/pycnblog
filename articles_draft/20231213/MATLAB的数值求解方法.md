## 3.1 方程组求解

### 3.1.1 直接方法：高斯消元法

高斯消元法是一种直接方法，用于解决线性方程组。它的核心思想是通过行操作将方程组转换为上三角矩阵，然后通过回代求解得到解。

高斯消元法的具体步骤如下：

1. 对方程组进行行交换，使得方程组的第一列中的第一行非零元素为1。

2. 对方程组的第一列以外的其他行进行行减法，使得第一列除第一行外其他元素都为0。

3. 重复第1步和第2步，直到方程组变为上三角矩阵。

4. 对方程组进行列交换，使得方程组的第一列中的第一行非零元素为1。

5. 对方程组的第一列外的其他列进行列减法，使得第一列除第一行外其他元素都为0。

6. 重复第4步和第5步，直到方程组变为对角线矩阵。

7. 对方程组进行回代求解，得到解。

### 3.1.2 迭代方法：Jacobi方法

Jacobi方法是一种迭代方法，用于解决线性方程组。它的核心思想是将方程组转换为单独的单变量方程，然后通过迭代求解得到解。

Jacobi方法的具体步骤如下：

1. 对方程组进行分解，得到单变量方程。

2. 对每个单变量方程进行迭代求解，直到满足某种停止条件。

3. 更新方程组的解。

### 3.1.3 MATLAB中的方程组求解函数

MATLAB提供了多种方程组求解函数，如`solve`、`linsolve`、`mldivide`等。这些函数可以根据方程组的类型和特点选择不同的求解方法。

高斯消元法也可以用于解决偏微分方程。在这种情况下，我们需要将偏微分方程转换为一个大型的线性方程组，然后使用高斯消元法进行求解。

### 3.2.2 迭代方法：梯度下降法

梯度下降法是一种迭代方法，用于解决最小化问题。它的核心思想是通过梯度信息，逐步更新变量，使目标函数的值逐渐减小。

梯度下降法的具体步骤如下：

1. 初始化变量。

2. 计算目标函数的梯度。

3. 更新变量。

4. 重复第2步和第3步，直到满足某种停止条件。

## 4.1 方程组求解

### 4.1.1 直接方法：高斯消元法

```matlab
% 定义方程组
A = [1, 2, 3; 4, 5, 6; 7, 8, 10];
b = [6; 15; 24];

% 使用高斯消元法求解方程组
x = A\b;

% 输出结果
fprintf('x = %f, %f, %f\n', x(1), x(2), x(3));
```

### 4.1.2 迭代方法：Jacobi方法

```matlab
% 定义方程组
A = [1, 2, 3; 4, 5, 6; 7, 8, 10];
b = [6; 15; 24];

% 使用Jacobi方法求解方程组
x = zeros(3, 1);
tol = 1e-6;
max_iter = 1000;
iter = 0;

while iter < max_iter
    x_old = x;
    x = (A(1, 1) * x_old(1) + A(1, 2) * x_old(2) + A(1, 3) * x_old(3) - b(1)) / A(1, 1);
    x(2) = (A(2, 1) * x_old(1) + A(2, 2) * x_old(2) + A(2, 3) * x_old(3) - b(2)) / A(2, 2);
    x(3) = (A(3, 1) * x_old(1) + A(3, 2) * x_old(2) + A(3, 3) * x_old(3) - b(3)) / A(3, 3);
    
    error = max(abs(x - x_old));
    iter = iter + 1;
    
    if error < tol
        break;
    end
end

% 输出结果
fprintf('x = %f, %f, %f\n', x(1), x(2), x(3));
```
