## 4.2数据清洗

首先，我们需要对数据集A和数据集B进行数据清洗。我们可以使用以下代码实现数据去重、数据填充和数据删除：

```python
import pandas as pd

# 数据去重
def remove_duplicates(data):
    return data.drop_duplicates()

# 数据填充
def fill_missing(data, fill_value):
    return data.fillna(fill_value)

# 数据删除
def remove_error(data, condition):
    return data[data[condition] == True]

# 数据集A
dataA = pd.read_csv('dataA.csv')
dataA = remove_duplicates(dataA)
dataA = fill_missing(dataA, 0)
dataA = remove_error(dataA, 'age > 100')

# 数据集B
dataB = pd.read_csv('dataB.csv')
dataB = remove_duplicates(dataB)
dataB = fill_missing(dataB, 0)
dataB = remove_error(dataB, 'temperature > 45')
```

最后，我们需要对数据集A和数据集B进行数据融合。我们可以使用以下代码实现数据合并、数据聚合和数据平均：

```python
# 数据合并
def merge_data(dataA, dataB, key):
    return pd.merge(dataA, dataB, on=key)

# 数据聚合
def aggregate_data(data, key, agg_func):
    return data.groupby(key).agg(agg_func)

# 数据平均
def mean_data(data, key):
    return data.groupby(key).mean()

# 数据合并
merged_data = merge_data(dataA, dataB, 'id')

# 数据聚合
aggregated_data = aggregate_data(merged_data, 'age', ['mean', 'median', 'mode'])

# 数据平均

```
