2. 排队模型：排队模型是用于描述排队系统行为的数学模型。排队模型可以是简单的队列模型，如M/M/1队列模型，也可以是复杂的队列模型，如M/G/1队列模型。

3. 排队性能指标：排队性能指标是用于评估排队系统性能的指标，例如平均等待时间、平均服务时间、平均吞吐率等。

def queue_model(lambda_, mu, q):
    t = 0
    n = 0
    while True:
        if n == q:
            t += np.random.exponential(1 / lambda_)
