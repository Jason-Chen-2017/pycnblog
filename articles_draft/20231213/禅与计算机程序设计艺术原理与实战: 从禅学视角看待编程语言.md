### 1.4.1 动态规划的代码实例

def fib(n):
    if n < 0:
        print("Incorrect input")
    elif n == 1:
        return 0
    elif n == 2:
        return 1
    else:
        return fib(n - 1) + fib(n - 2)

### 1.4.2 贪心算法的代码实例

```python
def coin_change(coins, amount):
    dp = [float("inf") for _ in range(amount + 1)]
    dp[0] = 0

    for i in range(amount + 1):
        for coin in coins:
            if i >= coin:
                dp[i] = min(dp[i], dp[i - coin] + 1)

    return dp[amount]

coins = [1, 2, 5]
amount = 11
print(coin_change(coins, amount))
```

这段代码实现了最少硬币数的求解，它使用了贪心算法的思想来解决问题。

### 1.4.3 回溯算法的代码实例

```python
def subset_sum(candidates, target):
    def backtrack(candidates, target, start, path):
        if target == 0:
            return True
        elif target < 0:
            return False
        else:
            for i in range(start, len(candidates)):
                num = candidates[i]
                if num > target:
                    continue
                else:
                    path.append(num)
                    if backtrack(candidates, target - num, i, path):
                        return True
                    path.pop()
                    return False

    candidates.sort()
    return backtrack(candidates, target, 0, [])

candidates = [2, 3, 6, 7]
target = 7
print(subset_sum(candidates, target))
```

这段代码实现了子集和的求解，它使用了回溯算法的思想来解决问题。
