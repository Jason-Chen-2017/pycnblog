# 2.核心概念与联系

自动驾驶汽车的核心概念包括：

1.自动驾驶系统：自动驾驶系统是自动驾驶汽车的核心部分，负责控制汽车的行驶。自动驾驶系统主要包括传感器、计算机和控制系统。

2.传感器：传感器是自动驾驶系统的重要组成部分，负责收集汽车周围的环境信息。传感器主要包括雷达、摄像头、激光雷达和 GPS。

3.计算机：计算器是自动驾驶系统的核心部分，负责处理传感器收集的信息，并根据算法和规则生成控制指令。

4.控制系统：控制系统是自动驾驶系统的重要组成部分，负责根据计算机生成的控制指令来控制汽车的行驶。

自动驾驶汽车的核心概念之间的联系如下：

1.传感器收集汽车周围的环境信息，并将信息传递给计算机。

2.计算机处理收集到的信息，并根据算法和规则生成控制指令。

3.控制系统根据计算机生成的控制指令来控制汽车的行驶。

```python
import numpy as np

# 定义PID控制器
class PID:
    def __init__(self, kp, ki, kd):
        self.kp = kp
        self.ki = ki
        self.kd = kd
        self.last_error = 0

    def step(self, error):
        self.error_sum += error
        self.output = self.kp * error + self.ki * self.error_sum + self.kd * (error - self.last_error)
        self.last_error = error

    def reset(self):
        self.error_sum = 0
```
