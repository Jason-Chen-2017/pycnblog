## 4.1 数据预处理

数据预处理是将原始数据转换为适合可视化的数据格式的过程。以下是一个简单的数据预处理代码实例：

```python
import pandas as pd

# 读取原始数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.dropna()

# 数据转换
data['new_feature'] = data['old_feature'] * 10

# 数据聚合
data_agg = data.groupby('category').mean()
```

在这个代码实例中，我们使用了pandas库来读取原始数据，进行数据清洗、数据转换和数据聚合等操作。

## 4.2 图形生成

图形生成是将预处理后的数据转换为图形形式的过程。以下是一个简单的图形生成代码实例：

```python
import matplotlib.pyplot as plt

# 创建条形图
plt.bar(data_agg.index, data_agg['new_feature'])

# 设置图形标签
plt.xlabel('category')
plt.ylabel('new_feature')

# 显示图形
plt.show()
```

在这个代码实例中，我们使用了matplotlib库来创建条形图，并设置了图形标签。

## 4.3 可视化交互

可视化交互是允许用户与图形进行交互的过程。以下是一个简单的可视化交互代码实例：

```python
import ipywidgets as widgets

# 创建交互式图形
interactive_plot = widgets.InteractiveOutput()

# 设置图形交互功能
def plot_interactive(data):
    interactive_plot.render(plt.bar(data.index, data['new_feature']))

# 显示交互式图形
interactive_plot.interactive_obj.enable_edit_tools(True)
```

在这个代码实例中，我们使用了ipywidgets库来创建交互式图形，并设置了图形交互功能。
