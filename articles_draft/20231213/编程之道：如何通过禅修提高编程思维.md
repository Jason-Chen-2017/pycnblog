### 4.1 算法设计实例

```python
def find_max(nums):
    max_num = nums[0]
    for num in nums[1:]:
        if num > max_num:
            max_num = num
    return max_num
```

上述代码实例是一个寻找数组中最大值的算法。通过遍历数组中的每个元素，我们可以找到数组中的最大值。

### 4.2 数据结构实例

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, value):
        if not self.head:
            self.head = Node(value)
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = Node(value)
```

import numpy as np

def solve_quadratic(a, b, c):
    d = b**2 - 4*a*c
    if d < 0:
        return None
    elif d == 0:
        x = (-b + np.sign(b)*np.sqrt(d)) / (2*a)
        return x
    else:
        x1 = (-b + np.sqrt(d)) / (2*a)
        x2 = (-b - np.sqrt(d)) / (2*a)
        return x1, x2
```

上述代码实例是一个解二次方程的函数。通过定义一个函数，我们可以更好地实现问题的解决。
