### 3.1.2 哈希算法

哈希算法是一种将数据转换为固定长度哈希值的算法，以保护数据的完整性和安全。常见的哈希算法有MD5、SHA-1等。哈希算法可以帮助确保数据的完整性和不可篡改性，从而实现数据隐私和安全的目标。

## 4.1 加密算法实现

以下是一个使用Python实现AES加密算法的代码实例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from base64 import b64encode, b64decode

def encrypt(key, data):
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(data)
    return cipher.nonce + tag + ciphertext

def decrypt(key, ciphertext):
    nonce = ciphertext[:16]
    tag = ciphertext[16:32]
    ciphertext = ciphertext[32:]
    cipher = AES.new(key, AES.MODE_EAX, nonce=nonce)
    return cipher.decrypt_and_verify(ciphertext, tag)

key = get_random_bytes(16)
data = b"Hello, World!"
encrypted_data = encrypt(key, data)
decrypted_data = decrypt(key, encrypted_data)
print(decrypted_data)
```

以下是一个使用Python实现MD5哈希算法的代码实例：

```python
import hashlib

def md5(data):
    return hashlib.md5(data.encode()).digest()

data = "Hello, World!"
hash_data = md5(data)
print(hash_data)
```

```python
import random
from Crypto.PublicKey import ECC
from Crypto.Signature import DSS
from Crypto.Hash import SHA256

def sign(key, data):
    signer = DSS.new(key, 'fips-186-3')
    digest = SHA256.new(data)
    signature = signer.sign(digest)
    return signature

def verify(key, data, signature):
    verifier = DSS.new(key, 'fips-186-3')
    digest = SHA256.new(data)
    try:
        verifier.verify(digest, signature)
        return True
    except ValueError:
        return False

key = ECC.generate(curve='P-256')
data = "Hello, World!"
signature = sign(key.export_key(), data)
is_verified = verify(key.export_key(), data, signature)
print(is_verified)
```
