在这个例子中，我们创建了一个贝叶斯网络，其中变量A、B、C和D之间存在条件依赖关系。

```python
# 设置参数
model.add_evidence(evidence={'A': True, 'C': True})
```

在这个例子中，我们设置了贝叶斯网络的参数，变量A和C的值已知。

### 4.3 计算条件概率

最后，我们需要计算贝叶斯网络的条件概率。我们可以使用Python的`pgmpy`库来计算贝叶斯网络的条件概率。

```python
# 计算条件概率
result = model.query_probs(['B'], evidence={'A': True, 'C': True})
```
