1. 期望值：假设随机变量X的期望值为E(X)，则有：

$$
E(X) = \sum_{i=1}^{n} x_i \cdot P(x_i)
$$

其中，x_i是随机变量X的取值，P(x_i)是随机变量X的概率分布。

1. 方差：假设随机变量X的方差为Var(X)，则有：

$$
Var(X) = E[ (X - E(X))^2 ]

```python
import random

def random_algorithm(n):
    # 初始化随机数生成器
    random.seed(1)

    # 设置循环次数
    for _ in range(n):
        # 生成随机数
        x = random.randint(1, 100)

        # 判断是否满足终止条件
        if x > 50:
            # 选择最佳解
            print(x)

# 设置循环条件
n = 10

# 执行随机算法
random_algorithm(n)
```
