```python
import numpy as np

def signal_transmit(signal, distance):
    attenuation = 20 * np.log10(distance / 1000)
    received_signal = signal * np.exp(-attenuation / 20)
    return received_signal
```

### 4.2 信道分配代码实例

我们可以使用Python的NumPy库来实现信道分配的代码。以下是一个简单的信道分配示例：

```python
import numpy as np

def channel_allocation(users, channels):
    allocated_channels = np.zeros(users.shape)
    for i in range(users.shape[0]):
        for j in range(channels.shape[0]):
            if users[i] == channels[j]:
                allocated_channels[i] = j
    return allocated_channels
```

在这个示例中，我们首先导入了NumPy库，然后定义了一个名为`channel_allocation`的函数。这个函数接受一个用户列表和一个通道列表作为输入，并根据用户和通道的匹配情况分配通道。
