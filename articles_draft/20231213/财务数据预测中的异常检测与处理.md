## 4.1 异常检测的代码实例

### 4.1.1 使用Z-score方法进行异常检测

```python
import numpy as np
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 数据预处理
data = data.dropna()

# 计算Z-score
z_scores = np.abs(np.std(data, axis=0) / np.mean(data, axis=0))

# 设置阈值
threshold = 3

# 检测异常数据
outliers = data[z_scores > threshold]

# 输出异常数据
print(outliers)
```

### 4.1.2 使用IQR方法进行异常检测

```python
import numpy as np
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 数据预处理
data = data.dropna()

# 计算Q1、Q3和IQR
Q1 = data.quantile(0.25)
Q3 = data.quantile(0.75)
IQR = Q3 - Q1

# 设置阈值
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

# 检测异常数据
outliers = data[(data < lower_bound) | (data > upper_bound)]

# 输出异常数据
print(outliers)
```

## 4.2 异常处理的代码实例

### 4.2.1 删除异常数据

```python

import pandas as pd

z_scores = np.abs(np.std(data, axis=0) / np.mean(data, axis=0))
```

### 4.2.2 修改异常数据

```python
import numpy as np
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 异常检测
z_scores = np.abs(np.std(data, axis=0) / np.mean(data, axis=0))
threshold = 3
outliers = data[z_scores > threshold]

# 修改异常数据
outliers['value'] = np.mean(data)

# 合并异常数据和正常数据
data = pd.concat([data, outliers], ignore_index=True)
```
