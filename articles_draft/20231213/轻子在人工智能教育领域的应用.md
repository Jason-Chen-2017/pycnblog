## 3.1机器学习算法原理

机器学习是人工智能的一个重要分支，旨在使计算机能够从数据中学习和自动化决策。机器学习算法的核心原理包括：

- 监督学习：使用标签数据训练模型，以便在新的未标签数据上进行预测。
- 无监督学习：没有标签数据，通过自动发现数据中的结构和模式来训练模型。
- 强化学习：通过与环境的互动来学习，以便在未来的决策中最大化奖励。

## 3.2深度学习算法原理

深度学习是机器学习的一个子分支，旨在使用神经网络来解决复杂问题。深度学习算法的核心原理包括：

- 卷积神经网络（Convolutional Neural Networks，CNN）：用于图像分类和处理。
- 循环神经网络（Recurrent Neural Networks，RNN）：用于序列数据处理，如文本和语音。
- 变压器（Transformer）：一种自注意力机制的神经网络，用于自然语言处理任务，如机器翻译和文本生成。

监督学习的一个简单代码实例是线性回归。以下是一个使用Python的Scikit-learn库实现线性回归的代码示例：

```python

from sklearn.metrics import mean_squared_error

# 训练数据
X_train = [[1], [2], [3], [4]]
y_train = [1, 2, 3, 4]

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X_train, y_train)

# 预测
X_test = [[5]]
y_pred = model.predict(X_test)

# 评估
print(mean_squared_error(y_test, y_pred))
```
