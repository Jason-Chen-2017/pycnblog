# 1.背景介绍


在电子集成工艺中，术语“抛光”通常指的是对导线或者信号线进行微小扭矩使其发生漂移或偏移，从而导致信号失真、失真区域增多、波形抖动等。抛光过程能够有效提升器件的导通性，进而降低功耗，但同时也会增加了器件内部的短路、晶体管损坏概率等问题。另一方面，“平坦化”又称为离层处理，意味着将无序导电介质中的电位进行调整和平整，以解决电子材料的物理和化学特性。如此，抛光能大幅提升导通性，而平坦化则可加强器件的耐久性和抗纤维腐蚀能力。
一般来说，抛光以及平坦化都是为了降低功耗和提升导通性的手段，但实际应用时往往还需要结合实际的器件和环境因素才能最终达到最佳效果。下面就具体分析一下抛光与平坦化相关的几个关键技术问题及其解决方案。
# 2.核心概念与联系
## 2.1 概念解析
抛光和平坦化的两个基本概念为：
- “平行方向位移”（parallel-direction movement）：即各条导线的延申距离不同，比如一条导线延申5公里，另一条导线延申7公里；
- “轴向位移”（axial movement）：即轴向运动，也叫做微位移，即两导线之间垂直于主轴方向移动的距离不同，比如一条导线位移4厘米，另一条导线位移6厘米；

如下图所示，蓝色的“平行方向位移”即“抛光”，黄色的“轴向位移”即“平坦化”。

抛光和平坦化的主要作用如下：
- 抛光：可以显著地改善器件的导通性，降低功耗，并减少高频噪声；
- 平坦化：可以增强器件的耐久性和抗纤维腐蚀能力，并防止器件在外界环境中遇到撞击。

因此，如果想要实现良好的导通性，一定要尽量采用平坦化技术；如果想要提升器件的耐久性和抗纤维腐蚀能力，一定要采用抛光技术。

## 2.2 抛光技术特点
抛光技术的特点包括：
- 在不改变原本导线长度的情况下，通过调节导线与电源之间的相对位置、导线本身的排布，以及导线导带的形状等方法，使导线发生漂移或偏移，从而增加信号失真；
- 由于器件绝缘性、工作温度等原因，往往不能完全避免导线偏移，不过一般可以通过设置限流器、热阻隔离器等电气隔离设备来限制偏移程度；
- 有些特定器件尺寸和条件下，可以通过“弯曲半径使导线延申”的方法代替正交分布来引入轴向位移，从而改善导线的导通性能。

## 2.3 平坦化技术特点
平坦化技术的特点包括：
- 通过调节导线的长度、导线的稳定性、导带的宽度、导体的厚度、孔隙的大小等方法，使导线得到统一，从而消除抖动、减少轴向位移；
- 平坦化往往需要预先设计并提供给客户，这样可以保证施工、测试人员和客户的安全；
- 由于平坦化后导线的平坦程度更高，所以受力影响更大，容易出现电力线绕开、纤维腐蚀等问题。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 概述
抛光和平坦化技术的核心技术原理非常复杂，这里仅讨论基础知识。抛光技术和平坦化技术的具体操作步骤和数学模型公式细致详解，这些技巧对实际的产品开发具有重要意义。

抛光技术包括：
- 使用导射片电镜调整导线长度：电镜可以改变导线的长度，从而调整导线长度。
- 将导线串联在一起：通过串联导线可以引起导线的轴向位移，从而抵消抛光的影响。
- 在导线两端加入阻焊层：导线两端的阻焊层可以阻断电磁干扰，抑制轴向位移，提高器件的导通性。
- 对导线进行回环拉伸：用相邻导线回环拉伸，可以减轻导线的震荡现象，同时还能改善导线的导通性。

平坦化技术包括：
- 使用切割工具切割导线：通过切割导线可以切除导线中的凹洞，消除轴向位移。
- 对导线进行倾斜：将导线倾斜，可以引入轴向位移。
- 用纸板固定导线：在导线两端固定导线，可以稳固导线。
- 铺设导线的平坦层：铺设导线的平坦层，可以使导线的平坦程度更高。

下面将具体介绍以上几个技术。

## 3.2 使用导射片电镜调整导线长度
抛光首先需要使用导射片电镜调整导线长度，具体操作步骤如下：

1. 拿起导射片，把电源接入其中。
2. 按下“改变导线长度按钮”，调出导线调节工具。
3. 把导线的长度调至适当的值。
4. 把电镜安装在导线两端的相应位置，调整导线长度。

## 3.3 将导线串联在一起
一般地，在使用导线串联的时候，需要注意以下几点：

1. 保持导线的平行状态。
2. 调整导线的导带角度。
3. 选择有助于导线连接的导线。
4. 调整导线的间距。

一般来说，可以在导线上安装杆状或其他配件，来模拟电源。这样可以更好地控制导线的位置。

## 3.4 在导线两端加入阻焊层
在两端加入阻焊层可以抑制轴向位移。在现有的金属导线两端安装一个至少五公分厚度的阻焊层，使导线发生轴向位移时无法抵消。虽然阻焊层增加了一些长度，但实际上还是缩短了导线长度。

## 3.5 对导线进行回环拉伸
回环拉伸是一种减轻导线震荡现象的措施。一般地，回环拉伸的操作可以帮助导线更加平滑，使得导线的导通性更高。具体操作步骤如下：

1. 用无源的钢丝把导线的一端粘连在一起。
2. 另一端绕过导线的中点，并用电钻划开一小段电线。
3. 把一段电线的另一端接到回环拉伸的电缆末端。
4. 将导线的另一端连接回第四步建立的回环电缆，再用电钻把回环电缆另一端与电源相连。
5. 当电缆任意一侧出现明显的震荡现象时，就把回环拉伸放开。

## 3.6 使用切割工具切割导线
切割导线一般用于消除轴向位移。由于导线只有两个点之间的距离可以被切割，因此切割点一般选在导线两端较远处，切割导线的中心线可以帮助消除轴向位移。具体操作步骤如下：

1. 找到需要切割的导线。
2. 设置刀具，将导线剪成适当的尺寸。
3. 找到导线的中间位置。
4. 在导线的中心位置刀片切割，从而切割导线。
5. 使用刀片砍去导线的凹洞，消除轴向位移。

## 3.7 对导线进行倾斜
倾斜导线可以引入轴向位移。将导线倾斜也可以提升导线的导通性。具体操作步骤如下：

1. 在导线两端安装测量仪表。
2. 用电锤将导线上下端对准某个方向（一般地，倾斜度越小，导线越平坦）。
3. 调整导线的弯曲程度，使其倾斜在指定方向。
4. 当导线轴向位移时，通过仪表判断导线是否还处于倾斜状态，直到导线平坦。

## 3.8 用纸板固定导线
在导线两端安装一个至少十公分厚度的纸板，在纸板上铺设导线，固定导线。该方法可以减少导线的摩擦力，并稳固导线。具体操作步骤如下：

1. 从两端清除导线上的杂质。
2. 在导线两端安装一个至少十公分厚度的纸板。
3. 用钉针将导线粘贴在纸板上。
4. 用电锤固定导线。

## 3.9 铺设导线的平坦层
通过铺设平坦层可以增强导线的平坦程度。但是，平坦层需要根据器件的设计情况来选择，否则可能会造成导线短路、接触到液体、脱落等问题。下面介绍两种平坦层的铺设方式。

### 方法一：用沥青粘贴在导线上
这种方法适用于铺设平坦层比较简单，且平坦层是单一颜色的情况下。具体操作步骤如下：

1. 从两端清除导线上的杂质。
2. 用平底橡胶粘贴平坦层，一边粘贴，另一边用绷带拼接。
3. 不要使平坦层过长，否则可能造成导线短路。
4. 铺设导线的平坦层。

### 方法二：用玻璃杯浸泡导线
这种方法适用于铺设平坦层颜色较多或结构复杂的情况。具体操作步骤如下：

1. 从两端清除导线上的杂质。
2. 用细菌灯泡导线，并用塑料棒挂住导线。
3. 用玻璃杯装饰导线周围，使之看起来平坦。
4. 反复堆叠玻璃杯，再用塑料桶装饰导线的边缘。
5. 再次反复堆叠玻璃杯，最后装饰整个导线周围。
