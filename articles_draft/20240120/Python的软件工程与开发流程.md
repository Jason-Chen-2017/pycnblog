## 4. 具体最佳实践：代码实例和详细解释说明

在Python软件工程中，最佳实践包括：

- 遵循PEP8编码规范：使用4个空格作为缩进，每行不超过80个字符，使用空行分隔不同的逻辑块等。
- 使用虚拟环境：为每个项目创建一个独立的虚拟环境，避免依赖冲突。
- 使用类和对象：将相关的代码组织到一个类中，提高代码的可读性和可维护性。
- 使用异常处理：使用try-except块来捕获和处理异常。
- 使用装饰器：使用装饰器简化代码，提高代码的可重用性。

代码实例：

```python
import math

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

def main():
    try:
        num = int(input("Enter a positive integer: "))
        if num < 0:
            raise ValueError("Invalid input")
        else:
            print("Factorial of", num, "is", factorial(num))
    except ValueError as e:
        print(e)

if __name__ == "__main__":
    main()
```

详细解释说明：

- 使用`try-except`块捕获和处理异常。
- 使用递归实现阶乘函数。
- 使用`if-else`语句判断输入值是否为正整数。
