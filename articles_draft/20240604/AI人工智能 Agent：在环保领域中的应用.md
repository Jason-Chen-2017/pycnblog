### 5.1 空气质量预测项目实践

我们可以使用以下代码实现空气质量预测：

```python
import numpy as np

def air_quality_prediction(Q, P, alpha, learning_rate):
    Q_new = Q + learning_rate * (P - Q)
    return Q_new

Q = np.array([50, 60, 55, 60, 58])
P = np.array([60, 55, 58, 60, 62])
alpha = 0.5
learning_rate = 0.1

for i in range(len(Q) - 1):
    Q[i + 1] = air_quality_prediction(Q[i], P[i], alpha, learning_rate)
print("Predicted Air Quality:", Q[-1])
```
