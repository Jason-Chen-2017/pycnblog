# 1.背景介绍


现在大家都知道，除了经济增长，房价一直都是全球最重要的也是持续吸引眼球的“炫富”话题。但由于中国GDP占世界比重巨大、地方债务问题等诸多原因，以及政府对房地产市场的管控，中国的房地产投资成果总体上仍然不及发达国家。那么作为一个程序员，我们应该如何参与到房地产投资的大门中呢？

首先，我们得明白什么是“股份制商业”，它是指以集体的名义购买土地、建楼房，然后让所有业主共享所有的收益。股份制商业在实质上和实体企业类似，只是后者要分散在各个工厂或工业园区，而前者则由集体所有权的形式出现。房地产投资也属于这种类型的企业，只不过房屋被看作是产权和价值的载体。

其次，我们应当清楚该如何参与房地产投资，特别是在中国。因为目前国内的房地产投资市场还处于初创期，投资风险非常高。首先，每年的新房销售预计将超过销售总量的90%以上，考虑到海外市场投资热潮，中国的房地产投资将会受到更多关注。同时，国内金融市场将会加速发展，资本市场流动性也将变得更为充裕。如果真的有钱的话，就可以通过参与房地产投资来积累更多的财富。

最后，房地产投资是一个持续且艰难的过程，需要我们不断努力寻找适合自己的方向，尽可能缩小投资风险，提升投资回报。

# 2.核心概念与联系
## 2.1 关于“股份制商业”
股份制商业（Corporate Stock Ownership）是一种以集体名义购买土地或房屋，所有权归共同所有人，通过设定董事会的方式共同管理和分配利润的一种商业模式。股份制商业和普通企业相比，有两个显著特征：

1.股东必须持有股份才能享有企业的全部权力；

2.股份制商业必须是高度竞争的，否则就无法生存下去。

通常来说，股份制商业是由许多股东共同投资设立的一个企业，其资产和收益由所有者分享。股份制商业的优点是可以降低资本支出，促进企业间的竞争，增加创新的活力，因而成为资本市场上最具竞争力的组织形式。

## 2.2 房地产投资的定义
房地产投资是指以集体的名义购买土地或房屋，再将这些房子让给所有业主共同拥有的一种投资方式。它属于股份制商业的一种类型。房地产投资的目标是利用土地或房屋的价值和地租收入建立大型的商品房产业，通过销售房屋，获得房产业的利润。房地产投资者通常需要了解以下概念：

1.项目投资计划（PPP）：房地产投资者为了建立一套房产业而进行的初始筹备工作，主要包括设计户型、装修方案、确定建筑材料、计划布局等。

2.地产开发银行：房地产开发银行就是房地产投资的发起人或经营者，他们负责向社会提供房地产投资产品，承接各种类型房地产的开发、销售、运输、租赁等业务。

3.产权证券化：房地产投资可以理解为土地及相关物业的买卖交易，每张产权证都对应着一套完整房屋的所有权。

4.土地注册：在房地产投资之前，买家须先缴纳土地的注册费，并取得土地所有权证书。

5.房屋出租及销售：房地产投资必须依靠土地和房屋的可持续性，并且能够带来持续的经济效益。所以很多地方都会限制住宅楼盘的数量，以防止房价过快上涨，这往往会影响房地产投资的热度。

6.业主委员会：业主委员会由业主组成，主要职责是协调、监督、维护资产安全，确保投资者的利益。业主委员会的选举以及其他活动，都会影响房地产投资者的收益和投资决策。

7.项目评估审批：房地产投资者在投资之前都需要通过专业的审查团队评估项目的基础条件是否符合要求，并与建筑工程师、勘察师等进行沟通，确保项目能够按时交付。

## 3.1 中国房地产投资的现状
### 3.1.1 地块供应

根据统计数据显示，2018 年土地供应总量约为 800 万平方米。平均每平方米土地可供出租，价格高达 2.2 万元。除此之外，还有种类繁多的商品房以及住宅小区，价格也随之飙升。但是，2018 年土地供应总量仅占 GDP 比重的 2.2% 。

### 3.1.2 房地产投资市场初探
由于房地产投资的诸多限制，导致中国的房地产投资市场还处于初创期。近几年，房地产市场的发展已经进入了第一次全面的爆发期，屡创纪录的房价曾连续七年全线直线上升。由于土地供应不足，导致房地产投资者需要极大的信心来进行资本投资，同时，要对自己进行尽职调查，对土地的产权了解的很透彻，因此，成功的房地产投资者往往具有良好的金融实力。

据中国住房和城乡建设部发布的数据，2018 年我国出让土地证券共计发行 1789 亿股，较 2017 年增长 17% 。其中，住宅占据绝大部分。截至 2019 年底，中国共有城镇住宅小区 1700 余万个，位居全国第二、三位。2019 年国内住宅销售总额已超百亿元，同比增长 30% 。其中，49 个国家、9 个省份的平均每平方米住宅销售均价均在 4000～5000 元之间。

截止 2019 年底，全国 42 个城市新建住宅小区累计完成销售总面积 1.29 万平方米，近六成的小区已售出。中国新房预计将再添 600 万套左右。

### 3.1.3 房地产投资陷阱
#### 3.1.3.1 “抢地王”模式
投资者无论是购买房子还是贷款买房，都会面临一个问题——到底该投多少，在哪里买。这是由于中国的土地供应极度短缺，且土地分布极不集中，造成房地产投资的难题。因此，一些投资者可能会选择跟风，采用抢地王模式，挖掘潜在的价值并快速获得一手房地产。

抢地王模式意味着采用少量资金，进行大规模土地抢占，通过过度出租，盗窃以及房地产过热等手段，迅速筹集资金用于购买更多的土地。抢地王模式导致的结果就是抛售土地后的产权变卖，并产生严重的金融风险。

#### 3.1.3.2 无效投资
许多房地产投资者在进行房地产投资时，没有深入了解房地产市场的规则和法律法规，只注重如何找到一个好的项目，在项目首期就布局完善的资金来源，却忽略了投资方法的核心原理——价值。

在中国，房地产投资的“价值”指的是房地产的长期回报，而不是单次的获利。这种高标准要求投资者追求一个持久的收益，因此，一些投资者会放弃单次投资而投资更多的房子，甚至更多的项目。在这样的背景下，投资者们容易形成一种无效的投资习惯——购置一堆不值得拥有的房子，浪费钱财，最终损失惨重。

#### 3.1.3.3 投机炒房
虽然中国的房地产投资已经成为中国人最爱、最依赖的消费品之一，但是，随着城镇化进程的推进，房地产市场的日渐萎缩，投机炒房的现象也越发常见。对于投资者来说，炒房不仅伤害了房地产市场的长远发展，而且带来了一定的金融风险。

在中国，炒房行为主要集中在两个领域：一是使用房地产抵押贷款，二是高送转、高利率的物业服务，比如房地产中介、电梯安装、装修服务等。虽然如此，炒房行为的确呈现出一种新的特征：通过散乱且不合规的手段获取暴利，然后又在抛售时产权变卖。

## 3.2 程序员如何参与房地产投资
### 3.2.1 找到合适的项目
在市场推广期，每个人都希望自己的项目成功，但如果没有行业经验或者知识储备，就可能会遭遇失败的窘境。比如，很多刚毕业的学生会希望得到一手好的房子，其实恰恰相反，他们可能缺乏足够的专业能力和知识。

一般情况下，有经验的投资者喜欢在自己擅长的领域尝试新的事物，房地产市场也一样。但对于初级投资者而言，尤其是那些刚入行的小白们，需要多加注意培养自己的专业知识和能力。

在这里，我们推荐四步走的方法来寻找适合自己的项目：

1. 了解自己的能力范围。首先，要确定自己的能力范围，不能以没有丰富经验为耻，以免招致忽视的骂名。根据自己的专业背景和年龄，选择自己感兴趣的行业，或者围绕自己所在行业，找到自己擅长的领域。

2. 拓宽知识面。房地产投资与许多其他行业不同，它的知识面比较狭窄。一般而言，投资者只需要懂得土地市场、房地产市场的基本原理、房地产投资的流程即可。了解基础知识、选股分析等，这些都是可以帮助投资者构建投资组合的基本能力。

3. 关注热点事件。许多投资者喜欢涉猎新闻，能够及时了解业界动态和热点事件，能够避免一些隐患。

4. 提高动手能力。投资是一个技术活儿，只有熟练掌握技术才能制定出色的投资策略。学会做交易、搭建项目、谋定制等，这些都是投资成功的必要条件。

### 3.2.2 选择行业领域
在选择行业领域的时候，也可以综合参考一下美国、欧洲、日本等国家的人才需求，来判断自己所在的行业是否偏好房地产市场。例如，美国的房地产投资市场占有率较高，但欧美国家的投资者却并不觉得房地产有那么重要。而日本的房地产投资也处于一个比较初级阶段，缺乏相应的经验和资源。

另外，中国的房地产投资市场还处于初创期，有一些行业还没有太多的公司和专家投入，因此，投资者们也更需要注意相应行业发展的规律性和长期趋势。同时，要结合自身实际情况，控制风险。

### 3.2.3 选择合适的规模
选择合适的规模对于房地产投资者来说，是一个比较难的问题。因为中国的房地产投资市场还处于初创期，房地产的房子很少，而且均价偏低。另外，很多投资者认为“大胆地投资”会带来巨大的回报，这也是受到“抢地王”模式和无效投资的影响。

通常情况下，初级投资者选择规模较小的项目，通过打造持续性的投资组合，来实现长期的收益。当然，也可以在较大的项目上进行有限的投资。但是，更大的项目的投资回报往往比小项目更为重要，因此，初级投资者需要慎重选择。

### 3.2.4 掌握必要的工具
选择适合自己的投资工具十分重要。投资工具的选择往往直接影响投资者的投资能力，对投资的效率、稳健性和安全性都有着决定性的作用。

房地产投资者应该在以下三个方面考虑自己的投资工具：

1. 投资账户。投资账户的选择，直接影响投资者的风险承受能力和投资效率。投资账户既要保证安全性，又要充满投资的动力。比如，选择一个安全可靠的平台账户，不要使用个人银行账户，这样可以避免金融危机带来的损失。

2. 买卖双方的关系。买方通常会给予投资者一个较低的购买价格，并且提供了一些短期的返点。投资者可以根据对方的策略，选择性地接受或拒绝购房需求。这可以有效减少税费开支。

3. 资产管理工具。房地产投资者往往需要使用专业的资产管理工具来记录自己的投资信息，并且定期更新相关的文件。掌握相应的工具，可以节省时间和精力，提升工作效率。

### 3.2.5 实践并检验自己的投资策略
实践和检验自己的投资策略十分重要。如果没有充足的实践经验，很难判断自己的投资策略是有效的，还是没什么用的。房地产投资者通常会选择较为保守的策略，以便在较长的一段时间内保持较高的收益。但随着时间的推移，投资者也需要不断完善自己的策略，根据市场环境、个人需求以及投资理念等，逐步提升策略水平。

房地产投资者需要时刻关注政策变化，适时的调整投资策略，并做好策略风险的把握。由于中国房地产市场的特殊性，投资者需要适应市场的节奏和波动，不能轻易妥协，而是充分利用市场的力量，为自己的投资获得最大的收益。
