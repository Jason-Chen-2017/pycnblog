4.3. 核心代码实现

以下是一个 Python 代码示例，用于获取 Wine Library 和 Wine Advantage 上相同年份的葡萄酒，并比较它们的价格：
```python
import requests
import json
from urllib.parse import urlencode

# 设置请求参数
url = 'https://www.wine-library.com/search/?q=Burgundy&v=1'

# 请求数据并解析
response = requests.get(url)
data = json.loads(response.text)

# 查找符合条件的产品
condition = {'vintage': 2018, 'origin': 'Burgundy'}
products = []
for product in data['products']:
    if condition:
        products.append(product)

# 比较价格
compare_url = 'https://www.wine-advantage.com/compare/Burgundy,2018/1'
compare_response = requests.get(compare_url)
data = json.loads(compare_response.text)
compare_products = []
for product in data['products']:
    if condition:
        compare_products.append(product)

# 输出结果
for product in compare_products:
    print(f'{product["name"]} - ${product["price"]}')
```
以上代码可查询 2018 年份的勃艮第葡萄酒，并输出不同葡萄酒的价格进行比较。
