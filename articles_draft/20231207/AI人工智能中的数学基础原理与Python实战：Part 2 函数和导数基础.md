## 3. 具体代码实例和详细解释说明

### 3.1 函数的具体代码实例

在Python中，我们可以使用以下代码来定义一个函数：

```python
def my_function(x):
    return x * x
```

在这个例子中，我们定义了一个名为`my_function`的函数，它接受一个参数`x`，并返回`x`的平方。

```python
import numpy as np

def my_function(x):
    return x * x

```

在这个例子中，我们定义了一个名为`my_function`的函数，它接受一个参数`x`，并返回`x`的平方。我们还定义了一个名为`my_function_derivative`的函数，它接受一个参数`x`，并返回`x`的二倍。

```python
import numpy as np
```
