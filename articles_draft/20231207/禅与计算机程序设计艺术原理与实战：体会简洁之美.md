3. 贪心算法：贪心算法是一种非常重要的算法思想，它可以帮助我们解决许多最优化问题。贪心算法的基本思想是在每个步骤中选择当前最佳的解决方案，然后将其作为下一个步骤的起点。

    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
