## 3.1 概率论
### 3.1.1 事件、样本空间、概率空间
事件是一个可能发生或不发生的结果，样本空间是所有可能结果的集合，概率空间是一个包含所有可能事件的集合。

### 3.1.2 独立性
两个事件之间的独立性，表示它们发生的概率不受对方发生的影响。

### 3.1.3 条件概率
条件概率是一个事件发生的概率，给定另一个事件已经发生。

### 3.1.4 贝叶斯定理
贝叶斯定理是一种用于计算条件概率的公式，它可以帮助我们更好地理解事件之间的关系。

## 3.2 统计学
### 3.2.1 参数估计
参数估计是一种用于估计参数值的方法，它可以帮助我们理解数据的特点和规律。

### 3.2.2 假设检验
假设检验是一种用于验证假设的方法，它可以帮助我们判断数据是否符合预期。

### 3.2.3 方差分析
方差分析是一种用于分析多个样本之间差异的方法，它可以帮助我们理解数据的特点和规律。
