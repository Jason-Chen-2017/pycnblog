```python
def lcs(X, Y):
    m = len(X)
    n = len(Y)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1):
        for j in range(n + 1):
            if i == 0 or j == 0:
                dp[i][j] = 0
            elif X[i - 1] == Y[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    return dp[m][n]
```

```python
import threading

def worker(tasks, results):
    while tasks:
        task = tasks.pop()
        result = task()
        results.append(result)

def calculate(x):
    return x * x

if __name__ == '__main__':
    tasks = [calculate(i) for i in range(100)]
    results = []
    threads = []
    for _ in range(4):
        t = threading.Thread(target=worker, args=(tasks, results))
        t.start()
        threads.append(t)
    for t in threads:
        t.join()
    print(results)
```

在上述代码中，我们使用多线程技术来解决并行计算问题。我们首先定义了一个worker函数，用于处理任务。然后，我们定义了一个calculate函数，用于计算一个数的平方。最后，我们创建了四个线程，并将任务分配给每个线程。每个线程都会调用worker函数来处理任务，并将结果存储到结果列表中。最后，我们等待所有线程完成后，打印出结果列表。
