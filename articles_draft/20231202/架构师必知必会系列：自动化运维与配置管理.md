### 1.4.1 自动化运维的具体代码实例

以下是一个简单的自动化运维代码实例：

```python
import time
from watchdog.observers import Observer
from watchdog.events import FileSystemEventHandler

class MyHandler(FileSystemEventHandler):
    def on_modified(self, event):
        print(f"File {event.src_path} has been modified")

if __name__ == "__main__":
    event_handler = MyHandler()
    observer = Observer()
    observer.schedule(event_handler, path='/path/to/your/directory', recursive=True)
    observer.start()

    try:
        while True:
            time.sleep(1)
    except KeyboardInterrupt:
        observer.stop()

    observer.join()
```

在这个代码实例中，我们使用了watchdog库来监控文件系统的状态和性能指标。当文件系统中的文件被修改时，我们的监控程序会触发on_modified方法，从而实现自动化运维工作流程。

### 1.4.2 配置管理的具体代码实例

以下是一个简单的配置管理代码实例：

```python
import json

class ConfigManager:
    def __init__(self, config_file):
        self.config_file = config_file
        self.config = None

    def load_config(self):
        with open(self.config_file, 'r') as f:
            self.config = json.load(f)

    def save_config(self):
        with open(self.config_file, 'w') as f:
            json.dump(self.config, f)

    def get_config(self, key):
        return self.config.get(key)

    def set_config(self, key, value):
        self.config[key] = value

if __name__ == "__main__":
    config_manager = ConfigManager('/path/to/your/config/file')
    config_manager.load_config()

    # 获取配置
    config = config_manager.get_config('key')
    print(config)

    # 设置配置
    config_manager.set_config('key', 'value')
    config = config_manager.get_config('key')
    print(config)

    # 保存配置
    config_manager.save_config()
```

在这个代码实例中，我们使用了json库来管理系统的配置。当我们需要获取或设置配置时，我们可以使用get_config和set_config方法来实现配置管理。
