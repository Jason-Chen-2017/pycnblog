# 2 核心概念与联系

在开发移动应用之前，我们需要了解一些核心概念，包括移动应用的类型、平台、开发工具和技术。

## 2.1 移动应用的类型

移动应用可以分为两类：原生应用和跨平台应用。原生应用是针对特定平台（如iOS或Android）开发的应用，而跨平台应用则可以在多个平台上运行。根据需要，我们可以选择开发原生应用或跨平台应用。

## 2.2 移动应用的平台

移动应用的平台主要包括iOS和Android。iOS平台是苹果公司开发的操作系统，主要用于苹果公司的设备，如iPhone和iPad。Android平台是谷歌开发的操作系统，主要用于各种智能手机和平板电脑。根据需要，我们可以选择开发iOS平台的应用、Android平台的应用或者同时开发两者。

## 2.3 移动应用的开发工具和技术

移动应用的开发工具和技术主要包括编程语言、开发框架和开发平台。例如，iOS平台的应用通常使用Objective-C或Swift作为编程语言，而Android平台的应用则使用Java或Kotlin。此外，还有许多开发框架和平台可以帮助我们更快地开发移动应用，如React Native、Flutter等。

## 4.1 案例介绍

我们将开发一个简单的计算器应用，该应用可以在iOS和Android平台上运行。我们将使用Swift编程语言开发iOS平台的应用，并使用Java编程语言开发Android平台的应用。

## 4.2 Swift代码实例

以下是iOS平台的计算器应用的Swift代码实例：

```swift
import UIKit

class ViewController: UIViewController {
    var resultLabel: UILabel!
    var result: Double = 0.0

    override func viewDidLoad() {
        super.viewDidLoad()
        setupUI()
    }

    func setupUI() {
        resultLabel = UILabel(frame: CGRect(x: 0, y: 0, width: view.frame.width, height: view.frame.height))
        view.addSubview(resultLabel)
    }

    func buttonPressed(button: UIButton) {
        if button.titleLabel?.text == "=" {
            result = round(result * 100) / 100
            resultLabel.text = String(result)
        } else {
            result *= 10
            resultLabel.text = String(button.titleLabel?.text ?? "")
        }
    }
}
```

## 4.3 Java代码实例

以下是Android平台的计算器应用的Java代码实例：

```java
import android.os.Bundle;
import android.view.View;
import android.widget.Button;
import android.widget.EditText;
import android.widget.TextView;

public class MainActivity extends AppCompatActivity {
    private EditText number1EditText;
    private EditText number2EditText;
    private TextView resultTextView;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        number1EditText = findViewById(R.id.number1EditText);
        number2EditText = findViewById(R.id.number2EditText);
        resultTextView = findViewById(R.id.resultTextView);
    }

    public void onButtonPressed(View view) {
        String operation = ((Button) view).getText().toString();
        double number1 = Double.parseDouble(number1EditText.getText().toString());
        double number2 = Double.parseDouble(number2EditText.getText().toString());

        if (operation.equals("=")) {
            double result = 0.0;
            if (number1 != 0 && number2 != 0) {
                switch (operation) {
                    case "+":
                        result = number1 + number2;
                        break;
                    case "-":
                        result = number1 - number2;
                        break;
                    case "*":
                        result = number1 * number2;
                        break;
                    case "/":
                        result = number1 / number2;
                        break;
                }
            }
            resultTextView.setText(String.valueOf(result));
        } else {
            number1EditText.setText(number1 + operation);
        }
    }
}
```
