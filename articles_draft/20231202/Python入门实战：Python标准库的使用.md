1.1.3 快速发展阶段（2000年-2010年）：随着Python的发展，它的社区也逐渐成熟，许多第三方库和框架被开发出来，为Python提供了更多的功能。在这个阶段，Python成为一种非常受欢迎的编程语言，被广泛应用于各种领域。

1.1.4 成为主流语言阶段（2010年-至今）：到现在为止，Python已经成为一种主流的编程语言，被广泛应用于各种领域，如Web开发、数据分析、机器学习、人工智能等。Python的社区也不断增长，许多新的库和框架被开发出来，为Python提供了更多的功能。

### 3.1 文件操作
```python
import os

# 打开文件
file = os.open("test.txt", os.O_RDWR | os.O_CREAT)

# 读取文件内容
content = os.read(file, 1024)

# 写入文件内容
os.write(file, "Hello, World!".encode())

# 关闭文件
os.close(file)
```
解释说明：

- 使用os.open()函数打开文件，并获取文件描述符。
- 使用os.read()函数读取文件内容。
- 使用os.write()函数写入文件内容。
- 使用os.close()函数关闭文件。

### 3.2 网络编程
```python
import socket

# 创建socket对象
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# 连接服务器
sock.connect(("localhost", 8080))

# 发送请求
sock.send("GET / HTTP/1.1\r\nHost: localhost\r\n\r\n".encode())

# 接收响应
response = sock.recv(1024)

# 关闭连接
sock.close()
```
解释说明：

- 使用socket.socket()函数创建socket对象。
- 使用socket.connect()函数连接服务器。
- 使用socket.send()函数发送请求。
- 使用socket.recv()函数接收响应。
- 使用socket.close()函数关闭连接。
