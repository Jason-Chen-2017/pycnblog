### 4.2. 应用实例分析

以预测某种疾病的发病率为例。首先，使用 Python 3、NumPy 和 Pandas 等库对数据进行预处理。然后，使用机器学习库中的决策树算法对数据进行建模，并训练模型。最后，使用模型对历史数据进行预测，得出对应的发病率预测值。

### 4.3. 核心代码实现

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import accuracy_score

# 读取数据
data = pd.read_csv('health_data.csv')

# 数据预处理
# 去除数据中存在的异常值、缺失值和重复值
# 数据均值设为 0，标准差设为 1
data_preprocessed = data.dropna().drop(columns=['age', 'gender']).drop(axis=1).fillna(0).fill(1)

# 特征选择
# 选择出年龄和性别作为特征
selected_features = ['age', 'gender']

# 构建并训练决策树模型
model = DecisionTreeClassifier()
model.fit(data_preprocessed[selected_features], data_preprocessed['target'])

# 预测发病率
predictions = model.predict(data_preprocessed[selected_features])

# 输出预测结果
print('预测发病率：', predictions)

# 比较实际结果与预测结果
accuracy = accuracy_score(data_preprocessed['target'], predictions)
print('Accuracy：', accuracy)
```
