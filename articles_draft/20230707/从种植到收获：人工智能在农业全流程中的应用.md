2.3. 代码实例和解释说明

以图像识别技术为例，使用 OpenCV 库实现一个简单的光谱图像分类任务。首先安装 OpenCV 库，然后编写代码实现图像分类：

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('iris.jpg')

# 特征提取
特征 = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 识别训练集图片
train_labels = [1, 0, 0, 1, 0, 1, 1, 0, 0, 1]
train_images = [img, img]

# 特征匹配
match_scores = []
for label in train_labels:
    match_scores.append(cv2.matchTemplate(train_images, (img, label), cv2.TM_CCOEFF_NORMED))

# 最大匹配分数
max_score = max(match_scores)
index = np.argmax(match_scores)

# 输出结果
print('Label:', train_labels[index])
```

上述代码使用 OpenCV 库读取一株郁金香花的图像，并使用灰度化处理后提取特征。然后使用训练集数据训练一个支持向量机（SVM）模型，最后使用测试集数据匹配图片，输出结果。
