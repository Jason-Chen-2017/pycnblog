# 1.简介
  

建仓基金，或者称为宽基金，指的是具有高度收益回报能力、覆盖全市场的多种风险资产的一种固定比例股票型基金产品。其特点在于通过长期投资者持有的股票而获得高收益，适用于广阔的投资领域。建仓基金主要包括QDII（Qualified Deposit-Insurance Institutions）产品、ETF基金、封闭式基金等。QDII产品可分为正规券商销售的公募基金和私募基金；ETF基金是指由全球范围内的企业共同投资组成的公募基金；封闭式基金是指投资者将所持有的特定标的（如股票、债券等）固定比例不变地放在基金账户中，按月定期支付本息。

宽基金和其他投资产品最大的不同在于，它不需要像传统投资产品那样设定初始投资额、管理费用等门槛。一般来说，宽基金的起步金额较低，但是需要为基金经理提供有关产品概况、优势、服务等详细信息。当初选取建仓基金时，应充分考虑投资风险和预期收益的匹配程度，根据自己的条件进行筛选。建仓基金的获利模式通常是每年或每季度给出固定的收益，没有分红或分润。因此，要注意对收益的把握，及时调整仓位。

本文旨在讨论不同风险类型的建仓基金以及如何选择它们的特色。其中包括黄金ETF基金、房地产信托基金、纯债基金、货币市场基金、短期国际商品基金等。我们将首先介绍宽基金的构成和运作方式，然后系统性地介绍它们各自的优缺点以及适合哪些投资目标和风险类型。最后，我们结合具体操作实例，分析不同风险类型的建仓基金的优势所在。

# 2.宽基金的构成与运作方式
## 2.1 宽基金的组成
宽基金通常由以下四个部分组成：

1. 投资标的：指建仓基金所投资的证券、贵金属、黄金等投资品种。

2. 投资管理公司：负责运营基金并分配基金份额。

3. 投资顾问：为客户提供专业化的咨询、指导、研究服务。

4. 投资交易所/基金公司：负责完成所有实际交易，保证基金投资标的的现货价格稳定。

对于建仓基金，一般都是以股票、债券、黄金等为主，因为这些资产都是具有超强回报的资产类别。他们可以提供长期、稳健、无风险收益，有助于满足投资者对于周期性盈利和收益的需求。同时，由于公司、交易所、顾问等方面都有专业人员的参与，可以保障投资安全、风险控制和投资者的满意度。

## 2.2 宽基金的运作方式
宽基金的运作方式有两种：

1. 普通建仓基金：一般指没有私募基金背景，仅靠证券公司、交易所、基金公司三方参与构建的基金。这种基金由基金经理来管理，有专业人士参与指导管理。目前国内最知名的普通建仓基金为：
   - 中证白酒业资源投资基金
   - 双汇投资基金
   - 大连商品交易所股票基金
   - 上海建工基金

2. QDII建仓基金：也称为正规券商销售的公募基金。这是一类特殊的建仓基金，其中投资资产主要来源于中国证监会批准的大型券商集团。根据规则，购买正规券商发行的股票，而非个人投资者发行的股票。该类基金产品只能以派生权的方式投资。目前国内最著名的QDII基金产品有：
   - 深证信托基金
   - 华泰柏瑞信托基金
   - 招商基金
   - 兴全所投资基金
   - 香港中小盘混合基金
   
# 3.不同的建仓基金及其特色
接下来我们来介绍不同风险类型的建仓基金，以及它们的特点。

## 3.1 黄金ETF基金
黄金ETF基金（Gold ETF Fund），又称黄金指数型基金，是指投资者将其持有的黄金指数（GLD、USD/GOLD）等金属作为投资标的，通过套利机会获得一定程度上折溢价的一种固定收益型基金。由于黄金是一个持续增长的资产，而且其吸引力和便利的属性使得它成为投资者追求的目标。所以，黄金ETF基金可以实现长期、稳健、无风险的收益。

优势：

1. 收益稳定：由于资产的稳定性，黄金ETF基金的每年都会给出相似的收益率。

2. 投资门槛低：无需购买黄金的数量，只需持有价值或估值相同的黄金指数即可。

3. 稀缺性：黄金是世界上最珍贵的宝石之一，并且黄金ETF基金也是全球范围内最具增长潜力的基金。

4. 不用担心黄金的过度配置：基于黄金ETF基金的投资不会导致黄金过度配置，因而不会造成巨大的过度储备。

5. 有利于经济运行：黄金ETF基金为投资者提供了低估值黄金的机会，可促进经济发展。

缺陷：

1. 流动性差：由于黄金与人民币之间的兑换价差异和交易时间的不确定性，黄金ETF基金无法做到100%保证金字塔。

2. 管理费用高：因为主要的投资标的都是黄金，而金属这种稀缺性质的资源，又不能直接融入黄金的持有，所以对它们的定期管理和交易结算费用较高。

适用对象：

1. 短线投资：主要投资黄金ETF基金的策略是为了寻找长期资产增值空间，而非短期利润上升。

2. 专业投资者：金属投资领域的专业投资者也可以利用黄金ETF基金来进行投资。

## 3.2 房地产信托基金
房地产信托基金（Real Estate Trust Fund，RETF），又称产信基金或产权基金，是指依托产权所受到信托的房地产资产和债券作为投资标的的一种股票型固定收益基金。房地产信托基金是现代资本市场上最活跃的资产类别之一。

优势：

1. 透明度高：房地产信托基金中的大量资产均受到产权控制，有利于客户查看和确认投资资产的真实性。

2. 投资便利：房地产信托基金的交易平台界面简洁，资金转移速度快，不受风控压力影响，方便客户进行快速布局。

3. 高收益率：房地产信托基金相对于其他金融产品更适合长期投资，能够提供相当于投资回报率百分之二十左右的收益。

4. 可控性强：房地产信托基金具有高度的税后收益率，投资者无需承担多余的费用，而且资金流向简单，容易投资管理和定期结算。

5. 稳健性高：房地产信托基金的定期理财服务体系强制执行信托协议，有利于确保房地产投资项目的稳定增值。

缺陷：

1. 风险高：房地产信托基金的资金来源来自于信托公司的运营风险，信托公司为客户提供高风险的担保和赎楼保护。

2. 部分波动性：房地产信托基金虽然面临着一定程度的波动，但仍然存在盈亏浮动甚至损失的可能性。

3. 投资周期长：房地产信托基金的投资周期较长，投资金额越大，收益率就越低。

适用对象：

1. 长期投资：房地产信托基金适合长期持有、稳健投资。

2. 高风险投资者：房地产信托基金的风险较高，投资者有较强的理财意识和保守性偏好，需要较高的投资知识和专业能力。

## 3.3 纯债基金
纯债基金（Bonds Fund），又称债券型基金，是指将一种固定债券作为投资标的，通过多元化持仓机制来获取较高的收益率的一种固定收益型基金。纯债基金有两种主要的代表：传统债券型基金和创新债券型基金。传统债券型基金以长期负债的债券作为投资标的，经历过经典的分级再平衡机制，具有良好的流动性。创新债券型基金则主要以较短期内迅速走弱的债券作为投资标的，可以通过创新的结构和理念，实现较高的回报率。

优势：

1. 无风险收益：纯债基金的资产来源主要来自传统的债券市场，有着极高的回报率，往往可以长期保持较高的净值。

2. 投资灵活性高：纯债基金的投资范围非常广泛，除了债券类资产，还可以包括货币市场基金、股票市场基金等其他资产的组合投资。

3. 杠杆率高：纯债基金的杠杆率高，可以满足低认购成本，投资风险较低的投资者。

4. 提供多样化的收益产品：包括封闭式的信用基金和开放式的资管计划，以满足不同投资者的需要。

缺陷：

1. 市场噪音：纯债基金的投资标的以债券为主，因此具有很高的市场分散性，会带来一些噪声。

2. 套牢风险：纯债基金的债务结构有利于长期投资，但也使得它们容易被套牢。

适用对象：

1. 对债券持有比较看重的人群：债券型基金对债券市场拥有独特的感情和兴趣，能够获得持有债券的高收益。

2. 希望提升信用级别的人群：债券型基金可以提供一个良好的基准，帮助投资者了解信用水平，做出理性的投资决策。

## 3.4 货币市场基金
货币市场基金（Currency Market Fund，CMF）是指投资者将多种货币市场（包括股票、债券、期货等）的资产组合作为投资标的，通过套利机会获得一定程度上折溢价的一种固定收益型基金。货币市场基金以货币市场为基础，提供多种风险准备，形成一定的避险作用，所以市场对它的抗风险能力很有信心。

优势：

1. 回报率高：货币市场基金的回报率略高于其他各类基金，但是仍然能达到惊人的回报水平。

2. 稳定性高：货币市场基金的资产来源来自银行间市场，具有较高的稳定性，不会出现大幅波动。

3. 投资频道多样：货币市场基金的投资领域包含多种不同形式的货币市场，包括股票市场、债券市场、期货市场等。

4. 发行效率高：货币市场基金的发行过程十分简单，只要投资者有相关的认识和准备就可以很快上手。

缺陷：

1. 低杠杆率：货币市场基金的杠杆率是不足以吸引很多投资者的，只有少部分投资者才会选择加入。

2. 管理费用高：货币市场基金的管理费用会很高，主要包括销售佣金、管理费、托管费等。

3. 投资周期长：货币市场基金的投资周期较长，需要投资者花费较长的时间和精力才能实现回报。

适用对象：

1. 资金量相对较大的投资者：如果您的资金量相对较大，可以尝试选择货币市场基金作为投资的首选。

2. 需要持续发展的人群：货币市场基金具有很高的持续性，能够长期保持其净值不变，适合持续增值的投资者。

## 3.5 短期国际商品基金
短期国际商品基金（Short Interest Stock Bond Fund，SISMBF），又称短债基金，是指短期间的股票国债（STIB）作为投资标的的一种固定收益型基金。短期国际商品基金的优势在于其适合发行短期国际商品，比如短期国债和货币市场基金，也可以通过套利的方式获得一定程度上的折溢价，而且易于管理，不需要理财产品。

优势：

1. 回报率高：短期国际商品基金的回报率超过了其他任何一种基金，而且可以远远超过黄金ETF基金。

2. 投资阶段性回报：短期国际商品基金的投资环境比较灵活，适合在投资前期不断积累信用，避免盲目投资。

3. 成本相对较低：短期国际商品基金的发行成本是有限的，一般在几千元到几万元之间。

4. 自由度高：短期国际商品基金的投资产品有各种形式，有利于客户选择投资对象，选择最适合自己的投资品种。

缺陷：

1. 回撤风险：短期国际商品基金的回撤风险较大，可能由于标的变动而受到损害。

2. 投资周期长：短期国际商品基金的投资周期比较长，投资者需要花费较长的时间才能实现回报。

3. 不能完全抗风险：短期国际商品基金的投资产品不是全品种投资，可能会损失部分投资收益，但仍然具有一定的应对风险的能力。

适用对象：

1. 对股市、债市波动比较敏感的人群：短期国际商品基金与其他基金不同，它们的投资标的不是股票、债券等传统的商品市场，而是短期国际商品，具有灵活的投资空间。

2. 具有创新精神和热情的人群：短期国际商品基金的优势在于可以随时改变产品投资方向，适合具有创新精神和热情的人群。
