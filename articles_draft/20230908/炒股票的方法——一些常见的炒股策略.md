## 2.基本概念术语说明
### 2.1.概念定义
- 日内均线（Day’s High/Low Line）：又称KD指标或DMI指标，它是一个用来跟踪价格走势的技术指标。用一根线段标记当前的最高价和最低价的平均值。当价格出现突破或者跌破日内均线时，可能表明将要开始买进或者卖出。日内均线一般是5日、10日、20日或者更长时间的一个移动均线。日内均线指标已经成为很多股票分析工具中的必备技术指标，广泛用于大盘走势、个股走势的研究、预测以及交易决策。
- 布林带（Bollinger Bands）：也叫做中轨线、布林通道，是由上轨线、中轨线和下轨线组成的股价波动范围。一般来说，上轨线上方为正偏离值（A），下轨线下方为负偏离值（B）。当收盘价位于上下轨线之间时，股价处于中性区域，是最安全的状态。当股价触碰下轨线时，表示股价处于下行趋势中，此时可能触发卖出信号；当股价触碰上轨线时，表示股价处于上行趋势中，此时可能触发买入信号。中轨线则处于上轨线与下轨线之间的位置。
- 布林带震荡（Bollinger Band® Breakout/Bearish Trend）：描述了上涨的股价从“中轨线”突破“上轨线”之后，出现明显的回调趋势，并持续到“下轨线”反弹回原始位置。
- 上涨支撑位（Reversal Support Level）：即上轨线的下方，是持股者认为投资者不会再回落的支撑水平，这时候如果股价继续上涨则容易被短期大跌打破。
- 下跌阻力位（Reversal Resistance Level）：即下轨线的上方，是持股者认为投资者难以站稳的回撤水平。这个时候，如果股价继续下跌，则可以承受较大的风险，但股价超过这一水平则很可能会进入新的周期。
- 多空头寸比例（Long/Short Hedge Ratio）：衡量的是一个证券投资组合中，在不同时间和市场条件下的股票占净持有数量的比例。多头头寸比例越高，则意味着持有更多的多头头寸（买入量）。反之亦然，相对于空头持仓，短期股票回报比较高。
- 对冲基准（Hedge Fund/Index Fund）：一种特殊的基金产品，它的投资方式源自于通过跟踪指数或者基准来投资。但是其目的是通过调整仓位，从而获得超额收益。
- 均衡仓位法（Equal Weighted Positioning Method）：一种系统化的仓位管理方法，其原理是在一定时间段内，将所有账户权益分摊给各个股票所占的份额，并确保所有投资品种的权益始终相同。
- 封闭市值法（Closed-Price Valuation Method）：一种以实时的股价计算市值的方法。根据公司最新公布的财务数据、公司业绩评估和预测数据以及公司业务模式，可以确定一个公司的价值。
- 单边布局（Conductivities）：也称趋势牛熊运动、趋势性强弱指标。它的主要作用是测量股票价格在过去一段时间内的牛和熊气势，并根据这些气势的大小来判断市场趋势的方向和强度。比如说，如果股票短期内表现出较强的趋势性，那就可以考虑买入。反之，如果短期内出现强烈的反转（如向下反转），则可以考虑卖出。
- 比率指标（Ratio Indicator）：指标是由两个相互竞争的价格指标决定的，包括价比指标和量比指标。其中，价比指标比较的是同类别商品的价格差异，而量比指标比较的是同类别商品的流动性状况。
- 均值回复策略（Mean Reversion Strategy）：适合追踪短期波动变化的投资策略。它利用股票的历史波动和趋势，找寻其转折点，然后按顺序买入卖出，以实现自我调控。
- 空头结构（Put Option Strategies）：是一种使用虚值期权作为投资标的进行期权交易的投资策略。通常情况下，空头结构采取的是“长期空头”，即买入虚值期权，等待价差实现并获利。
- 大趋势策略（Big Swings Strategy）：是一种非常复杂的策略体系，可以预测股票未来的走势，并且调整仓位，抓住主要的趋势机会。这种策略侧重长期预测和仓位管理，在股市短线非常高的波动情况下也能够生存。
- 市场情绪指标（Sentiment Indicators）：是指用于监测、分析和预测股市的情绪信息的数据。包括：负面情绪指标（如“恐慌情绪指数”）、正面情绪指标（如“投资情绪指数”）、投机情绪指标（如“抛售转型情绪指数”）、主动性指标（如“涉幅变动指数”）等。它们可以通过跟踪和分析不同的技术指标、宏观经济数据以及公众舆论等，来帮助投资者掌握市场的动态。
- 期权定价模型（Option Pricing Models）：采用概率密度函数来表示期权价格的变动过程。根据参数的不同，可以分为几种类型，包括：
    - 求得期权价格的矩法（Moment-matching method）：此方法通过求解期权价格的矩形式，得到期权的价格模型。
    - 蒙特卡洛模拟法（Monte Carlo simulation method）：是随机模拟市场中每个可行的价格，然后统计出现的结果，得到概率分布，以此估计期权价格。
    - 二项期权模型（Binomial option model）：由两个互斥事件构成，分别对应期权的两种状态——看跌期权和看涨期权。在指定期间内，可以计算出期权的发生次数，通过数学推导，可以得出期权的价格。
    - 隐含波动率模型（Implied Volatility Model）：是根据理论计算期权价格模型的一种方法。
    - 模拟神经网络模型（Neural Network Simulation Model）：基于神经网络的机器学习方法，可以对多个变量进行预测，用来确定期权的价格。

以上，是本文涉及到的一些概念和术语的定义。
## 3.核心算法原理和具体操作步骤以及数学公式讲解
本节主要介绍一些常见的炒股票的策略方法。
### 3.1. 日内均线策略
日内均线策略（DMI）是通过跟踪一段时间内的股价平均线（日内均线）来判断股价的趋势。日内均线策略使投资者能够快速判断股价的变动情况，在出现买卖信号时作出应对。此外，由于指标的易于操作性，日内均线策略被认为是一种简单易用的技术分析工具。

日内均线策略由以下几个步骤构成：
1. 设置买卖水平：设置买卖水平对日内均线策略的胜算非常关键。通常情况下，交易双方都会配合设定一致的买入卖出价格，因此设置太高或太低的价格都无法取得好效果。设置的买入卖出价格需要与策略使用的技术指标匹配。
2. 研判趋势：通过分析日内均线，判断股价的趋势。一般来说，向上涨的股价，日内均线一般比前一日高一些，或者持平；向下跌的股价，日内均线一般比前一日低一些，或者持平。
3. 扫描指标图表：检查是否存在底背驰、顶背驰和实体背驰。如果存在这三种情况中的任何一种，都应作出应对。
4. 交易信号确认：交易信号确认阶段将对买卖信号进行最后确认。一般情况下，技术分析师将根据日内均线的走势判断股票的趋势，然后决定买还是卖。如果出现买卖信号，双方都应根据实际情况协商，然后执行相应操作。

#### （1）日内均线买入策略
日内均线买入策略是指利用日内均线的强弱来决定买入信号的强弱，它的基本思路是：如果股价突破日内均线，则买入；如果股价跌破日内均线，则卖出。设置买卖价格一般为均线后面的支撑位。

1. 设置买卖价格：
   - 开盘前的10秒钟：使用ATR技术分析的波幅值加上股价往上的7倍，作为日内均线的支撑位。
   - 白天：一般是日线级别。
   - 夜盘：使用日线级别的开盘价，再减去15%至25%的ATR作为日内均线的支撑位。

2. 判断趋势：
   - 如果日内均线上穿5日均线，买入信号，股价处于强势阶段；
   - 如果日内均线上穿10日均线，则重复前面的步骤；
   - 如果日内均线上穿20日均线，则重复前面的步骤；
   - 如果日内均线下穿5日均线，卖出信号，股价处于弱势阶段；
   - 如果日内均线下穿10日均线，则重复前面的步骤；
   - 如果日内均线下穿20日均线，则重复前面的步骤；

3. 检查买卖信号：
   - 如果股价超过支撑位，买入信号送出；
   - 如果股价低于支撑位，卖出信号送出。

#### （2）日内均线卖出策略
日内均线卖出策略是指利用日内均线的强弱来决定卖出信号的强弱，它的基本思路是：如果股价跌破日内均线，则卖出；如果股价突破日内均线，则买入。设置买卖价格一般为均线前面的阻力位。

1. 设置买卖价格：
   - 开盘前的10秒钟：使用ATR技术分析的波幅值减去股价往下的7倍，作为日内均线的阻力位。
   - 白天：一般是日线级别。
   - 夜盘：使用日线级别的开盘价，再加上15%至25%的ATR作为日内均线的阻力位。

2. 判断趋势：
   - 如果日内均线下穿5日均线，卖出信号，股价处于强势阶段；
   - 如果日内均线下穿10日均线，则重复前面的步骤；
   - 如果日内均线下穿20日均线，则重复前面的步骤；
   - 如果日内均线上穿5日均线，买入信号，股价处于弱势阶段；
   - 如果日内均线上穿10日均线，则重复前面的步骤；
   - 如果日内均线上穿20日均线，则重复前面的步骤；

3. 检查买卖信号：
   - 如果股价低于阻力位，卖出信号送出；
   - 如果股价超过阻力位，买入信号送出。

#### （3）日内均线价格震荡策略
日内均线价格震荡策略是指利用日内均线抄底后的价格变化情况，来判断股票价格的短期震荡情况。根据日内均线的形态，有如下四种震荡类型：

1. 上涨趋势底部反转型：日内均线向上突破5日线，5日线上影线向上突破20日线；
2. 下跌趋势顶部反转型：日内均线向下跌破5日线，5日线下影线向下跌破20日线；
3. 弱势上升态：日内均线落在20日线以上，上影线在20日线以下；
4. 强势下降态：日内均线落在20日线以下，下影线在20日线以上。

对不同的震荡类型采取不同的操作策略：

1. 上涨趋势底部反转型：此时日内均线下穿5日线，则判定股票价格底部反转，可能出现买卖机会，应根据时机及自身情况决定是否执行；
2. 下跌趋势顶部反转型：此时日内均线上穿5日线，则判定股票价格顶部反转，可能出现买卖机会，应根据时机及自身情况决定是否执行；
3. 弱势上升态：此时日内均线位于20日线以下，向上突破，存在反抽查机会，应发出警告信号，及时停止交易；
4. 强势下降态：此时日内均线位于20日线以上，向下突破，存在爬坡抄底机会，应该首先提前抄底。

#### （4）日内均线套利策略
日内均线套利策略是指利用日内均线的强弱来进行套利，它的基本思路是：在股价高位时将空头开仓，在股价低位时将多头开仓。

1. 设置套利价格：设置套利价格是日内均线套利策略的关键。一般来说，套利价格的设定需要根据自己的心理因素进行调整。例如，如果投资者认为股价将向上推升，则可以选择比均线低一些的价格，如果投资者认为股价将向下推降，则可以选择比均线高一些的价格。

2. 操作步骤：
   - 当日内均线上穿5日均线时，多头开仓；
   - 当日内均线下穿5日均线时，空头开仓。

3. 退出条件：
   - 当股价触碰日内均线时，自动平仓。

#### （5）日内均线策略注意事项
1. 日内均线策略是一个快速、方便、有效的技术分析工具。同时，它还具有较高的操作效率，不需要耗费太多的人力物力，是个人投资者及专业投资者经常使用的一种技术分析工具。
2. 使用日内均线策略需要考虑不同的市场因素，特别是商品期货市场，存在不同类型的均线，需要区别对待。
3. 日内均线策略也存在一定的滞后性，因为指标的实时性并不好。
4. 在进行日内均线策略交易时，记得及时关注交易结果，保持耐心等待指标变化后的交易信号。
