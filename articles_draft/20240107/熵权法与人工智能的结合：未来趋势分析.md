## 4.1 代码实例
以下是一个简单的熵权法代码实例，它用于计算两种资源的权重，并根据权重进行资源分配。

```python
import math

# 资源和状态的定义
resource1 = [0.4, 0.6]
resource2 = [0.5, 0.5]

# 熵值的计算
def entropy(prob):
    return -sum(p * math.log2(p) for p in prob)

# 权重的计算
def weight(entropy_list):
    total_entropy = sum(entropy_list)
    return [e / total_entropy for e in entropy_list]

# 资源分配
def allocate_resource(resource, weight):
    return [r * w for r, w in zip(resource, weight)]

# 计算熵值
entropy1 = entropy(resource1)
entropy2 = entropy(resource2)

# 计算权重
weight1 = weight([entropy1])
weight2 = weight([entropy2])

# 资源分配
allocated_resource1 = allocate_resource(resource1, weight1)
allocated_resource2 = allocate_resource(resource2, weight2)

print("资源1的分配结果:", allocated_resource1)
print("资源2的分配结果:", allocated_resource2)
```
