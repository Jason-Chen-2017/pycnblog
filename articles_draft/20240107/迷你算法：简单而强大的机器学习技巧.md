# 2.核心概念与联系

## 2.1 机器学习的基本概念

在开始学习迷你算法之前，我们需要了解一些基本的机器学习概念。

### 2.1.1 训练集和测试集

训练集是用于训练机器学习模型的数据集，它包含了输入和输出的对应关系。测试集是用于评估模型性能的数据集，它不被用于训练模型。

### 2.1.2 特征和标签

特征是用于描述数据的属性，而标签是我们希望机器学习模型预测的值。

### 2.1.3 过拟合和欠拟合

过拟合是指机器学习模型在训练集上表现良好，但在测试集上表现不佳的现象。欠拟合是指机器学习模型在训练集和测试集上表现都不好的现象。

### 2.2.1 简化的算法

### 2.2.2 自动调参

## 3.1 迷你决策树算法的核心原理

迷你决策树算法是一种基于决策树的机器学习算法，它通过构建简化的决策树来进行分类和回归任务。迷你决策树算法的核心原理是通过递归地划分数据集，以找到最佳的特征来进行分类或回归。

### 3.1.1 信息增益和信息熵

信息熵是用于衡量数据集纯度的指标，它反映了数据集中各个类别的不确定度。信息增益是用于衡量特征对于分类任务的贡献的指标，它反映了特征能够减少数据集纯度的程度。

### 3.1.2 递归地划分数据集

迷你决策树算法通过递归地划分数据集，以找到最佳的特征来进行分类或回归。在每一次划分中，算法会计算每个特征的信息增益，并选择信息增益最大的特征进行划分。

### 3.1.3 停止条件

在递归地划分数据集的过程中，需要设置停止条件，以避免过拟合。常见的停止条件包括最小样本数、最大深度和最小信息增益等。

## 3.2 迷你决策树算法的具体操作步骤

以下是迷你决策树算法的具体操作步骤：

1. 从训练集中随机选择一个特征，作为当前节点的特征。
2. 计算当前节点的信息增益，以评估特征的贡献。
3. 选择信息增益最大的特征，作为当前节点的分割特征。
4. 将数据集按照当前节点的分割特征进行划分，得到左右两个子节点。
5. 递归地对左右两个子节点进行上述操作，直到满足停止条件。
6. 对每个叶子节点进行回归或分类，得到最终的预测结果。

entropy 函数是用于计算信息熵的函数，它可以通过以下公式定义：

$$
