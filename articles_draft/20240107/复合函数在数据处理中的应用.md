## 1.1 复合函数的定义和基本概念

复合函数是由多个函数组成的函数，它们的输出作为输入传递给下一个函数。这种组合方式可以帮助我们更好地处理和分析数据，特别是在处理复杂数据时。

复合函数的基本概念包括：

- 函数的组合：将多个函数组合在一起，形成一个新的函数。
- 函数的嵌套：将一个函数作为另一个函数的输入。
- 函数的组合规则：定义了如何将多个函数组合在一起。

假设我们有一个包含缺失值的数据集，我们可以使用复合函数来处理这些缺失值。

```python
import pandas as pd
import numpy as np

# 创建一个包含缺失值的数据集
data = {'name': ['Alice', 'Bob', 'Charlie', 'David'],
        'age': [25, np.nan, 30, 35],
        'gender': ['F', 'M', 'M', 'M']}
df = pd.DataFrame(data)

# 定义一个函数来处理缺失值
def handle_missing_values(df):
    df.fillna(value='Unknown', inplace=True)
    return df

# 使用复合函数处理缺失值

print(df)
```

在这个示例中，我们首先创建了一个包含缺失值的数据集。然后，我们定义了一个函数 `handle_missing_values` 来处理缺失值，该函数使用 `fillna` 方法将缺失值替换为 'Unknown'。最后，我们使用复合函数处理缺失值，并打印处理后的数据集。

## 4.2 数据转换示例

假设我们有一个包含年龄和性别的数据集，我们可以使用复合函数来将这些数据转换为新的格式。

```python
import pandas as pd
import numpy as np

# 创建一个包含年龄和性别的数据集
data = {'age': [25, 30, 35, 40],
        'gender': ['F', 'M', 'M', 'F']}
df = pd.DataFrame(data)

# 定义一个函数来将年龄转换为年龄段
def age_range(age):
    if age < 18:
        return 'Child'
    elif 18 <= age < 30:
        return 'Young Adult'
    elif 30 <= age < 60:
        return 'Adult'
    else:
        return 'Senior'

# 定义一个函数来将性别转换为性别代码
def gender_code(gender):
    if gender == 'F':
        return 'Female'
    else:
        return 'Male'

# 使用复合函数将数据转换为新的格式
df['age_range'] = df['age'].apply(age_range)
df['gender_code'] = df['gender'].apply(gender_code)
print(df)
```

在这个示例中，我们首先创建了一个包含年龄和性别的数据集。然后，我们定义了两个函数 `age_range` 和 `gender_code` 来将年龄和性别转换为新的格式。最后，我们使用复合函数将数据转换为新的格式，并打印处理后的数据集。
