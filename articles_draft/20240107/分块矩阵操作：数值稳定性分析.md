## 3.1 分块矩阵加法
分块矩阵加法是指将两个分块矩阵相加，得到一个新的分块矩阵。具体操作步骤如下：

1. 对于每个分块，分别将对应位置的两个矩阵相加。
2. 将所有分块相加后的矩阵组合成一个新的分块矩阵。

数学模型公式为：
$$
C_{ij} = A_{ij} + B_{ij}
$$
其中，$C_{ij}$ 表示新的分块矩阵的第 $i$ 行第 $j$ 列元素，$A_{ij}$ 和 $B_{ij}$ 分别表示原矩阵 $A$ 和 $B$ 的第 $i$ 行第 $j$ 列元素。

## 3.2 分块矩阵乘法
分块矩阵乘法是指将两个分块矩阵相乘，得到一个新的分块矩阵。具体操作步骤如下：

1. 对于每个分块，分别将对应位置的第一个矩阵的行与第二个矩阵的列相乘。
2. 将所有分块相乘后的矩阵组合成一个新的分块矩阵。

数学模型公式为：
$$
C_{ij} = A_{i.} \cdot B_{.j}
$$
其中，$C_{ij}$ 表示新的分块矩阵的第 $i$ 行第 $j$ 列元素，$A_{i.}$ 和 $B_{.j}$ 分别表示原矩阵 $A$ 的第 $i$ 行和 $B$ 的第 $j$ 列。

## 3.3 迭代方法
迭代方法是解线性方程组的一种常见方法，如Jacobi方法、Gauss-Seidel方法等。这些方法通常需要考虑数值稳定性问题，以确保计算结果的准确性。

### 3.3.1 Jacobi方法
Jacobi方法是一种迭代方法，通过将线性方程组拆分为若干个简单的线性方程，逐步求解。具体操作步骤如下：

1. 对于每个变量，将线性方程组拆分为若干个简单的线性方程。
2. 使用分块矩阵乘法和加法，逐步求解每个变量的值。
3. 重复步骤2，直到收敛。

数值稳定性分析：
Jacobi方法的数值稳定性主要取决于迭代方程的估计误差。如果迭代方程的估计误差较小，则算法的数值稳定性较好。

### 3.3.2 Gauss-Seidel方法
Gauss-Seidel方法是一种迭代方法，通过将线性方程组拆分为若干个简单的线性方程，逐步求解。具体操作步骤如下：

1. 对于每个变量，将线性方程组拆分为若干个简单的线性方程。
2. 使用分块矩阵乘法和加法，逐步求解每个变量的值。
3. 重复步骤2，直到收敛。

数值稳定性分析：
Gauss-Seidel方法的数值稳定性比Jacobi方法更好，因为它利用了上一次迭代的结果来更新当前变量的值。这有助于减少迭代方程的估计误差，从而提高算法的数值稳定性。

## 4.3 Jacobi方法
```python
import numpy as np

A = np.array([[4, -1], [-1, 4]])
b = np.array([1, 1])

x = np.zeros(2)

for i in range(100):
    x[0] = (b[0] - A[1, 0] * x[1]) / A[0, 0]
    x[1] = (b[1] - A[0, 1] * x[0]) / A[1, 1]

    if np.linalg.norm(A @ x - b) < 1e-10:
        break

print(x)
```
输出结果：
```
[1. 1.]
```
在这个例子中，我们使用Jacobi方法求解线性方程组 $Ax = b$。可以看到，算法的数值稳定性较好，收敛于正确的解。
