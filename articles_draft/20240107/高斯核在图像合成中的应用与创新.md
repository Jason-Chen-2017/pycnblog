## 2.1 高斯核的基本概念

高斯核（Gaussian kernel），又称高斯函数，是一种常用的连续函数，可以用于实现多种数学和统计计算。高斯核的定义如下：

$$
K(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{x^2}{2\sigma^2}}
$$

其中，$x$ 是输入变量，$\sigma$ 是标准差，用于控制高斯核的宽度和高度。

高斯核的特点：

1. 对称性：$K(-x) = K(x)$。
2. 单峰性：$K(x)$ 在 $x=0$ 处取得最大值。
3. 连续性：$K(x)$ 是一个连续的函数。

## 4.1 高斯核函数的实现

首先，我们需要实现高斯核函数。以下是 Python 代码实现：

```python
import numpy as np

def gaussian_kernel(x, y, sigma):
    return (1 / (np.sqrt(2 * np.pi * sigma ** 2) * sigma)) * np.exp(-(x ** 2 + y ** 2) / (2 * sigma ** 2))
```

在这个函数中，`x` 和 `y` 是输入变量，`sigma` 是标准差。
