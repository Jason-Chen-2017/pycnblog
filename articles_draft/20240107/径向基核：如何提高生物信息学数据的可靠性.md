# 2.核心概念与联系

## 2.1 径向基函数

径向基函数（Radial Basis Function，RBF）是一种用于解决高维问题的函数，它的定义如下：

$$
f(x) = \sum_{i=1}^{N} c_i \phi(\|x-c_i\|)
$$

其中，$c_i$ 是基函数中心，$\phi(\|x-c_i\|)$ 是基函数值，$N$ 是基函数数量。

径向基函数通常用于解决以下问题：

1. 插值问题：给定一组数据点和它们的值，找到一个函数来描述这些数据点。
2. 拟合问题：给定一组数据点和一个模型，找到一个函数来拟合这些数据点。
3. 分类问题：给定一组数据点和一组类别，找到一个函数来分类这些数据点。

## 2.2 径向基核

径向基核（Radial Basis Kernel，RBK）是一种用于计算两个数据点之间距离的函数，它的定义如下：

$$
K(x, y) = \sum_{i=1}^{N} c_i \phi(\|x-c_i\|) \phi(\|y-c_i\|)
$$

其中，$c_i$ 是基函数中心，$\phi(\|x-c_i\|)$ 是基函数值，$N$ 是基函数数量。

径向基核通常用于解决以下问题：

1. 距离度量：计算两个数据点之间的距离。
2. 相似性度量：计算两个数据点之间的相似性。
3. 支持向量机：用于解决分类和回归问题的一种机器学习算法。

def gaussian_kernel(x, y, sigma=1.0):
    """
    高斯基函数核
    """
    return np.exp(-np.linalg.norm(x - y)**2 / (2 * sigma**2))

def radial_basis_kernel(X, Y, sigma=1.0):
    """
    径向基核
    """
    K = np.zeros((len(X), len(Y)))
    for i, x in enumerate(X):
        for j, y in enumerate(Y):
            K[i, j] = gaussian_kernel(x, y, sigma)
    return K

X = np.array([[1, 2], [3, 4], [5, 6]])
Y = np.array([[7, 8], [9, 10], [11, 12]])

print(K)
