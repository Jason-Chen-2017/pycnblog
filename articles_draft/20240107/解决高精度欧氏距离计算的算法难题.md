# 2. 核心概念与联系

## 2.1 欧氏距离

欧氏距离是指在欧氏空间中，两个点之间的距离。欧氏空间是一个n维空间，其中n为整数。在欧氏空间中，两个点之间的距离可以通过以下公式计算：

$$
d(x, y) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)^2}
$$

其中，$x = (x_1, x_2, ..., x_n)$ 和 $y = (y_1, y_2, ..., y_n)$ 是欧氏空间中的两个点，$d(x, y)$ 是它们之间的欧氏距离。

## 2.2 高精度欧氏距离计算
