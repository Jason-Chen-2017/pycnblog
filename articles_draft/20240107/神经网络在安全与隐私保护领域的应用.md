## 2.1 神经网络基础概念

神经网络是一种模仿生物大脑结构和工作原理的计算模型。它由多个相互连接的节点组成，这些节点被称为神经元或神经网络。神经网络可以进行输入、处理和输出信息，并通过学习和调整权重来优化其性能。

神经网络的基本组成部分包括：

1. 输入层：接收输入信息的节点。
2. 隐藏层：进行信息处理和计算的节点。
3. 输出层：输出处理结果的节点。
4. 权重：连接不同节点之间的权重，用于调整信息传递。
5. 激活函数：用于控制神经元输出值的函数。

### 4.1.1 加密代码实例

```python
def encrypt(data, weights):
    input_data = np.array(data)
    output_data = np.dot(input_data, weights)
    output_data = sigmoid(output_data)
    return output_data

weights = [0.1, 0.2, 0.3, 0.4]
encrypted_data = encrypt(data, weights)
print(encrypted_data)
```
