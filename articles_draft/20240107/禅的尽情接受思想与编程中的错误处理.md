## 4.1 代码实例1：除法错误

在本代码实例中，我们将演示一个除法错误的处理。当我们尝试将一个整数除以0时，会出现错误。我们将使用try-except语句来捕获错误，并在出现错误时采取相应的处理措施。

```python
def divide(a, b):
    try:
        result = a / b
    except ZeroDivisionError:
        print("错误：分母不能为零")
        return None
    return result

a = 10
b = 0
print(divide(a, b))
```

在上面的代码中，我们定义了一个函数divide，它接受两个参数a和b，并尝试将a除以b。如果b为0，则会出现ZeroDivisionError错误。我们使用try-except语句来捕获这个错误，并在出现错误时打印一个错误提示信息，并返回None。如果b不为0，则返回除法结果。

当我们调用divide函数时，如果b为0，则会打印错误提示信息“错误：分母不能为零”，并返回None。如果b不为0，则返回除法结果。

## 4.2 代码实例2：文件操作错误

在本代码实例中，我们将演示一个文件操作错误的处理。当我们尝试打开一个不存在的文件时，会出现错误。我们将使用try-except语句来捕获错误，并在出现错误时采取相应的处理措施。

```python
def read_file(filename):
    try:
        with open(filename, 'r') as f:
            content = f.read()
    except FileNotFoundError:
        print(f"错误：文件 {filename} 不存在")
        return None
    return content

filename = "nonexistent_file.txt"
print(read_file(filename))
```

在上面的代码中，我们定义了一个函数read_file，它接受一个参数filename，并尝试使用with语句打开filename文件并读取其内容。如果filename文件不存在，则会出现FileNotFoundError错误。我们使用try-except语句来捕获这个错误，并在出现错误时打印一个错误提示信息，并返回None。如果filename文件存在，则返回文件内容。

当我们调用read_file函数时，如果filename文件不存在，则会打印错误提示信息“错误：文件 nonexistent_file.txt 不存在”，并返回None。如果filename文件存在，则返回文件内容。
