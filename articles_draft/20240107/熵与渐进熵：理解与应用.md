# 2.核心概念与联系

## 2.1 熵的定义

熵是信息论中的一个基本概念，它用于衡量一个随机变量的不确定性。熵的定义如下：

$$
H(X)=-\sum_{i=1}^{n}P(x_i)\log_2 P(x_i)
$$

其中，$X$是一个随机变量，$x_i$是$X$的取值，$P(x_i)$是$x_i$的概率。

熵的性质：

1. 非负性：$H(X)\geq0$
2. 连加性：$H(X_1, X_2, \dots, X_n)=\sum_{i=1}^{n}H(X_i)$
3. 条件性：$H(X_1, X_2|Y)=\sum_{y}P(y)H(X_1, X_2|Y=y)$
4. 不等性：$H(X_1, X_2)\leq H(X_1)+H(X_2)$

熵的性质是对熵概念的一系列数学描述，它们有助于我们更好地理解熵的性质和应用。以下是熵的四个主要性质：

2. 连加性：熵可以看作是随机变量的一种“加权平均值”，因此熵具有连加性。
3. 条件性：熵可以通过给定其他随机变量来计算，这就是条件熵的概念。
4. 不等性：熵是一个非负数，因此熵的连加是不大于等于原始熵的和的，这就是熵的不等性。
