## 2.1 欧氏距离的定义
欧氏距离是一种常用的几何距离度量，它可以用来衡量两个点之间的距离。给定两个点 $P(x_1, y_1)$ 和 $Q(x_2, y_2)$ 在二维平面上，欧氏距离 $d$ 可以通过以下公式计算：

$$
d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}
$$

同样，在三维空间中，给定两个点 $P(x_1, y_1, z_1)$ 和 $Q(x_2, y_2, z_2)$，欧氏距离 $d$ 可以通过以下公式计算：

$$
d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2}
$$

## 4.1 图像分割示例
```python
import cv2
import numpy as np

# 读取图像

# 转换为灰度图像
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 计算图像的高度和宽度
height, width = gray_image.shape

# 创建一个用于存储分割结果的数组
segmented_image = np.zeros((height, width), dtype=np.uint8)

# 遍历图像中的每个像素点
for y in range(height):
    for x in range(width):
        # 获取当前像素点的邻域像素点
        neighbors = [(y - 1, x), (y + 1, x), (y, x - 1), (y, x + 1)]
        
        # 计算邻域像素点之间的欧氏距离
        distances = [np.linalg.norm(np.array(gray_image[neighbor[0], neighbor[1]]) - np.array(gray_image[y, x])) for neighbor in neighbors]
        
        # 根据计算出的欧氏距离，将像素点分为不同的类别
        if min(distances) < 50:
            segmented_image[y, x] = 255

# 显示分割后的图像
cv2.imshow('Segmented Image', segmented_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
