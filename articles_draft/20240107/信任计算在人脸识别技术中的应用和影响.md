## 4.3 模型训练和验证

我们可以使用OpenCV库中的`face_recognizer.train(images, labels)`方法来训练人脸识别模型，并使用`predict(image)`方法来验证模型的准确性和稳定性。

```python
import cv2

def train_model(images, labels):
    # 训练人脸识别模型
    recognizer = cv2.face.LBPHFaceRecognizer_create()
    recognizer.train(images, labels)
    return recognizer

def validate_model(recognizer, images, labels):
    # 验证人脸识别模型
    accuracy = 0
    for image, label in zip(images, labels):
        prediction = recognizer.predict(image)
        if prediction == label:
            accuracy += 1
    return accuracy / len(images)
```
