# 2.核心概念与联系
基函数和函数内积是密切相关的概念。基函数可以被看作是函数空间中的一种基础元素，而函数内积则是用于度量这些基函数之间的相似性和差异。在本节中，我们将详细介绍这两个概念的定义、性质和联系。

## 2.1 基函数
基函数（basis function）是用于表示其他函数的基本元素。在数学和计算机科学中，基函数通常用于构建更复杂的函数，如通过线性组合来表示。基函数的一些典型例子包括：

- 一元一次多项式：$1, x, x^2, x^3, \dots$
- 三角函数：$\sin x, \cos x$
- 高斯基函数：$e^{-x^2}, xe^{-x^2}, x^2e^{-x^2}, \dots$

基函数的主要特点是：

1. 线性无关：基函数之间不存在线性关系。
2. 完全：通过线性组合可以表示任何满足一定条件的函数。

## 2.2 函数内积
函数内积（functional inner product）是两个函数之间的一个量，用于度量它们之间的相似性和差异。函数内积的定义为：

$$
\langle f, g \rangle = \int_{-\infty}^{\infty} f(x)g^*(x) dx
$$

其中，$f(x)$ 和 $g(x)$ 是实值函数，$g^*(x)$ 是 $g(x)$ 的复共轭函数。函数内积具有以下性质：

1. 对称性：$\langle f, g \rangle = \langle g, f \rangle$
2. 线性性：$\langle af + bg, h \rangle = a\langle f, h \rangle + b\langle g, h \rangle$
3. 正定性：$\langle f, f \rangle \geq 0$，且$\langle f, f \rangle = 0$ 当且仅当 $f(x) = 0$ 在某个区间内。

## 4.4 函数内积的计算
```python
import numpy as np

def inner_product(f, g):
    return np.dot(f, g.conj())

g = np.array([np.cos(x), np.sin(x)])

```
在这个例子中，我们首先导入了numpy库，然后定义了函数内积的计算函数`inner_product`。接着，我们使用`np.array`函数生成了两个包含$\sin x$和$\cos x$的数组，并计算了它们的内积。
