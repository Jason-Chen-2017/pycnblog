# 1.简介
  

我认为职场经验不仅仅是工作技巧、解决问题的方法论、能力提升手段，更重要的是一门精通的软技能，它需要长期的积累，才能使得个人在工作中获得自我的价值和满足感。
但由于职场复杂性、身份各异性、行业特点等原因，获取职场经验并不简单。有些人一步登天，马上就能领悟到工作中的各种秘诀；有的人需要经历千辛万苦才有了成功，才能最终成为一名出色的领导者或管理者；还有的人经历了种种磨炼，只不过一旦失去机会便无缘往事。
因此，如何更好地获取职场经验，这是每一个职场人士都要面临的难题。

本文通过总结国外众多优秀的职场咨询公司和个人的职场生涯分享，从不同角度提出“如何获取更多职场经验”的相关问题，希望能够帮助大家理清职场中的“不易”。

为了帮助大家掌握更多职场经验技能，作者将从以下三个方面进行介绍：

1. 从实际案例聊起——关于获取职场经验的正确姿势
2. 职场生涯的必备技能——现代职场必备技能汇总
3. 开源职业指南——分享自主创新的职业发展方法

通过阅读本文，您可以获取到：

- 在职场工作遇到的各种问题及其原因分析；
- 学习新技能所需准备的关键步骤和关键词；
- 有助于职业发展的资源和工具推荐；
- 更加成熟、全面的职场生涯规划。

# 2. 获取职场经验的正确姿势
## 2.1 从找到适合自己的职位开始
首先，要确认自己是否真的适合这个职位。从这个职位来看是否与自己的兴趣匹配？是否有足够的能力胜任？注意不要沉溺于职位而忽略了自己的内心需求。

## 2.2 使用社交网络、搜索引擎、微博客等渠道获取职场资讯
在寻找职业发展方向时，一定要关注不同于其他职业领域的信息源，例如社会新闻、媒体报道，还有前同事或同事的推荐。通过参与社交网络（如Facebook、Twitter）、搜索引擎上的职场讨论、阅读热点职场新闻，不仅可以了解到其他人的职场建议，还能增强自己对该行业的认识。

## 2.3 拥抱变化
作为职场人的，我们经常会被不断更新的信息所左右，对于自己的职业生涯而言，这一点尤为重要。拥抱变化意味着我们要持续跟进最新潮流、挖掘个人的长处，而不是盲目跟风。时刻保持警惕，不断反省自己，发现自己的缺点和短板，提高职场竞争力。

## 2.4 培养职业发展的必备品质
在职场中，除了技术能力之外，很多人都会有自己的职业观、职业理念、职业追求等。这些东西往往是决定一个人职业成败的关键因素。在选择职业方向时，需要具备广泛的视野和开放心态，不仅要考虑现实生活里的工作需求，更要考虑自己长远的发展方向和未来抱负。

同时，要善于借鉴他人优秀的经验，尽可能地将自己的职业发展融入其中。即便是公司的晋升机制、薪酬福利也都是一项非常重要的工作内容。不妨在取得优异成绩之后，与身边的同事或领导交流、探讨职业发展的经验，以帮助自己加快职业生涯的进程。

## 2.5 互相支持、鼓励、帮助
与他人建立长久的联系是获取职场经验的关键之一。通过积极向他人提问、反馈意见，可以帮助自己增强沟通技巧、协作能力，从而促进自己的职业发展。同时，在工作中出现问题时也要及时寻求帮助，这也有助于改善工作环境和团队氛围。最后，将自己的工作成果展示出来也是一种不错的方式，能让别人认可自己的工作能力。

## 2.6 提升职业技能、技术水平
除了具有职业经验外，个人的职业技能也很重要。这包括计算机基础知识、英语口语表达、团队合作能力、项目管理能力等等。这些技能是职场生涯不可或缺的一部分。

在职场中，技术能力扮演着举足轻重的角色，并且在不断升级。这里建议学习计算机编程语言，掌握数据结构和算法，有助于应对复杂的软件开发任务。另外，应尽量保持较高的英语水平，因为这是一个能令工作轻松完成的关键因素。

# 3. 现代职场必备技能汇总
## 3.1 英语能力
首先要做到的是熟练掌握英语。英语口语表达能力在日常工作中至关重要，所以学习英语可以帮助我们顺利完成日常事务。要注意修养方法，比如关注语法和拼写，并且在学习中时常复习。此外，还有一些零散的技能可以帮助我们更好地进行日常用途：包括使用笔记、听力、翻译、简历撰写、书面表达等。

## 3.2 逻辑思维能力
对于复杂的问题，我们需要善于运用逻辑思维能力。逻辑思维能力可以帮助我们把握复杂信息的关系和逻辑，并依据这些信息制定策略。比如，对于某个项目，我们可以先梳理它的大致框架，再审视细节，提出解决方案。此外，在面试的时候，逻辑思维能力可以锻炼我们的注意力、记忆力和表达能力。

## 3.3 技术知识
作为一名优秀的工程师，除了掌握计算机技能外，更重要的是理解业务、产品和客户需求，以及对解决方案的理解。了解业务细节、产品特性、市场趋势，并结合自身情况制定商业计划、技术方案。另外，还需要掌握操作系统、数据库、网络安全、Web开发、编程语言等技术知识。

## 3.4 团队合作能力
在工作中，我们需要团队合作才能把事情做成。为此，我们需要能够跟踪并分析工作内容、与同事沟通、给予合理的反馈。团队合作能力能够有效减少矛盾、促进共赢，提升工作效率。

## 3.5 沟通能力
作为一名优秀的职场人士，我们应该善于与人打交道。因此，我们需要擅长沟通，能够快速准确地传达意见，能够识别他人说话的内容。这不仅对我们的职业生涯来说十分重要，也是我们在日常生活中的交际能力、倾听能力和表达能力的基础。

## 3.6 时间管理能力
我们需要掌握良好的时间管理习惯，这样才能高效地完成工作。有效的时间管理需要长期的训练、坚持，以及良好的安排。

## 3.7 组织能力
在工作中，我们需要善于组织工作，以免造成混乱和效率低下。组织能力可以通过学习工作流程、制订制度、检查管理等方式来提高。

## 3.8 职业操守
职业操守是指遵守社会公德、维护个人权益、尊重他人劳动、保护环境等行为，它们对职场人士的影响至关重要。在职场中，我们需要承担起职责、履行义务，明白自己的义务，并承担相应的后果。职业操守会影响个人的职业前途，为个人的生活和社会带来巨大的价值。

# 4. 开源职业指南
## 4.1 为什么要选择开源？
开源并不是新鲜、神秘的概念。正如开源软件一样，源代码都是公开的，任何人都可以修改、扩展、添加功能、改进性能。甚至还有其它开源项目的组件可以组合利用。而且，随着云计算和大数据的发展，开源也越来越受到青睐。

那么为什么现在越来越多的人选择开源？简单来说，因为有很多优势：

1. 透明度：开源项目的源代码都是公开的，所有人都可以看到，能够比较清楚地知道项目的目的、目标、限制条件、具体实现过程和结果。
2. 创新能力：开源的社区鼓励创新，充满激情地加入到项目中，形成新的想法和实践模式，推动创新的潮流。
3. 服务社会：开源的优势在于对社会的贡献，能够服务于很多不同领域的人群。而且，它的社区都是由志愿者组成的，有大量优秀的技术人员为开源贡献代码，帮助其成长。

## 4.2 开源的优势究竟有哪些？
在开源中，我们可以获取到丰富的经验和技能，包括：

1. 深度思考：开源项目的源代码几乎全部是可读的，并且经过世界各地的开发者的贡献，是一个高度复杂、庞大且活跃的社区。
2. 学习能力：开源项目的文档、代码、设计思路、讨论都充满了学习的价值。
3. 问题解决：开源项目中的问题会得到及时的解答，并有专门的支持团队帮助解答。
4. 共享知识：开源的经验和技能对于个人的职业发展都非常有益，可以用于自己工作以外的地方。

## 4.3 怎样参与开源项目？
开源项目都是公开的，社区是活跃的。为了参与到开源项目中来，你可以：

1. 寻找感兴趣的项目：去 GitHub 上找感兴趣的开源项目，或者转到相关领域的网站上查看相关信息，了解该领域的趋势。
2. 浏览项目源码：下载项目的代码，仔细研究一下项目的架构，运行一下项目，能够帮助你对项目有一个整体的认识。
3. 研究项目需求：研究一下项目的用户故事、功能列表、开发计划、测试报告等文档，了解项目背后的业务背景和需求，能够帮助你做出决策。
4. 编写代码：如果觉得自己能够解决项目中的问题，就可以尝试修改项目的代码，提交 pull request 让别人帮你审核和评估你的代码。
5. 提交 Bug 和 Issues：如果遇到了问题，也可以在项目的 issue 中提交你的问题，或者直接发邮件给项目的开发者。

## 4.4 如何成为开源项目的贡献者？
开源项目是一个包罗万象的集合，任何人都可以在里面做贡献。如果你是一个热爱开源的开发者，又有能力参与其中，那就可以贡献自己的一份力量。

首先，你需要熟悉项目的开发流程，了解项目的架构和规范，能够清楚的表达你的想法、疑问、意见和见解。

其次，你需要关注项目的开发进展，关注项目的规模和活跃度，持续跟踪项目的动态，掌握项目最新的技术更新、版本发布、用户反馈等消息。

最后，当你在某个阶段对某项功能有兴趣，但是又不确定该怎么实现时，可以提交一个 proposal ，阐述你的意图、解决方案和实现思路。这样的话，其他人可以帮你完善你的方案。
