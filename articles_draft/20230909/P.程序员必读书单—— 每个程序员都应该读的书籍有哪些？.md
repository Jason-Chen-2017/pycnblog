# 1.简介
  


首先，我们要清楚地定义下什么叫做程序员。程序员是指负责编写计算机软件的人，也就是“写代码”的人。这个群体在国内外非常庞大，规模也是异常的庞大。大家熟知的程序员有以下几类：

* Web前端工程师：负责网站页面设计、制作、性能优化等工作，要求精通HTML、CSS、JavaScript等前端技术；
* Android开发工程师：负责Android手机应用的开发工作，要求精通Java语言，掌握面向对象的编程思想、深刻理解各种设计模式和系统架构；
* iOS开发工程师：负责iPhone、iPad手机应用的开发工作，要求精通Objective-C或Swift语言，了解多线程编程、内存管理、系统框架等知识；
* 后端工程师：负责服务器后台业务逻辑的实现，主要用到的技术栈包括Python/PHP/JAVA等；
* 数据分析师：负责收集、整理、分析并呈现海量数据信息，对数据进行分析和挖掘从而发现业务价值，提升公司产品和服务质量，需要懂得数据采集、清洗、处理、分析、可视化等方面的技能；
* 测试人员：负责测试软件功能的正确性、效率、安全、兼容性等，通常也会参与到需求开发阶段。


因此，作为程序员，要学习相关技术，还不止于阅读一本技术书籍。除了技术书籍，还有很多其他方面的书籍是程序员必备的，比如：

  * 操作系统原理：掌握操作系统的基本原理及其设计思想，能够更好的理解软件运行机制，以及如何合理分配资源；
  * 数据库原理：掌握数据库的基本原理和核心组件，以及索引、锁、事务等相关概念；
  * 网络基础：了解TCP/IP协议族、HTTP协议、Socket编程、SSH、VNC等网络技术，能够构建自己的网络应用程序；
  * 软件工程实践：通过实际项目经验，了解软件开发过程中的一些最佳实践，能够提高生产力和质量；
  * 系统架构：系统架构设计的原则、方法论和理论，能帮助你设计出健壮、高效、可扩展的软件系统。
  
综上所述，阅读这些书籍能够使你的编程能力有提升，并且有更多的时间去学习新技术，从而提升职场竞争力。同时，阅读这些书籍也可以帮助你更好地与团队中的其他成员沟通交流，以及进一步提升自己的能力。当然，每个人的学习习惯不同，不同的书籍适用于自己。所以，想要成为一个优秀的程序员，就需要广泛阅读各类书籍。下面我将根据自己的经验，整理出几个适合程序员阅读的书籍。

# 2.程序员必读书单（持续更新）
## 第一部分：入门必读
### 1. C程序设计(第四版)
这是一本由丹尼斯·库克联合剽窃的经典教材，相信每一个程序员都会读过这本书。如果你是刚入门，或者想要了解一下C语言的基本语法，可以看一看这本书。这本书讲述了程序设计的一些基本概念，如变量、表达式、语句、数组、指针、函数、结构体、文件输入输出、递归、递推、排序、搜索、随机数生成、多线程、图形用户界面、字符串处理、动态内存管理等，并配有详细的示例程序。如果你已经有一定经验，但仍然觉得这本书有点枯燥，你也可以跳过前几章的内容，直接进入到第五章——指针、内存管理、字符串处理，这些内容更加困难些。但对于初学者来说，这本书是一个不错的选择。

《C程序设计》第四版,作者：丹尼斯·库克著, 译者：陈治军 赵一鸣 张冠昊, 机械工业出版社, 1988年7月第二版. 


### 2. ANSI C 标准(第三版)
如果你有较强的C语言基础，但希望了解C语言的最新规范，可以看一看这本书。这本书从头至尾介绍了ANSI C语言标准，包括基本的语法、运算符优先级、类型转换、位操作、数组、指针、结构体、枚举、联合体、预处理器、I/O控制、标准库等内容。如果你没有很强的C语言基础，建议先阅读《C程序设计》再阅读该书。

《ANSI C 标准(第三版)》, 作者:蒲柏瑜著, 译者：刘昆 苏霁红 王子春, 科学出版社, 2006年6月第一版.


### 3. 程序员修炼之道 (第二版)
这是一本值得一读的程序员修炼手册。这本书介绍了程序员应该具备的基本素养、思维方式、编程风格、编程习惯、编码规范、文档约定等内容。该书提供了一个循序渐进的方法，教会你在复杂的代码中快速找寻问题，而不是一味追求完美。另外，如果你愿意，也可以将书籍中的内容应用到自己的实际工作中。

《程序员修炼之道 (第二版)》，作者：韩奇伟、施耐著，机械工业出版社，2005年9月第二版。


### 4. C++ Primer Plus(第六版)
如果你是一名C++的老司机，又或者是希望更全面地了解C++的特性，可以看一看这本书。这本书提供了足够的信息来帮助你了解C++的各个方面，包括面向对象编程、泛型编程、模板编程、iostream编程、异常处理、内存管理、反汇编、调试、字符串处理、并行编程等。这本书堪称经典，而且涵盖面很广，适合有一定经验的C++老司机阅读。

《C++ Primer Plus(第六版)》，作者：Bjarne Stroustrup 著，Addison-Wesley Professional, 1999 年 10 月第六版。


### 5. 深入理解计算机系统(第三版)
如果你想要真正理解计算机系统是怎样运行的，那么这本书绝对是必须读的。这本书系统地介绍了计算机系统的方方面面，从微观层面介绍了计算机硬件、软件、系统结构，到宏观层面介绍了操作系统、编译系统、分布式系统，甚至还介绍了体系结构、嵌入式系统等。如果你之前没有接触过计算机系统的相关知识，这本书可能就会让你觉得它过于艰深难懂。但是，如果你认真读完这本书，你就会明白为什么计算机发展的这么迅速，以及为什么一些新的计算机体系架构如ARM架构这么受欢迎。这本书既有工程实践的思维，又有知识总结的功底，是一本值得一读的参考书。

《深入理解计算机系统(第三版)》，作者：陈向、林锦福译，清华大学出版社，2012年9月第一版。


### 6. Linux命令行大全
Linux命令行的学习曲线比较陡峭，因为它有很多命令。这本书就像你身边的一个“新手指南”，帮你快速上手。如果你是刚入门Linux，可以看一看这本书，里面介绍了Linux的基本概念、shell脚本、文本处理、查找替换、压缩与解压、用户权限与目录管理、进程与任务管理等内容。另外，你可以借助这本书查阅一些常用的Linux命令的用法。

《Linux命令行大全》，作者：鸟哥著，电子工业出版社，2012年10月第一版。


### 7. 极客兔兔 Git飞行规则
Git是一个开源版本控制系统，虽然简单易用，却十分强大。如果想更加深入地了解Git，你需要阅读这本书。这本书就像“兔子之旅”，带你领略Git的风采，学习Git的工作原理，掌握Git的核心概念。如果你刚开始使用Git，这本书也许就能给你一些帮助。

《极客兔兔 Git飞行规则》，作者：黄玄、李硕译，电子工业出版社，2015年11月第一版。


## 第二部分：Web前端必读
### 1. HTML & CSS基础教程
HTML和CSS是构建网页的基石。这两门技术就是让web开发变得更容易、更美观。这本书详细介绍了HTML、CSS的基础知识，并附带实例演示了HTML和CSS的基本用法。如果你是一位经验丰富的前端工程师，可以跳过这本书，直接进入到“JavaScript基础教程”。

《HTML & CSS基础教程》，作者：李荣潮著，湖南教育出版社，2013年8月第一版。


### 2. JavaScript基础教程
JavaScript是一种强大的脚本语言，用于与浏览器互动。这本书详细介绍了JavaScript的基础语法、DOM编程、异步编程、浏览器兼容性、Nodejs、WebAssembly等内容，并配有丰富的实例练习。如果你是一位需要提高技能的Web开发人员，这本书绝对是必读的。

《JavaScript基础教程》，作者：阮一峰著，人民邮电出版社，2013年11月第一版。


### 3. jQuery技术内幕
jQuery是一个非常流行的JavaScript库，它可以简化HTML文档的操纵，使得JavaScript的编程更高效。这本书从多个角度剖析了jQuery的内部构造及实现，并通过丰富的实例讲解了jQuery的使用方法。如果你对jQuery有兴趣，这本书绝对值得一读。

《jQuery技术内幕》，作者：姜戎、张晓龙译，电子工业出版社，2015年9月第一版。


### 4. Ajax实战
Ajax是一个在不刷新整个页面的情况下，局部更新页面的一种技术。这本书通过实例教会你使用Ajax从零开始实现局部刷新页面，并展示了一些注意事项。如果你需要实现后台数据的增删改查，这本书绝对值得一读。

《Ajax实战》，作者：张鑫旭著，电子工业出版社，2014年1月第一版。


### 5. Node.js权威指南
Node.js是一种基于Chrome V8引擎的JavaScript运行环境，它可以让JavaScript变成一门服务器编程语言。这本书通过详细介绍了Node.js的核心模块、异步编程模型、流媒体处理、WebSocket实时通信等内容，并通过丰富的实例，帮助你学习并掌握Node.js。如果你正在寻找一本深入浅出的Node.js教程，这本书一定适合你。

《Node.js权威指南》，作者：朴灿烈、葛成章译，人民邮电出版社，2015年10月第一版。


### 6. React入门教程
React是一个用于构建用户界面的JavaScript库。这本书详细介绍了React的起源、生态、虚拟DOM、状态管理、组件化开发、路由配置等内容，并附带了多个实例，帮助你快速上手。如果你正在考虑使用React构建应用程序，这本书非常适合你。

《React入门教程》，作者：阮一峰著，人民邮电出版社，2018年6月第一版。


### 7. Vue.js 2 技术揭秘
Vue.js是一个用于构建用户界面的渐进式框架，它的独特之处在于双向数据绑定。这本书从多个角度讲解了Vue.js的核心技术，包括虚拟DOM、模板编译、组件系统、路由系统、插件系统等。如果你是一位Vue.js的专家，这本书一定会是你的一份佳品。

《Vue.js 2 技术揭秘》，作者：尤雨溪著，人民邮电出版社，2017年5月第一版。


### 8. TypeScript从零重构axios
TypeScript是一个由微软发布的超级强大的编程语言。这本书从零开始重构了 Axios，并让你对其有更深入的理解。Axios 是主流的 Promise 风格 HTTP 请求库，它支持浏览器端和 Node.js 端。它提供了 TypeScript 的类型定义文件，方便使用 TypeScript 来开发基于 Axios 的项目。如果你要开始使用 TypeScript，这本书是一本不错的选择。

《TypeScript从零重构axios》，作者：黄鲂蘊著，电子工业出版社，2018年7月第一版。


### 9. 框架全景图
这是一本可读性很高的书，它收录了所有常用的Web开发框架，并给予了它们的概览、优缺点、使用的场景、API等信息。如果你是一位喜欢研究各种技术的程序员，这本书绝对值得一读。

《框架全景图》，作者：赵达森、李银河、冯东书、杜子聪、姜玉敏译，电子工业出版社，2018年10月第一版。
