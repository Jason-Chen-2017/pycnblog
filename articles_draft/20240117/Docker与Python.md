# 3.3 Docker与Python的具体操作步骤
1. 安装Docker：根据自己的操作系统选择对应的安装包，安装Docker。

2. 创建Dockerfile：创建一个名为Dockerfile的文件，用于定义容器化的应用。

3. 编写Dockerfile：在Dockerfile中编写以下内容：
```
FROM python:3.7

WORKDIR /app

COPY requirements.txt .

RUN pip install -r requirements.txt

COPY . .

CMD ["python", "app.py"]
```

4. 构建Docker镜像：使用以下命令构建Docker镜像：
```
docker build -t my-python-app .
```

5. 运行Docker容器：使用以下命令运行Docker容器：
```
docker run -p 8000:80 my-python-app
```
