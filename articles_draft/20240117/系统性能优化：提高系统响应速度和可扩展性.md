## 4.1 算法优化的代码示例

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[0]
        left = [x for x in arr[1:] if x <= pivot]
        right = [x for x in arr[1:] if x > pivot]
        return quick_sort(left) + [pivot] + quick_sort(right)
```

## 4.2 硬件优化的代码示例

```python
import numpy as np

def matrix_multiply(A, B):
    rows = A.shape[0]
    cols = B.shape[1]
    C = np.zeros((rows, cols))
    for i in range(rows):
        for j in range(cols):
            for k in range(A.shape[1]):
                C[i, j] += A[i, k] * B[k, j]
    return C
```

## 4.3 软件优化的代码示例

```python
import time

def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

start_time = time.time()
fibonacci(30)
end_time = time.time()

print("Time taken:", end_time - start_time)
```
