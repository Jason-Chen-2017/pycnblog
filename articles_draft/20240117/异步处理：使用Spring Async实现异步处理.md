# 4. 具体代码实例和详细解释说明

## 4.1 异步处理的代码实例

以下是一个使用回调函数实现异步处理的代码实例：

```java
import java.util.concurrent.Future;
import java.util.concurrent.ExecutorService;
import java.util.concurrent.Executors;

public class AsyncExample {
    public static void main(String[] args) {
        ExecutorService executor = Executors.newFixedThreadPool(10);
        Future<String> future = executor.submit(() -> {
            // 执行异步任务
            return "Hello, World!";
        });
        // 主线程继续执行其他任务
        System.out.println("Waiting for result...");
        // 在任务完成后调用回调函数
        System.out.println("Result: " + future.get());
        executor.shutdown();
    }
}
```

## 4.2 Spring Async的代码实例

以下是一个使用Spring Async实现异步处理的代码实例：

```java
import org.springframework.scheduling.annotation.Async;
import org.springframework.stereotype.Service;

@Service
public class AsyncService {
    @Async
    public void asyncTask() {
        // 执行异步任务
        System.out.println("Async task started...");
        try {
            Thread.sleep(3000);
        } catch (InterruptedException e) {
            e.printStackTrace();
        }
        System.out.println("Async task completed.");
    }
}
```

## 4.3 代码实例的详细解释说明

异步处理的代码实例使用Java的线程池和回调函数实现异步处理。主线程在等待异步任务完成之前继续执行其他任务。在异步任务完成后，回调函数会被调用，从而得到异步任务的结果。

Spring Async的代码实例使用Spring框架的异步处理功能实现异步处理。通过使用`@Async`注解，我们可以将方法标记为异步方法，从而让Spring框架在后台执行这些方法。在异步方法完成后，Spring框架会调用回调函数，从而得到异步方法的结果。
