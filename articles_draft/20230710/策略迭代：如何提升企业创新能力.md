3.2. 核心模块实现

创建一个名为 `strategy_iterator` 的 Python 模块，并实现以下函数：

```python
def strategy_iterator(environment, initial_strategy, max_iterations=100):
    strategy = initial_strategy
    while True:
        try:
            strategy.execute(environment)
            result = strategy.get_result()
            if result:
                environment = environment.update(result)
        except Exception as e:
            print(f"Error: {e}")
```
