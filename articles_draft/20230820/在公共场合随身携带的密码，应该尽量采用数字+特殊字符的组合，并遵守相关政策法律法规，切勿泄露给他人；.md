# 一、背景介绍

20世纪90年代末，美国政府颁布了《通用电气公司（GEM）应急设备实施指南》。该指南提到要采取一系列的应急措施，如建立紧急情况下的灾难恢复中心、建立突发事件响应中心、实施完备的灾难防御体系、全面部署数字化灾难恢复、定期检查和测试以及响应人员培训。该指南第十章“密码”部分详细阐述了密码的重要性和作用。

## （一）密码的由来及分类

密码的历史可追溯到古代。古希腊的哲学家亚里士多德就认为密码就是这样一种方法，通过一种独特的方式来加密讯息。他说："所有的讯息都要由另一个想与之沟通的人作保管，那个人只能透过一个与他有密钥关系的装置来解开这些讯息。"为了使这个密钥看上去不那么明显，人们又创造了各种各样的符号、文字、图形以及其他物品来装载它，这些东西常常被称作密码本，有时也会称它们为密码。例如古罗马军队使用的凭证，就是用来存放武器、食品等绝密材料的密码本。

到了近代，密码出现了三个主要类型：

1. 单表查字密码：这是最早的密码形式。它把字母、数字以及符号等所有可能的字符组合成密钥，然后再将消息中的每一个字母、数字或者符号与相应的密钥进行对应替换，得到加密后的结果。这种加密方法很简单，但由于密钥只有几个，所以破译起来也比较困难。

2. 几何密码：几何密码是通过将明文中的字母和数字表示成不同的图案来构造密钥的。比如，字母 A 可以作为矩形方块，字母 B 可以作为菱形或圆角形，字母 C 可以作为叉形等。这样就可以将一段文字映射成为图形，这个过程通常叫做编码。因为密钥通常都比较复杂，所以这类密码更安全。

3. 循环密码：循环密码是利用数列来构造密钥的。首先，把原始密码序列分割成若干子序列，每个子序列有相同数量的字符。然后，把这些子序列组成一个新的序列，重复这个过程，直到整个密码序列都被反复使用。这样就构造出了一个比较长的密钥。例如，一个长度为 n 的密码序列可以划分为 n 个连续的子序列，重复这个过程，最终可以形成一个比较大的密钥。为了更方便破译，还可以对密钥进行混淆，然后通过更多的异或运算来折叠这些密码。

目前，数字密码已经成为主流。它在加密过程中使用的是数字，并且将数字划分为不同的位数，这种密码可以更好地抵御暴力攻击。同时，由于数字密码不容易被猜测，因此也使得数字密码系统更安全。

# 二、基本概念术语说明

## （一）密码的含义

密码是一种机械的方法，用于保障数据的机密性、完整性和可用性。它的主要目的是隐藏发送者和接收者之间通信的内容，只让授权的双方能够读取信息。密码并不是自然界存在的实体，而是为了保护数据的措施。

## （二）常用密码术语

- 密码密钥：是为了保护数据的加密解密秘钥，包含数字、字母、符号等。
- 加密/解密：是指将明文数据转换成密文数据的过程。
- 暗号：用于传递加密信息的信道，包括纸、木头、铅笔或其他符号。
- 卡拉ok码：一种可随身携带的密码。
- 口令：一种设置密码的名称，用来确定访问某个服务或账户的权限。
- 唯一标识符：指个人身份识别号码、邮箱地址、手机号码或其他用于登录网站的账号。
- 真正的随机数：是指具有完全随机性、独立于其他随机数生成器、不能预测并且经过一定计算才能得到的数字。

# 三、核心算法原理和具体操作步骤

## （一）密码生成方法

### 方法一：利用计算机自动生成密码

1. 创建随机数种子：从某种随机数产生器或特定时间点获取的随机数，作为加密初始值。

2. 将随机数转换成密码字符串：将随机数转换成固定长度的字符串，其长度一般为8至16个字符。

3. 生成密码：根据一定规则，将随机数中的字符按照固定顺序排列组合，输出密码字符串。

4. 提交密码：提交密码字符串，此后再使用相同的随机数种子生成的密码均属于同一类。

### 方法二：手动输入密码

1. 获取基础密码：手动输入符合密码要求的密码，并将其记录下来。

2. 混淆密码：手动输入密码时，先通过乱序和替换等方式对其进行混淆，使得密码看上去不易辨认。

3. 使用验证码：对于较长或复杂的密码，需要用验证码验证用户是否输入正确。

### 方法三：混合以上两种方法

1. 使用多种方法组合：既可以使用计算机自动生成的密码，也可以手工输入。

2. 设置复杂性要求：密码复杂度越高，防止暴力破解的难度越大，但同时也越容易被破解。

## （二）密码存储策略

保障个人隐私和数据安全，首先就要对个人敏感信息进行保护。那么，如何存储密码也是一条关键环节。根据国际标准，密码应当具有高强度、低熵、易变换等特征，且密码库应当保密。

- 存储在笔记本电脑上：不可取，笔记本电脑容易丢失、被盗、被黑客入侵等。
- 存储在互联网上：建议加密传输，避免信息被截获。
- 不保存密码：密码泄露导致资产损失等严重后果。
- 保存密码最好不要连带生日、身份证号、银行卡号、支付宝账号等个人信息。
- 通过密码管理器管理密码，可以保证密码安全。
- 使用多个密码组合：可以实现彩虹表攻击。

## （三）密码管理策略

- 只允许同事知晓密钥：共享密钥容易造成泄漏、被盗、被篡改等安全风险。
- 对登录密码等重要信息进行双因素验证：两次验证，增强账户安全性。
- 定期更改密码：确保密码的有效性，降低被他人破解的风险。
- 开启多因素验证：增加账户安全性。
- 定期备份密码：防止数据丢失。
- 如果遇到财务危机，请及时通知所有相关人员修改密码。

## （四）密码传播方式

- 密钥不要透露给他人：密钥如果泄露，可能会造成巨额损失。
- 密钥不要在不安全的网络环境下保存：密钥泄露或被盗，可能会导致严重经济损失。
- 密码使用后请立即修改：随时更新密码是保持帐户安全的重要措施。
- 用随机字母、数字生成密码：密钥不易被猜测，增加密码复杂度。

## （五）注意事项

- 记住密码：在填写任何网站、注册新账号时，请务必牢记密码。
- 多重身份验证：多个身份信息的组合，比如姓名、身份证号、生日、银行卡号等，能有效防止账户被盗。
- 断网记录：不要在公共场所保存重要信息，请尽量在开放 Wi-Fi 环境下保存。
- 通知账户权限变动：账户权限变动时，需通知所有相关人员修改密码。
- 更改密码频率：定期更换密码，能防止遗忘密码。
- 防范黑客攻击：制定好密码策略，并定期检查网站安全日志。
