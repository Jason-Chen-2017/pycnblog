## 4. 具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来说明如何设计可重用的代码：

```python
# 模块化
def add(x, y):
    return x + y

def sub(x, y):
    return x - y

# 接口
def operation(x, y, op):
    if op == 'add':
        return add(x, y)
    elif op == 'sub':
        return sub(x, y)

# 抽象
def calculate(x, y, op):
    return operation(x, y, op)

# 封装
class Calculator:
    def __init__(self, x, y, op):
        self.x = x
        self.y = y
        self.op = op

    def calculate(self):
        if self.op == 'add':
            return self.add()
        elif self.op == 'sub':
            return self.sub()

    def add(self):
        return self.x + self.y

    def sub(self):
        return self.x - self.y

# 使用
calculator = Calculator(10, 5, 'add')
result = calculator.calculate()
print(result)  # 输出: 15
```

在这个代码实例中，我们首先将代码划分为多个模块，每个模块负责完成特定的功能。然后，我们设计了接口，以便不同的模块之间可以互相调用。接下来，我们对代码进行了抽象，将复杂的问题简化为更简单的问题，以便更容易解决。最后，我们对代码进行了封装，将相关的数据和操作封装在一个单元中，以便更好地组织代码。
