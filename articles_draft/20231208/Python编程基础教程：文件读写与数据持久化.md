$$
f(x) = \sum_{i=1}^{n} a_i x^i

### 4.2数据持久化的具体代码实例


### 6.1文件读写常见问题与解答

#### 问题1：如何读取文件的第n行内容？

答案：使用`readlines()`方法读取文件的所有行内容，并使用`enumerate()`函数获取行号，然后使用索引访问第n行内容。

```python
# 打开文件
file = open("data.txt", "r")

# 读取文件内容
content = file.readlines()

# 关闭文件
file.close()

# 获取第n行内容
n = 2
line = content[n-1]
print(line)
```

#### 问题2：如何写入文件的第n行内容？

答案：使用`write()`方法写入文件内容，并使用`enumerate()`函数获取行号，然后使用索引访问第n行内容。

```python
# 创建Python对象
data = [1, 2, 3, 4, 5]

# 打开文件
file = open("data.txt", "w")

# 写入文件内容
for i, item in enumerate(data):
    file.write(str(item) + "\n")
    if i == 1:
        file.write("这是第2行内容")

# 关闭文件
file.close()
```
