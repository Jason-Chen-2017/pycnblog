## 3.1 线性回归模型
线性回归模型是一种常用的数学模型，用于描述两个变量之间的关系。在健康与福祉研究中，我们可以使用线性回归模型来描述幸福感与健康状况、生活质量之间的关系。

线性回归模型的数学公式为：
$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$
其中，$y$是因变量（幸福感），$x_1, x_2, \cdots, x_n$是自变量（健康状况、生活质量等），$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$是回归系数，$\epsilon$是误差项。

## 3.2 随机森林模型
随机森林模型是一种常用的数学模型，用于分类和回归分析。在健康与福祉研究中，我们可以使用随机森林模型来分析多个因素对幸福感的影响。

随机森林模型的数学公式为：
$$
\hat{y} = \frac{1}{K} \sum_{k=1}^K f_k(x)
$$
其中，$\hat{y}$是预测值，$K$是决策树的数量，$f_k(x)$是第$k$个决策树的预测值。

## 3.3 支持向量机模型
支持向量机模型是一种常用的数学模型，用于分类和回归分析。在健康与福祉研究中，我们可以使用支持向量机模型来分析多个因素对幸福感的影响。

支持向量机模型的数学公式为：
$$
f(x) = \text{sgn} \left( \sum_{i=1}^N \alpha_i y_i K(x_i, x) + b \right)
$$
其中，$f(x)$是预测值，$N$是训练样本数量，$y_i$是训练样本的标签，$K(x_i, x)$是核函数，$\alpha_i$是拉格朗日乘子，$b$是偏置项。

## 4.1 线性回归模型
```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 收集数据
x = np.array([[1, 2], [3, 4], [5, 6]])
y = np.array([1, 2, 3])

# 训练模型
model = LinearRegression()
model.fit(x, y)

# 预测
x_new = np.array([[6, 7]])
y_pred = model.predict(x_new)
print(y_pred)
```

## 4.2 随机森林模型
```python
import numpy as np
from sklearn.ensemble import RandomForestRegressor

# 收集数据
x = np.array([[1, 2], [3, 4], [5, 6]])
y = np.array([1, 2, 3])

# 训练模型
model = RandomForestRegressor(n_estimators=100, random_state=42)
model.fit(x, y)

# 预测
x_new = np.array([[6, 7]])
y_pred = model.predict(x_new)
print(y_pred)
```
