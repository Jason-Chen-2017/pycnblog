### 4.1 数据仓库的具体代码实例

```python
import pandas as pd

# 数据源
data_source = pd.read_csv('data.csv')

# 数据集成
data_integration = data_source.groupby('key').mean()

# 数据存储
data_integration.to_csv('data_integration.csv')

# 数据访问
data_access = pd.read_csv('data_integration.csv')
```

### 4.2 数据集成架构的具体代码实例

```python
# 数据源
data_source1 = pd.read_csv('data1.csv')

# 数据转换
data_transform = data_source1.merge(data_source2, on='key')

# 数据存储
data_transform.to_csv('data_transform.csv')

# 数据访问
data_access = pd.read_csv('data_transform.csv')
```
