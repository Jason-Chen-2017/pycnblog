### 4.1 代码实例

```python
import http.server
import socketserver
import json
import base64
import hashlib
import hmac
import time
import os

class Handler(http.server.BaseHTTPRequestHandler):
    def do_GET(self):
        self.send_response(200)
        self.send_header('Content-type', 'application/json')
        self.end_headers()
        data = {
            'timestamp': str(int(time.time())),
            'nonce': str(os.urandom(10)),
            'signature': self.generate_signature(self.path)
        }
        self.wfile.write(json.dumps(data).encode('utf-8'))

    def generate_signature(self, path):
        key = b'your_secret_key'
        h = hmac.new(key, path.encode('utf-8'), hashlib.sha256)
        return base64.b64encode(h.digest()).decode('utf-8')

if __name__ == '__main__':
    HOST, PORT = "localhost", 8080
    server = socketserver.TCPServer((HOST, PORT), Handler)
    server.serve_forever()
```

### 4.2 详细解释说明

在这个代码实例中，我们实现了一个简单的HTTP服务器，用于处理智能家居设备的社交化和分享化的请求。服务器通过以下几个步骤来处理这些请求：

1. 接收客户端的请求。
2. 生成一个随机的timestamp和nonce。
3. 使用客户端提供的path和secret_key生成一个签名。
4. 将生成的timestamp、nonce和签名返回给客户端。

这个代码实例中的Handler类继承自http.server.BaseHTTPRequestHandler类，用于处理HTTP请求。在do_GET方法中，我们首先发送一个200的响应状态码，然后发送一个Content-type的响应头，表示响应体是一个JSON格式的数据。接下来，我们生成一个timestamp和nonce，并使用客户端提供的path和secret_key生成一个签名。最后，我们将这些数据以JSON格式返回给客户端。
