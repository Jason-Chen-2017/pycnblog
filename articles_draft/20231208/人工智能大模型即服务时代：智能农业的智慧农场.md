## 3.6 数学模型公式详细讲解

人工智能算法的数学模型公式主要包括：

1. 线性回归：y = wTx + b
2. 逻辑回归：P(y=1) = 1 / (1 + exp(-(wTx + b)))
3. 支持向量机：min (1/2 ||w||^2) 主条件：yi(wTxi + b) >= 1, i = 1, 2, ..., n
4. 聚类：min Σd(xi, mi)，其中d(xi, mi)是xi与mi之间的距离，xi是数据点，mi是簇中心
5. 主成分分析：Σxixi = Σxjxj，其中xij是数据点的第i个特征值，xj是数据点的第j个特征值
6. 奇异值分解：A = UΣV^T，其中A是数据矩阵，U是左奇异向量矩阵，Σ是奇异值矩阵，V是右奇异向量矩阵
7. Q-学习：Q(s, a) = Q(s, a) + α(r + γmaxQ(s', a') - Q(s, a))
8. 策略梯度：Δθ = -2/N Σ[g(s, a) * ∇log(πθ(a|s)) * (r + γV(s'))]
