### 3.1加密技术

#### 3.1.1哈希函数

哈希函数是一种将任意长度输入转换为固定长度输出的函数。哈希函数具有以下特点：

1. 确定性：对于任意输入，哈希函数始终产生相同的输出。
2. 唯一性：对于任意输出，哈希函数始终产生相同的输入。
3. 敏感性：对于任意输入，哈希函数对输入的任何变化都会产生不同的输出。

在区块链技术中，哈希函数用于生成区块的哈希值，以实现数据的安全性和完整性。

#### 3.1.2公钥加密和私钥解密

公钥加密和私钥解密是一种加密技术，通过使用一对公钥和私钥来实现数字签名和身份验证。

公钥加密：使用公钥加密数据，只有拥有对应的私钥才能解密数据。公钥可以公开分享，因为只有私钥才能解密数据。

私钥解密：使用私钥解密数据，只有拥有对应的公钥才能加密数据。私钥不能公开分享，因为只有私钥才能解密数据。

在区块链技术中，公钥加密和私钥解密用于实现交易的数字签名和身份验证。

### 3.2共识算法

共识算法是区块链技术中的一种机制，用于确定哪些交易是有效的，哪些区块是有效的。共识算法需要满足以下条件：

1. 一致性：所有节点必须同意哪些交易是有效的，哪些区块是有效的。
2. 容错性：即使部分节点失效，也能保证系统的正常运行。
3. 快速性：共识算法需要快速达成一致性，以保证系统的高效运行。

在区块链技术中，共识算法是通过节点之间的交互来实现的。例如，比特币使用工作量证明（PoW）算法，以太坊使用共识算法（PoS）。

### 4.2创建一个区块

```python
class Block:
    def __init__(self, data):
        self.data = data
        self.timestamp = time.time()
        self.prev_hash = self.calc_prev_hash()
        self.hash = self.calc_hash()

    def calc_prev_hash(self):
        return self.prev_hash

    def calc_hash(self):
        return hashlib.sha256(str(self.data + self.timestamp + self.prev_hash).encode('utf-8')).hexdigest()
```

def link_blocks(prev_block, current_block):

def consensus(blocks):
