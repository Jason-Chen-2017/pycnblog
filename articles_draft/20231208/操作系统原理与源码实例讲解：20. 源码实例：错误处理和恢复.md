# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释错误处理和恢复的实现方法。

假设我们有一个简单的计算器程序，程序需要对两个数进行加法运算。当用户输入的数字不是数字时，程序需要进行错误处理和恢复。

```python
def add(a, b):
    try:
        result = a + b
        return result
    except TypeError:
        print("输入的数字不是数字，请重新输入")
        return None

a = input("请输入第一个数字：")
b = input("请输入第二个数字：")

if a and b:
    result = add(a, b)
    if result is not None:
        print("结果为：", result)
    else:
        print("计算错误，请重新输入")
else:
    print("输入的数字不能为空")
```

在这个代码实例中，我们使用try-except语句来捕获TypeError异常，当用户输入的数字不是数字时，程序会输出错误信息并提示用户重新输入。当用户输入的数字是数字时，程序会进行加法运算并输出结果。
