### 3.1随机变量的概念

随机变量是一种可以取任意值的变量，其值的分布是随机的。在人工智能和机器学习中，我们经常需要处理随机变量，例如数据分布、模型评估、预测等。随机变量可以分为两类：离散型随机变量和连续型随机变量。离散型随机变量只能取有限个值，而连续型随机变量可以取任意值。

### 3.2分布函数

分布函数是随机变量的一个重要特征，用于描述随机变量的取值概率。分布函数F(x)是一个非负函数，满足以下条件：

1. F(x)为0，当x<a时，其中a是随机变量的最小值。
2. F(x)为1，当x≥b时，其中b是随机变量的最大值。
3. F(x)是单调递增的函数。

### 3.3概率密度函数

概率密度函数是连续型随机变量的一个重要特征，用于描述随机变量在某一区间内的概率密度。概率密度函数f(x)是一个非负函数，满足以下条件：

1. 积分为1，即∫f(x)dx=1，当x的范围为整个实数域时。
2. f(x)≥0，当x∈[a,b]时，其中a和b是随机变量的取值范围。

### 3.4期望

期望是随机变量的一个重要统计量，用于描述随机变量的中心趋势。期望是随机变量的一种平均值，可以通过概率密度函数计算。期望的公式为：

E[X]=∫x f(x) dx

### 3.5方差

方差是随机变量的一个重要统计量，用于描述随机变量的离散程度。方差是随机变量的一种平均值，可以通过概率密度函数计算。方差的公式为：

### 4.1期望

期望是随机变量的一个重要统计量，用于描述随机变量的中心趋势。期望是随机变量的一种平均值，可以通过概率密度函数计算。期望的公式为：

以下是一个Python代码实例，用于计算随机变量的期望：

```python
import numpy as np

def expectation(f, a, b):
    # 计算积分的区间
    x = np.linspace(a, b, 1000)
    y = f(x)
    # 计算积分的值
    integral = np.trapz(y, x)
    return integral

def f(x):
    return 1 / (x * np.sqrt(2 * np.pi)) * np.exp(-1 * (x ** 2) / 2)

a = -10
b = 10
print("期望为：", expectation(f, a, b))
```
