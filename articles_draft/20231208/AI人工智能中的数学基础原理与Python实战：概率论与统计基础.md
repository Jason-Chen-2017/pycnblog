## 3.1 概率论

### 3.1.1 概率的基本概念

概率是一个事件发生的可能性，它的值范围在0到1之间。概率的计算主要有两种方法：

1.直接计算法：直接计算法是指通过计算事件发生的可能性来得出概率值。例如，在一个六面骰子上，事件A是“骰子上的点数为3”，那么事件A的概率为1/6。

2.定义法：定义法是指通过定义一个事件的样本空间来得出概率值。例如，在一个六面骰子上，事件A是“骰子上的点数为3”，那么事件A的样本空间为{1,2,3,4,5,6}，事件A的概率为1/6。

### 3.1.2 概率的几种性质

1.非负性：概率的值必须大于等于0。

2.完整性：概率的值必须在0到1之间。

3.一致性：如果事件A和事件B是互斥的，那么事件A和事件B的概率之和必须等于1。

### 3.1.3 概率的几种计算方法

1.直接计算法：直接计算法是指通过计算事件发生的可能性来得出概率值。例如，在一个六面骰子上，事件A是“骰子上的点数为3”，那么事件A的概率为1/6。

2.定义法：定义法是指通过定义一个事件的样本空间来得出概率值。例如，在一个六面骰子上，事件A是“骰子上的点数为3”，那么事件A的样本空间为{1,2,3,4,5,6}，事件A的概率为1/6。

3.条件概率：条件概率是指在已知某个事件发生的条件下，另一个事件发生的概率。例如，在一个六面骰子上，事件A是“骰子上的点数为3”，事件B是“骰子上的点数为偶数”，那么事件A和事件B的条件概率为1/3和1/2。

### 3.1.4 概率的几种应用

1.随机变量：随机变量是指在某个事件发生时，可能取不同值的变量。例如，在一个六面骰子上，事件A是“骰子上的点数为3”，那么随机变量X的取值为3。

2.概率分布：概率分布是指在某个事件发生时，各种可能结果的概率之和为1的分布。例如，在一个六面骰子上，事件A是“骰子上的点数为3”，那么概率分布为{1/6,1/6,1/6,1/6,1/6,1/6}。

3.贝叶斯定理：贝叶斯定理是指在已知某个事件发生的条件下，另一个事件发生的概率。例如，在一个六面骰子上，事件A是“骰子上的点数为3”，事件B是“骰子上的点数为偶数”，那么事件A和事件B的条件概率为1/3和1/2。

## 3.2 统计

### 3.2.1 统计的基本概念

统计是一种用于处理大量数据的方法，用于发现数据中的模式和规律。统计主要有两种方法：

1.描述性统计：描述性统计是指通过计算数据的一些基本统计量来描述数据的特点。例如，在一个数据集中，我们可以计算平均值、中位数、方差等基本统计量。

2.分析性统计：分析性统计是指通过对数据进行分析来发现数据中的模式和规律。例如，在一个数据集中，我们可以使用线性回归来预测某个变量的值。

### 3.2.2 统计的几种方法

1.样本方法：样本方法是指通过从大量数据中随机抽取一部分数据来进行统计分析。例如，在一个数据集中，我们可以从大量数据中随机抽取100个数据来计算平均值。

2.总体方法：总体方法是指通过对大量数据进行全部的统计分析。例如，在一个数据集中，我们可以对所有数据进行计算平均值。

### 3.2.3 统计的几种应用

1.数据清洗：数据清洗是指通过对数据进行预处理来消除数据中的噪声和错误。例如，在一个数据集中，我们可以对数据进行去除重复数据、填充缺失数据等操作。

2.数据分析：数据分析是指通过对数据进行分析来发现数据中的模式和规律。例如，在一个数据集中，我们可以使用线性回归来预测某个变量的值。

3.数据可视化：数据可视化是指通过对数据进行可视化表示来帮助我们更好地理解数据的特点。例如，在一个数据集中，我们可以使用条形图来表示某个变量的分布。

## 4.1 概率论

### 4.1.1 直接计算法

```python
import random

def probability_direct_calculate(n, p):
    total = 0
    for _ in range(n):
        if random.random() < p:
            total += 1
    return total / n

print(probability_direct_calculate(10000, 0.5))
```

### 4.1.2 定义法

```python
import random

def probability_define(n, p):
    total = 0
    for _ in range(n):
        if random.random() < p:
            total += 1
    return total / n

print(probability_define(10000, 0.5))
```

### 4.1.3 条件概率

```python
import random

def conditional_probability(n, p, q):
    total = 0
    for _ in range(n):
        if random.random() < p:
            if random.random() < q:
                total += 1
    return total / n

print(conditional_probability(10000, 0.5, 0.3))
```
