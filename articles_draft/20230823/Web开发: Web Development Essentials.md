# 2.Web开发概述
Web开发的任务是使用Web技术构建网站，可以分为前端、后端和数据库三部分。如下图所示。


1. **前端**：负责设计、制作和实现Web页面的用户界面。主要技术包括HTML/CSS、JavaScript、jQuery、AJAX、Bootstrap、ReactJS、AngularJS、VueJS等。
2. **后端**：负责处理客户端（浏览器）的请求，从数据库中获取数据，生成动态响应，并返回给客户端。主要技术包括Python、Java、PHP、Ruby on Rails、Node.js、ASP.NET MVC、Laravel等。
3. **数据库**：用于存储和管理数据的仓库，是网站运行的基础。主要技术包括MySQL、PostgreSQL、Oracle、MongoDB等。
一般来说，一个完整的Web应用程序由前端和后端组成，它们之间通过HTTP协议进行通信，并共享同一套数据模型。前端负责构造网站的静态页面和交互组件，后端则提供API接口、数据访问、安全保障、性能优化等后台服务功能。
Web开发通常采用前后端分离的方式，也就是说，前端人员负责网站的静态页面、视觉效果、用户交互体验，后端人员负责网站的后台服务、数据处理、安全防护等。因此，Web开发者往往分工明确，而且可以同时兼顾前端、后端开发工作。
除了上述3种技术外，还有一些Web开发常用的技术模式，比如MVC模式、RESTful API模式、Microservices架构模式等。这些模式虽然不是必需品，但对Web开发的实践方式、效率和质量都有着积极作用。另外，除了Web技术之外，还有大量其他技术也被应用于Web开发领域，如云计算、移动开发、DevOps、可视化编程等。
# 3.Web开发环境配置
## 3.1 软件安装
对于Web开发人员来说，操作系统、文本编辑器、浏览器以及版本控制工具都是开发过程中不可缺少的工具。因此，我们应该在开始开发之前，做好相应的环境配置。

- 操作系统：Web开发需要安装操作系统，一般来说，推荐使用Mac OS或者Linux操作系统。Windows操作系统下也可以使用虚拟机，比如VMWare、VirtualBox。
- 文本编辑器：推荐使用集成开发环境IDE，如Visual Studio Code、Sublime Text、Atom等。
- 浏览器：Chrome、Firefox、Safari等主流浏览器均可以开发Web应用，当然还可以安装第三方插件或开发者工具来提高开发效率。
- 版本控制工具：使用Git、SVN或Mercurial版本控制工具进行项目管理。

## 3.2 开发环境搭建
经过配置之后，我们就可以开始进行Web开发了。不过，为了能够正常编写代码，我们还需要安装必要的开发环境。

### 安装WebServer
首先，我们需要安装WebServer，也就是我们开发Web应用时使用的Web服务器软件。常用的有Apache、Nginx等。

- Apache: 可以通过Homebrew进行安装，输入以下命令即可：`brew install apache` 。
- Nginx: 可以通过Homebrew进行安装，输入以下命令即可：`brew install nginx` 。

安装成功后，我们可以通过浏览器打开 `http://localhost/` ，如果看到默认的Apache欢迎页面，则表示WebServer安装成功。

### 安装数据库
我们需要安装数据库，即我们的网站数据存储位置。常用的数据库有MySQL、Postgresql、SQL Server等。这里，我们以MySQL为例进行安装。

- MySQL: 可以通过Homebrew进行安装，输入以下命令即可：`brew install mysql` 。
- Postgresql: 可以通过Homebrew进行安装，输入以下命令即可：`brew install postgresql` 。

安装成功后，我们可以通过MySQL客户端软件连接到本地的MySQL数据库，然后通过SQL语句创建表、插入数据等。

### 配置环境变量
最后，我们还需要配置环境变量，让操作系统能够识别并调用相应的工具。比如，我们的数据库客户端可能安装在`/usr/local/bin`目录下，则我们需要添加环境变量`PATH`，使得我们可以在任何地方执行该目录下的所有可执行文件。

- 设置环境变量：编辑 `~/.bash_profile` 文件，添加以下两行内容：

  ```
  export PATH="/usr/local/bin:$PATH"
  source ~/.bash_profile
  ```

至此，Web开发环境配置完毕。
# 4.HTML(HyperText Markup Language)
HTML(超文本标记语言)，是建立在Web页面上的显示网页内容的一种标记语言。它由一系列标签和属性构成，用以描述页面的内容、结构、行为，并用于呈现给用户阅读。

HTML由两个主要部分组成：

- **文档类型声明**（Document Type Declaration，DTD），用来指定当前文档的类型和版本。

- **元素**（Element），用来定义网页的结构、样式和行为。

HTML的语法严格遵循标准通用标记语言SGML规范，具有严谨、简洁、灵活、易用等特点。HTML采用了很长的历史，由蒂姆·李维斯特（Tim Berners-Lee）和万维网联盟（World Wide Web Consortium）共同维护。由于HTML本身的特殊性，导致它不能完全独立存在，只能嵌入到其他的文档或标记语言中。

## HTML基本结构
HTML文件的基本结构如下：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
</head>
<body>
    
</body>
</html>
```

- `<!DOCTYPE>` 是 HTML 的文档类型申明，告诉 Web 浏览器用什么版本的规范来解析这个文档；
- `<html>` 表示整个 HTML 文档，`lang` 属性是用来设置网页的语言，比如中文就是 `lang="zh-CN"`；
- `<head>` 表示网页的头部信息，比如 `<title>` 标签用来设置网页的标题；
- `<meta>` 标签用来设置网页的元信息，比如编码、刷新频率、关键词、描述等；
- `<body>` 表示网页的主体内容。

## HTML标签
HTML标签提供了描述网页内容的语义化标签，使页面结构更加清晰。

常用的 HTML 标签有：

- `<div>` : 块级元素，用来包裹其他HTML元素，常用于布局；
- `<span>` : 内联元素，用来包裹文字或图片，常用于文本的修饰；
- `<a>` : 链接标签，用于链接到其他页面或资源；
- `<img>` : 插入图像标签，用于在页面插入图片；
- `<input>` : 用户输入标签，用于收集用户输入；
- `<form>` : 表单标签，用于收集、验证用户输入；
- `<table>` 和 `<tr>`、`<td>` : 表格标签，用于制作表格；
- `<ul>` 和 `<li>` : 无序列表标签，用于列出项目列表；
- `<ol>` 和 `<li>` : 有序列表标签，用于编号项目列表；
- `<style>` : 样式标签，用于定义网页的样式；
- `<script>` : 脚本标签，用于包含 JavaScript 代码。
