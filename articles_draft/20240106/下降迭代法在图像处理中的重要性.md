## 3.3 数学模型公式

下面我们给出下降迭代法在图像恢复任务中的数学模型公式。

1. 噪声信息量：在图像恢复任务中，目标函数可以是图像的均值噪声信息量（MSE）。MSE是图像中噪声能量的函数，定义为：

$$
MSE = \frac{1}{MN} \sum_{i=1}^{M} \sum_{j=1}^{N} (g_{ij} - f_{ij})^2
$$

其中，$M$ 和 $N$ 是图像的行数和列数，$g_{ij}$ 是原始图像的灰度值，$f_{ij}$ 是恢复后的图像的灰度值。

2. 梯度下降法：在图像恢复任务中，可以使用梯度下降法实现像素点值的更新。更新规则是像素点的新值为其当前值减去一个学习率乘以梯度。 matlab 代码如下：

```matlab
learn_rate = 0.01; % 学习率
for iter = 1:max_iter
    for i = 1:M
        for j = 1:N
            gradient = -2 * (g(i, j) - f(i, j)); % 计算梯度
            f(i, j) = f(i, j) - learn_rate * gradient; % 更新像素点值
        end
    end
    error = mean(mean((g - f).^2)); % 计算目标函数的变化率
    if error < threshold
        break;
    end
end
```

% 定义目标函数：均值噪声信息量
M = size(gray_img, 1);
N = size(gray_img, 2);
mse = @(f) mean(mean((gray_img - f).^2));

% 初始化恢复后的图像
f = zeros(M, N);

% 设置学习率和收敛阈值
learn_rate = 0.01;
threshold = 1e-6;

% 设置最大迭代次数
max_iter = 100;

% 进行下降迭代法恢复处理
for iter = 1:max_iter
    for i = 1:M
        for j = 1:N
            gradient = -2 * (gray_img(i, j) - f(i, j)); % 计算梯度
            f(i, j) = f(i, j) - learn_rate * gradient; % 更新像素点值
        end
    end
    error = mse(f); % 计算目标函数的变化率
    if error < threshold
        break;
    end
end

% 输出恢复后的图像
imshow(f, []);
