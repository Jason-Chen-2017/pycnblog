### 4.1策略模拟代码实例

```python
import random

def strategy_simulation(strategies, iterations):
    results = []
    for _ in range(iterations):
        strategy = random.choice(strategies)
        result = strategy()
        results.append(result)
    return results
```

1. 首先导入了 `random` 模块，用于生成随机数。

3. 在函数体内，首先创建了一个空列表 `results`，用于存储模拟过程的结果。
4. 然后使用一个 `for` 循环进行模拟过程，循环次数为 `iterations`。在每次循环中，首先随机选择一个策略，然后调用该策略的 `()` 方法，并将结果添加到 `results` 列表中。
