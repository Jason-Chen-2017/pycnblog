# 2. 核心概念与联系

贝塔分布是一种连续概率分布，用于描述一个随机变量的取值范围内的概率分布。贝塔分布的概率密度函数（PDF）定义为：

$$
f(x) = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha) \Gamma(\beta)} \cdot \frac{x^{\alpha - 1} \cdot (1 - x)^{\beta - 1}}{\beta^\alpha \cdot (1 - \beta)^\beta}
$$

其中，$\alpha$ 和 $\beta$ 是贝塔分布的参数，$\Gamma$ 是伽马函数。

在自然语言处理中，贝塔分布的应用主要有以下几个方面：

## 4.2 文本分类

我们以一个简单的文本分类任务为例，使用贝塔分布进行文本分类。

```python
# 训练数据
train_data = [
    ("I love this product", "positive"),
    ("This is a bad product", "negative"),
    ("I am happy with this purchase", "positive"),
    ("I am disappointed with this purchase", "negative")
]

# 测试数据
test_data = [
    "I am not satisfied with this product",
    "I am very happy with this purchase"
]

# 贝塔分布参数估计
positive_alpha = 0
positive_beta = 0
negative_alpha = 0
negative_beta = 0

for text, label in train_data:
    if label == "positive":
        positive_alpha += len(text)
        positive_beta += len(text)
    else:
        negative_alpha += len(text)
        negative_beta += len(text)

# 测试数据分类
for text in test_data:
    positive_prob = beta.pdf(1, positive_alpha, positive_beta)
    negative_prob = beta.pdf(1, negative_alpha, negative_beta)
    
    if positive_prob > negative_prob:
        print(f"{text} -> positive")
    else:
        print(f"{text} -> negative")
```
