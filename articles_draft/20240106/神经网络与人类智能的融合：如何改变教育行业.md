# 2. 核心概念与联系

## 2.2 神经网络

神经网络是一种模拟人类大脑结构和工作方式的计算模型。它由多个节点（神经元）和它们之间的连接（权重）组成。每个节点都可以接收输入信号，进行处理，并输出结果。神经网络通过训练来学习，训练过程中会调整权重，以便更好地处理输入信号。

# 3. 核心算法原理和具体操作步骤以及数学模型公式详细讲解

在这一部分，我们将详细讲解神经网络的核心算法原理、具体操作步骤以及数学模型公式。

## 3.1 神经网络的基本结构

一个典型的神经网络包括以下几个部分：

1. 输入层：用于接收输入数据的节点。
2. 隐藏层：用于处理输入数据的节点。
3. 输出层：用于输出处理结果的节点。

每个节点之间通过权重连接，权重表示节点之间的关系。

## 3.2 激活函数

激活函数是神经网络中的一个关键组件，它用于控制节点的输出。常见的激活函数有：

1. 步函数：$$ f(x) = \begin{cases} 1, & x \geq 0 \\ 0, & x < 0 \end{cases} $$
2.  sigmoid 函数：$$ f(x) = \frac{1}{1 + e^{-x}} $$
3.  hyperbolic tangent 函数（tanh）：$$ f(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}} $$
4.  ReLU 函数：$$ f(x) = \max (0, x) $$

## 3.3 前向传播

前向传播是神经网络中的一个关键操作，它用于计算节点的输出。具体步骤如下：

1. 对输入层的每个节点，将输入数据作为其输入。
2. 对隐藏层的每个节点，计算其输入为：$$ x_i = \sum_{j=1}^{n} w_{ij} \cdot a_j + b_i $$
3. 对隐藏层的每个节点，计算其输出为：$$ a_i = f(x_i) $$
4. 对输出层的每个节点，计算其输入为：$$ x_i = \sum_{j=1}^{n} w_{ij} \cdot a_j + b_i $$
5. 对输出层的每个节点，计算其输出为：$$ a_i = f(x_i) $$

## 3.4 后向传播

后向传播是神经网络中的一个关键操作，它用于更新节点的权重。具体步骤如下：

1. 对输出层的每个节点，计算其误差：$$ \delta_i = f'(x_i) \cdot (t_i - a_i) $$
2. 对隐藏层的每个节点，计算其误差：$$ \delta_i = f'(x_i) \cdot \sum_{j=1}^{m} w_{ij} \cdot \delta_j $$
3. 更新节点的权重：$$ w_{ij} = w_{ij} - \eta \cdot \delta_i \cdot a_j $$

import numpy as np

# 定义激活函数
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def sigmoid_derivative(x):
    return x * (1 - x)

# 定义神经网络
class NeuralNetwork:
    def __init__(self, input_nodes, hidden_nodes, output_nodes):
        self.input_nodes = input_nodes
        self.hidden_nodes = hidden_nodes
        self.output_nodes = output_nodes

        # 初始化权重
        self.weights_input_hidden = np.random.rand(self.input_nodes, self.hidden_nodes)
        self.weights_hidden_output = np.random.rand(self.hidden_nodes, self.output_nodes)

    def forward(self, input_data):
        # 前向传播
        self.hidden_activation = sigmoid(np.dot(input_data, self.weights_input_hidden))
        self.output_activation = sigmoid(np.dot(self.hidden_activation, self.weights_hidden_output))

    def backward(self, input_data, output_data):
        # 后向传播
        output_errors = output_data - self.output_activation
        output_delta = output_errors * sigmoid_derivative(self.output_activation)

        hidden_errors = np.dot(self.weights_hidden_output.T, output_delta)
        hidden_delta = hidden_errors * sigmoid_derivative(self.hidden_activation)

        # 更新权重
        self.weights_hidden_output += np.dot(self.hidden_activation.T, output_delta)
        self.weights_input_hidden += np.dot(input_data.T, hidden_delta)

# 训练神经网络
