## 4.1 代码实例

我们以一个简单的智能建筑案例为例，实现一个简单的能源管理系统。

### 4.1.1 数据收集与预处理

我们假设一个建筑物有多个能源设备，如电源、热源等，每个能源设备都有一个传感器，用于实时收集能源使用量的数据。我们可以使用 Python 编程语言，编写一个简单的数据收集与预处理程序，如下所示：

```python
import time
import random

class Sensor:
    def __init__(self, id):
        self.id = id
        self.data = []

    def collect_data(self):
        self.data.append(random.randint(0, 100))
        print(f"Sensor {self.id} collect data: {self.data[-1]}")

def preprocess_data(sensors):
    data = []
    for sensor in sensors:
        data.append(sensor.data[-1])
    return data

sensors = [Sensor(i) for i in range(5)]
for _ in range(10):
    for sensor in sensors:
        sensor.collect_data()
    time.sleep(1)

data = preprocess_data(sensors)
print(data)
```

### 4.1.4 整体代码

将上述三个程序整合在一起，得到一个简单的智能建筑能源管理系统的整体代码，如下所示：

```python
import time
import random

class Sensor:
    def __init__(self, id):
        self.id = id
        self.data = []

    def collect_data(self):
        self.data.append(random.randint(0, 100))
        print(f"Sensor {self.id} collect data: {self.data[-1]}")

def preprocess_data(sensors):
    data = []
    for sensor in sensors:
        data.append(sensor.data[-1])
    return data

def analyze_data(data):
    total = sum(data)
    average = total / len(data)
    print(f"Total: {total}, Average: {average}")
    return average

def make_decision(average):
    if average > 80:
        return "Turn off some lights."
    else:
        return "Keep the lights on."

if __name__ == "__main__":
    sensors = [Sensor(i) for i in range(5)]
    for _ in range(10):
        for sensor in sensors:
            sensor.collect_data()
        time.sleep(1)

    data = preprocess_data(sensors)
    average = analyze_data(data)
    decision = make_decision(average)
    print(f"Decision: {decision}")
```
