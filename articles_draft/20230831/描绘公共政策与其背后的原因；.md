# 1.简介
  

作为政府机构，公共政策和政策制定过程既可以说是最复杂的也是最值得期待的事情之一。随着现代社会的复杂性提升、经济转型不断加剧、人口老龄化程度日益高涨等诸多因素的影响，各国都面临着一个很重要的问题——如何将国家治理结构以及行政决策权力纳入到应对新形势、实现发展目标等大背景下宏观政策调控体系中。而有效执行政府部门的政策往往需要通过制定公共政策实现。本文将围绕此问题进行探讨，阐述我国公共政策制定的关键环节及其背后所蕴含的社会、经济、政治等多方面的原因。

# 2.基本概念、术语与定义
## 2.1 公共政策制定
公共政策制定主要指政府机关按照国务院或地方政府授权的内容、要求和方向制定政策措施，提出决策和作出的具体方案，以保障公民享有平等受教育的权利、参与公共事务的自由、获得充分就业、平等参与公共事物的权利、推动经济社会发展的活力、实现全体公民的共同富裕。公共政策的制定具有决定性意义，其作用在于帮助政府适应社会、经济、政治变革需求，更好地服务于公众，促进社会稳定、经济增长。

## 2.2 公共政策主体
公共政策制定的主体包括政府、社区组织、企业、个体户、个体经营者等。一般来说，公共政策制定由政府直接主导，但也存在非政府组织、企业、个体户等制定公共政策的情况。公共政策制定的主体既要兼顾各方利益，又要注意考虑到公共政策的效益和效能。

## 2.3 公共政策的层次与分类
公共政策按制定时间、类型、性质、对象、范围和职责可分为五层。

1.计划层：计划层的公共政策由政府制定长远目标并实施相应的政策措施，如设立专项计划、制定长远规划、制定战略计划、制定生态系统治理计划等。计划层公共政策旨在实现政府的整体目标，即促进经济增长、保持经济运行良好、维护社会安定、保障公民权利和尊严。计划层的公共政策涉及政府管理部门，如财政部、国土资源部、卫生部、教育部、科技部等。

2.政策层：政策层的公共政策制定一般是短期目标下的政策建议，涉及政府管理部门的具体政策议题，如债务管理、金融市场管理、养老保险制度、教育发展、城市维护等。政策层公共政策旨在改善社会环境、提升个人生活品质，为公民提供便利和服务。政策层公共政olicy往往收效甚微，往往会成为被忽视甚至是误导性的政策，需要政府建立更多的监督机制来实现公共政策的实质性改进。

3.规范层：规范层的公共政策由政府部门颁布行业标准或规则，提出法律、制度或组织的规范要求。规范层的公共政策是基于公众的需求，旨在确保公众的合法权益得到充分保护。规范层的公共政策主要涉及公共管理、社会政策、法律法规等部门。

4.战略层：战略层的公共政策是指政府为在某个领域开辟新的发展道路或更好地满足消费者需求而制定的政策。战略层公共政策旨在发挥政府在全局治理中的优势，推动经济全球化和区域协调发展。战略层的公共政olicy往往是一切政策的集中体现，一旦失去了发展方向，就会被淘汰或边缘化。

5.操作层：操作层的公共政策是在特定事件发生或特殊时期政府为了解决某些突发问题所采取的一系列具体措施。如救灾、疫情防控、环境保护等。操作层的公共政策的设计往往相对简单，但却给公众带来重大的影响。操作层的公共政策常常与其他层级的公共政策相互矛盾或重复，同时也可能因为过度滥用而造成政策效果的偏差。

## 2.4 公共政策的制定方式
公共政策制定的方式有多种。

一是法律法规。最早的公共政策制定方式则是由法律法规规定。例如，1949年《巴黎公约》规定，工人阶级享有平等权利，但绝对不得妨碍劳动者的集体经济、依靠雇佣关系谋求福利和报酬。1980年《物联网法》规定，在物联网应用场景中，政府应当对用户隐私权保护予以关注。2017年《数据安全法》规定，要保障公民数据信息安全，包括收集、存储、使用、共享、保护等方面要遵守法律法规。此外，还有一些其他规定或法律文件还可以用来制定公共政策。

二是自治条例。自治条例是一种公共政策制定的自上而下途径，它允许公共机构或团体自己做出决策，并根据自己的利益制定相应的政策。自治条例的制定往往比较自由和独立，它们可以创造条件让不同的机构或团体之间做出自己认为正确的选择。自治条件可以是单一条款，也可以是具有一定规模的共识，因此，它们制订的政策具有较高的合理性。例如，乡镇村委会可以制订自己的自治条例，包括本地政府关于土地利用的规定，或者采用县令或直隶省政府提供的举措来保护村民的合法权益。

三是规则性文件。虽然自治条例能够创造条件让不同机构或团体做出自己的选择，但它们的特点是缺乏可操作性和长久性，且容易变更。规则性文件是由全社会共同制订的公共政策，它的制定具有法律效力，且对所有相关方均有约束力。这种文件通常由专门的公共政策制定机构或机构代表，如“美国联邦储备委员会”、“日本财政厅”等，制定出台与财政收入密切相关的支出限制、税收补贴、货币贬值的政策。规则性文件的优点是具有较强的制度约束力和持续性，可以创造长久的影响。

四是选举程序。为了更好地分配公共政策资源，减少腐败，各级政府往往设置选举程序来产生新的公共政策主张。选举程序一般周期较长，而且产生公共政策的影响也有限。目前，中国的选举程序分为立法（全国人大）、国资委、人大常委会、政府工作报告、人民代表大会等七种。人大常委会每两年召开一次，有2/3席为常委，共40名议员，其中常委一人为君主，其他19名为臣子。此外，还有一些地方政府也通过选举产生自己的公共政策。

五是会议制度。与其他形式相比，会议制度往往更加随意，更具包容性，可以使不同利益群体或群体之间的意见交流畅通。2018年9月，美国国会就提出“与中国分享休闲旅游开放领域”这一议案，国会通过该议案获得“众院赞成”，引起轩然大波。会议制度有助于促进公众对政府决策的知情权、参与权、表达权，以及政策的参与热情，并促使公众充分参与公共政策进程。

六是意见征集制。这是一种在公众中广泛传播、收集和反馈政策建议的方法。2014年，北京市政府启动了首期意见征集制试点，意在鼓励和支持在公众中透明公开和参与公共政策的讨论。意见征集制的一个突出特征是将信息以问答形式呈现给公众，使公众有机会表达个人观点、意见和建议，从而形成积极的公共政策信息。

七是众筹制。众筹制是一种在公众的一定预算内募集资金的制度。它可以激励公众投入资金，同时也有助于促进公众对政府决策的知情权、参与权、表达权，以及政策的参与热情。众筹制的典型例子就是美国种族仇恨犯罪中心的“回声室”活动，一群受害者将捐赠一些公款，希望通过捐款，反映他们的诉求，改变局势。

1.发现问题：公共政策是一个动态的过程，它将随着社会的变化而不断更新、完善。在制定公共政策之前，应该认清当前公共政策制定存在的问题，以便更好地决策和规划公共政策的发展方向。公共政策的发现可以从以下三个方面进行：
   - 概念和名词：识别并分析当前政策制定的概念和名词，并制定更新的概念和名词，以反映政策制定方向。
   - 目标和制定：梳理现有的政策目标，并确定新的政策目标。制定新的政策目标，并确定相应的政策措施。
   - 实际项目：跟踪和分析当前政策实际运行情况，制定政策措施，以符合政策目标。

2.研究背景：公共政策制定始终需要考虑政策背景。研究公共政策制定背景的目的在于了解公共政策制定时公共政策制定者的意愿、利益和追求。理解当前公共政olicy制定的背景，可以帮助制定更加合理的公共政策。
   - 制定者背景：询问制定公共政策的专家，了解制定者的兴趣爱好、精神状态、专业知识、家庭背景、职业以及感情状况。
   - 社会背景：了解制定公共政策的社会环境，包括民众的生活满意度、工作满意度、公共参与意识和向往等。
   - 国际背景：研究制定公共政策的国际环境和国际发展趋势，尤其是公共政策制定的层次、速度、范围和空间等。

3.研究现状：研究现有公共政策制定的经验和理论，包括学习、调查、比较、研究、分析等。研究现有公共政策制定的经验，可以帮助公共政策制定者制定更有针对性的政策措施。
   - 制度和法规：检查法律法规、制度和规范，确认现有公共政策是否符合制度和规范。
   - 研究结果：阅读其他国家、政府部门、学者的研究成果，学习民众对公共政策的看法。
   - 数据统计：收集、分析和呈现有关数据，如经济、社会、环境、健康、教育、基础设施、房价、医疗等，以帮助公共政策制定者分析现状。

4.制定计划：制定计划的目的是根据研究结果和当前的国际形势，制定公共政策制定的规划。制定计划后，将政策的发展目标和实施计划展示给公众。
   - 制定目标：制定公共政策发展的目标，并列出优先级。
   - 制定时间表：制定计划的实施时间表。
   - 确定资金来源：列出财政资金来源和政府支持的项目，确定资金和项目配比。
   - 分配资源：分配资源，确定政策优先级和难度级别。

5.制定策略：制定策略的目的是制定公共政策的实施计划。制定策略之后，公共政策专家就可以开展公共政策实施工作。
   - 制定措施：制定公共政策的具体措施。
   - 提供支持：向支持的政府部门提供资金和资源，以帮助实现政策目标。
   - 测试实施：测试实施公共政策，并将实施情况和效果反馈给专家。

6.评估和跟踪：在实施公共政策期间，需要跟踪和评估实施情况。评估过程中，公共政策专家可以分析实施情况，并根据情况调整实施计划。
   - 制定跟踪机制：制定跟踪机制，包括实施进度、完成情况、反馈等。
   - 实施数据统计：收集和分析实施数据的数量、质量、范围、效率、交付率、政策效果等。
   - 对照目标：分析实施效果和目标，并总结经验教训。
   - 总结与检视：对制定的公共政策进行总结，检视其与目标的完美结合。

# 5.未来发展趋势与挑战
公共政策制定面临着许多挑战。其中最突出的挑战是如何在短时间内快速且准确地制定出足够多的、真正能够落实的公共政策。另外，公共政策制定过程需要承担的角色越来越多，如公共关系、媒体采访、政策研究等。这些因素都会影响公共政策的制定过程。例如，公众的不满可能会激发公共政策制定者制定反对意见，这会影响公共政策的落实。另外，公共政策制定也面临着公共舆论的压力和反弹，这会影响公共政olicy制定者制定的政策效果。

在未来，公共政策制定应该更加注重公共品牌建设，以增强公众对政府的信任和依赖，提升政策的可信度和代表性。同时，公共政策制定者需要学会分析、解读和运用多种渠道获取的信息，让公共政策制定更加科学、客观、专业。最后，公共政策制定者还应该加强监督管理，确保公共政策制定的全面性、准确性和有效性。

# 6.附录常见问题与解答
1.为什么要制定公共政策？
公共政策制定是为了促进社会经济发展、保障公民权利、保障社会稳定、促进公共利益。公共政策制定能够提高社会的公共服务水平，促进经济的发展，为公共利益创造公平可靠的财富基础。公共政策制定可以为公共事务创造新的内在规则，也为公共服务提供更好的方式。

2.什么是公共政策？
公共政策是政府部门或社会团体根据其内部的目标、任务、方向以及规定，制定和修改的政策措施。公共政策是一份由政府颁布的文件，设定政府的管理和运作方针，包括财政、税收、教育、社会保障、金融、就业、社会事务、外交政策等方面。

3.如何衡量公共政策的效益？
公共政策的效益是指公共政策的社会和经济效益，它是衡量公共政策成功与否的重要标准。比如，一个政策的社会效益是指政策对社会的贡献，比如老百姓的福祉、平安、幸福。一个政策的经济效益是指政策对经济的贡献，比如经济的增长、就业的增加、人民的收入增加等。

4.哪些因素会影响公共政策的制定？
公共政策的制定过程受到多方面的影响，如经济、社会、政治、法律等因素的影响。其中，经济因素影响着公共政策制定过程，社会因素影响着公共政策制定人士的思维方式和风格，政治因素影响着公共政策制定的政治前景，法律因素影响着公共政策的效能和正确性。

5.公共政策制定过程有哪些阶段？
公共政策制定过程一般分为六个阶段。第一阶段是发现问题，第二阶段是研究背景，第三阶段是研究现状，第四阶段是制定计划，第五阶段是制定策略，第六阶段是评估和跟踪。
