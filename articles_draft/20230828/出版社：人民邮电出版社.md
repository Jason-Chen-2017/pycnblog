## 3.1 最大子序列问题
给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其和。例如，给定 [-2,1,-3,4,-1,2,1,-5,4], 返回 6 。

**贪心法：**

1. 初始化 max_sum 和 curr_sum 为 nums[0] 。
2. 在遍历 nums 的过程中，对于当前位置 i ，若 curr_sum + nums[i] >= 0 ，则增加 curr_sum += nums[i]；否则，对比 curr_sum 和 max_sum，并更新 max_sum 和 curr_sum 。
3. 最后更新 max_sum 为全局的最大值。
```python
def maxSubArray(nums):
    if not nums:
        return None
    
    # Initialize variables for the algorithm
    max_sum = float('-inf')
    curr_sum = 0

    # Iterate through the list and update the variables accordingly
    for num in nums:
        curr_sum += num
        
        if curr_sum > max_sum:
            max_sum = curr_sum
            
        if curr_sum < 0:
            curr_sum = 0
    
    return max_sum
```

时间复杂度 O(n)，空间复杂度 O(1)。

**Kadane's Algorithm**：

在 Kadane's Algorithm 中，使用动态规划的方法，记录下当前位置的最大子序列和。如果当前位置的子序列和比之前的最大子序列和大，则更新最大子序列和。

1. 初始化 max_sum 和 curr_sum 为 nums[0] 。
2. 在遍历 nums 的过程中，对于当前位置 i ，更新 curr_sum 为 max(curr_sum + nums[i], nums[i]) 。
3. 更新 max_sum 为全局的最大值。
```python
def maxSubArray(nums):
    if not nums:
        return None
    
    # Initialize variables for the algorithm
    max_sum = nums[0]
    curr_sum = nums[0]

    # Iterate through the list and update the variables accordingly
    for i in range(1, len(nums)):
        curr_sum = max(curr_sum + nums[i], nums[i])
        max_sum = max(max_sum, curr_sum)
    
    return max_sum
```

时间复杂度 O(n)，空间复杂度 O(1)。

## 3.2 欧拉函数和阶乘

欧拉函数 e (Euler's number) 是无穷远点的数量，即数轴上所有连线上的点的个数。E = Σ k=0 to infinity of k! = 1 × 2 ×... × k。显然，e 是一个无理数，因此无法用浮点数表示。但是，可以使用任意精度的有理数表示形式，如：

- 使用分数形式：e ≈ (lim n->∞ sum_{k=0}^n 1/k!)
- 使用连分数形式：e = lim x->1 (1 - x)/(x - 1)^n

### 求阶乘
阶乘就是从一个数字乘积所有的正整数。例如 5！ = 5 × 4 × 3 × 2 × 1 = 120。

递归方法求解阶乘：

```python
def factorial(num):
    if num == 0 or num == 1:
        return 1
    else:
        return num * factorial(num - 1)
```

迭代方法求解阶乘：

```python
def factorial(num):
    fact = 1
    while num!= 0:
        fact *= num
        num -= 1
    return fact
```

时间复杂度 O(n)，空间复杂度 O(1)。

### 求根号

#### 开平方根

开平方根，又称平方根或者二次根，是指在正数平方数轴上某个点 x 对直角坐标系作垂线时，与该点垂直的一条线段上的一点。由于该直角坐标系上点的排列关系，存在唯一的某个点，使得在该点与坐标轴相切的这条线段上再画两点所围成的正方形内切单位圆。根据勾股定理，我们可以计算出相应的半径 r。所以当两个点之间的距离等于 r 时，它们就处于同一水平线上。又因为曲线是有弦的，所以另一条半径相同的曲线是不存在的。

开平方根是个非常重要的数学运算。很多常用函数（如 `sqrt` 函数）都是基于开平方根的算法。以下给出两种求开平方根的方法：

**Babylonian Method:** 

Babylonian Method 基于牛顿迭代法，先猜测一个初始值的开平方根，然后重复迭代，新值等于旧值减去函数值除以值的商。这里使用的函数是 `(y + x/y)/2`，其中 y = x/2^k，k 为不断加 1 后得到的值。

```python
import math

def sqrt(num):
    guess = num / 2   # Guess a starting value
    while abs((guess*guess - num)) > 0.0000001:    # Iterate until converges within tolerance
        guess = (guess + num/(2*guess))/2   # Update the guess
        
    return guess    
```

**Newton Raphson Method:**

Newton Raphson Method 也属于牛顿迭代法。这个方法的思路是：首先选择一个初始点 z0 来作为近似值，计算 f(z0)、f'(z0) 和 f''(z0)。然后，设置一阶导数导数值为负函数值的倒数，即 dz/dt=-1/f(t)dx/dz，得到 z1=z0-f(z0)/f'(z0)。重复以上过程直到满足某一条件（如迭代次数或误差范围），即可得到最终近似值 z。

```python
def sqrt(num):
    def func(x):      # Define the function whose root we want to find
        return x*x - num  
        
    def deriv(x):       # Compute first derivative of the function
        return 2*x
        
    guess = num/2      # Initial guess at square root
    epsilon = 0.000001  
    iterations = 1000
     
    while abs(func(guess)) > epsilon and iterations > 0:    # Iterate until convergence is achieved
        guess -= func(guess) / deriv(guess)              # Use Newton-Raphson formula
        iterations -= 1
         
    return round(guess, 10)        # Round off the result to 10 decimal places
```

时间复杂度 O(log n)，空间复杂度 O(1)。
