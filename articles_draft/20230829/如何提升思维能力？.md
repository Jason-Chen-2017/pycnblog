# 1.简介
  

思维能力是指对事物进行分析、决策、解决问题的能力。越来越多的人喜欢做一些抽象、概括性的活动，比如说对某件事情作一个类比或模型。然而这些活动往往还不足以掌握真正的问题本质和核心。因此，如何通过学习、应用新颖的思维方式和方法，加强自身的思维能力就成为每个人都需要面对的课题。
在这个系列的第17篇文章中，笔者将从以下几个方面对提升思维能力进行探讨：

1) 一问一答式问答
2）逻辑思维训练法
3）拓展视野法
4）发散思维方法
5）生活领域中的思维训练技巧
6）历史与哲学的思维训练方法
7）学习的过程要刻意引导自己，不要轻举妄动。

这些方法是不是都能够帮助到读者提升思维能力呢?让我们一起来看看吧！

## 一问一答式问答
首先，回答一些简单但却有意思的提升思维能力的问题吧!
### 1.你有哪些提升思维能力的方法？可以分享一下吗？

答：目前我最喜欢的一句话就是——“什么都可以尝试一下”。试着去发现别人的优秀之处、分析自己的弱点并以此作为自己进步的阶梯。同时也要多和朋友们交流，了解他们的思维方式、擅长的领域等等，一起讨论、分享自己的想法。当然，还有许多其他的方法，比如逻辑思维训练法、拓展视野法、发散思维方法、生活领域中的思维训练技巧等等。

### 2.除了思维能力，你还想分享你有哪些其他的方法来提高你的工作效率？

答：除了思维能力外，我认为工作效率同样也是每天都应当重视的一个重要能力。工作效率可以通过以下方法来提升：

1. 减少不必要的沟通，避免信息过载。可以采用自动化工具，将重复性工作自动化处理，避免因工作繁忙、错过重要时机而造成的遗漏；

2. 建立合适的组织结构。合理地分配任务、管理时间、完善项目管理工具；

3. 制定目标清晰的任务列表。明确任务的优先级、交付标准、追踪进度；

4. 使用可视化工具。制作任务板、Kanban board、任务网络图等，更直观地显示任务之间的依赖关系和关系密切程度，帮助管理人员快速识别关键路径、瓶颈环节；

5. 提倡主动承担责任。当遇到问题无法自行解决时，需要及时跟上反馈，主动寻求帮助，而不是等着其他人替代；

6. 注重细节。每一项工作都需要严格执行细则和流程，包括表单填写、文件格式、文档编排等，确保工作质量；

7. 培养个人品牌。花费精力打磨个人形象，展示自己的才华，增强个人魅力；

以上七条，基本可以作为提升工作效率的手段。

## 逻辑思维训练法
逻辑思维训练法（Logical Thinking Training Method，LST），是一个由马克·吐温创立的启蒙教育方法。它基于几何思维方式，通过教授学生用词、语法、阅读、辨析和分类等逻辑思维方法，逐步培养学生运用逻辑推理能力、理解世界、改善判断、提高智商和记忆力。其主要步骤如下：

1. 课堂会议。LSTD从普及科学理论开始，循序渐进地深入浅出地教授学生。在每堂课开头，老师先布置一道练习题目，学生应当认真完成，然后提交给他人评分，最后互相比较得出评价结果，进一步熟悉教材的内容。每堂课结束后，学生根据自己对教材内容的理解，从字面上复述或者给出自己的答案，反思自己是否正确。这样可以帮助学生学习知识，掌握新鲜感。

2. 讨论。在每堂课之后，LSTD邀请两至三名学生参与讨论。在这期间，老师使用具体例子，演示各种逻辑推理和分析问题的能力。讨论过程中，学生应该提出疑问、批判性地分析、陈述自己的观点、提供不同的解读，帮助老师理解学生的想法，促使自己更好地理解知识。

3. 期末考试。每位学生要针对已掌握的知识、理论、技能，在教材基础上进行一场完整的逻辑思维和解决实际问题的考试。考试一般不超过三个小时，考生要在规定的时间内，完成相应的逻辑思维题目，并写出自己的分析、观点和解答。在之后的一段时间里，大家各抒己见，互相交流，互相鼓励，共同进步。这不仅能够帮学生理解更多的内容，而且还能锻炼学生的表达能力、团队合作精神和社会交际能力。

## 拓展视野法
拓展视野法（Expand Your Horizon Method，EHT）是一种增强个人思考能力的方法，由美国儿童心理学家苏珊·韦恩哈特创立。该方法的基本原理是将所学到的知识系统化，再以新的视角、不同的角度重新审视，然后根据自己的理解，综合利用这些知识构建新的见解。方法具体包括四个步骤：

1. 浏览：在阅读学习时，要注意留意课文中的相关知识点，尤其是那些较难或经典的观点。在学习的过程中，应该记录下自己比较感兴趣的观点，并在课后总结出来。

2. 演讲：把自己所学到的知识以不一样的视角进行讲解。拓展视野法所谓的“不一样”，就是用不同的角度观察和研究课文中的知识。为了达到这种效果，学生可以在课前选取一个问题作为突破口，然后根据自己的理解来阐述。例如，如果觉得课上的观点缺乏实践性，就可以让学生自己去构想实验，从而验证自己的观点。

3. 创建模型：拓展视野法要求学生从现实世界中抽象出概念和模式。所以，在教学过程中，学生应该能够通过绘画、数学建模等工具，创建自己的模型。这种模型既可以证明自己的观点，又能激发学生的思考。

4. 模拟测试：拓展视野法要求学生在课余的时间内，利用自己所学到的知识来解决实际问题。所以，可以安排一些经典问题，模拟测试学生的能力。学生在解决这些问题时，应该能够运用自己学到的知识，找出解决方案。并在课后把自己的解决方案和正确答案进行比较，以检验自己的能力。

## 发散思维方法
发散思维法（Expanded Mindset Method，EM）是一种培养学生发散思维的有效方式。它认为，只有当学生从内心发展，把各种可能性纳入考虑时，才能真正掌握复杂的知识和技能。因此，在学习的过程中，需要带着问题思考，充满创造性。具体来说，发散思维法有以下几个要素：

1. 挑战习惯。要教会学生自己去承受挑战，做一个独立的人。在学习过程中，要向他们传授知识，而不是被迫接受。

2. 鼓舞动力。要鼓励学生挑战自己，找到内在驱动力，坚持刻苦学习。

3. 自我检视。要以自我审视的方式来检视自己，仔细分析自己的学习行为。一旦发现差距，要及时调整自己的学习方式。

4. 清晰目标。发散思维法要求学生以直觉、感觉和直观的方式思考问题，所以要设定清晰的学习目标。对目标的定义也很重要。

## 生活领域中的思维训练技巧
生活领域中，如何提升思维能力呢？这里分享一些生活领域中的思维训练技巧吧！

1. 以“为什么”开始学习。很多时候，我们容易陷入问题的表象和假象之间，而忽略了原因。以“为什么”开始学习，可以帮助我们更全面地理解这个世界。

2. 抽象思考。在生活中，我们很容易陷入琐碎的事务，而忽略了整体。抽象思考能帮助我们洞穿其中，看到全局。

3. 从问题中学习。生活中会遇到各种问题，不断尝试和思考才能解决。面对问题，我们需要保持开放心态，在不同层次上思考。

4. 意识到自己正在被激发。人类活在复杂的社会中，往往有许多潜在的机会，而我们却往往被诱惑着去选择最具吸引力的、最有挑战性的事情。意识到自己正在被激发，其实可以帮助我们改掉内心的习气。

5. 学习实践。生活中有许多任务是无法轻松完成的，而我们却往往过于乐观。通过实践，我们可以了解到其他解决方法的优缺点。

6. 创造性思维。生活中的任何事情都不可避免地包含了无限的可能性，我们需要宽容地去面对。在解决问题时，创造性思维可以帮助我们找到独特的思路。

## 历史与哲学的思维训练方法
对于历史与哲学这两个领域，我们有什么思维训练技巧可以参考呢？

1. 关注关键事件。古希腊、罗马、近代、工业革命以及近年来的科技革命，都是深刻影响我们的历史。跟随历史进程，学会关注关键事件，可以学到一些观念上的转变。

2. 反思历史。学会反思历史，可以发现我们过去的错误，分析它们产生的原因，树立新的正确观点。

3. 回顾曾经的历史。古今中外的历史故事都会给人带来启发，有利于我们认识真理。通过回顾历史，可以弥补自己的不足，重新塑造人生观。

4. 以知人善治国为己任。亲近他人，学会与人沟通，树立共识，助人为乐。治国安邦，这是每个人应尽的义务。

5. 对事物保持敬畏。任何事物都有其深刻的意义，但不要盲目崇拜。了解事物的真相，保持对它的敬畏，才是更加宝贵的财富。

## 学习的过程要刻意引导自己，不要轻举妄动
最后，笔者想说的是，在学习的过程中，不要轻易放弃，不要刻意控制自己，不要徘徊在熟悉的轨道上。一味地孤陋寡闻地学习只会让人迷失方向，失败的危险也就不可避免。只有勇于探索、开拓，才能做到知行合一。
