# 1.简介
  

如今移动互联网的崛起给传统互联网行业带来了新的机遇和挑战。在这个信息爆炸的时代，用户对于商品、服务和营销信息的接受能力都变得更加迅速、多样化。因此，传统的互联网公司需要从根本上进行转型升级。相比于传统互联网行业的独裁模式，移动互联网给创业者带来的机会更多、更宽阔。然而，如何开拓创新空间、找到市场需求点、建立自己的优势并不容易。

为帮助创业者们开启一个新的创业之旅，我们整理了《移动互联网时代的创业之路》，面向的是对这一新领域感兴趣、有所准备的技术人才，并试图通过分享自己的理解、洞察、经验及见解等方面，让创业者们在关键时刻获得最大限度的收益。

# 2.移动互联网介绍
## 什么是移动互联网？
移动互联网（Mobile Internet）是指通过无线网络连接的个人计算机、便携设备或手机等移动终端，利用互联网接入和应用功能的网络服务。其主要特点包括：

1. 快速且广泛覆盖
	移动互联网能够提供的信息、服务和体验近乎随身携带。用户可以随时随地访问任意网站、应用程序或购物应用。

2. 低成本
    随着移动设备的普及，消费者越来越多地使用他们的移动设备，使得移动互联网的价值显著提升。它不仅降低了运营成本，而且可以节省资金、提高效率。

3. 普适性强
   用户可以通过各个平台、设备及距离访问同一套服务，以满足各种需求。同时，平台也可以根据用户的习惯及偏好做出调整，将品牌推向每个细分市场。

## 移动互联网的功能
移动互联网为用户提供了以下功能：

1. 通信
	通过短信、即时通讯、电话、视频聊天、社交媒体及其他方式进行通信。

2. 娱乐
	通过游戏、视频剪辑、音乐播放、美食推荐、免费 Wi-Fi 和便利贴等方式享受轻松愉快的娱乐活动。

3. 购物
	通过商家提供的移动购物APP进行快速、方便的购物体验，无需携带大额现金。

4. 日常生活
	用户可通过地图导航、共享照片、阅读新闻及其它日常生活用品。

5. 旅游
	用户可以利用免费 Wi-Fi 及酒店预订、旅游攻略、门票优惠等功能，进行旅游体验。

## 移动互联网的创业需求
移动互联网创业的需求主要体现在以下几个方面：

1. 技术要求

    一方面，由于移动互联网技术的复杂性，这些产品必须能够采用最先进的技术手段提升用户的体验。另外，还要考虑到产品的流量、性能、存储等要求。例如，一个基于Android系统的移动通信APP，需要精心设计APP界面、优化APP运行速度、提高数据安全性、提供流畅的浏览体验。
    
2. 市场推广

    在移动互联网初期，许多创业者面临着资金困难的问题。不过，随着移动互联网的发展，这些创业者将成为全球领先的服务提供商、中间商或代理商。因此，他们的产品将会被用户广泛采用，进入到各个细分市场，形成规模经济。
    
3. 数据分析

    通过收集用户的数据及使用日志，我们可以更好的了解用户的行为习惯和需求，从而更好的开发移动互联网产品。例如，我们可以通过分析用户使用APP的时间、频率、功能使用情况、地区分布等指标，获得有关APP市场竞争力的关键信息。

# 3.移动互联网行业概况
## 移动互联网公司类型
移动互联网公司分为两类，一类是企业级的互联网公司，如腾讯、阿里巴巴、百度、搜狗等；另一类是个人开发者创业型的互联网公司，如Uber、GrabBike、LimeBike、MiniBus、Moovit等。

## 移动互联网行业市占率

据调查，移动互联网行业市占率目前位列第五。调查显示，截至2017年底，中国移动互联网占有率达到75.1%，其中，中小型互联网公司的市占率达到49.6%。移动互联网行业已经成为中国互联网发展的重要组成部分，占到了整个互联网总产值的40.3%，成为今年全国第三大产业。

# 4.核心技术
## Android平台
Android是一个开源的操作系统，由Google开发。其主要特性有：

- 开源、免费：Android是自由软件，用户可以完全免费获取。
- 可移植性：Android可以运行在多个手机厂商的不同设备上，实现了跨平台特性。
- 安全性：Android具有高度的安全性，能够防止恶意软件的侵害。
- 多屏支持：Android支持多种屏幕尺寸，适用于不同大小的手机。
- 易学习性：Android拥有丰富的SDK，开发人员可以快速上手。
- 性能优异：Android的运行速度非常快，尤其是在处理图像和动画方面。

## iOS平台
iOS是一个苹果公司推出的手机操作系统，其主要特性有：

- 使用 Objective-C 编程语言：iOS使用Objective-C编程语言，是一种面向对象的编程语言。
- 闭源、收费：Apple严格限制了外部开发者的开发工作，只能以收费的方式才能得到其支持。
- 安全性：由于闭源，导致其代码可能存在安全漏洞。
- UI设计师友好：Apple官方提供了Mac OS X SDK，其内置了众多的UI组件。
- APP审核周期长：iPhone和iPad上的APP需要经过苹果的审核，审核时间通常需要半年左右。
- 有限的资源：iPhone和iPad的硬件资源相对较少，所以iOS也有一些限制。
- 性能较弱：iPhone和iPad的性能仍然很弱，尤其是在处理图像、视频等高性能任务上。

## HTML5技术
HTML5是Web技术的最新版本，目前由W3C组织制定标准。HTML5的主要特性有：

- 拥有超链接、图片、音频、视频、Canvas、Web Storage和Web SQL Database。
- CSS3增强了样式表的功能，可以对页面元素进行平滑动画、交互效果等。
- SVG引入矢量图形功能，可以创建类似游戏、绘画和图片的视觉效果。
- Web Messaging API为Web开发人员提供了跨浏览器的消息传递功能。

## 云计算技术
云计算是一种将资源通过网络动态分配给应用的技术。其主要特征有：

- 按需付费：云计算服务的使用费用可以根据实际使用量和用量大小进行计费。
- 自动伸缩：云计算服务可以根据应用的负载情况自动扩容和缩容，保证资源的高可用性。
- 弹性网络：云计算服务提供多种方式供用户选择，用户可以在户内、本地、外网、第三方服务器等不同位置访问应用。

## 大数据技术
大数据技术是指海量数据的存储、分析和挖掘技术。目前，大数据技术正在成为移动互联网的热门研究方向。其主要特征有：

- 海量数据存储：通过云计算平台可以实现海量数据的存储。
- 数据分析挖掘：利用大数据技术可以对海量数据进行实时的分析、挖掘，发现隐藏的价值。
- 手机app统计：通过大数据技术可以对用户进行精准的个性化广告投放。

# 5.创业故事

## 悦活酒店App
悦活酒店是一个中餐馆和酒店连锁经营管理的移动互联网公司，他们通过App把餐饮和住宿结合起来，通过外卖、呼叫中心等渠道与顾客打交道。

在创业的第一年，悦活酒店采用的是Google App Engine框架开发移动App。因为该框架免费、轻量、快速、可扩展性强，并且能够和其他App集成。但是，随着时间的推移，Google App Engine逐渐变成了一项被动的开发环境，不能满足业务发展的需求。因此，悦活酒店决定在互联网公司中引进技术人员重新组建团队来支撑业务发展。

## Viriadyne数码相框
Viriadyne数码相框是一家数字化影像公司，通过App来管理客户的数字相框。公司秉承一切为了用户的理念，在保证高质量的同时，也确保客户的隐私安全。

在创业的第二年，Viriadyne采用的是WordPress CMS框架开发网站，该框架属于老牌的建站工具，能够快速部署和维护网站。但是，由于WordPress的插件更新缓慢、缺乏安全性支持，使得公司面临安全风险。因此，Viriadyne决定在互联网公司中引进技术人员重新组建团队来支撑业务发展。

## Palooza奶茶会
Palooza奶茶会是一个以咖啡为主的移动互联网公司，通过App的形式让消费者在手机上享受到健康的生活方式。

在创业的第三年，Palooza采用的是PHP+MySQL开发网站。由于该技术栈过时、代码质量低、缺乏前后端分离的能力，公司的网站经常出现安全漏洞。因此，Palooza决定在互联网公司中引进技术人员重新组建团队来支撑业务发展。
