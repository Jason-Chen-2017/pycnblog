# 1.简介
  

移动支付和电子现金支付（简称E-cash）作为现代生活中不可或缺的一部分，越来越多的人选择使用移动支付或其他数字化支付方式。究竟移动支付和电子现金支付有什么不同？它们之间又有哪些联系和区别呢？本文将从以下几个方面对移动支付和电子现金支付进行探讨：

# 1) 定义及基本特性
移动支付和电子现金支付分别指的是一种应用在移动设备上的数据传输、处理和交易的方式，在经济层面上也存在着巨大的差异。两者都是利用数字技术实现各种支付功能的有效途径。但它们也有一些共同之处，比如都可以提供个人消费者和商户的消费服务。

首先，移动支付和电子现金支付的定义及基本特性。

“移动支付”是指以移动网络或者无线网络技术为基础的信息技术运用，通过移动终端设备如手机、平板电脑、手持设备等，远程采集用户的账户信息和交易信息，并通过信息传输和数据存储技术，完成收款、付款或转账过程。“移动支付”与传统的支付宝、微信支付等生活中的支付方式形成了互补关系。在移动支付的场景下，支付主体一般是个人消费者。移动支付产品的特点主要包括：

1. 成本低廉：相对于传统的实体店支付机构，移动支付的成本可谓便宜得多。目前，移动支付基本都是免费的，绝大部分用户不需要担心支付手续费、不用缴纳服务费。

2. 使用方便：消费者只需要打开手机上的相应应用程序，就可以轻松完成网购、银行业务、医疗保险、养老保险等各种支付需求。

3. 用户群体广泛：目前，中国移动互联网用户规模达到7亿人，占全国人口的9%以上，使得移动支付成为一种畅通无阻的支付渠道。

4. 可靠性高：移动支付的支付成功率较高，其覆盖范围覆盖大部分用户的支付需求。同时，移动支付平台还可实时提供每笔交易的详细状态，保障用户权益得到充分保障。

5. 安全性高：移动支付的安全性依赖于用户的个人信息安全、网络安全、密码安全和交易数据安全等。该系统采用多重身份认证和验证机制，确保用户交易信息的安全。

“电子现金支付”是指使用电子信息技术为支付双方提供直接的现金转账服务，是传统支付方式的一种新型形式。“电子现金支付”通过国家标准组织ECC/ISO及其他相关认证机构认证，所支付金额可以在中国境内任意地点使用。电子现金支付具有以下主要特征：

1. 消费者易用：电子现金支付本质上是现金的一种数字形式，消费者在使用过程中完全感受不到中间媒介的存在。消费者可以通过手机App、支付宝小程序、微信支付等直接与商家进行交易，而无需携带身份证或银行卡。

2. 技术支持：电子现金支付的普及，使得移动支付领域的服务提升迅速。通过开放API接口，电子现金支付应用市场正在扩张。

3. 支付方式多样：电子现金支付除了现金转账以外，还包括银行卡转账、借记卡转账、电汇转账等多种支付方式。

4. 服务范围广：电子现金支付已成为世界各国用户的首选支付方式。中国的发改委已经承诺将今年开始试点推动“商务支付”，也就是利用互联网支付服务进行跨境支付。

5. 数据安全：电子现金支付的数字货币系统采用公钥加密法，确保数据在传输过程中不被泄露或篡改。

# 2) 发展历史和影响
移动支付和电子现金支付的发展历程。

移动支付和电子现金支付是近几年来两大支付方式的代表性技术。随着移动支付产品的逐步完善和普及，移动支付成为消费者最主要的支付选择。目前，移动支付已经成为最主要的支付方式，也是支付方式的增长引擎。但是，移动支付和电子现金支付之间还是存在着许多差异。下面我们将从两个方面对移动支付和电子现金支付的发展历史和影响进行探讨。

## 2.1 移动支付的发展历程
20世纪90年代初，美国国家安全局（NSA）为了应对移动通信技术革命带来的潜在危害，就开始在美国各州建立研究中心，这些研究中心先后产生了三种移动支付方案：

第一款是商业品牌卡支付，它属于即时支付模型，由信用卡公司发行，消费者根据银行预设的指令按月或季度结算，在消费额度内支付消费者的全部现金或部分货币。这种支付方案的最大优点是简单快捷，消费者几乎无需关注支付手续费，但缺点也很明显，即无法实时提供付款状态，以及银行方面的结算风险。

20世纪90年代末，日本兴起了移动支付行业。日本通过提供购物支付宝（PayPal）和携号转网（OboCash）等产品，为消费者提供了便捷的支付方式。虽然PayPal和OboCash为消费者提供了便利，但它们仍然依赖信用卡进行结算，消费者仍然需要关注结算时的银行结算风险。

20世cellrow，美国的ATM取代了商业品牌卡支付，这是继日元和美元之后在美国流行的第二大支付手段。但由于ATM硬件昂贵、安装难度高、发卡量有限，因此电话卡支付（telegraph card payment）也在20世纪90年代出现。

20世纪90年代末期，欧洲各国也纷纷启动移动支付产品。主要的产品有三星Pay、SFR Mobile Money、Euromoney、Mango Mobile、Twilio Mobile Payment等。这些产品均属于第三方支付公司，直接与电信运营商合作，通过在电信公司的服务器上维护客户资料，消费者通过客户端软件或者网站直接与支付公司进行交易。虽然这种支付方式具有安全性，但缺乏实时交易信息、交易失败风险等问题。

2000年左右，手机短信支付出现。随着信息技术的飞速发展，现在的手机短信往往比单纯的短信更加便捷有效。消费者可以发送一条短信给商家，通过短信平台直接完成支付。手机短信支付本身没有任何实质性的优势，只是单纯的降低了支付的成本，增加了支付的便利性，但其最大的问题就是信息不对称性和数据安全性。

2006年，中国移动推出蚂蚁花呗、滴滴快车、支付宝等移动支付平台。截止到2020年，中国的移动支付总额已经接近5万亿元人民币，成为中国支付宝、微信支付、余额宝的最大用户群。

## 2.2 移动支付的影响
移动支付的影响。

移动支付在近几年快速发展，尤其是在2018年，移动支付的整体支出比预期的要高很多。据估计，中国移动支付的收入超过55亿美元，位列全球第四大移动支付服务提供商。同时，2018年移动支付交易规模预计达到3.78万亿元人民币。

移动支付已经成为支付界的重点和热门方向。据观察，在中国，各行各业都在借助移动支付的便利和强劲的消费能力，抢滩互联网金融、移动支付、云计算、大数据、物联网等新经济领域。移动支付领域的创新也促进了经济发展，并为不同行业的企业带来新的商业模式。另外，移动支付在促进社会的公平正义，推动经济平稳健康发展，也引起了人们对法律的高度重视。

# 3) 基本概念和术语
移动支付和电子现金支付之间的区别。

移动支付和电子现金支付各自都有自己独有的基本概念和术语，下面我们将介绍它们的一些共同点和不同点。

## 3.1 基本概念

### 3.1.1 支付对象
移动支付涉及两种不同类型的用户：

- 个人消费者
- 商户

个人消费者通常通过移动设备或其他数字设备访问商家的支付平台，向商家付款或接受支付。商户也通过自己的移动支付平台发布商品或服务，并接受消费者的付款。

### 3.1.2 支付方式
移动支付通常有两种支付方式：

- 即时支付
- 委托支付

即时支付：即时支付是指消费者支付请求通过移动支付平台立即发出，由商家直接扣除消费者的钱款。即时支付的特点是消费者看到支付结果的速度快，价格透明。

委托支付：委托支付则是指消费者的支付请求由商家委托第三方支付平台（如支付宝、微信支付等）来处理，消费者并不能立刻看见支付结果，但商家可以实时掌握消费者的支付状态。

### 3.1.3 支付系统
移动支付平台是一个由第三方公司提供的在线支付平台，由此用户的身份信息、交易记录、支付数据、交易通知、结算清算系统等安全可靠的信息资源组成。移动支付平台的运营者依据不同业务规则，制定相应的规范，开发符合支付环境要求的支付流程和服务。例如，当交易金额超出商家可承受范围时，支付系统会自动退款或限制交易，以保证用户的支付权益得到充分保障。

### 3.1.4 支付网关
支付网关是指支付服务的中间服务节点，负责接收用户订单的创建、查询、支付确认、退款等请求，并与支付系统进行交互，生成交易结果和支付通知。支付网关的作用主要是对支付系统的压力进行缓冲，并减少与支付系统的直接交互次数。

## 3.2 术语

### 3.2.1 支付交易
支付交易是指消费者与商家的一次支付行为，包括订单的生成、支付请求的提交、支付结果的确认、消费者的退款等。支付交易的生命周期可以分为如下五个阶段：

1. 创建订单：消费者向商家发起一个购买交易请求，填写相关信息并上传商品信息、收货地址、支付方式等。
2. 提交支付请求：商家接收到消费者的购买交易请求后，根据交易额和支付方式，向支付网关发起支付请求。
3. 支付确认：支付网关向支付系统发起支付请求，通过双向验证、账期等一系列安全检查，确认消费者的支付资金能够顺利通过支付系统。
4. 支付成功：支付系统完成支付清算后，发送支付成功的消息至支付网关。
5. 支付成功通知：支付网关向消费者发送支付成功的通知，包括支付结果、交易流水号、商品详情等。

### 3.2.2 交易类型
移动支付和电子现金支付之间的区别。

移动支付的交易类型：

- 支付宝支付
- 微信支付
- Apple Pay
- Google Pay

电子现金支付的交易类型：

- 银行卡转账
- 借记卡转账
- 电汇
- 点对点支付

移动支付和电子现金支付之间的区别：

- 交易对象不同：移动支付适用于个人消费者，电子现金支付适用于商户。
- 支付方式不同：移动支付的支付方式包括支付宝、微信支付等，电子现金支付的支付方式包括银行卡转账、借记卡转账、电汇等。
- 数据安全性不同：移动支付采用安全的网络加密技术，保证消费者支付信息的安全；电子现金支付采用数字签名技术保证交易数据的真实性、完整性和不可否认性。
