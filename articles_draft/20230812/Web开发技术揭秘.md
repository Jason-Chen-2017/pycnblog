# 1.简介
  

随着移动互联网、智能手机等新兴设备的普及，网页应用（web application）已经成为用户日常生活中的一部分。由于它能够快速响应、节省时间、节约金钱，并且可以提供丰富的服务功能，使得用户在线上享受到前所未有的便利。因此，越来越多的人选择将自己的网站或者应用程序部署在云端，利用云计算平台为自己提供服务，从而实现自主可控。而网站技术、云计算技术、web服务器技术等相关知识也正在蓬勃发展，并由此带动了新的领域的出现——前端开发。前端开发主要负责的是网站页面的呈现，即如何把数据展现给用户，并通过交互的方式提升用户体验。因此，掌握前端开发技能对于互联网行业来说是十分必要的。本书旨在系统阐述web前端开发的各个方面，帮助读者更好地理解web前端开发的原理，加强其实际应用能力。
# 2.主要内容
## 2.1	HTML基础
- HTML(Hypertext Markup Language)：超文本标记语言，用于创建网页结构以及内容的标记语言。
- HTML文档结构
  - <!DOCTYPE>声明：定义文档类型，告诉浏览器使用的规范或版本。
  - <html>元素：文档根元素，所有其他元素必须嵌套在这里。
  - <head>元素：头部元素，包括<title>元素、<meta>元素、<link>元素和<script>元素。
  - <body>元素：内容元素，包含文档正文内容。
  - <h1>~<h6>元素：标题元素，用来显示文章的不同层次。
  - <p>元素：段落元素，用来分隔不同段落的内容。
  - <a>元素：链接元素，用来创建超链接。
  - <img>元素：图片元素，用来插入图片。
  - <ul>元素：无序列表元素，用来创建无序列表。
  - <ol>元素：有序列表元素，用来创建有序列表。
  - <li>元素：列表项元素，用来定义列表项。
  - <div>元素：块级容器元素，用来包裹其他元素。
  - <span>元素：行内容器元素，用来包裹文字。
- HTML标签属性
  - id属性：定义唯一标识符。
  - class属性：定义类别名称，可以用作自定义样式。
  - style属性：定义内联样式。
  - title属性：定义额外信息。
  - href属性：定义超链接的目标地址。
  - src属性：定义图像文件路径。
  - alt属性：定义图像替代文字。
  - align属性：设置图片的对齐方式。
  - type属性：定义输入字段的类型。
  - value属性：定义输入字段的值。
  - maxlength属性：限制输入字段的最大长度。
  - checked属性：指定默认选中状态。
  - disabled属性：禁用输入字段。
  - readonly属性：指定输入字段为只读。
## 2.2	CSS基础
- CSS(Cascading Style Sheets)：层叠样式表，用于设置网页的视觉样式。
- CSS语法
  - Selector：选择器，用来选取页面上的元素。
  - Property：属性，用来设置元素的样式。
  - Value：值，属性的具体取值。
  - Rule Set：规则集，包含多个选择器和样式声明。
  - Media Query：媒体查询，根据不同的屏幕尺寸和设备特性调整样式。
- CSS选择器
  - 标签选择器：选择特定标签名的元素，例如：<p>、<h1>等。
  - ID选择器：选择特定ID值的元素，例如：<div id="content">。
  - 类选择器：选择特定class值的元素，例如：<div class="container">。
  - 元素选择器：选择特定元素类型的元素，例如：<input type="checkbox">。
  - 通配符选择器：匹配任意标签、ID、class的元素，例如：*.container。
  - 属性选择器：选择具有特定属性的元素，例如：<a href="#">。
  - 伪类选择器：匹配特定状态的元素，例如：<a:visited>、<a:active>等。
  - 组合选择器：将多个选择器合并成一个选择器，例如：<div.container #nav>.item。
- CSS属性
  - background-color：设置元素的背景颜色。
  - color：设置字体颜色。
  - font-size：设置字体大小。
  - text-align：设置文本的水平对齐方式。
  - padding：设置元素的内边距。
  - margin：设置元素的外边距。
  - border：设置元素的边框。
  - width/height：设置元素的宽度和高度。
  - display：设置元素的显示方式。
  - position：设置元素的定位方式。
  - top/bottom/left/right：设置元素的定位偏移量。
  - float：设置元素的浮动方向。
- CSS单位
  - px：像素。
  - em：相对于当前字体的字号。
  - %：百分比。
  - vw/vh：视窗的宽高。
- CSS布局
  - 流体布局：基于内容大小和宽度，自动调整行间距和字号，不考虑任何外部约束。
  - 弹性布局：基于Flexbox模型，允许弹性缩放和换行。
  - 网格布局：基于Grid模型，可实现复杂的二维布局。
## 2.3	JavaScript基础
- JavaScript(Java Script)：一种解释型、面向对象、动态的编程语言。
- JavaScript语法
  - 数据类型
    - Number：数值类型。
    - String：字符串类型。
    - Boolean：布尔类型。
    - Object：对象类型。
    - Function：函数类型。
  - 操作符
    - 赋值运算符：=。
    - 算术运算符：+、-、*、/、%。
    - 关系运算符：==、!=、>、<、>=、<=。
    - 逻辑运算符：!、&&、||。
    - 条件运算符：?:。
    - 三目运算符：condition? expressionIfTrue : expressionIfFalse。
    - 逗号运算符：表达式1，表达式2。
    - 语句
      - var x = y;
      - if (condition) {statement}
      - for (initialization; condition; increment/decrement) {statement}
      - while (condition) {statement}
      - do {statement} while (condition);
      - break;
      - continue;
      - return expression;
  - 函数
    - function myFunction(parameter1, parameter2){function body}
    - 参数：传递给函数的值。
    - 返回值：函数执行结果。
  - 对象
    - 创建对象：var objectName = {};
    - 添加属性：objectName.propertyName = propertyValue;
    - 获取属性：propertyValue = objectName.propertyName;
    - 删除属性：delete objectName.propertyName;
    - 判断属性是否存在：propertyName in objectName;
    - 对象类型转换：String(), Number()等。
- BOM和DOM
  - BOM(Browser Object Model)：描述了web页面的窗口对象模型。
  - window对象：描述全局变量、浏览器窗口属性和方法。
  - location对象：描述当前页面URL相关信息。
  - navigator对象：描述浏览器及其插件的信息。
  - history对象：描述浏览器的历史记录。
  - document对象：描述当前页面的文档对象模型。
  - DOM(Document Object Model)：描述了处理网页内容的模型，提供了对网页各元素的各种操作。
  - createElement()：创建元素节点。
  - createTextNode()：创建文本节点。
  - getElementById()：根据ID获取元素。
  - querySelector()：根据选择器获取元素。
  - appendChild()：添加子节点。
  - removeChild()：删除子节点。
  - parentNode：父节点。
  - childNodes：子节点集合。
  - nextSibling：下一个兄弟节点。
