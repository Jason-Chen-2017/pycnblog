2. 时间分配代码实例

```python
import datetime

def allocate_time(tasks):
    now = datetime.datetime.now()
    for task in tasks:
        if now < task.deadline:
            time_left = task.deadline - now
            time_needed = task.priority * task.workload / task.resources
            if time_left >= time_needed:
                now += time_needed
            else:
                now = task.deadline
    return now
```

3. 任务依赖关系代码实例

```python
def schedule_tasks(tasks):
    schedule = []
    for task in tasks:
        if not schedule or schedule[-1].name != task.depends_on:
            schedule.append(task)
        else:
            schedule[schedule.index(task.depends_on)].depends_on.append(task)
    return schedule
```

4. 资源分配代码实例

```python
def allocate_resources(tasks):
    resources = {}
    for task in tasks:
        if task.name in resources:
            resources[task.name] += task.resources
        else:
            resources[task.name] = task.resources
    return resources
```
