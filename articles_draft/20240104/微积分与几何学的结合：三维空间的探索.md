## 4.2 计算斜率

import numpy as np

def partial_derivative(f, x0, y0, z0):
    x = np.array([x0])
    y = np.array([y0])
    z = np.array([z0])
    grad = np.array([np.gradient(f, x, edge_order=2), np.gradient(f, y, edge_order=2), np.gradient(f, z, edge_order=2)])
    m = np.sum(grad)
    return m

f = lambda x, y, z: x**2 + y**2 + z**2
x0 = 1
y0 = 2
z0 = 3

m = partial_derivative(f, x0, y0, z0)
print("The slope of the function at the point (", x0, ",", y0, ",", z0, ") is:", m)
```
在这个例子中，我们首先导入了NumPy库，然后定义了一个函数`partial_derivative`，该函数接受一个函数`f`和一个点`(x0, y0, z0)`为参数，并通过计算梯度来计算斜率。然后我们给出了一个函数`f`和一个点的具体值，并调用该函数计算斜率。

## 4.3 计算距离

import math

def calculate_distance(x1, y1, z1, x2, y2, z2):
    dx = x2 - x1
    dy = y2 - y1
    dz = z2 - z1
    D = math.sqrt(dx**2 + dy**2 + dz**2)
    return D

x1 = 1
y1 = 2
z1 = 3
x2 = 4
y2 = 5
z2 = 6

distance = calculate_distance(x1, y1, z1, x2, y2, z2)
print("The distance between the two points is:", distance)
```
在这个例子中，我们首先定义了一个函数`calculate_distance`，该函数接受六个参数，分别表示两个点的坐标，并通过计算距离公式`D = sqrt((x2 - x1)^2 + (y2 - y1)^2 + (z2 - z1)^2)`计算距离。然后我们给出了两个点的具体坐标值，并调用该函数计算距离。
