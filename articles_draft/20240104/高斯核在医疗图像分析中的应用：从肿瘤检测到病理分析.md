# 2.核心概念与联系
# 2.1 高斯核的定义与特点
高斯核（Gaussian Kernel）是一种常用的核函数，它的定义如下：
$$
K(x, y) = \exp \left(-\frac{\|x-y\|^2}{2 \sigma^2}\right)
$$
其中，$x$ 和 $y$ 是输入向量，$\|x-y\|^2$ 是欧氏距离的平方，$\sigma$ 是标准差。高斯核具有以下特点：

1. 对称性：$K(x, y) = K(y, x)$。
2. 正定性：$K(x, x) > 0$。
3. 高斯分布：核函数值随着输入向量之间的距离增加而迅速衰减。

# 2.2 高斯核在医疗图像分析中的应用

2. 图像分割：通过高斯核进行图像分割可以将图像划分为多个区域，从而实现对特定结构的提取。

```python
import numpy as np

def gaussian_kernel(x, y, sigma):
    diff = x - y
    return np.exp(-np.square(diff) / (2 * np.square(sigma)))
```

```python
import numpy as np

def region_growing(image, sigma, threshold):
    regions = []
    labels = np.zeros_like(image)
    region_ids = 1

    for i in range(image.shape[0]):
        for j in range(image.shape[1]):
            if labels[i, j] == 0:
                region = []
                region.append((i, j))
                labels[i, j] = region_ids
                queue = [(i, j)]

                while queue:
                    x, y = queue.pop(0)
                    for dx, dy in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
                        nx, ny = x + dx, y + dy
                        if nx >= 0 and ny >= 0 and nx < image.shape[0] and ny < image.shape[1]:
                            if np.square(image[x, y] - image[nx, ny]) <= sigma:
                                if labels[nx, ny] == 0:
                                    labels[nx, ny] = region_ids
                                    region.append((nx, ny))
                                    queue.append((nx, ny))

                regions.append(region)
                if len(region) > threshold:
                    region_ids += 1

    return labels, regions
```
