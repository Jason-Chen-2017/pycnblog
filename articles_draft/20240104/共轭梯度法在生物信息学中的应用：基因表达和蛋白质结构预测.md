## 3.1 共轭梯度法的数学模型

共轭梯度法的数学模型可以表示为以下公式：

$$
\min_{x \in \mathbb{R}^n} \frac{1}{2}x^T A x - b^T x
$$

其中，$A \in \mathbb{R}^{n \times n}$ 是方程组的系数矩阵，$b \in \mathbb{R}^n$ 是方程组的常数项向量，$x \in \mathbb{R}^n$ 是变量向量。

共轭梯度法的核心思想是通过构建一个与原方程相互对应的对偶问题，然后通过迭代地求解这个对偶问题来逐步逼近原方程的解。对偶问题可以表示为：

$$
\max_{y \in \mathbb{R}^n} - \frac{1}{2}y^T A^{-1} y - b^T A^{-1} y
$$

共轭梯度法的迭代步骤如下：

1. 选择一个初始向量$x_0$。
2. 对于每个迭代步骤$k$，执行以下操作：
   - 计算搜索方向$d_k = -A_k x_k + b_k$。
   - 选择步长$\alpha_k$，使得$f(x_k + \alpha_k d_k) \leq f(x_k)$。
   - 更新向量$x_{k+1} = x_k + \alpha_k d_k$。

import numpy as np

def conjugate_gradient(A, b, x0=None, tol=1e-9, max_iter=1000):
    if x0 is None:
        x0 = np.zeros(A.shape[1])

    r0 = b - A @ x0
    d0 = r0
    k = 0

    while np.linalg.norm(r0) > tol and k < max_iter:
        alpha = (r0 @ r0) / (d0 @ A @ d0)
        x1 = x0 + alpha * d0
        r1 = b - A @ x1
        beta = (r1 @ r1) / (r0 @ r0)
        d1 = r1 + beta * d0
        r0 = r1
        d0 = d1
        x0 = x1
        k += 1

    return x0, k

A = np.array([[2, -1], [-1, 2]])
b = np.array([3, -1])
x0 = np.array([0, 0])

x, iterations = conjugate_gradient(A, b, x0)
print("Iterations:", iterations)
print("Solution:", x)
