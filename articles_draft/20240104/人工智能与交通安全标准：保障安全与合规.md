## 4.1 数据收集与预处理

### 4.1.1 数据收集

使用 Python 的 `pandas` 库进行数据收集：

```python
import pandas as pd

data = pd.read_csv('traffic_data.csv')
```

### 4.1.2 数据预处理

使用 Python 的 `pandas` 库进行数据预处理：

```python
# 数据清洗
data = data.dropna()

# 数据规范化
data['speed'] = (data['speed'] - data['speed'].mean()) / data['speed'].std()

# 特征提取
data['time_of_day'] = data['time'].apply(lambda x: x.hour)
data['day_of_week'] = data['time'].apply(lambda x: x.weekday())
```

## 4.2 模型训练与优化

### 4.2.1 支持向量机（SVM）

使用 Python 的 `sklearn` 库进行 SVM 模型训练和优化：

```python
from sklearn.svm import SVC

# 训练数据和标签
X_train = data.drop(['time', 'accident'], axis=1)
y_train = data['accident']

# 创建 SVM 模型
svm = SVC(kernel='linear')

# 训练模型
svm.fit(X_train, y_train)
```

### 4.2.2 决策树

使用 Python 的 `sklearn` 库进行决策树模型训练和优化：

```python
from sklearn.tree import DecisionTreeClassifier

# 训练数据和标签
X_train = data.drop(['time', 'accident'], axis=1)
y_train = data['accident']

# 创建决策树模型
dt = DecisionTreeClassifier()

# 训练模型
dt.fit(X_train, y_train)
```

### 4.2.3 神经网络

使用 Python 的 `tensorflow` 库进行神经网络模型训练和优化：

```python
import tensorflow as tf

# 训练数据和标签
X_train = data.drop(['time', 'accident'], axis=1)
y_train = data['accident']

# 创建神经网络模型
model = tf.keras.Sequential([
    tf.keras.layers.Dense(16, activation='relu', input_shape=(X_train.shape[1],)),
    tf.keras.layers.Dense(8, activation='relu'),
    tf.keras.layers.Dense(1, activation='sigmoid')
])

# 编译模型
model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

# 训练模型
model.fit(X_train, y_train, epochs=100)
```
