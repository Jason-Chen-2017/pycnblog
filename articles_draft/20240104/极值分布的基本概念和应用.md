## 4.1 泊松分布的Python实现

泊松分布是一种离散的概率分布，它描述了数据中极端值的出现概率。我们可以使用Python的`scipy.stats`库来实现泊松分布的计算。以下是一个泊松分布的Python实例：

```python
import numpy as np
import scipy.stats as stats

# 设置参数
lambda = 3
x = np.arange(0, 10, 1)

# 计算泊松分布的概率
prob = stats.poisson.pmf(x, lambda)

# 打印结果
print(prob)
```

在这个实例中，我们设置了泊松分布的参数$\lambda=3$，并计算了取值为0到9的概率。通过打印结果，我们可以看到泊松分布的概率分布情况。

## 4.2 正态分布的Python实现

正态分布是一种连续的概率分布，它描述了数据中极端值的出现概率。我们可以使用Python的`scipy.stats`库来实现正态分布的计算。以下是一个正态分布的Python实例：

```python
import numpy as np
import scipy.stats as stats

# 设置参数
mu = 5
sigma = 2
x = np.linspace(mu - 4*sigma, mu + 4*sigma, 100)

# 计算正态分布的概率密度函数
pdf = stats.norm.pdf(x, mu, sigma)

# 打印结果
print(pdf)
```

在这个实例中，我们设置了正态分布的参数$\mu=5$，$\sigma=2$，并计算了取值为$\mu\pm4\sigma$范围内的概率密度函数。通过打印结果，我们可以看到正态分布的概率密度函数分布情况。
