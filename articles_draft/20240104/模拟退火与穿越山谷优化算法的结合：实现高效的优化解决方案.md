# 2.核心概念与联系

## 2.1模拟退火（Simulated Annealing, SA）

模拟退火是一种基于概率的优化算法，它的核心思想是通过随机搜索来寻找问题空间中的最优解。算法的过程可以理解为在一个高温状态下随机搜索，逐渐降低温度，逐步收敛到最优解。模拟退火算法的核心步骤如下：

1. 从问题空间中随机选择一个初始解。
2. 根据当前解生成一个邻域解。
3. 计算当前解和邻域解的差值。
4. 根据差值和温度参数决定是否接受邻域解。
5. 更新温度参数，并返回步骤2。

## 2.2穿越山谷优化算法（Hill Climbing, HC）

穿越山谷优化算法是一种基于梯度的优化算法，它的核心思想是通过在问题空间中逐步沿梯度向上走来寻找最优解。穿越山谷优化算法的核心步骤如下：

1. 从问题空间中随机选择一个初始解。
2. 计算当前解的梯度。
3. 根据梯度向上走，生成一个邻域解。
4. 更新当前解为邻域解。
5. 重复步骤2-4，直到找到最优解。

## 2.3结合模拟退火与穿越山谷优化算法

结合模拟退火与穿越山谷优化算法的核心思想，我们可以在模拟退火算法的过程中引入穿越山谷优化算法的思想，以提高算法的搜索效率。具体来说，我们可以在模拟退火算法的搜索过程中，根据当前解的温度参数，动态调整步长，以便更有效地搜索问题空间。这种方法可以在大规模问题中实现高效的优化解决方案。

    return (x - 3) ** 2

def sa_hc_combine(f, T_max, T_min, alpha, n_iter):
    x_current = np.random.uniform(-10, 10)
    T = T_max
    n_accept = 0

    for _ in range(n_iter):
        x_neighbor = x_current + np.random.uniform(-1, 1)
        delta = f(x_neighbor) - f(x_current)

        if delta < 0 or np.random.rand() < np.exp(-delta / T):
            x_current = x_neighbor
            n_accept += 1

    return x_current, n_accept / n_iter
