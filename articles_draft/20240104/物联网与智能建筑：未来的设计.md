5. 安全协议的实现：

```python
import ssl

context = ssl.create_default_context()
s = context.wrap_socket(socket.socket(socket.AF_INET, socket.SOCK_STREAM), server_hostname='example.com')
s.connect(('example.com', 8080))
response = s.recv(4096)
```
