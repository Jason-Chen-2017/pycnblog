## 2.核心概念与联系
### 2.1 长期记忆与短期记忆
长期记忆和短期记忆是大脑中两种不同类型的记忆。短期记忆是一种临时的记忆，用于存储我们在几秒钟内需要访问的信息。例如，当我们在学习一个新的概念时，我们可能会将其存储在短期记忆中，以便在稍后的几秒钟内访问。然而，短期记忆很快会消失。

长期记忆，则是一种持久的记忆，用于存储我们的经历、知识和技能。这些记忆可以持续很长时间，甚至是整个生命周期。例如，当我们学习一门新语言时，我们会将其存储在长期记忆中，以便在未来访问和使用。

### 2.2 数据库与知识组织
数据库是一种结构化的存储系统，用于存储和管理数据。数据库通常包括表、关系和查询语言等组件。表是数据库中的基本结构，用于存储数据。关系是表之间的连接，用于查询和组合数据。查询语言是用于查询和操作数据的语言。

知识组织是大脑中知识的结构化存储和管理。知识组织可以通过数据库的概念来理解。例如，在大脑中，我们可以将知识存储在表中，并通过关系来查询和组合知识。这种组织方式使得大脑能够快速地访问和使用知识。

class Database:

        self.conn = sqlite3.connect(name)

        self.cursor.execute(f"CREATE TABLE {name} ({', '.join(columns)})")

        self.conn.commit()
