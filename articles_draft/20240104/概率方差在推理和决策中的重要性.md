贝叶斯定理的公式为：

$$

条件概率可以通过以下公式计算：

$$
P(A|B) = \frac{P(A\cap B)}{P(B)}

贝叶斯定理可以通过以下公式计算：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}

### 3.3.4 方差的计算

方差可以通过以下公式计算：

$$
\text{方差}(X) = E[(X - \mu)^2]
$$

其中，$E[X]$ 表示X的期望，$\mu$ 表示X的均值。

## 4.2 使用Python计算方差

在本节中，我们将使用Python计算方差。

```python
# 定义数据集X
X = [1, 2, 3, 4, 5]

# 计算X的均值
mu = sum(X) / len(X)

# 计算X的方差
variance = sum((x - mu) ** 2 for x in X) / len(X)

# 输出结果
print("方差(X) =", variance)
```
