```python
import numpy as np
import matplotlib.pyplot as plt

# 随机漫步过程的生成
def random_walk(n, steps):
    x = np.zeros(n+1)
    for i in range(n):
        x[i+1] = x[i] + np.random.choice([-1, 1])
    return x

# 随机漫步过程的绘图
def plot_random_walk(x):
    plt.plot(x)
    plt.xlabel('时间')
    plt.ylabel('位置')
    plt.title('随机漫步过程')
    plt.show()

# 主函数
if __name__ == '__main__':
    n = 1000
    steps = 100
    x = random_walk(n, steps)
    plot_random_walk(x)
```

上述代码首先导入了numpy和matplotlib库，然后定义了一个随机漫步过程的生成函数random_walk，该函数通过循环生成了n个随机漫步点的位置。接着定义了一个随机漫步过程的绘图函数plot_random_walk，该函数通过matplotlib库绘制了随机漫步过程的图像。最后，通过主函数调用了随机漫步过程的生成和绘图函数。
