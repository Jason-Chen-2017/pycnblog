## 4.1 逻辑回归算法实例

在这个例子中，我们将使用逻辑回归算法来预测一个人是否会喜欢一个电影。我们将使用一个简单的数据集，包括电影的类型、时长和评分。

### 4.1.1 数据准备

首先，我们需要准备一个数据集。我们将使用一个简单的数据集，包括电影的类型、时长和评分。

```python
import pandas as pd

data = {
    'type': ['动作', '喜剧', '爱情', '悬疑', '科幻'],
    'length': [120, 90, 105, 110, 135],
    'rating': [8.5, 7.0, 9.0, 7.5, 8.0]
}

df = pd.DataFrame(data)
```

### 4.1.2 数据预处理

接下来，我们需要对数据进行预处理。我们将使用`LabelEncoder`来编码电影类型，并将时长转换为分钟。

```python
from sklearn.preprocessing import LabelEncoder

le = LabelEncoder()
df['type'] = le.fit_transform(df['type'])
df['length'] = df['length'].astype(int)
```

### 4.1.3 训练模型

接下来，我们需要训练一个逻辑回归模型。我们将使用`LogisticRegression`来实现这个目标。

```python
from sklearn.linear_model import LogisticRegression

lr = LogisticRegression()
lr.fit(df[['type', 'length']], df['rating'])
```

### 4.1.4 预测

最后，我们需要使用训练好的模型来预测一个人是否会喜欢一个电影。我们将使用一个新的数据点来进行预测。

```python
new_movie = {
    'type': '喜剧',
    'length': 90
}

new_movie_df = pd.DataFrame([new_movie])

prediction = lr.predict(new_movie_df[['type', 'length']])
print(f'This person will probably like the movie: {prediction[0] > 0.5}')
```
