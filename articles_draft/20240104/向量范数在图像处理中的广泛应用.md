# 2.核心概念与联系
## 2.1 向量范数的定义
在数学中，向量范数是一个数学概念，用于衡量向量空间中向量的大小。向量范数的定义如下：

$$
\| \mathbf{v} \| = \sqrt{\mathbf{v}^T \mathbf{v}}
$$

其中，$\mathbf{v}$ 是一个向量，$^T$ 表示转置，$\mathbf{v}^T \mathbf{v}$ 是向量内积的形式。向量范数的性质如下：

1. 非负性：对于任何向量 $\mathbf{v}$，有 $\| \mathbf{v} \| \geq 0$，且仅当 $\mathbf{v} = \mathbf{0}$ 时，$\| \mathbf{v} \| = 0$。
2. 对称性：对于任何向量 $\mathbf{v}$ 和 $\mathbf{u}$，有 $\| \mathbf{v} + \mathbf{u} \| = \| \mathbf{u} + \mathbf{v} \|$。
3. 三角不等式：对于任何向量 $\mathbf{v}$ 和 $\mathbf{u}$，有 $\| \mathbf{v} + \mathbf{u} \| \leq \| \mathbf{v} \| + \| \mathbf{u} \|$。

## 2.2 向量范数在图像处理中的应用
在图像处理中，向量范数可以用来度量图像的亮度、对比度和细节信息等方面。具体应用包括：

1. 图像压缩：通过计算图像的向量范数，可以衡量图像的纹理和细节信息，从而实现图像压缩。
2. 图像分割：通过计算图像的向量范数，可以将图像划分为多个区域，从而实现图像分割。
3. 图像识别：通过计算图像的向量范数，可以将图像特征提取，从而实现图像识别。

import cv2
import numpy as np

def compress_image(image_path, output_path, compression_rate):
    # 读取原始图像
    image = cv2.imread(image_path)
    # 计算原始图像的向量范数
    vector_norm = np.linalg.norm(image)
    # 根据压缩率计算新图像的大小
    new_size = int(vector_norm * compression_rate)
    # 压缩图像
    resized_image = cv2.resize(image, (new_size, new_size))
    # 保存压缩后的图像
    cv2.imwrite(output_path, resized_image)

from sklearn.model_selection import train_test_split

X, y = digits.data, digits.target

accuracy = accuracy_score(y_test, y_pred)
