# 2.核心概念与联系

## 2.1 TOPSIS 法的基本概念

TOPSIS 法是一种多因素决策分析方法，它的核心思想是将各个选项的各个因素评分转换为相似度值，然后根据这些相似度值来评估和排序各个选项。具体来说，TOPSIS 法包括以下几个步骤：

1. 确定决策对象和决策因素。
2. 对各个决策因素进行权重评价。
3. 对各个决策选项的各个决策因素进行评分。
4. 将各个决策选项的各个决策因素评分转换为相似度值。
5. 根据相似度值来评估和排序各个决策选项。

## 2.2 TOPSIS 法与其他决策分析方法的联系

TOPSIS 法与其他决策分析方法有一定的联系，例如：

1. 与 Analytic Hierarchy Process（AHP）：TOPSIS 法和 AHP 都是多因素决策分析方法，它们都包括对决策对象和决策因素的评价以及对各个决策选项的评分和排序。不过，TOPSIS 法和 AHP 的主要区别在于，TOPSIS 法将各个决策选项的各个决策因素评分转换为相似度值，然后根据这些相似度值来评估和排序各个决策选项，而 AHP 则通过对各个决策因素进行排序和权重评价来评估和排序各个决策选项。

2. 与 Technique for Order of Preference by Similarity to Ideal Solution（TOPSIS）：TOPSIS 法和 TOPSIS 是同一种决策分析方法，它们的名字只是不同的表述。
