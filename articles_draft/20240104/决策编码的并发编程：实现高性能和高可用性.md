## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来展示如何使用决策编码来实现并发编程。

### 4.1 示例代码

我们将使用一个简单的示例来演示决策编码的并发编程。在这个示例中，我们有两个任务，一个是打印“Hello, World!”，另一个是打印“Hello, Decision Coding!”。这两个任务之间没有依赖关系，我们可以同时执行它们。

```python
import threading

def task1():
    print("Hello, World!")

def task2():
    print("Hello, Decision Coding!")

if __name__ == "__main__":
    thread1 = threading.Thread(target=task1)
    thread2 = threading.Thread(target=task2)

    thread1.start()
    thread2.start()

    thread1.join()
    thread2.join()
```

### 4.2 详细解释说明

在这个示例中，我们首先导入了`threading`模块，它提供了一个线程类，我们可以使用它来实现并发编程。

接下来，我们定义了两个任务，一个是`task1`函数，它打印“Hello, World!”，另一个是`task2`函数，它打印“Hello, Decision Coding!”。

在`if __name__ == "__main__":`块中，我们创建了两个线程，`thread1`和`thread2`，它们分别执行`task1`和`task2`函数。然后我们启动这两个线程，使用`thread1.start()`和`thread2.start()`方法。

最后，我们使用`thread1.join()`和`thread2.join()`方法来等待这两个线程结束。这样我们就实现了一个简单的并发编程示例。
