### 1.4.1 阳性安全代码实例

在阳性安全代码实例中，我们将通过一个简单的密码检查器来演示阳性安全的应用。密码检查器的代码如下：

```python
import re

def check_password(password):
    if len(password) < 8:
        return False
    if not re.search(r'[A-Z]', password):
        return False
    if not re.search(r'[a-z]', password):
        return False
    if not re.search(r'[0-9]', password):
        return False
    if not re.search(r'[~!@#$%^&*()_+-=]', password):
        return False
    return True

password = '123456'
if check_password(password):
    print('密码安全')
else:
    print('密码不安全')
```

在这个代码实例中，我们通过一个简单的密码检查器来演示阳性安全的应用。密码检查器通过检查密码的长度、大小写字母、数字和特殊字符来确定密码是否安全。如果密码满足所有的条件，则返回True，表示密码安全；否则返回False，表示密码不安全。
