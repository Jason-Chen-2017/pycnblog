# 2.核心概念与联系

## 2.1 关联分析

关联分析是一种数据挖掘技术，用于发现数据中的隐藏模式和关系。它通过分析数据中的相关性，以便预测未来的行为和趋势。关联分析可以用于各种领域，如市场营销、金融、医疗保健、物流等。

关联分析的主要任务是找到数据中的关联规则。关联规则是一种表达形式，包括两个或多个条件（左边）和一个结果（右边）。例如，如果购买奶酪，那么很可能购买奶酪。关联规则的支持（support）是指条件发生的概率，而寿命（confidence）是指条件和结果共同发生的概率。

关联分析的主要算法有Apriori算法和FP-Growth算法。Apriori算法是一种基于频繁项集的算法，它首先找到频繁项集，然后找到基于频繁项集的关联规则。FP-Growth算法是一种基于频繁项的算法，它首先将数据转换为频繁项，然后找到基于频繁项的关联规则。

## 3.1 Growth算法

Growth算法是一种基于频繁项的算法，它首先将数据转换为频繁项，然后找到基于频繁项的灰色关联规则。Growth算法的主要步骤如下：

1. 数据预处理：将数据转换为频繁项集。
2. 生成初始候选项：从频繁项集中随机选择一个项目作为初始候选项。
3. 扩展候选项：为初始候选项添加其他项目，生成新的候选项。
4. 计算支持：计算每个候选项的支持。
5. 筛选支持：从候选项中筛选出支持超过阈值的候选项。
6. 计算寿命：计算每个筛选出的候选项的寿命。
7. 筛选寿命：从筛选出的候选项中筛选出寿命超过阈值的候选项。
8. 输出灰色关联规则：输出寿命超过阈值的候选项作为灰色关联规则。

Growth算法的数学模型公式如下：

- 支持：$$ supp(X) = \frac{count(X)}{count(D)} $$
- 寿命：$$ conf(X \rightarrow Y) = \frac{count(X \cup Y)}{count(X)} $$

其中，$X$ 和 $Y$ 是关联规则的左边和右边，$D$ 是数据集，$count(X)$ 是项目 $X$ 出现的次数。

## 4.1 Growth算法实例

```python
from growth import Growth

# 数据预处理
data = [['奶酪', '坚果'], ['奶酪', '苹果'], ['奶酪', '面包'], ['坚果', '苹果'], ['坚果', '面包'], ['苹果', '面包']]

# 生成初始候选项
initial_candidates = ['奶酪', '坚果', '苹果', '面包']

# 扩展候选项
candidates = []
for item in initial_candidates:
    candidate = [item]
    while len(candidate) < len(initial_candidates):
        for item in initial_candidates:
            if item not in candidate:
                candidate.append(item)
        candidates.append(candidate)

# 计算支持
support = {}
for candidate in candidates:
    support[candidate] = len([transaction for transaction in data if set(candidate).issubset(set(transaction))]) / len(data)

# 筛选支持
threshold = 0.5
frequent_items = {candidate: support[candidate] for candidate in candidates if support[candidate] >= threshold}

# 计算寿命
confidence = {}
for frequent_item in frequent_items:
    for item in frequent_items:
        if item not in frequent_item:
            candidate = frequent_item | {item}
            confidence[candidate] = support[candidate] / support[frequent_item] if support[frequent_item] > 0 else 1

# 筛选寿命
threshold = 0.7
gray_rules = {candidate: confidence[candidate] for candidate in frequent_items if confidence[candidate] >= threshold}

# 输出灰色关联规则
for rule in gray_rules:
    print(f"{rule[0]} -> {rule[1]} (支持: {support[rule]}, 寿命: {confidence[rule]})")
```
