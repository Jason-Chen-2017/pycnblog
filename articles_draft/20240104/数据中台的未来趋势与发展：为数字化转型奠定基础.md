## 4.4 数据安全

### 4.4.1 Python代码实例

```python
import pandas as pd
from cryptography.fernet import Fernet

# 生成密钥
key = Fernet.generate_key()

# 初始化密钥
cipher_suite = Fernet(key)

# 读取数据源
data = pd.read_csv('data.csv')

# 加密
encrypted_data = data.apply(lambda x: cipher_suite.encrypt(x.encode()), axis=1)

# 保存到数据中台
encrypted_data.to_csv('data_security.csv', index=False)
```

### 4.4.2 详细解释说明

1. 导入pandas库和Fernet加密算法。
2. 生成密钥，并存储到key变量中。
3. 初始化密钥，并存储到cipher_suite变量中。
4. 读取数据源，将数据源存储在data变量中。
5. 加密，将data中的数据加密，并存储到encrypted_data变量中。
6. 保存数据到数据中台，将encrypted_data保存到data_security.csv文件中。
