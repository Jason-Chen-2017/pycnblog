# 2.核心概念与联系

## 2.1模糊综合评价的基本概念

模糊综合评价是一种基于模糊逻辑的评价方法，它可以根据不确定、模糊的信息来进行综合评价。模糊综合评价的核心概念包括：

- 模糊集：模糊集是一种用于描述不确定信息的数据结构，它可以表示一个数值域内的元素集合。模糊集可以用来描述用户的兴趣、喜好等信息。
- 模糊关系：模糊关系是一种描述不确定关系的关系，它可以用来描述用户之间的关系、用户与项目之间的关系等。
- 模糊判断：模糊判断是一种根据模糊关系来进行判断的方法，它可以用来判断用户的兴趣、喜好等信息。

## 2.2模糊综合评价与传统综合评价的区别

传统综合评价通常基于明确、确定的信息来进行评价，而模糊综合评价则基于不确定、模糊的信息来进行评价。因此，模糊综合评价与传统综合评价的主要区别在于：

- 评价信息的性质：模糊综合评价处理的是不确定、模糊的信息，而传统综合评价处理的是明确、确定的信息。
- 评价方法的特点：模糊综合评价使用了模糊逻辑和模糊数学的方法来进行评价，而传统综合评价使用了传统数学和统计方法来进行评价。

# 构建模糊集
def build_fuzzy_set(data):
    fuzzy_set = {}
    for item in data:
        fuzzy_set[item] = np.random.uniform(0, 1)
    return fuzzy_set

# 建立模糊关系
def build_fuzzy_relation(user_relation_data):
    fuzzy_relation = {}
    for user_id, relation in user_relation_data.items():
        fuzzy_relation[user_id] = {}
        for friend_id, relation_value in relation.items():
            fuzzy_relation[user_id][friend_id] = np.random.uniform(0, 1)
    return fuzzy_relation

# 进行模糊判断
def fuzzy_judgment(fuzzy_set, fuzzy_relation):
    judgment = {}
    for user_id, fuzzy_set_value in fuzzy_set.items():
        judgment[user_id] = {}
        for item, fuzzy_set_value in fuzzy_set_value.items():
            judgment[user_id][item] = fuzzy_set_value * np.max(fuzzy_relation[user_id].values())
            if np.isnan(judgment[user_id][item]):
                judgment[user_id][item] = 0
    return judgment

# 测试代码
if __name__ == '__main__':
    # 构建模糊集
    user_data = [['user1', 'item1', 0.8], ['user1', 'item2', 0.9], ['user2', 'item1', 0.7], ['user2', 'item2', 0.6]]
    fuzzy_set = build_fuzzy_set(user_data)

    # 建立模糊关系
    user_relation_data = {'user1': {'user2': 0.5}, 'user2': {'user1': 0.4}}
    fuzzy_relation = build_fuzzy_relation(user_relation_data)

    # 进行模糊判断
