## 3.1 次梯度定义的基本形式
次梯度定义的基本形式如下：

$$
\begin{aligned}
\min_{x \in \mathbb{R}^n} & \quad c^T x \\
\text{s.t.} & \quad Ax \preceq b \\
& \quad x \in \mathbb{R}^n
\end{aligned}
$$

其中，$c \in \mathbb{R}^n$是目标函数的系数向量，$A \in \mathbb{R}^{m \times n}$是约束矩阵，$b \in \mathbb{R}^m$是约束向量。

## 3.2 次梯度定义的解析解
次梯度定义的解析解可以通过以下步骤得到：

1. 将原始问题转换为标准形式：

$$
\begin{aligned}
\min_{x \in \mathbb{R}^n} & \quad c^T x \\
\text{s.t.} & \quad Ax \preceq b \\
& \quad x \succeq 0
\end{aligned}
$$

其中，$x \succeq 0$表示$x$的每个分量都大于等于0。

2. 将标准形式的次梯度定义转换为二次规划问题：

$$
\begin{aligned}
\min_{x \in \mathbb{R}^n} & \quad \frac{1}{2} x^T H x + c^T x \\
\text{s.t.} & \quad x \succeq 0
\end{aligned}
$$

其中，$H \in \mathbb{R}^{n \times n}$是对称正定矩阵。

3. 使用标准的二次规划求解方法（如内点法或外点法）来求解二次规划问题。

# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个具体的代码实例来展示次梯度定义的应用。我们将使用Python的CVXPY库来实现次梯度定义的解析解和数值解。

## 4.1 导入所需库
```python
import numpy as np
import cvxpy as cp
```

## 4.2 定义目标函数和约束条件
```python
n = 3
c = np.array([1, -2, 3])
A = np.array([[1, 0, 0],
              [0, 1, 0],
              [0, 0, 1]])
b = np.array([1, 1, 1])
```

## 4.3 使用解析解方法求解次梯度定义
```python
x = cp.Variable(n)
objective = cp.Minimize(cp.sum(c * x))
constraints = [A * x <= b, x >= 0]
problem = cp.Problem(objective, constraints)
problem.solve()
```

## 4.4 使用数值解方法求解次梯度定义
```python
x = cp.Variable(n)
objective = cp.Minimize(cp.sum(c * x))
constraints = [A * x <= b, x >= 0]
problem = cp.Problem(objective, constraints)
problem.solve()
```
