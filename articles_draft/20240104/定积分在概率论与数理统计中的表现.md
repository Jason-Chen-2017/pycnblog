期望是一种描述随机变量平均值的统计量，它可以用来衡量随机变量的中心趋势。定积分在计算期望时主要用于计算面积。

方差是一种描述随机变量离中心趋势的统计量，它可以用来衡量随机变量的不确定性。定积分在计算方差时主要用于计算面积。

### 3.1 定积分的基本公式

定积分的基本公式为：

$$
\int_{a}^{b} f(x) dx = F(b) - F(a)
$$

其中，$F(x)$ 是函数$f(x)$ 的反函数。

## 4. 具体代码实例和详细解释说明

### 4.1 计算概率密度函数（PDF）

```python
import numpy as np
import scipy.integrate as spi

def f(x):
    return np.exp(-x**2)

a = 0
b = 1
result, error = spi.quad(f, a, b)
print("PDF 值为：", result)
```

### 4.2 计算累积分布函数（CDF）

```python
import numpy as np
import scipy.integrate as spi

def F(x):
    return spi.quad(lambda t: np.exp(-t**2) * np.arctan(t) / np.sqrt(2 * np.pi), -np.inf, x)

x = 0.5
result, error = spi.quad(F, -np.inf, x)
print("CDF 值为：", result)
```

### 4.3 计算期望

```python
import numpy as np
import scipy.integrate as spi

def f(x):
    return x * np.exp(-x**2)

a = 0

```

### 4.4 计算方差

```python
import numpy as np
import scipy.integrate as spi

def f(x):
    return x**2 * np.exp(-x**2)

a = 0
b = 1
result, error = spi.quad(f, a, b)
variance = result - np.square(spi.quad(lambda t: t * np.exp(-t**2), a, b))

print("方差 值为：", variance)
```

### 6.1 定积分与积分的区别

定积分是求积分的一种形式，它可以用来计算多项式、函数的面积、曲线下方程等。而积分是一种数学概念，它描述了一个函数在一个区间内的面积。

### 6.2 定积分的计算方法有哪些？
