## 4.1 大脑的多任务处理代码实例

def task1():
    # 模拟大脑处理任务1的代码
    pass

def task2():
    # 模拟大脑处理任务2的代码
    pass

def main():
    # 创建两个线程，分别处理任务1和任务2
    thread1 = threading.Thread(target=task1)
    thread2 = threading.Thread(target=task2)

    # 启动线程
