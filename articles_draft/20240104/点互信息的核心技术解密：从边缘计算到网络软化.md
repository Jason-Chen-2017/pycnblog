# 2.核心概念与联系

## 2.1边缘计算

边缘计算（Edge Computing）是一种新型的计算模式，将数据处理和应用程序从中心化的服务器移动到边缘设备（如路由器、交换机等），从而降低了数据传输成本和延迟。边缘计算具有以下特点：

1. 分布式：边缘设备分布在整个网络中，可以实现大规模并行计算。
2. 智能化：边缘设备可以自主地进行数据处理和决策。
3. 安全：边缘计算可以保护数据的隐私和安全性。

## 2.2网络软化

网络软化（Software-Defined Networking, SDN）是一种新型的网络架构，将网络控制和数据平面分离，使得网络可以通过软件控制。SDN具有以下特点：

1. 灵活性：SDN可以根据需求快速调整网络配置。
2. 可扩展性：SDN可以支持大规模的网络设备和流量。
3. 可见性：SDN可以实时监控网络状态和流量。

## 2.3点互信息

点互信息（Point-to-Point Mutual Information, PMI）是一种用于度量两个变量之间的相关性的统计量。在边缘计算和网络软化中，点互信息可以用于解决数据传输、处理和安全等问题。具体来说，点互信息可以用于：

$$
H(X) = -\sum_{x \in X} P(x) \log_2 P(x)
$$

其中，$X$是一个随机变量的取值集合，$P(x)$是随机变量$X$取值$x$的概率。

$$
I(X;Y) = H(X) - H(X|Y)

其中，$I(X;Y)$是点互信息，$H(X)$是随机变量$X$的熵，$H(X|Y)$是随机变量$X$给定随机变量$Y$的熵。

public class PMI {
    public static double pmi(double[] x, double[] y) {
        double p_xy = 0.0;
        double p_x = 0.0;
        double p_y = 0.0;
        double p_x_y = 0.0;
        int n = x.length;

            int x_bin = (int) (x[i] * 100);
            int y_bin = (int) (y[i] * 100);
            p_xy += (double) (Arrays.stream(Arrays.copyOfRange(x_bin, x_bin, x_bin + 1)).count()) * (Arrays.stream(Arrays.copyOfRange(y_bin, y_bin, y_bin + 1)).count());
