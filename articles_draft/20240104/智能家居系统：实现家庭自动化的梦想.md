# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的智能家居系统实例进行说明。这个系统包括一个智能灯泡和一个智能门锁。我们使用Python编程语言实现这个系统。

```python
import time
import datetime

class SmartLight:
    def __init__(self):
        self.status = False

    def turn_on(self):
        self.status = True
        print("SmartLight is turned on.")

    def turn_off(self):
        self.status = False
        print("SmartLight is turned off.")

class SmartLock:
    def __init__(self):
        self.status = False
        self.unlock_time = datetime.time(18, 0, 0)

    def unlock(self):
        if self.status:
            print("SmartLock is already unlocked.")
        else:
            self.status = True
            print("SmartLock is unlocked.")

    def lock(self):
        if self.status:
            self.status = False
            print("SmartLock is locked.")
        else:
            print("SmartLock is already locked.")

def main():
    smart_light = SmartLight()
    smart_lock = SmartLock()

    while True:
        current_time = datetime.datetime.now().time()
        if current_time >= smart_lock.unlock_time:
            smart_lock.unlock()
        else:
            smart_lock.lock()

        if smart_light.status:
            print("SmartLight is on.")
        else:
            print("SmartLight is off.")

        time.sleep(60)

if __name__ == "__main__":
    main()
```

这个实例中，我们定义了两个类`SmartLight`和`SmartLock`，分别表示智能灯泡和智能门锁。`SmartLight`类有两个方法`turn_on`和`turn_off`，用于控制灯泡的开关状态。`SmartLock`类有两个方法`unlock`和`lock`，用于控制门锁的开关状态。`main`函数中，我们创建了一个智能灯泡和一个智能门锁的实例，并实现了智能门锁的自动锁定功能。
