# 4. 具体代码实例和详细解释说明

具体代码实例如下：

1. 数据预处理：

```python
import pandas as pd
from sklearn.preprocessing import MinMaxScaler

data = pd.read_csv('employee_performance.csv')
scaler = MinMaxScaler()
data['skill_level'] = scaler.fit_transform(data['skill_level'].values.reshape(-1, 1))
data['work_efficiency'] = scaler.fit_transform(data['work_efficiency'].values.reshape(-1, 1))
data['communication_ability'] = scaler.fit_transform(data['communication_ability'].values.reshape(-1, 1))
```

1. 特征提取：

```python
from sklearn.preprocessing import StandardScaler

data['mean_skill_level'] = data.groupby('department')['skill_level'].transform(StandardScaler().mean_)
data['std_skill_level'] = data.groupby('department')['skill_level'].transform(StandardScaler().std_)
data['mean_work_efficiency'] = data.groupby('department')['work_efficiency'].transform(StandardScaler().mean_)
data['std_work_efficiency'] = data.groupby('department')['work_efficiency'].transform(StandardScaler().std_)
data['mean_communication_ability'] = data.groupby('department')['communication_ability'].transform(StandardScaler().mean_)
data['std_communication_ability'] = data.groupby('department')['communication_ability'].transform(StandardScaler().std_)
```

1. 模型构建：

```python
from sklearn.ensemble import RandomForestClassifier

X = data[['mean_skill_level', 'std_skill_level', 'mean_work_efficiency', 'std_work_efficiency', 'mean_communication_ability', 'std_communication_ability']]
y = data['department']
model = RandomForestClassifier()
model.fit(X, y)
```

1. 模型评估：

```python
from sklearn.metrics import classification_report

y_pred = model.predict(X)
print(classification_report(y, y_pred))
```

1. 模型优化：

```python
from sklearn.model_selection import GridSearchCV

param_grid = {'n_estimators': [100, 200, 300], 'max_depth': [5, 10, 15]}
grid_search = GridSearchCV(estimator=model, param_grid=param_grid, cv=5)
grid_search.fit(X, y)
print(grid_search.best_params_)
```
