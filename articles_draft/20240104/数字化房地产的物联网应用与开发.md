1. 线性回归模型：用于预测房屋设备的状态和使用情况。线性回归模型的公式为：
$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$
其中，$y$ 是预测值，$x_1, x_2, ..., x_n$ 是输入变量，$\beta_0, \beta_1, ..., \beta_n$ 是参数，$\epsilon$ 是误差项。

2. 逻辑回归模型：用于预测房屋设备的故障情况。逻辑回归模型的公式为：
$$
P(y=1|x_1, x_2, ..., x_n) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - ... - \beta_nx_n}}
$$
其中，$P(y=1|x_1, x_2, ..., x_n)$ 是预测概率，$\beta_0, \beta_1, ..., \beta_n$ 是参数。

3. 决策树模型：用于分析房屋设备的使用规律和趋势。决策树模型的公式为：
$$
\arg \max_{c} \sum_{i=1}^n I(d_i = c) P(d_i|x_1, x_2, ..., x_n)
$$
其中，$c$ 是类别，$I(d_i = c)$ 是指示函数，$P(d_i|x_1, x_2, ..., x_n)$ 是条件概率。
