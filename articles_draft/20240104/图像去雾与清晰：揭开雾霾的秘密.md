## 3.1 均值滤波

均值滤波是一种简单的图像去雾技术，它的核心思想是通过将图像中的邻域像素值进行求和，然后将得到的和除以邻域像素数量来得到滤波后的像素值。均值滤波可以有效地消除图像中的噪声，但是它也会导致图像的边缘模糊化。

具体操作步骤如下：

1. 定义一个邻域大小，例如3x3。
2. 对于每个像素点，将其邻域内的像素值求和。
3. 将得到的和除以邻域像素数量来得到滤波后的像素值。

数学模型公式如下：

$$
G(x,y) = \frac{1}{N} \sum_{i=0}^{N-1} \sum_{j=0}^{N-1} f(x+i,y+j)
$$

其中，$G(x,y)$ 是滤波后的像素值，$f(x,y)$ 是原始像素值，$N$ 是邻域像素数量。

## 3.2 中值滤波

中值滤波是一种更高级的图像去雾技术，它的核心思想是通过将图像中的邻域像素值排序，然后将排名中间的像素值作为滤波后的像素值。中值滤波可以有效地消除图像中的噪声，同时也能保留图像的边缘信息。

具体操作步骤如下：

1. 定义一个邻域大小，例如3x3。
2. 对于每个像素点，将其邻域内的像素值排序。
3. 将排名中间的像素值作为滤波后的像素值。

数学模型公式如下：

$$
G(x,y) = f(x,y)
$$

其中，$G(x,y)$ 是滤波后的像素值，$f(x,y)$ 是原始像素值。

## 3.3 高斯滤波

高斯滤波是一种最常用的图像去雾技术，它的核心思想是通过将图像中的邻域像素值与高斯核进行卷积，从而得到滤波后的像素值。高斯滤波可以有效地消除图像中的噪声，同时也能保留图像的边缘信息。

具体操作步骤如下：

1. 定义一个高斯核，例如3x3。
2. 对于每个像素点，将其邻域内的像素值与高斯核进行卷积。
3. 将得到的卷积结果作为滤波后的像素值。

数学模型公式如下：

$$
G(x,y) = f(x,y) * K(x,y)
$$

其中，$G(x,y)$ 是滤波后的像素值，$f(x,y)$ 是原始像素值，$K(x,y)$ 是高斯核。

## 4.1 均值滤波

```python
import cv2
import numpy as np

def mean_filter(image, kernel_size):
    rows, cols = image.shape[:2]
    filtered_image = np.zeros((rows, cols))

    for i in range(rows):
        for j in range(cols):
            sum_value = 0
            count = 0
            for k in range(kernel_size):
                for l in range(kernel_size):
                    if i - k >= 0 and j - l >= 0 and i + k < rows and j + l < cols:
                        sum_value += image[i - k][j - l]
                        count += 1
            filtered_image[i][j] = sum_value / count

    return filtered_image

kernel_size = 3
filtered_image = mean_filter(image, kernel_size)
cv2.imshow('Filtered Image', filtered_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

```python
import cv2
import numpy as np

def median_filter(image, kernel_size):
    rows, cols = image.shape[:2]
    filtered_image = np.zeros((rows, cols))

    for i in range(rows):
        for j in range(cols):
            pixel_values = []
            for k in range(kernel_size):
                for l in range(kernel_size):
                    if i - k >= 0 and j - l >= 0 and i + k < rows and j + l < cols:
                        pixel_values.append(image[i - k][j - l])
            pixel_values.sort()
            filtered_image[i][j] = pixel_values[kernel_size // 2]

    return filtered_image

kernel_size = 3
filtered_image = median_filter(image, kernel_size)
cv2.imshow('Filtered Image', filtered_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

```python
import cv2
import numpy as np

def gaussian_filter(image, kernel_size, sigma_x, sigma_y):
    rows, cols = image.shape[:2]
    filtered_image = np.zeros((rows, cols))

    for i in range(rows):
        for j in range(cols):
            gaussian_kernel = cv2.getGaussianKernel(kernel_size, sigma_x, sigma_y)
            filtered_image[i][j] = cv2.filter2D(image[i][j], -1, gaussian_kernel)

kernel_size = 3
sigma_x = 1
sigma_y = 1
```
