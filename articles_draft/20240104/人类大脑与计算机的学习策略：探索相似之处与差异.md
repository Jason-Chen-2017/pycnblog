## 3.1 监督学习：逻辑回归

逻辑回归是一种监督学习算法，用于分类问题。逻辑回归假设存在一个分隔超平面，可以将数据分为不同的类别。逻辑回归的目标是找到一个最佳的分隔超平面，使得在该超平面上的错误率最小。

逻辑回归的数学模型公式如下：

$$
P(y=1|\mathbf{x};\mathbf{w}) = \frac{1}{1+\exp(-\mathbf{w}^T\mathbf{x})}
$$

其中，$P(y=1|\mathbf{x};\mathbf{w})$ 是输入向量 $\mathbf{x}$ 通过权重向量 $\mathbf{w}$ 得到的输出概率。$\exp$ 是指数函数。

具体操作步骤如下：

1. 初始化权重向量 $\mathbf{w}$。
2. 计算输入向量 $\mathbf{x}$ 通过权重向量 $\mathbf{w}$ 得到的输出概率。
3. 使用梯度下降法来优化权重向量 $\mathbf{w}$，使得输出概率与实际标签 $y$ 之间的差距最小。

## 3.2 无监督学习：聚类

聚类是一种无监督学习算法，用于分类问题。聚类的目标是找到数据集中的簇，使得同一簇内的数据点相似，而不同簇内的数据点不相似。

常见的聚类算法有：

- K均值聚类：K均值聚类假设存在 K 个簇，并尝试将数据点分配到这些簇中。K均值聚类的目标是最小化数据点与其所属簇中心的距离。
- 层次聚类：层次聚类是一种自动选择簇数的聚类算法。层次聚类通过逐步合并数据点或分裂簇来形成一个层次结构。

具体操作步骤如下：

1. 初始化簇中心。
2. 将数据点分配到最靠谱的簇中。
3. 更新簇中心。
4. 重复步骤2和步骤3，直到收敛。

## 3.3 强化学习：Q-学习

强化学习是一种通过与环境的互动来学习如何做出最佳决策的学习方法。Q-学习是一种强化学习算法，用于解决定向的Markov决策过程（MDP）问题。Q-学习的目标是找到一个最佳的动作策略，使得总收益最大。

Q-学习的数学模型公式如下：

$$
Q(s,a) = R(s,a) + \gamma \max_{a'} Q(s',a')
$$

其中，$Q(s,a)$ 是状态 $s$ 和动作 $a$ 的价值，$R(s,a)$ 是状态 $s$ 和动作 $a$ 的奖励，$\gamma$ 是折扣因子。

具体操作步骤如下：

1. 初始化Q值。
2. 从初始状态开始，选择一个动作，并执行该动作。
3. 更新Q值。
4. 重复步骤2和步骤3，直到收敛。

```python
def gradient_descent(X, y, learning_rate, iterations):
    m, n = X.shape
    w = np.zeros(n)
    for _ in range(iterations):
        prediction = X.dot(w)
        loss = (y - prediction).dot(np.log(prediction) - np.log(1 - prediction))
        gradients = X.T.dot(y - prediction) / m
        w -= learning_rate * gradients
    return w
```

```python
import numpy as np

def update_q_values(q_values, next_q_values, learning_rate, discount_factor):
    return np.maximum(q_values + learning_rate * (next_q_values - q_values) * discount_factor, 0)

state = 0
action = 0
reward = 10
next_state = 1
next_q_values = np.zeros(2)

q_values = np.zeros(2)
learning_rate = 0.1
discount_factor = 0.99

for _ in range(1000):
    next_q_values[action] = reward + discount_factor * np.max(q_values)
    q_values = update_q_values(q_values, next_q_values, learning_rate, discount_factor)
    state = next_state

print(q_values)
```
