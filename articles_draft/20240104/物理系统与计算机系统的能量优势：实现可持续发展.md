## 3.1 高效算法
### 3.1.1 动态规划
动态规划（Dynamic Programming，DP）是一种优化算法，它可以用于解决最优化问题。动态规划的核心思想是将问题拆分成子问题，并将子问题的解存储在一个表格中，以便在后续的计算中重复使用。例如，求Fibonacci数列的值可以使用动态规划算法。

动态规划的公式为：
$$
dp[i] = \begin{cases}
    a_i, & \text{if } i=1 \\
    dp[i-1] + a_i, & \text{if } i>1
\end{cases}
$$

### 3.1.2 贪心算法
贪心算法（Greedy Algorithm）是一种基于当前状态下最优解的算法。贪心算法的核心思想是在每个步骤中选择能够带来最大收益的选项，直到问题得到解决。例如，求最大子集和问题可以使用贪心算法。

贪心算法的公式为：
$$
\max_{S \subseteq T} \sum_{x \in S} x
$$

## 4.1 动态规划实例
### 4.1.1 Fibonacci数列
```python
def fibonacci(n):
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + 1
    return dp[n]
```

### 4.1.2 最大子集和
```python
def max_subset_sum(nums):
    sums = [0] * (len(nums) + 1)
    for i in range(1, len(nums) + 1):
        sums[i] = max(nums[i - 1], sums[i - 1] + nums[i - 1])
    return sums[-1]
```
