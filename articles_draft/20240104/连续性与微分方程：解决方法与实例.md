## 1.2 核心概念与联系

在本节中，我们将介绍连续性与微分方程的核心概念和联系。

### 1.2.1 连续性

连续性是指一个函数在某个点上的限值与该点的函数值相等。在数学上，连续性通常表示为：对于任何给定的ε>0，都存在一个δ>0，使得对于任何x满足|x-a|<δ，都有|f(x)-f(a)|<ε。

### 1.2.2 微分方程

微分方程是一种描述函数变化的方程，它包含了函数的一阶或多阶微分与函数本身之间的关系。微分方程可以用来描述各种自然现象和技术系统的变化规律。

### 1.4.1 解微分方程的Python代码实例

在本节中，我们将介绍如何使用Python的SymPy库来解微分方程。

#### 1.4.1.1 解一阶微分方程的Python代码实例

```python
from sympy import symbols, Eq, diff, solve

x, y = symbols('x y')

# 定义微分方程
eq = Eq(diff(y, x), x**2 + y)

# 解微分方程
solution = solve(eq, y)

print(solution)
```

#### 1.4.1.2 解二阶微分方程的Python代码实例

```python
from sympy import symbols, Eq, diff, solve

x, y = symbols('x y')

# 定义微分方程
eq = Eq(diff(y, x, x), x**2 - y)

# 解微分方程
solution = solve(eq, y)

print(solution)
```

### 1.4.2 解偏微分方程的Python代码实例

在本节中，我们将介绍如何使用Python的SymPy库来解偏微分方程。

#### 1.4.2.1 解热传导方程的Python代码实例

```python
from sympy import symbols, Eq, diff, solve

x, y, z, T = symbols('x y z T')

# 定义热传导方程
eq = Eq(diff(T, x, x) + diff(T, y, y) + diff(T, z, z), 0)

# 解热传导方程
solution = solve(eq, T)

print(solution)
```
