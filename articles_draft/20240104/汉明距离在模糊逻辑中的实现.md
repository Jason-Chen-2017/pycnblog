### 3.1汉明距离的计算公式

汉明距离的计算公式如下：

$$
d_H(x, y) = \frac{1}{2} \sum_{i=0}^{n-1} |x_i - y_i|
$$

其中，$x$ 和 $y$ 是两个长度为 $n$ 的二进制序列，$x_i$ 和 $y_i$ 分别是序列 $x$ 和 $y$ 的第 $i$ 个位置的取值，$|x_i - y_i|$ 是两个位置取值的差异，$\frac{1}{2} \sum_{i=0}^{n-1} |x_i - y_i|$ 是所有位置取值差异的和，最终除以 $2$ 得到汉明距离。

    binary_mat = np.zeros((len(values), 2))
    for i, value in enumerate(values):
        binary_mat[i, value] = 1
    return binary_mat

# 汉明距离计算
def hamming_distance(binary_mat1, binary_mat2):
    distance = np.sum(binary_mat1 != binary_mat2, axis=1) / 2
    return distance

def fuzzy_connect(binary_mat1, binary_mat2, operation):
    if operation == 'AND':
        return np.logical_and(binary_mat1, binary_mat2)
    elif operation == 'OR':
        return np.logical_or(binary_mat1, binary_mat2)
    elif operation == 'NOT':
        return np.logical_not(binary_mat1)

        return np.logical_and(binary_mat1, binary_mat2)
    elif laws == 'OR':
        return np.logical_or(binary_mat1, binary_mat2)
    elif laws == 'NOT':

汉明距离与其他距离度量的区别在于它们衡量的是不同类型的差异。汉明距离用于衡量二进制序列之间的差异，而其他距离度量如欧氏距离、马氏距离等用于衡量多维空间中的点之间的距离。汉明距离在信息论、编码论等领域具有重要的应用价值。
