## 3.1正则表达式的基本组成部分

3.特殊字符（special character）：特殊字符是一种特殊的元字符，它们用于匹配特定的字符串模式。例如，“[0-9]”表示一个数字，“[A-Za-z]”表示一个字母。

```python
def is_valid_phone_number(phone_number):
    # 中国手机号码验证正则表达式
    china_phone_regex = r'^1[3-9]\d{9}$'
    # 美国手机号码验证正则表达式
    us_phone_regex = r'^\(1-9\) \d{3}-\d{3}-\d{4}$'
    # 英国手机号码验证正则表达式
    uk_phone_regex = r'^\+\d{4} 7\d{3} \d{7}$'
    
    # 匹配中国手机号码
    if re.match(china_phone_regex, phone_number):
        return "中国手机号码有效"
    # 匹配美国手机号码
    elif re.match(us_phone_regex, phone_number):
        return "美国手机号码有效"
    # 匹配英国手机号码
    elif re.match(uk_phone_regex, phone_number):
        return "英国手机号码有效"
    else:
        return "手机号码无效"

# 测试手机号码验证函数
phone_number = "13912345678"
print(is_valid_phone_number(phone_number))
```
