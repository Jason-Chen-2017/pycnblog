## 1.1 贝叶斯网络的基本概念

贝叶斯网络是一种概率图模型，用于表示随机变量之间的条件依赖关系。它们的主要优点在于可以有效地处理不完全观测的问题，并在有限的数据集下进行预测和推理。贝叶斯网络的核心概念包括：

1. 节点（nodes）：表示随机变量。
2. 边（edges）：表示变量之间的依赖关系。
3. 条件概率（conditional probability）：表示一个变量给定其他变量时的概率。
4. 贝叶斯定理：用于计算条件概率。

## 2.1 贝叶斯网络的核心概念

### 2.1.1 节点和边

节点表示随机变量，边表示变量之间的依赖关系。节点可以分为两类：观测变量（evidence variables）和隐变量（latent variables）。观测变量是可以直接观测到的，而隐变量是无法直接观测到的。

### 2.1.2 条件概率和贝叶斯定理

条件概率是贝叶斯网络的基本概念之一。给定其他变量，一个变量的概率表示为：

$$
P(A|B) = \frac{P(A \cap B)}{P(B)}
$$

贝叶斯定理是贝叶斯网络的核心算法。它允许我们根据已知信息更新我们对未知信息的信念。贝叶斯定理可以表示为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

### 2.1.3 贝叶斯网络的结构

贝叶斯网络的结构是一个有向无环图（DAG），其节点表示随机变量，边表示变量之间的条件依赖关系。贝叶斯网络的结构可以通过学习（例如，使用最大似然估计）或手动指定。

### 2.1.4 贝叶斯网络的参数

贝叶斯网络的参数是它们的概率分布。给定一个贝叶斯网络的结构，我们可以使用最大似然估计、贝叶斯估计或其他方法来估计其参数。

## 4.1 使用Python的pgmpy库实现贝叶斯网络

Python的pgmpy库是一个强大的贝叶斯网络库，可以用于构建、学习和使用贝叶斯网络。以下是一个使用pgmpy库实现贝叶斯网络的示例：

```python
from pgmpy.models import BayesianNetwork
from pgmpy.factors.discrete import TabularCPD
from pgmpy.inference import VariableElimination

# 构建贝叶斯网络
model = BayesianNetwork([('A', 'B'), ('B', 'C'), ('C', 'D')])

# 学习贝叶斯网络的参数
cpds = {
    'A': TabularCPD(variable='A', variable_card=2, domain=[0, 1]),
    'B': TabularCPD(variable='B', variable_card=2, domain=[0, 1],
                    evidence=['A'],
                    values=[[0.8, 0.2], [0.7, 0.3]]),
    'C': TabularCPD(variable='C', variable_card=2, domain=[0, 1],
                    evidence=['B'],
                    values=[[0.6, 0.4], [0.5, 0.5]]),
    'D': TabularCPD(variable='D', variable_card=2, domain=[0, 1],
                    evidence=['C'],
                    values=[[0.4, 0.6], [0.3, 0.7]])
}

model.add_cpds(cpds)

# 使用贝叶斯网络进行推理和预测
inference = VariableElimination(model)
result = inference.query(variables=['D'], evidence={'C': 1})
print(result)
```

model.add_node('C')
model.add_node('D')

model.add_edge('B', 'C')
model.add_edge('C', 'D')

    'B': ConditionalProbabilityTable([[0.8, 0.2], [0.7, 0.3]]),
