# 4.具体代码实例和详细解释说明

## 4.1 主动学习的代码实例
主动学习的代码实例如下：

```python
import numpy as np
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

# 加载数据
X, y = load_data()

# 初始化学习算法
clf = LogisticRegression()

# 初始化样本集
P = np.random.rand(len(X)) < 0.2
X_train, X_test, y_train, y_test = train_test_split(X[P], y[P], test_size=0.8, random_state=42)

# 训练学习算法
clf.fit(X_train, y_train)

# 查询策略
def query_strategy(clf, X_train, X_test, y_train, y_test):
    uncertainty = np.var(clf.predict_proba(X_test), axis=0)
    idx = np.argsort(uncertainty)[-int(len(X_test) * 0.2):]
    X_query, y_query = X_test[idx], y_test[idx]
    return X_query, y_query

# 主动学习
while True:
    X_query, y_query = query_strategy(clf, X_train, X_test, y_train, y_test)
    X_train, y_train = np.vstack((X_train, X_query)), np.hstack((y_train, y_query))
    clf.fit(X_train, y_train)
    acc = accuracy_score(y_test, clf.predict(X_test))
    if acc >= 0.95:
        break

print("主动学习的准确率：", acc)
```

## 4.2 详细解释说明
上述代码实例中，我们首先加载了数据，并初始化了学习算法和样本集。然后，我们使用查询策略选择一些样本进行训练。在每一轮训练后，我们计算模型的准确率，如果准确率达到0.95，则停止训练。
