## 4.1 旋转

```python
import numpy as np

def rotate(matrix, point, angle):
    angle_rad = np.deg2rad(angle)
    cos_angle = np.cos(angle_rad)
    sin_angle = np.sin(angle_rad)
    rotation_matrix = np.array([[cos_angle, -sin_angle, 0], [sin_angle, cos_angle, 0]])
    translated_point = np.dot(rotation_matrix, point)
    return translated_point
```

在上述代码中，我们首先将旋转角度从度转换为弧度，然后构建旋转矩阵，将旋转矩阵与原始点进行点积运算，得到旋转后的点。
