## 4. 具体代码实例和详细解释说明

在这里，我们以一个简单的优化问题为例，来展示鱼群算法的具体实现。

### 4.1 问题描述

求解一个简单的优化问题：
$$
f(x) = -x^2 + 4x
$$
其中，$x \in [0, 10]$。

### 4.2 代码实现

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义目标函数
def objective_function(x):
    return -x**2 + 4*x

# 初始化鱼群
def initialize_fish_swarm(num_fish, search_space):
    return np.random.uniform(search_space[0], search_space[1], num_fish)

# 更新鱼群中心
def update_fish_swarm_center(fish_swarm):
    return np.mean(fish_swarm, axis=0)

# 更新鱼的位置
def update_fish_position(fish_swarm, fish_swarm_center, search_space, w, c1, c2):
    for i in range(len(fish_swarm)):
        r1 = np.random.rand()
        r2 = np.random.rand()
        fish_swarm[i] = fish_swarm[i] + w*r1*(fish_swarm_center - fish_swarm[i]) + c1*r2*(search_space[1] - fish_swarm[i]) + c2*r2*(fish_swarm_center - fish_swarm[i])
    return fish_swarm

# 主函数
def main():
    num_fish = 50
    search_space = [0, 10]
    w = 0.7
    c1 = 1.5
    c2 = 1.5
    max_iter = 100

    fish_swarm = initialize_fish_swarm(num_fish, search_space)
    best_fish = fish_swarm[0]
    best_fish_value = objective_function(best_fish)

    for i in range(max_iter):
        fish_swarm_center = update_fish_swarm_center(fish_swarm)
        fish_swarm = update_fish_position(fish_swarm, fish_swarm_center, search_space, w, c1, c2)
        best_fish_index = np.argmin(objective_function(fish_swarm))
        if objective_function(fish_swarm[best_fish_index]) < best_fish_value:
            best_fish = fish_swarm[best_fish_index]
            best_fish_value = objective_function(best_fish)

    print("最优解：", best_fish)
    print("最优值：", best_fish_value)

    plt.plot(fish_swarm, 'o', label='Fish Swarm')
    plt.plot([best_fish, best_fish], np.min(search_space), np.max(search_space), 'r-', label='Best Fish')
    plt.xlabel('Fish Position')
    plt.ylabel('Fish Value')
    plt.legend()
    plt.show()

if __name__ == '__main__':
    main()
```

### 4.3 解释说明

1. 首先，我们定义了目标函数`objective_function`，并设定了搜索空间。
2. 接着，我们使用`initialize_fish_swarm`函数初始化鱼群，将候选解随机分布在搜索空间中。
3. 然后，我们使用`update_fish_swarm_center`函数计算鱼群中心，并使用`update_fish_position`函数更新鱼的位置。
4. 最后，我们使用主函数`main`进行迭代，直到满足终止条件，即达到最优解。
5. 在这个例子中，我们使用了`matplotlib`库绘制了鱼群的分布和最优解的位置。
