以下是一些具体的代码实例和详细解释说明：

1. Python：

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 删除缺失值
data = data.dropna()

# 填充缺失值
data['age'] = data['age'].fillna(data['age'].mean())

# 修复错误值
data['age'] = data['age'].apply(lambda x: min(max(x, 0), 120))

# 过滤错误值
data = data[data['age'] <= 100]
```

2. R：

```R
# 读取数据
data <- read.csv('data.csv')

# 删除缺失值
data <- na.omit(data)

# 填充缺失值
data$age <- ifelse(is.na(data$age), mean(data$age, na.rm = TRUE), data$age)

# 修复错误值
data$age <- ifelse(data$age < 0 | data$age > 120, 0, data$age)

# 过滤错误值
data <- data[data$age <= 100, ]
```
