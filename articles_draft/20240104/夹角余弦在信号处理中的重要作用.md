## 1.1 信号处理的基本概念

信号处理是对信号的数学分析和处理，主要包括信号的定义、分析、变换、滤波、恢复等方面。信号通常是时间域或频域的函数，可以表示物理现象的变化，如声音、光线、电磁波等。信号处理的目的是提取信号中的有用信息，消除噪声和干扰，改变信号的特性，实现信号的传输、存储、识别等功能。

## 1.2 夹角余弦的基本概念

夹角余弦（cosine）是一种数学函数，表示两个向量之间的夹角。夹角余弦函数的定义为：

$$
\cos(\theta) = \frac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\| \|\mathbf{b}\|}
$$

其中，$\mathbf{a}$ 和 $\mathbf{b}$ 是向量，$\cdot$ 表示点积，$\|\mathbf{a}\|$ 和 $\|\mathbf{b}\|$ 表示向量的模（长度）。夹角余弦函数的范围是 $[-1,1]$，表示角度为 $0^\circ$ 到 $180^\circ$ 的取值。

傅里叶变换是信号处理中的一个重要工具，可以将时域信号转换为频域信号，从而方便分析和处理。傅里叶变换的定义为：

$$
X(f) = \int_{-\infty}^{\infty} x(t) e^{-j2\pi ft} dt
$$

$$
x(t) = \int_{-\infty}^{\infty} X(f) e^{j2\pi ft} df
$$

其中，$x(t)$ 是时域信号，$X(f)$ 是频域信号。傅里叶变换可以表示为夹角余弦的积分形式：

$$
X(f) = \sum_{n=-\infty}^{\infty} x(n) \delta(f - f_s n) = \sum_{n=-\infty}^{\infty} x[n] e^{-j2\pi f_s n}

其中，$f_s$ 是采样频率，$\delta(f)$ 是谐元函数。从上述公式可以看出，傅里叶变换中，信号的相位信息是通过夹角余弦来表示的。因此，夹角余弦在傅里叶变换中具有重要作用。

import matplotlib.pyplot as plt

def low_pass_filter(cutoff_freq, sample_freq, signal):
    nyquist_freq = sample_freq / 2
    normalized_cutoff_freq = cutoff_freq / nyquist_freq
    filter_coefficients = 0.5 * (1 - np.cos(2 * np.pi * normalized_cutoff_freq))
    filter_coefficients = np.concatenate(([1], filter_coefficients, [1]))
    filtered_signal = np.convolve(signal, filter_coefficients, mode='valid')
    return filtered_signal

def phase_recovery(signal):

Q: 夹角余弦为什么能表示信号之间的相位关系？
A: 夹角余弦是一个向量的内积的正弦值，它可以表示两个向量之间的夹角。当两个信号的相位关系保持一致时，它们的夹角余弦为1，表示正相位；当两个信号的相位关系相反时，它们的夹角余弦为-1，表示负相位。因此，夹角余弦可以用来表示信号之间的相位关系。

Q: 夹角余弦滤波有什么优缺点？
A: 优点：
1. 可以根据信号的频率特性进行滤波。
2. 可以保留或消除某些频率成分，实现信号的过滤。
缺点：
1. 滤波器设计较为复杂，需要根据信号的频率特性进行设计。
2. 滤波后的信号可能会损失一些信息。
