         # 2.函数式编程概述
         
         ## 什么是函数式编程？
         
         函数式编程（functional programming）是一种编程范式，它试图通过对函数式编程的纯粹性、无副作用、数据不可变性等特性的运用来构建软件系统。其特征是高阶函数（Higher-order functions）和不可变数据结构。简单来说，就是对数据的计算方式进行了抽象。只在输入参数发生变化的时候才会产生输出，且没有副作用。换句话说，就是通过“函数”的方式来编程。
         
         ### 函数式编程与命令式编程的区别
         
         命令式编程指的是基于语句的编程语言，如C、Java、Python等。其中，每一条语句都是执行某个操作，依赖上一条指令的结果，并产生一个结果。函数式编程在这种编程方式下倾向于避免共享状态，尽量降低数据依赖。这就是为什么函数式编程可以实现并行化，能更好地解决问题，适合用于并行计算。
         
         ### 为什么要学习函数式编程？
         
         通过学习函数式编程，你可以更好地了解计算机科学，理解世界的本质。还可以更有效地应对实际场景中的问题，解决复杂的问题。例如，JavaScript在过去几年里已经成为最热门的前端开发语言，它可以用来编写函数式编程风格的代码。这就是为什么学习函数式编程非常重要。
         
         ### 函数式编程的优缺点
         
         #### 优点
         
         **更少的代码**：函数式编程往往意味着代码更少，这使得代码易读、易懂、易维护。如果你的代码可以被抽象成更小的函数，那么阅读和修改它的难度就会降低。

         　**可靠性**：函数式编程所提供的很多特性可以让你的代码更加健壮，避免很多潜在的错误。

         　**并行性**：通过并行化，函数式编程可以有效地利用计算机的多个核心来提升性能。

         　**调试简单**：因为函数式编程避免了共享状态，所以你的代码的调试和测试会变得更加容易。

         　**抽象能力强**：函数式编程可以帮助你更好地处理复杂的问题。

         　**更加贴近数学模型**：函数式编程可以更加贴近数学模型，这是其他编程语言所不能比拟的。

         　**自然语言描述**：函数式编程的函数看起来很像数学上的函数。如果你熟悉函数式编程语言，你就可以用自然语言来描述函数式编程的问题，并直接验证你的猜想是否正确。


         #### 缺点

         　**不容易学习**：函数式编程可能比较难以学习，因为它要求你学习一些较新的编程理念。

         　**复杂度高**：函数式编程通常比命令式编程更复杂，需要你有一定的数学功底才能理解它的一些特性。

         　**无法应用于所有的场景**：函数式编程不一定适合所有场景。虽然它的一些特性可以在某些特定场景下使用得当，但大多数情况下还是无法替代命令式编程。


         ## 实践方法论
         
         如果你是一个初学者或是一个刚接触函数式编程的人，下面是几个你可以参考的资源：
         
         ### MOOCs/MOOCs

          　MOOCs（Massive Open Online Courses）即大规模开放式课程，你可以找到很多免费的函数式编程课程。其中Coursera的Functional Programming in Scala课程和MITx的6.006 Introduction to Computer Science and Programming in Python都提供了详细的教程。
         
          　当然，我自己也在学习函数式编程，并会更新此系列视频的发布进度。

         
         ### Books

         　如果你想要购买一本有关函数式编程的书籍，可以考虑这些资源：

         　· Harry Wampler's Essential Clojure：如果你想了解Clojure，这个书就值得一看。这本书覆盖了函数式编程的所有方面，包括数据结构、集合、递归、宏、惰性求值和类型系统等。

         　· David Beazley's Functional Programming Languages：这本书可以帮你快速了解函数式编程的各种语言。你可以选择一门你熟练使用的语言来阅读。
