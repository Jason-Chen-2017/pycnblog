# 1.简介
         
　　“中国数字政权”是一个充满争议的新兴政治课题，其理论基础、社会机制、实践方式等复杂性都无法完全确立。基于对此类问题的关注和探索，我们试图对数字政权的构建进行一个系统的整体设计方案，并将其落地实施。本文综合国内外研究成果，分析当前数字政权建设存在的问题及局限，提出了一种可能性的方案。
        
         # 2.相关背景
         ## 2.1 数字技术的兴起和应用
       　　 “2008年底，我国科技部科学技术委员会召开关于打造全国信息化工作十年行动方案座谈会，共同决定了“2009—2019”规划，明确了建设国家信息化体系的目标。在这期间，由于我国数字经济发展迅速，出现了一批数字产业、服务领域的创新案例，如互联网金融、移动互联网、云计算、物联网、大数据、人工智能、区块链等，而这些产品和服务引起了广泛的社会关注。“数字政府”成为当时热词，涌现了一系列与政务信息相关的数字化平台，如政府网站、政务大厅、办事指南、法律问询、公众参与等。
        
         在此背景下，中国的政治参与渠道也越来越多样化，包括微博、微信、电话、网络投票、选举、参与各种活动……网络参与活动也呈现爆炸性增长，如2017年，网民协调的组织“世界民主自由联盟”通过网络集结、讨论民主、人权、平等等议题，而2018年即使是在社交媒体上的流行病号“不忧国忧民”也极大的冲击了普通人的参与意识。
        
       ![](https://pic2.zhimg.com/v2-f38c1a0d0ddcb7d6761b2cf11238fffa_r.jpg)
        
        ## 2.2 普通民众对政治参与的需求变化
       　　随着经济的发展，人们越来越关注利益分配。政府也逐渐成为资本运作的对象，它通过税收收入、土地征用、债券赎回等方式筹措公共资源。但普通民众对公共事务的关注度也越来越高，他们希望政务部门能够提供更有效率的服务，提升生活品质、保障公共权力。
        
       ![](https://pic1.zhimg.com/v2-fc9d49a4fdadce938aa5de60cfaf52bb_r.jpg)
        
        ## 2.3 政府角色转型升级——“微观中国”的出现
      　　“微观中国”是指全面建设小康社会，在高度发达的民主宪政的同时，实现财富效益最大化。目前，我国正处于转型发展的关键时期，“中国梦”的脚步也越来越近。政府对于“微观中国”的愿景也越来越清晰，它所承担的任务是全面推进政府运行、完善政治制度、培育强大的人才队伍、促进社会公平正义。
       
       ![](https://pic3.zhimg.com/v2-0fced0f36bc0a81e4cfacbf8e6a6dfca_r.jpg)
        
       # 3.数字政权建设的必要性
       　　“数字政权”的定义模糊且缺乏统一标准，这里我们试图从宏观视角总结数字政权的必要性。数字政权建设有助于解决如下四个问题：
        
        1．防止垄断和弱肉强食
        在“911”事件发生后，美国、加拿大等一些国家的人民币、港币等货币贬值引发巨额美元抛售潮，引发全球金融危机，更有甚者导致中国的外汇储备几乎耗尽。中国数字货币联盟（CCB）主席孙春兰表示：“如果没有数字货币，就不会有今天的这种情况。”
        
        2．提升国家治理水平
        近年来，各国在解决贫困问题、保护环境、化解疾病、改善教育等方面都开展了多种尝试。但是，如何将政务信息整合到国家治理结构中，解决现有治理模式中的不足？中国政法大学李克强教授提出的“数字新政府”理念就是这样的一个尝试。
        
        3．确保公民个人权利
        当前，越来越多的人把精力放在数字化上，希望通过数字技术让生活变得更便捷、高效、安全。但一些公民发现自己的权利被剥夺了，如权利获得，权利申辩，权利追究等。只有建立起数字政权，公民个人权利才能得到公平而可靠的保障。
        
        4．促进社会和谐
        数字政权可以通过各种形式来解决社会不公平问题，如支持弱势群体、保护法治秩序，以及改善民生福祉。通过数字政权，每个公民都可以享受到公平、公正、透明的待遇，也许某些项目可能会惠及某些特殊群体。
        
        # 4.政党制度的发展方向
       　　在传统的民主制度下，政党由一定的政治权威、候选人产生，并得到普遍认可，成为政权的主要依据。随着国家治理结构的日益复杂化，政党制度也面临着越来越多的挑战。“我国政党制度正在发生革命性的变化”。
        
       ![](https://pic2.zhimg.com/v2-0fb63c94f9c3b81e16ab7f4c5a1487d9_r.jpg)
        
        ### 4.1 数字政党的形态
       　　数字政党实际上是数字化的公民团体。它与其他公民团体的区别在于，它不是依靠一党制执行政治决策，而是透过网络舆论来进行竞争，具有更高的灵活性和弹性。它也受制于社会机制的约束，并依赖于程序性规则，不能自发产生或退出。它还受制于严格控制信息流通渠道的影响，因而难以走向独立的政治实体。
        
        1． 垂直型政党
        
        垂直型政党通常由核心成员组成，与新闻、出版、电影、音乐等特定领域有关。它们通过组织各种抗争活动、提升声势，或者凝聚共识来推动社会问题的解决。例如，农村地区出现反腐败斗争的“全国青年五月天”，歌手徐诗琪的“胡歌梦”、英国工党的“左翼文宣阵线”。
        
        2． 水平型政党
        
        水平型政党由平民组成，在各个领域均有涉猎，代表着不同利益阶层的声音。它们采用一套简单、直接的表达方式，鼓励年轻人发表看法，进行公民教育，以期激发大家的社会责任感。例如，胡锦涛夫妻等担任高级官员的政党，陈希、蔡奇、张铁志等担任高校教师的政党，陈光诚等为抗衡习近平的政党。
        
        3． 联邦制政党
        
        联邦制政党是指政党体系由单一的政权所组成，通常具备较少的地区差异性。它们的特点是对社会各阶层均有代表性，具有较强的政治影响力。联邦制政党一般由众多成员组成，通过政治竞争和选举产生。例如，美国的“民主党”和“自由党”就是典型的联邦制政党。
        
        # 5.当前政党制度存在的弊端和局限
       　　当前的政党制度在很大程度上受制于当今社会的信息技术和信息分配方式。政党通过公布政纲，推动政府进行政策制定，将信息信息技术手段如微信、微博、论坛等部署到政务系统中。因而，政党制度不仅对公众来说难以接受，而且有可能导致政权内部的不信任感。
        
        更为重要的是，政党制度又往往导致政治权力过分集中，导致少数政治精英掌握超越国家控制权限，引发政权内部的分裂和内部冲突。而政党制度又无法真正做到对政府的全覆盖，因此也会成为国家政治生活中不可忽略的一部分。
        
        
        # 6.政党制度的可能性
       　　随着互联网技术的飞速发展，越来越多的人将更多的注意力放置在数字技术上，而非政治层面。因此，在国际上出现了一种“开放式政党制度”，指公众可以在网上注册自己的政党并加入其所属的政党，进行政治竞争。
        
        
       ![](https://pic1.zhimg.com/v2-92935a292264cd2c6c6a2cfcebf19bcf_r.jpg)
        
        随着这种公开竞争的政党制度的出现，很多新型的政党制度可能具有以下特征：
        
        1. 政治性强
        
        通过竞争性选举产生，以期削弱政党的地域性、民族主义倾向。
        
        2. 公开性
        
        公众可以任意加入，并通过政党纲领、报告或演说的方式参与到政党的决策过程之中。
        
        3. 自治性
        
        每个政党拥有自己的数据中心，可以根据自己的策略来实施政策，而无需等待组织上层的命令。
        
        4. 分散性
        
        政党之间相互独立，互不干扰，公众没有义务保持对某个政党的信任。
        
        不过，这种政党制度是否适用于中国是一个非常重要的问题。作为最早的国家级政党，以“中国特色社会主义阵营”的组成以及2016年“中共中央关于加强和改进新形势下党的作风建设若干问题的决定”，两派政党展开激烈的争论。这种冲突并没有触及到现有体制本身，只是推动政党制度的发展。如今，在十八大之后，党内矛盾的出现让政党制度再次成为有争议的问题。
        
        如果数字政党制度真正走向市场，那么政党制度还有以下的一些挑战：
        
        1. 竞争激烈
        
        从“先锋队”到“火箭队”，需要极其精细的竞争机制。
        
        2. 数据中心的规模
        
        有太多的数据需要保存，保证政党决策的实时性。
        
        3. 信息管控
        
        政党通过竞争性选举产生，但政治决策还是由一群政客来完成，需要有专门的信息室来对外发布政策。
        
        此外，要想将政党制度引入中国，首先要解决以下两个问题：
        
        1. 发展党员与建立党组织关系
        
           为什么党员越来越多？党员数量有多大？党组织又是如何建立的？这些都是问题。如果不能顺利解决这些问题，数字政党制度势必难以走向市场。
           
        2. 如何对公众负责
        
        对公众负责，不能满足于让政党占据有限的空间；对公众负责，也不能因为一方面的言行不当而成为一个阴谋集团。
        
        在解决以上两个问题的过程中，政党制度的可能性也将越来越大。
