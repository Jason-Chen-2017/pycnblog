# 1.简介
  


随着智能设备、移动互联网应用和实体物品等新型交互形式的兴起，用户对界面设计的需求越来越高。然而，在现实世界中，很多UI设计师并不具备专业的美观视觉设计知识，因此往往无法通过简单的复制粘贴完成好的界面设计。因此，作为UI设计领域的专业人员，除了掌握技术外，更需要能够有关美感、创意、情感、专业的综合能力。

本文从用户体验设计三个方面，包括认知心理学、广告设计、计算机图形学、平面设计等角度，阐述了用户界面设计的理论基础、方法和技巧。文章还介绍了不同类型产品的UI设计策略，其中包括电子产品、手机APP、PC端GUI界面、汽车界面等，为读者提供了一个全面的视野。

本文作者为国际顶级的UI设计专家、PhD级的导演、编剧、影评人Terry Pereira。他是加拿大多伦多大学商学院的PhD生，曾任Facebook设计总监，担任过几年UI设计师、互动设计师、产品经理等职务。他的研究主要集中在人机交互设计、广告设计、游戏UI设计等领域，涉及到许多复杂的理论和技术。

# 2.认知心理学与用户体验设计

“认知心理学”（Cognitive psychology）是指利用各种认知机制分析人的行为和活动，从而研究人的思维、判断力、注意力、推理能力、归纳理解能力等。美国哈佛大学的教授马修·奥布林和斯坦福大学的教授詹姆士·卡尼曼将认知心理学分为五个层次，即认知（perception），运用（application），探索（exploration），情绪（emotional），和个性化（personalization）。

在UI设计过程中，“认知心理学”可以帮助我们提升产品的易用性、舒适性、满足度，并保证用户满意度。在日益复杂的世界环境下，人们逐渐习惯了快速反应，而对于稍显拖沓的操作，却难以察觉其中的不自然。因此，当用户对系统产生困扰时，良好的用户体验设计将极大地增强用户的满意度。

如同一张纸张，它既有线条又有颜色，但同时也有透明感和亲切的气息。如果想要美观且易于使用的界面，就需要关注用户的使用习惯，从而塑造出符合人类习惯的界面。

## 2.1 认知特征

认知特征可以帮助我们更好地理解用户的能力，以及他们可能遇到的各种情况。常用的认知特征如下表所示：

1. 直觉性（Sense of Smell and Taste）

   用户对界面元素的色彩、材质、布局、大小等因素的直觉。例如，假设一个按钮上有一个放大的镜像图像，通过这种直觉就知道该按钮是一个可点击的按钮。

2. 形象化（Aesthetic Experience）

   使人能够建立强烈的视觉震撼，并在脑海中留下深刻印象，也就是所谓的“脸部刺激”。例如，当看到一些酷炫的UI元素时，都会流露出一种强烈的赞叹，而不是仅凭直觉或预期。

3. 概念化（Conceptual Knowledge）

   能够借助抽象的符号来抽象地理解事物，也就是所谓的“概念网络”。例如，当看到某个物件看似毫无意义却又突兀的样子时，就会认为这是由某种先天属性决定的，而不是某个因果关系。

4. 社会性（Sociality and Relationships Awareness）

   在社交互动中，人们有能力处理复杂的情感关系。例如，当在网上购物时，能够准确判断价格和库存信息，并且有能力帮助顾客解决疑惑。

## 2.2 使用场景

不同类型的产品需要不同的UI设计方法，为了更好的满足用户的需求，我们应该考虑这些特征：

1. 效率

   UI设计师应该注重界面的设计，以便用户能够快速找到想要的信息或者进行任务，并减少额外的时间开销。

2. 兼容性

   应当为用户提供尽可能广泛的适用范围，以便于更好的满足不同的平台。

3. 直观性

   对用户来说，要直观而精准的呈现信息和功能是非常重要的。UI设计师应该熟练掌握各种绘画工具和标志设计技法，并善于运用动画效果，使得UI元素的变化更加生动有趣。

4. 可访问性

   UI设计应当考虑用户残障、阅读障碍、移动设备和屏幕阅读器的可用性。

## 2.3 决策过程

用户体验设计的决策过程由以下几个阶段组成：

1. 需求调研：收集目标用户的使用数据、竞品分析、市场调研、用户访谈、观察、用户测试等。
2. 用户研究：收集用户的视觉、听觉、触觉、嗅觉、味觉、身体感觉等能力，进行相关测试。
3. 产品设计：根据用户的能力、场景和需求进行产品设计，并进行可用性测试。
4. 投产部署：把产品转化为实际产品，通过市场调查、受众群体宣传等方式推广。

# 3.产品设计策略

## 3.1 电子产品设计策略

电子产品是最容易被忽略的一类产品，特别是那些对我们周遭的生活影响很大的产品。它的设计策略有很多种，这里列举一些常见的设计策略。

1. 结合科技与生活：结合科技与生活的产品设计，可让产品更真实地融入我们的生活，提升用户的消费体验。

2. 多样化的功能：电子产品具有丰富的功能，为了更好地服务用户，需要提供更多样化的功能。

3. 个性化定制：个性化定制能让用户拥有个性化的产品体验，可提升用户的留存率。

4. 尊重隐私：保护用户隐私是电子产品的基本原则，做好个人信息保护，也是电子产品的设计之一。

5. 数据安全：考虑到用户的个人隐私，电子产品的存储空间也要相对较小。数据安全可以成为电子产品的设计关键点之一。

## 3.2 手机APP设计策略

手机APP是手机用户接触最频繁的应用，也是最容易被忽略的一种互联网产品。为了满足用户的需求，APP设计者会充分考虑人机交互的设计原则。以下是一些常见的APP设计策略：

1. 简单直接：APP简单易懂的界面设计，能够吸引用户使用。

2. 清晰有效：清晰有效的导航菜单，能让用户快速定位到自己需要的功能。

3. 有趣有益：玩游戏、社交分享、新闻阅读，都是典型的有趣有益的功能。

4. 个性化推荐：APP推荐功能，能够给用户提供个性化推荐，促进用户增长。

5. 更贴近生活：APP在设计时，应当时尚化、符合大众审美的风格，并结合用户日常生活的习惯进行改造。

## 3.3 PC端GUI界面设计策略

PC端GUI界面是多年来人们最喜欢的计算应用模式。为了提升用户体验，PC端GUI设计者需要关注以下几个方面：

1. 操作便利：为了方便用户操作，需要做到界面简单、操作简洁。

2. 美观大方：界面设计要不遗余力地呈现出高雅美观的外观。

3. 用户友好：界面设计要考虑用户的认识和理解能力。

4. 功能丰富：界面设计应该提供足够丰富的功能。

5. 数据安全：在数据的处理上，要有适当的安全防护措施。

## 3.4 汽车界面设计策略

汽车是一个比较特殊的产品类，用户需要在驾驶中接受外界信息，并做出相应的调整。因此，汽车的界面设计需要结合驾驶员的视觉、肢体和心理感知。以下是一些汽车界面的设计策略：

1. 专注操控：汽车的操控系统太复杂，需要专门设计一种操控手段。

2. 时尚元素：汽车需要时尚化的外观和配色方案，让驾驶员在驾驶中更容易区分自己和驾驶室。

3. 小巧轻量：满足用户的空间感知需求，汽车的界面需要保持最小的尺寸。

4. 自动化提示：让驾驶员在驾驶中获得提示，降低后视镜对驾驶员的影响。

5. 与驾驶习惯匹配：充分利用驾驶习惯，不断迭代产品设计，优化用户驾驶体验。
