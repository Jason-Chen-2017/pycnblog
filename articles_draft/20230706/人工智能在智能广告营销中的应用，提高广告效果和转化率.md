- 模型预测结果：将预测结果与实际购买记录进行对比，得出定向投放广告的效果。

## 4.3. 核心代码实现

首先，安装所需的 Python 库：`pandas`，`numpy`，`mlxtend` 和 `sklearn`。然后，编写以下代码实现核心模块：
```python
import pandas as pd
import numpy as np
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

# 读取数据
def read_data(data_file):
    data = []
    with open(data_file, 'r') as f:
        for line in f:
            data.append([float(x) for x in line.strip().split(',')])
    return np.array(data)

# 数据预处理
def preprocess(data):
    # 提取特征
    X = data[:, :-1]
    y = data[:, -1]
    # 划分训练集和测试集
    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)
    # 线性回归模型训练
    model = LinearRegression()
    model.fit(X_train, y_train)
    # 预测
    y_pred = model.predict(X_test)
    # 计算 MSE
    mse = mean_squared_error(y_test, y_pred)
    return (X_test, y_test, mse)

# 应用示例
data = read_data('user_data.csv')
y_pred, mse, _ = preprocess(data)

# 计算模型的预期效果
print("Mean Squared Error (MSE): ", mse)

# 测试模型的效果
X_test, y_test, _ = preprocess(data)
model = LinearRegression()
model.fit(X_test, y_test)
y_pred = model.predict(X_test)
print("Model Prediction: ", y_pred)
```

数据预处理的具体方法根据问题的实际情况而异。通常，可以采用以下一些方法：
