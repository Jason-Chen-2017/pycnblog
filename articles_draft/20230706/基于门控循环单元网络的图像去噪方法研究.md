# 2. 技术原理及概念

## 2.1. 基本概念解释

去噪算法可以分为两大类：基于统计学的方法和基于机器学习的方法。其中，基于机器学习的方法又可以分为监督学习和无监督学习两种。

监督学习是指利用已有的数据来训练模型，从而实现去噪的功能。常见的监督学习方法包括：均方误差（MSE）、结构相似性指数（SSIM）、差分图（DIF）等。

无监督学习是指利用已有的数据来训练模型，但不提供任何有用的信息来指导模型，从而实现去噪的功能。常见的无监督学习方法包括：滤波、降噪、生成等。

3. 确定损失函数：根据问题的不同，选择合适的损失函数来评估模型的性能，如MSE、SSIM、DIF等。

4. 训练模型：使用准备好的数据集，按照一定的训练步骤来训练GRU模型，并最终得到去噪的效果。

## 2.3. 相关技术比较

目前，图像去噪的方法有多种，主要包括均方误差（MSE）、结构相似性指数（SSIM）、差分图（DIF）等。而基于GRU的图像去噪方法是近年来比较流行的一种方法。

相比于传统的统计学方法，基于GRU的图像去噪方法具有以下优点：
