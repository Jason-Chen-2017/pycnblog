1. 协同过滤算法：$$
R_{ij} = \sum_{k \in N_i} \frac{sim(i,k)}{|N_i|} \cdot R_{kj}
$$

其中，$R_{ij}$ 表示用户 $i$ 对物品 $j$ 的评分，$N_i$ 表示与用户 $i$ 相似的用户集合，$sim(i,k)$ 表示用户 $i$ 和用户 $k$ 之间的相似度，$|N_i|$ 表示与用户 $i$ 相似的用户集合的大小。

1. 基于内容的推荐算法：$$
P(q|d) = \frac{p(d|q) \cdot p(q)}{\sum_{q'} p(d|q') \cdot p(q')}

其中，$P(q|d)$ 表示给定文档 $d$ 的相关性度量，$p(d|q)$ 表示文档 $d$ 与查询 $q$ 的相关性，$p(q)$ 表示查询 $q$ 的概率，$p(q')$ 表示其他查询的概率。

其中，$R_{ij}$ 表示用户 $i$ 对物品 $j$ 的评分，$R_{ij}^{content}$ 表示基于内容的评分，$R_{ij}^{collaborative}$ 表示基于协同过滤的评分，$\alpha$ 表示混合推荐的权重。

### 1.4.2 学习资源推荐

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 基于内容的推荐
def content_based_recommendation(user_interest, resources):
    vectorizer = TfidfVectorizer()
    resource_matrix = vectorizer.fit_transform(resources)
    user_vector = vectorizer.transform([user_interest])
    similarity = cosine_similarity(user_vector, resource_matrix)
    recommended_resources = np.argsort(similarity[0])[::-1]
    return recommended_resources

# 基于协同过滤的推荐
def collaborative_filtering_recommendation(user_id, user_ratings, resource_ratings):
    user_similarity = {}
    for user in user_ratings.keys():
        if user != user_id:
            similarity = 1 - np.linalg.norm(user_ratings[user] - user_ratings[user_id]) / np.linalg.norm(user_ratings[user] - user_ratings[user_id])
            user_similarity[user] = similarity
    weighted_ratings = {}
    for user, similarity in user_similarity.items():
        weighted_ratings[user] = user_ratings[user] * similarity
    recommended_resources = np.argsort(np.sum(weighted_ratings[user_id]))[::-1]
    return recommended_resources

# 基于内容与协同过滤的混合推荐
def hybrid_recommendation(user_interest, user_ratings, resource_ratings):

    collaborative_recommended_resources = collaborative_filtering_recommendation(user_id, user_ratings, resource_ratings)
    hybrid_recommended_resources = set(content_recommended_resources).intersection(set(collaborative_recommended_resources))
```
