## 1.4 具体代码实例和详细解释说明

以下是一个简单的物联网设备数据收集和传输的代码实例：

```python
import paho.mqtt.client as mqtt

def on_connect(client, userdata, flags, rc):
    print("连接成功")

def on_message(client, userdata, msg):
    print("收到消息：", msg.payload.decode())

client = mqtt.Client()
client.on_connect = on_connect
client.on_message = on_message

client.connect("127.0.0.1", 1883, 60)
client.loop_start()

client.publish("test/topic", "Hello, world!")
```

这个代码实例使用了Paho库，它是一个开源的MQTT库。首先，我们定义了两个回调函数，一个是连接成功的回调函数，一个是收到消息的回调函数。然后，我们创建了一个MQTT客户端对象，并为它设置了连接成功的回调函数和收到消息的回调函数。接着，我们使用connect方法连接到服务器，并使用loop_start方法启动循环。最后，我们使用publish方法发布一条消息。
