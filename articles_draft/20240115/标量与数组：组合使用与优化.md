## 4.1 标量与数组的组合使用

### 4.1.1 示例1：计算数组和

```python
def array_sum(arr):
    total = 0
    for i in arr:
        total += i
    return total

arr = [1, 2, 3, 4, 5]
print(array_sum(arr))  # Output: 15
```

### 4.1.2 示例2：计算数组平均值

```python
def array_mean(arr):
    total = 0
    for i in arr:
        total += i
    return total / len(arr)

arr = [1, 2, 3, 4, 5]
print(array_mean(arr))  # Output: 3.0
```

### 4.1.3 示例3：计算数组和的方差和标准差

```python
def array_var(arr):
    mean = array_mean(arr)
    total = 0
    for i in arr:
        total += (i - mean) ** 2
    return total / len(arr)

def array_std(arr):
    return array_var(arr) ** 0.5

arr = [1, 2, 3, 4, 5]
print(array_var(arr))  # Output: 2.0
print(array_std(arr))  # Output: 1.4142135623730951
```
