1. 欧几里得距离：欧几里得距离是用来衡量两个地理空间对象之间距离的方法。欧几里得距离公式为：

$$

$$

其中，$d$ 是欧几里得距离，$x_1$ 和 $y_1$ 是第一个地理空间对象的坐标，$x_2$ 和 $y_2$ 是第二个地理空间对象的坐标。

$$

$$

其中，$d$ 是曼哈顿距离，$x_1$ 和 $y_1$ 是第一个地理空间对象的坐标，$x_2$ 和 $y_2$ 是第二个地理空间对象的坐标。

3. 余弦相似度：余弦相似度是用来衡量两个地理对象的属性之间相似性的方法。余弦相似度公式为：

$$
sim(A, B) = \frac{A \cdot B}{\|A\| \|B\|}

其中，$sim(A, B)$ 是两个地理对象的属性之间的相似度，$A$ 和 $B$ 是两个地理对象的属性向量，$\|A\|$ 和 $\|B\|$ 是两个属性向量的长度。

```python
# 定义两个地理空间对象的坐标

```

2. 使用 Python 的 numpy 库计算曼哈顿距离：

```python

point2 = np.array([3, 4])

# 计算曼哈顿距离

print(manhattan_distance)
```

