1. 线性回归模型：线性回归模型可以用于预测和分析连续型数据，其公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

其中，$y$ 是预测值，$\beta_0$ 是截距，$\beta_1$、$\beta_2$、...、$\beta_n$ 是系数，$x_1$、$x_2$、...、$x_n$ 是自变量，$\epsilon$ 是误差。

2. 逻辑回归模型：逻辑回归模型可以用于预测和分析离散型数据，其公式为：

$$
P(y=1|x_1,x_2,...,x_n) = \frac{1}{1+e^{-\beta_0-\beta_1x_1-\beta_2x_2-...-\beta_nx_n}}
$$

其中，$P(y=1|x_1,x_2,...,x_n)$ 是预测值，$\beta_0$ 是截距，$\beta_1$、$\beta_2$、...、$\beta_n$ 是系数，$x_1$、$x_2$、...、$x_n$ 是自变量。

3. 支持向量机模型：支持向量机模型可以用于分类和回归问题，其公式为：

$$
y = \text{sgn}(\beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon)
$$

其中，$y$ 是预测值，$\beta_0$ 是截距，$\beta_1$、$\beta_2$、...、$\beta_n$ 是系数，$x_1$、$x_2$、...、$x_n$ 是自变量，$\epsilon$ 是误差。

## 4.1 线性回归模型代码实例

```python
import numpy as np

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 3 * x + 2 + np.random.randn(100, 1)

# 训练线性回归模型
from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(x, y)

# 预测
x_test = np.array([[0.5]])
y_pred = model.predict(x_test)
print(y_pred)
```

## 4.2 逻辑回归模型代码实例

```python

# 生成数据

y = np.where(y > 0.5, 1, 0)

# 训练逻辑回归模型
from sklearn.linear_model import LogisticRegression

model.fit(x, y)

# 预测

print(y_pred)
```
