# 2.核心概念与联系

## 2.1 径向基核函数

径向基核函数（Radial Basis Function，RBF）是一类定义在欧几里得空间上的函数，它们的特点是以其中一个点为参考点，以该点为圆心的球面为支持域，在支持域内具有正值，在支持域外具有负值或零值。常见的 RBF 函数有多项式基核函数、高斯基核函数、万有引力基核函数等。

### 2.1.1 多项式基核函数

多项式基核函数（Polynomial Kernel Function）可以表示为：

$$
K(x, y) = (x^T y + 1)^d
$$

其中，$x, y \in \mathbb{R}^n$ 是输入向量，$d$ 是多项式度。

### 2.1.2 高斯基核函数

高斯基核函数（Gaussian Kernel Function）可以表示为：

$$
K(x, y) = \exp(-\gamma \|x - y\|^2)
$$

其中，$x, y \in \mathbb{R}^n$ 是输入向量，$\gamma$ 是高斯核参数。

## 2.2 径向基核函数在图像增强中的应用

径向基核函数在图像增强中的应用主要体现在以下几个方面：

1. 图像模糊化：通过将输入图像映射到特征空间，实现图像的模糊化效果，从而减弱图像中的噪声和干扰。
2. 图像锐化：通过在特征空间进行处理，实现图像的锐化效果，从而提高图像的细节和对比度。
3. 图像增强：通过在特征空间中对图像特征进行调整，实现图像的增强效果，从而提高图像的可视化效果和信息提取性能。

```python
import numpy as np
import cv2
import matplotlib.pyplot as plt

# 读取输入图像

# 计算高斯基核函数值
def gaussian_kernel(image, gamma, kernel_size):
    rows, cols = image.shape[:2]
    gaussian_kernel = np.zeros((rows, cols))
    for i in range(rows):
        for j in range(cols):
            sum_kernel = 0
            for x in range(i - kernel_size // 2, i + kernel_size // 2 + 1):
                for y in range(j - kernel_size // 2, j + kernel_size // 2 + 1):
                    if x >= 0 and x < rows and y >= 0 and y < cols:
                        sum_kernel += np.exp(-gamma * ((i - x)**2 + (j - y)**2))
            gaussian_kernel[i, j] = sum_kernel
    return gaussian_kernel

# 根据高斯基核函数值进行增强处理
def image_enhancement(image, gaussian_kernel):
    rows, cols, channels = image.shape
    enhanced_image = np.zeros((rows, cols, channels))
    for i in range(rows):
        for j in range(cols):
            for c in range(channels):
                enhanced_image[i, j, c] = np.sum(image[i - kernel_size // 2:i + kernel_size // 2 + 1,
                                                  j - kernel_size // 2:j + kernel_size // 2 + 1,
                                                  c] * gaussian_kernel[i - kernel_size // 2:i + kernel_size // 2 + 1,
                                                                          j - kernel_size // 2:j + kernel_size // 2 + 1]) / np.sum(gaussian_kernel[i - kernel_size // 2:i + kernel_size // 2 + 1,
                                                                                                                              j - kernel_size // 2:j + kernel_size // 2 + 1])
    return enhanced_image

# 参数设置
gamma = 0.01
kernel_size = 5

# 计算高斯基核函数值
gaussian_kernel = gaussian_kernel(image, gamma, kernel_size)

# 进行图像增强处理
enhanced_image = image_enhancement(image, gaussian_kernel)

# 显示原始图像和增强后的图像
plt.subplot(1, 2, 1), plt.imshow(cv2.cvtColor(image, cv2.COLOR_BGR2RGB))
plt.title('Original Image')
plt.subplot(1, 2, 2), plt.imshow(cv2.cvtColor(enhanced_image, cv2.COLOR_BGR2RGB))
plt.title('Enhanced Image')
plt.show()
```
