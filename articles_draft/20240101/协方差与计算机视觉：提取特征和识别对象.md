## 2.1 协方差的定义与性质

协方差是一种衡量两个随机变量之间线性关系的方法。给定两个随机变量X和Y，其协方差定义为：

$$
\text{Cov}(X,Y) = E[(X - \mu_X)(Y - \mu_Y)]
$$

其中，$\mu_X$和$\mu_Y$分别是X和Y的期望值。从公式中可以看出，协方差是通过计算X和Y的差分，然后再计算它们的平均值来得到的。协方差的正值表示X和Y之间存在正相关关系，负值表示X和Y之间存在负相关关系，而零表示X和Y之间没有相关关系。

协方差的一些性质包括：

1. 如果X和Y之间存在线性关系，那么它们的协方差不等于零。
2. 如果X和Y之间存在正相关关系，那么它们的协方差为正。
3. 如果X和Y之间存在负相关关系，那么它们的协方差为负。
4. 如果X和Y之间没有相关关系，那么它们的协方差为零。

```python
import numpy as np

# 读取图像

# 提取灰度特征

# 提取颜色特征
colors = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)

# 提取边缘特征
edges = cv2.Canny(image, 100, 200)

# 计算特征之间的协方差

cov_gray_colors = np.cov(gray.flatten(), colors.flatten())

print('协方差矩阵：', cov_gray_colors)
```
