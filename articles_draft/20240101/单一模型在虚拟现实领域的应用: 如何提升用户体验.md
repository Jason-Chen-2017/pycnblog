## 4.1 代码实例

```python

def data_processing(x):
    return x * 2

def simulation(x, t):
    return x * np.exp(-t)

def optimization(x):
    return x ** 2

def visualization(x):
    return x + 1

x = np.linspace(0, 10, 100)
y = data_processing(x)
z = simulation(y, x)
w = optimization(z)
v = visualization(w)

import matplotlib.pyplot as plt

plt.plot(x, y, label='Data Processing')
plt.plot(x, z, label='Simulation')
plt.plot(x, w, label='Optimization')
plt.plot(x, v, label='Visualization')
plt.legend()
plt.show()
```
