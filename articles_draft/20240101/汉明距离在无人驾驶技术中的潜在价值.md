2. 初始化一个计数器，设置为0。

4. 当遍历完所有位置后，计数器的值就是汉明距离d(x, y)。

$$

在这里，我们将提供一个简单的Python代码实例，展示如何使用汉明距离计算两个二进制序列之间的距离。

```python
def hamming_distance(x, y):
    n = len(x)
    distance = 0
    for i in range(n):
        if x[i] != y[i]:
            distance += 1
    return distance

x = '1101'
y = '1001'

distance = hamming_distance(x, y)
print('Hamming distance:', distance)
```
