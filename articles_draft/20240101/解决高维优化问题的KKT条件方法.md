# 2. 核心概念与联系

## 2.1 优化问题

优化问题是求解一个函数在满足一定约束条件下的最大值或最小值的问题。在实际应用中，优化问题可以表示为：

$$
\begin{aligned}
\min & \quad f(x) \\
s.t. & \quad g_i(x) \leq 0, \quad i = 1,2,\cdots,m \\
& \quad h_j(x) = 0, \quad j = 1,2,\cdots,l
\end{aligned}
$$

其中，$f(x)$ 是目标函数，$g_i(x)$ 和 $h_j(x)$ 是约束函数，$x$ 是决策变量。

## 2.2 KKT条件

KKT条件是一种用于解决约束优化问题的方法，它的核心思想是将约束条件和目标函数融合在一起，从而使得求解过程更加简洁和高效。KKT条件可以表示为：

$$
\begin{aligned}
& \nabla f(x) + \sum_{i=1}^m \lambda_i \nabla g_i(x) + \sum_{j=1}^l \mu_j \nabla h_j(x) = 0 \\
& \lambda_i \geq 0, \quad i = 1,2,\cdots,m \\
& g_i(x) \leq 0, \quad i = 1,2,\cdots,m \\
& \mu_j h_j(x) = 0, \quad j = 1,2,\cdots,l \\
& h_j(x) = 0, \quad j = 1,2,\cdots,l
\end{aligned}
$$

其中，$\lambda_i$ 和 $\mu_j$ 是拉格朗日乘子，表示约束条件的贡献度。

## 4.1 代码实例

```python
import numpy as np

def f(x):
    return -x[0]**2 - x[1]**2

def g(x):
    return x[0]**2 + x[1]**2 - 1

def h(x):
    return x[0]

def gradient_f(x):
    return np.array([-2*x[0], -2*x[1]])

def gradient_g(x):
    return np.array([2*x[0], 2*x[1]])

def gradient_h(x):
    return np.array([1])

def kkt_method(x0, tol=1e-6, max_iter=1000):
    x = x0
    lambda_ = np.zeros(1)
    mu = np.zeros(1)
    while True:
        grad_f = gradient_f(x)
        grad_g = gradient_g(x)
        grad_h = gradient_h(x)
        grad_sum = grad_f + lambda_ * grad_g + mu * grad_h
        if np.linalg.norm(grad_sum) < tol:
            break
        x_new = x - np.linalg.inv(np.vstack((np.eye(2), grad_g)).T.dot(np.hstack((lambda_, mu)))) * grad_sum
        x = x_new
        if np.linalg.norm(x - x_new) < tol:
            break
    return x

x0 = np.array([0.5, 0.5])
x_optimal = kkt_method(x0)
print("优化解: ", x_optimal)
```
