1. 线性回归模型：线性回归模型是一种常用的数据可视化方法，用于分析两个变量之间的关系。线性回归模型的数学模型公式为：

$$
y = \beta_0 + \beta_1x + \epsilon
$$

其中，$y$ 是因变量，$x$ 是自变量，$\beta_0$ 是截距，$\beta_1$ 是斜率，$\epsilon$ 是误差项。

2. 多元线性回归模型：多元线性回归模型是一种用于分析多个变量之间关系的数据可视化方法。多元线性回归模型的数学模型公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是因变量，$x_1, x_2, \cdots, x_n$ 是自变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。

3. 逻辑回归模型：逻辑回归模型是一种用于分析二分类问题的数据可视化方法。逻辑回归模型的数学模型公式为：

$$
P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x}}
$$

其中，$P(y=1|x)$ 是因变量，$x$ 是自变量，$\beta_0$ 是截距，$\beta_1$ 是斜率。

4. 决策树模型：决策树模型是一种用于分类和回归问题的数据可视化方法。决策树模型的数学模型公式为：

$$
\text{if } x_1 \leq t_1 \text{ then } y = c_1 \\
\text{else if } x_2 \leq t_2 \text{ then } y = c_2 \\
\cdots \\
\text{else } y = c_n
$$

其中，$x_1, x_2, \cdots, x_n$ 是自变量，$t_1, t_2, \cdots, t_n$ 是阈值，$c_1, c_2, \cdots, c_n$ 是类别。

5. 主成分分析（PCA）模型：主成分分析（PCA）模型是一种用于降维和数据可视化的方法。PCA模型的数学模型公式为：

$$
z = W^Tx
$$

其中，$z$ 是新的特征向量，$W$ 是旋转矩阵，$x$ 是原始特征向量。

6. 欧氏距离：欧氏距离是一种用于计算两个点之间距离的数据可视化方法。欧氏距离的数学模型公式为：

$$
d(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2 + \cdots + (x_n - y_n)^2}
$$

其中，$x$ 和 $y$ 是两个点，$x_1, x_2, \cdots, x_n$ 和 $y_1, y_2, \cdots, y_n$ 是点的坐标。

```python
import matplotlib.pyplot as plt

# 生成随机数据
x = np.random.rand(100)
y = 2 * x + 1 + np.random.randn(100)

# 计算线性回归模型参数
beta_1 = np.cov(x, y)[0, 1] / np.cov(x, x)[0, 0]
beta_0 = np.mean(y) - beta_1 * np.mean(x)

# 绘制线性回归图形
plt.scatter(x, y)
plt.plot(x, beta_0 + beta_1 * x)
plt.show()
```
