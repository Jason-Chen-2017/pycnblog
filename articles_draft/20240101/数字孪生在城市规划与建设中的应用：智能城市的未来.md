$$
\frac{\partial \rho}{\partial t} = \nabla \cdot (\rho v) + \sigma
$$

其中，$\rho$ 是交通流量，$t$ 是时间，$v$ 是速度，$\sigma$ 是交通源。我们可以使用 Finite Element Method（FEM）来解决这个微分方程。我们可以使用 Python 来实现 FEM：

```python
import numpy as np
import matplotlib.pyplot as plt

def fem(rho, v, sigma):
    # 创建网格
    x = np.linspace(0, 1, 100)
    y = np.linspace(0, 1, 100)
    X, Y = np.meshgrid(x, y)

    # 计算梯度
    grad_rho = np.gradient(rho, X, Y)

    # 计算流量
    flow = rho * v

    # 计算交通源
    source = sigma

    # 计算梯度分布
    grad_dist = grad_rho + flow + source

    return grad_dist

# 测试数据

# 计算梯度分布
grad_dist = fem(rho, v, sigma)

# 可视化
plt.imshow(grad_dist, cmap='hot', interpolation='nearest')
plt.colorbar()
plt.show()
```
