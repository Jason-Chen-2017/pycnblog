1. 线性回归：线性回归是一种常用的统计学方法，用于预测因变量的值。在社交媒体数据分析中，我们可以使用线性回归来预测粉丝数量、点赞数量、转发数量等的值。线性回归的数学模型公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是因变量，$x_1, x_2, \cdots, x_n$ 是自变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。

2. 逻辑回归：逻辑回归是一种常用的统计学方法，用于预测二值变量的值。在社交媒体数据分析中，我们可以使用逻辑回归来预测用户是否会点赞、转发等。逻辑回归的数学模型公式为：

$$
P(y=1|x_1, x_2, \cdots, x_n) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \cdots - \beta_nx_n}}
$$

其中，$P(y=1|x_1, x_2, \cdots, x_n)$ 是预测概率，$x_1, x_2, \cdots, x_n$ 是自变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

3. 决策树：决策树是一种常用的机器学习方法，用于解决分类和回归问题。在社交媒体数据分析中，我们可以使用决策树来分析用户行为和预测用户行为。决策树的数学模型公式为：

$$
\text{if } x_1 \leq a_1 \text{ then } y = b_1 \\
\text{else if } x_2 \leq a_2 \text{ then } y = b_2 \\
\cdots \\
\text{else } y = b_n
$$

其中，$x_1, x_2, \cdots, x_n$ 是自变量，$a_1, a_2, \cdots, a_n$ 是分割阈值，$b_1, b_2, \cdots, b_n$ 是预测结果。

```python
# 训练模型
model = LogisticRegression()
model.fit(X, y)

# 预测

y_pred = model.predict(X_new)
print(y_pred)
```
