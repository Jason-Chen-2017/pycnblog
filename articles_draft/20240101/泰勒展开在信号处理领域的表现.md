# 2.核心概念与联系
泰勒展开是一种用于逼近函数的数学方法，它通过将函数展开为一系列项的和来表示。泰勒展开的基本公式为：

$$
f(x) \approx \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n
$$

其中，$f(x)$ 是要逼近的函数，$a$ 是逼近的参考点，$f^{(n)}(a)$ 表示第 $n$ 阶导数在参考点 $a$ 的值。

在信号处理领域，泰勒展开主要用于以下几个方面：

1. **信号分析**：通过泰勒展开，我们可以将复杂的信号分解为多个基函数的线性组合，从而方便地进行分析。这在傅里叶分析、波分分析等方面都有应用。

2. **滤波**：泰勒展开可以用于实现各种滤波算法，如高通滤波、低通滤波、锐化滤波等。通过保留不同阶数的项，我们可以实现不同类型的滤波效果。

3. **波形识别**：在波形识别任务中，我们需要将输入信号映射到特定的类别。泰勒展开可以用于将波形表示为多个基函数的线性组合，从而方便地进行特征提取和模式识别。

import numpy as np
import matplotlib.pyplot as plt

# 定义信号
def signal(t):
    return np.sin(2 * np.pi * 5 * t) + np.sin(2 * np.pi * 10 * t)

# 计算泰勒展开系数
def taylor_coefficients(signal, t0, N):
    dt = 1 / (2 * N)
    t = np.linspace(t0 - dt, t0 + dt, 2 * N)
    x = signal(t)
    c = np.zeros(2 * N)
    for n in range(2 * N):
        c[n] = np.trapz(x * np.exp(-2 * np.pi * 1j * n * t), t)
    return c

# 高通滤波
def high_pass_filter(signal, t0, N, cutoff_frequency):
    c = taylor_coefficients(signal, t0, N)
    filtered_signal = np.sum(c[N:] * np.exp(-2 * np.pi * 1j * (np.arange(len(c[N:])) + cutoff_frequency) * t0) * np.exp(-2 * np.pi * 1j * (np.arange(len(c[N:])) + cutoff_frequency) * t))
    return filtered_signal

# 测试
t0 = 0
N = 100
cutoff_frequency = 5
signal = signal(t0)
filtered_signal = high_pass_filter(signal, t0, N, cutoff_frequency)

plt.figure()
plt.plot(signal, label='Original Signal')
plt.plot(filtered_signal, label='Filtered Signal')
plt.legend()
plt.show()
