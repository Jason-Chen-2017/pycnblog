## 4.1 求解线性方程组时利用单调性的代码实例

下面是一个求解线性方程组时利用单调性的代码实例：

```python
import numpy as np

# 定义线性方程组
A = np.array([[1, 1], [1, -1]])
b = np.array([2, 1])

# 求解线性方程组
x = np.linalg.solve(A, b)

# 分析解的单调性
if x[0] <= x[1]:
    print("解是单调递增的")
elif x[0] >= x[1]:
    print("解是单调递减的")
else:
    print("解不是单调的")
```

在这个代码实例中，我们首先定义了一个线性方程组，然后使用 `numpy` 库的 `linalg.solve` 函数来求解这个方程组。最后，我们分析了解的单调性，并输出了结果。
