## 4.1 监控模块的代码实例

以下是一个简单的监控模块的代码实例：

```python
import os
import json

class MonitorModule:
    def __init__(self):
        self.logs = []

    def collect_logs(self):
        log_files = ['/var/log/auth.log', '/var/log/syslog', '/var/log/secure']
        for log_file in log_files:
            with open(log_file, 'r') as f:
                self.logs.extend(f.readlines())

    def process_logs(self):
        processed_logs = []
        for log in self.logs:
            processed_log = self.analyze_log(log)
            if processed_log:
                processed_logs.append(processed_log)
        return processed_logs

    def analyze_log(self, log):
        # 对于这个简单的示例，我们只关注登录失败事件
        if 'failed' in log:
            return {'event_type': 'login_failure', 'timestamp': log.split(' ')[0], 'username': log.split(' ')[2]}
        return None
```

在上述代码中，我们首先定义了一个 `MonitorModule` 类，该类包含了三个方法：`collect_logs`、`process_logs` 和 `analyze_log`。`collect_logs` 方法用于收集登录日志；`process_logs` 方法用于对收集到的登录日志进行处理；`analyze_log` 方法用于分析登录日志，以检测登录失败事件。

## 4.2 报警模块的代码实例

```python
class AlarmModule:
    def __init__(self):
        self.rules = []

    def add_rule(self, rule):
        self.rules.append(rule)

    def trigger_alarm(self, event):
        for rule in self.rules:
            if rule.matches(event):
                return rule.generate_alarm(event)
        return None

class AlarmRule:
    def __init__(self, event_type, threshold):
        self.event_type = event_type
        self.threshold = threshold

    def matches(self, event):
        return event['event_type'] == self.event_type

    def generate_alarm(self, event):
        return {'alarm_type': 'login_failure', 'event': event}
```

在上述代码中，我们首先定义了一个 `AlarmModule` 类，该类包含了两个方法：`add_rule` 和 `trigger_alarm`。`add_rule` 方法用于添加报警规则；`trigger_alarm` 方法用于根据报警规则触发报警。

我们还定义了一个 `AlarmRule` 类，该类包含了两个方法：`matches` 和 `generate_alarm`。`matches` 方法用于判断给定的事件是否满足报警规则；`generate_alarm` 方法用于根据报警规则生成报警信息。
