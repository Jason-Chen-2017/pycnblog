# 2.核心概念与联系

## 2.1 熵权法基本概念

熵权法是一种基于信息论的价格调控方法，其核心概念是熵权（Entropy-Based Pricing）。熵权法的基本思想是通过对市场参与者的行为信息进行量化和分析，从而得出一个公平、合理的价格。

熵权法的核心概念包括：

- 信息熵：信息熵是信息论中的一个概念，用于衡量信息的不确定性。信息熵越高，信息的不确定性越大，反之越小。
- 价格熵：价格熵是熵权法中的一个核心概念，它用于衡量市场参与者在设定价格时的不确定性。价格熵越高，市场参与者在设定价格时的不确定性越大，反之越小。
- 熵权价格：熵权价格是熵权法中的一个核心概念，它是通过对市场参与者的行为信息进行量化和分析得出的一个公平、合理的价格。

## 2.2 熵权法与其他价格调控方法的联系

熵权法与其他价格调控方法有以下联系：

- 与代价调控（Cost-Based Pricing）：代价调控是一种基于成本的价格调控方法，它认为市场价格应该等于生产成本加上一定的利润。熵权法与代价调控的不同在于，熵权法关注市场参与者的行为信息，而代价调控关注生产成本。
- 与市场调控（Market-Based Pricing）：市场调控是一种基于市场供需关系的价格调控方法，它认为市场价格应该根据供需关系进行调整。熵权法与市场调控的不同在于，熵权法关注市场参与者的行为信息，而市场调控关注供需关系。
- 与规定调控（Regulatory Pricing）：规定调控是一种基于政府或者监管机构设定的价格标准的价格调控方法，它认为市场价格应该根据政府或者监管机构设定的标准进行调整。熵权法与规定调控的不同在于，熵权法关注市场参与者的行为信息，而规定调控关注政府或者监管机构设定的标准。

# 市场参与者的行为信息矩阵

              [0.2, 0.3, 0.2, 0.3],
              [0.1, 0.1, 0.4, 0.4]])

# 计算价格熵
def entropy(P):
    return -np.sum(P * np.log2(P))

# 得出熵权价格
def entropy_based_pricing(P):
    H = entropy(P)
    P_star = np.sum(P * np.log2(P)) / H
    return P_star

# 计算并输出熵权价格
P_star = entropy_based_pricing(P)
print("熵权价格:", P_star)
