$$

其中，$y$ 是预测值，$\beta_0$ 是截距，$\beta_1,\beta_2,\cdots,\beta_n$ 是系数，$x_1,x_2,\cdots,x_n$ 是特征变量，$\epsilon$ 是误差项。

$$
\min_{\mathbf{w},b} \frac{1}{2}\mathbf{w}^T\mathbf{w} + C\sum_{i=1}^n\xi_i
$$

其中，$\mathbf{w}$ 是权重向量，$b$ 是偏置项，$\xi_i$ 是松弛变量，$C$ 是正则化参数。

3.决策树模型：决策树模型是一种常用的分类和回归模型，它可以用于预测物流运输的成本、时间、距离等。决策树模型的数学模型公式为：

$$

\text{else if } x_2 \text{ is } a_2 \text{ then } y = b_2 \\
\cdots \\
\text{else if } x_n \text{ is } a_n \text{ then } y = b_n
$$

其中，$x_1,x_2,\cdots,x_n$ 是特征变量，$a_1,a_2,\cdots,a_n$ 是特征值，$b_1,b_2,\cdots,b_n$ 是预测值。
