# 加载气候数据

在这个代码实例中，我们首先使用 Pandas 库加载气候数据，然后使用 dropna() 函数对数据进行清洗，最后使用 corr() 函数计算相关性，以挖掘气候变化问题的规律。

## 4.2 人工智能与机器学习代码实例

我们以生态保护问题为例，使用 Python 的 Scikit-learn 库来进行人工智能与机器学习。

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# 加载生态数据
data = load_iris()

# 数据清洗
X_train, X_test, y_train, y_test = train_test_split(data.data, data.target, test_size=0.2, random_state=42)

# 模型训练
model = SVC(kernel='linear')
model.fit(X_train, y_train)

# 结果应用
y_pred = model.predict(X_test)
print(accuracy_score(y_test, y_pred))
```

在这个代码实例中，我们首先使用 Scikit-learn 库加载生态数据，然后使用 train_test_split() 函数对数据进行清洗和分割，接着使用 SVC() 函数训练支持向量机模型，最后使用 predict() 函数进行预测，并使用 accuracy_score() 函数计算预测的准确率。
