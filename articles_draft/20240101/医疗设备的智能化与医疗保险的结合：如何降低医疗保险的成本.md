## 4.1 数据收集

我们假设我们有一个智能血压计设备，可以通过蓝牙连接与智能手机进行数据传输。我们可以使用以下代码来收集血压值：

```python
import bluetooth

def discover_devices():
    nearby_devices = bluetooth.discover_devices(lookup_names=False)
    return nearby_devices

def connect_device(address):
    server_sock = bluetooth.BluetoothSocket(bluetooth.RFCOMM)
    server_sock.connect((address, 1))
    return server_sock

def read_data(sock):
    data = sock.recv(1024)
    return data.decode('utf-8')

devices = discover_devices()
address = devices[0]
sock = connect_device(address)
data = read_data(sock)
print(data)
```
