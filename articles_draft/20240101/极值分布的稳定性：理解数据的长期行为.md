### 4.1 正态分布

我们可以使用Python的numpy库来进行正态分布的计算。

```python
import numpy as np

# 设置均值和标准差
mu = 0
sigma = 1

# 生成正态分布的随机数
x = np.random.normal(mu, sigma, 1000)

# 计算正态分布的概率密度函数
pdf = np.exp(-(x - mu)**2 / (2 * sigma**2)) / np.sqrt(2 * np.pi * sigma**2)

# 计算正态分布的累积分布函数
cdf = np.cumsum(pdf) / np.sum(pdf)
```

### 4.2 泊松分布

```python
import scipy.stats as stats

# 生成泊松分布的随机数
x = stats.poisson.rvs(lambda_, size=1000)

# 计算泊松分布的概率密度函数
pdf = stats.poisson.pmf(x, lambda_)

# 计算泊松分布的累积分布函数

```
