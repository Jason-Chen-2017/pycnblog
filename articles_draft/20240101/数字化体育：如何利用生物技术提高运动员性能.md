## 4.1心率监测代码实例
以下是一个心率监测代码实例：
```python
import time

class HeartRateSensor:
    def __init__(self):
        self.heart_rate = 0

    def measure_heart_rate(self):
        self.heart_rate = randint(60, 100)
        print("心率：", self.heart_rate, "次/分")

sensor = HeartRateSensor()
for _ in range(5):
    sensor.measure_heart_rate()
    time.sleep(1)
```
这个代码实例中，我们定义了一个`HeartRateSensor`类，该类包含一个`measure_heart_rate`方法，用于监测心率。在主程序中，我们创建了一个`HeartRateSensor`对象，并通过调用`measure_heart_rate`方法，每秒监测一次心率。
