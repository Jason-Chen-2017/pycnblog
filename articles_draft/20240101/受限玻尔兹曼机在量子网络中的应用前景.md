1. 单位性量子门（Identity gate）：不改变量子位的状态。
2. 阶乘量子门（Hadamard gate）：将量子位从 $| 0 \rangle$ 状态转移到 $| 1 \rangle$ 状态，反之亦然。
3. 控制量子门（Controlled gate）：仅在控制量子位为 $| 1 \rangle$ 状态时才对目标量子位进行操作。

    return np.array([[1, 0], [0, 1]]) @ state

        return target
    else:
        return np.array([[0, 1], [1, 0]]) @ target

def shortest_path(graph, start, end):
    n = len(graph)
    states = [np.array([1, 0])]
    energies = [0]

    for _ in range(n * (n - 1)):
        new_states = []
        new_energies = []
