# 2.核心概念与联系

## 2.1夹角余弦

夹角余弦（angle cosine）是一种几何概念，用于描述两个向量之间的关系。它定义为：

$$
\cos(\theta) = \frac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\| \|\mathbf{b}\|}
$$

其中，$\mathbf{a}$ 和 $\mathbf{b}$ 是两个向量，$\theta$ 是它们之间的夹角，$\cdot$ 表示向量内积，$\|\mathbf{a}\|$ 和 $\|\mathbf{b}\|$ 分别表示向量 $\mathbf{a}$ 和 $\mathbf{b}$ 的长度。

## 2.2信道估计

信道估计是一种在通信系统中用于估计通信信道参数的技术。信道参数包括信道功率、信道延迟、信道噪声等。信道估计可以帮助我们更好地进行信号传输，提高通信系统的性能。

## 2.3调制解调

调制解调是一种在通信系统中用于实现信号的调制和解调的技术。调制是指将信息符号转换为信号电平，以便在通信信道上传输。解调是指在通信信道上接收信号电平，将其转换回信息符号。调制解调技术可以实现数据的传输和接收，提高通信系统的效率和可靠性。

3. 使用夹角余弦公式计算信道延迟。

数学模型公式为：

$$
\tau = \frac{\|\mathbf{r} - \mathbf{a}\|}{\|\mathbf{v}\|}

其中，$\tau$ 是信道延迟，$\mathbf{r}$ 是接收到的信号向量，$\mathbf{a}$ 是发送信号向量，$\mathbf{v}$ 是信道传输速度。

其中，$\mathbf{s}$ 是调制后的信号向量，$\mathbf{a}$ 和 $\mathbf{b}$ 是信息符号对应的方向向量，$\theta$ 是调制角度。

### 3.2.2解调

$$

其中，$\mathbf{a}$ 是解调后的信息符号对应的方向向量，$\mathbf{s}$ 是接收到的信号向量，$\theta$ 是解调角度。

# 信道功率估计
def channel_power_estimation(r, a):
    return np.linalg.norm(r)**2 / np.linalg.norm(a)**2

# 信道延迟估计
def channel_delay_estimation(r, a, v):
    return np.linalg.norm(r - a) / np.linalg.norm(v)

# 调制
def modulation(a, b, theta):
    return a * np.cos(theta) + b * np.sin(theta)

# 解调
def demodulation(s, b, theta):
    return s * np.cos(theta) - b * np.sin(theta)

# 测试数据
a = np.array([1, 0])
b = np.array([0, 1])
v = np.array([1, 0])
r = np.array([0.8, 0.6])
theta = np.arctan2(b[1], b[0])

# 信道功率估计
P = channel_power_estimation(r, a)
print("信道功率估计结果：", P)

# 信道延迟估计
tau = channel_delay_estimation(r, a, v)
print("信道延迟估计结果：", tau)

# 调制
s = modulation(a, b, theta)
print("调制后的信号向量：", s)

# 解调
a_demod = demodulation(s, b, theta)
print("解调后的信息符号对应的方向向量：", a_demod)
