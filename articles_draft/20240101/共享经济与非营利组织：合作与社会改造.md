## 1.1 共享经济的发展

共享经济的发展可以追溯到20世纪90年代初的租赁经济（sharing economy）概念。在2008年，Airbnb成立，它是一家提供房屋租赁服务的互联网平台。随后，Uber、Lyft等出租车服务平台也开始兴起，共享经济成为一种新型的经济模式。

## 1.2 非营利组织的发展

非营利组织（NGO）是一种独立的社会组织，它们通常为解决社会问题而努力，如环保、公益事业、人权、发展帮助等。非营利组织的发展可以追溯到19世纪初的慈善组织，如红十字会。随着社会问题的增多和全球化的进程，非营利组织的数量和影响力也在不断增长。

## 4.1 资源共享代码实例

假设我们有一个房屋共享平台，它可以为非营利组织提供房屋资源。以下是一个简单的Python代码实例：

```python
class Nonprofit:
    def __init__(self, name):
        self.name = name
        self.resources = []

    def add_resource(self, resource):
        self.resources.append(resource)

class Resource:
    def __init__(self, name, location, price):
        self.name = name
        self.location = location
        self.price = price

def match_resource_need(nonprofit, resource):
    for need in nonprofit.resources:
        if need.location == resource.location and need.price <= resource.price:
            return True
    return False

nonprofit = Nonprofit("Red Cross")
resource = Resource("House", "New York", 100)
nonprofit.add_resource(resource)
```

在这个代码实例中，我们首先定义了一个`Nonprofit`类和一个`Resource`类。然后我们创建了一个`Red Cross`非营利组织，并为它添加了一个房屋资源。最后，我们定义了一个`match_resource_need`函数，它用于判断资源与需求是否匹配。
