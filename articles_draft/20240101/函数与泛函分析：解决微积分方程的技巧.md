### 4.1 代码实例1：求解微积分方程的初值问题

假设我们有一个微积分方程：

$$
y'(x) = y(x) + x^2
$$

我们想要求解这个方程的初值问题，即在x=0时，y(0) = 1。我们可以使用以下代码来解决这个问题：

```python
import numpy as np
from scipy.integrate import odeint

def f(t, y):
    return y + t**2

t = np.linspace(0, 1, 100)
y0 = 1
y = odeint(f, y0, t)

print(y[-1, 0])
```
