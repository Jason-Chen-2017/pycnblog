## 3.5 数学模型公式详细讲解
在进行Python数据分析时，我们需要了解一些数学模型公式，以便更好地理解和应用算法原理。例如：

- 平均值：Average = Sum(x) / N，其中x是数据集中的每个值，N是数据集的大小。
- 标准差：Standard Deviation = sqrt(Sum((x - Average)^2) / N)，其中x是数据集中的每个值，Average是数据集的平均值，N是数据集的大小。
- 相关性：Pearson相关性 = Sum((x - Average_x)(y - Average_y)) / sqrt(Sum((x - Average_x)^2) * Sum((y - Average_y)^2))，其中x和y是数据集中的两个变量，Average_x和Average_y是两个变量的平均值。
