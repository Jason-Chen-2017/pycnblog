# 2.核心概念与联系

## 2.1异常处理

异常处理是指在程序运行过程中，当发生错误时，采取相应的措施进行处理。Python中的异常处理主要包括try、except、finally等关键字。

- try：用于尝试执行某段代码，如果发生异常，则执行except子句。
- except：用于捕获并处理异常。
- finally：无论是否发生异常，都会执行的代码块。

例如：
```python
try:
    # 尝试执行的代码
    x = 1 / 0
except ZeroDivisionError as e:
    # 处理异常
    print("发生了除零错误:", e)
finally:
    # 无论是否发生异常，都会执行的代码
    print("这里是finally块")
```

## 2.2调试

调试是指在程序出现问题时，通过查找和修复问题的过程来确保程序正常运行。Python中的调试主要包括设置断点、查看变量值、步进执行代码等功能。

- 设置断点：可以通过设置断点来暂停程序的执行，从而查看程序在某一行代码处的状态。
- 查看变量值：可以通过查看变量值来了解程序在某一行代码处的状态。
- 步进执行代码：可以通过步进执行代码来逐行执行程序，从而找到程序出现问题的原因。

例如：
```python
import pdb

# 设置断点
pdb.set_trace()

# 程序代码
x = 1
y = 0
z = x / y
print(z)
```
当程序执行到设置的断点时，会暂停执行，并进入交互模式。我们可以通过输入相关命令来查看变量值、步进执行代码等。
