## 4.1 组件组织实例

### 4.1.1 组件拆分实例

在这个实例中，我们将一个简单的文件系统拆分为多个组件，包括文件读取组件、文件写入组件、文件删除组件等。

```python
class FileReader:
    def read(self, file_path):
        # 文件读取逻辑
        pass

class FileWriter:
    def write(self, file_path, content):
        # 文件写入逻辑
        pass

class FileDeleter:
    def delete(self, file_path):
        # 文件删除逻辑
        pass
```

### 4.1.2 组件组合实例

在这个实例中，我们将上述的文件系统组件组合成一个完整的文件系统。

```python
class FileSystem:
    def __init__(self):
        self.reader = FileReader()
        self.writer = FileWriter()
        self.deleter = FileDeleter()

    def read(self, file_path):
        return self.reader.read(file_path)

    def write(self, file_path, content):
        return self.writer.write(file_path, content)

    def delete(self, file_path):
        return self.deleter.delete(file_path)
```

### 4.1.3 组件交互实例

在这个实例中，我们将上述的文件系统组件进行交互。

```python
file_system = FileSystem()
content = "Hello, World!"
file_path = "test.txt"

file_system.write(file_path, content)
content = file_system.read(file_path)
file_system.delete(file_path)
```
