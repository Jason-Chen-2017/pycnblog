```c
#include <stdio.h>
#include <avr/io.h>
#include <util/delay.h>

int main(void) {
    DDRB |= (1 << PB0); // 设置PORTB的第0位为输出
    while (1) {
        PORTB ^= (1 << PB0); // 使LED灯闪烁
        _delay_ms(500); // 延时500毫秒
    }
    return 0;
}
```

这个代码实例的解释说明如下：

1. 首先，我们包含了必要的头文件，包括标准输入输出头文件、AVR IO头文件和延时头文件。
2. 然后，我们在主函数中设置PORTB的第0位为输出，这样我们就可以控制LED灯的闪烁。
3. 接下来，我们进入一个无限循环，每次循环中我们使LED灯闪烁，然后延时500毫秒。
4. 最后，我们返回0，表示程序结束。

```c
}
```

4. 最后，我们延时1秒，然后重复上述操作。

4. 同样，我们使用printf函数提示用户输入第二个数字，然后使用scanf函数读取用户输入的数字。
5. 然后，我们计算a和b的和，并将结果存储在变量sum中。
6. 最后，我们使用printf函数打印出a和b的和。
