假设我们有一个简单的计算器程序，如下所示：

```python
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):

```

我们可以将这个程序进行代码重构，如下所示：

```python

    return reduce(op, [a, b])

def add(a, b):
    return operation(a, b, lambda x, y: x + y)

def subtract(a, b):
    return operation(a, b, lambda x, y: x - y)

def multiply(a, b):
    return operation(a, b, lambda x, y: x * y)
```

在这个例子中，我们将原始的四个函数简化为一个通用的函数，从而提高了代码的可读性、可维护性和可扩展性。

## 4.2 性能优化

假设我们有一个简单的排序程序，如下所示：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

我们可以将这个程序进行性能优化，如下所示：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr
```

在这个例子中，我们将原始的排序算法改为一个更高效的算法，从而提高了程序的响应速度。
