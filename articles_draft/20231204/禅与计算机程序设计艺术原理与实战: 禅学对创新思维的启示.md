## 4.1 递归实例

递归是一种解决问题的方法，它可以将问题分解为多个相同或相似的子问题，然后递归地解决这些子问题。递归是一种非常重要的算法原理，它可以帮助我们更好地解决问题。

以下是一个递归的代码实例：

```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)
```

在这个代码实例中，我们定义了一个名为`factorial`的函数，它用来计算一个数的阶乘。这个函数使用递归的方法来解决问题，它将问题分解为多个相同的子问题，然后递归地解决这些子问题。

## 4.2 动态规划实例

动态规划是一种解决问题的方法，它可以将问题分解为多个相互依赖的子问题，然后递归地解决这些子问题。动态规划是一种非常重要的算法原理，它可以帮助我们更好地解决问题。

以下是一个动态规划的代码实例：

```python
def longest_common_substring(s1, s2):
    m = len(s1)
    n = len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    longest_length = 0
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                longest_length = max(longest_length, dp[i][j])
    return longest_length
```

在这个代码实例中，我们定义了一个名为`longest_common_substring`的函数，它用来计算两个字符串的最长公共子串。这个函数使用动态规划的方法来解决问题，它将问题分解为多个相互依赖的子问题，然后递归地解决这些子问题。

以下是一个贪心算法的代码实例：

```python
def knapsack(items, capacity):
    n = len(items)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if items[i - 1][1] <= w:
                dp[i][w] = max(items[i - 1][0] + dp[i - 1][w - items[i - 1][1]], dp[i - 1][w])
            else:
                dp[i][w] = dp[i - 1][w]
    return dp[n][capacity]
```

在这个代码实例中，我们定义了一个名为`knapsack`的函数，它用来解决一个背包问题。这个问题是一个经典的贪心算法问题，它可以将问题分解为多个相互独立的子问题，然后选择最优的子问题来解决问题。
