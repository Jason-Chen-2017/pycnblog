### 4.2 数据分析代码实例

以下是一个简单的数据分析代码实例：

```python
import pandas as pd
import matplotlib.pyplot as plt

# 数据清洗
data = pd.read_csv('sales_data.csv')
data = data.dropna()

# 数据分析
sales = data['sales'].sum()
average_sales = data['sales'].mean()

# 数据可视化

plt.xlabel('类别')
```
