## 3.1概率论
### 3.1.1概率的基本概念
概率是一种度量不确定性的方法，用于描述事件发生的可能性。概率是一个数值，范围在0到1之间。

### 3.1.2概率的计算方法
概率的计算方法有多种，包括：

1.直接计算法：直接计算法是通过计算事件发生的可能性来计算概率的方法。例如，如果有10个球，其中5个是红色的，那么红色球的概率是5/10=1/2。

2.定义法：定义法是通过定义一个事件的概率来计算概率的方法。例如，如果定义一个事件的概率是0.5，那么这个事件的概率就是0.5。

3.贝叶斯定理：贝叶斯定理是一种通过条件概率来计算概率的方法。例如，如果事件A和事件B之间存在条件概率关系，那么可以通过贝叶斯定理来计算事件A和事件B的概率。

### 3.1.3概率的应用

## 4.1概率论
### 4.1.1直接计算法
```python
import random

# 生成10个球，其中5个是红色的
balls = [random.choice(['red', 'blue']) for _ in range(10)]

# 计算红色球的概率
red_probability = sum(ball.startswith('red') for ball in balls) / len(balls)

print(red_probability)
```

### 4.1.2定义法
```python
# 定义一个事件的概率是0.5
event_probability = 0.5

# 计算事件发生的概率
event_occurrence_probability = event_probability

print(event_occurrence_probability)
```

### 4.1.3贝叶斯定理
```python
# 定义一个事件的概率是0.5
event_probability = 0.5

# 定义一个条件概率是0.5
condition_probability = 0.5

# 计算贝叶斯定理
bayes_theorem = event_probability * condition_probability / (event_probability * condition_probability + (1 - event_probability) * (1 - condition_probability))

print(bayes_theorem)
```
