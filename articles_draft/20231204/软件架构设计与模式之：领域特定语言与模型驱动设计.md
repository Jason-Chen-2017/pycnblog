### 4.1.4实现

我们可以根据计算器的语法和语义，实现计算器的解释器或编译器。例如，我们可以使用Python语言来实现计算器的解释器，如下：

```python
def expression(tokens):
    stack = []
    for token in tokens:
        if token in ['+', '-', '*', '/']:
            op2 = stack.pop()
            op1 = stack.pop()
            result = operate(op1, op2, token)
            stack.append(result)
        else:
            stack.append(token)
    return stack.pop()

def operate(op1, op2, op):
    if op == '+':
        return op1 + op2
    elif op == '-':
        return op1 - op2
    elif op == '*':
        return op1 * op2
    elif op == '/':
        return op1 / op2

def parse(expression_str):
    tokens = expression_str.split()
    return expression(tokens)

if __name__ == '__main__':
    expression_str = '2 + 3 * 4'
    result = parse(expression_str)
    print(result)
```
