### 4.1.1 单例模式

单例模式是一种设计模式，它确保一个类只有一个实例，并提供全局访问点。以下是单例模式的具体代码实例：

```python
class Singleton:
    _instance = None

    @classmethod
    def get_instance(cls):
        if cls._instance is None:
            cls._instance = cls()
        return cls._instance
```

### 4.1.2 工厂模式

工厂模式是一种设计模式，它定义了一个创建对象的接口，但不定义该对象的具体类。以下是工厂模式的具体代码实例：

```python
class Factory:
    def create(self, type):
        if type == 'A':
            return A()
        elif type == 'B':
            return B()
        else:
            return None
```

### 4.3.1 排序算法

排序算法是一种常用的算法，它可以对数据进行排序。以下是排序算法的具体代码实例：

```python
def sort(arr):
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] > arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
result = sort(arr)
print(result)
```
