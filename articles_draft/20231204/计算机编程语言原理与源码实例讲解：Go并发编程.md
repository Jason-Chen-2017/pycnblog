## 4.具体代码实例和详细解释说明

### 4.1 Goroutine的使用示例

```go
package main

import "fmt"

func main() {
    // 创建一个Goroutine
    go func() {
        fmt.Println("Hello, World!")
    }()

    // 主线程等待Goroutine完成
    fmt.Scanln()
}
```

在上面的代码中，我们创建了一个Goroutine，它会打印出“Hello, World!”。然后，主线程会等待Goroutine完成后再继续执行。

### 4.2 Channel的使用示例

```go
package main

import "fmt"

func main() {
    // 创建一个Channel
    ch := make(chan int)

    // 创建一个Goroutine，用于向Channel发送数据
    go func() {
        ch <- 1
    }()

    // 主线程等待Goroutine发送数据
    fmt.Scanln()

    // 主线程接收Channel中的数据
    fmt.Println(<-ch)
}
```

在上面的代码中，我们创建了一个Channel，然后创建了一个Goroutine，用于向Channel发送数据。主线程会等待Goroutine发送数据后再接收数据。

### 4.3 Sync包的使用示例

```go
package main

import "fmt"
import "sync"

func main() {
    // 创建一个Mutex锁
    var mux sync.Mutex

    // 创建一个Goroutine，用于尝试获取Mutex锁
    go func() {
        mux.Lock()
        fmt.Println("Hello, World!")
        mux.Unlock()
    }()

    // 主线程等待Goroutine完成
    fmt.Scanln()
}
```

在上面的代码中，我们创建了一个Mutex锁，然后创建了一个Goroutine，用于尝试获取Mutex锁。主线程会等待Goroutine完成后再继续执行。
