```python
my_instance = MyClass()
```

## 4.3装饰器的函数、装饰器的参数和装饰器的返回值

```python
def decorator(func):
    def wrapper(*args, **kwargs):
        print("Before calling the function")
        result = func(*args, **kwargs)
        print("After calling the function")
        return result
    return wrapper

@decorator
def my_function():
    print("Inside the function")

my_function()
```

在上述代码中，我们定义了一个装饰器`decorator`，它的参数是要装饰的函数`func`。然后我们用`@decorator`语法将`my_function`装饰了一遍。最后我们调用了`my_function`。
