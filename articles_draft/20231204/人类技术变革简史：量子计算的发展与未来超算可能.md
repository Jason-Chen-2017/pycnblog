### 1.4.1 量子幻想算法实现

```python
from qiskit import QuantumCircuit, Aer, transpile, assemble

# 初始化量子位
qc = QuantumCircuit(2)

# 创建纠缠状态
qc.h(0)
qc.cx(0, 1)

# 执行计算
qc.h(0)
qc.cx(0, 1)
qc.measure([0, 1], [0, 1])

# 读取结果
result = qc.get_counts()
print(result)
```
