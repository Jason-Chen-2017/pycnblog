### 4.1 代码实例
```

    int n;
    scanf("%d", &n);
    if (n % 2 == 0) {
        printf("n是偶数\n");
    } else {
        printf("n是奇数\n");
    }
    return 0;
}
```

```
+----------------+

+----------------+
|                 |
|    输出n是奇数  |
+----------------+
```
从上述流程图可以看出，程序的行为包括：输入n，判断n是否为偶数，输出n是偶数或奇数。
