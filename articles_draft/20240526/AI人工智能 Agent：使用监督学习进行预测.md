## 2. 核心概念与联系

监督学习是一种训练模型的方法，它使用一组已知的输入输出对进行训练，这些输入输出对被称为训练数据集。在监督学习中，训练数据集由两部分组成：特征集和标签集。特征集是输入数据，标签集是预期的输出。

监督学习与其他学习方法的主要区别在于，其他学习方法如无监督学习和强化学习不依赖于已知的输出标签。

## 3. 核心算法原理具体操作步骤

在监督学习中，使用的主要算法有线性回归、逻辑回归、支持向量机、决策树等。我们将讨论其中的两个算法：线性回归和逻辑回归。

### 线性回归

线性回归是一种用于拟合线性模型的方法，它假设输入数据与输出数据之间存在线性关系。线性回归的目标是找到一条最佳线，使得预测值与实际值之间的误差最小。

线性回归的核心公式为：

$$y = mx + b$$

其中，y 是输出值，x 是输入值，m 是斜率，b 是截距。

### 逻辑回归

逻辑回归是一种用于分类问题的方法，它可以将输入数据映射到一个概率分布上。逻辑回归的目标是找到一条最佳线，使得预测概率与实际概率之间的误差最小。

逻辑回归的核心公式为：

$$\log(\frac{p}{1-p}) = mx + b$$

其中，p 是预测概率，x 是输入值，m 是斜率，b 是截距。

其中，$$\hat{y}$$ 是预测值，$$\bar{x}$$ 和 $$\bar{y}$$ 是输入数据集和输出数据集的平均值。

在训练数据集上使用最大似然估计，我们得到以下公式：

$$\log(\frac{p}{1-p}) = mx + b$$

其中，p 是预测概率，x 是输入值，m 是斜率，b 是截距。

Q1：监督学习与无监督学习的区别？

A1：监督学习需要使用标记过的数据进行训练，而无监督学习则不需要标记过的数据。监督学习的目标是预测未知数据，而无监督学习的目标是发现数据间的结构和关系。

Q2：线性回归和逻辑回归有什么区别？

A2：线性回归是一种用于拟合线性模型的方法，而逻辑回归是一种用于分类问题的方法。线性回归的目标是找到一条最佳线，使得预测值与实际值之间的误差最小，而逻辑回归的目标是找到一条最佳线，使得预测概率与实际概率之间的误差最小。
