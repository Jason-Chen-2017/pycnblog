```python
import requests

def select_platform():
    platforms = ['Amazon', 'Alibaba', 'eBay']
    for platform in platforms:
        response = requests.get(f'https://api.example.com/{platform}')
        if response.status_code == 200:
            return platform
    return None
```

这个代码实例使用Python的requests库发送请求到各个电商平台的API，返回第一个响应状态为200的平台名称。
