```python
import datetime

class Order:
    def __init__(self, order_time, product_id, warehouse_id):
        self.order_time = order_time
        self.product_id = product_id
        self.warehouse_id = warehouse_id

    def calculate_delivery_time(self):
        warehouse_time = 2  # 库存时间（小时）
        logistics_time = 1  # 物流时间（小时）

        return self.order_time + datetime.timedelta(hours=warehouse_time + logistics_time)

order = Order(datetime.datetime.now(), "P001", "W001")
print("预计送达时间：", order.calculate_delivery_time())
```

这个代码实例中，我们定义了一个`Order`类，用于表示订单。在这个类中，我们定义了一个`calculate_delivery_time`方法，用于计算订单的预计送达时间。这个方法中，我们使用了一个简单的数学模型来计算预计送达时间。
