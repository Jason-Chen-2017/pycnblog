# 2.核心概念与联系
加密货币交易的核心概念包括：加密货币、区块链、去中心化、数字签名、交易所等。

## 2.1 加密货币
加密货币是一种数字货币，使用加密技术进行交易和存储。比特币是最著名的加密货币，其核心概念是通过加密技术实现货币的安全传输和交易。

## 2.2 区块链
区块链是加密货币交易的基础技术，它是一种去中心化的分布式数据存储和交易系统。区块链由一系列的区块组成，每个区块包含一组交易记录和一个时间戳。每个区块都包含前一个区块的哈希值，这样一来，当一个区块被修改时，后续所有区块的哈希值都会发生变化，从而保证了区块链的不可篡改性。

## 2.3 去中心化
去中心化是加密货币交易的核心特征，它意味着没有任何中心化机构对交易进行控制。这使得加密货币交易更加安全和隐私。

## 2.4 数字签名
数字签名是加密货币交易的一种安全机制，它使用公钥和私钥进行加密和解密。在加密货币交易中，发送方使用私钥对交易信息进行加密，接收方使用发送方的公钥解密交易信息。这样一来，交易信息的安全性得到保障。

## 2.5 交易所
交易所是加密货币交易的平台，它提供了买卖加密货币的服务。交易所通常提供各种加密货币的交易对，如比特币与美元、比特币与其他加密货币等。

```python
import hashlib
import hmac
import base64

def sign_message(message, private_key):
    # 使用SHA-256算法对消息进行哈希
    digest = hashlib.sha256(message.encode()).digest()

    # 使用HMAC算法进行签名
    signature = hmac.new(private_key, digest, hashlib.sha256).digest()

    # 使用Base64编码将签名数据转换为字符串
    encoded_signature = base64.b64encode(signature)

    return encoded_signature

def verify_signature(message, signature, public_key):
    # 使用SHA-256算法对消息进行哈希
    digest = hashlib.sha256(message.encode()).digest()

    # 使用HMAC算法进行验证
    try:
        hmac.compare_digest(signature, hmac.new(public_key, digest, hashlib.sha256).digest())
        return True
    except TypeError:
        return False

# 示例代码
message = "这是一个加密货币交易的消息"
private_key = b"私钥"
public_key = b"公钥"

signature = sign_message(message, private_key)
is_valid = verify_signature(message, signature, public_key)

print(signature)  # 输出：b'签名数据'
print(is_valid)   # 输出：True
```

在这个代码实例中，我们首先导入了`hashlib`、`hmac`和`base64`模块。然后定义了两个函数：`sign_message`和`verify_signature`。

`sign_message`函数首先使用SHA-256算法对消息进行哈希，然后使用HMAC算法进行签名，最后使用Base64编码将签名数据转换为字符串。

`verify_signature`函数首先使用SHA-256算法对消息进行哈希，然后使用HMAC算法进行验证，如果验证成功，则返回True，否则返回False。

在示例代码中，我们首先定义了一个消息、私钥和公钥。然后使用`sign_message`函数对消息进行签名，并使用`verify_signature`函数进行验证。最后，我们输出了签名数据和验证结果。
