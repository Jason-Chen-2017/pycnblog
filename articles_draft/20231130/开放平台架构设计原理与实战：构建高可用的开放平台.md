## 4.2 排序算法实例
以下是一个使用Python实现的快速排序算法实例：

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [3, 5, 2, 1, 4]
sorted_arr = quick_sort(arr)
print(sorted_arr)
```

在这个实例中，我们使用Python的列表推导式来实现快速排序算法。我们首先定义了一个quick_sort函数，该函数接受一个数组参数并返回该数组的排序结果。然后，我们使用这个函数来排序一个数组。

## 4.3 搜索算法实例
以下是一个使用Python实现的二分搜索算法实例：

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

arr = [1, 2, 3, 4, 5]
target = 3
index = binary_search(arr, target)
print(index)
```

在这个实例中，我们使用Python的while循环来实现二分搜索算法。我们首先定义了一个binary_search函数，该函数接受一个数组和一个目标值参数并返回目标值在数组中的索引。然后，我们使用这个函数来查找一个数组中的一个目标值。
