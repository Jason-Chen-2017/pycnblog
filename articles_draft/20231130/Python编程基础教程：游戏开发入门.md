## 2.1 游戏循环
游戏循环是游戏的核心，它负责处理游戏的所有逻辑和更新。游戏循环通常包括以下几个步骤：

1. 获取用户输入
2. 更新游戏对象的状态
3. 绘制游戏界面
4. 检查游戏是否结束

这些步骤会不断重复，直到游戏结束。
