# 1.背景介绍

在1980年代，Pascal语言已经成为一种广泛使用的编程语言，其应用范围涵盖了各种领域，如操作系统、数据库、软件开发等。在这一阶段，Pascal语言的标准化工作也得到了进一步完善，使其更加稳定和可靠。

# 2.核心概念与联系

在Pascal语言中，过程和函数是两种非常重要的程序结构，它们用于组织代码，使其更加易于理解和维护。在这一节中，我们将详细介绍过程和函数的核心概念、联系以及区别。

## 2.1 过程

过程（procedure）是一种用于实现某个功能的程序结构，它可以接受输入参数、执行一系列操作，但不能返回任何值。过程的主要特点是：

1. 过程是一种无返回值的程序结构。
2. 过程可以接受输入参数，但不能返回任何值。
3. 过程可以修改局部变量和全局变量的值。
4. 过程可以调用其他过程或函数。

## 2.2 函数

函数（function）是一种用于实现某个功能的程序结构，它可以接受输入参数、执行一系列操作，并返回一个值。函数的主要特点是：

1. 函数是一种有返回值的程序结构。
2. 函数可以接受输入参数，并返回一个值。
3. 函数可以修改局部变量的值，但不能修改全局变量的值。
4. 函数可以调用其他过程或函数。

## 2.3 过程与函数的区别

过程和函数的主要区别在于返回值的处理方式。过程不返回任何值，而函数则返回一个值。此外，函数可以修改局部变量的值，但不能修改全局变量的值，而过程可以修改局部变量和全局变量的值。

## 4.1 过程的使用方法

以下是一个简单的过程示例：

```pascal
program Example;
var
  x, y: integer;
begin
  x := 10;
  y := 20;
  writeln('x = ', x);
  writeln('y = ', y);
  procedure PrintSum;
  begin
    writeln('x + y = ', x + y);
  end;
  PrintSum;
end.
```

在上述代码中，我们定义了一个名为`PrintSum`的过程，它用于计算`x`和`y`的和并输出结果。过程的定义和调用都是在程序的主体部分中进行的。

## 4.2 函数的使用方法

以下是一个简单的函数示例：

```pascal
program Example;

  x, y: integer;
```
