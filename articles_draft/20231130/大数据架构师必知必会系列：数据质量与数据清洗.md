# 2.核心概念与联系

## 2.1 数据质量

数据质量是指数据的准确性、完整性、一致性、时效性和可靠性等方面的度量。数据质量问题主要包括数据错误、数据缺失、数据冗余、数据不一致等。

## 2.2 数据清洗

数据清洗是指对数据进行预处理、校验、修正、补全等操作，以提高数据质量的过程。数据清洗的目的是为了消除数据中的错误、缺失、冗余等问题，从而提高数据的可靠性和可用性。

## 2.3 数据质量与数据清洗的联系

数据质量与数据清洗是密切相关的。数据清洗是提高数据质量的重要手段，而数据质量是数据清洗的目标。数据质量问题的发现和解决是数据清洗的重要内容。
