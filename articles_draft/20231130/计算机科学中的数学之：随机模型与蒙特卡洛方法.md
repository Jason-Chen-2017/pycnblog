# 2.核心概念与联系

## 2.1 随机变量

随机变量是一种可以取不同值的变量，其取值的概率可以用一个概率分布来描述。随机变量可以是离散的（如掷骰子的点数）或连续的（如气温）。随机变量的期望是它的所有可能取值的乘以它们的概率和，方差是期望值与实际取值之间的差异的平方和。

## 2.2 概率

概率是一种度量事件发生的可能性的数学概念。概率通常用0到1之间的一个数来表示，表示事件发生的可能性。概率可以通过频率、几何、朗贝尔定理等方法来计算。

期望是随机变量的一个数学期望，表示随机变量的平均值。期望可以用随机变量的概率分布来计算。期望是随机变量的一个重要性质，可以用来描述随机变量的行为。

## 2.4 方差

方差是随机变量的一个数学量，表示随机变量的离散程度。方差可以用随机变量的概率分布来计算。方差是随机变量的一个重要性质，可以用来描述随机变量的行为。

## 2.5 随机过程

随机过程是一种随时间变化的随机变量序列，每个时刻的随机变量都可以取不同的值。随机过程可以是离散的（如掷骰子的点数序列）或连续的（如气温序列）。随机过程的期望是它们的所有可能取值的乘以它们的概率和，方差是期望值与实际取值之间的差异的平方和。
