## 4.1 求导

```python
import numpy as np

def derivative(f, x):
    return (f(x + h) - f(x)) / h

def f(x):
    return x**2

h = 0.001

result = derivative(f, x)
print(result)
```

在上述代码中，我们首先导入了NumPy库，然后定义了一个求导的函数`derivative`。接着，我们定义了一个函数`f`，该函数的输入是一个变量`x`，输出是`x`的平方。然后，我们设置了变量`x`的值为2，以及求导的步长`h`为0.001。最后，我们调用`derivative`函数，并将结果打印出来。

## 4.2 积分

```python
import numpy as np

def integral(f, a, b, n):
    h = (b - a) / n
    s = 0
    for i in range(n):
        x = a + i * h
        s += f(x)
    return s * h

def f(x):
    return x**2

a = 0
b = 2
n = 1000

result = integral(f, a, b, n)
print(result)
```

在上述代码中，我们首先导入了NumPy库，然后定义了一个积分的函数`integral`。接着，我们定义了一个函数`f`，该函数的输入是一个变量`x`，输出是`x`的平方。然后，我们设置了积分的区间`a`和`b`的值为0和2，以及积分的步长`n`为1000。最后，我们调用`integral`函数，并将结果打印出来。
