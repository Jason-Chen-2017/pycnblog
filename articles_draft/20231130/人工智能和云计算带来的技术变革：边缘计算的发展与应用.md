# 2.核心概念与联系

在了解边缘计算的核心概念之前，我们需要了解一些基本概念：

- **云计算**：云计算是一种基于互联网的计算模式，它允许用户在远程服务器上存储和处理数据，而无需购买和维护自己的硬件和软件。这使得用户可以更轻松地扩展和优化其计算能力。

- **人工智能**：人工智能是一种通过算法和机器学习来模拟人类智能的技术。它涉及到自然语言处理、计算机视觉、机器学习等多个领域。

- **边缘计算**：边缘计算是一种将计算能力推向设备、传感器和网络边缘的计算模式。它旨在减少数据传输到云服务器的时间和带宽需求，从而提高系统的响应速度和可靠性。

边缘计算与云计算和人工智能之间的联系如下：

- **边缘计算与云计算的联系**：边缘计算可以与云计算相结合，以实现更高效的数据处理和存储。通过将计算能力推向边缘设备，我们可以减少对云服务器的依赖，从而降低延迟和带宽需求。

- **边缘计算与人工智能的联系**：边缘计算为人工智能提供了更多的可能性。通过在边缘设备上进行实时数据处理，我们可以更快地获取有关环境、设备和用户的信息，从而为人工智能系统提供更丰富的数据来源。
