# 4.具体代码实例和详细解释说明
## 4.1 定义变量的代码实例
```rust
fn main() {
    let x = 5;
    println!("x = {}", x);
}
```
在这个代码实例中，我们定义了一个名为x的变量，并将其初始值设为5。然后，我们使用println!宏来输出变量x的值。

## 4.2 定义函数的代码实例
```rust
fn add(x: i32, y: i32) -> i32 {
    x + y
}

fn main() {
    let result = add(3, 4);
    println!("result = {}", result);
}
```
在这个代码实例中，我们定义了一个名为add的函数，它接受两个i32类型的参数，并返回一个i32类型的结果。然后，我们在main函数中调用add函数，并将结果输出到控制台。

## 4.3 定义结构体的代码实例
```rust
struct Point {
    x: i32,
    y: i32,
}

fn main() {
    let origin = Point { x: 0, y: 0 };
    println!("origin = {:?}", origin);
}
```
在这个代码实例中，我们定义了一个名为Point的结构体，它有两个i32类型的字段：x和y。然后，我们在main函数中创建了一个名为origin的Point实例，并将其输出到控制台。
