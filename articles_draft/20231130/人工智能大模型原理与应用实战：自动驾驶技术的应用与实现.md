## 4.1代码实例

```python
import cv2
import numpy as np
import math

# 读取图像

# 图像处理
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
blur = cv2.GaussianBlur(gray, (5, 5), 0)

# 特征提取
edges = cv2.Canny(blur, 50, 150)

# 目标识别
contours, hierarchy = cv2.findContours(edges, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

# 控制算法
for cnt in contours:
    x, y, w, h = cv2.boundingRect(cnt)
    if w > 50 and h > 50:
        cv2.rectangle(img, (x, y), (x + w, y + h), (0, 255, 0), 2)

# 显示结果
cv2.imshow('result', img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## 4.2详细解释说明

上述代码实例主要包括以下步骤：

1. 读取图像：使用`cv2.imread()`函数读取图像，并将其转换为灰度图像。

2. 图像处理：使用`cv2.cvtColor()`函数将图像转换为灰度图像，并使用`cv2.GaussianBlur()`函数对图像进行高斯滤波。

3. 特征提取：使用`cv2.Canny()`函数对图像进行边缘检测，以提取特征。

4. 目标识别：使用`cv2.findContours()`函数对边缘图像进行轮廓检测，并将其存储在`contours`变量中。

5. 控制算法：遍历所有的轮廓，并使用`cv2.boundingRect()`函数计算每个轮廓的边界框。如果边界框的宽度和高度都大于50，则将其绘制在原始图像上。

6. 显示结果：使用`cv2.imshow()`函数显示原始图像，并使用`cv2.waitKey()`和`cv2.destroyAllWindows()`函数等待用户按任意键并关闭窗口。
