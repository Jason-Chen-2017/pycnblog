# 2.核心概念与联系

在探讨人类航空航天技术的进步与探索之前，我们需要了解一些核心概念和联系。

## 2.1 航空技术

航空技术是指人类利用飞行器（如飞机、直升机、无人驾驶飞行器等）进行空中旅行的技术。航空技术的发展历程可以分为以下几个阶段：

1. 古代飞行器：人类从古代就开始研究飞行器，如箭、翼、箭簇等。这些飞行器的设计和制造技术是人类早期航空技术的基础。
2. 汽机时代：19世纪末，汽机技术的发展为人类航空技术提供了新的动力源。汽机引擎驱动的飞行器开始出现，如汽飞机、汽翼等。
3. 电机时代：20世纪初，电机技术的发展为人类航空技术提供了更高效的动力源。电机引擎驱动的飞行器开始广泛应用，如电机飞机、电翼等。
4. 无人驾驶飞行器：21世纪初，无人驾驶技术的发展为人类航空技术提供了新的飞行方式。无人驾驶飞行器可以根据预设的路径自动飞行，如无人驾驶飞机、无人驾驶直升机等。

## 2.2 航天技术

航天技术是指人类利用火箭发射器进行宇宙探索的技术。航天技术的发展历程可以分为以下几个阶段：

1. 古代火箭：人类从古代就开始研究火箭技术，如中国的火箭兵、希腊的火箭人等。这些火箭技术是人类早期航天技术的基础。
2. 汽机时代：20世纪初，汽机技术的发展为人类航天技术提供了新的动力源。汽机引擎驱动的火箭发射器开始出现，如汽火箭、汽箭等。
3. 电机时代：20世纪中叶，电机技术的发展为人类航天技术提供了更高效的动力源。电机引擎驱动的火箭发射器开始广泛应用，如电火箭、电箭等。
4. 无人驾驶火箭：21世纪初，无人驾驶技术的发展为人类航天技术提供了新的飞行方式。无人驾驶火箭可以根据预设的轨道自动飞行，如无人驾驶火箭、无人驾驶火箭发射器等。
