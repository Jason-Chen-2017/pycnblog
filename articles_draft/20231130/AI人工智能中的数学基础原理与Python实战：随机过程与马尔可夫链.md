例子：一个简单的六面骰子游戏

在这个游戏中，我们有一个六面骰子，每次掷骰子都会得到一个1到6的结果。我们想要模拟这个游戏的过程，并计算掷骰子的平均值。

接下来，我们需要确定状态转移。在这个例子中，状态转移是随机的，我们可以使用随机数生成器来生成每次掷骰子的结果。

最后，我们需要计算期望值。在这个例子中，期望值是掷骰子的平均值。我们可以使用Python的numpy库来计算这个平均值。

以下是Python代码实例：

```python
import numpy as np

# 生成随机变量的分布
probability_distribution = np.ones(6) / 6

# 状态转移
def roll_dice():
    return np.random.choice(range(1, 7), p=probability_distribution)

# 计算期望值
def calculate_average(num_rolls):
    total_value = 0
    for _ in range(num_rolls):
        total_value += roll_dice()
    return total_value / num_rolls

# 主程序
num_rolls = 1000
average_value = calculate_average(num_rolls)
print("掷骰子的平均值为：", average_value)
```
