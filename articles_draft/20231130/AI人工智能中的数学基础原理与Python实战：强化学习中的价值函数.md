# 3.1价值迭代算法
价值迭代算法是一种用于计算价值函数的算法，它通过迭代地更新价值函数来逼近最优价值函数。价值迭代算法的核心思想是通过迭代地更新价值函数来逼近最优价值函数。

价值迭代算法的具体操作步骤如下：

1. 初始化价值函数：将所有状态的价值函数初始化为0。

2. 迭代更新价值函数：对于每个状态，计算其与所有可能的行为相关联的价值函数的期望。

3. 更新价值函数：将更新后的价值函数存储到价值函数中。

4. 重复步骤2和3，直到价值函数收敛。

价值迭代算法的数学模型公式如下：

V(s) = max(R(s) + γ * max(V(s'))), 其中s'是s的所有可能的下一状态。
