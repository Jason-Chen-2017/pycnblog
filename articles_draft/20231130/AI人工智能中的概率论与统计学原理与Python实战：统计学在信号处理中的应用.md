## 3.1概率论
### 3.1.1事件
事件是随机过程中可能发生的结果。在信号处理中，我们经常需要处理随机信号，因此概率论的事件是非常重要的。

### 3.1.2样本空间
样本空间是所有可能发生的事件集合。在信号处理中，我们需要对随机信号进行分析和处理，因此样本空间是非常重要的。

### 3.1.3概率
概率是事件发生的可能性。在信号处理中，我们需要处理随机信号，因此概率是非常重要的。概率的计算公式为：

P(A) = n(A) / n(S)

其中，P(A)是事件A的概率，n(A)是事件A的样本数，n(S)是样本空间的样本数。

### 3.1.4条件概率
条件概率是一个事件发生的可能性，给定另一个事件已经发生的情况。在信号处理中，我们经常需要处理随机信号，因此条件概率是非常重要的。条件概率的计算公式为：

P(A|B) = P(A∩B) / P(B)

其中，P(A|B)是事件A发生的概率，给定事件B已经发生的情况，P(A∩B)是事件A和事件B同时发生的概率，P(B)是事件B的概率。

## 3.2统计学
### 3.2.1均值
均值是一个随机变量的期望值。在信号处理中，我们经常需要对数据进行分析和处理，因此均值是非常重要的。均值的计算公式为：

μ = Σ(xi * pi)

其中，μ是均值，xi是随机变量的取值，pi是随机变量的概率。

### 3.2.2方差
方差是一个随机变量的分散度。在信号处理中，我们经常需要对数据进行分析和处理，因此方差是非常重要的。方差的计算公式为：

σ^2 = Σ((xi - μ)^2 * pi)

其中，σ^2是方差，xi是随机变量的取值，μ是均值，pi是随机变量的概率。

### 3.2.3协方差
协方差是两个随机变量之间的相关性。在信号处理中，我们经常需要对数据进行分析和处理，因此协方差是非常重要的。协方差的计算公式为：

cov(X,Y) = Σ((xi - μx)(yi - μy) * pi)

其中，cov(X,Y)是协方差，xi是随机变量X的取值，yi是随机变量Y的取值，μx是随机变量X的均值，μy是随机变量Y的均值，pi是随机变量X和Y的概率。

### 4.1.4条件概率

```python
import random

# 定义事件
event_A = True
event_B = False

# 生成随机数
random_number = random.random()

# 判断随机数是否大于0.5
if random_number > 0.5:
    event_A = True
else:
    event_A = False

if random_number < 0.5:
    event_B = True
else:
    event_B = False

# 计算事件A和事件B同时发生的概率
probability_A_and_B = event_A * event_B

# 计算事件B的概率
probability_B = event_B / 2

# 计算条件概率
conditional_probability_A_given_B = probability_A_and_B / probability_B

# 打印条件概率结果
print("条件概率A给定B:", conditional_probability_A_given_B)
```
