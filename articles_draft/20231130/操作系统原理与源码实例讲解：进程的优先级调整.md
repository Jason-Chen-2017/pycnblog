# 2.核心概念与联系

在操作系统中，进程优先级是指操作系统调度进程时考虑的一个因素，用于确定进程在队列中的排序顺序。进程优先级高的进程在队列中排在优先级低的进程前面，因此在调度时得到更快的响应和执行。

进程优先级调整的核心概念包括：

- 优先级：进程优先级是一个整数，用于表示进程在调度队列中的优先顺序。优先级高的进程在队列中排在优先级低的进程前面，因此在调度时得到更快的响应和执行。
- 调度策略：操作系统中的调度策略是指操作系统如何选择哪个进程进行调度的规则。常见的调度策略有先来先服务（FCFS）、最短作业优先（SJF）、优先级调度等。
- 资源分配：进程优先级调整会影响进程的资源分配。高优先级的进程通常可以获得更多的系统资源，如CPU时间片、内存空间等，从而提高其执行速度。

    int current_process = 0;
    int next_process = 0;

                next_process = i;
