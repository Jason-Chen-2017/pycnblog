# 2.核心概念与联系

结构体是一种用户自定义的数据类型，它可以包含多个字段，这些字段可以是基本类型或其他结构体类型。结构体可以实现多态性，因为它可以包含多个不同类型的字段。

接口是一种抽象类型，它可以定义一组方法，这些方法可以被实现类型的实例调用。接口可以实现多态性，因为它可以被多个不同类型的实例实现。
