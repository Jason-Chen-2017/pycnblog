## 3.1 加法
在数值计算中，加法是一种非常基本的运算。它是指将两个数字相加的过程。在Python中，我们可以使用加法运算符（+）来实现加法。

数学模型公式：a + b = c

具体操作步骤：
1. 定义两个数字a和b。
2. 使用加法运算符（+）将a和b相加。
3. 将结果存储在一个变量中。

Python代码实例：
```python
a = 5
b = 3
c = a + b
print(c)  # 输出: 8
```

## 3.2 减法
在数值计算中，减法是一种非常基本的运算。它是指将一个数字从另一个数字中减去的过程。在Python中，我们可以使用减法运算符（-）来实现减法。

数学模型公式：a - b = c

具体操作步骤：
1. 定义两个数字a和b。
2. 使用减法运算符（-）将a从b中减去。
3. 将结果存储在一个变量中。

Python代码实例：
```python
a = 5
b = 3
c = a - b
print(c)  # 输出: 2
```

## 3.3 乘法
在数值计算中，乘法是一种非常基本的运算。它是指将两个数字相乘的过程。在Python中，我们可以使用乘法运算符（*）来实现乘法。

数学模型公式：a * b = c

具体操作步骤：
1. 定义两个数字a和b。
2. 使用乘法运算符（*）将a和b相乘。
3. 将结果存储在一个变量中。

Python代码实例：
```python
a = 5
b = 3
c = a * b
print(c)  # 输出: 15
```

## 3.4 除法
在数值计算中，除法是一种非常基本的运算。它是指将一个数字除以另一个数字的过程。在Python中，我们可以使用除法运算符（/）来实现除法。

数学模型公式：a / b = c

具体操作步骤：
1. 定义两个数字a和b。
2. 使用除法运算符（/）将a除以b。
3. 将结果存储在一个变量中。

Python代码实例：
```python

b = 3
c = a / b

```
