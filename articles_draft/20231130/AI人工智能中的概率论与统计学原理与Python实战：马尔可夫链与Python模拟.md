## 3.1 马尔可夫链的基本概念
马尔可夫链是一种随机过程，其中当前状态只依赖于前一状态，而不依赖于之前的状态。在马尔可夫链中，状态可以是任何事物，例如：

- 人的行为
- 天气
- 股票价格
- 网页浏览

马尔可夫链的核心概念是：当前状态只依赖于前一状态，而不依赖于之前的状态。这意味着，在马尔可夫链中，只需要知道前一状态和状态转移概率，就可以预测下一状态。

## 3.2 马尔可夫链的状态转移概率
在马尔可夫链中，状态转移概率是从一个状态到另一个状态的概率。状态转移概率可以用一个概率矩阵表示，其中每一行表示从一个状态到另一个状态的概率。

例如，假设我们有一个三个状态的马尔可夫链，状态1到状态2的概率为0.5，状态1到状态3的概率为0.5。那么，状态转移概率矩阵将是：

```
| 0.5  0.5  0.0 |

```

在这个矩阵中，第一行表示从状态1到其他状态的概率，第二行表示从状态2到其他状态的概率，第三行表示从状态3到其他状态的概率。

