# 2.核心概念与联系

在数据中台架构中，数据监控和数据治理是两个重要的概念。数据监控是指对数据的实时监控，以便及时发现问题和优化性能。数据治理则是指对数据的管理，包括数据的创建、存储、使用和删除等。

数据监控和数据治理之间存在密切的联系。数据监控可以帮助数据治理提高数据的质量和可靠性。同时，数据治理也可以帮助数据监控更好地管理和监控数据。
