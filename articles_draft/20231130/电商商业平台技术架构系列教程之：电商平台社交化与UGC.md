# 2.核心概念与联系

## 2.1 社交化

社交化是指在电商平台上，用户可以进行互动、分享、评论等操作，从而建立起社交关系。社交化可以帮助平台建立起用户的信任感，提高用户的互动度，从而提高销售额和盈利能力。

社交化的核心概念包括：

- 用户关系：用户之间的关系，可以是好友关系、关注关系等。
- 用户互动：用户之间的互动，可以是评论、点赞、分享等。
- 用户生成内容：用户生成的内容，可以是评论、评价、图片等。

## 2.2 UGC

UGC（User Generated Content，用户生成内容）是指用户在电商平台上自主创作的内容，例如评论、评价、图片等。UGC可以帮助平台建立起用户的信任感，提高用户的互动度，从而提高销售额和盈利能力。

UGC的核心概念包括：

- 用户评论：用户对商品或服务的评价。
- 用户评价：用户对商品或服务的评价。
- 用户图片：用户上传的图片。

    return 1 - distance.euclidean(user1, user2)

# 推荐新的关注对象
