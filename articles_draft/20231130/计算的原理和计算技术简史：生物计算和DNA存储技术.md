```python
from Bio import SeqIO
from Bio.Seq import Seq
from Bio.SeqRecord import SeqRecord
from Bio.Align import PairwiseAligner

# 读取序列文件
seq1_file = "seq1.fasta"
seq2_file = "seq2.fasta"

seq1 = SeqIO.read(seq1_file, "fasta")
seq2 = SeqIO.read(seq2_file, "fasta")

# 进行序列比对

# 输出比对结果
print(alignment)
```

## DNA存储技术

以下是一个将文本数据编码为DNA序列的Python代码实例：

```python
import random
import string

def encode_dna(text):
    dna_table = str.maketrans("ATCG", "TAGC")
    dna_seq = "".join(text.translate(dna_table))
    dna_seq = dna_seq.upper()
    dna_seq = dna_seq.translate(str.maketrans("", "", string.punctuation))
    dna_seq = dna_seq.translate(str.maketrans("", "", string.whitespace))
    return dna_seq

text = "Hello, World!"
dna_seq = encode_dna(text)
print(dna_seq)
```

在这个代码实例中，我们首先定义了一个encode_dna函数，该函数将文本数据编码为DNA序列。然后我们使用这个函数将文本数据"Hello, World!"编码为DNA序列，并输出结果。
