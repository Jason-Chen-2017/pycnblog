# 1.背景介绍


后期工艺(post-processing) 是将模组完成的设计转化成完整的集成电路的过程。它包括从制造过程后期制备的测试样机进行反馈采集、设备性能评价、工作参数设定等后处理过程。此外还有对设备结构进行封装，完成材料封装和元器件焊接。后期工艺是确保集成电路能满足需求，并具有可靠性和可维护性的一项重要环节。
传统的线形制造方法要求设计人员必须自己动手制作电路板，而作为制造者，需要掌握好钻孔、焊接、烧结、打粗、焊盘和刷铜这些先进工艺技巧。不管是采用何种封装材料，制作完成的集成电路板都必须经过后期工艺才能进行封装。在这个过程中，还要考虑各种反映不良率(corrosion rate)、组件间相互作用、噪声以及其他因素。因此，后期工艺技术是一个技术活跃且充满挑战的领域。
目前，人们已经开发出了很多的后期工艺技术，如焊接技术、金属腐蚀技术、温度补偿技术、薄膜封装技术、电解电容器技术等等。这些技术的实用性及其精益程度均已得到验证。然而，在实际生产中，仍然存在着各种问题，例如缺乏专业知识、高昂的工厂成本、无法满足客户的速度要求等。因此，有必要开发一套完整的后期工艺流程体系，提高效率，降低成本，达到产品质量和用户满意度的高度统一。
为了有效地完成后期工艺流程，一定要正确认识其中的关键环节和环节之间的关系。下面，就让我们一起来看看芯片制造技术的后期工艺流程与技术。
# 2.核心概念与联系
## 2.1 封装层次与封装材料
封装层次： 按照电气连接点位置来分层。即上层封装上表面缺陷或开路，下层封装裸晶圆和绝缘子。一般从电源、引脚、集成电路芯片及其接口层开始，逐层向内封装直至基底。

封装材料： 主要有微米级别的硅胶、微纳米级别的镀锌板、厚度介于两者之间，颜色灰度由浅到深的石墨等。

两种不同规格封装材料的比较：微米级硅胶/微纳米级镀锌板（用于封装电路板） vs 微米级硅胶（用于封装绝缘子、电阻器等）

| 比较项目 | 微米级硅胶/微纳米级镀锌板 | 微米级硅胶   |
| -------- | ---------------------- | ------------ |
| 沉积性   | 低                     | 高           |
| 硬度      | 中                     | 低           |
| 耐候     | 长                     | 短           |
| 温度      | -                      | 冷           |
| 熔融      | 不熔                   | 可熔         |
| 可擦拭    | 不可擦拭               | 可擦拭       |

## 2.2 后期工艺流程
后期工艺流程分为前期准备工作、模具制作、工艺试验、焊接、封装、测试、验证、质量管理五个主要阶段。
### 2.2.1 前期准备工作
包括设计人员和工程师对模组进行综合设计、熟悉仪器设备、了解后期工艺原理、沟通工作分配。
### 2.2.2 模具制作
制作模具：模具为机械零件提供支撑，用来固定集成电路板上的元器件并锚定电源。目前，模具制作的方法有：加工电镜、温水切割法、扫尾法、热轧法、手动压敷法等。

电镜模具制作：电镜模具制作是利用电镜将模具固定在集成电路板上的过程。电镜模具制作优点是简单、经济、快速。缺点是精度较差、易受电路板的变形影响。适用于小型或小型批量生产。

温水切割法模具制作：温水切割法模具制作是通过搓面、画线等方式来切割模具，实现模具固定在集成电路板上的目的。优点是容易进行、便宜、精度高。缺点是切割方向不确定、结果不可控。适用于中型批量生产。

扫尾法模具制作：扫尾法模具制作是将模具沿某一方向移出原有的位置，然后堆叠到另一个位置，并将模具旋转一定角度后再固定在集成电路板上。优点是精度高、应用范围广泛。缺点是费时、费力、无控制能力。适用于复杂的集成电路板。

热轧法模具制作：热轧法模具制作是利用矿石或金属板上的热电荷进行切割、修剪、打磨和摩擦模具，固定模具在集成电路板上的目的。优点是切割速度快、容易控制模具质量、模具坚固性强。缺点是有风险、费用高、难度大。适用于大型集成电路板。

手动压敷法模具制作：手动压敷法模具制作是利用手工工具在模具表面加厚，然后再将其放入压射台中，按压模具使模具固定在集成电路板上的过程。优点是容易操作、费用低、快捷、操作方便。缺点是模具容易起皮、坏死、损坏。适用于小型和中型批量生产。

### 2.2.3 工艺试验
工艺试验：将模具制作好的模具装配到集成电路板上进行试验，根据模具固定情况调整各项参数，使得集成电路板上的元器件固定在模具上并连上电源。若模具失败，则需重新制作模具。

测试对象：模具、绝缘子、电池、电源电缆等。

工艺试验方案：

(1)电源电压测试

将模具固定在集成电路板上，通过连接电源电路板与试验电路板的方式测试电源电压。电压测试的目的是检测模具与绝缘子、电池、电源电缆等间的阻抗匹配程度。如果出现阻抗偏差，则需要进行检查并进行重新制作。

(2)绝缘子测试

将模具固定在集成电路板上，通过连接多种类型绝缘子、电阻器等测试绝缘子的抗极化性、缺陷密度等性能。如果发现异常，则需要重新制作模具。

(3)电容测试

将模具固定在集成电路板上，通过连接电容等电子元件测试电容的抗漏失效、功耗、电感等性能。如果发现异常，则需要重新制作模具。

(4)电流测试

将模具固定在集成电路板上，通过示波器测量电流在模具的不同位置是否恒定的，以及当前流是否能够吸收和导通至元件。如果出现不符合预期的现象，则需要重新制作模具。

(5)热传递测试

将模具固定在集成电路板上，通过热压计、温度计、示波器等测试集成电路板的热传递。如果测试结果显示信号失真或噪声严重，则可能需要重新制作模具。

### 2.2.4 焊接
焊接：在焊接过程中，模具与封装材料会发生碰撞、摩擦、温度升降、气泵回流等现象。因此，在模具焊接之前，必须对焊接场地、焊接工具、焊接条件等进行充分准备。

焊接工具：包括：电钻、电冲击、电镊、电水槽、蒸汽压板等。

焊接场地：选择隔离环境的干净、无杂质、通风空调好的地面，避免开辟任何异物。

焊接条件：尽量维持常温、高湿度，以保证焊接效果。

### 2.2.5 封装
封装：封装就是将所有电路器件或者部分电路器件组装成一个整体，并且组装之后的模具可以直接安装到母板上使用。这一过程主要是为了将杂乱无章的电路元器件进行整齐划一的排列，提高其完美的设备表现。

一般的封装技术分为三种：微表面包层封装（MSL）、光刻胶电镀封装（LPC）和自然膜封装（NFF）。

MSL封装：微表面包层封装是一种简单又实用的封装技术，通过向表面施加微小的张力来实现材料的聚合，从而达到对晶体的全覆盖。MSL封装方法可以有效解决室温、低温下的多个微小设备间隔的不良影响，避免设备之间的相互干扰、联系，增加了封装性能。但是微表面包层封装容易产生温度升高、功耗增加等问题。

LPC封装：光刻胶电镀封装(Light-Passive Circuit Technique, LPC)是一种基于纳米颗粒晶体管的可靠、精密、透明、易打理、轻便、无螺栓、容易分离、防护等特点的封装技术，可以将导电材料嵌入到母片表层的微晶孔中，并在外部侧壁上施加外涂层，同时还能防止异物侵入。LPC封装也可充分利用导电材料本身的性质来提高封装的性能，具有很高的封装密度和良好的耐磨能力。

NFF封装：自然膜封装(Natural Film Fabrication, NFF)是一种采用大范围自然膜生产工艺的方法，能够显著缩短晶圆封装的时间、降低成本和降低了制成过程中的温度、湿度和光照条件，同时还具有耐候性、耐候时间长、精度高等优点。这种封装也被称为UVM(Uninterruptible Voltage Mask)材料，这种材料对电子组件的功能、耐久性、缺陷容错率等性能做出了很大的贡献。

### 2.2.6 测试
测试：将制作好的集成电路测试于生产线上，确认集成电路板的实际运行特性。测试过程中还要注意各类测试设备的维护、检修、改造等方面的工作。

测试设备：包括示波器、示波器分析仪、电阻耐久性测试仪、电容耐久性测试仪、耐力测试仪、高压压力表、低压压力表、高压与低压步进电机、集成电路单元测试仪、模拟变压器、万用表、声光表、马达、探头等。

测试方法：一般的测试方法有仪器验证法、工艺质量监督法、产品合格性检验法、质量保证审核法。

### 2.2.7 验证
验证：验证是指对最终的集成电路板进行一些定性和定量的测试，以验证其品质是否符合设计要求。

定性验证：指将生产好的集成电路与标准的验证模型比较，从而判断集成电路的实际性能是否符合设计标准。常用的定性验证模型有：

微观测试法：指采用微观测试方法对集成电路各个部件的性能、延迟、错误率等进行检测，检测结果为静态测试结果；

静态测试法：指对整个集成电路进行静态测试，检测其工作电压、电流、功率、阻抗等各项指标，检测结果为静态测试结果；

动态测试法：指对整个集成电路的特定输入输出进行操作后，检测其工作电压、电流、功率、阻抗、相关功率等指标，检测结果为动态测试结果；

定量验证：指对测试产品的内部结构、外部表现、尺寸、重量、材质等进行定量的测试。如采用电路印制工艺测试微晶电路板，通过声波、雷击等方式对电路进行测试。

### 2.2.8 质量管理
质量管理：质量管理是对产品制造过程中，对各项环节产品质量的管理，保障产品质量和产品的一致性。质量管理的目标是确保产品的质量与规格之间的平衡，控制产品出现缺陷后造成的损失，确保公司产品的持续竞争力和市场占有率。

质量管理包括以下几个方面：

1、验收管理：验收管理是产品进入质量管理链条的第一个环节，主要是为了确保产品在制造过程中没有质量瑕疵，并对产品的基本性能与外观等进行初步验收。在验收前，公司必须对所有生产、交付环节进行检查，确保产品在产品质量管理的各个环节得到满足。

2、投产前质量控制：投产前质量控制是将产品送往专业生产工厂之前对产品的质量进行控制，目的是减少产品在使用过程中出现的缺陷，降低设备故障率，提高产品的实际使用率。

3、产品质量控制：产品质量控制是产品生产过程中质量保证与监控，主要用于对产品的表征产品质量、过程质量和可靠性、运输质量进行有效的控制。对产品的整体质量控制可以促进产品的协同性和一致性，消除工艺瑕疵和生产过程的不确定性，保障产品质量和利益。

4、产品退货管理：产品退货管理是对产品质量管理链条中最重要的环节之一，也是对生产过程中的失误、不合格、不良事件等进行分析处理、制止、排除的过程。

5、产品质量安全管理：产品质量安全管理是通过建立正确的产品质量管理机制来支持企业的产品质量安全。其核心是建立规范、制度、流程和责任体系，以保证公司产品的质量安全。

后期工艺技术将提供一种完整的后期工艺流程和方法，帮助企业在制造过程后期控制成本、提高产品质量，降低产品开发周期，提高企业的竞争力。
