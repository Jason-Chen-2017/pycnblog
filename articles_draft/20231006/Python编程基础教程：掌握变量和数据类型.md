# 2.核心概念与联系
## 变量和数据类型
- 数据类型是指储存在变量中的值的类型。例如，整数（int）、浮点数（float）、字符串（str）、布尔值（bool）等。
- 变量是在程序运行期间创建的名字，它用于存储值或引用数据，可在程序的任何位置访问和使用。变量名通常采用小驼峰命名法，即首字母小写，中间每个单词的首字母大写。
- 在Python中，变量不需要声明类型。每个变量在使用前都必须赋值，变量类型会根据赋给它的初始值自动确定。
```python
x = "hello world" # str类型变量
y = 10          # int类型变量
z = True        # bool类型变量
print(type(x))   # <class'str'>
print(type(y))   # <class 'int'>
print(type(z))   # <class 'bool'>
```
## 操作符
操作符是一种特殊符号，它告诉计算机进行特定运算或比较。Python支持丰富的运算符，包括算术运算符、逻辑运算符、位运算符、比较运算符、赋值运算符等。本文只介绍其中几个重要的运算符。
### 算术运算符
- `+` : 加法，返回两个数字的总和
- `-` : 减法，返回两个数字的差
- `*` : 乘法，返回两个数字的积
- `/` : 除法，返回两个数字的商
- `%` : 取模，返回除法的余数
- `**` : 求幂，返回一个数的次方
```python
a = 5 + 3      # 8
b = 7 - 4      # 3
c = 2 * 4      # 8
d = 10 / 3     # 3.33...
e = 10 % 3     # 1
f = 2 ** 3     # 8
```
### 赋值运算符
- `=` : 将右侧的值赋值给左侧的变量。
- `+=` : 把左侧变量的值增加指定的值。
- `-=` : 从左侧变量中减去指定的值。
- `*=` : 给左侧变量赋值自身乘以指定的值。
- `/=` : 给左侧变量赋值自身除以指定的值，结果为浮点型。
- `%=` : 给左侧变量赋值自身取模指定的值。
- `**=` : 给左侧变量赋值自身的指定次方的值。
```python
num = 10           # 赋值运算符示例
num += 5           # num = num + 5
num -= 3           # num = num - 3
num *= 2           # num = num * 2
num /= 2.0         # num = num / 2.0 (注意此处使用浮点数)
num %= 3           # num = num % 3
num **= 2          # num = num ** 2
```
### 逻辑运算符
- `and` : 用于Boolean表达式，只有两个都为True时才为True。
- `or` : 用于Boolean表达式，只要两个有一个为True就为True。
- `not` : 用于Boolean表达式，取反操作。
```python
# 逻辑运算符示例
flag1 = False       # flag1为False
flag2 = True        # flag2为True
result = flag1 and flag2    # result = False
result = flag1 or flag2     # result = True
result = not flag1          # result = True
```
## 控制流语句
### if、else、elif
if语句是最基本的条件语句。它用来判断某个条件是否成立，如果成立，则执行if块内的代码；否则，跳过if块，继续执行后续代码。在if语句中，还可以使用elif子句（else if），用来表示多个分支，当第一个分支条件不满足时，选择第二个条件。
```python
score = 90                    # 假设学生的成绩为90分
if score >= 90:               # 判断成绩是否大于等于90分
    print("优秀")             # 如果成绩大于等于90分，输出“优秀”
elif score >= 80:             # elif子句表示“否则，如果”
    print("良好")             # 如果成绩大于等于80分，输出“良好”
else:                         # else子句表示“否则”
    print("不及格")            # 如果成绩不满足上述条件，输出“不及格”
```
### for 和 while
for语句用来遍历序列（如列表、元组、字典）中的元素，每次迭代将当前元素赋值给指定的变量，然后执行语句块。while语句用来重复执行一个语句块，直到某个条件满足为止。
```python
fruits = ['apple', 'banana', 'orange']              # 创建一个包含水果名称的列表
for fruit in fruits:                              # 使用for循环遍历列表中的元素
    print("I like", fruit)                        # 每次循环打印出当前水果的名称

count = 0                                         # 初始化计数器
while count <= len(fruits):                       # 当计数器小于等于水果数量时，循环执行以下语句块
    print("I like", fruits[count])                 # 每次循环打印出第count个水果的名称
    count += 1                                    # 计数器加1
```
## 函数
函数是一个可以重复使用的代码块，它接受输入参数，对这些参数进行处理，然后返回一个输出值。在Python中，定义函数使用def关键字，并添加函数名、括号()和冒号:。函数体一般被缩进。
```python
def greetings(name):                   # 定义了一个名为greetings的函数，它接受一个参数——姓名
    print("Hello,", name)             # 打印欢迎消息

greetings('Alice')                     # 调用greetings函数，传入参数'Alice'
```
