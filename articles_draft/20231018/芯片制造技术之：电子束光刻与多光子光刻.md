# 1.背景介绍


电子束光刻（Electron Beam Deposition，EBD）、多光子光刻（Multilayer Deposition，MLD），是两种常用的芯片制造工艺方法。电子束光刻技术主要应用于先导层及封测等专用芯片的制造，而多光子光刻技术则广泛用于一般的微型芯片及集成电路制造。近年来，随着半导体产业的快速发展，各类芯片都开始采用多光子光刻技术制造。所以，了解两者的基本原理和特点，对掌握正确的生产工艺方法和设计技巧非常重要。本文首先简要介绍EBD和MLD的基本知识，然后介绍它们的关键区别和相同点，最后阐述EBD和MLD的应用领域，并给出一些典型案例进行介绍。
# 2.核心概念与联系
## 2.1 电子束光刻与多光子光刻
### 2.1.1 电子束光刻
电子束光刻（EBD）是指利用电子束在导电介质表面上快速排列钢丝的方式制备、维持以及后期修正微晶片、玻璃芯片、CMOS、FET和硅基器件等电子器件，其特点是在极短时间内完成低温工艺过程，并具有很高的可靠性和重复性。所谓电子束光刻，就是在普通电子学家以外的人员参与了制作过程。它由四个步骤组成：放电；增强；聚焦；提升。其中，放电是指将需要制备的物品通过电磁波放射到一个能量载入器上，即“下料”环节；增强是指将下料后的材料通过对准感光元件或者调整设备位置使得在高温环境下制备，即“加热”环节；聚焦是指调整焊盘或物体上的孔径使得电流运动迅速平滑，即“聚焦”环节；提升是指在高温条件下施加特定微小的增压作用，以促进电子融合以及使结构纤维化，即“提升”环节。

电子束光刻的优点有：
 - 操作简单
 - 速度快
 - 抗氧化能力强
 - 费用低

电子束光刻的缺点有：
 - 存在开槽现象
 - 容易受到反应物质影响
 - 不宜长期使用

### 2.1.2 多光子光刻
多光子光刻（MLD）是一种芯片制造工艺，它利用在晶体表面的不同光谱范围内形成多种多层、带状结构的电子气泡。这种结构中的电子气泡具有很强的空间分辨率，能够记录并识别微观尺寸小的颗粒，因此可以精确地刻划复杂且稳定的电路布局。MLD的技术基础是微电子学，它是一种利用电子气泡构造晶体电子结构的新型工艺，利用每秒数十亿至数百亿级的电子粒子进行连续的掺杂分布，生成高度非均匀的电子晶体结构。多光子光刻的主要流程包括：电子束加速→激光加速→引力加速→介质冲洗→晶格化。

1.电子束加速：电子束加速又称为Lumerical Ebeam（LE）方案，它是一种将计算机控制的光电子发射管以及计算机模拟处理信号的数据转换到晶片表面的控制策略，使用最简单的加速电子束的方法直接产生高速多光子器件。LE方式的制备过程将原始设计文件经过精心处理、加工、烘焙、冷却等环节，输出加速电子束，通过电子束对准晶片表面形成高效率的高光照射。

2.激光加速：激光加速是多光子光刻的一种先进工艺，其特点是不使用电子束而是直接使用激光，直接在晶片表面形成高光束并对准物体形成电流，产生高速多层光栅结构。激光加速的制备工艺包括太阳能、紫外线和红外线照射等，产生的多层光栅具有良好的空间隔离性、完整性和耐腐蚀性。

3.引力加速：引力加速的优点是没有空隙，结构简单、可靠、较低耗能，适用于微型、小型和功能部件。而在碳纳米管等应用中，由于表面结构受到机械磁场和力场的干扰，会出现紊乱、间隙等问题。引力加速的制备工艺包含利用重力加劲推动的均质高斯分布法、利用多轴并行设计技术等。

4.介质冲洗：介质冲洗是一种重要工序，它是在晶片表面进行修饰、磨损以及抹胶的过程，可以消除微粒子的影响。介质冲洗的重要性在于减少多层结构之间的相互作用，防止电子粒子之间发生摩擦，并使结构的韧性更好。目前主流的介质冲洗工艺有涂布、镀锌、镀钛、溶解、冷冻、熔融等。

5.晶格化：晶格化的过程是在晶体的不同层次之间填充导电材料，从而形成具有一定宽度和厚度的晶格。它具有多种用途，如单晶硅片的制备、CMOS芯片的制备等。多光子光刻的制备方法涵盖了电子束加速、激光加速、引力加速、介质冲洗和晶格化五个环节，但通常每一项工艺都是独立的。另外，多光子光刻工艺还需要考虑微晶片的种类、制备设备、维持条件等因素。

多光子光刻的优点有：
 - 导电材料的高度非均匀，能够制备出不同的材料
 - 在低温下，不需要接触导电材料，产生较低的成本
 - 可以高速制备大型、复杂、多功能的芯片
 - 晶格化后具有较好的导电性能，可防止腐蚀和热膨胀

多光子光刻的缺点有：
 - 需要精密的控制电路布局
 - 有时会出现开槽现象
 - 制备过程比较复杂，耗费时间

## 2.2 关键区别与相同点
电子束光刻和多光子光刻在技术实现上有较大的不同。具体来说，电子束光刻是通过将材料和电流一起以电磁波形式传播到工作台上的移动电子束，形成高能的电子束。而多光子光刻则是利用微粒子、微尺寸电子、旋转的旋极结构和电子场来形成具有空间分辨率的电子晶体结构。两者的关键区别与相同点如下：

1.使用材料：电子束光刻中使用的材料为钢丝，而多光子光刻使用的材料为玻璃、金属材料等。

2.制备周期：电子束光刻的制备周期较短，通常在几十分钟内完成；而多光子光刻的制备周期较长，平均需几个月甚至更久的时间。

3.工艺流程：电子束光刻制备通常只有“放电-增强-聚焦-提升”四个环节，而多光子光刻的制备流程则是“电子束加速-激光加速-引力加速-介质冲洗-晶格化”，每个工艺环节都有其独特的特性。

4.难点：电子束光刻难点在于开槽现象和反应物质的影响，而多光子光刻的难点则在于控制精度、复杂的设计和制作工艺。

5.制造工艺：电子束光刻的制作工艺依赖于技术的创新和制造经验，其制备时间也依赖于电源的供应时间和精细化程度；而多光子光刻则是一个复杂的工艺流程，需要进行多次迭代，才能最终达到满意的效果。

6.价格比：电子束光刻的成本较低，但制备周期长，往往占到产品生命周期的70%，而且容易受到非均匀性材料的影响。而多光子光刻的成本相对较高，但制备周期短，能提供全面的解决方案。

综上所述，电子束光刻和多光子光刻在制备技术、制作工艺、价格比、硬件、生产、维护、技术创新等方面存在很大的差异。对于需要高速、精确、成本低廉的产品，电子束光刻就显得更加适合；而对于要求灵活、可控、精密、经济的应用，多光子光刻就会更加有效。
# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
本文将首先简要介绍EBD和MLD技术的基本原理，之后将重点阐述它们的应用领域，再通过具体的例子详细阐述一下相关应用。

## 3.1 EBD
EBD作为一种历史悠久的技术，其早期代表工作的目的是制造透明胶，其基本的制备过程可以概括为以下的4个步骤：
1. 实物放电（IC Implantation）：在一定的温度条件下，将待制备的物品（比如像样的胶水、透明胶等）通过特殊方式放射到导电材料上。
2. 激光排列（Emission Spectroscopy）：在排放后期光的过程中，通过探测微晶片表面电子结构的变化来获取电子元件的电场强度。
3. 导纳（Dislocation Nucleation）：通过设计电场的大小以及导电材料的挥动方向，实现导电材料中的多种奇异点，从而实现纳米颗粒的聚集。
4. 机械开槽（Electroplating）：将纳米颗粒填入导电材料中，并用挖掘机开槽、摧毁等方法实现纳米片的结构稳定。

总的来说，EBD作为一种应用落后于微电子学的技术，它大幅缩短了制备周期，但是由于开槽现象的存在，导致制备成果的精度受限。当今的制备方案更注重自动化，提升了效率，同时降低了人工操作的复杂度。

## 3.2 MLD
多光子光刻作为一种最新技术，其核心原理是用多种不同波长的微电子来构造晶体电子结构。其主要的制备工艺流程为：

1. 电子束加速（EBeam Acceleration）：利用计算机生成激光束，对晶片表面形成高速电子束，从而产生高光阶的高速多光子器件。

2. 激光加速（Xray Accelerated Deposition）：利用极端放射激光将多光子构件嵌入晶体表面，以获得高层次的电子结构。

3. 引力加速（Gravitational Accelerated Deposition）：借助重力加力，在晶体表面形成多层紧凑的光栅电子架構。

4. 介质冲洗（Chemical Cleaning）：在晶体表面进行精心设计的修饰，从而消除介质粒子的影响。

5. 晶格化（Patterning）：通过填充不同厚度的导电层，从而形成具有固定宽度和厚度的晶体。

可以看出，多光子光刻制备的工艺流程是由多个环节组合而成的，并且每一个环节都有其独特的特性。MLD是制备高速多层晶体电子结构的前沿技术，其优点是无空隙、结构简单、可靠、较低耗能，适用于微型、小型和功能部件。然而，由于所需的控制精度和制作工艺比较复杂，仍处于起步阶段，并未得到广泛应用。

## 3.3 应用领域
### 3.3.1 集成电路制造
由于电子束光刻技术的成功，集成电路制造领域正在借鉴和学习。IC制备采用EBD的发展路径，制备过程依旧分为放电、增强、聚焦、提升4个环节。其中，放电过程负责将电路材料（通常是玻璃或者橡皮）放射到电子束表面，增强过程利用各种方式提升放射出的电子束的能量，通过改变微晶片内部的微观结构，使它具备良好的工作状态，聚焦过程针对需要制备的电路板，调整焊盘位置，使电流可以在材料中流动顺畅，提升过程就是向晶片中加入特定级别的增压，以降低电子的碎裂和反复。目前，IC制备已经逐渐被多光子光刻所取代，在制备过程中只使用微粒子带通滤波器，并以较低的功率进行提升，获得了比较好的制备效果。

### 3.3.2 可穿戴电子产品制造
智能手机、平板电脑、穿戴电子产品和可穿戴互联网设备等产品，都需要可靠的IC制备。智能手表的制造，除了利用LED光源投射，还有其他方法，例如用EBD来制备微晶片，通过电子束形成良好的膜结构。在制备过程中，需要制备的部分往往直接嵌入到电路板上，而不是外延到材料表面。这就要求IC制备技术要有较高的制作精度。

### 3.3.3 内存芯片制造
内存芯片的制备过程需要经历复杂的工艺流程，如电子束加速、激光加速、引力加速、介质冲洗、晶格化等。目前，制备电路板的工艺已经基本统一，但是制备内存芯片的工艺仍然复杂，需要对现有的工艺流程进行优化。

### 3.3.4 航天器装配电路制造
目前，航天器的制造越来越依赖于集成电路，而集成电路的制备过程仍然需要通过电子束光刻技术来完成。虽然多光子光刻已成为制备航天器电路的主流方法，但是其制备工艺仍有很多不完善之处。

### 3.3.5 生物医疗芯片制造
目前，生物医疗芯片的制备仍然采用电子束光刻技术，因为其技术成熟度较高，可靠性高，制作周期短。由于电子束加速的工艺流程也比较简单，所以制备周期并不是问题。
