# 1.背景介绍


“线宽”（Pitch）又称为“广度”，是指电声或光线传播距离的一半，通常用字母P表示。一般情况下，线宽越窄，则能够反射得相对更少的光，即所谓的“窄带衰减”。
然而在芯片制造中，由于工艺的限制，往往存在着极大的不均匀性。不同器件之间的线宽并不相同，有的器件线宽差距很大，有的器件线宽相当。这就需要相应的技术手段来消除这种不均匀性，保证集成电路整体性能的统一。
通常的做法是：通过调整器件之间的排列关系、优化连接方式等方法，使线宽尽可能的小，且能够充分利用所有信号通道。但这种“微调”的方式也不可避免地会引入较多噪声。另外，不同的器件类型、功耗、带宽要求等因素也会影响到线宽的大小。因此，如何准确、有效地测量并控制线宽，是确保集成电路整体性能的关键。
# 2.核心概念与联系
“线宽分布”(Width Distribution)与“均匀性控制”(Uniformity Control)是我国芯片制造领域两项重要技术领域。为了更好地理解线宽分布与均匀性控制，下面将给出一些概念的定义和联系。
## 2.1 线宽分布
线宽分布是指器件在导通电子信号时占据的导通区域宽度。
一般来说，器件的线宽分布包括如下几种情况:
- 直接分布：器件上只有一种导通宽度，如晶体管或电容；
- 间接分布：器件上导通宽度呈间断分布，如电感，其中某些导通区间比较宽，而另一些导通区间比较窄；
- 分层分布：器件上导通宽度分为多层次结构，例如微扼波线；
- 混合分布：器件上导通宽度呈多种异质混杂状态，如氢化镍芯片、金属耦合硅片。
### 2.1.1 直接分布
在直接分布条件下，器件只有一种导通宽度，即线宽。比如，线宽为320nm的晶体管，或者线宽为1.03mm的金属开关。


图：晶体管和金属开关的直观示意图。

对于直接分布的器件，其线宽主要由以下三个方面决定：

1. 材料特性：通常认为材料的导通率较高时，才能获得足够的线宽。

2. 使用技术：器件的制造方法、使用场合及质量都可能影响到其线宽。例如，有的器件用硫化氢、磷酸、氮化物等物质结晶制造，其导通率高而线宽较窄；有的器件用压敏电阻结晶制造，其导通率低但线宽也较宽。

3. 设计者意图：不同器件之间可以根据设计者的意图进行改进，使线宽符合用户的需求。例如，如果用户希望特定功能实现，可以通过调整器件排列位置、扩散空间来获得更大的线宽；如果用户对性能要求苛刻，可以使用多种材料及制作工艺来获得更好的性能。

在这一类器件中，我们可以看到的例子有液晶显示器、微型计算机、单晶硅片电容、GPS等。

### 2.1.2 间接分布
在间接分布条件下，器件导通宽度呈间断分布，即有的导通区间比较宽，而另一些导通区间比较窄。典型例子是电感。


图：电感器件间接分布的示意图。

对于间接分布的器件，其线宽主要由以下三个方面决定：

1. 设计者意图：不同器件之间也可以根据设计者的意图进行改进，使线宽符合用户的需求。例如，有的器件采用窄轨铁电缆配套双端走线，可以提高线宽；有的器件采用导电介质电缆来制作，可以节省材料费用；还有的器件采用加厚包覆材料来提升抗膨胀能力。

2. 源件特性：不同器件之间的导通宽度取决于它们之间的连接方式。比如，用导线连接的器件，导通宽度受铜丝杆和插头长度的影响；而用跳线连接的器件，导通宽度依赖于跳线电气特性和跳线封装材料的设计。

3. 测量技术：有了一定设计经验的工艺工程师可以依靠特殊测量工具来精确定位导通宽度。

在这一类器件中，我们可以看到的例子有电感器件、水冷器件、锂电池电感器等。

### 2.1.3 分层分布
在分层分布条件下，器件导通宽度被分为若干层，每一层在导通电子信号时占据的导通区域宽度相等。典型例子是微扼波线。


图：微扼波线的分层分布示意图。

对于分层分布的器件，其线宽主要由以下三个方面决定：

1. 材料特性：不同层面的导通宽度依赖于材料属性，如导通率、硬度、导电性、电导率等。

2. 设计者意图：设计者可以根据器件形状、构造、电气特性等特点，使各层之间的导通宽度相等或接近。

3. 测量技术：有了一定设计经验的工艺工程师可以依靠特殊测量工具来精确定位导通宽度。

在这一类器件中，我们可以看到的例子有微扼波线、半导体MOSFET、电力电子单元等。

### 2.1.4 混合分布
在混合分布条件下，器件上导通宽度既有直接分布的部分，也有间接分布的部分，并且每一层的导通宽度也是不均匀的。典型例子是氢化镍芯片。


图：氢化镍芯片的混合分布示意图。

对于混合分布的器件，其线宽主要由以下四个方面决定：

1. 器件结构：器件的结构决定了其线宽的分布。多层结构可以导致多层导通宽度不均匀，以促进导通能力的提高。而单层结构可以使导通宽度逐步趋向于平坦。

2. 源件特性：不同器件之间的导通宽度取决于它们之间的连接方式。比如，通过导线连接的器件，其线宽受铜丝杆、插头长度等参数的影响；而通过跳线连接的器件，其线宽依赖于跳线电气特性、跳线封装材料的设计。

3. 测量技术：有了一定设计经验的工艺工程师可以依靠特殊测量工具来精确定位导通宽度。

4. 连接技术：器件的连接方式影响了导通宽度的分布。有时候，多个器件通过导线连接可以获得更大的线宽，而直接连接就不能获取这种效果。

在这一类器件中，我们可以看到的例子有氢化镍芯片、集成电路IC等。

## 2.2 均匀性控制
均匀性控制是指器件在导通电子信号时，导通宽度应当保持相同或近似相同。通常情况下，均匀性控制的目标是：削弱或减缓不同器件之间的不均衡，使器件集成为整体提供良好的性能。
均匀性控制一般包含两类方法：
- 对称分布：通过控制器件之间的分布对称性，减少或消除不必要的不均衡，使导通宽度的变化趋于一致。
- 均衡设计：通过优化器件设计，提高器件之间的导通能力比例，使导通宽度的变化趋于一致。
### 2.2.1 对称分布
在对称分布条件下，器件导通宽度会在一定范围内对称分布。最基本的方法就是采用同轴控制法。同轴控制法是指让不同器件在同一个方向上移动，直到导通宽度保持一致或相似。

在同轴控制法下，器件之间的排列关系非常重要。比如，在同一排列中，相同的导通器件排在一起，不同的导通器件排在一起，甚至是部分器件排在其他部分之前，都有助于减少不必要的不均衡。再比如，当导线组或器件之间出现环形排列时，就可以实现无限接近对称分布。


图：同轴控制法示意图。

同轴控制法还有一个优点是便宜，不需要复杂的设备，只需简单移动器件即可。但是缺点是施加到器件之间以及导线之间的均匀控制较弱，可能会导致速度过慢或导通难度过高。所以同轴控制法适用于极少数器件，而且只适用于对称分布。

在这一类器件中，我们可以看到的例子有微波炉、空气处理器件、数控机床等。

### 2.2.2 均衡设计
在均衡设计条件下，设计者除了对导通器件的数量和排列关系进行控制外，还可以通过调整器件之间的相互作用，来达到减少或消除不必要的不均衡，使导通宽度的变化趋于一致。

最简单的调整方式是采用“多聚焦”法。多聚焦法是指同时连接多个不同器件，共同产生一定的电流，并将其导通到导线上。这样做的目的是，使每条导线的导通宽度趋于一致，从而避免不同器件之间的不均衡。

多聚焦法可以采用变压器、反相器、结合回路、电容器、耦合器等各种电路元件，以及延迟网络、信号处理器等逻辑元件来实现。不过，多聚焦法还不是通用的解决方案，因为不同器件之间的设计特性、分工等方面可能无法兼顾到。


图：多聚焦法示意图。

相反，采用“单聚焦”法可以解决不同器件之间的不均衡。单聚焦法是指仅用一条导线连接多个不同器件，然后通过调整电流的流动路径，来实现导通宽度的一致。该方法的优点是简单、经济，可以满足大多数应用场景。但缺点是，它只能控制导通宽度，不能改变各器件的导通能力。
