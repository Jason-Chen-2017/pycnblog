# 1.背景介绍


封装是芯片制造过程中不可缺少的一环。它是将电子元器件进行固定在基板上、连接与隔离等功能的集合，有助于提高芯片的可靠性和稳定性。由于物理限制，芯片的封装材质一般都是硅玻璃或其他高性能表面材料。但由于硅损坏率较低、分辨率高、结构复杂，在电路板上封装晶圆型材质仍然占主导地位。因此，封装材料成为各类主流芯片制造商争相追逐的目标。
通常，芯片封装的主要技术有两种：胶合层法和镀锌法。两种方法都依赖于锚定层的加工及其技术标准。而两种技术又可以分为两种类型：横向和纵向。横向的封装主要采用金属层；而纵向则主要基于半导体材料和电介质层（电极）的组合。
芯片制造商也对各种封装材料进行了比较研究，选取最佳封装材料至关重要。下面就来看一下，什么样的封装材料能够实现更高的集成度、导通性和耐腐蚀能力？为什么要选择这些封装材料？
# 2.核心概念与联系
## 2.1.基板
在封闭的设计世界里，基板是连接所有器件的地方，所有的物理连接都会通过基板的接口（引脚）完成。一个完整的电路板往往由多个基板组成，每一个基板中都包含若干组件，如电阻、电容、管脚等，通过这些组件，信号就可以从输入端流动到输出端，产生对应的电压和电流。
## 2.2.封装层
封装层就是用来固定基板上的器件。基板上所有的器件（包括电源、电阻、电容、OPAmp等等）都会被固定在封装层上面，然后才能把电路装到基板上。封装层是在工艺上为了减少由于电子材料烧焊导致的静态失效、静电粒子侵入导致的动态失效、反射干扰等问题所采用的防护层。
主要封装层有金属层和半导体层两种。其中，金属层具有硬化、隔离作用，对于短期内的、局部的静电干扰效果尤为明显；而半导体层则具有广泛的导通性、耐腐蚀性、抗紫外线等特性，是实现高集成度、高导通性和高抗腐蚀性的保障。
## 2.3.电极
电极是半导体与电介质之间连接的一层。在电极之间的任何电信号都会被捕获、处理后传递到另一端。电极还负责隔离、保护和保护电子元器件免受环境影响的干扰。各种材料都可以作为电极，如硅、陶瓷、金属等，它们的类型、厚度、材料比例和位置都不一样，满足不同客户的需求。常见的电极包括：微芯片电极（Microchip Leads）、卡西欧电极（Cessna Leads）、高斯电极（Gauss Leads）等。
## 2.4.基材与填充层
基材是用于构筑封装层的材料，在极化条件下，基材可以保证导电性、导通性和耐磨性。除此之外，基材还起到连接封装层的作用。填充层是由绝缘材料包裹的铜垢或其他沉积物，也是为了保护封装层和电极免受潮湿或其他外部刺激带来的损害。
## 2.5.膜与套件
膜是一种用于固定封装材料到基板上的设备，在厚度上可以达到几十公分以上。膜中的氧化物能够将封装材料固定在基板上，并帮助保持基板的完好状态。而套件是由不同型号、大小、性能等级的膜和其它配套零件组成的设备，能够提供更高的集成度和耐用性。
## 2.6.测试电流
测试电流是一个特殊的电流，只用于测试电路的功能。它一般用于产生刺激信号，例如电感量程的10%。测试电流也可以用来测量电气属性和场效应的参数，例如谐波功率。
# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1.胶合层法
胶合层法，即将封装材料层层覆盖在基板上，用单层或多层半导体贴膜的方法进行封装。这里涉及到的技术难点主要有：

1. 基板密度不足、缺乏连接性，导致无法形成密堆集电。
2. 没有考虑导通性、导电性，导致信号传输不良。
3. 单层贴膜容易发生大面积坍塌或断裂，造成大面积开口。
4. 导电距大于导热距离，导通性不足，无法实现高效率。

胶合层法的基本原理是：将封装材料覆盖在基板上，从基板边缘向里移动，利用底层的贴膜的导通性对齐和固定基板上的元器件，并避免设计孔洞。胶合层法的具体操作步骤如下：

1. 首先，在较大范围内清除基板的杂质，然后用刮片机擦去基板上易燃物质、沙砾等异物。
2. 在基板边缘施加较大的碰撞力，使基板的表面散射干扰流线。
3. 将贴膜固定在基板上，贴上贴膜后，在贴膜表面采用各种硼片和防水膜材料使贴膜完全嵌入基板。
4. 把绝缘层封装在基板上，覆上全薄金属表面及贴膜，直到基板表面达到一定厚度。
5. 按顺序层层堆叠不同厚度的硅、钢材料、绝缘材料及贴膜，层层嵌入，形成高导通性的封装材料。
6. 用电泵反复吸收和输送电子元件，使基板上所有元件的导电力均匀分布，实现封装的导通和导电功能。
7. 使用热噪声电流测试电路是否存在漏洞。
8. 对铣削过的外壳进行干燥，确保金属层的导通性，便于温度变化时快速恢复。

## 3.2.镀锌法
镀锌法，即利用多种元件（如硅片、导电片、微粒子、翘曲层及透明层等）在基板上制作贴锌体，从而保证导通性、导电性和韧性。这里涉及到的技术难点主要有：

1. 导电材料的绝缘性不足，容易反复伸缩导致基板破坏。
2. 贴锌层不能准确控制导电间隙，致使导电片的滞留、聚焦和衬托效应。
3. 需要消耗更多的工作量和资源，不利于实施批量生产。

镀锌法的基本原理是：在基板上设置多种元件，利用它们的贴合、导通、配平等功能对封装材料进行预处理，形成导通性好的锌化层。镀锌法的具体操作步骤如下：

1. 首先，用刮片机擦去基板上易燃物质、沙砾等异物，确保基板的外观整洁、无凹槽等瑕疵。
2. 铣削外壳，熨烫金属，稀释出足够厚度的羽化层，等待接下来的镀锌过程。
3. 在基板四周设置多种元件，如微粒子、导电片、翘曲层等，它们可以在不同的方向上贴合在基板上，并且可以固定导电片或微粒子使导电片进入基板，进而导电。
4. 选择适当厚度的锌化材料，如硅基锌、陶瓷锌等，在金属表面制造柱状排列的锌化层。
5. 根据所需的封装材料大小、厚度和材料比例，在金属表面设置不同厚度的锌化层，做到高度导通、平滑处理。
6. 用机械或电动工具加速锌化层堆积，使锌化层密集地堆积在基板上。
7. 通过接触试验、冲击试验、发光试验、电动测试等验证封装的导通性、导电性、韧性及反应性能。

## 3.3.横向封装材料
对于横向封装材料，主要有金属丝芯（Graphite）、陶瓷丝芯（Ceramic）和光刻胶。
### 3.3.1.金属丝芯(Graphite)
金属丝芯是一种横向封装材料，用金属芯片固定在基板上，中间层薄如膜状的金属丝。这种金属丝又称为银芯，可以长久存在、韧性强、耐腐蚀性强。但是，银芯材质较脆、成本较高，且反应慢，不适宜在硅片和其他高能材料上联。它的特点是：

1. 制造简单、成本低廉、性能优越、不易腐蚀。
2. 横向导通性好、导电性强、可靠性高。
3. 操作简单、实现高密度封装。

其制作方式为：

1. 用低温冷却金属芯片。
2. 加热至金属表面的顶点温度，使金属包裹在一起，将金属片条状的路径形成。
3. 采用多层裂缝组织构成金属丝芯。
4. 按要求设置导电层、连接层及温馨层。

其优势是：

1. 制造简单、成本低廉、性能优越、不易腐蚀。
2. 横向导通性好、导电性强、可靠性高。
3. 操作简单、实现高密度封装。
4. 集成度高、导通性好。

### 3.3.2.陶瓷丝芯(Ceramic)
陶瓷丝芯也叫瓷丝芯，是一种用于封装的软化金属材料，是一种轻巧、坚固、导通性强的化学制品。它的优点是：

1. 导电性强、可靠性高。
2. 丝芯上施加小范围的反射光照射，可防止内部膨胀。
3. 操作简单、工作速度快、外观漂亮。

其制作方式为：

1. 用银箔夹成固体卡盘，然后浸泡在低温下，表面呈现出亚金属化合物的晶莹剔透状态。
2. 用蒸馏法熔化，液化，熔化后的陶瓷芯变成平滑的金属网格状。
3. 再次用低温冷却，保持固化，一般需要几天时间。
4. 用预裂钳固定金属丝芯和导电层。

它的特点是：

1. 制造简单、成本低廉、性能优越、不易腐蚀。
2. 横向导通性好、导电性强、可靠性高。
3. 操作简单、实现高密度封装。
4. 集成度高、导通性好。

### 3.3.3.光刻胶(Electroplating)
光刻胶是一种横向封装材料，用于将导电材料镀锌化后固定在基板上。它的特点是：

1. 不含硅，不容易腐蚀。
2. 可靠性好，导通性好。
3. 制造及反应速度快。

其制作方式为：

1. 在导电层表面涂有锂离子或碘离子，或乙炔。
2. 用电动机加速施放光刻二氧化硅激发物对导电层进行反应，使导电层上锌化物和原有的导电材料发生接触。
3. 对导电层进行冷却、锌化，使导电层结构塑形，形成密集结晶。
4. 用接触测试、冲击测试、发光测试、热力测试等检测封装材料的导电性、导通性、反应性能。

它的应用场景：

1. 集成度要求高，适用高性能集成电路。
2. 能够提供长期耐用和低价廉价。

## 3.4.纵向封装材料
纵向封装材料主要有包覆层(Filling Layer)和电极(Leads)。
### 3.4.1.包覆层(Filling Layer)
包覆层(Filling Layer)是用于构建封装层的材料。主要有金属丝芯(graphite)，陶瓷丝芯(ceramic)，以及其它高分子材料(organic)。它是固定元器件和连接器件的坚固层，并防止生物作用。其制作方法有烘焊或烧结。
### 3.4.2.电极(Leads)
电极(Lead)是一种介质，位于基板的任意一侧，并以导电性材料为其材料，用于连接基板上导电元件之间的互联，是构建半导体基板结构、缓解电源失压等功能的必要元素。其制作方法有摆布或成型。
