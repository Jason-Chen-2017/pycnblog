# 2.核心概念与联系

## 2.1 L1正则化

L1正则化是一种常用的正则化方法，它在最小化损失函数的同时，通过引入L1正则项来约束模型的复杂度，从而避免过拟合。L1正则化的目标函数可以表示为：

$$
J(\theta) = \frac{1}{2n} \sum_{i=1}^n (h_\theta(x_i) - y_i)^2 + \lambda \sum_{j=1}^m |w_j|
$$

其中，$h_\theta(x_i)$ 是模型的预测值，$y_i$ 是真实值，$x_i$ 是输入特征，$w_j$ 是模型中的权重参数，$\lambda$ 是正则化参数，$n$ 是训练样本的数量，$m$ 是模型中权重参数的数量。

L1正则化的优点包括：

1. 稀疏性：L1正则化可以将一些权重参数设置为0，从而实现模型的稀疏化。
2. 鲁棒性：L1正则化可以减小模型对输入数据的敏感性，从而提高模型的鲁棒性。
3. 简化模型：L1正则化可以简化模型，从而减少模型的复杂性和计算成本。

# 模型构建
def l1_regularization(theta, X, y, lambda_):
    m = len(y)
    theta_pred = X.dot(theta)
    J = (1 / (2 * m)) * np.sum((theta_pred - y) ** 2) + lambda_ * np.sum(np.abs(theta))
    return J

# 梯度下降求解
def gradient_descent(theta, X, y, lambda_, alpha, iterations):
    m = len(y)
    J_history = np.zeros((iterations, 1))
    for i in range(iterations):
        theta_pred = X.dot(theta)
        gradients = (1 / m) * (2 * X.transpose().dot(theta_pred - y)) + lambda_ * np.sign(theta)
        theta = theta - alpha * gradients
        J_history[i] = l1_regularization(theta, X, y, lambda_)
    return theta, J_history

iterations = 1000

# 模型训练

# 模型评估
