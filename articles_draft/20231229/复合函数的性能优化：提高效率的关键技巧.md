$$

$$

$$
F(x) = \begin{cases}
x, & \text{if } x \leq 0 \\
F(x-1) + f(g(h(x))), & \text{otherwise}
\end{cases}
$$

```python
    return x * x

def g(x):

    else:

```

### 4.2.2 解释说明

在这个代码实例中，我们将三个基本函数 f(x)、g(x) 和 h(x) 按照递归方式组合成一个新的函数 F(x)。然后，我们使用一个输入 x=5 来计算 F(x) 的值。

## 4.3 并行组合

### 4.3.1 代码实例

```python
import threading

def f(x):
    return x * x

def g(x):
    return x + 1

def h(x):
    return x * x

def F(x):
    t1 = threading.Thread(target=f, args=(x,))
    t2 = threading.Thread(target=g, args=(x,))
    t3 = threading.Thread(target=h, args=(x,))
    t1.start()
    t2.start()
    t3.start()
    t1.join()
    t2.join()
    t3.join()
    return t1.result() + t2.result() + t3.result()

x = 5
print(F(x))
```

### 4.3.2 解释说明

在这个代码实例中，我们将三个基本函数 f(x)、g(x) 和 h(x) 按照并行方式组合成一个新的函数 F(x)。然后，我们使用一个输入 x=5 来计算 F(x) 的值。
