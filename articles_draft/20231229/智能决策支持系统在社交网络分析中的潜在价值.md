## 3.3 数学模型公式详细讲解
在社交网络分析中，常用的数学模型公式包括：

1. 度（Degree）：节点的度为与其相连的其他节点的数量，公式为：
$$
D(v) = |\{u \in V | (v,u) \in E \}|
$$
2.  Betweenness Centrality（Betweenness）：节点的 Betweenness 为它在所有短路径中的数量，公式为：
$$
BC(v) = \sum_{s \neq v \neq t} \frac{\sigma_{st}(v)}{\sigma_{st}}
$$
其中，$\sigma_{st}$ 为从节点 $s$ 到节点 $t$ 的所有短路径数量，$\sigma_{st}(v)$ 为经过节点 $v$ 的短路径数量。

3.  closeness Centrality（Closeness）：节点的 Closeness 为它到其他所有节点的平均距离的逆数，公式为：
$$
CC(v) = \frac{n-1}{\sum_{u \neq v} d(u,v)}
$$
其中，$n$ 为节点数量，$d(u,v)$ 为节点 $u$ 到节点 $v$ 的距离。

4.  PageRank：节点的 PageRank 为它的入度和出度的权重平均值，公式为：
$$
PR(v) = (1-d) + d \sum_{u \in Adj(v)} \frac{PR(u)}{L(u)}
$$
其中，$d$ 为漫步概率，$Adj(v)$ 为节点 $v$ 的邻接节点集合，$L(u)$ 为节点 $u$ 的入度。

在本节中，我们将通过一个简单的例子来展示智能决策支持系统在社交网络分析中的应用。

## 4.1 示例：Twitter数据分析
### 4.1.1 数据收集和预处理
我们可以通过Twitter API获取Twitter数据，如用户关注关系、微博内容等。首先，我们需要注册Twitter Developer账户，并获取API访问权限。然后，我们可以使用Python的tweepy库来获取Twitter数据：
```python
import tweepy

# 设置API密钥和访问密钥
consumer_key = 'your_consumer_key'
consumer_secret = 'your_consumer_secret'
access_token = 'your_access_token'
access_token_secret = 'your_access_token_secret'

# 设置API对象
auth = tweepy.OAuthHandler(consumer_key, consumer_secret)
auth.set_access_token(access_token, access_token_secret)

# 创建API对象
api = tweepy.API(auth)

# 获取用户关注关系
followers = api.followers(screen_name='your_screen_name')

# 获取用户关注关系
friends = api.friends(screen_name='your_screen_name')
```
### 4.1.2 构建社交网络图
我们可以使用NetworkX库来构建社交网络图。首先，我们需要将Twitter数据转换为图的节点和边：
```python
import networkx as nx

# 创建空图
G = nx.Graph()

# 添加节点
for user in followers + friends:
    G.add_node(user.screen_name)

# 添加边
for follower in followers:
    G.add_edge(f'your_screen_name', follower.screen_name)

# 绘制图
nx.draw(G, with_labels=True)
```
### 4.1.3 社交网络分析
我们可以使用NetworkX库来分析社交网络的拓扑特征，如度中心性、 Betweenness Centrality、 closeness Centrality等。例如，我们可以计算节点的度：
```python
# 计算节点度
degrees = nx.degree(G)

# 绘制度分布
plt.hist(degrees.values(), bins=10)
plt.xlabel('Degree')
plt.ylabel('Frequency')
plt.show()
```
