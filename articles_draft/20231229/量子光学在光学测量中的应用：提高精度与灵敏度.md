# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释量子光学在光学测量中的应用。

代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt
from qiskit import QuantumCircuit, Aer, transpile, assemble

# 创建量子电路
qc = QuantumCircuit(2, 2)

# 初始化量子态
qc.initialize([1, 0], 0)
qc.initialize([0, 1], 1)

# 实现量子门
qc.cx(0, 1)

# 将量子电路编译成可执行的量子电路
qc_executable = transpile(qc, Aer.get_backend('qasm_simulator'))

# 执行量子电路
result = assemble(qc_executable).run().result()
counts = result.get_counts()

# 绘制结果
plt.bar(qc_executable.name, counts.values())
plt.xlabel('Quantum circuit')
plt.ylabel('Counts')
plt.show()
```

代码解释：

1. 导入所需的库：`numpy`、`matplotlib.pyplot` 和 `qiskit`。
2. 创建一个量子电路 `qc`，输入两个量子比特。
3. 初始化量子态：将第一个量子比特的状态设为 `[1, 0]`，第二个量子比特的状态设为 `[0, 1]`。
4. 实现量子门：将第一个量子比特与第二个量子比特之间的控制位进行 XOR 运算。
5. 将量子电路编译成可执行的量子电路 `qc_executable`。
6. 执行量子电路并获取结果。
7. 绘制结果。
