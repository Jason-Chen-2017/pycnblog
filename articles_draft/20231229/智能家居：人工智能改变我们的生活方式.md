### 4.1 数据收集代码实例

以下是一个简单的温度、湿度、空气质量的数据收集代码实例：

```python
import time
import Adafruit_DHT

# 设置传感器类型和数据收集间隔
sensor = Adafruit_DHT.DHT22
interval = 5

while True:
    # 数据收集
    humidity, temperature = Adafruit_DHT.read_retry(sensor)
    if humidity is not None and temperature is not None:
        print("温度: {:.1f}℃, 湿度: {:.1f}%".format(temperature, humidity))
    else:
        print("无法收集数据")

    # 数据处理
    temperature = temperature - 23
    humidity = humidity * 0.8

    # 数据存储
    with open("data.csv", "a") as f:
        f.write("{:.1f},{:.1f}\n".format(temperature, humidity))

    # 数据发送
    import requests
    requests.post("http://your_server_url/data", json={"temperature": temperature, "humidity": humidity})

    # 休眠
    time.sleep(interval)
```
