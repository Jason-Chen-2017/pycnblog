在进行数字化转型时，需要掌握以下几个核心数学模型公式：

1. 线性回归模型：线性回归模型是一种常用的数字化模型，用于预测因变量的值。线性回归模型的数学模型公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是因变量，$x_1, x_2, \cdots, x_n$ 是自变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。

2. 逻辑回归模型：逻辑回归模型是一种常用的数字化模型，用于预测二值变量的值。逻辑回归模型的数学模型公式为：

$$
P(y=1|x_1, x_2, \cdots, x_n) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \cdots - \beta_nx_n}}
$$

其中，$P(y=1|x_1, x_2, \cdots, x_n)$ 是预测概率，$x_1, x_2, \cdots, x_n$ 是自变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

3. 决策树模型：决策树模型是一种常用的数字化模型，用于预测类别变量的值。决策树模型的数学模型公式为：

$$
\text{if } x_1 \text{ is } A_1 \text{ then } y = B_1 \\
\text{else if } x_2 \text{ is } A_2 \text{ then } y = B_2 \\
\cdots \\
\text{else if } x_n \text{ is } A_n \text{ then } y = B_n
$$

其中，$x_1, x_2, \cdots, x_n$ 是自变量，$A_1, A_2, \cdots, A_n$ 是条件变量，$B_1, B_2, \cdots, B_n$ 是预测值。

```python
# 读取原始数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.drop_duplicates()
data = data.fillna(0)
data = data.replace(to_replace='error', value=0)

# 数据转换
data['age'] = data['age'].astype(int)
data['gender'] = data['gender'].astype(str)

# 数据存储
data.to_csv('cleaned_data.csv', index=False)
```
