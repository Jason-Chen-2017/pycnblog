## 4.1 随机生成算法实例

### 4.1.1 生成随机地图

我们可以使用以下代码来生成一个随机地图：

```python
import random

def generate_map(width, height):
    map = [[0 for _ in range(width)] for _ in range(height)]
    for x in range(width):
        for y in range(height):
            if random.random() < 0.3:
                map[x][y] = 1
    return map
```

这段代码首先导入了 `random` 模块，然后定义了一个 `generate_map` 函数，该函数接受 `width` 和 `height` 作为参数，并返回一个随机生成的地图。在生成地图的过程中，我们使用了随机数生成器（Random Number Generator，RNG）来生成地图的每个格子的值。

### 4.1.2 生成随机角色

我们可以使用以下代码来生成一个随机角色：

```python
import random

def generate_character(attributes):
    character = {}
    for attribute, values in attributes.items():
        character[attribute] = random.choice(values)
    return character
```

这段代码首先导入了 `random` 模块，然后定义了一个 `generate_character` 函数，该函数接受 `attributes` 作为参数，并返回一个随机生成的角色。在生成角色的过程中，我们使用了随机数生成器（Random Number Generator，RNG）来生成角色的每个属性的值。
