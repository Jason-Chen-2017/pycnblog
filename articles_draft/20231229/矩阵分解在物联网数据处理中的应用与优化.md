## 3.3 数学模型公式详细讲解

### 3.3.1 矩阵分解的数学模型

矩阵分解的数学模型可以表示为：

$$
\mathbf{X} = \mathbf{A} \mathbf{B}^T + \mathbf{E}
$$

其中，$\mathbf{X}$ 是原始矩阵，$\mathbf{A}$ 和 $\mathbf{B}$ 是分解后的矩阵，$\mathbf{E}$ 是误差矩阵。

### 3.3.2 非负矩阵分解的数学模型

非负矩阵分解的数学模型可以表示为：

$$
\mathbf{X} = \mathbf{A} \mathbf{B}^T
$$

其中，$\mathbf{A}$ 和 $\mathbf{B}$ 是非负矩阵，$\mathbf{X}$ 是原始矩阵。

### 3.3.3 正则化的数学模型

正则化的数学模型可以表示为：

$$
\min_{\mathbf{A}, \mathbf{B}} \frac{1}{2} ||\mathbf{X} - \mathbf{A} \mathbf{B}^T||_F^2 + \lambda \left( ||\mathbf{A}||_F^2 + ||\mathbf{B}||_F^2 \right)
$$

其中，$\lambda$ 是正则化参数，$||\cdot||_F$ 是矩阵的范数。

### 3.3.4 随机梯度下降的数学模型

随机梯度下降的数学模型可以表示为：

$$
\mathbf{A}^{(t+1)} = \mathbf{A}^{(t)} - \eta \frac{\partial L}{\partial \mathbf{A}}
$$

$$
\mathbf{B}^{(t+1)} = \mathbf{B}^{(t)} - \eta \frac{\partial L}{\partial \mathbf{B}}
$$

其中，$L$ 是损失函数，$\eta$ 是学习率。

## 4.1 非负矩阵分解的Python实现

```python
import numpy as np
from scipy.sparse.linalg import svds

def non_negative_matrix_factorization(X, rank, max_iter=100, tol=1e-6):
    U, s, Vt = svds(X, k=rank)
    U = np.maximum(U, 0)
    Vt = np.maximum(Vt, 0)
    for i in range(max_iter):
        W = U @ Vt
        diff = X - W
        U, s, Vt = svds(diff, k=rank)
        U = np.maximum(U, 0)
        Vt = np.maximum(Vt, 0)
        if np.linalg.norm(X - U @ Vt) < tol:
            break
    return U, Vt

X = np.random.rand(100, 100)
rank = 10
U, Vt = non_negative_matrix_factorization(X, rank)
```

```python
import numpy as np
from scipy.sparse.linalg import svds

def regularized_non_negative_matrix_factorization(X, rank, lambda_param, max_iter=100, tol=1e-6):
    U, s, Vt = svds(X, k=rank)
    U = np.maximum(U, 0)
    Vt = np.maximum(Vt, 0)
    for i in range(max_iter):
        W = U @ Vt
        diff = X - W
        U, s, Vt = svds(diff + lambda_param * (U @ U.T + Vt @ Vt), k=rank)
        U = np.maximum(U, 0)
        Vt = np.maximum(Vt, 0)
        if np.linalg.norm(X - U @ Vt) < tol:
            break
    return U, Vt

X = np.random.rand(100, 100)
rank = 10
lambda_param = 0.1
U, Vt = regularized_non_negative_matrix_factorization(X, rank, lambda_param)
```
