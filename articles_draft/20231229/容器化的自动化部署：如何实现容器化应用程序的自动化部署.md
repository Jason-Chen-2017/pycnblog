## 4.1 代码实例

我们将通过一个简单的Python应用程序来演示如何实现容器化应用程序的自动化部署。我们将使用Docker来构建容器镜像，并将其推送到Docker Hub上。然后，我们将从Docker Hub上拉取容器镜像，并运行容器。

首先，我们需要创建一个Dockerfile文件，用于描述容器镜像的构建过程：

```
FROM python:3.7

WORKDIR /app

COPY requirements.txt .

RUN pip install -r requirements.txt

COPY . .

CMD ["python", "app.py"]
```

接下来，我们需要创建一个requirements.txt文件，用于描述应用程序的依赖项：

```
Flask==1.0.2
```

接下来，我们需要创建一个app.py文件，用于描述应用程序的运行过程：

```
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=80)
```

接下来，我们需要使用Docker命令来构建容器镜像：

```
docker build -t my-app .
```

接下来，我们需要使用Docker命令来推送容器镜像到Docker Hub上：

```
docker push my-app
```

接下来，我们需要使用Docker命令来从Docker Hub上拉取容器镜像：

```
docker pull my-app
```

接下来，我们需要使用Docker命令来运行容器：

```
docker run -p 80:80 my-app
```
