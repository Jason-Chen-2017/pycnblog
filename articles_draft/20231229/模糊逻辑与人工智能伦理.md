模糊逻辑有一些基本规则，这些规则可以用来处理模糊关系和模糊逻辑表达式。

### 规则1（交换律）

给定模糊和、模糊或、模糊差和模糊积运算符A、B和C，有：

- A $\oplus$ B = B $\oplus$ A

### 规则2（结合律）

给定模糊和、模糊或、模糊差和模糊积运算符A、B和C，有：

- (A $\oplus$ B) $\oplus$ C = A $\oplus$ (B $\oplus$ C)

- (A $\ominus$ B) $\ominus$ C = A $\ominus$ (B $\ominus$ C)

- A $\ominus$ (B $\cup$ C) = (A $\ominus$ B) $\cup$ (A $\ominus$ C)

# 定义模糊集合
def define_fuzzy_set(name, values):
    return {label: value for label, value in zip(name, values)}

# 模糊和
def fuzzy_and(a, b):
    return {label: min(value_a, value_b) for label, (value_a, value_b) in zip(a.keys(), b.items())}

# 模糊或
def fuzzy_or(a, b):
    return {label: max(value_a, value_b) for label, (value_a, value_b) in zip(a.keys(), b.items())}

# 模糊差
def fuzzy_difference(a, b):
    return {label: max(value_a - value_b, 0) for label, (value_a, value_b) in zip(a.keys(), b.items())}

# 模糊积
def fuzzy_product(a, b):
    return {label: min(value_a * value_b, 1) for label, (value_a, value_b) in zip(a.keys(), b.items())}

# 定义模糊集合
A = define_fuzzy_set("A", [0, 0.5, 1])
B = define_fuzzy_set("B", [0, 0.5, 1])

# 计算模糊和
C_and = fuzzy_and(A, B)
print("模糊和：", C_and)

# 计算模糊或
C_or = fuzzy_or(A, B)
print("模糊或：", C_or)

# 计算模糊差
C_difference = fuzzy_difference(A, B)
print("模糊差：", C_difference)

# 计算模糊积
C_product = fuzzy_product(A, B)
print("模糊积：", C_product)
