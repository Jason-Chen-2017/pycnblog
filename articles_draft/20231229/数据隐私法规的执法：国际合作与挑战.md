### 4.1 数据加密

数据加密是一种通过加密算法将个人信息加密为不可读形式的技术。在这里，我们将通过一个简单的AES加密算法实例来说明数据加密的过程。

```python
from Crypto.Cipher import AES

# 定义密钥和明文
key = b'abcdefgh'
plaintext = b'Hello, World!'

# 创建AES加密对象
cipher = AES.new(key, AES.MODE_ECB)

# 对明文进行加密
ciphertext = cipher.encrypt(plaintext)

print('加密后的密文：', ciphertext)
```

在这个例子中，我们使用了AES加密算法对明文 'Hello, World!' 进行了加密。加密后的密文是一个字节序列，不能直接读懂明文的内容。


在这个例子中，我们使用了混淆算法对明文 'Hello, World!' 和随机数 '1234567890' 进行了异或运算，得到了掩码后的密文。掩码后的密文不能直接读懂明文的内容，但是可以通过知道随机数来还原明文。

数据擦除是一种通过将数据从存储设备上完全删除的方式来保护数据安全和隐私的技术。在这里，我们将通过一个简单的数据擦除实例来说明数据擦除的过程。
