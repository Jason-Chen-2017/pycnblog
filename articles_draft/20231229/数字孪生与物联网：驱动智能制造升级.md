### 4.2 物联网代码实例

在这个代码实例中，我们将通过Python编程语言实现一个简单的物联网系统，包括设备的连接和通信、数据的收集和处理以及平台的构建和管理。

```python
# 设备的连接和通信
class Device:
    def __init__(self, id, name):
        self.id = id
        self.name = name

# 数据的收集和处理
class DataCollector:
    def __init__(self, devices):
        self.devices = devices

    def collect_data(self):
        data = []
        for device in self.devices:
            data.append({'id': device.id, 'name': device.name})
        return data

# 平台的构建和管理
class Platform:
    def __init__(self, data_collector):
        self.data_collector = data_collector

    def manage_data(self):
        data = self.data_collector.collect_data()
        print(data)

# 测试
device1 = Device(1, 'Sensor1')
device2 = Device(2, 'Sensor2')
data_collector = DataCollector([device1, device2])
platform = Platform(data_collector)
platform.manage_data()
```
