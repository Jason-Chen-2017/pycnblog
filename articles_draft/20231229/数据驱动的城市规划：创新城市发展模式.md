# 3.3 数学模型公式详细讲解
数据驱动的城市规划的数学模型公式主要包括以下几种：

- **线性回归模型**：线性回归模型用于预测城市数据的变化趋势。它的基本公式是：
$$
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_n x_n + \epsilon
$$
其中，$y$ 是预测变量，$x_1, x_2, \cdots, x_n$ 是预测因素，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。
- **逻辑回归模型**：逻辑回归模型用于预测二值性质的城市数据。它的基本公式是：
$$
P(y=1|x_1, x_2, \cdots, x_n) = \frac{1}{1 + e^{-\beta_0 - \beta_1 x_1 - \beta_2 x_2 - \cdots - \beta_n x_n}}
$$
其中，$P(y=1|x_1, x_2, \cdots, x_n)$ 是预测概率，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。
- **决策树模型**：决策树模型用于分类和预测城市数据。它的基本公式是：
$$
\arg \max_{c} P(c|x_1, x_2, \cdots, x_n)
$$
其中，$c$ 是类别，$P(c|x_1, x_2, \cdots, x_n)$ 是条件概率。
