# 4.具体代码实例和详细解释说明

## 4.1列式存储的代码实例
以下是一个简单的列式存储代码实例：

```python
import pandas as pd

# 创建表结构
df = pd.DataFrame({
    'id': [1, 2, 3, 4, 5],
    'name': ['Alice', 'Bob', 'Charlie', 'David', 'Eve'],
    'age': [25, 30, 35, 40, 45],
    'gender': ['F', 'M', 'M', 'M', 'F']
})

# 插入数据
df.to_csv('data.csv', index=False)

# 创建索引
df = pd.read_csv('data.csv')
df['age'] = df['age'].astype(int)
df['gender'] = df['gender'].astype('category')
df.set_index('id', inplace=True)

# 执行查询
query = 'gender == "M"'
result = df.query(query)
print(result)
```

## 4.2多维数据处理的代码实例
以下是一个简单的多维数据处理代码实例：

```python
import pandas as pd

# 创建数据模型
data = {
    'dim_date': ['2021-01-01', '2021-01-02', '2021-01-03'],
    'fact_sales': [100, 200, 300]
}

# 插入数据
df = pd.DataFrame(data)
df.set_index('dim_date', inplace=True)

# 创建计算和聚合规则
def calculate_sales(row):
    return row['fact_sales']

# 执行查询
query = 'dim_date == "2021-01-01"'
result = df.query(query).apply(calculate_sales, axis=1)
print(result)
```
