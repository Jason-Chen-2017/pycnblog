假设我们有一个股票价格数据序列，我们想要对这个数据序列进行描述性分析，以便更好地理解其中心趋势和离散程度。我们可以使用 Python 语言编写如下代码：

```python
import numpy as np
import pandas as pd

# 加载股票价格数据
stock_price_data = pd.read_csv('stock_price.csv')

# 计算平均值
average_price = stock_price_data['price'].mean()

# 计算中位数
median_price = stock_price_data['price'].median()

# 计算方差
variance_price = stock_price_data['price'].var()

# 计算标准差
std_dev_price = stock_price_data['price'].std()

# 打印结果
print('平均股票价格:', average_price)
print('中位数股票价格:', median_price)
print('方差股票价格:', variance_price)
print('标准差股票价格:', std_dev_price)
```
