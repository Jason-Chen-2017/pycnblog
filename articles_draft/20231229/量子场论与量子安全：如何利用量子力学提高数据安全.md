## 3.1量子密钥交换：BB84协议
BB84协议是一种量子密钥交换协议，它利用量子通信的特性来实现安全的密钥交换。BB84协议的具体操作步骤如下：

1.  Alice将随机选择一组线性无关的单位向量，将每个向量表示为一个量子位（qubit），并将这些量子位发送给Bob。
2.  Bob将每个量子位以两种基础状态进行测量，即正向基础状态和反向基础状态。
3.  Alice和Bob在通信过程中约定好一个基础状态，例如正向基础状态。Alice将每个向量的测量结果以二进制形式发送给Bob。
4.  Bob根据Alice发送的测量结果和基础状态选择出一组相同的向量，这些向量组成了共享的密钥。

BB84协议的数学模型公式如下：

$$
|0\rangle = \frac{1}{\sqrt{2}}(|H\rangle +|V\rangle ) \\
|1\rangle = \frac{1}{\sqrt{2}}(|H\rangle -|V\rangle )
$$

其中，|0\rangle 和|1\rangle 是量子位的基础状态，|H\rangle 和|V\rangle 是线性无关的单位向量。

量子门模型是一种量子加密技术，它利用量子位（qubit）的特性来实现安全的数据加密。量子门模型的具体操作步骤如下：

3.  Bob将量子位应用于量子门进行加密。
4.  Bob将加密后的量子位发回给Alice。
5.  Alice将加密后的量子位解码为原始数据。

```python
import random
import numpy as np

def generate_random_basis():
    return random.choice(['Z', 'X'])

def bb84_protocol(n):
    Alice = QuantumCircuit(n)
    Bob = QuantumCircuit(n)

    # Alice generates n random basis and n qubits
    basis = [generate_random_basis() for _ in range(n)]
    qubits = [Control('c' + str(i)) for i in range(n)]

    # Alice prepares qubits in |+⟩ state
    Alice.h(qubits)

    # Alice sends qubits to Bob
    Alice.cz(qubits[0], Bob.qubits[0])
    for i in range(1, n):
        Alice.cz(qubits[i], Bob.qubits[i])

    # Bob measures qubits in random basis
    for i, basis in enumerate(basis):
        if basis == 'Z':
            Bob.z(Bob.qubits[i])
        else:
            Bob.x(Bob.qubits[i])

    # Alice and Bob share classical information
    key = [0] * n
    for i, basis in enumerate(basis):
        if basis == 'Z' and Bob.get_measurement_result(Bob.qubits[i]) == 0:
            key[i] = 1

    return key
```

```python
import numpy as np

def quantum_gate(state, gate):
    if gate == 'H':
        return np.array([[1/np.sqrt(2), 1/np.sqrt(2)],
                         [1/np.sqrt(2), -1/np.sqrt(2)]]) @ state
    elif gate == 'X':
        return np.array([[0, 1],
                         [1, 0]]) @ state
    elif gate == 'Y':
        return np.array([[0, -1j],
                         [1j, 0]]) @ state
    elif gate == 'Z':
        return np.array([[1, 0],
                         [0, -1]]) @ state
    else:
        raise ValueError('Invalid gate')

def quantum_encryption(state, key):
    encrypted_state = np.kron(np.eye(2**len(key)), quantum_gate('Z', key)) @ state
    return encrypted_state

def quantum_decryption(state, key):
    decrypted_state = np.kron(np.eye(2**len(key)), quantum_gate('Z', key).conj().T) @ state
    return decrypted_state
```
