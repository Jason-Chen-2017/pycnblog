## 4.1 代码实例

```python
import numpy as np
import random

def fitness_function(x):
    # 目标函数，例如最小化x的平方和
    return np.sum(x**2)

def update_velocity(v, p, c1, c2, r1, r2):
    return c1 * r1 * (p - v) + c2 * r2 * (p - v)

def update_position(x, v):
    return x + v

def update_best_position(x, best_x):
    return x if fitness_function(x) < fitness_function(best_x) else best_x

def firefly_algorithm(n, x_min, x_max, max_iter):
    # 初始化蜻蜓群
    x = np.random.uniform(x_min, x_max, n)
    v = np.zeros(n)
    p = x.copy()
    best_x = x[np.argmin(fitness_function(x))]

    for t in range(max_iter):
        for i in range(n):
            v[i] = update_velocity(v[i], p[i], c1, c2, random.random(), random.random())
            x[i] = update_position(x[i], v[i])
            p[i] = update_best_position(x[i], p[i])

        best_x = update_best_position(best_x, p.min())

    return best_x, fitness_function(best_x)

n = 20
x_min = -10
x_max = 10
max_iter = 100

best_x, best_fitness = firefly_algorithm(n, x_min, x_max, max_iter)
print("最佳解: ", best_x)
print("最佳解对应的适应度: ", best_fitness)
```

## 4.2 详细解释说明

在上述代码实例中，我们首先定义了目标函数`fitness_function`，该函数用于评估蜻蜓群中每个蜻蜓的适应度。接着，我们定义了`update_velocity`、`update_position`和`update_best_position`三个函数，分别用于更新蜻蜓的速度、位置和个体最佳位置。

接下来，我们初始化了蜻蜓群，包括蜻蜓的位置、速度、个体最佳位置和全群最佳位置。然后，我们使用`for`循环进行迭代，在每一轮迭代中更新蜻蜓的速度和位置，并更新个体最佳位置和全群最佳位置。

最后，我们返回全群最佳位置和对应的适应度，并打印输出。
