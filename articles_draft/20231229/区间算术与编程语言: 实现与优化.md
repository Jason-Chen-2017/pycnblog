# 2.核心概念与联系

在深入探讨区间算术之前，我们需要了解一些基本概念。首先，我们需要明确什么是区间。区间是一个包含有限个元素的有序集合，其中每个元素都在一个有限的数值范围内。例如，在整数域中，我们可以定义一个区间为`[a, b]`，其中`a`和`b`是整数，`a <= b`。

接下来，我们需要了解区间算术的基本操作。这些操作包括：

- 加法：将两个区间相加，得到一个新的区间。
- 减法：将一个区间从另一个区间中减去，得到一个新的区间。
- 乘法：将两个区间相乘，得到一个新的区间。
- 除法：将一个区间除以另一个区间，得到一个新的区间。

这些基本操作可以用来实现更复杂的算法，如求区间的交集、并集、差集等。

```python
class Interval:
    def __init__(self, start, end):
        self.start = start
        self.end = end

    def __add__(self, other):
        if not isinstance(other, Interval):
            raise ValueError("Invalid interval")
        new_start = self.start + other.start
        new_end = self.end + other.end
        if new_start > new_end:
            new_start, new_end = new_end, new_start
        return Interval(new_start, new_end)

    def __sub__(self, other):
        if not isinstance(other, Interval):
            raise ValueError("Invalid interval")
        new_start = self.start - other.end
        new_end = self.end - other.start
        return Interval(new_start, new_end)

    def __mul__(self, other):
        if not isinstance(other, Interval):
            raise ValueError("Invalid interval")
        new_start = self.start * other.start
        new_end = self.end * other.end
        return Interval(new_start, new_end)

    def __truediv__(self, other):
        if not isinstance(other, Interval):
            raise ValueError("Invalid interval")
        new_start = self.start / other.end
        new_end = self.end / other.start
        return Interval(new_start, new_end)
```
