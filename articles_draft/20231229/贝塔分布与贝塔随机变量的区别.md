## 2.1 贝塔分布

贝塔分布是一个连续概率分布，用于描述一个随机变量的取值范围在0和1之间的分布。贝塔分布的概率密度函数（PDF）定义为：

$$
f(x) = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha) \Gamma(\beta)} x^{\alpha - 1} (1 - x)^{\beta - 1}
$$

其中，$\alpha$ 和 $\beta$ 是贝塔分布的参数，$\Gamma$ 是伽马函数。

贝塔分布具有以下性质：

- 当 $x \rightarrow 0$ 时，$f(x) \rightarrow \infty$，当 $x \rightarrow 1$ 时，$f(x) \rightarrow \infty$。
- 贝塔分布是一个连续概率分布，其取值范围在0和1之间。
- 贝塔分布的均值和方差分别为 $\frac{\alpha}{\alpha + \beta}$ 和 $\frac{\alpha \beta}{(\alpha + \beta)^2 (\alpha + \beta + 1)}$。

## 2.2 贝塔随机变量

贝塔随机变量是一个连续随机变量，其分布遵循贝塔分布。贝塔随机变量可以通过以下方式生成：

- 从一个Uniform(0, 1)随机变量中抽取一个值。
- 从一个Exponential($\lambda$)随机变量中抽取一个值，然后将其除以$\lambda$。

贝塔随机变量具有以下性质：

- 贝塔随机变量的取值范围在0和1之间。
- 贝塔随机变量的均值和方差分别为 $\frac{\alpha}{\alpha + \beta}$ 和 $\frac{\alpha \beta}{(\alpha + \beta)^2 (\alpha + \beta + 1)}$。

## 2.3 贝塔分布和贝塔随机变量之间的联系

贝塔分布和贝塔随机变量之间的关系是密切的。贝塔随机变量的分布遵循贝塔分布，因此可以用贝塔分布的概率密度函数来描述贝塔随机变量的取值概率。

## 4.1 使用Python生成贝塔随机变量

我们可以使用Python的`scipy.stats`库来生成贝塔随机变量。以下是一个示例代码：

```python
import numpy as np
from scipy.stats import beta

# 生成1000个贝塔随机变量
x = beta.rvs(a=5, b=6, size=1000)

# 计算贝塔随机变量的均值和方差
mean = np.mean(x)
variance = np.var(x)

print("均值:", mean)
print("方差:", variance)
```

在这个示例中，我们使用了`beta.rvs`函数生成了1000个贝塔随机变量。然后，我们计算了贝塔随机变量的均值和方差。

## 4.2 使用Python计算贝塔分布的概率密度函数

我们可以使用Python的`scipy.stats`库来计算贝塔分布的概率密度函数。以下是一个示例代码：

```python
import numpy as np
from scipy.stats import beta

# 设置贝塔分布的参数
a = 5
b = 6

# 计算贝塔分布在x=0.5的概率密度值
pdf_value = beta.pdf(0.5, a, b)

print("贝塔分布在x=0.5的概率密度值:", pdf_value)
```

在这个示例中，我们使用了`beta.pdf`函数计算了贝塔分布在$x=0.5$的概率密度值。
