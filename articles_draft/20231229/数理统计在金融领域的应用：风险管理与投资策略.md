在金融领域，随机变量是表示未知或不可预测的因素的变量。概率分布是描述随机变量可能取值和出现概率的函数。常见的概率分布包括均匀分布、二项分布、泊松分布、董事会分布和正态分布等。

## 2. 期望和方差

期望是随机变量的均值，表示其中心趋势。方差是期望和实际值之间的平方和的平均值，表示随机变量的离散程度。在金融领域，期望和方差是评估风险和回报的关键指标。

## 1. 最小二乘法

最小二乘法是一种用于估计线性回归模型参数的方法。它的基本思想是最小化残差平方和，使得预测值与实际值之间的差异最小。在金融领域，最小二乘法用于预测股票价格、利率和通货膨胀等。

数学模型公式：

$$
\min \sum_{i=1}^{n}(y_i-\hat{y}_i)^2
$$

## 2. 蒙特卡洛方法

蒙特卡洛方法是一种通过随机抽样 approximates 函数值和积分的方法。在金融领域，蒙特卡洛方法用于估计复杂模型的预测值和风险。

数学模型公式：

$$
\hat{y}=\frac{1}{N}\sum_{i=1}^{N}y_i
$$

## 3. bootstrap 方法

bootstrap 方法是一种通过随机抽样来估计参数和统计量的方法。在金融领域，bootstrap 方法用于估计模型的置信区间和预测误差。

数学模型公式：

## 3. bootstrap 方法

```python
import numpy as np
import pandas as pd

# 加载数据
data = pd.read_csv('stock_data.csv')

# 选取特征和目标变量
X = data[['open', 'high', 'low', 'volume']]
y = data['close']

# 设置参数
n = 10000
n_bootstrap = 1000

# 生成bootstrap样本
bootstrap_samples = []
for i in range(n_bootstrap):
    sample = np.random.choice(X.index, size=X.shape[0])
    bootstrap_samples.append(X.loc[sample])

# 计算预测值
predictions = np.mean(bootstrap_samples, axis=0)
```
