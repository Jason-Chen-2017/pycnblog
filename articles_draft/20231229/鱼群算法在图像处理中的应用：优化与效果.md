# 2.核心概念与联系

## 2.1 鱼群算法的基本概念

鱼群算法是一种基于自然现象的优化算法，它模拟了鱼群中的行为规律，以解决复杂优化问题。主要包括以下几个概念：

1. 鱼群：鱼群是算法中的主要实体，每个鱼群包含多个鱼，鱼在行动空间中随机移动。
2. 鱼：鱼是鱼群中的单个实体，它有自己的位置、速度、能量等属性。
3. 能量：能量是鱼群中的一个重要参数，它反映了鱼的适应性，越高的能量表示越优越适应。
4. 相互作用：鱼群中的鱼之间存在相互作用，这些作用可以通过邻域选择、领导力等手段实现。
5. 自然选择：在每一轮迭代中，根据鱼的能量值进行自然选择，选出适应性最强的鱼作为下一轮迭代的领导者。

1. 初始化鱼群和能量值。
