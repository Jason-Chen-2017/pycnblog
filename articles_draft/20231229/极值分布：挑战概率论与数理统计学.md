极值定理是一种描述随机变量极端值分布特征的定理。它主要包括以下两个结论：

1. 标准正态分布的极大值（或极小值）的分布是泊松分布。
2. 随机变量的极小值（或极大值）的分布可以通过对应的随机变量的分布函数的反函数得到。

### 1.2.4 极值分布与其他分布的关系

极值分布与其他分布之间存在很强的联系。例如，正态分布是最常见的分布之一，它的极大值分布可以通过极值定理得到。此外，极值分布还可以用于描述其他分布的极端值，如泊松分布、指数分布等。

### 1.3.3 极值分布的数学模型公式

在本节中，我们将详细讲解极值分布的数学模型公式。

#### 1.3.3.1 极大值分布（Type I Extreme Value Distribution）

极大值分布的概率密度函数（PDF）为：

$$
f(x; \mu, \sigma) = \frac{1}{\sigma} e^{-(x-\mu) / \sigma} e^{-e^{-(x-\mu) / \sigma}}
$$

其中，$\mu$ 表示位置参数，$\sigma$ 表示形状参数。

#### 1.3.3.2 极小值分布（Type II Extreme Value Distribution）

极小值分布的概率密度函数（PDF）为：

$$
f(x; \mu, \sigma) = \frac{1}{\sigma} e^{(-x-\mu) / \sigma} e^{-e^{(-x-\mu) / \sigma}}
$$

其中，$\mu$ 表示位置参数，$\sigma$ 表示形状参数。

#### 1.3.3.3 泊松极值分布（Poisson Extreme Value Distribution）

泊松极值分布的概率密度函数（PDF）为：

$$
f(x; \lambda) = \frac{\lambda^x e^{-\lambda}}{x!}
$$

其中，$\lambda$ 表示参数。



### 1.4.4 使用极值分布进行分析

最后，我们可以使用极值分布的数学模型公式来计算极端值的概率。我们将使用 `scipy` 库的 `extreme_rv` 函数来计算极大值和极小值的概率。
