```solidity
pragma solidity ^0.5.0;

contract DataAnalysis {
    uint256 private data;
    address private owner;

    constructor(uint256 _data, address _owner) public {
        data = _data;
        owner = _owner;
    }

    function getData() public view returns (uint256) {
        require(msg.sender == owner, "Only owner can access this data.");
        return data;
    }
}
```

在这个智能合约中，我们定义了一个私有变量`data`来存储数据，一个私有变量`owner`来存储数据的所有者。在构造函数中，我们初始化这两个变量。

我们还定义了一个公共函数`getData`，它允许某个地址访问数据。在这个函数中，我们使用`require`语句来检查调用者的地址是否与数据的所有者相同，如果不同，则抛出一个错误。
