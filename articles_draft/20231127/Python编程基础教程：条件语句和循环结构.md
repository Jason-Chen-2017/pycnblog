# 2.核心概念与联系
## 变量
在计算机中，变量是存储数据的内存位置或寄存器中的一个名称。在Python中，我们可以使用变量来临时存储数据或用于保存中间结果。变量名必须遵循标准的标识符命名规则，即只能包含字母数字下划线，且不能以数字开头。同时，变量名通常用小写字母表示。

```python
a = 1   # 整数赋值
b = 'hello'    # 字符串赋值
c = [1, 2, 3]    # 列表赋值
d = (1, 2, 3)    # 元组赋值
e = {'name': 'Alice', 'age': 25}     # 字典赋值
f = True      # 布尔值赋值
g = None      # 空值赋值
h = 1 + 2j    # 复数赋值
```

### 数据类型
在Python中，所有的数据都属于某个特定的数据类型，不同的数据类型有不同的特征和使用范围。常见的数据类型有：

1. 整数（int）
2. 浮点数（float）
3. 字符串（str）
4. 布尔值（bool）
5. 列表（list）
6. 元组（tuple）
7. 字典（dict）
8. 集合（set）
9. 复数（complex）

Python的动态语言特性使其能够灵活地处理不同类型的数据，并且允许用户自定义数据类型。

## 控制流语句
### if-else语句
if-else语句是最常用的流程控制语句之一。它的一般形式如下：

```python
if condition:
    # 如果condition成立，执行以下代码块
    
else:
    # 如果condition不成立，执行以下代码块
```

如果if后的条件判断表达式为True，则执行if后的代码块；否则，执行else后的代码块。多个elif子句可以用来添加多个条件判断。

```python
if age < 18:
    print('未成年')
elif age >= 18 and age <= 60:
    print('成年')
else:
    print('老年')
```

### for-in语句
for-in语句是一种重复执行某段代码的机制。它的一般形式如下：

```python
for variable in sequence:
    # 对sequence中的每个元素执行的代码块
```

其中variable是一个临时变量，表示序列中的当前元素；sequence代表一系列值，如一个列表或元组。for-in语句可以遍历任何可迭代对象，如列表、元组、字符串、字典等。例如，可以将字符串中的每一个字符打印出来：

```python
s = "Hello, world!"
for c in s:
    print(c)
```

输出为：

```python

l
l
o
,
 

w
o
r
l
d
!
```

### while语句
while语句也称作“无限循环”或者“死循环”，当条件满足时，就一直执行循环体中的代码。它的一般形式如下：

```python
while condition:
    # 当condition满足时，执行以下代码块
```

当循环体中的代码被执行了一次之后，会计算一次条件表达式，如果还是为True，就会再次进入循环，直到条件表达式为False为止。例如，实现求一个数的绝对值函数abs()：

```python
def abs(x):
    if x >= 0:
        return x
    else:
        return -x
```

利用while语句，可以实现同样的功能：

```python
def abs_v2(x):
    y = x
    while y < 0:
        y += x * 2
    return y
```

这里，y是一个变量，表示输入的x的值，而x*2是为了消除负号的影响。当y小于0时，表示x的值为负数，此时继续乘以x，得到新的y值，并继续判断；否则，表示x的值已经是非负数，直接返回。这样，abs_v2()就可以正确地求出任意整数或浮点数的绝对值。

### pass语句
pass语句用于补充占位符。由于if-else语句的分支缺失，导致代码无法正常运行，或者出现语法错误。在这种情况下，我们可以使用pass语句来填充缺失的位置。

```python
num = input("请输入数字:")
try:
    num = int(num)
    print(num)
except ValueError:
    pass
```

这里，我们尝试将用户输入的字符串转换为整数。如果输入的不是有效的整数，则程序抛出ValueError异常，但是由于没有对该异常进行处理，程序崩溃。我们可以使用pass语句跳过对该异常的捕获，避免程序崩溃：

```python
num = input("请输入数字:")
try:
    num = int(num)
    print(num)
except ValueError:
    pass
finally:
    print("程序结束")
```

这里，finally语句保证程序最后一定会执行，即使有异常发生也是如此。

## 函数定义
函数是组织代码的有效手段。在Python中，我们可以定义函数来实现特定功能，然后在其他地方调用该函数。函数定义使用def关键字，后跟函数名、括号和参数。函数体由缩进的代码行构成。函数调用使用函数名加括号，并传入相应的参数。

```python
def greet():
    print("Hello!")

greet()
```

这里，我们定义了一个greet()函数，它没有参数，仅包含一条print语句。调用greet()函数时，程序输出"Hello！"。

```python
def add(x, y):
    return x + y

z = add(2, 3)
print(z)
```

这里，我们定义了一个add()函数，它接受两个参数x和y，并返回它们的和。我们调用add()函数并传入参数2和3，得到的结果是5，所以将其打印出来。
