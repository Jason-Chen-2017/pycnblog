## 2. 技术原理及概念

2.1. 基本概念解释

2.1.1. 什么是动态规划？

动态规划（Dynamic Programming，简称DP）是解决很多多状态、多决策变量的组合优化问题的一种有效方法。动态规划将原问题分解成子问题，通过求解子问题的最优解来解决原问题。动态规划的核心思想是利用子问题的最优解来构建整体问题的最优解，从而避免了重复计算。

2.1.2. 什么是编辑距离？

编辑距离是指将一个字符串转换成另一个字符串所需的最小操作次数。在自然语言处理中，编辑距离主要用于计算两个字符串之间的相似度。

2.1.3. 如何使用动态规划解决编辑距离问题？

使用动态规划解决编辑距离问题的基本思路如下：

1) 定义一个二维数组 dp，其中 dp[i][j] 表示将字符串 s 转换成字符串 t 的最小编辑距离（包括插入、删除、替换）。

2) 定义一个字符串集合（或者称为邻接表）原字符串集合（O超集），以及一个操作集合（或者称为操作集）。操作集合包括插入一个字符、删除一个字符和替换一个字符。

3) 利用动态规划的基本思想，从原字符串集合中找到一个子字符串 t'（与编辑距离无关，但与子字符串集合中的元素相对应），使得 t' 和 dp[i][j] 达到最小。然后将 t 替换为 t'，继续寻找子字符串 t''（与编辑距离无关，但与子字符串集合中的元素相对应），使得 t'' 和 dp[i][j] 达到最小。重复以上步骤，直到找到一个子字符串 t''，使得 t'' 和 dp[i][j] 达到最小。

2.2. 技术原理介绍:算法原理，操作步骤，数学公式等

2.2.1. 算法原理

动态规划技术的核心在于利用子问题的最优解来构建整体问题的最优解。在解决编辑距离问题时，我们可以将原问题拆分成子问题，通过求解子问题的最优解来解决原问题。通过不断尝试子问题的最优解，最终找到一个满足编辑距离最小要求的子字符串，从而构建出整体问题的最优解。

2.2.2. 操作步骤

（1）初始化动态规划数组 dp，以及原字符串集合 O 和操作集合 S。

（2）从 O 集合中找到一个子字符串 t，初始化距离 dp[0][0]=0。

（3）遍历 S 集合，找到第一个使得 t 中的某个字符与编辑距离相关的子字符串，将其添加到 dp 数组中，更新距离。

（4）重复步骤（3），直到找到一个满足编辑距离最小要求的子字符串。

（5）将找到的子字符串 t 替换为 O 集合中的字符串，并更新距离为 0。

（6）重复步骤（2）-（5），直到原字符串集合中的所有子字符串的距离都为 0。

2.2.3. 数学公式

假设编辑距离为 min编辑距离（dp[i][j]），编辑距离的计算公式为：

min编辑距离（dp[i][j]）= min(|s[i]-t[i]|, |s[i]-t[i]|)

其中，| | 表示取绝对值。

import set

def min_edit_distance(s, t):
    dp = [[None for _ in range(len(s))] for _ in range(len(t))]
    return dp

def dp_vector(dp):
    return [[dp[i][j] if j == i else None for i in range(len(dp)) for j in range(len(dp[0]))]

def vector_operation(dp, operation):
    return [dp[i][j] if operation else None for i in range(len(dp)) for j in range(len(dp[0]))]

def soft_min(dp):
    dp = [0] + dp
    return min(dp)

def max_dp(dp):
    dp = [float('inf')] + dp
    return max(dp)

def dynamic_program(s, t):
    dp = min_edit_distance(s, t)
    dp_vec = dp_vector(dp)
    return soft_min(dp_vec), max_dp(dp)

def main():
    s = set([
        'aa', 'ab', 'ac', 'a', 'b', 'bc', 'abc', 'abd', 'c', 'bd', 'be', 'b', 'c', 'ce', 'cb', 'cde'
    ])
    t = set(['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j', 'k', 'l','m', 'n', 'o', 'p', 'q', 'r','s', 't', 'u', 'v', 'w', 'x', 'y', 'z', 'aa', 'ab', 'ac', 'a', 'b', 'bc', 'abc', 'abd', 'c', 'bd', 'be', 'b', 'c', 'ce', 'cb', 'cde'])
    dp = dynamic_program(s, t)
    print(dp)

if __name__ == '__main__':
