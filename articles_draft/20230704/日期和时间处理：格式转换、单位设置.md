* yyyy-MM-DD HH:MM:SS 格式：将日期和时间转换为 yyyy-MM-DD HH:MM:SS 格式，包括YYYY-MM-DD HH:MM:SS等。

* 秒：将时间转换为秒，适用于需要快速响应的场景。
* 毫秒：将时间转换为毫秒，适用于需要更高精度的时间控制场景。

4. 应用示例与代码实现讲解
---------------------------------

4.1. 应用场景介绍

本文将介绍如何使用 Python 实现日期和时间的格式转换和单位设置。

4.2. 应用实例分析

首先，我们需要获取用户输入的日期和时间，然后对输入的日期和时间进行格式转换和单位设置，最后将结果输出给用户。

4.3. 核心代码实现

```python
import datetime

# 获取用户输入的日期和时间
date_time_str = input("请输入日期和时间（格式：YYYY-MM-DD HH:MM:SS）：")

# 将输入的日期和时间格式转换为 datetime.datetime 类型
date_time = datetime.datetime.strptime(date_time_str, "%Y-%m-%d %H:%M:%S")

# 对日期和时间进行格式转换和单位设置
date_format = "%Y-%m-%d"
time_format = "%H:%M:%S"

date_str = date_time.strftime(date_format)
time_str = date_time.strftime(time_format)

# 将格式化后的日期和时间输出给用户
print("格式化后的日期为：", date_str)
print("格式化后的时间为：", time_str)
```

4.4. 代码讲解说明

在上面的代码中，我们首先通过 input 函数获取用户输入的日期和时间，并使用 datetime.datetime.strptime 函数将输入的日期和时间格式化为 datetime.datetime 类型。

接着，我们对日期和时间进行了格式转换和单位设置，将日期和时间分别转换为 date_format 和 time_format 格式，最后将格式化后的日期和时间输出给用户。
