                 

# 1.背景介绍

  
近几年来，随着科技的发展、人们生活水平的提高，引力波的频率和能量的增长速度越来越快，已成为引起广泛关注的问题。根据统计数据，中国每年捕捉到超过1000个引力波，其中不少于1/3的人工引力波源出自地球上。在宇宙学层面，宇宙中存在大量的具有双星际质量比的恒星系统，这些恒星质量比在5-7之间，这种恒星系统被称为多信使系统（Multiple Messenger System）。多信使系统在空间上的分布也有很多规律可循，例如星团形成和星系团合成等，但在宇宙学的研究中，多信使系统的观测却并非易事。  
最近，我国在多信使系统的调查上取得了突破性进展。作为观测第一线的地球物理专家团队，国家天文台共同开发了一套多信使系统观测工具箱——“中国引力波天文观测平台”(GBTCS)，其目标是通过对接各种数据资源，实现多信使系统观测全流程自动化、精准、有效。平台的建设也受到了欧洲和日本的多方支持。GBTCS主要包括五大功能模块：  
①多信使系统观测自动化
通过对接各类资源并结合数据分析理论，平台可以自动识别多信使系统特征并对其进行时空分布的估计，从而帮助科研人员快速进行多信使系统观测和参数测定。

②高分辨率分析及建模
由于多信使系统的复杂性、多样性以及宇宙的不确定性，无法依靠传统的空间相位谱分析或连续谱测量获得精确信息。GBTCS采用光谱仪、红外传感器等高分辨率分析方法进行建模处理，以获取更加符合实际情况的数据。

③数字图像监控
随着技术的飞速发展，科技实验室正在积极投入到宇宙学领域，对多信使系统的各种性质有更深刻的认识。为了能够准确、迅速地对此种多样性的宇宙结构进行探测与分析，平台引入了数字图像监控与跟踪（Digital Image Monitoring and Tracking）模块，在计算机视觉算法的基础上引入机器学习技术，实现了多信使系统的目标识别。

④观测网络规划与设计
作为一个具有国际影响力的地球物理研究机构，GBTCS的观测网络需要保持稳定的高效运行。在此过程中，需要充分利用全球多元数据资源，为地球物理学家提供无限可能。因此，平台还提供了“观测网络规划与设计”模块，为观测者设计专属的观测网格，满足对多信使系统不同区域的观测需求。

2.核心概念与联系  
引力波是一种电磁波，其传播速度超过光速，由加速度所引起的离心力场作用。通过观测引力波可以发现宇宙中存在的微小粒子，这是研究宇宙中物质运动和物质演化的重要途径。宇宙学中多信使系统指的是星体、恒星系统以及植物、动物等众多生命体在空间上密集聚集的系统，它由数以百万计的恒星团团围绕、形成的巨大膨胀结构组成。多信使系统是由无数的引力子体按照固定轨道相互作用形成的宇宙结构，是宇宙中重要的天然奇点和热电核磁体。但是，如何发现和观测多信使系统一直是一个难题。  
多信使系统观测的核心问题就是如何找到多信使系统所在的位置、大小、形状、时间跨度以及分布规律。这一过程涉及到对宇宙的整体了解、掌握物理的基本原理、建模的方法、数据处理的能力、算法的优化等知识技能。观测多信使系统的关键是建立能够有效识别多信使系统特征的算法模型，提取其位置、大小、形状、多样性等多方面的信息。本文将基于多信使系统的定义和相关原理，以引力波为例，描述多信使系统的观测流程。

3.1 引力波理论
引力波是一种带电粒子的电磁波，其传播方向和强度依赖于周围空间的总引力。引力波在宇宙中扮演了重要角色，具有诸如黑洞、流体、星云、大气、重力等广泛的应用。一般来说，引力波具有以下几条性质:

① 发散性：由于引力场的存在，引力波会出现奇异的发散现象，即其长宽比可能会发生变化；

② 矩形特征：引力波呈现出矩形结构，即二维波动模式具有水平分辨率、垂直分辨率均为正整数倍；

③ 复杂性：引力波的复杂性来源于动力学中包含的诸多噪声，包括分辩率降低、失真、混叠等；

④ 实验性质：引力波在实验过程中具有丰富的物理特性，如超导性、阻抗匹配、多能级等。

为观测引力波，首先要对引力波的基本理论有所了解。引力波理论研究了引力波的基本物理过程以及它们产生的物理效应。物理定律表明，引力波的传播速度、大小、方向和强度都依赖于宇宙中尤其是超级质量体系的存在以及引力场的构造。

3.2 多信使系统简介
多信使系统是一个复杂的系统，其构建方式主要有星团形成和星系团合成两种，这两者之间的区别就在于最终形成的系统的宏观尺度。比如，星团形成的多信使系统具有宇宙范围内的宏观尺度，主要分布在星系内部；而星系团合成的多信使系统则具有宏观尺度很大的系统，广义坐标系内的位置是由大质量对象确定的。多信使系统又可细分为单星团多信使系统、双星际质量比多信使系统等。

3.3 GBTCS多信使系统观测流程

主要流程说明：

1. 收集数据：通过各种方式获取和处理多信使系统的数据。目前主要的工具包括红外线、红外遥感、高分辨率天文望远镜等。

2. 数据处理：对数据进行预处理，去除干扰项、处理噪声、提取特征。

3. 模型构建：构建多信使系统的空间分布模型，包括构建空间分布密度函数、多信使系统所在的相位分布以及其他动态过程。

4. 特征检测：对空间分布模型的结果进行特征检测，找寻多信使系统特征，如相位偏转、聚集结构、聚团结构等。

5. 参数估计：对多信使系统的特征进行反向计算，得到多信使系统的参数，如密度、半径等。

6. 结果展示：将观测结果显示出来，包括空间分布图像、系统尺度图像以及参数曲线。

通过GBTCS平台的多信使系统观测，可以快速且准确地找到多信使系统的位置、大小、形状、密度、分辨率、方向等信息。
