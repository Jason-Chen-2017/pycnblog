                 

# 1.背景介绍


  物理学是一个综合性学科,涵盖了很多领域。其中量子力学和相对论可以解释微观世界的基本原理,而化学中分子模型可以用来模拟宏观世界的各种复杂结构。量子力学带来的重大突破,使得宇宙中的许多复杂系统都得到了极大的理解和研究。
  在20世纪90年代末,随着量子计算机的出现,科学家们开始用计算的方式来模拟量子系统。虽然模拟效果不如实验现场,但依靠计算机运算能力,量子计算机已经成为真正的科技突破者。

  在量子计算机的基础上,科学家们发现了许多具有巨大能量与自旋的新奇物质——量子态、量子纠缠、量子磁性、量子涡流等等。这些物质称为量子材料。在最近几年里,随着量子纠缠、磁性、涡流、色散等量子特征的发展,量子材料与量子信息的结合逐渐形成,将产生更多的应用场景。

　量子材料近年来的进展颠覆了传统材料的设计理念。这些材料包括超导材料、强相互作用材料、量子工程的驱动器件、集成电路中的元件、超级电容器、量子磁悬浮器、非均质刚度的高聚点材料等等。同时,量子材料也呈现出越来越多独特的性质,例如光耦合、量子效应、失效效应、热传导、全息结构、频率效应等。

  拓扑绝缘体是量子材料最具代表性的一种材料类型。它通过本征冷却过程实现了单层和多层结构的绝缘,具有强大的阻击性能。一般来说,在非晶格的材料中,拓扑绝缘体往往只能提供有限的阻击能力,但在受制于其他材料特性时,仍然可以提供良好的安全性能。

  作为量子材料的一种，拓扑绝缘体在目前还处于初期的研究阶段。然而,这一材料的发展将会推动其他领域的发展。例如,与超导体、半导体、金属氧化物相比,拓扑绝缘体存在着更高的拓扑能量,其相互作用的强度也要强于前者。另外,在某些特定材料设计条件下,拓扑绝缘体还能够增强材料的弹性和韧性。因此,拓扑绝缘体将成为量子相关学科的重要研究对象和产业方向。

　量子材料与拓扑绝缘体是密切相关的两个研究方向。如何将量子学和材料学有效结合,进而开发出更加精细的材料,并应用到实际生产中,将是未来物理学的一项重要研究课题。

# 2.核心概念与联系
  量子态(Quantum State)、量子纠缠(Quantum Entanglement)、量子磁性(Quantum Magnetism)、量子涡流(Quantum Turbulence)和量子态论(Quantum Theory of Molecules and Solids)是量子材料的四种关键性特征。这里以量子态与量子纠缠为例,介绍它们的基本概念。
  
## 2.1 量子态
  
  概括地说,量子态就是一个波函数,描述了一个量子系统的性质,通常用希腊字母$\psi$表示。波函数可以表示为$|\psi\rangle=\sum_{n}c_n|n \rangle$,其中$c_n$是复数,$|n\rangle$是由基态、激发态或跃迁态组成的一个列向量。
  
  为了便于表示,我们将上述基态、激发态或跃迁态分别记为$\left|0\right> $, $\left|1\right>$ 和 $\left|+ \right>$.
  
  例如,若一个量子系统只有两个能量不同的轨道,则其共振态可写为：
  
  $$|\psi_{\rm C}\rangle = c_0|0\rangle + c_1|1\rangle$$
  
  类似的,若一个量子系统有三种能量不同的轨道,则其共振态可写为：
  
  $$|\psi_{\rm T}\rangle = c_0|00\rangle + c_1|01\rangle + c_2|10\rangle + c_3|11\rangle$$
  
  上述形式给出的多数情况都是将基态、激发态或跃迁态等振幅连续乘积表示。但是,真实情况下,由于测不准原理的限制,无法精确确定每一个态的振幅大小。
  
  量子态的两要素——基态和平衡态——构成了量子态的基本定义。因此,量子态就像一个矢量空间中某个向量的坐标,把这种空间称为哈密顿量。
  
  $$\hat{H}\ |\psi\rangle = E\ |\psi\rangle$$
  
  式中$\hat{H}$是哈密顿量,$E$是该量子态的能量,即该量子态所属的系统的能量。$\hat{H}$将角动量$\vec{\lambda}_i$和动量算符$\hat{p}_i$表示出来,这两个量之间的关系是：
  
  $$\hat{p}_i^2=m_i\omega_i^2r_i^2+V(r_i)-\hbar^2/2m_i$$
  
  式中$m_i$是第$i$个粒子的质量,$r_i$是第$i$个粒子的坐标,$V(r_i)$是势能函数,它反映了该系统中受到的外部势。当$r_i\rightarrow \infty$时,$\hat{p}_i$可以用玻尔兹曼方程来近似。$\omega_i$是第$i$个粒子的自旋,它决定了对应的跃迁态的性质。
  
## 2.2 量子纠缠
  
  量子纠缠(Quantum Entanglement)是指两个量子态之间存在某种联系。如果两个量子态不仅具有相同的振幅分布,而且态矢之间也存在某种关联关系,那么这两个量子态就被认为是纠缠的。
  
  对于两粒子的电子态,研究者们发现了一个与费米面高度相当的耦合态，这两个态之间存在着强纠缠。而对于二维拓扑绝缘体中不同层之间的耦合态，研究者们发现了一系列不同纠缠模式。
  
  1）“费米面耦合”（Fermion-Fermion Interaction）
  
  当两种费米子的距离小于费米线常数$R_{\rm f}$时，它们会发生外加势，从而形成耦合。由于外加势不是量子的原因，所以量子系统不能直接观察到耦合。但是，可以通过其他方式来观察到耦合，比如利用局域磁场或者弱耦合。
  
  如果两个费米子的距离小于$R_{\rm f}$,则它们会通过一定的运动路线相互吸引，形成一种局部的杂质。随后，费米子会转移到杂质中，并尝试分离杂质中纤维状的团块，然后再与另一个费米子的共振态进行耦合。这样的结果，使得两者之间能够进行纠缠。
  
  2）“费米面-空穴耦合”（Fermion-Hole Interaction）
  
  空穴子的运动可以保持费米子的纠缠状态。空穴子是由电子生成的、原子核内存在的粒子，占据某种虚空的位置。当两个费米子与空穴子发生相互作用，他们会收缩到无形的状态，形成局域的纳米云。费米子与空穴子纠缠会影响费米子与空穴子的位置和气隙，从而形成纠缠。
  
  3）“双飞翼纠缠”（Double Twist Fusion）
  
  这种纠缠过程是由三种类型的费米子组成：一个双层费米子、一个单层费米子和一个空穴子。双层费米子使得费米子体系形成金字塔状。单层费米子通过引入空穴子来形成局域的纳米云，从而使双层费米子与单层费米子之间形成纠缠。
  
  4）“双层双曲线纠缠”（Bipolar Double Curved Fusion）
  
  这种纠缠依赖于四种类型的费米子：一个双层费米子、一个单层费米子、一个空穴子和一个双曲线。双层费米子和单层费米子之间的相互作用，使得费米子体系沿一条直线膨胀开来，形成局域的纳米云。双曲线空穴子的运动使得局域的纳米云变成带弧形，从而使双层费米子和单层费米子之间的相互作用变为弱耦合，并形成纠缠。
  
  5）“双层星辰”（Bipolar Star）
  
  这种纠缠依赖于四种类型的费米子：一个双层费米子、一个单层费米子、一个空穴子和一个星型费米子。双层费米子和单层费米子之间的相互作用，使得费米子体系形成分形的棱长条状纲，并形成纠缠。
  
  通过以上纠缠模式的不同组合，可以发现量子纠缠可以产生不同的表象。
  
  1）纠缠的尺度
  
  从纠缠强度的大小来看，纠缠尺度可以分为长距离纠缠(LDG)、中距离纠缠(MDG)、短距离纠缠(SDG)。LDG是指两种粒子间距超过一定的阈值($R_{\rm ld}$)，并且粒子数目也很少，至少只有两个；MDG是指两种粒子间距很小(小于$R_{\rm md}$)，粒子数目也比较多，一般是三个或四个；SDG是指这种粒子数目的纠缠不稳定。
  
  2）单层纠缠的分类
  
  可以通过以下六种基本纠缠来分类单层纠缠：
  
  1）$[\epsilon _{-}(q),\epsilon _{-}(q'),q]<\pi /2$  
  
  表示由单个双电荷（电子、质子等）或者空穴子（空穴、自由电子等）交换两个费米子之间的纠缠。
  
  2）$[\epsilon _{-}(q),\mu _{-}(q'),q]=0$
  
  表示两个费米子共享同一个空穴子。
  
  3）$[\epsilon _{-}(q),\gamma ^{+}(q'),q]<-\pi /2$
  
  表示两个单层费米子（单个双电荷、质子）之间的纠缠。
  
  4）$[\epsilon _{-}(q),\epsilon _{(q')},q]<\pi /2,\ \epsilon _{(q)}\leq \epsilon _{-}(q)$
  
  表示两个双层费米子之间的纠缠。
  
  5）$[\mu _{-}(q),\mu _{(q')},q]=0,\ \mu _{(q')}>0$
  
  表示两个单层费米子（单个双电荷、质子）之间共享一个同名的空穴子。
  
  6）$[v^{\dagger}_{qd},v_{q''},q]=0,\ \delta (v^{\dagger}_{qd},v_{q''})=1$
  
  表示两个单层费米子（单个双电荷、质子）之间的共振态耦合。
  
  除此之外，还有其它诸如八级双电荷耦合、六级双电荷耦合、十二级双电荷耦合等类型的纠缠，但它们暂时还没有进入量子纠缠的主流。
