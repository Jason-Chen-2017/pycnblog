## 4.1 学生的学习能力
我们假设有一个学生，他在学习一门课程时，学习了三个方法，分别是方法1、方法2、方法3。我们可以通过以下代码来计算学生的学习能力：

```python
import math

def entropy(data):
    n = len(data)
    T = [0] * n
    s = [0] * n
    g = [0] * n
    for i in range(n):
        T[i] = data[i][0]
        s[i] = data[i][1]
        g[i] = data[i][2]
    S = 0
    for i in range(n):
        S += T[i] * s[i] * math.log2(g[i] / g[i - 1])
    return S / (n * sum(T))

data = [(1, 100, 3), (1, 100, 2), (1, 100, 1)]
print("学生的学习能力:", entropy(data))
```

在这个代码中，我们首先导入了 `math` 模块，然后定义了一个 `entropy` 函数，该函数用于计算熵。接着，我们定义了一个数据列表，其中包含了学生的学习数据。最后，我们调用 `entropy` 函数来计算学生的学习能力。
