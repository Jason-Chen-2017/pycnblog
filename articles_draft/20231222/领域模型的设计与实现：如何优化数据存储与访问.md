## 4.1 代码实例介绍

我们将实现一个简单的电商系统，其中包括用户、订单、商品等实体。我们将使用Python的SQLAlchemy库来实现数据存储和访问，使用Python的Flask库来实现业务逻辑。

## 4.2 实体设计

首先，我们需要设计实体。我们将设计以下实体：

- 用户实体：表示用户，包括用户名、密码、邮箱等属性。
- 订单实体：表示订单，包括订单号、用户ID、商品ID、数量等属性。
- 商品实体：表示商品，包括商品ID、名称、价格、库存等属性。

我们将使用以下Python代码来实现这些实体：

```python
from sqlalchemy import create_engine, Column, Integer, String, Float
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker

Base = declarative_base()

class User(Base):
    __tablename__ = 'users'
    id = Column(Integer, primary_key=True)
    username = Column(String(50), unique=True, nullable=False)
    password = Column(String(50), nullable=False)
    email = Column(String(100), unique=True, nullable=False)

class Order(Base):
    __tablename__ = 'orders'
    id = Column(Integer, primary_key=True)
    user_id = Column(Integer, nullable=False)
    product_id = Column(Integer, nullable=False)
    quantity = Column(Integer, nullable=False)

class Product(Base):
    __tablename__ = 'products'
    id = Column(Integer, primary_key=True)
    name = Column(String(100), nullable=False)
    price = Column(Float, nullable=False)
    stock = Column(Integer, nullable=False)

engine = create_engine('sqlite:///electronic_commerce.db')
Base.metadata.create_all(engine)
Session = sessionmaker(bind=engine)
session = Session()
```

## 4.3 业务逻辑实现

接下来，我们需要实现业务逻辑。我们将实现以下业务逻辑：

- 用户注册：用户提供用户名、密码、邮箱等信息，系统将创建一个新的用户账户。
- 用户登录：用户提供用户名和密码，系统将验证用户账户的有效性。
- 用户下单：用户选择商品，指定数量，系统将创建一个新的订单。
- 用户查询订单：用户查询自己的订单信息，系统将返回相应的订单信息。

我们将使用以下Python代码来实现这些业务逻辑：

```python
def register(username, password, email):
    user = User(username=username, password=password, email=email)
    session.add(user)
    session.commit()

def login(username, password):
    user = session.query(User).filter_by(username=username, password=password).first()
    if user:
        return True
    else:
        return False

def place_order(user_id, product_id, quantity):
    order = Order(user_id=user_id, product_id=product_id, quantity=quantity)
    session.add(order)
    session.commit()

def query_order(user_id):
    orders = session.query(Order).filter_by(user_id=user_id).all()
    for order in orders:
        product = session.query(Product).filter_by(id=order.product_id).first()
        print(f'订单号：{order.id}, 用户ID：{order.user_id}, 商品ID：{order.product_id}, 数量：{order.quantity}, 商品名称：{product.name}, 商品价格：{product.price}')
```
