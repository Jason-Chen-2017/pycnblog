# 2.核心概念与联系

在计算机视觉中，概率分布是一种描述随机变量取值概率的方法。在图像和视频处理中，概率分布可以用来描述像素值、边缘、形状等特征的分布。概率分布可以帮助我们理解和预测这些特征的行为，从而提高计算机视觉系统的性能。

概率分布可以分为两类：离散概率分布和连续概率分布。离散概率分布用于描述离散随机变量的取值概率，如像素值、颜色等。连续概率分布用于描述连续随机变量的取值概率，如边缘、形状等。

在计算机视觉中，概率分布的应用主要包括以下几个方面：

1. 图像压缩：通过对像素值的概率分布进行建模，可以有效地减少图像数据的量，从而实现图像压缩。
2. 图像分割和段落化：通过对像素值的概率分布进行分析，可以将图像划分为多个区域，从而实现图像分割和段落化。
3. 边缘检测：通过对像素邻域的概率分布进行分析，可以找出图像中的边缘和线条。
4. 形状识别：通过对形状特征的概率分布进行分析，可以识别图像中的不同形状。
5. 目标检测和识别：通过对目标特征的概率分布进行分析，可以识别图像中的目标和目标特征。

## 3.1 高斯分布

高斯分布是一种连续概率分布，用于描述数据集中的中心趋势。高斯分布的概率密度函数为：

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$\mu$ 是均值，$\sigma$ 是标准差。

在计算机视觉中，高斯分布可以用于描述像素值的分布，从而实现图像压缩。具体操作步骤如下：

1. 计算像素值的均值和标准差。
2. 使用高斯分布模型对像素值进行建模。
3. 根据高斯分布模型，将原始像素值转换为压缩后的像素值。

## 3.2 多变量高斯分布

多变量高斯分布是一种连续概率分布，用于描述多个随机变量之间的关系。多变量高斯分布的概率密度函数为：

$$
f(x_1, x_2, \dots, x_n) = \frac{1}{(2\pi)^{\frac{n}{2}}|\Sigma|^{\frac{1}{2}}}e^{-\frac{1}{2}(x-\mu)^T\Sigma^{-1}(x-\mu)}
$$

其中，$\mu$ 是均值向量，$\Sigma$ 是协方差矩阵。

在计算机视觉中，多变量高斯分布可以用于描述像素邻域的概率分布，从而实现边缘检测。具体操作步骤如下：

3. 根据多变量高斯分布模型，计算像素点与边缘的概率。

P(X=k) = \frac{e^{-\lambda}\lambda^k}{k!}

import numpy as np
import cv2
import matplotlib.pyplot as plt

# 读取图像

# 计算像素值的均值和标准差
mean = np.mean(img)
std = np.std(img)

# 使用高斯分布模型对像素值进行建模
def gaussian_distribution(x, mean, std):
    return 1 / (std * np.sqrt(2 * np.pi)) * np.exp(-(x - mean)**2 / (2 * std**2))

# 根据高斯分布模型，将原始像素值转换为压缩后的像素值
compressed_img = np.array([gaussian_distribution(x, mean, std) for x in img]).reshape(img.shape)

# 显示原始图像和压缩后的图像
plt.subplot(1, 2, 1)
plt.imshow(img, cmap='gray')
plt.title('Original Image')

plt.subplot(1, 2, 2)
plt.imshow(compressed_img, cmap='gray')
plt.title('Compressed Image')

plt.show()
