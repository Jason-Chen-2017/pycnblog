# 1.背景介绍

语音特征提取主要包括：

3. 时频域特征提取：如波形分析、梅尔频率泊松化等

语音模型识别主要包括：

1. 隐马尔科夫模型（HMM）：一种基于概率的语音模型，它将语音信号模型为一个隐藏马尔科夫链，通过观测到的语音特征与预训练的HMM进行比较，从而得到最佳匹配的文本信号。
2. 深度学习模型：如卷积神经网络（CNN）、循环神经网络（RNN）、长短期记忆网络（LSTM）等。
