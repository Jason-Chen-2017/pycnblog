## 4.1 Hadoop示例
### 4.1.1 准备数据
首先，我们需要准备一个大数据集，这里我们使用一个包含10亿条数据的文本文件作为示例。

### 4.1.2 编写MapReduce程序
接下来，我们需要编写一个MapReduce程序，该程序将读取数据文件，并计算每个单词的出现次数。

```python
from __future__ import print_function
from pyspark import SparkContext

if __name__ == "__main__":
    sc = SparkContext("local", "WordCount")

    # 读取数据文件
    lines = sc.textFile("input.txt")

    # 将每行数据切分为单词
    words = lines.flatMap(lambda line: line.split(" "))

    # 计算每个单词的出现次数
    word_counts = words.map(lambda word: (word, 1)).reduceByKey(lambda a, b: a + b)

    # 输出结果
    word_counts.saveAsTextFile("output")
```

### 4.1.3 运行Hadoop程序
最后，我们需要运行Hadoop程序，以便在Hadoop集群上执行MapReduce任务。


## 4.3 Flink示例
### 4.3.1 准备数据
首先，我们需要准备一个大数据集，这里我们使用一个包含10亿条数据的文本文件作为示例。

### 4.3.2 编写Flink程序
接下来，我们需要编写一个Flink程序，该程序将读取数据文件，并计算每个单词的出现次数。

```python
from pyflink.datastream import StreamExecutionEnvironment
from pyflink.datastream.connectors import TextOutputFormat

def tokenize(line):
    return [word for word in line.split(" ")]

def count_words(words):
    return {word: words.count(word) for word in words}

if __name__ == "__main__":
    env = StreamExecutionEnvironment.get_execution_environment()

    # 读取数据文件
    data_stream = env.read_text_file("input.txt")

    # 将每行数据切分为单词
    words = data_stream.flat_map(tokenize)

    # 计算每个单词的出现次数
    word_counts = words.map(count_words).key_by("__key__").sum("value")

    # 输出结果
    word_counts.output_for_balanced_sink(
        "output",
        TextOutputFormat().set_coding(None),
    )

    env.execute("WordCount")
```

### 4.3.3 运行Flink程序
最后，我们需要运行Flink程序，以便在Flink集群上执行任务。

```bash
$ cd path/to/flink/bin
$ ./start-cluster.sh
$ ./start-local.sh
$ ./flink run -p 1 localhost:8081 WordCount.py
```
