### 3.1 色彩空间

色彩空间是一个用于表示颜色的三维空间，其中三个维度分别表示红色、绿色和蓝色的分量。色彩空间可以帮助我们理解颜色之间的关系，并选择合适的颜色组合。

色彩空间的数学模型是基于RGB（红、绿、蓝）模型的。RGB模型是一种用于表示颜色的数字模型，其中每个颜色都可以表示为一个由三个数字组成的向量。这三个数字分别表示红色、绿色和蓝色的分量。

### 3.2 色彩对比度

色彩对比度是指两个颜色之间的差异。高对比度的颜色更容易被观察者区分和识别。低对比度的颜色则更难被观察者识别。在数据可视化中，高对比度的颜色可以帮助观察者更好地区分不同的数据点和分组。

色彩对比度的数学模型是基于颜色差异的。颜色差异可以通过以下公式计算：

$$
\Delta E = \sqrt{(L_1 - L_2)^2 + (a_1 - a_2)^2 + (b_1 - b_2)^2}
$$

其中，$\Delta E$是颜色差异，$L_1$和$L_2$是两个颜色的亮度分量，$a_1$和$a_2$是两个颜色的红绿蓝（CIELAB）色彩分量，$b_1$和$b_2$是两个颜色的绿蓝色彩分量。

### 3.3 色盲友好性

色盲友好性是指颜色方案对色盲人群可读性的友好程度。色盲人群无法正确识别一些颜色，因此需要选择一种颜色方案，使得色盲人群也能够理解数据可视化。

色盲友好性的数学模型是基于颜色的鉴别度。鉴别度是指颜色之间的区分能力。高鉴别度的颜色方案更容易被色盲人群区分和识别。

色盲友好性的数学模型是基于颜色的鉴别度。鉴别度可以通过以下公式计算：

$$
D = \frac{\Delta E}{\sigma}
$$

其中，$D$是鉴别度，$\Delta E$是颜色差异，$\sigma$是颜色方差。

接下来，我们需要选择一种颜色方案。我们可以选择一种色彩空间，例如RGB空间，并根据数据的范围选择合适的颜色。为了确保颜色的高对比度，我们可以选择颜色的饱和度较高。为了确保色盲友好性，我们可以选择颜色的鉴别度较高。

以下是一个具体的颜色方案示例：

- -50°C到-40°C：红色
- -40°C到-30°C：浅红色
- -30°C到-20°C：橙色
- -20°C到-10°C：浅橙色
- -10°C到0°C：黄色
- 0°C到10°C：浅黄色
- 10°C到20°C：黄绿色
- 20°C到30°C：浅黄绿色

- 40°C到50°C：浅绿色

### 4.2 代码实现

我们可以使用Python的matplotlib库来实现这个数据可视化示例。以下是一个简单的代码示例：

```python
import matplotlib.pyplot as plt
import numpy as np

# 生成随机数据
data = np.random.rand(10) - 0.5

# 创建一个颜色列表
colors = ['red', 'orange', 'yellow', 'lime', 'blue', 'purple', 'cyan', 'magenta', 'maroon', 'navy']

# 创建一个图表
plt.figure()

# 为每个数据点绘制一个条形图
for i, d in enumerate(data):
    plt.barh(i, d, color=colors[i])

# 设置图表标签
plt.xlabel('Temperature (°C)')
plt.ylabel('Index')
plt.title('Temperature Variation')

# 显示图表
plt.show()
```

这个代码将生成一个条形图，用于表示地球温度变化。每个条形的颜色都是根据数据的范围选择的。
