# 2. 核心概念与联系

智能交通系统的核心概念包括：大数据、人工智能、互联网、物联网、云计算等。这些技术手段的联系如下：

- 物联网技术可以帮助智能交通系统实现设备互联、数据传感、事件触发等功能，从而实现交通设施的智能化管理。
