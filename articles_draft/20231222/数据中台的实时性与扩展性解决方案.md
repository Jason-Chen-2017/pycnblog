# 2. 核心概念与联系

在数据中台中，实时性和扩展性是两个关键要素。实时性指的是数据处理和传输的速度，扩展性指的是系统能够处理更大量的数据和用户请求。为了实现这两个目标，我们需要了解一些核心概念和技术。

## 2.1 数据流处理

数据流处理（Data Stream Processing）是一种处理实时数据的技术，它允许我们在数据到达时进行处理，而不需要等待所有数据都到达。这种处理方式对于实时应用非常重要，因为它可以确保数据的时效性。

数据流处理通常使用一种称为“流”（Stream）的数据结构，流是一种不断到达的数据序列。流处理系统通常包括一个或多个处理器，它们可以对流进行过滤、转换、聚合等操作。

## 2.2 分布式系统

分布式系统（Distributed System）是一种将多个计算节点连接在一起的系统，这些节点可以在不同的位置或网络中。分布式系统可以提供高可用性、高扩展性和高性能等优势。

在数据中台中，分布式系统可以帮助我们实现数据的高可用性和高扩展性。通过将数据和处理任务分布在多个节点上，我们可以确保系统能够处理更多的数据和请求。

## 2.3 数据库

数据库（Database）是一种用于存储和管理数据的系统。数据库可以是关系型数据库（Relational Database）或非关系型数据库（Non-Relational Database）。

在数据中台中，数据库可以用于存储和管理数据。关系型数据库通常用于处理结构化数据，而非关系型数据库通常用于处理非结构化数据。
