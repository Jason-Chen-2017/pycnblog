### 4.1 无向图的半正定核矩阵计算

```python
import networkx as nx
import numpy as np

# 构建无向图
G = nx.Graph()
G.add_edges_from([(1, 2), (1, 3), (2, 3), (2, 4), (3, 4), (3, 5)])

# 计算关系强度
relationship_strength = nx.graph_from_edgelist(G.edges())

# 构建半正定核矩阵
K = np.zeros((G.number_of_nodes(), G.number_of_nodes()))
for i, j in G.edges():
    K[i-1][j-1] = relationship_strength[i][j]

# 计算指标
centrality = nx.degree_centrality(G)
clustering_coefficient = nx.average_clustering(G)
```
