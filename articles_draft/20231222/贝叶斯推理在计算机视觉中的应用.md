# 2.核心概念与联系
贝叶斯推理是一种基于概率论的推理方法，它主要通过计算条件概率来得出结论。在计算机视觉中，贝叶斯推理可以帮助我们解决许多问题，例如目标检测、图像分类、对象识别等。

贝叶斯推理的核心概念有两个：条件概率和贝叶斯定理。条件概率是指给定某个事件已发生，另一个事件的概率。贝叶斯定理则是用于计算条件概率的公式，它可以帮助我们更好地理解和解决问题。

在计算机视觉中，贝叶斯推理的应用主要有以下几个方面：

1.图像分类：通过计算图像中各种对象的概率，从而将图像分类到不同的类别。

2.目标检测：通过计算图像中各种目标的概率，从而识别和定位图像中的目标。

3.对象识别：通过计算图像中各种对象的概率，从而识别并识别图像中的对象。
