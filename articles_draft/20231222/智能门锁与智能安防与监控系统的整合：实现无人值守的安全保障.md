```python
def get_alarm_info(url, api_key):
```

```python
import base64

def decrypt_data(encoded_data, key):
    decoded_data = base64.b64decode(encoded_data).decode()
    return decoded_data
```

最后，我们需要设置数据解析：

```python
# 数据解析

    return json.loads(data)
```
