## 3.1 线性回归
线性回归是一种用于预测连续变量的方法。它的基本思想是找到一个最佳的直线，使得该直线可以最好地拟合数据。

### 3.1.1 算法原理
线性回归的算法原理是通过最小化误差来找到最佳的直线。误差是指预测值与实际值之间的差异。我们需要找到一个最佳的直线，使得误差的平方和最小。

### 3.1.2 具体操作步骤
1. 首先，我们需要准备一个数据集，包括一个输入变量（X）和一个输出变量（Y）。
2. 然后，我们需要计算输入变量和输出变量之间的平均值。
3. 接下来，我们需要计算输入变量和输出变量之间的协方差。
4. 最后，我们需要计算输入变量和输出变量之间的相关系数。

### 3.1.3 数学模型公式详细讲解
线性回归的数学模型公式如下：

Y = b0 + b1 * X

其中，Y是输出变量，X是输入变量，b0是截距，b1是斜率。

## 3.2 逻辑回归
逻辑回归是一种用于预测分类变量的方法。它的基本思想是找到一个最佳的分界线，使得该分界线可以最好地分隔数据。

### 3.2.1 算法原理
逻辑回归的算法原理是通过最大化概率来找到最佳的分界线。我们需要找到一个最佳的分界线，使得数据在两侧的概率最大。

### 3.2.2 具体操作步骤
1. 首先，我们需要准备一个数据集，包括一个输入变量（X）和一个输出变量（Y）。
2. 然后，我们需要将输入变量和输出变量转换为标准化的形式。
3. 接下来，我们需要计算输入变量和输出变量之间的协方差。
4. 最后，我们需要计算输入变量和输出变量之间的相关系数。

### 3.2.3 数学模型公式详细讲解
逻辑回归的数学模型公式如下：

P(Y=1|X) = sigmoid(b0 + b1 * X)

其中，P(Y=1|X)是输出变量为1的概率，sigmoid是sigmoid函数，b0是截距，b1是斜率。

## 3.3 梯度下降
梯度下降是一种用于优化函数的方法。它的基本思想是通过不断地更新参数，使得函数的梯度最小。

### 3.3.1 算法原理
梯度下降的算法原理是通过计算函数的梯度，然后更新参数，使得梯度最小。我们需要找到一个最佳的参数，使得函数的梯度最小。

### 3.3.2 具体操作步骤
1. 首先，我们需要准备一个数据集，包括一个输入变量（X）和一个输出变量（Y）。
2. 然后，我们需要计算输入变量和输出变量之间的梯度。
3. 接下来，我们需要更新参数，使得梯度最小。

### 3.3.3 数学模型公式详细讲解
梯度下降的数学模型公式如下：

X = X - α * ∇f(X)

其中，X是参数，α是学习率，∇f(X)是函数的梯度。

## 4.3 梯度下降
```python
import numpy as np

# 准备数据
X = np.array([[1, 2], [3, 4], [5, 6]])
Y = np.array([2, 4, 6])

# 初始化参数
X = np.array([[1, 2], [3, 4], [5, 6]])
b = np.array([0, 0])

# 学习率
alpha = 0.01

# 迭代次数
iterations = 1000

# 梯度下降
for i in range(iterations):
    # 计算梯度
    grad = np.dot(X.T, np.dot(X, b) - Y)

    # 更新参数
    b = b - alpha * grad

# 预测
pred = np.dot(X, b)
```
