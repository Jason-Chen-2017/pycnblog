1. 在需要捕获异常的代码块中使用try关键字，将可能抛出异常的代码放在try代码块中。
2. 在try代码块中，如果发生异常，程序会自动跳转到与try代码块对应的catch代码块。
3. 在catch代码块中，可以捕获异常并进行处理。可以使用throw new Exception()语句来抛出异常，也可以使用try-catch-finally语句来捕获异常。

## 4.1 异常处理的具体代码实例

以下是一个Java异常处理的具体代码实例：

```java
public class ExceptionExample {
    public static void main(String[] args) {
        try {
            int[] numbers = {1, 2, 3, 4, 5};
            System.out.println(numbers[5]);
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.println("Array index out of bounds exception occurred");
        } finally {
            System.out.println("Finally block executed");
        }
    }
}
```

在这个代码实例中，我们使用try-catch-finally语句来捕获ArrayIndexOutOfBoundsException异常。当我们尝试访问数组numbers中的第6个元素时，会抛出ArrayIndexOutOfBoundsException异常。在catch代码块中，我们捕获这个异常并输出一条错误信息。在finally代码块中，我们输出一条消息，表示finally代码块已经执行完成。

## 6.1 异常处理常见问题与解答

Q：如何捕获多个异常？

A：可以使用多个catch代码块来捕获多个异常。例如：

```java
try {
    // 可能会抛出异常的代码
} catch (Exception1 e1) {
    // 捕获异常1并进行处理
} catch (Exception2 e2) {
    // 捕获异常2并进行处理
} finally {
    // 执行一些不受异常控制的代码
}
```

Q：如何抛出自定义异常？

A：可以使用throw语句来抛出自定义异常。例如：

```java
public class CustomException extends Exception {
    public CustomException(String message) {
        super(message);
    }
}

try {
    throw new CustomException("自定义异常抛出");
} catch (CustomException e) {
    // 捕获自定义异常并进行处理
}
```

## 6.2 调试常见问题与解答

Q：如何设置断点？

A：可以使用调试器的设置断点功能来设置断点。例如，在Eclipse中，可以使用鼠标点击代码左边的行号，然后选择“设置断点”。

Q：如何使用调试器逐步执行程序？

A：可以使用调试器的步入、步出、步过等功能来逐步执行程序。例如，在Eclipse中，可以使用“步入”（step into）功能来进入方法的内部，“步出”（step out）功能来退出当前方法，“步过”（step over）功能来跳过当前方法的内部。
