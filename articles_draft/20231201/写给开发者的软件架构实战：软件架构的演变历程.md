## 4.1 模块化设计实例

在这个实例中，我们将使用Python语言来实现一个简单的模块化设计。我们将创建一个名为`math`的模块，该模块包含两个函数：`add`和`sub`。

```python
# math.py
def add(a, b):
    return a + b

def sub(a, b):
    return a - b
```

然后，我们将创建一个名为`main`的模块，该模块使用`math`模块中的函数来进行加法和减法运算。

```python
# main.py
import math

def main():
    a = 10
    b = 5
    print(math.add(a, b))
    print(math.sub(a, b))

if __name__ == '__main__':
    main()
```

在这个实例中，我们将`math`模块和`main`模块分开，以便更容易地理解和维护。

## 4.2 层次化设计实例

在这个实例中，我们将使用Python语言来实现一个简单的层次化设计。我们将创建一个名为`model`的层次，该层次包含两个类：`Person`和`Student`。

```python
# model.py
class Person:
    def __init__(self, name):
        self.name = name

    def say_hello(self):
        print('Hello, my name is', self.name)

class Student(Person):
    def __init__(self, name, grade):
        super().__init__(name)
        self.grade = grade

    def say_hello(self):
        print('Hello, my name is', self.name, 'and I am a', self.grade, 'student.')
```

然后，我们将创建一个名为`main`的层次，该层次使用`model`层次中的类来创建和使用对象。

```python
# main.py
from model import Person, Student

def main():
    person = Person('Alice')
    person.say_hello()

    student = Student('Bob', 'freshman')
    student.say_hello()

if __name__ == '__main__':
    main()
```

在这个实例中，我们将`model`层次和`main`层次分开，以便更容易地开发和维护。
