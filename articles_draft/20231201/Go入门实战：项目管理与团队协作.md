## 4.1 并发编程实例
```go
package main

import (
    "fmt"
    "sync"
)

func main() {
    var wg sync.WaitGroup
    wg.Add(2)

    go func() {
        fmt.Println("Hello")
        wg.Done()
    }()

    go func() {
        fmt.Println("World")
        wg.Done()
    }()

    wg.Wait()
}
```
在这个例子中，我们使用了Go语言的并发编程功能来实现两个goroutine的并发执行。我们使用了sync.WaitGroup来等待所有goroutine完成后再继续执行。

## 4.2 模块化实例
```go
package main

import (
    "fmt"
    "os"
)

func main() {
    fmt.Println("Hello, World!")
    fmt.Println(os.Args)
}
```
在这个例子中，我们使用了Go语言的模块化功能来实现一个简单的命令行应用程序。我们使用了os.Args来获取命令行参数。

## 4.3 依赖管理实例
```go
package main

import (
    "fmt"
    "log"
    "os"

    "github.com/urfave/cli"
)

func main() {
    app := cli.NewApp()
    app.Name = "Hello World"
    app.Usage = "A simple hello world application"

    app.Commands = []cli.Command{
        {
            Name:  "hello",
            Action: func(c *cli.Context) error {
                fmt.Println("Hello, World!")
                return nil
            },
        },
    }

    if err := app.Run(os.Args); err != nil {
        log.Fatal(err)
    }
}
```
在这个例子中，我们使用了Go语言的依赖管理功能来实现一个命令行应用程序。我们使用了github.com/urfave/cli库来实现命令行解析和处理。

## 4.4 测试驱动开发实例
```go

    "testing"

    result := Add(2, 3)
    if result != 5 {
```

在这个例子中，我们使用了Go语言的测试驱动开发功能来实现一个简单的测试用例。我们使用了testing.T来实现测试用例的断言和错误处理。
