4.2 邮件发送代码实例：
```python

# 创建SMTP对象
smtp = smtplib.SMTP("smtp.example.com", 587)

# 启用TLS加密
smtp.starttls()

# 登录SMTP服务器
smtp.login("your_username", "your_password")

# 发送邮件
message = "Subject: Hello\n\nHello, World!"
smtp.sendmail("your_email@example.com", "recipient@example.com", message)

# 关闭SMTP服务器
smtp.quit()
```
