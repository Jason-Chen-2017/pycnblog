# 1.背景介绍


## 一、课程简介
### 1.1 开课背景及其影响
2019年9月2日，“计算机科学与技术（Computer Science and Technology，简称CS）”专业正式开设在香港中文大学。作为人工智能、机器学习、深度学习等高新科技领域的先驱，香港中文大学是国内高等院校中同时拥有最强AI和最多资源的语言大学。截至目前，该专业共设有八个学部。

根据国家对计算机科学与技术的定义，"计算机科学与技术"是指利用信息技术进行计算的系统工程、理论、方法和原理的研究。因此，该专业倡导并实践对计算机的系统性、规范化和高效率的利用，提升计算机应用的能力，促进人类智慧文明的进程。

近几年来，随着数字经济的崛起，越来越多的企业将人工智能、机器学习、深度学习等技术应用到业务中。为了帮助企业解决复杂的问题，并且取得更好的成果，相关的技术手段已经逐渐成为企业竞争力的一项重要组成部分。然而，如何正确地运用这些技术不仅需要丰富的理论知识和技术能力，更要配合丰富的工程实践能力和创造性思维。

香港中文大学的计算机科学与技术专业，以交叉学科的多学科视角，以基础知识为主线，结合计算机科学、应用数学、计算理论、工程技术、软件工程、人工智能等多个学科进行综合培养。通过不断探索、整合、实践，学生能够掌握面向对象编程、数据结构、数据库、算法设计、操作系统、网络通信、体系结构、编译原理、人工智能、模式识别等多学科的核心技术。相信这样的专业训练能够帮助学生具备深厚的专业素质、广阔的道路前景和无限的求知欲。

5．了解并掌握数据库理论、数据库系统、数据挖掘、信息检索、自然语言处理等领域的最新技术和理论；

6．具有机器学习、深度学习等人工智能方向的知识、技能，并有良好的应用能力；

7．具有较强的英语阅读、书面表达、口头交流等能力；

# 2.核心概念与联系
## 一、计算机概述
### 2.1 计算机的定义及分类
计算机是一种基于程序的电子装置，它由机械硬件、电子元器件和固态硬盘三大基本组成部分构成，完成特定的计算功能。它可以执行存储在计算机内或外的数据和程序，可以通过用户输入、输出的方式与外界沟通。按照功能不同，计算机又分为不同的种类，如：

- 个人计算机(PC)：是一种单板计算机，集成了集成电路、运算器、控制器、内存、输入输出设备等设备，可作为个人使用或者用于局域网网络服务。
- 服务器计算机(SCADA)：是一种嵌入式计算机，主要用来运行工控、过程控制、能源监测、仪表自动化等数字化过程控制系统。
- 智能手机：是一种嵌入式计算机，集成了操作系统、处理器、摄像头、麦克风、屏幕等传感器与硬件单元，可以携带人体形态签名等生物特征信息，能够识别用户的行为习惯和喜好。
- 游戏主机：是一种游戏机的外壳，主要由微处理器、内存、硬盘、显示屏等组成，可以通过网络、串口、光驱接口等方式与主机上的玩家互动。
- 微型计算机(MCU): 是一种嵌入式计算机，小型、低功耗，通常集成于消费电子产品、移动终端、车载电子设备、机器人等系统中。
- 超级计算机(HC)：是一种基于芯片的计算机，由若干节点组成，节点间通过互联网络连接，能够同时处理庞大的任务量。

除了上述定义外，还有一些通用的概念：

- 数据(Data)：计算机所处理的信息、文字、图像、声音、视频等符号的集合。
- 指令(Instruction)：由CPU执行的基本操作指令，指令系统负责解释和执行指令。
- 程序(Program)：用某种高级语言编写的可执行指令序列，指导计算机完成特定功能的程序，也叫做软件。
- 操作系统(Operating System)：控制程序执行、分配系统资源、提供公共服务的系统软件，包括命令提示符、文件管理、存储管理、设备管理、文件I/O、作业调度、错误处理等。

### 2.2 计算机的组成及基本概念
#### 2.2.1 CPU与计算机之间的关系
计算机系统中最关键的部分之一就是中央处理器(Central Processing Unit,CPU)，它是一个采用二进制指令集的程序执行引擎，负责指令的解译、执行和结果的输出。由于CPU的作用是进行计算，所以CPU的设计和性能直接影响着计算机的运算速度、处理能力、处理能力。一般来说，CPU的制造商都会将功能相同且性能相似的CPU放在一起比较，以便用户选择合适的配置。CPU的主要组成如下图所示:


从上图可以看到，CPU由以下几个组件构成：

1. 指令单元：负责指令的解译、执行和结果的输出。指令单元的功能是分析并执行程序中的指令。指令可以是算术、逻辑、控制、数据传输等指令。

2. 寄存器：存储CPU的运算和地址。寄存器可以分为：

   - 数据寄存器：专门用来存储数据，可以划分为整数寄存器(IR)、浮点寄存器(FR)、条件码寄存器(CCR)和变址寄存器(AR)。
   - 指令寄存器：存储正在被执行的指令。
   - 地址寄存器：存储数据的地址。
   
3. 缓存：CPU和主内存之间的数据传递和存取速度差异较大，缓存的作用是减少对主内存的访问。它分为指令缓存和数据缓存。

    - 指令缓存：存储指令，当CPU执行指令时，首先从指令缓存中取出指令，然后再从主存中取出数据。
    - 数据缓存：存储数据，当CPU读写数据时，首先从缓存中取出数据，然后再从主存中写入数据。
    
4. 总线：CPU和其他部件通过总线进行信息交换。总线是一条双绞铜管，通过一条导线互连两端的所有设备，信号线的电压范围通常为0~3.3V或0~5V。总线分为数据总线和地址总线。

    - 数据总线：数据总线的宽度通常为32位、64位或128位。
    - 地址总线：地址总线的宽度依赖于CPU的位数。
    
#### 2.2.2 计算机网络的定义及结构
计算机网络是指利用通信技术实现的分布式计算机的集合，每个计算机都通过网络与其他计算机进行连接，计算机网络可以跨越多个城市甚至全球各地。一般来说，计算机网络由节点、链路、路由器、交换机、控制器等组成，每台计算机均处于网络之中，通过交换机连接在一起，实现数据传输、信息共享、应用协同等功能。

计算机网络的结构如下图所示:


从上图可以看出，计算机网络主要由以下几个模块构成：

1. 主机：指可以进行通信的计算机，可以是一台、多台甚至是整个网络。

2. 链路层：负责主机之间的数据传输，主要包括：

    - MAC层：负责数据链路层的帧的发送与接收。MAC层把网络层传下来的IP数据报装配成帧，并封装成符合介质传输的比特流。
    - LAN(Local Area Network)层：为在一个小区域内使用的网络，通常包括许多计算机，链路层工作在物理层，而不在数据链路层以上。
    - WAN(Wide Area Network)层：为跨越很远距离的两个网络之间的通信，链路层工作在物理层，通常使用专门的信号传输介质。
    - 以太网：一种数字传输标准，它规定了主机和交换机之间、交换机和交换机之间的数据传输方式。

3. 互联网层：规定计算机之间如何相互连接，以及数据如何在网络上传输。互联网层一般包括：

    - IP层：规定了在互联网上进行数据通信的规则。它把数据报文分割成单位长度的数据包，每个数据包都有一个目的地址。
    - ICMP(Internet Control Message Protocol)：提供互联网控制消息协议，ICMP报文是IP数据报无法到达时的通知消息。
    - ARP(Address Resolution Protocol)：把IP地址转换成MAC地址的协议。
    - RARP(Reverse Address Resolution Protocol)：把MAC地址转换成IP地址的协议。
    
4. 传输层：定义了两个实体之间通过怎样的传输方式进行通信。传输层协议有TCP、UDP和SCTP等。
    
    TCP(Transmission Control Protocol)：一种可靠的、面向连接的、基于字节流的传输层协议。它建立在IP协议之上，提供可靠的、按序到达的数据传输服务。
    
    UDP(User Datagram Protocol)：一种不可靠的、无连接的、基于数据报的传输层协议。它只提供不保证数据顺序到达的数据传输服务。
    
5. 应用层：定义了应用程序如何使用网络。主要有HTTP、FTP、SMTP、DNS、Telnet等。
