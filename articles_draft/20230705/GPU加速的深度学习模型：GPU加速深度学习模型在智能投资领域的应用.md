### 4.3. 核心代码实现

以下是一个使用TensorFlow 2.x版本和Boston Housing数据集来实现上述示例的代码实现。
```python
import numpy as np
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

# 准备数据
df = keras.datasets.boston_housing.load_data()

# 准备模型
inputs = keras.Input(shape=(13,))
model = keras.models.Sequential([
    layers.Dense(64, activation='relu', input_shape=inputs),
    layers.Dense(32, activation='relu'),
    layers.Dense(1)
])

# 编译模型
model.compile(optimizer='adam', loss='mean_squared_error', metrics=['accuracy'])

# 训练模型
model.fit(df.data, df.target, epochs=100, batch_size=32)

# 评估模型
df.plot(x='target', y='value', data=df.data, labels=['Actual'], color='red')
df.plot(x='predict', y='value', data=df.data, labels=['Predict'], color='blue')
```
在上面的代码中，我们首先使用numpy和tensorflow库来读取Boston Housing数据集中的股票价格数据和目标数据。然后，我们创建一个神经网络模型，使用ReLU激活函数，并使用平均平方误差作为损失函数。我们使用Adam优化器来训练模型，并在训练过程中使用20%的训练数据来进行验证。最后，我们在训练集和测试集上对模型进行评估，并使用实际股票价格和预测股票价格来绘制图表。
