### 4.2 应用实例分析

假设有一个社交网络，其中包含 50 个节点，每个节点表示一个用户，每个边表示这两个用户之间的交互（如“用户 A 喜欢用户 B 的电影”）。我们可以使用相关性学习来分析这个社交网络，以了解用户之间的喜好。

下面是一个相关性学习的 Python 代码示例：

```python
import numpy as np
import pandas as pd
from scipy.stats import pearsonr

# 准备数据
users = pd.read_csv('users.csv')
movies = pd.read_csv('movies.csv')

# 构建社交网络
network = users[['user_id','movie_id']].join(movies['movie_id'])

# 提取节点和边
nodes, edges = network.shape

# 提取节点特征
features = users[['user_id','movie_id', 'rating']].drop('rating', axis=1)

# 提取节点之间的关系
relations = edges.apply(lambda x: x[1][x[0]]).dropna()

# 计算相关性
correlation = pearsonr(relations['rating'], features['rating'])

# 输出结果
print('相关性系数:
', correlation)
```

在这个代码中，我们首先使用 Pandas 库读取用户和电影数据，然后使用 Scikit-learn 库中的 pearsonr 函数来计算相关性。最后，我们将结果输出到控制台上。
