### 4.1 代码实例

以下是一个简单的嵌入式系统加密解密示例：

```c
#include <stdio.h>
#include <string.h>
#include <openssl/aes.h>

int main() {
    char plaintext[] = "Hello, World!";
    char key[] = "12345678";
    char ciphertext[16];

    // 加密
    AES_KEY aes_key;
    AES_set_encrypt_key(key, 128, &aes_key);
    AES_encrypt(plaintext, ciphertext, &aes_key);

    // 解密
    AES_set_decrypt_key(key, 128, &aes_key);
    AES_decrypt(ciphertext, plaintext, &aes_key);

    printf("Plaintext: %s\n", plaintext);
    printf("Ciphertext: %s\n", ciphertext);

    return 0;
}
```

### 4.2 详细解释说明

在这个代码实例中，我们使用了OpenSSL库来实现AES加密解密。首先，我们定义了一个明文字符串`plaintext`和一个密钥字符串`key`。然后，我们使用AES_set_encrypt_key函数设置加密密钥，并使用AES_encrypt函数进行加密。接着，我们使用AES_set_decrypt_key函数设置解密密钥，并使用AES_decrypt函数进行解密。最后，我们打印出明文和密文。

这个代码实例展示了如何在嵌入式系统中实现最佳的安全策略。通过使用OpenSSL库，我们可以轻松地实现AES加密解密，从而保护敏感数据和控制系统的安全性。
