### 3.3 数学模型公式详细讲解

在本节中，我们将详细讲解社交媒体数据分析的数学模型公式。

#### 3.3.1 中心性

中心性是用于衡量节点在社交网络中的重要性的一个指标。中心性可以分为度中心性、路径中心性和 clos 中心性等。

1. 度中心性：度中心性是用于衡量节点的连接程度的一个指标。度中心性可以计算为节点的度（即节点的邻居数）的平均值。

2. 路径中心性：路径中心性是用于衡量节点在社交网络中的中心性的一个指标。路径中心性可以计算为节点到其他节点的最短路径的平均值。

3. clos 中心性：clos 中心性是用于衡量节点在社交网络中的中心性的一个指标。clos 中心性可以计算为节点到其他节点的最短路径的最大值。

#### 3.3.2 聚类

聚类是用于将相似节点组合在一起的一个方法。聚类可以分为基于距离的聚类、基于密度的聚类、基于随机游走的聚类等。

1. 基于距离的聚类：基于距离的聚类是一种将相似节点组合在一起的方法。基于距离的聚类可以计算为节点之间的距离的平均值。

2. 基于密度的聚类：基于密度的聚类是一种将相似节点组合在一起的方法。基于密度的聚类可以计算为节点之间的密度的平均值。

3. 基于随机游走的聚类：基于随机游走的聚类是一种将相似节点组合在一起的方法。基于随机游走的聚类可以计算为节点之间的随机游走的平均值。

#### 3.3.3 关联规则

关联规则是用于发现数据之间的关联关系的一个方法。关联规则可以计算为节点之间的关联关系的概率。

关联规则可以计算为：

$$
P(A \cup B) = P(A) + P(B) - P(A \cap B)
$$

其中，$P(A \cup B)$ 表示节点 A 和节点 B 之间的关联关系的概率，$P(A)$ 表示节点 A 的概率，$P(B)$ 表示节点 B 的概率，$P(A \cap B)$ 表示节点 A 和节点 B 的交集的概率。

### 4.2 构建社交网络

我们可以使用 NetworkX 库来构建社交网络：

```python
import networkx as nx

G = nx.Graph()

for row in df.iterrows():
    G.add_node(row[1]['user_id'], user_id=row[1]['user_id'])
    G.add_edge(row[1]['user_id'], row[1]['friend_id'], weight=row[1]['weight'])
```

### 4.3 分析社交网络

我们可以使用 NetworkX 库来分析社交网络：

```python
import networkx as nx

centrality = nx.degree_centrality(G)
closeness = nx.closeness_centrality(G)
clustering = nx.clustering(G)

print('degree_centrality:', centrality)
print('closeness_centrality:', closeness)
print('clustering_coefficient:', clustering)
```
