### 4.1 使用Matplotlib绘制条形图

Matplotlib是一个流行的数据可视化工具，它提供了一系列的绘图函数，可以用于绘制各种类型的图表和图形。

首先，我们需要导入Matplotlib库：

```python
import matplotlib.pyplot as plt
```

然后，我们需要创建一个数据集：

```python
data = [5, 10, 15, 20, 25]
```

接下来，我们需要使用Matplotlib的bar函数绘制条形图：

```python
plt.bar(range(len(data)), data)
```

最后，我们需要使用show函数显示条形图：

```python
plt.show()
```

### 4.2 使用Seaborn绘制箱线图

