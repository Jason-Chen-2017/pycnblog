假设我们需要分配农业资源，以实现农业资源的节约。我们有以下约束条件：

- 水资源的总量为 $1000$ 立方米
- 气候条件为适宜农业生产

我们需要最小化以下目标函数：

$$
\min \quad c^T x \\
s.t. \quad Ax \leq b \\
x \geq 0
$$

其中，$c = [1, 1]$，$A = \begin{bmatrix} 1 & 1 \end{bmatrix}$，$b = 1000$，$x = \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}$。

具体的Python代码实现如下：

```python
from scipy.optimize import linprog

c = [1, 1]
A = [[1, 1]]
b = 1000
x0 = [0, 0]

result = linprog(c, A_ub=A, b_ub=b, bounds=[(0, None), (0, None)], options={'disp': True})
print(result)
```
