## 4.1 量子场的代码实例

```python
import numpy as np

# 定义一个量子场的波函数
def quantum_field(x):
    return np.exp(-x**2)

# 计算量子场的值

y = quantum_field(x)

# 绘制量子场的波函数
import matplotlib.pyplot as plt
plt.plot(x, y)
plt.xlabel('x')
plt.ylabel('y')
plt.title('Quantum Field')
plt.show()
```

import numpy as np

# 定义一个量子操作的矩阵
def quantum_operation(x):
    return np.array([[np.exp(-x**2), np.cos(x)], [-np.cos(x), np.exp(-x**2)]])
