- 晶体研究：晶体研究是研究晶体结构和性能的科学。晶体结构是晶体的基本结构单元，它们决定了晶体的性能。通过研究晶体结构，材料科学家可以为各种行业提供更好的材料选择。

- 液体研究：液体研究是研究液体性质和性能的科学。液体是一种流动的物质，它们可以被压缩和流动。液体研究可以为各种行业提供更好的材料选择。

- 电子材料：电子材料是一种可以用来制造电子产品的材料。电子材料可以为各种行业提供更好的材料选择。

    return current / (voltage * area)
