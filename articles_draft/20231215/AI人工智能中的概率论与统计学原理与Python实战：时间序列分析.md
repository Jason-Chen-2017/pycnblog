## 3.1 概率论基础

### 3.1.1 概率的基本定理

概率的基本定理是概率论中最基本的数学定理，它表示了两个独立事件的概率之积。

$$
P(A \cup B) = P(A) + P(B) - P(A \cap B)
$$

### 3.1.2 条件概率

条件概率是给定另一个事件已经发生的时候，一个事件发生的可能性。条件概率可以表示为：

$$
P(A|B) = \frac{P(A \cap B)}{P(B)}
$$

### 3.1.3 贝叶斯定理

贝叶斯定理是一种概率推理方法，它可以用来更新我们对事件的信念。贝叶斯定理可以表示为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

## 3.2 统计学基础

### 3.2.1 均值、方差和标准差

均值是随机变量的数学期望，它表示随机变量的平均值。方差是随机变量的数学方差，它表示随机变量的离散程度。标准差是方差的平方根，它表示随机变量的离散程度的度量。

$$
\mu = E[X] = \sum_{i=1}^{n} x_i p(x_i)
$$

$$
\sigma^2 = Var[X] = E[X^2] - (E[X])^2
$$

$$
\sigma = \sqrt{Var[X]}
$$

### 3.2.2 相关性和协方差

相关性是两个随机变量之间的数学关系，它表示两个随机变量之间的关系。协方差是两个随机变量之间的数学关系，它表示两个随机变量之间的关系。

$$
\rho(X,Y) = \frac{Cov(X,Y)}{\sigma_X \sigma_Y}
$$

$$
Cov(X,Y) = E[(X - \mu_X)(Y - \mu_Y)]
$$

### 3.2.3 最大似然估计

最大似然估计是一种用于估计参数的方法，它基于数据的概率密度。最大似然估计可以表示为：

$$
\hat{\theta} = \arg \max_{\theta} L(\theta)
$$

### 3.2.4 信息论

信息论是一种用于衡量信息的方法，它可以用来衡量数据的不确定性和熵。熵是信息论中的一个重要概念，它表示数据的不确定性。

$$
H(X) = -\sum_{i=1}^{n} p(x_i) \log p(x_i)
$$

## 4.1 均值、方差和标准差

```python
import numpy as np

# 生成随机数列
x = np.random.normal(loc=0, scale=1, size=1000)

# 计算均值
mean = np.mean(x)
print("Mean:", mean)

# 计算方差
variance = np.var(x)
print("Variance:", variance)

# 计算标准差
std_dev = np.std(x)
print("Standard Deviation:", std_dev)
```

## 4.2 相关性和协方差

```python
import numpy as np

# 生成随机数列
x = np.random.normal(loc=0, scale=1, size=1000)
y = np.random.normal(loc=0, scale=1, size=1000)

# 计算协方差
covariance = np.cov(x, y)
print("Covariance:", covariance)

# 计算相关性
correlation = np.corrcoef(x, y)[0, 1]
print("Correlation:", correlation)
```

## 4.3 最大似然估计

```python
import numpy as np

# 生成随机数列
x = np.random.normal(loc=0, scale=1, size=1000)

# 定义最大似然估计函数
def likelihood(theta):
    return -np.sum((x - theta)**2) / (2 * len(x))

# 计算最大似然估计
theta_hat = np.argmax(likelihood(np.linspace(-5, 5, 100)))
print("Maximum Likelihood Estimate:", theta_hat)
```

## 4.4 信息论
