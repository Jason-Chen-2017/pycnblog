## 1.5 智能家居的具体代码实例

以下是一个简单的智能家居系统的代码实例：

```python
import time
import random

class SmartHomeSystem:
    def __init__(self):
        self.devices = []

    def add_device(self, device):
        self.devices.append(device)

    def get_device_status(self):
        status = {}
        for device in self.devices:
            status[device.name] = device.get_status()
        return status

    def control_device(self, device_name, command):
        device = self.get_device(device_name)
        if device:
            device.control(command)

    def get_device(self, device_name):
        for device in self.devices:
            if device.name == device_name:
                return device
        return None

class Device:
    def __init__(self, name):
        self.name = name

    def get_status(self):
        return random.choice(["ON", "OFF"])

    def control(self, command):
        if command == "ON":
            self.status = "ON"
        elif command == "OFF":
            self.status = "OFF"

# 创建智能家居系统
smart_home_system = SmartHomeSystem()

# 添加家居设备
smart_home_system.add_device(Device("Light"))
smart_home_system.add_device(Device("AirConditioner"))

# 获取家居设备状态
device_status = smart_home_system.get_device_status()
print(device_status)

# 控制家居设备
smart_home_system.control_device("Light", "ON")
smart_home_system.control_device("AirConditioner", "ON")
```

这个代码实例中，我们定义了一个`SmartHomeSystem`类，用于管理家居设备。我们还定义了一个`Device`类，用于表示家居设备。通过这个代码实例，我们可以看到如何创建一个简单的智能家居系统，并控制家居设备进行执行。
