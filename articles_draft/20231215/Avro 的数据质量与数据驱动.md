## 4.1 数据清洗

我们将通过一个简单的数据清洗示例来演示如何使用 Avro 来清洗数据。

```python
from avro.io import DatumReader
from avro.datafile import DataFileReader

# 读取数据文件
reader = DataFileReader(open("data.avro", "rb"), DatumReader(schema=SCHEMA))

# 去除重复数据
data = list(reader)
data = list(set(data))

# 填充缺失值
for record in data:
    if record["age"] is None:
        record["age"] = 0

# 转换数据类型
for record in data:
    record["age"] = int(record["age"])

# 编码数据
for record in data:
    record["age"] = record["age"].encode("utf-8")
```

在这个示例中，我们首先读取了 Avro 文件，然后使用了去除重复数据、填充缺失值、转换数据类型和编码数据等功能来清洗数据。
