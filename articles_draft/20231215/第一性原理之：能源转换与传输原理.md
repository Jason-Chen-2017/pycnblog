## 4.具体代码实例和详细解释说明

### 4.1 能源转换代码实例

以下是一个简单的能源转换代码实例，将热能转换为电能：

```python
import numpy as np

def heat_to_electricity(heat, efficiency):
    """
    将热能转换为电能
    :param heat: 热能（J）
    :param efficiency: 转换效率（-1~1）
    :return: 电能（J）
    """
    return heat * efficiency

if __name__ == '__main__':
    heat = 1000  # 热能（J）
    efficiency = 0.3  # 转换效率
    electricity = heat_to_electricity(heat, efficiency)
    print("电能：", electricity, "J")
```

### 4.2 能源传输代码实例

以下是一个简单的能源传输代码实例，将电能传输到远端：

```python
import numpy as np

def electricity_transport(electricity, distance, power_loss):
    """
    将电能传输到远端
    :param electricity: 电能（J）
    :param distance: 传输距离（m）
    :param power_loss: 传输损失（-1~1）
    :return: 传输后的电能（J）
    """
    return electricity / (1 - power_loss)

if __name__ == '__main__':
    electricity = 1000  # 电能（J）
    distance = 1000  # 传输距离（m）
    power_loss = 0.1  # 传输损失
    transport_electricity = electricity_transport(electricity, distance, power_loss)
    print("传输后的电能：", transport_electricity, "J")
```
