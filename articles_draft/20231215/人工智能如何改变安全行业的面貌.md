- 梯度下降的Python实现：

```python
import numpy as np

# 损失函数
def loss(x, y, theta):
    return np.sum((x * theta - y) ** 2)

# 梯度
def grad(x, y, theta):
    return 2 * np.dot(x, theta - y)

# 初始化模型参数
theta = np.array([0, 0])

# 学习率
alpha = 0.01

# 训练数据
X = np.array([[1, 1], [1, 2], [2, 1], [2, 2]])
y = np.array([1, 2, 1, 2])

# 训练模型
for i in range(1000):
    grad_theta = grad(X, y, theta)
    theta = theta - alpha * grad_theta

# 预测
pred = theta.dot(X)
print(pred)
```

- 随机梯度下降的Python实现：

```python
import numpy as np

# 损失函数
def loss(x, y, theta):
    return np.sum((x * theta - y) ** 2)

# 梯度
def grad(x, y, theta):
    return 2 * np.dot(x, theta - y)

# 初始化模型参数
theta = np.array([0, 0])

# 学习率
alpha = 0.01

# 训练数据
X = np.array([[1, 1], [1, 2], [2, 1], [2, 2]])
y = np.array([1, 2, 1, 2])

# 训练模型
for i in range(1000):
    # 随机选择一个训练数据样本
    idx = np.random.randint(0, len(X))
    grad_theta = grad(X[idx], y[idx], theta)
    theta = theta - alpha * grad_theta

# 预测
pred = theta.dot(X)
print(pred)
```
