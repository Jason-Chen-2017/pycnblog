## 2.1 事件
事件是概率论中的一个基本概念，它表示某个特定的结果或结果集合。例如，在抛硬币的例子中，事件可以是“硬币正面”或“硬币反面”。

## 2.2 样本空间
样本空间是概率论中的一个基本概念，它表示所有可能的结果集合。例如，在抛硬币的例子中，样本空间是“硬币正面”和“硬币反面”的集合。

## 2.3 概率
概率是概率论中的一个基本概念，它表示事件发生的可能性。概率通常用数字0到1表示，表示事件发生的可能性。例如，在抛硬币的例子中，硬币正面的概率是0.5，硬币反面的概率也是0.5。

## 2.4 条件概率
条件概率是概率论中的一个基本概念，它表示事件发生的可能性，给定另一个事件已经发生。例如，在抛硬币的例子中，硬币正面的条件概率给定硬币已经抛出，是0.5。

## 2.5 独立事件
独立事件是概率论中的一个基本概念，它表示两个事件发生的可能性，不受另一个事件的影响。例如，在抛硬币的例子中，硬币正面和硬币反面是独立的事件，它们之间的发生不受影响。

## 3.1 概率的计算
### 3.1.1 直接计算概率
直接计算概率是一种简单的方法，我们可以直接计算事件发生的可能性。例如，在抛硬币的例子中，硬币正面的概率是0.5，硬币反面的概率也是0.5。

### 3.1.2 统计方法计算概率
统计方法是一种通过收集数据和计算统计量来计算概率的方法。例如，我们可以通过收集大量硬币抛出的数据，计算硬币正面和硬币反面的概率。

## 3.2 条件概率的计算
### 3.2.1 直接计算条件概率
直接计算条件概率是一种简单的方法，我们可以直接计算事件发生的可能性，给定另一个事件已经发生。例如，在抛硬币的例子中，硬币正面的条件概率给定硬币已经抛出，是0.5。

### 3.2.2 贝叶斯定理计算条件概率
贝叶斯定理是一种通过计算先验概率和后验概率来计算条件概率的方法。贝叶斯定理的公式是：
$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$
其中，$P(A|B)$ 是条件概率，$P(B|A)$ 是条件概率，$P(A)$ 是先验概率，$P(B)$ 是后验概率。

## 3.3 独立事件的计算
### 3.3.1 直接计算独立事件
直接计算独立事件是一种简单的方法，我们可以直接计算两个事件发生的可能性，不受另一个事件的影响。例如，在抛硬币的例子中，硬币正面和硬币反面是独立的事件，它们之间的发生不受影响。

### 3.3.2 乘法定理计算独立事件
乘法定理是一种通过计算每个事件的概率并乘积来计算独立事件的方法。乘法定理的公式是：
$$
P(A \cap B) = P(A)P(B)
$$
其中，$P(A \cap B)$ 是两个事件发生的概率，$P(A)$ 是事件A的概率，$P(B)$ 是事件B的概率。

## 4.1 直接计算概率
我们可以使用Python的random模块来直接计算概率。例如，我们可以使用random.choice()函数来随机选择一个事件，然后计算事件发生的概率。

```python

# 定义事件
events = ["硬币正面", "硬币反面"]

# 随机选择一个事件

# 计算事件发生的概率

print("事件发生的概率为：", probability)
```
