## 4.数学模型和公式详细讲解举例说明

### 4.1数学模型

一个基本的细胞自动机由一个二维矩阵组成，其中每个单元格可以具有有限数量的状态。每个单元格的状态在每个时间步由其邻居的状态决定。以下是一个基本的数学模型：

$$
s(t+1) = f(s(t), s_{neigh}(t))
$$

其中，$s(t)$表示单元格在时间步$ t $的状态，$s_{neigh}(t)$表示其邻居的状态，$f$表示状态更新规则。

### 4.2公式举例

我们将使用一个简单的例子，即一维细胞自动机来说明。假设我们有一个长度为$ n $的单元格数组$ s $，其中每个单元格可以具有0或1两种状态。状态更新规则如下：

1. 如果单元格的状态为0，则在下一个时间步中，状态将保持不变。
2. 如果单元格的状态为1，则在下一个时间步中，状态将变为0。

我们可以使用以下Python代码来实现这个例子：

```python
def cellular_automata(s):
    n = len(s)
    s_new = s.copy()
    for i in range(n):
        if s[i] == 1:
            s_new[i] = 0
        else:
            s_new[i] = s[i]
    return s_new
```
