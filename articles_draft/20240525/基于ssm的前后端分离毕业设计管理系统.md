## 1. 背景介绍

随着软件开发的不断发展，前后端分离已经成为一种流行的开发模式。这种模式可以提高开发效率，降低系统的复杂性，使得前端和后端的开发人员可以分别专注于自己的领域。SSM（Spring、Spring MVC、MyBatis）是一种强大的Java框架，能够让开发人员更轻松地实现前后端分离。我们将基于SSM来构建一个毕业设计管理系统。

## 2. 核心概念与联系

毕业设计管理系统是一个用于管理学生毕业设计的系统，包括学生信息、课程信息、教师信息和毕业设计信息等。系统的核心概念包括：

* 学生信息：包括学生ID、姓名、性别、班级等。
* 课程信息：包括课程ID、课程名称、教师ID等。
* 教师信息：包括教师ID、姓名、职务等。
* 毕业设计信息：包括毕业设计ID、学生ID、课程ID、标题、摘要、提交时间等。

这些信息之间相互关联，构成了一种复杂的关系网络。我们需要通过后端编程来实现这些信息的存储、查询、更新和删除等功能。

// 用户登录
public boolean login(String username, String password) {
    String md5Password = MD5Util.md5(username + password);
    Student student = studentDao.findStudentByUserName(username);
    if (student != null && md5Password.equals(student.getPassword())) {
        return true;
    }
    return false;
}

// 数据库查询
public List<Course> findAllCourses() {
