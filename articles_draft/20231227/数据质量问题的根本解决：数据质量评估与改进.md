假设我们有一个包含学生成绩的数据集，我们需要检查这些成绩的准确性。首先，我们需要比较这些成绩与来源的统计特性，如平均值、方差和相关系数。

```python

import pandas as pd

# 加载数据
data = pd.read_csv('student_scores.csv')

# 计算平均值
average = data['score'].mean()

# 计算方差
variance = data['score'].var()

# 计算相关系数
correlation = data['score'].corr(data['gender'])
```

### 4.1.2 数据统计
