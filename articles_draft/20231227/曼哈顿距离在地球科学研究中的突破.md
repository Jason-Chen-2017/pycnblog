# 2.核心概念与联系

曼哈顿距离是一种简单的距离计算方法，它只考虑水平和垂直的距离。给定两个点（x1, y1）和（x2, y2），曼哈顿距离（Manhattan distance）可以通过以下公式计算：

$$
d = |x1 - x2| + |y1 - y2|

在本节中，我们将通过一个具体的代码实例来展示曼哈顿距离在地球科学研究中的应用。

假设我们有一个地理信息系统（GIS）中的两个地理对象，它们的坐标 respectively are (x1, y1) and (x2, y2)。我们需要计算它们之间的曼哈顿距离。

首先，我们需要将这两个坐标转换为二维坐标系，即（x, y）。然后，我们可以使用曼哈顿距离公式计算它们之间的距离：

$$
d = |x1 - x2| + |y1 - y2|
$$

具体的Python代码实现如下：

```python
import math

def manhattan_distance(x1, y1, x2, y2):
    return abs(x1 - x2) + abs(y1 - y2)

x1, y1 = 1, 2
x2, y2 = 3, 4

distance = manhattan_distance(x1, y1, x2, y2)
print("曼哈顿距离：", distance)
```

在这个例子中，我们首先导入了`math`模块，然后定义了一个名为`manhattan_distance`的函数，该函数接受四个参数（x1, y1, x2, y2），并返回它们之间的曼哈顿距离。最后，我们调用这个函数并打印了结果。
