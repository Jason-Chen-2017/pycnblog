# 4.具体代码实例和详细解释说明
## 4.1 代码实例
以下是一个简单的能源管理系统的代码实例：

```python
import paho.mqtt.client as mqtt
import numpy as np
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans

# 连接MQTT服务器
def on_connect(client, userdata, flags, rc):
    print("Connected with result code " + str(rc))
    client.subscribe("energy/data")

# 数据处理
def on_message(client, userdata, msg):
    data = np.array(msg.payload.decode().split(','))
    df = pd.DataFrame(data, columns=["time", "power", "voltage", "current"])
    df["time"] = pd.to_datetime(df["time"])
    df = df.set_index("time")
    scaler = StandardScaler()
    scaled_df = scaler.fit_transform(df)
    kmeans = KMeans(n_clusters=3)
    kmeans.fit(scaled_df)
    print(kmeans.labels_)

client = mqtt.Client()
client.on_connect = on_connect
client.on_message = on_message
client.connect("mqtt.eclipse.org", 1883, 60)
client.loop_forever()
```

## 4.2 详细解释说明
上述代码实例主要包括以下部分：

1. 连接MQTT服务器：通过`paho-mqtt`库，我们可以连接到MQTT服务器，并订阅能源数据的主题。
2. 数据处理：当收到能源数据时，我们将数据解析并转换为DataFrame格式。然后，使用`StandardScaler`对数据进行标准化处理。接着，使用`KMeans`算法对数据进行聚类分析，并输出聚类结果。
