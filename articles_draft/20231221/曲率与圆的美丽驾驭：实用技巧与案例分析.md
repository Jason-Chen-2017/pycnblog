## 2.1 曲率与圆的基本概念

### 2.1.1 曲率的定义

曲率是描述一个曲线在某一点上弧度和方向的量。在计算机图形学中，曲率是计算曲线的形状和特性的关键因素之一。曲率可以通过以下公式计算：

$$
k = \frac{||\vec{T}'(t)||}{||\vec{r}'(t)||}
$$

其中，$\vec{r}(t)$ 是曲线的参数表示，$\vec{r}'(t)$ 是曲线的速度向量，$\vec{T}(t)$ 是速度向量的单位法向量，$\vec{T}'(t)$ 是单位法向量的切向导数。

### 2.1.2 圆的定义

圆是一种特殊的曲线，它的每一点到其中心的距离是相等的。圆的定义可以通过以下公式表示：

$$
(x - a)^2 + (y - b)^2 = r^2
$$

其中，$(a, b)$ 是圆心，$r$ 是半径。

## 2.2 曲率与圆的核心概念与联系

### 2.2.1 圆的曲率

圆是一种特殊的曲线，它的曲率在任何点上都是恒定的。根据圆的定义，圆的曲率可以通过以下公式计算：

$$
k = \frac{1}{r}
$$

其中，$r$ 是圆的半径。

### 2.2.2 圆的椭圆曲率

椭圆是一种特殊的曲线，它的曲率在任何点上都是恒定的，但是曲率的方向会发生变化。椭圆的定义可以通过以下公式表示：

$$
\frac{(x - a)^2}{r_1^2} + \frac{(y - b)^2}{r_2^2} = 1
$$

其中，$(a, b)$ 是椭圆的中心，$r_1$ 和 $r_2$ 是椭圆的长半轴和短半轴。椭圆的曲率可以通过以下公式计算：

$$

$$

其中，$\theta$ 是椭圆与x轴的夹角。

### 2.2.3 圆与曲率的关系

圆和曲率之间的关系是密切的。在圆的定义中，圆的每一点到其中心的距离是相等的，这意味着圆的曲率在任何点上都是恒定的。这使得圆在计算机图形学中成为了一个重要的基本形状，因为它可以通过简单的数学公式来描述和计算。

```python
import numpy as np

def curvature(r, r_prime):
    T = r_prime / np.linalg.norm(r_prime)
    T_prime = np.diff(T) / np.linalg.norm(np.diff(T))
    k = np.linalg.norm(T_prime) / np.linalg.norm(r_prime)
    return k

r_prime = np.array([[1, 0], [1, 2], [1, 3], [1, 4]])

print(k)
```
