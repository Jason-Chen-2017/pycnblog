以下是一个使用Python的SHA256哈希算法的代码实例：

```python
import hashlib

def hash_password(password):
    salt = os.urandom(16)
    hashed_password = hashlib.pbkdf2_hmac('sha256', password.encode('utf-8'), salt, 100000)
    return salt + hashed_password

def verify_password(password, hashed_password):
    salt = hashed_password[:16]
    hashed_password = hashed_password[16:]
    return hashlib.pbkdf2_hmac('sha256', password.encode('utf-8'), salt, 100000) == hashed_password

password = "my_password"
hashed_password = hash_password(password)
print(verify_password(password, hashed_password))
```

在这个例子中，我们使用了SHA256哈希算法来哈希密码。首先，我们导入了SHA256哈希算法，然后定义了`hash_password`和`verify_password`函数。这两个函数分别用于哈希和验证密码。最后，我们使用这两个函数来哈希和验证一段文本。
