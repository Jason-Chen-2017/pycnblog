## 4.1 哈希分布实例

### 4.1.1 代码实例

```python
import hashlib

def hash_distribution(data, storage_num):
    hash_function = hashlib.sha256()
    hash_table = {}
    for i in range(storage_num):
        hash_table[i] = []
    for key, value in data.items():
        hash_function.update(str(key).encode('utf-8'))
        hash_value = hash_function.hexdigest()
        storage_id = int(hash_value, 16) % storage_num
        hash_table[storage_id].append((key, value))
    return hash_table
```

### 4.1.2 解释说明

在这个代码实例中，我们使用了哈希分布算法来实现数据扩展性的可维护性与可靠性。首先，我们使用了哈希函数`hashlib.sha256()`来计算数据键的哈希值。然后，我们将哈希值对应的存储设备ID存储在`hash_table`字典中。最后，我们将数据键值对存储在对应的存储设备ID的列表中。
