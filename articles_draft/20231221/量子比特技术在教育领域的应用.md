### 1.4.1 量子幂指数法（QPE）代码实例

```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile, assemble

# 定义量子电路
qc = QuantumCircuit(2, 2)

# 初始化量子态
qc.initialize([1, 0], 0)
qc.initialize([0, 1], 1)

# 应用量子门
qc.h(0)

# 绘制量子电路
qc.draw()

# 运行量子电路
simulator = Aer.get_backend('qasm_simulator')
qobj = assemble(transpile(qc, simulator), shots=1024)
result = simulator.run(qobj).result()

# 获取结果
counts = result.get_counts()
print(counts)
```

上述代码实例展示了量子幂指数法（QPE）的具体应用。在此示例中，我们定义了一个量子电路，并将其运行在QASM模拟器上。最后，我们获取了运行结果，并将其打印到控制台。
