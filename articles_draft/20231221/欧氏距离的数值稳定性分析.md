## 1.背景介绍

欧氏距离是一种基于欧几里得空间的距离度量，用于计算两点之间的距离。它在各种计算机视觉、机器学习和数据挖掘等领域具有广泛的应用。欧氏距离的定义为：

$$
d(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2 + \cdots + (x_n - y_n)^2}
$$

其中，$x = (x_1, x_2, \ldots, x_n)$ 和 $y = (y_1, y_2, \ldots, y_n)$ 是两个 $n$-维向量，$d(x, y)$ 是它们之间的欧氏距离。

2. 计算每个坐标之间的差值：

$$
\Delta_i = x_i - y_i, \quad i = 1, 2, \ldots, n
$$

3. 计算每个差值的平方：

$$
\Delta_i^2 = \Delta_i \cdot \Delta_i, \quad i = 1, 2, \ldots, n

$$
\sum_{i=1}^n \Delta_i^2 = \sum_{i=1}^n (\Delta_i \cdot \Delta_i)
$$

5. 计算欧氏距离：

$$
d(x, y) = \sqrt{\sum_{i=1}^n \Delta_i^2} = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}
$$

```python
import numpy as np

def euclidean_distance(x, y):
    """
    计算两点欧氏距离
    
    Parameters:
    x (array): 第一个点的坐标
    y (array): 第二个点的坐标
    
    Returns:
    float: 两点欧氏距离
    """
    # 计算每个坐标之间的差值
    delta = x - y
    # 计算每个差值的平方
    delta_squared = np.square(delta)
    # 计算所有差值平方的和
    sum_delta_squared = np.sum(delta_squared)
    # 计算欧氏距离
    distance = np.sqrt(sum_delta_squared)
    return distance

# 测试数据
x = np.array([1, 2, 3])
y = np.array([4, 5, 6])

# 计算欧氏距离
distance = euclidean_distance(x, y)
print("欧氏距离:", distance)
```
