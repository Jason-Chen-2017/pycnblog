## 4.1 滤波示例

### 4.1.1 低通滤波器实现

```python
import numpy as np

def lowpass_filter(x, fc, fs):
    N = 2 * np.pi * fc / fs
    H = np.ones(len(x)) / (1 + 1j * N * np.arange(len(x)) / len(x))
    y = np.convolve(x, H, mode='valid')
    return y
```

### 4.1.2 高通滤波器实现

```python
import numpy as np

def bandpass_filter(x, fc1, fc2, fs):
    N1 = 2 * np.pi * fc1 / fs
    N2 = 2 * np.pi * fc2 / fs
    H = (1j * N1 * np.ones(len(x))) / (1 + 1j * (N1 - N2) * np.arange(len(x)) / len(x))
    y = np.convolve(x, H, mode='valid')
    return y
```

## 4.3 信号解析示例

### 4.3.1 傅里叶变换实现

```python
import numpy as np

def fft_transform(x):
    N = len(x)
    X = np.fft.fft(x)
    f = np.fft.fftfreq(N, d=1/N)
    Y = X / N
    return Y, f
```

### 4.3.2 傅里叶逆变换实现

```python
import numpy as np

def ifft_transform(X, N):
    x = np.fft.ifft(X)

```
