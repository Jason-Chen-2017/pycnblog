# 2.核心概念与联系

在进行环境数据分析之前，我们需要了解一些核心概念。这些概念包括：

- 卫星数据：卫星数据是指从地球卫星上收集的数据，包括远程感知数据和导航数据。远程感知数据包括地面图像、海洋图像、大气图像等。导航数据包括地球磁场、地球形状等。
- 海洋变化：海洋变化是指海洋水位、温度、水质等指标的变化。这些变化可能是由于自然因素（如地貌变化、气候变化）或人类活动（如海洋污染、海洋捕捞）引起的。
- 环境数据分析：环境数据分析是指对环境数据进行处理和分析，以便于了解环境变化和预测未来的环境状况。

在进行环境数据分析时，我们需要使用一些算法来处理和分析数据。这些算法包括：

接下来，我们需要加载卫星数据。我们将使用 Landsat 卫星数据来分析海洋变化：

```python
landsat = ee.ImageCollection("LANDSAT/LC08/C01/T1_SR")
```

接下来，我们需要设置分析区域：

```python
region = ee.Geometry.Rectangle([110, 15, 130, 30])
```

接下来，我们需要设置时间范围：

```python
start_date = "2018-01-01"
end_date = "2018-12-31"
```

接下来，我们需要设置分析参数：

```python
cloud_threshold = 30
```

接下来，我们需要过滤云晴天数据：

```python
filtered_data = landsat.filterDate(start_date, end_date).filterMetadata("cloud_coverage_percent", "less_than", cloud_threshold)
```

接下来，我们需要计算海洋温度和水位：

```python
temp_data = filtered_data.map(lambda image: image.select("SR_B5").gsub(10000, 300).divide(10000))
```

接下来，我们需要将数据下载到本地：

```python
temp_data = ee.batch.Export.image.toDrive(
    description="temp_data",
    folder="temp_data",
    fileNamePrefix="temp_data",
    scale=30,
    region=region,
    fileFormat="GeoTIFF",
    maxPixels=1e13
)

temp_data.start()
```

在上面的代码中，我们首先导入了必要的库，并设置了 Google Earth Engine 的项目信息。接下来，我们加载了 Landsat 卫星数据，并设置了分析区域和时间范围。接下来，我们过滤了云晴天数据，并计算了海洋温度和水位。最后，我们将数据下载到本地。
