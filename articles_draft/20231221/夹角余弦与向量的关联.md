# 2.核心概念与联系

## 2.1 夹角余弦的定义与性质

夹角余弦是两个向量之间的一个度量标准，用于衡量它们之间的夹角。夹角余弦的定义为：

$$
\cos(\theta) = \frac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\| \|\mathbf{b}\|}
$$

其中，$\mathbf{a}$ 和 $\mathbf{b}$ 是两个向量，$\cdot$ 表示向量点乘，$\|\mathbf{a}\|$ 和 $\|\mathbf{b}\|$ 分别表示向量 $\mathbf{a}$ 和 $\mathbf{b}$ 的长度。

夹角余弦具有以下性质：

1.  $-1 \leq \cos(\theta) \leq 1$，表示夹角范围从 $180^\circ$ 到 $0^\circ$。
2.  $\cos(180^\circ - \theta) = -\cos(\theta)$，表示垂直和平行的向量之间的夹角余弦分别为正和负。
3.  $\cos(0^\circ) = 1$，表示两个向量是平行的。
4.  $\cos(180^\circ) = -1$，表示两个向量是垂直的。

## 2.2 夹角余弦与向量的关联

夹角余弦与向量之间的关联主要体现在以下几个方面：
