```python
import random

# 原始数据集
data = [(1, 'is_a', 2), (2, 'is_a', 3), (3, 'is_a', 4)]

# 随机替换一些数据
for i in range(len(data)):
    for j in range(len(data)):
        if i != j:
            data[i] = data[j]

# 生成新的数据集
new_data = []
for d in data:
    new_data.append((d[0], d[1], d[2]))

print(new_data)
```

在上面的代码中，我们首先导入了 `random` 模块，然后定义了一个原始数据集 `data`。接着，我们使用一个嵌套循环随机替换一些数据，并将剩余的数据添加到新的数据集 `new_data` 中。最后，我们打印出新的数据集。
