## 4.1 排序算法实例
我们选择快速排序（Quick Sort）作为示例，它是一种常用的排序算法。快速排序的基本思想是：通过选择一个基准元素，将数组分为两部分，一部分元素小于基准元素，一部分元素大于基准元素，然后递归地对两部分元素进行排序。

以下是快速排序的伪代码：
```
function quick_sort(arr, low, high) {
    if (low < high) {
        // 选择基准元素
        pivot = partition(arr, low, high);
        // 递归地对两部分元素进行排序
        quick_sort(arr, low, pivot - 1);
        quick_sort(arr, pivot + 1, high);
    }
}

function partition(arr, low, high) {
    // 选择基准元素
    pivot = arr[high];
    i = low - 1;
    // 对于非基准元素，如果小于基准元素，则交换位置
    for (j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(arr, i, j);
        }
    }
    // 基准元素的最终位置
    swap(arr, i + 1, high);
    return i + 1;
}

function swap(arr, i, j) {
    temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
}
```
