# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

在回归分析中，我们通常使用以下几种标准化处理方法：

1. 均值标准化：将因变量和自变量的值减去均值，然后除以标准差。
2. 最小-最大归一化：将因变量和自变量的值乘以一个范围（通常是0-1），使其落在0-1之间。
3. 对数转换：将因变量和自变量的值替换为对数。

以下是具体的数学模型公式：

均值标准化：
$$
z = \frac{x - \mu}{\sigma}
$$

最小-最大归一化：
$$
z = \frac{x - \min}{\max - \min}
$$

对数转换：
$$
z = \log(x)
$$

import numpy as np

# 创建一个数据框
data = {
    '因变量': [1, 2, 3, 4, 5],
    '自变量': [10, 20, 30, 40, 50]

# 计算均值和标准差
