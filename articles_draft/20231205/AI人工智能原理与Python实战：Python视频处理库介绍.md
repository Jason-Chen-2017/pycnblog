# 2.核心概念与联系

在进行视频处理之前，我们需要了解一些核心概念和联系。

## 2.1 视频的基本概念

视频是一种动态的多媒体内容，由一系列的图像（帧）组成。每一帧都是一个二维的图像，包含了视频中的所有信息。视频的帧率（frames per second，FPS）是指每秒钟播放的帧数，通常情况下，视频的帧率为24-60帧每秒。视频的分辨率（resolution）是指视频的宽度和高度，例如1080p（1920x1080）、4K（3840x2160）等。

## 2.2 视频处理的核心概念

视频处理的核心概念包括：

- 帧提取：从视频中提取出每一帧的图像，以便进行后续的处理。
- 帧处理：对每一帧的图像进行处理，例如旋转、翻转、裁剪等。
- 帧融合：将多个帧进行融合，以生成新的视频内容。
- 帧分析：对每一帧的图像进行分析，以提取出有意义的信息，例如人脸识别、物体检测等。

## 2.3 人工智能与机器学习的联系

人工智能和机器学习是相互联系的。机器学习是人工智能的一个子分支，它研究如何让计算机从数据中学习，以便进行预测、分类和决策等任务。深度学习是机器学习的一个子分支，它研究如何利用多层神经网络来处理复杂的问题。

人工智能和机器学习的联系可以从以下几个方面来看：

- 人工智能可以使用机器学习技术来进行决策和预测。
- 机器学习可以利用人工智能技术来提高其性能和效率。
- 深度学习可以用于处理复杂的人工智能和机器学习问题。

for frame in frames:
    # 对图像进行旋转
