### 4.2 拦截器实例

以下是一个简单的Go语言拦截器实例，它实现了一个简单的日志记录功能：

```go
package main

import (
	"fmt"
)

type Interceptor interface {
	Intercept(msg string) string
}

type SimpleInterceptor struct{}

func (i *SimpleInterceptor) Intercept(msg string) string {
	return fmt.Sprintf("Intercepted: %s", msg)
}

type Logger struct {
	interceptor Interceptor
}

func (l *Logger) Log(msg string) {
	msg = l.interceptor.Intercept(msg)
	fmt.Println(msg)
}

func main() {
	interceptor := &SimpleInterceptor{}
	logger := &Logger{interceptor}
	logger.Log("Hello, World!")
}
```

在这个实例中，我们首先定义了一个Interceptor接口，它包含一个Intercept方法。然后，我们实现了一个SimpleInterceptor结构体，它实现了Interceptor接口。最后，我们创建了一个Logger结构体，它包含一个Interceptor成员变量，并调用其Log方法来记录日志。
