# 2.核心概念与联系

在讨论文件系统损坏恢复技术之前，我们需要了解一些基本的概念。文件系统是操作系统中用于管理文件和目录的数据结构，它负责将文件存储在磁盘上，并提供了一种逻辑上的组织方式。文件系统损坏是指文件系统结构或数据发生了错误，导致文件系统无法正常工作。

文件系统损坏恢复技术的核心是通过检查文件系统的结构和数据，并尝试修复损坏的部分。这种技术可以分为两种类型：一种是基于文件系统的元数据（如文件名、目录结构等）进行恢复，另一种是基于文件系统的数据块进行恢复。
