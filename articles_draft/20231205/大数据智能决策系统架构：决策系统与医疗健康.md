### 4.1代码实例

我们将通过一个简单的医疗健康决策系统来详细解释大数据智能决策系统的实现过程。这个决策系统的核心功能是根据患者的医疗记录、生物标志物、基因数据等大量数据，来进行疾病诊断和治疗方案推荐。

我们将使用Python语言来实现这个决策系统。以下是这个决策系统的具体代码实例：

```python
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
from sklearn.decomposition import PCA
from sklearn.feature_selection import SelectKBest
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

# 数据预处理
data = pd.read_csv('medical_data.csv')
data = data.dropna()

# 数据清洗
data = data.drop_duplicates()

# 数据去除噪声
data = data.replace(to_replace='', value=None)

# 数据填充缺失值
data = data.fillna(data.mean())

# 数据分析
X = data.drop('disease', axis=1)
y = data['disease']

# 数据聚类
kmeans = KMeans(n_clusters=3)
kmeans.fit(X)

# 数据降维
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)

# 数据特征选择
selector = SelectKBest(k=5)
X_selected = selector.fit_transform(X_pca, y)

# 决策模型构建
X_train, X_test, y_train, y_test = train_test_split(X_selected, y, test_size=0.2, random_state=42)

# 决策模型选择
model = RandomForestClassifier(n_estimators=100, random_state=42)

# 决策模型训练
model.fit(X_train, y_train)

# 决策模型测试
accuracy = model.score(X_test, y_test)
print('Accuracy:', accuracy)
```

这个代码实例中，我们首先使用pandas库来读取医疗数据，并进行数据预处理。数据预处理包括数据清洗、数据去除噪声、数据填充缺失值等操作。

接下来，我们使用sklearn库来进行数据分析。数据分析包括数据聚类、数据降维、数据特征选择等操作。

最后，我们使用RandomForestClassifier模型来进行决策模型构建。决策模型构建包括决策模型选择、决策模型训练、决策模型测试等操作。
