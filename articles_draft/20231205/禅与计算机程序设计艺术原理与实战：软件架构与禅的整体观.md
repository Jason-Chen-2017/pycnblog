## 4.1 算法原理

我们将通过一个简单的排序算法来详细解释算法原理。排序算法是计算机程序设计中的一种常见算法，它可以将一个数据序列按照某种规则重新排列。

### 4.1.1 选择排序

选择排序是一种简单的排序算法，它的基本思想是在每次迭代中选择一个最小（或最大）的元素，并将其放在当前位置。选择排序的时间复杂度为O(n^2)，其中n是输入序列的长度。

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

### 4.1.2 冒泡排序

冒泡排序是一种简单的排序算法，它的基本思想是在每次迭代中将最大（或最小）的元素放在当前位置。冒泡排序的时间复杂度为O(n^2)，其中n是输入序列的长度。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```
