## 4.2 数据清洗代码实例

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 数据缺失值处理
data_filled = data.fillna(data.mean())

# 数据噪声处理
data_filtered = data_filled.drop(data_filled[data_filled > 3 * data_filled.std()].index)

# 数据错误处理
data_corrected = data_filtered.replace({'value': 'correct_value'})
```
