## 4.2 敏捷开发思想的代码实例

在这个代码实例中，我们将通过一个简单的购物车程序来展示敏捷开发思想的应用。

```python
class Product:
    def __init__(self, name, price):
        self.name = name
        self.price = price

    def get_total_price(self, quantity):
        return self.price * quantity

class ShoppingCart:
    def __init__(self):
        self.products = []

    def add_product(self, product, quantity):
        self.products.append((product, quantity))

    def remove_product(self, product):
        for i in range(len(self.products)):
            if self.products[i][0] == product:
                del self.products[i]
                break

    def get_total_price(self):
        total_price = 0
        for product, quantity in self.products:
            total_price += product.get_total_price(quantity)
        return total_price

if __name__ == '__main__':
    cart = ShoppingCart()
    cart.add_product(Product('苹果', 1.5), 5)
    cart.add_product(Product('香蕉', 2), 3)
    cart.remove_product(Product('苹果', 1.5))
    print('购物车总价为：', cart.get_total_price())
```

在这个代码实例中，我们定义了一个购物车程序，包括一个产品类和一个购物车类。我们可以通过添加、移除和获取购物车总价来实现购物车的功能。在这个代码实例中，我们可以看到敏捷开发思想的应用：我们通过快速迭代和团队协作来完成任务，从而更快地满足客户的需求。
