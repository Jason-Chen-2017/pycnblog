# 4.具体代码实例和详细解释说明

下面是一个具体的滑动窗口算法的代码实例：

```python
def sliding_window(arr, k):
    window_start = 0
    window_end = k - 1
    window_sum = sum(arr[window_start:window_end + 1])
    min_window_sum = window_sum

    while window_end < len(arr) - 1:
        window_end += 1
        window_sum += arr[window_end]

        while window_sum >= arr[window_start]:
            window_sum -= arr[window_start]
            window_start += 1

        if window_sum < min_window_sum:
            min_window_sum = window_sum

    return min_window_sum
```

上述代码实例是寻找数组中的最小子数组的滑动窗口算法实现。该算法的时间复杂度为O(n)，空间复杂度为O(1)。

```python
def sliding_window(s, k):
    window_start = 0
    window_end = k - 1
    window_chars = set(s[window_start:window_end + 1])
    unique_chars = len(window_chars)

    while window_end < len(s) - 1:
        window_end += 1
        window_chars.add(s[window_end])
        unique_chars += 1

        while len(window_chars) > k:
            window_chars.remove(s[window_start])
            unique_chars -= 1
            window_start += 1

        if unique_chars < k:
            return s[window_start:window_end + 1]

    return s[window_start:window_end + 1]
```
