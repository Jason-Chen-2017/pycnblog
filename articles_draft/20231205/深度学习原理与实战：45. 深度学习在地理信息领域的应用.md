# 2.核心概念与联系

在深度学习中，我们需要了解以下几个核心概念：

1.神经网络：神经网络是由多个节点（神经元）组成的图，每个节点都有一个输入和一个输出。神经网络可以通过训练来学习一个函数，将输入映射到输出。

2.深度学习：深度学习是一种神经网络的子集，它由多层神经网络组成。每一层神经网络都可以学习不同的特征，从而提高模型的准确性和泛化能力。

3.卷积神经网络（CNN）：卷积神经网络是一种特殊的深度学习模型，它通过卷积层来学习图像的特征。卷积神经网络在图像分类、目标检测等任务中取得了很好的成果。

4.递归神经网络（RNN）：递归神经网络是一种特殊的深度学习模型，它可以处理序列数据。递归神经网络在自然语言处理、时间序列预测等任务中取得了很好的成果。
