# 1.简介
  

C++（通称“CPP”）是一种高级、面向对象的、静态ally typed、结构化的编程语言。它不但可以编写系统级程序，而且还可以编写游戏、GUI和Web应用等应用软件。它的开发者为贝尔实验室的史蒂夫·乔布斯（Dennis Ritchie）。目前，C++已经成为众多领域最受欢迎的编程语言。

2003年，在一个名叫“龙芯”的计算机实验室里，当时只有3个成员，但很快就扩展到7个人，并于同年发布了第一个C++编译器——GNU C Compiler (GCC)。1983年，Bjarne Stroustrup接管该实验室，目前仍然在开发C++。自1998年至今，C++已经成为开发者最喜欢的语言之一。

本文作为系列教程的第一篇，主要介绍C++的基础语法，并通过几个简单的示例带领读者了解C++的基本用法。文章的内容将分成以下几部分：

1. 变量、常量和数据类型
2. 数据输入输出
3. 运算符、条件语句和循环语句
4. 函数、类和面向对象编程
5. STL、字符串处理、多线程等高级特性

# 2.变量、常量和数据类型
## 2.1 什么是变量？
变量是存储数据的占位符，在程序运行过程中可以随时修改其值。在C++中，所有的变量都必须先声明后使用。声明变量一般采用如下格式：
```cpp
<数据类型> <变量名称>;
```
其中，`数据类型`指明变量所保存的数据类型，例如int表示整数型数据；`变量名称`由字母、数字或下划线组成，不能以数字开头。

例如：
```cpp
int age;    // 声明一个整数型变量age
double salary = 5000.0;   // 声明一个浮点型变量salary并初始化赋值
char gender;      // 声明一个字符型变量gender
bool isMarried = false;     // 声明一个布尔型变量isMarried并初始化赋值
```
## 2.2 什么是常量？
常量是固定不变的值。在C++中，使用关键字const定义常量，格式如下：
```cpp
const <数据类型> <常量名称> = <常量值>;
```
其中，`数据类型`和`<常量值>`定义了常量的类型及初始值。常量只能赋值一次，之后无法修改。

例如：
```cpp
const int MAX_AGE = 100;        // 定义了一个整数型最大年龄常量
const double PI = 3.1415926;   // 定义了一个双精度浮点数圆周率常量
const char NAME[] = "John";    // 定义了一个字符串常量，值为"John"
```
## 2.3 数据类型
C++支持的标准数据类型包括：整型、浮点型、布尔型、字符型和指针类型等。下面对这些数据类型进行详细介绍。
### 2.3.1 整型
整型又分为短整型(short)、整数型(int)、长整型(long)、长长整型(long long)四种形式。下面介绍它们的特点和用途。
#### short类型
short类型范围是-32768～32767，占2字节内存空间。适用于频繁访问的数据。通常在做计算或需要存储大量小型整数值的场合使用。

例如：
```cpp
short a = -10;       // 短整型变量a
short b = 20000;     // 短整型变量b
cout << a + b << endl;    // 输出结果:19990
```
#### int类型
int类型范围是-2147483648~2147483647，占4字节内存空间。它是标准的变量类型，通常用于整数运算和索引运算。

例如：
```cpp
int a = -1000000000;          // 整数型变量a
int b = 2000000000;           // 整数型变量b
unsigned int c = 4000000000U;  // 无符号整数型变量c
cout << a * b + static_cast<long long>(c) / 1000 << endl;
                            // 输出结果:-9999999995000000000
```
#### long类型
long类型是整数型的另一种形式，范围是-2^31~2^31-1，占4字节内存空间。当整数较大时可以使用此类型。

例如：
```cpp
long a = -1000000000;         // 长整型变量a
long b = 2000000000;          // 长整型变量b
unsigned long c = 4000000000UL;// 无符号长整型变量c
cout << a * b + static_cast<long long>(c) / 1000 << endl;
                             // 输出结果:-9999999995000000000
```
#### long long类型
long long类型是整数型的第三种形式，范围是-2^63~2^63-1，占8字节内存空间。当整数较大时可以使用此类型。

例如：
```cpp
long long a = -1000000000000000000LL;    // 长长整型变量a
long long b = 2000000000000000000LL;     // 长长整型变量b
unsigned long long c = 4000000000000000000ULL;
                                       // 无符号长长整型变量c
cout << a * b + static_cast<long long>(c) / 1000 << endl;
                                        // 输出结果:-99999999995000000000
```
### 2.3.2 浮点型
浮点型又分为单精度浮点型(float)和双精度浮点型(double)两种形式。下面介绍它们的特点和用途。
#### float类型
float类型是一种单精度浮点型，可以表示小数点后6位有效数字。它的取值范围约为±3.4e+38。

例如：
```cpp
float f = 3.14f;             // 单精度浮点型变量f
double d = f + static_cast<double>(i);
                           // 将整数i转换为浮点型后相加
cout << fixed << setprecision(5) << d << endl;
                          // 使用fixed和setprecision函数控制输出格式
```
#### double类型
double类型是一种双精度浮点型，可以表示小数点后15位有效数字。它的取值范围约为±1.8e+308。

例如：
```cpp
double x = 1.234567891234567;  // 双精度浮点型变量x
float y = static_cast<float>(x);  // 将双精度浮点型变量转换为单精度浮点型y
cout << scientific << setprecision(4) << y << endl;
                                   // 使用scientific和setprecision函数控制输出格式
```
### 2.3.3 bool类型
bool类型是一个值只有true和false两个取值的数据类型。它用来表示逻辑表达式的真假性。

例如：
```cpp
bool flag = true;            // 布尔型变量flag
if (!flag) {                 // 如果flag为false
    cout << "Flag is false." << endl;
} else {                     // 如果flag为true
    cout << "Flag is true." << endl;
}
```
### 2.3.4 字符型
char类型是单个ASCII字符的数据类型。它通常用来表示单个字符或者编码字符集中的某个字符。char类型占用一个字节的内存空间。

例如：
```cpp
char ch1 = 'A';              // ASCII码为65的字符'A'
char ch2[10];                // 存放10个字符的数组
ch2[0] = 'H'; ch2[1] = 'e';  // 初始化数组元素
for (int i=0; i<10; ++i) {
    cout << ch2[i] << " ";
}                           // 输出结果:"H e l l o    "
```
### 2.3.5 指针类型
指针类型是一种特殊的数据类型，它指向某一块内存的首地址。在C++中，指针变量本身占用四个字节的内存空间，它的值存放的是实际指向的内存地址。由于指针类型变量不像其他变量那样直接存储值，所以不需要事先声明大小。

例如：
```cpp
int num = 100;               // 整型变量num
int* p = &num;               // 指针变量p，指向num变量的地址
cout << "*p=" << *p << endl; // 输出结果:*p=100
*p = 200;                    // 修改num的值
cout << "*p=" << *p << endl; // 输出结果:*p=200
```
