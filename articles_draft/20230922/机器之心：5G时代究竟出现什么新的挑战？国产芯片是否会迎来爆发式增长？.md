# 2 5G技术演变历史
## 2.1 5G网络技术的历史发展历程
### 2.1.1 5G技术的定义、分类、功能、架构及应用场景
**5G**（第五代移动通信技术）是指采用新的无线传输技术、信号处理方法、计算系统、存储系统、应用系统等等构建的第三代移动通信技术。它将信息传输速度提升至每秒1兆比特以上，并且使得各种业务活动都能够像平板电脑一样方便快捷地进行。

2019年7月，国际标准化组织（ISOR）正式发布了“第五代移动通信（5G）”的相关标准，该标准将由一系列的技术标准、协议、工具和测试组成。

根据定义，5G可以分为以下三大类：

1. 物理层（Physical Layer）：采用新的通信物理层协议，提升信道频谱宽带。
2. 连接层（Connection Layer）：新增了新的信令控制协议，增加了可靠性和吞吐量。
3. 网络层（Network Layer）：建立新的用户体验，可提供更好的服务质量。

目前5G主要应用于以下几种场景：

1. 大容量广播通信：5G大幅度提升宽带分配效率和支持小容量视频、音频流的传输。
2. 智慧城市：5G通信能够让城市变成一个超级智能、绿色环保的城市。
3. 增强现实、虚拟现实、人工智能、机器人等：人们将拥有更加复杂、真实的环境、虚拟世界、人工智能虚拟助手。
4. 健康医疗卫生：帮助患者享受智能化医疗服务。
5. 工业互联网：促进高速的工业生产，降低生产成本，提升产品品质。
6. 电力与能源：用于远程监控、远程预测和远程管理。
7. 游戏开发：提供全新游戏玩法。

### 2.1.2 5G技术及标准的快速发展
2015年2月，美国NIST（National Institute of Standards and Technology，美国国家标准与技术研究院）发布了3GPP（3rd Generation Partnership Project，第三代合作计划）第四阶段第六次改进的“第五代移动通信(5G)”标准，正式命名为LTE-Advanced+，即运营商级第五代移动通信(5G)标准。2018年5月，欧洲核子中心（ETSI）发布了第五代移动通信系统Evolved Universal Terrestrial Radio Access Network（E-Utra）标准，即全球统一覆盖、综合效能的5G标准。

2019年11月，欧洲核子中心（ETSI）发布了5G专题报告，将5G定义为建立在新的技术基础上，面向大众的开放式、灵活的无线通信网络，旨在实现互联网的升级换代。

2020年1月，美国白宫宣布：白宫已准备在2020年启动新的5G基建投资计划。

### 2.1.3 5G技术的核心组成
根据5G技术的定义，5G网络的核心组成如下图所示。


1. 核心网：**核心网**是5G设备和设备之间交换数据的中心枢纽，也是无论是边缘节点还是骨干节点都能访问的最快网络。核心网的组成是**终端基础设施**、**重构数据中心**、**网络策略和决策引擎**等。
2. 终端基础设施：**终端基础设施**由UE（用户端）、NAS（网络接入服务器）、RAN（Radio Access Network，基站）、NES（Next Enhancing System，下一代增强系统）、HSS（Home Subscriber Server，家庭无线局域网接入服务器）等组成。其中，UE是手机、平板电脑、电视盒子、智能穿戴设备等终端设备；NAS是连接用户端设备的设备，如Wi-Fi路由器；RAN是承载核心网通信的无线电子频段；NES用于语音识别、图像处理等智能服务；HSS是用户身份鉴权的服务器。
3. 数据中心：**数据中心**由计算中心、存储中心、网络中心和安全中心组成。计算中心由机器学习计算资源池、人工智能资源池组成；存储中心由可扩展的分布式存储集群组成；网络中心由高速千兆以太网交换机组成；安全中心由高级防火墙和访问控制系统组成。
4. 网络策略和决策引擎：**网络策略和决策引擎**负责推动核心网的部署、运行和管理。

### 2.1.4 5G关键技术及标准
目前5G技术仍处于蓬勃发展阶段，各方面技术均取得重要进展。5G关键技术及标准如下：

1. LTE-Advanced+标准：**LTE-Advanced+**是5G运营商级标准，其中包括物理层、连接层、MAC层、RLC层、PDCP层、RRC层、PHY层等。

2. 全球第一次独立5G系统无线电频谱：**第一次独立5G系统无线电频谱**为期约10年左右的全球范围内第一次独立申请、使用5G无线电频谱，这是为了应对欧洲天气异常变化、环境恶化、军事威胁等因素带来的影响。

3. ETWS-TM5G：**ETWS-TM5G**（Evolved Trusted Wireless Systems for the Masses）是一项由欧洲核子研究中心(ETSI)主导开发的可持续无线通信技术，打破了5G当前存在的技术瓶颈、规模化难度等问题。

4. AMF3（Adaptive Multi-Rated Frame Structure）：**AMF3**是一个新的适应性多档帧结构，能够有效解决频谱利用率不足的问题。

5. D2D迅速组网通信（Distributed Coordinated Communications）：**D2D迅速组网通信**是一种新的组网方式，可以极大地缩短组网时间，提高通信速度。

6. PCF（Packet Control Function）：**PCF**作为5G的控制单元，能够为RAN中的所有设备提供动态、准确的控制。

7. ABS（Advancing Base Station Technologies）：**ABS**是一种通信系统升级方案，适用于5G初期，将更高的性能带宽和高速数据传输能力引入到5G系统中。

## 2.2 4G到5G技术演变历史
### 2.2.1 4G技术演变历史
**4G**（第三代移动通信技术）是指采用毫米波、微波、光频等技术，且能支持上百万条并发服务的高速移动通信技术。

1990年，中国政府提出用数字通信技术改造现有的固定网络，试图将通信领域从通信方式转向计算机通信领域。随后，中国提出了第一版56K标准。

1997年，中国开发的第一款手机即安卓。

2000年，蜂窝移动通信标准3GPP1已经成为国际标准。

2005年，3GPP推出了第二版56K标准。

2008年，3GPP推出了第三版56K标准，并开始应用于第三代手机。

2012年，3GPP推出了第四版56K标准，并被称为4G标准。

### 2.2.2 4G和5G的区别及联系
不同的是，**4G**是在GSM、CDMA技术基础上发展而来的新一代移动通信技术，而**5G**是第五代移动通信技术。相对于**4G**来说，**5G**提升的主要特点有以下两点：

1. 提升带宽：5G使用毫米波或更高频段的技术，在频段带宽上与4G相比，有显著的提升。

2. 更高速率：5G的用户速率达到了10GBps以上，比4G提升了近十倍。

同时，由于5G采用了不同的物理层、连接层、网络层协议，因此5G也能为个人用户提供更多的便利。

总结起来，**4G**和**5G**之间的差异主要是通信技术的升级和带宽提升，但是两者之间又是有关联的。4G主要应用于固定电话、GSM、CDMA等传统移动通信系统，5G则主要应用于非固定移动通信系统。

另外，中国目前还处于第三代移动通信技术的实践阶段，未来将进入第四代移动通信技术的时代，届时将不断革新和更新。
