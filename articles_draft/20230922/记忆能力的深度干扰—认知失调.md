# 2. 基础概念和术语
## 2.1 认知神经科学与认知科学概览
**认知神经科学**：研究人类的行为、感知、思维、思想过程中的神经网络组织，以及它们在不同认知任务下的作用。它是对大脑机理的研究，包括：大脑结构、功能、运作方式、机制等。

**认知科学**：主要关注对人类各种信息处理、认识与表达行为的学习、记忆、思维、推理、判断和决策的各种规律和机制。它从多个角度研究认知生理、生化、心理、语言、艺术、社会、情绪、情感、动机等方面的各种现象和特征，促使我们开展关于如何塑造和塑造我们的认知、构建和使用我们的思维、预测和分析我们的世界，以及改善我们的生活方式等问题的研究。

## 2.2 记忆的定义和分类
**记忆**：是指对过去或正在发生事件的形成的图像、声音、符号、文字、数字等形式的记录。记忆既可以是事先编码好的知识，也可以是正在被检索、学习和创造出来的。

记忆的分类可分为五种：
- 直接存储记忆(Direct Storage Memory)：指记忆材料存储在后天产生的神经末梢中；
- 间接存储记忆(Indirect Storage Memory)：指记忆材料存储在脑区的细胞或神经元中；
- 联想记忆(Episodic Memory)：指能够跨时间点的回忆；
- 活性神经回路记忆(Active Neural Networking Memories)：指具有激活神经回路的存储模块；
- 路径依赖记忆(Path Dependent Memory)：指跨时空记忆链接起来的一系列记忆模式。

## 2.3 认知失调的定义
**认知失调(Cognitive Dissonance)**，又称认知对立，是指由多种不同来源的信息组合产生的不一致性。在学习过程中，我们的大脑无法清晰准确地获取、处理和整合信息，甚至出现错误、混乱甚至混淆的状况，因而产生认知失调。

认知失调常表现为如下症状：
- 重读困难：我们常遇到这种情况，我们阅读材料时总是只记得一半，另外一半却毫无印象，而且没有留意到重复出现的信息，造成效率低下。
- 错读错误：当我们看到一些材料时，虽然并不十分确定其含义，但由于它已经被我们习惯性地接受，就会产生错误的假设。例如，我们收到了一条广告，上面的某些词语看起来很像学校名字，就可能受此影响而错读其含义。
- 分心思维：即使是最熟悉的材料，当我们把注意力放在其他事务上时，也可能发生错误的想法和思维。例如，我们在阅读或做笔记时，突然想起一些事情，就有可能带着那些思维进入下一个注意点。
- 重复记忆：当我们以错误的方式复述或反复阅读某些材料时，我们可能会产生错误的记忆，进而出现认知上的偏差。

# 3. 记忆机制原理和功能
## 3.1 大脑的记忆机制
### 3.1.1 记忆的基本功能
记忆作为大脑的重要功能之一，其基本功能是把各种信息形成图像、声音、符号等形式的记忆素，并将其永久存储在大脑里，以便后续快速获取、理解和利用。

记忆的过程包括三个阶段：编码、检索和运用。下面我们逐个介绍：

**编码阶段**：在这一阶段，大脑完成对外部刺激的感知，然后将信息转换成神经活动信号，这些信号通过光感受器、味觉触觉器、神经传导通路传送给各个神经元，形成神经元内的运动神经冲动。这一过程称为编码，也是对外部刺激进行认识、加工、抽象、综合、排列、理解的过程。编码后，信息会长期存储在大脑里，这就是刚才提到的各种形式的记忆素。

**检索阶段**：在检索阶段，大脑根据我们记忆中的关键词、事件或材料，通过搜索引擎、学习规则、回忆、想象、符号之间的联系等方式，快速找到相应的神经元，并从这些神经元中获取记忆信息。

**运用阶段**：在运用阶段，大脑从内存中取出所需的材料，并将其呈现出来。例如，当我们需要处理文本材料、听到的声音、看到的图像时，都可以从大脑的记忆库中找到相应的材料并加以利用。

### 3.1.2 注意与兴奋
了解了记忆的基本功能之后，让我们来了解一下为什么人类的大脑容易出现认知失调。一般来说，人类的大脑处于一个比较平静状态，大部分信息都会被无意识地吸收，但是在学习、记忆、运用等各种活动中，人脑会产生两种类型不同的信息输入，它们之间互相抵消、干扰和冲突。

前者通常来自于当前正在进行的任务，如读课本、做题、学习新东西，并且它们往往能获得较大的处理速度，不会引起其它信息的干扰。而后者则来自于过去的经验、直觉、欲望，它们经历了一个漫长的时间积累，可能也比当前正在进行的任务耗费更多的时间，且容易受到其它信息的干扰。

因此，如果想让大脑正确处理信息，就需要从两个方向上考虑：减少当前正在进行的任务的干扰，从而保证任务顺利进行；同时增强过去的经验、直觉、欲望的刺激，从而增加它们对当前任务的影响。这样，才能有效避免认知失调。

注意与兴奋理论认为，人类的大脑在学习、记忆和运用各项任务时，都会产生一种“警觉性”或者“兴奋性”，这种性格对人类的学习和记忆活动至关重要，因为它可以提高学习效率、减少认知失调。

## 3.2 不同记忆模式
人类的大脑有三种记忆模式：
1. **短期记忆(Short-term memory，STM)**：短期记忆指的是人脑中短暂的记忆单元，其容量有限，只能存放短时间的信息。其特点是易被遗忘，且容易受到各种干扰，适用于处理快速变化的环境。
2. **长期记忆(Long-term memory，LTM)**：长期记忆指的是人脑中持久的记忆单元，其容量较大，可以储存长时间的信息。其特点是易被遗忘，但对环境变化不敏感，适用于处理静态环境。
3. **超意识记忆(Hyperconsciousness Memory)**：超意识记忆是指在学习、记忆、工作等各种任务中人脑自动生成和存储的各种信息。其特点是不受注意、不被约束，无法被主动遗忘，适用于特殊场景下的灵活运用。

为了更好地理解记忆模式，我们还需要了解一下三个要素：
- 信息：是指物质世界中的客观事实、观念或符号，是关于客体的一组属性及其关系的集合。
- 存储：是指将客观事实、观念或符号转换成存储于计算机或大脑中的二进制代码的过程。
- 查找：是指将存储于计算机或大脑中的信息检索出来的过程。

在实际应用中，人们常常将短期记忆模式和长期记忆模式分别称为**工作记忆模式(Working Memory)**和**持久记忆模式(Permanent Memory)，**两者在功能、结构和功能上存在区别。

## 3.3 信息存储系统
### 3.3.1 连接网络
人类大脑的存储系统是一个多层级的存储网络，每一层都是一个个体或群体的存储器。在不同层级中，大脑可以选择性地进行不同类型的信息的存储，比如图片、文字、听觉信息等。在不同层级之间的连接有两种模式：集中式和分布式。

集中式连接模式又称为显式连接(Explicit Connections)，指多种功能和信息存储器共同连接在一起，共享信息存储空间，且所有存储器都能够全面访问整个网络。典型的集中式连接模式如图3所示。


分布式连接模式又称为隐式连接(Implicit Connections)，指各种功能和信息存储器彼此独立，不共享信息存储空间，各个存储器只对自己负责的区域内的数据进行访问。典型的分布式连接模式如图4所示。


### 3.3.2 高速缓存存储器
人脑具有复杂而稀疏的连接网络，其连接结构及连接模式决定了信息的存储结构，但在存储信息时，仍然依赖于“快存储器”和“慢存储器”。快存储器存储、处理、检索最近获取到的信息，如指令、图像、音频、视频等。慢存储器存储、处理远距离获取到的信息，如物理信息、记忆模式、观念等。由于快速存储器的容量有限，所以可能难以保存完整的物理信息。为了方便信息的存储和检索，人脑会自动将某些信息复制到慢存储器中，再从慢存储器中检索出相关信息，这就是“拷贝术”。

### 3.3.3 记忆模块
人脑的记忆模块包括四个部分：
- **海马体(Hippocampus)**：海马体主要负责短期记忆。它有660万个突触核，并对编码、识别和存储信息发挥重要作用。海马体的位置位于躯干正中，上下分布，上部的海马体支配下部的海马体，受额叶、枕叶、皮质组织支配。
- **胶质层(Neurotransmitter Layer)**：胶质层负责短期记忆。它有三种不同类型的胶质——恒河沙玛利亚胶质(Aspergillus niger)、蔓延反射胶质(Reticular Coaty)和香气胶质(Xylitol)。胶质在神经细胞和神经轴突间的传递，是构建信息的重要因素。
- **长期记忆系统(Long Term Memory System)**：长期记忆系统是人脑的高级存储设备，属于容量大、性能强、功能丰富的记忆模块。它包括海马体网状结构、瞳孔和五官调节中心、后脑勺动物区、天牛脑区和海马体的顶端，可以支持长期的数字、符号、图像、文字和语音的存储。
- **认知状态编码系统(Cognitive State Encoding System)**：认知状态编码系统将脑电波或图像数据与记忆模式相对应。不同的信息通过不同的脑电波或图像信号传输到认知状态编码系统，通过脑电波或图像数据的提取，可以检索到相关的记忆模式。

# 4. 深度干扰的原因及其预防措施
深度干扰(Deep Cognitive Impairment，DCI)是指认知功能差的突出特点。它由以下原因引起：
- 重复性学习：学习任务中，试图记住过去学习过的内容，往往会出现重复性学习。在学习新知识时，如果重复阅读已学过的材料，人类大脑会变得迟钝，导致认知功能障碍。
- 深度记忆：深度记忆是指记忆的生理构造限制。人类大脑具有超过60亿个神经细胞，每个细胞都包含大约2400个突触，这一特点导致深度记忆的困难。长期记忆存在的局限性会导致大脑的深度记忆产生干扰。
- 注意缺陷：注意缺陷主要表现为各种注意力下降或注意力缺陷。注意力缺陷一般是在某段时间内，由于突发事件或复杂环境导致的注意力不集中，导致人的大脑缺乏专注力。注意力缺陷的发生在两个方面，一是由于注意力不能集中，另一是由于执行任务时没有充分利用注意力，导致认知失败。
- 学习压力：学习压力主要指在学习新任务、新知识时，人类大脑由于自身的需要而产生的紧张情绪。学习压力往往来自于外部刺激的刺激，如奖赏、压力、诱惑、竞争等。它会令人们产生焦虑、恐惧和厌恶情绪，进而影响人的正常学习、记忆和工作。
- 暂时记忆障碍(Sporadic Remembering Deficit，SRD)：SPoradic Remembering Deficit，临时记忆缺陷是指在学习过程中由于学习经历的重复性导致记忆力下降。由于知识的提炼、组织和压缩，会导致记忆者出现不连贯的思维和观点，导致临时记忆缺陷。

针对以上种种深度干扰，制定如下预防措施：
- 更好地训练：通过刻苦练习和刻意模仿，人们可以养成良好的学习习惯，更好地培养学习的能力，提高学习的效率和质量。训练时，应注意选择具体的目标、任务和材料，提升技巧和能力。
- 使用专注模式：人们在学习时，应该一直专注于某一件事或某一道题，同时尽量减少其它注意力的干扰。长期专注可以提高学习效率和质量。
- 通过比较和学习来提高记忆力：通过比较、模拟、联想、归纳等方式来实现记忆，可以提高记忆力。更进一步，通过学习知识与记忆的关联性，可以形成记忆模式，更好地理解、记忆信息。
- 健康饮食保持良好睡眠：摄入营养保护剂、多吃水果、蛋白质 rich food 和肉类，可以提高大脑的免疫系统，改善认知功能。多睡眠可以减少睡眠障碍对学习的影响，增强记忆力。
