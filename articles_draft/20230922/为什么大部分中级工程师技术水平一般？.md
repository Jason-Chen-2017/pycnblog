# 1.简介
  

随着计算机技术的发展，技术人员越来越多地加入到开发行业中，其中包括全栈工程师、后端工程师、算法工程师等。而由于在不同领域的知识广度、深度不一，导致大部分中级工程师的技术水平普遍偏低。

本文将从如下几个方面进行阐述：
1）为什么技术人员要选择该职业？
2）不同的职业需要具备哪些能力或技能？
3）目前国内各个公司对新入职技术人员的培训程度？
4）如何看待技术人员“技术不进则退”的问题？
5）如何才能提升自己的技术水平？

# 2. 职业划分及其要求
## （1）前端工程师
前端工程师主要负责网页的开发、设计、性能优化等工作，主要要求：
1. 熟练掌握HTML、CSS、JavaScript相关技术；
2. 有扎实的基础学过计算机网络、数据结构、操作系统等课程；
3. 有丰富的UI设计经验并有良好的交互设计风格；
4. 具有良好的学习能力、解决问题的能力和创新的精神；
5. 在理解业务需求的前提下，能够快速把控产品的开发进度。

## （2）后端工程师
后端工程师主要负责服务器的开发、维护和运维，主要要求：
1. 对数据库有一定了解；
2. 对Linux、Unix、Windows有一定了解；
3. 具有分布式系统开发经验；
4. 拥有较强的逻辑思维和分析能力，具备较高的抽象思维能力和独立解决问题的能力；
5. 善于沟通、协调、组织团队成员，有一定编码能力。

## （3）移动端工程师
移动端工程师主要负责iOS和Android应用的开发，主要要求：
1. 掌握Objective-C/Swift编程语言；
2. 有良好的产品意识和产品意图，擅长用户体验设计；
3. 对产品有敏锐的洞察力和清晰的头脑，能够快速理解复杂的产品需求；
4. 熟悉多线程、事件处理机制，并能适应产品的变动；
5. 有很强的学习能力、决策能力和创新能力。

## （4）算法工程师
算法工程师主要负责研究、实现、测试各种算法，主要要求：
1. 对数学、线性代数、数据结构有一定了解；
2. 有较强的逻辑思维和解决问题的能力，良好的分析问题和解决问题的能力；
3. 对软件工程有一定的关注，能够应用到实际开发中；
4. 具备良好的沟通能力、团队合作能力和创造力，能充分利用个人优势和资源。

## （5）系统工程师
系统工程师主要负责整个系统的架构设计和研发，主要要求：
1. 掌握UNIX/LINUX系统的基本使用；
2. 对分布式系统架构有深刻的理解；
3. 掌握高性能计算、存储、搜索、消息队列等技术；
4. 能够完整的完成项目的生命周期管理，对运营有一定能力；
5. 善于协同合作，能够指导和带领其他工程师完成工作。

# 3. 各大公司对新入职技术人员的培训情况
不同公司对于技术人员的培训标准差别较大，但有一个共同的特点就是专业能力。因此，为了保证技术人员技术能力的统一，需要通过各种方式对所有技术人员进行培训。这里列举了一些比较知名的公司的培训情况供参考。
## （1）腾讯
腾讯作为互联网领域知名的游戏公司，在2017年推出了针对技术人员的编程培训课程——“《C++程序员入门》”，主要面向初级、中级工程师。该课程包括C++语言基础、数据结构和算法、内存管理、编程规范、软件工程方法论、设计模式、调试技术、计算机网络、软件测试、Web开发、机器学习等知识。截止目前（2021年），该课程已经有近2000余人报名参加，已经覆盖了腾讯多个开发岗位。

腾讯云作为亚洲领先的云服务提供商，也是在2017年推出了一门技术方向为“AI+互联网”的“《腾讯云高级技术工程师》”课程，邀请业界顶尖的机器学习、深度学习、图像识别等领域的专家担任讲师，帮助技术人员快速掌握云计算平台的使用技巧、工具和最佳实践。该课程目前已经累计开设5期，已经吸引了超过1000位学生报名参加。

## （2）阿里巴巴
阿里巴巴作为中国最大的电商平台，2017年推出了一门“《阿里云开发工程师实习生训练营》”，由阿里云技术专家组成的实习生团队进行一对一远程教学，课程涵盖阿里云基础知识、编程实践、云产品使用技巧、技术架构和业务拓展四大部分。截至目前（2021年），该实习生训练营共有50余名同学报名参加，其中有七位实习生获得阿里云认证。

另一方面，阿里巴巴也在2019年推出了一门面向中小型互联网企业的“《Spring Cloud微服务开发实践》”课程，由阿里云云栖社区技术专家担任授课嘉宾，课程包括Spring Cloud基础教程、微服务架构设计、实战案例分享、工具使用技巧、容器化部署、DevOps自动化运维、Kubernetes运维指南等知识，适合企业的产品技术选型和开发能力提升。该课程于2月1日开课，已累计报名1万余人。

## （3）百度
百度作为中国领先的搜索引擎公司，在2017年推出了一门“《百度工程师技术学院系列之JAVA开发工程师(认证升级版)》”课程，由百度云计算技术专家担任授课嘉宾，课程主要介绍Java技术基础、框架应用、分布式系统开发、异步并发编程、数据库使用及优化、Web安全等内容，适用于正在准备Java开发工作的技术人群。该课程于4月份发布完结，总共开设两期，估计1万余名学生报名。

另一方面，百度也在2019年推出了一门面向学生的“《百度信息安全技术精英班》”，邀请优秀的安全工程师、安全专家作为老师，邀请学生学习、实践，结合实际的安全攻防场景，由百度云安全专家担任总监讲授，课程主要介绍计算机安全、网络安全、数据库安全、分布式安全等知识，并且提供实际的攻防练习，适合技术人才的自修教育活动。该课程目前有近1000人报名参加，第一期课程已于5月底结束。

# 4. “技术不进则退”的现象
“技术不进则退”是一个比较流行的话题，但似乎真的存在着。下面是一些案例：
## （1）微软放弃ASP.NET Core转投Xamarin开发
在2016年，微软宣布将其开源.NET Framework版本ASP.NET卸载，转投Xamarin开发领域，其迁移过程被认为影响了整个微软的技术生态。微软CEO Bill Gates接受了采访时表示，“虽然ASP.NET Core有一些明显优势，但是如果没有这些优势，我们最终会放弃这个框架。” 实际上，在 Xamarin 生态中有很多跨平台开发的方案，包括 React Native 和 Flutter 。所以，微软考虑的不是完全抛弃 ASP.NET Core ，而是建议放弃 ASP.NET Core 来切换到其他更符合 Xamarin 概念的框架。然而，据相关部门统计，目前 Xamarin 在市场份额上仍占有重要的地位，仅次于 iOS 和 Android 。
## （2）Facebook停掉众多程序员的实习机会
在2017年，Facebook宣布暂停众多实习机会，原因包括招聘者因个人原因不具备在Facebook工作的条件。Facebook CEO马克·瓦特尼在接受BBC采访时表示：“我们没有试图剥夺实习生或者其他类型的岗位，只是暂停了新人的实习机会。但我们不会给他们的申请者提供任何补偿，因为补偿只会让他们更难获得工作。”
## （3）谷歌放弃Python转投Java开发
在2019年，谷歌宣布从Python编程语言转向Java，其主要原因是希望其能够取得更好的Java市场份额。不过，对于此次转型，谷歌透露，“Java有更丰富的工具和库支持开发者，而Python是更灵活的一种编程语言，可以编写更高效的算法。”谷歌还表示，谷歌一直坚持使用Python来进行内部工具开发，“以前的编程语言都是因为特定用途才选择的，而今天，随着越来越多的公司都采用了Java来开发，所以我们决定支持更多的语言”。

# 5. 提升技术能力的几种方法
## （1）多学新技术
学习新技术可以有效提升技术能力，这是很多人努力追求的目标。下面是一些常用的技术，大家可以通过阅读相关书籍、网站和视频来了解它们的特点和应用。

1）数据结构与算法：数据结构和算法是程序员必备的功课，其涉及到非常多的计算机理论知识，如时间复杂度、空间复杂度、最优算法等。

2）数据库技术：数据库技术是保存数据的基础，不懂数据库的人无法正常使用计算机。数据库技术包括关系数据库、NoSQL数据库、键值对数据库和文件数据库。

3）网络协议：网络协议是互联网运行的基石，不懂协议的程序无法正常通信。协议包括TCP/IP协议、HTTP协议、WebSocket协议等。

4）操作系统原理：操作系统是计算机的内核，是计算机能运行的关键。操作系统原理涉及系统调用、虚拟内存、进程间通信、死锁等。

5）软件工程理论：软件工程理论为软件工程师提供了一套可落地的理论体系，能帮助其更好地理解软件开发的过程和方法。

## （2）积极参与开源项目
参与开源项目可以提升技术能力。通过参与开源项目，可以学习到更多的编程技术和工程实践，能够激发自己的创造力。GitHub上有很多著名的开源项目，如Redis、Django、Flask等。通过参与开源项目，可以促进自己技术能力的提升。

## （3）参加编程比赛
参加编程比赛可以锻炼自己的编程能力。编程比赛往往涉及到编程的各个方面，例如竞赛形式、规则设置、编译器错误等。通过参加编程比赛，可以更好的锻炼自己的编程能力。目前，国内著名的编程比赛有Codeforces、TopCoder、Leetcode等。

## （4）编程做项目
编程做项目可以锻炼自己的项目经验和解决问题的能力。在项目中，你可以接触到不同技术的应用，也可以锻炼自己的问题解决能力。你也可以自己开发一些小型应用或游戏来锻炼自己的编程能力。
