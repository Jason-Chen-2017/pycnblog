# 1.简介
  


什么是矩阵？矩阵是一个数字表格，由若干行和若干列组成，每一项表示一个变量或表达式的系数。例如：
$$\begin{bmatrix}a & b \\ c & d \end{bmatrix}$$
这是一个 $2 \times 2$ 的矩阵，其中 $a$, $b$, $c$, $d$ 为任意实数或复数。矩阵可以做很多事情，包括向量空间、几何学中的变换、物理学中的动力学、信号处理中的滤波器设计等等。因此，理解矩阵对于学习高等线性代数非常重要。

那么为什么要用矩阵呢？因为矩阵运算可以进行更多的计算。在数值分析、图论、控制论、优化理论等领域都有广泛的应用。矩阵和相关运算还有很多其他方面的联系，比如几何变换矩阵、幂级数矩阵、傅里叶级数矩阵、微分方程矩阵、稀疏矩阵、随机矩阵、对称矩阵、非对角矩阵等等。这些都是高等线性代数研究中的重要概念。所以，了解矩阵的特性和用法有助于更好地理解高等线性代数的内容。

接下来，我们就来看一下如何学习线性代数中最重要的两个概念——特征值与向量空间。如果你对它们还不是很熟悉的话，我建议你先学习一下数学公式的知识，尤其是一些矩阵运算的技巧。学习完之后再回过头来阅读本视频的内容就会比较轻松。

# 2.基本概念术语说明

1. 矩阵

   在数学中，矩阵（matrix）是一种用来表示二维数组的工具。它是一个矩形块状的数据结构，其中每一格元素的值代表某种统计学上的属性或关系，如速度、质量、温度、价格等。

举例来说，假设有一个二维数组A=[[a,b],[c,d]],其中a,b,c,d都是实数或者复数，那么这个数组就是一个矩阵。矩阵的行数和列数分别为2，即矩阵的阶数为2。

另外，如果用文字来定义矩阵，我们也可以这样写：

    A = | a b |
        | c d |
        
这里的"|"和"\\"分别用来表示矩阵中的行与列的边界，将每一列数据用"\\"隔开，将每一行数据用"|"隔开。这样的形式相比上面用[[ ]]的方式更加直观易懂。

2. 向量

   向量（vector）也被称作基矢量、坐标或一维数组，它是一个具有方向的量，可以表示出某个空间或某个对象的大小及方向，如位置矢量、速度矢量、电流强度向量、浓度分布向量等等。

同样，向量也可以用文字来定义：

    x = (x1, x2,..., xn), n个实数或者复数
    
x1, x2,..., xn 表示向量的各分量。

3. 向量空间

   向量空间（vector space）是一个集合，它由一组向量所张成。向量空间中的任何向量都可以通过一些线性组合得到，且没有冗余的部分。向量空间通常定义在一个集合上，该集合中的元素可以进行加减乘除等运算。
   
举个例子，空间R中包含了所有的向量，则R为R向量空间。

4. 零向量

   零向量（null vector），也称零坐标，它是一个长度为零但指向任一方向的向量，例如，零向量只有一个分量，分量值为0。零向量不参与向量空间的任何运算。
   
5. 向量积

   向量积（dot product）又叫点积或内积，是把两个相同维度的向量中对应分量相乘并求和的结果，记作：<a, b>。

例如，如果a=(a1, a2,..., an)，b=(b1, b2,..., bn)为向量，那么<a, b>=a1*b1 + a2*b2 +... + an*bn。

6. 范数

   范数（norm）是一个映射，它把向量映射到一个非负实数。范数通常指的是向量的模长（magnitude）。我们一般采用欧氏范数来衡量向量的大小。
   
7. 列向量

   列向量（column vector）是具有n个元素的向量，其中每个元素都是一个列向量。例如，一个3×1的列向量可以写作：
   
    C = [a]
       [b]
       
   其中C的每一列元素都是一个1 × 1的列向量。
   
8. 行向量

   行向量（row vector）是具有m个元素的向量，其中每个元素都是一个行向量。例如，一个1×3的行向量可以写作：
   
   R = [a b c]
     
   其中R的每一行元素都是一个1 × m的行向量。
   
9. 转置矩阵

   转置矩阵（transpose matrix）是矩阵的镜像对称性质的矩阵，即<A^T,B>=<A,B>。例如，如果A=|a b|
                                        |c d|,
    
    那么A的转置矩阵A^T=|a c|
                         |b d|
                         
   此时，A和A^T是同型矩阵。
   
   
# 3.核心算法原理和具体操作步骤以及数学公式讲解

1. 矩阵运算

   矩阵乘法：AB是一个m×n的矩阵，C是n×p的矩阵，那么C=AB可以定义为：
   
   $$C=\left[\begin{array}{cccc}a_{11} b_{11}+a_{12} b_{21}&...&a_{1n} b_{1n}+a_{1p} b_{pn}\\...&\cdots&\cdots\\a_{m1} b_{11}+a_{m2} b_{21}&...&a_{mn} b_{1n}+a_{mp} b_{pn}\end{array}\right]=\sum_{i=1}^{m} \sum_{j=1}^{n}\left[a_{ij} \cdot \left(\begin{array}{ccc}b_{ji}\\...\end{array}\right)\right].$$
   
  这里，$\cdot$ 是一种符号，表示两个矩阵对应的元素按行乘积的和。

   逆矩阵：对于 invertible matrix A，存在矩阵 B 满足：A * B = B * A = I_k （I_k 是单位矩阵）。换句话说，A 乘以它的转置等于它的逆矩阵。其定义如下：
   
   $$\begin{equation*}
   AA^{-1}=AA^{−1}=\begin{bmatrix}1&0\\0&1\end{bmatrix},\quad AX=B\Rightarrow X=A^{-1}B,\quad X=BX^{-1}.
   \end{equation*}$$
   
2. 特征值与特征向量

   特征值与特征向量是利用矩阵的一些特性而来的。

   ① 对称矩阵的特征值及其对应的特征向量: 对称矩阵是方阵，其特征值均为实数，对应的特征向量为实数 eigenvector 。证明：对称矩阵$M$有$n$个特征值，且$eig(M)=\{λ1,λ2,...,λn\}$。
   
   当方阵$M$可逆，并且$M^TM=A$是满秩矩阵，则$det(M)>0$，当$det(M)<0$时，$M$无真实的特征值。
   
  证明：如果$M$可逆，并且$M^TM=A$是满秩矩阵，则有$MM^T=A=M(M^T)^T$(证明中考虑了矩阵转置与逆矩阵)。由此可得$det(M)=det(MM^T)=det(M)(det(M^T))=det(M)^2$。
  
  当$det(M)>0$时，显然有$eig(M)$的非负特征值，并且$eig(M)=\{\lambda i:\text{$M-\lambda I_n$满秩}\}$.
  
  当$det(M)<0$时，根据列交换可知$M^T=-M$，有$eig(-M)=eig(M^T)$。令$Y=M-λI_n$,则$det(Y)<0$。根据特征方程，有$X^TY=XY^TX$，且$X^T$为$Y$的特征向量。因为$X^T$可以作为$Y$的一个特征向量，则$X$可以作为$-Y$的特征向量。

  推论：对称矩阵$M$的$λ_1$对应的特征向量$v_1$，即$M v_1 = λ_1 v_1$，并且$||v_1||=1$。（实际应用中，特征向量可能需要归一化）
   
   证明：对称矩阵$M$的特征向量$eigvec(M)$也是唯一的，而且是正交向量。也就是说，对于任意$v_1,v_2$，有：
   
   $\forall v_1,v_2\in eigvec(M),$$\begin{aligned}
   Mv_1 &= λ_1 v_1 \\
   Mv_2 &= λ_2 v_2
   \end{aligned}$
   
   根据正交定理，$v_1^Tv_2=0$。
