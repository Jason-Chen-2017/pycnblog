# 2.二项分布
## 2.1 二项分布概述
二项分布又称伯努利分布，是一种极其简单的离散分布，它描述的是随机实验中事件发生一次或者多次，概率为p的情况。具体来说，就是在每次试验中，只有两个可能的结果，比如抛硬币，成功或者失败。假设每次试验的结果都是独立且同分布的，那么其联合分布就可以用二项分布来表示。
## 2.2 二项分布的参数估计
给定n个独立的随机变量X1，X2，……Xn，i=1,2,…,n,它们服从二项分布，记作X=X1+X2+…+Xn，其中Xi的取值只能为0或1。则n次独立试验中事件发生k次的概率为:
P(k)=C(n,k)*p^k*(1-p)^(n-k)

其中，C(n,k)为组合数。根据binomial theorem可以得到这个概率。
## 2.3 二项分布的应用
### （1）抛掷硬币n次，正面朝上的次数
假设抛掷n枚硬币，其中前m枚为正面朝上的，那么抛掷这n枚硬币得到第m次正面的概率为：
P(m;n)=C(n,m)*p^m*(1-p)^(n-m)
### （2）投掷骰子n次，每种点数出现次数的期望
假设投掷n颗骰子，各个点数的出现频数服从二项分布，记作X1, X2,..., X6。那么每个点数出现的概率为：
p=[1/6, 1/6, 1/6, 1/6, 1/6, 1/6]
那么一共投掷了n次骰子，所有点数的出现次数的期望为：
E[X]=np
### （3）一批学生考试中成绩及格比不及格的概率
假设有n名学生参加考试，分两组A、B，其中组A得到a分，组B得到b分，则两组学生总得分的期望为：
E[X]=np+(nb-na)/2=70
设好分数线为60，那么组A中成绩及格的概率为：
P(X≥60|A)>P(X≥60|B)=(9*a-5)/(4*n)<=(11*a-7)/(4*n)=P(X≥60)
则两组学生考试中成绩及格的概率为：
P(X>60)=P(X≥60)(1-P(X≤60))=P(X≥60)*(1-(1/(1+exp(-(X-70)/10))))<=(9*a-5)/(4*n)*(1-(1/(1+exp(-(X-70)/10))))=<=(b-a)/2
即两组学生考试中成绩及格的概率相等。
# 3.多项式分布
## 3.1 多项式分布概述
多项式分布又称伯努利-沙巴克尔马尔可夫分布，它是一个离散分布，用来描述多次独立重复试验中事件发生某个指定次数的次数。在n次试验中，有k次事件发生的概率为：
P(k)=C(n,k)*(p_1^k)*(p_2^(n-k))*(p_3^(n-k-1))*...*(p_K^(n-k-K+1))

其中，p1, p2, …, pk是各自事件发生的概率，C(n,k)为组合数。多项式分布适用于依赖于不同物质条件的组合爆炸试验，例如核爆试验中研究核燃料对星际飞船残骸破坏的影响。
## 3.2 多项式分布的参数估计
### （1）抛掷硬币n次，第m枚硬币为正面的概率
假设有n枚硬币，每枚硬币正面朝上概率为pi，则每枚硬币的抛掷概率为pi，则抛掷n枚硬币，第m枚硬币为正面的概率为：
P(X=m;n,p_i)=C(n,m)*p_i^m*(1-p_i)^(n-m)
### （2）投掷骰子n次，每种点数出现次数的期望
假设投掷n颗骰子，各个点数的出现频数服从多项式分布，记作X1, X2,..., X6。令pi=P(X=i), i=1,2,...,6。则每个点数出现的概率为：
p=[pi, pi, pi, pi, pi, pi]
那么一共投掷了n次骰子，所有点数的出现次数的期望为：
E[X]=∑i=1to6pi*i!
### （3）一批学生考试中成绩及格比不及格的概率
假设有n名学生参加考试，每组学生具有ai名成员，考试分数为70，则有两种分数分配方案：
1. 分配两组，每组五人，第一组中高于平均分的占比为1/4，第二组中高于平均分的占比为3/4。则组A中成绩及格的概率为：
    P(X>=60|A)=1/4*(1-pi)*e^(-abs(X-70)/10) + 3/4*(1-pi)*e^((70-X)/10)
    P(X<=60|A)=1-P(X>=60|A)=1/4*(1-pi)*e^((-X+70)/10) + 3/4*(1-pi)*e^(-abs(X-70)/10)
    组B同理。

2. 一组成绩高于平均分的概率为：
    P(X>=60|B)=1/2*(1-pi)*e^(-abs(X-70)/10)+1/2*(1-pi)*e^((70-X)/10)
    P(X<=60|B)=1-P(X>=60|B)=1/2*(1-pi)*e^((-X+70)/10)+(1/2-pi)*e^((-X+70)/10)
    此时两组成绩的差异越小，则组B中成绩及格的概率越大。
    
由此可见，分组策略（1）可能导致组B中成绩及格的概率高于组A，但如果考虑到组A的成绩分散性，这种情况下应该选用分组策略（2）。
# 4.泊松分布
## 4.1 概述
泊松分布（Poisson distribution），也叫泊松分布、泊松过程或泊松现象，是一个离散概率分布。泊松分布适用于当一个事件在一个确定的时间间隔内发生的次数为不可知的，而这些事件发生彼此独立，即不知道前一个事件结束后，下一个事件将会发生多少次，泊松分布可以准确预测这一事件发生的次数。泊松分布是指单位时间内随机事件发生次数的概率分布。泊松分布的典型应用领域是计算机网络流量、通信传输数据包数量、设备故障数量、汽车尾气消耗量、DNA序列突变数、氢核反应产出速率等。
## 4.2 参数估计
### （1）一天之内发生m起交通事故的次数的概率
在一个平常的时间里，一天之内发生m起交通事故的概率为：
P(X=m)=e^(-lambda)*lambda^m/m!
where lambda is the average number of occurrences per interval.

平均每分钟发生一次事故的电话呼叫是个例子。
### （2）一年之内发生m起交通事故的次数的概率
一年之内发生m起交通事故的概率为：
P(X=m)=e^(-lambda)*lambda^m/m!*exp(-lambda)/T
where T is the total time period being considered (for example, a year).

年终奖金活动是另一个例子。
## 4.3 泊松分布的应用
### （1）网站访问次数的预测
对新闻网站进行访问统计的目标是了解用户对新闻的兴趣程度。假设网站每天都有N个新闻页需要更新，每页都存在一定数量的静态资源，每个静态资源加载时间平均为t秒，然后再经历平均处理时间t'秒，那么每天访问网站的用户访问次数可以近似认为服从泊松分布，于是预测每日用户访问数量的泊松分布参数λ可以用：
λ=Np/t

其中，N是网页总数，t是平均页面加载时间，t'是平均页面处理时间。这样预测每日访问用户数量的泊松分布参数λ可以作为网站营销、广告设计等的基础。
### （2）手机APP下载次数的预测
对手机APP进行下载统计的目标是了解用户对APP功能的喜爱程度，假设每个APP安装大小约为sMB，下载完成后需要解压，每个解压文件耗时t'秒，那么每周用户下载APP的数量可以近似认为服从泊松分布，于是预测每周用户下载数量的泊松分布参数λ可以用：
λ=Ns/t

其中，S是APP总体大小，t是单个APP的下载耗时，t'是解压耗时。这样预测每周用户下载数量的泊松分布参数λ可以作为市场调研、产品运营等的基础。
# 5.指数分布
## 5.1 概念
指数分布（exponential distribution）又称为幂律分布、伽玛分布或泊松分布，是指若随机变量X随着时间的推移服从指数分布，那么X的概率密度函数为：
f(x)=\lambda e^{-\lambda x}

其中λ是指数分布的参数，其概率密度函数曲线呈锯齿状。指数分布可以用来描述一段时间内平均发生率的泊松分布。由于其紧凑的概率密度函数，使其广泛应用于许多实际问题。如一辆车在单位时间内驶过某个距离所需的时间是一个符合指数分布的随机变量；服务器故障导致服务延迟随着时间的增长是一个符合指数分布的随机变量；电话呼叫等待时间模型中，分摊因素造成等待时间的均值为指数分布。
## 5.2 参数估计
### （1）电视机频道订阅的用户数
假设一台电视机每天的播放时间是t小时，电视每天播出的节目种类是Θ个，且每个节目的观看时长分别服从独立的指数分布。也就是说，观看第j类的节目至少t/Θ小时才能完成，观看其他节目则只需要较短的时间。电视机的用户人数可以近似认为服从泊松分布，于是计算电视机每日订户人数的指数分布参数λ如下：
λ=exp(-t/Θ)

其中λ为平均每天用户数。可以估计，每周有6.5天是闰年，每月有31天是普通月份，每季度有91天是3个月的季度，每半年有183天是6个月的半年。用泊松分布估计每半年或每月每日订户人数的过程，除了用泊松分布估计每天的订户人数外，还需要乘以相应的天数。
