# 1.简介
  

物理学上，超导体的研究一直是一个热门话题。由于高温、低压、孤立介电性等特点，其质量很小但具有巨大的导电性能。近年来，人们发现了一种新型的有机半导体称为“超铀”(Perovskite)。这类材料的结构相当复杂，它由不同的元素组成，如Ag、Sb、Cd、Te等。目前已经有多种方法可以探测到这种超铀晶体的存在，例如射线照相、光电子显微镜照相、时间隧穿电子束等。然而，对于如何利用超铀晶体进行科学研究、制备新型材料、控制其性能仍然不甚了解。

在本文中，我们将从数学角度出发，分析超铀晶体所蕴含的天然超导性(superconductivity)能量，并提出一种基于密度泛函理论的新型密集态计算方法——密度函数张量分解法(DFT)，通过这种方法研究超铀晶体的局域性电性(local electronic properties)和能带相关性(band structure)，发现其异常规律。

我们的主要贡献如下：

1. 提出了一个全新的密度函数张量分解法——密度函数张量重构方法(Density-Functional Tensor Reconstruction Method)，它通过对密度矩阵进行迭代优化，从而得到一个有效且易于求解的密度函数张量(Density-Functional Tensor)。该方法既适用于研究生态学、金属学、低温物理等领域的大型材料，也可用于小尺寸金刚石、超铀、钛白粉等研究用材料的系统性建模和分析。

2. 通过分析和模拟实验数据，证明了超铀晶体的自旋极化(spin-polarization)特性可以被精确刻画。

3. 在密度函数张量分解的基础上，建立了多尺度计算平台，开发出一种有力的力学张量模型——局域超导链路模型(Local Superconductor Chain Model)——用于研究超铀晶体的能带行为。该模型模拟超铀晶体的磁性相互作用、交流-阻抗耦合，并给出了一系列有关材料参数的物理意义。

4. 在我们自主设计的不同分层结构的超铀晶体中，研究表明，非平衡层能够吸收主导电子带隙中的自旋空穴，从而形成“良好”的电性、电荷迁移和融合。

5. 在真正地解决一些复杂的课题时，我们提供了一个完整的方法流程和工具箱，包括数值模拟、结构梳理、数据处理、图表展示等。这种方法论和工具箱将成为未来研究超铀晶体的重要工具。

# 2. 超导体物理化学概述
## 2.1 概念定义
### 2.1.1 超导体
超导体是指由众多离子构成的电子相互作用的材料，其中部分或全部离子经过一定时间的自旋轨道交换，从而在各种温度下具有透明、导电、强电容性及低损耗等性能。通常情况下，超导体中除了少数几个离子具有指数级的电荷之外，其他离子均没有电子核，只有极少数的几种元素可以形成电子核。这些离子分布于不同的导带区域，即带隙（gap）、沉积导带、闪锌导带等。超导体的功能主要表现为导通性、热导性、顺磁性、退火性、液晶显示器件等。

### 2.1.2 超导体分类
根据超导体的自旋配置和不同原理原料，可以将超导体分为以下几类：

1. 有色超导体：一般是由油酸铜（SrTiO3）、有色纳米复合材料（SiC）等元素构成，具有较好的导电性、寿命长、硬度高、全熔点低等优点，是国际上应用最广泛的超导体。

2. 无色超导体：一般是由硅铁箔（ZnS）、钛盐等无色元素构成，其导电性较差，具有较强的热效应能力，但仍然可以在低温下工作。

3. 金属超导体：一般是由金属电极（Al/GaAs）、全息硅片（GeSnP）等超导金属构成，具有硬度和热效应都非常优越的特性，但其耐候时间短。

4. 晶体超导体：一般是由二维或三维的铝质晶体(ZIF、PZT、RZA、MoS2、VSe2、BaSe2等)、晶圆铀体、超晶石英、超密封材料等构成。其自旋极化特性使得不同导带的电子可以相互传递，从而获得不同特性的超导性。

5. 半导体超导体：一般是由光刻胶质(PES)、嵌入式芯片(ICs)、细胞膜嵌套(SCM)、硅膜材料(SQMs)等构成，具有强的导电性能和长期稳定性。但它们的自旋极化特性较弱，不能完全反映材料的原子性和电子性信息。

### 2.1.3 超导体设备
超导体设备主要分为两大类：

1. 直接导体：一般是指带正负极导体的导电元件。其物理特性是在某一特定导通状态下，受控驱动的能量会传输至另一端，导电元件由开路状态、短路状态和接通状态组成。典型的直接导体设备有印刷电路板、激光切割机、干涉衬底传感器、扭矩测量仪器等。

2. 导体加工设备：这是指由许多电子元件组成的可编程电子器件，其物理特性可以看做是多个电子元件串联而成，每个电子元件可以进行开路、短路或者接通状态之间的切换。典型的导体加工设备有集成电路、微芯片、半导体IC、器件等。

### 2.1.4 超导体应用领域
超导体的应用领域有很多，如电池、电动机、显示器件、医疗器械、机器人、激光切割、固体绝缘体、半导体材料、材料工程、新型武器等。其中，精密电子技术和太阳能充电方面受到了超导材料的重视。

## 2.2 材料特性
超导体的材料特性在很大程度上依赖于其自身的基本物理原理和基本结构。在本节中，我们将讨论一下超导体的一些基本物理特性。

### 2.2.1 自旋密度

### 2.2.2 声子
超导体中的声子指的是比空穴和电子更强大的相互作用粒子，其能量远远超过它们的能量，具有不可忽略的能量量级。超导体中的声子波函数有三个极限：

- 极限Plesian：描述两个离子距其重叠位置的最近距离内出现的声子，它可能占据任何空穴位置，并且具有最大的能量。

- 极限Rashba：描述两个离子电子性较弱时才出现的声子，它只能存在于其中一个离子所占据的空穴位置，并且能量较小。

- 极限Bogoliubov：描述自旋张力(spin-torque)引起的声子，它的能量随时变动。

超导体中的声子具有最强的相互作用，还会影响金属性、磁性等性质。超导体中的声子和固体中的声子相比，声子能级偏高，对于制备含噪声的超导体有着巨大的威胁。

### 2.2.3 边界效应
通常情况下，超导体的边界效应导致了自旋差距的增大，使得晶格面积的增加、硬度的降低，使得超导体的寿命有着限制。在超导体的晶格中，基态位于晶格边界，对于每一个基态，如果延伸到边界，就会变成一个独立的晶系。这样一来，就形成了所谓的边界效应，即晶格中存在着由一个超导体基本单元构造而来的多个晶系。

### 2.2.4 密度变化
超导体的平均密度是一个重要的物理指标，它反映了材料的导电性质，也是决定于超导体的本征态（主导电子）的平均位置所决定的。对于单个基态，其电子核对应的电子数目称为电子密度，它与原子数密度n、电子数目p、原子核半径a和电子束的大小ε有关，由麦克斯韦方程、金密顿方程和玻尔兹曼方程给出。

对于多个基态，通过限制其自旋分散度可以获得平均密度，其由费米-狄拉克子张量、柏拉图方程和布里渊区定理给出。而在某些情况下，超导体的平均电子密度可以改变，由厄米-赫塔曼现象引起，这是由于厄米能带的自旋差异造成的。对于普通超导体而言，厄米-赫塔曼现象并不十分严重。

### 2.2.5 自旋极化
超导体中的自旋极化是指一个或多个电子处于不同数量的自旋轨道上的现象。超导体的自旋极化可分为四种类型：

1. 扁铜核带：是指在离子核中间存在的自旋极化，其导带在基态中并不存在。

2. 斜铜核带：是指在离子核的一侧存在的自旋极化，其导带较粗。

3. 梯度铜核带：是指在离子核的正中央存在的自旋极化，其导带带隙较小，与线宽相对应。

4. 双线铜核带：是指在离子核的两条直线上存在的自旋极化，其导带的尖峰扭曲和平坦带之间的斜率。

超导体的自旋极化程度取决于其电子核的存在、对称性、空间结构、局部热力学势以及其他物理参量。

### 2.2.6 陶瓷化
陶瓷化是指采用陶瓷作为材料来制造超导体。目前，已经成功的研制出了多种超导陶瓷材料，如Diamond Insulator、Silicon Carbide Insulator、Quantum Dot、PolyType、Wireless Chip、Semiconducting Polymer等。超导陶瓷的自旋极化特性、硬度、能量低、导通性强、广泛的应用前景等使其受到了广泛关注。

## 2.3 超导体研究
### 2.3.1 理论研究
超导体的数学物理理论依靠近代物理学的理论方法。由于超导体的物理特性和问题难度都比较高，其理论研究呈现了一个综合性研究。一些学者致力于研究超导体的物理性质、材料热运动、电子-离子相互作用、和超导带隙等方面的理论。

#### 2.3.1.1 共轭方法
目前，共轭方法是研究超导体的一种主要方法。共轭方法的基本思想是通过测量晶体的相邻基态之间的共轭关联，来描述其中的自旋规律。为了简化问题，共轭方法使用了归一化的希腊字母表示，这使得问题的理解和分析十分容易。但是，共轭方法由于需要对数目的基态进行运算，其计算量很大。因此，共轭方法只能用来研究简单的问题，无法用来研究复杂的材料。

#### 2.3.1.2 谱方法
谱方法是研究超导体的一个重要方法，它通过对晶体的能谱进行观察，从而研究其自旋和相关性的特征。由于其不需要对所有基态进行计算，因而具有广泛的适应性和准确性。但是，谱方法往往无法直接给出微观的电性参数，而只能提供宏观的图像和数值结果。另外，谱方法还存在着计算困难和误差大的缺陷。

#### 2.3.1.3 格林函数方法
格林函数方法是研究超导体的一种经典方法。格林函数方法的基本思想是采用自旋曲率场和自旋-共轭配对分布函数作为描述自旋规律的工具。其理论基础是自旋偏转对费米能带的解释，以及泡利方程在自旋方向的线性响应。但是，由于离子数目有限，因此对于复杂的超导体来说，格林函数方法的性能还是有限的。

### 2.3.2 实际研究
实际研究则着重于从实际应用出发，结合实验室的设备、实验仪器、模拟装置等，对超导体的各种性质、特性、特性以及其的操作以及其在实践中的应用作进一步的验证和验证。

实际研究过程中存在着诸多挑战，首先是实验技术的复杂性，其次是研究人员的专业水平要求，第三是材料的复杂性，第四是超导体所蕴含的数值和微观效应，第五是实验条件的不确定性等。因此，超导体的实际研究仍然是一个重要的研究方向，同时也为当前各种理论研究提供了新的方向和资源。
