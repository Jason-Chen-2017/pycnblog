# 线性回归与逻辑回归：监督学习的两种基本算法

### 1. 背景介绍

监督学习（Supervised Learning）是一种机器学习算法，它通过学习已知的训练数据来预测未知的结果。监督学习算法广泛应用于数据分析、预测、分类和回归等领域。其中，线性回归（Linear Regression）和逻辑回归（Logistic Regression）是监督学习中两种最基本和最常用的算法。

### 2. 核心概念与联系

线性回归和逻辑回归都是回归算法，但它们的目标和应用场景不同。线性回归旨在预测连续值的输出，而逻辑回归旨在预测离散值的输出。

线性回归假设输入数据之间存在线性关系，并尝试找到一个最佳的线性模型来拟合数据。逻辑回归假设输入数据之间存在非线性关系，并尝试找到一个逻辑函数来拟合数据。

### 3. 核心算法原理具体操作步骤

#### 3.1线性回归

1. 数据预处理：将输入数据标准化或归一化，以便模型训练。
2. 模型初始化：初始化线性模型的参数。
3. 损失函数计算：计算训练数据的损失函数，即均方误差（Mean Squared Error）。
4. 参数更新：使用梯度下降算法或其他优化算法更新模型参数。
5. 模型评估：使用验证数据评估模型的性能。

#### 3.2逻辑回归

1. 数据预处理：将输入数据标准化或归一化，以便模型训练。
2. 模型初始化：初始化逻辑模型的参数。
3. 损失函数计算：计算训练数据的损失函数，即对数损失函数（Log Loss）。
4. 参数更新：使用梯度下降算法或其他优化算法更新模型参数。
5. 模型评估：使用验证数据评估模型的性能。

### 4. 数学模型和公式详细讲解举例说明

#### 4.1线性回归

线性回归模型可以表示为：

$$y = w^T x + b$$

其中，$y$是输出变量，$x$是输入变量，$w$是权重向量，$b$是偏置项。

损失函数可以表示为：

$$L(w, b) = \frac{1}{2} \sum_{i=1}^N (y_i - (w^T x_i + b))^2$$

其中，$N$是训练数据的数量。

#### 4.2逻辑回归

逻辑回归模型可以表示为：

$$p = \sigma (w^T x + b)$$

其中，$p$是输出变量的概率，$\sigma$是sigmoid函数，$w$是权重向量，$b$是偏置项。

损失函数可以表示为：

$$L(w, b) = - \sum_{i=1}^N (y_i \log p_i + (1 - y_i) \log (1 - p_i))$$

其中，$y_i$是训练数据的标签。

### 5. 项目实践：代码实例和详细解释说明

下面是一个使用 Python 和 scikit-learn 库实现线性回归和逻辑回归的示例代码：
```python
import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.linear_model import LogisticRegression
from sklearn.datasets import load_boston
from sklearn.model_selection import train_test_split

# 加载 Boston  datasets
boston = load_boston()
X = boston.data
y = boston.target

# 将数据分为训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)

# 实例化线性回归模型
lr_model = LinearRegression()
lr_model.fit(X_train, y_train)

# 预测测试集
y_pred = lr_model.predict(X_test)

# 实例化逻辑回归模型
lr_model = LogisticRegression()
lr_model.fit(X_train, y_train)

# 预测测试集
y_pred = lr_model.predict(X_test)
```
