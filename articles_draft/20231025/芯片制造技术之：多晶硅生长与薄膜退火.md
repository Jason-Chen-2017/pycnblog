# 1.背景介绍


## 1.1 什么是多晶硅？
多晶硅（polycrystalline silicon，PCS），是指含有多个不同晶体结构的硅。如一般的硅是单晶，那么多晶硅就是具有多个晶体结构的硅。通常情况下，多晶硅分为两类——同种异构晶体与杂化晶体。同种异构晶体指的是同一种类型的晶体成对共用；杂化晶体则相当于所有晶体都混在一起。比如，在一次冷开采样中采集到9颗不同晶体，称为杂化晶体，而在一次洗浴过程中由于浮气的作用形成的纯粹硅晶体即为同种异构晶体。

## 1.2 为什么要用多晶硅？
多晶硅的应用主要包括以下几方面：

1. 增强结构强度: 在电子设备、计算机、显示器等领域，多晶硅通常比单晶硅更容易产生一个较大的电容来增强其电阻性，提高其抗干扰能力和可靠性。

2. 提升低温性能：多晶硅结晶厚度可以达到厘米级甚至毫米级，在激光器、微波炉、照明灯等领域，多晶硅能够提供更高的低温性能。

3. 降低耐腐蚀率：多晶硅的稳定性得益于它的杂化结构，使得各个晶体的温度保持在均匀状态，从而降低了耐腐蚀率。

4. 提高灵敏度：多晶硅的薄膜层能够吸收高能量的电磁波并转变为低频的信号，从而提高其灵敏度。

5. 改善设备结构：由于其结晶厚度更大，多晶硅在集成电路、半导体器件及其他电子元器件中的嵌入，使得整机结构简化、尺寸缩小、抗震烧能力增强、出线可靠性更高，是提升产品的可靠性、成本效益和可维护性的有效方式。

## 1.3 多晶硅的种类
### 1.3.1 同种异构晶体
同种异构晶体（homogeneous polymorphous silicon）,也称单晶硅，这是一种典型的多晶硅结构。其含有的全部晶体的晶格大小、偏旋方向、排列方式都是相同的，只是所含有的原子数量不同。

同种异构晶体的材料由晶体团簇（crystal clusters）组成，其原理是在晶体团簇之间配制一定的缺陷，使得晶体团簇的每一层的原子数量不一样。比如，单个晶体团簇中含有10^9个原子，而另一个晶体团簇中含有10^8个原子，这样就使得两种晶体团簇之间出现了一定的差异。通过这种方式，就可以获得两个不同的晶体结构。

### 1.3.2 杂化晶体
杂化晶体（dispersed polycrystalline silicon，DPS），是指采用多种晶体结构结合的方式，而不是直接将不同晶体堆叠起来。常用的混合方法有：颗粒状聚合物（granular aggregate）法、颗粒状网络法（nanopillar network）、风洞法（winding pattern）、液态分离法（liquid phase separation）等。

## 1.4 什么是薄膜退火？
薄膜退火（thin film deposition，TFD），是指在金属表面的特定位置掺杂特殊的氢离子层，使得金属表面呈现出薄膜状结构。由于特殊的结构，该特异的薄膜结构往往被设计用来制造各种工艺品或装置，如微芯片、光刻胶、印刷电路板等。

图2：薄膜退火示意图

根据薄膜退火过程中的反应时间，又可以分为干燥步（heating step）、涂层步骤（annealing step）和晶化步骤（crystallization step）。其中，晶化步骤完成之后，所得到的晶体便是真正意义上的多晶硅。

# 2.核心概念与联系
## 2.1 显著尺寸
显著尺寸是指特定原子核的相对坐标相对于整个晶体中央点的距离，通常以nm为单位。显著尺寸越小，代表原子核的位置就越靠近晶体中心，它能促进晶体团簇的分布和紧密聚集，因此其对应着晶体的精细程度。但显著尺寸越大，代表原子核的位置就越远离晶体中心，因而它能减少晶体团簇之间的相互作用，导致晶体的疏松程度。显著尺寸取决于材料的结构特性以及温度。

显著尺寸有些类似于晶体的疏密程度，但是显著尺寸表示的是晶体中某种原子核与晶体中心的距离，所以显著尺寸其实还隐含着一种空间度量信息，反映了晶体的均匀性。

图3：显著尺寸关系示意图

## 2.2 平均振动速率
平均振动速率是指构成晶体的不同原子核之间，平衡力的平均推力。由于同种异构晶体（Homogeneous Polymorphous Silicon，HPS）所含有的不同晶体团簇之间的摩擦力较小，所以平均振动速率通常很小。

而杂化晶体（Dispersed Polycrystalline Silicon，DPS）具有大量的晶体团簇，且这些晶体团簇之间的摩擦力很大，因此平均振动速率可以达到几百万赫兹。

## 2.3 裂变模式
裂变模式是指晶体发生融合时，融合的速度和方向。裂变模式有三种基本模式：

- 斜射模式：一般来说，若两个晶体的圆心间距不大于同一倍的晶胞周期（LCM，Least Common Multiple），则发生裂变。斜射模式代表了最小的融合速度，通常仅仅用在单晶体的成熟期，或者用在轻量级的测试中。

- 普朗克模式：在普朗克模式中，受压晶体会随着时间的推移逐渐向外扩散，直到撤除受压晶体内的所有氢原子后才停止扩散。普朗克模式代表了晶体融合的最快速度，并且可以产生小尺度晶体，例如亚稳态晶体。

- 拉曼金森模式：拉曼金森模式中的两个晶体会同时进入到一个临界温度，然后由其中较热的一块晶体接触到另一块晶体，形成新的晶体。拉曼金森模式代表了晶体融合的最慢速度，通常用于产生庞大尺度晶体，例如称重晶体。

## 2.4 薄膜退火的处理流程
薄膜退火的处理流程可以分为以下几个阶段：

1. 熔化处理（Thermal Annealing）：先进行熔化处理，主要是将晶体液态锈蚀物挤出至合适的温度，然后将晶体冻结至固态表面。

2. 涂层处理（Patterning）：加入特殊的氢离子层作为模板，将其放置在所需的位置上。在晶格尺度下，氢离子层被称为“孔”。

3. 腐蚀处理（Emulsion Diffusion）：在涂层的基础上进行腐蚀处理，将受压孔腐蚀掉，将水分子侵蚀到孔中。

4. 反制处理（Rinsing & Purification）：清洁溶液，在喷涂剂中加入氢离子油，以充分融化孔中残余的水分子，然后将晶体返回样品架。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 总结
多晶硅是一种具有高度空间群分数的无序晶体，具有极高的拓扑纬度，是自然界中具有复杂特性的重要组成部分。不同晶体结构的杂交可以形成多种高维的组合，提供丰富的功能和性能。作为电子工业、计算机系统、显示器等领域的基础材料，多晶硅与常见的单晶硅不同之处，首先是具有较好的硬度和导热性能。其次，多晶硅具有巨大的制造难度，因为晶体结构的各个参数需要通过实验和计算才能确定。除了以上优点外，多晶硅还有一些独特的结构和性质。如，平均振动速率大，物理定律控制尺寸起伏，合成结构中存在多个晶体团簇，不同晶体之间的摩擦力很大。此外，存在多种低温、低电压、灵敏度高、延迟响应速度快等技术。多晶硅在各个领域中得到广泛的应用，促进了科技的进步。

在多晶硅的生产过程中，将杂交晶体浸润或加入不同溶液，以控制多晶结构的纵横比和层间距。也可以使用薄膜退火的方法，将氢离子层投影到晶体表面，使晶体结晶时具有极高的厚度。按照处理流程，多晶硅的生产可以分为四个步骤，分别是熔化处理、涂层处理、腐蚀处理、反制处理。熔化处理是为了防止凝结剂影响晶体的结构，涂层处理是为了使晶体具有薄膜结构，腐蚀处理是为了消除残留的水分子和杂质，最后一步是为了去除喷涂剂中的氢离子，使晶体恢复正常。

## 3.2 熔化处理（Thermal Annealing）
熔化是利用固体液态挤出或融化柱状石墨烯为各种含有不同成分的熔化介质的过程，它可使晶体的表面积和质量扩大，使热运动引起的原子错位发生改变，使结构坍塌，可消除表面附着的氢离子。在这里，固体液态挤出后的晶体具有较大的厚度，故常用来模拟分散表面。熔化之后的晶体具有比较均匀的厚度分布，比单晶硅具有更好的硬度。

具体处理步骤如下：

1. 选取高分子（如有机盐、乙烯）、溶剂（如砂锈、粘土、水、硫酸钾）、介质（如HCN）等作为熔化介质。

2. 将固体石墨烯（CH3OH）预热至300～500°C，使其完全融化。

3. 将低温的晶体放在预热的石墨烯上，允许蒸馏过程持续至95~100°C，一般将其温度定为80°C至90°C。

4. 当温度升至90°C时，熔化介质开始升华，其会使晶体中的氢原子碎裂形成氢原子层，具有一定程度的溶解，而晶体内部的其他原子则趋向于聚集，并形成稳定的晶体。

5. 此时，晶体的平整度达到一定程度，可以用平扫的方式进行检查，如果发现孔隙过大或者孔隙过小，即可调整温度并重复步骤4。

6. 熔化完成后，晶体会向外扩展，具有非常厚的厚度。

## 3.3 涂层处理（Patterning）
涂层是指在晶体表面添加特殊层，并与晶体结构进行接触，使晶体发生表面上的膨胀。目前，市场上常见的涂层技术有硅酸钠、氢、氧、氮和氧化物、氘化物、聚苯酚、珠宝玻璃钢、基质、苏打、光刻胶、半导体材料等。具体步骤如下：

1. 准备涂层材料，一般选择氢、氧或氮等微小颗粒，注意材料的厚度。

2. 用刀切割出所需的涂层片，将片状涂层片垫在晶体上。

3. 用研磨、挤压或注水的方法，将涂层片弯曲缩放到所需大小。

4. 在晶体的平滑边缘附近粘贴两端刚好平行的两条轴，从而形成膨胀的晶体结构。

5. 将涂层材料添加到每一个涂层片上，涂层层数视情况而定。

6. 隔离晶体表面，避免使晶体周围组织膨胀。

7. 测试晶体的特征，观察表面是否膨胀。

## 3.4 腐蚀处理（Emulsion Diffusion）
腐蚀是指一种物理现象，即物体在液体和固体中结合成清晰的图像、形状或轮廓。在我们的晶体表面，腐蚀主要由氢和氦原子带来的孔隙扩张所致。目前，市场上流行的腐蚀材料有聚乙烯、聚氯乙烯、聚氯化钙、二氧化硅、海藻糖等。腐蚀处理的步骤如下：

1. 将晶体浸泡在恒温的液体中，然后加热至温度在300°C左右，保持30秒钟左右。

2. 在温度达到400°C左右时，使用专门的辊温腔温腔封闭晶体表面。

3. 在腔内加速液体的传导，直至腔体外部温度达到370°C。

4. 检测晶体的孔隙，将孔隙两侧的液体涂成浅色或深色的温差色。

5. 对孔隙位置进行微调，使晶体恢复平整。

## 3.5 反制处理（Rinsing & Purification）
反制处理是将通过腐蚀温度过低导致的孔隙膨胀捕获残渣、水分子、杂质，并将其过滤掉的过程。处理的目的是使晶体的结构恢复正常。常见的反制处理方法有开壳反制、制冰反制、电解反制、超声反制等。具体步骤如下：

1. 将晶体浸泡在淡水或其他低温空气中，经历几个小时的反制，待温度达到400°C时进行。

2. 使用专用成分过滤机过滤掉晶体表面上所有的温差色、孔隙残渣、水分子、杂质。

3. 浸泡湿盐（如酒精、石墨烯）可以清洁晶体上的孔隙残渣。

4. 如果晶体表面存在大量的孔隙残渣，可以使用制冰反制进行净化。

5. 通过微波或者毛胚技术扩散聚乙烯等带有不易溶解的物质，如磷酸盐、铬酸钾等。

6. 结束反制后，晶体表面呈现漆黑状态，要求再进行一遍反制处理。
