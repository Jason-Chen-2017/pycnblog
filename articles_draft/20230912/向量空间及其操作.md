# 1.简介
  


向量空间，英文名称是Vector Space。它是一个非常重要的抽象概念，可以用于数学、物理等多种领域。本文将尝试从一个非常简单的例子入手，通过抽象的方式，来理解什么是向量空间。

## 求解空间

设有一个代数方程，如：
$y = ax + b$ ，其中，$a$ 和 $b$ 是未知数，而 $x$ 和 $y$ 是变量。当给定某个值 $c$ 时，方程右端的值不变。那么这个方程组的解集就是一个空间。

比如说，$ax+b=1$ 的解集是实数轴上所有取值为正整数的点 $(x,y)$ 。

或者，$ax^2+bx+c=0$ 的解集则是整个实数平面上的任一根号分数形式的曲线。


由此可见，求解空间是一个非常重要的问题。

## 向量空间

向量空间（Vector space）是一个抽象的概念，由若干向量组成，这些向量之间可以相加、相减、相乘。通常来说，向量空间中的向量表示某个对象的某些特征，并用数值集合表示。

在线性代数中，向量空间被定义为由向量构成的集合，满足两个条件：

1. 向量空间中的每个向量都是一个元素；
2. 每个向量的维度相同，且都在非负整数集$\mathbb{N}$中取值。

为了更好地理解向量空间的概念，我们可以举几个实际例子。

### 一元二次方程

假设有一个一元二次方程 $f(x)=ax^2+bx+c$ ，对于给定的某个实数 $c$ ，方程的解集就是直线。这是因为任何一个 $x$ 值，都会对应着一条直线的切线斜率。而直线在二维坐标系中的表示方式是一条斜率为 $-\frac{b}{2a} \pm \sqrt{\left(\frac{b}{2a}\right)^2-c}$ 的射线。

因此，对任意给定的 $c$ ，方程 $f(x)=ax^2+bx+c$ 在所有的实数上都有唯一的解，即该直线上的切线斜率为 $-\frac{b}{2a} \pm \sqrt{\left(\frac{b}{2a}\right)^2-c}$ 。


我们称这样的一条直线为 $ax^2+bx+c$ 在某个固定的 $c$ 下的解，并将其记作 $\vec v=(\frac{-b}{2a},\frac{b^2}{4a})$ 。

即使对不同的 $c$ ，这条直线的切线都相同。也就是说，每一类方程都对应着同一个向量空间。

### 复数空间

另一个例子是复数空间。复数空间由若干复数构成，它们可以进行加法、减法、乘法和数乘操作。

一般来说，如果对复数空间中的任意一个复数 $\alpha$ ，都存在与 $\alpha$ 共轭的复数 $\beta$ ，则 $\alpha+\beta=\beta+\alpha$ ， $\alpha-\beta=-(\beta-\alpha)$ ， $\alpha\beta=\overline{\beta}\overline{\alpha}$ ， $\alpha\cdot\beta=|\alpha|^2|\beta|$ 。因此，任何一个复数都可以通过一定形式的转化得到。


我们知道，复数 $\alpha$ 可以写成 $\alpha=r(\cos \theta + i\sin \theta)$ （其中 $r$ 表示实部， $\theta$ 表示虚部）。因此，通过数学变换，任何一个复数都可以表示成这个形式。

因此，复数空间也是一种向量空间。但它比一元二次方程的情况要复杂得多。

### 矢量空间

最后一个例子是矢量空间。矢量空间也是一个抽象的概念，由若干向量组成，这些向量之间可以相加、相减、相乘。一般来说，矢量空间中的向量可以用来表示事物的位置或速度、方向等。

例如，三维欧氏空间 $\mathbb R^3$ 中任一点 $p$ 的位置可以用矢量 $\vec p=(x_1, x_2, x_3)^T$ 来表示。矢量也可以用来描述力、速度、位移等。

矢量空间同样具有以下性质：

1. 对矢量空间中的任意向量 $\vec a$ ，都存在对应的反向矢量 $\vec a^*$ ，满足 $\vec a+\vec a^*=0$ 。
2. 对矢量空间中的任意向量 $\vec a$ ，都存在一个单位矢量 $\hat a$ ，满足 $\vec a\cdot\hat a=\|\vec a\|=1$ 。
3. 如果 $\vec a$ 为任意矢量，$\lambda$ 为实数，则 $\lambda\vec a$ 也是矢量。


## 向量空间运算

向量空间的运算很多，这里我们仅讨论最基础的加法和乘法操作。

### 向量的加法

如果 $\vec u$, $\vec v$ 分别代表两个向量，那么 $\vec w=\vec u+\vec v$ 代表的就是这两个向量的加法。

显然，向量的加法满足结合律：$(\vec u+\vec v)+\vec w=\vec u+( \vec v+\vec w )$ 。

另外，零向量 $\vec o$ 满足 $\vec o+\vec v=\vec v$ 。

### 向量的数量积

如果 $\vec u$, $\vec v$ 分别代表两个向量，那么 $\vec u\cdot \vec v$ 或 $\|\vec u\|\ |\vec v\|$ 代表的是这两个向量的数量积（标量积）。

对于向量 $\vec u$, $\vec v$ 来说，矢量的数量积表示的是两个向量夹角的余弦值乘以它们长度的乘积。在高等数学中，一般把它叫做点积 (dot product)。

矢量 $\vec u\cdot \vec v$ 等价于 $\|\vec u\|\ |\vec v\|\ cos\theta$, $\theta$ 表示两者之间的夹角。


显然，如果 $\vec u$，$\vec v$ 是单位矢量，那么 $\vec u\cdot \vec v$ 等于这两个向量的数量积。

## 从向量到矩阵的演进

上面我们看到了矢量空间的基本概念。但是向量空间可以有很多种表示方法，例如坐标形式或基底形式。每种表示方法都可以转化为矩阵的形式。

具体到实数向量空间 $\mathbb R^n$ 中，坐标形式就是 $n\times n$ 的矩阵。

```
[ u_1 ]   [ x_{11} u_1 + x_{12} u_2 +... + x_{1n} u_n ]
[ u_2 ] = [ x_{21} u_1 + x_{22} u_2 +... + x_{2n} u_n ]
...
[ u_n ]   [ x_{n1} u_1 + x_{n2} u_2 +... + x_{nn} u_n ]
```

例如，矢量 $\vec{u}=(1,-2,3)^T$ 可以用矩阵形式 $[\vec{u}]=[1 -2 3]^T$ 来表示。

通用的矩阵形式：

$$ A = 
\begin{bmatrix}
    a & b \\ c & d
\end{bmatrix} $$

其中，$A$ 表示的是 $a$ 和 $d$ 的矩阵。它的行列式（det）等于 $ad-bc$ 。

向量空间的运算也可以通过矩阵乘法来实现。例如，对于 $\mathbb R^2$ 中的矢量 $\vec{u}=(1,-2)^T,\vec{v}=(3,4)^T$ ，它们的点积可以用矩阵乘法来计算：

$$ [\vec{u}]\cdot[\vec{v}] = [(1)(3)+((-2))(4)]=[-2] $$

而矩阵乘法则可以用来计算两个矩阵的乘积。
