数学模型公式详细讲解：

1. 卷积神经网络（CNN）：
$$
y = f(Wx + b)
$$
其中，$x$ 是输入数据，$W$ 是权重矩阵，$b$ 是偏置向量，$f$ 是激活函数。

1. 递归神经网络（RNN）：
$$
h_t = f(Wx_t + Uh_{t-1} + b)
$$
其中，$x_t$ 是时间步 t 的输入数据，$h_t$ 是时间步 t 的隐藏状态，$W$ 是输入到隐藏层的权重矩阵，$U$ 是隐藏层到隐藏层的权重矩阵，$b$ 是偏置向量，$f$ 是激活函数。

1. 自注意力机制（Attention）：
$$
a_{ij} = \frac{\exp(s(h_i, h_j))}{\sum_{k=1}^{T}\exp(s(h_i, h_k))}
$$
$$
h_j' = \sum_{i=1}^{T} a_{ij} h_i
$$
其中，$a_{ij}$ 是注意力权重，表示第 i 个时间步对第 j 个时间步的注意力，$s$ 是同态编码函数，$h_i$ 是第 i 个时间步的隐藏状态，$h_j'$ 是注意力机制后的隐藏状态。

from tensorflow.keras import layers

# 构建自注意力机制（Attention）模型
class Attention(layers.Layer):
    def call(self, inputs):
        query = inputs[0]
        value = inputs[1]
        score = tf.matmul(query, value)
        score = tf.nn.softmax(score)
        context = tf.matmul(score, value)
        return context
