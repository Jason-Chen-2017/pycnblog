## 4.1 代码实例

假设我们需要编写一个简单的密码验证程序，要求程序员需要考虑以下几个问题：

- 密码的长度需要大于8位。
- 密码中需要包含大写字母、小写字母、数字和特殊字符。
- 密码中不能包含可能被用于攻击的敏感词汇。

根据上述要求，我们可以编写以下代码：

```python
import re

def is_valid_password(password):
    if len(password) < 8:
        return False
    if not re.search(r'[A-Z]', password):
        return False
    if not re.search(r'[a-z]', password):
        return False
    if not re.search(r'[0-9]', password):
        return False
    if not re.search(r'[!@#$%^&*(),.?":{}|<>]', password):
        return False
    if re.search(r'[sensitive_word]', password):
        return False
    return True

sensitive_words = ['password', '123456', 'admin']
password = 'P@ssw0rd!'

if is_valid_password(password):
    print('密码验证成功')
else:
    print('密码验证失败')
```

## 4.2 详细解释说明

上述代码实现了一个简单的密码验证程序，通过检查密码的长度、字符类型、敏感词汇等条件来判断密码是否有效。具体来说，代码中使用了以下几个步骤：

1. 首先，我们导入了`re`模块，用于正则表达式匹配。
2. 然后，我们定义了一个`is_valid_password`函数，该函数接受一个密码参数，并检查密码是否满足以下条件：
     - 密码的长度需要大于8位。
     - 密码中需要包含大写字母、小写字母、数字和特殊字符。
     - 密码中不能包含可能被用于攻击的敏感词汇。
3. 接下来，我们定义了一个`sensitive_words`列表，用于存储敏感词汇。
4. 然后，我们定义了一个`password`变量，用于存储要验证的密码。
5. 最后，我们调用`is_valid_password`函数来判断密码是否有效，并输出结果。
