### 4.1 社交网络的邻居推荐

以下是一个简单的Python代码实例，实现了社交网络的邻居推荐：

```python
import numpy as np

def similarity(u, v):
    n = len(u.intersection(v))
    return n / np.linalg.norm(u) / np.linalg.norm(v)

u = set([1, 2, 3, 4, 5])
v = set([2, 3, 4, 5, 6])
print(similarity(u, v))
```

在这个例子中，我们首先计算了用户 $u$ 和用户 $v$ 的相交部分，然后通过计算相似度公式得到了相似度。
