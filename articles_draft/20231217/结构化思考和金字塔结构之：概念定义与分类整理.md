## 1.1 结构化思考的背景与意义

结构化思考的核心在于将复杂问题拆分成较小的子问题，然后逐步解决这些子问题，最终得到问题的解决方案。这种思维方式可以让我们更好地理解问题的本质，提高解决问题的效率和质量。

以一个简单的排序问题为例，展示结构化思考的代码实例和详细解释说明。

def sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        mid = len(arr) // 2
        left = arr[:mid]
        right = arr[mid:]
        return merge(sort(left), sort(right))

def merge(left, right):
    result = []
    while len(left) > 0 and len(right) > 0:
        if left[0] <= right[0]:
            result.append(left.pop(0))
        else:
            result.append(right.pop(0))
    result.extend(left)
    result.extend(right)
    return result
```

解释说明：

- 首先，我们将问题分解为两个子问题：左半部分和右半部分。
- 然后，我们递归地解决这两个子问题，直到问题的规模小于等于1。
- 最后，我们将两个子问题的解决结果通过合并操作得到问题的解决方案。

    def __init__(self, name):
        self.name = name
        self.subdirectories = []
        self.files = []

    def add_subdirectory(self, subdirectory):
        self.subdirectories.append(subdirectory)

    def add_file(self, file):
        self.files.append(file)

    def search(self, keyword):
        results = []
        for file in self.files:
            if keyword in file.name:
                results.append(file)
        for subdirectory in self.subdirectories:
            results.extend(subdirectory.search(keyword))
        return results
```

解释说明：

- 首先，我们将文件目录结构分为多个层次，每个层次包含一个或多个子目录和文件。
