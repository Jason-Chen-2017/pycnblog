## 2. 贪婪算法（Greedy Algorithm）

### 问题描述

给定一个正整数数组，找出和最小的连续子数组。

### 代码实例

```python
def min_subarray_sum(nums):
    n = len(nums)
    min_sum = float('inf')
    curr_sum = 0
    min_len = float('inf')
    curr_len = 0
    for i in range(n):
        curr_sum += nums[i]
        curr_len += 1
        while curr_sum < min_sum:
            min_sum = curr_sum
            min_len = curr_len
            curr_sum -= nums[i - curr_len]
            curr_len -= 1
    return min_len
```

### 解释说明

1. 在每个步骤中，选择当前最优解。
2. 将当前最优解作为下一步的起点。
3. 重复步骤1和步骤2，直到问题得到解决。
