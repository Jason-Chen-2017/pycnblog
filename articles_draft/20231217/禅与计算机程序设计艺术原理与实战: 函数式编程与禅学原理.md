```python

        print_number(2),
        print_number(3)
    )

```

在这个例子中，我们使用Python的`asyncio`库来实现高度并发。我们定义了一个`print_number`函数，它接受一个整数参数并打印它。我们使用`asyncio.gather`函数来并行执行多个`print_number`任务，因此满足高度并发原则。

```python

    return x + y
```
