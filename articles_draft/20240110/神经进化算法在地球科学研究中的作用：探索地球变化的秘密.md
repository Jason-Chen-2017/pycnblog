# 2. 核心概念与联系

## 2.1 神经进化算法（NEA）

神经进化算法（NEA）是一种基于生物进化的优化算法，它结合了生物进化学的进化策略和神经网络的表示和学习方法。NEA的核心思想是通过自然进化过程中的选择和变异来优化问题解 space。具体来说，NEA通过以下几个步骤进行优化：

1. 初始化：生成一个随机的解（神经网络）集合。
2. 评估：根据问题的目标函数对每个解进行评估，得到每个解的适应度。
3. 选择：根据适应度选择一定数量的解进行变异。
4. 变异：对选择到的解进行变异操作，生成新的解。
5. 替代：将新的解加入解集，替代适应度较低的解。
6. 终止：当满足终止条件时，算法终止并返回最佳解。
