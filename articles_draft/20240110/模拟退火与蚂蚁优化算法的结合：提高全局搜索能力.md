# 2.核心概念与联系

## 2.1模拟退火
模拟退火是一种基于温度的优化算法，它通过随机搜索解决方案并根据温度调整搜索步长来逐渐找到最优解。算法的核心思想是将优化问题看作一个能量最小化的问题，然后通过随机搜索来找到能量最低的状态。在这个过程中，温度作为一个参数用于控制搜索的粒度，当温度降低时，搜索的粒度也会逐渐减小，从而逐渐收敛到最优解。

## 2.2蚂蚁优化算法
蚂蚁优化算法是一种基于蚂蚁的自然优化过程的模拟，它通过模拟蚂蚁在寻食过程中的行为来搜索最优解。算法的核心思想是将优化问题看作一个寻食过程，蚂蚁通过在环境中搜索食物来找到最佳的寻食路径。在这个过程中，蚂蚁通过交互和互动来更新自己的寻食路径，从而逐渐找到最优解。

$$
f(x) = -x^2
$$

我们将使用Python编程语言来实现这个算法，代码如下：

```python
import random
import math

def f(x):
    return -x**2

def simulated_annealing(T, T_min, max_iter):
    x = random.uniform(-10, 10)
    y = f(x)
    T_current = T

    for _ in range(max_iter):
        x_new = random.uniform(-10, 10)
        y_new = f(x_new)

        if y_new < y:
            x = x_new
            y = y_new

        delta = y_new - y
        if delta < 0:
            T_current = max(T_min, T_current * 0.99)
        else:
            T_current = max(T_min, T_current * 0.999)

    return x, y

def ant_colony_optimization(T, T_min, max_iter):
    x = random.uniform(-10, 10)
    y = f(x)
    T_current = T

    for _ in range(max_iter):
        x_new = random.uniform(-10, 10)
        y_new = f(x_new)

        if y_new < y:
            x = x_new
            y = y_new

        delta = y_new - y
        if delta < 0:
            T_current = max(T_min, T_current * 0.99)
        else:
            T_current = max(T_min, T_current * 0.999)

    return x, y

T = 100
T_min = 0.1
max_iter = 1000

x1, y1 = simulated_annealing(T, T_min, max_iter)
x2, y2 = ant_colony_optimization(T, T_min, max_iter)

print("Simulated Annealing: x = {}, y = {}".format(x1, y1))
```
