# 2.核心概念与联系

## 2.1 贝叶斯定理

贝叶斯定理是概率论中的一个基本定理，它可以用来计算条件概率。贝叶斯定理的数学表达式如下：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示条件概率，即给定事件$B$发生的情况下，事件$A$的概率；$P(B|A)$ 表示联合概率，即事件$A$发生的情况下，事件$B$的概率；$P(A)$ 表示事件$A$的概率；$P(B)$ 表示事件$B$的概率。

## 3.3 数学模型公式详细讲解

贝叶斯分类器的数学模型公式如下：

1. 类别概率估计：

$$
P(C_i) = \frac{\sum_{x \in C_i} P(x)}{\sum_{j=1}^{n} \sum_{x \in C_j} P(x)}
$$

其中，$P(C_i)$ 表示类别$C_i$的概率；$x$表示数据点；$n$表示类别数量。

2. 条件概率估计：

$$
P(C_i|x) = \frac{P(x|C_i)P(C_i)}{P(x)}
$$

其中，$P(C_i|x)$ 表示给定数据点$x$，类别$C_i$的概率；$P(x|C_i)$ 表示给定类别$C_i$，数据点$x$的概率；$P(x)$ 表示数据点$x$的概率。

3. 分类决策：

$$
\hat{y} = \operatorname{argmax}_{C_i} P(C_i|x)
$$

其中，$\hat{y}$ 表示预测结果；$C_i$ 表示类别；$x$ 表示数据点。

接下来，我们需要训练贝叶斯分类器。这里我们简单地计算每个词汇的概率分布。

```python
# 计算每个词汇的概率分布
word_probabilities = {}
for data_point in data_points:
    if data_point in word_to_id:
        data_point_id = word_to_id[data_point]
        if data_point not in word_probabilities:
            word_probabilities[data_point] = [0] * len(word_to_id)
        word_probabilities[data_point][data_point_id] += 1

for word, probabilities in word_probabilities.items():
    total_count = sum(probabilities)
    word_probabilities[word] = [count / total_count for count in probabilities]
```
