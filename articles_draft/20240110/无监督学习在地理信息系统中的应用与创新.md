## 4.3 自组织映射的代码实例

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机数据
X = np.random.rand(100, 2)

# 设置网格大小
grid_size = 10

# 实现自组织映射
def self_organizing_map(X, grid_size):
    # 初始化网格
    grid = np.zeros((grid_size, grid_size))
    # 计算数据点与网格点的距离
    distances = np.sqrt(np.sum((X - grid)**2, axis=1))
    # 更新网格点
    for i in range(grid_size):
        for j in range(grid_size):
            grid[i, j] = X[np.argmin(distances[i*grid_size + j])]
    return grid

grid = self_organizing_map(X, grid_size)

# 可视化自组织映射
plt.imshow(grid, cmap='hot', interpolation='nearest')
plt.show()
```

详细解释说明：

- 首先，我们从numpy库中导入np.random.rand函数，从而生成随机数据。
- 然后，我们设置网格大小为10。
- 接着，我们实现自组织映射，并将其可视化。
