## 3.1 图像压缩
### 3.1.1 均匀分布
在图像压缩中，均匀分布是一种常用的概率分布模型。假设图像灰度的概率分布为 $p(x) = \frac{1}{L}$，其中 $x \in [0, L-1]$，$L$ 是灰度级别。则图像的平均灰度为 $\mu = \sum_{x=0}^{L-1} x \cdot p(x)$。

### 3.1.2 高斯分布
高斯分布是另一种常用的概率分布模型，它可以更好地描述图像灰度的分布。高斯分布的概率密度函数为：

$$
p(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中 $\mu$ 是均值，$\sigma$ 是标准差。通过最大化概率密度函数，可以得到图像的均值和方差。

### 3.1.3 压缩算法
根据上述概率分布模型，可以得到不同的压缩算法。例如，均匀分布下可以使用均值方差编码（MSE）算法，高斯分布下可以使用JPEG算法。这些算法通过对概率分布的建模，将图像数据压缩为更小的尺寸，从而减少存储和传输的计算成本。

## 3.2 图像模糊化
### 3.2.1 指数分布
指数分布是一种常用的概率分布模型，它可以描述图像像素值的分布。指数分布的概率密度函数为：

$$
p(x) = \frac{1}{\lambda}e^{-\frac{x}{\lambda}}
$$

其中 $\lambda$ 是参数。通过最大化概率密度函数，可以得到图像的像素值分布。

### 3.2.2 模糊化算法
根据上述概率分布模型，可以得到不同的模糊化算法。例如，指数分布下可以使用高斯模糊算法，它通过对图像像素值的概率分布进行建模，生成模糊化的图像，从而保护隐私信息。

## 3.3 图像分割
### 3.3.1 混合高斯分布
混合高斯分布是一种常用的概率分布模型，它可以描述图像像素值的分布。混合高斯分布的概率密度函数为：

$$
p(x) = \sum_{k=1}^K \alpha_k \frac{1}{\sqrt{2\pi\sigma_k^2}}e^{-\frac{(x-\mu_k)^2}{2\sigma_k^2}}
$$

其中 $\alpha_k$ 是混合权重，$\mu_k$ 是均值，$\sigma_k$ 是标准差。通过最大化概率密度函数，可以得到图像的像素值分布。

### 3.3.2 分割算法

import numpy as np

def mse_encode(image, L):
    # 计算图像的均值和方差
    mean = np.mean(image)
    variance = np.var(image)
    
    # 编码
    encoded_image = []
    for x in image:
        if x < mean - np.sqrt(variance):
            encoded_x = 0
        elif x < mean - np.sqrt(2 * variance):
            encoded_x = 1
        elif x < mean:
            encoded_x = 2
        elif x < mean + np.sqrt(2 * variance):
            encoded_x = 3
        elif x < mean + np.sqrt(variance):

            encoded_x = 5
        encoded_image.append(encoded_x)

## 4.3 图像分割
### 4.3.1 K-means算法
```python
import numpy as np
from sklearn.cluster import KMeans

def k_means_segmentation(image, K):
    # 归一化
    image_normalized = (image - np.min(image)) / (np.max(image) - np.min(image))
    
    # 分割
    kmeans = KMeans(n_clusters=K, random_state=0).fit(image_normalized.reshape(-1, 1))
    labels = kmeans.labels_
    
    # 统计每个聚类中像素点的数量
    counts = np.bincount(labels)
    
    # 获取聚类中像素点的均值
    centroids = kmeans.cluster_centers_
    return centroids, counts
```
