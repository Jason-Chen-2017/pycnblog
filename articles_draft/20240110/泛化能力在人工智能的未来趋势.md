## 3.3 数学模型公式详细讲解

在这一部分，我们将详细讲解泛化能力的数学模型公式。

### 3.3.1 线性回归

线性回归是一种简单的机器学习算法，它可以用来预测连续变量。线性回归的数学模型公式如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

在这个公式中，$y$是预测变量，$x_1, x_2, ..., x_n$是输入变量，$\beta_0, \beta_1, \beta_2, ..., \beta_n$是参数，$\epsilon$是误差。

### 3.3.2 逻辑回归

逻辑回归是一种用于预测二值变量的机器学习算法。逻辑回归的数学模型公式如下：

$$
P(y=1|x_1, x_2, ..., x_n) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - ... - \beta_nx_n}}
$$

在这个公式中，$P(y=1|x_1, x_2, ..., x_n)$是预测概率，$\beta_0, \beta_1, \beta_2, ..., \beta_n$是参数。

### 3.3.3 支持向量机

支持向量机是一种用于分类和回归的机器学习算法。支持向量机的数学模型公式如下：

$$
\min_{\mathbf{w}, b} \frac{1}{2}\mathbf{w}^T\mathbf{w} + C\sum_{i=1}^n\xi_i
$$

$$
y_i(\mathbf{w}^T\mathbf{x_i} + b) \geq 1 - \xi_i, \xi_i \geq 0
$$

在这个公式中，$\mathbf{w}$是权重向量，$b$是偏置项，$C$是正则化参数，$\xi_i$是松弛变量。

### 3.3.4 神经网络

神经网络是一种复杂的机器学习算法，它可以用来预测连续变量和二值变量。神经网络的数学模型公式如下：

$$
z_j^l = \sum_{i=1}^n w_{ij}^l x_i^l + b_j^l
$$

$$
a_j^l = f(z_j^l)
$$

在这个公式中，$z_j^l$是层$l$的神经元$j$的输入，$x_i^l$是层$l$的神经元$i$的输出，$w_{ij}^l$是层$l$的神经元$j$和$i$的权重，$b_j^l$是层$l$的神经元$j$的偏置，$f$是激活函数。

```python
import numpy as np

# 生成数据
np.random.seed(0)
X = np.random.rand(100, 1)
y = 3 * X + 2 + np.random.rand(100, 1)

# 训练模型
def sigmoid(z):
    return 1 / (1 + np.exp(-z))

def cost_function(X, y, theta):
    m = len(y)
    h = sigmoid(X.dot(theta))
    cost = (-1/m) * np.sum(y.dot(np.log(h)) + (1 - y).dot(np.log(1 - h)))
    return cost

def gradient_descent(X, y, theta, alpha, iterations):
    m = len(y)
    cost_history = np.zeros(iterations)
    for i in range(iterations):
        h = sigmoid(X.dot(theta))
        gradient = (1/m) * X.T.dot(h - y)
        theta = theta - alpha * gradient
        cost_history[i] = cost_function(X, y, theta)
    return theta, cost_history

# 预测
X_test = np.array([[0.5]])
y_pred = sigmoid(X_test.dot(theta)).round()
```
