## 3.3 数学模型公式
在本节中，我们将介绍一些常用的数学模型公式，用于分析随机试验和实验室测试的数据。

### 3.3.1 均值和方差
均值（$\mu$）是一组数字的集合中的数字总和除以集合大小的结果。方差（$\sigma^2$）是一组数字的集合中数字与均值之间差异的平均值的平方。这两个概念是评估样本分布的重要指标。

$$
\mu = \frac{1}{N} \sum_{i=1}^{N} x_i
$$

$$
\sigma^2 = \frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2
$$

### 3.3.2 相关性和相关系数
相关性是两个变量之间的关系，它们在同一集合中的变化呈现相同的趋势。相关系数（$r$）是一个介于-1和1之间的数字，用于度量两个变量之间的相关性。

$$
r = \frac{\sum_{i=1}^{N} (x_i - \mu)(y_i - \mu)}{\sqrt{\sum_{i=1}^{N} (x_i - \mu)^2} \sqrt{\sum_{i=1}^{N} (y_i - \mu)^2}}
$$

### 3.3.3 方程组解
在某些情况下，我们需要解决一组方程来获取样本或系统的参数。这可以通过各种方法实现，例如替代法、消元法和矩阵法等。

## 4.1 随机试验的代码实例
```python
import numpy as np
import pandas as pd
from scipy import stats

# 生成随机样本
np.random.seed(42)
sample_size = 100
x = np.random.randn(sample_size)

# 应用不同的条件
condition1 = x > 0
condition2 = x < 0

# 计算条件下的均值和方差
mean1, var1 = stats.tmean(x[condition1], nan_policy='omit'), stats.tvar(x[condition1], nan_policy='omit')
mean2, var2 = stats.tmean(x[condition2], nan_policy='omit'), stats.tvar(x[condition2], nan_policy='omit')

# 计算相关系数
correlation = stats.pearsonr(x[condition1], x[condition2])[0]

print("Condition 1 mean:", mean1)
print("Condition 1 variance:", var1)
print("Condition 2 mean:", mean2)
print("Condition 2 variance:", var2)
print("Correlation:", correlation)
```

## 4.2 实验室测试的代码实例
```python
import numpy as np
import pandas as pd

# 生成测试数据
test_conditions = [0.5, 1.0, 1.5, 2.0, 2.5]
test_results = [100, 105, 110, 115, 120]

# 计算均值和方差
mean_results = np.mean(test_results)
var_results = np.var(test_results)

# 计算相关性和相关系数
correlation = np.corrcoef(test_conditions, test_results)[0, 1]

print("Mean results:", mean_results)
print("Variance results:", var_results)
print("Correlation:", correlation)
```
