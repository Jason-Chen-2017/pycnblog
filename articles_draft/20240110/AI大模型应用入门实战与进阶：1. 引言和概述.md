# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 神经网络的基本结构

神经网络由多个节点（神经元）和连接这些节点的权重组成。每个节点接收来自其他节点的输入，进行计算后输出结果。神经网络通过训练来学习和决策。

### 3.1.1 节点（神经元）

节点（神经元）是神经网络的基本组成单元。它接收来自其他节点的输入，进行计算后输出结果。节点的计算公式为：

$$
y = f(x) = f(\sum_{i=1}^{n} w_i * x_i + b)
$$

其中，$x$ 是输入向量，$w_i$ 是权重向量，$b$ 是偏置项，$f$ 是激活函数。

### 3.1.2 权重

权重是连接不同节点的连接线的权重。权重用于调整节点之间的关系，使得模型能够学习和决策。权重的更新通常使用梯度下降法或其他优化算法来实现。

### 3.1.3 激活函数

激活函数是神经网络中的一个关键组件。它用于将节点的输入映射到输出。常见的激活函数有 sigmoid、tanh 和 ReLU 等。

## 3.2 深度学习的基本算法

深度学习的基本算法包括前向传播、后向传播和梯度下降等。

### 3.2.1 前向传播

前向传播是深度学习中的一个关键步骤。它用于将输入数据通过多个节点传递给输出节点。前向传播的公式为：

$$
y = f(x) = f(\sum_{i=1}^{n} w_i * x_i + b)
$$

其中，$x$ 是输入向量，$w_i$ 是权重向量，$b$ 是偏置项，$f$ 是激活函数。

### 3.2.2 后向传播

后向传播是深度学习中的一个关键步骤。它用于计算节点的梯度，以便进行权重的更新。后向传播的公式为：

$$
\frac{\partial L}{\partial w_i} = \frac{\partial L}{\partial y} * \frac{\partial y}{\partial w_i}
$$

其中，$L$ 是损失函数，$y$ 是输出节点的输出。

### 3.2.3 梯度下降

梯度下降是深度学习中的一个关键算法。它用于根据梯度来更新权重，使模型能够学习和决策。梯度下降的公式为：

$$
w_{i+1} = w_i - \alpha * \frac{\partial L}{\partial w_i}
$$

其中，$w_{i+1}$ 是更新后的权重，$w_i$ 是当前的权重，$\alpha$ 是学习率。

        self.fc1 = nn.Linear(10, 20)
        self.fc2 = nn.Linear(20, 1)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# 创建神经网络实例
net = Net()

# 定义损失函数和优化器

optimizer = optim.SGD(net.parameters(), lr=0.01)
