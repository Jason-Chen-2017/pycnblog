### 1.4.1 代码实例

以下是一个简单的数字支付代码实例：

```python
import random

def generate_key_pair():
    p, q = 1009, 1013
    n = p * q
    phi = (p - 1) * (q - 1)
    e = 65537
    d = pow(e, -1, phi)
    return (e, n, d)

def encrypt(m, e, n):
    return pow(m, e, n)

def decrypt(c, d, n):
    return pow(c, d, n)

def sign(m, d, n):
    return decrypt(encrypt(hash(m), e, n), d, n)

def verify(m, s, e, n):
    return encrypt(hash(m), s, n) == m

e, n, d = generate_key_pair()
m = 123456
s = sign(m, d, n)
if verify(m, s, e, n):
    print("验证通过")
else:
    print("验证失败")
```

### 1.4.2 代码解释

1. `generate_key_pair`：生成RSA密钥对，包括公钥（e, n）和私钥（d）。
2. `encrypt`：对消息（m）进行加密，使用公钥（e, n）。
3. `decrypt`：对加密后的消息（c）进行解密，使用私钥（d, n）。
4. `sign`：对消息（m）进行签名，使用私钥（d, n）。
5. `verify`：验证消息（m）是否来自签名者，使用公钥（e, n）和签名（s）。
