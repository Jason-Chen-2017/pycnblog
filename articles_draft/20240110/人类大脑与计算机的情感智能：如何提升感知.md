### 4.1情感识别代码实例

我们将使用一个简单的情感分析示例来演示情感识别的过程。这个示例使用 Python 和 NLTK 库来实现情感分析。

```python
import nltk
from nltk.sentiment import SentimentIntensityAnalyzer

# 初始化情感分析器
sia = SentimentIntensityAnalyzer()

# 测试数据
texts = ["I love this product!", "I hate this product!"]

# 情感分析
for text in texts:
    sentiment = sia.polarity_scores(text)
    print(f"Text: {text}, Sentiment: {sentiment}")
```

输出结果：

```

Text: I hate this product!, Sentiment: {'neg': 0.8, 'neu': 0.0, 'pos': 0.2, 'compound': -0.806}
```

在这个示例中，我们使用了 NLTK 库的情感分析器来对测试数据进行情感识别。情感分析器会返回四个值：负面情感、中性情感、正面情感和总体情感。
