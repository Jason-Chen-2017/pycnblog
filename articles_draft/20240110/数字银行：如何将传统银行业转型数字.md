## 4.1 存款业务

存款业务是数字银行的核心业务之一。以下是一个简单的存款业务代码实例：

```python
class Account:
    def __init__(self, account_number, owner, balance=0):
        self.account_number = account_number
        self.owner = owner
        self.balance = balance

    def deposit(self, amount):
        self.balance += amount
        print(f"存款成功，余额为：{self.balance}")

    def withdraw(self, amount):
        if amount > self.balance:
            print("余额不足")
        else:
            self.balance -= amount
            print(f"取款成功，余额为：{self.balance}")
```

这个代码实例中，我们定义了一个 `Account` 类，用于表示银行账户。这个类有三个属性：账户号、账户持有人、账户余额。这个类还有两个方法：存款方法 `deposit` 和取款方法 `withdraw`。

## 4.2 贷款业务

贷款业务是数字银行的另一个核心业务之一。以下是一个简单的贷款业务代码实例：

```python
class Loan:
    def __init__(self, loan_number, borrower, loan_amount, interest_rate, loan_term):
        self.loan_number = loan_number
        self.borrower = borrower
        self.loan_amount = loan_amount
        self.interest_rate = interest_rate
        self.loan_term = loan_term

    def calculate_monthly_payment(self):
        monthly_interest_rate = self.interest_rate / 12 / 100
        loan_term_months = self.loan_term * 12
        monthly_payment = (self.loan_amount * monthly_interest_rate) / (1 - (1 + monthly_interest_rate) ** -loan_term_months)
        return monthly_payment

    def display_monthly_payment(self):
        monthly_payment = self.calculate_monthly_payment()
        print(f"每月还款金额为：{monthly_payment}")
```

这个代码实例中，我们定义了一个 `Loan` 类，用于表示贷款。这个类有五个属性：贷款编号、贷款人、贷款金额、年利率、贷款期限。这个类还有两个方法：计算每月还款金额方法 `calculate_monthly_payment` 和显示每月还款金额方法 `display_monthly_payment`。
