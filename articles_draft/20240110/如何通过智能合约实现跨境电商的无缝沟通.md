# 4.具体代码实例和详细解释说明

## 4.1 一个简单的转账智能合约

以下是一个简单的转账智能合约的代码实例：

```solidity
pragma solidity ^0.5.0;

contract Transfer {
    address public sender;
    address public receiver;
    uint public amount;

    event TransferEvent(address indexed _sender, uint _amount);

    constructor(address _sender, address _receiver, uint _amount) public {
        sender = _sender;
        receiver = _receiver;
        amount = _amount;
    }

    function transfer() public {
        require(sender.balance >= amount);
        sender.transfer(amount);
        receiver.transfer(amount);
        emit TransferEvent(sender, amount);
    }
}
```

该智能合约包括以下几个部分：

1. 状态变量：sender、receiver、amount。
2. 事件：TransferEvent。
3. 构造函数：初始化 sender、receiver、amount。
4. transfer 函数：执行转账操作。

## 4.2 一个简单的购物智能合约

以下是一个简单的购物智能合约的代码实例：

```solidity
pragma solidity ^0.5.0;

contract Shopping {
    address public buyer;
    address public seller;
    uint public price;
    uint public quantity;

    event PurchaseEvent(address indexed _buyer, uint _quantity, uint _price);

    constructor(address _seller, uint _price, uint _quantity) public {
        buyer = msg.sender;
        seller = _seller;
        price = _price;
        quantity = _quantity;
    }

    function purchase() public payable {
        require(msg.value >= price * quantity);
        seller.transfer(msg.value);
        buyer.transfer(msg.value);
        emit PurchaseEvent(buyer, quantity, price);
    }
}
```

该智能合约包括以下几个部分：

1. 状态变量：buyer、seller、price、quantity。
2. 事件：PurchaseEvent。
3. 构造函数：初始化 buyer、seller、price、quantity。
4. purchase 函数：执行购物操作。
