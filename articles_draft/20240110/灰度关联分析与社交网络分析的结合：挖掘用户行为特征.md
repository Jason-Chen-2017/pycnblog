```python
# 读取原始数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.dropna()

# 数据转换
data['age'] = data['age'].astype(int)
data['gender'] = data['gender'].map({'male': 0, 'female': 1})

# 数据矫正
data = data[(data['age'] >= 18) & (data['age'] <= 60)]
```

## 4.2 社交网络构建

接下来，我们需要根据用户的行为数据，构建一个社交网络，其中节点表示用户，边表示用户之间的关系。以下是一个简单的社交网络构建代码实例：

```python
from networkx import Graph

# 创建一个空的社交网络
G = Graph()

# 添加节点
for user in data['user_id'].unique():
    G.add_node(user)

# 添加边
for index, row in data.iterrows():
    if row['user_id'] != row['friend_id']:
        G.add_edge(row['user_id'], row['friend_id'])
```

## 4.3 关联规则挖掘

然后，我们需要根据用户行为数据，进行关联规则挖掘，以发现一种事物与另一种事物之间存在关联关系的规则。以下是一个简单的关联规则挖掘代码实例：

```python
from mlxtend.frequent_patterns import apriori
from mlxtend.frequent_patterns import association_rules

# 生成频繁项集
frequent_itemsets = apriori(data['behavior'], min_support=0.05, use_colnames=True)

# 生成关联规则
rules = association_rules(frequent_itemsets, metric='lift', min_threshold=1)
```
