## 4.1 代码实例

我们以一个简单的多目标优化问题为例，来展示决策平面的应用。

假设我们有一个两目标优化问题，目标是最小化目标函数$f_1(x)$和$f_2(x)$，同时满足约束条件$g(x) \leq 0$和$h(x) = 0$。具体问题描述如下：

$$
\begin{aligned}
\min_{x} & \quad f_1(x) = x_1^2 + x_2^2 \\
\min_{x} & \quad f_2(x) = -x_1^2 - x_2^2 \\
s.t. & \quad g(x) = x_1 - x_2 \leq 0 \\
& \quad h(x) = x_1^2 + x_2^2 - 1 = 0 \\
& \quad x \in \mathbb{R}^2
\end{aligned}
$$

我们可以使用Python的`scipy.optimize`库来解决这个问题。首先，我们需要导入相关库：

```python
from scipy.optimize import minimize
```

然后，我们需要定义目标函数和约束条件：

```python
def f1(x):
    return x[0]**2 + x[1]**2

def f2(x):
    return -x[0]**2 - x[1]**2

def g(x):
    return x[0] - x[1]

def h(x):
    return x[0]**2 + x[1]**2 - 1
```

接下来，我们需要定义约束条件：

```python
constraints = [{'type': 'ineq', 'fun': g},
               {'type': 'eq', 'fun': h}]
```

最后，我们可以使用`minimize`函数来求解问题：

```python
result = minimize(lambda x: [f1(x), f2(x)], x0=[0.5, 0.5], constraints=constraints)
```


从结果中我们可以看到，最优解为$x^* = [1, 1]$，最优值为$f_1(x^*) = -1$和$f_2(x^*) = 1$。
