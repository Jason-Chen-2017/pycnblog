线性分类的目标是找到一个最佳的分界线，使得在该线上的数据点属于一个类别，而在线上的数据点属于另一个类别。这个过程通常使用梯度下降法来实现，其中梯度下降法是一种优化算法，用于最小化一个函数的值。

## 4.3 训练模型

然后，我们需要使用梯度下降法来训练模型，找到最佳的权重向量和偏置项。例如，我们可以使用以下代码来训练模型：

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def loss(y_true, y_pred):
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))

def gradient_descent(X, y, w, b, learning_rate, iterations):
    for i in range(iterations):
        y_pred = sigmoid(X.dot(w) + b)
        gradients = X.T.dot(y_pred - y)
        w -= learning_rate * gradients
        b -= learning_rate * np.mean(y_pred - y)
    return w, b

learning_rate = 0.01
iterations = 1000
w, b = gradient_descent(X, y, w, b, learning_rate, iterations)
```
