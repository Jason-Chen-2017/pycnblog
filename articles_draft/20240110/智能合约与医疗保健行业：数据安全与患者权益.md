## 4. 具体代码实例和详细解释说明

### 4.1 智能合约的代码实例

以下是一个简单的智能合约的代码实例，它使用了 Solidity 语言编写：

```solidity
pragma solidity ^0.5.0;

contract MedicalContract {
    address public owner;
    mapping(address => uint256) public patientBalance;
    uint256 public totalBalance;

    event PatientDeposit(address indexed patient, uint256 amount);
    event PatientWithdraw(address indexed patient, uint256 amount);

    constructor() public {
        owner = msg.sender;
    }

    function deposit() public payable {
        require(msg.value > 0, "Amount must be greater than 0");
        patientBalance[msg.sender] += msg.value;
        totalBalance += msg.value;
        emit PatientDeposit(msg.sender, msg.value);
    }

    function withdraw() public {
        require(msg.sender == owner, "Only owner can withdraw");
        uint256 amount = patientBalance[msg.sender];
        require(amount > 0, "Insufficient balance");
        patientBalance[msg.sender] = 0;
        totalBalance -= amount;
        payable(msg.sender).transfer(amount);
        emit PatientWithdraw(msg.sender, amount);
    }
}
```

### 4.2 代码实例的详细解释说明

这个智能合约的代码实例包括以下几个部分：

1. 合约定义：合约定义了合约的名称（MedicalContract）和使用的语言版本（Solidity ^0.5.0）。
2. 变量定义：合约定义了一些变量，例如所有者（owner）、患者余额（patientBalance）和总余额（totalBalance）。
3. 事件定义：合约定义了两个事件，用于记录患者存款和取款的操作。
4. 构造函数：构造函数用于初始化合约，设置所有者为创建合约的地址。
5. 存款函数：存款函数允许患者通过支付以太坊gas费来存款。
6. 取款函数：取款函数允许所有者通过将患者的余额转移到自己的地址来取款。
