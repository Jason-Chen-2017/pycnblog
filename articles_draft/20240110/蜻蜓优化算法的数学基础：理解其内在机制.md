### 3.2具体操作步骤

1. 初始化蜻蜓群：生成一个包含n个蜻蜓的群，每个蜻蜓表示为一个在解决空间中的向量。
2. 计算每个蜻蜓的闪烁光线强度：根据目标函数的值，计算每个蜻蜓的闪烁光线强度。
3. 更新蜻蜓的位置：根据闪烁光线强度和距离其他蜻蜓的距离，更新蜻蜓的位置。
4. 更新蜻蜓的闪烁光线强度：根据新的位置，更新蜻蜓的闪烁光线强度。
5. 重复步骤2-4，直到满足终止条件。

### 3.3数学模型公式

蜻蜓优化算法的数学模型可以表示为：

$$
\begin{aligned}
&x_i(t+1) = x_i(t) + \beta_0 \exp(-\gamma r_{ij}^2) \cdot \beta_1 \cdot (x_j(t) - x_i(t)) + \alpha \cdot \beta_2 \cdot rand() \\
&f(x_i(t+1)) = f(x_i(t))
\end{aligned}
$$

其中，$x_i(t)$表示第i个蜻蜓在第t时刻的位置，$r_{ij}$表示第i个蜻蜓与第j个蜻蜓之间的距离，$\beta_0$、$\beta_1$、$\beta_2$是常数，$\alpha$是一个随机数。

```python
import numpy as np
import matplotlib.pyplot as plt

def firefly_algorithm(f, x_min, x_max, n, max_iter):
    # 初始化蜻蜓群
    x = np.random.uniform(x_min, x_max, (n, len(x_min)))
    y = f(x)

    # 计算闪烁光线强度
    beta0 = 1
    gamma = 0.5
    beta1 = 1
    alpha = 0.5
    beta2 = 1

    for t in range(max_iter):
        for i in range(n):
            r = np.random.randint(n)
            r_ij = np.linalg.norm(x[i] - x[r])
            beta = beta0 * np.exp(-gamma * r_ij**2) * beta1
            x[i] = x[i] + beta * (x[r] - x[i]) + alpha * beta2 * np.random.randn(len(x_min))

        # 计算新的闪烁光线强度
        y_new = f(x)

        # 更新蜻蜓群
        idx = np.argmax(y_new)
        x = x[idx:]
        y = y[:len(x)]

    return x, y

# 测试函数
def f(x):
    return -(x[0]**2 + x[1]**2)

# 参数设置
x_min = [-10, -10]
x_max = [10, 10]
n = 10
max_iter = 100

# 运行蜻蜓优化算法
x_opt, y_opt = firefly_algorithm(f, x_min, x_max, n, max_iter)

# 绘制结果
plt.scatter(x_opt[:, 0], x_opt[:, 1], c=y_opt, cmap='viridis', marker='o')
plt.xlabel('x1')
plt.ylabel('x2')
plt.colorbar(label='Amplitude')
plt.show()
```
