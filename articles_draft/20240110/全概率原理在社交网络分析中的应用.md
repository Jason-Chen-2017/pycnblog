# 2.核心概念与联系

## 2.1 社交网络

社交网络是一种由人们之间建立的关系和互动组成的网络。社交网络可以用图形模型表示，其中节点表示人员，边表示关系或互动。社交网络可以用以下几个基本属性来描述：

- 节点（Vertex）：节点表示社交网络中的个体，如人、组织等。
- 边（Edge）：边表示个体之间的关系或互动。
- 权重（Weight）：边可以具有权重，表示关系或互动的强度。
- 属性（Attribute）：节点和边可以具有属性，如个人信息、组织信息等。

## 2.2 全概率原理

全概率原理是一种用于解决高维概率模型的随机采样方法，它基于贝叶斯定理和条件独立性。全概率原理的核心思想是，通过对每个参数进行独立采样，得到其边际分布，从而得到整个模型的分布。

全概率原理的主要步骤包括：

1. 确定模型的参数和分布。
2. 为每个参数定义一个条件独立性条件。
3. 对每个参数进行独立采样。
4. 计算边际分布。
