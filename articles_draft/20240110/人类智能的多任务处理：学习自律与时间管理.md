### 4.1 任务调度算法实现

```python
def schedule(tasks):
    scores = []
    for task in tasks:
        priority = task['priority']
        completion_time = task['completion_time']
        score = priority * completion_time
        scores.append(score)
    return max(scores)
```

任务调度算法实现的详细解释说明：

1. 定义一个 `schedule` 函数，接收一个任务列表 `tasks` 作为参数。
2. 创建一个空列表 `scores`，用于存储任务调度得分。
3. 遍历任务列表，对于每个任务，计算任务调度得分：优先级 * 完成时间。
4. 将得分添加到 `scores` 列表中。
5. 返回 `scores` 列表中的最大得分。
