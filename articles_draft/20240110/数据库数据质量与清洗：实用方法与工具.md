## 4.2 数据质量评估代码实例
```python
import pandas as pd

# 读取数据
data = pd.read_csv('clean_data.csv')

# 数据准确性评估
accuracy = data.apply(lambda x: x == x.replace(r'[^\w\s]', '', regex=True))
accuracy_score = accuracy.mean()

# 数据完整性评估
completeness = data.isnull().sum() / data.shape[0]
completeness_score = 1 - completeness

# 数据一致性评估
consistency = data.duplicated().sum() / data.shape[0]
consistency_score = 1 - consistency

# 数据时效性评估
timeliness = data['date'].max() - data['date'].min()
timeliness_score = timeliness.days / (data.shape[0] * 30)

# 数据可用性评估
availability = data.shape[0] / len(data.dropna())
availability_score = 1 - (1 - availability)

# 保存评估结果
result = pd.DataFrame({
    '准确性': [accuracy_score],
    '完整性': [completeness_score],
    '一致性': [consistency_score],
    '时效性': [timeliness_score],
    '可用性': [availability_score]
})
result.to_csv('quality_evaluation.csv', index=False)
```
