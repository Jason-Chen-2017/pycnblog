## 4.1 线性回归示例

以下是一个简单的线性回归示例：

```python
import numpy as np

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 2 * x + 1 + np.random.randn(100, 1) * 0.5

# 训练模型
X = np.hstack((np.ones((100, 1)), x))
theta = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)

# 预测
x_test = np.array([[0.5], [1], [1.5]])
X_test = np.hstack((np.ones((3, 1)), x_test))
y_predict = X_test.dot(theta)

# 评估
mse = np.mean((y - y_predict) ** 2)
print("MSE:", mse)
```

在这个示例中，我们首先生成了一组线性回归数据，然后使用最小二乘法训练了线性回归模型，接着使用训练好的模型对测试数据进行预测，最后计算了均方误差（MSE）来评估模型的性能。

## 4.2 逻辑回归示例

以下是一个简单的逻辑回归示例：

```python
import numpy as np

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 1 if x > 0.5 else 0 + np.random.randn(100, 1) * 0.5

# 训练模型
X = np.hstack((np.ones((100, 1)), x))
theta = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)

# 预测
x_test = np.array([[0.5], [1], [1.5]])
X_test = np.hstack((np.ones((3, 1)), x_test))
y_predict = 1 / (1 + np.exp(-X_test.dot(theta)))

# 评估
accuracy = np.mean(y_predict > 0.5 == y)
print("Accuracy:", accuracy)
```

在这个示例中，我们首先生成了一组逻辑回归数据，然后使用最大似然估计训练了逻辑回归模型，接着使用训练好的模型对测试数据进行预测，最后计算了准确率（Accuracy）来评估模型的性能。
