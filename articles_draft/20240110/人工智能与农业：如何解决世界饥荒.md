## 4. 具体代码实例和详细解释说明

### 4.1 预测气候变化

我们使用 Python 编程语言来编写代码实例。

```python
import numpy as np
import matplotlib.pyplot as plt

# 加载气候数据
data = np.loadtxt('climate_data.txt')

# 划分训练集和测试集
train_data = data[:int(0.8*len(data))]
test_data = data[int(0.8*len(data)):]

# 训练模型
model = np.polyfit(train_data[:, 0], train_data[:, 1], 3)

# 预测气候变化
x = np.linspace(0, len(test_data), 100)
y = model[0] + model[1]*x + model[2]*x**2 + model[3]*x**3
x = np.linspace(0, len(train_data), 100)
y = model[0] + model[1]*x + model[2]*x**2 + model[3]*x**3

# 绘制图像
plt.plot(x, y, 'r-', label='Predicted')
plt.plot(test_data[:, 0], test_data[:, 1], 'bo', label='Actual')
plt.legend()
plt.show()
```
