```python
import numpy as np

def parallel_processing(data, process_func):
    n = len(data)
    results = []
    for i in range(n):
        results.append(process_func(data[i]))
    return results

data = np.random.rand(1000)
process_func = lambda x: x * 2
results = parallel_processing(data, process_func)
print(results)
```

在这个代码实例中，我们使用了Python的NumPy库来生成1000个随机数作为输入数据。然后，我们定义了一个`parallel_processing`函数，它接受一个数据列表和一个处理函数作为参数。这个函数将数据列表分成多个部分，并同时对每个部分进行处理。最后，我们将处理结果存储在一个列表中，并返回这个列表。

import multiprocessing as mp
