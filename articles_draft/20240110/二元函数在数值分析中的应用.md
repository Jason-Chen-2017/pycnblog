### 1.4.1 求解方程组的例子

在本节中，我们将通过一个求解方程组的例子来说明二元函数在数值分析中的应用。我们需要求解以下方程组：

$$
\begin{cases}
x + y = 1 \\
x - y = 1
\end{cases}
$$

我们可以使用固点迭代法来解决这个问题。具体的操作步骤如下：

1. 选择一个初始点（例如，$(0, 0)$）。
2. 计算二元函数的值。
3. 更新迭代点。
4. 重复步骤2和步骤3，直到满足某个停止条件。

通过以下代码实现：

```python
def fixed_point_iteration(x0, y0, tol=1e-6, max_iter=1000):
    x, y = x0, y0
    for _ in range(max_iter):
        x_new = x + y - 1
        y_new = x - y + 1
        if abs(x_new - x) < tol and abs(y_new - y) < tol:
            break
        x, y = x_new, y_new
    return x, y

x0, y0 = 0, 0
x, y = fixed_point_iteration(x0, y0)
print("解：x =", x, ", y =", y)
```

### 1.4.2 最小化最大化问题的例子

在本节中，我们将通过一个最小化问题的例子来说明二元函数在数值分析中的应用。我们需要找到使以下目标函数取得最小值的点：

$$
f(x, y) = x^2 + y^2
$$

我们可以使用梯度下降法来解决这个问题。具体的操作步骤如下：

1. 计算二元函数的梯度。
2. 更新迭代点。
3. 重复步骤1和步骤2，直到满足某个停止条件。

通过以下代码实现：

```python
import numpy as np

def gradient_descent(x0, y0, alpha=0.01, tol=1e-6, max_iter=1000):
    x, y = x0, y0
    for _ in range(max_iter):
        grad = np.array([2 * x, 2 * y])
        x_new = x - alpha * grad[0]
        y_new = y - alpha * grad[1]
        if abs(grad.sum()) < tol:
            break
        x, y = x_new, y_new
    return x, y

x0, y0 = 0, 0
x, y = gradient_descent(x0, y0)
print("解：x =", x, ", y =", y)
```
