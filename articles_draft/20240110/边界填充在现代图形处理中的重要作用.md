## 4. 具体代码实例和详细解释说明

### 4.1 具体代码实例

以下是一个使用 Python 编写的边界填充代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义图形对象的边界
def draw_rectangle(x, y, width, height):
    x0, y0 = x, y
    x1, y1 = x + width, y + height
    points = [(x0, y0), (x1, y0), (x1, y1), (x0, y1)]
    return points

# 判断当前点是否在图形对象的边界上
def is_point_in_rectangle(point, points):
    x, y = point
    x0, y0 = points[0]
    x1, y1 = points[1]
    x2, y2 = points[2]
    x3, y3 = points[3]
    return (x >= min(x0, x1, x2, x3) and x <= max(x0, x1, x2, x3) and
            y >= min(y0, y1, y2, y3) and y <= max(y0, y1, y2, y3))

# 填充颜色
def fill_color(point, color, points):
    x, y = point
    points.append((x, y))
    return points

# 绘制填充的矩形
def draw_filled_rectangle(x, y, width, height, color):
    points = draw_rectangle(x, y, width, height)
    filled_points = []
    for point in points:
        if is_point_in_rectangle(point, points):
            filled_points = fill_color(point, color, filled_points)
    plt.fill(*zip(*filled_points))
    plt.plot(*zip(*points), color='black')
    plt.show()

# 主程序
if __name__ == '__main__':
    draw_filled_rectangle(0, 0, 200, 200, 'blue')
```

上述代码实例主要包括以下几个部分：

1. 定义图形对象的边界：`draw_rectangle` 函数用于绘制一个矩形，并返回矩形的四个顶点。
2. 判断当前点是否在图形对象的边界上：`is_point_in_rectangle` 函数用于判断给定的点是否在矩形的边界内。
3. 填充颜色：`fill_color` 函数用于填充矩形的颜色，并返回填充后的矩形点列表。
4. 绘制填充的矩形：`draw_filled_rectangle` 函数用于绘制一个填充颜色的矩形，并显示在图像中。
