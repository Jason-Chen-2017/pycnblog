### 2.1 函数与泛函的定义与特点

函数是从一个集合到另一个集合的映射关系，通常用符号f(x)表示。函数的主要特点是：

1. 对于每个输入x，只有一个输出y，即f(x) = y。
2. 输入和输出之间存在明确的关系。

泛函是一种更一般的函数，它可以接受一个或多个函数作为输入，并返回一个数值。泛函的主要特点是：

1. 对于每个输入函数，只有一个输出值。
2. 输入和输出之间存在明确的关系。

### 4.2 梯度下降示例

梯度下降是一种常见的优化算法，可以用于最小化一个或多个目标函数。以下是一个简单的梯度下降示例：

```python
import numpy as np

# 定义目标函数
def f(x):
    return x**2 + 1

# 梯度下降算法
def gradient_descent(x0, alpha, iterations):
    x = x0
    for _ in range(iterations):
        grad = 2*x
        x -= alpha * grad
    return x

# 初始值
x0 = 10
# 学习率
alpha = 0.1
# 迭代次数
iterations = 100

# 运行梯度下降算法
x_min = gradient_descent(x0, alpha, iterations)

print("最小值：", x_min)
```

在这个示例中，我们首先定义了一个简单的目标函数 `f(x) = x**2 + 1`，然后使用梯度下降算法对其进行最小化。通过设置初始值、学习率和迭代次数，我们可以得到目标函数的最小值。
