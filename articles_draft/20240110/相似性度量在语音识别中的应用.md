欧氏距离的公式为：

$$

其中，$x$ 和 $y$ 是两个音频片段的特征向量，$n$ 是特征向量的维度，$x_i$ 和 $y_i$ 是特征向量的各个元素。

import numpy as np

def euclidean_distance(x, y):

print("欧氏距离:", distance)
