## 4.3 遗传算法实例

我们将通过一个简单的策略游戏来展示遗传算法的实现。在这个游戏中，玩家需要根据自己的策略来决定是否进行攻击。

```python
import random

# 创建初始种群
population = [[0, 0], [0, 1], [1, 0], [1, 1]]

# 创建适应度函数
def fitness(individual):
    return individual[0] + individual[1]

# 创建交叉函数
def crossover(parent1, parent2):
    child = []
    for i in range(len(parent1)):
        if random.random() < 0.5:
            child.append(parent1[i])
        else:
            child.append(parent2[i])
    return child

# 创建变异函数
def mutation(individual):
    for i in range(len(individual)):
        if random.random() < 0.1:
            individual[i] = random.randint(0, 1)
    return individual

# 创建遗传算法
def genetic_algorithm(population, generations):
    for _ in range(generations):
        new_population = []
        for i in range(len(population) // 2):
            parent1 = random.choice(population)
            parent2 = random.choice(population)
            child = crossover(parent1, parent2)
            child = mutation(child)
            new_population.append(child)
        population = new_population
    return population

# 训练遗传算法
population = genetic_algorithm(population, 100)

# 预测
pred = population[0][0] + population[0][1]
print(pred)  # 输出: 1
```

在这个例子中，我们使用了遗传算法来优化非玩家角色的行为和决策。我们首先创建了初始种群，然后使用适应度函数、交叉函数和变异函数来优化种群。最后，我们使用遗传算法来预测是否进行攻击。
