## 3.1 梯度下降

梯度下降是一种优化算法，它使用梯度信息来最小化一个函数。梯度下降的目标是找到一个最小值，使得函数的梯度为零。梯度下降的公式为：

$$
\theta_{t+1} = \theta_t - \alpha \nabla J(\theta_t)
$$

其中，$\theta$ 是参数，$t$ 是时间步，$\alpha$ 是学习率，$\nabla J(\theta_t)$ 是梯度。

## 3.2 随机梯度下降

随机梯度下降是一种梯度下降的变种，它使用随机梯度信息来最小化一个函数。随机梯度下降的目标是找到一个最小值，使得函数的梯度为零。随机梯度下降的公式为：

$$
\theta_{t+1} = \theta_t - \alpha \nabla J(\theta_t, i_t)
$$

其中，$\theta$ 是参数，$t$ 是时间步，$\alpha$ 是学习率，$\nabla J(\theta_t, i_t)$ 是随机梯度。

## 3.3 批量梯度下降

批量梯度下降是一种梯度下降的变种，它使用批量梯度信息来最小化一个函数。批量梯度下降的目标是找到一个最小值，使得函数的梯度为零。批量梯度下降的公式为：

$$
\theta_{t+1} = \theta_t - \alpha \nabla J(\theta_t, S_t)
$$

其中，$\theta$ 是参数，$t$ 是时间步，$\alpha$ 是学习率，$\nabla J(\theta_t, S_t)$ 是批量梯度。

## 4.1 梯度下降代码实例

```python
# 定义函数
def f(x):
    return x**2 + 2*x + 1

# 定义梯度
def grad_f(x):
    return 2*x + 2

# 定义学习率
alpha = 0.01

# 初始化参数
x = 0

# 开始迭代
for t in range(1000):
    # 计算梯度
    grad = grad_f(x)
    # 更新参数

```
