### 2.1 线性规划

线性规划是一种工业优化方法，它的目标函数和约束条件都是线性的。线性规划可以用来解决各种资源分配、生产计划等问题。

线性规划的基本形式是：

$$
\begin{aligned}
\text{最大化/最小化} & \quad z = c^T x \\
\text{subject to} & \quad Ax \leq b \\
& \quad x \geq 0
\end{aligned}
$$

其中，$x$是决策变量向量，$c$是目标函数系数向量，$A$是约束矩阵，$b$是约束向量。

### 3.1 线性规划的Python代码实例

```python
from scipy.optimize import linprog

# 目标函数系数向量
c = [1, 2, 3]

# 约束矩阵
A = [[1, 1, 1], [2, 2, 2]]

# 约束向量
b = [4, 6]

# 初始决策变量向量
x0 = [0, 0, 0]

# 使用linprog函数解决线性规划问题
res = linprog(c, A_ub=A, b_ub=b, x0=x0)

# 输出解
print(res)
```

### 3.2 非线性规划的Python代码实例

```python
from scipy.optimize import minimize

# 目标函数
def f(x):
    return x[0]**2 + x[1]**2

# 约束函数
def g(x):
    return x[0] + x[1] - 5

# 初始决策变量向量
x0 = [0, 0]

# 使用minimize函数解决非线性规划问题
res = minimize(f, x0, constraints={'type': 'eq', 'fun': g})

# 输出解
print(res)
```
