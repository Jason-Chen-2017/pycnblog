- 使用Python的NumPy库来实现梯度下降：
```python
import numpy as np

# 定义损失函数
def loss(w, x, y):
    return np.mean((y - (w @ x).reshape(-1, 1)) ** 2)

# 定义梯度
def grad(w, x, y):
    return (x.T @ (x @ w - y)).reshape(-1, 1)

# 初始化权重
w = np.random.randn(x.shape[1], 1)

# 设置学习率
learning_rate = 0.01

# 训练模型
for epoch in range(1000):
    w = w - learning_rate * grad(w, x, y)

# 输出权重
print('Weight:', w)
```
