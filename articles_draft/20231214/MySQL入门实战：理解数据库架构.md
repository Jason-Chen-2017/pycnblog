# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释MySQL的使用方法。

## 4.1 创建数据库

首先，我们需要创建一个数据库。我们可以使用以下SQL语句来创建一个名为“mydb”的数据库：

```sql
CREATE DATABASE mydb;
```

## 4.2 使用数据库

接下来，我们需要使用我们创建的数据库。我们可以使用以下SQL语句来使用“mydb”数据库：

```sql
USE mydb;
```

## 4.3 创建表

接下来，我们需要创建一个表。我们可以使用以下SQL语句来创建一个名为“user”的表：

```sql
CREATE TABLE user (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    age INT NOT NULL
);
```

在上面的SQL语句中，我们创建了一个名为“user”的表，它有三个列：id、name和age。id列是主键，它的值会自动增长。name列是一个字符串类型的列，它的长度是255。age列是一个整数类型的列，它不能为空。

## 4.4 插入数据

接下来，我们需要插入一些数据。我们可以使用以下SQL语句来插入一条数据：

```sql
INSERT INTO user (name, age) VALUES ('John', 25);
```

在上面的SQL语句中，我们插入了一条数据，它的名字是“John”，年龄是25。

## 4.5 查询数据

接下来，我们需要查询数据。我们可以使用以下SQL语句来查询所有的用户信息：

```sql
SELECT * FROM user;
```

在上面的SQL语句中，我们查询了所有的用户信息。
