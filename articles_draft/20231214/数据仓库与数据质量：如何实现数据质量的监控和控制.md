## 4.1数据清洗示例

假设我们有一个数据仓库中的表，表中有一列名为“年龄”的数据，但是这些数据有一些重复和缺失的情况。我们需要对这些数据进行清洗，以提高数据质量。

### 4.1.1数据去重

我们可以使用哈希表来实现数据去重的功能。首先，我们需要创建一个哈希表，然后遍历数据仓库中的“年龄”列，将每个不同的年龄值添加到哈希表中。最后，我们可以从哈希表中获取唯一的年龄值，并更新数据仓库中的“年龄”列。

```python
# 创建哈希表
age_set = set()

# 遍历数据仓库中的“年龄”列
for row in data_warehouse_age_column:
    age_set.add(row['age'])

# 更新数据仓库中的“年龄”列
for age in age_set:
    data_warehouse_age_column = [row for row in data_warehouse_age_column if row['age'] == age]
```

### 4.1.2数据填充

我们可以使用插值法来实现数据填充的功能。首先，我们需要计算每个缺失年龄值的平均值，然后将这个平均值填充到缺失的年龄值中。

```python
# 计算每个缺失年龄值的平均值
average_age = sum(row['age'] for row in data_warehouse_age_column if row['age'] is None) / len(data_warehouse_age_column)

# 填充缺失的年龄值
for row in data_warehouse_age_column:
    if row['age'] is None:
        row['age'] = average_age
```

### 4.1.3数据转换

我们可以使用类型转换来实现数据转换的功能。首先，我们需要检查数据仓库中的“年龄”列的类型，然后将其转换为整型。

```python
# 检查数据仓库中的“年龄”列的类型
if data_warehouse_age_column[0]['age'].__class__ == str:
    # 将“年龄”列的值转换为整型
    data_warehouse_age_column = [{'age': int(row['age'])} for row in data_warehouse_age_column]
```

### 4.1.4数据过滤

我们可以使用条件判断来实现数据过滤的功能。首先，我们需要设置一个年龄范围，然后遍历数据仓库中的“年龄”列，将不符合年龄范围的数据删除。

```python
# 设置年龄范围
min_age = 18
max_age = 60

# 遍历数据仓库中的“年龄”列
data_warehouse_age_column = [row for row in data_warehouse_age_column if min_age <= row['age'] <= max_age]
```
