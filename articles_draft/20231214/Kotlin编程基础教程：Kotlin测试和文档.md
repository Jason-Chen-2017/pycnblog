# 4.4 委托示例
```kotlin
class Calculator {
    operator fun plus(other: Calculator): Calculator {
        val sum = this.value + other.value
        return Calculator(sum)
    }
}

fun main(args: Array<String>) {
    val calculator1 = Calculator(10)
    val calculator2 = Calculator(20)

    val calculator3 = calculator1 + calculator2

    println(calculator3.value)
}
```
在这个示例中，我们定义了一个名为`Calculator`的类，它有一个`value`属性。然后，我们使用`operator`关键字来定义一个`plus`方法，它可以让我们在不修改原始类的基础上添加新功能。最后，我们使用`println`函数来输出这个新的`Calculator`对象的`value`属性值。

# 4.5 协程示例
```kotlin
import kotlinx.coroutines.*

fun main(args: Array<String>) {
    GlobalScope.launch {
        delay(1000)
        println("Hello, World!")
    }
}
```
在这个示例中，我们导入了`kotlinx.coroutines`包，然后使用`GlobalScope.launch`函数来创建一个协程。这个协程会在1秒钟后输出“Hello, World!”。
