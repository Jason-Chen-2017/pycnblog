Python是一种强大的编程语言，它具有简洁的语法和易于学习。在Python中，模块是代码的组织和重用的基本单元。模块可以包含函数、类、变量等，使得代码更加模块化、可维护。本文将详细介绍Python模块的导入和定义方法，并提供具体的代码实例和解释。

## 1.1 Python模块的概念与特点

Python模块是一个包含一组函数、类或变量的Python文件。模块可以让我们将相关的代码组织在一个文件中，以便于复用和维护。模块的特点如下：

- 模块可以包含函数、类、变量等，使得代码更加模块化、可维护。
- 模块可以被其他程序导入和使用，实现代码的重用。
- 模块可以通过命名空间将全局变量和函数隐藏起来，避免命名冲突。

## 1.2 Python模块的导入与定义

### 1.2.1 导入模块

在Python中，我们可以使用`import`关键字导入模块。例如，要导入`math`模块，可以使用以下代码：

```python
import math
```

导入模块后，我们可以直接使用模块中的函数和变量。例如，要使用`math`模块中的`sqrt`函数，可以这样做：

```python
import math
print(math.sqrt(4))  # 输出：2.0
```

如果我们只需要使用某个模块的某个函数或变量，可以使用`from ... import ...`语句进行导入。例如，要导入`math`模块中的`sqrt`函数，可以这样做：

```python
from math import sqrt
print(sqrt(4))  # 输出：2.0
```

### 1.2.2 定义模块

要定义一个Python模块，只需要将代码保存到一个`.py`文件中即可。例如，我们可以创建一个名为`mymodule.py`的文件，并在其中定义一个函数：

```python
# mymodule.py
def hello():
    print("Hello, World!")
```

然后，我们可以在其他Python文件中导入并使用这个模块：

```python
# main.py
import mymodule
mymodule.hello()  # 输出：Hello, World!
```

### 1.2.3 导入模块的注意事项

- 在导入模块时，如果模块名包含点（.），则需要使用双引号（""）或单引号（'）将模块名括起来。例如，`import "module.name"`。
- 在导入模块时，如果模块名包含中文或其他特殊字符，则需要使用中文文字或其他特殊字符括起来。例如，`import 模块名`。
- 在导入模块时，如果模块名包含空格，则需要使用双引号（""）或单引号（'）将模块名括起来。例如，`import "module name"`。

### 1.3.1 模块的组成

导入模块和定义模块之间的联系是：导入模块是实现模块代码的重用和模块化的关键，定义模块是实现模块代码的组织和维护的关键。
