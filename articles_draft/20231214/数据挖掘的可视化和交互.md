## 4.2 交互

### 4.2.1 交互决策树

```python
import numpy as np
from sklearn.tree import DecisionTreeClassifier

# 数据
X = np.array([[1, 2], [3, 4], [5, 6]])
y = np.array([0, 1, 1])

# 构建决策树
clf = DecisionTreeClassifier()
clf.fit(X, y)

# 交互
while True:
    # 输入特征
    feature = input("请输入特征: ")
    # 调整决策树参数
    clf.fit(np.array([[feature]]), y)
    # 输出结果
    print(clf.predict([[feature]]))
```

### 4.2.2 交互支持向量机

```python
import numpy as np
from sklearn.svm import SVC

    # 调整支持向量机参数

    # 输出结果
    print(clf.predict([[feature]]))
```
