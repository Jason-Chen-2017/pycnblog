#### 3.1.1概率的基本定义

概率的基本定义是：对于一个随机事件，它的概率是发生该事件的方法数除以总方法数的乘积。 mathematically, the probability of an event is the number of favorable outcomes divided by the total number of outcomes.

#### 3.1.2概率的计算

概率的计算可以通过以下方法进行：

1. 直接计算：直接计算所有可能的结果，并计算满足条件的结果的数量。
2. 采样方法：从总体中随机抽取样本，并计算满足条件的结果的数量。
3. 定理方法：利用概率定理进行计算。

def compute_average(data):

# 计算中位数
def compute_median(data):
    sorted_data = np.sort(data)
    n = len(data)
    if n % 2 == 0:
        return (sorted_data[n//2 - 1] + sorted_data[n//2]) / 2
    else:
        return sorted_data[n//2]

# 计算方差
def compute_variance(data):
    return np.var(data)

# 计算标准差

    return np.std(data)
