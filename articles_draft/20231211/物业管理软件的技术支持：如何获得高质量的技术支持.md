### 4.1 培训实例

假设我们需要实现一个在线培训系统，以帮助物业管理人员学习软件功能。我们可以使用以下代码实现：

```python
import requests
from bs4 import BeautifulSoup

def get_training_materials():
    url = 'https://www.example.com/training-materials'
    response = requests.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')
    materials = soup.find_all('div', class_='training-material')
    return materials

def create_training_plan(materials):
    plan = []
    for material in materials:
        title = material.find('h2').text
        content = material.find('p').text
        plan.append({'title': title, 'content': content})
    return plan

def start_training(plan):
    for item in plan:
        print(item['title'])
        print(item['content'])

if __name__ == '__main__':
    materials = get_training_materials()
    plan = create_training_plan(materials)
    start_training(plan)
```

这段代码首先从一个在线教育平台获取培训材料，然后将这些材料组织成一个培训计划，最后根据计划进行培训。
