### 4.1 C++模板代码实例

```cpp

}

    int b = 20;
    int c = add(a, b);
    std::cout << "a + b = " << c << std::endl;

```

在这个代码实例中，我们定义了一个模板函数add，接受两个类型参数a和b，并返回它们的和。然后，我们使用模板函数add，将a和b指定为整型参数，并输出结果。

### 4.2 C++异常处理代码实例

```cpp
#include <iostream>

int divide(int a, int b) {
    if (b == 0) {
        throw std::runtime_error("Divide by zero");
    }
    return a / b;
}

int main() {
    int a = 10;
    int b = 0;
    try {
        int c = divide(a, b);
        std::cout << "a / b = " << c << std::endl;
    } catch (const std::runtime_error& e) {
        std::cerr << "Error: " << e.what() << std::endl;
    }
    return 0;
}
```

在这个代码实例中，我们定义了一个函数divide，接受两个整型参数a和b，并返回a除以b的结果。如果b为0，我们会抛出一个异常，并在catch块中捕获并处理异常。
