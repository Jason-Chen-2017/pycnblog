假设我们需要实现一个简单的集合类，用于存储一组整数。我们可以使用Python的内置数据结构list来实现。

```python
class SimpleCollection:
    def __init__(self):
        self.data = []

    def add(self, value):
        self.data.append(value)

    def remove(self, value):
        self.data.remove(value)

    def find(self, value):
        return value in self.data
```

在这个例子中，我们的集合类包含了四个方法：add、remove、find。这些方法的时间复杂度分别为O(1)、O(n)和O(n)。
