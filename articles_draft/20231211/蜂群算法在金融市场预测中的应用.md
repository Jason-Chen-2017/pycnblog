```python
import numpy as np

# 初始化蜂群
n_particles = 50
n_dimensions = 2
positions = np.random.rand(n_particles, n_dimensions)
velocities = np.random.rand(n_particles, n_dimensions)

# 设定参数
w = 0.7
c1 = 1.5
c2 = 1.5

# 定义目标函数
def fitness(x):
    return np.sum(x**2)

# 主循环
for t in range(100):
    # 更新速度
    for i in range(n_particles):
        r1 = np.random.rand()
        r2 = np.random.rand()
        velocities[i] = w * velocities[i] + c1 * r1 * (positions[i] - positions[i]) + c2 * r2 * (gbest - positions[i])

    # 更新位置
    for i in range(n_particles):
        positions[i] = positions[i] + velocities[i]

    # 更新最佳位置
    pbest = np.min(fitness(positions), axis=0)

    # 更新全局最佳位置
    if np.min(fitness(positions)) < np.min(fitness(gbest)):
        gbest = positions

# 输出结果
print(gbest)
```

这个代码实例中，我们首先初始化了蜂群的大小、初始位置、初始速度、最大速度、惯性因子等参数。然后我们定义了一个目标函数，即我们要优化的函数。接下来，我们进入了主循环，每次迭代中，我们更新了速度、位置、最佳位置和全局最佳位置。最后，我们输出了全局最佳位置，即最优解。
