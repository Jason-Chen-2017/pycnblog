1. 收集用户的历史购买行为数据，包括用户购买的产品、购买时间等信息。
2. 对用户购买行为数据进行预处理，如数据清洗、缺失值处理等。
3. 计算用户与产品之间的相似性，可以使用各种算法，如欧氏距离、余弦相似度等。

5. 对推荐结果进行排序，将最相似的产品排在前面。
6. 对推荐结果进行展示，让用户可以直接从推荐结果中选择所需的产品。

推荐系统的数学模型可以简化为以下公式：

$$
similarity(u, p) = f(user\_features(u), product\_features(p))

from scipy.spatial.distance import cosine

    user_ratings_normalized = np.linalg.norm(user_ratings, axis=1)
    product_ratings_normalized = np.linalg.norm(product_ratings, axis=1)
    similarity = cosine(user_ratings, product_ratings)

def recommend_products(user_ratings, product_ratings, user_id, top_n):
    user_ratings_normalized = np.linalg.norm(user_ratings, axis=1)
    product_ratings_normalized = np.linalg.norm(product_ratings, axis=1)
    similarity = cosine(user_ratings, product_ratings)
    similarity = np.dot(user_ratings_normalized, product_ratings_normalized.T)
