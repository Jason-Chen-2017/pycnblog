# 4.具体代码实例和详细解释说明

## 4.1量子调控的具体代码实例

### 4.1.1量子态的初始化

```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile

# 创建一个量子电路
qc = QuantumCircuit(1)

# 初始化量子位
qc.h(0)

# 将量子电路转换为可传输的形式
qc_transpiled = transpile(qc, Aer.get_backend('qasm_simulator'))

# 执行量子电路
result = qc_transpiled.run()

# 打印结果
print(result.get_counts())
```
