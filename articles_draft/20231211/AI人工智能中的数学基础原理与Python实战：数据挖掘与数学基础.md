6.梯度下降的Python实现：

```python
import numpy as np

# 创建数据
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([1, 2, 3, 4])

# 创建模型
def gradient_descent(X, y, alpha=0.01, iterations=1000):
    m, n = X.shape
    theta = np.zeros(n)
    for _ in range(iterations):
        h = np.dot(X, theta)
        gradient = np.dot(X.T, (h - y)) / m
        theta = theta - alpha * gradient
    return theta

# 训练模型
theta = gradient_descent(X, y)

# 预测
pred = np.dot(X, theta)
```
