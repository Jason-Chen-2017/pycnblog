## 4.1 数据加密
我们可以使用Python的cryptography库来实现数据加密和解密操作。以下是一个简单的加密和解密示例：

```python
from cryptography.fernet import Fernet

# 生成密钥
key = Fernet.generate_key()

# 加密数据
cipher_suite = Fernet(key)
encrypted_data = cipher_suite.encrypt(b"Hello, World!")

# 解密数据
decrypted_data = cipher_suite.decrypt(encrypted_data)

print(decrypted_data.decode())  # 输出：Hello, World!
```

在这个示例中，我们首先生成了一个密钥，然后使用这个密钥对数据进行加密和解密。最后，我们将解密后的数据打印出来。

## 4.2 数据脱敏
我们可以使用Python的random库来实现数据脱敏操作。以下是一个简单的脱敏示例：

```python
import random

# 原始数据
data = [
    {"name": "John Doe", "age": 30, "address": "123 Main St"},
    {"name": "Jane Smith", "age": 25, "address": "456 Elm St"},
    {"name": "Alice Johnson", "age": 28, "address": "789 Oak St"}
]

# 脱敏函数
def anonymize_data(data):
    anonymized_data = []
    for item in data:
        anonymized_item = {}
        anonymized_item["name"] = "***"
        anonymized_item["age"] = random.randint(18, 65)
        anonymized_item["address"] = "***"
        anonymized_data.append(anonymized_item)
    return anonymized_data

# 脱敏后的数据
anonymized_data = anonymize_data(data)
print(anonymized_data)
```

在这个示例中，我们首先定义了一组原始数据，然后使用脱敏函数对数据进行处理。脱敏函数会将名字、年龄和地址替换为星号（***），然后返回脱敏后的数据。最后，我们将脱敏后的数据打印出来。

## 4.3 数据清洗
我们可以使用Python的pandas库来实现数据清洗操作。以下是一个简单的清洗示例：

```python
import pandas as pd

# 原始数据
data = {
    "name": ["John Doe", "Jane Smith", "Alice Johnson"],
    "age": [30, 25, 28],
    "address": ["123 Main St", "456 Elm St", "789 Oak St"]
}

# 创建DataFrame
df = pd.DataFrame(data)

# 去除重复数据
df = df.drop_duplicates()

# 填充缺失值
df["age"].fillna(df["age"].mean(), inplace=True)

# 修正错误数据
df["address"] = df["address"].str.replace("St", "Street")

# 清洗后的数据
cleaned_data = df.to_dict("records")
print(cleaned_data)
```

在这个示例中，我们首先定义了一组原始数据，然后使用pandas库创建了一个DataFrame。接下来，我们使用了几个清洗函数来去除重复数据、填充缺失值和修正错误数据。最后，我们将清洗后的数据打印出来。

## 4.4 数据标准化
我们可以使用Python的sklearn库来实现数据标准化操作。以下是一个简单的标准化示例：

```python
from sklearn.preprocessing import StandardScaler

# 原始数据
data = {
    "age": [30, 25, 28],
    "height": [180, 165, 175],
    "weight": [75, 60, 65]
}

# 创建DataFrame
df = pd.DataFrame(data)

# 标准化数据
scaler = StandardScaler()
df_scaled = scaler.fit_transform(df)

# 标准化后的数据
standardized_data = pd.DataFrame(df_scaled, columns=df.columns)
print(standardized_data)
```

在这个示例中，我们首先定义了一组原始数据，然后使用sklearn库创建了一个StandardScaler对象。接下来，我们使用这个对象对数据进行标准化处理。最后，我们将标准化后的数据打印出来。
