## 3. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释迁移学习在图像识别中的应用。

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torchvision import datasets, transforms

# 加载源域数据集
source_data = datasets.ImageFolder(root='/path/to/source/data', transform=transforms.ToTensor())
source_loader = torch.utils.data.DataLoader(source_data, batch_size=128, shuffle=True)

# 加载目标域数据集
target_data = datasets.ImageFolder(root='/path/to/target/data', transform=transforms.ToTensor())
target_loader = torch.utils.data.DataLoader(target_data, batch_size=128, shuffle=True)

# 加载源域模型
source_model = torch.hub.load('torchvision.models', 'resnet18', pretrained=True)
source_model.fc = nn.Linear(source_model.fc.in_features, target_data.classes)

# 加载目标域模型
target_model = torch.hub.load('torchvision.models', 'resnet18', pretrained=False)

# 训练源域模型
for epoch in range(10):
    for data, label in source_loader:
        optimizer.zero_grad()
        output = source_model(data)
        loss = criterion(output, label)
        loss.backward()
        optimizer.step()

# 迁移源域模型到目标域
target_model.load_state_dict(source_model.state_dict())

# 训练目标域模型
for epoch in range(10):
    for data, label in target_loader:
        optimizer.zero_grad()
        output = target_model(data)
        loss = criterion(output, label)
        loss.backward()
        optimizer.step()
```

在上述代码中，我们首先加载了源域和目标域的数据集，并将其分别加载到源域数据加载器和目标域数据加载器中。然后，我们加载了源域模型和目标域模型，并将源域模型的最后一层替换为一个与目标域数据集类别数相同的全连接层。接着，我们训练了源域模型，并将其参数迁移到目标域模型中。最后，我们训练了目标域模型，以实现更好的图像识别效果。
