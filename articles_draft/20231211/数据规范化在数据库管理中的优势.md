数据规范化的核心算法原理包括：

1.第一范式（1NF）：第一范式是数据库设计中的一种规范化级别，它要求每个表中的每一列都是独立的，不能包含重复的数据。

2.第二范式（2NF）：第二范式是数据库设计中的一种规范化级别，它要求每个表中的每一列都是依赖于主键的，不能包含依赖于非主键的数据。

3.第三范式（3NF）：第三范式是数据库设计中的一种规范化级别，它要求每个表中的每一列都是依赖于主键的，不能包含依赖于其他表的数据。

        try {
            Connection conn = DriverManager.getConnection('jdbc:sqlite:students.db');
            Statement stmt = conn.createStatement();
            ResultSet rs = stmt.executeQuery('SELECT * FROM students WHERE student_id = ?', (1));

            while (rs.next()) {
                int student_id = rs.getInt('student_id');
                String name = rs.getString('name');

                System.out.println("Name: " + name);

            rs.close();
            stmt.close();
            conn.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
