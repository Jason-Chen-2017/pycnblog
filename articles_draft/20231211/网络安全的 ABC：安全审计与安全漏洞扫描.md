# 2. 核心概念与联系

在讨论安全审计与安全漏洞扫描之前，我们需要了解一些核心概念。

## 2.1 安全审计

安全审计是一种用于评估网络安全状况的方法，它包括以下几个方面：

1. 对网络设备、系统和应用程序进行审计，以确保它们符合安全标准。
2. 对网络流量进行监控，以确保其符合预期的行为。
3. 对网络设备和系统进行漏洞扫描，以确保它们没有被攻击者利用的漏洞。

安全审计的目的是为了确保网络系统的安全性，以及为了防止潜在的安全事件。

## 2.2 安全漏洞扫描

安全漏洞扫描是一种用于发现网络系统中潜在安全漏洞的方法，它包括以下几个方面：

1. 对网络设备和系统进行漏洞扫描，以确保它们没有被攻击者利用的漏洞。
2. 对应用程序进行代码审计，以确保其没有潜在的安全漏洞。
3. 对网络流量进行监控，以确保其符合预期的行为。

安全漏洞扫描的目的是为了发现网络系统中的潜在安全漏洞，以便可以在攻击者发现这些漏洞之前进行修复。

## 2.3 联系

安全审计和安全漏洞扫描是两种不同的方法，但它们之间有密切的联系。安全审计是一种用于评估网络安全状况的方法，而安全漏洞扫描是一种用于发现网络系统中潜在安全漏洞的方法。这两种方法可以相互补充，以确保网络系统的安全性。
