## 4.1 配置管理的代码实例

以下是一个简单的配置管理的代码实例：

```python
import json

class ConfigManager:
    def __init__(self):
        self.config_info = {}

    def add_config(self, key, value):
        self.config_info[key] = value

    def get_config(self, key):
        return self.config_info.get(key)

    def update_config(self, key, value):
        self.config_info[key] = value

    def delete_config(self, key):
        if key in self.config_info:
            del self.config_info[key]

    def save_config(self, file_path):
        with open(file_path, 'w') as f:
            json.dump(self.config_info, f)

    def load_config(self, file_path):
        with open(file_path, 'r') as f:
            self.config_info = json.load(f)
```

在上述代码中，我们定义了一个 `ConfigManager` 类，它负责管理配置信息。`ConfigManager` 类提供了以下几个方法：

- `add_config`：添加配置信息
- `get_config`：获取配置信息
- `update_config`：更新配置信息
- `delete_config`：删除配置信息
- `save_config`：保存配置信息到文件
- `load_config`：加载配置信息从文件
