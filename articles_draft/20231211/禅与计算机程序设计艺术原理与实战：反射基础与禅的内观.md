假设我们有一个简单的类：

```java
public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}
```

我们可以使用反射来获取这个类的元数据，并通过反射来操作这个类的方法。以下是一个具体的代码实例：

```java
import java.lang.reflect.Field;
import java.lang.reflect.Method;

public class ReflectionExample {
    public static void main(String[] args) throws Exception {
        // 获取 Person 类的 Class 对象
        Class<Person> personClass = Person.class;

        // 获取 Person 类的所有属性
        Field[] fields = personClass.getDeclaredFields();
        for (Field field : fields) {
            System.out.println(field.getName());
        }

        // 获取 Person 类的所有方法
        Method[] methods = personClass.getDeclaredMethods();
        for (Method method : methods) {
            System.out.println(method.getName());
        }

        // 创建 Person 对象
        Person person = new Person("John", 20);

        // 通过反射获取 Person 对象的 name 属性
        Field nameField = personClass.getDeclaredField("name");
        String name = (String) nameField.get(person);
        System.out.println("Name: " + name);

        // 通过反射设置 Person 对象的 age 属性
        Field ageField = personClass.getDeclaredField("age");
        int age = (int) ageField.get(person);
        System.out.println("Age: " + age);

        // 通过反射调用 Person 对象的 setName 方法
        Method setNameMethod = personClass.getDeclaredMethod("setName", String.class);
        setNameMethod.invoke(person, "Jack");
        System.out.println("Name: " + name);

        // 通过反射调用 Person 对象的 setAge 方法
        Method setAgeMethod = personClass.getDeclaredMethod("setAge", int.class);
        setAgeMethod.invoke(person, 21);
        System.out.println("Age: " + age);
    }
}
```

在这个代码实例中，我们首先获取了 Person 类的 Class 对象。然后，我们使用反射来获取 Person 类的所有属性和方法。接着，我们创建了一个 Person 对象，并使用反射来获取和设置这个对象的属性和方法。
