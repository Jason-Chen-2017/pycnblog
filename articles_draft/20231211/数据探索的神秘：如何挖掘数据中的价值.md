数据清洗代码：

```python
import pandas as pd
import numpy as np

# 读取数据
data = pd.read_csv('data.csv')

# 处理缺失值
data = data.fillna(data.mean())

# 处理异常值
data = data[np.abs(data - data.mean()) < 3 * data.std()]

# 处理噪声
data = data.rolling(window=5).mean()
```
