### 4.1.1 计算移动平均线

```python
def calculate_moving_average(prices, window_size):
    return [sum(prices[:window_size]) / window_size] + \
           [(sum(prices[i:window_size+i]) / window_size) for i in range(1, len(prices) - window_size + 1)]
```

这个函数接受两个参数：prices（股票价格列表）和window_size（窗口大小）。它返回一个列表，该列表包含了计算后的移动平均线值。

### 4.1.2 计算Relative Strength Index（RSI）

```python
def calculate_rsi(prices, window_size):
    gains = []
    losses = []
    for i in range(window_size, len(prices)):
        gain = (prices[i] - prices[i-window_size]) / prices[i-window_size]
        if gain > 0:
            gains.append(gain)
        else:
            losses.append(-gain)
    return 100 - 100 / (1 + sum(gains) / sum(losses))
```

这个函数接受两个参数：prices（股票价格列表）和window_size（窗口大小）。它返回一个数值，该数值表示股票的Relative Strength Index（RSI）。

### 4.1.3 计算Bollinger Band

```python
def calculate_bollinger_band(prices, window_size, multiplier):
    mean = sum(prices) / len(prices)
    std = (sum([(price - mean)**2 for price in prices]) / len(prices))**0.5
    return [mean - multiplier * std, mean + multiplier * std]
```

这个函数接受三个参数：prices（股票价格列表），window_size（窗口大小）和multiplier（倍数）。它返回一个列表，该列表包含了计算后的Bollinger Band值。
