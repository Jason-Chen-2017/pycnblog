## 4.2 创建图数据

我们创建一个简单的图数据，包含两个节点和一条边：

```python
import networkx as nx

# 创建一个空图
G = nx.Graph()

# 添加节点
G.add_node(1)
G.add_node(2)

# 添加边
G.add_edge(1, 2)
```

## 4.3 分布式存储图数据

我们将图数据分为两个部分，每个部分存储在不同的计算节点上：

```python
# 将图数据分为两个部分
G1 = G.subgraph(nodes=[1])
G2 = G.subgraph(nodes=[2])

# 将G1存储在计算节点1上
calculation_node1 = CalculationNode(G1)

# 将G2存储在计算节点2上
calculation_node2 = CalculationNode(G2)
```

## 4.4 分布式查询图数据

我们将查询任务分解为多个子任务，并在多个计算节点上并行执行：

```python
# 定义查询任务
def query_task(G):
    # 查询G中所有与1节点相连的节点
    return list(G.neighbors(1))

# 将查询任务分发给计算节点
query_result1 = calculation_node1.execute(query_task)
query_result2 = calculation_node2.execute(query_task)
```
