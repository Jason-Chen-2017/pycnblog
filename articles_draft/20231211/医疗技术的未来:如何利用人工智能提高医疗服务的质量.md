### 4.1 逻辑回归

逻辑回归是一种监督学习方法，它可以用来预测二元类别变量。逻辑回归可以帮助医生更好地诊断疾病，提高治疗效果，降低医疗成本。

#### 4.1.1 代码实例

```python
from sklearn.linear_model import LogisticRegression
from sklearn.datasets import load_breast_cancer

# 加载数据集
data = load_breast_cancer()
X = data.data
y = data.target

# 创建逻辑回归模型
model = LogisticRegression()

# 训练模型
model.fit(X, y)

# 预测
preds = model.predict(X)
```

#### 4.1.2 解释说明

逻辑回归是一种监督学习方法，它可以用来预测二元类别变量。逻辑回归的工作原理是，通过最小化损失函数来找到最佳的权重向量。损失函数是指的模型预测的结果与实际结果之间的差异。逻辑回归的损失函数是对数损失函数。

### 4.3 聚类

聚类是一种无监督学习方法，它可以用来分组数据。聚类可以帮助医生更好地诊断疾病，提高治疗效果，降低医疗成本。

#### 4.3.1 代码实例

```python
from sklearn.cluster import KMeans
from sklearn.datasets import load_breast_cancer

# 加载数据集
data = load_breast_cancer()
X = data.data

# 创建聚类模型
model = KMeans(n_clusters=3)

# 训练模型
model.fit(X)

# 预测
labels = model.labels_
```

#### 4.3.2 解释说明

聚类是一种无监督学习方法，它可以用来分组数据。聚类的工作原理是，通过最小化内部距离来找到最佳的聚类中心。内部距离是指每个数据点到其最近聚类中心的距离之和。聚类的目标是将相似的数据点分组，将不同的数据点分组。
