## 4.1 信号处理算法的代码实例

以下是一个简单的信号处理算法的代码实例，用于滤波处理生物传感器获取的信号：

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机信号
t = np.linspace(0, 1, 100)
s = np.sin(2 * np.pi * 10 * t) + np.cos(2 * np.pi * 20 * t)

# 滤波处理
f = np.fft.fft(s)
f_filt = f * (1 / (1 + (f / 10) ** 2))
s_filt = np.fft.ifft(f_filt)

# 绘制原始信号和滤波后信号
plt.plot(t, s, label='原始信号')
plt.plot(t, s_filt, label='滤波后信号')
plt.legend()
plt.show()
```

在上述代码中，我们首先生成了一个随机信号，然后使用傅里叶变换对信号进行滤波处理，最后绘制了原始信号和滤波后信号。
