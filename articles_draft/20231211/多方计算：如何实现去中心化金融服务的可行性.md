## 4.具体代码实例和详细解释说明

### 4.1 多方计算代码实例

以下是一个多方计算代码实例：

```python
import random
from cryptography.fernet import Fernet

# 生成密钥
key = Fernet.generate_key()

# 加密数据
cipher_suite = Fernet(key)
data = "Hello, World!"
encrypted_data = cipher_suite.encrypt(data.encode())

# 存储加密数据
stored_data = encrypted_data

# 多方协作计算
def compute_sum(data):
    return sum(data)

# 解密计算结果
def decrypt_data(encrypted_data):
    cipher_suite = Fernet(key)
    decrypted_data = cipher_suite.decrypt(encrypted_data)
    return decrypted_data.decode()

# 主程序
if __name__ == "__main__":
    # 生成多个节点的输入数据
    node_data = [random.randint(1, 100) for _ in range(10)]

    # 存储加密数据
    for data in node_data:
        encrypted_data = cipher_suite.encrypt(data.to_bytes(4, byteorder='big'))
        stored_data.append(encrypted_data)

    # 多方协作计算
    sum_data = compute_sum(node_data)

    # 解密计算结果
    decrypted_sum_data = decrypt_data(sum_data)

    # 输出计算结果
    print(decrypted_sum_data)
```

### 4.2 代码解释说明

上述代码实例的解释说明如下：

1. 生成密钥：使用 `cryptography.fernet` 库生成加密密钥。
2. 加密数据：使用生成的密钥对输入数据进行加密。
3. 存储加密数据：将加密后的数据存储在多个节点上。
4. 多方协作计算：实现多方协作计算的目标，即计算所有节点的输入数据之和。
5. 解密计算结果：将计算结果进行解密，以获得最终的计算结果。
6. 主程序：实现上述步骤的主程序。
