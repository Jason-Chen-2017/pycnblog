### 4.1 定时器
以下是一个使用Python的定时器示例：

```python
import time

def timer(seconds):
    print("Starting timer...")
    time.sleep(seconds)
    print("Timer finished!")

timer(5)
```

在这个示例中，我们使用Python的`time`模块来创建一个定时器。我们定义了一个`timer`函数，它接受一个参数`seconds`，表示定时器的持续时间。然后，我们使用`time.sleep`函数来暂停程序的执行，直到定时器结束。

### 4.3 通知推送
以下是一个使用Python的通知推送示例：

```python
import requests

def send_notification(title, message):
    url = "https://fcm.googleapis.com/fcm/send"
    headers = {
        "Authorization": "key=YOUR_API_KEY",
        "Content-Type": "application/json"
    }
    data = {
        "to": "/topics/all",
        "notification": {
            "title": title,
            "body": message
        }
    }
    response = requests.post(url, headers=headers, json=data)
    return response.json()

print(response)
```

在这个示例中，我们使用Python的`requests`库来创建一个通知推送。我们定义了一个`send_notification`函数，它接受两个参数`title`和`message`，表示通知的标题和内容。然后，我们使用Google的Firebase Cloud Messaging（FCM）API来发送通知。
