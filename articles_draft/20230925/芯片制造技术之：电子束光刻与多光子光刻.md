# 1.简介
  

电子束光刻（Electron Beam Lithography, EBLL）、多光子光刻（Multi-Wavelength Lithography, MWL）等都是基于固态照明（Solid-State Lighting, SSL）的晶圆印制工艺，也是在信息产业及航空航天领域非常重要的一种工艺。本文从电子束光刻和多光子光刻的原理出发，结合其具体操作，阐述了其原理及方法，并将介绍相关关键技术的应用，以及未来的发展方向。

# 2.前置知识
## 2.1 光刻与光栅
光刻（Lithography）是指用不同的粒子对一个或多个材料进行照射，使其具有特定形状的过程。而光栅（Raster）则是指在固定网格上采用一定间距的微小的点阵，将物理对象刻画到平面图象上的过程。由于光栅技术在制作电路板、显示屏幕等高分辨率的成像上具有重要作用，因此了解它们的一些特性和原理是很有必要的。

### 2.1.1 电子束光刻与晶圆结构
电子束光刻（EBLL）是指利用电子束对晶体化学品形成的方法。电子束是由个别自由电子组成，具有极短的传播距离，可以快速、准确地击穿晶体表面。它既可以在静止状态下发生，也可以通过激光等其它方式实现光刻制作。电子束光刻依赖于电子束的快速传播及精确的晶体位置调配，因此保证了晶体形成的精确程度。通常情况下，电子束光刻所用到的材料是具有晶圆结构的半导体材料，如GaAs、InP、SiC等。电子束光刻的主要步骤如下：

1. 设计晶体结构
电子束能够击穿晶体表面，因此需要有一个具有晶圆结构的晶体模板。设计好晶体模板后，可以利用水平扫描电子束对其进行刻画。

2. 构建光栅
使用电子束光刻时，首先要确定使用的光栅尺寸、厚度、刻度等参数。如果是模拟显示屏幕或生产测试用途，还需要考虑光栅精度、色彩饱和度以及动态范围等方面。

3. 漏电功能
为了提高光栅的抗漏电能力，可以使用双层或多层光栅结构，也可以使用激光等增强的方式来补偿。

4. 对晶体进行刻画
电子束光刻一般会以二值模式进行刻画。首先将电子束引入到电镀好的基底层中，通过镀锡等方法熔化，熔化完成后，便可使用挤出电子的方式切断电子束。然后，对剩下的电子进行分压，使其中的自由电子跟着剥离，这样就可得到含有二值的晶体。

### 2.1.2 多光子光刻与多普勒效应
多光子光刻（MWL）是指利用非均匀分布的多光子对晶体进行形成，且其波长范围可以覆盖多个真正的频率，使得晶体结构的不同频率分量可以同时存在。多光子光刻的特点是能够实现同一晶体的不同频率纹路模拟，并且整个过程是无缝连接的。对于一个完整的晶体结构来说，包括吸收、焊接、电镀、膜镀、封闭、反相等各个环节都需要有相应的处理，才能达到最终的目的。

多光子光刻中的多光子源实际上就是把原本散乱的电子束聚集到一起，成为由连续粒子构成的直线分布的多光子束。但是，多光子源必须注意避免产生高速旋转、跃迁，否则会导致多光子运动的不稳定，影响多光子的轨道选择及光谱分析结果。

多光子光刻有两种主要类型，即激光多光子光刻（ALS）和荧光多光子光刻（DLS）。前者是利用光的偏振现象对多种波长的多光子进行束加速，在不同频率处有不同程度的散射现象；而后者是在对晶体进行电子束形成之前，先用吸收镜和高功率激光对其电子进行电击操作，获得高精度的成像效果。

多普勒效应（Doppler effect）是指由于光速的变化引起的物体空间分布随时间而发生的变化称为多普勒效应。多普勒效应与电子束光刻、多光子光刻、荧光光刻等光刻技术密切相关，因为这些光刻技术都需要借助晶体结构的光谱分布特性，来达到不同的性能。由于不同晶体的空间结构和多普勒效应的存在，对于同一晶体的不同频率光刻，其光谱特征也不相同。

# 3.电子束光刻技术原理
电子束光刻技术利用电子束对晶体化学品进行形成，是目前应用最广泛的一种晶圆印制工艺。该技术利用了电子束的快速传播、精确晶体位置调配和精确性的制作过程。下图展示了一个电子束光刻过程示意图。

电子束光刻的主要流程分为以下四个阶段：

1. 准备：设计需要制作的晶体结构，根据预期的刻画效果选取适当的光栅尺寸。

2. 模拟：利用水平扫描电子束刻画晶体表面，通过构建电子束栅格来模拟光栅表面的反映。

3. 制作：把晶体表面和模拟区域隔开，采用电子束对晶体的某些区域施加电压，从而将晶体表面刻画成二值模式。

4. 清洗：擦去光栅上的电子流和杂质残留，使晶体表面清晰可见。

## 3.1 电子束光刻器件
电子束光刻器件是用来制作电子束光刻的装置，主要分为照相机类、集成电路板类、离心光刻类、激光类等。主要用于电子束光刻实验室设备如下图所示。

### 3.1.1 单摄像头类
单摄像头类电子束光刻器件包括显微摄像机、照相测量仪、影像分析仪、成像系统、电子束光刻系统及相关软件。主要用于扫描低分辨率图像，例如微距摄影机、X光机、红外摄影机等。

### 3.1.2 集成电路板类
集成电路板类电子束光刻器件主要用于激光光刻实验室设备，包括场镜类、电子束工作台类、三维扫描仪类等。主要用于精细刻画具有复杂晶体结构的实验对象。

### 3.1.3 离心光刻类
离心光刻类电子束光刻器件主要用于离心光刻实验室设备，包括数字光刻机、线性光刻机、电子束切削机及相关软件。主要用于精细刻画具有复杂晶体结构的实验对象，尤其适用于制备巨型晶体。

### 3.1.4 激光类
激光类电子束光刻器件主要用于激光光刻实验室设备，包括激光刻盘、激光刻纹机、成像系统、电子束光刻系统及相关软件。主要用于制备大尺寸的晶体。

## 3.2 电子束光刻原理
电子束光刻（Electron Beam Lithography, EBLL）是利用电子束对晶体化学品形成的方法。电子束是由个别自由电子组成，具有极短的传播距离，可以快速、准确地击穿晶体表面。它既可以在静止状态下发生，也可以通过激光等其它方式实现光刻制作。电子束光刻依赖于电子束的快速传播及精确的晶体位置调配，因此保证了晶体形成的精确程度。通常情况下，电子束光刻所用到的材料是具有晶圆结构的半导体材料，如GaAs、InP、SiC等。

电子束光刻主要步骤如下：

1. 设计晶体结构

电子束光刻的优点主要有：

1. 快捷：电子束光刻过程快，晶体的形成时间较短，因此可以满足各种制作要求。

2. 精确：电子束光刻技术依赖于电子束的快速传播及精确的晶体位置调配，因此保证了晶体形成的精确程度。

3. 简单的工艺：电子束光刻工艺简单易学，仅需掌握刻画技巧和切割技巧即可。

4. 灵活性强：电子束光刻可以在实验室内或远程进行，灵活方便。

电子束光刻的缺点主要有：

1. 晶体大小受限：电子束光刻不能用于制备大尺寸晶体，只能用于制备小尺寸晶体，而且容易产生晶体电流效应。

2. 不耐磨损：电子束光刻过程中容易出现污染，导致晶体失真或破坏。

3. 缺乏控制：电子束光刻通常采用默认设置，没有针对性的优化，无法得到理想的制作效果。

4. 价格昂贵：电子束光刻器件和软件价格昂贵，安装费用较高。

# 4.多光子光刻技术原理
多光子光刻（Multi-Wavelength Lithography, MWL）是利用非均匀分布的多光子对晶体进行形成，且其波长范围可以覆盖多个真正的频率，使得晶体结构的不同频率分量可以同时存在。多光子光刻的特点是能够实现同一晶体的不同频率纹路模拟，并且整个过程是无缝连接的。对于一个完整的晶体结构来说，包括吸收、焊接、电镀、膜镀、封闭、反相等各个环节都需要有相应的处理，才能达到最终的目的。

多光子光刻的主要步骤如下：

1. 准备：准备一个足够大的实验平台，用光谱仪和激光探测器检查晶体的结构和功能。

2. 布置光源：选取一个适当的材料和功能，如激光、X射线等，作为多光子光刻用的光源。

3. 测量光源的类型和功率：测量光源的类型和功率，确定光源能照到晶体的哪些区域。

4. 拍摄多光子图片：用激光探测器拍摄多光子图片，获得一张光谱图。

5. 将光谱图转换为指令：转换后的指令告诉光刻机如何刻画多光子。

6. 执行指令：电子束机执行指令，进行电子束光刻。

7. 检查和调整：对电子束光刻完工的晶体进行检查，以及调整指令，直至获得满意的结果。

## 4.1 多光子源分类
多光子光刻技术使用非均匀分布的多光子对晶体进行形成。与传统的电子束光刻技术一样，多光子光刻技术也是利用一个光源打到晶体表面上形成晶体的。那么，究竟什么样的光源更适合多光子光刻呢？多光子光刻常用的光源包括：

1. 高角聚焦激光：高角聚焦激光（Hyper Focal Irradiation, HIFI），又称双偏振光，是一种集束照相和多重色度收集技术。HIFI将激光束聚焦于一个单独的入射点，使用多种波长，可收集到不同颜色的光。其优点是有利于理解晶体表面的各个纹路，可以提供精确的晶体结构，但激光信号越多、占用带宽越宽，就越容易受到噪声影响。

2. 活塞耦合光源：活塞耦合光源（Active Shielding Source, ASS），是一种利用两个或多个激光辉光器组成的激光系统，每根激光系统包含有多种波长的光束。ASS将光线束聚焦于晶体表面，可同时收集不同波长的光。优点是可以收集多种波长的光，以达到不同频率纹路模拟，但输出功率很高，同时占用太多带宽资源。

3. 超冷激光：超冷激光（Supercooled Plasma Irradiation, SCI），是一种利用超冷量子液体做成的激光系统。SCI在冬季炽热的环境中释放出非均匀的光束，可以收集到晶体不同频率纹路之间的气泡，其输出功率很低。

4. 飞行激光：飞行激光（Flying Electron Irradiation, FEI），是一种利用飞入气流中的微弱电子做成的激光系统。FEI可以立即捕捉到晶体表面上发生的纹路变化，具有超高速率、高精度的能力，但其成像速度慢、成本高。

## 4.2 多光子光刻器件
多光子光刻技术的实验设备很多，主要有两类：

1. 材料工程类：主要用于制备晶体的材料工程类实验设备。包括微型样品装置、干涉模式探测器、光管成像仪器、光纤成像仪器、激光成像仪器等。

2. 工艺实验类：主要用于制备晶体的工艺实验类实验设备。包括微晶表面刻胶机、粒子加速器、多光子光刻机、低功率激光光刻机等。

## 4.3 多光子光刻实验过程
多光子光刻实验过程分为四步：

1. 布置实验平台：包括测量晶体，设置光源，布置探测器，建立光学基础设施等。

2. 设置光源：选择适当的材料和功率，调整光源的集束条件，使其能够有效穿透晶体表面，取得良好的多光子分布。

3. 制备晶体：包括采样晶体，给晶体加氢，使其平滑，然后用激光对晶体进行电击。

4. 使用多光子光刻机：多光子光刻机按照指令，将收集到的多光子射束投射到晶体表面，形成不同的纹路，并对晶体进行电击以固定其位置。

# 5.应用与实例
## 5.1 电子束光刻应用场景
电子束光刻的主要应用场景包括：

1. 制造信息化设备：电子束光刻可用于制造电路板、显示屏、激光显示屏等信息化设备。

2. 医疗成像：电子束光刻可用于医疗成像，利用特定透射模式对晶体进行刻画，以达到很高的成像精度。

3. 彩色显示屏制作：电子束光刻可用于制作具有不同颜色分量的彩色显示屏。

4. 电池设计：电子束光刻可用于电池设计，以电子束作为底层结构，并采用不同大小的晶体作为上层结构。

## 5.2 电子束光刻技术实例
### 5.2.1 GBIC（无源光刻机）
GBIC（无源光刻机）是无源光刻技术的一种形式。该机械装置可实现用电子束刻画晶圆，即使用电子束对晶体表面进行实时刻画，不需要经过其他工序。GBIC主要有两种类型，即回旋模式和水平扫描模式。

回旋模式是GBIC的标准模式，即直接用一个固定带宽的激光束射入晶体表面，使电子束以螺旋式旋转方式运动，模拟晶体结构的形成过程。这种模式主要适用于模拟工艺要求不高、快速制作需求高的场景。

水平扫描模式是GBIC的另一种模式，它将一个较窄的水平激光束与晶体表面垂直焦点对齐，然后沿着晶体表面扫描，模拟晶体结构的形成过程。这种模式对模拟精度要求比较高、制作工艺要求不严苛的晶体来说，效果还是比较好的。

GBIC的优点是材料和工艺成本低廉，易于使用。但是，由于电子束速率有限，GBIC无法用于制作精细晶体，需要借助激光光刻机来制作精细晶体。
