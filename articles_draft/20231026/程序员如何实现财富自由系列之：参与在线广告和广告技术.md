# 1.背景介绍


在线广告是互联网时代获取用户流量的一种重要方式，它可以帮助企业增加收入、提高竞争力、扩大规模，同时也给个人创造了新的收入机会。由于受到“大数据”、“云计算”等新技术的影响，广告投放领域也已经发生了巨变。那么，作为一个技术人员，是否也可以作为一名合格的广告技术专家，从事在线广告相关工作？
如果回答是肯定的话，那么首先需要掌握一些关于广告投放、广告营销等相关的基础知识。这里将提供一些基本的概念和术语介绍。
# 1.1 概念
广告（Advertisement）是指向消费者发出宣传品或产品信息的一项活动。一般来说，广告有以下几个特征：
- 内容性广告：这种广告向消费者展示的是某种商品或服务的信息，比如介绍产品功能、价格优惠等等。
- 宣传性广告：这种广告通过发布一些相关的图片、文字、视频等，来增加网民对某件事物的认识和喜爱度，并促进网民的购买行为。
- 行为性广告：这种广告主要依赖于顾客对特定商品或服务的不同消费习惠，如低价、折扣券、返利等。
# 1.2 媒介
媒介又称为广告平台，是广告的载体，包括互联网、平面媒介、广播电视等。目前互联网占据了绝大多数广告销售量，所以广告平台也是最主要的广告载体。
# 1.3 媒介类型
媒介类型通常由以下几类：
- 网络媒介：包括搜索引擎、社交媒体、门户网站、博客、论坛等。
- 移动应用媒介：包括手机 APP、微信、支付宝等。
- 消息推送媒介：包括短信、邮件、微信公众号、小红点、客户端推送等。
- 媒体资源：包括印刷品、电子产品、海报、宣传单等。
# 1.4 汇集点
汇集点是广告的投放位置。一般来说，汇集点分为两种：
- 搜索结果页：广告展示在网站或者 app 的搜索结果页面中。
- 应用内广告：广告展示在 app 或手机浏览器的内部页面中。
# 1.5 受众群体
受众群体通常由年龄、性别、教育水平、职业、收入、地域、婚姻状况、家庭状况等因素决定。根据受众群体的不同，广告制作会有所差异，因此广告技术人员需要针对不同的受众群体进行优化。
# 1.6 广告系列
广告系列是指同一广告形式下相同受众群体的广告集合，比如多个app中的推广活动。广告系列的目标就是达到高效、协同、统一的效果。广告系列通常由媒介公司管理，但可以分成多个子系列，每个子系列由不同的广告商管理。
# 2.核心概念与联系
广告技术人员经常用到的一些核心概念和联系如下图所示：

- 媒介平台与工具：媒介平台负责把广告创意打包形成广告素材，再通过各种方式呈现给用户。而各种工具则用来辅助媒介平台完成这一任务。比如，有些工具可以生成带有线索数据的海报，有些工具可以自动化地上传广告素材并通过审核。
- 流量分配：流量分配就是让广告素材在不同的媒介平台上展现出来，为其投入足够的点击率，最终达到广告主的目的。不同的广告主可能会使用不同的流量分配策略，比如CPM、CPI、CPA等。
- 数据分析：数据的收集和分析是广告工作不可或缺的一环。广告的数据可以反映用户的行为，包括搜索查询、查看产品详情、购买交易等；还可以通过数据分析发现用户喜好和偏好的变化，以便改善广告的设计和投放。
- 服务与支持：服务与支持是在线广告最重要的部分之一。广告主需要付费购买各种服务，比如专属顾问、设计团队、数据分析师、SEO推广等，这些都是为了确保广告素材能够快速准确地反应用户的需求。

## 3.1 广告投放的流程
1. 投放准备：在确定了广告系列之后，广告主就可以开始投放准备工作了。这其中包括内容策划、广告创意设计、素材拍摄、素材准备、流量分配、广告账户准备等。
2. 素材分发：准备好广告素材后，就可以把它们分发给各个媒介平台。有些平台要求广告主进行个人身份验证，这样才能保证广告的真实性和完整性。
3. 数据监测：分发成功的素材只是第一步，接下来就要进行数据监测，以评估广告效果。数据监测的方法很多，比如使用像Google Analytics这样的第三方数据分析工具，或自己开发自己的分析软件。
4. 定期维护：随着时间的推移，广告主可能需要不断更新广告内容、调整投放策略，因此需要定期进行维护工作。比如每半年检查一次投放效果、进行广告内容更新、调整流量分配比例等。
5. 维持生态：在日益壮大的互联网广告市场中，广告主需要与其他广告主建立长久稳固的关系。维持生态涉及到广告主之间的合作、渠道分成等工作。

## 3.2 广告创意设计的原则
在广告创意设计方面，广告主有一些基本原则和规范：
1. 创意思路：广告创意应该具有独特性，既有新颖又有别样，有别于同行的竞争对手。
2. 创意逻辑：广告创意需要包含符合人性化的观感和情感，并且能够把用户带入到广告界。
3. 创意视觉：广告创意应该充满表现力，突出创意主人的性格特点、品牌形象。
4. 创意方向：广告创意需要能够吸引用户的注意力，即要与用户需求相符，要有利于品牌的推广、销售和市场。
5. 创意执行：创意执行方面，需要考虑到广告的运营成本、创意素材的制作难度、项目实施周期等，还要考虑到广告投放后可能产生的效果。

## 3.3 素材拍摄和素材编辑技巧
素材拍摄是广告创意设计的一个环节，但是仍然需要一些技巧。以下是一些素材拍摄的建议：
1. 拍摄精准：拍摄的素材尽量让人感觉清晰、明了，不要出现背景杂乱、光线不均匀的情况。
2. 拍摄完整：广告主应该对素材进行完整的创作，从头到尾完整呈现整个广告画面。
3. 拍摄衬托物：拍摄衬托物可以增加广告的鲜明气质。
4. 素材命名：素材的命名要保持一致性，方便后续制作、管理。
5. 拍摄角度：拍摄时一定要有一定的观察角度，不要盲目跟风，否则可能会被打入地雷。
6. 拍摄角度：拍摄时一定要留意曝光条件，避免过度曝光、使得其他东西看不到。

## 3.4 流量分配的原理和方法
流量分配是广告过程中很重要的一环。流量分配原理简单说就是通过某种算法，让不同的广告创意投放到最适合用户的位置，以达到广告主的最大化利益。流量分配的方法有很多，以下是一些常用的流量分配方法：
1. CPM（Cost per Mille，每千次曝光计费）：以每次千次曝光计费的方式进行广告投放。
2. CPC（Cost per Click，每次点击付费）：以每次点击付费的方式进行广告投放。
3. CPA（Cost per Acquisition，每次访问付费）：以每次访问付费的方式进行广告投放。
4. 网络效果定向（Net Promoter Score，NPS）：利用用户对关键品牌的喜好程度、满意度、评论、投诉等反馈，来决定要不要继续推广某个产品。

## 3.5 流量分配的方案
流量分配的方案一般有两种：百分比和份额两种。百分比的流量分配意味着按固定百分比分配广告位；而份额的流量分配意味着按广告创意的属性，把广告位分成不同的份额。以下是一些常用的流量分配方案：
1. 随机分发：以随机的方式把所有的广告位分给不同的创意。
2. 历史分发：按曝光率、点击率、下载率等历史数据分发广告位。
3. 竞价排名分发：通过竞价的方式，来确定每个广告位的价格。
4. 配对分发：按一些重叠的区域分发不同的广告位。

## 3.6 数据分析的技能
数据分析是广告过程中不可或缺的一环。数据分析可以为广告主提供更多的投放建议，帮助广告主改善广告质量，提升品牌知名度和营收。以下是一些数据分析的技能：
1. 用户画像：数据分析可以帮助广告主更全面的了解用户的特征、行为习惠和偏好。
2. 时序数据：时序数据可以记录用户对广告创意的浏览、点击、转化等情况。
3. 行为数据分析：行为数据分析可以帮助广告主分析用户是怎样使用广告创意的。
4. 人工智能：人工智能技术的发展使得数据分析更加精准和可靠。

## 3.7 广告推广服务的提供
在线广告的成功离不开各种各样的服务，比如专属顾问、设计团队、数据分析师、SEO推广等等。为广告主提供各种服务的同时，还需要对他们进行培训，使他们熟悉在线广告的相关知识和业务。

### 4.1 内容策划
广告内容是一个大的话题，涉及的内容很多。这里只举一个简单的例子——如何策划一款游戏广告。
1. 游戏定位：游戏定位可以让玩家更容易理解游戏的含义。例如《魔兽世界》的定位就是“帕拉塞尔”，玩家要征服各种恶魔。
2. 游戏类型：游戏类型可以让玩家更直观地理解游戏的风格。例如《魔兽世界》的类型有“塔防”、“动作”、“角色扮演”等。
3. 受众群体：受众群体的构成对于玩家的收益非常重要。例如孩子可能对魔兽世界比较陌生，但老年人可能更容易接受。
4. 游戏画面：游戏画面是广告的亮点。广告创意应该突出游戏中的细节，包括物品、建筑、npc、背景音乐等。
5. 游戏机制：游戏机制能增强玩家的参与感。例如在游戏中玩家可获得技能，这些技能可以在游戏中产生作用。

在策划游戏广告时，广告主可以参考以上五个方面，用一两句话来表达广告内容。

### 4.2 广告创意设计
广告创意设计也是一个复杂的话题，有的人会花上几天时间设计出十几张不同风格的创意，还有的人会花上半年时间设计出几个精美的广告模板。不过无论多么繁琐，你都需要有一个初步的设计，让广告主看到你的创意。以下是一个广告创意的示例：


1. 品牌标识：品牌标识的设计要突出自己品牌的特征和个性。
2. 商品图片：商品图片的设计要突出商品的性价比。
3. 促销语词：促销语词的设计要突出商品的优惠政策和促销活动。
4. 购买按钮：购买按钮的设计要突出按钮的目的，让用户知道这个按钮的含义。
5. 字体颜色：字体颜色的选择要与商品的色调相呼应。

广告创意的设计主要有两个层面，一是创意视觉，即如何让创意看起来更漂亮；二是创意逻辑，即如何用语言把创意传达给用户。

### 4.3 素材拍摄
素材拍摄也是一个复杂的话题。刚才提到了素材命名的重要性，素材拍摄过程中还需注意一些细节，比如照片的曝光时间、空间分布、焦距等。

### 4.4 素材编辑
素材编辑的过程要配合一定的软件使用，包括像Adobe Photoshop这样的专业图像编辑软件、Premiere Pro这样的视频编辑软件等。编辑过程中还需注意导出文件的格式，为不同的平台准备不同的素材。

### 4.5 流量分配
流量分配其实是一个动态的过程，随着时间的推移，流量分配的模式、比例会发生变化。流量分配的范围一般有：搜索、应用、社交媒体等。

### 4.6 数据分析
数据分析的过程也是一个迭代的过程。数据收集、处理、分析、整理、展示的过程需要不断重复才能得到令人满意的结果。

### 4.7 广告推广服务
广告推广服务一般分为以下三大类：
1. 媒介支持服务：比如Facebook、Google、YouTube、Instagram、Twitter等。
2. 数据分析服务：比如Google Analytics、Chartbeat、Woopra等。
3. 活动支持服务：比如活动策划、活动执行、活动效果分析等。
