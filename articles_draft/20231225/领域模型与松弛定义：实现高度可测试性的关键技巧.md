# 2.核心概念与联系

## 2.1领域模型

领域模型是一种用于表示某个特定领域知识的形式化模型。它通常包括一组实体、属性、关系和规则，这些元素可以用来描述领域中的对象、属性和行为。领域模型可以帮助我们更好地理解和表示问题领域，从而提高系统的可测试性。

## 2.2松弛定义

松弛定义是一种用于处理实际应用中可能出现的不确定性和不完全性的方法。它允许我们在满足某些条件的前提下，对一个属性或关系进行宽松的定义。松弛定义可以帮助我们更好地处理实际问题，从而提高系统的可测试性。

    elif order.payment_method == 'alipay' and payment.payment_status == 'completed':
        return True
    elif order.payment_method == 'wechat_pay' and payment.payment_status == 'paid':
