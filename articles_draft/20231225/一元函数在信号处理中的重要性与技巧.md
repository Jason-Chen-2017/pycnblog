```python
# 信号采样

# 绘制信号

plt.xlabel('t')
plt.ylabel('f(t)')
plt.legend()
plt.show()
```

```python

plt.xlabel('t')
plt.ylabel('f'(t))
plt.legend()
plt.show()
```

```python
# 逆变换
f_prime = np.fft.ifft(f_prime_d)

plt.ylabel('f'(t))
```
