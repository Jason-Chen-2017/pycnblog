## 2.核心概念与联系

### 2.1 矩阵转置的定义和性质

矩阵转置是一种线性代数的基本操作，它可以将一维向量或二维矩阵的行列转换为列向量或行矩阵。对于一个二维矩阵A，其转置为A^T，其中A^T的行数等于列数，列数等于行数。

矩阵转置具有以下性质：

1. 对于任意矩阵A，(A^T)^T = A
2. 对于任意矩阵A和B，(A + B)^T = A^T + B^T
3. 对于任意矩阵A和B，(A - B)^T = A^T - B^T
4. 对于任意矩阵A和B，(AB)^T = B^T A^T
