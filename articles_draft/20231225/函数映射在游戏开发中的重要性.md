### 4.1 线性映射实例

以下是一个线性映射的具体代码实例：

```python
import numpy as np

def linear_mapping(x):
    a = 2
    b = 3
    y = a * x + b
    return y

x = np.array([1, 2, 3, 4, 5])
y = linear_mapping(x)
print(y)
```

在这个代码实例中，我们定义了一个线性映射函数`linear_mapping`，它接受一个输入`x`，并根据线性映射关系`y = 2x + 3`计算输出`y`。然后我们使用NumPy库创建一个输入数据集`x`，并调用`linear_mapping`函数计算输出数据集`y`。最后，我们使用`print`函数输出输出数据集`y`。
