## 4.1 信息熵计算
在这个例子中，我们将计算一个随机序列的信息熵。

```python
import numpy as np

# 随机序列
data = np.random.randint(0, 10, 100)

# 计算信息熵
def entropy(data):
    prob = np.bincount(data) / len(data)
    return -np.sum(prob * np.log2(prob))

print("信息熵:", entropy(data))
```

在这个例子中，我们首先生成了一个随机序列，然后使用`entropy`函数计算了序列的信息熵。

## 4.2 互信息计算
在这个例子中，我们将计算两个随机序列之间的互信息。

```python
import numpy as np

# 随机序列1
data1 = np.random.randint(0, 10, 100)

# 随机序列2
data2 = np.random.randint(0, 10, 100)

# 计算互信息
def mutual_information(data1, data2):
    prob1 = np.bincount(data1) / len(data1)
    prob2 = np.bincount(data2) / len(data2)
    prob12 = np.bincount([i1, i2] for i1, i2 in zip(data1, data2)) / len(data1)
    return -np.sum(prob1 * np.log2(prob1)) - np.sum(prob2 * np.log2(prob2)) + np.sum(prob12 * np.log2(prob12))

print("互信息:", mutual_information(data1, data2))
```

在这个例子中，我们首先生成了两个随机序列，然后使用`mutual_information`函数计算了两个序列之间的互信息。
