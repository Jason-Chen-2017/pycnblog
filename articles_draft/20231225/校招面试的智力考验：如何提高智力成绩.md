### 4.2逻辑问题代码实例

1. 判断一个数是否为素数：

```python
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

n = 7
print(is_prime(n))
```

2. 判断一个数是否为回文数：

```python
def is_palindrome(n):
    return str(n) == str(n)[::-1]

n = 12321
print(is_palindrome(n))
```

### 4.3语言问题代码实例

```python
def is_palindrome_string(s):
    return s == s[::-1]

s = "racecar"
print(is_palindrome_string(s))
```

2. 判断一个字符串是否为回文字符串（忽略大小写和空格）：

```python
def is_palindrome_string_ignore_case_and_space(s):
    s = s.lower().replace(" ", "")
    return s == s[::-1]
```
