## 4.1 线性规划算法代码实例

```python
import numpy as np
from scipy.optimize import linprog

# 定义目标函数
c = [-1, -2]

# 定义约束条件
A = [[2, 1], [1, 2]]
b = [4, 4]

# 求解最优解
x, slack = linprog(c, A_ub=A, b_ub=b, bounds=[(0, None), (0, None)])

print("最优解:", x)
print("约束条件:", slack)
```

在这个例子中，我们的目标是最小化能源消耗，约束条件是满足能源需求。通过线性规划算法的求解，我们可以得到最优解和约束条件。

## 4.2 遗传算法代码实例

```python
import numpy as np
from sklearn.metrics import mean_squared_error

# 定义适应度函数
def fitness(x):
    y = np.array([x[0]**2 + x[1]**2])
    return mean_squared_error(y, np.array([1, 1]))

# 初始化种群
population = np.random.rand(10, 2)

# 选择操作
selected = population[np.argsort(fitness(population))][-4:]

# 交叉操作
crossover_rate = 0.5
for i in range(len(selected)):
    for j in range(i+1, len(selected)):
        if np.random.rand() < crossover_rate:
            crossover_point = np.random.randint(1, 3)
            child1 = np.concatenate((selected[i][:crossover_point], selected[j][crossover_point:]))
            child2 = np.concatenate((selected[j][:crossover_point], selected[i][crossover_point:]))
            population = np.vstack((population, child1, child2))

# 变异操作
mutation_rate = 0.1
for individual in population:
    for i in range(len(individual)):
        if np.random.rand() < mutation_rate:
            individual[i] = np.random.rand()

# 评估适应度
fitness_values = np.array([fitness(individual) for individual in population])

# 终止条件
termination_condition = False

# 得到最优解
if not termination_condition:
    population = np.array([individual for individual in population[np.argsort(fitness_values)]])
    population = population[:2]

print("最优解:", population)
print("适应度:", fitness_values)
```

在这个例子中，我们的目标是通过遗传算法优化一个简单的函数，即最小化`x^2 + y^2`。通过遗传算法的求解，我们可以得到最优解和适应度。
