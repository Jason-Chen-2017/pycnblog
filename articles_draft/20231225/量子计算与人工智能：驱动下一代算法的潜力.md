### 1.4.1 量子位（qubit）的基本操作代码实例

```python
from qiskit import QuantumCircuit, execute, Aer

# 创建一个量子电路
qc = QuantumCircuit(2)

# 将第一个量子位设置为基态
qc.initialize([1, 0], 0)

# 应用Hadamard门
qc.h(0)

# 将量子电路执行
simulator = Aer.get_backend('qasm_simulator')
job = execute(qc, simulator)
result = job.result()

# 度量量子位
qc.measure([0], [0])

# 将量子电路执行
job = execute(qc, simulator)
result = job.result()
```
