实时分析在物流行业中的数学模型主要包括以下几种：

1. 线性回归模型：线性回归模型是一种常用的预测模型，它可以用来预测运输成本、运输时间等。线性回归模型的公式如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是预测变量，$x_1, x_2, \cdots, x_n$ 是预测因子，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。

2. 逻辑回归模型：逻辑回归模型是一种常用的分类模型，它可以用来预测货物是否会出现问题、是否需要特殊处理等。逻辑回归模型的公式如下：

$$
P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \cdots - \beta_nx_n}}
$$

其中，$P(y=1|x)$ 是预测概率，$x_1, x_2, \cdots, x_n$ 是预测因子，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

3. 决策树模型：决策树模型是一种常用的分类模型，它可以用来预测货物是否会出现问题、是否需要特殊处理等。决策树模型的公式如下：

$$
\text{if } x_1 \text{ is } A_1 \text{ then } \cdots \text{ if } x_n \text{ is } A_n \text{ then } y
$$

其中，$x_1, x_2, \cdots, x_n$ 是预测因子，$A_1, A_2, \cdots, A_n$ 是条件值，$y$ 是预测变量。
