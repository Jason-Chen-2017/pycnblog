# 4.3 机器人运动的具体代码实例
机器人运动的具体代码实例如下：

```python
import rospy
import geometry_msgs.msg

# 初始化节点
rospy.init_node('robot_move', anonymous=True)

# 创建发布者
pub = rospy.Publisher('/cmd_vel', geometry_msgs.msg.Twist, queue_size=10)

# 创建订阅者
rospy.Subscriber('/scan', Float32, callback)

# 发布移动命令
def callback(data):
    move = geometry_msgs.msg.Twist()
    move.linear.x = data
    pub.publish(move)
```
