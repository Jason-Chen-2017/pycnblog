# 2.核心概念与联系

## 2.1 概率模拟

概率模拟是一种通过随机样本来估计不确定性问题解的方法。它的核心思想是通过大量的随机样本来近似地计算期望值、概率等统计量。这种方法的优点是它无需知道解的表达式，只需知道问题的概率模型即可。
