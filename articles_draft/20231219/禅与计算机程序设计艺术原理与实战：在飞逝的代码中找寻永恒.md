## 3.1 动态规划

动态规划是一种常用的算法设计方法，它可以用来解决一类递归问题。动态规划的核心思想是将问题分解为子问题，然后将子问题的解存储在一个表格中，以便后续使用。

具体操作步骤如下：

1. 定义一个状态函数，用于描述问题的状态。
2. 根据状态函数，定义一个递归关系。
3. 根据递归关系，构建一个表格，用于存储子问题的解。
4. 根据表格，计算问题的解。

数学模型公式为：

$$
dp[i] = \max_{0 \leq j \leq i} \{ dp[j] + f(i, j) \}
$$

## 3.2 贪心算法

贪心算法是一种常用的算法设计方法，它的核心思想是在每个步骤中做出最佳的选择，以便最终得到最佳的解决方案。

具体操作步骤如下：

1. 定义一个目标函数，用于评估解决方案的质量。
2. 根据目标函数，在每个步骤中做出最佳的选择。
3. 重复步骤2，直到问题得到解决。

数学模型公式为：

$$
\arg \max_{x \in X} f(x)
$$

## 3.3 分治算法

分治算法是一种常用的算法设计方法，它的核心思想是将问题分解为子问题，然后将子问题的解组合成原问题的解。

具体操作步骤如下：

1. 将问题分解为子问题。
2. 对于每个子问题，递归地应用分治算法。
3. 将子问题的解组合成原问题的解。

## 4.1 动态规划实例

考虑一个 classic 的动态规划问题：最长公共子序列（LCS）。给定两个字符串 $s$ 和 $t$，找到它们的最长公共子序列。

```python
def lcs(s, t):
    m, n = len(s), len(t)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m):
        for j in range(n):
            if s[i] == t[j]:
                dp[i + 1][j + 1] = dp[i][j] + 1
            else:
                dp[i + 1][j + 1] = max(dp[i + 1][j], dp[i][j + 1])
    result = []
    i, j = m, n
    while i > 0 and j > 0:
        if s[i - 1] == t[j - 1]:
            result.append(s[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            i -= 1
        else:
            j -= 1
    return ''.join(reversed(result))
```

在这个实例中，我们首先定义了一个状态函数 `dp[i][j]`，用于描述字符串 $s$ 的前 $i$ 个字符和字符串 $t$ 的前 $j$ 个字符的最长公共子序列的长度。然后，我们根据状态函数定义了一个递归关系，并构建了一个表格用于存储子问题的解。最后，我们根据表格计算问题的解，并将结果以逆序输出。

考虑一个 classic 的贪心算法问题：最小Cut 问题。给定一个无向图 $G = (V, E)$，找到一个 cut $(S, V - S)$ 使得 $S$ 中的边数最少，同时满足 $S$ 中的所有 vertex 都在 $S$ 的边数大于等于 $k$。

```python
def min_cut(G, k):
    n = len(G)
    cut = [None] * n
    for i in range(n):
        cut[i] = ([], [])
    for u, v in G.edges():
        if len(cut[u][1]) < k:
            cut[u][1].append(v)
        elif len(cut[u][0]) < k:
            cut[u][0].append(v)
        else:
            cut[u][0].append(v)
            cut[v][1].append(u)
    S = [i for i in range(n) if len(cut[i][0]) < k]
    return (S, [i for i in range(n) if i not in S])
```

在这个实例中，我们首先初始化一个 cut 数组，用于存储每个 vertex 所属的 cut。然后，我们遍历所有的 edge，将其分配到不同的 cut 中。最后，我们找到一个满足条件的 cut 并返回。
