## 4.1 线性回归示例

以下是一个简单的线性回归示例：

```python
import numpy as np

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 3 * x + 2 + np.random.rand(100, 1)

# 训练模型
def linear_regression(x, y, alpha=0.01, iterations=1000):
    m, n = x.shape
    theta = np.zeros(n)
    for _ in range(iterations):
        predictions = x.dot(theta)
        errors = predictions - y
        gradient = (x.T).dot(errors) / m
        theta -= alpha * gradient
    return theta

# 预测
theta = linear_regression(x, y)

# 评估
predictions = x.dot(theta)
mse = np.mean((predictions - y) ** 2)
print("MSE:", mse)
```

在这个示例中，我们首先生成了一组线性回归数据，然后使用梯度下降法来训练线性回归模型。最后，我们使用训练好的模型来进行预测，并计算出均方误差（MSE）来评估模型的性能。

## 4.2 逻辑回归示例

以下是一个简单的逻辑回归示例：

```python
import numpy as np

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 1 * (x > 0.5) + 0

# 训练模型
def logistic_regression(x, y, alpha=0.01, iterations=1000):
    m, n = x.shape
    theta = np.zeros(n)
    for _ in range(iterations):
        h = 1 / (1 + np.exp(-x.dot(theta)))
        predictions = h * (y > 0.5) + (1 - h) * (y <= 0.5)
        gradient = (x.T).dot(predictions - y) / m
        theta -= alpha * gradient
    return theta

# 预测
theta = logistic_regression(x, y)

# 评估
predictions = (1 / (1 + np.exp(-x.dot(theta)))) > 0.5
accuracy = np.mean(predictions == y)
print("Accuracy:", accuracy)
```

在这个示例中，我们首先生成了一组逻辑回归数据，然后使用梯度下降法来训练逻辑回归模型。最后，我们使用训练好的模型来进行预测，并计算出准确率来评估模型的性能。
