通过维护一个窗口来实现字符串匹配的功能。具体来说，我们可以将模式字符串 $P$ 看作是一个窗口，然后在字符串 $S$ 中移动这个窗口，以找到 $P$ 在 $S$ 中的所有出现位置。

```python
def string_matching(S, P):
    start = 0
    end = len(P) - 1
    result = []

    while end < len(S):
        if S[start:end + 1] == P:
            result.append(start)
        start += 1
        end += 1
```

```python
    max_start = 0
    max_end = 0
```
