# 4.具体代码实例和详细解释说明

在智能家居领域，人工智能算法的具体代码实例如下：

1.智能插座：

```python
import time
import RPi.GPIO as GPIO

GPIO.setmode(GPIO.BCM)
GPIO.setup(2, GPIO.OUT)

def control_socket(state):
    if state == "on":
        GPIO.output(2, GPIO.HIGH)
    elif state == "off":
        GPIO.output(2, GPIO.LOW)

control_socket("on")
time.sleep(5)
control_socket("off")
```

2.智能温湿度传感器：

```python
import time
import DHT

DHT.setup(4)

def get_temperature():
    return DHT.read_temperature()

def get_humidity():
    return DHT.read_humidity()

temperature = get_temperature()
humidity = get_humidity()
print("Temperature: {}, Humidity: {}".format(temperature, humidity))
```

3.智能门锁：

```python
import time
import RPi.GPIO as GPIO

GPIO.setmode(GPIO.BCM)
GPIO.setup(2, GPIO.IN, pull_up_down=GPIO.PUD_UP)

def check_lock_status():
    return GPIO.input(2)

lock_status = check_lock_status()
print("Lock status: {}".format("locked" if lock_status else "unlocked"))
```
