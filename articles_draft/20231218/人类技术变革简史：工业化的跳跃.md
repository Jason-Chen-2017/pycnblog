### 1.4.1 农业革命代码实例

农业革命是人类历史上第一个工业化的跳跃。这个跳跃发生在18世纪末，使人类从农业社会跳跃到工业社会。以下是农业革命代码实例：

```python
import numpy as np
import pandas as pd

# 加载数据
data = pd.read_csv('agriculture_data.csv')

# 数据预处理
data['year'] = pd.to_datetime(data['year'])
data['year'] = data['year'].dt.year

# 计算生产力
data['productivity'] = data['output'] / data['labor']

# 计算生产成本
data['cost'] = data['input'] / data['output']

# 计算生产效率
data['efficiency'] = data['productivity'] / data['cost']

# 绘制生产力曲线

plt.xlabel('Year')
plt.ylabel('Productivity')
plt.title('Agricultural Productivity Over Time')
plt.show()

# 绘制生产成本曲线
plt.plot(data['year'], data['cost'])
plt.xlabel('Year')
plt.ylabel('Cost')
plt.title('Agricultural Cost Over Time')
plt.show()

# 绘制生产效率曲线
plt.plot(data['year'], data['efficiency'])
plt.xlabel('Year')
plt.ylabel('Efficiency')
plt.title('Agricultural Efficiency Over Time')
plt.show()
```
