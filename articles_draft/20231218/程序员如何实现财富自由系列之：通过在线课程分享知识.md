## 4.1 代码实例
以下是一个简单的 Python 代码实例，用于计算课程收入、利润和盈利率：

```python
def calculate_income(sales, price):
    return sales * price

def calculate_profit(revenue, cost):
    return revenue - cost

def calculate_profit_rate(profit, cost):
    return (profit / cost) * 100

# 课程销量
sales = 1000
# 课程价格
price = 99
# 课程成本
cost = 5000

# 计算课程收入
income = calculate_income(sales, price)
print(f"课程收入: {income}")

# 计算课程利润
profit = calculate_profit(income, cost)
print(f"课程利润: {profit}")

# 计算课程盈利率
profit_rate = calculate_profit_rate(profit, cost)
print(f"课程盈利率: {profit_rate}%")
```

## 4.2 详细解释说明
上述代码实例主要包括以下几个函数和变量：

- `calculate_income` 函数：用于计算课程收入，输入参数包括课程销量和课程价格。
- `calculate_profit` 函数：用于计算课程利润，输入参数包括课程收入和课程成本。
- `calculate_profit_rate` 函数：用于计算课程盈利率，输入参数包括课程利润和课程成本。
- `sales` 变量：表示课程销量。
- `price` 变量：表示课程价格。
- `cost` 变量：表示课程成本。

通过调用这些函数并传入相应的参数，可以计算出课程收入、利润和盈利率，从而了解课程的经济效益。
