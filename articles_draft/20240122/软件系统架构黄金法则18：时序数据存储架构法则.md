时序数据存储架构的核心算法原理包括：

- **时间序列数据压缩**：通过压缩算法，减少时间序列数据的存储空间。
- **时间序列数据分区**：将时间序列数据按照时间范围划分为多个分区，以提高查询效率。
- **时间序列数据索引**：通过索引技术，加速时间序列数据的查询。

具体操作步骤如下：

1. 收集时间序列数据。
2. 对时间序列数据进行压缩处理。
3. 将压缩后的时间序列数据分区。
4. 为分区数据创建索引。
5. 通过索引查询分区数据。

数学模型公式详细讲解：

- **时间序列数据压缩**：

$$
C = \frac{S_c}{S_o}
$$

其中，$C$ 是压缩率，$S_c$ 是压缩后的数据大小，$S_o$ 是原始数据大小。

- **时间序列数据分区**：

$$
P = \frac{T}{t}
$$

其中，$P$ 是分区数，$T$ 是时间序列数据总时间范围，$t$ 是单个分区的时间范围。

- **时间序列数据索引**：

$$
Q = \frac{T_q}{t_q}
$$

其中，$Q$ 是查询速度，$T_q$ 是查询时间，$t_q$ 是单个查询的时间。

## 4. 具体最佳实践：代码实例和详细解释说明

具体最佳实践的代码实例如下：

```python
import pandas as pd

# 收集时间序列数据
data = pd.read_csv('data.csv')

# 对时间序列数据进行压缩处理
data_c = data.compress()

# 将压缩后的时间序列数据分区
data_p = data_c.partition(30)

# 为分区数据创建索引
data_p.index = pd.MultiIndex.from_tuples([(i, j) for i in range(30) for j in range(24)])

# 通过索引查询分区数据
query = data_p.loc[(1, 12)]
```

详细解释说明：

1. 使用 pandas 库收集时间序列数据。
2. 使用 compress() 函数对时间序列数据进行压缩处理。
3. 使用 partition() 函数将压缩后的时间序列数据分区。
4. 使用 MultiIndex.from_tuples() 函数为分区数据创建索引。
5. 使用 loc() 函数通过索引查询分区数据。
