## 5. 具体最佳实践：代码实例和详细解释说明

在本节中，我们将讨论具体最佳实践：代码实例和详细解释说明。以下是一个简单的插件示例：

```java
@SpringBootApplication
public class DemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }

    @Bean
    public MyPlugin myPlugin() {
        return new MyPlugin();
    }

}

public class MyPlugin {

    public void doSomething() {
        System.out.println("Doing something...");
    }

}
```

在上述示例中，我们创建了一个名为`MyPlugin`的插件。`MyPlugin`类包含一个名为`doSomething`的方法。`DemoApplication`类中，我们使用`@Bean`注解创建了一个`MyPlugin`实例，并将其注入到Spring容器中。然后，我们调用`doSomething`方法，输出“Doing something...”。
