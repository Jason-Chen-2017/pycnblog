# 2.核心概念与联系
首先，让我们来认识下几个关键术语：

- **变量**：即存储数据的内存空间，需要先声明再使用。在计算机程序运行时，变量的值随着计算或输入而发生变化。通常情况下，变量是用来保存信息和值的，例如：整数、浮点数、字符串等。
- **数据类型**：表示一个变量所存储的数据种类，不同的数据类型决定了变量能够保存哪些值。在Python中，主要有以下几种数据类型：
  - 整型（int）：整型变量可以保存整数值，包括正负、零、负零。在Python中，整数类型使用`int`关键字表示，如`num = 10`，其中`num`是一个整数变量。
  - 浮点型（float）：浮点型变量可以保存小数或科学计数法的值。在Python中，浮点型类型使用`float`关键字表示，如`pi = 3.14`。
  - 字符串型（str）：字符串型变量可以保存文本形式的值，可由单引号、双引号或三重引号包裹的文本组成。在Python中，字符串类型使用`str`关键字表示，如`name = 'John'`。
  - 布尔型（bool）：布尔型变量只能保存两个值，即`True`或者`False`。布尔型变量用于逻辑判断，在条件语句、循环语句中起作用。在Python中，布尔型类型使用`bool`关键字表示。
  - 列表型（list）：列表型变量是按顺序排列的一系列值，可以使用索引访问其中的元素，也支持增删改查操作。列表中的值可以不同类型。在Python中，列表类型使用方括号`[]`表示，如`fruits = ['apple', 'banana', 'orange']`。
  - 元组型（tuple）：元组型变量类似于列表型变量，只是元素不能修改，也不支持增删操作。元组类型使用圆括号`()`表示，如`point = (x, y)`。
  - 字典型（dict）：字典型变量保存键值对，键必须是不可变类型（数字、字符串、元组），值可以是任意类型。字典类型使用花括号`{}`表示，如`person = {'name': 'John', 'age': 30}`。
  - 集合型（set）：集合型变量类似于列表，但不支持重复的元素。集合类型使用尖括号`<>`表示，如`nums = {1, 2, 3, 4}`.

接下来，我们将根据以上关键概念，详细讲述下什么是变量和数据类型。
## 变量（variable）
变量就是存储数据的内存空间，需要先声明再使用。在计算机程序运行时，变量的值随着计算或输入而发生变化。在Python中，变量需要指定数据类型，如整数型、浮点型、字符串型等。我们可以通过赋值运算符`=`给变量赋值，如：
```python
# 创建整数型变量num
num = 10
print(num) # 输出结果：10
```
这里，我们创建了一个名为`num`的整数型变量，并赋予初值为`10`。然后，我们打印变量的值，输出结果为`10`。如果要重新给变量赋值，也可以使用如下方式：
```python
num = 20
print(num) # 输出结果：20
```
## 数据类型（data type）
数据类型是指变量所存储的数据种类，不同的数据类型决定了变量能够保存哪些值。在Python中，主要有以下几种数据类型：
### 整型（integer）
整型变量可以保存整数值，包括正负、零、负零。在Python中，整数类型使用`int`关键字表示，如`num = 10`，其中`num`是一个整数变量。
```python
num = 10
print(type(num)) # <class 'int'>
```
上面代码创建了一个名为`num`的整数型变量，并打印变量的数据类型。输出结果为`<class 'int'>`，表明该变量的数据类型为整数型。
### 浮点型（float）
浮点型变量可以保存小数或科学计数法的值。在Python中，浮点型类型使用`float`关键字表示，如`pi = 3.14`，其中`pi`是一个浮点型变量。
```python
pi = 3.14
print(type(pi)) # <class 'float'>
```
上面代码创建了一个名为`pi`的浮点型变量，并打印变量的数据类型。输出结果为`<class 'float'>`，表明该变量的数据类型为浮点型。
### 字符串型（string）
字符串型变量可以保存文本形式的值，可由单引号、双引号或三重引号包裹的文本组成。在Python中，字符串类型使用`str`关键字表示，如`name = 'John'`,其中`name`是一个字符串型变量。
```python
name = "John"
print(type(name)) # <class'str'>
```
上面代码创建了一个名为`name`的字符串型变量，并打印变量的数据类型。输出结果为`<class'str'>`，表明该变量的数据类型为字符串型。
### 布尔型（boolean）
布尔型变量只能保存两个值，即`True`或者`False`。布尔型变量用于逻辑判断，在条件语句、循环语句中起作用。在Python中，布尔型类型使用`bool`关键字表示。
```python
flag = True
print(type(flag)) # <class 'bool'>
```
上面代码创建了一个名为`flag`的布尔型变量，并打印变量的数据类型。输出结果为`<class 'bool'>`，表明该变量的数据类型为布尔型。
### 列表型（list）
列表型变量是按顺序排列的一系列值，可以使用索引访问其中的元素，也支持增删改查操作。列表中的值可以不同类型。在Python中，列表类型使用方括号`[]`表示，如`fruits = ['apple', 'banana', 'orange']`。
```python
fruits = ["apple", "banana", "orange"]
print(type(fruits)) # <class 'list'>
print(fruits[0])    # apple
fruits.append("grape")
print(fruits[-1])   # grape
del fruits[1]
print(fruits)       # [apple, orange, grape]
```
上面代码创建了一个名为`fruits`的列表型变量，并进行了一些操作，包括获取第一个元素、添加元素、删除元素等。
### 元组型（tuple）
元组型变量类似于列表型变量，只是元素不能修改，也不支持增删操作。元组类型使用圆括号`()`表示，如`point = (x, y)`。
```python
coord = (3, 5)
print(type(coord))      # <class 'tuple'>
print(len(coord))       # 2
coord_x, coord_y = coord
print(coord_x)          # 3
```
上面代码创建了一个名为`coord`的元组型变量，并尝试访问其长度和元素。
### 字典型（dictionary）
字典型变量保存键值对，键必须是不可变类型（数字、字符串、元组），值可以是任意类型。字典类型使用花括号`{}`表示，如`person = {'name': 'John', 'age': 30}`。
```python
person = {"name": "John", "age": 30}
print(type(person))     # <class 'dict'>
print(person["name"])   # John
person["address"] = "123 Main St."
print(person)           # {'name': 'John', 'age': 30, 'address': '123 Main St.'}
```
上面代码创建了一个名为`person`的字典型变量，并尝试添加和读取键值对。
### 集合型（set）
集合型变量类似于列表，但不支持重复的元素。集合类型使用尖括号`<>`表示，如`nums = {1, 2, 3, 4}`.
```python
numbers = set([1, 2, 3, 4])
print(type(numbers))        # <class'set'>
print(2 in numbers)         # True
numbers.add(5)
numbers.remove(3)
print(numbers)              # {1, 2, 4, 5}
```
上面代码创建了一个名为`numbers`的集合型变量，并尝试添加和移除元素。
