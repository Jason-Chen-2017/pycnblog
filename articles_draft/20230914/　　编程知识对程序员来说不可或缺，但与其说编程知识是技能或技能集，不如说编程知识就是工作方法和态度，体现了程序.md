# 2.编程语言及其特点
## 2.1.什么是编程语言？
　　编程语言（Programming Language）是一门计算机指令和语法的集合，它定义了一种新的计算机编程的方法、规范，并赋予计算机以编写程序的能力。编程语言通常分为解释型语言和编译型语言两大类。

　　1.解释型语言
　　解释型语言是在运行时才把代码翻译成机器语言。解释器读取源代码，然后逐行解释执行。例如：Python，JavaScript，PHP，Ruby，Perl。

　　2.编译型语言
　　编译型语言是在代码编译过程中把代码翻译成机器语言。编译器把源码编译成字节码文件，然后虚拟机或硬件直接运行字节码文件。例如：C，C++，GoLang。

## 2.2.编程语言分类
　　按照语言类型，可以将编程语言分为三种：

　　1.高级语言（High-Level Language）：一般由编译器或解释器转换成机器语言。如：C，C++，Java，GoLang等。

　　2.低级语言（Low-Level Language）：是一种机器语言，直接被计算机所识别。如：汇编语言，Fortran语言。

　　3.脚本语言（Scripting Languages）：用来编写自动化脚本，可以让程序实现某项特定功能。如：Shell Script，PowerShell，Lua等。

## 2.3.变量类型及作用域
### 2.3.1.基本数据类型（Primitive Data Type）
　　基本数据类型指的是在程序中最简单的、不可再分的数据类型。

　　　　1.整型（Integer）：整型数据类型用于存储整数值，包括有符号整数、无符号整数和字符编码。主要包括如下几种：

　　　　　　①有符号整数：包括短整型（short），整型（int），长整型（long）。其中，short占两个字节，int占四个字节，long占八个字节；在计算机内存中，它们都是二进制补码表示形式。

　　　　　　②无符号整数：包括无符号短整型（unsigned short），无符号整型（unsigned int），无符号长整型（unsigned long）。其中，unsigned short占两个字节，unsigned int占四个字节，unsigned long占八个字节；它们的值范围均为0到2^n-1。

　　　　　　③字符编码：char类型用来存储单个字符，占两个字节。它的值范围是0到2^16-1。

　　　　　　④布尔类型（boolean）：只有true/false两种取值。它占一位存储空间。

　　　　2.浮点型（Floating Point）：浮点型数据类型用于存储小数值，包括单精度浮点型和双精度浮点型。它们之间的区别在于存储大小不同，double类型占八个字节，float类型占四个字节。

　　　　　　①单精度浮点型：float占四个字节，它的存储范围是±3.4E38~±1.2E-38，精度可达7位有效数字。

　　　　　　②双精度浮点型：double占八个字节，它的存储范围是±1.7E308~±4.9E-324，精度可达15位有效数字。

　　　　3.字符串型（String）：字符串型数据类型用来存储文本信息，如：姓名、地址、生日等。它是一个由零个或多个字符组成的序列。

　　　　　　①固定长度字符串：使用预先分配好的一段内存来存储字符串，字符串的长度固定且不可变。如：char name[10] = "zhangsan";

　　　　　　②可变长度字符串：使用堆栈或堆内存来存储字符串，字符串的长度可以动态变化，且可以通过指针访问。如：std::string str; //声明字符串变量str
               std::cin >> str; //从标准输入流读取字符串
               printf("%s", str.c_str()); //输出字符串

### 2.3.2.引用数据类型（Reference Data Type）
　　引用数据类型是指在程序中使用的变量或参数。这些变量或参数指向其他数据结构，而不是包含数据。

　　　　1.数组（Array）：数组是一个连续的内存块，用来保存同类型数据的集合。数组中的元素可以通过索引来访问，索引是从0开始的。数组的长度在创建时指定，之后不能改变。如：int arr[10];

　　　　　　①一维数组：每一个元素占用相同的内存空间，都存在数组的起始位置，数组的第一个元素的地址是固定的，而数组的第i个元素的地址是固定的偏移量i*(sizeof(data type))。

　　　　　　②多维数组：多维数组类似于矩阵。

　　　　2.指针（Pointer）：指针是一个变量，它存储另一个变量的内存地址。指针可以指向任意类型的变量，包括基本数据类型和引用数据类型。

　　　　　　①指针作为参数：函数的参数可以是指针类型，这样就可以修改调用者传递的参数的值。

　　　　　　②指针作为返回值：函数的返回值也可以是指针类型，这样就可以返回一个指向结果值的指针。

　　　　3.结构体（Struct）：结构体是一个用户自定义的数据类型，它可以包含多个成员变量，并且可以定义方法。结构体可以作为参数和返回值。

　　　　4.枚举（Enum）：枚举是一个命名的数据类型，它由若干个称为枚举常量的整型常量组成。枚举常量的值就是它们在枚举定义时的顺序。枚举常量可以作为参数和返回值。

## 2.4.运算符和表达式
### 2.4.1.运算符
　　运算符是对算术运算符、关系运算符和逻辑运算符的统称，用来处理各种运算任务。

　　　　1.算术运算符：包括加法、减法、乘法、除法、取余、幂运算符。

　　　　　　①+、-、*、/、%、**：分别表示加、减、乘、除、取余、幂运算。

　　　　　　②+=、-=、*=、/=、%=：这些运算符也是加、减、乘、除、取余的复合赋值运算符。

　　　　2.关系运算符：包括等于、不等于、大于、小于、大于等于、小于等于。

　　　　　　①==、!=、>、<、>=、<=：分别表示等于、不等于、大于、小于、大于等于、小于等于。

　　　　3.逻辑运算符：包括与（&&）、或（||）、非（!）、异或（XOR）。

　　　　　　①&&、||、!：这三个运算符是逻辑运算符，可以组合使用。

　　　　　　②^：这是异或运算符，可以计算两个相异的数的唯一值。

### 2.4.2.表达式
　　表达式是由运算符和变量组成的一个完整的语句。表达式可以用来计算一个值，也可以表示一个条件或语句的结果。

　　　　1.赋值表达式：表达式右侧的值可以赋值给左侧的变量。如a=b=10;

　　　　2.逻辑表达式：可以用来控制程序的流程，包括if语句、while语句等。如a>b && b<c || a == c;

　　　　3.条件表达式：可以根据条件是否满足来决定表达式的值。如a?b:c;

## 2.5.控制语句
### 2.5.1.选择语句（Selection Statement）
　　选择语句可以基于不同的条件来执行不同的代码块。

　　　　1.if语句：if语句是条件选择语句，它判断一个条件是否满足，并根据条件的结果来执行相应的代码块。

　　　　　　①if语句：if (condition) { statement }；

　　　　　　②if…else语句：if (condition) { statement1 } else { statement2 }；

　　　　　　③嵌套if语句：if (condition1) { if (condition2) { statement }; }

　　　　2.switch语句：switch语句根据某个表达式的值来执行相应的case子句。

　　　　　　①switch语句：switch (expression) { case constant: statement; break; default: statement; }

　　　　　　②注意：switch语句匹配的是常量，而不是变量，所以不会发生变量比较的问题。

　　　　3.goto语句：goto语句是无条件跳转语句，它强制程序跳到指定的标签处。

### 2.5.2.循环语句（Loop Statement）
　　循环语句是用来重复执行一个代码块的语句。

　　　　1.for语句：for语句是一种特殊的循环语句，它可以一次性完成初始化、条件判断和迭代的代码。

　　　　　　①for语句：for (initialization; condition; iteration) { statement }；

　　　　　　②注意：初始化、条件和迭代三个部分的顺序非常重要。

　　　　2.while语句：while语句是一种普通的循环语句，它依次执行条件语句代码块，直到条件表达式为假时结束。

　　　　　　①while语句：while (condition) { statement }；

　　　　3.do…while语句：do…while语句是一种特殊的循环语句，它保证至少执行一次语句块。

　　　　　　①do…while语句：do { statement } while (condition);；

## 2.6.函数
　　函数是独立的可执行代码片段，它接受输入参数并返回输出结果。函数由函数名、参数列表和函数体组成。

　　　　1.定义函数：定义函数时，需要给定函数名称、参数列表、返回类型（如果有）、函数体。如：int add(int x, int y){ return x + y; }

　　　　2.调用函数：调用函数时，只需要传入正确的参数即可。如：int result = add(x,y);
