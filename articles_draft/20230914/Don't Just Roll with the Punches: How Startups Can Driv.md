# 1.简介
  

2021年，我国将迎来税收“万箭穿心”的一年。如果说去年美国纳税人收入激增、工资单上缴占比急剧下降是一个突破口的话，今年国家税务总局在印发《税收征管规定》后，税务收入占比也出现同样的表现。然而，更令人担忧的是“一刀切”的惯性操作会导致税务负担过重，引起社会不满和抗议。要打造更高质量的企业，首先要克服这个弊端。近几年，随着互联网经济的蓬勃发展，越来越多的创新型初创企业主选择了互联网模式，试图用科技驱动商业变革。这些新型企业家面临的首要任务就是如何吸引投资，怎样降低税负。本文将从税率到税务管理三个方面，阐述我国商业初创企业的税务策略，给出详细操作指导建议。 

# 2.基本概念术语说明
## 2.1.背景介绍
### 2.1.1.什么是税务？
按照《中华人民共和国行政处罚法》第二条第一款，“对违反国家规定的行为实施处罚”，故称“税务”。

### 2.1.2.为什么要缴纳税？
- 从法律上说，为了维护社会公正和经济秩序，保护国家财产安全，税制是各个国家义务保障公民权利的重要组成部分；
- 从事活动涉及国民经济利益时，需要缴纳相应的税费。比如企业、个人开展生产经营活动需要缴纳所得税，个人所得税代表个人所得和收入中的个别项目的税额；
- 在缴纳税费之前，还要考虑到维持社会稳定所需的开支。比如，为了使城市居民能够拥有基本生活必需品，政府就可能征收物业服务税或保洁服务税；
- 当经济危机时，税收减免、缓缴甚至免税也是合理的做法。

### 2.1.3.税率
1. 营业税：企业所得税、个人所得税等。
2. 增值税：除了营业税外，公司收取的其他各种税费都属于增值税，如印花税、消费税、燃气税等。增值税的种类多达十几种，以不同税率适用不同的项目。
3. 城建税：城市建设、拆迁、改扩建等税费。
4. 消费税：购买商品和服务所需交付的地方政府提供的消费税。
5. 教育费附加税（通常称为教育费增值税）：购买公共教育课外教育费用的附加税。
6. 预扣税率：由于企业应当缴纳预缴的税款，所以应该严格按照规定预缴，并将剩余的税额按年度预扣的方式缴纳。预扣税率则是指预缴的税款部分所对应的税率。
7. 消费税率调整：根据实际情况调整消费税税率，防止过高或过低。

## 2.2.一般收入和所得税
### 2.2.1.一般收入
一般收入包括：收入、债务、利息、配偶收入、礼金、雇佣金、租金、遗产捐赠、退休金、婚姻费、子女教育费、分红、股票收益、储户存款、透支、汇兑。

### 2.2.2.所得税
所得税是指企业为其雇员缴纳的税额，一般按照个人所得税率缴纳。但是，企业可能会向省、自治区、直辖市税务部门申报企业所得税的部分。具体地，一旦企业从境内或者境外获得收入超过某个金额（金额不低于某个最低限额），即可向税务部门申请企业所得税抵扣。所得税适用于企业所有制形式——自然人股份有限公司、有限责任公司和其他类似企业，以及小微企业。以下是关于企业所得税的一些规定：

1. 个体户所得税率：适用企业登记证上的核定所得税率，个体户缴纳个人所得税时，仍要缴纳个体户所得税率，否则视为免收所得税。个体户所得税率在实际使用中要受到法律保护，不能随意修改。个体户所得税率在2021年末时已经调整至0.3%左右。

2. 小规模纳税人收入金额：对于收入金额低于2500元的纳税人（含个体户、农村集体企业主、承包经营户和其他适用个人所得税补助规定的纳税人），应缴纳个人所得税。缴纳金额不少于15%；收入超过2500元的纳税人缴纳个人所得税的比例可适当提高到20%。

3. 非纳税人收入金额：对于非纳税人（如基金、信托、个人投资者、合伙企业等）缴纳个人所得税的金额一般为所缴纳收入的3%至5%，但具体缴纳比例仍要结合纳税人的实际收入。

4. 跨省、跨区域纳税人所得税优惠：如果纳税人同时从两个以上省、自治区、直辖市缴纳个人所得税，且满足一定条件，可以享受一个月的跨省、跨区域所得税优惠。

5. 限额累进税率：个人所得税的总金额不得超过每年个人收入的限额，即所得税税率不得高于等于所得额的8%。如果超出限额，须缴纳增值税。

6. 植被收入免征个人所得税：如果在江、沪、浙三角地区开展植树造林活动，免征个人所得税。

7. 农村集体企业主和个人所得税补助：适用农村集体企业主和个人缴纳农业生产资料、办公设备及相关税金的纳税人，其所得税由企业统一征收。个人所得税补助用于补助少数特困人员等个人缴纳农业生产资料、办公设备等税金。

## 2.3.个人所得税税率
### 2.3.1.税率标准
2021年中国个人所得税税率规定如下：

1. 基本每月扣除：税前净收入扣除五险一金后的每月实际到手的收入，超过部分每月直接计入个人所得税（税额按3%征收）。如不足一月，按实际到手计算所得税。2020年的企业所得税个人所得税标准也是按5%计算。

2. 每月实缴比例：企业上年度缴纳的个人所得税（未经过个人征收减半）加上个人贡献增值的税金后，计入本次缴纳的个人所得税实缴比例为：
   - （个人所得税税率／增值税率／所得增值税税率调节系数×附加税扣除率）+（所得税折旧摊销率／增值税率）。
   - 其中，所得税税率调节系数通常为1.1至1.4，附加税扣除率为0.2。如超过10万元，可适当增加所得税税率调节系数。

   根据此规定，2021年个人所得税计算方式如下：

   - 基本每月扣除：根据申报信息核定个人所得税税率。如果申报信息不正确，则默认收入超过10万元的所得税税率为9%；2020年的企业所得税个人所得税标准也是按5%计算。
   - 每月实缴比例：企业上年度缴纳的个人所得税（未经过个人征收减半）加上个人贡献增值的税金后，除以本次缴纳的实收金额。

   此方法既保障了每个人缴纳所得税的公平，又促进了税收收入的分享。

3. 年终奖励税收抵扣：除扣除基本每月扣除部分之外，年终奖励税收可以抵扣个人所得税。年终奖励税收通常包括：
   - 年终奖金：作为社保福利基金、稳岗基金、奖金、年度奖励等来源缴纳的年终奖励。
   - 年终业绩奖励：企业向年度业绩考核结果说明的激励性奖励，包括表彰、奖励、裨益等。
   - 年终业绩与规划奖励：根据企业年度计划和目标，奖励其完成该计划的业绩和效益。
   - 年终绩效奖励：对主管领导、员工以及其他单位的业务表现进行评价，评出优秀的工作先进、保持高度的道德品质等，给予鼓励和奖励。

   2021年个人所得税的年终奖励税收抵扣限额为：
   - 个人扣除基础每月扣除部分之后年终奖励总额的四分之一；
   - 2020年为每月1310元，2021年为每月1500元，如果当期年终奖励税额超出限制，只按其实际到手额扣除个人所得税。

### 2.3.2.适用人群
个人所得税主要适用于个人和小微企业。2021年中国个人所得税税率最高到43.6%，分别适用于：

1. 个人户主：户均年收入不超过6万元的自然人；
2. 一般小规模纳税人：每月实际到手收入不超过2500元的小规模企业主。
3. 一般中小纳税人：每月实际到手收入不超过5000元的中小企业主。
4. 特困、困难劳动者、残疾人：每月实际到手收入不超过3万元的特殊困难家庭。
5. 半边裔、女性、自愿捐赠者：每月实际到手收入不超过10万元的多样化税收人群。

### 2.3.3.不适用人群
- 国企、事业单位职工：职工个人所得税由所在单位统一缴纳；
- 教师、学生：在教育、培训方面的收入不含个人所得税；
- 不可披露信息的收入：指个人隐私信息泄漏导致没有足够的资料、权限追查来源、超出个人逃避欲望而逃税的情况。这类情况只能由国家监察部门通过专项检查识别和依法采取法律手段进行处罚。

## 2.4.企业所得税税率
### 2.4.1.适用对象
企业所得税主要适用于符合规定的企业，具体包括：

1. 有限责任公司、股份有限公司、有限合伙企业和其他类型的企业；
2. 国有企业、集体企业和其他组织：国有企业、集体企业、专营店、经营性附属企业、农牧渔、房地产开发等企业必须缴纳企业所得税；
3. 上市公司、企业状态不明、暂停营业、注销的企业以及其他原因无法确定是否缴纳企业所得税的企业。

### 2.4.2.税率标准
2021年中国企业所得税税率规定如下：

1. 大额纳税人缴纳企业所得税率：大于2000万元的纳税人纳税额3%，5000万元以下的纳税人纳税额6%。企业缴纳个人所得税的所得额小于等于800万元时，免缴企业所得税。

2. 三证合一的企业缴纳企业所得税：公司注册地与营业场所所在地一致，并且存在实体完整文档。

3. 行政处罚企业的可伤害性：可影响其正常经营的严重事件，包括突发性大规模火灾、爆炸、水灾、地震、龙卷风、台风、战争等。

4. 专票企业缴纳企业所得税：适用于自然人股东和控股股东的中长期投资企业。

5. 企业所得税特别偏远地区及高风险行业企业的个人所得税缴纳额加一倍。

6. 一二线城市率较低，需多增征一部分税款。

7. 可退税企业的税率适当降低，一般为3%~4%。

8. 可被认定为破坏游戏规则的行为的企业的税率最低降至最低税率的百分之六。

9. 企业所得税征收的部分可适当减半或全额退还。

### 2.4.3.企业所得税缴纳方式
1. 分级征收：在税负率较低的情况下，适用分级征收，逐级递减。第一级所得税为2.5%，第二级为1.5%，第三级为0.5%，最高不超过0.25%；
2. 毛利扣除：企业销售毛利额的5%，扣除之后的5%归个人所得税。也可以使用固定税率方式缴纳企业所得税；
3. 拆分计税：对经营成果进行拆分，上交的部分为减值部分，下交的部分为增加值部分，上交部分2.5%，下交部分1.5%。按照年度税收年底计入个人所得税；
4. 可供选择的其他缴纳方式。

## 2.5.税务管理
### 2.5.1.税务分类
1. 国税与地税：关乎国家经济运行和国家安全的税种。国税主要包括：企业所得税、个人所得税、增值税、消费税、城建税、教育费附加税、预扣税、附加扣除和契税。地税主要包括：土地增值税、车船税、郊区建设税、农业用地税、农村土地承包经营权转让税、耕地宅基地使用权税、环保税。
2. 地方税：着重对当地居民的收入，涉及市容、住房、教育、医疗、交通、水利、金融、公共设施、环境、资源利用、物业服务等方面。
3. 海关税：关乎国际贸易的税种。
4. 关税：关乎国际金融的税种。
5. 印花税：面向个人或小微企业，提供印刷专用商品的消费税。

### 2.5.2.税务管理原则
1. 减轻扣除：减少、避免或取消减轻对纳税人申报义务。如免税、滞纳金、补缴和其他。
2. 提高收入分配：充分考虑企业产品质量、技术进步、顾客需求和区域差异性，提升纳税人收入分配。
3. 降低税率：适度降低对不同行业、不同收入水平的税率。
4. 规范政策：坚持税收征管和收费管理的有效性，改善税务执法部门的执法能力。
5. 监督纳税：落实税收征管，尤其是严格监督所得税实缴情况。

### 2.5.3.如何设立个人所得税专票账户
设立个人所得税专票账户，目的是将个人所得税专项实缴预扣减半、企业所得税专项抵扣预扣、公司预缴预扣减半等情况的个人所得税余额自动转移到专票账户，以便做好个人所得税的专项管理。设立个人所得税专票账户的具体操作如下：

1. 缴纳费用。个人所得税专票账户专门设立，无需缴纳额外的费用。

2. 修改个人所得税报税列表。报税时需在“报税种类”栏目中选择“个人所得税专票账户”类型。

3. 设置专票账户余额。登录用户中心，进入“设置”页面，点击“个人所得税专票账户”，输入个人所得税账户余额。

4. 报税时将超出专票账户余额部分转至个人所得税专票账户。如交易发生在个人所得税账户以外，系统将自动从个人所得税账户转至专票账户。

5. 缴纳金额超限的部分。缴纳金额超过专票账户余额时，系统将会提示您缴纳金额超限。请您根据提示转账到专票账户。

6. 提交普通报税。提交普通报税时系统将自动从专票账户转至您的个人所得税账户。

7. 查看个人所得税专票账户历史。登录用户中心，点击“我的账户”，找到“个人所得税专票账户”，可查看、提现、账单等。
