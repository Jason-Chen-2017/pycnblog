                 

# 1.背景介绍



什么是量化投资？作为一个从事量化分析和研究工作的人，我首先需要了解量化投资的定义、特点、作用以及未来的发展方向。

## 1.1 什么是量化投资

量化投资（Quantitative investment）或称量化交易，是指利用各种算法技术和模式对市场价格进行预测、分析、预测，并进行有效的交易。其基本方法是在历史数据中发现规律性，构建模型对未来趋势进行预测和研判，然后通过计算自动化执行交易策略来实现盈利。

量化投资包含以下三个层次：

1. 技术层面：主要包括基于机器学习、人工神经网络等技术的深度学习模型、预测分析技术等。

2. 理论层面：主要包括统计学、金融学、控制论、经济学、博弈论等理论，探讨市场的内在规则、行为与风险之间的关系，提出关于市场模型的假设、建立动态系统模型、抽象化经济行为特征、回归分析模型等理论。

3. 应用层面：主要包括国际、跨境期货品种的套利、股票的选股、债券的定价、基金的运作、企业的投资决策等实际应用。

目前，量化投资最火爆的领域就是金融衍生品，如商品期权、国债期权、股指期权等。量化投资通过计算机仿真技术、大数据分析技术、智能投顾系统、风险管理平台等将传统的非计算机视角的研究方法扩展到计算机技术领域。

## 1.2 量化投资的特点

1. 数据驱动：量化投资直接面对的数据是市场中的大量信息，采用历史数据的机器学习模型能够帮助个人投资者更好地理解市场动向。

2. 模型驱动：量化交易的关键在于模型构建能力。所谓的模型是指通过某些分析手段、建立某种计算模型来预测、分析和模拟市场的走势，它可以用来评估个别公司、个别投资标的的投资回报、风险评估、仓位管理等。

3. 高频率高带宽：目前，整个行业都处于高频率高带宽的变化过程中，量化投资的应用模式也会随之发生相应的变化。比如股票市场每天都会产生巨大的变化，而一般的交易所只能在交易日进行下单交易。因此，量化交易平台往往需要同时兼顾实时性和高频率低延迟，确保足够的实时响应速度。

4. 自动化操作：量化投资的交易策略往往是由程序自动生成，不需要依赖人工参与，降低了交易成本。

5. 稳健性高：量化交易存在着一定的不确定性，但在一定时间周期内，其收益与波动幅度应该都不会太差。而且，量化交易所遵循的“透明”及“可追溯”的原则，不会被政府部门追踪、监控。

6. 灵活性强：量化交易可以根据用户的不同需求，选择不同的策略进行操作，满足用户多样化的交易需求。

## 1.3 量化投资的作用

1. 智能化交易：量化交易能够根据市场情况做出正确的交易决策，避免了盲目跟风，提高了交易效率。

2. 精准化管理：量化交易的模型可以提供有效的管理工具，帮助投资者达到精准化的持仓管理水平。

3. 大众化应用：量化投资已经成为大众化应用的领域，有很多金融产品和服务的研发商、销售商希望能够借助量化技术来解决投资问题。

4. 定制化服务：量化交易的服务也可以根据客户的需求，提供定制化的量化服务，满足用户个性化的交易需求。

5. 科技进步：由于量化交易的理念在电子游戏行业得到应用，在自动驾驶方面取得了先发优势，科技人员也逐渐把目光移向量化交易领域。

# 2.核心概念与联系

在正式进入介绍具体的量化投资技术之前，我们需要对量化投资的相关术语和概念进行必要的了解。以下内容将介绍一些核心的术语与概念。

## 2.1 TPS (Ticks Per Second)

TPS是指每秒钟发出的交易次数，通常TPS的范围在5-20万之间。

## 2.2 K线图（K Line Chart）

K线图是一种常用的图表，用于呈现股价走势，分为五日、十日、月线和周线。如下图所示：

- 五日K线（5 Day K-Line，简称K线）：表示过去5天每天的开盘价、收盘价、最高价和最低价，一般用5根竖线表示；
- 十日K线（10 Day K-Line）：表示过去10天每天的开盘价、收盘价、最高价和最低价，一般用10根竖线表示；
- 月K线（Month K-Line）：表示过去一个月每天的开盘价、收盘价、最高价和最低价，一般用20根竖线表示；
- 周K线（Week K-Line）：表示过去一周每天的开盘价、收盘价、最高价和最低价，一般用5根横线表示；

以上就是四种K线图。可以看到，K线图可以很直观地反映出市场的走势，而且各种K线图代表着不同时间长度的市场走势，具有广泛的适应性。

## 2.3 MACD

MACD（Moving Average Convergence Divergence，移动平均convergence/divergence线），是由<NAME>在1979年提出的统计技术指标。它与长短期移动平均线（MA5、MA10、MA20），加上连续的移动平均线（CMF）比较，创造了一个新的买卖讯号信号。MACD指标对一段时间内的股价波动起到一定的作用，所以经常用以观察股价趋势。

如下图所示：


- EMA12（Exponential Moving Average，指数移动平均线）：用最近的12天的数据计算得来，是一个平滑的数据，用来判断价格趋势。EMA(n) = （前一日EMA(n-1)×11/13 + 今日的收盘价×2/13）。
- DIF（DIfference）= EMA12 - EMA26：是MACD的基础。DIF(n)表示短期EMA与长期EMA的差值，即上升动向。
- DEA（DEviation of Oscillator）= （9×DIF(n-1)-6×DEM1(n-1)+ 3×DEM2(n-1)）/ 3： 是DIF的简单移动平均。DEA(n)表示DIFF的均线，用来抑制噪声。
- MACD（Moving Average Convergence Divergence）= 2 × （DIF(n)-DEA(n))：两倍的DIFF减去DEA，即为MACD柱状图。
- Signal Line = 9 x DEA(n) - 15 x DEM1(n): 是MACD的快线，用来快速确认趋势。Signal Line(n) > MACD(n) 表示该股票处于趋势上升阶段，买入信号。Signal Line(n) < MACD(n) 表示该股票处于趋势下跌阶段，卖出信号。

上面是MACD的主要计算过程。通过MACD可以分析出股票的走势方向、买卖时机。

## 2.4 Bollinger Bands

布林带（Bollinger Bands）是由美国纳什维尔·阿克塞姆·希特勒(<NAME>)于1980年提出的。他的意识形态与感觉相当激烈。布林带是一种统计显示股价波动范围的方法，用来判断股价是否偏离正常范围，属于超买超卖指标之一。

它的构造方法是将股价上涨或者下跌幅度与标准差相乘，从而得到宽度。其中标准差代表了波动范围。

布林带由三条线组成，上轨线（Upper Band）、中轨线（Middle Band）和下轨线（Lower Band）。

- 上轨线：上轨线是股价的压力位，距离中轨线的宽度为两倍的标准差，超过上轨线则可能面临着超买现象，需要增强警惕。
- 中轨线：中轨线是股价的震荡位，距离上下轨线的宽度为两个标准差的平均值，即中轨线至上下轨线之间的距离近似为零。
- 下轨线：下轨线是股价的支撑位，距离中轨线的宽度为两倍的标准差，小于下轨线则可能面临着超卖现象，需要增强警惕。

## 2.5 RSI (Relative Strength Index)，相对强弱指数

RSI（Relative Strength Index，相对强弱指数）是通过一段时间内股价涨跌幅的比例，衡量当前股价在波动过程中被人们认为处于的位置，也叫做相对强弱指数，是一个量度股价变动的重要工具。

RSI计算方式：

RSI = 100 – (100/(1+RS))

RS（RSI的相对强弱指数） = Avg Gains / Avg Losses * 100

Avg Gains: 当天收盘价大于昨天收盘价时，与昨天收盘价的较大的绝对值之和除以当天收盘价的绝对值之和。若没有任何上涨，Avg Gains取值为0。

Avg Losses: 当天收盘价小于昨天收盘价时，与昨天收盘价的较小的绝对值之和除以今天收盘价的绝对值之和。若没有任何下跌，Avg Losses取值为0。

Ave RS（RSI相对强弱指数的平均值）：

(Avg Gain 1 + Avg Gain n +... + Avg Gain N) / (N+1) – ((Avg Loss 1 + Avg Loss n +... + Avg Loss N)/(N+1)

N 为交易日数量。

## 2.6 Stochastic Oscillator (KD)

随机指标又称KD指标（Stochastic oscillator）。它是一种 momentum indicator，通过计算股价的随机值，来识别股价的短期移动趋势，其根据 STOCH（%K，%D）曲线来计算。当%K上穿（下穿）50线时，意味着股价短期趋势反转，建议买入。当%K与%D出现交叉时，意味着股价处于下跌通道。当%K与%D平行且%D<30时，表示股价底部寻底阶段。当%K与%D平行且%D>70时，表示股价顶部反弹阶段。当%K与%D平行且%D>30且%D<70时，表示股价上涨通道。

随机指标是一种比较复杂的指标，要比其他指标更为敏锐，并且有各自的优缺点。但是，它也是一种老牌指标，被广泛应用于股票、外汇、贵金属、加密货币等市场。

KD曲线由K曲线（%K）和D曲线(%D)组成。

- %K： 随机指标的计算结果。当K线图上一天的最高价大于等于上方BB（上轨线）时，K线为100%，当K线图上一天的最低价小于等于下方BB（下轨线）时，K线为0%。其计算方法为：K = 100 * (Current Close - LL) / (HH - LL), LL为上一天的最低价，HH为上一天的最高价。
- %D： %D曲线取的是%K曲线在N日内的简单移动平均值。其计算方法为：D = MA(%K,N), N为平滑天数，一般取N=3。
