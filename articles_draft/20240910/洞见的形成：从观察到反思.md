#### 问题1：二分查找算法

**题目：** 实现一个二分查找算法，在有序数组中查找目标元素。

**答案：** 

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

**解析：** 通过观察有序数组的特性，结合中值比较，二分查找能够在对数级别的时间复杂度内找到目标元素。

#### 问题2：最长公共子序列

**题目：** 给定两个字符串，找到它们的最长公共子序列。

**答案：** 

```python
def longest_common_subsequence(str1, str2):
    dp = [[0] * (len(str2) + 1) for _ in range(len(str1) + 1)]
    for i in range(1, len(str1) + 1):
        for j in range(1, len(str2) + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    return dp[-1][-1]
```

**解析：** 通过观察字符串的字符匹配，使用动态规划方法计算最长公共子序列。
