# 2.核心概念与联系

## 2.1 图像分割与语义分割的区别

图像分割与语义分割的主要区别在于它们的目标。图像分割的目标是将图像划分为多个区域，每个区域代表不同的物体或场景。例如，在一个街景图像中，我们可能需要将图像划分为建筑物、路面、车辆等区域。而语义分割的目标是将图像划分为不同的语义类别，如人、植物、建筑物等。例如，在一个人群场景中，我们可能需要将图像划分为人、背景等类别。

## 2.2 图像分割与语义分割的联系

图像分割与语义分割是密切相关的。语义分割可以看作是图像分割的一个特殊类型，它将图像划分为不同的语义类别。因此，在实际应用中，我们可以将语义分割视为图像分割的一种特殊情况。
