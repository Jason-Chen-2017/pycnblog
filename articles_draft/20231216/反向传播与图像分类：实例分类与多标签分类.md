### 3.1 反向传播算法原理

反向传播算法是一种常用的神经网络训练方法，它的核心思想是通过计算损失函数的梯度，从而更新神经网络中的权重和偏置。反向传播算法的主要步骤如下：

1. 前向传播：通过输入数据进行前向传播计算，得到输出结果。
2. 损失函数计算：根据输出结果计算损失函数。
3. 梯度计算：通过计算损失函数的梯度，得到权重和偏置的梯度。
4. 权重和偏置更新：根据梯度信息更新权重和偏置。

def loss(x, y, theta):
    return np.sum((x @ theta - y) ** 2)

# 定义梯度
def grad(x, y, theta):
    return x.T @ (x @ theta - y)

# 训练模型
