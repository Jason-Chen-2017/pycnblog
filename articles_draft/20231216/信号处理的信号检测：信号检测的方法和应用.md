### 信号预处理

信号预处理主要包括滤波、增益、平滑等处理，以去除噪声、增强特征、减少干扰。具体的代码实例如下：

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.signal import butter, lfilter

# 信号生成
t = np.linspace(0, 1, 100)
x = np.sin(2 * np.pi * 5 * t) + np.random.randn(100) * 0.1

# 滤波
b, a = butter(2, 0.1, 'low')
y = lfilter(b, a, x)

# 增益
y = y * 2

# 平滑
y = np.convolve(y, np.ones(5) / 5, 'same')

# 绘制信号
plt.plot(t, x, label='原信号')
plt.plot(t, y, label='预处理后信号')
plt.legend()
plt.show()
```

### 信号分析
