    def __init__(self):

        self.facts = []

    def add_rule(self, rule):
        self.rules.append(rule)

    def add_fact(self, fact):

            if rule.is_applicable(self.facts):
                rule.execute(self.facts)

1. 规则引擎的优缺点是什么？

- 规则引擎具有高度的可扩展性，可以处理大量的规则和事实。
