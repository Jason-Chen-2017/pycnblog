## 4.1 基于用户的协同过滤（User-based Collaborative Filtering）

基于用户的协同过滤算法的核心思想是，根据当前用户的兴趣，找出与当前用户相似的其他用户，并根据这些用户的喜好推荐产品。具体的实现步骤如下：

1. 数据预处理：将用户和产品的基本信息和历史行为数据进行清洗和转换。
2. 用户相似度计算：根据用户的历史行为计算用户之间的相似度。
3. 用户相似度排序：根据用户相似度对用户进行排序，选择与当前用户最相似的用户。
4. 产品推荐：根据选择出的用户的喜好推荐产品。

以下是基于用户的协同过滤算法的具体代码实例：

```python
import numpy as np
from scipy.spatial.distance import cosine

# 数据预处理
# 假设数据已经被预处理并存储在变量中
# ratings = ...

# 用户相似度计算
def calculate_similarity(ratings):
    user_ratings = ratings.todense()
    user_ratings = user_ratings.astype(float)
    user_similarity = 1 - cosine(user_ratings)
    return user_similarity

# 用户相似度排序
def sort_similarity(similarity):
    sorted_indices = np.argsort(similarity, axis=0)
    return sorted_indices

# 产品推荐
def recommend_products(ratings, user_id, num_recommendations):
    user_similarity = calculate_similarity(ratings)
    sorted_indices = sort_similarity(user_similarity)
    user_index = ratings.index[user_id]
    recommended_indices = sorted_indices[user_index][user_index:user_index+num_recommendations]
    recommended_products = ratings.index[recommended_indices]
    return recommended_products

# 测试
user_id = 1
num_recommendations = 5
recommended_products = recommend_products(ratings, user_id, num_recommendations)
print("推荐产品：", recommended_products)
```
