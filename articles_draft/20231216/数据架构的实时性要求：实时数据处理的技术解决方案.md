# 2.核心概念与联系
在讨论实时数据处理之前，我们需要了解一些核心概念。

## 2.1 实时数据
实时数据是指在数据产生后立即进行处理的数据。与批处理数据相对，批处理数据是指在数据产生后一段时间后进行处理的数据。实时数据可以进一步分为近实时数据和远程实时数据。近实时数据是指数据在产生后几秒钟或几毫秒内进行处理的数据，例如实时聊天、实时监控等。远程实时数据是指数据在产生后几分钟或几小时内进行处理的数据，例如实时统计、实时报警等。

## 2.2 数据流
数据流是指数据在产生后不断流动的过程。数据流可以是结构化的，例如文本、图像等；也可以是非结构化的，例如日志、传感器数据等。数据流可以通过各种传输方式传输，例如TCP/IP、UDP等。数据流的处理需要考虑到数据的速度、大小、格式等因素。

## 2.3 数据处理
数据处理是指对数据进行转换、分析、存储等操作的过程。数据处理可以是批处理的，例如批量导入、批量统计等；也可以是实时的，例如实时分析、实时存储等。数据处理需要考虑到数据的类型、结构、质量等因素。
