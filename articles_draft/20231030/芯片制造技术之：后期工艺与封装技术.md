# 1.背景介绍


光刻胶（PCB）是一个重要的芯片制造组件，它在集成电路中的作用是导电、隔离信号、起到保护电路板的作用。PCB的制作通常需要长时间的精心设计、多次测试、严格的生产工艺标准和流程。而且每一个PCB制造企业都面临着独特的制造流程、经济条件、国际竞争力等诸多限制。因此，要想做好PCB制造也是一件非常复杂的事情。
如何更好地把PCB制造做好，是近年来电子化工行业的一项重要需求。例如，苹果公司就使用了PCB后期处理工艺来保证其产品质量。另外，台积电公司也采用了在PCB上使用的微纳层处理技术来提高PCB的性能。华为公司则使用电镀技术来实现PCB的电化学性能和防护性能。在这种背景下，本文将试图通过对PCB后期工艺的研究、探索和总结，阐述PCB封装技术的原理及其应用。
PCB封装技术一般分为两个阶段：PCB封装前期（PCB Place & Route，简称PPR）与PCB封装后期（PCB Encapsulation and Test）。PPR是将预制好的封装材料如焊盘或硅基喷涂，嵌入pcb孔中进行铺装。然后利用自动化工具完成对PCB器件引脚之间的互连。而PCB封装后的功能测试是用来检测pcb上的元器件是否能正常工作。

# 2.核心概念与联系
## 2.1 PCB封装概述
PCB封装指的是将PCB上所有的元器件和连接件组装成一个大的电路板。主要包括两种类型：矩形封装和六边形封装。按照外形大小划分，矩形封装又可分为小矩形与大矩形。而六边形封装又可以分为圆角六边形与直角六边形。如下图所示：

### 2.1.1 PCB封装前期
PCB封装前期的目标是将所有元器件（电阻、电容、印刷电路板材料等）装进设计好的板上，同时连接器件之间的引脚。PCB封装前期主要有三个过程：

1. PCB布线 - 在不改变PCB基板结构的情况下，用普通的钢丝、铜丝或者玻璃笔电将元器件固定在PCB上。在正常的打样、组装过程中，需要注意防止击穿、磨损、晒伤等问题。
2. SMD焊盘放电 - 将电容器、电阻等金属件按照要求焊在SMD焊盘上，能够有效降低对PCB基板表面的影响，同时消除元器件间的干扰。
3. 通过场效应堆栈提升电阻抗 - 使用场效应堆栈将电阻间距拉近，减少反射，增强真空效应，确保高效率地传输电流。

### 2.1.2 PCB封装后期
PCB封装后期的目标是把铺好的PCB外包装起来，使得PCB具有良好的隔离效果，并且测试PCB上的元器件的运行状态。主要有以下几个方面：

1. 测试电压 - 测试电压用于校准元件以及检测PCB是否处于工作状态。通常情况下，测试电压为直流2.5V~3.3V。
2. 感应电极 - 感应电极（RHE）用于将测试电压信号传送到电路板上，测量元器件对电流的响应。
3. 晶体挡板 - 晶体挡板用于隔离电路板与外部环境。通常来说，需要两种类型的挡板：一种是利用焊盘粘贴在设计好的PCB上，另一种是加厚的玻璃棉棒。
4. 模拟检测仪 - 模拟检测仪用于检查电路板上的信号流，并记录信号强度与电流。
5. 稳态测试 - 稳态测试用于判断PCB封装是否产生开关电路以及电路功能是否正常。
6. PCB焊接 - 最后一步是将焊好的外壳带回生产环节，等待下一部PCB的制造。

### 2.1.3 PCB封装前期和后期区别
PCB封装技术可以分为PCB封装前期（PPR）和PCB封装后期三步，但是这些步不是一蹴而就的。开发设计人员必须选择适合自己的封装方式，具体做法是在PCB设计的最初就决定使用何种封装材料、外壳材料、封装件类型，以及后续的测试、封装和铺装方法。因此，PCB封装技术的确也存在“先天不足”的情况。

PCB封装前期的目的主要是将所有元器件和连接件装进设计好的板上，通过铺好的外壳来连接器件之间的引脚。由于耗时长，效率不高，因此PCB制造企业都会有专门团队负责PCB封装前期的工作。相反，PCB封装后期的目标是把铺好的PCB外包装起来，使得PCB具有良好的隔离效果，并且测试PCB上的元器件的运行状态。需要设计人员根据实际的业务需求和测试结果，选择相应的测试电压、感应电极、挡板等设备，将PCB连接到电源、接口电缆或其他组件上进行测试。这样就可以最大限度地保证PCB封装的成功性。

## 2.2 封装材料
### 2.2.1 印刷电路板材料
由于尺寸庞大，手感丰富，易于刻蚀，印刷电路板材料是目前PCB上用的最多的材料。但由于密度偏高，在使用过程中容易出现缺陷，所以还需要一些特殊措施来解决这个问题。常见的印刷电路板材料有：

1. LED液晶电路板材料 - 以LED电子技术显示图像，能够显示的颜色范围比普通电子技术要宽。LED液晶电路板材料一般是由长的EMC层和薄薄的电子层组成。因此，在插卡之前需要对EMC层进行擦除，使其成为电子层。擦除了之后，才能安装PCB。
2. TCR化学材料 - 用TCR化学材料可以达到较高的耐氧、耐腐蚀性，而且还不会腐蚀任何原料。可以用含有TCR化学物质的卷板将印刷电路板紧紧的贴在一起，也可以用双层TCR化学纸张把板子包裹起来。
3. GALVAD防火材料 - GALVAD防火材料可以很好的防止PCB板件对环境的侵害。GALVAD材料是由原子级的硅原子凝结而成，可以无缝融入电路板板模，保护板件不受到任何损坏。
4. 绝缘材料 - 可以加入电阻、电容等导体，改善板件的绝缘性能，提高其抗污染能力。

### 2.2.2 阻玻璃
用于PCB的封闭材料可以选择阻性玻璃、超薄膜材料或MDF材料等。但是对于PCB封闭材料，还有一些特殊注意事项。比如，为了防止PCB在安装的时候发生锈蚀，可以使用护目镜将阻性玻璃放在高温环境下，或在安装时仔细扎牢板体。在安装时还应避免使用油性或硫酸含量较高的材料，否则可能导致锈蚀。最后，还应尽量选用热敏电阻、绝缘层，防止PCB在摔落时短路。

### 2.2.3 塑料
为了使PCB更耐久、耐腐蚀、耐磨损、耐高温，可以选择厚重的塑料，甚至更厚的膜状塑料。但必须注意，选择厚的塑料可能会导致焊接异常，且防水性能差。需要选择比较耐用、耐磨损的材料。

### 2.2.4 纺织合金
纺织合金可以应用于PCB的封装材料。纺织合金由聚乙烯、聚氯乙烯、丙烯、异形聚氯乙烯、聚丙烯、硅化合物等各种材料组成。可以选择常规纺织品混凝土、导电纺织品、芯片用纺织品、电容器用纺织品等。

### 2.2.5 硅
PCB的封闭材料除了使用阻性玻璃或MDF材料外，还可以使用硅作为封闭材料。硅可以被冷藏、浸泡、铣刨、烧制或熔融。但由于PCB的尺寸特别大，因此需要注意防止过度熔化，使PCB板件变形。

## 2.3 封装材料的挤压处理
在PCB的设计过程中，每当布线或元器件的位置调整，PCB就会出现挤压，而这种挤压会引入额外的磨损、气泡等杂质，降低PCB的封装性。因此，为了尽可能的避免挤压，可以采用不同类型的封装材料，以及相应的挤压处理方法。常见的封装材料有：

1. 电镀硅 - 电镀硅是一类经过电镀处理的纤维材料，具有耐电压、耐磨损、耐腐蚀、耐高温、耐挤压、耐风险、防腐蚀、防静电、易清洁等特性。
2. 铅芯材料 - 是一种高级的封闭材料，它具有坚固、透明、轻盈、耐磨损、耐磕碾、防腐蚀、防静电、易清洁等优点。常用的铅芯材料有炭化铅芯材料、铅锌材料、二氧化铅聚苯乙烯管材料等。
3. 混凝土 - 混凝土具有耐腐蚀性、耐氧化剂、防腐蚀、防静电等特性。如果需要制作大面积的PCB，建议用聚乙烯或聚氯乙烯混凝土。
4. 棱纹聚合剂 - 聚合剂可以在金属材料内部产生棱纹，从而增加金属材料的抗漏化和粘性。常用的棱纹聚合剂有石墨烯聚合剂、重氮聚合剂、六氟化碳聚合剂、环己烯聚合剂、柳橙烯聚合剂、二氧化碳聚合剂等。
5. 非金属封闭层 - 可以在PCB上添加非金属材料，如陶瓷、珐琅、钛矿、银锡等，来增强PCB的隔离性能。不过，非金属封闭层不能完全隔离PCB，仍然需要使用电镀硅材料或氧化剂来完善封闭性能。

## 2.4 封装算法
PCB封装的算法有两种，分别为极大连通子集算法（Maximal Connected Subgraph Algorithm，简称MCSA）和最小面积覆盖算法（Minimum Area Covering Algorithm，简称MACA）。前者通过找出最大的连接子集来得到连接的电路板，后者通过找到最小面积覆盖来获得连接的电路板。

### 2.4.1 极大连通子集算法
MCSA算法的基本思想是选择连接元件的组合，使得该组合内的所有元件之间都有一条路径可达，且每条路径上使用的交联材料数量最少。这种算法的效率很高，且不需要搜索所有可能的排列组合，只需计算出单个元件间的路径长度即可。但该算法有如下缺点：

1. 算法运行时间较长，无法适用于大型PCB。
2. 不一定能找到一个最佳的封装。
3. 对PCB布局有较大的依赖性。

### 2.4.2 最小面积覆盖算法
MACA算法的基本思想是通过确定每个元件的位置，找出整个PCB的最小面积覆盖，即使各元件之间没有路径可达，也能将它们与周围元件相连。MACA算法的速度较快，适用于大型PCB。

## 2.5 封装设备
### 2.5.1 印刷电路板封闭制作工艺
制作封闭PCB的过程一般包括以下四个步骤：

1. 定位 - 选择电路板基板上的两个粗略的间距区域，用导线将两侧隔离。
2. 拼接 - 用拼接设备将隔离后的基板拼接在一起，形成完整的板块。
3. 插卡 - 将导线固定在卡座上，连接在一起的导线固定在一起。
4. 美化 - 简单修饰一下，比如补上间距，调整孔洞大小，让导线顺滑流动。

### 2.5.2 电镀设备
电镀是利用电流、磁场、电阻、放电过程来改变物体的性质。电镀过程将被电子设备所制成的PCB纤维材料经过加工，使其具有良好的硬度和耐腐蚀性。电镀的原理就是利用电流来推动电子运动。常用的电镀设备有电钻、电锤、电钳、电雕、电烙铁、电镐等。

电镀设备的使用方法：

1. 首先准备好相应的电镀设备，电镀设备可以选择机械，也可以选择手动。
2. 在准备好的设备上加上镀层，将准备好的纤维纸转移到导线的旁边。
3. 将导线通过电镀设备，使电流通过纤维，形成板材。

### 2.5.3 镀层材料
镀层材料是指在电镀过程中，电子与电流之间的平衡，从而产生各种不同的工艺效果。常用的镀层材料有铅锌材料、炭化铅芯材料、丁香膏材料、绝缘材料、GALVAD材料、绝缘釉、GaN箔片、丙烯凝胶等。

### 2.5.4 氧化剂
在PCB封装的后期阶段，还可以通过将封装后的PCB放置在缓冲溶液中，加入适量的氧化剂，加速PCB的组装。氧化剂可以让封装后的PCB具有较好的隔离性和防护性能。常用的氧化剂有石墨烯氧化剂、重氮氧化剂、环己烯氧化剂、二氧化碳氧化剂、聚氯乙烯氧化剂等。

### 2.5.5 防火材料
对于需要封闭的PCB来说，防火材料非常重要。目前，PCB制造领域已经推出了各种PCB封闭材料，其中包括硅基阻隔材料、碳阻隔材料、GALVAD材料等。防火材料能够使PCB上多个元件之间的联系较弱，因此也起到了减小PCB打开风险的作用。
