# 2.核心概念与联系
## 2.1 自助式软件咨询公司(Self-service software consultancy)
顾名思义，自助式软件咨询公司就是由客户自主选择、购买、学习、实施的一站式服务型企业。它通过线上、线下、远程的方式，向客户提供全方位的技术支持和服务。其特点是用户无需购买授权、不受专利保护，客户可以在线上学习相关技术，解决实际问题，无缝连接到本地服务人员。当然，自助式软件咨询公司除了为客户提供技术支持外，还可以提供运营服务、法律咨询、财务规划、HR服务、培训服务、销售渠道拓展等更多服务。
## 2.2 投资建议类别
自助式软件咨询公司的投资建议类别主要分为以下三种:

1. 技术增长类建议
2. 市场营销类建议
3. 管理咨询类建议

### 技术增长类建议
技术增长类建议主要包括基础设施建设、项目方案制定、技术培训、公司内部工具建设等。

例如，客户有兴趣开发一款云计算平台应用。自助式软件咨询公司可以协助完成云计算基础设施建设、购买服务器硬件、配置网络环境、部署软件、进行性能测试等工作，协助客户提升技术水平。

### 市场营销类建议
市场营销类建议主要涉及到客户关系管理、营销策略制定、销售渠道开拓、品牌宣传推广、网络营销活动等。

例如，客户想利用微信支付服务收款。自助式软件咨询公司可以提供微信支付接口文档、SDK下载、设置及测试等一系列教程，帮助客户快速接入微信支付服务，实现微信支付功能。

### 管理咨询类建议
管理咨询类建议主要包含企业文化建设、管理层培训、企业风险管控、组织结构调整、职权分工管理、人力资源规划、薪酬福利管理等。

例如，客户担心自己没有足够的人才储备，想寻找一份IT部门的管理经验。自助式软件咨询公司可以开设专门的管理课堂，提供企业文化建设、管理经验分享、职权分工管理、薪酬福利管理等优质服务。

## 2.3 合作社类
自助式软件咨询公司还可以通过建立起合作社的方式，帮助客户找到志同道合的伙伴，共同构建自己的技术团队。

合作社类包括如下几种:

1. 免费试用期
2. 付费试用期
3. 服务提供商
4. 供应商

### 免费试用期
免费试用期意味着客户只需要向公司提交订单即可获得免费试用期。免费试用期一般持续3个月，在此期间，客户可以体验产品功能、使用流程、访问技术文档，以及完成简单的技术培训。如果客户觉得产品符合要求，并对公司提供的服务满意，就可以购买付费会员。

### 付费试用期
付费试用期是在免费试用期结束后，客户须支付一定金额才能享受完整的服务。付费试用期一般持续1年，客户可以根据自身需求购买服务。付费会员可以享受全套的服务，并按月或按年结算。

### 服务提供商
服务提供商是指向自助式软件咨询公司提供IT服务的第三方机构，如公司的合作伙伴、服务代理商、设备厂商等。服务提供商可在自助式软件咨询公司举办培训、技术支持等服务，降低客户购买自助式软件咨询公司服务的成本，提高客户满意度。

### 供应商
供应商是指自助式软件咨询公司委托的第三方供应商，如服务器租赁、数据中心、IDC 工程等。供应商可提供自助式软件咨询公司需要的计算机设备及相关配套服务。

## 3.2 投资建议

### 技术增长类建议

#### 基础设施建设
基础设施建设可以为软件开发提供非常重要的资源。这些资源包括服务器、存储、网络带宽、IDC机房、域名注册等。如果没有足够的基础设施，软件开发的效率就会显著下降。所以，我们应该优先推荐客户购买云计算平台服务，他们可以节省大量的硬件投入，让软件开发者不需要购买服务器，从而降低开发成本。

另外，软件开发人员还需要对软件开发所用的编程语言有一定了解，需要掌握基础的计算机知识。比如，要熟练掌握Python、Java、C++等语言，能够编写软件应用程序。如果客户对这些语言不是很熟悉，可能导致学习曲线陡峭。

#### 项目方案制定
项目方案制定阶段最耗时的环节，也是最容易出现错误的环节。尤其是在开始开发软件时，我们需要对功能、性能、可用性、安全、可靠性、可扩展性等多个维度进行评估，设计出符合公司业务目标的软件系统。项目方案制定的重要作用在于保证项目的成功，并有效地分配项目资源，避免资源浪费。

#### 技术培训
技术培训对于任何一个技术人员都是不可或缺的。自助式软件咨询公司可以组织针对不同技术领域的培训班，帮助软件开发者掌握各项技术。不过，很多初级开发人员学习新技术往往比较困难，因此自助式软件咨询公司需要提供一些简单易懂的视频教程，以及技术论坛，来帮助客户快速学习。

#### 公司内部工具建设
对于软件开发者来说，开发工具也是必不可少的。公司内部工具包括开发工具、项目管理工具、代码库、代码审查工具、CI/CD自动化工具等。这些工具极大的提高了软件开发的效率，并减少了重复工作，有利于软件开发的规范化和可追溯性。

#### IT部门管理经验
IT部门管理经验不仅对软件开发人员的工作生活影响深远，同时也对软件开发者的个人发展、创业梦想产生重大影响。所以，对于想加入或开展软件开发工作的客户，我们都会帮助他们搜集相关信息，为他们提供管理经验。

职权分工管理是企业文化建设的一个重要组成部分。企业需要建立起一个合理的工作流程和制度，来规范公司内部各职能人员的职责和权力，确保工作效率，防止各种工作冲突，保持企业的运行稳定。自助式软件咨询公司可以提供专门的管理课堂，提供企业文化建设、管理经验分享、职权分工管理、薪酬福利管理等优质服务。

## 4.1 登录验证
```python
import re
def login_verify(username, password):
    if len(password)<8 or not re.match("^[a-zA-Z0-9]+$", username):
        return False # 密码长度不能少于8位并且用户名只能包含数字字母
    else:
        return True
```
这个函数是用来验证用户名和密码是否符合规则的。它的输入参数分别是用户名和密码。首先判断密码长度是否小于8位，并且用户名只能包含数字字母。如果条件不满足，则返回False；否则，返回True。

## 4.2 消息通知
```python
from datetime import timedelta
import smtplib
import ssl

def send_notification():
    sender_email = "sender@example.com"
    receiver_email = "receiver@example.com"
    password = "password"

    message = """\
    Subject: Hi there

    This message is sent from Python."""

    try:
        context = ssl.create_default_context()
        with smtplib.SMTP_SSL("smtp.gmail.com", 465, context=context) as server:
            server.login(sender_email, password)
            server.sendmail(sender_email, receiver_email, message)

        print("Notification Sent Successfully")
    except Exception as e:
        print("Error Occurred:", str(e))

if __name__ == "__main__":
    while True:
        now = datetime.datetime.now().time()
        start_time = datetime.time(hour=7)   # 早上7点发送提醒
        end_time = datetime.time(hour=19)    # 下午6点发送提醒
        delta_start = (start_time - now).total_seconds() / 60
        delta_end = (end_time - now).total_seconds() / 60

        if delta_start >= 0 and delta_end <= 0:
            print("Sending Notification...")
            send_notification()
            time.sleep(timedelta(days=1).total_seconds())   # 每天提醒一次
        elif delta_start > 0 and delta_end < 0:
            time.sleep(delta_start * 60 + 30)          # 提前30秒再次尝试
        else:
            pass                                       # 不需要提醒
```
这个函数是用来给指定邮箱发送消息通知的。它的输入参数为空。首先是收信人和发信人邮箱地址，以及发信人的邮件服务密码。然后，准备一条包含主题和正文的邮件。这里的主题是“Hi there”，正文是默认的文字。接着创建了一个ssl上下文，用来加密发送邮件，以防止中间人攻击。然后，利用smtplib库连接到Gmail的SMTP服务器，并登陆到自己的账号。最后，利用server.sendmail()函数将邮件发送给指定的收信人。

注意，在函数内部，还添加了一段时间延迟的代码，每隔一天检查一下当前的时间，确定是否发送邮件。
