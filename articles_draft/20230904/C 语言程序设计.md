# 1.简介
  


C 是一种通用、静态的编程语言，被广泛应用于开发各种系统软件、嵌入式软件、驱动程序等。它最初由贝尔实验室的丹尼斯·里奇和丹尼斯·沃霍尔于 1972 年联合创立，并于 1983 年推出第 2 版。该版本于 1988 年作为 ISO/IEC 标准释出。

C 具有高效率的特点，在资源受限的系统中可以用于开发系统软件、驱动程序、网络协议栈等，也可以用来编写应用程序，例如编写一个 Linux 命令行工具、编写一个简单的图形用户界面（GUI）程序等。

C 的特点还有其强大的运算能力，使得它可以在系统层面上实现高性能的计算任务，比如矩阵运算、图像处理、生物信息分析等。同时，C 提供了丰富的数据结构、控制流机制及函数库，可以帮助开发人员快速构建复杂的软件。因此，无论是在嵌入式系统还是大型服务器集群上，都可以选择 C 作为开发语言。

本系列文章将以实践为主线，逐步向读者介绍 C 语言的基础知识和典型应用场景。每一章节结束时，都会提出一些练习题，让读者熟悉所学的内容。



# 2.基本概念术语说明
## 2.1 编译与链接
**编译器**：通过源代码生成目标代码的文件，称为编译器。

**源文件**：存储着可执行或库函数的 C 源代码，也叫做源程序。扩展名通常为 `.c`。

**目标文件**：编译器把源文件转换成目标文件的过程称为目标代码生成（Code Generation）。目标文件是一个二进制文件，包含机器指令和数据。

**预处理器**：把宏定义替换成实际值的程序。一般用于条件编译、头文件包含和依赖关系管理等功能。

**编译选项**：可以影响编译过程的设置，如警告级别、优化等。

**链接器**：作用是将几个目标文件合并成一个可执行文件或库文件。链接器负责解决不同目标文件间的依赖关系、符号引用和地址分配等问题。

**库**：存放已经编译好的可重用代码的集合，可以通过 #include 来引用这些代码。库提供了特定功能的代码，例如 printf() 函数就是存放在 libstdio.a 库中的。

**可执行文件**：在运行时被加载到内存并执行的二进制文件。它包含了最终的可执行代码以及相关的库依赖项。它的扩展名通常为 `.exe` 或 `.out`。

## 2.2 数据类型
C 语言支持八种数据类型：整型、浮点型、逻辑型、字符型、指针型、数组型和结构体型。

- **整型**：分为短整型（short）、整数型（int）、长整型（long）、长长整型（long long），它们的取值范围和容量都不一样。不同类型的整型可以相互转换。
- **浮点型**：分为单精度浮点型（float）和双精度浮点型（double），它们都是按 IEEE-754 标准进行表示的。
- **逻辑型**：只有两种状态——真（true）或假（false）。它对应的值是 0 和非零。
- **字符型**：表示单个 ASCII 字符。
- **指针型**：用来保存变量地址的变量。
- **数组型**：用来保存固定数量元素的一种数据类型。
- **结构体型**：用来保存多个数据项的聚集结构。

## 2.3 标识符
**标识符**：是指程序中用于命名各种变量、数据结构、函数、文件等元素的一串字符序列。由字母、数字、下划线、美元符号组成，但不能以数字开头。

**关键字**：在 C 中具有特殊意义的词汇。有些关键字不能作为标识符。

```c
auto break case char const continue default do double else enum extern float for goto if int long register return short signed sizeof static struct switch typedef union unsigned void volatile while
``` 

## 2.4 基本语法
C 语言采用的是结构化编程风格，语句以分号结尾，块状结构以花括号 {} 包裹，表达式之间以空格隔开。

**注释**：单行注释以两个斜杠 // 开始，多行注释以 /* */ 开始和结束。

**程序结构**：一个 C 程序由若干函数、全局变量和主函数（main() 函数）组成。每个源文件包含一个或多个函数，源文件中的全局变量声明只能在文件内有效，不能跨文件使用。

**变量声明**：变量声明包括类型说明符和变量名称。类型说明符包括 int、char、float 等关键字，后跟变量名。

**赋值语句**：用 = 号连接左边的变量或表达式和右边的值，赋值语句会改变变量的值。

**表达式**：是一种算术或逻辑操作符及其对应的操作数构成的序列。表达式的值依赖于操作数的值和运算符的性质。

**语句**：可以是表达式，也可以是声明或控制语句。

**控制语句**：包括选择语句、循环语句、跳转语句和宏定义。

```c
if (condition) statement;   // 选择语句
while (condition) statement;  // 循环语句
for (initialization; condition; iteration) statement;   // 初始化表达式、条件表达式、迭代表达式
do {statement} while (condition);    // do-while 循环
switch (expression) {case constant: statement;...;}    // switch 语句
goto label;      // 无条件跳转语句
label:           // 标签
return expression;     // 返回语句
break;        // 退出当前循环
continue;     // 跳过剩余语句，继续执行当前循环
#define macro_name replacement text;    // 宏定义
```

**函数**：C 语言的函数是第一类对象，可以作为参数传入另一个函数，也可以作为返回值返回给调用方。

**程序流程**：程序从 main() 函数开始，首先完成必要的初始化工作，然后按照顺序调用其他函数，最后释放掉不需要的资源并正常退出。

**预编译**：预编译是对源文件进行预处理的过程，主要完成三件事情：词法分析、语法分析和语义分析。

**编译**：编译是把源代码文件经过词法分析、语法分析和语义分析后生成目标代码文件的过程。

**汇编**：汇编是把汇编代码转换成机器代码的过程。

**链接**：链接是把多个目标代码文件组合成为一个可执行文件或者动态库的过程。

**异常处理**：C 支持异常处理机制，当出现运行期错误时，可以通过异常处理机制捕获并处理异常。

**线程**：C 支持线程机制，允许程序同时执行多个子任务，提高程序的并发性和运行效率。

**调试**：程序在开发过程中必然存在一些逻辑上的错误，C 提供了一系列的调试手段帮助程序员定位错误位置并修正错误。
