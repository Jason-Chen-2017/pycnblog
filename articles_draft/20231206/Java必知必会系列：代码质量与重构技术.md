# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释Java代码质量与重构技术的具体操作步骤。

## 4.1代码优化实例

### 4.1.1代码

```java
public class Test {
    public static void main(String[] args) {
        int sum = 0;
        for (int i = 0; i < 100000; i++) {
            sum += i;
        }
        System.out.println(sum);
    }
}
```

### 4.1.2优化后代码

```java
public class Test {
    public static void main(String[] args) {
        int sum = 0;
        for (int i = 0; i < 100000; i++) {
            sum += i;
        }
        System.out.println(sum);
    }
}
```

### 4.1.3解释说明

在这个代码实例中，我们需要计算1到100000之间的和。通过观察可以发现，这个代码是正确的，但是性能并不高。因为每次循环都需要对`sum`进行加法运算，这会导致性能下降。

为了提高性能，我们可以使用代数公式来计算和。和的公式为：`sum = n * (n + 1) / 2`，其中`n`是循环次数。通过使用这个公式，我们可以将循环次数从100000减少到1，从而提高性能。
