## 3.1 排序算法
排序算法是计算机程序设计中的一个基本概念，用于对数据进行排序。常见的排序算法有选择排序、插入排序、冒泡排序等。

### 3.1.1 选择排序
选择排序是一种简单的排序算法，它的核心思想是在每次迭代中选择最小的元素，并将其放入正确的位置。具体操作步骤如下：

1. 从数组中选择第一个元素作为当前最小元素。
2. 从当前元素后面的元素中选择最小的元素，并将其与当前元素交换。
3. 重复步骤2，直到数组排序完成。

数学模型公式：
$$
T(n) = \Theta(n^2)
$$

### 3.1.2 插入排序
插入排序是一种简单的排序算法，它的核心思想是将一个元素插入到已排序的序列中的正确位置。具体操作步骤如下：

1. 从第二个元素开始，将其与前一个元素进行比较。
2. 如果当前元素小于前一个元素，将其插入到前一个元素的正确位置。
3. 重复步骤2，直到数组排序完成。

数学模型公式：
$$
T(n) = \Theta(n^2)
$$

### 3.1.3 冒泡排序
冒泡排序是一种简单的排序算法，它的核心思想是通过多次交换相邻的元素，将最大的元素逐渐冒泡到数组的末尾。具体操作步骤如下：

1. 从第一个元素开始，将其与下一个元素进行比较。
2. 如果当前元素大于下一个元素，将其与下一个元素交换。
3. 重复步骤2，直到数组排序完成。

数学模型公式：
$$
T(n) = \Theta(n^2)
$$

## 3.2 搜索算法
搜索算法是计算机程序设计中的一个基本概念，用于在数据结构中查找特定的元素。常见的搜索算法有线性搜索、二分搜索等。

### 3.2.1 线性搜索
线性搜索是一种简单的搜索算法，它的核心思想是从数组的第一个元素开始，逐个比较每个元素，直到找到目标元素或者遍历完整个数组。具体操作步骤如下：

1. 从数组的第一个元素开始，将其与目标元素进行比较。
2. 如果当前元素等于目标元素，则返回当前元素的下标。
3. 如果当前元素不等于目标元素，则将当前元素的下标加1，并继续比较下一个元素。
4. 重复步骤2和3，直到找到目标元素或者遍历完整个数组。

数学模型公式：
$$
T(n) = \Theta(n)
$$

### 3.2.2 二分搜索
二分搜索是一种高效的搜索算法，它的核心思想是将数组分为两个部分，然后将目标元素与中间元素进行比较，根据比较结果将搜索范围缩小到一个更小的部分。具体操作步骤如下：

1. 将数组分为两个部分，左半部分和右半部分。
2. 将目标元素与中间元素进行比较。
3. 如果目标元素等于中间元素，则返回中间元素的下标。
4. 如果目标元素小于中间元素，则将搜索范围缩小到左半部分。
5. 如果目标元素大于中间元素，则将搜索范围缩小到右半部分。
6. 重复步骤2-5，直到找到目标元素或者搜索范围缩小到空。

数学模型公式：
$$
T(n) = \Theta(\log n)
$$

## 4.1 选择排序实例
```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
print(selection_sort(arr))
```

在这个实例中，我们实现了一个选择排序的函数，它接收一个数组作为输入，并将其排序。具体实现方法如下：

1. 首先，我们定义了一个函数`selection_sort`，它接收一个数组作为输入。
2. 然后，我们获取数组的长度，并使用两个循环来实现选择排序的过程。
3. 在外部循环中，我们从第一个元素开始，将其标记为当前最小元素。
4. 在内部循环中，我们从当前元素后面的元素中选择最小的元素，并将其与当前元素交换。
5. 重复内部循环，直到找到最小的元素。
6. 然后，我们将当前元素与最小元素交换，使得当前元素的位置变为最小元素的位置。
7. 重复外部循环，直到数组排序完成。
8. 最后，我们返回排序后的数组。

## 4.2 插入排序实例
```python
def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr

arr = [5, 2, 8, 1, 9]
print(insertion_sort(arr))
```

在这个实例中，我们实现了一个插入排序的函数，它接收一个数组作为输入，并将其排序。具体实现方法如下：

1. 首先，我们定义了一个函数`insertion_sort`，它接收一个数组作为输入。

4. 在内部循环中，我们将当前元素与前一个元素进行比较，如果当前元素小于前一个元素，则将其与前一个元素交换。
5. 重复内部循环，直到找到正确的位置。

## 4.3 冒泡排序实例
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [5, 2, 8, 1, 9]
print(bubble_sort(arr))
```

在这个实例中，我们实现了一个冒泡排序的函数，它接收一个数组作为输入，并将其排序。具体实现方法如下：

1. 首先，我们定义了一个函数`bubble_sort`，它接收一个数组作为输入。
2. 然后，我们获取数组的长度，并使用两个循环来实现冒泡排序的过程。
3. 在外部循环中，我们从第一个元素开始，将其标记为当前元素。
4. 在内部循环中，我们将当前元素与下一个元素进行比较，如果当前元素大于下一个元素，则将其与下一个元素交换。
5. 重复内部循环，直到找到最大的元素。
