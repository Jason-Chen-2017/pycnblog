# 2.核心概念与联系

在物联网与边缘计算中，有几个核心概念需要我们了解：

3. 数据处理：在物联网与边缘计算中，数据处理是一个重要的环节。数据可以在边缘设备上进行预处理、实时分析、智能决策等功能。

5. 安全性：物联网与边缘计算系统需要考虑安全性问题，如数据加密、身份验证、访问控制等。
