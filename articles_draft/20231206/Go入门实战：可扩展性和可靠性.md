## 3.1 并发原语

### 3.1.1 Goroutine

Goroutine是Go语言中的轻量级线程，它们是Go语言中的用户级线程，由Go运行时创建和管理。Goroutine之间之间是独立的，可以并行执行。

Goroutine的创建和使用步骤如下：

1. 定义一个Goroutine函数，例如：
```go
func myGoroutine() {
    // 执行Goroutine的任务
}
```
2. 使用go关键字创建Goroutine，例如：
```go
go myGoroutine()
```
3. 在主线程中等待所有Goroutine完成执行，例如：
```go
fmt.Scan()
```

### 3.1.2 Channel

Channel是Go语言中的一种通信原语，它可以用于实现Goroutine之间的同步和通信。Channel是一个可以存储和传输数据的数据结构，它可以用于实现Goroutine之间的同步和通信。

Channel的创建和使用步骤如下：

1. 定义一个Channel变量，例如：
```go
ch := make(chan int)
```
2. 使用send操作符发送数据到Channel，例如：
```go
ch <- 10
```
3. 使用recv操作符从Channel中读取数据，例如：
```go
v := <-ch
```

## 3.2 垃圾回收

Go语言提供了自动垃圾回收机制，可以自动回收不再使用的内存，减少内存泄漏的风险。垃圾回收机制的工作原理如下：

1. 当一个变量被创建时，Go语言的运行时会为其分配内存。
2. 当一个变量被赋值时，Go语言的运行时会检查变量是否已经被分配了内存。如果没有，则为其分配内存。
3. 当一个变量被释放时，Go语言的运行时会将其标记为不再使用。
4. 当一个变量的生命周期结束时，Go语言的运行时会回收其分配的内存。

## 4.1 并发示例

### 4.1.1 Goroutine示例

```go
package main

import "fmt"

func myGoroutine() {
    fmt.Println("Hello, World!")
}

func main() {
    go myGoroutine()
    fmt.Scan()
}
```

在上述代码中，我们定义了一个名为myGoroutine的Goroutine函数，它将打印“Hello, World!”。然后，我们使用go关键字创建了一个Goroutine，并在主线程中等待所有Goroutine完成执行。

### 4.1.2 Channel示例

```go
package main

import "fmt"

func myGoroutine(ch chan int) {
    for i := 0; i < 5; i++ {
        ch <- i
    }
}

func main() {
    ch := make(chan int)
    go myGoroutine(ch)

    for i := 0; i < 5; i++ {
        fmt.Println(<-ch)
    }

    fmt.Scan()
}
```

在上述代码中，我们定义了一个名为myGoroutine的Goroutine函数，它将通过Channel发送0到4之间的整数。然后，我们使用go关键字创建了一个Goroutine，并在主线程中使用recv操作符从Channel中读取数据。
