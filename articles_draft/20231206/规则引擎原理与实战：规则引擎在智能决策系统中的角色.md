```python

        "action": "normal_price"
    },
    {
        "condition": "age >= 60",
        "action": "senior_discount"
    }
```

## 4.2 解析规则库

接下来，我们需要解析规则库，以便能够根据规则库中的规则来进行决策。以下是一个简单的解析规则库的示例：

```python
def parse_rules(rules):
    parsed_rules = []
    for rule in rules:
        parsed_rule = {
            "condition": rule["condition"],
            "action": rule["action"]
        }
        parsed_rules.append(parsed_rule)
    return parsed_rules

parsed_rules = parse_rules(rules)
```

## 4.3 匹配规则

根据输入的数据和当前的状态，我们需要匹配规则库中的规则，以便能够找到适用于当前情况的规则。以下是一个简单的匹配规则的示例：

```python

    matched_rules = []
    for rule in parsed_rules:
        if eval(rule["condition"]):
            matched_rules.append(rule)
    return matched_rules

matched_rules = match_rules(parsed_rules, data)
```

## 4.4 执行规则

根据匹配到的规则，我们需要执行规则，以便能够生成决策结果。以下是一个简单的执行规则的示例：

```python
def execute_rules(matched_rules, data):
    results = []
    for rule in matched_rules:
        result = rule["action"](data)
        results.append(result)
    return results

results = execute_rules(matched_rules, data)
```

## 4.5 生成决策结果

根据规则的执行结果，我们需要生成决策结果，以便能够进行决策。以下是一个简单的生成决策结果的示例：
