### 4.2Mockito框架设计的具体代码实例

Mockito框架设计的具体代码实例如下：

```java
import org.mockito.Mockito;

public class MockitoTest {
    public static void main(String[] args) {
        // 创建一个模拟对象
        TestObject testObject = Mockito.mock(TestObject.class);
        // 设置模拟对象的方法返回值
        Mockito.when(testObject.testMethod()).thenReturn("This is a test method.");
        // 调用模拟对象的方法
        String result = testObject.testMethod();
        // 输出结果
        System.out.println(result);
    }
}

public class TestObject {
    public String testMethod() {
        return "This is a test method.";
    }
}
```

在这个代码实例中，我们使用Mockito框架创建了一个模拟对象，并设置了模拟对象的方法返回值。然后，我们调用了模拟对象的方法，并输出了结果。
