```python
# 读取数据源
data1 = pd.read_csv('data1.csv')
data2 = pd.read_csv('data2.csv')

# 合并数据
data = pd.merge(data1, data2, on='key')

# 保存数据
data.to_csv('data.csv', index=False)
```

在这个示例中，我们使用了pandas库来读取和合并数据。首先，我们使用`pd.read_csv`函数读取两个数据源`data1.csv`和`data2.csv`。然后，我们使用`pd.merge`函数将两个数据源合并到一个新的数据框中，并根据`key`列进行合并。最后，我们使用`data.to_csv`函数将合并后的数据保存到`data.csv`文件中。
