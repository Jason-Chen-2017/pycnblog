## 4.1 智能合约代码实例

以下是一个简单的智能合约代码实例：

```solidity
pragma solidity ^0.5.16;

contract SimpleContract {
    address public owner;
    uint256 public totalSupply;

    constructor() public {
        owner = msg.sender;
        totalSupply = 1000000;
    }

    function transfer(address payable recipient, uint256 amount) public payable {
        require(msg.value == amount, "Invalid amount");
        recipient.transfer(amount);
    }
}
```

这个智能合约的详细解释说明如下：

1.`pragma solidity ^0.5.16;`：这是一个Solidity版本声明，表示这个智能合约使用的Solidity版本是0.5.16。

2.`contract SimpleContract {`：这是一个智能合约的声明，名称为`SimpleContract`。

3.`address public owner;`：这是一个公共变量，名称为`owner`，类型为`address`，表示合约的所有者。

4.`uint256 public totalSupply;`：这是一个公共变量，名称为`totalSupply`，类型为`uint256`，表示合约的总供应量。

5.`constructor() public {`：这是一个构造函数，用于初始化合约的状态。

6.`owner = msg.sender;`：这是一个赋值操作，将合约的所有者设置为调用合约的地址。

7.`totalSupply = 1000000;`：这是一个赋值操作，将合约的总供应量设置为1000000。

8.`function transfer(address payable recipient, uint256 amount) public payable {`：这是一个可以接收ETH的函数，名称为`transfer`，参数为`recipient`（接收地址）和`amount`（接收ETH的数量），类型为`public`和`payable`。

9.`require(msg.value == amount, "Invalid amount");`：这是一个要求操作，表示接收ETH的数量必须与传入的数量相等，否则会抛出一个错误。

10.`recipient.transfer(amount);`：这是一个调用接收地址的`transfer`函数，将ETH转移给接收地址。
