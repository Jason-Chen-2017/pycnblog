2.1. 基本概念解释

4.2. 应用实例分析

假设我们的项目中有一个计算器应用，需要计算一个列表的平均值。在没有使用设计模式之前，我们需要先创建一个计算器类，并实现计算平均值的算法。然后，在主应用中调用这个计算器的类，来实现计算功能。

4.3. 核心代码实现

```python
# application.py

class Calculator:
    def __init__(self):
        self.instance = None
    
    def set_calculator(self, calculator):
        self.instance = calculator
    
    def calculate_average(self):
        if not self.instance:
            return None
        return self.instance.average

# calculator.py
```
