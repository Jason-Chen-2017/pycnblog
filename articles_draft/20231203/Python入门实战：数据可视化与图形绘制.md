## 1.6 Python的数据可视化与图形绘制的代码实例

在Python中，数据可视化与图形绘制的代码实例包括：

- 条形图的代码实例：
```python
import matplotlib.pyplot as plt
import pandas as pd

# 导入数据
data = pd.read_csv('data.csv')

# 数据预处理
x = data['x']
y = data['y']

# 图表选择
plt.bar(x, y)

# 图形绘制
plt.xlabel('x')
plt.ylabel('y')
plt.title('条形图')

# 图表美化
plt.grid(True)
plt.legend(loc='best')

# 显示图表
plt.show()
```
- 折线图的代码实例：

- 饼图的代码实例：
```python
import matplotlib.pyplot as plt
import pandas as pd

# 导入数据
data = pd.read_csv('data.csv')

# 数据预处理
labels = data['labels']
sizes = data['sizes']

# 图表选择
plt.pie(sizes, labels=labels, autopct='%1.1f%%')

# 图形绘制
plt.axis('equal')
plt.title('饼图')

# 图表美化
plt.legend(loc='best')

# 显示图表
plt.show()
```
