## 4.1量子纠缠

```python
from qiskit import QuantumCircuit, Aer, transpile
from qiskit.visualization import plot_histogram

# 创建量子电路
qc = QuantumCircuit(2)

# 设置两个量子比特的初始状态
qc.h(0)
qc.cx(0, 1)

# 绘制量子电路

result = simulator.run(qc).result()
```
