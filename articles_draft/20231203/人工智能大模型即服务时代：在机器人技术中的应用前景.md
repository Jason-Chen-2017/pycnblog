### 4.1 代码实例

以下是一个简单的AIaaS技术的代码实例：

```python
import requests

# 设置请求头
headers = {
    'Content-Type': 'application/json'
}

# 设置请求体
data = {
    'text': '你好，我是一个机器人'
}

# 发送请求
response = requests.post('https://api.example.com/ai/aaas', headers=headers, json=data)

# 获取响应结果
result = response.json()

# 输出结果
print(result)
```
