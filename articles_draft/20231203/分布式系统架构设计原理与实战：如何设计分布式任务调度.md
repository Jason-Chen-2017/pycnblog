### 4.1代码实例

我们将通过一个简单的代码实例来详细解释说明分布式任务调度系统的核心概念和算法原理。

```python
import threading
import time

class Task:
    def __init__(self, name, priority, resource_requirement):
        self.name = name
        self.priority = priority
        self.resource_requirement = resource_requirement

class TaskScheduler:
    def __init__(self):
        self.tasks = []

    def add_task(self, task):
        self.tasks.append(task)

    def schedule_tasks(self):
        for task in self.tasks:
            print(f"Scheduling task {task.name} with priority {task.priority} and resource requirement {task.resource_requirement}")
            time.sleep(task.priority)

class TaskExecutor:
    def __init__(self):
        self.tasks = []

    def add_task(self, task):
        self.tasks.append(task)

    def execute_tasks(self):
        for task in self.tasks:
            print(f"Executing task {task.name} with priority {task.priority} and resource requirement {task.resource_requirement}")
            time.sleep(task.priority)

if __name__ == "__main__":
    task_scheduler = TaskScheduler()
    task_scheduler.add_task(Task("Task 1", 1, 1))
    task_scheduler.add_task(Task("Task 2", 2, 2))
    task_scheduler.add_task(Task("Task 3", 3, 3))
    task_scheduler.schedule_tasks()

    task_executor = TaskExecutor()
    task_executor.add_task(Task("Task 1", 1, 1))
    task_executor.add_task(Task("Task 2", 2, 2))
    task_executor.add_task(Task("Task 3", 3, 3))
    task_executor.execute_tasks()
```

在主程序中，我们创建了一个`TaskScheduler`对象，并添加了三个任务。然后，我们调用`schedule_tasks`方法来分配任务。在`schedule_tasks`方法中，我们遍历任务列表，并根据任务的优先级和资源需求来分配任务。

接下来，我们创建了一个`TaskExecutor`对象，并添加了三个任务。然后，我们调用`execute_tasks`方法来执行任务。在`execute_tasks`方法中，我们遍历任务列表，并根据任务的优先级和资源需求来执行任务。
