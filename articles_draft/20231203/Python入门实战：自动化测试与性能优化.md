## 4.2 性能优化的代码实例
```python
import time

def optimize_code(func):
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        print(f'Function {func.__name__} took {end_time - start_time} seconds to execute.')
        return result
    return wrapper

@optimize_code
def slow_function():
    time.sleep(1)
```
在上述代码中，我们使用了Python的decorator来实现代码性能优化。我们定义了一个optimize_code函数，它是一个高阶函数，用于修改函数的执行时间。我们使用@optimize_code装饰器将slow_function函数进行性能优化。
