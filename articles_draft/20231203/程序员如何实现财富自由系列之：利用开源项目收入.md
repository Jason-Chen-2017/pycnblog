```python
class Calculator:
    def __init__(self):
        self.result = 0

    def add(self, num):
        self.result += num
        return self.result

    def subtract(self, num):
        self.result -= num
        return self.result

    def multiply(self, num):
        self.result *= num
        return self.result

    def divide(self, num):
        self.result /= num
        return self.result

if __name__ == '__main__':
    calculator = Calculator()
    print(calculator.add(5))
    print(calculator.subtract(3))
    print(calculator.multiply(2))
    print(calculator.divide(4))
```

## 6.1 如何选择合适的开源协议？

选择合适的开源协议主要需要考虑以下因素：

1. 许可范围：开源协议的许可范围可以包括商业用途、修改后的版本等。你需要选择一个许可范围适合你的项目的开源协议。

2. 版权保护：开源协议的版权保护可以包括著作权、专利等。你需要选择一个版权保护适合你的项目的开源协议。

3. 社区参与：开源协议的社区参与可以包括贡献代码、提供反馈等。你需要选择一个社区参与适合你的项目的开源协议。

4. 法律支持：开源协议的法律支持可以包括法律解释、法律援助等。你需要选择一个法律支持适合你的项目的开源协议。

一些常见的开源协议包括GPL、Apache、BSD等。你可以根据上述因素来选择合适的开源协议。
