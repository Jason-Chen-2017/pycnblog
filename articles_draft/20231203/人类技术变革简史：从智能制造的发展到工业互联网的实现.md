## 4.具体代码实例和详细解释说明

### 4.1 智能制造的代码实例

在智能制造中，我们可以使用Python语言编写代码来实现机器人的运动控制。以下是一个简单的机器人运动控制的代码实例：

```python
import rospy
from geometry_msgs.msg import Twist

class RobotController:
    def __init__(self):
        rospy.init_node('robot_controller', anonymous=True)
        self.publisher = rospy.Publisher('/cmd_vel', Twist, queue_size=10)
        self.rate = rospy.Rate(10) # 10Hz

    def move(self, linear_speed, angular_speed):
        cmd = Twist()
        cmd.linear.x = linear_speed
        cmd.angular.z = angular_speed
        self.publisher.publish(cmd)
        self.rate.sleep()

if __name__ == '__main__':
    try:
        controller = RobotController()
        controller.move(0.5, 0.0) # 前进
        rospy.sleep(2)
        controller.move(0.0, 0.5) # 旋转
        rospy.sleep(2)
        controller.move(0.0, 0.0) # 停止
        rospy.sleep(2)
    except rospy.ROSInterruptException:
        pass
```

### 4.2 工业互联网的代码实例

在工业互联网中，我们可以使用Python语言编写代码来实现设备之间的信息交换。以下是一个简单的设备信息交换的代码实例：

```python
import socket

def send_data(ip, port, data):
    try:
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.connect((ip, port))
        sock.sendall(data)
        sock.close()
    except Exception as e:
        print(e)

def receive_data(ip, port):
    try:
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.bind((ip, port))
        sock.listen(1)
        conn, addr = sock.accept()
        with conn:
            data = conn.recv(1024)
            print(data)
        sock.close()
    except Exception as e:
        print(e)

if __name__ == '__main__':
    send_data('192.168.1.100', 8888, b'Hello, World!') # 发送数据
    receive_data('192.168.1.100', 8888) # 接收数据
```
