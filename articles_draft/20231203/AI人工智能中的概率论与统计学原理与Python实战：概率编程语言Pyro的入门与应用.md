# 2.核心概念与联系
在概率论与统计学中，概率是一个随机事件发生的度量。概率可以用来描述一个事件发生的可能性，也可以用来描述一个事件发生的频率。在人工智能中，概率论与统计学是一种用于处理不确定性和随机性的方法。

概率论与统计学的核心概念包括：

1.随机变量：随机变量是一个随机事件的取值。随机变量可以用来描述一个事件的不确定性。

2.概率分布：概率分布是一个随机变量的取值概率的函数。概率分布可以用来描述一个事件的随机性。

4.方差：方差是一个随机变量的方差。方差可以用来描述一个事件的不确定性。

5.协方差：协方差是两个随机变量的协方差。协方差可以用来描述两个事件之间的相关性。
