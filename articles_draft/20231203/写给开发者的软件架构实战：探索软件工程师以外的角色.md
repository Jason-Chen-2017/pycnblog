```python
# 定义一个模块化的类

    def __init__(self, name):
        self.name = name

    def do_something(self):
        print("do something")

# 创建一个模块化的对象
module = Module("module1")

# 调用模块化对象的方法
module.do_something()
```

在这个代码实例中，我们定义了一个模块化的类`Module`，并创建了一个模块化的对象`module`。`Module`类的`do_something`方法是这个模块化对象的一个方法。通过调用`module`对象的`do_something`方法，我们可以看到`do something`的输出。

```python

# 定义一个抽象的类
class Abstract:
    def __init__(self, name):
        self.name = name

    def do_something(self):
        raise NotImplementedError("子类需要实现do_something方法")

# 创建一个抽象的对象
abstract = Abstract("abstract")

# 调用抽象对象的方法
abstract.do_something()
```

在这个代码实例中，我们定义了一个抽象的类`Abstract`，并创建了一个抽象的对象`abstract`。`Abstract`类的`do_something`方法是这个抽象对象的一个方法。通过调用`abstract`对象的`do_something`方法，我们可以看到`NotImplementedError`的错误提示。
