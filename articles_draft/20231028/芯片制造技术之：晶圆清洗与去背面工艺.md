# 1.背景介绍


晶圆表面电极层、空间、内部的各类分层材料等都存在着较多的高温、低电压等异常状况，称之为“杂质”，而晶圆在制造过程中的主要处理步骤为晶圆晾掸、晶圆表面电极层清洗、电极去除、晶圆去背面清洗等。这些处理方法的目的就是为了使晶圆表面上不存在杂质，降低晶圆表面开裂伤害、保护晶圆内陶瓷化合物、降低晶圆成本及整体性能。本文将详细介绍晶圆表面电极层清洗与去背面清洗技术。
# 2.核心概念与联系
晶圆表面电极层清洗：指对晶圆表面电极层进行清洗液喷涂后使其纳入分层清洁剂，如铅酸钠或氧化铝来分离杂质，从而达到减少杂质和提高纳米材料性能的效果。该技术主要用于杀菌、去除苯并防止刺激性环境下晶圆爆炸，具有较好的材料耐腐蚀能力和工作良好，但工艺复杂，且会引入一定的污染物质。


晶圆去背面清洗：指在生产前后清洗晶圆外壳上的电子材料，清洗掉不必要的电子束、点片和孔隙，并切断墙壁和导体的导线引起的回流，同时还要注意去除和防止由于晶圆体积过大导致的断层现象。此外，需要注意的是，晶圆去背面清洗也是一种重要的环保措施。

## 晋升版晶圆表面电极层清洗技术
晶圆表面电极层清洗技术是在晶圆表面上增加杂质、杀菌、分层、纳米化、增强纳米性能和保护沉积物的方法。其基本流程是通过选择适当的清洗液、预先喷洒至低温（-100~-150摄氏度）、放置干燥湿润的容器中，然后打开时速机器修复部件。
### 操作步骤
1. 在晶圆表面电极层选择适当的清洗液、适量用油或者无油粘土（涂抹于电极层周围），进行擦洗，等待确诊；
2. 将清洗液与碘酸钠混合至一定比例，如1:5000，加入适量用水，按100毫升计量的速度，逐渐倒入到上面步骤中的无油粘土上，从顶部向底部擦洗，一次只需擦干即可；
3. 使用干净手指轻敲晶圆表面的电极层，至少擦出一层清洗液，以消除残留液滴和杂质；
4. 对残留液滴进行及时冲洗，烘干干净后，转移至干净的容器中保存；
5. 在准备好的空气中等待至少一个小时，然后关闭容器，并保持静止至24小时；
6. 当晶圆表面电极层清洗后的情况出现明显改善时，便可以重复以上步骤。
### 原理及理论依据
晶圆表面电极层的结构主要包括电极层、电极自噬层、结构层和分层材料层四个层次，其中电极层位于晶圆表面和晶圆之间的稳定区，主要负责电子辐射和外界的环境影响；电极自噬层则位于晶圆表面中部，扮演了屏障作用，可有效控制电极的外溢。由于电极自噬层直接接触空气，因而在电极层周围留下充满微生物的“杂质”。如今，越来越多的人认为杂质对纳米材料的性能有着巨大的负面影响，因此很多纳米材料的制造厂商都进行了进一步的研究，希望能够找到一种特殊的清洗液能够在晶圆表面电极层中完全去除杂质，降低杂质浓度、提升纳米材料性能，获得更优异的性能。

晶圆表面电极层清洗法的关键就在于所选用的清洗液。在一般情况下，清洗液应当具备以下几种特点：
- 清洁，具有杀菌作用，且易于排出微生物，从而使电极层中存在的杂质尽可能少；
- 满足要求，其含有的硫氧化物的含量在90%左右，能够达到纳米材料清洗的标准要求。

常用的清洗液有金属酸盐（如铅酸钠）、氧化铝、红黄色超聚氮苯酚、乙二醇、对甲酸（Teflon）。但这些清洗液的用量都比较大，容易造成杀菌效应，对于一些特殊的纳米材料（如石墨烯）来说，这种杀菌效应可能不太起作用。所以，对于不同工艺领域或特定纳米材料，必须寻找合适的清洗液来进行电极层清洗。

目前，晶圆表面电极层清洗的工艺通常包括三个阶段：
- 第一阶段：将原有的电极层的杂质全部分离，将新分离出的杂质加上清洗液进行处理；
- 第二阶段：利用轻巧易携带的设备对晶圆表面电极层进行快速的连续清洗，进行初步的消除；
- 第三阶段：再次进入前面介绍的步骤进行处理，最后实现电极层的纯净化，并加固电极。

晶圆表面电极层清洗的方法还有一些其他的原理、操作方针和技巧，如下所示：
- 采用高速度的热交换方式对晶圆表面进行清洗，使其保持冷静，从而避免手指伤及晶圆表面，减少运动损伤；
- 通过多个清洗道路对晶圆表面进行清洗，使得晶圆表面电极层有利于分层和清洁；
- 采用电磁感应的形式，通过使用特殊的清洗工具，可以将杂质直接清除出晶圆表面；
- 一般每隔三五天，晶圆表面电极层应当重新清洗一次，以期待获得改善；
- 需要定期对晶圆表面进行检查，以保证电极层没有残留物质；
- 不宜同时进行多个纳米材料的电极层清洗。

## 晋升版晶圆去背面清洗技术
晶圆去背面清洗技术是指在生产前后清洗晶圆外壳上的电子材料，清洗掉不必要的电子束、点片和孔隙，并切断墙壁和导体的导线引起的回流，同时还要注意去除和防止由于晶圆体积过大导致的断层现象。
### 操作步骤
1. 在晶圆外壳上选择适当的清洗剂，如铝垢、锰酸或氧化铝溶液，对电子材料进行擦洗，吸气后使电子结晶在湿润的环境中晾晒10分钟；
2. 用干净的手指轻轻敲击晶圆外壳上的孔洞，使孔洞内的颗粒被完全驱除，一旦孔洞内被完全驱除，电子便无法运动，产生的电流和辐射不会对电子材料造成影响；
3. 关闭晶圆外壳上的盖子，将孔洞封闭；
4. 如果晶圆已经处于正常成型状态，请对晶圆外部采用清洗剂，如铝垢、锰酸或氧化铝溶液，对孔洞、导线和导体进行全面清洗；
5. 在晶圆处于静止状态前，应立即对导体进行移除和封堵；
6. 晶圆表面受到电子辐射的损害是晶圆性能的决定因素，因此在晶圆制造完成后需要进行清洗，以促进晶圆表面电极层的纳米化。
### 原理及理论依据
晶圆外壳的电子材料表面，常发生缺陷、破损或过热，如：液化、沉淀、残余气体、电子、微生物等，这些材料对晶圆造成极大的伤害。清洗晶圆外壳上的电子材料，可以消除这些污染物，提升晶圆性能。

去背面清洗法的实质是由光电子在晶圆外壳的孔洞内加速，使电子浓度达到相对饱和状态。这一过程是通过对晶圆外壳的压力、温度、速度以及金属薄膜层的磁场进行控制实现的。当孔洞内的电子浓度达到饱和状态时，孔洞内的气体便被排出，孔洞空白处的电子便被驱除，留下的只有极少量的电子残余。

晶圆外壳的外部，除了孔洞、导线和导体，还有许多孤立的亚元件，如电极、晶圈、材料贴片、微丝、玻璃钢等。这些孤立物散落在晶圆外壳的表面上，有时候甚至是无生命的，可以通过去除亚元件的方法，来清除孔洞。

为了防止晶圆体积过大导致的断层现象，晶圆去背面清洗技术还应注重保持材料的平整和完整，以保证晶圆表面电极层的纳米化。

晶圆去背面清洗技术的操作方法及相关原理、操作方针和技巧，如下：
- 消除电子材料：在晶圆外部的孔洞、导线、导体、缺陷等位置上，采用高效率的清除剂对孔洞内的电子进行驱除，如氧化铝溶液、铝垢、锰酸等；
- 封闭孔洞：将晶圆外部的孔洞封闭起来，使孔洞内的电子无法运动，并避免直接接触外部环境；
- 检查孔洞：每隔半个月对晶圆外壳的孔洞进行检查，检验孔洞是否已被完全驱除；
- 继续清除孔洞：如果孔洞仍然被完全驱除，可以继续用清除剂进行清除，直到孔洞空白处没有残余电子；
- 去除导体：在清除完晶圆孔洞之后，还应继续清除晶圆外部的导体；
- 禁止接触：在清除完晶圆外壳之后，还应禁止晶圆外壳的任何接触，以免出现其他异物；
- 保持完美：经过晶圆去背面清洗之后，晶圆的外壳应保持完好，无瑕疵，并保持完美，保证晶圆表面电极层的纳米化。
