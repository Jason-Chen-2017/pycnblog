## 1.1 欧氏距离的定义

欧氏距离（Euclidean Distance）是一种度量空间中两点之间距离的方法，它可以用来计算两个向量之间的距离。给定两个向量 $A$ 和 $B$，欧氏距离可以通过以下公式计算：

$$
d(A, B) = \sqrt{\sum_{i=1}^{n}(a_i - b_i)^2}
$$

其中，$n$ 是向量 $A$ 和 $B$ 的维数，$a_i$ 和 $b_i$ 是向量 $A$ 和 $B$ 的第 $i$ 个元素。

## 1.2 欧氏距离的应用

欧氏距离在机器学习中的应用非常广泛，主要有以下几个方面：

- **聚类算法**：K-均值聚类、DBSCAN 等聚类算法中，欧氏距离用于计算数据点之间的距离，从而实现数据点的分组。
- **分类算法**：支持向量机（SVM）、K 近邻（KNN）等分类算法中，欧氏距离用于计算样本之间的距离，从而实现样本的分类。
- **回归算法**：线性回归、多项式回归等回归算法中，欧氏距离用于计算预测值与实际值之间的差异，从而实现模型的训练。

## 1.3 欧氏距离的局限性

尽管欧氏距离在许多情况下表现出色，但它也存在一些局限性。例如，对于非正态分布的数据，欧氏距离可能会产生偏见；对于高维数据，欧氏距离可能会导致计算效率的下降。因此，在实际应用中，需要根据具体问题选择合适的距离度量标准。

## 4.1 代码实例

def euclidean_distance(A, B):
    n = len(A)
    distance = 0
    for i in range(n):
        distance += (A[i] - B[i]) ** 2
    return np.sqrt(distance)

A = np.array([1, 2, 3])
B = np.array([4, 5, 6])

print(euclidean_distance(A, B))

接下来，我们遍历向量 `A` 和 `B` 的每个元素，计算每个元素之间的差值，并将差值平方。然后，我们将所有元素的平方和存储到变量 `distance` 中。

最后，我们计算 `distance` 的平方根，并将结果返回。
