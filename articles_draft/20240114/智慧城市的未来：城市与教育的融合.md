# 4.具体代码实例和详细解释说明

在教育与智慧城市的融合中，具体代码实例和详细解释说明如下：

1. 使用Python的scikit-learn库实现线性回归模型：
```python
from sklearn.linear_model import LinearRegression

# 训练数据
X_train = [[1], [2], [3], [4], [5]]
y_train = [1, 2, 3, 4, 5]

# 创建线性回归模型
model = LinearRegression()

# 训练线性回归模型
model.fit(X_train, y_train)

# 预测新数据
X_new = [[6]]
y_pred = model.predict(X_new)
print(y_pred)  # 输出：[6]
```

2. 使用Python的numpy库实现多项式回归模型：
```python
import numpy as np

# 训练数据
X_train = np.array([[1], [2], [3], [4], [5]])
y_train = np.array([1, 2, 3, 4, 5])

# 创建多项式回归模型
model = np.poly1d(np.polyfit(X_train, y_train, 2))

# 预测新数据
X_new = np.array([[6]])
y_pred = model(X_new)
print(y_pred)  # 输出：[6]
```
