数据清洗的具体操作步骤包括：

- 数据检查：检查数据是否满足一定的格式、范围和约束条件。
- 数据纠正：修正数据中的错误、不完整和不合适的信息。
- 数据过滤：删除不符合预期的数据，如重复、缺失和异常值。

# 4.具体代码实例和详细解释说明

以下是一个简单的数据治理的生命周期管理示例：

```python
import pandas as pd
import numpy as np

# 数据收集
data = pd.read_csv('data.csv')

# 数据存储
data.to_csv('data_processed.csv', index=False)

# 数据处理
data['age'] = data['age'].apply(lambda x: x if x > 0 else np.nan)
data['income'] = data['income'].apply(lambda x: x if x > 0 else np.nan)
data = data.dropna()

# 数据分析
mean_age = data['age'].mean()
mean_income = data['income'].mean()

# 数据使用
print(f'Average age: {mean_age}')
print(f'Average income: {mean_income}')
```

在这个示例中，我们首先使用pandas库收集数据，然后使用to_csv方法存储数据。接着，我们使用apply方法对age和income列进行处理，将负值替换为NaN。然后，我们使用dropna方法删除缺失值。最后，我们使用mean方法计算平均值，并使用print方法输出结果。
