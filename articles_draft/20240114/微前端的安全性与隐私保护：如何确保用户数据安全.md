## 4.1 数据加密示例

### 4.1.1 AES加密示例

在这个示例中，我们将使用AES算法对用户密码进行加密。

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad

# 生成密钥
key = get_random_bytes(16)

# 生成加密对象
cipher = AES.new(key, AES.MODE_ECB)

# 生成明文
plaintext = b"password"

# 加密明文
ciphertext = cipher.encrypt(pad(plaintext, AES.block_size))

print("加密后的密文：", ciphertext)
```

### 4.1.2 RSA加密示例

在这个示例中，我们将使用RSA算法对用户密码进行加密。

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# 生成密钥对
key = RSA.generate(2048)

# 生成公钥
public_key = key.publickey()

# 生成私钥
private_key = key

# 生成明文
plaintext = b"password"

# 使用公钥加密明文
cipher = PKCS1_OAEP.new(public_key)
ciphertext = cipher.encrypt(plaintext)

print("使用公钥加密后的密文：", ciphertext)

# 使用私钥解密密文
cipher = PKCS1_OAEP.new(private_key)
plaintext = cipher.decrypt(ciphertext)

print("使用私钥解密后的明文：", plaintext)
```

## 4.3 跨站脚本攻击防护示例

### 4.3.1 输入过滤示例

在这个示例中，我们将使用输入过滤来防止XSS攻击。

```javascript
function escapeHTML(input) {
  return input
    .replace(/&/g, '&amp;')
    .replace(/</g, '&lt;')
    .replace(/>/g, '&gt;')
    .replace(/"/g, '&quot;')
    .replace(/'/g, '&#39;');
}

const input = '<script>alert("XSS")</script>';
const safeInput = escapeHTML(input);

console.log(safeInput); // &lt;script&gt;alert("XSS")&lt;/script&gt;
```
