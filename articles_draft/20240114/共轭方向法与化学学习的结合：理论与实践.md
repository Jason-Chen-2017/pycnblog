1. 初始化：从一个随机点开始，设置初始解x0和初始共轭向量d0。
2. 计算共轭梯度：计算共轭梯度g0=∇f(x0)·d0。
3. 选择步长：选择适当的步长α，使得f(x0+αd0)最小。
4. 更新解和共轭向量：更新解x1=x0+αd0，并计算新的共轭向量d1=g0+βd0，其中β是一个适当的参数。
5. 重复上述步骤，直到满足收敛条件。

## 4.1 共轭方向法的Python实现
```python
import numpy as np

def conjugate_gradient(A, b, x0, tol=1e-9, max_iter=1000):
    r = b - A @ x0
    d = r / np.linalg.norm(r)
    g = A @ d
    x = x0
    r_prev = r
    k = 0
    while np.linalg.norm(r) > tol and k < max_iter:
        alpha = (r @ r) / (d @ g)
        x += alpha * d
        r -= alpha * g
        g = A @ r + beta * g
        d = r / np.linalg.norm(r)
        r_prev = r.copy()
        k += 1
    return x, k
```
