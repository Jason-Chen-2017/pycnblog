# 2.核心概念与联系
# 2.1 TensorFlow的核心概念
TensorFlow的核心概念是张量（Tensor）。张量是一种多维数组，用于表示神经网络中的数据和计算。张量可以是整数、浮点数、复数等不同类型的数据。张量可以用于表示输入数据、权重、偏置、输出等。

TensorFlow还提供了一种称为图（Graph）的数据结构，用于表示神经网络的计算过程。图是一种有向无环图（DAG），其中每个节点表示一个操作，每条边表示数据的流动。图可以用于表示神经网络的结构和计算过程，使得TensorFlow能够在不同设备上进行高效的计算。

# 2.2 Keras的核心概念
Keras的核心概念是高层次的神经网络API。Keras提供了一种简单、可扩展的方法来构建、训练和评估神经网络模型。Keras支持多种不同类型的神经网络，如卷积神经网络（CNN）、循环神经网络（RNN）、自编码器等。

Keras还提供了一种称为层（Layer）的数据结构，用于表示神经网络的计算过程。层可以用于表示输入层、隐藏层、输出层等，每个层都可以用于表示一个操作。层可以用于表示神经网络的结构和计算过程，使得Keras能够在不同设备上进行高效的计算。

# 2.3 TensorFlow与Keras的联系
TensorFlow与Keras之间有很强的联系。Keras是一个高层次的神经网络API，可以在TensorFlow上运行。Keras使用TensorFlow作为底层计算引擎，可以利用TensorFlow的强大计算能力来构建、训练和评估神经网络模型。同时，Keras提供了一种简单、可扩展的方法来构建、训练和评估神经网络模型，使得构建和训练神经网络变得更加简单和高效。

import tensorflow as tf

# 定义一个简单的神经网络模型
class SimpleNN(tf.keras.Model):
    def __init__(self):
        super(SimpleNN, self).__init__()
        self.dense1 = tf.keras.layers.Dense(64, activation='relu')
        self.dense2 = tf.keras.layers.Dense(10, activation='softmax')

    def call(self, inputs):
        x = self.dense1(inputs)
        return self.dense2(x)

# 构建、训练和评估神经网络模型
model = SimpleNN()
