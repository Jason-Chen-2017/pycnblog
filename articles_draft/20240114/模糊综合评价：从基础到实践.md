# 2. 核心概念与联系

## 2.1 模糊集
模糊集（Fuzzy Set）是一种用于描述不确定性和不完全信息的概念。它可以用来表示一个集合中元素的属于某个属性的程度。例如，我们可以用模糊集来描述一个学生的成绩，其成绩可能属于“良好”、“中等”和“差”等不同程度。模糊集可以通过定义一个属于度函数来表示元素与属性之间的关系。

## 2.2 模糊关系
模糊关系（Fuzzy Relation）是一种用于描述多个因素之间复杂关系的概念。它可以用来表示不同因素之间的相互作用和影响。例如，在评价企业绩效时，我们可能需要考虑多个因素，如收入、利润、市值等。这些因素之间可能存在复杂的关系，例如收入可能会影响市值，而利润可能会影响收入。模糊关系可以通过定义一个关系矩阵来表示这些因素之间的关系。

## 2.3 模糊判定
模糊判定（Fuzzy Decision）是一种用于处理不确定性和不完全信息的方法。它可以用来根据不同因素的属于度和关系矩阵来进行评价。例如，在评价学生成绩时，我们可以根据不同学生的成绩属于度来进行排名。模糊判定可以通过定义一个判定函数来实现。
