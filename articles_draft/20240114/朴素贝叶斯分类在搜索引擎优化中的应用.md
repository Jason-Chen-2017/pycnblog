# 4. 具体代码实例和详细解释说明

## 4.1 代码实例

以下是一个简单的朴素贝叶斯分类代码实例：

```python
from sklearn.naive_bayes import MultinomialNB
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 训练数据集
data = [
    ("汽车新车", "汽车新车"),
    ("汽车二手车", "汽车二手车"),
    ("家居装修", "家居装修"),
    ("家居建筑", "家居建筑"),
    ("电子产品", "电子产品"),
    ("电子配件", "电子配件"),
]

# 特征和标签
X, y = zip(*data)

# 特征向量化
vectorizer = CountVectorizer()
X_vectorized = vectorizer.fit_transform(X)

# 训练数据集和测试数据集的分割
X_train, X_test, y_train, y_test = train_test_split(X_vectorized, y, test_size=0.2, random_state=42)

# 训练朴素贝叶斯分类模型
model = MultinomialNB()
model.fit(X_train, y_train)

# 测试朴素贝叶斯分类模型
y_pred = model.predict(X_test)

# 测试准确度
accuracy = accuracy_score(y_test, y_pred)
print("准确度:", accuracy)
```

## 4.2 详细解释说明

上述代码实例中，我们首先创建了一个训练数据集，其中包含了一些例子和它们的标签。然后，我们使用`CountVectorizer`进行特征向量化，以便于模型训练。接着，我们使用`train_test_split`函数将数据集分割为训练数据集和测试数据集。

接下来，我们使用`MultinomialNB`进行朴素贝叶斯分类模型训练。在训练完成后，我们使用模型进行预测，并计算准确度。

答案：类别不平衡问题可以通过重采样、权重调整等方法来解决。重采样可以通过随机抑制或随机放大，增加少数类别的样本数量；权重调整可以通过给少数类别的样本分配更高的权重，来提高分类模型的准确性。
