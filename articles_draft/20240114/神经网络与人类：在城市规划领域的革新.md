## 3.5 数学模型公式详细讲解

在神经网络中，常用的数学模型公式包括：

1. 激活函数：激活函数是神经网络中的关键组成部分，用于实现非线性变换。常用的激活函数有sigmoid、tanh和ReLU等。

$$
sigmoid(x) = \frac{1}{1 + e^{-x}}
$$

$$
tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}
$$

$$
ReLU(x) = max(0, x)
$$

1. 损失函数：损失函数用于衡量模型预测结果与真实值之间的差距，常用的损失函数有均方误差、交叉熵等。

$$
MSE(y, \hat{y}) = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
$$

$$
CrossEntropy(y, \hat{y}) = -\frac{1}{n} \sum_{i=1}^{n} [y_i \log(\hat{y}_i) + (1 - y_i) \log(1 - \hat{y}_i)]
$$

1. 梯度下降：梯度下降是神经网络训练中的一种常用优化算法，用于更新神经网络的权重和偏置参数。

$$
\theta = \theta - \alpha \nabla_{\theta} J(\theta)
$$

其中，$\alpha$ 是学习率，$J(\theta)$ 是损失函数。

# 4. 具体代码实例和详细解释说明

在这个部分，我们将提供一个具体的代码实例，以展示神经网络在城市规划领域的应用。

```python
import numpy as np
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

# 数据预处理
X_train = np.load('X_train.npy')
Y_train = np.load('Y_train.npy')
X_test = np.load('X_test.npy')
Y_test = np.load('Y_test.npy')

# 模型构建
model = Sequential()
model.add(Dense(64, input_dim=X_train.shape[1], activation='relu'))
model.add(Dense(32, activation='relu'))
model.add(Dense(1, activation='linear'))

# 模型训练
model.compile(loss='mean_squared_error', optimizer='adam')
model.fit(X_train, Y_train, epochs=100, batch_size=32, validation_split=0.2)

# 模型评估
loss = model.evaluate(X_test, Y_test)
print('Loss:', loss)

# 模型应用
predictions = model.predict(X_test)
```

在这个代码实例中，我们使用了 TensorFlow 和 Keras 库来构建、训练和评估神经网络模型。首先，我们对数据进行了预处理，然后构建了一个简单的神经网络模型，包括两个隐藏层和一个输出层。接下来，我们使用了 Adam 优化器和均方误差损失函数来训练模型，并在训练集和验证集上进行了训练。最后，我们使用了测试数据来评估模型的性能，并使用了训练好的模型来进行预测。
