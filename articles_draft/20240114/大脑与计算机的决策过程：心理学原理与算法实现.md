4. 神经网络：

```python
import numpy as np

# 假设有一组数据，X为输入变量，y为输出变量
X = np.array([[1, 2], [3, 4], [5, 6], [7, 8]])
y = np.array([0, 1, 0, 1])

# 假设有一层输入层和一层输出层的神经网络
input_layer = 2
output_layer = 1

# 初始化权重和偏置
weights = np.random.rand(input_layer, output_layer)
bias = np.random.rand(output_layer)

# 使用梯度下降算法训练模型
learning_rate = 0.01
epochs = 1000

for epoch in range(epochs):
    output = np.dot(X, weights) + bias
    error = y - output
    gradients = 2 * X.T.dot(error)
    weights -= learning_rate * gradients
    bias -= learning_rate * error

# 使用模型预测y值
y_pred = np.where(output > 0.5, 1, 0)

# 打印预测结果
print("预测结果：", y_pred)
```
