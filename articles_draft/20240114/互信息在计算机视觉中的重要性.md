## 3.1 互信息的计算

互信息的计算可以通过以下公式进行：

$$
I(X;Y) = H(X) - H(X|Y)
$$

其中，$H(X)$ 是随机变量 $X$ 的熵，$H(X|Y)$ 是随机变量 $X$ 给定随机变量 $Y$ 的熵。

熵的计算可以通过以下公式进行：

$$
H(X) = -\sum_{x \in X} P(x) \log P(x)
$$

$$
H(X|Y) = -\sum_{x \in X, y \in Y} P(x,y) \log P(x|y)
$$

其中，$P(x)$ 是随机变量 $X$ 取值 $x$ 的概率，$P(x,y)$ 是随机变量 $X$ 和 $Y$ 同时取值 $x$ 和 $y$ 的概率，$P(x|y)$ 是随机变量 $X$ 给定随机变量 $Y$ 取值 $y$ 的概率。

## 4.1 计算互信息的Python代码实例

以下是一个计算互信息的Python代码实例：

```python
import numpy as np

def entropy(p):
    return -np.sum(p * np.log2(p))

def mutual_information(p, q):
    p_xy = p * q
    h_x = entropy(p)
    h_y = entropy(q)
    h_xy = entropy(p_xy)
    return h_x - h_xy + h_y

# 示例数据
p = np.array([0.1, 0.2, 0.3, 0.4])
q = np.array([0.4, 0.3, 0.2, 0.1])

# 计算互信息
mi = mutual_information(p, q)
print("互信息:", mi)
```
