# 4.具体代码实例和详细解释说明
## 4.1 逻辑回归示例
```python
import numpy as np
from sklearn.linear_model import LogisticRegression

# 创建数据
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([0, 1, 0, 1])

# 创建逻辑回归模型
model = LogisticRegression()

# 训练模型
model.fit(X, y)

# 预测
print(model.predict([[2, 3]]))
```
