# 4.具体代码实例和详细解释说明

## 4.1 物流资源安全算法实例

```python
import numpy as np

def resource_safety(data):
    n = len(data)
    R = np.zeros(n)
    for i in range(n):
        R[i] = data[i]['complete'] + data[i]['available'] - data[i]['damaged']
    S_r = np.mean(R)
    return S_r

data = [
    {'complete': 100, 'available': 100, 'damaged': 0},
    {'complete': 100, 'available': 100, 'damaged': 0},
    {'complete': 100, 'available': 100, 'damaged': 0},
]

S_r = resource_safety(data)
print(S_r)
```
