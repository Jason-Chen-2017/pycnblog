# 4. 具体代码实例和详细解释说明

在本节中，我们将提供一个具体的数字化证券交易平台的代码实例，并详细解释说明其工作原理。

```python
import hashlib
import json
import time

class DigitalStock:
    def __init__(self, symbol, price, quantity):
        self.symbol = symbol
        self.price = price
        self.quantity = quantity

    def sign(self, private_key):
        message = json.dumps({'symbol': self.symbol, 'price': self.price, 'quantity': self.quantity})
        signature = hashlib.sha256((message + private_key).encode('utf-8')).hexdigest()
        return signature

    def verify(self, public_key, signature):
        message = json.dumps({'symbol': self.symbol, 'price': self.price, 'quantity': self.quantity})
        return hashlib.sha256((message + signature).encode('utf-8')).hexdigest() == public_key

# 示例代码
stock = DigitalStock('AAPL', 150, 10)
private_key = 'my_private_key'
public_key = 'my_public_key'
signature = stock.sign(private_key)
print(stock.verify(public_key, signature))
```

在上述代码中，我们定义了一个 DigitalStock 类，用于表示数字化股票。DigitalStock 类包括以下方法：

- sign()：用于生成数字签名，确保交易信息的安全传输。
- verify()：用于验证数字签名，确保交易信息的安全传输。

示例代码中，我们创建了一个 DigitalStock 实例，并生成了一个数字签名。然后，我们使用 verify() 方法来验证数字签名的有效性。
