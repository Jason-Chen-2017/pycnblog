## 4.1 公钥加密与私钥加密

在Python中，我们可以使用`rsa`库来实现公钥加密和私钥加密。

```python
from rsa import RSA

# 生成公钥和私钥
key = RSA.generate(2048)
public_key = key.publickey()
private_key = key

# 使用公钥加密数据
data = b'Hello, World!'
encrypted_data = public_key.encrypt(data)

# 使用私钥解密数据
decrypted_data = private_key.decrypt(encrypted_data)
```

## 4.2 数字证书

在Python中，我们可以使用`rsa`库来实现数字证书的生成和验证。

```python
from rsa import RSA, sign, verify

# 生成公钥和私钥
key = RSA.generate(2048)
public_key = key.publickey()
private_key = key

# 使用私钥对数据进行签名
data = b'Hello, World!'
signature = sign(data, private_key)

# 使用公钥对数据进行验证
is_valid = verify(signature, data, public_key)
```
