## 4.1 智能浴缸的代码实例
智能浴缸的代码实例可以通过以下Python代码来实现：

```python
import time

class SmartShower:
    def __init__(self):
        self.water_temperature = 38
        self.water_pressure = 10
        self.water_flow = 10

    def set_water_temperature(self, temperature):
        self.water_temperature = temperature

    def set_water_pressure(self, pressure):
        self.water_pressure = pressure

    def set_water_flow(self, flow):
        self.water_flow = flow

    def get_water_temperature(self):
        return self.water_temperature

    def get_water_pressure(self):
        return self.water_pressure

    def get_water_flow(self):
        return self.water_flow

if __name__ == '__main__':
    shower = SmartShower()
    shower.set_water_temperature(38)
    shower.set_water_pressure(10)
    shower.set_water_flow(10)
    print(shower.get_water_temperature())
    print(shower.get_water_pressure())
    print(shower.get_water_flow())
```
