# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明正则表达式与正则分组的使用和应用。

假设我们有一个包含以下字符串的文本：

```
Hello, my name is John Doe. I live in New York.
```

我们想要从这个文本中提取名字和城市。为了实现这个功能，我们可以使用正则表达式和正则分组。

首先，我们可以使用正则表达式来匹配名字和城市的模式：

```
name: ([A-Z][a-zA-Z]*) Doe
city: New ([A-Z][a-zA-Z]*)
```

这里的正则表达式可以匹配名字和城市的模式，其中name模式匹配以Doe结尾的名字，city模式匹配以New结尾的城市。

接下来，我们可以使用正则分组来捕获名字和城市的子串：

```
name: ([A-Z][a-zA-Z]*) Doe
city: New ([A-Z][a-zA-Z]*)
```

这里的正则分组可以捕获名字和城市的子串，其中name分组匹配名字，city分组匹配城市。

最后，我们可以使用正则表达式和正则分组来提取名字和城市：

```python
import re

text = "Hello, my name is John Doe. I live in New York."
name_pattern = r"name: ([A-Z][a-zA-Z]*) Doe"
city_pattern = r"city: New ([A-Z][a-zA-Z]*)"

name_match = re.search(name_pattern, text)
city_match = re.search(city_pattern, text)

if name_match:
    name = name_match.group(1)
    print(f"Name: {name}")
if city_match:
    city = city_match.group(1)
    print(f"City: {city}")
```

这里的代码可以从文本中提取名字和城市，并将其保存到名字和城市变量中。
