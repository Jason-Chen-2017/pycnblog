## 2.1 查准（Precision）

查准是指模型在正例中的准确性。查准可以用来衡量模型在正例中的有效性，即模型在正例中的真阳性率。查准可以通过以下公式计算：

$$
Precision = \frac{TP}{TP + FP}
$$

其中，TP表示真阳性，FP表示假阳性。

## 2.2 查全（Recall）

查全是指模型在正例中的完整性。查全可以用来衡量模型在正例中的覆盖率，即模型在正例中的真阳性率。查全可以通过以下公式计算：

$$
Recall = \frac{TP}{TP + FN}
$$

其中，TP表示真阳性，FN表示假阴性。

## 2.3 查准-查全

查准-查全是一种评估分类器性能的指标，用于衡量模型在正例和负例中的准确性和完整性。查准-查全可以通过以下公式计算：

$$
F1 = 2 \times \frac{Precision \times Recall}{Precision + Recall}
$$

其中，F1表示F1分数，Precision表示查准，Recall表示查全。

# 4.具体代码实例和详细解释说明

## 4.1 查准-查全的计算

在Python中，我们可以使用以下代码计算查准-查全：

```python

y_true = [1, 1, 0, 0, 1]
y_pred = [1, 0, 0, 1, 1]

precision, recall, f1, _ = precision_recall_fscore_support(y_true, y_pred, average='binary')

print('Precision:', precision)
print('Recall:', recall)
print('F1:', f1)
```

在这个例子中，我们使用了`sklearn.metrics.precision_recall_fscore_support`函数来计算查准-查全。`y_true`表示真实标签，`y_pred`表示预测标签。`average='binary'`表示二分类情况下的查准-查全。

