## 3.1 共轭梯度方法

共轭梯度方法（Conjugate Gradient Method）是一种用于解决线性方程组的迭代方法，它的核心思想是利用共轭方向来更有效地搜索最优解。共轭梯度方法的具体操作步骤如下：

1. 初始化：选择一个初始向量x0，并计算梯度g0。
2. 构建共轭方向：将当前梯度g0与上一次的共轭方向c1相加，得到新的共轭方向ci。
3. 更新向量：将当前向量x0加上共轭方向ci的一个正比数，得到新的向量xi。
4. 计算梯度：计算新的向量xi的梯度g。
5. 更新共轭方向：将新的梯度g与上一次的共轭方向ci相加，得到新的共轭方向ci。
6. 判断终止条件：如果梯度g小于一个预设的阈值，则终止迭代，返回最优解。否则，继续从步骤2开始。

共轭梯度方法的数学模型公式如下：

$$
c_i = -g_i + \beta_i c_{i-1} \\
x_{i+1} = x_i + \alpha_i c_i \\
g_{i+1} = f'(x_{i+1}) \\
\beta_i = \frac{g_i^T g_i}{g_{i-1}^T g_{i-1}} \\
\alpha_i = \frac{g_{i+1}^T g_{i+1}}{g_i^T g_i}
$$

其中，$c_i$ 是共轭方向，$x_i$ 是当前向量，$g_i$ 是当前梯度，$f'(x)$ 是目标函数的一阶导数，$\beta_i$ 和$\alpha_i$ 是步长因子。

4. 更新共轭方向：将新的梯度g与上一次的共轭方向ci相加，得到新的共轭方向ci。

def quantum_conjugate_gradient(A, b, x0, tol, max_iter):
    g0 = A @ x0 - b
    c0 = np.zeros_like(g0)
    x = x0
    g = g0
    c = c0
    for i in range(max_iter):
        alpha = np.dot(g, g) / np.dot(c, A @ c)
        x_new = x + alpha * c
        g_new = A @ x_new - b
        beta = np.dot(g_new, g_new) / np.dot(g, g_new)
        c_new = g_new - beta * c
        x = x_new
        g = g_new
        c = c_new
        if np.linalg.norm(g) < tol:
            break
    return x, c, g
