# 4.具体代码实例和详细解释说明

## 4.1 React Native代码实例

React Native的代码实例如下：

```javascript
import React from 'react';
import { View, Text, Button } from 'react-native';

class MyApp extends React.Component {
  render() {
    return (
      <View>
        <Text>Hello, React Native!</Text>
        <Button title="Click me" onPress={() => alert('Button clicked!')} />
      </View>
    );
  }
}

export default MyApp;
```

上述代码实例中，我们使用React Native的组件化开发方式，实现了一个简单的移动应用。我们使用`View`、`Text`和`Button`等组件，实现了移动应用的UI和功能。

## 4.2 Flutter代码实例

Flutter的代码实例如下：

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Hello, Flutter!'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text(
              'Hello, Flutter!',
            ),
            RaisedButton(
              onPressed: () {
                print('Button clicked!');
              },
              child: Text('Click me'),
            ),
          ],
        ),
      ),
    );
  }
}
```

上述代码实例中，我们使用Flutter的UI框架，实现了一个简单的移动应用。我们使用`MaterialApp`、`Scaffold`、`AppBar`、`Text`和`RaisedButton`等组件，实现了移动应用的UI和功能。
