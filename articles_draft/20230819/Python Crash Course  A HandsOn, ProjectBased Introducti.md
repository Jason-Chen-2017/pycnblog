# 1.简介
  

## 1.1 什么是Python？
Python 是一种高级编程语言，可以用来编写应用程序、进行数据处理等。它被设计出来用于可移植性，并具有易于学习的特性。Python 已经成为最受欢迎的编程语言之一，其语法简单易懂，而且内置丰富的库支持。而且 Python 在很多领域都有很广泛的应用，比如科学计算、web开发、机器学习、数据分析、游戏制作、图像处理等。所以，Python 成为当今最流行的编程语言之一。

如果你是一个程序员或者想要从事程序员工作，那么学习 Python 将会非常有帮助。首先，因为 Python 有许多非常有用的模块和库，可以用来解决各种各样的问题。其次，Python 的简单性让初学者入门容易，而且学习曲线平滑。第三，因为 Python 可以运行在许多不同的操作系统平台上，因此，你的代码将具有更好的兼容性。最后，因为 Python 是开源的，任何人都可以在上面进行自由地贡献自己的代码，这也会促进开源社区的蓬勃发展。总之，学习 Python 对你来说是一个理想的选择！

# 2.核心概念与术语
## 2.1 什么是变量？
变量是存储数据的地方。变量的名称通常用一个小写字母开始，后面跟着零个或多个字母数字下划线。变量的值可以通过赋值语句来改变，就像在普通算术中一样。举例如下：

```python
a = 5   # a 是一个整数变量，它的值为 5
b = "Hello"  # b 是一个字符串变量，它的值为 "Hello"
c = True    # c 是一个布尔变量，它的值为 True
d = None    # d 是一个特殊值 None，表示没有值
e = [1, 2, 3]     # e 是一个列表变量，它的元素是 1, 2, 3
f = (1, 2, 3)     # f 是一个元组变量，它的元素是 1, 2, 3
g = {"name": "Alice", "age": 25}   # g 是一个字典变量，它包含键值对 "name": "Alice" 和 "age": 25
h = range(5)      # h 是一个范围对象，它生成序列 0, 1, 2, 3, 4
i = {j for j in range(3)}   # i 是一个集合变量，它包含 0, 1, 2 三个元素
```

除了以上示例，Python 中还可以使用组合变量，即将多个变量组合成一个复合变量，这样就可以方便地访问到它们的内容。如：

```python
x, y, z = 10, 'hello', True   # x=10, y='hello', z=True
data = {'name': 'Alice', 'age': 25}   # data={'name': 'Alice', 'age': 25}
```

## 2.2 数据类型
Python 支持以下几种数据类型：

### 2.2.1 整型（int）
整数（integer）是不可变的数字，没有小数点，如：`1`, `2`, `3`，`-99`。

### 2.2.2 浮点型（float）
浮点数（floating point number）是带小数的数字，如：`3.14`, `-2.5`，`1E-5`。

### 2.2.3 复数型（complex）
复数（complex number）由实部和虚部构成，如：`(3+4j)`, `(1.5-3.5j)`。

### 2.2.4 布尔型（bool）
布尔值只有两种，分别是 `True` 和 `False`，用关键字 `and`、`or` 和 `not` 表示逻辑与、逻辑或、逻辑非运算。

### 2.2.5 字符型（str）
字符串（string）是由单引号 `' '` 或双引号 `" "` 括起来的任意文本序列，如："hello world"。

### 2.2.6 列表型（list）
列表（list）是按顺序排列的一系列值的集合，如：[1, 2, 3], ["apple", "banana"], [[1, 2], [3, 4]]。

### 2.2.7 元组型（tuple）
元组（tuple）类似于列表，但是元素不能修改，如：`(1, 2, 3)`, `("apple", "banana")`。

### 2.2.8 字典型（dict）
字典（dictionary）是一组无序的键值对的集合，每个键都对应一个值，如：{"name": "Alice", "age": 25}。

### 2.2.9 集合型（set）
集合（set）是一组无序的、不可重复的元素的集合，如：{1, 2, 3}, {"apple", "banana"}。

### 2.2.10 None 类型
None 表示特殊的空值，相当于 Java 中的 null。

## 2.3 语句、表达式和标识符
Python 中有三种类型的语句：

### 2.3.1 赋值语句（assignment statement）
赋值语句用于给变量赋值，如：`a = 5`，把右边的值赋给左边的变量 `a`。

### 2.3.2 条件语句（conditional statement）
条件语句用来控制程序执行路径，分为 `if` 语句、 `elif` 语句、 `else` 语句。

```python
if condition1:
    # if 语句块
elif condition2:
    # elif 语句块
else:
    # else 语句块
```

其中，`condition` 是一个表达式，当该表达式的值为真时才执行相应的语句块。

### 2.3.3 循环语句（looping statements）
循环语句用于重复执行相同的代码，共有四种循环语句：

1. `for` 语句
2. `while` 语句
3. `break` 语句
4. `continue` 语句

```python
for variable in iterable:
    # for 语句块
    
while condition:
    # while 语句块
    
for i in range(10):
    print(i)
    if i == 5:
        break
        
for letter in ['a', 'b', 'c']:
    if letter == 'b':
        continue
    print(letter)
```

其中，`iterable` 可以是一个列表、元组、集合或字符串，表示要迭代遍历的对象；`variable` 是循环变量，每次迭代都会改变其值；`range()` 函数产生一个整数序列，一般用于 `for` 循环；`break` 语句终止当前循环，`continue` 语句跳过当前循环，直接进入下一次迭代。

## 2.4 函数
函数（function）是组织好的、可重复使用的代码块，它接受输入参数（arguments），做出输出（return values）。

定义函数的方法：

```python
def function_name(argument1, argument2,...):
    """文档字符串"""
    # 语句块
```

其中，`function_name` 是函数名，可以自定义；`argumentN` 是传入的参数名，可以自定义，默认值可用 `=` 指定；文档字符串（docstring）是描述函数功能的文字，通常用三个双引号 `"""` 或单引号 `'''` 来指定。

调用函数的方法：

```python
result = function_name(value1, value2,...)
```

其中，`result` 是函数的返回值。

## 2.5 模块
模块（module）是 Python 文件，包含了 Python 代码和数据。通过导入（import）模块，可以将模块中的函数、变量、类、方法等引入当前程序的命名空间，使得程序可以调用这些函数、变量、类、方法。

一般情况下，模块文件以 `.py` 结尾，文件名就是模块名。可以使用 `help()` 函数查看某个模块的帮助信息，也可以使用 `dir()` 函数获取某个模块里面的函数、变量、类的列表。

## 2.6 包
包（package）是指包含多个模块的文件夹。创建一个包文件夹，并在里面创建 `__init__.py` 文件即可。然后，就可以在这个文件夹外层添加 `__init__.py` 文件，以便让包被导入。
