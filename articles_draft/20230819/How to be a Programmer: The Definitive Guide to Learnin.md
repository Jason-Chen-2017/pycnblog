# 3.初级技术
## 数据类型和变量
数据类型指的是存储在变量中的值的类型，可以分为两种：基本数据类型（Primitive data types）和引用数据类型（Reference data types）。基本数据类型包括整数型、浮点型、字符型、布尔型等，这些类型的变量占用固定大小的内存空间，可以直接存取其值。而引用数据类型包括类、结构体、数组、指针等，这些类型的变量存储的是指向其他数据的地址，可以在运行时创建、修改和销毁，比较耗费内存。
### 基本数据类型
#### 整数型
整数型的数据类型包括整型和长整型。整型是最简单的一种类型，占用4个字节（32位机上），可以表示正负整数、0，范围为-2^31到2^31-1，可以通过关键字int来声明一个整型变量。长整型则比整型大的内存，一般用关键字long long来声明，占用8个字节（64位机上），可以表示正负整数、0，范围为-2^63到2^63-1。
```cpp
// 整型
int age = 30;
printf("age is %d", age); // output: age is 30

// 长整型
long long salary = -9223372036854775808L; // or just -9223372036854775808
printf("salary is %lld\n", salary); // output: salary is -9223372036854775808
```
#### 浮点型
浮点型的数据类型包括单精度浮点型（float）和双精度浮点型（double）。单精度浮点型占用4个字节，可以表示小数，精确度为7位，可以通过关键字float来声明一个单精度浮点型变量。双精度浮点型占用8个字节，可以表示小数，精确度为15位，可以通过关键字double来声明一个双精度浮点型变量。
```cpp
// float
float height = 1.73f;
printf("%.2f", height); // output: 1.73

// double
double weight = 72.5;
printf("%.1lf pounds\n", weight * 2.2); // output: 96.0 pounds
```
#### 字符型
字符型的数据类型包括字符型、宽字符型和多字节字符型。字符型变量只保存一个ASCII码字符，占用1个字节，可以使用单引号(')或双引号(")括起来。宽字符型和多字节字符型都是用来保存多个字符的，并且允许使用Unicode字符集。宽字符型占用2个字节（UTF-16编码），即使只保存一个中文汉字也需要两个字节。多字节字符型（如GBK、Big5等）可以保存较为复杂的文字，但占用的内存也更多。
```cpp
char letterA = 'A';
wchar_t wLetterA = L'Á'; // for UTF-16 encoding
char multiByte[] = "你好"; // GBK encoded Chinese character string
```
#### 布尔型
布尔型的数据类型只含两个值，true和false。它主要用于逻辑判断，用于控制条件语句的执行流程。布尔型变量可以使用关键字bool来声明。
```cpp
bool flag = true;
if (flag == true) {
    printf("The flag is on");
} else {
    printf("The flag is off");
}
```
### 引用数据类型
#### 字符串
字符串是一个连续的字符序列，可以通过字符数组（char[]）来表示。字符串是一种非常常见的数据结构，对于一些复杂的任务来说，比如文本处理、文件读取、网络通信等都需要用到字符串。
```cpp
char name[10];
strcpy(name, "Alice"); // copy the value of Alice into the array
strcat(name, ", ");
strcat(name, "Bob");
puts(name); // output: Alice, Bob
```
#### 数组
数组是一个有序集合，其元素都有相同的数据类型。数组的声明形式如下：数据类型 数组名称[数组长度]；数组的每个元素的索引值从0开始，最大值为数组长度减1。数组是一种重要的数据结构，很多编程语言都提供了内置的数组支持，方便用户进行数据管理。
```cpp
int numbers[10]; // initialize an int array with length 10
numbers[0] = 10;
numbers[9] = 100;
for (int i=0; i<10; ++i){
    printf("%d ", numbers[i]); // output: 10 0 0 0 0 0 0 0 100 0
}
```
#### 指针
指针是一个存储变量地址的变量，其类型由数据类型决定，包括指向其他数据类型变量的指针、指向函数的指针等。指针可以让程序动态分配内存、访问内存中的数据、调用函数等。指针的声明形式如下：数据类型*指针变量名；指针变量指向的数据类型必须是动态申请的，否则就会导致程序崩溃。
```cpp
int num = 10;
int* ptrNum = &num; // pointer variable that points to num's address
printf("*ptrNum = %d\n", *ptrNum); // output: *ptrNum = 10
```
## 运算符
运算符是用来对表达式的值进行计算的符号。C++语言支持的运算符共有以下几种：
- 算术运算符：+、-、*、/、%、++、--等。
- 关系运算符：<、>、<=、>=、==、!=等。
- 逻辑运算符：&&、||、!。
- 赋值运算符：=、+=、-=、*=、/=等。
- 三元运算符：? :。
- 位运算符：&、|、~、^、<<、>>、<<<、>>>等。
- 成员运算符：.和->。
- 数组下标运算符：[ ].
- 函数调用运算符：().
- sizeof运算符：用于获取某个变量或类型占用多少内存空间。
```cpp
int a = 10;
int b = 5;
int c = a + b; // addition operator
a *= b; // multiplication and assignment operator
c /= b++; // division and post-increment operators
b += (++c) / 2; // compound expressions
cout << sizeof(int) << endl; // output: size of int in bytes
```
## 控制结构
控制结构是用来控制程序执行流程的语句。C++语言支持的控制结构包括以下几种：
- if-else语句：根据条件判断是否执行某段代码。
- switch语句：类似于if-else语句，用于选择不同的分支执行代码块。
- while循环：重复执行语句块直到指定的条件为假。
- do-while循环：同样是重复执行语句块直到指定的条件为假，不过do-while语句至少会执行一次。
- for循环：用来重复执行语句块指定次数。
- break语句：立刻终止当前循环或switch语句的执行。
- continue语句：跳过当前迭代，继续执行下一轮循环。
- goto语句：跳转到指定位置。
```cpp
// example code for control structures
#include <iostream>
using namespace std;

int main()
{
   int x = 10;

   if(x > 15)
      cout<<"x is greater than 15"<<endl;
   else if(x < 15 && x!= 0)
      cout<<"x is between 1 and 15 inclusive"<<endl;
   else if(x == 0)
      cout<<"x equals zero"<<endl;
   else 
      cout<<"x is less than one but not equal to zero"<<endl;
   
   return 0;
}
```
