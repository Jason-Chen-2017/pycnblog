## 4.具体代码实例和详细解释说明

### 4.1 使用Python实现泊松分布

在Python中，可以使用`scipy.stats`模块实现泊松分布。以下是一个简单的示例：

```python
import numpy as np
import scipy.stats as stats

# 设定参数
lambda_value = 5

# 生成随机样本
sample_size = 1000
random_sample = stats.poisson(lambda_value).rvs(sample_size)

# 计算概率密度函数
pdf_value = stats.poisson(lambda_value).pdf(random_sample)

# 计算累积分布函数
cdf_value = stats.poisson(lambda_value).cdf(random_sample)

# 绘制概率密度函数
import matplotlib.pyplot as plt

plt.plot(random_sample, pdf_value)
plt.xlabel('Value')
plt.ylabel('Probability Density')
plt.title('Poisson Distribution PDF')
plt.show()

# 绘制累积分布函数
plt.plot(random_sample, cdf_value)
plt.xlabel('Value')
plt.ylabel('Probability')
plt.title('Poisson Distribution CDF')
plt.show()
```

### Q1：泊松分布与正态分布的区别是什么？

泊松分布和正态分布都是连续随机变量的分布，但它们在应用和特性上有很大不同。泊松分布用于描述随机事件发生的次数，而正态分布用于描述随机事件的取值。泊松分布适用于低频事件，正态分布适用于高频事件。

### Q2：如何选择泊松分布的参数$\lambda$？

泊松分布的参数$\lambda$可以通过实验或观测得到。一种常见的方法是计算已知数据集中的平均值，将其作为$\lambda$的估计。另一种方法是使用最大似然估计（MLE）或贝叶斯估计（BE）等方法进行参数估计。

泊松分布是一种连续随机变量的分布，其取值范围是$(0, \infty)$。虽然泊松分布通常用于描述整数值的随机变量，如缺陷数量、孔隙数量等，但它也可以用于描述非整数值的随机变量，如电子分布、原子位置等。在这种情况下，可以将非整数值近似为整数值，然后使用泊松分布进行分析。
