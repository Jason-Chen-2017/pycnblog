```python
from sklearn.ensemble import VotingClassifier

from sklearn.linear_model import LogisticRegression

```

接下来，我们需要加载数据集：

```python
data = load_iris()
X = data.data
y = data.target
```

然后，我们需要训练多个模型：

```python
model1 = LogisticRegression()
model2 = SVC()
```

接下来，我们需要将多个模型组合在一起：

```python
model = VotingClassifier(estimators=[('lr', model1), ('svc', model2)])
```

最后，我们需要训练组合后的模型：

```python
model.fit(X, y)
```
