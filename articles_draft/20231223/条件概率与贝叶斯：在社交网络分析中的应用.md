# 2.核心概念与联系

## 2.1 条件概率

条件概率是一种描述事件发生的概率，给定另一个事件已经发生的情况下的概率。形式上，条件概率可以表示为：

$$
P(A|B) = \frac{P(A \cap B)}{P(B)}
$$

其中，$P(A|B)$ 是条件概率，表示事件 $A$ 发生的概率，给定事件 $B$ 已经发生；$P(A \cap B)$ 是联合概率，表示事件 $A$ 和事件 $B$ 同时发生的概率；$P(B)$ 是事件 $B$ 发生的概率。

在社交网络分析中，条件概率可以用来计算两个节点之间的关系概率，给定其他节点已经知道的信息。例如，我们可以计算两个用户之间的好友关系概率，给定它们都关注了同一个社交组织。

## 2.2 贝叶斯定理

贝叶斯定理是一种用于更新先验知识（即，在观察数据之前的知识）为后验知识（即，在观察数据之后的知识）的方法。贝叶斯定理可以表示为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 是条件概率，表示事件 $A$ 发生的概率，给定事件 $B$ 已经发生；$P(B|A)$ 是条件概率，表示事件 $B$ 发生的概率，给定事件 $A$ 已经发生；$P(A)$ 是事件 $A$ 发生的概率；$P(B)$ 是事件 $B$ 发生的概率。

在社交网络分析中，贝叶斯定理可以用来更新节点之间的关系概率，给定其他节点已经知道的信息。例如，我们可以更新两个用户之间的好友关系概率，给定它们都关注了同一个社交组织。

## 3.1 计算两个节点之间的关系概率

要计算两个节点之间的关系概率，我们需要知道以下信息：

1. 节点总数 $n$
2. 节点之间的关系矩阵 $R$，其中 $R_{ij}$ 表示节点 $i$ 和节点 $j$ 之间的关系概率。

我们可以使用以下公式计算两个节点之间的关系概率：

$$
P(A|B) = \frac{R_{AB}}{\sum_{j=1}^{n} R_{Aj}}
$$

其中，$P(A|B)$ 是两个节点 $A$ 和 $B$ 之间的关系概率；$R_{AB}$ 是节点 $A$ 和节点 $B$ 之间的关系概率；$R_{Aj}$ 是节点 $A$ 与其他节点之间的关系概率。

## 3.2 基于条件概率的社交网络聚类

要使用条件概率进行社交网络聚类，我们需要以下信息：

1. 节点总数 $n$
2. 节点之间的关系矩阵 $R$，其中 $R_{ij}$ 表示节点 $i$ 和节点 $j$ 之间的关系概率。

我们可以使用以下步骤进行聚类：

1. 计算每个节点与其他节点之间的关系概率。
2. 根据关系概率将节点分组，使得同一组内的节点之间的关系概率较高，而与其他组的节点之间的关系概率较低。
3. 重复步骤2，直到聚类稳定。

## 3.3 基于贝叶斯定理的社交网络推荐

要使用贝叶斯定理进行社交网络推荐，我们需要以下信息：

1. 节点总数 $n$
2. 节点之间的关系矩阵 $R$，其中 $R_{ij}$ 表示节点 $i$ 和节点 $j$ 之间的关系概率。
3. 用户 $A$ 的兴趣向量 $V_A$，其中 $V_{Ai}$ 表示用户 $A$ 对节点 $i$ 的兴趣程度。

我们可以使用以下步骤进行推荐：

1. 计算用户 $A$ 与其他节点之间的关系概率。
2. 使用贝叶斯定理更新用户 $A$ 对其他节点的兴趣程度。
3. 根据更新后的兴趣程度对节点进行排序，并返回排名靠前的节点作为推荐结果。

```python
import numpy as np

n = 100
R = np.random.rand(n, n)
R = R + R.T
R = R / np.sum(R, axis=1)[:, np.newaxis]

V_A = np.random.rand(n)

P_A_given_B = np.zeros(n)
for i in range(n):
    P_A_given_B[i] = R[0, i] * V_A[i] / np.sum(R[0, :] * V_A[:])

recommended_nodes = np.argsort(P_A_given_B)[::-1]
```
