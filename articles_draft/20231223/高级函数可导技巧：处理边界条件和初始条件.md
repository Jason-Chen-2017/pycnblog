## 4.1 代码实例1：求函数的导数

在这个例子中，我们将求一个简单的函数的导数。

```python
import numpy as np

def f(x):
    return x**2

def df(x):
    return 2*x

x = np.linspace(-10, 10, 100)
y = f(x)
dy = df(x)

import matplotlib.pyplot as plt

plt.plot(x, y, label='f(x)')
plt.plot(x, dy, label='f'(x))
plt.legend()
plt.show()
```

在这个例子中，我们定义了一个简单的函数f(x) = x**2，并计算了其导数df(x) = 2*x。然后我们使用numpy和matplotlib来绘制函数和其导数的图像。

## 4.3 代码实例3：求函数的积分

在这个例子中，我们将求一个函数的积分。

```python
import numpy as np

def f(x):
    return x**2

a = -1
b = 1

dx = 0.01
area = 0

for x in np.arange(a, b, dx):
    area += (f(x) + f(x+dx)) * dx / 2

print(f'当x趋近于{b}时，积分值为{area}')
```

在这个例子中，我们定义了一个函数f(x) = x**2，并使用一个简单的积分法来计算其积分值。

解答：积分法的选择取决于函数的复杂性和精度要求。常见的积分法有梯形积分、Simpson积分等。梯形积分是一种简单的积分法，适用于简单的函数；Simpson积分是一种高精度的积分法，适用于复杂的函数。
