## 1.1 贝塔分布的基本概念
贝塔分布是一种连续概率分布，用于描述随机变量的不确定性。它的概率密度函数（PDF）定义为：

$$
f(x) = \frac{x^{\alpha-1}(1-x)^{\beta-1}}{\text{B}(\alpha, \beta)}
$$

其中，$\alpha$ 和 $\beta$ 是贝塔分布的参数，$\text{B}(\alpha, \beta)$ 是贝塔函数。

贝塔分布具有两个参数：$\alpha$ 和 $\beta$。当 $\alpha > 0$ 和 $\beta > 0$ 时，贝塔分布的支持区间为 $[0, 1]$。当 $\alpha$ 和 $\beta$ 都大于 0 时，贝塔分布是一个有界的、连续的、单峰的概率分布。

# 3.2 贝塔函数的定义
贝塔函数的定义为：

$$
\text{B}(\alpha, \beta) = \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}
$$

其中，$\Gamma(\cdot)$ 是伽马函数。

# 3.3 贝塔分布的累积分布函数
贝塔分布的累积分布函数（CDF）定义为：

$$
F(x) = \int_{0}^{x} f(t) dt = \frac{x^{\alpha}(1-x)^{\beta}}{\text{B}(\alpha, \beta)}
$$

```python
import numpy as np
from scipy.special import betainc, gammaln

def beta_pdf(x, alpha, beta):
    a_ln = gammaln(alpha) + gammaln(beta) - gammaln(alpha + beta)
    b_ln = betainc(alpha, beta, x) - np.log(betainc(alpha + 1, beta + 1, x))
    return np.exp(a_ln - b_ln)
```

```python
def mle_beta(data):
    alpha = np.mean(data)
```
