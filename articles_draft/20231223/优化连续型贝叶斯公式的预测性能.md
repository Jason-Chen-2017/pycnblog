# 2.核心概念与联系

## 2.1 贝叶斯方法
贝叶斯方法是一种概率推理方法，它基于贝叶斯定理来更新先验概率和观测数据的后验概率。贝叶斯定理可以表示为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示条件概率，$P(B|A)$ 表示后验概率，$P(A)$ 表示先验概率，$P(B)$ 表示边际概率。

在机器学习和数据挖掘领域，贝叶斯方法可以用于模型建立、参数估计和预测等任务。

## 2.2 连续型贝叶斯公式
连续型贝叶斯公式是一种用于处理连续变量的贝叶斯方法。它可以用于对连续变量进行预测和分类。连续型贝叶斯公式的核心是利用先验分布和观测数据来得到后验分布，从而得到预测和分类的结果。

连续型贝叶斯公式的主要步骤包括：

1. 确定先验分布：根据问题的先验知识，确定连续变量的先验分布。
2. 观测数据：收集并观测数据，得到观测数据的概率分布。
3. 计算后验分布：根据先验分布和观测数据的概率分布，计算后验分布。
4. 预测和分类：根据后验分布，得到预测和分类的结果。
