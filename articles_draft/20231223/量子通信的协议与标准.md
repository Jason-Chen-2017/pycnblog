## 4.2 导入Qiskit库

导入Qiskit库，并初始化一个量子计算机实例：

```python
import qiskit
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 初始化一个量子计算机实例
backend = Aer.get_backend('qasm_simulator')
```

## 4.3 创建量子比特和量子门

创建两个量子比特，并定义量子门：

```python
# 创建两个量子比特
qc = QuantumCircuit(2)

# 定义量子门
qc.h(0)  # 超位门
qc.cx(0, 1)  # CNOT门
```

## 4.4 绘制量子电路

绘制量子电路，以便更好地理解其结构：

```python
# 绘制量子电路
qc.draw()
```


## 4.5 执行量子计算

执行量子计算，并将结果存储到类别记录器中：

```python
# 执行量子计算
qobj = qc.execute(backend)

# 将结果存储到类别记录器中
counts = qobj.get_counts()
```

## 4.6 绘制类别记录器

绘制类别记录器，以便更好地理解结果：

```python
# 绘制类别记录器

```


## 4.7 解释结果

根据类别记录器的结果，可以得出量子比特的状态。在这个例子中，我们可以看到量子比特的状态为 $|11\rangle$ ，这表示量子比特处于第二个状态。
