## 1.1 相对论的基本概念

相对论是现代物理学的一个重要的理论框架，它揭示了时间和空间之间的关系，并且解释了重力的本质。相对论的核心观念是：时间和空间是相互联系的，并且受到物体之间的相对运动的影响。在相对论中，时间和空间形成了一个名为“时空”的四维空间，其中三个空间维度和一个时间维度相互联系。

### 1.1.1 时空坐标系

在相对论中，我们使用时空坐标系来描述物体的位置和时间。时空坐标系由四个坐标组成：三个空间坐标（x、y、z）和一个时间坐标（t）。这四个坐标可以用来描述一个事件在时空中的位置。

### 1.1.2 相对性

相对性是相对论的核心观念，它表示物体之间的运动是相对的，而不是绝对的。在相对性下，两个物体在相对于另一个物体的运动中，它们所经历的时间和空间都会发生变化。这意味着，两个物体之间的时间和空间关系是相对的，而不是绝对的。

def hyper_speed_motion(initial_speed, target_speed, acceleration, time):
    speed = initial_speed
    for t in range(time):
        acceleration = (target_speed - speed) / time
        speed += acceleration
    return speed
