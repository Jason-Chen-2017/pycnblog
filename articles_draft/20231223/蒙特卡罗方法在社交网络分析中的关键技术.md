import random

def generate_random_node_subset(graph, ratio):
    num_nodes = len(graph.nodes())
    num_subset_nodes = int(num_nodes * ratio)
    subset_nodes = random.sample(list(graph.nodes()), num_subset_nodes)
    return subset_nodes

我们可以使用以下代码来计算一个节点的中心性：

```python
def centrality(graph, node):
    degree = len(list(graph.neighbors(node)))
    return 1 / degree
```

### 4.1.3 估计中心性

我们可以使用以下代码来估计中心性：

```python
def estimate_centrality(graph, ratio, num_samples):
    subset_nodes = []
    centrality_sum = 0
    for _ in range(num_samples):
        subset_nodes = generate_random_node_subset(graph, ratio)
        subset_centrality = sum(centrality(graph, node) for node in subset_nodes)
        centrality_sum += subset_centrality
    return centrality_sum / (num_samples * ratio)
```
