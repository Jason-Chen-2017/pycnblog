# 2.核心概念与联系

## 2.1 超导体

超导体是一类具有零电阻的金属，它们在特定温度以下可以无损地传导电流。超导体的发现可以追溯到1911年，当时荷兰物理学家赫尔曼·杜姆（Heike Kamerlingh Onnes）在实验中发现氢的液体状态可以无限低下电导率。随后，他还发现了氢的同类物质氢化钠（NaCl）、氢化钙（CaO）等也具有超导性。

1986年，美国物理学家约翰·勒曼（John L. Vaughan）和他的团队在实验中发现了第一个非氢元素超导体——氧化镍（V2O5）。随后，超导体的种类逐渐增多，包括氧化钙（CaO）、氧化镍（V2O5）、氧化铬（Nb3Sn）等。

超导体的应用主要集中在电磁 levitation、量子计算、超导磁场等领域。例如，超导磁场可以用于医学成像、物理实验等，而电磁 levitation 则可以用于无粘性悬浮系统、高速铁路等。

## 2.2 纤维组合材料

纤维组合材料是一种结合了强度、轻量化和耐用性的材料，通常由纤维（如纤维纤维、玻璃纤维等）与矩阵材料（如塑料、金属等）结合而成。纤维组合材料具有高强度、低密度、高耐用性等特点，因此在车辆、建筑、航空等重型行业中得到了广泛应用。

纤维组合材料的主要类型包括：

1. 纤维纤维组合材料（例如：碳纤维纤维组合材料、金属纤维纤维组合材料等）
2. 玻璃纤维组合材料（例如：高强度玻璃纤维组合材料、轻型玻璃纤维组合材料等）
3. 木纤维组合材料（例如：木纤维纤维组合材料、木纤维塑料组合材料等）

## 2.3 拓扑超导体

拓扑超导体是一种新型的超导体，它的超导性不依赖于零温度，而是依赖于超导体在特定的拓扑结构中的布局。拓扑超导体的发现为超导体的应用开辟了新的可能，尤其是在低温、高强度、高速等场景下的应用方面具有重要意义。

拓扑超导体的主要特点包括：

1. 不依赖于温度的超导性：拓扑超导体的超导性不受温度的影响，因此在低温环境下仍然可以保持高效的电导性。
2. 高强度：拓扑超导体具有较高的强度，可以应用于高强度场景下，如高速铁路、航空等。
3. 高速：拓扑超导体可以在低温环境下实现高速电导，因此在高速传输场景下具有优势。
