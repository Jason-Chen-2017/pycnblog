
## 4.3 模糊规则的建立
我们将模糊规则建立为如果低消耗则低消耗、如果中消耗则中消耗、如果高消耗则高消耗。

```python
if low_consumption:
    then_low_consumption = np.array([1, 2, 3, 4, 5])

if medium_consumption:
    then_medium_consumption = np.array([6, 7, 8, 9, 10])

if high_consumption:
    then_high_consumption = np.array([11, 12, 13, 14, 15])
```

## 4.4 模糊控制的实现
我们将模糊控制实现为根据输入模糊集选择对应的输出模糊集。

```python
def fuzzy_control(input_consumption):
    if np.all(input_consumption <= low_consumption):
        return then_low_consumption
    elif np.all(input_consumption >= medium_consumption):
        return then_medium_consumption
    else:
        return then_high_consumption
```
