### 4.1 受限玻尔兹曼机代码实例

以文本分类问题为例，我们可以使用受限玻尔兹曼机来解决这个问题。首先，我们需要构建一个文本分类模型，然后计算文本的熵，最大化熵，从而实现文本分类。

```python
import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.metrics import accuracy_score

# 文本数据
texts = ["I love machine learning", "Quantum computing is amazing", "Deep learning is cool"]

# 构建文本分类模型
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

# 训练分类器
classifier = MultinomialNB()
classifier.fit(X, texts)

# 预测
y_pred = classifier.predict(vectorizer.transform(["I like quantum computing", "Machine learning is fun"]))

# 评估
print("Accuracy:", accuracy_score(texts, y_pred))
```
