# 2.核心概念与联系

## 2.1 边缘计算

边缘计算（Edge Computing）是一种计算模型，将计算能力推向边缘设备，例如传感器、通信设备和存储设备，以便在数据产生的地方进行实时处理和分析。这种模型可以减少数据传输和存储成本，提高数据处理效率，并实现更快的响应时间。

边缘计算的主要特点包括：

- 数据处理在边缘设备上进行，减少了数据传输到云端的延迟
- 边缘设备具有局部智能，可以进行实时处理和分析
- 边缘设备可以协同工作，共同完成复杂任务
