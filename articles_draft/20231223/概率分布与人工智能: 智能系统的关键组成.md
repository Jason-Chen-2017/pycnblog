## 3.1 离散概率分布
离散概率分布用于描述离散随机变量的分布情况。常见的离散概率分布有：泊松分布、二项分布、多项分布等。

### 3.1.1 泊松分布
泊松分布是一种描述连续随机事件发生的概率分布。泊松分布的概率密度函数为：

$$
P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}
$$

其中，$\lambda$ 是参数，$k$ 是取值为非负整数的随机变量。

### 3.1.2 二项分布
二项分布是一种描述连续随机事件发生的概率分布。二项分布的概率密度函数为：

$$
P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}
$$

其中，$n$ 是事件发生次数，$p$ 是事件发生的概率，$k$ 是取值为0到$n$的随机变量。

### 3.1.3 多项分布
多项分布是一种描述连续随机事件发生的概率分布。多项分布的概率密度函数为：

$$
P(X=k) = \frac{n!}{k!(n-k)!} p^k (1-p)^{n-k}
$$

其中，$n$ 是事件发生次数，$p$ 是事件发生的概率，$k$ 是取值为0到$n$的随机变量。

## 3.2 连续概率分布
连续概率分布用于描述连续随机变量的分布情况。常见的连续概率分布有：正态分布、摇动分布、幂分布等。

### 3.2.1 正态分布
正态分布是一种描述连续随机变量的概率分布。正态分布的概率密度函数为：

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$\mu$ 是均值，$\sigma$ 是标准差。

### 3.2.2 摇动分布
摇动分布是一种描述连续随机变量的概率分布。摇动分布的概率密度函数为：

$$
f(x) = \frac{2}{\pi\sigma^2} e^{-\frac{x^2}{\sigma^2}} \sin(x\sigma)
$$

其中，$\sigma$ 是参数。

### 3.2.3 幂分布
幂分布是一种描述连续随机变量的概率分布。幂分布的概率密度函数为：

$$
f(x) = \frac{\alpha}{\sigma} \frac{x^{\alpha-1}}{\sigma} e^{-\frac{x^{\alpha}}{\sigma}}
$$

其中，$\alpha$ 是参数，$\sigma$ 是参数。

## 4.1 泊松分布的Python实现
```python
import numpy as np
import matplotlib.pyplot as plt

def poisson_pmf(k, lambda_):
    return np.exp(-lambda_) * (lambda_ ** k) / np.math.factorial(k)

lambda_ = 3
k = np.arange(0, 10)
plt.plot(k, poisson_pmf(k, lambda_))
plt.xlabel('k')
plt.ylabel('P(X=k)')
plt.title('Poisson Probability Mass Function')
plt.show()
```
上述代码实现了泊松分布的概率密度函数，并绘制了分布曲线。

## 4.2 二项分布的Python实现
```python
import numpy as np
import matplotlib.pyplot as plt

def binomial_pmf(k, n, p):
    return np.math.comb(n, k) * (p ** k) * ((1 - p) ** (n - k))

n = 10
p = 0.5
k = np.arange(0, n+1)
plt.plot(k, binomial_pmf(k, n, p))
plt.xlabel('k')
plt.ylabel('P(X=k)')
plt.title('Binomial Probability Mass Function')
plt.show()
```
上述代码实现了二项分布的概率密度函数，并绘制了分布曲线。

## 4.3 正态分布的Python实现
```python
import numpy as np
import matplotlib.pyplot as plt

def normal_pdf(x, mu, sigma):
    return (1 / (np.sqrt(2 * np.pi) * sigma)) * np.exp(-(x - mu) ** 2 / (2 * sigma ** 2))

mu = 0
sigma = 1
x = np.linspace(-10, 10, 100)
plt.plot(x, normal_pdf(x, mu, sigma))
plt.xlabel('x')
plt.ylabel('P(X=x)')
plt.title('Normal Probability Density Function')
plt.show()
```
上述代码实现了正态分布的概率密度函数，并绘制了分布曲线。

# 6.附录常见问题与解答
在这一部分，我们将解答一些常见问题：

Q: 概率分布和概率模型有什么区别？

A: 概率分布是一种描述随机变量取值概率的数学模型，它可以用概率密度函数或者概率质量函数来描述。概率模型是一种将随机变量映射到概率分布的数学框架。

Q: 如何选择合适的概率分布？

A: 选择合适的概率分布需要考虑数据的特点和应用需求。例如，如果数据具有正态分布特征，可以选择正态分布；如果数据具有摇动分布特征，可以选择摇动分布。

Q: 如何计算概率分布的参数？

A: 可以使用最大似然估计（MLE）或者贝叶斯估计（BE）来计算概率分布的参数。MLE是一种基于数据最大化似然函数的方法，而BE是一种基于先验概率和数据后验概率的方法。
