### 4.1 自动生成图表

```python
import matplotlib.pyplot as plt

# 生成随机数据
x = [i for i in range(10)]
y1 = [i * 2 for i in x]
y2 = [i * 3 for i in x]

# 选择合适的图表类型
plt.figure()
plt.plot(x, y1, label='y1')
plt.plot(x, y2, label='y2')

# 设置图表参数
plt.title('自动生成图表')
plt.xlabel('x')
plt.ylabel('y')
plt.legend()

# 生成图表
plt.show()
```

### 4.2 数据预处理

```python
import pandas as pd

# 加载原始数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.dropna()
data = data.drop_duplicates()

# 数据转换
data['date'] = pd.to_datetime(data['date'])

# 数据归一化
data['value'] = (data['value'] - data['value'].mean()) / data['value'].std()
```

### 4.3 交互式可视化

```python

# 处理用户输入
fig.show()
```

### 4.4 智能建议

```python
from sklearn.linear_model import LinearRegression

# 加载原始数据
data = pd.read_csv('data.csv')

# 智能建议：推荐图表类型
if data['value'].std() < 1e-6:
    print('推荐图表类型：直线图')
else:
    print('推荐图表类型：折线图')

# 智能建议：推荐图表参数
model = LinearRegression().fit(data[['date']], data['value'])
print('推荐图表参数：斜率：{:.2f}, 截距：{:.2f}'.format(model.coef_[0], model.intercept_))
```
