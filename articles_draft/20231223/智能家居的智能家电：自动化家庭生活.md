# 4.具体代码实例和详细解释说明
## 4.1 智能家电的代码实例
以一个智能空调为例，我们来看一个简单的智能家电控制代码实例：

```python
import time
import requests

# 设备连接
url = 'http://192.168.1.100/api/control'
headers = {'Content-Type': 'application/json'}

# 数据收集
def get_temperature():
    response = requests.get(url, headers=headers)
    return response.json()['temperature']

# 数据处理
def adjust_temperature(target_temperature):
    control_data = {
        'target_temperature': target_temperature,
        'operation': 'adjust'
    }
    response = requests.post(url, json=control_data, headers=headers)
    return response.json()['status']

# 控制执行
current_temperature = get_temperature()
target_temperature = 25
adjust_temperature(target_temperature)

# 人机交互
print('当前温度：', current_temperature, '℃')
print('目标温度：', target_temperature, '℃')
```

在这个代码实例中，我们首先通过HTTP请求连接到智能空调设备，然后获取当前温度，对比目标温度，如果不同，则发送调整温度的控制命令。最后，通过打印当前温度和目标温度，实现与用户的交互。

## 4.2 智能家居系统的代码实例
以一个智能家居网关为例，我们来看一个简单的智能家居系统控制代码实例：

```python
import time
import requests

# 设备连接
url = 'http://192.168.1.100/api/gateway'
headers = {'Content-Type': 'application/json'}

# 数据收集
def get_device_list():
    response = requests.get(url, headers=headers)
    return response.json()['devices']

# 数据处理
def turn_on_device(device_id):
    control_data = {
        'device_id': device_id,
        'operation': 'turn_on'
    }
    response = requests.post(url, json=control_data, headers=headers)
    return response.json()['status']

# 控制执行
device_list = get_device_list()
device_id = device_list[0]['id']
turn_on_device(device_id)

# 人机交互
print('设备已开启')
```

在这个代码实例中，我们首先通过HTTP请求连接到智能家居网关，然后获取所有设备列表，选择一个设备进行开启。最后，通过打印设备已开启的信息，实现与用户的交互。
