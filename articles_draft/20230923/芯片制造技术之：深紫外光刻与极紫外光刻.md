# 2.深紫外光刻原理及其制作过程
## 2.1 深紫外光刻原理
深紫外光刻原理是指通过对待击穿材料的微透镜或者电镜，在光照条件下使得被射出的入射光能够穿透材料表面的孔隙，并留下深红色的晶体。在制作二维结构化件时，用深紫外光刻技术可以使得结构紧凑而不至于容易打断，并且保持较高的精度，同时还可以在深层次上施加电压。下面就深紫外光刻原理、制作方法及其步骤做一些介绍。
### 2.1.1 深紫外光刻原理简介
深紫外光刻的原理是利用超声波在晶体表面的感应作用，激发离子流失以形成一种放射状的晶体，从而使得晶体表面出现大量紫外线反射。由于离子的浓度低于原子的浓度，所以只会发生在晶格中微米级别的位置上，因此称为“深紫外光刻”。

在深紫外光刻过程中，首先需要准备好所需的工具，如晶体制作模板、晶体切割工具、微透镜、旋转装置等。然后，使用晶体模板对晶体进行切割，再通过微透镜在晶体表面张开一定宽度的孔隙，用于紫外光刻。通过激活感光元件生成紫外线，在孔隙内形成紫外晶体。最后，将紫外晶体整体旋转一定角度，将紫外晶体分离出来，即完成深紫外光刻。
## 2.2 深紫外光刻制作步骤
### 2.2.1 制作前准备工作
首先进行第一道准备工作，就是材料购买和预处理。首先需要购买紫外晶体模具，一般都会有多款产品供选择，这里推荐购买德州仪器（TDK）产品，它提供的模具质量保证绝对有保证，质量出众。其次，需对模具进行预处理，一般需要清洗、锈蚀、去除粘着剂，然后加入锌镉金属酸、碳酸钾溶液，使其完全溶解。
### 2.2.2 模具清洗、焊接
制作完模具后，第一步应该是进行模具清洗、焊接，目的是为了使模具处于良好的状态，此时模具没有受到其他因素的影响。清洗模具的方法可以是用布巾擦拭模具表面，另一个方法则是将模具放在浴缸中浸泡2-3小时左右。之后在浴缸中进行烘烤，保证模具的烘焙质量。焊接模具的时候，记得注意焊接模具与模具之间的间距不要太小，否则可能会导致模具损坏。
### 2.2.3 模具修补
制作完模具后，模具需要进行修补，修补的目的是为了消除模具中的缺陷，提升模具的抵抗力。一般的修补方法有透镜修补、侧切修补等。透镜修补时，需要将模具中的一块薄片放在透镜面上，让透镜扫到模具内部的每一处缺陷，然后将修正后的模具放回原位。侧切修补时，就是将模具从一端切断，然后在另一端填补，消除了模具中间存在的裂纹。
### 2.2.4 模具切割
模具修补完毕后，模具就可以进行切割了，切割之前需要考虑模具的尺寸大小和切割数量，切割后得到的都是清晰、无杂质、耐腐蚀的切片。对于大的模具，通常需要切割成四个较小的切片，这样才能更方便进行深紫外光刻。切割时需要注意选择准确且均匀的切割深度，避免切割出较大的尖孔。
### 2.2.5 模具组装
切割完模具后，模具需要进行组装。如果一张模具可以制作多个结构，那么可以直接将各个切片依次堆叠起来，但对于单个结构来说，还需要根据需求进行组装。组装时需要将模具中的一块薄片对准模具的下部中心，按照一个固定方向进行固定。另外，也可以选择将模具放置于一个工作台上，然后配合刮削工具进行模具的刮削、切割。
### 2.2.6 电源接通与深紫外光刻器件连接
电源的连接对于深紫外光刻的制作非常重要，因为电源会产生紫外光，紫外光能够对晶体产生很强的反射效应，从而制作成品中的紫外晶体。通常情况下，深紫外光刻需要使用特定的制作设备，该设备能够帮助制作者把光线投射到晶体表面，并激活晶体的自由破碎。在连接电源之前，需要确认制作设备的类型、功率和空间要求。
### 2.2.7 深紫外光刻器件选取
深紫外光刻器件的选取决定着制作过程中的实际性能。制作二维结构化件最关键的一步就是深紫外光刻过程，因此选择恰当的深紫外光刻器件对制作的成功至关重要。通常情况下，制作深紫外光刻需要采用高功率的紫外光刻器件，例如760nm或890nm的红外光刻机或470nm或540nm的探测器。同时，还需要搭配紫外线调节器，用于调节光线的颜色、强度等参数。除此之外，还需要注意制作设备的防护措施，比如说防止反复震动、保护电池等。
### 2.2.8 深紫外光刻激光系统设置
深紫外光刻的最后一步，便是在设定激光系统。激光系统包括聚焦装置和放大装置。聚焦装置用于设定系统的工作波长，也就是测量制作成品的有效波长，因此，通常都需要使用更高的波长进行制作。而放大装置则用于提高激光束的大小，使得深紫外光刻制作过程更加细腻，达到高精度的目的。在设置激光系统时，还需要注意光路的长度是否合适、滤波器是否选择合适、各个装置是否安装正确、有无漏电、测量功能是否正常运行等。
### 2.2.9 制作过程总结
1.制作前准备工作：购买和预处理模具；
2.模具清洗、焊接：在浴缸中进行烘烤和焊接；
3.模具修补：透镜修补或侧切修补；
4.模具切割：选择准确且均匀的切割深度；
5.模具组装：按照顺序放置模具；
6.电源接通与深紫外光刻器件连接：确认制作设备、功率、空间要求；
7.深紫外光刻器件选取：选择高功率的紫外光刻器件和紫外线调节器；
8.深紫外光刻激光系统设置：选择合适的激光系统；
9.激光系统测试：检查激光系统的功能。
### 2.2.10 深紫外光刻实践环节
深紫外光刻是一个复杂、繁琐、耗时的制作过程，但只要认真对待，不断进行实践，并熟悉相关技能，深刻理解深紫外光刻原理，就可以轻松完成制作任务。下面给大家列举一下一些实践环节，以帮助大家快速入手实践：

1.模具购买和预处理：德州仪器提供的模具质量绝对值得信赖，它们的产品经过严格检测，保证原创和后续工程质量。可购买的模具有蓝图DIY模具、白模具、环形模具等。

2.模具清洗、焊接：模具清洗、焊接比较简单，直接用布巾擦拭即可，但最好还是在浴缸中烘烤一段时间，然后再进行焊接。尤其注意模具间距是否合适。

3.模具修补：模具修补的目的是为了消除模具中的缺陷，从而增加模具的抵抗力。一般有透镜修补、侧切修补两种方式。透镜修补时，需要将模具里的一个薄片在透镜面上扫到每个缺陷，再放回原位。侧切修补时，就是将模具从一端切断，然后在另一端填补，消除了模具中间存在的裂纹。

4.模具切割：模具切割的目的就是将模具切割成足够小的大小，这才可以进行深紫外光刻。切割时要选择正确且均匀的切割深度，切割出来的切片不要太大，否则容易腐蚀，导致模具切割混乱。切割时也要注意模具尺寸大小是否合适，尺寸越大，切割效率越低。

5.模具组装：模具组装需要满足不同结构的需求，但最终目的都是制作紫外晶体，因此制作时需要按照固定的工作流程来。组装时需要注意模具的摆放方向和固定方向，以及模具与切片之间的距离。

6.电源接通与深紫外光刻器件连接：深紫外光刻需要使用的器件都具有不同程度的复杂性，在选择和连接时都需要十分小心。建议最好采用带有显示屏的紫外光刻机器，这样可以直观地看到制作进度。

7.深紫外光刻器件选取：高功率的紫外光刻器件和紫外线调节器是深紫外光刻制作过程的基础。为了取得良好的制作效果，可以使用专门设计的激光光刻机，这样的光刻机一般都具有自动调节能力。紫外线调节器的作用是调整光线的亮度，使得紫外晶体光滑而不会生锈。

8.深紫外光刻激光系统设置：制作时，激光系统需要根据不同的工艺流程来设定。通常情况下，制作二维结构化件不需要太大的激光束，但是在较大型的工艺流程中，可能需要更大的激光束。在设置激光系统时，还需要注意光路的长度是否合适、滤波器是否选择合适、各个装置是否安装正确、有无漏电、测量功能是否正常运行等。

# 3.什么是极紫外光刻？为什么要进行极紫外光刻？极紫外光刻如何实现？
## 3.1 什么是极紫外光刻？
极紫外光刻（EIT）技术是通过加热电压对晶体表面的极化、干涉、刺激等产生有机的晶体，从而获得二维结构化件的制作。所谓极化就是晶体中离子的位置分布在晶体表面的某些平行平面的极点附近。人们早期对极化和晶体学的研究并非从宏观层面详细探讨，而是局限于晶体表面受到电场和磁场力的影响。从微观层面看，晶体的晶胞结构可以通过化学原理对离子的运动进行控制，从而将晶体划分为极点附近的区域和远离极点的区域，这种控制离子运动的方式称为“费米效应”（Fermi Effect）。因此，晶体的自然极化是极紫外光刻制作的基石。
## 3.2 为什么要进行极紫外光刻？
通常情况下，制作二维结构化件都采用传统的深紫外光刻（DIC）技术。但是，随着技术的发展，越来越多的研究者发现，传统的DIC制作工艺存在诸多缺陷，最突出的问题之一就是晶体的弯曲程度不够，导致晶体表面无法形成紫外晶体。于是，许多人开始关注DIC技术存在的问题，希望找到一种新的制作技术能够解决这些问题。所以，极紫外光刻（EIT）技术应运而生。
## 3.3 极紫外光刻如何实现？
极紫外光刻基于晶体的自然极化，制作过程如下：

1. 待制作晶体预处理
首先需要进行预处理，预处理的目的是消除水分、氧化物、杂质，保证晶体的自然极化。

2. 提供电压
第二步是使用提供电压器件供电，使晶体表面发生自然极化。这个电压的大小和频率都需要根据制作成品的品质、大小和要求来确定。

3. 自然极化
第三步是使晶体表面发生自然极化。这时候还不能称为真正的自然极化，只有晶体内部的局部区域发生自然极化。这个范围一般是靠近晶核的，不到晶核边缘的区域暂时还不能称为自然极化。但随着电压的增大，整个晶体就会发生自然极化，并且越来越多的晶核会向外扩张。

4. 制造紫外晶体
第四步就是制造紫外晶体。有两种制造紫外晶体的工艺，即傅立叶变换（Fourier Transform）法和格林函数法。傅立叶变换法需要用激光照射晶体表面，将光子束在晶体表面反射出，并转换为电磁波，然后用探测器测量电磁波的幅度和相位信息。格林函数法不需要用激光照射晶体表面，而是利用晶体的属性来计算出晶体内部的能量密度分布情况，从而得知晶体的内部电荷分布情况。两种制造方法都可以获得同样的结果，都是晶体中所有电子的电荷分布情况。但是，格林函数法计算电荷分布需要更多的时间，且只能在晶体的限定范围内进行。傅立叶变换法的优点是不需要进入晶体内部，测量效率高，而且精度也高。

5. 电路设计与测试
之后，电路设计与测试就和普通的模拟电路一样。将晶体制作出的晶体件作为元器件进行连接和组合，完成电路的设计与测试。测试时需要注意电气规范、测试方案的制订等。测试完成后，即可生产出一个制作良好的二维结构化件。
## 3.4 EIT与DIC区别和联系？
极紫外光刻与传统的深紫外光刻（DIC）制作工艺之间的区别有以下几点：

1. 电压作用
DIC中的晶体表面受到的电压是由晶体表面的马达驱动产生的，一般占据了一半以上的电压，因此，它的制作速度慢。而EIT制作的晶体表面受到的电压不是由马达驱动，而是由微电容器和其他辅助元件提供。因此，它的制作速度快，制作出来的晶体结构拥有更高的精度。

2. 器件类型和制造工艺
传统的深紫外光刻（DIC）技术需要使用电压驱动的光刻机，它的制造工艺是无纺织结构。而极紫外光刻（EIT）技术则需要使用超频微电子器件，它的制造工艺是带有金属材料的金属构件。

3. 制作方式
传统的深紫外光刻（DIC）技术需要用微透镜与光刻机组合一起工作，即先将晶体表面吹出微弱的紫外光，再将光线投射到晶体表面，产生紫外晶体。而极紫外光刻（EIT）技术则不需要用光刻机，而是用微电容器和其他辅助元件来驱动晶体发生自然极化，并使得晶体发生紫外晶体的变化。

4. 制作温度
传统的深紫外光刻（DIC）技术的制作温度较低，在室温下能够制作出较大的晶体。而极紫外光刻（EIT）技术的制作温度可以达到0到1000℃之间，只要能够实现电压驱动就可以制作出较为精准的晶体。

综上所述，EIT与DIC的区别和联系如下：

1. 电压作用：EIT技术中的晶体表面受到的电压由微电子容器和其他辅助元件提供，因此，它的制作速度快，而DIC制作的晶体表面受到的电压由晶体表面的马达驱动产生，它的制作速度慢。

2. 器件类型和制造工艺：EIT技术中的器件需要使用超频微电子器件，因此其制造工艺是带有金属材料的金属构件，而传统的深紫外光刻（DIC）技术需要使用电压驱动的光刻机，其制造工艺是无纺织结构。

3. 制作方式：EIT技术中的制作方式不需要用光刻机，而是用微电容器和其他辅助元件来驱动晶体发生自然极化，并使得晶体发生紫外晶体的变化，因此它制作方式不同于传统的深紫外光刻（DIC）。

4. 制作温度：EIT技术中的制作温度可以达到0到1000℃之间，只要能够实现电压驱动就可以制作出较为精准的晶体，而传统的深紫外光刻（DIC）技术的制作温度较低，在室温下能够制作出较大的晶体。
