# 2.背景介绍
## 2.1 数据隐私
数据隐私（Data Privacy）是指由个人或组织提取或者利用一些个人信息，而不告知该个人的法律主体的一种行为方式。当今，由于信息技术的普及和社会经济的快速发展，越来越多的人将自己的个人信息用于各种目的。数据隐私是一个重要的法律要求，并受到越来越多的法律和行政部门的关切。数据隐私并不是绝对的，它应该是有约束力的。数据隐私就是关于“我”的信息，在这种信息共享中，涉及到个人的隐私权、个人的自由、个人的自由意志、个人的个人生活、组织的内部、组织的外部……所有人的隐私权都被侵犯时，才会考虑数据隐私。数据隐私也是个人信息保护的一个重要组成部分。
数据隐私涉及的数据可以分为三类：

1. 静态数据：静态数据是在收集时就存在的原始数据，也就是不会因时间的推移而发生变化的数据，如姓名、身份证号码、联系方式等；
2. 可变数据：可变数据指的是具有生命周期特征的数据，如住宅地址、工作单位、电子邮箱等，它们在使用前会对其进行收集和储存，并且随着使用过程的不断推进，数据也会不断更新，如动态电话簿、博客、留言板等；
3. 个人数据：个人数据是指能够单独或者与其他信息结合识别特定自然人身份的一切数据，如生物识别信息、精神病史记录、银行卡信息、居住地信息、食谱、照片、记录、音乐等。
## 2.2 数据隐私定义
数据隐私，一般来说，指的是保护用户个人信息的权利和义务，即组织、企业、政府应通过某种手段，保障用户个人信息不被非法使用、泄露、披露、篡改、毁损或丢失。
数据隐私的定义较为宽泛，它涵盖的内容比较广泛，从基础的个人信息保护，到商业模式的隐私保护，再到数据安全中的数据隐私控制，甚至还有跨境数据传输中的隐私保护。但是一般认为，数据隐私的核心要素还是个人信息的保护。数据隐私也是信息安全、网络安全的一个重要方面，是个人信息保护的保护伞。
## 2.3 数据隐私的价值
数据隐私的价值首先体现在信息安全上，只有保护好用户个人信息，才能确保用户的隐私安全，防止信息泄露、违法使用。其次，数据隐私还可以促进信息共享和价值观的形成，让不同的用户群体产生共鸣，从而促进社会进步。第三，数据隐私对于国家经济发展和社会进步有着重要作用。如今，越来越多的企业和机构都在使用大数据分析、人工智能技术、云计算技术，大数据存储产生海量数据，这些数据存储在中心化和联邦的服务器上，很容易造成信息泄露、数据泄露等隐私问题。如果没有数据隐私保护，那么这些隐私数据就会很容易遭到窃取、滥用。
## 2.4 数据隐私的法律规范
目前，中国有多部法律、法规、标准和文件依据数据隐私的保护，如国务院办公厅、国土资源部、国家信息管理局、公安部、工信部、福建省政法委、全国人大常委会、全国计算机信息系统安全专项检查评估基准体系，以及《公民个人信息保护条例》等。
根据《国务院办公厅关于印发〈信息安全与个人信息保护条例〉的通知》，2003年9月2日起施行，《信息安全与个人信息保护条例》(征求意见稿)试点期间内，修订后的《信息安全与个人信息保护条例》正式发布，将健全数据安全、个人信息保护制度、信息技术措施和监督管理，保障公民、法人和其他组织依法行使个人信息权利。除此之外，《信息安全与个人信息保护条例》还有保障知识产权保护、消费者权益维护、信息公开及网站运营秩序维护等诸多内容。
# 3.云计算数据隐私与合规
## 3.1 云计算数据隐私与合规概述
云计算（Cloud Computing），也称为“一云通百”，是利用网络计算机（云端）、存储空间、软件服务和相关的通信服务等资源，建立属于用户的服务器集群，运行应用程序、服务和网络环境等。这样的计算资源可以随时按需访问和使用，从而实现高度的伸缩性、灵活性和弹性。云计算显著的优势在于按需访问、按使用付费，具备极高的可靠性和可用性，可以支持快速响应的业务需求。虽然云计算确实提供了新的便利，但同时也带来了新的复杂性。其中，数据隐私与合规是云计算的两个突出难题。
数据隐私与合规是指数据的所有者对个人数据隐私的承诺，以及数据使用者对个人数据的处理方式、存储位置和安全保障等规则的承诺。云计算数据隐私与合规实际上是云计算安全保障的两个重要环节。
### 3.1.1 云计算数据隐私与合规背景
为了保障用户的个人信息在云计算平台上处理、保密，云计算公司一般会制定相应的隐私保护和合规协议。但是，云计算数据隐私与合规并不完全一样，因为云计算平台尚处于刚刚出现阶段，如何充分地保障数据隐私与合规尚未得到充分认识。因此，为了更好的保障云计算数据隐私与合规，云计算公司需要根据如下背景进行辨析：

1. 发展阶段：云计算刚刚兴起，作为新兴的一种计算模式，并没有进行过严格的隐私与合规的教育和培训，因此很多公司在制定隐私与合规制度时还缺乏足够的经验。比如，腾讯公司在“云计算数据隐私保护与合规”白皮书中就提到，在云计算平台上收集到的用户数据，可能会保存很久的时间，可能存在被复制、违规使用的风险。

2. 技术阶段：云计算平台带来了新的技术革命，涌现了一批新的服务和产品，例如云服务器、虚拟机、容器等，这些新技术带来的巨大的自由度和弹性，使得云计算平台上的用户数据不仅具有新颖的价值，而且具有众多的潜在危险性。

3. 体制阶段：云计算的发展和壮大，也引起了行政部门和政府对数据隐私与合规的重视程度的提高，并且出现了不同类型的合规要求。比如，苹果公司推出的iCloud，要求对用户数据进行安全、一致的管理；联想集团旗下的墨迹天气APP，收集用户数据，要求用户不分享；微信和支付宝公司都推出了严格的个人信息保护制度。

综上所述，云计算数据隐私与合规存在三个阶段性问题：

1. 尚未成熟：云计算刚刚出现，相关的技术和合规问题仍在逐步解决中；

2. 规范模糊：目前，云计算平台上收集、处理、存储和使用用户数据的方式还没有达到标准化，因此合规的方式也没有统一的模板。

3. 缺少机制：目前，没有成熟的机制可以确保云计算平台上数据的真实性和安全性，缺乏真正的管控力度。
## 3.2 云计算数据隐私与合规定义
云计算数据隐私与合规，是指云计算平台上的用户数据进行保护、管理和使用过程中，所涉及的隐私风险、数据泄露、违规使用等问题。云计算数据隐私与合规的内容非常广泛，从数据存储、数据处理、数据共享、数据删除、数据传输等多个方面进行处理，对用户隐私和合规的保障非常重要。
### 3.2.1 数据存储
数据存储是云计算数据隐私与合规中最为重要的环节。它涉及用户数据在云计算平台的生命周期，以及存储、加密、访问权限、数据删除等方面的保障措施。具体说来，数据存储可以分为以下四个方面：

1. 持久存储：数据的持久存储指的是将用户数据长期存储至硬盘上，并且不能删除或修改，只有在用户删除账号或退出登录后，才会被彻底删除或不可访问。

2. 异地冗余存储：数据的异地冗余存储是指将用户数据存储至另一地域，通过互联网连接至云计算平台，实现数据的快速冗余备份。

3. 多重加密：数据的多重加密是指对用户数据进行加密处理，同时对相同数据重复加密，以提升云计算平台数据的安全性。

4. 数据权限控制：数据的权限控制是指设置不同级别的用户权限，限制不同用户对用户数据进行不同的操作，确保用户数据安全。

### 3.2.2 数据处理
数据处理是云计算数据隐私与合规中另一重要环节。它主要是指云计算平台对用户数据进行分析、处理、转换、统计、查询等过程中，所涉及的风险、风险检测、风险预防、风险排查、数据恢复、数据违规处理等方面的保障措施。具体来说，数据处理可以分为以下四个方面：

1. 消息加密：消息加密是指云计算平台上数据的传输过程进行加密，防止第三方截获、篡改用户数据的情况。

2. 日志审计：日志审计是指记录云计算平台上用户数据的操作行为，通过对日志进行分析和监测，检测异常行为的发生。

3. 智能数据分析：智能数据分析是指云计算平台对用户数据进行分析，识别、跟踪和预测用户行为习惯，并提供个性化的服务。

4. 数据泄漏保护：数据泄漏保护是指云计算平台上用户数据的泄露风险，包括用户数据被非法盗用、泄露、篡改、销毁等风险，需要有相应的应急处理措施。

### 3.2.3 数据共享
数据共享是云计算数据隐私与合规中，对数据拥有者和数据使用者双方的权利和义务，包括使用数据的权利、保护数据的权利、获取数据的权利、转移数据的权利、删除数据的权利等。具体来说，数据共享可以分为以下四个方面：

1. 用户权限管理：用户权限管理是指对不同用户之间的权限进行管理，确保用户数据安全。

2. 数据披露限制：数据披露限制是指限制云计算平台上用户数据的泄露，防止数据被错误使用、盗用、泄露、遗忘等。

3. 合规共享标准：合规共享标准是指明确云计算平台上用户数据的类型、使用范围、使用目的、使用方法等，并将其转换为标准协议，确保云计算平台上用户数据的合规共享。

4. 数据共享监控：数据共享监控是指对云计算平台上不同用户之间共享数据的监控，确保用户数据安全。

### 3.2.4 数据删除
数据删除是云计算数据隐私与合规中，对云计算平台上用户数据的删除、注销或匿名化处理过程中的保护措施。数据删除主要是指对用户数据在云计算平台上的永久性或长期删除、改写或匿名化处理过程中的保护措施。具体来说，数据删除可以分为以下四个方面：

1. 数据时效性：数据时效性是指云计算平台上用户数据的保留时限，超过时限的数据需要被删除。

2. 数据完整性：数据完整性是指云计算平台上用户数据的完整性，不存在无效数据、数据缺失等。

3. 数据隐私保护义务：数据隐私保护义务是指云计算平台上用户数据使用方应遵守数据使用条件，不得泄露用户数据或超出使用范围。

4. 数据生命周期管理：数据生命周期管理是指云计算平台上用户数据的生命周期管理过程，确保用户数据的安全和保密。

### 3.2.5 数据传输
数据传输是云计算数据隐私与合规中，指云计算平台上用户数据传输过程中，所涉及的风险、安全措施、监测能力等方面的内容。具体来说，数据传输可以分为以下四个方面：

1. 数据加密传输：数据加密传输是指云计算平台上用户数据在传输过程中采用加密、密钥管理等技术，保障用户数据的安全。

2. 数据传输合规审核：数据传输合规审核是指审核云计算平台上用户数据在传输过程中的合规情况，确保符合相关法律法规、监管要求。

3. 网络监控技术：网络监控技术是指对云计算平台上用户数据传输过程中的网络状况进行监控，确保数据传输的安全。

4. 数据传输工具：数据传输工具是指云计算平台上数据传输过程中采用的工具，确保数据传输的准确性、可靠性和效率。

### 3.2.6 数据保险箱
数据保险箱是云计算数据隐私与合规中，对于用户数据的不满意、丢失、泄露等事件，将用户数据暂时存放在一个专门的隔离区域，待事情解决后再将其销毁。这么做的目的是为了防止数据遭到未经授权的泄露。数据保险箱具有一定的数据保存时间，一般不超过七天。
