## 4.1 代价曲线实例

假设我们有一个简单的推理推测算法，它接受一个整数列表作为输入，并返回这个列表中最大的数。我们想要绘制这个算法在不同输入大小下的代价曲线。

首先，我们需要选择一个或多个算法进行性能测试。在这个例子中，我们只有一个算法。然后，我们需要设定不同的输入大小，以便对算法性能进行全面的测试。在这个例子中，我们可以设定输入大小为 10、100、1000、10000 等。

接下来，我们需要测试算法的性能。在这个例子中，我们可以使用 Python 的 time 模块来测试算法的时间消耗。我们可以对每个输入大小进行多次测试，并计算平均值。

最后，我们可以将测试结果绘制在坐标系中，以便直观地观察算法在不同输入大小下的性能。在这个例子中，我们可以使用 Python 的 matplotlib 库来绘制代价曲线。

## 4.2 代价曲线实现代码

以下是一个简单的 Python 代码实例，用于绘制代价曲线：

```python
import time
import matplotlib.pyplot as plt

# 定义算法函数
def find_max(lst):
    max_val = lst[0]
    for num in lst:
        if num > max_val:
            max_val = num
    return max_val

# 设定输入大小
input_sizes = [10, 100, 1000, 10000]

# 测试算法性能
times = []
for size in input_sizes:
    lst = [i for i in range(size)]
    start_time = time.time()
    find_max(lst)
    end_time = time.time()
    time_cost = end_time - start_time
    times.append(time_cost)

# 绘制代价曲线
plt.plot(input_sizes, times, marker='o')
plt.xlabel('Input Size')
plt.ylabel('Time Cost')
plt.title('Cost Curve of Find Max Algorithm')
plt.grid(True)
plt.show()
```

在这个代码实例中，我们首先定义了一个 find\_max 函数，用于找到列表中最大的数。然后，我们设定了不同的输入大小，并对算法性能进行了测试。最后，我们使用 matplotlib 库绘制了代价曲线。
