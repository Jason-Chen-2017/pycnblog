### 2.1 欧氏距离

欧氏距离是一种常用的距离度量，它可以衡量两点之间的距离，通常用于欧几里得空间中。欧氏距离的公式为：

$$
d(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2 + \cdots + (x_n - y_n)^2}
$$

其中，$x = (x_1, x_2, \cdots, x_n)$ 和 $y = (y_1, y_2, \cdots, y_n)$ 是欧几里得空间中的两个点，$d(x, y)$ 是它们之间的欧氏距离。

## 4. 具体代码实例和详细解释说明

### 4.1 用户相似性计算

以下是一个用户相似性计算的具体代码实例：

```python
import numpy as np

# 用户的历史行为数据
users = [
    [1, 2, 3],
    [2, 3, 4],
    [3, 4, 5]
]

# 计算用户之间的欧氏距离
def euclidean_distance(u, v):
    return np.sqrt(np.sum((u - v) ** 2))

# 计算用户相似性
def user_similarity(users):
    similarity = []
    for i in range(len(users)):
        for j in range(i + 1, len(users)):
            distance = euclidean_distance(users[i], users[j])
            similarity.append((i, j, 1 - distance / np.sqrt(np.sum(users[i] ** 2) * np.sum(users[j] ** 2))))
    return similarity

# 输出用户相似性
print(user_similarity(users))
```
