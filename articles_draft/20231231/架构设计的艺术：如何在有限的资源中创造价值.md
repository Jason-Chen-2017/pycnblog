## 4.1 分布式算法实例

在分布式算法中，我们可以使用 Hadoop 来实现大数据处理任务。以下是一个简单的 Hadoop 示例代码：

```python
from hadoop.mapreduce import Mapper, Reducer

class WordCountMapper(Mapper):
    def map(self, line, context):
        words = line.split()
        for word in words:
            context.emit(word, 1)

class WordCountReducer(Reducer):
    def reduce(self, key, values, context):
        count = sum(values)
        context.write(key, count)

if __name__ == '__main__':
    input_file = 'input.txt'
    output_file = 'output.txt'
    Mapper = WordCountMapper
    Reducer = WordCountReducer
    hadoop.job(input_file, output_file)
```

在这个示例中，我们使用 Hadoop 来计算一个文本文件中每个单词的出现次数。首先，我们定义了一个 Mapper 类，它负责将输入数据划分为多个子任务。然后，我们定义了一个 Reducer 类，它负责将子任务的结果合并到最终结果中。最后，我们使用 Hadoop 的 job 函数来运行这个任务。

```python
def gradient_descent(X, y, learning_rate=0.01, iterations=100):
    m, n = X.shape
    theta = np.zeros(n)
    for _ in range(iterations):
        predictions = X.dot(theta)
        errors = predictions - y
        theta -= learning_rate * (X.T.dot(errors)) / m
    return theta

# 加载数据

X = data[:, :-1]
y = data[:, -1]

# 使用梯度下降算法训练模型

```

在这个示例中，我们使用 Gradient Descent 来训练一个简单的线性模型。首先，我们加载了数据，并将其划分为特征和目标变量。然后，我们使用梯度下降算法来训练模型，其中 learning_rate 和 iterations 是可选参数。最后，我们返回训练好的模型参数。
