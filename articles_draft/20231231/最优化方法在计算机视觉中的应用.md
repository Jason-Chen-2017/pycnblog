## 4.1 线性回归

线性回归是一种简单的最优化方法，可以用于解决线性模型中的问题。以下是一个Python代码实例：

```python
import numpy as np

# 数据
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])

# 目标函数
def f(beta):
    return np.sum((y - (beta[0] + beta[1]*x))**2)

# 梯度下降
def gradient_descent(x, y, learning_rate, iterations):
    beta = np.zeros(2)
    for i in range(iterations):
        gradients = 2 * (y - (beta[0] + beta[1]*x))
        beta -= learning_rate * gradients
    return beta

# 优化
beta = gradient_descent(x, y, learning_rate=0.01, iterations=1000)
print("beta:", beta)
```

在这个例子中，我们使用梯度下降法求解线性回归问题。首先，我们定义了一个目标函数`f`，它是一个二次项，用于衡量模型的误差。然后，我们实现了一个`gradient_descent`函数，它使用梯度下降法求解目标函数。最后，我们调用`gradient_descent`函数进行优化，并打印出最终的`beta`值。

## 4.2 逻辑回归

逻辑回归是一种用于解决二分类问题的最优化方法。以下是一个Python代码实例：

```python
import numpy as np

# 数据
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([0, 0, 1, 1])

# 目标函数
def f(theta):
    return np.sum(y * (np.log(1 + np.exp(theta.dot(X))) - np.log(1 + np.exp(-theta.dot(X)))))

# 梯度下降
def gradient_descent(X, y, learning_rate, iterations):
    theta = np.zeros(2)
    for i in range(iterations):
        gradients = (X.T.dot(y * (np.exp(theta.dot(X))) - (1 - y) * (np.exp(-theta.dot(X))))) / X.shape[0]
        theta -= learning_rate * gradients
    return theta

# 优化
theta = gradient_descent(X, y, learning_rate=0.01, iterations=1000)
print("theta:", theta)
```

在这个例子中，我们使用梯度下降法求解逻辑回归问题。首先，我们定义了一个目标函数`f`，它是一个对数损失函数，用于衡量模型的误差。然后，我们实现了一个`gradient_descent`函数，它使用梯度下降法求解目标函数。最后，我们调用`gradient_descent`函数进行优化，并打印出最终的`theta`值。
