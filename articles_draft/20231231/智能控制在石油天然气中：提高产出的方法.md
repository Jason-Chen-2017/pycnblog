$$

$$

其中，$p(k+1)$ 表示未来时刻的井孔压力，$p(k)$ 表示当前时刻的井孔压力，$T_s$ 表示采样时间，$u(k)$ 表示控制输入。

具体代码实例如下：

```python
import numpy as np

def model_predictive_control(p_ref, T_s, p_init, u_max):
    p = p_init
    u = 0
    while p < p_ref:
        p_future = p + T_s * u
        if abs(p_future - p_ref) < 0.1:
            break
        if -u_max <= u <= u_max:
            u = u_max
        elif u_max < u:
            u = u - 0.1
        else:
            u = u + 0.1
        p = p_future
    return p, u

p_ref = 100  # 目标井孔压力
```
