# 4.具体代码实例和详细解释说明

## 4.1 线性规划示例

### 4.1.1 问题描述

在一个物流网络中，有 $n$ 个城市，每个城市之间有 $m$ 条物流路线，每条路线的容量为 $c$。现在需要从城市 $s$ 向城市 $t$ 运输 $d$ 个单位货物，求最短路径。

### 4.1.2 数学模型

根据问题描述，可以建立一个线性规划模型，目标函数为运输成本，约束条件为路线容量和货物要求。具体来说，目标函数可以表示为：

$$
minimize \quad \sum_{i=1}^{n} \sum_{j=1}^{n} c_{ij} x_{ij}
$$

约束条件可以表示为：

$$
\sum_{j=1}^{n} x_{ij} = d, \quad i=1,2,\cdots,n
$$

$$
x_{ij} \leq c_{ij}, \quad i,j=1,2,\cdots,n
$$

### 4.1.3 代码实现

```python
import numpy as np
from scipy.optimize import linprog

# 定义城市数量、路线容量、货物要求
n = 5
c = np.array([[0, 1, 1, 1, 1],
                 [1, 0, 1, 1, 1],
                 [1, 1, 0, 1, 1],
                 [1, 1, 1, 0, 1],
                 [1, 1, 1, 1, 0]])
d = np.array([1, 1, 1, 1, 1])

# 定义起始城市、终止城市
s = 0
t = 4

# 构建线性规划问题
A = np.vstack((np.ones(n), c)).T
b = np.hstack((d, np.zeros(n)))

# 求解线性规划问题
res = linprog(b, A_ub=A, bounds=(0, 1), method='highs')

# 输出结果
print('最短路径:', res.x)
print('最短路径成本:', res.fun)
```

## 4.2 动态规划示例

### 4.2.1 问题描述

在一个物流中心有 $n$ 个货物需求点，每个需求点需要运输 $d_i$ 个单位货物。现在有 $m$ 条物流路线，每条路线的容量为 $c_j$。求最短路径。

### 4.2.2 数学模型

根据问题描述，可以建立一个动态规划模型，目标函数为运输成本，约束条件为路线容量和货物要求。具体来说，状态转移方程可以表示为：

$$
dp[i][j] = min\{dp[i-1][j] + c_j, \quad j=1,2,\cdots,m\}
$$

### 4.2.3 代码实现

```python
import numpy as np

# 定义货物需求点数量、货物需求、物流路线容量
n = 5
d = np.array([1, 1, 1, 1, 1])
c = np.array([1, 1, 1, 1, 1])

# 定义起始城市、终止城市
s = 0
t = 4

# 初始化动态规划表
dp = np.zeros((n, n))

# 填充动态规划表
for i in range(n):
    for j in range(n):
        if i == 0:
            dp[i][j] = 0
        else:
            dp[i][j] = dp[i-1][j] + c[j]

# 输出结果
print('最短路径:', dp)
print('最短路径成本:', dp[t][t])
```
