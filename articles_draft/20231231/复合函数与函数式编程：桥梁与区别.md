## 2.1 复合函数

复合函数是函数式编程中的一个基本概念，它描述了如何将一个函数与另一个函数组合在一起，形成一个新的函数。这种组合方式可以用来构建更复杂的函数，并且可以帮助我们更好地理解和表达复杂的计算过程。

### 2.1.1 定义

复合函数可以通过以下定义来描述：

给定两个函数 $f(x)$ 和 $g(x)$，复合函数 $h(x)$ 可以通过以下方式定义：

$$
h(x) = f(g(x))
$$

### 2.1.2 例子

例如，我们可以使用复合函数来表示如何将一个数字从摄氏温度转换为华氏温度，然后再转换回摄氏温度。这里，我们可以定义两个函数 $f(x)$ 和 $g(x)$：

$$
f(x) = \frac{9}{5}x + 32
$$

$$
g(x) = \frac{5}{9}(x - 32)
$$

然后，我们可以使用复合函数来表示从摄氏温度转换为华氏温度的过程：

$$
h(x) = f(g(x)) = f(\frac{5}{9}(x - 32)) = \frac{9}{5}(\frac{5}{9}(x - 32)) + 32 = x
$$

这个例子展示了如何使用复合函数来表示从摄氏温度转换为华氏温度的过程，然后再转换回摄氏温度。这个过程可以通过使用复合函数来描述，这使得我们可以更好地理解和表达这个计算过程。

### 2.2.1 特点

函数式编程有以下几个特点：

1. 函数式编程强调使用函数来表示计算，而不是使用变量和过程。
2. 函数式编程中的函数是无状态的，这意味着函数不能修改其他变量的值。
3. 函数式编程中的函数是纯粹的，这意味着函数的结果仅仅依赖于其输入，不能依赖于外部状态或随机性。
4. 函数式编程中的函数是可组合的，这意味着函数可以被组合在一起，形成更复杂的函数。

### 2.2.2 优点

函数式编程有以下几个优点：

1. 函数式编程可以帮助我们更好地理解和表达计算过程，因为它强调使用函数来表示计算。

3. 函数式编程可以帮助我们更好地优化代码，因为它强调使用可组合的函数。

## 4.1 复合函数的代码实例

### 4.1.1 摄氏温度转华氏温度的代码实例

我们可以使用复合函数来表示从摄氏温度转换为华氏温度的过程，然后再转换回摄氏温度。这里，我们可以使用以下代码实例来表示这个过程：

```python
def celsius_to_fahrenheit(celsius):
    return (celsius * 9 / 5) + 32

def fahrenheit_to_celsius(fahrenheit):
    return (fahrenheit - 32) * 5 / 9

celsius = 100
fahrenheit = celsius_to_fahrenheit(celsius)
celsius_again = fahrenheit_to_celsius(fahrenheit)
print(celsius_again)
```

这个代码实例首先定义了两个函数 `celsius_to_fahrenheit` 和 `fahrenheit_to_celsius`，然后使用复合函数来表示从摄氏温度转换为华氏温度的过程，然后再转换回摄氏温度。最后，这个代码实例使用了复合函数来计算从摄氏温度转换为华氏温度再转换回摄氏温度的过程。

### 4.1.2 数学表达式求值的代码实例

我们可以使用复合函数来表示数学表达式求值的过程。这里，我们可以使用以下代码实例来表示这个过程：

```python
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    return x / y

def calculate(expression):
    tokens = expression.split()
    operators = ['+', '-', '*', '/']
    result = eval(tokens[0])
    for i in range(1, len(tokens)):
        if tokens[i] in operators:
            operator = tokens[i]
            right = eval(tokens[i + 1])
            if operator == '+':
                result = add(result, right)
            elif operator == '-':
                result = subtract(result, right)
            elif operator == '*':
                result = multiply(result, right)
            elif operator == '/':
                result = divide(result, right)
    return result

expression = "3 + 5 * 2 - 4 / 2"
result = calculate(expression)
print(result)
```

这个代码实例首先定义了四个数学运算函数 `add`、`subtract`、`multiply` 和 `divide`，然后使用复合函数来表示数学表达式求值的过程。最后，这个代码实例使用了复合函数来计算给定数学表达式的值。

我们可以使用函数式编程来表示数列求和的过程。这里，我们可以使用以下代码实例来表示这个过程：

```python
from functools import reduce

def add(x, y):
    return x + y
```
