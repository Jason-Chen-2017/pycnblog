## 4.1 粒子动力学模型

### 4.1.1 代码实现
```python
import numpy as np
import matplotlib.pyplot as plt

def force(x, k, c):
    return -k * x - c * x

def velocity(x, v, dt):
    return v + force(x, k, c) * dt

def simulate(x0, v0, k, c, dt, steps):
    x = np.zeros(steps)
    v = np.zeros(steps)
    x[0] = x0
    v[0] = v0
    for i in range(1, steps):
        x[i] = velocity(x[i - 1], v[i - 1], dt)
        v[i] = velocity(x[i - 1], v[i - 1], dt)
    return x, v

k = 1
c = 0.1
dt = 0.01
steps = 1000
x0 = 0
v0 = 0

x, v = simulate(x0, v0, k, c, dt, steps)
plt.plot(x, v)
plt.xlabel('Position')
plt.ylabel('Velocity')
plt.title('Particle Dynamics')
plt.show()
```
### 4.1.2 解释说明
在这个代码实例中，我们首先定义了粒子受到的力：弦张力和粘液力。然后，我们定义了粒子速度的更新函数。接着，我们使用简单的粒子动力学模型进行模拟，并绘制了粒子在不同时间尺度上的速度分布。
