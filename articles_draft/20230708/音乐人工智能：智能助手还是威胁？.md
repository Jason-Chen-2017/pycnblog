### 4.3.2 机器学习（ML）

在ML方面，我们可以使用scikit-learn库来实现推荐功能。以下是一个简单的示例代码：

```python
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

def ml_recommend(data, model):
    # 实现协同过滤算法
    similarities = []
    for u, v in data:
        # 计算相似度
        similarity = cosine_similarity(v)
        similarities.append(similarity)
    # 对相似度进行排序
    similarities.sort()
    # 获取相似度最高的用户
    user_id = np.argmin(similarities)
    # 返回相似度最高的歌曲
    return np.argmax(similarities[user_id])

```
