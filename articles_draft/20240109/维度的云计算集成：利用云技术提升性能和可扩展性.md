```python
costs = [[10, 20, 30], [40, 50, 60], [70, 80, 90]]

# 资源调度
dependencies = [(0, 1), (1, 2)]

# 自动扩展
strategies = ['strategy1', 'strategy2', 'strategy3']

# 计算负载
loads = np.array([np.sum(weights[i] * costs[i]) for i in range(len(resources))])

# 调度任务
scheduled_tasks = []
for i, resource in enumerate(resources):
    for j, (task, cost) in enumerate(zip(tasks, costs[i])):
        if i not in dependencies[j]:
            scheduled_tasks.append((resource, task, costs[i][j]))

# 扩展资源
extended_resources = []
for strategy in strategies:
    if strategy == 'strategy1':
        extended_resources.append(resources + ['resource4', 'resource5'])
    elif strategy == 'strategy2':
        extended_resources.append(resources + ['resource6', 'resource7'])
    else:
        extended_resources.append(resources + ['resource8', 'resource9'])

# 选择最佳扩展策略
best_strategy = max(strategies, key=lambda strategy: np.sum(loads[i] * costs[i] for i in extended_resources[strategy]))
```

在这个代码实例中，我们首先发现所有可用的云计算资源，并计算每个资源的负载。然后，我们根据任务的依赖关系调度任务，并动态地扩展或缩减云计算资源。最后，我们选择最佳的扩展策略。
