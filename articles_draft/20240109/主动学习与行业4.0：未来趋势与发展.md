主动学习是一种人工智能算法，它可以让模型在有限的标签数据集下，更有效地学习。主动学习的核心思想是让模型主动选择需要获取标签的数据，以便更好地优化模型。主动学习的具体实现方法包括：

- 查询策略：模型根据当前的知识，主动选择需要获取标签的数据。
- 模型更新：根据获取的新标签数据，更新模型的参数。

## 3.3 主动学习算法数学模型公式
主动学习算法的数学模型公式如下：

$$
\begin{aligned}
&w_{t+1} = w_t + \eta_t \nabla J(w_t) \\
&\nabla J(w_t) = \sum_{i=1}^n \delta(f(x_i, w_t)) \nabla_w \delta(f(x_i, w_t)) \\
&\delta(f(x_i, w_t)) = \begin{cases}
1, & \text{if } x_i \text{ is queried} \\
0, & \text{otherwise}
\end{cases}
\end{aligned}
$$

其中，$w_t$ 表示模型参数在第t次迭代时的值，$\eta_t$ 表示学习率，$J(w_t)$ 表示模型损失函数，$x_i$ 表示数据，$f(x_i, w_t)$ 表示模型在第t次迭代时对数据$x_i$的预测，$\nabla_w \delta(f(x_i, w_t))$ 表示模型对数据$x_i$的梯度。

class ActiveLearning:
    def __init__(self, X, y, query_strategy, model):
        self.X = X
        self.y = y
        self.query_strategy = query_strategy
        self.model = model
        self.unlabeled_indices = np.where(self.y == -1)[0]
        self.labeled_indices = np.where(self.y == 1)[0]

    def query(self):
        unlabeled_x = self.X[self.unlabeled_indices]
        unlabeled_y = self.y[self.unlabeled_indices]

        self.unlabeled_indices = np.delete(self.unlabeled_indices, np.argsort(self.query_strategy(unlabeled_x)))

    def train(self, num_queries):
        for _ in range(num_queries):
            self.query()
            self.model.fit(self.X[self.labeled_indices], self.y[self.labeled_indices])
