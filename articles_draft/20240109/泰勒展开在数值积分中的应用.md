# 2.核心概念与联系

## 2.1 泰勒展开

泰勒展开是一种逼近的数学工具，它可以用来逼近一个函数在某个点的值。泰勒展开的基本思想是将一个函数展开为一系列项的和，这些项是基于该函数的导数和底数构建的。泰勒展开的一般形式如下：

$$
f(x) \approx \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n
$$

其中，$f(x)$ 是要逼近的函数，$a$ 是逼近的点，$f^{(n)}(a)$ 是 $f(x)$ 的 $n$ 阶导数在点 $a$ 的值，$n!$ 是 $n$ 的阶乘。

泰勒展开的一个重要特点是，当 $x$ 接近 $a$ 时，泰勒展开的误差趋于零，因此可以用来逼近函数值。

## 2.2 数值积分

数值积分是一种求积分值的方法，它主要通过将积分区间内的函数值进行累加来近似求解积分值。数值积分的一个重要特点是，它可以在有限的计算时间内得到积分值的近似解。

数值积分的一个常见方法是分区法，它将积分区间划分为多个等长子区间，然后在每个子区间内将积分函数近似为梯形公式或者曲线段逼近函数，并将这些近似值累加得到积分值。

def f(x):
    return np.exp(-x**2)

b = 1
n = 10

# 计算泰勒展开的和
def taylor_sum(x, a, n):
    f_prime = lambda x: f(x)
    f_prime_prime = lambda x: f_prime(x)
    taylor_sum = 0
    for i in range(n+1):
        taylor_sum += f_prime_prime(a) / (i * np.math.factorial(i)) * (x - a) ** i
        f_prime_prime = lambda x: f_prime_prime(x) - f_prime(x) * f_prime(x)
    return taylor_sum

# 计算泰勒展开积分的近似值
def taylor_integral(a, b, n):
    integral = 0
    x = np.linspace(a, b, 1000)
    for i in range(len(x)):
        integral += taylor_sum(x[i], a, n)
    return integral

result = taylor_integral(a, b, n)
print("泰勒展开积分的近似值:", result)
