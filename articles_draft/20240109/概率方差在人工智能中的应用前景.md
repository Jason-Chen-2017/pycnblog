## 2.1 概率

概率是一种数学工具，用于描述事件发生的可能性。概率通常表示为一个数值，范围在0到1之间，其中0表示事件不可能发生，1表示事件必然发生。

概率可以通过以下几种方法计算：

- 经验概率：通过对事件发生的历史记录进行分析，得到事件发生的频率。
- 理论概率：通过对事件的描述进行分析，得到事件发生的可能性。
- 条件概率：通过对事件发生的前提条件进行分析，得到事件发生的可能性。

## 2.2 方差

方差是一种数学工具，用于描述数据集中数据点相对于平均值的离散程度。方差可以通过以下公式计算：

$$
\sigma^2 = \frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2
$$

其中，$\sigma^2$ 表示方差，$N$ 表示数据点数量，$x_i$ 表示数据点，$\mu$ 表示平均值。

方差的主要特点是：

- 方差是一个非负数，其值越大，数据点相对于平均值的离散程度越大。
- 如果数据点相对于平均值是紧凑的，那么方差就会较小；如果数据点相对于平均值是分散的，那么方差就会较大。

```python
import numpy as np

def gradient_descent(X, y, theta, alpha, iterations):
    m = len(y)
    for i in range(iterations):
        hypothesis = np.dot(X, theta)
        gradient = (1 / m) * np.dot(X.T, (hypothesis - y))
        theta = theta - alpha * gradient
    return theta
```

在上述代码中，我们实现了梯度下降算法。在这个算法中，我们使用了概率方差来计算梯度的方差，从而影响模型的收敛速度。

```python
import tensorflow as tf

class NeuralNetwork(object):
    def __init__(self, input_size, hidden_size, output_size):
        self.input_size = input_size
        self.hidden_size = hidden_size
        self.output_size = output_size
        self.weights = {
            'hidden': tf.Variable(tf.random_normal([input_size, hidden_size])),
            'output': tf.Variable(tf.random_normal([hidden_size, output_size]))
        }
        self.biases = {
            'hidden': tf.Variable(tf.random_normal([hidden_size])),
            'output': tf.Variable(tf.random_normal([output_size]))
        }

    def forward(self, x):
        hidden_layer = tf.add(tf.matmul(x, self.weights['hidden']), self.biases['hidden'])
        hidden_layer = tf.nn.relu(hidden_layer)
        output_layer = tf.add(tf.matmul(hidden_layer, self.weights['output']), self.biases['output'])
        return output_layer
```

在上述代码中，我们实现了一个简单的神经网络。在这个神经网络中，我们使用了概率方差来计算激活函数的非线性性，从而影响神经网络的收敛速度。
