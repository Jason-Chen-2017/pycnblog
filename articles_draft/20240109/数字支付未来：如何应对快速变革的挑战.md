```python
class PaymentSystem:
    def __init__(self, private_key, public_key):
        self.private_key = private_key
        self.public_key = public_key

    def encrypt(self, data):
        encrypted_data = hashlib.sha256(data.encode()).digest()
        return base64.b64encode(encrypted_data).decode()

    def decrypt(self, encrypted_data):
        decrypted_data = base64.b64decode(encrypted_data.encode())
        return hashlib.sha256(decrypted_data).hexdigest()

    def sign(self, data):
        signature = hashlib.sha256((data + self.private_key).encode()).hexdigest()
        return signature

    def verify(self, data, signature):
        computed_signature = hashlib.sha256((data + self.public_key).encode()).hexdigest()
        return signature == computed_signature
```

在这个类中，我们定义了四个方法：`encrypt`、`decrypt`、`sign`和`verify`。这四个方法分别实现了数据加密、解密、签名和验证功能。
