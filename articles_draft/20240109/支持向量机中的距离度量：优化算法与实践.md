# 2.核心概念与联系

在SVM中，距离度量是一个非常重要的概念，它用于衡量数据点之间的距离。常见的距离度量包括欧氏距离、马氏距离、曼哈顿距离等。在SVM中，选择合适的距离度量可以帮助算法更好地学习数据的结构，从而提高模型的性能。

## 2.1 欧氏距离

欧氏距离是一种常用的距离度量，用于计算两个点之间的距离。在二维空间中，欧氏距离公式为：

$$
d(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2}
$$

在多维空间中，欧氏距离公式为：

$$
d(x, y) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}
$$

## 2.2 马氏距离

马氏距离是一种另一种常用的距离度量，用于计算两个点之间的距离。在二维空间中，马氏距离公式为：

$$
d(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2}
$$

在多维空间中，马氏距离公式为：

$$
d(x, y) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}
$$

## 2.3 曼哈顿距离

曼哈顿距离是一种另一种常用的距离度量，用于计算两个点之间的距离。在二维空间中，曼哈顿距离公式为：

$$
d(x, y) = |x_1 - y_1| + |x_2 - y_2|
$$

在多维空间中，曼哈顿距离公式为：

$$
d(x, y) = \sum_{i=1}^n |x_i - y_i|
$$

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明如何在SVM中使用距离度量。我们将使用Python的scikit-learn库来实现SVM模型，并使用欧氏距离作为距离度量。

```python
from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# 加载数据集
iris = datasets.load_iris()
X = iris.data
y = iris.target

# 数据预处理
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 训练集和测试集的分割
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# 创建SVM模型
svm = SVC(kernel='linear', C=1.0, random_state=42)

# 训练SVM模型
svm.fit(X_train, y_train)

# 预测测试集的标签
y_pred = svm.predict(X_test)

# 计算准确率
accuracy = accuracy_score(y_test, y_pred)
print(f'准确率：{accuracy:.4f}')
```

在上面的代码中，我们首先导入了所需的库，并加载了iris数据集。接着，我们对数据进行了标准化处理，以便于模型训练。然后，我们将数据集分为训练集和测试集。接着，我们创建了一个SVM模型，并使用线性核进行训练。最后，我们使用训练好的模型对测试集进行预测，并计算准确率。
