## 2.1 连续性

连续性是微积分中的一个基本概念，它描述了一个函数在某个点上的变化规律。连续性可以分为两种类型：

1. 连续：在某个点上，函数的值和它周围的值之间存在连续性，即函数在该点处有定义。
2. 不连续：在某个点上，函数的值和它周围的值之间没有连续性，即函数在该点处未定义或有跳跃变化。

连续性是微积分的基本要素，后续的极限、积分和微分都依赖于连续性的概念。

## 2.2 极限

极限是微积分中的一个重要概念，它描述了一个函数在某个点的近邻点上的行为。极限可以用来描述无限接近某个值的过程。

极限的基本概念可以表示为：

$$
\lim_{x \to a} f(x) = L
$$

其中，$a$ 是极限的点，$f(x)$ 是函数，$L$ 是极限的值。

极限是微积分的基本工具，它可以用于解决许多问题，如求函数的导数和积分。

## 2.3 积分

积分是微积分中的一个核心概念，它描述了一个函数在某个区间内的面积。积分可以用来解决许多问题，如求面积、体积、工作量等。

积分的基本概念可以表示为：

$$
\int_{a}^{b} f(x) dx
$$

其中，$a$ 和 $b$ 是积分区间的端点，$f(x)$ 是积分函数。

积分是微积分的基本工具，它可以用于解决许多问题，如求面积、体积、工作量等。

## 2.4 微分

微分是微积分中的一个核心概念，它描述了一个函数在某个点的斜率。微分可以用来解决许多问题，如求导数、速度、加速度等。

微分的基本概念可以表示为：

$$
\frac{dy}{dx} = f'(x)
$$

其中，$y$ 是被微分函数，$x$ 是微分函数，$f'(x)$ 是函数的导数。

微分是微积分的基本工具，它可以用于解决许多问题，如求导数、速度、加速度等。

## 2.5 微积分的联系

微积分的核心概念之间存在着密切的联系。连续性、极限、积分和微分都是微积分的基本要素，它们之间的联系可以用以下公式表示：

$$
\frac{d}{dx} \int_{a}^{x} f(t) dt = f(x)
$$

这个公式表示了积分和微分之间的关系，即积分是微分的反操作。

## 3.1 极限的计算

极限的计算主要包括两种情况：

1. 直接求极限：直接将$x$ 替换为$a$，并计算结果。
2. 趋于极限：将$x$ 趋于$a$，并计算结果。

例如，计算极限$\lim_{x \to 0} \frac{\sin x}{x}$：

1. 直接求极限：将$x$ 替换为$0$，得到$\frac{0}{0}$，这是一个未定义的极限。
2. 趋于极限：将$x$ 趋于$0$，可以用恒等关系$\lim_{x \to 0} \frac{\sin x}{x} = 1$来解决。

## 3.2 积分的计算

积分的计算主要包括两种情况：

1. 直接积分：直接将积分公式应用于函数，并计算结果。
2. 积分技巧：使用积分技巧，如常数分离、交换限制符、积分规则等，来计算积分。

例如，计算积分$\int_{0}^{1} 2x dx$：

1. 直接积分：将积分公式应用于函数，得到$2x^2|_0^1 = 2(1^2) - 2(0^2) = 2$。
2. 积分技巧：使用常数分离技巧，将$2x$ 分解为$2 \cdot x$，然后分别积分，得到同上结果。

## 3.3 微分的计算

微分的计算主要包括两种情况：

1. 直接微分：直接将微分公式应用于函数，并计算结果。
2. 微分技巧：使用微分技巧，如常数下降、链式法则、产品法则等，来计算微分。

例如，计算微分$\frac{d}{dx} (3x^2 + 4x)$：

1. 直接微分：将微分公式应用于函数，得到$6x + 4$。
2. 微分技巧：使用常数下降技巧，将$3x^2$ 分解为$3 \cdot x^2$，然后分别微分，得到同上结果。

## 3.4 常用微积分公式

微积分中有许多常用的公式，如下所示：

1. 积分公式：$\int x^n dx = \frac{x^{n+1}}{n+1} + C$
2. 微分公式：$\frac{d}{dx} x^n = nx^{n-1}$
3. 恒等关系：$\int_{a}^{b} dx = b - a$
4. 积分规则：$\int (f(x) \pm g(x)) dx = \int f(x) dx \pm \int g(x) dx$
5. 积分技巧：$\int x^n dx = \frac{x^{n+1}}{n+1} + C$
6. 微分技巧：$\frac{d}{dx} (f(x) \pm g(x)) = \frac{df(x)}{dx} \pm \frac{dg(x)}{dx}$

## 4.1 极限的代码实例

在Python中，可以使用`sympy`库来计算极限。例如，计算极限$\lim_{x \to 0} \frac{\sin x}{x}$：

```python
import sympy as sp

x = sp.Symbol('x')
f = sp.sin(x)/x
limit = sp.limit(f, x, 0)
print(limit)
```

输出结果为`1`，表示极限存在。

## 4.2 积分的代码实例

在Python中，可以使用`sympy`库来计算积分。例如，计算积分$\int_{0}^{1} 2x dx$：

```python
import sympy as sp

x = sp.Symbol('x')
f = 2*x
integral = sp.integrate(f, (x, 0, 1))
print(integral)
```

输出结果为`2/3`，表示积分结果。

## 4.3 微分的代码实例

在Python中，可以使用`sympy`库来计算微分。例如，计算微分$\frac{d}{dx} (3x^2 + 4x)$：

```python
import sympy as sp

x = sp.Symbol('x')
f = 3*x**2 + 4*x
derivative = sp.diff(f, x)
print(derivative)
```

输出结果为`6*x + 4`，表示微分结果。

## 4.4 常用微积分公式的代码实例

在Python中，可以使用`sympy`库来计算常用微积分公式。例如，计算积分$\int x^2 dx$：

