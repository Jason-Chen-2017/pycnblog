其中，$f(x)$ 是提取的特征，$w_{i}$ 是特征权重，$n$ 是生理数据的个数。


### 4.2 特征提取

```python
# 提取特征

```

### 4.3 模型训练

```python
# 训练SVM模型
from sklearn.svm import SVC

X = data_features.values
y = data['label'].values

```
