### 4.1.2 模型定义

接下来，我们需要定义一个深度学习模型。在这个示例中，我们将使用一个简单的线性模型。我们还需要在模型中添加一个成本层，以便在训练过程中考虑成本。

```python
import tensorflow as tf

class CostSensitiveLinearRegression(tf.keras.Model):
    def __init__(self):
        super(CostSensitiveLinearRegression, self).__init__()
        self.linear = tf.keras.layers.Dense(1, input_shape=(1,))

    def call(self, x, training=False):
        y_pred = self.linear(x)
        cost = cost_function(y_pred)
        return y_pred, cost
```

```python

    optimizer.apply_gradients(zip(gradients, model.trainable_variables))
```
