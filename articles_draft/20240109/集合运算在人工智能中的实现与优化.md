## 2.1 集合的基本概念

集合是一种数据结构，用于表示一组具有特定属性的元素的集合。集合中的元素可以是任何类型的数据，包括数字、字符、对象等。集合可以用括号表示，例如：{1, 2, 3}。

集合运算的主要操作包括：

- 创建集合：使用{}或createSet()函数创建一个新的集合。
- 添加元素：使用add()函数将元素添加到集合中。
- 删除元素：使用remove()函数将元素从集合中删除。
- 查询元素：使用in操作符或contains()函数判断元素是否在集合中。
- 集合运算：包括并集、交集、差集、笛卡尔积等。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 集合运算的基本概念

### 3.1.1 并集

并集是两个集合中所有元素的集合。并集运算符用 | 表示，例如：A | B。

### 3.1.2 交集

交集是两个集合中共同元素的集合。交集运算符用 & 表示，例如：A & B。

### 3.1.3 差集

差集是一个集合中不在另一个集合中的元素的集合。差集运算符用 - 表示，例如：A - B。

### 3.1.4 笛卡尔积

笛卡尔积是两个集合中所有可能组合的集合。笛卡尔积运算符用 × 表示，例如：A × B。

## 3.2 集合运算的数学模型公式

### 3.2.1 并集

$$
A | B = \{x | x \in A \text{或} x \in B\}
$$

### 3.2.2 交集

$$
A & B = \{x | x \in A \text{且} x \in B\}
$$

### 3.2.3 差集

$$
A - B = \{x | x \in A \text{且} x \notin B\}
$$

### 3.2.4 笛卡尔积

$$
A × B = \{ (a, b) | a \in A \text{且} b \in B\}
$$

## 3.3 集合运算的具体操作步骤

### 3.3.1 并集

1. 创建两个集合A和B。
2. 遍历集合A中的每个元素，将其添加到结果集合中。
3. 遍历集合B中的每个元素，将其添加到结果集合中。
4. 返回结果集合。

### 3.3.2 交集

1. 创建两个集合A和B。
2. 创建一个新集合C。
3. 遍历集合A中的每个元素，如果它也在集合B中，将其添加到集合C中。
4. 返回集合C。

### 3.3.3 差集

1. 创建两个集合A和B。
2. 创建一个新集合C。
3. 遍历集合A中的每个元素，将其添加到集合C中。
4. 遍历集合B中的每个元素，如果它也在集合C中，将其从集合C中删除。
5. 返回集合C。

### 3.3.4 笛卡尔积

1. 创建两个集合A和B。
2. 创建一个新集合C。
3. 遍历集合A中的每个元素，将其添加到集合C中。
4. 遍历集合B中的每个元素，将一个元组（a, b）添加到集合C中，其中a是集合A中的元素，b是集合B中的元素。
5. 返回集合C。

# 4.具体代码实例和详细解释说明

## 4.1 并集实例

```python
A = {1, 2, 3}
B = {3, 4, 5}
C = A | B
print(C)  # 输出: {1, 2, 3, 4, 5}
```

## 4.2 交集实例

```python
A = {1, 2, 3}
B = {3, 4, 5}
C = A & B
print(C)  # 输出: {3}
```

## 4.3 差集实例

```python
A = {1, 2, 3}
B = {3, 4, 5}
C = A - B
print(C)  # 输出: {1, 2}
```

## 4.4 笛卡尔积实例

```python
A = {1, 2, 3}
B = {4, 5, 6}
C = A × B
print(C)  # 输出: {(1, 4), (1, 5), (1, 6), (2, 4), (2, 5), (2, 6), (3, 4), (3, 5), (3, 6)}
```
