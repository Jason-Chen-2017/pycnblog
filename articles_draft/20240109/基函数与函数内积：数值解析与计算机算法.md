# 2.核心概念与联系

## 2.1 基函数

基函数是数值解析和计算机算法中的一个基本概念，它用于表示函数或多项式。基函数可以是线性独立的函数，如基于元素的方法中的基函数；也可以是指基于多项式的表示，如多项式近似和多项式插值。

### 2.1.1 基于元素的基函数

在基于元素的方法中，如元素公式法、有限元方法等，基函数通常是元素内的节点函数。这些基函数可以表示为：

$$
\phi_i(x) = \begin{cases}
1, & x = x_i \\
0, & x \neq x_i
\end{cases}
$$

这些基函数是线性独立的，可以用于表示元素内的函数。

### 2.1.2 多项式基函数

多项式基函数通常用于多项式近似和多项式插值。常见的多项式基函数有：

- 常数多项式：$1$
- 线性多项式：$x$
- 二次多项式：$x^2$
- 三次多项式：$x^3$

这些基函数可以用于表示多项式。

## 2.2 函数内积

函数内积是在数值解析和计算机算法中表示向量和向量之间的内积的一个概念。函数内积在许多计算机算法中发挥着重要作用，例如在线性代数、线性方程组解和优化问题中。

### 2.2.1 数学模型

函数内积可以表示为：

$$
\langle f, g \rangle = \int_{a}^{b} f(x)g(x)dx
$$

其中，$f(x)$ 和 $g(x)$ 是两个函数，$a$ 和 $b$ 是积分区间。

### 2.2.2 计算机算法实现

在计算机算法中，我们需要计算函数内积。这可以通过数值积分方法实现，如梯形法、Simpson法等。例如，使用梯形法计算函数内积可以表示为：

$$
\langle f, g \rangle \approx \sum_{i=0}^{n-1} \frac{w_i}{6}(f(x_i) + 4f(x_{i+1}) + f(x_{i+2}))g(x_i)
$$

其中，$w_i$ 是梯形法中的权重，$x_i$ 是积分区间的分点。

## 4.1 基于元素的基函数

考虑一个二维元素，元素内的节点函数可以表示为：

$$
\phi_1(x, y) = \begin{cases}
1, & x \leq 1 \text{ and } y \leq 1 \\
0, & \text{otherwise}
\end{cases}
$$

$$
\phi_2(x, y) = \begin{cases}
1, & 1 \leq x \leq 2 \text{ and } y \leq 1 \\
0, & \text{otherwise}
\end{cases}
$$

我们可以使用Python的NumPy库来计算基函数的值：

```python
import numpy as np

def phi1(x, y):
    return np.where((x <= 1) & (y <= 1), 1, 0)

def phi2(x, y):
    return np.where((1 <= x) & (y <= 1), 1, 0)
```

## 4.3 函数内积计算

考虑两个函数：

$$
f(x) = 2x
$$

$$
g(x) = x^2
$$

我们需要计算函数内积：

$$
\langle f, g \rangle = \int_{-1}^{1} f(x)g(x)dx
$$

我们可以使用Python的NumPy库和Scipy库来计算函数内积：

```python
import numpy as np
from scipy.integrate import simps

def f(x):
    return 2 * x

def g(x):
    return x**2

x = np.linspace(-1, 1, 1000)
dx = x[1] - x[0]

integral = simps(f(x) * g(x), x)

print("函数内积：", integral)
```
