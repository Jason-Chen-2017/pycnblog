## 2.1 汉明距离定义

汉明距离是一种用于计算两个二进制序列之间的差异的度量标准。给定两个长度为n的二进制序列x和y，汉明距离（H）可以定义为这两个序列中不同位的数量。换句话说，汉明距离是两个序列中不同位的数量。

$$
H(x, y) = \sum_{i=1}^{n} \delta(x_i, y_i)
$$

其中，$\delta(x_i, y_i)$ 是一个指示函数，如果$x_i \neq y_i$，则$\delta(x_i, y_i) = 1$，否则$\delta(x_i, y_i) = 0$。

## 4.1 汉明距离算法Python实现

以下是汉明距离算法的Python实现：

```python
def hamming_distance(x, y):
    n = len(x)
    distance = 0
    for i in range(n):
        if x[i] != y[i]:
            distance += 1
    return distance

# 测试数据
x = "1101"
y = "1001"

# 计算汉明距离
result = hamming_distance(x, y)
print("Hamming distance:", result)
```

```java
public class HammingDistance {
    public static int hammingDistance(String x, String y) {
        int n = x.length();
        int distance = 0;
        for (int i = 0; i < n; i++) {
            if (x.charAt(i) != y.charAt(i)) {
                distance += 1;
            }
        }
        return distance;
    }

    public static void main(String[] args) {
        String x = "1101";
        String y = "1001";

        int result = hammingDistance(x, y);
        System.out.println("Hamming distance: " + result);
    }
}
```
