# 2.核心概念与联系

首先，我们需要了解边界计算和区块链技术的核心概念。

## 2.1 边界计算（Edge Computing）

边界计算是一种计算模式，将计算任务推向边缘设备，以降低网络延迟和提高数据处理效率。边界计算的主要优势包括：

- 降低网络延迟：边界计算将计算任务推向边缘设备，减少了数据传输距离，从而降低了网络延迟。
- 提高数据处理效率：边界计算可以将大量的计算任务分散到多个边缘设备上，提高了数据处理效率。
- 降低网络负载：边界计算可以减少数据传输量，降低网络负载。

## 2.2 区块链技术

区块链技术是一种去中心化的数字交易记录系统，通过将数据存储在多个节点上，提高了数据安全性和透明度。区块链技术的核心特点包括：

- 去中心化：区块链技术不依赖于中心化的服务器或机构，所有的节点都具有相同的权重，提高了数据安全性。
- 不可篡改：区块链技术通过加密算法，使得已经存储在区块链上的数据不可以被篡改。
- 透明度：区块链技术通过公开的分布式账本，使得所有的交易记录都可以被所有节点查看，提高了数据透明度。
