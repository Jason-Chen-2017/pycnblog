## 2.核心概念与联系

### 2.1 余弦距离

余弦距离是一种用于衡量两个向量之间相似性的度量方法，它是通过计算两个向量之间的余弦相似度得到的。余弦相似度是一种基于角度的相似度度量方法，它通过计算两个向量之间的角度来衡量其相似性。

余弦距离的公式为：

$$
d_{cos}(\mathbf{x}, \mathbf{y}) = 1 - \cos(\theta) = 1 - \frac{\mathbf{x} \cdot \mathbf{y}}{\|\mathbf{x}\| \|\mathbf{y}\|}
$$

其中，$\mathbf{x}$ 和 $\mathbf{y}$ 是两个向量，$\cdot$ 表示点积，$\|\mathbf{x}\|$ 和 $\|\mathbf{y}\|$ 分别表示向量 $\mathbf{x}$ 和 $\mathbf{y}$ 的长度。

### 2.2 欧几里得距离

欧几里得距离的公式为：

$$
d_{Euclidean}(\mathbf{x}, \mathbf{y}) = \sqrt{(\mathbf{x} - \mathbf{y})^T (\mathbf{x} - \mathbf{y})}

其中，$\mathbf{x}$ 和 $\mathbf{y}$ 是两个向量，$(\mathbf{x} - \mathbf{y})^T$ 表示向量 $\mathbf{x}$ 和 $\mathbf{y}$ 之间的点积。

### 4.1 余弦距离的代码实例

```python
import numpy as np

def cosine_similarity(x, y):
    dot_product = np.dot(x, y)
    norm_x = np.linalg.norm(x)
    norm_y = np.linalg.norm(y)
    cosine_similarity = 1 - dot_product / (norm_x * norm_y)
    return cosine_similarity

x = np.array([1, 2, 3])
y = np.array([4, 5, 6])

cosine_similarity(x, y)
```

```python
def euclidean_distance(x, y):
    diff = x - y
    euclidean_distance = np.sqrt(np.dot(diff, diff))
```
