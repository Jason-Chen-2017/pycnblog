$$
\int_a^b f(x) dx

其中，$f(x)$ 是一个函数，$a$ 和 $b$ 是区间的端点。不定积分的计算方法有很多，主要包括直接积分、积分表和软件工具等。

## 3.1 求最小公倍数
求最小公倍数的公式为：

$$
\text{最小公倍数} = \frac{a \times b}{\text{最大公约数}}
$$

其中，$a$ 和 $b$ 是两个整数。通过不定积分，可以得到最大公约数的公式：

$$
\text{最大公约数} = \int_{x=0}^{x=1} \left\lfloor ax/b \right\rfloor dx

其中，$\left\lfloor \cdot \right\rfloor$ 表示向下取整。具体操作步骤如下：

1. 将两个整数 $a$ 和 $b$ 分别取绝对值。

def gcd(a, b):
    return int(math.fabs(a) * math.fabs(b) / math.gcd(a, b))

    while b:
        a, b = b, a % b

## 4.4 欧拉函数的代码实例
```python
import math

def prime_factors(n):
    factors = []
    i = 2
    while i * i <= n:
        if n % i:
            i += 1
        else:
            n //= i
            factors.append(i)
    if n > 1:
        factors.append(n)
    return factors

def euler_function(n):
    factors = prime_factors(n)
    result = n
    for factor in factors:
        result *= (factor - 1)
    return result

n = 12
result = euler_function(n)
print(result)
```
