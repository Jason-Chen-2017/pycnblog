### 3.3 数学模型公式详细讲解

在SOP流程的数据驱动决策中，可以使用各种数学模型来分析和处理数据。以下是一些常见的数学模型公式：

1. 均值（Mean）：均值是一种描述数据集合的中心趋势的统计量。它可以通过以下公式计算：
$$
\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i
$$
其中，$x_i$ 是数据集合中的第 $i$ 个元素，$n$ 是数据集合的大小。
2. 中位数（Median）：中位数是一种描述数据集合的中心趋势的统计量。它可以通过以下公式计算：
$$
\text{Median} = \left\{ \begin{array}{ll}
\frac{x_{(n+1)/2} + x_{n/(2)}} {2} & \text{if } n \text{ is odd} \\
x_{n/(2)} & \text{if } n \text{ is even}
\end{array} \right.
$$
其中，$x_{(n+1)/2}$ 和 $x_{n/(2)}$ 是数据集合中的第 $(n+1)/2$ 和 $n/(2)$ 个元素。
3. 方差（Variance）：方差是一种描述数据集合的散度的统计量。它可以通过以下公式计算：
$$
\sigma^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2
$$
其中，$x_i$ 是数据集合中的第 $i$ 个元素，$n$ 是数据集合的大小，$\bar{x}$ 是数据集合的均值。
4. 标准差（Standard Deviation）：标准差是一种描述数据集合的散度的统计量。它可以通过以下公式计算：
$$
\sigma = \sqrt{\sigma^2}
$$
其中，$\sigma$ 是方差。
5. 相关系数（Correlation Coefficient）：相关系数是一种描述两个变量之间的关系的统计量。它可以通过以下公式计算：
$$
r = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n} (x_i - \bar{x})^2} \sqrt{\sum_{i=1}^{n} (y_i - \bar{y})^2}}
$$
其中，$x_i$ 和 $y_i$ 是数据集合中的第 $i$ 个元素，$n$ 是数据集合的大小，$\bar{x}$ 和 $\bar{y}$ 是数据集合的均值。

3. 生产结束

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 加载生产数据
data = pd.read_csv('production_data.csv')

# 计算生产吞吐量
throughput = data['end_time'] - data['start_time'].shift(1)

# 计算生产效率
efficiency = data['quantity'] / data['time']

# 计算生产质量
quality = data['defective'] / data['quantity']

# 绘制生产吞吐量曲线
plt.plot(data['start_time'], throughput)
plt.xlabel('时间')
plt.ylabel('吞吐量')
plt.title('生产吞吐量曲线')
plt.show()

# 绘制生产效率曲线
plt.plot(data['start_time'], efficiency)
plt.xlabel('时间')
plt.ylabel('效率')
plt.title('生产效率曲线')
plt.show()

# 绘制生产质量曲线
plt.plot(data['start_time'], quality)
plt.xlabel('时间')
plt.ylabel('质量')
plt.title('生产质量曲线')
plt.show()
```
