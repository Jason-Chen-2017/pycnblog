$$

### 4.1 温度控制算法实现

```python
import numpy as np

def pid_controller(target_temp, current_temp, Kp, Ti, Td):
    error = target_temp - current_temp
    derivative = np.sign(error) * np.abs(error)
    integral = np.integrate(error, Ti)
    output = Kp * error + Td * derivative + integral
    return output
```
