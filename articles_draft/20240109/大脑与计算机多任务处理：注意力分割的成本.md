```python
import numpy as np

class BrainTask:
    def __init__(self, weight, function):
        self.weight = weight
        self.function = function

    def process(self, input_data):
        output_data = self.function(input_data)
        return output_data

def attention_distribution(tasks):
    attention = np.sum([task.weight * task.function for task in tasks])
    return attention

task1 = BrainTask(0.5, lambda x: x * 0.5)
task2 = BrainTask(0.5, lambda x: x * 0.5)
task3 = BrainTask(0.5, lambda x: x * 0.5)

tasks = [task1, task2, task3]
attention = attention_distribution(tasks)
print("Attention distribution:", attention)
```

### 问题1：大脑是如何实现多任务处理的？

### 问题2：大脑多任务处理的成本是什么？
答案：大脑在处理多任务时，注意力分割的成本主要表现在以下几个方面：

1. 信息丢失：当你注意力集中在一个任务上时，你可能会忽略其他任务，导致信息丢失。
2. 处理延迟：当你需要在多个任务之间切换注意力时，可能会导致处理延迟。
3. 错误率增加：当你注意力分散在多个任务上时，可能会导致错误率增加。

### 问题3：如何减少大脑多任务处理的成本？
答案：减少大脑多任务处理的成本可以通过以下几种方法实现：

1. 优先级分配：根据任务的重要性和紧急性，为任务分配不同的优先级，以便更有效地分配注意力。
2. 任务分解：将复杂的任务分解为多个简单的任务，以便更容易地处理。
3. 注意力训练：通过培养注意力训练，可以提高注意力的集中力和稳定性，从而减少多任务处理的成本。
