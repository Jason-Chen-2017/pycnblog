# 2.核心概念与联系

## 2.1 多目标决策（Multi-objective Decision Making, MODM）

多目标决策是一种在面临多个目标时采取的决策方法，该方法需要考虑多个目标之间的权衡和平衡。这些目标可能是矛盾相互矛盾的，或者是相互补充的。在实际应用中，多目标决策问题通常是非线性、非确定性和多约束的，因此需要采用适当的优化方法来解决。

## 2.2 优化模型

优化模型是一种数学模型，用于描述一个系统的目标和约束条件。优化模型可以用来描述一个系统的最优解，即使这个系统包含多个目标和多个约束条件。在多目标决策中，优化模型可以用来描述多个目标之间的权衡和平衡。

## 2.3 Pareto优化

Pareto优化是一种多目标优化方法，它基于Pareto优势关系来描述多个目标之间的权衡和平衡。Pareto优势关系是指一个解相对于另一个解具有更好的目标函数值。在Pareto优化中，我们寻找一个Pareto最优解集，即一个包含所有Pareto最优解的集合。

def random_search(f1, f2, constraint, bounds, n_iter):
