## 4.1 动力学代码实例

### 4.1.1 力学模型

```python
import numpy as np

class RobotArm:
    def __init__(self):
        self.mass = 10  # kg
        self.length = 1  # m
        self.inertia = self.mass * self.length**2  # kg*m^2

    def force(self, acceleration):
        return np.array([self.mass * acceleration, self.inertia * acceleration])
```

## 4.2 控制代码实例

### 4.2.1 位置控制

```python
import numpy as np

class PIDController:
    def __init__(self, Kp, Ki, Kd):
        self.Kp = Kp
        self.Ki = Ki
        self.Kd = Kd
        self.integral = 0

    def update(self, error, dt):
        self.integral += error * dt
        derivative = (error - self.last_error) / dt
        self.last_error = error
        output = self.Kp * error + self.Ki * self.integral + self.Kd * derivative
        return output
```
