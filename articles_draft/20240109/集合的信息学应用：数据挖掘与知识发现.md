# 3.1 聚类分析
聚类分析是一种无监督学习方法，它的目标是从一组数据中找出具有相似性的对象，将它们分组成不同的类别。聚类分析可以用于发现数据中的模式和关系，并用于数据压缩和数据清洗等任务。

## 3.1.1 K-均值聚类
K-均值聚类是一种常用的聚类分析方法，它的核心思想是将数据分成K个类别，每个类别的中心是已知的聚类中心，通过迭代的方法来找出最佳的聚类中心。

### 3.1.1.1 K-均值聚类的步骤

1.随机选择K个聚类中心。

2.根据聚类中心，将数据分成K个类别。

3.计算每个类别的均值，更新聚类中心。

4.重复步骤2和步骤3，直到聚类中心不再变化或者变化的速度很小。

### 3.1.1.2 K-均值聚类的数学模型公式

假设我们有一个数据集D，包含N个样本点，每个样本点有P个特征。我们希望将这些样本点分成K个类别。

对于每个类别i（i=1,2,...,K），我们有K个聚类中心Ci，每个聚类中心Ci包含P个特征。

我们希望找到一个最佳的聚类中心，使得每个样本点距离它所在类别的聚类中心最近。这可以通过最小化以下目标函数来实现：

$$
J(C_1,C_2,...,C_K)=\sum_{i=1}^{K}\sum_{x\in C_i}||x-C_i||^2
$$

其中，$||x-C_i||^2$表示样本点x和聚类中心Ci之间的欧氏距离的平方。

通过优化目标函数J，我们可以找到最佳的聚类中心。这可以通过迭代的方法来实现，具体步骤如下：

1.随机选择K个聚类中心。

2.根据聚类中心，将数据分成K个类别。

3.计算每个类别的均值，更新聚类中心。

4.重复步骤2和步骤3，直到聚类中心不再变化或者变化的速度很小。

## 3.1.2 层次聚类
层次聚类是一种另一种聚类分析方法，它的核心思想是通过逐步将数据分成更小的类别来构建一个层次结构的聚类。

### 3.1.2.1 层次聚类的步骤

1.计算数据集中每对样本点之间的距离，找出最近的两个样本点。

2.将这两个样本点合并成一个类别。

3.计算新形成的类别与其他类别之间的距离，找出最近的两个类别。

4.将这两个类别合并成一个类别。

5.重复步骤3和步骤4，直到所有的样本点被合并成一个类别。

### 3.1.2.2 层次聚类的数学模型公式

层次聚类的数学模型公式可以通过构建一个距离矩阵来表示。距离矩阵D是一个N×N的矩阵，其中N是数据集中的样本点数量。每个元素D[i][j]表示样本点i和样本点j之间的距离。

通过计算距离矩阵中的每个元素，我们可以找出最近的两个样本点，将它们合并成一个类别。然后更新距离矩阵，并重复这个过程，直到所有的样本点被合并成一个类别。

# 3.2 关联规则挖掘
关联规则挖掘是一种无监督学习方法，它的目标是从事务数据中发现关联规则，例如从市场篮库数据中发现顾客购买某一商品的可能性。

## 3.2.1 支持度和信息增益
关联规则挖掘的核心概念是支持度和信息增益。支持度是指一个项目集合在整个数据集中出现的次数占总次数的比例，信息增益是指一个项目集合在整个数据集中出现的次数占总次数的比例。

### 3.2.1.1 支持度

支持度是指一个项目集合在整个数据集中出现的次数占总次数的比例。假设项目集A在数据集D中出现了A的次数，数据集D中的总次数为T，则支持度可以表示为：

$$
sup(A)=\frac{A}{T}
$$

### 3.2.1.2 信息增益

信息增益是指一个项目集合在整个数据集中出现的次数占总次数的比例。假设项目集A在数据集D中出现了A的次数，数据集D中的总次数为T，则信息增益可以表示为：

$$
ig(A)=\frac{A}{T}\log_2\left(\frac{A}{T}\right)
$$

## 3.2.2 Apriori算法
Apriori算法是一种常用的关联规则挖掘方法，它的核心思想是通过迭代地找出频繁项目集来发现关联规则。

### 3.2.2.1 Apriori算法的步骤

1.找出数据集中的所有单项目集。

2.计算每个单项目集的支持度，选择支持度大于阈值的项目集。

3.找出选择的项目集中的所有可能的项目集合，并计算它们的支持度。

4.选择支持度大于阈值的项目集合。

5.重复步骤3和步骤4，直到不再找到新的关联规则。

### 3.2.2.2 Apriori算法的数学模型公式

Apriori算法的数学模型公式可以通过构建一个项目集合的频率表来表示。频率表是一个包含所有可能的项目集合及其在数据集中的出现次数的表。

通过计算频率表中的每个项目集的支持度，我们可以选择支持度大于阈值的项目集。然后更新频率表，并重复这个过程，直到不再找到新的关联规则。

# 4.1 K-均值聚类
```python
from sklearn.cluster import KMeans
import numpy as np

# 数据集
data = np.array([[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]])

# 设置聚类中心数量
k = 2

# 创建KMeans聚类器
kmeans = KMeans(n_clusters=k)

# 训练聚类器
kmeans.fit(data)

# 获取聚类中心
centers = kmeans.cluster_centers_

# 获取每个样本点的类别
labels = kmeans.labels_

print("聚类中心：", centers)
print("样本点类别：", labels)
```
# 4.2 层次聚类

```python
from scipy.cluster.hierarchy import dendrogram, linkage
import numpy as np

# 数据集
data = np.array([[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]])

# 创建距离矩阵

```
# 4.3 关联规则挖掘
```python
from mlxtend.frequent_patterns import apriori
from mlxtend.frequent_patterns import association_rules
import pandas as pd

# 数据集
data = pd.DataFrame({
    'Transaction': ['12345', '12345', '12345', '56789', '56789', '56789'],
    'Item': ['milk', 'milk', 'bread', 'bread', 'eggs', 'eggs']
})

# 找出频繁项目集
frequent_items = apriori(data, min_support=0.5, use_colnames=True)

# 找出关联规则
rules = association_rules(frequent_items, metric='lift', min_threshold=1)

print("关联规则：", rules)
```
