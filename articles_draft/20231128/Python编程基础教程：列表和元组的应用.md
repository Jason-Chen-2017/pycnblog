# 2.核心概念与联系
## 2.1 列表
列表是一个有序序列的数据类型。它是由若干逗号分隔的值组成的集合，可以包含不同类型的数据，并且可以通过索引来访问其中的元素。列表是一种可变的、动态的数据类型，可以随时增减其中的元素。列表是一种被广泛使用的数据类型，被用于许多方面，如保存文件目录、数据库记录、网页信息、用户输入等。

列表是用方括号([])包围的逗号分隔值列表，每一个值都对应着一个下标(index)或位置。列表的第一个元素的下标是0，第二个元素的下标是1，依次类推。如下图所示：


## 2.2 元组
元组是一个不可改变的序列数据类型。它类似于列表，但只允许读取，不允许修改。元组的元素不能删除、插入和修改。元组是按顺序排列的一系列元素。元组与列表之间最主要的区别就是一旦定义后，就不能再修改了。

元组是用圆括号(())包围的逗号分隔值列表，其特点是只能读取元素，不能修改元素。元组的索引从0开始。如下图所示：


# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
本节将主要介绍如何访问、修改、遍历、切片等列表、元组的基本操作方法。

## 3.1 访问列表中的元素
要访问列表中的元素，可以使用方括号[]和索引来访问。列表索引从0开始，即第一个元素的索引为0，第二个元素的索引为1，依此类推。如下示例：

```python
myList = [1, "hello", True]
print(myList[0])    #输出: 1
print(myList[1])    #输出: hello
print(myList[-1])   #输出: True (最后一个元素)
```

还可以使用for循环来遍历列表，代码如下：

```python
myList = ["apple", "banana", "orange"]
for fruit in myList:
    print(fruit)     #输出: apple
                    #       banana
                    #       orange
```

## 3.2 修改列表中的元素
要修改列表中的元素，可以使用方括号[]和索引来访问，再赋值给对应的元素。如下示例：

```python
myList = [1, 2, 3, 4, 5]
myList[1] = 'hello'
print(myList)      #输出：[1, 'hello', 3, 4, 5]
```

如果想在列表末尾添加新的元素，可以使用append()方法。如果想在某个位置前插入新元素，可以使用insert()方法，指定位置和值。如下示例：

```python
myList = ['apple', 'banana', 'orange']
myList.append('peach')        #在列表末尾添加元素peach
print(myList)                 #输出：['apple', 'banana', 'orange', 'peach']

myList.insert(1, 'pear')      #在第2个位置前插入元素pear
print(myList)                 #输出：['apple', 'pear', 'banana', 'orange', 'peach']
```

## 3.3 删除列表中的元素
要删除列表中的元素，可以使用del语句或者pop()方法。如果使用del语句，则可以一次性删除多个元素，如果仅删除一个元素，则可以使用pop()方法。如下示例：

```python
myList = [1, 2, 3, 4, 5]
del myList[1:]             #删除列表中所有除第一个元素外的所有元素
print(myList)              #输出：[1]

myList.pop(1)              #删除第2个元素
print(myList)              #输出：[1]
```

## 3.4 列表切片操作
列表切片操作可以从已有的列表中取出部分元素，并生成一个新的列表。切片语法如下：

```python
listName[start:end+1:step]
```

其中，start表示切片开始位置的索引；end+1表示切片结束位置的索引，不包含该位置的元素；step表示步长，默认为1。举例如下：

```python
myList = [1, 2, 3, 4, 5]
newList = myList[:3]          #从列表的开头切片到第三个元素
print(newList)               #[1, 2, 3]

newList = myList[::2]         #从列表的开头切片到结尾，每隔两个取一个元素
print(newList)               #[1, 3, 5]

newList = myList[::-1]        #反转整个列表
print(newList)               #[5, 4, 3, 2, 1]
```

## 3.5 创建元组
元组是不可更改的数据类型，其内容一旦定义完成后便不能修改。元组的创建形式与列表相同，只需要在左右两边加上圆括号即可。

```python
tupleName = ('apple', 'banana', 'orange')
print(type(tupleName))       #<class 'tuple'>
```

## 3.6 比较列表和元组
列表和元组的最大区别在于是否可修改，列表可以增删改查，而元组则不可以。另外，列表和元组在很多情况下可以互换使用，比如说当你只是需要访问列表中的某些元素，但是需要返回的是一个元组时，你可以把列表转换为元组。

## 3.7 Python的多维列表
Python支持多维列表，也就是列表里面还能嵌套列表。如下示例：

```python
myList = [[1, 2], [3, 4]]
print(myList[0][1])           #输出: 2
```

这样就可以实现二维数组的效果。但是，在这种情况下，建议不要在列表之间直接拷贝数据，因为这样容易导致数据的一致性问题。

## 3.8 列表和元组性能方面的探讨
由于列表和元组都是Python自带的数据类型，因此在效率方面来说，它们的速度都很快，而且存储空间也相差无几。因此，选择何种数据类型完全取决于你的需求和场景。
