- Lyft：Lyft是一个美国的共享单车服务平台，其推出的共享单车产品名为“lyft bike”，可以为滴滴出行的用户提供一个方便快捷的共享单车选择。
- Bolt and Go：Bolt and Go是一款由Amtrucks公司推出的全自动轿车配送车。
- Google Maps：谷歌地图是由Google公司提供的免费且强大的地图工具，可用来查找和导航路径、地点、地形信息等。
- Uber：优步是一个美国的综合性移动应用支付平台，依靠AI和大数据技术，帮助用户解决生活中的各种困难，包括打车、租车、外卖和派遣。
- Self Driving Car：自动驾驶汽车是未来必然的方向，在这个新领域里，自动驾驶汽车将会成为真正意义上的“自律机器”。
- Deep Learning：深度学习是指机器学习中的一种方法，它使用多层神经网络的组合来学习数据的特征表示形式，使得机器能够以更高的准确率做出预测或决策。
- Object Detection：目标检测是一种计算机视觉技术，其目标是识别和分类图像中的目标对象，并对它们进行定位。
- Neural Network：神经网络是人工神经元网络的简称，是一种模拟人类大脑工作原理的神经网络模型，其基本结构由输入层、隐藏层和输出层组成。
- Long Short Term Memory (LSTM)：长短期记忆网络（Long Short-Term Memory，LSTM）是一种特殊类型的RNN，其中包含三个门结构，可以实现记忆特性。
- Convolutional Neural Network (CNN)：卷积神经网络（Convolutional Neural Network，CNN）是一种类型神经网络，它是特别适合处理具有空间相关性的数据的神经网络。
- CNN for Pedestrian Detection：卷积神经网络（Convolutional Neural Network，CNN）用于行人检测与跟踪。
- Traffic Sign Recognition：交通标志识别（Traffic sign recognition，TSR）是根据环境中的交通标志，将其识别出来并转换成信号系统，帮助警察及时制止车辆的违法行为。
- GAN：生成对抗网络（Generative Adversarial Networks，GANs），是一种深度学习模型，通过对抗的方式训练两个神经网络，其中一个网络生成模拟真实数据，另一个网络则去判别生成的数据是不是真实的。

## 3.1 AGV自动巡逻车

### 3.1.1 AGV常用功能介绍

AGV（自动巡逻车、全面巡逻车）是指由多台无人机或其他具有巡逻功能的车辆组成的集装箱货运载具，主要用途是进行快速、准确地对大型运输线路中的空闲车站进行监控，发现危险物品和威胁，以及实时跟踪货物在运输过程中可能遇到的任何异常状况。

主要功能：

- 巡检：巡检功能是指通过无人机固定频率不间断的扫描地面区域和路段，以便发现潜在的危险事件或违规行为，例如，油毒、财物遗失、人员非法进入。
- 报警：报警功能是指无人机在发现危险后向相关部门发送报警，包括交通管理部门、警务部门和勘查部门。
- 油泄预警：无人机巡逻时，检测到油污、污染源等危险状况时，可以向相关部门发送通知预警。
- 边缘检测：无人机巡逻时，可以检测到边缘区域的移动情况，判断是否有人员聚集或发生碰撞。
- 摄像头监控：无人机携带摄像头，在前方实时监控整条运输路线，能够辨识危险区域，以及识别运输中存在的人员，以防止人员盗窃或恐怖袭击。
- 轨迹规划：无人机巡逻时，可以通过GPS芯片获取当前位置，并根据系统的计算规划出一条高效的巡航路径。

### 3.1.2 AGV运作流程

AGV的运作流程如下所示：

1. 雷达探测范围：AGV雷达探测范围一般设置在60米到120米之间，不少大型运输线路可以设在300米以内，以满足对运输路线周围的安全保障需求。
2. 摄像头拍照：AGV前部安装多种摄像头，能够实时的拍摄整条运输路线，并且进行分析处理，获取运输过程中的状况信息。
3. 红外避障：AGV采用红外避障装置，能够识别和避免高速路上、铁道桥梁上的敌机。
4. 自主避障：无人机能够在自身内部构造避障塔，检测和过滤掉一切障碍物，精准侦测车辆。
5. GPS：由于AGV实时接收卫星定位数据，可以精确的进行轨迹规划和地图导航，有效控制运输速度和减少违法损失。
6. 智能巡逻：通过摄像头采集的图像数据进行图像分析处理，检测到路况变化或者特殊物件时，能立即停止或调整车辆运行方向。
7. 模糊识别：在各种环境复杂的情况下，无人机巡逻时，AGV还可以进行模糊识别，识别出周围环境的变化，并及时调整运行策略，保障安全。
8. 实时通信：无人机之间的通信系统，可实时跟踪车辆的状态信息，做到及时响应。
9. 设备维护：无人机需要定期检查、维护，确保系统正常运行。

## 3.2 Lyft共享单车

### 3.2.1 Lyft共享单车概述

Lyft是美国的一个共享单车服务平台，其推出的共享单车产品名为“lyft bike”，可以为滴滴出行的用户提供一个方便快捷的共享单车选择。

Lyft bike共有两辆车型：一个是全黑色的Model S，另一个是白色的Model 3。除此之外，还有第三种颜色的单车：绿色Model X。


Lyft Model S：黑色，边框紫色，有两种扬声器、后座位、侧踏板、一档功能。 容量：9.4磅，可以携带八个人。 价格：$28每小时。

Lyft Model 3：白色，边框红色，有两种扬声器、后座位、侧踏板、三档功能。 容量：11.4磅，可以携带六个人。 价格：$35每小时。

Lyft Model X：绿色，边框蓝色，有四种扬声器、后座位、侧踏板、四档功能。 容量：12.4磅，可以携带四个人。 价格：$38每小时。

### 3.2.2 Lyft共享单车运作流程

Lyft的共享单车运作流程如下：

1. 用户注册：用户首先注册账号，然后登录Lyft APP或者Web端，选择自己的共享单车。
2. 预约单车：用户选择想要的共享单车，并点击“预约”按钮，即可完成订单预约。
3. 取车：当订单状态为“待接单”，即用户已完成订单预约，系统就会匹配到最近的一个可用车辆。
4. 行程开始：开始行程后，共享单车与乘客一起开始行驶。
5. 充电：共享单车根据运动距离以及当前电池的容量，估算出需要充电的时间。
6. 服务：在行驶过程中，共享单车会不定期的进行服务。
7. 结束行程：当行程结束后，共享单车会休息几分钟，让乘客准备离开。
8. 归还单车：当乘客准备离开共享单车时，需要通过APP点击“归还”按钮，系统会为他们提供退还车辆的方法。

### 3.2.3 Lyft共享单车优缺点

#### 优点：

1. 简单易用：用户只需要打开手机App或浏览器，就可以轻松找到想要的共享单车。
2. 成本低廉：Lyft的共享单车价格较高，但比起其他服务商的共享单车更加经济。
3. 直观显示：Lyft的共享单车APP提供了直观的剩余电量，实时展示车辆信息。
4. 社区氛围浓厚：Lyft的共享单车APP上提供了众多好评的评论，能够激发更多人的加入。

#### 缺点：

1. 时延：Lyft的共享单车时延较高，基本上秒级响应，但还是不够稳定。
2. 短期内没有车源：Lyft的共享单车目前仅有少部分可用，如果下单数量过多，可能会导致车辆库存不足。
3. 不一定有车：Lyft的共享单车可能会出现“无车可上”的情况。
