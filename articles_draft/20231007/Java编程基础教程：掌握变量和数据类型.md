# 2.核心概念与联系

　　首先，需要对数据类型进行归纳总结。由于Java是一门静态面向对象的编程语言，因此Java语言的基础数据类型只有两种：整型和浮点型。而其他的基本数据类型则由对象来定义，例如Java语言中的String就是一种字符串对象。

　　整数类型：

　　　　1.byte(8位)：-128~127

　　　　2.short(16位)：-32768~32767

　　　　3.int(32位)：-2147483648~2147483647

　　　　4.long(64位)：-9223372036854775808~9223372036854775807

         注：Java语言支持的最大整型数字为Long.MAX_VALUE和Integer.MAX_VALUE之间的值。

　　　　5.BigInteger：可以存储任意大小的整数值，但性能比直接用long差别不大。

　　　　实数类型：

　　　　1.float(32位)：单精度浮点数，适合储存小数。如3.14f或314e-2f，默认值为0.0f。

　　　　2.double(64位)：双精度浮点数，适合储存较大的小数。如3.14d或314e-2d，默认值为0.0d。

　　　　字符类型：char(16位)：用于表示单个字符，可以用'a', 'b', 'c'这样的形式表述。

　　　　布尔类型：boolean：true/false，可以用来控制程序的执行流程。

　　　　Java语言还提供了以下数据类型：

　　　　1.引用类型：用于指向一个对象的地址，主要有3种引用类型：Object、Class和Array。其中Object表示所有类的基类，Class表示Java运行时编译生成的字节码文件，Array表示一个元素的集合。

　　　　2.void类型：Java没有空值，即没有真正意义上的"空值"，void表示没有任何返回值的函数或者方法。

　　　　3.泛型类型：泛型的概念允许不同类型的数据被安全地存储于同一个集合中，它提供了一个统一的接口，使得集合中的元素可以按照不同的类型被操作，提高了代码的灵活性和复用性。Java泛型有两个重要特点：参数化类型和类型擦除。

　　　　4.枚举类型：枚举类型是一个由enum关键字定义的特殊类，它只能保存特定的数据类型（包括int、double、char、boolean等基本类型）或者String，不能保存其他数据类型，并且每个枚举类型都对应唯一的一个常量值。使用枚举类型可以使代码更加易懂、可读、可维护。

# 3.变量的声明、初始化、作用域和生命周期

　　Java语言中，变量是保存在堆内存中的一段内存空间。当我们定义一个变量时，JVM就会在栈内存中创建一个对应的变量，用于存放该变量的值，但是不会在堆内存中开辟新的内存空间。直到真正使用这个变量时，才会在堆内存中开辟内存空间。

　　Java语言中，变量分为两类：实例变量和类变量。

　　　　实例变量（Instance Variable）：是指类的成员变量，属于对象自己的一部分，每个对象都拥有自己独立的实例变量副本。实例变量可以在多个方法中访问，这些方法被称为实例变量的方法。实例变量可以通过“this”关键字访问。实例变量只能在对象创建的时候赋初始值，之后就一直有效直到对象被销毁。实例变量也可以通过构造器方法来赋值。

　　　　类变量（Class Variable）：又叫静态变量，是属于整个类共享的变量，仅有一个副本，所有对象共享这个变量。类变量可以通过“类名.变量名”来访问。类变量在内存中只占据一份空间，所以在系统启动后，类的变量便可以直接使用。类变量的值可以被修改，所以可以通过“类名.变量名=new值”的方式来修改类变量的值。类变量的生命周期始终持续到程序结束。

　　变量的声明方式如下所示：

```java
//实例变量的声明
public class MyClass {
    int instanceVariable;

    //构造器方法给实例变量赋值
    public MyClass() {
        instanceVariable = 10;
    }

    //方法调用实例变量的值
    void printValue(){
        System.out.println("instanceVariable: " + this.instanceVariable);
    }
}

//类变量的声明
public class MyOtherClass {
    static String classVariable = "hello world";
}
```

实例变量和类变量都是在堆内存中开辟新的内存空间，而且在创建对象的时候都会被赋初始值。但是实例变量的方法内部只能访问当前对象的实例变量，类变量的方法内部可以访问整个类的类变量。

　　变量的作用域决定了变量能否在某个范围内使用。Java的变量作用域包括全局作用域、局部作用域和块作用域。

 　　全局作用域：这种作用域变量可以在整个程序中使用。通常情况下，我们习惯将变量的声明语句放在程序的最前面，所有的变量都可以在全局作用域使用。

 　　局部作用域：局部作用域是指在代码块中定义的变量，只能在此代码块内使用。

 　　块作用域：Java还引入了块作用域的概念，块作用域是指在花括号{}中定义的变量，只能在此块内使用。块作用域的变量可以与局部变量重名。

  　　变量的生命周期决定了变量何时应该被释放，一般情况是当对象被垃圾回收时释放内存空间。对象的生命周期包括创建、使用、销毁三部分过程。

    创建阶段：创建对象时，分配内存空间，并在内存中创建对象，初始化成员变量的值。

    使用阶段：对象创建后，就可以使用了，可以为其成员变量设置值、调用方法。

    销毁阶段：当对象不再被使用时，系统自动销毁对象，释放内存空间。对象销毁之后，之前创建的对象引用无法再被访问。

　　因此，变量的生命周期可以简单概括为：创建->使用->销毁。

# 4.进制转换及进制之间的比较

## 4.1.进制

　　　　1.二进制：也就是以0b或0B开头，它的数值表示方式是“0”和“1”，所以数值只能是0和1。0b或0B可以与其他进制混合运算。

　　　　2.八进制：以0开头，数字0到7，数值范围为0～111。八进制可以与其他进制混合运算。

　　　　3.十六进制：以0x或0X开头，数字0到9和A到F（或者a到f），数值范围为0～FFFFH。十六进制也可以与其他进制混合运算。

　　一般来说，二进制、八进制和十六进制均可互相转换。下面的程序演示了进制之间的转换：

```java
public class BinaryDemo {

    public static void main(String[] args) {

        byte b = (byte) 0xff;
        short s = (short) b;
        int i = b;
        long l = b;

        System.out.println(b);    //输出：-1
        System.out.println(s);    //输出：-1
        System.out.println(i);    //输出:-1
        System.out.println(l);    //输出:-1

        System.out.println("=============================");

        Integer ii = new Integer("-1");
        Long ll = new Long("0xFFFL");

        System.out.println(ii.toBinaryString());   //输出：11111111111111111111111111111111
        System.out.println(ll.toBinaryString());   //输出：1111111111111111

        Short ss = new Short((short)-1);
        Byte bb = new Byte((byte)0xff);
        Boolean bo = true? false : true;

        System.out.println(ss.toBinaryString());     //输出：1111111111111111
        System.out.println(bb.toBinaryString());     //输出：11111111
        System.out.println(bo.toString());           //输出："true"
    }
}
```

结果显示，所有整数类型都可以转换为其二进制表示。但由于整数的表示范围限制，大于整数范围的值会丢失其最低位，因此小数类型和Boolean类型也是可以转换成其二进制的。
