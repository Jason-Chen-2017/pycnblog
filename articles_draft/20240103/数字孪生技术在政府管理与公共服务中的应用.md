3. 应用阶段（2017年至今）：在这一阶段，数字孪生技术已经从理论和实验阶段进入了实际应用阶段，被广泛应用于各个行业和领域，包括制造业、能源、交通运输、医疗健康等。

```python
import numpy as np

def pid_control(setpoint, process_variable, Kp, Ki, Kd):
    integral = 0
    derivative = 0
    output = 0

    while True:
        error = setpoint - process_variable
        integral += error
        derivative = (error - previous_error)
        previous_error = error
        output = Kp * error + Ki * integral + Kd * derivative
        process_variable += output
```
