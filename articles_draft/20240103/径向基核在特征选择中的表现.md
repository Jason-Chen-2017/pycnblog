# 2.核心概念与联系

## 2.1 支持向量机（SVM）

支持向量机（Support Vector Machine, SVM）是一种常用的监督学习方法，它可以用于分类和回归问题。SVM的核心思想是通过找出最大间隔的超平面来将数据分割为不同的类别。在实际应用中，由于数据集可能存在非线性关系，因此需要使用核函数（kernel function）将原始数据映射到高维空间，以便在新的空间中找到最大间隔的超平面。

## 2.2 径向基核（RBF）

径向基核（Radial Basis Function, RBF）是一种常用的核函数，它可以用于描述数据点之间的相似性。径向基核的基本形式如下：

$$
K(x, y) = \exp(-\gamma \|x - y\|^2)
$$

其中，$x$ 和 $y$ 是数据点，$\gamma$ 是一个正参数，用于控制核函数的宽度。

1. 径向基核可以用于计算数据点之间的相似性，从而帮助我们选择那些对模型性能有最大贡献的特征。
2. 径向基核可以用于构建支持向量机（SVM）模型，SVM模型在特征选择方面具有很好的表现。
3. 径向基核可以用于处理高维数据，并在高维空间中找到最大间隔的超平面，从而提高模型的泛化能力。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来展示径向基核在特征选择中的表现。

```python
import numpy as np
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score
from sklearn.feature_selection import SelectKBest, f_classif

# 加载鸢尾花数据集
iris = load_iris()
X = iris.data
y = iris.target

# 数据预处理
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 训练集和测试集划分
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# 特征选择
selector = SelectKBest(f_classif, k=2)
X_train_selected = selector.fit_transform(X_train, y_train)
X_test_selected = selector.transform(X_test)

# SVM模型训练
svc = SVC(kernel='rbf', gamma='auto')
svc.fit(X_train_selected, y_train)

# 模型评估
y_pred = svc.predict(X_test_selected)
accuracy = accuracy_score(y_test, y_pred)
print(f'准确率：{accuracy:.4f}')
```

在上述代码中，我们首先加载了鸢尾花数据集，并对数据进行了预处理。然后，我们使用径向基核（RBF）的特征选择方法（`SelectKBest` 和 `f_classif`）来选择那些对模型性能有最大贡献的特征。接着，我们使用支持向量机（SVM）模型进行训练和评估，并计算了模型的准确率。
