我们使用均方误差（MSE）作为损失函数：

$$
L(\theta) = \frac{1}{2n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
$$

### 4.1.4 禁忌搜索应用

我们将禁忌搜索应用于优化模型参数 $\theta$，以最小化损失函数 $L(\theta)$。

```python
import numpy as np
from tabu import TabuSearch

# 数据准备
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
Y = np.array([1, 2, 3, 4])

# 模型定义
def linear_model(X, theta):
    return np.dot(X, theta)

# 损失函数定义
def mse_loss(Y, Y_hat):
    return np.mean((Y - Y_hat) ** 2)

# 禁忌搜索应用
tabu_search = TabuSearch(X.shape[1], mse_loss, linear_model, max_iter=1000)
theta_opt = tabu_search.search(Y)

print("最优参数：", theta_opt)
```
