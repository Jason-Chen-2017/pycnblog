### 交互算法的代码实例

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

class Ball {
public:
    Ball(float x, float y, float z, float radius)
        : x_(x), y_(y), z_(z), radius_(radius) {}

    void move(float dt) {
        x_ += dt;
        y_ += dt;
        z_ += dt;
    }

    float distance(const Ball& other) const {
        float dx = x_ - other.x_;
        float dy = y_ - other.y_;
        float dz = z_ - other.z_;
        return std::sqrt(dx * dx + dy * dy + dz * dz);
    }

private:
    float x_, y_, z_;
    float radius_;
};

int main() {
    Ball ball1(0, 0, 0, 1);
    Ball ball2(2, 2, 2, 1);
    float dt = 0.1;
    while (true) {
        float distance = ball1.distance(ball2);
        std::cout << "Distance: " << distance << std::endl;
        ball1.move(dt);
        ball2.move(dt);
        if (distance < 1) {
            break;
        }
    }
    return 0;
}
```

上述代码实例是一个简单的交互算法示例，它模拟了两个球体的运动和相互作用。代码首先定义了一个名为`Ball`的类，该类包括位置和半径等属性，以及移动和计算距离等方法。接着，在`main`函数中，创建两个球体，设置时间步长，并进入一个循环中，在循环中，计算两个球体之间的距离，并将球体移动一定距离。当两个球体之间的距离小于1时，循环停止。

### 多人互动算法的代码实例

```cpp
#include <iostream>
#include <thread>
#include <mutex>

std::mutex mtx;

void producer() {
    int count = 0;
    while (true) {
        std::lock_guard<std::mutex> lock(mtx);
        std::cout << "Producer: " << ++count << std::endl;
        std::this_thread::sleep_for(std::chrono::seconds(1));
    }
}

void consumer() {
    int count = 0;
    while (true) {
        std::lock_guard<std::mutex> lock(mtx);
        std::cout << "Consumer: " << ++count << std::endl;
        std::this_thread::sleep_for(std::chrono::seconds(2));
    }
}

int main() {
    std::thread t1(producer);
    std::thread t2(consumer);
    t1.join();
    t2.join();
    return 0;
}
```

上述代码实例是一个简单的多人互动算法示例，它模拟了生产者-消费者问题。代码首先定义了一个名为`mtx`的互斥锁，然后定义了两个线程`producer`和`consumer`，分别模拟了生产者和消费者的行为。生产者在一个循环中不断生产产品，消费者在另一个循环中不断消费产品。两个线程使用互斥锁保护共享资源，确保线程安全。
