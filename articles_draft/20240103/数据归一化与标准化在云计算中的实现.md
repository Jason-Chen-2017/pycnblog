## 1.2 核心概念与联系

数据归一化是一种数据预处理方法，它的目的是为了使数据更加规范、可读性更强，以便于进行后续的数据分析、机器学习等工作。数据归一化可以将数据转换为一个有限的范围内的数值，以便于后续的数据处理。

数据归一化可以分为以下几种：

1. 范围归一化：将数据的范围限制在一个有限的范围内，例如将数据的范围限制在0到1之间。
2. 比例归一化：将数据的范围限制在一个固定的比例范围内，例如将数据的范围限制在0到100之间。
3. 标准化：将数据转换为一个标准的格式，例如将数据转换为浮点数。

2. 将数据的最大值和最小值赋给一个常量。

$$

## 1.4 具体代码实例和详细解释说明

### 1.4.1 范围归一化代码实例和详细解释说明

```python
import numpy as np

# 原始数据
data = np.array([1, 2, 3, 4, 5])

# 计算最大值和最小值
max_val = np.max(data)
min_val = np.min(data)

# 范围归一化
normalized_data = (data - min_val) / (max_val - min_val)

print(normalized_data)
```

### 1.4.2 比例归一化代码实例和详细解释说明
