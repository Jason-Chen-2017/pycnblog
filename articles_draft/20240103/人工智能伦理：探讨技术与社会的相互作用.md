## 3.1 监督学习

监督学习是一种机器学习方法，它需要一组已知的输入和输出数据来训练模型。通过这些数据，模型可以学习出如何从输入中预测输出。监督学习的一个常见应用是分类问题，例如图像识别、语音识别等。

### 3.1.1 逻辑回归

逻辑回归是一种用于二分类问题的监督学习算法。它通过学习一个逻辑函数来预测输入数据的两个类别之间的关系。逻辑回归的数学模型公式如下：

$$
P(y=1|x;\theta) = \frac{1}{1 + e^{-(\theta_0 + \theta_1x_1 + \theta_2x_2 + ... + \theta_nx_n)}}
$$

其中，$x$ 是输入特征向量，$y$ 是输出类别（1或0），$\theta$ 是模型参数，$e$ 是基数常数。

### 3.1.2 梯度下降

梯度下降是一种优化算法，用于最小化一个函数。在逻辑回归中，梯度下降可以用于最小化损失函数，从而更新模型参数。梯度下降的数学公式如下：

$$
\theta_{new} = \theta_{old} - \alpha \nabla_{\theta} J(\theta)
$$

其中，$\theta_{new}$ 是更新后的模型参数，$\theta_{old}$ 是旧的模型参数，$\alpha$ 是学习率，$J(\theta)$ 是损失函数，$\nabla_{\theta}$ 是梯度。

## 3.2 无监督学习

无监督学习是一种机器学习方法，它不需要已知的输入和输出数据来训练模型。无监督学习的一个常见应用是聚类问题，例如图像分类、文本摘要等。

### 3.2.1 聚类算法：K均值

K均值是一种无监督学习算法，它将输入数据分为K个聚类。K均值的数学模型公式如下：

$$
\arg\min_{\theta} \sum_{i=1}^K \sum_{x \in C_i} ||x - \mu_i||^2
$$

其中，$C_i$ 是第i个聚类，$\mu_i$ 是第i个聚类的中心。

## 4.1 监督学习：逻辑回归

我们将通过一个简单的二分类问题来演示逻辑回归的应用。假设我们有一组数据，其中包含两种类别的输入特征，我们需要预测这些特征所属的类别。

```python
import numpy as np

# 生成数据
X = np.random.rand(100, 2)
y = (X[:, 0] > 0.5).astype(int)

# 初始化模型参数
theta = np.zeros(2)

# 设置学习率和迭代次数
alpha = 0.01
iterations = 1000

# 训练模型
for i in range(iterations):
    predictions = X.dot(theta)
    loss = (1 / len(X)) * np.sum(np.logadder(1 + np.exp(-y * predictions)))
    gradients = (1 / len(X)) * np.dot(X.T, (y - 1 / (1 + np.exp(-y * predictions))) * np.exp(-y * predictions))
    theta = theta - alpha * gradients

# 预测
X_test = np.array([[0.1, 0.2], [-0.3, -0.4]])
predictions = X_test.dot(theta)
```

在这个例子中，我们首先生成了一组随机数据，并将其划分为两个类别。然后，我们初始化了模型参数，设置了学习率和迭代次数，并使用梯度下降算法训练逻辑回归模型。最后，我们使用训练好的模型对新数据进行预测。
