## 4.1 代码实例
考虑一个简单的线性回归问题，我们的目标是在一个有限的表示空间中找到一个最佳的线性回归模型。这个问题可以通过以下代码实例来解决：

```python
import numpy as np

# 生成训练数据
np.random.seed(0)
X_train = np.random.rand(100, 2)
y_train = np.dot(X_train, np.array([1, -1])) + np.random.randn(100)

# 定义目标函数
def loss(w, X_train, y_train):
    y_pred = np.dot(X_train, w)
    return np.mean((y_pred - y_train) ** 2)

# 定义搜索策略
def search(X_train, y_train, w_min, w_max, num_iter):
    w = (w_min + w_max) / 2
    for _ in range(num_iter):
        w -= 0.01 * np.gradient(loss(w, X_train, y_train), w)
    return w

# 执行搜索
w_min = -10
w_max = 10
num_iter = 1000
w = search(X_train, y_train, w_min, w_max, num_iter)

# 预测
X_test = np.array([[0.5, 0.5], [-0.5, -0.5]])
y_pred = np.dot(X_test, w)
print("y_pred:", y_pred)
```

在这个代码实例中，我们首先生成了一个训练数据集，并定义了一个目标函数（损失函数）来衡量模型的性能。接着，我们定义了一个搜索策略，这个策略通过梯度下降来优化目标函数。最后，我们执行了搜索，并用得到的最佳权重来预测新的测试数据。

## 4.2 详细解释说明
在这个代码实例中，我们首先生成了一个训练数据集，其中包括了2个特征的100个样本。接着，我们定义了一个线性回归模型，这个模型通过计算预测值和真实值之间的均方误差来衡量性能。接着，我们定义了一个搜索策略，这个策略通过梯度下降来优化目标函数。最后，我们执行了搜索，并用得到的最佳权重来预测新的测试数据。
