## 1.2 核心概念与联系

### 1.2.1 密切圆的定义和性质

密切圆是一种在给定点上与某条曲线接触的圆。密切圆可以分为两种类型：内密切圆和外密切圆。内密切圆是在曲线上方的圆，而外密切圆是在曲线下方的圆。密切圆的中心称为该点的切心，切心位于曲线弧长为零时。密切圆的半径称为该点的切径。

密切圆具有以下性质：

- 对于任何一条连续可导的曲线，在任何一点上都有且仅有一个内密切圆和一个外密切圆。
- 密切圆的半径与曲线的切角成正比，即 $r = \frac{1}{|\kappa|}$，其中 $r$ 是密切圆的半径，$\kappa$ 是曲率值。
- 密切圆的中心位于曲线的切点，即 $C = P + r\frac{dP}{ds}$，其中 $C$ 是密切圆的中心，$P$ 是曲线上的任意点，$r$ 是密切圆的半径，$\frac{dP}{ds}$ 是曲线在点 $P$ 处的切向量。

### 1.2.2 曲率的定义和性质

曲率是描述曲线在某一点上弧度变化率的量。曲率可以通过以下公式计算：

$$
\kappa = \frac{|d\vec{T}/ds|}{||\vec{r}'(s)||}
$$

其中 $\kappa$ 是曲率值，$\vec{T}$ 是曲线在点 $P$ 处的切向量，$\vec{r}'(s)$ 是曲线的速度向量。

曲率具有以下性质：

- 曲率值的正负号表示曲线在该点的方向。
- 曲率值与曲线的弧长成正比，即 $\kappa = \frac{1}{R}$，其中 $R$ 是曲线在该点的半径。
- 曲率值与曲线的变化率成正比，即 $\kappa = \frac{|d\vec{T}/ds|}{||\vec{r}'(s)||}$。

### 1.2.3 密切圆与曲率的关系

密切圆和曲率之间存在密切的关系。根据上述定义和性质，我们可以得出以下关系：

$$
r = \frac{1}{|\kappa|}
$$

其中 $r$ 是密切圆的半径，$\kappa$ 是曲率值。这个关系表明，密切圆的半径与曲线在该点的曲率值成反比。

在本节中，我们将提供一个 Python 代码实例，以展示如何计算密切圆和曲率。

```python
import numpy as np

def curvature(r, dp_ds):
    return np.abs(np.dot(dp_ds, r)) / np.linalg.norm(r)

def tangent_circle(r, dp_ds, s):
    r_inv = 1 / curvature(r, dp_ds)
    center = s + r_inv * dp_ds
    return center

# 示例曲线
r = np.array([0, 1, 2, 3, 4, 5])
dp_ds = np.array([1, 2, 3, 4, 5, 6])
s = np.linspace(0, 1, 6)

# 计算曲率
kappa = curvature(r, dp_ds)
print("曲率值:", kappa)

# 计算密切圆
center = tangent_circle(r, dp_ds, s[0])
print("密切圆中心:", center)
```
