## 2. 核心概念与联系

在数据挖掘中，距离度量是一种用于衡量两个数据点之间距离的方法。距离度量可以帮助我们了解数据之间的关系和相似性，进而实现数据挖掘的目标。常见的距离度量包括欧几里得距离、曼哈顿距离、余弦相似度等。

### 2.1 欧几里得距离

欧几里得距离是一种用于衡量两个点之间距离的方法，它是基于欧几里得几何空间中的距离定义的。欧几里得距离的公式为：

$$
d(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2 + \cdots + (x_n - y_n)^2}
$$

其中，$x = (x_1, x_2, \cdots, x_n)$ 和 $y = (y_1, y_2, \cdots, y_n)$ 是两个数据点，$n$ 是数据点的维度。

### 2.2 曼哈顿距离

曼哈顿距离是一种用于衡量两个点之间距离的方法，它是基于曼哈顿几何空间中的距离定义的。曼哈顿距离的公式为：

$$
d(x, y) = |x_1 - y_1| + |x_2 - y_2| + \cdots + |x_n - y_n|
$$

其中，$x = (x_1, x_2, \cdots, x_n)$ 和 $y = (y_1, y_2, \cdots, y_n)$ 是两个数据点，$n$ 是数据点的维度。

### 2.3 余弦相似度

余弦相似度是一种用于衡量两个向量之间相似度的方法，它是基于两个向量之间的内积和其长度的关系定义的。余弦相似度的公式为：

$$

$$

其中，$x$ 和 $y$ 是两个向量，$x \cdot y$ 是 $x$ 和 $y$ 的内积，$\|x\|$ 和 $\|y\|$ 是 $x$ 和 $y$ 的长度。

### 4.1 欧几里得距离

```python
import numpy as np

def euclidean_distance(x, y):
    return np.sqrt(np.sum((x - y) ** 2))

x = np.array([1, 2, 3])
y = np.array([4, 5, 6])

distance = euclidean_distance(x, y)
print(distance)
```
