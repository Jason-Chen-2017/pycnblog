# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的销售预测示例来展示数据驱动的供应链管理的具体代码实例和解释。

1. **数据收集**：

我们假设已经收集到了一份销售数据，包括时间、产品类别、销售额等信息。

```python
import pandas as pd

data = {
    'time': ['2021-01', '2021-02', '2021-03', '2021-04', '2021-05'],
    'product_category': ['A', 'A', 'A', 'B', 'B'],
    'sales': [100, 120, 150, 200, 250]
}

df = pd.DataFrame(data)
```

2. **数据处理**：

我们需要将数据转换为适合训练模型的格式。

```python
X = df[['product_category']]
y = df['sales']
```

3. **模型训练**：

我们使用线性回归模型进行销售预测。

```python
from sklearn.linear_model import LinearRegression

model = LinearRegression()
model.fit(X, y)
```

4. **模型应用**：

我们可以使用训练好的模型进行销售预测。

```python
X_new = pd.DataFrame({'product_category': ['C']})
y_pred = model.predict(X_new)
print(y_pred)
```
