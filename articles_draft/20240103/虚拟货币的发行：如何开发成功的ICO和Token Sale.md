## 1.1 ICO简介

ICO（Initial Coin Offering）是一种通过区块链技术实现的资金筹集方式，通过发行数字货币（即代币）来吸引投资者。ICO的核心思想是通过代币销售来获取资金，并将这些资金用于项目的开发和运营。ICO的目的是为了帮助创新项目获取资金，以实现其业务模式的发展和扩展。

## 1.2 Token Sale简介

Token Sale是一种通过区块链技术实现的资产分配方式，通过发行数字代币来吸引投资者。Token Sale的核心思想是通过代币销售来分配资产，并将这些资产用于项目的开发和运营。Token Sale的目的是为了帮助创新项目分配资产，以实现其业务模式的发展和扩展。

## 1.3 ICO和Token Sale的区别

虽然ICO和Token Sale在实现上有所不同，但它们的核心思想和目的是一样的。ICO通常用于筹集资金，而Token Sale通常用于分配资产。在实际应用中，这两种方式往往会相互结合，以实现更为全面的资源整合和项目发展。
