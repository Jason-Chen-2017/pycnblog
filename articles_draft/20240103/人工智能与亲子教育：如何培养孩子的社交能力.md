```python
import pandas as pd

data = {
    'name': ['Alice', 'Bob', 'Charlie'],
    'age': [10, 12, 14],
    'interest': ['reading', 'sports', 'music'],
    'learning_ability': [80, 90, 85]
}

df = pd.DataFrame(data)
print(df)
```

为了分析孩子的个性化信息，可以使用以下代码实例：

```python
def analyze_data(df):
    df['total_score'] = df['learning_ability'] + df['interest'].apply(lambda x: {'reading': 100, 'sports': 100, 'music': 90}.get(x))
    df['average_score'] = df['total_score'] / df['age']
    return df

df = analyze_data(df)
print(df)
```

# 4.3 根据分析结果提供个性化教育方法
为了根据分析结果提供个性化教育方法，可以使用以下代码实例：

```python
def provide_education(df):
    df['education'] = df.apply(lambda row: 'reading' if row['interest'] == 'reading' else 'sports' if row['interest'] == 'sports' else 'music', axis=1)
    return df

df = provide_education(df)
print(df)
```
