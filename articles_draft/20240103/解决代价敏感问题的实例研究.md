3. 根据当前节点的代价信息，选择最佳的分裂特征。
4. 递归地对当前节点的子节点进行相同的操作，直到满足停止条件。
5. 返回构建好的决策树。

### 4.1 代价敏感决策树
我们将通过一个简单的例子来展示如何使用代价敏感决策树来解决代价敏感问题。假设我们有一个数据集，包含两个特征和一个代价信息，我们需要找到最佳的决策树模型。

```python

from sklearn.tree import DecisionTreeClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 加载数据集

X = iris.data
y = iris.target

# 添加代价信息

X = np.column_stack((X, cost))
```
