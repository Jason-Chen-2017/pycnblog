## 4.1问题描述

考虑一个简单的优化问题，目标是最小化以下目标函数：

$$
f(x) = -x^2 + 4x
$$

其中，$x$ 是一个实数。我们需要找到使目标函数取得最小值的 $x$ 的值。

## 4.2代码实现

```python
import numpy as np

def f(x):
    return -x**2 + 4*x

def bat_algorithm(pop_size, max_iter, lower_bound, upper_bound):
    # 初始化蝙蝠群的位置和速度
    x = np.random.uniform(lower_bound, upper_bound, pop_size)
    v = np.zeros(pop_size)
    
    # 初始化最优解
    best_x = x[np.argmin(f(x))]
    best_f = f(best_x)
    
    # 主循环
    for _ in range(max_iter):
        # 更新蝙蝠的速度和位置
        for i in range(pop_size):
            v[i] = v[i] + c * np.random.rand() * (best_x - x[i]) + (best_x - x[i]) * np.random.rand()
            x[i] = x[i] + v[i]
            
            # 更新蝙蝠群的最优解
            if f(x[i]) < best_f:
                best_f = f(x[i])
                best_x = x[i]
                
    return best_x, best_f

# 设置参数
pop_size = 20
max_iter = 100
lower_bound = -10
upper_bound = 10

# 运行蝙蝠算法
best_x, best_f = bat_algorithm(pop_size, max_iter, lower_bound, upper_bound)

# 输出结果
print("最优解: x =", best_x, "f(x) =", best_f)
```

在 `bat_algorithm` 函数中，我们首先初始化了蝙蝠群的位置和速度，并找到了初始最优解。接着，我们进入了主循环，每次循环中，我们根据蝙蝠的速度和位置更新蝙蝠的位置，并更新蝙蝠群的最优解。主循环结束后，我们返回了最优解和最优值。
