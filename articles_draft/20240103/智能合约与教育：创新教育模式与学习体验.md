# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释智能合约的实现过程。

## 4.1 智能合约代码实例

我们以一个简单的智能合约实例来进行说明，该智能合约用于自动评估学生的成绩。

```solidity
pragma solidity ^0.5.0;

contract GradeContract {
    struct Student {
        uint id;
        string name;
        uint score;
    }

    Student[] public students;

    function addStudent(uint id, string memory name, uint score) public {
        students.push(Student(id, name, score));
    }

    function getStudentScore(uint id) public view returns (uint) {
        for (uint i = 0; i < students.length; i++) {
            if (students[i].id == id) {
                return students[i].score;
            }
        }
        return 0;
    }
}
```

## 4.2 详细解释说明

1. 首先，我们使用Solidity编程语言编写智能合约代码。
2. 我们定义了一个结构`Student`，包含学生的ID、姓名和成绩等信息。
3. 我们定义了一个公共数组`students`，用于存储学生信息。
4. 我们定义了一个`addStudent`函数，用于添加学生信息到数组中。
5. 我们定义了一个`getStudentScore`函数，用于获取学生的成绩。
6. 我们使用`pragma solidity ^0.5.0`指令，表示使用Solidity版本0.5.0。
