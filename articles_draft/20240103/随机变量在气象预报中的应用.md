随机变量的期望和方差是随机变量的基本统计量，可以用来描述随机变量的性质。

### 3.1.1 期望

随机变量的期望是一个数值，用来描述随机变量的平均值。期望的公式为：

$$
E[X] = \sum_{i=1}^{n} x_i P(x_i)
$$

其中，$x_i$ 是随机变量的取值，$P(x_i)$ 是随机变量的概率分布。

### 3.1.2 方差

随机变量的方差是一个数值，用来描述随机变量的不确定性。方差的公式为：

$$
Var[X] = E[X^2] - (E[X])^2
$$

其中，$E[X^2]$ 是随机变量的二次期望，$(E[X])^2$ 是随机变量的平方期望。

## 3.2 随机变量的独立性

两个随机变量$X$和$Y$是独立的，如果它们的联合分布等于积分分布，即：

# 假设雨量、温度、风力的分布为正态分布

temp_mu, temp_sigma = 20, 2
wind_mu, wind_sigma = 10, 2

wind_mean = np.exp(np.log(wind_mu) + 0.5 * np.log(1 + wind_sigma**2))

# 计算雨量、温度、风力的方差

## 4.2 气象模型的建立和验证

```python
import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

# 假设气象数据中的雨量、温度、风力是独立的随机变量，其分布为正态分布
rain_data = np.random.normal(rain_mu, rain_sigma, 1000)
temp_data = np.random.normal(temp_mu, temp_sigma, 1000)
wind_data = np.random.normal(wind_mu, wind_sigma, 1000)

# 建立气象模型
X = np.column_stack((rain_data, temp_data, wind_data))
y = np.sin(X.sum(axis=1)) + np.random.normal(0, 0.1, 1000)

model = LinearRegression()
model.fit(X, y)

# 验证气象模型
y_pred = model.predict(X)
mse = mean_squared_error(y, y_pred)
print('模型准确性：', 1 - mse)
```
