    def __init__(self, name, probability):
        self.name = name
        self.probability = probability

# 定义事件独立性的函数
def event_independence(events):
    for i in range(len(events)):
        for j in range(i+1, len(events)):
            if events[i].name == events[j].name:
                return False
    return True

# 创建事件数据

# 检查事件独立性

## 4.2 事件存储的代码实例

我们可以使用 Python 语言来实现事件存储的代码实例。以下是一个简单的例子：

```python
# 定义事件数据的类
class Event:
    def __init__(self, name, attributes):
        self.name = name
        self.attributes = attributes

# 定义事件存储的类
class EventStore:
    def __init__(self):
        self.events = []

    def add_event(self, event):
        self.events.append(event)

    def get_event(self, name):
        for event in self.events:
            if event.name == name:
                return event
        return None

# 创建事件存储对象
event_store = EventStore()

# 创建事件数据
event1 = Event("A", {"attribute1": "value1", "attribute2": "value2"})
event2 = Event("B", {"attribute1": "value3", "attribute2": "value4"})

# 添加事件到事件存储
event_store.add_event(event1)
event_store.add_event(event2)

# 获取事件
event = event_store.get_event("A")
print(event.name, event.attributes)
```

在这个例子中，我们首先定义了一个事件数据的类，并创建了两个事件对象。然后，我们使用事件存储的类来创建一个事件存储对象，并将这两个事件添加到事件存储中。最后，我们使用事件存储的 get_event 方法来获取一个事件对象，并打印出事件名称和属性。
