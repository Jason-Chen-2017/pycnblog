# 2. 核心概念与联系

## 2.1 数值积分

数值积分是数值计算中一个重要的问题，它涉及到将多变函数的积分求值为一个数值。数值积分方法的主要目标是将积分问题转化为求和问题，然后通过计算求和的近似值来得到积分的近似值。

常见的数值积分方法有：

1. 左端积分（Left Endpoint Integration）
2. 右端积分（Right Endpoint Integration）
3. 中点积分（Midpoint Integration）
4. 梯形法（Trapezoidal Rule）
5. 三角法（Simpson's Rule）
6. 蒙特卡洛积分（Monte Carlo Integration）

## 2.2 蒙特卡洛方法

蒙特卡洛方法是一种基于随机性的数值计算方法，它主要应用于解决数值积分、求解方程和优化问题。这种方法的核心思想是通过大量的随机样本来估计不确定性问题的解，从而得到近似的结果。

蒙特卡洛方法的主要优点是它无需知道解的表达式，只需要知道问题的概率分布和约束条件，因此对于许多复杂问题具有很大的优势。

## 3.1 蒙特卡洛积分公式

假设我们要求解一个区间 $[a, b]$ 上的积分：

$$
\int_a^b f(x) dx
$$

其中 $f(x)$ 是一个已知的函数。我们可以将区间 $[a, b]$ 划分为 $n$ 个等长子区间，并为每个子区间选取一个随机点 $x_i$，然后计算这些随机点的函数值的和：

$$
\sum_{i=1}^n f(x_i) \Delta x
$$

其中 $\Delta x = \frac{b-a}{n}$ 是子区间的长度。随着 $n$ 的增加，这个和的估计值将逼近真实的积分值。

## 4.1 简单的蒙特卡洛积分示例

我们来看一个简单的蒙特卡洛积分示例，假设我们要求解一个区间 $[0, 1]$ 上的积分：

$$
\int_0^1 x^2 dx
$$

我们可以将区间 $[0, 1]$ 划分为 $n$ 个等长子区间，并为每个子区间选取一个随机点 $x_i$，然后计算这些随机点的函数值的和：

$$
\sum_{i=1}^n f(x_i) \Delta x
$$

其中 $\Delta x = \frac{1}{n}$ 是子区间的长度。随着 $n$ 的增加，这个和的估计值将逼近真实的积分值。

以下是一个简单的Python代码实现：

```python
import random

def f(x):
    return x**2

def monte_carlo_integration(a, b, n):
    delta_x = (b - a) / n
    sum_x = 0
    for _ in range(n):
        x = random.uniform(a, b)
        sum_x += f(x)
    return sum_x * delta_x

a = 0
b = 1
n = 10000
result = monte_carlo_integration(a, b, n)
print(result)
```
