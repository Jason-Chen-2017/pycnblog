## 2.1 矩阵迹的基本定义

给定一个矩阵A，其元素为a\_{ij}，矩阵迹定义为：

$$
tr(A) = \sum_{i=1}^{n} a_{ii}
$$

其中，n是矩阵A的阶数。

## 2.2 矩阵迹的性质

1. 线性性：对于任意的矩阵A和B，以及实数k，有：

$$
tr(kA + B) = ktr(A) + tr(B)
$$

2. 交换性：对于任意的矩阵A和B，有：

$$
tr(AB) = tr(BA)
$$

3. 分配性：对于任意的矩阵A、B和C，有：

$$
tr(A(BC)) = tr((AB)C)
$$

## 4.1 平均滤波算法实现

```python
import numpy as np
import cv2

def average_filtering(image, kernel_size):
    rows, cols = image.shape[:2]
    kernel = np.ones((kernel_size, kernel_size), np.float32) / (kernel_size * kernel_size)
    result = cv2.filter2D(image, -1, kernel)
    return result

kernel_size = 3
filtered_image = average_filtering(image, kernel_size)
cv2.imshow('Filtered Image', filtered_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

在上述代码中，我们首先导入了numpy和cv2库，然后定义了一个平均滤波函数`average_filtering`，其中`image`是输入图像，`kernel_size`是滤波核的大小。在函数中，我们首先获取图像的行数和列数，然后创建一个滤波核，其中元素值为滤波核的大小的 reciprocal，即`1 / (kernel_size * kernel_size)`。接着，我们使用`cv2.filter2D`函数对图像进行滤波处理，最后显示处理后的图像。

## 4.2 自适应均值减少增强算法实现

```python
import numpy as np
import cv2

def adaptive_mean_reduction_enhancement(image, block_size, scale_factor):
    rows, cols = image.shape[:2]
    result = np.zeros_like(image)
    for i in range(rows):
        for j in range(cols):
            block = image[i:i+block_size, j:j+block_size]
            mean_value = np.mean(block)
            result[i, j] = image[i, j] + scale_factor * (mean_value - image[i, j])
    return result

block_size = 3
scale_factor = 10
enhanced_image = adaptive_mean_reduction_enhancement(image, block_size, scale_factor)
cv2.imshow('Enhanced Image', enhanced_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

在上述代码中，我们首先导入了numpy和cv2库，然后定义了一个自适应均值减少增强函数`adaptive_mean_reduction_enhancement`，其中`image`是输入图像，`block_size`是块大小，`scale_factor`是缩放因子。在函数中，我们首先获取图像的行数和列数，然后创建一个与输入图像大小相同的结果图像。接着，我们遍历图像的每个像素点，获取其周围的块，计算块的均值，并将当前像素点的值增强为均值值加上缩放因子的差值。最后显示处理后的图像。
