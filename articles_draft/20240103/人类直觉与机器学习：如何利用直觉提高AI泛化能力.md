## 4.1 直觉规则系统实例

```python
class Rule:
    def __init__(self, condition, action):
        self.condition = condition
        self.action = action

class RuleSystem:
    def __init__(self):
        self.rules = []

    def add_rule(self, rule):
        self.rules.append(rule)

    def match(self, condition):
        for rule in self.rules:
            if rule.condition == condition:
                return rule.action
        return None

# 添加直觉规则
rule1 = Rule("天气好", "去外面散步")
rule2 = Rule("天气糟", "留在家里")
rule3 = Rule("天气好", "穿短裤")

# 创建直觉规则系统
rs = RuleSystem()
rs.add_rule(rule1)
rs.add_rule(rule2)
rs.add_rule(rule3)

# 匹配规则并执行
condition = "天气好"
action = rs.match(condition)
print(action)  # 输出：去外面散步
```

在上述代码实例中，我们首先定义了`Rule`类，用于表示直觉规则。然后定义了`RuleSystem`类，用于表示直觉规则系统。接着，我们添加了一些直觉规则，并创建了一个直觉规则系统。最后，我们匹配了规则并执行了相应的决策。
