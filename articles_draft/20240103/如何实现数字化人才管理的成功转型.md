## 4.具体代码实例和详细解释说明

### 4.1 数据挖掘算法代码实例

```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression

# 加载数据
data = pd.read_csv('data.csv')

# 选择特征和目标变量
X = data[['age', 'salary', 'experience']]
X = X.values
y = data['performance']
y = y.values

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 预测
X_new = np.array([[25, 3000, 5]])
y_predict = model.predict(X_new)
print(y_predict)
```
