对偶变换是将原向量空间中的基转换为对偶空间中的基的过程。对偶变换通常用T表示，可以表示为：

$$

1. 对偶变换是线性的，即对于任意向量v1, v2在V中和函数f, g在V*中，有：

$$

$$

$$

2. 对偶变换可以用来计算原向量空间中向量的内积、范数等属性。

### 4.1 最小二乘法

```python
import numpy as np

A = np.array([[1, 2], [3, 4]])
b = np.array([5, 6])

x = np.linalg.lstsq(A, b, rcond=None)[0]
print(x)
```

上述代码使用numpy库中的lstsq函数来解决线性方程组，得到最小二乘法的解。

### 4.2 高斯消元

```python

```

上述代码使用numpy库中的solve函数来解决线性方程组，得到高斯消元的解。

### 4.3 奇异值分解

```python
import numpy as np

A = np.array([[1, 2], [3, 4]])

U, S, V = np.linalg.svd(A)

```

上述代码使用numpy库中的svd函数来计算矩阵A的奇异值分解。
