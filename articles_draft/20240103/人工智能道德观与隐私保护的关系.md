### 4.1数据脱敏代码实例

在本节中，我们将通过一个Python代码实例来演示如何将姓名转换为代码：

```python
import hashlib

def anonymize_name(name):
    name_hash = hashlib.sha256(name.encode('utf-8')).hexdigest()
    anonymized_name = name_hash[:8]
    return anonymized_name

name = "John Doe"
anonymized_name = anonymize_name(name)
print(anonymized_name)
```

在这个代码实例中，我们使用了SHA-256哈希算法将姓名转换为唯一的代码。这样，我们可以确保姓名不会被直接识别。

### 4.2数据加密代码实例

在本节中，我们将通过一个Python代码实例来演示如何使用AES算法对数据进行加密：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

def encrypt_data(data, key):
    cipher = AES.new(key, AES.MODE_ECB)
    encrypted_data = cipher.encrypt(data)
    return encrypted_data

data = b"Hello, World!"
key = get_random_bytes(16)
encrypted_data = encrypt_data(data, key)
print(encrypted_data)
```

在这个代码实例中，我们使用了AES算法将数据加密为只有具有特定密钥才能解密的形式。这样，我们可以确保数据不会被未经授权的访问。

    return groups
