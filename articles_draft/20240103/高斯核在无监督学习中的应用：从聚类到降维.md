## 2.1高斯核函数

高斯核函数是一种常用的核函数，其定义为：

$$
K(x, y) = \exp(-\frac{\|x - y\|^2}{2\sigma^2})
$$

其中，$x$ 和 $y$ 是数据点，$\|x - y\|^2$ 是欧氏距离的平方，$\sigma$ 是核参数。

## 2.2核函数与内积空间

核函数是一种将高维空间映射到低维内积空间的方法。通过使用核函数，我们可以在低维内积空间中进行计算，而无需显式地计算高维空间中的向量。这种方法被称为内积空间（Hilbert space）方法。

```python
import numpy as np
from sklearn.cluster import KMeans
from sklearn.metrics.pairwise import rbf_kernel

# 生成随机数据
X = np.random.rand(100, 2)

# 计算数据点之间的欧氏距离
d = np.sqrt(np.sum((X - X[:, np.newaxis]) ** 2, axis=2))

# 使用高斯核函数计算相似性矩阵
K = rbf_kernel(X, X)

# 使用K-均值算法对数据进行分类

# 打印聚类结果
print(y_kmeans)
```
