## 4.1 代码实例

我们以一个简单的能源资源分配问题为例，通过线性规划算法实现能源资源分配的优化。

```python
from scipy.optimize import linprog

# 能源资源的生产和消费数据
supply = [10, 20, 30]
demand = [25, 15, 10]

# 能源资源的交易价格
price = [1, 2, 3]

# 能源资源分配的目标：最小化交易成本
cost = sum(price[i] * supply[i] for i in range(len(supply)))

# 能源资源分配的约束：供不能超过求
for i in range(len(supply)):
    linprog(cost, [supply[i] - demand[i]], [1])
```

## 4.2 详细解释说明

在这个代码实例中，我们首先导入了`linprog`函数，该函数是Scipy库中的一种线性规划算法，可以用来解决最小化或最大化线性目标函数的约束优化问题。

然后，我们定义了能源资源的生产和消费数据，以及能源资源的交易价格。接着，我们设定能源资源分配的目标为最小化交易成本，即最小化能源资源的交易价格与生产量的乘积之和。

最后，我们使用`linprog`函数对能源资源分配问题进行优化，并将结果输出。
