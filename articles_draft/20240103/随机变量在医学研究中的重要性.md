4. 预测和决策：随机变量可以帮助研究者更好地预测和做决策。例如，研究者可以使用随机变量来预测患者的生存率、疗效和副作用等，从而帮助医生制定更好的治疗方案。

$$
E[X] = \sum_{i=1}^{n} x_i P(x_i)

$$

例如，我们需要估计患者的生存率，可以使用贝叶斯估计方法。首先，我们需要收集患者的生存数据，然后计算生存概率分布函数，最后根据数据进行估计。

```python
import numpy as np
import pandas as pd
from scipy.stats import binom

# 收集患者生存数据
data = pd.read_csv('survival_data.csv')

# 计算生存概率分布函数
survival_prob = data['survival'].mean()

# 根据数据进行估计
estimate = binom.fit(data['survival'], survival_prob)
```

2. 期望和方差的计算：

例如，我们需要计算患者的年龄对治疗效果的期望和方差。可以使用numpy库计算这些指标。

```python
import numpy as np

# 假设我们有一组患者的年龄和治疗效果数据
age = np.array([25, 30, 35, 40, 45, 50, 55, 60])
treatment_effect = np.array([0.8, 0.7, 0.6, 0.5, 0.4, 0.3, 0.2, 0.1])

# 计算期望
expectation = np.sum(age * treatment_effect) / len(treatment_effect)

# 计算方差
variance = np.sum((age - expectation) ** 2 * treatment_effect) / len(treatment_effect)
```

例如，我们需要预测患者的生存率，可以使用线性回归方法。首先，我们需要收集患者的生存数据，然后使用scikit-learn库进行预测。

