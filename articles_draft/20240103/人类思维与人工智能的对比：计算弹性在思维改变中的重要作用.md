## 4.1 弹性计算的具体代码实例

```python
import time

def allocate_resources(resource_type, resource_num):
    resources = {}
    for i in range(resource_num):
        resources[resource_type] = i
    return resources

def adjust_resources(resources, task_need):
    for resource_type in resources.keys():
        if task_need[resource_type] > resources[resource_type]:
            resources[resource_type] += 1
    return resources

resources = allocate_resources('CPU', 10)
task_need = {'CPU': 15}
resources = adjust_resources(resources, task_need)
print(resources)
```
