# 2.基本概念术语说明
## 2.1 中英文简历的差异
中文简历（Resume in Chinese）简称简历，英文简历（Resume in English）简称Résumé。中文简历一般采用纸质的制作，长度一般在5到7页，以简历的内容和基本信息为主。而英文简历则是采用计算机或移动设备上的应用程序，可以根据用户填写的简历模板生成符合相关要求的PDF文件。根据简历的要求不同，两者也存在一些区别。

### 中文简历特点
- 用简洁的文字表达技能清晰，语言表达能力强。
- 内容的详细、突出程度较高，并突出个人的职业生涯。
- 有助于向HR筛选简历，加强职业竞争力。
- 可以作为英文口语习惯训练的工具。
- 更容易被搜索引擎收录。

### 英文简历特点
- 可借鉴其它人的简历写作，克服语言障碍。
- 内容更丰富，更细致，掌握不同的技术领域。
- 设计的专业化程度比中文简历更高。
- 对HR来说，英文简历的格式和排版更加友好，更易于扫描。

总体来说，中英文简历各有特色。如果没有特别需要，建议用中文简历。

## 2.2 应聘的岗位类型
招聘岗位类型分类：

- Junior Programmer：初级程序员，一般的工作年限为1~3年；
- Senior Programmer：高级程序员，一般的工作年限为3~5年；
- Lead Programmer：领导者，负责管理团队成员、项目，需对大型软件、Web应用有深刻的理解和能力；
- Architect：架构师，负责整体技术架构设计，需对软件架构、高性能计算、云计算有深刻的理解；
- Manager：经理，负责管理项目，协调资源，并向上级汇报工作进展；
- CTO：首席技术官，负责整个技术战略和管理层面的工作，需具有卓越的管理技能和执行力。

## 2.3 名词释义
下面列举一些常用的名词和对应的英文翻译，供大家查阅。

- Algorithm：算法，英文缩写为Algo。它是一系列解决特定问题的方法，指导计算机完成任务的指令集合。
- API：应用程序接口，应用程序编程接口，它是一套调用规范，定义了某类软件组件提供的服务及其通信机制。API是一个软件抽象层，通过它可以屏蔽底层实现的不同，使各种功能相对容易地被访问、使用。
- Array：数组，即一组按一定顺序排列的数据。
- Backtracking：回溯法，是一种穷举搜索方法，他的基本思路是从上到下、从左到右的尝试所有可能的路径，当找到一条路径导致目标函数值无效时，就退回一步重新考虑。
- Bit Manipulation：位运算，是计算机科学里处理数的一种技术。它利用计算机的低级位操作指令来进行逻辑、算术运算，并不需要借助于更为抽象的算法。
- Boolean Expression：布尔表达式，是指用逻辑运算符表示的命题陈述。比如A && B代表同时满足A和B两个条件。
- Brute Force：暴力破解，是指一种简单的直接计算方法，把所有的可能情况都试一遍，直到找到结果为止。
- Cache：缓存，是计算机存储的数据临时的暂存器，它可以加快数据的获取速度。
- Compilers：编译器，是一种将编程语言源代码转换成机器码的程序。编译器通常包括前端和后端两个部分，前端负责分析和解析源代码，后端负责产生机器码。
- Computer Science：计算机科学，又称CS，是研究电脑系统、计算机程序设计、计算机科学技术的学术领域。
- Data Structure：数据结构，是指用来组织、存储和管理数据的集合，是计算机中的基础。
- Decomposition：分解，是指将复杂问题分解为多个子问题。
- Design Patterns：设计模式，是一套反复使用的、多种多样的方案，是解决各种问题的有效方法。
- Dynamic Programming：动态规划，是一种求最值问题的优化技术，它的基本思想是将复杂问题分解成若干个小问题，求解每个子问题只记录必要的信息。
- Edge Case：边界情况，是指输入数据中的特殊情况。
- Encryption：加密，是指保密过程，将明文信息通过某种编码方式转换为无法识别的形式。
- Efficiency：效率，是指完成某项任务所需要的时间和资源开销的度量。
- Hash Function：散列函数，是一种将任意长度的消息压缩到固定长度的消息摘要的函数。
- Heap：堆，是树形结构，父节点的值一定不大于等于它的孩子节点的值，通常是用链表实现。
- Graph Theory：图论，是研究图形、图像、关系网络、数学等各种复杂系统的结构与演变过程的学术研究。
- Greedy Approach：贪婪算法，是一种基于局部最优策略的搜索算法。它在每一次迭代时，都会做出当前状态下的最优选择。
- Input Output：输入输出，指的是信息在存储、处理和传输过程中所遵循的次序，包括输入、处理、输出三个阶段。
- Interviewing Skills：面试技巧，是指通过有针对性的准备，采用合理的技术流程，进行一场面试，从而在面试官看来，是一个能够胜任工作岗位的真实的候选人。
- Linked List：链表，是由一系列节点组成的线性数据结构，每个节点都包含指向下一个节点的引用。
- Memory Management：内存管理，是计算机系统中分配和回收计算机资源的过程。
- Memoization：记忆化，是指对复杂计算过程的结果进行缓存，避免每次重复计算，从而提升运行效率。
- Merge Sort：归并排序，是一种稳定、高效的排序算法，它是采用分治法的一个典型例子。
- Pointer：指针，是一种变量，用于存放内存地址。
- Quick Sort：快速排序，是另一种排序算法。它的平均时间复杂度是O(nlogn)，但它在最坏情况下的时间复杂度却是O(n^2)。
- Redo and Undo：撤销与重做，是事务处理系统中常见的概念，即当事务执行失败或者被取消时，可以通过撤销操作将系统回滚到之前的状态。
- Recursive Algorithm：递归算法，是指在解决问题时，把一个大问题拆分成几个小问题，分别解决，最后合并其结果得到最终的解决方案。
- Regular Expression：正则表达式，是一种文本匹配的技术，它定义了一系列字符的组合，用来匹配或搜索某种模式的字符串。
- Search Space：搜索空间，是指在解决问题时，对问题的所有可能解进行分类、排列和组合的方式。
- Selection Sort：选择排序，是一种简单、直观的排序算法。它的基本思想就是在待排序队列中，找到最小（最大）元素，放在序列的起始位置，然后，再从剩余未排序元素中继续寻找最小（最大）元素，加入到已排序序列末尾。
- Stack：栈，它是一种线性的数据结构，仅允许在表尾进行插入和删除操作。
- String Matching Algorithms：字符串匹配算法，是指在一个长字符串中查找短字符串出现的位置，可以利用子串匹配算法、编辑距离算法或模式匹配算法来实现。
