# 2.核心概念与联系

## 2.1 鲸鱼优化算法简介

鲸鱼优化算法是一种基于自然界中鲸鱼行动模型的优化算法，它模拟了鲸鱼在寻找食物和逃跑自然敌人时的行为。鲸鱼优化算法的核心思想是将鲸鱼群的行为模拟为一种优化过程，通过鲸鱼群中每个鲸鱼的位置和速度来表示优化问题中的变量，从而实现优化问题的解决。
