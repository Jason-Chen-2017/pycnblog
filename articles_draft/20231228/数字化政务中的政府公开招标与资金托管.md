## 4.1 政府公开招标的具体代码实例

在政府公开招标中，可以使用以下Python代码实现竞标信息的收集、审核和评估：

```python
import requests
import json

# 发布采购需求
url = 'https://api.example.com/purchase/publish'
data = {'purchase_id': '12345', 'purchase_name': '购买电脑', 'purchase_budget': 100000}
response = requests.post(url, data=json.dumps(data))

# 提交竞标信息
url = 'https://api.example.com/bid/submit'
data = {'bid_id': '67890', 'purchase_id': '12345', 'bid_price': 90000, 'bid_quality': '高质量'}
response = requests.post(url, data=json.dumps(data))

# 收集竞标信息
url = 'https://api.example.com/bid/list'
response = requests.get(url)
bids = json.loads(response.text)

# 审核和评估竞标信息
for bid in bids:
    if bid['bid_price'] < bid['purchase_budget'] and bid['bid_quality'] == '高质量':
        print(f'合格供应商：{bid["supplier_name"]}, 竞标价格：{bid["bid_price"]}, 竞标质量：{bid["bid_quality"]}')
```
