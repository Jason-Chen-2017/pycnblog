## 4. 具体代码实例和详细解释说明

在环境保护领域，数据可重用性的具体代码实例如下：

1. 数据清洗和预处理：

```python
import pandas as pd
import numpy as np

# 加载原始数据
raw_data = pd.read_csv('raw_data.csv')

# 处理缺失值
raw_data = raw_data.fillna(method='ffill')

# 去除噪声
raw_data = raw_data[np.abs(raw_data - raw_data.mean()) < 3 * raw_data.std()]

# 数据标准化
raw_data = (raw_data - raw_data.min()) / (raw_data.max() - raw_data.min())
```

2. 数据集成：

```python
# 加载不同来源的数据
data1 = pd.read_csv('data1.csv')
data2 = pd.read_csv('data2.csv')

# 整合数据
integrated_data = pd.concat([data1, data2], axis=0)

# 数据单位转换
integrated_data['temperature'] = integrated_data['temperature'].convert_units('celsius')
```

3. 数据分析和模型构建：

```python
# 加载环境保护决策模型
from sklearn.ensemble import RandomForestRegressor

# 构建模型
model = RandomForestRegressor()

# 训练模型
model.fit(integrated_data[['temperature', 'humidity']], integrated_data['air_quality'])
```

4. 模型评估和优化：

```python
# 加载真实值数据
true_values = pd.read_csv('true_values.csv')

# 评估模型
predictions = model.predict(true_values[['temperature', 'humidity']])

# 计算模型精度
accuracy = model.score(true_values[['temperature', 'humidity']], true_values['air_quality'])

print('模型精度:', accuracy)
```
