### 4.1 代码实例

假设我们需要解决一个多目标优化问题，其中有三个决策变量，优化目标如下：

$$
\begin{aligned}
\min_{x \in \mathbb{R}^3} & \quad f(x) = x_1^2 + x_2^2 + x_3^2 \\
\text{s.t.} & \quad g_1(x) = x_1 + x_2 + x_3 - 10 \leq 0 \\
& \quad h_1(x) = x_1 - x_2 = 0
\end{aligned}
$$

我们可以使用以下代码来解决这个问题：

```python
import numpy as np
from scipy.optimize import minimize

def objective_function(x):
    return x[0]**2 + x[1]**2 + x[2]**2

def constraint1(x):
    return x[0] + x[1] + x[2] - 10

def constraint2(x):
    return x[0] - x[1]

x0 = np.array([1, 1, 1])
constraints = ({'type': 'ineq', 'fun': constraint1}, {'type': 'eq', 'fun': constraint2})

result = minimize(objective_function, x0, constraints=constraints)
print(result.x)
```

在这个代码实例中，我们首先导入了 `numpy` 和 `scipy.optimize` 库，然后定义了优化目标函数和约束条件。接下来，我们设定了初始决策向量 `x0`，并定义了约束条件。最后，我们使用 `minimize` 函数来解决优化问题，并输出了最优决策向量。
