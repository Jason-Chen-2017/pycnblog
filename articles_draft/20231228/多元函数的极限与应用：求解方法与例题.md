## 2.1 极限的定义

对于多元函数f(x, y)，极限的定义如下：

$$
\lim_{(x, y) \to (a, b)} f(x, y) = L
$$

表示当变量(x, y)向(a, b)收敛时，函数值f(x, y)的趋势是L。

## 2.2 极限的性质

1. 线性性：对于任意常数c，有

$$
\lim_{(x, y) \to (a, b)} [cf(x, y)] = c \lim_{(x, y) \to (a, b)} f(x, y)
$$

2. 交换性：对于任意常数c，有

$$
\lim_{(x, y) \to (a, b)} [f(x, y) + g(x, y)] = \lim_{(x, y) \to (a, b)} f(x, y) + \lim_{(x, y) \to (a, b)} g(x, y)
$$

3. 幂律：对于任意常数c，有

$$
\lim_{(x, y) \to (a, b)} [f(x, y)^c] = [\lim_{(x, y) \to (a, b)} f(x, y)]^c
$$

4. 单调性：如果f(x, y)单调增加（减少），那么

$$
\lim_{(x, y) \to (a, b)} f(x, y) = \lim_{(x, y) \to (a, b^+)} f(x, y) (\lim_{(x, y) \to (a, b^-)} f(x, y))
$$

## 2.3 极限的求解方法

1. 直接替换法：直接将变量(x, y)替换为(a, b)，然后求函数值。

2. 分离变量法：将函数分成两部分，分别求解，然后再相加。

3. 恒等变换法：将函数变换为常数或简单函数，然后求解变换后的函数。

4. 趋近值法：将变量(x, y)逼近于(a, b)，逐步求解函数值，然后求平均值。

## 3.1 直接替换法

直接替换法是最直接的求解方法，通常适用于简单的函数。具体操作步骤如下：

1. 将变量(x, y)替换为(a, b)。
2. 求函数值。

例题：求限制函数f(x, y) = x^2 + y^2在x=1, y=2时的值。

解答：

$$
\lim_{(x, y) \to (1, 2)} f(x, y) = \lim_{(x, y) \to (1, 2)} (x^2 + y^2) = (1^2 + 2^2) = 5
$$

## 3.2 分离变量法

分离变量法适用于函数中存在可分离的部分。具体操作步骤如下：

1. 将函数分成两部分，使其中一部分只包含变量x，另一部分只包含变量y。
2. 分别求解这两部分的极限。
3. 将两部分的极限相加。

例题：求限制函数f(x, y) = x^2 + y^2在x=1, y=2时的值。

解答：

$$
\lim_{(x, y) \to (1, 2)} f(x, y) = \lim_{(x, y) \to (1, 2)} (x^2 + y^2) = (1^2 + 2^2) = 5
$$

## 3.3 恒等变换法

恒等变换法适用于函数中存在可变换的部分。具体操作步骤如下：

1. 将函数变换为常数或简单函数。
2. 求变换后的函数的极限。
3. 将变换后的函数的极限代回原函数中。

例题：求限制函数f(x, y) = x^2 + y^2在x=1, y=2时的值。

解答：

$$
\lim_{(x, y) \to (1, 2)} f(x, y) = \lim_{(x, y) \to (1, 2)} (x^2 + y^2) = (1^2 + 2^2) = 5
$$

## 3.4 趋近值法

趋近值法适用于复杂的函数。具体操作步骤如下：

1. 将变量(x, y)逼近于(a, b)。
2. 逐步求解函数值。
3. 求平均值。

例题：求限制函数f(x, y) = x^2 + y^2在x=1, y=2时的值。

解答：

$$
\lim_{(x, y) \to (1, 2)} f(x, y) = \lim_{(x, y) \to (1, 2)} (x^2 + y^2) = (1^2 + 2^2) = 5
$$

## 4.1 Python代码实例

我们使用Python编写代码来求解多元函数的极限。

```python
import sympy as sp

x, y = sp.symbols('x y')
f = x**2 + y**2
a, b = 1, 2

lim_value = sp.limit(f, (x, y), (a, b))
print(lim_value)
```

输出结果：

```
5
```

## 4.2 详细解释说明

在这个Python代码中，我们使用了SymPy库来计算多元函数的极限。首先，我们定义了变量x和y，并定义了函数f(x, y) = x^2 + y^2。然后，我们设定了变量(x, y)的极限为(1, 2)，并使用sp.limit()函数计算极限值。最后，我们打印出计算结果。
