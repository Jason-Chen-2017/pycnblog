### 4.1 代码实例

以下是一个简单的智能家居代码实例，它使用了线性回归算法来预测家居设备的使用量。

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# 生成随机数据
np.random.seed(0)
X = np.random.rand(100, 1)
y = 3 * X.squeeze() + 2 + np.random.randn(100)

# 训练线性回归模型
model = LinearRegression()
model.fit(X, y)

# 预测
X_new = np.array([[0.5], [0.8], [1.0]])
y_pred = model.predict(X_new)

# 可视化
plt.scatter(X, y, label='原始数据')
plt.plot(X, model.predict(X), color='red', label='预测结果')
plt.legend()
plt.show()
```

### 4.2 详细解释说明

1. 首先，我们导入了所需的库，包括`numpy`、`matplotlib`和`sklearn`。
2. 然后，我们生成了一组随机的X和y数据，其中X是一个100x1的数组，y是一个100的数组。
3. 接下来，我们使用`sklearn`库中的`LinearRegression`类来训练线性回归模型，并使用原始数据进行训练。
4. 然后，我们使用训练好的模型来预测新的X数据的y值，并将预测结果存储在`y_pred`变量中。
5. 最后，我们使用`matplotlib`库来可视化原始数据和预测结果，并显示出来。
