```python
def classify_and_collect_waste(waste):
    classified_waste = {}
    for waste_type in waste:
        if waste_type not in classified_waste:
            classified_waste[waste_type] = []
        classified_waste[waste_type].append(waste)
    return classified_waste

waste = ["废纸", "塑料垃圾", "玻璃瓶"]  # 原始废物
classified_waste = classify_and_collect_waste(waste)
print("分类和收集后的废物：", classified_waste)
```

在这个代码实例中，我们通过对废物的分类和收集来减少资源消耗。具体来说，我们将原始废物按类别分组，并将其存储在字典中。
