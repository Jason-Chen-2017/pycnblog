## 4.1 密码加密算法实现
以下是一个使用Python实现AES密码算法的代码示例：
```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad

# 密钥
key = get_random_bytes(16)

# 明文
plaintext = b"Hello, world!"

# 加密
cipher = AES.new(key, AES.MODE_ECB)
ciphertext = cipher.encrypt(pad(plaintext, AES.block_size))

# 解密
plaintext = unpad(cipher.decrypt(ciphertext), AES.block_size)
```

## 4.3 门锁控制算法实现
以下是一个使用Python实现门锁控制算法的代码示例：
```python
# 门锁状态
lock_status = False

# 开锁
def unlock(code):
    global lock_status
    if code == "1234":
        lock_status = True
    else:
        lock_status = False

# 关锁
def lock():
    global lock_status
    lock_status = False

# 报警
def alarm():
    print("门锁报警！")

# 测试门锁控制算法
unlock("1234")
if lock_status:
    print("门已开锁！")
else:
    print("错误的密码！")
lock()
if not lock_status:
    print("门已关锁！")
else:
    print("门未关锁！")
alarm()
```
