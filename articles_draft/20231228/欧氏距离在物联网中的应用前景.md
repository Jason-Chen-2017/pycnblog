# 2. 核心概念与联系

## 2.1 欧氏距离简介

欧氏距离是一种计算两点距离的方法，它是在欧几里得几何空间中定义的。给定两个点（向量）P(x1, y1)和Q(x2, y2)，欧氏距离D可以通过以下公式计算：

$$
D = \sqrt{(x2 - x1)^2 + (y2 - y1)^2}
$$

欧氏距离可以用于计算多个点之间的距离，也可以用于计算多维空间中的距离。在物联网中，欧氏距离可以应用于数据聚类、数据降维、数据可视化等方面。

## 4.1 计算两点欧氏距离

```python
import math

def euclidean_distance(point1, point2):
    x1, y1 = point1
    x2, y2 = point2
    return math.sqrt((x2 - x1)**2 + (y2 - y1)**2)

point1 = (1, 2)
point2 = (4, 6)
print(euclidean_distance(point1, point2))
```

## 4.2 计算多点欧氏距离
