r = np.array([big_bang(t[i]) for i in range(len(t))])

## 4.2 星系形成的算法实现
我们可以使用Python来实现星系形成的算法，如下所示：

```python
import numpy as np

def gravitational_force(m1, m2, r):
    G = 6.674 * 10**(-11)
    return G * m1 * m2 / r**2

def star_system_formation(m1, m2, r, dt):
    F = gravitational_force(m1, m2, r)
    a = F / m1
    r_new = r + a * dt
    return r_new

m1 = 10**30
m2 = 10**30
r = 10**12
dt = 1000

r_new = star_system_formation(m1, m2, r, dt)
print('New distance:', r_new)
```

这个代码实现了星系形成的算法原理，其中 `m1` 和 `m2` 是两个物质的质量，`r` 是它们之间的距离，`dt` 是时间步长。通过这个代码，我们可以看到星系的形成是由物质的分布不均匀和引力力导致的。
