## 4.1 创建一个简单的区块链网络

首先，我们需要创建一个简单的区块链网络。我们可以使用Python编程语言来实现这个功能。以下是一个简单的Python代码实例：

```python
import hashlib
import json
import socket
import threading

class Node:
    def __init__(self, ip, port):
        self.ip = ip
        self.port = port
        self.sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        self.sock.bind((self.ip, self.port))
        self.sock.listen(5)
        self.connections = []

    def broadcast(self, message):
        for connection in self.connections:
            connection.send(json.dumps(message).encode())

    def receive(self):
        while True:
            connection, address = self.sock.accept()
            self.connections.append(connection)
            print(f"Connected to {address}")
            threading.Thread(target=self.handle_connection, args=(connection,)).start()

    def handle_connection(self, connection):
        while True:
            try:
                data = json.loads(connection.recv(1024).decode())
                self.broadcast(data)
            except Exception as e:
                print(f"Error: {e}")
                break

if __name__ == "__main__":
    node = Node("localhost", 5000)
    node.receive()
```

在这个代码实例中，我们创建了一个简单的区块链网络，包括创建节点、创建连接、广播消息等功能。

## 4.2 创建一个简单的区块

接下来，我们需要创建一个简单的区块。我们可以使用Python编程语言来实现这个功能。以下是一个简单的Python代码实例：

```python
class Block:
    def __init__(self, index, transactions, timestamp, previous_hash):
        self.index = index
        self.transactions = transactions
        self.timestamp = timestamp
        self.previous_hash = previous_hash
        self.hash = self.calculate_hash()

    def calculate_hash(self):
        block_string = json.dumps(self.__dict__, sort_keys=True).encode()
        return hashlib.sha256(block_string).hexdigest()

    def __repr__(self):
        return f"Block(index={self.index}, timestamp={self.timestamp}, previous_hash={self.previous_hash})"
```

在这个代码实例中，我们创建了一个简单的区块类，包括创建区块、计算区块哈希等功能。
