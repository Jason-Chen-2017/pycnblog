然后，编写代码进行商品信息的处理和特征提取：

```python
import jieba
from spacy import ner

# 读取商品信息
text = "这是一款商品，原价：19.99，优惠：5%"
doc = ner.load("en_core_web_sm")

# 分析商品信息
items = [token.text for token in doc if token.is_stop!= True and token.is_punct!= True]

# 分词
words = list(jieba.cut(text))

# 特征提取
features = [{"word": token.lower(), "label": "noun"}] for token in words if token.is_stop!= True and token.is_punct!= True]

# 储存
with open("product_features.txt", "w") as f:
    for feature in features:
        f.write(feature["word"] + " " + str(feature["label"]) + "
")
```

接着，编写代码进行模型训练和评估：

```python
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

# 读取数据
data = pd.read_csv("product_data.csv")

# 将数据分为训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(data["feature"], data["label"], test_size=0.2, random_state=0)

# 训练模型
model = LogisticRegression()
model.fit(X_train, y_train)

# 评估模型
accuracy = accuracy_score(y_test, model.predict(X_test))
print("模型准确率：", accuracy)
```
