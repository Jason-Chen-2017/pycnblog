# 4.具体代码实例和详细解释说明

## 4.1创建数据库

```sql
CREATE DATABASE mydb;
```

创建数据库的详细解释说明：

1. `CREATE DATABASE` 是 MySQL 中的一个关键字，用于创建数据库。
2. `mydb` 是数据库的名称。

## 4.2创建表

```sql
CREATE TABLE mytable (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(255) NOT NULL,
    age INT NOT NULL
);
```

创建表的详细解释说明：

1. `CREATE TABLE` 是 MySQL 中的一个关键字，用于创建表。
2. `mytable` 是表的名称。
3. `id` 是表的主键，`INT` 是数据类型，`PRIMARY KEY` 是主键约束，`AUTO_INCREMENT` 是自动增长约束。
4. `name` 是表的列，`VARCHAR(255)` 是数据类型，`NOT NULL` 是非空约束。
5. `age` 是表的列，`INT` 是数据类型，`NOT NULL` 是非空约束。

## 4.3插入数据

```sql
INSERT INTO mytable (name, age) VALUES ('John', 20);
```

插入数据的详细解释说明：

1. `INSERT INTO` 是 MySQL 中的一个关键字，用于插入数据。
2. `mytable` 是表的名称。
3. `(name, age)` 是表的列名称。
4. `VALUES ('John', 20)` 是插入的数据。

## 4.4查询数据

```sql
SELECT * FROM mytable WHERE age = 20;
```

查询数据的详细解释说明：

1. `SELECT` 是 MySQL 中的一个关键字，用于查询数据。
2. `*` 是所有列。
3. `FROM` 是 MySQL 中的一个关键字，用于指定表。
4. `mytable` 是表的名称。
5. `WHERE` 是 MySQL 中的一个关键字，用于指定查询条件。
6. `age = 20` 是查询条件。
