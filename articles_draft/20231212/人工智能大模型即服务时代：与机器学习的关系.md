## 3.1 线性回归

线性回归是一种简单的机器学习算法，用于预测连续型变量的值。它的基本思想是找到一个最佳的直线，使得该直线可以最佳地拟合训练数据集。

线性回归的数学模型如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

其中，$y$是预测变量，$x_1, x_2, ..., x_n$是输入变量，$\beta_0, \beta_1, ..., \beta_n$是模型参数，$\epsilon$是误差项。

线性回归的具体操作步骤如下：

1. 初始化模型参数$\beta_0, \beta_1, ..., \beta_n$为随机值。
2. 使用梯度下降算法迭代更新模型参数，以最小化训练数据集上的损失函数。
3. 当模型参数收敛时，停止迭代。
4. 使用得到的模型参数预测测试数据集上的$y$值。

## 3.2 逻辑回归

逻辑回归是一种用于二分类问题的机器学习算法。它的基本思想是找到一个最佳的分隔超平面，使得该超平面可以最佳地将训练数据集划分为两个类别。

逻辑回归的数学模型如下：

$$
P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n)}}
$$

其中，$y$是类别变量，$x_1, x_2, ..., x_n$是输入变量，$\beta_0, \beta_1, ..., \beta_n$是模型参数。

逻辑回归的具体操作步骤与线性回归类似，只是损失函数不同。

## 3.3 支持向量机

支持向量机（SVM）是一种用于二分类和多分类问题的机器学习算法。它的基本思想是找到一个最佳的分隔超平面，使得该超平面可以最佳地将训练数据集划分为不同的类别。

支持向量机的数学模型如下：

$$
f(x) = \text{sign}(\sum_{i=1}^n \alpha_i y_i K(x_i, x) + b)
$$

其中，$f(x)$是输出函数，$K(x_i, x)$是核函数，$\alpha_i$是模型参数，$y_i$是训练数据集的标签。

支持向量机的具体操作步骤如下：

1. 初始化模型参数$\alpha_1, \alpha_2, ..., \alpha_n$为随机值。
2. 使用梯度下降算法迭代更新模型参数，以最小化训练数据集上的损失函数。
3. 当模型参数收敛时，停止迭代。
4. 使用得到的模型参数预测测试数据集上的类别。

## 4.1 线性回归

以下是一个使用Python的Scikit-Learn库实现的线性回归代码示例：

```python
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

# 训练数据集
X_train = [[1], [2], [3], [4]]
y_train = [2, 4, 6, 8]

# 测试数据集
X_test = [[5], [6], [7], [8]]
y_test = [10, 12, 14, 16]

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X_train, y_train)

# 预测测试数据集上的y值
y_pred = model.predict(X_test)

# 计算预测误差
mse = mean_squared_error(y_test, y_pred)
print("预测误差：", mse)
```

在这个代码示例中，我们首先创建了一个线性回归模型，然后使用训练数据集训练该模型。接着，我们使用测试数据集预测y值，并计算预测误差。

## 4.2 逻辑回归

以下是一个使用Python的Scikit-Learn库实现的逻辑回归代码示例：

```python

from sklearn.metrics import accuracy_score

# 训练数据集

y_train = [0, 1, 1, 0]

# 测试数据集
X_test = [[1, 0], [1, 1], [0, 0], [0, 1]]
y_test = [0, 1, 1, 0]

# 创建逻辑回归模型
model = LogisticRegression()

# 训练模型
model.fit(X_train, y_train)

# 预测测试数据集上的y值
y_pred = model.predict(X_test)

# 计算预测准确率
accuracy = accuracy_score(y_test, y_pred)
print("预测准确率：", accuracy)
```

在这个代码示例中，我们首先创建了一个逻辑回归模型，然后使用训练数据集训练该模型。接着，我们使用测试数据集预测y值，并计算预测准确率。
