## 3.2 几何几何的基本定理
1. 点到点的距离：点到点的距离是两个点之间的距离，可以用欧几里得距离公式计算，公式为：d = √((x2 - x1)² + (y2 - y1)²)。
2. 线段到点的距离：线段到点的距离是线段两端点与点之间的距离，可以用欧几里得距离公式计算，公式为：d = √((x2 - x1)² + (y2 - y1)²)。
3. 面到点的距离：面到点的距离是面三个点与点之间的距离，可以用欧几里得距离公式计算，公式为：d = √((x2 - x1)² + (y2 - y1)²)。

## 3.3 几何几何的基本操作
1. 点的移动：可以通过更改点的坐标来移动点。例如，将点（x，y）移动到点（x + a，y + b）。
2. 点的旋转：可以通过更改点的坐标来旋转点。例如，将点（x，y）旋转角度θ，得到点（x'，y'），其中x' = x * cosθ - y * sinθ，y' = x * sinθ + y * cosθ。

## 4.1 代码实例
```python
import math

# 定义一个点类
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def move(self, a, b):
        self.x += a
        self.y += b

    def rotate(self, theta):
        x_prime = self.x * math.cos(theta) - self.y * math.sin(theta)
        y_prime = self.x * math.sin(theta) + self.y * math.cos(theta)
        self.x = x_prime
        self.y = y_prime

    def scale(self, k):
        self.x *= k
        self.y *= k

# 定义一个线段类
class Segment:
    def __init__(self, p1, p2):
        self.p1 = p1
        self.p2 = p2

    def length(self):
        return math.sqrt((self.p2.x - self.p1.x) ** 2 + (self.p2.y - self.p1.y) ** 2)

# 定义一个面类
class Plane:
    def __init__(self, p1, p2, p3):
        self.p1 = p1
        self.p2 = p2
        self.p3 = p3

    def distance(self, p):
        a = self.p2.x - self.p1.x
        b = self.p2.y - self.p1.y
        c = -a * self.p1.x - b * self.p1.y
        d = -a * self.p3.x - b * self.p3.y
        e = a * p.x + b * p.y - c
        f = -a * p.x - b * p.y - d
        return abs(e) / math.sqrt(e ** 2 + f ** 2)
```

## 4.2 详细解释说明
在上述代码中，我们定义了三个类：Point、Segment和Plane。

1. Point类：表示一个点，其中x和y是点的坐标。该类有三个方法：move、rotate和scale，分别用于移动、旋转和缩放点。
2. Segment类：表示一个线段，其中p1和p2是线段的两个端点。该类有一个方法：length，用于计算线段的长度。
3. Plane类：表示一个面，其中p1、p2和p3是面的三个顶点。该类有一个方法：distance，用于计算面到点的距离。
