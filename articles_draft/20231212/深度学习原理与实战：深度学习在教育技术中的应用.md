# 2.核心概念与联系

深度学习的核心概念包括神经网络、前向传播、反向传播、损失函数、梯度下降等。这些概念是深度学习的基础，理解它们对于掌握深度学习技术至关重要。

## 2.1 神经网络

神经网络是深度学习的基本结构，它由多个节点（神经元）和连接这些节点的权重组成。每个节点接收输入，进行计算，并输出结果。这些计算通过连接的权重进行传播，以形成输出。

神经网络的基本结构包括输入层、隐藏层和输出层。输入层接收输入数据，隐藏层进行计算，输出层输出结果。每个层之间通过权重连接，这些权重在训练过程中会被调整。

## 2.2 前向传播

前向传播是神经网络中的一个过程，它用于计算输出。在前向传播过程中，输入数据通过各个层进行计算，最终得到输出结果。

前向传播的计算公式为：

$$
y = f(Wx + b)
$$

其中，$y$ 是输出，$f$ 是激活函数，$W$ 是权重矩阵，$x$ 是输入，$b$ 是偏置。

## 2.3 反向传播

反向传播是神经网络中的一个过程，它用于调整权重。在反向传播过程中，从输出层向前向后，通过计算梯度，调整各个层的权重。

反向传播的计算公式为：

$$
\Delta W = \frac{\partial L}{\partial W} = \frac{\partial L}{\partial y} \cdot \frac{\partial y}{\partial W}
$$

其中，$\Delta W$ 是权重的梯度，$L$ 是损失函数，$y$ 是输出，$W$ 是权重。

## 2.4 损失函数

损失函数是用于衡量模型预测结果与实际结果之间差异的指标。常用的损失函数包括均方误差（MSE）、交叉熵损失（Cross-Entropy Loss）等。损失函数在训练过程中会被最小化，以便得到更好的预测结果。

## 2.5 梯度下降

梯度下降是优化模型参数的一种方法，它通过不断调整参数，使损失函数的值逐渐减小。梯度下降的公式为：

$$
W_{new} = W_{old} - \alpha \cdot \Delta W
$$

其中，$W_{new}$ 是新的权重，$W_{old}$ 是旧的权重，$\alpha$ 是学习率，$\Delta W$ 是权重的梯度。

## 4.1 梯度下降

```python
import numpy as np

def gradient_descent(X, y, theta, alpha, num_iterations):
    m = len(y)
    X = np.c_[np.ones(m), X]
    for _ in range(num_iterations):
        hypothesis = X.dot(theta)
        loss = (hypothesis - y) ** 2
        gradient = X.T.dot(loss) / m
        theta = theta - alpha * gradient
    return theta
```

在上述代码中，我们实现了梯度下降算法。`X` 是输入数据，`y` 是输出数据，`theta` 是模型参数，`alpha` 是学习率，`num_iterations` 是训练次数。我们首先扩展输入数据，然后计算损失，并计算梯度。最后，我们更新模型参数。

```python
def backward_propagation(X, y, theta1, theta2, learning_rate):
    m = len(y)
    gradients = {}
    L = X.shape[1]

    # Compute the loss
    loss = (1 / m) * np.sum(np.square(X.dot(theta2) - y))

    # Compute the gradient of the loss with respect to theta2
    gradients["theta2"] = (1 / m) * X.T.dot(np.multiply(X.dot(theta2) - y, X))

    # Compute the gradient of the loss with respect to theta1
    gradients["theta1"] = (1 / m) * np.dot(X.T.dot(theta2.T), theta2)

    # Update the parameters
    theta1 = theta1 - learning_rate * gradients["theta1"]
    theta2 = theta2 - learning_rate * gradients["theta2"]

    return theta1, theta2
```

在上述代码中，我们实现了反向传播算法。`X` 是输入数据，`y` 是输出数据，`theta1` 和 `theta2` 是模型参数，`learning_rate` 是学习率。我们首先计算损失，并计算梯度。最后，我们更新模型参数。
