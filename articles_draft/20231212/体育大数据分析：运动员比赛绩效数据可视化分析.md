## 4.1 数据预处理
### 4.1.1 数据清洗
```python
import pandas as pd

# 读取数据
data = pd.read_csv('athlete_data.csv')

# 删除重复数据
data.drop_duplicates(inplace=True)

# 修正错误数据
data['score'] = data['score'].apply(lambda x: x if x >= 0 else 0)
```

### 4.1.2 数据缺失处理
```python
# 删除缺失值
data.dropna(inplace=True)

# 填充缺失值
data['age'] = data['age'].fillna(data['age'].mean())
```

### 4.1.3 数据归一化
```python
# 最小-最大归一化
data['score_min_max'] = (data['score'] - data['score'].min()) / (data['score'].max() - data['score'].min())

# 标准化
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
data['score_std'] = scaler.fit_transform(data[['score']])
```
