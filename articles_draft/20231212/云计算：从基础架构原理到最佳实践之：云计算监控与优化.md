## 4.1 性能监控代码实例

以下是一个性能监控的代码实例：

```python
import time
import psutil

def performance_monitor():
    while True:
        cpu_usage = psutil.cpu_percent()
        memory_usage = psutil.virtual_memory().percent
        disk_usage = psutil.disk_usage('/').percent
        network_usage = psutil.net_io_counters(pernic=True)

        print('CPU Usage:', cpu_usage)
        print('Memory Usage:', memory_usage)
        print('Disk Usage:', disk_usage)
        print('Network Usage:', network_usage)

        time.sleep(1)

if __name__ == '__main__':
    performance_monitor()
```

在这个代码实例中，我们使用了 `psutil` 库来获取系统的性能指标，如 CPU 使用率、内存使用率、磁盘使用率、网络使用率等。我们将这些性能指标打印出来，并每秒更新一次。

## 4.2 资源监控代码实例

以下是一个资源监控的代码实例：

```python
import time
import psutil

def resource_monitor():
    while True:
        cpu_count = psutil.cpu_count(logical=False)
        cpu_times = psutil.cpu_times(percpu=True)
        memory_info = psutil.virtual_memory()
        disk_partitions = psutil.disk_partitions(all=True)

        print('CPU Count:', cpu_count)
        print('CPU Times:', cpu_times)
        print('Memory Info:', memory_info)
        print('Disk Partitions:', disk_partitions)

        time.sleep(1)

if __name__ == '__main__':
    resource_monitor()
```

在这个代码实例中，我们使用了 `psutil` 库来获取系统的资源状态，如 CPU 核数、CPU 使用率、内存信息、磁盘分区等。我们将这些资源状态打印出来，并每秒更新一次。

## 4.3 安全监控代码实例

以下是一个安全监控的代码实例：

```python
import time
import psutil

def security_monitor():
    while True:
        processes = psutil.process_iter()
        for process in processes:
            try:
                process_name = process.info['name']
                process_pid = process.info['pid']
                process_username = psutil.Process(process_pid).username()

                print('Process Name:', process_name)
                print('Process PID:', process_pid)
                print('Process Username:', process_username)
            except (KeyError, psutil.NoSuchProcess):
                pass

        time.sleep(1)

if __name__ == '__main__':
    security_monitor()
```

在这个代码实例中，我们使用了 `psutil` 库来获取系统的进程信息，如进程名称、进程 PID、进程用户名等。我们将这些进程信息打印出来，并每秒更新一次。
