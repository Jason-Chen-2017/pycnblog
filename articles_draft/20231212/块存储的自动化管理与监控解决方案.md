## 4.1 存储资源的分配
以下是一个简单的Python代码实例，用于实现存储资源的分配：

```python
def allocate_storage(need, total):
    if need > total:
        raise ValueError("Need exceeds total")
    return need
```

这个函数接收两个参数：需求（need）和总量（total）。如果需求大于总量，则会抛出一个ValueError异常。否则，函数会返回需求的值，表示分配的存储资源。

## 4.3 存储空间监控
以下是一个简单的Python代码实例，用于实现存储空间的监控：

```python
def monitor_storage(storage):
    used = storage.used()
    total = storage.total()
    return used / total
```

这个函数接收一个存储对象（storage）。函数会调用存储对象的used()方法获取已使用的空间，并调用total()方法获取总空间。然后，函数会返回已使用空间的占用率。
