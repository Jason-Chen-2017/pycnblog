
### 4.2 数据仪表盘实例

```python
import dash
import dash_core_components as dcc
import dash_html_components as html
from dash.dependencies import Input, Output
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 创建仪表盘
app = dash.Dash(__name__)

app.layout = html.Div([
    dcc.Graph(
        id='graph',
        figure={
            'data': [
                {'x': data['x'], 'y': data['y'], 'type': 'scatter'}
            ],
            'layout': {
                'title': '数据仪表盘示例',
                'xaxis': {'title': 'X轴'},
                'yaxis': {'title': 'Y轴'}
            }
        }
    )
])

# 运行仪表盘
if __name__ == '__main__':
    app.run_server(debug=True)
```

在这个实例中，我们使用了Dash库来创建一个简单的数据仪表盘。我们首先读取了数据，然后使用Dash的`Graph`组件创建了一个散点图。最后，我们使用`run_server`方法运行仪表盘。
