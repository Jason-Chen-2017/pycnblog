## 2.1 数据仓库与数据集成的概念

## 4.3 数据整合代码实例

```python
import pandas as pd

# 读取数据源1
data1 = pd.read_csv('data1.csv')

# 读取数据源2
data2 = pd.read_csv('data2.csv')

# 数据格式转换
data1['date'] = pd.to_datetime(data1['date'])
data2['date'] = pd.to_datetime(data2['date'])

# 数据结构转换
data1 = data1[['date', 'age', 'gender']]
data2 = data2[['date', 'age', 'gender']]

# 数据清洗
data1 = data1.dropna()
data2 = data2.dropna()

# 数据整合
data = pd.concat([data1, data2])

# 存储到数据仓库
data.to_csv('data_warehouse.csv', index=False)
```

在这个代码实例中，我们首先使用pandas库读取两个数据源，然后对数据进行格式转换、类型转换和清洗。最后，我们将数据整合到一个统一的数据仓库中，并存储到数据仓库中。
