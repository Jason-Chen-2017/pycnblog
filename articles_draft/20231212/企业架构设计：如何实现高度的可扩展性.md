我们可以使用一种简单的负载均衡算法，即轮询算法。轮询算法是一种将请求分配给可用节点的简单方法，它会按顺序将请求分配给每个可用节点。这种算法可以确保分布式系统中的所有节点都能够处理相同的负载。

```python
import random

def load_balancer(requests, nodes):
    request_index = 0
    node_index = 0
    while request_index < len(requests):
        node = nodes[node_index]
        if node.is_available():
            node.process_request(requests[request_index])
            request_index += 1
        node_index = (node_index + 1) % len(nodes)
```
