## 4.1概率模型

在Python中，我们可以使用numpy库来创建概率模型。以下是一个简单的示例：

```python
import numpy as np

# 创建一个概率模型
probability_model = np.array([0.3, 0.7])

# 使用概率模型进行预测
prediction = np.random.choice(2, p=probability_model)
print(prediction)
```

在这个示例中，我们创建了一个概率模型，其中0的概率为0.3，1的概率为0.7。然后，我们使用这个概率模型进行预测，并输出预测结果。

## 4.2随机变量

在Python中，我们可以使用numpy库来创建随机变量。以下是一个简单的示例：

```python
import numpy as np

# 创建一个随机变量
random_variable = np.random.normal(loc=0, scale=1, size=1000)

# 使用随机变量进行统计分析
mean = np.mean(random_variable)
std = np.std(random_variable)
print(mean)
print(std)
```

在这个示例中，我们创建了一个正态分布的随机变量，其均值为0，标准差为1。然后，我们使用这个随机变量进行统计分析，并输出均值和标准差。

## 4.3条件概率

在Python中，我们可以使用numpy库来计算条件概率。以下是一个简单的示例：

```python
import numpy as np

# 创建一个条件概率表
condition_probability_table = np.array([
    [0.5, 0.5],
    [0.3, 0.7]
])

# 使用条件概率表进行预测
prediction = np.random.choice(2, p=condition_probability_table[0, 1])
print(prediction)
```

在这个示例中，我们创建了一个条件概率表，其中当事件1发生时，事件2的概率为0.5，当事件2发生时，事件1的概率为0.3。然后，我们使用这个条件概率表进行预测，并输出预测结果。
