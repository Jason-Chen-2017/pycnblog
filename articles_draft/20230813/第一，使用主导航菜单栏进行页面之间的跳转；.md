# 1.简介
  

目前网页设计中，菜单栏经常作为重要的导航工具出现，用于在不同页面之间快速切换或定位到目标区域。但是，实际上网站的导航设计并不仅仅局限于菜单栏，比如分类、标签、搜索等也是可以被当做页面间的切换工具，但它们通常不会占据菜单栏的空间，而是采用类似按钮的方式呈现在页面底部或侧边栏。那么，如何合理地分配页面间的跳转方式，才能让用户更便捷的浏览和获取信息呢？本文将介绍一些常见的页面跳转方法及其优缺点，让读者对这些方法有一个全面的认识，选择适合自己的跳转方式。
# 2.主要术语及概念
## 2.1 网站导航
### 2.1.1 目录结构（Site Map）
网站的导航结构就是网站页面的一种树形结构。它是指网站内页面、文件的分层次结构，用来方便访问者快速了解整个网站的内容及其位置。
网站目录结构属于静态文档，也就是说，网站的所有页面都已按照结构化数据保存在服务器上。目录结构一般会自动生成，不需要用户手动创建。目录结构中的每一个节点代表一个网站的页面或文件，每个节点都会显示该节点所对应的名称、链接地址、子节点个数以及页面摘要（如果有的话）。通过目录结构，用户可以通过点击各个节点进入相应的页面或文件，就可以轻東地找到自己想要的信息。
网站目录结构的作用有三方面：

1.方便访问者：通过目录结构，用户可以很直观的找到自己想访问的页面或者文件，而无需先翻阅所有页面的标题。

2.搜索引擎优化（SEO）：由于搜索引擎对目录结构的识别非常有效，所以网站制作者可以根据网站目录结构提升网站的权威性，让网站的关键词排名靠前。

3.快捷访问：通过目录结构，用户可以快速的定位自己需要的信息，进一步加快网站的响应速度。

### 2.1.4 链接页
链接页是指将多个网站链接在一起，形成一个完整的网络。在网站中，链接页常用在广告宣传和推广页面上，通过将多个网站链接在一起，能够大大促进网站流量，提高站长的宣传效果。但是，由于链接过多，可能会造成用户信息泄露和骚扰，因此在做链接页时，应该注意防范安全隐患。

### 2.1.5 搜索框
搜索框一般位于网站导航栏或页面顶端，用来让用户输入关键字快速搜索相关信息。搜索框是实现网站导航功能的重要途径，具有十分重要的意义。搜索框的作用主要有以下四个方面：

1.查找信息：通过搜索框，用户可以快速检索出自己所需要的信息，不需要翻阅长篇大论。

2.提升用户体验：搜索框帮助网站改善用户体验，提升用户满意度，尤其是在搜索结果不够准确或需要反复查询时，用户可以更方便地找到需要的信息。

3.降低搜索排名：由于搜索框通常位于导航栏的顶端，使得用户容易被搜索引擎收录，提升网站排名。

4.节约系统资源：搜索框的功能性也使得网站管理员不必频繁使用服务器，减少服务器负载，节约硬件成本。

## 2.2 页面跳转
页面跳转又称为页面间的跳转或链接，是指在同一个域名下两个相邻页面之间发生的跳转动作。通过页面跳转，用户可以在网站上自由穿梭，发现新的信息，解决问题，获得感兴趣的内容。常见的页面跳转方法有：

### 2.2.1 链接法
链接法是指通过超链接（Hyperlink）实现页面跳转。超链接是一个文本中的特殊符号，它使得用户可以从当前页面跳转至其他页面。超链接可以点击，也可以鼠标悬停，甚至可以用键盘快捷键激活。超链接的基本形式如下：

```
<a href="目标页面URL">超链接文字</a>
```

其中，href属性定义了超链接的目标URL，文字则表示超链接的显示文字。通常情况下，用户点击超链接后，将会转到目标页面。超链接的好处有以下几点：

1.直观易懂：超链接是用户与网站互动的最常见方式，其直接体现了页面之间的连接关系，对用户来说比较直观。

2.方便快捷：超链接提供了一种即快又省力的方法，用户可以轻東、迅速地跳转到所需的页面，且不用离开当前页面。

3.避免死链：网站维护人员经常会遇到失效的链接，如果没有备案，那么当用户点击该链接时，将无法访问该页面，此时就可能出现死链。

4.利于搜索：因为搜索引擎可以抓取超链接，所以网站上发布的超链接越多，其收录也就越好。而且，由于搜索引擎将超链接作为重要的索引信息，因此，其抓取也会受到网站本身的影响。

### 2.2.2 表单提交法
表单提交法是指利用HTML中的表单元素（如input标签和select标签）实现页面跳转。表单提交法即在当前页面中嵌入一个表单，然后将用户输入的数据提交到另一个页面。表单提交可以是同步提交，也可以是异步提交，区别在于处理方式不同。对于同步提交，浏览器等待表单提交完成再加载新页面，但在用户看来却像是网页在刷新，对于异步提交，浏览器在接收到服务器返回的结果后再加载新页面。表单提交的基本形式如下：

```
<form action="目标页面URL" method="GET|POST">
   ... 表单元素
</form>
```

其中，action属性定义了表单数据的提交目标URL，method属性定义了数据的传输方式，可以是GET或POST。表单元素可以包括文本框、密码框、单选框、复选框、下拉列表、隐藏字段等，通过这些表单元素，用户可以输入数据并提交到指定的页面。表单提交法的好处有以下几点：

1.安全性：表单提交法可以保证用户的隐私数据在传输过程中不被窃取，确保用户信息的安全。

2.易用性：表单提交法可以简单、直观、易用，用户只需要简单填写表单并提交即可。

3.动态性：表单提交法可以实现动态更新，使得网站具有较强的实时性。

4.可定制性：由于表单提交可以提交各种类型的数据，因此，用户可以自定义表单提交页面的样式。

### 2.2.3 服务器重定向法
服务器重定向法是指服务器告诉客户端，请求的页面已经转移到了另外一个地方，并将新的URL发送给客户端。客户端随后将重新发送请求，并请求新的页面。服务器重定向的基本形式如下：

```
HTTP/1.1 301 Moved Permanently
Location: 目标页面URL
Content-Type: text/html; charset=utf-8
Connection: keep-alive
```

其中，301状态码表示永久重定向，Location头域指定了新的URL。服务器重定向法的好处有以下几点：

1. SEO优化：服务器重定向可以帮助搜索引擎更新它的索引。

2. 权限管理：服务器重定向可以控制用户访问权限，阻止未授权的用户访问特定页面。

3. 用户体验：服务器重定向可以给用户带来较好的用户体验，用户会察觉到网址的变化，不会认为是个错误的页面。

### 2.2.4 脚本跳转法
脚本跳转法是指利用JavaScript脚本实现页面跳转。脚本跳转法的基本形式如下：

```
window.location.href = "目标页面URL";
```

其中，window.location.href表示当前页面的URL，赋值操作可以改变其值，即实现页面跳转。脚本跳转法的好处有以下几点：

1. 用户体验：脚本跳转法可以给用户带来较好的用户体验，用户可以直接从当前页面跳到目标页面。

2. 可定制性：由于脚本跳转可以执行任意JavaScript代码，因此，可以实现更复杂的页面跳转。

3. SEO优化：脚本跳转法可以帮助搜索引擎更新索引。

综上，页面跳转是网页设计中非常重要的一环，如何合理地设计页面跳转，对提升用户体验、提升网站流量、提升网站品牌都有着重要的作用。
