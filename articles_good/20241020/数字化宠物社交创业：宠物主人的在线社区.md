##### 附录C：数学模型和数学公式

1. **协同过滤算法中的评分预测**：
   协同过滤算法中的评分预测通常使用用户-物品评分矩阵来计算。以下是一个简单的评分预测公式：

   $$ \hat{r}_{ui} = r_{ui} + \sum_{j \in N_{u}} (r_{uj} - \bar{r}_{uj}) \cdot (r_{vi} - \bar{r}_{vi}) $$

   其中，$r_{ui}$ 是用户对物品 $i$ 的真实评分，$r_{uj}$ 是用户对物品 $j$ 的评分，$\bar{r}_{uj}$ 是用户对所有物品的平均评分，$N_{u}$ 是与用户 $u$ 相似的用户集合。

   $$ \bar{r}_{uj} = \frac{\sum_{i \in I} r_{ui}}{|I|} $$

   其中，$I$ 是所有物品的集合，$|I|$ 是物品的数量。

```python
    existing_user = User.query.filter_by(username=username).first()
    if existing_user:
        return jsonify({'error': 'Username already exists'}), 400
    
    new_user = User(username=username, password=password)
    db.session.add(new_user)
    db.session.commit()

    db.create_all()
    app.run(debug=True)
```

##### 附录C：数学模型和数学公式

在推荐系统和用户行为分析中，以下是一些常用的数学模型和数学公式：

1. **协同过滤算法中的评分预测**：
   $$ \hat{r}_{ui} = r_{ui} + \sum_{j \in N_{u}} (r_{uj} - \bar{r}_{uj}) \cdot (r_{vi} - \bar{r}_{vi}) $$
   其中，$\hat{r}_{ui}$ 是预测的用户 $u$ 对物品 $i$ 的评分，$r_{ui}$ 是用户 $u$ 对物品 $i$ 的真实评分，$r_{uj}$ 是用户 $u$ 对物品 $j$ 的评分，$\bar{r}_{uj}$ 是用户对所有物品的平均评分，$N_{u}$ 是与用户 $u$ 相似的用户集合。

   $$ \text{similarity}(x, y) = \frac{\sum_{i \in I} x_i \cdot y_i}{\sqrt{\sum_{i \in I} x_i^2} \cdot \sqrt{\sum_{i \in I} y_i^2}} $$
   其中，$x$ 和 $y$ 是两个向量，$I$ 是所有特征项的集合，$x_i$ 和 $y_i$ 是向量中对应的特征值。

   其中，$w_i$ 是特征项 $i$ 的权重，$\text{content\_vector}(i)$ 是物品 $i$ 的内容向量。

```python
app = Flask(__name__)

db = SQLAlchemy(app)

def home():

def register():
    if request.method == 'POST':
        username = request.form['username']
        password = request.form['password']
        if not username or not password:
            flash('请填写用户名和密码！', 'warning')
            return redirect(url_for('register'))
        if User.query.filter_by(username=username).first():
            flash('用户名已存在！', 'warning')
            return redirect(url_for('register'))
        new_user = User(username=username, password=generate_password_hash(password))
        db.session.add(new_user)
        db.session.commit()

        return redirect(url_for('home'))
    return render_template('register.html')

    if request.method == 'POST':
        username = request.form['username']
        password = request.form['password']
        user = User.query.filter_by(username=username).first()
        if not user or not check_password_hash(user.password, password):
            flash('用户名或密码错误！', 'warning')
            return redirect(url_for('login'))
        flash('登录成功！', 'success')
        return redirect(url_for('home'))
    return render_template('login.html')

    db.create_all()
```
