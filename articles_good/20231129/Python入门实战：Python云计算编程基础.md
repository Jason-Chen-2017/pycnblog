## 4.1 基础设施即服务（IaaS）

IaaS提供了虚拟机、存储、网络等基础设施服务，用户可以通过互联网访问和管理这些服务。以下是一个使用Python访问IaaS服务的代码实例：

```python
import boto3

# 创建IaaS客户端
ec2 = boto3.client('ec2')

# 创建虚拟机
response = ec2.run_instances(
    ImageId='ami-0c94855ba951c16d6',
    MinCount=1,
    MaxCount=1,
    InstanceType='t2.micro'
)

# 获取虚拟机的IP地址
instance_id = response['Instances'][0]['InstanceId']
instance = ec2.describe_instances(InstanceIds=[instance_id])
ip_address = instance['Reservations'][0]['Instances'][0]['PublicIpAddress']

# 打印虚拟机的IP地址
print(ip_address)
```

在上述代码中，我们首先导入了Boto库，这是一个用于访问AWS服务的Python库。然后，我们创建了一个IaaS客户端，并使用该客户端创建了一个虚拟机。最后，我们获取了虚拟机的IP地址并打印了它。
