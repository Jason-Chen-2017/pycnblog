## 4.1数据清洗

### 4.1.1数据的去除

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 去除重复数据
data = data.drop_duplicates()

# 去除缺失数据
data = data.dropna()

# 去除错误数据
data = data.replace(to_replace='error', value=None)
data = data.dropna()

# 保存数据
data.to_csv('data_cleaned.csv', index=False)
```

### 4.1.2数据的填充

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 填充缺失数据
data['city'] = data['city'].fillna('unknown')

# 保存数据
data.to_csv('data_filled.csv', index=False)
```

### 4.1.3数据的转换

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 转换数据格式
data['age'] = data['age'].astype(int)

# 转换数据类型
data['gender'] = data['gender'].astype('category')

# 保存数据
data.to_csv('data_transformed.csv', index=False)
```

### 4.1.4数据的编码

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 编码数据值
data['gender'] = data['gender'].cat.codes

# 保存数据
data.to_csv('data_encoded.csv', index=False)
```

import pandas as pd
