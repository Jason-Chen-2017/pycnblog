# 1.背景介绍


在中国，创业企业往往都需要一些资金支持才能正常运营。一方面，要靠个人或者团队的努力获得足够的资金激活公司，另一方面，企业也可能拥有自己的私募基金或其他形式的资本来源，但随着市场化改革、人工智能等新技术的兴起，越来越多的人加入了投资者群体，他们希望通过股权机制来实现财富自由。
股权计划（Equity Plan）作为一种能够激励股东并扩大股份数量的机制，其目的就是给予创始人的参与权、鼓励他们参与公司的管理活动，从而使其获得更高的回报。除了创始人在企业运营中所产生的价值外，股权计划还可以帮助创始人享受到各种福利待遇和条件。在过去几年里，股权计划已经成为创业公司的一个重要渠道。
那么，怎样才能设计出一个成功的创业公司的股权计划？下面就让我们一起学习一下如何参与创业公司的股权计划。
# 2.核心概念与联系
首先，我们需要了解股权计划的基本概念。股权计划由四个主要组成部分构成，即股权分级、股权分配、增持计划、终止计划。下面分别进行介绍。
## （1）股权分级
股权分级是指根据公司实际情况对其股东进行等级划分，将股份按一定比例分配给各等级股东。不同的等级对应的股份量不同，最低等级的股份量最少，最高等级的股份量最多。一般来说，股权分级的目的是为了提升创始人的收益，比如雇佣人员的晋升或培训费用减免。同时，股权分级也是为了保障创业公司的长期健康发展，确保创始人在资本市场上取得合法的地位和影响力。
## （2）股权分割
股权分割是在公司正常运作时发生的，它是指公司出售股份以赚取利润或扩大股权结构。由于创业公司的起步阶段较早，通常没有足够的资金及能力来进行股权分割，因此很少有创业公司做股权分割。当创业公司进入良性循环后，若出现资金短缺，则会考虑向外借款，或寻找第三方投资者提供担保。
## （3）增持计划
增持计划是指公司发行新股或现金分红的方式，向原有的股东或新的股东分发股份，其目的是为了促进创始人对公司的认同感和参与感，增强创始人的地位和影响力。增持计划一般包括增持股份数量和增持方式两个部分。增持股份数量是指公司向指定股东增加的股份数量；增持方式一般包括增持比例股份和优先股。优先股是指除增持比例股份外，优先购买某些股份的资格要求。例如，创业公司可以优先购买增持计划中的特定股份，也可以按照优先股的价格在指定时间段内购买。
## （4）终止计划
终止计划是指公司正常运营期间遇到困难或市场变化，主动停止发放股权或提前终止股权转让的过程。终止计划存在的原因有很多，比如创始人离职、增持股份分配不均等。如果有需要，创始人可以在终止计划之前申请垫付等方式补偿股权损失。
## 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## （1）股权分割模型
股权分割模型是基于风险资本模型的，它假定创业公司的价值主要来自于稀缺资源（如时间、智慧和人才），而股权分配是一个非平等博弈的过程。由于股权分割是创业公司的一个关键环节，因此研究股权分割模型也是创业公司领导者不可或缺的一项技能。股权分割模型可用于评估不同规模的创业公司是否适宜进行股权分割，以帮助公司制定相应的战略选择。以下是股权分割模型的基本公式：

1. 产品溢价：单位时间内新增产品销售额占总销售额的比例，反映创业公司对市场需求的快速响应程度。

2. 流动性溢价：单位时间内流入资金占总资金的比例，反映创业公司在资金周转中的效率。

3. 投资溢价：单位时间内投入资本支出占总支出的比例，反映创业公司对资本市场投资热情。

4. 制度风险：企业的行为规范及相关政策，包括组织结构、决策机制、薪酬福利、监管制度、知识产权保护等，其对创业公司价值的影响程度。

5. 风险资本：创业公司拥有的最大流动资金或企业核心资产，风险资本在股权分割过程中具有决定性作用。

6. 预期收益率：预测未来市场回报率的模型，反映创业公司未来的盈利能力。

7. 外部环境影响：外界环境因素对创业公司股权分割进程及效率的影响。

根据股权分割模型，可以计算出每一轮股权分割所需要支付的资本成本（又称为银行风险），作为创业公司的股权分割决策依据。股权分割的最终目标是最大化预期收益率与最小化银行风险之间的权衡。
## （2）增持计划模型
增持计划模型是基于MSTAR模型的，MSTAR模型是全球首个全面且客观的跨行业模型，它将财务数据、产品质量、商业模式三个维度综合分析，形成了一个科学框架，揭示了公司发展的驱动因素。在股权计划中，增持计划模型可以用来制定创业公司的增持策略，帮助创始人吸引更多的投资者，降低获利风险。以下是增持计划模型的基本公式：

1. 资金占用：企业投资资金占比，反映企业在融资上的投入情况。

2. 现金流压力：企业现金流与增持计划目标现金流之间的差距，反映企业现金流的相对困难程度。

3. 资本利用率：企业资本投入的有效性，反映企业对资本市场投资状况的把握。

4. 投资环境：企业与市场、竞争对手及个人的关系，反映企业对投资机会和条件的把握。

5. 创始人参与度：企业创始人的参与程度及能力，反映企业创始人的自我感觉和价值观。

6. 竞争优势：竞争对手在同类产品或服务上的竞争优势，反映企业在竞争上处于什么位置。

7. 营收影响：企业与竞争对手营收差距的大小，反映企业营收表现的好坏。

8. 现金流水平：企业现金流的运行状况，反映企业经营状况的真实性。

增持计划模型基于以上八个方面对创业公司的增持计划进行分析，为创业公司制定增持策略提供参考。
## （3）终止计划模型
终止计划模型的建立及完善依赖于企业内外部对股权转让机制的讨论、研究和研究成果。终止计划模型从两个角度阐述了股权转让的效应：收益和风险。

收益方面，终止计划模型认为，股权转让对创始人的声誉、企业凝聚力、核心价值等产生正面的影响，这在保障创始人的权益、增强创始人的参与感、加速创业公司的成功方面是有益的。但同时，股权转让对创业公司长期健康发展也带来了潜在的风险，因此，终止计划模型将股权转让的效益与风险共同考虑，并通过计算退休后的获益率以及延迟退休对创业公司的经济影响，帮助创业公司制定终止计划。

风险方面，终止计划模型认为，目前的股权转让机制存在诸多问题，比如效率低下、管理难度大、失去原股东的控制权、无法为创始人提供所需的就业机会等。终止计划模型将这些问题通过一定的模型和指标来描述，并对现行股权转让机制进行改革。改革措施包括允许现有股东继续持有股权，鼓励创始人与新投资人合伙经营，增加受托经纪人等。

具体操作步骤如下：
### 操作步骤一：签署协议——创始人与公司达成协议，选择股权结构。
创始人在注册时将授权委托律师代为填写所有必要信息，包括股权结构（如优先权股份占比、优先购买股份、增持比例股份占比）。律师将确保创始人注册资料真实准确无误，确保协议遵循法律规定。
### 操作步骤二：查阅会议记录——审核创始人提交的资料，核对股权计划资料的完整性、正确性。
创始人提交的股权计划资料，包括项目说明书、财务报告、业务计划书、股权结构图等，需由公司内部人员审核确认，确保无误后再进行下一步操作。
### �apping操作三：梳理资产负债表、利润表、现金流量表——充分理解公司资产负债表、利润表、现金流量表的内容，核对上半年、下半年、年底资产负债表、利润表、现金流量表的变动情况。
创始人和股东的资产负债表、利润表、现金流量表，都是公司财务报告必备的文件，需要进一步分析、梳理。确认每一项资产负债表、利润表、现金流量表的明细，核对账面金额与股权结构是否一致。
### 操作步骤四：制定增持计划——根据公司现有财务报表、股权结构、增持目标、增持范围等，制定增持计划。
增持计划涉及公司现金流、利润、成本、市场份额等方面，可在公司规划或内部协调下完成，其中“增持目标”指设置的增持金额、时间、方式，“增持范围”指优先选取的股票，具体内容由公司内部讨论并签订协议确定。
### 操作步骤五：签约子公司——合伙协议、分立协议、董事会决议等，签署合伙协议、分立协议、董事会决议。
创始人须与子公司达成合伙协议、分立协议或董事会决议，确认子公司的归属、分工、管理办法和权利义务。
### 操作步骤六：下半年财务复核——预计下半年财务情况，核实公司预算、费用、预收账款、财务报表是否完善。
截至目前，公司完成了对所有材料的审核、确认和初审，但尚未进行收尾核实。此时可以推断公司的财务状况，制定进一步的增持计划。
### 操作步骤七：补充资产负债表、利润表、现金流量表——补充或完善公司资产负债表、利润表、现金流量表的内容，如增加银行存款、现金、账款等。
公司的资产负债表、利润表、现金流量表内容需要保持更新，以便随时掌握公司的资金、收益情况。根据公司现有资金状况、风险承受能力和增持计划情况，采取合理的安排，补充资产负债表、利润表、现金流量表。
### 操作步骤八：签约上市——根据公司条件和发行条件等，签署公司上市协议。
公司条件满足上市条件，拟对外发行股份，须与证券交易所或其他经销商签订上市协议。签约签署完成后，即可启动对外交易。
