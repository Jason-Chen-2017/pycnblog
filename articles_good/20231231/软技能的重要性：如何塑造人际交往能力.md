```python
def chat_bot():
    responses = [
        "我不太明白你的意思，请问能否再解释一下？",
        "很抱歉，我暂时没有理解你的问题。请问能否重新说明一下？",
        "请问你能提供一些更多的信息，以帮助我更好地理解你的问题？"
    ]

    while True:
        user_input = input("你说什么？\n")
        if user_input.lower() == "退出":
            break
        print(random.choice(responses))

if __name__ == "__main__":
    chat_bot()
```
