## 4.1 物流资源调度：车辆调度算法实例

在这个实例中，我们将使用一种常用的车辆调度算法——贪婪算法，来解决一个物流资源调度问题。

问题描述：

有 $$ n $$ 个运输需求，每个运输需求的大小分别为 $$ d_1, d_2, ..., d_n $$，有 $$ m $$ 个车辆，每个车辆的容量分别为 $$ v_1, v_2, ..., v_m $$，运输成本为 $$ c_{ij} $$。求如何安排车辆的运输任务，使运输成本最小。

代码实例：

```python
import numpy as np

def vehicle_scheduling(demands, vehicles, costs):
    # 初始化运输成本
    total_cost = np.inf
    # 初始化运输计划
    schedule = None
    # 遍历所有可能的运输计划
    for i in range(len(demands)):
        # 如果当前车辆容量大于运输需求，则将运输需求分配给当前车辆
        if demands[i] <= vehicles[i]:
            # 更新运输成本
            total_cost = min(total_cost, costs[i])
            # 更新运输计划
            schedule = [(i, demands[i])]
        else:
            # 将运输需求分配给下一个车辆
            for j in range(i+1, len(demands)):
                # 如果当前车辆容量大于运输需求，则将运输需求分配给当前车辆
                if demands[j] <= vehicles[j]:
                    # 更新运输成本
                    total_cost = min(total_cost, costs[i] + costs[j])
                    # 更新运输计划
                    schedule = [(i, demands[i]), (j, demands[j])]
                else:
                    # 将运输需求分配给下一个车辆
                    break
    return total_cost, schedule

# 测试数据
demands = [2, 3, 4, 5]
vehicles = [4, 3, 2, 1]
costs = [1, 2, 3, 4]

# 运输成本和运输计划
total_cost, schedule = vehicle_scheduling(demands, vehicles, costs)
print("运输成本:", total_cost)
print("运输计划:", schedule)
```

## 4.3 物流网络优化：路径规划算法实例

在这个实例中，我们将使用一种常用的路径规划算法——贪婪算法，来解决一个物流网络优化问题。

问题描述：

有 $$ n $$ 个节点，每个节点的权重分别为 $$ w_1, w_2, ..., w_n $$，求如何从起点到终点找出最短路径。

代码实例：

```python
import numpy as np

def shortest_path(weights):
    # 初始化最短路径
    shortest_path = np.inf
    # 初始化当前节点
    current_node = 0
    # 遍历所有可能的路径
    for i in range(len(weights)):
        # 如果当前节点的权重小于最短路径，则更新最短路径和当前节点
        if weights[i] < shortest_path:
            shortest_path = weights[i]
            current_node = i
        # 如果当前节点是终点，则停止遍历
        if i == len(weights) - 1:
            break
        # 更新当前节点
        current_node = i
    return current_node, shortest_path

# 测试数据
weights = [0, 3, 1, 2, 1, 4, 1, 0]

# 最短路径和终点
shortest_path, end_node = shortest_path(weights)
print("最短路径:", shortest_path)
print("终点:", end_node)
```
