## 1.2 蒙特卡罗方法简介

蒙特卡罗方法（Monte Carlo method）是一种基于随机样本的数值计算方法，它的核心思想是利用大量随机数生成的样本来近似地求解某个问题的解。这种方法的名字起源于法国蒙特卡罗国际气象台的一位工程师M. Monte Carlo，他在1940年代开始使用这种方法来解决一些气象模型的问题。

随机数在蒙特卡罗方法中发挥着关键作用，它们可以通过随机生成来近似地求解某个问题的解。这种方法的优点是易于实现，不需要知道问题的具体解，而且可以处理复杂的问题。但是，它的缺点也是明显的，即计算结果的准确性取决于随机数的质量和数量，因此需要大量的计算资源来获得较为准确的结果。

    length = len(password)
    uppercase = sum(c.isupper() for c in password)
    lowercase = sum(c.islower() for c in password)
    digits = sum(c.isdigit() for c in password)
    special_chars = sum(not c.isalnum() for c in password)
