#### 3.1.1数据去重

数据去重是数据清洗的一个重要环节，它的目的是确保数据中不存在重复的记录，以实现数据的有效管理和应用。数据去重可以通过以下几种方式实现：

1. 使用哈希表实现数据去重：

在哈希表中，每个键值对应一个值。我们可以将数据中的每个记录的键值作为哈希表的键，值为1。如果哈希表中已经存在该键值，则说明该记录已经存在，我们可以将其过滤掉。

2. 使用排序和遍历实现数据去重：

首先，我们需要对数据进行排序，以确保相同的记录在相邻的位置。然后，我们可以遍历数据，如果当前记录与前一条记录相同，则将其过滤掉。

#### 3.1.2数据填充

数据填充是数据清洗的一个重要环节，它的目的是确保数据中不存在缺失的值，以实现数据的有效管理和应用。数据填充可以通过以下几种方式实现：

1. 使用平均值填充：

我们可以将数据中的每个缺失值替换为其他相关属性的平均值。例如，如果我们有一个包含年龄、性别和收入的数据集，我们可以将缺失的收入替换为年龄和性别的平均收入。

2. 使用最近邻填充：

我们可以将数据中的每个缺失值替换为与其他相关属性最相似的记录的值。例如，如果我们有一个包含年龄、性别和收入的数据集，我们可以将缺失的收入替换为与年龄和性别最相似的记录的收入。

```python
def remove_duplicates(data):

    result = []
    for item in data:
        if item not in seen:
            seen.add(item)
            result.append(item)
    return result
```

```python
                new_row[mapping[column]] = value
            else:
                new_row[column] = value
        result.append(new_row)
    return result
```
