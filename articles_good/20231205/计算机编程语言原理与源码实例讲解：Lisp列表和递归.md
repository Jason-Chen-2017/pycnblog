# 2.核心概念与联系

## 2.1 Lisp列表

Lisp列表是一种灵活的数据结构，可以用于表示各种不同的数据。列表是由一系列元素组成的有序序列，每个元素都有一个值和一个类型。列表可以包含其他列表，这使得它们可以表示复杂的数据结构。

Lisp列表的语法是通过括号和空格来表示的。列表的第一个元素称为“头”，后面的元素称为“尾”。例如，(1 2 3) 是一个列表，其中1是头，2和3是尾。

Lisp列表可以用于表示各种不同的数据结构，例如树、图、图表等。它们的灵活性使得它们在人工智能和计算机视觉等领域非常受欢迎。

## 2.2 递归

递归是一种编程技巧，可以用于解决许多问题。递归是一种迭代过程，其中一个函数调用自身，直到满足某个条件为止。递归可以用于解决许多问题，例如计算阶乘、求和等。

递归的核心概念是“基本情况”和“递归情况”。基本情况是递归过程中的终止条件，递归情况是递归过程中的递归调用。例如，在计算阶乘的递归过程中，基本情况是n=1时的阶乘为1，递归情况是n>1时的阶乘为n*（n-1）。

递归是Lisp语言的核心特征之一，它使得Lisp语言能够表示和解决许多复杂的问题。

以下是一个递归的代码实例：

```lisp
(defun factorial (n)
  (if (= n 1)
      1
      (* n (factorial (- n 1)))))
```

在这个代码实例中，我们定义了一个递归函数factorial，用于计算阶乘。函数的参数n表示要计算阶乘的数。函数的基本情况是n=1时的阶乘为1。递归情况是n>1时的阶乘为n*（n-1）。
