## 3.1 基于矢量空间的方法

基于矢量空间的方法主要包括插值、卷积、滤波等技术。这些技术的核心思想是通过对低分辨率图像的像素点进行操作，从而提高其分辨率。

### 3.1.1 插值

插值是一种简单的超分辨率技术，通过对低分辨率图像的像素点进行插值来生成高分辨率图像。常见的插值方法有线性插值、双三次插值等。

线性插值的公式为：
$$
I_{h}(x,y) = I(x,y) + \frac{\partial I}{\partial x}(x,y)(x-x_0) + \frac{\partial I}{\partial y}(x,y)(y-y_0)
$$

双三次插值的公式为：
$$
I_{h}(x,y) = I(x,y) + \frac{\partial I}{\partial x}(x,y)(x-x_0)^2 + \frac{\partial I}{\partial y}(x,y)(y-y_0)^2 + \frac{\partial^2 I}{\partial x^2}(x,y)(x-x_0)^3 + \frac{\partial^2 I}{\partial y^2}(x,y)(y-y_0)^3 + \frac{\partial^2 I}{\partial x \partial y}(x,y)(x-x_0)^2(y-y_0)^2
$$

### 3.1.2 卷积

卷积是一种常用的超分辨率技术，通过对低分辨率图像的像素点进行卷积来生成高分辨率图像。卷积操作的核心是卷积核，卷积核是一种滤波器，可以用于提取图像中的特征信息。

卷积公式为：
$$
I_{h}(x,y) = I(x,y) * k(x,y)
$$

### 3.1.3 滤波

滤波是一种常用的超分辨率技术，通过对低分辨率图像的像素点进行滤波来生成高分辨率图像。滤波操作的核心是滤波器，滤波器可以用于消除图像中的噪声和干扰。

常见的滤波方法有均值滤波、中值滤波、高斯滤波等。

均值滤波的公式为：
$$
I_{h}(x,y) = \frac{1}{N} \sum_{i=-n}^{n} \sum_{j=-n}^{n} I(x+i,y+j)
$$

中值滤波的公式为：
$$
I_{h}(x,y) = \text{median}\left\{I(x+i,y+j) \mid -n \leq i,j \leq n\right\}
$$

高斯滤波的公式为：
$$
I_{h}(x,y) = \frac{1}{2\pi\sigma^2} \exp\left(-\frac{(x-x_0)^2 + (y-y_0)^2}{2\sigma^2}\right)
$$

## 4.1 插值

```python
import numpy as np
import cv2
import matplotlib.pyplot as plt

def bicubic_interpolation(img, scale_factor):
    rows, cols = img.shape
    new_rows = int(rows * scale_factor)
    new_cols = int(cols * scale_factor)
    new_img = np.zeros((new_rows, new_cols, 3), dtype=np.uint8)
    for i in range(new_rows):
        for j in range(new_cols):
            i0 = int(i * (1 + (i + 1) / (2 * rows)))
            j0 = int(j * (1 + (j + 1) / (2 * cols)))
            i1 = int(i * (1 + (i + 2) / (2 * rows)))
            j1 = int(j * (1 + (j + 2) / (2 * cols)))
            w0 = (i - i0) * (i - i0) * (i - i0)
            w1 = ((i1 - i) * (i1 - i) * (i1 - i) + (j1 - j) * (j1 - j) * (j1 - j))
            new_img[i, j, :] = img[i0, j0, :] * w0 + img[i1, j1, :] * w1
    return new_img

scale_factor = 4
new_img = bicubic_interpolation(img, scale_factor)
plt.imshow(new_img)
plt.show()
```

import cv2
import matplotlib.pyplot as plt

def convolution(img, kernel):
    rows, cols, channels = img.shape
    new_img = np.zeros((rows, cols, channels), dtype=np.uint8)
    for c in range(channels):
        for i in range(rows):
            for j in range(cols):
