线性回归模型的数学模型公式为：

$$

逻辑回归模型的数学模型公式为：

$$
P(y=1|x_1,x_2,\cdots,x_n) = \frac{1}{1+e^{-\beta_0-\beta_1x_1-\beta_2x_2-\cdots-\beta_nx_n}}
$$

其中，$P(y=1|x_1,x_2,\cdots,x_n)$ 是目标变量的概率，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

$$

## 4.2 智能空气质量检测器示例

### 4.2.1 硬件连接

```python
import Adafruit_ADS1x15
import time

ads = Adafruit_ADS1x15.ADS1115()

# 读取ADC值
def read_adc():
    adc_value = ads.read_adc(0, gain=1)
    return adc_value

# 计算空气质量
def calculate_air_quality(adc_value):
    air_quality = (adc_value - 500) / 10
    return air_quality

# 获取空气质量
air_quality = calculate_air_quality(read_adc())
print("空气质量: ", air_quality)
```
