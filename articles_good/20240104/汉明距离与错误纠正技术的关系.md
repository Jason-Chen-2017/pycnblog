## 1.1 汉明距离的定义与基本概念

汉明距离是一种度量两个二进制字符串之间的距离，通常用于比较两个数据的不同程度。它的定义为：在两个二进制字符串中，相同位置上bit值不同的位数。例如，对于字符串A = 1011和字符串B = 1100，它们的汉明距离为2，因为它们在相同位置上有两个bit值不同。

汉明距离具有以下性质：

1. 对于任意二进制字符串X和Y，其汉明距离d(X, Y)满足0 ≤ d(X, Y) ≤ min(length(X), length(Y))。
2. 对于任意二进制字符串X和Y，如果X = Y，那么其汉明距离d(X, Y) = 0。
3. 对于任意二进制字符串X和Y，如果X ≠ Y，那么其汉明距离d(X, Y) > 0。
4. 对于任意二进制字符串X和Y，如果X ≠ Y，那么交换X和Y中的任意两个bit值可以使其汉明距离减少1。

    distance = 0

            distance += 1
    return distance
