## 4.1 代码实例

```python
import numpy as np

# 初始化模型参数
theta = np.random.rand(10)

# 定义损失函数
def loss_function(theta):
    return np.sum(theta**2)

# 定义梯度函数
def gradient(theta):
    return 2 * theta

# 定义梯度校准函数
def gradient_correction(theta, learning_rate):
    grad = gradient(theta)
    new_theta = theta - learning_rate * grad
    return new_theta

# 训练模型
learning_rate = 0.01
for i in range(1000):
    theta = gradient_correction(theta, learning_rate)
    loss = loss_function(theta)
    print(f"Iteration {i+1}, Loss: {loss}")
```

## 4.2 代码解释

在这个代码实例中，我们首先初始化模型参数`theta`。然后，我们定义了损失函数`loss_function`和梯度函数`gradient`。接下来，我们定义了梯度校准函数`gradient_correction`，它接受模型参数和学习率作为输入，并返回调整后的模型参数。

在训练模型的过程中，我们使用梯度校准函数来调整模型参数，并计算每次迭代后的损失值。通过这种方式，我们可以看到模型参数逐渐收敛，损失值逐渐降低。
