# 1.简介
  

互联网技术的飞速发展已经给企业带来了巨大的变革机遇和挑战。作为互联网公司，无论是创业初期，还是日益壮大的用户群体、竞争激烈的市场环境、快速发展的电商平台，都需要不断地寻找新的增长模式、打造品牌优势。SEO（Search Engine Optimization）作为优化互联网信息排名、提高网站流量的一个重要工具，在实现网站价值和建立忠诚度方面发挥着至关重要的作用。

随着互联网爆炸式增长、多元化发展和信息化进程的加快，SEO已经成为企业突破瓶颈的新途径之一。SEO从古至今已经成为一种行业内外共同认可的最佳做法，而一些经验老道、见解独到的SEO专家也成为了“SEO先锋”。本文将介绍十三条基本的SEO技巧，帮助企业更好地实现网站的全面优化。

# 2.概念和术语
## 2.1 什么是SEO？
SEO即搜索引擎优化，它是一个通过对网站进行技术手段和管理方法，有效地提升网站在搜索引擎上的排名和收录，从而获得更多的流量和用户。其核心是通过关键词选择性的建站、设置目标页URL结构、提供有意义的链接、创建有价值的副标题、优化页面设计、提高网站内容质量、利用外部资源如图片、视频、表单等提升网站的互动性，并最大程度地提升网站的权威性和影响力。

## 2.2 为什么要优化网站呢？
SEO是一种综合性的解决方案，它既涉及到网站策划、建设、运营、内容制作、维护，还要兼顾网站本身的策略、布局、外观、功能。网站的优化旨在提升网站的搜索引擎自然排名、提升网站知名度、增加网站流量和转化率。因此，优化网站的目的就是为了让网站更容易被检索、被发现，从而达到优化营销效果、提升企业形象的目的。

## 2.3 搜索引擎的基础知识
### 2.3.1 什么是搜索引擎？
搜索引擎是网络上最广泛使用的信息查找工具。它主要用于检索互联网索引文件中的大型数据库，以获取相关的信息。搜索引擎可以分为两大类：免费的、商业的搜索引擎。

免费搜索引擎包括谷歌、Bing、Yahoo等，这些搜索引擎提供基于文本的搜索服务，对于个人和小型组织的搜索需求来说，它们足够满足需求；而付费的搜索引擎则提供了更为丰富的搜索功能和更为精准的结果排序。

### 2.3.2 关键字和搜索词的区别？
关键字是人们用语音、文字或图像描述的主题、产品或者事物，搜索词是指计算机通过搜索引擎向搜索结果显示的关键词。一般情况下，关键字会使用主动语态（例如“咖啡”）进行描述，而搜索词则一般使用被动语态（例如“喝杯咖啡”）。

关键字是指人们赋予网站的名称，搜索引擎通过对关键字进行分析、挖掘和整理，从海量的互联网数据中找到客户所需的内容；而搜索词是指用户输入的搜索请求，它通常是关键词的组成部分，而且它的语言、表达方式往往与用户习惯不符。

### 2.3.3 为什么会出现重复内容？
重复内容是指相同的或相似的内容，比如重复的广告、重复的宣传语、相同的图片、相同的视觉风格等。当用户搜索某个关键词时，如果网站内容过于丰富，搜索结果就会呈现出很多重复内容。

为了解决这个问题，SEO专家们通常会采用以下几种策略：

1. 尽可能避免使用重复的内容：减少同类网站的数量，不使自己的网站产生垃圾邮件，不要重写网站的导航栏和标签，不要使用重复的照片或视觉元素。
2. 提高内容质量：网站的每一个页面都应该保持更新，包括正文内容、照片、视频、音频等，确保内容的准确性。
3. 使用外链：网站的作者可以利用互联网资源，把相关的文档、图片、视频等外链到网站的各个页面上，为自己的网站增加更多的价值和链接。
4. 创建适当的跳转链接：把重要的关键词链接到其他网站上，用户可以在搜索结果中直接访问到网站内容。
5. 不要刻意推广太多的内容：网站的内容越多，搜索引擎的工作就越繁重，因此需要制定相应的政策和规划来控制搜索量。

### 2.3.4 什么叫做爬虫？
爬虫又称网络蜘蛛，它是一种自动获取信息的程序，它会按照网站的目录结构、页面连接关系，对网页内容进行抓取。而SEO专家们的目标就是通过爬虫来获取网站的内容。

### 2.3.5 如何通过爬虫来获取网站的内容？
SEO专家们通常采用的方式是模拟浏览器的行为，也就是通过开发脚本，模仿用户浏览网页的过程，自动打开网页、提交表单、点击链接等，然后通过解析页面获取所需信息。

这种方式虽然简单易行，但也存在一定的限制，它无法像真实用户一样获得网站的所有信息。另一方面，通过爬虫的方式只能获取当前网站的内容，不能获知网站的上下文环境，可能会漏掉一些更为重要的信息。因此，SEO专家们还需结合其他的方法，如搜索引擎爬虫、通用爬虫等，来进一步获取网站的上下文环境。

# 3.SEO核心技术
## 3.1 URL结构优化
### 3.1.1 什么是URL？
URL（Uniform Resource Locator），中文译作统一资源定位符，用来标识互联网上的资源位置。

### 3.1.2 URL的构成？
URL由三部分组成：协议、域名、网址。如下图所示：


- 协议：表示数据通信的规则、规范、尺度等。http表示超文本传输协议，https表示安全超文本传输协议。
- 域名：域名是互联网世界中的唯一名称，它是网页的地址。它由一串字母数字组成，通常是顶级域名（com，cn，org）和二级域名（www，bbs，shop）组成。
- 网址：网址是域名后面的那一串字符，用来确定网站的具体位置。网址可以是域名，也可以是域名下的子目录或文件。

### 3.1.3 为什么要优化URL？
URL是作为网站的门牌号码，它承载着网站的身份、特征和价值，同时也是用户检索网站时的关键因素。

由于URL的重要性，所以网站的URL一定要清晰、简洁，且具有代表性。但是很多网站的URL仍然比较生硬，难以形成良好的搜索排名。

因此，要优化URL的关键点就是要缩短网站URL的长度、增加关键词的数量、使用正确的网址编码方式。

### 3.1.4 URL的分类与优化
#### 3.1.4.1 固定链接
固定链接就是指用固定的网址去指向一个特定的页面，这种方式主要用于网站首页、品牌专题页、内部链接、邮箱订阅等。固定链接的形式一般都是直接访问，不需要输入关键字，所以会受到排名的影响。

例如，搜索引擎会把关键词输入框置于固定链接之后，如“http://www.example.com/”，这样的网址会受到搜索引擎欢迎。

#### 3.1.4.2 动态链接
动态链接是指用参数化的网址去指向特定的页面，这种方式主要用于搜索页、列表页、详情页、归档页等。动态链接允许用户根据自己的需求，通过输入关键词进行过滤筛选，所以不受固定链接的影响。

例如，搜索页的网址可以设为“http://www.example.com/?s=关键词”，这样的网址不会受到搜索排名的影响，但可能会降低用户的流量。

### 3.1.5 SEO利器——URL优化器
许多SEO专家都建议使用URL优化器来优化网站的URL。URL优化器是一个独立的软件，它可以自动识别并标记页面中的重复内容、提取网站关键词、生成优化的URL，并实时更新网站的URL，降低网站的排名波动。

URL优化器可以帮助网站的建设者避免使用重复的URL，改善网站的结构、提升网站的品牌形象，并提升网站的SEO效果。

## 3.2 关键词密度和排名位置调整
### 3.2.1 什么是关键词密度？
关键词密度是指某一个主题下，关键词的重复次数与该主题关联度的比值。它反映了网站的重要性，也会影响到其收录、排名、展示的顺位。

### 3.2.2 怎么才能提高关键词密度？
要提高关键词密度，除了要注意建设具有吸引力的品牌关键词外，还应遵循以下几条建议：

1. 用最相关的关键词：使用最能描述网站主题的关键词，而不是各种乱七八糟的关键词。
2. 短小的关键词：使用较短的关键词，因为搜索引擎的检索范围是有限的。
3. 分层次的关键词：网站的不同页面使用不同的关键词，可以有效地提升不同页面的关键词密度。
4. 广泛的关键词：使用多个不同领域的关键词，可以有效地提升网站的关键词覆盖度。
5. 切忌侧重于堆砌数字：使用关键词时切忌过度夸张，导致页面排名位置偏离。

### 3.2.3 优化关键词排名算法的四个步骤
当多个关键词出现在同一个页面时，搜索引擎的排名算法决定了哪些词将排名靠前。下面是SEO专家们经常使用的优化关键词排名算法的四个步骤：

1. 验证投放位置：验证自己的网站是否放在合适的位置，将重要的关键词放在重要的位置。
2. 优化内容：优化网页内容，使之突出网站的主要功能和产品优势。
3. 优化网页结构：优化网页结构，使其符合搜索引擎的检索规则。
4. 使用外部资源：使用外部资源如图片、视频、表单等来增加网页的互动性。

### 3.2.4 浏览量和排名
浏览量是指某个网页被搜索引擎记录的总次数，排名是指网站在全球范围内的搜索结果中，排名的顺序。

搜索引擎的算法决定了一个网页的排名。Google的算法会考虑到网页的原创性、性价比、所链接的其他网页等因素，同时也会将网页的关键词和用户查询的匹配度作为依据。因此，了解搜索引擎对网页的判读算法，以及提升排名的一些技巧，可以提升网站的流量。

### 3.2.5 重定向排名
重定向排名是指搜索引擎将搜索结果引导至其他网站的形式。此时，搜索引擎认为目标网站的质量要胜过原始网站，所以将其排名提升。重定向排名通常用于网站改版和品牌建设等重大活动。

例如，搜索引擎在发现您输入的是“小米手机”，它可能就会将搜索结果引导至“小米官方网址”，然后再将您引导回到“小米官网”。

### 3.2.6 上下文匹配排名
上下文匹配排名是指搜索引擎根据用户搜索的历史记录、行为习惯和网络内容特征，匹配候选网站的相关性，将其排列在搜索结果的首位。上下文匹配可以提高网站的流量、用户满意度和权威性。

例如，搜索用户喜欢吃的东西时，它可能会将“美食”这一词与相关的网站排在搜索结果的首位。

## 3.3 建设有影响力的SEO策略
### 3.3.1 什么是影响力策略？
影响力策略是一系列针对SEO人员、网站建设者和目标用户的决策和实施方案，旨在提升网站的知名度和品牌影响力。影响力策略往往包括网站营销策略、搜索引擎营销、SEM、社交媒体、内容营销、白皮书、案例研究、品牌识别、品牌定位、品牌形象设计、品牌营销、PR活动、营销推广包装、品牌宣传、品牌提升计划等。

### 3.3.2 怎么提升网站影响力？
要提升网站影响力，首先需要有良好的品牌形象和口碑。对于品牌形象，只要在产品和服务方面有突出的特色，就可以吸引用户关注，提升网站的曝光率。

对于品牌影响力，可以通过建立品牌识别、品牌定位、品牌形象设计、品牌营销、品牌宣传等方面，完成品牌形象的塑造、搜索引擎的推荐和社会的关注。

除此之外，还可以采取以下策略来提升网站影响力：

1. 通过热点事件的报道：大型事件、热点新闻、突发事件、政局变化等都可以发布至各种媒体，来刺激用户参与互动，从而提升网站的受众参与度。
2. 通过SEO策略提升网站权威性：使用有价值的关键词，通过竞争力的网站内容，提升网站的影响力，扩大搜索引擎收录，实现网站价值最大化。
3. 积极参与到本地和国际的活动中：举办活动、参与竞赛，都可以提升网站的品牌形象，扩大影响力。

## 3.4 SEO建设方法论
### 3.4.1 三个层级的SEO建设
在SEO建设中，通常按三个层级进行建设：战略层、平台层、核心层。

1. 战略层：战略层是SEO建设的最高层，一般都是公司的最高层或副总裁在进行。战略层的任务是整体规划和实现公司的SEO策略。

2. 平台层：平台层是SEO建设的第一步，是面向网站所有用户的基础建设。平台层的任务是为网站搭建基础技术架构，搭建网站基础数据平台，搭建网站管理系统。

3. 核心层：核心层是SEO建设的最后一步，也是最具创造性的一环。核心层的任务是为网站的SEO做专业的管理、培训、研发、部署、维护，确保网站的稳健运行和持续提升。

### 3.4.2 SEO的内容建设
SEO内容建设可以分为内容策划、网站建设、营销推广、SEM、社交媒体等。

1. 内容策划：内容策划主要是围绕企业的核心业务，深入挖掘网站的客户痛点，制定内容规划，确立网站建设方向。

2. 网站建设：网站建设是SEO内容建设的第一步，是确立网站目标，进行网站设计，编写网站代码，部署网站服务器。网站建设内容包括：
    - 网站标志
    - 网站架构
    - 网站基础信息
    - 网站内容设置

3. 营销推广：营销推广是SEO内容建设的第二步，主要是策划和执行网站营销活动。营销推广内容包括：

    - 营销策略：制定营销战略，定义目标客户群体，制定营销推广活动，推送营销内容。
    - 营销工具：熟悉SEO推广工具，构建有效的营销渠道，提升营销效率。
    - 营销推广方案：制定营销推广方案，编写推广文案，进行推广活动。

4. SEM：SEM（Search Engine Marketing）即搜索引擎营销，是SEO内容建设的第三步。SEM的内容包括：
    - SEM策略：制定SEM战略，制定SEM策略，搭建SEM工具，测试SEM效果，优化SEM效果。
    - SEM效果评估：分析SEM效果，制定SEM改进方案。
    - SEM内容制作：制作高品质的SEM内容，投放到搜索引擎中。

5. 社交媒体：社交媒体是SEO内容建设的第四步。社交媒体内容包括：
    - 内容生产：写好社交媒体内容，制作成果分享。
    - 技术支持：搭建社交媒体运营平台，建立内容生产和传播机制。
    - 发推：发推时，注意遵守社交媒体平台规则，选择适当的图片、视频，并添加标签。

### 3.4.3 SEO的管理方法
SEO管理方法是指通过管理和控制来提升网站的核心竞争力。管理方法往往分为内部管理和外部管理。

1. 内部管理：内部管理是指采用内部人的管理办法来提升网站的竞争力。内部管理内容包括：
    - 编辑部管理：为网站编辑团队提供支持，鼓励其提高文章质量，完善文章审核流程，增强网站内容创作能力。
    - 技术管理：促进网站技术开发、维护、更新，提升网站的技术能力，维护网站的稳定性。
    - 财务管理：关注网站的资金支出情况，持续提升网站的收入和盈利能力。

2. 外部管理：外部管理是指采用第三方公司、机构的管理办法来提升网站的竞争力。外部管理的内容包括：
    - 服务提供商管理：从服务提供商处购买、租用服务器、域名，为网站架设稳固的网络平台。
    - 第三方审查：委托第三方公司进行网站的审核，增强网站的安全性、完整性、可信度。
    - 数据分析：使用数据分析工具对网站进行分析，提升网站的流量、访客、留存、转化率等指标。
