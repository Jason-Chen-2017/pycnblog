# 1.背景介绍


计算机科学教育系统的发展是一个全新的领域,它至今仍处在蓬勃发展阶段。由于IT从业人员对计算机知识的需求日益提升,越来越多的大学、研究机构及企业开始投入计算机科学教育体系建设与管理。作为国内最早将计算机科学教育作为核心教育的一所知名高等院校——北京航空航天大学（BIT）的一项研究报告指出,截止2017年末,中国拥有计算机科学相关专业的博士生数量达到19万人左右,仅次于美国。

因此,为了满足现代信息技术产业对专业人才培养的需要,建立起一支适合国内需求的计算机科学教育系统是十分重要的。对于新兴的计算机科学领域,如何实现精准且有效的教学制度是一个关键问题。目前,多种计算机科学教育模式已经被提出,如应用型计算机科学课程设计、项目实践训练、实验室基础教育、开放课程和互动式学习等。它们都或多或少地吸收了计算机科学的实际情况,但各自也存在一些共性和不同之处。本文将基于计算机科学作为一个研究方向进行讨论,主要阐述基于实践经验对计算机科学教育系统的构建与改进的一些探索和建议。

# 2.核心概念与联系
计算机科学作为一门研究和开发计算机系统及硬件的人工智能技术的学科,其课程设置与教学方式是非常灵活的。一般情况下,一个计算机科学教育计划可以包括以下方面:

1. 核心课程学习:由重点课堂、典型案例、作业辅导、课后问题分析、期末考试等组成,是学生熟练掌握课堂内容并具备应试能力的重要途径。

2. 普通课程学习:通过综合多个课程,使学生了解各个领域的基础知识和能力,培养学生的创新能力。

3. 跨专业课程学习:通过课程间的交流、结合不同学科的知识与技能,鼓励学生更好地融会贯通,提升学生的综合能力。

4. 学分制度学习:是一种很好的评估学生能力的方法,能够给予老师更多的指导和鼓励。但是,在实践中,往往存在因时间和经济负担不足而导致的课程难度不够或课程不完整的问题。

5. 完善的反馈机制:为了保证学生及时获得反馈,可设置完善的考核机制、平时反馈、教师评价等。

6. 可持续发展:计算机科学教育需要不断地向前发展,促进科技的发展、创新与应用。

## 3.1课程设置
在计算机科学教育中,课程设置通常分为预备课程、核心课程、标准课程、自由选修、专业级选修、实习课程等六个层次。

预备课程:这些课程对学生的知识结构和求职技能提升具有重大意义,一般设置在所有学生入学之前。比如,理论课程、计算概论、程序语言基础、数据结构、通信网络、电子工程等。

核心课程:核心课程为学科发展的基石,是对学科知识和技能的考核和培养,一般包括必修课程和任选课程两种。必修课程设置在开设学科课程的基础上,一般占总课时的40%-50%,并且是必修课。任选课程设置在核心课程之后,一般占总课时的20%-30%,并且按指定条件确定是否招收,如已有相关课程或时间,则不需要再次招收该课程。

标准课程:一般来说,一门学科涉及到的知识和技能很多,难免会漏掉某些重要的主题。因此,除了核心课程外,还设置了一批标准课程,强化学生学习某些专业名词和技能。如信号与系统、数据库、编译原理等。

自由选修:即使在非核心课程中,也可设置一些自由选修课程,如专业方向课程、个人兴趣课程、研究项目课程等。这些课程既有助于学生了解其他课程的内容,又可帮助学生选择自己感兴趣的课程。

专业级选修:专业级选修课程设置更加专业,一般包括研究生课程、硕士课程、博士课程等。学校或机构根据学生的研究或毕业方向,在选修课的范围内为学生提供更丰富的教学内容。

实习课程:实习课程可以加强学生的职业竞争力,有利于学生在校工作及晋升的成功。实习课程设置的形式可以是系列实习、考察实习、个人实习等。

## 3.2授课方式
计算机科学教育的授课方式有多种,如线下授课、线上授课、混合授课、问答课等。其中,线上授课较为普遍,主要采用直播、录屏、直播回看的方式。线下授课一般采用教室教学,由教授和助教主讲,也有助于学生的自主学习和交流。

授课内容:授课内容通常以讨论式教学为主,围绕着一些基础理论、原理和实践应用进行阐述。教材的编写需要遵循一定的规范,从教师的角度出发,注重启发学生的思维,并引导学生运用所学知识解决实际问题。

教学方法:授课过程中,一般采取由教师带领学生演练、拆解和复习等学习方法,相信学生能更好地理解、消化和记住所学的内容。

## 3.3学生目标与培养方案
在计算机科学教育中,教师需要针对学生的特点、能力、综合素质等多方面进行教学安排,形成相应的教学方案。具体包括以下几个方面:

1. 学生认知:教师应该站在学生的角度,引导学生从学习目的、任务描述、知识背景等各个方面理性思考。此外,还需帮助学生形成正确的学习习惯和效率工具,树立正确的学习态度和能力评价指标。

2. 学生能力:教师应该帮助学生掌握基础理论、编程技巧、数学分析、统计学、机器学习等计算机科学领域的基本知识,并能运用这些知识解决实际问题。

3. 学生独立性:教师应坚持学生参与学习、完成作业的原则,鼓励学生自行决定是否参与实验、研究等活动。同时,也应引导学生了解学习过程中的风险和注意事项,做好信息安全的防护。

4. 学生综合素质:教师应该关注学生在学习和生活中的综合发展,将知识、能力、情商和责任心融于教学之中,塑造学生的全面发展。

# 5.未来发展趋势与挑战
随着计算机科学教育的发展,新世纪的科技革命正在席卷世界。而当前的计算机科学教育还面临许多问题。首先,新世纪的计算机科学离不开快速的计算机发展速度、大规模的数据存储、人工智能和物联网技术的快速发展。但同时,计算机科学的教育面临许多挑战,如科技水平落后、理论水平偏低、学生需求不足、学生满意度不高等。所以,要让计算机科学教育朝着更美好的方向发展,就需要从以下几个方面进行探索:

1. 提升教育科技水平:在计算机科学教育的整个过程中,教师和学生都需要接触到最新的科技理论、实践工具、技术方法等,但绝大部分教学资源都是古老的.所以,教育科技部门需要跟上时代的步伐,加快科技迭代,推出全新的教学内容。例如,基于智能机器人的编程课程、人工智能算法课程、云计算课程、数据结构与算法课程等。

2. 拓宽教学内容:当前,计算机科学教育的教学内容仍停留在基本语法、数据结构、算法、系统编程等方面,对于实际工作中的应用开发等更高级的内容缺乏支持。所以,教师需要拓宽自己的教学视野,提升教学内容的深度和广度。在内容上,计算机科学教育可以从新型互联网应用、大数据分析、机器学习、高性能计算、量子计算、芯片设计等方向进行延伸。在教学形式上,可以设置开放课程、项目实践训练等方式,让学生真正感受到实践的乐趣。

3. 扩大学生群体:虽然计算机科学的教育在数量上远远小于其他学科,但在学生群体方面却有待提高。例如,可以从青少年教育和早期教育的角度考虑,引入编程实验课、竞赛实训课、科技辅导课、创客教育等项目,激发学生的创新潜力。另外,也应该重视中学计算机科学教育,增强对计算机科学、软件工程和网络工程等专业技术的培养。

4. 营造良好的学习氛围:无论是在线教育还是实体教育,学习氛围始终是一个重要的方面。通过引导学生保持良好的心态、积极主动参与学习,教师可以在学生身上塑造良好的品牌形象,营造一套令人愉悦的学习氛围。除了上述的四项策略,还可以通过社会分享、志愿服务等方式,为学生提供更多的自主学习空间。

# 6.附录常见问题与解答
Q：什么样的计算机科学课程最适合非计算机专业的学生？
A：可以参考网络科学、信息安全、分布式计算、数据可视化、云计算、机器学习等专业课程。

Q：为什么越高等的计算机专业，计算机科学课程越难学？
A：首先，计算机科学的知识面和技能要求是越来越复杂的。只有掌握了基础的计算机理论知识，才能掌握更复杂的知识和技能。其次，课程设置与授课方式也越来越复杂。比如，数据结构与算法教程，既需要系统的理论知识，又需要编写代码实现。而且，计算机科学的内容在变得越来越多，新生代的研究人员和工程师们也需要不断地学习和更新技能。

Q：如何构建一支完整的计算机科学教育系统？
A：构建计算机科学教育系统，可以参照如下建议进行建设：

1. 课程设置:应以计算机科学为核心内容，逐步引入数据结构与算法、操作系统、计算机网络、数据库、计算机图形学、人工智能、机器学习等专业课程。

2. 授课方式:授课方式可以以线下为主、线上为辅的方式进行。线下课程一般采用教室教学方式，可以举办计算机班、项目班、比赛班等，邀请更多的计算机专业学生上课。线上课程可以采用平台授课、直播授课、录屏回看的方式，让学生在线上获得良好的学习效果。

3. 学习辅助工具:可以设置编程辅助工具、画图工具、虚拟机环境、图书馆系统、作业帮系统等，提升学生的动手能力和解决问题能力。

4. 反馈机制:采用课后自动评定、可视化学习措施、参与式学习等方式，帮助学生及时获得反馈，改进教学效果。

5. 面向学生的教育培训模式:可分为微职业培训、职业教育模式、专业教育模式等。微职业培训以实际项目驱动，如大学生创客、移动应用开发、算法竞赛等；职业教育模式以核心专业课程为主，如计算机科学技术、软件工程、网络工程、信息安全等；专业教育模式以全英语授课，设立专业、学术讲座、学术研讨会等。
