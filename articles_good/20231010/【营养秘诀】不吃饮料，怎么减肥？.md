# 1.背景介绍


大家都知道减肥不可怕，可怕的是什么时候减肥，还能坚持下去。而想要坚持减肥，首先就需要明确自己的减肥目标和方式。
但偶尔会听到一些人说“不吃饮料，怎么减肥”，“就算吃饮料也没用，我就是吃太多了”。如果不能正确认识减肥过程中“不可避免的代谢损失”，“心脏活动减慢”等现象，对减肥的效果可能就会受阻。

所以本文将从宏观层面探讨“如何减肥不吃饮料”，提出一个比较客观的方案——“改善身体机能”，并通过科学模型计算，给出每日运动、饮食调整建议。

# 2.核心概念与联系
## 饮食：宇宙万物之源，人体消化道重要器官，其主要功能是为了获取营养分子，并进行初步的转化成为人体所需的微量元素（如碳水化合物），促进身体机能的恢复和肌肉强壮。
## 运动：运动是人体的基础保障，无论什么工作、学习、生活中，都会有大量的运动活动，能够调节身体各个器官的功能。
## 代谢：代谢是指在体内某些微生物从细胞组织分泌出来释放的能量。这是导致体重增加的根本原因。
## 易摄入食物：易摄入食物包括蔬菜、水果、肉类、蛋奶等。由于身体组织结构及基因的作用，导致部分胆固醇、核糖体等各种血糖化合物无法被消化吸收，进而造成代谢产物紊乱、性功能障碍等一系列后果。
## 钠盐浓度：盐的含量越高，代谢产物会相对更多地被吸收，造成身体代谢功能下降，导致体重下降。并且盐容易被腐蚀或渗透，导致皮肤干燥、畸形等问题。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 一、饮食方面：
1、早餐：适量食用维C富含的豆制品。其中的花生和豆乳富含丰富的维生素C，能够增强免疫力和抵抗病菌。另外，还有油炸牛排、米饭和方便面都可以让你的味蕾舒缓、口感好吃。

2、午餐：适量食用蛋白质、鱼、鸡蛋。蛋白质含量较低，还具有防癌、抗癌、补充骨质的功效。鱼、鸡蛋含有的纤维素、钙质和磷质可以帮助消化。

3、晚餐：适量食用全麦粉，因为全麦粉里面含有的维生素A、维生素B1、维生素E、叶酸等可以改善大脑和神经系统的功能。

## 二、运动方面：
1、锻炼身体机能：选择户外跑步或举铁/举杠铃等冷门运动。锻炼大脑、眼睛、耳朵、呼吸等各个器官。

2、遵循德国人的“七小时工作制”：早上6点起床，开始7小时的训练，按照计划完成训练。

3、减少油腻或高盐饮食：少吃油腻的食物和高盐度的饮食，这样可以有效减轻体内油脂的堆积。并且可以避免体内钠盐超标导致的身体代谢功能低下、运动能力下降、体重减轻。

## 三、代谢方面：
1、控制饮食：低盐低糖的食谱，可以有效减少甘薄咖啡因的摄入，也能控制药物副作用。

2、多吃瘦肉、鱼类、蔬菜等瘦身食品：减少胖食肉、鱼类、蔬菜，可以改善体重，同时不改变生活方式。

3、运动与锻炼：多做力量训练、冥想练习，可以增加代谢的活性物质，提升身体机能。

function calculateCaloriesNeeded() {
  const heightCm = document.getElementById("height").value; // 获取身高 cm
  const weightKg = document.getElementById("weight").value; // 获取体重 kg
  const ageYears = document.getElementById("age").value; // 获取年龄 years
  const activityLevel = document.getElementById("activity-level").value; // 获取活跃度等级 1-5
  const dietMode = document.getElementById("diet-mode").value; // 获取饮食模式

  let BMR = Math.round((10 * weightKg) + (6.25 * heightCm ** 2) - (5 * ageYears) + 5); // 基础代谢率公式

  let extraCalorieLoss = 0; // 没有额外减少的卡路里数量
  switch (dietMode) {
    case "vegetarian":
      extraCalorieLoss = 0.8; // 蔬食减少量，范围0.5-1
      break;
    case "paleo":
      extraCalorieLoss = 0.4; // 鱼瘦减少量，范围0.25-0.5
      break;
    default:
      break;
  }

  let totalCalories = Math.max(((BMR * activityLevel) - (BMR * activityLevel * extraCalorieLoss)), 0);

  return `You need to consume ${Math.ceil(totalCalories / 4)} portions of food per day.`;
}

document.querySelector("#calculateBtn").addEventListener("click", function () {
  alert(calculateCaloriesNeeded());
});

function [bmr,kcalMin,kcalMax] = getBMRAndLimits(weight,height,age)
%Get the basal metabolic rate and maximum and minimum caloric values for the given set of demographics
%Inputs: 
%    weight     in KG   floating point number
%    height     in CM   floating point number
%    age        in Years   integer number
%Outputs:
%    bmr         basal metabolic rate   floating point number
%    kcalMin     minimum allowable caloric range   floating point number
%    kcalMax     maximum allowable caloric range   floating point number

%Constants used in calculations
const c=10^3;% mmHg to MPa conversion factor
const h0=150;% resting heart rate in beats per minute
const rho0=1.2;% resting water density in g per cc

%Slope and y-intercept terms for calculating BMR
const slope=(c*(h0**(-1)));%slope is negative because we are finding logarithmic units
const intercept=log(rho0)-(slope*log(height/100)); 

%Derived constants used in calculations
const AGEFAKTOR=0.85;% Multiplier for BMR reduction as age increases

%Calculating min and max limits for caloric intake
const minHeights=[160];%minimum heights above which no further adjustment is made
const maxHeights=[220];%maximum heights beyond which no further adjustment is made

minRange=(minHeights/100).^(2./3)*1.6/.75.*height.^(1./3)*AGEFAKTOR*.45;% minimum allowable caloric range for given gender and age
maxRange=(maxHeights/100).^(2./3)*1.6/.75.*height.^(1./3)*AGEFAKTOR*.45;% maximum allowable caloric range for given gender and age

bmr=kcalBase=kcalFinal=0; % Initialize variables
if age<30 && height>=160 && height<=220 
    bmr=intercept+(slope*log(weight)); % Basal metabolic rate calculation using regression equation
    kcalBase=round((weight*bmr)/(10*math.pi*(height/100)^2)); % Baseline calories calculated using BMR
    kcalMin=floor(kcalBase-minRange); % Minimum limit on daily caloric intake
    kcalMax=ceil(kcalBase+maxRange); % Maximum limit on daily caloric intake  
end
    
end
