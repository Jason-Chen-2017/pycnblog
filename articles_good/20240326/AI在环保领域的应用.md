## 4. 具体最佳实践：代码实例和详细解释说明

以下是一个基于机器视觉的智能垃圾分类系统的代码示例:

```python
import cv2
import numpy as np
from tensorflow.keras.models import load_model

# 加载预训练的垃圾分类模型
model = load_model('garbage_classification_model.h5')

# 定义垃圾类别
garbage_classes = ['paper', 'plastic', 'metal', 'glass', 'organic']

# 实时捕获视频流
cap = cv2.VideoCapture(0)

while True:
    # 读取视频帧
    ret, frame = cap.read()
    
    # 预处理图像
    img = cv2.resize(frame, (224, 224))
    img = np.expand_dims(img, axis=0)
    
    # 使用模型进行垃圾分类
    predictions = model.predict(img)
    predicted_class = garbage_classes[np.argmax(predictions[0])]
    
    # 在视频帧上显示分类结果
    cv2.putText(frame, predicted_class, (50, 50), cv2.FONT_HERSHEY_SIMPLEX, 1, (0, 255, 0), 2)
    
    # 显示视频帧
    cv2.imshow('Garbage Classification', frame)
    
    # 按 'q' 退出
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# 释放资源
cap.release()
cv2.destroyAllWindows()
```

该代码实现了一个基于实时视频流的智能垃圾分类系统。主要步骤包括:

1. 加载预训练的垃圾分类模型
2. 定义垃圾类别
3. 实时捕获视频流并读取视频帧
4. 对视频帧进行预处理,输入到模型进行垃圾分类
5. 在视频帧上显示分类结果
6. 显示视频并等待用户退出

通过这样的实时垃圾分类系统,可以帮助提高城市的垃圾分类效率,为后续的资源回收利用奠定基础。
