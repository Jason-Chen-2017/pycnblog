## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

神经网络的核心是前向传播和反向传播算法。以下是这两个过程的数学模型：

#### 前向传播

输入层到隐藏层的激活函数为 $a_{ij} = \sigma(w_{ij}x_j + b_j)$，其中 $w_{ij}$ 是权重，$b_j$ 是偏置，$\sigma$ 是激活函数（例如 sigmoid 函数）。

输出层的输出为 $y_k = \sigma(\sum_{j=1}^{n} w_{kj}a_{ij} + b_k)$。

#### 反向传播

损失函数通常使用均方误差（MSE），即 $L = \frac{1}{2}\sum_{k=1}^{m} (y_k - \hat{y}_k)^2$。

通过梯度下降算法调整权重和偏置：

$$\Delta w_{ij} = -\alpha \frac{\partial L}{\partial w_{ij}}, \Delta b_j = -\alpha \frac{\partial L}{\partial b_j}$$

其中 $\alpha$ 是学习率。

### 4.2 公式推导过程

假设我们有 $n$ 个输入变量 $x_1, x_2, \ldots, x_n$，以及 $m$ 个输出变量 $y_1, y_2, \ldots, y_m$。假设我们有一个多层神经网络，其中 $l$ 层的输出为 $a_{l} = \sigma(W_{l}a_{l-1} + b_{l})$，其中 $W_{l}$ 是权重矩阵，$b_{l}$ 是偏置向量。

#### 前向传播的推导

$$a_{l} = \sigma(W_{l}a_{l-1} + b_{l})$$

#### 反向传播的推导

假设损失函数为 $L = \frac{1}{2}\sum_{k=1}^{m} (y_k - \hat{y}_k)^2$。

首先，计算输出层的误差：

$$\delta_{k} = \frac{\partial L}{\partial a_{l}} = (y_k - \hat{y}_k) \cdot \sigma'(a_{l})$$

然后，计算隐藏层的误差：

$$\delta_{l-1} = \sum_{k=1}^{m} \delta_{k}W_{l}^{T}\sigma'(a_{l-1})$$

通过反向传播，我们可以得到：

$$\Delta w_{ij} = -\alpha \frac{\partial L}{\partial w_{ij}} = -\alpha \delta_{k}a_{l-1}^j$$

$$\Delta b_j = -\alpha \frac{\partial L}{\partial b_j} = -\alpha \delta_{k}$$

### 4.3 案例分析与讲解

假设我们有一个简单的神经网络，用于对二分类问题进行分类。输入层有 2 个神经元，隐藏层有 3 个神经元，输出层有 1 个神经元。我们使用 sigmoid 函数作为激活函数。

#### 数据集

我们有一个包含 100 个样本的数据集，每个样本有 2 个特征。样本标签为 0 或 1。

#### 模型构建

输入层到隐藏层的权重矩阵为 $W_1$，隐藏层到输出层的权重矩阵为 $W_2$。

#### 模型训练

我们使用均方误差作为损失函数，学习率为 0.1。

#### 训练过程

1. **数据预处理**：将数据标准化为 0 到 1 之间的值。
2. **模型构建**：随机初始化权重矩阵。
3. **模型训练**：进行 1000 次迭代，每次迭代计算损失函数，更新权重矩阵。
4. **模型评估**：在验证集上评估模型的性能。

通过上述过程，我们可以训练出一个能够对二分类问题进行有效分类的神经网络。

### 5.2 源代码详细实现

以下是简单的神经网络实现代码：

```python
import numpy as np
import tensorflow as tf

# 初始化参数
input_size = 2
hidden_size = 3
output_size = 1

# 随机初始化权重和偏置
W1 = np.random.rand(input_size, hidden_size)
b1 = np.random.rand(hidden_size)
W2 = np.random.rand(hidden_size, output_size)
b2 = np.random.rand(output_size)

# 激活函数
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

# 前向传播
def forward(x):
    hidden = sigmoid(np.dot(x, W1) + b1)
    output = sigmoid(np.dot(hidden, W2) + b2)
    return output

# 损失函数
def loss(y, y_pred):
    return 0.5 * np.mean((y - y_pred)**2)

# 反向传播
def backward(x, y, y_pred):
    dW2 = np.dot(hidden.T, (y_pred - y) * sigmoid'(output))
    db2 = (y_pred - y) * sigmoid'(output)
    
    hidden = sigmoid(np.dot(x, W1) + b1)
    dW1 = np.dot(x.T, (np.dot(W2.T, (y_pred - y) * sigmoid'(output))) * sigmoid'(hidden))
    db1 = np.dot((y_pred - y) * sigmoid'(output), sigmoid'(hidden))
    
    return dW1, dW2, db1, db2

# 模型训练
for i in range(1000):
    hidden = sigmoid(np.dot(x, W1) + b1)
    output = sigmoid(np.dot(hidden, W2) + b2)
    loss_val = loss(y, output)
    
    dW1, dW2, db1, db2 = backward(x, y, output)
    
    W1 -= 0.1 * dW1
    b1 -= 0.1 * db1
    W2 -= 0.1 * dW2
    b2 -= 0.1 * db2

# 模型评估
y_pred = forward(x)
accuracy = np.mean((y_pred > 0.5) == y)
print("Accuracy:", accuracy)
```
