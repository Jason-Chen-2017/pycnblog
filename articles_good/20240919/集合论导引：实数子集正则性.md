### 集合论的基本概念

集合是由一组确定的、互不相同的对象组成的整体。这些对象称为集合的元素。集合可以用大括号 `{}` 表示，例如，{1, 2, 3} 表示一个包含三个元素的集合。

集合的基本运算包括并集、交集、补集和差集：

- **并集（Union）**：两个集合 A 和 B 的并集是一个包含所有 A 和 B 中元素的集合，记作 A ∪ B。
- **交集（Intersection）**：两个集合 A 和 B 的交集是一个只包含同时属于 A 和 B 的元素的集合，记作 A ∩ B。
- **补集（Complement）**：集合 A 的补集是一个包含所有不属于 A 的元素的集合，记作 A'。
- **差集（Difference）**：两个集合 A 和 B 的差集是一个包含所有属于 A 但不属于 B 的元素的集合，记作 A \ B。

这些基本概念为理解和分析集合的性质奠定了基础。

### 实数的性质

实数集是数学中最基本的集合之一，包含了所有的有理数和无理数。实数的性质对于集合论和实数子集的研究至关重要。以下是一些关键的实数性质：

- **稠密性（Denseness）**：实数集在数轴上是没有间隙的，任意两个不同实数之间都存在无穷多个其他实数。
- **完备性（Completeness）**：实数集是完备的，即每一个有界实数集合都有一个最值。
- **顺序性（Ordering）**：实数集具有顺序性，任意两个实数之间都可以比较大小。

这些性质使得实数集成为分析问题和证明数学定理的有力工具。

### 4.2 公式推导过程

在实数子集正则性研究中，常用的数学公式包括：

1. **稠密性公式**：用于判断子集是否稠密。
   $$\forall x, y \in S, \exists z \in S \text{ such that } |z - x| < |y - x|$$
2. **连通性公式**：用于判断子集是否连通。
   $$\forall x, y \in S, \exists \text{ a path } P \text{ from } x \text{ to } y \text{ in } S$$
3. **孤立性公式**：用于判断子集是否存在孤立点。
   $$\exists x \in S, \forall y \in S, |x - y| > r \text{ for some } r > 0$$

```python
import numpy as np
import matplotlib.pyplot as plt

# 检测稠密性
def is_dense(subset):
    sorted_subset = sorted(subset)
    for i in range(len(sorted_subset) - 1):
        if sorted_subset[i+1] - sorted_subset[i] > 1:
            return False
    return True

# 检测连通性
def is_connected(subset):
    # 这里使用简单的方法来检查连通性，实际应用中可能需要更复杂的算法
    return len(set(subset)) == 1

# 检测孤立性
def is_isolated(subset):
    for x in subset:
        distance = min([abs(x - y) for y in subset])
        if distance <= 0.1:
            return False
    return True

# 判断正则性
def is_regular(subset):
    return is_dense(subset) and is_connected(subset) and not is_isolated(subset)

# 测试子集
input_subset = [1.1, 2.2, 3.3, 4.4]
result = is_regular(input_subset)
print("子集是正则子集" if result else "子集不是正则子集")

# 绘制子集
if is_dense(input_subset) and is_connected(input_subset):
    plt.plot(input_subset, 'ro')
    plt.axis([0, 5, 0, 5])
    plt.show()
```
