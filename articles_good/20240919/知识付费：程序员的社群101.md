### 5.2 源代码详细实现

以下是知识付费平台的核心代码实现：

```python
from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///knowledge_pay.db'
db = SQLAlchemy(app)

class Course(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(50), nullable=False)
    price = db.Column(db.Float, nullable=False)
    content = db.Column(db.Text, nullable=False)

@app.route('/courses', methods=['GET', 'POST'])
def courses():
    if request.method == 'POST':
        course_data = request.get_json()
        new_course = Course(name=course_data['name'], price=course_data['price'], content=course_data['content'])
        db.session.add(new_course)
        db.session.commit()
        return jsonify({'message': 'Course added successfully.'}), 201

    courses = Course.query.all()
    courses_list = [{'id': course.id, 'name': course.name, 'price': course.price, 'content': course.content} for course in courses]
    return jsonify(courses_list)

if __name__ == '__main__':
    db.create_all()
    app.run(debug=True)
```

### 5.3 代码解读与分析

上述代码实现了一个简单的在线教育平台，主要包括以下功能：

1. **数据库模型**：

   - `Course` 类代表课程信息，包括课程名称、价格和内容。

2. **路由和视图函数**：

   - `/courses` 路由支持两种HTTP方法：GET和POST。
   - POST方法用于添加新的课程信息。
   - GET方法用于获取所有课程信息。

3. **数据库操作**：

   - 使用SQLAlchemy进行数据库操作，确保数据的完整性和一致性。
