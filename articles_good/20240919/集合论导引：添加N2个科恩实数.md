2. **唯一性**：对于任意元素 x，如果 x ∈ A ∪ B，则 x 不能同时属于 A 和 B。

证明如下：

1. **封闭性**：

   - 如果 x ∈ B，则 x ∈ A ∪ B 成立。
   - 因此，对于任意 x ∈ A ∪ B，都有 x ∈ A 或 x ∈ B，证明了封闭性。

2. **唯一性**：
   - 假设 x ∈ A ∪ B，且 x 同时属于 A 和 B。

   - 如果 x ∈ B，则 x 不能同时属于 A，与假设矛盾。
   - 因此，x 不能同时属于 A 和 B，证明了唯一性。

综上所述，我们证明了并集运算满足封闭性和唯一性，即对于任意两个集合 A 和 B，其并集 A ∪ B 满足上述性质。
