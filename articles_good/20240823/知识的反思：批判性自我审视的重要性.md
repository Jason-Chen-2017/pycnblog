### 5.2 源代码详细实现

下面是一个简单的二分查找算法的实现：

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

### 5.3 代码解读与分析

这段代码首先定义了一个二分查找函数，它接收一个有序数组 `arr` 和一个目标值 `target`。通过不断缩小区间，最终找到目标值的位置。

### 5.4 运行结果展示

```python
arr = [1, 3, 5, 7, 9]
target = 5
result = binary_search(arr, target)
print("Element found at index:", result)
```

输出结果为：

```bash
Element found at index: 2
```

这表明目标值 5 在数组中的索引为 2。
