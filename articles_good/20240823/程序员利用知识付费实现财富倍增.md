### 5.2 源代码详细实现

以下是一个简单的线性回归模型的实现：

```python
import numpy as np
import matplotlib.pyplot as plt

def linear_regression(X, Y):
    X_trans = np.append(np.ones((X.shape[0], 1)), X, axis=1)
    beta = np.linalg.inv(X_trans.T.dot(X_trans)).dot(X_trans.T).dot(Y)
    return beta

X = np.array([[1, 2, 3, 4], [1, 2, 3, 5], [1, 2, 4, 5], [2, 3, 4, 5]])
Y = np.array([10, 8, 9, 11])

beta = linear_regression(X, Y)
print("模型参数：", beta)

Y_pred = X.dot(beta)
print("预测得分：", Y_pred)

plt.scatter(X[:, 1], Y)
plt.plot(X[:, 1], Y_pred, color='red')
plt.xlabel('特征1')
plt.ylabel('得分')
plt.show()
```

- 第1行：导入 numpy 库，用于数学运算。
- 第2行：导入 matplotlib.pyplot 库，用于绘图。
- 第4行：定义线性回归模型函数，输入为用户行为特征矩阵 $X$ 和课程推荐得分矩阵 $Y$。
- 第5行：计算 $X$ 的转置。
- 第6行：将常数项 1 添加到 $X$ 的每一行，形成新的矩阵 $X_{trans}$。
- 第7行：计算 $X_{trans}^T \cdot X_{trans}$ 的逆矩阵。
- 第8行：计算 $X_{trans}^T \cdot Y$。
- 第9行：计算模型参数 $\beta$。
- 第10行：使用模型参数预测得分 $Y_{pred}$。
- 第12行：绘制散点图，显示实际得分与预测得分的关系。
