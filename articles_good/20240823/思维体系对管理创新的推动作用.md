import numpy as np
from scipy.optimize import linprog

# 约束条件
A = np.array([[1, 2], [2, 1]])
b = np.array([5, 5])
x0 = np.array([0, 0])

# 目标函数
c = np.array([1, 2])

# 求解线性规划问题
res = linprog(c, A_ub=A, b_ub=b, x0=x0, method='highs')

# 输出结果
print("最优解：", res.x)
print("最优值：", res.fun)
```

3. **定义目标函数**：定义目标函数系数向量$c$。

4. **求解线性规划问题**：使用linprog函数求解线性规划问题。

5. **输出结果**：输出最优解和最优值。
