### 5.1 开发环境搭建

### 5.2 源代码详细实现

以下是一个简单的Python代码实例，用于爬取知识付费平台上的课程信息和用户评价：

```python
import requests
from bs4 import BeautifulSoup

def get_course_info(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.content, 'html.parser')
    
    # 获取课程名称
    course_name = soup.find('h1', class_='course-title').text
    
    # 获取课程简介
    course_intro = soup.find('div', class_='course-description').text
    
    # 获取课程价格
    course_price = soup.find('span', class_='price').text
    
    # 获取用户评价
    reviews = soup.find_all('div', class_='review')
    review_texts = [review.find('p', class_='review-text').text for review in reviews]
    
    return {
        'course_name': course_name,
        'course_intro': course_intro,
        'course_price': course_price,
        'review_texts': review_texts
    }

# 获取课程信息
url = 'https://example.com/course/12345'
course_info = get_course_info(url)

# 打印课程信息
print(course_info)
```

### 5.3 代码解读与分析

该代码实例主要实现了以下功能：

1. **发送HTTP请求**：使用requests库向知识付费平台发送GET请求，获取课程页面内容。
2. **解析HTML内容**：使用BeautifulSoup库解析HTTP响应内容，提取课程名称、简介、价格和用户评价等信息。
3. **存储数据**：将提取到的课程信息存储为一个字典，以便后续处理。
