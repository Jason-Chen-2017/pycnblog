## 4. 数学模型和公式详细讲解举例说明

### 4.1 线性回归模型

线性回归模型的公式如下：

$$
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n
$$

其中：

*   $y$ 是目标变量。
*   $x_i$ 是特征变量。
*   $\beta_i$ 是模型参数，表示每个特征对目标变量的影响。

### 4.2 逻辑回归模型

逻辑回归模型使用 sigmoid 函数将线性组合转换为概率：

$$
P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + ... + \beta_n x_n)}}
$$

其中：

*   $P(y=1|x)$ 是样本属于类别 1 的概率。
*   $x_i$ 是特征变量。
*   $\beta_i$ 是模型参数。 

## 5. 项目实践：代码实例和详细解释说明

### 5.1 房价预测

**目标：** 使用线性回归模型预测房价。

**代码示例：**

```python
import pandas as pd
from sklearn.linear_model import LinearRegression

# 加载数据
data = pd.read_csv('house_prices.csv')

# 特征选择
features = ['area', 'bedrooms', 'bathrooms']
X = data[features]
y = data['price']

# 创建模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 预测房价
new_data = [[1500, 3, 2]]
predicted_price = model.predict(new_data)

print(f'Predicted price: ${predicted_price[0]:.2f}')
```

**解释：**

1.  导入必要的库。
2.  加载房价数据集。
3.  选择特征变量和目标变量。
4.  创建线性回归模型实例。
5.  使用训练数据拟合模型。
6.  使用新数据预测房价。

### 5.2 垃圾邮件分类

**目标：** 使用逻辑回归模型对邮件进行垃圾邮件分类。

**代码示例：**

```python
import pandas as pd
from sklearn.linear_model import LogisticRegression
from sklearn.feature_extraction.text import CountVectorizer

# 加载数据
data = pd.read_csv('emails.csv')

# 文本特征提取
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(data['text'])
y = data['spam']

# 创建模型
model = LogisticRegression()

# 训练模型
model.fit(X, y)

# 预测邮件是否为垃圾邮件
new_email = ['Free money!']
predicted_spam = model.predict(vectorizer.transform(new_email))

print(f'Spam: {predicted_spam[0]}')
```

**解释：**

1.  导入必要的库。
2.  加载邮件数据集。
3.  使用 CountVectorizer 将文本数据转换为数值特征。
4.  创建逻辑回归模型实例。
5.  使用训练数据拟合模型。
6.  使用新邮件预测是否为垃圾邮件。
