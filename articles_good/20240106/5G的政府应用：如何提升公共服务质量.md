```python
import random

def classify_task(task_type, task_params):
    if task_type == 'high_compute':
        return task_params['compute'] > 50
    elif task_type == 'low_compute':
        return task_params['compute'] > 20
    else:
        return task_params['compute'] > 10

def task_dispatch(task, edge_node):
    if task['type'] == 'high_compute':
        return edge_node * 0.6
    elif task['type'] == 'low_compute':
        return edge_node * 0.4
    else:
        return edge_node * 0.2

def result_aggregation(edge_nodes, task_results):
    for edge_node, result in zip(edge_nodes, task_results):
        print(f'{edge_node}, {result}')

def main():
    task = {'type': 'high_compute', 'compute': 70}
    edge_nodes = [1, 2, 3]
    task_results = [task_dispatch(task, node) for node in edge_nodes]
    result_aggregation(edge_nodes, task_results)

if __name__ == '__main__':
    main()
```

在这个代码实例中，我们首先定义了一个`classify_task`函数，用于根据任务类型和参数将任务分发到边缘网络上。然后定义了一个`task_dispatch`函数，用于根据任务类别为边缘节点分配计算任务。最后，在`main`函数中，我们通过一个示例任务和边缘节点列表来演示边缘计算技术的具体操作。
