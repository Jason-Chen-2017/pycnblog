## 4.1 基于规则的监测与预警代码实例

```python
import re

# 定义网络安全事件的规则
rules = [
    {'pattern': r'login_failed', 'action': 'send_alert'},
    {'pattern': r'login_success', 'action': 'ignore'},
]

# 监测网络流量
def monitor(traffic):
    for rule in rules:
        if re.search(rule['pattern'], traffic):
            rule['action'](traffic)

# 发送警告
def send_alert(traffic):
    print(f'Alert: {traffic}')

# 忽略事件
def ignore(traffic):
    pass

# 测试代码
traffic = 'someone failed to login'
monitor(traffic)
```

在这个代码实例中，我们首先定义了一些网络安全事件的规则，包括登录失败和登录成功。然后，我们定义了一个监测网络流量的函数`monitor`，该函数会根据规则匹配结果触发对应的处理措施。最后，我们测试了代码，当监测到登录失败时，会触发发送警告的处理措施。
