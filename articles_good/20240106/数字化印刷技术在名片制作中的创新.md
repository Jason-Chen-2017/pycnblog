## 4.2 数字打印的代码实例
```python
from PIL import Image
from PIL import ImageDraw
from PIL import ImageFont

def draw_rectangle(image, x, y, width, height):
    draw = ImageDraw.Draw(image)
    draw.rectangle([(x, y), (x + width, y + height)], fill=255)

def draw_text(image, x, y, text):
    draw = ImageDraw.Draw(image)
    font = ImageFont.truetype("arial.ttf", 12)
    draw.text((x, y), text, font=font, fill=255)

def main():
    image = Image.new("RGB", (400, 600), color="white")
    draw_rectangle(image, 100, 100, 200, 300)
    draw_text(image, 150, 200, "John Doe")
    draw_text(image, 150, 250, "Software Engineer")
    draw_text(image, 150, 300, "1234567890")

if __name__ == "__main__":
    main()
```
这个代码实例使用了 PIL 库来创建一个 PNG 名片。首先，我们定义了一个名片的大小（400x600 像素），然后使用 `draw_rectangle` 函数绘制了一个矩形，再使用 `draw_text` 函数绘制了名片上的文本。最后，我们使用 `image.save()` 方法保存了名片。
