## 2. 核心概念与联系

### 2.1 类模板

类模板是C++中的一种泛型编程工具，它允许我们定义一种类的通用版本，并在编译时根据实际类型生成具体的类实现。类模板可以提高代码的可重用性和可维护性，同时减少编写和维护代码的时间和成本。

类模板定义如下：

```cpp
template <typename T>
class MyClass {
public:
    T value;
    MyClass(T val) : value(val) {}
    T getValue() const { return value; }
};
```

在上述示例中，`MyClass`是一个泛型类，它可以接受任何类型的实例化。

### 2.2 类模板特化

类模板特化是一种在特定情况下为特定类型提供特定实现的方法。通过类模板特化，我们可以为特定类型提供更具体的行为，从而实现更高效的代码和更好的性能。

类模板特化定义如下：

```cpp
template <typename T>
class MyClass {
public:
    T value;
    MyClass(T val) : value(val) {}
    T getValue() const { return value; }
};

// 特化版本
template <>
class MyClass<int> {
public:
    int value;
    MyClass(int val) : value(val) {}
    int getValue() const { return value; }
};
```

在上述示例中，我们为`int`类型提供了一个特化版本的`MyClass`。这个特化版本具有与原始模板不同的实现。

### 2.3 类模板实例化

类模板实例化是在特定类型上实例化类模板的过程。通过类模板实例化，我们可以为特定类型生成具体的类实现。

类模板实例化定义如下：

```cpp
MyClass<int> myIntClass;
```

在上述示例中，我们实例化了一个`int`类型的`MyClass`。这将生成一个具体的类实现，其中`T`被替换为`int`。

### 4.1 代码实例

```cpp
#include <iostream>
#include <vector>

template <typename T>
class MyClass {
public:
    T value;
    MyClass(T val) : value(val) {}
    T getValue() const { return value; }
};

// 特化版本
template <>
class MyClass<int> {
public:
    int value;
    MyClass(int val) : value(val) {}

};

int main() {
    MyClass<int> myIntClass(42);
    std::cout << "MyClass<int> value: " << myIntClass.getValue() << std::endl;

    MyClass<double> myDoubleClass(3.14);
    std::cout << "MyClass<double> value: " << myDoubleClass.getValue() << std::endl;

    return 0;
}
```

在上述示例中，我们定义了一个泛型类`MyClass`，并为`int`类型提供了一个特化版本。在`main`函数中，我们实例化了`int`和`double`类型的`MyClass`，并输出了它们的值。

1. 我们定义了一个泛型类`MyClass`，它可以接受任何类型的实例化。
2. 我们为`int`类型提供了一个特化版本的`MyClass`，它具有与原始模板不同的实现。
3. 我们实例化了`int`和`double`类型的`MyClass`，并输出了它们的值。
