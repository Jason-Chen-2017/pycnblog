### 4. 医学优化

在医学优化中，我们可以使用粒子群优化算法来优化治疗方案。以下是一个简单的粒子群优化算法的代码实例：

```python
import numpy as np

def f(x):
    return x**2

def pso(dim, max_iter, w, c1, c2, lr):
    particles = np.random.uniform(low=-10, high=10, size=(dim, 2))
    velocities = np.zeros((dim, 2))
    best_positions = particles.copy()
    best_fitness = np.inf

    for _ in range(max_iter):
        r1, r2 = np.random.rand(dim)
        velocities += w * r1 * (best_positions - particles) + c1 * r2 * (particles[:, 0] - particles[:, 1]) + c2 * r2 * (particles[:, 1] - particles[:, 0])
        particles += velocities
        fitness = f(particles)

        if fitness < best_fitness:
            best_fitness = fitness
            best_positions = particles.copy()

    return best_positions, best_fitness
```
