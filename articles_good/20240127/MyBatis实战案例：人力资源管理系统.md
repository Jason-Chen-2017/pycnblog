## 4. 具体最佳实践：代码实例和详细解释说明

在本节中，我们将通过一个简单的代码实例来说明MyBatis实战案例：人力资源管理系统。

### 4.1 创建数据库表

首先，我们需要创建一个员工信息表：

```sql
CREATE TABLE employee (
    id INT PRIMARY KEY,
    name VARCHAR(255),
    age INT,
    gender CHAR(1),
    position VARCHAR(255),
    salary DECIMAL(10,2)
);
```

### 4.2 创建SQL映射文件

接下来，我们需要创建一个SQL映射文件，用于存储员工信息的查询、添加、修改、删除操作：

```xml
<mapper namespace="com.example.hr.mapper.EmployeeMapper">
    <select id="selectAll" resultType="com.example.hr.model.Employee">
        SELECT * FROM employee;
    </select>
    <insert id="insert" parameterType="com.example.hr.model.Employee">
        INSERT INTO employee (name, age, gender, position, salary)
        VALUES (#{name}, #{age}, #{gender}, #{position}, #{salary});
    </insert>
    <update id="update" parameterType="com.example.hr.model.Employee">
        UPDATE employee
        SET name = #{name}, age = #{age}, gender = #{gender}, position = #{position}, salary = #{salary}
        WHERE id = #{id};
    </update>
    <delete id="delete" parameterType="int">
        DELETE FROM employee WHERE id = #{id};
    </delete>
</mapper>
```

### 4.3 创建Java对象

接下来，我们需要创建一个Java对象来表示员工信息：

```java
package com.example.hr.model;

public class Employee {
    private int id;
    private String name;
    private int age;
    private char gender;
    private String position;
    private double salary;

    // getter and setter methods
}
```

### 4.4 使用MyBatis操作数据库

最后，我们需要使用MyBatis操作数据库：

```java
package com.example.hr.service;

import com.example.hr.mapper.EmployeeMapper;
import com.example.hr.model.Employee;
import org.apache.ibatis.session.SqlSession;
import org.apache.ibatis.session.SqlSessionFactory;

import java.util.List;

public class EmployeeService {
    private SqlSessionFactory sqlSessionFactory;
    private EmployeeMapper employeeMapper;

    public EmployeeService(SqlSessionFactory sqlSessionFactory) {
        this.sqlSessionFactory = sqlSessionFactory;
        this.employeeMapper = sqlSessionFactory.openSession().getMapper(EmployeeMapper.class);
    }

    public List<Employee> selectAll() {
        return employeeMapper.selectAll();
    }

    public void insert(Employee employee) {
        employeeMapper.insert(employee);
    }

    public void update(Employee employee) {
        employeeMapper.update(employee);
    }

    public void delete(int id) {
        employeeMapper.delete(id);
    }
}
```
