```python
class User:
    def __init__(self, username, password):
        self.username = username
        self.password = password

    def __init__(self, username, password):
        super().__init__(username, password)
        self.admin_privilege = True

class UserManager:
    def __init__(self):
        self.users = []

    def add_user(self, user):
        self.users.append(user)

    def remove_user(self, user):
        self.users.remove(user)

# 使用最小权限原则，普通用户和管理员有不同的权限

user_manager.add_user(user1)
user_manager.add_user(admin)
```

在这个例子中，我们定义了一个`User`类和一个`Admin`类，其中`Admin`类继承自`User`类。`Admin`类具有`admin_privilege`属性，表示该用户是管理员。`UserManager`类负责管理用户，并且只有管理员才能添加和删除用户。这样，我们就实现了最小权限原则。

```python
class Database:
    def __init__(self):
        self.data = {}

    def put(self, key, value):
        self.data[key] = value

    def get(self, key):
        return self.data.get(key)

class DatabaseManager:
    def __init__(self):
        self.databases = []

    def add_database(self, database):
        self.databases.append(database)

    def remove_database(self, database):
        self.databases.remove(database)

database_manager.add_database(database)
```

在这个例子中，我们定义了一个`Database`类和一个`DatabaseManager`类。`Database`类负责管理数据，`DatabaseManager`类负责管理数据库。这样，我们就实现了分层安全性。
