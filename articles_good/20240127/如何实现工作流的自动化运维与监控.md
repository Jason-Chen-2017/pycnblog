## 4. 具体最佳实践：代码实例和详细解释说明

以下是一个简单的 Python 代码实例，用于实现工作流自动化运维与监控：

```python
import time
import threading

def task1():
    print("任务1开始执行")
    time.sleep(2)
    print("任务1执行完成")

def task2():
    print("任务2开始执行")
    time.sleep(3)
    print("任务2执行完成")

def monitor():
    while True:
        print("监控任务执行中")
        time.sleep(5)

if __name__ == "__main__":
    t1 = threading.Thread(target=task1)
    t2 = threading.Thread(target=task2)
    t3 = threading.Thread(target=monitor)

    t1.start()
    t2.start()
    t3.start()

    t1.join()
    t2.join()
```

在这个代码实例中，我们定义了三个任务：任务1、任务2和监控任务。任务1和任务2分别使用线程进行并行执行，监控任务使用线程进行实时监控。在主程序中，我们启动了三个线程，并使用 join 方法等待任务执行完成。
