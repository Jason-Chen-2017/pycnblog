### 4.1 性能测试的代码实例

以下是一个简单的性能测试代码实例：

```python
import time
import random

def test_function():
    time.sleep(random.uniform(0.1, 0.5))

if __name__ == '__main__':
    start_time = time.time()
    for i in range(1000):
        test_function()
    end_time = time.time()
    total_time = end_time - start_time
    average_time = total_time / 1000
    print(f"Average response time: {average_time:.2f}s")
```

### 4.2 压力测试的代码实例

以下是一个简单的压力测试代码实例：

```python
import time
import random
import threading

def generate_load():
    for i in range(100):
        time.sleep(random.uniform(0.1, 0.5))

def test_function():
    time.sleep(random.uniform(0.1, 0.5))

if __name__ == '__main__':
    threads = []
    for i in range(1000):
        t = threading.Thread(target=generate_load)
        threads.append(t)
        t.start()
    for t in threads:
        t.join()
    start_time = time.time()
    for i in range(1000):
        test_function()
    end_time = time.time()
    total_time = end_time - start_time
    print(f"Total response time: {total_time:.2f}s")
```

压力测试的代码实例中，我们定义了一个名为`generate_load`的函数，该函数模拟了一个负载的生成。然后，我们使用多线程来执行该函数1000次，并记录总的执行时间。最后，我们计算总的响应时间。
