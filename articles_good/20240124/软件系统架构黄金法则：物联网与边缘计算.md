### 4.1 代码实例

以下是一个简单的物联网设备之间通信和数据交换的代码实例：

```python
import threading

class Device:
    def __init__(self, name, ip_address):
        self.name = name
        self.ip_address = ip_address
        self.data = None

    def send_data(self, data):
        self.data = data
        print(f"{self.name} 发送数据: {self.data}")

    def receive_data(self):
        while True:
            data = input(f"{self.name} 接收数据: ")
            print(f"{self.name} 接收到数据: {data}")

def main():
    device1 = Device("Device1", "192.168.1.1")
    device2 = Device("Device2", "192.168.1.2")

    device1_thread = threading.Thread(target=device1.receive_data)
    device2_thread = threading.Thread(target=device2.send_data, args=(device1,))

    device1_thread.start()
    device2_thread.start()

if __name__ == "__main__":
    main()
```

### 4.2 详细解释说明

在这个代码实例中，我们创建了两个物联网设备，分别是 `Device1` 和 `Device2`。`Device1` 的职责是接收数据，而 `Device2` 的职责是发送数据。这两个设备之间通过网络进行通信和数据交换。

在主函数中，我们创建了两个线程，分别是 `device1_thread` 和 `device2_thread`。`device1_thread` 负责执行 `device1` 的 `receive_data` 方法，而 `device2_thread` 负责执行 `device2` 的 `send_data` 方法。这两个线程可以同时运行，从而实现设备之间的通信和数据交换。
