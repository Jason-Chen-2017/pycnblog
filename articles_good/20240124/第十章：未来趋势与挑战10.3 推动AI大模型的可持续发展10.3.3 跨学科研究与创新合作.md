## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 分布式计算实例

在实际应用中，我们可以使用Python的`multiprocessing`模块来实现分布式计算。以下是一个简单的示例：

```python
import multiprocessing
import time

def worker(name):
    print(f"{name} is working...")
    time.sleep(2)
    return f"{name} has finished working."

if __name__ == "__main__":
    processes = []
    for i in range(5):
        p = multiprocessing.Process(target=worker, args=(f"worker_{i}",))
        processes.append(p)
        p.start()

    for p in processes:
        p.join()
```

在这个示例中，我们创建了5个工作进程，并在不同的进程中执行不同的任务。通过这种方式，我们可以降低计算资源的消耗。

### 4.2 能源消耗优化实例

在实际应用中，我们可以使用Python的`psutil`库来监控系统资源，并根据资源状况调整任务执行策略。以下是一个简单的示例：

```python
import psutil
import time

def task():
    while True:
        time.sleep(1)

if __name__ == "__main__":
    cpu_percent = psutil.cpu_percent(interval=1)
    if cpu_percent < 50:
        task()
    else:
        print("CPU usage is too high, skip the task.")
```

在这个示例中，我们监控系统的CPU使用率，如果CPU使用率低于50%，则执行任务。通过这种方式，我们可以降低能源消耗。

### 4.3 数据安全与隐私保护实例

在实际应用中，我们可以使用Python的`cryptography`库来实现数据加密。以下是一个简单的示例：

```python
from cryptography.fernet import Fernet

def generate_key():
    return Fernet.generate_key()

def encrypt_message(key, message):
    f = Fernet(key)
    return f.encrypt(message.encode())

def decrypt_message(key, encrypted_message):
    f = Fernet(key)
    return f.decrypt(encrypted_message).decode()

if __name__ == "__main__":
    key = generate_key()
    message = "Hello, World!"
    encrypted_message = encrypt_message(key, message)
    print(f"Encrypted message: {encrypted_message}")
    decrypted_message = decrypt_message(key, encrypted_message)
    print(f"Decrypted message: {decrypted_message}")
```

在这个示例中，我们使用`cryptography`库实现了数据加密和解密。通过这种方式，我们可以保障数据安全与隐私。
