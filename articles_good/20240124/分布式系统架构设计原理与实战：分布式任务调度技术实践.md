## 5. 具体最佳实践：代码实例和详细解释说明

具体最佳实践的代码实例如下：

```python
import threading
import queue

def worker(task_queue):
    while not task_queue.empty():
        task = task_queue.get()
        # 执行任务
        result = task()
        # 存储结果
        result_queue.put(result)
        task_queue.task_done()

def main():
    task_queue = queue.Queue()
    result_queue = queue.Queue()
    threads = []

    # 添加任务
    for i in range(10):
        task_queue.put(lambda: f"task_{i}")

    # 启动工作线程
    for _ in range(4):
        t = threading.Thread(target=worker, args=(task_queue,))
        t.start()
        threads.append(t)

    # 等待所有任务完成
    task_queue.join()

    # 关闭工作线程
    for t in threads:
        t.join()

    # 获取结果
    while not result_queue.empty():
        print(result_queue.get())

if __name__ == "__main__":
    main()
```

代码实例详细解释说明如下：

- 定义一个`worker`函数，用于执行任务。函数接收一个任务队列作为参数，从队列中取出任务，执行任务，并将结果存储到结果队列中。
- 定义一个`main`函数，用于添加任务、启动工作线程、等待所有任务完成、关闭工作线程和获取结果。
- 使用`threading`模块创建多个工作线程，每个线程执行`worker`函数。
- 使用`queue`模块创建任务队列和结果队列，用于存储任务和结果。
