## 2. 核心概念与联系

在讨论流程图的安全与防护之前，我们首先需要了解一下流程图的核心概念。流程图是一种有向图，用于描述算法或程序的执行流程。流程图的主要组成元素包括：

- 流程框：表示程序或算法的一个或多个步骤。
- 处理符号：表示程序或算法的执行操作。
- 连线：表示程序或算法的执行顺序。

流程图的安全与防护主要关注以下几个方面：

- 数据安全：确保流程图中涉及的数据不被恶意访问、篡改或泄露。

# 生成密钥

# 生成初始化向量
iv = get_random_bytes(16)

# 生成明文

# 生成密文

ciphertext = cipher.encrypt(pad(plaintext, AES.block_size))

# 生成密钥和初始化向量

# 生成密文
