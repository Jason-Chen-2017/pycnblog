### 4.1 密码学实践

在软件架构实战中，开发者可以使用多种密码学技术，例如AES、RSA、SHA等。以下是一个AES加密和解密的代码实例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad

key = get_random_bytes(16)
cipher = AES.new(key, AES.MODE_CBC)

plaintext = b"Hello, World!"
ciphertext = cipher.encrypt(pad(plaintext, AES.block_size))

cipher.close()

decrypted_text = unpad(cipher.decrypt(ciphertext), AES.block_size)
print(decrypted_text.decode())
```

### 4.2 加密实践

在软件架构实战中，开发者可以使用多种加密技术，例如AES、RSA、SHA等。以下是一个RSA加密和解密的代码实例：

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

key = RSA.generate(2048)
public_key = key.publickey().export_key()
private_key = key.export_key()

cipher = PKCS1_OAEP.new(public_key)
plaintext = b"Hello, World!"
ciphertext = cipher.encrypt(plaintext)

cipher.close()

decipher = PKCS1_OAEP.new(private_key)
decrypted_text = decipher.decrypt(ciphertext)

print(decrypted_text.decode())
```

### 4.4 授权实践

在软件架构实战中，开发者可以使用多种授权技术，例如角色分配、权限分配、访问控制等。以下是一个基于角色分配的授权的代码实例：

```python
class User:
    def __init__(self, username, password):
        self.username = username
        self.password = password

class Role:
    def __init__(self, name, permissions):
        self.name = name
        self.permissions = permissions

class Permission:
    def __init__(self, name):
        self.name = name

class Authorization:
    def __init__(self, user, role):
        self.user = user
        self.role = role

    def has_permission(self, permission):
        return permission in self.role.permissions

user = User("admin", "password")
role = Role("admin", [Permission("read"), Permission("write")])
authorization = Authorization(user, role)

print(authorization.has_permission(Permission("read")))
print(authorization.has_permission(Permission("write")))
```
