## 1. 背景介绍

多边形是二维空间中的一种基本的几何形状，它由一组连续的点组成，这些点按顺序形成一个闭合的多边形。多边形可以用一组顶点、边和面来表示。在计算机图形学中，多边形是用来表示物体形状的基本单元，而多边形包含关系和多边形交叉是判断两个多边形关系的重要方法。

## 2. 核心概念与联系

### 2.1 多边形包含关系

多边形包含关系是指一个多边形内部包含另一个多边形的关系。例如，一个矩形内部包含一个三角形，则说明矩形包含三角形。多边形包含关系有三种情况：

- 完全包含：一个多边形完全包含在另一个多边形内部。
- 部分包含：一个多边形部分包含在另一个多边形内部。
- 不包含：一个多边形不包含在另一个多边形内部。

### 2.2 多边形交叉

多边形交叉是指两个多边形的边相交的情况。例如，两个矩形的边相交，则说明这两个矩形相交。多边形交叉有三种情况：

- 完全交叉：两个多边形完全相交，即边相交且顶点相交。
- 部分交叉：两个多边形部分相交，即边相交但顶点不相交。
- 不交叉：两个多边形不相交，即边不相交。

### 3.1 多边形包含关系算法原理

- 线段在多边形内部的判断：通过判断给定线段是否在多边形内部，从而判断多边形包含关系。

多边形交叉的算法原理是通过判断两个多边形的边是否相交来实现的。这可以通过以下几种方法实现：

- 多边形边的关系判断：通过判断两个多边形的边关系，从而判断多边形交叉。

    x, y = point
    flag = False
    for i in range(n):
        x1, y1 = polygon[i]
        x2, y2 = polygon[(i + 1) % n]
        if y1 >= y and y2 < y:
            if x <= x1 and x >= x2 or x1 >= x and x2 < x:
