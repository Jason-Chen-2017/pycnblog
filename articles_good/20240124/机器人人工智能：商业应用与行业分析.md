## 4. 数学模型公式详细讲解

### 4.1 线性回归

线性回归是一种用于预测连续变量的机器学习算法，可以用以下数学模型公式表示：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

其中，$y$ 是预测值，$x_1, x_2, ..., x_n$ 是输入变量，$\beta_0, \beta_1, ..., \beta_n$ 是参数，$\epsilon$ 是误差。

### 4.2 逻辑回归

逻辑回归是一种用于预测类别变量的机器学习算法，可以用以下数学模型公式表示：

$$
P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n)}}
$$

其中，$P(y=1|x)$ 是预测概率，$x_1, x_2, ..., x_n$ 是输入变量，$\beta_0, \beta_1, ..., \beta_n$ 是参数。

### 4.3 神经网络

神经网络是一种用于预测连续变量或类别变量的深度学习算法，可以用以下数学模型公式表示：

$$
y = f(Wx + b)
$$

其中，$y$ 是预测值，$x$ 是输入变量，$W$ 是权重矩阵，$b$ 是偏置向量，$f$ 是激活函数。

## 5. 具体最佳实践：代码实例和详细解释说明

### 5.1 机器学习实例：线性回归

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据生成
X = np.random.rand(100, 1)
y = 2 * X + 1 + np.random.randn(100, 1) * 0.1

# 模型训练
model = LinearRegression()
model.fit(X, y)

# 预测
X_new = np.array([[0.5]])
y_pred = model.predict(X_new)
print(y_pred)
```

### 5.2 深度学习实例：神经网络

```python
import tensorflow as tf

# 数据生成
X = np.random.rand(100, 1)
y = 2 * X + 1 + np.random.randn(100, 1) * 0.1

# 模型定义
model = tf.keras.Sequential([
    tf.keras.layers.Dense(1, input_shape=(1,))
])

# 编译
model.compile(optimizer='adam', loss='mean_squared_error')

# 训练
model.fit(X, y, epochs=100)

# 预测
X_new = np.array([[0.5]])
y_pred = model.predict(X_new)
print(y_pred)
```
