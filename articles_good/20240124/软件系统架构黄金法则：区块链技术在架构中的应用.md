## 2. 核心概念与联系

区块链技术的核心概念包括：

- 分布式存储：区块链中的数据是通过多个节点来存储的，这样可以提高数据的安全性和可靠性。
- 去中心化：区块链中的节点是相互联系的，没有一个中心节点来控制整个网络。
- 加密技术：区块链中的数据通过加密技术来保护，确保数据的安全性。
- 共识算法：区块链中的节点通过共识算法来达成一致，确保数据的一致性和完整性。

在软件系统架构中，这些核心概念可以帮助我们构建一个更安全、更透明、更可靠的系统。

import hashlib
import time

class Block:
    def __init__(self, index, prev_hash, timestamp, data, hash):
        self.index = index
        self.prev_hash = prev_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(0, "0", time.time(), "Genesis Block", self.calculate_hash("Genesis Block"))

    def add_block(self, data):
        index = len(self.chain)
        prev_hash = self.chain[-1].hash
        timestamp = time.time()
        hash = self.calculate_hash(str(index) + str(prev_hash) + str(timestamp) + str(data))
        new_block = Block(index, prev_hash, timestamp, data, hash)
        self.chain.append(new_block)
        return new_block

    def calculate_hash(self, data):
        return hashlib.sha256(data.encode()).hexdigest()

    def is_valid(self):
        for i in range(1, len(self.chain)):
            current_block = self.chain[i]
            prev_block = self.chain[i-1]
            if current_block.hash != self.calculate_hash(str(current_block.index) + str(prev_block.hash) + str(current_block.timestamp) + str(current_block.data)):
                return False
        return True

# 创建区块链
blockchain = Blockchain()

# 添加区块
blockchain.add_block("Block 1")
blockchain.add_block("Block 2")
blockchain.add_block("Block 3")

# 验证区块链
print(blockchain.is_valid())
