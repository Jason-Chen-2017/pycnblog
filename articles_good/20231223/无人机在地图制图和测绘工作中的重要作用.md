## 3.1 数据采集
### 3.1.1 影像数据采集
无人机搭载相机采集影像数据是无人机在地图制图和测绘工作中最常见的数据采集方式。影像数据采集的主要步骤如下：

1. 确定采集区域和采集时间。

2. 设定相机参数，如焦距、快门速度、ISO速度等。

3. 通过无人机控制系统控制相机进行采集。

影像数据采集的数学模型公式为：
$$
I(x, y) = K \cdot \tau(x, y) \cdot A(x, y) + n(x, y)
$$
其中，$I(x, y)$ 是采集到的影像灰度值；$K$ 是照相机光圈常数；$\tau(x, y)$ 是场景亮度函数；$A(x, y)$ 是照相机传感器响应函数；$n(x, y)$ 是噪声。

### 3.1.2 激光雷达数据采集

3. 通过无人机控制系统控制激光雷达进行采集。

$$
d = \frac{c \cdot t}{2}

from PIL import Image

# 加载处理后的影像数据

# 像元统计
pixels = image.getdata()
hist = [0] * 256
for pixel in pixels:
    hist[pixel[0]] += 1
print(hist)

# 纵向统计
rows, cols = image.size
row_stats = [0] * rows
for i in range(rows):
    row_stats[i] = sum(pixel[0] for pixel in image.getdata(box=(0, i, cols, i+1))) / cols
print(row_stats)

# 空间统计
row_stats = [0] * rows
col_stats = [0] * cols
for i in range(rows):
    for j in range(cols):
        row_stats[i] += image.getpixel((j, i))[0]
        col_stats[j] += image.getpixel((j, i))[0]
print(row_stats)
print(col_stats)
```

import fiona

# 创建地图空间模型
driver = "ESRI Shapefile"

    "geometry": "Polygon",
    "properties": {
        "name": "string",
        "area": "float"
    }
}
with fiona.open("map.shp", "w", driver=driver, schema=schema) as c:
    for i in range(rows):
        for j in range(cols):
            area = (i + 1) * (j + 1)
            geometry = {"type": "Polygon", "coordinates": [([(j, i), (j+1, i), (j+1, i+1), (j, i+1), (j, i)])]}
            properties = {"name": "area_{}_{}".format(i, j), "area": area}
            c.write({"geometry": geometry, "properties": properties})
```

    }
