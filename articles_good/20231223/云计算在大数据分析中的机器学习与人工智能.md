### 3.3 数学模型公式

#### 3.3.1 线性回归

线性回归是一种用于预测连续型变量的方法，它假设变量之间存在线性关系。线性回归的数学模型公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是目标变量，$x_1, x_2, \cdots, x_n$ 是预测变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。

#### 3.3.2 逻辑回归

逻辑回归是一种用于预测二值型变量的方法，它假设变量之间存在逻辑关系。逻辑回归的数学模型公式为：

$$
P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \cdots - \beta_nx_n}}
$$

其中，$y$ 是目标变量，$x_1, x_2, \cdots, x_n$ 是预测变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

#### 3.3.3 决策树

决策树是一种用于预测类别型变量的方法，它将数据空间划分为多个区域，每个区域对应一个类别。决策树的数学模型公式为：

$$
\text{if } x_1 \text{ is } A_1 \text{ then } y = C_1 \\
\text{else if } x_2 \text{ is } A_2 \text{ then } y = C_2 \\
\cdots \\
\text{else if } x_n \text{ is } A_n \text{ then } y = C_n
$$

其中，$x_1, x_2, \cdots, x_n$ 是预测变量，$A_1, A_2, \cdots, A_n$ 是条件表达式，$C_1, C_2, \cdots, C_n$ 是类别。

#### 3.3.4 支持向量机

支持向量机是一种用于解决线性不可分问题的方法，它通过寻找支持向量来将数据分类。支持向量机的数学模型公式为：

$$
\min_{w,b} \frac{1}{2}w^2 \\
\text{s.t.} \ y_i(w \cdot x_i + b) \geq 1, \forall i
$$

其中，$w$ 是权重向量，$b$ 是偏置项，$x_i$ 是输入向量，$y_i$ 是标签。

#### 3.3.5 随机森林

随机森林是一种用于预测连续型或类别型变量的方法，它通过构建多个决策树来进行预测。随机森林的数学模型公式为：

$$
\hat{y} = \frac{1}{K} \sum_{k=1}^K f_k(x)
$$

其中，$\hat{y}$ 是预测值，$K$ 是决策树的数量，$f_k(x)$ 是第$k$个决策树的预测值。
