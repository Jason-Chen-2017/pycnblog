## 3.3 数学模型公式详细讲解
### 3.3.1 描述性分析
1. 平均值（Mean）：$$ \bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_{i} $$
2. 中位数（Median）：$$ x_{med} = \left\{ \begin{array}{ll} \frac{x_{(n+1)/2} + x_{(n+2)/2}}{2} & \text{if } n \text{ is even} \\ x_{(n+1)/2} & \text{if } n \text{ is odd} \end{array} \right. $$
3. 方差（Variance）：$$ \sigma^{2} = \frac{1}{n} \sum_{i=1}^{n} (x_{i} - \bar{x})^{2} $$
4. 标准差（Standard Deviation）：$$ \sigma = \sqrt{\sigma^{2}} $$

### 3.3.2 关联分析
1. 相关系数（Correlation Coefficient）：$$ r = \frac{\sum_{i=1}^{n} (x_{i} - \bar{x})(y_{i} - \bar{y})}{\sqrt{\sum_{i=1}^{n} (x_{i} - \bar{x})^{2}} \sqrt{\sum_{i=1}^{n} (y_{i} - \bar{y})^{2}}} $$

### 3.3.3 聚类分析
1. 欧式距离（Euclidean Distance）：$$ d(x, y) = \sqrt{\sum_{i=1}^{n} (x_{i} - y_{i})^{2}} $$
2. 曼哈顿距离（Manhattan Distance）：$$ d(x, y) = \sum_{i=1}^{n} |x_{i} - y_{i}| $$
3. 闵可夫斯基距离（Minkowski Distance）：$$ d(x, y) = (\sum_{i=1}^{n} |x_{i} - y_{i}|^{p})^{\frac{1}{p}} $$

### 3.3.4 决策树
1. 信息熵（Information Entropy）：$$ H(S) = -\sum_{i=1}^{n} p_{i} \log_{2} p_{i} $$
2. 条件信息熵（Conditional Information Entropy）：$$ H(S|A) = -\sum_{v=1}^{m} p_{v} \log_{2} p_{v} $$
3. 信息增益（Information Gain）：$$ G(S, A) = H(S) - H(S|A) $$

### 3.3.5 支持向量机
1. 核函数（Kernel Function）：$$ K(x, y) = \phi(x)^{T} \phi(y) $$
2. 支持向量（Support Vectors）：$$ x_{i} \text{ s.t. } \frac{1}{n} \sum_{i=1}^{n} y_{i} K(x_{i}, x_{j}) + C \geq 0, j=1, 2, \dots, n $$

### 3.3.6 神经网络
1. 激活函数（Activation Function）：$$ f(x) = \frac{1}{1 + e^{-x}} $$
2. 梯度下降法（Gradient Descent）：$$ w_{i} = w_{i} - \eta \frac{\partial J}{\partial w_{i}} $$

```python
import pandas as pd

# 读取市场信息
data = pd.read_csv('market_data.csv')
```
### 4.1.2 数据预处理
```python
# 去重
data = data.drop_duplicates()

# 填充
data = data.fillna(method='ffill')

# 转换
data['category'] = data['category'].astype('category')
```

```python
# 标准差
std_age = data['age'].std()
```
### 4.2.2 关联分析
```python
# 相关系数
corr_age_income = data['age'].corr(data['income'])
```
### 4.2.3 聚类分析
```python
from sklearn.cluster import KMeans

# 欧式距离
euclidean_distance = lambda x, y: sum((x - y) ** 2 for x, y in zip(x, y))
```
