class AthleteManager {
    private $conn;

    public function __construct() {
        $this->conn = new mysqli("localhost", "username", "password", "database");
        if ($this->conn->connect_error) {
            die("Connection failed: ". $this->conn->connect_error);
        }
    }

    public function registerAthlete($athlete) {
        $stmt = $this->conn->prepare("INSERT INTO athletes (id, name, height, weight, best_time, current_score) VALUES (?,?,?,?,?,?)");
        $stmt->bind_param("iss", $athlete->getId(), $athlete->getName(), $athlete->getHeight(), $athlete->getWeight(), $athlete->getBestTime(), $athlete->getCurrentScore());

        if ($stmt->execute()) {
            return true;
        } else {
            return false;
        }
    }

    public function updateAthlete($id, $name, $height, $weight, $best_time, $current_score) {
        $stmt = $this->conn->prepare("UPDATE athletes SET name =?, height =?, weight =?, best_time =?, current_score =? WHERE id =?");
        $stmt->bind_param("iss", $name, $height, $weight, $best_time, $current_score);
        $stmt->execute();

        return $stmt->query("SELECT * FROM athletes WHERE id =?");
    }

    public function getAllAthletes($id) {
        $stmt = $this->conn->prepare("SELECT * FROM athletes WHERE id =?");
        $stmt->bind_param("i", $id);
        $stmt->execute();

        $stmt->close();
        $result = $stmt->get_result();

        return $result->fetch_all(MYSQLI_ASSOC);
    }

    public function getAthleteById($id) {
        $stmt = $this->conn->prepare("SELECT * FROM athletes WHERE id =?");
        $stmt->bind_param("i", $id);
        $stmt->execute();

        $stmt->close();
        $result = $stmt->get_result();

        return $result->fetch_assoc();
    }
}

7. 附录：常见问题与解答

### 常见问题

7.1. 为什么无法保存用户登录信息？

在实现用户登录功能时，需要对用户输入的用户名和密码进行存储和验证。如果在注册用户时没有正确填写用户名和密码，或者注册后没有进行登录，那么将无法保存用户登录信息。

7.2. 为什么在赛事浏览页面无法获取到赛事信息？

在赛事浏览页面，需要从数据库中获取赛事信息，并在网页上进行展示。如果在数据库中没有赛事信息，或者获取不到赛事信息，那么在赛事浏览页面将无法获取到赛事信息。

7.3. 为什么在运动员管理页面无法添加运动员？

在运动员管理页面，需要添加运动员信息，并将其保存到数据库中。如果在数据库中没有运动员信息，或者添加的运动员信息有误，那么在运动员管理页面将无法添加运动员。

### 常见解答

7.1. 可以在数据库中创建一个名为“user_login”的表，用于存储用户登录信息。在用户登录时，需要将用户名和密码保存到该表中。如果需要将用户名和密码与数据库中的信息进行验证，可以使用 PHP 的 PDO 扩展进行操作。

7.2. 如果在数据库中没有赛事信息，可以在数据库中创建一个名为“events”的表，用于存储赛事信息。在赛事浏览页面，需要从该表中获取赛事信息，并在网页上进行展示。

7.3. 在运动员管理页面，需要添加运动员信息。在数据库中，需要创建一个名为“athletes”的表，用于存储运动员信息。在添加运动员信息时，需要对运动员信息进行验证，并将其保存到该表中。

7.4. 在进行数据库操作时，需要使用 PHP 和 MySQLi 或 PDO 扩展进行操作。
