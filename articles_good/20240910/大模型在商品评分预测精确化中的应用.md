#### 1. 数据预处理

**题目：** 编写一个 Python 函数，实现以下数据预处理步骤：

- 数据清洗：去除空值和异常值；
- 数据转换：将字符串数据转换为数值数据；
- 缺失值处理：使用平均值、中位数或最近邻插值填补缺失值。

**答案：**

```python
import pandas as pd
from sklearn.impute import SimpleImputer
from sklearn.preprocessing import StandardScaler

def preprocess_data(data):
    # 数据清洗
    data = data.dropna()
    
    # 数据转换
    data = pd.get_dummies(data)
    
    # 缺失值处理
    imputer = SimpleImputer(strategy='mean')
    data = pd.DataFrame(imputer.fit_transform(data), columns=data.columns)
    
    # 数据标准化
    scaler = StandardScaler()
    data = pd.DataFrame(scaler.fit_transform(data), columns=data.columns)
    
    return data
```

**解析：** 此函数使用 Pandas 库进行数据处理，使用 Scikit-learn 库进行缺失值处理和标准化。

#### 2. 特征工程

**题目：** 编写一个 Python 函数，实现以下特征工程步骤：

- 特征选择：使用 F-tests 进行特征选择；
- 特征变换：将类别型特征转换为数值型特征；
- 特征构造：计算商品销量、用户购买率等新特征。

**答案：**

```python
import pandas as pd
from sklearn.feature_selection import f_classif

def feature_engineering(data):
    # 特征选择
    f_tests = f_classif(data.select_dtypes(include='number'), data['rating'])
    selected_features = data.columns[f_tests > 5].tolist()
    data = data[selected_features + ['rating']]
    
    # 特征变换
    data = pd.get_dummies(data)
    
    # 特征构造
    data['sales'] = data.groupby('item_id')['rating'].transform('sum')
    data['purchase_rate'] = data.groupby('user_id')['rating'].transform('mean')
    
    return data
```

**解析：** 此函数使用 Pandas 库进行数据处理，使用 Scikit-learn 库进行特征选择。
