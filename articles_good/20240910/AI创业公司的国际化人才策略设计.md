##### 1. 字符串匹配算法

**题目：** 实现一个字符串匹配算法，找到字符串s中的所有子字符串t的位置。

**输入：** s = "hello", t = "ll"

**输出：** [2, 3]

**答案解析：**

```python
def search(s, t):
    n, m = len(s), len(t)
    result = []
    for i in range(n - m + 1):
        if s[i:i+m] == t:
            result.append(i)
    return result

s = "hello"
t = "ll"
print(search(s, t))  # 输出 [2, 3]
```

##### 2. 最长公共前缀

**题目：** 编写一个函数来查找字符串数组中的最长公共前缀。

**输入：** ["flower","flow","flight"]

**输出：** "fl"

**答案解析：**

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        while not s.startswith(prefix):
            prefix = prefix[:-1]
            if not prefix:
                return ""
    return prefix

strs = ["flower","flow","flight"]
print(longestCommonPrefix(strs))  # 输出 "fl"
```

##### 3. 合并两个有序链表

**题目：** 将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

**输入：** l1 = [1,2,4], l2 = [1,3,4]

**输出：** [1,1,2,3,4,4]

**答案解析：**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeTwoLists(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        curr = dummy
        p, q = l1, l2
        while p and q:
            if p.val < q.val:
                curr.next = p
                p = p.next
            else:
                curr.next = q
                q = q.next
            curr = curr.next
        curr.next = p or q
        return dummy.next

# 构建链表
l1 = ListNode(1, ListNode(2, ListNode(4)))
l2 = ListNode(1, ListNode(3, ListNode(4)))
# 合并链表
result = Solution().mergeTwoLists(l1, l2)
# 输出结果
while result:
    print(result.val, end=" ")
    result = result.next
```

**输出：** 1 1 2 3 4 4
