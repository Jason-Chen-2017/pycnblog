```python
from typing import List

def energy_matching(energy_supply: List[int], energy_demand: List[int]) -> float:
    total_demand = sum(energy_demand)
    total_supply = sum(energy_supply)
    if total_supply == 0:
        return 0.0
    return min(total_supply, total_demand) / total_supply
```

**解析：** 匹配度等于供应和需求的最小值除以供应量。

**3. 编写一个Python函数，计算给定能源供应序列和能源需求序列之间的最大匹配度。**

**题目：** 编写一个Python函数`max_energy_matching(energy_supply: List[int], energy_demand: List[int]) -> float`，计算给定能源供应序列和能源需求序列之间的最大匹配度。

**示例：**


**答案：**

```python
from typing import List

def max_energy_matching(energy_supply: List[int], energy_demand: List[int]) -> float:
    n = min(len(energy_supply), len(energy_demand))
    dp = [[0] * (n + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for j in range(1, n + 1):
            dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
            if energy_supply[i - 1] == energy_demand[j - 1]:
                dp[i][j] = max(dp[i][j], dp[i - 1][j - 1] + 1)

    return dp[n][n] / max(n, n)
```

**解析：** 使用动态规划计算最大匹配度，最大匹配度等于动态规划表中最后一个元素的值除以供应和需求序列的最小长度。

```python
from typing import List

def balance_degree(energy_supply: List[int], energy_demand: List[int]) -> float:
    total_supply = sum(energy_supply)
    total_demand = sum(energy_demand)
    return abs(total_supply - total_demand) / max(total_supply, total_demand)
```

**解析：** 供需平衡度等于供需差值的绝对值除以供需量中的较大值。

**10. 编写一个Python函数，计算给定能源供应序列和能源需求序列之间的供需优化度。**

**题目：** 编写一个Python函数`优化度(energy_supply: List[int], energy_demand: List[int], target_balance: float) -> float`，计算给定能源供应序列和能源需求序列之间的供需优化度。

```python
def 优化度(energy_supply, energy_demand, target_balance):
    # 请在这里实现代码
```

**答案：**

```python
from typing import List

def 优化度(energy_supply: List[int], energy_demand: List[int], target_balance: float) -> float:
    balance = abs(sum(energy_supply) - sum(energy_demand))
    return 1 - abs(balance - target_balance) / target_balance
```
