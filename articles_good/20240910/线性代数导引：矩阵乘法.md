**题目：** 如何计算两个矩阵的乘积？

**答案：** 矩阵乘法的计算规则如下：

- 两个矩阵 A 和 B 的乘积 C = AB，只有当 A 的列数等于 B 的行数时，乘积 C 才有意义。
- 乘积矩阵 C 的大小为 A 的行数乘以 B 的列数。
- C 的每个元素 C[i][j] 是 A 的第 i 行与 B 的第 j 列对应元素的乘积之和。

**公式：**

\[
C_{ij} = \sum_{k=1}^{n} A_{ik} \times B_{kj}
\]

其中，\(A\) 是 \(m \times n\) 矩阵，\(B\) 是 \(n \times p\) 矩阵，\(C\) 是 \(m \times p\) 矩阵。

**举例：**

```markdown
A = |1 2|
    |3 4|

B = |5 6|
    |7 8|

C = A \* B = |1*5 + 2*7  1*6 + 2*8|
             |3*5 + 4*7  3*6 + 4*8|

C = |19 20|
    |43 46|
```

### 2. 如何通过矩阵乘法求解线性方程组？

**题目：** 如何使用矩阵乘法求解线性方程组？

**答案：** 线性方程组 Ax = b 可以通过矩阵乘法求解，其中 A 是系数矩阵，x 是未知数向量，b 是常数向量。

1. 将线性方程组转化为矩阵形式 Ax = b。
2. 计算矩阵 A 的逆矩阵 \(A^{-1}\)。
3. 用逆矩阵乘以常数向量 b，即 \(x = A^{-1}b\)。

**举例：**

```markdown
方程组：
1x + 2y = 3
3x + 4y = 7

矩阵形式：
A = |1 2|
    |3 4|

b = |3|
    |7|

计算逆矩阵 A^{-1}：
A^{-1} = \frac{1}{(1 \times 4) - (2 \times 3)} |4 - 7|
                                       |-(3 \times 2) - (1 \times 1)| = \frac{1}{-2} |-3|
                                                                         |7| = |-1.5|
                                                                               |-7|

计算 x = A^{-1}b：
x = |-1.5| |3| = (-1.5 \times 3) = -4.5
   |-7|   |7|        

x = -4.5
y = -7 \times (-1.5) = 10.5
```

### 6. 如何判断两个矩阵是否可交换？

**题目：** 如何判断两个矩阵 A 和 B 是否可交换？

**答案：** 如果两个矩阵 A 和 B 可交换，即 AB = BA，则它们是交换矩阵。

判断方法如下：

1. **直接计算：** 计算 AB 和 BA，如果两者相等，则 A 和 B 可交换。
2. **特征值判断：** 如果 A 和 B 都是方阵，且它们的特征值相同，则它们可交换。

### 8. 矩阵乘法的时间复杂度是多少？

**题目：** 矩阵乘法的时间复杂度是多少？

**答案：** 矩阵乘法的时间复杂度取决于矩阵的大小。假设矩阵 A 是 \(m \times n\) 矩阵，矩阵 B 是 \(n \times p\) 矩阵，则矩阵乘法的时间复杂度为 \(O(mnp)\)。
