##### 1. 集合的基本运算

**题目：** 给定两个集合 A = {1, 2, 3} 和 B = {3, 4, 5}，求它们的并集、交集和差集。

**答案：**

- 并集 A ∪ B = {1, 2, 3, 4, 5}
- 交集 A ∩ B = {3}
- 差集 A - B = {1, 2}

**解析：** 并集是包含两个集合中所有元素的集合，交集是同时属于两个集合的元素构成的集合，差集是一个集合中去除另一个集合中元素的集合。

##### 2. 集合的基数

**题目：** 一个集合的基数是指集合中元素的数量，给定集合 A = {1, 2, 3, 4, 5}，求集合 A 的基数。

**答案：** 集合 A 的基数为 5。

**解析：** 集合的基数就是集合中元素的数量，可以通过计数的方法得到。

##### 3. 集合的等价关系

**题目：** 判断以下两个集合是否等价：A = {1, 2, 3} 和 B = {4, 5, 6}。

**答案：** 集合 A 和 B 不等价。

**解析：** 等价关系是指两个集合具有相同的基数，并且每个元素的集合相等。由于 A 和 B 的元素集合不相等，所以它们不等价。

##### 4. 集合的划分

**答案：** 集合 A 的划分如下：

- {1}
- {2}
- {3}
- {4}
- {5}
- {6}
- {1, 2}
- {1, 3}

**解析：** 集合的划分是指将集合划分为若干个子集，使得每个子集都是集合的真子集，并且所有子集的并集等于原集合。

##### 5. 集合的映射

**题目：** 给定集合 A = {1, 2, 3} 和集合 B = {a, b, c}，求从 A 到 B 的映射 f。

**答案：** 一种可能的映射 f 如下：

- f(1) = a
- f(2) = b
- f(3) = c

**解析：** 映射是指将集合中的每个元素映射到另一个集合中的一个元素，保持元素的顺序。

##### 6. 集合的极限

**题目：** 给定集合 A = {1/n : n ∈ N}，求集合 A 的极限。

**答案：** 集合 A 的极限为 0。

**解析：** 集合的极限是指随着 n 趋近于无穷大时，集合中元素的极限值。在这个例子中，随着 n 趋近于无穷大，1/n 的值趋近于 0。

#### 算法编程题库

##### 1. 计算两个集合的并集、交集和差集

**题目：** 编写一个函数，计算两个集合的并集、交集和差集。

**答案：** 以下是一个使用 Python 实现的示例：

```python
def set_operations(A, B):
    union = A.union(B)
    intersection = A.intersection(B)
    difference = A.difference(B)
    return union, intersection, difference

A = {1, 2, 3}
B = {3, 4, 5}
union, intersection, difference = set_operations(A, B)

print("并集:", union)
print("交集:", intersection)
print("差集:", difference)
```

**解析：** 使用 Python 的集合（`set`）数据结构，可以方便地计算两个集合的并集、交集和差集。`union()`、`intersection()` 和 `difference()` 是集合的内置方法。

##### 2. 求集合的基数

**题目：** 编写一个函数，求集合的基数。

**答案：** 以下是一个使用 Python 实现的示例：

```python
def set_cardinality(A):
    return len(A)

A = {1, 2, 3}
cardinality = set_cardinality(A)

print("集合 A 的基数:", cardinality)
```

**解析：** 使用 `len()` 函数可以方便地求得集合的基数。
