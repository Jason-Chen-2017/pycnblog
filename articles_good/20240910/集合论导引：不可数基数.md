在集合论中，不可数基数是一个重要的概念，它描述了某些集合的大小无法用自然数来一一对应。本文将介绍与不可数基数相关的一些典型面试题和算法编程题，并给出详尽的答案解析。

### 1. 如何证明实数集是不可数的？

**题目：** 请解释并证明实数集（\(R\)）是不可数的。

**答案：** 

实数集 \(R\) 是不可数的，可以通过以下步骤证明：

1. **构造一个枚举序列：** 首先，我们可以尝试构造一个实数的枚举序列。例如，我们可以对 \(R\) 中的有理数进行排序，然后添加无理数，以构造一个实数的序列。

2. **证明序列不完整：** 通过康托尔对角线法，我们可以证明这个序列不能覆盖 \(R\) 中的所有实数。具体地，我们可以通过以下步骤构造一个新的实数，该实数与序列中的每一个实数都不同。

3. **得出结论：** 如果存在一个实数序列能够枚举 \(R\) 中的所有实数，那么这个序列必定是完整的。由于我们已经证明了不存在这样的序列，因此 \(R\) 是不可数的。

**代码示例：**

```python
def construct_inequality(seq, n, decimal_places):
    return abs(seq[n][decimal_places] - float(n) / decimal_places) > 0.5

def cantor_diagonal_proof(seq, decimal_places=10):
    new_seq = []
    for n, row in enumerate(seq):
        new_seq.append([row[i] + construct_inequality(seq, n, decimal_places) for i in range(len(row))])
    return new_seq

# 示例：有理数和无理数的序列
seq = [
    [x, x**2] for x in range(10)] + [
    [math.pi, math.pi**2] for i in range(10)]
new_seq = cantor_diagonal_proof(seq)
print(new_seq)
```

### 2. 如何判断两个集合是否等势？

**题目：** 请编写一个函数，判断两个集合 \(A\) 和 \(B\) 是否等势（即存在双射映射）。

**答案：**

函数实现如下：

```python
def are_bijective(A, B):
    if len(A) != len(B):
        return False

    mapping = {}
    for a in A:
        if a in mapping:
            return False
        for b in B:
            if a == b and (a not in mapping or mapping[a] != b):
                mapping[a] = b
                break
        else:
            return False

    return True

# 测试
A = [1, 2, 3]
B = [4, 5, 6]
print(are_bijective(A, B)) # 输出：False
```

    try:
        iterator = iter(A)
