**1. 编写一个Python程序，实现两个电商平台的订单数据融合。**

**答案：** 请参考以下Python代码：

```python
import pandas as pd

def merge_orders(order1, order2):
    df1 = pd.DataFrame(order1)
    df2 = pd.DataFrame(order2)
    
    df1['source'] = 'platform1'
    df2['source'] = 'platform2'
    
    merged_df = pd.concat([df1, df2])
    merged_df.drop_duplicates(subset=['order_id'], inplace=True)
    
    return merged_df

order1 = [{'order_id': 1, 'customer_id': 1001, 'amount': 100.0},
          {'order_id': 2, 'customer_id': 1002, 'amount': 200.0}]

order2 = [{'order_id': 3, 'customer_id': 1001, 'amount': 150.0},
          {'order_id': 4, 'customer_id': 1003, 'amount': 300.0}]

merged_orders = merge_orders(order1, order2)
print(merged_orders)
```

**2. 编写一个Python程序，实现基于K-means算法的电商用户群体聚类。**

**答案：** 请参考以下Python代码：

```python
import pandas as pd
from sklearn.cluster import KMeans

def cluster_users(users, k):
    df = pd.DataFrame(users)
    
    # 提取用户特征
    features = df[['age', 'income', 'spending']]
    
    # 使用K-means算法进行聚类
    kmeans = KMeans(n_clusters=k, random_state=0)
    kmeans.fit(features)
    
    # 添加聚类结果到原始数据
    df['cluster'] = kmeans.labels_
    
    return df

users = [{'user_id': 1, 'age': 25, 'income': 5000, 'spending': 2000},
         {'user_id': 2, 'age': 30, 'income': 6000, 'spending': 3000},
         {'user_id': 3, 'age': 35, 'income': 7000, 'spending': 4000}]

clustered_users = cluster_users(users, 2)
print(clustered_users)
```
