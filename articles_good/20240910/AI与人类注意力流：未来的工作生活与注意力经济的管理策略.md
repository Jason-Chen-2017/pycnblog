#### **面试题：** 深度学习中的注意力机制是如何工作的？请举例说明。

**答案：** 深度学习中的注意力机制是一种用于提高模型性能的技术，它通过赋予模型对输入数据的不同部分不同的重要性来改善特征提取和模型决策。

**示例：** 以Transformer模型中的自注意力（Self-Attention）为例：

```python
import torch
from torch.nn import MultiheadAttention

# 假设输入序列的长度为10，每个词的维度为512
input_sequence = torch.rand(10, 512)

# 定义自注意力机制
multihead_attn = MultiheadAttention(embed_dim=512, num_heads=8)

# 计算自注意力
output, _ = multihead_attn(input_sequence, input_sequence, input_sequence)

# 输出维度为(10, 512)
print(output.shape)
```

**解析：** 在这个例子中，`MultiheadAttention` 对象使用了自注意力机制。输入序列经过线性层转换后，通过计算查询（Query）、键（Key）和值（Value）之间的相似度来生成注意力权重。这些权重用于对输入序列进行加权求和，生成输出序列。
