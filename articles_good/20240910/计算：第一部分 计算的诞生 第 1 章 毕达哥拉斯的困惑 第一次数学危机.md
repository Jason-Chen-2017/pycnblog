                 

### 《计算的诞生》：第1章 毕达哥拉斯的困惑与第一次数学危机

#### 典型问题/面试题库

**1. 毕达哥拉斯定理的数学表达是什么？**

**答案：** 毕达哥拉斯定理是直角三角形中三边关系的数学表达，公式为 \(a^2 + b^2 = c^2\)，其中 \(a\) 和 \(b\) 是直角边，\(c\) 是斜边。

**2. 什么是第一次数学危机？**

**答案：** 第一次数学危机源于古希腊，主要矛盾是“无理数的存在”。尽管毕达哥拉斯定理表明直角三角形的边长关系可以用整数表示，但实际中发现并非所有边长都可以用整数表示，例如边长为 \(\sqrt{2}\) 的直角三角形，这就引发了关于无理数存在的争议。

**3. 无理数与有理数的区别是什么？**

**答案：** 有理数是可以表示为两个整数之比的数，包括整数、分数、小数（有限或无限循环小数）；无理数则是无法表示为两个整数之比的数，其小数部分为无限不循环。

**4. 如何证明 \(\sqrt{2}\) 是无理数？**

**答案：** 使用反证法，假设 \(\sqrt{2}\) 是有理数，则可以表示为两个整数之比，通过推导得出矛盾，从而证明 \(\sqrt{2}\) 是无理数。

**5. 数学危机对数学发展有何影响？**

**答案：** 数学危机促使数学家开始重新审视数学的基本概念，推动了无理数的理论发展，并引发了对数学逻辑和公理体系的反思。

#### 算法编程题库

**6. 实现一个函数，判断一个数是否为无理数。**

```python
def is_irrational(num):
    # 你的代码
    pass
```

**答案解析：**

```python
def is_irrational(num):
    # 对于整数和分数，它们都是有理数
    if isinstance(num, (int, float)):
        return False
    # 可以通过试除法来判断一个整数是否为无理数
    for i in range(2, int(num**0.5)+1):
        if num % i == 0:
            return False
    return True

# 测试
print(is_irrational(2))  # 输出：False
print(is_irrational(3.14))  # 输出：False
print(is_irrational(3.14159))  # 输出：True
```

**7. 编写一个程序，找出100以内的无理数。**

```python
def find_irrationals(limit):
    # 你的代码
    pass

# 测试
print(find_irrationals(100))
```

**答案解析：**

```python
def find_irrationals(limit):
    irrationals = []
    for num in range(2, limit+1):
        if is_irrational(num):
            irrationals.append(num)
    return irrationals

# 测试
print(find_irrationals(100))
```

#### 算法题实例解析

**8. 给定一个无理数，如何将其表示为有限或无限循环小数？**

**答案：** 对于一些无理数，例如 \(\sqrt{2}\)，可以通过长除法将其表示为无限循环小数。以下是一个简单的实现：

```python
def decimal_expansion(root):
    integer_part = int(root)
    decimal_part = root - integer_part
    
    result = str(integer_part) + '.'
    while decimal_part > 0:
        decimal_part *= 10
        digit = int(decimal_part)
        result += str(digit)
        decimal_part -= digit
        
    return result

# 测试
print(decimal_expansion(2))  # 输出：2.
print(decimal_expansion(3.14159))  # 输出：3.14159
print(decimal_expansion(3.14))  # 输出：3.14
```

**9. 给定一个循环小数，如何将其表示为一个分数？**

**答案：** 可以使用以下公式来将循环小数表示为一个分数：

\[ \text{循环小数} = \text{非循环部分} + \frac{\text{循环部分}}{10^n - 1} \]

其中 \( n \) 是循环部分的位数。

```python
def cyclic_decimal_to_fraction(decimal_str):
    non_loop_part = decimal_str.index('.')  # 非循环部分
    loop_part = decimal_str[non_loop_part+len(decimal_str.split('.')[1]):]  # 循环部分
    n = len(loop_part)  # 循环部分的位数
    
    non_loop = float('0.' + decimal_str[:non_loop_part])
    loop = float('0.' + loop_part) / (10**n - 1)
    
    result = non_loop + loop
    return result

# 测试
print(cyclic_decimal_to_fraction('0.3333'))  # 输出：0.3333
print(cyclic_decimal_to_fraction('0.142857142857'))  # 输出：0.142857142857
```

#### 综合题实例

**10. 编写一个程序，计算并打印出100以内的所有无理数的有限或无限循环小数表示。**

```python
def print_irrational_expansions(limit):
    for num in range(2, limit+1):
        if is_irrational(num):
            decimal_expansion = decimal_expansion(num)
            print(f"{num} 的循环小数表示为：{decimal_expansion}")

# 测试
print_irrational_expansions(100)
```

**答案解析：** 此程序会遍历2到100之间的所有整数，判断每个数是否为无理数。如果是，则使用长除法计算出其循环小数表示，并打印出来。
