#### 1. 计算运动员速度
**题目描述：** 给定运动员的起始位置、终止位置和运动时间，计算运动员的平均速度。
**答案解析：** 可以利用勾股定理计算运动员的位移，再除以运动时间得到平均速度。
```go
import "math"

func calculateSpeed(x1, y1, x2, y2, time float64) float64 {
    distance := math.Sqrt((x2 - x1) * (x2 - x1) + (y2 - y1) * (y2 - y1))
    speed := distance / time
    return speed
}
```

#### 2. 判断运动员是否越位
**题目描述：** 给定足球比赛中的运动员位置，判断运动员是否处于越位状态。
**答案解析：** 可以通过计算运动员与对方球门线的距离和与对方球员的最近距离，来判断运动员是否越位。
```go
func isOffside(x, y, goalX, goalY, opponentX, opponentY float64) bool {
    distanceToGoal := math.Sqrt((x - goalX) * (x - goalX) + (y - goalY) * (y - goalY))
    distanceToOpponent := math.Sqrt((x - opponentX) * (x - opponentX) + (y - opponentY) * (y - opponentY))
    
    if distanceToGoal < distanceToOpponent {
        return true
    }
    return false
}
```
