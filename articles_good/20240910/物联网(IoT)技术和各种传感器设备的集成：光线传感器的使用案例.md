**题目：** 请简述光线传感器的基本工作原理。

**答案：** 光线传感器通常基于光敏电阻或光敏二极管的工作原理。当光线照射到传感器时，光子能量会导致光敏材料中的电子跃迁，从而产生电流或电压变化，通过测量这个变化，可以得到光线强度的信息。

**解析：** 这个问题考察应聘者对传感器原理的理解。正确回答应该包含光敏电阻或光敏二极管的原理，以及它们如何将光信号转换为电信号。

        return 0
    return sum(data) / len(data)

#### 2. 光线传感器数据滤波

**题目：** 使用移动平均滤波算法对光线传感器数据进行滤波，以去除噪声。

**答案：**

```python
def moving_average(data, window_size):
    if not data or window_size <= 1:
        return data
    filtered_data = []
    for i in range(len(data)):
        if i < window_size:
            filtered_data.append(sum(data[:i+1]) / (i+1))
        else:
            filtered_data.append(sum(data[i-window_size+1:i+1]) / window_size)
    return filtered_data

# 示例数据
sensor_data = [100, 101, 110, 120, 121, 130, 131, 140]

# 滤波后的数据
filtered_data = moving_average(sensor_data, 3)
print("滤波后光线强度：", filtered_data)
```

**解析：** 这个函数使用移动平均滤波算法，对输入的光线传感器数据进行滤波。对于每个数据点，计算一个滑动窗口内的平均值，作为滤波后的值。示例数据集的输出滤波后数据去除了噪声，提高了数据平滑度。
