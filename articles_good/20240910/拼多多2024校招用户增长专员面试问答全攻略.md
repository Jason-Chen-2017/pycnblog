### 三、算法编程题库

#### 3.1 算法与数据结构

**1. 请实现一个堆排序算法。**

**代码实例：**
```python
def heapify(arr, n, i):
    largest = i
    left = 2 * i + 1
    right = 2 * i + 2

    if left < n and arr[i] < arr[left]:
        largest = left

    if right < n and arr[largest] < arr[right]:
        largest = right

    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

def heapSort(arr):
    n = len(arr)

    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)

    for i in range(n - 1, 0, -1):
        arr[i], arr[0] = arr[0], arr[i]
        heapify(arr, i, 0)

arr = [12, 11, 13, 5, 6, 7]
heapSort(arr)
print("排序后的数组：", arr)
```

**解析：**
- **堆排序算法**：首先将数组构建成最大堆，然后交换堆顶元素（最大元素）与最后一个元素，再对剩余元素重新构建最大堆，重复此过程，直到堆中只剩下一个元素，此时数组已排序。

#### 3.2 数学与逻辑

**2. 请实现一个查找旋转排序数组中的最小元素的算法。**

**代码实例：**
```python
def searchMin(nums):
    low, high = 0, len(nums) - 1

    while low < high:
        mid = (low + high) // 2

        if nums[mid] > nums[high]:
            low = mid + 1
        else:
            high = mid

    return nums[low]

nums = [4, 5, 6, 7, 0, 1, 2]
print("最小元素：", searchMin(nums))
```

**解析：**
- **查找旋转排序数组中的最小元素**：通过二分查找的方法，比较中间元素和最右端元素的大小关系，不断缩小区间，直到找到最小元素。
