## 2. 核心概念与联系

### 2.1 黎曼空间

黎曼空间是一个光滑的流形，它具有一个内在的度量张量，即黎曼度量张量。黎曼度量的存在使得我们可以定义空间中的距离、角度等几何量。

### 2.2 对称空间

一个流形$M$被称为对称空间，如果存在一个非退化的对称性群$G$，使得对任意的$g \in G$和$p \in M$，有$g \cdot p = p$。这里的$\cdot$表示对称性群的作用。

### 2.3 黎曼对称空间

黎曼对称空间是黎曼空间和对称空间的一个结合，它既满足黎曼空间的性质，又满足对称空间的性质。

### 5.2 源代码详细实现

下面是球面和欧氏空间黎曼对称空间模型的Python代码实现：

```python
import numpy as np

class RiemannianSurface:
    def __init__(self, metric, group):
        self.metric = metric
        self.group = group

    def area(self):
        return np.trace(self.metric) * np.pi

    def curvature(self):
        return -np.linalg.det(self.metric) / np.trace(self.metric)

class Sphere(RiemannianSurface):
    def __init__(self, radius):
        super().__init__(np.array([[1, 0], [0, radius**2]]), np.pi * 2)

class EuclideanSpace(RiemannianSurface):
    def __init__(self):
        super().__init__(np.array([[1, 0], [0, 1]]), np.inf)
```

`Sphere`类和`EuclideanSpace`类分别表示球面和欧氏空间，它们继承自`RiemannianSurface`类。在`Sphere`类中，我们使用二维单位球面的度量张量来初始化`metric`属性，将对称性群设置为$\pi \times 2$。在`EuclideanSpace`类中，我们使用单位正方体的度量张量来初始化`metric`属性，将对称性群设置为无穷大。
