## 2.1 贝叶斯定理

贝叶斯定理是概率论中的一个重要定理，它描述了如何更新先验知识（prior knowledge）为新的观测数据（observational data）提供条件概率（conditional probability）。贝叶斯定理的数学表达式为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示条件概率，即给定事件$B$发生，事件$A$的概率；$P(B|A)$ 表示条件概率，即给定事件$A$发生，事件$B$的概率；$P(A)$ 表示事件$A$的先验概率；$P(B)$ 表示事件$B$的先验概率。

1. 条件概率计算：

$$

$$

其中，$w_i$ 表示词汇$i$，$C_k$ 表示类别$k$；$P(w_i|C_k)$ 表示给定类别$k$，词汇$i$的概率；$P(C_k|w_i)$ 表示给定词汇$i$，类别$k$的概率；$P(w_i)$ 表示词汇$i$的先验概率；$\sum_{j=1}^{|V|} P(C_k|w_j)P(w_j)$ 表示类别$k$下所有词汇的概率和。

2. 类别概率计算：

$$
P(C_k) = \frac{\sum_{i=1}^{|D|} \delta(C_k, d_i)}{\sum_{j=1}^{|D|} \delta(C_j, d_i)}
$$

其中，$D$ 表示数据集，$d_i$ 表示数据集中的第$i$个文本；$\delta(C_k, d_i)$ 表示数据集中的第$i$个文本属于类别$k$。

3. 词汇映射关系计算：

$$
f(w_i) = \operatorname*{arg\,max}_k P(C_k|w_i)
$$

其中，$f(w_i)$ 表示词汇$i$的翻译；$\operatorname*{arg\,max}_k P(C_k|w_i)$ 表示给定词汇$i$，最大化类别$k$的概率。

```python
import numpy as np

def conditional_probability(word_index, X_train, y_train):
    N = len(X_train)
    V = len(word_index)
    C = np.unique(y_train)
    P_C = len(C) / N
    P_W = np.zeros((V, C))
    P_WC = np.zeros((V, C))
    for i, word in enumerate(word_index.keys()):
        P_W[i, :] = np.zeros(C)
        P_WC[i, :] = np.zeros(C)
    for i, x in enumerate(X_train):
        for j, word in enumerate(x):
            P_W[j, y_train[i]] += 1
            P_WC[j, y_train[i]] += 1
    P_WC += np.eye(C) * (np.sum(P_W, axis=0) - P_W.sum(axis=1, keepdims=True))
    P_W /= P_WC
    return P_W, P_C
```

```python
    translated_words = []
    for word in words:
        if word in word_index:
            translated_index = np.argmax(model[word_index[word]])
            translated_words.append(list(word_index.keys())[translated_index])
```
