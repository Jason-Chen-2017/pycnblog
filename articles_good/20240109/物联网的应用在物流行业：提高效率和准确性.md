4. 数据展示：物流管理系统可以将运输轨迹展示给物流企业，帮助企业更准确地了解物品的运输情况。

```python
import gps
import time

def gps_update(gps_data):
    print("GPS data updated:")
    print("Time: ", gps_data.timestamp)
    print("Latitude: ", gps_data.latitude)
    print("Longitude: ", gps_data.longitude)

def main():
    gps_cw = gps.gps()
    gps_cw.start()
    try:
        while True:
            gps_data = gps_cw.next()
            gps_update(gps_data)
    except KeyboardInterrupt:
        gps_cw.stop()
        print("GPS tracking stopped.")

if __name__ == "__main__":
    main()
```
