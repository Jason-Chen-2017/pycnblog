### 4.1 数据清洗的具体代码实例

```python
import re

def clean(data):
    if isinstance(data, str):
        data = re.sub(r'\W+', ' ', data)
        data = re.sub(r'\s+', ' ', data).strip()
    return data

data = '123-456-7890'
cleaned_data = clean(data)
print(cleaned_data)  # Output: 123 456 7890
```

在上面的代码实例中，我们定义了一个名为 `clean` 的函数，用于对字符串数据进行清洗。函数首先检查数据是否为字符串，然后使用正则表达式将非字母数字字符替换为空格，然后使用另一个正则表达式将多个空格替换为单个空格，并去除头尾空格。最后，返回清洗后的数据。

```python
def convert(data, from_format, to_format):
    if from_format == 'int' and to_format == 'float':
        return float(data)
    elif from_format == 'float' and to_format == 'int':
        return int(data)
    else:
        raise ValueError('Unsupported format')

from_format = 'int'
to_format = 'float'
converted_data = convert(data, from_format, to_format)

```

在上面的代码实例中，我们定义了一个名为 `convert` 的函数，用于对整数数据进行转换。函数首先检查输入数据的格式和目标格式，然后根据格式进行转换。如果输入数据为整数，则将其转换为浮点数；如果输入数据为浮点数，则将其转换为整数。如果输入数据的格式不支持转换，则会引发 ValueError 异常。
