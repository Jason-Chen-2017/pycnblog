在Python中，我们可以使用NumPy库来实现随机漫步。以下是一个简单的随机漫步示例：

```python
import numpy as np

def random_walk(n_steps, prob_matrix):
    current_state = 0
    states = np.arange(prob_matrix.shape[0])
    for _ in range(n_steps):
        next_state = np.random.choice(states, p=prob_matrix[current_state])
        current_state = next_state
    return current_state

prob_matrix = np.array([[0.5, 0.5],
                        [0.3, 0.7]])
n_steps = 10
current_state = random_walk(n_steps, prob_matrix)
print(current_state)
```
