## 3.1 线性回归

线性回归是一种常用的机器学习算法，用于预测连续型变量。线性回归的基本思想是通过拟合一条直线来最小化误差，从而预测目标变量的值。线性回归的数学模型如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是目标变量，$x_1, x_2, \cdots, x_n$ 是输入变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差。

线性回归的具体操作步骤如下：

1. 收集和准备数据。
2. 计算参数。
3. 预测目标变量的值。

线性回归的参数可以通过最小二乘法求解。最小二乘法的公式如下：

$$
\min \sum_{i=1}^n (y_i - (\beta_0 + \beta_1x_{1i} + \beta_2x_{2i} + \cdots + \beta_nx_{ni}))^2
$$

## 3.2 逻辑回归

逻辑回归是一种常用的机器学习算法，用于预测二值型变量。逻辑回归的基本思想是通过拟合一个sigmoid函数来预测目标变量的值。逻辑回归的数学模型如下：

$$
P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \cdots - \beta_nx_n}}
$$

其中，$y$ 是目标变量，$x_1, x_2, \cdots, x_n$ 是输入变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

逻辑回归的具体操作步骤如下：

1. 收集和准备数据。
2. 计算参数。
3. 预测目标变量的值。

逻辑回归的参数可以通过最大似然估计求解。最大似然估计的公式如下：

$$
\max \prod_{i=1}^n P(y_i|x_i)^{\hat{y}_i}(1 - P(y_i|x_i))^{1 - \hat{y}_i}
$$

其中，$\hat{y}_i$ 是预测值。

## 3.3 支持向量机

支持向量机是一种常用的机器学习算法，用于解决二分类问题。支持向量机的基本思想是通过找出支持向量来将不同类别的数据分开。支持向量机的数学模型如下：

$$
\min \frac{1}{2}\|\omega\|^2 + C\sum_{i=1}^n\xi_i
$$

其中，$\omega$ 是分隔超平面的法向量，$C$ 是正则化参数，$\xi_i$ 是松弛变量。

支持向量机的具体操作步骤如下：

1. 收集和准备数据。
2. 计算参数。
3. 预测目标变量的值。

支持向量机的参数可以通过松弛最大 margin 方法求解。松弛最大 margin 方法的公式如下：

$$
\min \frac{1}{2}\|\omega\|^2 \\
\text{s.t.} \quad y_i(\omega \cdot x_i + b) \geq 1 - \xi_i, \xi_i \geq 0, i = 1, 2, \cdots, n
$$

## 3.4 随机森林

随机森林是一种常用的机器学习算法，用于解决多分类和回归问题。随机森林的基本思想是通过构建多个决策树来预测目标变量的值，并通过平均法得到最终的预测值。随机森林的数学模型如下：

$$
\hat{y} = \frac{1}{K}\sum_{k=1}^K f_k(x)
$$

其中，$K$ 是决策树的数量，$f_k(x)$ 是第$k$个决策树的预测值。

随机森林的具体操作步骤如下：

1. 收集和准备数据。
2. 构建决策树。
3. 预测目标变量的值。

随机森林的参数可以通过随机抽取特征和训练数据集来求解。随机抽取特征的公式如下：

$$
S = \{j \in \{1, 2, \cdots, p\}\}
$$

其中，$p$ 是特征的数量，$S$ 是随机抽取的特征集。

随机抽取训练数据集的公式如下：

$$
T = \{i \in \{1, 2, \cdots, n\}\}
$$

其中，$n$ 是样本的数量，$T$ 是随机抽取的训练数据集。

## 4.1 线性回归

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 3 * x.squeeze() + 2 + np.random.randn(100, 1)

# 训练模型
model = LinearRegression()
model.fit(x, y)

# 预测
x_test = np.array([[0.5], [0.8], [1.0]])
y_predict = model.predict(x_test)

# 绘图
plt.scatter(x, y)
plt.plot(x, model.predict(x), color='red')
plt.show()
```

## 4.2 逻辑回归

```python
from sklearn.linear_model import LogisticRegression

# 生成数据

# 训练模型
model = LogisticRegression()
model.fit(x, y)

# 预测
x_test = np.array([[0.5], [0.8], [1.0]])
y_predict = model.predict(x_test)

# 绘图
plt.scatter(x, y)
plt.plot(x, model.predict(x) / np.max(model.predict(x)), color='red')
plt.show()
```
