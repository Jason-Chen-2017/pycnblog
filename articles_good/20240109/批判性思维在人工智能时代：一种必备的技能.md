### 3.3.1 线性回归

线性回归的数学模型公式如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是输出变量，$x_1, x_2, \cdots, x_n$ 是输入变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。

### 3.3.2 逻辑回归

逻辑回归的数学模型公式如下：

$$
P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \cdots - \beta_nx_n}}
$$

其中，$P(y=1|x)$ 是输出变量，$x_1, x_2, \cdots, x_n$ 是输入变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

### 3.3.3 支持向量机

支持向量机的数学模型公式如下：

$$
\min_{\mathbf{w},b} \frac{1}{2}\|\mathbf{w}\|^2 \\
s.t. \quad y_i(\mathbf{w} \cdot \mathbf{x}_i + b) \geq 1, \quad i = 1,2,\cdots,n
$$

其中，$\mathbf{w}$ 是权重向量，$b$ 是偏置项，$y_i$ 是输出变量，$\mathbf{x}_i$ 是输入变量。

## 4.1 线性回归

以下是一个简单的线性回归示例代码：

```python
import numpy as np

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 2 * x + 1 + np.random.randn(100, 1)

# 初始化参数
beta_0 = np.random.randn(1)
beta_1 = np.random.randn(1)

# 训练模型
learning_rate = 0.01
for i in range(1000):
    y_pred = beta_0 + beta_1 * x
    error = y - y_pred
    grad_beta_0 = -np.mean(error)
    grad_beta_1 = -np.mean(error * x)
    beta_0 -= learning_rate * grad_beta_0
    beta_1 -= learning_rate * grad_beta_1

# 预测
x_test = np.array([[0.5], [0.8], [1.1]])
y_pred = beta_0 + beta_1 * x_test
print(y_pred)
```

在这个示例中，我们首先生成了一组线性回归数据，然后使用梯度下降法训练了一个简单的线性回归模型，最后使用训练好的模型对新的数据进行预测。

## 4.3 支持向量机

以下是一个简单的支持向量机示例代码：

```python
import numpy as np

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 2)
y = 2 * x[:, 0] + 1 + np.random.randn(100, 1)

# 训练模型
C = 1
tol = 1e-3
max_iter = 1000

# 初始化参数
w = np.random.randn(2, 1)
b = 0

# 训练支持向量机
for i in range(max_iter):
    y_pred = np.sign(w.dot(x) + b)
    error = y - y_pred
    grad_w = np.dot(x.T, error)
    grad_b = np.mean(error)
    w -= C * grad_w
    b -= C * grad_b
    if np.linalg.norm(grad_w) < tol and np.linalg.norm(grad_b) < tol:
        break

# 预测
x_test = np.array([[0.5], [0.8], [1.1]])
y_pred = np.sign(w.dot(x_test) + b)
print(y_pred)
```

在这个示例中，我们首先生成了一组支持向量机数据，然后使用顺序最小化法训练了一个简单的支持向量机模型，最后使用训练好的模型对新的数据进行预测。
