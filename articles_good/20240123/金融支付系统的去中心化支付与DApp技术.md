## 1. 背景介绍

金融支付系统是现代经济活动的基石，它使人们能够轻松地进行支付和交易。然而，传统的金融支付系统存在一些挑战，如高成本、低效率和安全性问题。因此，去中心化支付和DApp技术在这些领域中发挥了重要作用。

### 1.1 去中心化支付

去中心化支付是一种新兴的支付方式，它摒弃了传统的中心化结构，而是将数据和交易权力分散到网络中的各个节点。这种方法可以提高系统的安全性、可靠性和效率。

### 1.2 DApp技术

DApp（Decentralized Application）技术是一种基于去中心化网络的应用开发技术，它使用区块链、智能合约等技术来实现去中心化的应用。DApp技术可以用于开发各种应用，包括金融支付系统。

import ecdsa

def generate_key_pair():
    private_key = ecdsa.SigningKey.generate()
    public_key = private_key.get_verifying_key()
    return private_key, public_key

def sign_transaction(private_key, transaction):
    signature = private_key.sign(transaction)
    return signature

def verify_transaction(public_key, transaction, signature):
    try:
        public_key.verify(signature, transaction)
        return True
    except Exception:
        return False
