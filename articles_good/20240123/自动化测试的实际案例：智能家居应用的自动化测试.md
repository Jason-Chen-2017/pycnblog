以下是一个智能家居应用的自动化测试代码实例：

```python
import unittest
from selenium import webdriver

class TestSmartHomeApp(unittest.TestCase):
    def setUp(self):
        self.driver = webdriver.Chrome()
        self.driver.implicitly_wait(10)

    def test_login(self):
        self.driver.get("https://www.example.com")
        self.driver.find_element_by_id("username").send_keys("admin")
        self.driver.find_element_by_id("password").send_keys("password")
        self.driver.find_element_by_id("login").click()
        self.assertIn("Dashboard", self.driver.title)

    def test_device_control(self):
        self.driver.find_element_by_link_text("Devices").click()
        devices = self.driver.find_elements_by_class_name("device")
        for device in devices:
            device.click()
            self.assertIn("Device Control", self.driver.title)

    def tearDown(self):
        self.driver.quit()

if __name__ == "__main__":
    unittest.main()
```

在这个代码实例中，我们使用Python的unittest模块和Selenium库来编写智能家居应用的自动化测试脚本。首先，我们导入了unittest和webdriver库，然后定义了一个TestSmartHomeApp类，继承于unittest.TestCase类。在TestSmartHomeApp类中，我们定义了setUp和tearDown方法来初始化和销毁测试环境，以及test_login和test_device_control方法来测试智能家居应用的登录和设备控制功能。最后，我们使用unittest.main()方法来执行测试用例。
