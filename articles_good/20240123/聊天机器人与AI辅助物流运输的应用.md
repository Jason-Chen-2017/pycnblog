### 4.1 聊天机器人实例

```python
import nltk
from nltk.chat.util import Chat, reflections

pairs = [
    [
        r"hi|hello|hey",
        ["Hello!", "Hi there!", "Hey!"]
    ],
    [
        r"how are you?",
        ["I'm good, thanks! How about you?", "I'm doing well, thanks for asking!"]
    ],
    [
        r"(.*) your name?",
        ["My name is Chatbot.", "I'm a chatbot."]
    ],
    [
        r"quit",
        ["Bye!", "See you later!"]
    ]
]

def chatbot():
    print("Hi, I'm Chatbot. How can I help you today?")
    chat = Chat(pairs, reflections)
    chat.converse()

if __name__ == "__main__":
    chatbot()
```
