```python
import numpy as np

# 自定义图表
plt.title('正弦曲线')
plt.xlabel('x轴')
plt.ylabel('y轴')
plt.legend('正弦')

# 保存图表

# 显示图表

```

在这个示例中，我们首先导入了NumPy和Matplotlib库。然后，我们创建了一组数据x和y，其中y是x的正弦值。接下来，我们使用`plt.plot()`函数创建了一个正弦曲线图表。然后，我们使用`plt.title()`、`plt.xlabel()`、`plt.ylabel()`和`plt.legend()`函数自定义了图表的标题、x轴标签、y轴标签和图例。最后，我们使用`plt.savefig()`函数保存了图表，并使用`plt.show()`函数显示了图表。

## 8. 附录：常见问题与解答

以下是一些PythonMatplotlib数据可视化的常见问题与解答：

**Q：如何创建多个子图？**

A：使用`plt.subplot()`函数可以创建多个子图。例如：
```python
plt.subplot(2, 2, 1)
plt.plot(x, y)
plt.title('子图1')
plt.subplot(2, 2, 2)
plt.plot(x, y)
plt.title('子图2')
plt.subplot(2, 2, 3)
plt.plot(x, y)
plt.title('子图3')
plt.subplot(2, 2, 4)
plt.plot(x, y)
plt.title('子图4')
plt.show()
```

**Q：如何保存图表为PDF文件？**

A：使用`plt.savefig()`函数，指定文件格式为'pdf'。例如：
```python
plt.savefig('图表名.pdf')
```

**Q：如何设置图表的尺寸？**

A：使用`plt.figure()`函数可以设置图表的尺寸。例如：
```python
plt.figure(figsize=(10, 6))
```

**Q：如何设置图表的背景色？**

A：使用`plt.axes()`函数可以设置图表的背景色。例如：
```python
plt.axes(facecolor='lightgrey')
```

**Q：如何设置图表的边距？**

A：使用`plt.subplots_adjust()`函数可以设置图表的边距。例如：
```python
plt.subplots_adjust(left=0.1, right=0.9, top=0.9, bottom=0.1)
```
