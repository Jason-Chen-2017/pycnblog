### 4.1 代码实例

以下是一个简单的代码实例，展示了如何在软件系统架构中实现团队协作与沟通：

```python
class TeamMember:
    def __init__(self, name, role):
        self.name = name
        self.role = role

    def communicate(self, message):
        print(f"{self.name} 说：{message}")

class Project:
    def __init__(self, name, members):
        self.name = name
        self.members = members

    def start(self):
        for member in self.members:
            member.communicate(f"项目 {self.name} 开始")

    def end(self):
        for member in self.members:
            member.communicate(f"项目 {self.name} 结束")

# 创建团队成员
member1 = TeamMember("Alice", "开发者")
member2 = TeamMember("Bob", "测试者")
member3 = TeamMember("Charlie", "设计师")

# 创建项目
project = Project("软件系统架构项目", [member1, member2, member3])

# 开始项目
project.start()

# 结束项目
project.end()
```

### 4.2 详细解释说明

上述代码实例中，我们创建了一个`TeamMember`类和一个`Project`类。`TeamMember`类表示团队成员，具有名字和角色两个属性。`Project`类表示项目，具有名字和成员列表两个属性。`Project`类中有一个`start`方法，用于开始项目，并通过调用成员的`communicate`方法，向成员发送开始项目的消息。`Project`类中有一个`end`方法，用于结束项目，并通过调用成员的`communicate`方法，向成员发送结束项目的消息。
