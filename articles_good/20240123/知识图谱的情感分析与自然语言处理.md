### 4.1 知识图谱构建

```python
import networkx as nx

# 创建一个有向图
G = nx.DiGraph()

# 添加实体和关系
G.add_edge("实体1", "实体2", "关系")

# 保存知识图谱
nx.write_gpickle(G, "知识图谱.pickle")
```

### 4.2 情感词表构建

```python
# 创建一个字典，用于存储情感词
情感词表 = {
    "好": 1,
    "不好": -1,
    "很好": 2,
    "很不好": -2,
}
```

### 4.3 情感分析算法实现

```python
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

# 创建一个文本数据集
文本数据集 = ["我觉得这个电影很好", "我觉得这个电影不好"]

# 创建一个词向量器
词向量器 = CountVectorizer()

# 将文本数据集转换为词向量
词向量 = 词向量器.fit_transform(文本数据集)

# 创建一个朴素贝叶斯分类器
分类器 = MultinomialNB()

# 训练分类器
分类器.fit(词向量, 情感词表.values())

# 测试分类器
测试文本 = "我觉得这个电影很不好"
测试词向量 = 词向量器.transform([测试文本])
测试情感 = 分类器.predict(测试词向量)
```
