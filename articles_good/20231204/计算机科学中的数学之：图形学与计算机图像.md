4.计算机视觉：计算机视觉是图形学与计算机图像的一个重要应用领域，它用于实现计算机与图像之间的交互和理解。计算机视觉需要我们关注图像学、计算机科学、人工智能等方面的知识，并使用各种算法和技术来实现图像的识别、分类、检测等任务。

    glLightfv(GL_LIGHT0, GL_POSITION, light_position);

    glEnable(GL_LIGHTING);
    glEnable(GL_LIGHT0);
}

void drawScene() {
    glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);
    glLoadIdentity();
