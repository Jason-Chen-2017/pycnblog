### 4.3.1 代码实例

from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression

X = [[0, 0], [1, 1]]

## 4.4 计算机科学家
### 4.4.1 代码实例
```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]

    left = merge_sort(left)
    right = merge_sort(right)

    return merge(left, right)

def merge(left, right):
    result = []
    left_index = 0
    right_index = 0

    while left_index < len(left) and right_index < len(right):
        if left[left_index] < right[right_index]:
            result.append(left[left_index])
            left_index += 1
        else:
            result.append(right[right_index])
            right_index += 1

    result.extend(left[left_index:])
    result.extend(right[right_index:])

    return result

arr = [5, 2, 8, 3, 1]
print(merge_sort(arr))
```
### 4.4.2 详细解释说明
在这个代码实例中，我们实现了一个合并排序算法。我们定义了merge_sort函数来递归地对数组进行分割，直到每个子数组长度为1。然后，我们定义了merge函数来将两个有序子数组合并成一个有序数组。最后，我们使用merge_sort函数来对数组进行排序，并打印排序后的结果。

## 4.5 资深程序员
### 4.5.1 代码实例
```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

n = 5
print(factorial(n))
```
### 4.5.2 详细解释说明
在这个代码实例中，我们实现了一个阶乘函数。我们定义了factorial函数来递归地计算阶乘。如果n等于0，则返回1，否则返回n乘以阶乘(n-1)。最后，我们使用factorial函数计算5的阶乘，并打印结果。
