## 4.1 机械化生产的具体代码实例

```python
import time

def machine_production(input_data, output_data):
    start_time = time.time()
    for i in range(len(input_data)):
        output_data[i] = input_data[i] * 2
    end_time = time.time()
    return output_data, end_time - start_time

input_data = [1, 2, 3, 4, 5]
output_data = []

output_data, time_cost = machine_production(input_data, output_data)
print("输出数据:", output_data)
print("时间消耗:", time_cost)
```

在这个代码实例中，我们使用Python语言实现了一个机械化生产的功能。通过使用`time`模块，我们可以计算出生产过程的时间消耗。
