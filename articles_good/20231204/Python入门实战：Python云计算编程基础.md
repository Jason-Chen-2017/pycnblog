## 4.1 创建和管理虚拟机

在Python中，可以使用Python的云计算库（如Boto、Google Cloud SDK等）来创建和管理虚拟机。以下是一个使用Boto创建和管理虚拟机的示例代码：

```python
import boto3

# 创建EC2客户端
ec2 = boto3.client('ec2')

# 创建虚拟机
response = ec2.run_instances(
    ImageId='ami-0c94855ba95c617d2',
    MinCount=1,
    MaxCount=1,
    InstanceType='t2.micro'
)

# 获取虚拟机的ID
instance_id = response['Instances'][0]['InstanceId']

# 获取虚拟机的公网IP
public_ip = response['Instances'][0]['PublicIpAddress']

# 启动虚拟机
ec2.start_instances(InstanceIds=[instance_id])

# 停止虚拟机
ec2.stop_instances(InstanceIds=[instance_id])

# 终止虚拟机
ec2.terminate_instances(InstanceIds=[instance_id])
```

在上述代码中，我们首先创建了一个EC2客户端，然后使用`run_instances`方法来创建虚拟机。我们可以通过`InstanceId`来获取虚拟机的ID，通过`PublicIpAddress`来获取虚拟机的公网IP。最后，我们可以使用`start_instances`、`stop_instances`和`terminate_instances`方法来启动、停止和终止虚拟机。

## 4.2 创建和管理容器

在Python中，可以使用Python的容器库（如Docker SDK for Python、Kubernetes Python Client等）来创建和管理容器。以下是一个使用Docker SDK for Python创建和管理容器的示例代码：

```python
from docker import Client

# 创建Docker客户端
docker_client = Client(base_url='unix://var/run/docker.sock',
                        version='1.34',
                        timeout=10)

# 创建容器
response = docker_client.create_container(
    Image='python:3.7',
    Cmd='python -c "print('Hello World!')"',
    Name='my_container'
)

# 启动容器
docker_client.start(container=response['Id'])

# 停止容器
docker_client.stop(container=response['Id'])

# 删除容器
docker_client.remove_container(container=response['Id'], force=True)
```

在上述代码中，我们首先创建了一个Docker客户端，然后使用`create_container`方法来创建容器。我们可以通过`Id`来获取容器的ID。最后，我们可以使用`start`、`stop`和`remove_container`方法来启动、停止和删除容器。

## 4.5 创建和管理虚拟私有网络

在Python中，可以使用Python的虚拟私有网络库（如Boto、Google Cloud SDK等）来创建和管理虚拟私有网络。以下是一个使用Boto创建和管理虚拟私有网络的示例代码：

```python
import boto3

# 创建VPC客户端
vpc = boto3.client('ec2')

# 创建虚拟私有网络
response = vpc.create_vpc(
    CidrBlock='10.0.0.0/16'
)

# 创建子网
response = vpc.create_subnet(
    VpcId=response['VpcId'],
    CidrBlock='10.0.1.0/24'
)

# 创建网络接口
response = vpc.create_network_interface(
    SubnetId=response['SubnetId'],
    Description='My Network Interface'
)

# 分配IP地址
response = vpc.assign_private_ip_addresses(
    NetworkInterfaceId=response['NetworkInterfaceId'],
    PrivateIpAddresses=[
        '10.0.1.100'
    ]
)

# 删除虚拟私有网络
vpc.delete_vpc(VpcId=response['VpcId'])
```

在上述代码中，我们首先创建了一个VPC客户端，然后使用`create_vpc`方法来创建虚拟私有网络。我们可以通过`CidrBlock`来获取虚拟私有网络的IP地址范围，通过`SubnetId`来获取子网的ID。最后，我们可以使用`create_subnet`、`create_network_interface`和`assign_private_ip_addresses`方法来创建子网、创建网络接口和分配IP地址。
