# 2.核心概念与联系
## 什么是风险投资？
风险投资（Risk-adjusted investment）简称“灾备”，是一种投资策略，它将风险经验和系统性风险管理融入到资本回报预测中，以更好地控制风险、降低损失并提高回报。

风险投资包括三个主要元素：
- 投资组合：是指以一定的比例投资于不同类型的资产，而并非单纯追求一种特定资产的平均收益水平。
- 漏夜药策略：是指根据已知的短期和长期风险状况，制定出针对某个投资产品或风险类的最佳投资计划。
- 套期保值：是指利用保障机制来规避可能出现的损失风险。该机制可以设定偿还贷款或赎回风险资产的期限，并给予一定程度的补偿。

风险投资的目标是寻找能够创造长期价值的产品和服务，并将其投资于以取得比较理想的回报，同时承担相应的风险。

## 为什么要参与风险投资？
众所周知，有些事情无法避免，但风险投资却能够帮助我们摆脱这种困境，因为它帮助我们减少损失，增加回报。

风险投资最大的好处之一是，它能够帮助我们管理复杂的金融风险，而不是仅依赖信用评级等简单衡量标准。它有助于在投资过程中识别出那些潜在的不良现象，并采取措施减缓这些风险的蔓延，从而产生巨大的经济效益。另一个好处是，它使我们拥有参与投资组合中的每一项资产所带来的个人价值，这在个人的金融生涯中极具吸引力。

风险投资可以帮助我们在预防、控制和应对系统性风险方面发挥至关重要的作用。这包括对金融市场的可预测性、稳健性和适当性的洞察力、对投资者风险承受能力的分析、对风险的绩效管理，以及对外部市场波动的适应性反应能力。

## 风险投资的分类
1. 个股风险投资：指在个别股票上或一组相关的股票上进行的投资活动，目的是通过风险投资的方式来增强投资者的回报。

2. 资产配置型风险投资：即策略性基金，它以投资组合为基础，按照一定的规则来分配资金投资不同的资产，并根据风险偏好调整投资组合的仓位，目的是通过选择更加符合长期经济效益和资产配置风险管理目的的资产组合，最大限度地提高整个资金结构的回报。

3. 恒生指数基金：是一种跟踪恒生指数的国际跨国型私募基金，主要投资于股票、债券、黄金等资产。

4. ETF基金：是一种由世界范围内的投资者参与管理的开放式基金，通过自律性管理，通过对证券市场的研究和分析发现、运用科学方法建立和实施有效的投资策略，帮助投资者实现资产配置和投资理念的统一。

5. 社区投资：属于非营利性机构向客户提供专业化产品或服务的一种活动，并以提供个人化的服务或资产以获得回报的形式进行投资，由公共社区的成员提供资金支持。

## 风险投资模型及定性分析
风险投资模型是用来描述投资组合的风险特征及其资产预测模型。常见的风险投资模型有VaR、ES、CVaR和BETA模型。下面我以两种具体的模型——波特五小战模型和均值回复模型为例，分别介绍其模型框架、定性分析和操作指导。

### 波特五小战模型
波特五小战模型是一个投资组合理论模型，是由巴里克·波特和丹尼尔·李·斯蒂格勒设计的。模型认为，一个投资者的风险资本由五种基本要素组成，分别是：超常获利能力、财务杠杆、独立决策者、大量的未知信息和社会压力。也就是说，每个投资者在不同情况下，都需要在不同层次上进行防御性投资。

该模型提供了一种基于对每种要素的建模和评估，以及对因子组合在全球范围内分散性的研究，来分析投资组合的风险，包括最大风险、变化风险、管理风险和适用性风险等。

#### 模型框架
该模型将人类行为的因素分为两大类——超常获利能力和财务杠杆。超常获利能力又细分为身临其境的智慧和能力、厌恶痛苦的直觉和勇气、不顾现实的爱憎分明、热衷冒险的勤奋、强烈的商业冲动、鲁莽的狂想、敢做敢为的创新精神等。财务杠杆又分为强盗式投资和团队协作式投资。

模型的第二步是考虑投资者可能面临的约束条件——独立决策者、大量的未知信息和社会压力。投资者在这些约束条件下，为了达到最好的结果，需要做出更多的预测，以便做出合理的决策。

第三步是选择模型参数——预测误差、预测幅度和长期折扣。该模型假设一个准确且可靠的预测模型和低估误差。长期折扣用来表示投资者对未来获利的期望。

#### 操作指导
波特五小战模型虽然较为理论化，但是其操作过程还是十分容易理解。首先，确定超常获利能力、财务杠杆、独立决策者、大量的未知信息和社会压力。其次，定义一个预测误差、预测幅度和长期折扣。然后，进行持续预测、建立因子模型、动态调整预测、完善的内部信息、可控性的长期折扣和充分的社会压力。

总的来说，模型具有高度抽象化的特点，能够为投资者提供一个高层次的风险指标和策略建议。但是，在实践中，我们仍然需要结合实际情况和经验，辅以决策工具和风险控制手段，才能在风险投资领域生存。

### 均值回复模型
均值回复模型又叫“投资优势模型”，是由奥地利经济学家卡尔·萨根提出的一种风险管理模型。它的基本假设是，投资者投资超额回报的概率较低，但是由于个人风险和随机事件的影响，可能会产生反弹回升。所以，均值回复模型试图构建一个反映市场回报和投资者风险之间的关系的模型。

均值回复模型将投资回报视为两个部分，即合理的预期回报和超额回报。合理的预期回报意味着投资者以一种可预测的方式来预测未来的股价、长期收益和风险，并在这种预期下作出最好的投资决定。超额回报则是指投资者获得超过其预期回报的部分回报。

均值回复模型假设交易者可以由下列四个过程组成：1）寻找市场中具有可预测性的变量；2）进行有效的回报评估；3）形成优化决策；4）实施投资决策。模型通过研究市场价格，寻找具有长期回报的变量，并对每种变量的回报进行预测，来计算回报的均值。此外，模型还考虑到交易者的风险水平，以及交易者自身的回报期望和风险偏好，来量化交易者在每个投资时刻的投资回报。

#### 模型框架
均值回复模型的基础是利用历史数据计算每天的收益率，并将其作为资产的预测模型。模型考虑了六个基本要素，包括市场变动、信心、情绪、个人收入、经济状况和市场资源。其中，市场变动代表了信息的快速更新、公司股价的波动，以及社会事件的影响。信心则反映了投资者对未来长期回报的信心。情绪则代表投资者对于不同的股价变动反应的适度性。个人收入则反映了投资者对资产的预期风险。经济状况则代表了投资者是否能够接受政府政策的限制。市场资源则反映了投资者对现金的需求。

#### 操作指导
均值回复模型虽然有理论依据，但是它也存在一些操作上的挑战。首先，模型假设交易者能够准确、可靠地预测资产价格，这可能需要专业的知识和技巧。其次，模型预测的未来回报还受到交易者的情绪、个人收入、经济状况、市场资源和其他因素的影响。第三，模型没有考虑风险投资者如何控制个人风险，以及投资者如何根据预期回报和风险调整自己的投资组合。

总的来说，均值回复模型以预测模型的思路，认为交易者可以根据历史数据，预测未来的股价回报并进行有效的回报评估，从而帮助投资者完成资产配置。但是，模型仍然需要结合实际情况和经验，尤其是在风险投资领域生存。

接着，编写计算期望超额收益率的函数。这里计算的是百分比形式的超额回报率。

```python
def expected_excess_return(portfolio_returns):
    N = len(portfolio_returns)
    xbar = portfolio_returns.mean()

    cov = np.cov(portfolio_returns, ddof=0)[0][1]
    var_x = portfolio_returns.var()

    beta = cov/(var_x*(N-1))
    alpha = xbar-(beta*(xbar**2))/2
    
    return round(((alpha+(beta*(portfolio_returns**2)).sum())**(1/2)), 3)
```

这里计算的是五年、十年和二十年平均超额回报率。

```python
portfolio_returns = []
for name in ['Asset1', 'Asset2', 'Asset3']:
    ret = np.random.normal(loc=returns[name], scale=0.2, size=(5,))
    portfolio_returns += list(ret)

excess_returns = []
durations = [5, 10, 20]
for dur in durations:
    excess_returns.append(expected_excess_return(portfolio_returns[:dur]))

print('5 年、10 年、20 年平均超额回报率:', excess_returns)
```

4. 将β、α和初始资产回报率代入计算每个投资日的平均超额收益率。
5. 绘制一条曲线图，展示各周期平均超额回报率的变化。

参与风险投资有很多原因：

1. 提高资产的价值：风险投资能够提高资产的价值，这是因为风险投资以收益率作为标准，通过对风险资产的投资，能够建立起投资组合并在牺牲部分风险的前提下创造出可观的回报。

2. 加强投资者的风险管理能力：风险投资能够强化投资者的风险管理能力，因为它通过收集信息、分析数据、开发模型、实施策略、管理资金，在一定程度上能够规避风险并实现投资理念的统一。

3. 对投资的长远考虑：风险投资能够引导投资者进行长远的考虑，因为它通过建立投资组合来预测可能的资产组合回报，可以帮助投资者避免过度的风险，从而投资理念的最大化。

## 问：风险投资有哪些分类？

风险投资有个别股票、一组相关的股票的个股风险投资、资产配置型风险投资、恒生指数基金、ETF基金、社区投资等分类。

## 问：如何参与风险投资？

参与风险投资的方法有：

1. 订阅投资期刊：每月或者每季度阅读投资杂志，了解相关的投资讯息。

2. 浏览网络金融平台：寻找不同行业的专业机构，注册账号，查看各类投资产品的投资指南。

3. 在线阅读与咨询：找专业咨询师进行深度分析，提供适合自己的咨询服务。

4. 创建投资组合：制定个人投资计划，设置投资目标，根据实际情况制定不同的投资策略。

## 问：如何确定风险投资的起始金额？

个人投资的起始金额可以取决于个人的资产负债表、年龄、收入、工作积极性、经验、家庭情况、教育水平、资本现金及收益回报率等多种因素。为了获得最佳的投资回报，应努力扩大风险投资资金的大小，但不能盲目增加投资金额。

## 问：什么时候结束风险投资？

风险投资的终止条件一般有三个：

1. 收益预期达到一定水平后停止：当个人的预期收益达到了某一级别后，就不再参加风险投资了。

2. 经历一段时间的盈利后再退出：如果投资者已经在事业上取得成功，那么停止风险投资的时间也应该相应地提早。

3. 风险承受能力降低后退出：如果投资者的风险承受能力降低，不再愿意承受任何风险，则可以终止风险投资。

## 问：风险投资有哪些优点？

1. 有助于个人的金融安全：风险投资能够帮助投资者识别到市场风险并采取相应的策略，从而保障个人的金融安全。

2. 有助于促进经济发展：风险投资可以促进经济的发展，因为它为投资者创造可观的回报，特别是对于市场不确定性很强的行业和领域。

3. 有利于个人的个人权益：风险投资有利于个人的个人权益，因为它为个人提供了保障和储蓄的渠道。
