### 4.1 数据清洗与预处理
在一个销售报告中，我们需要对CRM平台中的数据进行清洗与预处理。以下是一个简单的Python代码实例：

```python
import pandas as pd

# 读取CRM平台数据
data = pd.read_csv('crm_data.csv')

# 数据筛选
data = data[data['sales'] > 0]

# 数据清洗
data = data.dropna()

# 数据转换
data['sales'] = data['sales'].astype('int')

# 数据归一化
data['sales'] = data['sales'] / data['sales'].max()
```

### 4.2 数据分析与报告
