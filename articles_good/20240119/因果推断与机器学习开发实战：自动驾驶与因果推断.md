### 4.1 潜在输入方法的代码实例
```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression

# 加载数据
data = pd.read_csv('data.csv')

# 选择因变量
y = data['target']

# 选择自变量
X = data.drop(['target'], axis=1)

# 训练模型
model = LinearRegression()
model.fit(X, y)

# 预测值
y_pred = model.predict(X)
```
