## 4. 具体最佳实践：代码实例和详细解释说明
在Go语言中，SQL数据库操作的最佳实践包括以下几个方面：

1. 使用`database/sql`包和相应的数据库驱动包。
2. 使用`sql.Open`函数创建数据库连接。
3. 使用`sql.Query`函数创建数据库查询。
4. 使用`sql.Rows`结构执行数据库查询。
5. 使用`sql.Close`函数关闭数据库连接。

以下是一个Go语言中的SQL数据库操作示例代码：

```go
package main

import (
	"database/sql"
	"fmt"
	"log"

	_ "github.com/go-sql-driver/mysql"
)

func main() {
	// 创建数据库连接
	db, err := sql.Open("mysql", "username:password@tcp(localhost:3306)/dbname")
	if err != nil {
		log.Fatal(err)
	}
	defer db.Close()

	// 创建数据库查询
	rows, err := db.Query("SELECT * FROM table_name")
	if err != nil {
		log.Fatal(err)
	}
	defer rows.Close()

	// 执行数据库查询
	for rows.Next() {
		var id int
		var name string
		var age int

		err := rows.Scan(&id, &name, &age)
		if err != nil {
			log.Fatal(err)
		}
		fmt.Printf("ID: %d, Name: %s, Age: %d\n", id, name, age)
	}

	// 关闭数据库连接
	err = db.Close()
	if err != nil {
		log.Fatal(err)
	}
}
```
