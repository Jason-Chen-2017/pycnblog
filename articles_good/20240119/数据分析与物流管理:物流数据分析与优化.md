#### 4.1.2 物流成本效率分析
```python
import pandas as pd
import numpy as np

data = pd.read_csv('material_cost.csv')
cost_data = data.groupby('cost_type')['cost'].sum()
cost_data['total'] = cost_data['transportation'] + cost_data['warehouse'] + cost_data['inventory']
cost_efficiency = cost_data['total'] / cost_data['sales']
cost_efficiency.plot(kind='line')
plt.show()
```
#### 4.1.3 物流成本预测
```python
import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression

data = pd.read_csv('material_cost.csv')
X = data[['sales']]
y = data['total']
model = LinearRegression()
model.fit(X, y)
y_pred = model.predict(X)
plt.scatter(X, y, label='Actual')
plt.plot(X, y_pred, color='red', label='Predicted')
plt.legend()
plt.show()
```
