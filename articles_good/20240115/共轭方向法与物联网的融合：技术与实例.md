## 4.1 共轴方向法的Python实现

在Python中，可以使用NumPy库来实现共轴方向法。以下是一个简单的共轴方向法的Python实现：

```python
import numpy as np

def conjugate_gradient(A, b, x0=None, tol=1e-9, max_iter=1000):
    if x0 is None:
        x0 = np.zeros_like(b)
    x = x0
    r = b - A @ x
    d = r / np.linalg.norm(r)
    k = 0
    while np.linalg.norm(r) > tol and k < max_iter:
        alpha = (r @ r) / (d @ A @ d)
        x = x + alpha * d
        r = r - alpha * A @ d
        d = r / np.linalg.norm(r)
        k += 1
    return x, k

# 示例

b = np.array([1, 1])
x, k = conjugate_gradient(A, b)
print("x:", x)
print("k:", k)
```
