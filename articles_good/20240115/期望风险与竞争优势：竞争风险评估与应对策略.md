## 4.1 期望风险评估算法实现

以下是一个简单的期望风险评估算法实现示例：

```python
import numpy as np

def risk_evaluation(data, weights, losses):
    risk = np.sum(weights * losses)
    return risk

# 示例数据
data = np.array([[1, 2], [3, 4], [5, 6]])
weights = np.array([0.5, 0.5])
losses = np.array([0.1, 0.2])

# 计算预期风险
risk = risk_evaluation(data, weights, losses)
print("预期风险:", risk)
```
