### 第二部分：技术面试专题

#### 第2章：数据结构与算法基础

##### 2.1 数据结构

**线性表**

- 定义：线性表是一种数据结构，其中的元素具有顺序关系。
- 特点：线性表中的元素只有一个前驱和一个后继。
- 常见实现方式：数组、链表。

**栈和队列**

- 栈：一种后进先出（LIFO）的数据结构。
- 队列：一种先进先出（FIFO）的数据结构。
- 常见实现方式：数组、链表。

**链表**

- 定义：链表由一系列节点组成，每个节点包含数据和指向下一个节点的指针。
- 特点：链表可以动态扩展，不需要固定的大小。
- 常见实现方式：单链表、双链表、循环链表。

**树与图**

- 树：一种层次结构，用于表示具有层级关系的数据。
- 图：一种由节点和边组成的数据结构，用于表示复杂关系。
- 常见实现方式：二叉树、红黑树、图矩阵。

##### 2.2 算法原理

**排序算法**

- 冒泡排序：比较相邻元素，若顺序错误则交换。
- 选择排序：找到未排序部分的最小元素，放到已排序部分的末尾。
- 插入排序：将未排序的元素插入到已排序部分的合适位置。

**搜索算法**

- 线性搜索：顺序遍历数组，找到目标元素。
- 二分搜索：有序数组中，通过递归或迭代查找目标元素。

**分治算法**

- 基本思想：将大问题分解成多个小问题，分别解决，最后合并结果。
- 应用场景：快速排序、合并排序等。

##### 2.3 算法实战

**动态规划问题**

- 背包问题：给定一组物品和其重量、价值，求最大化总价值的方法。
- 最长公共子序列：给定两个序列，求它们的最长公共子序列的长度。

**贪心算法问题**

- 最小公倍数（LCM）：求两个整数的最小公倍数。

**简单算法题实现**

- 题目：实现一个函数，判断一个整数是否是素数。
- 代码实现：

```python
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

# 测试
print(is_prime(11))  # 输出：True
print(is_prime(12))  # 输出：False
```

**中级算法题解析**

- 题目：实现一个函数，求两个有序数组的交集。
- 代码实现：

```python
def intersection(arr1, arr2):
    arr1.sort()
    arr2.sort()
    result = []
    i = j = 0
    while i < len(arr1) and j < len(arr2):
        if arr1[i] == arr2[j]:
            result.append(arr1[i])
            i += 1
            j += 1
        elif arr1[i] < arr2[j]:
            i += 1
        else:
            j += 1
    return result

# 测试
print(intersection([1, 2, 3, 4], [2, 4, 6, 8]))  # 输出：[2, 4]
```

**高级算法题挑战**

- 题目：实现一个函数，求最长公共子串。
- 代码实现：

```python
def longest_common_substring(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    max_len = 0
    max_end = 0
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                if dp[i][j] > max_len:
                    max_len = dp[i][j]
                    max_end = i
            else:
                dp[i][j] = 0
    return s1[max_end - max_len: max_end]

# 测试
print(longest_common_substring("abcdefg", "azced"))  # 输出："azced"
```

##### 3.2 算法编程题解析

**常见面试算法题解析**

- 题目：实现一个函数，求旋转数组的最小值。
- 代码实现：

```python
def min_array(nums):
    left, right = 0, len(nums) - 1
    while left < right:
        mid = (left + right) // 2
        if nums[mid] > nums[right]:
            left = mid + 1
        else:
            right = mid
    return nums[left]

# 测试
print(min_array([3, 4, 5, 1, 2]))  # 输出：1
```

- 代码实现（Python）：

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

# 测试
print(gcd(60, 48))  # 输出：12
```

**高级算法技巧应用**

- 题目：实现一个函数，求二叉树的直径。
- 代码实现：

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def diameter_of_binary_tree(root):
    def dfs(root):
        if not root:
            return 0
        left = dfs(root.left)
        right = dfs(root.right)
        max_len = max(left, right)
        nonlocal diameter
        diameter = max(diameter, left + right)
        return max_len + 1

    diameter = 0
    dfs(root)
    return diameter

# 测试
# 构造二叉树：1，2，4，4，5，6，7
root = TreeNode(1)
root.left = TreeNode(2)
root.right = TreeNode(4)
root.left.left = TreeNode(4)
root.left.right = TreeNode(5)
root.right.left = TreeNode(6)
root.right.right = TreeNode(7)

print(diameter_of_binary_tree(root))  # 输出：4
```
