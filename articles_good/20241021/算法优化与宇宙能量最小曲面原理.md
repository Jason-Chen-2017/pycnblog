**3.1 动态规划**

动态规划（Dynamic Programming，简称DP）是一种用于求解最优化问题的算法策略。它通过将问题分解为子问题，并在子问题上进行重叠计算来提高算法的效率。动态规划的核心思想是“最优子结构”和“子问题重叠”。

#### **最优子结构**

最优子结构是指一个最优解包含其子问题的最优解。也就是说，如果一个问题可以通过某种方式分解成多个子问题，那么每个子问题的最优解组合起来就能得到整个问题的最优解。

#### **子问题重叠**

子问题重叠是指不同子问题之间可能会出现相同的子问题，这意味着在求解过程中某些子问题会被多次计算。动态规划通过保存子问题的解来避免重复计算，从而提高算法效率。

#### **动态规划伪代码**

以下是一个动态规划求解斐波那契数列的伪代码示例：

```python
def dp_solution(input_data):
    # 初始化数组
    dp = [0] * len(input_data)
    # 初始化边界条件
    dp[0] = input_data[0]
    dp[1] = input_data[1]
    # 动态规划递推过程
    for i in range(2, len(input_data)):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[-1]
```

在动态规划中，通常可以使用递推关系式来表示子问题之间的关系。以下是一个典型的递推关系式：

$$

$$

其中，\(dp[i]\) 表示第 \(i\) 个子问题的最优解，\(input_data[i]\) 表示第 \(i\) 个输入数据。

#### **举例说明**
