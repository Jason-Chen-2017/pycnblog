## 3.1 热传导问题
热传导问题是物理仿真的一个典型例子，它涉及到热量在物体中的传播。热传导问题可以用热传导方程来描述，其形式如下：

$$
\rho C_p \frac{\partial T}{\partial t} = k \nabla^2 T + Q
$$

其中，$\rho$ 是材料密度，$C_p$ 是热容，$T$ 是温度，$t$ 是时间，$k$ 是热导率，$Q$ 是热源。

要解决这个方程，我们可以使用稠密网格方法（Finite Element Method, FEM）或有限差分方法（Finite Difference Method, FDM）等数值方法。这些方法通过将方程转换为离散形式，然后通过迭代求解得到温度分布。

## 3.2 流动问题
流动问题是物理仿真的另一个典型例子，它涉及到流体在物体中的运动。流动问题可以用纳西妥-霍尔方程组来描述，其形式如下：

$$
\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{v}) = 0
$$

$$
\rho (\frac{\partial \mathbf{v}}{\partial t} + \mathbf{v} \cdot \nabla \mathbf{v}) = -\nabla p + \mu \nabla^2 \mathbf{v} + \mathbf{F}
$$

其中，$\rho$ 是流体密度，$\mathbf{v}$ 是流体速度向量，$p$ 是压力，$\mu$ 是动力粘滞系数，$\mathbf{F}$ 是外力矢量。

要解决这个方程组，我们可以使用有限元方法（Finite Element Method, FEM）或有限差分方法（Finite Difference Method, FDM）等数值方法。这些方法通过将方程转换为离散形式，然后通过迭代求解得到流体速度和压力分布。

## 4.1 热传导问题

```python
import numpy as np

Cp = 1000.0  # 热容 (J/(kg*K))
k = 1.0  # 热导率 (W/(m*K))
Q = 0.0  # 热源 (W/m^3)
dt = 0.01  # 时间步长 (s)

```

接下来，我们需要构建一个有限差分网格，以便于求解问题：

```python

N = 100
x = np.linspace(0, L, N)
dx = x[1] - x[0]

```

然后，我们可以使用有限差分方法来求解问题：

```python
T = np.zeros(N)  # 温度分布
T[N//2] = 100.0  # 初始温度梯度

while T.max() > 1e-6:
    T_new = np.zeros(N)
    for i in range(1, N-1):
        T_new[i] = (T[i] + dt*k*((T[i+1] - 2*T[i] + T[i-1])/dx**2) + dt*Q*dx) / (rho*Cp*(1 + 2*dt*k/dx**2))
    T = T_new

print(T)

```

这个代码实例将求解一个一维热传导问题，并输出温度分布。可以看到，通过迭代求解，温度分布逐渐趋于稳定。


接下来，我们需要构建一个有限差分网格，以便于求解问题：

```python
L = 1.0  # 问题域的长度 (m)
N = 100
x = np.linspace(0, L, N)
dx = x[1] - x[0]

```

然后，我们可以使用有限差分方法来求解问题：

```python
v_new = np.zeros(3)
p_new = np.zeros(3)

while np.linalg.norm(v_new - v) > 1e-6:
    for i in range(1, N-1):
        for j in range(3):
            term1 = -p[j]*(1/(rho*dx))*(x[i+1] - x[i])
            term2 = -p[j]*(1/(rho*dx))*(x[i] - x[i-1])
            term3 = mu*(v[j]/(dx**2))*(x[i+1] - x[i])
            term4 = mu*(v[j]/(dx**2))*(x[i] - x[i-1])
            term5 = F[j]/(rho*dx)
            v_new[j][i] = v[j][i] + dt*(term1 - term2 + term3 - term4 + term5)
    v, p = v_new, p_new

print(v)

```

这个代码实例将求解一个一维流动问题，并输出流体速度和压力分布。可以看到，通过迭代求解，流体速度和压力分布逐渐趋于稳定。
