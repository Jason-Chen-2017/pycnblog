# 2.核心概念与联系

## 2.1 云计算基本概念

云计算是一种基于互联网的计算资源分配和共享模式，它使得用户可以在需要时轻松获取计算资源，而无需购买和维护自己的硬件和软件。云计算主要包括以下几个基本概念：

1. 虚拟化：虚拟化是云计算的基石，它使得多个虚拟的计算资源可以共享同一个物理资源。虚拟化可以将物理资源划分为多个虚拟资源，从而实现资源的共享和利用。

2. 软件即服务（SaaS）：软件即服务是云计算的一种应用模式，它使得用户可以在线获取软件服务，而无需购买和维护软件。软件即服务使得用户可以轻松获取软件服务，而无需担心软件的更新和维护。

3. 平台即服务（PaaS）：平台即服务是云计算的一种应用模式，它使得用户可以在线获取平台服务，而无需购买和维护平台硬件和软件。平台即服务使得用户可以轻松获取平台服务，而无需担心平台的更新和维护。

4. 基础设施即服务（IaaS）：基础设施即服务是云计算的一种应用模式，它使得用户可以在线获取基础设施服务，而无需购买和维护基础设施硬件和软件。基础设施即服务使得用户可以轻松获取基础设施服务，而无需担心基础设施的更新和维护。
