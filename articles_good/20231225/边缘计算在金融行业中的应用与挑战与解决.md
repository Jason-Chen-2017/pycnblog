$$
P(y=1|x;\theta)=sigmoid(w^Tx+b)
$$
其中，$P(y=1|x;\theta)$ 是输出概率，$sigmoid$ 是sigmoid函数，$w$ 是权重向量，$x$ 是输入特征向量，$b$ 是偏置项。
2. 支持向量机：支持向量机是一种多分类算法，它可以根据输入特征预测输出类别。支持向量机的数学模型公式如下：
$$
minimize\frac{1}{2}w^Tw+C\sum_{i=1}^n\xi_i

$$
其中，$w$ 是权重向量，$x$ 是输入特征向量，$b$ 是偏置项，$C$ 是正则化参数，$\xi_i$ 是松弛变量。
3. 决策树：决策树是一种分类算法，它可以根据输入特征预测输出类别。决策树的数学模型公式如下：
$$
if\ x_1\ is\ A_1\ then\ output\ A_2
$$
其中，$x_1$ 是输入特征，$A_1$ 是条件，$A_2$ 是输出类别。

```python
import numpy as np

class LogisticRegression:
    def __init__(self, learning_rate=0.01, num_iterations=1000):
        self.learning_rate = learning_rate
        self.num_iterations = num_iterations

    def fit(self, X, y):
        m, n = X.shape
        self.weights = np.zeros(n)
        for _ in range(self.num_iterations):
            linear_model = np.dot(X, self.weights)
            y_predicted = self.sigmoid(linear_model)
            dw = (1 / m) * np.dot(X.T, (y_predicted - y))
            self.weights -= self.learning_rate * dw

    def predict(self, X):
        linear_model = np.dot(X, self.weights)
        y_predicted = self.sigmoid(linear_model)
        return y_predicted

    def sigmoid(self, z):
        return 1 / (1 + np.exp(-z))
```

```python
import numpy as np

class KMeans:
    def __init__(self, num_clusters=3):
        self.num_clusters = num_clusters

    def fit(self, X):
        self.centroids = np.max(X, axis=0)
        for _ in range(100):
            self.distances = np.sqrt(np.sum((X - self.centroids[:, np.newaxis]) ** 2, axis=2))
            self.labels = np.argmin(self.distances, axis=0)
            self.centroids = np.array([X[self.labels == i].mean(axis=0) for i in range(self.num_clusters)])

    def predict(self, X):
        self.distances = np.sqrt(np.sum((X - self.centroids[:, np.newaxis]) ** 2, axis=2))
        return self.labels[np.argmin(self.distances, axis=0)]
```
