```python

import matplotlib.pyplot as plt

# 信号生成
fs = 1000

plt.legend()
plt.show()
```

以Python为例，我们来实现一个非线性最小二乘估计的代码实例。

```python
import numpy as np

# 信号生成
fs = 1000
Ts = 1/fs
t = np.arange(0, 1, Ts)
x = 2 * np.sin(2 * np.pi * 50 * t) + 0.5 * np.sin(2 * np.pi * 100 * t)

# 非线性模型
def model(a, b):
    return a * np.sin(2 * np.pi * 50 * t) + b * np.sin(2 * np.pi * 100 * t)

# 非线性最小二乘估计
a_est = np.linalg.lstsq(x.reshape(-1, 1), model(1, 1), bounds=(0, None))[0][0]
b_est = np.linalg.lstsq(x.reshape(-1, 1), model(1, 1), bounds=(None, 1))[0][0]

# 估计结果绘制
plt.figure()
plt.plot(t, x, label='原信号')
plt.plot(t, model(a_est, b_est), label='非线性最小二乘估计')
plt.legend()
plt.show()
```

在上述代码中，我们首先生成了一个频率为50Hz和100Hz的信号，并将其加噪。然后，我们使用多项式模型实现了一个非线性模型，并使用非线性最小二乘估计算法估计模型参数a和b。最后，我们绘制了原信号和非线性模型的拟合结果，可以看到非线性最小二乘估计成功地估计了模型参数。
