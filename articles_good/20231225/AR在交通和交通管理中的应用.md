## 4.1 交通状况实时监控

### 4.1.1 算法实现

```python
import cv2
import numpy as np

def count_cars(frame):
    cars = 0
    for i in range(frame.shape[0]):
        for j in range(frame.shape[1]):
            if frame[i, j, 0] > 0:
                cars += 1
    return cars

def main():
    cap = cv2.VideoCapture('traffic_video.mp4')
    while cap.isOpened():
        ret, frame = cap.read()
        if not ret:
            break
        cars = count_cars(frame)
        print(f'Cars: {cars}')
        cv2.imshow('frame', frame)
        if cv2.waitKey(1) & 0xFF == ord('q'):
            break
    cap.release()
    cv2.destroyAllWindows()

if __name__ == '__main__':
    main()
```

### 4.1.2 解释说明

在上述代码中，我们首先使用OpenCV库读取了交通视频。然后，我们定义了一个`count_cars`函数，该函数通过遍历视频帧的每个像素点来计算车辆数量。最后，我们使用`cv2.imshow`函数显示视频帧，并使用`cv2.waitKey`函数监听用户输入，如果用户按下`q`，程序将退出。
