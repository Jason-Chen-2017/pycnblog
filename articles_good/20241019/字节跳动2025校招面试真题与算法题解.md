以下是一些典型的经典算法题示例：

1. **最长公共子序列（LCS）**：

   题目描述：给定两个字符串`str1`和`str2`，找出它们的最长公共子序列。

   示例：

   ```python
   str1 = "ABCD"
   str2 = "ACDF"
   ```

   解答：

   ```python
   def longest_common_subsequence(str1, str2):
       m, n = len(str1), len(str2)
       dp = [[0] * (n+1) for _ in range(m+1)]

       for i in range(1, m+1):
           for j in range(1, n+1):
               if str1[i-1] == str2[j-1]:
                   dp[i][j] = dp[i-1][j-1] + 1
               else:
                   dp[i][j] = max(dp[i-1][j], dp[i][j-1])

       return dp[m][n]

   print(longest_common_subsequence("ABCD", "ACDF"))
   ```

2. **最小生成树（MST）**：

   题目描述：给定一个无向图，求出其最小生成树。

   示例：

   ```python
   graph = [[0, 2, 4, 6], [2, 0, 1, 3], [4, 1, 0, 5], [6, 3, 5, 0]]
   ```

   解答：

   ```python
   import heapq

   def prim_mst(graph):
       mst = []
       visited = [False] * len(graph)
       heapq.heapify(graph[0])

       while graph:
           weight, u, v = heapq.heappop(graph[0])
           if visited[u] or visited[v]:
               continue
           mst.append((u, v, weight))
           visited[u] = visited[v] = True
           for edge in graph[v]:
               if not visited[edge[1]]:
                   heapq.heappush(graph[0], edge)

       return mst

   print(prim_mst([[0, 2, 4, 6], [2, 0, 1, 3], [4, 1, 0, 5], [6, 3, 5, 0]]))
   ```

3. **硬币找零问题**：

   题目描述：给定一组硬币的面值和总金额，求出最少需要多少枚硬币来凑齐总金额。

   示例：

   ```python
   coins = [1, 5, 10, 25]
   amount = 30
   ```

   解答：

   ```python
   def coin_change(coins, amount):
       dp = [float('inf')] * (amount + 1)
       dp[0] = 0
       for coin in coins:
           for i in range(coin, amount + 1):
               dp[i] = min(dp[i], dp[i - coin] + 1)

       return dp[amount] if dp[amount] != float('inf') else -1

   print(coin_change([1, 5, 10, 25], 30))
   ```

**题目描述**：给定一个整数数组，找出其中的峰值元素。峰值元素是指其值大于左右相邻元素的元素。

**示例**：

```python
nums = [1, 2, 3, 1]
```

**解题思路**：

1. **直接遍历**：遍历数组，找出峰值元素；
2. **二分查找**：利用二分查找，降低时间复杂度。

**实战案例**：

```python
def find_peak_element(nums):
    left, right = 0, len(nums) - 1

    while left < right:
        mid = (left + right) // 2
        if nums[mid] > nums[mid + 1]:
            right = mid
        else:
            left = mid + 1

    return left

print(find_peak_element([1, 2, 3, 1]))
```

1. **排序算法**：

   - **冒泡排序**：比较相邻元素，若逆序则交换，重复该过程直到排序完成。

   ```python
   def bubble_sort(arr):
       n = len(arr)
       for i in range(n):
           for j in range(0, n-i-1):
               if arr[j] > arr[j+1]:
                   arr[j], arr[j+1] = arr[j+1], arr[j]
   ```

   - **选择排序**：每次从剩余元素中找到最小元素，将其放到已排序序列的末尾。

   ```python
   def selection_sort(arr):
       n = len(arr)
       for i in range(n):
           min_idx = i
           for j in range(i+1, n):
               if arr[j] < arr[min_idx]:
                   min_idx = j
           arr[i], arr[min_idx] = arr[min_idx], arr[i]
   ```

   - **插入排序**：将元素插入到已排序序列的正确位置，以完成排序。

   ```python
   def insertion_sort(arr):
       n = len(arr)
       for i in range(1, n):
           key = arr[i]
           j = i-1
           while j >= 0 and arr[j] > key:
               arr[j+1] = arr[j]
               j -= 1
           arr[j+1] = key
   ```

2. **查找算法**：

   - **二分查找**：在有序数组中，通过不断缩小区间，找到目标元素。

   ```python
   def binary_search(arr, target):
       left, right = 0, len(arr) - 1
       while left <= right:
           mid = (left + right) // 2
           if arr[mid] == target:
               return mid
           elif arr[mid] < target:
               left = mid + 1
           else:
               right = mid - 1
       return -1
   ```

#### 5.2 动态规划与贪心算法

动态规划（Dynamic Programming，简称DP）和贪心算法（Greedy Algorithm）是解决复杂问题的两大算法策略。

1. **动态规划**：

   动态规划通常用于解决最优子结构问题和重叠子问题。其核心思想是：将原问题分解为子问题，通过保存子问题的解，避免重复计算。

   - **斐波那契数列**：

     ```python
     def fib(n):
         dp = [0] * (n+1)
         dp[1] = 1
         for i in range(2, n+1):
             dp[i] = dp[i-1] + dp[i-2]
         return dp[n]
     ```

   - **最长公共子序列（LCS）**：

     ```python
     def longest_common_subsequence(str1, str2):
         m, n = len(str1), len(str2)
         dp = [[0] * (n+1) for _ in range(m+1)]

         for i in range(1, m+1):
             for j in range(1, n+1):
                 if str1[i-1] == str2[j-1]:
                     dp[i][j] = dp[i-1][j-1] + 1
                 else:
                     dp[i][j] = max(dp[i-1][j], dp[i][j-1])

         return dp[m][n]
     ```

2. **贪心算法**：

   贪心算法通过每次选择当前最优解，来逐步达到整体最优解。其核心思想是：在每个决策点上选择局部最优解，以期望最终达到全局最优解。

   - **活动选择问题**：

     ```python
     def activity_selection(s, f):
         n = len(s)
         max_activities = 0
         last_end = -1

         for i in range(n):
             if s[i] > last_end:
                 max_activities += 1
                 last_end = f[i]

         return max_activities
     ```

   - **硬币找零问题**：

     ```python
     def coin_change(coins, amount):
         dp = [float('inf')] * (amount + 1)
         dp[0] = 0
         for coin in coins:
             for i in range(coin, amount + 1):
                 dp[i] = min(dp[i], dp[i - coin] + 1)

         return dp[amount] if dp[amount] != float('inf') else -1
     ```

#### 5.3 树与图算法

树（Tree）和图（Graph）是两种重要的数据结构，广泛应用于算法设计。

1. **树**：

   - **二叉树**：每个节点最多有两个子节点。

   ```python
   class TreeNode:
       def __init__(self, val=0, left=None, right=None):
           self.val = val
           self.left = left
           self.right = right
   ```

   - **二叉搜索树（BST）**：左子树的所有节点值小于根节点，右子树的所有节点值大于根节点。

   ```python
   def insert(root, val):
       if root is None:
           return TreeNode(val)
       if val < root.val:
           root.left = insert(root.left, val)
       else:
           root.right = insert(root.right, val)
       return root
   ```

2. **图**：

   - **邻接矩阵**：用二维数组表示图，行和列分别表示顶点，元素值表示边。

   ```python
   graph = [
       [0, 1, 1],
       [1, 0, 1],
       [1, 1, 0]
   ]
   ```

   - **邻接表**：用一维数组表示图，数组每个元素对应一个顶点，元素值是一个链表，链表中的节点表示与该顶点相邻的其他顶点。

   ```python
   graph = {
       0: [1, 2],
       1: [0, 2],
       2: [0, 1]
   }
   ```

3. **树与图算法**：

   - **深度优先搜索（DFS）**：

     ```python
     def dfs(root):
         if root is None:
             return
         print(root.val)
         dfs(root.left)
         dfs(root.right)
     ```

   - **广度优先搜索（BFS）**：

     ```python
     from collections import deque

     def bfs(graph, start):
         queue = deque([start])
         visited = set()

         while queue:
             node = queue.popleft()
             if node not in visited:
                 print(node.val)
                 visited.add(node)
                 for neighbor in graph[node]:
                     queue.append(neighbor)
     ```

   - **拓扑排序**：

     ```python
     def topology_sort(graph):
         in_degrees = [0] * len(graph)
         for node in graph:
             for neighbor in graph[node]:
                 in_degrees[neighbor] += 1

         queue = deque([node for node, _ in enumerate(in_degrees) if in_degrees[node] == 0])

         while queue:
             node = queue.popleft()
             print(node.val)
             for neighbor in graph[node]:
                 in_degrees[neighbor] -= 1
                 if in_degrees[neighbor] == 0:
                     queue.append(neighbor)
     ```

#### 7.1 编码实战案例

以下是一个常见的编程题目示例，要求实现一个函数，计算两个字符串的最长公共子序列。

**题目描述**：给定两个字符串`str1`和`str2`，找出它们的最长公共子序列。

**输入**：

```
str1 = "ABCD"
str2 = "ACDF"
```

**输出**：

```
"ACD"
```

**解题思路**：

使用动态规划算法求解最长公共子序列。定义一个二维数组`dp`，其中`dp[i][j]`表示`str1`的前`i`个字符和`str2`的前`j`个字符的最长公共子序列的长度。

**解题步骤**：

1. 初始化二维数组`dp`，其中`dp[0][j] = dp[i][0] = 0`；
2. 遍历字符串`str1`和`str2`的每个字符，更新`dp`数组；
3. 如果`str1[i-1] == str2[j-1]`，则`dp[i][j] = dp[i-1][j-1] + 1`；
4. 否则，`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`；
5. 返回`dp[m][n]`，其中`m`和`n`分别为字符串`str1`和`str2`的长度。

**代码实现**：

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if str1[i-1] == str2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[m][n]

print(longest_common_subsequence("ABCD", "ACDF"))
```

   解法：使用动态规划算法。

   ```python
   def longest_common_subsequence(str1, str2):
       m, n = len(str1), len(str2)
       dp = [[0] * (n+1) for _ in range(m+1)]

       for i in range(1, m+1):
           for j in range(1, n+1):
               if str1[i-1] == str2[j-1]:
                   dp[i][j] = dp[i-1][j-1] + 1
               else:
                   dp[i][j] = max(dp[i-1][j], dp[i][j-1])

       return dp[m][n]
   ```

2. **最小生成树（MST）**：

   题目描述：给定一个无向图，求出其最小生成树。

   解法：使用Prim算法。

   ```python
   import heapq

   def prim_mst(graph):
       mst = []
       visited = [False] * len(graph)
       heapq.heapify(graph[0])

       while graph:
           weight, u, v = heapq.heappop(graph[0])
           if visited[u] or visited[v]:
               continue
           mst.append((u, v, weight))
           visited[u] = visited[v] = True
           for edge in graph[v]:
               if not visited[edge[1]]:
                   heapq.heappush(graph[0], edge)

       return mst
   ```

   ```python
   def coin_change(coins, amount):
       dp = [float('inf')] * (amount + 1)
       dp[0] = 0
       for coin in coins:
           for i in range(coin, amount + 1):
               dp[i] = min(dp[i], dp[i - coin] + 1)

       return dp[amount] if dp[amount] != float('inf') else -1
   ```

4. **二分查找**：

   ```python
   def binary_search(arr, target):
       left, right = 0, len(arr) - 1
       while left <= right:
           mid = (left + right) // 2
           if arr[mid] == target:
               return mid
           elif arr[mid] < target:
               left = mid + 1
           else:
               right = mid - 1
       return -1
   ```

5. **排序算法**：

   题目描述：实现一个排序算法，对数组进行排序。

   解法：实现快速排序。

   ```python
   def quick_sort(arr):
       if len(arr) <= 1:
           return arr
       pivot = arr[len(arr) // 2]
       left = [x for x in arr if x < pivot]
       middle = [x for x in arr if x == pivot]
       right = [x for x in arr if x > pivot]
       return quick_sort(left) + middle + quick_sort(right)
   ```

   ```python
           return max(left_height, right_height) + 1
   ```
