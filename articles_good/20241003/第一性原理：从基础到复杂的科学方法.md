### 4.2. 公式讲解

以下是对第一性原理中的几个关键公式进行详细讲解：

1. **牛顿定律**：

   $$ F = ma $$

   其中，$F$ 表示力，$m$ 表示质量，$a$ 表示加速度。

2. **麦克斯韦方程**：

   $$ \nabla \cdot E = \frac{\rho}{\varepsilon_0} $$

   $$ \nabla \cdot B = 0 $$

   $$ \nabla \times E = -\frac{\partial B}{\partial t} $$

   $$ \nabla \times B = \mu_0 J + \mu_0 \varepsilon_0 \frac{\partial E}{\partial t} $$

   其中，$E$ 表示电场，$B$ 表示磁场，$\rho$ 表示电荷密度，$\varepsilon_0$ 表示真空电容率，$\mu_0$ 表示真空磁导率，$J$ 表示电流密度。

3. **薛定谔方程**：

   $$ i\hbar \frac{\partial \psi}{\partial t} = -\frac{\hbar^2}{2m} \nabla^2 \psi + V \psi $$

   其中，$i$ 表示虚数单位，$\hbar$ 表示约化普朗克常数，$m$ 表示粒子质量，$\psi$ 表示波函数，$V$ 表示势能。

### 5.2. 源代码详细实现和代码解读

以下是一个基于第一性原理的简单例子：计算一个物体在重力作用下的运动轨迹。

**代码实现：**

```python
import numpy as np
import matplotlib.pyplot as plt

def gravity_trajectory(v0, g, t_max):
    """
    计算物体在重力作用下的运动轨迹。

    参数：
    - v0：初始速度
    - g：重力加速度
    - t_max：时间最大值

    返回：
    - 速度时间序列
    - 位置时间序列
    """

    # 时间步长
    dt = 0.01

    # 时间序列
    t = np.arange(0, t_max + dt, dt)

    # 计算速度和位置
    v = v0 - g * t
    x = v0 * t - 0.5 * g * t ** 2

    return v, x

# 参数设置
v0 = 10  # 初始速度
g = 9.8  # 重力加速度
t_max = 10  # 时间最大值

# 计算运动轨迹
v, x = gravity_trajectory(v0, g, t_max)

# 绘制速度-时间图
plt.plot(t, v)
plt.xlabel("时间 (s)")
plt.ylabel("速度 (m/s)")
plt.title("速度-时间图")
plt.grid()
plt.show()

# 绘制位置-时间图
plt.plot(t, x)
plt.xlabel("时间 (s)")
plt.ylabel("位置 (m)")
plt.title("位置-时间图")
plt.grid()
plt.show()

```
