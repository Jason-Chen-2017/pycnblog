```python
import numpy as np

def linear_regression(X, y):
    # 计算斜率和截距
    m = np.mean(X)
    b = np.mean(y - X)
    return m, b

def predict(X, m, b):
    # 预测输出值
    return X * m + b

# 训练数据集
X = np.array([0, 1, 2, 3, 4])
y = np.array([1, 2, 3, 4, 5])

# 计算斜率和截距
m, b = linear_regression(X, y)

# 预测输出值
y_pred = predict(X, m, b)

# 输出预测结果
print("预测结果：", y_pred)
```

2. **函数实现**：`linear_regression` 函数用于计算斜率和截距，`predict` 函数用于根据斜率和截距预测输出值。
