## 2. 核心概念与联系
### 2.1 矩阵理论

矩阵理论是研究矩阵的运算、性质及其应用的一门学科。在定常线性系统分析中，矩阵理论的核心概念包括矩阵的加减法、乘法、逆矩阵、行列式等。

### 2.2 定常线性系统

定常线性系统是指系统参数不随时间变化的线性系统。其数学模型通常可以表示为：

$$
\dot{x} = Ax + Bu
$$

其中，$x$ 为系统状态向量，$u$ 为输入向量，$A$ 和 $B$ 为系统矩阵。

### 2.3 可观测性与可控性

可观测性与可控性是描述系统性能的两个重要概念。

- **可观测性**：如果系统状态可以通过输出信号完全描述，则称该系统为可观测的。
- **可控性**：如果系统状态可以通过输入信号完全控制，则称该系统为可控的。

可观测性与可控性可以通过分析系统的传递矩阵 $C = BCA^{-1}$ 判断。

## 3. 核心算法原理 & 具体操作步骤
### 3.1 算法原理概述

判断定常线性系统的可观测性与可控性的核心算法包括以下几种：

- **能控规范形**：通过将系统矩阵 $A$ 进行行变换，将其转化为能控规范形，判断系统是否可控。
- **能观规范形**：通过将系统矩阵 $A$ 进行行变换，将其转化为能观规范形，判断系统是否可观测。
- **李雅普诺夫稳定性理论**：通过李雅普诺夫函数判断系统的稳定性。

### 3.2 算法步骤详解

以下以能控规范形算法为例，介绍判断系统可控性的具体步骤：

1. 对系统矩阵 $A$ 进行行变换，将其转化为上三角矩阵。
2. 在上三角矩阵中，对每一列进行行变换，使其变为形如 $\begin{pmatrix} 1 & 0 & 0 & \cdots & 0 \\ 0 & 1 & 0 & \cdots & 0 \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & 0 & \cdots & 1 \end{pmatrix}$ 的规范形。
3. 如果规范形中的对角元素均非零，则系统可控；否则，系统不可控。

### 3.3 算法优缺点

- **能控规范形**：算法简单，易于实现，但可能需要大量计算。
- **能观规范形**：算法简单，易于实现，但可能需要大量计算。
- **李雅普诺夫稳定性理论**：算法复杂，但能够判断系统的稳定性，具有一定的通用性。

### 3.4 算法应用领域

以上算法在控制系统、信号处理、通信系统等领域具有广泛的应用。

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 数学模型构建

以下为一个二阶定常线性系统的例子：

$$
\dot{x}_1 = x_2
$$

$$
\dot{x}_2 = -x_1 - x_2 + u
$$

其中，$x_1$ 和 $x_2$ 分别表示系统的状态变量，$u$ 为输入信号。

### 4.2 公式推导过程

将上述方程组表示为矩阵形式：

$$
\dot{x} = \begin{pmatrix} 0 & 1 \\ -1 & -1 \end{pmatrix} x + \begin{pmatrix} 0 \\ 1 \end{pmatrix} u
$$

### 4.3 案例分析与讲解

首先，判断系统的可控性。根据能控规范形算法，对系统矩阵 $A$ 进行行变换：

$$
\begin{pmatrix} 0 & 1 \\ -1 & -1 \end{pmatrix} \xrightarrow{\text{行变换}} \begin{pmatrix} 1 & 0 \\ 1 & -1 \end{pmatrix}
$$

由于规范形中的对角元素均非零，因此该系统可控。

接下来，判断系统的可观测性。根据能观规范形算法，对系统矩阵 $A$ 进行行变换：

$$
\begin{pmatrix} 0 & 1 \\ -1 & -1 \end{pmatrix} \xrightarrow{\text{行变换}} \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}
$$

由于规范形中的对角元素均非零，因此该系统可观测。

```python
import numpy as np
from scipy.linalg import eig

def is_controllable(A):
    n = A.shape[0]
    for i in range(n):
        A_i = np.eye(n) - A[:, :i+1] @ np.linalg.inv(A[:i+1, :i+1])
        if np.linalg.det(A_i) == 0:
            return False
    return True

def is_observable(A):
    n = A.shape[0]
    for i in range(n):
        A_i = np.eye(n) - A[:i+1, :i+1] @ np.linalg.inv(A[:i+1, :i+1])
        if np.linalg.det(A_i) == 0:
            return False
    return True

# 示例系统
A = np.array([[0, 1], [-1, -1]])
B = np.array([[0], [1]])

print("Is the system controllable?", is_controllable(A))
print("Is the system observable?", is_observable(A))
```
