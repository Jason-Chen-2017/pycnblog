#### 1. 太空探索基础知识

##### 问题1：什么是行星？

**答案：** 行星是围绕恒星运行的天体，具有足够的质量使其成为球形，并且已经摆脱了行星环的影响。

**解析：** 行星是太阳系中的基本组成单位，通常围绕恒星运行，具有稳定轨道。它们通过引力与其他天体相互作用，并按照开普勒定律运行。

##### 问题2：如何探测行星？

**答案：** 通过发射探测器、使用望远镜观测、分析行星大气成分等方法探测行星。

**解析：** 探测行星的方法包括直接观测、间接观测和探测器实地探测。直接观测使用望远镜获取行星的表面图像和大气数据；间接观测通过分析行星的引力效应和光谱特性来推断其存在。

#### 2. 行星探测任务解析

##### 问题3：火星探测器的任务是什么？

**答案：** 火星探测器的任务包括研究火星的地质结构、气候、大气成分，以及寻找生命的迹象。

**解析：** 火星探测器旨在收集关于火星的宝贵数据，以帮助科学家了解这颗红色星球的过去和现在。这些数据有助于制定未来的探索计划和寻找潜在的人类栖息地。

##### 问题4：木星探测器的主要目标是什么？

**答案：** 木星探测器的主要目标是研究木星的大气层、磁场、极光，以及其卫星的表面特征。

**解析：** 木星是太阳系中最大的行星，其拥有复杂的大气层和磁层。探测器的研究有助于揭示木星的内部结构、磁场起源以及其卫星的地质活动。

##### 问题11：编写一个程序，模拟行星轨道计算。

**答案：** 
```python
import math

def calculate_orbit(semi_major_axis, eccentricity):
    # 计算行星轨道的周期
    period = 2 * math.pi * semi_major_axis * math.sqrt(1 / (1 - eccentricity**2))
    return period

# 示例：地球的轨道
earth_semi_major_axis = 149.6e6 * 1000  # 单位：公里
earth_eccentricity = 0.0167
earth_period = calculate_orbit(earth_semi_major_axis, earth_eccentricity)
print("地球轨道周期为：", earth_period, "秒")

# 示例：火星的轨道
mars_semi_major_axis = 227.9e6 * 1000  # 单位：公里
mars_eccentricity = 0.0934
mars_period = calculate_orbit(mars_semi_major_axis, mars_eccentricity)
print("火星轨道周期为：", mars_period, "秒")
```

**解析：** 此程序计算行星轨道的周期，基于开普勒第三定律，即行星轨道周期的平方与轨道半长轴的立方成正比。该计算可以帮助科学家预测行星的运动轨迹。

##### 问题12：编写一个程序，分析行星探测器的数据。

**答案：**
```python
import pandas as pd

def analyze_data(data):
    # 加载数据
    df = pd.read_csv(data)
    
    # 计算大气温度的平均值和标准差
    temp_avg = df['temperature'].mean()
    temp_std = df['temperature'].std()
    
    # 计算大气压力的平均值和标准差
    pressure_avg = df['pressure'].mean()
    pressure_std = df['pressure'].std()
    
    return temp_avg, temp_std, pressure_avg, pressure_std

# 示例：分析火星探测器的数据
mars_data = "mars_data.csv"
temp_avg, temp_std, pressure_avg, pressure_std = analyze_data(mars_data)
print("火星大气温度平均值：", temp_avg)
print("火星大气温度标准差：", temp_std)
print("火星大气压力平均值：", pressure_avg)
print("火星大气压力标准差：", pressure_std)
```

**解析：** 此程序使用 Pandas 库加载和解析 CSV 数据，计算大气温度和压力的平均值和标准差，帮助科学家分析行星探测数据。
