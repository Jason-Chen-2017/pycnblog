### 算法编程题库

#### 5. 群体智慧算法：粒子群优化（PSO）

**题目：** 请实现一个粒子群优化（PSO）算法，用于求解一个简单的函数优化问题。

**答案：** 粒子群优化（PSO）算法是一种基于群体智慧的优化算法，通过模拟鸟群觅食行为，实现搜索空间的优化。以下是一个基于 Python 的粒子群优化算法示例：

```python
import numpy as np

# 粒子群优化算法
def particle_swarm_optimization(func, n_particles, max_iterations, w=0.5, c1=1, c2=2):
    # 初始化粒子群
    particles = np.random.uniform(-5, 5, (n_particles, 2))
    velocities = np.zeros((n_particles, 2))
    p_best = particles.copy()
    g_best = particles[0]

    # 迭代过程
    for _ in range(max_iterations):
        for i in range(n_particles):
            # 计算适应度
            fitness = func(particles[i])

            # 更新个体最优解
            if fitness < func(p_best[i]):
                p_best[i] = particles[i]

            # 更新全局最优解
            if fitness < func(g_best):
                g_best = particles[i]

        # 更新速度和位置
        velocities = w * velocities + c1 * np.random.random() * (p_best[i] - particles[i]) + c2 * np.random.random() * (g_best - particles[i])
        particles += velocities

    return g_best

# 函数优化问题
def rosenbrock(x):
    return sum(100 * (x[1:] - x[:-1]**2)**2 + (1 - x[:-1])**2)

# 求解优化问题
best_solution = particle_swarm_optimization(rosenbrock, 50, 100)
print("最优解：", best_solution)
```

#### 6. 群体智慧算法：遗传算法（GA）

**题目：** 请实现一个遗传算法（GA），用于求解一个简单的函数优化问题。

**答案：** 遗传算法（GA）是一种基于自然进化过程的优化算法，通过选择、交叉、变异等操作，实现搜索空间的优化。以下是一个基于 Python 的遗传算法示例：

```python
import numpy as np

# 遗传算法
def genetic_algorithm(func, n_particles, max_iterations, crossover_rate=0.8, mutation_rate=0.1):
    # 初始化种群
    population = np.random.uniform(-5, 5, (n_particles, 2))

    # 迭代过程
    for _ in range(max_iterations):
        # 计算适应度
        fitness = np.array([func(individual) for individual in population])

        # 选择
        selected = np.random.choice(np.arange(n_particles), size=int(n_particles/2), replace=False, p=fitness/fitness.sum())

        # 交叉
        offspring = np.zeros((n_particles, 2))
        for i in range(n_particles):
            if np.random.random() < crossover_rate:
                parent1, parent2 = selected[i//2*2], selected[i//2*2+1]
                offspring[i] = (population[parent1] + population[parent2]) / 2
            else:
                offspring[i] = population[i]

        # 变异
        for i in range(n_particles):
            if np.random.random() < mutation_rate:
                offspring[i] += np.random.normal(0, 1, size=2)

        # 更新种群
        population = offspring

    # 返回最优解
    best_fitness = np.min(fitness)
    best_solution = population[np.argmin(fitness)]
    return best_solution

# 函数优化问题
def rosenbrock(x):
    return sum(100 * (x[1:] - x[:-1]**2)**2 + (1 - x[:-1])**2)

# 求解优化问题
best_solution = genetic_algorithm(rosenbrock, 50, 100)
print("最优解：", best_solution)
```
