#### **2.2 编程挑战**

**题目 1：** 编写一个程序，模拟一个简单的生态系统，包括植物、动物和食物链。分析生态系统中的动态变化，并预测某个物种的灭绝时间。

**答案：**
```go
package main

import (
	"fmt"
	"time"
)

type Species struct {
	Name     string
	Count    int
	FeedRate int
}

func (s *Species) update(count int) {
	s.Count += count * s.FeedRate
}

func main() {
	plants := &Species{"Plants", 100, 1}
	herbivores := &Species{"Herbivores", 50, 0.5}
	carnivores := &Species{"Carnivores", 10, 0.2}

	for {
		plants.update(herbivores.Count)
		herbivores.update(carnivores.Count)
		carnivores.update(0)

		if carnivores.Count <= 0 {
			fmt.Println("Carnivores extinct at time:", time.Now())
			break
		}

		time.Sleep(time.Second)
	}
}
```

**解析：** 该程序模拟了一个简单的生态系统，包括植物、食草动物和食肉动物。通过不断更新各个物种的数量，模拟生态系统中的动态变化。当食肉动物灭绝时，程序输出相应的信息。

**题目 2：** 编写一个程序，实现一个简单的交通信号灯控制系统。考虑不同交通流量的影响，优化信号灯的切换策略。

**答案：**
```go
package main

import (
	"fmt"
	"time"
)

type TrafficLight struct {
	Red     bool
	Yellow  bool
	Green   bool
}

func (tl *TrafficLight) switchLight() {
	if !tl.Red && !tl.Yellow && !tl.Green {
		tl.Red = true
	} else if tl.Red && !tl.Yellow && !tl.Green {
		tl.Yellow = true
	} else if tl.Red && tl.Yellow && !tl.Green {
		tl.Green = true
	} else if tl.Red && tl.Yellow && tl.Green {
		tl.Green = false
	}
}

func main() {
	light := TrafficLight{false, false, true}

	for {
		light.switchLight()
		if light.Red {
			fmt.Println("Red Light")
		} else if light.Yellow {
			fmt.Println("Yellow Light")
		} else if light.Green {
			fmt.Println("Green Light")
		}
		time.Sleep(time.Second)
	}
}
```

**解析：** 该程序实现了一个简单的交通信号灯控制系统。根据信号灯的状态，不断切换信号灯的颜色。在实际应用中，可以根据不同交通流量调整信号灯切换策略。
