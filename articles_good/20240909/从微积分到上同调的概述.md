#### 1. 导数的计算

**题目：** 已知函数 $f(x) = x^2 + 2x + 1$，求其在 $x=1$ 处的导数。

**答案：** 导数的计算通常涉及到求极限。对于给定的函数 $f(x) = x^2 + 2x + 1$，其在 $x=1$ 处的导数可以通过以下步骤计算：

$$
f'(1) = \lim_{h \to 0} \frac{f(1+h) - f(1)}{h}
$$

将 $f(x)$ 代入上式，得到：

$$
f'(1) = \lim_{h \to 0} \frac{(1+h)^2 + 2(1+h) + 1 - (1^2 + 2 \cdot 1 + 1)}{h}
$$

简化后得到：

$$
f'(1) = \lim_{h \to 0} \frac{h^2 + 2h + 1}{h} = \lim_{h \to 0} (h + 2) = 2
$$

因此，$f(x)$ 在 $x=1$ 处的导数为 $2$。

#### 2. 积分的求解

**题目：** 求解以下定积分：

$$
\int_{0}^{1} x^2 \, dx
$$

**答案：** 求解定积分通常可以通过找到对应的原函数，然后计算原函数在积分区间上的差值。对于给定的积分，我们可以找到原函数 $F(x) = \frac{x^3}{3}$。因此，该定积分的值为：

$$
\int_{0}^{1} x^2 \, dx = F(1) - F(0) = \frac{1^3}{3} - \frac{0^3}{3} = \frac{1}{3}

### 二、上同调

#### 3. 上同调群的计算

**题目：** 已知一个拓扑空间 $X$，且 $X$ 的上同调群 $H^1(X)$ 是零群。求证：$X$ 是连通空间。

**答案：** 上同调群 $H^1(X)$ 是零群意味着对于任意的闭链 $c$，都存在一个链 $d$ 使得 $c = d \partial b$，其中 $\partial b$ 是边界链。现在假设 $X$ 不是连通空间，即存在两个非空开集 $A$ 和 $B$ 使得 $X = A \cup B$，且 $A \cap B = \emptyset$。

取 $A$ 和 $B$ 的边界分别为 $a$ 和 $b$，则 $a$ 和 $b$ 是闭链，因为它们的边界是空集。根据上同调群的性质，存在链 $d_a$ 和 $d_b$ 使得 $a = d_a \partial b$ 和 $b = d_b \partial a$。

将这两个等式相加，得到：

$$
a + b = (d_a \partial b) + (d_b \partial a) = (d_a + d_b) (\partial b + \partial a) = 0
$$

由于 $H^1(X)$ 是零群，这意味着 $a + b$ 是边界链。但是，$a$ 和 $b$ 分别是 $A$ 和 $B$ 的边界，这意味着 $A$ 和 $B$ 都是闭集，这与 $A$ 和 $B$ 是开集相矛盾。因此，$X$ 必须是连通空间。

##### 3. 拓扑空间的可约性

**题目：** 已知拓扑空间 $X$ 的第一同调群 $H^1(X)$ 是零群，证明 $X$ 是可约空间。

**答案：** 根据上同调理论，如果拓扑空间 $X$ 的第一同调群 $H^1(X)$ 是零群，那么存在一个链分解使得 $X$ 可以表示为两个闭链的并集。这意味着 $X$ 是可约的。

证明：假设 $X$ 是一个拓扑空间，且 $H^1(X) = 0$。根据上同调群的性质，存在链 $c_1$ 和 $c_2$ 使得 $X = c_1 + c_2$，其中 $c_1$ 和 $c_2$ 是闭链。

因为 $H^1(X) = 0$，这意味着 $c_1 + c_2$ 是边界链。因此，存在链 $b_1$ 和 $b_2$ 使得 $c_1 + c_2 = b_1 \partial b_2$。

将等式 $X = c_1 + c_2$ 代入上式，得到：

$$
X = (b_1 \partial b_2) \partial b_2 = b_1 (\partial b_2 \partial b_2) = b_1 \partial \emptyset = \partial b_1
$$

这意味着 $X$ 是可约的，因为 $X$ 可以表示为闭链 $b_1$ 的边界。

**题目：** 使用辛普森法则计算以下定积分：

$$
\int_{0}^{1} e^x \, dx
$$

**答案：** 辛普森法则是数值积分的一种方法，其基本思想是将积分区间等分为 $n$ 个小区间，然后在每个小区间上使用抛物线逼近函数，最后计算抛物线下的面积。

$$

##### 2. 拓扑排序

**题目：** 给定一个有向图，实现一个拓扑排序的算法。

**答案：** 拓扑排序是一种用于对有向无环图（DAG）进行排序的算法。其基本思想是找出所有入度为零的顶点，然后依次删除这些顶点，同时更新其他顶点的入度。

以下是拓扑排序的 Python 代码实现：

```python
from collections import deque

def topological_sort(graph):
    in_degree = {v: 0 for v in graph}
    for v in graph:
        for neighbor in graph[v]:
            in_degree[neighbor] += 1

    queue = deque()
    for v in in_degree:
        if in_degree[v] == 0:
            queue.append(v)

    result = []
    while queue:
        v = queue.popleft()
        result.append(v)
        for neighbor in graph[v]:
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)

    return result
```

例如，给定一个图如下：

```
graph = {
    1: [2, 3],
    2: [4],
    3: [4],
    4: []
}
```

调用 `topological_sort(graph)` 将得到排序结果 `[1, 2, 3, 4]`。

##### 3. 计算曲率

**题目：** 给定一条曲线的参数方程，实现一个计算曲率的算法。

**答案：** 曲率是描述曲线弯曲程度的量，可以通过计算曲线的导数来求解。

对于参数方程 $x(t)$ 和 $y(t)$ 的曲线，其曲率可以通过以下公式计算：

$$
k(t) = \frac{|x'(t)y''(t) - x''(t)y'(t)|}{(x'(t)^2 + y'(t)^2)^{3/2}}
$$

以下是计算曲率的 Python 代码实现：

```python
import numpy as np

def curvature(x, y):
    x_prime = np.gradient(x)
    y_prime = np.gradient(y)
    x_double_prime = np.gradient(x_prime)
    y_double_prime = np.gradient(y_prime)

    k = np.abs(x_prime * y_double_prime - x_double_prime * y_prime) / (np.square(x_prime) + np.square(y_prime)) ** (3/2)
    return k
```

例如，给定参数方程：

```
x(t) = t^3
y(t) = t^2
```

调用 `curvature(x, y)` 将得到曲率值。
