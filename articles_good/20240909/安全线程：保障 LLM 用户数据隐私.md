#### 3. 题目：请给出一个实现安全线程的简单示例。

**答案：** 下面是一个使用Go语言实现的简单安全线程示例：

```go
package main

import (
    "fmt"
    "sync"
)

var (
    counter int
    mu      sync.Mutex
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    counter++
}

func main() {
    var wg sync.WaitGroup
    for i := 0; i < 1000; i++ {
        wg.Add(1)
        go func() {
            defer wg.Done()
            increment()
        }()
    }
    wg.Wait()
    fmt.Println("Counter:", counter)
}
```

**解析：** 在这个例子中，我们使用互斥锁（Mutex）来保护共享变量 `counter`，确保在同一时间只有一个 goroutine 可以访问 `counter`，从而保障数据的安全。
