**题目 4：** 编写一个算法，计算一个基于体育赞助的商业项目在给定时间内的预期收益。

**问题描述：**
假设一个体育赞助项目在一年内进行，项目初始成本为100万元，预计可以吸引5000名新用户。每个新用户在项目期间的平均消费为200元，且项目结束后，每个新用户的终身价值（LTV）为500元。请编写一个算法，计算该项目在一年内的预期收益。

**答案：**
```python
def calculate_project_revenue(initial_cost, new_users, avg_consume, ltv):
    total_new_user_revenue = new_users * avg_consume
    total_ltv_revenue = new_users * ltv
    total_revenue = total_new_user_revenue + total_ltv_revenue
    return total_revenue - initial_cost

initial_cost = 1000000  # 初始成本（元）
new_users = 5000         # 新用户数量
avg_consume = 200        # 每个新用户平均消费（元）
ltv = 500               # 每个新用户的终身价值（元）

revenue = calculate_project_revenue(initial_cost, new_users, avg_consume, ltv)
print("项目预期收益为：", revenue, "元")
```

**解析：**
此算法首先计算新用户在项目期间的总消费收入，然后计算新用户的总终身价值收入。最后，将两者相加，并减去项目的初始成本，得到项目的预期收益。预期收益可以帮助企业评估体育赞助项目的经济效益。

exposure = 10000  # 活动期间曝光次数

awareness_impact = calculate_brand_awareness_impact(exposure, new_followers)
