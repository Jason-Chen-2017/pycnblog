2. **商品兴趣建模：** 使用大模型对商品进行建模，提取商品特征的语义表示。
3. **相似度计算：** 计算用户和商品的兴趣向量之间的相似度，通常使用余弦相似度、欧氏距离等方法。
4. **推荐生成：** 根据用户和商品的相似度生成个性化推荐列表。

**示例代码：** 使用BERT模型进行用户个性化推荐：

```python

# 用户兴趣向量

# 商品兴趣向量

    model.encode("科幻电影《星际穿越》"),
    model.encode("动作电影《速度与激情9》"),
    # ... 其他商品的兴趣向量 ...
]

cosine_similarities = []
for item_interest in item_interests:
    similarity = torch.nn.functional.cosine_similarity(user_interest, item_interest)
    cosine_similarities.append(similarity)

# 生成个性化推荐列表
recommended_items = [item for _, item in sorted(zip(cosine_similarities, item_interests), reverse=True)]

# 输出推荐结果
```
