**功能 3：环境风险评估**

* **数据结构：** 使用优先队列（Priority Queue）存储高风险企业。
* **算法：** 根据碳排放量和环境风险指标，对企业进行排序，提取高风险企业。

**源代码示例：**

```java
import java.util.*;

public class GreenSupplyChain {
    // 图结构表示供应链网络
    private Map<String, List<String>> supplyChain;

    // 哈希表存储碳排放系数
    private Map<String, Double> carbonEmissionsCoefficients;

    public GreenSupplyChain() {
        supplyChain = new HashMap<>();
        carbonEmissionsCoefficients = new HashMap<>();
    }

    // 添加企业到供应链网络
    public void addCompanyToSupplyChain(String company, List<String> partners) {
        supplyChain.put(company, partners);
    }

    // 设置碳排放系数
    public void setCarbonEmissionsCoefficient(String company, double coefficient) {
        carbonEmissionsCoefficients.put(company, coefficient);
    }

    // 计算碳排放量
    public double calculateCarbonEmissions(String company) {
        double emissions = 0;
        Queue<String> queue = new LinkedList<>();
        queue.add(company);

        while (!queue.isEmpty()) {
            String currentCompany = queue.poll();
            double coefficient = carbonEmissionsCoefficients.getOrDefault(currentCompany, 0.0);
            emissions += coefficient;

            for (String partner : supplyChain.getOrDefault(currentCompany, Collections.emptyList())) {
                queue.add(partner);
            }
        }

        return emissions;
    }

    // 主函数
    public static void main(String[] args) {
        GreenSupplyChain system = new GreenSupplyChain();

        // ...
```
