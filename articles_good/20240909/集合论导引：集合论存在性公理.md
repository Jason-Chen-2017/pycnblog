#### 1. 集合的基本运算

**题目：** 给定三个集合 A = {1, 2, 3}, B = {3, 4, 5}, C = {5, 6, 7}，求并集 A ∪ B 和交集 A ∩ B。

**答案：** A ∪ B = {1, 2, 3, 4, 5}，A ∩ B = {3}。

**解析：** 并集包含两个集合中所有的元素，而交集只包含两个集合中都包含的元素。

**代码示例：**

```python
A = [1, 2, 3]
B = [3, 4, 5]
C = [5, 6, 7]

# 求并集
union = list(set(A) | set(B))
print("并集:", union)

# 求交集
intersection = list(set(A) & set(B))
print("交集:", intersection)
```

#### 2. 集合的笛卡尔积

**题目：** 给定两个集合 A = {1, 2}, B = {a, b}，求 A 和 B 的笛卡尔积。

**答案：** A × B = {(1, a), (1, b), (2, a), (2, b)}。

**解析：** 笛卡尔积是指将两个集合中的每个元素进行组合，生成一个新的集合。

**代码示例：**

```python
A = [1, 2]
B = ['a', 'b']

# 求笛卡尔积
cartesian_product = [(x, y) for x in A for y in B]
print("笛卡尔积:", cartesian_product)
```

#### 5. 集合的补集

**题目：** 给定一个集合 A = {1, 2, 3} 和全集 U = {1, 2, 3, 4, 5}，求 A 的补集 A'。

**答案：** A' = {4, 5}。

**解析：** 集合的补集是指全集 U 中不属于集合 A 的元素组成的集合。

**代码示例：**

```python
A = [1, 2, 3]
```

#### 34. 集合的基数和子集

**题目：** 给定一个集合 A，求 A 的基数 |A| 和 A 的所有子集。

**答案：** |A| 是集合 A 的基数，A 的所有子集可以通过递归或位掩码生成。

**代码示例（递归）**：

```python
def subsets(A):
    n = len(A)
    return [] if n == 0 else [subset + [A[i]] for subset in subsets(A[:i+1]) for i in range(n)]

A = [1, 2, 3]

# 求所有子集
all_subsets = subsets(A)
print("所有子集:", all_subsets)

# 求集合A的基数
cardinality_A = len(A)
print("集合A的基数:", cardinality_A)
```

**代码示例（位掩码）**：

```python
def subsets(A):
    n = len(A)
    return [A[i] for i in range(2**n) if (i & (1 << j)) != 0 for j in range(n)]

A = [1, 2, 3]

# 求所有子集
all_subsets = subsets(A)
print("所有子集:", all_subsets)

# 求集合A的基数
cardinality_A = len(A)
print("集合A的基数:", cardinality_A)
```

#### 35. 集合的对称差和并集

**题目：** 给定两个集合 A 和 B，求 A 和 B 的对称差 A Δ B 和 A 和 B 的并集 A ∪ B。

**答案：** A Δ B 是 A 和 B 的对称差，即 A ∪ B - (A ∩ B)。A ∪ B 是 A 和 B 的所有元素的集合。

**代码示例：**

```python
A = [1, 2, 3]
B = [3, 4, 5]

# 求对称差
symmetric_difference = list(set(A) ^ set(B))
print("对称差:", symmetric_difference)

# 求并集
union = list(set(A) | set(B))
print("并集:", union)
```

#### 36. 集合的基数和补集

**题目：** 给定一个集合 A 和全集 U，求 A 的基数 |A| 和 A 的补集 A'。

**答案：** |A| 是集合 A 的基数，A' 是全集 U 中不属于 A 的元素组成的集合。

**代码示例：**

```python
U = [1, 2, 3, 4, 5]
A = [1, 2]

# 求集合A的基数
cardinality_A = len(A)
print("集合A的基数:", cardinality_A)

# 求集合A的补集
complement_A = [x for x in U if x not in A]
print("集合A的补集:", complement_A)
```

#### 37. 集合的交集和并集

**题目：** 给定三个集合 A、B 和 C，求 A 和 B 的交集 A ∩ B，以及 A 和 B 的并集 A ∪ B。

**答案：** A ∩ B 是 A 和 B 的公共元素，A ∪ B 是 A 和 B 的所有元素的集合。

```python
A = [1, 2, 3]

C = [5, 6, 7]

# 求交集
intersection = list(set(A) & set(B))
```

#### 38. 集合的差集和对称差

**题目：** 给定两个集合 A 和 B，求 A 和 B 的差集 A - B 和 A 和 B 的对称差 A Δ B。

**答案：** A - B 是属于 A 但不属于 B 的元素组成的集合，A Δ B 是属于 A 或 B 但不同时属于 A 和 B 的元素组成的集合。

**代码示例：**

```python
A = [1, 2, 3]
B = [3, 4, 5]

# 求差集
difference = list(set(A) - set(B))
print("差集:", difference)

# 求对称差
symmetric_difference = list(set(A) ^ set(B))
print("对称差:", symmetric_difference)
```

#### 39. 集合的基数和子集

**题目：** 给定一个集合 A，求 A 的基数 |A| 和 A 的所有子集。

**答案：** |A| 是集合 A 的基数，A 的所有子集可以通过递归或位掩码生成。

**代码示例（递归）**：

```python
def subsets(A):
    n = len(A)
    return [[]] if n == 0 else [subset + [A[i]] for subset in subsets(A[:i+1]) for i in range(n)]

A = [1, 2, 3]

# 求所有子集
all_subsets = subsets(A)
print("所有子集:", all_subsets)

# 求集合A的基数
cardinality_A = len(A)
print("集合A的基数:", cardinality_A)
```

**代码示例（位掩码）**：

```python
def subsets(A):
    n = len(A)
    return [A[i] for i in range(2**n) if (i & (1 << j)) != 0 for j in range(n)]

A = [1, 2, 3]

# 求所有子集
all_subsets = subsets(A)
print("所有子集:", all_subsets)

# 求集合A的基数
cardinality_A = len(A)
print("集合A的基数:", cardinality_A)
```
