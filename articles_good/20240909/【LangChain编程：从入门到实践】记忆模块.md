### 3. 记忆模块的使用：API接口和查询示例
**题目：** 请展示如何使用LangChain记忆模块的API接口，并提供一个简单的查询示例。

**答案：**
以下是使用LangChain记忆模块的API接口的一个简单示例：
```python
from langchain.memory import Memory

# 创建一个简单的记忆模块实例
memory = Memory()

# 添加一条记忆信息
memory.add("person", "name", "Alice")
memory.add("person", "age", 30)

# 查询记忆信息
name = memory.get("person", "name")
age = memory.get("person", "age")

print(f"Name: {name}, Age: {age}")
```

**解析：**
在这个示例中，我们首先创建了一个Memory对象，然后使用`add`方法添加了关于一个人的两条记忆信息。接着，我们使用`get`方法根据键值查询了姓名和年龄，并打印出来。
