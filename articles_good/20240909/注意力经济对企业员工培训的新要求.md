### 5. 如何设计一个基于用户关注度的内容推荐系统？

**题目：** 设计一个算法，根据用户的历史行为（如浏览、点赞、分享等），为用户推荐内容。

**答案：** 可以采用基于协同过滤的推荐算法，如矩阵分解、基于用户的最近邻（User-Based CF）等。以下是一个简单的基于用户的最近邻算法示例：

```python
import numpy as np

def calculate_similarity(user_profiles, user_index):
    # 计算用户与所有其他用户的相似度
    similarity_matrix = []
    for i, other_index in enumerate(range(len(user_profiles))):
        if i == user_index:
            continue
        similarity = np.dot(user_profiles[user_index], user_profiles[other_index])
        similarity_matrix.append(similarity)
    return similarity_matrix

def content_recommendation(user_profile, user_index, user_profiles, top_n=5):
    # 根据相似度推荐内容
    similarity_matrix = calculate_similarity(user_profiles, user_index)
    similar_users = np.argsort(similarity_matrix)[::-1]
    recommended_contents = []
    for user in similar_users[1:top_n+1]:
        recommended_contents.extend(user_profiles[user])
    return recommended_contents
```
