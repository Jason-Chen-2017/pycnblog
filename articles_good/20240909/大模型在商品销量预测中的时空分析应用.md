#### 3. 题目：如何评估商品销量预测模型的性能？

**题目：** 在开发商品销量预测模型后，如何评估其性能？

**答案：**

**方法：**

1. **均方误差（MSE）：** 衡量预测值与实际值之间的平方误差，越小越好。
2. **均方根误差（RMSE）：** MSE的平方根，数值越小表示预测越准确。
3. **平均绝对误差（MAE）：** 预测值与实际值之间的绝对误差的平均值，越小越好。
4. **决定系数（R²）：** 衡量模型解释变量的比例，值越接近1表示模型越好。

**举例：** 使用Python评估销量预测模型性能：

```python
from sklearn.metrics import mean_squared_error, mean_absolute_error, r2_score

# 预测结果
y_pred = model.predict(X_test)

# 计算性能指标
mse = mean_squared_error(y_test, y_pred)
mae = mean_absolute_error(y_test, y_pred)
r2 = r2_score(y_test, y_pred)

print("MSE:", mse)
print("MAE:", mae)
print("R²:", r2)
```

**解析：** 通过这些性能指标，可以全面评估销量预测模型的准确性、稳定性和解释能力。

2. **置信区间：** 根据预测的概率分布计算置信区间，提供预测结果的可靠性范围。
3. **敏感性分析：** 分析模型输入变量对预测结果的影响，识别关键变量并减小不确定性。

**举例：** 使用Python计算置信区间：

```python
import numpy as np

# 预测结果
predictions = ...

# 计算概率分布
probabilities = np.exp(predictions) / np.sum(np.exp(predictions), axis=1, keepdims=True)

# 计算置信区间（95%）
confidence_level = 0.95
alpha = 1 - confidence_level
confidence_intervals = np.percentile(probabilities, [alpha/2*100, (1-alpha/2)*100], axis=1)
```

**解析：** 通过计算置信区间，可以提供预测结果的可靠性范围，帮助决策者评估预测风险。
