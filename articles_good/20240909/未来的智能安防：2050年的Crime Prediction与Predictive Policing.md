### 2. 如何处理缺失值和异常值？

**题目：** 在Crime Prediction数据集中，如何处理缺失值和异常值？

**答案：** 

1. **缺失值处理：**

**解析：** 缺失值和异常值会影响模型的训练效果，因此需要采取适当的处理方法，以提高模型的性能。

### 4. 如何评估 Crime Prediction 模型的性能？

**题目：** 在评估Crime Prediction模型性能时，有哪些常用的评估指标？

**答案：**

1. **准确率（Accuracy）：** 分类正确的样本数占总样本数的比例。
2. **精确率（Precision）：** 精确率表示预测为正类的样本中，实际为正类的比例。
3. **召回率（Recall）：** 召回率表示实际为正类的样本中，被预测为正类的比例。
4. **F1 分数（F1 Score）：** 精确率和召回率的调和平均，综合考虑了分类的精确性和召回性。
5. **ROC 曲线和 AUC 值：** ROC 曲线下的面积（AUC）用于评估分类器的性能。

**解析：** 选择合适的评估指标可以根据业务需求和数据集的特点，全面评估模型的性能。

**答案：**

2. **欠采样（Under-sampling）：** 删除多数类样本，减少其在数据集中的比例。

4. **集成方法：** 结合多种方法，如SMOTE与欠采样相结合。
5. **权重调整：** 在训练模型时，给少数类样本更高的权重。
