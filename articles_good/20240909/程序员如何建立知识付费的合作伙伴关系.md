### 4. 搜索推荐算法

**题目：** 设计一个简单的基于用户行为的搜索推荐算法。

**答案：** 一个简单的基于用户行为的搜索推荐算法可以基于用户的浏览历史和购买行为。算法的核心思想是找出与其他用户行为相似的用户，并推荐这些用户热门的内容。

以下是算法的基本步骤：

1. **用户画像构建：** 根据用户的浏览历史和购买行为构建用户画像。
2. **相似用户发现：** 使用协同过滤算法（如用户基于物品的协同过滤）找到与其他用户行为相似的潜在用户。
3. **内容推荐：** 根据相似用户的兴趣推荐相应的内容。

以下是一个简单的协同过滤算法的伪代码：

```python
# 伪代码

# 构建用户画像
def build_user_profile(user_actions):
    profile = {}
    for action in user_actions:
        item = action['item']
        if item not in profile:
            profile[item] = 0
        profile[item] += 1
    return profile

# 计算用户相似度
def calculate_similarity(user1_profile, user2_profile):
    common_items = set(user1_profile.keys()) & set(user2_profile.keys())
    if not common_items:
        return 0
    sum_similarity = sum(min(user1_profile[item], user2_profile[item]) for item in common_items)
    return sum_similarity / len(common_items)

# 推荐内容
def recommend_content(user_profile, all_user_profiles, content_popularity, top_n=5):
    similarity_scores = {}
    for profile in all_user_profiles:
        similarity_scores[profile['user_id']] = calculate_similarity(user_profile, profile['profile'])
    
    recommended_items = []
    for user_id, score in sorted(similarity_scores.items(), key=lambda x: x[1], reverse=True):
        for item in all_user_profiles[user_id]['profile']:
            if item not in recommended_items and item in content_popularity:
                recommended_items.append(item)
                if len(recommended_items) == top_n:
                    break
    
    return recommended_items
```
