### 6. 设计一个注意力分配算法

**题目：** 设计一个简单的注意力分配算法，用于优化多任务处理。

**答案：** 基于优先级和任务完成时间，设计一个简单的注意力分配算法。

```python
def attention_allocation(tasks):
    sorted_tasks = sorted(tasks, key=lambda x: (x['priority'], x['duration']))
    attention_score = 100  # 初始注意力分数
    completed_tasks = []

    for task in sorted_tasks:
        if attention_score >= task['duration']:
            completed_tasks.append(task)
            attention_score -= task['duration']
        else:
            break

    return completed_tasks

tasks = [
    {'task_id': 1, 'priority': 3, 'duration': 20},
    {'task_id': 2, 'priority': 2, 'duration': 10},
    {'task_id': 3, 'priority': 1, 'duration': 30},
]

print(attention_allocation(tasks))
```

### 7. 注意力焦点切换算法

**题目：** 设计一个算法来模拟注意力焦点在多个任务间的切换。

**答案：** 使用状态机来模拟注意力焦点在不同任务之间的切换。

```python
class AttentionSwitcher:
    def __init__(self, tasks):
        self.tasks = tasks
        self.current_task = None

    def switch_attention(self, task_id):
        for task in self.tasks:
            if task['task_id'] == task_id:
                if self.current_task:
                    self.current_task['duration'] -= 1
                self.current_task = task
                break

    def report(self):
        return {
            'current_task': self.current_task,
            'tasks': self.tasks,
        }

tasks = [
    {'task_id': 1, 'duration': 20},
    {'task_id': 2, 'duration': 10},
    {'task_id': 3, 'duration': 30},
]

switcher = AttentionSwitcher(tasks)
switcher.switch_attention(1)
print(switcher.report())
```
