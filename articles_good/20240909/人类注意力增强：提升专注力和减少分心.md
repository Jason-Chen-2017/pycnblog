#### 二、算法编程题解析

### 3. 暴力破解密码

**题目：** 编写一个算法，用于检测用户输入的密码是否正确。如果密码长度超过 6 位，则密码强度较高，需要通过暴力破解方式来验证。

**答案：** 可以采用暴力破解算法，具体实现如下：

```python
def check_password(password):
    if len(password) <= 6:
        return True  # 密码长度小于等于 6 位，直接返回正确
    
    for i in range(len(password)):
        for j in range(i+1, len(password)+1):
            if j-i > 1 and password[i:j] in ["123456", "password"]:
                return False  # 密码为常见弱密码，返回错误
    return True  # 密码强度较高，返回正确

# 示例
print(check_password("123456"))  # 输出：False
print(check_password("password"))  # 输出：False
print(check_password("qwerty"))  # 输出：True
```

**解析：** 通过暴力破解算法，可以检测用户输入的密码是否为常见弱密码。实际应用中，可以根据实际情况调整弱密码列表和密码长度限制。

### 4. 字符串匹配算法

**题目：** 编写一个算法，用于在给定的文本中查找是否存在指定模式。

**答案：** 可以采用 KMP 算法，具体实现如下：

```python
def kmp_search(text, pattern):
    def build_lps(pattern):
        lps = [0] * len(pattern)
        length = 0
        i = 1
        while i < len(pattern):
            if pattern[i] == pattern[length]:
                length += 1
                lps[i] = length
                i += 1
            else:
                if length != 0:
                    length = lps[length - 1]
                else:
                    lps[i] = 0
                    i += 1
        return lps

    lps = build_lps(pattern)
    i = j = 0
    while i < len(text):
        if pattern[j] == text[i]:
            i += 1
            j += 1
        if j == len(pattern):
            return True
        elif i < len(text) and pattern[j] != text[i]:
            if j != 0:
                j = lps[j - 1]
            else:
                i += 1
    return False

# 示例
print(kmp_search("ababcabcab", "abc"))  # 输出：True
print(kmp_search("ababcabcab", "abcd"))  # 输出：False
```

**解析：** KMP 算法是一种高效的字符串匹配算法，通过预计算最长公共前后缀（LPS）数组，避免了字符串匹配过程中的重复计算，提高了匹配速度。
