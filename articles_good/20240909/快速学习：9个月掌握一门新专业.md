### 1. 数据结构与算法基础
**题目：** 什么是时间复杂度？请举例说明线性时间复杂度和对数时间复杂度。

**答案：** 时间复杂度是指算法执行时间与数据规模之间的增长关系。线性时间复杂度（O(n)）表示算法执行时间与数据规模成正比；对数时间复杂度（O(log n)）表示算法执行时间与数据规模的对数成正比。

**举例：** 
- 线性时间复杂度：遍历一个数组
- 对数时间复杂度：二分查找一个有序数组

```python
class StackQueue:
    def __init__(self):
        self.stack_in = []
        self.stack_out = []

    def enqueue(self, item):
        self.stack_in.append(item)

    def dequeue(self):
        if not self.stack_out:
            while self.stack_in:
                self.stack_out.append(self.stack_in.pop())
        return self.stack_out.pop()
```

```python
def has_cycle(graph):
    visited = set()
    for node in graph:
        if node not in visited:
            if has_cycle_util(graph, node, visited):
                return True
    return False

def has_cycle_util(graph, node, visited):
    visited.add(node)
    for neighbor in graph[node]:
        if neighbor not in visited:
            if has_cycle_util(graph, neighbor, visited):
                return True
        elif neighbor != node:

    return False
```

### 18. 算法面试题
**题目：** 请实现一个函数，计算两个大整数的和。

**答案：**
```python
def add_large_numbers(num1, num2):
    carry = 0
    result = []
    i, j = len(num1) - 1, len(num2) - 1

    while i >= 0 or j >= 0 or carry:
        x = int(num1[i]) if i >= 0 else 0
        y = int(num2[j]) if j >= 0 else 0
        sum = x + y + carry
        carry = sum // 10
        result.append(str(sum % 10))
        i, j = i - 1, j - 1

    if carry:
        result.append(str(carry))

    return ''.join(result[::-1])
```

### 19. 算法面试题
**题目：** 请实现一个函数，找出字符串中的最长无重复子串。

**答案：**
```python
def longest_substring_without_repeating_characters(s):
    start = 0
    max_length = 0
    used_char = {}

    for i, char in enumerate(s):
        if char in used_char and start <= used_char[char]:
            start = used_char[char] + 1
        used_char[char] = i
        max_length = max(max_length, i - start + 1)

    return max_length
```

### 21. 算法面试题

**答案：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_sorted_lists(l1, l2):
    dummy = ListNode()
    curr = dummy

    while l1 and l2:
        if l1.val < l2.val:
            curr.next = l1
            l1 = l1.next
        else:
            curr.next = l2
            l2 = l2.next
        curr = curr.next

    curr.next = l1 or l2
    return dummy.next
```

### 22. 算法面试题
**题目：** 请实现一个函数，判断一个二叉树是否是平衡二叉树。

**答案：**
```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def is_balanced(root):
    def check_height(node):
        if not node:
            return 0

        left_height = check_height(node.left)
        if left_height == -1:
            return -1
        right_height = check_height(node.right)
        if right_height == -1:
            return -1

        if abs(left_height - right_height) > 1:
            return -1

        return max(left_height, right_height) + 1

    return check_height(root) != -1
```
