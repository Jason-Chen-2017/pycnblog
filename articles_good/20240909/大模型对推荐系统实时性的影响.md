**题目：** 请描述如何使用协同过滤算法实现一个简单的推荐系统。

**答案：** 协同过滤算法主要包括基于用户的协同过滤（User-based Collaborative Filtering）和基于物品的协同过滤（Item-based Collaborative Filtering）。

**基于用户的协同过滤：**

```python
# Python 示例代码
import numpy as np

# 用户行为数据矩阵，行表示用户，列表示物品
rating_matrix = np.array([[1, 2, 0, 0],
                          [2, 0, 3, 1],
                          [0, 1, 4, 2]])

# 计算相似度矩阵
similarity_matrix = np.dot(rating_matrix.T, rating_matrix) / np.linalg.norm(rating_matrix.T, axis=1) * np.linalg.norm(rating_matrix, axis=0)

# 按照相似度排序
sorted_similarity = np.argsort(similarity_matrix[0])

# 推荐结果
recommendation = rating_matrix[0, sorted_similarity[-5:]] * similarity_matrix[0, sorted_similarity[-5:]]
print("Recommendation:", recommendation)
```
