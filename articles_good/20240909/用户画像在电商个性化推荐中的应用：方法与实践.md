### 算法编程题库

#### 5. 请实现一个基于用户行为数据的用户画像构建方法。
**题目：** 编写一个程序，接收用户行为数据，输出用户画像。

**答案：** 

```python
# 假设用户行为数据为：用户ID、购买商品类别、购买时间、浏览商品类别
user_behavior = [
    ('user1', '服装', '2023-01-01', '数码'),
    ('user1', '食品', '2023-01-02', '家居'),
    ('user2', '数码', '2023-01-01', '服装'),
    # 更多数据...
]

from collections import defaultdict
import pandas as pd

def build_user_profile(behavior_data):
    user_profile = defaultdict(dict)
    behavior_df = pd.DataFrame(behavior_data)
    
    for index, row in behavior_df.iterrows():
        user_id, category, _, _ = row
        if category not in user_profile[user_id]:
            user_profile[user_id][category] = 0
        user_profile[user_id][category] += 1
    
    return user_profile

user_profile = build_user_profile(user_behavior)
for user, profile in user_profile.items():
    print(f"{user}: {profile}")
```

**解析：** 该程序首先将用户行为数据转换为 DataFrame 格式，然后遍历数据，为每个用户构建一个包含购买类别及其次数的字典，形成用户画像。

#### 6. 请实现一个基于用户画像的个性化推荐算法。
**题目：** 编写一个程序，根据用户画像为用户推荐商品。

**答案：**

```python
# 假设用户画像和商品信息如下：
user_profile = {
    'user1': {'服装': 3, '食品': 2, '家居': 1},
    'user2': {'数码': 5, '服装': 1},
    # 更多用户画像...
}

item_info = {
    '商品1': {'类别': '服装', '评分': 4.5},
    '商品2': {'类别': '食品', '评分': 4.8},
    '商品3': {'类别': '数码', '评分': 4.6},
    # 更多商品信息...
}

def recommend_items(profile, items, n=3):
    recommendations = []
    for item, info in items.items():
        if info['类别'] in profile:
            recommendations.append((item, info['评分']))
    
    recommendations.sort(key=lambda x: x[1], reverse=True)
    return recommendations[:n]

recommendations = recommend_items(user_profile['user1'], item_info)
print("推荐的商品：", recommendations)
```

**解析：** 该程序首先根据用户画像筛选出符合用户兴趣类别的商品，然后根据商品评分进行排序，返回推荐结果。
