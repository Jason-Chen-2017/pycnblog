#### 算法编程题库

**4. 实现一个二分查找算法。**

**题目描述：** 给定一个排序后的整数数组 `nums` 和一个目标值 `target`，编写一个函数搜索 `nums` 中的 `target`，如果目标值存在返回它的索引，否则返回 `-1`。

**答案：**

```python
def binary_search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

# 测试
nums = [1, 2, 3, 4, 5]
target = 3
print(binary_search(nums, target))  # 输出：2
```

**解析：** 二分查找算法是一种高效搜索算法，通过不断缩小搜索范围，快速找到目标值。卓越管理者应熟悉算法原理，并应用于实际项目管理中，提升团队效率。

**5. 设计一个双向链表。**

**题目描述：** 实现一个支持迭代、添加节点、删除节点、插入节点、获取节点值的双向链表。

**答案：**

```python
class Node:
    def __init__(self, value):
        self.value = value
        self.next = None
        self.prev = None

class DoublyLinkedList:
    def __init__(self):
        self.head = None
        self.tail = None

    def append(self, value):
        new_node = Node(value)
        if not self.head:
            self.head = new_node
            self.tail = new_node
        else:
            self.tail.next = new_node
            new_node.prev = self.tail
            self.tail = new_node

    def remove(self, node):
        if node.prev:
            node.prev.next = node.next
        if node.next:
            node.next.prev = node.prev
        if node == self.head:
            self.head = node.next
        if node == self.tail:
            self.tail = node.prev
        node.next = None
        node.prev = None

    def insert(self, node, value):
        new_node = Node(value)
        if node == self.tail:
            self.append(value)
            return
        new_node.next = node.next
        new_node.prev = node
        node.next.prev = new_node
        node.next = new_node

    def __iter__(self):
        current = self.head
        while current:
            yield current.value
            current = current.next

# 测试
dll = DoublyLinkedList()
dll.append(1)
dll.append(2)
dll.append(3)
dll.insert(Node(2), 2.5)
dll.remove(Node(1))
for value in dll:
    print(value)  # 输出：3 2.5 3
```

**解析：** 双向链表是一种常见的数据结构，通过在节点中添加前后指针，实现快速插入、删除操作。卓越管理者应具备设计数据结构的能力，提升团队技术水平。
