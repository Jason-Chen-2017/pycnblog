##### 3. 在云计算服务中如何确保数据安全？

**题目描述：** 请解释在云计算服务中如何确保用户数据的安全，并设计相应的数据安全策略。

**答案解析：**
- **数据加密：** 对用户数据进行加密，确保数据在传输和存储过程中不被窃取。
- **访问控制：** 实施严格的访问控制机制，只有授权用户才能访问特定数据。
- **安全审计：** 定期进行安全审计，检测潜在的安全威胁和漏洞。
- **备份和恢复：** 定期备份用户数据，并建立灾难恢复计划，确保数据在灾难发生时能够迅速恢复。

**代码示例：**（以下为Python示例代码）

```python
import bcrypt

def hash_password(password):
    hashed = bcrypt.hashpw(password.encode('utf-8'), bcrypt.gensalt())
    return hashed

def check_password(hashed_password, password):
    return bcrypt.checkpw(password.encode('utf-8'), hashed_password)

# 假设用户输入密码为 "password123"
password = "password123"
hashed = hash_password(password)
print("hashed_password:", hashed)

# 验证密码
result = check_password(hashed, password)
print("Password is correct:", result)
```
