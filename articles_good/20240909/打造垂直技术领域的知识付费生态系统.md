    "fmt"
    "os"
)

    UserID   string

    Amount   float64
    Status   string

        ID:     "1001",
        Name:   "李四",
        Balance: 100.0,

    fmt.Println("User Balance:", user.Balance)

**解析：** 在这个例子中，我们创建了一个 `User` 结构体和一个 `Order` 结构体，分别用于存储用户信息和订单信息。用户支付后，系统会更新用户的余额和订单状态。
