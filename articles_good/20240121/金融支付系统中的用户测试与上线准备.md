### 4.1 压力测试模型

压力测试模型用于评估系统的性能和稳定性。在用户测试中，可以使用压力测试模型来模拟大量用户的访问请求，以评估系统的性能。压力测试模型的公式为：

$$
P = \frac{N \times T}{R}
$$

其中，$P$ 表示压力测试的强度，$N$ 表示请求数量，$T$ 表示请求时间，$R$ 表示系统的响应时间。

### 4.2 性能指标模型

性能指标模型用于评估系统的性能。在用户测试中，可以使用性能指标模型来计算系统的平均响应时间、吞吐量等指标。性能指标模型的公式为：

$$
Q = \frac{N}{T}
$$

$$
R = \frac{1}{N} \sum_{i=1}^{N} t_i
$$

其中，$Q$ 表示吞吐量，$N$ 表示请求数量，$T$ 表示时间间隔，$R$ 表示平均响应时间，$t_i$ 表示第$i$个请求的响应时间。

```java
// 创建JMeter测试计划
TestPlan testPlan = new TestPlan("Financial Payment System Test Plan");

// 创建线程组
ThreadGroup threadGroup = new ThreadGroup("Financial Payment System Thread Group");
threadGroup.setNumThreads(100);
threadGroup.setRampUp(10);

// 创建HTTP请求
HTTPSamplerProxy httpRequest = new HTTPSamplerProxy("http://www.example.com/payment");
httpRequest.setMethod("POST");
httpRequest.setPath("/payment");

// 添加请求参数
NameValuePair[] parameters = new NameValuePair[1];
parameters[0] = new NameValuePair("amount", "100");

httpRequest.setArguments(parameters);

// 添加请求头
Header[] headers = new Header[1];
headers[0] = new Header("Content-Type", "application/json");

httpRequest.setHeaders(headers);

// 添加请求到线程组
threadGroup.addSampler(httpRequest);

// 添加线程组到测试计划
testPlan.addThreadGroup(threadGroup);

// 保存测试计划
testPlan.writeToFile("FinancialPaymentSystemTestPlan.jmx");
```
