### 4.1 代码实例

以下是一个简单的Redis分布式图数据挖掘示例：

```python
import redis

# 连接Redis
r = redis.StrictRedis(host='localhost', port=6379, db=0)

# 创建图数据
nodes = ['node1', 'node2', 'node3', 'node4']
edges = [('node1', 'node2'), ('node2', 'node3'), ('node3', 'node4')]

# 存储节点数据
for node in nodes:
    r.sadd('nodes', node)

# 存储边数据
for edge in edges:
    r.zadd('edges:%s:%s' % (edge[0], edge[1]), 1)

# 查询边数据
edges = r.zrange('edges:node1:node2')
print(edges)
```

### 4.2 详细解释说明

在上述代码中，我们首先连接到Redis服务器，然后创建一些节点和边数据。接着，我们将节点数据存储到Redis的集合数据结构中，并将边数据存储到Redis的有序集合数据结构中。最后，我们查询边数据并打印出来。
