## 2. 核心概念与联系

### 2.1 智能家居系统

### 2.2 知识图谱

知识图谱是一种用于表示实体、关系和属性的数据结构，可以帮助人工智能系统进行更高效、准确的信息查询和推理。知识图谱的主要组成部分包括：

- 属性：实体的特性，可以是基本属性、复合属性、派生属性等。

## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 知识图谱构建实例

以下是一个简单的知识图谱构建实例：

```python
from rdflib import Graph, URIRef, Literal, Namespace

# 定义命名空间
ns = Namespace("http://example.org/")

# 创建图
g = Graph()

# 添加实体
house = URIRef(ns.house)
bedroom = URIRef(ns.bedroom)
living_room = URIRef(ns.living_room)

# 添加关系
g.add((house, ns.has_room, bedroom))
g.add((house, ns.has_room, living_room))

# 添加属性
g.add((bedroom, ns.room_type, Literal("bedroom")))
g.add((living_room, ns.room_type, Literal("living room")))

# 打印图
print(g.serialize(format="turtle"))
```
