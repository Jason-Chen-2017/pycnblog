## 2. 核心概念与联系
在自然语言处理中，命名实体识别（NER）是一种分类任务，其目标是将文本中的命名实体标记为预定义的类别，如人名、地名、组织名等。NER可以分为以下几种类型：

- **实体类别**：NER可以识别的命名实体类别，如人名、地名、组织名、位置名等。
- **实体标注**：NER在文本中识别的命名实体，如“蒂姆·伯尼斯”、“纽约”、“联合国”等。
- **实体边界**：NER在文本中识别的命名实体的边界，如“蒂姆·伯尼斯”的起始位置和结束位置。

NER的核心概念与联系包括：

- **词性标注**：NER与词性标注相关，因为命名实体通常具有特定的词性，如人名通常是名词，地名通常是名词或地名等。
- **语义角色标注**：NER与语义角色标注相关，因为命名实体通常扮演特定的语义角色，如主体、目标、动作等。
- **信息抽取**：NER与信息抽取相关，因为命名实体通常是文本中的有用信息，如人名、地名、组织名等。

数学模型公式详细讲解：

- **支持向量机**：支持向量机（Support Vector Machines，SVM）是一种监督学习算法，用于解决二分类问题。它的核心思想是通过寻找最大间隔来实现分类。支持向量机的数学模型公式如下：

  $$
  \min_{w,b} \frac{1}{2}w^T w + C \sum_{i=1}^n \xi_i \\
  s.t. \ y_i (w^T \phi(x_i) + b) \geq 1 - \xi_i, \ \xi_i \geq 0, \ i = 1, \dots, n
  $$

  其中，$w$ 是权重向量，$b$ 是偏置项，$\phi(x_i)$ 是输入向量 $x_i$ 通过非线性映射后的特征向量，$C$ 是正则化参数，$\xi_i$ 是误差项。

- **决策树**：决策树（Decision Tree）是一种监督学习算法，用于解决分类和回归问题。它的核心思想是通过递归地划分特征空间来实现分类。决策树的数学模型公式如下：

  $$
  \arg \max_{c \in \{1, \dots, k\}} \sum_{i=1}^n I(y_i = c) P(x_i)
  $$

  其中，$c$ 是类别，$k$ 是类别数量，$I(y_i = c)$ 是指示器函数，$P(x_i)$ 是输入向量 $x_i$ 的概率分布。

- **循环神经网络**：循环神经网络（Recurrent Neural Network，RNN）是一种深度学习算法，用于解决序列数据的分类和回归问题。它的核心思想是通过循环连接隐藏层来实现长距离依赖。循环神经网络的数学模型公式如下：

  $$
  h_t = \sigma(W_{hh} h_{t-1} + W_{xh} x_t + b_h) \\
  y_t = \sigma(W_{hy} h_t + b_y)
  $$

  其中，$h_t$ 是隐藏层状态，$y_t$ 是输出，$\sigma$ 是激活函数，$W_{hh}$ 是隐藏层到隐藏层的权重矩阵，$W_{xh}$ 是输入到隐藏层的权重矩阵，$W_{hy}$ 是隐藏层到输出的权重矩阵，$b_h$ 是隐藏层的偏置项，$b_y$ 是输出的偏置项。

          StanfordCoreNLP pipeline = new StanfordCoreNLP(props);

                      System.out.println(token.word() + " - " + ner);
                  }
              }
          }
      }
  }
