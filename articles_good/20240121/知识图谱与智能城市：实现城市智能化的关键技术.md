## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 代码实例

以下是一个简单的智能城市与知识图谱的代码实例：

```python
import networkx as nx
import numpy as np

# 创建图
G = nx.Graph()

# 添加节点
G.add_node("城市管理")
G.add_node("城市服务")
G.add_node("城市生活")

# 添加边
G.add_edge("城市管理", "城市服务")
G.add_edge("城市管理", "城市生活")
G.add_edge("城市服务", "城市生活")

# 查询节点
print(G.nodes)

# 查询边
print(G.edges)

# 查询节点的邻接节点
print(G.neighbors("城市管理"))

# 查询节点的度
print(G.degree("城市管理"))

# 查询节点的距离
print(nx.shortest_path_length(G, "城市管理", "城市生活"))
```

### 4.2 详细解释说明

上述代码实例中，我们首先导入了`networkx`和`numpy`库，然后创建了一个图`G`，添加了节点`城市管理`、`城市服务`、`城市生活`，并添加了边`城市管理-城市服务`、`城市管理-城市生活`、`城市服务-城市生活`。接着，我们查询了节点、边、邻接节点、度、距离等信息。
