```python
url = 'http://example.com/api/products'
response = requests.get(url)
data = response.json()
```

### 4.2 数据处理

```python

df = pd.DataFrame(data)
df['price'] = df['price'].apply(lambda x: x / 100)
df['sales'] = df['sales'].apply(lambda x: x / 1000)
```

### 4.3 数据存储

```python
from sqlalchemy import create_engine
```
