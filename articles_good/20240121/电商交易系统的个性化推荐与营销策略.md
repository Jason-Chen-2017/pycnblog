- 协同过滤的相似性计算公式：
$$
sim(u,v) = \frac{\sum_{i \in I(u) \cap I(v)} (r_{ui} - \bar{r_u})(r_{vi} - \bar{r_v})}{\sqrt{\sum_{i \in I(u)} (r_{ui} - \bar{r_u})^2} \sqrt{\sum_{i \in I(v)} (r_{vi} - \bar{r_v})^2}}
$$
其中，$sim(u,v)$ 表示用户 $u$ 和用户 $v$ 之间的相似性，$I(u)$ 表示用户 $u$ 的历史记录集合，$r_{ui}$ 表示用户 $u$ 对产品 $i$ 的评分，$\bar{r_u}$ 表示用户 $u$ 的平均评分。

- 内容过滤的相似性计算公式：
$$
sim(i,j) = \frac{1}{1 + \frac{\sum_{k=1}^{n} (d_k^i - d_k^j)^2}{\sum_{k=1}^{n} d_k^i + \sum_{k=1}^{n} d_k^j}}
$$
其中，$sim(i,j)$ 表示产品 $i$ 和产品 $j$ 之间的相似性，$d_k^i$ 表示产品 $i$ 的特征值，$n$ 表示特征的数量。

```python
import numpy as np
from scipy.spatial.distance import cosine

def cosine_similarity(u, v):
    dot_product = np.dot(u, v)
    norm_u = np.linalg.norm(u)
    norm_v = np.linalg.norm(v)
    return dot_product / (norm_u * norm_v)

def recommend_items(user_id, user_ratings, item_ratings, num_recommendations):
    user_ratings_matrix = np.array(user_ratings)
    user_ratings_matrix = user_ratings_matrix[user_ratings_matrix.nonzero()][0]
    user_similarities = {}

    for item_id in user_ratings_matrix.nonzero()[0]:
        item_similarities = {}
        for other_user_id in user_ratings_matrix.nonzero()[1]:
            if other_user_id not in user_similarities:
                user_similarities[other_user_id] = {}
            item_similarities[other_user_id] = cosine_similarity(user_ratings_matrix[item_id], user_ratings_matrix[other_user_id])
        item_similarities = {other_user_id: value for other_user_id, value in item_similarities.items() if other_user_id != user_id}
        user_similarities[item_id] = item_similarities

    recommended_items = []
    for item_id in user_ratings_matrix.nonzero()[0]:
        if item_id not in user_similarities[item_id].keys():
            continue
        similar_users = sorted(user_similarities[item_id].keys(), key=lambda x: user_similarities[item_id][x], reverse=True)
        for similar_user in similar_users:
            if len(recommended_items) >= num_recommendations:
                break
            for other_item_id in item_ratings[similar_user].keys():
                if other_item_id not in user_ratings_matrix.keys():
                    continue
                if other_item_id not in recommended_items:
                    recommended_items.append(other_item_id)
                    break

    return recommended_items
```

在这个实例中，我们首先计算用户之间的相似性，然后根据相似性为用户推荐最合适的产品。
