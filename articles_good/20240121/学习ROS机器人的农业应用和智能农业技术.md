### 4.1 定位与导航示例

```python
import rospy
from nav_msgs.msg import Odometry
from geometry_msgs.msg import Pose

def callback(odom):
    global current_pose
    current_pose = odom.pose.pose

def main():
    rospy.init_node('robot_localization')
    rospy.Subscriber('/odom', Odometry, callback)
    rospy.spin()
```
