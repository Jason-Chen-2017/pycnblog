import pandas as pd

### 4.4 数据分析

```python
import pandas as pd
from sklearn.cluster import KMeans

data = pd.read_csv('processed_data.csv')
kmeans = KMeans(n_clusters=3)
kmeans.fit(data)
data['cluster'] = kmeans.labels_
data.to_csv('clustered_data.csv', index=False)
```
