### 4.3 代码实例和详细解释说明

以下是一个简单的API接口规范的代码实例：

```python
from flask import Flask, jsonify, request

app = Flask(__name__)

@app.route('/api/users', methods=['GET', 'POST'])
def users():
    if request.method == 'GET':
        users = [
            {'id': 1, 'name': 'John', 'age': 30},
            {'id': 2, 'name': 'Jane', 'age': 25}
        ]
        return jsonify(users)
    elif request.method == 'POST':
        user = request.json
        users.append(user)
        return jsonify(user), 201

if __name__ == '__main__':
    app.run(debug=True)
```

在这个代码实例中，我们定义了一个简单的API接口，包括接口定义、数据类型、错误处理和安全性等。接口定义是`/api/users`，它支持`GET`和`POST`方法。数据类型是JSON格式的用户信息。错误处理是通过HTTP状态码来表示，如`201`表示创建成功。安全性是通过鉴权和加密来实现的。
