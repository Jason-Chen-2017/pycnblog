```python
import os

def deploy(environment):
    if environment == 'production':
        os.system('python manage.py deploy')
    elif environment == 'staging':
        os.system('python manage.py deploy_staging')

deploy('production')
```

解释说明：

- 使用`os`模块编写自动化部署代码。
- 定义一个`deploy`函数，接收`environment`参数。
- 根据`environment`参数执行不同的部署命令。
- 调用`deploy`函数，传入`production`参数进行部署。

### 4.4 自动化监控

```python
import time

def monitor(monitor_strategy):
    if monitor_strategy == 'cpu':
        while True:
            cpu_usage = os.popen('top -l 1').readlines()[1].split()[1]
            print(f'CPU Usage: {cpu_usage}%')
            time.sleep(5)
    elif monitor_strategy == 'memory':
        while True:
            memory_usage = os.popen('free -m').readlines()[1].split()[2]
            print(f'Memory Usage: {memory_usage}MB')
            time.sleep(5)

monitor('cpu')
```

解释说明：

- 使用`time`模块编写自动化监控代码。
- 定义一个`monitor`函数，接收`monitor_strategy`参数。
- 根据`monitor_strategy`参数执行不同的监控命令。
