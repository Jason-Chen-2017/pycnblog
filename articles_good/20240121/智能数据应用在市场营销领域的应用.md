5. 模型评估：对模型进行评估，以确保其效果和准确性。

数学模型公式详细讲解：

- 线性回归：线性回归是一种预测客户行为的方法，公式为：y = a * x + b，其中y是预测值，x是自变量，a是斜率，b是截距。
- 决策树：决策树是一种分类和预测客户行为的方法，公式为：if x1 <= v1 then y = f1 else if x2 <= v2 then y = f2 else ...，其中x1、x2...是特征，v1、v2...是阈值，f1、f2...是预测值。

from sklearn.linear_model import LinearRegression

# 数据预处理
data = data.dropna()

# 客户分析
X = data[['age', 'gender', 'income']]
y = data['purchase']
model = LinearRegression()
model.fit(X, y)

# 目标客户定位
X_test = [[25, 'female', 50000]]
y_pred = model.predict(X_test)

# 营销活动优化

y_train = data['purchase']
model = DecisionTreeClassifier()
model.fit(X_train, y_train)
