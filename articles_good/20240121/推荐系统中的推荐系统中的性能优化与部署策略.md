以下是一个基于Python的推荐系统的代码实例：

```python
import numpy as np
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv('data.csv')

# 计算物品之间的相似度
similarity = cosine_similarity(data)

# 计算用户的兴趣向量
user_interest = data['user_id'].value_counts()

# 计算物品的推荐得分
recommend_score = user_interest.multiply(similarity).sum(axis=1)

# 对推荐得分进行排序
recommend_list = recommend_score.sort_values(ascending=False)

# 输出推荐列表
print(recommend_list)
```
