### 4.1 导入流程图的最佳实践

我们可以使用以下代码实例来演示如何将流程图导入到代码中：

```python
import graphviz

dot = graphviz.Digraph(comment='My Flowchart')
dot.node('Start', 'Start')
dot.node('Process1', 'Process 1')
dot.node('Decision', 'Decision')
dot.node('End', 'End')

dot.edge('Start', 'Process1')
dot.edge('Process1', 'Decision')
dot.edge('Decision', 'End')

dot.view()
```

在这个例子中，我们使用了 `graphviz` 库来创建一个流程图。我们首先创建了一个有注释的有向图，然后添加了各个节点，并使用 `edge` 函数来表示节点之间的关系。最后，我们使用 `view` 函数来展示流程图。

### 4.2 导出流程图的最佳实践

我们可以使用以下代码实例来演示如何将代码导出为流程图：

```python
from graphviz import Digraph

dot = Digraph(comment='My Flowchart')
dot.attr('node', shape='box')

dot.node('Start', 'Start')
dot.node('Process1', 'Process 1')
dot.node('Decision', 'Decision')
dot.node('End', 'End')

dot.edge('Start', 'Process1')
dot.edge('Process1', 'Decision')
dot.edge('Decision', 'End')

dot.render('my_flowchart', view=True)
```

在这个例子中，我们使用了 `graphviz` 库来创建一个流程图。我们首先创建了一个有注释的有向图，然后添加了各个节点，并使用 `edge` 函数来表示节点之间的关系。最后，我们使用 `render` 函数来导出流程图，并使用 `view` 函数来展示流程图。
