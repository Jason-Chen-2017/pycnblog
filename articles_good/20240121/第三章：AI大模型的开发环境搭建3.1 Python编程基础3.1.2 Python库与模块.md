### 4.1 NumPy库

NumPy库是Python中最重要的数学库之一。NumPy库提供了大量的数学函数和数据结构，可以帮助我们快速处理大量数据。以下是NumPy库的一个简单示例：

```python
import numpy as np

# 创建一个1维数组
arr1 = np.array([1, 2, 3, 4, 5])
print(arr1)

# 创建一个2维数组
arr2 = np.array([[1, 2, 3], [4, 5, 6]])
print(arr2)

# 使用NumPy库进行数学运算
result = np.add(arr1, arr2)
print(result)
```

### 4.2 Pandas库

Pandas库是Python中最重要的数据分析库之一。Pandas库提供了DataFrame、Series等数据结构和函数，可以帮助我们快速处理和分析数据。以下是Pandas库的一个简单示例：

```python
import pandas as pd

# 创建一个DataFrame
df = pd.DataFrame({'A': [1, 2, 3, 4, 5], 'B': [6, 7, 8, 9, 10]})
print(df)

# 使用Pandas库进行数据分析
mean = df.mean()
print(mean)
```

TensorFlow库是Google开发的一个深度学习框架。TensorFlow库可以帮助我们快速构建和训练深度学习模型。以下是TensorFlow库的一个简单示例：

```python

    tf.keras.layers.Dense(units=32, activation='relu', input_shape=(10,)),
    tf.keras.layers.Dense(units=10, activation='softmax')

# 使用TensorFlow库训练模型
```
