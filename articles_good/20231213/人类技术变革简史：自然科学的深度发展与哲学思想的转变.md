```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [3, 5, 1, 6, 2, 4]
print(quick_sort(arr))
```

详细解释说明：

1. 首先，我们定义了一个 quick_sort 函数，它接收一个参数 arr（数组）并返回一个排序后的数组。

2. 然后，我们检查数组的长度，如果长度小于等于 1，说明数组已经排序，直接返回数组。

3. 接着，我们选择一个中间值作为基准值，然后将数组分为三个部分：小于基准值的部分、等于基准值的部分、大于基准值的部分。

4. 最后，我们递归地对小于基准值的部分和大于基准值的部分进行快速排序，然后将排序后的部分与等于基准值的部分拼接在一起，得到排序后的数组。
