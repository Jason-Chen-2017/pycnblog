以下是一个简单的线性回归代码实例：

```python
import numpy as np

def linear_regression(X, y):
    # 添加常数项
    X = np.hstack((np.ones((X.shape[0], 1)), X))
    # 求解参数
    theta = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)
    return theta

X = np.array([[1, 2], [2, 3], [3, 4]])
y = np.array([3, 4, 5])

theta = linear_regression(X, y)
print("参数theta:", theta)
```

4. 求解参数，使用最小二乘法。
