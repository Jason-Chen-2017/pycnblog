### 3. 算法编程题

**题目1：** 编写一个函数，实现数据安全加密和解密的功能。

**答案：** 
```python
import base64
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad

# 加密函数
def encrypt(plaintext, key):
    cipher = AES.new(key, AES.MODE_CBC)
    ct_bytes = cipher.encrypt(pad(plaintext.encode(), AES.block_size))
    iv = base64.b64encode(cipher.iv).decode('utf-8')
    ct = base64.b64encode(ct_bytes).decode('utf-8')
    return iv, ct

# 解密函数
def decrypt(iv, ct, key):
    try:
        iv = base64.b64decode(iv)
        ct = base64.b64decode(ct)
        cipher = AES.new(key, AES.MODE_CBC, iv)
        pt = unpad(cipher.decrypt(ct), AES.block_size)
        return pt.decode('utf-8')
    except (ValueError, KeyError):
        return None

# 示例
key = b'mqkxvwuyn1hujexl'  # 必须是16位
plaintext = "Hello, World!"
iv, encrypted_text = encrypt(plaintext, key)
print("Encrypted:", encrypted_text)
print("IV:", iv)

decrypted_text = decrypt(iv, encrypted_text, key)
print("Decrypted:", decrypted_text)
```

**解析：** 使用PyCryptoDome库实现AES加密与解密。`encrypt`函数将明文转换为密文，并返回IV和加密后的文本。`decrypt`函数使用IV和加密后的文本恢复明文。

**题目2：** 编写一个函数，实现图像去噪功能。

**答案：**
```python
import cv2
import numpy as np

# 去噪函数
def denoise_image(image_path, filter_size=5, sigma=1.0):
    image = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)
    denoised_image = cv2.bilateralFilter(image, filter_size, sigma, sigma*2)
    return denoised_image

# 示例
image_path = "example.jpg"
denoised_image = denoise_image(image_path)
cv2.imshow('Original', cv2.imread(image_path))
cv2.imshow('Denoised', denoised_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

**解析：** 使用OpenCV库实现图像双边滤波去噪。`denoise_image`函数读取图像并应用双边滤波器，返回去噪后的图像。`filter_size`控制滤波器的范围，`sigma`控制空间和响应的权重。
