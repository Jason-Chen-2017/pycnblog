**题目：** 对以下数据进行清洗，去除重复记录、无效数据和异常值。

```
[
    {"id": 1, "name": "张三", "age": 25, "email": "zhangsan@example.com"},
    {"id": 2, "name": "李四", "age": 30, "email": "lisi@example.com"},
    {"id": 3, "name": "王五", "age": 35, "email": "wangwu@example.com"},
    {"id": 4, "name": "张三", "age": 28, "email": "zhangsan@example.com"},
    {"id": 5, "name": "赵六", "age": 40, "email": "zhailiu@example.com"},
    {"id": 6, "name": "李四", "age": 32, "email": "lisi@example.com"},
    {"id": 7, "name": "刘七", "age": 25, "email": "liuqi@example.com"}
]
```

**答案：**

```python
import json

data = [
    {"id": 1, "name": "张三", "age": 25, "email": "zhangsan@example.com"},
    {"id": 2, "name": "李四", "age": 30, "email": "lisi@example.com"},
    {"id": 3, "name": "王五", "age": 35, "email": "wangwu@example.com"},
    {"id": 4, "name": "张三", "age": 28, "email": "zhangsan@example.com"},
    {"id": 5, "name": "赵六", "age": 40, "email": "zhaaliu@example.com"},
    {"id": 6, "name": "李四", "age": 32, "email": "lisi@example.com"},
    {"id": 7, "name": "刘七", "age": 25, "email": "liuqi@example.com"}
]

# 去除重复记录
unique_data = []
for item in data:
    if item not in unique_data:
        unique_data.append(item)

# 删除无效数据和异常值
clean_data = [item for item in unique_data if 0 < item['age'] <= 100]

# 输出清洗后的数据
print(json.dumps(clean_data, indent=2))
```

**解析：** 该代码首先使用列表推导式去除重复记录，然后使用列表推导式过滤掉无效数据和异常值。最终输出清洗后的数据。

### 2. 数据统计

**题目：** 统计以下数据中的年龄分布情况，并计算每个年龄段的人数占比。

```
[
    {"id": 1, "name": "张三", "age": 25},
    {"id": 2, "name": "李四", "age": 30},
    {"id": 3, "name": "王五", "age": 35},
    {"id": 4, "name": "赵六", "age": 40},
    {"id": 5, "name": "刘七", "age": 45},
    {"id": 6, "name": "王八", "age": 50},
    {"id": 7, "name": "陈九", "age": 55},
    {"id": 8, "name": "赵十", "age": 60}
]
```

**答案：**

```python
import json
from collections import defaultdict

data = [
    {"id": 1, "name": "张三", "age": 25},
    {"id": 2, "name": "李四", "age": 30},
    {"id": 3, "name": "王五", "age": 35},
    {"id": 4, "name": "赵六", "age": 40},
    {"id": 5, "name": "刘七", "age": 45},
    {"id": 6, "name": "王八", "age": 50},
    {"id": 7, "name": "陈九", "age": 55},
    {"id": 8, "name": "赵十", "age": 60}
]

# 统计年龄分布情况
age_distribution = defaultdict(int)
for item in data:
    age_distribution[item['age']] += 1

# 计算每个年龄段的人数占比
total = sum(age_distribution.values())
age占比 = {age: (count / total) * 100 for age, count in age_distribution.items()}

# 输出年龄分布情况及占比
print(json.dumps(age_distribution, indent=2))
print("年龄占比：")
print(json.dumps(age占比, indent=2))
```

**解析：** 该代码首先使用 defaultdict 统计每个年龄段的人数，然后计算每个年龄段的人数占比。最终输出年龄分布情况和占比。

### 4. 关联规则挖掘

**题目：** 使用 Apriori 算法进行关联规则挖掘，对以下交易数据集挖掘频繁项集和关联规则。

```
[
    [1, 2, 3],
    [1, 3, 4],
    [1, 5],
    [2, 3, 5],
    [2, 4, 5],
    [3, 4],
    [4, 5]
]
```

**答案：**

```python
import json
from mlxtend.frequent_patterns import apriori
from mlxtend.frequent_patterns import association_rules

transactions = json.loads('[1, 2, 3],[1, 3, 4],[1, 5],[2, 3, 5],[2, 4, 5],[3, 4],[4, 5]')

# 使用 Apriori 算法挖掘频繁项集
fp = apriori(transactions, min_support=0.5, use_colnames=True)

# 挖掘关联规则
rules = association_rules(fp, metric="lift", min_threshold=1)

# 输出频繁项集和关联规则
print("频繁项集：")
print(fp)
print("关联规则：")
print(rules)
```

**解析：** 该代码首先使用 mlxtend 库的 Apriori 算法挖掘频繁项集，然后使用关联规则算法挖掘关联规则。最终输出频繁项集和关联规则。

```
[
    [1, 2, 3],
    [2, 3, 4],
    [3, 5, 6],
    [4, 7, 8],
    [5, 9, 10],
    [6, 11, 12],
    [7, 13, 14],
```

```python
import pandas as pd
from sklearn.impute import SimpleImputer
from sklearn.preprocessing import StandardScaler
from sklearn.feature_selection import SelectKBest
from sklearn.feature_selection import f_classif

data = [
    [1, 2, 3],
    [2, 3, 4],
    [3, 5, 6],
    [4, 7, 8],
    [5, 9, 10],
    [6, 11, 12],
    [7, 13, 14],
    [8, 15, 16],
    [9, 17, 18],
    [10, 19, 20]
]

# 创建 DataFrame
df = pd.DataFrame(data)

# 缺失值处理
imputer = SimpleImputer(strategy="mean")
df = imputer.fit_transform(df)

# 数据转换
scaler = StandardScaler()
df = scaler.fit_transform(df)

# 特征选择
selector = SelectKBest(score_func=f_classif, k=2)
df = selector.fit_transform(df)

# 输出预处理后的数据

```
```
X = [[1, 2],
     [2, 3],

     [4, 5],
     [5, 6],
     [6, 7],
     [7, 8],
     [8, 9],
```

```python
from sklearn.cluster import KMeans

```

**解析：** 该代码首先创建 K-Means 聚类模型，然后使用训练数据进行模型训练，最后使用模型进行聚类。最终输出聚类结果。
