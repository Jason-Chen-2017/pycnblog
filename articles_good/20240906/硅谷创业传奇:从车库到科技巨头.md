## 算法编程题解析

### 1. 单调栈

**题目：** 实现一个单调栈，支持 push、pop、top 操作。

**答案：**

```python
class MonotonicStack:
    def __init__(self):
        self.stack = []

    def push(self, x: int) -> None:
        while self.stack and self.stack[-1] < x:
            self.stack.pop()
        self.stack.append(x)

    def pop(self) -> int:
        return self.stack.pop()

    def top(self) -> int:
        return self.stack[-1]
```

**解析：** 单调栈是一种特殊的栈，它保证了栈内的元素按升序或降序排列。在实现时，我们利用一个辅助栈来维护单调性。当新元素大于栈顶元素时，将栈顶元素弹出，直到新元素大于或等于栈顶元素。

### 2. 环形缓冲区

**题目：** 设计一个环形缓冲区，支持入队、出队和获取队首元素。

**答案：**

```python
from collections import deque

class CircularBuffer:
    def __init__(self, k: int):
        self.buffer = deque(maxlen=k)

    def enqueue(self, val: int) -> None:
        self.buffer.append(val)

    def dequeue(self) -> int:
        if not self.buffer:
            return -1
        return self.buffer.popleft()

    def peek(self) -> int:
        if not self.buffer:
            return -1
        return self.buffer[0]
```

**解析：** 环形缓冲区是一种循环利用的缓冲区。在这里，我们使用 `deque` 来实现。当缓冲区满时，新加入的元素会覆盖最早的元素，从而实现循环利用。

### 3. 合并区间

**题目：** 给定一组区间，合并所有重叠的区间。

**答案：**

```python
def merge(intervals):
    if not intervals:
        return []

    intervals.sort()
    merged = [intervals[0]]

    for i in range(1, len(intervals)):
        last = merged[-1]
        current = intervals[i]

        if last[1] >= current[0]:
            merged[-1] = [last[0], max(last[1], current[1])]
        else:
            merged.append(current)

    return merged
```

**解析：** 首先将区间按起始位置排序，然后依次合并区间。当当前区间的起始位置小于或等于前一个区间的结束位置时，合并两个区间。否则，将当前区间添加到合并后的区间列表中。

### 4. 最长公共前缀

**题目：** 找出字符串数组中的最长公共前缀。

**答案：**

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""

    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s) and prefix[i] == s[i]:
            i += 1
        prefix = prefix[:i]
        if not prefix:
            break

    return prefix
```

**解析：** 首先选择第一个字符串作为初始前缀，然后依次与前一个字符串比较，更新前缀。当两个字符串的对应字符不匹配时，更新前缀。重复这个过程，直到找到最长的公共前缀。

### 5. 合并两个有序链表

**题目：** 将两个升序链表合并为一个新的升序链表。

**答案：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def mergeTwoLists(l1, l2):
    dummy = ListNode()
    current = dummy

    while l1 and l2:
        if l1.val < l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next

    current.next = l1 or l2
    return dummy.next
```

**解析：** 创建一个虚拟头节点，用于简化链表的拼接过程。然后，比较两个链表的头节点，选择较小的值添加到新链表中，并将对应链表的头节点向后移动。重复这个过程，直到一个链表为空。最后，将非空链表的剩余部分接在合并后的链表后面。

### 6. 接雨水

**题目：** 给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

**答案：**

```python
def trap(height):
    if not height:
        return 0

    left_max = height[0]
    right_max = height[-1]
    water = 0

    for i in range(1, len(height)):
        left_max = max(left_max, height[i - 1])
        right_max = max(right_max, height[len(height) - i - 1])

        water += min(left_max, right_max) - height[i]

    return water
```

**解析：** 我们使用两个指针分别从左右两端遍历数组，同时维护左右两端的最大高度。对于当前位置，计算当前高度与左右两端最大高度的较小值，将其减去当前高度，即为当前位置能接的雨水。最后累加所有位置能接的雨水，即为最终答案。

### 7. 二进制表示中的质数个数

**题目：** 给定一个正整数 n，计算它在二进制表示中包含的质数个数的总和。

**答案：**

```python
def countPrimes(numOnes):
    if numOnes < 2:
        return 0

    primes = [True] * (numOnes + 1)
    primes[0] = primes[1] = False

    for i in range(2, int(numOnes ** 0.5) + 1):
        if primes[i]:
            primes[i*i: numOnes+1: i] = [False] * len(primes[i*i: numOnes+1: i])

    return sum(primes)
```

**解析：** 我们使用埃拉托斯特尼筛法（Sieve of Eratosthenes）来筛选出小于等于 numOnes 的所有质数。然后，遍历每个质数，将其在二进制表示中的位数相加。由于质数的二进制表示只包含 '1' 和 '0'，我们可以直接计算其位数。

### 8. 最小路径和

**题目：** 给定一个包含非负整数的 m x n 网格，找到一条从左上角到右下角的路径，使得路径上的数字总和为最小值。

**答案：**

```python
def minPathSum(grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]

    dp[0][0] = grid[0][0]

    for i in range(1, m):
        dp[i][0] = dp[i - 1][0] + grid[i][0]

    for j in range(1, n):
        dp[0][j] = dp[0][j - 1] + grid[0][j]

    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]

    return dp[-1][-1]
```

**解析：** 使用动态规划的方法。我们定义一个二维数组 dp，其中 dp[i][j] 表示到达坐标 (i, j) 的最小路径和。初始状态下，左上角的值等于 grid[0][0]。然后，我们遍历网格，计算每个位置的值，最后返回右下角的值。

### 9. 搜索旋转排序数组

**题目：** 搜索一个旋转排序的数组。

**答案：**

```python
def search(nums, target):
    left, right = 0, len(nums) - 1

    while left <= right:
        mid = (left + right) // 2

        if nums[mid] == target:
            return mid

        if nums[left] <= nums[mid]:
            if nums[left] <= target < nums[mid]:
                right = mid - 1
            else:
                left = mid + 1
        else:
            if nums[mid] < target <= nums[right]:
                left = mid + 1
            else:
                right = mid - 1

    return -1
```

**解析：** 搜索旋转排序数组可以看作是二分搜索的一种变体。我们需要确定旋转点的位置，然后根据旋转点的位置调整二分搜索的区间。

### 10. 两数之和

**题目：** 给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**答案：**

```python
def twoSum(nums, target):
    nums_dict = {v: i for i, v in enumerate(nums)}

    for i, v in enumerate(nums):
        complement = target - v
        if complement in nums_dict and nums_dict[complement] != i:
            return [i, nums_dict[complement]]

    return []
```

**解析：** 我们使用一个字典来存储每个数字及其下标。然后，遍历数组，对于每个数字，计算其补数，并在字典中查找补数是否存在。如果存在且下标不同，返回两个数字的下标。
