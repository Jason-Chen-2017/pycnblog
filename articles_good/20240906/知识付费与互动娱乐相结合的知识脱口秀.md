#### 算法编程题

**1. 如何实现一个简单的时间戳服务？**

**题目：** 编写一个 Go 程序，实现一个可以获取当前时间戳的服务。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

func currentTimeStamp() int64 {
    return time.Now().UnixNano()
}

func main() {
    fmt.Println("Current timestamp:", currentTimeStamp())
}
```

**解析：** 这个程序使用 `time.Now().UnixNano()` 方法获取当前时间戳，单位为纳秒。

**2. 如何实现一个简单的缓存系统？**

**题目：** 编写一个 Go 程序，实现一个缓存系统，支持添加、获取和删除缓存项。

**答案：**

```go
package main

import (
    "fmt"
    "sync"
)

type Cache struct {
    sync.RWMutex
    items map[string]interface{}
}

func NewCache() *Cache {
    return &Cache{
        items: make(map[string]interface{}),
    }
}

func (c *Cache) Set(key string, value interface{}) {
    c.Lock()
    defer c.Unlock()
    c.items[key] = value
}

func (c *Cache) Get(key string) (interface{}, bool) {
    c.RLock()
    defer c.RUnlock()
    value, ok := c.items[key]
    return value, ok
}

func (c *Cache) Delete(key string) {
    c.Lock()
    defer c.Unlock()
    delete(c.items, key)
}

func main() {
    cache := NewCache()
    cache.Set("key1", "value1")
    value, ok := cache.Get("key1")
    if ok {
        fmt.Println("Get:", value)
    }
    cache.Delete("key1")
    _, ok = cache.Get("key1")
    if !ok {
        fmt.Println("Key not found")
    }
}
```

**解析：** 这个程序定义了一个 `Cache` 结构体，用于实现缓存系统。`Set`、`Get` 和 `Delete` 方法分别用于添加、获取和删除缓存项。为了确保并发安全，使用了读写锁（`sync.RWMutex`）。
