##### 4. 数据去重算法

**题目：** 编写一个算法，实现对一个整数数组进行去重操作。

**答案：** 可以使用哈希表实现数据去重。

```python
def remove_duplicates(arr):
    seen = set()
    result = []
    for num in arr:
        if num not in seen:
            seen.add(num)
            result.append(num)
    return result
```

**解析：** 通过哈希表记录已出现的元素，实现高效的去重操作。

##### 5. 数据清洗算法

**题目：** 编写一个算法，对以下数据集进行清洗：

```python
data = [
    {"name": "张三", "age": 25, "city": "北京"},
    {"name": "李四", "age": 35, "city": "上海"},
    {"name": "王五", "age": 30, "city": "北京"},
    {"name": "赵六", "age": 28, "city": "北京"},
]
```

要求：去除重复数据，统一字段命名（如 city 转换为 city_lower）。

**答案：**

```python
def clean_data(data):
    cleaned_data = []
    seen = set()
    for item in data:
        key = tuple(sorted(item.items()))
        if key not in seen:
            seen.add(key)
            cleaned_item = {k.lower(): v for k, v in item.items()}
            cleaned_data.append(cleaned_item)
    return cleaned_data

cleaned_data = clean_data(data)
print(cleaned_data)
```

**解析：** 通过哈希表记录已处理的数据项，并统一字段命名，实现数据清洗。

##### 6. 数据分类算法

**题目：** 编写一个算法，对以下数据集进行分类，按照年龄分组：

```python
data = [
    {"name": "张三", "age": 25, "city": "北京"},
    {"name": "李四", "age": 35, "city": "上海"},
    {"name": "王五", "age": 30, "city": "北京"},
    {"name": "赵六", "age": 28, "city": "北京"},
]
```

**答案：**

```python
def classify_by_age(data):
    age_groups = {}
    for item in data:
        age = item["age"]
        if age not in age_groups:
            age_groups[age] = []
        age_groups[age].append(item)
    return age_groups

age_groups = classify_by_age(data)
print(age_groups)
```

**解析：** 根据年龄字段，将数据集按照年龄分组，实现数据分类。
