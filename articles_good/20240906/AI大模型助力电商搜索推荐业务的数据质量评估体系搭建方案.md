**题目：** 请描述一种方法来检测电商数据中的重复项。

**答案：** 可以使用哈希表（或字典）来实现。首先，遍历数据集中的每个记录，将其中的关键字段（如商品ID、用户ID等）作为键存储在哈希表中。如果发现键已存在，则说明存在重复项。

**示例代码：**

```python
def find_duplicates(data):
    seen = {}
    duplicates = []
    for item in data:
        key = tuple(item[0:2])  # 假设我们只检查前两个字段
        if key in seen:
            duplicates.append(item)
        else:
            seen[key] = item
    return duplicates
```

    # 假设recommendations是推荐列表，context是上下文信息
    relevant = 0

    # 假设recommendations是推荐列表，sales_data是销售数据
    sales = 0
    for item in recommendations:
        if item in sales_data:
            sales += sales_data[item]
    return sales
