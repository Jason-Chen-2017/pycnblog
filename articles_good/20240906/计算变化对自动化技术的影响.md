**1. 求最大公约数（GCD）**

**题目描述：**  
给定两个整数 `a` 和 `b`，求它们的最大公约数。

**输入：**  
输入包含两个整数 `a` 和 `b`。

**输出：**  
输出 `a` 和 `b` 的最大公约数。

**代码示例：**

```python
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a

a = int(input("请输入第一个整数："))
b = int(input("请输入第二个整数："))

print("最大公约数为：", gcd(a, b))
```

**4. 快速排序算法**

**题目描述：**  
实现快速排序算法，对输入的列表进行排序。

**输入：**  
输入包含一个列表，其中包含多个整数。

**输出：**  
输出排序后的列表。

**代码示例：**

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = list(map(int, input("请输入一个列表，用空格分隔：").split()))
print("排序后的列表：", quick_sort(arr))
```

**5. 合并两个有序链表**

**题目描述：**  
给定两个已排序的单链表，合并它们成为一个有序的单链表。

**输入：**  
输入包含两个字符串，分别表示两个有序链表，每个链表的节点值用空格分隔。

**输出：**  
输出合并后的有序链表。

**代码示例：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_sorted_lists(l1, l2):
    dummy = ListNode()
    current = dummy
    while l1 and l2:
        if l1.val < l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next
    current.next = l1 or l2
    return dummy.next

l1_values = list(map(int, input("请输入第一个链表，用空格分隔：").split()))
l2_values = list(map(int, input("请输入第二个链表，用空格分隔：").split()))

l1 = ListNode(l1_values[0])
current = l1
for value in l1_values[1:]:
    current.next = ListNode(value)
    current = current.next

l2 = ListNode(l2_values[0])
current = l2
for value in l2_values[1:]:
    current.next = ListNode(value)
    current = current.next

merged_head = merge_sorted_lists(l1, l2)
while merged_head:
    print(merged_head.val, end=" ")
    merged_head = merged_head.next
```

**题目描述：**  
在有序数组中查找给定值，返回其索引。如果不存在，返回 -1。

**输入：**  
输入包含两个部分：第一部分是一个有序数组，第二部分是一个整数，表示要查找的值。

**输出：**  
输出查找结果，即给定值的索引或 -1。

**代码示例：**

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

arr = list(map(int, input("请输入有序数组，用空格分隔：").split()))
target = int(input("请输入要查找的值："))

result = binary_search(arr, target)
print("索引为：", result)
```

**8. 矩阵乘法**

**题目描述：**  
实现矩阵乘法算法，计算两个给定矩阵的乘积。

**输入：**  
输入包含两个部分：第一部分是一个矩阵，第二部分是另一个矩阵。

**输出：**  
输出两个矩阵的乘积。

**代码示例：**

```python
def matrix_multiply(A, B):
    rows_A, cols_A = len(A), len(A[0])
    rows_B, cols_B = len(B), len(B[0])
    if cols_A != rows_B:
        return None
    result = [[0] * cols_B for _ in range(rows_A)]
    for i in range(rows_A):
        for j in range(cols_B):
            for k in range(cols_A):
                result[i][j] += A[i][k] * B[k][j]
    return result

A = [list(map(int, input().split())) for _ in range(int(input("请输入矩阵A的行数：")))]
B = [list(map(int, input().split())) for _ in range(int(input("请输入矩阵B的行数：")))]

result = matrix_multiply(A, B)
if result is None:
    print("无法进行矩阵乘法")
else:
    for row in result:
        print(" ".join(map(str, row)))
```

**输出：**

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def insert(root, val):
    if root is None:
        return TreeNode(val)
    if val < root.val:
        root.left = insert(root.left, val)
    else:
        root.right = insert(root.right, val)
    return root

def inorder_traversal(root):
    if root:
        inorder_traversal(root.left)
        print(root.val, end=" ")
        inorder_traversal(root.right)

values = list(map(int, input("请输入整数序列，用空格分隔：").split()))

for val in values:
    root = insert(root, val)

inorder_traversal(root)
```

**输入：**  
输入一个区间数组 `intervals`。

**输出：**  
输出不重叠的区间数组。

```python
def merge(intervals):
    if not intervals:
        return []
    intervals.sort(key=lambda x: x[0])
    result = [intervals[0]]
    for interval in intervals[1:]:
        last_end = result[-1][1]
        if interval[0] <= last_end:
            result[-1][1] = max(last_end, interval[1])
        else:
            result.append(interval)
    return result

intervals = [list(map(int, input().split())) for _ in range(int(input("请输入区间数组长度：")))]

print(merge(intervals))
```

**16. 最长公共前缀**

**题目描述：**  
编写一个函数来查找字符串数组中的最长公共前缀。

**输入：**  
输入包含一个字符串数组。

**输出：**  
输出最长公共前缀。

**代码示例：**

```python
def longest_common_prefix(strs):
    if not strs:
        return ""
    prefix = ""
    for char in strs[0]:
        for s in strs[1:]:
            if len(s) < len(prefix) or s[:len(prefix)] != prefix:
                return prefix
        prefix += char
    return prefix

strs = input("请输入字符串数组，用空格分隔：").split()

print("最长公共前缀：", longest_common_prefix(strs))
```

**17. 两数之和**

**题目描述：**  
给定一个整数数组 `nums` 和一个目标值 `target`，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**输入：**  
输入包含两个部分：第一部分是一个整数数组 `nums`，第二部分是一个整数 `target`。

**输出：**  
输出两个整数在数组中的索引，满足 `nums[index1] + nums[index2] = target`。

**代码示例：**

```python
def two_sum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

nums = list(map(int, input("请输入整数数组，用空格分隔：").split()))
target = int(input("请输入目标值："))

result = two_sum(nums, target)
print("索引为：", result)
```

**18. 盗贼的得手**

**题目描述：**  
一个专业的盗贼来到一个小镇，发现这里有 `N` 座房子，每座房子都有一个价值。盗贼想尽可能多地偷取财物，但不能连续偷两座房子。

**输入：**  
输入包含两个部分：第一部分是一个整数 `N`，表示房子的数量；第二部分是一个整数数组 `nums`，其中 `nums[i]` 表示第 `i` 座房子的价值。

**输出：**  
输出盗贼能偷取的最大财物价值。

**代码示例：**

```python
def rob(nums):
    if len(nums) == 1:
        return nums[0]
    prev_prev = nums[0]
    prev = max(nums[0], nums[1])
    for i in range(2, len(nums)):
        curr = max(prev, prev_prev + nums[i])
        prev_prev = prev
        prev = curr
    return prev

nums = list(map(int, input("请输入整数数组，用空格分隔：").split()))

print("最大财物价值：", rob(nums))
```

**19. 合并有序链表**

**题目描述：**  
给定两个有序单链表，将它们合并为一个有序单链表。

**输入：**  
输入包含两个部分：第一部分是一个字符串，表示第一个有序单链表，每个节点值用空格分隔；第二部分是一个字符串，表示第二个有序单链表，每个节点值用空格分隔。

**输出：**  
输出合并后的有序单链表。

**代码示例：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_sorted_lists(l1, l2):
    dummy = ListNode()
    current = dummy
    while l1 and l2:
        if l1.val < l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next
    current.next = l1 or l2
    return dummy.next

l1_values = list(map(int, input("请输入第一个链表，用空格分隔：").split()))
l2_values = list(map(int, input("请输入第二个链表，用空格分隔：").split()))

l1 = ListNode(l1_values[0])
current = l1
for value in l1_values[1:]:
    current.next = ListNode(value)
    current = current.next

l2 = ListNode(l2_values[0])
current = l2
for value in l2_values[1:]:
    current.next = ListNode(value)
    current = current.next

merged_head = merge_sorted_lists(l1, l2)
while merged_head:
    print(merged_head.val, end=" ")
    merged_head = merged_head.next
```

**20. 最长回文子串**

**题目描述：**  
给定一个字符串 `s`，找到其最长的回文子串。

**输入：**  
输入包含一个字符串 `s`。

**输出：**  
输出最长回文子串。

**代码示例：**

```python
def longest_palindromic_substring(s):
    start, max_len = 0, 1
    for i in range(len(s)):
        left, right = i, i
        while right < len(s) - 1 and s[right] == s[right + 1]:
            right += 1
        while left > 0 and right < len(s) and s[left - 1] == s[right]:
            left, right = left - 1, right + 1
        if right - left + 1 > max_len:
            start, max_len = left, right - left + 1
        left, right = i, i + 1
        while left > 0 and right < len(s) and s[left - 1] == s[right]:
            left, right = left - 1, right + 1
        if right - left + 1 > max_len:
            start, max_len = left, right - left + 1
    return s[start:start + max_len]

s = input("请输入字符串：")

print("最长回文子串：", longest_palindromic_substring(s))
```
