**题目：** 请解释被动房与主动房的主要区别。

**答案：** 被动房和主动房都是旨在实现高效能源利用的建筑类型，但它们的实现方式有所不同。

**被动房（Passive House）：** 被动房采用超高效的建筑设计和材料，以最大限度地减少能源需求。这些房屋通常具有极高的隔热性、密封性和被动式太阳能加热系统。它们的主要目标是实现极低的能源消耗，通常通过可再生能源来实现能源供给。

**主动房（Active House）：** 主动房则更加注重动态控制和优化能源使用。它们不仅采用了高效的设计和材料，还利用先进的控制系统来实时监测和调整能源使用，以达到更高的能源效率和舒适度。主动房通常配备有太阳能板、储能系统和其他智能设备。

**解析：** 被动房和主动房的主要区别在于实现方式。被动房依赖于被动式技术和设计，而主动房则更加依赖主动控制和优化技术。这两种建筑类型各有优势，可以根据具体需求和条件选择适用。

### 14. 建筑碳排放计算方法

**题目：** 请介绍建筑碳排放计算的方法和步骤，并给出一个计算实例。

**答案：** 建筑碳排放计算方法包括以下步骤：

1. **能源消耗数据收集：** 收集建筑各用能系统的能源消耗数据，如电力、天然气、石油等。
2. **能源碳排放系数获取：** 获取每种能源的碳排放系数，即单位能源消耗所产生的二氧化碳排放量。
3. **碳排放计算：** 将能源消耗数据乘以相应的碳排放系数，计算建筑总碳排放量。
4. **排放源分析：** 对建筑碳排放的主要排放源进行分析，如供暖、空调、照明等。

**实例：** 一栋办公楼的碳排放计算：

- **数据：** 办公楼年度电力消耗为100万kWh，天然气消耗为50万立方米。
- **碳排放系数：** 电力碳排放系数为0.7kgCO2e/kWh，天然气碳排放系数为3.2kgCO2e/m³。
- **计算：** 电力碳排放量 = 100万kWh × 0.7kgCO2e/kWh = 70,000kgCO2e；天然气碳排放量 = 50万m³ × 3.2kgCO2e/m³ = 160,000kgCO2e；总碳排放量 = 70,000kgCO2e + 160,000kgCO2e = 230,000kgCO2e。

**解析：** 通过碳排放计算，可以了解建筑碳排放情况，为减少碳排放提供依据。

**答案：** 建筑碳排放核算的方法和步骤包括：

1. **能源消耗数据收集：** 收集建筑各用能系统的能源消耗数据，如电力、天然气、石油等。
2. **碳排放系数获取：** 获取每种能源的碳排放系数，即单位能源消耗所产生的二氧化碳排放量。
3. **碳排放计算：** 将能源消耗数据乘以相应的碳排放系数，计算建筑总碳排放量。
4. **排放源分析：** 对建筑碳排放的主要排放源进行分析，如供暖、空调、照明等。

**核算实例：** 计算一栋商业建筑的碳排放：

- **数据：** 年度电力消耗为100万kWh，天然气消耗为50万立方米。
- **碳排放系数：** 电力碳排放系数为0.7kgCO2e/kWh，天然气碳排放系数为3.2kgCO2e/m³。
- **计算：** 电力碳排放量 = 100万kWh × 0.7kgCO2e/kWh = 70,000kgCO2e；天然气碳排放量 = 50万m³ × 3.2kgCO2e/m³ = 160,000kgCO2e；总碳排放量 = 70,000kgCO2e + 160,000kgCO2e = 230,000kgCO2e。

**解析：** 通过核算建筑碳排放，可以了解建筑对环境的影响，为节能减排和绿色建筑发展提供依据。

- **计算：** 电力碳排放量 = 100万kWh × 0.7kgCO2e/kWh = 70,000kgCO2e；天然气碳排放量 = 50万m³ × 3.2kgCO2e/m³ = 160,000kgCO2e；总碳排放量 = 70,000kgCO2e + 160,000kgCO2e = 230,000kgCO2e。
