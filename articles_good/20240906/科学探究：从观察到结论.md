#### 一、典型面试题

##### 1. 如何实现二分查找算法？

**答案：** 二分查找是一种高效的查找算法，它的工作原理是通过将有序数组的中点与目标值比较，从而决定是继续查找左侧还是右侧。以下是二分查找的伪代码：

```pseudo
function binarySearch(arr, target):
    low = 0
    high = arr.length - 1

    while low <= high:
        mid = (low + high) / 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return -1
```

**解析：** 这个算法的时间复杂度是 O(log n)，适用于大规模数据的查找操作。

##### 2. 如何实现快速排序算法？

**答案：** 快速排序是一种分治算法，通过递归地将数组分割成较小的数组，然后对它们进行排序。以下是快速排序的伪代码：

```pseudo
function quicksort(arr):
    if arr.length <= 1:
        return arr

    pivot = arr[arr.length // 2]
    left = []
    middle = []
    right = []

    for x in arr:
        if x < pivot:
            left.append(x)
        elif x == pivot:
            middle.append(x)
        else:
            right.append(x)

    return quicksort(left) + middle + quicksort(right)
```

**解析：** 快速排序的时间复杂度平均为 O(n log n)，但最坏情况下为 O(n^2)。因此，在实际应用中，有时会选择其他排序算法。

##### 3. 如何实现贪心算法？

**答案：** 贪心算法是一种在每一步选择上采取当前最优解的策略，从而得到全局最优解。以下是贪心算法的一个例子：找到数组中的最大子序列和。

```python
def maxSubArray(nums):
    cur_sum = max_sum = nums[0]
    for num in nums[1:]:
        cur_sum = max(num, cur_sum + num)
        max_sum = max(max_sum, cur_sum)
    return max_sum
```

**解析：** 贪心算法通常适用于具有局部最优解导致全局最优解的问题。

#### 二、算法编程题

##### 1. 寻找旋转排序数组中的最小值

假设按照升序排序的数组在预先未知的某个点上进行了旋转。

请你找出并返回数组的最小元素。

示例 1：

输入：nums = [3,4,5,1,2]
输出：1
解释：原数组为 [1,2,3,4,5]，在索引 3 处旋转，最小元素是 1。

示例 2：

输入：nums = [4,5,6,7,0,1,2]
输出：0
解释：原数组为 [0,1,2,4,5,6,7]，在索引 4 处旋转，最小元素是 0。

示例 3：

输入：nums = [1]
输出：1
解释：原数组为 [1]，未旋转，最小元素是 1。

**答案：**

```java
public int findMin(int[] nums) {
    int left = 0, right = nums.length - 1;
    while (left < right) {
        int mid = (left + right) / 2;
        if (nums[mid] > nums[right]) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    return nums[left];
}
```

**解析：** 使用二分查找的方法，当中间值大于最右侧的值时，说明最小值在右侧，否则在左侧。

##### 2. 最长公共前缀

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

示例 1：

输入：strs = ["flower","flow","flight"]
输出："fl"

示例 2：

输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。

**答案：**

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        while not s.startswith(prefix):
            prefix = prefix[:-1]
    return prefix
```

**解析：** 遍历每个字符串，从第一个字符串开始，不断缩短前缀，直到找到一个公共前缀。

##### 3. 两数之和

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

示例：

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9，所以返回 [0, 1]。

**答案：**

```java
public int[] twoSum(int[] nums, int target) {
    int[] result = new int[2];
    Map<Integer, Integer> numMap = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int complement = target - nums[i];
        if (numMap.containsKey(complement)) {
            result[0] = numMap.get(complement);
            result[1] = i;
            return result;
        }
        numMap.put(nums[i], i);
    }
    return result;
}
```

**解析：** 使用哈希表存储已遍历的数值及其索引，对于当前遍历的数值，计算其补数，并检查哈希表中是否存在补数。
