#### 算法编程题库

1. **最小生成树算法**

   **题目：** 请使用 Prim 算法实现一个函数，用于计算无向图的最小生成树。

   **答案：** Prim 算法的基本思想是从一个顶点开始，逐步增加一个顶点和与该顶点相邻的边，直到所有顶点都被包含在最小生成树中。

   ```python
   def prim算法(G):
       # G 是无向图的邻接矩阵
       n = len(G)
       mst = []  # 最小生成树的边集合
       visited = [False] * n  # 记录已访问的顶点
       start = 0  # 从顶点 0 开始
       visited[start] = True
       for _ in range(n - 1):
           # 寻找与已访问顶点相邻的最小权重边
           min_edge = None
           for i in range(n):
               for j in range(n):
                   if G[i][j] != 0 and not visited[j]:
                       if min_edge is None or G[i][j] < G[min_edge[0]][min_edge[1]]:
                           min_edge = (i, j)
           # 添加最小权重边到最小生成树中
           mst.append(min_edge)
           # 标记新加入的顶点为已访问
           visited[min_edge[1]] = True
       return mst
   ```

2. **最短路径算法**

   **题目：** 请使用 Dijkstra 算法实现一个函数，用于计算单源最短路径。

   **答案：** Dijkstra 算法的基本思想是使用一个优先队列来维护当前找到的最短路径，并逐步扩展到相邻的顶点。

   ```python
   import heapq

   def dijkstra(G, start):
       # G 是无向图的邻接矩阵
       n = len(G)
       dist = [float('inf')] * n  # 初始化最短路径长度
       dist[start] = 0
       visited = [False] * n  # 记录已访问的顶点
       priority_queue = [(0, start)]  # 优先队列
       while priority_queue:
           # 取出优先队列中最短路径的顶点
           current_dist, current_vertex = heapq.heappop(priority_queue)
           if visited[current_vertex]:
               continue
           visited[current_vertex] = True
           for neighbor, weight in enumerate(G[current_vertex]):
               if weight != 0 and not visited[neighbor]:
                   # 更新邻居顶点的最短路径长度
                   new_dist = current_dist + weight
                   if new_dist < dist[neighbor]:
                       dist[neighbor] = new_dist
                       heapq.heappush(priority_queue, (new_dist, neighbor))
       return dist
   ```

3. **排序算法**

   **题目：** 请实现一个快速排序算法。

   **答案：** 快速排序的基本思想是通过一趟排序将待排序的记录分割成独立的两部分，其中一部分记录的关键字均比另一部分的关键字小，然后递归地对这两部分继续进行排序。

   ```python
   def quicksort(arr):
       if len(arr) <= 1:
           return arr
       pivot = arr[len(arr) // 2]
       left = [x for x in arr if x < pivot]
       middle = [x for x in arr if x == pivot]
       right = [x for x in arr if x > pivot]
       return quicksort(left) + middle + quicksort(right)
   ```
