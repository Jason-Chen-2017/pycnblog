### 1. 调节光照强度

**题目：** 编写一个算法，根据用户生物节律调节室内光照强度。

**答案：**

```python
def adjust_lighting(biorhythm_data):
    """
    调节光照强度的函数
    :param biorhythm_data: 用户生物节律数据
    :return: 调节后的光照强度值
    """
    # 根据生物节律数据计算光照强度
    light_intensity = 100  # 初始光照强度为 100
    for data in biorhythm_data:
        if data['activity'] == 'sleep':
            light_intensity -= 10  # 睡眠时降低光照强度
        elif data['activity'] == 'work':
            light_intensity += 10  # 工作时增加光照强度
    return max(0, min(100, light_intensity))  # 光照强度限制在 0 到 100 之间

# 示例数据
biorhythm_data = [
    {'activity': 'sleep', 'time': '00:00'},
    {'activity': 'work', 'time': '08:00'},
    {'activity': 'sleep', 'time': '22:00'}
]

# 调节光照强度
adjusted_lighting = adjust_lighting(biorhythm_data)
print("调整后的光照强度：", adjusted_lighting)
```

**解析：** 该算法根据用户生物节律数据中的活动类型和时间，动态调整光照强度，以满足用户需求。

### 2. 调节室内温度

**题目：** 编写一个算法，根据用户生物节律调节室内温度。

**答案：**

```python
def adjust_temperature(biorhythm_data):
    """
    调节室内温度的函数
    :param biorhythm_data: 用户生物节律数据
    :return: 调节后的温度值
    """
    # 根据生物节律数据计算温度
    temperature = 22  # 初始温度为 22°C
    for data in biorhythm_data:
        if data['activity'] == 'sleep':
            temperature -= 2  # 睡眠时降低温度
        elif data['activity'] == 'work':
            temperature += 2  # 工作时增加温度
    return max(18, min(30, temperature))  # 温度限制在 18°C 到 30°C 之间

# 示例数据
biorhythm_data = [
    {'activity': 'sleep', 'time': '00:00'},
    {'activity': 'work', 'time': '08:00'},
    {'activity': 'sleep', 'time': '22:00'}
]

# 调节室内温度
adjusted_temperature = adjust_temperature(biorhythm_data)
print("调整后的温度：", adjusted_temperature)
```

**解析：** 该算法根据用户生物节律数据中的活动类型和时间，动态调整室内温度，以满足用户需求。
