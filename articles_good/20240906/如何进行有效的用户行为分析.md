* 计算用户登录次数
* 统计用户活跃天数
* 分析用户访问间隔

**举例：**

```go
// 统计用户活跃天数
func activeDays(users []string) map[string]int {
    days := make(map[string]int)
    for _, user := range users {
        days[user] = 0
    }
    for _, user := range users {
        for _, day := range users {
            if user == day {
                days[user]++
                break
            }
        }
    }
    return days
}

// 输出：map[user1:7, user2:5, user3:3]
```

**解析：** 通过统计用户登录次数、活跃天数和访问间隔，可以了解用户的使用频率，进而优化产品运营策略。
