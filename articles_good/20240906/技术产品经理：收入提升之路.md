## 算法编程题库

### 1. 如何实现一个LRU（最近最少使用）缓存？

**答案：** LRU缓存是一种常用的缓存算法，其实现思路是利用一个队列来记录数据的访问顺序，当缓存容量达到上限时，根据访问顺序移除最近最少使用的数据。

以下是Python实现的一个简单LRU缓存：

```python
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = OrderedDict()

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.cache.move_to_end(key)
        return self.cache[key]

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)
```

### 2. 如何求解二分查找的最左/最右边界？

**答案：** 对于一个有序数组，二分查找的最左边界是第一个满足条件的位置，最右边界是最后一个满足条件的位置。

以下是Python实现的二分查找最左边界：

```python
def search_left(nums, target):
    left, right = 0, len(nums) - 1
    while left < right:
        mid = (left + right) // 2
        if nums[mid] < target:
            left = mid + 1
        else:
            right = mid
    return left if left < len(nums) and nums[left] == target else -1
```

最右边界类似，只需在找到目标值时，将`left`更新为`right`：

```python
def search_right(nums, target):
    left, right = 0, len(nums) - 1
    while left < right:
        mid = (left + right + 1) // 2
        if nums[mid] > target:
            right = mid - 1
        else:
            left = mid
    return left if left < len(nums) and nums[left] == target else -1
```

### 3. 如何实现一个优先队列（堆）？

**答案：** 优先队列是一种特殊的队列，元素按照优先级顺序排列。在Python中，可以使用堆（Heap）来实现优先队列。

以下是Python实现的优先队列：

```python
import heapq

class PriorityQueue:
    def __init__(self):
        self.heap = []
    
    def push(self, item, priority):
        heapq.heappush(self.heap, (-priority, item))
    
    def pop(self):
        return heapq.heappop(self.heap)[1]
    
    def is_empty(self):
        return len(self.heap) == 0
```

通过`push`方法添加元素，元素按照优先级排序，其中优先级越高，值越小。`pop`方法获取并移除优先级最高的元素。
