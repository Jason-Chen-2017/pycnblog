##### 1. 如何优化餐饮配送路线？

**题目：** 请简述如何使用算法优化餐饮配送路线。

**答案：** 
使用基于贪心算法的最近邻（Nearest Neighbor）方法，或者基于图论的Dijkstra算法和A*算法。

**解析：**
- **最近邻算法：** 选择当前未配送的餐厅中离当前配送员最近的餐厅作为下一站，直到所有餐厅配送完毕。
- **Dijkstra算法：** 以配送员位置为起点，逐步计算到每个餐厅的最短路径，每次选择未配送的餐厅中路径最短的一个作为下一站。
- **A*算法：** 结合距离估算和实际距离，优先选择估计总路径最短的餐厅进行配送。

**示例代码：**
```go
// 最近邻算法示例
func optimizeRoutes(restaurantCoords map[int][2]float64) []int {
    unassigned := make([]int, 0, len(restaurantCoords))
    for id := range restaurantCoords {
        unassigned = append(unassigned, id)
    }
    
    route := make([]int, 0)
    for len(unassigned) > 0 {
        // 假设 current 是配送员当前位置，findNext 找到离 current 最近的未配送餐厅
        next := findNext(restaurantCoords, unassigned)
        route = append(route, next)
        unassigned = removeElement(unassigned, next)
    }
    
    return route
}
```
