#### 2. 如何利用数据分析提升行动力与管理效果？

**编程题：** 编写一个程序，利用数据分析提升行动力与管理效果。

**输入：** 一组员工工作记录数据，包括工作时间、工作效率、工作质量等指标。

**输出：** 优化后的员工工作计划，提升管理效果。

**代码实例：**

```python
import pandas as pd

# 假设工作记录数据存储在一个名为data.csv的文件中
data = pd.read_csv('data.csv')

# 计算每个员工的工作效率平均值
efficiency_avg = data['工作效率'].mean()

# 计算每个员工的工作质量平均值
quality_avg = data['工作质量'].mean()

# 筛选出工作效率和工作质量高于平均值的员工
top_employees = data[(data['工作效率'] > efficiency_avg) & (data['工作质量'] > quality_avg)]

# 根据筛选结果优化员工工作计划
optimized_plan = top_employees[['员工姓名', '工作时间', '工作效率', '工作质量']].sort_values(by='工作效率', ascending=False)

# 输出优化后的工作计划
print(optimized_plan)
```

**解析：** 该程序利用数据分析，筛选出工作效率和工作质量较高的员工，并根据结果调整工作计划，从而提升管理效果。在实际应用中，可以根据具体情况调整数据分析和优化策略。
