#### 3. 算法编程题库

**算法编程题1：** 最长公共子序列（LCS）

**题目描述：** 给定两个字符串 `s1` 和 `s2`，找出它们的最长公共子序列。

**答案：** 使用动态规划算法实现。

```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[m][n]

s1 = "AGGTAB"
s2 = "GXTXAYB"
print(longest_common_subsequence(s1, s2))  # 输出 4
```

**算法编程题2：** 单调栈

**题目描述：** 给定一个数组 `arr`，使用单调栈实现一个函数，找出每个元素左边和右边的第一个比它大的元素。

**答案：** 使用两个栈实现。

```python
def next_greater_elements(arr):
    n = len(arr)
    left = [None] * n
    right = [None] * n
    stack = []

    for i in range(n):
        while stack and arr[stack[-1]] < arr[i]:
            stack.pop()
        if stack:
            left[i] = arr[stack[-1]]
        stack.append(i)

    stack = []
    for i in range(n-1, -1, -1):
        while stack and arr[stack[-1]] <= arr[i]:
            stack.pop()
        if stack:
            right[i] = arr[stack[-1]]
        stack.append(i)

    return left, right

arr = [4, 5, 2, 25]
print(next_greater_elements(arr))  # 输出 ([None, None, 25, 25], [None, 25, 25, None])
```
