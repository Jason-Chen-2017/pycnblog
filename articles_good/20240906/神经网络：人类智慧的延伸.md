## 三、神经网络算法编程题

### 1. 简单神经网络实现

**题目：** 编写一个简单的神经网络，实现输入和输出之间的映射。

**答案：** 请参考以下Python代码实现：

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def neural_network(input_data, weights, bias):
    return sigmoid(np.dot(input_data, weights) + bias)

# 示例
input_data = np.array([1, 0])
weights = np.array([[0.5, 0.5], [0.5, 0.5]])
bias = -0.5

output = neural_network(input_data, weights, bias)
print(output)
```

### 2. 多层神经网络实现

**题目：** 编写一个多层神经网络，实现输入和输出之间的映射。

**答案：** 请参考以下Python代码实现：

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def neural_network(input_data, layers, biases):
    activation = input_data
    for layer, bias in zip(layers, biases):
        activation = sigmoid(np.dot(activation, layer) + bias)
    return activation

# 示例
input_data = np.array([1, 0])
weights1 = np.array([[0.5, 0.5], [0.5, 0.5]])
weights2 = np.array([[0.5, 0.5], [0.5, 0.5]])
biases1 = -0.5
biases2 = -0.5

output = neural_network(input_data, [weights1, weights2], [biases1, biases2])
print(output)
```

### 5. 生成对抗网络（GAN）

**题目：** 请简述生成对抗网络（GAN）的基本原理和优缺点。

**答案：** GAN是一种由生成器和判别器对抗训练的神经网络模型，用于生成具有真实数据分布的样本。基本原理是生成器和判别器之间的对抗性训练，生成器试图生成逼真的数据，判别器则试图区分真实数据和生成数据。GAN的优点包括：

* 强大的生成能力，可以生成高质量的数据。
* 可以学习数据的高层次特征。

但GAN也存在一些缺点：

* 训练过程不稳定，容易陷入模式崩溃。
* 需要大量的训练数据和计算资源。
