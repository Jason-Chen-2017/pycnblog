#### 编程题 1：优先队列实现

**题目：** 实现一个优先队列，支持插入和删除操作，并按照优先级顺序返回队列中的元素。

**答案：** 以下是一个基于堆（heap）实现的优先队列：

```python
import heapq

class PriorityQueue:
    def __init__(self):
        self.heap = []
        self.count = 0

    def insert(self, element, priority):
        entry = (priority, self.count, element)
        heapq.heappush(self.heap, entry)
        self.count += 1

    def remove(self):
        _, _, element = heapq.heappop(self.heap)
        return element

    def is_empty(self):
        return len(self.heap) == 0
```

**解析：** 该优先队列使用最小堆实现，插入和删除操作的时间复杂度均为 O(log n)。

### 源代码实例

#### 源代码示例：注意力监控系统

以下是一个简化的注意力监控系统的源代码示例，用于记录用户在网站上的活动并生成注意力报告。

```python
import time

class ActivityLogger:
    def __init__(self):
        self.activity_log = []

    def log_activity(self, user_id, activity, duration):
        entry = {
            'user_id': user_id,
            'activity': activity,
            'duration': duration,
            'timestamp': time.time()
        }
        self.activity_log.append(entry)

    def generate_report(self, user_id):
        report = []
        for entry in self.activity_log:
            if entry['user_id'] == user_id:
                report.append(entry)
        return report
```

**解析：** 该代码示例实现了用户活动记录和报告生成功能，可用于监控和分析用户注意力分布。
