#### 算法编程题 1：最长公共子序列

**题目描述：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。最长公共子序列是指同时出现在 `str1` 和 `str2` 中，且最长的不相交子序列。

**答案：** 可以使用动态规划的方法求解。

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

# 示例
str1 = "ABCD"
str2 = "ACDF"
print(longest_common_subsequence(str1, str2))  # 输出 3
```

**解析：** 使用二维数组 `dp` 来记录状态，其中 `dp[i][j]` 表示 `str1` 的前 `i` 个字符和 `str2` 的前 `j` 个字符的最长公共子序列的长度。

#### 算法编程题 2：股票买卖最佳时机

**题目描述：** 给定一个整数数组 `prices`，其中 `prices[i]` 是第 `i` 天的股票价格。如果在一个交易日，你最多只允许完成一笔交易（即买入和卖出一股股票），设计一个算法来计算你所能获取的最大利润。

**答案：** 可以使用一次遍历的方法求解。

```python
def max_profit(prices):
    max_profit = 0
    for i in range(1, len(prices)):
        profit = prices[i] - prices[i - 1]
        max_profit = max(max_profit, profit)
    return max_profit

# 示例
prices = [7, 1, 5, 3, 6, 4]
print(max_profit(prices))  # 输出 5
```

**解析：** 遍历数组，每次计算当前价格与前一天价格的差值，更新最大利润。

#### 算法编程题 3：最长连续序列

**题目描述：** 给定一个未排序的整数数组，找出最长连续序列的长度。

**答案：** 可以使用哈希表的方法求解。

```python
def longest_consecutive(nums):
    if not nums:
        return 0
    num_set = set(nums)
    max_length = 0

    for num in num_set:
        if num - 1 not in num_set:
            current_num = num
            current_length = 1
            while current_num + 1 in num_set:
                current_num += 1
                current_length += 1
            max_length = max(max_length, current_length)

    return max_length

# 示例
nums = [100, 4, 200, 1, 3, 2]
print(longest_consecutive(nums))  # 输出 4
```

**解析：** 遍历哈希表，找到每个数的连续序列长度，更新最大长度。
