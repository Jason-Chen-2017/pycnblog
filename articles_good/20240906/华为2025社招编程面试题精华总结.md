## 算法与数据结构

### 1. 快排（Quick Sort）

**题目：** 实现一个快速排序算法。

**答案：** 快速排序是一种分治算法，其基本思想是通过一趟排序将待排序的记录分隔成独立的两部分，其中一部分记录的关键字均比另一部分的关键字小，则可分别对这两部分记录继续进行排序，以达到整个序列有序。

```go
func quickSort(arr []int, low, high int) {
    if low < high {
        pi := partition(arr, low, high)
        quickSort(arr, low, pi-1)
        quickSort(arr, pi+1, high)
    }
}

func partition(arr []int, low, high int) int {
    pivot := arr[high]
    i := low
    for j := low; j < high; j++ {
        if arr[j] < pivot {
            arr[i], arr[j] = arr[j], arr[i]
            i++
        }
    }
    arr[i], arr[high] = arr[high], arr[i]
    return i
}
```

### 2. 最长公共子序列（Longest Common Subsequence）

**题目：** 给定两个字符串，找到它们的最长公共子序列。

**答案：** 使用动态规划求解。定义一个二维数组 dp，其中 dp[i][j] 表示字符串 s1 的前 i 个字符和字符串 s2 的前 j 个字符的最长公共子序列长度。

```go
func longestCommonSubsequence(s1, s2 string) int {
    m, n := len(s1), len(s2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if s1[i-1] == s2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

## 编程语言基础

### 3. Golang 中函数参数传递方式

**题目：** Golang 中函数参数是如何传递的？

**答案：** Golang 中函数参数传递方式是值传递，即传递的是参数的拷贝。

```go
package main

import "fmt"

func modify(x int) {
    x = 100
}

func main() {
    a := 10
    modify(a)
    fmt.Println(a) // 输出 10，而不是 100
}
```

### 4. Golang 中 Goroutine 的调度策略

**题目：** Golang 中 Goroutine 的调度策略是什么？

**答案：** Golang 中的 Goroutine 调度策略是基于 M:N 调度模型。其中 M 表示运行的 Goroutine 数量，N 表示系统线程数量。Goroutine 会按照一定策略分配到系统线程上执行。

## 并发编程

### 5. Golang 中如何实现线程安全？

**题目：** Golang 中如何实现线程安全？

**答案：** 可以使用以下方法实现线程安全：

* **互斥锁（Mutex）：** 保护共享资源，防止并发访问。
* **读写锁（RWMutex）：** 允许多个读取操作同时进行，但只允许一个写入操作。
* **原子操作（Atomic）：** 保证对共享变量的原子性操作。

```go
package main

import (
    "fmt"
    "sync"
)

var (
    counter int
    mu      sync.Mutex
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    counter++
}

func main() {
    var wg sync.WaitGroup
    for i := 0; i < 1000; i++ {
        wg.Add(1)
        go func() {
            defer wg.Done()
            increment()
        }()
    }
    wg.Wait()
    fmt.Println("Counter:", counter)
}
```
