### 2. 逻辑回归模型

**问题：** 请解释逻辑回归模型的原理和应用场景。

**答案：** 逻辑回归模型是一种用于预测概率的二分类模型。它的原理是通过寻找一个线性函数来表示概率的对数，然后使用 Sigmoid 函数将结果映射到 0 和 1 之间。应用场景包括垃圾邮件检测、欺诈检测、客户流失预测等。

**解析：** 逻辑回归模型通过最大化似然估计来训练模型，适用于二分类问题。它可以帮助我们预测某个样本属于正类的概率，从而进行分类决策。

### 3. 决策树模型

**解析：** 决策树模型易于理解和解释，适用于各种类型的数据和问题。它可以提供直观的决策路径，帮助用户理解数据背后的关系。

```python
import numpy as np

def compute_cost(X, y, theta):
    m = len(y)
    h = X.dot(theta)
    error = (h - y)
    J = (1 / (2 * m)) * (error.T.dot(error))
    return J

def gradient_descent(X, y, theta, alpha, iters):
    m = len(y)
    for i in range(iters):
        h = X.dot(theta)
        error = h - y
        theta = theta - (alpha / m) * (X.T.dot(error))
        J = compute_cost(X, y, theta)
        print(f"Iteration {i+1}: J = {J}")
    return theta

# 初始化参数
X = np.array([[1, 1], [1, 2], [1, 3], [1, 4]])
y = np.array([2, 4, 6, 8])
theta = np.zeros((2, 1))
alpha = 0.01
iters = 1500

# 训练模型
theta_final = gradient_descent(X, y, theta, alpha, iters)
print("Final theta:", theta_final)

# 预测
X_new = np.array([[1, 5]])
y_new = X_new.dot(theta_final)
print("Predicted value:", y_new)

**解析：** 在这个示例中，我们使用了梯度下降算法来训练线性回归模型。`compute_cost` 函数用于计算成本函数 J，`gradient_descent` 函数用于迭代更新参数 theta。通过调用这些函数，我们可以训练模型并进行预测。

import numpy as np

def entropy(y):
    hist = np.bincount(y)
    probabilities = hist / len(y)
    return -np.sum([p * np.log2(p) for p in probabilities if p > 0])

def information_gain(y, y_left, y_right):
    p_left = len(y_left) / len(y)
    p_right = len(y_right) / len(y)
    return entropy(y) - (p_left * entropy(y_left) + p_right * entropy(y_right))

def find_best_split(X, y):
    best_feature = None
    best_value = None
    best_info_gain = -1
    current_entropy = entropy(y)

    for feature in range(X.shape[1]):
        unique_values = np.unique(X[:, feature])
        for value in unique_values:
            left_indices = np.where(X[:, feature] == value)[0]
            right_indices = np.where(X[:, feature] != value)[0]
            y_left = y[left_indices]
            y_right = y[right_indices]
            info_gain = information_gain(y, y_left, y_right)
            if info_gain > best_info_gain:
                best_info_gain = info_gain
                best_feature = feature
                best_value = value

    return best_feature, best_value

# 示例数据
X = np.array([[1, 1], [1, 2], [1, 1], [0, 0], [0, 1], [0, 1]])
y = np.array([1, 1, 1, 0, 0, 0])

# 构建决策树
best_feature, best_value = find_best_split(X, y)
print("Best feature:", best_feature)
print("Best value:", best_value)
```
