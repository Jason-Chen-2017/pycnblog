**题目：** 编写一个算法，计算社交网络中信息扩散的影响范围。

**答案：** 基于影响力模型，可以采用以下算法计算社交网络中信息扩散的影响范围：

1. 初始化一个影响范围数组，所有元素的初始值为0。
2. 遍历社交网络中的用户，对于每个用户，将其影响力值加到其好友的影响范围上。
3. 输出所有用户的影响范围。

**示例代码（Python）：**

```python
from collections import defaultdict

def calculate_influence_network(network, influence_values):
    influence_range = defaultdict(int)
    for user, friends in network.items():
        for friend in friends:
            influence_range[friend] += influence_values[user]
    return influence_range
```
