## 4.2 自动化备份和恢复的代码实例

以下是一个自动化备份和恢复的代码实例：

```python
import mysql
import time

# 连接数据库
conn = mysql.connect(host='localhost', user='root', password='123456', database='test')
cursor = conn.cursor()

# 执行备份操作
def backup():
    backup_file = 'test_backup.sql'
    cursor.execute('SELECT * FROM test')
    data = cursor.fetchall()
    with open(backup_file, 'w') as f:
        for row in data:
            f.write(' '.join(map(str, row)) + '\n')
    print('备份成功')

# 执行恢复操作
def restore(backup_file):
    with open(backup_file, 'r') as f:
        data = f.readlines()
        cursor.executemany('INSERT INTO test VALUES (%s, %s, %s)', data)
        conn.commit()
    print('恢复成功')

# 执行备份操作
backup()

# 执行恢复操作
restore('test_backup.sql')
```

在上述代码实例中，我们首先通过自动化工具连接到数据库，然后执行备份操作。接着，我们执行恢复操作，并输出恢复结果。通过这种方式，我们可以对数据库进行备份和恢复。
