## 4.1 交通信息查询
我们可以使用 Python 编程语言来实现一个简单的交通信息查询系统。首先，我们需要收集一些交通数据，如交通流量、交通状况等数据。然后，我们可以使用这些数据来实现交通信息查询的功能。

```python
import requests

def get_traffic_info(city):
    url = f"http://api.map.baidu.com/tea/v2/traffic/city?city={city}&ak=你的 ak"
    response = requests.get(url)
    data = response.json()
    return data['data']['traffic']

city = "北京"
traffic_info = get_traffic_info(city)
print(traffic_info)
```

在这个代码中，我们首先导入了 `requests` 库，然后定义了一个 `get_traffic_info` 函数，该函数接收一个城市名称作为参数，并通过发送 HTTP 请求来获取该城市的交通信息。最后，我们调用了 `get_traffic_info` 函数，并将获取到的交通信息打印出来。

## 4.2 交通信息可视化
我们还可以使用 Matplotlib 库来可视化交通信息。首先，我们需要安装 Matplotlib 库，然后我们可以使用这个库来绘制交通信息的图表。

```python

plot_traffic_info(traffic_info)

在这个代码中，我们首先导入了 Matplotlib 库，然后定义了一个 `plot_traffic_info` 函数，该函数接收一个交通信息矩阵作为参数，并使用 Matplotlib 库来绘制该矩阵的图表。最后，我们调用了 `plot_traffic_info` 函数，并将获取到的交通信息的图表显示出来。
