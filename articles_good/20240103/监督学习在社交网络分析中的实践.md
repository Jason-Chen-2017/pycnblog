### 3.3 数学模型公式详细讲解

在本节中，我们将详细介绍监督学习中的线性回归和逻辑回归的数学模型公式。

#### 3.3.1 线性回归

线性回归是一种简单的监督学习算法，它假设输入特征和输出标签之间存在线性关系。线性回归的目标是找到最佳的线性模型，使得预测值与实际值之间的差异最小。

线性回归的数学模型公式为：

$$
y = \theta_0 + \theta_1x_1 + \theta_2x_2 + \cdots + \theta_nx_n + \epsilon
$$

其中，$y$ 是预测值，$x_1, x_2, \cdots, x_n$ 是输入特征，$\theta_0, \theta_1, \theta_2, \cdots, \theta_n$ 是模型参数，$\epsilon$ 是差异。

线性回归的目标是最小化均方误差（MSE）：

$$
MSE = \frac{1}{m} \sum_{i=1}^{m} (y_i - \hat{y}_i)^2
$$

其中，$m$ 是训练数据集的大小，$y_i$ 是实际值，$\hat{y}_i$ 是预测值。

通过梯度下降算法，我们可以求得线性回归的最佳模型参数：

$$
\theta_j = \theta_j - \alpha \frac{\partial}{\partial \theta_j} MSE
$$

其中，$\alpha$ 是学习率。

#### 3.3.2 逻辑回归

逻辑回归是一种用于二分类问题的监督学习算法，它假设输入特征和输出标签之间存在逻辑关系。逻辑回归的目标是找到最佳的逻辑模型，使得预测概率最接近实际概率。

逻辑回归的数学模型公式为：

$$
P(y=1|x) = \frac{1}{1 + e^{-(\theta_0 + \theta_1x_1 + \theta_2x_2 + \cdots + \theta_nx_n)}}
$$

其中，$P(y=1|x)$ 是预测概率，$x_1, x_2, \cdots, x_n$ 是输入特征，$\theta_0, \theta_1, \theta_2, \cdots, \theta_n$ 是模型参数。

逻辑回归的目标是最大化对数似然函数：

$$
L = \sum_{i=1}^{m} [y_i \log(\hat{y}_i) + (1 - y_i) \log(1 - \hat{y}_i)]
$$

其中，$m$ 是训练数据集的大小，$y_i$ 是实际值，$\hat{y}_i$ 是预测概率。

通过梯度上升算法，我们可以求得逻辑回归的最佳模型参数：

$$
\theta_j = \theta_j - \alpha \frac{\partial}{\partial \theta_j} L
$$

其中，$\alpha$ 是学习率。

### 4.1 线性回归代码实例

我们将通过一个简单的线性回归代码实例来演示监督学习在社交网络分析中的应用。在这个例子中，我们将预测用户的关注度，根据用户的历史关注行为。

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机数据
np.random.seed(0)
X = np.random.randn(100, 1)
y = 2 * X + 1 + np.random.randn(100, 1) * 0.5

# 初始化模型参数
theta = np.zeros(1)
alpha = 0.01
iterations = 1000

# 梯度下降算法
for i in range(iterations):
    gradients = (1 / X.shape[0]) * X.transpose() * (y - X.dot(theta))
    theta = theta - alpha * gradients

# 预测
X_new = np.array([[2]])
y_pred = X_new.dot(theta)

# 绘制
plt.scatter(X, y)
plt.plot(X, X.dot(theta), color='r')
plt.xlabel('X')
plt.ylabel('y')
plt.show()
```

在这个代码实例中，我们首先生成了随机数据，其中$X$ 表示输入特征，$y$ 表示输出标签。然后我们初始化了模型参数$\theta$ 和学习率$\alpha$，并使用梯度下降算法进行模型训练。最后，我们使用训练好的模型进行预测，并绘制了结果。

plt.xlabel('X')
plt.ylabel('y')
plt.show()
