假设我们有一个函数 $f(x) = x^2 - 4x + 4$，我们要求解这个函数在区间 $[0, 4]$ 内的最小值、最大值和零点。

```python
import numpy as np

def f(x):
    return x**2 - 4*x + 4

def bisection(a, b, tol):
    x1 = (a + b) / 2
    if abs(f(x1)) < tol:
        return x1
    elif f(x1) * f(b) < 0:
        return bisection(a, x1, tol)
    else:
        return bisection(x1, b, tol)

def secant(x0, x1, tol):
    while abs(x1 - x0) > tol:
        x2 = f(x1) * (x1 - x0) / (f(x1) - f(x0))
        x0 = x1
        x1 = x2
    return x1

a, b = 0, 4
tol = 1e-6

x_min = bisection(a, b, tol)
x_max = bisection(a, b, tol)
x_zero = secant(0, 0, tol)

print("最大值：", x_max)
print("零点：", x_zero)
```

在以上公式中，$y_{i+1}$ 表示下一步的近似解，$y_i$ 表示当前步的近似解，$h$ 表示步长，$f(x_i, y_i)$ 表示函数值，$c_i$ 表示系数，$\phi_i(x)$ 表示基函数，$x$ 表示数值，$a$ 表示精度要求，$m$ 表示有限域。
