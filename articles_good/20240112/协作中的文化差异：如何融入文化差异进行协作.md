## 4.1 自然语言处理示例

以下是一个自然语言处理示例：


```
['我', '爱', '中', '国']
```

## 4.2 机器学习示例

以下是一个机器学习示例：

```python
from sklearn.linear_model import LogisticRegression

X = [[1, 2], [2, 3], [3, 4]]
y = [0, 1, 1]

model = LogisticRegression()
model.fit(X, y)

print(model.predict([[2, 2]]))
```

这个示例使用了sklearn库进行逻辑回归。输出结果为：

```
[1]
```

## 4.3 数据挖掘示例

以下是一个数据挖掘示例：

```python
from sklearn.cluster import KMeans

X = [[1, 2], [2, 3], [3, 4]]
model = KMeans(n_clusters=2)
model.fit(X)

print(model.labels_)
```

这个示例使用了sklearn库进行聚类。输出结果为：

```
[0, 1, 1]
```
