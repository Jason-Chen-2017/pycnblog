## 4.1 计算文本文件的熵

### 4.1.1 代码实现

```python
import os
import math

def calculate_entropy(file_path):
    with open(file_path, 'r', encoding='utf-8') as f:
        text = f.read()
        char_count = {}
        for char in text:
            if char in char_count:
                char_count[char] += 1
            else:
                char_count[char] = 1

        total_char = len(char_count)
        entropy = 0
        for count in char_count.values():
            probability = count / total_char
            entropy -= probability * math.log2(probability)

        return entropy

file_path = 'example.txt'
entropy = calculate_entropy(file_path)
print(f'文本文件的熵：{entropy}')
```

### 4.1.2 解释说明

这个代码实例中，我们首先定义了一个`calculate_entropy`函数，该函数接受一个文件路径作为输入，并计算该文件中字符的熵。具体实现如下：

1. 打开文件并读取文本内容。
2. 统计文本中每个字符的出现次数。
3. 计算文本中每个字符的概率。
4. 使用熵定义公式计算文本文件的熵。

最后，我们调用`calculate_entropy`函数计算指定文件的熵，并输出结果。
