```python
def get_location(vehicle_id):
    url = f"http://api.example.com/vehicle/{vehicle_id}/location"
    response = requests.get(url)
    if response.status_code == 200:
        return response.json()
    else:
        return None

def get_delivery_status(vehicle_id):
    url = f"http://api.example.com/vehicle/{vehicle_id}/delivery_status"
    response = requests.get(url)
    if response.status_code == 200:
        return response.json()
    else:
        return None

def main():
    vehicle_id = "12345"
    location = get_location(vehicle_id)
    if location:
        print(f"Vehicle {vehicle_id} location: {location}")
    else:
        print(f"Failed to get location for Vehicle {vehicle_id}")

    delivery_status = get_delivery_status(vehicle_id)
    if delivery_status:
        print(f"Vehicle {vehicle_id} delivery status: {delivery_status}")
    else:
        print(f"Failed to get delivery status for Vehicle {vehicle_id}")

if __name__ == "__main__":
    main()
```

### 7.5 如何评估物流实时监控系统的效果？

物流实时监控系统的效果可以通过以下几个指标进行评估：

1. 客户满意度：通过收集客户反馈，评估物流实时监控系统对客户满意度的提高。
2. 物流成本：通过比较物流实时监控系统前后的成本，评估系统对物流成本的优化效果。
3. 物流效率：通过比较物流实时监控系统前后的效率指标，如物流通量、运输时间等，评估系统对物流效率的提高。
4. 物流风险管理：通过分析物流实时监控系统对物流风险管理的效果，评估系统对物流风险降低的能力。
