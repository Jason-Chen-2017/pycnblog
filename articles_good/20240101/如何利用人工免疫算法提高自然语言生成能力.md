## 4.1代码实例

```python
import numpy as np

# 初始化抗原和抗体集合
antigens = ['sunny', 'cloudy', 'rainy', 'snowy']
antibodies = ['sunny day', 'cloudy day', 'rainy day', 'snowy day']

# 计算抗原与抗体的相似度
def similarity(antigen, antibody):
    return sum([1 for a, b in zip(antigen, antibody) if a == b]) / len(antigen)

# 选择具有较高相似度的抗体进行激活
activated_antibodies = [antibody for antibody in antibodies if similarity(antigen, antibody) > 0.5]

# 通过激活的抗体生成新的抗体
def generate_new_antibodies(activated_antibodies):
    new_antibodies = []
    for antibody in activated_antibodies:
        for word in ['sunny', 'cloudy', 'rainy', 'snowy']:
            new_antibody = antibody.replace(word, 'partly ' + word)
            new_antibodies.append(new_antibody)
    return new_antibodies

# 重复步骤2-4，直到满足终止条件
for _ in range(10):
    activated_antibodies = [antibody for antibody in antibodies if similarity(antigen, antibody) > 0.5]
    antibodies = generate_new_antibodies(activated_antibodies)

# 输出结果
for antibody in antibodies:
    print(antibody)
```

在这个代码实例中，我们首先初始化了抗原和抗体集合。抗原集合表示天气状况，包括'sunny', 'cloudy', 'rainy', 'snowy'。抗体集合表示描述天气的文本，包括'sunny day', 'cloudy day', 'rainy day', 'snowy day'。

接下来，我们定义了一个`similarity`函数，用于计算抗原与抗体的相似度。在这个例子中，我们使用了简单的字符串匹配来计算相似度。

然后，我们选择具有较高相似度的抗体进行激活。激活的抗体表示搜索过程中的有效解决方案。

接下来，我们定义了一个`generate_new_antibodies`函数，用于通过激活的抗体生成新的抗体。在这个例子中，我们将原始抗体中的单词替换为其他形式，以生成新的抗体。

最后，我们重复步骤2-4，直到满足终止条件。在这个例子中，我们设定了10次迭代作为终止条件。

最后，我们输出结果，即生成的描述天气的文本。
