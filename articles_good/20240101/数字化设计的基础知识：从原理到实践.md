## 3.4 数学模型公式详细讲解

### 3.4.1 数字滤波数学模型公式详细讲解

数字滤波的数学模型主要包括：

1. 零阶滤波：零阶滤波的数学模型公式为：
$$
y[n] = K \sum_{k=0}^{N-1} x[n-k]
$$
其中，$x[n]$ 是输入信号，$y[n]$ 是输出信号，$K$ 是系数，$N$ 是窗口长度。

2. 一阶滤波：一阶滤波的数学模型公式为：
$$
y[n] = K \sum_{k=0}^{N-1} x[n-k] h[k]
$$
其中，$x[n]$ 是输入信号，$y[n]$ 是输出信号，$K$ 是系数，$h[k]$ 是一阶滤波器的系数，$N$ 是窗口长度。

3. 二阶滤波：二阶滤波的数学模型公式为：
$$
y[n] = K \sum_{k=0}^{N-1} x[n-k] (h_1[k] + h_2[k] n)
$$
其中，$x[n]$ 是输入信号，$y[n]$ 是输出信号，$K$ 是系数，$h_1[k]$ 和 $h_2[k]$ 是二阶滤波器的系数，$N$ 是窗口长度。

### 3.4.2 数字调制数学模型公式详细讲解

数字调制的数学模型主要包括：

1. 霍尔调制：霍尔调制的数学模型公式为：
$$
s(t) = \sum_{n=0}^{\infty} A_n p(t - nT) \cos(\omega_c t + \phi_n)
$$
其中，$s(t)$ 是调制信号，$A_n$ 是调制信号的幅值，$p(t)$ 是信号的载波，$\omega_c$ 是载波角频率，$\phi_n$ 是调制信号的相位，$T$ 是时间间隔。

2. 频率调制：频率调制的数学模型公式为：
$$
s(t) = \cos(\omega_c t + \sum_{n=0}^{\infty} A_n \cos(\omega_n t))
$$
其中，$s(t)$ 是调制信号，$\omega_c$ 是载波角频率，$\omega_n$ 是调制信号的频率，$A_n$ 是调制信号的幅值。

3. 位调制：位调制的数学模型公式为：
$$
s(t) = \sum_{n=0}^{\infty} A_n p(t - nT) \delta(t - nT)

其中，$s(t)$ 是调制信号，$A_n$ 是调制信号的幅值，$p(t)$ 是信号的载波，$\delta(t)$ 是Diracδ函数，$T$ 是时间间隔。

import matplotlib.pyplot as plt

# 零阶滤波
def zero_phase_filter(x, K, N):
    y = np.zeros(len(x))
    for n in range(len(x)):
        for k in range(N):
            y[n] += K * x[n-k]
    return y

# 一阶滤波
def first_filter(x, K, h):
    y = np.zeros(len(x))
    for n in range(len(x)):
        for k in range(N):
            y[n] += K * x[n-k] * h[k]
    return y

# 二阶滤波
def second_filter(x, K, h1, h2):
    y = np.zeros(len(x))
    for n in range(len(x)):
        for k in range(N):
            y[n] += K * x[n-k] * (h1[k] + h2[k] * n)
    return y

# 测试数据

import numpy as np
import matplotlib.pyplot as plt

# 霍尔调制
def amplitude_modulation(x, A, T, fc, phi):
    y = np.zeros(len(x))
    for n in range(len(x)):
        for k in range(int(T)):
            y[n] += A * np.cos(2 * np.pi * fc * n * T + phi)
    return y

# 频率调制
def frequency_modulation(x, A, T, fc, fm):
    y = np.zeros(len(x))
    for n in range(len(x)):
        for k in range(int(T)):
            y[n] += A * np.cos(2 * np.pi * (fc + fm * x[n]) * n * T)
    return y

# 位调制
def pulse_amplitude_modulation(x, A, T, p):
    y = np.zeros(len(x))
    for n in range(len(x)):
        if p[n] == 1:
            y[n] = A
    return y
