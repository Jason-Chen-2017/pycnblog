### 4.1 基于内容的推荐实例

```python
# 收集用户的兴趣和需求信息

# 收集物品的内容信息
item_content = {'music': {'pop': 10, 'rock': 8, 'jazz': 6},
                'movie': {'action': 10, 'comedy': 8, 'horror': 6},
                'book': {'fiction': 10, 'science': 8, 'history': 6}}

# 计算用户与物品之间的相似度
similarity = {}
for item in item_content.keys():
    similarity[item] = sum([user_interest[key] * item_content[item][key] for key in user_interest.keys()])

# 根据相似度排序，为用户推荐与其相关的内容
recommended_items = sorted(similarity.items(), key=lambda x: x[1], reverse=True)
```
