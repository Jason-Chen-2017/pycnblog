### 4.3 交通规则识别

代码实例：

```python
import re

# 文本数据
text = '速度限制为60公里/小时，禁止行驶区域内行驶。'

# 正则表达式匹配
pattern = re.compile(r'速度限制为(\d+)公里/小时')
match = pattern.search(text)

# 提取结果
if match:
    speed_limit = match.group(1)
    print(f'速度限制为{speed_limit}公里/小时')
else:
    print('未找到速度限制')
```

详细解释说明：

1. 使用正则表达式库匹配文本数据。
2. 使用正则表达式库提取结果。
