### 4.1 知识图谱构建实例

在这个实例中，我们将构建一个简单的知识图谱，用于表示一些人和他们的关系。

```python
from rdflib import Graph, Namespace, Literal, URIRef
from rdflib.namespace import RDF, RDFS

# 创建一个知识图谱
g = Graph()

# 创建一个命名空间
ns = Namespace("http://example.org/")

# 创建实体
alice = URIRef(ns["Alice"])
bob = URIRef(ns["Bob"])

# 创建属性
age = URIRef(ns["age"])

# 创建关系
g.add((alice, RDF.type, RDFS.Person))
g.add((bob, RDF.type, RDFS.Person))
g.add((alice, age, Literal(25)))
g.add((bob, age, Literal(30)))
g.add((alice, RDF.name, "Alice"))
g.add((bob, RDF.name, "Bob"))
g.add((alice, RDF.knows, bob))
g.add((bob, RDF.knows, alice))
```

### 4.2 个性化学习实例

在这个实例中，我们将构建一个简单的个性化学习系统，用于推荐适合自己的学习资源。

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 学习资源
resources = ["Python编程", "机器学习", "深度学习", "自然语言处理"]

# 学生需求
student_need = "机器学习"

# 构建TF-IDF向量器
vectorizer = TfidfVectorizer()

# 构建资源向量
resource_vector = vectorizer.fit_transform(resources)

# 构建学生需求向量
student_vector = vectorizer.transform([student_need])

# 计算相似度
similarity = cosine_similarity(student_vector, resource_vector)

# 推荐资源
recommended_resource = resources[similarity.argmax()]
```
