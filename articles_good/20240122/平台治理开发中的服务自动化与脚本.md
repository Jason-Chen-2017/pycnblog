## 4. 具体最佳实践：代码实例和详细解释说明

在平台治理开发中，服务自动化和脚本技术的具体最佳实践包括：

### 4.1 代码实例

以下是一个使用Python编写的自动化脚本示例：

```python
import os
import sys

def start_service(service_name):
    command = f"service {service_name} start"
    os.system(command)

def stop_service(service_name):
    command = f"service {service_name} stop"
    os.system(command)

def restart_service(service_name):
    command = f"service {service_name} restart"
    os.system(command)

if __name__ == "__main__":
    service_name = sys.argv[1]
    action = sys.argv[2]

    if action == "start":
        start_service(service_name)
    elif action == "stop":
        stop_service(service_name)
    elif action == "restart":
        restart_service(service_name)
    else:
        print("Invalid action")
```

### 4.2 详细解释说明

上述代码实例是一个使用Python编写的自动化脚本，用于实现对服务的自动启动、停止和重启操作。脚本接收两个命令行参数：服务名称和操作类型。根据操作类型，脚本调用相应的函数来实现对服务的自动操作。
