代码实例：

```python
from pyspark import SparkContext
from pyspark.sql import SQLContext

# 初始化 Spark 环境
sc = SparkContext()
sqlContext = SQLContext(sc)

# 读取数据
data = sqlContext.read.json("data.json")

# 数据清洗
data_clean = data.dropDuplicates().fillna()

# 数据存储
data_lake = data_clean

# 数据处理
data_processed = data_lake.groupBy("transaction_type").agg({"amount": "sum"})

# 数据可视化
data_processed.show()

# 数据应用
data_processed.write.json("data_processed.json")
```

1. 首先，我们使用 Spark 环境来处理大量数据。

3. 接着，我们将清洗后的数据存储到数据湖中。
4. 然后，我们对数据湖中的数据进行处理，并将处理后的数据可视化。
5. 最后，我们将处理后的数据应用于金融支付系统的业务和决策。
