### 4.1 实体识别

实体识别是自然语言处理系统识别文本中实体的过程。以下是一个基于规则的实体识别示例：

```python
import re

def entity_recognition(text):
    # 定义一系列实体的正则表达式
    date_pattern = r'\d{4}-\d{2}-\d{2}'
    number_pattern = r'\d+'
    # 使用正则表达式匹配实体
    date_entities = re.findall(date_pattern, text)
    number_entities = re.findall(number_pattern, text)
    # 返回实体列表
    return date_entities + number_entities

text = "2021-03-15 1000"
print(entity_recognition(text))
```

### 4.2 关系抽取

关系抽取是自然语言处理系统抽取实体之间关系的过程。以下是一个基于规则的关系抽取示例：

```python
import re

def relation_extraction(text):
    # 定义一系列关系的正则表达式
    location_pattern = r'位于|在'
    # 使用正则表达式匹配关系
    location_relations = re.findall(location_pattern, text)
    # 返回关系列表
    return location_relations

text = "这家餐厅位于中心区"
print(relation_extraction(text))
```

### 4.3 问题答案推理

问题答案推理是自然语言处理系统通过知识图谱推理问题答案的过程。以下是一个基于规则的问题答案推理示例：

```python
def question_answering(knowledge_graph, question):
    # 使用知识图谱查询问题答案
    answer = knowledge_graph.query(question)
    # 返回答案
    return answer

# 假设knowledge_graph是一个知识图谱对象
question = "这家餐厅位于哪里？"
answer = question_answering(knowledge_graph, question)
print(answer)
```
