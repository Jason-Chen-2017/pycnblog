## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 创建ROS节点

首先，我们需要创建一个ROS节点，以便实现机器人的功能。以下是一个简单的ROS节点创建示例：

```python
#!/usr/bin/env python

import rospy

def main():
    rospy.init_node('my_node', anonymous=True)
    rate = rospy.Rate(1) # 1hz
    while not rospy.is_shutdown():
        rospy.loginfo('My Node is running!')
        rate.sleep()

if __name__ == '__main__':
    try:
        main()
    except rospy.ROSInterruptException:
        pass
```

### 4.2 创建ROS发布者

接下来，我们需要创建一个ROS发布者，以便发布消息到主题。以下是一个简单的ROS发布者创建示例：

```python
#!/usr/bin/env python

import rospy
from std_msgs.msg import Int32

def main():
    rospy.init_node('my_publisher', anonymous=True)
    pub = rospy.Publisher('chatter', Int32, queue_size=10)
    rate = rospy.Rate(1) # 1hz
    while not rospy.is_shutdown():
        msg = Int32()
        msg.data = 10
        pub.publish(msg)
        rate.sleep()

if __name__ == '__main__':
    try:
        main()
    except rospy.ROSInterruptException:
        pass
```

### 4.3 创建ROS订阅者

最后，我们需要创建一个ROS订阅者，以便订阅消息从主题。以下是一个简单的ROS订阅者创建示例：

```python
#!/usr/bin/env python

import rospy
from std_msgs.msg import Int32

def callback(data):
    rospy.loginfo(rospy.get_caller_id() + ' I heard %d', data.data)

def main():
    rospy.init_node('my_subscriber', anonymous=True)
    sub = rospy.Subscriber('chatter', Int32, callback)
    rate = rospy.Rate(1) # 1hz
    while not rospy.is_shutdown():
        rate.sleep()

    try:
        main()
    except rospy.ROSInterruptException:
        pass
```
