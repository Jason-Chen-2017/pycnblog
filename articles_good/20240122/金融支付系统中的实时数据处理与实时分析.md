### 4.1 实时数据处理

实时数据处理的代码实例如下：

```python
import queue

def process_data(data):
    # 对接收到的数据进行处理
    return data * 2

def main():
    # 创建一个队列
    q = queue.Queue()

    # 添加数据到队列
    q.put(1)
    q.put(2)
    q.put(3)

    # 处理队列中的数据
    while not q.empty():
        data = q.get()
        processed_data = process_data(data)
        print(f"Processed data: {processed_data}")

if __name__ == "__main__":
    main()
```
