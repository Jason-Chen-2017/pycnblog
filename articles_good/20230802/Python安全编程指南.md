         # 2.概念术语介绍
         ## 2.1 Python简介
         Python（英国发音为/ˈpaɪθən/）是一个高级语言，拥有比其他语言更简洁的代码结构，支持多种编程范式，能够进行自动内存管理，并且具有强大的类库和丰富的第三方模块生态系统。Python通常被称为脚本语言或可脚本语言，因为它可以轻松地调用命令行工具和系统API。
         
         ## 2.2 数据类型
         ### 2.2.1 数字类型
         Python中的数字类型包括：整数(int)、长整数(long int)、浮点数(float)，布尔值(bool)。其中，整数和长整数都表示整数；浮点数表示实数或者复数；布尔值只有True和False两个取值。
         
         ### 2.2.2 字符串类型
         在Python中，字符串用单引号(')或双引号(")括起来。
         + 使用三重双引号('''...''')或三重单引号("""...""")来创建多行字符串。
         + 使用r前缀创建无转义字符的原始字符串。
         + 使用反斜杠(\)转义特殊字符。
         + 如果字符串内部含有单引号，则可以在其外围加上双引号。反之亦然。
         
         ### 2.2.3 列表类型
         Python中的列表(list)是一个有序集合，元素可以是任意类型。列表是一种有序集合数据类型，它的元素的数量不固定，可以动态增删。列表可以使用方括号([])来表示。
         
         ### 2.2.4 元组类型
         Python中的元组(tuple)是一个不可变的有序集合，元素可以是任意类型。元组是一种有序集合数据类型，它的元素的数量也不固定，但是一旦定义就不能修改。元组可以使用圆括号(())来表示。
         
         ### 2.2.5 字典类型
         Python中的字典(dict)是一个键-值对(key-value)存储结构。字典是一种存储映射关系的数据结构，它使用键来访问对应的值。字典是一种无序集合，字典可以通过键来访问对应的值，键不能重复。字典使用花括号({})表示。
         
         ### 2.2.6 集合类型
         Python中的集合(set)是一个无序且元素唯一的集合。集合是一种无序集合数据类型，它只存储一组不重复元素。集合使用花括号({})表示。
         
         ## 2.3 操作符与表达式
         ### 2.3.1 算术运算符
         Python支持以下四种算术运算符：+ (加法)、- (减法)、* (乘法)、/ (除法)。
         
         ### 2.3.2 比较运算符
         Python支持以下六种比较运算符：< (小于)、> (大于)、<= (小于等于)、>= (大于等于)、== (等于)、!= (不等于)。
         
         ### 2.3.3 赋值运算符
         Python支持以下八种赋值运算符：= (简单的赋值)、+= (加法赋值)、-= (减法赋值)、*= (乘法赋值)、/= (除法赋值)、%= (取余赋值)、&= (按位与赋值)、|= (按位或赋值)。
         
         ### 2.3.4 逻辑运算符
         Python支持以下三种逻辑运算符：and (逻辑与)、or (逻辑或)、not (逻辑非)。
         
         ### 2.3.5 成员运算符
         Python提供成员运算符用于判断元素是否属于某一集合。
         
        |运算符|描述|
        |:-----:|:----------|
        |in|如果左侧对象存在于右侧序列中，返回True，否则返回False。|
        |not in|如果左侧对象不存在于右侧序列中，返回True，否则返回False.|
         
         ### 2.3.6 身份运算符
         Python提供以下两种身份运算符。
         
        |运算符|描述|
        |:--------|:------|
        is|如果两边对象相同，则返回True，否则返回False|
        is not|如果两边对象不同，则返回True，否则返回False|
         
         ### 2.3.7 位运算符
         Python提供了以下位运算符。
         
        |运算符|描述|
        |:------|:------|
        & (按位与)|参与运算的两个值,如果两个相应位都为1,则该位的结果为1,否则为0|
        \| (按位或)|只要对应的二个二进位有一个为1时，结果位就为1|
        ^ (异或)|当两对应的二进位相异时，结果为1|
        ~ (按位取反)|对数据的每个二进制位取反,即把1变为0,把0变为1|
        << (左移动)|左移运算是把"<<"左边的运算数的各二进位全部左移若干位，由"<<"右边的数字指定了移动的位数，高位丢弃，低位补0。|
        >> (右移动)|右移运算是把">>"左边的运算数的各二进位全部右移若干位，">>"右边的数字指定了移动的位数。|
         
         ### 2.3.8 运算符优先级
         从最高到最低的优先级依次为：
         1. **
         2. ~ + -
         3. * / % //
         4. + -
         5. >> <<
         6. < > <= >= ==!=
         7. &
         8. ^
         9. \|
         
         ## 2.4 控制流语句
         ### 2.4.1 if-else语句
         ```python
         x = input()   # 获取用户输入的整数
         if x > 0:
             print("x is positive")
         elif x < 0:
             print("x is negative")
         else:
             print("x is zero")
         ```
         
         ### 2.4.2 for循环
         ```python
         lst = [1, 2, 3, 4]   # 创建列表
         for num in lst:    # 对列表进行遍历
             print(num)     # 输出当前元素
         ```
         
         ### 2.4.3 while循环
         ```python
         i = 0               # 初始化变量i
         while i < 5:        # 当i小于5时循环
             print(i)       # 输出i
             i += 1         # 将i增加1
         ```
         
         ### 2.4.4 函数定义及调用
         ```python
         def my_function():
             return "Hello world!"
         
         result = my_function()   # 执行函数，获取结果
         print(result)            # 输出结果
         ```
         
         ### 2.4.5 try-except语句
         ```python
         try:
            value = 10 / 0      # 尝试执行除零错误
        except ZeroDivisionError:
            print("You can't divide by zero!")   # 输出异常信息
         ```

         ## 2.5 模块导入与包管理
         ### 2.5.1 模块导入
         通过import语句可以导入模块。引入模块后可以使用模块名直接访问模块内的函数或类。
         ```python
         import math   # 引入math模块
         pi = math.pi   # 使用math模块计算圆周率
         ```
         
         ### 2.5.2 自定义模块
         可以在文件中编写自己的模块，然后通过import导入。自定义模块的名称应该尽量有意义，且符合标识符的命名规范。模块一般以.py为扩展名保存。模块包含以下三个部分：
         1. 模块文档注释(optional): 模块的相关信息，如作者、版本、日期等。
         2. 模块全局变量声明(optional): 模块使用的全局变量。
         3. 模块功能实现(mandatory): 模块实现的功能，包括函数、类等。
         
         ```python
         # filename: mymodule.py
         PI = 3.14159   # 模块全局变量声明
         
         def circumference(radius):   # 模块功能实现
             """Calculate the circumference of a circle"""   # 函数文档注释
             return 2 * PI * radius
         ```

         ## 2.6 文件操作
         Python提供了file对象用于处理文件。
         
         ### 2.6.1 文件打开
         ```python
         file = open('/path/to/file','mode')
         ```
         mode参数可以是'r'代表读模式、'w'代表写模式、'a'代表追加模式。另外，还可以指定编码方式，例如'utf-8':
         ```python
         file = open('/path/to/file', encoding='utf-8')
         ```
         
         ### 2.6.2 文件读写
         #### 读取所有内容
         read()方法可以一次性读取文件的全部内容。
         ```python
         data = file.read()
         ```
         
         #### 逐行读取
         readline()方法可以读取文件的一行内容，并返回。
         ```python
         line = file.readline()
         ```
         
         #### 指定行号读取
         seek()方法可以调整文件指针位置到指定的行号，从第0行开始计数。
         ```python
         file.seek(offset, from_what)
         ```
         offset参数表示偏移的字节数，from_what参数表示参考的起始位置。设定为0表示从头开始，1表示从当前位置开始，2表示从文件末尾开始。
         ```python
         file.seek(0)                  # 设置文件指针到开头
         current_position = file.tell()  # 获取当前位置
         ```
         tell()方法可以查看当前文件指针位置。
         
         #### 写入文件
         write()方法可以向文件写入内容。
         ```python
         file.write('some text')
         ```
         
         #### 删除文件
         unlink()方法可以删除文件。
         ```python
         os.unlink('/path/to/file')
         ```
         
         ## 2.7 其它特性
         ### 2.7.1 默认参数
         函数的参数可以设置默认值，这样可以简化调用函数时的参数传递过程。
         ```python
         def greeting(name="world"):
             return "Hello, {}!".format(name)
         
         print(greeting())     # Hello, world!
         print(greeting("Alice"))  # Hello, Alice!
         ```
         
         ### 2.7.2 可迭代对象与生成器
         Python中的列表、字符串、元组、集合都是可迭代对象(iterable)，可以通过for语句或其他迭代机制进行迭代。同时，也可以通过生成器(generator)生成一个新的可迭代对象。
         ```python
         def squares(n):
             for i in range(n):
                 yield i**2
         
         for num in squares(5):
             print(num)    # output: 0 1 4 9 16
         
         nums = tuple(squares(5))   # 生成器转换为元组
         assert isinstance(nums, tuple) and nums == (0, 1, 4, 9, 16)
         ```
         
         ### 2.7.3 属性访问与方法调用
         对象的方法调用遵循先查找属性，再查找方法的顺序。对象的属性访问遵循链式查询，直到找到目标属性或方法。
         ```python
         obj = {'name': 'Alice'}
         name = getattr(obj, 'name')   # 查找属性
         say_hello = getattr(obj,'say_hello', lambda: "Hello!")   # 查找方法，没有则创建新方法并返回
         hello_str = say_hello()     # 调用方法
         ```
