## 3.1 前向传播算法原理

前向传播算法是神经网络的基本算法，它用于计算神经网络的输出结果。前向传播算法的原理是通过输入层接收输入数据，然后将数据传递给隐藏层进行计算，最后将计算结果传递给输出层输出。

前向传播算法的具体操作步骤如下：

1. 初始化神经网络的权重和偏置。
2. 输入层接收输入数据。
3. 隐藏层进行计算，将计算结果传递给输出层。
4. 输出层输出计算结果。

前向传播算法的数学模型公式如下：

$$
y = f(\sum_{i=1}^{n} w_i x_i + b)
$$

其中，$y$ 是输出结果，$f$ 是激活函数，$w_i$ 是权重，$x_i$ 是输入数据，$b$ 是偏置，$n$ 是输入数据的数量。

## 3.2 后向传播算法原理

后向传播算法是神经网络的基本算法，它用于调整神经网络的权重和偏置。后向传播算法的原理是通过输出层接收计算结果，然后将结果传递给输出层进行反馈，最后将反馈结果传递给输入层进行调整。

后向传播算法的具体操作步骤如下：

1. 初始化神经网络的权重和偏置。
2. 输入层接收输入数据。
3. 隐藏层进行计算，将计算结果传递给输出层。
4. 输出层接收计算结果，将结果传递给输入层进行反馈。
5. 输入层调整权重和偏置，以便最小化损失函数。

后向传播算法的数学模型公式如下：

$$
\Delta w = \alpha \delta_j x_i
$$

$$
\Delta b = \alpha \delta_j
$$

其中，$\Delta w$ 是权重的梯度，$\Delta b$ 是偏置的梯度，$\alpha$ 是学习率，$\delta_j$ 是隐藏层的反馈结果，$x_i$ 是输入数据。

## 3.3 梯度下降算法原理

梯度下降算法是神经网络的基本算法，它用于调整神经网络的权重和偏置。梯度下降算法的原理是通过计算损失函数的梯度，然后将梯度与学习率相乘，以便调整权重和偏置。

梯度下降算法的具体操作步骤如下：

1. 初始化神经网络的权重和偏置。
2. 计算神经网络的输出结果。
3. 计算损失函数的梯度。
4. 调整神经网络的权重和偏置。
5. 重复步骤2-4，直到损失函数达到最小值。

梯度下降算法的数学模型公式如下：

$$
w_{new} = w_{old} - \alpha \nabla L(w)
$$

其中，$w_{new}$ 是新的权重，$w_{old}$ 是旧的权重，$\alpha$ 是学习率，$\nabla L(w)$ 是损失函数的梯度。

db = np.sum(dy, axis=0)

b += 0.1 * db
