## 1.背景介绍

区块链技术的发展历程可以分为以下几个阶段：

1. 2008年，Satoshi Nakamoto发表了一篇论文《Bitcoin: A Peer-to-Peer Electronic Cash System》，提出了一种基于区块链技术的数字货币系统。
2. 2009年，Satoshi Nakamoto实现了第一个基于区块链技术的数字货币系统——Bitcoin。
3. 2011年，Laszlo Hanyecz使用了第一个基于区块链技术的数字货币进行购物的交易。
4. 2013年，Ethereum项目被提出，它是第一个基于区块链技术的开源平台，允许开发者创建和部署智能合约。
5. 2014年，Ethereum项目成功完成了开发和测试，并开始进行公开测试。
6. 2015年，Ethereum项目正式发布，并开始进行商业化应用。

从以上历史可以看出，区块链技术的发展非常迅猛，尤其是智能合约这一技术，它为区块链技术的应用提供了更多的可能性。

### 4.1 智能合约的代码实例

以下是一个简单的智能合约的代码实例：

```solidity
pragma solidity ^0.5.0;

contract SimpleContract {
    address public owner;
    uint256 public balance;

    constructor() public {
        owner = msg.sender;
        balance = 1000000;
    }

    function transfer(address payable recipient, uint256 amount) public payable {
        require(msg.value == amount, "Amount must match");
        recipient.transfer(amount);
        balance -= amount;
    }
}
```
