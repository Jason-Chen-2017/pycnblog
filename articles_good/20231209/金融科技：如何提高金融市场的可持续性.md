## 4.3 区块链技术

### 4.3.1 去中心化

区块链技术是一种去中心化的、安全的、透明的、高效的交易平台，它不需要任何中心化的权威机构来管理和验证交易。以下是一个简单的去中心化的代码实例：

```python
import hashlib

# 创建区块
class Block:
    def __init__(self, index, previous_hash, timestamp, data, hash):
        self.index = index
        self.previous_hash = previous_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

    def calculate_hash(self):
        sha = hashlib.sha256()
        sha.update((str(self.index) + str(self.previous_hash) + str(self.timestamp) + str(self.data)).encode('utf-8'))
        return sha.hexdigest()

# 创建链
class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(0, "0", 0, "Genesis Block", self.calculate_hash("Genesis Block"))

    def add_block(self, index, previous_hash, timestamp, data):
        block = Block(index, previous_hash, timestamp, data, self.calculate_hash(str(index) + str(previous_hash) + str(timestamp) + str(data)))
        self.chain.append(block)

    def calculate_hash(self, data):
        sha = hashlib.sha256()
        sha.update((str(data)).encode('utf-8'))
        return sha.hexdigest()

# 创建链
blockchain = Blockchain()
blockchain.add_block(1, blockchain.chain[-1].hash, 1544832496, "First block")
blockchain.add_block(2, blockchain.chain[-1].hash, 1544832497, "Second block")
```
