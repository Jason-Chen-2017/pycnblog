## 3.1数据压缩算法

### 3.1.1lossless 压缩算法

lossless 压缩算法的核心思想是通过找出数据中的重复和冗余信息，并将其删除或压缩，从而减少数据的存储空间。常见的 lossless 压缩算法有 Huffman 编码、Lempel-Ziv 编码（LZ77、LZ78、LZW）等。

Huffman 编码是一种基于字符频率的编码方法，它将每个字符对应一个不同的二进制编码，常用字符对应的编码更短，从而减少了数据的存储空间。Huffman 编码的具体操作步骤如下：

1.统计数据中每个字符的频率。

2.根据字符频率构建一个字符频率树。

3.从字符频率树中选择两个最小频率的字符，将它们合并为一个新的节点，并将其频率设为两个字符的频率之和。

4.重复步骤3，直到字符频率树中只剩下一个根节点。

5.根据字符频率树生成 Huffman 编码表。

6.将数据中每个字符对应其对应的 Huffman 编码。

Lempel-Ziv 编码是一种基于字符序列的编码方法，它将数据中的重复和冗余信息压缩为一个或多个代表该信息的代码。Lempel-Ziv 编码的具体操作步骤如下：

1.将数据分为多个长度相同的字符序列。

2.对于每个字符序列，检查是否存在与之前出现过的字符序列相同的子序列。

3.如果存在，则将该子序列替换为一个代表该子序列的代码。

4.将数据中每个字符序列对应其对应的代码。

### 3.1.2loss 压缩算法

loss 压缩算法的核心思想是通过对数据进行一定程度的信息丢失，从而减少数据的存储空间。常见的 loss 压缩算法有 JPEG、MP3 等。

JPEG 是一种基于分频的图像压缩算法，它将图像分为多个频率区域，并对每个频率区域进行量化和编码。量化是将频率区域中的数值映射到一个有限的数值集合，从而减少数值的精度。编码是将量化后的数值序列转换为二进制编码。JPEG 的具体操作步骤如下：

1.将图像分为多个频率区域。

2.对每个频率区域进行量化。

3.对量化后的数值序列进行编码。

4.将编码后的数值序列存储为压缩后的图像文件。

MP3 是一种基于预测编码的音频压缩算法，它将音频信号分为多个频率区域，并对每个频率区域进行预测和编码。预测是根据音频信号的历史值预测其未来值，从而减少数值的精度。编码是将预测后的数值序列转换为二进制编码。MP3 的具体操作步骤如下：

1.将音频信号分为多个频率区域。

2.对每个频率区域进行预测。

3.对预测后的数值序列进行编码。

4.将编码后的数值序列存储为压缩后的音频文件。

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [5, 2, 8, 3, 5, 2, 8, 1, 9, 3]
arr = quick_sort(arr)
```

def huffman_encode(arr):
    frequency = {}
    for i in range(len(arr)):
        if arr[i] in frequency:
            frequency[arr[i]] += 1
        else:
            frequency[arr[i]] = 1

    huffman_tree = build_huffman_tree(frequency)
    huffman_code = build_huffman_code(huffman_tree)

    encoded_arr = []
    for i in range(len(arr)):
        encoded_arr.append(huffman_code[arr[i]])
    return encoded_arr

def build_huffman_tree(frequency):
    heap = []
    for key, value in frequency.items():
        heap.append((value, [key]))
    heapify(heap)

    while len(heap) > 1:
        left = heappop(heap)
        right = heappop(heap)
        for node in left[1:]:
            node[0] = (left[0] + right[0], left[1] + right[1], left[2] + right[2])
            heappush(heap, node)
        for node in right[1:]:
            node[0] = (left[0] + right[0], left[1] + right[1], left[2] + right[2])
            heappush(heap, node)

    return heap[0]

def build_huffman_code(huffman_tree):
    code = {}
    def dfs(node, code_so_far):
        if len(node[1]) == 1:
            code[node[1][0]] = code_so_far
        else:
            dfs(node[1][0], code_so_far + '0')
            dfs(node[1][1], code_so_far + '1')
    dfs(huffman_tree, '')
    return code

encoded_arr = huffman_encode(arr)
