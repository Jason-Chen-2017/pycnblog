### 4.3.5 模型评估

对训练好的模型进行评估，以确定模型的性能。可以使用Python的NLTK库来进行模型评估。

```python
import nltk

def evaluate_model(model, data):
    correct = 0
    total = 0
    for text in data:
        prediction = model.predict_output_word(text, topn=1)[0][0]
        if prediction == text:
            correct += 1
        total += 1
    accuracy = correct / total
    return accuracy

accuracy = evaluate_model(model, data)
```
