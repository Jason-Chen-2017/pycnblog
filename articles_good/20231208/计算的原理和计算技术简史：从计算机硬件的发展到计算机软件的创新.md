### 4.1 排序算法实例

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

arr = [64, 34, 25, 12, 22, 11, 90]
bubble_sort(arr)
print("排序后的数组为：", arr)
```

在这个代码实例中，我们使用了冒泡排序算法对一个数组进行排序。首先，我们定义了一个名为 `bubble_sort` 的函数，它接受一个数组作为参数。在函数内部，我们使用了两个循环来比较数组中的每个元素，并将较大的元素移动到数组的末尾。最后，我们使用 `print` 函数输出排序后的数组。

### 4.2 搜索算法实例

```python
def dfs(graph, start):
    visited = set()
    stack = [start]
    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(neighbors for neighbors in graph[vertex] if neighbors not in visited)
    return visited

graph = {
    'A': set(['B', 'C']),
    'B': set(['A', 'D', 'E']),
    'C': set(['A', 'F']),
    'D': set(['B']),
    'E': set(['B', 'F']),
    'F': set(['C', 'E'])
}
start = 'A'
print("深度优先搜索结果为：", dfs(graph, start))
```

在这个代码实例中，我们使用了深度优先搜索算法对一个图进行搜索。首先，我们定义了一个名为 `dfs` 的函数，它接受一个图和一个起始节点作为参数。在函数内部，我们使用了一个栈来存储当前节点，并使用一个集合来存储已访问的节点。我们使用一个 `while` 循环来遍历图中的每个节点，并将其添加到已访问的集合中。最后，我们使用 `print` 函数输出搜索结果。

### 7.1 计算机硬件与软件的区别是什么？
