def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

arr = [64, 25, 12, 22, 11]

## 9. 附录：常见问题与解答

### 9.1 问题1

**问题：如何快速排序？**

**解答：** 快速排序是一种高效的排序算法，其核心思想是通过一趟排序将待排序的数据分割成独立的两部分，其中一部分的所有数据都比另一部分的所有数据要小，然后再按此方法对这两部分数据分别进行快速排序，整个排序过程可以递归进行，以此达到整个数据变成有序序列。

### 9.2 问题2

**问题：什么是TCP/IP协议？**

**解答：** TCP/IP协议是互联网的基础协议，由传输控制协议（TCP）和因特网协议（IP）组成。TCP负责数据传输的可靠性和流量控制，而IP负责数据包的路由和传输。

### 9.3 问题3

**问题：什么是动态规划？**

**解答：** 动态规划是一种用于求解最优子结构问题的算法思想，其核心思想是将问题分解为子问题，并利用子问题的解来构建原问题的解，从而避免重复计算。
