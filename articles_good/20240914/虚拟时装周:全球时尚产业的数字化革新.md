const nftContract = new ethers.Contract(nftContractAddress, NFT_ABI, wallet);

// 获取NFT详细信息

##### 3. 如何使用Python实现基于区块链的溯源系统？

**答案：** 使用Python实现基于区块链的溯源系统，可以使用PyQt5库创建图形界面，使用Hyperledger Fabric库实现区块链功能。以下是一个简单的示例：

```python
import sys
from PyQt5.QtWidgets import QApplication, QWidget, QVBoxLayout, QPushButton, QLabel
from hyperledger.fabric import client

class App(QWidget):
    def __init__(self):
        super().__init__()
        self.initUI()

    def initUI(self):
        self.setWindowTitle('Blockchain溯源系统')
        self.setGeometry(300, 300, 300, 200)

        layout = QVBoxLayout()

        self.label = QLabel('请输入溯源编码：')
        layout.addWidget(self.label)

        self.edit = QLineEdit(self)
        layout.addWidget(self.edit)

        self.button = QPushButton('查询')
        layout.addWidget(self.button)

        self.button.clicked.connect(self.query)
        self.setLayout(layout)

    def query(self):
       溯源编码 = self.edit.text()
        response = client.query('query溯源编码 where 溯源编码 = \'%s\'' % 溯源编码)
        result = response.json()
        if result['result']['records']:
            self.label.setText('溯源结果：\n%s' % result['result']['records'][0]['溯源信息'])
        else:
            self.label.setText('未找到相关溯源信息')

if __name__ == '__main__':
    app = QApplication(sys.argv)
    ex = App()
    ex.show()
    sys.exit(app.exec_())
```

**解析：** 该示例使用Hyperledger Fabric库创建了一个简单的溯源系统界面，用户可以输入溯源编码进行查询。实际应用中，需要根据具体需求调整区块链配置、查询语句等。
