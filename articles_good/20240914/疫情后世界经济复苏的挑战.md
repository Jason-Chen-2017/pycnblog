**题目：** 疫情对全球供应链造成了哪些影响？企业可以采取哪些措施来缓解这些影响？

**答案：** 疫情对全球供应链造成了显著影响，主要包括以下几个方面：

1. **供应链中断：** 疫情导致工厂停工、物流受限，导致供应链中断，原材料供应和产品交付受到影响。
2. **物流成本上升：** 疫情期间，国际货运成本显著上升，海运和空运需求增加，运费上涨。
3. **订单延误：** 供应链中断和物流成本上升导致订单交付时间延长，客户满意度降低。
4. **供应链透明度不足：** 企业难以实时掌握供应链上下游的运行情况，增加了供应链风险。

**应对策略：**

1. **多元化供应链：** 企业可以通过建立多元化的供应链，降低对单一供应商的依赖，提高供应链韧性。
2. **加强供应链风险管理：** 企业应制定完善的供应链风险管理计划，包括供应链风险评估、应急响应机制等。
3. **提升供应链透明度：** 通过引入数字化工具，提高供应链的实时监控和数据分析能力，增强供应链透明度。
4. **优化库存管理：** 企业应采用科学的库存管理方法，合理调整库存水平，降低库存成本，提高资金周转率。

**代码示例：** 
（以下为 Python 代码示例，用于模拟供应链风险管理）

```python
import random

class SupplyChain:
    def __init__(self):
        self.status = "正常"
        self.stock = 100

    def risk_assessment(self):
        risk_level = random.choice(["低", "中", "高"])
        if risk_level == "高":
            self.status = "风险中"
            self.stock -= 50
        elif risk_level == "中":
            self.status = "风险低"
            self.stock -= 20
        print(f"当前供应链状态：{self.status}, 库存：{self.stock}")

    def emergency_response(self):
        if self.status == "风险中":
            self.status = "风险已缓解"
            self.stock += 30
        print(f"应急响应后，当前供应链状态：{self.status}, 库存：{self.stock}")

# 实例化供应链对象
supply_chain = SupplyChain()

# 进行风险评估
supply_chain.risk_assessment()

# 执行应急响应
supply_chain.emergency_response()
```

1. **多元化市场：** 企业可以通过拓展新的市场，降低对单一市场的依赖，分散风险。
2. **加强风险管理：** 企业应建立国际贸易风险管理机制，包括风险评估、风险预警、风险应对等。
