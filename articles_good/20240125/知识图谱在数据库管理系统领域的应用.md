### 4.1 实体识别

实体识别可以使用自然语言处理技术，如词性标注、命名实体识别等，来提取文本中的实体。以下是一个简单的Python代码实例：

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.tag import pos_tag

text = "莱斯坦大学位于伦敦，是一所著名的大学。"
tokens = word_tokenize(text)
tags = pos_tag(tokens)

entities = []
for i in range(len(tags)):
    if tags[i][1] in ['NN', 'NNS', 'NNP', 'NNPS']:
        entities.append(tags[i][0])

print(entities)
```

### 4.2 关系识别

关系识别可以使用自然语言处理技术，如依赖解析、语义角色标注等，来提取文本中的关系。以下是一个简单的Python代码实例：

```python
import nltk
from nltk.tokenize import sent_tokenize
from nltk.parse.dependency import DependencyParser

text = "莱斯坦大学位于伦敦，是一所著名的大学。"
sentences = sent_tokenize(text)
parser = DependencyParser(nltk.app.pos_tagger())

for sentence in sentences:
    tree = parser.parse(sentence)
    for subtree in tree.subtrees():
        if subtree.label() == 'nsubj':
            head = subtree.root()
            dependents = subtree.subtrees()
            relations = []
            for dependent in dependents:
                if dependent.label() == 'pobj':
                    relations.append((head, dependent[0]))
            print(relations)
```

### 4.3 实例生成

实例生成可以使用规则引擎、机器学习等技术，来生成知识图谱中的实例。以下是一个简单的Python代码实例：

```python
class Entity:
    def __init__(self, name, type):
        self.name = name
        self.type = type

class Relation:
    def __init__(self, subject, predicate, object):
        self.subject = subject
        self.predicate = predicate
        self.object = object

def generate_instance(entities, relations):
    instance = []
    for relation in relations:
        subject = entities[relation.subject]
        predicate = relation.predicate
        object = entities[relation.object]
        instance.append((subject, predicate, object))
    return instance

entities = [Entity('莱斯坦大学', 'organization'), Entity('伦敦', 'city')]
relations = [Relation(0, 'located_in', 1)]
instance = generate_instance(entities, relations)
print(instance)
```
