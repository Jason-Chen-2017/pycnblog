### 4.1 代码实例

以下是一个UFT的UI自动化测试脚本示例：

```vbscript
Sub TestLogin()
    ' 打开浏览器
    Browser("http://www.example.com").Open

    ' 输入用户名
    Browser("username").Set "admin"

    ' 输入密码
    Browser("password").Set "password"

    ' 点击登录按钮
    Browser("login").Click

    ' 验证是否登录成功
    If Browser("welcome").Exist Then
        MsgBox "登录成功"
    Else
        MsgBox "登录失败"
    End If
End Sub
```

### 4.2 详细解释说明

在这个示例中，我们创建了一个名为`TestLogin`的测试用例，它使用VBScript编写。这个测试用例的目的是测试一个Web应用程序的登录功能。

首先，我们使用`Browser`对象打开一个浏览器，并访问一个URL。然后，我们使用`Browser`对象的`Set`方法输入用户名和密码。接着，我们使用`Browser`对象的`Click`方法点击登录按钮。最后，我们使用`If`语句来验证是否登录成功，如果成功则显示“登录成功”，否则显示“登录失败”。
