### 3.1 序列比对

序列比对是一种比较两个序列相似性的方法，常用于比较DNA、RNA或蛋白质序列。常见的序列比对算法有Needleman-Wunsch算法、Smith-Waterman算法等。

Needleman-Wunsch算法的原理是通过动态规划求解最佳匹配，其核心思想是将序列比对问题转换为求解一个矩阵的最优路径问题。具体操作步骤如下：

1. 初始化一个矩阵，矩阵的行数为序列1的长度，列数为序列2的长度。
2. 填充矩阵中的初始值，矩阵的第0行和第0列的值为-∞，其他值为0。
3. 对于矩阵中的每个单元格，计算其左上方和上方单元格的值，并选择较大的值。如果序列1中的字符与序列2中的字符相同，则加上一个常数（例如1）；如果不同，则减去一个常数（例如-1）。
4. 重复第3步，直到矩阵的最后一个单元格。
5. 找到矩阵的最大值，该值是序列比对的最佳匹配值。

Smith-Waterman算法的原理是通过动态规划求解最佳局部匹配，其核心思想是将序列比对问题转换为求解一个矩阵的最优路径问题。具体操作步骤如下：

1. 初始化一个矩阵，矩阵的行数为序列1的长度，列数为序列2的长度。
2. 填充矩阵中的初始值，矩阵的第0行和第0列的值为-∞，其他值为0。
3. 对于矩阵中的每个单元格，计算其左上方和上方单元格的值，并选择较大的值。如果序列1中的字符与序列2中的字符相同，则加上一个常数（例如1）；如果不同，则减去一个常数（例如-1）。
4. 重复第3步，直到矩阵的最后一个单元格。
5. 找到矩阵中的最大值，该值是序列比对的最佳局部匹配值。

def needleman_wunsch(seq1, seq2):
    len1, len2 = len(seq1), len(seq2)
    dp = [[-float('inf')] * (len2 + 1) for _ in range(len1 + 1)]
    for i in range(len1 + 1):
        dp[i][0] = 0
    for j in range(len2 + 1):
        dp[0][j] = 0
    for i in range(1, len1 + 1):
        for j in range(1, len2 + 1):
            match = 0 if seq1[i - 1] != seq2[j - 1] else 1
            dp[i][j] = max(0, dp[i - 1][j - 1] + match, dp[i - 1][j] - 1, dp[i][j - 1] - 1)
    return dp[len1][len2]
