以下是一个简单的Python代码实例：

```python
import matplotlib.pyplot as plt
import pandas as pd

# 读取CRM平台数据
data = pd.read_csv('crm_data.csv')

# 数据预处理
data = data.dropna()

# 数据可视化

plt.plot(data['客户数量'], data['销售额'])

plt.ylabel('销售额')
plt.title('客户数量与销售额关系')
plt.show()
```
