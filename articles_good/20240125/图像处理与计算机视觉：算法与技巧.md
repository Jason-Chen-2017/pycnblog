### 4.1 滤波算法

滤波算法是一种用于减少图像噪声的方法，例如均值滤波、中值滤波、高通滤波等。以下是一个简单的均值滤波算法的实现：

```python
import numpy as np
import cv2

def mean_filter(image, kernel_size):
    rows, cols = image.shape
    pad_rows, pad_cols = kernel_size // 2, kernel_size // 2
    padded_image = np.pad(image, ((pad_rows, pad_rows), (pad_cols, pad_cols)), 'constant')
    filtered_image = np.zeros_like(image)

    for i in range(rows):
        for j in range(cols):
            filtered_image[i, j] = np.mean(padded_image[i:i+kernel_size, j:j+kernel_size])

    return filtered_image

kernel_size = 5
filtered_image = mean_filter(image, kernel_size)
cv2.imshow('Original Image', image)
cv2.imshow('Filtered Image', filtered_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

在这个实例中，我们使用了均值滤波算法来减少图像噪声。首先，我们使用 `numpy` 库来实现滤波算法，然后使用 `cv2` 库来读取图像并显示原始图像和滤波后的图像。

### 4.2 边缘检测算法

边缘检测算法是一种用于识别图像中的边缘和对象边界的方法，例如迪菲尔德算法、罗尔算法、科夫斯基算法等。以下是一个简单的迪菲尔德算法的实现：

```python
import numpy as np
import cv2
import matplotlib.pyplot as plt

def sobel_filter(image, kernel_size):
    rows, cols = image.shape
    pad_rows, pad_cols = kernel_size // 2, kernel_size // 2
    padded_image = np.pad(image, ((pad_rows, pad_rows), (pad_cols, pad_cols)), 'constant')
    filtered_image = np.zeros_like(image)

    sobel_x = np.array([[-1, 0, 1], [-2, 0, 2], [-1, 0, 1]])
    sobel_y = np.array([[-1, -2, -1], [0, 0, 0], [1, 2, 1]])

    for i in range(rows):
        for j in range(cols):
            gx, gy = 0, 0
            for k in range(kernel_size):
                gx += sobel_x[k][j] * padded_image[i+k, j]
                gy += sobel_y[k][j] * padded_image[i, j+k]
            filtered_image[i, j] = np.sqrt(gx**2 + gy**2)

    return filtered_image

kernel_size = 5
filtered_image = sobel_filter(image, kernel_size)

plt.subplot(1, 2, 1), plt.imshow(image, cmap='gray')
plt.title('Original Image'), plt.xticks([]), plt.yticks([])
plt.subplot(1, 2, 2), plt.imshow(filtered_image, cmap='gray')
plt.title('Sobel Filtered Image'), plt.xticks([]), plt.yticks([])
plt.show()
```

在这个实例中，我们使用了迪菲尔德算法来检测图像中的边缘。首先，我们使用了 `numpy` 库来实现滤波算法，然后使用 `cv2` 库来读取图像并显示原始图像和滤波后的图像。
