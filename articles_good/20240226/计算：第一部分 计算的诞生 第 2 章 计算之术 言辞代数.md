### 常见运算

* **加减乘除**

加减乘除是最基本的运算。它们的定义如下：

$$a + b = c$$
$$a - b = d$$
$$a \times b = e$$
$$a \div b = f$$

* **指数运算**

指数运算是指数与底数的运算。它的定义如下：

$$a^b = c$$

### 常见函数

* **线性函数**

线性函数是直线上的点的映射。它的定义如下：

$$f(x) = kx + b$$

* **二次函数**

二次函数是抛物线上的点的映射。它的定义如下：

$$f(x) = ax^2 + bx + c$$
