**4.1 销售量预测模型**

销售量预测模型是限时优惠策略中的重要工具，用于预测在设定的时间范围内，通过优惠活动可以实现的销售额。以下是销售量预测模型的基本公式：

$$
S = p \times Q
$$

其中：

- \( S \) 表示预测的销售额（单位：元）
- \( p \) 表示优惠力度（单位：折扣比例，例如8折表示0.8）
- \( Q \) 表示预测的销售量（单位：件）

**4.2 用户参与度模型**

用户参与度模型用于预测在限定的时间内，有多少潜在用户会参与优惠活动。以下是用户参与度模型的基本公式：

$$
R = \frac{C \times D}{1 + e^{-(k \times T)}}
$$

其中：

- \( R \) 表示用户参与度（单位：比例，例如0.8表示80%的用户参与）
- \( C \) 表示潜在用户总数（单位：人）
- \( D \) 表示用户参与活动的概率（单位：比例，例如0.1表示10%的用户参与活动）
- \( k \) 表示时间衰减系数（单位：1/天，例如0.1表示每天衰减10%）
- \( T \) 表示活动持续时间（单位：天）

**4.3 优化策略模型**

在限时优惠策略的实施过程中，企业需要不断优化策略，以提高销售量和用户参与度。以下是一个简单的优化策略模型：

$$
p_{\text{new}} = p_{\text{current}} + \alpha \times (p_{\text{max}} - p_{\text{current}})
$$

其中：

- \( p_{\text{new}} \) 表示新的优惠力度（单位：折扣比例）
- \( p_{\text{current}} \) 表示当前的优惠力度（单位：折扣比例）
- \( p_{\text{max}} \) 表示最大的优惠力度（单位：折扣比例，例如9折表示0.9）
- \( \alpha \) 表示优化步长（单位：比例，例如0.1表示每次优化增加10%）

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 参数设置
C = 1000  # 潜在用户总数
D = 0.1   # 用户参与活动的概率
k = 0.1   # 时间衰减系数
p_max = 0.8  # 最大优惠力度
p_current = 0.7  # 初始优惠力度
alpha = 0.1  # 优化步长
T = 30     # 活动持续时间（天）

# 用户参与度模型
def user_involvement(C, D, k, T):
    R = C * D / (1 + np.exp(-k * T))
    return R

# 优化策略模型
def optimize_discount(p_current, p_max, alpha):
    p_new = p_current + alpha * (p_max - p_current)
    return p_new

# 计算并打印用户参与度
R = user_involvement(C, D, k, T)
print(f"初始用户参与度：{R:.2f}")

# 优化优惠力度并打印结果
p_new = optimize_discount(p_current, p_max, alpha)
print(f"优化后优惠力度：{p_new:.2f}")

# 重复优化过程并绘制图表
R_values = []
p_values = []
for i in range(T):
    p_values.append(p_current)
    R = user_involvement(C, D, k, i + 1)
    R_values.append(R)
    p_current = optimize_discount(p_current, p_max, alpha)

plt.plot(range(T), R_values, label='User Involvement')
plt.plot(range(T), p_values, label='Discount Rate')
plt.xlabel('Days')
plt.ylabel('Value')
plt.title('User Involvement and Discount Rate Optimization')
plt.legend()
plt.show()
