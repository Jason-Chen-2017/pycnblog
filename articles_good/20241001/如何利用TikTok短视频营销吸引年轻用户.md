### 4. 数学模型和公式 & 详细讲解 & 举例说明

#### 数学模型

为了提高短视频的推荐效果，可以采用基于用户行为的协同过滤算法。其基本公式为：

$$
\hat{r_{ui}} = \frac{\sum_{j \in N(i)} r_{uj} \cdot sim(i, j)}{\sum_{j \in N(i)} sim(i, j)}
$$

其中，$r_{ui}$ 表示用户 $u$ 对项目 $i$ 的评分预测，$N(i)$ 表示与项目 $i$ 相似的项目集合，$r_{uj}$ 表示用户 $u$ 对项目 $j$ 的真实评分，$sim(i, j)$ 表示项目 $i$ 和项目 $j$ 的相似度。

#### 举例说明

假设有两个用户 $u_1$ 和 $u_2$，以及两个短视频 $i_1$ 和 $i_2$。用户 $u_1$ 对短视频 $i_1$ 给予5星评价，对短视频 $i_2$ 给予4星评价；用户 $u_2$ 对短视频 $i_1$ 给予3星评价，对短视频 $i_2$ 给予5星评价。假设两个短视频的相似度为1，则根据上述公式，用户 $u_1$ 对短视频 $i_2$ 的评分预测为：

$$
\hat{r_{u1i2}} = \frac{5 \cdot 1 + 4 \cdot 1}{1 + 1} = 4.5
$$

import numpy as np

def cosine_similarity(x, y):
    return np.dot(x, y) / (np.linalg.norm(x) * np.linalg.norm(y))

def collaborative_filtering(ratings, items, k=5):
    user_item_similarity = {}
    for i in items:
        for j in items:
            if i != j:
                user_item_similarity[i][j] = cosine_similarity(ratings[i], ratings[j])
    return user_item_similarity

def predict_rating(ratings, user_item_similarity, user, item, k=5):
    sim_sum = 0
    sim_sum_rating = 0
    for j in user_item_similarity[user]:
        sim_sum += user_item_similarity[user][j]
        sim_sum_rating += user_item_similarity[user][j] * ratings[user][j]
    if sim_sum == 0:
        return np.mean(list(ratings.values()))
    else:
        return sim_sum_rating / sim_sum

# 测试数据
ratings = {

    'u2': [0, 0, 3, 5, 0],

}

items = ['i1', 'i2', 'i3', 'i4', 'i5']

user_item_similarity = collaborative_filtering(ratings, items, k=5)

# 预测用户 u1 对 i5 的评分
predicted_rating = predict_rating(ratings, user_item_similarity, 'u1', 'i5')
