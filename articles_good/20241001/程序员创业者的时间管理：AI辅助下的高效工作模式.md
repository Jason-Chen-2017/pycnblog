## 4. 数学模型和公式 & 详细讲解 & 举例说明（Detailed Explanation and Examples of Mathematical Models and Formulas）

### 4.1 任务优先级排序模型

在任务优先级排序中，我们可以使用如下数学模型：

$$
P(i) = \frac{E(i) \times D(i)}{U(i)}
$$

其中，$P(i)$ 表示任务 $i$ 的优先级，$E(i)$ 表示任务 $i$ 的紧急程度，$D(i)$ 表示任务 $i$ 的重要性，$U(i)$ 表示任务 $i$ 的紧急程度。

### 4.2 工作时间优化模型

工作时间优化模型的目标是使工作效率最大化。我们可以使用如下数学模型：

$$
\max_{x} W(x)
$$

$$
s.t. \quad C(x) \leq T
$$

其中，$W(x)$ 表示工作效率，$C(x)$ 表示工作完成所需时间，$T$ 表示总工作时间。

### 4.3 举例说明

假设我们有以下三个任务：

- 任务A：紧急程度 $E(A) = 3$，重要性 $D(A) = 4$，紧急程度 $U(A) = 2$
- 任务B：紧急程度 $E(B) = 2$，重要性 $D(B) = 3$，紧急程度 $U(B) = 1$
- 任务C：紧急程度 $E(C) = 1$，重要性 $D(C) = 5$，紧急程度 $U(C) = 3$

根据任务优先级排序模型，我们可以计算出每个任务的优先级：

$$
P(A) = \frac{E(A) \times D(A)}{U(A)} = \frac{3 \times 4}{2} = 6
$$

$$
P(B) = \frac{E(B) \times D(B)}{U(B)} = \frac{2 \times 3}{1} = 6
$$

$$
P(C) = \frac{E(C) \times D(C)}{U(C)} = \frac{1 \times 5}{3} = \frac{5}{3}
$$

根据优先级排序，任务A和任务B的优先级最高，应先完成。

根据工作时间优化模型，我们可以计算出每个任务的工作时间：

$$
W(x) = \max \left( \frac{C(A)}{C(B)}, \frac{C(B)}{C(C)} \right)
$$

$$
s.t. \quad C(A) + C(B) + C(C) \leq T
$$

假设总工作时间 $T = 24$ 小时，我们可以计算出每个任务的工作时间：

$$
W(A) = \frac{C(A)}{C(B)} = \frac{4}{3} = 1.33 \quad \text{小时}
$$

$$
W(B) = \frac{C(B)}{C(C)} = \frac{3}{5} = 0.6 \quad \text{小时}
$$

$$
W(C) = T - W(A) - W(B) = 24 - 1.33 - 0.6 = 22.07 \quad \text{小时}
$$

因此，任务A需要1.33小时，任务B需要0.6小时，任务C需要22.07小时。

```python
import pandas as pd
import numpy as np
from sklearn.cluster import KMeans

# 5.2.1 数据收集
def collect_data():
    data = pd.DataFrame({
        'Task': ['A', 'B', 'C'],
        'E': [3, 2, 1],
        'D': [4, 3, 5],
        'U': [2, 1, 3]
    })
    return data

# 5.2.2 任务优先级排序
def task_priority_sort(data):
    data['P'] = (data['E'] * data['D']) / data['U']
    sorted_data = data.sort_values(by='P', ascending=False)
    return sorted_data

# 5.2.3 工作时间优化
def work_time_optimization(data, total_time):
    kmeans = KMeans(n_clusters=3)
    data['Cluster'] = kmeans.fit_predict(data[['E', 'D', 'U']])
    work_time = {}
    for i in range(3):
        work_time[f'W_{i+1}'] = total_time / kmeans.inertia_
    return work_time

# 5.2.4 主函数
def main():
    data = collect_data()
    sorted_data = task_priority_sort(data)
    work_time = work_time_optimization(sorted_data, total_time=24)
    print("排序后的任务：")
    print(sorted_data)
    print("工作时间优化：")
    print(work_time)

if __name__ == "__main__":
    main()
```

- **5.3.4 主函数**：`main` 函数调用其他函数，完成数据收集、任务排序和工作时间优化，并打印结果。

```python
# 5.2.2 任务优先级排序
def task_priority_sort(data):
    data['P'] = (data['E'] * data['D']) / data['U']
    sorted_data = data.sort_values(by='P', ascending=False)
    return sorted_data

# 5.2.3 工作时间优化
def work_time_optimization(data, total_time):
    kmeans = KMeans(n_clusters=3)
    data['Cluster'] = kmeans.fit_predict(data[['E', 'D', 'U']])
    work_time = {}
    for i in range(3):
        work_time[f'W_{i+1}'] = total_time / kmeans.inertia_
    return work_time

# 5.2.4 主函数
def main():
    data = collect_data()
    sorted_data = task_priority_sort(data)
    work_time = work_time_optimization(sorted_data, total_time=24)
    print("排序后的任务：")
    print(sorted_data)
    print("工作时间优化：")
    print(work_time)

if __name__ == "__main__":
    main()
```

- **5.3.2 任务优先级排序**：`task_priority_sort` 函数根据任务优先级排序模型，计算每个任务的优先级，并返回排序后的任务数据。
- **5.3.3 工作时间优化**：`work_time_optimization` 函数使用K-Means聚类算法，根据任务的紧急程度和重要性，为每个任务分配工作时间。
- **5.3.4 主函数**：`main` 函数调用其他函数，完成数据收集、任务排序和工作时间优化，并打印结果。
