## 2. 核心概念与联系

### TensorFlow

TensorFlow是由Google Brain团队开发的一个开源的机器学习框架，它支持各种类型的机器学习模型，包括神经网络、决策树、支持向量机等。TensorFlow的主要特点是灵活性、可扩展性和可移植性，它可以在各种平台上运行，包括移动设备、服务器和浏览器。

### Keras

Keras是一个高级神经网络API，它由纯Python编写而成，可以在TensorFlow、Microsoft Cognitive Toolkit、PlaidML和RAPIDS CuDNN等后端运行。Keras的设计理念是代码简洁、模块性强，是一个高度模块化和可扩展的神经网络库，非常适合快速实验。

### 联系

TensorFlow和Keras之间有着密切的联系。TensorFlow是一个底层的机器学习框架，它提供了各种组件和API，用于构建和训练机器学习模型。Keras是一个高层的API，它建立在TensorFlow之上，提供了更加简洁易用的API，可以快速构建和训练神经网络模型。因此，许多机器学习开发者会使用Keras作为前端，使用TensorFlow作为后端。

### TensorFlow

以下是一个使用TensorFlow构建和训练神经网络模型的简单代码示例：
```python
import tensorflow as tf

# 定义神经网络模型
model = tf.keras.models.Sequential([
    tf.keras.layers.Flatten(input_shape=(28, 28)),
    tf.keras.layers.Dense(128, activation='relu'),
    tf.keras.layers.Dropout(0.2),
    tf.keras.layers.Dense(10, activation='softmax')
])

# 定义损失函数
loss_fn = tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True)

# 定义优化器
optimizer = tf.keras.optimizers.Adam()

# 定义训练步骤
@tf.function
def train_step(images, labels):
    with tf.GradientTape() as tape:
        predictions = model(images, training=True)
        loss = loss_fn(labels, predictions)
    gradients = tape.gradient(loss, model.trainable_variables)
    optimizer.apply_gradients(zip(gradients, model.trainable_variables))
    return loss

# 训练神经网络模型
for images, labels in train_data:
    loss = train_step(images, labels)
    print(f'Loss: {loss.numpy()}')
```
