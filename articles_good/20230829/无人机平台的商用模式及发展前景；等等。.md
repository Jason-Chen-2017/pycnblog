# 2.基本概念术语说明
## 2.1 无人机
无人机（Unmanned Aerial Vehicle，UAV）是一种运用无人驾驶技术的机器，主要用于空间、海洋和空中航行的应用，由无人机制造公司、研究机构或国际合作机构制造。无人机通常具备各种复杂功能，包括拍摄、视频监控、巡航、测绘、导航、精确定位、识别目标等。它的命名来源于美国电影《蒙提·D·摩尔》中的飞机。

目前世界上有三种类型的无人机：

1. 静态无人机：静态无人机一般是指没有移动能力，因此它们不能执行远距离跟踪任务。例如军用无人机，用于敌方打击演习、侦察搜索、情报收集等。

2. 近卫无人机：近卫无人机则具有较大的移动范围，可以在极小的时间内进行多种任务，如无人监视、搜救、遥感调查、地面监测等。

3. 宇宙无人机：宇宙无人机与太空飞行器一样，具备穿梭在宇宙空间的能力，可以进行星际飞行、星际旅行等任务。

无人机平台是指能够提供无人机操作系统、无人机控制、通信、数据传输、信息处理、图像处理、云计算、数据分析等功能的硬件和软件平台，它向用户提供一系列完整的应用服务，包括：

1. 无人机机载控制：通过无人机操纵杆、遥控器或网页控制器，使得用户能够通过无人机完成各种任务，比如悬停、起降、着陆、拍照、视频监控等。

2. 数据上传下载：平台可以通过无线方式将数据上传到云端，或者从云端下载数据。

3. 信息共享：平台可以将用户的各种信息发送给其他平台，也可以接收并处理其他平台发送来的信息。

4. 服务支持：平台还可以提供各种服务，如自主学习、自适应优化、轨迹规划、遥感分析、实时检疫、弹道导弹防御、垂直起降、底盘可控性测试等。

无人机平台除了提供上面这些服务之外，还可以提供以下一些附加功能：

1. 远程控制：无人机可以通过无线或有线网络远程控制，这样可以让用户在不离开自己的住处的情况下操控无人机。

2. 扩展功能：无人机平台除了提供普通的无人机控制功能外，还可以额外提供一系列拓展功能，如激光雷达、地图导航、图像识别、声纳传感器、气压计等。

3. 教育培训：无人机平台可以提供专门针对儿童和老年人的教育训练服务，帮助学校、家庭、企业等提供适合学生学习的课程。

总而言之，无人机平台可以帮助用户解决日益增长的无人机市场发展的需求，并且它还可以提供无人机服务的定制化、个性化。因此，无人机平台建设是一个十分复杂和艰难的过程。

## 2.2 平台形态
无人机平台包括两大类：

1. 服务型平台：平台本身直接提供某项特定的无人机服务，如自动化取样、目标识别、地图导航、无人机引导等。

2. 智能型平台：平台为无人机提供全套的整体方案，包括整机定制、设备安装、招募人才、培训人员、开发工具、云服务等。

静态无人机平台：这是最原始的无人机平台形态。它通常由专业的工程师、设备制造商、技术顾问和领域专家组成，根据客户的需求进行一站式服务。但由于缺乏支付能力、资源限制等原因，价格不菲。静态无人机平台的典型特征：

1. 大批量、昂贵：静态无人机平台的购买成本非常高，尤其是在成本较低的区域，比如中国。购买数量上万台，单台价格约千元左右。

2. 服务时间长、质量差：静态无人机平台通常只能按固定时间段进行服务，即便是熟练的技工也无法及时完成每一次任务。而且由于缺乏成熟的软件，服务质量很差。

3. 服务水平一般：静态无人机平台的服务水平比较一般，因为往往只是为某一类用户提供某项特定的服务。

近卫无人机平台：近卫无人机平台的历史可以追溯到上世纪70年代末，是静态无人机平台的改进版。近卫无人机平台的特征如下：

1. 小批量、便宜：近卫无人机平台的购买成本相比静态无人机平台要低很多，价格普遍在百元左右。购买数量上万台左右。

2. 服务时间短、质量高：近卫无人机平台可以快速响应客户需求，并且拥有高性能、高可靠的软硬件设施，所以服务时间短，服务质量高。

3. 服务水平高：近卫无人机平台的服务水平相当高，覆盖了静态无人机平台所不具备的各类服务，如自动化取样、无人机引导、地图导航、无人机精准定位等。

宇宙无人机平台：宇宙无人机平台的出现，标志着无人机向更加高维度的发展方向迈出了一大步。宇宙无人机平台的特征如下：

1. 高度可定制：宇宙无人机平台的整机可以根据客户的需要进行高度定制，如传感器选择、精密尺寸、材料、动力装置、计算机配置等。

2. 紧张的市场需求：宇宙无人机平台的需求一直都在持续增长，而且市场的供需情况也非常不确定。

3. 更大的市场空间：宇宙无人机平台的部署可以满足更多的市场需求，比如军事、科研、卫星等领域。

综上所述，无人机平台可以分为两大类：静态无人机平台和近卫无人机平台。静态无人机平台市场占有率较低，但是服务时间长，服务质量差；而近卫无人机平台价格便宜，服务时间短，服务质量高，同时覆盖了静态无人机平台所不具备的各类服务。宇宙无人机平台目前市场上还处于早期阶段，但是却有着前景，在不久的将来一定会成为主流无人机平台形态。

## 2.3 业务流程
无人机平台的业务流程包括四个部分：

1. 订单采购：在正式购买之前，客户首先需要提交订单，要求平台提供相应的服务，并填写相关申请表格。平台收到订单后，首先进行订单审核，然后依据订单信息进行设备选购、安装等过程。

2. 服务交付：平台按照订单指定的要求，安装必要的设备、软件，为客户提供无人机服务。服务一般包括两种形式：远程服务和固定位置服务。

3. 服务维护：平台为了确保无人机平台的稳定运行，需要定期对平台进行维护，维修设备，升级软件等。

4. 技术支持：平台需要为客户提供相关技术支持，包括硬件故障排除、软件调试、拓展功能的集成、新技术的研究开发等。

其中，订单采购、服务交付、服务维护、技术支持四个环节构成了无人机平台的核心业务流程。订单采购、服务维护、技术支持环节主要依赖人力投入，而服务交付环节则依赖设备硬件、软件的开发和测试工作量巨大。

## 2.4 业务模式
无人机平台的业务模式主要包括：

1. 服务型平台：服务型平台主要通过自己开发的软件、硬件系统，实现与客户的无人机服务。

2. 智能型平台：智能型平台根据客户的需求，提供不同类型、级别的无人机服务，如定制开发、试验认证、培训、运营等。智能型平台通常具有预算充裕、知识结构完善、专业人才训练的优势。

3. 混合型平台：混合型平台既提供无人机服务，又承担智能型平台的某些工作，如设备开发、运营等。这种平台既有高端的技术人员参与，又有较强的经验积累。

### （1）服务型平台
服务型平台为无人机用户提供了各种无人机操作、数据上传、信息共享、无人机引导等服务，如远程控制、目标识别、底盘测试、地图导航、数据分析等。服务型平台的特点是价格较高、专业人员培训成本低，但服务质量难以保证。服务型平台的典型代表是西安一汽大众。

### （2）智能型平台
智能型平台由专业的工程师、设备制造商、技术顾问和领域专家组成，提供统一的、全面的、集成化的服务，包括定制开发、试验认证、培训、运营等。智能型平台具有专业人员数量多、团队协作配合能力强、预算充裕、财政政策灵活等优势。智能型平台的典型代表是微软、谷歌、NVIDIA等国际知名科技公司。

### （3）混合型平台
混合型平台既具有服务型平台和智能型平台的特性，既为用户提供无人机服务，又承担智能型平台的部分工作。混合型平台的部署包括先购买服务型平台、再购买或租赁智能型平台等。混合型平台的优势在于能够兼顾技术和资源上的优势，在适合的场景下，可以实现无人机服务的有效整合。混合型平台的典型代表是北京九龙仓无人机公司、浙江英德无人机创业公司等。

## 2.5 市场营销策略
无人机平台的市场营销策略是指通过宣传推广、品牌营销、渠道代理、线下活动、网络营销、社区服务、促销等方式吸引和留住目标客户。在无人机平台推广中，营销策略必须符合多种目标受众。一般分为消费者目标受众、品牌管理者目标受众、技术人员目标受众。

1. 消费者目标受众：消费者目标受众是指平台的最终用户群体，主要是普通消费者、家庭主妇等。无人机平台推广时，应该专注于消费者群体，力求增加产品的知名度和口碑，同时要注意营销活动的时效性。

2. 品牌管理者目标受众：品牌管理者目标受众是指负责品牌宣传的人员，主要是商业、媒体、公关等相关人员。无人机平台推广时，应该考虑品牌理念、宣传内容、品牌口碑等，将平台的产品融入到品牌价值观中。

3. 技术人员目标受众：技术人员目标受众是指负责平台开发、生产、运营的人员，主要是科技公司、政府部门、高校等机构。无人机平台推广时，应该优先关注技术人员的满意度、接受度，同时也要注重技术硬件的适用性。

## 2.6 技术路线图
无人机平台的技术路线图是指无人机平台的技术构架、功能模块、技术规范、标准化进程和管理机制，以及交付的计划。技术路线图主要包括以下几个方面：

1. 平台框架：平台框架通常包括无人机机架、无人机控制中心、信息采集中心、数据分析中心、云计算中心、机器学习中心等。平台框架决定了无人机平台的整体结构。

2. 平台功能模块：平台功能模块是指无人机平台中的各项功能模块，如无人机控制、图像识别、通信、无人机轨迹规划、云端存储等。平台功能模块的数量、分布、功能特性均影响平台的性能和稳定性。

3. 标准化过程：标准化过程是指平台的开发过程中，从工程角度、管理角度、组织角度、法律角度等多个角度推行平台的标准化。标准化有利于提升平台的标准化程度，减少相关问题的出现。

4. 管理机制：管理机制是指平台的各项管理措施，如服务与支持、设备质保、安全管理、工程管理、HR管理、财务管理、人力资源管理、技术支持等。管理机制是无人机平台的基石，必须得到高度重视。

5. 投产计划：投产计划是指平台的计划生命周期中，各环节投入的时间、资源投入比例、质量评估标准、目标、达成率等，及平台上线后的发展路径。投产计划的实施对平台的整体生态、盈利能力、竞争力、技术创新能力等，起到重要作用。

## 2.7 团队构成
无人机平台的团队构成包括以下几个部分：

1. 产品设计与研发部门：产品设计与研发部门负责平台的整体设计、规划、开发、测试、系统集成与支持等工作。

2. 售前支持部门：售前支持部门负责对客户的需求进行分析，并向客户提供必要的技术支持。

3. 售后服务部门：售后服务部门主要负责向客户提供包括设备安装、使用教育、技术咨询、售后维护等服务。

4. 测试与验证部门：测试与验证部门为平台的开发、测试提供专业的人力资源。

5. 销售部门：销售部门为客户提供各种无人机服务，如无人机控制、无人机引导、目标识别、底盘测试等。

6. 市场营销部门：市场营销部门负责平台推广、宣传、促销等工作。

7. 人力资源部门：人力资源部门负责平台人员的招聘、培训、管理等工作。

8. 技术支持部门：技术支持部门为平台的技术人员提供定制化的技术服务，如软件开发、设备安装、软件调试等。

以上八个部门构成了无人机平台的全面团队。

## 2.8 管理模式
管理模式是指无人机平台的管理制度和执行方式。管理模式包括以下几种：

1. 服务型平台管理模式：服务型平台管理模式，主要是指服务型平台的管理人员由专业人才构成。管理人员通过对客户的需求进行专业化管理，最大限度地满足客户的需求。

2. 智能型平台管理模式：智能型平台管理模式，主要是指智能型平台的管理人员由一批有专业知识、经验、能力的技术人才构成。管理人员通过对无人机平台的整体技术能力进行持续培养、评估和提升，确保无人机平台持续稳定发展。

3. 混合型平台管理模式：混合型平台管理模式，是指混合型平台的管理人员既是技术人员也是管理人员。该管理人员既负责平台的开发和维护，又能对平台进行指导、管理和监督。

## 2.9 商业模式
商业模式是指无人机平台在市场中的竞争策略、组织架构、资源配置、销售渠道、客户关系等方面的行为模式和结果。无人机平台的商业模式，主要包括以下几个方面：

1. 服务收费模式：服务收费模式是指无人机平台收取服务费用的模式。主要包括标准模式和自定义模式。标准模式是指无人机平台为客户提供固定的服务费用，例如，一些服务型平台的服务费用是一定的，而一些智能型平台的服务费用按套餐收费，客户按照套餐的服务费用进行消费。

2. 平台商业模式：平台商业模式主要是指无人机平台的销售模式。主要包括自销售模式、托管销售模式、服务卖场模式、平台连锁店模式等。自销售模式是指无人机平台直接在平台提供销售服务，如智能型平台通过网站、微信、APP等渠道进行销售。托管销售模式是指无人机平台委托第三方平台来进行销售。服务卖场模式是指无人机平台与第三方平台建立合作关系，将自身提供的服务卖给第三方平台的用户。平台连锁店模式是指无人机平台在平台之间构建多层级联营体系，通过平台间的组合推出更高级的服务。

3. 定制开发模式：定制开发模式是指无人机平台通过内部、外部合作者的协同配合，提供不同级别的定制开发服务。无人机平台可以在内部进行软件开发、硬件开发、算法优化、平台升级等，也可以利用外部合作者的资源，如算法模型、图像识别技术、云计算平台等进行定制开发。

4. 服务保障模式：服务保障模式是指无人机平台对客户的服务质量进行保障。主要包括免费保障、收费保障、优惠保障等。免费保障是指无人机平台不收取任何费用为客户提供服务。收费保障是指无人机平台向客户收取一定的服务费用，以此作为平台的营收来源。优惠保障是指无人机平台向部分客户提供优惠券，以此吸引部分客户消费。

## 2.10 发展前景
无人机平台的发展前景主要包括三个方面：

1. 销量：无人机平台的销量是指无人机平台的货物或劳务的数量和质量。目前，无人机平台的销量已经远超过传统的运输工具，例如，中国航空10号舱的销量已占到整个飞机公司销量的三分之一。因此，无人机平台将继续保持稳健的增长势头。

2. 规模：无人机平台的规模是指无人机平台的设备数量和规模。目前，无人机平台的规模仍然处于初级阶段，但是随着未来发展的不断推动，无人机平台将逐渐壮大，进入中高级阶段。

3. 专业化：无人机平台的专业化是指无人机平台的研发水平。目前，无人机平台的研发人员水平仍然落后于主流行业。因此，无人机平台的研发人员将继续高举创新理念，开发高质量的无人机软件、硬件和算法。

## 2.11 存在的关键问题
无人机平台存在的关键问题主要有以下几点：

1. 专业知识和技术缺乏：目前，无人机平台的专业人员较少，只有少数技术人员，技术水平较低，产品和服务水平都较低。另外，当前的无人机平台技术平台较为封闭，无法满足各领域的需求，导致各方面服务水平不足，导致无人机平台的普及率不高。

2. 人才和技术储备缺乏：目前，无人机平台还缺乏高素质的人才和专业技术储备。虽然无人机平台的管理结构有一定的健全性，但各方面的专业人才缺乏，加上技术储备薄弱，使得平台技术成果不断受到质疑和挑战。

3. 服务模式缺乏创新：无人机平台的服务模式还处于初级阶段，产品和服务水平仍然较低。导致服务模式缺乏创新性，客户对无人机平台的体验也不好。

## 2.12 突出的优势
无人机平台有两个突出的优势：

1. 规模优势：无人机平台规模优势是指无人机平台的设备数量和规模越来越庞大。目前，无人机平台的设备数量已达到5亿，占据无人机市场的70%。虽然还有很多潜力待挖，但是目前看来，无人机平台的规模优势不会消失。

2. 用户体验优势：无人机平台的用户体验优势是指无人机平台的用户体验越来越好。无人机平台为用户提供了更高级、更专业的服务，用户的使用体验更好，因此，用户体验优势将会越来越明显。
