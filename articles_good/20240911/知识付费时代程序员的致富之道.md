#### 典型面试题库

**1. 什么是面向对象编程？请举例说明。**

**答案：** 面向对象编程是一种编程范式，它将软件设计基于对象的概念。对象是具有属性（数据）和方法（功能）的实体，它们通过封装、继承、多态等机制相互作用。

**举例：** 

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def bark(self):
        print(f"{self.name} is barking!")

dog = Dog("Buddy", 5)
dog.bark()  # 输出: Buddy is barking!
```

**解析：** 在这个例子中，`Dog` 是一个类，它定义了对象的属性（`name` 和 `age`）和方法（`bark`）。创建 `Dog` 类的实例 `dog`，并调用 `bark` 方法打印出相应的信息。

**2. 请解释单例模式及其应用场景。**

**答案：** 单例模式是一种设计模式，确保一个类只有一个实例，并提供一个全局访问点。它通常用于控制对某些资源（如数据库连接、文件系统操作、配置对象）的访问。

**应用场景：**

* 确保关键资源不被重复创建。
* 系统中的某些组件需要共享状态。
* 控制对共享资源的访问。

**示例代码：**

```python
class Database:
    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super(Database, cls).__new__(cls)
        return cls._instance

    def connect(self):
        print("连接数据库")

database = Database()
database.connect()  # 输出: 连接数据库
```

**解析：** 在这个例子中，`Database` 类使用 `__new__` 方法来确保只有一个实例。每次尝试创建 `Database` 对象时，都会检查 `_instance` 是否为 `None`。如果是，则创建一个新实例，否则返回现有实例。

#### 算法编程题库

**1. 实现一个函数，判断一个字符串是否为回文。**

**答案：**

```python
def is_palindrome(s: str) -> bool:
    return s == s[::-1]

# 示例
print(is_palindrome("racecar"))  # 输出: True
print(is_palindrome("python"))   # 输出: False
```

**解析：** 该函数通过比较字符串与其逆序来判断是否为回文。

**2. 实现一个函数，找出数组中的第二大元素。**

**答案：**

```python
def find_second_largest(nums: List[int]) -> int:
    if len(nums) < 2:
        return -1
    
    max1, max2 = nums[0], nums[1]
    
    for num in nums[1:]:
        if num > max1:
            max2 = max1
            max1 = num
        elif num > max2:
            max2 = num
    
    return max2

# 示例
print(find_second_largest([1, 2, 3, 4, 5]))  # 输出: 4
print(find_second_largest([5, 5, 5, 5]))     # 输出: -1
```

**解析：** 该函数通过遍历数组，维护两个最大值来找出第二大元素。
