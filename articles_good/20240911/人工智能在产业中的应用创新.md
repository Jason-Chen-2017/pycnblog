## 算法编程题库与解析

### 1. K近邻算法

**题目：** 实现K近邻算法，完成对给定测试数据的分类。

**答案：** K近邻算法是一种基于距离的监督学习算法。给定训练数据和测试数据，计算测试数据与训练数据之间的距离，选取距离最近的K个邻居，然后根据邻居的分类结果进行投票，选取出现次数最多的类别作为测试数据的分类结果。

**示例代码：**

```python
import numpy as np
from collections import Counter

def euclidean_distance(x1, x2):
    return np.sqrt(np.sum((x1 - x2) ** 2))

def k_nearest_neighbors(train_data, train_labels, test_data, k):
    distances = []
    for test_sample in test_data:
        dist = euclidean_distance(test_sample, train_data)
        distances.append(dist)
    sorted_distances = sorted(zip(distances, train_labels))
    neighbors = sorted_distances[:k]
    neighbor_labels = [label for _, label in neighbors]
    most_common = Counter(neighbor_labels).most_common(1)
    return most_common[0][0]

# 示例数据
train_data = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
train_labels = np.array([0, 0, 1, 1])
test_data = np.array([[2, 2.5]])

# 预测
predicted_label = k_nearest_neighbors(train_data, train_labels, test_data, 2)
print("Predicted label:", predicted_label)
```

### 2. 支持向量机

**题目：** 实现支持向量机（SVM）算法，完成对给定测试数据的分类。

**答案：** 支持向量机（SVM）是一种线性分类模型。给定训练数据和测试数据，通过寻找最优超平面，将数据划分为不同的类别。

**示例代码：**

```python
import numpy as np
from numpy.linalg import inv

def svm(train_data, train_labels, C=1.0):
    # 数据预处理
    X = train_data
    y = train_labels.reshape(-1, 1)
    
    # 求解最优超平面
    alpha = np.zeros(len(y))
    b = 0
    for i in range(len(y)):
        grad = np.dot(X, alpha * y) - y[i] * np.dot(X[i], X[i]) + b
        alpha[i] -= 1 / C * grad
    
    # 求解拉格朗日乘子
    L = np.dot(X.T, X) - np.diag(alpha) - np.diag(y * np.diag(X @ alpha)) - b * np.ones((len(y), 1))
    P = L - np.diag(L) + np.diag(alpha * y)
    Q = np.dot(inv(P), L)
    Q = np.array([[q[0], q[1]] for q in Q])
    
    # 求解最优超平面
    w = Q @ alpha * y
    b = y - np.dot(X, w)
    
    # 分类
    def predict(x):
        return np.sign(np.dot(x, w) + b)
    
    return predict

# 示例数据
train_data = np.array([[1, 1], [1, 2], [-1, 0], [-1, -1]])
train_labels = np.array([1, 1, -1, -1])
test_data = np.array([[0, 0]])

# 训练SVM模型
model = svm(train_data, train_labels)

# 预测
predicted_label = model(test_data)
print("Predicted label:", predicted_label)
```

### 3. 决策树

**题目：** 实现决策树算法，完成对给定测试数据的分类。

**答案：** 决策树是一种基于特征划分的监督学习算法。给定训练数据和测试数据，通过递归划分数据，构建一棵树，树的叶子节点代表分类结果。

**示例代码：**

```python
import numpy as np

def decision_tree(train_data, train_labels, depth=0, max_depth=3):
    # 判断是否达到最大深度
    if depth == max_depth:
        return np.mean(train_labels)
    
    # 计算特征的重要性
    feature_importance = np.std(train_labels, axis=0)
    best_split = np.argmax(feature_importance)
    
    # 构建子树
    left_data = train_data[train_data[:, best_split] < 0]
    left_labels = train_labels[train_data[:, best_split] < 0]
    right_data = train_data[train_data[:, best_split] >= 0]
    right_labels = train_labels[train_data[:, best_split] >= 0]
    
    # 计算叶节点
    left_label = decision_tree(left_data, left_labels, depth+1, max_depth)
    right_label = decision_tree(right_data, right_labels, depth+1, max_depth)
    
    return (best_split, left_label, right_label)

# 示例数据
train_data = np.array([[1, 1], [1, 2], [-1, 0], [-1, -1]])
train_labels = np.array([1, 1, -1, -1])
test_data = np.array([[0, 0]])

# 构建决策树
tree = decision_tree(train_data, train_labels)

# 打印决策树
print(tree)
```
