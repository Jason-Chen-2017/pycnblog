#### 1. 绘制二维坐标系中的直线

**题目：** 编写一个函数，绘制二维坐标系中的直线，给定直线的两个端点坐标。

**答案：** 可以使用 Bresenham 算法绘制二维坐标系中的直线。

```go
package main

import (
    "fmt"
)

func drawLine(x1, y1, x2, y2 int) {
    dx := x2 - x1
    dy := y2 - y1

    if dx == 0 && dy == 0 {
        return
    }

    if abs(dx) > abs(dy) {

        p := 2*abs(dx) - abs(dy)
        for i := 0; i <= steps; i++ {
            printPoint(x1 + (dx * i)/abs(dy), y1 + i*sign(dy))
        }
    }
}

func abs(x int) int {
    if x < 0 {
        return -x
    }
    return x
}

func sign(x int) int {
    if x > 0 {
        return 1
    }
    return -1
}

func printPoint(x, y int) {
    fmt.Printf("(%d, %d)\n", x, y)
}

func main() {
    drawLine(2, 2, 5, 5)
}
```

**解析：** Bresenham 算法是一种高效绘制直线的算法，适用于像素级的绘制。该算法通过计算决策参数，逐步绘制出直线上的像素点。

#### 2. 计算两点之间的距离

**题目：** 编写一个函数，计算二维坐标系中两点之间的距离。

**答案：**

```go
package main

import (
    "fmt"
    "math"
)

func distance(x1, y1, x2, y2 float64) float64 {
    dx := x2 - x1
    dy := y2 - y1
    return math.Sqrt(dx*dx + dy*dy)
}

func main() {
    x1, y1 := 1.0, 2.0
    x2, y2 := 4.0, 6.0
    fmt.Printf("Distance: %f\n", distance(x1, y1, x2, y2))
}
```

**解析：** 该函数使用勾股定理计算两点之间的距离。通过计算两点横坐标和纵坐标的差的平方和，再开方得到距离。
