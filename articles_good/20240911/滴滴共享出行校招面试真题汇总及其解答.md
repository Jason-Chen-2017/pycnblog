#### 1. 最长公共子序列（LCS）

**题目描述：** 给定两个字符串，找出它们的**最长公共子序列**。

**输入：**
```
str1 = "ABCD"
str2 = "ACDF"
```

**输出：**
```
LCS: "ACD"
```

**解答：**

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    result = []
    i, j = m, n
    while i > 0 and j > 0:
        if str1[i - 1] == str2[j - 1]:
            result.append(str1[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            i -= 1
        else:
            j -= 1

    return ''.join(result[::-1])

str1 = "ABCD"
str2 = "ACDF"
print(longest_common_subsequence(str1, str2))  # 输出 "ACD"
```

#### 2. 两数之和

**题目描述：** 给定一个整数数组 `nums` 和一个目标值 `target`，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**输入：**
```
nums = [2, 7, 11, 15]
target = 9
```

**输出：**
```
输出长度为 2 的数组 [0, 1]，其中 nums[0]+nums[1] = 9
```

**解答：**

```python
def two_sum(nums, target):
    hashmap = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hashmap:
            return [hashmap[complement], i]
        hashmap[num] = i

    return []

nums = [2, 7, 11, 15]
target = 9
print(two_sum(nums, target))  # 输出 [0, 1]
```

#### 3. 排序算法

**题目描述：** 实现一个排序算法，如快速排序、归并排序、冒泡排序等。

**输入：**
```
arr = [64, 34, 25, 12, 22, 11, 90]
```

**输出：**
```
排序后的数组：[11, 12, 22, 25, 34, 64, 90]
```

**解答（快速排序）：**

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr

    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]

    return quick_sort(left) + middle + quick_sort(right)

arr = [64, 34, 25, 12, 22, 11, 90]
sorted_arr = quick_sort(arr)
print("排序后的数组：", sorted_arr)
```

#### 4. 负载均衡算法

**题目描述：** 设计一个负载均衡算法，支持以下功能：
- 添加服务器
- 移除服务器
- 获取下一个服务器

**输入：**
```
操作 1：添加服务器 ["add", "192.168.1.1"]
操作 2：移除服务器 ["remove", "192.168.1.1"]
操作 3：获取下一个服务器 ["next"]
```

**输出：**
```
操作 1：添加服务器成功
操作 2：移除服务器成功
操作 3：获取下一个服务器："192.168.1.2"
```

**解答：**

```python
from collections import deque

class LoadBalancer:
    def __init__(self):
        self.servers = deque()

    def add_server(self, server):
        self.servers.append(server)
        print(f"添加服务器成功：{server}")

    def remove_server(self, server):
        self.servers.remove(server)
        print(f"移除服务器成功：{server}")

    def get_next_server(self):
        if not self.servers:
            return None
        server = self.servers.popleft()
        self.servers.append(server)
        return server

# 测试
lb = LoadBalancer()
lb.add_server("192.168.1.1")
lb.add_server("192.168.1.2")
lb.remove_server("192.168.1.1")
print(lb.get_next_server())  # 输出 "192.168.1.2"
```

#### 5. 缓存系统

**题目描述：** 设计一个缓存系统，支持以下功能：
- 添加数据
- 获取数据
- 删除数据
- 缓存容量限制

**输入：**
```
操作 1：添加数据 ["add", "key1", "value1"]
操作 2：获取数据 ["get", "key1"]
操作 3：删除数据 ["delete", "key1"]
操作 4：缓存容量限制 ["limit", 2]
```

**输出：**
```
操作 1：添加数据成功
操作 2：获取数据：value1
操作 3：删除数据成功
操作 4：缓存容量限制成功，当前缓存大小：2
```

**解答：**

```python
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity):
        self.capacity = capacity
        self.cache = OrderedDict()

    def get(self, key):
        if key not in self.cache:
            return -1
        value = self.cache.pop(key)
        self.cache[key] = value
        return value

    def put(self, key, value):
        if key in self.cache:
            self.cache.pop(key)
        elif len(self.cache) >= self.capacity:
            self.cache.popitem(last=False)
        self.cache[key] = value

    def show_cache_size(self):
        return len(self.cache)

# 测试
cache = LRUCache(2)
cache.put("key1", "value1")
print(cache.get("key1"))  # 输出 "value1"
cache.put("key2", "value2")
print(cache.show_cache_size())  # 输出 2
cache.delete("key1")
print(cache.show_cache_size())  # 输出 1
```

#### 6. 反转链表

**题目描述：** 实现一个函数，反转单链表。

**输入：**
```
节点列表：[1, 2, 3, 4, 5]
```

**输出：**
```
反转后的节点列表：[5, 4, 3, 2, 1]
```

**解答：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverse_linked_list(head):
    prev = None
    current = head
    while current:
        next_node = current.next
        current.next = prev
        prev = current
        current = next_node
    return prev

# 创建链表
head = ListNode(1)
head.next = ListNode(2)
head.next.next = ListNode(3)
head.next.next.next = ListNode(4)
head.next.next.next.next = ListNode(5)

# 反转链表
new_head = reverse_linked_list(head)

# 打印反转后的链表
while new_head:
    print(new_head.val, end=" ")
    new_head = new_head.next
```

#### 7. 合并区间

**题目描述：** 给出一个区间的集合，请合并所有重叠的区间。

**输入：**
```
区间列表：[ [1,3], [2,6], [8,10], [15,18] ]
```

**输出：**
```
合并后的区间列表：[ [1,6], [8,10], [15,18] ]
```

**解答：**

```python
def merge_intervals(intervals):
    if not intervals:
        return []

    intervals.sort(key=lambda x: x[0])
    result = [intervals[0]]

    for interval in intervals[1:]:
        last_interval = result[-1]
        if last_interval[1] >= interval[0]:
            last_interval[1] = max(last_interval[1], interval[1])
        else:
            result.append(interval)

    return result

intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]
print(merge_intervals(intervals))  # 输出 [[1, 6], [8, 10], [15, 18]]
```
