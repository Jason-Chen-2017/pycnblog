### 3. 算法编程题

**题目：** 编写一个Python程序，使用K-means算法对一组用户行为数据进行聚类。

```python
from sklearn.cluster import KMeans
import numpy as np

# 用户行为数据
data = np.array([[1, 2], [1, 4], [1, 0],
                 [10, 2], [10, 4], [10, 0]])

# 创建KMeans模型，设定聚类中心数量为2
kmeans = KMeans(n_clusters=2, random_state=0).fit(data)

# 输出聚类结果
print("聚类中心：", kmeans.cluster_centers_)
print("每个样本的聚类标签：", kmeans.labels_)

# 根据聚类标签对数据进行重新分配
reallocated_data = np.zeros((data.shape[0], 2))
for i, label in enumerate(kmeans.labels_):
    reallocated_data[i] = kmeans.cluster_centers_[label]

print("重新分配后的数据：", reallocated_data)
```

**解析：** 该程序使用K-means算法对用户行为数据进行了聚类，并输出了聚类中心、每个样本的聚类标签以及重新分配后的数据。
