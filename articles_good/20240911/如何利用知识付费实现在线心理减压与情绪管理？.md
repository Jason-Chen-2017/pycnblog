2. **个性化课程推荐**

**题目：** 编写一个算法，根据用户画像推荐个性化的心理课程。

**答案：** 个性化课程推荐可以通过以下步骤实现：

```python
# 示例代码：个性化课程推荐
courses = [
    {'name': '压力管理', 'rating': 4.8, 'duration': 60},
    {'name': '情绪调节', 'rating': 4.9, 'duration': 90},
    {'name': '心理疏导', 'rating': 4.7, 'duration': 120},
    {'name': '自我认知', 'rating': 4.5, 'duration': 45}
]

def recommend_courses(user_profile):
    recommended_courses = []
    for course in courses:
        if course['rating'] >= user_profile['activity_level']:
            recommended_courses.append(course)
    return recommended_courses

print(recommend_courses(user_profile))
```

3. **情绪状态检测**

**题目：** 编写一个算法，根据用户输入的文本检测情绪状态。

**答案：** 情绪状态检测可以通过以下步骤实现：

```python
# 示例代码：情绪状态检测
from textblob import TextBlob

def detect_emotion(text):
    analysis = TextBlob(text)
    if analysis.sentiment.polarity > 0:
        return '正面情绪'
    elif analysis.sentiment.polarity < 0:
        return '负面情绪'
    else:
        return '中性情绪'

text = "今天的工作真的很累，感觉心情很差。"
print(detect_emotion(text))
```
