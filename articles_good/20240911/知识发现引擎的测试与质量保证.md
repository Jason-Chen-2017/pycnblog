为了更直观地理解知识发现引擎的相关概念和算法，我们提供以下源代码实例：

1. **计算两个集合的交集和并集**

```python
def intersection(nums1, nums2):
    return list(set(nums1) & set(nums2))

def union(nums1, nums2):
    return list(set(nums1) | set(nums2))
```

这两个函数分别计算两个集合的交集和并集。`intersection` 函数使用集合的交集操作 `&` 来计算交集，而 `union` 函数使用集合的并集操作 `|` 来计算并集。

2. **找出数组中的重复元素**

```python
def find_duplicates(nums):
    return [x for x in set(nums) if nums.count(x) > 1]
```

这个函数使用集合操作和列表推导式来找出数组中的重复元素。首先，将数组转换为集合以去除重复元素，然后通过计数器统计每个元素的个数，最后返回重复元素。

3. **计算两个字符串的相似度**

```python
def similarity(str1, str2):
    return sum(a == b for a, b in zip(str1, str2))
```

这个函数使用 `zip` 函数将两个字符串按索引对齐，然后计算字符串中对应字符的匹配度。通过比较对应字符是否相同来计算相似度。

4. **将字符串转换为驼峰式命名**

```python
def to_camel_case(s):
    return ''.join(word.title() for word in s.split('_'))
```

这个函数将下划线分隔的字符串转换为驼峰式命名。首先，使用 `split('_')` 函数将字符串按下划线分割成单词，然后使用 `word.title()` 函数将每个单词的首字母大写，最后使用 `''.join()` 函数将单词拼接成完整的字符串。
