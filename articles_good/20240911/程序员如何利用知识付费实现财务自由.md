### 算法编程题库

#### 4. 如何实现一个简单的在线课程平台？

**题目：** 请使用 Python 编写一个简单的在线课程平台，支持课程发布、报名、评论等功能。

**答案：** 可以使用 Flask 框架实现一个简单的在线课程平台。

```python
from flask import Flask, request, jsonify

app = Flask(__name__)

courses = [
    {"id": 1, "title": "Python 基础", "instructor": "张三"},
    {"id": 2, "title": "数据分析", "instructor": "李四"},
]

@app.route('/courses', methods=['GET'])
def get_courses():
    return jsonify(courses)

@app.route('/courses', methods=['POST'])
def create_course():
    course_data = request.get_json()
    courses.append(course_data)
    return jsonify(course_data), 201

@app.route('/courses/<int:course_id>', methods=['GET'])
def get_course(course_id):
    course = next((c for c in courses if c['id'] == course_id), None)
    if course:
        return jsonify(course)
    else:
        return jsonify({"error": "课程未找到"}), 404

@app.route('/courses/<int:course_id>', methods=['DELETE'])
def delete_course(course_id):
    global courses
    courses = [course for course in courses if course['id'] != course_id]
    return jsonify({"message": "课程已删除"})

if __name__ == '__main__':
    app.run(debug=True)
```

**解析：** 这是一个简单的在线课程平台实现，包括课程列表查询、课程发布、课程详情查询和课程删除等功能。使用 Flask 框架可以快速搭建，并根据需求进行扩展。

#### 5. 如何实现一个简单的技术博客平台？

**题目：** 请使用 Java 编写一个简单的技术博客平台，支持博客发布、评论、分类等功能。

**答案：** 可以使用 Spring Boot 框架实现一个简单的技术博客平台。

```java
@RestController
@RequestMapping("/posts")
public class PostController {

    @Autowired
    private PostService postService;

    @GetMapping
    public ResponseEntity<List<Post>> listPosts() {
        List<Post> posts = postService.listPosts();
        return ResponseEntity.ok(posts);
    }

    @PostMapping
    public ResponseEntity<Post> createPost(@RequestBody Post post) {
        Post savedPost = postService.savePost(post);
        return ResponseEntity.status(HttpStatus.CREATED).body(savedPost);
    }

    @GetMapping("/{id}")
    public ResponseEntity<Post> getPost(@PathVariable Long id) {
        Post post = postService.findPostById(id);
        if (post != null) {
            return ResponseEntity.ok(post);
        } else {
            return ResponseEntity.notFound().build();
        }
    }

    @DeleteMapping("/{id}")
    public ResponseEntity<Void> deletePost(@PathVariable Long id) {
        boolean isDeleted = postService.deletePostById(id);
        if (isDeleted) {
            return ResponseEntity.noContent().build();
        } else {
            return ResponseEntity.notFound().build();
        }
    }
}
```

**解析：** 这是一个简单的技术博客平台实现，包括博客列表查询、博客发布、博客详情查询和博客删除等功能。使用 Spring Boot 框架可以快速搭建，并根据需求进行扩展。
