**问题 3：如何在决策中使用决策树？**

**面试题：** 请解释决策树在决策过程中的应用，并给出一个简单的决策树示例。

**答案解析：** 
1. **决策树**是一种图形化工具，用于表示决策过程，通过评估不同选择的结果来指导决策。
2. **应用**：决策树可以帮助识别关键因素，评估不同选择的可能结果，从而做出更明智的决策。
3. **示例**：以下是一个简单的决策树示例，用于决定是否购买股票。

![决策树示例](https://example.com/dtg_example.png)

**代码示例：**
```python
# Python 代码示例：创建决策树进行决策
from decisiontree import DecisionTree

# 创建决策树
tree = DecisionTree()

# 添加决策节点
tree.add_decision_node("股票价格", ["high", "low"])
tree.add_decision_node("市场趋势", ["up", "down"])

# 添加结果节点
tree.add_result_node("购买", ["high", "up"])
tree.add_result_node("不购买", ["low", "down"])

# 使用决策树进行决策
decision = tree.decide(["high", "up"])
print("决策结果：", decision)
```
