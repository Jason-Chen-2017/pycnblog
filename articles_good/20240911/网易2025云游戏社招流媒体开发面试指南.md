#### 二、编程题

##### 1. 编写一个函数，实现一个简单的流媒体播放器。

**题目：** 编写一个函数 `play`，接收一个字符串参数 `url`，实现一个简单的流媒体播放器，可以播放视频或音频文件。

**答案：**

```go
package main

import (
	"fmt"
	"net/http"
	"os"
)

func play(url string) {
	resp, err := http.Get(url)
	if err != nil {
		fmt.Println("Error fetching content:", err)
		return
	}
	defer resp.Body.Close()

	file, err := os.Create("output.mp4")
	if err != nil {
		fmt.Println("Error creating file:", err)
		return
	}
	defer file.Close()

	_, err = io.Copy(file, resp.Body)
	if err != nil {
		fmt.Println("Error copying content:", err)
		return
	}

	fmt.Println("File downloaded successfully!")
}
```

##### 2. 编写一个函数，实现一个简单的缓冲器，用于处理流媒体播放过程中的缓冲。

**题目：** 编写一个函数 `buffer`，接收一个整数参数 `size`，实现一个简单的缓冲器，用于处理流媒体播放过程中的缓冲。

**答案：**

```go
package main

import (
	"fmt"
	"sync"
)

type Buffer struct {
	data []byte
	mu   sync.Mutex
}

func (b *Buffer) Write(p []byte) (n int, err error) {
	b.mu.Lock()
	defer b.mu.Unlock()

	n = len(p)
	b.data = append(b.data, p...)

	return
}

func (b *Buffer) Read(p []byte) (n int, err error) {
	b.mu.Lock()
	defer b.mu.Unlock()

	n = len(p)
	if len(b.data) < n {
		n = len(b.data)
		copy(p, b.data)
		b.data = b.data[n:]
	} else {
		copy(p, b.data[:n])
		b.data = b.data[n:]
	}

	return
}

func main() {
	buff := &Buffer{}
	buff.Write([]byte("Hello, World!"))
	buff.Read([]byte("Hello, "))
	fmt.Println(string([]byte("World!")))
}
```
