#### 算法编程题库与解析

### 3. 如何设计一个简单的投票系统？

**题目：** 设计一个简单的投票系统，支持候选人的投票和投票结果的统计。

**答案：** 

```python
# Python 代码示例

# 存储候选人列表
candidates = ["A", "B", "C"]

# 存储投票结果
votes = {"A": 0, "B": 0, "C": 0}

# 投票函数
def vote(candidate):
    if candidate in candidates:
        votes[candidate] += 1
        print(f"投票成功：{candidate}")
    else:
        print("无效投票，请重新选择候选人。")

# 统计投票结果
def show_results():
    for candidate, vote_count in votes.items():
        print(f"{candidate}: {vote_count}票")

# 测试投票
vote("A")
vote("B")
vote("D")  # 无效投票
show_results()
```

**解析：** 该示例代码实现了一个简单的投票系统，包括投票和投票结果的统计功能。

### 4. 如何实现一个简单的文件加密和解密算法？

**题目：** 使用 Python 实现一个简单的文件加密和解密算法，采用凯撒密码。

**答案：**

```python
# Python 代码示例

# 加密函数
def encrypt(plain_text, shift):
    encrypted_text = ""
    for char in plain_text:
        if char.isalpha():
            shifted_char = chr((ord(char) - ord('a') + shift) % 26 + ord('a'))
            encrypted_text += shifted_char
        else:
            encrypted_text += char
    return encrypted_text

# 解密函数
def decrypt(encrypted_text, shift):
    decrypted_text = ""
    for char in encrypted_text:
        if char.isalpha():
            shifted_char = chr((ord(char) - ord('a') - shift) % 26 + ord('a'))
            decrypted_text += shifted_char
        else:
            decrypted_text += char
    return decrypted_text

# 测试加密和解密
shift = 3
original_text = "Hello, World!"
encrypted_text = encrypt(original_text, shift)
print(f"加密后：{encrypted_text}")
decrypted_text = decrypt(encrypted_text, shift)
print(f"解密后：{decrypted_text}")
```

**解析：** 该示例代码实现了一个简单的凯撒密码加密和解密算法，可用于对字符串进行加密和解密。
