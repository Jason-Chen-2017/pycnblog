### 1. 什么是知识图谱？如何构建知识图谱？

**答案：** 知识图谱是一种语义网络，用于表示实体和实体之间的关系。构建知识图谱通常包括数据采集、实体识别、关系抽取、实体关系映射等步骤。

**源代码示例：** 

```python
class Node:
    def __init__(self, name):
        self.name = name
        self.edges = []

class Edge:
    def __init__(self, source, target, relation):
        self.source = source
        self.target = target
        self.relation = relation

# 构建知识图谱
graph = {
    '人': Node('人'),
    '动物': Node('动物'),
    '猫': Node('猫')
}

# 添加关系
graph['人'].edges.append(Edge(graph['人'], graph['动物'], '属于'))
graph['动物'].edges.append(Edge(graph['动物'], graph['猫'], '包含'))

# 打印知识图谱
for node in graph.values():
    print(node.name)
    for edge in node.edges:
        print(edge.relation, edge.source.name, edge.target.name)
```
