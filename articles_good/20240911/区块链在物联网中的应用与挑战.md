**答案：**
区块链在物联网设备身份验证中的应用示例：

1. **设备注册：** 设备在加入物联网网络时，需要通过区块链进行注册，生成唯一的设备身份标识，并将其记录在区块链上。
2. **身份验证：** 当设备进行通信时，会通过区块链验证其身份，确保通信双方都是合法的设备。
3. **设备签名：** 每个设备在发送数据时都会使用私钥进行签名，接收方可以通过公钥验证签名的真实性，确保数据的完整性。

示例代码：

```go
// 设备注册过程
func registerDevice(deviceID string) {
    // 生成设备私钥和公钥
    privKey, err := crypto.GenerateKey()
    if err != nil {
        log.Fatal(err)
    }
    pubKey := privKey.PublicKey

    // 创建设备身份信息
    deviceInfo := DeviceInfo{
        ID:       deviceID,
        PublicKey: pubKey,
    }

    // 将设备信息上传到区块链
    blockchain.AddDevice(deviceInfo)
}

// 设备身份验证
func verifyDevice(deviceID string, signature []byte, data []byte) bool {
    // 从区块链获取设备公钥
    deviceInfo, err := blockchain.GetDevice(deviceID)
    if err != nil {
        return false
    }

    // 验证签名
    return crypto.VerifySignature(deviceInfo.PublicKey, data, signature)
}
```
