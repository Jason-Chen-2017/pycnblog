#### 1. 数据采集与整合

项目管理的基础在于对项目全生命周期数据的采集与整合。大数据技术可以通过各种传感器、日志、用户反馈等多渠道收集项目相关的数据。这些数据包括项目进度、资源分配、人员绩效、风险管理等信息。通过整合这些数据，项目经理可以获得全面的视角，了解项目的实际运行状态。

**典型面试题：** 请简述项目管理中数据采集的常见方法和挑战。

**题目描述：** 给定一个包含项目任务及其开始时间和持续时间的列表，编写一个函数，计算项目的最早完成时间。

**答案解析：**

```python
def calculate_earliest_completion_time(tasks):
    tasks.sort(key=lambda x: x['start_time'])
    completion_time = tasks[0]['start_time'] + tasks[0]['duration']
    
    for i in range(1, len(tasks)):
        if tasks[i]['start_time'] > completion_time:
            completion_time = tasks[i]['start_time'] + tasks[i]['duration']
        else:
            completion_time += tasks[i]['duration']
    
    return completion_time

tasks = [
    {'task_id': 1, 'start_time': 0, 'duration': 2},
    {'task_id': 2, 'start_time': 2, 'duration': 3},
    {'task_id': 3, 'start_time': 5, 'duration': 1},
    {'task_id': 4, 'start_time': 6, 'duration': 2}
]

print(calculate_earliest_completion_time(tasks))  # 输出：10
```

#### 算法编程题 2：资源优化调度

**题目描述：** 给定一个包含项目任务及其开始时间、持续时间和所需资源的列表，编写一个函数，优化资源分配，使得项目完成时间最短。

**答案解析：**

```python
def optimize_resource_allocation(tasks):
    tasks.sort(key=lambda x: x['start_time'])
    allocation = {}
    current_time = tasks[0]['start_time']
    for task in tasks:
        if current_time < task['start_time']:
            current_time = task['start_time']
        if current_time + task['duration'] <= task['required_resources']:
            allocation[task['task_id']] = task['required_resources']
            current_time += task['duration']
        else:
            allocation[task['task_id']] = current_time + task['duration']
            current_time = task['required_resources']
    
    return allocation

tasks = [
    {'task_id': 1, 'start_time': 0, 'duration': 2, 'required_resources': 3},
    {'task_id': 2, 'start_time': 2, 'duration': 3, 'required_resources': 4},
    {'task_id': 3, 'start_time': 5, 'duration': 1, 'required_resources': 2},
    {'task_id': 4, 'start_time': 6, 'duration': 2, 'required_resources': 3}
]

print(optimize_resource_allocation(tasks))
```

def calculate_project_risk(tasks):
    risk_values = [task['probability'] * task['impact'] for task in tasks]
    return sum(risk_values)
