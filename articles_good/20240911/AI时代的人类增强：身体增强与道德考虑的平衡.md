#### 二、算法编程题库

##### 1. 计算体脂率

**题目：** 编写一个函数，根据身高、体重和年龄计算一个人的体脂率。

**答案：** 

```python
def calculate_body_fat_rate(height, weight, age):
    # 使用皮尔·路易斯公式计算体脂率
    # 体脂率 = (1.209 * weight) + (0.414 * height) - (0.452 * age) - 48.3
    body_fat_rate = (1.209 * weight) + (0.414 * height) - (0.452 * age) - 48.3
    return round(body_fat_rate, 2)

# 示例
height = 170  # 身高：170厘米
weight = 60   # 体重：60公斤
age = 30      # 年龄：30岁

# 计算体脂率
body_fat_rate = calculate_body_fat_rate(height, weight, age)
print(f"体脂率：{body_fat_rate}%")
```

**解析：** 本题使用皮尔·路易斯公式计算体脂率，该公式较为常用，适用于一般人群。计算结果需要保留两位小数。

##### 2. 计算最大心率

**题目：** 编写一个函数，根据年龄计算一个人的最大心率。

**答案：**

```python
def calculate_max_heart_rate(age):
    # 使用哈金斯公式计算最大心率
    # 最大心率 = 220 - 年龄
    max_heart_rate = 220 - age
    return round(max_heart_rate)

# 示例
age = 30  # 年龄：30岁

# 计算最大心率
max_heart_rate = calculate_max_heart_rate(age)
print(f"最大心率：{max_heart_rate}次/分钟")
```

**解析：** 本题使用哈金斯公式计算最大心率，该公式较为简单，适用于一般人群。计算结果需要保留整数。

##### 3. 运动监测与训练

**题目：** 编写一个函数，根据用户输入的运动数据（如步数、距离、时长），计算用户的运动强度，并给出训练建议。

**答案：**

```python
def calculate_training_intensity(steps, distance, duration):
    # 计算平均每分钟步数和平均每分钟距离
    avg_steps_per_minute = steps / duration
    avg_distance_per_minute = distance / duration
    
    # 根据平均每分钟步数和平均每分钟距离，判断运动强度
    if avg_steps_per_minute > 100 and avg_distance_per_minute > 1:
        intensity = "高强度"
        training_advice = "建议进行有氧运动，如慢跑、游泳等，以增强心肺功能。"
    elif avg_steps_per_minute > 70 and avg_distance_per_minute > 0.5:
        intensity = "中等强度"
        training_advice = "建议进行中等强度的有氧运动，如快走、骑自行车等，以保持身体健康。"
    else:
        intensity = "低强度"
        training_advice = "建议进行轻度运动，如散步、做瑜伽等，以放松身心。"
    
    return intensity, training_advice

# 示例
steps = 1200  # 步数：1200步
distance = 5  # 距离：5公里
duration = 30  # 时长：30分钟

# 计算运动强度和训练建议
intensity, training_advice = calculate_training_intensity(steps, distance, duration)
print(f"运动强度：{intensity}")
print(f"训练建议：{training_advice}")
```

**解析：** 本题根据用户输入的运动数据，计算平均每分钟步数和平均每分钟距离，根据这两个指标判断运动强度，并给出相应的训练建议。
