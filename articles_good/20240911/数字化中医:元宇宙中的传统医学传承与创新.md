**4. 编写一个算法，根据中医的五行学说（金、木、水、火、土）判断两种草药是否相生相克。**

**答案：** 五行相生相克的关系如下：

- 相生：木生火，火生土，土生金，金生水，水生木
- 相克：木克土，土克水，水克火，火克金，金克木

```python
def is_compatible(herb1, herb2):
    herbs = {
        '木': ['火', '金'],
        '火': ['土', '水'],
        '土': ['金', '火'],
        '金': ['木', '水'],
        '水': ['火', '土']
    }
    return herb2 in herbs[herb1] or herb1 in herbs[herb2]

# 测试
print(is_compatible('木', '火'))  # True，木生火
print(is_compatible('金', '水'))  # True，金生水
print(is_compatible('火', '土'))  # True，火生土
print(is_compatible('水', '木'))  # False，水克木
print(is_compatible('金', '火'))  # False，金克火
```

**5. 编写一个算法，根据中医脉象分析结果预测病人的健康状况。**

```python
def predict_health(pulse_type, pulse_strength):
    health_status = {
        ('浮', '强'): '健康',
        ('浮', '弱'): '气虚',
        ('沉', '强'): '实证',
        ('沉', '弱'): '虚证',

        ('滑', '弱'): '气血两虚',
        ('涩', '强'): '血瘀',
        ('涩', '弱'): '血虚'
    }
    return health_status.get((pulse_type, pulse_strength), '未知')

# 测试

```

**6. 编写一个算法，计算一组草药的总体功效，并根据功效对草药进行排序。**

**答案：** 草药的功效通常用数字表示，数字越大，表示功效越强。以下是一个简单的算法示例：

```python
def calculate_total_efficacy(herbs):
    efficacy_scores = {'人参': 10, '黄芪': 8, '白术': 6, '甘草': 5, '黄连': 9}
    total_efficacy = sum(efficacy_scores[herb] for herb in herbs if herb in efficacy_scores)
    return total_efficacy

def sort_herbs_by_efficacy(herbs):
    efficacy_scores = {'人参': 10, '黄芪': 8, '白术': 6, '甘草': 5, '黄连': 9}
    return sorted(herbs, key=lambda x: efficacy_scores.get(x, 0), reverse=True)

# 测试
herbs = ['人参', '白术', '黄连', '黄芪', '甘草']
total_efficacy = calculate_total_efficacy(herbs)
sorted_herbs = sort_herbs_by_efficacy(herbs)
print("Total Efficacy:", total_efficacy)
print("Sorted Herbs:", sorted_herbs)
```

```python
    elif pulse_strength == "弱":
        strength_score = 0.5
```
