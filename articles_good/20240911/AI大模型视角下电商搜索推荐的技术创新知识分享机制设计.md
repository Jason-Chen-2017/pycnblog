### 1. 电商搜索推荐的基石：向量表示与相似性度量

**题目：** 请解释在电商搜索推荐系统中，如何使用向量表示商品和用户，并如何计算它们之间的相似度？

**答案：** 在电商搜索推荐系统中，商品和用户通常使用向量表示。常用的向量表示方法包括：

- **基于属性的表示：** 将商品和用户的属性（如分类、标签、用户偏好等）转换为向量。
- **基于内容表示：** 利用自然语言处理技术（如Word2Vec、BERT等）将商品和用户的描述转换为向量。

相似度计算方法主要包括：

- **余弦相似度：** 用于衡量两个向量方向上的相似程度。
- **欧氏距离：** 用于衡量两个向量之间的距离，越近表示相似度越高。

**代码示例：**

```python
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

# 假设商品和用户的向量表示
user_vector = np.array([0.1, 0.2, 0.3])
item_vector = np.array([0.4, 0.5, 0.6])

# 计算余弦相似度
similarity = cosine_similarity([user_vector], [item_vector])[0][0]
print("Similarity:", similarity)
```
