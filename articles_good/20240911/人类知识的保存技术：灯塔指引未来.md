### 二、算法编程题库

#### 1. 给定一个字符串，找出其中不重复的最长子串。

**答案：** 可以使用滑动窗口的方法实现。

```python
def longest_substring(s):
    left, right = 0, 0
    max_len, start = 0, 0
    while right < len(s):
        if s[right] not in s[left:right]:
            right += 1
            if right - left > max_len:
                max_len = right - left
                start = left
        else:
            left = right
        right += 1
    return s[start:start+max_len]
```

**解析：** 该算法通过维护一个滑动窗口，逐个移动右边界，检查窗口内的字符串是否重复。当发现重复字符时，将左边界移动到重复字符的下一个位置。

#### 2. 设计一个有序链表，支持在链表中间插入节点。

**答案：** 可以使用双向链表实现。

```java
class Node {
    int val;
    Node prev;
    Node next;
    Node(int val) {
        this.val = val;
    }
}

public class SortedLinkedList {
    private Node head;
    private Node tail;
    
    public void insert(int val) {
        Node newNode = new Node(val);
        if (head == null) {
            head = newNode;
            tail = newNode;
        } else if (val <= head.val) {
            newNode.next = head;
            head.prev = newNode;
            head = newNode;
        } else if (val >= tail.val) {
            newNode.prev = tail;
            tail.next = newNode;
            tail = newNode;
        } else {
            Node current = head.next;
            while (current != null && current.val < val) {
                current = current.next;
            }
            newNode.prev = current.prev;
            newNode.next = current;
            current.prev.next = newNode;
            current.prev = newNode;
        }
    }
}
```

**解析：** 该算法通过遍历链表，找到合适的位置插入新节点，确保链表保持有序。
