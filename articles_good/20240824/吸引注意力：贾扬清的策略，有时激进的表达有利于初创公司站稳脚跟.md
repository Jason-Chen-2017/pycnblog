### 5.2 源代码详细实现

以下是一个简单的广告发布项目的代码实现：

```python
from flask import Flask, request, jsonify
import requests

app = Flask(__name__)

# 广告内容列表
ad_contents = [
    {
        "id": 1,
        "title": "颠覆性技术，引领未来",
        "content": "我们研发出了一种具有颠覆性的技术，将彻底改变你的生活。",
        "image_url": "https://example.com/image1.jpg"
    },
    {
        "id": 2,
        "title": "无与伦比的性能",
        "content": "我们的产品在性能上无与伦比，是你最佳的解决方案。",
        "image_url": "https://example.com/image2.jpg"
    }
]

@app.route('/advertisements', methods=['GET'])
def get_advertisements():
    return jsonify(ad_contents)

@app.route('/advertisements/<int:ad_id>', methods=['GET'])
def get_advertisement(ad_id):
    ad = next((ad for ad in ad_contents if ad['id'] == ad_id), None)
    if ad:
        return jsonify(ad)
    else:
        return jsonify({"error": "广告不存在"}), 404

if __name__ == '__main__':
    app.run(debug=True)
```

- **单个广告内容获取**：通过 `/advertisements/<int:ad_id>` 接口获取指定 ID 的广告内容。

这个项目演示了如何使用 Flask 框架快速搭建一个广告发布系统，并通过 API 接口提供广告内容。
