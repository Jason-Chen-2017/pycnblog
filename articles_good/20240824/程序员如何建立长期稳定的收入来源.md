```python
import numpy as np
import matplotlib.pyplot as plt

def linear_regression(x, y):
    x_mean = np.mean(x)
    y_mean = np.mean(y)
    num = 0
    den = 0
    for i in range(len(x)):
        num += (x[i] - x_mean) * (y[i] - y_mean)
        den += (x[i] - x_mean) ** 2
    beta_1 = num / den
    beta_0 = y_mean - beta_1 * x_mean
    return beta_0, beta_1

x = np.array([1, 2, 3, 4])
y = np.array([2, 4, 6, 8])
beta_0, beta_1 = linear_regression(x, y)
print("拟合直线：y = {} + {} \cdot x".format(beta_0, beta_1))

x_fit = np.linspace(0, 5, 100)
y_fit = beta_0 + beta_1 * x_fit
plt.scatter(x, y)
plt.plot(x_fit, y_fit)
plt.show()
```

#### 10. 附录：常见问题与解答
##### 10.1 如何选择适合自己的学习资源？
- 了解自己的需求：明确自己需要学习的内容，选择相应的教材和课程。
- 考虑难度：选择适合自己水平的教材和课程，避免难度过高或过低。
- 互动性：选择具有互动性的教材和课程，如在线课程、编程社区等。

##### 10.2 如何建立有效的职业规划？
- 设定目标：明确自己的职业目标，如晋升、创业等。
- 制定计划：根据目标，制定具体的计划，包括学习、工作等方面。
- 定期评估：定期评估职业规划的执行情况，根据实际情况进行调整。
