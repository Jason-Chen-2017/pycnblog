### 5.2 源代码详细实现

下面是道德积分交易系统的核心代码实现：

```python
from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy
from flask_migrate import Migrate

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///integral.db'
db = SQLAlchemy(app)
migrate = Migrate(app, db)

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    integral = db.Column(db.Integer, default=0)

@app.route('/add_integral', methods=['POST'])
def add_integral():
    username = request.form['username']
    integral = int(request.form['integral'])
    user = User.query.filter_by(username=username).first()
    if user:
        user.integral += integral
        db.session.commit()
        return jsonify({'status': 'success', 'message': 'Integral added successfully.'})
    else:
        return jsonify({'status': 'error', 'message': 'User not found.'})

@app.route('/get_integral', methods=['GET'])
def get_integral():
    username = request.args.get('username')
    user = User.query.filter_by(username=username).first()
    if user:
        return jsonify({'status': 'success', 'integral': user.integral})
    else:
        return jsonify({'status': 'error', 'message': 'User not found.'})

if __name__ == '__main__':
    db.create_all()
    app.run(debug=True)
```

- **积分增加接口**：`add_integral`函数用于增加用户的道德积分，接收用户名和积分参数，通过SQLAlchemy查询用户信息，更新积分，并返回处理结果。
- **积分查询接口**：`get_integral`函数用于查询用户的道德积分，接收用户名参数，通过SQLAlchemy查询用户信息，并返回积分。
