## 1. 背景介绍

金融支付系统是指一系列用于处理金融交易的系统和网络，包括支付卡系统、电子支付系统、移动支付系统等。这些系统为人们提供了方便快捷的支付方式，使得金融交易变得更加便捷。随着科技的发展，金融支付系统也不断发展和演进，不断拓展到更多领域。

## 2. 核心概念与联系

### 2.1 支付卡系统

支付卡系统是一种以支付卡为中心的支付方式，包括借记卡、贷记卡、预付卡等。支付卡通常使用磁带技术或RFID技术，可以在POS终端进行支付。支付卡系统的核心概念包括：

- 卡片管理系统（CMS）：负责管理支付卡的生命周期，包括卡片激活、注销、更新等。
- 交易管理系统（TMS）：负责处理支付卡交易，包括授权、清算、撤销等。
- 风险管理系统（RMS）：负责对支付卡交易进行风险控制，包括欺诈检测、异常处理等。

### 2.2 电子支付系统

电子支付系统是一种以电子形式进行支付的方式，包括网上支付、手机支付、QR代码支付等。电子支付系统的核心概念包括：

- 支付网关：负责接收来自商户的支付请求，并将请求转发给支付平台。
- 支付平台：负责处理支付请求，并将支付结果返回给支付网关。
- 支付渠道：负责将支付结果传递给银行卡或支付账户。

### 2.3 移动支付系统

移动支付系统是一种通过手机进行支付的方式，包括NFC支付、移动支付应用（如微信支付、支付宝支付等）等。移动支付系统的核心概念包括：

- 移动支付应用：是一种特殊的电子支付系统，通过手机应用程序提供支付服务。
- NFC技术：是一种无线通信技术，可以在短距离内进行数据传输，适用于移动支付。

import hashlib

def encrypt(data, key):
    cipher = AES.new(key, AES.MODE_ECB)
    ciphertext = cipher.encrypt(data)
    return ciphertext

def decrypt(ciphertext, key):
    cipher = AES.new(key, AES.MODE_ECB)
    plaintext = cipher.decrypt(ciphertext)
    return plaintext

data = "支付数据"
key = "密钥"
