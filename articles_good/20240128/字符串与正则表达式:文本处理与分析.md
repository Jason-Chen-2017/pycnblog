### 4.1 使用Python的re模块实现正则表达式

Python的re模块提供了一种简单的方法来实现正则表达式的匹配、搜索和替换功能。以下是一个简单的例子：

```python
import re

# 匹配字符串中的数字
pattern = r'\d+'
text = '123456'
match = re.match(pattern, text)
if match:
    print('匹配成功')
else:
    print('匹配失败')

# 搜索字符串中的数字
pattern = r'\d+'
text = '1234567890'
search = re.search(pattern, text)
if search:
    print('搜索成功')
else:
    print('搜索失败')

# 替换字符串中的数字
pattern = r'\d+'
text = '1234567890'
replaced_text = re.sub(pattern, '*', text)
print(replaced_text)
```
