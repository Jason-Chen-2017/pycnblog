# 2.核心概念与联系

逻辑回归与支持向量机的核心概念在于它们如何处理数据以及它们之间的联系。

## 2.1 逻辑回归

逻辑回归是一种基于概率模型的二分类算法，它假设输入变量的组合线性相关于输出变量。逻辑回归通过最小化损失函数来学习参数，从而使预测值与实际值之间的差距最小化。

逻辑回归的核心概念包括：

- 概率模型：逻辑回归假设输入变量的组合线性相关于输出变量，通过概率模型来表示输出变量的概率分布。
- 损失函数：逻辑回归通过最小化损失函数来学习参数，损失函数通常是交叉熵损失函数。
- 正则化：为了防止过拟合，逻辑回归通常使用正则化技术，如L1正则化和L2正则化。

## 2.2 支持向量机

支持向量机是一种基于最大边界值分类器的二分类算法，它通过最大化边界值和类别间距离来学习参数。支持向量机可以处理高维数据和不均衡数据，并且具有较好的泛化能力。

支持向量机的核心概念包括：

- 最大边界值分类器：支持向量机通过最大化边界值和类别间距来学习参数，从而实现对数据的最大分类。
- 核函数：支持向量机可以通过核函数处理高维数据，如径向基函数、多项式基函数等。
- 松弛变量：为了处理不均衡数据，支持向量机通过松弛变量来处理误分类问题。

## 2.3 逻辑回归与支持向量机的联系

逻辑回归和支持向量机在处理二分类问题上有一定的联系。它们都通过学习参数来实现对数据的分类，并且都可以处理高维数据和不均衡数据。不过，逻辑回归通过概率模型和损失函数来学习参数，而支持向量机通过最大边界值分类器和核函数来学习参数。

### 3.1.3 数学模型公式详细讲解

逻辑回归模型的数学模型可以表示为：

$$
P(y=1|x;w) = \frac{1}{1+e^{-(w_0 + w^Tx)}}
$$

其中，$w$ 是权重向量，$w_0$ 是偏置项，$x$ 是输入特征向量，$y$ 是输出类别（1 或 0）。

逻辑回归的损失函数通常使用交叉熵损失函数：

$$
L(y, \hat{y}) = -\frac{1}{N} \sum_{i=1}^{N} [y_i \log(\hat{y}_i) + (1 - y_i) \log(1 - \hat{y}_i)]
$$

其中，$N$ 是数据集的大小，$y_i$ 是真实输出，$\hat{y}_i$ 是预测输出。

通过梯度下降算法，我们可以得到逻辑回归模型的参数更新规则：

$$
w_{new} = w_{old} - \eta \nabla L(y, \hat{y})
$$

其中，$\eta$ 是学习率，$\nabla L(y, \hat{y})$ 是损失函数的梯度。

```python

    y_pred = 1 / (1 + np.exp(-(w_bias + np.dot(X, w))))
    loss = logistic_loss(y, y_pred)
    
    # 计算梯度

    # 参数更新
    w -= learning_rate * dw
    w_bias -= learning_rate * db
    
    # 输出损失函数值
```
