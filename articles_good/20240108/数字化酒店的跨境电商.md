## 4.1 预订系统

以下是一个简单的预订系统代码实例：

```python
import random

class Room:
    def __init__(self, room_id, status):
        self.room_id = room_id
        self.status = status

class Order:
    def __init__(self, order_id, room_id, price, status):
        self.order_id = order_id
        self.room_id = room_id
        self.price = price
        self.status = status

def check_room_available(rooms, room_id):
    for room in rooms:
        if room.room_id == room_id and room.status == "available":
            return True
    return False

def place_order(orders, rooms, room_id, price):
    if not check_room_available(rooms, room_id):
        return None
    order = Order(len(orders) + 1, room_id, price, "pending")
    orders.append(order)
    room = Room(room_id, "booked")
    rooms.append(room)
    return order

def process_payment(orders, order_id, price):
    for order in orders:
        if order.order_id == order_id:
            order.status = "paid"
            return order
    return None

rooms = [Room(i, "available") for i in range(1, 11)]
orders = []
order_id = 1
price = 100
while order_id <= 5:
    order = place_order(orders, rooms, order_id, price)
    if order:
        process_payment(orders, order.order_id, price)
    order_id += 1
```

在这个代码实例中，我们定义了两个类 `Room` 和 `Order`，分别表示酒店房源和预订信息。我们定义了三个函数 `check_room_available`、`place_order` 和 `process_payment`，分别表示检查房源可用性、提交预订请求和处理支付请求。

在主程序中，我们创建了一些房源对象并将它们存储在一个列表中。然后我们通过一个循环来模拟预订和支付过程。在每次循环中，我们首先调用 `place_order` 函数来提交预订请求，然后调用 `process_payment` 函数来处理支付请求。
