### 4.1 语义理解代码实例和详细解释说明

在这个例子中，我们将使用Python编程语言来实现一个简单的语义理解系统。我们将使用NLTK库来进行词汇解析和语义角色标注，以及spaCy库来进行关系抽取。

首先，我们需要安装NLTK和spaCy库：

```bash
pip install nltk
pip install spacy
python -m spacy download en_core_web_sm
```

然后，我们可以使用以下代码来实现语义理解系统：

```python
import nltk
import spacy
from nltk import pos_tag
from nltk import chunk

# 加载spaCy模型
nlp = spacy.load("en_core_web_sm")

# 文本示例
text = "John gave Mary a book."

# 使用spaCy进行关系抽取
doc = nlp(text)
for token in doc:
    print(token.text, token.lemma_, token.pos_, token.dep_, token.head.text)

# 使用NLTK进行词汇解析和语义角色标注
tokens = nltk.word_tokenize(text)
pos_tags = nltk.pos_tag(tokens)
chunks = chunk.ne_chunk(pos_tags)
print(chunks)
```

这个代码实例首先加载spaCy模型，然后使用spaCy进行关系抽取。接着，使用NLTK进行词汇解析和语义角色标注。最后，将结果打印出来。
