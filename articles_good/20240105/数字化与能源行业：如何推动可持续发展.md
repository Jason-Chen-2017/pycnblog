## 4.4 物联网

### 4.4.1 代码实例

```python
import paho.mqtt.client as mqtt

# 连接服务器
client = mqtt.Client()
client.connect("broker.hivemq.com", 1883, 60)

# 订阅主题
client.subscribe("energy/data")

# 消费者回调函数
def on_message(client, userdata, message):
    print(message.topic + " " + str(message.payload))

# 设置消费者回调函数
client.on_message = on_message

# 循环运行
client.loop_forever()
```

### 4.4.2 解释说明

这个代码实例使用Python编程语言和paho.mqtt.client库实现了物联网。首先，我们导入了mqtt库，然后创建了一个MQTT客户端。接着，我们连接了服务器。接下来，我们订阅了一个主题。最后，我们设置了一个消费者回调函数，用于处理接收到的消息。然后，我们启动了客户端循环运行。
