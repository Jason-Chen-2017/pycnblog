## 3.3 数学模型公式详细讲解
智能运维的数学模型公式主要包括：

1. 设备利用率公式：$$ UtilizationRate = \frac{ActualUsage}{PotentialUsage} $$
2. 资源利用率公式：$$ ResourceUtilization = \frac{ActualResourceUsage}{TotalResource} $$
3. 成本效益公式：$$ CostBenefit = \frac{Benefit}{Cost} $$

其中，$UtilizationRate$ 是设备利用率，$ActualUsage$ 是实际使用量，$PotentialUsage$ 是潜在使用量；$ResourceUtilization$ 是资源利用率，$ActualResourceUsage$ 是实际资源使用量，$TotalResource$ 是总资源量；$CostBenefit$ 是成本效益，$Benefit$ 是收益，$Cost$ 是成本。

import json

url = 'http://api.example.com/device/data'
headers = {'Content-Type': 'application/json'}
data = {'deviceId': '12345'}

response = requests.get(url, headers=headers, data=json.dumps(data))
if response.status_code == 200:
    device_data = response.json()
