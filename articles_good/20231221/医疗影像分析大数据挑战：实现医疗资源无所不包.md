### 4.2 机器学习算法示例

在这个示例中，我们将使用 Python 和 scikit-learn 库来实现一个简单的机器学习算法，即支持向量机算法。支持向量机算法是一种常用的机器学习算法，它可以用于解决二分类、多分类、回归等问题。

```python
from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# 加载医疗影像数据
data = datasets.load_breast_cancer()

# 设置训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(data.data, data.target, test_size=0.2, random_state=42)

# 对医疗影像数据进行标准化处理

X_train = scaler.fit_transform(X_train)
X_test = scaler.transform(X_test)

# 对医疗影像数据进行支持向量机模型训练
svm = SVC(kernel='linear', C=1)
svm.fit(X_train, y_train)

# 对支持向量机模型进行测试
y_pred = svm.predict(X_test)

# 评估支持向量机模型的预测效果
accuracy = accuracy_score(y_test, y_pred)
```

### 4.3 深度学习算法示例

在这个示例中，我们将使用 Python 和 TensorFlow 库来实现一个简单的深度学习算法，即卷积神经网络算法。卷积神经网络算法是一种常用的深度学习算法，它可以用于解决图像分类、回归、检测等问题。

```python
import tensorflow as tf
from tensorflow.keras.datasets import mnist
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense
from tensorflow.keras.utils import to_categorical

# 加载医疗影像数据
(X_train, y_train), (X_test, y_test) = mnist.load_data()

# 设置参数
input_shape = (28, 28, 1)
num_classes = 10
batch_size = 128
epochs = 10

# 预处理医疗影像数据
X_train = X_train.reshape(X_train.shape[0], *input_shape).astype('float32') / 255
X_test = X_test.reshape(X_test.shape[0], *input_shape).astype('float32') / 255
y_train = to_categorical(y_train, num_classes)
y_test = to_categorical(y_test, num_classes)

# 构建卷积神经网络模型
model = Sequential([
    Conv2D(32, (3, 3), activation='relu', input_shape=input_shape),
    MaxPooling2D((2, 2)),
    Conv2D(64, (3, 3), activation='relu'),
    MaxPooling2D((2, 2)),
    Flatten(),
    Dense(128, activation='relu'),
    Dense(num_classes, activation='softmax')
])

# 对卷积神经网络模型进行编译
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# 对卷积神经网络模型进行训练
model.fit(X_train, y_train, batch_size=batch_size, epochs=epochs, validation_data=(X_test, y_test))

# 对卷积神经网络模型进行测试
test_loss, test_acc = model.evaluate(X_test, y_test)
print(f'Test accuracy: {test_acc:.4f}')
```
