## 2.核心概念与联系

区块链是一种分布式、去中心化的数据存储和交易方式，其核心概念包括：

1. 区块：区块是区块链的基本单元，它包含一组交易数据和一个指向下一个区块的指针。每个区块都有一个时间戳，表示该区块在网络中的创建时间。

2. 链：区块链由多个区块组成，这些区块之间通过指针相互连接。这种结构使得数据在整个网络中都可以被同步和验证，从而实现了数据的安全性、可靠性和透明度。

```python
import hashlib
import time

class Block:
    def __init__(self, index, previous_hash, timestamp, data, hash):
        self.index = index
        self.previous_hash = previous_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

    def calc_hash(self):
        sha = hashlib.sha256()
        sha.update(str(self.index) + str(self.previous_hash) + str(self.timestamp) + str(self.data))
        return sha.hexdigest()

def create_genesis_block():
    return Block(0, "0", time.time(), "Genesis Block", "0")

def create_block(previous_block, data):
    index = previous_block.index + 1
    timestamp = time.time()
    hash = previous_block.calc_hash()
    return Block(index, hash, timestamp, data, hash)
```
