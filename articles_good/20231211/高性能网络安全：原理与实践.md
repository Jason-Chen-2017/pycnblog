    ciphertext = cipher.encrypt(data)
    return ciphertext

def rsa_decrypt(ciphertext, private_key):
    cipher = PKCS1_OAEP.new(private_key)
    plaintext = cipher.decrypt(ciphertext)

2. 网络安全：网络安全是网络安全的一个重要组成部分，它包括网络设备安全、网络通信安全和网络应用安全等方面。

## 6.2 网络安全常见问题与解决方案

1. 密码强度不足：密码强度不足是网络安全中的一个常见问题，可以通过设置复杂密码和定期更新密码来解决。
2. 未授权访问：未授权访问是网络安全中的一个常见问题，可以通过实施访问控制和身份验证来解决。
3. 网络漏洞：网络漏洞是网络安全中的一个常见问题，可以通过定期进行漏洞扫描和修复来解决。
