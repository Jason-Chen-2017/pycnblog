### 5.2 源代码详细实现

以下以Java实现Wi-Fi通信为例，展示源代码：

```java
import java.io.InputStream;
import java.io.OutputStream;
import java.net.Socket;

public class Wi-FiCommunication {
    private Socket socket;

    public Wi-FiCommunication(String ip, int port) throws Exception {
        socket = new Socket(ip, port);
    }

    public void sendData(String data) throws Exception {
        OutputStream output = socket.getOutputStream();
        output.write(data.getBytes());
    }

    public String receiveData() throws Exception {
        InputStream input = socket.getInputStream();
        StringBuilder buffer = new StringBuilder();
        byte[] bufferBytes = new byte[1024];
        int length;
        while ((length = input.read(bufferBytes)) != -1) {
            buffer.append(new String(bufferBytes, 0, length));
        }
        return buffer.toString();
    }

    public void close() throws Exception {
        socket.close();
    }
}
```

### 5.3 代码解读与分析

以上代码展示了Java实现Wi-Fi通信的基本步骤：

1. 创建`Wi-FiCommunication`对象，指定目标设备的IP地址和端口号。
2. 使用`sendData`方法发送数据。
3. 使用`receiveData`方法接收数据。
4. 使用`close`方法关闭socket连接。
