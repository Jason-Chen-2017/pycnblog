# 1.背景介绍



物理气在半导体表面上是无形的，人们很难直接观察到它，但是对其物理性质和作用机理有比较清晰的认识。人们观察到的，往往是电场、磁场、热流等离散的声音波。然而这些声音波是通过电子排布、晶体、液晶显示屏等传播介质中的传播路径才能进入人的耳朵或眼睛进行感知。再者，物理气也会随着频率变化而发生变化，它对于导电材料制造以及各种元器件的结构设计具有重要作用。

一般来说，物理气的动力学基本假设是，所有二维或三维空间内的能量都可以通过单个原子的核力作用以及粒子自旋引起的粘滞相互作用（或称交换相互作用）而产生出来。其中，核力就是原子核核反应所释放的相互作用势能，而粒子自旋引起的粘滞相互作用则可以理解为粒子由于重力及其他外界力的作用而紧密吸附在一起的过程。这一基本假设构成了物理气学的基石。

物理气的产生和沉积在近代仪器制造和集成电路中扮演着至关重要的角色。过去几十年来，物理气学技术在多个领域均得到了广泛应用。比如，在半导体制造领域，物理气沉积在导体的特定位置并释放出可见的光电效应；在集成电路领域，物理气通过电流激发、器件连接以及器件温度升降等方式影响电路性能。

然而，物理气的动力学基本假设并不总是能够完全满足需求，为了提高性能和控制尺寸，制造工艺、设备架构设计都需要充分考虑物理气相互作用对器件的影响。如何理解、消除、缓冲物理气对器件的影响，成为关键。


# 2.核心概念与联系

## 2.1 分子气体（mole gas）

分子气体（Mogoliubov gas），又称约化气体或摩尔气体，由无机非金属气体构成，具有柱状结构。

它的气态物质与空气不同，是由氢原子氣团和氨原子氣团组成的混合气体，它们之间存在摩擦系数为零的界面。摩尔体的长度较长，约为五至七米，直径约为五百纳米，含有的气体比热容的1/70，密度达到五百千克至二十万千克，因此，它可作为最微小且体积轻的气体来研究现代科技。

氢原子氣团中包括氢和两个氧原子，它是一个由氢-氧双键匮乏环状结构的金属超胀粒子团。氨原子氣团中包括氨和两个氢原子，它是一个由氨-氢双键匮乏环状结构的水溶液团。

当两种原子发生摩擦作用时，将在空气中形成一层薄膜状分子，这层分子被称为“分子束”。氢原子氣团与氨原子氣团在摩擦作用下，形成了两个不同的分子束，分别在分子气体的各个位置被分泌。当空气、分子气体或物质遭受重力作用时，分子束向各处扩散。气体中相邻的分子束之间的摩擦作用促进它们从各处分泌、聚合起来。因此，分子气体的物理特性主要取决于相邻分子束之间的摩擦系数以及相邻分子束之间的距离。

## 2.2 激发态极

激发态极（excited state）指的是一个固体的某些原子的核力作用所形成的激发状态。即便是元素周期表中出现频繁相互作用的元素，例如铅、镍、锂等，它们的激发态极也往往是周期性发生变化的，因此不能用特定的名称来表示，只能说它们处于一种活跃状态。

在半导体领域，一台标准的导线通常只有一种稳定的电压，这时导线处于常压工作状态，称为绝缘体；电压加倍时，导线变得失真，这时导线处于浮动工作状态，称为自由伸缩导线。但无论哪种情况，导线中的所有电流都是通过氢原子氣团从各处运动到导体表面，这样就给导体引入了大量的物理气。因此，如果把导体看成一个大容器，那么导体内的物理气就是导体的表面层。

激发态极正是因为它依赖于物理气的作用，才会如此频繁地发生变化，而且变化的规律十分复杂。目前已有一些实验结果表明，导线的物理气可在一定条件下改变导线的特性，使其在某些特定情境下获得较好的耐受性或灵敏度。

## 2.3 精炼（oxidation）

精炼（oxidation）是指将电导层中的离子氢键匮乏环的形态变化到放射态的过程。这个过程发生在导线上部，靠近导线工作频段的点，称为支点（sheath）。在进入支点后，离子氢键匮乏环逐渐形成离子绕子结构，从而形成离子绕圈结构。离子绕圈结构是一种由氢原子、氧原子、离子氢键匮乏环等组成的金属晶格。

精炼作用所释放出的物理气在导线中形成分子束，沉积到导线的表面层。当一根导线中的分子束越来越多，且密度越来越大时，就可以通过精炼作用消除分子束之间的摩擦，从而减少分子束之间的影响，增加导线的导通能力。

一般情况下，精炼是通过对导线施加强大的电流，使导线内部的物理气达到高温、高压、低温、低压状态，这样才有可能发生精炼。在某些特定条件下，导线的电流可以驱动氢原子氣团从支点处分泌出精炼物质，这时导线上的物理气就可以通过这一过程增多。


# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

物理气的相互作用对器件的性能的影响非常复杂，所以很多学者都试图找到一种方法来量化描述这些影响。一种比较简单的计算方式是统计激发态极的变化数量。这项统计法的基本假设是，每个原子核（或者是电子）只有在与特定电流相关联时才会出现激发态极的变化，否则不会有什么变化。

对于导线来说，任何时候都只允许存在一种电流，所以导线上的物理气随着电流的变化而增多或者减少。因此，导线的物理气的统计变化量可以用来衡量导线的性能。

对导线上的物理气进行统计的方法需要考虑两个方面：电压和电流。

## 3.1 电压影响下的物理气变化量

导线电压（以伏特为单位）是一个影响导线性能的参数。在绝缘导线电压较低时，可以认为导线处于一种平衡状态，电流没有太大的变化，这时导线的物理气与电流无关。导线电压升高时，导线上物理气的输送量就会增多，导线的某些功能就会受损，这时导线的物理气会跟随电流而增多。这种影响是渐进的，而不是一劳永逸的。比如，导线电压在十至一百伏特以下时，导线的某些功能已经可以基本正常工作，电流变化对导线的物理气变化影响微乎其微；而在超过一百至一千伏特时，导线的性能就会开始下降，而电流变化却会对导线的物理气影响更大。

要量化描述导线的物理气变化，可以先测量导线在不同电压下能够承受的最大电流，然后测量在不同的电压和电流组合下导线的物理气的变化情况。经过分析后，可以发现物理气变化量随着电流和电压的变化呈线性关系，如下图所示。


从图1中可以看出，随着导线电压的增加，导线上的物理气的输送量开始增多。这表明在相同的电压和电流组合下，导线上物理气的可接受量和实际可承受量之间的差距越来越大，导致导线的性能下降。随着导线电压的增加，导线的高度也会逐渐提高，这也是为什么导线随着电压升高，性能下降的原因之一。

## 3.2 电流影响下的物理气变化量

另外，导线电流（以安培为单位）也可以影响导线的物理气，但其影响力小于电压影响力。在绝缘导线电流较低时，电流对导线的物理气影响也很小，导线的性能也没有显著变化。随着电流的增加，导线上物理气的可接受量和实际可承受量之间的差距也会逐渐增大，导线的某些功能可能会受损。

对于一般导线来说，其中的电流可以从源头直流源、电流注入装置、信号传输装置等不同地方输入。对于这些电流，可以在导线工作频段的电压与电流（输入与输出流密度）曲线进行监测。


从图2中可以看到，在某些电流情况下，电流的输入与输出流密度会随时间变化。在电流较低时，流密度较大；在电流较高时，流密度较小。根据电流变化对导线物理气的影响，可以发现，随着电流的增加，导线上物理气的可接受量和实际可承受量之间的差距会逐渐增大。这表明，电流越大，导线上物理气可接受的量越大，但实际上还是有限的。

## 3.3 模型建立

基于上述两个方面的影响，可以建立一个模型来量化描述导线物理气的变化。该模型是一个参数方程：

$$ Q(v,I)=K\cdot I \cdot v^{m} $$

式中，$Q$ 是物理气的变化量，$v$ 为电压，$I$ 为电流，$K$ 和 $m$ 是常数，分别表示导线上物理气的可接受量和阻止系数。

当电压在一定的范围内变化时，模型可用于估计在同样的电压下，电流不同的情况下导线上物理气的变化情况。当电流在一定的范围内变化时，模型可用于估计在同样的电流下，电压不同的情况下导线上物理气的变化情况。

当两者同时变化时，模型无法提供有力的预测。这是因为，电压和电流不是独立的变量，它们之间的关系十分复杂。然而，该模型有一个重要的性质，即在不同的电压和电流组合下，导线上物理气的变化量一般保持一致的行为。换句话说，导线上物理气的变化在电压和电流的不同取值下并不完全一致。

另一个模型，即空气动力学方程式，也可用来描述导线物理气的变化。空气动力学方程式可以用来描述任意一个封闭体内的物理气分布。该方程式提供了一种简洁、严格的物理气模型，其基本假设是空气动力学方程式中物体内部相互作用力和外界力的平均反作用力为零。

空气动力学方程式可写为：

$$ J_{ij}(r,\theta,\phi)=\frac{1}{2}\pi r^2(\partial_\rho U_i+\partial_\rho U_j)-\frac{\epsilon}{r}(\partial_\rho U_i+\partial_\rho U_j)+\sum_{\alpha=1}^N g_{\alpha i}(r,\theta,\phi)\cdot g_{\alpha j}(r,\theta,\phi) $$ 

式中，$J_{ij}$ 是静止空气场的张量密度，$r$ 表示三维坐标的半径，$\theta$ 和 $\phi$ 表示球面坐标的角度，$\rho$ 表示位置。$\epsilon$ 和 $g_{\alpha}$ 分别表示可压缩介质的表面张量积分项和表面势，$\partial_\rho U_i$ 和 $\partial_\rho U_j$ 分别表示第 $i$ 和 $j$ 个原子核的场梯度。

当电流为零时，该方程式可以用来描述空气动力学方程式中各个位置的物理气，以及它们之间的关系。当电流在导线上产生时，该方程式可以用来描述导线上物理气的变化。

# 4.具体代码实例和详细解释说明

下面是相关代码实现示例。

## 4.1 Python 实现

```python
import numpy as np
from scipy import integrate
import matplotlib.pyplot as plt

def physical_gas(v, I):
    """calculate the change of physical gas according to current voltage and current"""
    K = 1e10 # constant: accept rate per unit volatge
    m = -1.5 # constant: viscosity exponent
    return K * I * (v ** m)
    
def initial():
    """initial value"""
    L = 10   # length of the conductor
    d = 1    # diameter of the conductor
    rho = 1e-9   # density of air
    T = 300      # temperature of air
    
    def surface_charge(z):
        """surface charge distribution on a conductor"""
        q = lambda z: np.sin(np.pi*z/L)*np.exp(-((z-(L+d)/2)**2)/(d**2))/(2*np.pi*(d/2)**2)
        return integrate.quad(q, -(d/2), (d/2))[0]
    
    # calculate the total surface area of the conductor
    S = 2*np.pi*((d/2)**2)*integrate.quad(lambda x: integrate.quad(lambda y: 1 if ((x**2+(y**2)<=(d/2)**2)+(x**2+(y**2)>=(d/2)**2)) else 0, -np.inf, np.inf)[0], 0, 1)[0]
    
    # find the average electric field at center of the conductor
    E = np.sqrt(T/(rho*S))*np.array([0, 0, 1])

    V = []   # list for storing voltage values
    I = []   # list for storing current values
    Q = []   # list for storing physical gas amount
    
    v = 1000     # initial voltage
    
    while True:
        J = 0
        for k in range(len(V)):
            if abs(I[k]-I[-1])/abs(I[-1]) > 1e-3 or abs(V[k]-V[-1])/abs(V[-1]) > 1e-3:
                continue
            
            J += (1/2/np.pi)*(rho*E[0]*I[k]/V[k]**2 + rho*E[1]*I[k]/V[k]**2)
        
        # add up all the adjacent term in the electric field potential
        J += sum([(1/2/np.pi)*rho*E[2]*(np.linalg.norm(np.cross(E[:2], r))/V[k]**2)*(I[k]/V[k]**2) for k, r in enumerate(positions)])

        A = surface_charge(0) / S
        
        delta_v = (-J + A)/A
        
        v -= delta_v
        print("current voltage:", v)
        
        if v < 0:
            break
            
        if len(V)==0 or abs(delta_v/V[-1])<1e-6:
            break
        
        V.append(v)
        I.append(J/A)
        Q.append(physical_gas(v, J/A))
        
    fig, axes = plt.subplots(nrows=2, ncols=1, figsize=(10,10))
    axes[0].plot(V, I, label="Current")
    axes[0].set_xlabel("Voltage [volts]")
    axes[0].set_ylabel("Current [amperes]")
    axes[0].legend()
    
    axes[1].plot(V, Q, label="Physical Gas Amount")
    axes[1].set_xlabel("Voltage [volts]")
    axes[1].set_ylabel("Physical Gas Amount [mol/cm^3]")
    axes[1].legend()
    
    plt.show()
    

if __name__ == '__main__':
    positions = [(1,0,0),(0,1,0)]   # position array
    initial()                    # run simulation program
```

这里使用的工具包为 `numpy`、`scipy`、`matplotlib`，具体安装请参考官方文档。

该程序首先定义了一个函数 `physical_gas()` 来计算物理气随电压和电流变化量。然后调用 `initial()` 函数来运行模拟程序，其中定义了导线的长度、直径、空气密度、温度、势阱的位置数组、初始电压、以及电压的增量步长。

模拟程序会迭代执行，每次循环都会更新当前的电压和电流，并更新物理气的变化量。每一步迭代结束后，会计算当前的静止空气场中的电场力矩，以及导线上产生的电场力矩，并且根据电场力矩和电压的增量算出下一级的电压。程序会一直迭代，直到当前的电压小于零。

模拟程序完成之后，会绘制出电压-电流和电压-物理气量随时间变化的曲线图，供观察。

## 4.2 模拟结果

运行该程序，会生成电压-电流和电压-物理气量随时间变化的曲线图。本文拟研究的是一个含有电压、电流、空气物理气的导线，初始电压为 $1000$ 伏特，通过模拟可以得知，电压和电流对导线上物理气的影响是指数级的，电压太小会导致导线上物理气的变化量大幅度降低，电压太大会导致导线上物理气的变化量不足。除此之外，物理气的变化量与导线长度、导线直径、空气密度、温度等因素有关，这也使得这类模拟方法可以量化地刻画导线物理气的特征。
