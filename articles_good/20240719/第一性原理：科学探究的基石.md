### 4.2 公式推导过程

> 公式推导：
1. **线性回归模型的推导**：
   - 样本数据：$(x_1, y_1), (x_2, y_2), \cdots, (x_n, y_n)$
   - 最小二乘法：$\min_{a, b} \sum_{i=1}^n (y_i - (ax_i + b))^2$
   - 解方程组：
     \begin{align*}
     \sum_{i=1}^n x_i &= \sum_{i=1}^n ax_i + \sum_{i=1}^n b \\
     \sum_{i=1}^n x_i^2 &= \sum_{i=1}^n x_i^2 a + \sum_{i=1}^n x_i b
     \end{align*}
   - 得到回归系数：
     \begin{align*}
     a &= \frac{\sum_{i=1}^n (x_i y_i) - n \bar{x} \bar{y}}{\sum_{i=1}^n x_i^2 - n \bar{x}^2} \\
     b &= \bar{y} - a \bar{x}
     \end{align*}

2. **统计假设检验的推导**：
   - 假设 $H_0$：$\mu_1 = \mu_2$，$H_a$：$\mu_1 \neq \mu_2$
   - 样本均值：$\bar{x}_1, \bar{x}_2$
   - 样本方差：$s_1^2, s_2^2$
   - T检验：
     \begin{align*}
     t &= \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}} \\
     p &= P(T > t) \text{ or } P(T < -t)
     \end{align*}
   - 确定结论：
     - 如果 $p < \alpha$，则拒绝 $H_0$，接受 $H_a$
     - 如果 $p > \alpha$，则接受 $H_0$，拒绝 $H_a$

   ```python
   import numpy as np

   # 数据集
   x = np.array([1, 2, 3, 4, 5])
   y = np.array([2, 4, 5, 4, 6])

   # 模型参数
   a = 0
   b = 0

   # 最小二乘法求解
   for i in range(len(x)):
       y_pred = a * x[i] + b
       error = y[i] - y_pred
       da = error * x[i]
       db = error
       a += da
       b += db

   # 输出结果
   print("a =", a)
   print("b =", b)
   ```

2. **统计假设检验**：
   ```python
   import numpy as np
   from scipy.stats import t

   # 数据集
   x1 = np.array([10, 12, 14, 16, 18])
   x2 = np.array([20, 22, 24, 26, 28])
   y1 = np.array([8, 9, 11, 12, 13])
   y2 = np.array([9, 11, 12, 14, 15])

   # 假设检验
   n1 = len(x1)
   n2 = len(x2)
   df = n1 + n2 - 2
   mean_diff = np.mean(x1) - np.mean(x2)
   std_diff = np.sqrt(np.var(x1) / n1 + np.var(x2) / n2)
   p_value = t.cdf(mean_diff / std_diff, df)

   # 输出结果
   print("p =", p_value)
   ```
