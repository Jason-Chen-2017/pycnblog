### 5.2 源代码详细实现

以下是一个基于以太坊智能合约的版权保护案例：

```solidity
pragma solidity ^0.8.0;

contract CopyrightProtection {
    struct Copyright {
        string title;
        string author;
        string contentHash;
        bool isProtected;
    }

    mapping(string => Copyright) public copyrights;

    function registerCopyright(string memory title, string memory author, string memory contentHash) public {
        require(!copyrights[contentHash].isProtected, "Copyright is already protected");
        copyrights[contentHash] = Copyright(title, author, contentHash, true);
    }

    function isCopyrightProtected(string memory contentHash) public view returns (bool) {
        return copyrights[contentHash].isProtected;
    }
}
```

### 5.3 代码解读与分析

以上代码实现了一个简单的版权保护智能合约，其功能如下：

1. **Copyright 结构**：定义版权信息结构，包括标题、作者、内容哈希值和版权保护状态。
2. **registerCopyright 函数**：允许用户注册版权，将版权信息存储到合约中。
3. **isCopyrightProtected 函数**：判断版权是否受保护。
