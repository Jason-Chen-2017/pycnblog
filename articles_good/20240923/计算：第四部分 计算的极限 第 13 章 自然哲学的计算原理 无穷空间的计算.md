### 4.3 案例分析与讲解

以下是一个简单的无穷空间计算案例：

假设我们在一个无穷大的网格中寻找一个特定的点，我们可以使用广度优先搜索算法来解决这个问题。

```python
from collections import deque

def bfs(grid, target):
    queue = deque([(0, 0)])
    visited = set([(0, 0)])

    while queue:
        x, y = queue.popleft()
        if (x, y) == target:
            return True
        for dx, dy in [(1, 0), (-1, 0), (0, 1), (0, -1)]:
            nx, ny = x + dx, y + dy
            if (nx, ny) not in visited:
                visited.add((nx, ny))
                queue.append((nx, ny))

    return False

grid = [[1, 1, 1], [1, 0, 1], [1, 1, 1]]
target = (2, 2)

print(bfs(grid, target))  # 输出：True
```
