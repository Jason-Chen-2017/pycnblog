## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

在双线布局中，我们可以构建以下数学模型：

$$
f(x, y) = \frac{1}{2}\left[(x - x_0)^2 + (y - y_0)^2\right]
$$

其中，$x$ 和 $y$ 分别代表两个独立战略线的指标，$x_0$ 和 $y_0$ 分别代表各自的目标值。

### 4.2 公式推导过程

1. **目标函数**：首先，我们定义目标函数为两个战略线指标的平方和的一半，即：

$$
f(x, y) = \frac{1}{2}(x^2 + y^2)
$$

2. **约束条件**：其次，我们设定以下约束条件：

$$
\begin{cases}
x \geq 0 \\
y \geq 0 \\
x + y \leq C
\end{cases}
$$

其中，$C$ 代表资源总量。

3. **拉格朗日乘数法**：为了解决上述约束条件，我们引入拉格朗日乘数法，得到：

$$
L(x, y, \lambda) = f(x, y) + \lambda(g(x, y) - C)
$$

其中，$\lambda$ 为拉格朗日乘数。

4. **求导**：对 $L(x, y, \lambda)$ 分别对 $x$、$y$ 和 $\lambda$ 求导，并令导数为零，得到：

$$
\begin{cases}
\frac{\partial L}{\partial x} = x - \lambda = 0 \\
\frac{\partial L}{\partial y} = y - \lambda = 0 \\
\frac{\partial L}{\partial \lambda} = g(x, y) - C = 0
\end{cases}
$$

5. **解方程组**：解上述方程组，得到：

$$
\begin{cases}
x = \lambda \\
y = \lambda \\
x + y = C
\end{cases}
$$

因此，$x = y = \frac{C}{2}$，即两个战略线指标应相等，以达到最优布局。

import numpy as np
import matplotlib.pyplot as plt

def f(x, y):
    return 0.5 * (x**2 + y**2)

def constraint(x, y, C):
    return x + y - C

def lagrange_multiplier(C):
    x = np.linspace(0, C, 100)
    y = C - x
    lambda_ = np.zeros_like(x)

    for i in range(len(x)):
        lambda_[i] = x[i]

    return x, y, lambda_

def plot_layout(x, y, lambda_):
    plt.figure()
    plt.plot(x, y, label="Strategy Line 1")
    plt.plot(y, x, label="Strategy Line 2")
    plt.scatter(x, y, color="red", label="Optimal Layout")
    plt.scatter(lambda_, lambda_, color="green", label="Lagrange Multipliers")
    plt.xlabel("x")
    plt.ylabel("y")
    plt.legend()
    plt.show()

if __name__ == "__main__":
    C = 100
    x, y, lambda_ = lagrange_multiplier(C)
    plot_layout(x, y, lambda_)
