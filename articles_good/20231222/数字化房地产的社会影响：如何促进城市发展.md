## 4.1 信息化代码实例

### 4.1.1 数据处理

```python
import pandas as pd

# 读取房地产数据
data = pd.read_csv('house_data.csv')

# 数据处理
data['price'] = data['price'] * 10000
data['area'] = data['area'] / 10
data['price_per_square_meter'] = data['price'] / data['area']

# 保存处理后的数据
data.to_csv('processed_house_data.csv', index=False)
```

### 4.1.2 数据库管理

```python
from sqlalchemy import create_engine

# 创建数据库连接
engine = create_engine('mysql+pymysql://username:password@localhost/dbname')

# 创建数据表
data_table = """
CREATE TABLE house_data (
    id INT PRIMARY KEY,
    price DECIMAL(10,2),
    area DECIMAL(10,2),
    price_per_square_meter DECIMAL(10,2)
)
"""

# 创建数据表
engine.execute(data_table)

# 导入处理后的数据
processed_data = pd.read_csv('processed_house_data.csv')
processed_data.to_csv('processed_house_data.sql', index=False)
```
