### 编程题库

#### 1. 平衡括号
**题目：** 编写一个函数，检查给定的字符串中是否包含平衡的括号。

**答案：**
```go
func isBalanced(s string) bool {
    stack := []rune{}
    for _, char := range s {
        switch char {
        case '(':
            stack = append(stack, ')')
        case '{':
            stack = append(stack, '}')
        case '[':
            stack = append(stack, ']')
        default:
            if len(stack) == 0 || char != stack[len(stack)-1] {
                return false
            }
            stack = stack[:len(stack)-1]
        }
    }
    return len(stack) == 0
}

// 测试
fmt.Println(isBalanced("()"))   // true
fmt.Println(isBalanced("({)}")) // false
fmt.Println(isBalanced("[{()}]")) // true
```
**解析：** 使用栈结构来存储左括号，当遇到右括号时，与栈顶元素比较是否匹配，不匹配则返回false。最后检查栈是否为空。

#### 2. 单词搜索
**题目：** 给定一个二维字符网格和一个单词，请判断该单词是否存在于网格中。

**答案：**
```go
var grid = [][]byte{
    {'A', 'B', 'C', 'E'},
    {'S', 'F', 'C', 'S'},
    {'A', 'D', 'E', 'E'},
}

func exist(board [][]byte, word string) bool {
    rows, cols := len(board), len(board[0])
    for i := 0; i < rows; i++ {
        for j := 0; j < cols; j++ {
            if dfs(board, i, j, word) {
                return true
            }
        }
    }
    return false
}

func dfs(board [][]byte, i, j int, word string) bool {
    if len(word) == 0 {
        return true
    }
    rows, cols := len(board), len(board[0])
    if i < 0 || i >= rows || j < 0 || j >= cols || board[i][j] != byte(word[0]) {
        return false
    }
    temp := board[i][j]
    board[i][j] = 0 // 标记已访问
    res := dfs(board, i+1, j, word[1:]) ||
           dfs(board, i-1, j, word[1:]) ||
           dfs(board, i, j+1, word[1:]) ||
           dfs(board, i, j-1, word[1:])
    board[i][j] = temp // 恢复原值
    return res
}

// 测试
fmt.Println(exist(grid, "ABCCED")) // true
fmt.Println(exist(grid, "SEE")) // true
fmt.Println(exist(grid, "ABCB")) // false
```
**解析：** 使用深度优先搜索（DFS）遍历网格，对于每个符合条件的起点，递归搜索周围的所有可能路径。

#### 3. 最大子序和
**题目：** 给定一个整数数组 nums ，找到一个具有最大和的连续子数组（至少长度为 1）。

**答案：**
```go
func maxSubArray(nums []int) int {
    maxSum := nums[0]
    curSum := nums[0]
    for i := 1; i < len(nums); i++ {
        curSum = max(nums[i], curSum+nums[i])
        maxSum = max(maxSum, curSum)
    }
    return maxSum
}

// 测试
fmt.Println(maxSubArray([]int{-2, 1, -3, 4, -1, 2, 1, -5, 4})) // 6
```
**解析：** 动态规划，每次更新当前子序列的最大和，并与全局最大和进行比较。

### 面试题库

#### 1. 数据结构与算法基础知识
**题目：** 请简述哈希表的工作原理以及解决哈希冲突的方法。

**答案：** 哈希表通过哈希函数将键映射到数组索引，从而实现快速查找、插入和删除操作。解决哈希冲突的方法有：
* **链地址法：** 数组中的每个槽包含一个链表，具有相同哈希值的元素存储在同一个链表中。
* **开放地址法：** 当发生冲突时，继续查找下一个槽，直到找到一个空的槽。
* **再哈希法：** 当发生冲突时，使用不同的哈希函数重新计算哈希值。

#### 2. 编程语言特性
**题目：** Go 语言中的 Goroutine 和 Channel 如何实现并发？

**答案：** Goroutine 是 Go 语言内置的轻量级线程，通过 `go` 关键字创建。Channel 是用于在 Goroutine 之间传递数据的通道。实现并发的方法包括：
* 使用 `select` 语句在多个通道上等待操作。
* 使用 `for range` 循环从通道接收数据。
* 使用 `close` 函数关闭通道。
