### 源代码实例

为了更好地理解注意力经济对城市规划的新要求，以下提供一个简单的源代码实例，用于展示如何通过编程实现城市规划中的一些关键概念。

```go
package main

import (
    "fmt"
)

// 定义城市规划结构体
type UrbanPlan struct {
    CityName   string
    Infrastructure map[string]int
    Attractions []string
    Population  int
    GreenArea   float64
}

// 添加基础设施的方法
func (up *UrbanPlan) AddInfrastructure(facility string, cost int) {
    up.Infrastructure[facility] = cost
}

// 添加景点的方法
func (up *UrbanPlan) AddAttraction(attraction string) {
    up.Attractions = append(up.Attractions, attraction)
}

// 计算人均基础设施成本的方法
func (up *UrbanPlan) CalculatePerCapitaCost() float64 {
    totalCost := 0
    for _, cost := range up.Infrastructure {
        totalCost += cost
    }
    return float64(totalCost) / float64(up.Population)
}

// 打印城市规划信息的方法
func (up *UrbanPlan) PrintPlan() {
    fmt.Printf("城市规划：城市名：%s\n", up.CityName)
    fmt.Printf("基础设施：\n")
    for facility, cost := range up.Infrastructure {
        fmt.Printf("%s：成本：%d\n", facility, cost)
    }
    fmt.Printf("景点：\n")
    for _, attraction := range up.Attractions {
        fmt.Printf("%s\n", attraction)
    }
    fmt.Printf("人均基础设施成本：%.2f\n", up.CalculatePerCapitaCost())
    fmt.Printf("绿地面积：%.2f 平方千米\n", up.GreenArea)
}

func main() {
    // 创建城市规划实例
    plan := UrbanPlan{
        CityName:   "智慧城",
        Infrastructure: map[string]int{
            "交通网络": 50000000,
            "供水系统": 30000000,
            "电力网络": 40000000,
        },
        Attractions: []string{"科技馆", "文化公园", "艺术中心"},
        Population:  1000000,
        GreenArea:   50.0,
    }

    // 添加新的基础设施
    plan.AddInfrastructure("废物处理中心", 25000000)

    // 添加新的景点
    plan.AddAttraction("儿童乐园")

    // 打印城市规划信息
    plan.PrintPlan()
}
```

**代码解析：**

- `UrbanPlan` 结构体定义了城市规划的关键属性，如城市名、基础设施、景点、人口和绿地面积。
- `AddInfrastructure` 方法用于添加新的基础设施及其成本。
- `AddAttraction` 方法用于添加新的景点。
- `CalculatePerCapitaCost` 方法用于计算人均基础设施成本。
- `PrintPlan` 方法用于打印城市规划的详细信息。
- 在 `main` 函数中，我们创建了一个 `UrbanPlan` 实例，并调用相关方法进行添加和打印。

此代码实例展示了如何通过编程来模拟城市规划的过程，以及如何处理注意力经济对城市规划的影响。在实际应用中，可以进一步扩展此代码，以包含更复杂的逻辑，如注意力经济指标的计算和评估。通过这些编程实践，可以帮助城市规划师更好地理解和应对注意力经济带来的挑战。
