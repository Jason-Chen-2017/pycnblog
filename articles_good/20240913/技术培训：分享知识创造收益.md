### 1. 数据结构与算法

#### 1.1 链表反转

**题目：** 编写一个函数，实现单链表反转。

```go
type ListNode struct {
    Val  int
    Next *ListNode
}

func reverseList(head *ListNode) *ListNode {
    // 解答内容
}
```

**答案解析：** 使用头插法，将原链表的每个节点插入到新链表的最前面。

```go
func reverseList(head *ListNode) *ListNode {
    var newHead *ListNode
    for head != nil {
        next := head.Next
        head.Next = newHead
        newHead = head
        head = next
    }
    return newHead
}
```

### 2. 并发编程

#### 2.1 同步多个 goroutine

**题目：** 使用通道和 WaitGroup 实现一个并发程序，其中多个 goroutine 并行执行，并在主 goroutine 中等待所有子 goroutine 完成。

```go
func main() {
    // 解答内容
}
```

**答案解析：** 创建 WaitGroup，启动多个 goroutine，并使用通道传递数据。

```go
func main() {
    var wg sync.WaitGroup
    c := make(chan struct{})
    for i := 0; i < 5; i++ {
        wg.Add(1)
        go func() {
            defer wg.Done()
            time.Sleep(time.Second)
            fmt.Println("Goroutine", i, "done")
            c <- struct{}{}
        }()
    }
    wg.Wait()
    close(c)
    fmt.Println("All goroutines done")
}
```

### 3. 网络编程

#### 3.1 HTTP 客户端

**题目：** 使用 Go 标准库编写一个简单的 HTTP 客户端，实现发送 GET 请求并解析响应。

```go
import (
    "bytes"
    "net/http"
)

func httpGet(url string) (string, error) {
    // 解答内容
}

func main() {
    // 解答内容
}
```

**答案解析：** 使用 `http.Get` 方法发送 GET 请求，并使用 `json.Unmarshal` 解析 JSON 响应。

```go
import (
    "bytes"
    "encoding/json"
    "net/http"
)

func httpGet(url string) (string, error) {
    resp, err := http.Get(url)
    if err != nil {
        return "", err
    }
    defer resp.Body.Close()

    var data struct {
        Message string `json:"message"`
    }
    if err := json.NewDecoder(resp.Body).Decode(&data); err != nil {
        return "", err
    }
    return data.Message, nil
}

func main() {
    message, err := httpGet("https://jsonplaceholder.typicode.com/todos/1")
    if err != nil {
        fmt.Println("Error:", err)
    } else {
        fmt.Println("Message:", message)
    }
}
```

### 4. 数据存储

#### 4.1 Redis 基本操作

**题目：** 使用 Go 的 Redis 客户端实现 Redis 的基本操作，如设置、获取和删除键值对。

```go
import (
    "github.com/go-redis/redis/v8"
)

func main() {
    // 解答内容
}
```

**答案解析：** 使用 `redis.Client` 连接 Redis，并执行基本操作。

```go
import (
    "github.com/go-redis/redis/v8"
)

func main() {
    rdb := redis.NewClient(&redis.Options{
        Addr:     "localhost:6379",
        Password: "", // no password set
        DB:       0,  // use default DB
    })

    err := rdb.Set("key", "value", 0).Err()
    if err != nil {
        panic(err)
    }

    val, err := rdb.Get("key").Result()
    if err != nil {
        panic(err)
    }
    fmt.Println("key", val)

    err = rdb.Del("key").Err()
    if err != nil {
        panic(err)
    }
}
```
