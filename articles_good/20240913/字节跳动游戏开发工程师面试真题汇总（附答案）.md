**1. 如何实现一个LRU缓存机制？**

**答案：** 可以使用哈希表和双向链表实现LRU缓存。哈希表用于快速访问节点，双向链表用于维护访问顺序。

```python
class Node:
    def __init__(self, key, value):
        self.key = key
        self.value = value
        self.prev = None
        self.next = None

class LRUCache:
    def __init__(self, capacity):
        self.capacity = capacity
        self.dict = {}
        self.head = Node(0, 0)
        self.tail = Node(0, 0)
        self.head.next = self.tail
        self.tail.prev = self.head

    def get(self, key):
        if key not in self.dict:
            return -1
        node = self.dict[key]
        self._remove(node)
        self._add(node)
        return node.value

    def put(self, key, value):
        if key in self.dict:
            node = self.dict[key]
            node.value = value
            self._remove(node)
            self._add(node)
        else:
            if len(self.dict) >= self.capacity:
                del self.dict[self.tail.prev.key]
                self._remove(self.tail.prev)
            self.dict[key] = self.head.next
            self._add(Node(key, value))

    def _remove(self, node):
        p = node.prev
        n = node.next
        p.next = n
        n.prev = p

    def _add(self, node):
        p = self.head
        n = self.head.next
        p.next = node
        n.prev = node
        node.prev = p
        node.next = n
```

**2. 如何在排序数组中查找元素的第一个和最后一个位置？**

**答案：** 分别使用二分查找算法，针对数组中的值分别找到第一个和最后一个位置。

```python
def searchRange(nums, target):
    def find_left(nums, target):
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            elif nums[mid] > target:
                right = mid - 1
            else:
                if mid == 0 or nums[mid - 1] != target:
                    return mid
                right = mid - 1
        return -1

    def find_right(nums, target):
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            elif nums[mid] > target:
                right = mid - 1
            else:
                if mid == len(nums) - 1 or nums[mid + 1] != target:
                    return mid
                left = mid + 1
        return -1

    return [find_left(nums, target), find_right(nums, target)]
```
