## **三、算法编程题库**

### 7. 计算最佳种植区域

**题目：** 给定一个农田的地图（用二维数组表示），每个单元表示土壤类型和产量潜力，编写一个算法找出最佳种植区域，使得产量最大化。

**答案：**
```go
package main

import (
    "fmt"
)

func maxProductRegion(lands [][]int) int {
    m, n := len(lands), len(lands[0])
    maxProd := 0

    for i := 0; i < m; i++ {
        for j := 0; j < n; j++ {
            maxProd = max(maxProd, maxProductFrom(lands, i, j))
        }
    }
    return maxProd
}

func maxProductFrom(lands [][]int, i int, j int) int {
    m, n := len(lands), len(lands[0])
    if i < 0 || i >= m || j < 0 || j >= n {
        return 0
    }

    if lands[i][j] == 0 {
        return 0
    }

    leftProd := maxProductFrom(lands, i, j-1)
    rightProd := maxProductFrom(lands, i, j+1)
    upProd := maxProductFrom(lands, i-1, j)
    downProd := maxProductFrom(lands, i+1, j)

    return lands[i][j] + max(leftProd, rightProd, upProd, downProd)
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}

func main() {
    lands := [][]int{
        {2, 1, 0, 1, 2},
        {1, 1, 0, 1, 1},
        {1, 1, 1, 1, 1},
        {2, 1, 1, 1, 2},
        {1, 1, 0, 1, 1},
    }
    fmt.Println(maxProductRegion(lands))
}
```

**解析：** 该算法通过递归计算每个单元格的四个相邻单元格的最大值，并加上当前单元格的值，得到最佳种植区域的产量。通过遍历整个二维数组，找到最大的产量。

### 8. 预测植物生长周期

**题目：** 给定植物的生长数据（包括土壤湿度、光照强度、温度等），编写一个算法预测植物的生长周期。

**答案：**
```go
package main

import (
    "fmt"
)

func predictGrowthCycle(data [][]int) int {
    maxGrowthRate := 0
    maxCycleLength := 0

    for i := 0; i < len(data)-1; i++ {
        currentGrowthRate := data[i+1][0] - data[i][0]
        cycleLength := data[i+1][1] - data[i][1]

        if currentGrowthRate > maxGrowthRate {
            maxGrowthRate = currentGrowthRate
            maxCycleLength = cycleLength
        }
    }

    return maxCycleLength
}

func main() {
    data := [][]int{
        {10, 1}, // 湿度，生长周期
        {12, 3},
        {15, 5},
        {18, 7},
        {20, 9},
    }
    fmt.Println(predictGrowthCycle(data))
}
```

**解析：** 该算法通过计算每两个连续数据点之间的生长速率（湿度变化）和生长周期（时间变化），找到最大的生长速率对应的生长周期。

### 9. 优化灌溉策略

**题目：** 给定农田的湿度分布和灌溉水源的位置，编写一个算法优化灌溉策略，使农田湿度分布最均匀。

**答案：**
```go
package main

import (
    "fmt"
)

func optimizeIrrigation(lands []int, waterSource int) []int {
    n := len(lands)
    totalWater := n * (lands[0] + lands[n-1]) / 2

    for i := 0; i < n; i++ {
        if i == waterSource {
            continue
        }
        diff := lands[waterSource] - lands[i]
        if diff > 0 {
            lands[i] += diff / 2
            lands[waterSource] -= diff / 2
            totalWater -= diff
        }
    }

    if totalWater > 0 {
        // 平均分配剩余的水量
        for i := 0; i < n; i++ {
            if i == waterSource {
                continue
            }
            lands[i] += totalWater / n
        }
    }

    return lands
}

func main() {
    lands := []int{3, 2, 1, 4, 3}
    waterSource := 2
    fmt.Println(optimizeIrrigation(lands, waterSource))
}
```

**解析：** 该算法首先将水源处的湿度与农田其他地方的不均匀湿度差平均分配，然后分配剩余的水量，使得农田湿度分布最均匀。
