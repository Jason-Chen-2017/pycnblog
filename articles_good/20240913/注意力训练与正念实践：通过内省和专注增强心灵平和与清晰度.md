#### 4. 专注力测试的算法实现

**算法编程题：** 实现一个专注力测试的算法，根据用户输入的专注时间记录，评估用户的专注力水平。

**答案：** 以下是一个简单的专注力测试算法实现：

```python
def calculate_focus_score(attention_logs):
    total_time = 0
    distracted_time = 0

    for log in attention_logs:
        if log['status'] == 'distracted':
            distracted_time += log['duration']
        else:
            total_time += log['duration']

    if total_time == 0:
        return 0  # 无法计算专注力得分

    focus_score = (total_time - distracted_time) / total_time
    return focus_score

# 示例数据
attention_logs = [
    {'status': 'focused', 'duration': 30},
    {'status': 'distracted', 'duration': 5},
    {'status': 'focused', 'duration': 20},
    {'status': 'distracted', 'duration': 10}
]

# 计算专注力得分
score = calculate_focus_score(attention_logs)
print(f"Focus Score: {score:.2f}")
```

**解析：** 该算法通过用户输入的专注时间记录，计算专注时间和分心时间的比例，从而得到专注力得分。分数越高，表示用户的专注力水平越高。
