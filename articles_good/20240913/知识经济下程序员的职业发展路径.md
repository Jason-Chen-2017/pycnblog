#### 面试题库

##### 1. 什么是单例模式？请举例说明。

**答案：** 单例模式是一种设计模式，用于确保一个类只有一个实例，并提供一个全局访问点。例如，在 Java 中，可以使用静态成员变量和静态工厂方法来实现单例模式。

**举例：**

```java
public class Singleton {
    private static Singleton instance;

    private Singleton() {}

    public static Singleton getInstance() {
        if (instance == null) {
            instance = new Singleton();
        }
        return instance;
    }
}
```

##### 2. 如何实现一个线程安全的单例模式？

**答案：** 有几种方法可以实现线程安全的单例模式：

1. 使用同步锁（synchronized）：
```java
public class Singleton {
    private static Singleton instance;

    private Singleton() {}

    public static synchronized Singleton getInstance() {
        if (instance == null) {
            instance = new Singleton();
        }
        return instance;
    }
}
```

2. 使用静态块初始化：
```java
public class Singleton {
    private static Singleton instance;

    private Singleton() {}

    static {
        if (instance == null) {
            instance = new Singleton();
        }
    }

    public static Singleton getInstance() {
        return instance;
    }
}
```

##### 3. 什么是事件驱动编程？请举例说明。

**答案：** 事件驱动编程是一种编程范式，它基于事件来触发程序中的行为。事件可以是用户输入、定时任务或其他系统事件。

**举例：** 在 Python 的 tkinter 库中，可以使用事件驱动编程来响应用户输入：

```python
import tkinter as tk

def on_button_click():
    print("Button clicked")

root = tk.Tk()
button = tk.Button(root, text="Click me", command=on_button_click)
button.pack()

root.mainloop()
```

#### 算法编程题库

##### 4. 寻找两个正序数组的中位数。

**题目描述：** 给定两个大小分别为 m 和 n 的正序数组 nums1 和 nums2。请从它们中找出中位数。

**答案：** 可以使用二分查找算法来解决这个问题。时间复杂度为 O(log(min(m, n)))。

```python
def findMedianSortedArrays(nums1, nums2):
    if len(nums1) > len(nums2):
        nums1, nums2 = nums2, nums1

    m, n = len(nums1), len(nums2)
    imin, imax, half_len = 0, m, (m + n + 1) // 2
    while imin <= imax:
        i = (imin + imax) // 2
        j = half_len - i
        if i < m and nums2[j - 1] > nums1[i]:
            imin = i + 1
        elif i > 0 and nums1[i - 1] > nums2[j]:
            imax = i - 1
        else:
            if i == 0:
                max_of_left = nums2[j - 1]
            elif j == 0:
                max_of_left = nums1[i - 1]
            else:
                max_of_left = max(nums1[i - 1], nums2[j - 1])
            if (m + n) % 2 == 1:
                return max_of_left
            min_of_right = min(nums1[i], nums2[j])
            return (max_of_left + min_of_right) / 2

nums1 = [1, 3]
nums2 = [2]
print(findMedianSortedArrays(nums1, nums2))
```

##### 5. 两数相加。

**题目描述：** 不使用运算符 + 和 - ，实现两个整数相加。

**答案：** 可以使用位操作来实现。时间复杂度为 O(max(n, m))。

```python
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        current = dummy
        carry = 0
        while l1 or l2 or carry:
            val1 = 0 if not l1 else l1.val
            val2 = 0 if not l2 else l2.val
            total = val1 + val2 + carry
            carry = total // 10
            current.next = ListNode(total % 10)
            current = current.next
            if l1:
                l1 = l1.next
            if l2:
                l2 = l2.next
        return dummy.next
```
