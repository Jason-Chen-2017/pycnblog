#### 4. 编写一个算法，实现对对话中关键信息的提取和存储。

**题目描述：**

编写一个算法，接收一段对话文本，提取出其中的关键信息（如用户名、日期、地点等），并将其存储在指定的数据结构中。

**答案：**

```python
import re

def extract_key_info(dialog):
    key_info = []
    pattern = re.compile(r'@[\w]+|\d{4}-\d{2}-\d{2}|[A-Za-z]+')
    matches = pattern.finditer(dialog)
    for match in matches:
        key_info.append(match.group())
    return key_info

dialog = "你好，我是张三，今天2023-10-01，我们见面地点在上海。"
print(extract_key_info(dialog))
```

**解析：**

这个问题考察应聘者对文本处理和正则表达式的掌握。通过使用正则表达式，可以高效地提取出对话中的关键信息。实际面试中，这类问题可能涉及更多的数据结构和算法知识。

#### 5. 编写一个算法，实现对对话中上下文的保存和查询。

**题目描述：**

编写一个算法，实现对对话中上下文的保存和查询。对话文本中的关键信息（如用户名、日期、地点等）将被视为上下文信息。要求算法能够根据用户输入的查询关键词，返回与关键词相关的上下文信息。

**答案：**

```python
class ContextMemory:
    def __init__(self):
        self.contexts = {}

    def save_context(self, user_id, context):
        if user_id in self.contexts:
            self.contexts[user_id].append(context)
        else:
            self.contexts[user_id] = [context]

    def query_context(self, user_id, keyword):
        if user_id in self.contexts:
            results = []
            for context in self.contexts[user_id]:
                if keyword in context:
                    results.append(context)
            return results
        else:
            return []

# 使用示例
memory = ContextMemory()
memory.save_context("user123", "你好，我是张三，今天2023-10-01，我们见面地点在上海。")
results = memory.query_context("user123", "张三")
print(results) # 输出：['你好，我是张三，今天2023-10-01，我们见面地点在上海。']
```

**解析：**

这个问题考察应聘者对数据结构的设计和算法的实现能力。通过实现一个简单的上下文记忆类，可以保存和查询对话中的上下文信息。实际面试中，这类问题可能结合具体的业务场景进行考察。
