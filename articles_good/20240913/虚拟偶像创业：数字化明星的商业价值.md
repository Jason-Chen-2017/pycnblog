**3. 如何实现虚拟偶像的表情识别？**

**题目：** 编写一个简单的表情识别程序，识别输入的图片中的表情，并输出对应的文字描述。

**答案：** 这里使用 Python 的 OpenCV 库实现表情识别。

```python
import cv2
import numpy as np

def detect_emotion(img_path):
    face_cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')
    emotion_cascade = cv2.CascadeClassifier('haarcascade_eye.xml')

    img = cv2.imread(img_path)
    gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

    faces = face_cascade.detectMultiScale(gray, 1.3, 5)
    for (x, y, w, h) in faces:
        cv2.rectangle(img, (x, y), (x+w, y+h), (255, 0, 0), 2)
        roi_gray = gray[y:y+h, x:x+w]
        roi_color = img[y:y+h, x:x+w]
        eyes = emotion_cascade.detectMultiScale(roi_gray)
        for (ex, ey, ew, eh) in eyes:
            cv2.rectangle(roi_color, (ex, ey), (ex+ew, ey+eh), (0, 255, 0), 2)

    cv2.imshow('Emotions', img)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

if __name__ == "__main__":
    detect_emotion('example.jpg')
```

**解析：** 该程序使用 OpenCV 库检测输入图片中的面部和眼睛，并在检测到面部时绘制矩形框。用户可以通过按任意键退出程序。
