### 四、算法编程题库

**1. 编写一个函数，实现数据清洗过程中的去重操作。**

**输入：** 数据列表（如[1, 2, 2, 3, 4, 4, 4]）

**输出：** 去重后的数据列表（如[1, 2, 3, 4]）

```python
def remove_duplicates(data):
    return list(set(data))

data = [1, 2, 2, 3, 4, 4, 4]
print(remove_duplicates(data))
```

**2. 编写一个函数，实现数据预处理中的数据填充。**

**输入：** 数据列表（如[1, 2, None, 4]）

**输出：** 填充后的数据列表（如[1, 2, 0, 4]）

```python
def fill_data(data):
    for i in range(len(data)):
        if data[i] is None:
            data[i] = 0
    return data

data = [1, 2, None, 4]
print(fill_data(data))
```

**3. 编写一个函数，实现数据预处理中的数据归一化。**

**输入：** 数据列表（如[1, 2, 3, 4]）

**输出：** 归一化后的数据列表（如[0.0, 0.25, 0.5, 0.75]）

```python
def normalize_data(data):
    min_val = min(data)
    max_val = max(data)
    return [(x - min_val) / (max_val - min_val) for x in data]

data = [1, 2, 3, 4]
print(normalize_data(data))
```
