**题目：** 编写一个函数，用于加密和解密数字资产数据。

**答案：**

```python
from cryptography.fernet import Fernet

def encrypt_data(data, key):
    f = Fernet(key)
    encrypted_data = f.encrypt(data.encode())
    return encrypted_data

def decrypt_data(encrypted_data, key):
    f = Fernet(key)
    decrypted_data = f.decrypt(encrypted_data).decode()
    return decrypted_data

# 示例
key = Fernet.generate_key()
data = "This is a secret message."
encrypted_data = encrypt_data(data, key)
print(encrypted_data)
decrypted_data = decrypt_data(encrypted_data, key)
print(decrypted_data)
```

**解析：** 该函数使用 Fernet 库实现 AES 加密算法，对数据进行加密和解密。

```python
import random

def analyze_risk(assets):
    risk_levels = {"low_risk": 0, "medium_risk": 0, "high_risk": 0}
    for asset in assets:
        if asset["status"] == "low_risk":
            risk_levels["low_risk"] += 1
        elif asset["status"] == "medium_risk":
            risk_levels["medium_risk"] += 1
        elif asset["status"] == "high_risk":
            risk_levels["high_risk"] += 1
    total_risk = sum(risk_levels.values())
    risk_percentage = {level: (count / total_risk) * 100 for level, count in risk_levels.items()}
    return risk_percentage

# 示例
assets = [
    {"id": 1, "name": "Bitcoin", "status": "low_risk"},
    {"id": 2, "name": "Facebook", "status": "high_risk"},
    {"id": 3, "name": "Google Drive", "status": "low_risk"},
]
print(analyze_risk(assets))
```

**解析：** 该函数计算数字资产的风险水平，并返回每个风险水平的百分比。
