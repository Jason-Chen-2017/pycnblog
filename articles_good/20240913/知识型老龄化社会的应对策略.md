#### 1. 老年人健康数据分析

**题目：** 给定一组老年人健康数据，包括身高、体重、血压、血糖等指标，编写程序计算老年人的健康指数，并根据健康指数划分健康等级。

**答案：**

```python
def calculate_health_index(height, weight, blood_pressure, blood_sugar):
    bmi = weight / (height / 100) ** 2
    health_index = (bmi + blood_pressure + blood_sugar) / 3
    return health_index

def classify_health_level(health_index):
    if health_index < 60:
        return "极差"
    elif health_index < 70:
        return "较差"
    elif health_index < 80:
        return "一般"
    elif health_index < 90:
        return "良好"
    else:
        return "优秀"

# 示例数据
height = 170  # 身高（厘米）
weight = 65   # 体重（千克）
blood_pressure = 120  # 血压（毫米汞柱）
blood_sugar = 5.5  # 血糖（mmol/L）

health_index = calculate_health_index(height, weight, blood_pressure, blood_sugar)
health_level = classify_health_level(health_index)

print("健康指数：", health_index)
print("健康等级：", health_level)
```

**解析：** 该程序首先计算健康指数，然后根据健康指数划分健康等级。可以用于评估老年人的健康状况。

#### 2. 老年人医疗费用预算

**题目：** 给定一组老年人医疗费用数据，包括门诊费用、住院费用、药品费用等，编写程序计算老年人的总医疗费用，并按照费用类型分类统计。

**答案：**

```python
def calculate_total_medical_expenses(medical_expenses):
    total_expenses = 0
    expense_types = {}

    for expense in medical_expenses:
        total_expenses += expense['amount']
        expense_type = expense['type']
        if expense_type in expense_types:
            expense_types[expense_type] += expense['amount']
        else:
            expense_types[expense_type] = expense['amount']

    return total_expenses, expense_types

# 示例数据
medical_expenses = [
    {'type': '门诊', 'amount': 200},
    {'type': '住院', 'amount': 3000},
    {'type': '药品', 'amount': 500},
]

total_expenses, expense_types = calculate_total_medical_expenses(medical_expenses)

print("总医疗费用：", total_expenses)
print("费用分类统计：", expense_types)
```

**解析：** 该程序计算老年人的总医疗费用，并按照费用类型进行分类统计，可以用于老年人医疗费用的管理和分析。

#### 4. 老年人活动记录分析

**题目：** 给定一组老年人活动记录数据，包括活动类型、活动时长等，编写程序分析老年人的活动情况，并生成活动报告。

**答案：**

```python
def analyze_activity_records(activity_records):
    activity_types = {}
    total_duration = 0

    for record in activity_records:
        activity_type = record['type']
        duration = record['duration']
        total_duration += duration

        if activity_type in activity_types:
            activity_types[activity_type] += duration
        else:
            activity_types[activity_type] = duration

    return activity_types, total_duration

def generate_activity_report(activity_types, total_duration):
    report = "活动报告：\n"
    report += "总活动时长：{}分钟\n".format(total_duration)
    for activity_type, duration in activity_types.items():
        report += "{}：{}分钟\n".format(activity_type, duration)

    return report

# 示例数据
activity_records = [

    {'type': '看电视', 'duration': 120},

activity_report = generate_activity_report(activity_types, total_duration)
```

```python
def calculate_mental_health_index(depression, anxiety):
    mental_health_index = (1 - depression) * (1 - anxiety)
    return mental_health_index

def assess_mental_health_state(mental_health_index):
    if mental_health_index < 0.4:
        return "心理健康状况较差"
    elif mental_health_index < 0.6:
```
