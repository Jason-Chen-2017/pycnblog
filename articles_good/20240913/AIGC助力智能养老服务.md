#### 2. 如何在智能养老系统中实现健康监测？

**算法编程题：** 设计一个健康监测系统，能够实时收集老年人的健康数据，包括心率、血压、血氧等指标，并分析这些数据，提供健康建议。

**答案：**

```python
import random

class HealthMonitor:
    def __init__(self):
        self.heart_rates = []
        self.blood_pressures = []
        self.oxygen_levels = []

    def collect_data(self, heart_rate, blood_pressure, oxygen_level):
        self.heart_rates.append(heart_rate)
        self.blood_pressures.append(blood_pressure)
        self.oxygen_levels.append(oxygen_level)

    def analyze_data(self):
        avg_heart_rate = sum(self.heart_rates) / len(self.heart_rates)
        avg_blood_pressure = sum(self.blood_pressures) / len(self.blood_pressures)
        avg_oxygen_level = sum(self.oxygen_levels) / len(self.oxygen_levels)

        if avg_heart_rate > 100:
            return "您的平均心率为 {}，建议您注意休息和运动。".format(avg_heart_rate)
        elif avg_blood_pressure > 140 or avg_blood_pressure < 90:
            return "您的平均血压为 {}，建议您咨询医生。".format(avg_blood_pressure)
        elif avg_oxygen_level < 95:
            return "您的血氧浓度为 {}%，建议您保持空气流通，避免缺氧。".format(avg_oxygen_level)
        else:
            return "您的健康状况良好，继续保持。"

# 测试
monitor = HealthMonitor()
monitor.collect_data(90, 120, 98)
monitor.collect_data(110, 130, 95)
monitor.collect_data(100, 110, 97)

print(monitor.analyze_data())
```

**解析：** 该健康监测系统可以收集心率、血压和血氧等健康数据，并计算这些数据的平均值。根据平均值提供相应的健康建议。
