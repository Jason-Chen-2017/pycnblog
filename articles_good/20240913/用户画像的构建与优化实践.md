**输入：**
```go
users := [][]int{
    {1, 1, 2, 3},
    {1, 2, 2, 3, 5},
    {2, 3, 4},
    {3, 5},
}
```

**输出：**
```go
[
    {"商品1", 2, 2.0},
    {"商品2", 1, 2.0},
    {"商品3", 1, 1.0},
    {"商品5", 1, 1.0},
]
```

**答案：**
```go
package main

import (
    "fmt"
)

func main() {
    users := [][]int{
        {1, 1, 2, 3},
        {1, 2, 2, 3, 5},
        {2, 3, 4},
        {3, 5},
    }

    result := make(map[int]struct{})
    for _, user := range users {
        for _, item := range user {
            result[item] = struct{}{}
        }
    }

    for item := range result {
        count := 0
        for _, user := range users {
            if contains(user, item) {
                count++
            }
        }
        avgCount := float64(count) / float64(len(users))
        fmt.Printf("%d\t%d\t%.1f\n", item, count, avgCount)
    }
}

func contains(slice []int, item int) bool {
    for _, value := range slice {
        if value == item {
            return true
        }
    }
    return false
}
```

**解析：** 该算法首先统计所有用户购买的商品种类，然后对每种商品计算其被购买的总次数和平均购买次数。

**题目：** 给定一个用户购买历史数据列表，为每个用户生成一个喜好得分。

**输入：**
```go
users := [][]int{
    {1, 1, 2, 3},
    {1, 2, 2, 3, 5},
    {2, 3, 4},
    {3, 5},
}
```

**输出：**
```go
[
    {1, 5.0},
    {2, 4.0},
    {3, 3.0},
    {5, 2.0},
]
```

**答案：**
```go
package main

import (
    "fmt"
)

func main() {
    users := [][]int{
        {1, 1, 2, 3},
        {1, 2, 2, 3, 5},
        {2, 3, 4},
        {3, 5},
    }

    result := make(map[int]map[int]float64)
    for _, user := range users {
        counts := make(map[int]int)
        for _, item := range user {
            counts[item]++
        }
        for item, count := range counts {
            if result[user[0]] == nil {
                result[user[0]] = make(map[int]float64)
            }
            result[user[0]][item] = float64(count)
        }
    }

    for user, items := range result {
        total := 0.0
        for _, count := range items {
            total += count
        }
        avg := total / float64(len(items))
        fmt.Printf("%d\t%.1f\n", user, avg)
    }
}
```

**解析：** 该算法首先对每个用户的购买历史进行计数，然后计算每个商品的购买次数占总购买次数的比例，以此作为喜好得分。
