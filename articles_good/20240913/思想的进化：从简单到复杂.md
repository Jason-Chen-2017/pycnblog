##### 题目 1: 简单递归计算阶乘

**题目描述：** 编写一个函数，使用递归的方式计算一个整数的阶乘。

**代码示例：**

```python
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

print(factorial(5)) # 输出 120
```

**答案解析：**

- 这个函数使用递归的方式计算阶乘，当 n 等于 0 时，返回 1，否则返回 n 乘以 n-1 的阶乘。
- 阶乘的定义是：n! = n * (n-1)!，其中 n! 表示 n 的阶乘。

##### 题目 2: 求两个数的最大公约数

**题目描述：** 编写一个函数，使用递归的方式求两个数的最大公约数。

**代码示例：**

```python
def gcd(a, b):
    if b == 0:
        return a
    return gcd(b, a % b)

print(gcd(12, 18)) # 输出 6
```

**答案解析：**

- 这个函数使用辗转相除法（也称为欧几里得算法）求最大公约数，当 b 等于 0 时，返回 a，否则返回 b 和 a 的余数的最大公约数。
- 最大公约数是两个数的公共因数中最大的那个数。

##### 题目 6: 求两个有序数组的合并

**代码示例：**

```python
def merge_sorted_arrays(nums1, nums2):
    m, n = len(nums1), len(nums2)
    nums1.extend(nums2)
    nums1.sort()
    return nums1[:m+n]

print(merge_sorted_arrays([1, 3, 5], [2, 4, 6])) # 输出 [1, 2, 3, 4, 5, 6]
```

**答案解析：**

- 有序数组是指数组中的元素按照从小到大的顺序排列。

##### 题目 7: 求两个字符串的最长公共子序列

**题目描述：** 编写一个函数，求两个字符串的最长公共子序列。

**代码示例：**

```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n+1) for _ in range(m+1)]
    for i in range(1, m+1):
        for j in range(1, n+1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

print(longest_common_subsequence("abcde", "ace")) # 输出 3
```

**答案解析：**

- 这个函数使用动态规划的方法求两个字符串的最长公共子序列，首先创建一个二维数组 `dp`，其中 `dp[i][j]` 表示 `s1` 的前 `i` 个字符和 `s2` 的前 `j` 个字符的最长公共子序列的长度。
- 动态规划是一种常用的算法设计技巧，用于求解最优化问题，通过维护一个状态表，将复杂的问题分解为多个子问题，并利用子问题的解来构建原问题的解。
