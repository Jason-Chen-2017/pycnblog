### 2.1 马尔可夫链

马尔可夫链是一种特殊的随机过程，其未来的状态仅取决于当前状态，而与过去的状态无关。这种“无记忆性”使得马尔可夫链易于分析和模拟。马尔可夫链在许多领域中都有应用，例如：

## 4. 数学模型和公式详细讲解举例说明

### 4.1 马尔可夫链的数学模型

马尔可夫链可以用转移概率矩阵来描述。转移概率矩阵 $P$ 是一个 $n \times n$ 的矩阵，其中 $P_{ij}$ 表示从状态 $i$ 转移到状态 $j$ 的概率。

$$
P = \begin{bmatrix}
P_{11} & P_{12} & \cdots & P_{1n} \\
P_{21} & P_{22} & \cdots & P_{2n} \\
\vdots & \vdots & \ddots & \vdots \\
P_{n1} & P_{n2} & \cdots & P_{nn}
\end{bmatrix}
$$

### 4.2 蒙特卡洛方法的数学原理

蒙特卡洛方法的数学原理是基于大数定律。大数定律指出，随着随机试验次数的增加，样本均值将收敛于总体均值。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 Python 代码示例：模拟马尔可夫链

```python
import numpy as np

# 定义状态空间和转移概率矩阵
states = ['A', 'B', 'C']
P = np.array([[0.5, 0.3, 0.2],
              [0.2, 0.6, 0.2],
              [0.1, 0.2, 0.7]])

# 模拟马尔可夫链
current_state = 'A'
for i in range(10):
    next_state = np.random.choice(states, p=P[states.index(current_state)])
    print(f"Step {i+1}: {current_state} -> {next_state}")
    current_state = next_state
```

### 5.2 Python 代码示例：使用蒙特卡洛方法计算π

```python
import random

def monte_carlo_pi(n):
    count = 0
    for i in range(n):
        x = random.random()
        y = random.random()
        if x**2 + y**2 <= 1:
            count += 1
    return 4 * count / n

# 计算π的近似值
pi_approx = monte_carlo_pi(10000)
print(f"π ≈ {pi_approx}")
```
