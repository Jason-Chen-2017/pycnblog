# 2.基本概念术语说明
为了更好的理解人工智能技术在制造业的应用情况，我们需要先了解一些基本的概念、术语。这些术语将帮助我们更好地理解实验室的团队和实验项目。
## 2.1 概念
* 机器学习(Machine Learning): 机器学习是指计算机通过数据、经验或反馈，自动发现数据的模式并进行改进，使得计算机的性能得到提升。一般情况下，机器学习用于分类、预测、聚类等任务，可以根据输入的数据进行训练生成模型。
* 深度学习(Deep Learning): 深度学习是指多层次神经网络，由多个隐藏层组成，每一层都是一个隐含层，包含一堆神经元，并且具有非线性激活函数。深度学习可以处理高维度和复杂的数据。
* 人工智能(Artificial Intelligence)：人工智能是一个与生俱来的能力，可以通过某些机器实现，如自主驾驶汽车、文字识别等。人工智能通常包含三个层次：符号主义(Symbolic AI)、连接主义(Connectionist AI)、规则系统主义(Rule-based Systems)。
* 强化学习(Reinforcement Learning): 强化学习是一种基于监督学习的算法，它试图最大化环境的奖励，同时也会在与环境互动时做出反应。强化学习可以用于制造过程控制、机器人控制、自动驾驶、游戏 playing agent等领域。
* 数据驱动(Data Driven): 数据驱动是指从大量的数据中学习并应用知识、技能，以取得成功。数据驱动的关键是利用数据的价值，挖掘数据中隐藏的潜在关系，提升效率。
* 自然语言理解(Natural Language Understanding): 自然语言理解(NLU)是指计算机系统能够理解自然语言。它的目标是在不依赖于语言资源、语境、语法结构等形式化语言的情况下，将自然语言转换成机器可读的形式，并运用计算的方式进行分析、理解和表达。NLU可用于多种应用场景，包括聊天机器人、语言理解、意图识别、文本摘要、机器翻译等。
* 虚拟现实(Virtual Reality): 虚拟现实(VR/AR)是指通过计算机模拟真实世界的情景和空间，通过眼镜、耳机、头戴设备、移动控制器、智能手机等进行沟通互动，最终达到增强现实的效果。
## 2.2 术语
* 任务(Task): 是指特定领域内的一个具体的问题，它需要解决的方法可能涉及不同的技术、工具、算法等。如图像分类任务、语音识别任务等。
* 数据集(Dataset): 是指包含输入输出对的训练数据，用于训练机器学习模型。数据集分为有监督数据集和无监督数据集。
* 模型(Model): 是指输入数据经过一系列转换后得到的结果，如分类器、回归模型等。
* 超参数(Hyperparameter): 是指模型训练过程中的参数，它影响模型的训练速度、精确度和稳定性。
* 推理(Inference): 是指模型运行过程，即模型应用到新的输入上产生输出。
* 评估指标(Evaluation Metrics): 是指衡量模型性能的标准指标。如准确率、召回率、F1值等。
* 优化器(Optimizer): 是指模型训练过程中使用的算法，用于更新模型参数，以减少损失函数的值。
* 蒸馏(Distillation): 是指一个大模型的压缩，得到小模型，目的是提升推理效率。
* 生成模型(Generative Model): 是指可以生成样本的模型，如VAE、GAN等。
* 编码器(Encoder): 是指对输入数据进行特征抽取的模型，如CNN等。
* 解码器(Decoder): 是指对编码后的表示进行逆向重构的模型。
* 注意力机制(Attention Mechanism): 是指基于注意力的模型，可以关注输入序列的不同部分，以提升模型的表现力。
* 循环神经网络(RNN): 是指具有内部循环连接的神经网络，适合处理序列数据。
* 长短期记忆网络(LSTM): 是一种特殊的RNN，它采用门结构，可以更好地捕获时间上的关联。
* 门控循环单元(GRU): 是一种更简单的RNN，它采用重置门和更新门，可以有效缓解梯度消失问题。

人工智能在制造业的应用主要分为以下几类：
1. 物料智能化：制造企业能够对原材料和辅助物料的品质和质量进行全面跟踪管理，自动化分配、整合和管理生产中的耗材和原料，提升生产效率。
2. 生产控制智能化：生产企业可以通过机器人等方式实现生产过程的自动化，进而降低成本，节省资源，提升竞争力。
3. 工艺流程智能化：工艺企业通过人工智能技术及其协同配套的工艺装备，智能化设计、调整工艺流程，实现无人化生产，缩短作业周期，提高产品品质。
4. 服务智能化：服务企业通过数字化手段实现供应链的整合，精细化的生产和营销的管控，提升客户满意度，提升竞争力。
5. 零售智能化：零售企业通过物联网、无人机等技术将实体店变成数字化平台，实现线下零售的自动化，提升效率，提高盈利能力。

## 3.1 物料智能化
### 3.1.1 ASSEMBLYAI：面向成品加工领域的智能物料管理系统
#### （1）功能概述
ASSEMBLYAI智能物料管理系统可以将为客户提供高质量的商品所需的全部原材料和辅助物料整合，自动分配、整合和管理生产中的耗材和原料。通过基于机器学习的图像识别技术，识别并标识物料的形状、颜色、尺寸、结构等信息，实现整体物料管理。

系统支持人员通过工单系统输入生产指令，系统根据工单指导生产车间生产，并提供实时的物料库存监控报告。系统采用基于RFID技术的智能卡系统进行物料信息的实时同步。

通过将物料的三维数据转化成二维图像，通过人工智能技术、图像处理技术和计算机视觉算法，将数据转换成物料的位置信息，保障生产的快速和精准。

#### （2）操作步骤
1. 人像识别：物料来源于整个产品的设计思路，需要先进行人像识别，确定产品的设计者、团队人员等信息。

2. 图像识别：物料的数量是所有原始材料和辅助物料的重要组成部分，需要根据产品的要求对所有的物料进行自动识别，包括各种大小、形状、纹理、样式、颜色等信息，然后将其分配给不同的部门进行管理。

3. 信息获取：自动识别后的物料信息需要存储起来，便于后续的管理和分配。

4. 物料库存查询：在需要查询物料库存的时候，可以向系统提供相应的信息，系统将返回当前物料库存的详细信息，方便管理人员查询。

5. 物料分配：系统可以根据需求完成智能化的物料分配工作。

总体来说，ASSEMBLYAI系统是一个具有全方位、灵活、智能的物料管理系统。通过机器学习算法，自动识别物料、图像处理、RFID技术、人脸识别等技术，实现对物料的全面整合和管理。
### 3.1.2 PROMOTECH：面向行业领域的物料智能推荐系统
#### （1）功能概述
PROMOTECH是面向行业领域的物料智能推荐系统，可以提升公司产品的物料推荐效果，促进公司产品的品牌形象建设。系统可将公司库存的物料数据进行收集、分析和挖掘，根据用户喜好和偏好生成符合用户需求的物料推荐列表。

该系统通过物料的颜色、形状、价格等属性，对物料进行排序、筛选，为用户提供个性化的推荐物料列表，满足用户的购买欲望。

#### （2）操作步骤
1. 数据采集：通过网站、微信公众号、应用等渠道收集用户的购买行为数据。

2. 特征工程：对收集到的用户数据进行特征工程，提取用户的喜好信息、偏好信息和购买行为信息等。

3. 推荐算法：根据用户的购买偏好、喜好等特征信息，选择合适的推荐算法，生成推荐物料列表。

4. 流程控制：系统可设置不同的推荐流程，进行测试、调优，保证推荐效果的稳定性。

总体来说，PROMOTECH系统是一个具有实用的、高效的物料推荐系统。通过收集、分析用户的购买行为数据，为用户提供个性化的推荐物料列表，提升用户的购买欲望，促进公司产品的品牌形象建设。
### 3.1.3 PRODUCEME：人才培养与晋升管理的智能化工具
#### （1）功能概述
PRODUCEME是一个人才培养与晋升管理的智能化工具，能够实时收集企业员工的个人信息，进行人才匹配、挖掘、招聘等流程，为企业搭建最具针对性的人才平台。

通过智能化推荐系统，PRODUCEME能够精准匹配职位和技能要求，建议匹配的人才。除此之外，PRODUCEME还具备人才池管理、岗位推荐、招聘启事发布等功能，打通人才匹配、聘用、绩效评价等环节，为企业搭建了一个极具价值的“人才工厂”。

#### （2）操作步骤
1. 登陆平台：登录平台进行人才搜索、投递等流程，在登录之后，即可进行相关业务操作。

2. 职位推荐：PRODUCEME可以根据用户的工作经历、职称、技能等要求，提供职位推荐。

3. 招聘启事发布：平台管理员可上传招聘启事，平台会将启事发送至内部群聊，接受用户的报名。

4. 投递简历：用户填写完简历，上传至平台进行投递。

5. 审核投递：系统审核投递信息，确认是否能够收到。

6. 晋升审批：平台可以审批用户的晋升申请，完成相关流程，为员工加薪或升职。

总体来说，PRODUCEME系统是一个具有灵活性、智能化、适应性的人才匹配系统。通过人才搜索、职位匹配、招聘启事发布、投递简历等功能，为企业提供最全面的人才信息和管理服务，满足员工就业需求。
## 3.2 生产控制智能化
### 3.2.1 CONTROLIO：在线管理生产设备的控制系统
#### （1）功能概述
CONTROLIO是一套在线管理生产设备的控制系统，能够根据实时数据自动控制生产设备，降低成本，提高生产效率。该系统能够实时监控设备的运行状态，识别异常情况，及时进行排查和维护。

CONTROLIO采用云计算技术，将云端的数据传输至本地控制终端，实现设备的远程监控。通过人工智能技术，控制终端可以识别和理解生产过程中的物料、工件、设备和信息，并据此控制设备的运行。

#### （2）操作步骤
1. 设备注册：用户在系统中注册生产设备，配置设备的相关信息。

2. 配置规则：用户在注册设备时，可以配置设备的控制规则，如电压、电流、温度等。

3. 实时监控：当生产设备运行发生异常时，控制终端将实时采集设备的实时数据，并通知系统管理员进行异常排查。

4. 远程控制：当系统管理员确认异常时，可以远程控制设备的启动和停止。

5. 历史数据查看：用户可以查看历史数据，了解生产过程中的运行状态，识别异常，提升工作效率。

总体来说，CONTROLIO系统是一个具有高度实时性、精准性的生产设备控制系统。通过使用云计算、机器学习等技术，能够实现对生产设备的全面监控、控制和故障诊断，并可直观呈现生产过程中的关键数据。
### 3.2.2 HARMONIX：数字化流程管理的智能系统
#### （1）功能概述
HARMONIX是一个数字化流程管理的智能系统，能够实现一站式的流程管理，提升企业的工作效率。该系统能够根据组织的标准和流程需求，将流程、文档、活动、事务等转换成自动化的管理系统。

HARMONIX采用基于机器学习的自动化配置方案，根据历史数据和管理人员的指导，对流程进行优化配置，实现标准化的流程执行，提升工作效率。同时，系统还提供了人员管理、资源管理、报警中心、工作流等功能，可以实现更多的协同管理。

#### （2）操作步骤
1. 配置流程：用户可以在系统中配置流程模板、条件节点、控制节点和文件节点等，实现流程管理的自动化。

2. 执行流程：当用户触发流程执行时，系统将自动解析模板和上下文数据，生成执行计划。

3. 查看结果：系统将实时显示流程的执行情况，并按照规定的通知频率发送消息给相关人员。

4. 操作日志记录：系统记录用户的操作记录，以便跟踪用户的操作轨迹。

5. 文件管理：用户可以在系统中管理文件的上传下载，查看文件历史版本等。

总体来说，HARMONIX系统是一个具有高度自动化、标准化、智能化的流程管理系统。通过自动化配置方案、大数据分析、深度学习算法等技术，系统可以实现流程标准化、流程自动化、自动化报告生成、协同管理等功能，提升企业的工作效率。
### 3.2.3 SUPERIOR：精益制造的生命周期管理系统
#### （1）功能概述
SUPERIOR是一个精益制造的生命周期管理系统，可以实时跟踪和管理产品的生命周期。该系统能够建立起完整的产品生命周期数据库，实时记录产品的生产、加工、交付、使用、废弃等过程，全程监控产品的成熟度，进行产品优化和生命周期管理。

SUPERIOR采用基于云计算的知识图谱、机器学习算法等技术，对制造企业的物料、工序、工艺路线、材料等信息进行智能分析和挖掘，建立起产品生命周期数据图谱，提供产品的生命周期风险分析、规划、优化、追溯等全过程的管理和优化服务。

#### （2）操作步骤
1. 生产管理：用户可以在系统中配置生产订单、生产设备、生产人员、质量标准、产品信息等，完成生产流程的管理。

2. 产品管理：系统能够将生产过程中的物料、工序、工艺路线等信息自动提取、匹配、分析、归纳，形成产品生命周期数据图谱。

3. 生命周期管理：通过分析产品生命周期数据图谱，用户可以找到优化的环节，提升产品的生命周期质量，为生产企业提供管理的决策依据。

4. 质量管理：对产品的生命周期数据图谱，系统可实时计算产品的生命周期质量，并通过合理的措施，防止产品出现质量问题。

5. 发货管理：用户可以创建发货单、查看订单状态、编辑发货信息，实时跟踪产品的运输和送达过程。

总体来说，SUPERIOR系统是一个具有高度自动化、智能化的产品生命周期管理系统。通过利用云计算、大数据分析等技术，系统能够对制造企业的生产和产品信息进行全面挖掘，形成完整的产品生命周期数据图谱，为生产企业提供全过程的生命周期管理和优化服务。

CIRCUS是一个工艺工序跟踪管理的智能化系统，可以自动化管理工艺工序，实现流程精准化、管理规范化。该系统能够根据产品的工艺路线图和工序控制清单，对产品的工艺工序进行自动化跟踪和管理。

CIRCUS采用大数据分析和信息技术，对制造企业的生产数据进行深度挖掘，形成产品生命周期数据图谱，提取产品的关键工序，通过分析产品的工艺工序，提升产品的制造效率，节省生产成本，提高企业的竞争力。

#### （2）操作步骤
1. 配置工艺路线：用户可以在系统中配置产品的工艺路线图，定义工序的执行顺序、工艺工时、标准规范、维护规范等。

5. 管理报告：系统将实时生成产品生产过程的管理报告，统计生产过程中的工序情况、效率、标准遵守情况等。

总体来说，CIRCUS系统是一个具有高度自动化、智能化的工序跟踪管理系统。通过分析产品的工艺工序、制造数据和工序控制清单，系统可以对工序进行精准化管理，提升制造企业的生产效率，节省生产成本，提高竞争力。
### 3.3.2 TACONIC：自动化质量管理的企业级系统
#### （1）功能概述
TACONIC是一个自动化质量管理的企业级系统，可以有效实现质量管控、过程可靠性、经济效益之间的平衡。该系统通过建立完整的生产数据图谱，将企业产品的质量管理、过程管理和合同管理连接起来，为产品的生产过程提供全方位的管理服务。

TACONIC采用云计算技术，将大量的生产数据汇集在一起，构建统一的质量管理体系，实现系统数据的集成和分析，形成全面、准确的质量数据。

#### （2）操作步骤
1. 数据采集：用户可以在系统中配置和自定义采集参数，实现自动数据采集、分析和存储。

2. 数据分析：系统通过建立数据模型，对生产数据进行聚合、分析和挖掘，形成完整的产品生命周期数据图谱。

3. 质量管理：系统能够自动识别生产数据中的质量问题，形成质量管理报告。

4. 过程管理：系统将产品生命周期数据图谱中的质量问题转化成生产过程中的缺陷，进行问题跟踪，提升生产过程的可靠性。

5. 合同管理：系统通过对合同信息、质量管理报告进行整合，形成完整的合同管理系统，为企业提供安全、可靠、经济、及时的合同管理服务。

总体来说，TACONIC系统是一个具有高度自动化、智能化的质量管理系统。通过采用云计算、大数据分析等技术，系统能够实现对企业的生产数据进行全面挖掘，形成完整的产品生命周期数据图谱，为企业提供完整的质量管理、过程管理和合同管理服务，提升企业的产品质量管理能力，提高生产绩效和经济效益。
