### 面试题：最长公共子序列

#### 算法原理

最长公共子序列（Longest Common Subsequence，LCS）是指两个序列中，最长的公共子序列。LCS问题是一个经典的动态规划问题。

#### 具体操作步骤

1. 创建一个二维数组`dp`，用于存储子问题的解。
2. 初始化第一行和第一列的元素为1，表示空序列与当前元素的公共子序列长度为1。
3. 从第二行第二列开始，遍历二维数组，根据状态转移方程计算每个元素的公共子序列长度。
4. 找到最大值对应的元素，即为最长公共子序列的长度。

#### 数学模型

状态转移方程为：
$$
dp[i][j] = \begin{cases}
dp[i-1][j-1] + 1, & \text{若} a[i-1] = b[j-1]; \\
\max(dp[i-1][j], dp[i][j-1]), & \text{否则}.
\end{cases}
$$

### 面试题：二分查找

#### 数学模型

二分查找是一种高效的查找算法，其核心思想是每次将查找范围缩小一半。

#### 公式

二分查找的时间复杂度为O(log n)，其中n为待查找序列的长度。

#### 详细讲解

1. 首先，将待查找序列排序。
2. 设待查找序列为A，目标值为x，初始左右边界为l = 0，r = n - 1。
3. 当l < r时，进入循环。
4. 计算中间位置mid = (l + r) / 2。
5. 若A[mid] = x，则查找成功，返回mid。
6. 若A[mid] > x，则将右边界更新为mid - 1。
7. 若A[mid] < x，则将左边界更新为mid + 1。
8. 循环结束后，返回-1表示查找失败。

#### 举例说明

假设待查找序列为A = [1, 3, 5, 7, 9]，目标值为x = 7。

1. 初始左右边界为l = 0，r = 4。
2. 计算中间位置mid = (0 + 4) / 2 = 2，A[mid] = 5 > x，将右边界更新为1。
3. 计算中间位置mid = (0 + 1) / 2 = 0，A[mid] = 1 < x，将左边界更新为1。
4. 计算中间位置mid = (1 + 1) / 2 = 1，A[mid] = 3 < x，将左边界更新为2。
5. 计算中间位置mid = (2 + 2) / 2 = 2，A[mid] = 7 = x，查找成功，返回mid = 2。

```python
def lcs(a, b):
    n, m = len(a), len(b)
    dp = [[0] * (m + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for j in range(1, m + 1):
            if a[i - 1] == b[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[n][m]

a = [1, 2, 3, 4, 5]
b = [2, 4, 5, 6, 7]
print(lcs(a, b))  # 输出：3
```

2. 初始化二维数组`dp`，用于存储子问题的解。
3. 使用两层循环遍历`dp`数组，根据状态转移方程计算每个元素的公共子序列长度。
4. 返回最长公共子序列的长度。

## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 二分查找

#### 数学模型

二分查找（Binary Search）是一种高效的查找算法，其核心思想是每次将查找范围缩小一半。

#### 公式

- 时间复杂度：$O(\log n)$，其中$n$为待查找序列的长度。

#### 详细讲解

1. **排序**：确保待查找序列已经排序。
2. **初始化**：设置左右边界$l = 0$，$r = n - 1$。
3. **循环**：当$l < r$时，计算中间位置$m = (l + r) / 2$。
4. **比较**：若$A[m] == x$，则查找成功，返回$m$；若$A[m] > x$，则更新$r = m - 1$；若$A[m] < x$，则更新$l = m + 1$。

#### 举例说明

假设待查找序列为$A = [1, 3, 5, 7, 9]$，目标值为$x = 7$。

1. $l = 0$，$r = 4$，$m = (0 + 4) / 2 = 2$，$A[m] = 5 > x$，更新$r = 2 - 1 = 1$。
2. $l = 0$，$r = 1$，$m = (0 + 1) / 2 = 0$，$A[m] = 1 < x$，更新$l = 0 + 1 = 1$。
3. $l = 1$，$r = 1$，$m = (1 + 1) / 2 = 1$，$A[m] = 3 < x$，更新$l = 1 + 1 = 2$。
4. $l = 2$，$r = 1$（此时$l > r$），查找失败，返回-1。
