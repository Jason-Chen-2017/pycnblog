### 4. 数学模型和公式 & 详细讲解 & 举例说明

#### 4.1 莫尔斯函数

莫尔斯函数是莫尔斯理论的核心概念之一。对于一个非线性动力系统，其莫尔斯函数可以表示为：

\[ f(\lambda) = \frac{1}{2} \sum_{i=1}^{n} (\lambda_i^2 - \lambda_{i+1}^2) \]

其中，\(\lambda_1, \lambda_2, ..., \lambda_n\) 是系统在不同参数下的临界点。

**举例说明**：

假设我们有一个简单的二阶非线性动力系统：

\[ \dot{x} = -x^3 + x \]
\[ \dot{y} = y^3 - y \]

首先，我们需要计算系统在不同参数下的临界点。通过数值计算，我们可以得到以下临界点：

\[ \lambda_1 = 0, \lambda_2 = 1, \lambda_3 = -1 \]

然后，我们可以计算莫尔斯函数：

\[ f(\lambda) = \frac{1}{2} \left[ (0^2 - 1^2) + (1^2 - (-1)^2) + ((-1)^2 - 0^2) \right] = \frac{1}{2} \left[ -1 + 1 - 1 \right] = -\frac{1}{2} \]

由此可见，莫尔斯函数的值在所有临界点处均为负值，这表明系统在这些临界点附近是稳定的。

#### 4.2 莫尔斯不等式

莫尔斯不等式是莫尔斯理论的重要工具，它将莫尔斯函数与系统拓扑性质联系起来。莫尔斯不等式可以表示为：

\[ \text{deg}(f) \leq \sum_{i=1}^{n} (-1)^{i+1} \text{sign}(f(\lambda_i)) \]

其中，\(\text{deg}(f)\) 是莫尔斯函数的指数，\(\text{sign}(f(\lambda_i))\) 是莫尔斯函数在临界点 \(\lambda_i\) 处的符号。

**举例说明**：

我们继续使用上面的二阶非线性动力系统，其莫尔斯函数已经计算得到 \(f(\lambda) = -\frac{1}{2}\)。根据莫尔斯不等式，我们有：

\[ \text{deg}(f) \leq \sum_{i=1}^{n} (-1)^{i+1} \text{sign}(f(\lambda_i)) = (-1)^{1+1} \text{sign}(-\frac{1}{2}) = 1 \]

由此可见，莫尔斯函数的指数不超过 1，这表明系统在所有临界点处的拓扑性质是有限的。

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

# 定义非线性动力系统
def model(y, t):
    x, y = y
    dxdt = -x**3 + x
    dydt = y**3 - y
    return [dxdt, dydt]

# 计算临界点
def find_critical_points():
    critical_points = []
    t = np.linspace(0, 10, 1000)
    for x0 in [-2, -1, 1, 2]:
        y0 = [x0, 0]
        sol = odeint(model, y0, t)
        x = sol[:, 0]
        y = sol[:, 1]
        if np.count_nonzero(np.diff(np.sign(x))) > 1:
            critical_points.append(x0)
    return critical_points

# 构造莫尔斯函数
def morse_function(critical_points):
    morse_func = 0
    for i in range(len(critical_points) - 1):
        morse_func += (critical_points[i]**2 - critical_points[i+1]**2)
    return morse_func

# 应用莫尔斯不等式
def morse_inequality(morse_func, critical_points):
    sign = np.sign(morse_func(critical_points))
    inequality = np.sum([(-1)**(i+1) * sign[i] for i in range(len(sign))])
    return inequality

# 绘制莫尔斯流形
def plot_morse_flow(critical_points):
    x, y = np.meshgrid(np.linspace(-2, 2, 100), np.linspace(-2, 2, 100))
    z = np.zeros_like(x)
    for cp in critical_points:
        z[x == cp] = 1
    plt.figure()
    plt.contour(x, y, z, levels=2, colors='r')
    plt.plot(x, y, 'k')
    plt.show()

# 执行示例
if __name__ == "__main__":
    critical_points = find_critical_points()
    morse_func = morse_function(critical_points)
    inequality = morse_inequality(morse_func, critical_points)
    print("Critical points:", critical_points)
    print("Morse function:", morse_func)
    print("Morse inequality:", inequality)
    plot_morse_flow(critical_points)
```

2. **临界点求解**：通过求解二阶线性动力系统的数值解，我们找到系统在不同参数下的临界点。

3. **莫尔斯函数构造**：根据临界点的信息，我们构造了莫尔斯函数。
