##### 1. 腾讯面试题：深度优先搜索（DFS）与非深度优先搜索（BFS）的区别

**题目：** 请解释深度优先搜索（DFS）与非深度优先搜索（BFS）的区别，并分别描述其实现思路。

**答案：** 深度优先搜索（DFS）和非深度优先搜索（BFS）是两种常见的图遍历算法。它们的区别在于：

- **遍历顺序：** DFS 是先深后宽，而 BFS 是先宽后深。
- **实现方式：** DFS 通常使用递归来实现，而 BFS 使用循环加队列实现。

**解析：**

**DFS 实现：**

```go
func dfs(graph [][]int, visited []bool, node int) {
    if visited[node] {
        return
    }
    visited[node] = true
    for _, neighbor := range graph[node] {
        dfs(graph, visited, neighbor)
    }
}
```

**BFS 实现：**

```go
func bfs(graph [][]int, start int) {
    visited := make([]bool, len(graph))
    queue := []int{start}
    for len(queue) > 0 {
        node := queue[0]
        queue = queue[1:]
        if visited[node] {
            continue
        }
        visited[node] = true
        for _, neighbor := range graph[node] {
            queue = append(queue, neighbor)
        }
    }
}
```

##### 2. 阿里巴巴面试题：如何实现一个单例模式？

**题目：** 请使用 Go 语言实现一个单例模式，并解释其原理。

**答案：** 单例模式是一种设计模式，确保一个类只有一个实例，并提供一个全局访问点。在 Go 语言中，可以使用懒汉式和饿汉式两种实现方式：

**解析：**

**懒汉式：**

```go
var instance *Singleton

func GetInstance() *Singleton {
    if instance == nil {
        instance = &Singleton{}
    }
    return instance
}
```

**饿汉式：**

```go
type Singleton struct{}

var instance Singleton

func GetInstance() *Singleton {
    return &instance
}
```

#### 二、算法编程题库

##### 1. 字节跳动算法题：最长公共子序列（LCS）

**题目：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。

**答案：** 最长公共子序列（LCS）是指两个序列中公共子序列中最长的序列。可以使用动态规划（DP）的方法求解。

**解析：**

```go
func longestCommonSubsequence(str1 string, str2 string) int {
    m, n := len(str1), len(str2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if str1[i-1] == str2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

##### 2. 腾讯算法题：最长递增子序列（LIS）

**题目：** 给定一个无序数组 `nums`，找出最长递增子序列的长度。

**答案：** 最长递增子序列（LIS）是指一个序列，它在原序列中是递增的，并且是原序列中最长的递增子序列。

**解析：**

```go
func lengthOfLIS(nums []int) int {
    dp := make([]int, len(nums))
    for i := range dp {
        dp[i] = 1
    }
    for i := 1; i < len(nums); i++ {
        for j := 0; j < i; j++ {
            if nums[i] > nums[j] {
                dp[i] = max(dp[i], dp[j]+1)
            }
        }
    }
    return max(dp...)
}

func max(nums ...int) int {
    res := nums[0]
    for _, num := range nums {
        if num > res {
            res = num
        }
    }
    return res
}
```
