### 2. 外微分形式与闭形式

**题目：** 请解释外微分形式（exact differential form）与闭形式（closed form）的区别。

**答案：** 外微分形式是指可以表示为某个函数的全微分的微分形式。闭形式则是指其外微分等于零的微分形式。在代数拓扑中，这两个概念用于研究流形的拓扑性质。

**解析：** 如果一个微分形式是外微分形式，那么它一定是一个闭形式。但反之不成立。外微分形式通常与流形的向量场相关联，而闭形式则更多地用于研究流形的同伦性质。

**示例代码：**

```python
# 定义一个外微分形式
exact_form = exact_form

# 定义一个闭形式
closed_form = closed_form

# 检查一个形式是否为外微分形式

# 检查一个形式是否为闭形式

```
