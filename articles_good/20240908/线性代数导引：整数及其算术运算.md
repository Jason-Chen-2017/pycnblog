##### 1. 求最大公约数（GCD）

**题目描述：** 给定两个整数 `a` 和 `b`，求它们的最大公约数。

**输入：** 两个整数 `a` 和 `b`。

**输出：** 最大公约数。

**示例：**
```
输入：a = 15, b = 5
输出：5

输入：a = 12, b = 18
输出：6
```

**算法解析：** 使用辗转相除法（也称为欧几里得算法）来求解最大公约数。该方法通过不断用较小数去除较大数，然后用得到的余数作为新的较小数，直到余数为0，此时的较小数即为最大公约数。

**答案：**
```go
func gcd(a int, b int) int {
    for b != 0 {
        temp := b
        b = a % b
        a = temp
    }
    return a
}
```

##### 2. 求最小公倍数（LCM）

**题目描述：** 给定两个整数 `a` 和 `b`，求它们的最小公倍数。

**输入：** 两个整数 `a` 和 `b`。

**输出：** 最小公倍数。

**示例：**
```
输入：a = 3, b = 4
输出：12

输入：a = 5, b = 7
输出：35
```

**算法解析：** 最小公倍数可以通过最大公约数来求解，公式为 `LCM(a, b) = (a * b) / GCD(a, b)`。

**答案：**
```go
func lcm(a int, b int) int {
    return a / gcd(a, b) * b
}
```

**输出：** 是否为素数（`true` 或 `false`）。

**示例：**
```
输入：n = 17
输出：true

输入：n = 18
输出：false
```

**算法解析：** 判断一个整数是否为素数，可以遍历从2到该数的平方根，若该数能被任何整数整除，则不是素数。

**答案：**
```go
func isPrime(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i*i <= n; i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}
```

**题目描述：** 给定一个整数 `n`，计算它的阶乘。

**输入：** 一个整数 `n`。

**输出：** 阶乘值。

**示例：**
```
输入：n = 5
输出：120

输入：n = 0
输出：1
```

**算法解析：** 阶乘可以通过递归或循环来计算。递归方法在 `n = 0` 时返回1，否则返回 `n` 与 `n - 1` 的阶乘的乘积。循环方法通过从 `n` 开始递减，累乘每个数。

**答案：**
```go
func factorial(n int) int {
    if n == 0 {
        return 1
    }
    result := 1
    for i := n; i > 0; i-- {
        result *= i
    }
    return result
}
```

##### 13. 计算两个整数的最大公约数（使用递归）

**题目描述：** 给定两个整数 `a` 和 `b`，使用递归方法计算它们的最大公约数。

**输入：** 两个整数 `a` 和 `b`。

**输出：** 最大公约数。

**示例：**
```
输入：a = 15, b = 5
输出：5

输入：a = 12, b = 18
输出：6
```

**算法解析：** 使用递归方法，将较大数作为 `a`，较小数作为 `b`，然后调用 `gcd` 函数。递归结束条件为 `b = 0`，此时 `a` 即为最大公约数。

**答案：**
```go
func gcdRecursive(a int, b int) int {
    if b == 0 {
        return a
    }
    return gcdRecursive(b, a%b)
}
```

##### 15. 判断一个整数是否为回文数

**示例：**
```

输出：true

输入：n = 123
输出：false
```

**算法解析：** 将整数转换为字符串，然后反转字符串，最后比较原字符串和反转后的字符串是否相同。

**答案：**
```go
func isPalindrome(n int) bool {
    str := strconv.Itoa(n)
    rev := reverseString(str)
    return str == rev
}

func reverseString(s string) string {
    runes := []rune(s)
    for i, j := 0, len(runes)-1; i < j; i, j = i+1, j-1 {
        runes[i], runes[j] = runes[j], runes[i]
    }
    return string(runes)
}
```

##### 16. 计算一个整数的幂

**题目描述：** 给定一个整数 `base` 和指数 `exponent`，计算 `base` 的 `exponent` 次幂。

**输入：** 整数 `base` 和指数 `exponent`。

**输出：** 幂的结果。

**示例：**
```
输入：base = 2, exponent = 3
输出：8

输入：base = 5, exponent = 2
输出：25
```

**算法解析：** 使用递归或循环来计算幂。递归方法在 `exponent = 0` 时返回1，否则返回 `base` 与 `exponent - 1` 的幂的乘积。循环方法通过从 `exponent` 开始递减，每次乘以 `base`。

**答案：**
```go
func power(base int, exponent int) int {
    if exponent == 0 {
        return 1
    }
    result := 1
    for i := exponent; i > 0; i-- {
        result *= base
    }
    return result
}
```

##### 21. 计算一个整数的奇数位和偶数位之和

```
输出：(1 + 3 + 5, 2 + 4)

输入：n = 67890
输出：(6 + 8, 7 + 9 + 0)
```

**算法解析：** 将整数转换为字符串，然后遍历字符串的每个字符，根据字符的位置（奇数位或偶数位）累加对应的数字。

**答案：**
```go
func oddEvenSum(n int) (int, int) {
    str := strconv.Itoa(n)
    oddSum := 0
    evenSum := 0
    for i, char := range str {
        num := int(rune(char) - '0')
        if i%2 == 0 {
            oddSum += num
        } else {
            evenSum += num
        }
    }
    return oddSum, evenSum
}
```

```go
        sum += int(math.Pow(float64(digit), 3))
    }
    return sum == n
}
```

##### 26. 计算一个整数的阶乘之和

**输入：** 一个整数 `n`。

**输出：** 阶乘之和。

**示例：**
```
输入：n = 3
输出：9

输入：n = 5
输出：153
```

**算法解析：** 可以通过递归或循环计算每个整数的阶乘，然后累加得到阶乘之和。递归方法在 `n = 0` 或 `n = 1` 时返回1，否则返回 `n` 与 `(n - 1)` 的阶乘之和。循环方法从 `n` 开始递减，每次计算阶乘并累加。

**答案：**
```go
func factorialSum(n int) int {
    if n <= 1 {
        return 1
    }
    result := 1
    for i := n; i > 0; i-- {
        result += factorial(i)
    }
    return result
}

func factorial(n int) int {
    if n <= 1 {
        return 1
    }
    return n * factorial(n-1)
}
```

**算法解析：** 将整数转换为字符串，然后遍历字符串的每个字符，将每个字符转换为数字并求和，最后除以位数得到平均值。

**答案：**
```go
func digitAverage(n int) float64 {
    str := strconv.Itoa(n)
    sum := 0
    for _, char := range str {
        sum += int(char - '0')
    }
    return float64(sum) / float64(len(str))
}
```

##### 29. 计算一个整数的各位数字的方差

**题目描述：** 给定一个整数 `n`，计算它的各位数字的方差。

**输入：** 一个整数 `n`。

**输出：** 各位数字的方差。

**示例：**
```
输入：n = 12345
输出：1.6666666666666667

输入：n = 67890
输出：2.3333333333333335
```

**算法解析：** 将整数转换为字符串，然后遍历字符串的每个字符，将每个字符转换为数字，计算平均值，然后计算每个数字与平均值的差的平方，最后求这些平方的平均值。

**答案：**
```go
func digitVariance(n int) float64 {
    str := strconv.Itoa(n)
    sum := 0
    for _, char := range str {
        sum += int(char - '0')
    }
    avg := float64(sum) / float64(len(str))
    var sumOfSquares float64
    for _, char := range str {
        digit := float64(int(char) - '0')
        sumOfSquares += (digit - avg) * (digit - avg)
    }
    return sumOfSquares / float64(len(str))
}
```

##### 30. 计算一个整数的各位数字的标准差

**题目描述：** 给定一个整数 `n`，计算它的各位数字的标准差。

**输入：** 一个整数 `n`。

**输出：** 各位数字的标准差。

**示例：**
```
输入：n = 12345
输出：1.2909

输入：n = 67890
输出：1.5514
```

**算法解析：** 标准差是方差的平方根。首先计算方差，然后对方差开平方。

**答案：**
```go
func digitStandardDeviation(n int) float64 {
    variance := digitVariance(n)
    return math.Sqrt(variance)
}
```
