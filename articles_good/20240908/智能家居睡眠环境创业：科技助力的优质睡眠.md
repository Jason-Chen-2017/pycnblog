#### 1. 睡眠监测数据的实时处理

**题目：** 设计一个算法来处理睡眠监测数据，并提取关键信息。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

type SleepData struct {
    Timestamp time.Time
    HeartRate int
    Motion    float64
}

func processSleepData(data []SleepData) (avgHeartRate int, avgMotion float64) {
    totalHeartRate := 0
    totalMotion := 0.0
    for _, d := range data {
        totalHeartRate += d.HeartRate
        totalMotion += d.Motion
    }
    avgHeartRate = totalHeartRate / len(data)
    avgMotion = totalMotion / float64(len(data))
    return
}

func main() {
    data := []SleepData{
        {Timestamp: time.Now(), HeartRate: 70, Motion: 0.1},
        {Timestamp: time.Now().Add(time.Minute), HeartRate: 72, Motion: 0.2},
        {Timestamp: time.Now().Add(2 * time.Minute), HeartRate: 68, Motion: 0.3},
    }
    avgHeartRate, avgMotion := processSleepData(data)
    fmt.Printf("Average Heart Rate: %d\n", avgHeartRate)
    fmt.Printf("Average Motion: %f\n", avgMotion)
}
```

**解析：** 这个算法通过处理一系列睡眠监测数据，计算平均心率（Heart Rate）和平均运动（Motion）数据。这可以帮助创业者了解用户的睡眠质量。

#### 2. 智能灯光调节

**题目：** 设计一个算法来调节智能灯光，根据用户的睡眠模式调整灯光亮度。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

func adjustLighting(heartRate int, lightLevel *int) {
    if heartRate < 60 {
        *lightLevel = 0 // 完全关闭灯光
    } else if heartRate < 80 {
        *lightLevel = 20 // 低亮度
    } else {
        *lightLevel = 100 // 高亮度
    }
}

func main() {
    var lightLevel int = 100
    heartRate := 65

    adjustLighting(heartRate, &lightLevel)
    fmt.Printf("Adjusted Light Level: %d\n", lightLevel)
}
```

**解析：** 这个算法根据心率（Heart Rate）来调整灯光亮度。对于心率较低的情况，将灯光调至较低亮度，有助于用户进入睡眠状态。

#### 3. 智能窗帘控制

**题目：** 设计一个算法来控制智能窗帘，根据室内光线强度调整窗帘状态。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

func adjustCurtains lichtIntensity float64, curtainsState *string) {
    if lichtIntensity < 200 {
        *curtainsState = "Open" // 打开窗帘
    } else {
        *curtainsState = "Close" // 关闭窗帘
    }
}

func main() {
    var curtainsState string = "Close"
    lichtIntensity := 300

    adjustCurtains(lichtIntensity, &curtainsState)
    fmt.Printf("Curtains State: %s\n", curtainsState)
}
```

**解析：** 这个算法根据室内光线强度（LichtIntensity）来调整窗帘状态。对于光线较暗的情况，将窗帘打开，有助于用户更好地休息。

#### 4. 智能音响控制

**题目：** 设计一个算法来控制智能音响，根据用户的语音指令播放音乐。

**答案：**

```go
package main

import (
    "fmt"
    "strings"
)

func playMusic(command string) {
    if strings.Contains(command, "play") {
        fmt.Println("Playing music...")
    } else {
        fmt.Println("Command not recognized.")
    }
}

func main() {
    command := "Play soft music."

    playMusic(command)
}
```

**解析：** 这个算法根据用户的语音指令（Command）来播放音乐。当指令包含“play”关键字时，算法将播放音乐。

#### 5. 智能环境监测

**题目：** 设计一个算法来监测室内环境参数，如温度、湿度等，并在超出正常范围时发出警报。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

type EnvironmentData struct {
    Temperature float64
    Humidity    float64
}

func checkEnvironmentData(data EnvironmentData) {
    if data.Temperature < 10 || data.Temperature > 30 {
        fmt.Println("Temperature alert!")
    }
    if data.Humidity < 30 || data.Humidity > 70 {
        fmt.Println("Humidity alert!")
    }
}

func main() {
    data := EnvironmentData{
        Temperature: 25,
        Humidity:    45,
    }

    checkEnvironmentData(data)
}
```

**解析：** 这个算法检查室内环境参数，如温度和湿度。如果温度低于10°C或高于30°C，或者湿度低于30%或高于70%，算法将发出警报。
