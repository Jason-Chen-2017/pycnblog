#### 1. 最长公共子序列（LCS）

**题目：** 给定两个字符串`str1`和`str2`，找出它们的最长公共子序列。

**输入：**

```
str1 = "ABCDGH"
str2 = "AEDFHR"
```

**输出：**

```
LCS = "ADH"
```

**答案：**

使用动态规划解决最长公共子序列问题。

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    result = []
    i, j = m, n
    while i > 0 and j > 0:
        if str1[i - 1] == str2[j - 1]:
            result.append(str1[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            i -= 1
        else:
            j -= 1

    return ''.join(result[::-1])

# 测试
str1 = "ABCDGH"
str2 = "AEDFHR"
print(longest_common_subsequence(str1, str2))
```

**输入：**

```
nums = [10, 9, 2, 5, 3, 7, 101, 18]
```

**输出：**

```
LIS length = 4
```

**答案：**

使用动态规划解决最长递增子序列问题。

```python
def length_of_lis(nums):
    if not nums:
        return 0

    dp = [1] * len(nums)
    for i in range(1, len(nums)):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)

    return max(dp)
```

#### 4. 最小路径和

**题目：** 给定一个包含非负整数的二维网格`grid`，找出从左上角到右下角的最小路径和。

**输入：**

```
grid = [
    [1, 3, 1],
    [1, 5, 1],
    [4, 2, 1]
]
```

**输出：**

```
Min path sum = 7
```

**答案：**

使用动态规划求解。

```python
def min_path_sum(grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]

    dp[0][0] = grid[0][0]
    for i in range(1, m):
        dp[i][0] = dp[i - 1][0] + grid[i][0]
    for j in range(1, n):
        dp[0][j] = dp[0][j - 1] + grid[0][j]

    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]

    return dp[-1][-1]

# 测试

    [1, 5, 1],

]

```

#### 5. 合并区间

**题目：** 给定一组区间，合并所有重叠的区间。

**输入：**

```
intervals = [
    [1, 3],
    [2, 6],
    [8, 10],
    [15, 18]
]
```

**输出：**

```
Merged intervals = [[1, 6], [8, 10], [15, 18]]
```

**答案：**

首先对区间按照起始点排序，然后合并重叠区间。

```python
def merge(intervals):
    if not intervals:
        return []

    intervals.sort()
    result = [intervals[0]]
    for i in range(1, len(intervals)):
        last_end, curr_start = result[-1][1], intervals[i][0]
        if curr_start <= last_end:
            result[-1][1] = max(last_end, intervals[i][1])
        else:
            result.append(intervals[i])

    return result

# 测试

    [1, 3],
    [2, 6],
    [8, 10],
    [15, 18]
]
print(merge(intervals))
```

#### 6. 二分查找

**题目：** 给定一个排序数组`nums`和一个目标值`target`，找出`target`在数组中的索引。

**输入：**

```
nums = [-1, 0, 3, 5, 9, 12]
target = 9
```

**输出：**

```
Index = 4
```

**答案：**

使用二分查找算法。

```python
def search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1

# 测试

target = 9
print(search(nums, target))
```

```python
def isValid(s):
    stack = []
    for c in s:
        if c in "({[":
            stack.append(c)
        elif not stack or (c == ')' and stack[-1] != '(' or c == ']' and stack[-1] != '[' or c == '}' and stack[-1] != '{'):
            return False
        else:
            stack.pop()

    return not stack
```

```python
def max_profit(prices):
    max_profit = 0

        min_price = min(min_price, price)
        max_profit = max(max_profit, price - min_price)

    return max_profit
```

#### 10. 股票买卖最佳时机 III

**输入：**

```
prices = [3, 3, 6, 5, 0, 3, 1, 4]
```

**输出：**

```
Max profit = 9
```

**答案：**

使用动态规划。

```python
def max_profit(prices):
    n = len(prices)
    buy1, sell1, buy2, sell2 = -prices[0], 0, -prices[0], 0

    for price in prices:
        buy1 = max(buy1, -price)
        sell1 = max(sell1, buy1 + price)
        buy2 = max(buy2, sell1 - price)
        sell2 = max(sell2, buy2 + price)

    return sell2

# 测试

```

```
Index = 4
```

**答案：**

使用二分查找算法。

```python
def search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] >= nums[left]:
            if target >= nums[left] and target < nums[mid]:
                right = mid - 1
            else:
                left = mid + 1
        else:
            if target > nums[mid] and target <= nums[right]:
                left = mid + 1
            else:
                right = mid - 1

    return -1
```


**答案：**

从数组的末尾开始填充，避免覆盖已填充的元素。

```python
def merge(nums1, m, nums2, n):
    p1, p2 = m - 1, n - 1
    p = m + n - 1

    while p1 >= 0 and p2 >= 0:
        if nums1[p1] > nums2[p2]:
            nums1[p] = nums1[p1]
            p1 -= 1
        else:
            nums1[p] = nums2[p2]
            p2 -= 1
        p -= 1

    while p2 >= 0:
        nums1[p] = nums2[p2]
        p2 -= 1
        p -= 1
```

#### 18. 编辑距离

**题目：** 给定两个字符串`word1`和`word2`，请计算将`word1`转换为`word2`所需的最小操作次数。操作包括插入、删除和替换一个字符。

```

```

**输出：**

```
Minimum number of operations = 3
```

**答案：**

使用动态规划。

```python
def min_distance(word1, word2):
    dp = [[0] * (len(word2) + 1) for _ in range(len(word1) + 1)]

    for i in range(len(word1) + 1):
        for j in range(len(word2) + 1):
            if i == 0:
                dp[i][j] = j
            elif j == 0:
                dp[i][j] = i
            elif word1[i - 1] == word2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]
            else:
                dp[i][j] = 1 + min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1])

    return dp[-1][-1]

# 测试
```

#### 19. 股票买卖的最佳时机 IV

**题目：** 给定一个数组`prices`，其中`prices[i]`是第`i`天的股票价格。可以完成最多`k`笔交易。每笔交易必须包含一个购买和一个出售操作。请找出能获得的最大利润。

**输入：**

```

```

**输出：**


**答案：**

使用动态规划。

```python
def max_profit(prices, k):
    n = len(prices)
    if n == 0:
        return 0

    dp = [[[0] * 2 for _ in range(k + 1)] for _ in range(n)]

    for j in range(1, k + 1):
        max_diff = -prices[0]
        for i in range(1, n):
            max_diff = max(max_diff, dp[i - 1][j - 1][1] - prices[i - 1])
            dp[i][j][0] = max(dp[i - 1][j][0], max_diff + prices[i - 1])
            dp[i][j][1] = max(dp[i - 1][j][1], dp[i - 1][j - 1][0] - prices[i - 1])

    return dp[-1][-1][0]
```

**题目：** 给定两个字符串`s1`和`s2`，找出它们的最长公共子串。


使用动态规划。

```python
def longest_common_substring(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    max_len, end_pos = 0, 0

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                if dp[i][j] > max_len:
                    max_len = dp[i][j]
                    end_pos = i
            else:
                dp[i][j] = 0

    return s1[end_pos - max_len: end_pos]
```

#### 27. 三数之和

**题目：** 给定一个数组`nums`，找出所有满足条件`i < j < k`且`nums[i] + nums[j] + nums[k] == 0`的三元组。

**输入：**

```
nums = [-1, 0, 1, 2, -1, -4]
```

**输出：**

```
Three-sum combinations: [(-1, 0, 1), (-1, -1, 2)]
```

**答案：**

使用哈希表和排序。

```python
def three_sum(nums):
    nums.sort()
    ans = []
    n = len(nums)
    for i in range(n):
        if i > 0 and nums[i] == nums[i - 1]:
            continue
        left, right = i + 1, n - 1
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            if total < 0:
                left += 1
            elif total > 0:
                right -= 1
            else:
                ans.append([nums[i], nums[left], nums[right]])
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                left += 1
                right -= 1

    return ans
```

```python
def is_anagram(s, t):
    if len(s) != len(t):
        return False

    cnt = [0] * 26
    for c in s:
        cnt[ord(c) - ord('a')] += 1

    for c in t:
        cnt[ord(c) - ord('a')] -= 1
        if cnt[ord(c) - ord('a')] < 0:
            return False

    return True
```
