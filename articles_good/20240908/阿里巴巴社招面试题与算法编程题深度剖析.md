## 数据结构与算法基础

### 1. 数据结构基础

- **数组**：数组是一种基础的数据结构，用于存储一系列元素。它的特点是查找和访问速度快，但插入和删除操作相对较慢。
- **链表**：链表由一系列节点组成，每个节点包含数据和指向下一个节点的指针。链表的优点是插入和删除操作灵活，但访问速度相对较慢。
- **树**：树是一种层次结构，用于表示具有父子关系的数据。常见的树结构包括二叉树、平衡树等。
- **图**：图是一种由节点和边组成的数据结构，用于表示复杂的关系。常见的图算法包括最短路径、拓扑排序等。

### 2. 算法基础

- **排序算法**：常见的排序算法有冒泡排序、选择排序、插入排序、快速排序等。
- **查找算法**：常见的查找算法有二分查找、顺序查找等。
- **动态规划**：动态规划是一种解决最优化问题的算法思想，通过将问题分解为子问题并求解子问题的最优解，从而得到原问题的最优解。

### 1. 题目10：寻找两个有序数组中的中位数

#### 题目描述

给定两个有序数组 nums1 和 nums2，找出这两个有序数组的中位数。

#### 解题思路

我们可以将两个数组合并，然后找到中位数。但由于数组的大小可能很大，直接合并会导致时间复杂度较高。因此，我们可以使用二分查找的方法来解决这个问题。

#### 解题步骤

1. 将两个数组分别命名为 nums1 和 nums2。
2. 判断两个数组的大小，较小的数组作为基准。
3. 使用二分查找的方法，找到中位数的索引。
4. 根据索引计算中位数。

#### 代码示例

```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        if len(nums1) > len(nums2):
            nums1, nums2 = nums2, nums1

        m, n = len(nums1), len(nums2)
        imin, imax, half_len = 0, m, (m + n + 1) // 2
        while imin <= imax:
            i = (imin + imax) // 2
            j = half_len - i
            if i < m and nums2[j - 1] > nums1[i]:
                imin = i + 1
            elif i > 0 and nums1[i - 1] > nums2[j]:
                imax = i - 1
            else:
                if i == 0:
                    max_of_left = nums2[j - 1]
                elif j == 0:
                    max_of_left = nums1[i - 1]
                else:
                    max_of_left = max(nums1[i - 1], nums2[j - 1])
                if (m + n) % 2 == 1:
                    return max_of_left
                min_of_right = min(nums1[i], nums2[j])
                return (max_of_left + min_of_right) / 2
```

### 2. 题目11：最长公共子序列

#### 题目描述

给定两个字符串 text1 和 text2，找出它们的最长公共子序列。

#### 解题思路

我们可以使用动态规划的方法来解决这个问题。定义一个二维数组 dp，其中 dp[i][j] 表示 text1 的前 i 个字符和 text2 的前 j 个字符的最长公共子序列的长度。

#### 解题步骤

1. 初始化一个二维数组 dp，大小为 (m+1)×(n+1)，其中 m 和 n 分别为 text1 和 text2 的长度。
2. 遍历 text1 和 text2 的每个字符，根据字符是否匹配更新 dp 数组。
3. 返回 dp[m][n]，即为最长公共子序列的长度。

#### 代码示例

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i - 1] == text2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1] + 1
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

        return dp[m][n]
```

### 3. 题目12：两数之和

#### 题目描述

给定一个整数数组 nums 和一个目标值 target，找出数组中两数之和等于 target 的两个数，并返回他们的下标。

#### 解题思路

我们可以使用哈希表来解决这个问题。遍历数组，对于每个元素，使用哈希表查找是否存在与该元素相加等于 target 的另一个元素。

#### 解题步骤

1. 初始化一个空哈希表。
2. 遍历数组，对于每个元素，判断哈希表中是否存在与该元素相加等于 target 的另一个元素。
3. 如果存在，返回这两个元素的下标。
4. 如果不存在，将当前元素的值和下标存储在哈希表中。

#### 代码示例

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        hashmap = {}
        for i, num in enumerate(nums):
            complement = target - num
            if complement in hashmap:
                return [hashmap[complement], i]
            hashmap[num] = i
        return []
```
