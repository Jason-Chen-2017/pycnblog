**题目：** 输入一个整数数组，输出所有数字的和。

**答案：**


##### 3. 判断是否为素数

给定一个整数，判断是否为素数。

**题目：** 输入一个整数，输出是否为素数。

**答案：**

```python
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            return False
    return True

num = 17
is_prime_num = is_prime(num)
print(is_prime_num)  # 输出 True
```

##### 4. 冒泡排序

给定一个整数数组，使用冒泡排序对其进行排序。

**题目：** 输入一个整数数组，使用冒泡排序对其进行排序。

**答案：**

```python
def bubble_sort(nums):
    n = len(nums)
    for i in range(n - 1):
        for j in range(n - i - 1):
            if nums[j] > nums[j + 1]:
                nums[j], nums[j + 1] = nums[j + 1], nums[j]

nums = [5, 3, 8, 4, 2]
bubble_sort(nums)
print(nums)  # 输出 [2, 3, 4, 5, 8]
```

##### 5. 回文判断

给定一个字符串，判断是否为回文。

**题目：** 输入一个字符串，输出是否为回文。

**答案：**

```python
def is_palindrome(s):
    return s == s[::-1]

s = "madam"
is_palindrome_s = is_palindrome(s)
print(is_palindrome_s)  # 输出 True
```
