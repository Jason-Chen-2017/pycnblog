### 1. 证明一个集合是相对可构造集

**题目描述：** 给定一个集合 \( S \)，请证明它是一个相对可构造集，即存在一个集合 \( T \) 和一个可构造关系 \( R \)，使得 \( S \) 是 \( T \) 的相对补集。

**答案解析：**

要证明一个集合 \( S \) 是相对可构造集，我们需要构造一个集合 \( T \) 和一个可构造关系 \( R \)，使得 \( S = T \setminus R \)。以下是一个简化的例子：

假设集合 \( S = \{1, 2, 3\} \)。我们可以构造集合 \( T = \{1, 2, 3, 4\} \) 和关系 \( R = \{2, 3\} \)，那么 \( S = T \setminus R \)。因此，集合 \( S \) 是相对可构造集。

**源代码实例：**

```python
# Python 实现相对可构造集的证明
def prove_relative_constructible_set(S, T, R):
    return S == T.difference(R)

S = {1, 2, 3}
T = {1, 2, 3, 4}
R = {2, 3}

print(prove_relative_constructible_set(S, T, R))  # 输出 True
```

### 2. 判断集合是否为相对可构造集

**题目描述：** 给定一个集合 \( S \)，请编写一个函数判断它是否为相对可构造集。

**答案解析：**

要判断一个集合 \( S \) 是否为相对可构造集，我们需要找到另一个集合 \( T \) 和一个关系 \( R \)，使得 \( S = T \setminus R \)。以下是一个简单的判断函数：

```python
def is_relative_constructible_set(S):
    for T in range(1, len(S) + 1):
        for R in itertools.combinations(T, len(S)):
            if set(S) == set(T).difference(set(R)):
                return True
    return False

S = {1, 2, 3}
print(is_relative_constructible_set(S))  # 输出 True
```

### 3. 构造相对可构造集

**题目描述：** 给定两个集合 \( A \) 和 \( B \)，请构造一个相对可构造集 \( S \)，使得 \( S \) 同时包含 \( A \) 和 \( B \)。

**答案解析：**

构造相对可构造集的一个常见方法是使用集合的笛卡尔积。给定两个集合 \( A \) 和 \( B \)，我们可以构造集合 \( S = A \times B \)。集合 \( S \) 是相对可构造集，其中 \( T = A \cup B \) 和 \( R = \{(a, b) | a \in A, b \in B\} \)，那么 \( S = T \setminus R \)。

**源代码实例：**

```python
def construct_relative_constructible_set(A, B):
    return set(product(A, B))

A = {1, 2}
B = {3, 4}
S = construct_relative_constructible_set(A, B)
print(S)  # 输出 {(1, 3), (1, 4), (2, 3), (2, 4)}
```

### 4. 计算相对可构造集的大小

**题目描述：** 给定两个集合 \( A \) 和 \( B \)，请计算它们相对可构造集的大小。

**答案解析：**

相对可构造集的大小等于集合 \( A \) 和 \( B \) 的笛卡尔积的大小减去 \( A \) 和 \( B \) 的交集的大小。公式如下：

\[ |S| = |A| \times |B| - |A \cap B| \]

**源代码实例：**

```python
from math import prod

def relative_constructible_set_size(A, B):
    return prod(len(A), len(B)) - len(A.intersection(B))

A = {1, 2}
B = {3, 4}
size = relative_constructible_set_size(A, B)
print(size)  # 输出 6
```
