## 3.1 线性回归

线性回归是一种预测方法，它可以用来预测一个变量的值，根据一个或多个其他变量的值。线性回归的数学模型公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

其中，$y$ 是预测的变量，$x_1, x_2, ..., x_n$ 是预测变量，$\beta_0, \beta_1, ..., \beta_n$ 是参数，$\epsilon$ 是误差。

线性回归的具体操作步骤为：

1. 数据清洗：对数据进行清洗，以确保数据的质量。
2. 数据处理：对数据进行处理，以确保数据的一致性。
3. 数据分析：对数据进行分析，以确定预测模型的参数。
4. 数据可视化：对数据进行可视化，以确定预测模型的效果。

## 3.2 逻辑回归

逻辑回归是一种分类方法，它可以用来分类一个变量的值，根据一个或多个其他变量的值。逻辑回归的数学模型公式为：

$$
P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n)}}
$$

其中，$y$ 是分类的变量，$x_1, x_2, ..., x_n$ 是分类变量，$\beta_0, \beta_1, ..., \beta_n$ 是参数。

逻辑回归的具体操作步骤为：

## 4.1 线性回归

```python
import numpy as np
import matplotlib.pyplot as plt

# 数据清洗
x = np.array([1, 2, 3, 4, 5])
y = np.array([1, 4, 9, 16, 25])

# 数据处理
x = x.reshape((-1, 1))

# 数据分析
beta_0 = np.mean(y) - np.mean(x) * np.mean(x)
beta_1 = np.mean(y) - np.mean(x) * np.mean(x)

# 数据可视化
plt.scatter(x, y)
plt.plot(x, beta_0 + beta_1 * x, color='red')
plt.show()
```

## 4.3 支持向量机

```python
import numpy as np
from sklearn import svm

# 数据清洗
x = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([1, 2, 3, 4])

# 数据处理
x = x.reshape((-1, 1, 2))

# 数据分析
clf = svm.SVC(kernel='linear')
clf.fit(x, y)

# 数据可视化
plt.scatter(x[:, 0], x[:, 1], c=y)
plt.plot(clf.support_vectors_[:, 0], clf.support_vectors_[:, 1], color='red')
plt.show()
```
