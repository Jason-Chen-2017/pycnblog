随着互联网的普及和数据的爆炸增长，云计算技术成为了企业和个人的核心基础设施之一。云计算的核心思想是将计算资源、存储资源和网络资源等基础设施通过网络共享和集中管理，实现资源的灵活性、可扩展性和可靠性。云计算可以分为公有云、私有云和混合云三种类型，各自适用于不同的场景和需求。

云计算的发展历程可以分为以下几个阶段：

1. 基础设施即服务（IaaS）：这是云计算的第一阶段，主要提供虚拟服务器、虚拟硬盘、虚拟交换机等基础设施服务。用户可以通过网络访问和管理这些基础设施资源，实现资源的共享和可扩展性。

2. 平台即服务（PaaS）：这是云计算的第二阶段，主要提供应用程序开发和部署平台。用户可以通过网络访问和管理这些平台资源，实现应用程序的快速开发和部署。

3. 软件即服务（SaaS）：这是云计算的第三阶段，主要提供软件应用服务。用户可以通过网络访问和使用这些软件应用，实现软件的快速部署和使用。

云计算的核心技术包括虚拟化技术、分布式技术、网络技术、存储技术、安全技术等。虚拟化技术是云计算的基础，可以实现资源的共享和可扩展性。分布式技术是云计算的核心，可以实现资源的负载均衡和容错。网络技术是云计算的基础设施，可以实现资源的访问和管理。存储技术是云计算的核心，可以实现数据的存储和备份。安全技术是云计算的关键，可以保证资源的安全性和可靠性。

云计算的发展趋势包括边缘计算、服务网格、容器技术、函数计算等。边缘计算是云计算的延伸，可以实现资源的分布和集中。服务网格是云计算的核心，可以实现资源的组合和管理。容器技术是云计算的基础，可以实现资源的隔离和移植。函数计算是云计算的新兴，可以实现资源的微服务和自动化。

        self.nodes = nodes
        self.hash_function = hashlib.md5
        self.node_to_shard = defaultdict(set)

    def add_node(self, node):
        self.nodes.add(node)
        for shard in self.shards():
            self.node_to_shard[node].add(shard)

    def remove_node(self, node):
        self.nodes.remove(node)
        for shard in self.shards():
            if node in self.node_to_shard[shard]:
                self.node_to_shard[shard].remove(node)
