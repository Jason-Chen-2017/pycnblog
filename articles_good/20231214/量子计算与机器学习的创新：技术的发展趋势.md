5. 损失函数：损失函数是机器学习算法的评估标准，它可以用来衡量算法的预测误差和准确性。

import numpy as np

# 创建量子电路

# 设置量子门参数
qc.h(0)

qc.h(1)

# 编译量子电路

# 获取结果
counts = qobj.get_counts()
plot_histogram(counts)
