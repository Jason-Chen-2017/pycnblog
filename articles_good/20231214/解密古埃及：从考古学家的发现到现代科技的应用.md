```python
def base_convert(num, from_base, to_base):
    # 将要转换的数字分解为各个位的数字
    digits = []
    while num > 0:
        digits.append(num % from_base)
        num //= from_base
    # 将各个位的数字转换为对应的进制系统下的数字
    converted_digits = []
    for digit in digits:
        converted_digits.append(str(digit))
    # 将各个位的数字拼接成一个新的数字
    converted_num = ''.join(reversed(converted_digits))
    return converted_num
```

上述代码实例中的`base_convert`函数用于将一个数字从一个进制系统转换为另一个进制系统。它接受三个参数：要转换的数字、要转换的进制系统和目标进制系统。
