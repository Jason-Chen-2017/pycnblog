### 4.2 公式推导过程

1. **奇偶性检测**：

   - 奇数：`n & 1`
   - 偶数：`n & 1 == 0`

2. **奇偶排序**：

   - 初始化两个数组：`oddArray` 和 `evenArray`
   - 遍历原数组：`array`
     - 如果 `array[i] & 1 == 0`，则将 `array[i]` 存储到 `evenArray`
     - 否则，将 `array[i]` 存储到 `oddArray`
   - 对 `evenArray` 和 `oddArray` 分别进行排序
   - 合并两个数组：`resultArray = evenArray + oddArray`

3. **奇偶校验码**：

   - 假设数据为 `data`
   - 计算奇偶校验位：`parityBit = data & 1`
   - 如果 `parityBit == 0`，则表示数据为偶数；否则，表示数据为奇数。

### 5.2 源代码详细实现

# 奇偶性检测
def check_parity(n):
    return n & 1

# 奇偶排序
def odd_even_sort(array):
    oddArray = []
    evenArray = []
    for i in range(len(array)):
        if check_parity(array[i]):
            oddArray.append(array[i])
        else:
            evenArray.append(array[i])
    oddArray.sort()
    evenArray.sort()
    resultArray = evenArray + oddArray
    return resultArray

# 奇偶校验码
def parity_check(data):
    return data & 1

# 测试代码
array = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]

print(sorted_array)
