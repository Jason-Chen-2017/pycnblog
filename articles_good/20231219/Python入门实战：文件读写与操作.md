## 4.1 文件读写代码实例

以下是一个读取和写入文件的代码实例：

```python
# 打开文件
with open('example.txt', 'r') as f:
    # 读取文件内容
    data = f.read()
    # 处理文件内容
    data = data.upper()
    # 写入文件
    with open('example.txt', 'w') as f:
        f.write(data)
```

详细解释说明：

1. 使用with语句打开文件，以确保文件在操作完成后自动关闭。
2. 使用open()函数打开文件，模式为'r'表示读取模式。
3. 使用文件对象的read()方法读取文件内容。
4. 处理文件内容，将其转换为大写。
5. 使用open()函数打开文件，模式为'w'表示写入模式。
6. 使用文件对象的write()方法写入文件内容。

# 6.附加问题

在本节中，我们将回答一些常见问题。

## 6.1 常见问题

1. 如何判断文件是否存在？

使用os.path.exists()函数可以判断文件是否存在。

```python
import os

file_path = 'example.txt'
if os.path.exists(file_path):
    print(f'文件{file_path}存在')
else:
    print(f'文件{file_path}不存在')
```

1. 如何获取文件的大小？

使用os.path.getsize()函数可以获取文件的大小。

```python
import os

file_path = 'example.txt'
file_size = os.path.getsize(file_path)
print(f'文件{file_path}的大小为{file_size}字节')
```

1. 如何获取文件的创建和修改时间？

使用os.path.getctime()和os.path.getmtime()函数可以获取文件的创建和修改时间。

```python
import os

file_path = 'example.txt'
creation_time = os.path.getctime(file_path)
modification_time = os.path.getmtime(file_path)

print(f'文件{file_path}的创建时间为{creation_time}')
print(f'文件{file_path}的修改时间为{modification_time}')
```

1. 如何获取文件的所有者和权限？

使用os.path.getpwuid()和os.stat()函数可以获取文件的所有者和权限。

```python
import os

file_path = 'example.txt'
uid = os.stat(file_path).st_uid
owner_name = os.path.getpwuid(uid)

print(f'文件{file_path}的所有者为{owner_name}')
```

1. 如何更改文件的所有者和权限？

使用os.chown()和os.chmod()函数可以更改文件的所有者和权限。

```python
import os

file_path = 'example.txt'
new_owner_uid = 1001
new_permissions = 0o644

os.chown(file_path, new_owner_uid, 0)
os.chmod(file_path, new_permissions)
```
