## 3.1 贪婪算法

贪婪算法是一种常用的算法设计方法，它的核心思想是在每个步骤中做出最佳的局部决策，以期达到全局最优解。这种算法的优点是简单易理解，缺点是不一定能够找到全局最优解。

贪婪算法的具体操作步骤如下：

1. 初始化问题，设定目标函数。
2. 找到当前状态下的最佳局部解。
3. 更新问题状态，将当前局部解作为下一步的初始状态。
4. 重复步骤2和步骤3，直到问题解决或达到最大迭代次数。

数学模型公式详细讲解：

贪婪算法的目标函数可以表示为：

$$
f(x) = \min_{x \in X} c(x)
$$

其中，$f(x)$ 是目标函数，$c(x)$ 是对象函数，$X$ 是解空间。

## 3.2 动态规划

动态规划是一种常用的算法设计方法，它的核心思想是将问题拆分成多个子问题，然后递归地解决这些子问题，最后将子问题的解组合成原问题的解。

动态规划的具体操作步骤如下：

1. 定义子问题。
2. 递归地解子问题。
3. 存储子问题的解。
4. 将子问题的解组合成原问题的解。

数学模型公式详细讲解：

动态规划可以用以下公式表示：

$$
f(n) = \min_{i=1}^{n} \{ f(i-1) + g(i) \}
$$

其中，$f(n)$ 是原问题的解，$f(i-1)$ 是子问题的解，$g(i)$ 是子问题与原问题之间的关系。

## 3.3 分治法

分治法是一种常用的算法设计方法，它的核心思想是将问题拆分成多个子问题，然后递归地解决这些子问题，最后将子问题的解合并成原问题的解。

分治法的具体操作步骤如下：

1. 将问题拆分成多个子问题。
2. 递归地解子问题。
3. 将子问题的解合并成原问题的解。

数学模型公式详细讲解：

分治法可以用以下公式表示：

$$
T(n) = 2T(\frac{n}{2}) + O(n)
$$

其中，$T(n)$ 是解问题所需的时间复杂度，$T(\frac{n}{2})$ 是解子问题所需的时间复杂度，$O(n)$ 是合并子问题所需的时间复杂度。

```python
def longest_common_subsequence(str1, str2):
    m = len(str1)
    n = len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
```

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])

    return result
```

详细解释说明：

1. 将问题拆分成多个子问题。
2. 递归地解子问题。
3. 将子问题的解合并成原问题的解。
