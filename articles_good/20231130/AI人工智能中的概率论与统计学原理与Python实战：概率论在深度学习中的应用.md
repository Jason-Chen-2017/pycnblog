# 2.核心概念与联系
# 2.1概率论基本概念
在深度学习中，我们使用概率论来描述模型的不确定性，以及模型预测的不确定性。概率论的基本概念包括事件、样本空间、概率、条件概率、独立性等。

事件：在概率论中，事件是一个可能发生的结果。事件可以是一个数字、一个区间、一个集合等。

样本空间：在概率论中，样本空间是所有可能发生的事件的集合。样本空间可以是一个有限集合、一个无限集合或者一个数字区间等。

概率：在概率论中，概率是一个事件发生的可能性。概率是一个数字，范围在0到1之间。概率为0的事件是不可能发生的，概率为1的事件是必然发生的。

条件概率：在概率论中，条件概率是一个事件发生的可能性，给定另一个事件已经发生了。条件概率是一个数字，范围在0到1之间。

独立性：在概率论中，独立性是两个事件发生的可能性之间的关系。如果两个事件是独立的，那么它们的发生或不发生是不会影响彼此的。

数据分布：在统计学中，数据分布是一个数据集合的分布情况。数据分布可以是一个连续分布、一个离散分布或者一个混合分布等。

参数：在统计学中，参数是一个数据分布的特征。参数可以是一个数字、一个向量等。

统计量：在统计学中，统计量是一个数据集合的描述性统计。统计量可以是一个中心趋势、一个散度等。

统计假设：在统计学中，统计假设是一个数据分布的假设。统计假设可以是一个无参数假设、一个参数假设等。
