## 3.1 线性回归

线性回归是一种简单的机器学习算法，它可以用来预测连续型变量的值。线性回归的基本思想是通过找到最佳的直线来最小化误差。

### 3.1.1 数学模型

线性回归的数学模型如下：

y = θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ

其中，y是预测值，x₁、x₂、...、xₙ是输入变量，θ₀、θ₁、...、θₙ是权重。

### 3.1.2 损失函数

损失函数是用来衡量模型预测值与实际值之间差异的指标。对于线性回归，常用的损失函数是均方误差（MSE）：

MSE = (1/m) * Σ(y_i - y_hat)^2

其中，m是数据集的大小，y_i是实际值，y_hat是预测值。

### 3.1.3 梯度下降算法

梯度下降算法是用来优化线性回归模型的方法。它的基本思想是通过不断地更新权重，使损失函数的值逐渐减小。梯度下降算法的公式如下：

θ = θ - α * ∇J(θ)

其中，θ是权重，α是学习率，∇J(θ)是损失函数的梯度。

## 3.2 逻辑回归

逻辑回归是一种用于分类问题的机器学习算法。它可以用来预测二元类别变量的值。

### 3.2.1 数学模型

逻辑回归的数学模型如下：

P(y=1) = 1 / (1 + exp(-(θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ)))

其中，y是预测值，x₁、x₂、...、xₙ是输入变量，θ₀、θ₁、...、θₙ是权重。

### 3.2.2 损失函数

逻辑回归的损失函数是交叉熵损失函数：

J = - (1/m) * Σ[y_i * log(y_hat_i) + (1 - y_i) * log(1 - y_hat_i)]

其中，m是数据集的大小，y_i是实际值，y_hat_i是预测值。

### 3.2.3 梯度下降算法

逻辑回归的梯度下降算法与线性回归相同，只是损失函数和数学模型不同。

## 3.3 支持向量机

支持向量机（SVM）是一种用于分类和回归问题的机器学习算法。它的核心思想是通过找到最佳的超平面来将不同类别的数据点分开。

### 3.3.1 数学模型

支持向量机的数学模型如下：

y = θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ

其中，y是预测值，x₁、x₂、...、xₙ是输入变量，θ₀、θ₁、...、θₙ是权重。

### 3.3.2 损失函数

支持向量机的损失函数是软边界损失函数：

J = 1/2 * Σ(θₙ^2) + C * Σ(max(0, 1 - y_i * (θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ)))

其中，C是正则化参数，用于平衡模型复杂度和误差。

### 3.3.3 梯度下降算法

支持向量机的梯度下降算法与线性回归相同，只是损失函数和数学模型不同。

## 4.1 线性回归

### 4.1.1 数据集

我们将使用一个简单的线性回归数据集，其中x是输入变量，y是输出变量：

### 4.1.2 模型定义

我们将使用Python的numpy库来定义线性回归模型：

import numpy as np

θ = np.array([0, 0])

### 4.1.3 损失函数

我们将使用均方误差（MSE）作为损失函数：

def mse(y_true, y_pred):
    return np.mean((y_true - y_pred)**2)

### 4.1.4 梯度下降算法

我们将使用梯度下降算法来优化线性回归模型：

def gradient_descent(x, y, θ, alpha, iterations):
    m = len(y)
    for _ in range(iterations):
        y_pred = np.dot(x, θ)
        gradients = 2/m * np.dot(x.T, y - y_pred)
       θ = θ - alpha * gradients
    return θ

### 4.1.5 训练模型

我们将使用梯度下降算法来训练线性回归模型：

### 4.1.6 预测
