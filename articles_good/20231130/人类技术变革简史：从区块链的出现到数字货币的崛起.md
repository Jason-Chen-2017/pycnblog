# 2.核心概念与联系

## 2.1区块链的基本概念

区块链是一种分布式、去中心化的数据存储和交易系统，它由一系列的区块组成，每个区块包含一组交易数据和一个时间戳，这些区块通过加密技术相互连接，形成一个不可篡改的链条。

区块链的核心特点有以下几点：

1. 去中心化：区块链不依赖于任何中心化的机构或实体，而是通过分布式网络来实现数据存储和交易。

2. 透明度：区块链的所有交易数据是公开可见的，任何参与网络的节点都可以查看和验证这些数据。

3. 安全性：区块链通过加密技术来保护数据的完整性和不可篡改性，同时通过共识机制来确保网络的一致性和稳定性。

4. 可扩展性：区块链的设计允许它们与其他网络和系统进行集成，从而实现更广泛的应用场景。

5. 计算哈希值：节点需要计算当前区块的哈希值，这个哈希值是唯一的、不可逆的、不可变的。

6. 解决难题：节点需要解决一个难题，这个难题的解决方案是一个数字，它满足以下条件：当前区块的哈希值小于一个预设的阈值。

8. 更新链条：当其他节点接收到当前区块后，它们需要更新自己的链条，这个链条是一个不可篡改的链条。

```python
import time

class Block:
    def __init__(self, index, previous_hash, timestamp, data, hash):
        self.index = index
        self.previous_hash = previous_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

    def calculate_hash(self):
        encoded_str = '{}:{}:{}:{}:{}'.format(self.index, self.previous_hash, self.timestamp, self.data, self.nonce)
        return hashlib.sha256(encoded_str.encode()).hexdigest()

    def mine_block(self, difficulty):
        while self.hash[0:difficulty] != '0' * difficulty:
            self.nonce += 1
            self.hash = self.calculate_hash()

class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(0, '0', time.time(), 'Genesis Block', '0')

    def get_last_block(self):
        return self.chain[-1]

    def add_block(self, new_block):
        new_block.previous_hash = self.get_last_block().hash
        new_block.mine_block(difficulty)
        self.chain.append(new_block)

# 创建一个区块链实例
blockchain = Blockchain()

# 创建一个交易
transaction = {
    'sender': 'Alice',
    'recipient': 'Bob',
    'amount': 50
}

# 创建一个新的区块
new_block = Block(blockchain.get_last_block().index + 1, blockchain.get_last_block().hash, time.time(), transaction, '0')

# 添加新的区块到区块链
blockchain.add_block(new_block)
```
