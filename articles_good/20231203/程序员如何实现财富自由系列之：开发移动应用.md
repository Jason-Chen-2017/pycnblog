```java
public class CalculatorActivity extends AppCompatActivity {

    private EditText mNumber1EditText;
    private EditText mNumber2EditText;
    private TextView mResultTextView;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_calculator);

        mNumber1EditText = findViewById(R.id.number1_edittext);
        mNumber2EditText = findViewById(R.id.number2_edittext);
        mResultTextView = findViewById(R.id.result_textview);

        findViewById(R.id.add_button).setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                double number1 = Double.parseDouble(mNumber1EditText.getText().toString());
                double number2 = Double.parseDouble(mNumber2EditText.getText().toString());
                double result = number1 + number2;
                mResultTextView.setText(String.valueOf(result));
            }
        });

        findViewById(R.id.subtract_button).setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                double number1 = Double.parseDouble(mNumber1EditText.getText().toString());
                double number2 = Double.parseDouble(mNumber2EditText.getText().toString());
                double result = number1 - number2;
                mResultTextView.setText(String.valueOf(result));
            }
        });
    }
}
```

详细解释说明：

1. 我们首先创建一个CalculatorActivity类，继承自AppCompatActivity类。

2. 我们声明了三个EditText类型的控件，用于输入数字1和数字2，以及显示结果的TextView类型的控件。

3. 在onCreate方法中，我们设置了布局文件，并找到了我们声明的控件。

4. 我们为加法按钮设置了一个OnClickListener，当用户点击加法按钮时，我们将输入的数字1和数字2转换为double类型，然后计算结果，并将结果设置到结果TextView中。

5. 我们为减法按钮设置了一个OnClickListener，当用户点击减法按钮时，我们将输入的数字1和数字2转换为double类型，然后计算结果，并将结果设置到结果TextView中。
