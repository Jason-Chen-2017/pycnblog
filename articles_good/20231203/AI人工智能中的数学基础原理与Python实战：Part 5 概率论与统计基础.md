## 1.1 概率论与统计的基本概念

概率论是一门研究随机事件发生的可能性和概率的学科，而统计是一门研究从数据中抽取信息的学科。在人工智能和机器学习中，我们需要对数据进行分析和预测，因此需要掌握概率论与统计的基本概念。

### 1.1.1 随机事件

随机事件是一种可能发生或不发生的事件，其发生概率不确定。例如，掷骰子的结果是一个随机事件，因为掷骰子的结果可能是1、2、3、4、5或6，但具体结果不可预测。

### 1.1.2 概率

概率是随机事件发生的可能性，通常表示为一个数值，范围在0到1之间。概率的计算方法有多种，例如：

1. 直接计数法：计算满足条件的事件数量与总事件数量的比值。例如，从一个包中挑出一个球的概率为1/10，因为一个包中有10个球。

2. 定义域法：将事件分为不可分的小区间，然后计算满足条件的小区间的总面积与总面积的比值。例如，计算一个球在一个圆盘上的概率，可以将圆盘分为100个等分的小区间，然后计算满足条件的小区间的总面积与总面积的比值。

3. 统计法：通过对大量实验数据进行统计，计算满足条件的事件数量与总事件数量的比值。例如，通过对1000次掷骰子的实验，计算出掷出6的事件数量与总事件数量的比值。

### 1.1.3 条件概率

条件概率是一个事件发生的概率，给定另一个事件已发生。例如，一个人在下雨天出门的概率是多少？给定已知今天是下雨天，这个问题的答案是100%。

### 1.1.4 独立事件

独立事件是两个或多个事件之间发生关系不存在的事件，它们的发生不会影响彼此的概率。例如，掷两个骰子的结果是一个独立事件，因为掷第一个骰子的结果不会影响掷第二个骰子的结果。

### 1.3.2 概率论的核心数学模型公式

1. 概率的加法定理：P(A或B) = P(A) + P(B) - P(A和B)

2. 概率的乘法定理：P(A和B) = P(A|B) * P(B) = P(B|A) * P(A)

3. 贝叶斯定理：P(A|B) = P(B|A) * P(A) / P(B)

### 1.3.3 统计的核心算法原理

1. 均值：计算数据集中所有数据点的平均值。

2. 方差：计算数据集中所有数据点与平均值之间的差异的平均值的平方。

3. 标准差：计算数据集中所有数据点与平均值之间的差异的平均值的平方的平根。

```python
from math import factorial

def probability_direct_count(n, r):
    return factorial(n) / (factorial(r) * factorial(n - r))

r = 3
```


### 1.4.3 统计法

```python
from random import randint

def probability_statistic(n, r):
    total_events = 0
    satisfied_events = 0
    for _ in range(n):
        event = randint(1, 10)
        if event <= r:
            satisfied_events += 1
        total_events += 1
    return satisfied_events / total_events

n = 1000
r = 3
print(probability_statistic(n, r))
```

### 1.4.4 贝叶斯定理

```python
def bayes_theorem(p_a, p_b_given_a, p_b):
    p_a_given_b = p_b_given_a * p_a / p_b
    return p_a_given_b

p_a = 0.2
p_b_given_a = 0.8
p_b = 0.3
print(bayes_theorem(p_a, p_b_given_a, p_b))
```

```python
def mean(data):
```

### 1.4.6 方差

```python
def variance(data):
    mean_data = mean(data)
    return sum((x - mean_data) ** 2 for x in data) / len(data)

data = [1, 2, 3, 4, 5]
print(variance(data))
```

### 1.4.7 标准差

```python
def standard_deviation(data):
    variance_data = variance(data)
    return variance_data ** 0.5

data = [1, 2, 3, 4, 5]
print(standard_deviation(data))
```
