## 3.1 线性回归

线性回归是一种常用的机器学习算法，它用于预测一个连续变量的值，根据一个或多个预测变量的值。线性回归的数学模型如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

其中，$y$ 是预测变量，$x_1, x_2, ..., x_n$ 是预测变量，$\beta_0, \beta_1, ..., \beta_n$ 是参数，$\epsilon$ 是误差。

线性回归的目标是找到最佳的参数值，使得预测值与实际值之间的差异最小。这可以通过最小化误差函数来实现：

$$
J(\beta_0, \beta_1, ..., \beta_n) = \frac{1}{2m}\sum_{i=1}^m (y_i - (\beta_0 + \beta_1x_{i1} + \beta_2x_{i2} + ... + \beta_nx_{in}))^2
$$

其中，$m$ 是训练数据的数量。

通过对误差函数的偏导数并设为零，可以得到参数的最佳值：

$$
\frac{\partial J}{\partial \beta_j} = 0, j = 0, 1, ..., n
$$

这样得到的参数值就是线性回归的最佳值。

## 3.2 支持向量机

支持向量机是一种常用的分类算法，它可以用于解决线性可分和非线性可分的分类问题。支持向量机的数学模型如下：

$$
y = w^Tx + b
$$

其中，$w$ 是权重向量，$x$ 是输入向量，$b$ 是偏置。

支持向量机的目标是找到最佳的权重向量和偏置，使得分类错误的样本数最小。这可以通过最小化误差函数来实现：

$$
J(w, b) = \frac{1}{2}w^Tw + C\sum_{i=1}^n \xi_i
$$

其中，$C$ 是正则化参数，$\xi_i$ 是损失函数的惩罚项。

通过对误差函数的偏导数并设为零，可以得到权重向量和偏置的最佳值：

$$
\frac{\partial J}{\partial w} = 0, \frac{\partial J}{\partial b} = 0
$$

这样得到的权重向量和偏置就是支持向量机的最佳值。

## 3.3 梯度下降

梯度下降是一种常用的优化算法，它可以用于最小化函数。梯度下降的数学模型如下：

$$
w_{k+1} = w_k - \alpha \nabla J(w_k)
$$

其中，$w_k$ 是第$k$ 次迭代的权重向量，$\alpha$ 是学习率，$\nabla J(w_k)$ 是函数$J(w_k)$ 的梯度。

梯度下降的目标是找到最佳的权重向量，使得误差函数最小。这可以通过迭代地更新权重向量来实现：

$$
w_{k+1} = w_k - \alpha \nabla J(w_k)
$$

通过重复这个过程，可以得到最佳的权重向量。

# 学习率

# 迭代次数
iterations = 1000

# 训练
for i in range(iterations):
    # 预测
    y_pred = beta_0 + beta_1 * X

    # 误差
    error = y - y_pred

    # 梯度
    grad_beta_0 = error.mean()

    # 更新参数
    beta_0 = beta_0 - alpha * grad_beta_0

# 得到最佳参数
print("最佳参数：", beta_0, beta_1)
