## 3.1 社交网络分析中的核心算法

社交网络分析中的核心算法包括：

- **中心性指数**：中心性指数用于衡量一个节点在社交网络中的中心性，它的计算公式为：

  $$
  C = \frac{1}{N - 1} \sum_{i=1}^{N} \frac{1}{d_i}
  $$

  其中，$N$ 是节点数，$d_i$ 是节点 $i$ 的度。

- ** closeness centrality**： closeness centrality 用于衡量一个节点在社交网络中的接近性，它的计算公式为：

  $$
  C = \frac{N - 1}{\sum_{i=1}^{N} d_i}
  $$

  其中，$N$ 是节点数，$d_i$ 是节点 $i$ 到其他节点的平均距离。

- ** betweenness centrality**： betweenness centrality 用于衡量一个节点在社交网络中的中介性，它的计算公式为：

  $$
  C = \sum_{s \neq i \neq t} \frac{\sigma_{st}(i)}{\sigma_{st}}
  $$

  其中，$s$ 和 $t$ 是节点 $i$ 之间的任意两个节点，$\sigma_{st}(i)$ 是节点 $i$ 在路径 $s$ 到 $t$ 中的数量，$\sigma_{st}$ 是节点 $i$ 之间的任意两个节点的数量。

  def gradient_descent(X, y, theta, alpha, iterations):

      for i in range(iterations):
          gradients = (1 / m) * X.T.dot(X.dot(theta) - y)
          theta -= alpha * gradients

  def backward_propagation(X, y, theta1, theta2, learning_rate):
      m = len(y)
      L = len(theta1)
      L2 = len(theta2)
      gradients = [0] * (L + L2)
      error = (1 / m) * X.T.dot(y - X.dot(theta1) - np.tanh(theta2))
      gradients[-L2:] = (1 / m) * X.T.dot((1 - np.tanh(theta2)) * np.tanh(theta2 - X.dot(theta1)))
      gradients[:L2] = (1 / m) * X.T.dot((1 - np.tanh(theta1)) * (1 - np.tanh(theta1)) * (y - X.dot(theta1) - np.tanh(theta2)))
      gradients = np.array(gradients)
      gradients /= learning_rate
      return gradients
