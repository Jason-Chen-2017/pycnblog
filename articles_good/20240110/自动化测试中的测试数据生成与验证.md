## 4.1 随机测试数据生成的代码实例

```python
import random

def generate_random_data(min_value, max_value, data_size):
    data = []
    for _ in range(data_size):
        value = random.randint(min_value, max_value)
        data.append(value)
    return data

min_value = 1
max_value = 100
data_size = 1000
random_data = generate_random_data(min_value, max_value, data_size)
print(random_data)
```

在上述代码中，我们首先导入了 `random` 模块，然后定义了一个 `generate_random_data` 函数，该函数接受三个参数：`min_value`、`max_value` 和 `data_size`。该函数通过循环生成 `data_size` 个随机整数，并将其存储到 `data` 列表中。最后，我们打印了生成的随机测试数据。

## 4.2 基于规则的测试数据验证的代码实例

```python
def validate_data_rule(data, rule):
    for value in data:
        if not rule(value):
            return False
    return True

def is_even(value):
    return value % 2 == 0

data = [1, 2, 3, 4, 5]
rule = is_even
valid = validate_data_rule(data, rule)
print(valid)
```

在上述代码中，我们首先定义了一个 `validate_data_rule` 函数，该函数接受两个参数：`data` 和 `rule`。该函数通过循环检查 `data` 中的每个值是否满足 `rule` 的条件，如果满足条件，则返回 `True`，否则返回 `False`。然后，我们定义了一个 `is_even` 函数，该函数检查一个整数是否为偶数。接下来，我们定义了一个 `data` 列表，并将其与 `is_even` 函数作为 `rule` 传递给 `validate_data_rule` 函数。最后，我们打印了验证结果。
