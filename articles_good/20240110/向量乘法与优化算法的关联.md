### 1.4.1 梯度下降算法实例

```python
import numpy as np

def gradient_descent(x0, alpha, f, iterations):
    x = x0
    for i in range(iterations):
        grad = f(x)
        x = x - alpha * grad
    return x

def f(x):
    return x**2

x0 = np.random.rand(1)
alpha = 0.1
iterations = 100
x = gradient_descent(x0, alpha, f, iterations)
```

```python
def newton_method(x0, alpha, f, iterations):
```
