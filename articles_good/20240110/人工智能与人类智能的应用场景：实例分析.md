### 4.1 知识表示和推理代码实例

```python
# 知识表示
knowledge = {
    "bird": ["can fly", "has feathers", "lays eggs"],
    "mammal": ["has fur", "gives birth to live young", "sucks milk"],
    "reptile": ["has scales", "lays eggs", "cold-blooded"]
}

# 知识推理
def is_bird(animal):
    for key, value in knowledge.items():
        if key == animal:
            return value
    return False

print(is_bird(animal))  # True
```

在这个代码实例中，我们首先定义了一个名为 `knowledge` 的字典，用于存储知识表示。然后，我们定义了一个名为 `is_bird` 的函数，用于判断一个动物是否是鸟类。最后，我们测试了这个函数，结果显示出“鸢尾”是一种鸟类。

### 4.2 学习和适应代码实例

学习和适应是人工智能算法的另一个基本组成部分，它们可以用于从大量数据中学习出有用的模式，并进行适应。以下是一个简单的学习和适应代码实例：

```python
# 数据
data = [(1, 2), (2, 3), (3, 4), (4, 5)]

# 学习
def learn_linear_regression(data):
    x = [x for x, y in data]
    y = [y for x, y in data]
    m = sum(x) / sum(x**2)
    b = y[0] - m * x[0]
    return m, b

# 适应
def predict(x, m, b):
    return m * x + b

m, b = learn_linear_regression(data)
x = 5
print(predict(x, m, b))  # 7.0
```

在这个代码实例中，我们首先定义了一个名为 `data` 的列表，用于存储数据。然后，我们定义了一个名为 `learn_linear_regression` 的函数，用于学习线性回归模型。最后，我们定义了一个名为 `predict` 的函数，用于根据学习到的模型进行预测。最后，我们测试了这个预测函数，结果显示出当 x 为 5 时，预测值为 7.0。

### 4.3 语言理解和生成代码实例

语言理解和生成是人工智能算法的另一个基本组成部分，它们可以用于与人类进行自然的对话，并生成自然语言的文本。以下是一个简单的语言理解和生成代码实例：

```python
# 自然语言处理库
import nltk

# 语言理解
def language_understanding(text):
    words = nltk.word_tokenize(text)
    tags = nltk.pos_tag(words)
    return tags

# 语言生成
def language_generation(tags):
    words = [word for word, tag in tags]
    text = " ".join(words)
    return text

text = "The quick brown fox jumps over the lazy dog."
tags = language_understanding(text)
print(tags)  # [('The', 'DT'), ('quick', 'JJ'), ('brown', 'NN'), ('fox', 'NN'), ('jumps', 'VBZ'), ('over', 'IN'), ('the', 'DT'), ('lazy', 'JJ'), ('dog', 'NN'), ('.', '.')]
text = language_generation(tags)
print(text)  # The quick brown fox jumps over the lazy dog.
```

在这个代码实例中，我们首先导入了一个名为 `nltk` 的自然语言处理库。然后，我们定义了一个名为 `language_understanding` 的函数，用于对自然语言文本进行理解。最后，我们定义了一个名为 `language_generation` 的函数，用于根据理解的结果生成自然语言文本。最后，我们测试了这个生成函数，结果显示出生成的文本与原文本相同。
