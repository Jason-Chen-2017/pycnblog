### 3.3 数学模型公式详细讲解

多模态学习在游戏领域的数学模型公式详细讲解包括：

- 卷积神经网络（CNN）：CNN是一种用于处理图像数据的神经网络，其公式如下：

$$
y = f(\sum_{i=1}^{k} w_i * x_{i:i+h} + b)
$$

其中，$x$ 是输入图像，$w_i$ 是卷积核，$h$ 是卷积核大小，$b$ 是偏置项，$f$ 是激活函数。

- 循环神经网络（RNN）：RNN是一种用于处理序列数据的神经网络，其公式如下：

$$
h_t = f(\sum_{i=1}^{n} w_i * h_{t-1} + v_i * x_t + b)
$$

其中，$x_t$ 是输入序列，$h_{t-1}$ 是上一个时间步的隐藏状态，$w_i$ 是权重，$v_i$ 是输入权重，$b$ 是偏置项，$f$ 是激活函数。

- 自注意力机制（Attention）：Attention 是一种用于关注输入序列中重要部分的机制，其公式如下：

$$
a_{ij} = \frac{\exp(s(h_i, h_j))}{\sum_{k=1}^{T} \exp(s(h_i, h_k))}
$$

$$
c_i = \sum_{j=1}^{T} a_{ij} * h_j
$$

其中，$h_i$ 是输入序列，$a_{ij}$ 是注意力分数，$c_i$ 是注意力聚焦后的输出。

def mcts_select(node):
    while node.children:
        child = max(node.children, key=lambda child: child.value / (1 + child.visits))

    return node

# 扩展节点
def mcts_expand(node):
    actions = node.game_state.get_valid_actions()
    for action in actions:

        node.value += value
        node.visits += 1
        node = node.parent
