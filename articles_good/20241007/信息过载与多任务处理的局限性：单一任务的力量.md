#### 4.1 数学概念和符号

- **任务完成度（Completion Rate）**：表示任务完成的程度，通常用百分比表示。任务完成度可以表示为：
  $$ C = \frac{完成任务所需时间}{总时间} $$
  
- **任务切换成本（Task Switching Cost）**：表示在处理任务时进行切换所需的时间和其他资源。任务切换成本可以表示为：
  $$ S = \alpha \times T $$
  其中，$ \alpha $ 为任务切换系数，$ T $ 为任务切换所需时间。

- **认知负担（Cognitive Load）**：表示在处理任务时所需的认知资源，包括注意力、记忆和思考等。认知负担可以表示为：
  $$ L = \beta \times T $$
  其中，$ \beta $ 为认知负担系数，$ T $ 为任务处理时间。

- **时间窗口（Time Window）**：表示在特定时间段内可以处理的任务数量。时间窗口可以表示为：
  $$ W = t \times n $$
  其中，$ t $ 为时间段长度，$ n $ 为任务数量。

#### 5.2 源代码详细实现和代码解读

```python
import numpy as np

# 定义任务类
class Task:
    def __init__(self, name, duration, switch_cost, cognitive_load):
        self.name = name
        self.duration = duration
        self.switch_cost = switch_cost
        self.cognitive_load = cognitive_load

# 定义单一任务处理算法
def single_task_processing(tasks, time_window):
    completion_rate = 0
    completed_tasks = []

    # 初始化任务队列
    task_queue = tasks.copy()
    task_queue.sort(key=lambda x: x.duration)

    # 循环处理任务
    while task_queue:
        current_task = task_queue.pop(0)
        remaining_time = current_task.duration

        # 如果剩余时间小于时间窗口
        if remaining_time < time_window:
            completion_rate += 1
            completed_tasks.append(current_task.name)
            print(f"完成任务：{current_task.name}")
        else:
            # 将当前任务加入待办队列
            task_queue.append(current_task)
            print(f"任务 {current_task.name} 暂停，待后续处理。")

    return completion_rate, completed_tasks

# 测试代码
if __name__ == "__main__":
    # 创建任务列表
    tasks = [
        Task("任务1", 10, 1, 1.5),
        Task("任务2", 20, 1, 1.5),
        Task("任务3", 10, 1, 1.5)
    ]

    # 设置时间窗口
    time_window = 60

    # 调用单一任务处理算法
    completion_rate, completed_tasks = single_task_processing(tasks, time_window)

    # 输出结果
    print(f"完成度：{completion_rate}")
    print(f"已完成的任务：{completed_tasks}")
```

1. **任务持续时间**：根据任务的预计持续时间，选择适当的时间窗口。
