### 5.2 源代码详细实现和代码解读

#### 5.2.1 合约代码

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract DataCopyright {

    // 数据指纹映射
    mapping(uint => string) public fingerprints;

    // 用户权限映射
    mapping(address => mapping(uint => bool)) public userPermissions;

    // 添加数据指纹
    function addFingerprint(uint index, string memory fingerprint) public {
        fingerprints[index] = fingerprint;
    }

    // 授予用户权限
    function grantPermission(address user, uint index) public {
        userPermissions[user][index] = true;
    }

    // 检查用户权限
    function checkPermission(address user, uint index) public view returns (bool) {
        return userPermissions[user][index];
    }

}
```

#### 5.2.2 代码解读

1. **数据指纹映射**：使用`mapping`定义一个指纹映射，用于存储数据指纹；
2. **用户权限映射**：使用`mapping`定义一个用户权限映射，用于存储用户的权限信息；
3. **添加数据指纹**：通过`addFingerprint`函数添加数据指纹到区块链；
4. **授予用户权限**：通过`grantPermission`函数授予用户对特定数据指纹的访问权限；
5. **检查用户权限**：通过`checkPermission`函数检查用户对特定数据指纹的访问权限。

#### 5.2.3 代码实现

1. **部署合约**：

```bash
truffle migrate --network development
```

2. **添加数据指纹**：

```python
from web3 import Web3
import json

# 连接到本地以太坊节点
web3 = Web3(Web3.HTTPProvider('http://127.0.0.1:8545'))

# 加载合约ABI和地址
with open('build/contracts/DataCopyright.json', 'r') as f:
    contract_abi = json.load(f)['abi']
with open('contract_address.txt', 'r') as f:
    contract_address = f.read()

# 创建合约实例
contract = web3.eth.contract(address=contract_address, abi=contract_abi)

# 添加数据指纹
contract.functions.addFingerprint(0, '0xhash_value').transact({'from': web3.eth.coinbase})
```

3. **授予用户权限**：

```python
# 授予用户权限
contract.functions.grantPermission('0xuser_address', 0).transact({'from': web3.eth.coinbase})
```

4. **检查用户权限**：

```python
# 检查用户权限
permission = contract.functions.checkPermission('0xuser_address', 0).call()
print(permission)
```
