### 4.2 公式推导过程

1. **计算相似度**：首先，计算查询向量$Q$和键向量$K$之间的相似度。相似度的计算公式为：

   $$  
   \text{normalized\_similarity}(Q, K) = \frac{\text{similarity}(Q, K)}{\sqrt{d_k}}

3. **应用softmax函数**：最后，应用softmax函数对归一化相似度进行转换，得到概率分布。softmax函数的公式为：

   $$  
   \text{softmax}(x) = \frac{e^x}{\sum_{i=1}^n e^x_i}  
   $$

   其中，$x$表示输入向量，$n$表示向量的维度。

```python  
import numpy as np  
import qiskit

# 初始化量子比特  
qubit = qiskit.QuantumRegister(3, name='q')  
circuit = qiskit.ClassicalRegister(3, name='c')  
qc = qiskit.QuantumCircuit(qubit, circuit)

# 初始化量子比特为叠加态  
qc.h(qubit[0])  
qc.h(qubit[1])  
qc.h(qubit[2])

# 应用量子门实现注意力分配  
qc.ccx(qubit[0], qubit[1], qubit[2])  
qc.cx(qubit[0], qubit[1])  
qc.cx(qubit[0], qubit[2])  
qc.ccx(qubit[1], qubit[2], qubit[0])

# 测量量子比特得到最终结果  
qc.measure(qubit, circuit)

# 运行量子电路  
backend = qiskit.Aer.get_backend('qasm_simulator')  
qc.run(backend, shots=1000).result().get_counts()

# 输出结果  
print(qc.draw())

# 分析结果  
print(qiskit.visualization.plot_barchart(qc.run(backend, shots=1000).result().get_counts()))
```
