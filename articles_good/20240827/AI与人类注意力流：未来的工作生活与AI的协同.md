## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

为了描述人类注意力流，我们可以构建一个线性回归模型。设\(x_1, x_2, ..., x_n\)为n个时间点的人类注意力流数据，\(y_1, y_2, ..., y_n\)为相应的任务完成情况数据，则线性回归模型可表示为：

$$y_i = \beta_0 + \beta_1 x_i + \epsilon_i$$

其中，\(\beta_0\)和\(\beta_1\)为模型的参数，\(\epsilon_i\)为随机误差。

### 4.2 公式推导过程

为了推导线性回归模型的公式，我们首先定义损失函数：

$$L(\beta_0, \beta_1) = \sum_{i=1}^n (y_i - (\beta_0 + \beta_1 x_i))^2$$

为了最小化损失函数，我们对\(\beta_0\)和\(\beta_1\)分别求导，并令导数为0，得到：

$$\frac{\partial L}{\partial \beta_0} = -2\sum_{i=1}^n (y_i - (\beta_0 + \beta_1 x_i)) = 0$$

$$\frac{\partial L}{\partial \beta_1} = -2\sum_{i=1}^n x_i (y_i - (\beta_0 + \beta_1 x_i)) = 0$$

解上述方程组，即可得到\(\beta_0\)和\(\beta_1\)的值。

### 4.3 案例分析与讲解

假设某人在一天中完成了5个任务，其注意力流数据如下表所示：

| 时间 | 注意力流 |
|------|----------|
| 8:00 | 0.4      |
| 9:00 | 0.6      |
| 10:00| 0.8      |
| 11:00| 0.5      |
| 12:00| 0.3      |

对应的任务完成情况如下表所示：

| 时间 | 任务完成情况 |
|------|-------------|
| 8:00 | 完成        |
| 9:00 | 完成        |
| 10:00| 完成        |
| 11:00| 未完成      |
| 12:00| 完成        |

我们可以使用线性回归模型来预测第11个任务的完成情况。根据4.1节中的模型构建，我们有：

$$y_i = \beta_0 + \beta_1 x_i$$

将数据代入，得到：

$$\beta_0 = 0.55, \beta_1 = 0.25$$

因此，第11个任务的完成情况预测为：

$$y_{11} = 0.55 + 0.25 \times 0.5 = 0.80$$

预测结果为80%，即有80%的概率完成任务。

### 5.2 源代码详细实现

以下是实现线性回归模型的Python代码：

```python
import numpy as np
import matplotlib.pyplot as plt

def linear_regression(x, y):
    # 添加常数项，转换为矩阵形式
    X = np.column_stack([np.ones(len(x)), x])
    # 求解参数
    beta = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)
    return beta

def predict(x, beta):
    # 预测结果
    y_pred = beta[0] + beta[1] * x
    return y_pred

# 数据
x = np.array([0.4, 0.6, 0.8, 0.5, 0.3])
y = np.array([1, 1, 1, 0, 1])

# 训练模型
beta = linear_regression(x, y)

# 预测
x_new = 0.5
y_pred = predict(x_new, beta)

print(f"预测结果：{y_pred}")

# 可视化
plt.scatter(x, y)
plt.plot(x, np.array([predict(x_i, beta) for x_i in x]), color='red')
plt.show()
```

### 5.3 代码解读与分析

- `linear_regression`函数用于求解线性回归模型的参数。输入为x和y的数据，输出为参数beta。
- `predict`函数用于预测新的x值对应的y值。
- 使用`np.column_stack`将x和常数项1组合成一个矩阵X。
- 使用`np.linalg.inv`求解X的逆矩阵，与X.T.dot(X)相乘得到参数beta。
- 使用`plt.scatter`绘制原始数据点，使用`plt.plot`绘制回归线。
