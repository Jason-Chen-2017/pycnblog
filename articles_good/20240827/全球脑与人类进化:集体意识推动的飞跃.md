### 5.2 源代码详细实现

以下是一个简单的全球脑算法实现，用于计算节点的影响力和全局影响力。

```python
import numpy as np
import networkx as nx

def compute_influence(G, x, p):
    """
    计算节点的影响力和全局影响力。

    参数：
    G：图对象，表示全球脑。
    x：特征向量，表示节点的知识、能力和状态等信息。
    p：传播概率矩阵，表示节点之间的互动关系。

    返回：
    I：节点影响力列表，I[i]表示节点i的影响力。
    IG：全局影响力，表示全球脑的整体影响力。
    """
    # 计算节点影响力
    I = np.dot(p, x)
    
    # 计算全局影响力
    IG = np.sum(I)
    
    return I, IG

# 创建图对象
G = nx.Graph()

# 添加节点和边
G.add_nodes_from([1, 2, 3, 4])
G.add_edges_from([(1, 2), (1, 3), (2, 3), (3, 4)])

# 定义特征向量
x = np.array([0.2, 0.3, 0.4, 0.5])

# 定义传播概率矩阵
p = np.array([[0.5, 0.2, 0.3], [0.2, 0.5, 0.3], [0.3, 0.2, 0.5], [0.1, 0.1, 0.8]])

# 计算节点影响力和全局影响力
I, IG = compute_influence(G, x, p)

print("节点影响力：", I)
print("全局影响力：", IG)
```

1. **节点影响力计算**：

   在函数中，我们首先通过`np.dot(p, x)`计算节点影响力。这里，`p`是传播概率矩阵，表示节点之间的互动关系；`x`是特征向量，表示节点的知识、能力和状态等信息。`np.dot`函数用于计算矩阵乘积，即每个节点的邻接矩阵与特征向量的内积。

2. **全局影响力计算**：

   接着，我们通过`np.sum(I)`计算全局影响力，即所有节点影响力的加权和。

3. **案例应用**：

   在主程序中，我们创建了一个简单的图对象`G`，表示全球脑。然后，我们定义了特征向量`x`和传播概率矩阵`p`，分别表示节点的知识和互动关系。最后，我们调用`compute_influence`函数，计算节点影响力和全局影响力，并打印结果。
