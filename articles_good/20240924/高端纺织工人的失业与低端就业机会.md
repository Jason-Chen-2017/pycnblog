\[ E = f(T, P, M) \]

- **生产过程自动化水平 \( P \)**：反映了生产过程中自动化的程度。自动化水平越高，生产效率越高。
- **机器技术水平 \( M \)**：反映了机器的性能和智能化程度。机器技术水平越高，生产效率越高。

举例说明：假设一个纺织厂有100名工人，其中50名具备高技能水平，另外50名技能水平较低。如果生产过程中90%的任务实现了自动化，且使用的机器技术水平较高，那么生产效率可以表示为：

\[ E = f(0.5, 0.9, 1) = 0.5 \times 0.9 \times 1 = 0.45 \]

即生产效率为45%。

```python
    return contours

    for contour in contours:

        if 100 < area < 500:
            x, y, w, h = cv2.boundingRect(contour)
            aspect_ratio = w / float(h)
            if 0.5 < aspect_ratio < 2:
                fibers.append(contour)
    return fibers
```
