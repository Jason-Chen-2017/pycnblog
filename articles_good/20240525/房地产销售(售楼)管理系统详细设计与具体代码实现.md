### 4.1 房产价格计算

房产价格可以通过以下公式计算：

$$
P = A \times R
$$

其中，$P$是房产价格，$A$是房产面积，$R$是房产价格_per_unit_area。

举例：假设一套房产的面积为100平方米，价格_per_unit_area 为3000元/平方米，那么房产价格为：

$$
P = 100 \times 3000 = 300000元
$$

```python
import sqlite3

# 连接数据库
conn = sqlite3.connect('property_management.db')
cursor = conn.cursor()

# 创建数据库表
cursor.execute('''
    CREATE TABLE IF NOT EXISTS property (
        id INTEGER PRIMARY KEY,
        address TEXT,
        area INTEGER,
        price INTEGER,
        image TEXT,
        video TEXT
    )
''')

cursor.execute('''
    CREATE TABLE IF NOT EXISTS customer (
        id INTEGER PRIMARY KEY,
        name TEXT,
        contact TEXT,
        demand TEXT
    )
''')

cursor.execute('''
    CREATE TABLE IF NOT EXISTS transaction (
        id INTEGER PRIMARY KEY,
        property_id INTEGER,
        customer_id INTEGER,
        transaction_time TEXT,
        transaction_status TEXT,
        FOREIGN KEY (property_id) REFERENCES property(id),
        FOREIGN KEY (customer_id) REFERENCES customer(id)
    )
''')

# 添加房产信息
property_id = 1
address = '上海市浦东新区'
area = 100
price = 300000
image = 'http://example.com/image.jpg'
video = 'http://example.com/video.mp4'
cursor.execute('''
    INSERT INTO property (id, address, area, price, image, video)
    VALUES (?, ?, ?, ?, ?, ?)
''', (property_id, address, area, price, image, video))

# 查询房产信息
cursor.execute('''
    SELECT * FROM property WHERE id = ?
''', (property_id,))

property = cursor.fetchone()
print(property)

# 修改房产信息
new_price = 320000
cursor.execute('''
    UPDATE property SET price = ? WHERE id = ?
''', (new_price, property_id))

# 删除房产信息
cursor.execute('''
    DELETE FROM property WHERE id = ?
''', (property_id,))

# 查询交易记录
cursor.execute('''
    SELECT * FROM transaction WHERE customer_id = ?
''', (customer_id,))

transactions = cursor.fetchall()
print(transactions)

# 关闭数据库连接
conn.close()
```
