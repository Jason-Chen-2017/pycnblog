### 5.2 源代码详细实现

以下是元宇宙数字孪生伦理委员会的核心算法实现：

```python
import pandas as pd
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.metrics.pairwise import cosine_similarity

class EthicalCommittee:
    def __init__(self, ethical_rules, threshold):
        self.ethical_rules = ethical_rules
        self.threshold = threshold

    def calculate_score(self, content):
        scores = {}
        for rule in self.ethical_rules:
            score = cosine_similarity([content], [rule])[0][0]
            scores[rule] = score
        return scores

    def make_decision(self, content):
        scores = self.calculate_score(content)
        total_score = sum(scores.values())
        decision_score = total_score / len(scores)
        if decision_score >= self.threshold:
            return "Accept"
        else:
            return "Reject"

if __name__ == "__main__":
    ethical_rules = ["Hate speech is not allowed.", "Pornography is not allowed.", "False information is not allowed."]
    committee = EthicalCommittee(ethical_rules, 0.5)
    content = "This is an article discussing social issues."
    decision = committee.make_decision(content)
    print(f"The decision for the content is: {decision}")
```

- **make_decision方法**：根据得分计算决策结果，判断是否接受内容。
