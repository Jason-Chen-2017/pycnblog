## 5. 项目实践：代码实例和详细解释说明

### 5.1 开发环境搭建

本文使用Python语言进行古印度数字系统的实现。

### 5.2 源代码详细实现

```python
def add(a, b):
    result = []
    carry = 0
    for i in range(max(len(a), len(b))):
        x = a[i] if i < len(a) else 0
        y = b[i] if i < len(b) else 0
        sum = x + y + carry
        result.append(sum % 10)
        carry = sum // 10
    if carry:
        result.append(carry)
    return result

# 测试代码
print(add([1, 2, 3], [4, 5, 6]))
```
