# 1.简介
  

2021年是互联网行业的一年。相对于过去的十几年间，互联网的崛起极大地促进了科技创新和产业变革。2021年以来，高校毕业生数量持续增加，大学生成为全球最具吸引力的职业选择之一。近年来，国内外越来越多的人开始选择加入互联网大厂，互联网行业火爆。相信每个人的心中都有自己的理想，但如何找到自己满意的工作却是一个难题。一方面，在短时间内找到满意的工作是件很难的事情；另一方面，选择恰当的企业拓展自己的能力也是非常重要的。因此，招聘网站作为一个时尚、功能丰富且受众广泛的门户网站，不仅能够给求职者提供丰富的招聘信息，还能帮助企业展示自己的产品或服务，增强其知名度，从而更容易被求职者邀约加入。

然而，现在招聘网站的界面设计并不统一，有的采用大图标+列表的形式，有的采用单列列表的形式，有的采用卡片式布局，这让用户阅读起来比较费劲。另外，有的网站通过搜索框找寻岗位，有的网站直接跳转到招聘页面，使得用户的使用体验较差。在这种情况下，怎样才能做到招聘频道的广泛覆盖？让所有用户都能快速了解到公司招聘的信息，并且可以直接申请到相关岗位呢？为了解决这个问题，本文将介绍一种新的招聘频道设计方式——通用搜索页面。

# 2.基本概念术语说明
## 2.1. 什么是招聘网站？
招聘网站，也称人才网、职位网、职业社区等，是网络招聘平台的统称。它是将招聘需求发布在互联网上，让求职者及求职者群体可以通过搜索、筛选、浏览、投递的方式找到合适的工作机会。招聘网站除了提供职位信息外，还承担招聘需求收集、跟踪招聘人员、简历审核、招聘成果展示等一系列职能。各类招聘网站的名称和特点各异，如猎头、智联招聘、Boss直聘、拉钩、顶住房贷、猎聘网等。

## 2.2. 什么是通用搜索页面？
通用搜索页面，又称为通用查询页面，是指一个网站上有多个不同类型的搜索栏目，通过统一的搜索框进行快速搜索的页面。一般情况下，通用搜索页面包括职位搜索、招聘项目搜索、招聘信息搜索、人才库搜索等，用户可以根据需求选择对应的搜索栏目，并输入关键词进行搜索。

## 2.3. 什么是通用导航页？
通用导航页，是指一个网站上有多个不同类型的内容菜单，通过不同的链接显示不同种类的信息的页面。一般情况下，通用导航页包括首页、新闻中心、产品展示、联系我们等，通过不同的链接，可以访问到不同类型的信息。

## 2.4. 为什么要设计通用搜索页面和导航页？
当前的招聘网站普遍存在以下问题：

1. 复杂搜索规则。目前的招聘网站搜索规则繁多，用户不易识别。
2. 无法直观地获得搜索结果。用户只能看到排版整齐的搜索结果，很少能直观地看到搜索结果的概括。
3. 搜索结果质量低下。搜索结果往往呈现的是全站的、非专业化的推荐，且不具有可靠性。
4. 不方便对结果排序。很多招聘网站的搜索结果都是默认排序的，用户很难把握高效率的筛选策略。
5. 用户体验差。用户在浏览过程中，需要先点击不同的菜单项、链接，再输入关键字进行搜索，效率低下。

这些问题，归根结底源自于招聘网站的缺乏通用性和可用性。通用搜索页面和导航页的设计目标就是要通过简洁、易用、精准的界面，实现信息的集中、分类、检索、分析和呈现。从而提升用户的搜索体验和效率，最终实现招聘网站的发展方向和规模。

# 3.核心算法原理和具体操作步骤以及数学公式讲解
## 3.1. 设计原则
为了实现招聘频道的广泛覆盖，我建议按照如下几个方面进行设计：

1. 使用户能快速找到相关招聘信息。在导航栏里设置搜索按钮，让用户点击一个链接即可进入搜索页面，这样用户就可以快速定位到搜索页面。
2. 提供简明易懂的搜索结果。通过标签化的分类系统，对搜索结果进行细致的划分，让用户在一瞬间就能知道招聘结果的大致范围。
3. 提供多种筛选条件。用户可以在搜索页面的右侧提供多个筛选条件，让用户根据自身的兴趣、能力、工作经验等因素进行选择，快速找到所需信息。
4. 充分利用互联网特性。借助互联网特性，如流量红利、数据增长、规模经济等，提升网站的收益。
5. 使用设计感。为了达到招聘频道广泛覆盖的目的，除了简单易用的导航栏和搜索框，还可以设计出符合品牌风格的设计风格、色彩搭配和信息结构。

## 3.2. 搜索结果展示
搜索结果展示是一个重要环节。为了提供简明易懂的搜索结果，我建议采取如下方式：

1. 通过标签化的分类系统，对搜索结果进行细致的划分。我认为标签化的分类系统能够更好地突出重要信息。例如，给与招聘岗位相关的标签如“人力资源”、“财务”、“IT工程师”等，给与公司信息相关的标签如“湖南大学”、“成都科技有限公司”等。通过标签的清晰划分，可以让搜索结果更加容易理解。
2. 对搜索结果进行排序。用户搜索得到的结果需要根据用户需求进行排序，比如按照最热门的、最新发布的、相关度高的等。
3. 在搜索结果的详情页提供公司信息。给予用户详细的公司信息，如公司主营业务、福利待遇、地址、电话号码等，能够让用户更好地确定是否适合自己的工作。
4. 鼓励用户分享招聘信息。鼓励用户分享自己的招聘信息，对于提升招聘信息的曝光率和热度具有积极作用。

## 3.3. 过滤条件的设计
搜索页面的过滤条件是搜索页面的一个亮点。它能够帮助用户准确地找到所需的信息，从而缩小搜索范围，提高搜索效率。我认为，在搜索页面的过滤条件设计上，应遵循如下原则：

1. 清楚明白的过滤条件。对用户来说，过滤条件是最难掌握的部分，所以首先应该清楚地告诉用户哪些选项是可以用来筛选的。同时，需要提供足够多的过滤条件，这样用户就可以灵活组合各种条件进行搜索。
2. 根据招聘岗位的要求进行筛选。根据招聘岗位的特征，制定相应的筛选条件，如要求男性/女性、应届生/工作经验、薪酬水平、技术水平、工作年限、公司规模等。通过设定这些具体的筛选条件，能够帮用户更准确地找到符合自己的需求的招聘信息。
3. 考虑用户可能经常使用的情况。用户可能经常使用相同的关键词进行搜索，因此可以预置一些常用的搜索词，减少搜索复杂度。同时，还可以针对常用的搜索词，给予特殊的提示，提升搜索效果。

## 3.4. 模板页面的制作
模板页面，是指一个完整的页面框架，它包括导航栏、搜索框、搜索结果、底部链接等。每款网站的设计风格不同，模板页面也有所不同。但是，我发现很多招聘网站的模板页面都存在很多重复的内容。因此，我建议对模板页面进行制作优化。

1. 导航栏和搜索框。对于已经固定下来的导航栏和搜索框，不要在模板页面中重新设计，只需要调整位置即可。
2. 搜索结果区域的模块化设计。搜索结果页面可以按照职位类别（如技术类、人力类、销售类）、城市、工作年限等维度，将搜索结果区域划分为多个模块。
3. 底部的链接。在模板页面的底部，可以设置一些便于用户获取更多信息的链接，如招聘流程、报名攻略、培训课程、相关法律法规等。

## 3.5. 流量推广的运用
流量推广，是通过各种方式将广告或者其他类型的付费宣传活动带入到目标群体中，以此来增加网站的日活跃用户和推广品牌形象。招聘频道的设计涉及到用户量较大，使用流量推广可以有效提升招聘信息的转化率。

1. 展示各种媒体渠道。在招聘频道的广告位上，可以展示各大媒体的相关视频、图片等内容，增加招聘信息的曝光率。
2. 提供短期优惠。为用户提供短期的优惠券、会员权益等，鼓励用户参加活动，提高用户黏性。
3. 建立友好的交流机制。鼓励用户分享自己的故事、感受和建议，建立互动氛围，提升互动度。
