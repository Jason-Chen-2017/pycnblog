## 2.核心概念与联系

### 2.1模块化

模块化是一种将软件系统划分为多个模块的方法，每个模块都有自己的功能和职责。模块化有助于提高代码的可读性、可维护性和可重用性。在框架设计中，模块化可以让我们将不同的功能组件分开开发和维护，从而更好地控制项目的复杂度。

### 2.2组件化

组件化是一种将软件系统划分为多个组件的方法，每个组件都有自己的功能和职责。组件化有助于提高代码的可重用性和可扩展性。在框架设计中，组件化可以让我们将不同的功能组件组合在一起，从而更好地实现功能的复用和扩展。

### 2.3依赖注入

依赖注入是一种将对象之间的依赖关系通过构造函数或接口注入的方法。依赖注入有助于提高代码的可测试性和可维护性。在框架设计中，依赖注入可以让我们将不同的组件之间的依赖关系明确定义，从而更好地控制组件之间的关系和依赖。

### 2.4反射

反射是一种在运行时动态获取和操作类的信息和功能的方法。反射有助于提高代码的可扩展性和灵活性。在框架设计中，反射可以让我们在运行时动态地加载和使用不同的组件，从而更好地实现功能的扩展和灵活性。

### 4.2组件化示例

在这个示例中，我们将实现一个简单的组件化系统，包括一个计算器组件和一个日志组件。我们将使用Python的组件化机制来实现这个系统。

```python
# 计算器组件
from abc import ABC, abstractmethod

class Calculator(ABC):
    @abstractmethod
    def add(self, a, b):
        pass

    @abstractmethod
    def sub(self, a, b):
        pass

# 实现计算器组件
class SimpleCalculator(Calculator):
    def add(self, a, b):
        return a + b

    def sub(self, a, b):
        return a - b

# 日志组件
from abc import ABC, abstractmethod

class Logger(ABC):
    @abstractmethod
    def log(self, message):
        pass

# 实现日志组件
class SimpleLogger(Logger):
    def log(self, message):
        logging.info(message)
```

### 4.3依赖注入示例

在这个示例中，我们将实现一个简单的依赖注入系统，包括一个计算器组件和一个日志组件。我们将使用Python的依赖注入机制来实现这个系统。

```python
# 依赖注入容器
from collections import namedtuple

Dependency = namedtuple("Dependency", ["component", "name"])

def get_component(name):
    if name == "calculator":
        return SimpleCalculator()
    elif name == "logger":
        return SimpleLogger()
    else:
        raise ValueError(f"Unknown component: {name}")

# 使用依赖注入容器
def add(a, b, calculator=Dependency("calculator", "calculator")):
    calculator = get_component(calculator.name)
    return calculator.add(a, b)

def sub(a, b, calculator=Dependency("calculator", "calculator")):
    calculator = get_component(calculator.name)
    return calculator.sub(a, b)
```

### 4.4反射示例

在这个示例中，我们将实现一个简单的反射系统，包括一个计算器组件和一个日志组件。我们将使用Python的反射机制来实现这个系统。

```python
# 反射系统
import types

def get_component(name):
    component = globals().get(name)
    if not isinstance(component, types.ModuleType):
        raise ValueError(f"Unknown component: {name}")
    return component

# 使用反射系统
def add(a, b, calculator="calculator"):
    calculator = get_component(calculator)
    return calculator.add(a, b)

def sub(a, b, calculator="calculator"):
    calculator = get_component(calculator)
    return calculator.sub(a, b)
```
