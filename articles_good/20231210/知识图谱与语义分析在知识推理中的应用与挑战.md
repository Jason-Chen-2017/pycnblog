## 4.3 知识推理

### 4.3.1 推理规则

```python
from rdfine import RDFine
from rdflib import Graph, Namespace

ns = Namespace("http://example.com/")
g = Graph()

g.add((ns.A, ns.age, ns.B))

query = """
ASK {
    ?x ?p ?y .
    FILTER (?p = ns:age)
    FILTER (?x = ns:A)
    FILTER (?y = ns:B)
}
"""

results = g.query(query)

if results:
    print("A's age is greater than B's age.")
else:
    print("A's age is not greater than B's age.")
```
