低通滤波是一种用于减少图像中噪声影响的算法。低通滤波的核心思想是利用数学模型来描述图像的特征，并根据这些特征进行处理和分析。低通滤波的数学模型公式为：

$$

其中，$G(u,v)$ 是滤波后的图像，$g(m,n)$ 是原始图像，$h(u-m,v-n)$ 是滤波核。

### 3.1.2 高通滤波

$$

其中，$G(u,v)$ 是滤波后的图像，$g(m,n)$ 是原始图像，$h(u-m,v-n)$ 是滤波核。

### 3.1.3 边缘检测

## 4.2 视频编码代码实例
```python
import cv2
import numpy as np

# 读取视频
cap = cv2.VideoCapture('sample.mp4')

# 视频编码
fourcc = cv2.VideoWriter_fourcc(*'XVID')
out = cv2.VideoWriter('output.avi', fourcc, 20.0, (640, 480))

# 读取视频帧
while True:
    ret, frame = cap.read()
    if not ret:
        break

    # 压缩视频帧
    compressed_frame = cv2.compress(frame, cv2.IMWRITE_AVI_CODEC)

    # 写入压缩视频帧
    out.write(compressed_frame)

# 释放资源
cap.release()
out.release()

# 显示结果
cv2.imshow('Original Frame', frame)
cv2.imshow('Compressed Frame', compressed_frame)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
