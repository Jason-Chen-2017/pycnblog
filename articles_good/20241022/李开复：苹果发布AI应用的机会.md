### 3.1 AI应用的定义与特征

AI应用是指利用人工智能技术，如机器学习、深度学习、自然语言处理等，实现特定任务和目标的软件系统或硬件设备。AI应用具有以下几个特征：

- **自动性**：AI应用能够自动执行任务，无需人工干预。
- **自适应**：AI应用可以根据输入数据进行自我调整和优化，以适应不同的环境和需求。
- **智能化**：AI应用能够模拟人类智能，实现图像识别、语音识别、自然语言理解等高级功能。
- **高效性**：AI应用能够快速处理大量数据，提供准确和高效的决策。

人工智能（AI）技术的发展离不开一系列核心技术的支持。在这些技术中，机器学习、深度学习、自然语言处理（NLP）和计算机视觉等技术尤为重要。以下将详细介绍这些技术的原理、分类和应用。

### 4.1 机器学习算法

机器学习（Machine Learning，ML）是一种让计算机通过数据学习并做出预测或决策的方法。它主要分为以下几类：

#### 4.1.1 监督学习算法

监督学习算法是指通过已标记的训练数据来训练模型，然后使用模型进行预测。常见的监督学习算法包括：

- **线性回归（Linear Regression）**：线性回归是一种用于预测数值型数据的简单算法。它通过找到一个线性函数来拟合数据，使得预测值与实际值之间的误差最小。
  $$ y = w_0 + w_1 \cdot x $$
  
- **决策树（Decision Tree）**：决策树是一种树形结构，通过一系列规则来对数据进行分类或回归。每个节点代表一个特征，每个分支代表该特征的取值。
  
- **支持向量机（Support Vector Machine，SVM）**：支持向量机是一种用于分类和回归的强大算法。它通过找到一个最优的超平面来分隔数据，使得分类边界最大化。

#### 4.1.2 无监督学习算法

无监督学习算法是指在没有标记数据的情况下，让计算机自动发现数据中的模式。常见的无监督学习算法包括：

- **K-均值聚类（K-Means Clustering）**：K-均值聚类是一种基于距离度量的聚类算法。它通过将数据点划分为K个簇，使得簇内数据点之间的距离最小。
  
- **主成分分析（Principal Component Analysis，PCA）**：主成分分析是一种降维技术，通过将数据投影到新的正交坐标系上，以减少数据的维度，同时保留主要的信息。

#### 4.1.3 强化学习算法

强化学习算法是一种通过试错来学习最优策略的算法。它通过奖励机制来指导学习过程，常见的强化学习算法包括：

- **Q学习（Q-Learning）**：Q学习是一种基于值函数的强化学习算法。它通过更新Q值来选择最优动作，使得策略逐渐收敛到最优解。

- **深度强化学习（Deep Reinforcement Learning）**：深度强化学习结合了深度学习和强化学习，通过神经网络来近似值函数或策略。它被广泛应用于复杂的决策问题，如自动驾驶和游戏AI。

### 4.2 深度学习模型

深度学习（Deep Learning，DL）是一种基于多层神经网络的学习方法。它通过自动提取数据的特征表示，使得计算机能够从大量数据中学习复杂的模式。以下是几种常见的深度学习模型：

#### 4.2.1 卷积神经网络（CNN）

卷积神经网络（Convolutional Neural Network，CNN）是一种用于图像处理的深度学习模型。它通过卷积层、池化层和全连接层来提取图像的特征。

- **卷积层**：卷积层通过卷积操作来提取图像的局部特征。
- **池化层**：池化层通过降维操作来减少数据的维度，同时保留主要的信息。
- **全连接层**：全连接层通过全连接来对特征进行分类或回归。

#### 4.2.2 循环神经网络（RNN）

循环神经网络（Recurrent Neural Network，RNN）是一种用于序列数据的深度学习模型。它通过循环连接来处理时间序列数据，使得模型能够记住过去的输入信息。

- **循环层**：循环层通过循环连接来更新隐藏状态，使得信息能够在时间上传递。
- **门控单元**：门控单元如长短期记忆（Long Short-Term Memory，LSTM）和门控循环单元（Gated Recurrent Unit，GRU），通过控制信息的流动来防止梯度消失和爆炸问题。

#### 4.2.3 生成对抗网络（GAN）

生成对抗网络（Generative Adversarial Network，GAN）是一种通过对抗训练来生成数据的模型。它由生成器和判别器两个部分组成，生成器生成数据，判别器判断数据是否真实。

- **生成器**：生成器通过噪声生成逼真的数据。
- **判别器**：判别器通过区分真实数据和生成数据来训练生成器。

### 4.3 自然语言处理技术

自然语言处理（Natural Language Processing，NLP）是一种让计算机理解和处理自然语言的技术。以下是几种常见的NLP技术：

#### 4.3.1 词向量表示

词向量表示是将单词映射到高维空间中的向量表示。常见的词向量模型包括：

- **Word2Vec**：Word2Vec是一种基于神经网络的词向量模型，通过上下文来学习单词的向量表示。
- **GloVe**：GloVe是一种基于全局语境的词向量模型，通过词频和词的共现关系来学习单词的向量表示。

#### 4.3.2 语言模型

语言模型是用于预测下一个单词的概率分布。常见的语言模型包括：

- **N-gram模型**：N-gram模型是一种基于历史n个单词的统计模型，通过计算单词序列的概率来预测下一个单词。
- **神经网络语言模型**：神经网络语言模型通过神经网络来学习单词序列的概率分布。

#### 4.3.3 序列标注

序列标注是将输入序列中的每个单词标注为不同的标签。常见的序列标注任务包括：

- **命名实体识别（Named Entity Recognition，NER）**：NER是将输入序列中的命名实体（如人名、地名等）标注为相应的类别。
- **情感分析（Sentiment Analysis）**：情感分析是将输入序列的情感标签为正面、负面或中性。

#### 4.3.4 跨语言语义理解

跨语言语义理解是让计算机在不同语言之间进行语义理解。常见的跨语言语义理解技术包括：

- **机器翻译**：机器翻译是将一种语言的文本翻译成另一种语言的文本。
- **语义对齐**：语义对齐是将不同语言的文本映射到共同的语义空间中。

### 4.4 计算机视觉技术

计算机视觉（Computer Vision，CV）是让计算机理解和解释图像和视频的技术。以下是几种常见的计算机视觉技术：

#### 4.4.1 目标检测

目标检测是将图像中的物体检测出来，并定位其位置。常见的目标检测算法包括：

- **R-CNN**：R-CNN是一种基于深度学习的目标检测算法，通过区域提议和分类来检测图像中的目标。
- **SSD**：SSD是一种基于卷积神经网络的快速目标检测算法，通过多尺度特征图来检测图像中的目标。

#### 4.4.2 图像分类

图像分类是将图像分为不同的类别。常见的图像分类算法包括：

- **卷积神经网络（CNN）**：CNN是一种用于图像分类的深度学习模型，通过卷积层、池化层和全连接层来提取图像的特征。
- **迁移学习**：迁移学习是一种利用预训练模型进行图像分类的方法，通过将预训练模型的权重迁移到新任务中，提高分类的准确性。

#### 4.4.3 人脸识别

人脸识别是将图像或视频中的人脸检测并识别出来。常见的人脸识别算法包括：

- **深度学习模型**：深度学习模型如CNN和LSTM等被广泛应用于人脸识别任务。
- **特征点定位**：通过定位人脸的关键点（如眼睛、鼻子、嘴巴等），进行人脸识别。

#### 4.1 监督学习算法

监督学习算法是一种通过已标记的训练数据来训练模型，然后使用模型进行预测的算法。以下是几种常见的监督学习算法：

**线性回归（Linear Regression）**

线性回归是一种用于预测数值型数据的简单算法。它通过找到一个线性函数来拟合数据，使得预测值与实际值之间的误差最小。

**伪代码**：

```
function LinearRegression(x, y):
    # x: 特征向量
    # y: 标签向量
    # 计算x的均值和方差
    mean_x = mean(x)
    variance_x = variance(x)
    
    # 计算x和y的协方差
    covariance_xy = covariance(x, y)
    
    # 计算x的方差
    variance_x = variance(x)
    
    # 计算斜率w1
    w1 = covariance_xy / variance_x
    
    # 计算截距w0
    w0 = mean(y) - w1 * mean(x)
    
    return w0, w1
```

**决策树（Decision Tree）**

决策树是一种树形结构，通过一系列规则来对数据进行分类或回归。每个节点代表一个特征，每个分支代表该特征的取值。

**伪代码**：

```
function DecisionTree(data, labels, features):
    # data: 特征数据
    # labels: 标签数据
    # features: 可选特征列表
    if len(data) <= 1 or not features:
        return LeafNode(labels)
    
    # 计算每个特征的信息增益

#### 4.2 深度学习模型

深度学习模型是一种基于多层神经网络的学习方法，用于处理复杂的数据和任务。以下是几种常见的深度学习模型：

**卷积神经网络（CNN）**

卷积神经网络是一种用于图像处理的深度学习模型。它通过卷积层、池化层和全连接层来提取图像的特征。

**伪代码**：

```
function ConvolutionalNeuralNetwork(x, w):
    # x: 输入数据
    # w: 权重参数
    # 卷积操作
    conv_output = Conv2D(x, w.conv_weights)
    
    # 池化操作
    pool_output = MaxPooling(conv_output, w.pool_size)
    
    # 全连接层
    fc_output = FullyConnected(pool_output, w.fc_weights)
    
    # 激活函数
    output = ReLU(fc_output)
    
    return output
```

**循环神经网络（RNN）**

循环神经网络是一种用于序列数据的深度学习模型。它通过循环连接来处理时间序列数据，使得模型能够记住过去的输入信息。

**伪代码**：

```
function RNN(input, hidden_state, weights):
    # input: 输入数据
    # hidden_state: 隐藏状态
    # weights: 权重参数
    
    # 计算输入和隐藏状态的加权和
    summed_input = input * weights.input_to_hidden
    
    # 加上偏置项
    summed_input += weights.hidden_to_hidden
    
    # 通过激活函数
    hidden_state = activation(summed_input)
    
    return hidden_state
```

**生成对抗网络（GAN）**

生成对抗网络是一种通过对抗训练来生成数据的模型。它由生成器和判别器两个部分组成，生成器生成数据，判别器判断数据是否真实。

**伪代码**：

```
function GAN(z, generator, discriminator):
    # z: 生成器的输入噪声
    # generator: 生成器模型
    # discriminator: 判别器模型
    
    # 生成器生成假样本
    fake_samples = generator(z)
    
    # 判别器判断真假样本
    fake_logits = discriminator(fake_samples)
    real_logits = discriminator(x)
    
    # 计算生成器的损失函数
    g_loss = -torch.mean(torch.log(fake_logits))
    
    # 计算判别器的损失函数
    d_loss = -torch.mean(torch.log(real_logits) + torch.log(1 - fake_logits))
    
    return g_loss, d_loss
```

#### 4.1.1 线性回归

线性回归是一种用于预测数值型数据的模型，它通过找到一条直线来拟合数据点，使得预测值与实际值之间的误差最小。

**数学模型**：

线性回归的数学模型可以表示为：

\[ y = w_0 + w_1 \cdot x \]

其中，\( y \) 是预测值，\( x \) 是输入特征，\( w_0 \) 和 \( w_1 \) 是模型的权重参数。

**公式讲解**：

1. **斜率（\( w_1 \)）**：斜率 \( w_1 \) 表示输入特征 \( x \) 对预测值 \( y \) 的影响程度。斜率越大，输入特征对预测值的影响越强。
2. **截距（\( w_0 \)）**：截距 \( w_0 \) 表示当输入特征 \( x \) 为0时，预测值 \( y \) 的初始值。

**举例说明**：

假设我们有一个数据集，其中包含房价 \( y \) 和房屋面积 \( x \)：

| 房屋面积 \( x \) | 房价 \( y \) |
|----------------|-------------|
| 1000           | 200,000     |
| 1500           | 300,000     |
| 2000           | 400,000     |

我们希望使用线性回归模型来预测一个面积为 1200 平方英尺的房子的价格。首先，我们需要计算斜率和截距：

\[ w_1 = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2} \]
\[ w_0 = \bar{y} - w_1 \cdot \bar{x} \]

其中，\( \bar{x} \) 和 \( \bar{y} \) 分别是输入特征 \( x \) 和预测值 \( y \) 的均值。

通过计算，我们得到斜率 \( w_1 = 100,000 \) 和截距 \( w_0 = -100,000 \)。

因此，预测公式为：

\[ y = 100,000 \cdot x - 100,000 \]

将 \( x = 1200 \) 代入公式，我们得到预测房价为：

\[ y = 100,000 \cdot 1200 - 100,000 = 22,000,000 \]

即预测价格为 220 万美元。

#### 4.1.2 决策树

决策树是一种用于分类和回归的树形结构，通过一系列规则来对数据进行决策。它通常由根节点、内部节点和叶节点组成。

**数学模型**：

决策树的构建过程可以表示为：

1. **选择最佳特征**：计算每个特征的信息增益或基尼不纯度，选择具有最大信息增益或最小基尼不纯度的特征作为节点。
2. **划分数据**：根据最佳特征的不同取值，将数据划分为多个子集。
3. **递归构建子树**：对每个子集递归构建子树，直到满足停止条件（如数据量不足、特征重要性不足等）。

**公式讲解**：

1. **信息增益（Information Gain）**：信息增益是衡量特征对分类效果的指标，计算公式为：

\[ IG(D, A) = H(D) - \sum_{v \in A} \frac{|D_v|}{|D|} H(D_v) \]

其中，\( D \) 是原始数据集，\( A \) 是特征集合，\( D_v \) 是特征 \( v \) 的取值集合。

2. **基尼不纯度（Gini Impurity）**：基尼不纯度是衡量数据集分类纯度的指标，计算公式为：

\[ GI(D) = 1 - \sum_{v \in A} \frac{|D_v|}{|D|}^2 \]

**举例说明**：

假设我们有一个包含三个特征（年龄、收入、家庭状况）的分类问题，数据如下：

| 年龄 | 收入 | 家庭状况 | 类别 |
|-----|-----|----------|------|
| 30  | 高   | 独生     | A    |
| 40  | 中   | 独生     | B    |
| 50  | 高   | 独生     | A    |
| 20  | 低   | 非独生   | B    |
| 35  | 中   | 非独生   | B    |

我们希望使用决策树来对数据进行分类。首先，我们需要计算每个特征的信息增益或基尼不纯度，选择具有最大信息增益或最小基尼不纯度的特征作为节点。

对于年龄特征：

\[ IG(\text{年龄}, D) = H(D) - \frac{2}{5} H(D_1) - \frac{3}{5} H(D_2) \]

其中，\( D_1 \) 表示年龄在30-40岁之间的数据子集，\( D_2 \) 表示年龄在20-30岁之间的数据子集。

计算得到：

\[ IG(\text{年龄}, D) = 0.735 - \frac{2}{5} (0.9) - \frac{3}{5} (0.85) = 0.355 \]

对于收入特征：

\[ GI(\text{收入}, D) = 1 - \frac{2}{5} \cdot (0.6^2) - \frac{3}{5} \cdot (0.5^2) \]

计算得到：

\[ GI(\text{收入}, D) = 0.68 \]

由于年龄特征的信息增益最大，我们选择年龄特征作为决策树的根节点，并根据年龄特征的不同取值将数据划分为两个子集：

- 年龄在30-40岁之间的数据子集（\( D_1 \)）：包含两个类别A和B，比例分别为1:1。
- 年龄在20-30岁之间的数据子集（\( D_2 \)）：包含两个类别A和B，比例分别为1:1。

对于收入特征，我们选择收入特征作为节点，并根据收入特征的不同取值将数据划分为两个子集：

- 收入为高的数据子集（\( D_{11} \)）：包含类别A，比例为1。
- 收入为中或低的数据子集（\( D_{12} \)）：包含类别B，比例为1。

递归构建子树，直到满足停止条件，我们得到一个决策树：

```
根节点：年龄
        /     \
       /       \
     30-40    20-30
    /   \     /   \
   高    中    低   中
   A     B     B    B
```

通过这个决策树，我们可以对新的数据进行分类。例如，对于年龄为30岁，收入为中，家庭状况为非独生的数据，我们可以根据决策树预测其类别为B。

#### 4.2.1 卷积神经网络（CNN）

卷积神经网络是一种用于图像识别和处理的深度学习模型，通过卷积层、池化层和全连接层来提取图像的特征。

**数学模型**：

1. **卷积层**：卷积层通过卷积操作来提取图像的局部特征，公式为：

\[ \text{conv}(x, w) = \sum_{i=1}^{k} w_i \cdot x_i \]

其中，\( x \) 是输入图像，\( w \) 是卷积核（滤波器），\( k \) 是卷积核的数量。

2. **池化层**：池化层通过降维操作来减少数据的维度，同时保留主要的信息。常用的池化操作包括最大池化和平均池化。

3. **全连接层**：全连接层通过全连接来对特征进行分类或回归。

**公式讲解**：

1. **卷积操作**：卷积操作可以通过卷积核在输入图像上滑动，提取局部特征。卷积核的数量决定了特征的多样性。

2. **激活函数**：激活函数（如ReLU函数）用于增加网络的非线性特性，使得网络能够学习更复杂的特征。

3. **反向传播**：通过反向传播算法，计算输出误差，并更新网络的权重参数。

**举例说明**：

假设我们有一个3x3的卷积核 \( w \) 和一个3x3的输入图像 \( x \)：

\[ w = \begin{bmatrix}
0 & 1 & 0 \\
1 & 1 & 1 \\
0 & 1 & 0
\end{bmatrix} \]
\[ x = \begin{bmatrix}
0 & 1 & 0 \\
1 & 1 & 1 \\
0 & 1 & 0
\end{bmatrix} \]

通过卷积操作，我们得到：

\[ \text{conv}(x, w) = \begin{bmatrix}
0 & 2 & 0 \\
2 & 5 & 2 \\
0 & 2 & 0
\end{bmatrix} \]

通过ReLU函数，我们得到激活后的特征图：

\[ \text{ReLU}(\text{conv}(x, w)) = \begin{bmatrix}
0 & 2 & 0 \\
2 & 5 & 2 \\
0 & 2 & 0
\end{bmatrix} \]

通过这个过程，我们可以提取输入图像的局部特征，用于后续的图像分类或处理。

#### 4.2.2 循环神经网络（RNN）

循环神经网络是一种用于处理序列数据的深度学习模型，通过循环连接来处理时间序列数据，使得模型能够记住过去的输入信息。

**数学模型**：

1. **隐藏状态**：RNN的隐藏状态 \( h_t \) 可以表示为：

\[ h_t = \text{ReLU}(W_h \cdot [h_{t-1}, x_t] + b_h) \]

其中，\( W_h \) 是隐藏状态权重，\( b_h \) 是隐藏状态偏置。

2. **输出**：RNN的输出 \( y_t \) 可以表示为：

\[ y_t = \text{softmax}(W_o \cdot h_t + b_o) \]

其中，\( W_o \) 是输出权重，\( b_o \) 是输出偏置。

**公式讲解**：

1. **隐藏状态更新**：通过循环连接，RNN的隐藏状态 \( h_t \) 可以记住过去的输入信息，使得模型能够处理长序列数据。

2. **输出**：RNN的输出通常用于分类或回归，通过softmax函数将隐藏状态转换为概率分布。

**举例说明**：

假设我们有一个时间序列数据 \( x_t \)，其中 \( x_t = [1, 2, 3, 4, 5] \)。我们使用一个简单的RNN模型进行分类，隐藏状态权重 \( W_h = [1, 2] \)，输出权重 \( W_o = [1, 2, 3] \)，隐藏状态偏置 \( b_h = 1 \)，输出偏置 \( b_o = 1 \)。

首先，我们初始化隐藏状态 \( h_0 = [0, 0] \)。

然后，我们逐个处理时间序列数据：

- 对于 \( x_1 = 1 \)：

\[ h_1 = \text{ReLU}([1, 2] \cdot [0, 1] + 1) = \text{ReLU}(1 + 1) = \text{ReLU}(2) = 2 \]

\[ y_1 = \text{softmax}([1, 2] \cdot [2, 0] + 1) = \text{softmax}(3) = [0.26, 0.74] \]

- 对于 \( x_2 = 2 \)：

\[ h_2 = \text{ReLU}([1, 2] \cdot [2, 2] + 1) = \text{ReLU}(5) = 5 \]

\[ y_2 = \text{softmax}([1, 2] \cdot [5, 2] + 1) = \text{softmax}(12) = [0.18, 0.82] \]

- 对于 \( x_3 = 3 \)：

\[ h_3 = \text{ReLU}([1, 2] \cdot [5, 3] + 1) = \text{ReLU}(12) = 12 \]

\[ y_3 = \text{softmax}([1, 2] \cdot [12, 3] + 1) = \text{softmax}(27) = [0.11, 0.89] \]

- 对于 \( x_4 = 4 \)：

\[ h_4 = \text{ReLU}([1, 2] \cdot [12, 4] + 1) = \text{ReLU}(29) = 29 \]

\[ y_4 = \text{softmax}([1, 2] \cdot [29, 4] + 1) = \text{softmax}(63) = [0.05, 0.95] \]

- 对于 \( x_5 = 5 \)：

\[ h_5 = \text{ReLU}([1, 2] \cdot [29, 5] + 1) = \text{ReLU}(64) = 64 \]

\[ y_5 = \text{softmax}([1, 2] \cdot [64, 5] + 1) = \text{softmax}(133) = [0.01, 0.99] \]

通过这个过程，我们可以得到每个时间点的隐藏状态和输出，用于分类或回归。

#### 4.2.3 生成对抗网络（GAN）

生成对抗网络是一种通过对抗训练来生成数据的模型，由生成器和判别器两个部分组成，生成器生成数据，判别器判断数据是否真实。

**数学模型**：

1. **生成器（Generator）**：生成器的目标是生成尽可能真实的数据。

\[ G(z) = \text{Generator}(z) \]

其中，\( z \) 是生成器的输入噪声。

2. **判别器（Discriminator）**：判别器的目标是区分真实数据和生成数据。

\[ D(x) = \text{Discriminator}(x) \]
\[ D(G(z)) = \text{Discriminator}(\text{Generator}(z)) \]

**公式讲解**：

1. **生成器损失函数**：生成器的损失函数通常为对抗损失，公式为：

\[ L_G = -\log(D(G(z))) \]

2. **判别器损失函数**：判别器的损失函数通常为二元交叉熵损失，公式为：

\[ L_D = -[\log(D(x)) + \log(1 - D(G(z)))] \]

**举例说明**：

假设我们有一个生成对抗网络，其中生成器的输入噪声维度为2，生成器生成的数据维度为3。

首先，我们初始化生成器的权重参数 \( W_G \) 和判别器的权重参数 \( W_D \)。

然后，我们进行以下步骤：

1. **生成器生成数据**：生成器使用输入噪声 \( z \) 生成数据 \( G(z) \)。
2. **判别器判断数据**：判别器对真实数据 \( x \) 和生成数据 \( G(z) \) 进行判断，计算判别器的损失函数 \( L_D \)。
3. **更新生成器**：通过梯度下降算法，更新生成器的权重参数 \( W_G \)，以减少生成数据的判别器损失。
4. **更新判别器**：通过梯度下降算法，更新判别器的权重参数 \( W_D \)，以区分真实数据和生成数据。

通过这个过程，生成器和判别器相互对抗，生成器不断优化生成数据，判别器不断优化区分数据，最终达到平衡状态。

import torch
import torch.nn as nn
import torchvision
import torchvision.transforms as transforms
from torch.utils.data import DataLoader
from torchvision.datasets import ImageFolder

# 定义CNN模型
class CNNModel(nn.Module):
    def __init__(self):
        super(CNNModel, self).__init__()
        # 卷积层
        self.conv1 = nn.Conv2d(3, 32, 3, padding=1)
        self.conv2 = nn.Conv2d(32, 64, 3, padding=1)

        self.pool = nn.MaxPool2d(2, 2)
        # 全连接层
        self.fc1 = nn.Linear(64 * 6 * 6, 128)

        self.relu = nn.ReLU()

    def forward(self, x):
        x = self.pool(self.relu(self.conv1(x)))
        x = self.pool(self.relu(self.conv2(x)))
        x = x.view(-1, 64 * 6 * 6)
        x = self.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# 实例化模型
model = CNNModel()

# 定义损失函数和优化器
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)

# 加载数据集
train_dataset = ImageFolder(root='./data/train', transform=transforms.ToTensor())
train_loader = DataLoader(dataset=train_dataset, batch_size=32, shuffle=True)

# 训练模型
for epoch in range(10):  # 数量为10个epoch
    running_loss = 0.0
    for i, data in enumerate(train_loader, 0):
        inputs, labels = data
        optimizer.zero_grad()
        outputs = model(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()
        running_loss += loss.item()
    print(f'Epoch {epoch + 1}, Loss: {running_loss / len(train_loader)}')

# 测试模型
test_dataset = ImageFolder(root='./data/test', transform=transforms.ToTensor())
test_loader = DataLoader(dataset=test_dataset, batch_size=32, shuffle=False)
with torch.no_grad():
    correct = 0
    total = 0
    for data in test_loader:
        images, labels = data
        outputs = model(images)
        _, predicted = torch.max(outputs.data, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()
print(f'Accuracy: {100 * correct / total}%')
```

```python
class CNNModel(nn.Module):
    def __init__(self):
        super(CNNModel, self).__init__()
        # 卷积层
        self.conv1 = nn.Conv2d(3, 32, 3, padding=1)
        self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
        # 池化层
        self.pool = nn.MaxPool2d(2, 2)
        # 全连接层
        self.fc1 = nn.Linear(64 * 6 * 6, 128)
        self.fc2 = nn.Linear(128, 2)
        # 激活函数
        self.relu = nn.ReLU()

    def forward(self, x):
        x = self.pool(self.relu(self.conv1(x)))
        x = self.pool(self.relu(self.conv2(x)))
        x = x.view(-1, 64 * 6 * 6)
        x = self.relu(self.fc1(x))
        x = self.fc2(x)
        return x
```

**解读**：

- **卷积层**：我们定义了两个卷积层 `self.conv1` 和 `self.conv2`。第一个卷积层将输入的3通道图片（RGB）映射到32个通道的特征图，卷积核大小为3x3。`padding=1` 用于保持输出特征图的大小不变。
- **池化层**：使用最大池化层 `nn.MaxPool2d`，池化窗口大小为2x2，步长也为2。
- **全连接层**：第一个全连接层 `self.fc1` 将64个特征图中的每个6x6像素区域映射到128个神经元，第二个全连接层 `self.fc2` 将128个神经元映射到2个神经元，用于分类。
- **ReLU激活函数**：我们在每个卷积层后添加ReLU激活函数，增加模型的非线性能力。
