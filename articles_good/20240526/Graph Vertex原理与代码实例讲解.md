## 2. 核心概念与联系

在图中，Vertex（顶点）是图中的基本元素，它表示图中的节点或点。Vertex 之间通过边（Edge）相互连接，形成一定的关系或结构。根据连接方式的不同，边可以分为有向边和无向边。

图的顶点和边之间的关系可以用邻接表（Adjacency List）或邻接矩阵（Adjacency Matrix）来表示。邻接表是一种线性数据结构，用于存储图中的顶点及其相邻顶点之间的关系。邻接矩阵是一种二维数据结构，用于存储图中的顶点及其相邻顶点之间的关系。

## 3. 核心算法原理具体操作步骤

在图中，顶点之间的关系可以通过DFS（深度优先搜索）和 BFS（广度优先搜索）算法进行遍历。下面我们分别介绍这两种算法的原理和操作步骤。

### 3.1 DFS（深度优先搜索）算法

DFS 算法是一种用于遍历图或树结构的算法，它从图中的一个顶点开始，沿着边向深度遍历顶点。 DFS 算法的主要操作步骤如下：

1. 从图中的一个顶点开始，标记为已访问。
2. 从已访问的顶点出发，沿着边向深度遍历相邻顶点，并将其标记为已访问。
3. 重复步骤 2，直到所有顶点都被访问完毕。

DFS 算法可以使用递归或栈数据结构实现。

### 3.2 BFS（广度优先搜索）算法

BFS 算法是一种用于遍历图或树结构的算法，它从图中的一个顶点开始，沿着边向广度遍历顶点。 BFS 算法的主要操作步骤如下：

1. 从图中的一个顶点开始，标记为已访问。
2. 将已访问的顶点放入队列中。
3. 从队列中取出一个顶点，沿着边向广度遍历相邻顶点，并将其标记为已访问。
4. 将新的已访问顶点放入队列中。
5. 重复步骤 3 和 4，直到所有顶点都被访问完毕。

BFS 算法可以使用队列数据结构实现。

### 4.1 图的度数序列

图的度数序列是一个用于描述图中顶点度数（即顶点连接边的数量）的序列。我们可以使用以下公式计算图的度数序列：

$$
D(v) = \sum_{u \in V} E(v, u)
$$

其中 $D(v)$ 表示顶点 $v$ 的度数，$V$ 表示图中的所有顶点，$E(v, u)$ 表示顶点 $v$ 和顶点 $u$ 之间的边数。

## 4. 项目实践：代码实例和详细解释说明

在本节中，我们将通过一个简单的项目实践，展示如何使用Python语言实现图的Vertex原理和DFS、BFS算法。我们将创建一个简单的图，表示一个社交网络，其中每个节点表示一个人，每条边表示两人之间的关注关系。

### 4.1 创建图的邻接表

首先，我们需要创建一个图的邻接表，表示每个顶点及其相邻顶点之间的关注关系。

```python
# 创建图的邻接表
graph = {
    'Alice': ['Bob', 'Charlie', 'David'],
    'Bob': ['Alice', 'Charlie', 'Eve'],
    'Charlie': ['Alice', 'Bob', 'Eve', 'Fred'],
    'David': ['Alice'],
    'Eve': ['Bob', 'Charlie', 'Fred'],
    'Fred': ['Charlie', 'Eve']
}
```

### 4.2 实现DFS算法

接下来，我们将实现DFS算法，用于遍历图中的顶点。

```python
# 实现DFS算法
def dfs(graph, start, visited=None):
    if visited is None:
        visited = set()
    visited.add(start)
    for neighbor in graph[start]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)
    return visited
```

### 4.3 实现BFS算法

最后，我们将实现BFS算法，用于遍历图中的顶点。

```python
# 实现BFS算法
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    visited.add(start)
    while queue:
        current = queue.popleft()
        for neighbor in graph[current]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    return visited
```
