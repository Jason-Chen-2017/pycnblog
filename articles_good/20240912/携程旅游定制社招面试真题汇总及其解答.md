### 算法编程题库及解析

#### 3. 编写一个算法，计算两个正整数的最大公约数。

**题目：** 编写一个函数 `gcd(a int, b int) int`，计算两个正整数 `a` 和 `b` 的最大公约数。

**答案：**

```go
func gcd(a, b int) int {
    for b != 0 {
        a, b = b, a%b
    }
    return a
}
```

**解析：** 该算法基于辗转相除法，通过不断用较小数去除较大数，直到余数为0，此时较大数即为最大公约数。

#### 4. 实现一个函数，检查一个整数是否为素数。

**题目：** 编写一个函数 `isPrime(n int) bool`，检查整数 `n` 是否为素数。

**答案：**

```go
func isPrime(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i*i <= n; i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}
```

**解析：** 该算法首先排除 `n` 小于等于1的情况，然后从2开始，逐个检查 `n` 是否能被小于等于其平方根的整数整除。如果不能，则 `n` 为素数。

#### 5. 编写一个算法，对整数数组进行快速排序。

**题目：** 实现快速排序算法，对一个整数数组进行升序排序。

**答案：**

```go
func quickSort(arr []int) {
    if len(arr) <= 1 {
        return
    }
    pivot := arr[len(arr)/2]
    left, right := 0, len(arr)-1
    for {
        for arr[left] < pivot {
            left++
        }
        for arr[right] > pivot {
            right--
        }
        if left >= right {
            break
        }
        arr[left], arr[right] = arr[right], arr[left]
        left++
        right--
    }
    quickSort(arr[:left])
    quickSort(arr[left:])
}
```

**解析：** 快速排序算法的核心思想是通过选择一个基准元素，将数组划分为两部分，一部分元素均小于基准元素，另一部分元素均大于基准元素。然后递归地对这两部分进行快速排序。
