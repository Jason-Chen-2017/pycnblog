#### 三、算法编程题库

**1. 编写一个算法，实现课程内容的分类。**

```python
def classify_courses(courses):
    # your code here
    pass
```

**答案：**

```python
def classify_courses(courses):
    categories = {}
    for course in courses:
        category = course.get('category', '未分类')
        if category not in categories:
            categories[category] = []
        categories[category].append(course)
    return categories
```

**2. 编写一个算法，实现课程推荐。**

```python
    # your code here
    pass
```

**答案：**

```python
def recommend_courses(user_profile, courses):
    recommendations = []
    for course in courses:
        if course['category'] in user_profile['interests']:
            recommendations.append(course)
    return recommendations
```
