## 第二部分：算法编程题库

### 1. 背包问题

**题目：** 字节跳动面试题：给定一个背包容量和一组物品，求出能够装入背包的最大价值。

**答案解析：**

1. **动态规划：** 定义一个二维数组 `dp[i][w]`，表示在前 `i` 个物品中选择，能够装入容量为 `w` 的背包的最大价值。
2. **状态转移方程：** 根据物品的重量和价值，构建状态转移方程。

**示例代码：**

```go
func knapsack(W, N int, weights []int, values []int) int {
    dp := make([][]int, N+1)
    for i := range dp {
        dp[i] = make([]int, W+1)
    }

    for i := 1; i <= N; i++ {
        for w := 1; w <= W; w++ {
            if weights[i-1] <= w {
                dp[i][w] = max(dp[i-1][w], dp[i-1][w-weights[i-1]]+values[i-1])
            } else {
                dp[i][w] = dp[i-1][w]
            }
        }
    }

    return dp[N][W]
}
```

### 2. 最短路径问题

**题目：** 京东面试题：给定一个无向图，求图中两点之间的最短路径。

**答案解析：**

1. **迪杰斯特拉算法：** 通过递增的方式逐步确定当前到达每个顶点的最短路径。
2. **状态转移方程：** 根据当前已确定的最短路径，更新其他顶点的最短路径。

**示例代码：**

```go
func dijkstra(graph [][]int, start int) []int {
    n := len(graph)
    dist := make([]int, n)
    for i := range dist {
        dist[i] = math.MaxInt32
    }
    dist[start] = 0
    visited := make([]bool, n)

    for i := 0; i < n; i++ {
        u := -1
        for _, v := range dist {
            if !visited[v] && (u == -1 || v < dist[u]) {
                u = v
            }
        }
        visited[u] = true

        for v, w := range graph[u] {
            if !visited[v] && dist[v] > dist[u]+w {
                dist[v] = dist[u] + w
            }
        }
    }

    return dist
}
```

### 3. 股票买卖问题

**题目：** 美团面试题：给定一个数组，表示每天的股票价格，求出最大利润。

**答案解析：**

1. **动态规划：** 定义一个一维数组 `dp[i]`，表示从第 `i` 天开始到第 `n` 天的最大利润。
2. **状态转移方程：** 根据前一天的最大利润，计算当前的最大利润。

**示例代码：**

```go
func maxProfit(prices []int) int {
    if len(prices) < 2 {
        return 0
    }
    dp := make([]int, len(prices))
    dp[0] = 0
    dp[1] = max(0, prices[1]-prices[0])

    for i := 2; i < len(prices); i++ {
        dp[i] = max(dp[i-1], dp[i-2]+prices[i]-prices[i-1])
    }

    return dp[len(prices)-1]
}
```
