##### 1. 腾讯 - 给定一个字符串，找出最长的无重复子串的长度。

**题目描述：** 给定一个字符串，找出最长的无重复子串的长度。

**答案：**

- **思路：** 使用滑动窗口的方法，维护一个当前无重复子串的起始位置和长度。
- **代码实现：**

```python
def length_of_longest_substring(s: str) -> int:
    n = len(s)
    start = 0
    max_len = 0
    used = {}

    for i in range(n):
        if s[i] in used and start <= used[s[i]]:
            start = used[s[i]] + 1
        else:
            max_len = max(max_len, i - start + 1)

        used[s[i]] = i

    return max_len
```

**解析：** 该算法的时间复杂度为 O(n)，其中 n 为字符串的长度。

##### 2. 阿里巴巴 - 给定一个整数数组，找到两个数使得它们的和等于目标值。

**题目描述：** 给定一个整数数组，找到两个数使得它们的和等于目标值。

**答案：**

- **思路：** 使用哈希表存储已遍历过的元素，通过查找哈希表来判断当前元素是否存在与之相加等于目标值的另一个元素。
- **代码实现：**

```python
def two_sum(nums: List[int], target: int) -> List[int]:
    n = len(nums)
    used = {}

    for i in range(n):
        complement = target - nums[i]
        if complement in used:
            return [used[complement], i]
        used[nums[i]] = i

    return []
```

**解析：** 该算法的时间复杂度为 O(n)，其中 n 为数组的长度。

##### 3. 百度 - 给定一个二叉树，找出其最大深度。

**题目描述：** 给定一个二叉树，找出其最大深度。

**答案：**

- **思路：** 使用递归的方法，计算左右子树的最大深度，取其中较大值再加 1。
- **代码实现：**

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def max_depth(root: TreeNode) -> int:
    if root is None:
        return 0
    return 1 + max(max_depth(root.left), max_depth(root.right))
```

**解析：** 该算法的时间复杂度为 O(n)，其中 n 为二叉树的节点数。

##### 5. 拼多多 - 给定一个字符串，找出其中第一个不重复的字符。

**题目描述：** 给定一个字符串，找出其中第一个不重复的字符。

**答案：**

- **思路：** 使用哈希表记录每个字符的出现次数，然后遍历字符串，找出第一个出现次数为 1 的字符。
- **代码实现：**

```python
def first_uniq_char(s: str) -> str:
    n = len(s)
    used = [0] * 26

    for c in s:
        used[ord(c) - ord('a')] += 1

    for c in s:
        if used[ord(c) - ord('a')] == 1:
            return c

    return ''
```

**解析：** 该算法的时间复杂度为 O(n)，其中 n 为字符串的长度。

##### 6. 京东 - 给定一个整数，找出其对应的格雷码。

**题目描述：** 给定一个整数，找出其对应的格雷码。

**答案：**

- **思路：** 使用异或运算，将当前数字和上一个数字进行异或操作，得到下一个格雷码。
- **代码实现：**

```python
def gray_code(n: int) -> List[int]:
    if n == 0:
        return [0]

    res = [0]
    for i in range(1, n + 1):
        res += [x ^ (1 << i - 1) for x in res[::-1]]

    return res
```

**解析：** 该算法的时间复杂度为 O(n)，其中 n 为整数的大小。

##### 7. 美团 - 给定一个数组，找出其中第二小的元素。

**题目描述：** 给定一个数组，找出其中第二小的元素。

**答案：**

- **思路：** 使用两个变量分别记录最小的元素和第二小的元素，遍历数组，更新这两个变量。
- **代码实现：**

```python
def find_second_smallest(nums: List[int]) -> int:
    if len(nums) < 2:
        return -1

    smallest = second_smallest = float('inf')
    for num in nums:
        if num < smallest:
            second_smallest = smallest
            smallest = num
        elif smallest < num < second_smallest:
            second_smallest = num

    return second_smallest if second_smallest != float('inf') else -1
```

**解析：** 该算法的时间复杂度为 O(n)，其中 n 为数组的长度。

##### 8. 快手 - 给定一个整数，找出其对应的二进制表示中 1 的个数。

**题目描述：** 给定一个整数，找出其对应的二进制表示中 1 的个数。

**答案：**

- **思路：** 使用位操作，将当前数字和上一个数字进行与运算，统计 1 的个数。
- **代码实现：**

```python
def count_bits(num: int) -> int:
    count = 0
    while num:
        count += num & 1
        num >>= 1

    return count
```

**解析：** 该算法的时间复杂度为 O(log n)，其中 n 为整数的大小。

##### 9. 滴滴 - 给定一个整数数组，找出其中第 k 个最大的元素。

**题目描述：** 给定一个整数数组，找出其中第 k 个最大的元素。

**答案：**

- **思路：** 使用快速选择算法，在数组中找到第 k 大的元素。
- **代码实现：**

```python
def find_kth_largest(nums: List[int], k: int) -> int:
    def quick_select(nums, left, right, k):
        pivot = nums[right]
        i = left
        for j in range(left, right):
            if nums[j] > pivot:
                nums[i], nums[j] = nums[j], nums[i]
                i += 1
        nums[i], nums[right] = nums[right], nums[i]
        if i == k:
            return nums[i]
        elif i < k:
            return quick_select(nums, i + 1, right, k)
        else:
            return quick_select(nums, left, i - 1, k)

    return quick_select(nums, 0, len(nums) - 1, k)
```

**解析：** 该算法的时间复杂度为 O(n)，其中 n 为数组的长度。

##### 10. 小红书 - 给定一个字符串，找出其中第一个重复出现的字符。

**题目描述：** 给定一个字符串，找出其中第一个重复出现的字符。

**答案：**

- **思路：** 使用哈希表记录每个字符的出现次数，遍历字符串，找出第一个出现次数大于 1 的字符。
- **代码实现：**

```python
def first_recurring_char(s: str) -> str:
    n = len(s)
    used = {}
    for i in range(n):
        if s[i] in used and used[s[i]] > 0:
            return s[i]
        used[s[i]] = 1

    return ''
```

**解析：** 该算法的时间复杂度为 O(n)，其中 n 为字符串的长度。
