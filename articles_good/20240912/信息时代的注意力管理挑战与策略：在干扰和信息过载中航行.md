##### 1. 请实现一个函数，用于计算给定文本中的有效单词数。

**题目描述：**
编写一个函数`count_valid_words(text)`，该函数接受一个字符串`text`作为输入，并返回文本中的有效单词数。有效单词是指由字母组成的连续字符序列。

**示例：**
```
text = "Hello, World!"
count = count_valid_words(text)  # 输出应为 2，即 "Hello" 和 "World"
```

**答案：**
```python
def count_valid_words(text):
    words = text.split()
    valid_words = 0
    for word in words:
        if word.isalpha():
            valid_words += 1
    return valid_words

# 测试
text = "Hello, World!"
print(count_valid_words(text))  # 输出应为 2
```


```python
def intersection(list1, list2):
    return list(set(list1) & set(list2))

# 测试
list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]
intersection = intersection(list1, list2)
print(intersection)  # 输出应为 [3, 4]
```

```python
    words = s.split()
    return dict(Counter(words))

print(word_frequency)  # 输出应为 {'Hello': 1, 'World': 1}
```
