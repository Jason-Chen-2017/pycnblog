    return np.convolve(data, np.ones(window), 'valid') / window

#### 13. 最小化供应链成本

**题目：** 设计一个算法，最小化整个供应链的运营成本。

**答案：** 使用线性规划。

**解析：**

```python
from scipy.optimize import linprog

# 定义目标函数和约束条件
c = [-1, -1]  # 成本系数
A = [[1, 1], [1, 0], [0, 1]]  # 约束条件矩阵
b = [100, 200, 150]  # 约束条件向量

x0 = [0, 0]  # 初始解
bounds = [(0, None), (0, None)]  # 变量的上下界

result = linprog(c, A_ub=A, b_ub=b, x0=x0, bounds=bounds, method='highs')

if result.success:
    print("最优解：", result.x)
else:
    print("没有找到最优解")
```
