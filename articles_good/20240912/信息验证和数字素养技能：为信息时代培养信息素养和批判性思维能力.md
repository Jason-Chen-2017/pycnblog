#### 1. 如何在 Python 中实现单例模式？

**题目：** 请使用 Python 实现一个单例模式。

**答案：**

```python
class Singleton:
    _instance = None
    
    def __new__(cls, *args, **kwargs):
        if not cls._instance:
            cls._instance = super(Singleton, cls).__new__(cls, *args, **kwargs)
        return cls._instance

# 使用示例
singleton1 = Singleton()
singleton2 = Singleton()
print(singleton1 is singleton2)  # 输出 True
```

**解析：** 在这个示例中，通过重写 `__new__` 方法，实现了单例模式。当创建第一个实例时，`_instance` 被赋值为该实例；后续创建实例时，直接返回 `_instance`。

#### 2. 如何在 JavaScript 中实现一个防抖（Debounce）函数？

**题目：** 请使用 JavaScript 实现一个防抖函数。

**答案：**

```javascript
function debounce(func, wait) {
    let timeout;
    return function ... {
        clearTimeout(timeout);
        timeout = setTimeout(func, wait);
    };
}

// 使用示例
const debouncedFunction = debounce(() => {
    console.log("Debounced function called");
}, 1000);
setInterval(debouncedFunction, 500);
```

**解析：** 在这个示例中，防抖函数通过清除上一次的定时器并在设定时间后执行目标函数来实现。当在设定时间内再次触发函数时，上一次的定时器会被清除，从而实现防抖效果。

#### 3. 如何在 Java 中实现二分查找算法？

**题目：** 请使用 Java 实现一个二分查找算法。

**答案：**

```java
public class BinarySearch {
    public static int binarySearch(int[] arr, int target) {
        int low = 0;
        int high = arr.length - 1;
        while (low <= high) {
            int mid = low + (high - low) / 2;
            if (arr[mid] == target) {
                return mid;
            } else if (arr[mid] < target) {
                low = mid + 1;
            } else {
                high = mid - 1;
            }
        }
        return -1;
    }

    public static void main(String[] args) {
        int[] arr = {1, 3, 5, 7, 9, 11};
        int target = 7;
        int result = binarySearch(arr, target);
        System.out.println("Element found at index: " + result);
    }
}
```

**解析：** 在这个示例中，二分查找算法通过不断缩小区间来查找目标元素。当目标元素在数组中时，返回其索引；否则，返回 -1。

#### 4. 寻找两个正序数组的中位数

**题目：** 给定两个按升序排列的数组 nums1 和 nums2，请找出并返回这两个正序数组的中位数。

**答案：**

```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        nums = nums1 + nums2
        nums.sort()
        n = len(nums)
        if n % 2 == 1:
            return float(nums[n // 2])
        else:
            return (nums[n // 2 - 1] + nums[n // 2]) / 2.0
```

**解析：** 这个示例中的解决方案将两个数组合并为一个数组，然后对合并后的数组进行排序。最后，根据数组的长度判断中位数的位置，并返回中位数。

#### 5. 最长公共子序列

**题目：** 给定两个字符串 text1 和 text2，返回这两个字符串的最长公共子序列的长度。

**答案：**

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i - 1] == text2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1] + 1
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

        return dp[m][n]
```

**解析：** 这个示例中的解决方案使用动态规划来求解最长公共子序列。通过填写一个二维数组 `dp`，记录每个位置的最长公共子序列长度。
