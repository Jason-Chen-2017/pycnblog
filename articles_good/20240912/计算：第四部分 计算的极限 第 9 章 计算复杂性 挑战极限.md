#### 1. KMP 算法

**题目：** 实现字符串匹配算法 KMP，并解释其时间复杂度。

**答案：** KMP（Knuth-Morris-Pratt）算法是一种高效字符串匹配算法，其核心思想是利用已经匹配的部分来避免重复比较。以下是 KMP 算法的实现和解析。

**代码：**

```python
def kmp(s, p):
    def compute_lps(p):
        lps = [0] * len(p)
        length = 0
        i = 1
        while i < len(p):
            if p[i] == p[length]:
                length += 1
                lps[i] = length
                i += 1
            else:
                if length != 0:
                    length = lps[length - 1]
                else:
                    lps[i] = 0
                    i += 1
        return lps

    lps = compute_lps(p)
    i = j = 0
    while i < len(s):
        if p[j] == s[i]:
            i += 1
            j += 1
        if j == len(p):
            return i - j
        elif i < len(s) and p[j] != s[i]:
            if j != 0:
                j = lps[j - 1]
            else:
                i += 1
    return -1
```

**解析：** KMP 算法的时间复杂度为 O(n+m)，其中 n 是字符串 s 的长度，m 是模式串 p 的长度。这个时间复杂度是因为计算 LPS（最长公共前后缀）数组的时间复杂度为 O(m)，字符串匹配过程的时间复杂度为 O(n)。

#### 2. 二分查找

**题目：** 实现一个二分查找算法，并解释其时间复杂度。

**答案：** 二分查找是一种高效的查找算法，其基本思想是逐步缩小查找范围，直到找到目标元素或确定目标元素不存在。

**代码：**

```python
def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

**解析：** 二分查找的时间复杂度为 O(log n)，其中 n 是数组的长度。这个时间复杂度是因为每次查找都能将查找范围缩小一半。

#### 3. 暴力解法与优化

**题目：** 给定一个整数数组，编写一个函数来找出数组中所有奇数的总和。

**答案：** 以下是一个简单的暴力解法和优化解法。

**代码：**

```python
# 暴力解法
def sum_of_odd(nums):
    return sum(x for x in nums if x % 2 != 0)

# 优化解法
def sum_of_odd_optimized(nums):
    return sum(nums[i] for i in range(len(nums)) if nums[i] % 2 != 0)
```

**解析：** 暴力解法的时间复杂度为 O(n)，其中 n 是数组的长度。优化解法通过使用列表推导式减少了重复的元素访问，时间复杂度仍为 O(n)，但减少了常数因子。

#### 5. 动态规划

**题目：** 给定一个整数数组，找出所有连续子数组的和的最大值。

**答案：** 动态规划是一种优化递归的方法，用于解决最优子结构问题。

**代码：**

```python
def max_subarray_sum(nums):
    max_ending_here = max_so_far = nums[0]
    for x in nums[1:]:
        max_ending_here = max(x, max_ending_here + x)
        max_so_far = max(max_so_far, max_ending_here)
    return max_so_far
```

**解析：** 动态规划的时间复杂度为 O(n)，其中 n 是数组的长度。

#### 8. 算法竞赛

**题目：** 给定一个数组和两个整数 k 和 sum，找出数组中所有连续子数组的和等于 sum 的子数组数量。

**答案：** 使用滑动窗口方法。

**代码：**

```python
def count_subarrays(nums, k, target):
    left = 0
    count = 0
    current_sum = 0
    for right in range(len(nums)):
        current_sum += nums[right]
        while current_sum > target or (left < right and left > 0 and current_sum - nums[left] == target):
            current_sum -= nums[left]
            left += 1
        if current_sum == target:
            count += right - left + 1
    return count
```

**解析：** 时间复杂度为 O(n)，其中 n 是数组的长度。
