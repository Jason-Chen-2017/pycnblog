#### 面试题库

##### 1. 什么是单例模式？请举例说明。

**答案：** 单例模式是一种设计模式，确保一个类仅有一个实例，并提供一个访问它的全局访问点。

**举例：**

```java
public class Singleton {
    private static Singleton instance;

    private Singleton() {}

    public static Singleton getInstance() {
        if (instance == null) {
            instance = new Singleton();
        }
        return instance;
    }
}
```

**解析：** 在这个例子中，`Singleton` 类的构造函数是私有的，确保无法通过外部直接创建实例。`getInstance()` 方法提供了获取实例的入口，通过双重检查锁确保实例的唯一性。

##### 2. 什么是链表？请实现一个单链表的基本操作。

**答案：** 链表是一种线性数据结构，由一系列节点组成，每个节点包含数据和指向下一个节点的指针。

**实现：**

```java
class Node {
    int data;
    Node next;
}

public class LinkedList {
    Node head;

    public void append(int data) {
        Node newNode = new Node();
        newNode.data = data;

        if (head == null) {
            head = newNode;
        } else {
            Node current = head;
            while (current.next != null) {
                current = current.next;
            }
            current.next = newNode;
        }
    }

    public void display() {
        Node current = head;
        while (current != null) {
            System.out.print(current.data + " ");
            current = current.next;
        }
    }
}
```

**解析：** 在这个例子中，`LinkedList` 类实现了一个单链表，提供了 `append()` 和 `display()` 方法，分别用于添加节点和显示链表内容。

##### 3. 请实现快速排序算法。

**答案：**

```java
public class QuickSort {
    public static void quickSort(int[] arr, int low, int high) {
        if (low < high) {
            int pivot = partition(arr, low, high);
            quickSort(arr, low, pivot - 1);
            quickSort(arr, pivot + 1, high);
        }
    }

    public static int partition(int[] arr, int low, int high) {
        int pivot = arr[high];
        int i = (low - 1);

        for (int j = low; j < high; j++) {
            if (arr[j] < pivot) {
                i++;

                int temp = arr[i];
                arr[i] = arr[j];
                arr[j] = temp;
            }
        }

        int temp = arr[i + 1];
        arr[i + 1] = arr[high];
        arr[high] = temp;

        return i + 1;
    }

    public static void main(String[] args) {
        int[] arr = {10, 7, 8, 9, 1, 5};
        quickSort(arr, 0, arr.length - 1);

    }
}
```

#### 算法编程题库

##### 1. 计算两个整数相加。

**题目：** 编写一个函数，计算两个整数相加。

**答案：**

```java
public class AddTwoIntegers {
    public static int add(int a, int b) {
        while (b != 0) {
            int carry = a & b;
            a = a ^ b;
            b = carry << 1;
        }
        return a;
    }

    public static void main(String[] args) {
        int a = 15;
        int b = 32;
        System.out.println("Sum: " + add(a, b));
    }
}
```

**解析：** 在这个例子中，`add` 函数使用位运算实现整数相加。通过不断计算进位和当前位，直到没有进位为止。

##### 2. 判断一个字符串是否是回文。

**题目：** 编写一个函数，判断一个字符串是否是回文。

**答案：**

```java
public class Palindrome {
    public static boolean isPalindrome(String str) {
        int left = 0;
        int right = str.length() - 1;

        while (left < right) {
            if (str.charAt(left) != str.charAt(right)) {
                return false;
            }
            left++;
            right--;
        }
        return true;
    }

    public static void main(String[] args) {
        String str = "racecar";
        System.out.println("Is palindrome: " + isPalindrome(str));
    }
}
```

**解析：** 在这个例子中，`isPalindrome` 函数通过比较字符串的两端字符，判断字符串是否是回文。
