#### 2. 编写一个基于LLM的医学文献检索系统
**题目：** 编写一个基于LLM的医学文献检索系统，能够接收关键词查询，并返回相关的医学文献。

**答案：**
```python
import spacy
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 加载nlp模型
nlp = spacy.load('en_core_web_md')

# 假设已有医学文献数据集
documents = ...

# 提取文档的向量表示
def get_document_vector(document):
    doc = nlp(document)
    return [token.vector for token in doc]

vectors = [get_document_vector(doc) for doc in documents]

# 构建TF-IDF向量空间
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(vectors)

# 查询函数
def search(query):
    query_vector = nlp(query)
    query_vector = [token.vector for token in query_vector]
    query_vector = vectorizer.transform([query_vector])
    similarity = cosine_similarity(query_vector, X)
    top_docs = similarity.argsort()[-5:][0]
    return [documents[i] for i in top_docs]

# 检索医学文献
query = "disease diagnosis"
results = search(query)
print(results)
```

**解析：** 本题使用spacy库进行文本处理，结合TF-IDF和余弦相似性，实现医学文献的检索功能。
