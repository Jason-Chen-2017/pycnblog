2. **编程题库答案解析：**
   - **题目 1：** 编写函数实现数字替换为汉字，可以使用哈希表存储数字和汉字的对应关系，例如 `{'1': '一', '2': '二', ...}`。遍历字符串，当遇到数字时，查找哈希表替换为汉字。
   - **题目 2：** 编写函数计算字符串相似度，可以使用动态规划算法计算编辑距离。编辑距离表示将一个字符串转换为另一个字符串所需的最小操作次数。具体实现可以参考以下代码：
   
     ```python
     def minEditDistance(str1, str2):
         m, n = len(str1), len(str2)
         dp = [[0] * (n + 1) for _ in range(m + 1)]

         for i in range(m + 1):
             for j in range(n + 1):
                 if i == 0:
                     dp[i][j] = j  # 替换操作
                 elif j == 0:
                     dp[i][j] = i  # 插入操作
                 elif str1[i - 1] == str2[j - 1]:
                     dp[i][j] = dp[i - 1][j - 1]  # 无需操作
                 else:
                     dp[i][j] = min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]) + 1

         return dp[m][n]

     # 示例
     str1 = "abcdef"
     str2 = "azced"
     print(minEditDistance(str1, str2))  # 输出 3
     ```
