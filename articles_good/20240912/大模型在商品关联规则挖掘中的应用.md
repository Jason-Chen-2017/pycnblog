**题目：** 编写一个基于 collaborative filtering 的商品推荐算法，输入用户-商品评分矩阵，输出用户相似度矩阵和推荐结果。

```python
def collaborative_filtering(user_item_matrix):
    # 步骤1：计算用户之间的相似度
    user_similarity = cos_similarity(user_item_matrix)

    # 步骤2：为每个用户生成推荐列表
    recommendations = {}
    for user, ratings in user_item_matrix.items():
        neighbor_ratings = {neighbor: ratings[neighbor] for neighbor in ratings if neighbor in user_similarity[user]}
        if not neighbor_ratings:
            continue
        average_rating = sum(neighbor_ratings.values()) / len(neighbor_ratings)
        recommendation_list = [item for item, rating in ratings.items() if rating < average_rating]
        recommendations[user] = recommendation_list

    return user_similarity, recommendations
```

**解析：** 该程序首先计算用户之间的相似度，然后基于相似度为每个用户生成推荐列表。其中，`cos_similarity` 函数用于计算用户之间的余弦相似度。
