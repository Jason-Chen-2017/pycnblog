def adaptive_control(temperature, humidity, target_temp=25, target_humidity=50):
    # 基于目标温度和湿度，调整环境参数
    if temperature < target_temp:
        # 若温度低于目标温度，增加加热功率
        heating_power = 10  # 增加加热功率
    else:
        heating_power = 0  # 不加热

    if humidity < target_humidity:
        # 若湿度低于目标湿度，增加加湿功率
        humidifying_power = 5  # 增加加湿功率
    else:
        humidifying_power = 0  # 不加湿

    return heating_power, humidifying_power

temperature = 23

print(f"加热功率：{heating_power}, 加湿功率：{humidifying_power}")
```

**解析：** 该算法根据环境温度和湿度的实际情况，自动调整加热和加湿功率，实现环境参数的自适应调节。

def deformation(temp, initial_shape, T1=100, A=0.1):
    # 温度变化引起形状变化，这里假设变形量与温差成线性关系
    delta_temp = temp - T1
    deformation = A * delta_temp
    new_shape = initial_shape + deformation
    return new_shape

initial_shape = 5  # 初始长度为5
temp = 150  # 温度升高到150度
new_shape = deformation(temp, initial_shape)
print(f"新形状：{new_shape:.2f}")
```

**解析：** 该算法假设形状记忆合金的变形量与温差成正比，通过简单的线性模型模拟温度变化引起的形状变化。

```python
def adjust_light_transparency(lux, base_lux=100, transparency_step=0.05):
    # 根据光照强度调节光线透过率
    # 假设光照强度与透过率成反比
    if lux < base_lux:
        transparency = 1 - (lux / base_lux) * transparency_step
    else:
        transparency = 1

    return transparency

current_lux = 80
new_transparency = adjust_light_transparency(current_lux)
print(f"新透过率：{new_transparency:.2f}")
```
