#### 1. 船舶调度问题

**题目：** 编写一个程序，实现船舶在码头的调度，优化船舶停靠时间。

**代码示例：**

```python
import heapq

def schedule_ships(ships, docks):
    # 这里使用优先队列来优化船舶停靠时间
    min_heap = []
    result = []

    # 初始化优先队列
    for dock in docks:
        heapq.heappush(min_heap, (dock.time, dock))

    for ship in ships:
        # 找到可用的最短时间码头
        _, dock = heapq.heappop(min_heap)
        result.append((ship, dock))
        # 计算新的码头可用时间
        dock.time = ship.arrival + ship.duration + dock.service_time

    # 将修改后的码头时间重新放入优先队列
    heapq.heappush(min_heap, (dock.time, dock))

    return result
```

**解析：** 该程序使用优先队列来优化船舶停靠时间，确保每次都能调度到可用的最短时间码头。

```python
def classify_goods(goods, rules):
    classified_goods = {'urgent': [], 'non-urgent': []}

    for good in goods:
        for rule in rules:
            if rule['condition'](good):
                classified_goods[rule['type']].append(good)
                break

    return classified_goods
```

**解析：** 该程序根据预设的规则对货物进行分类，规则可以是函数，用于判断货物是否符合某个分类条件。

#### 3. 装卸效率优化

**题目：** 编写一个程序，根据码头装卸效率，优化货物装卸顺序。

**代码示例：**

```python
import heapq

def optimize Loading(loading_plan, efficiency):
    # 这里使用堆来优化装卸顺序
    max_heap = []
    for item in loading_plan:
        heapq.heappush(max_heap, (-item['duration'], item))

    optimized_plan = []
    while max_heap:
        _, item = heapq.heappop(max_heap)
        optimized_plan.append(item)
        if efficiency > 0:
            efficiency -= item['duration']
        else:
            break

    return optimized_plan
```

**解析：** 该程序使用堆来优化装卸顺序，确保每次都能优先处理耗时最长的任务，从而提高整体装卸效率。
