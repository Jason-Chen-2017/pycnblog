### 4. 区块链在能源交易中的算法编程题

**题目：** 请设计一个区块链系统中的算法，用于实现能源交易的记录和验证。

**答案：**

**算法设计：**

1. **数据结构设计：** 设计一个区块结构，包含交易记录、时间戳、前一个区块哈希等字段。

   ```python
   class Block:
       def __init__(self, transactions, timestamp, previous_hash):
           self.transactions = transactions
           self.timestamp = timestamp
           self.previous_hash = previous_hash
           self.hash = self.calculate_hash()
   
       def calculate_hash(self):
           return hashlib.sha256(
               str(self.timestamp).encode('utf-8') +
               str(self.previous_hash).encode('utf-8') +
               str(self.transactions).encode('utf-8')
           ).hexdigest()
   ```

2. **交易记录验证：** 设计一个验证交易记录的算法，确保交易数据的完整性和正确性。

   ```python
   def verify_transactions(block):
       for transaction in block.transactions:
           # 验证交易数据的格式和完整性
           if not is_valid_transaction(transaction):
               return False
       return True
   ```

3. **区块链构建：** 设计一个构建区块链的算法，包括创建新区块、添加区块到链中等操作。

   ```python
   class Blockchain:
       def __init__(self):
           self.unconfirmed_transactions = []
           self.chain = [self.create_genesis_block()]
   
       def create_genesis_block(self):
           return Block([], time.time(), "0")
   
       def add_new_transaction(self, transaction):
           self.unconfirmed_transactions.append(transaction)
   
       def mine(self):
           if not self.unconfirmed_transactions:
               return "No transactions to mine"
           last_block = self.chain[-1]
           new_block = Block(self.unconfirmed_transactions, time.time(), last_block.hash)
           if verify_transactions(new_block):
               self.chain.append(new_block)
               self.unconfirmed_transactions = []
           else:
               return "Invalid transactions"
   
       def is_chain_valid(self):
           for i in range(1, len(self.chain)):
               current_block = self.chain[i]
               previous_block = self.chain[i - 1]
               if current_block.hash != current_block.calculate_hash():
                   return False
               if current_block.previous_hash != previous_block.hash:
                   return False
           return True
   ```

4. **测试代码：** 编写测试代码，验证算法的正确性和性能。

   def test_blockchain():
       blockchain = Blockchain()
       blockchain.add_new_transaction("Transaction 1")

       assert blockchain.chain[1].transactions == ["Transaction 1", "Transaction 2"]
