#### 线性回归

**代码实例：**

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据准备
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([1, 2, 3, 4])

# 模型训练
model = LinearRegression()
model.fit(X, y)

# 预测
y_pred = model.predict(X)

# 打印结果
print("模型参数：", model.coef_, model.intercept_)
print("预测结果：", y_pred)
```

**解析：**

- 导入必要的库。
- 准备输入特征矩阵 X 和目标变量 y。
- 创建 LinearRegression 模型并调用 fit 方法进行训练。
- 使用 predict 方法对 X 进行预测。
- 打印模型的参数（系数和截距）以及预测结果。
