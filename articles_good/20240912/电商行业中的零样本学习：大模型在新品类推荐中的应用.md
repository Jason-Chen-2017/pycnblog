#### 3. 零样本学习模型的评估方法

**题目：** 请列举几种评估零样本学习模型性能的方法，并说明各自的优缺点。

**答案：** 常见的评估方法包括：

* **准确率（Accuracy）：** 简单但可能过于乐观，尤其在类别分布不平衡时。
* **F1 分数（F1 Score）：** 考虑到精确度和召回率，更适合类别分布不平衡的情况。
* **宏平均（Macro Average）：** 平衡所有类别的性能，但可能忽略类别之间的差异。
* **微平均（Micro Average）：** 综合所有类别的性能，但可能忽略类别内的差异。

**解析：** 不同的评估方法适用于不同的场景和数据分布。例如，F1 分数在类别分布不平衡的情况下更为合适，因为它同时考虑了精确度和召回率。

**源代码示例：**

```python
from sklearn.metrics import accuracy_score, f1_score

# 假设我们有一组真实的标签和预测结果
true_labels = [0, 1, 2, 0, 1]
predicted_labels = [0, 1, 2, 1, 0]

# 计算准确率
accuracy = accuracy_score(true_labels, predicted_labels)
print("Accuracy:", accuracy)

# 计算F1分数
f1 = f1_score(true_labels, predicted_labels, average='weighted')
print("F1 Score:", f1)
```
