#### 1. 生成对抗网络（GAN）的基本概念和结构

**题目：** 请简要介绍生成对抗网络（GAN）的基本概念和结构。

**答案：** 生成对抗网络（GAN）是由生成器（Generator）和判别器（Discriminator）两部分组成的框架，用于生成高质量的假样本。生成器从随机噪声中生成样本，判别器的任务是区分真实样本和生成样本。

**解析：**
- 生成器：接收随机噪声作为输入，生成与真实样本相似的样本。
- 判别器：接收真实样本和生成样本，输出它们是真实样本的概率。

**解析：** 这些策略与优化方法可以帮助GAN更好地训练，避免过拟合和模式崩溃。

#### 5. 实现一个简单的GAN模型

import tensorflow as tf
from tensorflow.keras.layers import Dense, Flatten, Reshape
from tensorflow.keras.models import Sequential

# 生成器模型
def generator(z, latent_dim):
    model = Sequential([
        Dense(128, activation="relu", input_shape=(latent_dim,)),
        Dense(28*28*1, activation="relu"),
        Reshape((28, 28, 1))
    ])
    return model(z)

# 判别器模型
def discriminator(x, latent_dim):
    model = Sequential([
        Flatten(input_shape=(28, 28, 1)),
        Dense(128, activation="relu"),
        Dense(1, activation="sigmoid")
    ])
    return model(x)

# GAN模型
def GAN(generator, discriminator, z):
    g_model = generator(z)
    d_model = discriminator(g_model)
    return tf.keras.Model(inputs=z, outputs=d_model)
