### 5.2 源代码详细实现

以下是一个简单的自然语言处理项目的代码实现，用于计算两个词语的相似度。

```python
import numpy as np
from gensim.models import KeyedVectors

# 读取预训练的词向量模型
word_vectors = KeyedVectors.load_word2vec_format('word2vec.bin', binary=True)

# 计算两个词语的相似度
def word_similarity(word1, word2):
    vector1 = word_vectors[word1]
    vector2 = word_vectors[word2]
    similarity = np.dot(vector1, vector2) / (np.linalg.norm(vector1) * np.linalg.norm(vector2))
    return similarity

# 测试代码
similarity = word_similarity('cat', 'dog')
print(similarity)
```

1. **读取预训练的词向量模型**：使用 `KeyedVectors.load_word2vec_format()` 函数，读取预训练的词向量模型。
2. **定义相似度计算函数**：定义一个函数 `word_similarity()`，用于计算两个词语的相似度。它首先获取两个词语的向量表示，然后使用余弦相似度公式计算它们的相似度。
