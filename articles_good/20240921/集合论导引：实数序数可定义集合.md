## 2. 核心概念与联系

### 实数的构造

实数可以通过不同的方式构造出来，其中最常见的两种方法是基于自然数和有理数的扩展。自然数和有理数都是集合，因此，实数的构造本质上是一个集合论问题。

**康托尔的基数理论**提供了实数构造的框架。康托尔通过基数（无限集合的“大小”）的概念，定义了实数集$\mathbb{R}$。具体来说，实数集$\mathbb{R}$可以通过集合$\mathbb{N}$（自然数集）和$\mathbb{Q}$（有理数集）的基数扩展来构造。

$$
\mathbb{R} = \{x \mid x \text{ 是一个基数大于等于 } \mathbb{Q} \text{ 的无限集合}\}
$$

### 序数的概念

序数是集合论中用于描述集合之间顺序关系的特殊集合。序数本质上是一个可以排序的集合，它满足全序关系和传递性质。

**序数的构造**基于自然数和集合的嵌套。具体来说，序数可以通过集合的递归定义来构造。最小的序数是自然数集$\mathbb{N}$，它是零序数。然后，通过将序数视为集合的基数，可以构造出更高阶的序数。

$$
\alpha_0 = \mathbb{N}
$$
$$
\alpha_{\beta+1} = \alpha_\beta \cup \{\alpha_\beta\}
$$
$$
\alpha_\lambda = \bigcup_{\beta<\lambda} \alpha_\beta
$$

其中，$\beta$和$\lambda$是序数。

### 可定义集合的概念

可定义集合是指可以通过有限步骤的集合论操作从基本集合中构造出来的集合。可定义集合的概念在形式化数学和计算机科学中非常重要。

**可定义集合的定义**：

一个集合$A$是可定义的，如果存在一个公式$\phi(x)$和一组参数$P$，使得对于任意的模型$\mathcal{M}$，如果$\mathcal{M} \models \phi(x)^P$，则$x \in A$。

$$
A = \{x \mid \mathcal{M} \models \phi(x)^P\}
$$

def construct_real_numbers():

    rational_numbers = set()
    for p in natural_numbers:

            if q != 0:

def construct_definable_set():
    # 定义公式和参数
    formula = sympy.PredicateExpression(x, sympy.Rel(x, sympy.Symbol('y'), sympy.Equality))
    parameters = []
    
    # 构造可定义集合
    definable_set = set()
    for r in construct_real_numbers():
