# 2.核心概念与联系

## 2.1. 计算机结构与组成

计算机结构与组成，主要是描述计算系统内部的机械、电气、机电元素之间的相互作用，以及这些元素的功能和特性。它涵盖了计算机内存、CPU结构、输入输出设备结构、存储器层次结构、总线结构、总线性能、计算机性能指标等方面的内容。其中，了解计算机的基本结构有助于我们理解程序执行过程、识别、优化程序中的错误、提升计算机的性能和扩展性等。

**CPU结构**：描述计算机的运算单元、控制器、寄存器以及各种控制信号之间的关系。它与数据、指令之间的交换、控制流转等有关。

**存储器层次结构**：指存储器从简单到复杂的存储器类型，并结合计算机的要求，定义出一个适合于计算机处理的数据存储结构。它考虑了速度、容量、价格等因素。

**总线结构**：描述计算机主板、扩展卡、接口卡等各种外设与计算机之间的通信方式，以及计算机与其他计算机之间的通信方式。

## 2.2. 数据表示与处理

数据表示与处理，是指计算机所用数据的格式、编码方式、大小端、字节序、浮点数精度、运算精度、并行计算等方面的内容。它涵盖了有符号整数、无符号整数、实数、复数、浮点数、向量、矩阵、图像、音频、视频等数据类型的定义、表示和转换方法，以及常用的存储、压缩、加密等方法。了解这些原理对程序开发人员来说，可以有效地避免不同数据类型间的混乱问题，提升数据处理效率。

## 2.3. 算术逻辑运算与控制

算术逻辑运算与控制，主要介绍算术运算、逻辑运算与控制结构。它详述了如何通过一系列的运算和指令完成指定的任务，以及如何更好地控制指令流，提升计算机性能。例如，它的运算逻辑结构可以用来实现加减乘除，条件判断，循环控制等；它的控制结构如if-else语句、case语句、跳转语句等，可以帮助程序执行指定的操作流程。

## 2.4. 算法与编码

算法与编码，主要涉及计算机程序设计语言的相关知识。它涵盖了各种算法的分类、建立时间、空间复杂度、正确性证明、复杂度分析、排序、搜索、图论、贪心算法、动态规划、递归算法、枚举法、字符串匹配算法、回溯算法、分治算法等方面内容。这些算法技术能够帮助计算机解决实际的问题，还能提供给工程师更好的解决问题的方法。

**编程语言**：是一种用于编写计算机软件的指令集合。主要包括汇编语言、高级语言和脚本语言。其中，高级语言又可细分为静态强类型语言和动态弱类型语言。

## 2.5. 操作系统与接口

操作系统与接口，主要关注操作系统的运行机制、调度策略、进程/线程管理、文件系统管理、虚拟内存管理、网络管理、驱动管理等方面内容。操作系统具有完整性、一致性、可用性和安全性，能够协调应用程序的运行和资源共享，并且保证应用程序的正常运行。

**进程/线程管理**：涉及操作系统如何管理进程和线程，以及如何保证并发执行的正确性。

**文件系统管理**：涉及操作系统如何组织和管理文件系统，以及文件如何存储、检索、管理等。

**虚拟内存管理**：对应用程序的地址空间进行抽象和管理，使得程序可以像访问真实内存一样访问存储在磁盘上的文件，提高程序的运行速度和稳定性。

**网络管理**：包括网络协议栈管理、路由协议管理、网络安全管理等方面。

**驱动管理**：包括设备注册、初始化、打开和关闭、读写操作、中断处理等方面。

## 2.6. 数据库管理系统

数据库管理系统（DBMS），是管理用户关系型数据集合并支持多种查询语言的软件。它由存储和检索数据的管理模块、SQL命令解析模块、事务管理模块、安全管理模块、日志记录模块等组成。它提供了对数据的安全、可靠性、完整性、并发访问、查询语言的支持和改进，具有很强的生命周期意识和持续发展的特征。

**SQL**：Structured Query Language，结构化查询语言，是一种数据库查询语言。它是在关系模型数据库管理系统中使用的标准语言。

**索引**：索引是一种特殊的数据结构，它用于快速找到某个值在一个大的集合中的位置。

**数据库事务**：数据库事务是由一组动作组成的一个工作单位，它包括对数据库中数据的插入、删除、修改等操作，可以确保数据处理的原子性、一致性和隔离性。

**缓存机制**：缓冲区就是内存中的临时存储区，它被用来暂时保存数据，以便后续处理。缓存机制的目的是为了提高系统的运行效率，降低数据库负载。

## 2.7. 机器学习与深度学习

机器学习与深度学习，是关于计算机系统如何利用经验自动调整参数，从而达到预测新情况、解决问题或优化性能的理论和方法。它们共同构建起的用于人工智能、机器人、自然语言处理等领域的计算机科学技术基础。

**机器学习**：是一类用于获取“知识”的算法，并根据该知识对未知数据进行预测、分类和评估。这种学习方法通常使用训练数据集来拟合模型参数，然后应用模型对新数据进行预测、分类和评估。机器学习算法的关键是利用训练数据来估计模型参数，从而让系统产生可行且具有说服力的推论。

**深度学习**：深度学习是机器学习的子领域，它是建立多个非线性函数的神经网络，模仿生物神经元构造的神经网络来进行数据处理和学习。深度学习是基于对数据中相互关联的模式进行学习，通过迭代训练的方式不断提高学习效果。

## 2.8. 分布式计算与云计算

分布式计算与云计算，是两个相关领域，涉及分布式系统的设计和云计算服务的提供。分布式计算是指多台计算机按照一定的规则一起工作，共同解决一件任务。分布式计算的目标是为了解决单台计算机无法处理的大数据量，采用分布式方式将工作量分布到多台计算机上，通过网络连接，共同完成整个任务。分布式计算的主要技术包括并行计算、集群计算、超级计算机、云计算等。

云计算是一种按需使用网络服务器的服务，它是一种通过网络快速部署应用程序、随时扩展或缩小计算能力的服务。云计算服务的客户可以在不同的地方同时使用同样的应用程序，获得比传统IT资源更高的利用率，并能节省IT支出。

## 3.1. RSA算法 

RSA算法(Rivest–Shamir–Adleman)是美国密码学家罗纳德·李维斯特（Rivest）、阿迪·舒马赫（Shamir）和安东尼·希姆莱（Adleman）于1977年提出的公钥密码算法。它是第一个能同时实现加密和数字签名的公钥密码算法，并且安全性非常高。 

### 算法概述 

RSA算法是一种公钥加密算法，由两部分组成，即密钥生成和密钥分配。

1. 密钥生成：生成公钥和私钥对，公钥所有人公开，私钥只有该用户自己知道。公钥和私钥是相互对应关系，任意一对公钥对应的私钥只能由唯一的该私钥对应的公钥来解密。

2. 密钥分配：在网络上传送公钥，私钥只有接收者可以得到，他就可以用公钥来加密信息，然后用私钥来解密信息。

3. 使用：加密：先选择一对密钥对(公钥pubKey和私钥priKey)，然后将待加密的信息m使用pubKey加密，得到c。解密：用私钥priKey解密得到m。 

### 具体操作步骤 

1. 生成一对公钥和私钥，公钥A=(n,e),私钥B=(n,d)。
2. 将公钥A发送给B。
3. B收到公钥A后，用自己的私钥B=privKeyDecrypt(n,d)解密得到d，他就可以用公钥A=pubKeyDecrypt(n,e)加密信息。
4. A得到的加密消息c可以用B的公钥B=pubKeyDecrypt(n,e)解密得到信息。 

### 模型公式 

#### 加密

加密过程：c = pubKeyEncrypt(m, e, n) 

pubKeyEncrypt() 函数的参数分别为待加密的消息 m ，公钥 exponent e 和 n 。该函数返回加密结果 c 。

公钥加密函数：

publicKeyEncrypt(m, e, n):
    if len(m)<n:
        # 补齐m长度为n的倍数
        for i in range((len(m)+n)%n):

        raise Exception('Message too long')
    # 普通加密过程
    return fastExpMod(int_to_bytes(long(hexlify(m), base=16)), e, int(n))

#### 解密

解密过程：m = privKeyDecrypt(c, d, n) 

privKeyDecrypt() 函数的参数分别为待解密的加密结果 c ，私钥 exponent d 和 n 。该函数返回解密结果 m 。

私钥解密函数：

privateKeyDecrypt(c, d, n):
    messageBytes = bytearray(fastExpMod(c, d, int(n)))
    messageString = ''
    for b in messageBytes:
        if b == '\x00': break;   // 遇到'\x00'结束
        messageString += chr(b);
    return messageString

## 3.2. AES算法 

AES(Advanced Encryption Standard)是美国联邦政府采用的一种区块加密标准。对称加密算法可以分成两步：加密和解密。第一次对称加密使用的是对称密钥，第二次对称加密使用的是随机数。AES是一个高级加密标准，速度快，安全级别高。

### 算法概述 

AES是对称加密算法，分组密码的一种，又称Rijndael加密算法。它的优点是加密速度快，目前已被广泛使用。

它有如下特点：

1. 对称性：算法采用对称加密方法，加密和解密使用相同的密钥，属于对称加密算法。

2. 块大小：块大小为128bit，即16字节。

3. 密钥：密钥可以长度为128、192或256bit，分别对应16字节、24字节和32字节。如果密钥较短，则会重复使用密钥，使得密钥重复次数越多，攻击者破解的时间就越长。建议密钥长度至少为128bit。

4. 分组加密：对每一段明文，先将其切割成若干个固定长度（默认为128位）的独立的块，然后对每个块进行加密操作。对最后一块使用填充方式进行填充。

5. 初始向量：每个块加密过程中，都会增加一个初始向量IV，该向量用于与当前块进行异或运算。IV也是随机生成，具有良好的随机性。

6. 校验码：在最后一个块的结尾添加了一个16进制的校验码。用于检验数据是否损坏。

7. 迭代轮数：默认迭代10轮。

### 具体操作步骤 

1. 设置密钥，密钥长度为128、192或256bit。
2. 用密钥加密或解密。

加密过程：

输入：明文 plaintext，初始向量 IV 
输出：密文 ciphertext

3. 初始化轮密钥。 
4. 将IV拓展为密钥和偏置。 
5. 根据128位的轮密钥、初始向量和输入明文进行轮密钥加密。 
6. 返回加密结果。

解密过程：

输入：密文 ciphertext 
输出：明文 plaintext

7. 初始化轮密钥。 
8. 根据128位的轮密钥、初始向量和输入密文进行轮密钥解密。 
9. 检查校验码。 
10. 返回解密结果。

### 模型公式 

#### 加密

加密过程：ciphertext = encryptBlock(plaintext, key, iv) 

encryptBlock() 函数的参数分别为待加密的明文 plaintext ，密钥 key ，初始向量 iv 。该函数返回加密结果 ciphertext 。

AES加密过程：

1. 首先，检查明文长度是否为16的倍数，不是的话，用空格补齐。 
2. 判断是否为AES加密算法。 
3. 获取初始密钥，判断是否存在初始向量。 
4. 如果不存在初始向量，则随机生成。 
5. 取出明文分组，使用AES加密算法进行加密。 
6. 拼接加密结果。 
7. 返回加密结果。

AES加密算法：

encryptBlock():
    aes = pyaes.AESModeOfOperationCBC(key, iv)
    blocks = [plaintext[i:i+16] for i in range(0, len(plaintext), 16)]    // 把明文分组为16字节的形式
    cipherBlocks = []
    for block in blocks:
        cipherBlocks.append(aes.encrypt(block))
    ciphertext = ''.join([str(cipher).encode('hex').upper() for cipher in cipherBlocks])
    paddingLength = 16 - (len(plaintext) % 16)     // 填充长度
    paddedPlaintext = str(bytearray(plaintext + ('\x00'*paddingLength)).encode('hex')).upper().ljust(len(ciphertext)*2,' ')      // 填充明文
    return paddedPlaintext

#### 解密

解密过程：plaintext = decryptBlock(ciphertext, key, iv) 

decryptBlock() 函数的参数分别为待解密的密文 ciphertext ，密钥 key ，初始向量 iv 。该函数返回解密结果 plaintext 。

AES解密过程：

1. 判断是否为AES加密算法。 
2. 获取初始密钥，判断是否存在初始向量。 
3. 如果不存在初始向vedn，则设置默认初始向量。 
4. 根据密文长度分组。 
5. 遍历每一分组，使用AES解密算法进行解密。 
6. 拼接解密结果。 
7. 返回解密结果。

AES解密算法：

    aes = pyaes.AESModeOfOperationCBC(key, iv)
    blocks = [(ciphertext[i*2:(i+1)*2]).decode('hex') for i in range(len(ciphertext)//2)]       // 把密文分组
