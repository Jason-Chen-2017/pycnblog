##### Linear Regression

#### 线性回归

Linear regression is a fundamental statistical technique used to model the relationship between a dependent variable and one or more independent variables. It assumes a linear relationship between the variables and aims to find the best-fitting line that minimizes the sum of squared errors.

线性回归是一种基本的统计方法，用于描述一个因变量与一个或多个自变量之间的关系。它假设变量之间存在线性关系，并旨在找到最小化平方误差和的最佳拟合线。

**Mathematical Model:**

$$
y = \beta_0 + \beta_1x + \epsilon
$$

Where:

- \( y \) is the dependent variable.
- \( x \) is the independent variable.
- \( \beta_0 \) is the intercept.
- \( \beta_1 \) is the slope.
- \( \epsilon \) is the error term.

**Example:**

Suppose we want to predict the sales of a product based on its advertising expenditure. We collect data on the advertising expenditure (\( x \)) and the corresponding sales (\( y \)) for different time periods. We can use linear regression to find the relationship between these variables.

**Mathematical Model:**

$$
y = \beta_0 + \beta_1x + \epsilon
$$

Where:

- \( y \) is the sales.
- \( x \) is the advertising expenditure.
- \( \beta_0 \) is the intercept.
- \( \beta_1 \) is the slope.
- \( \epsilon \) is the error term.

We can use the following data points:

| \( x \) | \( y \) |
|---------|---------|
| 1000    | 5000    |
| 1500    | 7000    |
| 2000    | 9000    |
| 2500    | 11000   |
| 3000    | 13000   |

Using linear regression, we can find the best-fitting line and the relationship between advertising expenditure and sales:

**Mathematical Model:**

$$
y = 2000 + 3x
$$

Where:

- \( \beta_0 \) = 2000 (Intercept)
- \( \beta_1 \) = 3 (Slope)

**Conclusion:**

For every unit increase in advertising expenditure (\( x \)), the sales (\( y \)) are expected to increase by 3 units.

##### Support Vector Machine (SVM)

#### 支持向量机（SVM）

Support Vector Machine (SVM) is a supervised machine learning algorithm used for classification and regression tasks. It aims to find the hyperplane that best separates the data points of different classes in a high-dimensional space.

支持向量机（SVM）是一种用于分类和回归任务的监督机器学习算法。它旨在找到高维空间中不同类别的数据点之间的最佳超平面。

**Mathematical Model:**

$$
\text{Maximize } \frac{1}{2} ||\textbf{w}||^2 \text{ subject to } y_i (\textbf{w} \cdot \textbf{x_i}) \geq 1 \text{ for all } i
$$

Where:

- \( \textbf{w} \) is the weight vector.
- \( \textbf{x_i} \) is the data point.
- \( y_i \) is the class label.
- \( \cdot \) represents the dot product.

**Example:**

Consider a binary classification problem with two classes, \( +1 \) and \( -1 \). We have a set of data points in a two-dimensional space:

| \( x_1 \) | \( x_2 \) | \( y \) |
|-----------|-----------|---------|
| 1         | 2         | +1      |
| 3         | 4         | +1      |
| 5         | 6         | +1      |
| 7         | 8         | -1      |
| 9         | 10        | -1      |
| 11        | 12        | -1      |

We want to find the hyperplane that best separates these data points. Using SVM, we can find the optimal hyperplane:

**Mathematical Model:**

$$
\text{Maximize } \frac{1}{2} ||\textbf{w}||^2 \text{ subject to } y_i (\textbf{w} \cdot \textbf{x_i}) \geq 1 \text{ for all } i
$$

Where:

- \( \textbf{w} \) is the weight vector.
- \( \textbf{x_i} \) is the data point.
- \( y_i \) is the class label.
- \( \cdot \) represents the dot product.

The optimal hyperplane is:

$$
\textbf{w} \cdot \textbf{x} + b = 0
$$

Where:

- \( b \) is the bias term.

The data points on the hyperplane satisfy the condition \( y_i (\textbf{w} \cdot \textbf{x_i}) \geq 1 \).

##### Neural Networks

#### 神经网络

Neural networks are a class of machine learning algorithms inspired by the structure and function of the human brain. They consist of interconnected nodes (neurons) that process and transmit information.

神经网络是一类受人类大脑结构和功能启发的机器学习算法。它们由相互连接的节点（神经元）组成，用于处理和传输信息。

**Mathematical Model:**

A simple neural network can be represented as follows:

$$
\textbf{z} = \textbf{W} \cdot \textbf{x} + b
$$

Where:

- \( \textbf{z} \) is the output of the neural network.
- \( \textbf{W} \) is the weight matrix.
- \( \textbf{x} \) is the input vector.
- \( b \) is the bias term.

The activation function is used to introduce non-linearity into the network:

$$
a = \sigma(\textbf{z})
$$

Where:

- \( a \) is the activation value.
- \( \sigma \) is the activation function (e.g., sigmoid, ReLU, tanh).

**Example:**

Consider a simple neural network with one input layer, one hidden layer, and one output layer. The input vector is \( \textbf{x} = [1, 2] \), and the weight matrix is \( \textbf{W} = \begin{bmatrix} 0.5 & 0.3 \\ 0.2 & 0.4 \end{bmatrix} \). The bias term is \( b = [0.1, 0.2] \). The activation function is \( \sigma(z) = \frac{1}{1 + e^{-z}} \).

The output of the network is:

$$
\textbf{z} = \textbf{W} \cdot \textbf{x} + b = \begin{bmatrix} 0.5 & 0.3 \\ 0.2 & 0.4 \end{bmatrix} \cdot \begin{bmatrix} 1 \\ 2 \end{bmatrix} + \begin{bmatrix} 0.1 \\ 0.2 \end{bmatrix} = \begin{bmatrix} 1.1 \\ 1.2 \end{bmatrix}
$$

The activation values are:

$$
a = \sigma(\textbf{z}) = \begin{bmatrix} \frac{1}{1 + e^{-1.1}} \\ \frac{1}{1 + e^{-1.2}} \end{bmatrix} \approx \begin{bmatrix} 0.7 \\ 0.6 \end{bmatrix}
$$

By training the neural network with labeled data, we can adjust the weights and biases to improve the network's performance.

#### Source Code Implementation

The following is the Python code for implementing a simple neural network for image recognition using TensorFlow:

```python
import tensorflow as tf
from tensorflow.keras import layers

# Load the dataset
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.mnist.load_data()

# Preprocess the data
x_train = x_train / 255.0
x_test = x_test / 255.0
x_train = x_train[..., tf.newaxis]
x_test = x_test[..., tf.newaxis]

# Define the neural network architecture
model = tf.keras.Sequential([
    layers.Flatten(input_shape=(28, 28)),
    layers.Dense(128, activation='relu'),
    layers.Dense(10, activation='softmax')
])

# Compile the model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Train the model
model.fit(x_train, y_train, epochs=5)

# Evaluate the model
test_loss, test_acc = model.evaluate(x_test, y_test, verbose=2)
print('\nTest accuracy:', test_acc)
```
