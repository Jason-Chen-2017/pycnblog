### 4. 数学模型和公式 & 详细讲解 & 举例说明（Detailed Explanation and Examples of Mathematical Models and Formulas）

#### 4.1 注意力分配模型

在注意力生物节律优化算法中，注意力分配模型是核心部分。该模型用于预测用户在不同生物节律状态下的注意力水平，并据此调整任务分配策略。

注意力分配模型可以表示为：

\[ A(t) = f(B(t), C(t)) \]

其中，\( A(t) \) 表示在时间 \( t \) 的注意力水平，\( B(t) \) 表示生物节律特征，\( C(t) \) 表示任务特征。函数 \( f \) 用于结合生物节律和任务特征，以预测注意力水平。

#### 4.2 生物节律特征提取

生物节律特征提取是注意力生物节律优化算法的关键步骤。本文采用了一种基于时间序列分析的方法，提取用户的昼夜节律和月相节律特征。

昼夜节律特征可以表示为：

\[ B_c(t) = \sin(2\pi t/T_c) \]

其中，\( T_c \) 表示昼夜周期，通常为24小时。

月相节律特征可以表示为：

\[ B_m(t) = \sin(2\pi t/T_m) \]

其中，\( T_m \) 表示月相周期，通常为29.5天。

#### 4.3 任务特征提取

任务特征提取用于描述任务的复杂性和紧急程度。本文采用了一种基于任务标签的词袋模型进行特征提取。

假设有 \( n \) 个任务，每个任务可以表示为一个标签集合。任务特征可以表示为：

\[ C(t) = [c_1(t), c_2(t), ..., c_n(t)] \]

其中，\( c_i(t) \) 表示任务 \( i \) 在时间 \( t \) 的特征值，可以通过词袋模型计算得到。

#### 4.4 举例说明

假设用户在一天中的不同时间段有四个任务需要完成，其生物节律特征和任务特征如下表所示：

| 时间 | 生物节律特征 \( B(t) \) | 任务特征 \( C(t) \) |
| ---- | ---------------------- | ------------------ |
| 8:00 | \( B_8 = 0.5 \)         | \( C_8 = [1, 0, 0, 0] \) |
| 12:00 | \( B_{12} = 0.8 \)      | \( C_{12} = [0, 1, 0, 0] \) |
| 16:00 | \( B_{16} = 0.3 \)      | \( C_{16} = [0, 0, 1, 0] \) |
| 20:00 | \( B_{20} = 0.6 \)      | \( C_{20} = [0, 0, 0, 1] \) |

根据注意力分配模型，可以预测用户在各个时间段的注意力水平，并据此调整任务分配策略。

### 4. Mathematical Models and Formulas & Detailed Explanation and Examples

#### 4.1 Attention Allocation Model

In the Attention-Biological Rhythm Optimization Algorithm, the attention allocation model is the core component. This model is used to predict the level of attention of a user at different biological rhythm states and adjust the task allocation strategy accordingly.

The attention allocation model can be represented as:

\[ A(t) = f(B(t), C(t)) \]

Where \( A(t) \) represents the level of attention at time \( t \), \( B(t) \) represents the biological rhythm features, and \( C(t) \) represents the task features. The function \( f \) is used to combine the biological rhythm and task features to predict the attention level.

#### 4.2 Extraction of Biological Rhythm Features

The extraction of biological rhythm features is a critical step in the Attention-Biological Rhythm Optimization Algorithm. This article adopts a time-series analysis-based method to extract the circadian and lunar rhythm features of the user.

The circadian rhythm feature can be represented as:

\[ B_c(t) = \sin(2\pi t/T_c) \]

Where \( T_c \) represents the circadian cycle, typically 24 hours.

The lunar rhythm feature can be represented as:

\[ B_m(t) = \sin(2\pi t/T_m) \]

Where \( T_m \) represents the lunar cycle, typically 29.5 days.

#### 4.3 Extraction of Task Features

Task feature extraction is used to describe the complexity and urgency of tasks. This article adopts a bag-of-words model based on task tags for feature extraction.

Assuming there are \( n \) tasks that need to be completed by the user at different time intervals, each task can be represented as a set of tags. The task features can be represented as:

\[ C(t) = [c_1(t), c_2(t), ..., c_n(t)] \]

Where \( c_i(t) \) represents the feature value of task \( i \) at time \( t \), which can be calculated using a bag-of-words model.

#### 4.4 Example

Assume that a user has four tasks to complete at different times throughout the day, with the biological rhythm features and task features as shown in the following table:

| Time | Biological Rhythm Feature \( B(t) \) | Task Feature \( C(t) \) |

| 20:00 | \( B_{20} = 0.6 \)      | \( C_{20} = [0, 0, 0, 1] \) |

According to the attention allocation model, the attention level of the user at each time interval can be predicted, and the task allocation strategy can be adjusted accordingly.
