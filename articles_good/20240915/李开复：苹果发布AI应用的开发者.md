#### 线性回归

#### 神经网络

- **前向传播**：$$z^{[l]} = \sigma(W^{[l-1]} a^{[l-1]} + b^{[l-1]})$$

- **反向传播**：$$\delta^{[l]} = \frac{\partial L}{\partial z^{[l]}} \odot \frac{\partial \sigma}{\partial z^{[l]}}$$

### 5.2 源代码详细实现

以下是一个简单的线性回归代码实现：

```python
import numpy as np
import matplotlib.pyplot as plt

# 数据集
X = np.array([[1, 2], [1, 4], [1, 6]])
y = np.array([3, 7, 11])

# 模型参数
theta = np.zeros((2, 1))

# 学习率
alpha = 0.01

# 迭代次数
num_iterations = 1000

# 梯度下降
for i in range(num_iterations):
    z = np.dot(X, theta)
    error = y - z
    d_error = X.T.dot(error)
    theta = theta - alpha * d_error

# 训练结果
plt.scatter(X[:, 1], y)
plt.plot(X[:, 1], X.dot(theta), color='red')
plt.show()
```

import numpy as np
import matplotlib.pyplot as plt

# 数据集
X = np.array([[1, 2], [1, 4], [1, 6]])
y = np.array([3, 7, 11])

# 模型参数
theta = np.zeros((2, 1))

# 学习率
alpha = 0.01

# 迭代次数
num_iterations = 1000

# 梯度下降
for i in range(num_iterations):
    z = np.dot(X, theta)
    error = y - z
    d_error = X.T.dot(error)
    theta = theta - alpha * d_error

# 训练结果
plt.scatter(X[:, 1], y)
plt.plot(X[:, 1], X.dot(theta), color='red')
plt.show()
```
