# 4.具体代码实例和详细解释说明

## 4.1 简单的命令行工具示例

以下是一个简单的Go语言命令行工具示例：

```go
package main

import (
	"flag"
	"fmt"
	"os"
)

func main() {
	// 定义命令行参数
	name := flag.String("name", "", "Your name")
	flag.Parse()

	// 处理命令行参数
	if *name == "" {
		fmt.Println("Please input your name.")
		os.Exit(1)
	}

	// 输出结果
	fmt.Printf("Hello, %s.\n", *name)
}
```

在上述示例中，我们使用`flag`包对命令行参数进行解析，并根据解析后的参数输出结果。

## 4.2 文件操作示例

以下是一个Go语言命令行工具示例，用于读取文件内容：

```go
package main

import (
	"flag"
	"fmt"
	"io/ioutil"
	"os"
)

func main() {
	// 定义命令行参数
	filename := flag.String("filename", "", "Input file name")
	flag.Parse()

	// 处理命令行参数
	if *filename == "" {
		fmt.Println("Please input file name.")
		os.Exit(1)
	}

	// 读取文件内容
	content, err := ioutil.ReadFile(*filename)
	if err != nil {
		fmt.Printf("Error: %v\n", err)
		os.Exit(1)
	}

	// 输出结果
	fmt.Printf("File content: %s\n", content)
}
```

在上述示例中，我们使用`io/ioutil`包实现文件读取操作，并根据文件内容输出结果。
