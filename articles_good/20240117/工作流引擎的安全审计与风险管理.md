   Risk\_Control = Risk - Control(Risk)

   其中，$Risk\_Control$ 表示控制后的风险，$Control(Risk)$ 表示风险控制的效果。

# 安全审计数学模型公式实现
def risk(probability, impact):
    return probability * impact

# 风险管理数学模型公式实现
def risk_control(risk, control):
    return risk - control

# 示例数据
