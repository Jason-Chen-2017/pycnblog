## 4.1 定义模型

定义模型是PyTorch的核心操作步骤，通过定义模型，实现模型的构建。以下是一个简单的神经网络模型的定义：

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.fc1 = nn.Linear(784, 128)
        self.fc2 = nn.Linear(128, 64)
        self.fc3 = nn.Linear(64, 10)

    def forward(self, x):
        x = F.relu(self.fc1(x))
        x = F.relu(self.fc2(x))
        x = self.fc3(x)
        return x

net = Net()
```

在上述代码中，我们定义了一个简单的神经网络模型，包括三个全连接层。`nn.Linear` 用于定义全连接层，`F.relu` 用于定义激活函数。

## 4.2 训练模型

训练模型是PyTorch的核心操作步骤，通过训练模型，实现模型的优化。以下是一个简单的神经网络模型的训练：

```python
import torch.optim as optim

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.01, momentum=0.9)

for epoch in range(10):
    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
        inputs, labels = data
        optimizer.zero_grad()
        outputs = net(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()
        running_loss += loss.item()
    print('Epoch: %d loss: %.3f' % (epoch + 1, running_loss / len(trainloader)))
```

在上述代码中，我们定义了一个简单的神经网络模型，包括三个全连接层。`nn.CrossEntropyLoss` 用于定义损失函数，`optim.SGD` 用于定义优化算法。

```python

        _, predicted = torch.max(outputs.data, 1)
```
