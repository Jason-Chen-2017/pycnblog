# 4.具体代码实例和详细解释说明

## 4.1 使用unittest框架编写测试用例

unittest是Python的一个内置模块，它提供了一种结构化的方式来编写和维护测试用例。以下是一个使用unittest框架编写测试用例的示例：

```python
import unittest

class TestCalculator(unittest.TestCase):

    def test_add(self):
        self.assertEqual(2 + 2, 4)

    def test_subtract(self):
        self.assertEqual(5 - 3, 2)

if __name__ == '__main__':
    unittest.main()
```

在上述示例中，我们定义了一个名为TestCalculator的测试类，该类继承自unittest.TestCase类。然后，我们定义了两个测试方法：test_add和test_subtract。在这两个测试方法中，我们使用assertEqual方法来验证输入和预期输出是否相等。最后，我们使用unittest.main()方法来执行测试用例。

## 4.2 使用pytest框架编写测试用例

pytest是Python的一个第三方框架，它提供了一种更加简洁的方式来编写和维护测试用例。以下是一个使用pytest框架编写测试用例的示例：

```python
import pytest

def add(x, y):
    return x + y

def test_add():
    assert add(2, 2) == 4

def test_subtract():
    assert add(5, -3) == 2
```

在上述示例中，我们定义了一个名为add的函数，该函数实现了加法功能。然后，我们定义了两个测试函数：test_add和test_subtract。在这两个测试函数中，我们使用assert关键字来验证输入和预期输出是否相等。最后，我们使用pytest框架自动执行测试用例。
