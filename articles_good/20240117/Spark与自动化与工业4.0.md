## 1.4 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释Spark在自动化与工业4.0中的应用。

假设我们有一个生产数据集，包含生产线上的各种设备的运行数据。我们希望通过Spark来实现对这些数据的处理、分析和挖掘，从而提高生产效率、降低成本、提高产品质量。

首先，我们需要创建一个SparkSession对象，并加载生产数据集：

```python
from pyspark.sql import SparkSession

spark = SparkSession.builder.appName("industry4.0").getOrCreate()

data = spark.read.csv("path/to/your/data.csv", header=True, inferSchema=True)
```

接下来，我们可以通过Spark的数据框架API来实现对生产数据的处理、分析和挖掘。例如，我们可以通过以下代码来实现对生产数据的统计分析：

```python
from pyspark.sql.functions import avg, sum, count

statistics = data.agg(avg("temperature").alias("avg_temperature"),
                      sum("pressure").alias("sum_pressure"),
                      count("device_id").alias("device_count"))

statistics.show()
```
