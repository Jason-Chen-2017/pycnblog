## 3.1 RSA算法原理和操作步骤

RSA算法是一种公钥加密算法，它的核心思想是使用两个不同的大素数生成一个大素数，然后使用这个大素数生成公钥和私钥。RSA算法的安全性主要依赖于大素数的难以解决的数学问题。

RSA算法的原理和操作步骤如下：

1. 选择两个大素数p和q，使得p和q互质，并且p>q。
2. 计算n=p*q。
3. 计算φ(n)=(p-1)*(q-1)。
4. 选择一个大素数e，使得1<e<φ(n)，且gcd(e,φ(n))=1。
5. 计算d=e^(-1)modφ(n)。
6. 公钥为(n,e)，私钥为(n,d)。

在RSA算法中，加密和解密的过程如下：

1. 对于明文m，使用公钥(n,e)进行加密，得到密文c：c=m^e mod n。
2. 使用私钥(n,d)解密密文c，得到明文m：m=c^d mod n。

import math

def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b)

def ext_gcd(a, b):
    if b == 0:
        x = 1
        y = 0
        d = a
    else:
        d, y, x = ext_gcd(b, a % b)
        y -= math.floor(a / b) * x

def rsa_key_gen(p, q):

    phi = (p - 1) * (q - 1)
