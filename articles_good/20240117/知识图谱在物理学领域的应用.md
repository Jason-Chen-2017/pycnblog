以下是一个简单的代码实例，用于表示和管理物理实验中的实体关系和实体属性：

```python
# 定义实体类
class Entity:
    def __init__(self, name, properties):
        self.name = name
        self.properties = properties

# 定义实体关系类
class Relation:
    def __init__(self, source, target, property):
        self.source = source
        self.target = target
        self.property = property

# 定义实体属性类
class Property:
    def __init__(self, name, value):
        self.name = name
        self.value = value

# 创建实体
entity1 = Entity("实验设备1", [Property("类型", "电磁波"), Property("能量", "100W")])
entity2 = Entity("实验条件1", [Property("温度", "298K"), Property("压力", "1atm")])

# 创建实体关系
relation1 = Relation(entity1, entity2, "能量")

# 打印实体关系
print(relation1.source.name, relation1.target.name, relation1.property.name)
```

在这个代码实例中，我们首先定义了实体类、实体关系类和实体属性类。然后我们创建了两个实体，分别表示实验设备和实验条件。接下来我们创建了一个实体关系，表示实验设备和实验条件之间的能量关系。最后我们打印了实体关系的源实体、目标实体和关系属性。
