## 1.2 知识图谱技术的基本概念

知识图谱（Knowledge Graph）是一种以实体（Entity）和关系（Relation）为基本单位的图形数据库，它可以表示实体之间的关系网络。知识图谱可以帮助计算机理解自然语言文本，从而实现自然语言处理、问答系统、推荐系统等应用。知识图谱技术的核心是实体识别、关系抽取、实体链接等，它们可以帮助计算机理解自然语言文本中的信息，并将信息转换为结构化的知识。

### 4.1 实体识别与链接的代码实例

实体识别与链接的代码实例如下：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 训练数据
train_data = [
    "智能空调",
    "智能灯泡",
    "智能门锁",
    "智能浴室",
    "智能窗帘"
]

# 测试数据
test_data = [
    "开启空调",
    "关闭灯泡",
    "锁门",
    "调整浴室温度",
    "打开窗帘"
]

# 词向量化
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(train_data)

# 计算相似度
similarity = cosine_similarity(X, vectorizer.transform(test_data))

# 输出结果
for i, query in enumerate(test_data):
    print(f"查询：{query}")
    print(f"相似度：{similarity[i]}")
    print(f"匹配结果：{train_data[similarity[i].argmax()]}")
    print()
```
