```python
import hashlib

class UserAuthentication:
    def __init__(self, username, password):
        self.username = username
        self.password = password

    def authenticate(self):
        password_hash = hashlib.sha256(self.password.encode()).hexdigest()
        return password_hash == self.password

user_authentication = UserAuthentication('admin', 'password123')
print(user_authentication.authenticate())
```

在这个例子中，我们创建了一个`UserAuthentication`类，它包含一个`authenticate`方法。这个方法使用了Python的`hashlib`模块来实现用户身份验证功能。它会将用户输入的密码进行SHA256哈希处理，并将其与存储在`UserAuthentication`对象中的密码进行比较。如果两者相等，则返回`True`，表示用户身份验证成功。
