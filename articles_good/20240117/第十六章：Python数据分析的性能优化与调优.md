2. 链表：链表是一种线性数据结构，每个元素包含一个指针，指向下一个元素。链表的时间复杂度为O(n)，空间复杂度为O(n)。

## 4.2 使用并行和分布式计算

以下是一个使用多线程的例子：

```python
import threading

# 使用多线程进行并行计算
def thread_compute(num):
    print(f"线程{num}开始计算")
    result = num * num
    print(f"线程{num}计算完成，结果为{result}")

threads = []
for i in range(5):
    t = threading.Thread(target=thread_compute, args=(i,))
    threads.append(t)
    t.start()

for t in threads:
    t.join()
```
