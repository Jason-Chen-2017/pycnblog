# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的例子来说明数据清洗的具体操作。假设我们有一个包含学生成绩的数据集，数据集中包含学生的姓名、年龄、成绩等信息。我们需要对这个数据集进行清洗，以提高数据质量。

```python
import pandas as pd

# 创建数据集
data = {'name': ['张三', '李四', '王五', '赵六'],
        'age': [20, 21, 22, 23],
        'score': [80, 90, 85, 75]}

# 创建DataFrame
df = pd.DataFrame(data)

# 数据校验
def check_age(age):
    return 0 <= age <= 100

df['age'] = df['age'].apply(check_age)

# 数据纠正
def correct_score(score):
    if score > 100:
        return 100
    elif score < 0:
        return 0
    else:
        return score

df['score'] = df['score'].apply(correct_score)

# 数据填充
def fill_missing_age(row):
    if pd.isnull(row['age']):
        return 20
    else:
        return row['age']

df['age'] = df['age'].fillna(fill_missing_age)

# 数据转换
def convert_age_to_years(age):
    return age / 10

df['age_years'] = df['age'].apply(convert_age_to_years)

# 数据整合
def integrate_data(df):
    return df

df_integrated = integrate_data(df)

# 打印结果
print(df_integrated)
```

在上述代码中，我们首先创建了一个包含学生成绩的数据集。然后，我们对数据集进行了数据校验、数据纠正、数据填充、数据转换和数据整合等操作。最后，我们打印了整合后的数据集。
