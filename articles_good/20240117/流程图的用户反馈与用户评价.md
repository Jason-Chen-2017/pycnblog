在流程图中，我们通常使用一些基本的数学模型公式来表示算法和程序的各种结构。这些基本数学模型公式包括：

- 顺序执行：$$ P_1;P_2;P_3;...;P_n $$

## 1.4 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明流程图的使用方法。

代码实例：

```python
# 计算100到200之间偶数的和
sum = 0
for i in range(100, 201):
    if i % 2 == 0:
        sum += i
print(sum)
```

