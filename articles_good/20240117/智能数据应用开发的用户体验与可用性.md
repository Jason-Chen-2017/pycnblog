## 3.1 线性回归

线性回归是一种用于预测连续变量的算法。它假设数据之间存在线性关系，通过最小二乘法，找到最佳的直线或平面来拟合数据。

数学模型公式：$$ y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon $$

## 3.2 逻辑回归

逻辑回归是一种用于预测离散变量的算法。它假设数据之间存在线性关系，通过最大似然估计，找到最佳的直线或平面来分类数据。

数学模型公式：$$ P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n)}} $$

## 3.3 支持向量机

支持向量机是一种用于分类和回归的算法。它通过找到最大间隔的超平面，将数据分为不同的类别。

数学模型公式：$$ f(x) = \text{sgn}(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n) $$

## 3.4 决策树

决策树是一种用于分类和回归的算法。它通过递归地构建树状结构，将数据分为不同的类别。

数学模型公式：$$ \text{if } x_1 \leq t_1 \text{ then } f(x) = L_1 \text{ else } f(x) = L_2 $$

## 3.5 随机森林

随机森林是一种用于分类和回归的算法。它通过构建多个决策树，并通过投票的方式，得到最终的预测结果。

数学模型公式：$$ f(x) = \text{mode}(\text{predict}(T_1(x)), \text{predict}(T_2(x)), \cdots, \text{predict}(T_n(x))) $$

## 3.6 神经网络

神经网络是一种用于分类和回归的算法。它通过构建多层的神经元网络，并通过反向传播算法，得到最佳的权重和偏置。

数学模型公式：$$ y = \sigma(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + b) $$

## 4.1 线性回归

线性回归的Python代码实例：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 生成数据
X = np.random.rand(100, 1)
y = 2 * X + 1 + np.random.randn(100, 1)

# 训练模型
model = LinearRegression()
model.fit(X, y)

# 预测
X_new = np.array([[0.5]])
y_pred = model.predict(X_new)
```
