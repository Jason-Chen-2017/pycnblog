## 4.1 加密算法实例

### 4.1.1 RSA算法实例

RSA算法是一种公开密钥加密算法，它使用两个大素数生成密钥对（公钥、私钥），公钥用于加密，私钥用于解密。

```python
import random
import math

def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a

def rsa_key_gen(p, q):
    n = p * q
    phi = (p - 1) * (q - 1)
    e = random.randint(1, phi)
    g = gcd(e, phi)
    while g != 1:
        e = random.randint(1, phi)
        g = gcd(e, phi)
    d = pow(e, phi - 1, phi)
    return (n, e, d)

def rsa_encrypt(n, e, m):
    c = pow(m, e, n)
    return c

def rsa_decrypt(n, d, c):
    m = pow(c, d, n)
    return m

p = random.randint(2, 10)
q = random.randint(2, 10)
if p != q:
    n, e, d = rsa_key_gen(p, q)
    m = random.randint(1, n)
    c = rsa_encrypt(n, e, m)
    m = rsa_decrypt(n, d, c)
    print("n =", n)
    print("e =", e)
    print("d =", d)
    print("m =", m)
    print("c =", c)
```

### 4.1.2 AES算法实例

AES算法是一种对称加密算法，它使用固定密钥加密和解密数据。

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad

key = get_random_bytes(16)
cipher = AES.new(key, AES.MODE_ECB)

plaintext = b"Hello, World!"
ciphertext = cipher.encrypt(pad(plaintext, AES.block_size))
print("ciphertext =", ciphertext)

decrypted = unpad(cipher.decrypt(ciphertext), AES.block_size)
print("decrypted =", decrypted)
```

## 4.2 签名算法实例

### 4.2.1 RSA签名算法实例

RSA签名算法是一种公开密钥签名算法，它使用私钥生成签名，使用公钥验证签名。

```python
from Crypto.PublicKey import RSA
from Crypto.Signature import pkcs1_15
from Crypto.Hash import SHA256

key = RSA.generate(2048)
signer = pkcs1_15.new(key)
hasher = SHA256.new(b"Hello, World!")

signature = signer.sign(hasher)
print("signature =", signature)

verifier = pkcs1_15.new(key)
hasher = SHA256.new(b"Hello, World!")
verified = verifier.verify(hasher, signature)
print("verified =", verified)
```

### 4.2.2 ECDSA签名算法实例

ECDSA签名算法是一种椭圆曲线密钥签名算法，它使用私钥生成签名，使用公钥验证签名。

```python
from Crypto.PublicKey import ECC
from Crypto.Signature import DSS
from Crypto.Hash import SHA256

key = ECC.generate(curve="P-256")
signer = DSS.new(key)
hasher = SHA256.new(b"Hello, World!")

signature = signer.sign(hasher)
print("signature =", signature)

verifier = DSS.new(key)
hasher = SHA256.new(b"Hello, World!")
verified = verifier.verify(hasher, signature)
print("verified =", verified)
```

## 4.4 风险控制算法实例

### 4.4.1 贝叶斯定理实例

贝叶斯定理是一种概率推理方法，它可以用于评估事件发生的概率。

```python
import math

P(A) = P(A|B) * P(B) / P(B|A)

P(A) = 0.8
P(B) = 0.6
P(A|B) = 0.9
P(B|A) = 0.7

P(A|B) = P(A) * P(B|A) / P(B)
P(A|B) = 0.8 * 0.7 / 0.6
P(A|B) = 0.8
```
