# 4.具体代码实例和详细解释说明

在这里，我们将提供一个具体的代码实例和详细解释说明。

```python
from flask import Flask, request, Response
from consul import Consul

app = Flask(__name__)
consul = Consul()

@app.route('/service')
def service():
    service_name = request.args.get('service_name')
    service_id = consul.catalog.service(service_name)[0].ID
    response = Response(f'Service ID: {service_id}')
    return response

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
```

在这个代码实例中，我们使用了Flask和Consul库来实现一个简单的服务发现示例。Flask是一个Python Web框架，Consul是一个开源的服务发现和配置库。

首先，我们创建了一个Flask应用，并使用Consul库初始化一个Consul客户端。然后，我们创建了一个/service路由，该路由接收一个服务名称作为参数。在该路由中，我们使用Consul客户端的catalog.service方法获取服务ID，并将其返回给客户端。

最后，我们使用if __name__ == '__main__':语句启动Flask应用，并指定主机和端口。
