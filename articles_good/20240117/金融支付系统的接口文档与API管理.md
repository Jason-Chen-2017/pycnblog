   ```python

   key = RSA.generate(2048)
   # 生成签名对象
   signer = pkcs1_15.new(key)
   # 生成消息摘要
   digest = SHA256.new("Hello, World!")
   # 签名
   signature = signer.sign(digest)
   # 验证签名
   verifier = pkcs1_15.new(key)
   verifier.verify(digest, signature)
   ```
