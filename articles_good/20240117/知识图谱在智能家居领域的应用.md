# 4. 具体代码实例和详细解释说明

## 4.1 代码实例

以下是一个简单的知识图谱在智能家居领域的应用示例：

```python
from knowledge_graph import KnowledgeGraph

# 创建知识图谱
kg = KnowledgeGraph()

# 添加实体
kg.add_entity('家居设备', '设备类型', '智能插座')
kg.add_entity('家庭成员', '姓名', '张三')
kg.add_entity('家庭成员', '年龄', 25)

# 添加关系
kg.add_relation('家居设备', '使用者', '家庭成员')
kg.add_relation('家庭成员', '使用设备', '家居设备')

# 查询结果
result = kg.query('家居设备', '使用者')
print(result)
```

## 4.2 详细解释说明

1. 首先，我们导入了`knowledge_graph`模块，并创建了一个知识图谱对象`kg`。
2. 然后，我们使用`kg.add_entity()`方法添加了两个实体：`家居设备`和`家庭成员`。
3. 接下来，我们使用`kg.add_relation()`方法添加了两个关系：`家居设备`的使用者是`家庭成员`，`家庭成员`使用的设备是`家居设备`。
4. 最后，我们使用`kg.query()`方法查询`家居设备`的使用者，并打印了查询结果。
