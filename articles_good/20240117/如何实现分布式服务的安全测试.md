# 4.具体代码实例和详细解释说明

```bash
nmap -sV -sC -p- -A 192.168.1.1
```

在这个命令中，-sV表示进行版本扫描，-sC表示使用默认扫描策略，-p-表示扫描所有端口，-A表示使用所有扫描技术。
