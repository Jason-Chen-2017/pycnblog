```python

    for i in range(num):
        print(i)

t2 = threading.Thread(target=print_num, args=(10,))

t1.start()
t2.start()

t1.join()
t2.join()
```

```python
# 发送数据
s.send("Hello, World!".encode())

# 接收数据
data = s.recv(1024).decode()

# 关闭套接字
s.close()

print(data)
```

