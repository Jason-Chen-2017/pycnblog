# 4.代码实例

在本节中，我们将通过一个简单的自动驾驶系统的实例来详细讲解自动驾驶系统的具体操作步骤。

```python
import cv2
import numpy as np
import rospy
from sensor_msgs.msg import Image
from cv_bridge import CvBridge

class AutonomousDrivingSystem:
    def __init__(self):
        self.bridge = CvBridge()
        self.image_sub = rospy.Subscriber('/camera/image_raw', Image, self.image_callback)
        self.pub = rospy.Publisher('/car/steering_angle', float, queue_size=10)

    def image_callback(self, data):
        try:
            cv_image = self.bridge.imgmsg_to_cv2(data, desired_encoding='bgr8')
            gray_image = cv2.cvtColor(cv_image, cv2.COLOR_BGR2GRAY)
            edges = cv2.Canny(gray_image, 50, 150)
            lines = cv2.HoughLines(edges, 1, np.pi / 180, 200)
            steering_angle = self.calculate_steering_angle(lines)
            self.pub.publish(steering_angle)
        except rospy.ROSInterruptException:
            pass

    def calculate_steering_angle(self, lines):
        if lines is None or len(lines) == 0:
            return 0.0
        else:
            rho = lines[0][0][0]
            angle = np.arctan(rho[1] / rho[0])
            return angle

if __name__ == '__main__':
    rospy.init_node('autonomous_driving_system', anonymous=True)
    autonomous_driving_system = AutonomousDrivingSystem()
    rospy.spin()
```

在这个实例中，我们首先导入了必要的库，然后定义了一个自动驾驶系统类。在类的初始化方法中，我们初始化了一些变量，并订阅了摄像头的图像数据。在订阅的回调方法中，我们将图像数据转换为灰度图，然后使用可ny盾边缘检测算法检测边缘，接着使用傅里叶变换检测线，最后根据线的倾角计算转向角度，并发布转向角度。
