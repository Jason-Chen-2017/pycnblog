4.1 聚类算法实例

```python
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt
import numpy as np

# 生成随机数据
X = np.random.rand(100, 2)

# 使用KMeans算法进行聚类
kmeans = KMeans(n_clusters=3, random_state=0).fit(X)

# 绘制聚类结果
plt.scatter(X[:, 0], X[:, 1], c=kmeans.labels_, cmap='rainbow')
plt.scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1], s=200, c='black')
plt.show()
```

在这个代码实例中，我们使用了KMeans算法进行聚类。首先，我们生成了一组随机数据，然后使用KMeans算法对数据进行聚类。最后，我们绘制了聚类结果，可以看到数据被分为了三个不同的类别。

4.2 分类算法实例

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# 加载鸢尾花数据集
iris = load_iris()
X = iris.data
y = iris.target

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)

# 使用随机森林算法进行分类
clf = RandomForestClassifier(n_estimators=100, random_state=0).fit(X_train, y_train)

# 预测测试集结果
y_pred = clf.predict(X_test)

# 计算分类准确率
accuracy = accuracy_score(y_test, y_pred)
print('分类准确率：', accuracy)
```

在这个代码实例中，我们使用了随机森林算法进行分类。首先，我们加载了鸢尾花数据集，然后将数据集划分为训练集和测试集。接着，我们使用随机森林算法对训练集进行分类，并预测测试集结果。最后，我们计算分类准确率。

4.3 回归算法实例

```python
from sklearn.datasets import load_boston
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestRegressor
from sklearn.metrics import mean_squared_error

# 加载波士顿房价数据集

X = boston.data
y = boston.target

# 划分训练集和测试集

# 使用随机森林算法进行回归

# 预测测试集结果

# 计算回归误差
```

在这个代码实例中，我们使用了随机森林算法进行回归。首先，我们加载了波士顿房价数据集，然后将数据集划分为训练集和测试集。接着，我们使用随机森林算法对训练集进行回归，并预测测试集结果。最后，我们计算回归误差。
