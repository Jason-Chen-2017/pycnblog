### 2.1线性空间

线性空间的一个重要特点是，它可以用来表示线性关系。线性关系是一种关系，它可以用一组线性方程来描述。例如，在二维空间中，线性关系可以用一条直线来描述。

### 2.2多项式回归

多项式回归是一种预测方法，可以用来预测一个变量的值，根据其他变量的值。多项式回归是一种特殊类型的回归分析，它使用多项式函数来模拟数据。多项式函数是一种数学函数，它可以用多个变量来表示。

多项式回归的一个重要特点是，它可以用来拟合非线性关系。例如，在三维空间中，多项式回归可以用一组曲线来描述非线性关系。

### 2.3联系

线性空间和多项式回归之间的联系在于它们都可以用来描述关系。线性空间可以用来描述线性关系，而多项式回归可以用来描述非线性关系。此外，多项式回归可以用线性空间的概念来解释。例如，多项式回归可以用来拟合一组数据点，这些数据点可以被看作是线性空间中的向量。

import numpy as np
from sklearn.linear_model import LinearRegression

# 定义多项式函数
def polynomial_function(x, a_0, a_1, a_2):
    return a_0 + a_1 * x + a_2 * x**2

# 拟合数据
def fit_data(x, y, a_0, a_1, a_2):
    m = len(x)
    sum_ = 0
    for i in range(m):
        sum_ += (y[i] - polynomial_function(x[i], a_0, a_1, a_2))**2
    a_0, a_1, a_2 = np.linalg.solve(np.array([[n**2, n, 1] for n in range(m)]), np.array([sum_ for _ in range(m)]))
    return a_0, a_1, a_2

# 预测值
def predict_value(x, a_0, a_1, a_2):
    return polynomial_function(x, a_0, a_1, a_2)

# 评估模型
def evaluate_model(x, y, a_0, a_1, a_2):
    m = len(x)
    sum_ = 0
    for i in range(m):
        sum_ += (predict_value(x[i], a_0, a_1, a_2) - y[i])**2
    return np.sqrt(sum_ / m)

# 主函数
if __name__ == "__main__":
    # 生成数据
    x = np.array([1, 2, 3, 4, 5])
    y = np.array([2, 4, 6, 8, 10])

    # 拟合数据
    a_0, a_1, a_2 = fit_data(x, y, 0, 0, 0)

    print("a_1 =", a_1)
