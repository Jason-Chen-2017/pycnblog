## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

在本文中，我们采用元胞自动机模型来模拟复杂系统中的涌现现象。元胞自动机模型由三个基本部分组成：元胞、状态和规则。

1. **元胞**：元胞是离散的网格点，每个元胞都有有限的状态，通常用0和1表示。

2. **状态**：元胞的状态表示其在某一时间步的取值。

3. **规则**：规则决定了每个元胞在下一个时间步的状态，通常是基于局部信息（如邻近元胞的状态）进行计算。

元胞自动机模型可以表示为以下数学公式：

$$
S(t+1) = F(S(t))
$$

其中，$S(t)$ 表示在时间步$t$的元胞状态，$F$ 表示状态转换规则。

### 4.2 公式推导过程

在推导元胞自动机的状态转换规则时，我们需要考虑以下几个方面：

1. **局部信息**：每个元胞的状态取决于其邻近元胞的状态。

2. **状态转换条件**：根据邻近元胞的状态，确定当前元胞在下一个时间步的状态。

3. **状态空间**：元胞的状态空间为$[0, 1]$，表示为实数。

基于上述考虑，我们可以推导出以下状态转换规则：

$$
S(t+1) = \sum_{i=1}^{n} w_i \cdot S(t-i)
$$

其中，$w_i$ 表示第$i$个邻近元胞对当前元胞状态的权重，$n$ 表示邻近元胞的数量。

class CellularAutomaton:
    def __init__(self, size, rules):
        self.size = size
        self.rules = rules
        self.grid = np.zeros((size, size), dtype=int)
    
    def update(self):
        new_grid = np.zeros((self.size, self.size), dtype=int)
        for i in range(self.size):
            for j in range(self.size):
                neighbors = self.get_neighbors(i, j)
                new_grid[i, j] = self.rules[neighbors]
        self.grid = new_grid
    
    def get_neighbors(self, i, j):
        neighbors = []
        for x in [-1, 0, 1]:
            for y in [-1, 0, 1]:
                if x == 0 and y == 0:

                ni, nj = i + x, j + y
                if 0 <= ni < self.size and 0 <= nj < self.size:

        return neighbors
    
    def run(self, steps):
