### 4.2 公式推导过程

以基于节点度的排名模型为例，介绍其公式推导过程。

设网络中节点集合为V，节点i的度数为di，影响力值f(i)与度数di成正比，即：

$$ f(i) = k \cdot di $$

其中，k为常数。

为了消除节点度数的影响，可以对影响力值进行归一化处理，得到：

$$ f'(i) = \frac{f(i)}{\sum_{j \in V} f(j)} = \frac{k \cdot di}{\sum_{j \in V} k \cdot dj} $$

因此，节点i的影响力值归一化后为：

$$ f'(i) = \frac{di}{\sum_{j \in V} dj} $$

# 影响力计算
def influence_computation(graph, k):
    influence_values = []
    for node in graph.nodes():
        influence_values.append(k * graph.degree(node))
    influence_values /= sum(influence_values)
    return influence_values

# 主函数
def main():
    # 构建网络图
    graph = nx.Graph()

    # 数据收集与预处理
    data = pd.read_csv('data.csv')
    data = data_preprocessing(data)

    # 添加节点与边
    graph.add_nodes_from(data['node'])
    graph.add_edges_from(data[['node', 'neighbor']])

    # 计算影响力值
    k = 0.1  # 常数
    influence_values = influence_computation(graph, k)

    # 影响力排名
    ranked_influences = sorted(zip(influence_values, graph.nodes()), reverse=True)

    # 联系与沟通
