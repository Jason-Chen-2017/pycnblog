### 5.2 源代码详细实现

下面是数字身份认证的实现代码。

```python
from Cryptodome.PublicKey import RSA
from Cryptodome.Signature import pkcs1_15
from Cryptodome.Hash import SHA256

def generate_keys():
    key = RSA.generate(2048)
    private_key = key.export_key()
    public_key = key.publickey().export_key()
    return private_key, public_key

def encrypt_message(public_key, message):
    rsakey = RSA.import_key(public_key)
    encrypted_text = rsakey.encrypt(message, 32)
    return encrypted_text

def sign_message(private_key, message):
    rsakey = RSA.import_key(private_key)
    h = SHA256.new(message)
    signature = pkcs1_15.new(rsakey).sign(h)
    return signature

def verify_signature(public_key, message, signature):
    rsakey = RSA.import_key(public_key)
    h = SHA256.new(message)
    try:
        pkcs1_15.new(rsakey).verify(h, signature)
        return True
    except (ValueError, TypeError):
        return False

# 生成公钥和私钥
private_key, public_key = generate_keys()

# 加密消息
message = "My ID is 123456"
encrypted_message = encrypt_message(public_key, message)

# 签名消息
signature = sign_message(private_key, message)

# 验证签名
is_valid = verify_signature(public_key, message, signature)
print(f"Message is valid: {is_valid}")
```

### 5.3 代码解读与分析

- `generate_keys()`：生成RSA密钥对。
- `encrypt_message()`：使用公钥加密消息。
- `sign_message()`：使用私钥对消息进行数字签名。
- `verify_signature()`：使用公钥验证签名。
