### 4.2 公式推导过程

1. **多层感知机（MLP）模型**：

   - 输入层：$x_1, x_2, ..., x_n$
   - 隐藏层：$h_1, h_2, ..., h_m$
   - 输出层：$y_1, y_2, ..., y_k$

   - 输出计算：

     $$y_i = \sigma(\sum_{j=1}^{m} w_{ji}h_j)$$

   - 权重更新：

     $$w_{ji} = w_{ji} - \alpha \cdot \delta y_i \cdot h_j$$

   - 激活函数：

     $$\sigma(x) = \frac{1}{1 + e^{-x}}$$

2. **神经活动模拟**：

   - 静息电位：

     $$V_r = -65 \text{mV}$$

   - 静息电阻：

     $$R_r = 150 \text{k}\Omega$$

   - 突触后电位（PSP）：

     $$V_p = V_r + I_p/R_r$$

   - 突触前电位（GAP）：

     $$V_g = V_r - I_g/R_r$$

### 5.2 源代码详细实现

以下是一个简单的多层感知机（MLP）实现：

```python
import tensorflow as tf
import numpy as np

# 设置随机种子
tf.random.set_seed(42)

# 设置超参数
input_size = 3
hidden_size = 5
output_size = 2
learning_rate = 0.1
epochs = 1000

# 初始化权重
w_ij = tf.random.normal([hidden_size, input_size])
w_ji = tf.random.normal([output_size, hidden_size])

# 初始化激活函数
sigma = tf.sigmoid

# 训练数据
x = np.array([[1, 0, 1], [0, 1, 0], [1, 1, 0]])
y = np.array([[1, 0], [0, 1], [1, 0]])

# 训练过程
for epoch in range(epochs):
    # 前向传播
    h = sigma(tf.matmul(w_ij, x))
    y_pred = sigma(tf.matmul(w_ji, h))

    # 计算损失函数
    loss = tf.reduce_mean(tf.square(y - y_pred))

    # 反向传播
    with tf.GradientTape() as tape:
        loss = tf.reduce_mean(tf.square(y - y_pred))

    grads = tape.gradient(loss, [w_ij, w_ji])
    w_ij -= learning_rate * grads[0]
    w_ji -= learning_rate * grads[1]

# 输出结果
print("Final weights:")
print(w_ij.numpy())
print(w_ji.numpy())
```
