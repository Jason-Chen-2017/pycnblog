## 4.1 简单游戏循环实例

以下是一个简单的游戏循环实例，它包括初始化、更新和绘制三个阶段：

```python
import pygame

# 初始化游戏
pygame.init()
screen = pygame.display.set_mode((800, 600))
clock = pygame.time.Clock()

# 游戏循环
running = True
while running:
    # 更新游戏
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    # 绘制游戏
    screen.fill((0, 0, 0))
    pygame.display.flip()

    # 更新游戏时间
    clock.tick(60)

# 结束游戏
pygame.quit()
```

在上述代码中，我们首先导入了pygame库，然后进行游戏的初始化，包括设置屏幕大小和创建时钟。接着，我们进入游戏循环，在循环中，我们首先更新游戏的事件，如关闭窗口等。然后，我们绘制游戏界面，将屏幕填充为黑色。最后，我们更新游戏时间，并检查是否需要退出游戏。当游戏结束后，我们关闭游戏。

## 4.2 简单事件处理实例

以下是一个简单的事件处理实例，它包括玩家输入处理和游戏物体碰撞检测：

```python
import pygame

# 初始化游戏
pygame.init()
screen = pygame.display.set_mode((800, 600))
clock = pygame.time.Clock()

# 创建游戏物体
player = pygame.Rect(400, 300, 50, 50)

# 游戏循环
running = True
while running:
    # 更新游戏
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
        elif event.type == pygame.KEYDOWN:
            if event.key == pygame.K_UP:
                player.move_ip(0, -5)
            elif event.key == pygame.K_DOWN:
                player.move_ip(0, 5)
            elif event.key == pygame.K_LEFT:
                player.move_ip(-5, 0)
            elif event.key == pygame.K_RIGHT:
                player.move_ip(5, 0)

    # 碰撞检测
    if player.colliderect(pygame.Rect(700, 500, 50, 50)):
        player.x = 400
        player.y = 300

    # 绘制游戏
    screen.fill((0, 0, 0))
    pygame.draw.rect(screen, (255, 0, 0), player)
    pygame.display.flip()

    # 更新游戏时间
    clock.tick(60)

# 结束游戏
pygame.quit()
```

在上述代码中，我们首先导入了pygame库，然后进行游戏的初始化，包括设置屏幕大小和创建时钟。接着，我们创建了一个游戏物体，它是一个矩形。在游戏循环中，我们首先更新游戏的事件，如关闭窗口和玩家输入等。然后，我们进行游戏物体的碰撞检测，如果游戏物体与另一个游戏物体发生碰撞，我们将游戏物体的位置重置为初始位置。最后，我们绘制游戏界面，将屏幕填充为黑色，并将游戏物体绘制为红色矩形。当游戏结束后，我们关闭游戏。
