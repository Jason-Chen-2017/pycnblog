## 3.1 梯度下降
梯度下降是一种优化算法，用于最小化一个函数。在机器学习中，我们经常需要最小化损失函数，以找到最佳的模型参数。梯度下降算法的基本思想是通过迭代地更新参数，使得损失函数逐步减小。

梯度下降算法的步骤如下：

1. 初始化模型参数。
2. 计算损失函数的梯度。
3. 更新模型参数。
4. 重复步骤2和3，直到收敛。

数学模型公式为：

$$
\theta = \theta - \alpha \nabla J(\theta)
$$

其中，$\theta$ 是模型参数，$J(\theta)$ 是损失函数，$\alpha$ 是学习率，$\nabla J(\theta)$ 是损失函数的梯度。

## 3.2 支持向量机
支持向量机（SVM）是一种二分类算法，它通过找到一个最佳的超平面来将数据分为两个类别。SVM 通过最大化边界条件的边际来优化模型参数。

SVM 的核心步骤如下：

1. 将数据映射到高维特征空间。
2. 找到支持向量。
3. 计算支持向量之间的间距。
4. 优化超平面。

数学模型公式为：

$$
\min_{\omega, b} \frac{1}{2}\|\omega\|^2 \\
s.t. \quad y_i(\omega^T x_i + b) \geq 1, \quad i=1,2,...,N
$$

其中，$\omega$ 是超平面的法向量，$b$ 是超平面的偏移量，$x_i$ 是输入向量，$y_i$ 是标签。

## 3.3 随机森林
随机森林是一种集成学习方法，它通过组合多个决策树来构建模型。随机森林的核心思想是通过多个不同的决策树来捕捉数据的不同方面，从而提高模型的准确性。

随机森林的核心步骤如下：

1. 生成多个决策树。
2. 对每个决策树进行训练。
3. 对输入数据进行预测。
4. 通过多个决策树的预测结果进行平均。

数学模型公式为：

$$
\hat{y} = \frac{1}{K} \sum_{k=1}^K f_k(x)
$$

其中，$\hat{y}$ 是预测结果，$K$ 是决策树的数量，$f_k(x)$ 是第$k$个决策树的预测结果。

## 4.1 梯度下降实例
在这个例子中，我们将实现一元梯度下降算法，用于最小化一元线性回归模型的损失函数。

```python
import numpy as np

def gradient_descent(X, y, learning_rate=0.01, num_iterations=1000):
    m, n = X.shape
    theta = np.zeros(n)
    y_pred = np.zeros(m)

    for i in range(num_iterations):
        y_pred = X @ theta
        errors = y - y_pred
        theta -= learning_rate * (X.T @ errors) / m

    return theta
```

## 4.2 支持向量机实例
在这个例子中，我们将实现一个简单的支持向量机算法，用于分类二元类别数据。

```python
import numpy as np
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC

# 生成二元类别数据
X, y = make_classification(n_samples=100, n_features=2, random_state=42)

# 数据预处理
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 训练测试分割
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# 训练支持向量机
svm = SVC(kernel='linear')
svm.fit(X_train, y_train)

# 预测
y_pred = svm.predict(X_test)

# 评估
accuracy = np.mean(y_pred == y_test)
print(f'Accuracy: {accuracy:.4f}')
```
