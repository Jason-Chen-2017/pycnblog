#### 算法编程题 1：给定一个字符串，找出其中最长回文子串。

**题目：** 编写一个函数，找出给定字符串中最长的回文子串。

**答案：** 可以使用动态规划的方法解决这个问题。以下是 Python 代码示例：

```python
def longest_palindrome(s: str) -> str:
    n = len(s)
    if n < 2:
        return s
    
    start = 0
    max_len = 1
    
    dp = [[False] * n for _ in range(n)]
    
    for i in range(n):
        dp[i][i] = True
    
    for i in range(n - 1, -1, -1):
        for j in range(i + 1, n):
            if s[i] == s[j]:
                if j - i == 0 or dp[i + 1][j - 1]:
                    dp[i][j] = True
                    if max_len < j - i + 1:
                        max_len = j - i + 1
                        start = i
            
            else:
                dp[i][j] = False
                
    return s[start:start + max_len]
```

#### 算法编程题 2：给定一个整数数组，找出所有三个数的组合，使得它们的和等于零。

**题目：** 编写一个函数，找出给定整数数组中所有三个数的组合，使得它们的和等于零。

**答案：** 可以使用排序和双指针的方法解决这个问题。以下是 Python 代码示例：

```python
def three_sum(nums: List[int]) -> List[List[int]]:
    nums.sort()
    ans = []

    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i - 1]:
            continue

        left, right = i + 1, len(nums) - 1

        while left < right:
            total = nums[i] + nums[left] + nums[right]

            if total == 0:
                ans.append([nums[i], nums[left], nums[right]])
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1

                left += 1
                right -= 1
            elif total < 0:
                left += 1
            else:
                right -= 1

    return ans
```

def subarray_sum(nums: List[int], target: int) -> List[List[int]]:
    prefix_sum = {0: -1}
    curr_sum = 0
    ans = []

    for i, num in enumerate(nums):

**答案：** 可以使用垂直扫描的方法解决这个问题。以下是 Python 代码示例：

```python
def longest_common_prefix(strs: List[str]) -> str:
    if not strs:
        return ""

    prefix = ""

    for i in range(len(strs[0])):
        char = strs[0][i]
        for s in strs[1:]:
            if i >= len(s) or s[i] != char:
                return prefix
        prefix += char

    return prefix
```

**答案：** 可以使用双指针的方法解决这个问题。以下是 Python 代码示例：

```python
def two_sum_closest(nums: List[int], target: int) -> int:
    nums.sort()
    left, right = 0, len(nums) - 1

    while left < right:
        total = nums[left] + nums[right]
        if total == target:
            return left
        elif total < target:
            left += 1
        else:
            right -= 1

    return min(left, right)
```
