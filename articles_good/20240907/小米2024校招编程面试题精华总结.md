#### 1. 斐波那契数列

**题目：** 实现 Fibonacci 数列。

**答案：** 可以使用递归或循环的方式实现。

**递归实现：**

```go
func Fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    return Fibonacci(n-1) + Fibonacci(n-2)
}
```

**循环实现：**

```go
func Fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    a, b := 0, 1
    for i := 2; i <= n; i++ {
        a, b = b, a+b
    }
    return b
}
```

**解析：** 斐波那契数列是指每一项都等于前两项之和。递归实现简单直观，但性能较差。循环实现时间复杂度为 O(n)，性能更优。

#### 2. 最长公共子序列

**题目：** 给定两个字符串，找出它们的最长公共子序列。

**答案：** 可以使用动态规划实现。

```go
func LongestCommonSubsequence(s1 string, s2 string) string {
    m, n := len(s1), len(s2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if s1[i-1] == s2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    // 恢复最长公共子序列
    var result []byte
    i, j := m, n
    for i > 0 && j > 0 {
        if s1[i-1] == s2[j-1] {
            result = append(result, s1[i-1])
            i--
            j--
        } else if dp[i-1][j] > dp[i][j-1] {
            i--
        } else {
            j--
        }
    }
    return string(result)
}
```

**解析：** 使用二维数组 `dp` 记录状态，`dp[i][j]` 表示 `s1` 的前 `i` 个字符和 `s2` 的前 `j` 个字符的最长公共子序列长度。最后通过回溯恢复最长公共子序列。

#### 3. 二分查找

**题目：** 给定一个有序数组，找出目标值。

**答案：** 使用二分查找算法实现。

```go
func binarySearch(nums []int, target int) int {
    left, right := 0, len(nums)-1
    for left <= right {
        mid := (left + right) / 2
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}
```

**解析：** 二分查找的时间复杂度为 O(logn)，是一种高效的查找算法。
