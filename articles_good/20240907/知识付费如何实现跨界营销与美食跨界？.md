#### 四、编程题解析

**6. 编写一个函数，实现将一个字符串中的数字转换为对应的大写金额表示。**

**题目：** 编写一个函数 `numberToChineseMoney(s string) string`，实现将一个字符串中的数字转换为对应的大写金额表示。

**答案：**

```go
package main

import (
    "fmt"
    "unicode"
)

func numberToChineseMoney(s string) string {
    // 数字与汉字的映射关系
    digits := []rune{"零", "一", "二", "三", "四", "五", "六", "七", "八", "九"}
    units := []string{"", "十", "百", "千", "万", "亿", "十亿", "百亿", "千亿", "万亿"}

    var result []rune
    i := 0
    for i < len(s) {
        // 跳过前导零
        if s[i] == '0' {
            i++
            continue
        }
        // 处理数字
        num := 0
        for i < len(s) && unicode.IsDigit(rune(s[i])) {
            digit := int(s[i] - '0')
            num = num*10 + digit
            i++
        }
        // 转换为大写金额表示
        unit := ""
        if num >= 10000 {
            unit = units[4]
            num /= 10000
        }
        if num >= 1000 {
            unit = units[3]
            num /= 1000
        }
        if num >= 100 {
            unit = units[2]
            num /= 100
        }
        if num >= 10 {
            unit = units[1]
            num /= 10
        }
        // 将数字与单位拼接起来
        result = append(result, digits[num]...)
        if unit != "" {
            result = append(result, []rune(unit)...)
        }
    }
    return string(result)
}

func main() {
    s := "123456789"
    fmt.Println(numberToChineseMoney(s)) // 一亿二千三百四十五万六千七百八十九
}
```

**解析：** 该函数通过遍历字符串中的数字，将其转换为对应的大写金额表示。注意处理数字的前导零和单位换算。

**7. 编写一个函数，实现将一个字符串中的大写金额表示转换为对应的数字。**

**题目：** 编写一个函数 `chineseMoneyToNumber(s string) float64`，实现将一个字符串中的大写金额表示转换为对应的数字。

**答案：**

```go
package main

import (
    "fmt"
    "regexp"
    "strings"
)

func chineseMoneyToNumber(s string) float64 {
    // 数字与汉字的映射关系
    digits := []string{"零", "一", "二", "三", "四", "五", "六", "七", "八", "九"}
    units := []string{"", "十", "百", "千", "万", "亿", "十亿", "百亿", "千亿", "万亿"}

    // 清除空格
    s = strings.TrimSpace(s)

    // 匹配大写金额表示
    matches := regexp.MustCompile(`零百|零千|零十`).FindAllString(s, -1)
    for _, match := range matches {
        s = strings.Replace(s, match, "", -1)
    }

    // 分解金额
    parts := strings.Split(s, "亿")
    var total float64
    for _, part := range parts {
        unitsMap := make(map[string]int)
        for i, unit := range units {
            unitsMap[unit] = i
        }
        num := 0
        lastDigit := 0
        for i := len(part) - 1; i >= 0; i-- {
            digit := strings.Index(digits, string(part[i]))
            if digit == -1 {
                continue
            }
            if lastDigit > 0 {
                num += digit * unitsMap[lastDigit]
            }
            lastDigit = unitsMap[part[i : i+1]]
        }
        total += float64(num) * math.Pow10(unitsMap[parts[0]])
    }
    return total
}

func main() {
    s := "一亿二千三百四十五万六千七百八十九"
    fmt.Println(chineseMoneyToNumber(s)) // 123456789
}
```

**解析：** 该函数通过正则表达式清除大写金额表示中的无效字符，然后分解金额并转换为对应的数字。注意处理零和单位的映射关系。
