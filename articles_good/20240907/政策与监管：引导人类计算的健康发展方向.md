```python
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad
from base64 import b64encode

def aes_encrypt(plaintext, key):
    cipher = AES.new(key, AES.MODE_CBC)
    ct_bytes = cipher.encrypt(pad(plaintext.encode('utf-8'), AES.block_size))
    iv = b64encode(cipher.iv).decode('utf-8')
    ct = b64encode(ct_bytes).decode('utf-8')
    return iv, ct
```

### 12. 机器学习算法

**题目：** 请简要介绍常见的机器学习算法，并讨论其优缺点。

**答案：** 常见的机器学习算法包括以下几种：

1. **监督学习算法**：
   - **线性回归**：通过拟合线性模型，预测连续值。
     - 优点：简单易懂，易于实现。
     - 缺点：预测能力有限，无法处理非线性问题。
   - **逻辑回归**：用于分类问题，输出概率。
     - 优点：易于理解和实现，计算效率高。
     - 缺点：对异常值敏感，无法处理非线性问题。
   - **支持向量机（SVM）**：通过找到最优分类边界，实现分类。
     - 优点：分类效果好，适用于小样本数据。
     - 缺点：计算复杂度高，难以处理大规模数据。
   - **决策树**：通过构建树形模型，实现分类或回归。
     - 优点：易于理解和解释，处理非线性问题。
     - 缺点：过拟合风险高，易受到数据噪声影响。
   - **随机森林**：通过构建多个决策树，进行集成学习。
     - 优点：分类效果较好，减少过拟合。
     - 缺点：计算复杂度高，难以处理大规模数据。

2. **无监督学习算法**：
   - **K-均值聚类**：将数据划分为 K 个聚类，寻找最优聚类中心。
     - 优点：简单易懂，易于实现。
     - 缺点：对初始聚类中心敏感，可能陷入局部最优。
   - **主成分分析（PCA）**：通过降维，减少数据维度，突出主要特征。
     - 优点：降低数据维度，突出主要特征。
     - 缺点：对异常值敏感，可能丢失部分信息。

3. **深度学习算法**：
   - **卷积神经网络（CNN）**：通过卷积层提取特征，实现图像识别等任务。
     - 优点：适用于图像、语音等任务，具有强大的特征提取能力。
     - 缺点：计算复杂度高，需要大量数据训练。
   - **循环神经网络（RNN）**：通过循环结构处理序列数据，实现语言建模等任务。
     - 优点：适用于序列数据，具有记忆能力。
     - 缺点：训练过程可能陷入梯度消失或爆炸问题。

**解析：** 不同的机器学习算法适用于不同的任务和数据类型，需要根据具体需求选择合适的算法。每种算法都有其优缺点，需要综合考虑算法性能、计算效率和可解释性等因素。
