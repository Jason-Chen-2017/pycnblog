#### 1. 社交网络的传播效应

**题目：** 在社交网络中，一个用户的影响范围如何计算？

**答案：** 社交网络中用户的影响范围可以通过计算其影响力扩散的深度和广度来确定。

**计算方法：**
- **深度（Reach Depth）：** 从用户出发，通过连接的社交网络节点，计算能够触及的最大深度。
- **广度（Reach Width）：** 用户在一次互动中能够触达到的不同用户的数量。

**示例：**

```go
// 示例代码：计算用户的社交网络影响范围
func calculateInfluence(graph map[int][]int, userId int) (int, int) {
    var depth, width int
    visited := make(map[int]bool)
    
    // 深度优先搜索
    func dfs(node int, level int) {
        if visited[node] {
            return
        }
        visited[node] = true
        depth = max(depth, level)
        width += len(graph[node])
        for _, neighbor := range graph[node] {
            dfs(neighbor, level+1)
        }
    }
    
    dfs(userId, 0)
    return depth, width
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

**解析：** 该示例代码使用深度优先搜索（DFS）来计算用户的社交网络影响范围。`calculateInfluence` 函数接收一个表示社交网络拓扑的图和用户ID，返回该用户的影响深度和广度。
