class DonationSystem:
    def __init__(self):
        self.donations = []
        self.ranking = []

    def add_donation(self, user, amount):
        self.donations.append((user, amount))
        self.ranking.append((amount, user))
        self.ranking.sort(reverse=True)

    def get_donation_history(self, user):
        return [(user, amount) for user, amount in self.donations if user == user]

    def get_donation_ranking(self):
        return [(amount, user) for amount, user in self.ranking]
```

**解析：** 此题考察应聘者对数据结构和排序算法的理解，以及如何使用它们实现一个简单的捐赠系统。

import java.util.Map;
import java.util.TreeMap;

class SponsorSystem {
    private TreeMap<Integer, String> ranking;

    public SponsorSystem() {
        ranking = new TreeMap<>(Collections.reverseOrder());
    }

    public void updateSponsorship(int userId, int newAmount) {
        ranking.put(newAmount, userId);
    }

    public void addSponsorship(int userId, int amount) {
        ranking.put(amount, userId);
    }

    public void printRanking() {
        for (Map.Entry<Integer, String> entry : ranking.entrySet()) {
            System.out.println("Sponsor: " + entry.getValue() + ", Amount: " + entry.getKey());
        }
    }
}
```
