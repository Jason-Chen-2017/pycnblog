#### 3. Flask中的路由和视图函数

**题目：** Flask中的路由和视图函数是什么？它们的作用是什么？

#### 5. 使用Flask构建机器学习API的示例

**题目：** 请给出一个使用Flask构建机器学习API的简单示例。

**答案：** 下面是一个简单的Flask机器学习API示例，该示例使用Scikit-learn库的线性回归模型。

```python
from flask import Flask, request, jsonify
from sklearn.linear_model import LinearRegression

app = Flask(__name__)

# 创建线性回归模型
model = LinearRegression()

# 加载模型权重
model.fit([[1], [2], [3]], [2, 4, 6])

@app.route('/predict', methods=['POST'])
def predict():
    # 获取请求体中的数据
    data = request.get_json()
    x = data['x']
    
    # 使用模型进行预测
    prediction = model.predict([[x]])
    
    # 构建响应对象
    response = {
        'prediction': prediction[0]
    }
    
    # 返回响应
    return jsonify(response)

if __name__ == '__main__':
    app.run(debug=True)
```

**解析：** 在这个示例中，我们创建了一个简单的Flask应用，并使用Scikit-learn的线性回归模型。我们定义了一个`/predict`端点，用于处理POST请求。在视图函数中，我们获取请求体中的数据，使用模型进行预测，并将预测结果返回给用户。
