#### 2. 数据结构与算法

**题目：** 实现一个有效的排序算法，并分析其时间复杂度。

**答案解析：**

排序算法有很多种，以下是几种常见的排序算法及其时间复杂度分析：

1. **冒泡排序：** 时间复杂度为 O(n^2)，空间复杂度为 O(1)。
2. **选择排序：** 时间复杂度为 O(n^2)，空间复杂度为 O(1)。
3. **插入排序：** 时间复杂度为 O(n^2)，但在部分有序的数据中表现较好。
4. **快速排序：** 平均时间复杂度为 O(n log n)，最坏情况下为 O(n^2)。
5. **归并排序：** 时间复杂度为 O(n log n)，空间复杂度为 O(n)。

**源代码实例：**

```go
// 冒泡排序示例
func bubbleSort(arr []int) {
    n := len(arr)
    for i := 0; i < n-1; i++ {
        for j := 0; j < n-i-1; j++ {
            if arr[j] > arr[j+1] {
                arr[j], arr[j+1] = arr[j+1], arr[j]
            }
        }
    }
}
```

#### 1. 寻找两个正序数组的中位数

**题目：** 给定两个已排序的数组 `nums1` 和 `nums2`，找出这两个数组的中位数。

**答案解析：**

我们可以通过合并两个数组并找到中位数来解决这个问题。或者，我们可以使用二分查找法，在其中一个数组中找到中位数的位置。

**源代码实例：**

```go
func findMedianSortedArrays(nums1 []int, nums2 []int) float64 {
    m, n := len(nums1), len(nums2)
    if m > n {
        nums1, nums2 = nums2, nums1
        m, n = n, m
    }
    imin, imax, halfLen := 0, m, (m+n+1)/2
    for imin <= imax {
        i := (imin + imax) / 2
        j := halfLen - i
        if i < m && nums2[j-1] > nums1[i] {
            // i 过大，需要减小
            imax = i - 1
        } else if i > 0 && nums1[i-1] > nums2[j] {
            // i 过小，需要增大
            imin = i + 1
        } else {
            // 找到中位数
            if i == 0 {
                maxOfLeft := nums2[j-1]
            } else if j == 0 {
                maxOfLeft := nums1[i-1]
            } else {
                maxOfLeft := max(nums1[i-1], nums2[j-1])
            }
            if (m+n)%2 == 1 {
                return float64(maxOfLeft)
            }
            minOfRight := 0
            if i < m {
                minOfRight = nums1[i]
            }
            if j < n {
                minOfRight = min(minOfRight, nums2[j])
            }
            return float64(maxOfLeft+minOfRight) / 2.0
        }
    }
    return 0
}
```

#### 2. 最长公共子序列

**题目：** 给定两个字符串 `text1` 和 `text2`，找出它们的最长公共子序列。

**答案解析：**

我们可以使用动态规划的方法来解决这个问题。定义一个二维数组 `dp`，其中 `dp[i][j]` 表示 `text1` 的前 `i` 个字符和 `text2` 的前 `j` 个字符的最长公共子序列的长度。

**源代码实例：**

```go
func longestCommonSubsequence(text1 string, text2 string) int {
    m, n := len(text1), len(text2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if text1[i-1] == text2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]
}
```
