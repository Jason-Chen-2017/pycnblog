   - **代码实例：**
   ```python
   import numpy as np

   # 用户行为矩阵
   user_behavior_matrix = np.array([[1, 0, 1], [1, 1, 0], [0, 1, 1]])

   # 计算用户相似度
   user_similarity_matrix = np.dot(user_behavior_matrix, user_behavior_matrix.T) / np.linalg.norm(user_behavior_matrix, axis=1)[:, np.newaxis]

   # 推荐物品
   for user_id in range(user_behavior_matrix.shape[0]):
       recommended_items = np.argsort(user_similarity_matrix[user_id])[::-1]
       print("User {} recommends: {}".format(user_id, recommended_items))
   ```
