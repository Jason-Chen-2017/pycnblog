**答案：** 常用的文本相似度计算方法包括余弦相似度、Jaccard相似度、编辑距离和Word2Vec。

**解析：** 余弦相似度通过计算两个向量的余弦值来衡量文本相似度；Jaccard相似度通过计算两个集合的交集和并集的比值来衡量文本相似度；编辑距离（Levenshtein距离）衡量两个字符串之间的差异程度；Word2Vec是一种将词语转换为向量的方法，通过计算向量之间的距离来衡量文本相似度。

**答案解析：** 搜索结果排序可以通过计算查询与结果之间的相似度实现。以下是一个基于TF-IDF的简单实现：

```python
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

def search_results_sort(results, query):
    # 将结果和查询转换为向量
    result_vectors = [[word2vec[word] for word in result if word in word2vec] for result in results]
    query_vector = [word2vec[word] for word in query if word in word2vec]

    # 计算TF-IDF并返回排序后的结果
    scores = [cosine_similarity([vector], [query_vector])[0][0] for vector in result_vectors]
    sorted_results = [result for _, result in sorted(zip(scores, results), reverse=True)]
    return sorted_results
```
