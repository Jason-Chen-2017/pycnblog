#### 2. Explain the concept of collaborative filtering in AI-based recommendation systems.

**Answer:** Collaborative filtering is a method used in recommendation systems that makes predictions about the interests of a user by collecting preferences from many users. It can be categorized into two types:

- **User-based Collaborative Filtering:** Finds similar users based on their past preferences and recommends items those similar users have liked.
- **Item-based Collaborative Filtering:** Identifies similar items based on their attributes and user interactions, then recommends items that are similar to what the user has previously liked.

#### 3. How does a decision tree work in predictive analytics for e-commerce search?

**Answer:** A decision tree is a popular machine learning model used for classification and regression tasks. It works as follows:

1. **Splitting Data:** The algorithm starts by splitting the data into subsets based on the values of one feature.
2. **Calculating Gini Impurity or Entropy:** For each split, the algorithm calculates the impurity or entropy of the resulting subsets to determine the best split.
3. **Recursively Building the Tree:** The process of splitting and evaluating the impurity is repeated for each subset until a stopping criterion is met (e.g., a maximum depth or a minimum number of samples per leaf).
