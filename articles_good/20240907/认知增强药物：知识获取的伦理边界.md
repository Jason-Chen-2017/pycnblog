### 1. 计算文本中关键词的权重

**题目：** 编写一个函数，计算给定文本中关键词的权重。关键词的权重取决于其在文本中出现的频率和重要性。

**输入：** 

```plaintext
text = "认知增强药物在学术研究中具有重要作用，有助于提高研究效率。"
keywords = ["认知增强药物", "学术研究", "研究效率"]
```

**输出：**

```plaintext
{"认知增强药物": 2.5, "学术研究": 2.0, "研究效率": 1.5}
```

**答案：** 

```python
def compute_keyword_weights(text, keywords):
    word_counts = {}
    for keyword in keywords:
        word_counts[keyword] = text.count(keyword)
    
    total_words = len(text.split())
    keyword_weights = {}
    for keyword, count in word_counts.items():
        keyword_weights[keyword] = count / total_words
    
    return keyword_weights

text = "认知增强药物在学术研究中具有重要作用，有助于提高研究效率。"
keywords = ["认知增强药物", "学术研究", "研究效率"]
result = compute_keyword_weights(text, keywords)
print(result)
```

**解析：** 该函数计算给定文本中每个关键词的权重，权重取决于关键词在文本中的频率和总词数。关键词的权重值介于 0 和 1 之间，表示其在文本中的重要程度。

### 2. 判断句子是否包含特定的关键词

**题目：** 编写一个函数，判断给定句子是否包含特定的关键词。关键词可以是单个单词或短语。

**输入：**

```plaintext
sentence = "认知增强药物在学术研究中具有重要作用。"
keywords = ["认知增强药物", "学术研究"]
```

**输出：**

```plaintext
True
```

**答案：**

```python
def contains_keywords(sentence, keywords):
    for keyword in keywords:
        if keyword in sentence:
            return True
    return False

sentence = "认知增强药物在学术研究中具有重要作用。"
keywords = ["认知增强药物", "学术研究"]
result = contains_keywords(sentence, keywords)
print(result)
```

**解析：** 该函数遍历每个关键词，检查句子是否包含该关键词。如果包含，则返回 True；否则，返回 False。

### 3. 计算文本中关键词的相似度

**题目：** 编写一个函数，计算给定文本中两个关键词的相似度。相似度可以通过两个关键词在文本中的共同词频来衡量。

**输入：**

```plaintext
text = "认知增强药物在学术研究中具有重要作用，有助于提高研究效率。"
keywords = ["认知增强药物", "学术研究"]
```

**输出：**

```plaintext
0.75
```

**答案：**

```python
def compute_similarity(text, keyword1, keyword2):
    word_counts = {}
    for word in text.split():
        word_counts[word] = word_counts.get(word, 0) + 1

    count1 = word_counts.get(keyword1, 0)
    count2 = word_counts.get(keyword2, 0)
    common_count = min(count1, count2)

    total_count = sum(word_counts.values())
    similarity = common_count / total_count

    return similarity

text = "认知增强药物在学术研究中具有重要作用，有助于提高研究效率。"
keywords = ["认知增强药物", "学术研究"]
result = compute_similarity(text, keywords[0], keywords[1])
print(result)
```

**解析：** 该函数计算两个关键词在文本中的共同词频，并将其与总词频进行比较，以计算相似度。相似度介于 0 和 1 之间，表示两个关键词在文本中的相似程度。
