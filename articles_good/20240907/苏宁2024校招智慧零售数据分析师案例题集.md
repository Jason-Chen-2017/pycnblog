##### 3. 数据排序

**题目：** 对给定的数据进行排序，要求时间复杂度为O(nlogn)。

**答案：** 
- **算法选择：** 选择时间复杂度为O(nlogn)的排序算法，如归并排序、快速排序等。
- **实现细节：** 实现排序算法，确保排序正确。

**代码示例：**

```python
# Python 代码示例，仅作参考
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])

    return result

# 用户输入
arr = list(map(int, input("请输入待排序的数字序列：").split()))

# 排序
sorted_arr = merge_sort(arr)

# 输出结果
print("排序后的序列：", sorted_arr)
```
