## 4.1 快速阅读代码实例
以下是一个快速阅读的代码实例：

```python
import re

def fast_read(text, keywords):
    words = re.findall(r'\w+', text)
    result = []
    for word in words:
        if word in keywords:
            result.append(word)
    return result

text = "This is a sample text for fast reading."
keywords = ["fast", "reading"]
print(fast_read(text, keywords))
```

这个代码实例中，我们首先导入了 `re` 模块，用于正则表达式匹配。然后定义了一个 `fast_read` 函数，该函数接受一个文本和一组关键词作为输入，通过正则表达式匹配文本中的关键词，并将其添加到结果列表中。最后，我们调用该函数，将一个示例文本和一组关键词作为输入，并打印出结果。
