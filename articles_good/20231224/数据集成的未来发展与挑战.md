# 4.具体代码实例和详细解释说明

## 4.1 数据整合代码实例

```python
# 数据整合代码实例

# 定义来源数据集
T1 = [{'name': 'Alice', 'age': 25, 'gender': 'Female'},
      {'name': 'Bob', 'age': 30, 'gender': 'Male'}]
T2 = [{'name': 'Alice', 'age': 26, 'gender': 'Female'},
      {'name': 'Charlie', 'age': 35, 'gender': 'Male'}]

# 数据整合
R = T1.union(T2)

# 打印整合后的结果集
print(R)
```

输出结果：

```
[{'name': 'Alice', 'age': 25, 'gender': 'Female'},
 {'name': 'Bob', 'age': 30, 'gender': 'Male'},
 {'name': 'Alice', 'age': 26, 'gender': 'Female'},
 {'name': 'Charlie', 'age': 35, 'gender': 'Male'}]
```

## 4.2 数据转换代码实例

```python
# 数据转换代码实例

# 定义来源数据集
T = [{'name': 'Alice', 'age': 25, 'gender': 'Female'}]

# 数据转换
T_out = {'name': ['name'], 'age': ['age'], 'gender': ['gender']}.fromkeys(T[0].keys())
for row in T:
    T_out.append(row)

# 打印转换后的结果集
print(T_out)
```

输出结果：

```
[{'name': 'Alice', 'age': 25, 'gender': 'Female'}]
```
