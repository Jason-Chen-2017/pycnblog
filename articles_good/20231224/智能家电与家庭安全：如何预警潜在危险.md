## 4.1 异常检测

异常检测可以通过以下代码实现：

```python
import numpy as np

def detect_anomaly(data, threshold):
    mean = np.mean(data)
    std = np.std(data)
    anomalies = []
    for x in data:
        z_score = (x - mean) / std
        if np.abs(z_score) > threshold:
            anomalies.append(x)
    return anomalies
```

在上述代码中，我们首先计算数据的均值和标准差。然后，我们遍历数据中的每个数据点，计算其Z分数。如果Z分数的绝对值大于阈值，则认为该数据点是异常点。
