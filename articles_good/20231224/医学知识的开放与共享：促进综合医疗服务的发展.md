```python
import requests
from bs4 import BeautifulSoup

url = 'https://www.example.com/medical-knowledge'
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

medical_knowledge = []
for item in soup.find_all('div', class_='medical-knowledge-item'):
    title = item.find('h2').text
    content = item.find('p').text
    medical_knowledge.append({'title': title, 'content': content})

print(medical_knowledge)
```

### 1.4.2 详细解释说明

这个代码实例使用了 Python 语言和 requests 库来获取一个医学知识网站的数据。首先，我们导入了 requests 库和 BeautifulSoup 库。然后，我们使用 requests.get() 方法来获取网站的数据。接着，我们使用 BeautifulSoup 库来解析获取到的数据。最后，我们使用 for 循环来遍历解析出来的数据，并将其存储到 medical_knowledge 列表中。
