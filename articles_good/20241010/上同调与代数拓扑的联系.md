##### 1.1 代数拓扑的基本概念

代数拓扑是拓扑学和代数学的交叉领域，主要研究拓扑空间上的代数结构。代数拓扑的基本概念包括拓扑空间、拓扑结构、闭包、开集、连通性等。

**定义：** 拓扑空间是由一个集合和这个集合上的一个拓扑结构构成的有序对 $(X, \tau)$，其中 $X$ 是集合，$\tau$ 是 $X$ 上的一个子集族，满足以下条件：
1. 空集和整个集合都属于 $\tau$。
2. $\tau$ 中的任意子集的并集仍属于 $\tau$。
3. $\tau$ 中任意两个子集的交集仍属于 $\tau$。

**基本性质：**
- 拓扑空间中的开集定义了空间中的“接近性”。
- 闭包是一个集合 $A$ 在拓扑空间中的所有极限点的集合，记作 $\overline{A}$。
- 连通性是指拓扑空间中任意两点都可以通过一条连续曲线相连。

##### 1.2 代数群与同调群

**代数群：** 在代数拓扑中，代数群是一个群，其元素是拓扑空间中的连续映射。代数群的主要研究对象是群的运算、子群、直积等。

- 0-同调群：描述了空间的连通性。
- 1-同调群：描述了空间的循环性。

#### 4.1 上同调群计算的基本方法

**间接法计算上同调群**

间接法是一种常用的计算上同调群的方法，它通过同态序列和短正合列来计算上同调群。

**定义**：对于拓扑空间 $X$ 和连通空间 $Y$，构造同态序列：
\[ 0 \to \text{ ker } d_n \to \text{ im } d_{n-1} \to \cdots \to \text{ ker } d_1 \to \text{ im } d_0 \to 0 \]
其中，$d_n: Y \to X$ 是边界映射。

**步骤**：
1. 计算短正合列的核和像。对于每个 $d_i$，计算其核 $\text{ ker } d_i$ 和像 $\text{ im } d_i$。
2. 构造上同调群。根据同态序列的核和像，构造上同调群：
\[ H^n(X) = \frac{\text{ ker } d_n}{\text{ im } d_{n-1}} \]

**例**：考虑拓扑空间 $X$ 和连通空间 $Y$，计算 $X$ 的上同调群。

- **计算同调群**：
  - $\text{ ker } d_0 = Y$，$\text{ im } d_0 = X$。
  - $\text{ ker } d_1 = \text{ ker } d_1 = X$，$\text{ im } d_1 = \text{ im } d_1 = \text{ ker } d_0$。
  - $\text{ ker } d_2 = \text{ ker } d_2 = \text{ im } d_1$，$\text{ im } d_2 = \text{ im } d_2 = \text{ ker } d_1$。

- **构造上同调群**：
  - $H^0(X) = \frac{\text{ ker } d_0}{\text{ im } d_0} = \frac{Y}{X} = 0$。
  - $H^1(X) = \frac{\text{ ker } d_1}{\text{ im } d_0} = \frac{X}{X} = 0$。
  - $H^2(X) = \frac{\text{ ker } d_2}{\text{ im } d_1} = \frac{\text{ im } d_1}{\text{ im } d_0} = 0$。

因此，$X$ 的上同调群为 $0$。

直接法是一种通过构造同调代数空间来计算上同调群的方法。它通过同态序列和直和分解来计算上同调群。

**步骤**：
1. 构造同调代数空间。对于每个 $d_i$，构造同调代数空间：

其中，$R$ 是一个交换环。
