### 4.1 词义标注
词义标注是将单词、短语或句子映射到其对应的词义上。词义标注的代码实例如下：

```python
import nltk
from nltk.corpus import wordnet

def get_synsets(word):
    return wordnet.synsets(word)

def get_word_meaning(word):
    synsets = get_synsets(word)
    meanings = []
    for synset in synsets:
        meanings.append(synset.definition())
    return meanings

word = "run"
meanings = get_word_meaning(word)
print(meanings)
```

### 4.2 语义关系抽取
语义关系抽取是将语言元素映射到其对应的语义关系上。语义关系抽取的代码实例如下：

```python
from nltk.corpus import wordnet

def get_hypernyms(word):
    hypernyms = []
    for synset in wordnet.synsets(word):
        for hypernym in synset.hypernyms():
            hypernyms.append(hypernym.name())
    return hypernyms

word = "dog"
hypernyms = get_hypernyms(word)
print(hypernyms)
```

### 4.3 语义角色识别
语义角色识别是将语言元素映射到其对应的语义角色上。语义角色识别的代码实例如下：

```python
from nltk.corpus import wordnet

def get_verbs(sentence):
    verbs = []
    for word in sentence.split():
        if wordnet.synsets(word):
            verbs.append(word)
    return verbs

def get_semantic_roles(sentence):
    verbs = get_verbs(sentence)
    roles = []
    for verb in verbs:
        for synset in wordnet.synsets(verb):
            for role in synset.lemmas():
                roles.append(role.role())
    return roles

sentence = "John gave Mary a book."
roles = get_semantic_roles(sentence)
print(roles)
```
