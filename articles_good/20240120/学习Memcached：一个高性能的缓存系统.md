```python

# 存储数据

# 获取数据

# 删除数据
client.delete('key')
```

- **服务端编程**：Memcached的服务端编程可以通过以下代码实现：

```python
import memcache

# 创建一个Memcached服务器实例
server = memcache.Server(['127.0.0.1:11211'])

# 处理客户端请求
while True:
    request = server.get_request()
    if request is None:
        break
    command, key, value = request
    if command == 'set':
        server.set(key, value)
    elif command == 'get':
        server.get(key)
    elif command == 'delete':
        server.delete(key)
```
