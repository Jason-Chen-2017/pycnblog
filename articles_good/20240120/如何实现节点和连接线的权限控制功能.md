### 4.1 使用访问控制列表（Access Control List，ACL）

访问控制列表（Access Control List，ACL）是一种常用的权限控制方法。访问控制列表是一个用于表示权限关系的列表。访问控制列表中的元素是一个包含用户、资源和权限的元组。访问控制列表可以用于实现权限检查功能。

以下是一个使用访问控制列表实现权限检查功能的代码实例：

```python
class AccessControlList:
    def __init__(self):
        self.acl = {}

    def add_permission(self, user, resource, permission):
        if resource not in self.acl:
            self.acl[resource] = []
        self.acl[resource].append((user, permission))

    def check_permission(self, user, resource):
        if resource in self.acl:
            for item in self.acl[resource]:
                if item[0] == user:
                    return item[1]
        return None

# 创建访问控制列表实例
acl = AccessControlList()

# 添加权限
acl.add_permission("user1", "resource1", "read")
acl.add_permission("user2", "resource1", "write")
acl.add_permission("user3", "resource2", "read")

# 检查权限
print(acl.check_permission("user1", "resource1"))  # Output: read
print(acl.check_permission("user2", "resource1"))  # Output: write
print(acl.check_permission("user3", "resource2"))  # Output: read
print(acl.check_permission("user4", "resource1"))  # Output: None
```

### 4.2 使用角色分配和权限组

角色分配和权限组是一种用于实现权限分配功能的方法。角色分配和权限组可以用于将用户分组为不同的角色，并为每个角色分配权限。

以下是一个使用角色分配和权限组实现权限分配功能的代码实例：

```python
class Role:
    def __init__(self, name):
        self.name = name
        self.permissions = []

    def add_permission(self, permission):
        self.permissions.append(permission)

class User:
    def __init__(self, name):
        self.name = name
        self.roles = []

    def add_role(self, role):
        self.roles.append(role)

# 创建角色和权限
admin_role = Role("admin")
admin_role.add_permission("read")
admin_role.add_permission("write")
admin_role.add_permission("execute")

user1 = User("user1")
user1.add_role(admin_role)

# 检查权限
print(user1.check_permission("read"))  # Output: True
print(user1.check_permission("write"))  # Output: True
print(user1.check_permission("execute"))  # Output: True
```
