### 3.3 数学模型公式详细讲解

在销售管理系统的优化实践中，常见的数学模型公式有：

1. 利润最大化模型：
$$
\max_{x} P(x) - C(x)
$$
其中，$P(x)$ 是销售额，$C(x)$ 是成本，$x$ 是决策变量。

2. 客户满意度最大化模型：
$$
\max_{x} U(x) = \sum_{i=1}^{n} w_i \cdot u_i(x_i)
$$
其中，$U(x)$ 是客户满意度，$w_i$ 是客户权重，$u_i(x_i)$ 是客户满意度函数。

3. 库存成本最小化模型：
$$
\min_{x} K(x) = h \cdot S + \sum_{i=1}^{n} c_i \cdot I_i
$$
其中，$K(x)$ 是库存成本，$h$ 是库存成本率，$S$ 是总库存，$c_i$ 是商品成本，$I_i$ 是商品库存。

### 4.1 贪心算法实例

```python
def knapsack(weights, values, capacity):
    n = len(values)
    dp = [[0 for _ in range(capacity + 1)] for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if weights[i - 1] <= w:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - weights[i - 1]] + values[i - 1])
            else:
                dp[i][w] = dp[i - 1][w]
    return dp[n][capacity]
```

### 4.2 动态规划算法实例

```python
def coin_change(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] = min(dp[i], dp[i - coin] + 1)
    return dp[amount] if dp[amount] != float('inf') else -1
```
