### 4.1 代码实例

假设我们要使用RPA自动化一个简单的API测试任务，这个API接口用于获取用户信息。以下是一个使用Python和RPA库UiPath的代码实例：

```python
from uipath import Activities

# 创建一个API请求活动
api_request_activity = Activities.CreateAPIRequestActivity()

# 设置API请求的URL
api_request_activity.SetURL("https://api.example.com/user")

# 设置API请求的方法
api_request_activity.SetMethod("GET")

# 设置API请求的头部信息
api_request_activity.SetHeaders({"Authorization": "Bearer {access_token}"})

# 设置API请求的参数
api_request_activity.SetParameters({"id": "123"})

# 执行API请求
response = api_request_activity.Execute()

# 检查API请求的结果
if response.status_code == 200:
    print("API请求成功")
else:
    print("API请求失败")
```
