### 3.3 数学模型公式详细讲解

深度学习中的数学模型主要包括以下几种：

- 线性回归：y = wx + b
- 逻辑回归：P(y=1|x) = 1 / (1 + exp(-wx - b))
- 多层感知机（MLP）：f(x) = g(w1x + b1)g(w2x + b2)...g(wkx + bk)
- 卷积神经网络（CNN）：f(x) = max(0, Wx + b)
- 递归神经网络（RNN）：f(x_t) = g(Wx_t + Uf(x_{t-1}) + b)
- 自编码器（Autoencoder）：x = s(W1y + b1)，y = s(W2x + b2)
- 生成对抗网络（GAN）：G(z) 和 D(x)

其中，x 是输入数据，y 是输出数据，w 是权重，b 是偏置，g 是激活函数，s 是压缩函数。

# 构建模型

model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))
model.add(MaxPooling2D((2, 2)))

model.add(Dense(10, activation='softmax'))

# 编译模型
model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])
