## 4.1 图像恢复示例

### 4.1.1 代码实现

```python
import numpy as np

def A_matrix(N, M, d):
    A = np.zeros((N, M))
    for i in range(N):
        for j in range(M):
            A[i][j] = np.exp(-(i**2 + j**2) / (2 * d**2))
    return A

def b_vector(N, M, f):
    b = np.zeros(N)
    for i in range(N):
        for j in range(M):
            b[i] += f[i][j]
    return b

def conjugate_gradient(A, b, x0, tol, max_iter):
    r0 = b - A @ x0
    r_prev = r0.copy()
    d = -r0
    k = 0
    while np.linalg.norm(r0) > tol and k < max_iter:
        alpha = (r0.T @ r0) / (d.T @ A @ d)
        x = x0 + alpha * d
        r0 = b - A @ x
        beta = (r0.T @ r0) / (r_prev.T @ r0)
        d = -r0 + beta * d
        r_prev = r0.copy()
        k += 1
    return x, k

N, M = 256, 256
d = 1
f = np.random.rand(N, M)
A = A_matrix(N, M, d)
b = b_vector(N, M, f)
x0 = np.zeros(N)
tol = 1e-6
max_iter = 100

x, iterations = conjugate_gradient(A, b, x0, tol, max_iter)
print("Iterations:", iterations)
```

### 4.1.2 代码解释

在这个示例中，我们使用共轭梯度法进行图像恢复。首先，我们定义了一个二维数组 `f` 来表示噪声图像，然后计算了 `A` 矩阵和 `b` 向量。接着，我们使用共轭梯度法进行迭代求解，直到收敛条件满足。最后，我们输出了迭代次数。
