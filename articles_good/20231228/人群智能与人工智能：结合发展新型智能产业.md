人群智能是指通过集体智慧和人类协作来解决问题的过程。它的发展可以追溯到19世纪的社会学家和心理学家，如格雷厄姆（Gustave Le Bon）和卢梭（Jean-Jacques Rousseau）。然而，人群智能的概念并不是一成不变的，它随着社会和技术的发展而不断演变。

人工智能是一门研究如何让机器具有智能行为和人类类似的思维过程的科学。它的研究历史可以追溯到1950年代的伯克利大学的夏克瓦尔德（Alan Turing）和伯努利（John von Neumann）。

- **深度学习（Deep Learning）**：通过多层神经网络，机器可以学习复杂的特征和模式。
- **自然语言处理（Natural Language Processing, NLP）**：通过算法和模型，机器可以理解和生成自然语言文本。
- **计算机视觉（Computer Vision）**：通过算法和模型，机器可以理解和分析图像和视频。

import numpy as np
from sklearn.linear_model import LinearRegression

fusion_pred = (gw_pred + ai_pred) / 2

print(fusion_pred)
