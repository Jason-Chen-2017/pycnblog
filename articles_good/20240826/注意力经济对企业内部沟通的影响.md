### 5. 项目实践：代码实例和详细解释说明

- 库：Natural Language Toolkit (NLTK)、Scikit-learn

#### 源代码详细实现

```python
import nltk
from nltk.tokenize import word_tokenize
from sklearn.feature_extraction.text import TfidfVectorizer

# 假设已有1000条信息
infos = ["信息1", "信息2", "信息3", ...]

# 使用TF-IDF模型对信息进行分类
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(infos)

# 根据TF-IDF值对信息进行排序
sorted_indices = X.sum(axis=1).sort_values(ascending=False).index

# 筛选出前10%的关键信息
key_infos = [infos[i] for i in sorted_indices[:int(len(sorted_indices) * 0.1)]]

# 将关键信息分配给员工处理
# 假设已有10名员工
employees = ["员工1", "员工2", ...]
for i, info in enumerate(key_infos):
    employee = employees[i % len(employees)]
    print(f"信息：{info}，分配给员工：{employee}")
```

上述代码首先使用NLTK库对信息进行分词，然后使用Scikit-learn库的TF-IDF模型对信息进行分类和标签化。接下来，根据TF-IDF值对信息进行排序，筛选出前10%的关键信息。最后，将关键信息分配给员工处理。
