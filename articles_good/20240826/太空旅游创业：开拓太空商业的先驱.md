### 5.2 源代码详细实现

以下是一个简单的Python代码实例，用于计算太空飞行器的轨道参数。

```python
import numpy as np
from scipy.integrate import solve_ivp

# 运动方程
def equation_of_motion(t, y):
    r, v = y
    G = 6.67430e-11  # 万有引力常数
    M = 5.972e24  # 地球质量
    a = -G * M / r**2
    dvdt = a
    drdt = v
    return [drdt, dvdt]

# 初始条件
initial_state = [1e7, 7.9e3]  # 轨道高度和初始速度

# 时间范围
t_span = [0, 2 * 3600]  # 2小时的时间范围

# 求解运动方程
solution = solve_ivp(equation_of_motion, t_span, initial_state)

# 绘制轨道
import matplotlib.pyplot as plt

plt.plot(solution.t, solution.y[0])
plt.xlabel('Time (s)')
plt.ylabel('Radius (m)')
plt.title('Orbit of a Satellite')
plt.show()
```

- **时间范围**：设定了2小时的时间范围，用于模拟飞行器的运动。
