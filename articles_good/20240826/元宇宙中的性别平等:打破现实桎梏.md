```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 数据加载
data = pd.read_csv('metaverse_data.csv')

# 数据预处理
data['gender'] = data['gender'].map({'male': 0, 'female': 1})
data['discrimination'] = data['discrimination'].map({True: 1, False: 0})

# 性别平等指数计算
N = len(data)
M = np.sum(data['gender'])
S = M / N

# 性别歧视指数计算
R = np.sum(data['discrimination'])
D = R / N

# 结果输出
print(f"性别平等指数 S: {S:.2f}")
print(f"性别歧视指数 D: {D:.2f}")

# 可视化
plt.bar(['性别平等', '性别歧视'], [S, D])
plt.xlabel('指数类型')
plt.ylabel('指数值')
plt.title('元宇宙性别平等分析')
plt.show()
```

3. **性别平等指数计算**：计算性别平等指数 $S$，其中 $M$ 为女性用户数，$N$ 为总用户数。
4. **性别歧视指数计算**：计算性别歧视指数 $D$，其中 $R$ 为性别歧视行为的发生率，$N$ 为总用户数。
