## 3.1 动态规划

动态规划是一种常用的优化问题解决方法，它可以用来解决具有最大化或最小化目标的子问题。动态规划的核心思想是将问题分解为多个子问题，然后递归地解决这些子问题，最后将解决的子问题的结果组合成最终的解。

动态规划的具体操作步骤如下：

1. 确定问题的状态：将问题分解为多个状态，每个状态都有一个独立的解。
2. 确定递归关系：找到每个状态的解与其他状态的解之间的关系。
3. 确定初始条件：找到问题的基本状态，即无需递归解决的状态。
4. 求解：根据递归关系和初始条件，递归地解决每个状态，并将解存储在一个表格中。
5. 回溯：根据表格中的解，回溯得到问题的最终解。

## 3.2 贪心算法

贪心算法是一种常用的优化问题解决方法，它的核心思想是在每个决策时 Always Choose the locally optimal solution (ACLOS)。也就是说，在每个决策点上，我们都尝试选择当前最优的解，而不考虑整个问题的全局最优解。

贪心算法的具体操作步骤如下：

1. 确定决策函数：找到一个可以评估当前决策的函数，这个函数应该能够衡量当前决策的优劣。
2. 选择当前最优决策：根据决策函数，选择当前最优的决策。
3. 更新决策函数：根据当前的决策，更新决策函数，以便在下一个决策点上进行新的评估。
4. 重复步骤2和3，直到问题得到解决。

## 3.3 分治算法

分治算法是一种常用的解决问题的方法，它的核心思想是将问题分解为多个子问题，然后递归地解决这些子问题，最后将解决的子问题的结果组合成最终的解。

分治算法的具体操作步骤如下：

1. 将问题分解为多个子问题：找到问题的子问题，并将其分解为多个更小的子问题。
2. 递归解决子问题：对于每个子问题，递归地解决它，直到找到最基本的子问题。

```python
def longest_palindrome(s: str) -> int:
    n = len(s)
    dp = [[False] * n for _ in range(n)]
    max_len = 0

    for i in range(n):
        dp[i][i] = True
        max_len = 1

    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            if s[i] == s[j] and dp[i + 1][j - 1]:
                dp[i][j] = True
                max_len = length

    return max_len
```
