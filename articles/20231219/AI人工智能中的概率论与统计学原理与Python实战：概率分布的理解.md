## 3.1 概率分布的类型

概率分布可以分为两类：离散型和连续型。离散型概率分布用于描述离散的随机事件，如掷骰子的结果。连续型概率分布用于描述连续的随机事件，如人的身高。

## 3.2 概率密度函数

概率密度函数（PDF）是概率分布的一个重要特性，它描述了某个随机变量在某个特定区间内的概率密度。概率密度函数可以用来计算某个区间内的概率。

## 3.3 累积分布函数

累积分布函数（CDF）是概率分布的另一个重要特性，它描述了某个随机变量在某个特定区间内的概率。累积分布函数可以用来计算某个区间内的概率。

## 3.4 常见的概率分布

### 3.4.1 泊松分布

泊松分布是一种离散型概率分布，用于描述在固定时间内发生固定类型事件的概率。泊松分布的概率密度函数为：

$$
P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}
$$

其中，$k$ 是事件发生的次数，$\lambda$ 是事件发生的平均率。

### 3.4.2 指数分布

指数分布是一种连续型概率分布，用于描述时间间隔的寿命。指数分布的累积分布函数为：

$$
F(t) = 1 - e^{-\lambda t}
$$

其中，$t$ 是时间间隔，$\lambda$ 是事件发生的平均率。

### 3.4.3 正态分布

正态分布是一种连续型概率分布，用于描述大量数据的分布。正态分布的概率密度函数为：

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$x$ 是随机变量，$\mu$ 是期望值，$\sigma$ 是标准差。

## 4.1 泊松分布的计算

```python
import numpy as np
from scipy.stats import poisson

# 设置平均率
lambda_ = 2

# 计算概率
k = 3
prob = poisson.pmf(k, lambda_)
print(f"泊松分布的概率：{prob}")
```

## 4.2 指数分布的计算

```python
import numpy as np
from scipy.stats import exponweib

# 设置平均率
lambda_ = 1

# 计算累积概率
t = 2

print(f"指数分布的累积概率：{cdf}")
```

## 4.3 正态分布的计算

```python
import numpy as np
from scipy.stats import norm

# 设置参数
mu = 0
sigma = 1

# 计算概率密度函数
x = 0
pdf = norm.pdf(x, mu, sigma)
print(f"正态分布的概率密度函数：{pdf}")

# 计算累积概率函数
```
