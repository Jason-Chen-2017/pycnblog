# 2.核心概念与联系

## 2.1 数据中台

数据中台是一种架构，它的目的是为了解决企业中的数据管理问题。数据中台可以帮助企业将数据存储、处理、分析等功能集中化管理，提高数据的可用性和可靠性。数据中台的主要组成部分包括：

- 数据清洗：对数据进行清洗和预处理，以便进行分析。
- 数据存储：提供一个中心化的存储系统，以便存储和管理数据。

数据湖是一种数据存储方式，它允许企业将结构化和非结构化数据存储在一个中心化的存储系统中，以便更好地管理和分析。数据湖的主要特点包括：

- 灵活性：数据湖允许企业将不同类型的数据存储在一个中心化的存储系统中，以便更好地管理和分析。
