## 3.1 深度优先搜索（DFS）

深度优先搜索（DFS）是一种常用的图遍历算法，它的核心思想是先深入一个节点，然后再广度遍历其子节点。这种思想在禅中可以看作是一种“直接指点”的方法。

具体的操作步骤如下：

1. 从图的一个节点开始，将该节点标记为已访问。
2. 从该节点出发，以深度为主，遍历其所有未访问的邻居节点。
3. 对于每个邻居节点，如果它还有未访问的邻居节点，则递归地应用深度优先搜索。
4. 当所有可能的节点都被访问完毕后，算法结束。

在数学模型中，深度优先搜索可以用递归的方式表示。例如，对于一个有向图G，深度优先搜索可以用以下递归公式表示：

$$
DFS(G, v) = \begin{cases}
    \text{访问节点v并标记为已访问} \\
    \text{对于每个从节点v出发的有向边(v, u)} \\
    \text{如果节点u未访问，则递归地调用DFS(G, u)} \\
\end{cases}
$$

## 3.2 广度优先搜索（BFS）

广度优先搜索（BFS）是另一种常用的图遍历算法，它的核心思想是先广度遍历一个节点的所有子节点，然后再深入其中一个子节点。这种思想在禅中可以看作是一种“自然妙行”的方法。

具体的操作步骤如下：

1. 从图的一个节点开始，将该节点标记为已访问。
2. 将该节点的所有未访问的邻居节点加入一个队列中。
3. 从队列中取出一个节点，将该节点标记为已访问。
4. 将该节点的所有未访问的邻居节点加入队列中。
5. 重复步骤3和4，直到队列为空为止。

在数学模型中，广度优先搜索可以用队列的数据结构来表示。例如，对于一个有向图G，广度优先搜索可以用以下公式表示：

$$
BFS(G, v) = \begin{cases}
    \text{访问节点v并标记为已访问} \\
    \text{将节点v的所有未访问的邻居节点加入队列中} \\
    \text{如果队列不为空，则取队列中的第一个节点u，将其标记为已访问} \\
    \text{将节点u的所有未访问的邻居节点加入队列中} \\
    \text{重复步骤3和4，直到队列为空为止} \\
\end{cases}
$$

## 4.1 深度优先搜索的Python实现

以下是一个使用Python实现深度优先搜索的代码示例：

```python
def dfs(graph, start):
    visited = set()
    stack = [start]

    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)

    return visited
```

在这个代码示例中，我们首先定义了一个`dfs`函数，该函数接受一个图`graph`和一个起始节点`start`作为输入。然后，我们创建了一个`visited`集合用于存储已访问的节点，并将起始节点推入`stack`栈中。接下来，我们进入一个`while`循环，每次循环中我们弹出栈顶节点`vertex`，并检查它是否已经被访问。如果没有，我们将其标记为已访问，并将其邻居节点推入栈中。循环继续，直到栈为空。最后，我们返回已访问的节点集合。

def bfs(graph, start):
    visited = set()
    queue = deque([start])
