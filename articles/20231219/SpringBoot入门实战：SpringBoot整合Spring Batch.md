Spring Boot 是一个用于构建新型 Spring 应用程序的优秀起点。它的目标是提供一种简单的配置，以便快速开发 Spring 应用程序。Spring Boot 的核心是一个独立的、平台上的应用程序，可以运行在单个 JAR 文件中。

Spring Batch 是一个用于构建批处理应用程序的框架。它提供了一些组件，如读取器、处理器、写入器和Job执行器，以便处理大量数据。Spring Batch 可以与 Spring Boot 整合，以便在 Spring Boot 应用程序中使用批处理功能。
