## 3.1 极限

极限是微积分的基本概念之一，它描述了一个变量在另一个变量接近某个特定值时的极限情况。在计算极限时，我们需要遵循以下规则：

1. 加法法则：对于两个函数f(x)和g(x)，如果lim x→a f(x)和g(x)都存在，那么lim x→a [f(x) + g(x)] = lim x→a f(x) + lim x→a g(x)。
2. 乘法法则：对于两个函数f(x)和g(x)，如果lim x→a f(x)和g(x)都存在，那么lim x→a [f(x) * g(x)] = lim x→a f(x) * lim x→a g(x)。
3. 除法法则：对于两个函数f(x)和g(x)，如果lim x→a g(x)≠0且lim x→a f(x)和g(x)都存在，那么lim x→a [f(x) / g(x)] = lim x→a f(x) / lim x→a g(x)。

## 3.2 导数

导数是微积分的核心概念之一，它描述了一个函数在某一点的变化率。在计算导数时，我们需要遵循以下规则：

1. 对于一个变量x的函数f(x)，其导数记为f'(x)，计算公式为f'(x) = lim (h→0) [f(x + h) - f(x)] / h。
2. 对于一个变量x和y的函数f(x, y)，其偏导数记为∂f/∂x和∂f/∂y，计算公式分别为∂f/∂x = lim (h→0) [f(x + h, y) - f(x, y)] / h和∂f/∂y = lim (h→0) [f(x, y + h) - f(x, y)] / h。

## 4.1 优化问题

在人工智能中，优化问题是非常常见的。我们可以使用微积分的梯度下降算法来解决这些问题。以下是一个简单的梯度下降示例：

```python
import numpy as np

def f(x):
    return x**2 + 2*x + 1

def gradient(x):
    return 2*x + 2

def gradient_descent(x0, learning_rate, iterations):
    x = x0
    for i in range(iterations):
        grad = gradient(x)
        x = x - learning_rate * grad
        print(f"Iteration {i+1}: x = {x}, f(x) = {f(x)}")
    return x

x0 = 0
learning_rate = 0.1
iterations = 100

x_min = gradient_descent(x0, learning_rate, iterations)
print(f"Minimum value of x: {x_min}")
```

在这个示例中，我们定义了一个简单的函数f(x) = x**2 + 2*x + 1，并计算了其导数gradient(x) = 2*x + 2。然后我们使用梯度下降算法来寻找这个函数的最小值。

## 4.2 函数的连续性和可导性

在人工智能中，我们经常需要分析函数的连续性和可导性。以下是一个简单的示例，用于检查一个函数是否连续或可导：

```python
import numpy as np

def f(x):
    if x < 0:
        return np.sin(1/x)
    else:
        return x**2

def is_continuous(x0, x1):
    return f(x0) == f(x1)

def is_differentiable(x0, epsilon=1e-7):
    return abs(f(x0 + epsilon) - f(x0 - epsilon)) < epsilon

x0 = 0
x1 = 1

print(f"Is f(x) continuous at x = {x0} and x = {x1}? {is_continuous(x0, x1)}")
print(f"Is f(x) differentiable at x = {x0}? {is_differentiable(x0)}")
```

在这个示例中，我们定义了一个函数f(x) = sin(1/x) if x < 0，否则x**2。然后我们使用连续性和可导性检查函数f(x)在x0和x1处的连续性和可导性。

## 4.3 多变量积分和梯度

在人工智能中，我们经常需要处理多变量的问题。以下是一个简单的示例，用于计算一个多变量函数的梯度：

```python
import numpy as np

def f(x, y):

```

在这个示例中，我们定义了一个多变量函数f(x, y) = x**2 + y**2，并计算了其梯度gradient(x, y) = [2*x, 2*y]。然后我们使用梯度来计算函数f(x, y)在(x, y)处的梯度。
