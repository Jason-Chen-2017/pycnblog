## 4.3 自然的流动

我们可以使用以下Python代码来实现动态规划和贪心算法：

```python
def dynamic_planning(n, W):
    dp = [0] * (n + 1)
    for i in range(1, n + 1):
        for j in range(i, n + 1):
            dp[j] = max(dp[j], dp[j - i] + W[i])
    return dp[n]

def greedy_algorithm(n, W):
    result = 0
    for i in range(n, 0, -1):
        if W[i] > 0:
            result += W[i]
            W[i] = 0
    return result
```

在这个例子中，我们首先使用dynamic_planning()函数实现动态规划算法，然后使用greedy_algorithm()函数实现贪心算法。这两个算法都是解决最大子集和问题的，动态规划算法的时间复杂度是O(n^2)，贪心算法的时间复杂度是O(n)。
