## 3.1 排序算法

排序算法是计算机科学中最基本的算法之一，它可以用来对一组数据进行排序。以下是一些常见的排序算法：

1. 冒泡排序：冒泡排序是一种简单的排序算法，它通过多次遍历数据，将较大的数据向后移动，将较小的数据向前移动，最终实现排序。具体的操作步骤如下：

   - 从第一个元素开始，与后面的每个元素进行比较。
   - 如果当前元素大于后面的元素，交换它们的位置。
   - 重复上述操作，直到整个数据序列有序。

2. 选择排序：选择排序是一种简单的排序算法，它通过多次遍历数据，找到最小的数据并将其放在最前面，最大的数据放在最后面，最终实现排序。具体的操作步骤如下：

   - 从第一个元素开始，找到最小的元素。
   - 将最小的元素与当前位置的元素交换。
   - 重复上述操作，直到整个数据序列有序。

3. 插入排序：插入排序是一种简单的排序算法，它通过将新的元素插入到已经排好序的数据中，最终实现排序。具体的操作步骤如下：

   - 从第一个元素开始，将其视为已经排好序的数据。
   - 从第二个元素开始，将它与已经排好序的数据进行比较，找到合适的位置插入。
   - 重复上述操作，直到整个数据序列有序。

4. 快速排序：快速排序是一种高效的排序算法，它通过将数据分为两部分，一部分较小的数据和一部分较大的数据，然后递归地对两部分数据进行排序，最终实现排序。具体的操作步骤如下：

   - 从第一个元素开始，将它视为基准元素。
   - 将其余元素分为两部分，一部分较小的数据和一部分较大的数据。
   - 将基准元素放在较小的数据的末尾，并递归地对较小的数据和较大的数据进行排序。
   - 重复上述操作，直到整个数据序列有序。

## 3.2 搜索算法

搜索算法是计算机科学中另一个基本的算法之一，它可以用来在一组数据中找到满足某个条件的元素。以下是一些常见的搜索算法：

1. 线性搜索：线性搜索是一种简单的搜索算法，它通过遍历数据，找到满足某个条件的元素。具体的操作步骤如下：

   - 从第一个元素开始，与给定的条件进行比较。
   - 如果当前元素满足条件，则返回它的位置。
   - 如果当前元素不满足条件，则继续遍历下一个元素。
   - 重复上述操作，直到找到满足条件的元素或者遍历完所有元素。

2. 二分搜索：二分搜索是一种高效的搜索算法，它通过将数据分为两部分，一部分较小的元素和一部分较大的元素，然后递归地对两部分数据进行搜索，最终找到满足某个条件的元素。具体的操作步骤如下：

   - 从第一个元素开始，将它视为基准元素。
   - 将其余元素分为两部分，一部分较小的元素和一部分较大的元素。
   - 如果基准元素满足条件，则返回它的位置。
   - 如果基准元素不满足条件，则将其放在较小的元素的末尾，并递归地对较小的元素和较大的元素进行搜索。
   - 重复上述操作，直到找到满足条件的元素或者遍历完所有元素。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(arr))
```
