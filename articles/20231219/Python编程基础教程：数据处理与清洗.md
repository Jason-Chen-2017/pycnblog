$$
X_{new} = X - X_{missing}

- 列均值填充法：
$$
X_{new}(i,j) = \bar{X}_i

2.数据噪声处理：

- 移动平均法：
$$
Y(t) = \frac{1}{2} \times (X(t-1) + X(t)) + \frac{1}{2} \times (X(t) + X(t+1))
$$

- 指数移动平均法：
$$
Y(t) = \alpha \times X(t) + (1-\alpha) \times Y(t-1)
$$

3.数据重复值处理：

- 去重法：
$$
X_{new} = X - X_{duplicate}
$$

# 4.1数据加载和预处理
```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 数据预处理
data = data.dropna()  # 删除缺失值
data = data.drop_duplicates()  # 删除重复值
```

# 4.2数据清洗
```python
# 数据清洗
data['age'] = data['age'].astype(int)  # 类型转换
data['age'].fillna(data['age'].mean(), inplace=True)  # 缺失值处理
```

# 4.3数据转换
```python
# 数据转换
data['gender'] = data['gender'].map({'M': 1, 'F': 0})  # 编码
data['age'] = (data['age'] - data['age'].mean()) / data['age'].std()  # 标准化
```

# 4.4数据整理
```python
# 数据整理
data_grouped = data.groupby('age').mean()  # 聚合
data_grouped.reset_index(inplace=True)  # 重置索引
```
