## 4.具体代码实例和详细解释说明

### 4.1.异常处理的具体代码实例

异常处理的具体代码实例如下：

```python
try:
    # 尝试执行的代码
    num = int(input("请输入一个整数："))
    result = 10 / num
except ValueError as e:
    # 处理异常的代码
    print("输入的不是一个整数，请重新输入！")
except ZeroDivisionError as e:
    # 处理异常的代码
    print("输入的整数不能为0，请重新输入！")
finally:
    # 恢复异常的代码
    print("程序运行完成！")
```

在这个代码实例中，我们首先尝试执行输入整数并进行除法运算。如果输入的不是一个整数，则捕获ValueError异常，并提示用户重新输入。如果输入的整数为0，则捕获ZeroDivisionError异常，并提示用户重新输入。最后，无论是否发生异常，都会执行finally语句，并提示程序运行完成。

### 4.2.调试的具体代码实例

调试的具体代码实例如下：

```python
def add(x, y):
    return x + y

def sub(x, y):
    return x - y

def mul(x, y):
    return x * y

def div(x, y):
    return x / y

if __name__ == "__main__":
    while True:
        try:
            x = float(input("请输入第一个数字："))
            y = float(input("请输入第二个数字："))
            op = input("请输入运算符：")
            if op == "+":
                result = add(x, y)
            elif op == "-":
                result = sub(x, y)
            elif op == "*":
                result = mul(x, y)
            elif op == "/":
                result = div(x, y)
            else:
                raise ValueError("输入的运算符不正确！")
            print("结果是：", result)
        except ValueError as e:
            print("输入的数据不是数字，请重新输入！")
        except ZeroDivisionError as e:
            print("除数不能为0，请重新输入！")
        except Exception as e:
            print("发生了未知错误，请重新输入！")
        finally:
            again = input("是否继续计算（y/n）：")
            if again != "y":
                break
```

在这个代码实例中，我们首先定义了四个基本的数学运算函数：add、sub、mul和div。然后，我们使用while循环来实现计算器的功能。用户可以输入两个数字和一个运算符，然后程序会根据运算符来执行相应的运算。如果用户输入的数据不是数字，则捕获ValueError异常，并提示用户重新输入。如果除数为0，则捕获ZeroDivisionError异常，并提示用户重新输入。最后，如果发生了未知错误，则捕获Exception异常，并提示用户重新输入。最后，无论是否发生异常，都会执行finally语句，并提示用户是否继续计算。
