# 4.具体代码实例和详细解释说明
具体代码实例如下：

1. 单例模式：
```python
class Singleton:
    _instance = None

    def __new__(cls, *args, **kwargs):
        if not cls._instance:
            cls._instance = super(Singleton, cls).__new__(cls, *args, **kwargs)
        return cls._instance
```
这是一个简单的单例模式实例，它使用了类的`__new__`方法，确保了一个类的实例只能通过该类的类方法创建。

2. 工厂模式：
```python
class Factory:

        pass

class ConcreteFactory(Factory):
    def create(self):
        return ConcreteProduct()

    pass
```
这是一个简单的工厂模式实例，它定义了一个工厂接口`Factory`，并实现了一个具体的工厂类`ConcreteFactory`，它可以创建一个具体的产品类`ConcreteProduct`。

3. 观察者模式：
```python
class Observer:
    def update(self, subject):
        pass

class ConcreteObserver(Observer):
    def update(self, subject):
        print(f"Observer: Received update from {subject}")

class Subject:
    def __init__(self):
        self._observers = []

    def attach(self, observer):
        self._observers.append(observer)

    def detach(self, observer):
        self._observers.remove(observer)

    def notify(self):
        for observer in self._observers:
            observer.update(self)
```
这是一个简单的观察者模式实例，它定义了一个观察者接口`Observer`，并实现了一个具体的观察者类`ConcreteObserver`。同时，它定义了一个主题接口`Subject`，并实现了一个具体的主题类`Subject`，它可以添加和移除观察者，并通知观察者更新。
