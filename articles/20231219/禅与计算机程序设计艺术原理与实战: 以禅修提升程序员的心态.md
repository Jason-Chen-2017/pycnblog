## 3.1 排序算法

排序算法是计算机程序设计中非常重要的一种算法，它的目的是将一组数据按照某种顺序进行排序。常见的排序算法有插入排序、选择排序、冒泡排序、快速排序等。

### 3.1.1 插入排序

插入排序是一种简单的排序算法，它的基本思想是将一个记录插入到已经排好序的有序列表中，从而得到一个新的有序列表。具体的操作步骤如下：

1. 从第一个元素开始，假设它是有序的。
2. 取下一个元素，与它前面的元素进行比较。
3. 如果当前元素小于前一个元素，将当前元素插入到前一个元素之前。
4. 重复步骤2和3，直到所有元素都排序。

### 3.1.2 选择排序

选择排序是一种简单的排序算法，它的基本思想是在未排序的元素中找到最小（或最大）元素，将其放在已排序的元素列表的末尾。具体的操作步骤如下：

1. 从第一个元素开始，假设它是有序的。
2. 取下一个元素，与它前面的元素进行比较。
3. 如果当前元素小于前一个元素，将当前元素插入到前一个元素之前。
4. 重复步骤2和3，直到所有元素都排序。

### 3.1.3 冒泡排序

冒泡排序是一种简单的排序算法，它的基本思想是通过多次交换相邻的元素，将最大（或最小）的元素逐渐移动到列表的末尾。具体的操作步骤如下：

1. 从第一个元素开始，假设它是有序的。
2. 取下一个元素，与它前面的元素进行比较。
3. 如果当前元素大于前一个元素，将当前元素和前一个元素交换位置。
4. 重复步骤2和3，直到所有元素都排序。

### 3.1.4 快速排序

快速排序是一种高效的排序算法，它的基本思想是通过选择一个基准元素，将其他元素分为两个部分：一个比基准元素小的部分，一个比基准元素大的部分。然后递归地对这两个部分进行排序。具体的操作步骤如下：

1. 从数组中选择一个基准元素。
2. 将比基准元素小的元素放在基准元素的左边，比基准元素大的元素放在基准元素的右边。
3. 对左边的子数组递归地进行快速排序。
4. 对右边的子数组递归地进行快速排序。

## 3.2 搜索算法

搜索算法是计算机程序设计中的另一种重要算法，它的目的是在一个数据结构中查找满足某个条件的元素。常见的搜索算法有线性搜索、二分搜索等。

### 3.2.1 线性搜索

线性搜索是一种简单的搜索算法，它的基本思想是从数据结构的开始处开始逐个检查元素，直到找到满足条件的元素为止。具体的操作步骤如下：

1. 从数据结构的开始处开始检查元素。
2. 检查当前元素是否满足条件。
3. 如果当前元素满足条件，返回当前元素的位置。
4. 如果当前元素不满足条件，继续检查下一个元素。
5. 重复步骤2和3，直到找到满足条件的元素。

### 3.2.2 二分搜索

二分搜索是一种高效的搜索算法，它的基本思想是将数据结构分为两个部分，然后根据被搜索的元素是否在两个部分的边界上来决定是否继续搜索。具体的操作步骤如下：

1. 从数据结构的开始处开始检查元素。
2. 检查当前元素是否满足条件。
3. 如果当前元素满足条件，返回当前元素的位置。
4. 如果当前元素不满足条件，根据被搜索的元素是否在当前元素的左边或右边来决定是否继续搜索。
5. 重复步骤2和3，直到找到满足条件的元素。

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
