## 4.1 代码实例
我们来看一个简单的代码实例，它是一个求两个数的最大公约数（GCD）的程序：

```python
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a
```

## 4.2 详细解释说明
上述代码实例中，我们定义了一个名为 `gcd` 的函数，它接受两个参数 `a` 和 `b`。函数的主体部分使用了一个 `while` 循环，循环的条件是 `b != 0`。在循环内部，我们使用了一个赋值语句来交换 `a` 和 `b`，并将 `b` 的值设置为 `a % b`。这个过程会一直持续到 `b` 的值为 0。最后，我们返回 `a` 的值，它就是两个数的最大公约数。
