## 4.2 代码的高效

以下是一个高效代码示例：

```python
def search(arr, target):
    """
    函数search实现二分查找
    """
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

在这个示例中，我们采用了以下几个高效的代码设计方法：

1. 使用二分查找算法：二分查找算法是一种高效的查找算法，时间复杂度为O(logn)。
2. 使用while循环实现：while循环可以更高效地实现二分查找算法，避免了递归调用带来的额外开销。

以下是一个可读性代码示例：

```python
def is_prime(n):
    """
    函数is_prime判断一个数是否为素数
    """
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True
```

在这个示例中，我们采用了以下几个可读性代码设计方法：

1. 使用清晰的变量名：变量名`n`清晰易懂，表示待判断的数。
