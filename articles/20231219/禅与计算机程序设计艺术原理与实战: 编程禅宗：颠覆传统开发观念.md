## 4.1 代码实例：排序算法

我们选择一个简单的排序算法——冒泡排序，来作为我们的代码实例。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

## 4.2 详细解释说明

通过上述代码实例，我们可以看到冒泡排序算法的核心原理是通过多次遍历数组，将较大的元素逐步冒泡到数组的末尾。这种方法的时间复杂度为O(n^2)，空间复杂度为O(1)。
