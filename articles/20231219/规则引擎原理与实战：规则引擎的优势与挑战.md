## 4.1 规则引擎的具体代码实例

以下是一个简单的规则引擎的具体代码实例：

```python
class RuleEngine:
    def __init__(self):
        self.knowledge_base = {}
        self.rules = []
        self.facts = []

    def load_knowledge_base(self, knowledge_base):
        self.knowledge_base = knowledge_base

    def evaluate_rules(self):
        for rule in self.rules:
            if self.knowledge_base.get(rule.condition, False):
                rule.action()

    def update_facts(self, facts):
        self.facts = facts

    def execute_decisions(self):
        for decision in self.facts:
            decision.execute()

```
