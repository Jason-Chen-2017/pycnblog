## 3.1抽样分布
抽样分布是一种用于估计参数的方法，它涉及从总体中随机抽取一部分样本，然后根据样本得出关于总体的结论。常见的抽样分布有单点估计、区间估计和比例估计等。

### 3.1.1单点估计
单点估计是一种用于估计参数的方法，它涉及从总体中随机抽取一部分样本，然后根据样本得出关于总体的结论。单点估计可以是点估计或者笛卡尔估计。

#### 3.1.1.1点估计
点估计是一种单点估计的方法，它涉及从总体中随机抽取一部分样本，然后根据样本中的一个值得出关于总体的结论。点估计的一个典型例子是样本均值。

#### 3.1.1.2笛卡尔估计
笛卡尔估计是一种单点估计的方法，它涉及从总体中随机抽取一部分样本，然后根据样本中的一个值得出关于总体的结论。笛卡尔估计的一个典型例子是样本中位数。

### 3.1.2区间估计
区间估计是一种用于估计参数的方法，它涉及从总体中随机抽取一部分样本，然后根据样本得出关于总体的结论。区间估计可以是置信区间估计或者信息区间估计。

#### 3.1.2.1置信区间估计
置信区间估计是一种区间估计的方法，它涉及从总体中随机抽取一部分样本，然后根据样本得出关于总体的结论。置信区间估计的一个典型例子是样本均值的置信区间。

#### 3.1.2.2信息区间估计
信息区间估计是一种区间估计的方法，它涉及从总体中随机抽取一部分样本，然后根据样本得出关于总体的结论。信息区间估计的一个典型例子是样本中位数的信息区间。

### 3.1.3比例估计
比例估计是一种用于估计比例的方法，它涉及从总体中随机抽取一部分样本，然后根据样本得出关于总体的结论。比例估计的一个典型例子是样本比例。

## 3.2假设检验
假设检验是一种用于检验某个假设的方法，它涉及从总体中随机抽取一部分样本，然后根据样本判断假设是否成立。假设检验可以是一样性假设检验或者相关性假设检验。

### 3.2.1一样性假设检验
一样性假设检验是一种假设检验的方法，它涉及从两个或多个总体中随机抽取一部分样本，然后根据样本判断这些总体是否相同。一样性假设检验的一个典型例子是两个样本的均值检验。

### 3.2.2相关性假设检验
相关性假设检验是一种假设检验的方法，它涉及从一个或多个总体中随机抽取一部分样本，然后根据样本判断这些变量是否存在相关性。相关性假设检验的一个典型例子是皮尔逊相关性检验。

## 4.1抽样分布
### 4.1.1单点估计
```python
import numpy as np
import scipy.stats as stats

# 生成一个正态总体
np.random.seed(0)
population = np.random.normal(loc=0, scale=1, size=10000)

# 从总体中随机抽取100个样本
sample = np.random.choice(population, size=100)

# 计算样本均值
sample_mean = np.mean(sample)

# 计算总体均值
population_mean = np.mean(population)

# 计算样本方差
sample_variance = np.var(sample)

# 计算总体方差
population_variance = np.var(population)

# 计算样本标准差
sample_stddev = np.std(sample)

# 计算总体标准差
population_stddev = np.std(population)

# 计算单点估计的置信区间
confidence_interval = stats.t.interval(alpha=0.95, df=len(sample)-1, loc=sample_mean, scale=sample_stddev/np.sqrt(len(sample)))

print("样本均值:", sample_mean)
print("总体均值:", population_mean)
print("样本方差:", sample_variance)
print("总体方差:", population_variance)
print("样本标准差:", sample_stddev)
print("总体标准差:", population_stddev)
print("单点估计的置信区间:", confidence_interval)
```

### 4.1.3比例估计
```python
import numpy as np
import scipy.stats as stats

# 生成一个二分类总体
np.random.seed(0)
population = np.random.choice([0, 1], size=10000, p=[0.5, 0.5])

# 从总体中随机抽取100个样本
sample = np.random.choice(population, size=100)

# 计算样本比例
sample_proportion = np.sum(sample) / len(sample)

# 计算总体比例
population_proportion = np.sum(population) / 10000

# 计算比例估计
proportion_estimate = stats.proportion_confint(sample_proportion, method='normal')

print("样本比例:", sample_proportion)
print("总体比例:", population_proportion)
print("比例估计:", proportion_estimate)
```
## 4.2假设检验
### 4.2.1一样性假设检验
```python
import numpy as np
import scipy.stats as stats

# 生成两个正态总体
np.random.seed(0)
population1 = np.random.normal(loc=0, scale=1, size=1000)
population2 = np.random.normal(loc=1, scale=1, size=1000)

# 从两个总体中随机抽取100个样本
sample1 = np.random.choice(population1, size=100)
sample2 = np.random.choice(population2, size=100)

# 计算两个样本的均值
sample1_mean = np.mean(sample1)
sample2_mean = np.mean(sample2)

# 一样性假设检验
t_statistic, p_value = stats.ttest_ind(sample1, sample2)

print("样本1均值:", sample1_mean)
print("样本2均值:", sample2_mean)
print("一样性假设检验的t统计量:", t_statistic)
print("一样性假设检验的p值:", p_value)
```
### 4.2.2相关性假设检验
```python
import numpy as np
import scipy.stats as stats

# 生成两个正态总体
np.random.seed(0)
population1 = np.random.normal(loc=0, scale=1, size=1000)
population2 = np.random.normal(loc=0, scale=1, size=1000)

# 从两个总体中随机抽取100个样本
sample1 = np.random.choice(population1, size=100)
sample2 = np.random.choice(population2, size=100)

# 计算两个样本之间的相关性
correlation_coefficient = np.corrcoef(sample1, sample2)[0, 1]

# 相关性假设检验
t_statistic, p_value = stats.ttest_corr(sample1, sample2)

print("相关性估计:", correlation_coefficient)
print("相关性假设检验的t统计量:", t_statistic)
print("相关性假设检验的p值:", p_value)
```
