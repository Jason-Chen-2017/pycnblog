## 3.2 数学模型公式详细讲解

### 3.2.1 深度优先搜索（DFS）

深度优先搜索（DFS）是一种搜索算法，它的核心思想是在每个节点深入探索可能的最深节点，直到无法继续探索为止。DFS 可以用递归来实现，其时间复杂度为 O(n!)，其中 n 是节点数。

### 3.2.2 广度优先搜索（BFS）

广度优先搜索（BFS）是一种搜索算法，它的核心思想是在每个节点沿着最短路径向外扩展，直到所有节点都被访问为止。BFS 可以用队列来实现，其时间复杂度为 O(n+m)，其中 n 是节点数，m 是边数。

### 3.2.3 动态规划（DP）

动态规划（DP）是一种解决最优化问题的算法，它的核心思想是将问题拆分成多个子问题，并将子问题的解存储在一个表格中，以便后续使用。DP 可以用递归来实现，其时间复杂度为 O(n)，其中 n 是问题的大小。

### 3.2.4 贪婪算法（GA）

贪婪算法（GA）是一种解决优化问题的算法，它的核心思想是在每个步骤中选择能够带来最大收益的选项，不考虑后续的影响。GA 可以用迭代来实现，其时间复杂度为 O(n)，其中 n 是问题的大小。

## 4.1 深度优先搜索（DFS）代码实例

```python
def dfs(graph, start):
    visited = set()
    stack = [start]
    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)
    return visited
```

## 4.2 广度优先搜索（BFS）代码实例

```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    while queue:
        vertex = queue.popleft()
        if vertex not in visited:
            visited.add(vertex)
            queue.extend(graph[vertex] - visited)
    return visited
```

## 4.3 动态规划（DP）代码实例

```python
def fib(n):
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n]
```

## 4.4 贪婪算法（GA）代码实例

```python
def coinChange(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] = min(dp[i], dp[i - coin] + 1)
    return dp[amount] if dp[amount] != float('inf') else -1
```
