```python
import hashlib
import base58

def create_address(private_key):
    public_key = int(private_key, 16) + 2**32
    public_key_hash = hashlib.sha256(public_key.to_bytes(32, byteorder='big')).digest()
    address_hash = base58.b58encode(public_key_hash)
    return address_hash.decode('utf-8')

private_key = '92d5a7c4e3e0f5f0'
address = create_address(private_key)
print(address)
```

在函数内部，我们首先将私钥转换为一个整数，然后将其加上 2**32。接着，我们使用 SHA-256 散列函数对公钥哈希，并将哈希结果转换为一个字节数组。最后，我们使用 base58 库对哈希结果进行编码，并将其解码为字符串形式。

最后，我们调用 `create_address` 函数，将一个示例私钥传入，并打印出生成的比特币地址。
