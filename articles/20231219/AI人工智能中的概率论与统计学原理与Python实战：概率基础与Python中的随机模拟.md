# 2.核心概念与联系

## 2.1 概率基础

概率是一种量化事件发生可能性的方法。概率通常用P（E）表示，其中E是事件，P（E）是事件E发生的可能性。概率通常取值在0到1之间，其中0表示事件不可能发生，1表示事件必然发生。

### 2.1.1 概率的计算

概率可以通过不同的方法来计算。以下是几种常见的概率计算方法：

1. 直接计数法：直接计数法是一种简单的概率计算方法，它需要计算事件发生的次数和总次数，然后将事件发生的次数除以总次数。例如，如果在100次实验中事件发生了50次，那么事件的概率为50/100=0.5。

2. 定理法：定理法是一种通过已知事件之间的关系来计算概率的方法。例如，如果事件A和事件B互相独立，那么事件A和事件B发生的概率是事件A的概率乘以事件B的概率的乘积。

3. 条件概率法：条件概率法是一种通过已知事件发生的条件来计算概率的方法。例如，如果事件A和事件B发生的概率是0.5和0.6，那么事件A发生时事件B发生的概率是0.6/0.5=1.2。

### 2.1.2 概率的几种常见定理

1. 总概率定理：总概率定理是一种用于计算多个事件发生概率的方法。总概率定理表示事件A和事件B发生的概率是事件A发生时事件B发生的概率乘以事件A发生的概率的乘积。例如，如果事件A的概率是0.5，事件B的概率是0.6，那么事件A和事件B发生的概率是0.5*0.6=0.3。

2. 贝叶斯定理：贝叶斯定理是一种用于计算条件概率的方法。贝叶斯定理表示事件A和事件B发生的概率是事件B发生时事件A发生的概率乘以事件B发生的概率的乘积。例如，如果事件A的概率是0.5，事件B的概率是0.6，那么事件A发生时事件B发生的概率是0.6/0.5=1.2。

3. 乘法法则：乘法法则是一种用于计算多个事件发生概率的方法。乘法法则表示事件A和事件B发生的概率是事件A发生时事件B发生的概率乘以事件A发生的概率的乘积。例如，如果事件A的概率是0.5，事件B的概率是0.6，那么事件A和事件B发生的概率是0.5*0.6=0.3。

例子：掷骰子的概率和随机模拟

3. 接下来，我们需要生成随机数来模拟掷骰子的过程。我们可以使用Python的random模块来生成随机数。

```python
def roll_dice():
    return random.randint(1, 6)

    result = roll_dice()
    print(result)
```
