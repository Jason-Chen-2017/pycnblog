## 4.1 数据归档代码实例

以下是一个简单的数据归档代码实例：

```python
import os
import time

def archive_data(data, archive_path):
    if not os.path.exists(archive_path):
        os.makedirs(archive_path)
    with open(os.path.join(archive_path, 'archive.txt'), 'a') as f:
        f.write(data + '\n')

data = 'This is a sample data'
archive_path = '/path/to/archive'
archive_data(data, archive_path)
```

在上述代码中，我们首先导入了`os`和`time`模块。接着定义了一个`archive_data`函数，该函数接收一个`data`和一个`archive_path`参数。如果`archive_path`不存在，则创建一个新的目录。接着，使用`with`语句打开一个文件，将`data`写入文件中。

## 4.2 冷热分离代码实例

以下是一个简单的冷热分离代码实例：

```python
import os
import time

def hot_data(data, hot_path):
    with open(os.path.join(hot_path, 'hot.txt'), 'a') as f:
        f.write(data + '\n')

def cold_data(data, cold_path):
    if not os.path.exists(cold_path):
        os.makedirs(cold_path)
    with open(os.path.join(cold_path, 'cold.txt'), 'a') as f:
        f.write(data + '\n')

hot_path = '/path/to/hot'
cold_path = '/path/to/cold'
data = 'This is a sample data'
hot_data(data, hot_path)
cold_data(data, cold_path)
```

在上述代码中，我们首先导入了`os`和`time`模块。接着定义了两个函数，分别是`hot_data`和`cold_data`。`hot_data`函数接收一个`data`和一个`hot_path`参数，将`data`写入`hot.txt`文件。`cold_data`函数接收一个`data`和一个`cold_path`参数，如果`cold_path`不存在，则创建一个新的目录。接着，使用`with`语句打开一个文件，将`data`写入文件中。
