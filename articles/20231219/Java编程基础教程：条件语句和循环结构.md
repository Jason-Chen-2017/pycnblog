## 2.核心概念与联系

### 2.1 条件语句

条件语句是一种用于根据某个条件是否满足来执行或跳过代码块的控制结构。在Java中，条件语句主要包括if语句、switch语句等。

#### 2.1.1 if语句

if语句是Java中最基本的条件语句，用于根据一个布尔表达式的值来执行或跳过代码块。if语句的基本格式如下：

```java
if (布尔表达式) {
    // 执行的代码块
}
```

布尔表达式的值可以是true或false。如果布尔表达式的值为true，则执行代码块；如果为false，则跳过代码块。

#### 2.1.2 switch语句

switch语句是一种用于根据一个变量的值执行不同代码块的控制结构。switch语句的基本格式如下：

```java
switch (变量) {
    case 值1:
        // 执行的代码块1
        break;
    case 值2:
        // 执行的代码块2
        break;
    // ...
    default:
        // 默认执行的代码块
}
```

switch语句中的变量可以是任何数据类型，但是case后面的值必须是常量表达式。switch语句中的每个case块后面必须有一个break语句，用于跳出switch语句。如果没有break语句，则会执行到下一个case块，直到执行到default块或者switch语句结束。

### 2.2 循环结构

循环结构是一种用于重复执行代码块的控制结构。在Java中，循环结构主要包括for语句、while语句、do-while语句等。

#### 2.2.1 for语句

for语句是Java中一种常用的循环结构，用于根据一个条件的值来重复执行代码块。for语句的基本格式如下：

```java
for (初始化语句; 条件表达式; 更新语句) {
    // 执行的代码块
}
```

初始化语句、条件表达式和更新语句都是可选的。如果初始化语句和更新语句不存在，则可以省略。

#### 2.2.2 while语句

while语句是Java中另一种常用的循环结构，用于根据一个条件的值来重复执行代码块。while语句的基本格式如下：

```java
while (条件表达式) {
    // 执行的代码块
}
```

条件表达式的值可以是true或false。如果条件表达式的值为true，则执行代码块；如果为false，则跳过代码块。

#### 2.2.3 do-while语句

do-while语句是Java中另一种常用的循环结构，用于根据一个条件的值来重复执行代码块。do-while语句的基本格式如下：

```java
do {
    // 执行的代码块
} while (条件表达式);
```

条件表达式的值可以是true或false。如果条件表达式的值为true，则执行代码块；如果为false，则跳出do-while语句。

## 4.具体代码实例和详细解释说明

### 4.1 条件语句的具体代码实例

```java
public class ConditionExample {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;
        if (a < b) {
            System.out.println("a < b");
        }
        switch (a % 3) {
            case 0:
                System.out.println("a 是 3 的倍数");
                break;
            case 1:
                System.out.println("a 不是 3 的倍数");
                break;
            default:
                System.out.println("a 是 3 的倍数");
                break;
        }
    }
}
```

在上述代码中，我们使用if语句和switch语句来判断两个变量的大小关系，以及变量a是否是3的倍数。

### 4.2 循环结构的具体代码实例

```java
public class LoopExample {
    public static void main(String[] args) {
        for (int i = 0; i < 10; i++) {
            System.out.println("i = " + i);
        }
        int i = 0;
        while (i < 10) {
            System.out.println("i = " + i);
            i++;
        }
        int i = 0;
        do {
            System.out.println("i = " + i);
            i++;
        } while (i < 10);
    }
}
```

在上述代码中，我们使用for语句、while语句和do-while语句来遍历0到9的整数，并将整数输出到控制台。

## 6.附录常见问题与解答

### 6.1 常见问题1：如何判断两个变量的大小关系？

答：可以使用if语句或switch语句来判断两个变量的大小关系。例如：

```java
int a = 10;
int b = 20;
if (a < b) {
    System.out.println("a < b");
}
```

### 6.2 常见问题2：如何判断一个变量是否是某个数字的倍数？

答：可以使用if语句或switch语句来判断一个变量是否是某个数字的倍数。例如：

```java
int a = 10;
if (a % 3 == 0) {
    System.out.println("a 是 3 的倍数");
}
```

### 6.3 常见问题3：如何实现循环遍历一个数组？

答：可以使用for语句、while语句或do-while语句来遍历一个数组。例如：

```java
int[] arr = {1, 2, 3, 4, 5};
for (int i = 0; i < arr.length; i++) {
    System.out.println("arr[" + i + "] = " + arr[i]);
}
```
