## 3.1 动态规划

动态规划是一种常用的优化问题解决方法，它可以用来解决具有最大或最小值的问题。动态规划的核心思想是将问题分解为子问题，并将子问题的解存储在一个表格中，以便后续使用。

### 3.1.1 动态规划的核心原理

动态规划的核心原理是将一个复杂的问题分解为多个子问题，并将子问题的解存储在一个表格中，以便后续使用。这种方法可以避免重复计算，从而提高算法的效率。

### 3.1.2 动态规划的具体操作步骤

1. 确定问题的状态：首先，我们需要确定问题的状态，即问题可以被分解为多个子问题的状态。

2. 确定基本状态：接下来，我们需要确定基本状态，即问题的最基本状态。

3. 确定状态转移方程：然后，我们需要确定状态转移方程，即从基本状态到其他状态的转移方式。

4. 确定终止条件：最后，我们需要确定终止条件，即问题的解已经得到。

### 3.1.3 动态规划的数学模型公式

动态规划的数学模型公式可以表示为：

$$
dp[i] = \max_{0 \leq j \leq i} \{ dp[j] + f(i, j) \}
$$

其中，$dp[i]$ 表示问题的解，$f(i, j)$ 表示从基本状态 $j$ 到状态 $i$ 的转移方式。

### 3.2.1 贪心算法的核心原理

贪心算法的核心原理是在每个步骤中选择当前状态下的最佳选择，并将其作为下一个步骤的起点。这种方法可以简化算法的设计，但并不一定能得到最优解。

### 3.2.2 贪心算法的具体操作步骤

1. 确定问题的目标函数：首先，我们需要确定问题的目标函数，即我们需要最大化或最小化的函数。

2. 确定问题的约束条件：接下来，我们需要确定问题的约束条件，即问题的解必须满足的条件。

3. 选择当前状态下的最佳选择：然后，我们需要选择当前状态下的最佳选择，并将其作为下一个步骤的起点。

### 3.2.3 贪心算法的数学模型公式

贪心算法的数学模型公式可以表示为：

$$

其中，$x^*$ 表示问题的解，$f(x)$ 表示目标函数，$X$ 表示问题的约束条件。

    n = len(nums)
    dp = [0] * (n + 1)
    max_sum = float('-inf')

        dp[i] = max(nums[i - 1], nums[i - 1] + dp[i - 1])
