```python
from typing import List

class Rule:
    def __init__(self, condition: str, action: str):
        self.condition = condition
        self.action = action

class RuleEngine:
    def __init__(self, rules: List[Rule]):
        self.rules = rules

    def execute(self, data):
        for rule in self.rules:
            if eval(rule.condition):
                eval(rule.action)

data = {'age': 25, 'gender': 'male'}
rules = [
    Rule('age > 20 and gender == "male"', 'print("恭喜，满足条件")'),
    Rule('age < 18', 'print("不满足条件")')
]

engine = RuleEngine(rules)
engine.execute(data)
```

在主程序中，我们创建了一个包含两个规则的规则引擎，并使用`execute`方法执行这些规则。

在上面的代码实例中，我们创建了一个简单的规则引擎和规则设计器。规则引擎通过`execute`方法来执行规则，它接收一个数据字典作为输入，并根据规则的条件部分来执行规则的动作部分。
