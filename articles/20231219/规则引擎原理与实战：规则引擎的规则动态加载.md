### 4.1.1 规则加载实现

```python
import json

class Rule:
    def __init__(self, id, condition, action, trigger):
        self.id = id
        self.condition = condition
        self.action = action
        self.trigger = trigger

class RuleEngine:
    def __init__(self):
        self.rules = []

    def load_rule(self, rule_file):
        with open(rule_file, 'r') as f:
            rule_data = json.load(f)
            rule = Rule(rule_data['id'], rule_data['condition'], rule_data['action'], rule_data['trigger'])
            self.rules.append(rule)

    def execute_rule(self, rule_id):
        for rule in self.rules:
            if rule.id == rule_id:
                if rule.trigger(rule.condition):
                    rule.action()
                    break
```

- `id`：规则的ID
- `condition`：规则的条件部分
- `action`：规则的动作部分
- `trigger`：规则的触发部分

我们还定义了一个`RuleEngine`类，用于表示规则引擎，它包括以下属性：

- `rules`：规则引擎的规则列表

`RuleEngine`类的`load_rule`方法用于加载新的规则，它通过读取规则文件，解析规则文件为`Rule`对象，并将其添加到规则列表中。

`RuleEngine`类的`execute_rule`方法用于执行规则，它通过遍历规则列表，找到与规则ID匹配的规则，并根据触发条件和动作部分执行规则。
