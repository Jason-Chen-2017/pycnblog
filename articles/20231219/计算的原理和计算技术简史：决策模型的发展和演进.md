## 4.1 排序算法实例

我们选择冒泡排序算法作为实例，代码如下：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个实例中，我们首先定义了一个名为 `bubble_sort` 的函数，接收一个数组 `arr` 作为参数。然后，我们使用两个嵌套循环来实现冒泡排序算法的过程。在内部循环中，我们比较相邻的两个元素，如果后面的元素小于前面的元素，则交换它们的位置。这个过程会重复执行，直到数组排序完成。

## 6.4 如何选择合适的排序算法？

选择合适的排序算法需要考虑以下几个因素：

- 数据规模：不同的排序算法适用于不同的数据规模。例如，冒泡排序适用于小规模数据，而快速排序适用于大规模数据。
- 时间复杂度：不同的排序算法的时间复杂度不同。例如，冒泡排序的时间复杂度为 O(n^2)，而快速排序的时间复杂度为 O(n log n)。
- 空间复杂度：不同的排序算法的空间复杂度不同。例如，快速排序的空间复杂度为 O(log n)，而归并排序的空间复杂度为 O(n)。

根据这些因素，可以选择合适的排序算法来满足不同的需求。
