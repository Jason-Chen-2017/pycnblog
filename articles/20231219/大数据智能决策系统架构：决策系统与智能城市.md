### 4.2.1 数据收集

```python
import requests

def get_traffic_data():
    url = 'http://api.smartcity.com/traffic'
    response = requests.get(url)
    data = response.json()
    return data
```

### 4.2.2 数据存储

```python
import pymysql

def store_traffic_data(data):
    connection = pymysql.connect(host='localhost', user='root', password='', database='smartcity')
    cursor = connection.cursor()
    for item in data:
        sql = 'INSERT INTO traffic (time, road_id, flow) VALUES (%s, %s, %s)'
        cursor.execute(sql, (item['time'], item['road_id'], item['flow']))
    connection.commit()
    connection.close()
```

### 4.2.3 数据处理
