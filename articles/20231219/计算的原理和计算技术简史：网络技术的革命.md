1. 排序算法：排序算法是计算机科学中的基本概念，它可以用来对数据进行排序。常见的排序算法有插入排序、选择排序、冒泡排序等。这些算法的时间复杂度和空间复杂度是不同的，需要根据具体情况选择合适的算法。

2. 搜索算法：搜索算法是用来在数据结构中查找特定元素的算法。常见的搜索算法有二分查找、深度优先搜索、广度优先搜索等。这些算法的时间复杂度和空间复杂度也是不同的，需要根据具体情况选择合适的算法。

3. 分治法：分治法是一种解决问题的方法，它将问题分解为子问题，然后递归地解决子问题。例如，快速幂算法就是一种分治法。

4. 动态规划：动态规划是一种解决问题的方法，它将问题分解为子问题，然后根据子问题的解求出原问题的解。例如，计算机硬盘的调度算法就是一种动态规划。

```python
def insert_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

2. 搜索算法的Python实现：

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

3. 分治法的Python实现：

```python
def quick_pow(x, n):
    if n == 0:
        return 1
    elif n == 1:
        return x
    else:
        return x * quick_pow(x, n - 1)
```

```python
def fib(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fib(n - 1) + fib(n - 2)
```
