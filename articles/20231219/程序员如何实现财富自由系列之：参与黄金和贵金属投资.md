## 4.1 获取黄金和贵金属价格的Python代码

```python
import requests
import json

def get_gold_price(currency='USD'):
    url = f'https://api.goldapi.io/api/XAU/{currency}'
    response = requests.get(url)
    data = json.loads(response.text)
    return data['price']

def get_silver_price(currency='USD'):
    url = f'https://api.goldapi.io/api/XAG/{currency}'
    response = requests.get(url)
    data = json.loads(response.text)
    return data['price']

def get_platinum_price(currency='USD'):
    url = f'https://api.goldapi.io/api/PTM/{currency}'
    response = requests.get(url)
    data = json.loads(response.text)
    return data['price']

def get_palladium_price(currency='USD'):
    url = f'https://api.goldapi.io/api/PAL/{currency}'
    response = requests.get(url)
    data = json.loads(response.text)
    return data['price']
```

上述代码通过调用GoldAPI的API来获取黄金、银、铂金和钯金的价格。这些价格以美元为单位。

## 4.2 计算黄金和贵金属价格变动的Python代码

```python
def calculate_price_change(price1, price2):
    change = (price1 - price2) / price2 * 100
    return change

gold_price1 = get_gold_price()
gold_price2 = get_gold_price(currency='EUR')
gold_price_change = calculate_price_change(gold_price1, gold_price2)

print(f'黄金价格变动：{gold_price_change}%')
```

上述代码通过计算两个不同货币下黄金价格的差值来计算黄金价格的变动。
