# 4.具体代码实例和详细解释说明

## 4.1 运动数据收集与处理

```python
import pandas as pd

# 读取运动数据
data = pd.read_csv('sports_data.csv')

# 数据预处理
data['distance'] = data['distance'].fillna(0)
data['heart_rate'] = data['heart_rate'].fillna(0)
data['speed'] = data['speed'].fillna(0)

# 数据归一化
data['distance'] = data['distance'] / data['distance'].max()
data['heart_rate'] = data['heart_rate'] / data['heart_rate'].max()
data['speed'] = data['speed'] / data['speed'].max()
```
