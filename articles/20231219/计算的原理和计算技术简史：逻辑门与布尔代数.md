## 3.1 布尔代数基本运算

布尔代数的基本运算包括：

1. 与（AND）：与运算是两个二值信息的运算，它的结果为true，只有当两个输入都为true。否则，结果为false。数学模型公式为：$$ a \wedge b = true \quad if \quad a = b = true \\ a \wedge b = false \quad otherwise $$

2. 或（OR）：或运算是两个二值信息的运算，它的结果为true，只要有一个或两个输入为true。否则，结果为false。数学模型公式为：$$ a \vee b = true \quad if \quad a = true \quad or \quad b = true \\ a \vee b = false \quad otherwise $$

3. 非（NOT）：非运算是一个二值信息的运算，它的结果为true，只有当输入为false。否则，结果为false。数学模型公式为：$$ \neg a = true \quad if \quad a = false \\ \neg a = false \quad otherwise $$

## 3.2 布尔代数基本定理

布尔代数的基本定理包括：

1. 交换律：与或运算的顺序不影响结果。数学模型公式为：$$ a \wedge b = b \wedge a \\ a \vee b = b \vee a $$

2. 结合律：与或运算可以分阶段进行。数学模型公式为：$$ (a \wedge b) \wedge c = a \wedge (b \wedge c) \\ (a \vee b) \vee c = a \vee (b \vee c) $$

3. 分配律：与或运算可以分配给非运算。数学模型公式为：$$ a \wedge (b \vee c) = (a \wedge b) \vee (a \wedge c) \\ a \vee (b \wedge c) = (a \vee b) \wedge (a \vee c) $$

4. 双补定理：对于任意的布尔表达式$P$，有$$ \neg P = (a \wedge \neg a) \vee (b \wedge \neg b) \vee (c \wedge \neg c) \vee ... $$

## 3.3 布尔代数基本定理的应用

布尔代数基本定理的应用包括：

1. 简化布尔表达式：通过使用交换律、结合律和分配律，我们可以将布尔表达式简化为等价的表达式。这有助于我们更好地理解和分析布尔表达式。
