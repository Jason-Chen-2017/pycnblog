常量折叠与传播技术的核心概念如下：

1. 常量折叠：常量折叠是指在表达式中，将多个相同的常量合并为一个常量。例如，在表达式`a + b + c`中，如果`a`、`b`和`c`都是相同的常量，则可以将其合并为`a + (b + c)`或`3 + (b + c)`。

2. 常量传播：常量传播是指在表达式中，将一个常量值替换为另一个常量值。例如，在表达式`a + b`中，如果`a`是一个常量值，则可以将其替换为`a + b`或`a + (b + 0)`。
