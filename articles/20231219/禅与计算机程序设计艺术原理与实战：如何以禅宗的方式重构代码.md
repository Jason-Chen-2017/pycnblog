## 4.1 冒泡排序实例

以下是一个冒泡排序的实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个实例中，我们使用了两个for循环来实现冒泡排序。在第一个for循环中，我们遍历了数组中的所有元素。在第二个for循环中，我们遍历了数组中的所有元素，并将较大的元素移动到数组的末尾。

## 4.2 选择排序实例

以下是一个选择排序的实例：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

在这个实例中，我们使用了一个for循环来实现选择排序。在第一个for循环中，我们遍历了数组中的所有元素。在第二个for循环中，我们遍历了数组中的所有元素，并将较小的元素移动到数组的开头。

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i-1
        while j >= 0 and key < arr[j]:
