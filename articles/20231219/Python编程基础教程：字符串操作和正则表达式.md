## 4.具体代码实例和详细解释说明

### 4.1 字符串操作的具体代码实例

```python
# 字符串的创建
str1 = 'Hello, World!'
print(str1)

# 字符串的修改
str1 = 'Hello, World!'
str1 = str1.replace('World', 'Python')
print(str1)

# 字符串的查找
str1 = 'Hello, World!'
index = str1.find('World')
print(index)

# 字符串的替换
str1 = 'Hello, World!'
str1 = str1.replace('World', 'Python')
print(str1)

# 字符串的分割
str1 = 'Hello, World!'
list1 = str1.split(',')
print(list1)

# 字符串的合并
str1 = 'Hello'
str2 = 'World'
str3 = str1 + ' ' + str2
print(str3)
```

### 4.2 正则表达式的具体代码实例

```python
import re

# 字符串的匹配
str1 = 'Hello, World!'
match = re.match(r'Hello, World!', str1)
print(match)

# 字符串的替换
str1 = 'Hello, World!'
str1 = re.sub(r'World', 'Python', str1)
print(str1)

# 字符串的分割
str1 = 'Hello, World!'
list1 = re.split(r',', str1)
print(list1)

# 字符串的提取
str1 = 'Hello, World!'
list2 = re.findall(r'[A-Za-z]', str1)
print(list2)
```
