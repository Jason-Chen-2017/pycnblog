# 2.核心概念与联系

在深入学习分布式系统之前，我们需要了解一些基本的概念和联系。以下是一些核心概念：

1. **分布式系统**：分布式系统是指由多个独立的计算机节点组成的系统，这些节点通过网络进行通信，共同完成某个任务。

2. **节点**：分布式系统中的每个计算机节点都称为节点。节点可以是服务器、个人电脑、移动设备等。

3. **网络**：节点之间的通信是通过网络实现的。网络可以是局域网（LAN）、广域网（WAN）或者互联网。

4. **数据一致性**：在分布式系统中，多个节点需要保持数据的一致性。这意味着，在任何时刻，任何节点查询到的数据都应该与其他节点的数据一致。

5. **容错性**：分布式系统需要具备容错性，即在某些节点出现故障的情况下，系统仍然能够正常运行。

6. **负载均衡**：分布式系统需要实现负载均衡，即在多个节点之间分散任务，以提高整体性能和可靠性。
