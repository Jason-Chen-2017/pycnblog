## 4.3 实现兼容性

```python
class FileOperation:
    def read_file(self, file_path):

            return f.read()

    def write_file(self, file_path, content):
        with open(file_path, 'w') as f:
            f.write(content)

    def delete_file(self, file_path):
        os.remove(file_path)
```
