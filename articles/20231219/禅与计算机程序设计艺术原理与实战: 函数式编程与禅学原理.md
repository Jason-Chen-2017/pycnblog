```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)
```

我们可以使用尾递归优化来解决这个问题。尾递归优化是一种编译器优化技术，它可以避免递归导致的栈溢出问题。我们可以使用以下代码来实现尾递归优化的阶乘函数：

```python

    if n == 0:
        return accumulator
    else:
        return factorial_tail_recursive(n - 1, n * accumulator)
```
