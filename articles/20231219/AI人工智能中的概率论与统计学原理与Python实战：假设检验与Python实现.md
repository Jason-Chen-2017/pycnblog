## 4.2假设检验示例

假设我们有一组数据，表示一组人的身高（以厘米为单位）：

```
data = [170, 165, 180, 175, 168, 185, 172, 169, 178, 171]
```

我们希望测试这组数据是否来自于一个均值为175的正态分布。这个问题可以用以下Null假设表示：

```
H0: μ = 175
```

我们可以使用t检验方法进行这个假设检验。首先，我们需要计算样本均值和样本方差：

```
import numpy as np
import pandas as pd
import scipy.stats as stats

data = np.array([170, 165, 180, 175, 168, 185, 172, 169, 178, 171])
sample_mean = np.mean(data)
sample_variance = np.var(data)
```

接下来，我们可以计算检验统计量。在这个例子中，我们将使用t检验方法，因为样本数量较小：

```
t_statistic, p_value = stats.ttest_1samp(data, 175)
```

现在，我们可以比较检验统计量与临界值。假设我们使用的是两侧检验，并且概率水平为0.05。我们可以从t分布表中获取临界值。在这个例子中，临界值为2.201。

```
if p_value < 0.05:
    print("Reject H0")
else:
    print("Fail to reject H0")
```

在这个例子中，我们的p值为0.06，小于0.05，因此我们拒绝Null假设。这意味着我们有足够的证据证明这组数据的均值不等于175。

# 6.附录常见问题与解答

在本节中，我们将回答一些常见问题及其解答。

## 6.1问题1：什么是假设检验？

答案：假设检验是一种统计方法，用于确定一个或多个假设是否成立。假设检验通常包括设定Null假设、选择统计检验方法、计算检验统计量、比较检验统计量与临界值以及结论的步骤。假设检验的目的是通过对数据进行分析，来决定是否接受或拒绝Null假设。

## 6.2问题2：为什么我们需要假设检验？

答案：我们需要假设检验，因为在实际应用中，我们通常不能直接观察到事实。假设检验可以帮助我们根据数据来评估假设的可信度，从而做出更有据据的决策。假设检验还可以帮助我们确定一个或多个假设是否成立，从而指导我们的研究和实践。

## 6.3问题3：如何选择合适的统计检验方法？

答案：选择合适的统计检验方法需要考虑以下几个因素：

1. 问题类型：根据问题的类型，选择合适的统计检验方法。例如，如果问题涉及到两组数据之间的比较，可以考虑使用t检验或秩和检验。
2. 数据类型：根据数据的类型，选择合适的统计检验方法。例如，如果数据是连续的，可以考虑使用t检验；如果数据是离散的，可以考虑使用卡方检验。
3. 假设：根据问题的假设，选择合适的统计检验方法。例如，如果问题涉及到均值的比较，可以考虑使用t检验；如果问题涉及到比例的比较，可以考虑使用卡方检验。

## 6.4问题4：如何解释p值？

答案：p值是一个概率，表示在接受Null假设的情况下，观察到的数据发生的概率。通常，我们使用两侧检验，并将p值设为0.05。如果p值小于0.05，我们认为有足够的证据拒绝Null假设。如果p值大于0.05，我们认为没有足够的证据拒绝Null假设。p值越小，证据越强，越有可能拒绝Null假设。
