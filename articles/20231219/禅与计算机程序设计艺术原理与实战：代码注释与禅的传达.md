## 3.1 排序算法

排序算法是计算机科学中最基本的算法之一，它的目的是将一组数据按照某种顺序排列。常见的排序算法有：冒泡排序、选择排序、插入排序、归并排序等。

### 3.1.1 冒泡排序

冒泡排序是一种简单的排序算法，它的基本思想是通过多次比较和交换元素来将较大的元素逐渐移动到数组的末尾。

具体的操作步骤如下：

1.从第一个元素开始，与后面的每个元素进行比较。如果当前元素大于后面的元素，则交换它们的位置。

2.重复第一步，直到整个数组被排序。

冒泡排序的时间复杂度为O(n^2)，其中n是数组的长度。

### 3.1.2 选择排序

选择排序是一种简单的排序算法，它的基本思想是通过多次选择最小（或最大）的元素并将其放在数组的正确位置来将数组排序。

具体的操作步骤如下：

1.从第一个元素开始，找出最小的元素。

2.将最小的元素与第一个元素交换位置。

3.重复第一步和第二步，直到整个数组被排序。

选择排序的时间复杂度为O(n^2)，其中n是数组的长度。

### 3.1.3 插入排序

插入排序是一种简单的排序算法，它的基本思想是将一个元素插入到已经排好序的子数组中的正确位置。

具体的操作步骤如下：

1.将第一个元素视为有序的子数组。

2.从第二个元素开始，将它与有序子数组中的元素进行比较。如果当前元素小于有序子数组中的元素，则将其插入到有序子数组的正确位置。

3.重复第二步，直到整个数组被排序。

插入排序的时间复杂度为O(n^2)，其中n是数组的长度。

### 3.1.4 归并排序

归并排序是一种高效的排序算法，它的基本思想是将一个大的数组分割成多个小的数组，然后将这些小的数组进行递归排序，最后将排序的小数组合并成一个大数组。

具体的操作步骤如下：

1.将数组分割成两个部分，直到每个部分只有一个元素。

2.将每个部分进行递归排序。

3.将排序的部分合并成一个大数组。

归并排序的时间复杂度为O(nlogn)，其中n是数组的长度。

## 3.2 搜索算法

搜索算法是计算机科学中另一个重要的算法类型，它的目的是在一个数据结构中找到满足某个条件的元素。常见的搜索算法有：线性搜索、二分搜索等。

### 3.2.1 线性搜索

线性搜索是一种简单的搜索算法，它的基本思想是通过遍历数据结构中的每个元素来找到满足某个条件的元素。

具体的操作步骤如下：

1.从第一个元素开始，逐个检查每个元素是否满足某个条件。

2.如果当前元素满足条件，则返回它的位置。

3.如果当前元素不满足条件，则继续检查下一个元素。

4.重复第一步和第二步，直到找到满足条件的元素或者遍历完整个数据结构。

线性搜索的时间复杂度为O(n)，其中n是数据结构的长度。

### 3.2.2 二分搜索

二分搜索是一种高效的搜索算法，它的基本思想是将一个有序的数据结构分割成两个部分，然后将中间的元素与目标元素进行比较，如果中间的元素与目标元素相等，则返回它的位置；如果中间的元素小于目标元素，则将搜索范围设置为右半部分；如果中间的元素大于目标元素，则将搜索范围设置为左半部分。

具体的操作步骤如下：

1.将数据结构分割成两个部分，左半部分和右半部分。

2.将中间的元素与目标元素进行比较。

3.如果中间的元素与目标元素相等，则返回它的位置。

4.如果中间的元素小于目标元素，则将搜索范围设置为右半部分。

5.如果中间的元素大于目标元素，则将搜索范围设置为左半部分。

6.重复第一步和第二步，直到找到满足条件的元素或者搜索范围为空。

二分搜索的时间复杂度为O(logn)，其中n是数据结构的长度。

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```
