# 2.核心概念与联系

深度学习是一种人工智能技术，它旨在模仿人类大脑对数据的处理方式。深度学习算法可以自动学习特征，从而无需人工指导，这使得它在处理大规模、高维度的数据集上具有显著优势。深度学习的核心概念包括：

## 3.1 神经网络基础

神经网络是深度学习的基本结构，由多层神经元组成。每个神经元接收来自前一层的输入，通过权重和偏置进行加权求和，然后通过激活函数得到输出。常见的激活函数包括 sigmoid、tanh 和 ReLU。

### 3.1.1 前向传播

在前向传播过程中，输入数据通过多层神经元逐层传递，直到得到最后一层的输出。具体步骤如下：

1. 将输入数据输入到第一层神经元。
2. 每个神经元对输入数据进行加权求和，然后通过激活函数得到输出。
3. 输出作为下一层神经元的输入，直到得到最后一层的输出。

### 3.1.2 反向传播

反向传播是一种优化算法，用于最小化损失函数。具体步骤如下：

1. 计算输出与真实值之间的损失。
2. 通过回传损失，计算每个神经元的误差。
3. 计算每个权重和偏置的梯度。
4. 更新权重和偏置，使损失最小化。

## 3.2 卷积神经网络（CNN）

卷积神经网络（CNN）是一种特殊类型的神经网络，主要用于图像处理和分类。CNN的核心组件是卷积层和池化层。

### 3.2.1 卷积层

卷积层通过卷积核对输入图像进行卷积，以提取特征。具体步骤如下：

1. 将卷积核应用于输入图像。
2. 计算卷积核与输入图像的内积。
3. 将内积累加起来，得到卷积后的特征图。

### 3.2.2 池化层

池化层用于减少特征图的尺寸，同时保留重要信息。常见的池化方法包括最大池化和平均池化。

## 3.3 循环神经网络（RNN）

循环神经网络（RNN）是一种能够处理序列数据的神经网络。RNN的核心特点是每个时间步都可以访问之前的信息。

### 3.3.1 LSTM

长短期记忆（Long Short-Term Memory, LSTM）是一种特殊类型的RNN，能够长距离保存信息。LSTM的核心组件是门（gate），包括输入门、遗忘门和输出门。

### 3.3.2 GRU

 gates递归单元（Gated Recurrent Unit, GRU）是一种更简化的RNN结构，相比LSTM更易于训练。GRU将输入门和遗忘门合并为更简单的更新门。
