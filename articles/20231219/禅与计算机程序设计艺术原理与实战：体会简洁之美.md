## 3.1 排序算法

排序算法是计算机程序设计中最常见的任务之一。我们将讨论两种常见的排序算法：冒泡排序和快速排序。

### 3.1.1 冒泡排序

冒泡排序是一种简单的排序算法，它通过多次比较和交换元素来达到排序的目的。具体操作步骤如下：

1. 从第一个元素开始，与后续的每个元素进行比较。如果当前元素大于后续元素，则交换它们的位置。
2. 重复第一步，直到整个数组被排序。

冒泡排序的时间复杂度为O(n^2)，其中n是数组的长度。

### 3.1.2 快速排序

快速排序是一种更高效的排序算法，它使用分治法（Divide and Conquer）的思想来实现。具体操作步骤如下：

1. 从数组的第一个元素开始，选择一个基准元素。
2. 将其他元素分为两组：一个包含小于基准元素的元素，另一个包含大于基准元素的元素。
3. 递归地对两个组进行快速排序。
4. 将基准元素放在正确的位置，并将两个组合并起来。

快速排序的平均时间复杂度为O(nlogn)，其中n是数组的长度。

## 3.2 搜索算法

搜索算法是计算机程序设计中另一个常见的任务。我们将讨论两种常见的搜索算法：深度优先搜索和广度优先搜索。

### 3.2.1 深度优先搜索

深度优先搜索（Depth-First Search，DFS）是一种搜索算法，它首先深入一个路径，然后回溯到上一个节点。具体操作步骤如下：

1. 从起始节点开始，访问它的所有邻居节点。
2. 从其中一个邻居节点开始，重复第一步。
3. 如果无法继续深入，则回溯到上一个节点并尝试其他邻居节点。

### 3.2.2 广度优先搜索

广度优先搜索（Breadth-First Search，BFS）是另一种搜索算法，它首先探索最近的节点，然后逐渐扩展到更远的节点。具体操作步骤如下：

1. 从起始节点开始，将它放入一个队列中。
2. 从队列中取出一个节点，访问它的所有邻居节点。
3. 将这些邻居节点放入队列中，并重复第二步。

## 4.1 冒泡排序实例

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

在这个例子中，我们使用了冒泡排序算法来对一个整数数组进行排序。我们将数组中的每个元素与后续的元素进行比较，如果当前元素大于后续元素，则交换它们的位置。这个算法的时间复杂度为O(n^2)，因此在处理大型数组时可能会较慢。

## 4.2 快速排序实例

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)
```

在这个例子中，我们使用了快速排序算法来对一个整数数组进行排序。我们首先选择一个基准元素，然后将其他元素分为两组：一个包含小于基准元素的元素，另一个包含大于基准元素的元素。然后递归地对两个组进行快速排序，并将基准元素放在正确的位置。这个算法的平均时间复杂度为O(nlogn)，因此在处理大型数组时更高效。
