## 2.2面向对象编程的基本概念

面向对象编程的基本概念包括：

- 对象（Object）：是一种数据类型，包含数据和方法。
- 类（Class）：是对象的模板，定义对象的属性和方法。
- 继承（Inheritance）：是一种代码重用方法，允许一个类从另一个类继承属性和方法。
- 多态（Polymorphism）：是一种允许不同类的对象在运行时以相同的方式交互的机制。

# 4.具体代码实例和详细解释说明

在这部分，我们将通过一个具体的代码实例来说明禅与面向对象编程的应用。我们将实现一个简单的动物类层次结构，包括动物、狗和猫这三个类。

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError("Subclasses must implement this method")

class Dog(Animal):
    def speak(self):
        return "Woof!"

class Cat(Animal):
    def speak(self):
        return "Meow!"

def main():
    dog = Dog("Rex")
    cat = Cat("Whiskers")

    print(f"{dog.name} says {dog.speak()}")
    print(f"{cat.name} says {cat.speak()}")

if __name__ == "__main__":
    main()
```

在这个代码实例中，我们首先定义了一个基类`Animal`，它包含了一个名字属性和一个抽象的`speak`方法。然后我们定义了两个子类`Dog`和`Cat`，它们 respective地实现了`speak`方法。最后，我们在`main`函数中创建了一个狗和猫的对象，并调用了它们的`speak`方法，以展示它们的不同的声音。

这个代码实例展示了面向对象编程的核心概念，包括类、对象、继承和多态。同时，它也展示了如何将禅的思想应用到编程中，以提高代码的质量和效率。
