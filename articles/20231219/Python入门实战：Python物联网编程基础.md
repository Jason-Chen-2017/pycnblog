## 4.1 代码实例：智能家居设备控制

在这个例子中，我们将通过Python编程来控制智能家居设备，例如开关、光线、温度等。具体的代码实例如下：

```python
import socket

# 创建一个socket对象
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# 设置socket对象的连接参数
s.bind(('localhost', 8888))
s.listen(5)

# 等待客户端的连接请求
client, addr = s.accept()

# 接收客户端发送过来的数据
data = client.recv(1024)

# 解析数据，获取设备类型和操作命令
device_type, command = data.decode().split(',')

# 根据设备类型和操作命令实现相应的控制功能
if device_type == 'switch':
    if command == 'on':
        print('开关打开')
    elif command == 'off':
        print('开关关闭')
elif device_type == 'light':
    if command == 'bright':
        print('光线亮')
    elif command == 'dim':
        print('光线暗')
elif device_type == 'temp':
    if command == 'up':
        print('温度上升')
    elif command == 'down':
        print('温度下降')

# 发送处理结果回到客户端
client.send(b'处理成功')

# 关闭客户端连接
client.close()
```

在这个代码实例中，我们首先创建了一个socket对象，并设置了连接参数。然后，我们等待客户端的连接请求，当客户端连接成功后，我们可以接收客户端发送过来的数据。接下来，我们根据设备类型和操作命令实现相应的控制功能，并将处理结果发送回客户端。最后，我们关闭客户端连接。
