## 4.1 排序算法

排序算法是一种常用的算法，它用于对数据进行排序。Python提供了多种排序算法，包括冒泡排序、插入排序、选择排序、归并排序等。

以下是一个简单的冒泡排序算法的实现：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在上面的代码中，我们使用了两个for循环来实现冒泡排序算法。内部循环用于比较相邻的元素，并将较大的元素交换其位置。外部循环用于重复内部循环，直到数据被完全排序。

## 4.2 搜索算法

搜索算法是一种常用的算法，它用于查找数据中的某个值。Python提供了多种搜索算法，包括线性搜索、二分搜索等。

以下是一个简单的线性搜索算法的实现：

```python
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1
```

在上面的代码中，我们使用了一个for循环来实现线性搜索算法。内部循环用于遍历数据，并检查当前元素是否等于目标值。如果找到目标值，则返回其索引；如果没有找到目标值，则返回-1。

## 4.3 数学算法

数学算法是一种常用的算法，它用于解决数学问题。Python提供了多种数学算法，包括最大公约数、最小公倍数、欧几里得算法等。

以下是一个简单的最大公约数算法的实现：

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a
```

在上面的代码中，我们使用了一个while循环来实现最大公约数算法。内部循环用于将a和b的值替换为b和a mod b的值，直到b为0。当b为0时，a的值就是最大公约数。
