## 排序算法

排序算法的主要目标是将一个数据集中的元素按照一定的顺序进行排序。常见的排序算法有：

1. 冒泡排序
2. 选择排序
3. 插入排序
4. 希尔排序
5. 快速排序
6. 归并排序

### 冒泡排序

冒泡排序是一种简单的排序算法，它的原理是通过多次比较和交换元素来将较大的元素逐渐移动到数组的末尾。具体的操作步骤如下：

1. 从第一个元素开始，与后面的每个元素进行比较。
2. 如果当前元素大于后面的元素，则交换它们的位置。
3. 重复上述操作，直到整个数组被排序。

### 选择排序

选择排序是一种简单的排序算法，它的原理是通过多次选择最小（或最大）的元素来将其放入正确的位置。具体的操作步骤如下：

1. 从第一个元素开始，找到最小的元素。
2. 将最小的元素与第一个元素交换位置。
3. 重复上述操作，直到整个数组被排序。

### 插入排序

插入排序是一种简单的排序算法，它的原理是通过将一个元素插入到已经排好序的子数组中，从而将整个数组排序。具体的操作步骤如下：

1. 将第一个元素视为已排序的子数组。
2. 从第二个元素开始，将它与已排序的子数组中的元素进行比较。
3. 如果当前元素小于已排序的子数组中的元素，则将其插入到已排序的子数组中的正确位置。
4. 重复上述操作，直到整个数组被排序。

### 希尔排序

希尔排序是一种插入排序的变种，它的原理是通过将一个数组分为多个子数组，然后对每个子数组进行插入排序来将整个数组排序。具体的操作步骤如下：

1. 选择一个大小为n/2的初始步长。
2. 将数组分为多个子数组，然后对每个子数组进行插入排序。
3. 逐渐减小步长，直到步长为1。
4. 重复上述操作，直到整个数组被排序。

### 快速排序

快速排序是一种分治法的排序算法，它的原理是通过将一个数组分为两个部分，然后对每个部分进行快速排序来将整个数组排序。具体的操作步骤如下：

1. 选择一个基准元素。
2. 将所有小于基准元素的元素放在其左边，将所有大于基准元素的元素放在其右边。
3. 对左边和右边的子数组进行快速排序。
4. 重复上述操作，直到整个数组被排序。

### 归并排序

归并排序是一种分治法的排序算法，它的原理是通过将一个数组分为两个部分，然后对每个部分进行归并排序来将整个数组排序。具体的操作步骤如下：

1. 将数组分为两个部分。
2. 对每个部分进行归并排序。
3. 将两个排序好的部分合并成一个排序好的数组。

## 搜索算法

搜索算法的主要目标是在一个数据集中找到满足某个条件的元素。常见的搜索算法有：

1. 线性搜索
2. 二分搜索

### 线性搜索

线性搜索是一种简单的搜索算法，它的原理是通过遍历一个数据集中的每个元素来找到满足某个条件的元素。具体的操作步骤如下：

1. 从第一个元素开始，逐个检查每个元素是否满足某个条件。
2. 如果当前元素满足条件，则返回该元素。
3. 如果当前元素不满足条件，则继续检查下一个元素。
4. 重复上述操作，直到找到满足条件的元素或者遍历完整个数据集。

### 二分搜索

二分搜索是一种有效的搜索算法，它的原理是通过将一个数据集分为两个部分，然后对每个部分进行二分搜索来找到满足某个条件的元素。具体的操作步骤如下：

1. 将数组分为两个部分。
2. 找到中间元素。
3. 如果中间元素满足条件，则返回该元素。
4. 如果中间元素不满足条件，则根据中间元素是否大于满足条件的元素来决定是将搜索范围设置为左半部分还是右半部分。
5. 重复上述操作，直到找到满足条件的元素或者搜索范围为空。

## 图算法

图算法的主要目标是在一个图中找到满足某个条件的节点或者边。常见的图算法有：

1. 最短路径算法
2. 最短路径算法
3. 最大流算法
4. 最小割算法

### 最短路径算法

最短路径算法的主要目标是在一个图中找到从一个节点到另一个节点的最短路径。常见的最短路径算法有：

1. 迪杰斯特拉算法
2. 贝尔曼福特算法

迪杰斯特拉算法的原理是通过将一个图分为多个部分，然后对每个部分进行迪杰斯特拉算法来找到从一个节点到另一个节点的最短路径。具体的操作步骤如下：

1. 将第一个节点视为已探索的部分。
2. 从第一个节点开始，将所有未探索的邻居节点加入到优先级队列中。
3. 从优先级队列中取出一个节点，将它视为已探索的部分。
4. 对已探索的部分的节点进行关键性质判断，如果满足条件，则将其邻居节点加入到优先级队列中。
5. 重复上述操作，直到找到目标节点或者优先级队列为空。

贝尔曼福特算法的原理是通过将一个图分为多个部分，然后对每个部分进行贝尔曼福特算法来找到从一个节点到另一个节点的最短路径。具体的操作步骤如下：

1. 将所有节点的距离设为无穷大。
2. 将第一个节点的距离设为0。
3. 对所有节点进行关键性质判断，如果满足条件，则更新其邻居节点的距离。
4. 重复上述操作，直到所有节点的距离都不变。

### 最大流算法

最大流算法的主要目标是在一个流网络中找到最大的流。常见的最大流算法有：

1. 福尔曼算法
2. 弗拉斯曼算法

福尔曼算法的原理是通过将一个流网络分为多个部分，然后对每个部分进行福尔曼算法来找到最大的流。具体的操作步骤如下：

1. 将第一个节点视为源节点，将第二个节点视为沿流节点。
2. 从源节点开始，将所有未探索的邻居节点加入到优先级队列中。
3. 从优先级队列中取出一个节点，将它视为当前节点。
4. 对当前节点的所有邻居节点进行关键性质判断，如果满足条件，则将其视为沿流节点，并更新其流量。
5. 重复上述操作，直到所有节点的流量都不变。

弗拉斯曼算法的原理是通过将一个流网络分为多个部分，然后对每个部分进行弗拉斯曼算法来找到最大的流。具体的操作步骤如下：

1. 将所有节点的流量设为0。

4. 重复上述操作，直到所有节点的流量都不变。

### 最小割算法

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

### 插入排序

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i-1
        while j >=0 and key < arr[j]:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr
```

### 希尔排序

```python
def shell_sort(arr):
    n = len(arr)
    gap = n//2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j-gap] > temp:
                arr[j] = arr[j-gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr
```

### 快速排序

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr)//2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

### 归并排序

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr)//2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

```python
import heapq

def dijkstra(graph, start):
    distances = {node: float('inf') for node in graph}
    distances[start] = 0
    pq = [(0, start)]
    while pq:
        current_distance, current_node = heapq.heappop(pq)
        if current_distance > distances[current_node]:
            continue
        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))
    return distances
```

```python
    distances[start] = 0
    for _ in range(len(graph)-1):
        for node in graph:
```

```python
from collections import deque

def ford_fulkerson(graph, source, target, flow):
    max_flow = 0
    while flow > 0:
        levels = [-1]*(len(graph))
        levels[source] = 0
        queue = deque([source])
        while queue:
            current = queue.popleft()
            for neighbor, capacity in graph[current].items():
                if capacity > 0 and levels[neighbor] == -1:
                    levels[neighbor] = levels[current] + 1
                    queue.append(neighbor)
        if levels[target] == -1:
            return max_flow
        path = [target]
        while path[-1] != source:
            current = path[-1]
            for neighbor, capacity in graph[current].items():
                if capacity > 0 and levels[neighbor] == levels[current] - 1:
                    path.append(neighbor)
        path.append(source)
        path.reverse()
        augmenting_path_capacity = flow
        for i in range(len(path)-1):
            current = path[i]
            neighbor = path[i+1]
            augmenting_path_capacity = min(augmenting_path_capacity, graph[current][neighbor])
        for i in range(len(path)-1):
            current = path[i]
            neighbor = path[i+1]
            graph[current][neighbor] -= augmenting_path_capacity
            graph[neighbor][current] += augmenting_path_capacity
        max_flow += augmenting_path_capacity
    return max_flow
```

```python
    stack = [source]
    while stack:
        current = stack.pop()
```
