## 4.1 算法实例

我们选取一个简单的排序算法——冒泡排序，来作为算法实例。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

冒泡排序是一种简单的排序算法，它的基本思想是通过多次比较和交换元素，将数组中的元素按照顺序排列。在上面的代码中，我们首先定义了一个函数`bubble_sort`，它接收一个数组`arr`作为参数。然后，我们使用两个嵌套的`for`循环来实现冒泡排序的过程。在内部循环中，我们比较当前元素和下一个元素，如果当前元素大于下一个元素，则交换它们的位置。最后，我们返回排序后的数组。

## 4.2 数据结构实例

我们选取一个简单的数据结构——链表，来作为数据结构实例。

```python
class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, value):
        if not self.head:
            self.head = Node(value)
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = Node(value)

    def print_list(self):
        current = self.head
        while current:
            print(current.value)
            current = current.next
```

链表是一种线性数据结构，它由一个或多个节点组成。每个节点包含一个值和一个指向下一个节点的指针。在上面的代码中，我们首先定义了一个`Node`类，它包含一个值和一个指针。然后，我们定义了一个`LinkedList`类，它包含一个头节点。我们实现了两个方法`append`和`print_list`，分别用于向链表中添加节点和打印链表。
