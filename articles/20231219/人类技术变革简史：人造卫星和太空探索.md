## 1.2 人造卫星的诞生
人造卫星是在地球的引力下稳定运行的天体，通常由金属或其他材料制成。它们的轨道可以是地球轨道、月轨道或其他轨道，用于各种目的，如气象观测、导航、通信、军事等。

人造卫星的诞生可以追溯到1957年，当时苏联发射了世界上第一个人造卫星——“斯莫斯顿”。从此，人类进入了太空时代。

$$

$$

2.轨道元件：
$$
a = \frac{p^2}{G(M_e+M_m)}
$$
$$
e = \sqrt{1+2\frac{E}{G(M_e+M_m)}a}
$$
$$
T = 2\pi\sqrt{\frac{a^3}{G(M_e+M_m)}}

## 4.1 卫星轨道计算
我们可以使用Python编程语言来编写一个简单的卫星轨道计算程序。以下是代码实例：

```python
import numpy as np

def satellite_orbit(semi_major_axis, eccentricity):
    periapsis = semi_major_axis * (1 - eccentricity)
    apoapsis = semi_major_axis * (1 + eccentricity)
    return periapsis, apoapsis

semi_major_axis = 6378.1 * 1000  # 地球半径1000公里
eccentricity = 0.0  # 无倾斜
periapsis, apoapsis = satellite_orbit(semi_major_axis, eccentricity)
print("卫星最近点距地球{}km，最远点距地球{}km".format(periapsis, apoapsis))
```

在这个例子中，我们首先导入了NumPy库，然后定义了一个`satellite_orbit`函数，用于计算卫星轨道的最近点和最远点。接着，我们设定了地球半径为1000公里，倾斜率为0（无倾斜轨道），然后调用`satellite_orbit`函数计算最近点和最远点距离。最后，我们打印出结果。
