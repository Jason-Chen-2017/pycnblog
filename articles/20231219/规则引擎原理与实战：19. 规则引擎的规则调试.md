# 4.具体代码实例和详细解释说明

## 4.1 规则引擎的规则调试代码实例
以下是一个简单的规则引擎的规则调试代码实例：

```python
from jython_rules.core import RuleEngine
from jython_rules.rule import Rule

# 创建规则引擎实例
rule_engine = RuleEngine()

# 创建规则实例
rule1 = Rule("IF age >= 18 AND age <= 60 THEN is_adult = true")
rule2 = Rule("IF is_adult = true THEN can_vote = true")

# 添加规则到规则引擎
rule_engine.add_rule(rule1)
rule_engine.add_rule(rule2)

# 设置事实数据
fact = {"age": 30}

# 执行规则引擎
rule_engine.fire(fact)

# 获取结果
print(fact)
```

## 4.2 规则引擎的规则调试代码解释
1. 首先，我们导入了`jython_rules`库中的`RuleEngine`和`Rule`类。
2. 然后，我们创建了一个规则引擎实例`rule_engine`。
3. 接着，我们创建了两个规则实例`rule1`和`rule2`。
4. 之后，我们将这两个规则添加到规则引擎中。
5. 然后，我们设置了事实数据`fact`。
6. 最后，我们执行了规则引擎，并获取了结果。
