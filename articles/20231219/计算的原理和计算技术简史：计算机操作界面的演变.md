1. 排序算法实例：以冒泡排序算法为例，下面是一个简单的Python实现：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

2. 搜索算法实例：以二分搜索算法为例，下面是一个简单的Python实现：

```python
def binary_search(arr, target):
    left, right = 0, len(arr)-1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

3. 计算机网络协议实例：以HTTP协议为例，下面是一个简单的Python实现：

```python
import socket

def http_request(url, method='GET', headers={}):
    host, path = url.split('/', 1)
    port = 80 if 'https' not in url else 443
    headers['Host'] = host
    headers['User-Agent'] = 'Mozilla/5.0'
    headers = '\r\n'.join(f'{k}: {v}' for k, v in headers.items())
    request = f'{method} {path} HTTP/1.1\r\nContent-Length: 0\r\n\r\n'
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    sock.connect((host, port))
    sock.sendall(request.encode('ascii'))
    response = sock.recv(4096)
    sock.close()
    return response
```
