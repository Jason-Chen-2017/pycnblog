# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来展示编程之禅的实践。我们将实现一个简单的排序算法：冒泡排序。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个例子中，我们通过冥想和专注来理解问题。我们注意到，冒泡排序的核心思想是通过多次遍历数组，将较大的元素逐步移动到数组的末尾。我们使用两个嵌套循环来实现这个过程。

在编写代码时，我们直接体验了问题和解决方案。我们注意到，冒泡排序的时间复杂度是O(n^2)，因此在处理大型数组时可能不是最佳选择。然而，这个算法简单易懂，对于理解排序算法的基本概念是有帮助的。
