例1：求解线性方程组的解

线性方程组是数学建模中最常见的问题类型，我们可以通过矩阵求解方法来求解线性方程组的解。以下是一个3元素线性方程组的例子：

x + 2y - z = 1
x - y + 2z = 2
2x + y - z = 3

我们可以将这个线性方程组表示为矩阵形式：

[ 1  2 -1 | 1 ]
[ 1 -1  2 | 2 ]
[ 2  1 -1 | 3 ]

通过矩阵求解方法，我们可以得到线性方程组的解：

x = 1
y = 2
z = 1

例2：求解二次方程的解

二次方程是数学建模中另一个常见的问题类型，我们可以通过求解方程的解来解决二次方程问题。以下是一个二次方程的例子：

x^2 - 5x + 6 = 0

我们可以将这个二次方程表示为：

(x - 3)(x - 2) = 0

通过求解方程的解，我们可以得到二次方程的解：

x = 3
x = 2
