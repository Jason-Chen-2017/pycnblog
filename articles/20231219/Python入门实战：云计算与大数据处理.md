## 4.1 云计算代码实例

### 4.1.1 Boto示例

Boto是一个用于访问Amazon Web Services（AWS）的Python库。以下是一个使用Boto访问AWS S3服务的示例：

```python
import boto3

# 创建一个S3客户端
s3 = boto3.client('s3')

# 上传一个文件到S3
s3.upload_file('test.txt', 'my-bucket', 'test.txt')

# 下载一个文件从S3
s3.download_file('my-bucket', 'test.txt', 'test_downloaded.txt')
```

### 4.1.2 Google Cloud Python Client Library示例

Google Cloud Python Client Library是一个用于访问Google Cloud Platform的Python库。以下是一个使用Google Cloud Python Client Library访问Google Cloud Storage服务的示例：

```python
from google.cloud import storage

# 创建一个Cloud Storage客户端
client = storage.Client()

# 上传一个文件到Cloud Storage
bucket = client.bucket('my-bucket')
blob = bucket.blob('test.txt')
blob.upload_from_filename('test.txt')

# 下载一个文件从Cloud Storage
blob = bucket.blob('test.txt')
blob.download_to_filename('test_downloaded.txt')
```

## 4.2 大数据处理代码实例

### 4.2.1 Hadoop示例

Hadoop是一个用于分布式大数据处理的开源框架。以下是一个使用Hadoop的Python库MapReduce示例：

```python
from hadoop.mapreduce import MapReduce

# 定义一个Map函数
def map_func(key, value):
    for word in value.split():
        yield (word, 1)

# 定义一个Reduce函数
def reduce_func(key, values):
    count = 0
    for value in values:
        count += value
    yield (key, count)

# 创建一个MapReduce实例
mr = MapReduce()

# 设置输入文件和输出文件
mr.set_input('input.txt')
mr.set_output('output')

# 设置Map和Reduce函数
mr.set_map_func(map_func)
mr.set_reduce_func(reduce_func)

# 运行MapReduce
mr.run()
```
