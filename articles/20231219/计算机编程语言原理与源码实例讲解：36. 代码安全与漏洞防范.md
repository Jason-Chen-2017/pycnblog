# 2.核心概念与联系

在讨论代码安全与漏洞防范之前，我们需要了解一些核心概念。首先，我们需要了解什么是漏洞（Vulnerability），以及如何将其与安全（Security）和隐蔽性（Stealth）区分开来。此外，我们还需要了解一些常见的攻击方法，如跨站请求伪造（Cross-site request forgery）、SQL注入（SQL injection）和代码注入（Code injection）等。

## 2.1 漏洞、安全和隐蔽性

漏洞是指计算机系统或软件中存在的一种已知的弱点，可以被攻击者利用来违反系统的安全策略。安全是指计算机系统或软件能够保护数据、资源和用户免受未经授权的访问和破坏的能力。隐蔽性是指攻击者无法被目标系统或软件的管理员及时发现的能力。

## 2.2 常见攻击方法

1. 跨站请求伪造（Cross-site request forgery）：这种攻击方法涉及到攻击者诱使用户执行未经授权的操作，例如在用户名和密码已经登录的情况下，攻击者诱使用户执行一些不合理的操作。

2. SQL注入（SQL injection）：这种攻击方法涉及到攻击者通过输入特殊的SQL语句来绕过程序的安全检查，从而访问或修改数据库中的数据。

3. 代码注入（Code injection）：这种攻击方法涉及到攻击者通过输入恶意代码来绕过程序的安全检查，从而执行恶意操作。

## 4.1 哈希函数实例

我们可以使用Python的hashlib库来实现一个简单的哈希函数。以下是一个MD5哈希函数的实例：

```python
import hashlib

def md5_hash(data):
    return hashlib.md5(data.encode()).hexdigest()

data = "Hello, World!"
print(md5_hash(data))
```

在这个实例中，我们使用了MD5哈希函数来计算字符串“Hello, World!”的哈希值。通过运行这个代码，我们可以看到哈希值为`5eb55452184b0b8e1750d0127f37b8a5`。

## 4.2 对称加密实例

我们可以使用Python的cryptography库来实现一个简单的对称加密。以下是一个AES加密和解密的实例：

```python
from cryptography.fernet import Fernet

# 生成一个密钥
key = Fernet.generate_key()

# 初始化加密器
cipher_suite = Fernet(key)

# 加密数据
data = "Hello, World!"
encrypted_data = cipher_suite.encrypt(data.encode())
print(encrypted_data)

# 解密数据
decrypted_data = cipher_suite.decrypt(encrypted_data).decode()
print(decrypted_data)
```

在这个实例中，我们使用了AES对称加密算法来加密和解密字符串“Hello, World!”。通过运行这个代码，我们可以看到加密后的数据为`b'gAAAAABf9X0AAAAAQAB'`，解密后的数据为“Hello, World!”。
