# 1.简介
         
　　　　《Core Java Volume I》由<NAME> 和 <NAME>编写并出版于2005年。这是一本入门级的Java参考书籍，它涵盖了Java编程语言的基础知识、关键特性及最佳实践方法。每章的内容都由一个小节组成，主要关注于某个主题。书中的示例代码都经过详细注释，是学习Java编程语言的良好素材。
          　　本书适用于Java程序员的学习、开发人员的职业生涯规划、高级工程师的技术参考等。本书并不仅仅是教给初级学习者，更重要的是可以提供一份深厚的基础，帮助读者在实际工作中提升自己的能力和竞争力。
         　　　　本书包括以下12章：
          　　第1章，概述Java虚拟机及Java平台；
          　　第2章，Java语言概述及其特点；
          　　第3章，面向对象程序设计基础；
          　　第4章，集合框架及Java事件模型；
          　　第5章，多线程、同步与协作；
          　　第6章，异常处理机制；
          　　第7章，Java输入/输出流处理；
          　　第8章，Java网络编程；
          　　第9章，数据库访问技术；
          　　第10章，Java性能优化；
          　　第11章，Java编译器技术；
          　　第12章，Java标准库（J2SE）技巧与典型用法。
         　　　　本书的作者深受国内外程序员的追捧，得到了很多名人的推荐，如Sun公司的总监陈立杰、IBM公司的创始人马丁爱桥、Oracle公司的技术经理何仕强等。此外，还有软件界著名程序员的集体作品——尤其是Core Javaworld周刊的主编吴军老师，他也是《Java编程思想》、《Head First Java》等经典著作的作者之一。
         　　　　因此，《Core Java Volume I》已成为 Java 学习的“圣经”，被誉为“Java程序员必备工具书”。

         　　　　本书的特色有以下几点：
          　　第一，本书以直观易懂的方式对Java的各种基础知识进行了全面介绍。对初学者来说，通过图文并茂的形式，能极大地帮助读者了解Java的各种特性和功能，快速掌握Java的使用方法。
          　　第二，本书注重内容连贯性和完整性。每个章节精炼易懂，内容紧扣主题，前后连贯，不乏例子丰富的实例代码。读者可以通过阅读本书学习到丰富的知识，同时还可将所学应用到实际项目当中。
          　　第三，本书通俗易懂，不会过分复杂化。书中融合了作者自己的理解，提供了丰富的实例，并配以生动形象的插图和动画效果，使得读者能够较快地理解相关知识。
          　　第四，本书符合中文习惯。本书采用简洁而直接的语句组织方式，清晰易懂的语言风格，便于读者理解。而且，作者把注意力集中放在了一些难懂的细枝末节上，以图准确地描述Java各项特性和使用方法。
