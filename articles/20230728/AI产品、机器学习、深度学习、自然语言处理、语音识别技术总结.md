# 3.机器学习（ML）
机器学习（Machine Learning）是一种让计算机具备“学习能力”的技术，它可以让计算机从数据中自动分析、归纳和预测出来，而无需事先指定或编程实现特定的规则。简单来说，机器学习就是利用计算机的算法，一步步地演化出可以解决特定问题的模型，最终达到机器能够自主解决问题的目的。机器学习以现有的海量数据为基础，通过不断迭代，它可以自动发现并利用数据的规律性和关联性，自动进行分类、回归、聚类、关联分析、预测等工作。机器学习的两种主要类型是监督学习和非监督学习。

### （1）监督学习
监督学习是指给机器学习算法提供已知正确答案的数据，算法通过分析学习到的知识建立模型，来预测新的样本的输出结果。监督学习的典型例子是分类问题，如垃圾邮件过滤、手写数字识别；回归问题，如房价预测、气温预测；排序问题，如推荐系统、广告排名；序列问题，如股票价格预测。监督学习可以分为两大类：
- 有标签分类(supervised classification)：训练集包含输入数据及其对应的标签，算法可以利用此信息进行学习，并根据新输入的数据预测标签。如二分类问题，预测销售额是否超过1万元；多分类问题，预测客户的购买意向；多标签分类问题，预测文档的主题标签。
- 无标签分类(unsupervised classification)：训练集没有相应的标签，算法需要自己确定数据之间的关系，对数据的分布形状、模式进行建模，然后再利用这些知识来预测新的输入数据属于哪一类别。如聚类、模式识别。

### （2）非监督学习
非监督学习是指不需要任何人为干预的情况下，机器学习算法通过分析数据中的结构和模式来自我学习，找寻数据的隐藏模式或分布规律，使得算法能够自动发现数据的相似性和异常值。非监督学习的典型例子是聚类(clustering)，它可以用来发现数据中的共同特征，将相似的数据分为一类。K-Means、DBSCAN、OPTICS、GMM、HDBSCAN、EM都是常用的非监督学习算法。

### （3）评估方法
在机器学习中，为了评估模型的性能，通常会用到一些标准指标，如准确率、召回率、F1 Score、ROC曲线、AUC值等。其中，准确率(Accuracy)、召回率(Recall/Sensitivity)、F1 Score三者可以用于分类问题，ROC曲线和AUC值可以用于二分类问题和多分类问题。其他的评估标准还有例如损失函数，即预测值与真实值之间的差距大小。总的来说，机器学习需要用大量的 labeled data 和 unlabeled data 来训练模型，然后用 test set 来评估模型的效果。

### （4）超参数调优
在机器学习过程中，往往存在一些超参数，比如说模型的复杂度，使用的算法，参数初始化方式等，这些超参数对模型的训练有着巨大的影响。因此，如何选择合适的超参数就显得尤为重要。常见的方法有 Grid Search、Random Search、贝叶斯优化和遗传算法等。

# 4.深度学习（DL）
深度学习（Deep Learning）是一类人工智能技术，它利用多层神经网络的形式，可以处理复杂的数据，从而实现端到端的学习。深度学习的目标是建立具有多个隐藏层的神经网络，每个隐藏层都由若干神经元组成，不同层之间通过权重链接。训练过程是在整个神经网络上反复迭代，修正网络中权值的过程，目的是使神经网络逐渐变得更聪明、更强健、更有效。深度学习应用的领域包括图像识别、视频分析、文本理解、语音识别、股市预测等。

### （1）卷积神经网络（CNN）
卷积神经网络（Convolutional Neural Network，CNN）是深度学习的一个子领域，它利用卷积层和池化层来提取图像特征，可以用于图像分类、物体检测、语义分割等任务。卷积层的作用是抽取图像中的局部特征，并压缩高维特征图；池化层的作用是进一步降低特征图的维度，减少计算量，防止过拟合。

### （2）循环神经网络（RNN）
循环神经网络（Recurrent Neural Networks，RNN）是深度学习另一个重要的子领域，它可以用于建模序列数据，如文本、时间序列等。RNN 的关键是循环连接，它把前面输出的信息传递给当前的单元，从而学习到长期依赖关系。它还具有记忆功能，可以帮助神经网络记住之前出现过的事件，从而提高推理速度。

### （3）递归神经网络（RNN）
递归神经网络（Recursive Neural Network，RNN）是一种深度学习算法，它可以用于建模树或者图结构数据，如语法树、网页结构等。递归神经网络的关键是多层递归连接，它可以让网络像自然语言一样处理多层结构的输入。

# 5.自然语言处理（NLP）
自然语言处理（Natural Language Processing，NLP）是人工智能的一个领域，它试图让计算机理解和生成人类的语言。它可以处理文本、音频、视频等信息，包括自然语言、对话、摘要、评论等。NLP 技术的基本任务是向机器提供一段文本或一段语言描述，然后返回能够代表整个文本或语言的单词、短语、句子甚至整个语句。

### （1）词嵌入
词嵌入（Word Embedding）是自然语言处理的一种特征表示方法，它将文本转换为固定维度的向量，向量中的每一维对应于一个词或词组，向量的值反映了该词或词组的含义。词嵌入可用于多种 NLP 任务，如文本分类、情感分析、命名实体识别等。

### （2）词向量
词向量（Word Vector）是一种用矢量空间表示词汇的语义的方法。词向量可以用于文本分类、情感分析等任务。

### （3）句子嵌入
句子嵌入（Sentence Embedding）是一种采用上下文语境的词嵌入方法，它可以表示一段文本的整体含义。句子嵌入可用于多种 NLP 任务，如文本相似度计算、问答系统、文档摘要、机器翻译等。

### （4）序列标注
序列标注（Sequence Labeling）是自然语言处理的一个子领域，它可以让计算机完成对文本序列中每个元素的标记。序列标注可用于文本分类、命名实体识别、事件抽取、文本蕴含等任务。

# 6.语音识别
语音识别（Speech Recognition）是自动将声音转化为文本的过程，是自然语言处理的一部分。它可以在语音数据中提取语音特征，并借助这些特征进行语音识别。在语音识别中，通常使用短时傅立叶变换（Short Time Fourier Transform，STFT）和梅尔频谱系数（Mel Frequency Cepstral Coefficients，MFCC）来提取语音特征。语音识别的应用包括拼写检查、语音交互、命令控制、日常生活场景下的语音助手等。
