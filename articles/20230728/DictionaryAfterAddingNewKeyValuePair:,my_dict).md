# 1.简介
         

         Python中字典类型是一个非常重要的数据结构，它可以存储任意数量的键值对。当我们向字典中添加一个新的键值对时，如果键已经存在于字典中，则更新对应的值；如果键不存在，则创建新的键值对。比如，给字典`my_dict = {"apple":1,"banana":2}`新增键值对`"orange":3`，其结果如下：
            ```python
            >>> my_dict["orange"] = 3
            >>> print(my_dict)
            {'apple': 1, 'banana': 2, 'orange': 3}
            ```
            如果想获取字典中某个键的值，可以使用`my_dict[key]`表达式，如`print(my_dict['banana'])`输出`2`。
            
            在实际应用中，我们经常需要将新数据添加到字典中去，而对于已有的键值对，我们也可能需要修改或删除它们。在Python中，使用字典的方法进行这些操作很方便，比如`pop()`、`update()`、`del`等方法。本文就来学习一下如何向字典中添加、删除或修改键值对。
         # 2.基本概念术语说明
         ## 2.1 字典（dictionary）
         ### 描述
        - **字典**（英语：dictionary），又称符号表、关联数组（associative array），是一个无序的计算机数据结构，其中每个元素都由一个键值对组成，键通常为字符串或者数字，值可以是任意类型。

        - 比较好记的说法：词典（Dictionary）是一种线性存储数据的集合，用关键码（Key）索引操纵数据。关键码使得查找数据变得简单快捷。
        
        ### 使用场景
        - Python中的字典，尤其适合用来存储具有映射关系的键值对，能够有效地提高查找效率，例如：保存用户信息、商品详情、联系方式等。

        - 有时候也可以用来作为字典树的实现。

        - 字典还可以用于统计多维数据之间的相关性。例如：假设我们有两个列表`listA=[a1, a2,..., an]`和`listB=[b1, b2,..., bm]`,想计算出一个矩阵C，C[i][j]=abs(ai-bi), i, j表示第i个元素和第j个元素的差值的绝对值。此时就可以使用字典构建矩阵C。

        - 更广义上来说，字典不仅可以用来保存键值对，还可以用来保存其他的编程语言所没有的东西。例如：在javascript中，字典与对象之间可以看作是对偶的。

    ## 2.2 键（key）
    ### 描述
    - 字典中的键（key）是一个不可改变的数据项，是唯一标识符，与值（value）之间建立起一一对应的关系。

    - 每个键对应着一个值，但一个键只能对应一个值，不能出现重复的键。

    - 可以使用任意的不可变数据类型作为键，一般使用字符串或者数字。
    
    ### 使用示例
    ```python
    d = {}                     # 创建空字典d
    d['name'] = "Alice"        # 添加键'name'并赋值为'Alice'
    d[123] = [1, 2, 3]         # 添加键123并赋值为[1, 2, 3]
    d[(1, 2)] = {1:2, 3:4}     # 添加键(1, 2)并赋值为{1:2, 3:4}
    ```
    在上面的例子中，字典d有三个键，分别是："name", 123, (1, 2)。
    
    ## 2.3 值（value）
    ### 描述
    - 字典中的值（value）可以是任何类型的变量，包括另一个字典。

    - 当值是一个字典的时候，该值也被称为子字典（sub dictionary）。
    
    ### 使用示例
    ```python
    user = {"name":"Alice","age":25}      # 创建一个用户信息字典
    address = {"street":"123 Main St.", "city":"Anytown"}    # 创建一个地址信息字典
    person = {"user":user, "address":address}          # 将用户信息和地址信息打包进一个个人信息字典person中
    ```
