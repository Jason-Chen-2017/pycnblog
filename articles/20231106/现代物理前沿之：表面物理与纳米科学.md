# 1.背景介绍


## 表面物理
表面物理研究物体在表面的特性。它以材料学、结构工程、计算物理学、化学、光学等多学科交叉领域为研究对象，从晶体结构到金属界面，再到石墨烯、玻璃和塑料，表面物理一直以来都有着广泛的应用需求。目前国际上主要研究表面的关注点集中在一类关键词——光的传播，其实现形式主要是衍射与反射。
## 纳米科学
纳米(Nanom)和微米(Micron)两词出自希腊语Νανομηδες，意指小而细的一截或极小的一粒。纳米科技就是指能量小、尺寸微的纳米材料与技术，其中包括微电子技术、新型材料制备技术、制药技术、生物技术、医疗器械、智能机器人等应用领域。纳米材料具有超高导通率、高透光性、高稳定性、高密度、低温度等特点，能够满足高速器件、显示器件、光源等应用需求。因此，纳米科技成为国际上颠覆性的科技革命的火车头。
## 宏观背景
近几年来，随着人工智能、机器学习、互联网的发展，计算机科学技术取得突破性进步，人们越来越重视自动化和自主化。人工智能、机器学习、模式识别、图像处理、语音识别、无人机、深度学习等领域都是人工智能的热门方向。
人工智能时代带来的变革不仅影响了物质层面的生产效率和经济利益，也改变了人类社会的很多方面。其中最重要的变化之一是信息爆炸时代，这个时代的信息存储、传输、处理、分析能力成为各行各业的重点难题。为此，工程师们尝试开发新的材料，希望能够提升材料性能、抵御环境侵蚀等性能指标。一方面，通过对这些材料进行解析、设计、制造和测试，可以大幅提高制造过程中的产品质量。另一方面，也可以帮助企业更好地理解用户需求，选择适合自己的解决方案。
但是，与其他物理科学不同，对纳米科学与材料的认识还处于萌芽阶段。纳米科学的历史比较悠久，但其出现对表面物理及相关材料的探索还有很大的挑战。由于材料本身的复杂性、特性过于复杂，所以对材料表象的认识非常困难。然而，最近几年里，人们发现通过合成纳米材料并将其与相应的技术结合，可以达到很高的性能水平。例如，通过合成荧光标记液，就能够精确跟踪物体的位置、角度、速度等信息。与此同时，微芯片技术、超声波技术、智能传感器等新兴技术也已开始广受欢迎。总的来说，纳米科学正在催生着一个全新的物理研究领域，即纳米科学与表面物理的结合。

# 2.核心概念与联系
## 周期性边界条件
在表面物理学中，周期性边界条件(Periodic Boundary Condition, PBC)是一个假设，它认为研究区域以周期性的方式进行周期性的复制，使得研究对象的表面具有自我复制的性质。PBC可以把研究对象看做是无限的单元格，而不是整体，并且每个单元格都具有相同的尺寸和形状。一般情况下，我们可以用周期性边界条件去模拟无限的布朗运动，或者研究类似莫尔维恩哈德的分布。如下图所示，红色的圆代表原子，蓝色的圆代表原子的周期性复制。

## 晶体结构的周期性边界条件
晶体结构可以分为两种类型：一种是平滑的结构，如类金刚石；另一种是精致的结构，如手掌玻璃、石墨烯、陶瓷。对于平滑结构而言，它的周期性边界条件可以保证研究区域内存在无穷多个相邻的单胞（简单单位）晶体，这样可以很方便的利用周期性对称性做研究。如下图所示，右侧所示的是平滑的类金刚石，左侧所示的是其周期性边界条件的模拟。

对于精致结构，例如手掌玻璃，没有周期性边界条件则无法模拟出这种具有自我复制的物理性质。不过，为了模拟这种自我复制，可以在模拟之前先进行适当的分割，例如将玻璃分割成大小合适的小块。分割后得到的小块组成的复合材料就可以用周期性边界条件来模拟自我复制。如下图所示，左侧所示的是手掌玻璃的自我复制模拟，右侧所示的是分割后的玻璃片段的周期性边界条件模拟。

综上所述，表面物理所关心的研究对象既可能是无限的复合材料（具有周期性边界条件），又可能是个体的简单单元晶体（不具有周期性边界条件）。对于复合材料，可以利用周期性边界条件来进行研究，例如研究其晶体结构、材料性能、自旋耦合、机械行为、天文学、宇宙学等；对于单胞晶体，可以利用周期性边界条件模拟自我复制，或者将其进行分割后利用周期性边界条件进行研究。而对于无周期性边界条件，则需要考虑一些特殊的模拟方法，例如拓扑计算、基于路径积分的方法等。

## 模拟方法
表面物理通常采用核磁共振技术、超声速振动技术、强磁场技术等多种模拟方法，通过设计各种实验装置、模拟实验来达到收集必要的数据，然后分析这些数据，进一步对研究对象的表象进行解释。核磁共振是最常用的模拟方法，它利用原子核与X射线之间的相互作用，通过测量两个原子核之间的相互作用强度来揭示其原子结构、电子状态和几何结构。如下图所示，左侧所示的是普通核磁共振(Normal NMR)，右侧所示的是强脉冲核磁共振(Fast IR NMR)。

超声速振动技术则利用高压气体激发出的粒子的束流来模拟结构的变形。与核磁共振不同，超声速振动技术不需要原子之间的相互作用，而只依赖于浮力、速度等因素，因此可以提供更高的自由度。如下图所示，左侧所示的是超声速高压气流动，右侧所示的是超声速冷却气流动。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 费米面电子的纳米弛豫半径算法
费米面电子的纳米弛豫半径算法(Frank-Condon Approximation)被广泛用于求解费米面电子的弛豫频率。该算法认为费米面电子处于弛豫态时，两端电子能量差距最大，可以近似视作二维费米面上的一个“熵点”，从而使费米面电子的实际弛豫频率可由熵的梯度决定的弛豫频率。具体步骤如下：
1. 在2D的平面上，确定费米面边界的线条形成的曲线，取该曲线的外接圆，记作R。
2. 用直线L连接R的两个焦点，作为纳米弛豫半径的定义域。
3. 用曲线L切开半径为R/e的圆环，得到费米面电子的两端电子，记作u1和u2。
4. 对于任意一条电子轨迹l(t)，记v(t)=e*l(t)/h，其中e为基本能量，h为轨道长度。
5. 分别找出u1和u2处电子的动量k1和k2，并设v1=vk1+v2，则有：v1^2=(vk1+v2)^2-2(v1)(vk1+v2)<|v1|>。
6. 将v1看作是一维的“熵”变量，利用牛顿-莱昂斯方程组求解该变量，得到v1的表达式。
7. 计算v1的一阶导数，得到v1'=dv1/dk，设为f1。
8. 对费米面电子的每一条电子轨迹，求解其v1值，并拟合函数v1=f1(k1)来获取电子的实际弛豫频率。

## 紧束缚光子的共振分析
通过结构定轴法（SFA）、多电子跃迁强度法（MEPS）等方法可以测出自由电子的能级分裂，进而计算不同能级的占据率和跃迁概率。
结构定轴法（SFA）的基本想法是在电子云密度很大的情况下对少量电子，进行定轴测量，定轴测量能够提供给定电子一个独特的投影（相空间坐标），从而使不同能级之间的相空间分布比例以及不同能级的光子的相空间分布可靠测量出来。
而多电子跃迁强度法（MEPS）的基本想法是通过在不同电子下施加受控的极化脉冲，测量出某些特定能级的跃迁强度。
通过进行多次重复的运行，可以获得不同能级光子的数量分布、占据率、跃迁概率、单电子跃迁强度。
MEPS中使用到的控制受控极化脉冲有：控制脉冲1（CP1），控制脉冲2（CP2），能级序列失衡脉冲（SQUID），弱动脉冲（GSB）。
