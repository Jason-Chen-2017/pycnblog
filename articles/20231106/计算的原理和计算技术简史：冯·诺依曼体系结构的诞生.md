# 1.背景介绍


计算机这个概念在近代以来占据了人类信息技术发展的中心位置，但是它产生的历史却不是一帆风顺的。从最早的电算机到中断式的微型计算机，再到今天集成电路的多核处理器，计算技术已经渗透到人们的生活中各个角落。作为一个工程师，研究者或者是普通用户都需要了解一些基本的计算原理和计算技术，从而更加了解计算机的工作机制。虽然计算机发展得如火如荼，但其核心原理和理论基础却是不变的，那就是冯诺依曼机构的制定。下面，就让我们一起回顾一下这一制度的诞生历史。
# 冯·诺依曼：一个科学怪人
“冯氏三部曲”中的第一部就是“计算机及其运算方法”，《计算机及其运算方法》一书于1947年由剑桥大学出版社出版，并获得图灵奖。作者丘奇威(约翰·麦卡锡)是美国著名数学家、逻辑学家，他创立了一种新的数字表示系统，称之为电子二进制法，可以进行更高效率的加减乘除等算术运算。这项提案得到了当时很多学者的支持，大家认为这是颠覆性的科技革命。但是很快，这种想法就会被另一个人——英国物理学家皮凯龙·莱纳德·图灵(Alan Turing Turing)所驳斥，他认为电子二进制法并不能解决计算机的数学问题，因此造成了计算机发展的停滞。
这时候，为了寻找突破口，1945年年底的爱迪生博士发明了集电气工程、电子工程、机械工程、通信工程、自动控制工程于一体的“超级计算机”。此后，这批科学家又开始了一场对计算机的重新审视。

# 诺伊曼机构的诞生
“冯氏三部曲”的第二部《论沿着程序设计方法的方向研究电子计算机器》是1947年三月刊载在《国际周刊》上的，是以图灵和丘奇威合著。这本书一经发表，便引起轩然大波，主要是因为它对计算机的描述过于抽象，不够系统，缺乏直观感受，读起来令人费解。因此，与此同时，图灵、丘奇威、麦克阿瑟等人相继成为了“电子计算机界的四大天王”，将其视作科技革新中不可或缺的一环。

1948年4月，尼古拉·哥德尔、艾兰·布劳恩、卡尔·马克思和埃米尔·笛卡尔联名写信给鲍勃·邱奇，请求建立一个由人工智能、符号理论、计算理论和实际应用驱动的研究机构。该机构的名称叫做“诺伊曼研究院”。邱奇给出的答复是：“我们正在筹建这样的机构，作为认真探索人工智能、符号理论、计算理论和实际应用之间的关系，使之走向前进的‘桥梁’。”可谓英雄所见略同。

1948年7月1日，诺伊曼研究院正式成立，主任是托马斯·冯·诺依曼。

1949年1月1日，诺伊曼研究院首次举办会议，宣读“摩尔定律的终结”报告，1月2日闭幕，该报告为整个研究领域的爆炸性事件，至今仍历历在目。

# 冯·诺依曼：从一串比特说开去
冯诺依曼就是著名的电子工程师，他在他的代表作《计算的本质》一书中，通过详尽的论述，阐释了计算机的运作原理。在这本书中，他用非常简单易懂的方式，就揭示了计算机的各种功能和特性，并为未来的计算机发展指明了方向。他成功地打通了计算机硬件和软件之间纷争的关节，使人们看到了一种全新的计算机世界，那就是“比特流动”的计算模式。

1950年，随着计算机技术的快速发展，另一位中国科学家陈冉通过了一份题为《计算的本质》的重要著作。两人的通信一直没有停止。1952年，杨振宁、罗素、西蒙·麦克纳马拉等人发起组织“国际计算学会”，主张把计算理论的研究成果普及到全球，并发起了一系列的国际研讨会，包括《计算的本质》的讨论。

1954年，由图灵、丘奇威和冯诺依曼发起的“百花齐放、百家争鸣”运动改变了计算领域的局面，推动了计算机技术的快速发展。

1957年，蒂姆·库因茨发表了“计算机程序设计语言——一种新的交互方式”一文，开创了“编译型编程语言”的先河。

1962年，苏姚·沃尔夫斯基发表了“数据导向编程——一种计算结构化方法”一文，提出了“数据流语言”的概念。

1965年，尼古拉斯·庇古发表了“存储器管理——计算机硬件层次结构中的动态重排”一文，首次将“动态存储分配”引入计算机。
