# 1.背景介绍


## 一段时间过去了...
　　在过去的一段时间里，人工智能（Artificial Intelligence，AI）和云计算（Cloud Computing）已经成为当今最热门的话题。无论是在智能手机、电脑、机器人等领域，还是在娱乐圈，都可以看到许多业界人士关注这一领域的最新进展。其中，从娱乐圈出来的人大多认为，AI和云计算正在颠覆传统的视频游戏制作方式，而且这种影响力还会持续到下一个互联网的爆炸式增长期。

　　对于企业来说，这是另一个非常迫切需要解决的问题。在虚拟现实、AR/VR、无人驾驶等技术的帮助下，人们越来越希望能够将更多的注意力放到玩虚拟世界，而不是玩真实世界中的游戏。但是在实际应用中，企业往往发现，如何让更多的人参与到这个体验中并得到最大化的收益，是一个非常重要的课题。

　　2017年9月，由微软创始人彼得-蒂姆·诺伊曼（Peter Nichols）发表在IEEE Spectrum上的一篇文章，提出了“阿尔法狗”的概念——基于AI的虚拟现实机器人。这个虚拟现实机器人包括两个部分，即虚拟引擎和人工智能。其虚拟引擎可以生成3D图像、声音等，而人的大部分认知能力可以透过这个引擎进行学习和理解。通过对人类的情绪、面部表情、动作等进行识别，这个机器人可以做出反应并给予相应的响应。

　　为了更好的让业界了解人工智能和云计算所带来的变化，国际标准化组织（ISO）在今年五月份发布了新标准ISO/IEC 42017，旨在阐述云计算的新兴趋势。该标准定义了云计算的定义、特征、服务和管理方式，并着重阐述了云计算的应用场景。

## 那么，什么是云计算呢？
　　云计算（Cloud computing）是一个分布式计算资源池，允许用户通过网络从事计算服务。云计算分为公共、私有、社区等多种形态，每个形态都具有独特的功能和优点。公共云计算是云计算的一种形式，提供各种基础设施（例如服务器、存储、网络等）给所有用户使用；私有云计算则是指由客户自己设立的云计算环境，仅为个人或小型机构提供服务；社区云计算则是指利用云计算技术为某个特定社群提供服务。一般情况下，云计算所提供的计算资源的容量和可靠性要高于本地的数据中心。

　　在2016年，全球公共云数据中心数量达到了两百多万个，每年新增的数据中心超过四千万个。根据IDC统计，目前全球已有七成以上的IT行业业务依赖于云计算。随着云计算技术的不断发展，越来越多的公司开始选择将自己的服务器和存储放在公共云上，进行虚拟化和云计算。

　　2015年《华尔街日报》曾经报道，为了应对全球金融危机、贸易战和信息技术的发展要求，英国政府推出了一项叫做“移交数据中心”（Offshore Data Centers）计划。计划目的是通过拆除部分数据中心，将它们转移至海外进行运营，以减少设备成本并提升整体的数据中心利用率。据估计，该计划可节省60%～70%的数据中心投资开支。

　　近年来，随着超级计算机、云计算、大数据、物联网、人工智能等技术的飞速发展，云计算也成为新的风口。正如3D打印一样，云计算的出现，将使得很多重复性工作自动化，并释放大量生产效率，实现更多产业的升级换代。因此，如果能够从娱乐业角度出发，充分利用云计算的价值，提升娱乐业的竞争力，将是非常值得考虑的事情。
