# 1.背景介绍


## 计算机的产生
古代人类开始发明各种工具用以记录科学和工程的数据。这些工具包括手帐、棍子、沙子等小型工具，后来人们发现利用这些工具可以解决一些复杂的问题。这项技术得到广泛应用。人类的发明创造是不断推动着科技的发展的。
在古希腊时期，哥白尼发明了望远镜，它可以用来看到星星和星系。通过光线调节器，他可以调整不同波长的光束从而拍摄出更精确的照片。现代的微观电子显然也是人们在研究和开发过程中所发明的，它的发现使得科学和工程领域迅速发展。
随着科技的进步，人们对计算机的需求也越来越强烈。就算没有计算机这个词语，也能明显感觉到人们对信息处理能力的需求激增。自然界的资源是无限的，但是人类的工具却很有限。于是，一些计算机程序员、建筑师或者农民等，利用其数学、逻辑、语言以及工程知识，创造出了能够模拟自然界的程序。这样一来，一些复杂的问题就可以用简单的命令或语句来表示，并由计算机执行。
1945年，美国人诺姆·卡尔-马克思第一次提出“计算机科学”的概念。从此，计算机领域有了自己的规范和体系。由于它的高度实用性和普及性，它被广泛地应用在各个领域中。
## 计算的基本原理
### 演绎法
先验公理：

1. 命题总是正确或错误。
2. 真值表只能是正确的或错误的。
3. 如果某些条件必须同时满足才可能发生一个事件，那么必定存在一些条件互斥的情况。
4. 没有任何事情可以永远保持相同的状态，所以任何事情都可以变化。
5. 任何试图证明一件事是真还是假的逻辑都有缺陷。

演绎法的过程如下：

1. 从最简单的命题开始，构造一张真值表。将每种情况标记成对或错。
2. 在每一行的基础上增加一列，显示结果取决于是否满足每个条件。如果有两个条件互斥，则结果只能是唯一确定的。
3. 根据新增加的列，填充余下的格子。对于每一行，如果所有条件都满足，则这一列标记为对，否则标记为错。
4. 重复步骤3直至真值表中的每一行都填满。如果某个条件在所有的情况中都不一致，那么说明有矛盾之处。
5. 使用演绎法所求证的命题的真值表作为初始条件，然后由每个条件以及在该条件下所作出的推论，一步步推导出所需结论。

演绎法是比较直观易懂的方法。它适用于一切简单直观的命题，但是对于某些复杂的问题，如代数关系、集合论、概率论等等，却不能完全运用演绎法。需要其他方法，如归纳法、递归法、层次分析法、因果律等。

### 图灵机模型
图灵机（Turing machine）是美国计算机科学家考克斯.艾伦（Alan Turing）于1936年提出的一种抽象模型，它是一种可编程计算机。图灵机由一个带头脑的读写头和一个无限的存储器组成。读写头读取指令，执行指令修改存储器的内容。计算机可以解决很多问题，如判定性逻辑推理、数据压缩、密码学、自动组合、游戏等。但是，由于图灵机并非由人的双眼所见，因此对于人来说仍然很难理解其工作方式。
## 计算技术的演变
### 单道机
单道机是最早的基于二进制的存储器的计算机。它由一个十分简单的芯片组成，即存储单元、控制器、输入/输出设备三个部分。CPU负责接收输入的信号、转换数据，将结果输出到打印机或者显示屏。这种结构被称为点式机，因为它只能进行数字处理。
### 脱机多道机
第一台电子计算机ENIAC，它的运算速度只有每秒300万次。它的优势是价格便宜，但缺乏高效率的优势。在实际应用中，这意味着必须等待几天甚至几周才能获得结果。第二台个人电脑IBM PC，在80年代曾经达到顶峰，但由于其磁盘容量较小，无法应付实际应用。第三台超级计算机-通用计算机的问世，解决了磁盘容量的问题。但是，整个计算机仍然十分昂贵。因此，出现了多道计算机。
### 时移寻址技术
虽然时移寻址技术的出现极大地降低了内存的成本，但仍然受制于机器的逻辑性和性能限制。因此，它被广泛地用于批处理任务、事务处理系统、信息处理系统等。
### 分布式计算技术
分布式计算技术包括集群计算和云计算。集群计算把多个计算机集中起来，共享硬件资源，实现共享的任务分配。云计算则是指把服务器托管给第三方服务商，用户可以在不购买服务器的情况下，直接使用服务器资源。相比于本地计算机，云计算具有非常好的经济性。但是，由于网络通信的开销，云计算的性能受限。因此，许多大型公司开始采用分布式计算技术。
### 矩阵运算技术
由于过多的加减乘除运算，导致计算机运算速度十分缓慢。为了解决这个问题，数学家们在19世纪末提出了矩阵运算技术。矩阵运算技术通过对大型矩阵进行快速乘法运算来提高计算机运算速度。
### 图形处理技术
图形处理技术的出现促进了计算机的视觉和图像处理能力的提高。它主要由计算机图形学、视觉识别、三维图形处理、虚拟现实技术、生物医学模拟等领域组成。其中，计算机图形学的发展取得了巨大的成功，成为目前计算机处理图像的主流方法。
## 计算机性能提升的措施
### 流水线技术
流水线技术就是利用多个阶段的处理单元并行处理任务，以提高运算速度。流水线技术能够并行地执行指令，能极大地缩短处理时间。现代计算机的流水线通常由五级结构组成，分别是数据发射、指令译码、指令队列、运算模块、结果存储。每一级都是串行的，通过设计多个晶体管来提升性能。
### 指令级并行技术
指令级并行技术是指在同一个时钟周期内，使用多个处理单元来处理不同的指令。AMD和英特尔就使用了这种技术来提高处理器的性能。当前，英特尔已将超标量、VLIW、SIMD(单指令多数据)等多种指令级并行技术结合到一起。
### 超频技术
超频技术是指将工艺设备的性能提升到极致。这要求设计人员在工艺上做大幅度的优化，将电源的最大功率提升至一定的程度，以维持所需的性能。由于超频往往会降低工艺质量，因此，功耗再提升之后还需要时间恢复到正常状态。目前，英特��、AMD、高通等公司都在研发超频技术。
### 神经网络技术
神经网络技术是指对大量的神经元进行联结，形成一个复杂的网络结构，从而能够对复杂的信息进行智能分析和预测。目前，人工智能领域占据了高收入行业。许多科技公司都试图使用神经网络技术来解决问题，比如滴滴、网易云音乐、亚马逊等。
