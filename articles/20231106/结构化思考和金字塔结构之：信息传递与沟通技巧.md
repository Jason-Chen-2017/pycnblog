# 1.背景介绍


## 1.1 信息传递的目的
信息传递作为人类社会生活不可或缺的一部分，它的目的是传递信息，帮助信息从产生到传递到接收者，传播知识、提高生产力和个人能力。任何时候都需要用到信息传递，无论对现实世界还是虚拟环境，无论是交流、沟通、学习、工作、娱乐等方面都离不开信息传递。信息传递的过程也是复杂的，需要解决很多问题，下面介绍信息传递的几个主要目标。

1.传递知识：信息的传递还可以促进知识的传递。通过分享各种各样的信息，比如历史、科学、文化等，可以培养广泛的人才素养。通过观看电影、听音乐、阅读小说、了解历史上的各种故事，也可以让大家了解世界。

2.提升生产力：在工作和学习中，通过信息的共享，可以更有效地集中精力。信息的传递可以提升生产力，达到工作效率的最大化。比如，通过微博、微信、QQ、知乎等平台上信息的传播，可以减少手机的使用时间，把更多的时间放在工作和学习上面。

3.增加自信心：在面对困难时，信息的传递也能增强自信心。通过告诉他人自己的坚持、努力，或者在分享过程中鼓励自己，都能建立起积极向上的心态，让人感觉轻松自在。

4.改善生活方式：通过信息的传递，我们可以在日常生活中发现新的模式和习惯，培养身边的人际关系。比如，通过短视频的内容创作，可以分享一些生活中的感悟、经验、情感、见闻等。通过阅读评论、讨论，可以获得不同视角的意见。通过音乐、电台节目、购物分享、结伴同行，可以实现一种全新的社交方式。

## 1.2 沟通技巧
在日常生活中，信息的传递越来越多元化，为了信息的准确传递，各种沟通技巧和方法层出不穷，下面介绍几种比较常用的沟通技巧。

1.图说法：图说法是指将复杂的问题简单化。通过使用简单的语言表达复杂的问题，可以让别人快速理解并做出回应。比如，如果有人问你家庭经济情况怎么样？你可以简单的描述一下家庭的收入分配结构、产出结构以及消费水平。这样的话，对方就可以立刻理解你的意思，并且做出相应的反馈。

2.归纳总结：归纳总结是指摘取重点，列举重点信息。通过阅读一些媒体或网站，可以发现许多复杂的问题都可以归结为几个关键词。比如，对于一个特定的国家或地区，可以提炼几个关键词，如“社会治理”、“公共服务”、“基建工程”等，然后针对性分析这些关键词所代表的实际问题。

3.复述陈述：复述陈述是指重复刚才讲过的内容。重复是语言交流的基本技能，通过重复别人的讲话，可以让对方了解自己的想法和观点。比如，你在给别人讲的时候，可以通过重复之前的讲话，让对方能够把握住你的意思。

4.比喻语言：比喻语言是指使用类似于生活中事物的比喻来讲述复杂的信息。比如，你可以讲述一个狗和蜥蜴的故事，让对方更容易接受这个消息。
