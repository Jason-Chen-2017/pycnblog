## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的 Python 代码示例，用于模拟一个简单的物理系统：

```python
import numpy as np
import matplotlib.pyplot as plt

def simulate_system(y0, k, t):
    dt = 0.01
    y = y0
    t_values = np.arange(0, t, dt)
    y_values = [y]
    for t in t_values[1:]:
        y = y - k * y * dt
        y_values.append(y)
    return t_values, y_values

y0 = 1
k = 0.1
t = 10
t_values, y_values = simulate_system(y0, k, t)

plt.plot(t_values, y_values)
plt.xlabel('Time')
plt.ylabel('Position')
plt.title('Position vs Time')
plt.show()
```

这个代码首先导入了 NumPy 和 Matplotlib 库，然后定义了一个模拟系统的函数 `simulate_system`。函数接受初始位置 `y0`、阻力系数 `k` 和模拟时间 `t` 作为参数，并返回模拟时间和位置的数组。最后，代码使用 Matplotlib 绘制了位置与时间的关系图。
