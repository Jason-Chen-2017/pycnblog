## 项目实践：代码实例和详细解释说明

以下是一个简单的Python代码示例，展示了如何创建一个AI Agent来调用函数：

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

class AIAgent:
    def __init__(self):
        self.weights = np.random.rand(10)

    def forward(self, x):
        y = np.dot(x, self.weights)
        y = sigmoid(y)
        return y

agent = AIAgent()
x = np.array([1, 2, 3, 4, 5])
y = agent.forward(x)
print(y)
```

在这个示例中，我们创建了一个简单的AI Agent，它可以接收输入向量`x`并调用`forward`函数。`forward`函数使用Sigmoid激活函数来计算输出`y`。
