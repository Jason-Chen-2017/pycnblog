## 项目实践：代码实例和详细解释说明

在这个部分，我们将提供代码实例和详细解释说明，帮助读者了解如何实现Agent。以下是一个简单的Python代码示例：

```python
import requests

def get_financial_data(keywords):
    # 调用财务信息API
    url = "https://api.example.com/financial_data"
    params = {"keywords": keywords}
    response = requests.get(url, params=params)
    
    # 解析API返回的数据
    data = response.json()
    return data

def display_financial_data(data):
    # 显示财务信息
    print("利润：", data["profit"])
    print("收入：", data["revenue"])
    print("成本：", data["cost"])

# 用户查询请求
query = "利润情况"

# 获取财务信息
financial_data = get_financial_data(query)

# 显示财务信息
display_financial_data(financial_data)
```
