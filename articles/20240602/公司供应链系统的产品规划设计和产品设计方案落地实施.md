```
Maximize: Z = c1*x1 + c2*x2 + ... + cn*xn
Subject to: a11*x1 + a12*x2 + ... + a1n*xn <= b1
            a21*x1 + a22*x2 + ... + a2n*xn <= b2
            ...
            am1*x1 + am2*x2 + ... + amn*xn >= bm
```

其中，Market是市场规模，Price是产品价格，Demand是产品需求，Supply是产品供应。
