## 背景介绍

## 核心概念与联系
AIGC可以分为以下几个核心概念：

1. 个人化：AIGC需要根据个人需求和特点提供个性化的服务。例如，根据个人喜好推荐音乐、电影等；根据个人生活习惯提醒用户完成重要任务等。

```python
import scrapy

class MusicSpider(scrapy.Spider):
    name = 'music'
    start_urls = ['https://music.163.com/']

    def parse(self, response):
        # 爬取音乐列表
        music_list = response.css('li[class="u-cover-img"] a::attr("href")').extract()
        for url in music_list:
            yield scrapy.Request('https://music.163.com' + url, self.parse_song)

    def parse_song(self, response):
        # 爬取音乐详细信息
        music_info = {
            'title': response.css('div.album_title ::text').extract_first(),
            'singer': response.css('div.album_info ::text').extract_first(),
            'album': response.css('div.album_detail ::text').extract_first(),
        }
        yield music_info
```
