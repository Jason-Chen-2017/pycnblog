## 核心概念与联系

在图数据结构中，边(Edge)是图中连接顶点(Vertex)的线。每条边都有两个端点，即两个顶点。边可以具有不同的权重，表示数据之间的关系强度。

图可以表示为一个由顶点集V和边集E组成的元组G = (V, E)。每条边e ∈ E都由两个顶点vi, vj组成，表示为e = (vi, vj)。

## 核心算法原理具体操作步骤

以下是Graph Edge的一些基本操作：

1. 添加边：向图中添加一条新的边。
2. 删除边：从图中删除一条边。
3. 查询边：查询图中是否存在一条特定边。
4. 更新边权重：更新图中某一条边的权重。

这些操作可以通过以下数据结构实现：

1. 邻接表(Adjacency List)：用一个数组存储每个顶点的邻接节点。
2. 邻接矩阵(Adjacency Matrix)：用一个二维数组表示图的邻接关系。

## 数学模型和公式详细讲解举例说明

在图数据结构中，边权重通常表示为一个实数。我们可以使用矩阵来表示图的邻接关系。设图G有n个顶点，则邻接矩阵A为n × n的矩阵，A[i][j]表示顶点i与顶点j之间的边权重。

数学模型可以表示为：

A = [a\_ij] \_ 1 ≤ i, j ≤ n

其中a\_ij表示顶点i与顶点j之间的边权重，a\_ij = w\_ij，其中w\_ij是实数。

## 项目实践：代码实例和详细解释说明

下面是一个Python实现Graph Edge的例子：

```python
class GraphEdge:
    def __init__(self, v1, v2, weight):
        self.v1 = v1
        self.v2 = v2
        self.weight = weight

    def __str__(self):
        return f"({self.v1}, {self.v2}, {self.weight})"

# 创建边
edge1 = GraphEdge(1, 2, 1)
edge2 = GraphEdge(2, 3, 1)

# 添加边
edges = [edge1, edge2]

# 删除边
edges.remove(edge1)

# 查询边
if edge1 in edges:
    print("边存在")
else:
    print("边不存在")

# 更新边权重
edge2.weight = 2
print(edge2)
```
