## 背景介绍

强连通分量（Strongly Connected Components, SCC）是计算机科学中一个重要的概念，它是指一个有向图中的一组顶点，这些顶点之间的边全部都是有向边，而且这些顶点之间没有出边指向其他顶点。这类组成的图称为强连通图（Strongly Connected Graph）。强连通分量在计算机科学中有着广泛的应用，例如网络流、数据压缩、程序分析等。

1. 对图中的每个顶点进行深度优先搜索（DFS），并记录每个顶点的探索顺序。

```python
import collections

def dfs(graph, start):
    visited, stack = set(), [start]
    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)
    return visited

def strongly_connected_components(graph):
    visited = set()
    scc, scc_list = [], []
    for vertex in graph:
        if vertex not in visited:
            visited.add(vertex)
            scc = dfs(graph, vertex)
            scc_list.append(scc)
    return scc_list
```
