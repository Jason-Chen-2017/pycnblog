人工智能（Artificial Intelligence，简称AI）是指模拟人类智能的计算机程序的科学。它涉及到人工智能算法、数据结构和语言等多个领域。人工智能的目的是让计算机能够像人类一样进行学习、思考、决策和沟通等智能行为。

在人工智能领域中，Agent（智能代理）是研究和开发的重要部分。Agent 是指能够执行某种任务的计算机程序或系统。Agent 可以自动执行任务，也可以与人类交互并获取信息。Agent 的主要目的是提高人类的工作效率和减轻人类的负担。
