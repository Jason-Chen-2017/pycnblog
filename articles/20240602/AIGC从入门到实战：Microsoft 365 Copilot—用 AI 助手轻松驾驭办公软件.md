## 5. 项目实践：代码实例和详细解释说明

Microsoft 365 Copilot 的项目实践涉及到多种技术，例如自然语言处理（NLP）和机器学习（ML）等。以下是一个简单的代码实例：

```python
import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

# 文本数据
texts = ['I love Python', 'Python is great', 'I hate Python', 'Python is powerful']

# 生成词袋模型
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

# 生成训练集
y = np.array([1, 1, 0, 1])

# 训练朴素贝叶斯分类器
clf = MultinomialNB()
clf.fit(X, y)

# 预测新文本
new_text = 'Python is useful'
new_X = vectorizer.transform([new_text])
prediction = clf.predict(new_X)
print(prediction)
```
