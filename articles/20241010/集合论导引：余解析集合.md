## 第一部分：集合论基础

### 第1章：集合的基本概念

#### 1.1 集合的定义与性质

集合是数学中的一个基础概念，可以理解为一些确定对象的总体。这些对象可以是任何形式，如数字、人物、地点等。集合通常用大写字母表示，例如A、B、C等。

**定义：** 集合是一组无序的、确定的对象的集合。这些对象称为集合的元素。如果x是集合A的一个元素，我们通常写作x ∈ A；如果x不是集合A的一个元素，我们写作x ∉ A。

**性质：** 
- **确定性**：集合中的元素是明确的，不会引起歧义。
- **互异性**：集合中的元素是不同的，不允许重复。
- **无序性**：集合中的元素没有固定的顺序。

#### 1.2 子集与真子集

**子集（Subset）：** 如果集合A的所有元素都是集合B的元素，那么称A是B的子集，记作A ⊆ B。换句话说，如果对于任意的x ∈ A，都有x ∈ B，那么A是B的子集。

**真子集（ Proper Subset）：** 如果集合A是集合B的子集，并且A不等于B，那么称A是B的真子集，记作A ⊂ B。也就是说，除了A是B的子集外，A和B不相等。

#### 1.3 集合的表示方法

集合的表示方法主要有两种：列表表示法和描述法。

- **列表表示法（Brace Notation）：** 用大括号{}将集合的元素括起来，元素之间用逗号隔开。例如，{1, 2, 3, 4} 表示一个包含四个元素的集合。

- **描述法（Set-Builder Notation）：** 用大括号{}将集合括起来，并在大括号后面用竖线|表示“满足”，然后写上集合的元素应满足的条件。例如，{x ∈ N | x > 5} 表示一个包含所有大于5的自然数的集合。

## 第二部分：集合运算

### 第2章：集合运算

#### 2.1 并集与交集

**并集（Union）：** 集合A和集合B的并集是由A和B中的所有元素组成的集合，记作A ∪ B。其数学表达式为：

\[ A \cup B = \{ x | x \in A \text{ 或 } x \in B \} \]

**交集（Intersection）：** 集合A和集合B的交集是由A和B中共有的元素组成的集合，记作A ∩ B。其数学表达式为：

\[ A \cap B = \{ x | x \in A \text{ 且 } x \in B \} \]

#### 2.2 差集与对称差

**差集（Difference）：** 集合A和集合B的差集是由属于A但不属于B的元素组成的集合，记作A - B 或 A \ B。其数学表达式为：

\[ A - B = \{ x | x \in A \text{ 且 } x \not\in B \} \]

**对称差（Symmetric Difference）：** 集合A和集合B的对称差是由属于A但不属于B的元素，以及属于B但不属于A的元素组成的集合，记作A Δ B。其数学表达式为：

\[ A \Delta B = \{ x | (x \in A \text{ 且 } x \not\in B) \text{ 或 } (x \in B \text{ 且 } x \not\in A) \} \]

#### 2.3 补集

**补集（Complement）：** 对于一个给定的集合U（通常称为全集），集合A的补集是所有不属于A但属于U的元素组成的集合，记作A' 或 AC。其数学表达式为：

\[ A' = U - A = \{ x | x \in U \text{ 且 } x \not\in A \} \]

### 第3章：集合的基数与势

#### 3.1 基数的定义

**基数（Cardinality）：** 集合中元素的数量称为该集合的基数，记作|A|。如果两个集合的基数相等，则称这两个集合是等势的。

#### 3.2 势的概念与运算

**势（ Cardinality）：** 势是集合的一个属性，表示集合中元素的数量。对于任意集合A，其势记作card(A)。

- **有限集合的势：** 如果集合A是有限集合，则其势为集合中元素的数量。
- **无限集合的势：** 如果集合A是无限集合，则其势为无穷大。

**势的运算：** 
- **加法：** 如果集合A和B都是有限集合，则A和B的并集的势等于A和B的势之和。
- **乘法：** 如果集合A和B都是有限集合，则A和B的交集的势等于A和B的势之积。
- **减法：** 如果集合A和B都是有限集合，则A和B的差集的势等于A的势减去B的势。

#### 3.3 可数集与不可数集

**可数集（Countable Set）：** 如果一个集合的元素可以与自然数建立一一对应关系，则称该集合是可数的。

**不可数集（Uncountable Set）：** 如果一个集合的元素不能与自然数建立一一对应关系，则称该集合是不可数的。
