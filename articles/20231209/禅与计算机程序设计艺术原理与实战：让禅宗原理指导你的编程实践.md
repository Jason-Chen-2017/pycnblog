在本节中，我们将通过一个简单的排序问题来展示如何将禅宗原理应用到编程实践中。

## 4.1 问题描述

给定一个整数数组，请编写一个函数，用于将数组中的元素按照升序排列。

## 4.2 解决方案

我们可以使用冒泡排序算法来解决这个问题。冒泡排序算法的核心思想是通过多次对数组中的元素进行交换，使得较大的元素逐渐向数组的末尾移动，较小的元素逐渐向数组的开头移动。

以下是冒泡排序算法的具体实现：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [5, 2, 8, 1, 9]
print(bubble_sort(arr))
```

在这个例子中，我们首先定义了一个名为 `bubble_sort` 的函数，用于对整数数组进行冒泡排序。然后，我们创建了一个整数数组 `arr`，并将其传递给 `bubble_sort` 函数，以便对其进行排序。最后，我们打印了排序后的数组。
