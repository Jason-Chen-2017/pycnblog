# 4.具体代码实例和详细解释说明
在这里，我们将通过一个简单的Python程序来演示如何遵循代码规范和最佳实践。

```python
# 导入模块
import math

# 定义函数
def factorial(n):
    # 检查输入参数
    if not isinstance(n, int) or n < 0:
        raise ValueError("输入参数必须是非负整数")
    # 递归计算阶乘
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

# 主程序
if __name__ == "__main__":
    # 输入值
    n = 5
    # 计算阶乘
    result = factorial(n)
    # 输出结果
    print("5的阶乘为:", result)
```

在这个程序中，我们遵循以下规范和最佳实践：
1. 使用中文注释解释代码的功能和逻辑。
2. 使用函数来组织代码，将相关功能放入一个函数中。
3. 使用`isinstance`函数来检查输入参数的类型。
4. 使用`raise`语句来抛出异常，当输入参数不符合要求时。
