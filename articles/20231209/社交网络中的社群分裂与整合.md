## 4.1Python代码实例

```python
import matplotlib.pyplot as plt

# 创建网络
G = nx.Graph()
G.add_edges_from([(0, 1), (0, 2), (0, 3), (1, 2), (1, 3), (2, 3)])

# 社群分裂
communities = nx.girvan_newman(G)

# 社群整合
communities = nx.girvan_newman(G, communities)

# 绘制网络
nx.draw(G, with_labels=True)
plt.show()
```

在这个代码实例中，我们使用Python的networkx库来创建一个简单的网络。我们首先创建一个包含5个节点和6个边的网络，然后使用Girvan-Newman算法进行社群分裂与整合。最后，我们使用matplotlib库来绘制网络图。
