# 2.核心概念与联系
# 2.1概率论基本概念
概率论的基本概念包括事件、样本空间、概率空间、随机变量、期望、方差等。

事件：是一个可能发生的结果。

样本空间：是所有可能发生的事件的集合。

概率空间：是一个包含样本空间和概率的集合。

随机变量：是一个可以取不同值的变量。

期望：是随机变量的数学期望。

方差：是随机变量的数学方差。

# 2.2统计学基本概念
统计学的基本概念包括样本、参数、统计量、假设检验、回归分析等。

样本：是从总体中随机抽取的一部分数据。

参数：是总体的一些特征。

统计量：是样本的一些特征。

假设检验：是用来检验一个假设是否成立的方法。

回归分析：是用来预测一个变量的值的方法。

# 2.3概率论与统计学的联系
概率论和统计学是相互联系的。概率论是统计学的基础，统计学是概率论的应用。概率论提供了概率的定义和计算方法，统计学提供了从数据中抽取信息和模式的方法。
