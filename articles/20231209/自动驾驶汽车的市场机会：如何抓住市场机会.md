```python
import numpy as np

# 读取雷达数据
data = np.load('radar_data.npy')

# 使用阈值滤波器滤除噪声
filtered_data = np.where(data > 30, 1, 0)

# 使用连通域分割算法分割目标
labels, num_labels = np.unique(filtered_data, return_inverse=True)

# 计算目标的位置和速度
positions = np.where(labels == 0)
speeds = np.diff(positions) / np.diff(labels)

# 显示目标的位置和速度
print('Target positions:', positions)
print('Target speeds:', speeds)
```

在这个代码实例中，我们首先使用NumPy库读取雷达数据，然后使用阈值滤波器滤除噪声，接着使用连通域分割算法分割目标，最后计算目标的位置和速度，并显示结果。
