# 2.核心概念与联系

## 2.1.随机过程

随机过程是一种随机变量序列，其中随机变量在时间上有序。随机过程可以分为两类：离散随机过程和连续随机过程。离散随机过程的随机变量只能取有限个值，而连续随机过程的随机变量可以取无限个值。

## 2.2.马尔可夫链

马尔可夫链是一种特殊类型的随机过程，其特点是随机变量之间的相关性仅依赖于它们之间的关系，而不依赖于整个过程的历史。换句话说，给定当前状态，将来状态的概率分布与过去状态无关。

## 2.3.局部马尔可夫链

局部马尔可夫链是一种特殊类型的马尔可夫链，其特点是随机变量之间的相关性仅依赖于它们之间的关系，而不依赖于整个过程的历史。与全局马尔可夫链不同，局部马尔可夫链的相关性仅在局部有效。

## 2.4.局部随机过程

局部随机过程是一种特殊类型的随机过程，其特点是随机变量之间的相关性仅依赖于它们之间的关系，而不依赖于整个过程的历史。与全局随机过程不同，局部随机过程的相关性仅在局部有效。

```python
# 初始状态向量
pi = np.array([0.5, 0.5])

# 马尔可夫链的转移概率矩阵
P = np.array([
    [0.8, 0.2],
    [0.3, 0.7]
])

# 局部马尔可夫链的局部转移概率矩阵

    [0.8, 0.2],
    [0.3, 0.7]
])

# 马尔可夫链的状态转移
next_state = np.dot(pi, P)
print("马尔可夫链的下一时刻状态:", next_state)

# 局部马尔可夫链的状态转移
next_state_local = np.dot(pi, Q)
print("局部马尔可夫链的下一时刻状态:", next_state_local)
```
