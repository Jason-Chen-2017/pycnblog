## 3.3 数学模型公式详细讲解

在这一部分，我们将详细讲解一些常用的数学模型公式，以帮助用户更好地理解数据分析的原理和方法。

### 3.3.1 线性回归

线性回归是一种用于预测因变量的方法，基于因变量与自变量之间的线性关系。线性回归的数学模型公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

其中，$y$ 是因变量，$x_1, x_2, ..., x_n$ 是自变量，$\beta_0, \beta_1, ..., \beta_n$ 是回归系数，$\epsilon$ 是误差项。

### 3.3.2 逻辑回归

逻辑回归是一种用于预测分类变量的方法，基于因变量与自变量之间的逻辑关系。逻辑回归的数学模型公式为：

$$
P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n)}}
$$

其中，$P(y=1)$ 是因变量的概率，$x_1, x_2, ..., x_n$ 是自变量，$\beta_0, \beta_1, ..., \beta_n$ 是回归系数。

### 3.3.3 支持向量机

支持向量机是一种用于解决线性可分问题的方法，可以处理非线性问题通过内积映射和核函数。支持向量机的数学模型公式为：

$$
f(x) = \text{sgn}(\sum_{i=1}^n \alpha_i y_i K(x_i, x) + b)
$$

其中，$f(x)$ 是因变量，$x_1, x_2, ..., x_n$ 是自变量，$y_1, y_2, ..., y_n$ 是标签，$\alpha_1, \alpha_2, ..., \alpha_n$ 是权重，$K(x_i, x)$ 是内积映射和核函数，$b$ 是偏置项。

### 3.3.4 随机森林

随机森林是一种用于解决分类和回归问题的方法，可以处理高维数据和非线性问题。随机森林的数学模型公式为：

$$
\hat{y} = \frac{1}{K} \sum_{k=1}^K f_k(x)
$$

其中，$\hat{y}$ 是预测值，$K$ 是决策树的数量，$f_k(x)$ 是第 $k$ 个决策树的预测值。

### 3.3.5 梯度下降

梯度下降是一种用于优化损失函数的方法，可以处理各种类型的损失函数和优化目标。梯度下降的数学模型公式为：

$$
\theta_{t+1} = \theta_t - \alpha \nabla J(\theta_t)
$$

其中，$\theta_{t+1}$ 是更新后的参数，$\theta_t$ 是当前参数，$\alpha$ 是学习率，$J(\theta_t)$ 是损失函数，$\nabla J(\theta_t)$ 是损失函数的梯度。

## 4.1 线性回归

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 5, 4, 5])

# 模型
model = LinearRegression()

# 训练
model.fit(x.reshape(-1, 1), y)

# 预测
pred = model.predict(x.reshape(-1, 1))

# 输出
print(model.coef_, model.intercept_)
print(pred)
```

## 4.5 梯度下降

```python
import numpy as np

# 数据
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 5, 4, 5])

# 模型
def loss(theta):
    return np.sum((y - (theta[0] + theta[1]*x))**2)

# 梯度
def grad(theta):
    return np.array([2*(y - (theta[0] + theta[1]*x)), 2*theta[1]*x])

# 参数
theta = np.array([0, 0])

# 学习率
alpha = 0.01

# 训练
for i in range(1000):
    grad_theta = grad(theta)
    theta = theta - alpha * grad_theta

# 预测
pred = theta[0] + theta[1]*x

# 输出
print(theta)
print(pred)
```
