# 2.核心概念与联系

在泛型编程中，我们使用泛型类型来表示一种数据类型的占位符。这个占位符可以在编译时被具体的数据类型替换。这种方法使得程序员可以编写更加通用的代码，而不需要为每种数据类型编写单独的实现。

泛型编程的核心概念包括泛型类型、类型参数、约束和实例化。泛型类型是一种表示数据类型的占位符，类型参数是用于替换泛型类型的实际数据类型，约束是用于限制类型参数的范围，实例化是用于创建具体的类型实例。

```python
class Stack:
    def __init__(self, T):
        self.items = []
        self.type_param = T

    def push(self, value):
        if not isinstance(value, self.type_param):
            raise ValueError("Value must be of type " + str(self.type_param))
        self.items.append(value)

    def pop(self):
        if len(self.items) == 0:
            raise IndexError("Stack is empty")
        return self.items.pop()
```
