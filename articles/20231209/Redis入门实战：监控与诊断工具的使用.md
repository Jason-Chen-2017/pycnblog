Redis（Remote Dictionary Server）是一个开源的高性能的键值存储系统，由 Salvatore Sanfilippo 开发。它支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以恢复起来。Redis 不仅仅支持简单的键值对类型的数据，同时还提供列表、集合、有序集合及哈希等数据结构的存储。

Redis 支持多种语言的客户端库，包括 Android、iOS、Java、C、C++、Python、Ruby、Go、Node.js 等，并且可以在内存中进行数据的存储和操作，因此 Redis 的性能远远超过任何的磁盘数据库。
