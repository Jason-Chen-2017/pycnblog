辐射温度公式可以用来计算宇宙背景辐射的辐射温度。辐射温度公式为：

$$
T = \frac{h \nu}{k T}

$$

辐射分布公式可以用来计算宇宙背景辐射的辐射分布。辐射分布公式为：

$$
I(\theta) = I_0 \cos^2 \theta

其中，$I(\theta)$ 是辐射强度，$I_0$ 是辐射强度在方向$\theta=0$时的值，$\theta$ 是观测方向角。

```python
import numpy as np

# 辐射温度计算
def calculate_temperature(h, nu, k, T):
    return h * nu / (k * T)

# 辐射波长计算
def calculate_wavelength(c, f):
    return c / f

# 辐射分布计算
def calculate_distribution(I0, theta):
    return I0 * np.cos(theta)**2

# 参数设置
h = 6.62607004e-34  # 普朗克常数，J·s
c = 299792458  # 光速，m/s
k = 1.38064852e-23  # 布林兹-卢梭常数，J/K
T = 2.725  # 宇宙背景辐射温度，K
nu = 2.4e11  # 微波频率，Hz
f = nu / c  # 微波频率，Hz
I0 = 2.725e-4  # 辐射强度在方向$\theta=0$时的值，W/m^2
theta = np.pi / 3  # 观测方向角，rad

# 计算结果
temperature = calculate_temperature(h, nu, k, T)
wavelength = calculate_wavelength(c, f)
distribution = calculate_distribution(I0, theta)

print("辐射温度：", temperature, "K")
print("辐射波长：", wavelength, "m")
print("辐射分布：", distribution, "W/m^2")
```
