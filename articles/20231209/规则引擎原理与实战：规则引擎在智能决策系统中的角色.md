## 4.1 代码实例

```python
# 定义规则
rules = [
    {
        "condition": "age >= 18",
        "action": "grant_access"
    },
    {
        "condition": "age < 18",
        "action": "deny_access"
    }
]

# 执行规则
for rule in rules:
    if eval(rule["condition"]):
        rule["action"]()
```

## 4.2 代码解释

### 4.2.1 规则定义

在代码实例中，我们通过一个列表来定义规则的结构和语法。每个规则包括一个条件部分（条件表达式）和一个操作部分（动作）。条件部分用于判断是否满足某个条件，操作部分用于执行某个动作。

### 4.2.2 规则执行

在代码实例中，我们通过一个循环来执行规则。首先，我们遍历规则列表，然后根据给定的条件部分来判断是否满足某个条件。如果满足条件，则执行对应的操作部分。如果不满足条件，则跳过当前规则并执行下一个规则。
