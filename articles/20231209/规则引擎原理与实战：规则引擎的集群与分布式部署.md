# 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来解释规则引擎的工作原理。我们将使用Python来编写规则引擎的代码。

```python
# 定义规则库
rules = [
    {
        "condition": "age >= 18",
        "action": "grant_access"
    },
    {
        "condition": "age < 18",
        "action": "deny_access"
    }
]

# 初始化工作内存
work_memory = {"age": 20}

# 执行规则
for rule in rules:
    if eval(rule["condition"]):
        rule["action"](work_memory)
        break

# 输出结果
print(work_memory)  # {"age": 20, "access": "grant_access"}
```

在这个代码实例中，我们定义了一个规则库，其中包含两个规则。每个规则包含一个条件和一个操作。我们还初始化了工作内存，并执行了规则。最后，我们输出了工作内存的结果。
