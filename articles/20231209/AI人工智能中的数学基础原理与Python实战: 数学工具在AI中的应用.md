## 4.1 线性代数

以下是一个线性方程组的Python代码实例：

```python
import numpy as np

# 定义线性方程组的系数和常数项
A = np.array([[1, 2], [3, 4]])
b = np.array([5, 6])

# 使用NumPy库的linalg.solve函数求解线性方程组

# 输出结果

```

在这个例子中，我们首先定义了线性方程组的系数和常数项，然后使用NumPy库的linalg.solve函数求解线性方程组。最后，我们输出了求解结果。
