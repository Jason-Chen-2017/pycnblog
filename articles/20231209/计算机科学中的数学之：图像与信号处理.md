# 2.核心概念与联系

在图像与信号处理中，信号和图像是两个核心概念。信号是时间域上的变化，图像是空间域上的变化。信号可以是连续信号（如音频信号）或离散信号（如数字图像），图像可以是彩色图像或黑白图像。

信号与图像处理的核心概念包括：

1. 信号与图像的表示：信号可以用时域函数、频域函数、时域序列、频域序列等多种方式来表示；图像可以用二维数组、灰度值、颜色值等方式来表示。

2. 信号与图像的处理：信号与图像处理包括滤波、变换、压缩、分析等多种方法，这些方法可以用来提取信号与图像中的有用信息，或者对信号与图像进行压缩、恢复、分析等操作。

3. 信号与图像的特征：信号与图像的特征是信号与图像中的一些特点，例如信号的频率、幅度、相位等特点；图像的边缘、纹理、颜色等特点。

4. 信号与图像的应用：信号与图像处理技术的应用范围非常广泛，包括图像处理、图像识别、图像压缩、信号处理、信号分析、通信技术等。

1. 滤波：滤波是图像与信号处理中的一种重要方法，用于去除信号或图像中的噪声。滤波可以分为空域滤波和频域滤波两种方法。空域滤波是通过使用卷积操作来应用滤波器来去除信号或图像中的噪声，频域滤波是通过使用频域滤波器来去除信号或图像中的噪声。滤波的核心思想是利用滤波器来保留信号或图像中的有用信息，同时去除噪声。

2. 变换：变换是图像与信号处理中的一种重要方法，用于将信号或图像从时域转换到频域，或者将信号或图像从空域转换到特征域。变换可以分为傅里叶变换、傅里叶逆变换、傅里叶频谱、傅里叶相位、傅里叶系数、傅里叶定理、谱密度、谱密度图等方法。变换的核心思想是利用变换来分析信号或图像中的特征，以提取有用信息。

3. 压缩：压缩是图像与信号处理中的一种重要方法，用于将信号或图像的大小减小，以减少存储空间和传输带宽。压缩可以分为有损压缩和无损压缩两种方法。无损压缩是通过使用压缩算法来将信号或图像的大小减小，但是压缩后的信号或图像与原始信号或图像具有相同的质量和精度；有损压缩是通过使用压缩算法来将信号或图像的大小减小，但是压缩后的信号或图像与原始信号或图像之间存在一定的质量损失。压缩的核心思想是利用压缩算法来减小信号或图像的大小，以减少存储空间和传输带宽。

4. 分析：分析是图像与信号处理中的一种重要方法，用于将信号或图像中的特征进行分析，以提取有用信息。分析可以分为特征提取、特征提取方法、特征选择、特征选择方法、特征表示、特征表示方法等方法。分析的核心思想是利用分析方法来分析信号或图像中的特征，以提取有用信息。
