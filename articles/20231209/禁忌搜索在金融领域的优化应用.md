# 4.具体代码实例和详细解释说明

在这里，我们通过一个简单的贷款利率优化问题来展示禁忌搜索的具体代码实例。

```python
import numpy as np
from itertools import product

# 初始化搜索空间
X = np.random.rand(100, 1)

# 初始化禁忌列表
T = []

# 初始化目标函数
f = lambda x: np.sum(x)

# 初始化探索步长
delta = 0.1

# 初始化终止条件
iter_max = 1000
theta = 0.9

# 初始化当前解
x_current = X[0]

# 初始化当前解加入禁忌列表
T.append(x_current)

# 初始化迭代次数
iter = 0

# 开始禁忌搜索
while iter < iter_max and np.max(f(X)) < theta:
    # 探索邻域
    N_x = [x for x in X if np.linalg.norm(x - x_current) <= delta]

    # 选择最优邻域解
    x_best = max(N_x, key=f)

    # 更新当前解
    x_current = x_best

    # 更新禁忌列表
    T.append(x_current)

    # 更新迭代次数
    iter += 1

# 输出最优解
print("最优解:", x_best)
```

在这个代码实例中，我们首先初始化了搜索空间、禁忌列表、目标函数、探索步长、终止条件、当前解等变量。然后我们开始禁忌搜索，通过探索邻域、选择最优邻域解、更新当前解和禁忌列表来找到最优解。最后，我们输出了最优解。
