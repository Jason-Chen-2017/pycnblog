## 1.1 背景介绍

人造卫星和太空探索的起源可以追溯到19世纪末和20世纪初的科学家们对宇宙的探索兴趣。在1926年，俄罗斯科学家弗朗索瓦·朔洛夫（Franz von Schubert）提出了人造卫星的概念，并提出了一种用于实现这一目标的火箭发动机的设计。

在1957年，俄罗斯成功发射了第一颗人造卫星，名为“斯莱瑟-1”（Sputnik 1），这是人类太空探索的开始。随后，美国也成功发射了第一颗人造卫星，名为“探索者-1”（Explorer 1）。

## 1.2 核心概念与联系

人造卫星是一种人工制造的天体，通常由金属或其他材料制成，可以绕地球或其他星球进行运动。人造卫星可以用于各种目的，如气候观测、地球观测、导航、通信、军事和太空探索。

太空探索是指人类探索太空的活动，包括发射人造卫星、探索月球、火星和其他行星，以及探索太空中的其他天体和现象。太空探索需要一系列的技术，包括火箭技术、太空舱技术、导航技术、通信技术、能源技术等。

```python
import numpy as np

def convert_coordinates(latitude, longitude, altitude, ellipsoid_semi_major_axis, ellipsoid_flattening):
    x = (ellipsoid_semi_major_axis * np.cos(latitude) * np.cos(longitude)) / np.sqrt(1 - ellipsoid_flattening * np.sin(latitude)**2)
    y = (ellipsoid_semi_major_axis * np.cos(latitude) * np.sin(longitude)) / np.sqrt(1 - ellipsoid_flattening * np.sin(latitude)**2)
    z = (ellipsoid_semi_major_axis * np.sin(latitude)) / np.sqrt(1 - ellipsoid_flattening * np.sin(latitude)**2)
    return x, y, z

latitude = np.deg2rad(40.7128)  # New York City's latitude
longitude = np.deg2rad(-74.0060)  # New York City's longitude
altitude = 400  # Altitude in meters
ellipsoid_semi_major_axis = 6378100  # Earth's equatorial radius
ellipsoid_flattening = 1 / 298.257223563  # Earth's flattening

x, y, z = convert_coordinates(latitude, longitude, altitude, ellipsoid_semi_major_axis, ellipsoid_flattening)
print("X:", x, "m")
print("Y:", y, "m")
print("Z:", z, "m")
```
