## 3.1 动态规划
动态规划是一种解决最优化问题的方法，它通过构建一个状态表格来逐步计算最优解。动态规划的核心思想是将问题分解为子问题，并将子问题的解组合成最终解。

动态规划的具体操作步骤如下：
1. 确定问题的状态：将问题分解为多个子问题，每个子问题的状态需要记录。
2. 确定状态转移方程：根据子问题的状态，确定状态转移方程，用于计算下一个状态的值。
3. 确定初始状态：确定问题的初始状态，并计算其对应的值。
4. 确定终止条件：确定问题的终止条件，当满足终止条件时，算法停止运行。
5. 计算最优解：根据状态转移方程，逐步计算最优解。

动态规划的数学模型公式为：
$$
dp[i] = \min_{0 \leq j \leq i-1} \{ dp[j] + cost[j, i] \}
$$

## 3.2 贪心算法
贪心算法是一种解决最优化问题的方法，它通过在每个步骤中选择最优的选择来逐步构建解。贪心算法的核心思想是在每个步骤中选择最优的选择，从而得到最优的解。

贪心算法的具体操作步骤如下：
1. 确定问题的目标：确定问题的目标，即要最小化或最大化的目标函数。
2. 确定问题的约束条件：确定问题的约束条件，即问题的可行解的条件。
3. 确定问题的选择：确定问题的选择，即在每个步骤中可以选择的选择。
4. 确定问题的贪心策略：确定问题的贪心策略，即在每个步骤中选择最优的选择。
5. 计算最优解：根据贪心策略，逐步计算最优解。

贪心算法的数学模型公式为：
$$
x^* = \arg \min_{x \in X} \{ f(x) \}
$$

分治算法的具体操作步骤如下：
1. 确定问题的分解：将问题分解为多个子问题，每个子问题的解需要记录。
2. 确定问题的合并：根据子问题的解，确定问题的合并方法，用于计算最终解。
3. 确定问题的初始状态：确定问题的初始状态，并计算其对应的值。
4. 确定问题的终止条件：确定问题的终止条件，当满足终止条件时，算法停止运行。
5. 计算最优解：根据分解和合并方法，逐步计算最优解。

```python
def dp(n):
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n]

print(dp(100))
```

```python
def greedy(n, k):
    res = 0
    for i in range(1, n + 1):
        if i % k == 0:
            res += i
    return res

print(greedy(100, 5))
```
在这个例子中，我们使用贪心算法来解决1到n的整数中，除以k的和的问题。我们定义了一个贪心策略，即每次选择除以k的整数。然后，我们根据贪心策略计算每个状态的值，最后返回最终的解。

```python
def divide_and_conquer(n):

    else:
```
