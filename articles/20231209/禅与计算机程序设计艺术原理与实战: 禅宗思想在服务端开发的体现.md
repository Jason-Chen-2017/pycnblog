禅宗思想起源于中国，是一种哲学思想，强调内心的平静和自然的行动。在计算机程序设计中，禅宗思想可以帮助我们更好地理解和应用计算机程序的设计原则。

禅宗思想的核心概念包括：

1. 直指人心，砥柱不渝：禅宗强调直接触及人心的真实，而不是通过抽象的理论来解释事物。在计算机程序设计中，这意味着我们应该关注代码的实际效果，而不是过于关注理论和抽象的概念。

2. 无所谓：禅宗认为，我们应该放弃对结果的预期，而是专注于当下的行动。在计算机程序设计中，这意味着我们应该关注当前的代码实现，而不是过于关注未来的预测和结果。

3. 无量寿：禅宗强调永恒的时间和空间，这意味着我们应该关注程序的长期稳定性和可靠性。在计算机程序设计中，这意味着我们应该关注程序的性能和可靠性，而不是过于关注短期的效率和速度。

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

我们可以通过以下步骤来解释这个代码的工作原理：

1. 首先，我们定义了一个变量 `min_index`，用于记录当前最小元素的下标。
2. 然后，我们使用内层循环来遍历数组，找到最小的元素并更新 `min_index`。
3. 最后，我们使用多重赋值来交换当前元素和最小元素的位置。
4. 我们重复这个过程，直到整个数组被排序。
