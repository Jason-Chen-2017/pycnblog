# 2.核心概念与联系
在实时数据处理和流式计算中，我们需要了解以下几个核心概念：

- **数据流**：数据流是一种连续的数据序列，通常用于表示实时数据。数据流可以是来自各种数据源，如传感器、网络流量、日志文件等。

- **流处理**：流处理是一种实时数据处理技术，它可以在数据流中进行实时分析和处理。流处理可以用于实时监控、实时报警、实时推荐等应用场景。

- **流计算**：流计算是一种基于流处理的计算模型，它可以用于实时数据处理和流式计算。流计算可以用于实时数据分析、实时数据挖掘、实时决策等应用场景。
