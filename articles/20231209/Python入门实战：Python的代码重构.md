## 1.6 Python的代码重构具体代码实例和解释
以下是一个Python的代码重构实例：
```python
# 原始代码
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    return x / y

# 重构后代码
from functools import reduce

def math_operation(x, y, operation):
    operations = {
        '+': add,
        '-': subtract,
        '*': multiply,
        '/': divide
    }
    return operations[operation](x, y)

# 使用重构后的代码
result = math_operation(10, 5, '+')
print(result)  # 输出: 15
```
