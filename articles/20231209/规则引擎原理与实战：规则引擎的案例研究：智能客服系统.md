## 2.1 规则

规则是指一种描述问题解决方案的语句，它由条件和动作组成。条件是一个布尔表达式，用于判断某个事实是否满足。动作是一种操作，当条件为真时会被执行。规则可以用以下形式表示：

```

```

例如，一个简单的规则可能是：

```
IF 用户问题是“我需要帮助” THEN 回复“我很高兴帮助您”
```

```

```

