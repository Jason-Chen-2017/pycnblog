# 4.具体代码实例和详细解释说明

在本节中，我们将通过具体代码实例来解释泛型编程的实际应用。

例子1：

```python
def generic_function(x: T) -> T:
    return x * 2
```

在这个例子中，我们定义了一个泛型函数`generic_function`，它接受一个类型参数`T`，并返回一个`T`类型的值。我们使用`x * 2`来计算`x`的双倍值。

例子2：

```python
class GenericClass:
    def __init__(self, x: T) -> None:
        self.x = x

    def method(self, y: T) -> T:
        return self.x + y
```

在这个例子中，我们定义了一个泛型类`GenericClass`，它接受一个类型参数`T`，并包含一个`method`方法。`method`方法接受一个`T`类型的参数`y`，并返回一个`T`类型的值。我们使用`self.x + y`来计算`self.x`和`y`的和值。
