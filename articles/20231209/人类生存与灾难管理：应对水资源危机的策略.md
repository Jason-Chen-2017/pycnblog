### 4.1 水资源供需分析

```python
import numpy as np
import pandas as pd
from scipy.optimize import linprog

# 收集水资源的供给和需求数据
data = pd.read_csv("water_data.csv")

# 使用线性规划模型分析水资源的供需关系
result = linprog(data["supply"],
                 A_eq=np.ones(len(data)),
                 b_eq=data["demand"],
                 bounds=[(0, None)] * len(data))

print(result)
```

### 4.2 水资源利用率

```python
import numpy as np
from sklearn.ensemble import RandomForestRegressor

# 收集水资源利用率数据
data = pd.read_csv("water_usage_data.csv")

# 使用随机森林算法预测水资源利用率

model.fit(data[["supply", "demand"]], data["usage"])

# 预测新的水资源利用率
new_data = np.array([[1000, 500]])
pred = model.predict(new_data)
print(pred)
```
