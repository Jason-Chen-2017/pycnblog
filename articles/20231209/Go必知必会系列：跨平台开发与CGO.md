# 4.具体代码实例和详细解释说明

以下是一个简单的CGO代码实例：

```go
package main

import "C"

//export HelloWorld
func HelloWorld() {
    C.printf("Hello, World!\n")
}

func main() {
    C.HelloWorld()
}
```

在这个例子中，我们创建了一个Go程序，它调用了一个C语言函数`HelloWorld`。`HelloWorld`函数使用C语言的`printf`函数打印出"Hello, World!"。

首先，我们需要使用gcc编译C语言代码：

```bash
$ gcc -shared -o hello.so hello.c
```

然后，我们可以在Go程序中调用C语言代码：

```go
package main

import "C"

//export HelloWorld
func HelloWorld() {
    C.printf("Hello, World!\n")
}

func main() {
    C.HelloWorld()
}
```

最后，我们可以使用go build命令将Go程序编译成可执行文件：

```bash
$ go build main.go
```

运行Go程序，并观察C语言代码的输出结果：

```bash
$ ./main
Hello, World!
```
