## 1.4 具体代码实例和详细解释说明

在混合云的数据湖中，可以使用以下代码实例来实现数据的存储、处理和分析：

1. 数据存储：

```python
from pyspark.sql import SparkSession

spark = SparkSession.builder.appName("DataLake").getOrCreate()

data = spark.read.csv("data.csv", header=True, inferSchema=True)

data.write.parquet("data_lake")
```

2. 数据处理：

```python
from pyspark.sql import SparkSession
from pyspark.sql.functions import col, when

spark = SparkSession.builder.appName("DataLake").getOrCreate()

data = spark.read.parquet("data_lake")

data = data.withColumn("age", when(col("age") > 18, col("age")).otherwise(0))

data.show()
```

3. 数据分析：

```python
from pyspark.sql import SparkSession
from pyspark.sql.functions import avg, count

spark = SparkSession.builder.appName("DataLake").getOrCreate()

data = spark.read.parquet("data_lake")

avg_age = data.select(avg("age")).collect()[0][0]

print("Average age:", avg_age)
```
