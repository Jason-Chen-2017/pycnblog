# 2.核心概念与联系

图神经网络是一种特殊类型的神经网络，它可以处理图结构数据。图结构是一种特殊类型的数据结构，由一组节点（vertices）和一组边（edges）组成。节点表示图中的实体，如人、地点或物品，而边表示实体之间的关系。

图神经网络的核心概念包括：

1.图卷积层：图卷积层是图神经网络的基本组成部分，它可以在图上进行卷积运算，以提取图结构中的特征。图卷积层可以看作是传统卷积神经网络（CNNs）的扩展，它可以处理不同类型的图结构。

2.邻域聚合：邻域聚合是图神经网络中的一种信息传播方法，它可以将节点的邻域信息聚合到节点本身，以捕捉图结构中的信息。邻域聚合可以看作是信息传播的一种方法，它可以在图上传播信息，以提高模型的性能。

3.读取节点特征：图神经网络可以读取节点的特征信息，以捕捉节点本身的信息。节点特征可以是节点的属性，如节点的类别、属性或关系。

4.读取边特征：图神经网络可以读取边的特征信息，以捕捉边之间的关系。边特征可以是边的属性，如边的权重、类别或关系。

5.非线性激活函数：图神经网络可以使用非线性激活函数，以捕捉图结构中的复杂关系。非线性激活函数可以帮助模型学习更复杂的模式，从而提高模型的性能。

2.对于每个节点，对其邻域信息进行聚合，以捕捉图结构中的信息。聚合操作可以使用各种方法，如平均值、最大值或权重平均值等。

3.对于每个节点，对其特征信息进行读取，以捕捉节点本身的信息。特征信息可以是节点的属性、类别或关系等。

4.对于每个节点，对其边特征信息进行读取，以捕捉边之间的关系。边特征信息可以是边的属性、类别或关系等。

6.对于每个节点，对其特征信息进行卷积运算，以提取图结构中的特征。卷积运算可以使用各种方法，如卷积核、滤波器或卷积层等。

from tensorflow.keras.layers import Input, Dense, Conv1D, GlobalMaxPooling1D
from tensorflow.keras.models import Model

# 定义输入层
input_layer = Input(shape=(input_dim,))

# 定义卷积层
conv_layer = Conv1D(filters=64, kernel_size=3, activation='relu')(input_layer)

# 定义邻域聚合层
pooling_layer = GlobalMaxPooling1D()(conv_layer)

# 定义全连接层
dense_layer = Dense(64, activation='relu')(pooling_layer)

# 定义输出层
output_layer = Dense(num_classes, activation='softmax')(dense_layer)

# 定义模型
model = Model(inputs=input_layer, outputs=output_layer)

# 编译模型
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# 训练模型
model.fit(x_train, y_train, epochs=10, batch_size=32)
