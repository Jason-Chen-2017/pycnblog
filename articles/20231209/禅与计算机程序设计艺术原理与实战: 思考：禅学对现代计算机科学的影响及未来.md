动态规划是一种解决最优化问题的算法方法。它通过将问题分解为子问题，并将子问题的解存储在一个动态表中，从而避免重复计算。在动态规划算法中，我们需要思考和分析问题，以及设计合适的动态表和子问题。通过学习“无思议”的思想，我们可以放下过于思考和概念的依赖，从而更好地体验动态规划算法的真实性。

## 3.2 直接体验与贪心算法

贪心算法是一种解决最优化问题的算法方法。它通过在每个决策中选择最优的选项，从而逐步构建最优解。在贪心算法中，我们需要直接体验问题的解，以及选择最优的选项。通过学习“直接体验”的思想，我们可以更好地理解和应用贪心算法，从而更好地解决问题。

## 3.3 简单与分治法

## 4.1 无思议与动态规划

```python
def dp(n):
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n]

print(dp(10))  # 55
```

在这个代码实例中，我们使用动态规划算法来解决斐波那契数列问题。我们将问题分解为子问题，并将子问题的解存储在一个动态表中。通过学习“无思议”的思想，我们可以放下过于思考和概念的依赖，从而更好地体验动态规划算法的真实性。

## 4.2 直接体验与贪心算法

```python
def greedy(n):
    coins = [1, 5, 10, 25]
    result = 0
    for coin in coins:
        result += n // coin
        n %= coin
    return result

print(greedy(100))  # 41
```

在这个代码实例中，我们使用贪心算法来解决最大面额硬币问题。我们通过在每个决策中选择最优的选项，从而逐步构建最优解。通过学习“直接体验”的思想，我们可以更好地理解和应用贪心算法，从而更好地解决问题。

## 4.3 简单与分治法

```python
def divide_and_conquer(n):
    if n <= 1:
        return n
    else:
        return divide_and_conquer(n // 2) + divide_and_conquer(n - n // 2)

print(divide_and_conquer(10))  # 4
```

在这个代码实例中，我们使用分治法来解决乘法问题。我们将问题分解为子问题，并将子问题的解组合为原问题的解。通过学习“简单与复杂”的思想，我们可以将问题简化，从而更好地解决问题。
