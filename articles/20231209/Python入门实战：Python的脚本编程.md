# 2.核心概念与联系

## 2.1 Python简介
Python是一种高级编程语言，由Guido van Rossum于1991年创建。它具有简洁的语法和易于阅读的代码，使得编写程序变得更加简单。Python支持多种编程范式，包括面向对象编程、函数式编程和过程式编程。

# 4.具体代码实例和详细解释说明

在这里，我们将提供一个Python脚本的实例，并详细解释其代码。

```python
# 导入模块
import os

# 定义函数
def greet(name):
    print(f"Hello, {name}!")

# 主函数
def main():
    # 获取用户输入的名字
    name = input("请输入您的名字：")

    # 调用greet函数
    greet(name)

# 执行主函数
if __name__ == "__main__":
    main()
```

这个脚本的主要功能是打印出用户输入的名字。它首先导入了`os`模块，然后定义了一个名为`greet`的函数，该函数接收一个名字作为参数，并打印出一个带有名字的问候语。最后，它定义了一个名为`main`的主函数，该函数获取用户输入的名字，并调用`greet`函数。
