## 4.1房地产价格预测算法的Python实现
```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 收集房地产市场的相关数据
data = np.array([[100, 200, 300], [200, 300, 400], [300, 400, 500]])

# 使用线性回归模型来预测房地产价格
model = LinearRegression()
model.fit(data[:, [0, 1]], data[:, 2])

# 根据预测结果来决定是否购买房地产
predicted_price = model.predict([[100, 200]])
if predicted_price > 300:
    print("购买房地产")
else:
    print("不购买房地产")
```
