## 4.1 经济可持续发展模型代码实例

以下是一个简单的经济可持续发展模型代码实例，用于描述和分析经济增长、消费、投资、通货膨胀等变量之间的关系：

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 数据加载
data = pd.read_csv('economic_data.csv')

# 变量定义
Y = data['GDP']
t = data['CPI']
r = data['Interest_Rate']
e = data['Exchange_Rate']

# 模型定义
def C(Y, t):
    # 消费函数
    return Y * t

def I(r, Y):
    # 投资函数
    return r * Y

def NX(Y, e):
    # 外汇支付余额函数
    return Y * e

# 模型计算
C = C(Y, t)
I = I(r, Y)
NX = NX(Y, e)

# 数据可视化
plt.plot(Y, C, label='Consumption')
plt.plot(Y, I, label='Investment')
plt.plot(Y, NX, label='Net_Exports')
plt.xlabel('GDP')
plt.ylabel('Value')
plt.legend()
plt.show()
```
