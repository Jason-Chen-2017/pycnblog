# 2.核心概念与联系

## 2.1 分治算法的基本思想

分治算法的基本思想是将问题划分为多个子问题，递归地解决这些子问题，然后将解决的结果合并成原问题的解。这种思想可以简单地理解为“分而治之”。

分治算法与动态规划、贪心算法等其他算法思想相比，有一定的联系。例如，动态规划也是将问题划分为多个子问题，但是它在解决子问题时需要考虑子问题之间的关系，而分治算法则是在解决子问题时相互独立。贪心算法则是在每个步骤中选择最优解，而分治算法则是在递归地解决子问题后，将解决的结果合并成原问题的解。

## 4.1 代码实例：快速排序

快速排序是一种常用的分治算法，它的核心思想是将问题划分为多个子问题，递归地解决这些子问题，然后将解决的结果合并成原问题的解。

### 4.1.1 算法步骤

1. 选择一个基准值（pivot）。
2. 将数组中所有小于基准值的元素放到左边，所有大于基准值的元素放到右边。
3. 递归地对左边和右边的子数组进行快速排序。
4. 将基准值放到正确的位置。

### 4.1.2 代码实现

```python
def quick_sort(arr, left, right):
    if left < right:
        pivot_index = partition(arr, left, right)
        quick_sort(arr, left, pivot_index - 1)
        quick_sort(arr, pivot_index + 1, right)

def partition(arr, left, right):
    pivot = arr[right]
    i = left - 1
    for j in range(left, right):
        if arr[j] < pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i + 1], arr[right] = arr[right], arr[i + 1]
    return i + 1

arr = [3, 5, 1, 7, 9, 2, 4, 6, 8]
quick_sort(arr, 0, len(arr) - 1)
print(arr)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

合并排序是另一种常用的分治算法，它的核心思想是将问题划分为多个子问题，递归地解决这些子问题，然后将解决的结果合并成原问题的解。

### 4.2.1 算法步骤

1. 将数组划分为两个子数组。
2. 递归地对子数组进行合并排序。
3. 将子数组合并成原数组。

### 4.2.2 代码实现

```python
def merge_sort(arr, left, right):
    if left < right:
        mid = (left + right) // 2
        merge_sort(arr, left, mid)
        merge_sort(arr, mid + 1, right)
        merge(arr, left, mid, right)

def merge(arr, left, mid, right):
    L = arr[left:mid + 1]
    R = arr[mid + 1:right + 1]
    i = j = 0
    for k in range(left, right + 1):
        if i >= len(L):
            arr[k] = R[j]
            j += 1
        elif j >= len(R):
            arr[k] = L[i]
            i += 1
        elif L[i] <= R[j]:
            arr[k] = L[i]
            i += 1
        else:
            arr[k] = R[j]
            j += 1

arr = [3, 5, 1, 7, 9, 2, 4, 6, 8]
merge_sort(arr, 0, len(arr) - 1)
print(arr)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
