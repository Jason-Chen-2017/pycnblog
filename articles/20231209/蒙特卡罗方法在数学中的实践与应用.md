```python
import random

def monte_carlo(n):
    x = 0
    for _ in range(n):
        y = random.uniform(0, 1)
        x += y**2
    return x / n

def main():
    n = 100000
    result = monte_carlo(n)
    print("蒙特卡罗方法的结果:", result)

if __name__ == "__main__":
    main()
```

在主函数中，我们设置了一个试验次数 `n`，并调用 `monte_carlo` 函数。最后，我们打印出蒙特卡罗方法的结果。
