## 2.1 工业互联网的核心概念

## 4.1 物联网的代码实例

```python
import time
import requests

# 设置设备的IP地址和端口号
device_ip = '192.168.1.100'
device_port = 8080

# 设置需要获取的数据类型
data_type = 'temperature'

# 发送请求获取数据
response = requests.get(f'http://{device_ip}:{device_port}/data/{data_type}')

# 解析响应数据
data = response.json()

# 打印数据
print(data)
```

在这个代码实例中，我们使用Python的requests库发送HTTP请求获取设备的数据。首先，我们设置了设备的IP地址和端口号，然后设置了需要获取的数据类型（在这个例子中是温度）。接着，我们发送了一个GET请求，并解析了响应数据。最后，我们打印了数据。

## 4.2 大数据分析的代码实例

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.dropna()

# 数据聚合
data = data.groupby('date').mean()

# 数据挖掘
data = data[data['temperature'] > 30]

# 数据可视化
data.plot()
```

在这个代码实例中，我们使用Python的pandas库进行大数据分析。首先，我们读取了数据文件（在这个例子中是一个CSV文件）。接着，我们对数据进行了清洗，删除了缺失值。然后，我们对数据进行了聚合，计算了每天的平均温度。接着，我们对数据进行了挖掘，筛选出了温度大于30度的数据。最后，我们对数据进行了可视化，绘制了温度变化的图表。
