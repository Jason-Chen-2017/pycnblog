### 4.1 排序算法实例

以下是一个简单的冒泡排序算法的实现：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

冒泡排序算法的实现原理是通过不断比较相邻的元素，将较大的元素向后移动，以便在每次迭代中将最大的元素移动到正确的位置。

### 4.2 搜索算法实例

以下是一个简单的深度优先搜索算法的实现：

```python
def dfs(graph, start):
    visited = set()
    stack = [start]

    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)

    return visited
```

深度优先搜索算法的实现原理是通过不断探索一个节点的邻居，直到探索完成或者无法继续探索为止。

### 4.3 优化算法实例

以下是一个简单的梯度下降算法的实现：

```python
def gradient_descent(x, y, learning_rate, epochs):
    m, b = 0, 0

    for _ in range(epochs):
        for xi, yi in zip(x, y):
            grad_m = -2 * (sum(xi - (m * xi + b)) / len(x))
            grad_b = -2 * (sum(yi - (m * xi + b)) / len(x))
            m -= learning_rate * grad_m
            b -= learning_rate * grad_b

    return m, b
```

梯度下降算法的实现原理是通过不断更新模型参数，以便最小化损失函数。
