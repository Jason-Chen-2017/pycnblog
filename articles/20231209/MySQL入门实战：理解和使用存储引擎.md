### 4.1 InnoDB存储引擎的示例

以下是一个使用InnoDB存储引擎创建表、插入数据、查询数据的示例：

```sql
CREATE TABLE `employee` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `name` varchar(45) NOT NULL,
  `age` int(11) NOT NULL,
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8;

INSERT INTO `employee` (`name`, `age`) VALUES ('John', 30), ('Alice', 25), ('Bob', 35);

SELECT * FROM `employee`;
```

在上述示例中，我们首先创建了一个名为`employee`的表，其中包含`id`、`name`和`age`等字段。然后我们使用INSERT语句插入了三条记录，最后使用SELECT语句查询了表中的所有记录。
