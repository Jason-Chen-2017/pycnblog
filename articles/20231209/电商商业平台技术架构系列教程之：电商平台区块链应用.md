# 2.核心概念与联系

1. 区块链
2. 加密技术
3. 智能合约
4. 去中心化

## 1.区块链

区块链是一种分布式、去中心化的数字账本技术，它可以确保数据的完整性、安全性和透明度。区块链由一系列的区块组成，每个区块包含一组交易数据和一个时间戳。每个区块的数据是通过加密技术加密的，并且与前一个区块通过一个加密链接。这样，一旦一个区块被添加到区块链中，就不能被修改。

## 2.加密技术

加密技术是区块链技术的基础。在区块链中，数据通过加密技术加密，以确保数据的安全性。同时，加密技术也用于确保区块链的完整性和透明度。例如，通过使用公钥和私钥，可以确保只有具有私钥的用户可以修改区块链中的数据。

## 3.智能合约

智能合约是一种自动执行的合约，它可以在区块链上执行。智能合约可以用于实现各种业务逻辑，例如交易、支付、投票等。智能合约可以确保交易的安全性、数据的完整性和交易的透明度等方面。

## 4.去中心化

去中心化是区块链技术的核心特征。去中心化意味着没有一个中心化的机构可以控制区块链。这样，一旦一个区块被添加到区块链中，就不能被修改。这样，去中心化可以确保交易的安全性、数据的完整性和交易的透明度等方面。

## 1.创建一个简单的区块链

首先，我们需要创建一个简单的区块链。我们可以使用Python来实现这个功能。以下是一个简单的代码实例：

```python
import hashlib
import json

class Block:
    def __init__(self, index, previous_hash, timestamp, data, hash):
        self.index = index
        self.previous_hash = previous_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

    def calculate_hash(self):
        block_string = json.dumps(self.__dict__, sort_keys=True).encode('utf-8')
        return hashlib.sha256(block_string).hexdigest()

    def __repr__(self):
        return 'Block(index={}, previous_hash={}, timestamp={}, data={}, hash={})'.format(
            self.index, self.previous_hash, self.timestamp, self.data, self.hash
        )

def create_genesis_block():
    index = 0
    previous_hash = 0
    timestamp = 1546300800
    data = 'Genesis Block'
    hash = hashlib.sha256(json.dumps(locals(), sort_keys=True).encode('utf-8')).hexdigest()
    return Block(index, previous_hash, timestamp, data, hash)

def create_block(previous_block):
    index = previous_block.index + 1
    timestamp = 1546300800 + index
    data = 'Block {}'.format(index)
    hash = hashlib.sha256(json.dumps(locals(), sort_keys=True).encode('utf-8')).hexdigest()
    return Block(index, previous_block.hash, timestamp, data, hash)

genesis_block = create_genesis_block()
block_1 = create_block(genesis_block)
block_2 = create_block(block_1)
```

在这个代码实例中，我们创建了一个简单的区块链。我们首先创建了一个`Block`类，用于表示区块。然后，我们创建了一个`create_genesis_block`函数，用于创建区块链的第一个区块（称为“基础区块”或“创世区块”）。最后，我们创建了一个`create_block`函数，用于创建其他区块。

## 2.创建一个简单的智能合约

接下来，我们需要创建一个简单的智能合约。我们可以使用Solidity来实现这个功能。以下是一个简单的代码实例：

```solidity
pragma solidity ^0.5.16;

contract SimpleContract {
    address public owner;
    uint public balance;

    constructor() public {
        owner = msg.sender;
        balance = 100 ether;
    }

    function transfer(address payable recipient, uint amount) public payable {
        require(msg.value == amount, "Transfer amount must match");
        recipient.transfer(amount);
    }
}
```

在这个代码实例中，我们创建了一个简单的智能合约。我们首先定义了一个`SimpleContract`合约，它有一个`owner`地址、一个`balance`余额和一个`transfer`函数。然后，我们在合约的构造函数中初始化`owner`地址和`balance`余额。最后，我们实现了一个`transfer`函数，用于将一定数量的ETH转到指定的收件人地址。

## 3.将区块链与智能合约联系起来

最后，我们需要将区块链与智能合约联系起来。我们可以使用Web3.js来实现这个功能。以下是一个简单的代码实例：

```javascript
const Web3 = require('web3');
const web3 = new Web3('http://localhost:8545');

const abi = [
    {
        "constant": false,
        "inputs": [
            {
                "name": "recipient",
                "type": "address"
            },
            {
                "name": "amount",
                "type": "uint256"
            }
        ],
        "name": "transfer",
        "outputs": [],
        "payable": true,
        "stateMutability": "payable",
        "type": "function"
    }
];

const contractAddress = '0x5B6...';
const contract = new web3.eth.Contract(abi, contractAddress);

contract.methods.transfer('0x5B6...', 1000).send({ from: '0x5B6...', value: 1000 })
    .on('transactionHash', (hash) => { console.log('Transaction Hash:', hash); })
    .on('receipt', (receipt) => { console.log('Receipt:', receipt); })
    .on('error', (error) => { console.error('Error:', error); });
```

在这个代码实例中，我们首先创建了一个Web3实例，用于与区块链进行交互。然后，我们定义了一个`abi`数组，用于描述智能合约的接口。接下来，我们创建了一个`contract`实例，用于与智能合约进行交互。最后，我们调用智能合约的`transfer`函数，将一定数量的ETH转到指定的收件人地址。
