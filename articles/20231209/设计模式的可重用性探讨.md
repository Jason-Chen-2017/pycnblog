## 4.1 代码实例

我们将通过一个简单的例子来说明设计模式的使用方法。假设我们需要实现一个简单的计算器，可以进行加法、减法、乘法和除法的计算。我们可以使用工厂模式来实现这个计算器。

```python
# 定义一个计算器类
class Calculator:
    def __init__(self):
        pass

    def add(self, a, b):
        return a + b

    def subtract(self, a, b):
        return a - b

    def multiply(self, a, b):
        return a * b

    def divide(self, a, b):
        return a / b

# 定义一个工厂类
class CalculatorFactory:
    def __init__(self):
        pass

    def create_calculator(self):
        return Calculator()

# 使用工厂类创建计算器
calculator = CalculatorFactory().create_calculator()

# 使用计算器进行计算
result = calculator.add(2, 3)
print(result)  # 输出: 5
```

在这个例子中，我们首先定义了一个计算器类，这个类提供了加法、减法、乘法和除法的计算方法。然后我们定义了一个工厂类，这个类提供了一个`create_calculator`方法，用于创建计算器对象。最后，我们使用工厂类创建计算器对象，并使用计算器对象进行计算。

## 4.2 详细解释说明

在这个例子中，我们使用了工厂模式来实现一个简单的计算器。工厂模式的核心思想是将对象的创建过程封装在一个工厂类中，这样我们可以通过调用工厂类的方法来创建不同类型的对象。
