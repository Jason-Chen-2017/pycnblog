# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的热传导问题来展示如何使用有限差分方法进行数值求解。

假设我们有一个长为1米的金属棒，其初始温度为0度，并且棒的两端被固定在100度。我们需要求解棒的温度分布在时间t=1秒时。

首先，我们需要设定问题：

- 问题类型：初值问题
- 偏微分方程：热传导方程
- 边界条件：棒的两端被固定在100度

接下来，我们选择有限差分方法进行求解。我们将棒划分为10个等分的小段，每个小段的长度为0.1米。我们还设定时间步长$\Delta t = 0.1$秒。

然后，我们使用有限差分方法来求解温度分布。具体操作步骤如下：

1. 初始化温度数组$u$，将所有元素设为0度。

2. 使用递推关系求解每个时间步的温度分布。

3. 迭代求解到所需的时间t=1秒。

最后，我们可以使用matplotlib库来可视化温度分布。

```python
import numpy as np
import matplotlib.pyplot as plt

# 设定问题
L = 1  # 棒长度
T = 1  # 时间t=1秒
N = 10  # 棒划分为10个等分的小段
dx = L / N
dt = 0.1  # 时间步长

# 初始化温度数组
u = np.zeros((N+1, N+1))

# 边界条件
u[:, 0] = 100
u[:, N] = 100

# 使用递推关系求解每个时间步的温度分布
for n in range(int(T / dt)):
    for i in range(1, N):
        for j in range(1, N):
            u[i, j] = u[i, j] + dt * alpha * (u[i+1, j] + u[i-1, j] + u[i, j+1] + u[i, j-1] - 4 * u[i, j])

# 可视化温度分布
plt.imshow(u, cmap='hot', extent=[0, L, 0, L])
plt.colorbar()
plt.xlabel('x')
plt.ylabel('y')
plt.title('Temperature Distribution')
plt.show()
```
