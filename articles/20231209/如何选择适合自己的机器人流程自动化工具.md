```python
import time
import random

class Robot:
    def __init__(self, task_list):
        self.task_list = task_list

    def execute_task(self, task):
        print(f"开始执行任务：{task}")
        time.sleep(random.uniform(1, 5))
        print(f"任务：{task} 执行完成")

    def execute_all_tasks(self):
        for task in self.task_list:
            self.execute_task(task)

if __name__ == "__main__":
    task_list = ["任务1", "任务2", "任务3"]
    robot = Robot(task_list)
    robot.execute_all_tasks()
```

这个代码实例定义了一个`Robot`类，它有一个`execute_task`方法用于执行单个任务，还有一个`execute_all_tasks`方法用于执行所有任务。在主程序中，我们创建了一个`Robot`对象，并调用其`execute_all_tasks`方法来执行所有任务。
