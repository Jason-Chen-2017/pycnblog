# 4.具体代码实例和详细解释说明

在这里，我们以物流路径规划为例，给出一个具体的代码实例和详细解释说明：

```python
import networkx as nx

# 创建图
G = nx.Graph()

# 添加节点
G.add_nodes_from([('A', {'weight': 0}), ('B', {'weight': 1}), ('C', {'weight': 2}), ('D', {'weight': 3}), ('E', {'weight': 4})])

# 添加边
G.add_edges_from([('A', 'B', {'weight': 1}), ('B', 'C', {'weight': 1}), ('C', 'D', {'weight': 1}), ('D', 'E', {'weight': 1})])

# 定义Dijkstra算法
def dijkstra(G, start, end):
    distances = {node: float('inf') for node in G.nodes()}
    distances[start] = 0
    visited = set()

    while visited != G.nodes():
        min_node = None
        for node in G.nodes():
            if node not in visited and (min_node is None or distances[node] < distances[min_node]):
                min_node = node
        if min_node is None:
            break
        visited.add(min_node)
        for neighbor in G.neighbors(min_node):
            new_distance = distances[min_node] + G[min_node][neighbor]['weight']
            if new_distance < distances[neighbor]:
                distances[neighbor] = new_distance

    return distances[end]

# 计算最短路径
shortest_path = dijkstra(G, 'A', 'E')
print(shortest_path)
```

在这个代码中，我们使用Python语言和NetworkX库来实现Dijkstra算法。首先，我们创建了一个图，并添加了节点和边。然后，我们定义了一个Dijkstra算法，该算法可以计算出从起始节点到终止节点的最短路径。最后，我们使用这个算法计算了从节点A到节点E的最短路径。
