# 2.核心概念与联系

## 2.1 随机拓扑

随机拓扑是一种网络结构，其中每个节点与其他节点之间的连接是随机的。这意味着，在随机拓扑中，任何两个节点之间的连接都有可能存在，而不是基于某种规则或约束。随机拓扑通常用于模拟实际网络的随机性，以及对网络性能和稳定性的研究。

## 2.2 马尔可夫链

马尔可夫链是一种有限状态机，其状态之间的转移遵循一定的概率分布。在马尔可夫链中，每个状态都有一个概率转移到下一个状态，而不依赖于之前的状态。这意味着，在马尔可夫链中，只需知道当前状态和转移概率，就可以预测下一个状态。马尔可夫链广泛应用于各种概率模型和随机过程的研究。

## 2.3 随机拓扑与马尔可夫链的联系

随机拓扑和马尔可夫链之间的联系主要体现在随机拓扑可以用来构建马尔可夫链的状态转移图。在这种情况下，每个节点可以被视为马尔可夫链的一个状态，而节点之间的连接可以被视为状态转移的概率。通过这种方式，我们可以利用随机拓扑的特性来生成和分析马尔可夫链。

1. 随机拓扑的连接概率公式：
$$
P(u,v) = \frac{k_u \cdot k_v}{\sum_{i=1}^{n} k_i^2}
$$
其中，$P(u,v)$ 表示节点 $u$ 与节点 $v$ 之间的连接概率，$k_u$ 和 $k_v$ 分别表示节点 $u$ 和节点 $v$ 的度，$n$ 表示网络中节点的数量。

$$

```python
import networkx as nx
import random

# 生成随机拓扑
def generate_random_topology(n, p):
    G = nx.erdos_renyi_graph(n, p)
    return G

# 构建马尔可夫链
def build_markov_chain(G):
    states = list(G.nodes)
    transition_probabilities = {}
    for u in states:
        for v in states:
            if u != v and G.has_edge(u, v):
                transition_probabilities[(u, v)] = 1.0 / G.degree(u)
    MC = nx.DiGraph()
    MC.add_nodes_from(states)
    for u, v, prob in G.edges(data=True):
        MC.add_edge(u, v, weight=prob)
    return MC

# 主函数
def main():
    n = 100
    p = 0.01
    G = generate_random_topology(n, p)
    MC = build_markov_chain(G)
    nx.draw(MC, with_labels=True)

if __name__ == '__main__':
    main()
```
