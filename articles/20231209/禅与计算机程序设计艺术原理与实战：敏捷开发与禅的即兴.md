2. 贪心算法：禅教的“简洁”思想与贪心算法的“贪心选择”思想相符。贪心算法通过在每个步骤中选择最优解，从而得到问题的最优解。

3. 回溯算法：禅教的“一心一意”思想与回溯算法的“深度优先搜索”思想相符。回溯算法通过从问题的根节点开始，逐步探索可能的解决方案，并在发现不可行解决方案时进行回溯，从而得到问题的所有可能解决方案。

## 4.1 动态规划实例
```python
def dp(n):
    dp = [0] * (n + 1)
    for i in range(1, n + 1):
        for j in range(i - 1, 0, -1):
            dp[j] = max(dp[j], dp[j - 1] + f(j, i))
    return dp[n]
```
在这个动态规划实例中，我们首先初始化一个dp数组，用于存储问题的最优解。然后，我们遍历问题的所有子问题，并根据动态规划算法的公式更新dp数组。最后，我们返回问题的最优解。

## 4.2 贪心算法实例

```python
def greedy(n):
    S = [f(i) for i in range(1, n + 1)]
    return min(S)
```
在这个贪心算法实例中，我们首先计算问题的所有可能解决方案，并将它们存储在S数组中。然后，我们使用贪心算法的公式找到问题的最优解。

## 4.3 回溯算法实例
```python
def backtracking(n):
    S = []
    for i in range(1, n + 1):
        S.append(f(i))
    return max(S)
```
在这个回溯算法实例中，我们首先计算问题的所有可能解决方案，并将它们存储在S数组中。然后，我们使用回溯算法的公式找到问题的最优解。
