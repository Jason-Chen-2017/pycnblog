以下是具体的代码实现：

```python
def sliding_window_min_max_sum(arr):
    n = len(arr)
    result = [0] * n
    min_sum = float('inf')
    max_sum = float('-inf')
    for i in range(n):
        min_sum = min(min_sum, arr[i])
        max_sum = max(max_sum, arr[i])
        for j in range(i, n):
            min_sum = min(min_sum, arr[j])
            max_sum = max(max_sum, arr[j])
            result[j] = (min_sum, max_sum)
    return result
```
