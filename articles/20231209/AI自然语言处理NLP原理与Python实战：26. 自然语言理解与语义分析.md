```python
# 准备数据集
data = ["I love you.", "You are my friend."]

# 预处理数据

vectorizer = TfidfVectorizer()

model = LogisticRegression()
model.fit(X_train, y_train)

preds = model.predict(X_test)

# 评估模型
print(preds)
```

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression

# 准备数据集
data = ["I love you.", "You are my friend."]

# 预处理数据
data = [d.lower() for d in data]

# 提取特征
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(data)

# 训练模型
X_train, X_test, y_train, y_test = train_test_split(X, data, test_size=0.2, random_state=42)
model = LogisticRegression()
model.fit(X_train, y_train)

# 测试模型
preds = model.predict(X_test)

# 评估模型
print(preds)
```

## 4.4 语义角色标注

### 4.4.1 算法原理

语义角色标注的算法原理包括规则引擎、统计模型和深度学习模型三个方面。

1. 规则引擎：使用规则来描述语义角色，以便识别文本中的实体和它们之间的关系。
2. 统计模型：使用概率来描述语义角色，以便识别文本中的实体和它们之间的关系。
3. 深度学习模型：使用神经网络来描述语义角色，以便识别文本中的实体和它们之间的关系。

### 4.4.2 具体操作步骤

### 4.4.3 数学模型公式

语义角色标注的数学模型公式包括：

1. 文法规则公式：$$ S \rightarrow NP + VP $$
2. 概率公式：$$ P(s) = \prod_{i=1}^n P(w_i | s) $$
3. 深度学习模型公式：$$ p(y | x) = \frac{1}{Z(x)} \exp(\sum_{i=1}^n \log p(y_i | x_i)) $$
