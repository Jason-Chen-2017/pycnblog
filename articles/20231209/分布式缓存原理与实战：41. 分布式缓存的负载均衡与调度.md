```python
import random

class CacheNode:
    def __init__(self, id, capacity):
        self.id = id
        self.capacity = capacity
        self.data = []

    def add_data(self, data):
        if len(self.data) < self.capacity:
            self.data.append(data)
        else:
            print(f"缓存节点 {self.id} 已满")

class CacheSystem:
    def __init__(self, nodes):
        self.nodes = nodes

    def add_data(self, data):
        for node in self.nodes:
            node.add_data(data)

    def get_data(self, id):
        for node in self.nodes:
            if id in node.data:
                return node.data[id]
        return None

nodes = [CacheNode(i, 10) for i in range(5)]
cache_system = CacheSystem(nodes)

data = {"key": "value"}
cache_system.add_data(data)

print(cache_system.get_data("key"))
```

1. 首先，我们定义了一个 `CacheNode` 类，它表示缓存节点，包括节点 ID、节点容量和缓存数据。
2. 然后，我们定义了一个 `CacheSystem` 类，它表示分布式缓存系统，包括缓存节点列表和添加数据和获取数据的方法。
3. 接下来，我们创建了五个缓存节点，并将它们添加到分布式缓存系统中。
4. 最后，我们添加了一个数据到分布式缓存系统中，并获取了该数据。
