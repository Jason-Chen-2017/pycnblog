# 4.具体代码实例和详细解释说明

在这个例子中，我们将使用动态规划算法来解决一个具有最优子结构的问题：最长公共子序列问题。

最长公共子序列问题的描述如下：给定两个字符串，找出它们的最长公共子序列。

我们可以使用动态规划算法来解决这个问题。具体的代码实现如下：

```python
def lcs(str1, str2):
    m = len(str1)
    n = len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(m):
        for j in range(n):
            if str1[i] == str2[j]:
                dp[i + 1][j + 1] = dp[i][j] + 1
            else:
                dp[i + 1][j + 1] = max(dp[i + 1][j], dp[i][j + 1])

    return dp[m][n]
```

这个函数的时间复杂度是O(m * n)，空间复杂度是O(m * n)。这种时间复杂度和空间复杂度可以通过动态规划算法来实现。
