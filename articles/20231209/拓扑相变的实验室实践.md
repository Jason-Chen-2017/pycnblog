## 1.5 拓扑相变的代码实例

以下是一个简单的拓扑相变的Python代码实例：

```python
import networkx as nx

# 创建一个简单的图
G = nx.Graph()
G.add_nodes_from([1, 2, 3])
G.add_edges_from([(1, 2), (2, 3)])

# 添加一个新的节点
G.add_node(4)

# 添加一个新的边
G.add_edge(1, 4)

# 删除一个边
G.remove_edge(1, 2)

# 修改一个节点的属性
G.nodes[1]['color'] = 'red'

# 获取图的拓扑状态
topology = nx.topological_sort(G)

# 打印拓扑状态
print(topology)
```

在这个代码实例中，我们使用了`networkx`库来创建和操作图。首先，我们创建了一个简单的图，然后添加了一个新的节点和边，删除了一个边，修改了一个节点的属性，并获取了图的拓扑状态。
