```python
import numpy as np
from scipy.spatial.distance import cosine

def cosine_similarity(a, b):
    return 1 - cosine(a, b)

def recommend(user_id, data, top_n=10):
    user_ratings = data[user_id]
    similarities = {}
    for item_id, rating in user_ratings.items():
        if rating != 0:
            for other_item_id, other_rating in data.items():
                if other_item_id != item_id and other_rating != 0:
                    similarity = cosine_similarity(user_ratings[item_id], data[other_item_id])
                    similarities[(item_id, other_item_id)] = similarity

    sorted_similarities = sorted(similarities.items(), key=lambda x: x[1], reverse=True)
    recommendations = [item for item, similarity in sorted_similarities[:top_n]]
    return recommendations
```

这个代码实例首先计算用户与项目之间的相似性，然后根据相似性推荐相关的项目。具体解释如下：

1. `cosine_similarity` 函数计算两个向量之间的余弦相似性。
2. `recommend` 函数接收用户 ID、数据和推荐数量作为参数。
3. 首先，获取用户的评分。
4. 然后，计算用户与项目之间的相似性。
5. 根据相似性，排序项目。
6. 最后，返回推荐的项目。
