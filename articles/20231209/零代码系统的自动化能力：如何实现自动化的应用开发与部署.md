# 2.核心概念与联系

在讨论零代码系统的自动化能力之前，我们需要了解一些核心概念。这些概念包括：

1. 自动化：自动化是指通过计算机程序来自动执行某些任务，而无需人工干预。自动化可以提高效率，减少错误，并降低成本。

2. 零代码系统：零代码系统是一种软件开发平台，它允许用户通过配置和组合预先定义的组件来创建软件应用程序，而无需编写代码。这种系统通常提供了图形用户界面，以便用户可以轻松地构建和部署应用程序。

3. 应用开发：应用开发是指通过编写代码来创建软件应用程序的过程。这可以包括设计、编码、测试和部署等各种活动。

4. 部署：部署是指将软件应用程序从开发环境移动到生产环境的过程。这可以包括设置服务器、配置网络和数据库等各种活动。
