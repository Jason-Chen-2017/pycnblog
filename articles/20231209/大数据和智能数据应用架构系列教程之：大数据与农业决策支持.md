在大数据与农业决策支持的应用架构中，数学模型公式详细讲解包括：

- 线性回归：线性回归是一种用于预测连续变量的统计模型，它的公式为：$$ y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon $$ 其中，$y$是预测变量，$x_1, x_2, ..., x_n$是解释变量，$\beta_0, \beta_1, ..., \beta_n$是模型参数，$\epsilon$是误差项。
- 逻辑回归：逻辑回归是一种用于预测二值变量的统计模型，它的公式为：$$ P(y=1|x_1, x_2, ..., x_n) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - ... - \beta_nx_n}} $$ 其中，$y$是预测变量，$x_1, x_2, ..., x_n$是解释变量，$\beta_0, \beta_1, ..., \beta_n$是模型参数，$e$是基数。
- 决策树：决策树是一种用于预测类别变量的机器学习模型，它的公式为：$$ D(x) = \begin{cases} c_1, & \text{if } x \in R_1 \\ c_2, & \text{if } x \in R_2 \\ ... & ... \\ c_n, & \text{if } x \in R_n \end{cases} $$ 其中，$D(x)$是决策树的预测结果，$c_1, c_2, ..., c_n$是类别变量，$R_1, R_2, ..., R_n$是决策树的决策规则。
- 支持向量机：支持向量机是一种用于解决线性可分二分类问题的机器学习模型，它的公式为：$$ f(x) = \text{sgn} \left( \sum_{i=1}^n \alpha_i y_i K(x_i, x) + b \right) $$ 其中，$f(x)$是支持向量机的预测结果，$K(x_i, x)$是核函数，$y_i$是训练数据的标签，$\alpha_i$是模型参数，$b$是偏置项。

  id INT,
  name STRING,
  age INT
);

- 数据分析：使用R实现数据分析。
```R
library(ggplot2)

data <- read.csv('data.csv')
ggplot(data, aes(x = age, y = name)) + geom_point()
```
- 数据挖掘：使用Apache Mahout实现数据挖掘。
```bash
mahout seqdirectory -i data.csv -o data
mahout seq2sparse -i data -o data -nv
mahout kmeans -i data -c 3 -o data -nv
```
- 数据可视化：使用D3.js实现数据可视化。
```javascript
<script src="https://d3js.org/d3.v5.min.js"></script>

const data = d3.csv('data.csv');
const svg = d3.select('svg');
const xScale = d3.scaleLinear().domain([0, d3.max(data, d => d.age)]).range([0, width]);
const yScale = d3.scaleLinear().domain([0, d3.max(data, d => d.name)]).range([height, 0]);
const line = d3.line()
  .x(d => xScale(d.age))
  .y(d => yScale(d.name));
svg.append('path')
  .datum(data)
  .attr('d', line)
  .attr('stroke', 'steelblue')
  .attr('stroke-width', 1.5)
  .attr('fill', 'none');
```
