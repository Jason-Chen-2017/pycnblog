# 2.核心概念与联系

## 2.1 太空探测器的类型

太空探测器可以分为以下几类：

1. 地球轨道探测器：这类探测器的轨道在地球周围，用于观测地球上的天体和地球自身的物理学现象。例如，地球轨道卫星（GEO）、太阳轨道卫星（HEO）等。

2. 月球探测器：这类探测器的轨道在月球周围，用于观测月球上的天体和月球自身的物理学现象。例如，月球卫星（Lunar Orbiter）、月球探测器（Lunar Prospector）等。

3. 行星探测器：这类探测器的轨道在行星周围，用于观测行星上的天体和行星自身的物理学现象。例如，火星探测器（Mars Exploration Rover）、木星探测器（Cassini-Huygens）等。

4. 外星系探测器：这类探测器的轨道在外星系中，用于观测外星系中的天体和外星系自身的物理学现象。例如，外星系探测器（Voyager）、外星系探测器（New Horizons）等。

## 2.2 太空探测器的主要组成部分

太空探测器的主要组成部分包括：

1. 载体：负责太空探测器在轨道上运行的能力，可以是火箭、火箭发动机、电磁推进器等。

2. 载具：负责太空探测器在地球轨道上运行的能力，可以是地球轨道卫星、太阳轨道卫星等。

3. 传感器：负责收集太空中的各种信息，如光学传感器、红外传感器、磁场传感器、微波传感器等。

4. 数据处理系统：负责收集到的信息的处理和分析，可以是数字信号处理系统、模拟信号处理系统等。

5. 通信系统：负责太空探测器与地球之间的数据传输，可以是无线电通信系统、光纤通信系统等。

6. 能源系统：负责太空探测器的运行和工作，可以是太阳能电池板、电池等。
