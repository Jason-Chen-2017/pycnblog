## 4.1 线性插值

线性插值是一种简单的插值方法，它使用两个已知的数据点来构建一个线性函数。线性插值的数学模型公式如下：

$$
f(x) = a_0 + a_1x
$$

其中，$a_0$ 和 $a_1$ 是插值函数的系数，它们可以通过以下公式计算：

$$
a_0 = f(x_0)
$$

$$
a_1 = \frac{f(x_1) - f(x_0)}{x_1 - x_0}
$$

其中，$x_0$ 和 $x_1$ 是已知的数据点，$f(x_0)$ 和 $f(x_1)$ 是这些数据点的值。

以下是一个Python代码实例，演示了线性插值的使用：

```python
import numpy as np

# 已知的数据点
x = np.array([1, 2, 3, 4, 5])
y = np.array([1, 4, 9, 16, 25])

# 线性插值
coefficients = np.polyfit(x, y, 1)

# 预测未知的数据点
x_new = np.array([0, 6])
y_new = np.polyval(coefficients, x_new)

print(y_new)
```

## 4.2 多项式插值

多项式插值是一种更复杂的插值方法，它使用多个已知的数据点来构建一个多项式函数。多项式插值的数学模型公式如下：

$$
f(x) = a_0 + a_1x + a_2x^2 + \cdots + a_nx^n
$$

其中，$a_0, a_1, a_2, \cdots, a_n$ 是插值函数的系数，它们可以通过以下公式计算：

$$
a_k = \frac{\sum_{i=0}^{m} w_i y_i x_i^k}{\sum_{i=0}^{m} w_i x_i^k}
$$

其中，$w_i$ 是数据点的权重，$y_i$ 是这些数据点的值，$m$ 是已知数据点的数量。

以下是一个Python代码实例，演示了多项式插值的使用：

```python
import numpy as np

# 预测未知的数据点
```
