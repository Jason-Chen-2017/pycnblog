## 4.1 代码实例

以下是一个使用Python和Lambda表达式进行跨平台开发的具体代码实例：

```python
import os
import platform

def get_platform():
    return platform.system()

def get_os():
    return os.name

def main():
    platform = get_platform()
    os_name = get_os()

    print(f"当前操作系统：{platform}")
    print(f"当前操作系统名称：{os_name}")

if __name__ == "__main__":
    main()
```

在这个代码实例中，我们使用了Python的`os`和`platform`模块来获取当前操作系统的信息。我们定义了一个`get_platform`函数，该函数使用`platform.system()`方法获取当前操作系统的名称，并返回这个名称。我们还定义了一个`get_os`函数，该函数使用`os.name`属性获取当前操作系统的名称，并返回这个名称。最后，我们在`main`函数中调用了这两个函数，并使用`print`函数输出当前操作系统的名称。

在这个代码实例中，我们使用了Python的`os`和`platform`模块来获取当前操作系统的信息。我们首先导入了这两个模块，然后定义了一个`get_platform`函数，该函数使用`platform.system()`方法获取当前操作系统的名称，并返回这个名称。我们还定义了一个`get_os`函数，该函数使用`os.name`属性获取当前操作系统的名称，并返回这个名称。最后，我们在`main`函数中调用了这两个函数，并使用`print`函数输出当前操作系统的名称。
