## 3.1 时间复杂度

时间复杂度是指算法在最坏情况下所需要的时间复杂度。时间复杂度是用来评估算法性能的一个重要指标。时间复杂度可以用大O符号表示，如O(n)、O(n^2)、O(2^n)等。

## 3.2 空间复杂度

空间复杂度是指算法在最坏情况下所需要的空间复杂度。空间复杂度是用来评估算法性能的一个重要指标。空间复杂度可以用大O符号表示，如O(n)、O(n^2)、O(2^n)等。

## 3.3 动态规划

动态规划是一种解决最优化问题的算法方法。动态规划算法通过分步递推来得到最优解。动态规划算法可以用来解决许多复杂的最优化问题，如最长子序列、最短路径等。

## 3.4 贪心算法

贪心算法是一种解决最优化问题的算法方法。贪心算法通过在每个步骤中选择最优解来得到最终解。贪心算法可以用来解决许多简单的最优化问题，如零一包裹问题、零一背包问题等。

## 4.1 动态规划实例

### 4.1.1 最长子序列

最长子序列问题是指在一个给定的序列中，找出一个最长的子序列，使得子序列中的任意两个元素之间都不相邻。

```python
def longest_subsequence(arr):
    n = len(arr)
    dp = [1] * n

    for i in range(1, n):
        for j in range(i):
            if arr[i] > arr[j] and dp[i] < dp[j] + 1:
                dp[i] = dp[j] + 1

    return max(dp)
```

### 4.1.2 最短路径

最短路径问题是指在一个给定的图中，从一个顶点到另一个顶点的最短路径。

```python
def shortest_path(graph, start, end):
    n = len(graph)
    dist = [float('inf')] * n
    dist[start] = 0

    for _ in range(n):
        for u, v, w in graph:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w

    return dist[end]
```

## 4.2 贪心算法实例

### 4.2.1 零一包裹问题

零一包裹问题是指在一个给定的物品集合中，选择一些物品放入一个容量为1的背包，使得背包中物品的价值最大。

```python
def knapsack(items, capacity):
    n = len(items)
    dp = [0] * (capacity + 1)

    for i in range(n):
        for w in range(capacity, -1, -1):
            if items[i][0] <= w:
                dp[w] = max(dp[w], dp[w - items[i][0]] + items[i][1])

    return dp[capacity]
```
