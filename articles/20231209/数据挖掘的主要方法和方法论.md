# 2. 核心概念与联系

在数据挖掘中，我们需要了解一些核心概念，包括数据、特征、标签、训练集、测试集、模型等。这些概念之间存在着密切的联系，我们需要理解这些概念的含义和联系，才能更好地进行数据挖掘。

1. 数据：数据是数据挖掘的基础，数据是由一组数据点组成的集合。数据点是由一组特征组成的向量，每个特征表示数据点的一个属性。

2. 特征：特征是数据点的属性，用于描述数据点的某个方面。特征可以是数值型的，如年龄、体重等，也可以是类别型的，如性别、职业等。

3. 标签：标签是数据点的类别或分类，用于表示数据点的某个属性。标签可以是数值型的，如分数、评分等，也可以是类别型的，如正常、异常等。

4. 训练集：训练集是用于训练模型的数据集，包含一组数据点和它们的标签。训练集用于训练模型，使模型能够在测试集上得到较好的性能。

5. 测试集：测试集是用于评估模型性能的数据集，包含一组数据点和它们的标签。测试集用于评估模型在未知数据上的性能，以便我们可以确定模型是否过拟合或欠拟合。

6. 模型：模型是数据挖掘的核心，用于描述数据的模式和规律。模型可以是数学模型，如线性回归、逻辑回归等，也可以是机器学习模型，如决策树、支持向量机等。

这些核心概念之间存在着密切的联系，我们需要理解这些概念的含义和联系，才能更好地进行数据挖掘。

5. 回归分析：回归分析是一种监督学习方法，用于预测数据点的值。回归分析算法包括：线性回归、逻辑回归、支持向量机等。

6. 决策树：决策树是一种监督学习方法，用于构建决策树模型。决策树算法包括：ID3、C4.5、CART等。
