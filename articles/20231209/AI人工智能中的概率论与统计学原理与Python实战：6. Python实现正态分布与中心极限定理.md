## 1. 核心概念与联系

### 1.1 正态分布

正态分布是一种概率分布，其概率密度函数是对称的、单峰的、全面的、可积的、可微的。正态分布的概率密度函数为：

$$
f(x;\mu,\sigma) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，μ是均值，σ是标准差。正态分布的特点包括：

- 随机变量的概率密度函数是对称的，即在均值μ处的概率最大。
- 随机变量的概率密度函数是单峰的，即只有一个极值点。
- 随机变量的概率密度函数是全面的，即在均值μ处的概率为0。
- 随机变量的概率密度函数是可积的，即积分在区间[-∞,+∞]内为1。
- 随机变量的概率密度函数是可微的，即对于任意x，f(x)的导数存在。

### 1.2 中心极限定理

中心极限定理是概率论和数学统计中的一个重要定理，它描述了随机变量在大样本中的分布近似为正态分布。中心极限定理的主要结论是：

对于任意固定的ε>0，当n→∞时，P(|X-μ|≥ε)≈2(1-Φ(ε/√(nσ^2)))

其中，X是随机变量，μ是均值，σ是标准差，Φ是标准正态分布函数。

中心极限定理的应用主要包括：

- 假设检验：通过计算P值来判断假设是否被拒绝。
- 置信区间估计：通过计算置信区间来估计参数的取值范围。
- 估计误差：通过计算估计误差来评估估计值的准确性。

### 3.1 正态分布的概率密度函数

```python
import numpy as np

def normal_distribution(x, mu, sigma):
    return (1 / (np.sqrt(2 * np.pi * sigma**2))) * np.exp(-(x - mu)**2 / (2 * sigma**2))

mu = 0
sigma = 1

```

def central_limit_theorem(x, mu, sigma, n):
