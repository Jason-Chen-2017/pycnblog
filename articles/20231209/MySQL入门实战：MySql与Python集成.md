## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释MySQL与Python的集成。

代码实例：

```python
# 导入MySQL驱动程序
import mysql.connector

# 设置连接参数
config = {
    "host": "localhost",
    "user": "root",
    "password": "password",
    "database": "test"
}

# 建立连接
cnx = mysql.connector.connect(**config)

# 构建SQL语句
sql = "SELECT * FROM users"

# 执行查询
cursor = cnx.cursor()
cursor.execute(sql)

# 处理结果
results = cursor.fetchall()
for row in results:
    print(row)

# 断开连接
cnx.close()
```

在上述代码中，我们首先导入了MySQL驱动程序，然后设置了连接参数。接着，我们建立了数据库连接，并构建了一个SQL语句，以便查询MySQL数据库中的数据。接着，我们执行了查询操作，并处理了查询结果。最后，我们断开了数据库连接。
