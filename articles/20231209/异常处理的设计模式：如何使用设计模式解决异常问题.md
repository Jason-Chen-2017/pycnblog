# 4.具体代码实例和详细解释说明
以下是一个具体的代码实例，用于说明异常处理的设计模式的使用方法：

```python
class MyException(Exception):
    pass

def calculate_square(x):
    try:
        if x < 0:
            raise MyException("Invalid input: x must be non-negative")
        return x * x
    except MyException as e:
        print(e)
        return None

result = calculate_square(-1)
print(result)
```

在这个代码实例中，我们首先定义了一个异常类MyException，继承自Exception类。然后，我们定义了一个函数calculate_square，用于计算x的平方。在函数中，我们使用try-except语句来捕获可能发生的异常。如果x小于0，我们将抛出MyException异常。如果捕获到异常，我们将打印异常信息，并返回None。最后，我们调用calculate_square函数，传入-1作为参数，并打印返回值。
