# 2.核心概念与联系

在深度学习中，概率论和统计学是两个密切相关的领域，它们的核心概念如下：

1.概率：概率是一个事件发生的可能性，通常用数字0-1表示，表示事件发生的可能性。

2.随机变量：随机变量是一个可能取多个值的变量，每个值都有一个相应的概率。

3.条件概率：条件概率是给定某个事件发生的情况下，另一个事件发生的概率。

4.期望：期望是随机变量的数学期望，表示随机变量的平均值。

5.方差：方差是随机变量的一种度量，表示随机变量的离散程度。

6.协方差：协方差是两个随机变量之间的一种度量，表示它们之间的相关性。

7.信息论：信息论是一种用于度量信息的理论框架，用于衡量随机变量之间的相关性和独立性。

这些概念在深度学习中具有重要意义，它们为深度学习提供了理论基础和方法论。

9.K-最近邻：K-最近邻是一种用于处理分类和回归问题的算法，用于找到最近的K个邻居。
