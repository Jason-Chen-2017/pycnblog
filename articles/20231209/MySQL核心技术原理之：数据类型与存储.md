# 2.核心概念与联系
在MySQL中，数据类型与存储是密切相关的。数据类型决定了数据在存储过程中的格式和长度，而存储则决定了数据在磁盘上的存储方式。我们需要了解这些概念的联系，以便更好地应对实际问题。

数据类型主要包括：整数类型、浮点类型、字符串类型、日期时间类型等。整数类型可以进一步分为有符号整数和无符号整数，浮点类型可以进一步分为单精度浮点数和双精度浮点数。字符串类型可以进一步分为定长字符串和可变长字符串。日期时间类型可以进一步分为日期、时间、时间戳等。

存储主要包括：磁盘存储和内存存储。磁盘存储是数据在磁盘上的存储方式，内存存储是数据在内存中的存储方式。磁盘存储主要包括：顺序存储和链式存储。顺序存储是数据在磁盘上按照顺序存储，链式存储是数据在磁盘上按照链表存储。内存存储主要包括：堆内存和堆外存储。堆内存是数据在内存中的存储方式，堆外存储是数据在内存中的存储方式。
