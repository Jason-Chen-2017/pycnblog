## 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的Python代码实例，用于计算城市的绿化率。绿化率是一种衡量城市绿化程度的指标，它是指城市绿地面积与总面积的比值。

```python
import math

def calculate_green_rate(green_area, total_area):
    if total_area == 0:
        return 0
    else:
        return green_area / total_area * 100

green_area = 100000  # 绿地面积，单位：平方米
total_area = 1000000  # 总面积，单位：平方米

green_rate = calculate_green_rate(green_area, total_area)
print("绿化率：%.2f%%" % green_rate)
```

在这个代码实例中，我们首先定义了一个名为 `calculate_green_rate` 的函数，它接受两个参数：绿地面积和总面积。然后，我们计算绿化率，并将其打印出来。
