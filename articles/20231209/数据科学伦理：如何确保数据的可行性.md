## 4.2 数据验证

以下是数据验证的具体代码实例：

```python
import re

# 检查数据是否满足特定的格式
def check_format(data):
    pattern = r'^\d{4}-\d{2}-\d{2}$'
    return all(re.match(pattern, data))

# 验证数据是否满足特定的规则
def validate_data(data):
    pattern = r'^[A-Za-z0-9]+$'
    return all(re.match(pattern, data))

# 使用数据验证函数
data['email'] = ['example@example.com', 'example123@example.com']
print(check_format(data['email']))  # True
print(validate_data(data['email']))  # False
```

在上述代码中，我们首先定义了两个数据验证函数，分别用于检查数据是否满足特定的格式和规则，然后使用这些函数来验证数据是否满足特定的要求。
