# 2.核心概念与联系

## 2.1 函数的概念

函数是编程中的一个基本概念，它可以将一段可重复使用的代码封装起来，以便在需要时调用。函数的主要特点是：

1. 可重用性：函数可以被多次调用，提高代码的可重用性。
2. 模块化：函数可以将复杂的任务拆分成多个小任务，提高代码的可读性和可维护性。
3. 参数传递：函数可以接收参数，使其更具灵活性。

## 2.2 函数的定义与调用

在Python中，我们可以使用`def`关键字来定义函数，并使用`()`来调用函数。函数的定义格式如下：

```python
def 函数名(参数列表):
    函数体
```

其中，`函数名`是函数的名称，`参数列表`是函数接收的参数，`函数体`是函数的具体实现。

我们可以通过以下步骤来调用函数：

1. 调用函数：使用`函数名()`来调用函数。
2. 传递参数：在调用函数时，可以传递参数给函数。
3. 返回值：函数可以返回一个值，我们可以通过`return`关键字来实现。

# 4.具体代码实例和详细解释说明

## 4.1 代码实例

以下是一个简单的Python函数的实例：

```python
def greet(name):
    print(f"Hello, {name}!")

greet("John")
```

在这个例子中，我们定义了一个名为`greet`的函数，它接收一个名为`name`的参数。当我们调用`greet("John")`时，函数会打印出`Hello, John!`。

## 4.2 详细解释说明

1. 定义函数：我们使用`def`关键字来定义一个名为`greet`的函数，并指定其接收一个名为`name`的参数。
2. 编写函数体：我们使用`print()`函数来打印出一个带有`name`参数的消息。
3. 调用函数：我们调用`greet("John")`来调用`greet`函数，并传递一个字符串参数`"John"`。
4. 返回值：`greet`函数没有返回值，但是它会打印出一个消息。
