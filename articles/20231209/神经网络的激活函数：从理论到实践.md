# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的代码实例来演示如何使用激活函数。

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

x = np.array([1.0, 2.0, 3.0])
y = sigmoid(x)
print(y)
```

在上述代码中，我们首先导入了 numpy 库，然后定义了一个 sigmoid 函数，该函数实现了激活函数的计算。接着，我们定义了一个输入数组 x，并将其传递给 sigmoid 函数进行计算。最后，我们将计算结果打印出来。
