Redis是一个开源的高性能键值存储系统，它支持数据的持久化，可基于内存（Redis）或磁盘（Redis-Persistent）。Redis 是一个开源的使用 ANSI C 语言编写、遵循 BSD 协议的高性能键值对服务器。Redis 可以用作数据库、缓存和消息队列。Redis 提供多种语言的 API。

Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。不同的是 Redis 的持久化可以选择性地进行，也就是说你可以选择性地保存数据。
