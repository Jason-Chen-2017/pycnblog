数据清洗的具体代码实例如下：

```python
import pandas as pd

# 读取数据源
data = pd.read_csv('data.csv')

# 数据去重
data = data.drop_duplicates()

# 数据缺失值处理
data = data.fillna(data.mean())

# 数据类型转换
data['age'] = data['age'].astype(int)

# 数据清洗完成
data.to_csv('data_cleaned.csv', index=False)
```

数据聚合的具体代码实例如下：

```python
import pandas as pd

# 读取数据源
data = pd.read_csv('data.csv')

# 数据汇总
data_aggregated = data.groupby('age').mean()

# 数据统计
data_aggregated = data_aggregated.agg({'age': 'sum', 'name': 'count'})

# 数据聚类
data_aggregated = data_aggregated.groupby('age').apply(lambda x: x.idxmin())

# 数据聚合完成
data_aggregated.to_csv('data_aggregated.csv', index=False)
```
