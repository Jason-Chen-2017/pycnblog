## 4.4 智能旅游代码实例

智能旅游代码实例包括以下几点：

- 使用Python的requests库来获取旅游景点的信息。
- 使用Python的BeautifulSoup库来解析HTML内容。
- 使用Python的pandas库来处理数据。

具体代码实例如下：

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd

# 获取旅游景点的信息
url = 'https://www.tripadvisor.com/Attractions-g60763-Activities-c1'
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

# 解析HTML内容
attractions = []
for attraction in soup.find_all('div', class_='result-info'):
    name = attraction.find('h2', class_='result-name').text
    rating = float(attraction.find('span', class_='ui-bubble-rating').text.split(' out of')[0])
    attractions.append((name, rating))

# 处理数据
df = pd.DataFrame(attractions, columns=['Name', 'Rating'])
df.set_index('Name', inplace=True)

# 保存数据
df.to_csv('attractions.csv')
```
