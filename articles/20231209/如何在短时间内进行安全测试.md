## 4.具体代码实例和详细解释说明

具体代码实例：

```python
import requests
from requests.exceptions import RequestException

def send_request(url, method, data=None):
    try:
        response = requests.request(method, url, data=data)
        response.raise_for_status()
        return response.text
    except RequestException as e:
        print(e)
        return None

def main():
    url = "http://example.com"
    method = "GET"
    data = {
        "username": "admin",
        "password": "password"
    }
    result = send_request(url, method, data)
    if result:
        print("请求成功")
        print(result)
    else:
        print("请求失败")

if __name__ == "__main__":
    main()
```

详细解释说明：

上述代码实例是一个简单的网络请求示例，用于发送 GET 请求到指定的 URL，并获取响应结果。代码首先导入了 `requests` 库，然后定义了一个 `send_request` 函数，用于发送请求。`send_request` 函数接受 URL、请求方法和请求数据作为参数，并尝试发送请求。如果请求成功，函数返回响应结果；如果请求失败，函数返回 `None`。

在主函数中，我们定义了一个 URL、请求方法和请求数据，并调用 `send_request` 函数发送请求。如果请求成功，我们打印响应结果；如果请求失败，我们打印错误信息。
