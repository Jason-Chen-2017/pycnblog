## 4.2 优化生产计划

我们可以使用Python的pulp库来实现线性规划模型。首先，我们需要定义目标函数和约束条件。然后，我们可以使用pulp库来解决这个线性规划问题。

```python
from pulp import LpProblem, LpMinimize, LpMaximize, LpVariable

# 定义目标函数和约束条件
def objective_function():
    return lpSum([cost * x for x in xs])

def constraints():
    return [
        lpSum([x for x in xs]) <= resource_capacity,
        lpSum([x for x in xs if x >= demand]) == demand,
        xs >= 0
    ]

# 创建线性规划问题
problem = LpProblem("Production Planning", LpMinimize)

# 添加目标函数和约束条件
problem += (objective_function(), constraints())

# 解决线性规划问题
problem.solve()

# 获取最优解
xs = [value(x) for x in problem.variables()]
print("Optimal production plan:", xs)
```
