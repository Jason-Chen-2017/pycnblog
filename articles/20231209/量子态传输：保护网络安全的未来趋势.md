# 2.核心概念与联系

## 2.1 量子态

量子态是量子计算中的基本概念，它可以表示为一种概率分布。量子态可以用向量表示，通常用ket符号表示。例如，一个二级量子态可以表示为|ψ⟩=α|0⟩+β|1⟩，其中α和β是复数，|0⟩和|1⟩是基态。

## 2.2 量子通信

量子通信是利用量子物理原理实现信息传输的一种方式。量子通信的核心概念是量子比特（qubit），它是量子计算中的基本单元。量子比特可以表示为一个二维向量，通常用ket符号表示。例如，一个量子比特可以表示为|ψ⟩=α|0⟩+β|1⟩。

## 2.3 量子态传输

量子态传输是一种量子通信技术，它可以实现在量子通信网络中高效地传输量子态。量子态传输的核心算法原理是利用量子态的纠缠性和量子态的传输特性，实现在量子通信网络中高效地传输量子态。
