### 4.1 监督学习

#### 4.1.1 线性回归

线性回归是一种用于回归问题的监督学习方法，它通过训练模型来预测输入数据的输出值。线性回归的数学模型公式为：

$$
y = w^Tx + b
$$

线性回归的具体代码实现如下：

```python
import numpy as np

def linear_regression(X, y):
    X_bias = np.c_[np.ones((X.shape[0], 1)), X]
    theta = np.linalg.inv(X_bias.T.dot(X_bias)).dot(X_bias.T).dot(y)
    return theta
```
