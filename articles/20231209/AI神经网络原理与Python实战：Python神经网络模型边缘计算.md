# 2 核心概念与联系

在这一部分，我们将讨论AI神经网络的核心概念，以及与其他相关概念之间的联系。

## 2.1 神经元

神经元是人工智能领域的基本构建块，它模仿了人类大脑中神经元的工作方式。每个神经元都包含输入、输出和权重。输入是从其他神经元接收的信号，输出是神经元根据输入和权重计算出的值。权重是用于调整输入和输出之间关系的参数。

## 2.2 神经网络

神经网络是由多个相互连接的神经元组成的网络。神经网络可以学习从输入到输出的映射关系，这种学习过程通常是通过优化权重来实现的。神经网络可以应用于各种任务，如图像识别、语音识别和自然语言处理等。

## 2.3 激活函数

激活函数是神经网络中的一个关键组件，它用于将神经元的输入转换为输出。常见的激活函数有 sigmoid、tanh 和 ReLU 等。激活函数的作用是为了引入非线性，使得神经网络能够学习更复杂的模式。

## 2.4 损失函数

损失函数是用于衡量神经网络预测值与实际值之间差距的函数。损失函数的目标是最小化，以便优化神经网络的权重。常见的损失函数有均方误差、交叉熵损失等。

## 2.5 反向传播

反向传播是训练神经网络的一个重要技术，它通过计算损失函数的梯度来优化权重。反向传播首先计算输出层的损失，然后逐层向前传播这个损失，最后逐层向后传播梯度。这种方法使得优化过程更加高效。

## 2.6 边缘计算

边缘计算是一种计算模式，它将计算能力推向边缘设备，而不是将所有数据发送到中央服务器进行处理。这有助于减少数据传输开销，提高数据处理速度，并提高数据安全性。
