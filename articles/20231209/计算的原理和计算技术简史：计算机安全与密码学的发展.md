# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 RSA密码学算法

RSA密码学算法是一种非对称密码学算法，它的核心思想是利用大素数的特性，通过选择两个大素数p和q，计算出n=pq，然后选择一个公共指数e（e为素数），计算出公钥和私钥。

### 3.1.1 RSA密码学算法的具体操作步骤

1. 选择两个大素数p和q，使得p和q互质，并计算n=pq。
2. 选择一个公共指数e，使得gcd(e, (p-1)(q-1)) = 1，并计算出私钥指数d，使得ed ≡ 1 (mod (p-1)(q-1))。
3. 使用公钥（n, e）进行加密，使用私钥（n, d）进行解密。

### 3.1.2 RSA密码学算法的数学模型公式

1. 加密公式：C ≡ M^e (mod n)
2. 解密公式：M ≡ C^d (mod n)

## 3.2 AES密码学算法

AES密码学算法是一种对称密码学算法，它的核心思想是利用替换、移位、选择反应和排列等操作来实现加密和解密过程。

### 3.2.1 AES密码学算法的具体操作步骤

1. 选择一个密钥key，并将其扩展为4个128位的子密钥。
2. 对输入的明文数据块进行分组，将其分为16个32位的字。
3. 对每个字进行替换、移位、选择反应和排列等操作，并使用子密钥进行加密。
4. 将加密后的字重新组合成加密后的数据块。

### 3.2.2 AES密码学算法的数学模型公式

1. 替换：对每个字进行替换操作，将其转换为另一个等价的字。
2. 移位：对每个字进行移位操作，将其位置进行调整。
3. 选择反应：对每个字进行选择反应操作，将其中的某些位置进行反转。
4. 排列：对每个字进行排列操作，将其位置进行重新排列。

```python
def rsa_key_pair_generator(n):

    q = random.randint(n, 2 * n)

        p = random.randint(n, 2 * n)
        q = random.randint(n, 2 * n)
    phi = (p - 1) * (q - 1)
    d = pow(e, -1, phi)
```

```python
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad

def aes_key_expander(key):
    key_schedule = [key]
    for i in range(39):
        key_schedule.append(key_schedule[i] << 1)
        if i % 2 == 1:
            key_schedule[i + 1] ^= 0x52
        if i % 4 == 3:
            key_schedule[i + 1] ^= 0x36
    return key_schedule

def aes_encrypt(plaintext, key):
    block_size = 16
    padding_length = block_size - (len(plaintext) % block_size)
    plaintext += (chr(padding_length) * padding_length)
    cipher = AES.new(key, AES.MODE_ECB)
    ciphertext = cipher.encrypt(pad(plaintext, block_size))
    return ciphertext

def aes_decrypt(ciphertext, key):
    block_size = 16
    cipher = AES.new(key, AES.MODE_ECB)
    plaintext = unpad(cipher.decrypt(ciphertext), block_size)
    return plaintext
```
