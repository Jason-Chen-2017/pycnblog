# 2.核心概念与联系

## 2.1 Spring Boot

Spring Boot 是一个用于构建原生 Spring 应用程序的框架。它提供了一种简化的配置，使得开发人员可以专注于编写代码，而不是为应用程序设置和配置。Spring Boot 提供了许多预先配置的 Spring 依赖项，这使得开发人员可以更快地开始编写代码，而不是花时间配置依赖项。

Spring Boot 还提供了许多有用的功能，如自动配置、嵌入式服务器、数据源抽象、安全性、元数据、外部化配置、监控和管理等。这些功能使得开发人员可以更快地开发和部署应用程序，而不是花时间设置和配置各种组件。

## 2.2 Netty

Netty 是一个高性能的网络应用框架，它提供了一种简单的方法来构建可扩展、高性能的网络应用程序。Netty 使用非阻塞 I/O 模型，这意味着它可以处理大量并发连接，而不会导致性能下降。Netty 还提供了许多有用的功能，如数据包解码、编码、连接管理、事件驱动等。

Netty 的核心组件包括：

- Channel：表示网络连接，可以是 TCP 连接或 UDP 连接。
- EventLoop：负责处理 I/O 事件，如接收数据、发送数据等。
- Pipeline：是一个 Channel 的处理器链，可以包含多个处理器，如解码器、编码器、处理器等。
- ChannelHandler：是一个处理器的抽象类，可以实现各种功能，如解码、编码、读取、写入等。
