```python

# 创建表
query = Query("CREATE_INDEX", {
    "name": "users",
    "source": "users"
})
result = client.execute(query)

# 插入数据
query = Query("CREATE", {
    "collection": "users",
    "data": {
        "name": "John Doe",
        "age": 30
    }
})

# 查询数据
query = Query("GET", {
    "collection": "users",
    "ref": result["generated_ref"]
})

query = Query("UPDATE", {
    "collection": "users",
    "ref": result["generated_ref"],
    "data": {
        "age": 31
    }
})
result = client.execute(query)
```

在这个代码实例中，我们首先创建了一个名为 "users" 的表。然后，我们插入了一条数据，包括一个名字和一个年龄。接下来，我们查询了这条数据，并更新了年龄为 31。
