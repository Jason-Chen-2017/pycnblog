在这个例子中，我们将实现一个简单的背包问题，其中有n个物品，每个物品都有一个重量和一个价值。我们的目标是找到一个物品组合，使得总重量不超过一个给定的重量限制，并且总价值最大。

我们可以通过动态规划算法来解决这个问题。首先，我们需要创建一个dp表，用于存储每个状态的最优解。然后，我们需要遍历所有的物品，并计算每个状态的最优解。最后，我们需要返回dp表中的最优解。

以下是这个问题的具体代码实例：

```python
def knapsack(weights, values, capacity):
    n = len(weights)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if weights[i - 1] <= w:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - weights[i - 1]] + values[i - 1])
            else:
                dp[i][w] = dp[i - 1][w]

    return dp[n][capacity]

weights = [2, 3, 4, 5]
values = [4, 5, 6, 7]
capacity = 7

print(knapsack(weights, values, capacity))  # 输出: 16
```
