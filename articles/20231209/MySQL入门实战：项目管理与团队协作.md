# 2.核心概念与联系

在MySQL中，有几个核心概念需要理解：

- 数据库：MySQL中的数据库是一个逻辑的容器，用于存储和管理数据。
- 表：数据库中的表是一个实体，用于存储具有相同结构的数据。
- 列：表中的列用于存储数据的属性。
- 行：表中的行用于存储具有相同属性的数据记录。
- 索引：索引是用于加速数据查询的数据结构。
- 约束：约束用于确保数据的完整性和一致性。

这些概念之间的联系如下：

- 数据库包含多个表，表包含多个列和行。
