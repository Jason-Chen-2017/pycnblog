```python
import numpy as np
from scipy.stats import norm

# 模拟数据
np.random.seed(42)
x = np.linspace(-10, 10, 1000)
y = norm.pdf(x, loc=0, scale=1)

# 最大似然估计
def mle(x, y):
    return np.mean(x), np.mean(y)

# 使用最大似然估计对数据进行拟合
mean_x, mean_y = mle(x, y)
fitted_y = norm.pdf(x, loc=mean_x, scale=mean_y)

# 绘制数据和拟合结果
import matplotlib.pyplot as plt
plt.plot(x, y, label='data')
plt.plot(x, fitted_y, label='fit')
plt.legend()
plt.show()
```

在这个代码实例中，我们使用Python和NumPy库进行数据处理和拟合。我们首先使用`numpy.random.seed()`函数设置随机数生成器的种子，以便在不同运行中得到相同的结果。然后，我们使用`numpy.linspace()`函数生成一个从-10到10的均匀分布的1000个点，并使用`scipy.stats.norm.pdf()`函数计算在这些点处的正态分布的密度。

接下来，我们使用`mle()`函数进行最大似然估计，以估计正态分布的参数（均值和标准差）。然后，我们使用`scipy.stats.norm.pdf()`函数计算拟合后的密度。

最后，我们使用`matplotlib.pyplot`库绘制数据和拟合结果。
