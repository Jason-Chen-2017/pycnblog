## 2.核心概念与联系

在虚拟现实与增强现实的实现过程中，我们需要了解一些核心概念和联系。这些概念包括：

2. 图形处理单元（GPU）：用于处理3D模型的渲染。GPU可以高效地处理大量的图形计算，从而实现虚拟环境的快速渲染。
