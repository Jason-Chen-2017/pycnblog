1. 设计：我们可以使用面向对象的设计方法来设计这个程序。我们可以定义一个计算器类，并定义四个运算方法：add、subtract、multiply 和 divide。

2. 编写：我们可以使用简洁、灵活和可读性的代码风格来编写这个程序。我们可以使用if-else语句来实现不同运算的逻辑，并使用变量和函数来实现代码的可读性。

3. 测试：我们可以使用单元测试来测试这个程序的正确性和可靠性。我们可以编写一些测试用例，以确保每个运算方法都能正确地进行计算。

4. 维护：我们可以使用代码审查和代码重构来维护这个程序。我们可以检查代码的简洁性、灵活性和可读性，并对代码进行优化和修改。

以下是这个计算器程序的具体代码实例：

```python
class Calculator:
    def __init__(self):
        pass

    def add(self, a, b):
        return a + b

    def subtract(self, a, b):
        return a - b

    def multiply(self, a, b):
        return a * b

    def divide(self, a, b):
        return a / b

if __name__ == '__main__':
    calculator = Calculator()
    a = 10
    b = 2
    print(calculator.add(a, b))
    print(calculator.subtract(a, b))
    print(calculator.multiply(a, b))
    print(calculator.divide(a, b))
```

在这个代码实例中，我们定义了一个Calculator类，并实现了四个运算方法：add、subtract、multiply 和 divide。我们使用简洁、灵活和可读性的代码风格来编写这个程序，并使用单元测试来测试这个程序的正确性和可靠性。
