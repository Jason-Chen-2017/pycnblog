以下是一个简单的智能合约的具体代码实例：

```solidity
pragma solidity ^0.5.16;

contract MedicalData {
    // 存储医疗数据
    mapping(address => mapping(uint => uint)) public medicalData;

    // 添加医疗数据
    function addMedicalData(address patient, uint age, uint height) public {
        medicalData[patient][age] = height;
    }

    // 查询医疗数据
    function queryMedicalData(address patient, uint age) public view returns (uint) {
        return medicalData[patient][age];
    }
}
```

在上述代码中，我们定义了一个智能合约`MedicalData`，用于存储和查询医疗数据。`addMedicalData`函数用于添加医疗数据，`queryMedicalData`函数用于查询医疗数据。

## 4.2 数据验证技术的具体代码实例

以下是一个简单的数据验证技术的具体代码实例：

```python
import hashlib

def verify_data(data, hash_value):
    # 计算数据的哈希值
    hash_object = hashlib.sha256(data.encode())
    calculated_hash = hash_object.hexdigest()

    # 比较计算的哈希值与输入的哈希值是否相等
    if calculated_hash == hash_value:
        return True
    else:
        return False
```

在上述代码中，我们定义了一个`verify_data`函数，用于验证数据的一致性。`verify_data`函数接收一个数据和一个哈希值作为输入，并计算数据的哈希值。然后，我们比较计算的哈希值与输入的哈希值是否相等。如果相等，则返回`True`，表示数据一致；否则，返回`False`，表示数据不一致。
