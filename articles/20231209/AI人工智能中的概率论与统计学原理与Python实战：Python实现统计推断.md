## 3.1 条件概率
条件概率是一种在给定某个事件发生的情况下，另一个事件发生的概率。我们可以用公式P(A|B)表示条件概率，其中P(A|B)表示在事件B发生的情况下，事件A的概率。

## 3.2 贝叶斯定理
贝叶斯定理是概率论中的一个重要定理，它可以帮助我们计算条件概率。贝叶斯定理的公式为：

P(A|B) = P(B|A) * P(A) / P(B)

其中，P(A|B)是我们要计算的条件概率，P(B|A)是事件B发生时事件A的概率，P(A)是事件A的概率，P(B)是事件B的概率。

## 3.3 最大似然估计
最大似然估计是一种用于估计参数的方法，它基于数据的似然性。我们可以用公式L(θ)表示似然性，其中θ是参数，L(θ)是数据集中所有可能的观测值的概率。我们的目标是找到使似然性最大的θ值。
