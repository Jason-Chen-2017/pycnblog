### 1.4.1 代码实例

在这里，我们以 Linux 操作系统为例，提供一个简单的进程挂起与唤醒的代码实例：

```c
#include <stdio.h>
#include <unistd.h>
#include <sys/wait.h>

int main() {
    pid_t pid = fork();

    if (pid == 0) {
        // 子进程
        printf("子进程开始执行\n");
        sleep(5);
        printf("子进程执行完成\n");
    } else {
        // 父进程
        printf("父进程开始执行\n");
        sleep(1);
        printf("父进程执行完成\n");
        wait(NULL);
        printf("父进程等待子进程完成\n");
    }

    return 0;
}
```

### 1.4.2 详细解释说明

在这个代码实例中，我们使用了 `fork()` 函数来创建一个子进程。子进程和父进程分别执行不同的任务，并在完成任务后进行挂起与唤醒。

子进程首先执行 `printf("子进程开始执行\n")`，然后进行挂起，使用 `sleep(5)` 函数休眠 5 秒。在子进程休眠后，它会被操作系统挂起，释放 CPU 资源。

父进程首先执行 `printf("父进程开始执行\n")`，然后进行挂起，使用 `sleep(1)` 函数休眠 1 秒。在父进程休眠后，它会被操作系统挂起，释放 CPU 资源。

当父进程休眠结束时，操作系统会唤醒父进程，并执行 `printf("父进程执行完成\n")`。父进程继续执行，直到完成任务。

当子进程休眠结束时，操作系统会唤醒子进程，并执行 `printf("子进程执行完成\n")`。子进程继续执行，直到完成任务。

最后，父进程调用 `wait(NULL)` 函数来等待子进程完成，并释放子进程占用的系统资源。当子进程完成任务后，操作系统会唤醒父进程，并执行 `printf("父进程等待子进程完成\n")`。
