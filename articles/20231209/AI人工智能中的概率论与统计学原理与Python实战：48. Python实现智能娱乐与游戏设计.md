# 4.具体代码实例和详细解释说明
以下是一个简单的Python代码实例，用于生成指数分布的随机数：

```python
import numpy as np

def generate_exponential_distribution(lambda_value):
    return np.random.exponential(scale=1/lambda_value)

# 生成1000个随机数
random_numbers = [generate_exponential_distribution(1) for _ in range(1000)]

# 计算随机数的均值
mean = np.mean(random_numbers)
print("均值：", mean)

# 计算随机数的方差
variance = np.var(random_numbers)
print("方差：", variance)
```

在这个代码实例中，我们首先导入了NumPy库，然后定义了一个名为`generate_exponential_distribution`的函数，用于生成指数分布的随机数。接着，我们使用这个函数生成了1000个随机数，并计算了这些随机数的均值和方差。
