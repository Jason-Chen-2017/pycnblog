# 2.核心概念与联系

在深入探讨网络安全与加密技术的核心概念和算法原理之前，我们需要了解一些基本的概念和联系。

## 2.1 加密与解密

加密是一种将原始信息转换为加密信息的过程，以保护信息的机密性。解密是一种将加密信息转换回原始信息的过程，以恢复信息的机密性。

## 2.2 对称加密与非对称加密

对称加密是一种使用相同密钥进行加密和解密的加密方法。非对称加密是一种使用不同密钥进行加密和解密的加密方法。

## 2.3 密钥与密码

密钥是一种用于加密和解密的信息，密码是一种用于验证身份的信息。密钥和密码之间的区别在于，密钥是用于保护信息的机密性，而密码是用于验证身份。

## 2.4 数字证书与数字签名

数字证书是一种用于验证身份和保护信息的方法，数字签名是一种用于验证数据完整性和身份的方法。数字证书和数字签名之间的联系在于，数字证书是一种数字签名的应用。
