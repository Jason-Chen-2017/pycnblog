```python

    if n == 0:
        return 0
    else:
```

1. 代码的结构清晰：代码的结构使用了递归的方式，每个递归的子代码块都有明确的逻辑，确保代码的结构清晰、简洁。

3. 代码的高效性：代码使用了递归的方式，避免了不必要的循环和递归，确保代码的高效性。
