## 4.1 网络优化案例：延迟优化

我们假设有一个网络，其中有多个节点和链路。我们的目标是降低网络延迟，以提高用户体验。

### 4.1.1 代码实例

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建网络
G = nx.Graph()

# 添加节点
G.add_node(1)
G.add_node(2)
G.add_node(3)

# 添加链路
G.add_edge(1, 2, weight=10)
G.add_edge(2, 3, weight=20)

# 计算最短路径
shortest_path = nx.shortest_path(G, source=1, target=3)

# 绘制网络
plt.figure(figsize=(10, 10))
nx.draw(G, with_labels=True)
plt.plot([shortest_path[0], shortest_path[-1]], [0, 0], 'r', linewidth=2)
plt.show()
```

### 4.1.2 解释说明

在这个案例中，我们使用Python的networkx库来创建和绘制网络。我们首先创建一个空的网络，然后添加节点和链路。接下来，我们使用nx.shortest_path函数计算最短路径，从节点1到节点3。最后，我们使用matplotlib库绘制网络，并在其中绘制最短路径。

## 4.2 网络优化案例：可用性优化

我们假设有一个网络，其中有多个节点和链路。我们的目标是提高网络可用性，以提高用户满意度。

### 4.2.1 代码实例

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建网络
G = nx.Graph()

# 添加节点
G.add_node(1)
G.add_node(2)
G.add_node(3)

# 添加链路
G.add_edge(1, 2, weight=10)
G.add_edge(2, 3, weight=20)

# 计算可用性
availability = nx.availability(G, nodes=[1, 2, 3])

# 绘制网络
plt.figure(figsize=(10, 10))
nx.draw(G, with_labels=True)
plt.plot([availability[1][0], availability[1][0]], [0, 0], 'r', linewidth=2)
plt.plot([availability[1][1], availability[1][1]], [0, 0], 'r', linewidth=2)
plt.plot([availability[1][2], availability[1][2]], [0, 0], 'r', linewidth=2)
plt.show()
```

### 4.2.2 解释说明

在这个案例中，我们使用Python的networkx库来创建和绘制网络。我们首先创建一个空的网络，然后添加节点和链路。接下来，我们使用nx.availability函数计算可用性，并将结果绘制在网络图中。
