```python
# 用户角色
user_roles = {
    'user1': ['view', 'edit'],
    'user2': ['view'],
    'user3': ['edit', 'delete'],
}

# 资源权限
resource_permissions = {
    'resource1': ['view', 'edit', 'delete'],
    'resource2': ['view'],
    'resource3': ['edit', 'delete'],
}

# 检查用户是否具有执行某个操作的权限
def check_permission(user, operation, resource):
    user_roles = user_roles[user]
    resource_permissions = resource_permissions[resource]
    return operation in user_roles and operation in resource_permissions

# 示例
user = 'user1'
operation = 'edit'
resource = 'resource1'

if check_permission(user, operation, resource):
    print('用户具有执行该操作的权限。')
else:
    print('用户不具有执行该操作的权限。')
```

在上述代码中，我们使用RBAC进行授权。首先，我们定义了用户角色和资源权限。然后，我们定义了一个检查用户是否具有执行某个操作的权限的函数。最后，我们使用这个函数检查用户是否具有执行某个操作的权限。
