# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的文本分类任务来展示如何进行提示工程。

任务：文本分类，将文本分为两类：正面和负面。

输入提示：“请将以下文本分为正面或负面：”

输出预期：“正面”或“负面”

模型：基于深度学习的文本分类模型，如BERT、GPT等。

具体代码实例如下：

```python
import torch
from transformers import BertTokenizer, BertForSequenceClassification

# 加载预训练模型和tokenizer
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
model = BertForSequenceClassification.from_pretrained('bert-base-uncased', num_labels=2)

# 设置输入提示
input_prompt = "请将以下文本分为正面或负面："

# 设置输入文本
input_text = "我非常喜欢这个电影，它非常有趣。"

# 将输入文本转换为输入序列
input_sequence = tokenizer.encode(input_prompt + input_text, add_special_tokens=True)

# 将输入序列转换为输入张量
input_tensor = torch.tensor(input_sequence).unsqueeze(0)

# 使用模型进行预测
outputs = model(input_tensor)

# 获取预测结果
predictions = torch.softmax(outputs.logits, dim=1)

# 输出预测结果
print(predictions.argmax().item())
```

在这个代码实例中，我们首先加载了预训练的BERT模型和tokenizer。然后，我们设置了输入提示和输入文本。接着，我们将输入文本转换为输入序列，并将输入序列转换为输入张量。最后，我们使用模型进行预测，并输出预测结果。
