贝叶斯定理的公式为：

$$
P(A|B) = \frac{P(B|A) \times P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示条件概率，即事件 A 发生的概率给定事件 B 发生；$P(B|A)$ 表示条件概率，即事件 B 发生的概率给定事件 A 发生；$P(A)$ 表示事件 A 发生的概率；$P(B)$ 表示事件 B 发生的概率。

from scipy.stats import chi2_contingency

    'event2': [1, 0, 1, 0, 0, 1, 0, 1, 0, 1]
})

# 使用贝叶斯定理计算事件之间的关系

# 使用贝叶斯定理计算事件之间的因果关系
odds_ratio, _ = chi2_contingency(data, correction=False)

# 使用贝叶斯定理计算事件之间的可能性

# 输出结果
print('p_value:', p_value)
print('odds_ratio:', odds_ratio)
print('p_event1:', p_event1)
print('p_event2:', p_event2)
