假设我们有一个Hadoop集群，其中包含一个名为“hadoop-user”的用户组，该用户组包含一个名为“user1”的用户。我们希望为“user1”用户分配一个名为“reader”的角色，并为该角色分配一个名为“read”的权限。

具体操作步骤如下：

1.创建角色：

```
hadoop-user$ hadoop fs -mkdir -p /user/reader
hadoop-user$ hadoop fs -chown reader:reader /user/reader
```

2.分配权限：

```
hadoop-user$ hadoop fs -chmod -R 750 /user/reader
```

3.分配角色：

```
hadoop-user$ hadoop fs -chown user1:reader /user/reader
```

4.验证权限：

```
user1$ hadoop fs -ls /user/reader
```
