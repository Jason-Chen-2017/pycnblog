```python
import numpy as np
import matplotlib.pyplot as plt

# 定义环境模型
class Environment:
    def __init__(self):
        self.state = 0

    def step(self, action):
        if action == 0:
            self.state += 1
        elif action == 1:
            self.state -= 1
        return self.state

# 定义智能控制算法
class IntelligentControl:
    def __init__(self, env):
        self.env = env
        self.q_table = np.zeros((env.state_space, env.action_space))

    def choose_action(self, state):
        action = np.argmax(self.q_table[state])
        return action

    def learn(self, state, action, reward, next_state):
        learning_rate = 0.8
        discount_factor = 0.9
        self.q_table[state, action] += learning_rate * (reward + discount_factor * np.max(self.q_table[next_state])) - self.q_table[state, action]

# 初始化环境和智能控制系统
env = Environment()
ic = IntelligentControl(env)

# 训练智能控制系统
for episode in range(1000):
    state = env.state
    done = False
    while not done:
        action = ic.choose_action(state)
        reward = 0
        next_state = env.step(action)
        ic.learn(state, action, reward, next_state)
        state = next_state
        if state == 0:
            done = True

# 可视化结果
plt.plot(ic.q_table[0])
plt.show()
```

在这个代码实例中，我们定义了一个简单的环境模型，并使用Q-学习算法实现了一个智能控制系统。我们首先定义了一个环境类，该类包含一个状态变量，并实现了一个step方法，用于根据输入的动作更新状态。然后，我们定义了一个智能控制类，该类包含一个Q表，用于存储每个状态-动作对的价值。我们实现了choose_action方法，用于根据Q表选择最佳动作，并实现了learn方法，用于根据收到的奖励更新Q表。

在训练智能控制系统的过程中，我们使用了1000个回合来训练智能控制系统。在每个回合中，我们从当前状态开始，选择一个动作，执行该动作，收到一个奖励，并更新Q表。我们继续这个过程，直到达到目标状态。最后，我们可视化了Q表，以观察智能控制系统是否已经学会了如何在环境中取得最佳性能。
