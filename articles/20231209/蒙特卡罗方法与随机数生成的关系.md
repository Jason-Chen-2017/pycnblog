## 4.1 蒙特卡罗方法的Python实现

```python
import random

def monte_carlo(n, f):
    result = 0
    for _ in range(n):
        x = random.random()
        result += f(x)
    return result / n

def func(x):
    return x**2

result = monte_carlo(100000, func)
print(result)
```

在上述代码中，我们首先定义了一个`monte_carlo`函数，它接收两个参数：`n`（试验次数）和`f`（目标函数）。然后我们定义了一个`func`函数，它接收一个参数`x`，并返回`x`的平方。最后，我们调用`monte_carlo`函数，并传入`100000`（试验次数）和`func`（目标函数）作为参数，得到蒙特卡罗方法的估计结果。

## 4.2 随机数生成的Python实现

```python
import random

def generate_random_numbers(n, min_value, max_value):
    result = []
    for _ in range(n):
        result.append(random.uniform(min_value, max_value))
    return result

print(result)
```

在上述代码中，我们首先定义了一个`generate_random_numbers`函数，它接收三个参数：`n`（生成随机数的个数）、`min_value`（随机数的最小值）和`max_value`（随机数的最大值）。然后我们定义了一个`result`列表，用于存储生成的随机数。最后，我们调用`generate_random_numbers`函数，并传入`10`（生成随机数的个数）、`0`（随机数的最小值）和`1`（随机数的最大值）作为参数，得到生成的随机数列表。
