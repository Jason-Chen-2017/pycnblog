                 

# 1.背景介绍

智能制造是一种利用人工智能、大数据、物联网、云计算等技术进行制造业数字化转型的新兴制造业模式。智能制造的核心是通过大数据、人工智能、物联网等技术，实现制造业数字化转型，提高制造业的生产效率、产品质量、环保性能，降低成本，提高竞争力。

智能制造的发展背景主要包括以下几点：

1. 全球化的加剧：随着全球化的加剧，国际市场变得越来越竞争，制造业需要通过提高生产效率、降低成本、提高产品质量等手段来提高竞争力。

2. 信息化的推进：信息化技术的不断推进，使得数据收集、处理、分析等技术得到了很大的发展，为智能制造提供了技术支持。

3. 人工智能的发展：人工智能技术的不断发展，使得机器学习、深度学习等技术得到了很大的发展，为智能制造提供了技术支持。

4. 物联网的发展：物联网技术的不断发展，使得物联网设备得到了很大的发展，为智能制造提供了技术支持。

5. 环保要求的加强：随着环保要求的加强，制造业需要通过提高生产效率、降低成本、提高产品质量等手段来满足环保要求。

6. 制造业数字化转型：制造业需要通过数字化转型，提高生产效率、降低成本、提高产品质量等手段来提高竞争力。

因此，智能制造的发展是制造业数字化转型的必然结果，也是制造业提高竞争力的重要手段。
