在CT图像处理中，滤波是一种常用的图像处理方法，可以用来去除图像中的噪声。在这里，我们使用MATLAB来实现一种常用的滤波算法——中值滤波。

中值滤波的核心思想是：对于每个像素点，找到它周围的邻域内的像素点，然后将这些像素点中的中值作为当前像素点的新值。具体操作步骤如下：

1. 读取CT图像。
2. 定义滤波核。
3. 遍历图像中的每个像素点。
4. 找到当前像素点的邻域内的像素点。
5. 对这些像素点进行排序。
6. 找到排序后的中间值。
7. 将中间值作为当前像素点的新值。
8. 更新图像。

```MATLAB
% 读取CT图像

% 定义滤波核

% 遍历图像中的每个像素点

    for j = 1:size(img,2)
        % 找到当前像素点的邻域内的像素点
        neighbor_pixels = img(i-1:i+1, j-1:j+1);
        [rows, cols] = size(neighbor_pixels);
        % 对这些像素点进行排序
        [sorted_pixels, sorted_indices] = sort(neighbor_pixels(:));

        median_value = sorted_pixels(ceil(rows*cols/2));

end
```

## 4.1 医学信号处理案例：心电图的滤波分析
在心电图处理中，滤波是一种常用的信号处理方法，可以用来去除心电图中的噪声。在这里，我们使用MATLAB来实现一种常用的滤波算法——低通滤波。

低通滤波的核心思想是：对于每个时间点，找到当前时间点的邻域内的时间点，然后将这些时间点中的平均值作为当前时间点的新值。具体操作步骤如下：

1. 读取心电图数据。
2. 定义滤波核。
3. 遍历心电图数据中的每个时间点。
4. 找到当前时间点的邻域内的时间点。
5. 计算邻域内的平均值。
6. 将平均值作为当前时间点的新值。
7. 更新心电图数据。

以下是具体的MATLAB代码实例：

```MATLAB

ecg = ecg_data.ecg;

% 定义滤波核

% 遍历心电图数据中的每个时间点
```
