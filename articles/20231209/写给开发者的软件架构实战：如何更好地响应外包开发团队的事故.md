### 4.2 组件化实例

我们可以使用以下代码实例来演示组件化的实现：

```python
# 定义一个组件
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def calculate_area(self):
        return self.width * self.height

# 使用组件
rectangle = Rectangle(10, 20)
area = rectangle.calculate_area()
print(area)
```

在这个例子中，我们定义了一个 `Rectangle` 类，它用于表示矩形。我们可以通过实例化这个类来使用这个组件。
