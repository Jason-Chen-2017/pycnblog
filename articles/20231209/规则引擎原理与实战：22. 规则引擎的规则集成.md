# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释规则集成的实现方法。

假设我们有一个简单的规则引擎，用于处理订单。订单可以是已付款、未付款或已取消。我们需要根据订单的状态执行不同的操作，例如发送邮件通知、更新订单状态等。

我们可以将这些规则组合成一个规则集，如下：

```python
rules = [
    (lambda order: order.status == 'paid', lambda order: send_email(order, 'paid')),
    (lambda order: order.status == 'unpaid', lambda order: send_email(order, 'unpaid')),
    (lambda order: order.status == 'cancelled', lambda order: send_email(order, 'cancelled'))
]
```

在这个例子中，我们定义了一个列表，其中每个元素是一个元组。元组的第一个元素是一个条件函数，用于判断订单的状态。元组的第二个元素是一个动作函数，用于执行相应的操作。

我们可以通过以下代码实现规则集的执行：

```python
def execute_rules(order):
    for rule in rules:
        if rule[0](order):
            rule[1](order)
            break
```

在这个例子中，我们定义了一个 `execute_rules` 函数，它接受一个订单对象作为参数。函数会遍历所有规则，并根据订单的状态执行相应的操作。
