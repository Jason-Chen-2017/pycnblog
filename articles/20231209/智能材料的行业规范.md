# 2.核心概念与联系

## 2.1 智能材料的基本概念

智能材料是一类具有自主行为能力的材料，它们可以根据环境的变化或外部的刺激进行反应。这种反应可以是物理的、化学的或者是结构的。智能材料的应用范围非常广泛，包括医疗、建筑、汽车、化工等多个领域。

## 2.2 智能材料的核心特性

智能材料的核心特性包括：

1. 自主行为能力：智能材料可以根据环境的变化或外部的刺激进行反应，从而实现自主行为。

2. 可编程性：智能材料可以根据不同的需求进行设计和制造，从而实现不同的功能。

3. 可调节性：智能材料可以根据不同的环境条件进行调节，从而实现不同的性能。

4. 可学习性：智能材料可以根据不同的环境条件进行学习，从而实现不同的应用。

## 2.3 智能材料与传统材料的区别

智能材料与传统材料的主要区别在于其具有自主行为能力和可编程性等特性。传统材料是固定性的，它们的性能和功能不能随着环境的变化而改变。而智能材料则可以根据环境的变化或外部的刺激进行反应，从而实现自主行为和可编程性等特性。
