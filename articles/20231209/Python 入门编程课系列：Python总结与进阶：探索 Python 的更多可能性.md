# 2.核心概念与联系

在深入探讨 Python 的核心概念之前，我们需要了解一些基本的概念。

## 2.1 变量

变量是 Python 中最基本的数据类型之一，用于存储数据。变量可以存储任何类型的数据，包括整数、浮点数、字符串、列表、字典等。

在 Python 中，我们可以使用赋值操作符（=）来创建一个变量。例如：

```python
x = 10
y = "Hello, World!"
z = [1, 2, 3]
```

在这个例子中，我们创建了三个变量：x、y 和 z。x 是一个整数，y 是一个字符串，z 是一个列表。

## 2.2 数据类型

Python 中的数据类型可以分为以下几种：

1. 整数（int）：整数是一种数值类型，用于存储整数值。例如：1、-1、0、100 等。
2. 浮点数（float）：浮点数是一种数值类型，用于存储小数值。例如：1.0、-1.0、0.0、3.14 等。
3. 字符串（str）：字符串是一种文本类型，用于存储文本值。例如："Hello, World!"、"Python"、"123" 等。
4. 列表（list）：列表是一种可变的有序序列类型，用于存储多个值。例如：[1, 2, 3]、["Hello", "World!"]、[100, -100, 0] 等。
5. 元组（tuple）：元组是一种不可变的有序序列类型，用于存储多个值。例如：(1, 2, 3)、("Hello", "World!")、(100, -100, 0) 等。
6. 字典（dict）：字典是一种键值对的无序映射类型，用于存储多个值。例如：{"name": "John", "age": 30, "country": "USA"}、{"x": 10, "y": 20, "z": 30} 等。

在 Python 中，我们可以使用类型() 函数来获取一个变量的数据类型。例如：

```python
x = 10
print(type(x))  # <class 'int'>

y = "Hello, World!"
print(type(y))  # <class 'str'>

z = [1, 2, 3]
print(type(z))  # <class 'list'>
```

在这个例子中，我们使用 type() 函数来获取变量 x、y 和 z 的数据类型。我们可以看到，x 的数据类型是 int，y 的数据类型是 str，z 的数据类型是 list。

## 2.3 控制结构

控制结构是 Python 中的一种重要概念，用于控制程序的执行流程。Python 中的控制结构包括以下几种：

1. 条件语句（if 语句）：条件语句用于根据某个条件来执行不同的代码块。例如：

```python
x = 10
if x > 0:
    print("x 是正数")
else:
    print("x 不是正数")
```

在这个例子中，我们使用 if 语句来判断变量 x 是否大于 0。如果 x 大于 0，则执行 print("x 是正数") 这个代码块；否则，执行 print("x 不是正数") 这个代码块。

2. 循环语句（for 语句和 while 语句）：循环语句用于重复执行某个代码块。例如：

```python
for i in range(5):
    print(i)
```

在这个例子中，我们使用 for 循环来遍历一个范围（range(5)），并在每次迭代中执行 print(i) 这个代码块。

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

在这个例子中，我们使用 while 循环来遍历一个范围（i < 5），并在每次迭代中执行 print(i) 这个代码块，然后更新 i 的值（i += 1）。

在这个例子中，我们定义了一个名为 add 的函数，它接受两个参数（x 和 y），并返回它们的和。我们可以通过调用 add() 函数来使用这个函数。例如：

## 3.1 排序算法

排序算法是一种常用的算法，用于对数据进行排序。Python 中有许多不同的排序算法，如冒泡排序、选择排序、插入排序、归并排序等。

### 3.1.1 冒泡排序

冒泡排序是一种简单的排序算法，它的原理是通过多次交换相邻的元素来逐渐将数据排序。冒泡排序的时间复杂度为 O(n^2)，其中 n 是数据的长度。

以下是冒泡排序的具体操作步骤：

1. 从第一个元素开始，与后续的每个元素进行比较。如果当前元素大于后续元素，则交换它们的位置。
2. 重复第一步，直到所有元素都被排序。
3. 对于每个元素，重复第一步和第二步，直到所有元素都被排序。

以下是冒泡排序的 Python 代码实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(arr))  # [11, 12, 22, 25, 34, 64, 90]
```

在这个例子中，我们定义了一个名为 bubble_sort 的函数，它接受一个数组（arr）作为输入参数，并返回排序后的数组。我们使用冒泡排序算法来对数组进行排序，并将排序后的数组输出。

### 3.1.2 选择排序

选择排序是一种简单的排序算法，它的原理是通过在每次迭代中选择最小（或最大）元素，并将其放在正确的位置。选择排序的时间复杂度为 O(n^2)，其中 n 是数据的长度。

以下是选择排序的具体操作步骤：

1. 从第一个元素开始，找到最小（或最大）元素。
2. 将最小（或最大）元素与当前位置的元素交换。
3. 重复第一步和第二步，直到所有元素都被排序。

以下是选择排序的 Python 代码实例：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(selection_sort(arr))  # [11, 12, 22, 25, 34, 64, 90]
```

在这个例子中，我们定义了一个名为 selection_sort 的函数，它接受一个数组（arr）作为输入参数，并返回排序后的数组。我们使用选择排序算法来对数组进行排序，并将排序后的数组输出。

### 3.1.3 插入排序

插入排序是一种简单的排序算法，它的原理是通过将元素一个一个地插入到已排序的序列中，从而实现排序。插入排序的时间复杂度为 O(n^2)，其中 n 是数据的长度。

以下是插入排序的具体操作步骤：

1. 将第一个元素视为已排序序列的一部分。
2. 从第二个元素开始，将其与已排序序列中的元素进行比较。如果当前元素小于已排序序列中的元素，则将其插入到已排序序列中的适当位置。
3. 重复第二步，直到所有元素都被排序。

以下是插入排序的 Python 代码实例：

```python
def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(insertion_sort(arr))  # [11, 12, 22, 25, 34, 64, 90]
```

在这个例子中，我们定义了一个名为 insertion_sort 的函数，它接受一个数组（arr）作为输入参数，并返回排序后的数组。我们使用插入排序算法来对数组进行排序，并将排序后的数组输出。

### 3.1.4 归并排序

归并排序是一种分治法的排序算法，它的原理是将数据分为两个部分，分别进行排序，然后将排序后的两个部分合并为一个有序的数组。归并排序的时间复杂度为 O(nlogn)，其中 n 是数据的长度。

以下是归并排序的具体操作步骤：

1. 将数据分为两个部分，直到每个部分只包含一个元素。
2. 对每个部分进行递归排序。
3. 将排序后的两个部分合并为一个有序的数组。

以下是归并排序的 Python 代码实例：

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]
    left = merge_sort(left)
    right = merge_sort(right)
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result

arr = [64, 34, 25, 12, 22, 11, 90]
print(merge_sort(arr))  # [11, 12, 22, 25, 34, 64, 90]
```

在这个例子中，我们定义了一个名为 merge_sort 的函数，它接受一个数组（arr）作为输入参数，并返回排序后的数组。我们使用归并排序算法来对数组进行排序，并将排序后的数组输出。

## 3.2 搜索算法

搜索算法是一种常用的算法，用于在数据中查找特定的元素。Python 中有许多不同的搜索算法，如线性搜索、二分搜索等。

### 3.2.1 线性搜索

线性搜索是一种简单的搜索算法，它的原理是通过逐个检查数据中的每个元素，直到找到目标元素。线性搜索的时间复杂度为 O(n)，其中 n 是数据的长度。

以下是线性搜索的具体操作步骤：

1. 从第一个元素开始，检查它是否等于目标元素。
2. 如果当前元素等于目标元素，则返回当前元素的索引。
3. 如果当前元素不等于目标元素，则将当前元素视为下一个元素，并重复第一步和第二步，直到找到目标元素或检查完所有元素。

以下是线性搜索的 Python 代码实例：

```python
def linear_search(arr, x):
    for i in range(len(arr)):
        if arr[i] == x:
            return i
    return -1

arr = [64, 34, 25, 12, 22, 11, 90]
x = 22
result = linear_search(arr, x)
if result == -1:
    print("元素不存在")
else:
    print("元素在数组的第", result, "个位置")
```

在这个例子中，我们定义了一个名为 linear_search 的函数，它接受一个数组（arr）和一个目标元素（x）作为输入参数，并返回目标元素在数组中的索引。我们使用线性搜索算法来查找目标元素，并将查找结果输出。

### 3.2.2 二分搜索

二分搜索是一种高效的搜索算法，它的原理是通过将数据分为两个部分，然后将目标元素与中间元素进行比较，从而缩小搜索范围。二分搜索的时间复杂度为 O(logn)，其中 n 是数据的长度。

以下是二分搜索的具体操作步骤：

1. 将数据分为两个部分，直到每个部分只包含一个元素。
2. 对每个部分进行递归搜索。
3. 将递归搜索的结果与中间元素进行比较，从而缩小搜索范围。

以下是二分搜索的 Python 代码实例：

```python
def binary_search(arr, x):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == x:
            return mid
        elif arr[mid] < x:
            low = mid + 1
        else:
            high = mid - 1
    return -1

arr = [64, 34, 25, 12, 22, 11, 90]
x = 22
result = binary_search(arr, x)
if result == -1:
    print("元素不存在")
else:
    print("元素在数组的第", result, "个位置")
```

在这个例子中，我们定义了一个名为 binary_search 的函数，它接受一个数组（arr）和一个目标元素（x）作为输入参数，并返回目标元素在数组中的索引。我们使用二分搜索算法来查找目标元素，并将查找结果输出。

1. 将数据分为两个部分，直到每个部分只包含一个元素。
2. 对每个部分进行递归排序。
3. 将排序后的两个部分合并为一个有序的数组。
