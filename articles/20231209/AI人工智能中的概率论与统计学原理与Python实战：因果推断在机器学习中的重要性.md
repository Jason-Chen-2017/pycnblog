## 3.1 概率论基础
### 3.1.1 概率的基本概念
概率是一个事件发生的可能性，它通常取值在0到1之间。我们可以用概率来描述事件发生的可能性，也可以用概率来描述事件不发生的可能性。

### 3.1.2 概率的计算方法
我们可以使用以下几种方法来计算概率：
1. 直接计算方法：直接计算事件发生的所有可能性，然后除以总可能性。
2. 定理方法：使用一些数学定理来计算概率，例如总概率定理、贝叶斯定理等。
3. 表格方法：使用概率表格来查找事件的概率。

### 3.1.3 独立事件
独立事件是指发生事件A不会影响事件B的发生。对于独立事件，我们可以使用乘法定理来计算它们的概率。

## 3.2 统计学基础
### 3.2.1 统计学的基本概念
统计学是一门应用数学学科，它利用数学方法来分析实际问题中的数据。在统计学中，我们可以使用描述性统计和性能统计两种方法来分析数据。

### 3.2.2 描述性统计
描述性统计是一种用于描述数据特征的方法，它可以帮助我们更好地理解数据的分布和特点。描述性统计包括中心趋势、离散程度和形状等方面。

### 3.2.3 性能统计
性能统计是一种用于评估模型性能的方法，它可以帮助我们选择最佳的模型。性能统计包括误差率、精度、召回率等方面。

### 3.2.4 统计学的基本方法
在统计学中，我们可以使用以下几种基本方法来分析数据：
1. 分类：将数据按照某个特征进行分类。
2. 比较：比较不同组别之间的数据。
3. 关联：分析数据中的关联关系。
4. 预测：根据历史数据预测未来数据。

## 4.1 概率论代码实例
### 4.1.1 直接计算方法
```python
from math import factorial

def probability_direct_method(n, r):
    return factorial(n) / (factorial(r) * factorial(n - r))

n = 10
r = 3
print(probability_direct_method(n, r))
```

### 4.1.2 定理方法
#### 4.1.2.1 总概率定理
```python
from math import factorial

def total_probability_theorem(P, Q):
    return P + Q

P = 0.5
Q = 0.5
print(total_probability_theorem(P, Q))
```
#### 4.1.2.2 贝叶斯定理
```python
from math import factorial

def bayes_theorem(P, Q):
    return P * Q

P = 0.5
Q = 0.5
print(bayes_theorem(P, Q))
```

### 4.1.3 表格方法
```python
from collections import defaultdict

def probability_table_method(data):
    table = defaultdict(int)
    for item in data:
        table[item] += 1
    return table

data = [1, 2, 3, 4, 5]
print(probability_table_method(data))
```

## 4.2 统计学代码实例
### 4.2.1 描述性统计
#### 4.2.1.1 中心趋势
```python
def central_tendency(data):
    return sum(data) / len(data)

data = [1, 2, 3, 4, 5]
print(central_tendency(data))
```
#### 4.2.1.2 离散程度
```python
def dispersion(data):
    return sum((x - mean)**2 for x in data) / len(data)

data = [1, 2, 3, 4, 5]
mean = central_tendency(data)
print(dispersion(data))
```
#### 4.2.1.3 形状
```python
def shape(data):
    return len(set(data))

data = [1, 2, 3, 4, 5]
print(shape(data))
```

### 4.2.2 性能统计
#### 4.2.2.1 误差率
```python
def error_rate(y_true, y_pred):
    return sum(y_true != y_pred) / len(y_true)

y_true = [0, 1, 1, 0, 1]
y_pred = [0, 1, 1, 0, 0]
print(error_rate(y_true, y_pred))
```
#### 4.2.2.2 精度
```python
def accuracy(y_true, y_pred):
    return sum(y_true == y_pred) / len(y_true)

y_true = [0, 1, 1, 0, 1]
y_pred = [0, 1, 1, 0, 0]
print(accuracy(y_true, y_pred))
```
#### 4.2.2.3 召回率
```python
def recall(y_true, y_pred):
    true_positive = sum(y_true == y_pred)
    false_negative = sum(y_true == 0 and y_pred == 1)
    return true_positive / (true_positive + false_negative)

y_true = [0, 1, 1, 0, 1]
y_pred = [0, 1, 1, 0, 0]
print(recall(y_true, y_pred))
```
