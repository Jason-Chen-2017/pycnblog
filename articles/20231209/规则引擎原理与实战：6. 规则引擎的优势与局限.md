# 4.具体代码实例和详细解释说明

具体代码实例如下：

```python
# 读取规则集合
rules = [
    {"condition": "天气好", "action": "开始游泳"},
    {"condition": "天气坏", "action": "停止游泳"}
]

# 对规则集合进行评估
for rule in rules:
    if eval(rule["condition"]):
        print(rule["action"])

# 根据评估结果执行操作
```

详细解释说明如下：

1. 读取规则集合：在这个例子中，我们读取了一个规则集合，其中每个规则包括一个条件部分和一个操作部分。

2. 对规则集合进行评估：我们使用Python的eval函数来评估规则的条件部分，并根据评估结果来执行操作。

3. 根据评估结果执行操作：我们根据规则的评估结果来执行操作，例如开始游泳或停止游泳。
