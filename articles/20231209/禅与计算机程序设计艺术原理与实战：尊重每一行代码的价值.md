## 动态规划

我们可以通过动态规划来解决最长递增子序列问题。这个问题的目标是找到一个序列中的最长递增子序列。我们可以使用动态规划来解决这个问题。

```python
def longest_increasing_subsequence(arr):
    n = len(arr)
    dp = [1] * n
    for i in range(1, n):
        for j in range(i):
            if arr[j] < arr[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```

在这个代码实例中，我们首先定义了一个动态规划数组dp，其中dp[i]表示以arr[i]结尾的最长递增子序列的长度。然后我们遍历数组arr，并更新dp数组。最后，我们返回dp数组中的最大值，即最长递增子序列的长度。

## 贪心算法

我们可以通过贪心算法来解决活动选择问题。这个问题的目标是选择一个子集，使得子集中的活动不会相互冲突，并且子集的总时长最长。我们可以使用贪心算法来解决这个问题。

```python
def activity_selection(activities):
    activities.sort(key=lambda x: x[1])
    result = []
    current_end = 0
    for activity in activities:
        if activity[0] >= current_end:
            result.append(activity)
            current_end = activity[1]
    return result
```

在这个代码实例中，我们首先对活动列表进行排序，使得活动的结束时间逐个递增。然后我们使用一个当前结束时间的变量来记录当前活动的结束时间。在遍历活动列表的过程中，如果当前活动的开始时间大于等于当前结束时间，我们就将当前活动加入结果列表，并更新当前结束时间。最后，我们返回结果列表，即选择的活动。

我们可以通过分治算法来解决求两个有序数组的交集问题。这个问题的目标是找到两个有序数组的公共元素。我们可以使用分治算法来解决这个问题。

```python
def intersection(arr1, arr2):
    m = len(arr1)
    n = len(arr2)
    result = []
    i = 0
    j = 0
    while i < m and j < n:
        if arr1[i] < arr2[j]:
            i += 1
        elif arr1[i] > arr2[j]:
            j += 1
        else:
            result.append(arr1[i])
            i += 1
            j += 1
    return result
```

在这个代码实例中，我们首先定义了一个结果列表result，用于存储两个数组的公共元素。然后我们使用两个指针i和j来遍历两个数组。在遍历过程中，我们比较两个数组的当前元素，如果当前元素相等，我们就将其加入结果列表，并将两个指针都后移一位。如果当前元素不相等，我们将指针后移一位，以便继续遍历。最后，我们返回结果列表，即两个有序数组的公共元素。
