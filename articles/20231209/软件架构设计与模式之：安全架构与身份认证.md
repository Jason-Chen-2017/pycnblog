```python
class User:
    def __init__(self, username, password):
        self.username = username
        self.password = password

    def authenticate(self, password):
        return self.password == password

class ServiceProvider:
    def __init__(self, user):
        self.user = user

    def authenticate(self, password):
        return self.user.authenticate(password)

class AuthenticationMachine:
    def __init__(self, user):
        self.user = user

    def authenticate(self, password):
        return self.user.authenticate(password)

user = User("user", "password")
service_provider = ServiceProvider(user)
authentication_machine = AuthenticationMachine(user)

if authentication_machine.authenticate("password"):
    print("Authentication successful")
else:
    print("Authentication failed")
```

我们创建了一个`User`实例，并将其传递给`ServiceProvider`和`AuthenticationMachine`实例。然后我们调用`authenticate`方法来验证用户密码。如果验证成功，我们打印“Authentication successful”，否则打印“Authentication failed”。
