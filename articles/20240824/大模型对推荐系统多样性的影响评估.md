$$

$$

然后，计算每个推荐结果的内容集合与其平均值之间的欧几里得距离：

$$
||C_j - \bar{C_i}||_2 = \sqrt{\sum_{d=1}^{D} (C_{jd} - \bar{C_{id}})^2}
$$

最后，计算每个推荐结果的多样性分数：

$$
Diversity_i = \frac{1}{M} \sum_{j=1}^{M} ||C_j - \bar{C_i}||_2

$$

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.metrics.pairwise import cosine_similarity

# 加载用户数据
user_data = pd.read_csv('user_data.csv')
user行为特征 = user_data[['点击数', '收藏数', '评论数']]
user行为特征 = user行为特征.values

# 加载内容数据
content_data = pd.read_csv('content_data.csv')
content特征 = content_data[['文本特征', '类别特征']]
content特征 = content特征.values

# 数据预处理
user行为特征 = np.array(user行为特征)
content特征 = np.array(content特征)
user行为特征，content特征 = train_test_split(user行为特征，content特征，test_size=0.2，random_state=42)

# 构建关系模型
关系矩阵 = cosine_similarity(user行为特征，content特征)

# 生成推荐结果
推荐结果 = []
for i in range(len(关系矩阵)):
    推荐结果.append(np.argsort(关系矩阵[i])[-5:][::-1])
推荐结果 = np.array(推荐结果)

# 多样性评估
多样性分数 = []
for i in range(len(推荐结果)):
    内容集合 = np.array([content特征[j] for j in 推荐结果[i]])
    平均值 = np.mean(内容集合, axis=0)
    多样性分数.append(np.sum(np.linalg.norm(内容集合 - 平均值, axis=1)))
多样性分数 = np.mean(多样性分数)

print('多样性分数：', 多样性分数)
```

4. **推荐生成**：代码根据关系矩阵生成推荐结果，为每个用户推荐5个最相似的内容。

5. **多样性评估**：代码计算每个推荐结果的内容集合的平均值，并计算每个内容集合与其平均值之间的欧几里得距离。最后，计算所有推荐结果的多样性分数。
