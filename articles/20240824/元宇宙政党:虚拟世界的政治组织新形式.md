```python
# 成员活跃度模型
class MemberActivityModel:
    def calculate_activity(self, activities, total_days):
        return len(activities) / total_days

# 民主投票模型
class VotingModel:
    def calculate_winner(self, votes):
        max_votes = max(votes.values())
        winner = [k for k, v in votes.items() if v == max_votes]
        return winner

# 法规制定模型
class LawMakingModel:
    def calculate_pass_rate(self, supports, total_votes):
        return (supports / total_votes) * 100
```

5.3 代码解读与分析

- `MemberActivityModel` 类实现了成员活跃度模型的计算方法，包括活动次数和总天数的比例。
- `VotingModel` 类实现了民主投票模型的计算方法，包括计算得票数和胜选者。
- `LawMakingModel` 类实现了法规制定模型的计算方法，包括计算支持票数和通过率。
