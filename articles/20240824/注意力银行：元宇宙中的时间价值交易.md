## 4. 数学模型和公式

### 4.1 数学模型构建

注意力银行的数学模型主要基于注意力积分的计算和分配。假设用户A和B在注意力银行平台进行交易，A拥有x个注意力积分，B拥有y个注意力积分。根据交易规则，平台将按照一定比例收取手续费，剩余积分将分配给A和B。

设手续费率为r%，剩余积分分配比例为α:β（α+β=1），则用户A和B最终获得的注意力积分分别为：

用户A：x' = x * (1 - r%) * α  
用户B：y' = y * (1 - r%) * β

### 4.2 公式推导过程

1. 注意力积分计算

用户A和B的交易金额分别为x和y，根据手续费率r%，平台将收取手续费r * min(x, y)。剩余积分将为x + y - r * min(x, y)。

2. 剩余积分分配

根据剩余积分分配比例α:β，用户A和B将分别获得剩余积分的α份和β份。即：

用户A：x' = (x + y - r * min(x, y)) * α  
用户B：y' = (x + y - r * min(x, y)) * β

3. 注意力积分结算

用户A和B最终获得的注意力积分分别为x'和y'，根据交易规则，平台将自动进行结算。

### 4.3 案例分析与讲解

假设用户A拥有100个注意力积分，用户B拥有200个注意力积分。平台手续费率为5%，剩余积分分配比例为3:2。

根据数学模型，用户A和B的交易金额分别为100和200，平台将收取手续费5个注意力积分。剩余积分将为100 + 200 - 5 = 295个注意力积分。

根据剩余积分分配比例，用户A和B将分别获得：

用户A：x' = 295 * 0.3 = 88.5个注意力积分  
用户B：y' = 295 * 0.2 = 59.0个注意力积分

最终，用户A和B将分别拥有：

用户A：100 - 88.5 = 11.5个注意力积分  
用户B：200 - 59.0 = 141.0个注意力积分

```python
# 注意力银行系统

import blockchain
import web_framework

class AttentionBank:
    def __init__(self):
        self.blockchain = blockchain.Blockchain()
        self.web_framework = web_framework.WebFramework()

    def register_user(self, user_id, real_name):
        # 用户注册
        self.blockchain.add_block(user_id, real_name)

    def get_attention_points(self, user_id, task_id):
        # 获取注意力积分
        return self.blockchain.get_attention_points(user_id, task_id)

    def trade_attention_points(self, user_id_a, user_id_b, attention_points):
        # 注意力积分交易
        self.blockchain.trade_attention_points(user_id_a, user_id_b, attention_points)

    def settle_account(self, user_id_a, user_id_b, attention_points_a, attention_points_b):
        # 注意力积分结算
        self.blockchain.settle_account(user_id_a, user_id_b, attention_points_a, attention_points_b)

    def start(self):
        # 启动系统
        self.web_framework.start(self.register_user, self.get_attention_points, self.trade_attention_points, self.settle_account)

if __name__ == "__main__":
    attention_bank = AttentionBank()
    attention_bank.start()
```
