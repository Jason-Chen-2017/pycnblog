### 4.1 数学模型构建

### 4.2 公式推导过程

1. **用户兴趣模型**：

   假设用户 u 的兴趣可以用向量表示为 \( \mathbf{u} \)，商品 i 的特征向量表示为 \( \mathbf{i} \)，则用户 u 对商品 i 的兴趣度可以通过余弦相似度计算：

   \[
   \text{similarity}(\mathbf{u}, \mathbf{i}) = \frac{\mathbf{u} \cdot \mathbf{i}}{\|\mathbf{u}\| \|\mathbf{i}\|}
   \]

2. **商品特征提取模型**：

   假设商品 i 的特征向量表示为 \( \mathbf{i} \)，则商品 i 的类别可以通过 K-means 聚类算法提取：

   \[
   \mathbf{i}_k = \frac{1}{N} \sum_{i=1}^{N} \mathbf{i}_i
   \]

   其中，\( N \) 为聚类中心个数，\( \mathbf{i}_k \) 为第 k 类别的商品特征向量。

3. **推荐模型**：

   假设用户 u 对商品 i 的兴趣度表示为 \( r_{ui} \)，则用户 u 的个性化推荐列表可以通过矩阵分解算法计算：

   \[
   R = UV^T
   \]

   其中，\( R \) 为用户 u 的兴趣度矩阵，\( U \) 为用户特征矩阵，\( V \) 为商品特征矩阵。

from sklearn.metrics.pairwise import cosine_similarity
from tensorflow.keras.models import Model
from tensorflow.keras.layers import Input, Dense

# 数据集加载与预处理
data = pd.read_csv('data.csv')
data = data.dropna()

# 用户行为数据
user_actions = data.groupby('user_id')['item_id'].apply(list).reset_index()

# 商品特征数据
item_features = data.groupby('item_id')['feature'].apply(list).reset_index()

# K-means 聚类
kmeans = KMeans(n_clusters=3)
clusters = kmeans.fit_predict(item_features['feature'])

# 余弦相似度计算
similarity_matrix = cosine_similarity(item_features['feature'])

# 用户兴趣度计算
user_similarity = np.array([[similarity_matrix[item_id].dot(similarity_matrix[other_item_id]) / np.linalg.norm(similarity_matrix[item_id]) / np.linalg.norm(similarity_matrix[other_item_id]) for other_item_id in item_actions] for item_id, item_actions in user_actions.values])

# 矩阵分解
input_user = Input(shape=(1,))
input_item = Input(shape=(1,))

user_embedding = Dense(10, activation='relu')(input_user)
item_embedding = Dense(10, activation='relu')(input_item)

dot_product = Dense(1, activation='sigmoid')(user_embedding * item_embedding)

model = Model(inputs=[input_user, input_item], outputs=dot_product)
model.compile(optimizer='adam', loss='binary_crossentropy')

# 训练模型
model.fit([user_similarity[:, i], similarity_matrix[:, i]] for i in range(user_similarity.shape[0]), epochs=10)

# 推荐列表生成
for user_id in user_actions.index:
    user_actions_at = user_actions.loc[user_id]
    recommendations = np.argsort(model.predict([user_similarity[user_id], similarity_matrix[user_id]]))[-5:]
    print(f'User {user_id} recommendations: {item_features.iloc[recommendations]['item_id'].values}')
