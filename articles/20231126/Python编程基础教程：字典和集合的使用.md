                 

# 1.背景介绍


在本教程中，你将学习到字典（Dictionary）和集合（Set）的一些基本知识，并通过实例和实践来加强对这两个数据结构的理解。字典和集合都是非常重要的数据结构，它们是编程中经常使用的两种主要数据类型。

字典（Dictionary）是一种无序的键值对组成的元素，其中每一个键（Key）都对应唯一的一个值（Value）。字典可以让我们根据某个键快速地查找对应的值。比如，我们可以使用“姓名”作为键，找到相应的人员信息。而集合（Set）则不允许重复的成员。比如，我们可以通过集合来判断某个人是否已经加入了某个群组。

## 数据类型定义
字典（Dictionary）：由一组无序的键值对构成的元素，其中每个键值对是一个项，每个项都由一个键和一个值组成；键值对的形式为 key:value 。键必须是不可变对象，而值则可以是任意的对象。字典用{}表示。例如：{"name": "John", "age": 30} 。字典中的值可以修改，但键不能。

集合（Set）：由无序的、不重复的元素组成的元素，它是一个容器，里面存放着许多元素，并提供一些操作方法。集合用{}表示。例如：{1, 2, 3, 3, 2, 1} 。

## 特点
字典和集合的共同特征是：数据存储、访问及遍历的方式类似。

- **键**：字典中的每个键值对都有一个独一无二的键，用来标识这个值。通常情况下，键必须是不可变对象。
- **值**：字典中的每个键值对都有一个与之相关的值，也就是说，字典中的值可以是任何类型。字典中的值可以修改，但是键不能。
- **遍历**：由于字典中的元素是无序的，因此我们无法确定其遍历顺序。然而，我们可以通过键或值的集合进行遍历。
- **查找速度**：对于字典来说，查找某个特定的值十分迅速，因为我们只需要查询键即可。而对于集合来说，查找元素的时间复杂度取决于集合的大小。

# 2.核心概念与联系
字典（Dictionary）是一种无序的、可变的序列类型，由一系列键值对组成，每个键值对是一个项。字典中的每一个键都对应唯一的一条值，并且键是不可变的。集合（Set）也是一个可变序列类型，但不同的是，它只能包含不可变对象。集合中元素的顺序不是固定的，且不允许重复的元素。

字典和集合都是非常重要的数据结构，它们之间存在着很多相似之处。以下是在使用上会遇到的一些关键概念：

1. **键**：字典的键（key）必须是不可变对象，这意味着它们必须是可哈希的，且不能被修改。
2. **映射**：字典（dict）是键值对（key-value pair）的集合，它使用键而不是索引来获取值。
3. **动态性**：字典（dict）支持动态添加、删除元素，而集合（set）不支持。
4. **创建方式**：字典（dict）是通过 {} 的语法创建的，而集合（set）是通过 set() 函数创建的。
5. **操作**：字典（dict）的操作包括添加、删除、修改键值对、获取值等，而集合（set）的操作仅包括添加和删除元素。
6. **键不存在时返回默认值：**如果试图获取不存在的键值对的值，就会抛出 KeyError 异常。但对于集合（set），若元素不存在则返回 False。
7. **性能比较**：由于字典（dict）具有更快的查找速度，所以对于需要频繁读取或者写入数据的场合，建议使用字典；而对于存储不可变对象、且不需要保持元素顺序的信息的场景，使用集合（set）效率更高。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 创建字典
创建一个空字典的方法如下所示：

```python
my_dict = {} # 这是字典的声明方式
```

创建一个字典并向其中添加键值对的方法如下：

```python
my_dict = {"apple": 1, "banana": 2, "orange": 3} # 通过字典字面量创建字典
```

上面创建了一个字典，其中包含三个键值对。

## 3.2 访问字典中的值
要访问字典中的某个值，可以通过如下的方法：

```python
print(my_dict["banana"]) # 返回值为 2
```

通过字典[key]来访问指定键的值。当指定的键不存在时，会抛出 KeyError 异常。

## 3.3 更新字典中的值
要更新字典中的某个值，可以使用赋值运算符号 (=) 来实现：

```python
my_dict["apple"] = 10
print(my_dict["apple"]) # 返回值为 10
```

通过字典[key]来访问指定键的值，然后对该值重新赋值即可。若键不存在，则会自动创建此键并赋值。

## 3.4 删除字典中的元素
要删除字典中的元素，可以使用 del 语句来实现：

```python
del my_dict["banana"]
print(my_dict) # 返回值为 {"apple": 10, "orange": 3}
```

通过 del 语句来删除指定键值对。注意：删除元素后，字典的长度不变。若尝试访问已删除的键，会抛出 KeyError 异常。

## 3.5 迭代字典
字典是可迭代对象的一种，可以通过 for...in 循环来迭代字典：

```python
for item in my_dict:
    print(item) # apple orange
```

这个例子展示了如何通过 for...in 循环来迭代字典的所有键。你可以通过 dict.keys() 方法来获取字典所有键的列表：

```python
keys = list(my_dict.keys())
print(keys) # ['apple', 'orange']
```

和通过 for...in 循环获取键值对的方式一样。不过，通过 keys() 方法，我们还可以获得键的集合。
