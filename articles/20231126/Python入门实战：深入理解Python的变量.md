                 

# 1.背景介绍


## 什么是变量？
在编程语言中，变量是一个容器，它可以用来存储、修改或传递数据。
计算机程序中的变量一般分为两种类型：
- 本地变量（Local variable）：程序运行时在函数内部声明的变量称为本地变量，它只能在函数内被访问；
- 全局变量（Global variable）：程序运行时在函数外部声明的变量，它可以在整个程序范围内被访问。

## 为何要用变量？
变量的作用主要有以下几个方面：
- 数据共享：通过变量，多个函数或者模块之间可以共享数据；
- 命名空间隔离：不同的变量名称可以起到相同的作用域的效果，有效防止命名冲突；
- 便于维护：当变量名称发生变化时，只需要修改一次即可；
- 提高性能：一些计算量大的运算或循环，如果重复执行的话，就可以将结果保存到变量中，避免重复计算；
- 增加可读性：变量名称通常有助于提高代码的可读性。

# 2.核心概念与联系
变量的基本属性包括：
- 标识符(identifier)：变量的名字；
- 数据类型(data type)：变量所存放的数据类型；
- 值(value)：变量当前的值；
- 地址(address)：内存地址，指向变量在内存中的位置。

## 标识符
标识符就是一个给变量起个易懂的名字，让代码更容易阅读、维护和管理。Python中要求变量名必须遵循标识符规则，即只能由字母、数字和下划线组成，且不能以数字开头。

```python
# 合法的标识符
a = 1
x_y_z = "hello world"

# 不合法的标识符
_abc = 9 # 以下划线开头的标识符不是合法的
@var = 7 # 以@开头的标识符不是合法的
1var = 3 # 以数字开头的标识符不是合法的
```

## 数据类型
Python中有几种内置的数据类型：
- Number（数字）：int、float、complex（复数）
- String（字符串）
- List（列表）
- Tuple（元组）
- Set（集合）
- Dictionary（字典）

Python中数据类型也分为动态类型与静态类型，动态类型是在运行时才检查数据类型，而静态类型则是在编译阶段就已经确定了数据类型。

```python
num = 10
print(type(num))    # <class 'int'>
num = "Hello World!"
print(type(num))    # <class'str'>
```

## 值
变量的值是指该变量当前存储的数据。对于局部变量来说，值的改变不会影响其他变量，对于全局变量来说，任何对它的赋值都会改变它的值。

```python
a = 10     # a 的值为 10
b = a      # b 的值为 10

a = 20     # 此处 a 的值为 20，并不影响 b 的值

def func():
    c = 30   # 在函数内部声明的变量 c 是局部变量
    
func()
print(c)    # 报错，c 没有定义
```

## 地址
Python的变量虽然可以通过标识符来访问，但每个变量都有一个唯一的内存地址，可以通过id()函数获取变量的地址。

```python
a = 10
print("变量 a 的地址为：", id(a))    # 输出：变量 a 的地址为： 140121686466248
```
