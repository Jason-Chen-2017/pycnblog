# 2.核心概念与联系
## 数据结构与算法
数据结构与算法（Data Structure and Algorithms）是一门研究计算机存储、组织数据的方式、以及对这些数据进行有效访问、操作和维护的一门学科。数据结构描述了如何将信息存储在计算机中的数据对象中，算法则是指用来操作数据的计算方法或操作规则。数据结构和算法之间存在着密切的联系，其中数据结构是指数据的组织方式，而算法则是在数据结构基础上的运算规则。数据结构与算法也是各种编程语言的基础，只有掌握了数据结构与算法，才能写出高效、健壮且正确的代码。下面是一些常用的数据结构和算法，供大家参考。

1.数组 Array:一个元素序列。数组是最基本的数据结构之一。它将相同类型元素集合起来，并使用一个唯一的标识符来引用每个元素，从而可以轻松地访问、修改和管理元素。在Python中，数组是列表Array或元组Tuple的别名。

2.链表 Linked List:一种动态的数据结构，它将一组零散的内存块串联成一个整体，使得任意位置的内存块都可直接被访问到。链表是一种灵活的、非连续的存储结构，可以在运行时动态增加或减少元素，因此非常适合插入、删除操作频繁的场合。Python中提供了collections模块下的deque类实现了双端队列。

3.栈 Stack:一种先进后出的线性表数据结构。栈具有两个基本操作：入栈push(添加元素)和出栈pop(删除元素)。栈经常用于回撤操作，例如函数调用、表达式求值、矩阵运算、迷宫寻路等。Python中提供了list或者deque类的模拟实现。

4.队列 Queue:一种先进先出线性表数据结构。队列具有两个基本操作：入队enqueue(添加元素)和出队dequeue(删除元素)。队列通常用在进程调度、消息传递、数据传输等应用场景。Python中提供了queue模块的Queue类实现。

5.树 Tree:树是一种数据结构，它是由n个结点组成的集合，其中每个结点有零个或多个子结点，构成一条又线性的链。树的定义允许出现环形链接，但不允许出现自环，并且在任意两个结点间只有唯一的一条路径相连。树具有如下重要性质：根节点无前驱，除根外的所有其他节点均有且仅有一个父节点，所有叶节点（即没有子节点的节点）均处于同一层，且不属于任何子树，也不属于另一棵树。有根树、无根树。二叉树（Binary Tree）是每个节点最多有两颗子树的树。完全二叉树（Full Binary Tree），所有叶节点均处于最后两层，且最底下两层的各叶节点都集中在左侧。满二叉树（Complete Binary Tree），除了最底层，所有层的节点数都达到最大值。

## 图论 Graph Theory
图论（Graph theory）是一门关乎对图形结构进行研究的一门学术科目，主要涉及图形的表示、分析、分类、生成和识别、复杂度、应用、优化、算法、理论等诸多方面。图论的基本想法是研究如何将复杂的问题转化为图论的形式化模型，然后利用图论的分析手段、技术手段、模型方法、理论方法等对问题进行有效的解决。常用的图论的模型有：带权图 Weighted Graph、网 Graph、森林 Forest、星星 Star、无向图 Undigraph、无环图 Undirected Graph、有向图 Directed Graph、子图 Subgraph、顶点 Vertex、边 Edge、路径 Path、哈密顿回路 Hamiltonian Circuit、欧拉回路 Eulerian Circuit、割 Cut、顶点覆盖 Vertex Cover、最小割问题 Minimum Cut Problem、无向图划分问题 Bipartite Partition Problem。
