# 2.核心概念与联系
## 2.1 数据可视化概述
数据可视化（Data Visualization）是指利用图表、图像、文本等各种媒体形式，直观地呈现数据特征及其之间的关系，帮助用户理解和发现数据间的模式，从而提升数据分析结果、改进决策制定、优化管理工作等方面的效率。数据可视化是一门独立的学术研究，涉及数学、统计学、计算机科学、信息论、工程技术、心理学等多个领域。

数据可视化的主要任务是探索性数据分析过程中的一个重要环节。在探索性数据分析过程中，数据科学家需要快速识别出数据中的模式、规律以及隐藏的信息，从而找出数据的价值所在，并根据这些发现制定更好的决策或解决方案。由于数据的复杂性和多样性，不同的数据类型有着不同的可视化方法和工具。因此，数据可视化有着广泛的应用领域，包括金融、市场营销、医疗保健、制造业运营、环境保护、教育、娱乐以及其它任何需要对数据进行清晰、准确、易懂的呈现的场景中。

## 2.2 基本数据可视化知识
### 2.2.1 什么是信息密度？
信息密度（Information Density）是指给定数据的无重复值的数量占比。即所谓信息密度就是指数据集中不重复的或者说可以被无损移除的变量个数所占的比例。换句话说，就是数据集中存在多少的独特的事物。比如我们有一个数据集，里面包含五个人名，那么这个数据集的信息密度为0.2，因为其中有两人重名，所以四个人才能构成完整的数据单元。数据可视化中也常用信息密度表示数据的复杂程度，也称数据空间的度量。数据空间的度量使得我们可以更好地判断数据集是否具有足够的复杂度，以及如何有效地分析数据。
