# 2.核心概念与联系
首先，了解一些基本的计算机视觉的相关概念及其之间的联系：

## 2.1 像素与图像

图像即由像素点组成，每个像素点代表一个笛卡尔坐标系中的点。一般情况下，我们通常将图像划分成若干个矩形区域(称为宫格)，每一个矩形区域里的所有像素共享相同的颜色属性。


图像的尺寸大小即指的是图片中包含的像素数量。一般而言，具有$n\times m$个像素的图像，通常记作$(n,m)$。当图像的尺寸太大时，便称之为超高分辨率(Ultra-High Resolution, UHR)图像；相反，图像的尺寸太小则称之为低分辨率(Low Resolution, LR)图像。

## 2.2 颜色空间

颜色空间是一个三维向量空间，它定义了颜色的取值范围。图像一般都是用数字的方式来表示颜色信息，但是不同图像设备、传感器等输出的信号可能有不同的编码方式。因此，我们需要对这些信号进行统一的编码，然后再将它们用于处理。

不同颜色空间之间存在一定的转换关系，这样就可以用一种颜色空间进行图像的表示，再转换到另一种颜色空间进行后续的图像处理、分析工作。常用的颜色空间有RGB、XYZ、HSL、HSV、CMYK四种。


## 2.3 像素值与灰度值

在不同的色彩模型下，像素值的含义也会有所不同。常见的色彩模型有RGB、CMYK、YCrCb、HSV等。对于彩色图像，其每个像素点的值通常为三个整数值表示红色、绿色、蓝色三个通道的亮度值。对于黑白图像，其每个像素点的值只是一个整数值表示灰度值。

## 2.4 图像增强

图像增强技术是指利用数学方法或光学原理对图像进行改善或改进，使其更具视觉效果。目前比较流行的图像增强技术包括去雾算法、直方图均衡化、锐化、锯齿模糊、运动模糊、质心法、CLAHE算法等。

## 2.5 OpenCV

OpenCV(Open Source Computer Vision Library)是一个跨平台计算机视觉库，由Intel开发并开源。OpenCV提供超过2500个经过优化的函数，支持几乎所有的主要操作系统。OpenCV被广泛应用于计算机视觉、图像处理、机器学习和医疗影像等领域。在本文中，我们将结合OpenCV进行图像处理与计算机视觉的实践。
