## 4.1 动态规划实例

考虑一个经典的背包问题，我们有n个物品，每个物品都有一个重量和价值，我们需要将这些物品放入一个容量为W的背包中，以最大化总价值。

```python
def knapsack(W, wt, val, n):
    K = [[0 for _ in range(W + 1)] for _ in range(n + 1)]

    for i in range(n + 1):
        for w in range(W + 1):
            if i == 0 or w == 0:
                K[i][w] = 0
            elif wt[i - 1] <= w:
                K[i][w] = max(val[i - 1] + K[i - 1][w - wt[i - 1]], K[i - 1][w])
            else:
                K[i][w] = K[i - 1][w]

    return K[n][W]
```

在这个代码实例中，我们使用了动态规划的思想，通过递归地解决子问题，来解决原问题。我们创建了一个二维数组K，用于存储每个物品和背包容量的最大价值。然后，我们通过遍历所有物品和背包容量，来计算最大价值。

```python
def pow(base, exponent):
    if exponent == 0:
        return 1
    elif exponent == 1:
        return base
    else:
        p = pow(base, exponent // 2)
        if exponent % 2 == 0:
            return p * p
        else:
            return p * p * base
```

在这个代码实例中，我们使用了分治的思想，通过将问题分割成多个子问题，然后将子问题合并为原问题的解决方案。我们首先判断指数是否为0或1，如果是，则直接返回结果。否则，我们将指数除以2，并递归地计算两个子问题的结果。如果指数为偶数，则将两个子问题的结果相乘；如果指数为奇数，则将两个子问题的结果相乘，并将基数加入结果中。
