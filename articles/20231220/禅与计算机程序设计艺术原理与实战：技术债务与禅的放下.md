## 4.1 代码实例
我们将通过一个简单的排序算法来展示禅学算法的实现。

```python
def fja_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

## 4.2 详细解释说明
上述代码实现了一个简单的排序算法，该算法使用了禅学的原理。通过直接体验现实，我们可以看到该算法的时间复杂度为$O(n^2)$，空间复杂度为$O(1)$。
