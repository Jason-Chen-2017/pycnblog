```
user_id | resource_id | score
```

接下来，我们需要计算用户之间的相似度。我们可以使用余弦相似度公式来计算这些相似度。例如：

```python
from scipy.spatial.distance import cosine

def similarity(user_a, user_b):
    return cosine(user_a, user_b)
```

然后，我们需要找到用户的邻居。邻居是那些相似度较高的用户。例如：

```python
def find_neighbors(user_id, similarity_threshold):
    neighbors = []
    for i, user in enumerate(user_data):
        if user['user_id'] != user_id:
            similarity = similarity(user[2:], user_data[user_id][2:])
            if similarity > similarity_threshold:
                neighbors.append((i, user))
    return neighbors
```

最后，我们需要推荐学习资源。我们可以使用用户的邻居来预测用户对未见过的学习资源的评分。例如：

```python
def recommend(user_id, resource_id, similarity_threshold):
    neighbors = find_neighbors(user_id, similarity_threshold)
    predictions = []
    for i, neighbor in enumerate(neighbors):
        user = neighbor[1]
        score = 0
        for j, resource in enumerate(user[2:]):
            if resource[0] == resource_id:
                score = resource[1]
                break
        predictions.append((i, score))
    return sorted(predictions, key=lambda x: x[1], reverse=True)
```
