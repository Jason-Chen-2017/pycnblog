## 4.3 代码实例

以下是一个具体的代码实例，用于计算两个矩阵的乘积：

```python
class Matrix:
    def __init__(self, rows, cols, elements):
        self.rows = rows
        self.cols = cols
        self.elements = elements

    def multiply(self, other):
        if self.cols != other.rows:
            raise ValueError("The number of columns in the first matrix must be equal to the number of rows in the second matrix.")

        result = Matrix(self.rows, other.cols, [0] * (self.rows * other.cols))
        for i in range(self.rows):
            for j in range(other.cols):
                for k in range(self.cols):
                    result.elements[i * other.cols + j] += self.elements[i * self.cols + k] * other.elements[k * other.cols + j]
        return result

# 创建两个矩阵
matrix1 = Matrix(2, 3, [1, 2, 3])
matrix2 = Matrix(3, 2, [4, 5, 6])

# 计算矩阵乘积
result = matrix1.multiply(matrix2)
print(result.elements)
```
