# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的例子来说明禅宗与编程的实践。我们将实现一个简单的计数器，用于计算从1到n的所有整数的和。

```python
def sum_of_integers(n):
    total = 0
    for i in range(1, n + 1):
        total += i
    return total
```

在这个例子中，我们首先定义了一个名为`sum_of_integers`的函数，该函数接受一个整数参数`n`。然后，我们创建了一个名为`total`的变量，用于存储整数的和。接下来，我们使用了一个`for`循环来遍历从1到`n`的所有整数，并将每个整数加到`total`变量中。最后，我们返回了`total`变量的值。
