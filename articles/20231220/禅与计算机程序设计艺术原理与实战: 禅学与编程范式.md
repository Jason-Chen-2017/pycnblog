冒泡排序是一种简单的排序算法，它的时间复杂度为O(n^2)。其中，n是输入序列的长度。冒泡排序的基本操作是通过比较相邻的元素，如果它们的顺序错误，就交换它们。这个过程重复地进行，直到所有元素都排序为止。

冒泡排序的数学模型公式可以表示为：

T(n) = n(n-1)/2

其中，T(n)是冒泡排序的时间复杂度，n是输入序列的长度。

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
