### 1.4.1 例子：并行计算求和

假设我们需要计算一个大整数的和，如：

$$ sum = a_1 + a_2 + a_3 + \cdots + a_n $$

为了更快地计算这个和，我们可以将这个问题分解为多个子问题，并将子问题分配给不同的处理单元进行并行计算。具体的代码实例如下：

```python
import multiprocessing as mp

def sum_partial(start, end, data):
    result = 0
    for i in range(start, end):
        result += data[i]
    return result

if __name__ == '__main__':
    data = [i for i in range(1, 1000000)]
    n_process = mp.cpu_count()
    pool = mp.Pool(processes=n_process)
    result = pool.map(sum_partial, [i for i in range(0, len(data), n_process)], [data])
    print("Sum:", sum(result))
```

在这个例子中，我们首先定义了一个名为 `sum_partial` 的函数，该函数接受一个区间和一个数据列表作为输入，并计算该区间内的和。然后，我们在主程序中创建了一个进程池，将数据列表分成多个子列表，并将子列表及其对应的数据列表分配给不同的处理单元进行并行计算。最后，我们通过 `pool.map` 函数将计算结果汇总到一个列表中，并输出和的结果。

### 1.4.2 解释说明

在这个例子中，我们通过将问题分解为多个子问题，并将子问题分配给不同的处理单元进行并行计算，实现了并行计算的效果。通过将计算任务分配给多个处理单元，我们可以充分利用计算机硬件的并行处理能力，显著提高计算性能。
