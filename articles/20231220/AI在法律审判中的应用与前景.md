## 4.1 文本分析代码实例

在本节中，我们将通过一个文本分析代码实例来详细解释AI在法律审判中的应用。

### 4.1.1 文本预处理

文本预处理是文本分析的一个重要部分，它可以帮助律师更快速地处理大量的法律文献和文书。以下是一个简单的文本预处理代码实例：

```python
import re

def preprocess_text(text):
    # 删除特殊字符
    text = re.sub(r'[^\w\s]', '', text)
    # 转换为小写
    text = text.lower()
    # 分割为词汇表
    words = text.split()
    return words
```

### 4.1.2 关键词提取

关键词提取是文本分析的一个重要部分，它可以帮助律师更好地理解文本的主题。以下是一个简单的关键词提取代码实例：

```python
from collections import Counter

def extract_keywords(words):
    # 计算词汇出现次数
    word_counts = Counter(words)
    # 获取最常见的词汇
    keywords = word_counts.most_common(10)
    return keywords
```

### 4.1.3 文本摘要

文本摘要是文本分析的一个重要部分，它可以帮助律师更快速地查看文本内容。以下是一个简单的文本摘要代码实例：

```python
def generate_summary(text, keywords):
    # 分割为词汇表
    words = text.split()
    # 获取关键词
    keyword_list = [word[0] for word in keywords]
    # 生成摘要
    summary = ' '.join(keyword_list)
    return summary
```

    text = text.lower()

### 4.2.2 文本分类

文本分类是文书自动化的一个重要部分，它可以帮助律师更好地将文本分类到不同的类别。以下是一个简单的文本分类代码实例：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import Pipeline

# 训练集
train_texts = ['合同', '诉讼文书', '婚姻案件', '仲裁']
# 类别
train_labels = ['contract', 'litigation', 'marriage', 'arbitration']

# 构建模型
model = Pipeline([
    ('vectorizer', TfidfVectorizer()),
    ('classifier', MultinomialNB()),
])

# 训练模型
model.fit(train_texts, train_labels)

# 测试文本
test_text = '这是一个合同'
# 预测类别
predicted_label = model.predict([test_text])[0]
print(predicted_label)  # 输出: contract
```
