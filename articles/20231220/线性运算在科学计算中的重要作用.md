# 4.具体代码实例和详细解释说明

1. 线性方程组的解：

```python
import numpy as np

A = np.array([[1, 1], [1, 1]])
B = np.array([2, 2])

X = np.linalg.solve(A, B)
print(X)
```

2. 矩阵的乘法和逆矩阵：

```python

print(C)

```

3. 矩阵分解和奇异值分解：

```python

A = np.array([[1, 1], [1, 1]])

U, S, V = np.linalg.svd(A)

```
