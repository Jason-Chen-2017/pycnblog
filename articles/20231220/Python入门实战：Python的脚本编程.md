# 4.具体代码实例和详细解释说明

## 4.1 代码实例1：求两个整数的和

```python
# 定义两个整数
a = 10
b = 20

# 求两个整数的和
c = a + b

# 打印和
print(c)
```

解释说明：

这个代码实例中，我们首先定义了两个整数a和b。然后，我们使用加法运算符+来求两个整数的和，并将和存储在变量c中。最后，我们使用print函数来打印和。

## 4.2 代码实例2：求两个字符串的连接

```python
# 定义两个字符串
str1 = "Hello"
str2 = "World"

# 求两个字符串的连接
result = str1 + " " + str2

# 打印连接
print(result)
```

解释说明：

这个代码实例中，我们首先定义了两个字符串str1和str2。然后，我们使用连接运算符+来求两个字符串的连接，并将连接结果存储在变量result中。最后，我们使用print函数来打印连接结果。
