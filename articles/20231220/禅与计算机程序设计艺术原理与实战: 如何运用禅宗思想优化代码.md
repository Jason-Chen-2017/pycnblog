## 4.1 排序算法实例

我们将通过一个简单的插入排序算法来展示排序算法的实现过程。

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

在这个实例中，我们首先遍历数组中的每个元素，然后将其与前面的元素进行比较。如果当前元素小于前面的元素，则将前面的元素向后移动，使得当前元素可以插入到正确的位置。这个过程会一直持续到所有元素都被排序为递增顺序。

## 4.2 搜索算法实例

我们将通过一个简单的二分搜索算法来展示搜索算法的实现过程。

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

在这个实例中，我们首先将数组的左右边界定义为`left`和`right`。然后，我们通过计算中间元素的索引`mid`，将数组划分为两个部分。如果中间元素等于目标元素，则返回其索引。如果中间元素小于目标元素，则将左边界更新为`mid + 1`。如果中间元素大于目标元素，则将右边界更新为`mid - 1`。这个过程会一直持续到找到目标元素或者边界重合。
