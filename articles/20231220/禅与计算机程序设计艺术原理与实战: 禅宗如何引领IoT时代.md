```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

# 测试代码

print(bubble_sort(arr))
```

通过对代码的专注，我们可以确保代码的质量和效率。例如，在编写代码时，我们需要专注于代码的结构和逻辑，以确保代码的可读性和可维护性。
