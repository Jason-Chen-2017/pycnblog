## 4.1 计算机视觉
```python
import cv2
import numpy as np

# 读取图像

# 转换为HSV颜色空间
hsv = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)

# 设置颜色范围
lower_color = np.array([30, 150, 50])
upper_color = np.array([255, 255, 180])

# 创建颜色掩膜
mask = cv2.inRange(hsv, lower_color, upper_color)

# 对原图像进行颜色掩膜操作
result = cv2.bitwise_and(image, image, mask=mask)

# 显示结果
cv2.imshow('result', result)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
在这个代码实例中，我们使用OpenCV库来实现图像的颜色掩膜操作。首先，我们读取一张图像，然后将其转换为HSV颜色空间。接着，我们设置了颜色范围，并创建了颜色掩膜。最后，我们对原图像进行颜色掩膜操作，并显示结果。

## 4.2 机器学习
```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC

# 加载数据集
iris = load_iris()
X = iris.data
y = iris.target

# 数据预处理
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 训练测试数据集
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# 构建模型
model = SVC(kernel='linear')

# 训练模型
model.fit(X_train, y_train)

# 评估模型
accuracy = model.score(X_test, y_test)
print('Accuracy:', accuracy)
```
在这个代码实例中，我们使用Scikit-learn库来实现鸢尾花数据集的分类。首先，我们加载数据集，然后对数据进行预处理。接着，我们将数据分为训练集和测试集，并构建一个线性SVM模型。最后，我们训练模型并评估其性能。

## 4.3 路径规划
```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建图
G = nx.Graph()

# 添加节点
G.add_node('A')
G.add_node('B')
G.add_node('C')

# 添加边
G.add_edge('A', 'B', weight=1)
G.add_edge('B', 'C', weight=2)
G.add_edge('C', 'A', weight=3)

# 寻找最短路径
shortest_path = nx.shortest_path(G, 'A', 'C', weight='weight')

# 绘制图
nx.draw(G, with_labels=True, node_color='lightblue', edge_color='gray')
plt.plot([shortest_path[0], shortest_path[-1]], color='red', linewidth=2)
plt.show()
```
在这个代码实例中，我们使用NetworkX库来实现一些节点和边的路径规划。首先，我们创建一个图，然后添加节点和边。接着，我们使用Dijkstra算法寻找最短路径。最后，我们绘制图并显示最短路径。
