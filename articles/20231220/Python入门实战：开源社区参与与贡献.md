# 4.具体代码实例和详细解释说明
# 4.1 代码实例
以下是一个简单的Python代码实例，它实现了一个简单的排序算法：

```
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

# 4.2 详细解释说明
这个代码实例实现了一个冒泡排序算法，它是一种简单的比较排序算法。它的基本思想是通过多次遍历数组，将每次最大的元素移动到数组的末尾，直到数组排序为止。
