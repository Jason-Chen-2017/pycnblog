# 4.具体代码实例和详细解释说明

以下是一个使用Tkinter库编写的简单Python图形用户界面示例：

```python
import tkinter as tk

def on_button_click():
    print("按钮被点击了")

app = tk.Tk()
app.title("简单GUI示例")

button = tk.Button(app, text="点击我")
button.pack()

button.config(command=on_button_click)

app.mainloop()
```

这个示例中，我们首先导入了Tkinter库，然后创建了一个主窗口实例`app`，设置了窗口的标题。接着，我们创建了一个按钮组件，使用`pack`方法将按钮添加到窗口中。最后，我们使用`config`方法将按钮的点击事件连接到`on_button_click`函数。当用户点击按钮时，`on_button_click`函数会被调用，打印出"按钮被点击了"。
