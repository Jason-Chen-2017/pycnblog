# 4.具体代码实例和详细解释说明

## 4.1 数据收集代码实例

以下是一个使用Python编写的温度传感器数据收集代码实例：

```python
import time
import Adafruit_ADS1x15  # 导入Adafruit_ADS1x15库

# 初始化ADS1115
ads = Adafruit_ADS1x15.ADS1115()

# 设置温度传感器的输入通道
channel = 0

# 循环收集温度数据
while True:
    # 读取温度数据
    temperature = ads.read_temperature(channel)
    
    # 打印温度数据
    print("Temperature: {:.2f}C".format(temperature))
    
    # 等待一秒
    time.sleep(1)
```

```python
import time

# 设置温度阈值
temperature_threshold = 25

# 假设这是处理后的温度数据
temperature_data_processed = [0.99, 1.00, 1.01, 1.02, 1.03, 1.04, 1.05, 1.06, 1.07, 1.08, 1.09]

        print("Turn on the air conditioner")
    else:
        print("Turn off the air conditioner")

    time.sleep(5)
```
