- 无论何时、何地：禅思维要求我们在任何时候和任何地方都能保持清醒和集中的注意力。

## 4.1 代码实例：求两个整数的最大公约数

在本节中，我们将介绍如何使用禅思维来设计一个求两个整数的最大公约数的算法和代码实现。

### 4.1.1 算法设计

1. 问题分解：将问题分解为求两个整数的最大公约数的子问题。
2. 算法选择：根据问题的特点，选择合适的算法来解决问题。在这个例子中，我们可以选择辗转相除法来解决问题。
3. 算法实现：根据算法的原理，编写代码来实现算法。
4. 算法优化：根据算法的性能指标，优化算法以提高性能。

### 4.1.2 代码实现

```python
def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b)
```

### 4.1.3 详细解释说明

在这个例子中，我们使用了辗转相除法来求两个整数的最大公约数。辗转相除法的原理是：如果两个整数的最大公约数是x，那么它们的最小公倍数一定是x*y，其中y是两个整数的最小公倍数。因此，我们可以通过不断地将两个整数的最小公倍数除以它们的最大公约数来求出它们的最大公约数。

在代码实现中，我们使用了递归来实现辗转相除法。如果b等于0，则返回a；否则，返回b和a%b的最大公约数。通过这种方式，我们可以不断地将两个整数的最小公倍数除以它们的最大公约数，直到得到最大公约数为止。
