# 2.核心概念与联系

## 2.1 禅思维的核心概念

禅思维的核心概念包括：

- 直接体验（直指人）：禅思维强调直接体验现实世界，而不是依赖于理论和抽象思维。
- 观察：禅思维强调观察现实世界，以便更好地理解问题。
- 无思议：禅思维强调放下思议，直接体验现实。

## 2.2 禅思维与游戏开发的联系
