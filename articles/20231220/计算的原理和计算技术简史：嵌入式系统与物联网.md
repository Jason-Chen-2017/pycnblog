$$
A \wedge B = 0 \\
A \vee B = 1 \\
\neg A = \overline{A}
$$

## 3.2 算法
算法是一种用于解决特定问题的步骤。算法通常由一系列操作组成，这些操作在特定的输入条件下执行。算法的基本原理可以通过以下公式表示：

$$
\text{输入} \rightarrow \text{操作1} \rightarrow \text{操作2} \rightarrow \ldots \rightarrow \text{输出}
$$

## 4.2 算法代码实例
以下是一个简单的数据压缩算法的代码实例：

```c
#include <stdio.h>
#include <string.h>

int compress(char *input, char *output) {
    int count = 0;
    char prev = input[0];
    int i;
    for (i = 1; input[i] != '\0'; i++) {
        if (input[i] != prev) {
            output[count++] = input[i];
            prev = input[i];
        }
    }
    output[count] = '\0';
    return count;
}

int main() {
    char input[] = "aaabbbccc";
    char output[strlen(input) + 1];
    int compressed_length = compress(input, output);
    printf("Compressed string: %s\n", output);
    printf("Compressed length: %d\n", compressed_length);
    return 0;
}
```
