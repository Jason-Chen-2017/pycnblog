# 2.核心概念与联系

## 2.1 区块链

区块链是一种基于分布式账本技术的数据结构，它能够实现去中心化、安全性和不可篡改的数字货币交易。区块链的核心组成部分包括交易、区块和链。交易是区块链中的基本操作单位，区块是一组交易的有序集合，链是区块之间的有序连接关系。

## 2.2 去中心化

去中心化是区块链技术的核心特征之一。在传统的金融系统中，中心化机构如银行、支付公司等负责管理和处理交易。而在区块链系统中，去中心化的设计使得各个节点直接之间进行交易，不需要依赖于中心化机构。这使得区块链系统更加安全、透明和去中心化。

## 2.3 加密技术

加密技术是区块链技术的基础。在区块链系统中，所有的交易都会通过加密算法进行加密，以确保交易的安全性。此外，加密技术还用于创建和管理数字货币的供应，以确保其价值稳定性。

## 2.4 分布式共识

分布式共识是区块链技术的核心原理之一。在区块链系统中，各个节点通过分布式共识算法达成一致，确保交易的有效性和一致性。这种共识机制使得区块链系统能够实现去中心化、安全性和不可篡改的数字货币交易。

```python
import hashlib
import time

class Block:
    def __init__(self, index, transactions, timestamp, previous_hash):
        self.index = index
        self.transactions = transactions
        self.timestamp = timestamp
        self.previous_hash = previous_hash
        self.hash = self.calculate_hash()

    def calculate_hash(self):
        block_string = f"{self.index}{self.transactions}{self.timestamp}{self.previous_hash}"
        return hashlib.sha256(block_string.encode()).hexdigest()

class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(0, [], time.time(), "0")

    def add_block(self, transactions):
        index = len(self.chain)
        previous_block = self.chain[index - 1]
        new_block = Block(index, transactions, time.time(), previous_block.hash)
        self.chain.append(new_block)

    def is_valid(self):
        for i in range(1, len(self.chain)):
            current = self.chain[i]
            previous = self.chain[i - 1]

            if current.hash != current.calculate_hash():
                return False

            if current.previous_hash != previous.hash:
                return False

        return True
```

接着，我们定义了一个`Blockchain`类，用于表示区块链的结构。`Blockchain`类包括一个区块链列表`chain`，用于存储区块。我们定义了一个`create_genesis_block`方法，用于创建区块链的第一个区块，即基本区块。

最后，我们定义了一个`add_block`方法，用于添加新的区块到区块链中。同时，我们定义了一个`is_valid`方法，用于检查区块链的有效性。
