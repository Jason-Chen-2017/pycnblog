禅思维，也称为禅宗思想，是一种来自中国的哲学思想。它强调直接体验和直觉，而不是依赖于理论和逻辑。禅思维的核心观念是“一心一意”，即将全身心投入到当前的任务中，全身心地专注于目标，而不分散注意力。这种专注的方式可以帮助程序员更有效地解决问题，提高编程效率。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 4.2 代码解释

这个冒泡排序算法的主要思想是通过多次遍历数组，将较大的元素向后移动，直到整个数组被排序为止。具体来说，它包括以下几个步骤：

1. 遍历数组，从第一个元素开始。
2. 比较当前元素和下一个元素的值。
3. 如果当前元素大于下一个元素，则交换它们的值。
4. 重复上述步骤，直到整个数组被排序为止。
