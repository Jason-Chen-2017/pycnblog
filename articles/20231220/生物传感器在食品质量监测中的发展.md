# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的Python代码实例来演示生物传感器在食品质量监测中的应用。

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LogisticRegression

# 生成示例数据
X = np.random.rand(100, 5)
y = (X[:, 2] > 0.5).astype(np.int)

# 训练逻辑回归模型
model = LogisticRegression()
model.fit(X, y)

# 预测
y_pred = model.predict(X)

# 绘制ROC曲线
plt.plot(y_pred, y)
plt.xlabel('预测值')
plt.ylabel('真实值')
plt.show()
```

上述代码首先导入了必要的库，然后生成了示例数据。接着，使用逻辑回归模型对数据进行训练，并进行预测。最后，绘制ROC曲线以评估模型的性能。
