# 2.核心概念与联系

## 2.1皮尔森距离

皮尔森距离（Pearson Correlation）是一种度量两个随机变量之间相关性的量，它可以用来衡量数据之间的线性关系。皮尔森距离的计算公式如下：

$$
r = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2}\sqrt{\sum_{i=1}^{n}(y_i - \bar{y})^2}}
$$

其中，$r$ 表示皮尔森距离的值，$x_i$ 和 $y_i$ 分别表示两个随机变量的取值，$\bar{x}$ 和 $\bar{y}$ 分别表示这两个随机变量的均值。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来展示如何使用皮尔森距离来处理高维数据和减少数据噪声。

```python
import numpy as np
from scipy.stats import pearsonr

# 生成高维数据
data = np.random.rand(100, 10)

# 计算皮尔森距离
corr, _ = pearsonr(data.ravel(), data.ravel())

print("皮尔森距离:", corr)
```

在这个代码实例中，我们首先导入了 `numpy` 和 `scipy.stats` 库。然后，我们生成了一个高维数据矩阵 `data`，其中包含了100个样本和10个特征。接着，我们使用 `pearsonr` 函数来计算皮尔森距离，并将其打印出来。
