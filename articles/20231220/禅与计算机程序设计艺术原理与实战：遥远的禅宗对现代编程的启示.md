## 3.1 深度优先搜索（DFS）

深度优先搜索（DFS）是一种常用的图算法，它的核心思想是“先搜索当前节点的所有子节点，然后搜索其他节点”。这种思想与禅宗的“无思无虑，只是这一刻”一致。当我们使用DFS时，我们需要关注当前节点，而不是过分关注其他节点。

DFS的具体操作步骤如下：

1. 从起始节点开始，将其标记为已访问。
2. 从当前节点选择一个未访问的邻居节点，将其标记为当前节点。
3. 重复步骤2，直到所有可能的邻居节点都被访问过或者没有未访问的邻居节点可以被选择。
4. 将当前节点标记为已访问，返回到步骤1。

DFS的数学模型公式为：

$$
T(n) = O(n^2)
$$

其中，$T(n)$ 表示DFS的时间复杂度，$n$ 表示图的节点数。

## 3.2 广度优先搜索（BFS）

广度优先搜索（BFS）是一种常用的图算法，它的核心思想是“先搜索当前节点的所有子节点，然后搜索其他节点”。这种思想与禅宗的“一切皆空”一致。当我们使用BFS时，我们需要关注问题的本质，而不是过分关注某种特定的实现方式。

BFS的具体操作步骤如下：

1. 从起始节点开始，将其标记为已访问。
2. 将当前节点的所有未访问的邻居节点加入队列。
3. 从队列中弹出一个节点，将其标记为当前节点。
4. 重复步骤2，直到队列为空或者没有未访问的邻居节点可以被选择。
5. 将当前节点标记为已访问，返回到步骤1。

BFS的数学模型公式为：

$$
T(n) = O(n + e)
$$

其中，$T(n)$ 表示BFS的时间复杂度，$n$ 表示图的节点数，$e$ 表示图的边数。

## 4.1 DFS实例

```python
from collections import deque

def dfs(graph, start):
    visited = set()
    stack = deque([start])

    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)

    return visited
```

上述代码实现了一个基本的DFS算法，它接受一个图和一个起始节点作为输入，并返回一个包含所有访问过的节点的集合。在这个例子中，我们使用了一个有向无环图（DAG）作为输入，并使用了一个邻接表表示法来表示图。

## 4.2 BFS实例

```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])

    while queue:
        vertex = queue.popleft()
        if vertex not in visited:
            visited.add(vertex)
            queue.extend(graph[vertex] - visited)

    return visited
```

上述代码实现了一个基本的BFS算法，它接受一个图和一个起始节点作为输入，并返回一个包含所有访问过的节点的集合。在这个例子中，我们使用了一个有向无环图（DAG）作为输入，并使用了一个邻接表表示法来表示图。
