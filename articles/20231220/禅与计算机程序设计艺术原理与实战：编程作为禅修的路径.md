## 3.1 排序算法

排序算法是一种用于对数据进行排序的方法。常见的排序算法有：

1. **冒泡排序**：冒泡排序是一种简单的排序算法，通过多次比较和交换元素来实现排序。冒泡排序的时间复杂度为O(n^2)。

2. **选择排序**：选择排序是一种简单的排序算法，通过多次选择最小（或最大）元素来实现排序。选择排序的时间复杂度为O(n^2)。

3. **插入排序**：插入排序是一种简单的排序算法，通过多次将元素插入到有序列表中来实现排序。插入排序的时间复杂度为O(n^2)。

4. **归并排序**：归并排序是一种高效的排序算法，通过将数组分割成两个部分，然后递归地排序每个部分，最后合并为一个有序数组来实现排序。归并排序的时间复杂度为O(n*log(n))。

5. **快速排序**：快速排序是一种高效的排序算法，通过选择一个基准元素，将数组分割成两个部分，然后递归地排序每个部分，最后合并为一个有序数组来实现排序。快速排序的时间复杂度为O(n*log(n))。

## 3.2 搜索算法

搜索算法是一种用于在数据结构中查找特定元素的方法。常见的搜索算法有：

1. **线性搜索**：线性搜索是一种简单的搜索算法，通过遍历数据结构中的每个元素来查找特定元素。线性搜索的时间复杂度为O(n)。

2. **二分搜索**：二分搜索是一种高效的搜索算法，通过将数组分割成两个部分，然后递归地查找特定元素的位置，最后返回结果。二分搜索的时间复杂度为O(log(n))。

## 3.3 数学模型公式

数学模型公式是用于描述算法和数据结构的方法。常见的数学模型公式有：

1. **时间复杂度**：时间复杂度是用于描述算法执行时间的方法。时间复杂度通常用大O符号表示，例如O(n)、O(n^2)、O(log(n))等。

2. **空间复杂度**：空间复杂度是用于描述算法所需要的额外空间的方法。空间复杂度通常用大O符号表示，例如O(n)、O(n^2)、O(log(n))等。

3. **秩**：秩是用于描述数据结构中元素的位置的方法。例如，在数组中，第一个元素的秩为1，第二个元素的秩为2，以此类推。

4. **递归关系**：递归关系是用于描述递归算法的方法。递归关系通常用方程式表示，例如F(n) = F(n-1) + n，其中F(n)是n个元素的排序所需的比较次数。

## 4.1 排序算法实例

### 4.1.1 冒泡排序

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 4.1.2 选择排序

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

### 4.1.3 插入排序

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i-1
        while j >= 0 and key < arr[j]:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr
```

### 4.1.4 归并排序

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]
    left = merge_sort(left)
    right = merge_sort(right)
    return merge(left, right)

def merge(left, right):
    result = []
    while left and right:
        if left[0] < right[0]:
            result.append(left.pop(0))
        else:
            result.append(right.pop(0))
    result.extend(left)
    result.extend(right)
    return result
```

### 4.1.5 快速排序

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

### 4.2.1 线性搜索

```python
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1
```

### 4.2.2 二分搜索

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```
