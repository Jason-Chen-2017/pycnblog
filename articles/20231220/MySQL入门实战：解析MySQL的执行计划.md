```sql

```

这个查询将返回名字和年龄大于18岁的用户，并按名字升序排序。执行计划如下：

1. 根据 WHERE 子句选择年龄大于18岁的用户。

## 4.2连接（JOIN）实例


1. 根据 WHERE 子句筛选满足条件的用户。
2. 根据 ON 子句连接用户和订单表。
3. 根据 ORDER BY 子句对结果集进行排序。

```sql
SELECT name, age FROM users WHERE age > 18 ORDER BY name ASC;
```

可以使用以下命令来解析执行计划：


执行上述命令后，MySQL将返回一个详细的执行计划，包括表的类型、possible_keys、key、key_len、ref、rows_examined、rows_produced和filter_condition等信息。
