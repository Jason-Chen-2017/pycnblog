假设我们需要编写一个程序，用于计算两个整数的和、差、积和商。我们可以通过以下步骤来实现这个程序：

1. 定义两个整数的变量，并赋值。
2. 计算两个整数的和。
3. 计算两个整数的差。
4. 计算两个整数的积。
5. 计算两个整数的商。
6. 输出计算结果。

以下是代码实例：

```python
# 定义两个整数的变量，并赋值
a = 10
b = 5

# 计算两个整数的和

# 计算两个整数的差

# 计算两个整数的积
product = a * b

# 计算两个整数的商
quotient = a / b

# 输出计算结果
print("和: ", sum)
print("差: ", difference)
print("积: ", product)
print("商: ", quotient)
```
