                 

# 1.背景介绍

图像超分辨率是一种利用深度学习技术提高图像质量的方法，它通过将低分辨率图像（LR）转换为高分辨率图像（HR），从而提高图像的细节和清晰度。这项技术在多个领域都有广泛的应用，例如视频编码、医疗诊断、无人驾驶等。在过去的几年里，深度学习在图像超分辨率任务中取得了显著的进展，尤其是在2016年左右，随着Convolutional Neural Networks（CNN）和Generative Adversarial Networks（GAN）等深度学习模型的应用，图像超分辨率技术的性能得到了显著提升。
