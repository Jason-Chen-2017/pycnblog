# 2.核心概念与联系
禅学的核心概念包括：

1. 直接体验（直指人心）：禅学强调直接体验事物本质，而不是通过理论和概念来理解。
2. 无思议（无量伏）：禅学认为，思维和情感是障碍，需要通过冥想来超越。
3. 无动静（无生）：禅学认为，动态和静态都是对事物的误解，事物本身是无动静的。
4. 无分别（无我）：禅学认为，我和世界之间的界限是虚构的，需要消除。
