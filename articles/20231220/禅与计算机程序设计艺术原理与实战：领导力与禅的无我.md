## 3.1 排序算法

排序算法是计算机程序设计中最基本的算法之一。我们将介绍两种常见的排序算法：冒泡排序和快速排序。

### 3.1.1 冒泡排序

冒泡排序是一种简单的排序算法，它的基本思想是通过多次交换相邻的元素，将较大的元素逐渐向后移动，而较小的元素向前移动。

具体的操作步骤如下：

1. 从第一个元素开始，与后面的每个元素进行比较。
2. 如果当前元素大于后面的元素，则交换它们的位置。
3. 重复上述步骤，直到整个数组被排序。

冒泡排序的时间复杂度为O(n^2)，其中n是数组的长度。

### 3.1.2 快速排序

快速排序是一种更高效的排序算法，它的基本思想是通过选择一个基准元素，将其他元素分为两部分：一个比基准元素小的部分，一个比基准元素大的部分。然后递归地对这两个部分进行排序。

具体的操作步骤如下：

1. 选择一个基准元素。
2. 将其他元素分为两部分：一个比基准元素小的部分，一个比基准元素大的部分。
3. 递归地对这两个部分进行排序。

快速排序的平均时间复杂度为O(nlogn)，其中n是数组的长度。

## 3.2 搜索算法

搜索算法是计算机程序设计中另一个基本的算法之一。我们将介绍两种常见的搜索算法：深度优先搜索和广度优先搜索。

### 3.2.1 深度优先搜索

深度优先搜索是一种搜索算法，它的基本思想是在当前节点上展开，并尽可能深入到子节点。当到达叶子节点时，才回溯到父节点，并继续搜索其他子节点。

具体的操作步骤如下：

1. 从根节点开始，将其标记为已访问。
2. 选择当前节点的一个子节点，将其标记为当前节点。
3. 如果当前节点是叶子节点，则回溯到父节点并继续搜索其他子节点。
4. 重复上述步骤，直到所有节点都被访问。

### 3.2.2 广度优先搜索

广度优先搜索是一种搜索算法，它的基本思想是在当前节点上展开，并尽可能广泛地搜索子节点。当所有子节点都被访问后，才继续搜索父节点的其他子节点。

具体的操作步骤如下：

1. 从根节点开始，将其标记为已访问。
2. 将当前节点的所有未被访问的子节点加入队列。
3. 从队列中取出一个节点，将其标记为当前节点。
4. 如果当前节点是叶子节点，则结束搜索。
5. 将当前节点的所有未被访问的子节点加入队列。
6. 重复上述步骤，直到所有节点都被访问。

## 4.1 冒泡排序实例

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

这个函数的工作原理是通过多次交换相邻的元素，将较大的元素逐渐向后移动，而较小的元素向前移动。具体来说，它首先获取数组的长度，然后进行多次循环。在每次循环中，它会将当前元素与后面的元素进行比较，如果当前元素大于后面的元素，则交换它们的位置。这个过程会一直持续到整个数组被排序为非递减顺序。

## 4.2 快速排序实例

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

这个函数的工作原理是通过选择一个基准元素，将其他元素分为两部分：一个比基准元素小的部分，一个比基准元素大的部分。然后递归地对这两个部分进行排序。具体来说，它首先检查数组的长度，如果长度为1或0，则直接返回数组。然后选择一个基准元素（这里选择了数组的中间元素），将其他元素分为两部分：一个比基准元素小的部分，一个比基准元素大的部分。最后，它会递归地对这两个部分进行排序，并将结果拼接在一起。
