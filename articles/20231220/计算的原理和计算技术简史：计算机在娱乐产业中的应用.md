## 1. Huffman 压缩算法实例

Huffman 压缩算法是一种基于哈夫曼编码的压缩算法，它可以根据字符的频率构建一个优先级树，从而实现数据的压缩。以下是一个简单的 Huffman 压缩算法实例：

```python
import heapq

def huffman_encode(data):
    # 统计字符频率
    frequency = {}
    for char in data:
        frequency[char] = frequency.get(char, 0) + 1
    # 构建优先级队列
    priority_queue = [[weight, [symbol, ""]] for symbol, weight in frequency.items()]
    heapq.heapify(priority_queue)
    # 构建哈夫曼树
    while len(priority_queue) > 1:
        lo = heapq.heappop(priority_queue)
        hi = heapq.heappop(priority_queue)
        for pair in lo[1:]:
            pair[1] = '0' + pair[1]
        for pair in hi[1:]:
            pair[1] = '1' + pair[1]
        heapq.heappush(priority_queue, [lo[0] + hi[0]] + lo[1:] + hi[1:])
    # 获取哈夫曼编码
    huffman_code = sorted(priority_queue[0][1], key=lambda p: (len(p[-1]), p))
    return dict(huffman_code)

data = "this is an example of huffman encoding"
huffman_code = huffman_encode(data)
print(huffman_code)
```

## 2. 快速傅里叶变换（FFT）实例

快速傅里叶变换（FFT）是一种计算傅里叶变换的算法，它可以将时域信号转换为频域信号。以下是一个简单的 FFT 实例：

```python
import numpy as np
from scipy.fft import fft

def fft_example():
    # 生成一组随机信号
    signal = np.random.rand(1024)
    # 计算 FFT
    fft_result = fft(signal)
    # 获取频域信号
    freq = np.fft.fftfreq(len(signal))
    # 绘制频域信号
    plt.plot(freq, np.abs(fft_result))
    plt.xlabel('Frequency (Hz)')
    plt.ylabel('Amplitude')
    plt.title('FFT Example')
    plt.show()

fft_example()
```
