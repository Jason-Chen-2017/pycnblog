## 3.1 新闻文本摘要
新闻文本摘要是将长文本转换为短文本的过程，主要应用于新闻网站、搜索引擎等。深度学习中常用的新闻文本摘要算法有：

- **抽取式摘要**：利用TF-IDF、Word2Vec等技术提取文本中的关键词，构建摘要。
- **生成式摘要**：利用序列生成模型（如RNN、LSTM、GRU）生成摘要。

### 3.1.1 抽取式摘要
抽取式摘要主要包括以下步骤：

1. 文本预处理：包括去除停用词、标点符号、数字等，以及词汇切分、词性标注、词嵌入等。
2. 关键词提取：使用TF-IDF、Word2Vec等技术，计算文本中每个词的重要性，选取TOP-N个词构成摘要。

### 3.1.2 生成式摘要
生成式摘要主要包括以下步骤：

1. 文本预处理：同抽取式摘要。
2. 模型训练：使用RNN、LSTM、GRU等序列生成模型，训练在大量新闻文本上，学习文本特征。
3. 摘要生成：输入原文本，使用训练好的模型生成摘要。

## 3.2 新闻推荐
新闻推荐是根据用户历史行为、兴趣等信息，为用户推荐相关新闻的过程。深度学习中常用的新闻推荐算法有：

- **协同过滤**：根据用户行为数据计算用户之间的相似度，推荐用户可能喜欢的新闻。
- **内容过滤**：利用神经网络模型（如MDN、AutoInt），根据新闻内容与用户兴趣相似度，推荐新闻。

### 3.2.1 协同过滤
协同过滤主要包括以下步骤：

1. 用户行为数据收集：记录用户阅读、点赞、评论等操作。
2. 用户相似度计算：使用Pearson相关、欧氏距离等方法，计算用户之间的相似度。
3. 新闻推荐：根据用户历史行为和相似用户的行为数据，推荐相似的新闻。

### 3.2.2 内容过滤
内容过滤主要包括以下步骤：

1. 文本预处理：同抽取式摘要。
2. 模型训练：使用MDN、AutoInt等神经网络模型，训练在大量新闻文本和用户兴趣数据上，学习文本特征和用户兴趣。
3. 新闻推荐：输入新闻文本，使用训练好的模型计算新闻与用户兴趣的相似度，推荐相似的新闻。

## 4.1 新闻文本摘要
### 4.1.1 抽取式摘要
```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 文本列表
texts = ["这是一个长文本，用于生成新闻摘要。", "这是另一个长文本，也用于生成新闻摘要。"]

# 文本预处理
vectorizer = TfidfVectorizer(stop_words='english', max_df=0.5)
X = vectorizer.fit_transform(texts)

# 关键词提取
similarity = cosine_similarity(X, X)
top_words = vectorizer.get_feature_names_out()

# 构建摘要
summary = ""
for i in range(len(similarity[0])):
    if similarity[0][0][i] > 0.5:
        summary += top_words[i] + " "
print(summary)
```
### 4.1.2 生成式摘要
```python
import numpy as np
import tensorflow as tf
from tensorflow.keras.preprocessing.sequence import pad_sequences
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Embedding, LSTM, Dense

# 文本列表
texts = ["这是一个长文本，用于生成新闻摘要。", "这是另一个长文本，也用于生成新闻摘要。"]

# 文本预处理
tokenizer = tf.keras.preprocessing.text.Tokenizer()
tokenizer.fit_on_texts(texts)
sequences = tokenizer.texts_to_sequences(texts)
vocab_size = len(tokenizer.word_index) + 1
embedding_dim = 100
max_length = max(len(sequence) for sequence in sequences)

# 模型构建
model = Sequential()
model.add(Embedding(vocab_size, embedding_dim, input_length=max_length))
model.add(LSTM(128, return_sequences=True))
model.add(LSTM(128))
model.add(Dense(vocab_size, activation='softmax'))

# 模型训练
model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])
model.fit(sequences, np.array([0, 1]), epochs=100, verbose=0)

# 摘要生成
input_text = "这是一个新闻摘要生成的测试文本。"
input_sequence = tokenizer.texts_to_sequences([input_text])[0]
input_sequence = pad_sequences([input_sequence], maxlen=max_length)
input_sequence = np.array(input_sequence)
input_sequence = np.reshape(input_sequence, (1, max_length, vocab_size))
predictions = model.predict(input_sequence, verbose=0)
predicted_word_index = np.argmax(predictions, axis=-1)

summary = ""
for i in range(max_length):
    predicted_word_index[i] = np.argmax(predictions[0][i])
    summary += tokenizer.index_word[predicted_word_index[i]] + " "
print(summary)
```

## 4.2 新闻推荐
### 4.2.1 协同过滤
```python
from scipy.spatial.distance import cosine

# 用户行为数据
user_behavior = {
    "user1": ["news1", "news2", "news3"],
    "user2": ["news4", "news5", "news6"],
    "user3": ["news7", "news8", "news9"]
}

# 用户相似度计算
user_similarity = {}
for user1 in user_behavior:
    for user2 in user_behavior:
        if user1 != user2:
            intersection = len(set(user_behavior[user1]).intersection(set(user_behavior[user2])))
            union = len(set(user_behavior[user1]).union(set(user_behavior[user2])))
            similarity = intersection / union if union != 0 else 0
            user_similarity[(user1, user2)] = similarity

# 新闻推荐
user1_news = user_behavior["user1"]
user2_news = user_behavior["user2"]
similar_news = [news for news in user2_news if news not in user1_news]
similar_news.sort(key=lambda x: user_similarity[("user2", x)], reverse=True)
print(similar_news)
```
