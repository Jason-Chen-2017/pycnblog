# 4.具体代码实例和详细解释说明

首先，我们来看一个简单的问题：求两个整数的最大公约数。这个问题可以使用辗转相除法来解决。通过观察，我们可以发现辗转相除法的核心思想是：如果两个整数的最大公约数不是它们之间的最大公约数，那么它们之间的最大公约数必定是它们的差的最大公约数。

通过领悟，我们可以得出以下算法：

1. 如果两个整数相等，那么它们的最大公约数就是它们本身。
2. 如果两个整数不相等，那么我们可以将较大的整数减去较小的整数，得到一个新的整数。
3. 然后，我们可以将这个新整数与原始整数中较小的整数相比较，如果它们相等，那么它们就是最大公约数，如果它们不相等，那么我们可以继续执行步骤2和步骤3，直到它们相等为止。

通过实践，我们可以得到以下代码实例：

```python
def gcd(a, b):
    while a != b:
        if a > b:
            a = a - b
        else:
            b = b - a
    return a
```

这个代码实例使用了禅宗的观察、领悟和实践思想来解决问题。通过观察，我们可以发现辗转相除法的核心思想。通过领悟，我们可以得出算法的核心步骤。通过实践，我们可以实现这个算法，并确保它能够解决问题。
