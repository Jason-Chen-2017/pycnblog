# 2.核心概念与联系

## 2.1随机模型

### 2.1.1随机变量

随机变量是一个事件的特征值，可以用来描述事件的不确定性。随机变量可以分为离散型和连续型两种类型。离散型随机变量只能取有限个或无限个但可数个值，而连续型随机变量可以取无限个连续的值。

### 2.1.2事件

事件是一个随机过程中可能发生的结果，可以用来描述随机过程的发生情况。事件可以分为独立事件和相依事件两种类型。独立事件之间发生的结果不会影响彼此，而相依事件之间发生的结果会影响彼此。

### 2.1.3概率分布

概率分布是一个随机变量取值的概率与其取值相对应的值之间的关系。概率分布可以用来描述随机变量的分布情况。常见的概率分布有均匀分布、指数分布、正态分布等。

### 2.1.4期望

期望是一个随机变量的数学期望，表示随机变量的平均值。期望可以用来描述随机变量的中心趋势。期望的计算公式为：

$$
E[X] = \sum_{i=1}^{n} x_i P(x_i)
$$

### 2.1.5方差

方差是一个随机变量的数学方差，表示随机变量的离散程度。方差可以用来描述随机变量的不确定性。方差的计算公式为：

$$
Var[X] = E[X^2] - (E[X])^2
$$

## 2.2蒙特卡洛方法

### 2.2.1基本思想

蒙特卡洛方法是一种基于随机数生成的数值计算方法，它通过生成大量的随机样本来估计某个数值。蒙特卡洛方法的基本思想是：通过随机数生成的样本来近似地求解问题的解。

### 2.2.2核心步骤

1. 建立随机模型：根据问题的特点，建立一个合适的随机模型。
2. 生成随机样本：根据随机模型，生成大量的随机样本。
3. 计算样本平均值：对生成的随机样本进行计算，得到样本平均值。
4. 估计解：通过样本平均值来估计问题的解。

## 2.3核心概念联系

随机模型和蒙特卡洛方法之间存在密切的联系。随机模型提供了问题的数学模型，蒙特卡洛方法则基于随机模型生成随机样本来近似求解问题。因此，随机模型和蒙特卡洛方法是相辅相成的，无法分离。

## 4.1代码实例

在本节中，我们将通过一个简单的代码实例来演示蒙特卡洛方法的具体应用。我们将使用蒙特卡洛方法来估计π的值。

```python
import random

def monte_carlo_pi(n):
    inside_circle = 0
    for _ in range(n):
        x = random.uniform(-1, 1)
        y = random.uniform(-1, 1)
        if x**2 + y**2 <= 1:
            inside_circle += 1
    return (inside_circle / n) * 4

n = 1000000
pi_estimate = monte_carlo_pi(n)
print("π 的估计值为：", pi_estimate)
```

## 4.2详细解释说明

上述代码实例中，我们使用了蒙特卡洛方法来估计π的值。具体来说，我们首先导入了`random`模块，然后定义了一个函数`monte_carlo_pi`，该函数接受一个参数`n`，表示生成的随机样本数。

在函数中，我们首先初始化一个变量`inside_circle`，用于记录在圆内部的点数。然后，我们使用一个for循环来生成`n`个随机点，每个点的x和y坐标均为[-1, 1]之间的随机数。如果这个点在圆内部（即x^2 + y^2 <= 1），则将`inside_circle`变量加1。

最后，我们将`inside_circle`变量除以`n`，并乘以4，得到π的估计值。通过调用`monte_carlo_pi`函数，我们可以得到π的估计值。
