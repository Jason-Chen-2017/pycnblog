2. 递推关系：接下来，我们需要找出子问题之间的递推关系，并根据递推关系来得到子问题的解。

1. 定义目标函数：首先，我们需要定义问题的目标函数，并找出目标函数的最大值或最小值。
2. 选择策略：接下来，我们需要选择一个策略，根据策略来选择能够带来最大收益的选择。

## 4.1 示例：猜数字游戏
猜数字游戏是一种简单的游戏，它的规则是：玩家需要通过猜测来猜出一个随机生成的数字。在这个示例中，我们将使用动态规划算法来实现猜数字游戏的功能。

### 4.1.1 代码实例
```python
import random

def guess_number():
    num = random.randint(1, 100)
    guess = 0
    while True:
        guess = int(input("请猜一个数字："))
        if guess < num:
            print("太小了")
        elif guess > num:
            print("太大了")
        else:
            print("恭喜你，猜对了！")
            break

if __name__ == "__main__":
    guess_number()
```
### 4.1.2 代码解释
在这个示例中，我们使用了动态规划算法来实现猜数字游戏的功能。具体的代码解释如下：

1. 首先，我们导入了random模块，并生成了一个随机的数字。
2. 接下来，我们使用一个while循环来实现猜数字的过程。在循环中，我们使用input函数来获取玩家的猜测，并使用if-elif语句来判断猜测是否正确。
3. 如果猜测太小，我们会提示玩家猜的数字太小；如果猜测太大，我们会提示玩家猜的数字太大；如果猜测正确，我们会提示玩家猜对了，并结束循环。
