# 4.具体代码实例和详细解释说明

在本节中，我们将提供一个简单的Python代码实例，以说明如何进行加密货币交易。

```python
import hashlib
import os
import random
import time

class Wallet:
    def __init__(self, private_key):
        self.private_key = private_key
        self.address = self.generate_address()

    def generate_address(self):
        return hashlib.sha256(os.urandom(32)).hexdigest()

    def sign_transaction(self, transaction, private_key):
        return hashlib.sha256(transaction + private_key).hexdigest()

class Transaction:
    def __init__(self, from_address, to_address, amount):
        self.from_address = from_address
        self.to_address = to_address
        self.amount = amount

    def sign(self, private_key):
        return Wallet.sign_transaction(self, private_key)

    def verify(self, signature, public_key):
        return hashlib.sha256(self + public_key).hexdigest() == signature

wallet1 = Wallet(os.urandom(32))
wallet2 = Wallet(os.urandom(32))

transaction = Transaction(wallet1.address, wallet2.address, 100)
signature = transaction.sign(wallet1.private_key)

if transaction.verify(signature, wallet1.public_key):
    print("Transaction verified successfully.")
else:
    print("Transaction verification failed.")
```

在上述代码中，我们首先定义了一个`Wallet`类，用于存储用户的私钥和地址。然后我们定义了一个`Transaction`类，用于表示一笔交易。最后，我们创建了两个钱包，并创建了一笔交易。我们然后使用私钥对交易进行签名，并使用公钥验证签名。如果验证成功，则打印“交易验证成功”；否则，打印“交易验证失败”。
