## 3.1 动态规划

动态规划是一种常用的算法方法，它可以用来解决一些具有最优子结构的问题。在动态规划中，我们需要找到一个状态转移方程，并通过递归地计算状态来找到最优解。

### 3.1.1 状态转移方程

在动态规划中，我们需要找到一个状态转移方程，以便于计算状态。状态转移方程的形式通常是：

$$
dp[i] = \min_{0 \leq j < i} \{ dp[j] + cost(i, j) \}
$$

其中，$dp[i]$ 表示到达状态 $i$ 的最小代价，$cost(i, j)$ 表示从状态 $j$ 到状态 $i$ 的代价。

### 3.1.2 具体操作步骤

1. 初始化：设置基础状态和基础代价。
2. 状态转移：根据状态转移方程计算每个状态的代价。
3. 求解：找到最优解。

### 3.1.3 禅学思想的应用

在动态规划中，禅学思想的应用主要表现在我们如何看待问题。通过直接体验和感知，我们可以更好地理解问题，并找到更好的解决方案。例如，在求解最长子序列问题时，我们可以通过直接体验和感知来找到最长子序列。

## 3.2 贪心算法

贪心算法是一种常用的算法方法，它通过在每一步中做出最佳的局部决策来找到全局最优解。

### 3.2.1 贪心策略

在贪心算法中，我们需要找到一个贪心策略，以便于在每一步中做出最佳的局部决策。贪心策略的形式通常是：

$$
\arg \max_{x \in X} f(x)
$$

其中，$X$ 是一个有限集，$f(x)$ 是一个函数，表示在状态 $x$ 下的收益。

### 3.2.2 具体操作步骤

1. 初始化：设置基础状态和基础决策。
2. 贪心决策：根据贪心策略进行决策。
3. 求解：找到最优解。

### 3.2.3 禅学思想的应用

在贪心算法中，禅学思想的应用主要表现在我们如何看待问题。通过直接体验和感知，我们可以更好地理解问题，并找到更好的解决方案。例如，在求解 knapsack 问题时，我们可以通过直接体验和感知来找到最佳的装载方案。

## 4.1 动态规划实例

### 4.1.1 最长子序列问题

最长子序列问题是一种典型的动态规划问题。给定一个字符串 $s$，找到一个最长的非空子序列，使得子序列中的所有字符都不相同。

```python
def longest_unique_substtr(s: str) -> int:
    n = len(s)
    dp = [0] * (n + 1)
    for i in range(n):
        dp[i] = 1
        for j in range(i):
            if s[i] != s[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```

在上面的代码中，我们通过直接体验和感知来找到最长子序列。我们首先初始化一个状态数组 `dp`，然后根据状态转移方程计算每个状态的值，最后找到最大值。

### 4.1.2 最小路径和问题

最小路径和问题是一种典型的动态规划问题。给定一个矩阵，从左上角开始，到达右下角的最小路径和。

```python
def min_path_sum(matrix: List[List[int]]) -> int:
    rows, cols = len(matrix), len(matrix[0])
    dp = [[0] * cols for _ in range(rows)]
    dp[0][0] = matrix[0][0]
    for i in range(1, rows):
        dp[i][0] = dp[i - 1][0] + matrix[i][0]
        for j in range(1, cols):
            dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + matrix[i][j]
    return dp[-1][-1]
```

在上面的代码中，我们通过直接体验和感知来找到最小路径和。我们首先初始化一个状态数组 `dp`，然后根据状态转移方程计算每个状态的值，最后找到最小值。

## 4.2 贪心算法实例

### 4.2.1 最大独立子集问题

最大独立子集问题是一种典型的贪心算法问题。给定一个集合 $S$，找到一个最大的独立子集，使得子集中的元素pairwise 不相交。

```python
def max_independent_set(S: Set[int]) -> int:
    S = sorted(S, reverse=True)
    result = []
    for x in S:
        if not any(y & x != 0 for y in result):
            result.append(x)
    return len(result)
```

在上面的代码中，我们通过直接体验和感知来找到最大独立子集。我们首先对集合 $S$ 进行排序，然后逐个检查每个元素是否能够加入结果集，最后返回结果集的大小。
