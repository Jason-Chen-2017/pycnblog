#### 4.2.2 直指人心的思考

2. 记录最大值。

#### 4.2.3 无思无虑的解决问题

我们可以根据以下步骤实现解决方案：

1. 遍历数组。
2. 记录最大值。

以下是一个具体的代码实例：

```python
def find_max_value(nums):
    max_value = nums[0]
    for num in nums:
        if num > max_value:
            max_value = num
    return max_value
```
