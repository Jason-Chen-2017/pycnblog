## 4.2 无思议

无思议的代码实例如下：

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]
    return merge(merge_sort(left), merge_sort(right))

def merge(left, right):
    result = []
    while left and right:
        if left[0] < right[0]:
            result.append(left.pop(0))
        else:
            result.append(right.pop(0))
    result.extend(left)
    result.extend(right)
    return result
```

在这个代码实例中，我们通过无思议来设计了一个简单的归并排序算法。我们可以看到，这个算法的时间复杂度是O(nlogn)，而不是O(n^2)。这就表明，通过无思议，我们可以设计更高效的算法。
