- 确定子问题：将原问题分解成若干个子问题。
- 确定基本状态：将子问题分解成基本状态。
- 确定状态转移方程：将基本状态之间的关系表示为状态转移方程。

### 3.2 贪心算法

贪心算法是一种解决优化问题的方法，它通过在每一步选择最优解来逐步得到最优解。在使用贪心算法时，我们需要注意以下几点：

- 确定优化目标：明确需要最小化或最大化的目标。
- 确定贪心策略：在每一步选择最优解时，使用哪种策略。
- 证明贪心策略的正确性：证明贪心策略能够得到最优解。

### 3.3 分治算法

分治算法是一种解决复杂问题的方法，它通过将问题分解成子问题并解决子问题来得到最终解。在使用分治算法时，我们需要注意以下几点：

- 确定子问题：将原问题分解成若干个子问题。
- 确定合并策略：将子问题的解合并成原问题的解。

### 4.1 动态规划实例

考虑一个经典的背包问题，我们需要将一组物品放入一个有限容量的背包中，使得背包的重量不超过限制，同时最大化价值。我们可以使用动态规划来解决这个问题。

```python
def knapsack(values, weights, capacity):
    n = len(values)
    dp = [[0 for _ in range(capacity + 1)] for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if weights[i - 1] <= w:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - weights[i - 1]] + values[i - 1])
            else:
                dp[i][w] = dp[i - 1][w]
    return dp[n][capacity]

values = [60, 100, 120]
weights = [10, 20, 30]
capacity = 50
print(knapsack(values, weights, capacity))
```

在这个例子中，我们使用了动态规划的四个核心概念：

- 确定子问题：将背包问题分解成若干个子问题，每个子问题包含一个物品和一个背包容量。
- 确定基本状态：将子问题分解成基本状态，每个基本状态包含一个物品和一个背包容量。
- 确定状态转移方程：将基本状态之间的关系表示为状态转移方程，如果物品可以放入背包中，则选择该物品，否则选择不选该物品。
- 确定终止条件：当基本状态为终止状态时，返回对应的值。

### 4.2 贪心算法实例

考虑一个经典的活动选择问题，我们需要选择一组活动，使得活动之间不会相互冲突，同时最大化活动的数量。我们可以使用贪心算法来解决这个问题。

```python
def activity_selection(activities):
    activities.sort(key=lambda x: x[1])
    selected_activities = []
    current_end_time = 0
    for activity in activities:
        if activity[0] >= current_end_time:
            selected_activities.append(activity)
            current_end_time = activity[1]
    return selected_activities

activities = [(1, 4), (2, 5), (3, 6), (5, 7), (8, 9)]
print(activity_selection(activities))
```

在这个例子中，我们使用了贪心算法的三个核心概念：

- 确定优化目标：最大化活动的数量。
- 确定贪心策略：在每一步选择结束时间最早的活动。
- 证明贪心策略的正确性：证明选择结束时间最早的活动能够得到最优解。

```python
def fast_pow(base, exponent):
    if exponent == 0:
        return 1
    if exponent == 1:
        return base
    result = fast_pow(base, exponent // 2)
    if exponent % 2 == 0:
        return result * result
    else:
        return result * result * base

base = 2
exponent = 10
print(fast_pow(base, exponent))
```

在这个例子中，我们使用了分治算法的三个核心概念：

- 确定子问题：将快速幂问题分解成若干个子问题，每个子问题包含一个基础的快速幂问题。
- 确定合并策略：将子问题的解合并成原问题的解，如果指数是偶数，则将子问题的解相乘，否则将子问题的解相乘并与基数相乘。
- 证明正确性：证明分治算法能够得到正确的解。
