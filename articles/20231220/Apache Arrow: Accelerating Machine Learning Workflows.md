```python

import pandas as pd

# 创建一个 Pandas 数据帧
data = {'name': ['Alice', 'Bob', 'Charlie'],
        'age': [25, 30, 35],
        'gender': ['M', 'F', 'M']}
df = pd.DataFrame(data)

# 将 Pandas 数据帧转换为 Apache Arrow 数据帧
arrow_df = df.to_arrow()

# 将 Apache Arrow 数据帧转换为另一个 Pandas 数据帧
new_df = arrow_df.to_pandas()
```

在这个代码实例中，我们首先创建了一个 Pandas 数据帧，然后将其转换为 Apache Arrow 数据帧。接着，我们将 Apache Arrow 数据帧转换回另一个 Pandas 数据帧。这个例子展示了如何使用 Apache Arrow 进行数据处理和传输。
