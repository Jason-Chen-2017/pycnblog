## 4.2 层次结构设计实例

在这个例子中，我们将通过一个简单的文件系统来演示层次结构设计的实现。我们将文件系统划分为以下几个层次：

- 文件系统顶层：包含整个文件系统的根目录。
- 目录：包含文件和子目录的容器。
- 文件：存储数据的容器。

```python
# 文件系统顶层
class FileSystem:
    def __init__(self):
        self.root = Directory()

# 目录
class Directory:
    def __init__(self):
        self.files = []
        self.subdirectories = []

    def add_file(self, file):
        self.files.append(file)

    def add_subdirectory(self, subdirectory):
        self.subdirectories.append(subdirectory)

# 文件
class File:
    def __init__(self, name, data):
        self.name = name
        self.data = data
```

在这个例子中，我们将文件系统划分为三个层次，每个层次负责一个特定的功能。这样，在需求增长时，我们可以轻松地扩展文件系统。
