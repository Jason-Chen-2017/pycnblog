# 2.核心概念与联系

全概率模型（Probabilistic Graphical Models, PGM）是一种用于描述随机变量之间关系的图模型，它具有很强的表达能力和泛化能力，因此在推荐系统中得到了广泛的应用。全概率模型的核心思想是通过构建一个有向无环图（Directed Acyclic Graph, DAG）来表示随机变量之间的关系，然后通过计算条件概率来预测未知变量的值。在推荐系统中，随机变量可以表示用户的行为、物品的特征等，通过构建全概率模型，可以更好地捕捉用户的兴趣和物品的特点，从而提高推荐系统的准确性。
