## 4.1 烧炼过程的代码实例

在这个例子中，我们将介绍一个简单的烧炼过程的代码实例。这个代码实例是一个简单的Python程序，它可以计算烧炼过程中的热量。

```python
import math

def calc_heat(mass, specific_heat, temp_diff):
    return mass * specific_heat * temp_diff

mass = 100  # 质量，单位为克
specific_heat = 0.5  # 热量吸收率，单位为千瓦秒/克度
temp_diff = 1000  # 温度差，单位为摄氏度

heat = calc_heat(mass, specific_heat, temp_diff)
print("烧炼过程中的热量为：", heat, "千瓦秒")
```

在这个代码实例中，我们首先导入了`math`模块，然后定义了一个名为`calc_heat`的函数，这个函数用于计算烧炼过程中的热量。在函数中，我们使用了热量公式`Q = mcΔT`，其中`m`是质量，`c`是热量吸收率，`ΔT`是温度差。然后我们定义了一些变量，如质量、热量吸收率和温度差，并将它们传递给`calc_heat`函数，最后打印出烧炼过程中的热量。
