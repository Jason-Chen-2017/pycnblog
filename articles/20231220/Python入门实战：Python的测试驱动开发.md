## 4.具体代码实例和详细解释说明

### 4.1 代码实例

以下是一个简单的Python代码实例，用于演示Python的测试驱动开发：

```python
# 编写测试用例
import unittest

class TestAddition(unittest.TestCase):

    def test_add(self):
        self.assertEqual(2 + 2, 4)

# 运行测试用例
if __name__ == '__main__':
    unittest.main()
```

### 4.2 详细解释说明

这个代码实例首先导入了Python的unittest模块，然后定义了一个名为TestAddition的测试类，该类继承了unittest.TestCase类。在TestAddition类中，定义了一个名为test_add的测试方法，该方法使用assertEqual方法来验证2 + 2的结果是否等于4。

最后，使用if __name__ == '__main__':语句来运行测试用例。如果这个文件作为主程序运行，则运行unittest.main()方法，否则，直接运行测试用例。
