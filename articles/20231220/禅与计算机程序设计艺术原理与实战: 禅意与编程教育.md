## 4.1 排序算法实例

我们将通过一个简单的冒泡排序实例来解释排序算法的工作原理。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个实例中，我们使用了两层循环来遍历数组中的每个元素，并将相邻的元素进行比较和交换。通过这种方式，我们可以逐渐将较大的元素移动到数组的末尾，最终实现排序。

## 4.2 搜索算法实例

我们将通过一个简单的二分搜索实例来解释搜索算法的工作原理。

```python
def binary_search(arr, target):
    left, right = 0, len(arr)-1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

在这个实例中，我们使用了一个while循环来遍历数组中的每个元素，并将相邻的元素进行比较。通过这种方式，我们可以逐渐将较大的元素移动到数组的末尾，最终实现排序。

## 4.3 图论算法实例

我们将通过一个简单的最短路径算法实例来解释图论算法的工作原理。

```python
import heapq

def dijkstra(graph, start):
    n = len(graph)
    dist = [float('inf')] * n
    dist[start] = 0
    pq = [(0, start)]
    while pq:
        _, u = heapq.heappop(pq)
        for v, w in graph[u]:
            if dist[v] > dist[u] + w:
                dist[v] = dist[u] + w
                heapq.heappush(pq, (dist[v], v))
    return dist
```

在这个实例中，我们使用了一个heapq库来实现一个优先级队列，并使用了一个while循环来遍历图中的每个节点。通过这种方式，我们可以逐渐将较短的路径移动到队列的末尾，最终实现最短路径的找出。
