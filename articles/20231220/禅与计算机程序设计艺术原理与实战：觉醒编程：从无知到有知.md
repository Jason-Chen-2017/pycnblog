给定一个整数数组，找出其中最大的数。

### 代码实现

```python
def find_max(arr):
    max_num = arr[0]
    for num in arr:
        if num > max_num:
            max_num = num
    return max_num
```

### 解释说明

在这个例子中，我们直接体验了问题，并根据问题设计了一个简单的算法。我们首先将数组中的第一个数作为最大数，然后遍历数组，并更新最大数。最后，我们返回最大数。

## 4.2 放下预判的代码实例

### 问题描述

给定一个整数数组，找出其中所有的奇数。

### 代码实现

```python
def find_odd_numbers(arr):
    odd_numbers = []
    for num in arr:
        if num % 2 != 0:
            odd_numbers.append(num)
    return odd_numbers
```

### 解释说明

在这个例子中，我们放弃了对问题的预判，并根据问题设计了一个简单的算法。我们首先创建一个空列表，并遍历数组。如果一个数是奇数，我们将其添加到列表中。最后，我们返回列表。
