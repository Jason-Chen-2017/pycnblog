我们将使用一个简单的排序算法来展示禅意编程的实际应用。这个排序算法是基于插入排序的思想，它的代码如下：

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

通过观察和实验，我们可以发现这个排序算法的核心思想是逐个将待排序的元素插入到已排序的元素中。这种思想是基于插入排序的思想，它的时间复杂度是 $O(n^2)$，其中 $n$ 是待排序的元素数量。
