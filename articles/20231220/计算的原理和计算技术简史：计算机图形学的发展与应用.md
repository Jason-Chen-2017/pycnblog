# 2.核心概念与联系

计算机图形学的核心概念包括：

1. 图形模型：用于表示图形信息的数据结构，如点、线、多边形、曲面等。
2. 图形变换：将一种图形转换为另一种图形的过程，如旋转、平移、缩放等。
3. 图形渲染：将图形模型转换为视觉表示，如颜色、光照、阴影等。
4. 图形输入：从实际环境中获取图形信息的方法，如摄像头、扫描仪等。
5. 图形输出：将计算机图形信息显示在人类眼球中的方法，如显示器、打印机等。

这些概念之间的联系如下：

1. 图形模型是计算机图形学的基础，用于存储和处理图形信息。
2. 图形变换和图形渲染是计算机图形学的核心技术，用于创建和显示图形。
3. 图形输入和图形输出是计算机图形学的应用领域，用于实现图形信息的获取和展示。
