## 4.1 例子：求两个数的最小公倍数
给定两个整数 a 和 b，求它们的最小公倍数。

### 4.1.1 算法设计
我们可以使用辗转相除法（Euclidean algorithm）来求最小公倍数。辗转相除法的原理是：对于两个整数 a 和 b，如果 a = b * k + r，其中 k 是一个整数，r 是一个非负整数，并且 r < b，那么 b 的最大公约数（gcd）与 r 的最大公约数相同，同时 a 的最小公倍数与 b 的最小公倍数相同。

### 4.1.2 算法实现
```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

def lcm(a, b):
    return a * b // gcd(a, b)

a = 12
b = 18
print(lcm(a, b))
```
在上面的代码中，我们首先定义了一个函数 gcd，用于计算两个整数的最大公约数。然后我们定义了一个函数 lcm，用于计算两个整数的最小公倍数。最后，我们测试了这两个函数，并输出了结果。
