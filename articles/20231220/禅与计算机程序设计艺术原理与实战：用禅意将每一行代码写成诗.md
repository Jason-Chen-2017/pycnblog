## 2.3 无思想

无思想是禅意学中的一个重要概念，它指的是在执行某个任务时，不需要思考，只需要直接做。在编程中，无思想可以帮助程序员更好地专注于任务，减少不必要的思考，提高效率。

## 3.1 动态规划

动态规划是一种常用的算法设计方法，它可以用来解决一些具有最优子结构的问题。在动态规划中，我们需要找到一个状态转移方程，并根据这个方程来计算每个状态的值。

### 3.1.1 状态转移方程

在动态规划中，状态转移方程是用来描述状态之间关系的。例如，在计算机程序设计中，我们经常需要计算字符串的最长公共子序列，这是一个动态规划问题，其状态转移方程如下：

$$
dp[i][j] = \begin{cases}
1, & \text{if } s[i-1] == t[j-1] \\
0, & \text{otherwise}
\end{cases}
$$

其中，$s$ 和 $t$ 是两个字符串，$dp[i][j]$ 是字符串 $s$ 和 $t$ 的最长公共子序列的长度。

### 3.1.2 具体操作步骤

动态规划的具体操作步骤如下：

1. 初始化：将所有状态的值设为负无穷。
2. 状态转移：根据状态转移方程计算每个状态的值。
3. 求解：根据状态转移方程计算最终状态的值。

### 3.1.3 禅意学思想应用

在动态规划中，禅意学的整体性思想可以帮助我们更好地设计算法。例如，在计算字符串的最长公共子序列时，我们可以将问题分解为子问题，并根据子问题的状态转移方程计算每个状态的值。这样，我们可以更好地看到整体，从而更好地设计算法。

贪心算法是一种常用的算法设计方法，它可以用来解决一些具有贪心性质的问题。在贪心算法中，我们需要在每个步骤中做出最佳的决策，并相信这些决策会导致最佳的结果。

### 3.2.1 贪心策略

在贪心算法中，贪心策略是用来描述算法的决策方式的。例如，在计算机程序设计中，我们经常需要计算最接近目标值的整数，这是一个贪心算法问题，其贪心策略如下：

$$

$$

### 3.2.2 具体操作步骤

贪心算法的具体操作步骤如下：

1. 初始化：将所有状态的值设为负无穷。
2. 贪心策略：根据贪心策略选择最佳的决策。
3. 求解：根据贪心策略计算最终状态的值。

def longest_common_subsequence(s, t):
    m = len(s)
    n = len(t)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(m):
        for j in range(n):
            if s[i] == t[j]:
                dp[i + 1][j + 1] = dp[i][j] + 1
            else:
                dp[i + 1][j + 1] = max(dp[i + 1][j], dp[i][j + 1])

    return dp[-1][-1]
```

def closest_integer(target, nums):
    nums.sort()
    left, right = 0, len(nums) - 1

    while left <= right:
        mid = (left + right) // 2
        if nums[mid] >= target:
            right = mid - 1
        else:
            left = mid + 1

    return nums[left]
```
