## 4.具体代码实例和详细解释说明

### 4.1 创建JSON数据类型的表

首先，我们需要创建一个JSON数据类型的表。以下是一个示例代码：

```sql
CREATE TABLE json_table (
    id INT PRIMARY KEY,
    json_column JSON
);
```

在这个示例中，我们创建了一个名为json_table的表，其中id列是整数类型的主键，json_column列是JSON数据类型的列。

### 4.2 插入JSON数据

接下来，我们需要插入JSON数据到表中。以下是一个示例代码：

```sql
INSERT INTO json_table (id, json_column) VALUES (1, '{"name": "John", "age": 30, "city": "New York"}');
```

在这个示例中，我们插入了一个JSON数据到json_table表中，其中id为1，json_column为一个包含name、age和city三个属性的JSON对象。

### 4.3 更新JSON数据

然后，我们需要更新JSON数据。以下是一个示例代码：

```sql
UPDATE json_table SET json_column = JSON_SET(json_column, '$.age', 31) WHERE id = 1;
```

在这个示例中，我们更新了json_table表中id为1的记录的json_column的age属性，将其值从30更新为31。

### 4.4 删除JSON数据

接下来，我们需要删除JSON数据。以下是一个示例代码：

```sql
DELETE FROM json_table WHERE id = 1;
```

在这个示例中，我们删除了json_table表中id为1的记录。

### 4.5 查询JSON数据

最后，我们需要查询JSON数据。以下是一个示例代码：

```sql
SELECT json_column FROM json_table WHERE id = 1;
```

在这个示例中，我们查询了json_table表中id为1的记录的json_column。

### 4.6 使用JSON函数

我们还可以使用MySQL的JSON函数来处理JSON数据。以下是一个示例代码：

```sql
SELECT JSON_EXTRACT(json_column, '$.name') AS name, JSON_EXTRACT(json_column, '$.age') AS age FROM json_table WHERE id = 1;
```

在这个示例中，我们使用JSON_EXTRACT函数来提取json_column中的name和age属性，并将其作为name和age两个列返回。
