```python

        elif op == '-':
            return a - b
        elif op == '*':
            return a * b
        elif op == '/':
            return a / b
        else:
            raise ValueError("Invalid operation")
```

通过以上操作，我们已经成功地将原始程序中的依赖关系简化了，使得程序更加简洁明了。
