## 4.1 直指人心

假设我们需要设计一个简单的排序算法，以便对一组整数进行排序。首先，我们需要理解问题的特点和要求，以便设计出高效和可靠的算法。

```python
def sort_numbers(numbers):
    for i in range(len(numbers)):
        for j in range(i + 1, len(numbers)):
            if numbers[i] > numbers[j]:
                numbers[i], numbers[j] = numbers[j], numbers[i]
    return numbers
```

在这个例子中，我们使用了简单的冒泡排序算法。这个算法的时间复杂度是O(n^2)，其中n是输入的整数个数。虽然这个算法不是最高效的，但它足够简单，可以在许多情况下得到满意的结果。

## 4.2 禅行

接下来，我们需要实现这个排序算法，以便在实际工作中得到最大的效益。这可以通过优化代码来完成。

```python
def sort_numbers(numbers):
    for i in range(len(numbers)):
        for j in range(len(numbers) - 1, i, -1):
            if numbers[j - 1] > numbers[j]:
                numbers[j - 1], numbers[j] = numbers[j], numbers[j - 1]
    return numbers
```

在这个例子中，我们优化了冒泡排序算法，使其在最坏情况下的时间复杂度从O(n^2)降低到O(n)。这个优化可以提高算法的性能，从而提高软件的效率。
