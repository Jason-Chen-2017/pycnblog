2. 语法规则：形式语言的语法规则用于生成符号序列，语法规则可以用产生式表示。产生式的基本格式是 A -> aB 或 A -> a ，其中 A 是非终结符，a 是终结符，B 是非终结符或终结符。

2. 转移规则：自动机的转移规则用于处理输入符号序列，转移规则可以用表示。转移规则的基本格式是 δ(q, a) = q' ，其中 q 是当前状态，a 是输入符号，q' 是下一个状态。

### 1.6.2 自动机的类型

自动机的类型主要包括以下几种：

1. 确定性自动机（Deterministic Finite Automaton，DFA）：确定性自动机的转移规则是明确定义的，对于同一个状态和输入符号，只有一个下一个状态。
2. 非确定性自动机（Non-deterministic Finite Automaton，NFA）：非确定性自动机的转移规则是不明确定义的，对于同一个状态和输入符号，可能有多个下一个状态。
3. 有限状态自动机（Finite State Automaton，FSA））：有限状态自动机是确定性自动机和非确定性自动机的一种总称，它的状态是有限的。
4. 推导式文法（Production Grammar）：推导式文法是一种用于描述形式语言的自动机，它的主要特点是用产生式表示语法规则。
5. 推导式自动机（Parsing Automaton）：推导式自动机是一种用于处理输入符号序列的自动机，它的主要特点是用推导式文法表示语法规则。
