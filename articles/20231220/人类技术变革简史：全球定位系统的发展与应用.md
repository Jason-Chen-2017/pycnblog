                 

# 1.背景介绍

全球定位系统（Global Positioning System，GPS）是一种由美国国防部开发的卫星导航系统，由24个卫星组成，可以在任何地方提供定位、导航和时间信息。GPS系统的发展历程可以分为以下几个阶段：

1.1 初步研究阶段（1960年代至1970年代）

在1960年代，美国国防部开始研究卫星导航系统的可能性，以应对苏联的导弹防御系统。在1970年代，美国开始实际开发这一系统，并于1978年开始部署卫星。

1.2 初步运行阶段（1980年代）

在1980年代，GPS系统已经部署了8颗卫星，但是在这一阶段，GPS主要用于军事目的，并且对公众没有开放。

1.3 公开使用阶段（1990年代至2000年代）

在1990年代，GPS系统已经部署了24颗卫星，并且开始向公众提供服务。在这一阶段，GPS逐渐成为一种广泛应用的技术，被用于导航、地理信息系统、地图制作等多种领域。

1.4 现代发展阶段（2000年代至今）

在2000年代以来，GPS系统不断发展完善，不仅提供更准确的定位信息，还开发出了更多的应用，如车载GPS、手机GPS、个人导航器等。此外，还有其他国家开始研究和部署自己的卫星导航系统，如中国的北斗系统、欧洲的Galileo系统等。
