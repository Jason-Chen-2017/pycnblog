# 2.核心概念与联系

## 2.1 条件概率

条件概率是概率论中的一个基本概念，它描述了一个事件发生的条件下另一个事件发生的概率。给定一个事件A发生的条件下，事件B发生的概率可以表示为：

$$
P(B|A) = \frac{P(A \cap B)}{P(A)}
$$

其中，$P(B|A)$ 是条件概率，$P(A \cap B)$ 是事件A和事件B同时发生的概率，$P(A)$ 是事件A发生的概率。

在自然语言处理中，条件概率常用于计算词汇在特定上下文中的出现概率，这有助于解决词汇顺序、语义标注等问题。

## 2.2 贝叶斯定理

贝叶斯定理是概率论中的一个重要定理，它描述了如何更新先验概率为后验概率。给定一个事件A和事件B，贝叶斯定理可以表示为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 是事件A发生的条件下事件B发生的概率，$P(B|A)$ 是事件B发生的条件下事件A发生的概率，$P(A)$ 是事件A发生的概率，$P(B)$ 是事件B发生的概率。

在自然语言处理中，贝叶斯定理常用于计算词汇在特定上下文中的概率，这有助于解决语义标注、文本分类等问题。

```python
import numpy as np

# 训练数据集
train_data = [
    ('apple', 'fruit'),
    ('banana', 'fruit'),
    ('carrot', 'vegetable'),
    ('lettuce', 'vegetable')
]

# 计算先验概率
fruit_count = 0
vegetable_count = 0
total_count = 0
for item, label in train_data:
    if label == 'fruit':
        fruit_count += 1
    elif label == 'vegetable':
        vegetable_count += 1
    total_count += 1

P_fruit = fruit_count / total_count
P_vegetable = vegetable_count / total_count

# 计算条件概率
P_fruit_given_apple = 1
P_vegetable_given_carrot = 1

# 测试数据集

    ('apple',),
    ('carrot',)
]

# 分类
for item, _ in test_data:
    if item == 'apple':
        P_fruit_given_apple = 1
    elif item == 'carrot':
        P_vegetable_given_carrot = 1

    P_fruit = P_fruit * P_fruit_given_apple
    P_vegetable = P_vegetable * P_vegetable_given_carrot

    if P_fruit > P_vegetable:
        print('Predicted class: fruit')
    else:
        print('Predicted class: vegetable')
```
