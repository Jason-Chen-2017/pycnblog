# 2.核心概念与联系

在计算几何中，函数映射是一种将一个空间映射到另一个空间的方法。它可以用来解决许多问题，如最近点对距离、最小包含凸包等。函数映射可以分为几种类型，如线性映射、非线性映射、仿射映射等。

## 2.1 线性映射

线性映射是一种将一个向量空间映射到另一个向量空间的方法。它可以用来解决许多问题，如最小二乘法、线性方程组等。线性映射可以表示为一个矩阵，其中每一行代表一个向量。

## 2.2 非线性映射

非线性映射是一种将一个空间映射到另一个空间的方法，但不是线性的。它可以用来解决许多问题，如最小化函数、求解方程等。非线性映射可以表示为一个函数，其中每一点代表一个向量。

## 2.3 仿射映射

仿射映射是一种将一个空间映射到另一个空间的方法，包括线性映射和非线性映射。它可以用来解决许多问题，如平移、旋转、缩放等。仿射映射可以表示为一个矩阵，其中每一行代表一个向量。
