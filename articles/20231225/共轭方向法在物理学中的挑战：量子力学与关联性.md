## 3.1 共轭方向法的核心算法原理

共轭方向法（Conjugate Gradient, CG）是一种用于解线性方程组的迭代方法，其核心算法原理如下：

1. 初始化：选择一个初始向量x0，并计算初始残差向量r0 = b - Ax0。
2. 共轭方向：计算当前步的共轭方向hk = -AkT * rk-1。
3. 线性方程组的解：通过迭代地求解当前步的残差向量，逐步Approximate解线性方程组。

## 3.2 具体操作步骤

共轭方向法的具体操作步骤如下：

1. 选择一个初始向量x0，并计算初始残差向量r0 = b - Ax0。
2. 计算共轭矩阵AkT，并计算当前步的共轭方向hk = -AkT * rk-1。
3. 更新当前步的解向量xk = xk-1 + αkhk，其中α是步长因子。
4. 计算当前步的残差向量rk = b - Axk。
5. 检查收敛性，如果满足收敛条件，则停止迭代，返回最终解向量xk。否则，返回步骤2。

## 4.1 代码实例

以下是一个使用共轭方向法求解线性方程组的Python代码实例：

```python
import numpy as np

# 定义方程组矩阵和向量
A = np.array([[2, -1], [-1, 2]])
b = np.array([1, -1])

# 初始化向量和残差向量
x = np.zeros(A.shape[0])
r = b - Ax

# 设置步长因子和收敛条件
alpha = 0.1
tol = 1e-6

# 迭代求解
k = 0
while True:
    # 计算共轭方向
    hk = -A.T @ r
    
    # 更新解向量
    xk = x + alpha * hk
    
    # 计算新的残差向量
    rk = b - Axk
    
    # 检查收敛性
    if np.linalg.norm(rk) < tol:
        break
    
    # 更新残差向量和迭代次数
    r = rk
    k += 1

# 输出结果
print("解向量：", xk)
```

## 4.2 详细解释说明

1. 首先，我们定义了方程组矩阵A和向量b。
2. 然后，我们初始化解向量x和残差向量r。
3. 设置步长因子alpha和收敛条件tol。
4. 进行迭代求解，计算共轭方向hk，更新解向量xk，计算新的残差向量rk，并检查收敛性。
5. 如果收敛条件满足，则停止迭代，输出最终解向量xk。否则，继续迭代。
