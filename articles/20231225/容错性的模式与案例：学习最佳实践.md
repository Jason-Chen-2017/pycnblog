## 4.1错误检测示例

我们可以通过以下代码实现错误检测：

```python
def checksum(data):
    return sum(data)

data = [1, 2, 3, 4, 5]
checksum_value = checksum(data)

if checksum_value != checksum(data):
    print("Error detected")
```

在这个示例中，我们通过计算数据的检查和值来检测错误。如果检查和值不匹配，则表示发生了错误，并输出错误信息。

## 4.2错误定位示例

我们可以通过以下代码实现错误定位：

```python
def find_error(data):
    for i in range(len(data)):
        if data[i] != i:
            return i
    return -1

data = [0, 1, 2, 3, 4, 5]
error_index = find_error(data)

if error_index != -1:
    print("Error located at index", error_index)
```

在这个示例中，我们通过遍历数据列表来定位错误。如果发现数据值与索引不匹配，则表示发生了错误，并输出错误信息。

## 4.3自愈示例

我们可以通过以下代码实现自愈：

```python
def recover_error(data, error_index):
    data[error_index] = error_index
    return data

error_index = find_error(data)
if error_index != -1:
    data = recover_error(data, error_index)
    print("Error recovered")
```

在这个示例中，我们通过修改数据列表中的错误值来实现自愈。如果发生错误，则调用`recover_error`函数进行修复并输出错误恢复信息。
