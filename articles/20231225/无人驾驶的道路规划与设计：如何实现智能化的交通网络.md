## 4.1 道路网络规划

我们可以使用Python的NetworkX库来实现道路网络的规划。以下是一个简单的例子：

```python
import networkx as nx

# 创建一个有向图
G = nx.DiGraph()

# 添加节点
G.add_node("A")
G.add_node("B")
G.add_node("C")

# 添加边
G.add_edge("A", "B")
G.add_edge("B", "C")

# 计算最短路径
shortest_path = nx.shortest_path(G, source="A", target="C")
print(shortest_path)
```

在这个例子中，我们创建了一个有向图，并添加了三个节点和两条边。然后，我们计算了从节点“A”到节点“C”的最短路径。
