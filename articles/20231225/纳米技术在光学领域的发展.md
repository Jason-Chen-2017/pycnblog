# 4.具体代码实例和详细解释说明

在这里，我们以一个简单的纳米光学滤波器设计为例，展示如何编写代码实现。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义纳米光学滤波器的传递函数
def nanophotonic_filter(wavelength, center_wavelength, bandwidth):
    # 计算频率
    frequency = wavelength / (2 * np.pi * center_wavelength)
    # 计算传递函数
    transmission = np.exp(-0.5 * ((frequency - center_wavelength) / bandwidth) ** 2)
    return transmission

# 设置波长数组
wavelengths = np.linspace(300, 1000, 1000)
# 设置中心波长和带宽
center_wavelength = 550
bandwidth = 50
# 计算传递函数
transmissions = nanophotonic_filter(wavelengths, center_wavelength, bandwidth)
# 绘制传递函数
plt.plot(wavelengths, transmissions)
plt.xlabel('Wavelength (nm)')
plt.ylabel('Transmission')
plt.title('Nanophotonic Filter')
plt.show()
```

上述代码首先导入了 numpy 和 matplotlib.pyplot 库，然后定义了一个 nanophotonic_filter 函数，用于计算纳米光学滤波器的传递函数。接着设置了波长数组、中心波长和带宽，并调用 nanophotonic_filter 函数计算传递函数。最后使用 matplotlib.pyplot 库绘制了传递函数图像。
