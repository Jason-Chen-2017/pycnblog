```python
# 创建一个控制变量
temperature = ctrl.Antecedent(np.arange(0, 101, 1), 'temperature')

# 创建一个控制变量
cooling = ctrl.Consequent(np.arange(0, 101, 1), 'cooling')

# 创建一个模糊逻辑规则
rule = ctrl.Rule(temperature['low'] <= 18, cooling['high'])
rule.then('cooling')

# 创建一个模糊逻辑系统
cl = ctrl.ControlSystem([rule])

# 创建一个模糊输入
t = ctrl.ControlVariable(name='temperature')

# 创建一个模糊输出
c = ctrl.ControlVariable(name='cooling')

# 创建一个模糊接口
sim = ctrl.ControlSystemSimulation(cl)

# 设置模糊输入
sim.input['temperature'] = 20

# 运行模糊逻辑系统
sim.compute()

# 获取模糊输出
print(sim.output['cooling'])
```

3. 接着，我们创建了一个控制变量`cooling`，它的取值范围也是0到100。

4. 然后，我们创建了一个模糊逻辑规则，它的条件是控制变量`temperature`的值小于或等于18，然后的动作是控制变量`cooling`的值为`high`。

5. 接着，我们创建了一个模糊逻辑系统，并将模糊逻辑规则添加到系统中。

6. 然后，我们创建了一个模糊输入`t`，并将其值设置为20。

7. 接着，我们创建了一个模糊输出`c`。
