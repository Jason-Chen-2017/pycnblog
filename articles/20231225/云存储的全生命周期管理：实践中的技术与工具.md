```python
import hashlib

class CloudStorage:
    def __init__(self):
        self.storage = {}

    def store(self, data):
        hash_value = hashlib.sha256(data.encode()).hexdigest()
        self.storage[hash_value] = data

    def retrieve(self, hash_value):
        return self.storage.get(hash_value)

    def delete(self, hash_value):
        if hash_value in self.storage:
            del self.storage[hash_value]

storage = CloudStorage()
data = b"hello, world!"
storage.store(data)
print(storage.retrieve(hash_value))
storage.delete(hash_value)
```

在上面的代码中，我们首先导入了`hashlib`库，然后定义了一个`CloudStorage`类。这个类有三个方法：`store`、`retrieve`和`delete`。`store`方法用于存储数据，`retrieve`方法用于检索数据，`delete`方法用于删除数据。

在主程序中，我们创建了一个`CloudStorage`对象，然后存储了一个字符串"hello, world!"。接着，我们通过`retrieve`方法检索了该数据，并通过`delete`方法删除了该数据。
