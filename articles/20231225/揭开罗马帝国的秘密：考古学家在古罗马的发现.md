### 4.1.1 考古物品的观察与分析

在这个环节中，我们将使用一个简单的Python程序来实现考古物品的观察与分析。首先，我们需要一个数据集，包括考古物品的形状、材料和工艺等特征。然后，我们可以使用K-近邻算法来分类这些考古物品。

```python
import numpy as np
from sklearn.neighbors import KNeighborsClassifier

# 考古物品的特征数据集
data = np.array([
    ['形状', '材料', '工艺'],
    ['圆形', '金', '镂空'],
    ['方形', '陶瓷', '烧瓷'],
    ['椭圆形', '金', '镂空'],
    ['长方形', '陶瓷', '烧瓷'],
    ['圆柱形', '金', '镂空'],
    ['椭圆柱形', '陶瓷', '烧瓷'],
])

# 训练K-近邻分类器
knn = KNeighborsClassifier(n_neighbors=3)
knn.fit(data[:, 0:2], data[:, 2])

# 对新的考古物品进行分类
new_item = np.array(['圆形', '金', '镂空'])
knn.predict(new_item[:, np.newaxis])
```
