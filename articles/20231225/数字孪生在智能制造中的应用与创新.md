## 4.1 数据采集与预处理

以下是一个简单的温度传感器数据采集和预处理的Python代码实例：
```python
import numpy as np
import pandas as pd

# 模拟温度传感器数据
def generate_temperature_data():
    np.random.seed(0)
    time = np.arange(0, 10, 0.1)
    temperature = np.sin(time) + 25
    return time, temperature

time, temperature = generate_temperature_data()
data = pd.DataFrame({'time': time, 'temperature': temperature})

# 滤波处理
def moving_average(data, window_size):
    return data.rolling(window=window_size).mean()

window_size = 5
data['filtered_temperature'] = moving_average(data['temperature'], window_size)
```
在这个代码实例中，我们首先模拟了温度传感器的数据，然后使用滤波处理方法（即移动平均）对数据进行预处理，以提高数据的质量。

```python
import sqlite3

# 创建数据库和表
def create_database(database_name):
    conn = sqlite3.connect(database_name)
    cursor = conn.cursor()
    cursor.execute('''CREATE TABLE IF NOT EXISTS temperature_data (time REAL, temperature REAL)''')
    conn.commit()
    return conn

# 存储数据
def store_data(conn, data):
    for index, row in data.iterrows():
        cursor = conn.cursor()
        cursor.execute('''INSERT INTO temperature_data (time, temperature) VALUES (?, ?)''', (row['time'], row['temperature']))
        conn.commit()

# 查询数据
def query_data(conn):
    cursor = conn.cursor()
    cursor.execute('''SELECT * FROM temperature_data''')
    return pd.DataFrame(cursor.fetchall(), columns=['time', 'temperature'])

conn = create_database('temperature_data.db')
store_data(conn, data)
data_query = query_data(conn)
```
在这个代码实例中，我们使用SQLite数据库来存储和管理温度传感器的数据。首先，我们创建了一个数据库和一个表，然后将数据存储到表中，最后查询数据并将其转换为DataFrame。
