## 3.1 前馈神经网络

前馈神经网络是一种最基本的深度学习模型，它由输入层、隐藏层和输出层组成。输入层接收输入数据，隐藏层进行数据处理，输出层产生预测结果。前馈神经网络的算法原理如下：

$$
y = f(XW + b)
$$

其中，$X$是输入数据，$W$是权重矩阵，$b$是偏置向量，$f$是激活函数。

## 3.2 卷积神经网络

卷积神经网络是一种用于图像处理的深度学习模型，它由卷积层、池化层和全连接层组成。卷积层用于提取图像的特征，池化层用于降维，全连接层用于产生预测结果。卷积神经网络的算法原理如下：

$$
C = f(K \ast X + b)
$$

其中，$X$是输入图像，$K$是卷积核，$\ast$是卷积操作，$f$是激活函数，$C$是输出特征图，$b$是偏置向量。

## 3.3 循环神经网络

循环神经网络是一种用于自然语言处理和时间序列预测的深度学习模型，它由循环层和全连接层组成。循环层可以记忆之前的输入，以便在当前时间步产生预测结果。循环神经网络的算法原理如下：

$$
h_t = f(W_{hh}h_{t-1} + W_{xh}x_t + b)
$$

其中，$h_t$是隐藏状态，$W_{hh}$是隐藏状态到隐藏状态的权重矩阵，$W_{xh}$是输入到隐藏状态的权重矩阵，$x_t$是当前输入，$b$是偏置向量。

```python
import torch
import torch.nn as nn
import torch.optim as optim

# 定义前馈神经网络模型
class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.fc1 = nn.Linear(10, 5)
        self.fc2 = nn.Linear(5, 1)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# 创建模型实例
net = Net()

# 定义损失函数和优化器
criterion = nn.MSELoss()
optimizer = optim.SGD(net.parameters(), lr=0.01)

# 训练模型
```
