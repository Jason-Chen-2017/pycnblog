### 3.1 集合的基本操作

集合的基本操作包括：

1. 求交集：求两个集合的共同元素。
2. 求并集：求两个集合的所有元素。
3. 求差集：求一个集合中不在另一个集合中的元素。

这些操作可以通过数学模型公式来表示：

$$
A \cap B = \{x | x \in A \text { and } x \in B\}
$$

$$
A \cup B = \{x | x \in A \text { or } x \in B\}
$$

$$
A \setminus B = \{x | x \in A \text { and } x \notin B\}
$$

## 4.具体代码实例和详细解释说明

在本节中，我们将介绍一个具体的代码实例，以及其详细解释说明。

### 4.1 代码实例

我们来看一个简单的代码实例，用于可视化地表示两个集合的交集、并集和差集。

```python
import matplotlib.pyplot as plt

A = [1, 2, 3, 4, 5]
B = [3, 4, 5, 6, 7]

# 求交集
intersection = set(A) & set(B)
print("交集:", intersection)

# 求并集
union = set(A) | set(B)
print("并集:", union)

# 求差集
difference = set(A) - set(B)
print("差集:", difference)

# 可视化地表示
plt.scatter(A, [1]*len(A), label="A")
plt.scatter(B, [1]*len(B), label="B")
plt.legend()
plt.show()
```

### 4.2 详细解释说明

这个代码实例首先导入了`matplotlib.pyplot`库，用于可视化地表示集合。然后定义了两个集合`A`和`B`。接着，通过`set(A) & set(B)`求得两个集合的交集，通过`set(A) | set(B)`求得两个集合的并集，通过`set(A) - set(B)`求得两个集合的差集。最后，通过`plt.scatter()`函数可视化地表示两个集合的元素，并使用`plt.legend()`函数添加图例。
