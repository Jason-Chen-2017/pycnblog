- 每个客户的总订单金额
- 每个产品的总销售额
- 每个订单日期的总订单金额

1. 创建一个 Pinot 表，包括以下字段：

    ```

        order_id INT,
        customer_id INT,
        product_id INT,
        order_date DATE,
        order_amount DECIMAL
    );
    ```

    ```

    GROUP BY customer_id;

    SELECT product_id, SUM(order_amount) AS total_sales_amount
    FROM sales_table
    GROUP BY product_id;

    FROM sales_table
    GROUP BY order_date;
    ```
