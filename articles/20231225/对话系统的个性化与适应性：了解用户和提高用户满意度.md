```python
from sklearn.metrics.pairwise import cosine_similarity

# 用户历史交互记录
user_history = ["你好，我想买一台电脑。", "我需要一个高性能的电脑。", "价格不迷茫。"]

# 用户兴趣和需求
user_interest = ["高性能", "价格合理"]

# 构建TF-IDF向量器
vectorizer = TfidfVectorizer()

# 将用户历史交互记录转换为TF-IDF向量
user_history_vector = vectorizer.fit_transform(user_history)

# 将用户兴趣和需求转换为TF-IDF向量
user_interest_vector = vectorizer.transform(user_interest)

# 计算用户历史交互记录与用户兴趣和需求之间的相似度
similarity = cosine_similarity(user_history_vector, user_interest_vector)

# 根据相似度调整对话策略
if similarity > 0.5:
    strategy = "推荐高性能且价格合理的电脑"
else:
    strategy = "提供更多关于高性能电脑的信息"

print(strategy)
```

在这个代码实例中，我们首先导入了TF-IDF向量器和相似度计算函数。然后，我们获取了用户的历史交互记录和兴趣需求。接着，我们将用户历史交互记录和兴趣需求转换为TF-IDF向量。最后，我们计算了用户历史交互记录与兴趣需求之间的相似度，并根据相似度调整了对话策略。
