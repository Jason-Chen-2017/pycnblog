### 步骤3：确定领导者
在鲸鱼优化算法中，领导者是最优解的鲸鱼。我们可以通过比较每个鲸鱼的适应度来确定领导者。具体来说，我们可以将领导者设置为适应度最大的鲸鱼。

$$

$$

### 步骤5：重复步骤2-4，直到满足终止条件
在鲸鱼优化算法中，我们需要重复步骤2-4，直到满足终止条件。这些终止条件可以是迭代次数达到某个值、适应度变化小于某个阈值等。

$$
X_{i}(t+1) = X_{i}(t) + A \times e^{(-B \times |T|^{2})}\times D
$$
我们可以通过以下代码来实现这个例子：
```python
import numpy as np

def f(x):
    return -x**2

def whale_optimization_algorithm(num_whales, num_iterations, x_bounds):
    # 初始化鲸鱼群的位置
    whales = np.random.uniform(x_bounds[0], x_bounds[1], num_whales)
    
    # 初始化领导者
    leader = whales[np.argmax(f(whales))]
    
    # 开始迭代
    for t in range(num_iterations):
        # 计算每个鲸鱼的适应度
        fitness = np.array([f(whale) for whale in whales])
        
        # 确定领导者
        leader = whales[np.argmax(fitness)]
        
        # 更新鲸鱼群的位置
        for i in range(num_whales):
            A = np.random.rand()
            B = 2
            T = np.random.rand()
            D = leader - whales[i]
            whales[i] = whales[i] + A * np.exp(-B * np.power(T, 2)) * D
            
    # 返回最优解
    return leader

# 设置参数
num_whales = 50
num_iterations = 100
x_bounds = (-100, 100)

# 运行鲸鱼优化算法
best_whale = whale_optimization_algorithm(num_whales, num_iterations, x_bounds)
print("最优解：", best_whale)
print("最优值：", -best_whale**2)
```
