## 4.1 生物计数的最大似然估计

我们假设某个生物种类在一定区域的数量遵循泊松分布，例如泊松分布。我们的目标是找到使观察数据的概率最大化的参数值。

具体代码实例如下：

```python
import numpy as np
from scipy.optimize import minimize

def poisson_likelihood(k, lambda_):
    return np.sum(np.log(np.exp(-lambda_ * k) * (lambda_ ** k) / np.math.factorial(k)))

def poisson_mle(k):
    result = minimize(lambda lambda_: -poisson_likelihood(k, lambda_), np.array([1]), bounds=[(0, 100)])
    return result.x[0]

k = 10
lambda_ = poisson_mle(k)
print("Estimated lambda:", lambda_)
```

在这个代码实例中，我们首先导入了numpy和scipy.optimize库。然后，我们定义了一个名为poisson_likelihood的函数，用于计算泊松分布的概率密度函数值。接着，我们定义了一个名为poisson_mle的函数，用于根据观察数据（在这个例子中，观察数据是10）估计某个生物种类在一定区域的数量。最后，我们调用poisson_mle函数，并打印估计结果。

```python
import numpy as np
from scipy.optimize import minimize

def linear_model_likelihood(y, beta, sigma):
    return -np.sum((y - (beta[0] + beta[1] * y)) ** 2) / (2 * sigma ** 2) - np.log(sigma) - np.log(np.sqrt(2 * np.pi))

def linear_model_mle(y, beta):
    sigma = np.std(y - beta[0] - beta[1] * y)
    result = minimize(lambda beta: -linear_model_likelihood(y, beta, sigma), beta, bounds=[(-10, 10), (-10, 10)])
    return result.x

y = np.array([1, 2, 3, 4, 5])
beta = linear_model_mle(y, np.array([0, 0]))
print("Estimated beta:", beta)
```
