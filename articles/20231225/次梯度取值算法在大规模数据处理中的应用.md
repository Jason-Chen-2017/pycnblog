# 2.核心概念与联系

次梯度取值算法是一种优化算法，它通过使用次梯度信息来加速优化过程。次梯度取值算法的核心概念包括：

1. 损失函数：损失函数是用于衡量模型预测值与真实值之间差距的函数。在机器学习中，损失函数通常是一个非负值，小的损失函数值表示预测结果更接近真实值。

2. 梯度下降：梯度下降是一种常用的优化算法，它通过迭代地更新模型参数来最小化损失函数。梯度下降算法的核心思想是通过在梯度方向上进行小步长的更新来逼近损失函数的最小值。

4. 次梯度取值算法（STOMP）：次梯度取值算法是一种优化算法，它通过使用次梯度信息来加速梯度下降算法。次梯度取值算法的核心思想是通过使用次梯度信息来更准确地估计梯度下降算法的步长，从而加速优化过程。
