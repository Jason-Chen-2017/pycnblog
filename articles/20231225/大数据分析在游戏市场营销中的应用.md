# 4. 具体代码实例和详细解释说明

以下是一个简单的Python代码实例，使用K均值聚类算法对用户行为数据进行聚类分析：

```python
from sklearn.cluster import KMeans
import pandas as pd
import numpy as np

# 加载用户行为数据
data = pd.read_csv('user_behavior_data.csv')

# 数据预处理
data = data.dropna()
data = data.fillna(0)
data = data.astype(int)

# 选择聚类算法
kmeans = KMeans(n_clusters=3)

# 参数设置
kmeans.fit(data)

# 聚类分析
data['cluster'] = kmeans.labels_

# 结果分析
data.groupby('cluster').mean()
```
