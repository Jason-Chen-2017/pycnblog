# 2.核心概念与联系

## 2.1 量子计算机

量子计算机是一种新型的计算机，它利用量子比特（qubit）和量子门（quantum gate）进行计算。量子比特不同于传统的比特，它可以同时处于多个状态上，这使得量子计算机具有巨大的并行计算能力。量子计算机的出现为许多复杂问题提供了新的解决方案。

## 2.2 RSA密码系统

RSA密码系统是一种公开密钥加密系统，它使用两个大素数作为密钥对生成。RSA密码系统的安全性主要依赖于大素数因子化问题的困难。如果能够高效地解决大素数因子化问题，则RSA密码系统将失去其安全性。
