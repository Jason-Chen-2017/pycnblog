## 4.1 数据掩码

我们可以使用Python的NumPy库来实现数据掩码。以下是一个简单的数据掩码示例：

```python
import numpy as np

# 原始数据
X = np.array([1, 2, 3, 4, 5])

# 噪声分布
noise_dist = np.random.normal(0, 1, size=X.shape)

# 掩码后的数据
Y = X + noise_dist

print(Y)
```

在这个示例中，我们首先导入NumPy库，然后创建一个原始数据数组`X`。接着，我们创建一个正态分布的噪声数组`noise_dist`，并将其添加到原始数据中，得到掩码后的数据`Y`。
