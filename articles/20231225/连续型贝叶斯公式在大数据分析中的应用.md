# 2.核心概念与联系

## 2.1 贝叶斯定理
贝叶斯定理是贝叶斯方法的基础，它提供了一种更新先验概率分布为后验概率分布的方法。给定一个事件A和一个条件B，贝叶斯定理的公式为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示条件概率，即给定B发生，A发生的概率；$P(B|A)$ 表示反条件概率，即给定A发生，B发生的概率；$P(A)$ 和 $P(B)$ 分别表示事件A和B的先验概率。

## 2.2 连续型贝叶斯公式
连续型贝叶斯公式是用于处理连续变量的贝叶斯公式。在大数据分析中，连续型贝叶斯公式可以用于估计连续变量的概率分布，以及根据新数据更新现有的概率分布。连续型贝叶斯公式的核心是利用先验分布和观测数据来估计后验分布。

## 3.3 数学模型公式详细讲解
连续型贝叶斯公式的数学模型公式可以表示为：

$$
f_{posterior}(x) = \int f_{likelihood}(x|θ) \cdot f_{prior}(θ) dθ
$$

其中，$f_{posterior}(x)$ 表示后验概率密度函数；$f_{likelihood}(x|θ)$ 表示似然性函数；$f_{prior}(θ)$ 表示先验概率密度函数。

```python
def posterior_pdf(x, mu_p, sigma_p, observations):
    prior_pdf = norm.pdf(x, mu_p, sigma_p)
    likelihood_pdf = norm.pdf(x, np.mean(observations), np.std(observations))
    posterior_pdf = prior_pdf * likelihood_pdf
    return posterior_pdf

x = np.linspace(150, 200, 100)
```
