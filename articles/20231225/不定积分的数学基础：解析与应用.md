不定积分的具体代码实例如下：

```python
import numpy as np
import matplotlib.pyplot as plt

def integral(f, a, b, h):
    x = np.arange(a, b, h)
    y = [f(i) for i in x]
    return np.trapz(y, x)

def f(x):
    return x**2 + 2*x + 1

a = 0
b = 4
h = 0.1
result = integral(f, a, b, h)
print("不定积分的值为：", result)

x = np.linspace(a, b, 100)
y = [f(i) for i in x]
plt.plot(x, y)
plt.xlabel('x')
plt.ylabel('y')
plt.title('不定积分的曲线图')
plt.show()
```

上述代码中，我们首先导入了 numpy 和 matplotlib.pyplot 库，然后定义了一个名为 integral 的函数，该函数用于计算不定积分的值。接着，我们定义了一个名为 f 的函数，该函数表示被积函数。然后，我们设定了积分的积分变量范围和步长，并调用 integral 函数计算不定积分的值。最后，我们使用 matplotlib.pyplot 库绘制了不定积分的曲线图。
