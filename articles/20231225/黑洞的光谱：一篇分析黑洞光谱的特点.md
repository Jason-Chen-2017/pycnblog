# 2.核心概念与联系

在分析黑洞光谱的特点之前，我们需要了解一些核心概念和联系。

## 2.1 黑洞的类型

目前，我们已经发现了两种主要类型的黑洞：超大黑洞和中心黑洞。

- 超大黑洞：这种类型的黑洞通常形成在巨大星的结果，其质量可达到几十万到几百万次太阳质量。超大黑洞通常位于星系的外层，与星系中心没有太大的关系。

- 中心黑洞：这种类型的黑洞通常位于星系的中心，与星系的形成和演化密切相关。中心黑洞的质量通常在几百万到几亿次太阳质量之间。

## 2.2 黑洞光谱的形成

黑洞光谱的形成主要受到以下几个因素的影响：

- 黑洞的质量和尺寸
- 周围环境的物质分布和运动情况
- 黑洞内部的物质状态和运动情况

由于黑洞内部的物质状态和运动情况无法直接观察，因此我们需要通过观察黑洞周围的环境和物质运动情况，从而推测黑洞光谱的特点。
