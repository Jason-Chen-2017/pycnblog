# 2.核心概念与联系

## 2.1 量子显微镜
量子显微镜是一种基于原子力学的显微镜技术，它可以在原子级别对样品进行观察和测量。量子显微镜通过将样品与一个量子系统相互作用，从而获取样品的原子结构和性质信息。量子显微镜可以用于观察单个原子、分子、纳米粒子等微观结构，具有极高的分辨率和敏感度。

生物标签化研究是一种研究方法，它涉及将特定的标签或标记物质附加到生物样品中，以便进行观察和测量。生物标签化研究在生物科学、生物技术和生物医学等领域具有广泛的应用，例如：
