2. 根据温度和密度计算宇宙的膨胀率。

4. 重复步骤2和步骤3，直到宇宙的膨胀停止。

$$

```python
import numpy as np

def expand_universe(R, t, G, rho):
    H = np.sqrt(8 * np.pi * G * rho / 3)
    dR = H * R * dt
    return R + dR

R = 0
t = 0
G = 6.67430e-11
rho = 1e-26
dt = 1e-10

for t in np.arange(0, 13.8e9, dt):
    R = expand_universe(R, t, G, rho)
    print(R)
```
