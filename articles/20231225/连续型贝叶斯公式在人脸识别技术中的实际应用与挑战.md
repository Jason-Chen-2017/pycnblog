# 2.核心概念与联系

## 2.1 人脸识别技术

人脸识别技术是一种基于人脸特征的识别技术，它通过对人脸的图像进行分析，从而实现人员识别的技术。人脸识别技术可以分为两种：一种是基于人脸的结构特征的识别，另一种是基于人脸的文本特征的识别。

结构特征包括眼睛、鼻子、嘴巴等，这些特征是不可变的，因此可以用于人脸识别。文本特征包括人脸的颜色、形状、大小等，这些特征是可变的，因此可以用于人脸识别。

人脸识别技术的主要应用领域包括安全、金融、医疗等多个领域。例如，在安全领域，人脸识别技术可以用于身份验证和访问控制；在金融领域，人脸识别技术可以用于贷款审批和信用卡支付；在医疗领域，人脸识别技术可以用于病人诊断和治疗。
