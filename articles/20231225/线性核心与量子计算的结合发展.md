## 4.1 线性核心的代码实例
```
import numpy as np

# 定义矩阵A和向量b
A = np.array([[1, 2], [3, 4]])
b = np.array([5, 6])

# 求解线性方程组Ax = b

```
## 4.2 量子计算的代码实例
```
import qiskit

# 定义量子电路
qc = qiskit.QuantumCircuit(2, 2)

# 添加量子门
qc.h(0)
qc.cx(0, 1)
qc.measure([0, 1], [0, 1])

# 执行量子电路
backend = qiskit.Aer.get_backend('qasm_simulator')
job = qiskit.execute(qc, backend, shots=1000)
result = job.result()

# 获取结果
counts = result.get_counts()
print(counts)
```
