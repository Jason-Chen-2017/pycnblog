# 4.具体代码实例和详细解释说明

在这里，我们将提供一个具体的代码实例来说明如何使用智能合约实现去中心化的医疗保健资源分配。

```solidity
pragma solidity ^0.5.0;

contract MedicalResourceAllocation {
    // 定义患者结构
    struct Patient {
        uint id;
        string diagnosis;
        uint treatmentCost;
    }

    // 定义医疗保健资源结构
    struct MedicalResource {
        uint id;
        uint quantity;
    }

    // 定义患者数组
    Patient[] public patients;

    // 定义医疗保健资源数组
    MedicalResource[] public medicalResources;

    // 资源分配函数
    function allocateResource(uint patientId, uint resourceId) public {
        require(patientId < patients.length, "Patient not found");
        require(resourceId < medicalResources.length, "Resource not found");

        Patient memory patient = patients[patientId];
        MedicalResource memory resource = medicalResources[resourceId];

        // 根据患者的诊断和治疗方案来分配医疗保健资源
        if (patient.diagnosis == "心脏病" && patient.treatmentCost <= resource.quantity) {
            resource.quantity -= patient.treatmentCost;
            patients[patientId].treatmentCost = 0;
        }
    }
}
```

在这个代码实例中，我们定义了一个智能合约`MedicalResourceAllocation`，它包括患者和医疗保健资源的结构以及资源分配的函数。在`allocateResource`函数中，我们根据患者的诊断和治疗方案来分配医疗保健资源。
