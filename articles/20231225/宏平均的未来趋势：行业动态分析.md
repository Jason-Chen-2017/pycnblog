# 2.核心概念与联系
## 2.1 宏平均的定义
宏平均是一种数据处理方法，它通过将数据点的值进行加权平均计算，从而得到一个代表性的数值。宏平均的计算公式如下：

$$
\bar{x} = \frac{\sum_{i=1}^{n} w_i x_i}{\sum_{i=1}^{n} w_i}
$$

其中，$x_i$ 表示数据点的值，$w_i$ 表示数据点的权重，$n$ 表示数据点的数量，$\bar{x}$ 表示宏平均值。

## 2.2 宏平均与其他平均值的区别
宏平均与其他平均值的区别在于权重。在宏平均中，数据点的值通过权重进行加权平均计算，从而得到一个代表性的数值。而在其他平均值中，如简单平均值和数值平均值，数据点的值通过等权平均计算，从而得到一个代表性的数值。

```java
public class MacroAverage {
    public static double macroAverage(double[] values, double[] weights) {
        double sum = 0.0;
        for (int i = 0; i < values.length; i++) {
            sum += values[i] * weights[i];
        }
        double average = sum / Arrays.stream(weights).sum();
        return average;
    }

    public static void main(String[] args) {
        double[] values = {1, 2, 3, 4, 5};
        double[] weights = {1, 2, 3, 4, 5};

        double result = macroAverage(values, weights);
        System.out.println(result);
    }
}
```
