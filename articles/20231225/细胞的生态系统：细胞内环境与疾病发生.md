
2. 定义系统生物学模型的数学模型公式：

```python
def model(t, X):
    x1, x2, x3 = X
    dx1dt = -a1 * x1 + a2 * x2
    dx2dt = a1 * x1 - a2 * x2
    dx3dt = a3 * x2 - a4 * x3

```

```python
t, X = spi.odeint(model, x0, [t0, tf, dt])
```

```python
plt.title('System Biological Model')
```
