游戏AI的发展历程可以分为以下几个阶段：

1. 早期游戏AI（1970年代至1980年代）：这一阶段的游戏AI主要是通过简单的规则和状态机来控制游戏角色的行为。这些规则和状态机通常是由游戏设计者手动编写的。

2. 基于规则的游戏AI（1990年代）：随着计算机的发展，游戏AI开始使用更复杂的规则和知识表示，以便更好地模拟人类的思维过程。这些规则和知识通常是通过专家系统或知识工程的方式得到编写的。

3. 基于机器学习的游戏AI（2000年代至2010年代）：随着机器学习技术的发展，游戏AI开始使用机器学习算法来学习和优化游戏角色的行为。这些算法通常包括决策树、神经网络、支持向量机等。

4. 现代游戏AI（2010年代至今）：目前的游戏AI已经开始使用深度学习和其他高级技术，以便更好地模拟人类的精神和情感。这些技术包括卷积神经网络、递归神经网络、自然语言处理等。

```python
class GameCharacter:
    def __init__(self):
        self.health = 100
        self.energy = 100

    def decide(self, situation):
        if situation == "enemy_attack":
            if self.energy > 50:
                return "attack_enemy"
            else:
                return "defend"
        elif situation == "friendly_fire":
            return "avoid"
        elif situation == "low_health":
            return "heal"
        else:
            return "idle"
```

在这个代码实例中，我们定义了一个名为`GameCharacter`的类，它有两个属性：`health`和`energy`。这两个属性分别表示角色的生命值和能量值。然后，我们定义了一个名为`decide`的方法，它接受一个`situation`参数，并根据不同的情境返回不同的决策。

例如，如果情境是敌人攻击，并且角色的能量值大于50，那么角色将选择攻击敌人。如果角色的能量值小于50，那么角色将选择防御。如果情境是友方火，那么角色将选择避免。如果情境是角色的生命值低，那么角色将选择治疗。如果情境不在这些中间，那么角色将选择保持静止。
