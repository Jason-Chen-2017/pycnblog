$$
R = \{ (``age'', (18, 25)) \}

接下来，我们将数据集$D$应用于规则$R$：

$$
D_{filtered} = D \times R
$$

最后，我们评估过滤后的数据质量。这里我们只关注过滤条件是否满足，因此我们可以简单地计算过滤后的数据集的比例：

$$
\frac{|D_{filtered}|}{|D|} = \frac{10000}{100000} = 0.1

这表明我们成功地过滤出了10%的用户。

### 代码实例

以下是一个基于规则的数据过滤的Python代码实例：

```python
import pandas as pd

# 加载数据集
D = pd.read_csv("user_behavior.csv")

# 定义规则
R = { "age": (18, 25) }

# 应用规则

# 计算过滤后的数据集比例

print("过滤后的数据集比例: {:.2f}".format(filtered_ratio))
```
