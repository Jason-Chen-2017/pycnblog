# 2.1 次梯度优化
次梯度优化（CG）算法是一种在线优化算法，它通过使用次梯度来近似目标函数的梯度，从而在大规模优化问题中获得更好的性能。次梯度优化算法的核心思想是通过使用近似梯度来减少计算梯度的开销，从而提高优化算法的速度和效率。

# 2.2 局部搜索与全局搜索
局部搜索和全局搜索是两种不同的优化策略，它们在解决优化问题时采用了不同的方法。局部搜索通过从当前解开始，逐步寻找邻域中更好的解，直到找到一个局部最优解。全局搜索则通过搜索整个解空间，找到全局最优解。

# 3.2 次梯度优化算法具体操作步骤
1. 初始化：选择一个初始解x0，设置步长参数α和舍弃参数σ。
2. 计算次梯度：计算当前解xk的次梯度gk。
3. 更新解：根据次梯度gk和步长参数α更新解xk+1。
4. 舍弃或继续：根据舍弃参数σ决定是否继续搜索。
5. 重复步骤2-4，直到满足终止条件。

# 3.3 数学模型公式
假设目标函数f(x)是二次方程，其梯度为g(x)，次梯度为h(x)，则次梯度优化算法的更新公式为：

$$
x_{k+1} = x_k - \alpha_k h_k
$$

其中，αk是步长参数，hk是次梯度。

# 4.1 次梯度优化算法Python实现
```python
import numpy as np

def f(x):
    return x**2

def g(x):
    return 2*x

def h(x):
    return 2

def cg(x0, alpha, sigma, max_iter):
    k = 0
    x = x0
    while k < max_iter:
        g_k = g(x)
        h_k = h(x)
        d_k = -g_k / h_k
        alpha_k = 0.1
        x_k_plus_1 = x - alpha_k * h_k * d_k
        if f(x_k_plus_1) < f(x) + 1e-6:
            x = x_k_plus_1
        else:
            k += 1
    return x

x0 = 10
alpha = 0.1
sigma = 0.1
max_iter = 100
x_opt = cg(x0, alpha, sigma, max_iter)
print("最优解:", x_opt)
```
