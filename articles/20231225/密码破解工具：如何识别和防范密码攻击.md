密码破解工具主要用于破解和分析密码，它们通常包括以下几个核心概念：

```python
def dict_attack(password_hash, dictionary):
    for word in dictionary:
        if hashlib.sha256(word.encode()).hexdigest() == password_hash:
            return word
    return None

dictionary = ["password", "123456", "qwerty", "admin", "root"]

result = dict_attack(password_hash, dictionary)
print(result)
```
