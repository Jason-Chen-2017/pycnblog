### 4.1.2 代码实例
```python
def find_common_elements(arr1, arr2):
    m, n = len(arr1), len(arr2)
    i, j = 0, 0
    while i < m and j < n:
        if arr1[i] == arr2[j]:
            yield arr1[i]
            i += 1
            j += 1
        elif arr1[i] < arr2[j]:
            i += 1
        else:
            j += 1
```
### 4.1.3 解释说明
在这个例子中，我们将两个有序数组的公共元素问题分解为了子问题，即在两个有序数组中分别查找公共元素。通过递归地解决子问题，我们可以得到最终的解决方案。

def fibonacci(n):
    dp = [0] * (n + 1)
    dp[0] = 0
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n]
```
### 4.2.3 解释说明

def max_subarray_sum(arr):
    max_sum = float('-inf')
    current_sum = 0
    for x in arr:
        current_sum = max(0, current_sum + x)
        max_sum = max(max_sum, current_sum)
    return max_sum
```
### 4.3.3 解释说明
