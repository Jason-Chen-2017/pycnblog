## 3.1 聚类分析

聚类分析是一种用于分析数据集中的数据点相似性的方法，可以帮助我们发现数据中的模式和规律。常见的聚类分析算法有K均值聚类、DBSCAN聚类等。

### 3.1.1 K均值聚类

K均值聚类是一种不依赖距离矩阵的聚类算法，通过迭代的方式将数据点分为K个类别。算法的核心步骤如下：

1.随机选择K个类别中心。
2.将每个数据点分配到与其距离最近的类别中心。
3.重新计算每个类别中心的位置，使其位于该类别中的数据点的平均位置。
4.重复步骤2和3，直到类别中心的位置不再变化或达到最大迭代次数。

### 3.1.2 DBSCAN聚类

DBSCAN（Density-Based Spatial Clustering of Applications with Noise）聚类算法是一种基于密度的聚类算法，可以发现不同形状和大小的簇。算法的核心步骤如下：

1.随机选择一个数据点，将其标记为已访问。
2.找到与该数据点距离不超过阈值的其他数据点，将它们标记为已访问。
3.如果已访问的数据点数量达到阈值，则将这些数据点分配到一个新的簇中。
4.重复步骤1和2，直到所有数据点都被访问。

## 3.2 预测分析

### 3.2.1 线性回归

线性回归是一种用于预测连续变量的方法，通过拟合数据中的线性关系来预测未来的值。算法的核心步骤如下：

1.将数据分为训练集和测试集。
2.使用训练集中的输入变量（特征）来预测输出变量（标签）。
3.计算预测值与实际值之间的误差，并使用梯度下降法优化模型参数。
4.使用测试集评估模型的性能。

### 3.2.2 逻辑回归

## 4.1 聚类分析

假设我们有一组医疗数据，包括患者的年龄、体重、血压等信息。我们可以使用K均值聚类算法将这些数据分为不同的类别，以便更好地了解患者的健康状况。

```python
from sklearn.cluster import KMeans
import numpy as np

# 加载数据
data = np.loadtxt('medical_data.txt', delimiter=',')

# 使用K均值聚类算法将数据分为2个类别
kmeans = KMeans(n_clusters=2)
kmeans.fit(data)

# 输出类别中心
print(kmeans.cluster_centers_)

# 输出每个数据点所属的类别
print(kmeans.labels_)
```

## 4.3 推荐系统

```python
# 加载数据

# 使用TfidfVectorizer将数据转换为向量

# 输出相似度矩阵
print(similarity)

# 根据相似度推荐产品

```
