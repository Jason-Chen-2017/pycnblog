# 2.核心概念与联系
移动应用漏洞扫描是一种自动化的安全测试方法，旨在检测移动应用中的漏洞和安全问题。这些漏洞可能包括 SQL 注入、跨站脚本攻击（XSS）、不安全的数据传输等。通过移动应用漏洞扫描，我们可以发现这些漏洞，并采取措施来修复它们，从而保护用户数据免受损失。

移动应用漏洞扫描可以分为两种类型：静态应用程序扫描（SAST）和动态应用程序扫描（DAST）。静态应用程序扫描是一种在不运行应用程序的情况下对其代码进行分析的方法。动态应用程序扫描则是在运行应用程序的情况下对其进行监控和分析的方法。这两种方法各有优劣，通常情况下我们需要结合使用，以获得更全面的安全保护。
