在这个例子中，我们将使用LUKASIEWICZ模糊逻辑运算符进行运算。LUKASIEWICZ模糊逻辑运算符的定义如下：

$$
\mu_{A\oplus B}(x) = \max(\mu_A(x) \oplus \mu_B(x))
$$

其中，$\mu_A(x)$和$\mu_B(x)$是模糊集A和模糊集B的隶属度函数，$\oplus$是模糊逻辑运算符。

```python
import numpy as np

def lukasiewicz_or(a, b):
    return np.maximum(a, b)

def lukasiewicz_and(a, b):
    return a * b

def lukasiewicz_not(a):
    return 1 - a

def lukasiewicz_implication(a, b):
    return lukasiewicz_not(a) or b

def lukasiewicz_equivalence(a, b):
    return not lukasiewicz_implication(a, b) and not lukasiewicz_implication(b, a)
```

其中，$I(x)$是预测的空气质量指数，$a_1$和$a_0$是线性插值方程的参数，可以通过最小化预测误差得到。

def linear_interpolation(x, y):
    x = np.sort(x)
    y = np.sort(y)
    index = np.searchsorted(x, y)
    a_1 = (x[index + 1] - x[index]) / (y[index + 1] - y[index])
    a_0 = y[index] - a_1 * x[index]
    return a_1, a_0
