# 4.1索引
我们可以使用以下代码来创建一个索引：
```python
import numpy as np

def create_index(data):
    index = {}
    for i, item in enumerate(data):
        for key, value in item.items():
            if key not in index:
                index[key] = []
            index[key].append(i)
    return index

data = [{'name': 'John', 'age': 25}, {'name': 'Jane', 'age': 30}, {'name': 'Doe', 'age': 28}]
index = create_index(data)
print(index)
```
在上面的代码中，我们首先导入了numpy库，然后定义了一个create_index函数，该函数接受一个数据列表作为输入，并创建一个索引字典。在函数中，我们遍历数据列表中的每个元素，并将其中的键值对添加到索引字典中。最后，我们打印了索引字典。
