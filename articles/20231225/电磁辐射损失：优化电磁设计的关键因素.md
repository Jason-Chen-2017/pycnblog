# 2.核心概念与联系
电磁辐射损失是指在电磁设计中，由于电磁波在空气中的辐射损耗，导致设备输出功率降低的现象。电磁辐射损失主要由以下几个因素造成：

1. 电磁波的长度：电磁波的长度会影响其辐射损耗。长波长的电磁波辐射损耗较小，短波长的电磁波辐射损耗较大。
2. 电磁波的强度：电磁波的强度会影响其辐射损耗。强电磁波辐射损耗较大，弱电磁波辐射损耗较小。
3. 电磁波的方向：电磁波的方向会影响其辐射损耗。垂直方向的电磁波辐射损耗较小，水平方向的电磁波辐射损耗较大。
