## 4.5 地图
以下是一个简单的地图示例，展示了一组数字的数据：

```python
import matplotlib.pyplot as plt
import geopandas as gpd

# 下载地图数据
url = 'https://raw.githubusercontent.com/johannesbauer/geopandas-examples/master/data/world.geojson'
world = gpd.read_file(url)

# 创建地图
ax = world.plot(column='pop2005', legend=True, figsize=(10, 5))

# 添加数据点
df = pd.DataFrame({'country': ['USA', 'China', 'Japan'], 'value': [300, 1300, 120]})
df = gpd.GeoDataFrame(df, geometry=gpd.points_from_xy(df.longitude, df.latitude))
df.plot(ax=ax, markersize=10, color='red')

plt.show()
```
