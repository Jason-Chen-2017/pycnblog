## 4.具体代码实例和详细解释说明

在这里，我们给出一个简化的冰川融化模型的Python代码实例。

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.ndimage import gaussian_filter

# 初始温度分布
T0 = np.ones((50, 50))

# 时间步长
dt = 0.1

# 热导率
alpha = 0.1

# 外界热源
Q = np.zeros((50, 50))

# 时间步长循环
for t in range(1000):
    T = T0 + alpha * dt * gaussian_filter(T0, sigma=1) + dt * Q
    T0 = T

# 绘制温度分布
plt.imshow(T, cmap='hot', extent=[0, 50, 0, 50])
plt.colorbar()
plt.show()
```

这个代码实例使用了Python的NumPy和Matplotlib库来实现简化的冰川融化模型。首先，我们定义了初始温度分布`T0`、时间步长`dt`、热导率`alpha`和外界热源`Q`。然后，我们进行时间步长循环，在每一步中更新冰川表面温度`T`。最后，我们使用Matplotlib库绘制温度分布。
