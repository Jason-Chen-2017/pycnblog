# 2.核心概念与联系

## 2.1 凸函数的定义

凸函数是一种特殊的函数，它在整个定义域内具有最小值。更正式地说，对于一个实值函数f(x)，如果对于任何给定的x1和x2，以及对应的y1=f(x1)和y2=f(x2)，则对于任何0≤λ≤1，有f(λx1+(1-λ)x2)≤λy1+(1-λ)y2。这个定义意味着，对于凸函数，函数值在给定域内的变化是有界的，且变化趋向于最小。

## 2.2 凸函数的性质

凸函数具有以下几个重要性质：

1. 凸函数在内部有最小值，在边界处有最小值或最大值。
2. 凸函数的梯度（导数）是凸的。
3. 凸函数的Hess矩阵（二阶导数矩阵）是对称的。
4. 凸函数在局部最小值处，梯度为0。
5. 凸函数在局部最大值处，梯度为0。

## 2.3 凸函数与非凸函数的区别

非凸函数是不满足凸函数定义的函数。它们在某些区域内可能具有多个局部最小值或最大值，或者在内部没有最小值。这种情况使得非凸函数在优化问题中更加复杂，需要更加复杂的算法来解决。

## 2.4 凸函数与线性函数的联系

线性函数是凸函数的特殊情况。线性函数在整个定义域内具有最小值，且梯度是常数。因此，线性函数在内部具有唯一的最小值，在边界处有最小值或最大值。
