# 2.核心概念与联系

## 2.1区块链技术简介

区块链技术是一种基于分布式、去中心化的数据存储和传输方式，它的核心概念是将数据以“区块”的形式存储和传输，每个区块包含一组交易数据，这些交易数据是有序的，每个区块的数据是与前一个区块的数据有关，形成了一条链。区块链技术的核心特点是数据的不可篡改性和不可抵赖性，因为每个区块的数据都是通过加密算法加密后的，并且每个区块的数据都是通过多个节点的计算结果得到的，因此，任何一个节点都无法单独改动或欺骗数据。

## 2.2医疗保健数据安全与共享的挑战

区块链算法的核心原理是基于一种称为“共识算法”的机制，共识算法的核心思想是通过多个节点的计算结果得到一致的结果，从而确保数据的不可篡改性和不可抵赖性。共识算法的核心步骤如下：

1.节点之间通过网络进行交流，交流的内容是各个节点的计算结果。

2.节点之间通过比较各个节点的计算结果，找到一致的结果。

3.一致的结果被确认为是正确的结果。

class Blockchain:
    def __init__(self):
        self.chain = []

    def create_block(self, proof, previous_hash):
        block = {
            'index': len(self.chain) + 1,
            'timestamp': time.time(),
            'proof': proof,
            'previous_hash': previous_hash
        }
        self.chain.append(block)
        return block

    def get_last_block(self):
        return self.chain[-1]

    def is_valid(self, block, previous_block):
        if block['index'] != previous_block['index'] + 1:
            return False
        if block['timestamp'] > previous_block['timestamp']:
            return False
        if block['proof'] > previous_block['proof']:
            return False
        if block['previous_hash'] != previous_block['hash']:
            return False
        return True

    def proof_of_work(self, last_block):
        proof = 0
        while self.valid_proof(last_block, proof) is False:
            proof += 1
        return proof

    def valid_proof(self, last_block, proof):
        if str(proof)[:4] != str(last_block['proof'][:4]):
            return False
        block_hash = hashlib.sha256(str(last_block['index'] + last_block['timestamp'] + str(proof)).encode()).hexdigest()
        if block_hash[:4] != str(last_block['previous_hash'][:4]):
            return False
        return True

    def add_block(self, proof, previous_hash):
        new_block = self.create_block(proof, previous_hash)
        self.chain.append(new_block)
        return new_block

    def add_transaction(self, sender, recipient, amount):
        transaction = {
            'sender': sender,
            'recipient': recipient,
            'amount': amount
        }
