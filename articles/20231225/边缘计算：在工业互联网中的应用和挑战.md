                 

# 1.背景介绍

边缘计算是一种在设备上进行数据处理和分析的计算模型，它的核心思想是将数据处理和分析任务推向设备边缘，从而减少数据传输和计算负载，提高系统效率和降低延迟。在工业互联网中，边缘计算已经成为一个重要的技术趋势和应用场景。

工业互联网是指通过互联网技术将传统行业与信息技术相结合，实现企业业务流程的数字化、智能化和网络化的过程。工业互联网涉及到的领域非常广泛，包括制造业、能源、交通运输、医疗健康等。在这些领域，数据量巨大，实时性要求严格，计算需求巨大，因此边缘计算在工业互联网中具有重要的应用价值。
