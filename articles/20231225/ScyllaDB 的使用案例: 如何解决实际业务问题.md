# 2. 核心概念与联系

ScyllaDB 的核心概念包括：

1. 分布式数据存储：ScyllaDB 是一个分布式数据库系统，它可以在多个节点上存储和处理数据。这种分布式架构可以提高数据库的可扩展性和性能。

2. 快速路由：ScyllaDB 使用了一种称为“快速路由”的技术，该技术可以在数据库中实现高性能的数据处理和查询。快速路由技术可以减少数据库中的网络延迟，并提高数据处理的速度。

3. 数据模型：ScyllaDB 支持多种数据模型，包括关系型数据模型和非关系型数据模型。这种多种数据模型的支持可以满足不同类型的应用需求。

4. 数据一致性：ScyllaDB 使用了一种称为“一致性一写”的技术，该技术可以确保在分布式环境中实现数据的一致性。

首先，我们需要创建一个购物车表：

```

    id UUID PRIMARY KEY,
    user_id UUID,
    product_id UUID,
    quantity INT,
    created_at TIMESTAMP
);
```

接下来，我们可以使用 ScyllaDB 的 CQL（Cassandra Query Language）来插入和查询购物车数据。例如，我们可以使用以下 CQL 命令来插入一个新的购物车项：

```

VALUES (uuid(), uuid(), uuid(), 1, toTimestamp(now()));
```

我们还可以使用以下 CQL 命令来查询一个用户的购物车数据：

```
SELECT * FROM shopping_cart WHERE user_id = uuid();
```
