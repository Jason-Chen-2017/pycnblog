# 2.核心概念与联系

## 2.1 齐次无序单项式向量空间（Homogeneous Unordered Point Set, HUPS）

HUPS 是一种新的数据结构，用于表示和处理无序的点集。它的核心概念是将点集表示为一个多项式，这个多项式的系数表示点的数量，而点本身则表示为多项式的指数。HUPS 可以用于表示和处理各种类型的点集，包括二维点集、三维点集等。

HUPS 的构建主要包括以下几个步骤：

2. 对多项式进行哈希：对转换后的多项式进行哈希，以便于快速查找和比较。
3. 构建 HUPS 数据结构：将哈希后的多项式存储到 HUPS 数据结构中，以便于后续的查找和比较操作。

HUPS 的查找主要包括以下几个步骤：

2. 在 HUPS 数据结构中查找：将哈希后的多项式与 HUPS 数据结构中的多项式进行比较，如果存在相同的多项式，则表示输入的点集与 HUPS 数据结构中的点集相同。

HUPS 的更新主要包括以下几个步骤：

import numpy as np
from hashlib import sha256

class HUPS:
    def __init__(self):
        self.data = {}

    def build(self, points):
        # 将点集转换为多项式
        polynomial = np.poly1d(points)
        # 对多项式进行哈希
        hash_value = sha256(polynomial.bytes()).hexdigest()
        # 构建 HUPS 数据结构
        self.data[hash_value] = polynomial

    def find(self, points):
        # 对输入的点集进行哈希
        hash_value = sha256(np.poly1d(points).bytes()).hexdigest()
        # 在 HUPS 数据结构中查找
        return self.data.get(hash_value, None)

    def update(self, points):
        # 对输入的点集进行哈希
        hash_value = sha256(np.poly1d(points).bytes()).hexdigest()
        # 在 HUPS 数据结构中更新
        self.data[hash_value] = np.poly1d(points)

# 创建一个 HUPS 实例
hups = HUPS()

# 构建 HUPS 数据结构
points = np.array([1, 2, 3])
hups.build(points)

# 查找点集
points2 = np.array([1, 2, 3])
result = hups.find(points2)
print(result)  # 输出: 1x3 Poly1d([1 2 3])

# 更新点集
points3 = np.array([1, 2, 4])
hups.update(points3)
result = hups.find(points3)
print(result)  # 输出: 1x3 Poly1d([1 2 4])
