# 2.核心概念与联系

下降迭代法（Descent Iteration）是一种优化算法，它通过逐步减小目标函数的梯度来找到最小值。在自动驾驶中，下降迭代法可以用于优化路径规划和控制问题。

下降迭代法的核心概念包括：

- 目标函数：表示需要最小化的量，如路径长度、时间等。
- 梯度：目标函数的一阶导数，表示函数在某一点的增长方向。
- 迭代：通过更新参数，逐步减小梯度，找到最小值。
