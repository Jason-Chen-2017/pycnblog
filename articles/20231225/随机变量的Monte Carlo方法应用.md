# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个具体的代码实例来解释Monte Carlo方法的应用。

例子：求解积分 $\int_0^1 x^2 dx$ 的值。

```python
import numpy as np

def integrate_mc(n_samples):
    x_values = np.random.uniform(0, 1, n_samples)
    y_values = x_values**2
    return np.mean(y_values)

n_samples = 100000
result = integrate_mc(n_samples)
print(f"Monte Carlo method result: {result}")
```

在这个例子中，我们使用了Monte Carlo方法来求解积分 $\int_0^1 x^2 dx$ 的值。我们首先生成了100000个随机的x值，然后计算它们的平方，最后通过求均值来得出结论。通过比较Monte Carlo方法的结果和真实的积分结果，可以看到Monte Carlo方法的准确性。
