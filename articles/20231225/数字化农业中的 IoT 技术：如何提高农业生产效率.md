# 4.具体代码实例和详细解释说明
在这里，我们将给出一个简单的Python代码实例，以展示如何使用IoT技术来监测气候。

```python
import time
import requests

# 设置气候传感器的API地址
weather_api_url = "http://api.openweathermap.org/data/2.5/weather?q=beijing&appid=YOUR_API_KEY"

# 循环获取气候数据
while True:
    response = requests.get(weather_api_url)
    weather_data = response.json()
    temperature = weather_data["main"]["temp"]
    humidity = weather_data["main"]["humidity"]
    print("当前气温：", temperature, "度")
    print("当前湿度：", humidity, "%")
    time.sleep(60)
```

在这个代码实例中，我们使用了Python的requests库来获取气候数据。我们设置了一个气候传感器的API地址，并使用while循环来不断获取气候数据。我们从获取到的数据中提取了气温和湿度，并将它们打印出来。
