## 4.1 图数据模型的实现
我们将使用Python的NetworkX库来实现图数据模型。首先，我们需要创建一个图对象，并添加节点和边。

```python
import networkx as nx

G = nx.Graph()

G.add_node(1)
G.add_node(2)
G.add_node(3)

G.add_edge(1, 2)
G.add_edge(2, 3)
```
