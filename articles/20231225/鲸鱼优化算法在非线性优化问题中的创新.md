# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明鲸鱼优化算法的应用。我们将使用鲸鱼优化算法来解决一维非线性优化问题：

$$
\min f(x) = -x^2 \sin(x) + 5x - 4
$$

其中，$x \in [-10, 10]$。

```python
import numpy as np

def f(x):
    return -x**2 * np.sin(x) + 5 * x - 4

def whale_optimization_algorithm(f, bounds, n_whales, n_iterations):
    # 初始化鲸鱼群的位置
    X = np.random.uniform(bounds[0], bounds[1], n_whales)
    # 初始化鲸鱼群的速度
    V = np.zeros(n_whales)
    # 初始化最佳鲸鱼的位置和值
    best_X = X[np.argmin(f(X))]
    best_f = f(best_X)

    for t in range(n_iterations):
        # 计算鲸鱼群的目标函数值
        f_values = f(X)
        # 找到鲸鱼群中的最佳鲸鱼
        best_index = np.argmin(f_values)
        best_X = X[best_index]
        best_f = f_values[best_index]
        # 更新鲸鱼群的位置和速度
        for i in range(n_whales):
            V[i] = V[i] + C * np.random.rand() * (best_X - X[i]) - best_X * np.abs(C * np.random.rand() - 0.5)
            X[i] = X[i] + V[i]

    return best_X, best_f

n_whales = 50
n_iterations = 100
bounds = (-10, 10)
best_X, best_f = whale_optimization_algorithm(f, bounds, n_whales, n_iterations)
print("最佳鲸鱼的位置：", best_X)
print("最佳鲸鱼的目标函数值：", best_f)
```

在上面的代码实例中，我们首先定义了目标函数$f(x)$，并设定了鲸鱼群的大小$n_{whales}$和迭代次数$n_{iterations}$。然后，我们使用鲸鱼优化算法来求解这个问题，并输出了最佳鲸鱼的位置和目标函数值。
