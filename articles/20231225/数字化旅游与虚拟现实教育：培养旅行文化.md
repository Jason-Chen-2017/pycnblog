# 创建场景
scene = bpy.context.scene

# 创建地面
mesh = bpy.data.meshes.new("ground")
obj = bpy.data.objects.new("ground", mesh)
scene.collection.objects.link(obj)

# 创建山脉
mesh = bpy.data.meshes.new("mountain")
obj = bpy.data.objects.new("mountain", mesh)
scene.collection.objects.link(obj)

# 设置山脉的形状
verts = [(0, 0, 0), (10, 0, 0), (10, 10, 0), (0, 10, 0)]
edges = []
faces = [(0, 1, 2, 3)]

# 创建山脉的网格
mesh.from_pydata((verts, edges, faces), "quads")
mesh.update()

# 6.附录常见问题与解答

1. Q: 虚拟现实技术与传统旅游有什么区别？

2. Q: 虚拟现实教育与传统教育有什么区别？
A: 虚拟现实教育可以为学生提供一种全新的学习体验，让他们在虚拟环境中学习相关知识。而传统教育则需要学生在实际环境中学习。

3. Q: 数字化旅游与传统旅游有什么区别？
A: 数字化旅游利用信息技术和互联网等新技术手段，为旅游行业提供新的服务和产品。而传统旅游则依赖于传统的旅游方式，如人工售票、纸质旅游资料等。

4. Q: 虚拟现实教育与数字化旅游有什么区别？
A: 虚拟现实教育是利用虚拟现实技术，为学生提供一种全新的学习体验。而数字化旅游则是利用信息技术和互联网等新技术手段，为旅游行业提供新的服务和产品。它们的主要目标和应用场景不同。
