### 1.4.1 最小二多项式实例

```python
import numpy as np

def least_squares(A, b):
    A_T = A.T
    ATb = A_T @ b
    AAT = A_T @ A
    x = np.linalg.solve(AAT, ATb)
    return x

A = np.array([[1, 1], [1, -1], [2, 1]])
b = np.array([3, 0, 2])
x = least_squares(A, b)
print(x)
```

在上面的代码中，我们使用了最小二多项式算法来解决线性方程组问题。首先，我们定义了一个矩阵A和向量b。然后，我们使用了numpy库中的linalg.solve函数来解决线性方程组。最后，我们打印了解决后的结果。
