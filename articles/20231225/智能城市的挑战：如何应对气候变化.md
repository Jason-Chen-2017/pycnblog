## 4.1 气候变化预测
以下是一个基于支持向量机的气候变化预测的Python代码实例：

```python
from sklearn import svm
import numpy as np
import pandas as pd

# 加载气候数据
data = pd.read_csv('climate_data.csv')

# 数据预处理
X = data.drop('temperature', axis=1)
y = data['temperature']

# 训练模型
model = svm.SVR(kernel='linear')
model.fit(X, y)

# 预测气候变化
X_test = np.array([[20, 10, 5, 3]])
y_pred = model.predict(X_test)
print('预测气温:', y_pred[0])
```

## 4.2 优化城市能源利用
以下是一个基于遗传算法的城市能源优化的Python代码实例：

```python
import numpy as np
import random

# 定义城市能源优化问题
def fitness_function(x):
    # 计算能源消耗
    energy_consumption = x[0] * x[1] * x[2]
    return energy_consumption

# 初始化种群
population_size = 100
population = np.random.rand(population_size, 3)

# 遗传算法主循环
for generation in range(1000):
    # 选择
    selected_individuals = np.array(sorted(population, key=fitness_function))[:population_size // 2]
    
    # 交叉
    offspring = []
    for i in range(0, population_size, 2):
        crossover_point = random.randint(1, 2)
        offspring.append(np.concatenate((selected_individuals[i][:crossover_point], selected_individuals[i+1][crossover_point:])))
    
    # 变异
    mutation_rate = 0.1
    for individual in offspring:
        mutation_point = random.randint(0, 2)
        individual[mutation_point] = random.randint(0, 100)
    
    # 替代
    population = np.vstack((selected_individuals, np.array(offspring)))
    
    # 评估适应度
    fitness_values = np.array(list(map(fitness_function, population)))
    print('第{}代适应度: {}'.format(generation, fitness_values.mean()))

# 得到最优解
best_individual = population[np.argmin(fitness_values)]
print('最优解:', best_individual)
```
