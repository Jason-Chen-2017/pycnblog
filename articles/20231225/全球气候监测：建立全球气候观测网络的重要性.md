# 4.具体代码实例和详细解释说明

在这里，我们将提供一个简单的Python代码实例，用于计算气温和风速的分布。

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机气温和风速数据
np.random.seed(0)
T = np.random.rand(100, 100)
V = np.random.rand(100, 100)

# 计算气温分布
plt.figure(figsize=(8, 6))
plt.imshow(T, cmap='viridis', extent=(0, 100, 0, 100))
plt.colorbar()
plt.xlabel('Longitude')
plt.ylabel('Latitude')
plt.title('Temperature Distribution')
plt.show()

# 计算风速分布
plt.figure(figsize=(8, 6))
plt.imshow(V, cmap='jet', extent=(0, 100, 0, 100))
plt.colorbar()
plt.xlabel('Longitude')
plt.ylabel('Latitude')
plt.title('Wind Speed Distribution')
plt.show()
```

这个代码实例首先导入了`numpy`和`matplotlib.pyplot`库，然后生成了随机的气温和风速数据。接着，使用`matplotlib.pyplot`库绘制了气温和风速的分布图。
