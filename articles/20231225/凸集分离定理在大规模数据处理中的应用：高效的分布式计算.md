# 2.核心概念与联系

## 2.1 凸集分离定理

凸集分离定理是一种用于解决多类别分类问题的方法，它的核心思想是将多个类别之间的分界线表示为凸集。具体来说，凸集分离定理可以用于解决以下问题：

- 二分类问题：将两个类别之间的分界线表示为一个凸集，从而可以用于解决二分类问题。
- 多类别分类问题：将多个类别之间的分界线表示为多个凸集，从而可以用于解决多类别分类问题。

## 2.2 分布式计算

分布式计算是一种在多个计算节点上并行执行的计算方法，它可以用于解决大规模数据处理中的问题。分布式计算的主要特点是：

- 并行性：多个计算节点同时执行任务，从而提高计算效率。
- 分布式性：计算节点分布在不同的地理位置，可以在网络中进行数据传输和通信。
- 可扩展性：通过增加计算节点，可以实现系统的扩展。
