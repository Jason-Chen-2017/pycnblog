### 2.1 概率空间

概率空间是概率论中的基本概念，是一个包含所有可能结果的集合，以及每个结果发生的概率。在计算几何中，概率空间可以是一个几何图形的集合，每个图形都有一个相应的概率。

### 2.3 条件概率

条件概率是一个随机事件发生的概率，给定另一个随机事件已发生的情况下。在计算几何中，条件概率可以用来评估一个给定属性的几何图形的概率。

### 2.4 独立性

```python
import math

def random_point_in_circle(radius):
    x = random.uniform(-radius, radius)
    y = random.uniform(-radius, radius)
    return (x, y)

def random_polygon(n, radius, center):
    points = []
    for i in range(n):
        angle = 2 * math.pi * i / n
        x = center[0] + radius * math.cos(angle)
        y = center[1] + radius * math.sin(angle)
        points.append((x, y))
    return points
```

```python
def random_nearest_pair(points, n):
    pairs = []
    for i in range(n):
        for j in range(i + 1, n):
            distance = math.sqrt((points[i][0] - points[j][0])**2 + (points[i][1] - points[j][1])**2)
            pairs.append((distance, (points[i], points[j])))
    return min(pairs)
```

在这里，我们将列出一些常见问题及其解答。

### Q1: 如何生成一个圆内的随机点？

A1: 可以使用 `random_point_in_circle` 函数，它接受圆的半径作为参数，并返回一个圆内的随机点。

### Q2: 如何在一个给定区域内随机生成多个点？

A2: 可以使用 `random_sample_in_circle` 函数，它接受圆的半径和生成点的数量作为参数，并返回一个包含多个圆内随机点的列表。

### Q3: 如何在一个给定区域内随机生成多个几何图形？

A3: 可以使用 `random_sample_in_polygon` 函数，它接受几何图形的参数、生成图形的数量和中心点作为参数，并返回一个包含多个随机生成的几何图形的列表。

### Q4: 如何在一个给定区域内找到最近的两个点？

A4: 可以使用 `random_nearest_pair` 函数，它接受点的列表和生成最近点对的数量作为参数，并返回一个包含最近点对的列表。

### Q5: 如何在一个给定区域内生成一个凸包？

A5: 可以使用 `random_convex_hull` 函数，它接受点的列表和生成凸包的数量作为参数，并返回一个包含凸包点的列表。
