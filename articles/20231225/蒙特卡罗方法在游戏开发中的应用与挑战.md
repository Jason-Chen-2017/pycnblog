# 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来解释蒙特卡罗方法的使用：

```python
import random
import numpy as np

def target_function(x):
    return x**2

def random_sampling_strategy():
    return random.uniform(0, 1)

N = 10000
x_values = [random_sampling_strategy() for _ in range(N)]
y_values = [target_function(x) for x in x_values]

X = np.mean(y_values)
print("Estimated value of X:", X)
```

在这个例子中，我们定义了一个目标函数f(x) = x**2，并通过随机生成10000个随机变量来估计X的值。通过计算随机变量的平均值，我们可以得到X的估计值。
