1. 创建一个 FaunaDB 数据库实例。
2. 定义一个数据模型，例如文档、关系或图形模型。
3. 使用计算式查询语言编写查询。

数学模型公式详细讲解：

计算式查询语言的基本数学模型公式如下：

$$
f(x) = y
$$

其中，$f(x)$ 表示函数，$x$ 表示输入变量，$y$ 表示输出变量。这个公式表示函数$f(x)$ 对于给定的输入变量$x$ 会产生一个输出变量$y$ 。

```python

document = client.create_document({
    "collection": "users",

        "name": "John Doe",
        "email": "john.doe@example.com"
    }

```

