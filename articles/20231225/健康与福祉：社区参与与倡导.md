## 1.4 具体代码实例和详细解释说明

在这里，我们给出一个简单的Python代码实例，用于计算社区参与与倡导的度量：

```python
import numpy as np
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 计算参与率
participation_rate = data['participation'] / data['total']

# 计算活跃度
activity = data['activity']

# 计算平均参与率和平均活跃度
average_participation_rate = np.mean(participation_rate)
average_activity = np.mean(activity)

print('平均参与率：', average_participation_rate)
print('平均活跃度：', average_activity)
```

这个代码实例首先读取了一个CSV文件，然后计算了参与率和活跃度，最后计算了平均参与率和平均活跃度。
