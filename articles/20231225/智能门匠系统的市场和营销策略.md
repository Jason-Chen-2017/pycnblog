## 4.1 门锁控制代码实例

以下是一个简单的门锁控制代码实例，使用Python编程语言：

```python
import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
GPIO.setup(23, GPIO.OUT)

try:
    while True:
        command = input("请输入门锁控制指令（open/close）：")
        if command == "open":
            GPIO.output(23, GPIO.HIGH)
            print("门锁已开锁")
        elif command == "close":
            GPIO.output(23, GPIO.LOW)
            print("门锁已关锁")
        else:
            print("无效指令")
except KeyboardInterrupt:
    GPIO.cleanup()
```

这个代码实例使用Python的GPIO库实现了门锁的开关控制。在代码中，我们首先设置了GPIO库的工作模式，并配置了门锁控制的引脚。在主循环中，我们接收用户输入的控制指令，根据指令执行门锁的开关操作。

```python
    if not data:

    if command == "open":
        client_sock.send("门锁已开锁".encode("utf-8"))
    elif command == "close":
        client_sock.send("门锁已关锁".encode("utf-8"))

        client_sock.send("无效指令".encode("utf-8"))

client_sock.close()
server_sock.close()
```
