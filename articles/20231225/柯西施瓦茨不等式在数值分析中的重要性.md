# 2.核心概念与联系

柯西-施瓦茨不等式是一个关于向量和矩阵的不等式，它表示了向量之间的关系。它的基本形式如下：

$$
(\sum_{i=1}^{n} a_i^2)(\sum_{i=1}^{n} b_i^2) \geq (\sum_{i=1}^{n} a_i b_i)^2
$$

其中，$a_i$ 和 $b_i$ 是实数序列，$n$ 是序列的长度。

```python
def gradient_descent(f, x0, alpha, max_iter):
    x_k = x0
    for k in range(max_iter):
        grad = f(x_k)
        grad_norm_squared = np.sum(grad**2)
        x_k_plus_1 = x_k - alpha * grad
        x_k_plus_1_norm_squared = np.sum(x_k_plus_1**2)
        csi = grad_norm_squared * x_k_plus_1_norm_squared
        if k > 0 and csi < 0:
            raise ValueError("CS inequality violated")
        x_k = x_k_plus_1
    return x_k
```
