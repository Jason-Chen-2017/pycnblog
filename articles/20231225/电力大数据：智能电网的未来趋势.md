# 4. 具体代码实例和详细解释说明

在这一节中，我们将通过一个具体的代码实例来详细解释如何对电力大数据进行处理、分析和挖掘。

## 4.1 代码实例

我们以一个简单的电力需求预测问题为例。假设我们有一个历史电力需求数据集，包括时间、天气、节假日等特征。我们的目标是使用这些特征来预测未来一周的电力需求。

首先，我们需要将数据存储到Hadoop中：

```python
from hadoop.fileimport import FileImport

# 创建一个Hadoop文件导入对象
fi = FileImport('electric_demand.csv')

# 将数据导入Hadoop
fi.import_data()
```

接下来，我们需要对数据进行处理和清洗：

```python
from pyspark.sql import SparkSession

# 创建一个SparkSession对象
spark = SparkSession.builder.appName('electric_demand_prediction').getOrCreate()

# 读取Hadoop中的数据
df = spark.read.csv('electric_demand.csv', header=True, inferSchema=True)

# 对数据进行处理和清洗
df = df.dropna()
df = df.withColumn('date', df['date'].cast('date'))
df = df.withColumn('weather', df['weather'].cast('int'))
df = df.withColumn('holiday', df['holiday'].cast('int'))
```

然后，我们需要使用机器学习方法进行模型构建：

```python
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error

# 将数据分为特征和标签
X = df[['date', 'weather', 'holiday']]
Y = df['electric_demand']

# 将数据分为训练集和测试集
X_train, X_test, Y_train, Y_test = train_test_split(X, Y, test_size=0.2, random_state=42)

# 使用线性回归进行模型构建
model = LinearRegression()
model.fit(X_train, Y_train)

# 对测试集进行预测
Y_pred = model.predict(X_test)

# 计算预测精度
mse = mean_squared_error(Y_test, Y_pred)
print('MSE:', mse)
```

最后，我们需要对模型进行评估和优化：

```python
from sklearn.metrics import r2_score

# 计算R^2值
r2 = r2_score(Y_test, Y_pred)
print('R^2:', r2)
```

通过这个代码实例，我们可以看到如何对电力大数据进行处理、分析和挖掘。
