# 2.核心概念与联系

## 2.1 泊松分布

泊松分布是一种概率分布，用于描述一定时间内事件发生的次数。它的概率密度函数为：

$$
P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}
$$

其中，$k$ 是事件发生的次数，$\lambda$ 是事件发生率，$e$ 是基数。

```python
import numpy as np
import scipy.stats as stats

# 收集数据
data = {
    'rainy': {'n': 5, 't': 100},
    'sunny': {'n': 2, 't': 100},
}

# 计算发生率
lambda_rainy = data['rainy']['n'] / data['rainy']['t']
lambda_sunny = data['sunny']['n'] / data['sunny']['t']

# 使用泊松分布进行安全性能分析
def poisson_safety_analysis(lambda_value):
    x = np.arange(0, 10)
    p = stats.poisson.pmf(x, lambda_value)
    return x, p

x_rainy, p_rainy = poisson_safety_analysis(lambda_rainy)
x_sunny, p_sunny = poisson_safety_analysis(lambda_sunny)

print("雨天安全性能分析：")
print("事件发生次数，概率：", dict(zip(x_rainy, p_rainy)))
print("晴天安全性能分析：")
print("事件发生次数，概率：", dict(zip(x_sunny, p_sunny)))
```
