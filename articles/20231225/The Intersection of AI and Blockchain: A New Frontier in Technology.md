# 2.核心概念与联系

### 4.1.1 文本分类

文本分类是一种通过计算机处理和分类自然语言文本的方法。以下是一个简单的文本分类代码实例：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import make_pipeline

# 训练数据
train_data = [
    ('这是一个好书', 'positive'),
    ('这是一个很好的书', 'positive'),
    ('这是一个不好的书', 'negative'),
    ('这是一个很不好的书', 'negative')
]

# 测试数据
test_data = ['这是一个棒书', '这是一个糟糕的书']

# 创建文本分类模型
model = make_pipeline(TfidfVectorizer(), MultinomialNB())

# 训练模型
model.fit(train_data)

# 预测
predictions = model.predict(test_data)

print(predictions)
```

## 4.2 区块链代码实例

### 4.2.1 创建一个简单的区块链

```python
import hashlib
import json

class Block:
    def __init__(self, index, prev_hash, timestamp, data, hash):
        self.index = index
        self.prev_hash = prev_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

    def calculate_hash(self):
        block_string = json.dumps(self.__dict__, sort_keys=True).encode()
        return hashlib.sha256(block_string).hexdigest()

# 创建第一个区块
index = 0
prev_hash = "0"
timestamp = "2021-01-01"
data = "Genesis Block"
block = Block(index, prev_hash, timestamp, data, prev_hash)

# 创建第二个区块
index = 1
timestamp = "2021-01-02"
data = "Second Block"
block2 = Block(index, block.hash, timestamp, data, block.calculate_hash())

# 创建一个区块链
blockchain = [block, block2]

print(json.dumps(blockchain, indent=4))
```

        server.listen()

        while True:
            client, address = server.accept()
            print(f'Connected to {address}')
            node_thread = threading.Thread(target=self.handle_connection, args=(client,))
