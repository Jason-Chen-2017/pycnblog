```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
from sklearn.preprocessing import PolynomialFeatures

# 生成一组随机的肌肉力量测量数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 3 * x + 2 + np.random.randn(100, 1)

# 使用多项式回归方法对肌肉力量测量数据进行拟合
poly = PolynomialFeatures(degree=2)
x_poly = poly.fit_transform(x)
model = LinearRegression()
model.fit(x_poly, y)

# 绘制肌肉力量测量数据和拟合曲线
plt.scatter(x, y, label='原始数据')
plt.plot(x, model.predict(poly.fit_transform(x)), label='拟合曲线')
plt.legend()
plt.show()
```

这个代码实例主要包括以下几个步骤：

1. 导入必要的库，包括 `numpy`、`matplotlib` 和 `scikit-learn`。
2. 生成一组随机的肌肉力量测量数据，其中 $x$ 表示肌肉力量测量值，$y$ 表示对应的运动员身高。
3. 使用多项式回归方法对肌肉力量测量数据进行拟合。首先，使用 `PolynomialFeatures` 函数将原始数据转换为多项式特征。然后，使用 `LinearRegression` 函数对转换后的数据进行拟合。
4. 绘制原始肌肉力量测量数据和拟合曲线，并显示图像。
