                 

# 1.背景介绍

Teradata Aster是一种高性能的大数据分析平台，它集成了Teradata数据库和Aster分析引擎，可以处理大规模的多源数据，并提供强大的数据挖掘和预测分析功能。Aster分析引擎基于SQL和R语言，支持多种机器学习算法，包括线性回归、逻辑回归、决策树、随机森林等。Aster还支持实时数据流分析、图形分析、文本分析等多种分析任务。

Aster分析引擎的核心技术是基于图形计算的机器学习算法，这种算法可以在多核CPU和GPU上高效地执行，提供了快速的分析结果和高效的资源利用。此外，Aster还支持数据库级别的并行处理和分布式计算，可以处理Tera字节级别的数据。

Aster分析引擎的核心概念包括：

1.图形模型：图形模型是Aster分析引擎中用于表示机器学习算法的核心概念。图形模型可以用来表示多种不同类型的算法，如线性回归、逻辑回归、决策树、随机森林等。

2.图形计算：图形计算是Aster分析引擎中用于执行机器学习算法的核心技术。图形计算可以在多核CPU和GPU上高效地执行，提供了快速的分析结果和高效的资源利用。

3.数据库集成：Aster分析引擎与Teradata数据库紧密集成，可以直接在数据库中执行分析任务，无需将数据导出到外部工具中。这种集成可以提高数据处理的效率，减少数据传输的延迟，并保证数据的安全性和完整性。
