数据清洗（Data Cleaning）是指在数据预处理阶段，通过对数据进行检查、纠正、去除不必要的信息以及填充缺失值等操作，以提高数据质量，使数据更适合进行数据分析和机器学习的过程。数据清洗是数据分析和机器学习的关键环节，因为数据质量直接影响分析结果的准确性和可靠性。

在现实生活中，数据来源于各种不同的渠道，如数据库、网络、传感器等，这些数据可能存在各种不规范、不完整、不一致、噪声、重复、缺失等问题。因此，在进行数据分析和机器学习之前，需要对数据进行清洗，以确保数据的质量和可靠性。

数据清洗的核心步骤包括：数据检查、数据纠正、数据去除、数据填充和数据转换等。这些步骤可以帮助我们提高数据质量，使数据更适合进行数据分析和机器学习。

## 1.1 值域检查
值域检查是指对数据的值域进行检查，以确保数据的值在预定义的范围内。值域检查可以帮助我们发现数据中的异常值和错误值，并进行相应的处理。

### 1.1.2 具体操作步骤
1. 对数据的每个字段进行值域检查。

\text{then } x = \text{NaN}

```python
import numpy as np

# 读取数据

# 数据检查
data['age'].fillna(data['age'].mean(), inplace=True)
data['age'] = data['age'].astype(int)

# 数据纠正
data['gender'] = data['gender'].map({'male': 0, 'female': 1})

# 数据去除
data.drop(['height', 'weight'], axis=1, inplace=True)

# 数据填充
data['income'].fillna(data['income'].median(), inplace=True)

# 数据转换
data['income'] = data['income'].astype(float)

# 保存数据
data.to_csv('clean_data.csv', index=False)
```
