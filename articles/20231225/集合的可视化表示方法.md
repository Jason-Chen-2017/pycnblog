## 3.3 集合可视化的数学模型公式

集合可视化的数学模型公式主要包括以下几个方面：

1. **集合大小**：集合大小是指集合中元素的数量，可以用符号 |A| 表示。例如，集合 A = {1, 2, 3} 的大小为 3，可以用 |A| = 3 表示。

2. **集合交集**：集合交集是指两个集合中共同元素的集合，可以用符号 A ∩ B 表示。例如，集合 A = {1, 2, 3} 和集合 B = {2, 3, 4} 的交集为 {2, 3}，可以用 A ∩ B = {2, 3} 表示。

3. **集合并集**：集合并集是指两个集合的所有元素的集合，可以用符号 A ∪ B 表示。例如，集合 A = {1, 2, 3} 和集合 B = {2, 3, 4} 的并集为 {1, 2, 3, 4}，可以用 A ∪ B = {1, 2, 3, 4} 表示。

4. **集合差集**：集合差集是指一个集合中不在另一个集合中的元素的集合，可以用符号 A - B 表示。例如，集合 A = {1, 2, 3} 和集合 B = {2, 3, 4} 的差集为 {1}，可以用 A - B = {1} 表示。
