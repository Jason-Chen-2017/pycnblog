# 2. 核心概念与联系

数理统计在金融市场中的核心概念包括：

1. 随机变量：随机变量是一种可能取多个值的变量，每个值都有一定的概率。在金融市场中，随机变量可以表示股票价格、利率、市场指数等。

2. 概率分布：概率分布是一种描述随机变量取值概率的方法。在金融市场中，常用的概率分布有正态分布、泊松分布、莱布尼茨分布等。

3. 期望：期望是随机变量取值平均值的概念。在金融市场中，期望可以用来预测未来市场行为、评估投资风险等。

4. 方差：方差是随机变量取值离平均值的程度。在金融市场中，方差可以用来衡量市场风险、评估投资回报等。

5. 相关性：相关性是两个随机变量之间的关系。在金融市场中，相关性可以用来分析市场之间的关系、评估投资组合风险等。

6. 最优化：最优化是寻找满足某种条件的最佳解的过程。在金融市场中，最优化可以用来优化投资组合、管理风险等。
