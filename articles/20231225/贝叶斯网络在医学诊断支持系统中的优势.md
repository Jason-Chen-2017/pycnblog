# 4.具体代码实例和详细解释说明

在本节中，我们将提供一个具体的代码实例来说明如何使用贝叶斯网络进行医学诊断支持。这个代码实例将使用Python的pgmpy库来构建一个简单的贝叶斯网络，并使用该网络来进行诊断。

```python
from pgmpy.models import BayesianNetwork
from pgmpy.factors.discrete import TabularCPD
from pgmpy.inference import VariableElimination

# 定义随机变量
variables = ['fever', 'cough', 'sore_throat', 'flu']

# 定义条件概率分布
fever_cpd = TabularCPD(variable='fever', variable_card=2,
                       values=[[0.9, 0.1], [0.1, 0.9]])
cough_cpd = TabularCPD(variable='cough', variable_card=2,
                       values=[[0.8, 0.2], [0.2, 0.8]])
sore_throat_cpd = TabularCPD(variable='sore_throat', variable_card=2,
                             values=[[0.7, 0.3], [0.3, 0.7]])
flu_cpd = TabularCPD(variable='flu', variable_card=2,
                     values=[[0.5, 0.5], [0.5, 0.5]])

# 构建贝叶斯网络
model = BayesianNetwork([('fever', 'cough'), ('fever', 'sore_throat'),
                         ('cough', 'flu'), ('sore_throat', 'flu')])
model.add_cpds(variables, {
    'fever': fever_cpd,
    'cough': cough_cpd,
    'sore_throat': sore_throat_cpd,
    'flu': flu_cpd
})

# 进行推理
inference = VariableElimination(model)
query = ['flu', 'cough']
result = inference.query(query)
print(result)
```

在这个代码实例中，我们首先定义了随机变量和条件概率分布。然后，我们使用pgmpy库构建了一个贝叶斯网络。最后，我们使用变量消除推理方法进行推理，并打印了推理结果。
