# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的智能门锁系统为例，展示智能家居家居安全系统的具体代码实例和解释。

```python
import time
import RPi.GPIO as GPIO

# 设置GPIO引脚
GPIO.setmode(GPIO.BCM)
GPIO.setup(23, GPIO.IN, pull_up_down=GPIO.PUD_UP)

# 定义门锁状态
lock_status = False

# 监控门锁状态
while True:
    # 读取门锁状态
    status = GPIO.input(23)

    # 判断门锁状态
    if status == GPIO.LOW:
        if not lock_status:
            print("门锁已打开")
            lock_status = True
    elif status == GPIO.HIGH:
        if lock_status:
            print("门锁已关闭")
            lock_status = False

    # 延时1秒
    time.sleep(1)
```

这个代码实例中，我们使用了Raspberry Pi的GPIO库来实现一个简单的智能门锁系统。通过监控门锁的状态，系统可以实现门锁的自动打开和关闭功能。
