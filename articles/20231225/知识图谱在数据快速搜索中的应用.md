# 4.具体代码实例和详细解释说明
# 4.1 构建知识图谱
我们可以使用Python的NetworkX库来构建知识图谱。NetworkX库是一个用于创建、操作和查询网络的Python库。

```python
import networkx as nx

G = nx.Graph()

# 添加实体节点
G.add_node("人A", attributes={"性别": "男", "年龄": 25})
G.add_node("人B", attributes={"性别": "女", "年龄": 22})
G.add_node("地点A", attributes={"城市": "北京", "国家": "中国"})

# 添加实体之间的关系
G.add_edge("人A", "人B", attributes={"关系": "朋友"})
G.add_edge("人A", "地点A", attributes={"位于": "北京"})
```
