# 4.具体代码实例和详细解释说明

## 4.1 代码实例
```python
import numpy as np

# 心理表现和生理指标数据
data = np.array([['情绪', 80], ['焦虑', 70], ['抑郁', 60], ['脉搏', 75], ['呼吸', 85]])

# 心理表现和生理指标的权重
weights = np.array([0.4, 0.3, 0.2, 0.1, 0.1])

# 心理健康评估
def mental_health_score(data, weights):
    score = np.dot(data.T, weights) / np.sum(weights)
    return score

# 打印心理健康评估结果
print('心理健康评估结果:', mental_health_score(data, weights))
```
