# 2.核心概念与联系

在高能物理实验室中，有几个核心概念需要了解：

1. **粒子物理学**：粒子物理学是研究微小粒子（如电子、氢子、中子等）的科学。这些粒子通常被称为“标量粒子”，它们具有特定的质量、电荷和其他属性。

2. **强力场**：强力场是一种引起粒子之间相互作用的力场，它是四维空间时间的非线性量。强力场在高能物理实验室中起着关键作用，因为它决定了粒子之间的相互作用。

3. **谱距**：谱距是粒子之间能量差的量，它可以用来描述粒子之间的相互作用。在高能物理实验室中，谱距是一个重要的参数，因为它可以帮助研究员了解粒子之间的相互作用。

4. **轨迹分析**：轨迹分析是一种用于分析粒子运动的方法，它通过观察粒子在检测器中的轨迹来确定粒子的能量、质量和其他属性。
