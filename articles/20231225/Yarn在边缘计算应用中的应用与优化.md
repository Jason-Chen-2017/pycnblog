# 2.核心概念与联系

NodeManager是Yarn中的一个节点管理器，它负责管理节点上的资源，并与ResourceManager进行通信。ResourceManager是Yarn中的一个资源管理器，它负责管理整个集群的资源，并分配资源给不同的应用程序。Container是Yarn中的一个容器，它是一个可以运行应用程序的资源分配单位。ApplicationMaster是Yarn中的一个应用程序管理器，它负责管理应用程序的生命周期，并与ResourceManager进行通信。

```python
# 初始化Yarn客户端

# 创建一个应用程序
app = YarnClientApplication(client)

# 设置应用程序的资源需求
app.set_resource_requirements("cpu:1,mem:1024")

# 设置应用程序的优先级
app.set_priority(10)

# 提交应用程序
app.submit()

# 等待应用程序完成
app.wait_for_completion()

# 取消应用程序
app.cancel()

# 关闭Yarn客户端
client.cleanup()
```
