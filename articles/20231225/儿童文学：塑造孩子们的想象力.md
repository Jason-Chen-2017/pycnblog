## 4.1 文本摘要

以下是一个使用Python的NLTK库实现文本摘要的代码示例：

```python
import nltk
from nltk.corpus import stopwords
from nltk.tokenize import word_tokenize, sent_tokenize

def text_summarization(text):
    # 去除停用词
    stop_words = set(stopwords.words('english'))
    word_tokens = word_tokenize(text)
    filtered_sentences = []

    for sentence in sent_tokenize(text):
        words = word_tokenize(sentence)
        filtered_sentence = [w.lower() for w in words if not w.lower() in stop_words]
        filtered_sentence = ' '.join(filtered_sentence)
        filtered_sentences.append(filtered_sentence)

    # 计算每个句子的词频
    word_freq = {}
    for sentence in filtered_sentences:
        words = word_tokenize(sentence)
        for word in words:
            if word not in word_freq:
                word_freq[word] = 1
            else:
                word_freq[word] += 1

    # 根据词频选择最重要的句子
    sorted_sentences = sorted(filtered_sentences, key=lambda s: sum(word_freq[word] for word in word_tokenize(s)), reverse=True)
    summary = ' '.join(sorted_sentences[:3])

    return summary

text = "This is a sample text for text summarization. It contains some words and sentences. We will use nltk library to extract the most important sentences and form a summary."
print(text_summarization(text))
```

这个代码首先使用NLTK库对文本进行分词，然后去除停用词，计算每个句子的词频，并根据词频选择最重要的句子作为摘要。

## 4.2 作品推荐

以下是一个使用Python的Pandas库实现作品推荐的代码示例：

```python
import pandas as pd

# 假设我们有一个包含儿童文学作品信息的数据框
data = {
    'title': ['Book1', 'Book2', 'Book3', 'Book4', 'Book5'],
    'author': ['Author1', 'Author2', 'Author3', 'Author4', 'Author5'],
    'theme': ['Theme1', 'Theme2', 'Theme3', 'Theme4', 'Theme5'],
    'read_count': [100, 200, 300, 400, 500]
}
df = pd.DataFrame(data)

# 假设我们有一个包含孩子的阅读历史的数据框
child_history = {
    'title': ['Book1', 'Book3'],
    'read_count': [1, 2]
}
df_child_history = pd.DataFrame(child_history)

# 计算孩子的阅读喜好
child_preference = df_child_history['read_count'].mean()

# 根据孩子的阅读喜好找到相似的作品
recommended_books = df[df['read_count'] > child_preference]
print(recommended_books)
```

这个代码首先创建了一个包含儿童文学作品信息的数据框，然后根据孩子的阅读历史计算出孩子的阅读喜好，最后根据孩子的阅读喜好找到相似的作品进行推荐。
