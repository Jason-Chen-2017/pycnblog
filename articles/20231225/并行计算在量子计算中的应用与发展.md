### 4.1 量子模拟

量子模拟是量子计算中一个重要的应用，它可以用来模拟量子系统的行为。以下是一个简单的量子模拟代码实例：

```python
import numpy as np
import qiskit

# 创建量子计算机对象
qc = qiskit.QuantumCircuit(2, 2)

# 初始化量子比特状态
qc.initialize([1, 0], 0)
qc.initialize([0, 1], 1)

# 应用Hadamard门
qc.h(0)

# 应用CNOT门
qc.cx(0, 1)

# 对量子比特进行测量
qc.measure([0, 1], [0, 1])

# 执行量子计算
aer_simulator = qiskit.Aer.get_backend('aer_simulator')
qobj = qc.bind_qubits(range(2), range(2)).run(aer_simulator)
result = qobj.result()

# 输出结果
print(result.get_counts())
```

上述代码实例中，我们首先创建了一个量子计算机对象，并初始化了两个量子比特的状态。接着，我们应用了Hadamard门和CNOT门，并对量子比特进行了测量。最后，我们使用Aer模拟器执行量子计算，并输出了结果。
