# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来解释 Hive 的使用方法。

假设我们有一个名为 `employee` 的表，表包含以下列：

- id：员工编号
- name：员工姓名
- age：员工年龄
- salary：员工薪资

我们想要查询员工表中的所有员工信息，并且要求员工年龄大于 30 岁。

我们可以使用以下 Hive 查询来实现这个需求：

```sql
SELECT id, name, age, salary FROM employee WHERE age > 30;
```

这个查询的解释如下：

- `SELECT` 是 Hive 查询语句的关键字，用来指定要查询的列。
- `id, name, age, salary` 是要查询的列，包括员工编号、员工姓名、员工年龄和员工薪资。
- `FROM` 是 Hive 查询语句的关键字，用来指定要查询的表。
- `employee` 是要查询的表名。
- `WHERE` 是 Hive 查询语句的关键字，用来指定查询条件。
- `age > 30` 是查询条件，用来指定要查询的员工年龄大于 30 岁。
