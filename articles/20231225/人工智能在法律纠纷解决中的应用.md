## 4.1 知识图谱构建

我们可以使用Python的NetworkX库来构建知识图谱。以下是一个简单的代码实例：

```python
import networkx as nx

# 创建一个空的知识图谱
G = nx.Graph()

# 添加实体和关系
G.add_node("人A", type="人")
G.add_node("人B", type="人")
G.add_edge("人A", "人B", type="归属")

# 查询实体和关系
print(G.nodes(data=True))
print(G.edges(data=True))
```

在这个代码实例中，我们创建了一个空的知识图谱，并添加了两个实体（人A和人B）以及一个关系（归属）。然后，我们查询了图谱中的实体和关系。

## 4.2 法律纠纷分析

我们可以使用Python的spaCy库来进行法律纠纷分析。以下是一个简单的代码实例：

```python
import spacy

# 加载spaCy模型
nlp = spacy.load("zh_core_web_sm")

# 加载法律文本
text = "某人违反了法律规定，导致了法律责任"

# 分析法律文本
doc = nlp(text)

# 提取实体和关系
for ent in doc.ents:
    print(ent.text, ent.label_)
for rel in doc.relations:
    print(rel.subject, rel.rel_type, rel.object_)
```

在这个代码实例中，我们加载了spaCy的中文模型，并加载了一个法律文本。然后，我们使用spaCy的实体识别和关系识别功能来提取文本中的实体和关系。
