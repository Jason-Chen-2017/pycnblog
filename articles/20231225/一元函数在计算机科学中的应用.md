### 4.1 快速排序算法实例

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

在这个快速排序算法实例中，我们使用了一元函数来实现排序功能。具体来说，我们首先选择一个基准值（pivot），然后将原始数组分为三个部分：小于基准值的数、等于基准值的数、大于基准值的数。接着，我们递归地对小于基准值的数和大于基准值的数进行排序，最后将这些排序后的数组与等于基准值的数组合并，得到最终的排序结果。

### 4.2 栈实例

```python
class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        else:
            raise IndexError("pop from an empty stack")

    def is_empty(self):
        return len(self.items) == 0
```

在这个栈实例中，我们使用一元函数来实现栈的push和pop操作。具体来说，我们将栈的数据存储在一个列表中，使用push方法将新的元素追加到列表的末尾，使用pop方法将列表的末尾元素弹出。同时，我们还提供了一个is_empty方法来判断栈是否为空。

### 6.1 一元函数与多元函数的区别

一元函数是将一个变量作为输入，并根据某种规则将其映射到另一个变量作为输出的函数。而多元函数是将多个变量作为输入，并根据某种规则将它们映射到一个输出的函数。因此，一元函数只有一个输入变量，多元函数有多个输入变量。
