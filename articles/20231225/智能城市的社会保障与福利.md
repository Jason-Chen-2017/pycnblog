# 2.核心概念与联系

## 2.1 社会保障

社会保障是指政府为确保公民在生活、工作、学习等方面的基本权益，提供的一系列政策措施和服务。社会保障的主要内容包括：

1. 养老保障：政府为老年人提供养老金、医疗保险等服务。
2. 医疗保障：政府为公民提供医疗保险，确保他们可以获得医疗服务。
3. 失业保障：政府为失业人员提供失业保险，确保他们可以维持生活。
4. 子女教育保障：政府为公民的子女提供教育保险，确保他们可以接受良好的教育。
5. 残疾人保障：政府为残疾人提供残疾人保险，确保他们可以获得适当的帮助和支持。

## 2.2 福利

福利是指政府为提高公民的生活水平、提高社会福祉，提供的一系列政策措施和服务。福利的主要内容包括：

1. 住房福利：政府为公民提供住房贷款、住房补贴等服务，帮助他们购买住房。
2. 社会保险：政府为公民提供社会保险，包括医疗保险、失业保险、养老保险等。
3. 教育福利：政府为公民提供教育资金，帮助他们接受高质量的教育。
4. 文化福利：政府为公民提供文化活动资金，帮助他们参与文化活动。
5. 环境福利：政府为公民提供环境保护资金，帮助他们保护环境。
