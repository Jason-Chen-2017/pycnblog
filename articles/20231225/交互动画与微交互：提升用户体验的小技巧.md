# 4.具体代码实例和详细解释说明

## 4.1 交互动画实例

以下是一个简单的交互动画实例，使用CSS和JavaScript实现的：

```html
<!DOCTYPE html>
<html>
<head>
<style>
  .box {
    width: 100px;
    height: 100px;
    background-color: red;
    transition: all 1s;
  }
</style>
<script>
  function changeColor() {
    var box = document.getElementById('box');
    box.style.backgroundColor = 'blue';
  }
</script>
</head>
<body>
  <div id="box" onclick="changeColor()">
  </div>
</body>
</html>
```

在这个实例中，我们使用了CSS的transition属性来实现动画效果，当用户点击div元素时，背景颜色从红色变为蓝色，动画持续时间为1秒。

## 4.2 微交互实例

以下是一个简单的微交互实例，使用CSS和JavaScript实现的：

```html
<!DOCTYPE html>
<html>
<head>
<style>
  .box {
    width: 100px;
    height: 100px;
    background-color: red;
    transition: all 1s;
  }
</style>
<script>
  function expand() {
    var box = document.getElementById('box');
    box.style.transform = 'scale(1.5)';
  }
</script>
</head>
<body>
  <div id="box" onclick="expand()">
  </div>
</body>
</html>
```

在这个实例中，我们使用了CSS的transform属性来实现动画效果，当用户点击div元素时，元素大小从原始大小扩展到1.5倍，动画持续时间为1秒。
