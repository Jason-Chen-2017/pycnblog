### 4.1 边缘计算算法代码实例

```python
import numpy as np
import pandas as pd

# 读取学生成绩数据
data = pd.read_csv('student_scores.csv')

# 计算学生成绩的平均值
average = data['score'].mean()

# 计算学生成绩的标准差
std_dev = data['score'].std()

# 计算学生成绩的均值和标准差
result = {'average': average, 'std_dev': std_dev}

# 返回结果
print(result)
```
