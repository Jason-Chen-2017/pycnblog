```python
def calculate_emotional_intelligence(empathy, self_awareness, self_management, social_skills):
    emotional_intelligence = (empathy + self_awareness + self_management + social_skills) / 4
    return emotional_intelligence

empathy = 80
self_awareness = 70
self_management = 85
social_skills = 90

emotional_intelligence = calculate_emotional_intelligence(empathy, self_awareness, self_management, social_skills)
print("情绪智力分数：", emotional_intelligence)
```

这个代码实例中，我们定义了一个函数`calculate_emotional_intelligence`，用于计算领导者的情绪智力分数。这个函数接受四个参数：`empathy`、`self_awareness`、`self_management`和`social_skills`，分别对应情绪识别、情绪管理、情绪沟通和社交技巧。这四个参数的分数范围为0到100，分别代表情绪智力的四个方面。我们将这四个参数相加，并将和除以4，得到领导者的情绪智力分数。
