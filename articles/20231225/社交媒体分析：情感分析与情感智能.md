## 4.1情感分析的代码实例

我们将使用Python的NLTK库来实现情感分析。首先，我们需要安装NLTK库：

```
pip install nltk
```

然后，我们可以使用以下代码来实现情感分析：

```python
import nltk
from nltk.sentiment import SentimentIntensityAnalyzer

# 下载需要的数据集和模型
nltk.download('vader_lexicon')

# 创建情感分析器对象
sia = SentimentIntensityAnalyzer()

# 测试文本
text = "I love this product!"

# 获取情感分析结果
sentiment = sia.polarity_scores(text)

print(sentiment)
```

上述代码首先导入了NLTK库和SentimentIntensityAnalyzer类，然后下载了需要的数据集和模型，创建了情感分析器对象，并使用该对象获取了情感分析结果。

## 4.2情感智能的代码实例

我们将使用Python的Scikit-learn库来实现情感智能。首先，我们需要安装Scikit-learn库：

```
pip install scikit-learn
```

然后，我们可以使用以下代码来实现情感智能：

```python
import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB
from sklearn.metrics import accuracy_score

# 训练数据
data = [
    ("I love this product!", "positive"),
    ("I hate this product!", "negative"),
    ("This is a great product!", "positive"),
    ("This is a terrible product!", "negative"),
    ("I am happy with this product!", "positive"),
    ("I am unhappy with this product!", "negative"),
]

# 分词和词汇矢量化
vectorizer = CountVectorizer()
X = vectorizer.fit_transform([d[0] for d in data])
y = [d[1] for d in data]

# 训练模型
clf = MultinomialNB()
clf.fit(X, y)

# 测试数据
test_data = ["I like this product!", "I dislike this product!"]
test_X = vectorizer.transform(test_data)

# 预测情感
predictions = clf.predict(test_X)

# 评估模型
accuracy = accuracy_score(y, predictions)
print("Accuracy:", accuracy)
```

上述代码首先导入了Scikit-learn库和相关的函数，然后创建了训练数据，并使用CountVectorizer进行分词和词汇矢量化。接着，使用MultinomialNB进行情感分类，并使用accuracy_score评估模型。
