# 2. 核心概念与联系

### 2.1.1 纳米尺度

纳米技术研究的主要对象是尺寸在0.1-100纳米之间的微小结构和系统。这个尺寸范围在原子和分子级别，包括了许多常见物质的微观结构。例如，水分子的直径约为0.1纳米，蛋白质的分子大小通常在1-10纳米之间，病毒的尺寸通常在0.05-0.3纳米之间。

### 2.1.2 纳米材料

纳米材料是具有纳米尺度结构的材料，它们的性能和功能通常与其尺寸、形状和组织有关。例如，纳米金属粒子可以用于制造高性能的光学和电子材料，如金属纳米粒子镀膜和纳米金属纳米晶体体。同样，纳米半导体材料可以用于制造高效的光电转换设备，如单模光电子成像传感器和高效光电切换器。

纳米设备是能够在纳米尺度进行操作和控制的设备，它们通常采用原子力学和分子力学方法进行研究和制造。例如，电子麦克罗夫斯成像（EM）和光原子成像（SNOM）是两种常用的纳米成像技术，它们可以用于观察和分析纳米尺度的材料和结构。
