概率分布是一种用于描述随机变量取值概率的数学模型。在物理学中，随机变量通常表示一个系统中的某个物理量，如气体分子的速度、电子的能量等。概率分布可以通过一种称为密度函数（density function）的函数来表示，密度函数的积分在某个区间内等于1。

### 4.1 代码实例：气体分子速度分布

在本例中，我们将计算气体分子速度分布的概率密度函数，并使用Matplotlib库进行可视化。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义赫兹分布的参数
m = 1.67e-27  # 分子质量（公鸡）
kB = 1.38e-23  # 布尔常数（J/K）
T = 298  # 温度（K）

# 计算速度分布的范围
v_min = 0
v_max = 10000

# 计算速度分布的步长
v_step = 100

# 计算速度分布的概率密度函数
v = np.arange(v_min, v_max, v_step)
f = np.sqrt((2 / np.pi * m * kB * T)) * (v ** 2) * np.exp(-(m * v ** 2) / (2 * kB * T))

# 可视化速度分布
plt.plot(v, f)
plt.xlabel('Speed (m/s)')
plt.ylabel('Probability Density')
plt.title('Maxwell-Boltzmann Speed Distribution')
plt.show()
```
