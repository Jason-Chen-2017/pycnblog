### 4.1 自动化控制示例

我们可以通过以下代码实现一个基本的PID控制器：

```python
import numpy as np

class PID:
    def __init__(self, Kp, Ki, Kd):
        self.Kp = Kp
        self.Ki = Ki
        self.Kd = Kd
        self.integral = 0
        self.last_error = 0

    def compute(self, error, dt):
        self.integral += error * dt
        derivative = (error - self.last_error) / dt
        output = self.Kp * error + self.Ki * self.integral + self.Kd * derivative
        self.last_error = error
        return output
```

在这个示例中，我们定义了一个PID控制器类，它可以用于控制火箭和卫星的各种参数。我们可以通过调整Kp、Ki和Kd参数来优化控制器的性能。
