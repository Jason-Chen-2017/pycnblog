## 2. 核心概念与联系

在本节中，我们将介绍矩阵逆的基本概念和联系。

### 2.1 矩阵逆的定义

矩阵逆是线性代数中的一个重要概念，它表示一个矩阵的逆运算结果。给定一个方阵A，如果存在一个矩阵B，使得A * B = I，其中I是单位矩阵，则称B是A的逆矩阵，记作A^(-1)。

### 2.2 矩阵逆的性质

矩阵逆具有以下性质：

1. 逆矩阵不是唯一的。例如，给定一个对称矩阵A，如果存在一个逆矩阵A^(-1)，则A^T * A^(-1)也是A的逆矩阵。
2. 矩阵的逆矩阵具有交换律。即，如果A和B是矩阵的逆矩阵，那么A * B = B * A = I，其中I是单位矩阵。
3. 矩阵的逆矩阵具有分配律。即，如果A和B是矩阵的逆矩阵，那么A * (B + C) = A * B + A * C，A * (B * C) = (A * B) * C，其中C是一个矩阵。

### 2.3 矩阵逆的应用

### 3.2 行reduction法

行reduction法是一种通过对矩阵行进行行交换和行缩放来得到上三角矩阵的方法。具体步骤如下：

1. 选择矩阵A的第一行，将其正规化为（1，0，0，...）。
2. 将第一行除以其长度，使其长度为1。
3. 选择矩阵A的第二行，将其正规化为（0，a12，a23，...）。
4. 将第二行除以a12，使其第一个非零元素为1。
5. 将第一行和第二行交换，使得第一行的第二个元素为0。
6. 将第三行及以后的行依次正规化和交换，直到得到上三角矩阵。
