## 4.1 代码实例

我们将通过一个简单的条形图来演示如何将数据可视化与故事结合起来。以下是一个简单的Python代码实例：

```python

# 数据

# 创建条形图
plt.bar(data.keys(), data.values())

# 添加标签
plt.xticks(range(len(data)), data.keys())
plt.yticks(range(0, max(data.values()) + 1, 500))

# 添加标题
plt.title('城市销售额对比')

# 显示图表
plt.show()
```

## 4.2 详细解释说明

在这个代码实例中，我们首先导入了`matplotlib.pyplot`库，然后定义了一个字典类型的数据，其中包含了三个城市的销售额信息。接着，我们使用`plt.bar()`函数创建了一个条形图，其中的x轴表示城市名称，y轴表示销售额。我们还添加了标签和标题，并使用`plt.show()`函数显示了图表。

通过这个简单的例子，我们可以看到如何将数据可视化与故事结合起来。在这个例子中，我们的故事是关于三个城市的销售额对比。通过这个条形图，我们可以快速地看到城市A的销售额最低，城市C的销售额最高，而城市B的销售额在中间。这个故事可以帮助我们更好地理解这三个城市的销售情况，并根据这些信息制定相应的决策。
