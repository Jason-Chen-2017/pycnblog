## 4.1 机器人的运动控制

运动控制是机器人执行任务的关键环节。通过运动控制算法，机器人可以实现预定义的目标和任务。以下是一个简单的机器人运动控制示例：

```python
import rospy
from geometry_msgs.msg import Twist

def robot_move(linear_speed, angular_speed):
    pub = rospy.Publisher('/cmd_vel', Twist, queue_size=10)
    rospy.init_node('robot_move', anonymous=True)
    rate = rospy.Rate(10)
    move = Twist()
    move.linear.x = linear_speed
    move.angular.z = angular_speed
    while not rospy.is_shutdown():
        pub.publish(move)
        rate.sleep()

if __name__ == '__main__':
    try:
        robot_move(1.0, 0.5)
    except rospy.ROSInterruptException:
        pass
```

感知处理是机器人与环境的接触点。通过感知处理算法，机器人可以获取和理解环境信息，以便更好地执行任务。以下是一个简单的机器人图像处理示例：

```python
import cv2
import numpy as np

def image_processing(image):
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    blur = cv2.GaussianBlur(gray, (5, 5), 0)
    edges = cv2.Canny(blur, 50, 150)
    lines = cv2.HoughLinesP(edges, 2, np.pi / 180, 100, np.array([]), minLineLength=40, maxLineGap=5)
    return lines

if __name__ == '__main__':
    cap = cv2.VideoCapture(0)
    while True:
        ret, image = cap.read()
        lines = image_processing(image)
        if lines is not None:
            for line in lines:
                x1, y1, x2, y2 = line[0]
                cv2.line(image, (x1, y1), (x2, y2), (0, 255, 0), 2)
        cv2.imshow('image', image)
        if cv2.waitKey(1) & 0xFF == ord('q'):
            break
    cap.release()
    cv2.destroyAllWindows()
```

决策执行是机器人执行任务的核心环节。通过决策执行算法，机器人可以根据环境和任务需求，生成合适的行动和控制策略。以下是一个简单的机器人决策执行示例：

```python
import rospy
from geometry_msgs.msg import Twist

def decision_execution(linear_speed, angular_speed):
    pub = rospy.Publisher('/cmd_vel', Twist, queue_size=10)
    rospy.init_node('decision_execution', anonymous=True)
    rate = rospy.Rate(10)
    move = Twist()
    move.linear.x = linear_speed
    move.angular.z = angular_speed
    while not rospy.is_shutdown():
        if linear_speed > 0:
            move.linear.x = linear_speed
        else:
            move.linear.x = 0
        if angular_speed > 0:
            move.angular.z = angular_speed
        else:
            move.angular.z = 0
        pub.publish(move)
        rate.sleep()

if __name__ == '__main__':
    try:
        decision_execution(1.0, 0.5)
    except rospy.ROSInterruptException:
        pass
```
