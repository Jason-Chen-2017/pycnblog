# 4.具体代码实例和详细解释说明

## 4.1 舆情分析代码实例
以Python语言为例，下面是一个简单的舆情分析代码实例：

```python
import jieba
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.cluster import KMeans

# 收集数据
data = ["政府政策很好", "政府政策很差", "政府行为很好", "政府行为很差"]

# 数据预处理
seg_list = jieba.cut(data)

# 特征提取
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(seg_list)

# 数据分类
kmeans = KMeans(n_clusters=2)
y = kmeans.fit_predict(X)

# 数据分析
print("支持度：", y == 0)
print("反对度：", y == 1)
```

在这个代码实例中，我们首先通过`jieba`库进行文本分词。然后通过`TfidfVectorizer`库进行特征提取。接着通过`KMeans`算法进行数据分类，并根据分类结果计算支持度和反对度。

## 4.2 公众参与代码实例
以Python语言为例，下面是一个简单的公众参与代码实例：

```python
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.cluster import KMeans

# 收集数据
data = {"参与方式": ["官方网站", "社交媒体", "公开咨询", "网络投票"],
        "参与度": ["在线参与", "面对面参与", "有效参与", "无效参与"]}
df = pd.DataFrame(data)

# 特征提取
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(df["参与方式"])

# 数据分类
kmeans = KMeans(n_clusters=2)
y = kmeans.fit_predict(X)

# 数据分析
df["参与方式"] = df["参与方式"].apply(lambda x: "在线参与" if y == 0 else "面对面参与")
print(df)
```

在这个代码实例中，我们首先通过`pandas`库创建一个数据框。然后通过`TfidfVectorizer`库进行特征提取。接着通过`KMeans`算法进行数据分类，并根据分类结果计算参与方式。
