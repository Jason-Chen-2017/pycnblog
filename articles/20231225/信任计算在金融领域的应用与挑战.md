### 4.1 RSA加密算法实例

RSA是一种公开密钥加密算法，通常用于加密和解密数据。以下是一个简单的RSA加密和解密示例：

```python
import random

def generate_keys(n):
    p = random.randint(2, n - 1)
    q = random.randint(2, n - 1)
    phi = (p - 1) * (q - 1)
    totient = pow(phi, -1, phi)
    d = totient - 1
    return (p, q, d)

def encrypt(m, e, n):
    return pow(m, e, n)

def decrypt(c, d, n):
    return pow(c, d, n)

n = 2048
e = 65537
p, q, d = generate_keys(n)

m = 123456
c = encrypt(m, e, n)
print(f"加密后的数据：{c}")

d_ = pow(e, d, phi)
m_ = decrypt(c, d_, n)
print(f"解密后的数据：{m_}")
```

### 4.2 HMAC数据完整性验证算法实例

HMAC是一种基于密钥的消息完整性验证算法，通常用于确保数据的准确性和完整性。以下是一个简单的HMAC完整性验证示例：

```python
import hmac
import hashlib

key = b'secret'
message = b'Hello, World!'

hmac_obj = hmac.new(key, message, hashlib.sha256)
signature = hmac_obj.digest()

print(f"HMAC签名：{signature}")

verification_obj = hmac.new(key, message, hashlib.sha256)
verification_obj.digest()
```

### 4.3 Homomorphic Encryption算法实例

Homomorphic Encryption是一种允许在加密数据上进行计算的加密算法，通常用于保护数据的安全和隐私。以下是一个简单的Homomorphic Encryption示例：

```python
from phe import enc

# 生成一个GSW密钥对
secret_key = enc.generate_gsw_secret_key()
public_key = enc.generate_gsw_public_key(secret_key)

# 加密数据

encrypted_data = enc.encrypt(data, public_key)
print(f"加密后的数据：{encrypted_data}")

# 在加密数据上进行计算

# 解密结果

```
