# 4.1 传播标签
```python
import numpy as np

def similarity(x, y):
    return np.exp(-np.linalg.norm(x - y)**2 / 2)

def propagate_labels(data, labels, similarity, threshold):
    labeled_data = data[labels != -1]
    unlabeled_data = data[labels == -1]
    labeled_labels = labels[labels != -1]
    
    for _ in range(100):
        similarities = similarity(labeled_data, unlabeled_data)
        predicted_labels = np.argmax(similarities, axis=0)
        unlabeled_data[predicted_labels] = labeled_labels
        labels[unlabeled_data == -1] = -1

data = np.random.rand(100, 10)
labels = np.random.randint(0, 2, 100)
propagate_labels(data, labels, similarity, 0.5)
```

def similarity(x, y):
    return np.exp(-np.linalg.norm(x - y)**2 / 2)

def adaptive_semi_supervised_learning(data, labels, similarity, threshold):
    labeled_data = data[labels != -1]
    unlabeled_data = data[labels == -1]
    labeled_labels = labels[labels != -1]
    
    if np.mean(np.linalg.norm(labeled_data - unlabeled_data, axis=1)) < threshold:

## 问题1：半监督学习与完全监督学习的区别是什么？
答：半监督学习与完全监督学习的区别在于数据标签的可用性。在半监督学习中，我们只有一部分数据标签可用，而在完全监督学习中，所有数据标签可用。

答：图像聚类与图像分类的区别在于任务目标。在图像聚类中，我们的目标是根据图像之间的相似性将其分组，而在图像分类中，我们的目标是根据图像的特征将其分类。

## 问题3：半监督学习可以解决过拟合问题吗？
