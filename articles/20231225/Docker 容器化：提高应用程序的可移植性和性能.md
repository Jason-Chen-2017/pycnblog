                 

# 1.背景介绍

Docker 是一种开源的应用容器引擎，它可以用来打包应用及其依赖项，以便在任何流行的平台上运行。Docker 使用一种称为容器的抽象层，将软件打包到一个可移植的环境中，以便在任何流行的平台上运行。这使得软件开发人员能够确保其应用程序在所有环境中都能正常运行，而无需担心因不同环境的差异而导致的问题。

Docker 的核心概念是容器，它是一个轻量级的、自给自足的、可移植的应用程序环境。容器包含了应用程序及其依赖项，以及运行所需的一切。容器可以在任何支持 Docker 的平台上运行，无需担心环境差异。
