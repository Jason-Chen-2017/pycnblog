# 2.核心概念与联系

## 2.1相对熵

相对熵是信息论中的一个重要概念，它可以用来衡量两个概率分布之间的差异。相对熵的定义如下：

$$
D_{KL}(P||Q) = \sum_{x} P(x) \log \frac{P(x)}{Q(x)}
$$

其中，$P$ 和 $Q$ 是两个概率分布，$x$ 是取值域，$P(x)$ 和 $Q(x)$ 是两个分布的概率密度函数。相对熵的含义是，给定一个概率分布 $P$，我们可以通过一个参考分布 $Q$ 来衡量 $P$ 与 $Q$ 之间的差异。

相对熵在自动驾驶系统中的应用主要是通过优化模型的预测分布来提高系统的准确性和可靠性。例如，在图像识别任务中，我们可以通过优化模型的预测分布来提高模型的识别准确性。
