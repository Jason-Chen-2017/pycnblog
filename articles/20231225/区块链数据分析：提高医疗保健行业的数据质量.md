## 2.1 区块链基础概念

区块链是一种分布式、去中心化的数据存储和处理技术，它由一系列相互连接的区块组成，每个区块包含一组交易，这些交易是有序的，每个交易都有一个时间戳，这样可以确保数据的完整性和一致性。区块链技术的核心特点是：

1. 去中心化：区块链不需要中心化的服务器或数据库来存储和处理数据，而是通过多个节点共同维护数据。

2. 不可改变：区块链的数据是不可改变的，因为每个区块都有一个时间戳，这个时间戳确保了数据的完整性和一致性。

3. 透明度：区块链的数据是透明的，因为每个节点都可以看到其他节点的数据，这样可以确保数据的可信度。

4. 安全：区块链的数据是安全的，因为每个区块都有一个哈希值，这个哈希值确保了数据的安全性。

## 4.1 创建一个区块

```python
import hashlib

class Block:
    def __init__(self, index, data, previous_hash):
        self.index = index
        self.data = data
        self.previous_hash = previous_hash
        self.timestamp = time.time()
        self.hash = self.calculate_hash()

    def calculate_hash(self):
        block_string = f"{self.index}{self.data}{self.timestamp}{self.previous_hash}"
        return hashlib.sha256(block_string.encode()).hexdigest()
```

在上面的代码中，我们定义了一个`Block`类，它包含了区块的索引、数据、前一个哈希值、时间戳和哈希值。我们还定义了一个`calculate_hash`方法，它用于计算区块的哈希值。

## 4.2 创建一个区块链

```python
class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(0, "Genesis Block", "0")

    def add_block(self, data):
        previous_hash = self.chain[-1].hash
        new_block = Block(len(self.chain), data, previous_hash)
        self.chain.append(new_block)
```

在上面的代码中，我们定义了一个`Blockchain`类，它包含了一个区块链列表。我们还定义了一个`create_genesis_block`方法，它用于创建一个初始的区块，并一个`add_block`方法，它用于添加新的区块到区块链中。
