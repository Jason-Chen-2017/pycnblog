## 1. 知识图谱构建代码实例

以下是一个简单的知识图谱构建代码实例：

```python
from rdflib import Graph

# 创建一个空的RDF图
g = Graph()

# 添加实体和关系
g.add((("Disease", "name", "Cancer"), "IS_A", "Disease"))
g.add((("Cancer", "symptom", "Pain"), "IS_A", "Symptom"))
g.add((("Pain", "treatment", "Analgesic"), "IS_A", "Treatment"))

# 保存知识图谱
g.serialize(destination="knowledge_graph.ttl")
```

在上述代码中，我们首先导入了`rdflib`库，然后创建了一个空的RDF图。接着，我们添加了一些实体和关系，并将其保存到一个文件中。

## 2. 疾病诊断代码实例

以下是一个简单的疾病诊断代码实例：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 疾病描述
disease_descriptions = [
    "I have a lump in my breast",
    "I have a persistent cough",
    "I have a fever and headache"
]

# 疾病名称
disease_names = ["Breast Cancer", "Pneumonia", "Flu"]

# 将疾病描述转换为向量
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(disease_descriptions)

# 计算疾病描述之间的相似度
similarity = cosine_similarity(X)

# 找到最相似的疾病名称
most_similar_disease = disease_names[np.argmax(similarity)]
print(most_similar_disease)
```

在上述代码中，我们首先导入了`TfidfVectorizer`和`cosine_similarity`函数。接着，我们定义了一些疾病描述和疾病名称。然后，我们将疾病描述转换为向量，并计算疾病描述之间的相似度。最后，我们找到了最相似的疾病名称并输出了结果。
