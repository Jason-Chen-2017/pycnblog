# 2.核心概念与联系
# 2.1 贝叶斯定理
贝叶斯定理是贝叶斯决策的基础，它是概率论中的一个重要公式。贝叶斯定理可以帮助我们计算条件概率，即给定某个事件已经发生，另一个事件的概率。贝叶斯定理的公式为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 是条件概率，表示已知事件B发生的情况下事件A的概率；$P(B|A)$ 是联合概率，表示事件A发生的情况下事件B的概率；$P(A)$ 是事件A的概率；$P(B)$ 是事件B的概率。

# 2.2 贝叶斯决策
贝叶斯决策是一种基于贝叶斯定理的决策方法。它的核心思想是，根据已有的信息（即条件概率），计算出各种可能结果的概率，然后选择那个概率最高的结果作为预测结果。这种方法的优势在于它可以根据不同的情况来调整预测结果，从而提高预测准确率。

# 3.1 贝叶斯决策的步骤
贝叶斯决策的主要步骤包括：

1. 收集数据：收集与问题相关的数据，以便计算条件概率。
2. 定义事件：将问题中的各种可能结果定义为事件，并计算它们的概率。
3. 计算条件概率：使用贝叶斯定理计算条件概率。
4. 选择最高概率的结果：根据计算出的概率，选择那个概率最高的结果作为预测结果。

# 3.2 贝叶斯决策的数学模型
贝叶斯决策的数学模型可以表示为：

$$
\arg\max_{h \in H} P(h|D) = \frac{P(D|h)P(h)}{P(D)}

- $P(垃圾邮件|关键词“免费”) = \frac{100}{500} = 0.2$

接下来，我们需要计算事件的概率。根据给定的信息，我们可以计算出以下事件的概率：

- $P(垃圾邮件) = \frac{500}{1000} = 0.5$

$$

$$

根据这个结果，我们可以判断一个包含关键词“免费”的邮件更可能是正常邮件，而不是垃圾邮件。
