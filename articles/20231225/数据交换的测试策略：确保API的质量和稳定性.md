# 4.具体代码实例和详细解释说明

在这个部分，我们将提供一个具体的代码实例，以展示如何使用测试策略来确保 API 的质量和稳定性。

```python
import random
import requests

def test_api():
    urls = ['https://api.example.com/v1/data', 'https://api.example.com/v2/data']
    for url in urls:
        for _ in range(10):
            random_param = random.randint(1, 100)
            response = requests.get(f'{url}?param={random_param}')
            if response.status_code == 200:
                print(f'Success: {url} with param {random_param}')
            else:
                print(f'Failure: {url} with param {random_param}')

if __name__ == '__main__':
    test_api()
```

在上面的代码实例中，我们首先导入了 `random` 和 `requests` 库。 `random` 库用于生成随机参数，`requests` 库用于发送 HTTP 请求。

接下来，我们定义了一个 `test_api` 函数，该函数将测试两个 API 的性能。 我们首先定义了一个包含两个 API 地址的列表。 然后，我们使用一个嵌套的 for 循环来遍历 API 地址和随机参数。 对于每个请求，我们使用 `requests.get` 函数发送请求，并检查响应状态码。 如果响应状态码为 200，表示请求成功，我们将输出成功信息。 否则，我们将输出失败信息。
