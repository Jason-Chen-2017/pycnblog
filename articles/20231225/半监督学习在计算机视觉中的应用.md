# 2.核心概念与联系

半监督学习是一种在训练数据中结合有标注数据和无标注数据的学习方法，它通过利用无标注数据来补充有限的标注数据，从而提高模型的泛化能力。在计算机视觉中，半监督学习可以应用于图像分类、目标检测、图像分割等任务。

半监督学习的核心概念包括：

- 有标注数据：已经被人工标注的数据，通常较少。
- 无标注数据：没有被人工标注的数据，通常较多。
- 半监督学习算法：将有标注数据和无标注数据结合，训练模型。

半监督学习与其他学习方法的联系：

- 与监督学习的区别在于，监督学习只使用有标注数据进行训练。
- 与无监督学习的区别在于，无监督学习只使用无标注数据进行训练。
- 半监督学习结合了监督学习和无监督学习的优点，可以提高模型的泛化能力。
