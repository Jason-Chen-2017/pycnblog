# 4.1 语音识别代码实例

以下是一个简单的语音识别代码实例，它使用Python的`speech_recognition`库来将语音信号转换为文本：

```python
import speech_recognition as sr

# 初始化识别器
recognizer = sr.Recognizer()

# 获取麦克风录音
with sr.Microphone() as source:
    print("请说话")
    audio = recognizer.listen(source)

# 将录音转换为文本
try:
    text = recognizer.recognize_google(audio)
    print("你说的是：" + text)
except sr.UnknownValueError:
    print("抱歉，我没有理解你的说话")
except sr.RequestError as e:
    print("错误：" + str(e))
```

这个代码首先导入`speech_recognition`库，然后初始化一个识别器对象。接着，它使用麦克风录制用户的语音，并将录音转换为文本。最后，它将文本打印出来。

# 4.4 语音合成代码实例

以下是一个简单的语音合成代码实例，它使用Python的`gTTS`库来将文本转换为语音信号：

```python
from gtts import gTTS
import os

# 文本
text = "你好，我是智能语音助手"

# 创建一个gTTS对象
tts = gTTS(text=text, lang='zh-cn', slow=False)

# 保存语音文件
tts.save("hello.mp3")

# 播放语音文件
os.system("start hello.mp3")
```

这个代码首先导入`gTTS`库，然后创建一个`gTTS`对象。接着，它将文本转换为语音信号，并将语音信号保存到一个MP3文件中。最后，它使用系统命令播放语音文件。
