## 4.1 代码实例

我们选择了一个简单的排序算法——冒泡排序，来说明如何使用算法来提高工程质量。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

## 4.2 详细解释说明

冒泡排序是一种简单的排序算法，它的基本思想是通过多次对交换元素的操作来实现排序。在这个算法中，我们首先需要确定需要排序的数组，然后通过多次循环来实现排序。在每次循环中，我们会比较相邻的两个元素，如果后一个元素比前一个元素大，则交换它们的位置。这个过程会一直持续到所有元素都被排序为非降序。
