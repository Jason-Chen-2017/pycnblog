# 2.核心概念与联系

LU分解是一种矩阵分解方法，它将一个方阵A分解为上三角矩阵L和下三角矩阵U的乘积，即A=LU。其中，L表示左三角矩阵，U表示上三角矩阵。这种分解方法在许多数值计算中得到广泛应用，如线性方程组求解、奇异值分解等。

LU分解的核心概念包括：

- 上三角矩阵U：U是一个方阵，其对角线元素为1，其他元素为非零值。
- 下三角矩阵L：L是一个方阵，其对角线元素为1，其他元素为非零值。
- 矩阵A的分解：A可以被分解为L和U的乘积，即A=LU。

LU分解的联系包括：

- 线性方程组求解：LU分解在线性方程组求解中得到广泛应用，因为它可以将线性方程组转换为上三角矩阵的求解问题。
- 奇异值分解：LU分解在奇异值分解中也得到广泛应用，因为它可以用来计算矩阵的奇异值和奇异向量。
