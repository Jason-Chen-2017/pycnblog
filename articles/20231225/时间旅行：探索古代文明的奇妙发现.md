# 4.1 时间旅行算法的具体代码实例
以下是一个简单的时间旅行算法的具体代码实例：

```python
import time

def time_travel(total_time, resolution, jump_count):
    current_time = 0
    for _ in range(jump_count):
        current_time += total_time / resolution
    return current_time

total_time = 1000
resolution = 10
jump_count = 10

start_time = time.time()
end_time = time_travel(total_time, resolution, jump_count)
print(f"回到过去的时间：{end_time - start_time}")
```

在函数内部，我们使用了一个 `for` 循环来实现时间的跳跃。在每次跳跃中，我们将当前时间 `current_time` 加上总时间 `total_time` 除以时间分辨率 `resolution` 的结果。这样，我们就实现了回到过去的目的。
