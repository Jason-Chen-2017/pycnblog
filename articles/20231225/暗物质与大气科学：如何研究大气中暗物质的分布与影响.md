## 3.1 气候模型
气候模型是一种数值模拟方法，用于预测气候变化和气候模式。气候模型通常包括以下几个部分：

1. 大气模型：大气模型用于预测大气中的气温、湿度、风速等变量。大气模型通常基于Navier-Stokes方程组，这是流体动力学中的一组基本方程。Navier-Stokes方程组可以用来描述大气中的流动和热传导。

2. 海洋模型：海洋模型用于预测海洋中的温度、湿度、盐度等变量。海洋模型通常基于海洋动力学方程组，这是一组用于描述海洋中的流动和热传导的方程。

3. 地表模型：地表模型用于预测地表中的温度、湿度、冰厚等变量。地表模型通常基于地球热传导方程组，这是一组用于描述地表中的热传导的方程。

4. 辐射传输模型：辐射传输模型用于预测大气中的辐射传输，包括太阳辐射和地球辐射。辐射传输模型通常基于辐射传输方程组，这是一组用于描述辐射传输的方程。

    T_atmosphere = T_surface + L_vaporization * (p_surface / p_surface - 1)

T_atmosphere, R_atmosphere = build_atmospheric_model(T_surface, R_dry_air, p_surface, T_lifted_condensation, L_vaporization)

# 分析结果

plt.ylabel('Relative Humidity')
