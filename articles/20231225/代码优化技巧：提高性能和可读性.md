### 4.1 循环优化实例

以下是一个简单的循环优化实例：

```python
# 原始代码
def sum_of_array(arr):
    total = 0
    for i in range(len(arr)):
        total += arr[i]
    return total
```

我们可以对这个代码进行优化，以提高性能：

```python
# 优化后代码
def sum_of_array(arr):
    return sum(arr)
```

在这个优化后的代码中，我们使用了内置的sum()函数来计算数组的和，这样可以减少循环内的操作，从而提高代码的性能。

```python
        else:
```

以下是一个简单的数据结构优化实例：

```python
# 原始代码
def unique_elements(arr):
    result = []
    for i in range(len(arr)):
        if arr[i] not in result:
            result.append(arr[i])
    return result
```

我们可以对这个代码进行优化，以提高性能：

```python

    return list(set(arr))
```

在这个优化后的代码中，我们使用了内置的set()函数来删除数组中的重复元素，从而提高代码的性能。
