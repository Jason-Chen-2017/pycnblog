# 2.核心概念与联系

## 2.1 凸集分离定理的基本概念

凸集分离定理是一种用于解决多类别分类问题的方法，它的核心思想是将数据集划分为多个不相交的子集，每个子集对应一个类别。具体来说，凸集分离定理可以用于解决以下问题：

1. 二分类问题：将数据集划分为两个子集，一个对应正类，另一个对应负类。
2. 多类别问题：将数据集划分为多个子集，每个子集对应一个类别。

凸集分离定理的核心概念包括：

1. 凸集（Convex Set）：一个包含于自然数域上的实数集合，如果对于任意两个元素a,b在集合中，它们的线性组合也在集合中。
2. 凸函数（Convex Function）：一个函数f(x)，如果对于任意x,y在域上，满足f(λx+(1-λ)y)≤λf(x)+(1-λ)f(y)，其中0≤λ≤1。
3. 支持 hyperplane：一个分离超平面，将两个凸集分开。
