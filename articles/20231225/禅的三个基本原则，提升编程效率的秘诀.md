我们将实现一个简单的排序算法，即冒泡排序。冒泡排序是一种简单的比较排序算法，它的基本思想是通过多次遍历待排序的元素，每次遍历都会将最大的元素移动到最后一个位置。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):

                arr[j], arr[j+1] = arr[j+1], arr[j]

```

这个代码实例实现了一个简单的冒泡排序算法。冒泡排序的基本思想是通过多次遍历待排序的元素，每次遍历都会将最大的元素移动到最后一个位置。在这个实现中，我们使用了两个嵌套的for循环，第一个循环遍历整个数组，第二个循环遍历未排序的部分数组。如果当前元素大于下一个元素，我们就交换它们的位置。这个过程会一直持续到整个数组被排序为非递减顺序。
