## 4.1 动态规划算法实例
以下是一个0-1背包问题的动态规划算法实例：
```python
def knapsack(weights, values, capacity):
    n = len(weights)
    dp = [[0 for _ in range(capacity + 1)] for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if weights[i - 1] <= w:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - weights[i - 1]] + values[i - 1])
            else:
                dp[i][w] = dp[i - 1][w]
    return dp[n][capacity]
```
在这个实例中，我们定义了一个目标函数`knapsack`，它接受一个物品的重量列表`weights`、价值列表`values`和背包的容量`capacity`作为参数。我们使用动态规划算法来解决0-1背包问题。我们创建了一个二维表格`dp`来存储子问题的解。我们使用两个嵌套循环来解决问题。在每个循环中，我们检查物品的重量是否小于或等于背包的容量。如果是，我们使用`max`函数来选择最大的价值。如果不是，我们使用`dp[i - 1][w]`来保持之前的解。最后，我们返回`dp[n][capacity]`作为问题的解。

## 4.2 贪心算法实例
以下是一个旅行商问题的贪心算法实例：
```python
def traveling_salesman(distances):
    n = len(distances)
    total_distance = 0
    path = [0]
    while len(path) < n:
        min_distance = float('inf')
        min_index = -1
        for i in range(n):
            if i not in path and distances[path[-1]][i] < min_distance:
                min_distance = distances[path[-1]][i]
                min_index = i
        path.append(min_index)
        total_distance += min_distance
    return total_distance, path
```
在这个实例中，我们定义了一个目标函数`traveling_salesman`，它接受一个距离矩阵`distances`作为参数。我们使用贪心算法来解决旅行商问题。我们使用一个`while`循环来遍历所有的城市。在每个循环中，我们使用`min`函数来选择最小的距离。如果距离小于当前的最小距离，我们更新最小距离和最小索引。最后，我们返回总距离和路径。
