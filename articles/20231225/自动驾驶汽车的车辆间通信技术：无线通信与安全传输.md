## 4.2安全传输技术的实现

以下是一个使用TLS（Transport Layer Security）协议实现安全传输的代码实例：

```python
import socket
import ssl

def create_socket(host, port):
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    sock.connect((host, port))
    return sock

def create_ssl_socket(sock):
    context = ssl.create_default_context()
    ssl_sock = context.wrap_socket(sock, server_side=False)
    return ssl_sock

if __name__ == "__main__":
    host = "localhost"
    port = 12345
    sock = create_socket(host, port)
    ssl_sock = create_ssl_socket(sock)
    data = ssl_sock.recv(1024)
    print("Received data:", data.decode())
    ssl_sock.close()
```

在这个代码实例中，我们首先导入了必要的库和模块，包括socket、ssl等。接着，我们定义了一个名为create_socket的函数，它用于创建一个socket对象并连接到指定的主机和端口。接着，我们定义了一个名为create_ssl_socket的函数，它用于创建一个ssl对象并将其与socket对象绑定。

在主函数中，我们首先调用create_socket函数创建一个socket对象并连接到指定的主机和端口。接着，我们调用create_ssl_socket函数创建一个ssl对象并将其与socket对象绑定。然后，我们使用ssl_sock.recv()方法接收数据，并将其解码为字符串。最后，我们使用ssl_sock.close()方法关闭ssl对象。
