```python
class FaultTolerancePattern:
    def __init__(self):
        self.status = "normal"

    def monitor(self):
        if self.status == "abnormal":
            self.recover()
        else:
            print("System status: ", self.status)

    def recover(self):
        print("System recovering...")
        self.status = "normal"

pattern = FaultTolerancePattern()
pattern.status = "abnormal"
pattern.monitor()
```

在这个代码实例中，我们定义了一个名为 `FaultTolerancePattern` 的类，该类包含两个方法：`monitor` 和 `recover`。`monitor` 方法用于监测系统的状态，如果系统状态为 "abnormal"，则调用 `recover` 方法进行故障恢复。`recover` 方法用于实现故障恢复，将系统状态设置为 "normal"。

通过运行这个代码实例，我们可以看到系统在出现故障时能够自动恢复并继续正常运行的能力。
