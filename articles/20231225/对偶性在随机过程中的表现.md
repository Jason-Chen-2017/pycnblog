# 2.核心概念与联系

## 2.1 随机过程

随机过程（stochastic process）是一个随时间变化的随机系统，它可以用一组随机变量的序列来描述。随机过程的每个时刻都有一个随机变量，这些随机变量可以是连续型的或离散型的。随机过程可以根据其时间域和状态空间的特性进一步分类为：

- 有限时间随机过程：时间范围有限，通常用整数序列表示。
- 无限时间随机过程：时间范围为无限，通常用实数序列表示。
- 有限状态随机过程：状态空间有限，通常用向量或矩阵表示。
- 无限状态随机过程：状态空间为无限，通常用序列或图表表示。

## 2.2 对偶性

对偶性（duality）是一种在数学、物理和工程等领域中广泛存在的概念，它描述了某种性质或关系的相互对应关系。在随机过程中，对偶性通常用于描述和分析随机过程的一些性质，如期望、方差、协方差等。

对偶性在随机过程中的表现主要体现在以下几个方面：

- 期望的对偶性：随机过程的期望可以通过对偶性得到描述和计算。
