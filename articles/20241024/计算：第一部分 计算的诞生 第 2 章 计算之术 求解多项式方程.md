                 

### 第1章 引言

在计算机科学和数学中，计算是一个基础而又关键的概念。计算不仅是我们日常生活中不可或缺的一部分，也是现代科技和工业进步的核心动力。从简单的算术运算到复杂的模拟与优化，计算能力在各个领域中都发挥着重要作用。本章将介绍计算的重要性，计算的发展历史，以及本书的整体结构。

#### 1.1 计算的重要性

计算在现代社会中具有无可替代的重要性。首先，计算是信息时代的基础，几乎所有的数字设备——从个人计算机到智能手机、服务器和云计算平台——都依赖于计算能力。无论是浏览网页、发送电子邮件、观看视频，还是进行复杂的科学计算，计算都是必不可少的。

其次，计算是推动科技创新的重要驱动力。在人工智能、机器学习、自动驾驶、生物信息学等领域，计算能力的提升直接决定了技术的进步和应用的广度。例如，深度学习算法需要大量的计算资源来训练复杂的神经网络，而量子计算则有望带来全新的计算范式。

此外，计算还在经济、社会、科学等多个领域产生深远影响。在经济学中，计算模型用于分析市场动态、优化资源分配；在社会学中，计算方法用于数据分析和社会模拟；在自然科学中，计算科学为复杂的物理现象建模和预测提供了强有力的工具。

#### 1.2 计算的发展历史

计算的历史可以追溯到古代，当时人们使用简单的算术工具进行计算。随着时间的推移，计算工具和技术的不断进步，计算的发展经历了几个重要的阶段。

1. **古代计算**：最早的计算工具可以追溯到古代中国、印度和阿拉伯，包括算盘和筹算。这些工具极大地提高了人们的计算效率，但仍然是手工操作。

2. **机械计算**：17世纪，数学家布莱兹·帕斯卡发明了第一台机械计算器，标志着计算从手工操作向机械化转变。随后的几百年中，计算器经历了多次改进，如莱布尼茨发明的差分机和对数机。

3. **电子计算**：20世纪中叶，电子计算机的出现是计算史上的一个重大突破。冯·诺伊曼的存储程序计算机概念为现代计算机体系结构奠定了基础。早期的电子计算机如ENIAC和EDVAC，虽然在体积和速度上与今天的计算机不可同日而语，但它们展示了计算能力的巨大潜力。

4. **计算机革命**：20世纪70年代以来，计算机技术经历了快速发展，微型计算机和后来的个人计算机使计算技术进入了千家万户。这一时期，计算硬件和软件的快速发展，推动了计算应用的普及和深入。

#### 1.3 本书的结构

本书旨在系统地介绍计算的基础知识、核心算法和实际应用。全书分为三大部分，每一部分都围绕不同的主题进行深入探讨。

## 第2章：计算之术 求解多项式方程

### 2.1 多项式方程的基本概念

多项式方程是数学中的一个重要概念，它在计算科学和工程应用中有着广泛的应用。本章将详细介绍多项式方程的基本概念，包括多项式的定义、多项式方程的定义以及多项式方程的分类。

#### 2.1.1 多项式的定义

多项式是由一系列的项组成的表达式，每一项包括一个系数和一个或多个变量的幂次。在数学中，一个多项式的一般形式可以表示为：

$$ p(x) = a_nx^n + a_{n-1}x^{n-1} + \cdots + a_1x + a_0 $$

其中，$a_n, a_{n-1}, \ldots, a_1, a_0$ 是常数，称为多项式的系数，$x$ 是变量。如果 $a_n \neq 0$，则多项式的次数为 $n$。

多项式的系数可以是任意实数，而变量 $x$ 可以是任意实数、复数或者在其他类型的域上。多项式可以根据其系数是否为零来分类，例如，常数项为零的多项式称为一次多项式，最高次数为2的多项式称为二次多项式，依此类推。

#### 2.1.2 多项式方程的定义

多项式方程是一个含有未知数的多项式等式，其中等式右侧为0。一个多项式方程的一般形式可以表示为：

$$ a_nx^n + a_{n-1}x^{n-1} + \cdots + a_1x + a_0 = 0 $$

其中，$a_n, a_{n-1}, \ldots, a_1, a_0$ 是常数，$x$ 是未知数。这个方程中的最高次数决定了多项式方程的阶数。例如，上述方程是一个 $n$ 阶多项式方程。

多项式方程的解是使方程成立的未知数的值。这些解被称为多项式方程的根或零点。一个多项式方程可能有多个根，这些根可以是实数、复数或者属于其他类型的域。

#### 2.1.3 多项式方程的分类

多项式方程可以根据其系数和次数进行分类。以下是一些常见的分类：

1. **一次多项式方程**：最高次数为1的多项式方程，形式为 $ax + b = 0$，其中 $a$ 和 $b$ 是常数，$a \neq 0$。

2. **二次多项式方程**：最高次数为2的多项式方程，形式为 $ax^2 + bx + c = 0$，其中 $a, b, c$ 是常数，$a \neq 0$。

3. **三次多项式方程**：最高次数为3的多项式方程，形式为 $ax^3 + bx^2 + cx + d = 0$，其中 $a, b, c, d$ 是常数，$a \neq 0$。

4. **四次多项式方程**：最高次数为4的多项式方程，形式为 $ax^4 + bx^3 + cx^2 + dx + e = 0$，其中 $a, b, c, d, e$ 是常数，$a \neq 0$。

5. **高次多项式方程**：次数大于4的多项式方程，形式类似于上述四次多项式方程，但次数更高。

根据多项式方程的系数，还可以将其分类为有理系数多项式方程和无理系数多项式方程。有理系数多项式方程的系数是有理数，而无理系数多项式方程的系数包含无理数。

#### 2.1.4 多项式方程的性质

多项式方程具有以下一些重要性质：

1. **唯一性**：如果多项式方程有解，则解是唯一的。这是因为多项式函数是严格单调的，因此在任何区间内只有一个根。

2. **有界性**：多项式方程的解在实数范围内是有界的。这意味着多项式方程的解不可能无限大或无限小。

3. **代数基本定理**：每个非零多项式方程至少有一个复数根。这是复分析中的一个基本定理。

4. **代数闭域**：复数域是一个代数闭域，这意味着任何在复数域上的多项式方程都可以在复数域内找到其根。这是复分析中的一个基本定理。

这些性质使得多项式方程在数学和计算机科学中具有重要的地位，并且为多项式方程的求解提供了理论基础。

### 2.2 因式分解法求解多项式方程

因式分解法是求解多项式方程的一种基本方法。这种方法的基本思路是将多项式方程因式分解，使得每个因式等于零，从而求解出未知数的值。以下是一些具体的步骤和例子。

#### 2.2.1 因式分解法的基本思路

因式分解法的基本思路是将多项式方程分解为几个因式的乘积，每个因式等于零。具体步骤如下：

1. **提取公因式**：首先，尝试从多项式中提取公因式，使得多项式可以表示为几个因式的乘积。

2. **因式分解**：对于提取出公因式后的多项式，尝试将其分解为几个更简单的因式的乘积。

3. **求解因式**：将每个因式等于零，求解出未知数的值。

4. **合并解**：如果多项式方程有多个解，需要将所有解合并起来，得到最终的解集。

#### 2.2.2 因式分解法的应用

因式分解法可以应用于多种类型的多项式方程。以下是一个例子：

#### 例题1：因式分解法求解 $x^3 - 2x^2 - 5x + 6 = 0$

**步骤1：提取公因式**

首先，观察多项式 $x^3 - 2x^2 - 5x + 6$，发现可以将其分解为 $(x - 2)(x^2 - 5x + 6)$。

**步骤2：因式分解**

接下来，对因式 $x^2 - 5x + 6$ 进行因式分解。找到两个数，它们的和为 $-5$，乘积为 $6$。这两个数是 $-2$ 和 $-3$，因此可以将 $x^2 - 5x + 6$ 分解为 $(x - 2)(x - 3)$。

**步骤3：求解因式**

将每个因式等于零，得到 $x - 2 = 0$ 和 $x - 3 = 0$。

解得 $x = 2$ 和 $x = 3$。

**步骤4：合并解**

因此，多项式方程 $x^3 - 2x^2 - 5x + 6 = 0$ 的解集为 $\{2, 3\}$。

#### 例题2：因式分解法求解 $x^2 + 2x + 1 = 0$

观察多项式 $x^2 + 2x + 1$，可以发现它是一个完全平方三项式，可以因式分解为 $(x + 1)^2$。

将 $(x + 1)^2 = 0$，得到 $x + 1 = 0$。

解得 $x = -1$。

因此，多项式方程 $x^2 + 2x + 1 = 0$ 的解集为 $\{-1\}$。

#### 2.2.3 因式分解法的注意事项

在使用因式分解法求解多项式方程时，需要注意以下几点：

1. **因式分解的唯一性**：多项式的因式分解不是唯一的。例如，多项式 $x^3 - 2x^2 - 5x + 6$ 可以因式分解为 $(x - 2)(x^2 - 5x + 6)$，也可以因式分解为 $(x - 3)(x^2 + x - 2)$。

2. **非零系数**：在进行因式分解时，需要注意多项式方程中的系数不能为零。否则，无法进行因式分解。

3. **多项式的可因式分解性**：并非所有的多项式都可以因式分解。例如，某些多项式可能无法分解为整数系数的因式。

4. **多个解的情况**：多项式方程可能存在多个解。例如，$x^2 + 2x + 1 = 0$ 有两个相同的解，$x = -1$。

因式分解法是一种求解多项式方程的有效方法，尤其在多项式方程的系数为整数时，因式分解法具有直观性和简洁性。然而，当多项式方程的系数为复数或有理数时，因式分解可能变得复杂。在这种情况下，求根公式法和数值方法可能更为适用。

### 2.3 求根公式法求解多项式方程

求根公式法是求解多项式方程的一种常见方法，特别适用于二次多项式方程。这种方法基于多项式方程的系数，通过公式直接计算得到方程的根。以下将详细描述求根公式法的基本概念、公式和具体应用。

#### 2.3.1 求根公式的基本概念

求根公式，又称二次公式，是用于求解二次多项式方程的公式。对于一般形式的二次多项式方程：

$$ ax^2 + bx + c = 0 $$

其中，$a, b, c$ 是常数，$a \neq 0$，求根公式为：

$$ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} $$

这个公式给出了二次方程的两个根，分别对应于 $\pm$ 符号的选择。其中，$b^2 - 4ac$ 称为判别式，它决定了方程根的性质：

1. **判别式大于0**：$b^2 - 4ac > 0$，方程有两个不相等的实数根。
2. **判别式等于0**：$b^2 - 4ac = 0$，方程有两个相等的实数根。
3. **判别式小于0**：$b^2 - 4ac < 0$，方程没有实数根，只有复数根。
