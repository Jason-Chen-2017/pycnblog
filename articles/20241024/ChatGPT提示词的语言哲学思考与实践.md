**3.2.1.2 代码实现与分析**

以下是使用 Python 编写的新闻文章生成代码：

```python
import openai
import os

# 设置 API 密钥
openai.api_key = os.getenv("OPENAI_API_KEY")

# 训练模型
model = openai.GPT2Model.from_pretrained("gpt2")

# 生成新闻文章
def generate_news_article(title):
    prompt = "标题：" + title + "\n请根据标题生成一篇新闻文章："
    response = model.generate(prompt)
    return response.decode("utf-8")

# 测试
title = "我国成功发射嫦娥五号探测器"
article = generate_news_article(title)
print(article)
```
