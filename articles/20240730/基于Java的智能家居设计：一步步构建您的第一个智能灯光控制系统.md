### 5.1 开发环境搭建

### 5.2 源代码详细实现

以下是使用Java和Spring Boot实现的智能灯光控制系统的源代码：

```java
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api")
public class LightController {
    // 灯光设备状态
    private List<Light> lights = new ArrayList<>();
    
    @GetMapping("/lights")
    public List<Light> queryLights() {
        return lights;
    }
    
    @PostMapping("/lights")
    public void controlLight(@RequestBody Light light) {
        for (Light existingLight : lights) {
            if (existingLight.getId().equals(light.getId())) {
                existingLight.setStatus(light.getStatus());
                break;
            }
        }
    }
}

// 灯光设备类
class Light {
    private String id;
    private String status;
    
    public String getId() {
        return id;
    }
    
    public void setId(String id) {
        this.id = id;
    }
    
    public String getStatus() {
        return status;
    }
    
    public void setStatus(String status) {
        this.status = status;
    }
}
```

通过上述代码，实现了RESTful API接口，提供对灯光设备的状态查询和控制功能。

### 5.3 代码解读与分析

本文的源代码主要分为两部分：

- **控制器**：使用Spring Boot的@RestController注解，定义了两个API接口，分别用于查询和控制灯光设备。
- **灯光设备类**：定义了灯光设备的属性和方法，用于存储和控制灯光设备的状态。

控制器中使用@GetMapping和@PostMapping注解，分别定义了GET和POST请求的API接口，实现了灯光设备的查询和控制功能。
