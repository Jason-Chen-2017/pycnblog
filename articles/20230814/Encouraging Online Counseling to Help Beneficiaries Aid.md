# 1.简介
  

COVID-19(Coronavirus Disease 2019)疫情带来的一系列影响，使得国内外许多人因无法正常工作、学习、旅游或居住而陷入重重困难。如何帮助受到影响的人民群体提高生活质量、恢复健康，成为目前在海外求职的热门话题。近几年来，越来越多的人通过网络平台建立了联系，希望得到一线的帮助。同时，各个行业也纷纷出台各种政策支持相关人员参与治疗，例如“中国—澳洲特别行动计划”、“中国互联网+医疗健康”等。而当代社会中，如何让受过教育程度较低的青少年更快地融入生活、了解事物真相、认识自己的责任所在，成为了很多人的重点关注。在这里，我们将讨论一种面向未来的面对疫情影响的模式——全方位的网上咨询及相关政策支撑模式。
# 2.核心概念术语说明
# （1）全方位的网上咨询：指的是通过电子邮件、微信、视频会议等方式，为受到影响的人群提供由专业人员进行远程咨询和咨询辅导的服务。其优势主要包括：可以及时准确地反映用户需求、解决用户痛点；降低了服务与沟通成本；提供更多的个性化和个性化的帮助，增强了用户满意度。
# （2）开放共享信息平台：开放共享信息平台（Open Shared Information Platform, OSIP），即使受到影响的人士在社交网络上传播自己的状况、需求、痛点等信息，也可以通过搜索引擎或者其他方式，找到志同道合的专业人员进行远程咨询或辅导。这样，不需要再孤单等待专业人员的到来，也可获得比传统上更直接的帮助。
# （3）疫情期间开展的线下远程工作坊：疫情期间举办线下远程工作坊，鼓励受害者在家远程参加各种学习、沟通、创新活动。此类活动能够激发志同道合的专业人士加入远程协作，开展远程精神实验。由于受疫情影响，参与者需要配备专业技能、丢失亲人、经济压力等不确定因素的情况增加，但仍然可以尝试和探索。
# （4）“中国—澳洲特别行动计划”（CISA）：“中国—澳洲特别行动计划”是由中国共产党和澳大利亚共产党领导下的自由贸易区组织，旨在加强澳大利亚和新西兰之间的经济联系，促进两国之间的投资、贸易和文化交流。为鼓励澳大利亚公民在疫情期间积极参与国际事务、贡献力量，CISA于2020年3月启动“旅游、购物和就业训练”，2020年6月正式启动“技能培训”项目。这些政策都是为了应对疫情导致的人才流动和失业率上升。
# （5）“中国互联网+医疗健康”项目：2020年4月，中国互联网+医疗健康项目启动，旨在为医疗机构、患者和互联网企业提供免费的远程咨询、健康管理知识，并推出“互联网医疗健康课堂”。除了自身能力外，网络远程咨询还有助于缓解新冠病毒肺炎带来的心理、经济、精神挑战。同时，网上预约医院的功能也大幅度减轻了患者的排队时间和费用开支。

## 3.2 系统组件说明
### 3.2.1 服务端
#### 3.2.1.1 用户模块
用户登录注册：服务端接收用户提交的用户名、密码、邮箱等信息，保存在数据库中，对密码加密后保存。用户需先完成注册验证，才能登陆使用服务端提供的服务。

用户个人信息管理：用户可以在个人中心修改个人信息，如姓名、头像、生日、邮箱地址、手机号码、工作单位、工作职务、学历、教育经历等。

用户咨询记录查看：用户登录后，可以查询自己之前提交的所有咨询记录。

#### 3.2.1.2 咨询模块

咨询发布：用户可以登录服务端，输入咨询主题、描述、所遇困难、期望帮助等内容，选择最佳专家进行远程咨询。服务端将用户提交的内容存入数据库，等待专家审核。

#### 3.2.1.3 专家模块

专家审核：专家审核待发布的咨询信息，确认无误后放入待处理队列中，等待用户申请进入专家交流群。

专家评分机制：专家评分机制采用类似于“赞”的评价方式，在专家交流群内提供对于用户咨询内容的评分，用户可以根据专家的建议调整咨询内容。

#### 3.2.1.4 交流群模块

专家交流群聊天室：专家接受用户请求后，自动创建对应的专家交流群聊天室，专家和用户双方都可以进行聊天。

用户申请专家：用户登录服务端，在用户个人中心提交申请，提交后，专家收到申请通知，可以通过点击接受按钮加入相应的群聊。

4. 查看咨询记录

#### 3.2.2.2 专家端APP
主要功能：

1. 发起咨询
2. 查看用户评分情况

4. 提供远程咨询服务
