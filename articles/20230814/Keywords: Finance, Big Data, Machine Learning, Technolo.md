# 2.背景介绍
由于人类一直处在信息化时代，财经数据被大量收集，从而形成了海量的数据资源。如何进行有效的分析和决策，成为一门重要的技术。传统上，人们采用过去比较简单的统计方法对数据进行分析，但随着数据量的增加，传统的方法已经无法应付这种复杂的数据。这时候，人们又转向更加复杂的方法——机器学习(Machine Learning) 。机器学习可以让计算机通过数据自动分析获得新的知识，帮助它预测未来的事件或做出决策。
人工智能领域的研究越来越火热，涉及到对图像处理、自然语言处理等领域，都极有可能用到机器学习技术。近年来，无论是企业还是个人都意识到了这一点，也许是因为我们越来越依赖机器学习技术，而对于它的认识还不够充分。因此，我想用一些视觉和动感的词汇来描述一下这一话题。
图1: 数据产业的变化
数据产业的革命性变革，使得各种数据的产生和收集变得越来越容易，包括金融数据、运营数据、医疗数据、社会数据、日常生活数据等。如今，世界正在进入一个数据驱动的时代，整个金融界和其他行业都正在响应这个变革，都意识到要把数据变成有价值的资产，并利用数据精准洞察客户行为，提升公司产品质量，降低成本，提高竞争力。
图2: 大数据时代下的金融系统
由于大数据技术的应用，使得各行各业都能够掌握海量数据，实现互联网金融服务。这将引发金融界的一场深刻变革，金融机构将面临新型的商业模式变革，通过新形式的数据分析，更好地进行金融投资决策。同时，科技巨头也会进军其中，积极探索新的商业模式。
图3: 数字化金融产业
截至目前，国际上已经有超过200个国家跟随美国开启了数字化金融体系。银行、保险、证券、电子支付、金融信息平台、交易所、资讯、媒体等各方都涌现出不同形式的数字化服务，助力传统金融业务的数字化升级，促进全球经济的发展。
