# 2.代码片段
“代码片段”这一概念很容易理解，它指的是简短的可运行的代码块或者脚本。一般来说，代码片段可以用来验证某个函数、方法是否正常工作，或者用于调试某些特定问题。如何编写高效、易懂且有用的代码片段也是本文关注的重点。下面我们举几个代码片段作为例子。
## 计算两数之和（Python）
```python
def add_numbers(a, b):
    return a + b

print(add_numbers(3, 7)) # Output: 10
```
上述代码片段实现了一个函数`add_numbers`，该函数接受两个参数`a`和`b`，并返回它们的和。代码中也用到了关键字`return`语句，这意味着函数的执行结果会被保存起来并输出。通过这种方式，你可以把函数当作一个黑盒来使用，只需要输入相应的参数即可得到预期的结果。除此之外，你也可以对函数进行修改，使其具有更多功能。比如，你可以添加打印日志的功能，记录每次调用函数时的输入值和输出值等。
## 检查元素是否存在于列表中（Python）
```python
my_list = [1, 2, 3]

if 2 in my_list:
    print("Element found!")
else:
    print("Element not found.")
```
上述代码片段检查列表`my_list`中的是否包含数字2。首先，定义了一个空的列表`my_list`。然后用关键字`in`判断数字2是否在`my_list`中。如果找到了，则打印"Element found!"；否则，打印"Element not found."。代码非常简单易懂，不需要注释，但足够完成任务。
## 将字符串反转（Python）
```python
def reverse_string(s):
    s = list(s)
    left, right = 0, len(s)-1

    while left < right:
        s[left], s[right] = s[right], s[left]

        left += 1
        right -= 1

    return ''.join(s)

print(reverse_string('hello')) # Output: 'olleh'
```
上述代码片段实现了一个函数`reverse_string`，该函数接受一个字符串参数`s`，并返回该字符串的反转形式。首先，把字符串转换成列表，并得到列表长度的一半作为索引左右指针。然后，使用双指针法反向遍历列表，交换每个元素的位置，直到左右指针相遇。最后，再把反转后的字符组装成字符串返回。代码中用到了一些循环、条件语句和赋值运算符，但它们都比较简单易懂。至于为什么要使用这种方法反转字符串，这是因为字符串是按照字母顺序排列的，逆序之后可以方便地查找或者匹配字符串中的指定内容。
