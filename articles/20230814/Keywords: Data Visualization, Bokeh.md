# 1.简介
  

Bokeh是一个基于Python的开源数据可视化库，它可以用于快速创建交互式Web应用程序和富交互式文档，并且具有简单而强大的功能集，包括高级交互工具、易于扩展的数据源支持和自定义主题等。以下将对Bokeh进行概述并阐述其基本用法。

## 1.1 Bokeh介绍
Bokeh是一个开源数据可视化库，最初由John McInerney开发，目前属于被Vega（一种高性能的声明式语法）和Vega-Lite（一种简洁的声明式语法）支持的第三方库。主要特性包括：

1. 可视化语言：支持多种可视化语法，如图形（如线条、点、直方图）、矢量图（如圆圈、标记、路径）和文本。
2. 框架内置工具：提供了丰富的内置工具，可方便地实现各种类型的数据可视化效果，如饼图、柱状图、散点图、折线图等。
3. 可扩展性：提供低级可视化接口，可以通过组合多个元素来实现复杂的可视化效果。
4. 交互能力：支持高级交互机制，如缩放、平移、拖动、悬停、单击、双击等。
5. 跨平台兼容：适用于各种浏览器、操作系统、显示器设备等。

### 1.2.2 基础示例
下面给出一个简单的Bokeh的基础示例，创建一个空白的HTML文件，然后绘制一条线段。

```python
from bokeh.plotting import figure, output_file, show

output_file("line.html")

p = figure(plot_width=400, plot_height=400)
p.line([1, 2, 3], [1, 2, 3])

show(p)
```

## 1.3 数据可视化的基本概念
### 1.3.1 数据类型
数据可视化中，最基本的数据单位是**观察值（observation）**，比如一组年龄、身高、体重或其他衡量指标的值，这些观察值在特定的坐标系下以不同的形式呈现出来。不同的数据类型对应着不同的可视化效果，常用的包括：

- **点图（scatter plots）**：表示的是变量之间的关系，可用来探索数据之间的相关性。例如，散点图可以展示变量之间的正相关关系；气泡图则可以更好地反映出各个观察值的大小。
- **柱状图（bar charts）**：表示的是一组分类变量的数值分布，通常用来比较不同组别的数量。
- **线图（line charts）**：表示的是一系列随时间变化的观察值。
- **面积图（area graphs）**：类似于线图，但在两个轴上画出曲线。
- **箱型图（box and whisker plots）**：提供了一种简洁的方式来描述数据的分布和范围，可用来识别异常值。
- **热力图（heatmaps）**：提供了一种很有效的方式来可视化矩阵数据，比如每个观察值都关联着一个特定颜色。
- **堆叠图（stacked area or bar charts）**：用于同时显示两个或多个变量之间的变化规律。
- **条形图（pie charts）**：表示的是各个分类项的相对大小，通过颜色或字体等其他视觉编码区分不同类别。

### 1.3.2 可视化方法
- **映射法（mapping method）**：即把观察值映射到一些二维平面上的点、线或区域，称为**视觉元素**（visual element）。这种方法要求观察值要有具体的空间意义，否则就无法进行可视化。
- **统计法（statistical method）**：通过计算、概率统计、聚类分析等方式，从原始数据中抽象出一些统计量，再根据这些统计量推导出某些结论。这种方法不需要考虑具体的空间意义，因此能够对任意一组观察值进行可视化。
- **符号法（symbolization method）**：也是通过符号编码的方式来显示信息，但是它一般没有提供足够的信息来对数据进行细粒度地分析。
- **比较法（comparative method）**：对两种或更多的观察值进行比较，得出它们之间的差异。这种方法也不需要考虑具体的空间意义，但它提供了一种很直观的方式来观察到底哪个观察值更好。

总之，数据可视化的方法并不是一成不变的，而是随着应用需求和数据本身的特点而有所变化。选择合适的方法可以帮助我们理解数据并发现其中的模式。
