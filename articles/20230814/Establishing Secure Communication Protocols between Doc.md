# 1.简介
  
  
随着疫情的到来，医疗保健行业面临着巨大的挑战。首先，疫情带来的经济损失将是不可估量的。其次，在医院中进行核酸检测、确诊患者治疗、护理等过程中，存在着一定的风险隐患，尤其是在面对复杂的病情时，可能会造成生命财产损失或意外伤害。因此，建立起更加安全的通信协议，是重点关注的重点。   
  
为了解决这个问题，本文提供了一种基于椭圆曲线密码算法（ECC）的密钥交换协议，用于医疗保健机构之间的医生用户之间的通信。该协议可以加密保存在云端的数据，避免数据的泄露。同时，协议还设计了多层级验证系统，以保证数据的完整性、真实性、可用性和身份认证。  

# 2. 关键术语  

## 2.1 椭圆曲线密码算法（ECC） 

椭圆曲线密码算法(Elliptic Curve Cryptography，简称 ECC)，一种快速而安全的公钥加密算法，由英国国家标准局（NIST）于 2006 年提出。它基于椭圆曲线离散对数问题（ECDLP）进行，其安全性依赖于两个随机数的不可预测性。    

椭圆曲线是一种相当形式化的几何图形，其切线是一个抛物线，这种图形具有特殊的性质。椭圆曲线通常用 $y^2 = x^3 + a*x + b$ 表示，其中 a 和 b 是两个不定参数。椭圆曲线的基点（也叫做基元）是 $(0,b)$ 和 $(q,0)$ ，其中 q 为素数。每条椭圆曲线都可以看作是由无限多个点所组成的集合，这些点通过共同的基点联系在一起。   

椭圆曲线密码算法的特点有以下几点：

1. 计算速度快：椭圆曲线加密和签名运算速度非常快。实际应用中一般每秒可完成数十万次以上运算。
2. 通信安全：椭圆曲线加密采用的是双方事先共享的公私钥，因此通信过程中的消息内容被加密，且难以破译。
3. 实现简单：椭圆曲线加密算法的公开参数少，易于部署。
4. 广泛运用：椭圆曲线加密算法已经得到广泛应用，包括TLS/SSL协议，OpenSSH，PGP，SMIME等。

## 2.2 加密学基础知识   

### 2.2.1 对称加密算法

对称加密算法又称为私钥加密算法，主要分为两种：

1. 数据加密标准（DES）
2. 高级加密标准（AES）

### 2.2.2 公钥加密算法

公钥加密算法采用非对称加密算法，也叫作公钥算法。公钥算法由两部分组成：公钥和私钥。其中，公钥用来加密数据，只有拥有私钥的用户才能解密；私钥用来解密数据，只有拥有公钥的用户才能加密。公钥加密算法有两种，分别为：

1. RSA
2. ElGamal
