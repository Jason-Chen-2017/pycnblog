# 1.简介
  

机器学习领域由于数据的快速增长而变得日益重要，但收集、处理和存储大量数据仍然是一个难题。为了解决这个问题，越来越多的人开始采用众包的方式进行数据的收集和标记工作。Jigsaw是一个通过众包的方式收集、整理、分析、分享高质量的数据集的平台。它的基本功能包括：

- 提供众包模式的数据标注工具；
- 智能分类器自动检测、整合、并过滤无效数据，提升标注效率；
- 支持多种语言的用户界面，让非计算机专业的用户也能轻松上手使用。

在2017年9月1日发布的第一版产品上线之后，Jigsaw已经成为众包数据集的标注平台和资源。截止到今年5月，其已被用作多个数据集标注工具的基础，包括COVID-19检测，音乐推荐系统，商品评论分类等。

基于以上优点，本文将详细介绍Jigsaw平台的基本功能以及使用方法。

# 2.基本概念术语说明
2.1 Jigsaw数据集
Jigsaw数据集是一个由众包用户提交的海量数据集合，其包含不同种类的标签。每个数据集都有一个唯一标识符，可用于对其进行引用。

2.2 数据标签
每条数据都有一个或多个标签，这些标签提供关于数据所属类别的信息，比如图像中的物体、图像中出现的场景、文本信息的内容、视频片段的事件类型等。Jigsaw允许用户为数据集中的每一个数据添加多个标签，每一个标签可附带自己的附加信息。

2.3 用户
Jigsaw支持多种类型的用户，包括匿名用户、实名用户和企业账户。实名用户可以登录Jigsaw管理后台查看自己提交的数据集，企业账户可享受数据集管理权限以及支持的特性。

2.4 模板
模板（Template）是指数据集的蓝图，即用户创建数据集时定义的各项参数和约定俗成的标准。模板为用户提供了一种简单、有效的方法来描述自己的真实数据集，从而减少了数据输入的错误、缺陷和噪声，也方便了其他用户对其进行参考和比较。Jigsaw目前共支持两种类型的模板：文字类模板和图像类模板。

2.5 算法
算法（Algorithm）是指用来训练模型的自动化过程，该过程根据模板生成的数据集来产生模型输出。算法是一种抽象的概念，它可能包括如分类、排序、回归等一系列操作。目前Jigsaw仅支持基于特征工程的算法。

2.6 拒绝标签
拒绝标签（Rejection Label）是指某条数据因为某种原因（例如其所属类别无法准确判断）而被标记为“不匹配”的标签。拒绝标签不会影响最终结果，但可以帮助研究人员区分出那些难以识别的数据。

2.7 AI训练器
AI训练器（Artificial Intelligence Trainer）是指智能助手，它们通过数据的自动标注和模拟人类标注过程，来评估模型在数据上的性能。训练器能够给出更及时的反馈，使模型开发过程更加顺畅、高效。

2.8 团队
团队（Team）是指由多名实名用户组成的小组，他们共享同一个标注任务，协同完成标注任务。团队可以分享数据集和算法，也可以获得额外的福利。团队成员可以修改自己标注的结果，并向其他团队成员提供建议。

2.9 数据集管理员
数据集管理员（Dataset Administrator）是指具有管理权限的企业账户，负责管理整个Jigsaw平台上的数据集。数据集管理员可设定访问权限，接收和处理用户提交的数据。他还可以创建、编辑、删除模板、管理团队和AI训练器等。

2.10 API
API（Application Programming Interface）是计算机软件与遵循特定协议的应用程序之间的接口。API可用于实现Jigsaw与其他应用或服务的通信。目前，Jigsaw支持通过RESTful API和GraphQL API进行数据的交互。

2.11 数据打包下载
数据打包下载（Data Packaging and Downloading）是指用户通过Jigsaw平台提交的数据集可以被打包压缩，并下载得到一份完整的数据文件。该文件可用于研究或商业用途。

2.12 数据集分析
数据集分析（Analysis of Dataset）是指对用户提交的数据集进行统计分析，得到有关数据的一些关键信息，如数据量、类别分布、标签分布、数据质量、标注效率等。分析结果可帮助研究者找出潜在的问题或瓶颈，并制定相应的优化策略。

2.13 搜索引擎
搜索引擎（Search Engine）是指通过文本、图像、视频、音频等方式检索和筛选相关数据集。搜索结果包括数据集名称、概述、关键词、分类等。

2.14 数据导出
数据导出（Export Datasets）是指用户可以从Jigsaw平台导出自己标注过的数据集。数据集的导出格式可以是JSON、CSV或XML。

2.15 数据集授权
数据集授权（Dataset Authorization）是指用户在Jigsaw平台上上传、标注或下载的数据集需要遵守所在地区法律的要求。

2.16 标签数据集
标签数据集（Labeled Dataset）是指经过初步标记的数据集，其中含有明确的标签。标签数据集通常已划分为训练集、验证集和测试集。

2.17 非标签数据集
非标签数据集（Unlabeled Dataset）是指没有进行初步标记的数据集。在这种情况下，用户需要手动或半自动的标记数据集。
