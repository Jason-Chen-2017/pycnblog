# 1.简介
  

随着信息化的发展和互联网的普及，计算机技术正在成为社会经济发展不可或缺的一部分。目前，全球有超过90%的职场人士都在使用计算机，每天都在触碰到计算机科技所带来的各种便利和方便。而就业方向、创新领域、前景等各个方面的需求也在不断增加。因此，计算机专业对于人才的培养和研究方向的确定，对求职者的就业前景有着至关重要的作用。

作为高等院校计算机专业学生，学习计算机专业的课程是十分必要的。但很多学生不知道如何从零开始学习这些技术。为了帮助学生更好地掌握和理解计算机知识，作者基于当前国内外计算机相关教材和经验，编写了一套适合于高校计算机专业学生的教材。

该教材分为上下两册。上册主要介绍计算机技术的基本原理、关键术语和核心算法。下册侧重于实际编程应用及项目实践，将核心技术应用于实际开发中，帮助学生加深对技术的理解和掌握。

本教材不仅适用于高校计算机专业本科生，还可以提供给本科毕业生参加计算机相关的实习岗位培训。同时，还可以在学习过程中了解到更多计算机专业的最新知识和技能。相信通过阅读本教材，能够提升学生的计算机素养，提高自身竞争力，开拓创新的机会。

# 3.基本概念和术语
## 3.1 计算机概论
首先，介绍计算机的一些基本概念和术语。

计算机是指利用数据处理器和其它装置组成的机械装置，用来存储、输入、输出信息和进行计算的机器。它具有自己独立运行的指令集，能够识别和执行指令，并产生各种运算结果。

计算机又分为两个系列：汇编语言和高级语言。汇编语言是一种直接用二进制代码表示计算机指令的方法。它一般被称为“机器语言”。高级语言是人类认识和使用的更易于交流和阅读的语言。编译器将汇编语言转换为高级语言，然后再由计算机运行。

汇编语言和高级语言的不同之处在于复杂程度和表达能力。汇编语言的语法简单，代码的可读性差；而高级语言的语法较复杂，代码的可读性好，往往可以做出更复杂的功能。

计算机有两种运行方式：个人电脑和服务器。个人电脑采用桌上型PC或笔记本电脑，一般只有一个CPU和主存（RAM）组成。而服务器一般由多块CPU、内存和磁盘组成，同时具备网络通信功能，用于承载海量数据。

还有一些其他术语需要了解，比如：

1. 数据：数据的最小单位叫做比特(bit)，数据总量叫做字节(byte)。

2. CPU：Central Processing Unit，中文翻译为中心处理单元，其是电脑的核心部件，负责接收用户指令、分析指令内容、执行指令并生成结果。

3. 指令：计算机指令就是计算机可以识别并执行的操作，它由助记符和操作码组成。助记符是指令的名称或缩略词，例如MOV、ADD、SUB等；操作码是指令的具体实现方法或功能，例如将指定寄存器的值复制到另一个寄存器中，将立即数值加到累加器中等。

4. 编码：编码是把信息变换成可以被计算机理解和识别的信息形式的过程，它决定了信息在传输、存储和处理过程中发生的变化，包括ASCII、UTF-8、GBK、Unicode等。

5. 操作系统：操作系统是一个管理计算机硬件和软件资源的程序，它是计算机的内核。操作系统负责管理计算机硬件设备，分配任务，协调进程，控制输入/输出设备，提供保护和安全环境，以及向用户提供各种服务和支持。

6. 网络：网络是将多台计算机连接起来的网络，由路由器、交换机、防火墙、VPN等构成。网络提供最高速的数据传输能力和安全保障。

7. 源码：源码是程序的原始文本，它包括了程序中的注释、标识符、变量名、函数名、语句序列、预处理指令等。它用于程序的编译和调试。

## 3.2 计算机网络协议
接下来介绍计算机网络协议。

计算机网络是指通过计算机互连共享信息和资源的通信系统，是人类活动范围扩大的现象。计算机网络是由网络节点、链路、传输协议、应用层协议等组成的网络系统。

计算机网络协议定义了计算机之间数据传输、计算机内部通信以及计算机之间的通信规则。计算机网络协议有许多种，常用的有TCP/IP协议、UDP协议、HTTP协议等。

1. TCP/IP协议
TCP/IP协议是Internet最基本的协议族，它包括三个协议：传输控制协议TCP、网际协议IP和互联网协议IGMP。

1) TCP协议
TCP是一种面向连接的、可靠的、基于字节流的传输层协议。它的建立连接过程如下图所示：


建立连接时，客户端先向服务器发送一个连接请求报文，其中包含了自己的初始化信息，如窗口大小、MSS（最大报文段长度）、最大序列号等；服务器收到后回复确认报文，同样包含了自己的初始化信息；此后双方可以正常通信，直到出现超时、连接关闭等原因终止连接。

2) IP协议
IP协议是网络层的协议，它的职责就是将不同主机间的数据包传送到目的地址。IP协议将数据包从源点到目的点传递，但是这个传递过程不能保证到达目的地址，这需要靠下一层的路由协议来实现。

3) IGMP协议
IGMP协议是Internet组管理协议，它用来管理主机加入或者退出多播组。

除此之外，还有许多其他的协议，比如ARP协议、RARP协议、BOOTP协议、DHCP协议等。

2. HTTP协议
HTTP协议是Hypertext Transfer Protocol的缩写，也就是超文本传输协议。它是基于TCP/IP协议族的应用程序协议。其主要特点是在请求-响应模型基础上，增加了内容协商、身份验证、状态管理等机制。HTTP协议广泛应用于WWW网站服务、移动APP服务、数据接口服务等。

3. DNS协议
DNS协议（Domain Name System，域名系统）用于将域名转换为IP地址，它位于TCP/IP四层模型的应用层。它提供域名解析服务，将域名转换为对应的IP地址。

## 3.3 操作系统
最后介绍操作系统的一些特性。

操作系统是一个管理计算机硬件和软件资源的程序，它是计算机的内核，负责管理计算机硬件设备，分配任务，协调进程，控制输入/输出设备，提供保护和安全环境，以及向用户提供各种服务和支持。

操作系统具有以下特征：

1. 并发性：操作系统能支持多道程序并行执行，即允许多个任务同时执行，有效提高计算机系统的工作效率。

2. 共享性：操作系统能实现多任务之间对资源的共同占用，使每个任务都能得到有效的使用资源。

3. 虚拟性：操作系统能创建虚拟对象，使得应用程序感觉像独享计算机一样。

4. 异步性：操作系统支持用户态和内核态的交互，允许任务在没有抢占CPU的情况下交替执行。

5. 保护性：操作System能提供访问控制和安全保护，保证计算机系统的安全运行。

6. 可靠性：操作系统能保证系统的正常运行，并且能尽力减少系统故障，使计算机保持稳定、健康、可用。
