# 1.简介
  

　　PyCharm是JetBrains公司推出的一款Python集成开发环境(IDE)。PyCharm本身非常强大，并且支持多种语言（包括Python、Java、C++等），可以实现自动完成、代码智能提示、语法高亮显示、运行调试等功能。但在实际应用中，PyCharm也存在以下一些问题：

　　1）费用问题：PyCharm是收费软件，需要付费购买或者试用。在国内访问其官网或下载软件可能由于网络原因导致失败，造成购买困难。

　　2）版本迭代快：在过去几年中，PyCharm官方团队发布了多个版本更新，新版本的功能更新比旧版本都要快。但这也给用户带来了更多的学习成本。

　　3）中文文档不全：相对于其它JetBrains产品来说，中文文档较少。这对外国用户来说，阅读英文文档会比较吃力。

　　4）使用门槛低：PyCharm作为JetBrains旗下产品，并非每个用户都有相关经验。学习曲线陡峭，初学者容易掉入坑中。

　　5）社区支持差：虽然PyCharm提供了许多学习资源，但在实际使用过程中，仍然无法解决所有问题。如果遇到问题需要寻求帮助，只能通过邮件列表或论坛进行反馈。

　　总之，如果希望像使用一般的工具一样，可以在任意地点随时随地就能开发Python应用，那么需要选用PyCharm这样的在线编程平台。但是由于上述几个弊端，因此市面上的其它Python集成开发环境也逐渐被抛弃。

# 2.PyCharm 安装及激活
　　PyCharm 的安装和激活非常简单，可以直接从官方网站下载安装包安装，也可以根据系统类型下载对应的安装包安装，并进行一步简单的激活即可。



　　如图所示，下载完毕后双击安装文件启动安装程序。点击“Next”按钮进入下一步安装向导。


　　在欢迎界面勾选并输入许可证信息，然后点击“Next”继续。


　　在“Choose a Directory for the Product”页面选择安装目录，然后点击“Next”。


　　在“Choose Start Menu Folder”页面设置开始菜单项，然后点击“Next”。


　　在“Create Desktop Shortcut”页面选择是否创建桌面快捷方式，然后点击“Next”。


　　在“Installing JetBrains PyCharm Community Edition”页面等待安装完成，然后点击“Finish”完成安装。


　　安装成功后，打开 PyCharm，第一次启动可能会要求输入 License Key，根据自己的情况输入即可。此处我已有 License Key，无需重复输入。


　　激活成功后，就可以愉快地使用 PyCharm 进行 Python 编程啦！
