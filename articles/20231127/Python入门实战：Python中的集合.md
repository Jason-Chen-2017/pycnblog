## 概念简介

- 集合（set）：无序且不重复的元素的集。它可以用于表示一个非重复元素的集合。可以用来进行集合运算、元素查找等。
- 字典（dict）：存储键值对的数据结构，键必须唯一且不可变，值可以取任何数据类型，它可以通过键访问对应的值。

```python
# 创建空集合
s = set()
print(type(s)) # <class'set'>
# 添加元素到集合中
s.add('a')
s.add(1)
s.add((2,))
s.add({'a': 1})
print(s) # {'a', (2,), 1}

# 通过集合创建字典
d = dict(one=1, two='two', three=[3])
print(type(d)) # <class 'dict'>
print(d['three']) # [3]
```

## 功能概述

| 方法                      | 描述                                                         |
| ------------------------ | ------------------------------------------------------------ |
| s.add(elem)               | 添加元素 elem 到集合 s 中                                    |
| s.remove(elem)            | 从集合 s 中删除元素 elem                                     |
| s.discard(elem)           | 删除集合 s 中的元素 elem，如果 elem 不在集合中则忽略错误     |
| s.pop()                   | 从集合 s 中随机选择一个元素并返回                           |
| len(s)                    | 返回集合 s 的长度                                            |
| x in s                    | 判断元素 x 是否存在于集合 s 中                               |
| for elem in s:...        | 用 for 循环遍历集合 s 中的所有元素                            |
| s1.union(s2[,s3...])      | 返回两个或多个集合的并集                                      |
| s1.intersection(*s2[,s3]) | 返回两个或多个集合的交集                                      |
| s1 - s2                   | 返回集合 s1 减去集合 s2                                       |
| s1.difference_update(s2)  | 将集合 s2 中的元素从集合 s1 中删除                            |
| s1.symmetric_difference(s2)| 返回两个集合的对称差                                         |
| s1^s2                     | 同上                                                         |


# 2.核心概念与联系

在 Python 中，列表、元组、字典都是可变对象。因此，它们具有动态扩容的特点。而集合则不同，集合是一个无序的不重复元素的集。这意味着集合中的元素不能被修改，只能添加或者移除元素。同时，集合也支持集合运算。结合这两者特性，可以形成集合的一些重要性质：

1. 不允许重复元素：集合中不允许出现相同元素，这就保证了元素的唯一性。
2. 无序性：集合中的元素无需按顺序排列，因此集合元素之间的位置关系没有规律可言。
3. 可变性：集合本身是可变对象，可以在运行时添加或删除元素。
4. 支持集合运算：集合支持数学运算、关系运算和逻辑运算等，可以实现复杂的集合操作。

## 相关名词

- 笛卡尔积：Cartesian product 或 cartesian power 是指将两个或更多集合（有限个）之间的所有可能组合形成的集合，用记号 $A \times B$ 表示。
- 子集：Subset 是指由某些元素所构成的集合，用符号 $A\subseteq B$ 表示。如 $A=\{1,2\}$，$B=\{1,2,3\}$，则 $A\subseteq B$。
- 真子集：真子集是指只包含原始集合 A 的元素的集合，即 $A\supseteq A$。
- 超集：Supperset 是指从某种角度看待集合的集合。它是包含另一个集合中的全部元素的集合。如 $C\supseteq A$ 和 $C\supseteq AB$。
- 幂集：Power set 是指包含自身和其任意单个元素的集合的集合。它用记号 $\wp (A)$ 表示。
- 划分集：Partition of a set 是指把一个集合划分成两个或更多子集的方式。如 $A=\{1,2,3,4,5,6,7,8,9\}$ 分割成三个集合 $\{\{1,2,3\},\{4,5,6\},\{7,8,9\}\}$。

## 集合运算符

集合运算符是指在集合上进行计算的运算符。以下是一些常用的运算符：

- ∪ : Union 操作符，返回两个集合的并集。$\{1,2\} \cup \{2,3\} = \{1,2,3\}$ 。
- ∩ : Intersection 操作符，返回两个集合的交集。$\{1,2\} \cap \{2,3\} = \{2\}$ 。
- - : Difference 操作符，返回第一个集合中不属于第二个集合的元素。$\{1,2\} - \{2,3\} = \{1\}$ 。
- ^ : Symmetric difference 操作符，返回两个集合的对称差。$(\{1,2\} \oplus \{2,3\}) = \{\{1,3\},\{2,3\}\}$ ，其中 ${1,3}$ 为集合 $\{1,2\}$ 和集合 $\{2,3\}$ 的对称差。

## 基本概念

- 集合：是包含零个或多个元素的整体，这些元素可以是数字、符号、文字、表达式，甚至是其他集合。
- 元素：是集合中具有特殊性质的对象。
- 空集：不含任何元素的集合。$\emptyset$ 。
- 属于关系：如果 $x$ 在集合 $S$ 中，那么说 $x$ 属于 $S$ 。记作 $x \in S$ 。
- 不属于关系：如果 $x$ 不在集合 $S$ 中，那么说 $x$ 不属于 $S$ 。记作 $x \notin S$ 。
- 等价关系：如果 $S$ 和 $T$ 有相同的元素，那么说 $S$ 和 $T$ 是等价的。记作 $S \equiv T$ 。
- 相等关系：如果 $S$ 和 $T$ 中所有的元素都相同，并且都属于同一个集合，那么说 $S$ 和 $T$ 是相等的。记作 $S = T$ 。
- 集合的大小：就是指集合中元素的个数。

## 数学概念

- 笛卡尔积：笛卡尔积是指将两个或更多集合（有限个）之间的所有可能组合形成的集合，它由笛卡尔积定义域的元素乘积组成。笛卡尔积通常用来描述函数的输出空间，而不是输入空间。
- 上下标：对于函数 $f(x)$ ，如果 $y = f(x)$ ，那么称 $y$ 是 $x$ 的函数值，并且有 $y_1,\cdots, y_n$ 形式地表示函数值。如果把函数值以某种元素 $b_i$ （可能包含 $+$ 或 $-$ 号）标记，那么称这个标记为函数值向量。例如，对于 $y = ax+b$ 函数，当 $x=2$ 时，函数值为 $4 + b$ ，相应的函数值向量为 $(4,b)$ 。

集合是一种非常有用的工具，可以快速高效地对一组数据进行处理。熟练掌握集合的各种操作和运用方法能够帮助我们更好地理解数据的结构和特征，提升编程能力，改善我们的工作效率。

## 创建集合

创建一个空集合可以使用 `set()` 语句，也可以通过提供一组元素作为参数来创建一个集合。

```python
# 创建空集合
mySet = set()

# 通过提供一组元素来创建一个集合
numbers = {1, 2, 3, 4, 5}
colors = {"red", "green", "blue"}
```

## 基本操作

集合支持几种基本操作：添加元素、删除元素、获取长度、判断成员、遍历集合。

### 添加元素

集合添加元素的语法如下：`collection.add(element)` 

```python
mySet = set([1, 2, 3])
mySet.add(4)   # 添加元素 4 
```

上面例子展示了一个添加元素到集合的方法。`.add()` 方法会直接对当前集合添加元素，如果元素已存在则不会报错，而 `.append()` 方法则需要先判断元素是否已经存在。

### 删除元素

集合删除元素的语法如下：`collection.remove(element)` 

```python
mySet = set([1, 2, 3])
mySet.remove(2)    # 删除元素 2
```

上面例子展示了一个删除元素从集合的方法。`.remove()` 方法会直接删除当前集合指定元素，如果不存在该元素则抛出 KeyError 异常。`.discard()` 方法不会产生 KeyErrpr 异常，它会自动跳过不存在的元素，所以它的效率要比 remove() 方法高。

### 获取长度

获取集合长度的语法如下：`len(collection)` 

```python
mySet = set([1, 2, 3])
length = len(mySet)   # 获取集合长度
```

上面例子展示了一个获取集合长度的方法。`len()` 方法会返回当前集合中元素的数量。

### 判断成员

判断某个元素是否属于集合的语法如下：`element in collection` 

```python
mySet = set([1, 2, 3])
if 1 in mySet:
    print("Yes")
else:
    print("No")
```

上面例子展示了一个判断某个元素是否属于集合的方法。`in` 操作符会判断左边元素是否在右边集合中。

### 遍历集合

遍历集合的语法如下：`for element in collection:` 

```python
mySet = set([1, 2, 3])
for num in mySet:
    print(num)
```

上面例子展示了一个遍历集合的方法。`for` 循环会逐个遍历集合中的元素。

## 集合运算

集合支持对集合进行常见的集合运算，包括并、交、差、对称差。

### 并集 union

求两个集合的并集的语法如下：`set1.union(set2)` 。

```python
set1 = {1, 2, 3}
set2 = {3, 4, 5}
result = set1.union(set2)   # {1, 2, 3, 4, 5}
```

上面例子展示了一个求两个集合的并集的方法。`.union()` 方法会返回两个集合的并集。

### 交集 intersection

求两个集合的交集的语法如下：`set1.intersection(set2)` 。

```python
set1 = {1, 2, 3}
set2 = {3, 4, 5}
result = set1.intersection(set2)   # {3}
```

上面例子展示了一个求两个集合的交集的方法。`.intersection()` 方法会返回两个集合的交集。

### 差集 difference

求两个集合的差集的语法如下：`set1 - set2` 。

```python
set1 = {1, 2, 3}
set2 = {3, 4, 5}
result = set1 - set2   # {1, 2}
```

上面例子展示了一个求两个集合的差集的方法。`-` 操作符会返回两个集合的差集。

### 对称差 symmetric_difference

求两个集合的对称差的语法如下：`set1.symmetric_difference(set2)` 。

```python
set1 = {1, 2, 3}
set2 = {3, 4, 5}
result = set1.symmetric_difference(set2)   # {1, 2, 4, 5}
```

上面例子展示了一个求两个集合的对称差的方法。`.symmetric_difference()` 方法会返回两个集合的对称差。

### 更新更新集合

将两个集合合并为一个集合的语法如下：`set1.update(set2)` 。

```python
set1 = {1, 2, 3}
set2 = {3, 4, 5}
set1.update(set2)
result = set1   # {1, 2, 3, 4, 5}
```

上面例子展示了一个将两个集合合并为一个集合的方法。`.update()` 方法会直接对当前集合进行更新，原集合不会改变。

### 删除元素

从集合中删除一个或多个元素的语法如下：`set.discard(element)` 

```python
mySet = {1, 2, 3, 4, 5}
mySet.discard(2)    # 删除元素 2
```

上面例子展示了一个从集合中删除元素的方法。`.discard()` 方法会直接从当前集合中删除指定元素，如果不存在该元素则会直接跳过。
