# 2.核心概念与联系
## Web开发的特点
### 静态页面网站
静态页面网站（又称静态网站），顾名思义，就是没有动态页面，只需要服务器生成一套固定内容的网页。这一类网站的特点是加载速度快，并且很容易被搜索引擎收录，但是缺乏交互性和用户体验。

静态网站的典型构成是HTML文档、CSS样式表、图片、音频、视频等。网站内容主要由浏览器访问，主要使用纯文本形式呈现，不含任何数据库查询、存储、处理功能，访问速度快，安全性高，但不利于用户交互、数据的实时更新。此外，静态网站并不能够提供动态交互功能，只能在页面上展示已有的内容。

### 动态页面网站
动态页面网站（又称动态网站），顾名思义，就是可以实现交互的网站。这一类网站的特点是加载速度慢，需要后台服务器来处理服务器端请求并返回相应的数据，访问速度缓慢。但是由于采用服务器端渲染技术，使得网站的响应速度相当快，用户体验好，而且有助于提高网站的流量。

动态网站的典型构成是服务器端脚本语言如PHP、ASP.NET、Node.js，数据库，以及前端渲染模板语言如Jinja、Twig、Nunjucks等。网站会根据用户的请求，生成新的页面内容，并且可以在浏览器上实时显示出来。动态网站可以提供丰富的用户交互功能，包括但不限于留言板、投票、购物车、会员系统等。网站的安全性也可以得到保障，用户的个人信息不会直接暴露给网站管理员，保证了数据隐私和安全。

### Web开发流程
Web开发流程描述的是从需求分析、设计、编码、测试、发布、维护等多个阶段所需的流程，以及每个环节的角色和职责。下图描述了Web开发的一般流程：
Web开发的一般流程可以分为以下几个阶段：

1.需求分析：分析客户的业务需求，确定目标用户群，制订网站的功能特性，制定开发计划和时间节点。

2.设计：设计网站的界面、版面布局、页面风格、交互逻辑、用户认证方式、数据库设计等。通常，设计人员会结合业务需求和客户提供的素材，经过多轮设计讨论，最终选择出最佳的设计方案。

3.编码：编写网站的代码实现功能逻辑、数据库连接、网页设计等。通常，开发人员会把编写的代码提交给测试人员进行测试，测试人员也会反馈代码的质量和可靠性，以便进行调整。

4.测试：测试人员会对网站进行自动化测试，检测网站的可用性、响应速度、兼容性、安全性等，确保网站正常运行，减少出现故障。

5.发布：最后，网站的测试结果已经确认无误，就可以正式上线运行了。网站的管理员还需要负责网站的日常维护、运营、安全升级等工作。

## 常用技术术语
为了让读者能更全面地了解Web开发，这里对一些常用的技术术语做简单的介绍。

- HTML(HyperText Markup Language):超文本标记语言，是一种用于创建网页的标准标记语言。它定义了如何表示网页的内容，比如文本、图像、表格、链接等，以及如何布局网页上的内容。

- CSS(Cascading Style Sheets):层叠样式表，是一种用于设置HTML样式的语言。它可以让网页更加美观、协调、完整。CSS允许网页作者定义各种各样的样式属性，如颜色、字体、大小、边框、阴影等，甚至可以控制网页的布局、效果。CSS也支持媒体查询，可以针对不同的屏幕尺寸和设备类型进行不同的样式设置。

- JavaScript:一种轻量级的、解释性的、跨平台的编程语言，其代码运行在浏览器上，用来给网页添加动态、交互性的内容。JavaScript是网页编程的基石，也是目前最流行的脚本语言。

- RESTful API:RESTful API，即“RESTful Web Services”，是在互联网上基于HTTP协议通信的分布式服务API。它的主要特点是：

1.客户端和服务器之间交换数据以获取资源的URL。
2.客户端通过HTTP方法，对资源的CRUD（Create-Read-Update-Delete）操作进行请求。
3.服务器对资源执行对应的操作，并向客户端返回响应。 

RESTful API与其他类型的API的区别主要在于：它是专门为Web而设计的，而不是泛指某种类型的API。它遵循HTTP协议约定的格式、状态码、URI和方法等规则，使得其具有简单易用的特点。

- AJAX(Asynchronous JavaScript And XML):异步JavaScript和XML，是一种在无需刷新整个页面的情况下，更新部分网页的技术。它使用AJAX技术可以实现局部页面的更新，而无需重载整个页面，提升用户体验。

- JSON(JavaScript Object Notation):JavaScript对象符号化，是一种轻量级的数据交换格式。它可以直接在Javascript脚本语言里使用，可以处理复杂的数据结构，非常适合于前后端的数据传输。

- MySQL:MySQL是一个开源的关系型数据库管理系统，属于DBMS阵营。它使用C、C++编写，支持多种平台，主要用于WEB应用开发。

- PostgreSQL:PostgreSQL是一个开源的关系型数据库管理系统，由瑞典IT创业公司西班牙PostgreSQL Global Development Group开发，其可提供高性能、高可用性、高扩展性。PostgreSQL使用SQL语言，提供关系型数据库服务。

- MongoDB:MongoDB是一个开源的NoSQL数据库管理系统。它支持灵活的数据结构，不需要固定的模式，非常适合动态的web应用程序。

- Nginx:Nginx是一个免费、开源的Web服务器，由俄罗斯的<NAME>夫创建，采用BSD许可协议，稳定性高，处理并发请求能力强，并发性能好。Nginx使用异步非阻塞IO模型，提升并发效率。

- Docker:Docker是一个容器技术框架，它提供虚拟化环境，简化开发环境配置，降低开发难度。它可以封装应用及其依赖环境，打包镜像，分发到不同机器，实现开发环境一致性。

- Git:Git是一个开源的分布式版本控制系统，可以有效避免不同人修改同一文件造成冲突，记录每次对文件的修改，可以追踪历史版本，方便团队协作。

- GitHub:GitHub是一个面向开源及私有软件项目的托管平台。用户可以利用GitHub来快速搭建自己的代码库，管理版本更新，并与他人分享项目。
