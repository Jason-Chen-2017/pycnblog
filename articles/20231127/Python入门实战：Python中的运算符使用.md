                 

# 1.背景介绍



作为一个从事Python开发的技术人员，我们经常需要用到一些高级语言中常用的运算符。但对于初学者来说，这些运算符可能比较晦涩难懂。因此，本文将通过实例、原理和代码展示各类运算符的功能以及如何使用。让读者能够更加容易地理解并应用到实际的编程工作中。

# 2.核心概念与联系

Python中的运算符主要分为以下几种类型：

1.算术运算符：用于对数字进行基本的四则运算
2.赋值运算符：用于给变量赋值或修改其值
3.比较运算符：用于比较两个值之间的大小关系
4.逻辑运算符：用于处理布尔值（True、False）
5.成员运算符：用于测试值是否属于某集合
6.身份运算符：用于比较两个对象的存储位置
7.位运算符：用于按位计算值的相关信息

下面我们逐一讨论每一种运算符的使用方法及示例。

# 3.算术运算符

算术运算符包括加号(+)，减号(-)，乘号(*)，除号(/)，取模(%)等。

## 3.1 + 运算符

```python
>>> 3+2
5
```

"+" 表示两个数相加，即使其中有一个不是数字也能执行运算，结果还是数字。

```python
>>> 'hello'+'world'
'helloworld'
```

同样，"+"可以对字符串进行拼接。

## 3.2 - 运算符

```python
>>> 9-3
6
```

"-"表示两个数相减，得到的是差值。

```python
>>> 9-2-1
7
```

多个连续的 "-" 操作也可以执行减法。

```python
>>> a=20
>>> b=-a #b=20*-1=-20
>>> c=a+b #c=20+(-20)=0
```

还可以使用 "+" 和 "-" 来实现加减乘除的运算。

```python
>>> (2+4)*3/2
7.5
```

```python
>>> 2**3 # 2^3
8
```

"**" 是指幂运算符，表示整数的乘方。

## 3.3 * 运算符

```python
>>> 2*3
6
```

"*" 表示两个数相乘，得到的是积。

```python
>>> 'hello'*3
'hellohellohello'
```

可以把字符串重复多次。

## 3.4 / 运算符

```python
>>> 8/2
4.0
```

"/" 表示除法运算，结果是浮点型数据。如果两个数都是整数的话，则结果也是整数。

```python
>>> 5//2
2
```

"//" 表示整除运算，就是不考虑小数点后的部分，只保留整数部分。

```python
>>> 8%3
2
```

"%" 表示求模运算，它返回余数，也就是除法运算后所得的余数。

## 3.5 ** 运算符

```python
>>> pow(2,3) #2的3次方
8
```

"pow()" 函数用于计算任意精度的整数次方。

```python
>>> pow(2,-3) # 2的负3次方
0.125
```

"pow()" 函数可以计算任意精度的非整数次方。

# 4.赋值运算符

赋值运算符包括"="、"+="、"-="、"*="、"/="、"%="等。

```python
>>> x = 10
>>> y = 5
>>> z = x + y
>>> print(z)
15
>>> x += y #等价于x = x + y
>>> print(x)
15
>>> x *= y #等价于x = x * y
>>> print(x)
75
>>> x /= y #等价于x = x / y
>>> print(x)
37.5
>>> x %= y #等价于x = x % y
>>> print(x)
3.75
```

"+="表示先加上右边的值，再赋给左边的变量；"*="表示先乘以右边的值，再赋给左边的变量；"/="表示先除以右边的值，再赋给左边的变量；"%="表示先求余数，再赋给左边的变量。

# 5.比较运算符

比较运算符包括等于(==)、不等于(!=)、大于(>)、小于(<)、大于等于(>=)、小于等于(<=)等。

```python
>>> x = 10
>>> y = 5
>>> if x == y:
        print('equal')
    elif x > y:
        print('greater than')
    else:
        print('less than')
greater than
```

">", "<", ">=", "<=" 可以用来判断大小关系。

# 6.逻辑运算符

逻辑运算符包括"and"、"or"、"not"三种。

## 6.1 and 运算符

```python
>>> True and False
False
```

"and" 运算符表示同时满足两个条件。

```python
>>> x = 5
>>> y = 10
>>> not x < y or y == x
True
```

"not" 运算符表示否定，即取反。

## 6.2 or 运算符

```python
>>> True or False
True
```

"or" 运算符表示任何一个条件成立就返回 True 。

```python
>>> int(True)
1
>>> int(False)
0
```

"int()" 函数可以把布尔值转换为整数。

# 7.成员运算符

成员运算符包括"in"、"not in"两种。

## 7.1 in 运算符

```python
>>> [1,2,3] in [4,5,6,[1,2,3]]
True
```

"in" 运算符表示在指定的序列中查找元素是否存在。

```python
>>> ['apple', 'banana'] in ['grape', 'pear', ('apple', 'banana')]
True
```

"in" 运算符也可以用来测试字典中是否存在某个键值。

## 7.2 not in 运算符

```python
>>> 'orange' not in ['grape', 'pear', 'watermelon']
True
```

"not in" 运算符表示不在指定的序列中查找元素是否存在。

# 8.身份运算符

身份运算符包括"is"、"is not"两种。

## 8.1 is 运算符

```python
>>> a=[1,2,3]
>>> b=a
>>> a is b
True
```

"is" 运算符表示判断两个对象是否指向同一个内存地址。

```python
>>> d=dict()
>>> e={}
>>> d is e #如果字典为空{}，e=d也会返回 True
False
```

"is" 运算符对空字典的处理不同于其它类型。

## 8.2 is not 运算符

```python
>>> a=[1,2,3]
>>> b=[]
>>> a is not b
True
```

"is not" 运算符表示判断两个对象是否指向不同的内存地址。

# 9.位运算符

位运算符包括按位与(&)、按位或(|)、按位异或(^)、按位取反(~)、左移(<<)、右移(>>)等。

```python
>>> bin(5 & 3) # &表示按位与
'0b1'
>>> bin(5 | 3) # |表示按位或
'0b7'
>>> bin(5 ^ 3) # ^表示按位异或
'0b6'
>>> bin(~5)   # ~表示按位取反
'-0b6'
>>> bin(5 << 1)    # <<表示左移
'0b1010'
>>> bin(5 >> 1)    # >>表示右移
'0b1'
```

"bin()" 函数用于显示二进制形式。

# 10.运算符优先级

运算符优先级如下表：

| 运算符         | 描述              | 优先级        |
|---------------|--------------------|------------------|
| **             | 指数               | 次方            |
| *, /, //, %   | 乘、除、整除、取模 | 乘法、除法、整除、取模 |
| +, -          | 加、减             | 加法、减法       |
| >, >=, <, <= | 比较运算符         | 关系运算符      |
| ==,!=        | 等于、不等于        | 等于运算符      |
| is, is not    | 身份运算符         | 身份运算符      |
| in, not in    | 成员运算符         | 成员运算符      |
| not           | 逻辑否定           | 逻辑运算符      |
| and           | 逻辑与             | 逻辑运算符      |
| or            | 逻辑或             | 逻辑运算符      |
