                 

# 1.背景介绍


## 一、什么是元组？
元组（Tuple）是一个不可变的有序序列，元素不能修改删除或增加。元组是一种内置类型，创建时需要在括号中指定元素，并用逗号隔开。其语法形式如下：

```python
tuple = (element1, element2,...) 
```

例如：

```python
>>> tuple1 = ('apple', 'banana', 'orange')
>>> tuple2 = () # 空元组
>>> tuple3 = (1,)   # 只有一个元素的元组
>>> print(type(tuple1))    # <class 'tuple'>
>>> print(type(tuple2))    # <class 'tuple'>
>>> print(type(tuple3))    # <class 'tuple'>
```

可以看到，元组是由若干个元素组成的有序集合，这些元素之间存在固定顺序。而列表则不是固定的顺序的集合。

## 二、为什么要使用元组？
### （1）便于数据的传递
元组作为不可变数据类型，能帮助开发者将数据安全地传递给函数或者方法。因为元组的所有元素都定义了名称，而且不允许被修改或增加新的元素，因此可以使用元组实现信息的封装和隐藏，使得函数调用更加方便灵活。
### （2）性能高效率
元组在存储和处理数据的过程中，性能要比列表要高效。因为元组占用的内存空间小，因此可以节省内存空间，提升程序运行效率。此外，元组的数据类型有层次结构，易于管理和维护。

## 三、如何创建元组？
### （1）使用圆括号()
创建一个元组只需把元素放在括号里，并用逗号分割即可，例如：

```python
tup = (1, 2, 3)
```

### （2）使用tuple()函数
如果元组中的元素个数不确定，就可以使用tuple()函数生成元组。该函数接受任意数量的参数并将它们打包成元组返回，例如：

```python
a = 1
b = 2
c = 3
tup = tuple([a, b, c])
print(tup)
```

输出结果：

```python
(1, 2, 3)
```

### （3）使用索引值访问元素
元组元素可以通过索引来访问，索引从0开始。例如：

```python
t = (1, 2, 3)
print(t[0])     # 输出1
print(t[1:])    # 输出(2, 3)
```

### （4）使用*运算符拆分元组
元组也可以使用星号(*)运算符进行解包操作，把整个元组分解成为多个变量，例如：

```python
t = (1, 2, 3)
x, y, z = t
print(x)      # 输出1
print(y)      # 输出2
print(z)      # 输出3
```

## 四、元组操作
### （1）连接两个元组
使用加号+对两个元组进行连接。示例代码如下：

```python
t1 = (1, 2, 3)
t2 = ('x', 'y', 'z')
t3 = t1 + t2        # 元组连接
print(t3)           # 输出(1, 2, 3, 'x', 'y', 'z')
```

注意，当元组中某个元素已经存在时，最后出现的那个元素会覆盖之前的元素。

### （2）重复元组
使用乘号*可以对元组进行复制，即每个元素重复n次。示例代码如下：

```python
t = (1, 2, 3) * 3       # 对元组重复3次
print(t)                # 输出(1, 2, 3, 1, 2, 3, 1, 2, 3)
```

### （3）比较两个元组是否相等
使用等于(==)判断两个元组是否相等。示例代码如下：

```python
t1 = (1, 2, 3)
t2 = [1, 2, 3]          # t2虽然内容相同但数据类型不同，所以不能比较相等
if t1 == t2:
    print('True')      # 不执行，因为t1和t2类型不同
else:
    print('False')     # 执行，输出False
```

但是，比较两个元组是否相等的时候，首先判断长度是否一致，然后再逐一元素进行比较。由于元组是不可变对象，因此通过比较地址是无法区分的，只能通过元素的值进行比较。

```python
t1 = (1, 2, 3)
t2 = (1, 2, 3)
if id(t1) == id(t2):         # 通过id()函数获取对象的地址
    print("t1 is the same object as t2")
else:
    print("t1 is different from t2")
    
t3 = (1, 2, 4)              # t3的第三个元素的值不同，所以不相等
if t1 == t3:                 
    print('True')
else:
    print('False')           
```

输出结果：

```python
t1 is the same object as t2
False
```
