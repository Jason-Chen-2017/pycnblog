# 数据结构
## 列表 List
列表是Python中的一种数据类型，可以保存多个元素。创建列表的方式有两种：
第一种，用中括号 [] 创建空列表；第二种，用中括号 [] 将元素逐个添加到列表中。如：
```python
list1 = [1, 'hello', True] # 第一种创建空列表
list2 = [1, 'hello'] + ['world'] # 第二种创建非空列表
```
### 获取列表元素
可以通过索引获取列表中的元素，索引从0开始，依次递增。如`list[0]`表示第一个元素，`list[-1]`表示最后一个元素。也可以通过切片获取子列表，语法为 `list[start:end:step]`。如：
```python
list = [1, 2, 3, 4, 5]
print(list[0])   # 输出结果：1
print(list[:3])  # 输出结果：[1, 2, 3]
print(list[::2]) # 输出结果：[1, 3, 5]
```

### 修改列表元素
可以使用索引直接修改列表中的元素，或者使用切片对子列表赋值。如：
```python
list = [1, 2, 3, 4, 5]
list[0] = 0          # 通过索引修改第一个元素的值为0
list[1:] = [9, 8, 7] # 使用切片修改从第二个元素开始后的所有元素的值
print(list)          # 输出结果：[0, 9, 8, 7, 5]
```

### 删除列表元素
可以直接删除列表中的元素，也可使用`del`语句删除子列表。如：
```python
list = [1, 2, 3, 4, 5]
del list[1]    # 删除第二个元素
list[:] = []   # 清空列表的所有元素
```

### 列表的长度、最大值、最小值、排序等
可以用内置函数len()获取列表的长度，max()和min()获取列表的最大值和最小值。还可以通过sorted()对列表进行排序。如：
```python
list = [4, 5, 1, 2, 3]
length = len(list)       # 获取列表的长度
maximum = max(list)      # 获取列表的最大值
minimum = min(list)      # 获取列表的最小值
sorted_list = sorted(list)# 对列表进行排序
```

## 元组 Tuple
元组与列表类似，不同的是元组一旦定义就不能修改。创建元组的方式和列表一样，也可以用圆括号 () 来指定元素。如：
```python
tuple1 = (1, 'hello')     # 创建元组
tuple2 = tuple(['a', 'b'])# 用列表创建元组
```

### 元组的索引与切片
与列表类似，可以用索引获取元组中的元素，也可以用切片获取子元组。如：
```python
tuple = ('apple', 'banana', 'orange', 'grape')
print(tuple[0])        # 输出结果：'apple'
print(tuple[::-1])      # 输出结果：('orange', 'grape', 'banana', 'apple')
```

### 不可变性
由于元组一旦定义就不能修改，所以对元组进行的任何修改操作都会导致生成新的元组，而不是影响原始的元组。例如：
```python
tuple1 = (1, 2, 3)
tuple2 = tuple1 + (4,)     # 在元组后面加上元素4
print(tuple1)             # 输出结果：(1, 2, 3)
print(tuple2)             # 输出结果：(1, 2, 3, 4)
```

## 集合 Set
集合（set）是由无序不重复元素组成的，可以用花括号 {} 表示。创建集合的方式如下：
```python
set1 = {1, 2, 3}         # 通过集合推导式创建集合
set2 = set([1, 2, 3, 1]) # 从列表创建集合，会自动去重
set3 = set()              # 创建空集合
```

### 添加、删除、判断元素是否存在于集合
可以用add()方法添加元素到集合中，可以用remove()方法删除元素。另外，可以用in关键字判断某个元素是否存在于集合中。如：
```python
fruits = {'apple', 'banana', 'orange'}
fruits.add('grape')            # 添加元素到集合
fruits.remove('orange')        # 删除元素
if 'banana' in fruits:         # 判断元素是否存在
  print('banana is a fruit.') 
else:
  print("I don't know what's bananas.")
```

### 集合运算
可以用`&`（交集）、`|`（并集）、`^`（异或）、`<=`（子集）四则运算符来计算两个集合之间的关系。
```python
A = {1, 2, 3}
B = {2, 3, 4}
C = A & B           # 输出结果:{2, 3}
D = A | B           # 输出结果:{1, 2, 3, 4}
E = A ^ B           # 输出结果:{1, 4}
F = A <= B          # 输出结果：True
```
