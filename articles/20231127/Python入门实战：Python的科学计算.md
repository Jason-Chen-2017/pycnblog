# 2.核心概念与联系
## 数据结构——数组（Array）
在计算机科学中，数组是一个存储元素集合的顺序集合。数组的每个元素都有一个唯一标识符，元素之间可以直接通过下标访问。对于同类型的元素，可以使用相同的下标进行引用，这就是所谓的关联数组（associative array）。

在Python中，列表（List）和元组（Tuple）都是数组的两种常用数据结构。列表和元组都可以存储一系列元素，但是列表是可变的，可以改变它的长度；元组则是不可变的，一旦创建就不能修改。

例如，给定一个包含整数的列表：[1, 2, 3, 4]，如果要获取第四个数，我们可以使用下标索引方式：`num = my_list[3]`。

## 数据结构——字典（Dictionary）
字典是另一种非常重要的数据结构。它类似于关联数组，即将键-值对组织起来。在字典中，键是唯一的，值可以重复。字典的特点是通过键来查找值，而不是通过位置来确定元素。

例如，假设我们有这样的一个字典：
```python
my_dict = {'apple': 3, 'banana': 2, 'orange': 4}
```
其中，键`'apple'`对应的值为`3`，键`'banana'`对应的值为`2`。如果要获取`'banana'`对应的数量，可以使用`value = my_dict['banana']`，或者简写为`value = my_dict.get('banana')`。

## 函数——定义与调用
函数是用来完成特定任务的一段程序。在Python中，函数的定义一般采用如下形式：

```python
def function_name(parameter):
    # do something here
    return result
```

其中，`function_name`表示函数名，`parameter`表示参数（函数执行所需的数据），`:`表示函数体开始。函数体内的代码块应该缩进，并采用`return`语句返回结果。

函数调用时需要指定函数名和参数。例如，定义了一个计算平方的函数`square`，就可以调用该函数来计算出一个数字的平方值：`result = square(9)`。
