                 

# 1.背景介绍


现代工厂、制造业是一个蓬勃发展的行业，产业链上下游关系日益紧密，越来越多的企业希望借助互联网远程办公、边缘计算、智慧工地等新技术降本增效，提升生产效率。在这个过程中，如何让更多的人参与到智慧工厂管理中来，提升工作效率和质量成为亟待解决的问题。工业4.0时代，数字孪生技术、机器人技术、人工智能技术、无人机等人类智能化工具正在催生新的工业革命。人工智能（AI）可以帮助工厂更高效、更准确地完成各种任务，帮助企业改善生产环节、提高生产力水平。但是，如果这些AI技术仅仅用于特定应用场景，对于工厂内部管理来说就无法应对日益复杂的业务流程和各种操作流程。因此，需要一个能够实现全面的流程自动化的平台。基于这个目的，降低AI与流程自动化平台之间的界限，我们提出了使用RPA(Robotic Process Automation)的方式将AI与流程自动化相结合，通过构建业务流程模型和配置，使AI能够进行自主学习，自动识别出其执行的任务，并完成相应操作。
为了进一步深入理解RPA与GPT大模型AI Agent在物联网与智能制造中的应用，我们搭建了一个基于SOEN 391大型项目的案例研究，引入GPT-3 (Generative Pre-trained Transformer 3)AI agent，通过实现业务流程自动化从而改善生产工艺。在案例中，我们主要关注于两条业务线：工业光伏电站供电管理和智能制造制造流程优化。

工业光伏电站供电管理:
需求背景：随着电力工业的发展，更多的地方都在使用变压器、直流电机、重力场能、风能、光伏等设备。因此，供电网络的可靠性及其运营维护成为了许多供电厂的首要考虑事项。一般情况下，由于有些原因导致的供电故障较多，致使整个供电系统出现故障甚至瘫痪。因此，为了保证电力供应的稳定运行，减少或避免因电源故障带来的损失，工业光伏电站的维护人员每天都要处理大量繁杂的工作，因此很难集中精力处理所有问题。如何通过自动化手段，提高电站的整体运行效率，缩短维护周期，加快配套设备的维修速度，这是提升电站服务能力的重要方向之一。在企业级应用开发中，我们可以通过RPA的方式为工业光伏电站管理部门提供便利。

解决方案：基于当前工业光伏电站的实际情况，我们选择了用RPA的方式为其建立了一套完整的供电管理流程，包括设备维修流程、停电预警、配电统计、电力系统诊断、单机故障诊断等多个环节。通过RPA系统，企业可以轻松、自动、快速地完成日常运维工作。

1.设备维修流程：如今的工业光伏电站通常会有大量的设备，而且随着技术的更新迭代，设备可能存在各式各样的问题。因此，采用自动化系统可以提高维修效率，同时减少操作时间。在设备维修流程中，通常需要操作人员根据维修记录找到对应设备，手动检查设备状况，然后根据维修记录上的指示对设备进行维护。这样做的费用较高，且维护周期长，操作不灵活。而通过RPA系统，企业可以在维修期间自动检测设备故障，对设备执行维修措施，并及时上报维修进度。此外，通过RPA系统还可以实现“满意度”反馈机制，通过问卷调查收集用户对维修过程的满意程度，有效改善维修流程。

2.停电预警：工业光伏电站供电需要24小时持续正常运行，但在极端情况下也可能发生停电事件。因此，需要预先对停电事件进行分析、跟踪、预测、及时处理。在停电预警流程中，首先要检查电网是否已关闭，并确定是哪台设备发生故障。接下来，通过机器学习算法预测将要发生的停电时刻，并提前给相关人员发送警告通知。此外，在停电预警过程中，也可以通过数据分析发现异常波动，即电力消费的增加和减少，帮助确定供电系统是否存在不当行为。

3.配电统计：每天都有很多的电流、电压数据流向电网，但由于采集技术、传输方式等限制，只能获取到部分数据。通过自动化系统收集、汇总和分析数据，能够有助于了解供电系统的运行状况。例如，可以预测到当天的最大电流峰值，预估供电总容量，估计每台设备的负荷等信息。通过这种方式，可以提高配电网络的稳定性、可靠性及其效率。

4.电力系统诊断：一般来说，光伏电站的电网设计存在一些技术缺陷、安全隐患、电力系统连接不畅等问题，严重影响供电系统的正常运行。因此，需要针对电网技术缺陷、安全隐患、系统连接不畅等现象，及时发现并进行诊断。在电力系统诊断过程中，要采取数据采集、归纳、分析和报告等方式，对光伏电站电力系统进行诊断。通过数据分析，可以发现电力系统运行的瓶颈所在，找出根本原因，并制定相应的维修策略。

5.单机故障诊断：由于互联网的普及、移动化和各种设备的普及，光伏电站经常面临着大量的单机故障。为了及时发现和定位故障，提升效率，降低成本，需要建立具有一定的分析能力的单机故障诊断系统。系统通过识别故障原因和漏洞，利用机器学习算法预测故障出现的时刻，并将预测结果上传至服务器。通过时序数据分析，就可以发现系统故障在什么时候、为什么发生、何处位置，从而可以实时跟踪故障并进行定位。

智能制造制造流程优化:
需求背景：随着人工智能（AI）技术的发展，智能制造已经逐渐成为制造领域的一个热门话题。据估计，在未来十年内，制造业中的大部分产业都会转向由人工智能代替人类操作。通过智能制造，可以降低生产成本、提高生产效率，并提升市场竞争力。智能制造可以分为两个方面：一是智能制造的核心技术，包括机器人技术、3D打印技术、虚拟现实技术、人工智能（AI）技术；二是智能制造的产品形态，包括互联网技术、人机交互技术等。

解决方案：在当前智能制造发展阶段，要想推进工厂制造过程的改善和智能化，首先需要考虑制造过程的自动化改革。基于工业4.0时代背景下的智能制造发展趋势，我们的目标是通过自动化工具提升制造过程的效率、准确性，最终实现零失误、零售假冒等问题的解决。

基于当前智能制造的实际情况，我们选择了用RPA的方式为其建立了一套完整的制造流程，包括生产准备流程、生产核准流程、生产排产流程、质量监控流程等多个环节。通过RPA系统，企业可以轻松、自动、快速地完成工厂生产工作。

1.生产准备流程：制造业是一个高度复杂的过程，涉及众多工序和子工件，需要非常强的生产技巧和良好的设备条件才能正常进行。因此，在生产准备环节，企业通常需要进行大量的数据输入、文档记录和图形化展示，耗时长、费用高。而通过RPA系统，企业可以用计算机代替人工进行制造材料的筛选、分装、分配、封装、分类等工作，大大节省生产资源。另外，在生产准备环节，企业还需要处理各种资料的扫描、检验、存档等繁琐工作。通过RPA系统，可以快速、精确地完成该环节的自动化。

2.生产核准流程：由于生产资料的保密性、复杂性和不一致性，企业需要通过各种手段进行生产资料的鉴别、验证、授权等流程，才能保证资料的正确性、完整性、合法性。而通过RPA系统，企业可以把这一流程自动化，加快其审批速度，缩短流程耗时。同时，通过RPA系统还可以实现“满意度”反馈机制，通过问卷调查收集用户对生产过程的满意程度，及时提升产品质量。

3.生产排产流程：制造企业需要按时、按顺序、按需生产，否则就会出现延迟、不足、超量等问题。如何实现自动化生产排产，降低人力投入，提升生产效率，是目前企业面临的重点难点问题。在生产排产流程中，企业需要制订标准时间表，并根据需要调整生产计划。而通过RPA系统，企业可以把这一流程自动化，按时、按顺序、按需生产。在这里，RPA可以协助生产工人按时上下班，提高生产效率，节约生产成本。

4.质量监控流程：为了保证生产过程中货物品质的质量、完整、有效，企业需要进行完备的质量监控，及时发现质量问题并采取补救措施。而通过RPA系统，企业可以把这一流程自动化，及时发现生产过程中的质量问题，并予以提醒、提示和解决。通过数据分析，还可以发现系统故障、生产效率等问题。通过RPA系统还可以实现“满意度”反馈机制，通过问卷调查收集用户对质量控制的满意程度，及时评估产品质量。

未来发展趋势：RPA与GPT大模型AI Agent在物联网与智能制造中的应用仍然处于起步阶段，我们还有很多工作要做。一方面，我们可以继续探索更多的场景和应用，开拓RPA与GPT大模型AI Agent的应用空间；另一方面，我们也要寻找未来AI与RPA融合的创新突破口，提升AI技术的自动化效能，助力更多的企业在自动化工程、智能制造、物联网等领域取得成功。
