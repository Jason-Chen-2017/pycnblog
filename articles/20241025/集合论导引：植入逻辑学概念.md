#### 第1章: 集合论的基本概念

## 1.1 集合的定义与表示

### 1.1.1 集合的定义

集合是指由确定的、互不相同的对象所组成的一个整体。这些对象称为集合的元素。形式化地，集合可以表示为包含其所有元素的大括号中的列表。

### 1.1.2 集合的表示方法

集合可以通过列举法、描述法和抽象法来表示。

- **列举法**：直接列出集合的所有元素，如 `{1, 2, 3}`。
- **描述法**：使用条件来描述集合的元素，如 `{x | x 是正整数}`。
- **抽象法**：使用数学符号或抽象表示法，如 `N` 表示自然数集合。

### 1.1.3 集合的元素特性

集合的元素具有确定性、互异性和无序性。

- **确定性**：集合中的元素是确定的，每个元素是否属于集合是可以判断的。
- **互异性**：集合中的元素各不相同。
- **无序性**：集合中的元素没有特定的顺序。

#### 第2章: 集合的性质与定理

## 2.1 集合的相等性

### 2.1.1 集合相等的定义

两个集合相等，当且仅当它们包含完全相同的元素，记作 `A = B`。

### 2.1.2 集合相等的性质

集合的相等性具有自反性、对称性和传递性。

- **自反性**：任何集合与自身相等。
- **对称性**：如果 `A = B`，则 `B = A`。
- **传递性**：如果 `A = B` 且 `B = C`，则 `A = C`。

### 2.1.3 集合相等与集合包含的关系

集合的相等是集合包含的特例。两个集合相等，当且仅当它们相互包含。

## 2.2 集合的势与基数

### 2.2.1 集合的势的定义

集合的势是指集合中元素的数量。无限集合的势可以是无限的，有限集合的势是有限的。

### 2.2.2 集合的基数

集合的基数是指集合的势。有限集合的基数是一个自然数，无限集合的基数可以是无穷大。

### 2.2.3 可数集与不可数集

可数集是指其基数等于自然数集合的集合。不可数集是指其基数大于自然数集合的集合。

## 10.1 集合的基本运算公式

- 并集：`A ∪ B = {x | x ∈ A 或 x ∈ B}`
- 交集：`A ∩ B = {x | x ∈ A 且 x ∈ B}`
- 差集：`A - B = {x | x ∈ A 且 x ∉ B}`
- 对称差：`A Δ B = (A ∪ B) - (A ∩ B)`

## 10.2 集合的性质与定理

- 子集：若 A ⊆ B，则 B - A 是 A 的真子集。
- 补集：集合 A 的补集为 U - A，其中 U 是全集。
- 分配律：`A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)`，`A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)`。
- 结合律：`A ∪ B ∪ C = A ∪ (B ∪ C)`，`A ∩ B ∩ C = A ∩ (B ∩ C)`。
- 交换律：`A ∪ B = B ∪ A`，`A ∩ B = B ∩ A`。
