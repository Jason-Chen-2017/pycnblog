## 4.1 条形图

条形图是一种常用的数据可视化图形，用于显示分类变量之间的比较。以下是一个使用Python的Matplotlib库创建条形图的代码实例：

```python
import matplotlib.pyplot as plt

# 创建数据
data = [5, 10, 15, 20, 25]

# 创建条形图
plt.bar(range(len(data)), data)

# 设置图表标题和坐标轴标签
plt.title('Bar Chart Example')
plt.xlabel('Categories')
plt.ylabel('Values')

# 显示图表
plt.show()
```

在这个代码实例中，我们首先导入了Matplotlib库。然后，我们创建了一组数据。接着，我们使用`plt.bar()`函数创建了一个条形图，将数据和分类变量作为参数传递给函数。最后，我们设置了图表标题和坐标轴标签，并使用`plt.show()`函数显示了图表。

## 4.2 折线图

折线图是一种常用的数据可视化图形，用于显示连续变量的变化趋势。以下是一个使用Python的Matplotlib库创建折线图的代码实例：

```python
import matplotlib.pyplot as plt

# 创建数据
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

# 创建折线图
plt.plot(x, y)

# 设置图表标题和坐标轴标签
plt.title('Line Chart Example')
plt.xlabel('X-Axis')
plt.ylabel('Y-Axis')

# 显示图表
plt.show()
```

在这个代码实例中，我们首先导入了Matplotlib库。然后，我们创建了一组数据，包括X轴和Y轴的值。接着，我们使用`plt.plot()`函数创建了一个折线图，将X轴和Y轴值作为参数传递给函数。最后，我们设置了图表标题和坐标轴标签，并使用`plt.show()`函数显示了图表。
