随机分配算法的代码实例：

```python
import random

def random_allocation(resources, participants):
    allocations = []
    for participant in participants:
        allocation = random.choice(resources)
        allocations.append(allocation)
    return allocations

resources = [1, 2, 3, 4, 5]
participants = ['A', 'B', 'C', 'D', 'E']
allocations = random_allocation(resources, participants)
print(allocations)
```

权重分配算法的代码实例：

```python
def weighted_allocation(resources, participants, weights):
    allocations = []
    for participant, weight in zip(participants, weights):
        allocation = resources * weight
        allocations.append(allocation)
    return allocations

resources = [1, 2, 3, 4, 5]
participants = ['A', 'B', 'C', 'D', 'E']
weights = [0.2, 0.3, 0.2, 0.2, 0.1]
allocations = weighted_allocation(resources, participants, weights)
print(allocations)
```
