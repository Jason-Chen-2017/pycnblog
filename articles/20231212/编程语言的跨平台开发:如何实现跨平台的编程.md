## 4.具体代码实例和详细解释说明

### 4.1 代码实例

以下是一个使用Python进行跨平台开发的简单示例：

```python
import os
import sys

def main():
    if sys.platform == "win32":
        print("当前平台为Windows")
    elif sys.platform == "darwin":
        print("当前平台为macOS")
    elif sys.platform == "linux":
        print("当前平台为Linux")
    else:
        print("当前平台未知")

if __name__ == "__main__":
    main()
```

### 4.2 详细解释说明

上述代码实例主要包括以下几个方面：

- 导入os和sys模块，用于获取系统信息。
- 定义一个main函数，用于判断当前平台。
- 使用sys.platform属性获取当前平台的信息。
- 根据不同平台的信息，打印出当前平台的信息。
