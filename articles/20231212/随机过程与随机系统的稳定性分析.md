```python
import numpy as np

def random_walk(n, p):
    x = 0
    for _ in range(n):
        if np.random.rand() < p:
            x += 1
        else:
            x -= 1
    return x
```

上述代码实例是一个随机漫步的生成函数，它接受两个参数：n（步数）和 p（概率）。函数返回一个随机漫步的取值。
