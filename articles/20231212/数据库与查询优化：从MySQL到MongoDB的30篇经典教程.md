### 4.1 MySQL查询优化

MySQL是一种关系型数据库管理系统，它使用结构化查询语言（SQL）进行查询和操作。在MySQL中，我们可以使用一些查询优化技术，如索引、查询缓存和查询优化器等。

以下是一个MySQL查询优化的具体代码实例：

```sql
CREATE INDEX idx_name ON table (name);
SELECT name FROM table WHERE name = 'John';
```

在这个例子中，我们首先创建了一个名为`idx_name`的索引，它用于优化`name`列的查询。然后，我们使用`SELECT`语句进行查询，并使用`WHERE`子句筛选出名字为`John`的记录。

### 4.2 MongoDB查询优化

MongoDB是一种NoSQL数据库，它使用BSON格式存储数据，并提供了一个易于使用的查询语言。在MongoDB中，我们可以使用一些查询优化技术，如索引、查询缓存和查询优化器等。

以下是一个MongoDB查询优化的具体代码实例：

```javascript
db.collection.createIndex({ name: 1 });
db.collection.find({ name: 'John' });
```

在这个例子中，我们首先创建了一个名为`name`的索引，它用于优化`name`字段的查询。然后，我们使用`find`方法进行查询，并使用`name`字段筛选出名字为`John`的记录。
