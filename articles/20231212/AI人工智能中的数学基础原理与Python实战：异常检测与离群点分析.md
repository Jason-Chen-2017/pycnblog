### 4.1 异常检测代码实例

以下是一个简单的异常检测代码实例：

```python
import numpy as np
import pandas as pd
from sklearn.ensemble import IsolationForest

# 加载数据
data = pd.read_csv('data.csv')

# 分离特征和标签
X = data.drop('target', axis=1)
y = data['target']

# 训练模型
model = IsolationForest(contamination=0.1)
model.fit(X)

# 预测异常数据
preds = model.predict(X)

# 标记异常数据
y_pred = np.where(preds == -1, 1, 0)

# 打印异常数据
print(y_pred)
```

在上述代码中，我们首先加载了数据，然后分离了特征和标签。接着，我们训练了一个IsolationForest模型，并对数据进行预测。最后，我们将预测结果与原始标签进行比较，并标记出异常数据。

### 4.2 离群点分析代码实例

以下是一个简单的离群点分析代码实例：

```python
import numpy as np
import pandas as pd
from sklearn.cluster import DBSCAN

# 加载数据
data = pd.read_csv('data.csv')

# 分离特征和标签
X = data.drop('target', axis=1)

# 训练模型
model = DBSCAN(eps=0.5, min_samples=5)
model.fit(X)

# 标记离群点
labels = model.labels_

# 打印离群点
print(labels)
```

在上述代码中，我们首先加载了数据，然后分离了特征和标签。接着，我们训练了一个DBSCAN模型，并对数据进行分类。最后，我们将离群点的标签打印出来。
