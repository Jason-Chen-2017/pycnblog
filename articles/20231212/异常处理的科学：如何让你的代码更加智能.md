# 2.核心概念与联系
异常处理的科学涉及到以下几个核心概念：

1.异常：异常是程序在运行过程中发生的错误或异常情况，例如空指针异常、数组越界异常等。异常可以分为检查异常和运行异常两种类型。检查异常是指程序员可以通过编写代码来检查的异常，如空指针异常。运行异常是指程序在运行过程中发生的异常，例如数组越界异常。

2.异常处理机制：异常处理机制是程序在发生异常时如何进行处理和恢复的机制。Java语言中的异常处理机制是通过try-catch-finally语句来实现的。在try块中编写可能会发生异常的代码，在catch块中编写异常处理逻辑，在finally块中编写无论是否发生异常都需要执行的代码。

3.异常处理策略：异常处理策略是指程序员在处理异常时采用的策略。常见的异常处理策略有捕获异常、忽略异常、重新抛出异常等。捕获异常是指程序员在catch块中捕获异常并进行处理，忽略异常是指程序员忽略异常并继续执行程序，重新抛出异常是指程序员捕获异常后重新抛出异常以便在更高层次的catch块中处理。

