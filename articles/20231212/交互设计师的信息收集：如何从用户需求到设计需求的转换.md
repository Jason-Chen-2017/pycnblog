# 2.核心概念与联系
在交互设计过程中，核心概念包括用户需求、设计需求、交互设计、用户体验等。这些概念之间存在密切联系，如下所示：

- 用户需求：用户需求是交互设计过程的起点，是用户对系统或产品的期望和需求。用户需求可以通过问卷调查、用户访谈、用户测试等方法收集。

- 设计需求：设计需求是交互设计师根据用户需求制定的具体设计要求，包括界面设计、交互设计、功能设计等。设计需求是用户需求的具体实现。

- 交互设计：交互设计是交互设计师根据用户需求和设计需求设计系统或产品的过程，包括界面设计、交互设计、功能设计等。交互设计是交互设计师的核心技能。

- 用户体验：用户体验是用户在使用系统或产品时的感受和反馈，包括易用性、可用性、可靠性等。用户体验是交互设计的重要目标，交互设计师需要关注用户体验的优化。
