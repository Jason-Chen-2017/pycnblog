### 4.1代码实例

以气候预报为例，我们可以使用Python的NumPy库来编写计算机程序来解决气候预报的有限差分方程。以下是一个简单的气候预报程序示例：

```python
import numpy as np

# 设置参数
L = 1000  # 区域长度
T = 100  # 时间长度
dx = 10  # 空间步长
dt = 1  # 时间步长

# 初始条件
u0 = np.zeros(L)

# 有限差分方程
for n in range(T):
    u = u0.copy()
    for i in range(1, L-1):
        u[i] = u0[i] + dt * (u0[i+1] - 2*u0[i] + u0[i-1]) / dx**2
    u0 = u

# 输出结果
print(u0)
```
