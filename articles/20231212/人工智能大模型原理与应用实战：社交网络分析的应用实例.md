## 3. 社交网络分析的具体代码实例与解释

在这部分，我们将通过一个具体的社交网络分析的代码实例来解释社交网络分析的具体代码实例与解释。

代码实例：

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建一个社交网络
G = nx.Graph()

# 添加节点
G.add_node(1)
G.add_node(2)
G.add_node(3)

# 添加边
G.add_edge(1, 2)
G.add_edge(2, 3)

# 绘制社交网络
plt.figure(figsize=(8, 6))
nx.draw(G, with_labels=True)
plt.show()
```

解释：

在这个代码实例中，我们使用了networkx库来创建一个社交网络。首先，我们创建了一个空的图对象G。然后，我们添加了三个节点1、2、3。接着，我们添加了两条边，分别是1和2之间的边，2和3之间的边。最后，我们使用matplotlib库来绘制社交网络。
