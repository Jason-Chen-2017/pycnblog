# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明如何编写可扩展的代码。

假设我们需要编写一个简单的计算器，可以进行加法、减法、乘法和除法的计算。我们可以使用工厂模式来实现这个功能。

```python
class Calculator:
    def __init__(self):
        self.operators = {
            "+": self.add,
            "-": self.sub,
            "*": self.mul,
            "/": self.div
        }

    def add(self, x, y):
        return x + y

    def sub(self, x, y):
        return x - y

    def mul(self, x, y):
        return x * y

    def div(self, x, y):
        return x / y

    def calculate(self, operator, x, y):
        return self.operators[operator](x, y)

calculator = Calculator()
result = calculator.calculate("+", 10, 20)
print(result)  # 30
```

在这个例子中，我们将计算器的功能拆分成多个方法，每个方法负责一个特定的操作。然后我们将这些方法存储在一个字典中，这个字典就是我们的操作符映射。当我们需要进行某个操作时，我们只需要从字典中获取对应的方法并调用即可。这样我们可以很容易地添加新的操作符和方法，从而实现代码的可扩展性。
