# 2.核心概念与联系

元学习的核心概念包括元知识、元学习、 upstairs learning 和 downstairs learning。元知识是指在学习过程中，模型能够根据当前任务的特点来调整学习策略的知识。元学习是指通过学习元知识来优化模型学习策略的过程。upstairs learning 和 downstairs learning 是两种元学习的主要方法，它们分别是基于 upstairs learning 的元知识学习和基于 downstairs learning 的元知识学习。

3. 使用元策略来调整模型的学习策略，从而提高模型的泛化能力。
