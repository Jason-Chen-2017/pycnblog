```python
import requests
from bs4 import BeautifulSoup

url = 'http://www.example.com/culture'
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')
resources = soup.find_all('div', class_='resource')

for resource in resources:
    title = resource.find('h2').text
    link = resource.find('a')['href']
    print(title, link)
```

这段代码使用 Python 的 requests 库和 BeautifulSoup 库从一个文化与娱乐网站上获取文化与娱乐资源的链接，并将其打印出来。

```python
import requests

events = soup.find_all('div', class_='event')

```
