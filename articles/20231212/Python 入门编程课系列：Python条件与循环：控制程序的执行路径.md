# 2.核心概念与联系

在Python中，条件和循环是编程的基本概念，它们可以帮助我们控制程序的执行路径。条件是一种判断语句，它可以根据某个条件是否满足来执行或跳过代码块。循环是一种重复执行代码块的机制，它可以根据某个条件来控制循环的次数。

## 2.1 条件

条件是一种判断语句，它可以根据某个条件是否满足来执行或跳过代码块。在Python中，条件是通过`if`语句来实现的。`if`语句后面可以跟一个条件表达式，如果条件表达式的值为`True`，则执行代码块；如果条件表达式的值为`False`，则跳过代码块。

例如，我们可以使用`if`语句来判断一个数是否为偶数：

```python
num = 5
if num % 2 == 0:
    print("数是偶数")
else:
    print("数是奇数")
```

在这个例子中，我们首先定义了一个变量`num`，然后使用`if`语句来判断`num`是否为偶数。如果`num`是偶数，则执行`print("数是偶数")`；如果`num`是奇数，则执行`print("数是奇数")`。

## 2.2 循环

循环是一种重复执行代码块的机制，它可以根据某个条件来控制循环的次数。在Python中，循环是通过`for`和`while`语句来实现的。

### 2.2.1 for循环

`for`循环是一种用于遍历集合（如列表、字典、集合等）的循环。它可以根据某个条件来控制循环的次数。`for`循环的基本语法如下：

```python
for 变量 in 集合:
    代码块
```

例如，我们可以使用`for`循环来遍历一个列表：

```python
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num)
```

在这个例子中，我们首先定义了一个列表`numbers`，然后使用`for`循环来遍历`numbers`。在每次循环中，`num`变量会接收列表中的一个元素，然后执行`print(num)`。

### 2.2.2 while循环

`while`循环是一种基于条件的循环。它会不断地执行代码块，直到某个条件为`False`。`while`循环的基本语法如下：

```python
while 条件:
    代码块
```

例如，我们可以使用`while`循环来输出1到10的数字：

```python
i = 1
while i <= 10:
    print(i)
    i += 1
```

在这个例子中，我们首先定义了一个变量`i`，然后使用`while`循环来输出1到10的数字。在每次循环中，`print(i)`会输出当前的数字，然后`i += 1`会增加`i`的值。循环会一直执行，直到`i`大于10，条件为`False`。
