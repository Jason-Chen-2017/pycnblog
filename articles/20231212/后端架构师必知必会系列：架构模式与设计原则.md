## 4.1排序算法实例

### 4.1.1冒泡排序

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 4.1.2快速排序

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

## 4.2搜索算法实例

### 4.2.1二分搜索

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

## 6.1如何选择合适的排序算法？

选择合适的排序算法需要考虑以下几个因素：

- 数据规模：不同的排序算法适用于不同的数据规模。例如，当数据规模较小时，可以选择快速排序；当数据规模较大时，可以选择归并排序或堆排序。
- 时间复杂度：不同的排序算法的时间复杂度不同。例如，快速排序的平均时间复杂度为O(n log n)，归并排序的时间复杂度为O(n log n)。
- 空间复杂度：不同的排序算法的空间复杂度不同。例如，快速排序的平均空间复杂度为O(log n)，归并排序的空间复杂度为O(n)。

## 6.2如何选择合适的搜索算法？

选择合适的搜索算法需要考虑以下几个因素：

- 数据结构：不同的数据结构适用于不同的搜索算法。例如，当数据结构是有序的时，可以选择二分搜索；当数据结构是无序的时，可以选择线性搜索。
- 时间复杂度：不同的搜索算法的时间复杂度不同。例如，二分搜索的时间复杂度为O(log n)，线性搜索的时间复杂度为O(n)。
- 空间复杂度：不同的搜索算法的空间复杂度不同。例如，二分搜索的平均空间复杂度为O(1)，线性搜索的空间复杂度为O(1)。
