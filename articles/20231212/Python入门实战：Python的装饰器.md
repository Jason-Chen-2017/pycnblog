# 4.具体代码实例和详细解释说明
下面是一个具体的装饰器代码实例：

```python
def log_decorator(func):
    def wrapper(*args, **kwargs):
        print(f"Calling {func.__name__}")
        result = func(*args, **kwargs)
        print(f"Finished {func.__name__}")
        return result
    return wrapper

@log_decorator
def add(x, y):
    return x + y

result = add(2, 3)
print(result)
```

在这个例子中，我们定义了一个名为log_decorator的装饰器函数，它接收一个函数作为参数。在装饰器函数中，我们定义了一个名为wrapper的内部函数，该函数接收任意数量的参数和关键字参数。在wrapper函数中，我们调用原函数，并在调用原函数之前和之后打印一些日志信息。最后，我们返回wrapper函数。

通过使用@log_decorator装饰器，我们可以将log_decorator装饰器应用于add函数。这样，当我们调用add函数时，会先调用log_decorator装饰器的wrapper函数，然后再调用add函数本身。
