# 2.核心概念与联系

在计算机视觉中，我们需要处理的数据主要是图像和视频。图像是由像素组成的二维矩阵，每个像素代表了图像在某一点的颜色和亮度信息。视频则是连续的图像序列。

## 3.1 边缘检测

边缘检测是计算机视觉中一个重要的任务，它旨在找出图像中的边缘。边缘是图像中亮度变化较大的区域，通常代表物体的边界。

### 3.1.1 数学模型公式

边缘检测的一个典型方法是利用拉普拉斯算子（Laplacian）来计算图像的二阶导数。拉普拉斯算子的定义为：

$$
L(x, y) = \frac{\partial^2 I(x, y)}{\partial x^2} + \frac{\partial^2 I(x, y)}{\partial y^2}
$$

其中，$I(x, y)$ 表示图像的灰度值，$x$ 和 $y$ 分别表示图像的横坐标和纵坐标。

### 3.1.2 具体操作步骤

1. 对图像进行高斯滤波：高斯滤波可以减弱图像中的噪声，同时保留边缘信息。
2. 计算拉普拉斯算子：使用上述数学公式计算图像的拉普拉斯值。
3. 阈值处理：将拉普拉斯值与阈值进行比较，将大于阈值的像素点标记为边缘点。
4. 二值化处理：将边缘点标记为白色，其他像素点标记为黑色，生成边缘图。

## 3.2 特征提取

特征提取是计算机视觉中一个重要的任务，它旨在从图像中提取有意义的特征，以便进行图像识别和分类等任务。

### 3.2.1 数学模型公式

一个常用的特征提取方法是利用SIFT（Scale-Invariant Feature Transform）算法。SIFT算法的核心思想是通过对图像进行多尺度分析，找到图像中的局部极值点。

SIFT算法的步骤如下：

1. 对图像进行高斯滤波：高斯滤波可以减弱图像中的噪声，同时保留边缘信息。
2. 计算图像的差分图：差分图是图像灰度值的差分，可以突出图像中的边缘和纹理信息。
3. 找到局部极值点：通过对差分图进行扫描，找到图像中的局部极值点。
4. 计算特征向量：对每个局部极值点，计算其周围像素点的梯度信息，生成特征向量。
5. 对特征向量进行稳定性检查：通过计算特征向量的稳定性，筛选出可靠的特征点。
