## 4.1 例子：蒙特卡洛方法求π

我们可以使用蒙特卡洛方法来估计一个圆的周长。假设我们有一个正方形，边长为1，然后在正方形内随机生成一些点。如果一个点在圆内，那么它的数量可以用来估计圆的周长。

```python
import random
import math

def random_point():
    x = random.uniform(-1, 1)
    y = random.uniform(-1, 1)
    return (x, y)

def is_in_circle(point):
    x, y = point
    return x**2 + y**2 <= 1

def monte_carlo_pi(n):
    count = 0
    for _ in range(n):
        point = random_point()
        if is_in_circle(point):
            count += 1
    return 4 * count / n

n = 100000
pi = monte_carlo_pi(n)
print("估计的π:", pi)
```

在这个例子中，我们首先定义了一个随机点生成函数`random_point()`，用于生成随机点。然后我们定义了一个判断点是否在圆内的函数`is_in_circle()`。最后，我们定义了一个蒙特卡洛估计π的函数`monte_carlo_pi()`，通过生成大量的随机点来估计π的值。
