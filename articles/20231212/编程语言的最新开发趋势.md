# 2.核心概念与联系

在讨论编程语言的最新开发趋势之前，我们需要了解一些核心概念。编程语言可以分为两类：编译型和解释型。编译型语言将源代码编译成机器代码，然后直接运行。解释型语言则将源代码逐行解释执行。另一个重要概念是类型系统，它定义了程序中变量和数据类型的规则。
