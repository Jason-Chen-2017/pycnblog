2.使用设计模式的实例：
```python
from abc import ABC, abstractmethod

class Animal(ABC):
    @abstractmethod
    def speak(self):
        pass

class Dog(Animal):
    def speak(self):
        return "Woof!"

class Cat(Animal):
    def speak(self):
        return "Meow!"

def make_animal_sound(animal: Animal):
    return animal.speak()

dog = Dog()
cat = Cat()

print(make_animal_sound(dog))  # 输出：Woof!
print(make_animal_sound(cat))  # 输出：Meow!
```
在这个例子中，我们使用了抽象工厂设计模式。我们定义了一个`Animal`类，它是一个抽象类，有一个抽象方法`speak`。我们定义了两个具体的类（`Dog`和`Cat`），它们都实现了`speak`方法。我们还定义了一个`make_animal_sound`函数，它接受一个`Animal`对象作为参数，并调用其`speak`方法。
