# 4.具体代码实例和详细解释说明

在这里，我们将提供一个具体的代码实例，以及其详细解释说明。

代码实例：

```python
import tensorflow as tf
from tensorflow.keras.layers import Dense, Input
from tensorflow.keras.models import Model

# 定义输入层
input_layer = Input(shape=(784,))

# 定义隐藏层
hidden_layer = Dense(128, activation='relu')(input_layer)

# 定义输出层
output_layer = Dense(10, activation='softmax')(hidden_layer)

# 定义模型
model = Model(inputs=input_layer, outputs=output_layer)

# 编译模型
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# 训练模型
model.fit(x_train, y_train, epochs=10, batch_size=32)
```

详细解释说明：

1.首先，我们导入了TensorFlow库，并从中导入了Dense和Input类。

2.然后，我们定义了输入层，其shape为(784,)，表示输入数据的维度。

3.接着，我们定义了隐藏层，其中使用了ReLU激活函数，并将输入层的输出作为输入。

4.然后，我们定义了输出层，其中使用了softmax激活函数，并将隐藏层的输出作为输入。

5.接着，我们定义了模型，将输入层和输出层作为输入和输出。

6.然后，我们编译模型，使用Adam优化器，并设置损失函数为categorical_crossentropy，并设置评估指标为accuracy。

7.最后，我们训练模型，使用x_train和y_train作为训练数据，设置训练轮数为10，设置批次大小为32。
