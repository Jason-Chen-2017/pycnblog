# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明禅与计算机程序设计的联系。我们将实现一个简单的排序算法，即冒泡排序。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在上述代码中，我们首先定义了一个冒泡排序的函数，它接受一个数组作为参数。然后，我们使用两层循环来遍历数组中的每个元素。在内层循环中，我们比较当前元素和下一个元素的值，如果当前元素大于下一个元素，则交换它们的位置。这样，在每次循环中，最大的元素都会“泡泡”到数组的末尾。最后，我们返回排序后的数组。
