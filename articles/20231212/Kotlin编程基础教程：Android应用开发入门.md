# 4.2编写Android应用程序
以下是一个Android应用程序的代码实例：
```kotlin
import android.app.Activity
import android.os.Bundle

class MainActivity : Activity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
    }
}
```
详细解释说明：
1.`import android.app.Activity`是导入Android应用程序所需的Activity类。
2.`import android.os.Bundle`是导入Android应用程序所需的Bundle类。
3.`class MainActivity : Activity()`是Android应用程序的主要类，它继承了Activity类。
4.`override fun onCreate(savedInstanceState: Bundle?)`是Android应用程序的主要方法，它用于创建Activity的视图。
5.`super.onCreate(savedInstanceState)`是调用父类的onCreate方法。
6.`setContentView(R.layout.activity_main)`是设置Activity的布局文件。
