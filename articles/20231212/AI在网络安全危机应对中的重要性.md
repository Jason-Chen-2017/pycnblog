### 1.4.1 基于规则的监测代码实例

```python
import re

# 定义规则
rules = [
    {"pattern": r"^GET \/.* HTTP\/1.1$", "action": "allow"},
    {"pattern": r"^POST \/.* HTTP\/1.1$", "action": "deny"},
]

# 监测网络流量
def monitor_traffic(traffic):
    for rule in rules:
        if re.match(rule["pattern"], traffic):
            return rule["action"]
    return "unknown"

# 示例用法
traffic = "GET /index.html HTTP/1.1"
print(monitor_traffic(traffic))  # 输出: allow
```

解释说明：

- 定义了一组规则，每个规则包括一个正则表达式和一个动作。
- 监测网络流量，根据规则来检测网络安全事件。
- 示例用法：检测一个 GET 请求，输出允许。

### 1.5.1 基于规则的监测实例

```python
import re

# 定义规则
rules = [
    {"pattern": r"^GET \/.* HTTP\/1.1$", "action": "allow"},
    {"pattern": r"^POST \/.* HTTP\/1.1$", "action": "deny"},
]

# 监测网络流量
def monitor_traffic(traffic):
    for rule in rules:
        if re.match(rule["pattern"], traffic):
            return rule["action"]
    return "unknown"

# 示例用法
traffic = "GET /index.html HTTP/1.1"
print(monitor_traffic(traffic))  # 输出: allow
```
