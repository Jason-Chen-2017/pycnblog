## 4.1 代码实例

我们将提供一个简单的代码实例，用于描述光的传播与散射过程。

```python
import numpy as np
from scipy.constants import speed_of_light

# 定义光的波长和方向
wavelength = 500e-9  # 波长，单位为纳米
direction = np.array([1, 0, 0])  # 方向，单位为弧度

# 定义光的速度和折射率
speed = speed_of_light * 1e9  # 光速，单位为米/秒
refractive_index = 1.5  # 折射率

# 计算光的传播距离和折射角
distance = speed * wavelength / speed_of_light  # 传播距离，单位为米
angle = np.arcsin(1 / refractive_index)  # 折射角，单位为弧度

print("传播距离：", distance, "米")
print("折射角：", angle, "弧度")
```

## 4.2 详细解释说明

在上述代码中，我们首先导入了numpy和scipy.constants库，以便进行数学计算。

接下来，我们定义了光的波长和方向，波长为500纳米，方向为[1, 0, 0]，单位为弧度。

然后，我们定义了光的速度和折射率，光速为3e8米/秒，折射率为1.5。

接下来，我们计算了光的传播距离和折射角。传播距离是光速与波长的乘积，单位为米。折射角是光在不同媒介之间传播时的结果，可以通过np.arcsin函数计算，单位为弧度。

最后，我们输出了传播距离和折射角的值。
