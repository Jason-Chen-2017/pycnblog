然后，我们需要定义一个函数来计算HMAC的哈希值：

```python
def hmac_hash(key, msg):
    return base64.b64encode(hmac.new(key, msg.encode('utf-8'), hashlib.sha256).digest())
```

接下来，我们需要定义一个函数来验证用户的身份：

```python
def verify_user(user_id, password, token):
    # 首先，我们需要计算用户的密码哈希值
    password_hash = hmac_hash(user_id, password)

    # 然后，我们需要计算用户的token哈希值
    token_hash = hmac_hash(user_id, token)

    # 最后，我们需要比较用户的密码哈希值和用户的token哈希值是否相等
    return password_hash == token_hash
```
