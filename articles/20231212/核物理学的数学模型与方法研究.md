# 2.核心概念与联系

在核物理学中，核子是原子核的基本构成部分，由一定数量的核子组成。核物理学的核心概念包括：核子、核能、核衰变、核反应等。

1.核子：核子是原子核的基本构成部分，由一定数量的核子组成。核子由两种基本粒子组成：正电子（proton）和中性子（neutron）。核子的数量称为核数，表示为A，同时还有一个质数，表示核子中正电子和中性子的数量。

2.核能：核能是指核子之间的相互作用产生的能量。核能可以分为两种：稳定核能和不稳定核能。稳定核能是指核子之间的相互作用产生的稳定能量，而不稳定核能是指核子之间的相互作用产生的不稳定能量，可能导致核衰变。

3.核衰变：核衰变是指核子之间的相互作用导致核子的腐蚀或分裂产生的过程。核衰变可以分为两种：Alpha衰变和Beta衰变。Alpha衰变是指核子发射两个中性子和两个正电子的过程，而Beta衰变是指核子发射一个正电子或中性子的过程。

4.核反应：核反应是指核子之间的相互作用导致核子的运动或变化的过程。核反应可以分为两种：碰撞反应和分裂反应。碰撞反应是指核子之间的碰撞产生新核子的过程，而分裂反应是指核子分裂产生新核子的过程。
