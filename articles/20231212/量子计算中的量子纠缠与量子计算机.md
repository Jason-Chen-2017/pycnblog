## 1.2 核心概念与联系

量子计算机的核心概念有两个：量子比特（Qubit）和量子纠缠（Quantum Entanglement）。

### 1.2.1 量子比特（Qubit）

量子比特（Qubit）是量子计算机中的基本单位，它可以存储和处理信息。与传统计算机中的比特（Bit）不同，量子比特可以同时存储0和1，这使得量子计算机具有更高的计算能力。

### 1.2.2 量子纠缠（Quantum Entanglement）

量子纠缠是量子计算中的一个重要概念，它是指两个或多个量子比特之间的相互联系。当两个量子比特相互纠缠时，它们的状态将相互影响，即使它们物理上分离得很远。这种相互影响使得量子计算机能够同时处理多个问题，从而提高计算能力。

以下是一个量子纠缠的代码实例：

import numpy as np
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 创建量子比特
qc = QuantumCircuit(2)

# 初始化量子比特的状态
qc.h(0)

result = simulator.run(assemble(qc)).result()

# 绘制结果
plot_histogram(result.get_counts())
