```python
import hashlib

def hash_password(password):
    """
    密码加密算法
    """
    # 使用SHA-256算法进行密码加密
    hashed_password = hashlib.sha256(password.encode('utf-8')).hexdigest()
    return hashed_password

def verify_password(password, hashed_password):
    """
    密码验证算法
    """
    # 使用SHA-256算法进行密码加密
    hashed_password = hashlib.sha256(password.encode('utf-8')).hexdigest()
    # 比较加密后的密码是否相同
    if hashed_password == hashed_password:
        return True
    else:
        return False
```

在这个代码实例中，我们实现了一个简单的密码加密和验证算法。这个算法使用了SHA-256算法进行密码加密和验证。首先，我们使用`hashlib`模块中的`sha256`函数进行密码加密。然后，我们比较加密后的密码是否相同，以确定密码是否验证通过。
