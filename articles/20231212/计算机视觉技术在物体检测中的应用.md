### 4.3 对象检测
在这个代码实例中，我们将使用Python的OpenCV库来实现对象检测：

```python
import cv2
import numpy as np

# 读取图像

# 创建对象检测器
object_detector = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')

# 检测对象
faces = object_detector.detectMultiScale(img, scaleFactor=1.1, minNeighbors=5, minSize=(30, 30), flags=cv2.CASCADE_SCALE_IMAGE)

# 显示对象图像
for (x, y, w, h) in faces:
    cv2.rectangle(img, (x, y), (x+w, y+h), (255, 0, 0), 2)
cv2.imshow('Object Detection', img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

在这个代码中，我们首先使用`cv2.imread`函数来读取图像，然后使用`cv2.CascadeClassifier`函数来创建对象检测器。最后，我们使用`object_detector.detectMultiScale`函数来检测对象，并使用`cv2.rectangle`函数来显示对象图像。
