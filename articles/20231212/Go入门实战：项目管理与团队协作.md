# 4.具体代码实例和详细解释说明
在这个部分，我们将通过一个具体的代码实例来详细解释Go语言的并发、模块化、接口和错误处理等核心概念的使用方法。

```go
package main

import (
    "fmt"
    "sync"
)

type Task struct {
    id   int
    name string
}

func (t *Task) Do() {
    fmt.Printf("任务%d: %s\n", t.id, t.name)
}

func main() {
    var wg sync.WaitGroup
    tasks := []Task{
        {1, "任务1"},
        {2, "任务2"},
        {3, "任务3"},
    }

    for _, task := range tasks {
        wg.Add(1)
        go task.Do()
    }

    wg.Wait()
}
```

在这个代码实例中，我们创建了一个`Task`结构体，它有一个`id`和一个`name`字段。我们还实现了一个`Do`方法，用于执行任务。在`main`函数中，我们创建了一个`sync.WaitGroup`对象，用于等待所有任务完成。我们创建了一个`tasks`数组，包含了三个任务。然后，我们使用`for`循环来遍历任务数组，并为每个任务创建一个Goroutine来执行任务。最后，我们调用`wg.Wait()`方法来等待所有任务完成。
