# 2.核心概念与联系

## 2.1 结构化思考
结构化思考是一种系统的思考方法，它旨在帮助我们将复杂的问题分解为更小的部分，从而更容易理解和解决。这种思考方法通常包括以下步骤：

1. 识别问题的核心问题：这包括确定问题的主要目标和约束条件。
2. 分解问题：将问题分解为更小的部分，以便更容易理解和解决。
3. 建立关系：确定问题的各个部分之间的关系，以便更好地理解问题的整体结构。
4. 解决子问题：通过解决问题的各个部分，逐步解决整个问题。

## 2.2 金字塔结构
金字塔结构是一种用于表示信息层次结构的图形模型，它可以帮助我们更好地组织和管理信息。金字塔结构通常包括以下组件：

1. 顶部：表示最高层次的信息，通常是问题的目标或约束条件。
2. 中间层：表示问题的各个部分，以及它们之间的关系。
3. 底部：表示问题的细节，如数据和算法。

```python

    x = (b * b - 4 * a * c) ** 0.5
    return (-b + x) / (2 * a), (-b - x) / (2 * a)

a = 1

c = 6

print("x1 =", x1)
print("x2 =", x2)
```
