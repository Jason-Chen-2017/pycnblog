# 2.核心概念与联系

首先，我们需要了解Mesos和Docker的核心概念。Mesos是一个集群资源调度和分配系统，它可以在集群中自动分配资源，以实现高效的资源利用。Docker是一个应用容器引擎，它可以将软件打包成一个独立的容器，以便在任何平台上快速部署和运行。
