```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机数据
x = np.random.rand(100)
y = np.random.rand(100)

# 使用线性回归模型进行拟合
coef = np.linalg.lstsq(x.reshape(-1, 1), y.reshape(-1, 1), rcond=None)[0][0]

# 绘制数据和拟合结果
plt.scatter(x, y)
plt.plot(x, coef * x + np.mean(y), color='red')
plt.xlabel('x')
plt.ylabel('y')
plt.show()
```

在这个代码实例中，我们首先生成了一组随机数据。然后，我们使用线性回归模型进行拟合，并绘制了数据和拟合结果。这个例子涉及到了线性代数（矩阵的逆运算）和数值分析（梯度下降法）的方法。
