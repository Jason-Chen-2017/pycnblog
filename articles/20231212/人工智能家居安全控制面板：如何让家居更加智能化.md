### 1.4.1 智能传感器的代码实例

以下是一个智能传感器的代码实例：

```python
import time
import requests

class Sensor:
    def __init__(self, url):
        self.url = url

    def read(self):
        response = requests.get(self.url)
        data = response.json()
        return data['temperature'], data['humidity'], data['air_quality']

sensor = Sensor('http://sensor.example.com/data')
temperature, humidity, air_quality = sensor.read()
print(f'Temperature: {temperature} C, Humidity: {humidity} %, Air Quality: {air_quality}')
```

### 1.4.2 控制器的代码实例

以下是一个控制器的代码实例：

```python
import time
import requests

class Controller:
    def __init__(self, url):
        self.url = url

    def control(self, temperature, humidity, air_quality):
        data = {
            'temperature': temperature,
            'humidity': humidity,
            'air_quality': air_quality
        }
        response = requests.post(self.url, json=data)
        return response.json()

controller = Controller('http://controller.example.com/control')
response = controller.control(temperature, humidity, air_quality)
print(response)
```
