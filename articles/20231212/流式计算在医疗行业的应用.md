## 4.1 代码实例

我们将通过一个简单的代码实例来详细解释流式计算的实现方法。这个代码实例是一个简单的流式计算程序，它可以用于实时监控患者的心率。

```python
import numpy as np
from sklearn.preprocessing import MinMaxScaler
from keras.models import Sequential
from keras.layers import Dense, LSTM

# 读取心率数据
heart_rate_data = np.load('heart_rate_data.npy')

# 数据预处理
scaler = MinMaxScaler()
heart_rate_data = scaler.fit_transform(heart_rate_data)

# 构建流式计算模型
model = Sequential()
model.add(LSTM(50, input_shape=(heart_rate_data.shape[1], 1)))
model.add(Dense(1))
model.compile(loss='mean_squared_error', optimizer='adam')

# 训练流式计算模型
model.fit(heart_rate_data, np.ones(heart_rate_data.shape[0]), epochs=100, batch_size=32)

# 预测心率
heart_rate_pred = model.predict(heart_rate_data)

# 输出预测结果
print(heart_rate_pred)
```
