# 2. 核心概念与联系
# 2.1 传统关系型数据库与新型SQL的区别

新型SQL数据库的核心概念包括：

1. 分布式数据库：新型SQL数据库通常采用分布式架构，可以在多个节点上分布存储和处理数据，从而实现高性能和可扩展性。
2. 高性能：新型SQL数据库通常采用高性能的存储引擎和优化算法，提供了更快的查询速度和更高的吞吐量。
3. 可扩展性：新型SQL数据库通常采用可扩展的架构，可以根据需要动态地添加或删除节点，从而实现更好的可扩展性。
4. 灵活性：新型SQL数据库通常支持更加灵活的数据模型，可以轻松地添加、删除、修改表结构，以及实现更加复杂的查询和操作。

1. 创建“employee”表：
```sql

    id INT PRIMARY KEY,
    name VARCHAR(255),
    age INT
);
```
2. 插入一条员工记录：
```sql
INSERT INTO employee (id, name, age) VALUES (1, 'John Doe', 30);
```
3. 查询员工记录：
```sql

```
4. 更新员工记录：
```sql

```
5. 删除员工记录：
```sql
DELETE FROM employee WHERE id = 1;
```

通过以上代码实例，我们可以看到新型SQL数据库的使用方法相对简单，并且提供了更高的性能、可扩展性和灵活性。
