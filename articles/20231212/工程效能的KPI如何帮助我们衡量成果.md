我们将使用Python来实现KPI的计算。以下是一个简单的代码实例：

```python

# 定义工程项目的数据
data = [
    {"project": "项目A", "quality": 80, "progress": 90, "collaboration": 85},
    {"project": "项目B", "quality": 75, "progress": 85, "collaboration": 80},
    {"project": "项目C", "quality": 90, "progress": 95, "collaboration": 90},

# 计算KPI

# 打印结果

```
