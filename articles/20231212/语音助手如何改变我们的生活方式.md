```python
import speech_recognition as sr

def recognize_speech():
    recognizer = sr.Recognizer()
    with sr.Microphone() as source:
        print("请说出你的命令：")
        audio = recognizer.listen(source)
    try:
        text = recognizer.recognize_google(audio)
        print("你说的是：", text)
        return text
    except:
        print("抱歉，我没有听清楚你的命令。")
        return None

def respond(text):
    if "打开网页" in text:
        print("正在打开网页...")
        # 这里可以添加代码来打开网页
    elif "设置闹钟" in text:
        print("正在设置闹钟...")
        # 这里可以添加代码来设置闹钟
    else:
        print("抱歉，我不知道如何回应你的命令。")

if __name__ == '__main__':
    text = recognize_speech()
    if text:
        respond(text)
```

在这个代码中，我们首先导入了SpeechRecognition库，然后定义了一个`recognize_speech`函数来识别用户的语音命令。在这个函数中，我们使用SpeechRecognition库的`Recognizer`类来识别用户的语音命令，并使用`Microphone`类来获取用户的语音输入。

接下来，我们定义了一个`respond`函数来回应用户的命令。在这个函数中，我们根据用户的命令来执行不同的操作，例如打开网页或设置闹钟。

最后，我们在主程序中调用`recognize_speech`函数来识别用户的语音命令，并调用`respond`函数来回应用户的命令。
