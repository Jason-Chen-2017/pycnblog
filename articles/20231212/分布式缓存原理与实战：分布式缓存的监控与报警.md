# 2.核心概念与联系

在分布式缓存中，我们需要了解以下几个核心概念：

1.缓存节点：缓存节点是分布式缓存系统中的基本组成单元，它负责存储和访问缓存数据。

2.缓存集群：缓存集群是多个缓存节点组成的分布式缓存系统。

3.缓存键：缓存键是缓存数据的唯一标识，用于在缓存节点之间进行数据分布和同步。

4.缓存值：缓存值是缓存键所对应的数据值。

5.缓存策略：缓存策略是用于控制缓存数据的存储和访问行为的规则，例如缓存时间、缓存替换策略等。
