# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释说明社交网络分析的具体操作步骤。

```python
import networkx as nx

# 创建一个空的社交网络图
G = nx.Graph()

# 添加节点
G.add_node(1)
G.add_node(2)
G.add_node(3)

# 添加边
G.add_edge(1, 2)
G.add_edge(2, 3)

# 计算每个节点的度
degrees = nx.degree(G)
print(degrees)

# 计算每个节点的中心性
centralities = nx.betweenness_centrality(G)
print(centralities)

# 模拟人们之间的信息传播
info_propagation = nx.info_propagation(G)
print(info_propagation)
```

在上述代码中，我们首先导入了`networkx`库，然后创建了一个空的社交网络图。接着，我们添加了三个节点和两个边，并计算了每个节点的度和中心性。最后，我们模拟了人们之间的信息传播。
