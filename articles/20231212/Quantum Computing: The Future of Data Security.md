# 2.核心概念与联系
# 2.1.量子比特（qubit）
量子比特（qubit）是量子计算机的基本单位。与传统的二进制比特（bit）不同，量子比特可以存储多个状态。量子比特可以表示为$$ |0\rangle, |1\rangle, |+\rangle, |-\rangle $$等多种状态。这种多状态存储能力使得量子计算机能够同时处理大量的数据，从而提高计算速度和效率。

# 2.2.量子位操作
量子位操作是量子计算机中的基本操作。量子位操作可以对量子比特进行操作，例如翻转、旋转等。量子位操作可以通过量子电路来表示。量子电路是量子计算机中的基本结构，用于实现量子算法。

# 2.3.量子门
量子门是量子电路中的基本组件。量子门可以对量子比特进行操作，例如H门（Hadamard gate）、CNOT门（Controlled NOT gate）等。量子门可以实现量子位操作，并且可以组合起来实现更复杂的计算。

# 2.4.量子纠缠
量子纠缠是量子计算机中的重要概念。量子纠缠是指两个或多个量子比特之间的相互依赖关系。量子纠缠可以提高量子计算机的计算能力，并且可以用于实现量子加密和量子通信等应用。

qc.h(1)

# 应用CNOT门

qc.x(0)
