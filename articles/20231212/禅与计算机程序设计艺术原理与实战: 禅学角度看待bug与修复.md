### 1.4.1 代码实例

我们来看一个简单的代码实例，它是一个求两个数之和的程序：

```python
def add(a, b):
    return a + b

if __name__ == '__main__':
    a = 10
    b = 20
    print(add(a, b))
```

在这个代码实例中，我们定义了一个名为`add`的函数，它接受两个参数`a`和`b`，并返回它们之和。在`if __name__ == '__main__':`的部分，我们定义了两个变量`a`和`b`，并调用`add`函数来计算它们之和。最后，我们使用`print`函数来输出结果。
