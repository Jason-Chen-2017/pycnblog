### 1.4.1 基于邻域的社群发现
我们可以使用以下代码来实现基于邻域的社群发现：

```python
import networkx as nx

def find_communities(graph, radius):
    communities = []
    for node in graph.nodes():
        neighbors = graph.neighbors(node)
        distances = [nx.shortest_path_length(graph, node, neighbor) for neighbor in neighbors]
        community = [node]
        for neighbor in neighbors:
            if all(distance <= radius for distance in distances):
                community.append(neighbor)
        communities.append(community)
    return communities

graph = nx.Graph()
graph.add_edges_from([(0, 1), (0, 2), (0, 3), (1, 2), (1, 3), (2, 3)])
communities = find_communities(graph, 1)
print(communities)
```

在上述代码中，我们首先导入了`networkx`库，然后定义了一个`find_communities`函数，该函数接受一个图和一个半径作为输入参数。在`find_communities`函数中，我们首先遍历图中的每个节点，然后计算每个节点的邻域中其他节点的距离。接着，我们根据距离规则（例如，最短路径）来识别具有相似距离的子集。最后，我们返回识别出的社群。
