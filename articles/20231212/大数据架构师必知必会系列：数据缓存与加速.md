LRU算法的具体操作步骤如下：

1. 当内存空间不足时，从内存中淘汰最近最少使用的数据。
2. 将新的数据存储到内存中。
3. 更新内存中的数据访问时间戳。

LRU算法的数学模型公式如下：

$$
LRU(x) = \frac{1}{x} \sum_{i=1}^{n} \frac{1}{t_i}
$$

其中，$x$ 是数据的权重，$n$ 是数据的数量，$t_i$ 是数据的访问时间戳。

        self.cache = {}
        self.access_time = []

    def get(self, key):
        if key not in self.cache:
            return -1
        self.access_time.remove(key)
        self.access_time.append(key)
        return self.cache[key]

    def set(self, key, value):
        if len(self.cache) >= self.capacity:

            del self.cache[oldest_key]

        self.cache[key] = value
