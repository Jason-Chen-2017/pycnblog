```c

    return a + b;

```

Go程序`example.go`：

```go

//export Add
func Add(a, b int) int {
    return a + b
}

func main() {
    result := C.add(1, 2)
    fmt.Println("The result is:", result)
}
```

详细解释说明：

1.C语言函数库`example.c`中，定义了一个`add`函数，用于将两个整数相加。

2.Go程序`example.go`中，使用`import "C"`导入C语言函数库。

3.Go程序`example.go`中，使用`//export Add`注释将Go函数`Add`导出给C语言。

4.Go程序`example.go`中，调用C语言函数`add`，并传递参数1和2。

5.Go程序`example.go`中，使用`fmt.Println`打印结果。
