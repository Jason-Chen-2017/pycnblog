## 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来说明5G技术如何提高地球探测的精度、效率和可扩展性。

```python
import numpy as np
from scipy.signal import butter, lfilter

def butter_lowpass(cutoff, fs, order=5):
    nyq = 0.5 * fs
    normal_cutoff = cutoff / nyq
    b, a = butter(order, normal_cutoff, btype='low', analog=False)
    return b, a

def butter_lowpass_filter(data, cutoff, fs, order=5):
    b, a = butter_lowpass(cutoff, fs, order=order)
    y = lfilter(b, a, data)
    return y

def process_data(data, cutoff, fs):
    filtered_data = butter_lowpass_filter(data, cutoff, fs)
    return filtered_data

# 读取地球探测数据
data = np.loadtxt('earth_data.txt')

# 设置滤波器参数
cutoff = 0.1
fs = 100

# 进行滤波处理
filtered_data = process_data(data, cutoff, fs)

# 保存滤波后的数据
np.savetxt('filtered_earth_data.txt', filtered_data)
```

在这个代码实例中，我们使用了Python的NumPy和SciPy库来实现一种低通滤波器。低通滤波器可以用来消除地球探测数据中的高频噪声，从而提高数据的精度。
