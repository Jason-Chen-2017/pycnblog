# 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来详细解释 Stream API 的使用方法。

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class StreamExample {
    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

        // 创建数据流
        Stream<Integer> numberStream = numbers.stream();

        // 对数据流进行操作
        Stream<Integer> evenNumbers = numberStream.filter(n -> n % 2 == 0);
        Stream<Integer> squaredNumbers = evenNumbers.map(n -> n * n);
        Stream<Integer> sortedNumbers = squaredNumbers.sorted();
        Stream<Integer> summedNumbers = sortedNumbers.reduce(0, (a, b) -> a + b);

        // 获取数据流的最终结果
        int result = summedNumbers.findFirst().get();

        System.out.println(result); // 输出：50
    }
}
```

在这个代码实例中，我们首先创建了一个 List 对象，包含了一组整数。然后，我们通过调用 `stream()` 方法来创建一个数据流。接下来，我们对数据流进行了一系列的操作，包括过滤、映射、排序和聚合等。最后，我们通过调用 `findFirst()` 方法来获取数据流的最终结果，并输出结果。
