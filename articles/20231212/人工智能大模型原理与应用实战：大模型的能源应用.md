# 2.核心概念与联系
在讨论大模型的能源应用之前，我们需要了解一些核心概念。

## 2.1 大模型
大模型是指具有大规模参数数量和复杂结构的机器学习模型。这些模型通常需要大量的计算资源和数据来训练，但在训练完成后，它们可以在相对较小的计算资源上进行推理，从而实现高效的预测和分类。大模型的应用范围广泛，包括自然语言处理、计算机视觉、语音识别等领域。

## 2.2 能源资源
能源资源是指可以用于实现各种工作和活动的能量。能源资源可以分为两类：可再生能源（如太阳能、风能、水能等）和非可再生能源（如石油、天然气、核能等）。能源资源的预测、优化和管理对于确保能源安全、可靠和可持续性至关重要。

    Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),

    MaxPooling2D((2, 2)),

    Dense(10, activation='softmax')
])

# 编译模型
model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])
