# 2.核心概念与联系

## 2.1 随机过程

随机过程是一种随时间变化的随机变量序列，每个时刻的随机变量都是独立的。随机过程可以分为离散随机过程和连续随机过程，离散随机过程的随机变量取值是离散的，而连续随机过程的随机变量取值是连续的。随机过程还可以分为有限维随机过程和无限维随机过程，有限维随机过程的随机变量是多维的，而无限维随机过程的随机变量是无限维的。

## 2.2 时间序列

时间序列是一种随时间变化的数值序列，每个时刻的数值都是确定的。时间序列可以分为离散时间序列和连续时间序列，离散时间序列的时间点是离散的，而连续时间序列的时间点是连续的。时间序列还可以分为有限维时间序列和无限维时间序列，有限维时间序列的数值序列是多维的，而无限维时间序列的数值序列是无限维的。

## 2.3 随机过程与时间序列的联系

随机过程与时间序列是相互联系的，时间序列可以被看作是随机过程的特殊情况。具体来说，时间序列可以被看作是随机过程的确定性部分，即随机过程中的随机性部分为零。因此，随机过程可以被看作是时间序列的泛化，时间序列可以被看作是随机过程的特例。
