# 2.核心概念与联系

- 超导电流：超导体在低温下可以导通极高电流，而不会产生电阻。这使得超导体在许多应用中具有优势，例如电磁 levitation 、量子计算、量子通信等。

- 超导体类型：超导体可以分为两类：类型I超导体和类型II超导体。类型I超导体通常是由纯金属或金属酸化物组成，如铅、锂、铜等。类型II超导体通常是由复合材料或复合金属组成，如铬盔铁、钛盔铁等。

- 超导体的应用：超导体在许多领域具有重要的应用价值，包括电磁 levitation 、量子计算、量子通信等。此外，由于超导体可以在低温下导通极高电流，因此它还可以应用于电力传输、磁共振成像（MRI）等领域。

    # 计算超导体的电阻
    R = 1 / (I * T)
    # 绘制超导体的电阻与温度关系

    plt.xlabel('Temperature (K)')
    plt.ylabel('Resistance (ohm)')
    plt.title('Superconductor Resistance vs Temperature')
    plt.show()

if __name__ == '__main__':
    T = 0.01
    superconductor_simulation(T)
