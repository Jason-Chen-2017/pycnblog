# 2.核心概念与联系

## 2.1 服务治理

服务治理是指在多云环境中，通过对服务的发现、调用、监控、管理等方面进行统一的管理和控制，以实现服务的可用性、可靠性、性能等方面的优化。服务治理包括服务发现、服务调用、服务监控、服务管理等几个方面。

## 2.2 服务网络

服务网络是指在多云环境中，通过对服务之间的网络连接进行统一的管理和控制，以实现服务之间的高可用性、高性能、高安全性等方面的优化。服务网络包括服务间的网络连接、网络负载均衡、网络安全等几个方面。

## 2.3 联系

服务治理和服务网络是多云环境下实现更高可用性和性能的关键技术之一。服务治理和服务网络之间存在密切的联系，服务治理和服务网络需要紧密协同，共同实现多云环境下的服务可用性和性能优化。
