我们可以使用Python的scikit-learn库来实现朴素贝叶斯：

```python

from sklearn.naive_bayes import MultinomialNB

# 生成文本数据
texts = ['这是一个Python程序', '这是一个Java程序', '这是一个C程序']

# 生成标签数据
```
