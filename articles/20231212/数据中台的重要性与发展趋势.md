# 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的数据集成示例：

```python
import pandas as pd

# 读取数据
data1 = pd.read_csv('data1.csv')
data2 = pd.read_csv('data2.csv')

# 合并数据
data = pd.merge(data1, data2, on='key')

# 保存数据
data.to_csv('data.csv', index=False)
```

在这个示例中，我们首先使用pandas库读取两个CSV文件，然后使用merge函数将这两个数据集合并为一个新的数据集。最后，我们使用to_csv函数将合并后的数据保存为一个新的CSV文件。
