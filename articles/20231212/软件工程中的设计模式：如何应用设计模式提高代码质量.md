1. 单例模式：
```python

    _instance = None

    @classmethod
    def get_instance(cls):
        if cls._instance is None:
            cls._instance = cls()
        return cls._instance
```

2. 工厂模式：
```python
class Factory:
    @classmethod
    def create(cls, type):
        if type == "A":
            return A()
        elif type == "B":
            return B()
        else:
            return None
```
