# 2.核心概念与联系
代码审查工具是一种自动化的代码检查工具，可以根据一定的规则和标准对代码进行检查，以发现潜在的问题和错误。代码审查工具可以分为静态代码分析工具和动态代码分析工具。静态代码分析工具可以在代码编译之前进行检查，而动态代码分析工具则在代码运行过程中进行检查。

代码审查工具与编译器、链接器、调试器等工具有密切的联系。编译器用于将高级语言代码转换为机器代码，链接器用于将多个对象文件合并成可执行文件，调试器用于在代码运行过程中进行调试。代码审查工具则用于在代码编写和编译阶段进行检查，以发现潜在的问题和错误。
