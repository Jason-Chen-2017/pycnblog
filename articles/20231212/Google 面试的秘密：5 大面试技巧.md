解答：

在这个例子中，我们定义了一个名为 `square` 的函数，它接受一个整数 `n` 作为参数，并返回 `n` 的平方。

# 4.2 数据结构和算法题实例
在 Google 面试中，你也可能会遇到一些数据结构和算法题。这些题目涉及到数据结构、算法等方面。以下是一个简单的数据结构和算法题实例：

题目：实现一个简单的栈数据结构，并实现 push、pop、peek、is_empty 等方法。

解答：
```python
class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        return self.items.pop()

    def peek(self):
        return self.items[-1]

    def is_empty(self):
        return len(self.items) == 0
```
在这个例子中，我们定义了一个名为 `Stack` 的类，它实现了一个简单的栈数据结构。我们实现了 push、pop、peek、is_empty 等方法，以便在需要时对栈进行操作。
