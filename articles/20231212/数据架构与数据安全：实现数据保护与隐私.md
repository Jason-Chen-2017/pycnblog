## 4.1 数据加密

数据加密是一种数据保护技术，它可以帮助我们保护数据免受未经授权的访问和滥用。数据加密可以将原始数据转换为不可读的形式，以保护数据的安全性。以下是一个使用Python的AES加密算法的代码实例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# 生成密钥
key = get_random_bytes(16)

# 加密数据
data = b'Hello, World!'
cipher = AES.new(key, AES.MODE_EAX)
ciphertext, tag = cipher.encrypt_and_digest(data)

# 存储加密数据
print(ciphertext)
```

在这个代码实例中，我们首先生成了一个16字节的随机密钥。然后，我们使用AES加密算法对原始数据进行加密。最后，我们将加密后的数据存储在变量`ciphertext`中。

## 4.2 数据哈希

数据哈希是一种数据保护技术，它可以帮助我们保护数据免受未经授权的访问和滥用。数据哈希可以将原始数据转换为固定长度的哈希值，以保护数据的完整性。以下是一个使用Python的SHA1哈希算法的代码实例：

```python
import hashlib

# 计算哈希值
data = b'Hello, World!'

hex_dig = hash_object.hexdigest()

```

在这个代码实例中，我们首先使用SHA1哈希算法对原始数据计算哈希值。然后，我们将计算出的哈希值存储在变量`hex_dig`中。
