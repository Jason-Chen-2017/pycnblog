# 2.核心概念与联系

在语音识别技术中，我们需要关注以下几个核心概念：

1. 语音信号：语音信号是人类发出的声音，它是由声波组成的。语音信号可以通过麦克风等设备捕捉，并转换为电信号。

2. 语音特征：语音特征是语音信号的一些重要属性，如频率、振幅、时间等。语音特征可以用来描述语音信号的不同方面，如音高、音量、音色等。

3. 语音模型：语音模型是用来描述语音信号和语音特征之间关系的模型。语音模型可以是统计模型，如Hidden Markov Model（HMM），也可以是深度学习模型，如深度神经网络（DNN）。
