# 2.核心概念与联系

元学习与传统的机器学习和深度学习有很大的区别。传统的机器学习和深度学习通常需要人工设计特定的算法和模型，以及大量的手工标注数据。而元学习则可以自动学习出如何学习，从而减少了人工的干预。
