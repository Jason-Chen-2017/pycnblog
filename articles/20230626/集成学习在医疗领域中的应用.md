4.3. 核心代码实现

在Python中，可以使用pandas、numpy等库实现集成学习。下面是一个简单的示例代码，用于实现集成学习对多个病情的分析和诊断：

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import accuracy_score

# 准备数据
data = pd.read_csv('patients.csv')

# 将数据分为训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(data.drop(['result'], axis=1), data['result'], test_size=0.2)

# 构建线性回归模型
lr = LinearRegression()

# 训练模型
lr.fit(X_train.values.reshape(-1, 1), y_train.values)

# 测试模型
y_pred = lr.predict(X_test.values.reshape(-1, 1))

# 计算准确率
accuracy = accuracy_score(y_test.values, y_pred.values)
print('Accuracy:', accuracy)
```
