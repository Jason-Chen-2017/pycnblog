2.1 索引类型

2.2 索引键

索引键是索引的基本构成部分,决定了索引的类型和唯一性。在SQL Server 2016中,索引键可以包含单个列或多列,并且可以根据实际情况进行约束。

2.3 索引排序

索引排序是SQL Server 2016中一个非常重要的概念,可以确保索引中存储的数据按照升序或降序排列。索引排序可以提高查询性能,因为索引中的数据按照升序或降序排列可以减少排序操作的次数。
