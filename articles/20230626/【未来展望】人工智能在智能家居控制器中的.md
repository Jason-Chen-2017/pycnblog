4.4. 代码讲解说明

下面是一个简单的自然语言处理模块的代码实现，用于实现对用户输入指令的语音识别功能：

```python
# 导入需要的库
import speech_recognition as sr

# 创建识别器
recognizer = sr.Recognizer()

# 读取用户输入的指令
user_input = input("请说出你的指令：")

# 使用recognizer识别指令
text = recognizer.recognize_sphinx(user_input)

# 将识别结果转换为小写
text = text.lower()

# 提取指令中的关键词
words = []
for word in text.split():
    words.append(word.strip())

# 提取关键词中的名词
nouns = []
for word in words:
    if word.endswith("的"):
        nouns.append(word.split("的")[0])
    else:
        nouns.append(word)

# 将指令中的名词与识别结果进行匹配
matches = []
for word in nouns:
    if word in text:
        matches.append(word)

# 输出匹配的结果
print("你说了：", "、".join(matches))
```
