## 5. 项目实践：代码实例和详细解释说明

Druid的项目实践主要包括数据预处理、模型训练和推理等步骤。以下是一个简单的代码实例和解释说明：

```python
import druid

# 数据预处理
graph = druid.Graph()
graph.add_nodes(3)
graph.add_edges([(0, 1), (1, 2), (2, 0)])

# 模型训练

model.train()

# 推理

print(result)
```
