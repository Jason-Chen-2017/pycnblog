```python
from sklearn import datasets
from sklearn import svm

# 加载数据集
iris = datasets.load_iris()
X = iris.data
y = iris.target

# 创建SVM模型
model = svm.SVC()

# 训练模型
model.fit(X, y)

# 预测风险
y_pred = model.predict(X)

```

2. 代码解释：以上代码实例中，首先导入了Scikit-learn库中的datasets和svm模块，然后加载了iris数据集，并将其分为特征数据X和标签数据y。接着创建了一个支持向量机（SVM）模型，并将其训练与数据集。最后使用训练好的模型对特征数据进行预测。
