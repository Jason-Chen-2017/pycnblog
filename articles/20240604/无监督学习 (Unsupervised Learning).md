## 5. 项目实践：代码实例和详细解释说明

在实际项目中，无监督学习可以应用于各种任务，例如数据挖掘、预测分析等。以下是一个代码实例，展示了如何使用K-means聚类进行无监督学习。

```python
from sklearn.cluster import KMeans
from sklearn.datasets import load_iris

# 加载数据
iris = load_iris()
X = iris.data

# 进行K-means聚类
kmeans = KMeans(n_clusters=3)
kmeans.fit(X)

# 预测新的数据点
new_data = [[5.1, 3.5, 1.4, 0.2]]
prediction = kmeans.predict(new_data)
print(prediction)
```
