## 4. 数学模型和公式详细讲解举例说明

在上述推荐系统例子中，数学模型的核心是协同过滤算法。以下是一个简化的协同过滤公式：

$$
R(u,i) = \sum_{j \in I_u} w_{ij} \cdot p_j + b_u + b_i
$$

其中，$R(u,i)$表示用户$u$对商品$i$的评分，$w_{ij}$表示用户$u$和商品$i$的相似性，$p_j$表示商品$i$的主观评分，$b_u$表示用户$u$的偏好，$b_i$表示商品$i$的偏好。

## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的Python代码实例，实现协同过滤推荐系统：

```python
import numpy as np
from scipy.spatial.distance import cosine

# 用户-商品评分矩阵
R = np.array([[1, 0, 0, 1],
              [0, 1, 1, 0],
              [0, 1, 0, 0],
              [1, 0, 0, 1]])

# 用户相似性矩阵
W = np.zeros((R.shape[0], R.shape[0]))

# 计算用户相似性
for u in range(R.shape[0]):
    for v in range(R.shape[0]):
        if u == v:
            continue
        W[u, v] = 1 - cosine(R[u], R[v])

# 推荐
def recommend(u, k=3):
    scores = np.dot(W[u], R.T)
    scores[u] = -np.inf
    top_k_indices = np.argsort(scores)[::-1][:k]
    return top_k_indices

print(recommend(0))
```
