## 背景介绍
图（Graph）是计算机科学中一个非常重要的数据结构。它可以用来表示关系、网络、社交链路等复杂的数据结构。图的遍历（Graph Traversal）是指在图中按照一定的规则访问每个节点。它是图中最重要的操作之一，应用非常广泛，包括搜索算法、网络协议、数据库查询等。
## 核心概念与联系
图遍历有两种主要类型：广度优先搜索（BFS）和深度优先搜索（DFS）。它们的主要区别在于访问顺序：广度优先搜索先访问与当前节点最近的节点，而深度优先搜索则是先访问与当前节点最深的节点。实际上，我们可以通过调整搜索顺序来解决不同的问题。
## 核心算法原理具体操作步骤
广度优先搜索（BFS）和深度优先搜索（DFS）都遵循以下基本步骤：

1. 初始化一个空队列或栈，并将起始节点放入队列或栈中。
2. 设置一个访问标记数组，记录每个节点是否已经被访问过。
3. 当队列或栈不为空时，取出队首或栈顶的节点进行访问。
4. 对于当前节点的每个邻接节点，若尚未被访问，则将其放入队列或栈中，并标记为已访问。
5. 重复步骤3和4，直至队列或栈为空。

## 数学模型和公式详细讲解举例说明
图可以用邻接矩阵或邻接列表来表示。邻接矩阵是一个 n x n 的矩阵，其中第 i 行第 j 列的元素表示节点 i 和节点 j 之间的边的存在情况。邻接列表则是一个列表，其中每个节点都有一個子列表，表示该节点的所有邻接节点。

广度优先搜索（BFS）可以用一个队列来实现。每次从队列中弹出一个节点，然后将其所有未访问的邻接节点加入队列。深度优先搜索（DFS）可以用一个栈来实现。每次从栈中弹出一个节点，然后将其所有未访问的邻接节点压入栈中。
## 项目实践：代码实例和详细解释说明
下面是一个 Python 实现的广度优先搜索（BFS）和深度优先搜索（DFS）示例：

```python
import collections

def bfs(graph, start):
    visited = set()
    queue = collections.deque([start])
    while queue:
        vertex = queue.popleft()
        if vertex not in visited:
            visited.add(vertex)
            print(vertex, end=' ')
            for neighbor in graph[vertex]:
                queue.append(neighbor)

def dfs(graph, start):
    visited = set()
    stack = [start]
    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            print(vertex, end=' ')
            for neighbor in reversed(graph[vertex]):
                stack.append(neighbor)

G = {'A': ['B', 'C'],
     'B': ['A', 'D', 'E'],
     'C': ['A', 'F'],
     'D': ['B'],
     'E': ['B', 'F'],
     'F': ['C', 'E']}

bfs(G, 'A')
print()
dfs(G, 'A')
```

## 实际应用场景
图遍历在许多实际应用场景中都有广泛的应用，包括但不限于：

1. 网络协议：路由选择、网络故障检测等。
2. 搜索引擎：网页爬虫和链接分析等。
3. 图像处理：图像分割、图像检索等。
4. 社交网络：好友关系推送、推荐系统等。
5. 数据库：数据库查询、关系图谱等。
