### 5.2 个性化推荐

接下来，我们将使用协同过滤算法对广告进行个性化推荐。以下是一个简单的个性化推荐的示例：

```python
from sklearn.neighbors import NearestNeighbors

# 用户行为矩阵
user_behavior_matrix = data.pivot_table(index='user_id', columns='item_id', values='behavior')

# 协同过滤
model = NearestNeighbors(n_neighbors=3)
model.fit(user_behavior_matrix)

# 个性化推荐
def recommend(user_id, items_seen, n_recommendations):
    user_row_number = user_id - 1
    distances, indices = model.kneighbors(user_row_number)
    
    for i in range(0, len(distances.flatten())):
        if i == 0:
            continue
        else:
            item_index = indices.flatten()[i]
            if items_seen[item_index] == 0:
                return item_index
    return -1

# 推荐广告
recommended_ad = recommend(user_id=1, items_seen=user_behavior_matrix[user_id - 1], n_recommendations=1)
```
