## 数学模型和公式详细讲解举例说明

蒙特卡罗方法的一个经典应用场景是计算π（π）值。我们可以通过以下步骤来计算π：

1. 生成随机数x和y，范围在-1到1之间。
2. 计算x^2+y^2的值。
3. 如果x^2+y^2小于1，则计数加1。
4. 重复步骤1-3，N次。
5. 计算N次的总计数，除以N，得到π/4的值。
6. π的值为π/4的结果乘以4。

## 项目实践：代码实例和详细解释说明

以下是一个Python代码实例，演示如何使用蒙特卡罗方法计算π值：

```python
import random

def monte_carlo_pi(N):
    count = 0
    for _ in range(N):
        x, y = random.uniform(-1, 1), random.uniform(-1, 1)
        if x**2 + y**2 <= 1:
            count += 1
    return count / N * 4

N = 1000000
pi = monte_carlo_pi(N)
print(f"π ≈ {pi}")
```
