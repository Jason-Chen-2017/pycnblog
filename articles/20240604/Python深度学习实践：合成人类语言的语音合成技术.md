首先，我们需要安装librosa库：

```python

```

然后，我们可以使用以下代码来生成语音信号：

```python
import numpy as np
import librosa

# 设置参数
Fs = 22050  # 样本率
duration = 1  # 持续时间
n_fft = 1024  # 快取尺寸
hop_length = 512  # 跳跃长度
win_length = 1024  # 窗口长度
f0 = 440  # 频率

# 生成音频信号
y = np.sin(2 * np.pi * f0 * np.arange(hop_length) / Fs)
y = librosa.effects.pitch_shift(y, Fs, n_steps=2)

# 生成音频波形图
librosa.display.specshow(y, x_axis='time', y_axis='linear')
```
