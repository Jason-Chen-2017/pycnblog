## 核心算法原理具体操作步骤

图的遍历算法的核心原理是基于递归或迭代的。算法的基本步骤如下：

1. 从图中选取一个起始点。
2. 记录起始点的邻接节点。
3. 将邻接节点添加到一个队列或栈中。
4. 从队列或栈中取出一个节点。
5. 如果该节点未被访问过，则标记为已访问，并将其邻接节点添加到队列或栈中。
6. 重复步骤4和5，直到队列或栈为空。

```python
import networkx as nx

# 创建一个无向图
G = nx.Graph()

# 添加节点和边
G.add_edges_from([(1, 2), (2, 3), (3, 4), (4, 1)])
```

## 附录：常见问题与解答

1. 图的遍历算法有什么不同？

图的遍历算法有两种：深度优先搜索（DFS）和广度优先搜索（BFS）。深度优先搜索（DFS）是一种递归或迭代的算法，它沿着图的深度方向遍历节点。广度优先搜索（BFS）是一种迭代的算法，它沿着图的宽度方向遍历节点。

2. 如何选择遍历算法？

遍历算法的选择取决于具体的计算需求。深度优先搜索（DFS）适用于需要深度探索图结构的场景，而广度优先搜索（BFS）适用于需要宽度探索图结构的场景。
