# 2.基本概念术语说明
## 2.1 echo框架概述
echo是一个Go语言编写的Web框架，它的主要特点如下：

1. 非常简洁和快速，基于标准库net/http开发，兼容Go版本1.x。
2. 支持RESTful API。
3. 提供多种内置中间件。
4. 支持Websockets。
5. 内置模板引擎，支持Haml、Pug、Mustache等模板语言。
6. 提供日志系统和访问统计工具。
7. 支持自动注册路由、自定义路由规则。
8. 提供自动TLS证书生成。
9. 使用灵活的数据绑定器进行参数解析。
10. 自带HTTP/2服务器。
