# 1.简介
         
1961年，<NAME>出生于加利福尼亚州费城的一个牧羊人家庭，在小时候他就拥有了智力。毕业后，他进入哈佛大学获得计算机科学学士学位，并在那里取得成功，一路上游历美国各地，不断探索新领域，拓宽眼界，最终发现了人工智能这个无边疆、无限可能的热点。

1970年代，Edison一心开发一款机器学习系统，创造性地提出了神经网络的概念，打破了传统计算模型，并成为当时最先进的研究方向。其手段之精巧，数据处理之高效，运算速度之快，也使得当时的计算机科学界称其为“计算之母”。

1980年，Edison担任美帝海军电脑实验室的副总工程师，领导研制出第一款计算机——EDVAC（Electronic Data Vaulting and Automatic Calculation），为这一新型存储技术奠定了坚实的基础。

1987年，Edison被任命为美国国家科学委员会办公室主任，负责联合国教科文组织的研制工作。他参与制定了第一次国际人工智能交流会议的议程，提出了“第三次浪潮”（Third Wave）的人工智能的概念。

1995年，Edison任职于Oracle公司，并曾多次出任总裁，因此他是业界知名的IT技术专家。

2011年，Edison在皇家航空研究中心出任CTO一职，一直到现在仍然是Oracle公司的首席技术专家。

3、基本概念术语说明

1、人工智能（Artificial Intelligence）
人工智能是指由人类智慧所建构出来的通用智能机器，它是以计算机技术作为工具，从经验中学习，并利用本身的推理能力对新出现的复杂问题进行求解和处理。

人工智能主要包括三个方面：智能学习、自主决策和模仿行为。其中智能学习即让机器从数据中学习，以改善它的决策能力；自主决策即机器可以自己做决定，不需要依赖于外部输入；模仿行为则是指机器可以模仿人的行为，自动执行重复性任务。

人工智能的应用领域十分广泛，如智能机器人、助理系统、图像识别、语言理解与生成、信息检索与分析、医疗健康管理、金融保险、计算机游戏、教育、社会关系处理等。

2、计算机视觉（Computer Vision）
计算机视觉是指通过摄像头或其他传感器捕获到图像后，对图像中的物体、空间、特征等进行识别、理解和描述的一门技术。

计算机视觉具有以下几个特点：

⑴ 计算机视觉技术用于辅助机器识别、理解和解决各种计算机视觉相关的问题，比如图像识别、目标检测、跟踪、分割等。

⑵ 对于一些常见的图像处理任务来说，如图像增强、图像修复、图像合成、图像修复等，计算机视觉都可以使用。

⑶ 计算机视觉技术的发展始于上个世纪九十年代，它通过对图像的快速扫描和图像特征的提取，能够将原始图像转换为便于计算机理解的信息形式。

3、机器学习（Machine Learning）
机器学习是一种让计算机基于数据自动学习的技术。

机器学习的目标就是让计算机从给定的训练数据中找寻规律、知识和模式，并依据此模式对新的、未见过的数据进行预测和分类。

机器学习的核心问题是如何从海量数据中找出有用的模式，从而自动化地实现某种特定功能。目前，机器学习已渗透到各个行业，有着广泛的应用前景。

机器学习的四个主要步骤：

步骤一：收集数据：机器学习需要大量的数据才能进行有效的训练。一般情况下，需要从不同来源收集数据，包括现实世界的数据（例如，图像、文本等）、互联网数据以及产品销售数据。


步骤二：特征工程：数据收集完成后，下一步就是对这些数据进行特征工程，也就是将它们转化为可以用来训练机器学习模型的形式。通常情况下，需要进行数据清洗、去噪、规范化、归一化等预处理操作。


步骤三：选择模型：机器学习的模型有很多种类，包括线性回归、支持向量机、朴素贝叶斯、K近邻法等。根据数据的情况，选择适合的模型可以取得更好的效果。


步骤四：训练模型：在经过了特征工程和模型选择之后，就可以开始训练机器学习模型了。首先，要准备好训练数据和测试数据，然后按照之前的步骤进行数据处理、特征工程等操作。接着，把训练数据喂入模型进行训练，最后对测试数据进行预测评估。如果预测准确率满足要求，就可以部署模型了。

# 2.IBM Watson平台简介

IBM Watson平台是一个商业AI平台，是Watson Systems的产品之一，是最具竞争力的商业AI平台之一。
它提供了一系列的AI服务，包括认知（Conversation）、机器翻译、语音识别、机器学习（ML）、自然语言理解（NLU）、视觉搜索、文本分析、图形分析、实体识别等。同时还提供了一个完整的平台，可以构建和运行机器学习模型，可用于促进企业业务流程和产品研发。
Watson Platform支持用户免费注册，提供了多个免费试用计划供用户选择。

1、认知（Conversation）

用户可以通过聊天界面与Watson Conversation交互。通过对话，用户可以直接与机器人进行互动，获取有关数据、产品、服务或建议。例如，用户可以询问关于火车票的信息，这样机器人就可以根据用户的意愿为用户安排火车票。

2、机器翻译

用户可以将文本翻译为另一种语言，帮助用户完成工作或了解最新消息。它通过一组通用翻译引擎，支持多种语言之间的互译。

3、语音识别

用户可以通过录制语音或从麦克风采集声音，用它来与Watson Speech to Text API进行语音识别。它可以将语音转换为文字，并返回结果。

4、机器学习（ML）

用户可以通过Watson ML Studio建立和运行机器学习模型，可以帮助完成不同的业务需求。它提供了一个简单易懂的图形界面，用户可以轻松地创建模型并部署。

5、自然语言理解（NLU）

用户可以通过Watson Natural Language Understanding API分析文本，并提取出有关其意义的信息。它包括各种场景下的自然语言理解，包括评论分析、文档分析、情绪分析、意图识别等。

6、视觉搜索

用户可以通过上传照片、视频或者网页链接，用它来查找和索引照片。通过图像搜索，用户可以快速找到想要的东西。

7、文本分析

用户可以将文本文件进行解析、分类、主题建模，用它来获取结构化数据。它可以分析网站的网页内容，并且通过聚类、关联分析等方法将内容进行分组。

8、图形分析

用户可以通过上传图像文件、表格或者文本文件，用它来进行图形数据分析。它可以分析图像中的对象、区域、边缘等，并通过映射展示出来。

9、实体识别

用户可以从文本中提取出有关命名实体的信息，用它来进行实体搜索、数据挖掘和知识库构建。它通过词嵌入算法和特征提取算法，可以识别各种类型实体。

除了以上这些服务外，Watson Platform还提供了一种统一的API接口，可以让开发者自由调用这些服务，使得他们可以集成到自己的应用程序中。
