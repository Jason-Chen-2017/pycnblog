# 1.简介
         
　　MyBatis是一个开源ORM框架，它可以作用在不同的持久层框架上，如JDBC、Hibernate等。为了更好的管理MyBatis，方便开发人员配置 MyBatis 的相关属性参数，Spring Boot 提供了 MyBatis starter 模块，它自动从 MyBatis 中加载 MyBatis 配置文件并注入到 Spring Boot 容器中。但是，一般情况下，开发者需要手动编写 MyBatis 的 XML 文件来进行 MyBatis 的配置，所以有必要研究一下 MyBatis 相关的配置项，如何通过 Spring Boot 来设置 MyBatis 属性参数。
         　　本文主要介绍 MyBatis 和 Spring Boot 中 MyBatis 配置文件的相关属性参数以及使用方法。
         　　阅读本文需要对 Java、SpringBoot、Maven 有一定的了解，掌握 MyBatis 相关配置和使用。
         # 2.知识点介绍
         　　1. Mybatis介绍：MyBatis 是 Apache 基金会开源的一款优秀的持久层框架。它支持自定义 SQL、存储过程以及高级映射。 MyBatis 相对于 Hibernate 更加简单，学习成本也比较低。
         　　2. MyBatis 配置文件：MyBatis 的配置文件（mybatis-config.xml）用来读取数据库连接信息、设定 MyBatis 的环境、Mapper 映射文件地址、其他一些 MyBatis 设置等。在 MyBatis 中，可以通过设置 properties 参数来覆盖默认值，也可以通过 mapperLocations 参数来指定多个 Mapper 文件地址。
         　　3. Spring Boot 配置 MyBatis：由于 MyBatis 可以通过引入 MyBatis starter 依赖自动加载 MyBatis 配置文件，所以一般不需要自己创建 MyBatis 的 XML 文件。Spring Boot 会自动扫描 classpath 下的 MyBatis 配置文件并注入到 Spring 容器中。因此，只需在 pom.xml 文件中添加 MyBatis starter 依赖即可。如下所示：

         ```
         <dependency>
            <groupId>org.mybatis</groupId>
            <artifactId>mybatis-spring-boot-starter</artifactId>
            <version>${mybatis.version}</version>
        </dependency>
         ```

         　　4. MyBatis 属性参数：MyBatis 支持多种类型参数，包括 String、Integer、Double、Float、Boolean、Date 等，还可以使用 #{property} 或 $[parameter] 来引用对象或参数的值。Spring Boot 也提供了 MyBatisProperties 类来设置 MyBatis 属性参数。以下是 MyBatisProperties 的属性列表：

         | 属性名 | 类型 | 描述 | 默认值 |
         | :-----| ----: | :------ | -----:|
         | configLocation | Resource[] | mybatis 配置文件的位置 | null|
         | typeAliasesPackage | String[] | 所有需要用到的 TypeAlias 的包路径 | (无) |
         | typeAliasesSuperType | Class<?> | 指定某个类的所有子类的别名 | null |
         | typeHandlersPackage | String[] | 所有需要用到的 TypeHandler 的包路径 | （无） |
         | executorType | ExecutorType | 执行器类型（SIMPLE、REUSE、BATCH） | SIMPLE |
         | lazyLoadingEnabled | boolean | 是否开启延迟加载 | false |
         | aggressiveLazyLoading | boolean | 是否开启积极的延迟加载 | false |
         | multipleResultSetsEnabled | boolean | 是否允许单语句返回多结果集 | true |
         | useGeneratedKeys | boolean | 是否使用数据库主键生成策略 | false |
         | defaultStatementTimeout | int | 缺省超时时间，单位秒 | （无） |
         | mapUnderscoreToCamelCase | boolean | 是否开启驼峰命名规则 | false |
         | callSettersOnNulls | boolean | 是否调用 SETTERS 方法（仅用于 NONEMPTY 参数模式） | false |
         | vfsImpl | VFS | 文件系统实现类 | （无） |

         想要详细了解 MyBatis 中的参数含义，请参考 MyBatis 官网文档或源码。
         
         在实际应用中，Spring Boot 能够自动化地配置 MyBatis，但是 MyBatis 本身拥有丰富的配置项，如果没有仔细配置 MyBatis ，可能会出现意想不到的效果。因此，了解 MyBatis 的各个配置项并合理地使用它们是非常重要的。
