# 1.简介
         
2017年下半年到2018年初，国内外关于人工智能领域的论文、期刊越来越多，而它们的内容又分散在各个领域，形成了许多种类的文章。因此，作为一名有一定经验的研究人员或者工程师，对于每一个方向都有哪些类型的文章以及这些类型应该如何分类，怎样选取合适的学习资源非常重要。
         2019年6月，华东师范大学传播学院吴洁教授接受了知乎网专栏作者匿名用户关于“每一个方向都有哪些类型的文章”的问题的采访，回答了这个问题。以下为吴老师的精彩答案：
         2017年下半年到2018年初，国内外关于人工智能领域的论文、期刊越来越多，而它们的内容又分散在各个领域，形成了许多种类的文章。而对于很多有相关研究经验的同学来说，如何才能快速、全面地获取到所需的信息呢？
         1）初级阶段：从最基础的文章开始学习的人群，关注点主要集中于技术路线、理论研究、深度学习、模型评测等；
         2）中级阶段：从具有扎实的统计功底的学生出发，涉及更多的学习方法，注重理论联系实际、应用系统、理论创新等；
         3）高级阶段：在进入机器学习、深度学习、图像处理、自然语言处理、强化学习、推荐系统、搜索引擎领域之后，可能成为职场上不可或缺的一环。
         对每个阶段来说，建议先了解一下该领域的热门前沿领域，并了解一些学术界的共识，例如机器学习已经被认为是人工智能的一个重要方向，所以你应当从这些热门领域入手，然后根据自己的兴趣爱好进行选择，并多阅读相关论文。相信通过积累，渐渐会有自己的判断能力，掌握阅读优质文章的技巧。不过，如果想要掌握精准的学习方法，你还需要懂得如何获取信息、分析信息、整合信息、梳理知识、归纳总结，这就要求你熟悉各种数据库查询技巧，并具备丰富的英文阅读能力。最后，如果你要进入某个领域成为专家，也需要培养自己的逻辑思维能力、分析问题能力、解决问题的能力等，这是一项长久的过程。
         总之，作为一名研究者、工程师，无论你处于什么阶段，只要有动力，就可以不断学习新的知识、方法、工具，磨炼自己的思维和能力，实现自我价值最大化。“逆水行舟，不进则退”。希望大家能够在关键时刻做到“学以致用”，成为自己想要的样子！
