# 1.简介
  

## 1.1 研究背景
在现代经济发展的今天，如何从宏观经济角度看待股市、债市、外汇市等金融市场，成为一种新的必然趋势？如何通过科学的金融模型对市场进行分析、预测、管理，成为一个新兴的研究方向呢？
## 1.2 研究目的
本文试图系统地阐述相关金融学理论和概念，并以国内比较知名的交易平台——聚宽为例，讨论与股票市场相关的量化投资方法，并根据应用场景对其进行实操讲解。
## 1.3 研究范围
- 量化投资基础理论
- 股市量化交易相关理论
- 股票期权定价模型
- 基于机器学习的股票策略设计及评价
- 个人投资建议及实盘交易操作指导
