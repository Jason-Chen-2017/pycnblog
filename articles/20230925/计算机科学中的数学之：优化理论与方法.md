# 2.优化理论简介
什么是优化问题？为了解决优化问题，人们提出了一系列方法和理论。这些方法可以归纳为以下三类：
## (1) 凸优化：定义了严格局部最小值或全局最小值的优化问题；
## (2) 割舍优化：允许有一部分变量不参加优化，从而取得更好的全局近似解；
## (3) 二次规划：把问题的约束条件等价于一个二次函数的形式，使得求解变得容易，得到唯一最优解。
其中凸优化是最重要的优化问题类型。它定义了优化问题的目标函数，它的值应该极小或极大化，而且是所有可行解的最优值。例如，在某个一维空间中，希望找到一根使曲线最小化的直线，或者在某个二维空间中，希望找到一个面积最大的平行四边形等。而凸优化的关键就是找出这样的目标函数以及如何确定目标函数的一阶和二阶导数。因此，凸优化理论至关重要。

## (1) 最速下降法（Gradient Descent）
最速下降法（Gradient Descent，GD）是最基本的优化算法之一。它的基本思想是寻找一处函数的下降方向，使得这一方向上函数值下降最快。该算法的主要特点是每一步迭代都朝着函数的梯度方向进行调整，使得函数值不断减小，直到达到最佳状态。

## (2) 牛顿法（Newton Method）
牛顿法（Newton Method）是解决最优化问题的一种计算方法。它通过海塞矩阵（Hessian Matrix）的逆矩阵来逼近目标函数的二阶导数，从而找出函数的极值点。对于二次型的目标函数，牛顿法非常有效。

## (3) 拟牛顿法（Quasi-Newton Method）
拟牛顿法（Quasi-Newton Method），又称为近似牛顿法，是一种利用海塞矩阵进行数值计算的方法。与牛顿法不同的是，拟牛顿法采用一系列的预处理过程，在迭代过程中利用海塞矩阵的近似来加快收敛速度。

## (4) 提升法（Ascent Method）
提升法（Ascent Method），也叫方向传播法，是用于求解凸优化问题的一种算法。它是利用目标函数的高度曲面（海森矩阵）来确定搜索方向，然后按照高度方向沿着梯度方向走，同时更新自变量，直到满足终止条件或接近最优解。

## (5) 支配法（Subgradient Method）
支配法（Subgradient Method），也称为表弟法，是另一种求解凸优化问题的算法。它与分支定理类似，寻找在可行区域内目标函数减小最慢的方向作为搜索方向，从而快速缩小搜索范围，获得可行解。

## （6） 单调方程法（Monotone Equation Methods）
单调方程法（Monotone Equation Methods）与支配法一样，也是求解凸优化问题的算法。它的基本思想是在不可行区域内，寻找不等式约束下的最优解，然后代入固定变量条件下，解相应方程。

# 3.线性规划
线性规划（Linear Programming）是优化问题的一种形式。它定义了一个标准型的优化问题，其中要最小化或最大化某个线性函数，限制条件为一个或多个线性约束。

## （1） 无约束线性规划
给定一个二维实数向量$x=(x_1, x_2)$，目标函数$f(x)=\sum_{i=1}^{n}c_ix_i+d$，且没有任何约束。求解目标函数的最小值，即$min\{ f(x): \forall x \in \R^2 \}$ 。线性规划问题可以转换为如下的二次规划问题：$\min_{x}\frac{1}{2}x^TQx + q^Tx,$ $\text{s.t.} Ax = b$。其中，$Q=(q_{ij}), A=(a_{ij})$, $b=(b_j), c=(c_i), d=d$.

## （2） 有限约束线性规划
给定一个二维实数向量$x=(x_1, x_2)$，目标函数$f(x)=\sum_{i=1}^{n}c_ix_i+d$，约束条件为$\sum_{i=1}^nx_i=\gamma,\sum_{i=1}^ny_i=\delta$。求解目标函数的最小值，即$min\{ f(x): \forall x \in \R^2, \sum_{i=1}^nx_i=\gamma, \sum_{i=1}^ny_i=\delta \}$ 。线性规划问题可以转换为如下的二次规划问题：$\min_{x}\frac{1}{2}x^TQx + q^Tx,$ $\text{s.t.} Ax = b,\quad Cx \leq d.$其中，$Q=(q_{ij}), A=(a_{ij}), b=(b_j), C=(c_i), D=(d_i), x=(x_1, x_2), y=(y_1, y_2)$。这里，$C$是等号约束，$D$是不等号约束。当不等号约束有界时，可以通过变换约束关系来实现等号约束。如若$\exists u>0$ s.t. $Cx \leq uD$，则$\max\{f(x)-u\}$可以转化成线性规划问题，由此可以求得线性规划问题的最优解$y$。

## （3） 双变量线性规划
给定一个二维实数向量$x=(x_1, x_2)$，目标函数$f(x)=\sum_{i=1}^{n}c_ix_i+d$，约束条件为$Ax \leq b$。求解目标函数的最小值，即$min\{ f(x): \forall x \in \R^2, Ax \leq b \}$ 。线性规划问题可以转换为如下的二次规划问题：$\min_{x}\frac{1}{2}x^TQx + q^Tx,$ $\text{s.t.} Ax \leq b.$其中，$Q=(q_{ij}), A=(a_{ij}), b=(b_j), c=(c_i), d=d$.

# 4.整数规划
整数规划（Integer Programming）是优化问题的一种形式。它定义了一个标准型的优化问题，其中要最小化或最大化某个线性整型函数，限制条件为一个或多个整数约束。

## （1） 布尔变量线性规划
给定一个二维整数向量$x=(x_1, x_2)$，目标函数$f(x)=\sum_{i=1}^{n}c_ix_i+d$，约束条件为$\sum_{i=1}^nx_i=\gamma,\sum_{i=1}^ny_i=\delta$。求解目标函数的最小值，即$min\{ f(x): \forall x \in Z^2, \sum_{i=1}^nx_i=\gamma, \sum_{i=1}^ny_i=\delta \}$ 。线性规划问题可以转换为如下的二次规划问题：$\min_{x}\frac{1}{2}x^TQx + q^Tx,$ $\text{s.t.} Ax = b,\quad Cx \leq d.$其中，$Q=(q_{ij}), A=(a_{ij}), b=(b_j), C=(c_i), D=(d_i), x=(x_1, x_2), y=(y_1, y_2)$。这里，$C$是等号约束，$D$是不等号约束。当不等号约束有界时，可以通过变换约束关系来实现等号约束。如若$\exists u>0$ s.t. $Cx \leq uD$，则$\max\{f(x)-u\}$可以转化成线性规划问题，由此可以求得线性规划问题的最优解$y$。

## （2） 容量约束线性规划
给定一个二维整数向量$x=(x_1, x_2)$，目标函数$f(x)=\sum_{i=1}^{n}c_ix_i+d$，约束条件为$\sum_{j=1}^m a_{ij}x_j\leq B$，$0\leq x_j \leq M$。求解目标函数的最小值，即$min\{ f(x): \forall x \in Z^2, \sum_{j=1}^ma_{ij}x_j\leq B, 0\leq x_j \leq M \}$ 。线性规划问题可以转换为如下的二次规划问题：$\min_{x}\frac{1}{2}x^TQx + q^Tx,$ $\text{s.t.} Ax \leq b,\quad x\geq 0, x\leq M.$其中，$Q=(q_{ij}), A=(a_{ij}), b=(b_j), c=(c_i), d=d$.
