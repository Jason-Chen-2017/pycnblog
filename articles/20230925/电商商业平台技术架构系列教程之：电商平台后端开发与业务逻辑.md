# 1.简介
  

## 1.1 关于本系列教程

随着互联网经济的蓬勃发展，电子商务平台迅速崛起，成为支撑实体经济的“避孕套”。在如今这个价值网络化的时代里，电商网站对于实体经济的重要性已经无法用“八股文”来形容了。

电商商业平台技术（Business Platform Technology）是指能够对接实体经济的电商公司内部管理、交易流程、客户关系、商品销售、支付结算等信息系统的设计和研发。

随着互联网的飞速发展，电商平台也越来越复杂，为了应对日益增长的用户规模和数据量，一些互联网公司和国内外技术公司都陆续推出了基于微服务架构的云计算平台，用于支持电商公司进行更高效的业务模式升级。

但是，如何确保电商平台后端开发工程师具备较强的编程能力、解决问题的思维能力、分析问题的能力、团队合作精神、敏捷开发能力、架构设计能力、性能调优能力和微服务的使用经验等诸多软技能，仍然是一个需要进一步研究和培养的方向。

而本系列教程中的电商平台后端开发技术架构，就是为了帮助电商平台相关技术人员快速成长，提升自己的知识体系、能力水平，并为企业提供一个快速的解决方案。在本教程中，将从以下六个方面对电商平台后端开发技术架构做深入剖析：

1. 基础概念术语说明：本章将对一些相关概念和术语做快速介绍；
2. 服务分层及通信协议：本章将介绍服务分层及通信协议之间的关系，以及各层功能划分；
3. 数据持久化方案：本章将介绍各种数据持久化方案，并结合电商平台的业务场景，给出推荐的存储技术选型；
4. 消息队列技术：本章将介绍消息队列的基本原理、应用场景以及实际案例分析；
5. 分布式缓存技术：本章将介绍分布式缓存技术，包括Redis和Memcached，以及实践案例分析；
6. 系统安全机制：本章将介绍常用的系统安全机制，包括身份验证、授权、访问控制、加密传输、日志记录等，并结合电商平台业务场景，给出实际可行的安全防护策略。

通过阅读和实践这些内容，可以帮助读者了解电商平台的后台技术架构，以及解决技术难题的能力和方法。

## 1.2 本系列教程与其它学习资料的区别

相信很多同学在学习或准备实习前已经看过或者听说过各种电商平台相关的学习资料，但大多不免觉得资料过于陈旧，缺乏实战能力和业务经验。

因此，笔者参考国内外知名电商平台产品的成功经验，对电商平台技术架构这一主题进行了全面的学习和总结，即“电商商业平台技术架构系列教程”。通过系统性的知识结构梳理，以及相关的项目实战案例，让读者掌握电商商业平台的技术架构原理，并且能够利用所学的知识点，快速开发具有完整业务功能的后端系统。

由于篇幅限制，本教程重点讲述了基于Java语言的电商平台技术架构，并且侧重于后端开发及框架方面的知识，希望能够有效提升读者的技术理解和职业生涯规划。同时，欢迎大家通过微信公众号“Java攻城狮”加入我们的讨论群。
