# 1.简介
  

## 什么是树
树(Tree)是一种无序的、由节点组成的集合，在现实生活中，很多事物都可以看作是树的形态，比如一棵庄稼或一幅中国地图都是树形结构。树在计算机科学中起到很重要的作用，它用作组织数据、表示计算模型、实现抽象数据类型以及进行其他很多应用。在数据结构与算法领域，树是一种重要的数据结构。通过对树的了解，可以更好地理解很多相关的问题，包括文件系统、编译器设计、数据库索引、最短路径算法等。

## 为什么要学习树
学习树，可以让我们更好的理解和使用树这种数据结构。很多实际的问题都可以通过树的形式来解决。树结构的数据组织方式使得对数据的快速检索和分析变得十分容易。因此，学习树的数据结构，对于掌握面试中的一些算法问题以及之后的工作也会有很大的帮助。

## 树的特点
树具有以下几个特征：

1. 每个节点最多只能有一个父亲节点；
2. 没有任何一个节点的子孙数量超过两个；
3. 从根节点到任意一个叶子节点的长度相同；
4. 一颗树只有一个根节点。

以上几条特性，给树带来了独特的结构，能够帮助我们更加直观地认识树。同时，它还具有以下优点：

1. 树是一种简单而又巧妙的表示复杂系统的方式；
2. 树的层次遍历（从上到下、从左到右）非常有用；
3. 对树进行动态更新和删除操作也比较方便。

# 2.基本概念
## 树的定义
树是由n（n>=0）个节点组成的一个具有层次关系的集合。对于任意一个节点，除了该节点外，都存在唯一一个指向它的连接边。如果没有连接边，则这个节点称为叶子节点，反之，则称为非叶子节点。根节点是树上的第一个节点。


如图所示，左侧树是一个二叉树，它的每个节点至多有两个孩子节点，即左孩子和右孩子，根节点只有左孩子。右侧树是一个完全二叉树，它的每个节点都有两个孩子节点，且其余节点均有双亲节点。

## 二叉树
二叉树是树的一种特殊类型，它每个节点最多有两颗子树，通常子树被称为左子树和右子树。

### 二叉搜索树
二叉搜索树(BST)，顾名思义，就是二叉树的一种，其中，左子树中的所有节点的值一定小于根节点的值，右子树中的所有节点的值一定大于根节点的值。它具有以下性质：

1. 若左子树不空，则左子树上所有节点的值均小于根节点的值；
2. 若右子树不空，则右子树上所有节点的值均大于根节点的值；
3. 左、右子树也分别为二叉搜索树；
4. 没有键值相等的节点。

二叉搜索树是一个自平衡的排序二叉树。它支持一下两种操作：

1. 插入元素：新节点插入后，仍然保持二叉搜索树的性质；
2. 查找元素：根据值来查找节点的时间复杂度为O(logN)。

## 堆
堆是一种特殊的树型数据结构，满足特定性质的完全二叉树。堆中的每一个节点都大于或等于其子树的所有节点，并且高度也是最低的。它可以用于实现优先队列或者求最大（最小）值。

### 大根堆与小根堆
堆有两种形式——大根堆与小根堆。若堆中所有节点的值都大于或等于（小于或等于）其子树的根节点的值，则称为大根堆；反之，则称为小根堆。

大根堆中，堆顶元素的值是所有节点中最大的。小根堆中，堆顶元素的值是所有节点中最小的。


### 堆的性质
1. 如果堆是一颗完全二叉树，那么堆顶元素(堆中最小的元素)肯定在树的第i层，i=ceil[(logn+1)/2]。
2. 在任意时刻，堆中最大元素(堆顶元素)总是位于数组的第一个位置。
3. 堆的高度h<=logn,其中n是数组大小。
