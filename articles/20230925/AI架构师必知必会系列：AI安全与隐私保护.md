# 2.基础概念和术语
## 什么是AI？
AI（Artificial Intelligence）是指由计算机模拟人的智能行为而实现的自然科技，属于人工智能的一个分支。它具有领域知识、学习能力、推理能力、分析能力、创造能力等特征，可以进行预测、决策、学习、 reasoning和探索等功能。
## 什么是机器学习？
机器学习（Machine Learning）是指让计算机程序在经验（Data）的基础上学习，改善它的行为，提升它的效率的方法。它把很多杂乱无章的、不可靠的、反复出现的、但又十分相关的问题抽象成一种模式，通过提取规则或者对数据的形式进行建模，使得计算机具备识别、预测和决策的能力。
## 什么是深度学习？
深度学习（Deep Learning）是指让计算机通过大量的训练数据自动找寻并学习数据的内在规律，从而实现某些目标任务的算法。深度学习的算法通常由多个非线性的、多层次的感知器组成，并且可以使用不同的优化方法，来有效地解决各种问题。深度学习目前取得了极大的成功，取得巨大的突破性进步。
## 什么是强化学习？
强化学习（Reinforcement Learning）是指让机器通过不断获取奖励和惩罚信号，不断试错去解决一个长期的控制任务。强化学习的环境是一个动力系统（agent）与其环境之间的交互，环境反馈给 agent 一个状态（state），agent 根据这个状态做出动作，环境根据 agent 的动作反馈给 agent 下一个状态，这样循环往复，agent 在这个过程中不断学习、调整策略，最终达到学习效用函数最大化。
## 什么是生成对抗网络GAN？
生成对抗网络（Generative Adversarial Network，GAN）是一种生成模型，它可以生成和欺骗他人。其主要特点是通过对抗的方式训练网络，使生成网络产生与真实数据尽可能相似的数据，从而获得令人信服的数据分布。GAN的两个网络，即生成网络G和判别网络D，彼此博弈，通过交替更新，让G逐渐地变越来越像D的分布。
## GANomaly
GANomaly是指通过对抗网络对异常行为进行分类的一种新型的异常检测方法。该方法不需要事先知道所要检测的异常类型，只需要提供足够数量的正常样本、异常样本及相关标签信息即可完成训练和测试。
## 什么是联邦学习？
联邦学习（Federated Learning）是一种跨设备、跨平台、跨组织的机器学习方案，它允许不同机构、组织之间共享数据进行机器学习，并且不受数据隐私泄露风险影响。联邦学习的目的是为了将本地数据进行集体学习，避免数据孤岛现象的发生，提升模型的泛化能力。
## 什么是加密技术？
加密技术（Cryptographic Technology）是指利用加密算法对数据进行处理、加/解密、签名/验证等操作，实现信息安全的一种技术。加密算法采用数学方法，用于在不安全的通讯环境中传递消息，使数据或通信内容完全隐藏。
### Homomorphic Encryption
Homomorphic Encryption（也称为Homo encryption）是指利用代数结构将加密运算同另一个可逆运算关联起来，这样就可以对加密后的数据再次进行加密运算而得到与原始数据相同的结果。可以对加密后的数据进行加密运算，也可对加密后的结果进行解密运算，但不能修改加密前的明文数据。
### Secure Multi-Party Computation
Secure Multi-Party Computation（MPC）是指通过多方计算协同完成计算任务，使计算过程不暴露参与计算的个人信息。这种计算方式一般要求所有参与计算的个人提供统一的密钥，只有获得该密钥的人才能进行解密和还原原始数据。
### Homomorphic Identification
Homomorphic Identification（HIPE）是一种用于身份鉴别的加密技术。它可以用来确定两个加密数字是否对应同一个实体。HIPE协议中，加密后的数据可以分配给不同的人进行分析，但是无法直接查看该加密数据的内容。如果两者的加密运算结果相同，就能判断两者是否对应同一个实体。HIPE协议能够提供出色的身份鉴别性能。
