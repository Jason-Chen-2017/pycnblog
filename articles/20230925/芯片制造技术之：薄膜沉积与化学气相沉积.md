# 1.简介
  

## 概述
电子技术、计算机技术和无线电技术是当前信息社会的三大科技领域。其中电子技术的主要研究重点是制造和应用电子组件及系统。电子技术的发展以摩尔定律和芯片制造技术的发展而成为“集约”时代的产物。芯片制造技术通过微型化、批量生产和智能控制来提升产品的可靠性和成本效益，进而推动人类信息化建设向前迈进。

传统的核聚变反应堆（CSTR）是世界上最先进的精细化合物燃料环，但它的固体燃料（如液氮）和高温等特性使得它无法用于高功率甚至是超导应用。由于核聚变燃料所需的天然气和塑料等燃料资源匮乏，因此国际上曾引起热议。

在当前技术水平下，其固体燃料还有其他替代方式吗？答案是否定的。因为在精密制程（机械设计、工艺精湛）方面需要成熟的专业知识和工程能力；而且，在大多数情况下，相比于核燃料，化学燃料更具商业价值，且可以解决核燃料供不应求的问题。

芯片制造业虽然占据了最高的利润率，但是却不是唯一选择。另一种重要的选择来自于纳米技术，它利用微型、便携、不易腐蚀、不容易破坏、不容易储存的晶体元件来形成面板材料或单个电路片。纳米技术对解决信息通信、生物医疗和工业自动化等应用领域的需求具有巨大的潜力。

这项工作的目的就是为了探讨为什么化学气相沉积会影响芯片制造中的性能，以及如何用化学气相控製的方法来减小这种影响。

## 物理原理
### 薄膜结构
薄膜结构（nanostructure），指由微观颗粒组成的晶格体系，每颗粒仅占晶格体的一小部分，形状类似电极。许多高性能集成电路（IC）都依赖于薄膜结构，如微芯片、集成电路芯片（FPGA）。因此，理解薄膜结构对于理解芯片制造的影响非常重要。

一般来说，薄膜结构的晶格分辨率可以达到1-2 nm，即每一颗粒大小为1-2 Å，有着独特的晶体性质，如下图所示。由于薄膜结构中每颗粒尺寸较小，晶体中存在较少的离子，因此该结构可以容纳高阶元素或高频离子。如同电极一样，薄膜结构也可以对物质进行收集，并通过光耦合与电荷耦合过程将离子传输到其它位置。这样，薄膜结构的功能就像一个激光扫描器一样，能够快速、准确地感知外部环境的变化。

### 芯片温度与温场
一般来说，普通PCB板上的晶格硅层是绝缘层，外表看起来很干净。但在制造嵌入式系统时，往往会在硅基层次引入一个嵌入式层，这就要求嵌入式层的温度要低于硅基层的温度，否则会发生结构失焊或过热现象。另外，嵌入式设备还包括外围元器件，这些元器件可能被嵌入硅基层中，如果它们的温度高于硅基层的温度，也可能导致结构失焊或过热现象。因此，芯片制造人员经常需要进行嵌入式设计，以避免这种温度冲击。

芯片制造过程通常受到各种各样的温度影响，例如：温度源的影响、加热材料的温度、硅的膨胀、厚度变化后的表面温度、氧化层温度、电源室温度等等。所有这些都可能会影响硅层的温度，从而影响芯片的性能。了解芯片的温度对理解芯片的性能影响至关重要。

温度场是指在特定温度条件下的局部空间分布的总体情况。根据温度场的不同特征，温度场可以分为四种类型：

- 布朗温度场：由热力学第二定律所建立的温度场。晶体结构物内层布朗运动，外层布朗无感觉。布朗温度场随晶体结构物的周期性变换而发生变化，在某个特定温度下，布朗温度场各点取值相同。布朗温度场常用于测量电场中电荷的平均存在时间、功耗、电阻等。
- 焓方温度场：由电磁场的物理性质和电磁波的波动性质所建立的温度场。从电流中产生的电场、磁场分布到晶体，从而产生空气柱状物理性质，每个空气柱状物理区域的高度由晶体周期决定的。焓方温度场随晶体的周期性变换而变化，在某个特定温度下，各点焓方温度场取值相同。焓方温度场常用于测量电场中电荷的平均带宽、跳跃损耗、时延等。
- 比温度场：晶体晶粒之间存在一定距离，这会使晶粒之间的相互作用受到温度的限制，因此，晶体温度越高，此距离也越大。在某些晶体温度下，比温度场就表示晶粒之间的距离差异。比温度场常用于计算交联带隙宽度、折射率、真空弯曲等。
- 高斯温度场：由物理上的热方程所建立的温度场，因此，实际上，它只是比温度场的特殊形式。在某些晶体温度下，高斯温度场存在，但比温度场较小。常用于计算晶体界面平衡配额、反向电导率等。

各式各样的温度场都可以影响芯片的性能，包括晶体的温度、氢和氧的温度、电压电流等参数的温度、加工工艺温度、内部温度、外界温度等等。

### 剥蚀过程
在制作IC之前，一般会采用膜法去除多余的孔洞、磕碰边缘。对于晶圆结构，剥蚀包括膜顶脱落、氧化、氮化、局部溶解、缝隙填充、片刻膨胀等过程。其中，膜顶脱落和氧化是常规操作，氮化和局部溶解是工艺工序。当晶圆在一些工艺工序中膨胀后，在剥蚀过程中就会出现短缺或者死亡的情况。

如果晶圆在某些情况下没有得到完整的膜脱落过程，就可能发生含氧量增多、紫外线暴露、物化学品杂质吸收等一系列问题，甚至会发生完全崩坏。所以，IC制造过程中对晶圆进行全面的剥蚀工作是十分必要的。

目前，国内已经有很多基于人工智能的剥蚀方法，以期望实现实时的、无人监督的IC剥蚀过程。例如，基于计算机视觉的IC消解技术，以及基于机器学习的端到端学习技术。但是，基于人工智能的IC消解技术目前还处于初级阶段，难以应用于实际生产过程。

## 方法
### 量化分析
目前，绝大多数芯片制造商都会选用计算机模拟分析工具来验证他们的设计方案。模拟分析工具的输出结果可以用来评估芯片的性能、可靠性、功耗、尺寸、价格等性能指标。但是，这类工具只能给出大致的数字结果，不足以判断有待改进的地方。

因此，芯片制造商可以采用更多的量化分析手段，如：测试、模拟、仿真、量化优化、优化模型等，获取更加具体的芯片性能数据。这些数据既可以直接反映芯片的性能，又可以提供对芯片的设计策略进行调整、改善的参考依据。同时，也可以对性能数据进行数据处理、统计分析、回归分析、数据可视化等，以帮助芯片制造商分析和预测芯片的未来表现。

### 温度补偿机制
随着制造工艺的不断进步和设备的复杂化，芯片制造过程常常遇到温度的不稳定性、失焊、负载失效等问题。为解决这一问题，芯片制造商们在制造过程中通常会采用温度补偿机制。

温度补偿机制的核心思想是通过控制温度的方式来消除温度差异。常用的温度补偿机制有两种：一种是恒温补偿，它是指让整个芯片均匀加热到一个稳定温度，然后再把芯片送出；另一种是局部温度补偿，它是指用浅层浸没法生产线的温度来降低生产线中间的温度。温度补偿机制可以有效消除温度差异，使芯片的设计变得更加稳定。

### 可控化工艺
芯片制造过程中通常会采用工艺措施来优化IC的性能。在可控化工艺中，芯片制造商会采用机器人、自动化工具、电路板、浴室等进行精心设计的工艺流程，使IC的生产周期缩短，以降低成本。同时，芯片制造商还可以在工艺过程中引入输送控制、制冷释放等技术，来降低制造过程中的油耗、风险、管理难度。

可控化工艺的优势主要有以下几点：

1. 缩短制造周期：制造周期短的同时，就可以增加订单量，提高生产率，从而降低成本。

2. 提高产品的生命周期：由于IC的质量、安全性、可维护性都有较高的标准要求，因此IC制造商在制造过程中引入可控化工艺可以降低这些标准要求，提高产品的生命周期。

3. 降低成本：由于工艺措施可以大大缩短制造周期，降低制造成本，因此IC制造商可以开展大规模、超长周期的芯片制造项目，从而降低成本。

4. 提高竞争力：由于可控化工艺可以在制造过程中增加控制难度、降低油耗，因此它可以削弱竞争对手的优势。
