# 2.基本概念术语说明
## 2.1 社交媒体营销的定义
社交媒体（Social Media） marketing ，简称 SMM （ Social Media Marketing）。SMM 是指利用互联网社交媒体平台提供的各种功能和服务，进行针对特定用户群体的营销活动。一般包括以下四个方面：

1. 搜寻定位：通过搜索引擎获取潜在目标群体。
2. 内容策划：制作符合目标消费者口味的内容推送。
3. 动因分析：了解消费者为什么喜欢你的品牌、产品或服务。
4. 互动营销：与用户进行沟通、促成消费行为，提升活跃度。

## 2.2 数据科学中的常用术语
1. 数据：计算机生成的各种数字、文本、图形等信息。
2. 数据源：数据的产生途径。如人工采集、网站自动跟踪、手机 App 收集。
3. 数据结构：数据中各项特征的逻辑关系。如记录、字段、域、关系。
4. 观测变量（Outcome Variable）：被试者或研究所关心的问题或结果。
5. 自变量（Explanatory Variables）：影响观测变量的变量。
6. 时间序列数据：指按照时间先后顺序排列的数据。
7. 分类数据：具有某种内在联系的有限个离散组别。
8. 回归数据：具有连续性的实数值变量。
9. 异常值：由于偶然原因导致的值偏离其正常分布范围。
10. 正态分布：一种广泛使用的概率密度函数。

## 2.3 Python 的基本语法
1. 打印输出：print() 函数用于向屏幕输出字符串。
2. if-else 语句：if-else 可以用来执行条件判断和分支流程。
3. for 循环：for 循环可以重复执行一个固定次数的操作。
4. while 循环：while 循环可以重复执行直到指定的条件为假。
5. 函数定义：def 语句用于创建函数，其后的字符串为函数名。
6. 列表推导式：列表推导式可以快速生成列表。
7. 字典推导式：字典推导式可以快速生成字典。
8. 集合推导式：集合推导式可以快速生成集合。
9. 文件操作：Python 提供了丰富的文件读写操作方法。

## 2.4 数据科学工具包
1. NumPy：是 Python 中用于进行科学计算的一项基础软件库。
2. Pandas：是基于NumPy构建的数据分析工具包。
3. Matplotlib：是Python中的绘图工具包，可用于制作各式各样的图表。
4. Seaborn：是基于Matplotlib开发的一款可视化库，提供了更美观的统计图表。
5. Scikit-learn：是基于 NumPy 和 SciPy 的 Python 机器学习库。
6. Statsmodels：是基于 SciPy 的 Python 统计学库，提供了许多统计模型和诊断测试。
