# 2.基本概念术语说明
## 2.1 什么是“跨语言学习”？
“跨语言学习”(Interlanguage learning) 是一种能力，它可以帮助人们解决日益增长的多语言学习需求，包括日常生活中的用语、语法、逻辑、社会交流等，帮助外籍学生能够顺利融入当地社群，并更好地适应当地环境。

“跨语言学习”的核心是能够在两种不同语言之间建立联系和交流。如何理解“跨语言学习”能力：“跨语言学习”是指能够使用不同的语言与他人交流，能够理解、使用不同语言的人类语言，能够翻译出英文、法文、德文、日文、韩文等，能够阅读、理解、表达另一种语言的人类语言。因此，通过“跨语言学习”，人们可以从另一种语言中获得新知识、技能、经验。

## 2.2 为何要“跨语言学习”？
由于互联网技术的普及、海量信息的呈现，人们越来越容易接触到全球各地的人，这些国家的语言也越来越多。而世界各国之间的差异使得学英语成为一件比较困难的事情，这就使得很多人望而却步。所以，想要了解另一种语言的人类语言，并翻译成自己的母语，是成为一名优秀的语言学习者的一项基本要求。

另一方面，当今时代，各种类型的新闻、科技产业、商业模式、政治观点、甚至法律条款的出现都要求各国语言之间相互了解，这也促使越来越多的企业和组织都采用跨语言交流的方式。那么，“跨语言学习”究竟有什么样的价值呢？

## 2.3 什么是“跨语言学习”的目标？
“跨语言学习”是为了更好的融入当地社会、更高效地适应当地环境。从这个角度看，目标应该是让学习者能够准确、正确地理解另一种语言的人类语言。除此之外，“跨语言学习”还可以帮助学习者提升沟通、口语表达、写作能力，因为这些能力都依赖于语言。

## 2.4 “跨语言学习”的方法和工具有哪些？
“跨语言学习”的方法和工具主要分为以下几种：

1. 通过翻译工具：将另一种语言的人类语言翻译为英语、法语、德语、日语或其他语言。这是一种简单直观的跨语言学习方式，只要有合适的翻译工具即可实现。

2. 通过阅读材料：通过阅读材料和词典的同步学习来提升学习效率。这一方式倾向于掌握一定的母语和至少一门第二语言。但是，在学习时间长、内容丰富的情况下，这一方式也可能会造成学习效率下降。

3. 通过面试模拟考试：在某个国家举行的面试中提问另一种语言的人类语言。这种方法可以在短时间内提升自我介绍的能力、口语表达的能力，而且有助于加深自己对该语言的了解。

4. 参加官方培训课程：通过官方的专业培训课程，学习者可以跟着授课者的手把手教学，这也是一种不错的选择。但首先，培训机构应该事先对学习者的受教育程度做出评估，确保授课者和学习者的水平匹配。其次，培训课程的内容一般要和学习者的实际情况相符，这样才能真正激发学习者的学习兴趣。

5. 通过在线教育平台：越来越多的在线教育平台提供了越来越多的跨语言学习资源，学习者可以利用这些平台来提升学习能力。例如，Coursera、Udacity、Udemy等都是非常知名的跨语言学习网站。但是，要注意的是，选择一个好的平台也是需要付出努力的。
