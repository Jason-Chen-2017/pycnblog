## 4. 具体最佳实践：代码实例和详细解释说明
这里我们以法律文书自动生成为例,给出一个简单的代码实现:

```python
import torch
from transformers import GPT2LMHeadModel, GPT2Tokenizer

# 加载预训练模型和tokenizer
model = GPT2LMHeadModel.from_pretrained('gpt2')
tokenizer = GPT2Tokenizer.from_pretrained('gpt2')

# 输入用户需求
prompt = "起草一份买卖合同,交易标的是一辆二手汽车..."

# 生成文书初稿
input_ids = tokenizer.encode(prompt, return_tensors='pt')
output = model.generate(input_ids, max_length=1024, num_return_sequences=1, do_sample=True, top_k=50, top_p=0.95, num_iterations=50)

# 输出生成的文书
generated_text = tokenizer.decode(output[0], skip_special_tokens=True)
print(generated_text)
```

在这个示例中,我们利用预训练的GPT-2模型,根据用户提供的需求信息自动生成了一份初步的买卖合同文书。模型通过学习大量历史合同范本,能够生成符合法律格式和惯例的文本内容。当然,在实际应用中还需要人工审核和完善,但这种自动生成技术已经大大提高了文书撰写的效率。
