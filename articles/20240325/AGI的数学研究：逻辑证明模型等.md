### 4.1 演绎推理实现
我们可以使用Python的SymPy库实现一个简单的演绎推理系统。首先定义一些命题变量和蕴含关系:

```python
from sympy import * 

A, B, C = symbols('A B C')
premises = [Implies(A, B), Implies(B, C)]
```

然后使用自然推理系统进行推理:

```python
from sympy.logic.algorithms.dpll import dpll_satisfiable

model = dpll_satisfiable(And(*premises))
if model:
    print(f"If A, then B; if B, then C. Therefore, if A, then C: {model}")
else:
    print("The premises are inconsistent.")
```

这段代码演示了如何使用数学逻辑工具进行演绎推理。

### 4.2 模糊推理实现
我们可以使用Python的Fuzzy库实现一个简单的模糊推理系统。首先定义模糊变量和隶属函数:

```python
import numpy as np
from fuzzy.membership.membership import Membership
from fuzzy.defuzzify.defuzz import Defuzz

temperature = Membership('temperature', np.arange(0, 101, 1))
temperature['cold'] = Membership.trimf(temperature.universe, [0, 15, 30])
temperature['warm'] = Membership.trimf(temperature.universe, [20, 35, 50])
temperature['hot'] = Membership.trimf(temperature.universe, [40, 60, 100])
```

然后定义模糊规则并进行推理:

```python
from fuzzy.control.controlsystem import ControlSystem
from fuzzy.control.membership import fuzz

control_system = ControlSystem([
    Rule(temperature['cold'], temperature['warm']),
    Rule(temperature['warm'], temperature['hot'])
])

simulation = control_system.control(25)
print(f"The temperature is {Defuzz.defuzz(simulation, temperature):.2f} degrees.")
```

这段代码演示了如何使用模糊集合论工具进行模糊推理。
