# 1.简介
         

## 1.1 什么是深度学习
深度学习(Deep Learning)是一种利用多层次人工神经网络来提取数据特征并且进行分类、预测或回归的机器学习方法。

深度学习可以理解成是多层神经网络的组合，每层由多个神经元组成，每个神经元负责提取输入特征并通过激活函数进行输出响应，最后通过堆叠多层神经网络，就可以解决复杂的问题。这种高度非线性的结构能够捕获数据的局部和全局信息，并且在训练过程中自动进行权重调整，使得模型不断优化，最终达到很高的准确率。

## 1.2 为什么要用深度学习？
传统机器学习方法的缺陷在于无法有效地处理高维、非线性的数据，因此受到越来越多研究人员的关注。而深度学习却可以在有效降低手工特征工程的情况下，直接学习到数据本身的特性，从而取得更好的效果。

深度学习还有很多优点，比如：

1. 模型参数少，即便面对庞大的样本量也只需要很少的参数就可完成模型训练；
2. 适用于各类型任务，包括分类、预测、回归等；
3. 通过反向传播算法实现端到端训练，不需要中间特征，具有自我监督学习能力；
4. 提供了多种优化算法选择，能够有效防止过拟合现象；
5. 在特定领域还取得了比较好的成果，如图像识别、自然语言处理、推荐系统等。

## 1.3 深度学习的应用场景
深度学习目前主要有以下几类应用场景:

1. 图像识别：深度学习在图像识别方面的应用最为广泛。可以识别出图片中存在的物体及其位置，如门牌号码、车辆牌照、道路标志等，以帮助安全部门及时发现潜在的违法犯罪行为。
2. 自然语言处理：深度学习在自然语言处理方面也取得了丰硕的成果。可以处理海量文本数据，对话系统、搜索引擎、机器翻译、聊天机器人等都有着深厚的技术底蕴。
3. 智能助手：基于深度学习的智能助手产品，如Siri、Google Now、小冰等都将在日益增长的市场份额下获得惊喜的发展。
4. 金融分析：智能投顾机器人的发展正在拓展金融分析的边界。当AI能够帮助个人选择证券投资时，投资者将更有可能采取更加理智的投资策略，从而获得更高收益。
5. 数据挖掘：在互联网行业，深度学习被应用于数据挖掘，解决了大数据的挖掘难题。

当然，未来深度学习技术也会带来新的应用领域，如跨视角学习、无监督学习、生成式模型等。这些新技术将极大地拓宽深度学习的应用场景。

# 2.基本概念术语说明
## 2.1 深度学习的基本概念
深度学习的基本概念分为四个方面：

1. 神经网络(Neural Network): 神经网络由多个神经元相互连接构成，神经元接受输入信号，并给予输出响应。每个神经元的输入来自前一层神经元的输出，并传递到后一层神经元。

2. 误差反向传播(Backpropagation algorithm): 误差反向传播算法是指在神经网络的训练过程中，根据误差（实际值与预测值的差距）计算梯度，然后沿着梯度方向改变神经元的权重，以减小误差。

3. 代价函数(Cost function): 代价函数用来衡量神经网络的预测值与真实值之间的差距。一般来说，代价函数越小，代表预测结果越准确。

4. 优化器(Optimizer): 优化器用来迭代更新神经网络中的权重，使得代价函数最小化。有一些常用的优化器如随机梯度下降法(SGD)、动量法(Momentum)、Nesterov动量法(NAG)、AdaGrad、RMSprop、Adam。
