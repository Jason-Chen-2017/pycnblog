# 1.简介
         

CryptoKitties是由区块链科技公司Sorare出品的一款基于ERC721标准的去中心化游戏。该游戏最初于2017年发布，它旨在将用户带入虚拟世界中，并控制属于自己的数字宠物。由于其简单易玩、免费提供服务等特点，近两年来已经成为热门话题。现在越来越多的人加入了这个热潮中，希望拥有一个属于自己的数字宠物。而CryptoKitties项目也正是为了解决这个痛点而诞生。

CryptoKitties的目的是建立一个去中心化、不可篡改且所有权确定的数字宠物收藏平台。其最主要功能之一就是能够让用户进行ERC-721协议上的数字宠物的创建、销毁、收购、交易等操作。用户可以自由地通过自己的区块链钱包上传自己喜欢的小动物照片，并与其他用户进行交易。此外，还可以设定价格和时限等限制条件，甚至可以让用户进行二次元表白等活动。CryptoKitties将整个过程都记录在区块链上，让所有相关方都能够验证和验证交易信息，从而实现价值和流通上的完全保密性。

# 2. 核心概念及术语
## 2.1 NFT（Non-fungible Token）
NFT，Non-Fungible Token的缩写，翻译过来就是不可能互换的非同质化代币。它的特点是：

1. 不可互换的。这意味着一张图片和另一张图片就算是不同图片，它们仍然是一样的NFT。也就是说，每张NFT只能有一个主角，而且所有的NFT都是独一无二的。
2. 非同质化的。这是指NFT中的每张图片都不是真实的实体物品。图片中的小动物、文字或者数字都不能与真实的物品相提并论。
3. 有价值的。其中的价值是可以被衡量的，无论是数量还是质量。

与此同时，NFT还是一种很有吸引力的类别，因为它可以赋予用户“非同质化”的身份，使得用户产生“动物守望者”、“美食爱好者”、“艺术家”、“环游世界”等新的认知，以及“买卖”、“创造”、“赚钱”等新的行动方式。

## 2.2 ERC721（Ethereum Request for Comment 721）
ERC，Ethereum Request for Comment的缩写，翻译过来就是以太坊社区征求意见稿。其中，ERC721是一个用来定义如何标准化创建、分发和管理非同质化代币的标准。该标准定义了哪些合约接口需要实现，以及它们之间的关系。比如，ERC721标准规定了如何创建一个NFT，以及如何使用不同的方法来管理这些NFT。

ERC721的核心接口包括三个：

1. approveAndCall()：允许另一个地址来执行某个函数。
2. balanceOf(): 查询某个账户持有的NFT数量。
3. ownerOf(): 查询某张NFT的所有者的地址。

## 2.3 MINTING（铸造）
MINTING，是指用户可以在市场上自由创建的过程。当用户在市场上发布某种NFT的时候，他实际上是在向系统申请创建某些东西。这种操作称为铸造。

## 2.4 TOKEN ID（代币ID）
TOKEN ID，Token Identifier的缩写，翻译过来就是代币标识符。它是一个唯一标识符，用于代表某一特定的NFT。

## 2.5 METADATA（元数据）
METADATA，Metadata的缩写，翻译过来就是数据的元数据。它是关于数据的数据，包含了诸如作者、日期、描述、分类标签、版权信息等数据。

## 2.6 ERC20（Ethereum Request for Comments 20）
ERC20，Ethereum Request for Comments 20的缩写，翻译过来就是以太坊社区征求意见稿20号。其中，ERC20是Ethereum协议的基础设施，其目的就是为智能合约开发者提供一套统一的接口。

ERC20的主要特性包括：

1. 支持多个 token 类型。
2. 按顺序分配 token 标识符。
3. 每个 token 可以拥有自己的名称、符号。
4. 智能合约开发者可以根据需要自定义 token 的属性。
5. 智能合约具有可靠的发行渠道。

## 2.7 CIRCLE TOKEN (CK)
Circle Token(CK)，CircleKoin的缩写，即环球币。它是CryptoKitties系列游戏的基础代币，在游戏中扮演着重要作用，包括：

1. CK的最小单位，即CK令牌。
2. CK的总供应量，决定了游戏世界的繁荣和衰落。
3. CK的初始发行量。
4. CK的铸造成本。
5. CK的汇率。
6. CK的治理机制。

## 2.8 CURRENCY SYSTEM
CURRENCY SYSTEM，即游戏的货币系统。它是一个类似支付宝、微信支付等的服务。玩家可以通过CK令牌或ETH来购买游戏内的道具。玩家也可以兑换为其他代币，例如NFTs、手续费等。

## 2.9 STOLEN ITEMS PENALTY
STOLEN ITEMS PENALTY，即被偷走的道具惩罚机制。它是为了保护游戏中的互利关系，防止因为恶意举报导致的各种问题。如果玩家发现自己的物品被偷走，可以向平台举报，平台会给予相应惩罚。

# 3. 操作步骤与原理
## 3.1 新人的开场白
大家都很熟悉nft的规则了吧？不用急着在意，我只想跟大家分享一下，游戏中那些实实在在的小动物。

首先，我们要知道什么是NFT。NFT是指可增值数字资产，而CryptoKitties就是一个发行了多个NFT的游戏。NFT的本质是基于以太坊区块链技术的去中心化数字资产，其具有不可替代、真实存在且无法复制的特征。

其次，游戏的机制。玩家可以自由创建自己的小动物NFT，并且可以在市场上进行交易。游戏的物品具有版权保护，玩家在游戏中的表现不会受到任何商业利益的影响。

最后，再来看看玩家怎么玩CryptoKitties。首先，玩家必须拥有加密货币，这里推荐使用MetaMask插件。然后，登录网站后，就可以开始创建自己的小动物了。具体操作如下：

1. 点击左侧菜单中的"Owned Creatures"按钮，进入已拥有的小动物列表页面。
2. 点击右上角的加号图标，跳转到创建小动物的页面。
3. 在左侧的输入框中上传自己的小动物照片，并填写其他相关的信息。
4. 点击下方的"Create"按钮，小动物NFT就生成啦！
5. 选择你的小动物NFT，点击右边的"Purchase"按钮即可购买。
6. 选择自己的NFT想要花掉多少CK令牌，点击“Confirm Purchase”按钮完成购买。

另外，为了防止恶意举报，游戏还设计了盗取小动物NFT的惩罚机制。玩家需要在每个月的最后一天向平台举报自己的小动物NFT被偷走。平台审核后，平台可以判定玩家是否构成盗窃的犯罪行为，并向该玩家处以一定的法律责任。

## 3.2 创建流程
### 3.2.1 创建你的第一个NFT
如果你是新玩家，那你肯定会感到很惊讶，这个游戏竟然可以创建自己的NFT。那么，相信随着你的了解，你一定会喜欢上这个游戏。

首先，你需要拥有一个支持ERC721标准的钱包，比如MetaMask或者火币钱包。

然后，打开CryptoKitties官方网站，点击左侧菜单栏中的'create creature'按钮，进入创建小动物NFT的页面。

在该页面中，你可以上传自己的小动物照片，并添加一些相关信息。当你准备好了之后，点击底部的'create'按钮，你的小动物NFT就会生成啦。


### 3.2.2 设置价格
既然小动物是NFT，那么自然是要设置价格的。你需要记住，你花的CK不是传统意义上的货币，是CK币，CK币其实是CryptoKitties系统中的一种虚拟代币。


CK是一个最小的单位，在游戏过程中，CK的数量是重要的衡量标准，比如，CK令牌的数量就表明了玩家拥有的小动物数量。

当然，你也可以选择设置一个期限，不过这不是必需的，你可以把期限设置为永久。

### 3.2.3 上传照片
CryptoKitties游戏的玩法之一就是自由上传照片。你可以把自己喜欢的小动物放进这个画廊里，也可以上传自己的绘画作品。

所以，请务必上传高清照片，尽量不要上传过大的照片，因为这样会影响小动物的显示效果。

### 3.2.4 分享你的NFT
玩完这个游戏后，你可能会觉得非常开心，但同时也会想问，是不是可以把我的NFT分享给朋友呢？

当然可以啦！你只需要把NFT的链接复制给你的朋友，他们就可以查看到你的小动物了。

### 3.2.5 交易NFT
另一件你可能经常做的事情就是交易NFT。与其它玩家交换自己的小动物。

交易之前，双方必须先确认好自己的交易细节。确认完成后，双方才能进行交易。


### 3.2.6 查看我的NFT
刚才我们讲到了如何上传照片，并且生成自己的NFT，那么接下来我们来讲一下如何查看我们的NFT。

首先，你需要有一个支持ERC721标准的钱包，比如MetaMask或者火币钱包。

然后，打开CryptoKitties官方网站，点击左侧菜单栏中的'owned creatures'按钮，进入已拥有的小动物列表页面。

在这个页面中，你可以看到你的所有NFT。你可以点击任一NFT，查看其详情，修改描述等信息。


### 3.2.7 收藏我的NFT
玩完这个游戏后，你可能有很多收集到很多NFT的冲动。那么，我建议你继续收集更多NFT。

收集NFT的途径有很多，你可以选择在各个网站上传，也可以直接下载到手机或电脑，扫描二维码进行交易。所以，玩起来，收集，再玩，积累。

# 4. NFT与数字货币的区别与联系
## 4.1 二者概括
NFT与数字货币之间最显著的区别就是两者存储的方式。数字货币是中心化的系统，所有操作都由银行来处理；NFT则是去中心化的系统，所有操作都由参与者进行协商和执行。

## 4.2 功能区别
数字货币是支付系统，是用来支付的；NFT是游戏中的一种商品，可以作为交换手段。与数字货币相比，NFT更擅长承载价值和流通。

例如，数字货币是用来支付的，它可以用于购买商品、贷款、交易等。但是，NFT不能单独用于交易，只能用于游戏中的交换。

另一方面，数字货币并不能保证用户的安全。在充值之前，用户必须进行KYC（Know Your Customer）核查。NFT提供了一种新型的商业模式——自由价值交换。虽然用户可以自由分享自己的NFT，但是它是一种价值交换，而不是商品的交换。

## 4.3 属性区别
数字货币一般有价值、流通和匿名三大属性，NFT除了拥有这些属性，还有以下几个特点：

1. 拥有权。在NFT市场上，玩家可以自由定价、出售、买卖，而且是真正的自由交易。
2. 可验证。NFT市场上可以证明某些数字资产的真实性，从而避免欺诈。
3. 可溯源。NFT的所有者可以追溯资金流向。
4. 可信赖。只有经过认证的第三方可以发行NFT，否则发行失败。
5. 时尚。以3D模型、动漫形象、真人模特等形式表现自己独一无二的造物。
6. 去中心化。NFT市场由各个玩家自己运营。

## 4.4 应用场景
NFT的发行可以归纳为两个主要领域：游戏内兑换、市场交易。

游戏内兑换。对于游戏玩家来说，使用NFT可以让他们轻松地在游戏中获得数字资产。

例如，在 CryptoKitties 中，玩家可以得到不同等级的机器人和卡通小动物。他们可以使用 CryptoKitties Wallet 来购买这些资产。

市场交易。NFT的市场交易体验与其它数字资产不同。尽管玩家可以自由交易，但是却无法使用数字货币作为交易媒介。

例如，交易 CryptoKitties 小动物不仅依赖于货币，还依赖于信用卡支付。此外，在线游戏通常有垄断竞争的嫌疑，NFT的发行可以缓解这一难题。

# 5. 发展前景及关键技术路线
## 5.1 发展前景
CryptoKitties项目的起点就很神秘，它几乎没有任何官方消息，甚至连它的创始人Jack也不清楚。不过，最近它在波士顿举办了首届全球黑客马拉松，估计会推出一个系列。

CryptoKitties项目已经接触了大量投资者，项目融资已经达到了6亿美元，目前估计超过700万美元。

CryptoKitties的大胆尝试似乎还有它的身影，它的项目管理方面还处于起步阶段，这也许能给项目创始人带来一些启示。

目前来看，CryptoKitties的研究者们还处于搭建这个项目的阶段，有很多问题需要解决，比如如何更好的激励用户，如何提升游戏效率，如何减少盗窃的风险，如何确保数据的有效性等等。

## 5.2 技术路线
CryptoKitties项目目前还处于搭建阶段，最主要的问题就是如何定义一个NFT。目前的标准是ERC721，里面有几个核心接口：

1. approveAndCall()：允许另一个地址来执行某个函数。
2. balanceOf(): 查询某个账户持有的NFT数量。
3. ownerOf(): 查询某张NFT的所有者的地址。

ERC721定义了NFT的主要功能，但是并没有对NFT的结构、功能、操作指南进行详细说明。

除此之外，CryptoKitties项目还涉及的技术栈还有Solidity语言、IPFS文件存储、以太坊网络等，还有未来的云计算方案等。

另外，CryptoKitties项目在定义NFT的标准上还存在一些疑问，比如：为什么需要ownerOf()函数？这些问题都还在探索当中。
