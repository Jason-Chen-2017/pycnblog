# 2.数据可视化的基本概念和术语
## 2.1 数据集
数据集（Dataset）是一个用来描述数据特征集合的统称。一般情况下，数据集包括两个要素：变量（Variable）和值（Value）。例如，我们有一个销售数据集，其中包含销售额（Variable），顾客年龄、性别、消费品种等多个维度（Variable），每个顾客对应的销售额就是其值（Value）。同样地，一个股票价格的数据集中，包含股票的日期、开盘价、收盘价、最低价、最高价等多个维度，每个交易日对应的收盘价就是其值。数据集具有以下几个重要属性：
- 有序性：数据集中的各个值按顺序排列，且每个值的位置唯一确定。
- 可重复性：数据集中的每组值都可以出现多次。
- 可变性：数据集随着时间的推移，新的数据加入或者旧的数据更新都会改变其结构和内容。

## 2.2 属性和变量
属性（Attribute）、变量（Variable）和维度（Dimension）都是用来表示数据的不同角度或维度的词汇。属性可以看成是变量的子集，而且两者之间存在一对一的对应关系。比如，一个学生数据集包含姓名、性别、年龄等信息，其中姓名、性别和年龄是属性；而学生的名字、性别和年龄是变量。当然，属性也可以扩展成包含多维度的，如某个国家的气候，它的三个维度分别是年份、月份和气候类型，它们也是属性。

变量（Variable）是指数据集中的某个具体统计学上的量，它可以是连续的也可以是离散的。连续变量通常表示实数值，例如身高、体重、价格等；离散变量通常表示分类值，例如学历、职业、商品类别等。

## 2.3 统计方法
统计方法（Statistical Method）是用于分析数据集的统计计算和推理过程。统计方法分为四类：
- 频率统计：主要用于描述数据在整个数据集中的分布情况。比如，某个城市的人口分布，就可以使用频率统计的方法，统计出城市中不同年龄段的人数。
- 聚合统计：主要用于描述数据在不同维度上的变化趋势。比如，某个商品的销售总额随着时间的变化情况，就可以使用聚合统计的方法，画出折线图。
- 分布统计：主要用于描述变量的概率分布情况。比如，人们常常喜欢用正态分布表示数据，因此可以使用分布统计的方法验证某些假设。
- 回归分析：主要用于描述两个或多个变量间的关系。比如，研究不同人群的身高和体重的关系，就可以使用回归分析的方法。

## 2.4 可视化方法
可视化方法（Visualization Method）是一种利用图像、图表、图形来呈现数据的方法。常用的可视化方法有：
- 描述性统计图表：描绘数据的基本统计信息，如直方图、饼状图、条形图等。
- 交互式可视化：使用鼠标交互的方式查看数据分布，探索数据的潜在关系。
- 地理可视化：利用地理信息，如经纬度坐标，来映射和展示数据。
- 主题映射：根据特定场景，制作定制化的可视化图表。

# 3.Python数据可视化工具
## 3.1 Matplotlib
Matplotlib是一个著名的Python数据可视化库。Matplotlib是python里面的一个第三方绘图库，它提供了一整套类似MATLAB的绘图功能。Matplotlib也支持Windows、Linux、Mac等主流平台。

Matplotlib的主要特点如下：
- 支持多种图表形式：Matplotlib支持常见的各种图表形式，如线图、柱状图、饼图、密度图等。
- 支持自定义图表样式：Matplotlib支持定义颜色、线宽、字体大小、图例位置等自定义选项。
- 支持保存图片：Matplotlib支持直接将图表输出到文件，如PDF、PNG等。
- 支持中文显示：Matplotlib支持中文显示，但需要安装相应的字体库。

## 3.2 Seaborn
Seaborn是一个基于Matplotlib库的Python数据可视化库，它提供了更高级的图表形式，如汇总统计图、线性回归图等。Seaborn具有以下优点：
- 使用简单：Seaborn提供了一系列便捷函数，可以快速生成高质量的图表。
- 提供更多效果：Seaborn还提供了更多高级图表形式，如热力图、空间可视化等。
- 更漂亮的默认设置：Seaborn内置了一套漂亮的默认设置，使得生成的图表更加美观、直观。

## 3.3 Plotly
Plotly是一个基于Web的开源可视化库，它提供交互式图表，同时也能直接嵌入网页。Plotly提供了丰富的图表类型，如散点图、曲线图、柱状图等。Plotly提供了许多高级特性，如动画、子图等。
