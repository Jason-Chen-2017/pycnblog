# 1.简介
         
1968年,约翰·塞缪尔·高斯（<NAME>）提出了著名的广义线性模型（general linear model），这是一种统计方法，主要用于对随机变量进行建模。现代统计学在数据分析中已经成为一种基本工具，并被广泛应用于各个领域。 
        1987年，李开复发表了基于贝叶斯观点的“贝叶斯线性回归”（Bayesian Linear Regression），其基本思想是建立一个联合概率分布，将数据集看作是依据某种线性模型生成的。 
        1990年，奥本海姆（<NAME>）等人在“基于密度估计的图像分割”（Image Segmentation Based on Density Estimation）方面开创性地提出了基于核的方法。 
         1997年，亚历山大·格罗宁（Arthur Grassberger）提出了“非参数方法”（nonparametric methods）的概念，即数据不假设任何特定分布，而是采用非参数的统计方法对数据进行建模。
         在过去的几十年里，图像处理、生物医疗和其他领域均受到参数化模型的启发，例如傅立叶变换、核方法、混合高斯模型、图形模型、Markov网络等。
          有了这些经典模型之后，为了更好地理解这些模型背后的数学原理，笔者希望通过这篇文章，对参数化模型及其发展方向有一个简单的介绍。 
        # 2.基本概念术语说明
        1. 参数化模型
            参数化模型（Parametric Model）指的是已知数据的分布，定义一组参数，并用这些参数来描述这一分布。参数模型具有优点，比如计算简单，学习效率高，输出结果容易解释。在计算机视觉领域，一般会使用两种参数化模型：一是关于图像结构的模型；二是关于物体位置、大小和形状的模型。 
            比如对于一张图片，可以使用像素值构成的直方图作为图像结构的模型，通过各类统计量和特征描述图像的各种分布，再结合某些约束条件，最终确定一些参数，比如最亮的颜色、暗处的饱和度，直方图的峰值位置等。然后，在图像上找出物体的位置、大小、形状时，就可以使用这些参数化模型来做预测。
            另一种参数化模型就是多项式拟合（Polynomial Fitting）。该方法将每个点按照一定顺序进行排序，通过一条曲线来近似表达这条曲线上的点，这种方法可以得到较为精确的曲线拟合，也比较适合拟合单调的曲线。
            2. 非参数模型
            非参数模型（Non-parametric Model）是一种在没有对数据分布做任何假设前提下，对数据进行建模的方法。与参数化模型相比，非参数模型有很大的自由度，能够捕获不同的数据模式，但是由于需要对整个数据集进行建模，因此计算量较大，且无法给出具体的输出结果。
             比如对于一张图片，可以使用相邻像素之间差值的平方和来进行图像结构建模，而后续的分类任务则可以使用K-NN方法，K=1或者K=3。此外，还有基于密度的连通组件分割、基于树结构的图像分割、以及基于神经网络的图像识别等。 
             除了上述两种参数化模型和非参数模型，还有很多其他的参数化模型和非参数模型，如基于高斯过程的状态空间模型、Markov随机场、混合模型、深度学习模型等。
          3. 泛函逼近
             泛函逼近（Functional Approximation）是指利用函数的形式，来近似某个复杂系统的行为，并在此基础上做预测或控制。它可以非常有效地刻画复杂系统的物理规律、经济学中的物价决策机制、优化问题的求解方式、以及现实世界的复杂系统，都可以运用泛函逼近来进行研究。 
          4. 混合模型
             混合模型（Mixture Model）是一种基于贝叶斯定理，将多个参数化模型的输出结果合并起来得到的新模型。它可以有效地解决当样本数量不足时，仍然可以对未知数据进行概率密度估计的问题。如在图像分割中，可以先使用多个区域检测算法检测出图像中的多个区域，然后用多元高斯混合模型对每个区域的信号进行建模，最后根据不同区域的混合程度，对图像进行划分。 
          5. 聚类
             聚类（Clustering）是一种无监督学习方法，将相似的对象归为一类，并且对每一类的对象赋予一个中心点。它可以帮助找到隐藏在数据中的模式，或实现数据降维、数据可视化等。
            总之，参数化模型与非参数模型是两种重要的统计学习方法，它们有着不同的优缺点，适用于不同的场景。参数化模型较为直接、准确，但是计算量大，难以应付复杂的场景；非参数模型则可以利用更多信息，但计算量小，但可能失去部分信息。泛函逼近、混合模型、聚类是三个相互联系的统计学习方法，它们共同解决数据分析中遇到的很多问题。
