# 1.简介
         

随着智能家居的普及和发展，家庭成为全方位生活的关键节点之一，但也越来越成为攻击者面临的主要目标。近年来，不断涌现的安全威胁让人们对智能家居更加依赖，而对于个人的安全保障也变得尤为重要。智能家居安全问题一直是社会关切的焦点，也是行业的关注热点，国内外已经出现了许多相关报道。
# 2.背景介绍
在这个时代，“合法”的物联网设备日益流行，各种智能家居产品层出不穷。在过去的一年里，欧洲、中国和美国等地区共发布了超过100款智能家居产品。安全是每一个家庭不可或缺的因素。如何保证自己的家庭及周边安全是许多家庭考虑的问题。尽管安全问题会影响到所有的家庭成员，但是能够将安全问题准确有效地监测并通知，可以极大的减少暴露风险，帮助用户更好地保护自己和家人。因此，基于IoT（物联网）的个人化安全通知系统正成为解决这一难题的有效方案。
# 3.基本概念术语说明
## 概念
1. 物联网（IoT）：物联网是由不同物体、传感器、终端等设备相互连接，从而形成的网络。它使物理世界中的各种设备能够被联结、交换数据信息，并通过云计算的方式进行整合管理。
2. 智能家居：智能家居是指采用数字技术实现的智能家具和环境自动化监控设备。包括智能摄像头、语音助手、微波炉、空调、洗衣机、电梯、电视盒子、电气开关等。
3. 智能防火墙：智能防火墙是一种基于智能终端的网络安全产品。它可以通过互联网、移动通信、宽带等方式收集各类安全信息，并实时分析处理，提升网络隐私、可用性、安全性。
4. 智能门锁：智能门锁是一种基于人工智能（AI）的门禁系统。它能在检测到异常行为时主动向用户发送警报消息，通过提醒用户保持在家中，保护其财产安全。
5. 加密技术：加密技术通过对称加密、非对称加密、哈希算法等方式对数据进行编码，以达到数据安全、数据共享的目的。
6. HTTPS：HTTPS (Hypertext Transfer Protocol Secure) 是一种利用SSL/TLS协议进行安全通信的网页服务器安全通信协议。
7. 用户认证：用户身份验证是指对用户访问系统时的身份进行核验，以判断是否具有权限对该系统进行访问。
8. RESTful API：RESTful API 即Representational State Transfer 的API。它是一组标准的接口规范，旨在提供互联网应用从客户端到服务器端的通信接口。RESTful API 可以简单且快速地集成到现有的Web应用程序中。
9. MQTT：MQTT（Message Queuing Telemetry Transport）是一个轻量级的发布/订阅型消息传输协议。它是基于TCP/IP协议的发布/订阅消息传输协议，适用于高负载的网络环境。
10. OAuth：OAuth（Open Authorization）是一个开放授权标准。它定义了一种标准协议，允许第三方应用获得有限的权限，而不需要向资源所有者提供用户名和密码。
11. JWT：JWT（JSON Web Token）是JSON对象，它定义了一个紧凑且自包含的方法用于安全地 transmitting JSON objects between two parties as a JSON object.
12. 小米推送服务：小米推送服务是MIUI开发的一个内部消息推送服务。它提供高度可靠、低延迟的消息推送功能，适用于大多数App场景下的消息推送需求。
13. 物联网安全漏洞：物联网安全漏洞是指由于产品设计、开发过程中的缺陷或代码实现错误导致的安全风险。物联网安全漏洞分为两大类型——硬件安全漏洞和软件安全漏洞。
14. 硬件安全漏洞：硬件安全漏洞是指物联网设备本身存在的安全问题。例如：恶意的固件版本、敏感信息泄露、易受攻击的接口等。
15. 软件安全漏洞：软件安全漏洞是指物联网设备的软件存在的安全问题。例如：漏洞扫描程序存在误报、恶意访问、后门漏洞等。
16. 中间人攻击：中间人攻击又称为Man in the Middle Attack，是指攻击者截获了客户端与服务器之间的通讯内容，并在不知情的情况下，篡改或者拦截了双方正常的通讯。
17. 物联网系统安全：物联网系统安全是指物联网系统所采用的技术、设备、网络结构、系统管理人员等所产生的可能的风险，包括恶意攻击、恶意控制、病毒感染、入侵检测和应急响应、信息泄露、认证和授权、可用性问题等。
18. DDOS攻击：DDoS（Distributed Denial of Service）攻击是指通过大量的网络请求淹没服务器，致使服务器资源耗尽而无法提供正常的服务。
19. DNS劫持：DNS劫持（域名服务器缓存劫持），是指黑客通过将目标网站的解析权转移至他人手中，篡改DNS记录，引导用户访问他人站点，通常通过伪造易受攻击的域名来实现。
20. SQL注入攻击：SQL注入攻击，也叫做数据库攻击，是指攻击者通过构造恶意SQL查询语句插入或删除数据，获取数据存储在数据库中的敏感信息。
21. 垃圾邮件：垃圾邮件是指通过各种形式发送的无用或者骚扰信息，它们往往包含病毒或者恶意链接，通过垃圾邮件往往导致用户投诉、纠纷甚至损失。
22. 命令注入攻击：命令注入攻击，也称为恶意代码攻击，是指攻击者通过插入恶意的代码段，破坏或修改服务器上的程序，执行任意命令，获取操作系统权限。
23. HTTP协议漏洞：HTTP协议漏洞是指HTTP协议存在的一些安全性问题，如：Cookie中可能含有敏感信息；HTTP重定向可能会泄露用户信息；缺乏加密措施等。
24. 漏洞扫描工具：漏洞扫描工具（Vulnerability Scanner）是用来检测和识别网络中计算机系统、网络设备、服务器上的安全漏洞的应用程序或工具。常见的漏洞扫描工具有Nessus、Nmap、openvas等。
## 术语
1. 主动防御：主动防御是指通过预设的规则、策略等手段，对攻击行为进行预防，避免发生故障。
2. 反爬虫机制：反爬虫机制是指通过加入验证码、限制请求频率等方式，降低爬虫的搜索量，防止被封禁。
3. 数据脱敏：数据脱敏是指对原始数据进行匿名化处理，或者按照用户的权限设置，只显示必要的数据，进一步保护用户的隐私。
4. IP封锁：IP封锁是指通过封禁某些IP地址来限制网络上特定类型的访问，保护网络的安全和合法权利。
5. 隐私保护：隐私保护是指保护用户的个人信息不被泄露、滥用和篡改。
6. 勒索软件：勒索软件是指通过诈骗、贩卖等方式对计算机、手机等设备进行盗窃、故意传播的恶意软件。
7. 黑客攻击：黑客攻击是指攻击者通过入侵系统、盗取数据、修改数据、破坏网络、控制服务器等方式，获取个人信息、金钱、甚至生命危险。
8. 访问控制：访问控制是指对用户访问系统的权限进行控制，比如登录密码是否复杂、密码存留期限、登录尝试次数限制等。
9. 服务质量：服务质量（Quality of Service，QoS）是指通过提高网络可用性、资源利用率、控制数据流量等方式，实现网络的性能优化，提升系统的整体性能水平。
10. 入侵检测系统：入侵检测系统（Intrusion Detection System，IDS）是一种特殊的网络安全设备，它能够实时检测和跟踪计算机系统、网络设备和网络的运行状态，识别和记录发生的网络攻击事件，并向管理员、系统管理员或其他指定的人员发送报警信息。
11. Web应用安全：Web应用安全是指Web应用程序（Web App）的安全设计、开发、测试、运维及部署等工作。
12. DMARC：DMARC（Domain-based Message Authentication, Reporting & Conformance）是一种验证邮件域是否属于某个域名的所有权人的协议。
13. XSS跨站脚本攻击：XSS跨站脚本攻击，也叫做CSSI（Client-Side Script Injection），是指攻击者通过向Web页面中插入恶意JavaScript代码，并通过浏览器的过滤机制绕过后台安全防徢措施，篡改页面的内容，获取用户信息甚至进行钓鱼等攻击手段。
14. SSL证书：SSL证书（Secure Socket Layer Certificate）是一种电子商务证明文件，用于验证商家或组织的电子交易环境是否合法，确认电子交易的真实性。
15. 审计日志：审计日志（Audit Logs）是指记录系统运行情况和活动日志，可以反映系统的安全状态，并提供有效的事后排查依据。
16. 请求响应模型：请求响应模型（Request-Response Model）是指交互式网络通信模型，包括请求和相应两个阶段，在请求阶段，用户向服务器发起请求；在响应阶段，服务器返回请求结果给用户。
17. 事件响应模型：事件响应模型（Event-driven Model）是一种非交互式网络通信模型，基于事件驱动的思想，服务器无需直接响应客户端请求，而是在收到请求事件后触发对应的响应事件。
18. DNS记录：DNS记录（Domain Name System Record）是指域名系统（DNS）服务器上保存的信息，包括A记录、CNAME记录、MX记录、NS记录、TXT记录等。
19. TLS协议：TLS协议（Transport Layer Security）是建立在SSL（Secure Socket Layer）之上的安全协议，它通过密钥交换、身份验证、加密数据等方式，为网络通信提供安全保障。
20. CNAME记录：CNAME记录（Canonical Name Record）是一种类型为别名的DNS记录，它可以把多个名字映射到同一个真实主机。
21. MX记录：MX记录（Mail eXchange Record）是一种类型为邮件交换的DNS记录，它指定某个域名的邮件服务器位置。
22. NS记录：NS记录（Name Server Record）是一种类型为命名服务器的DNS记录，它指定了该域名由哪台服务器提供。
