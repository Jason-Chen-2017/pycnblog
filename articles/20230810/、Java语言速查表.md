# 2.Java语言简介
## Java概述
- “可伸缩的”：Java支持动态内存分配，可以轻松应对不同大小的内存需求；
- “面向对象”：Java是一门面向对象的语言，它支持封装、继承、多态等特性；
- “分布式”：Java支持分布式开发，通过RMI或JAX-RPC可以方便地进行远程服务调用；
- “健壮”：Java有丰富的异常处理机制，使得程序可以很好地适应错误条件；
- “类级编程”：Java是静态类型的语言，需要严格按照数据类型定义变量，编译器就可以确保变量的正确性；
- “虚拟机”：Java虚拟机（JVM）可以运行在各种平台上，并提供安全、可靠的环境；
- “微内核”：Java采用了微内核模式，只包括必要的功能，并通过插件扩展其功能；
- ……
Java是在Sun Microsystems公司推出的一款面向对象、分布式、健壮的类级编程语言，具有跨平台能力。主要用于开发移动应用程序、基于浏览器的Web应用、桌面客户端程序、嵌入式系统应用、游戏引擎等。截至2021年，Java已经被广泛应用于各个领域，并且在云计算、物联网、金融、区块链、人工智能等领域也得到了广泛应用。
## Java版本介绍
目前，OpenJDK和Oracle JDK都获得了长足的发展，OpenJDK是免费、开放源代码、完全向后兼容的OpenJDK项目，并拥有庞大的开源生态系统；而Oracle JDK是Oracle Corporation 收购的商业产品，是Oracle 旗下一款商用的Java开发工具包，由多个组件组成，包括JDK、JavaFX、JRE、Java Web Edition、Java Control Panel、Java HotSpot Client等，包含了Java开发的许多方面的工具，如：Java调试工具JDWP、Java Profiler、JavaBeans、Java Remote Method Invocation（RMI）、Java Architecture for XML Binding（JAXB）、Java Transaction API（JTA）等。
## Java语法和关键字
- class关键字：用来声明一个类、接口或者抽象类的关键字；
- interface关键字：用来声明一个接口的关键字；
- abstract关键字：用来声明抽象类的方法、变量及构造方法的关键字；
- extends关键字：用来实现子类从父类继承的方法、变量及构造方法的关键字；
- implements关键字：用来实现接口的方法、变量及构造方法的关键字；
- import关键字：用来导入一个类的关键字；
- package关键字：用来声明包的关键字；
- this关键字：代表当前对象的引用；
- super关键字：代表父类的引用；
- final关键字：表示不可变的关键字；
- static关键字：表示静态的关键字；
- synchronized关键字：用来同步线程执行的关键字；
- volatile关键字：volatile 可以保证变量在线程之间可见，即一个线程修改了这个变量的值，其他线程立刻能看到修改后的最新值。synchronized 关键字无法做到这一点。volatile 适合解决多线程访问同一个变量时状态不一致的问题，比如 i++ 操作，使用 volatile 修饰 i 之后，保证了每次读取该变量时，都是直接从主存中读取最新的值，而不是从寄存器中读，从而保证数据的一致性。
