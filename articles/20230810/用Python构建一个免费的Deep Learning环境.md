## 2.1 深度学习简介
深度学习（deep learning）是机器学习的分支，也是当前占据主流地位的机器学习技术。深度学习的关键特点就是由多个处理层组成，每一层都可以提取高级特征，并且这些特征之间存在层次关系。深度学习的层次结构使得它可以逐渐从原始输入中提取出抽象的、高度通用的特征表示，并进一步用于预测、分类或回归任务。

目前，深度学习的发展已经取得了一定的成果。2012年Hinton等人的神经网络论文发表，开启了深度学习的新时代，随后吴恩达等人将深度学习推向了更高的境界。目前，深度学习已经成为许多领域的基础性技术，如图像识别、文本分析、语音识别、行为识别等。

## 2.4 scikit-learn深度学习案例

下面，让我们以最简单的线性回归模型——线性回归（Linear Regression）为例，使用scikit-learn库实现一个小例子。

```python
from sklearn import linear_model

# 创建训练集
X = [[1], [2], [3]]
y = [1, 2, 3]

# 创建线性回归模型
regr = linear_model.LinearRegression()

# 拟合模型
regr.fit(X, y)

# 预测结果
print("预测结果:", regr.predict([[4]]))
```

上面代码主要做了三件事情：

1. 导入linear_model模块
2. 创建训练集X和对应的真实值y
3. 创建线性回归模型，拟合训练集

当运行上面的代码，输出的结果为：

```
预测结果: [[3.93799991]]
```

这是因为我们的模型只能拟合一条直线，因此它预测的值会出现一些偏差。如果要拟合的曲线比较复杂，那这个误差就会变得更加严重。不过，即使在这种情况下，线性回归还是非常有用的一个模型。
