# 1.简介
  

云计算（Cloud Computing）是一个广义上的概念，它指通过网络将计算资源、数据存储、应用平台等基础设施服务商、第三方提供商及最终用户联系在一起形成的计算平台。它可以使各种计算资源及相关服务（例如服务器、存储、数据库、网络等）按需、灵活地分配给用户，让用户能够按照自己的需求快速部署、扩容、释放计算资源，从而实现经济高效可靠的IT资源共享。

云计算产品或服务主要分为以下几类：
1) 基础设施即服务（Infrastructure as a Service，IaaS）

2) 平台即服务（Platform as a Service，PaaS）

3) 软件即服务（Software as a Service，SaaS）

4) 服务型软件即服务（Service-Oriented Software as a Service，SOA）

5) 混合云（Hybrid Cloud）

6) 联合云（Federated Cloud）

7) 多云融合（Multi-Cloud Fusion）

8) 数据中心即服务（Data Center as a Service，DCAAS）

9) 机器学习即服务（Machine Learning as a Service，MLaaS）

10) 消息传递即服务（Message Passing as a Service，MPaas）

11) 容器即服务（Container as a Service，CaaS）

本文将逐条分析每个云计算产品或服务，并简要阐述其功能。
# 2.基础设施即服务(IaaS)
## 2.1 AWS
Amazon Web Services（AWS）是一种基于云计算的服务，它提供了大量的基础设施服务，包括EC2、S3、VPC、EBS、RDS、Lightsail、API Gateway、Lambda等等。AWS主要通过其计算、存储、网络、数据库、安全和分析等服务，帮助客户快速构建应用程序，并且免费提供付费服务。AWS为企业提供托管、运维、管理、扩展、迁移、开发等全生命周期管理的一站式解决方案。

### EC2
弹性云计算服务（Elastic Cloud Compute，EC2），为用户提供虚拟化的计算平台，包括CPU、内存、网络、硬盘等计算资源，用户可以使用控制台或者API接口调用进行配置、创建、启动、停止、删除等操作，帮助用户快速部署需要的应用环境，实现资源按需付费、按小时收费、自动伸缩的计算资源管理。

支持运行Linux、Windows、Microsoft SQL Server、Oracle、IBM DB2、MySQL、PostgreSQL、MongoDB等多种主流操作系统，也可以自定义镜像满足复杂的应用场景。可以通过AMI、EBS快照、本地磁盘映像等方式进行存储的管理。

除了EC2外，AWS还提供其他基础设施服务，如：
- VPC：通过VPC，用户可以在一个逻辑隔离的网络空间内创建和配置自己的网络环境。
- IAM：IAM是AWS提供的一种访问控制和身份管理服务，用于保障不同AWS账号之间的安全。
- S3：S3对象存储服务，用于存储大容量、低价、高度可用的数据，提供高可用、高性能、安全、可扩展等特点。
- Lightsail：Lightsail是AWS提供的一系列简单易用、完全托管的虚拟私有云（VPS）产品，包括了Web服务器、关系数据库服务器、负载均衡器、SSD驱动器等资源。
- Route 53：AWS域名服务，提供DNS域解析、负载均衡、SSL证书、监控告警等功能。

### EBS
Elastic Block Store (EBS)，为用户提供块存储设备的按需付费服务，用户可以在线创建新的EBS，对其进行快照备份、扩容、迁移等操作。

EBS支持多种规格的磁盘，包括超高IOPS、吞吐量最大64TB/s的GP3类型磁盘、容量最大500GB的GP2类型磁盘、容量最大16TB的IO1类型磁盘、容量最大4000GB的SC1类型磁盘等。

### RDS
关系型数据库服务（Relational Database Service，RDS），为用户提供托管的关系型数据库服务，包括MYSQL、SQL Server、PostgreSql、MariaDB、Aurora等，用户可以使用控制台或者API接口调用进行配置、创建、启动、停止、删除等操作，帮助用户快速部署需要的数据库环境，实现资源按需付费、自动备份的数据库管理。

### Lightsail
Lightsail是AWS提供的一系列简单易用、完全托管的虚拟私有云（VPS）产品，包括了Web服务器、关系数据库服务器、负载均衡器、SSD驱动器等资源。用户可以在几秒钟内创建服务器，随时按需调整计算资源、磁盘大小、RAM数量，享受经济实惠的计算资源。

### API Gateway
API Gateway是AWS提供的一种API网关服务，帮助用户创建、发布、维护、监控和保护API。用户可以使用API Gateway直接与前端、移动应用、企业应用程序、后端服务和第三方服务集成。

API Gateway支持RESTful API和GraphQL API，并通过认证授权、速率限制、缓存、监控和日志记录等功能提升API的可用性、安全性和易用性。

# 3.平台即服务(PaaS)
## 3.1 Google Cloud Platform
谷歌云平台（Google Cloud Platform，GCP）是由谷歌公司推出的公共云平台，提供包括Compute Engine、App Engine、Storage、BigQuery、Kubernetes Engine、Cloud Functions、Artifact Registry等多个产品和服务，帮助用户轻松、快速、安全地部署应用程序。

### Compute Engine
谷歌云平台的计算引擎（Compute Engine，GCE），是一项基于虚拟机的云端服务，提供给用户高度可配置、可伸缩的计算资源，包括CPU、GPU、内存、存储等。GCE提供了按需付费的计费模式，用户只需为使用的计算资源付费，无须预先支付预留的资源。

GCE支持包括Ubuntu、CentOS、Red Hat、Debian、SUSE、FreeBSD、CoreOS、Gentoo等众多主流Linux发行版本的虚拟机镜像。

### App Engine
谷歌云平台的应用引擎（App Engine，GAE），是一项云端服务，提供用户部署、运行、管理Web应用的能力，包括免费的HTTP和HTTPS协议支持、动态负载均衡、横向和纵向扩展、自动修复、自动扩容等功能。

GAE通过提供完整的框架和库支持，帮助用户开发出色的Web应用，降低开发难度，节省时间。同时，用户也可以选择自己熟悉的编程语言，例如Java、Python、PHP、Node.js、Go、Ruby等。

### Kubernetes Engine
谷歌云平台的容器引擎（Kubernetes Engine，GKE），是一项托管的、自动扩展的Kubernetes服务，帮助用户部署、管理和更新容器化的应用。GKE允许用户部署基于Docker的容器化应用，同时，还会根据应用的需要自动扩展和缩减集群的节点数目，确保应用的高可用性。

### Storage
谷歌云平台的对象存储（Storage，GCS），是一款基于分布式网络存储的高可靠、高性能、高可用的数据存储服务。GCS支持多种存储类型，包括云端硬盘存储、云端文件存储、云端数据库存储、NoSQL数据库存储等。

### BigQuery
谷歌云平台的大数据分析服务（BigQuery，GBQ），是一款云端的分布式分析服务，提供结构化数据的查询和分析能力。GBQ可连接到各种来源的海量数据，并进行复杂的分析处理，具备高性能、低延迟、高可靠等特征。

### Artifact Registry
谷歌云平台的制品仓库（Artifact Registry，GAR）是一款托管的、面向软件开发人员的Docker镜像托管服务。GAR为用户提供免费、高可用的Docker镜像仓库，可以用于存放持续交付流水线生成的Docker镜像，提升工作效率，方便用户部署和管理应用程序。

## 3.2 Microsoft Azure
微软Azure（Microsoft Azure）是一家知名的云服务提供商，为用户提供包括VMware、Hyper-V、Cosmos DB、Functions等多个产品和服务。Azure为开发者提供了包括基础设施即服务（IaaS）、平台即服务（PaaS）、软件即服务（SaaS）、DevOps、Analytics、AI、IoT、Blockchain、Edge Computing等多个领域的云服务，帮助用户在公有云、私有云、混合云之间选择最适合自己的云服务。

### VMWare Solution
Azure的VMware Solution，是一款基于VMware技术的云端基础设施服务，支持用户在私有云上部署和管理VMware虚拟机，实现对VMware软件的完全控制。

### Hyper-V
Azure的超级计算机虚拟化（Hyper-V），是一款基于Windows Server Hyper-V技术的云端计算服务，提供包括VM、SDN、SAN、NSX、SR-IOV、SLB、Load Balancing等多个计算资源。

### Cosmos DB
Azure的文档数据库服务（DocumentDB，Cosmos DB），是一款完全托管的多区域分布式数据库服务，可以全局缩放，具有低延迟、高可用、快速索引、无限存储等特性。

### Function
Azure的函数计算（Function，AF），是一项事件驱动的serverless计算服务，提供包括HTTP Trigger、Event Grid Trigger、Timer Trigger、Cosmos DB Trigger、Blob Trigger等多种触发器，帮助用户开发无状态、高性能、弹性可扩展的应用。

# 4.软件即服务(SaaS)
## 4.1 Salesforce
Salesforce.com 是一家以电子商务为核心的软件即服务（Software-as-a-Service，SaaS）平台，提供包括Marketing Cloud、Service Cloud、SuccessFactors、Einstein Analytics等多个产品和服务。

### Marketing Cloud
Salesforce的营销云（Marketing Cloud，MC），是一款针对电子商务的营销工具套件，包括营销自动化、营销机器人、营销决策、Email Marketing、Campaign Management、Marketing Automation等功能。MC帮助客户建立强大的营销能力，优化客户转化率，达成收益增长目标。

### Service Cloud
Salesforce的客服云（Service Cloud，SC），是一款支持内部和外部客户服务的云服务。SC提供了一系列客户服务工具，包括强大的客户关系管理、知识库、案例管理、客户满意度调查等功能。

### SuccessFactors
Salesforce的成功商业组织（SuccessFactors，SF），是一款为人力资源、销售、财务、采购、项目管理、生产管理、供应链管理等多个业务部门提供综合协同的HR云服务。

### Einstein Analytics
Salesforce的认知分析服务（Einstein Analytics，EA），是一款为客户提供行为洞察和分析的智能化服务，包括预测模型训练、数据挖掘、数据整合、仪表板设计等功能。EA可以用于销售预测、顾客研究、风险评估、客户忠诚度管理、活动跟踪等多个领域。

## 4.2 Dropbox
Dropbox是一个全球性的网络文件存储服务平台，为用户提供包括协作、个人云、应用程序、商店、团队、生活、生活品味等多个产品和服务。

### Collaboration
Dropbox的协作（Collaboration，DC），是一款基于云端的协作编辑工具，可同步任意位置的文件，协助用户共同编辑文档、文档库、日历、任务、地址簿等。

### Personal Cloud
Dropbox的个人云（Personal Cloud，PC），是一款为用户提供个人私密存储空间的服务。PC可以存储照片、音乐、视频、文档、软件等私密信息，且数据可以进行加密保护。

### Application
Dropbox的应用程序（Application，DA），是一款为用户提供Dropbox客户端的软件，用户可以通过该软件与其它用户分享文件、存储空间等资源。

### Shopping
Dropbox的商城（Shopping，DM），是一款提供Dropbox用户购物、消费、付款等应用场景的服务。DM通过可定制的主题模板、个性化推荐、丰富的支付方式，帮助用户完成订单交易。

### Team
Dropbox的团队（Team，DT），是Dropbox提供的一个完整的团队协作工具箱。DT为用户提供包括沟通、文件共享、日程安排、即时通信、文件备份、文档协作等应用功能。

### Photos and Videos
Dropbox的照片和视频（Photos and Videos，DP），是Dropbox提供的基于云端的图片、视频分享平台，用户可以上传自己的图片、视频、音乐、视频短片、截屏等，与他人共享。

## 4.3 Atlassian
Atlassian是一个基于云计算的软件开发服务提供商，为用户提供包括Jira、Confluence、Bitbucket、Trello、Statuspage等多个产品和服务。

### Jira
Atlassian的看板（Jira，JS），是一款高效的敏捷项目管理工具。JS可以帮助团队成员高效执行项目计划、跟踪进度、处理问题、制作报告等。

### Confluence
Atlassian的知识库（Confluence，CK），是一款基于云端的Wiki工具，提供协作、搜索、评论、版本管理、标签、附件等功能。CK可以帮助团队成员分享和储存知识，构建知识库，促进团队合作。

### Bitbucket
Atlassian的源代码管理（Bitbucket，BB），是一款基于云端的代码仓库管理工具，可以帮助团队成员维护、查看代码、管理版本、协作开发、单元测试等。

### Trello
Atlassian的任务板（Trello，TK），是一款基于云端的任务管理工具，提供清晰的任务板、优先级排序、拖动式卡片、便捷的跨平台同步等功能。TK可以帮助团队成员高效执行任务，提升工作效率。

### Statuspage
Atlassian的状态页（Statuspage，ST），是一款基于云端的服务状态监控工具，提供包括内部和外部客户的服务健康状况、消息通知、紧急事件响应等功能。ST可以帮助团队成员及时了解服务的运行情况，提升客户满意度。
