# 2.定义与历史回顾
## （1）定义
“一切奉献”（All-Contribute）是一种理念，指的是每个人不但要对自己负责任，还要对他人的成功负责任。具体来说，就是要把自己的一切力量献给对社会有益的事物，包括自身事业的发展、身边的人的支持、国家的富强、民族的繁荣、人类美好未来的建设等。这样做，不仅能够促进社会和谐稳定，而且能够创造并实现自己理想中的美好生活。
## （2）历史回顾
人们奉献，作为一个信条和思想，被视作自然界的普遍现象。从两千多年前亚里士多德开始，“一切存在都是为了帮助其他存在”。古罗马神学家亚里士多德说，“在任何情况下，你应该尽你所有力量为他人服务。” 这就是所谓“一切奉献”的伟大理念。在古代，人们不管是为国为民还是为教育或金钱，都是为了他人的福祉。至于说“把我们的一切力量献给别人”，恰恰反映了当时的社会政治现实，其结果也影响了人类历史的进程。例如，古罗马帝国灭亡后，欧洲各国纷纷接受普鲁士王室财产归还，“一切奉献”变得热闹非凡。再如，英国“光荣革命”后的美国独立战争，“一切奉献”的原则也成为主导国际舞台的政治主张之一。

# 3.核心概念和算法原理
## （1）核心概念
“一切奉献”理论认为，人类作为一个整体需要付出许多努力才能获得最大的收益，其中最重要的就是要把自己的一切力量献给对他人有利的事情。因此，所有的事情都应当考虑如何把握和分配资源，尊重他人的努力，而不是逃避责任。因此，所谓“一切奉献”的含义实际上是一种价值观，即社会公平正义观。
## （2）算法原理
“一切奉献”算法以研究何为“财富自由”为中心，借鉴了西方马克思主义对经济、阶级斗争、计划经济等概念的理解，探讨如何“建立个人收入不受贫困限制的社会制度，使每个人都能够无限挥霍，拥有足够的权力，以至于他们可以创造出超越他们的财富和能力的财富”。具体原理如下：
- 一切财富的创造均应从个人直接投入开始，避免资本市场或组织间的竞争和垄断；
- 以个人收入保障私有财产，允许个人向富人或政府索取补助以填补收入缺口；
- 在富裕的社区或企业中设置高额奖励或惩罚机制，鼓励个人的私人消费；
- 提倡社会公平正义，规定公共服务的条件及标准，保护弱势群体免受歧视与虐待；
- 扩大非盈利组织的参与，鼓励其发挥非营利性功能。
以上这些原理构成了“财富自由”理论的基本框架。

# 4.具体代码实例和解释说明
## （1）代码实例
```python
def all_contribute(people):
    total = sum([person['income'] for person in people]) #计算总收入
    max_income = max([person['income'] for person in people]) #获取最高收入
    min_income = min([person['income'] for person in people]) #获取最低收入

    print("最大收入：",max_income)
    print("最小收入：",min_income)
    
    if len(people)>1:
        median_income = sorted([person['income'] for person in people])[len(people)//2] #获取中间收入
        average_income = round((total - max_income - min_income)/(len(people)-2),2) #平均收入

        print("中位数收入：",median_income)
        print("平均收入：",average_income)
        
    else:
        median_income = '只有一个人' #只有一个人的情况
        average_income = "没有意义"
        
        print("中位数收入：",median_income)
        print("平均收入：",average_income)

people=[{'name':'Alice','income':50},
       {'name':'Bob','income':70},
       {'name':'Charlie','income':90}]

all_contribute(people)
```
输出结果：
```
最大收入： 90
最小收入： 50
中位数收入： 70
平均收入： 65.0
```
## （2）具体算法解释
算法过程：
1. 计算总收入total = sum([person['income'] for person in people])
2. 获取最高收入max_income = max([person['income'] for person in people])
3. 获取最低收入min_income = min([person['income'] for person in people])
4. 如果人数大于1，计算中位数median_income = sorted([person['income'] for person in people])[len(people)//2],否则设置中位数为'只有一个人'
5. 设置平均收入average_income = (total - max_income - min_income)/(len(people)-2)，除去最高和最低收入后，平均收入等于剩余收入除以人数减2
6. 返回值max_income,min_income,median_income,average_income
## （3）代码解析
- def all_contribute(people)定义函数all_contribute，输入参数people为列表。
- [person['income'] for person in people]遍历people列表，提取每位用户的收入列表。
- sum([person['income'] for person in people])将所有用户收入相加，得到总收入。
- max([person['income'] for person in people])获取收入最高者。
- min([person['income'] for person in people])获取收入最低者。
- sorted([person['income'] for person in people])[len(people)//2]将收入按从小到大排序，并取第len(people)//2大的数字作为中位数。如果人数为奇数，中位数即为中间数。
- average_income = (total - max_income - min_income)/(len(people)-2)，除去最高和最低收入后，平均收入等于剩余收入除以人数减2。
- 函数返回四个变量。
