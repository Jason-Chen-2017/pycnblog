# 1.简介
  

## 1.1 为什么要写这篇文章？
我国在2021年首次启动了基因改造计划，成为国家重点培育基因工程的重大课题。这项工作以国家战略为导向，是引领高质量发展方向的重要力量。随着基因技术的广泛应用，越来越多的人们开始关注基因编辑技术的应用。如何选择最适合自己的基因编辑技术并进行科学、准确的编辑，是人们提出这个问题的重要课题之一。
为此，本文通过对基因编辑技术的介绍及其发展历程、不同类型、分类和优劣分析等方面，详细阐述了基因编辑技术的种类及其优缺点、选择方法，希望能够帮助读者在实际使用中进行正确选择。
## 1.2 本文想要达到的效果
1）提供全面的基因编辑技术介绍；

2）根据生物学、信息科学、生命科学、计算机科学等相关知识介绍基因编辑技术；

3）为读者提供从基础理论到实践应用的全面指导；

4）为读者解决疑问提供参考；

5）激发读者对自身兴趣爱好或需求，为其推荐更适合自己的基因编辑技术。
# 2.背景介绍
## 2.1 什么是基因编辑技术
基因编辑技术（Gene Editing Technology）是利用遗传信息编码、复制、删除或替换某些特定区域的分子细胞，在人体的不同组织上进行高效而精准地改造、重新构建基因的手段。它可以改变遗传密码的结构、调控细胞生化、调节细胞间信号通道、改变细胞发育和功能、以及产生新的分子。
## 2.2 基因编辑技术的发展历史
### 2.2.1 单核细胞制药的时代
首先，以单核细胞制药时期的癌症治疗为代表，通过利用变异致病基因的方法，单个的DNA分子被修改后，得到具有新功能的变异细胞，这些变异细胞可以通过用另一种方法培养的方式转移给正常细胞群。
这种方式虽然能够带来较好的治愈效果，但是仍然存在一些局限性：第一，改造后的基因只能用于特定的肿瘤；第二，因为耗费大量的人力物力，产出的改造细胞较少；第三，对于普通人来说，没有能力获得有效的治疗方法。
因此，这一时期的基因编辑技术主要集中在医疗领域，用于对特殊类型的癌症细胞进行修复。由于只能对特定的肿瘤进行治疗，并且需要大规模的人力物力投入，因此发展速度缓慢。
### 2.2.2 流行病时代
随着抗击疫情的发展，特别是新冠病毒的肆虐，以基因疫苗为代表的临床药物开始火热。作为一种技术，基因疫苗可以克服疾病的致病性，即把人体免疫系统中的致病位点标记出来，然后将标记上的区域进行编辑，从而在感染者体内获得致病病毒抑制剂，阻止疾病进一步扩散。
这是一种创新性的技术。在过去，由于医学水平的限制，基因编辑技术还没有得到充分利用，因此只能满足于生物武器。但由于能够在短时间内产生较大的收益，因此引起了一场持续发展的进程。
流行病的过程中，由于损伤发生率的增加，导致大量的患者不得不接受基因治疗，因此基因编辑技术也进入到了医疗领域。
### 2.2.3 大规模集成电路芯片技术时代
目前，人们已经逐渐意识到，能够用更高端的数字技术来进行大规模的基因编辑，将是一件前所未有的事情。为了实现这一目标，以芯片集成电路(Field-Programmable Gate Array，FPGA)技术为代表的大规模集成电路技术已经崭露头角。基于FPGA的基因编辑可以让人类在短时间内编辑多个基因，并同时生成多个编辑受体，用于不同的目的。
在这一阶段，各个基因编辑技术仍处于相互竞争的状态。不同种类的基因编辑技术对人体的影响也各不相同。因此，下一个时代的基因编辑技术仍需要更多的研究和进步。
