# 1.简介
  

什么是产品价值观？产品价值观是指企业为了达成业务目标而制定的价值观或标准，是对企业产品或服务能够产生何种影响、为什么产生这些影响以及如何实现这些影响的一种判断力。产品价值观具有重要作用，它使企业在制定产品或服务时更全面考虑用户需求，更准确地洞察用户痛点，并充分认识到产品与市场之间的关系。它的运用也促进了企业在市场竞争中处于领先优势地位。根据中国产品行业研究中心发布的《2019中国互联网+产业白皮书》，截至2019年第四季度，中国互联网+领域的产业规模已超过1700亿元人民币，且已形成比较完善的产业生态系统。同时，移动互联网、物联网等新兴产业也已经得到充分布局，不断提升经济发展潜力。因此，理解产品价值观对于合理设定产品战略和提供有益于用户的产品帮助至关重要。本文将阐述产品价值观的作用和意义，并基于此介绍如何设定产品价值观，定位产品战略。
# 2.产品价值观定义
产品价值观（Product Value Orientation）是指企业为了达成业务目标而制定的价值观或标准，是对企业产品或服务能够产生何种影响、为什么产生这些影响以及如何实现这些影响的一种判断力。其核心特征是三个方面：
- 可信度：产品价值观所体现的内涵和信念具有多样性、复杂性和层次性，能够超越单纯依靠设计、营销等手段而真正地塑造产品消费者心理和社会认知。
- 责任感：产品价值观要求产品开发者重视产品所带来的“带给用户”和“带给社会”的实际影响，具有对产品负责任的品牌意识，能够真正关注消费者的满意度、喜好甚至恶意评价。
- 尊重用户：产品价值观鼓励消费者的权利与自主，消费者所拥有的个人数据、隐私权、自由意志等都应得到尊重。这是因为产品让消费者享受着产品带来的便利，但同时也意味着消费者自己也要负起责任，尽可能地为自己的隐私权、安全权益和个人利益负责。
# 3.产品价值观设置方法
设定产品价值观的方法有很多，下面就以“三个价值观”为例进行说明：
## （一）真实性
真实性代表着产品服务应该真实有效，能够表达出消费者的真实体现。例如，电子商务网站的价格体系需要反映出物品的真实价格，而不是虚假价格；手机应用的功能需要符合消费者日常生活场景，而不是贩卖广告或刷屏效果；游戏平台的虚拟道具的特性应该与真实道具相符。因此，真实性产品价值观要求企业要提供产品或服务，做到用心传达真情实感，用心倾听消费者的真实需求。
## （二）客观性
客观性代表着产品或服务所传达的信息必须是可靠的、客观的、有信息量的，不能过分迎合消费者的感官享受。例如，餐饮类产品的价格不能完全依赖消费者的想象力和个人感情；网络支付的服务应该基于消费者的消费习惯和能力，而不是企业的利益考虑。因此，客观性产品价值观要求企业在产品或服务的设计、功能选择上，体现出科学逻辑、客观准确和真诚的态度。
## （三）亲密性
亲密性代表着消费者对产品或服务拥有感情上的亲切关爱，能够增强消费者对产品或服务的忠诚度。例如，日常用品的使用体验应该是亲密、友好的，不会因产品外观的变化而改变用户对产品的认识；购物类产品的价格选择应该能够满足消费者的需求，并且能够令消费者间接得到产品的感受。因此，亲密性产品价值观要求企业的产品或服务必须体现出高端感觉、亲切友好、令人留住的特质。
# 4.产品战略定位方法
产品战略定位是指基于产品价值观，明确产品的发展方向、价值创造和规模效益，寻找产品的长期价值和生命周期价值。其一般步骤包括以下五个方面：
1. 定义战略目标：首先明确产品的发展方向，确定企业的目标和愿景。产品的价值创造有助于企业实现目标，在未来一段时间内的市场份额占比可以作为衡量成功的标准。
2. 消费者需求分析：根据消费者对产品的需求，分析其长期需求、短期需求以及优先级。从而把握产品在市场中的主导地位。
3. 提炼核心价值：通过对产品的分析和调研，得出产品的核心价值和潜在的盈利模式。把握产品的价值所在。
4. 定位路径规划：根据消费者需求和企业目标，建立产品的发展路线图。确定产品发展的主要路径，并针对每条路径制定具体的策略。
5. 执行产品策略：从战略定位的第一步开始，不断优化产品策略，通过有效满足消费者的需求。并确保产品的持续迭代，一直走向用户的最佳体验。
# 总结
产品价值观是企业产品或服务的一项重要元素。它对于企业产品或服务的设计、开发、推广、营销、改进、迭代都起到重要作用。设立产品价值观有助于产品和市场的交流沟通，更好地参与市场竞争，提升企业在市场中的影响力。产品战略定位则是基于产品价值观的产品发展方式，通过提炼产品的核心价值、分析用户需求、制定发展路线图，以及落实执行策略，为产品的长远发展提供方向和指引。
