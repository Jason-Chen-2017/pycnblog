# 2.基本概念
## 2.1 语言与文字
语言是人类用来交流沟通的一种方式。语言是由动词、名词、形容词、副词、连词、介词、时态词、代词、叹词、感叹词、问号、否定词等构成的符号串，并不是某个物体的名字或属性。换句话说，语言就是人们用自己的符号来表示客观世界的各种事物。

文字是语言的一个载体，它在物理上是用笔书写出来，在逻辑上是按照一定规则排列组合而成的符号串。文字制作者一般都是具有艺术天赋或专业知识的人。

例如，下面这段英文语句："I love playing soccer."。"I"、"love"、"playing"、"soccer"等都属于名词，它们的意义可以直接被读懂。而"I love playing soccer"则是一个完整的句子，它代表一个情感上的情绪，表达了作者对自己的喜好。

## 2.2 词汇、语法、句法与句子
### 2.2.1 词汇
词汇是语言结构的基础，它由单词和短语组成。单词是语言的最小单位，通常由一个或多个音节组成。短语是由两个以上单词组成的结构。单词之间有时会用连接符如标点符号或空格隔开。

例如，下面的例子中，"The quick brown fox jumps over the lazy dog"这个句子共计10个单词。

```python
the = determiner/article   # 限定词
quick = adjective          # 形容词
brown = color              # 颜色名词
fox = noun                 # 名词
jumps = verb               # 动词
over = preposition         # 介词
lazy = adjective           # 形容词
dog = noun                 # 名词
```

### 2.2.2 语法
语法是语言结构的规则。语法是指任何语言的总体性质，是所有句法单元的集合以及这些单元之间的相互关系和次序规律。它主要分为三大类：
1. 语词语法：描述词性、形式和功能。如名词性、动词性、形容词性、副词性、介词性等。
2. 句法结构：由短语或整句组成，包括主干、谓语、宾语、状语、定语、助词等。
3. 修辞语法：包括一些代词性的动词、副词、形容词、名词、介词等。

例如，下面的句子含有语法错误，"does not eat food"。第二个单词"eat"是介词，应与后面的名词"food"平行。"not"是一个否定词，位于动词"eats"之前。

```python
John does not eat food      # 正确
John eats a food            # 错误
```

### 2.2.3 槽位与句法树
槽位是语法结构中的一个重要特征，它表示特定类型的词汇可以在哪些位置被插入。常见的槽位包括：主语（S）、谓语（V）、宾语（O）、定语（D）、状语（P）、间接宾语（IO）、主题（T）。

如下图所示，"John likes reading books on my iPad"是一个完整的句子，它的槽位包括：

1. S 是名词 "John" 的主语；
2. V 是动词 "likes" 的谓语；
3. O 是名词 "reading" 的宾语；
4. D 是形容词 "books" 的定语；
5. P 是介词 "on" 的状语；
6. IO 是名词 "my iPad" 的间接宾语；
7. T 是形容词 "iPad" 的主题。


由于每种语言的语法规则都是独特的，所以句法树也不可能一一列举，只能通过一些示例来感受一下不同语言的句法差异。
