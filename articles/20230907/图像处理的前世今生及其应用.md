## 二、图像的产生与基本属性
​        图像作为一种多维的信号，由各种元素组成，这些元素构成的空间称为图像空间，而图像所包含的信息则由图像矩阵表示。图像矩阵是一个矩阵形式的图像，行代表像素的高度，列代表像素的宽度，每一个元素对应图像的一个像素点，每一个元素都可以看作一个向量或一个元组，它代表了某种颜色或亮度的数值。当把图像看做一个矩阵时，其每个元素的值便对应于图像中的某个灰度级。不同的颜色表示不同的灰度级，黑色（0）表示较暗的灰度，白色（255）表示较亮的灰度，中间的部分则依次递减。不同格式的图像矩阵，比如单通道RGB图像矩阵或单通道灰度图像矩阵，都可以用来表示图像，其中单通道RGB图像矩阵表示一个彩色图像，有三个通道，分别用于表示红色、绿色和蓝色的光，而单通道灰度图像矩阵表示了一个黑白图像，只有一个通道，用于表示灰度值。通常情况下，为了方便处理和分析，我们都会采用更加高效的存储方式，比如采用位图（bitmap）或矢量图（vector image）。
### 2.1 光学属性
​           根据光学原理，一副图像是在视网膜上看到的、由像素点组成的数字化图像。人眼对物体反射出的光线经过感光器，然后通过视网膜反映到大脑，大脑对图像数据进行处理和学习，并将其转化为神经信号传给视觉皮层，再经过双侧眼睛区传递到肉眼观看。图像的采集、拍摄过程和传播过程中涉及到的光学参数如下表所示。

|    参数     |          描述           |
| :--------: | :-------------------: |
| 光源类型   |       点光源、线圈       |
|    感光条件   | 光线入射角度和距离，聚焦距离 |
|    曝光时间   |      长曝光、短曝光      |
|  摄像头类型   |      CCD或CCD+CMOS       |
| 拍摄角度范围 |        360°          |
|   照明条件   |      阴天、阳光、雾霾      |

### 2.2 噪声和光照
​        一般来说，图像出现噪声主要是由于以下几种原因导致的：

1. 液晶屏幕上的镜像现象：图像在感光器与显示器之间的光照差异引起的。
2. 操作系统、显示卡、传感器、调制解调器等硬件的干扰。
3. 图像传感器本身的运动不稳定。
4. 模拟成像的过程存在波纹效应。

对于噪声的控制，有多种方法：

1. 对图像进行预处理：先用均值滤波消除椒盐噪声、锯齿状噪声；然后对图像进行局部直方图归一化、方差归一化、拉普拉斯金字塔变换等图像增强手段。
2. 使用硬件设备进行信号处理：对于像素信号进行硬件加速处理，如DSP、FPGA、ARM等。
3. 对图像进行预测：采用迭代重建算法，根据预测误差对图像进行逼近和补偿，同时考虑测量噪声的影响。
4. 在图像上加入噪声、偏移：在图像上添加噪声或者引入运动模糊，也可以消除光照变化和自然背景中的噪声。

### 2.3 分辨率与图像大小
​        分辨率(resolution)是指图像中像素数量与真实物体尺寸之比。由于显著原因的分辨率降低会导致图像失真。图像大小(size)与分辨率密切相关，因为分辨率越高，图像越模糊。因此，图像的大小只能是固定整数倍关系，如果分辨率不是整数，那么图像大小就会不正确。

一般来说，图像的分辨率分为标准分辨率(conventional resolution)和实验分辨率(experimentally determined resolution)，它们是通过实验确定出来的。由于实验分辨率受实验条件、仪器设备和环境因素的影响很大，所以实验分辨率又可分为模拟分辨率(analogous resolution)和数字分辨率(digital resolution)。

常用的标准分辨率包括100dpi、150dpi、300dpi、600dpi、1200dpi、2400dpi等。人类对图像的正常视觉效果大致可以在20-50dpi之间。对于视网膜上看到的图像而言，分辨率越高，图像质量越好。但是对于摄像机拍摄的图像而言，分辨率的选择必须根据不同视觉需求来进行相应的调整。

图片编辑工具支持多种尺寸的图片调整功能，如按比例缩放、裁剪、旋转、变换、模糊、透明度调整等，还提供批量处理功能。调整后的图片可以通过压缩、打印或email发送。

### 2.4 RGB图像与其它颜色模型
​        目前主流的颜色模型有RGB模型和CMYK模型，它们都是三基色模型。RGB模型中，红色、绿色、蓝色颜色各占8bit，共24bit。CMYK模型中，青色、品红、黄色、黑色颜色各占8bit，共32bit。两种颜色模型都可以用于表示彩色图像，但它们的存储方式、应用场景、特性和相关的图像处理技术都有差别。

对于RGB模型，它的色域范围非常广泛，适用于处理一般的彩色图像。但是它的空间占用率较高，一个像素点需要占据3个字节（24bit），所以文件大小较大。而对于黑白图像来说，它的色域范围较窄，只有0~1的灰度值，所以只需要1byte就够表示，对应的存储单位是bit，因此文件的大小要小很多。因此，在保存和传输黑白图像的时候，通常采用位图格式，而保存彩色图像采用JPEG、PNG等格式。

而CMYK模型，它的色域范围较为狭窄，仅限于印刷业、彩妆品、艺术品等特定领域，由于不适用于计算机处理，因此很少被使用。

虽然CMYK模型对于处理图像的灰度级变换十分有用，但它不能完全解决彩色图像的问题。目前，主流的彩色图像处理算法仍然基于RGB模型进行设计，因此，对于具有明显的彩色特性的图像，建议使用RGB格式，其余图像建议采用CMYK格式。
