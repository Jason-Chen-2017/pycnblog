# 2.基本概念术语说明
## keras简介
Keras是一个开源的Python库，它是一个高层次的、用户友好的接口，用于构建和训练深度学习模型。它可以运行在 TensorFlow 或 Theano 上，并且具有独特的功能特性。主要包括以下四个部分:

1. layers：这一部分中包含各类神经网络层，例如卷积层、池化层、全连接层等等。

2. models：这是Keras中的模型模块，里面提供了各种常用模型的实现。你可以用它快速构建复杂的神经网络。

3. optimizers：这是Keras中的优化器模块，里面提供了各种常用优化算法的实现。你可以利用它们来训练神经网络。

4. callbacks：这是Keras中的回调函数模块，它提供了一些用于监控和保存模型训练过程的钩子。

除了上述四个部分之外，Keras还有一个函数式API（Functional API），允许用户通过链式调用多种层构造模型。这个API有助于更好地理解Keras的设计理念。

## 激活函数
激活函数（Activation Function）是指用来将输入信号转换成输出信号的非线性函数。深度学习模型中最常用的激活函数是ReLU（Rectified Linear Unit）函数，它的公式为 max(x,0)，即若输入 x > 0，则输出值为 x，否则输出值为 0 。它能够在一定程度上抑制不必要的梯度下降，从而加速模型收敛速度。除此之外，还有tanh、sigmoid、softmax、softplus、softsign、hardtanh等常用激活函数。
## 损失函数
损失函数（Loss Function）是衡量模型输出结果与实际值差异的指标。深度学习常用的损失函数有均方误差（Mean Squared Error，MSE）、交叉熵（Cross Entropy Loss，CE）、KL散度、迪里希科赫损失（Dice Loss）等。均方误差直接衡量输出值与实际值的差距大小，一般用于回归任务；交叉熵衡量不同概率分布之间的距离，常用于分类任务；KL散度衡量两概率分布之间的相似度，可以用来衡量生成模型与真实数据的相似度。
## 优化器
优化器（Optimizer）是更新神经网络权重的算法。深度学习常用的优化器有SGD（随机梯度下降）、Adam、Adagrad、Adadelta、RMSprop等。SGD对所有参数同时进行更新，往往导致局部最小值，因此收敛速度缓慢；Adam在每次迭代中都会调整学习率，它对学习率衰减非常敏感；Adagrad在每个迭代过程中只考虑自变量的一阶导数信息，因此对每个自变量的更新幅度会受到约束；Adadelta在每次迭代中都会保留之前的变化率矩，从而对小的更新值做出响应。RMSprop是一种自适应学习率的优化器，它的动机是为了抑制过大的学习率带来的震荡。
