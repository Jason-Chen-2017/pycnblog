# 1.简介
  

2017年1月1日，中华人民共和国成立，世界进入信息时代。随之而来的，是数字化、网络化的社会大变革。如今，社会生活中的各个角落都被数字技术所接入，每天产生海量数据，这些数据承载着巨大的价值。任何一个公司或组织都不可能做到真正客观地了解所有的数据，因此需要对数据进行保护，在使用前提下向用户提供必要的信息，保障个人数据的安全性。此时，数据的跨境流动成为关键。在不同国家，存在不同的数据隐私法规。本文将探讨目前多国对个人信息保护法律的不同程度，以及中国特色的数据保护方式。
# 2.基本概念术语说明
## 2.1 数据分类
个人信息包括“生物识别信息”（例如姓名、身份证号码等）、“行为特征信息”（例如浏览习惯、搜索记录、购买历史等）、“非行为特征信息”（例如兴趣爱好、健康状况、生活习惯等）。数据保护法律要求，对个人信息的收集和处理应遵循数据分类法，分为两类——敏感信息和一般信息。
### 敏感信息
敏感信息是指在确定用户个人资料、识别特定个体或保护特定群体利益、促进经济发展和社会稳定方面具有特殊重要性的信息。包括以下几类：

1. 身份标识信息：指能够唯一标识自然人的信息，如姓名、身份证号码、护照号码等；
2. 交易隐私信息：指与交易有关的信息，如银行账户信息、交易密码等；
3. 健康隐私信息：指与个人生命健康相关的信息，如疾病史、过敏史、个人症状等；
4. 婚姻家庭隐私信息：指与离婚、丧失配偶权力等关系密切的个人信息；
5. 法律规定的其他敏感信息。

### 一般信息
一般信息是指不涉及用户特定需求、特别不需要保护的公共信息，如所在地区、职业、教育经历、联系方式等。对于一般信息的保护，主要采取披露、出售、访问限制等措施。

## 2.2 数据跨境传输
根据2016年12月27日国务院办公厅发布的《关于建立健全个人信息（人口信息）保护制度的意见》，人口信息是指从事人口普查、人口统计、人口调查、公共卫生、社会福利、计划生育、医疗卫生、教育培训、民政服务、司法侦查等业务活动过程中，收集、管理、存储、使用、公开披露的人口画像、人口属性、人口数量、人口因素、个人信息、身份信息等信息，其主要包括身份信息、社会保险信息、健康信息、住址信息、住宿信息、工作单位信息等。这些信息对各国公民的生活、工作、财产安全、社会治安、公共利益等方面都具有重要的影响，需受到充分保护。

2017年8月，根据欧洲议会通报2016/2696(欧盟2016年第四次全国人口普查委员会第十一次会议决议)要求，旨在促进欧洲各国间的数据共享，推动欧盟对数据保护制度的完善，增强欧盟对数据主权的坚持，确保欧盟公民的个人信息得到充分保护。

2018年4月，联合国秘书长马尔萨斯向记者展示了首批欧盟个人信息保护框架草案。根据新框架规定，欧盟对于个人信息保护实行如下五项措施：

1. 主体权利：由欧盟授权代表人員进行获取、访问、使用、存储、转移、删除、注销、传输、交换、联络、公开披露、加工、监督、处置等操作的人身依据条款；

2. 目的权限：个人信息的使用仅限于满足欧盟法律、法规和国际组织的指令、请求或者需要，不得用于任何其他用途；

3. 范围：欧盟的个人信息仅限于与欧盟境内或与欧盟境外的特定国家及地区的人士之间共享；

4. 同等保护：若有异议，则可以通过书面申请转移至其他受到欧盟的个人信息保护机构进行审查；

5. 安全：采用符合欧盟标准的安全保护措施，包括加密传输、匿名传输、授权访问、访问审计、数据保护分级制度等。

联合国秘书长还表示，欧盟尊重各国的主权和自主选择，鼓励各国共同保护个人信息，通过倡议制定数据保护标准、行业标准、标准组织和协作机制来推动个人信息保护运动，推动欧盟各成员国共同构建起个人信息保护体系，共同应对欧洲信息安全风险。

综上所述，在面对不同国家的数据隐私法规时，中国政府可以参考以上各项规定，在基础上制定自己的法律政策。同时，借鉴欧盟个人信息保护框架，中国也可借助国际组织的力量，与其他国家分享个人信息保护制度，共同构建起个人信息保护体系。
