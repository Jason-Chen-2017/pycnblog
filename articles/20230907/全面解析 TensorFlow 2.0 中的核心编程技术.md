# 2.概览
## 2.1 TensorFlow 是什么？
TensorFlow（简称 TF）是一个开源的机器学习库，它最初由Google设计，是一个开源软件库，用于机器学习和深度神经网络的研究与开发。它提供一整套系统工具，包括张量计算(tensor computation)、自动微分(automatic differentiation)和数据管道(data pipeline)，可以有效地执行各种机器学习任务。

早期的 TensorFlow 只支持静态图模型，而随着时间的推移，静态图模型逐渐失去了它的优势，所以很多人都转向了动态图模型。动态图模型使用更方便的API接口，能直接运行Python或C++程序。相比静态图模型，动态图模型的运行效率更高，并且更容易被跨平台部署。因此，目前大部分深度学习框架都是采用动态图模型。

至于为什么叫 TensorFlow，其实就是谷歌内部的一些内部使用的名字。但随着时间的推移，很多知名的机器学习框架都开始改名，比如 PyTorch 和 MXNet。但是，作为一个社区，TensorFlow 还是始终在坚持这个名字。

## 2.2 TensorFlow 2.0 是什么？
TensorFlow 2.0 是 TensorFlow 的下一代版本，相对于旧版来说，它新增了许多新特性。其中，最显著的变化之一，是从 Keras API 转换到 TensorFlow 2.0 API。这是因为 Keras 是一种现有的、流行的高级接口，它使得构建、训练和部署复杂的神经网络变得非常简单。然而，Keras 本身就是基于 TensorFlow 框架开发的，而且它会自动处理许多繁琐的细节。TensorFlow 2.0 使用新的 eager execution 模式，能像 NumPy 或 Matplotlib 那样直观地创建、运行和调试神经网络。此外，TensorFlow 2.0 提供了一个全面的生态系统，其中包含大量的组件和扩展，可以帮助您轻松地解决日益增长的机器学习挑战。

除此之外，TensorFlow 2.0 还添加了强大的性能优化功能。例如，它可以在多个 GPU 上并行执行同一模型，还能利用自动图形优化器和其他编译器优化技巧来加快模型的速度。此外，TensorFlow 2.0 提供了分布式训练功能，用户可以轻松地将模型扩展到多台服务器上，并利用数据并行方法来提高训练速度。最后，TensorFlow 2.0 支持广泛的硬件配置，包括 CPUs、GPUs 和 TPU (tensor processing unit)，无论您是刚开始还是已经积累了一定的经验，都可以用 TensorFlow 2.0 来快速构建、训练和部署您的模型。
