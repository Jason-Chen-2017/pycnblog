# 1.简介
  

## 1.1.背景介绍
随着新一代的无人机、汽车、机器人等智能终端设备的普及，越来越多的人已经意识到自身在未来的生活中将会掌握一个新的领域——边缘计算。简单来说，边缘计算就是指将算法、模型部署在网络边缘、数据中心内的计算资源上，对用户数据的处理进行加速、缩短响应时间、提升系统整体的处理能力。
作为中国移动集团公司的高级副总裁兼CEO陈亚军博士，我们希望通过本文向读者介绍一下中国移动公司的“边缘计算”服务架构，其中包括边缘云平台、边缘数据中心、边缘计算服务、边缘计算应用场景等方面，帮助读者能够更全面的了解并运用相关的知识。
## 1.2.基本概念术语说明
### （1）边缘云平台
边缘云平台（Edge Cloud Platform）是中国移动公司针对边缘计算领域的云平台，它包含边缘数据中心、边缘计算服务、云平台安全、边缘计算应用开发工具等组件，其主要功能包括：
- 数据存储：将实时产生的数据从传统的云端迁移至边缘数据中心，实现边缘分析；
- 数据计算：对边缘设备上的业务数据进行实时的计算分析，确保数据的精准性、及时性和一致性；
- 计算资源：为不同业务场景的边缘设备提供不同的计算资源，减少运算密集型任务的响应延迟，提升系统整体性能；
- 服务安全：通过各种安全技术来保证数据的隐私、安全和完整性；
- 应用开发：提供完善的应用开发环境，支持复杂的边缘计算应用场景的开发。
### （2）边缘数据中心
边缘数据中心（Edge Data Center）是中国移动公司在云计算平台之外部署的一套硬件基础设施，它采用离线方式部署在区域边缘，同时为客户提供可靠的低延迟计算服务。其主要功能包括：
- 数据隔离：将不同业务场景的边缘设备的数据完全隔离开，确保数据的隐私、安全和完整性；
- 计算资源：对不同业务场景的边缘设备配置不同的计算资源，通过集群的方式共享计算资源，降低成本；
- 网络连接：通过专门设计的路由器或交换机技术，使边缘数据中心与其他边缘数据中心之间建立低延迟、高带宽的网络连接；
- 电源管理：在需要时通过电源管理系统来控制各个边缘设备的电源状态，以达到节省功耗和节约能源的目的。
### （3）边缘计算服务
边缘计算服务（Edge Computing Service）是中国移动公司基于边缘数据中心所提供的计算能力，为客户提供轻量级的计算服务。其主要功能包括：
- 模块化管理：边缘计算模块化管理方案能够灵活应对异构、变化的边缘设备需求；
- 弹性伸缩：根据边缘设备的实时负载状况和资源利用率，自动调整计算资源的数量和规模，确保满足用户的使用需求；
- 弹性调度：根据边缘设备的位置信息、上下行流量和设备性能等因素，在边缘数据中心内动态分配计算资源，确保服务质量和效率；
- 服务监控：提供丰富的服务监控手段，能够及时发现和解决运行过程中出现的问题，确保服务的高可用性。
### （4）边缘计算应用场景
边缘计算应用场景（Edge Computing Application Scenario）是中国移动公司对于边缘计算技术发展趋势和发展前景所展开的一系列技术合作，主要聚焦于如何通过边缘计算技术来提升用户体验、降低用户等待时间、优化网络通信效率、改进设备运营模式等方面。目前，边缘计算的应用场景包括视频监控、车联网、智慧医疗、无人驾驶、智能电网等。
