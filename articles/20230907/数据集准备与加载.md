# 2. MNIST数据集
## 1. 概述
MNIST数据集是一个手写数字识别数据集，它共有70000张训练图片和10000张测试图片，图像大小为$28\times 28$，像素值范围为0到9，其中50000张图片用于训练，10000张图片用于测试。它的特点是偏简单、直观、易于理解，是机器学习领域的一个经典数据集。其格式如下：

 - 训练集：mnist_train.csv (60000条记录，包括图像ID及其对应的标签) 
 - 测试集：mnist_test.csv （10000条记录）
 - 每张图片：黑白灰度图，大小为28*28个像素点，像素值范围为0~255。
 
## 2. 数据加载
MNIST数据集在机器学习领域中被广泛应用，目前已成为深度学习任务中必不可少的数据集之一，尤其是在图像分类、自动驾驶等场景下。

加载MNIST数据集可以使用`tensorflow.keras.datasets.mnist.load_data()`函数，该函数会返回两个元组：（x_train, y_train），(x_test, y_test)，分别表示训练集特征和目标，测试集特征和目标。 x_train/x_test 是MNIST训练集/测试集的特征数据，y_train/y_test是对应样本的类别标签。

```python
import tensorflow as tf

# Load the mnist dataset from keras datasets module
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.mnist.load_data()

print("Number of training examples:", len(x_train))
print("Number of test examples:", len(x_test))
```

