# 1.简介
  

## 1.1 Diffusion Maps简介
Diffusion maps是一个高维数据可视化方法。它可以将复杂的分布数据（如图像、生物特征等）转换为低维的空间曲面。它的特点是能够捕捉到局部的复杂结构以及在数据集中的相似性信息。它利用了数据空间的局部流形结构，通过不断迭代扩散的方式，使得局部密度分布逐渐接近全局密度分布，从而实现数据的降维。其数学模型及迭代推算过程十分简单直观。它具有良好的鲁棒性和适应性，可以应用于多种场景下的复杂数据分析。本文主要围绕Diffusion Maps进行讨论。

## 1.2 Diffusion Maps的优缺点
### 1.2.1 优点
- **计算速度快**：Diffusion Maps可以在$O(n^2 \log n)$的时间内计算得到图形嵌入结果，相比传统的机器学习算法，其计算时间更短。
- **易于理解**：Diffusion Maps可以有效地表示高维数据，并发现数据内部的结构关系。因此，它很容易被研究者接受。此外，对底层算法的理解也比较容易。
- **可解释性强**：Diffusion Maps生成的图形嵌入能够反映各变量之间的相互影响，为人们提供更直观的理解。

### 1.2.2 缺点
- **缺乏全局观察力**：因为Diffusion Maps只能表示局部的结构信息，因此无法观测全局的整体特性。这是由于它仅考虑了局部邻域的信息，没有考虑整个数据集的全貌。
- **限制较多**：Diffusion Maps模型受限于局部密度分布，因此对于非平滑的分布无法进行处理。除此之外，其输出图形仍然存在一定程度的空间切片现象，无法直接反映数据的全局形状。
- **难以处理海量数据**：Diffusion Maps的计算复杂度随着样本数量的增长呈线性增长，这会导致其处理海量数据的效率非常低下。因此，Diffusion Maps在当代实际中还存在一些局限性。
