# 1.简介
  

在很多实际应用场景下，高维数据（high-dimensional data）是不可缺少的。由于要处理的数据量非常大，传统的线性方法往往无法很好的满足需求。为了降低维度，采用非线性投影的方法通常效果更好。常用的非线性投影方法包括主成分分析(PCA)，次模态分析(ICA)，以及局部线性嵌入(LLE)。然而，这些方法并不能完全保证降低维度后的结果的准确性和可解释性，因为它们都是基于欧氏距离的距离度量。本文提出了一种随机化的方法——随机欧氏投影(Random Euclidean Projection, REP)来构造一个优化的欧氏投影。

# 2.背景介绍
现代统计学技术已经对大规模数据集建模、预测等任务产生了巨大的影响。利用机器学习技术可以有效地处理海量数据，促进了科技的发展。但是，高维数据的复杂性与样本不足带来了极大的挑战。尤其是在很多情况下，我们不仅希望对数据进行建模，还需要保留尽可能多的信息，因此，降维至低维度是一个关键步骤。现有的降维技术包括主成分分析法(PCA), 次模态分析法(ICA), 和局部线性嵌入法(LLE)。这些方法都试图找到一种低维空间能够最佳表示原始高维数据，同时又保持原始数据中的信息损失最小。由于目标函数和约束条件限制，这些方法只能在有限的时间内收敛。对于大型数据集来说，即使使用优化算法也会很耗时。

另一方面，当目标数据维度较小时，使用最简单的欧氏距离作为度量时，欧氏投影(Euclidean projection)算法可以得到比较理想的结果。其思路就是找到原始数据中的一组方向向量，使得这些方向向量构成的子空间能够最有效的表达原始数据的特征。通过这种方式，我们可以在一个低维空间中呈现原始高维数据的主要模式。目前，许多文献研究了欧氏投影算法的性能，但仍然存在一些缺陷。

# 3.基本概念术语说明
## 3.1. 高维数据 high-dimensional data
高维数据一般指的是具有多个变量（attributes或features）的数据。举例来说，身高、体重、年龄、工作经验、学历、地址等信息都属于高维数据。

## 3.2. 低维数据 low-dimensional data
低维数据指的是只有少数变量的数据。如经典的“维恩图”所示，这是一张2维图像，它的坐标轴上有两个变量：纵坐标表示某种属性，横坐标表示另外一种属性。在这个例子中，高维数据有7个变量（$x_1$,$x_2$,...,$x_n$），而低维数据只有两个变量($z_1$, $z_2$)来描述相同的信息。

## 3.3. 投影 projection
投影是将高维数据映射到低维空间的过程。一般来说，投影的方式有两种：
1. 投影到某个超平面的距离（欧氏距离）。
2. 投影到一个基底空间中的线性组合。

## 3.4. 欧氏距离 Euclidean distance
欧氏距离是指两个点之间的直线距离。对于两个点$(p=(p_1,p_2,...p_d),q=(q_1,q_2,...,q_d))$，其欧氏距离定义如下：
$$\sqrt{\sum_{i=1}^d[(p_i-q_i)^2]}$$

其中，$d$表示数据中的变量个数。

## 3.5. 欧氏投影 Euclidean projection
欧氏投影就是找到一个方向向量，使得该方向上的投影误差最小。其一般形式是：
$$proj_{\vec{v}}=\argmin_{w \in \mathbb R^m}\|X-\mu_W^{T}V^{*}(Vw)\|_F$$

其中，$\vec v$表示搜索的方向向量，$X$表示原始数据矩阵，$\mu_W$表示数据均值向量，$V^{*} = V^{-1/2}$是奇异值分解（SVD）矩阵，$Vw$表示从数据空间转换到系数空间。

欧氏距离的假设是各个变量之间服从独立同分布的正态分布。因此，欧氏投影一定能够找到一个具有低方差的低维空间，使得原始数据经过投影后仍然能够保持原来的信息。

## 3.6. 随机欧氏投影 random euclidean projection
在实践中，欧氏投影算法需要对数据进行拟合和计算，这在数据规模较大时耗费时间和资源。因此，我们可以通过随机化的方法来加快收敛速度。

假设原始数据集中共有$n$个数据点，则随机欧氏投影方法首先随机选择$k$个数据点，并计算相应的均值向量$\mu_k$和协方差矩阵$\Sigma_k$. 然后，对每个数据点$x_i$，随机生成一个旋转矩阵$R_i$,并将它乘以数据点的均值向量的标准差矩阵，得到$\tilde x_i = RX_i + N(0,\sigma^2 I)$, $\forall i = 1, 2,..., n$. 最后，根据新的随机数据集$\tilde X = [\tilde x_1, \cdots, \tilde x_n]^T$, 对$\tilde X$进行一次欧氏投影即可。这样的方法可以减少计算时间，提升效率。
