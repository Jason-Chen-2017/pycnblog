# 1.简介
  

文件（file）和目录（directory）是Linux系统中最基本的两个存储单位。目录用来组织文件，可以把多个文件放在同一个目录下，使得文件更加容易管理、查找和访问。而文件的权限则决定了谁有权对文件进行读、写或执行等操作。本文将会教你使用Shell命令行对文件和目录进行各种操作，并深入了解它们背后的机制原理。
## 1. Linux文件类型
在Linux中，每个文件都有一个特定的类型，比如：
- directory（目录）：用来存放其他文件的文件夹，它不直接包含数据，但可以包含子目录；
- link（符号链接）：类似Windows里的快捷方式，指向另外一个路径名；
- character device（字符设备）：一种特殊的设备文件，例如鼠标、键盘等；
- block device（块设备）：硬盘、磁盘驱动器、光驱等外设；
- socket（套接字）：网络通信中的一种抽象概念。
除了上面提到的这些类型之外，还有更多的文件类型，你可以通过以下命令查看：
```
ls --file-type
```
## 2. Shell中的文件操作命令
- ls：显示当前目录下的文件列表；
- cd：切换到指定目录；
- mkdir：创建新目录；
- rm：删除文件或目录；
- cp：复制文件或目录；
- mv：移动或重命名文件或目录；
- touch：新建空白文件；
- cat：显示文件内容；
- head/tail：显示文件开头或结尾的内容；
- less/more：分页显示文件内容；
- chmod：修改文件权限；
- chown：修改文件所有者；
- df：查看磁盘分区信息；
- du：查看文件或目录的磁盘使用情况；
- find：搜索文件或目录；
- grep：搜索文本文件；
- ln：创建文件链接；
- gzip/gunzip：压缩或解压文件；
- bunzip2：解压bz2压缩包；
- zip/unzip：压缩或解压zip文件。
以上命令列举了一些重要的文件操作命令，后面会逐一详细介绍。
## 3. 查看文件内容命令
### 3.1 cat 命令
cat 命令用于显示文件内容，默认情况下，cat命令从第一行开始输出内容，可以利用“-n”参数让其显示行号。命令如下所示：
```
cat [-n] filename
```
示例：
```
$ cat test.txt 

Hello World!
This is a text file.
```
如果想将文件内容追加到另一个文件末尾，可以使用以下命令：
```
cat >> new_file.txt < old_file.txt
```
此时，old_file的内容会被追加到new_file的末尾。
### 3.2 less 命令
less 命令也是显示文件内容的命令，但是比cat命令更高级，除了可以向上翻页和向下翻页外，还可以利用键盘上的箭头进行滚动，并且它还可以方便地保存并退出文件。命令如下所示：
```
less filename
```
示例：
```
$ less test.txt 
Hello World!
This is a text file.

^C
```
按下Ctrl+C组合键可退出less命令，但是如果文件比较大，建议先用命令“head”检查前几行内容。
### 3.3 more 命令
more 命令和less命令类似，也是分页显示文件内容的命令。不同的是，more只能向下翻页，不能向上翻页。所以，如果需要查看文件中间的内容，则需要多次运行该命令。命令如下所示：
```
more filename
```
示例：
```
$ more test.txt 
Hello World!
This is a text file.
Press any key to continue... 
```
按下回车键可继续浏览剩余内容，直至文件结尾。
