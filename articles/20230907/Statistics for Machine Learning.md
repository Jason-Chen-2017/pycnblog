# 2.统计学的基础知识
## 2.1 随机变量（Random Variable）
统计学中的随机变量是一个具有自身分布的变量。换句话说，随机变量描述的是一个抽象的现实世界中的变量，而其取值则由不同条件下的特定结果决定的。
比如，抛硬币游戏的结果就是随机变量，它可以取值为正面的概率或者反面的概率。而投掷骰子的结果就是另一个随机变量，它可以取值为1到6之间的每一种情况的出现频率。
随机变量由三个要素组成：取值空间（Value Space），概率密度函数（Probability Density Function），概率分布（Probability Distribution）。前两个要素称之为定义，第三个要素则称之为估计或假设。
### 2.1.1 随机变量的性质
1. 可列可加性：对任意的两个随机变量X和Y，都存在一个随机变量Z，使得对任意的a属于X的取值范围，都有Z(ax)=az+y(1-z),其中z是单位区间上的随机变量。即，将随机变量进行线性组合，得到的仍然是一个随机变量。
   > 证明：若随机变量X的取值空间为R，则有|Rx|=|x|+1,且存在c∈R,使得yc=cx,即每个取值都有一个对应的比例。由于每个值都有着自己的比例，所以该组合也是取值空间的一个划分。因此，Z=aX+(1−a)Y恰好可以按比例分配每个取值。而且，Z是X和Y的线性组合，也就必然满足上述性质。

2. 分布的独立性：若随机变量X和Y之间不相关，则它们的联合分布也是相互独立的。如果X和Y之间有相关性，那么它们的联合分布也是相关的。
   > 证明：如果X和Y之间无关，则存在非负常数λ1,λ2>0，使得P(X=x,Y=y)=λ1^X(1-λ1)^Yλ2^(xy)(1-λ2)^(1-x-y)，即X与Y的联合分布与它们各自的分布都是独立的。相似的，如果X和Y存在相关性，则存在参数ρ，使得P(X=x,Y=y)=ρ^|x-y|-|x-y|<1，则X与Y的联合分布也是相关的。

3. 期望和方差的关系：方差越小，随机变量的变化幅度就越小；方差越大，随机变量的变化幅度就越大。
   > 证明：对任何随机变量X，均值E(X)等于所有可能取值的乘积与这些取值的概率之和。换言之，E(X)等于平均值。当X是离散型随机变量时，如抛硬币的结果，这个平均值等于正面朝上的概率，即1/2。另一方面，方差Var(X)等于所有可能取值的乘积与这些取值的差平方和的商，除以总体数量N。换言之，方差衡量了平均值的变异程度。当X的取值空间是R时，方差为Var(X)=E[(X-E[X])^2]/N。当X的取值空间是R^n时，方差为Var(X)=E[(X-E[X])·(X'-E[X'])]/(N-1)。
   
4. 连续型随机变量的积分定律：对连续型随机变量X，定义F(x)为定义域为(−∞,∞)的函数，F(x)表示X小于等于x的概率。则当a≤b时，有：
   F(b)-F(a)=P(X<b)-P(X<a),若X的概率密度函数是关于x的单调递增函数，则有：
   P(a≤X≤b)=F(b)-F(a).
   > 证明：1. 对任意x,有P(X<x)=∫_{-∞}^{x}f(t)dt。F(x)取到一个值后，函数的值恒定，所以F(b)-F(a)=P(X<b)-P(X<a)。当x趋近于a或b时，求导后可消掉x的影响。
   2. 如果X的概率密度函数是关于x的单调递增函数，则对任意x',有f(x')>=f(x),令t=max\{tx'+(1-t)x:x'<=x\},则有f(t)>=(1-t)f(x')+tf'(x'),因此F(t)=P(X≤t)=P((1-t)X'<t)+tf'(x').左边等于右边，根据概率的线性叠加法则，即F(b)-F(a)=P((1-t)X'<b)+(tb-ta)f'(x),左边是关于t的函数，右边是关于x的函数，故两边都关于t单调递减，所以F(b)-F(a)=F'(ta),这里的ta=(b-a)/2，所以F'(ta)<0。取极限可知，当a≤b时，F(b)-F(a)=0。

## 2.2 随机变量的分布函数（Distribution Function）
分布函数是指定义在某个定义域上的函数，其值表示对应于此点（或某些点）的概率。在概率论中，分布函数用于研究随机变量随时间的行为或演化。比如，若X为一次抛掷骰子的结果，则其分布函数代表了不同点数出现的概率。在研究随机变量的分布函数时，通常会采用概率密度函数或累积分布函数。
### 2.2.1 概率密度函数（Probability Density Function）
概率密度函数是一个函数，它描述了一个随机变量取值分布在某个给定区间内的概率。概率密度函数的图形呈现出变量取值的概率密集程度，所以也称为密度函数。概率密度函数常用来表示随机变量的概率密度，但也有一些特殊形式的概率密度函数，例如指数分布、正态分布等。
#### （1）定义域为连续型随机变量的概率密度函数
设X是连续型随机变量，其概率密度函数为f(x),x∈R, f(x)>=0.
1. 非负性：若X=a,则f(a)=0,否则f(x)>=0.
2. 规范性：存在非负常数μ,σ>0,使得对所有的x∈R,有：f(x)=p(x∣μ,σ)f(x−μ)/σ,其中p(x)是标准正态分布的概率密度函数。
3. 归一化性：对所有的x∈R,有sum_(-∞)^(∞)dx f(x)=1,即概率密度函数的积分为1.
4. 最大值定理：设X的概率密度函数为f(x),则f(x)的最大值发生在某个x*，记作μ*,有max_xf(x)=f(μ*)
5. 积分性：设X的概率密度函数为f(x),对任意a,b∈R,有：
   ∫a^bf(x)dx=F(b)-F(a) (2)
   其中，F(x)为X的累积分布函数。
#### （2）定义域为离散型随机变量的概率密度函数
设X为离散型随机变量，其概率质量函数为p(k),k=1,2,…,n.
1. 非负性：p(k)>=0.
2. 规范性：存在非负实数ξ1,…,ξn>0,使得对所有的i=1,2,…,n,有p(i)=π(xi),π(xi)是标准的多项式分布。
3. 归一化性：对所有的i=1,2,…,n,有sum_1^np(k)=1.
4. 期望值：E[X]=∑kp(k).
#### （3）典型的概率密度函数
##### （1）常见的连续型概率密度函数
1. 二元高斯分布：p(x)=1/sqrt(2πσ^2)exp(-(x-μ)^2/(2σ^2)), μ为期望值，σ为标准差。
2. 三角形分布：p(x)=|x|<=1, x=0时取值0.5。
3. 绝对值正态分布：p(x)=1/sqrt(2πσ^2)exp(-|x|^2/(2σ^2))
4. 伯努利分布：p(x)=p^kq^(1-q),其中p,q∈(0,1), k为成功次数，1-q为失效次数。
##### （2）常见的离散型概率质量函数
1. Bernoulli分布：p(k)=p^k(1-p)^{1-k}, p∈(0,1), k=0,1。
2. 二项分布：p(k)=C^k q^kq^(n-k)/k!, n为试验次数, C为组合常数。
3. 多项式分布：p(k)=C^k q^kq^(n-k)/k!, n为试验次数, C为组合常数。
4. 几何分布：p(k)=q^k(1-q)^{n-k}, n为试验次数, q为概率。
5. Poisson分布：p(k)=λ^ke^(-λ), λ为每单位时间的事件发生的次数，单位时间为某个固定长度T，λ=k/T。
6. 泊松分布：p(k)=λ^ke^(-λ)/k!，λ>0，k>=0。
7. 负二项分布：p(k)=C^k q^kq^(n-k)/k!, n为试验次数, C为组合常数。
