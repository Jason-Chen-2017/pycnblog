# 1.简介
  

## VR虚拟现实简介
虚拟现实(Virtual Reality,VR)一直都是人们最热门的话题之一。近年来随着VR设备的不断涌现，其应用也越来越广泛。传统的三维动画、电影制作等，已经逐渐转型为在虚拟现实环境中进行体验。通过在虚拟现实中呈现现实世界的内容，引起观众的注意，让他们产生共鸣和亲切感，增加互动性。
目前市面上主流的VR设备有HTC VIVE、OCULUS Rift、Samsung Gear VR等，其价格昂贵，同时对于各个VR设备的性能要求也不同，这也导致了很多公司、开发者为了提高性能而投入巨额资金。
## LUI项目介绍
LUI（Lenghi Universal Interface）是一个VR开源项目，旨在为开发者提供一套基于Unity3D的VR游戏引擎解决方案。LUI致力于提供一整套完整的VR解决方案，包括VR交互设计工具、VR虚拟现实开发框架、VR UI组件库、VR体验测试工具、VR商城模块等。并且，LUI项目还推出了一款名为‘Horizon Zero Dawn’的游戏Demo，即VR开源建筑和模拟游戏。
## LUI VR交互设计工具
LUI 提供了一套VR交互设计工具，能够帮助设计师快速创建符合规范的VR交互体验。该工具支持多种手势、手势组合、功能按钮、虚拟对象交互等交互方式，并提供了丰富的可视化编辑器，帮助设计师进行交互流程的设计和调试。

## LUI VR Virtual Reality Development Framework
LUI VR Virtual Reality Development Framework 是LUI项目的一个主要模块。它基于Unity3D开发，提供了VR的各项功能。其中包括：
- 支持主流VR硬件：目前已支持 HTC Vive 和 Oculus Rift VR 设备；
- 拥有完整的VR体验层：游戏内、VR外、虚拟现实头戴显示等；
- 完善的VR输入系统：可以识别用户的各种手势、控制器动作、头部姿态变化等；
- VR UI组件库：提供了丰富的常用UI组件，如按钮、滑条、标签等，开发者无需重复造轮子；
- VR音频插件：提供了基本的音效播放功能，还可以通过脚本控制音量、位置等；
- VR物理插件：允许开发者导入FBX文件，在VR场景中运动，可以模拟实景中的物理效果；
- 其他插件：如控制器驱动插件、天空盒插件、视角跟踪插件等；



## LUI VR User Interface Components Library
LUI VR User Interface Components Library 为开发者提供了一套完整的VR UI组件库。其中包括：
- 可视化编辑器：提供了UI组件的创建和配置界面，通过拖放的方式，可以轻松添加、删除UI组件，并对其属性进行设置；
- UI控件：如文本框、按钮、进度条、标签、开关、列表等，可以方便地实现各种类型的交互界面；
- VR特色动画：LUI支持多种可视化的动画，如透明度、缩放、旋转、平移等，可以满足各种类型的动画需求；
- 2D可视化编辑器：如果需要将UI组件用于2D场景，LUI提供了一个额外的可视化编辑器；
- VR UI集成：开发者也可以将自有的UI组件融入到VR UI组件库中，为VR游戏增添新的视觉效果；




## LUI VR Experience Testing Tools
LUI VR Experience Testing Tools 为开发者提供了一系列VR体验测试工具。其中包括：
- SteamVR虚拟现实监控器：可实时查看玩家的VR头显状态，方便开发者定位错误信息；
- 第三方VR模拟器：LUI项目推出了一款基于Unreal Engine的VR模拟器——HOUDINI HDRI，具有较高的真实度；
- SteamVR Overlay Renderer：可渲染VR界面的屏幕叠加层，可以测试多种类型的UI组件；





## LUI VR Shop Module
LUI VR Shop Module 是LUI项目的一个主要模块，主要提供VR商城功能。该模块目前处于开发阶段，后续将推出更多功能。包括：
- 用户管理系统：提供用户注册、登录、修改密码等功能；
- 商品管理系统：提供商品发布、上下架、编辑、删除等功能；
- 消费订单管理系统：提供消费订单查询、确认、退货等功能；
- VR物品交易平台：提供购买、出售、交换VR物品的平台；




# 2.基本概念术语说明
## 什么是虚拟现实？
虚拟现实(Virtual Reality,VR)是利用计算机仿真技术创建的一种全新艺术、科技形式。它是一种让用户沉浸在虚拟环境中进行三维活动、交互的全新体验。虚拟现实技术的应用范围从娱乐到医疗、教育、商业等领域均十分广泛。
## 虚拟现实的历史
1997 年，日本的Nikola Puzzles首次推出了第一代虚拟现实系统。这一系统采用一块卡口摄像头，模拟移动物体的运动，使得观看者看到一个四方的空间，这种现实很逼真，但无法与现实世界完全匹配。
1999 年，美国的惠普实验室开发出一套基于Windows的虚拟现实系统Oculus VR。它的最大特点是虚拟现实镜头采用了“沉浸式”模式，让用户无需离开实际的现实世界就能进入虚拟空间。
2008 年，索尼推出PSVR，是第一款支持虚拟现实的电子游戏。它包括虚拟现实头戴装置，可以在户外玩各种沉浸式的游戏。
2011 年，Apple推出iPhone，配备了由3D摄像头组成的AR眼镜，使得人们可以像使用真实世界一样使用虚拟世界。
2014 年，任天堂推出了Wii U，这是第一个支持虚拟现实的游戏机。它采用两张投影屏幕，将虚拟画面投射到大屏幕上，让玩家在虚拟世界中生活。
2016 年，微软推出了HoloLens，这是一台兼顾效率和创新性的全新产品。它集成了深度相机、麦克风、陀螺仪、屏幕等硬件，并配备有高清晰度的头戴显示屏。
2017 年，Facebook开启了VR开发者社区，其开发者们计划在这个领域投入巨资。
2018 年，腾讯宣布其即将与Facebook合作，构建分布式VR产业。
## VR头显
VR头显，通常指的是由物理显示设备和虚拟技术构成的全新显示技术。它能够让用户以更具想象力和 immersive 的形式享受 VR 体验。
VR头显通常包括以下几个组成部分：
- 显示设备：通过显示设备上的物理技术，将图像投射到用户眼睛前，达到虚拟现实的效果。
- 处理器：负责对图像进行处理，如采集、同步、显示、分析等。
- 传感器：负责捕捉头部运动、姿态、体温、环境光线等信息。
- 内存：用来存储VR头显相关的算法和数据。
- 通信接口：用于连接计算机、手机或其他VR设备。
## 什么是VRP（Virtual Reality Platforms）？
VRP（Virtual Reality Platforms）一般指的是支持虚拟现实（VR）的平台，比如Oculus VR、HTC Vive、Microsoft HoloLens等。这些平台通过提供底层硬件接口、运行时环境、图形渲染、输入输出接口等服务，使得开发者能够快速、低成本地开发VR应用。
