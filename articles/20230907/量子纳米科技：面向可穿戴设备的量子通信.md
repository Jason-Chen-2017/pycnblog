# 1.简介
  

## 1.1 产品背景
随着近几年来的智能手表、智能手环、智能健身环等越来越多地进入人们的生活，越来越多的人开始关注人工智能领域对于人类的影响。其中，在传统的电信通信领域已经应用了太多的物理信道，并且无处不在，导致通信距离很远。如今，数字通信技术在智能手表、智能手环、智能健身环等设备上广泛运用，但是也产生了一些新的通信难题。

由于某些原因（如功耗过高、信号衰减、天线开口不够大），使得传统的电信号无法再满足需求。于是在2017年，量子通信被提出来，它可以解决传统通信中存在的问题，其优点在于可以突破时间和空间的限制，能够提供更大的带宽。

## 1.2 产品特色
- 使用了量子通信，解决了传统通信存在的性能瓶颈，能提升通信距离。
- 可穿戴设备，用户可以将其安装在自己的手腕、肩膀或其他任何可以承受重量的地方。
- 支持WiFi、蓝牙、Zigbee等传统通信方式，适用于不同场景。
- 更大的数据传输容量，可发送更大的数据包。

## 1.3 市场估值
根据《中国互联网行业报告2019》显示，2018年全国消费者对可穿戴设备的预期月增长率达到15.3%，有望超越可穿戴互联网，成为新的核心零售方式。

据报道，目前可穿戴通信领域共有超过两万家公司，同比增长超过80％。其中，华为、苹果、微软、OPPO、小米等巨头均在可穿戴通信领域占有重要份额，占比分别达到28.1%、20.4%、16.3%、14.3%和13.9%。

根据IDC发布的2019年第三季度预测，可穿戴通信将占2.16亿个制造商、2160万个手机品牌、3780万个智能手表、2700万个智能手环和1600万个智能健身环等一体化消费领域，并推动制造业的创新升级。
