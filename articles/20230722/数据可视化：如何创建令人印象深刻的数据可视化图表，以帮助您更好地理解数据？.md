# 1.简介

# 2 数据可视化的定义及其分类
数据可视化(Data Visualization)是指将数据以图表形式展现出来，它作为对数据的一种直观表达，是通过图形的方式展现数据特点、规律、变化，用于发现事物间的关系和特征，启发我们洞察复杂的信息。数据可视化也是计算机科学领域的一个分支，主要涉及计算机视觉、信息论、数学统计等多个领域。

目前，数据可视化的定义较为宽泛。根据数据可视化的实际应用，可分为两类：

1. 关于数据的呈现，如通过图表和图像来呈现数据，使得数据更容易被人们所认识和理解；
2. 关于数据的分析，如通过比较和比较两个或多个数据集之间的差异，或者绘制某种统计图表展示数据趋势、分布情况、相关性等，用来提升分析师和决策者对数据的理解、分析、判断能力，进而做出更多有效的决策。

另外，数据可视化可以按照数据来源的不同分类：

1. 结构化数据可视化：包括数据模型中明显具有结构化特征的数据，如数据库中的关系型数据，例如，客户信息、产品销售记录、订单记录等。这种数据通常需要按照一定的逻辑规则或模式进行分析，可以直接利用图表和图形表示出来；
2. 非结构化数据可视化：包括网络数据、文本数据等，这些数据没有预先设定好的结构，也没有严格的行列关系，通常需要先进行文本处理、信息提取、分析之后才能进行可视化；
3. 模型数据可视化：包括经过机器学习建模得到的结果数据，这些数据需要结合其他数据进行分析和解释，通常用作预测、监控等用途。

除了以上三类数据可视化方式外，还有一种即时可视化技术叫做快速响应可视化（Reactive Visual Analytics），它借鉴了传感器技术的一些特性，将实时的输入数据转化成图表、图形、图像甚至声音，帮助用户更快、更直观地了解并分析数据。由于快速响应可视化还处于初期阶段，相关理论尚未成熟，因此，对于它的讨论主要局限在理论上，而不会有太多的实际应用案例。

综上所述，数据可视化是指将数据以图表形式展现出来，从而增强对数据的理解、把握，促进业务决策。不同的数据来源会选择不同的可视化方式，如结构化数据可视化主要依赖图表，而非结构化数据可视化依赖树状图、脑图等方式进行可视化。同时，数据可视化需要关注可视化效果的评价标准，只有确保满足相应的标准才能够真正的实现业务价值。
