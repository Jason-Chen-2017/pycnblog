# 2.核心概念与联系

## 2.1数据分析

数据分析是指通过对数据进行清洗、转换、汇总、比较和模型建立等操作，从中抽取有用信息，并用以支持决策的过程。数据分析可以帮助我们找出数据中的趋势、模式和异常，从而更好地理解数据。

## 2.2数据可视化

数据可视化是指将数据以图形、图表或其他视觉方式表示的过程。数据可视化可以帮助我们更直观地理解数据，从而更好地做出决策。

## 2.3数据分析与可视化的联系

数据分析和数据可视化是相互联系的。数据分析是对数据进行深入分析的过程，而数据可视化是将分析结果以图形、图表等形式呈现的过程。数据分析和数据可视化共同构成了数据分析和可视化的整个过程。

# 4.具体代码实例和详细解释说明

## 4.1数据清洗

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 去除缺失值
data = data.dropna()

# 填充缺失值
data['age'] = data['age'].fillna(data['age'].mean())

# 转换格式
data['age'] = data['age'].astype(int)
```

## 4.2数据汇总

```python
# 求和
sum_age = data['age'].sum()

# 求平均值
mean_age = data['age'].mean()
```

## 4.3数据比较

```python
# 比较大小
max_age = data['age'].max()
min_age = data['age'].min()

# 比较相等
equal_age = data['age'] == data['age'].mean()
```

## 4.4数据模型建立

```python
# 线性回归
from sklearn.linear_model import LinearRegression

# 训练模型
model = LinearRegression().fit(data[['age']], data['salary'])
```
