# 2.核心概念与联系

在博弈论中，我们需要关注两个主要概念：策略和结果。策略是智能体在不同情况下采取的行为规则，结果是智能体采取不同策略后的最终效果。博弈论的核心思想是通过分析不同策略的结果，来预测和优化智能体在不同情况下的行为。

博弈论可以分为两种类型：完全信息博弈和不完全信息博弈。完全信息博弈是指所有智能体都知道对方的策略和行为，而不完全信息博弈是指智能体之间存在信息不完全或者隐藏的情况。博弈论还可以分为两种类型：零和博弈和非零和博弈。零和博弈是指智能体之间的利益是相互独立的，而非零和博弈是指智能体之间的利益是相互影响的。

博弈论与人工智能的联系主要体现在以下几个方面：

1. 博弈论可以用来分析智能体之间的竞争关系，帮助人工智能系统更好地理解和预测对方的行为。
2. 博弈论可以用来设计智能体的策略，帮助人工智能系统更好地决策和行动。
3. 博弈论可以用来优化智能体之间的协作关系，帮助人工智能系统更好地协同工作。
