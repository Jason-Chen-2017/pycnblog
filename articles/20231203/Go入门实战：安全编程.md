## 4.2 并发安全示例

```go
package main

import "fmt"

func main() {
    var wg sync.WaitGroup
    wg.Add(1)
    go func() {
        defer wg.Done()
        fmt.Println("Hello, World!")
    }()
    wg.Wait()
}
```

在这个示例中，我们使用了Go语言的并发原语goroutine来编写并发安全的代码。我们创建了一个sync.WaitGroup变量wg，并使用Add方法来表示我们将要启动一个goroutine。然后，我们使用go关键字来启动一个匿名函数，这个函数会在另一个goroutine中执行。在这个匿名函数中，我们使用defer关键字来确保在函数执行完成后调用wg.Done()方法。最后，我们使用wg.Wait()方法来等待所有的goroutine完成后再继续执行。
