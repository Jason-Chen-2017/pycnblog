## 3.1 最大似然估计
最大似然估计是一种用于估计参数的方法，它的基本思想是找到使数据概率最大的参数值。假设我们有一个随机变量X，它的概率密度函数为f(x|θ)，其中θ是参数。我们有一组观测数据x1, x2, ..., xn，我们可以计算出这组数据的概率为：

L(θ) = P(x1, x2, ..., xn|θ) = Π f(xi|θ)

我们的目标是找到使L(θ)最大的θ值。这个过程可以通过计算L(θ)的对数来简化，因为对数函数是单调增加的。我们可以使用梯度下降法或其他优化方法来找到θ的最大值。

## 3.2 参数估计
参数估计是一种用于估计参数的方法，它的基本思想是找到使数据的可能性最大的参数值。假设我们有一个随机变量X，它的概率密度函数为f(x|θ)，其中θ是参数。我们有一组观测数据x1, x2, ..., xn，我们可以计算出这组数据的似然性为：

L(θ) = P(x1, x2, ..., xn|θ) = Π f(xi|θ)

我们的目标是找到使L(θ)最大的θ值。这个过程可以通过计算L(θ)的对数来简化，因为对数函数是单调增加的。我们可以使用梯度下降法或其他优化方法来找到θ的最大值。

我们可以使用Python的numpy和scipy库来实现这个过程。首先，我们需要定义我们的数据和模型：

```python
import numpy as np
from scipy.stats import norm

# 定义数据
x = np.array([1, 2, 3, 4, 5])

# 定义模型
mu = np.mean(x)
sigma = np.std(x)
```

接下来，我们可以使用Python的scipy库来计算数据的似然性：

```python
# 计算似然性
likelihood = norm.logpdf(x, loc=mu, scale=sigma)
```

最后，我们可以使用Python的numpy库来计算θ的最大值：

```python
# 计算θ的最大值
max_theta = np.argmax(likelihood)
```
