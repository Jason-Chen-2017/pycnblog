### 4.1 创建一个新的 Spring Boot 项目

### 4.2 配置 Spring Security

接下来，我们需要配置 Spring Security。我们可以在项目的 `application.properties` 文件中添加以下配置：

```
spring.security.user.name=user
spring.security.user.password=password
```

这将创建一个用户名为 `user` 和密码为 `password` 的用户。

### 4.3 创建一个新的 REST 控制器

接下来，我们需要创建一个新的 REST 控制器。我们可以在项目的 `src/main/java` 目录中创建一个新的 Java 文件，并将其命名为 `SecurityController.java`。在这个文件中，我们可以添加以下代码：

```java
import org.springframework.security.core.context.SecurityContextHolder;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class SecurityController {

    @GetMapping("/")
    public String index() {
        return "Hello World!";
    }

    @GetMapping("/secure")
    public String secure() {
        Object principal = SecurityContextHolder.getContext().getAuthentication().getPrincipal();
        if (principal instanceof org.springframework.security.core.userdetails.User) {
            return "Secure page!";
        } else {
            return "Access denied!";
        }
    }
}
```

这个 REST 控制器包含两个端点：

- `/`：这个端点返回一个 "Hello World!" 的字符串。
- `/secure`：这个端点检查用户是否已经登录。如果用户已经登录，则返回一个 "Secure page!" 的字符串。否则，返回一个 "Access denied!" 的字符串。
