1. 将问题空间划分为多个小区域。
2. 选择基函数。
3. 将问题函数和微分项表示为有限个基函数的线性组合。
4. 将问题中的微分项替换为基函数的微分。
5. 将问题中的积分替换为基函数的积分。
6. 将问题中的微分方程转换为线性方程组。
7. 解决线性方程组。

$$
\sum_{i=1}^{N} c_i \int_{\Omega} \nabla N_i \cdot \nabla N_j dx = \int_{\Omega} f N_j dx

# 定义有限差分方法
def solve_pde(x, T, k, f):
    u = np.zeros((N+1, N+1))
    u[0, :] = 0.0
    u[-1, :] = 0.0
    for t in range(N):

    return u

# 解决问题
u = solve_pde(x, T, k, f)
