# 2.核心概念与联系

## 2.1 错误处理与恢复策略的定义

错误处理与恢复策略是指软件系统在发生错误时采取的措施，以确保系统能够继续正常运行或恢复到正常状态。错误处理与恢复策略可以分为以下几种：

1. 错误捕获：在运行时捕获错误，以便能够在错误发生时采取相应的措施。
2. 错误传播：将错误信息传递给相关的组件或系统，以便能够在错误发生时采取相应的措施。
3. 错误恢复：在错误发生时采取措施，以便能够恢复到正常状态。
4. 错误处理：在错误发生时采取措施，以便能够处理错误并继续运行。
