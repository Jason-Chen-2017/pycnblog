## 4.1 创建一个简单的游戏
我们将创建一个简单的猜数字游戏。在这个游戏中，用户需要猜测一个随机生成的数字，游戏会给出提示，告诉用户猜测的数字是大于还是小于猜测的数字。

```python
import random

def generate_random_number():
    return random.randint(1, 100)

def get_user_guess():
    while True:
        try:
            guess = int(input("请输入你的猜测："))
            if 1 <= guess <= 100:
                return guess
            else:
                print("请输入1到100之间的数字")
        except ValueError:
            print("输入错误，请重新输入")

def check_guess(guess, target):
    if guess < target:
        return "小于"
    elif guess > target:
        return "大于"
    else:
        return "等于"

def main():
    target = generate_random_number()
    guess = get_user_guess()
    while guess != target:
        print(f"你猜测的数字 {guess} {check_guess(guess, target)} 目标数字")
        guess = int(input("请输入你的下一次猜测："))
    print("恭喜你，猜对了！目标数字是 {}".format(target))

if __name__ == "__main__":
    main()
```

在这个例子中，我们首先使用`random.randint(1, 100)`函数生成一个随机数字作为目标数字。然后，我们使用`get_user_guess()`函数获取用户的猜测。接着，我们使用`check_guess()`函数判断用户的猜测是否正确，并给出提示。最后，我们使用`while`循环来判断用户的猜测是否正确，直到用户猜对目标数字为止。
