
2. 添加文件和目录：

```python

```

3. 提交版本：

```python

```

4. 查看版本历史：

```python
for commit in repo.iter_commits():
    print(commit.message)
```

5. 回滚到特定版本：

```python
repo.git.checkout('HEAD~1')
```

6. 合并版本：

```python
repo.git.merge('master')
```

7. 解决冲突：

```python
with open('my_file.txt', 'r') as f:
    content = f.read()

repo.git.checkout('--', 'my_file.txt')
repo.git.reset('--', 'my_file.txt')
repo.git.checkout('master', 'my_file.txt')

with open('my_file.txt', 'w') as f:
    f.write(content)
```
