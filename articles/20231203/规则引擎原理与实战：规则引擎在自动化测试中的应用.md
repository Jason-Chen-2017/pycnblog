# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释规则引擎的实现方法。

```python
# 定义规则
def rule1(input):
    if input > 10:
        return True
    else:
        return False

# 定义测试用例
test_cases = [
    {"input": 5, "expected": False},
    {"input": 10, "expected": True},
    {"input": 15, "expected": True},
]

# 执行规则
results = []
for case in test_cases:
    result = rule1(case["input"])
    results.append({"input": case["input"], "expected": case["expected"], "result": result})

# 判断结果
for result in results:
    if result["result"] == result["expected"]:
        print("通过")
    else:
        print("失败")
```

在上述代码中，我们首先定义了一个规则`rule1`，它接受一个输入参数并返回一个布尔值。然后，我们定义了一组测试用例`test_cases`，每个测试用例包括一个输入参数和一个预期结果。接下来，我们执行了规则`rule1`，并将结果存储在`results`列表中。最后，我们判断每个测试用例的结果是否与预期结果一致，并输出相应的结果。
