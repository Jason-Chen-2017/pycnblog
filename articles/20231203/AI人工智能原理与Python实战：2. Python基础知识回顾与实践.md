人工智能（Artificial Intelligence，AI）是计算机科学的一个分支，研究如何使计算机能够像人类一样思考、学习、决策和解决问题。人工智能的目标是创建智能机器人，这些机器人可以理解自然语言、学习新知识、自主决策、理解环境、进行自主行动、进行自主思考、进行自主学习、进行自主创造等。
