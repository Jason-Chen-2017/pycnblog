                 

# 1.背景介绍

自动机器学习（AutoML）是一种自动化的机器学习方法，它旨在帮助用户在没有专业知识的情况下，快速构建高性能的机器学习模型。自动机器学习的核心思想是自动化地选择最佳的机器学习算法，以及调整算法的参数，从而实现对数据的自动化分析和预测。

自动机器学习的发展背景主要包括以下几个方面：

1. 数据量的快速增长：随着数据的快速增长，人们需要更快地构建机器学习模型，以满足业务需求。自动机器学习可以帮助用户快速构建高性能的机器学习模型，从而满足业务需求。

2. 算法复杂度的增加：随着机器学习算法的不断发展和增加，人们需要更多的专业知识来选择和调整算法。自动机器学习可以自动化地选择最佳的机器学习算法，以及调整算法的参数，从而降低人工成本。

3. 人工智能的发展：随着人工智能的发展，机器学习成为了人工智能的重要组成部分。自动机器学习可以帮助人工智能系统快速构建高性能的机器学习模型，从而提高人工智能系统的性能。
