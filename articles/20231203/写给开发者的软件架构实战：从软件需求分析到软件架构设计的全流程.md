```python
# 定义一个简单的数据结构
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

# 定义一个简单的算法
def insert(head, data):
    new_node = Node(data)
    new_node.next = head
    head = new_node
    return head

# 测试代码
head = None
head = insert(head, 1)
head = insert(head, 2)
head = insert(head, 3)
```

在这个代码实例中，我们定义了一个简单的数据结构`Node`，并实现了一个简单的算法`insert`。这个算法用于在链表的头部插入一个新的节点。
