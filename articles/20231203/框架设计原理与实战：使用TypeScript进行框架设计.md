## 2.核心概念与联系

在TypeScript框架设计中，有几个核心概念需要理解：

3. 泛型：泛型是TypeScript中的一种编程技术，它允许开发者创建可重用的组件，这些组件可以处理不同类型的数据。泛型可以帮助开发者编写更灵活和可重用的代码。

首先，我们需要定义一个接口来描述计算器的形状：

```typescript
interface Calculator {
  add(a: number, b: number): number;
  subtract(a: number, b: number): number;
  multiply(a: number, b: number): number;
  divide(a: number, b: number): number;
}
```

接下来，我们可以创建一个实现了Calculator接口的类来实现计算器的基本功能：

```typescript
class BasicCalculator implements Calculator {
  add(a: number, b: number): number {
    return a + b;
  }

  subtract(a: number, b: number): number {
    return a - b;
  }

  multiply(a: number, b: number): number {
    return a * b;
  }

  divide(a: number, b: number): number {
    return a / b;
  }
}
```
