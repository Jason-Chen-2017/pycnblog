## 4.1 数据预处理

我们可以使用Python的NumPy库来进行数据预处理。以下是一个数据清洗的代码实例：

```python
import numpy as np

# 数据清洗
data = np.array([1, 2, np.nan, 4, 5])
data_clean = np.where(np.isnan(data), np.mean(data), data)
print(data_clean)
```

在这个代码实例中，我们使用NumPy的`np.isnan()`函数来检查数据中是否存在缺失值，并使用`np.where()`函数来填充缺失值。
