# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的命名实体识别和信息抽取任务来展示Python代码实例。

## 4.1 命名实体识别

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.tag import pos_tag

def named_entity_recognition(text):
    tokens = word_tokenize(text)
    tagged_tokens = pos_tag(tokens)
    named_entities = []

    for token, tag in tagged_tokens:
        if tag == 'NNP':
            named_entities.append(token)

    return named_entities

text = "Barack Obama is the 44th President of the United States."
named_entities = named_entity_recognition(text)
print(named_entities)
```

在这个代码中，我们使用了NLTK库来实现命名实体识别。我们首先将文本分词，然后将分词结果与词性标签相对应。接着，我们遍历标记结果，找到所有的名词单数（NNP），并将它们作为命名实体返回。
