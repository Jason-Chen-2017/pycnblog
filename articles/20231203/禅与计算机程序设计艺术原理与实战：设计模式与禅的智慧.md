   ```python
   def knapsack(items, capacity):
       max_value = 0
       for item in items:
           if item[1] <= capacity:
               max_value += item[0]
               capacity -= item[1]
       return max_value
   ```

   在这个代码实例中，我们定义了一个贪心算法问题 `knapsack`，它用于计算背包中最大的价值。我们选择最优解的具体操作步骤如下：

   1. 遍历所有物品。
   2. 如果物品的重量小于或等于背包的容量，则将物品的价值加入到最大价值中，并将背包的容量减少。
   3. 重复步骤 2，直到背包的容量为 0。
   4. 返回最大价值。
