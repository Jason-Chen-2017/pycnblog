## 4.1 递归算法实例

递归算法是一种通过函数调用自身来实现的算法，它是指一种通过函数调用自身来实现的算法。递归算法的核心思想是将一个复杂的问题拆分成多个简单的问题，然后逐步解决这些简单的问题。

以下是一个递归算法的实例：

```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)
```

在这个实例中，我们定义了一个名为`factorial`的函数，它用于计算一个数的阶乘。函数的参数`n`表示要计算阶乘的数。

函数的实现是通过递归的方式来实现的。当`n`等于0时，函数返回1。否则，函数返回`n`乘以`factorial(n-1)`的结果。

通过递归的方式，我们可以计算任意一个数的阶乘。

以下是一个贪心算法的实例：

```python
def coin_change(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0

    for i in range(1, amount + 1):
        for coin in coins:
            if coin <= i:
                dp[i] = min(dp[i], dp[i - coin] + 1)

    return dp[amount]
```

函数的实现是通过贪心的方式来实现的。我们首先初始化一个长度为`amount + 1`的列表`dp`，其中每个元素都初始化为`float('inf')`。然后，我们遍历每个金额，对于每个金额，我们遍历每个硬币，如果硬币小于等于当前金额，我们更新`dp`列表中当前金额的值为最小值。

通过贪心的方式，我们可以计算给定硬币组成的硬币组合的最小数量。
