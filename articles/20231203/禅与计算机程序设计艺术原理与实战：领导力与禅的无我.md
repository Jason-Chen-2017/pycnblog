## 4.1 模块化设计原则的应用

# 模块化设计原则的应用

# 定义一个计算器模块
class Calculator:
    def add(self, a, b):
        return a + b

    def subtract(self, a, b):
        return a - b

# 定义一个数学模型模块
class MathModel:
    def solve_quadratic_equation(self, a, b, c):
        discriminant = (b ** 2) - (4 * a * c)
        if discriminant >= 0:
            x1 = (-b + (discriminant ** 0.5)) / (2 * a)
            x2 = (-b - (discriminant ** 0.5)) / (2 * a)
            return x1, x2
        else:
            return None

# 使用计算器模块和数学模型模块
calculator = Calculator()
math_model = MathModel()

result = calculator.add(math_model.solve_quadratic_equation(1, -2, 1)[0], 1)
print(result)  # 输出: 2.0
```

# 抽象设计原则的应用

# 定义一个抽象类
class AbstractClass:
    def abstract_method(self):
        pass

# 定义一个具体类
class ConcreteClass(AbstractClass):
    def abstract_method(self):
        return "Hello, World!"

# 使用抽象类和具体类
abstract_class_instance = AbstractClass()
concrete_class_instance = ConcreteClass()

print(abstract_class_instance.abstract_method())  # 输出: 无
print(concrete_class_instance.abstract_method())  # 输出: Hello, World!
```

在这个代码实例中，我们将一个复杂的问题抽象成一个简单的问题，即一个抽象类和一个具体类。抽象类定义了一个抽象方法，具体类实现了这个抽象方法。通过这种抽象设计原则的应用，我们可以更好地理解和应用计算机程序的设计原则。
