## 1.排序算法的原理和步骤

排序算法是计算机程序设计中的一种常用算法，用于对数据进行排序。排序算法的原理主要包括：

- 比较排序：比较排序是指通过比较两个元素的关键字来决定它们的顺序的排序算法，如冒泡排序、快速排序等。
- 交换排序：交换排序是指通过交换两个元素的位置来决定它们的顺序的排序算法，如插入排序、选择排序等。

排序算法的具体操作步骤如下：

1. 比较排序：
   - 冒泡排序：
     - 从第一个元素开始，与后续的每个元素进行比较，如果当前元素大于后续元素，则交换它们的位置。
     - 重复上述步骤，直到整个数组被排序。
   - 快速排序：
     - 从数组的第一个元素开始，选择一个基准元素。
     - 将数组中小于基准元素的元素放在基准元素的左侧，大于基准元素的元素放在基准元素的右侧。
     - 对左侧和右侧的子数组重复上述步骤，直到整个数组被排序。
2. 交换排序：
   - 插入排序：
     - 从第一个元素开始，将其与后续的每个元素进行比较。
     - 如果当前元素小于后续元素，则将其与后续元素进行交换。
     - 重复上述步骤，直到整个数组被排序。
   - 选择排序：
     - 从数组的第一个元素开始，找到最小的元素。
     - 将最小的元素与当前元素进行交换。
     - 重复上述步骤，直到整个数组被排序。

## 2.搜索算法的原理和步骤

搜索算法是计算机程序设计中的一种常用算法，用于在数据结构中查找特定的元素。搜索算法的原理主要包括：

- 顺序搜索：顺序搜索是指从数据结构的第一个元素开始，逐个比较每个元素，直到找到目标元素或者遍历完整个数据结构为止。
- 二分搜索：二分搜索是指在有序数据结构中，将数据结构分为两个部分，然后比较目标元素与中间元素的关键字，根据比较结果将搜索范围缩小到相应的部分。

搜索算法的具体操作步骤如下：

1. 顺序搜索：
   - 从数据结构的第一个元素开始，逐个比较每个元素。
   - 如果当前元素等于目标元素，则找到目标元素并结束搜索。
   - 如果当前元素小于目标元素，则将搜索范围缩小到后续的元素。
   - 重复上述步骤，直到找到目标元素或者遍历完整个数据结构为止。
2. 二分搜索：
   - 将数据结构分为两个部分，左侧和右侧。
   - 比较目标元素与中间元素的关键字。
   - 如果目标元素等于中间元素，则找到目标元素并结束搜索。
   - 如果目标元素小于中间元素，则将搜索范围缩小到左侧的部分。
   - 如果目标元素大于中间元素，则将搜索范围缩小到右侧的部分。
   - 重复上述步骤，直到找到目标元素或者搜索范围缩小到空为止。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

解释：

- 从第一个元素开始，与后续的每个元素进行比较，如果当前元素大于后续元素，则交换它们的位置。
- 重复上述步骤，直到整个数组被排序。

### 1.2 快速排序

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)
```

解释：

- 从数组的第一个元素开始，选择一个基准元素。
- 将数组中小于基准元素的元素放在基准元素的左侧，大于基准元素的元素放在基准元素的右侧。

```python
    right = len(arr) - 1
    while left <= right:
```

```python
import heapq

def dijkstra(graph, start):

    dist = [float('inf')]*n
    dist[start] = 0
    queue = [(0, start)]
    while queue:
        d, u = heapq.heappop(queue)
```
