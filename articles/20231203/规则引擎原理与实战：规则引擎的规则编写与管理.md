# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来演示规则引擎的实现过程：

```python
# 定义规则库
rules = [
    {
        "condition": "age >= 18",
        "action": "grant_access"
    },
    {
        "condition": "age < 18",
        "action": "deny_access"
    }
]

# 初始化工作内存
work_memory = {"age": 20}

# 执行规则
for rule in rules:
    if eval(rule["condition"]):
        rule["action"]()
    else:
        continue

# 输出结果
print(work_memory)  # {"age": 20, "access": "grant_access"}
```

在这个例子中，我们定义了一个简单的规则库，包括两个规则。每个规则包括一个条件和一个操作。我们将规则库加载到内存中，并初始化工作内存。然后，我们遍历规则库，根据工作内存中的数据来执行规则。最后，我们输出工作内存的状态。
