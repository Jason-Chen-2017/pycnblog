# 2.核心概念与联系

在处理可重用性问题时，我们需要了解以下几个核心概念：

- **可重用性**：可重用性是指在不同的上下文中可以重复使用的内容。在提示中，可重用性可以是指可以重复使用的问题、答案或其他信息。
- **提示**：提示是指向大规模语言模型的输入，用于指导模型生成特定类型的输出。
- **上下文**：上下文是指提示中包含的背景信息，可以是问题、答案或其他相关信息。

在处理可重用性问题时，我们需要考虑以下几个方面：

- **可重用性的识别**：我们需要识别出可以重复使用的信息，以便在提示中进行重用。
- **可重用性的组织**：我们需要将可重复使用的信息组织成有意义的结构，以便在提示中进行重用。
- **可重用性的应用**：我们需要在提示中应用可重复使用的信息，以便更好地指导大规模语言模型生成所需的输出。
