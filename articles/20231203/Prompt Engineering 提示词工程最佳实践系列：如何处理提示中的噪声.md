# 2.核心概念与联系

在处理提示中的噪声时，我们需要了解以下几个核心概念：

1. **自然语言处理（NLP）**：NLP是一种通过计算机程序处理自然语言的技术，涉及到语音识别、语义分析、文本生成等多个方面。

2. **大规模语言模型（LLM）**：LLM是一种基于深度学习的模型，可以根据输入的文本生成相关的文本。GPT-3和GPT-4等模型都属于LLM的范畴。

3. **提示（Prompt）**：提示是指向模型的输入文本，用于指导模型生成特定的输出文本。

4. **噪声（Noise）**：噪声是指提示中的干扰信息，可能来自于语法错误、拼写错误、语义不清晰等。

5. **清洗（Cleaning）**：清洗是指通过删除、修改或添加提示中的信息，以减少噪声的过程。

6. **过滤（Filtering）**：过滤是指通过设定一定的规则，从提示中删除包含噪声的部分的过程。

7. **生成（Generation）**：生成是指通过模型生成文本的过程。
