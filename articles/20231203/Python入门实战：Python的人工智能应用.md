1.8.6 梯度下降：
```python
import numpy as np

# 定义损失函数
def loss(w, X, y):
    return np.mean((X @ w - y) ** 2)

# 定义梯度
def grad(w, X, y):
    return X.T @ (X @ w - y)

# 定义优化函数
def optimize(w, X, y, learning_rate, num_iterations):
    for _ in range(num_iterations):
        grad_w = grad(w, X, y)
        w = w - learning_rate * grad_w
    return w

# 训练模型
w = optimize(np.zeros(X.shape[1]), X, y, learning_rate=0.01, num_iterations=1000)
```

1.8.7 反向传播：
```python
import numpy as np

# 定义损失函数
def loss(w, X, y):
    return np.mean((X @ w - y) ** 2)

# 定义梯度
def grad(w, X, y):
    return X.T @ (X @ w - y)

# 定义优化函数
def optimize(w, X, y, learning_rate, num_iterations):
    for _ in range(num_iterations):
        grad_w = grad(w, X, y)
        w = w - learning_rate * grad_w
        loss_value = loss(w, X, y)
    return w, loss_value

# 训练模型
w, loss_value = optimize(np.zeros(X.shape[1]), X, y, learning_rate=0.01, num_iterations=1000)
```
