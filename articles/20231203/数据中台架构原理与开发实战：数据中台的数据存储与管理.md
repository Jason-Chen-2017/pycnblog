# 2.核心概念与联系

在数据中台架构中，数据存储与管理是其核心组件之一，它负责对企业内部和外部的各种数据资源进行存储、管理、维护等操作，为其他数据中台组件提供数据支持。数据存储与管理的核心概念包括：

- 数据存储：数据存储是指将企业内部和外部的各种数据资源存储到适当的存储设备上，以便在需要时进行读取和写入操作。数据存储可以分为本地存储、远程存储、分布式存储等多种类型。

- 数据管理：数据管理是指对数据存储的各种操作进行规范化管理，包括数据的收集、清洗、整合、分析、应用等环节。数据管理的目的是为了保证数据的质量、安全、可靠性等方面的要求。

- 数据存储与管理的联系：数据存储与管理是相互联系的，数据存储是数据管理的基础，数据管理是数据存储的必要条件。数据存储提供了数据的存储支持，数据管理提供了数据的规范化管理。

2. 数据清洗：对收集到的数据进行清洗操作，包括数据的去重、数据的填充、数据的转换等环节。

5. 数据应用：对分析后的数据进行应用操作，包括数据的报表、数据的预测、数据的推荐等环节。
