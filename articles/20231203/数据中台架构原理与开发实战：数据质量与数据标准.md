### 数据清洗

数据清洗是一种常见的数据预处理技术，它用于处理数据中的错误和不完整性。以下是一个简单的数据清洗示例：

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 处理缺失值
data = data.fillna(data.mean())

# 处理错误值
data = data[data['age'] >= 0]

# 处理噪声
data = data[data['temperature'] <= 100]

# 保存数据
data.to_csv('cleaned_data.csv', index=False)
```

在这个示例中，我们使用了pandas库来读取和处理数据。我们首先读取了一个CSV文件，然后使用fillna函数处理了缺失值，使用了大于等于0的值处理了错误值，并使用了小于等于100的值处理了噪声。最后，我们将处理后的数据保存到一个新的CSV文件中。

### 数据标准化

data = pd.read_csv('data.csv')
