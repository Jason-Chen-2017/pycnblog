## 4.1 规则引擎的代码实例

以Python语言为例，下面是一个简单的规则引擎的代码实例：

```python
class RuleEngine:
    def __init__(self, rules):
        self.rules = rules

    def match(self, data):
        for rule in self.rules:
            if rule.match(data):
                return rule
        return None

    def execute(self, data):
        rule = self.match(data)
        if rule:
            rule.execute(data)

data = {"age": 20, "gender": "male"}
engine = RuleEngine([
    Rule(lambda x: x["age"] > 18, lambda x: print("满18岁")),
    Rule(lambda x: x["gender"] == "male", lambda x: print("是男性"))
])
engine.execute(data)
```
