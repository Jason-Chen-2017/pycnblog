```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(arr))
```

在这个代码实例中，我们首先定义了一个名为`bubble_sort`的函数，它接受一个数组作为参数。然后，我们使用两层循环来遍历数组中的每个元素，并将相邻的元素进行交换。最后，我们返回排序后的数组。通过这个简单的例子，我们可以看到禅意编程的思想是如何应用到编程中的。
