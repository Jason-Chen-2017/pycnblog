# 2.核心概念与联系
在图像处理中，我们需要了解一些基本的概念，如像素、图像矩阵、图像通道、图像滤波、图像变换等。这些概念将帮助我们更好地理解图像处理的原理和方法。

## 2.1 像素
像素（picture element）是图像的基本单元，它表示图像的颜色和亮度值。像素通常以行和列的形式组织，形成一个矩阵。每个像素都有一个坐标（行和列），用于标识其在图像中的位置。

## 2.2 图像矩阵
图像矩阵是一个二维矩阵，其中每个元素表示图像中的一个像素。图像矩阵的行数表示图像的高度，列数表示图像的宽度。通过对图像矩阵进行操作，我们可以对图像进行各种处理，如旋转、缩放、裁剪等。

## 2.3 图像通道
图像通道是图像矩阵的一个维度，用于表示图像的颜色信息。通常，图像有三个通道：红色、绿色和蓝色（RGB）。每个通道都包含与图像矩阵相同数量的像素值。通过对图像通道进行操作，我们可以对图像进行颜色转换和调整。

## 2.4 图像滤波
图像滤波是一种用于减少图像噪声和提高图像质量的方法。通过应用各种滤波器，我们可以对图像进行平滑、锐化、边缘提取等操作。

## 2.5 图像变换
图像变换是一种用于对图像进行特征提取和特征表示的方法。通过对图像进行各种变换，我们可以将图像转换为其他形式，以便更容易地提取有用的信息。

图像滤波是一种用于减少图像噪声和提高图像质量的方法。通过应用各种滤波器，我们可以对图像进行平滑、锐化、边缘提取等操作。

### 3.1.1 均值滤波
均值滤波是一种简单的滤波方法，它通过将周围像素的值求和并除以周围像素的数量来计算当前像素的值。这种方法可以用于减少图像中的噪声。

均值滤波的公式如下：
$$
G(x,y) = \frac{1}{N} \sum_{i=-n}^{n} \sum_{j=-n}^{n} f(x+i,y+j)
$$

### 3.1.2 中值滤波
中值滤波是一种更复杂的滤波方法，它通过将周围像素的值排序并选择中间值来计算当前像素的值。这种方法可以用于减少图像中的噪声，同时保留图像的边缘信息。

中值滤波的公式如下：
$$
G(x,y) = \text{median}\{f(x+i,y+j)\}
$$

### 3.1.3 高斯滤波
高斯滤波是一种常用的滤波方法，它通过将周围像素的值与一个高斯函数相乘来计算当前像素的值。这种方法可以用于减少图像中的噪声，同时保留图像的细节信息。

高斯滤波的公式如下：
$$
G(x,y) = \frac{1}{2\pi\sigma^2} \exp\left(-\frac{(x-a)^2 + (y-b)^2}{2\sigma^2}\right)
$$
