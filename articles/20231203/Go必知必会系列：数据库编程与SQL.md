## 4.1 Go语言中的数据库连接代码实例

```go
package main

import (
    "database/sql"
    "fmt"
    _ "github.com/go-sql-driver/mysql"
)

func main() {
    db, err := sql.Open("mysql", "user:password@tcp(localhost:3306)/dbname")
    if err != nil {
        panic(err)
    }
    defer db.Close()

    rows, err := db.Query("SELECT * FROM table_name")
    if err != nil {
        panic(err)
    }
    defer rows.Close()

    for rows.Next() {
        var id int
        var name string
        err := rows.Scan(&id, &name)
        if err != nil {
            panic(err)
        }
        fmt.Println(id, name)
    }
}
```

## 4.3 Go语言中的数据库事务代码实例

```go
package main

import (
    "database/sql"
    "fmt"
    "log"
    _ "github.com/go-sql-driver/mysql"
)

func main() {
    db, err := sql.Open("mysql", "user:password@tcp(localhost:3306)/dbname")
    if err != nil {
        log.Fatal(err)
    }
    defer db.Close()

    tx := db.Begin()
    defer tx.Commit()

    _, err = tx.Exec("INSERT INTO table_name (id, name) VALUES (?, ?)", 1, "John")
    if err != nil {
        tx.Rollback()
        log.Fatal(err)
    }

    _, err = tx.Exec("UPDATE table_name SET name = ? WHERE id = ?", "Doe", 1)
    if err != nil {
        tx.Rollback()
        log.Fatal(err)
    }

    rows, err := tx.Query("SELECT * FROM table_name")
    if err != nil {
        tx.Rollback()
        log.Fatal(err)
    }
    defer rows.Close()

    for rows.Next() {
        var id int
        var name string
        err := rows.Scan(&id, &name)
        if err != nil {
            tx.Rollback()
            log.Fatal(err)
        }
        fmt.Println(id, name)
    }
}
```

在上述代码中，我们首先使用sql.Open函数打开数据库连接，并传入数据库类型、用户名、密码、主机名和端口号等参数。然后，我们使用db.Begin函数开始事务，并传入事务参数。接着，我们使用tx.Exec函数执行SQL插入和更新操作，并传入SQL语句和参数。如果执行成功，我们使用tx.Commit函数提交事务，否则使用tx.Rollback函数回滚事务。最后，我们使用rows.Next函数遍历查询结果，并使用rows.Scan函数扫描查询结果。最后，我们使用fmt.Println函数输出查询结果。
