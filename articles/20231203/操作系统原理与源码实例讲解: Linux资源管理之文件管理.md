## 4.1 创建文件的代码实例

创建文件的代码实例可以通过系统调用来实现。具体的代码实现如下：

```c
int create_file(const char *filename) {
    int fd = open(filename, O_CREAT | O_WRONLY, 0644);
    if (fd < 0) {
        perror("open");
        return -1;
    }
    close(fd);
    return 0;
}
```
