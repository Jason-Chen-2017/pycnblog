## 4.1 概率论代码实例
在这个代码实例中，我们将通过Python来计算硬币投掷实验中“硬币正面”和“硬币反面”的概率。

```python
import random

# 硬币投掷实验次数
n = 1000

# 硬币投掷结果
result = [random.choice(["正面", "反面"]) for _ in range(n)]

# 硬币正面的数量
positive_count = sum(1 for r in result if r == "正面")

# 硬币正面的概率
positive_probability = positive_count / n

print("硬币正面的概率:", positive_probability)
```

在这个代码实例中，我们首先通过Python的random模块来生成1000次硬币投掷结果。然后，我们通过计算“硬币正面”的数量来计算“硬币正面”的概率。最后，我们通过print函数来输出“硬币正面”的概率。
