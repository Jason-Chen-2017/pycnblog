# 2.核心概念与联系

- 多模态数据：多模态数据是指不同类型的数据，例如图像、文本、音频等。这些数据可以在不同的应用场景中进行学习和分析。
- 跨模态学习：跨模态学习是指在不同类型的数据上进行学习的过程。这种学习方法可以在不同类型的数据上进行特征学习、模型训练和预测等。
- 多模态学习任务：多模态学习任务是指在多模态数据上进行的学习任务，例如图像分类、文本分类、语音识别等。这些任务可以通过多模态学习方法进行解决。

多模态学习与传统的单模态学习方法的联系在于，多模态学习可以在不同类型的数据上进行学习，从而更好地利用数据的多样性，提高学习模型的性能。
