## 4.1 计算函数的导数

我们可以使用Python的numpy库来计算函数的导数。以下是一个示例代码：

```python
import numpy as np

def derivative(x, h=1e-6):
    return (f(x + h) - f(x)) / h

def f(x):
    return x**2

x = np.linspace(-10, 10, 1000)
h = 1e-6

derivative_values = [derivative(x_i) for x_i in x]

plt.plot(x, derivative_values)
plt.xlabel('x')
plt.ylabel('derivative')
plt.title('Derivative of f(x) = x^2')
plt.show()
```

在这个示例中，我们定义了一个`derivative`函数，它接受一个变量`x`和一个可选的步长`h`参数。我们使用了一个简单的差分方法来计算函数的导数。然后，我们使用numpy的`linspace`函数来生成一个连续的x值列表，并使用`derivative`函数来计算导数值。最后，我们使用matplotlib库来绘制导数值的图像。
