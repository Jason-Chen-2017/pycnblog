# 2.核心概念与联系

在Rust中，集合和迭代器是编程中的基本概念。集合是一种数据结构，它可以存储多个值。迭代器是一种用于遍历集合中的元素的机制。

集合在Rust中有多种类型，包括Vec、Array、HashSet和HashMap等。这些集合类型都实现了Iterator trait，这意味着它们可以通过迭代器遍历其元素。

迭代器是一种用于遍历集合中的元素的机制。它们实现了Iterator trait，并提供了一种方便的方式来遍历集合中的元素。

# 4.具体代码实例和详细解释说明

在Rust中，可以使用Vec、Array、HashSet和HashMap等集合类型来创建集合对象。这些集合类型都实现了Iterator trait，可以通过iter方法返回一个迭代器对象。

以下是一个具体的代码实例：

```rust
fn main() {
    let numbers = vec![1, 2, 3, 4, 5];
    let mut iterator = numbers.iter();

    while let Some(number) = iterator.next() {
        println!("{}", number);
    }
}
```

在这个例子中，我们创建了一个Vec对象numbers，并调用iter方法返回一个迭代器对象。然后，我们使用while循环和next方法遍历迭代器对象，并访问其元素。
