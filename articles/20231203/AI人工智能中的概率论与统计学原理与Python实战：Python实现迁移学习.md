# 2.核心概念与联系
# 2.1概率论基本概念
概率论是一门数学分支，用于描述事件发生的可能性。概率论的基本概念包括事件、样本空间、概率空间、事件的独立性等。

事件：在概率论中，事件是一个可能发生的结果。事件可以是一个数字、一个区间或一个集合等。

样本空间：样本空间是所有可能发生的事件集合。在概率论中，样本空间用大写字母S表示。

概率空间：概率空间是一个包含样本空间和概率函数的数学结构。在概率论中，概率空间用大写字母Ω表示。

事件的独立性：两个事件之间的独立性是指它们发生的概率不受彼此影响。在概率论中，两个事件A和B之间的独立性可以用以下公式表示：P(A∩B)=P(A)×P(B)。

# 2.2概率模型
概率模型是一种用于描述事件发生概率的数学模型。概率模型可以是离散的或连续的。

离散概率模型：离散概率模型用于描述离散事件的发生概率。例如，抛硬币的结果是正面或反面。

连续概率模型：连续概率模型用于描述连续事件的发生概率。例如，气温的分布。

# 2.3条件概率
条件概率是一种描述事件发生概率的概率模型。条件概率是一个事件发生的概率，给定另一个事件已经发生。在概率论中，条件概率可以用以下公式表示：P(A|B)=P(A∩B)/P(B)。

# 2.4贝叶斯定理
贝叶斯定理是一种描述事件发生概率的概率模型。贝叶斯定理可以用以下公式表示：P(A|B)=P(B|A)×P(A)/P(B)。
