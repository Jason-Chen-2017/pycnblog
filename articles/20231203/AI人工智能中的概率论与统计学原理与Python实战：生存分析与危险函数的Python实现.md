## 4.1生存分析
```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机数据
np.random.seed(0)
t = np.linspace(0, 10, 100)
s = np.exp(-t)

# 计算生存函数
S = 1 - s

# 计算失效函数
F = s

# 计算危险函数
h = s / S

# 绘制图像
plt.plot(t, s, label='s(t)')
plt.plot(t, F, label='F(t)')
plt.plot(t, h, label='h(t)')
plt.legend()
plt.show()
```
在这个代码实例中，我们首先生成了随机数据，然后计算了生存函数、失效函数和危险函数。最后，我们绘制了这些函数的图像。
