### 1.4.2 Mockito实例

在这个例子中，我们将通过Mockito的设计原理来实现一个简单的模拟框架。

```java
public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
}

public class CalculatorTest {
    @Test
    public void testAdd() {
        Calculator calculator = mock(Calculator.class);
        when(calculator.add(1, 2)).thenReturn(3);
        int result = calculator.add(1, 2);
        assertEquals(3, result);
    }
}
```

在这个例子中，我们首先定义了一个Calculator类，它有一个add方法。然后，我们定义了一个CalculatorTest类，它包含了一个testAdd方法。在testAdd方法中，我们使用mock方法来创建一个Calculator对象的模拟实例。然后，我们使用when方法来设置add方法的返回值。最后，我们调用add方法，并使用assertEquals方法来验证结果是否正确。
