# 2.核心概念与联系
网络安全咨询是一项跨学科的技术，涉及计算机科学、信息安全、网络安全、密码学等多个领域。网络安全咨询师需要具备广泛的知识和技能，包括操作系统、网络协议、安全框架、加密算法等。

网络安全咨询的核心概念包括：

- 网络安全：网络安全是指保护计算机系统和网络资源免受未经授权的访问、篡改和破坏的能力。网络安全包括防火墙、防病毒、加密、身份认证等多种技术手段。
- 安全框架：安全框架是一种用于构建安全系统的方法和模型。常见的安全框架有ISO/IEC 27001、NIST SP 800-53等。
- 加密算法：加密算法是一种用于保护数据和信息的方法。常见的加密算法有AES、RSA、SHA等。
