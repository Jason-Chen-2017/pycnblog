# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的规则引擎实例来说明规则引擎的性能优化策略。

```python
# 定义规则库
rules = [
    {
        "condition": "age >= 18",
        "action": "grant_access"
    },
    {
        "condition": "age < 18",
        "action": "deny_access"
    }
]

# 定义事实库
facts = [
    {
        "name": "Alice",
        "age": 20
    },
    {
        "name": "Bob",
        "age": 17
    }
]

# 定义规则引擎
def rule_engine(rules, facts):
    for fact in facts:
        for rule in rules:
            if eval(rule["condition"]):
                rule["action"](fact)

# 执行规则引擎
rule_engine(rules, facts)
```

在上述代码中，我们定义了一个简单的规则引擎，它包括一个规则库和一个事实库。规则库包括两个规则，分别用于判断年龄是否大于或等于18岁，并执行相应的动作。事实库包括两个事实，分别表示Alice和Bob的年龄。我们通过调用`rule_engine`函数来执行规则引擎，并根据规则库和事实库的条件来执行相应的动作。
