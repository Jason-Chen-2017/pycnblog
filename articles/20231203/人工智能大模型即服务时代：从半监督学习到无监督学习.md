## 4.2无监督学习的代码实例
我们以K-均值聚类（K-means clustering）为例，来展示无监督学习的代码实例。K-均值聚类是一种常用的无监督学习方法，它将数据分为k个簇，使得每个簇内的数据点之间相互接近，而不同簇之间相互远离。

```python
import numpy as np
from sklearn.cluster import KMeans

# 数据预处理
X = np.random.rand(100, 10)

# K-均值聚类模型定义
kmeans = KMeans(n_clusters=3, random_state=0)

# 模型训练
kmeans.fit(X)

# 模型评估
labels = kmeans.labels_
centroids = kmeans.cluster_centers_
print('Labels:', labels)
print('Centroids:', centroids)
```
