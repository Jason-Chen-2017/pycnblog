## 3.1 概率密度函数
概率密度函数是用于描述一个随机变量的概率分布的函数。它可以帮助我们更好地理解随机变量的分布特征。概率密度函数的定义如下：

$$
f(x) = \frac{dP(x)}{dx}
$$

其中，$f(x)$ 是概率密度函数，$P(x)$ 是随机变量的概率分布函数，$dP(x)/dx$ 是随机变量的概率分布函数的导数。

通过概率密度函数，我们可以计算随机变量在某个区间内的概率。例如，对于一个正态分布的随机变量，其概率密度函数为：

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$\mu$ 是随机变量的均值，$\sigma$ 是随机变量的标准差。

## 3.2 分布函数
分布函数是用于描述一个随机变量的概率分布的累积概率。它可以帮助我们更好地理解随机变量的概率值。分布函数的定义如下：

$$
F(x) = P(X \leq x)
$$

其中，$F(x)$ 是分布函数，$P(X \leq x)$ 是随机变量的概率分布在某个区间内的概率。

通过分布函数，我们可以计算随机变量在某个区间内的概率。例如，对于一个正态分布的随机变量，其分布函数为：

$$
F(x) = \frac{1}{2}erfc(\frac{x-\mu}{\sqrt{2}\sigma})
$$

其中，$erfc$ 是错函数，$\mu$ 是随机变量的均值，$\sigma$ 是随机变量的标准差。

# 4.具体代码实例和详细解释说明
在本节中，我们将通过具体的代码实例来帮助你更好地理解概率密度函数和分布函数的概念。

## 4.1 正态分布的概率密度函数和分布函数
我们可以使用Python的numpy库来计算正态分布的概率密度函数和分布函数。以下是一个示例代码：

```python
import numpy as np

# 正态分布的参数
mean = 0
std = 1

# 计算正态分布的概率密度函数
def normal_pdf(x, mean, std):
    return 1 / (std * np.sqrt(2 * np.pi)) * np.exp(-(x - mean)**2 / (2 * std**2))

# 计算正态分布的分布函数
def normal_cdf(x, mean, std):
    z = (x - mean) / std
    return 0.5 * (1 + np.erf(z))

# 测试
x = np.linspace(-3, 3, 100)
pdf = normal_pdf(x, mean, std)
cdf = normal_cdf(x, mean, std)

# 绘制概率密度函数和分布函数
import matplotlib.pyplot as plt
plt.plot(x, pdf, label='PDF')
plt.plot(x, cdf, label='CDF')
plt.legend()
plt.show()
```

在上面的代码中，我们首先定义了正态分布的参数，即均值和标准差。然后我们定义了正态分布的概率密度函数和分布函数，并使用numpy库来计算它们的值。最后，我们使用matplotlib库来绘制概率密度函数和分布函数的图像。

## 4.2 多项式分布的概率密度函数和分布函数
我们也可以使用Python的scipy库来计算多项式分布的概率密度函数和分布函数。以下是一个示例代码：

```python
import numpy as np
from scipy.stats import binom

# 多项式分布的参数
n = 10
p = 0.5

# 计算多项式分布的概率密度函数
def binomial_pdf(x, n, p):
    return np.power(p, x) * np.power(1 - p, n - x) * np.power(n, n) / np.math.factorial(n)

# 计算多项式分布的分布函数
def binomial_cdf(x, n, p):
    return binom.cdf(x, n, p)

# 测试
x = np.arange(0, n + 1)
pdf = binomial_pdf(x, n, p)
cdf = binomial_cdf(x, n, p)

# 绘制概率密度函数和分布函数
import matplotlib.pyplot as plt
plt.plot(x, pdf, label='PDF')
plt.plot(x, cdf, label='CDF')
plt.legend()
plt.show()
```

在上面的代码中，我们首先定义了多项式分布的参数，即试验次数和成功概率。然后我们定义了多项式分布的概率密度函数和分布函数，并使用scipy库来计算它们的值。最后，我们使用matplotlib库来绘制概率密度函数和分布函数的图像。
