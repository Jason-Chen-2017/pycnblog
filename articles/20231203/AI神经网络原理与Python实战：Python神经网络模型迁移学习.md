# 2.核心概念与联系

## 数学模型公式详细讲解

在这一部分，我们将详细讲解迁移学习的数学模型公式。

### 损失函数

损失函数是用来衡量模型预测值与真实值之间差异的函数。在迁移学习中，我们通常使用均方误差（MSE）作为损失函数。MSE的公式如下：

$$
MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
$$

其中，$n$ 是样本数量，$y_i$ 是真实值，$\hat{y}_i$ 是预测值。

### 梯度下降

梯度下降是一种用于优化神经网络的算法。它通过计算模型的梯度，然后更新模型的权重和偏置来最小化损失函数。梯度下降的公式如下：

$$
\theta_{t+1} = \theta_t - \alpha \nabla J(\theta_t)
$$

其中，$\theta$ 是模型的参数，$t$ 是迭代次数，$\alpha$ 是学习率，$\nabla J(\theta_t)$ 是损失函数的梯度。

2. 任务不相关：迁移学习需要任务相关的数据，因此，任务不相关的数据可能会影响迁移学习的性能。

3. 计算资源限制：迁移学习需要大量的计算资源，因此，计算资源限制可能会影响迁移学习的性能。
