## 4.1 收集性能监测数据
我们可以使用Prometheus来收集系统的性能监测数据。以下是一个使用Prometheus收集CPU使用率数据的示例代码：

```python
import os
import subprocess

def collect_cpu_usage():
    cpu_usage = subprocess.check_output(["top", "-b", "-n", "1", "-d", "1", "-p", str(os.getpid()), "-q", "1"]).decode("utf-8")
    cpu_usage_list = cpu_usage.split("\n")
    cpu_usage_data = [float(cpu_usage_list[i].split()[1]) for i in range(1, len(cpu_usage_list) - 1)]
    return cpu_usage_data
```
