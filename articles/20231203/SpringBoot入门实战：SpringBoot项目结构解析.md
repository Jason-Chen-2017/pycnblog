Spring Boot 是一个用于构建原生的 Spring 应用程序的快速开始点和 PaaS 平台。它的目标是减少开发人员通过 XML 配置来配置和编写代码的量。Spring Boot 提供了一种简单的方法来配置 Spring 应用程序，使其能够自动配置。

Spring Boot 的核心概念是“自动配置”，它通过使用 Spring 的内置 beans 和自动配置来简化 Spring 应用程序的开发。Spring Boot 的核心组件是 Spring 框架的一部分，它提供了一种简单的方法来配置 Spring 应用程序，使其能够自动配置。
