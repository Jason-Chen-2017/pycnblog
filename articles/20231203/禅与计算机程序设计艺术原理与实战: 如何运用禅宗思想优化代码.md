# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的代码实例来说明如何使用禅宗思想来优化代码。

假设我们需要编写一个程序来计算两个数的和。我们可以使用以下代码来实现这个功能：

```python
def add(a, b):
    return a + b

if __name__ == '__main__':
    a = 10
    b = 20
    print(add(a, b))
```

在这个代码中，我们首先定义了一个名为`add`的函数，这个函数接受两个参数`a`和`b`，并返回它们的和。然后，我们在主程序中调用了这个函数，并打印了结果。

通过使用禅宗思想，我们可以对这个代码进行优化。具体来说，我们可以将`add`函数的定义和主程序的调用合并到一起，这样可以让代码更加简洁。

```python
def add(a, b):
    return a + b

```

在这个优化后的代码中，我们将`add`函数的定义和主程序的调用合并到一起，这样可以让代码更加简洁。
