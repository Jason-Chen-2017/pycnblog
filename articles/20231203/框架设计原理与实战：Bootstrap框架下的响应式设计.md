# 2.核心概念与联系

## 2.1 响应式设计的核心概念

响应式设计是一种网页设计方法，它允许网页根据不同的设备和屏幕尺寸自动调整布局和样式。这种设计方法的核心概念包括：流体布局（Fluid Layout）、流畅的图像（Fluid Images）和媒体查询（Media Queries）。

### 2.1.1 流体布局

流体布局是指网页元素的宽度和高度以屏幕宽度和高度为基础进行自动调整。通过使用相对单位（如百分比）来设置元素的宽度和高度，我们可以让元素随着屏幕尺寸的变化而自动调整大小。

### 2.1.2 流畅的图像

流畅的图像是指图像的大小会根据屏幕尺寸进行自动调整，以保持图像的清晰度和比例。通过使用相对单位（如百分比）来设置图像的宽度和高度，我们可以让图像随着屏幕尺寸的变化而自动调整大小。

### 2.1.3 媒体查询

媒体查询是一种CSS3功能，允许我们根据设备和屏幕尺寸来设置不同的样式。通过使用媒体查询，我们可以为不同的设备和屏幕尺寸提供不同的样式，从而实现响应式设计。

## 2.2 Bootstrap框架的核心概念

Bootstrap是一个基于HTML、CSS和JavaScript的前端框架，它提供了许多有用的组件和工具，可以帮助我们快速开发响应式网页。Bootstrap的核心概念包括：网格系统（Grid System）、组件（Components）和JavaScript扩展（JavaScript Extensions）。

### 2.2.1 网格系统

Bootstrap的网格系统是一个基于12列的流体布局系统，它可以帮助我们快速构建响应式网页布局。通过使用Bootstrap的类名和媒体查询，我们可以轻松地实现不同设备和屏幕尺寸下的不同布局。

### 2.2.2 组件

Bootstrap提供了许多有用的组件，如按钮（Buttons）、表单（Forms）、导航（Navbar）等，这些组件都是响应式的，可以根据设备和屏幕尺寸进行自动调整。

### 2.2.3 JavaScript扩展

Bootstrap还提供了许多JavaScript扩展，如模态框（Modal）、滚动条（Scrollspy）、下拉菜单（Dropdown）等，这些扩展都是响应式的，可以根据设备和屏幕尺寸进行自动调整。

```css

    background-color: red;
  }
}
```

```css
@media (max-width: 768px) {
  .element {
    font-size: 14px;
  }
}

@media (min-width: 768px) {
  .element {
    font-size: 16px;
  }
}
```
