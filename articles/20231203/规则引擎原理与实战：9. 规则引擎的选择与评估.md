```python
# 规则引擎的编写
rules = [
    {
        "condition": "age >= 18",
        "action": "grant_access"
    },
    {
        "condition": "age < 18",
        "action": "deny_access"
    }
]

# 规则引擎的执行
def execute_rules(rules, data):
    for rule in rules:
        if eval(rule["condition"], data):
            return rule["action"]
    return None

# 规则引擎的监控
def monitor_rules(rules, data):
    result = execute_rules(rules, data)
    if result:
        print(f"Access granted for user {data['name']}")
    else:
        print(f"Access denied for user {data['name']}")

# 规则引擎的使用
data = {"name": "John", "age": 20}
monitor_rules(rules, data)
```

在上述代码中，我们首先定义了一组规则，其中包括了条件和操作。然后，我们定义了一个`execute_rules`函数，用于根据数据来执行规则。接着，我们定义了一个`monitor_rules`函数，用于根据数据来监控规则执行结果。最后，我们使用了`monitor_rules`函数来执行规则并监控结果。
