# 4.具体代码实例和详细解释说明

以下是一个具体的代码实例，用于实现目标设定与目标层次的算法：

```python
class GoalHierarchy:
    def __init__(self, goals):
        self.goals = goals

    def set_goals(self, goals):
        self.goals = goals

    def set_weights(self, weights):
        self.weights = weights

    def calculate_total_goal(self):
        total_goal = 0
        for i in range(len(self.goals)):
            total_goal += self.weights[i] * self.goals[i]
        return total_goal

# 示例使用
goals = [10, 20, 30]
weights = [0.3, 0.4, 0.3]

goal_hierarchy = GoalHierarchy(goals)
goal_hierarchy.set_goals(goals)
goal_hierarchy.set_weights(weights)

total_goal = goal_hierarchy.calculate_total_goal()
print("总目标：", total_goal)
```
