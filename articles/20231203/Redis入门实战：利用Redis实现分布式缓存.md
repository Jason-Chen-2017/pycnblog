                 

# 1.背景介绍

Redis（Remote Dictionary Server）是一个开源的高性能的key-value存储系统，它支持数据的持久化，可基于内存（in-memory）也可基于磁盘。Redis 提供多种语言的 API，包括：Ruby、Python、Java、C 和 C#，因此可以在任何语言中使用。

Redis 是一个使用 ANSI C 语言编写、遵循 BSD 协议的开源（GPL）、跨平台（Windows 和 Linux）的高性能 key-value 数据库，它支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。

Redis 可以用作数据库、缓存和消息队列。

Redis 的优点：

1. Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。
2. Redis 的网络传输层实现支持多种协议（TCP、HTTP、AWS、UnixSocket 等）。
3. Redis 支持数据的备份，即 Master-Slave 模式。
4. Redis 支持列式存储（Redis 的 String 类型是 Redis 列式存储的一种特例），可以用于存储大量的列数据。
5. Redis 支持 publish/subscribe 模式。
6. Redis 支持数据的有序存储。
7. Redis 支持数据的分片（Sharding）。
8. Redis 支持 Lua 脚本（Redis 脚本）。
9. Redis 支持事务（Redis 事务）。
10. Redis 支持定时任务。
