# 2.核心概念与联系

在Python中，模块是代码的组织和重用的基本单元。模块可以包含函数、类、变量等，可以通过导入模块的方式使用其内容。模块的导入与定义是Python中的一个重要概念，它有助于提高代码的可读性、可维护性和可重用性。

## 2.1 模块的定义

模块是Python中的一个文件，文件名以`.py`为后缀。模块可以包含函数、类、变量等。模块的定义主要包括以下几个步骤：

1. 创建一个Python文件，文件名以`.py`为后缀。
2. 在文件中定义函数、类、变量等。
3. 保存文件。

例如，我们可以创建一个名为`math_utils.py`的模块，定义一个`add`函数：

```python
# math_utils.py
def add(a, b):
    return a + b
```

## 2.2 模块的导入

模块的导入是指将一个模块的内容加载到当前的Python解释器环境中，以便可以使用其内容。模块的导入主要包括以下几个步骤：

1. 使用`import`关键字导入模块。
2. 使用点（`.`）访问模块的内容。

例如，我们可以导入`math_utils`模块，并使用`add`函数：

```python
import math_utils

result = math_utils.add(1, 2)
print(result)  # 输出: 3
```

## 2.3 模块的使用

模块的使用是指在当前的Python解释器环境中使用导入的模块内容。模块的使用主要包括以下几个步骤：

1. 使用导入的模块内容。
2. 使用模块内容完成某个任务。

例如，我们可以使用`math_utils`模块中的`add`函数，完成一个简单的加法计算任务：

```python
import math_utils

result = math_utils.add(1, 2)
print(result)  # 输出: 3
```
