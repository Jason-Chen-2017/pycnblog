## 4.1 字符编码转换实例

字符编码转换是计算机编程语言的基本操作，它涉及到字符集的转换、编码格式的转换等方面。以下是一个简单的字符编码转换实例：

```python
import codecs

def convert_encoding(input_file, output_file, input_encoding, output_encoding):
    with open(input_file, 'r', encoding=input_encoding) as f_in:
        with open(output_file, 'w', encoding=output_encoding) as f_out:
            for line in f_in:
                f_out.write(line)

convert_encoding('input.txt', 'output.txt', 'utf-8', 'gbk')
```

## 4.2 语言包管理实例

语言包管理是计算机编程语言的重要操作，它涉及到语言包的加载、使用、更新等方面。以下是一个简单的语言包管理实例：

```python
import gettext

def init_gettext(domain):
    gettext.bindtextdomain(domain, '/path/to/locale')
    gettext.textdomain(domain)

def translate(text):
    return gettext.gettext(text)

def ngettext(singular, plural, n):
    return gettext.ngettext(singular, plural, n)

init_gettext('myapp')
print(translate('Hello, World!'))
print(ngettext('1 item', '1 item', 1))
```
