## 4.具体代码实例和详细解释说明

### 4.1代码实例1：排序算法

排序算法是计算机程序中非常常见的一种算法，它可以将一组数据按照某种规则进行排序。以下是一个简单的排序算法实例：

```python
def sort(arr):
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] > arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
sorted_arr = sort(arr)
print(sorted_arr)
```

在这个代码实例中，我们使用了冒泡排序算法进行排序。冒泡排序的时间复杂度为O(n^2)，空间复杂度为O(1)。

### 4.2代码实例2：搜索算法

搜索算法是计算机程序中另一种非常常见的算法，它可以在一组数据中查找某个特定的值。以下是一个简单的搜索算法实例：

```python
def search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

arr = [5, 2, 8, 1, 9]
target = 8
index = search(arr, target)
if index != -1:
    print("找到了，下标为：", index)
else:
    print("没有找到")
```

在这个代码实例中，我们使用了线性搜索算法进行搜索。线性搜索的时间复杂度为O(n)，空间复杂度为O(1)。
