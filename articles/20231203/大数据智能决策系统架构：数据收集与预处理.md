# 2.核心概念与联系

在大数据智能决策系统中，数据收集和预处理是两个关键的环节。数据收集是指从各种数据源中收集数据，并将其存储在数据仓库中。预处理是指对收集到的数据进行清洗、转换和整理，以便进行分析和决策。

数据收集包括以下几个环节：

1. 数据源识别：识别出需要收集的数据源，并确定如何从中收集数据。
2. 数据采集：使用适当的技术和工具从数据源中收集数据。
3. 数据存储：将收集到的数据存储在数据仓库中，以便进行分析和决策。

预处理包括以下几个环节：

1. 数据清洗：对收集到的数据进行清洗，以移除错误、缺失值和噪声。
2. 数据转换：将收集到的数据转换为适合进行分析和决策的格式。
3. 数据整理：对数据进行整理，以便更容易进行分析和决策。
