# 4.具体代码实例和详细解释说明

在这里，我们以一个简单的Assembly代码实例为例，详细解释其执行过程：

```assembly
MOV R1, #10
ADD R2, R1, #5
SUB R3, R2, #3
```

1.加载数据和指令到寄存器：

在这个例子中，我们将数字10加载到寄存器R1中，数字5加载到寄存器R2中，数字3加载到寄存器R3中。

2.执行指令：

- MOV指令：将数字10加载到寄存器R1中。
- ADD指令：将寄存器R1和寄存器R2的值相加，结果存储到寄存器R2中。
- SUB指令：将寄存器R2和寄存器R3的值相减，结果存储到寄存器R3中。

3.存储结果：

在这个例子中，我们的结果存储在寄存器R3中。
