大数据智能决策系统是一种利用大量数据进行智能决策的系统，它可以根据数据分析结果自动进行决策。这种系统通常包括数据收集、数据处理、决策规则定义、推理引擎和决策执行等模块。

# 2.核心概念与联系

## 2.1决策规则
决策规则是大数据智能决策系统中的核心组成部分，它定义了在特定条件下采取的决策行动。决策规则可以是基于规则引擎的规则，也可以是基于机器学习模型的预测。

## 2.2推理引擎
推理引擎是大数据智能决策系统中的另一个核心组成部分，它负责根据决策规则和输入数据进行推理，从而得出决策结果。推理引擎可以是基于规则的推理，也可以是基于机器学习的推理。

## 2.3联系
决策规则和推理引擎之间的联系是大数据智能决策系统的核心。决策规则定义了系统的决策逻辑，推理引擎则负责根据这些规则进行推理。这种联系使得大数据智能决策系统能够根据实际情况自动进行决策，从而实现智能化。

```python
import datetime

def is_weekend(date):
    return date.weekday() >= 5

def is_electronic_product(product):
    return product.category == 'electronic'

def provide_coupon(user):
    if is_weekend(datetime.date.today()) and is_electronic_product(user.last_purchase):
        return True
    else:
        return False
```
