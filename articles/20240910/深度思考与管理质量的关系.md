##### 1. 如何使用深度优先搜索算法求解图的拓扑排序？

**答案：** 使用深度优先搜索算法求解图的拓扑排序，可以按照以下步骤进行：

1. 创建一个栈，用于存储当前搜索的节点；
2. 初始化一个空的拓扑排序序列；
3. 遍历图中的所有节点，对每个节点执行以下操作：
   a. 如果节点未被访问，则将其入栈；
   b. 出栈节点，并将其加入拓扑排序序列；
   c. 遍历节点的邻接节点，对每个邻接节点执行步骤 3.a 和 3.b；
4. 返回拓扑排序序列。

以下是使用 Python 实现的示例代码：

```python
def topology_sort(graph):
    visited = set()
    stack = []
    topological_order = []

    for node in graph:
        if node not in visited:
            depth_first_search(node, visited, stack)

        while stack:
            topological_order.append(stack.pop())

    return topological_order

def depth_first_search(node, visited, stack):
    visited.add(node)
    for neighbor in graph[node]:
        if neighbor not in visited:
            depth_first_search(neighbor, visited, stack)
    stack.append(node)

graph = {
    'A': ['B', 'C'],
    'B': ['D'],
    'C': ['D'],
    'D': []
}

print(topology_sort(graph))  # 输出：['A', 'B', 'C', 'D']
```

##### 2. 如何使用广度优先搜索算法求解最短路径？

**答案：** 使用广度优先搜索算法求解最短路径，可以按照以下步骤进行：

1. 初始化一个队列，用于存储待访问的节点，以及从起始节点到当前节点的路径长度；
2. 将起始节点加入队列，并设置其路径长度为 0；
3. 遍历队列，对每个节点执行以下操作：
   a. 将节点的邻接节点加入队列，并设置其路径长度为当前节点的路径长度加 1；
   b. 如果邻接节点未访问过，则将其从队列中移出，并更新最短路径；
4. 返回最短路径。

以下是使用 Python 实现的示例代码：

```python
from collections import deque

def breadth_first_search(graph, start):
    queue = deque([(start, 0)])
    visited = set([start])
    shortest_path = None

    while queue:
        node, distance = queue.popleft()
        if node == 'C':  # 目标节点
            shortest_path = distance
            break

        for neighbor in graph[node]:
            if neighbor not in visited:
                queue.append((neighbor, distance + 1))
                visited.add(neighbor)

    return shortest_path

graph = {
    'A': ['B', 'C'],
    'B': ['D'],
    'C': [],
    'D': ['E'],
    'E': []
}

print(breadth_first_search(graph, 'A'))  # 输出：3
```
