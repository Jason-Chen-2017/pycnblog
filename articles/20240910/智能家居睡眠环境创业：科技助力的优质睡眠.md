#### 算法编程题库

**1. 编写一个算法，计算用户睡眠时长。**

**题目描述：** 给定一个用户入睡和醒来的时间，编写一个算法计算用户的睡眠时长。

**输入：** 入睡时间（小时）和醒来时间（小时）。

**输出：** 睡眠时长（小时）。

**示例：**
```python
input:
- 入睡时间：22
- 醒来时间：6

output:
- 睡眠时长：8
```

**答案：**
```python
def calculate_sleep_duration(sleep_time, wake_time):
    if wake_time < sleep_time:
        sleep_duration = 24 - sleep_time + wake_time
    else:
        sleep_duration = wake_time - sleep_time
    return sleep_duration

sleep_time = 22
wake_time = 6
print(calculate_sleep_duration(sleep_time, wake_time))
```

**解析：** 这个算法通过计算用户入睡和醒来时间之差，得到用户的睡眠时长。

**2. 编写一个算法，分析用户睡眠质量。**

**题目描述：** 给定一个用户睡眠时的心率数据，编写一个算法分析用户的睡眠质量。

**输入：** 心率数据（列表）。

**输出：** 睡眠质量（字符串，如“良好”、“一般”、“较差”）。

**示例：**
```python
input:
- 心率数据：[70, 75, 80, 85, 90, 75, 70, 65, 60, 55, 50, 45, 40, 35, 30]

output:
- 睡眠质量：“良好”
```

**答案：**
```python
def analyze_sleep_quality(heart_rate_data):
    average_heart_rate = sum(heart_rate_data) / len(heart_rate_data)
    if average_heart_rate < 60:
        sleep_quality = "较差"
    elif average_heart_rate < 70:
        sleep_quality = "一般"
    else:
        sleep_quality = "良好"
    return sleep_quality

heart_rate_data = [70, 75, 80, 85, 90, 75, 70, 65, 60, 55, 50, 45, 40, 35, 30]
print(analyze_sleep_quality(heart_rate_data))
```

**解析：** 这个算法通过计算心率数据的平均值，判断用户的睡眠质量。
