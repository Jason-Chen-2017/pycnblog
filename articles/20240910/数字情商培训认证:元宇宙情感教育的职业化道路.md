### 三、算法编程题库及解析

#### 1. 编写一个算法，根据用户输入的文本，判断文本中是否包含情感词汇。
**代码实例：**
```python
def contains_emotion(text):
    emotions = ["happy", "sad", "angry", "surprised"]
    for emotion in emotions:
        if emotion in text.lower():
            return True
    return False

# 测试
print(contains_emotion("I am happy today"))  # 输出：True
print(contains_emotion("I am feeling sad"))  # 输出：True
print(contains_emotion("I am angry at this situation"))  # 输出：True
print(contains_emotion("This is a surprise"))  # 输出：True
print(contains_emotion("This is a normal day"))  # 输出：False
```
