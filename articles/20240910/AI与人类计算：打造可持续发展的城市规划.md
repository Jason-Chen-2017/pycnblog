#### 二、算法编程题库

##### 4. 编写一个算法，根据居民出行数据预测某个区域的交通需求。

**题目描述：**

输入一组居民出行数据，包括出发时间、出发位置、目的地位置。编写一个算法，预测某个时间段内某个区域的交通需求。

**答案：**

```python
def predict_traffic_demand(travel_data, start_time, end_time, region):
    # 统计在指定时间段内，从start_time到end_time，进入region区域的出行次数
    demand = 0
    for record in travel_data:
        if record['start_time'] <= end_time and record['end_time'] >= start_time and record['destination'] == region:
            demand += 1
    return demand
```

**解析：**

该算法根据输入的出行数据和指定的时间段、区域，统计在时间段内进入该区域的出行次数，从而预测交通需求。

##### 5. 编写一个算法，根据环境监测数据，评估某个区域的空气质量。

**题目描述：**

输入一组环境监测数据，包括空气质量指标（如PM2.5、PM10、CO、SO2等）和对应的监测时间。编写一个算法，评估某个区域的空气质量。

**答案：**

```python
def assess_air_quality(air_quality_data, region):
    # 计算每个指标的均值
    mean_values = {}
    for record in air_quality_data:
        if record['region'] == region:
            for key in record['metrics'].keys():
                if key in mean_values:
                    mean_values[key] += record['metrics'][key]
                else:
                    mean_values[key] = record['metrics'][key]
    
    for key in mean_values:
        mean_values[key] /= len(air_quality_data)
    
    # 根据均值评估空气质量
    quality = '良好'
    for key in mean_values:
        if mean_values[key] > 50:  # 以PM2.5为例，超过50视为较差
            quality = '较差'
            break
    
    return quality
```

**解析：**

该算法根据输入的环境监测数据，计算指定区域内各个空气质量指标的均值，并根据均值评估空气质量。如果某个指标的均值超过预设阈值，则认为空气质量较差。
