### 1. 运动员表现分析算法

**题目：** 编写一个函数，用于计算运动员的平均得分和命中率。

**答案：**
```python
def calculate_stats(scores, shots):
    average_score = sum(scores) / len(scores)
    hit_percentage = sum(shots) / len(shots)
    return average_score, hit_percentage
```

**解析：** 该函数接收两个列表，`scores` 和 `shots`，分别表示运动员的得分和投篮次数。通过计算得分平均值和命中率，为教练和运动员提供有针对性的指导。
