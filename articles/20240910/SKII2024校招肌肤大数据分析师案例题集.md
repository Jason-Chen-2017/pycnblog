#### 1. 数据清洗

**题目：** 编写一个Python函数，用于清洗包含缺失值和异常值的肌肤数据。

**答案：** 以下是一个简单的Python函数，用于清洗包含缺失值和异常值的肌肤数据：

```python
import pandas as pd

def clean_data(data):
    # 填充缺失值
    data = data.fillna(method='ffill')
    
    # 删除异常值
    Q1 = data.quantile(0.25)
    Q3 = data.quantile(0.75)
    IQR = Q3 - Q1
    data = data[~((data < (Q1 - 1.5 * IQR)) | (data > (Q3 + 1.5 * IQR))).any(axis=1)]
    
    return data
```

**解析：** 这个函数首先使用前向填充方法（`method='ffill'`）来填充缺失值，然后使用三倍标准差方法（`1.5 * IQR`）来删除异常值。
