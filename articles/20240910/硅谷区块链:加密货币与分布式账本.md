### 算法编程题库

#### 1. 区块链数据结构实现

**题目：** 使用 Go 语言实现一个简单的区块链数据结构，包括区块（Block）和链（Chain）。

**答案：** 下面是一个简单的区块链数据结构的实现示例：

```go
package main

import (
    "crypto/sha256"
    "encoding/hex"
    "fmt"
)

// 区块定义
type Block struct {
    Index     int
    Timestamp string
    Data      string
    Hash      string
    PrevHash  string
}

// 链定义
type Blockchain struct {
    chain []Block
}

// 创建一个新的区块
func (bc *Blockchain) NewBlock(data string, prevHash string) *Block {
    index := len(bc.chain) + 1
    timestamp := time.Now().Format(time.RFC3339)
    hash := calculateHash(index, timestamp, data, prevHash)
    block := Block{index, timestamp, data, hash, prevHash}
    bc.chain = append(bc.chain, block)
    return &block
}

// 计算哈希值
func calculateHash(index int, timestamp string, data string, prevHash string) string {
    hashInput := fmt.Sprintf("%d-%s-%s-%s", index, timestamp, data, prevHash)
    hash := sha256.Sum256([]byte(hashInput))
    return hex.EncodeToString(hash[:])
}

// 主函数
func main() {
    bc := NewBlockchain()
    bc.NewBlock("First block", "")
    bc.NewBlock("Second block", bc.chain[0].Hash)
    bc.NewBlock("Third block", bc.chain[1].Hash)
    for _, block := range bc.chain {
        fmt.Println(block)
    }
}
```

**解析：** 这个简单的区块链实现了区块和链的基本数据结构。每个区块包含索引、时间戳、数据和哈希值。区块通过哈希链接形成链条，保证了数据的不可篡改性。

#### 2. 加密货币挖矿算法

**题目：** 使用 Go 语言实现一个简单的加密货币挖矿算法，要求求解一个哈希值，使其满足特定条件。

**答案：** 下面是一个简单的挖矿算法实现示例：

```go
package main

import (
    "crypto/sha256"
    "encoding/hex"
    "math/big"
    "time"
)

// 挖矿算法
func mineBlock(data string, difficulty int) (string, error) {
    var hashValue, targetHashValue string
    start := time.Now()
    nonce := 0

    for {
        nonce++
        hashInput := fmt.Sprintf("%s%d", data, nonce)
        hash := sha256.Sum256([]byte(hashInput))
        hashValue = hex.EncodeToString(hash[:])

        bigHash := big.NewInt(0)
        bigHash.SetBytes([]byte(hashValue))

        bigTarget := big.NewInt(0)
        for i := 0; i < difficulty; i++ {
            bigTarget.Lsh(bigTarget, 1)
        }

        if bigHash.Cmp(bigTarget) == -1 {
            break
        }

        if nonce%1000 == 0 {
            fmt.Printf("nonce: %d, hash: %s\n", nonce, hashValue)
        }
    }

    duration := time.Since(start)
    fmt.Printf("Found a block after %v, nonce: %d, hash: %s\n", duration, nonce, hashValue)
    return hashValue, nil
}

// 主函数
func main() {
    data := "Hello, blockchain!"
    difficulty := 16

    hash, err := mineBlock(data, difficulty)
    if err != nil {
        fmt.Println(err)
        return
    }

    fmt.Println("Block hash:", hash)
}
```

**解析：** 这个简单的挖矿算法通过不断递增 nonce 值，计算哈希值，直到找到一个满足特定难度的哈希值。挖矿过程会输出挖矿进度，并在找到满足条件的哈希值时输出结果。
