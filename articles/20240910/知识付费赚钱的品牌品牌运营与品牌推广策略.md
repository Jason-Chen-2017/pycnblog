#### 3. 品牌推荐算法

**题目：** 编写一个算法，根据用户的浏览历史和购买记录，推荐与之相关的品牌。

**输入：** 
- 用户行为数据（字典数组，每个字典包含品牌名称和行为类型）

**输出：** 
- 推荐品牌列表（字符串数组）

**示例：** 
```python
def brand_recommendation(user_actions):
    brand_counts = {}
    for action in user_actions:
        brand_counts[action['brand']] = brand_counts.get(action['brand'], 0) + 1
    sorted_brands = sorted(brand_counts.items(), key=lambda x: x[1], reverse=True)
    return [brand[0] for brand in sorted_brands[:3]]

user_actions = [
    {"brand": "品牌A", "type": "浏览"},
    {"brand": "品牌B", "type": "购买"},
    {"brand": "品牌C", "type": "浏览"},
    {"brand": "品牌D", "type": "浏览"},
]

print(brand_recommendation(user_actions))
```
