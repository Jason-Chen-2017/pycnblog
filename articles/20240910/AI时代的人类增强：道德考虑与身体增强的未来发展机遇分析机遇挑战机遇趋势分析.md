#### 6.1 最长公共子序列（LCS）

**题目：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。编写一个算法来解决这个问题，并给出代码实例。

**答案：** 最长公共子序列（Longest Common Subsequence，LCS）问题是计算机科学中经典的问题，可以通过动态规划算法来解决。

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

# 示例
str1 = "AGGTAB"
str2 = "GXTXAYB"
print(longest_common_subsequence(str1, str2))  # 输出 "GTAB"
```

**解析：** 这个算法使用一个二维数组 `dp` 来存储中间结果。`dp[i][j]` 表示 `str1` 的前 `i` 个字符和 `str2` 的前 `j` 个字符的最长公共子序列的长度。通过填充这个数组，我们可以得到最终的结果。

#### 6.2 动态规划求解背包问题

**题目：** 使用动态规划算法解决0/1背包问题，编写一个算法来求解给定物品的最大价值。

**答案：** 0/1背包问题是一个经典的优化问题，可以通过动态规划算法来解决。

```python
def knapsack(values, weights, capacity):
    n = len(values)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if weights[i - 1] <= w:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - weights[i - 1]] + values[i - 1])
            else:
                dp[i][w] = dp[i - 1][w]

    return dp[n][capacity]

# 示例
values = [60, 100, 120]
weights = [10, 20, 30]
capacity = 50
print(knapsack(values, weights, capacity))  # 输出 220
```

**解析：** 这个算法使用一个二维数组 `dp` 来存储中间结果。`dp[i][w]` 表示在前 `i` 个物品中，能够装入容量为 `w` 的背包的最大价值。通过填充这个数组，我们可以得到最终的结果。

#### 6.3 最短路径算法：迪杰斯特拉算法

**题目：** 使用迪杰斯特拉算法求解无权图的最短路径问题。

**答案：** 迪杰斯特拉算法（Dijkstra's Algorithm）是一个用于求解单源最短路径的贪心算法。

```python
import heapq

def dijkstra(graph, start):
    n = len(graph)
    dist = [float('inf')] * n
    dist[start] = 0
    priority_queue = [(0, start)]

    while priority_queue:
        current_dist, current vertex = heapq.heappop(priority_queue)
        
        if current_dist > dist[current vertex]:
            continue
        
        for neighbor, weight in graph[current vertex].items():
            distance = current_dist + weight
            if distance < dist[neighbor]:
                dist[neighbor] = distance
                heapq.heappush(priority_queue, (distance, neighbor))

    return dist

# 示例
graph = {
    0: {1: 4, 7: 8},
    1: {0: 4, 2: 8, 7: 11},
    2: {1: 8, 3: 7, 8: 2},
    3: {2: 7, 4: 9, 8: 14},
    4: {3: 9, 5: 10},
    5: {4: 10, 6: 2},
    6: {5: 2, 7: 1},
    7: {0: 8, 1: 11, 6: 1},
    8: {2: 2, 3: 14, 5: 10}
}
print(dijkstra(graph, 0))  # 输出 [0, 4, 12, 10, 19, 22, 9, 8]
```

**解析：** 这个算法使用一个优先队列来存储未访问的节点，每次选择距离源点最近的节点进行扩展。通过更新节点的距离，我们可以找到从源点到其他所有节点的最短路径。
