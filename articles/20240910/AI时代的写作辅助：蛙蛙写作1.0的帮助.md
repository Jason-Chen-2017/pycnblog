### 2.1 面试题：文本纠错算法

**题目描述：** 请实现一个文本纠错算法，能够自动检测并纠正文本中的错误。

**答案解析：**

```python
def correct_text(text):
    # 假设有一个包含常见错误的文本库
    error_library = {"错别字": "正确字", "错句": "正确句"}

    # 检测文本中的错误
    for error, correction in error_library.items():
        if error in text:
            # 找到错误的位置
            error_index = text.find(error)
            # 替换错误
            text = text[:error_index] + correction + text[error_index + len(error):]

    return text
```

**解析：** 这个简单的文本纠错算法利用了一个包含常见错误和正确文本的库。当检测到文本中有错误时，它会找到错误的位置并替换为正确的文本。
