##### 5. 虚拟现实社交中的数据挖掘

**题目：** 请编写一个算法，用于分析虚拟现实社交平台上的用户行为数据，提取用户兴趣和偏好。

**答案：** 可以使用聚类算法进行分析。

```python
from sklearn.cluster import KMeans

def extract_interests(data, n_clusters):
    kmeans = KMeans(n_clusters=n_clusters)
    kmeans.fit(data)
    return kmeans.labels_
```

**解析：** 本题使用 KMeans 聚类算法分析用户行为数据，提取用户兴趣和偏好，要求考生理解聚类算法的基本原理和实现。
