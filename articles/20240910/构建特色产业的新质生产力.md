### 一、算法面试题解析

#### 1. 股票买卖

**题目：** 给定一个整数数组 `prices`，其中 `prices[i]` 是第 `i` 天股票的价格。如果在一个时期内最多只允许完成一笔交易，设计一个算法来找出最大的利润。

**答案：** 

```python
def maxProfit(prices):
    min_price = float('inf')
    max_profit = 0
    for price in prices:
        min_price = min(min_price, price)
        profit = price - min_price
        max_profit = max(max_profit, profit)
    return max_profit
```

**解析：** 维护一个 `min_price` 记录价格最低的那一天，然后计算这一天和当前天的差值，更新 `max_profit`。

#### 2. 有效的括号

**题目：** 给定一个字符串 `s` ，判断 `s` 是否是有效的括号字符串，即它满足以下条件：

1. 它是一个空字符串，或者
2. 它有一个打开的括号，并且有一个与之匹配的相应的关闭的括号。
3. 左右括号必须以正确的顺序关闭。

**答案：** 

```python
def isValid(s: str) -> bool:
    stack = []
    for c in s:
        if c in "([{":
            stack.append(c)
        elif not stack:
            return False
        elif (
            c == ")" and stack[-1] != "(" or
            c == "}" and stack[-1] != "{" or
            c == "]" and stack[-1] != "["
        ):
            return False
        else:
            stack.pop()
    return not stack
```

**解析：** 使用栈来存储遇到的左括号，当遇到右括号时，判断栈顶元素是否匹配，若匹配则出栈，否则返回 `False`。

### 二、编程题解析

#### 1. 求最大子序列和

**题目：** 给定一个整数数组 `nums` ，找到一个具有最大和的连续子序列（子数组）并返回其和。

**答案：** 

```python
def maxSubArray(nums):
    if not nums:
        return 0
    max_so_far = nums[0]
    curr_max = nums[0]
    for i in range(1, len(nums)):
        curr_max = max(nums[i], curr_max + nums[i])
        max_so_far = max(max_so_far, curr_max)
    return max_so_far
```

**解析：** 维护两个变量，`max_so_far` 记录最大子序列和，`curr_max` 记录当前子序列和。遍历数组，更新这两个变量。

#### 2. 二进制求和

**题目：** 给定两个二进制字符串，返回它们的和（用二进制表示）。

**答案：** 

```python
def addBinary(a: str, b: str) -> str:
    def add(x, y):
        while y:
            carry, x = divmod(x + y, 2)
            y = carry
        return x

    return bin(add(int(a, 2), int(b, 2)))[2:]
```

**解析：** 将字符串转化为整数，使用自定义的 `add` 函数进行二进制加法，最后将结果转化为字符串。
