```python
import numpy as np

def kmeans(data, k, max_iter=100):
    centroids = data[np.random.choice(data.shape[0], k, replace=False)]
    for _ in range(max_iter):
        clusters = assign_clusters(data, centroids)
        centroids = update_centroids(clusters, data, k)
    return centroids, clusters

def assign_clusters(data, centroids):
    distances = np.linalg.norm(data - centroids, axis=1)
    return np.argmin(distances, axis=1)

def update_centroids(clusters, data, k):
    new_centroids = np.zeros((k, data.shape[1]))
    for i in range(k):
        cluster_points = data[clusters == i]
        if cluster_points.shape[0] > 0:
            new_centroids[i] = np.mean(cluster_points, axis=0)
    return new_centroids
```

在电商搜索推荐系统中，常见的用户行为序列异常检测算法包括 K-均值聚类、局部异常因子（LOF）、孤立森林（Isolation Forest）等。以下是对这些算法的对比与分析：

| 算法         | 优点                                                                 | 缺点                                                                                         | 适用场景          |
| ------------ | -------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- | ----------------- |
| K-均值聚类   | 简单易懂，易于实现；聚类效果较好。                                      | 对初始聚类中心敏感；不适合处理高维度数据。                                                      | 低维度数据聚类   |
| 局部异常因子 | 对异常样本的检测效果较好；适用于高维度数据。                            | 可能会检测出多个局部异常；计算复杂度较高。                                                      | 高维度数据异常检测 |
| 孤立森林     | 对异常样本的检测效果较好；计算复杂度较低。                              | 对噪声敏感；不适合处理高维度数据。                                                               | 中低维度数据异常检测 |
