#### 二、集体行动中的算法编程题

**1. 如何设计一个投票系统，确保公平性？**

**题目：** 设计一个投票系统，要求保证每个选民只能投票一次，且投票结果公开透明。

**答案：** 可以使用哈希表来实现，存储选民ID和对应的投票结果。

```python
class VotingSystem:
    def __init__(self):
        self.votes = {}  # 存储投票结果，键为选民ID，值为投票选项

    def vote(self, voter_id, candidate):
        if voter_id not in self.votes:
            self.votes[voter_id] = candidate
        else:
            print("Voter has already voted.")

    def display_results(self):
        results = Counter(self.votes.values())
        for candidate, count in results.items():
            print(f"{candidate}: {count} votes")
```

**解析：** 通过哈希表来确保每个选民只能投票一次，同时可以快速统计投票结果。
