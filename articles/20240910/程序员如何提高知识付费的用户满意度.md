#### 算法题库

**1. 最长公共子序列（LCS）**
- **题目描述：** 给定两个字符串，找出它们的最长公共子序列。
- **算法思路：** 使用动态规划，构建二维数组，记录子序列的长度，最后返回最大长度。
- **代码示例：**
```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    return dp[m][n]
```

**2. 图的最短路径（Dijkstra算法）**
- **题目描述：** 给定一个无向图和一个起始节点，求图中到达其他各节点的最短路径。
- **算法思路：** 使用优先队列，每次选择未访问过的节点中距离起点最短的节点进行扩展。
- **代码示例：**
```python
import heapq

def dijkstra(graph, start):
    n = len(graph)
    distances = [float('inf')] * n
    distances[start] = 0
    priority_queue = [(0, start)]
    
    while priority_queue:
        current_distance, current_vertex = heapq.heappop(priority_queue)
        
        if current_distance > distances[current_vertex]:
            continue
        
        for neighbor, weight in graph[current_vertex].items():
            distance = current_distance + weight
            
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(priority_queue, (distance, neighbor))
    
    return distances
```

**3. 文本相似度计算（余弦相似度）**
- **题目描述：** 给定两个文本，计算它们之间的相似度。
- **算法思路：** 计算两个文本向量的余弦相似度，公式为两个向量点积除以两个向量长度的乘积。
- **代码示例：**
```python
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

def text_similarity(text1, text2):
    vector1 = np.array([word2vec(text1)])
    vector2 = np.array([word2vec(text2)])
    
    similarity = cosine_similarity(vector1, vector2)[0][0]
    return similarity
```

**4. 在线用户活跃度分析（滑动窗口算法）**
- **题目描述：** 给定一组用户行为数据，计算每个用户在一段时间内的活跃度。
- **算法思路：** 维护一个滑动窗口，对窗口内的数据进行统计和分析，窗口滑动时更新统计结果。
- **代码示例：**
```python
def user_activity(data, window_size):
    activities = []
    current_window = []
    
    for event in data:
        current_window.append(event)
        
        if len(current_window) > window_size:
            activities.append(average_activity(current_window))
            current_window.pop(0)
    
    activities.append(average_activity(current_window))
    
    return activities

def average_activity(window):
    return sum(window) / len(window)
```
