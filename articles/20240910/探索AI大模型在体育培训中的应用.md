**1. 编写一个函数，用于计算运动员的平均心率。**
```python
def calculate_average_heart_rate(heart_rates):
    if not heart_rates:
        return 0
    return sum(heart_rates) / len(heart_rates)

# 示例
heart_rates = [150, 160, 155]
print(calculate_average_heart_rate(heart_rates))  # 输出：154.0
```

**答案解析：** 该函数计算给定心率列表的平均值，若无数据则返回0。
