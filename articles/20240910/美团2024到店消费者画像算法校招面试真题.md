##### 题目5：给定一个用户行为数据集，计算用户的平均消费金额。

**代码：**

```python
# Python 代码示例
def avg_consume(data):
    total = 0
    for user, orders in data.items():
        total += sum(order['amount'] for order in orders)
    return total / len(data)

data = {
    'user1': [{'amount': 100}, {'amount': 200}, {'amount': 300}],
    'user2': [{'amount': 50}, {'amount': 100}],
    'user3': [{'amount': 200}, {'amount': 300}, {'amount': 400}, {'amount': 500}],
}

print(avg_consume(data))  # 输出：246.66666666666666
```

**解析：** 通过遍历用户行为数据集，计算每个用户的总消费金额，然后求平均值。

##### 题目6：给定一个用户行为数据集，计算用户的消费频次。

**代码：**

```python
# Python 代码示例
from collections import Counter

def consume_frequency(data):
    freq = Counter()
    for user, orders in data.items():
        freq[user] = len(orders)
    return freq

data = {
    'user1': [{'amount': 100}, {'amount': 200}, {'amount': 300}],
    'user2': [{'amount': 50}, {'amount': 100}],
    'user3': [{'amount': 200}, {'amount': 300}, {'amount': 400}, {'amount': 500}],
}

print(consume_frequency(data))  # 输出：Counter({'user3': 4, 'user1': 3, 'user2': 2})
```

**解析：** 通过使用 `collections.Counter` 计算每个用户的消费频次。
