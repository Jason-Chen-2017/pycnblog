#### 3. 深海路径规划算法

**题目描述：** 深海仿生机器人需要从起点到达终点，但在海底场景中存在障碍物。请设计一个算法，计算从起点到终点的最短路径。场景由一个二维数组表示，其中‘A’表示可以通行的区域，‘B’表示障碍物，其他字符表示海洋。

**输入格式：** 两个二维数组，分别表示起点和终点的坐标。

**输出格式：** 一个二维数组，表示从起点到终点的最短路径。

**样例输入：**
```
4 4
0 0
2 3
AABAB
```

**样例输出：**
```
0 1 1 1 2
0 0 1 1 2
1 1 1 1 2
1 1 1 1 2
```

**代码实现：**

```python
def find_shortest_path(width, height, start, end, grid):
    # 采用BFS算法寻找最短路径
    from collections import deque

    queue = deque([start])
    visited = [[False for _ in range(width)] for _ in range(height)]
    visited[start[0]][start[1]] = True

    while queue:
        x, y = queue.popleft()
        if (x, y) == end:
            break
        for dx, dy in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
            nx, ny = x + dx, y + dy
            if 0 <= nx < height and 0 <= ny < width and not visited[nx][ny] and grid[nx][ny] == 'A':
                visited[nx][ny] = True
                queue.append((nx, ny))

    # 从终点回溯到起点构建路径
    path = []
    x, y = end
    while (x, y) != start:
        path.append((x, y))
        for dx, dy in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
            nx, ny = x + dx, y + dy
            if 0 <= nx < height and 0 <= ny < width and visited[nx][ny]:
                x, y = nx, ny
    path.reverse()

    return path

# 测试样例
width, height = map(int, input().split())
start = tuple(map(int, input().split()))
end = tuple(map(int, input().split()))
grid = [input() for _ in range(height)]
path = find_shortest_path(width, height, start, end, grid)
for row in path:
    print(' '.join(map(str, row)))
```

#### 6. 深海障碍物规避算法

**题目描述：** 深海仿生机器人在前进过程中遇到障碍物，需要计算并执行规避路径。请设计一个算法，计算并输出从当前点到障碍物，再返回到原点的最优规避路径。假设机器人每次只能向上下左右四个方向移动。

**输入格式：** 四个整数，分别表示机器人的当前位置（x, y）和障碍物的位置（a, b）。

**输出格式：** 一个二维数组，表示规避路径。

**样例输入：**
```
1 1
3 2
```

**样例输出：**
```
1 2
3 2
3 1
1 1
```

**代码实现：**

```python
def evade_obstacle(x, y, a, b):
    # 使用BFS算法找到从(x, y)到(a, b)的路径
    queue = deque([(x, y)])
    visited = set([(x, y)])
    paths = {(-1, 0): [], (1, 0): [], (0, -1): [], (0, 1): []}

    while queue:
        x, y = queue.popleft()
        if (x, y) == (a, b):
            break
        for dx, dy in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
            nx, ny = x + dx, y + dy
            if 0 <= nx < 10 and 0 <= ny < 10 and (nx, ny) not in visited and grid[nx][ny] == 'A':
                visited.add((nx, ny))
                queue.append((nx, ny))
                paths[(nx - x, ny - y)].append((x, y))

    # 找到从(a, b)返回到(x, y)的路径
    reverse_paths = {(-1, 0): [], (1, 0): [], (0, -1): [], (0, 1): []}
    for key, path in paths.items():
        reverse_paths[(-key[0], -key[1])].extend(path)

    # 构建完整的规避路径
    evade_path = []
    for i in range(len(reverse_paths[(-1, 0)])):
        evade_path.extend(reverse_paths[(-1, 0)][i:])
        evade_path.extend(reverse_paths[(1, 0)][i:])
        evade_path.extend(reverse_paths[(0, -1)][i:])
        evade_path.extend(reverse_paths[(0, 1)][i:])

    return evade_path

# 测试样例

    print(point[0], point[1])
```

#### 23. 深海生物行为预测——遗传算法版

**题目描述：** 使用遗传算法预测深海生物的未来行为。请设计一个算法，通过交叉、变异等遗传操作，优化生物行为的预测。

**输入格式：** 多行，每行包含两个整数，分别表示生物的历史行为（纬度，经度）。

**输出格式：** 输出预测的未来行为。

**样例输入：**
```
10 10

10 20
```

**样例输出：**
```

  [11, 11]

```

**代码实现：**

```python
import random

def crossover(parent1, parent2):
    # 单点交叉
    crossover_point = random.randint(1, len(parent1) - 1)
    child = parent1[:crossover_point] + parent2[crossover_point:]
    return child

def mutate(child):
    # 变异操作，随机改变一个点
    mutate_point = random.randint(0, len(child) - 1)
    child[mutate_point] = (child[mutate_point][0] + random.randint(-1, 1), child[mutate_point][1] + random.randint(-1, 1))
    return child

def genetic_algorithm(behavior, generations, mutation_rate):
    population = [behavior]  # 初始种群
    for _ in range(generations):
        # 交叉操作
        new_population = []
        for _ in range(len(population) // 2):
            parent1, parent2 = random.sample(population, 2)
            child = crossover(parent1, parent2)
            new_population.append(child)
        
        # 变异操作
        for child in new_population:
            if random.random() < mutation_rate:
                mutate(child)

        population = new_population

    # 选择最佳个体
    best_child = min(population, key=lambda x: sum([abs(x[i][0] - behavior[i][0])**2 + abs(x[i][1] - behavior[i][1])**2 for i in range(len(behavior))]))
    return best_child

# 测试样例

print(best_child)
```

**题目描述：** 使用遗传算法优化深海仿生机器人的能效。请设计一个算法，通过交叉、变异等遗传操作，找到最优的能量消耗模式。

**输入格式：** 多行，每行包含两个整数，分别表示任务的复杂度和能量消耗。

**输出格式：** 输出最优的能量消耗模式。

**样例输入：**
```
2 3
1 2
3 5
4 7
```

**样例输出：**
```
[
  [3, 1],
  [2, 4]
]
```

**代码实现：**

```python
import random

def crossover(parent1, parent2):
    # 顺序交叉
    crossover_point = random.randint(1, len(parent1) - 1)
    child = parent1[:crossover_point] + parent2[crossover_point:]
    return child

def mutate(child):
    # 变异操作，随机改变任务顺序
    mutate_point = random.randint(0, len(child) - 1)
    child[mutate_point] = child[mutate_point - 1]
    child[mutate_point - 1] = child[mutate_point]
    return child

def genetic_algorithm(tasks, generations, mutation_rate):
    population = [tasks]  # 初始种群
    for _ in range(generations):
        # 交叉操作
        new_population = []
        for _ in range(len(population) // 2):
            parent1, parent2 = random.sample(population, 2)
            child = crossover(parent1, parent2)
            new_population.append(child)
        
        # 变异操作
        for child in new_population:
            if random.random() < mutation_rate:
                mutate(child)

        population = new_population

    # 选择最佳个体
    best_child = min(population, key=lambda x: sum([tasks[i][1] for i in range(len(tasks)) if x[i][0] == tasks[i][0]]))
    return best_child

# 测试样例
tasks = [tuple(map(int, input().split())) for _ in range(int(input()))]
best_child = genetic_algorithm(tasks, 50, 0.1)
print(best_child)
```
