**代码示例：** 使用 Python 设计一个优惠券系统：

```python

class Coupon:
    def __init__(self, code, discount, validity, type):
        self.code = code
        self.discount = discount
        self.validity = validity
        self.type = type

    def apply_coupon(self, price):
        if self.type == "new_user" and price >= 100:
            return price * (1 - self.discount)
        elif self.type == "recurring" and price >= 200:
            return price * (1 - self.discount)
        else:
            return price

coupon2 = Coupon("RECURRING20", 0.2, "2023-04-30", "recurring")

print(f"Original price: {price}")
print(f"Price after applying coupon {coupon1.code}: {coupon1.apply_coupon(price)}")
print(f"Price after applying coupon {coupon2.code}: {coupon2.apply_coupon(price)}")
```
