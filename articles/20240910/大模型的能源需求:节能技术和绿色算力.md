#### 算法编程题库

**1. 编写一个算法，计算给定字符串中重复字符的个数。**

**题目描述：** 给定一个字符串，编写一个算法，计算字符串中重复字符的个数。

**示例：**
输入：`"hello"`，输出：`4`

**答案：**

```python
def count_repeated_chars(s):
    count = 0
    char_set = set()
    for char in s:
        if char in char_set:
            count += 1
        else:
            char_set.add(char)
    return count

s = "hello"
print(count_repeated_chars(s))  # 输出：4
```

**解析：** 该算法使用一个集合（set）来存储已经遍历过的字符。遍历字符串，对于每个字符，如果已经在集合中，则表示该字符重复，计数器加一；否则，将该字符加入集合。遍历完成后，返回计数器的值。

**2. 编写一个算法，实现两个整数数的加法，不使用 `+` 或 `-` 运算符。**

**题目描述：** 编写一个算法，实现两个整数数的加法，不使用 `+` 或 `-` 运算符。

**示例：**
输入：`3` 和 `5`，输出：`8`

**答案：**

```python
def add_without_plus_minus(a, b):
    while b != 0:
        carry = a & b
        a = a ^ b
        b = carry << 1
    return a

a = 3
b = 5
print(add_without_plus_minus(a, b))  # 输出：8
```

**解析：** 该算法使用位运算实现加法操作。首先计算两个数的按位与（`&`），得到进位；然后计算两个数的按位异或（`^`），得到无进位的和；最后将进位移到下一位。循环执行这个过程，直到进位为0，此时无进位的和即为两个数的和。
