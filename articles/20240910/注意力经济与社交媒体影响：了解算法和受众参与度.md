### 7. 实现一个基于K-means算法的聚类函数

**题目：** 编写一个Python函数，实现K-means算法，将一组数据划分为K个簇。

**答案：**

```python
import numpy as np

def k_means(data, k, max_iterations=100):
    # 初始化簇中心
    centroids = data[np.random.choice(data.shape[0], k, replace=False)]
    for _ in range(max_iterations):
        # 计算每个数据点到簇中心的距离
        distances = np.linalg.norm(data[:, np.newaxis] - centroids, axis=2)
        # 分配簇
        clusters = np.argmin(distances, axis=1)
        # 更新簇中心
        new_centroids = np.array([data[clusters == i].mean(axis=0) for i in range(k)])
        # 判断收敛
        if np.all(centroids == new_centroids):
            break
        centroids = new_centroids
    return clusters, centroids
```

**解析：** 该函数首先随机初始化K个簇中心，然后通过迭代更新簇中心，直到簇中心不再变化或达到最大迭代次数。在每次迭代中，计算每个数据点到簇中心的距离，根据距离最近的簇中心将数据点分配到相应的簇，并更新簇中心。

```python
import numpy as np

def content_based_recommender(user_interests, item_features, similarity_metric='cosine', k=5):
    # 计算用户兴趣和项目特征之间的相似度矩阵
    similarity_matrix = cosine_similarity(user_interests.reshape(1, -1), item_features)
    # 选择相似度最高的 k 个项目
    recommended_items = np.argpartition(similarity_matrix, -k, axis=1)[:, -k:]
    return recommended_items
```

**解析：** 该函数首先计算用户兴趣和项目特征之间的相似度矩阵，然后选择相似度最高的 k 个项目作为推荐结果。在计算相似度时，可以使用不同的相似度度量方法，如余弦相似度、欧氏距离等。
