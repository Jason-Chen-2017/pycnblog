#### 三、答案解析

**1. 替换重复单词**

```python
def replace_duplicate_words(sentence, replace_str):
    words = sentence.split()
    word_set = set()
    result = []

    for word in words:
        if word not in word_set:
            result.append(word)
            word_set.add(word)
        else:
            result.append(replace_str)

    return ' '.join(result)
```

**2. 提取数字并排序**

```python
def extract_and_sort_numbers(sentence):
    numbers = [int(s) for s in sentence.split() if s.isdigit()]
    numbers.sort(reverse=True)
    return numbers
```

**3. 判断变形词**

```python
def is_anagram(str1, str2):
    return sorted(str1) == sorted(str2)
```
