### 5. 请实现基于用户协同过滤的推荐算法。

**答案：** 基于用户协同过滤的推荐算法可以通过计算用户之间的相似度矩阵，根据相似度矩阵为用户生成推荐列表。

```python
# 假设用户行为数据存储在user_item_matrix中，其中user_item_matrix[i][j]表示用户i对商品j的评分。
def user_based_cf(user_item_matrix):
    # 计算用户之间的相似度矩阵
    similarity_matrix = compute_similarity(user_item_matrix)
    
    # 为用户生成推荐列表
    recommendations = []
    for user in range(len(user_item_matrix)):
        user_similarity_scores = []
        for other_user in range(len(user_item_matrix)):
            if other_user == user:
                continue
            similarity_score = similarity_matrix[user][other_user]
            user_similarity_scores.append((other_user, similarity_score))
        user_similarity_scores.sort(key=lambda x: x[1], reverse=True)
        
        # 根据用户相似度分数生成推荐列表
        for other_user, similarity_score in user_similarity_scores:
            for item in user_item_matrix[other_user]:
                recommendations.append((item, similarity_score))
    return recommendations
```
