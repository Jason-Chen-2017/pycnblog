#### 一、面试题

##### 1. 集合的基本运算有哪些？

**题目：** 请简要介绍集合的基本运算，并给出每个运算的示例。

**答案：** 集合的基本运算包括：

- 并集（Union）：将两个集合中的所有元素合并成一个集合，去除重复元素。
- 交集（Intersection）：取两个集合共有的元素组成一个新的集合。
- 差集（Difference）：从第一个集合中移除与第二个集合共有的元素。
- 对称差（Symmetric Difference）：两个集合中不同的元素组成的新集合。

**示例：**

```python
# Python 示例

# 定义两个集合
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

# 并集
union = A | B  # {1, 2, 3, 4, 5, 6}

# 交集
intersection = A & B  # {3, 4}

# 差集
difference = A - B  # {1, 2}

# 对称差
symmetric_difference = A ^ B  # {1, 2, 5, 6}
```

##### 2. 如何判断一个集合是否为另一个集合的子集？

**题目：** 请使用 Python 编写一个函数，判断一个集合是否为另一个集合的子集。

**答案：**

```python
def is_subset(set1, set2):
    return set1.issubset(set2)

# 示例
A = {1, 2, 3}
B = {1, 2, 3, 4, 5}

print(is_subset(A, B))  # 输出 True
```

##### 3. 如何对集合进行排序？

**题目：** 请使用 Python 编写一个函数，对集合中的元素进行排序。

**答案：**

```python
def sort_set(s):
    return sorted(s)

# 示例
A = {3, 1, 4, 2}
sorted_A = sort_set(A)
print(sorted_A)  # 输出 [1, 2, 3, 4]
```

#### 二、算法编程题

##### 1. 判断两个集合的交集和并集

**题目：** 编写一个函数，计算两个集合的交集和并集。

**答案：**

```python
def intersection_and_union(set1, set2):
    intersection = set1.intersection(set2)
    union = set1.union(set2)
    return intersection, union

# 示例
A = {1, 2, 3}
B = {3, 4, 5}
intersection, union = intersection_and_union(A, B)
print("Intersection:", intersection)  # 输出 {3}
print("Union:", union)  # 输出 {1, 2, 3, 4, 5}
```
