#### 1. 用户留存率定义及计算

**题目：** 请解释用户留存率的定义，并给出计算用户留存率的公式。

**答案：** 用户留存率是指在特定时间段内，回到平台进行互动的用户占该时间段内首次访问用户的比例。计算公式如下：

\[ \text{用户留存率} = \frac{\text{第 } n \text{ 天回访用户数}}{\text{第 } n \text{ 天首次访问用户数}} \times 100\% \]

**举例：** 假设第7天的回访用户数为100，第7天的首次访问用户数为500，则第7天的用户留存率为：

\[ \text{用户留存率} = \frac{100}{500} \times 100\% = 20\% \]
