### 4. 算法编程题库

**题目1：** 编写一个算法，计算给定数据集中各环境指标的标准差。

**答案：** 

```python
def calculate_std_dev(data):
    n = len(data)
    mean = sum(data) / n
    variance = sum((x - mean) ** 2 for x in data) / n
    std_dev = variance ** 0.5
    return std_dev

data = [10, 20, 30, 40, 50]
std_dev = calculate_std_dev(data)
print("标准差：", std_dev)
```

**解析：** 该算法首先计算给定数据集的平均值，然后计算方差，最后求方差的平方根得到标准差。

**题目2：** 编写一个算法，判断给定字符串是否为有效的环保标语。

**答案：**

```python
def is_valid_slogan(slogan):
    valid_characters = set('abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789-,;:._')
    words = slogan.split()
    for word in words:
        if not all(char in valid_characters for char in word):
            return False
    return True

slogan = "绿色生活，美好未来！"
if is_valid_slogan(slogan):
    print("有效的环保标语")
else:
    print("无效的环保标语")
```

**解析：** 该算法首先将字符串按空格分隔成单词，然后判断每个单词中的字符是否为有效的环保标语字符。如果所有单词都符合条件，则返回 True，否则返回 False。
