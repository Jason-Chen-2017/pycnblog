## 三、算法编程题库及解析

### 1. 如何实现基于距离的聚类算法？

**题目：** 请实现一个基于距离的聚类算法，将一组数据划分为若干个簇。

**答案：** 基于距离的聚类算法（如 K-Means 算法）的基本步骤如下：

1. **初始化聚类中心**：随机选择 K 个数据点作为聚类中心。
2. **分配数据点**：计算每个数据点到聚类中心的距离，将数据点分配到距离最近的聚类中心。
3. **更新聚类中心**：重新计算每个簇的平均值，作为新的聚类中心。
4. **迭代直至收敛**：重复步骤 2 和 3，直到聚类中心不再发生显著变化。

**代码示例：**

```python
import numpy as np

def k_means(data, k, max_iter=100):
    # 初始化聚类中心
    centroids = data[np.random.choice(data.shape[0], k, replace=False)]
    
    for _ in range(max_iter):
        # 分配数据点
        distances = np.linalg.norm(data[:, np.newaxis] - centroids, axis=2)
        labels = np.argmin(distances, axis=1)
        
        # 更新聚类中心
        new_centroids = np.array([data[labels == k].mean(axis=0) for k in range(k)])
        
        # 判断是否收敛
        if np.linalg.norm(new_centroids - centroids) < 1e-5:
            break

        centroids = new_centroids
    
    return centroids, labels

# 测试数据
data = np.random.rand(100, 2)
k = 3

# 执行 K-Means 算法
centroids, labels = k_means(data, k)

# 输出聚类结果
print("聚类中心：", centroids)
print("数据点标签：", labels)
```

### 2. 如何实现神经网络的前向传播算法？

**题目：** 请实现一个简单的神经网络，实现前向传播算法。

**答案：** 神经网络的前向传播算法包括以下几个步骤：

1. **初始化参数**：包括权重（weights）和偏置（biases）。
2. **输入数据**：将输入数据通过网络的每一层进行计算。
3. **激活函数**：使用激活函数（如 sigmoid、ReLU）对中间层和输出层的计算结果进行非线性变换。
4. **计算输出**：得到网络最终的输出结果。

**代码示例：**

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def forwardPropagation(x, weights, biases):
    # 初始化输出
    a = x
    
    # 遍历网络层，计算前向传播
    for i in range(len(weights)):
        z = np.dot(a, weights[i]) + biases[i]
        a = sigmoid(z)
    
    return a

# 测试数据
x = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])

# 初始化权重和偏置
weights = [np.random.randn(y, x) for x, y in zip(x.shape, (x.shape[0], 1))]
biases = [np.random.randn(y, 1) for y in x.shape]

# 执行前向传播
output = forwardPropagation(x, weights, biases)

# 输出输出结果
print("输出结果：", output)
```
