### 1.1 链表反转

**题目：** 实现一个函数，反转单链表。

```go
func reverseList(head *ListNode) *ListNode {
    // 请在这里实现代码
}
```

**答案解析：**

```go
func reverseList(head *ListNode) *ListNode {
    var prev, curr *ListNode
    curr = head
    for curr != nil {
        nextTemp := curr.Next
        curr.Next = prev
        prev = curr
        curr = nextTemp
    }
    return prev
}
```

### 1.2 快排

**题目：** 实现快速排序算法。

```go
func quickSort(nums []int) []int {
    // 请在这里实现代码
}
```

**答案解析：**

```go
func quickSort(nums []int) []int {
    if len(nums) <= 1 {
        return nums
    }
    pivot := nums[len(nums)/2]
    left, right := 0, len(nums)-1
    for {
        for nums[left] < pivot {
            left++
        }
        for nums[right] > pivot {
            right--
        }
        if left >= right {
            break
        }
        nums[left], nums[right] = nums[right], nums[left]
        left++
        right--
    }
    quickSort(nums[:left])
    quickSort(nums[left:])
    return nums
}
```

**答案解析：**

```go
func preOrder(root *TreeNode) []int {
    result := []int{}
    if root != nil {
        result = append(result, root.Val)
        result = append(result, preOrder(root.Left)...)
        result = append(result, postOrder(root.Right)...)
    }
    return result
}

func inOrder(root *TreeNode) []int {
    result := []int{}
    if root != nil {
        result = append(result, inOrder(root.Left)...)
        result = append(result, root.Val)
        result = append(result, inOrder(root.Right)...)
    }
    return result
}

func postOrder(root *TreeNode) []int {
    result := []int{}
    if root != nil {
        result = append(result, postOrder(root.Left)...)
        result = append(result, postOrder(root.Right)...)
        result = append(result, root.Val)
    }
    return result
}
```

### 2.2 TCP、UDP 协议区别

**题目：** TCP 和 UDP 协议的区别是什么？

**答案：**

TCP（传输控制协议）和 UDP（用户数据报协议）是两种常用的网络传输协议，它们的主要区别如下：

* **连接性：** TCP 是面向连接的，UDP 是无连接的。TCP 需要在通信双方建立连接后才能传输数据，而 UDP 则不需要。
* **可靠性：** TCP 提供可靠的数据传输，通过确认和重传机制保证数据的完整性；UDP 则不提供可靠性，数据传输过程中可能会丢失或重复。
* **速度：** TCP 由于需要进行连接建立、数据确认等操作，传输速度相对较慢；UDP 则没有这些开销，传输速度相对较快。
* **应用场景：** TCP 适用于对数据完整性要求较高的应用，如文件传输、邮件传输等；UDP 适用于对实时性要求较高的应用，如视频直播、在线游戏等。

### 3.2 接口与抽象类

**题目：** Go 语言中如何实现接口和抽象类？

**答案：**

Go 语言中没有抽象类，但是可以通过接口（interface）来实现类似的功能。接口是一种抽象的类型，它只定义了方法而不实现任何方法，实现了接口的类型必须实现接口中的所有方法。

```go
// 接口定义
type Animal interface {
    Speak() string
}

// 实现接口
type Dog struct {}

func (d *Dog) Speak() string {
    return "汪汪汪"
}

func main() {
    dog := Dog{}
    animal := Animal(dog)
    fmt.Println(animal.Speak()) // 输出：汪汪汪
}
```
