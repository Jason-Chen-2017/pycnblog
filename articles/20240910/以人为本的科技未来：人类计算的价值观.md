## 算法编程题库

### 1. 暴力解法

**题目：** 给定一个整数数组，找到两个数，它们的和等于目标值。写出你的算法并给出一个实例。

**答案：** 
```python
def two_sum(nums, target):
    for i in range(len(nums)):
        for j in range(i+1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]
    return []

# 实例
nums = [2, 7, 11, 15]
target = 9
result = two_sum(nums, target)
print(result)  # 输出 [0, 1]
```

### 2. 动态规划

**题目：** 给定一个整数数组，找到最长的等差数列的长度。写出你的算法并给出一个实例。

**答案：** 
```python
def longest_ascending_subsequence(nums):
    n = len(nums)
    dp = [1] * n
    for i in range(1, n):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

# 实例
nums = [3, 10, 2, 1, 20]
result = longest_ascending_subsequence(nums)
print(result)  # 输出 4
```

### 3. 回溯算法

**题目：** 给定一个字符串，找出其中所有的回文字符串。写出你的算法并给出一个实例。

**答案：** 
```python
def find_palindromes(s):
    def backtrack(start, end):
        if start >= end:
            palindromes.append(s[start:end+1])
            return
        for i in range(start, end+1):
            if s[start] == s[i]:
                backtrack(start+1, i-1)
                backtrack(i+1, end)

    palindromes = []
    backtrack(0, len(s)-1)
    return palindromes

# 实例
s = "abaxyzzyxf"
result = find_palindromes(s)
print(result)  # 输出 ['xyzzyx', 'axyza', 'axyz', 'xyzy', 'xyz', 'x', 'y', 'z']
```

### 4. 前缀树

**题目：** 实现一个前缀树（Trie），包括插入、搜索和搜索前缀的功能。

**答案：** 
```python
class Trie:
    def __init__(self):
        self.children = {}
        self.is_end_of_word = False

    def insert(self, word):
        node = self
        for char in word:
            if char not in node.children:
                node.children[char] = Trie()
            node = node.children[char]
        node.is_end_of_word = True

    def search(self, word):
        node = self
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end_of_word

    def search_prefix(self, prefix):
        node = self
        for char in prefix:
            if char not in node.children:
                return False
            node = node.children[char]
        return True

# 实例
trie = Trie()
trie.insert("apple")
trie.insert("app")
trie.insert("bat")
print(trie.search("apple"))  # 输出 True
print(trie.search("app"))  # 输出 True
print(trie.search("bat"))  # 输出 True
print(trie.search("appz"))  # 输出 False
print(trie.search_prefix("app"))  # 输出 True
print(trie.search_prefix("appz"))  # 输出 False
```

### 4. 暴力解法

暴力解法是一种简单的算法思路，通过遍历数组，尝试所有可能的组合，找到满足条件的两个数。这种方法的时间复杂度为 O(n^2)，适用于数据量较小的场景。

```python
def two_sum(nums, target):
    for i in range(len(nums)):
        for j in range(i+1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]
    return []
```
