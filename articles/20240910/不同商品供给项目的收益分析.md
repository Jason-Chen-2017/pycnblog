#### 4. 最小生成树算法（Prim算法）

**题目：** 使用Prim算法求解一个无向图的最小生成树，并输出树的权重总和。

**答案：** 

```python
from collections import defaultdict

def prim_algorithm(graph):
    n = len(graph)
    parent = [None] * n
    key = [float('inf')] * n
    mst = []
    key[0] = 0
    in_mst = [False] * n

    for _ in range(n):
        min_key = float('inf')
        min_index = -1

        for i in range(n):
            if not in_mst[i] and key[i] < min_key:
                min_key = key[i]
                min_index = i

        in_mst[min_index] = True
        mst.append((min_index, parent[min_index], key[min_index]))

        for i in range(n):
            if not in_mst[i]:
                if graph[min_index][i] < key[i]:
                    key[i] = graph[min_index][i]
                    parent[i] = min_index

    return mst

# 输入图的数据
graph = [
    [0, 2, 4, 0, 0],
    [2, 0, 1, 3, 0],
    [4, 1, 0, 6, 2],
    [0, 3, 6, 0, 7],
    [0, 0, 2, 7, 0]
]

mst = prim_algorithm(graph)
print("最小生成树：", mst)
print("权重总和：", sum([edge[2] for edge in mst]))
```

**解析：** Prim算法通过迭代选择最小权重边来构建最小生成树，适合于权值较小的稀疏图。

#### 5. 贪心算法求解硬币找零问题

**题目：** 给定一个硬币面额数组和一个目标金额，使用贪心算法求解最小硬币数量。

**答案：** 

```python
def coin_change(coins, amount):
    coins.sort(reverse=True)
    result = 0
    for coin in coins:
        while amount >= coin:
            amount -= coin
            result += 1
    return -1 if amount > 0 else result

coins = [1, 2, 5]
amount = 11
print("最小硬币数量：", coin_change(coins, amount))
```

**解析：** 贪心算法通过从大到小选取硬币，每次选取尽可能多的硬币，以达到最小硬币数量。
