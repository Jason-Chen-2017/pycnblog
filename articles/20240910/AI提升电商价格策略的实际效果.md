#### 5. 编写一个价格预测算法

**题目：** 编写一个简单的价格预测算法，根据历史销售数据和当前市场情况预测商品未来价格。

**答案：**

```python
import numpy as np

def price_prediction(historical_sales, current_market):
    """
    价格预测算法
    :param historical_sales: 历史销售数据
    :param current_market: 当前市场情况数据
    :return: 预测的商品未来价格
    """
    # 计算历史平均价格
    historical_avg_price = np.mean(historical_sales)

    # 考虑当前市场情况调整价格
    if current_market['demand'] > historical_avg_price:
        predicted_price = historical_avg_price * 1.05  # 需求高于历史平均，适当提价
    else:
        predicted_price = historical_avg_price * 0.95  # 需求低于历史平均，适当降价

    return predicted_price
```
