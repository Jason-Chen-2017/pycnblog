1. **题目：给定一个整数数组，找出满足条件的子序列，使得子序列中任意两个元素的差值不超过 k。**

   **解析：** 可以使用哈希表记录数组中每个元素的出现次数，然后遍历数组，对于当前元素，判断其是否与之前出现的元素差值不超过 k，如果是，则更新结果。

   ```python
   def findSubsequence(nums, k):
       cnt = Counter(nums)
       ans = 0
       for v in nums:
           if cnt[v-k] > 0:
               ans += cnt[v-k]
               cnt[v-k] = 0
           cnt[v] -= 1
       return ans
   ```

2. **题目：给定一个整数数组，找出满足条件的子序列，使得子序列中任意两个元素的差值恰好为 k。**

   **解析：** 可以使用贪心算法，先对数组进行排序，然后从大到小遍历数组，对于当前元素，判断其与下一个元素的差值是否为 k，如果是，则将下一个元素加入子序列，否则跳出循环。

   ```python
   def findSubsequence(nums, k):
       nums.sort(reverse=True)
       ans = []
       for v in nums:
           if len(ans) == 0 or v-ans[-1] != k:
               break
           ans.append(v)
       return ans
   ```

       cnt = Counter(nums)
       ans = 0
       for v in nums:
           if cnt[v-k] > 0:
               ans += cnt[v-k]
               cnt[v-k] = 0
           cnt[v] -= 1
       return ans
   ```

2. **题目：给定一个整数数组，找出满足条件的子序列，使得子序列中任意两个元素的差值恰好为 k。**

   **解析：** 可以使用贪心算法，先对数组进行排序，然后从大到小遍历数组，对于当前元素，判断其与下一个元素的差值是否为 k，如果是，则将下一个元素加入子序列，否则跳出循环。

   ```python
   def findSubsequence(nums, k):
       nums.sort(reverse=True)
       ans = []
       for v in nums:
           if len(ans) == 0 or v-ans[-1] != k:
               break
           ans.append(v)
       return ans
   ```
