## 一、算法编程题解析

### 1. 颠倒整数

**题目描述：** 给出一个 32 位有符号整数，你需要将这个整数旋转 180 度（0 保留），返回旋转后的整数。

**示例：** 输入：123，输出：321。

**答案：**

```python
class Solution:
    def reverse(self, x: int) -> int:
        ans = 0
        for _ in range(32):
            if x == 0:
                break
            ans = (ans << 1) + (ans << 3)
            t = x % 10
            x = x // 10
            ans += t
        if ans < -2**31 or ans > 2**31 - 1:
            return 0
        return ans
```

**解析：** 该题主要考察对位运算和循环控制的理解。通过循环将输入整数的每一位逆序放入结果中，最后判断结果是否在 32 位整数的范围内。

### 2. 最长公共前缀

**题目描述：** 编写一个函数来查找字符串数组中的最长公共前缀。

**示例：** 输入：["flower","flow","flight"]，输出："fl"。

**答案：**

```python
class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        if not strs:
            return ""
        prefix = ""
        for c in sorted(strs, key=len)[0]:
            prefix += c
            if not all(s.startswith(prefix) for s in strs):
                break
        return prefix
```

**解析：** 该题主要考察对字符串和排序的理解。通过比较每个字符串的长度，找到最短字符串的前缀，然后逐个比较字符是否相同，直到找到一个不满足条件的字符。

## 二、面试题解析

### 1. 快排

**题目描述：** 实现快速排序算法。

**答案：**

```python
class Solution:
    def quicksort(self, arr: List[int]) -> List[int]:
        if len(arr) <= 1:
            return arr
        pivot = arr[len(arr) // 2]
        left = [x for x in arr if x < pivot]
        middle = [x for x in arr if x == pivot]
        right = [x for x in arr if x > pivot]
        return self.quicksort(left) + middle + self.quicksort(right)
```

**解析：** 该题主要考察对分治算法和递归的理解。通过选择一个基准值，将数组分为小于基准值、等于基准值和大于基准值的三个部分，然后对这三个部分分别进行快速排序。

### 2. 链表相交

**题目描述：** 给定两个链表，找出它们的第一个公共节点。

**答案：**

```python
class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> ListNode:
        pa, pb = headA, headB
        while pa != pb:
            pa = pa.next if pa else headB
            pb = pb.next if pb else headA
        return pa
```

**解析：** 该题主要考察对链表和循环的理解。通过让两个指针分别遍历两个链表，当两个指针相遇时，即为它们的第一个公共节点。
