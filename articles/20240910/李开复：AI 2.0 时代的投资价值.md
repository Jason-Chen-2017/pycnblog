##### 1. 用Python实现一个简单的KNN分类器。

**答案：** KNN（K-Nearest Neighbors）分类器是一种基于实例的机器学习算法，其基本思想是找到训练集中与测试实例最相似的K个邻近样本，并基于这K个样本的标签进行预测。

```python
from collections import Counter
import numpy as np

def euclidean_distance(x1, x2):
    return np.sqrt(np.sum((x1 - x2)**2))

class KNNClassifier:
    def __init__(self, k):
        self.k = k
        
    def fit(self, X, y):
        self.X_train = X
        self.y_train = y
        
    def predict(self, X):
        predictions = []
        for sample in X:
            distances = [euclidean_distance(sample, x) for x in self.X_train]
            k_indices = np.argsort(distances)[:self.k]
            k_nearest_labels = [self.y_train[i] for i in k_indices]
            most_common = Counter(k_nearest_labels).most_common(1)[0][0]
            predictions.append(most_common)
        return np.array(predictions)
```

##### 2. 实现一个基于梯度下降的线性回归模型。

**答案：** 线性回归是一种用于预测数值型输出的机器学习算法，其目标是找到一条直线，使得预测值与实际值之间的误差最小。梯度下降是一种常用的优化算法，用于寻找最小化损失函数的参数。

```python
import numpy as np

class LinearRegression:
    def __init__(self):
        self.w = None
    
    def fit(self, X, y):
        self.w = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)
        
    def predict(self, X):
        return X.dot(self.w)
```
