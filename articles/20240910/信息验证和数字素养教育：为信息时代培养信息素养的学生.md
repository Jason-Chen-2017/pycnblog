**答案及解析：**

```python
def clean_text(text):
    valid_chars = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789"
    cleaned_text = ""
    for char in text:
        if char in valid_chars:
            cleaned_text += char
    return cleaned_text

# 测试代码
text = "这是一个测试文本！123"
print(clean_text(text))  # 输出：thisisatesttext123
```

```python
    # 请在此编写代码

```

**答案及解析：**

```python
def search_keyword(text, keyword):
    result = []
    index = text.find(keyword)
    while index != -1:
        result.append(index)
        index = text.find(keyword, index + 1)
    return result

# 测试代码
text = "这是一段测试文本，用于测试搜索功能。"
keyword = "测试"
print(search_keyword(text, keyword))  # 输出：[10, 18]
```

**解析：** 该函数使用 `find` 方法查找关键词在文本中的位置，每次找到关键词的位置后，将位置添加到结果列表 `result` 中，然后继续从下一个位置开始查找，直到找不到为止。最后返回 `result` 作为结果。


**答案及解析：**

```python
from collections import Counter

def keyword_frequency(text):
    words = text.split()
    word_counts = Counter(words)
    sorted_counts = sorted(word_counts.items(), key=lambda x: x[1], reverse=True)
    return dict(sorted_counts)

# 测试代码
text = "这是一段测试文本，用于测试搜索功能。测试功能非常重要。"
print(keyword_frequency(text))  # 输出：{'测试': 3, '一段': 1, '文本': 1, '用于': 1, '搜索': 1, '功能': 2, '非常': 1, '重要': 1}
```

**解析：** 该函数首先将文本按照空格分割成单词列表 `words`，然后使用 `Counter` 类统计每个单词的出现次数。最后，将统计结果按照出现次数从高到低排序，并转换为字典返回。
