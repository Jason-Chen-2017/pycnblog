#### 二、算法编程题库

1. **编写一个函数，用于计算患者的健康评分。**
   - **输入：** 患者的年龄、体重、身高、血压等生理参数。
   - **输出：** 患者的健康评分（0-100分）。
   - **答案：** 

```python
def calculate_health_score(age, weight, height, blood_pressure):
    # 计算体重指数（BMI）
    bmi = weight / (height / 100) ** 2
    
    # 计算血压等级
    if blood_pressure <= 120:
        blood_pressure_level = 1
    elif blood_pressure <= 140:
        blood_pressure_level = 2
    else:
        blood_pressure_level = 3
    
    # 计算健康评分
    health_score = (age * 0.5) + (bmi * 2) + (blood_pressure_level * 3)
    
    # 返回健康评分
    return health_score

# 示例
print(calculate_health_score(30, 70, 175, 110))
```

2. **编写一个函数，用于预测患者的患病风险。**
   - **输入：** 患者的病史、家族病史等数据。
   - **输出：** 患者的患病风险（低、中、高）。
   - **答案：**

```python
def predict_disease_risk(history, family_history):
    # 计算病史权重
    history_weight = len(history)
    
    # 计算家族病史权重
    family_history_weight = len(family_history)
    
    # 计算总权重
    total_weight = history_weight + family_history_weight
    
    # 计算患病风险
    if total_weight <= 2:
        risk = '低'
    elif total_weight <= 4:
        risk = '中'
    else:
        risk = '高'
    
    # 返回患病风险
    return risk

# 示例
print(predict_disease_risk(['高血压'], ['高血压']))
```
