#### 1. 编写一个函数，实现特征提取与融合

**题目描述：** 给定两个特征矩阵 `X1` 和 `X2`，编写一个函数 `extract_and_merge_features`，实现特征提取和融合，返回融合后的特征矩阵。

**答案：**

```python
import numpy as np

def extract_and_merge_features(X1, X2):
    # 特征提取
    feature1 = np.mean(X1, axis=1)
    feature2 = np.mean(X2, axis=1)
    
    # 特征融合
    merged_features = np.hstack((feature1, feature2))
    
    return merged_features
```

**解析：** 该函数首先对两个特征矩阵进行平均操作，提取每个矩阵的特征表示，然后使用水平堆叠（`hstack`）将两个特征矩阵融合为一个。
