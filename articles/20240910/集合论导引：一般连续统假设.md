**答案：** 康托尔集可以通过以下算法生成：

```python
def generate_cantor_set(low, high, depth):
    if depth == 0:
        return [low, high]
    mid = (low + high) / 2
    left = generate_cantor_set(low, mid, depth - 1)
    right = generate_cantor_set(mid, high, depth - 1)
    return left + right

# 生成康托尔集
cantor_set = generate_cantor_set(0, 1, 3)
print(cantor_set)
```
