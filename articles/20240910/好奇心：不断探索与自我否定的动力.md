### 算法编程题

**1. 实现一个函数，判断一个字符串是否为回文。**

**代码示例：**

```python
def is_palindrome(s):
    return s == s[::-1]
```

**解析：** 这个函数通过字符串切片的方式，将输入字符串 `s` 反转并与原字符串比较，如果相同则返回 `True`，表示字符串是回文。

**2. 请编写一个函数，找出一个字符串中的所有回文字符串。**

**代码示例：**

```python
def find_palindromes(s):
    n = len(s)
    palindromes = []
    for i in range(n):
        for j in range(i+1, n+1):
            if is_palindrome(s[i:j]):
                palindromes.append(s[i:j])
    return palindromes
```

**解析：** 这个函数通过嵌套循环，遍历字符串中的所有子串，并使用 `is_palindrome` 函数判断子串是否为回文。如果是，则将其添加到结果列表中。

**3. 请编写一个函数，找出一个字符串中的最长回文字符串。**

**代码示例：**

```python
def longest_palindrome(s):
    n = len(s)
    max_len = 0
    start = 0
    for i in range(n):
        for j in range(i+1, n+1):
            if is_palindrome(s[i:j]) and j-i > max_len:
                max_len = j-i
                start = i
    return s[start:start+max_len]
```

**解析：** 这个函数通过嵌套循环，遍历字符串中的所有子串，并使用 `is_palindrome` 函数判断子串是否为回文。如果子串是回文且长度大于当前最大长度，则更新最大长度和起始索引。最后返回最长回文字符串。

def is_dna_palindrome(dna):
    nucleotides = "ACGTACGT"
    for i in range(len(dna)):
