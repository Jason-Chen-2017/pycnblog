#### 二、算法编程题库及解析

1. **面试题：** 给定一个整数数组 nums，判断是否存在三个元素 a、b、c 使得 a + b + c = 0。请你判断三个数是否存在。

   **答案：** 可以使用哈希表实现。首先遍历数组，对于每个元素 x，判断 -x 是否存在于哈希表中。如果存在，则找到了三个元素使得它们的和为 0。否则，将 x 的值和索引存入哈希表。

   ```python
   def threeSum(nums):
       nums.sort()
       ans = []
       length = len(nums)
       for i in range(length):
           # 遍历每个元素 x
           if i > 0 and nums[i] == nums[i-1]:
               continue
           # 查找 -x
           left, right = i+1, length-1
           while left < right:
               total = nums[i] + nums[left] + nums[right]
               if total == 0:
                   ans.append([nums[i], nums[left], nums[right]])
                   while left < right and nums[left] == nums[left+1]:
                       left += 1
                   while left < right and nums[right] == nums[right-1]:
                       right -= 1
                   left += 1
                   right -= 1
               elif total < 0:
                   left += 1
               else:
                   right -= 1
       return ans
   ```

2. **面试题：** 给定一个字符串 s 和一个字符 t，判断 t 是否是 s 的有效字符。

   **答案：** 可以使用双指针实现。定义两个指针 i 和 j，分别表示当前已比较的字符位置。初始时，i=0，j=1。如果 s[i] == t，则返回 True；否则，如果 s[i] != t，则移动 j，直到找到 s[i] == t 或 j 到达字符串末尾。如果 j 到达末尾，则返回 False。

   ```python
   def is_valid_char(s, t):
       i = 0
       j = 1
       while i < len(s) and j < len(s):
           if s[i] == t:
               return True
           j = i + 1
       return False
   ```
