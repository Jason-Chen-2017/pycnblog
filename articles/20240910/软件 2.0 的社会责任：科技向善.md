### 算法编程题库

#### 3. 给定一个整数数组，找到其中所有出现次数超过一半的数字。

**题目描述：** 在一个整数数组中，除了一个数字之外，其他数字都出现了两次。请找出这个只出现一次的数字。

**示例：**

```go
输入：[1, 2, 3, 2, 3, 5]
输出：5
```

**解决方案：**

```go
package main

import "fmt"

func findSingle(arr []int) int {
    result := 0
    for _, num := range arr {
        result ^= num
    }
    return result
}

func main() {
    arr := []int{1, 2, 3, 2, 3, 5}
    fmt.Println(findSingle(arr)) // 输出 5
}
```

**解析：** 这个问题可以使用位运算中的异或运算（^）来解决。异或运算具有以下性质：任何数字与 0 异或都等于其本身，任何数字与自身异或都等于 0。因此，将数组中所有数字进行异或运算，最终得到的结果即为只出现一次的数字。

#### 4. 给定一个字符串，找出其中第一个只出现一次的字符。

**题目描述：** 给定一个字符串，找出其中第一个只出现一次的字符。如果不存在，则返回 -1。

**示例：**

```go
输入："abcabc"
输出：'c'
```

**解决方案：**

```go
package main

import "fmt"

func firstUniqChar(s string) byte {
    counts := [26]int{}
    for i := 0; i < len(s); i++ {
        counts[s[i]-'a']++
    }
    for i := 0; i < len(s); i++ {
        if counts[s[i]-'a'] == 1 {
            return s[i]
        }
    }
    return -1
}

func main() {
    s := "abcabc"
    fmt.Println(firstUniqChar(s)) // 输出 'c'
}
```

**解析：** 这个问题可以使用一个长度为 26 的数组 `counts` 来记录每个字符出现的次数。遍历字符串，更新数组中的计数。然后再次遍历字符串，查找第一个出现次数为 1 的字符。
