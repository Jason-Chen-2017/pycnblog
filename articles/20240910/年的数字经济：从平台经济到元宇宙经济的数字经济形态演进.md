##### 1. 编写一个算法，找出两个字符串的最近公共祖先。

**题目：** 给定两个字符串 s1 和 s2，编写一个算法，找出它们的最长公共前缀。如果不存在公共前缀，返回空字符串。

```python
def longest_common_prefix(s1, s2):
    # 实现最长公共前缀算法
    pass
```

**答案：**

```python
def longest_common_prefix(s1, s2):
    # 找出两个字符串的最小长度
    min_len = min(len(s1), len(s2))

    # 遍历两个字符串的前 min_len 个字符
    for i in range(min_len):
        if s1[i] != s2[i]:
            return s1[:i]

    return s1[:min_len]
```

##### 2. 编写一个算法，实现快速排序。

**题目：** 给定一个整数数组 nums，实现一个快速排序算法。

```python
def quick_sort(nums):
    # 实现快速排序算法
    pass
```

**答案：**

```python
def quick_sort(nums):
    if len(nums) <= 1:
        return nums

    pivot = nums[len(nums) // 2]
    left = []
    middle = []
    right = []

    for num in nums:
        if num < pivot:
            left.append(num)
        elif num == pivot:
            middle.append(num)
        else:
            right.append(num)

    return quick_sort(left) + middle + quick_sort(right)
```
