#### **二、算法编程题库**

##### **1. 最长公共子序列（LCS）**

**题目描述：** 给定两个字符串，求它们的最长公共子序列。

**输入：** `s1 = "AGGTAB"`, `s2 = "GXTXAYB"`

**输出：** `"GTAB"`

**代码示例：**

```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return ''.join([s1[i - 1] for i, j in enumerate(dp[-1]) if dp[i][j] == dp[-1][-1]])

print(longest_common_subsequence("AGGTAB", "GXTXAYB"))
```

**解析：** 使用动态规划求解最长公共子序列，时间复杂度为 O(mn)。

##### **2. 背包问题**

**题目描述：** 给定一个物品数组和一个背包容量，求背包能装下的最大价值。

**输入：** `weights = [2, 3, 4, 5]`, `values = [3, 4, 5, 6]`, `capacity = 8`

**输出：** `9`

**代码示例：**

```python
def knapsack(weights, values, capacity):
    n = len(weights)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for j in range(1, capacity + 1):
            if weights[i - 1] <= j:
                dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weights[i - 1]] + values[i - 1])
            else:
                dp[i][j] = dp[i - 1][j]

    return dp[n][capacity]

print(knapsack([2, 3, 4, 5], [3, 4, 5, 6], 8))
```

**解析：** 使用动态规划求解背包问题，时间复杂度为 O(nC)，其中 C 为背包容量。

##### **3. 求最长递增子序列**

**题目描述：** 给定一个整数数组，求最长递增子序列的长度。

**输入：** `[10, 22, 9, 33, 21, 50, 41, 60]`

**输出：** `5`

**代码示例：**

```python
def length_of_lis(nums):
    dp = [1] * len(nums)
    for i in range(1, len(nums)):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)

    return max(dp)

print(length_of_lis([10, 22, 9, 33, 21, 50, 41, 60]))
```

**解析：** 使用动态规划求解最长递增子序列的长度，时间复杂度为 O(n²)。
