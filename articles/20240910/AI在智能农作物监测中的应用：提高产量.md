### 二、算法编程题库

#### 1. 使用K-means算法对农作物生长数据聚类

**题目：** 请使用K-means算法，对一组农作物生长数据进行聚类，并输出每个聚类中心及其对应的农作物数据。

**答案：** 
```python
import numpy as np
from sklearn.cluster import KMeans

# 假设生长数据存储在数组 data 中
data = np.array([[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]])

# 使用KMeans算法进行聚类，选择 k=2
kmeans = KMeans(n_clusters=2, random_state=0).fit(data)

# 输出聚类结果
print("聚类中心：", kmeans.cluster_centers_)
print("每个数据点的聚类标签：", kmeans.labels_)

# 查看每个聚类中心对应的农作物数据
for i in range(2):
    print(f"聚类 {i+1} 的农作物数据：")
    print(np.array(data)[kmeans.labels_ == i])
```

**解析：** 通过调用`KMeans`类并使用`fit`方法，我们可以对数据进行聚类。`cluster_centers_`属性保存了每个聚类中心的位置，而`labels_`属性保存了每个数据点的聚类标签。

#### 2. 使用决策树算法预测农作物产量

**题目：** 请使用决策树算法，根据农作物生长数据预测产量，并输出决策树的结构。

**答案：**
```python
from sklearn.tree import DecisionTreeRegressor
from sklearn.model_selection import train_test_split
import matplotlib.pyplot as plt

# 假设生长数据存储在数组 X 中，产量存储在数组 y 中
X = [[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]]
y = [1, 2, 1, 4, 4, 0]

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)

# 使用决策树算法进行训练
regressor = DecisionTreeRegressor(random_state=0)
regressor.fit(X_train, y_train)

# 输出决策树结构
from sklearn.tree import plot_tree
plt.figure(figsize=(12, 8))
plot_tree(regressor, filled=True, feature_names=['特征1', '特征2'], class_names=['产量'])
plt.show()

# 使用测试集进行预测
y_pred = regressor.predict(X_test)
print("预测结果：", y_pred)
```

**解析：** 在这个例子中，我们首先划分训练集和测试集，然后使用`DecisionTreeRegressor`进行训练。通过`plot_tree`函数，我们可以可视化决策树的结构。最后，使用训练好的模型对测试集进行预测。
