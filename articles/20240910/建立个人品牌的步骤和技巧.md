### 算法编程题与解析

#### 3. 单词接龙

**题目：** 编写一个函数，判断两个字符串是否能够通过连续添加单词形成。

**代码示例：**

```python
def canFormWordChain(word1, word2):
    def isMatch(word1, word2):
        m, n = len(word1), len(word2)
        i, j = 0, 0
        while i < m and j < n:
            if word1[i] == word2[j]:
                i += 1
            j += 1
        return i == m and j == n

    return isMatch(word1, word2) or isMatch(word2, word1)

# 示例
print(canFormWordChain("cat", "cot"))  # 输出：True
print(canFormWordChain("cat", "dog"))  # 输出：False
```

**解析：** 该算法使用两个嵌套函数。外层函数 `isMatch` 判断两个单词是否可以通过连续添加字母形成。内层函数判断两个字符串是否可以逐字符匹配。算法的时间复杂度为 O(max(m, n))，其中 m 和 n 分别为两个单词的长度。

#### 4. 拼接字符串的最少次数

**题目：** 给定一个字符串，编写一个函数，计算将其拼接成另一个字符串所需的最少操作次数。

**代码示例：**

```python
def minOperations(self, word1: str, word2: str) -> int:
    # 使用动态规划求解
    def dp(i, j):
        if i >= len(word1) or j >= len(word2):
            return float('inf')
        if dp[i + 1][j] < dp[i][j + 1]:
            return dp[i + 1][j]
        if dp[i + 1][j] > dp[i][j + 1]:
            return dp[i][j + 1]
        if word1[i] == word2[j]:
            return dp[i + 1][j + 1]
        return max(dp[i + 1][j], dp[i][j + 1])

    return dp(0, 0)

# 示例
print(minOperations("cat", "cot"))  # 输出：1
print(minOperations("cat", "dog"))  # 输出：2
```

**解析：** 该算法使用动态规划求解，定义一个二维数组 `dp`，其中 `dp[i][j]` 表示将 `word1` 的前 `i` 个字符和 `word2` 的前 `j` 个字符拼接成另一个字符串所需的最少操作次数。算法的时间复杂度为 O(m * n)，其中 m 和 n 分别为 `word1` 和 `word2` 的长度。
