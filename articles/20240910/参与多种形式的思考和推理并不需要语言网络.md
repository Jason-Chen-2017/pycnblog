### 一、逻辑推理

#### 题目1：鸡兔同笼问题

**题目描述：** 一个笼子里有鸡和兔子，共计35只，共有94只脚。请问笼子里各有多少只鸡和兔子？

**答案解析：** 设鸡有x只，兔子有y只，则有以下两个方程：
\[ x + y = 35 \]
\[ 2x + 4y = 94 \]

通过解方程组，我们可以得到：
\[ x = 23 \]
\[ y = 12 \]

所以，笼子里有23只鸡和12只兔子。

**源代码实例：**
```python
x, y = 0, 0
for i in range(36):
    if 2*i + 4*(35-i) == 94:
        x, y = i, 35-i
        break
print(f'鸡的数量为：{x}，兔子的数量为：{y}')
```

### 二、图形推理

#### 题目2：图形填充

**题目描述：** 下面这个图形中，缺失的图形应该是什么？

```
+---+---+
|   |   |
+---+---+
|   |   |
+---+---+
|   |   |
+---+---+
```

**答案解析：** 根据图形的规律，每一行的中间部分应该是一个正方形。因此，缺失的图形应该是一个正方形。

**源代码实例：**
```python
def fill_missing_graph():
    graph = [
        ['+', '+', '+'],
        ['+', ' ', '+'],
        ['+', '+', '+'],
        ['+', ' ', '+'],
        ['+', '+', '+'],
    ]
    for i, row in enumerate(graph):
        if i % 2 == 1:
            graph[i][1] = ' '
    for row in graph:
        print(''.join(row))
fill_missing_graph()
```

### 三、数学推理

#### 题目3：最大公因数

**题目描述：** 求两个整数36和60的最大公因数。

**答案解析：** 使用辗转相除法（欧几里得算法）可以求得最大公因数。

\[ 60 \mod 36 = 24 \]
\[ 36 \mod 24 = 12 \]
\[ 24 \mod 12 = 0 \]

所以，36和60的最大公因数是12。

**源代码实例：**
```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

print(gcd(36, 60))
```

### 四、综合推理

#### 题目4：程序输出

**题目描述：** 下面这段程序的输出是什么？

```python
def func(a, b):
    if a > b:
        return a
    else:
        return b

x = 5
y = 10
z = func(x, y)
print(z)
```

**答案解析：** 函数`func`比较两个数的大小并返回较大的那个。在这段程序中，`z`的值为10。

**源代码实例：**
```python
def func(a, b):
    if a > b:
        return a
    else:
        return b

x = 5
y = 10
z = func(x, y)
print(z)  # 输出 10
```
