### 1. 无穷序列的计算

**题目：** 给定一个无穷序列 {a1, a2, a3, ...}，其中 an = n * n!。请实现一个函数，计算前 n 项的和。

**答案解析：** 这道题目主要考察对无穷序列的理解和计算能力。可以通过递归或动态规划的方式求解。

**代码实例：**

```go
func factorial(n int) int {
    if n == 0 {
        return 1
    }
    return n * factorial(n-1)
}

func infiniteSeriesSum(n int) int {
    sum := 0
    for i := 1; i <= n; i++ {
        sum += i * factorial(i)
    }
    return sum
}
```
