### 4. 算法编程题库及解析

#### 问题5：最长公共子序列

**算法题：** 给定两个字符串，求它们的最长公共子序列。

**答案解析：**

- **算法思路：** 动态规划，使用二维数组存储中间状态。
- **代码实例：**

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

# 测试
str1 = "ABCD"
str2 = "ACDF"
print(longest_common_subsequence(str1, str2))  # 输出 2
```

#### 问题6：二分查找

**算法题：** 在一个有序数组中，查找一个给定元素，并返回它的索引。

**答案解析：**

- **算法思路：** 二分查找，不断缩小查找范围。
- **代码实例：**

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return -1

# 测试
arr = [1, 2, 3, 4, 5, 6]
target = 4
print(binary_search(arr, target))  # 输出 3
```
