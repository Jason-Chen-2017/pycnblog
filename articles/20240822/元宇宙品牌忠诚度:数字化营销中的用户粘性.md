元宇宙品牌忠诚度的构建过程，可以通过数学模型来表示。记品牌忠诚度评分为 $C$，其数学模型为：

$$
C = f(X, \theta)
$$

其中 $X$ 为用户的虚拟行为数据，$\theta$ 为模型参数。

假设品牌忠诚度评分 $C$ 与用户停留时间 $T$、点击次数 $C_{click}$、购买频率 $P$ 等特征有关，则可以将模型表示为：

$$
C = \alpha_1 T + \alpha_2 C_{click} + \alpha_3 P + \beta_1 + \beta_2 + \beta_3
$$

其中 $\alpha_1$、$\alpha_2$、$\alpha_3$ 为特征权重，$\beta_1$、$\beta_2$、$\beta_3$ 为截距项。

### 4.2 公式推导过程

为了构建元宇宙品牌忠诚度评分模型，我们需要对上述线性模型进行训练。假设我们有 $N$ 个用户的虚拟行为数据 $(x_i, y_i)$，其中 $x_i$ 为第 $i$ 个用户的虚拟行为特征向量，$y_i$ 为对应的品牌忠诚度评分。

模型的最小化问题可以表示为：

$$
\min_{\alpha, \beta} \sum_{i=1}^{N} (y_i - (\alpha_1 x_{1,i} + \alpha_2 x_{2,i} + \alpha_3 x_{3,i} + \beta_1 + \beta_2 + \beta_3))^2
$$

通过对上式求偏导，可以得到特征权重的求解公式：

$$
\alpha_1 = \frac{\sum_{i=1}^{N} (y_i - \beta_1 - \beta_2 - \beta_3) x_{1,i}}{\sum_{i=1}^{N} x_{1,i}^2}
$$

$$
\alpha_2 = \frac{\sum_{i=1}^{N} (y_i - \beta_1 - \beta_2 - \beta_3) x_{2,i}}{\sum_{i=1}^{N} x_{2,i}^2}
$$

$$
\alpha_3 = \frac{\sum_{i=1}^{N} (y_i - \beta_1 - \beta_2 - \beta_3) x_{3,i}}{\sum_{i=1}^{N} x_{3,i}^2}
$$

通过求解上述公式，可以得到模型的特征权重 $\alpha$ 和截距项 $\beta$。

### 5.2 源代码详细实现

首先，定义元宇宙品牌忠诚度评分的训练函数：

```python
from torch.utils.data import Dataset
import torch
import torch.nn as nn
import torch.optim as optim

class UserBehaviorDataset(Dataset):
    def __init__(self, behaviors, labels):
        self.behaviors = behaviors
        self.labels = labels
        
    def __len__(self):
        return len(self.behaviors)
    
    def __getitem__(self, index):
        features = self.behaviors[index]
        label = self.labels[index]
        return {'features': features, 'label': label}

# 定义模型参数和优化器
model = nn.Linear(3, 1)
optimizer = optim.SGD(model.parameters(), lr=0.01)
criterion = nn.MSELoss()

# 定义训练函数
def train(model, dataset, optimizer, criterion, num_epochs=100, batch_size=32):
    train_loader = torch.utils.data.DataLoader(dataset, batch_size=batch_size, shuffle=True)
    for epoch in range(num_epochs):
        running_loss = 0.0
        for i, data in enumerate(train_loader, 0):
            inputs, labels = data['features'], data['label']
            optimizer.zero_grad()
            outputs = model(inputs)
            loss = criterion(outputs, labels)
            loss.backward()
            optimizer.step()
            running_loss += loss.item()
        print(f'Epoch {epoch+1}, loss: {running_loss/len(train_loader)}')

# 训练模型
behaviors = [[20, 5, 3], [10, 8, 1], [30, 10, 5], [15, 2, 4], [25, 6, 2]]
labels = [7.2, 4.5, 12.1, 6.2, 9.3]
dataset = UserBehaviorDataset(behaviors, labels)
train(model, dataset, optimizer, criterion)
```

接着，定义模型预测函数：

```python
# 定义模型预测函数
def predict(model, dataset, batch_size=32):
    predictions = []
    predict_loader = torch.utils.data.DataLoader(dataset, batch_size=batch_size)
    with torch.no_grad():
        for batch in predict_loader:
            inputs = batch['features']
            outputs = model(inputs)
            predictions.extend(outputs.tolist())
    return predictions

# 预测用户忠诚度
dataset = UserBehaviorDataset(behaviors, labels)
predictions = predict(model, dataset)
print(predictions)
```

以上就是使用PyTorch构建元宇宙品牌忠诚度评分的完整代码实现。可以看到，通过简单的线性回归模型，我们就能够根据用户的虚拟行为数据，预测其品牌忠诚度评分。
