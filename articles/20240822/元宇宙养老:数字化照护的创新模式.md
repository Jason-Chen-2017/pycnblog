### 4.2 公式推导过程

我们以一个简单的线性回归模型为例，推导健康状态预测公式：

设 $\theta = (w_1,w_2,\dots,w_n,b)$ 为模型的参数，则线性回归模型的预测公式为：

$$
y = f(x) = w_1x_1 + w_2x_2 + \dots + w_nx_n + b
$$

其中 $w_i$ 为第 $i$ 个特征的权重，$b$ 为截距。通过最小化均方误差损失函数 $L = \frac{1}{N}\sum_{i=1}^N (y_i - f(x_i))^2$，我们可以求解模型参数 $\theta$，使得预测值与实际值尽可能接近。

以下是一个简单的Python代码示例，用于实现老年人跌倒预测的线性回归模型：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 模拟老年人活动数据
X = np.array([[2, 3, 1], [1, 4, 2], [4, 2, 3], [3, 3, 2]])
y = np.array([0, 0, 1, 1])

# 训练线性回归模型
model = LinearRegression()
model.fit(X, y)

# 预测新样本
new_sample = np.array([[3, 4, 1]])
predicted = model.predict(new_sample)

print(f"预测跌倒概率：{predicted}")
```
