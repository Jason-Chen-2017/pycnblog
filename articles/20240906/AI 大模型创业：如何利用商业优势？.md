**题目 4：** 请编写一个算法，实现对给定文本的语义分析，提取关键词。

**答案解析：**

以下是一个简单的语义分析算法示例，使用词频统计方法提取关键词：

```python
from collections import Counter

def extract_keywords(text):
    # 分词处理
    words = text.split()
    # 统计词频
    word_counts = Counter(words)
    # 提取高频关键词
    keywords = word_counts.most_common(10)
    return keywords

# 示例
text = "AI 大模型在商业应用中具有广泛的前景，可以提升企业的竞争力。"
print(extract_keywords(text))
```
