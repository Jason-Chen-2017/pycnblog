**1. 如何评估创业项目的投资回报率？**

**题目：** 在大模型时代，创业者需要如何评估一个项目的投资回报率（ROI）？

**答案：** 投资回报率是评估投资项目盈利能力的重要指标。可以通过以下公式计算：

\[ ROI = \frac{（收益 - 成本）}{成本} \]

**解析：** 

- **收益：** 项目实施后所产生的净利润。
- **成本：** 投资项目的总投入成本。

**示例代码：**

```python
def calculate_roi(income, cost):
    roi = (income - cost) / cost
    return roi

# 示例数据
income = 100000  # 收益
cost = 50000  # 成本

# 计算ROI
roi = calculate_roi(income, cost)
print("投资回报率（ROI）:", roi)
```
