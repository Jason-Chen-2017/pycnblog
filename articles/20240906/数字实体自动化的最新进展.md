```python
def recommend_items(user, items, similarity_matrix):
    scores = []
    for item in items:
        score = similarity_matrix[user][item]
        scores.append(score)
    recommended_items = [item for item, score in sorted(zip(items, scores), reverse=True)]
    return recommended_items
```

**解析：** 该算法根据用户与商品之间的相似度矩阵，计算每个商品对用户的推荐分数，然后按分数从高到低排序，返回推荐商品列表。
