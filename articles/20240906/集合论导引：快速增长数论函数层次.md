#### 一、典型问题与面试题库

##### 1. 筛选素数（素数是只能被1和它本身整除的自然数）

**题目：** 编写一个程序，使用埃拉托斯特尼筛法（Sieve of Eratosthenes）筛选出小于等于n的所有素数。

**答案：**

```go
package main

import "fmt"

func SieveOfEratosthenes(n int) []int {
    primes := make([]bool, n+1)
    for i := 2; i <= n; i++ {
        primes[i] = true
    }

    for p := 2; p*p <= n; p++ {
        if primes[p] {
            for i := p * p; i <= n; i += p {
                primes[i] = false
            }
        }
    }

    var result []int
    for p := 2; p <= n; p++ {
        if primes[p] {
            result = append(result, p)
        }
    }
    return result
}

func main() {
    n := 100
    primes := SieveOfEratosthenes(n)
    fmt.Println("Prime numbers:", primes)
}
```

**解析：** 该算法通过排除所有非素数来筛选出素数。初始化一个布尔数组，所有位置都设置为真，然后逐个排除非素数，最后返回素数数组。

##### 2. 欧拉函数（欧拉函数φ(n)表示不大于n且与n互质的整数数量）

**题目：** 编写一个函数，计算欧拉函数φ(n)的值。

**答案：**

```go
package main

import "fmt"

// 欧拉函数φ(n)的值计算
func EulerPhi(n int) int {
    result := n
    p := 2
    for p * p <= n {
        if n%p == 0 {
            while (n /= p) != 0 {
                result *= (1 - 1.0/float64(p))
            }
        }
        p++
    }
    return int(result)
}

func main() {
    n := 10
    phi := EulerPhi(n)
    fmt.Println("EulerPhi(", n, "):", phi)
}
```

**解析：** 使用欧拉函数的公式：如果n是一个大于1的自然数，且n的质因数分解为\( n = p_1^{k1} p_2^{k2} \cdots p_s^{ks} \)，则欧拉函数为 \( \phi(n) = n \left(1 - \frac{1}{p_1}\right) \left(1 - \frac{1}{p_2}\right) \cdots \left(1 - \frac{1}{p_s}\right) \)。

##### 3. 素数判定

**题目：** 编写一个函数，判断一个给定的整数是否为素数。

**答案：**

```go
package main

import "math"

func isPrime(n int) bool {
    if n <= 1 {
        return false
    }
    if n <= 3 {
        return true
    }
    if n%2 == 0 || n%3 == 0 {
        return false
    }
    for i := 5; i <= int(math.Sqrt(float64(n))); i += 6 {
        if n%i == 0 || n%(i+2) == 0 {
            return false
        }
    }
    return true
}

func main() {
    n := 29
    if isPrime(n) {
        fmt.Println(n, "是素数")
    } else {
        fmt.Println(n, "不是素数")
    }
}
```

**解析：** 通过检查数能否被小于或等于其平方根的素数整除来判断。

#### 二、算法编程题库

##### 1. 最大素数

**题目：** 编写一个程序，找出一个给定的范围内最大的素数。

**答案：**

```go
package main

import (
    "fmt"
    "math"
)

// 检查一个数是否是素数
func isPrime(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i <= int(math.Sqrt(float64(n))); i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

// 找到范围内最大的素数
func largestPrimeWithinRange(start, end int) int {
    for i := end; i >= start; i-- {
        if isPrime(i) {
            return i
        }
    }
    return -1
}

func main() {
    start, end := 1, 1000
    largestPrime := largestPrimeWithinRange(start, end)
    fmt.Printf("在%d到%d的范围内最大的素数是：%d\n", start, end, largestPrime)
}
```

**解析：** 从指定范围的最右侧开始，依次检查每个数是否是素数，直到找到第一个素数。

##### 2. 素数生成器

**题目：** 编写一个素数生成器，能够根据输入的数生成所有小于该输入的素数。

**答案：**

```go
package main

import (
    "fmt"
)

// 判断一个数是否是素数
func isPrime(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i*i <= n; i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

// 素数生成器
func primeGenerator(upperLimit int) {
    for num := 2; num <= upperLimit; num++ {
        if isPrime(num) {
            fmt.Println(num)
        }
    }
}

func main() {
    upperLimit := 100
    primeGenerator(upperLimit)
}
```

**解析：** 使用一个简单的循环，生成并打印所有小于等于上限的素数。

##### 3. 素数对（卡迈克尔数）

**题目：** 编写一个程序，找出所有小于等于n的卡迈克尔数，即两个素数的乘积。

**答案：**

```go
package main

import (
    "fmt"
    "math"
)

// 检查一个数是否是素数
func isPrime(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i <= int(math.Sqrt(float64(n))); i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

// 检查两个素数p和q的乘积是否是卡迈克尔数
func isCarmichaelNumber(n int) bool {
    if !isPrime(n) {
        return false
    }
    for i := 2; i <= n/2; i++ {
        if isPrime(i) && (n/i) != i && isPrime(n/i) {
            return true
        }
    }
    return false
}

func main() {
    n := 1000
    for i := 2; i <= n; i++ {
        if isCarmichaelNumber(i) {
            fmt.Println(i, "是卡迈克尔数")
        }
    }
}
```

**解析：** 检查每个数是否为素数，然后检查其因子是否也为素数，以确定是否为卡迈克尔数。
