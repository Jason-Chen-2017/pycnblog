### 二、算法编程题库及答案解析

#### 4. 虚拟身份的随机生成

**题目：** 编写一个函数，用于生成一个虚拟身份（包括姓名、性别、年龄等），要求姓名、性别和年龄具有随机性。

**答案：**

```python
import random

def generate_virtual_identity():
    names = ["Alice", "Bob", "Charlie", "Diana"]
    genders = ["男", "女"]
    ages = random.randint(18, 70)
    
    name = random.choice(names)
    gender = random.choice(genders)
    age = ages
    
    return {"name": name, "gender": gender, "age": age}

print(generate_virtual_identity())
```

**解析：** 通过随机选择姓名、性别和年龄，生成一个虚拟身份。这个例子使用 Python 的 `random` 模块来生成随机数。

#### 5. 虚拟资产的交易

**题目：** 编写一个函数，用于处理虚拟资产的交易。要求支持买入和卖出操作，并记录交易历史。

**答案：**

```python
class VirtualAsset:
    def __init__(self, name, price):
        self.name = name
        self.price = price
        self.transaction_history = []

    def buy(self, quantity):
        cost = self.price * quantity
        self.transaction_history.append(('buy', quantity, cost))
        print(f"Bought {quantity} {self.name}s at {self.price} each.")

    def sell(self, quantity):
        if quantity > len(self.transaction_history):
            print("Insufficient assets to sell.")
            return
        sold_quantity = min(quantity, len(self.transaction_history))
        total_cost = self.transaction_history[-sold_quantity:][0][2]
        profit = (self.price - total_cost / sold_quantity) * sold_quantity
        self.transaction_history = self.transaction_history[:-sold_quantity]
        print(f"Sold {sold_quantity} {self.name}s at {self.price} each. Profit: {profit}.")

crypto = VirtualAsset("BTC", 50000)
crypto.buy(2)
crypto.sell(1)
```

**解析：** 通过创建 `VirtualAsset` 类，处理买入和卖出操作，并记录交易历史。这个例子使用 Python 类和对象的概念来模拟虚拟资产交易。
