### 2.2 算法编程题：使用神经网络权重实现分类算法

**题目：** 使用神经网络权重实现一个简单的二分类算法。

**答案：**
```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def classify(input_data, weights):
    output = sigmoid(np.dot(input_data, weights))
    return '1' if output > 0.5 else '0'

# 示例
input_data = np.array([1, 0])
weights = np.array([-2, 1])
print(classify(input_data, weights))
```

**解析：** 在这个例子中，我们使用了一个简单的 Sigmoid 函数来计算输出概率，并根据该概率进行分类决策。
