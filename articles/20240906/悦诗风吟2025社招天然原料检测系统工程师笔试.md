#### 面试题

**1. 请解释什么是RESTful API？并举例说明。**

**答案：** RESTful API是一种基于HTTP协议的应用程序接口设计规范，旨在通过REST设计风格来构建和设计API。它使用HTTP协议的动词（GET、POST、PUT、DELETE等）来表示资源的操作，并使用URI（统一资源标识符）来标识资源的路径。

**举例：** 假设有一个博客系统，可以通过以下URL访问：

- `GET /api/blogs`：获取所有博客
- `POST /api/blogs`：创建一个新的博客
- `GET /api/blogs/{id}`：获取特定ID的博客
- `PUT /api/blogs/{id}`：更新特定ID的博客
- `DELETE /api/blogs/{id}`：删除特定ID的博客

**2. 如何在Golang中实现并发控制？**

**答案：** 在Golang中，可以使用以下方法实现并发控制：

- **互斥锁（Mutex）：** 保证同一时间只有一个goroutine可以访问共享资源。
- **读写锁（RWMutex）：**允许多个goroutine同时读取共享资源，但只允许一个goroutine写入。
- **通道（Channel）：** 用于在goroutine之间传递数据和同步。

**3. 请解释什么是深度优先搜索（DFS）和广度优先搜索（BFS）。并分别给出一个示例。**

**答案：** 深度优先搜索（DFS）和广度优先搜索（BFS）是两种常见的图搜索算法。

- **深度优先搜索（DFS）：** 从起点开始，沿着一条路径一直走到尽头，然后再回溯到上一个节点，继续沿着另一条路径进行搜索。

**示例：** 假设有一个图如下：

```
A -- B -- C
|    |
D -- E
```

从A开始进行DFS，可能得到的搜索顺序为：A -> B -> C -> E -> D

- **广度优先搜索（BFS）：** 从起点开始，按照访问顺序依次访问相邻的节点。

**示例：** 从A开始进行BFS，可能得到的搜索顺序为：A -> B -> D -> E -> C

#### 算法编程题

**1. 请实现一个函数，判断一个字符串是否是回文。**

**输入：** 一个字符串

**输出：** 是否是回文（True/False）

**示例：**

```
Input: "level"
Output: True

Input: "hello"
Output: False
```

**答案：** 

```python
def is_palindrome(s: str) -> bool:
    return s == s[::-1]

# 示例
print(is_palindrome("level"))  # 输出：True
print(is_palindrome("hello"))  # 输出：False
```

**2. 请实现一个函数，计算两个整数之和，不使用加法运算符。**

**输入：** 两个整数a和b

**输出：** 两个整数之和

**示例：**

```
Input: a = 1, b = 2
Output: 3

Input: a = -2, b = 3
Output: 1
```

**答案：** 

```python
def add(a: int, b: int) -> int:
    while b != 0:
        carry = a & b
        a = a ^ b
        b = carry << 1
    return a

# 示例
print(add(1, 2))  # 输出：3
print(add(-2, 3))  # 输出：1
```
