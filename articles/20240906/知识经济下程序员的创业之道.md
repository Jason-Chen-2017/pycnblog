### 编程题库与答案解析

#### 4. 数据结构与算法

**题目：** 实现一个函数，对链表进行逆序。

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverseList(head: ListNode) -> ListNode:
    pass
```

**答案：**

```python
def reverseList(head: ListNode) -> ListNode:
    prev = None
    curr = head
    while curr:
        next_node = curr.next
        curr.next = prev
        prev = curr
        curr = next_node
    return prev
```

**解析：** 通过迭代方式对链表进行逆序，每次迭代将当前节点指向前一个节点，实现链表逆序。

#### 5. 并发编程

**题目：** 使用协程和通道实现一个生产者-消费者模型。

```python
import asyncio

async def produce(queue: asyncio.Queue):
    pass

async def consume(queue: asyncio.Queue):
    pass
```

**答案：**

```python
async def produce(queue: asyncio.Queue):
    for i in range(5):
        await queue.put(i)
        print(f"Produced {i}")

async def consume(queue: asyncio.Queue):
    while True:
        item = await queue.get()
        print(f"Consumed {item}")
        await queue.task_done()

async def main():
    queue = asyncio.Queue()
    await produce(queue)
    await consume(queue)

asyncio.run(main())
```

**解析：** 生产者协程 `produce` 往通道 `queue` 中放入数值，消费者协程 `consume` 从通道中获取数值并打印，实现生产者-消费者模型。
