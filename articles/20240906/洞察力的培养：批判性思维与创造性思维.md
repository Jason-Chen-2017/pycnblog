##### 1. 打印从1到最大的n位数

**题目来源：** 字节跳动面试题

**题目：** 输入一个整数n，按顺序打印出从1到最大的n位数的所有数字。

**满分答案解析：**

```python
class Solution:
    def printNumbers(self, n: int) -> List[str]:
        return [str(i) for i in range(1, 10**n)]
```

**解析：** 通过循环从1遍历到10的n次方，将每个数字转换为字符串并加入结果列表。

##### 2. 等差数列划分

**题目来源：** 阿里巴巴面试题

**题目：** 给定一个整数数组nums和一个整数k，请返回该数组的所有k-diff数对的数量。

**满分答案解析：**

```python
def findPairs(nums, k):
    nums.sort()
    n = len(nums)
    count = 0
    left, right = 0, 1
    while right < n:
        if nums[right] - nums[left] == k:
            count += 1
            left += 1
            right += 1
        elif nums[right] - nums[left] > k:
            left += 1
        else:
            right += 1
    return count
```

**解析：** 对数组进行排序，使用双指针left和right遍历数组。当nums[right] - nums[left] > k时，移动left指针；当nums[right] - nums[left] < k时，移动right指针；当nums[right] - nums[left] == k时，计数并移动left和right指针。
