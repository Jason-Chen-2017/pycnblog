### 3. AI 与创意思维相关的算法编程题

#### 题目：
设计一个算法，使用 AI 技术生成一首符合用户情感需求的诗歌。

**答案：**

```python
import random

# 生成一首诗歌，情感需求为「喜悦」
def generate_poetry(desire):
    emotions = {
        '喜悦': ['欢快', '美好', '幸福', '欢笑'],
        '悲伤': ['忧伤', '哀愁', '失落', '哭泣'],
        '愤怒': ['暴躁', '恼怒', '气愤', '狂躁']
    }

    sentiment = emotions[desire]
    words = random.sample(sentiment, k=5)
    poetry = f"{words[0]}的{words[1]}，{words[2]}的{words[3]}，{words[4]}的{words[0]}。"

    return poetry

# 测试
print(generate_poetry('喜悦'))
```

**解析：** 该算法使用随机生成情感词汇，组成一首简短的诗歌。通过调整情感词汇的列表，可以生成不同情感需求的诗歌。实际应用中，可以结合自然语言处理技术，根据用户输入的情感描述，生成更加个性化的诗歌。
