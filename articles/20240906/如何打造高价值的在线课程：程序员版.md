## 面试题与解析

### 1. 算法复杂度分析

**题目：** 请解释时间复杂度和空间复杂度，并分别给出一个计算时间和空间的例子。

**答案：** 

时间复杂度是指算法在处理规模为 n 的输入数据时，所需计算时间的增长速度。空间复杂度是指算法在处理规模为 n 的输入数据时，所需存储空间的增长速度。

**时间复杂度例子：** 计算一个数组中元素的和。

```python
def sum_array(arr):
    result = 0
    for i in range(len(arr)):
        result += arr[i]
    return result
```

该函数的时间复杂度为 O(n)，因为计算时间和数组长度成正比。

**空间复杂度例子：** 使用递归计算斐波那契数列的第 n 项。

```python
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```

该函数的空间复杂度为 O(n)，因为递归调用栈的深度与 n 成正比。

### 2. 数据结构与算法

**题目：** 请解释以下数据结构及其常用算法：链表、栈、队列、树、图。

**答案：**

**链表：** 链表是一种线性数据结构，每个节点包含数据和一个指向下一个节点的指针。

- **插入和删除操作：** 平均时间复杂度为 O(1)。
- **查找操作：** 时间复杂度为 O(n)。

**栈：** 栈是一种后进先出（LIFO）的数据结构。

- **入栈和出栈操作：** 平均时间复杂度为 O(1)。
- **查找操作：** 时间复杂度为 O(n)。

**队列：** 队列是一种先进先出（FIFO）的数据结构。

- **入队和出队操作：** 平均时间复杂度为 O(1)。
- **查找操作：** 时间复杂度为 O(n)。

**树：** 树是一种层次结构的数据结构，包含一个根节点和若干个子节点。

- **插入和删除操作：** 平均时间复杂度为 O(log n)。
- **查找操作：** 时间复杂度为 O(log n)。

**图：** 图是一种复杂的数据结构，由节点和边组成。

- **插入和删除操作：** 平均时间复杂度为 O(n)。
- **查找操作：** 时间复杂度为 O(n)。

### 3. 编码实践

**题目：** 实现一个简单的排序算法，如冒泡排序或选择排序。

**答案：**

冒泡排序：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 4. 设计模式

**题目：** 请解释以下设计模式：单例模式、工厂模式、观察者模式。

**答案：**

**单例模式：** 确保一个类只有一个实例，并提供一个访问它的全局访问点。

**工厂模式：** 在创建对象时，不是直接使用 new 运算符，而是将其封装在方法中返回。

**观察者模式：** 定义对象间的一种一对多的依赖关系，使得每当一个对象状态发生改变时，其所有依赖者都会得到通知并自动更新。

## 算法编程题

### 1. 两数之和

**题目：** 给定一个整数数组 `nums` 和一个目标值 `target`，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**答案：**

```python
def two_sum(nums, target):
    hashmap = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hashmap:
            return [hashmap[complement], i]
        hashmap[num] = i
    return []
```

### 2. 寻找旋转排序数组中的最小值

**题目：** 假设按照升序排序的数组在预先未知的某个点上进行了旋转。

**答案：**

```python
def find_min(nums):
    left, right = 0, len(nums) - 1
    while left < right:
        mid = (left + right) // 2
        if nums[mid] > nums[right]:
            left = mid + 1
        else:
            right = mid
    return nums[left]
```

### 3. 合并两个有序链表

**题目：** 将两个升序链表合并为一个新的升序链表并返回。链表节点包含 `val` 和 `next` 指针。

**答案：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_two_lists(l1, l2):
    dummy = ListNode(0)
    current = dummy
    while l1 and l2:
        if l1.val < l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next
    current.next = l1 or l2
    return dummy.next
```
