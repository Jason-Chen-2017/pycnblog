#### 算法编程题库

##### 1. 码头货物分类问题

**题目：** 给定一个码头货物列表，其中包含货物的名称和重量，请设计一个算法，将货物按照重量分为大型、中型和小型三类。

**答案：**

```python
def classify_goods(goods_list):
    large = []
    medium = []
    small = []
    
    for item in goods_list:
        if item['weight'] > 1000:
            large.append(item)
        elif item['weight'] > 500:
            medium.append(item)
        else:
            small.append(item)
    
    return large, medium, small

# 示例
goods_list = [
    {'name': '货物1', 'weight': 1500},
    {'name': '货物2', 'weight': 800},
    {'name': '货物3', 'weight': 300},
    {'name': '货物4', 'weight': 200}
]

print(classify_goods(goods_list))
```

**解析：** 通过遍历货物列表，根据货物的重量进行分类，实现简单且高效。

##### 2. 码头堆叠问题

**题目：** 给定一个二维平面上的码头，其中有一些已经堆叠好的货物。请设计一个算法，计算可以在该码头上堆叠的最大货物数量。

**答案：**

```python
def max_stacked_goods(grid):
    max_goods = 0
    for i in range(len(grid)):
        for j in range(len(grid[0])):
            if grid[i][j] == 1:
                max_goods += 1
    return max_goods

# 示例
grid = [
    [1, 1, 0],
    [1, 1, 1],
    [0, 1, 0]
]

print(max_stacked_goods(grid))
```

**解析：** 通过遍历二维平面，计算已堆叠的货物数量，实现简单且高效。
