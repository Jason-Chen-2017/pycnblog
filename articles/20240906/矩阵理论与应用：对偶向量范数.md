### 一、对偶向量范数的基本概念

**定义：** 设\(A\)是一个\(m \times n\)的矩阵，\(x\)是一个\(n\)维向量，\(y\)是一个\(m\)维向量。则\(y\)称为\(x\)的**对偶向量**，满足以下关系：

\[
\langle x, y \rangle = x^T y = \sum_{i=1}^{n} x_i y_i
\]

其中，\(\langle \cdot, \cdot \rangle\)表示内积。

**对偶向量范数：** 对偶向量范数是一种度量向量“大小”的方式，定义为：

\[
\|x\|_* = \sup_{\|y\|_* \leq 1} \langle x, y \rangle
\]

其中，\(\|x\|_*\)表示\(x\)的对偶范数，\(\|y\|_*\)表示\(y\)的对偶范数。

**性质：**

1. 正定性：对于任意向量\(x\)，有\(\|x\|_* \geq 0\)，且\(\|x\|_* = 0\)当且仅当\(x = 0\)。
2. 同一性：对于任意向量\(x\)，有\(\|x\|\|y\|_* \geq \langle x, y \rangle \geq -\|x\|\|y\|_*\)。
3. 子空间范数：对于任意子空间\(S\)，有\(\|x\|_* = \min_{y \in S, \|y\|_* = 1} \langle x, y \rangle\)。

### 二、对偶向量范数的实际应用

```python
def dual_norm(x):
```

#### 2. 最小二乘解

**题目：** 给定一个\(m \times n\)矩阵\(A\)和一个\(m\)维向量\(b\)，求线性方程组\(Ax = b\)的最小二乘解。

**答案：**

```python
import numpy as np

def least_squares(A, b):
    return np.linalg.lstsq(A, b, rcond=None)[0]
```

```python
import numpy as np

def svd(A):
    return np.linalg.svd(A, full_matrices=True)
```

#### 4. 信号处理

**题目：** 给定一个信号序列\(x\)，使用对偶向量范数进行去噪。

**答案：**

```python
import numpy as np

def denoise(x, threshold):
    y = np.sign(x) * np.maximum(np.abs(x) - threshold, 0)
    return y
```
