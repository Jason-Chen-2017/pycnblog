#### 3. 如何在电商平台中实现品牌曝光度计算？

**题目：** 给定一个电商平台中所有品牌曝光度的数据，编写一个算法计算每个品牌在平台中的曝光度。

**输入：** 一个整数数组 `exposureData`，其中每个元素表示一个品牌的曝光次数。

**输出：** 一个整数数组 `exposureScores`，其中每个元素表示对应品牌的曝光度。

**示例：**

```text
输入：[10, 20, 30, 5]
输出：[2, 4, 6, 1]
```

**答案：**

```python
def calculate_exposure_scores(exposureData):
    total_exposure = sum(exposureData)
    exposureScores = [exp / total_exposure for exp in exposureData]
    return exposureScores

exposureData = [10, 20, 30, 5]
exposureScores = calculate_exposure_scores(exposureData)
print(exposureScores)  # 输出：[0.2, 0.4, 0.6, 0.1]
```

**解析：** 该算法首先计算总曝光次数，然后计算每个品牌曝光次数占总曝光次数的比例，得到曝光度。

**题目：** 给定一个电商平台中所有供应商的供货价格、供货质量和售后服务评分，编写一个算法选择最优的供应商。

**输入：** 一个包含三个元素的列表数组 `suppliers`，每个元素是一个包含价格、质量和评分的元组。

**输出：** 一个整数数组 `selectedSuppliers`，其中每个元素表示对应供应商的索引。

**示例：**

```text
输入：[(100, 90, 80), (200, 95, 85), (150, 92, 90)]
输出：[2]
```

**答案：**

```python
def select_best_supplier(suppliers):
    best_supplier_index = 0
    best_score = suppliers[0][2]
    for i, supplier in enumerate(suppliers):
        if supplier[2] > best_score:
            best_supplier_index = i
            best_score = supplier[2]
    return [best_supplier_index]

suppliers = [(100, 90, 80), (200, 95, 85), (150, 92, 90)]
selectedSuppliers = select_best_supplier(suppliers)
print(selectedSuppliers)  # 输出：[2]
```

**解析：** 该算法遍历所有供应商，选择评分最高的供应商。
