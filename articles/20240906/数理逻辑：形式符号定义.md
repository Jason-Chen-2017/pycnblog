##### 1. 逻辑电路设计

**题目：** 设计一个 Golang 程序，实现一个逻辑电路，根据输入的布尔值计算输出。

**答案：** 可以使用函数和结构体来实现这个逻辑电路。

```go
type LogicCircuit struct {
    a, b bool
}

func (lc *LogicCircuit) andGate() bool {
    return lc.a && lc.b
}

func (lc *LogicCircuit) orGate() bool {
    return lc.a || lc.b
}

func (lc *LogicCircuit) notGate() bool {
    return !lc.a
}
```

func prove(p, q bool) bool {
    if p {
        return true
    }
    return q && prove(p, false)
}
