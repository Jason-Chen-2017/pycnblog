   ```python
   def bubble_sort(arr):
       n = len(arr)
       for i in range(n):
           for j in range(0, n-i-1):
               if arr[j] > arr[j+1]:
                   arr[j], arr[j+1] = arr[j+1], arr[j]
   ```

2. **题目：** 如何在 Golang 中实现一个并发安全的全局变量？

   ```go
   var mu sync.Mutex
   var globalVariable int

   func setGlobalVariable(value int) {
       mu.Lock()
       defer mu.Unlock()
       globalVariable = value
   }

   func getGlobalVariable() int {
       mu.Lock()
       defer mu.Unlock()
       return globalVariable
   }
   ```

3. **题目：** 如何在 Python 中实现一个二分查找算法？
   **答案：** 二分查找是一种高效的查找算法。以下是 Python 实现二分查找的示例代码：
   ```python
   def binary_search(arr, target):
       low = 0
       high = len(arr) - 1
       while low <= high:
           mid = (low + high) // 2
           if arr[mid] == target:
               return mid
           elif arr[mid] < target:
               low = mid + 1
           else:
               high = mid - 1
       return -1
   ```
