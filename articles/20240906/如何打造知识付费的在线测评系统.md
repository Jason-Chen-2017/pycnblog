#### 5. 实现一个简单的自动评分系统。

**题目：** 实现一个自动评分系统，对选择题、填空题和判断题进行评分。

**答案：**
- **输入处理：** 读取考生的答案，如选择题的选项、填空题的答案、判断题的判断结果。
- **答案匹配：** 将考生答案与标准答案进行匹配，判断答案的正确性。
- **评分计算：** 根据题目的分值和答案的正确性计算总分。

**代码实例：**

```python
def grade_answers(questions, answers):
    score = 0
    for i, question in enumerate(questions):
        correct_answer = question['answer']
        user_answer = answers[i]
        if correct_answer == user_answer:
            score += question['score']
    return score

def main():
    questions = [
        {'type': '选择题', 'answer': 'A', 'score': 2},
        {'type': '填空题', 'answer': '北京', 'score': 3},
        {'type': '判断题', 'answer': 'True', 'score': 2},
    ]
    answers = ['A', '北京', 'True']
    score = grade_answers(questions, answers)
    print(f"您的得分是：{score}/9")

if __name__ == '__main__':
    main()
```

**解析：** 通过输入处理、答案匹配和评分计算，可以实现对考生答案的自动评分。
