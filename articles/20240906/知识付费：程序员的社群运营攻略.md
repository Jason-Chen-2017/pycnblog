```python
# Python 示例代码
class MemberManager:
    def __init__(self):
        self.members = {}

    def add_member(self, member):
        self.members[member] = True

    def delete_member(self, member):
        if member in self.members:
            del self.members[member]

    def query_member(self, member):
        return member in self.members

    def update_member(self, member, new_info):
        if member in self.members:
            self.members[member] = new_info
```

2. **编程题：设计一个社群活跃度监控系统**

**题目描述：** 设计一个社群活跃度监控系统，统计每个成员的活跃度，并按活跃度排序。

**答案解析：** 使用字典记录每个成员的活跃度，定期统计并排序。

```python
# Python 示例代码
class ActivityMonitor:
    def __init__(self):
        self.activities = {}

    def record_activity(self, member, score):
        if member in self.activities:
            self.activities[member] += score
        else:
            self.activities[member] = score

    def get_top_members(self):
        return sorted(self.activities.items(), key=lambda x: x[1], reverse=True)
```
