#### 算法编程题与解析：

#### 6. 数据去重

**题目：** 编写一个算法，实现从给定的列表中去除重复元素。

**答案：**

```python
def remove_duplicates(nums):
    return list(set(nums))

nums = [1, 2, 2, 3, 4, 4, 5]
print(remove_duplicates(nums))  # 输出：[1, 2, 3, 4, 5]
```

**解析：** 通过将列表转换为集合，可以去除重复元素。然后，将集合再次转换为列表，以获得去重后的结果。

#### 7. 数据排序

**题目：** 编写一个算法，实现快速排序。

**答案：**

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [3, 6, 8, 10, 1, 2, 1]
print(quick_sort(arr))  # 输出：[1, 1, 2, 3, 6, 8, 10]
```

**解析：** 快速排序是一种高效的排序算法。通过选择一个基准元素（pivot），将数组划分为三个部分：小于pivot的元素、等于pivot的元素和大于pivot的元素，然后递归地对小于和大于pivot的部分进行排序。

#### 8. 数据聚合

**题目：** 编写一个算法，实现将多个数据源的数据聚合到一个数据结构中。

**答案：**

```python
def aggregate_data(datasources):
    aggregated_data = {}
    for datasource in datasources:
        for key, value in datasource.items():
            aggregated_data[key] = aggregated_data.get(key, []) + [value]
    return aggregated_data

datasources = [
    {"name": "Alice", "age": 25},
    {"name": "Bob", "age": 30},
    {"name": "Charlie", "age": 35}
]

print(aggregate_data(datasources))  # 输出：{'name': ['Alice', 'Bob', 'Charlie'], 'age': [25, 30, 35]}
```

**解析：** 通过遍历多个数据源，将相同键的值聚合到一个字典中。使用字典的 `get` 方法获取键对应的值，并将其添加到一个列表中。
