#### 典型面试题与算法编程题

##### 1. 数据结构基础知识

**题目：** 请解释堆（Heap）和哈希表（Hash Table）的区别及各自的应用场景。

**答案：** 

- **堆（Heap）：** 堆是一种特殊的树形数据结构，具有最高效的插入和删除操作。堆通常用于实现优先队列，可以在常数时间内找到最大（或最小）元素。堆的应用场景包括：任务调度、图算法中的最短路径算法（如 Dijkstra 算法）、优先级队列等。

- **哈希表（Hash Table）：** 哈希表是一种通过哈希函数将关键字映射到数组索引的数据结构，支持快速的插入、删除和查找操作。哈希表的应用场景包括：实现字典、缓存、数据库索引等。

##### 2. 算法与数据结构面试题

**题目：** 请实现一个快速排序算法，并分析其时间复杂度和空间复杂度。

**答案：** 

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

# 时间复杂度：平均情况 O(n log n)，最坏情况 O(n^2)
# 空间复杂度：O(log n)
```

##### 3. 算法编程题

**题目：** 请实现一个函数，计算一个字符串中两个单词的最短距离。

**答案：**

```python
def shortest_distance(s, word1, word2):
    index1, index2 = -1, -1
    for i in range(len(s)):
        if s[i:i+len(word1)] == word1:
            index1 = i
        if s[i:i+len(word2)] == word2:
            index2 = i
        if index1 != -1 and index2 != -1:
            distance = abs(index1 - index2)
            return distance
    return -1

# 示例：
s = "wordgoodgoodgoodbestword"
word1 = "word"
word2 = "good"
print(shortest_distance(s, word1, word2))  # 输出 1
```
