## 算法编程题库

### 1. 编写一个函数，实现思维导图的基本功能

**题目：** 编写一个函数，用于生成简单的思维导图。思维导图包含节点和边，其中节点表示概念，边表示概念之间的关系。

**答案：** 

```python
class Node:
    def __init__(self, name):
        self.name = name
        self.children = []

def create_mindmap(root_name, parent=None):
    node = Node(root_name)
    if parent:
        parent.children.append(node)
    return node

def print_mindmap(node, level=0):
    print(" " * level * 2 + "- " + node.name)
    for child in node.children:
        print_mindmap(child, level + 1)

# 示例
root = create_mindmap("Python")
child1 = create_mindmap("Data Structure", root)
child2 = create_mindmap("Algorithm", root)
create_mindmap("List", child1)
create_mindmap("Set", child1)
create_mindmap("Tree", child2)
create_mindmap("Graph", child2)

print_mindmap(root)
```

### 2. 实现一个概念地图的算法

**题目：** 实现一个算法，用于生成概念地图。概念地图包含概念及其之间的连接关系。

**答案：** 

```python
class Concept:
    def __init__(self, name):
        self.name = name
        self.connected_to = []

    def add_connection(self, concept):
        self.connected_to.append(concept)

def generate_concept_map(concepts):
    concept_map = {}
    for concept in concepts:
        concept_map[concept.name] = concept
    return concept_map

def print_concept_map(concept_map):
    for concept in concept_map.values():
        print(concept.name)
        for connected_concept in concept.connected_to:
            print("  -> " + connected_concept.name)

# 示例
concept1 = Concept("Python")
concept2 = Concept("Data Structure")
concept3 = Concept("Algorithm")
concept1.add_connection(concept2)
concept2.add_connection(concept3)

concept_map = generate_concept_map([concept1, concept2, concept3])
print_concept_map(concept_map)
```
