**算法编程题：**

**题目：** 请实现一个函数，计算两个正整数的最大公约数（GCD）。

**答案：**

```go
package main

import (
    "fmt"
)

func gcd(a, b int) int {
    for b != 0 {
        a, b = b, a%b
    }
    return a
}

func main() {
    x := 48
    y := 18
    fmt.Printf("The GCD of %d and %d is %d.\n", x, y, gcd(x, y))
}
```

**解析：** 这个函数使用辗转相除法（欧几里得算法）来计算两个正整数的最大公约数。算法的基本思想是：用较大数除以较小数，然后用余数替换较大数，继续进行相除，直到余数为 0，此时较小数即为最大公约数。
