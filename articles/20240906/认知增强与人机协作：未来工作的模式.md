##### 1. 编写一个算法，实现对文本数据进行情感分析。

**题目描述：** 编写一个函数 `analyzeSentiment(text string) (string, error)`，接收一个文本字符串作为输入，返回该文本的情感分析结果（正面、负面或中性）以及可能出现的错误。

**答案：** 
```go
package sentiment

import (
    "errors"
    "strings"
)

func analyzeSentiment(text string) (string, error) {
    if text == "" {
        return "", errors.New("input text is empty")
    }

    positiveWords := []string{"好", "喜欢", "满意"}
    negativeWords := []string{"坏", "讨厌", "失望"}

    text = strings.ToLower(text)
    words := strings.Fields(text)

    posCount, negCount := 0, 0
    for _, word := range words {
        for _, posWord := range positiveWords {
            if word == posWord {
                posCount++
                break
            }
        }
        for _, negWord := range negativeWords {
            if word == negWord {
                negCount++
                break
            }
        }
    }

    if posCount > negCount {
        return "正面", nil
    } else if negCount > posCount {
        return "负面", nil
    } else {
        return "中性", nil
    }
}
```

**解析：** 这个算法使用了一个简单的词表来进行情感分析。通过对输入文本进行分词，并统计词表中正面词和负面词的出现次数，从而判断文本的情感倾向。
