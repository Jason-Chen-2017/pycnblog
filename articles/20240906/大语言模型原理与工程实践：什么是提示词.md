#### 算法编程题解析

**3. 编写一个算法，实现根据提示词生成文本摘要。**

**输入：** 一段文本和提示词。

**输出：** 文本的摘要，摘要长度不超过一定限制。

**代码实例：**

```python
import nltk
from nltk.tokenize import sent_tokenize, word_tokenize

def generate_summary(text, prompt, max_sentences=5):
    sentences = sent_tokenize(text)
    words = word_tokenize(text)
    
    prompt_words = set(word_tokenize(prompt))
    relevant_sentences = []
    
    for sentence in sentences:
        words_in_sentence = set(word_tokenize(sentence))
        if len(prompt_words.intersection(words_in_sentence)) > 0:
            relevant_sentences.append(sentence)
            
    if len(relevant_sentences) >= max_sentences:
        summary = " ".join(relevant_sentences[:max_sentences])
    else:
        summary = " ".join(relevant_sentences)
        
    return summary

text = "Your text here."
prompt = "Your prompt here."
print(generate_summary(text, prompt))
```

**解析：** 该代码实现了一个简单的文本摘要算法，通过分析输入文本和提示词，提取出与提示词相关的句子，并生成摘要。
