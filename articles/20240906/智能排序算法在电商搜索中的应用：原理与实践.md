#### 4. 实现基于评分的Top-k算法

**题目：** 请实现一个基于商品评分的Top-k算法，从一组商品中返回评分最高的k个商品。

**答案：** 
```python
def top_k_ratings(products, k):
    # 按评分从高到低排序
    sorted_products = sorted(products, key=lambda x: x['rating'], reverse=True)
    # 返回前k个商品
    return sorted_products[:k]

# 测试数据
products = [{'name': '商品1', 'rating': 4.5}, {'name': '商品2', 'rating': 4.8}, {'name': '商品3', 'rating': 4.3}]
k = 2
print(top_k_ratings(products, k))
```

**解析：** 该算法首先对商品进行排序，然后取出评分最高的k个商品，是一种简单但高效的排序算法。
