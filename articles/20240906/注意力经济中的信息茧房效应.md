#### 算法编程题库

##### 1. 计算文本相似度

**题目：** 编写一个函数，计算两个文本的相似度。

**答案：** 可以使用余弦相似度算法计算两个文本的相似度：

```python
from sklearn.metrics.pairwise import cosine_similarity
from sklearn.feature_extraction.text import CountVectorizer

def text_similarity(text1, text2):
    vectorizer = CountVectorizer()
    X = vectorizer.fit_transform([text1, text2])
    return cosine_similarity(X)[0][1]

text1 = "注意力经济中的信息茧房效应"
text2 = "信息茧房效应在社交媒体的影响"
similarity = text_similarity(text1, text2)
print(f"文本相似度：{similarity}")
```

##### 2. 社交网络社区检测

**题目：** 编写一个算法，使用 Louvain 方法检测社交网络中的社区结构。

**答案：** 可以使用 Python 的网络分析库 `networkx` 来实现：

```python
import networkx as nx
import matplotlib.pyplot as plt

def find_communities(G, n_communities):
    communities = nx.algorithms.community.girvan_newman(G)
    return communities

# 示例图
G = nx.Graph()
G.add_edges_from([(1, 2), (1, 3), (2, 4), (3, 4), (4, 5)])

# 检测社区结构
communities = find_communities(G, 2)
print(f"社区：{communities}")

# 绘制图
nx.draw(G, with_labels=True)
plt.show()
```
