#### 算法编程题库

1. **排序算法（冒泡排序）**

   **题目：** 编写一个冒泡排序算法，对给定数组进行排序。

   **答案：**

   ```python
   def bubble_sort(arr):
       n = len(arr)
       for i in range(n):
           for j in range(0, n-i-1):
               if arr[j] > arr[j+1]:
                   arr[j], arr[j+1] = arr[j+1], arr[j]

   # 示例
   arr = [64, 34, 25, 12, 22, 11, 90]
   bubble_sort(arr)
   print("排序后的数组：", arr)
   ```

2. **查找算法（二分查找）**

   **题目：** 编写一个二分查找算法，在有序数组中查找给定元素。

   **答案：**

   ```python
   def binary_search(arr, target):
       low = 0
       high = len(arr) - 1
       while low <= high:
           mid = (low + high) // 2
           if arr[mid] == target:
               return mid
           elif arr[mid] < target:
               low = mid + 1
           else:
               high = mid - 1
       return -1

   # 示例
   arr = [2, 4, 6, 8, 10, 12, 14, 16]
   target = 10
   result = binary_search(arr, target)
   if result != -1:
       print("元素在数组中的索引为：", result)
   else:
       print("元素不在数组中。")
   ```
