##### 题目2：算法编程题——设计一个算法来统计字符串中单词出现的次数

**题目描述：** 编写一个函数，统计给定字符串中各个单词出现的次数。字符串中的单词以空格分隔，且不区分大小写。

**答案：**

```python
def word_count(s):
    # 将字符串转换为小写，并用空格分隔
    words = s.lower().split()
    # 初始化一个字典来存储单词及其出现次数
    count = {}
    # 遍历单词列表
    for word in words:
        # 如果单词已经在字典中，则将其出现次数加一
        if word in count:
            count[word] += 1
        # 如果单词不在字典中，则将其添加到字典中，出现次数为1
        else:
            count[word] = 1
    return count

# 示例
s = "Hello world hello World"
print(word_count(s))  # 输出：{'hello': 2, 'world': 2}
```

**解析：** 

这个算法通过初始化一个字典来存储单词及其出现次数。首先，将输入的字符串转换为小写，并用空格分隔成单词列表。然后遍历这个列表，对于每个单词，如果它在字典中，则将其出现次数加一；如果它不在字典中，则将其添加到字典中，出现次数初始化为1。
