##### 1. 什么是CUI？请简要介绍其特点。

**答案：** CUI是对话式用户界面，是一种通过自然语言交互来满足用户需求的技术。其特点包括：

- **自然性：** 用户可以使用自然语言与系统进行交流，无需遵循特定的语法和命令。
- **智能化：** CUI通过人工智能技术实现自然语言理解、生成和对话管理，能够提供个性化、智能化的服务。
- **交互性：** CUI能够根据用户的需求，主动发起对话，引导用户完成特定任务。

##### 2. CUI与传统的GUI、CLI相比，有哪些优势？

**答案：** CUI相较于传统的GUI（图形用户界面）和CLI（命令行界面），具有以下优势：

- **易用性：** CUI使得用户无需学习复杂的操作步骤，即可快速完成所需任务。
- **高效性：** CUI能够快速响应用户请求，提高用户工作效率。
- **个性化：** CUI可以根据用户的历史行为和偏好，提供定制化的服务。

from textblob import TextBlob

def get_sentiment(input_text):
    analysis = TextBlob(input_text)
    if analysis.sentiment.polarity > 0:
        return "正面"
    elif analysis.sentiment.polarity < 0:
        return "负面"
    else:
        return "中性"

def get_reply(input_text, sentiment):
    if sentiment == "正面":
        return "很高兴听到你的好消息！"
    elif sentiment == "负面":
        return "很抱歉听到你的困扰，有什么我可以帮忙的吗？"
    else:
        return "我不太明白你的意思，能否详细描述一下？"

while True:
    user_input = input("请输入：")
    sentiment = get_sentiment(user_input)
    reply = get_reply(user_input, sentiment)
    print("系统回复：", reply)
