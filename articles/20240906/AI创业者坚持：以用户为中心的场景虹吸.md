#### 二、算法编程题库及答案解析

##### 1. 编写一个Python函数，计算给定列表中所有元素的最大公约数。

**题目：**

```python
def gcd(a, b):
    # 请在此处编写代码
```

**答案：**

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a
```

**解析：** 该函数使用辗转相除法（欧几里得算法）计算两个数的最大公约数。算法思路是不断用较小数去除较大数，然后用余数替换较大数，直到余数为0，此时较大数即为最大公约数。

##### 2. 编写一个Python函数，实现字符串的匹配查找。

**题目：**

```python
def findSubstring(haystack, needle):
    # 请在此处编写代码
```

**答案：**

```python
def findSubstring(haystack, needle):
    return haystack.find(needle)
```

**解析：** 该函数使用Python内置的`find()`方法查找字符串`needle`在字符串`haystack`中的位置。如果找到，返回位置索引；否则，返回-1。

##### 3. 编写一个Python函数，实现冒泡排序。

**题目：**

```python
def bubbleSort(arr):
    # 请在此处编写代码
```

**答案：**

```python
def bubbleSort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

**解析：** 该函数实现冒泡排序算法，通过两重循环，不断比较相邻元素的值，将较大的元素交换到右侧，最终实现数组的有序排列。
