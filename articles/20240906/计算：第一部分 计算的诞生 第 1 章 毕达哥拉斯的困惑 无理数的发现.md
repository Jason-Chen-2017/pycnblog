#### 数学面试题

**题目 1：无理数的定义是什么？**

**答案：** 无理数是指不能表示为两个整数之比的数，即不能表示为形式为 \( \frac{a}{b} \) 的数，其中 \( a \) 和 \( b \) 是整数且 \( b \neq 0 \)。

**解析：** 这个定义是基本的数学知识，了解无理数对于理解数学结构和计算方法至关重要。

**题目 2：如何证明 \( \sqrt{2} \) 是无理数？**

**答案：** 采用反证法，假设 \( \sqrt{2} \) 是有理数，可以表示为两个整数之比 \( \frac{a}{b} \)，然后通过逻辑推理得到一个矛盾，从而证明假设不成立。

**解析：** 这种证明方法展示了数学中的推理技巧，是面试中常见的数学题目。

**题目 3：请解释什么是勾股定理，并给出一个应用实例。**

**答案：** 勾股定理是指直角三角形两条直角边的平方和等于斜边的平方，即 \( a^2 + b^2 = c^2 \)。一个应用实例是在直角三角形中，如果已知两条边的长度，可以计算第三条边的长度。

**解析：** 勾股定理是解决直角三角形问题的关键定理，广泛应用于建筑和工程设计。

#### 算法编程题

**题目 4：实现一个判断一个数是否为无理数的函数。**

**答案：** 可以通过编程实现一个函数，接收一个数作为输入，判断它是否为无理数。以下是一个简单的示例：

```python
import math

def is_irrational(num):
    try:
        math.sqrt(num)
        return False
    except ValueError:
        return True

# 示例
print(is_irrational(2))  # 输出 True
print(is_irrational(4))  # 输出 False
```

**解析：** 这个函数利用 Python 的 `math.sqrt()` 函数，尝试对输入的数进行开方运算。如果引发 `ValueError` 异常，则说明输入的数为无理数。

**题目 5：编写一个计算直角三角形斜边长度的函数。**

**答案：** 可以通过勾股定理编写一个函数，计算直角三角形的斜边长度。以下是一个简单的示例：

```python
import math

def calculate_hypotenuse(a, b):
    return math.sqrt(a**2 + b**2)

# 示例
print(calculate_hypotenuse(3, 4))  # 输出 5.0
```

**解析：** 这个函数直接应用了勾股定理，计算斜边长度。
