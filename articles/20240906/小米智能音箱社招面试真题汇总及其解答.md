```python
def longest_substring(s: str) -> str:
    left, right = 0, 0
    max_len = 0
    max_start = 0
    seen = set()
    
    while right < len(s):
        if s[right] in seen:
            left = max(left + 1, seen[s[right]] + 1)
            seen.clear()
        seen.add(s[right])
        if max_len < right - left + 1:
            max_len = right - left + 1
            max_start = left
        right += 1

    return s[max_start:max_start+max_len]
```

```python
        return arr

def merge(left, right):

    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])

    return result
```

**题目5：最小路径和**

**题目描述：** 给定一个包含非负整数的二维数组，找到和最小的路径。

**答案解析：** 从右下角开始，每一格的值为其上方和左方的最小值加上当前格子自身的值。

**源代码示例：**

```python
def minPathSum(grid):
    m, n = len(grid), len(grid[0])
    for i in range(m-1, -1, -1):
        for j in range(n-1, -1, -1):
            if i < m-1:
                grid[i][j] += min(grid[i+1][j], grid[i+1][j+1])
            if j < n-1:
                grid[i][j] += min(grid[i][j+1], grid[i+1][j+1])
    return grid[0][0]
```

#### 面试题4：合并两个有序链表

**题目描述**：将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

**答案解析**：

这个问题可以通过创建一个新的哑节点作为头节点，然后遍历两个链表，将较小的节点链接到新链表中。这样，最终可以得到一个有序的新链表。

**代码实例**：

```python

        self.next = next

def mergeTwoLists(l1, l2):

    while l1 and l2:
        if l1.val < l2.val:
            curr.next = l1
            l1 = l1.next
        else:
            curr.next = l2
            l2 = l2.next
        curr = curr.next
    curr.next = l1 or l2
    return dummy.next
```
