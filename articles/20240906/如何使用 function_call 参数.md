### 面试题库

#### 1. `function_call` 参数的传递方式

**题目：** 在 C++ 中，函数的参数是如何传递的？

**答案：** 函数参数的传递方式主要有两种：值传递和引用传递。

- **值传递：** 将实参的值复制给形参，形参的改变不会影响实参。
- **引用传递：** 将实参的引用传递给形参，形参的改变会直接影响实参。

**解析：** 在 C++ 中，默认情况下是值传递，但可以通过引用来模拟引用传递。

#### 2. 函数重载与函数覆盖

**题目：** 什么是函数重载？什么是函数覆盖？请分别给出示例。

**答案：**

- **函数重载：** 允许多个同名函数存在，但参数列表必须不同。
- **函数覆盖：** 子类中的函数与父类中的函数同名，参数列表和返回类型必须相同。

**示例：**

```cpp
class Base {
public:
    void show(int n) {
        cout << "Base show " << n << endl;
    }
};

class Derived : public Base {
public:
    void show(int n) {
        cout << "Derived show " << n << endl;
    }
};
```

**解析：** 函数重载是通过参数列表的不同来实现的多态，而函数覆盖是通过继承和同名函数来实现的多态。

#### 3. 函数指针与函数指针数组

**题目：** 什么是函数指针？什么是函数指针数组？请分别给出示例。

**答案：**

- **函数指针：** 指向函数的指针，可以通过函数指针调用函数。
- **函数指针数组：** 一个数组，数组元素都是指向函数的指针。

**示例：**

```cpp
void func() {
    cout << "Function called" << endl;
}

int main() {
    void (*funcPtr)();
    funcPtr = &func;
    (*funcPtr)(); // 调用函数指针
    return 0;
}
```

**解析：** 函数指针和函数指针数组在回调函数和函数指针数组中的应用非常广泛。

### 算法编程题库

#### 4. 最大子序列和

**题目：** 给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个数）。

**示例：**

```cpp
int maxSubArray(vector<int>& nums) {
    int maxSum = INT_MIN, curSum = 0;
    for (int num : nums) {
        curSum = max(num, curSum + num);
        maxSum = max(maxSum, curSum);
    }
    return maxSum;
}
```

**解析：** 该算法使用贪心策略，当当前和小于0时，重置为0，相当于从下一个元素开始新的子序列。

#### 5. 回文数

**题目：** 判断一个整数是否是回文数。

**示例：**

```cpp
bool isPalindrome(int x) {
    if (x < 0 || (x % 10 == 0 && x != 0)) {
        return false;
    }
    int reversed = 0;
    while (x > reversed) {
        reversed = reversed*10 + x%10;
        x /= 10;
    }
    return x == reversed || x == reversed/10;
}
```

**解析：** 该算法通过反转一半的数字并与另一半进行比较，来判断是否是回文数。
