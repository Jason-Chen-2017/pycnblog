##### 1. 编写一个算法，计算一个整数的位数

**题目：** 编写一个函数，输入一个整数 n，返回它的位数。

**答案：** 

```python
def digit_count(n):
    if n < 0:
        n = -n
    count = 0
    while n > 0:
        count += 1
        n //= 10
    return count
```

##### 2. 编写一个算法，实现两个整数的二进制加法

**题目：** 编写一个函数，实现两个整数的二进制加法。

**答案：**

```python
def binary_addition(a, b):
    while b != 0:
        carry = a & b
        a = a ^ b
        b = carry << 1
    return a
```

##### 3. 编写一个算法，实现字符串匹配

**题目：** 编写一个函数，实现字符串匹配算法。

**答案：**

```python
def str_match(s, pattern):
    m, n = len(s), len(pattern)
    dp = [[False] * (n + 1) for _ in range(m + 1)]
    dp[0][0] = True
    for i in range(m + 1):
        for j in range(n + 1):
            if i < m and j < n and (s[i] == pattern[j] or pattern[j] == '?'):
                dp[i][j] = dp[i - 1][j - 1]
            if j < n and pattern[j] == '*':
                dp[i][j] = dp[i][j - 1] or dp[i - 1][j]
    return dp[-1][-1]
```
