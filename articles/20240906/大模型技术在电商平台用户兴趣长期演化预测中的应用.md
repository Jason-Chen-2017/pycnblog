#### 6. 编写一个函数，计算两个时间序列之间的相似度。

**输入：** 两个时间序列 `seq1` 和 `seq2`。

**输出：** 相似度分数。

```python
import numpy as np

def similarity(seq1, seq2):
    # 计算两个时间序列的余弦相似度
    dot_product = np.dot(seq1, seq2)
    norm_seq1 = np.linalg.norm(seq1)
    norm_seq2 = np.linalg.norm(seq2)
    return dot_product / (norm_seq1 * norm_seq2)
```
