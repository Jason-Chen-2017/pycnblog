##### 6. Golang编程题

**题目：** 编写一个Golang程序，使用通道实现一个简单的生产者-消费者模型。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

func producer(ch chan<- int) {
    for i := 0; i < 10; i++ {
        ch <- i
        time.Sleep(1 * time.Second)
    }
    close(ch)
}

func consumer(ch <-chan int) {
    for i := range ch {
        fmt.Println("Received:", i)
        time.Sleep(2 * time.Second)
    }
}

func main() {
    ch := make(chan int, 5)
    go producer(ch)
    consumer(ch)
}
```

**解析：** 这个Golang程序展示了如何使用通道实现生产者-消费者模型，是并发编程的基础。
