#### 1. 预处理任务：图像尺寸调整和归一化

**题目：** 编写一个函数，接收一张图像文件，将其调整为指定尺寸，并对像素值进行归一化处理。

**输入：** 
- `image_path`: 图像文件路径
- `output_size`: 目标尺寸，如 (224, 224)
- `scale`: 归一化比例，如 (0, 1)

**输出：** 
- 调整尺寸并归一化的图像数据

**代码示例：**
```python
import cv2
import numpy as np

def preprocess_image(image_path, output_size, scale):
    # 读取图像
    image = cv2.imread(image_path)
    # 调整尺寸
    image = cv2.resize(image, output_size)
    # 归一化
    image = image.astype(np.float32) / 255.0
    # 标准化
    mean = np.mean(image)
    std = np.std(image)
    image = (image - mean) / std
    return image
```

**解析：** 该函数首先使用 OpenCV 库读取图像，然后使用 `resize` 函数调整尺寸，接着将像素值从 [0, 255] 范围内归一化到 [0, 1]，最后进行标准化处理，以减少方差并提高训练稳定性。

#### 2. 数据增强：随机裁剪和旋转

**题目：** 编写一个函数，对输入图像进行随机裁剪和旋转，以增加训练数据多样性。

**输入：** 
- `image`: 图像数据
- `crop_size`: 裁剪尺寸，如 (224, 224)
- `rot_range`: 旋转角度范围，如 (-30, 30)

**输出：** 
- 裁剪和旋转后的图像数据

**代码示例：**
```python
import cv2
import numpy as np

def augment_image(image, crop_size, rot_range):
    # 随机裁剪
    h, w = image.shape[:2]
    crop_h, crop_w = crop_size
    x = np.random.randint(0, w - crop_w)
    y = np.random.randint(0, h - crop_h)
    image = image[y:y+crop_h, x:x+crop_w]

    # 随机旋转
    angle = np.random.uniform(rot_range[0], rot_range[1])
    M = cv2.getRotationMatrix2D((w/2, h/2), angle, 1)
    image = cv2.warpAffine(image, M, (w, h))
    
    return image
```

**解析：** 该函数首先对图像进行随机裁剪，然后使用 `getRotationMatrix2D` 和 `warpAffine` 函数进行随机旋转。这些操作能够增加训练数据的多样性，提高模型的泛化能力。

**题目：** 使用 TensorFlow 和 Keras 实现一个简单的卷积神经网络，用于面部特征的提取。

**输入：** 
- `images`: 图像数据，形状为 (batch_size, height, width, channels)

**输出：** 
- 面部特征向量，形状为 (batch_size, feature_size)

**代码示例：**
```python
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

def build_cnn(input_shape, feature_size):
    model = Sequential([
        Conv2D(32, (3, 3), activation='relu', input_shape=input_shape),
        MaxPooling2D((2, 2)),
        Conv2D(64, (3, 3), activation='relu'),
        MaxPooling2D((2, 2)),
        Conv2D(128, (3, 3), activation='relu'),
        Flatten(),
        Dense(feature_size, activation='softmax')
    ])
    return model

# 示例输入
input_shape = (224, 224, 3)
feature_size = 128
model = build_cnn(input_shape, feature_size)
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])
```

**解析：** 该函数构建了一个简单的卷积神经网络，包括卷积层、池化层和全连接层。卷积层用于提取图像特征，池化层用于降低维度，全连接层用于分类。这个模型可以用于面部特征的提取，为后续的皮肤质量评估提供基础。
