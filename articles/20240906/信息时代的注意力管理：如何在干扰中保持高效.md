#### 二、算法编程题库及答案解析

**1. 如何在多个线程中共享一个全局变量？**

**题目：** 在多线程环境中，如何安全地共享一个全局变量，以避免数据竞争？

**答案：** 使用互斥锁（Mutex）或读写锁（RWMutex）来保护共享变量的访问。

**代码示例：**

```go
package main

import (
    "fmt"
    "sync"
)

var (
    counter int
    mu      sync.Mutex
)

func increment() {
    mu.Lock()
    counter++
    mu.Unlock()
}

func main() {
    var wg sync.WaitGroup
    for i := 0; i < 1000; i++ {
        wg.Add(1)
        go func() {
            defer wg.Done()
            increment()
        }()
    }
    wg.Wait()
    fmt.Println("Counter:", counter)
}
```

**解析：** 在这个示例中，我们使用互斥锁 `mu.Lock()` 和 `mu.Unlock()` 来确保同一时间只有一个线程可以修改全局变量 `counter`，从而避免数据竞争。

**2. 如何实现一个生产者-消费者模型？**

**题目：** 编写一个 Go 程序，实现生产者-消费者模型，使用通道（channel）来传递数据。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

func producer(ch chan<- int) {
    for i := 0; i < 10; i++ {
        ch <- i
        time.Sleep(1 * time.Second)
    }
    close(ch)
}

func consumer(ch <-chan int) {
    for i := range ch {
        fmt.Println("Consumed:", i)
        time.Sleep(2 * time.Second)
    }
}

func main() {
    ch := make(chan int, 5)
    go producer(ch)
    consumer(ch)
}
```

**解析：** 在这个示例中，我们创建了一个带缓冲的通道 `ch`，缓冲区大小为 5。生产者 `producer` 函数将数据发送到通道中，并在发送完所有数据后关闭通道。消费者 `consumer` 函数从通道中接收数据，并打印出来。
