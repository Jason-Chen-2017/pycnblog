### 算法编程题库与源代码实例

#### 4. 自主武器系统的伦理决策算法

**题目：** 设计一个简单的自主武器系统，使其能够在面对道德困境时做出合理的决策。

**源代码实例：**

```python
class AutonomousWeaponSystem:
    def __init__(self, rules):
        self.rules = rules

    def make_decision(self, situation):
        for rule in self.rules:
            if rule.matches(situation):
                return rule.action
        return "No decision"

class Rule:
    def __init__(self, condition, action):
        self.condition = condition
        self.action = action

    def matches(self, situation):
        return self.condition(situation)

# 示例规则
rules = [
    Rule(lambda s: s['type'] == 'civilian', 'avoid'),
    Rule(lambda s: s['type'] == 'military', 'attack'),
]

# 示例情况
situations = [
    {'type': 'civilian', 'position': (0, 0)},
    {'type': 'military', 'position': (1, 1)},
]

aws = AutonomousWeaponSystem(rules)

for situation in situations:
    decision = aws.make_decision(situation)
    print(f"Decision for {situation}: {decision}")
```

**解析：** 该代码实现了一个简单的自主武器系统，包含两个规则：避免攻击平民和攻击军事目标。通过匹配规则和当前情况，系统将做出相应的决策。
