### 问答示例

**问题 1：** 请简述分布式系统中的 CAP 定理。

**答案：** CAP 定理指出，分布式系统在一致性（Consistency）、可用性（Availability）和分区容错性（Partition tolerance）这三个特性中，最多只能同时保证两项。具体来说，当一个分布式系统发生网络分区时，系统需要在一致性和可用性之间做出选择。

**问题 2：** 请实现一个快速排序算法。

**答案：** 快速排序是一种高效的排序算法，其基本思想是选择一个基准元素，将数组分成两个子数组，一个小于基准元素，另一个大于基准元素，然后递归地排序子数组。以下是快速排序的伪代码实现：

```
// 输入：数组arr、起始索引low、结束索引high
// 输出：排序后的数组

if low < high:
    pi = partition(arr, low, high)
    quicksort(arr, low, pi - 1)
    quicksort(arr, pi + 1, high)

// 分区操作
// 输入：数组arr、起始索引low、结束索引high
// 输出：分区后基准元素的索引

pivot := arr[high]
i := low - 1

for j from low to high - 1:
    if arr[j] < pivot:
        i++
        swap(arr[i], arr[j])

swap(arr[i + 1], arr[high])
return i + 1
```
