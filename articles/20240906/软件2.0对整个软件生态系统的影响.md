#### 算法编程题库

**1. 编写一个算法，根据用户输入的文本，判断其是否为垃圾邮件。**

**题目描述：** 垃圾邮件的判断标准包括：包含特定关键词、含有大量链接、文本格式不规范等。编写一个算法，输入一段文本，判断其是否为垃圾邮件。

**答案解析：** 可以使用字符串匹配算法和模式识别算法来判断文本是否为垃圾邮件。以下是一个简单的示例代码：

```python
def is_spam(text):
    keywords = ["offer", "free", "click here", "buy now"]
    if any(keyword in text for keyword in keywords):
        return True
    if "<a" in text or "</a" in text:
        return True
    if not text.isprintable():
        return True
    return False

text = input("请输入文本：")
if is_spam(text):
    print("这是一封垃圾邮件。")
else:
    print("这不是一封垃圾邮件。")
```

**2. 编写一个算法，实现文本分类。**

**题目描述：** 给定一组文本数据，将其分类为两类：科技类和非科技类。

**答案解析：** 可以使用机器学习算法中的文本分类算法来实现。以下是一个简单的示例代码：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB

# 假设已经有一组文本数据 labels
texts = ["科技新闻", "娱乐新闻", "科技新闻", "体育新闻", "娱乐新闻"]
labels = [0, 1, 0, 2, 1]

# 使用 TF-IDF 转换文本数据
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(texts)

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, labels, test_size=0.2, random_state=42)

# 使用朴素贝叶斯分类器进行分类
classifier = MultinomialNB()
classifier.fit(X_train, y_train)

# 测试分类器
predictions = classifier.predict(X_test)
print(predictions)
```
