**题目 1：给定一组用户对课程的评价数据，编写一个基于内容的推荐算法。**

```python
# Python代码示例
def content_based_recommendation(user_history, courses):
    # 假设用户历史评价为字典格式，courses为列表
    user_history = {'course1': 5, 'course2': 4, 'course3': 5}
    courses = ['course1', 'course2', 'course3', 'course4', 'course5']
    
    # 计算用户对每个课程的兴趣度
    interest_scores = {}
    for course in courses:
        if course in user_history:
            interest_scores[course] = user_history[course]
        else:
            interest_scores[course] = 0
    
    # 对兴趣度进行排序，返回推荐列表
    recommended_courses = sorted(interest_scores.items(), key=lambda x: x[1], reverse=True)
    return [course for course, _ in recommended_courses]
```

**答案解析：** 该代码示例实现了一个简单的基于内容的推荐算法，通过计算用户对每个课程的兴趣度，并按兴趣度排序，返回推荐列表。
