### 算法编程题库与解析
#### 4. 简化复杂句子
**题目：** 编写一个函数，简化复杂句子，去除不必要的冗余信息。

**答案：**
```python
def simplify_sentence(sentence):
    # 去除标点符号
    sentence = sentence.replace('.', '').replace('!', '').replace('?', '')
    # 分割句子为单词
    words = sentence.split()
    # 去除重复的单词
    unique_words = []
    for word in words:
        if word not in unique_words:
            unique_words.append(word)
    # 重新组合句子
    simplified_sentence = ' '.join(unique_words)
    return simplified_sentence

# 示例
sentence = "这是一个复杂的句子，它包含了很多不必要的冗余信息！"
print(simplify_sentence(sentence))
```

**解析：** 该函数通过去除标点符号，分割句子为单词，并去除重复的单词，从而简化复杂句子。简化后的句子去除了冗余信息，更加简洁明了。

#### 5. 信息筛选
**题目：** 编写一个函数，用于筛选出信息中的关键词，并统计关键词出现的次数。

**答案：**
```python
from collections import Counter

def filter_keywords(info, keywords):
    # 分割信息为句子
    sentences = info.split('. ')
    # 初始化关键词计数器
    keyword_counter = Counter()
    # 遍历句子，筛选关键词并计数
    for sentence in sentences:
        words = sentence.split()
        for word in words:
            if word in keywords:
                keyword_counter[word] += 1
    return keyword_counter

# 示例
info = "人工智能是一种模拟人类智能的技术。人工智能可以分为机器学习、深度学习和自然语言处理。机器学习是一种人工智能的方法，它通过训练模型来学习数据。深度学习是机器学习的一种，它通过神经网络来模拟人类大脑。自然语言处理是一种人工智能的方法，它用于处理自然语言。"
keywords = ["人工智能", "机器学习", "深度学习", "自然语言处理"]
print(filter_keywords(info, keywords))
```

**解析：** 该函数首先将信息分割为句子，然后遍历每个句子中的单词，如果单词在关键词列表中，则将其计数。通过`Counter`类可以方便地统计关键词出现的次数。
