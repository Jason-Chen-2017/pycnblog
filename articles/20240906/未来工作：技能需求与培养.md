#### 3. 数据结构与算法

**题目：** 实现一个堆排序算法。

**答案：**

```python
def heapify(arr, n, i):
    largest = i
    left = 2 * i + 1
    right = 2 * i + 2

    if left < n and arr[largest] < arr[left]:
        largest = left

    if right < n and arr[largest] < arr[right]:
        largest = right

    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

def heap_sort(arr):
    n = len(arr)

    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)

    for i in range(n - 1, 0, -1):
        arr[i], arr[0] = arr[0], arr[i]
        heapify(arr, i, 0)

    return arr
```

**解析：** 堆排序算法是一种高效的排序算法，利用堆这种数据结构来实现。该代码首先定义了一个 `heapify` 函数，用于将一个子堆调整成最大堆。`heap_sort` 函数则使用 `heapify` 函数实现堆排序。

#### 4. 设计模式

**题目：** 使用工厂模式实现一个简单的数据库连接。

**答案：**

```python
class DatabaseConnection:
    def connect(self):
        print("连接到数据库")

class MySQLConnection(DatabaseConnection):
    def connect(self):
        print("连接到 MySQL 数据库")

class PostgreSQLConnection(DatabaseConnection):
    def connect(self):
        print("连接到 PostgreSQL 数据库")

class ConnectionFactory:
    @staticmethod
    def create_connection(db_type):
        if db_type == "mysql":
            return MySQLConnection()
        elif db_type == "postgresql":
            return PostgreSQLConnection()
        else:
            raise ValueError("不支持的数据库类型")

# 使用示例
db = ConnectionFactory.create_connection("mysql")
db.connect()
```

**解析：** 该代码示例使用工厂模式来创建数据库连接。`ConnectionFactory` 类提供了一个静态方法 `create_connection`，用于根据传入的数据库类型创建对应的连接类实例。
