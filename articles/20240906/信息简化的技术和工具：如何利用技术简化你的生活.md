### 二、算法编程题库

#### 4. 实现一个文本摘要算法。

**题目：** 编写一个文本摘要算法，将长文本简化为关键句子。

**答案：** 使用基于TF-IDF的文本摘要算法。

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

def text_summary(text, num_sentences=3):
    # 将文本分割为句子
    sentences = text.split('.')
    sentences = [s.strip() for s in sentences if s.strip()]

    # 创建TF-IDF向量模型
    vectorizer = TfidfVectorizer()
    X = vectorizer.fit_transform(sentences)

    # 计算句子相似度
    similarity_matrix = cosine_similarity(X)

    # 构建句子相似度排序
    similarity_scores = []
    for i, sentence in enumerate(sentences):
        scores = similarity_matrix[i]
        max_score = max(scores)
        similarity_scores.append(max_score)

    # 根据相似度排序选取关键句子
    sorted_sentences = sorted(range(len(sentences)), key=lambda i: similarity_scores[i], reverse=True)
    summary_sentences = [sentences[i] for i in sorted_sentences[:num_sentences]]

    # 拼接摘要文本
    summary = '. '.join(summary_sentences)
    return summary

# 示例
text = "这是一个非常长的文本，它包含了大量的信息。我们需要从中提取出最关键的信息，以简化阅读。这里是一个示例文本，用于演示文本摘要算法。"
print(text_summary(text))
```

**解析：** 该算法使用TF-IDF计算句子间的相似度，并根据相似度排序选取最关键的句子。

#### 5. 实现一个信息过滤算法。

**题目：** 编写一个信息过滤算法，去除文本中的无关信息。

**答案：** 使用关键词过滤和正则表达式。

```python
import re

def filter_info(text, keywords=[]):
    # 构建关键词的正则表达式
    keyword_regex = '|'.join(keywords)
    regex = re.compile(f"({keyword_regex})", re.IGNORECASE)

    # 去除关键词
    filtered_text = regex.sub('', text)

    # 去除其他无关信息，如数字、符号等
    filtered_text = re.sub(r"[^a-zA-Z\s]", '', filtered_text)

    return filtered_text.strip()

# 示例
text = "这是一个包含大量信息的文本，其中有一些信息是无关的。例如，这里的数字和符号都是无关信息。"
keywords = ["信息", "文本", "示例"]
print(filter_info(text, keywords))
```

**解析：** 该算法首先根据关键词构建正则表达式，然后使用正则表达式去除文本中的关键词和无关信息。
