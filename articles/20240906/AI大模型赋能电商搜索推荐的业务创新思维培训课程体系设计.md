### 二、算法编程题库

#### 1. 货币兑换的最少货币数

**题目：** 给定一个正整数数组 coins 表示不同面额的硬币，以及一个总金额 amount，计算并返回可以凑成总金额所需的最少硬币个数。如果无法返回解决方案，则返回 -1。

**输入：** coins = [1, 2, 5], amount = 11

**输出：** 3 （11 = 5 + 5 + 1）

**解析：** 使用动态规划求解。定义 dp[i] 为凑成金额 i 所需的最少硬币个数，初始条件为 dp[0] = 0，dp[1] = 1。遍历硬币数组，更新 dp 数组。

**源代码：**

```python
def coinChange(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for i in range(1, amount + 1):
        for coin in coins:
            if i - coin >= 0:
                dp[i] = min(dp[i], dp[i - coin] + 1)
    return dp[amount] if dp[amount] != float('inf') else -1
```

#### 2. 最长公共子序列

**题目：** 给定两个字符串 text1 和 text2，找出他们的最长公共子序列。

**输入：** text1 = "ABCD", text2 = "ACDF"

**输出：** "ACD"

**解析：** 使用动态规划求解。定义 dp[i][j] 为 text1 的前 i 个字符和 text2 的前 j 个字符的最长公共子序列长度。

**源代码：**

```python
def longestCommonSubsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    return ''.join([text1[i - 1] for i, j in enumerate(dp[-1]) if j > 0])
```
