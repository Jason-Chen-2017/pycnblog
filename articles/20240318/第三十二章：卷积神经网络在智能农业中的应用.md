### 1.2 卷积神经网络的基本概念

卷积神经网络（Convolutional Neural Network, CNN）是一种深度学习算法，主要用于处理 grid-like data，如 images and audio signals。CNN 由多个 convolutional layers 组成，每个 convolutional layer 由若干 filters 组成，每个 filter 都具有一个 local receptive field，通过学习到的 weights，将输入图像的局部区域与 filter 做卷积运算，从而得到 feature maps。这些 feature maps 会被 feed 到后续的 convolutional layers 或 fully connected layers 进行进一步的处理。

## 核心概念与联系

### 2.1 卷积神经网络的主要组成部分

卷积神经网络主要包括 convolutional layers, pooling layers, fully connected layers 和 activation functions 等几个重要的组成部分。

#### 2.1.1 Convolutional Layers

convolutional layers 是 CNN 的核心部分，主要负责从输入的 image 中提取有用的 features。convolutional layers 的工作原理是将多个 filters 滑动到输入的 image 上，每次滑动时，计算 filters 与当前滑动窗口内的 image patch 的点积，得到一个 scalar value。将所有的 scalar values 连接起来，就得到了 convolutional layer 的输出 feature map。通常，输入 image 会被多个 filters 进行 convolution 操作，从而得到多个 feature maps。

#### 2.1.2 Pooling Layers

pooling layers 主要负责减小输入的 spatial resolution，以减少 computational cost 和防止 overfitting。常见的 pooling methods 包括 max pooling、average pooling 和 L2-norm pooling。

#### 2.1.3 Fully Connected Layers

fully connected layers 主要负责将 earlier learned features 转换为目标 task 的输出。例如，在 image classification 任务中，输入的 image 经过多个 convolutional layers 和 pooling layers 的处理后，得到的 high-level features 会被 feed 到一个 fully connected layer 中，该 fully connected layer 的输出 unit 数量等于需要预测的类别数量。

#### 2.1.4 Activation Functions

activation functions 主要负责对输入的 linear function 的 output 进行非线性映射。常见的 activation functions 包括 sigmoid、tanh、ReLU 等。
