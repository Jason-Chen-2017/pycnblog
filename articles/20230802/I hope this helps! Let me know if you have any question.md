         # 2.基本概念术语说明
         机器学习的基本概念、术语和算法包括：
            1.定义：
                橡皮鸭大学李宏毅老师在20世纪80年代末提出的“计算机程序可以模仿生物体”这一命题被广泛认为是认识计算机发展的一个里程碑事件。当时，人们对这个命题的看法有两种截然不同的理解，一方面是主张机器能够像人一样思考、学习、解决问题；另一方面则认为只要引入足够多的训练数据，机器便可以“学习”到某种模式或规律。从此，人们便产生了机器学习的观念，并将其应用到许多领域，如图像识别、文本分析、分类系统、语音识别等。
            
            与之对应的是，机器学习的目的也是为了让计算机具有“学习能力”，它可以从数据中提取出有效的信息，并据此进行预测、决策、推荐等任务。
            
            2.概念：
                监督学习（Supervised Learning）：
                    是指由已知的输入数据（Input Data）和对应的正确输出结果（Output Data），利用学习算法训练模型，使得模型对于新输入数据能给出预测性的输出结果。
                    
                    有监督学习又分为两类：
                        - 分类问题：
                            分类问题一般用于对数据的不同类别进行区分。例如，图像识别就是一个典型的分类问题。

                            根据样本数据的特征向量、标签，用统计学习的方法（如逻辑回归、支持向量机、神经网络）建立一个模型，对新的输入样本进行分类，确定其所属的类别。

                        - 回归问题：
                            回归问题一般用于预测数值型变量。例如，房价预测就是一个典型的回归问题。

                            根据样本数据的特征向量、标签，用统计学习的方法（如线性回归、多项式回归、神经网络）建立一个模型，对新的输入样本进行预测，确定其数值的大小。

                无监督学习（Unsupervised Learning）：
                    是指对数据的结构信息进行分析，发现潜在的相似性或者不相干的结构，但没有对应的正确输出结果。

                    无监督学习有三种主要方法：
                        - 聚类方法（Clustering Method）：
                            通过对数据进行聚类，自动将相似的数据划分到一起，形成几个簇。常用的聚类方法有K-means方法、层次聚类、高斯混合模型。

                        - 关联规则学习（Association Rule Learning）：
                            使用数据库中的多个事务间的关系来发现频繁出现的联系，并根据这些联系进行交易。

                        - 异常检测方法（Anomaly Detection Method）：
                            异常检测方法通过对数据分布进行建模，找出数据的中性的、不正常的行为，并进行报警、监控。
                            
                强化学习（Reinforcement Learning）：
                    强化学习是一种交互式的基于学习的控制策略，旨在实现自动选择、优化问题，而不需要人为干预。

                    在强化学习中，智能体（Agent）接收环境（Environment）提供的状态（State）信息，并通过与环境的交互，生成行动（Action）指令，从而调整自身状态。环境会根据智能体的行动，给予奖励（Reward）或者惩罚（Penalty），以此影响智能体的行为。智能体通过一定的策略，学习如何在给定约束条件下最大化累计奖赏。

                    
            机器学习算法：
                1. 分类算法
                    支持向量机（SVM，Support Vector Machine）、朴素贝叶斯（Naive Bayes）、隐马尔可夫模型（HMM，Hidden Markov Model）、K最近邻算法（KNN，k Nearest Neighbors）、决策树（Decision Tree）等。
                2. 回归算法
                    线性回归（Linear Regression）、岭回归（Ridge Regression）、Lasso回归（Lasso Regression）、局部加权线性回归（Locally Weighted Linear Regression）等。
                3. 模型融合算法
                    集成学习（Ensemble Learning）、随机森林（Random Forest）、AdaBoost、GBDT（Gradient Boost Decision Tree）等。
                4. 降维算法
                    PCA（Principal Component Analysis）、LDA（Linear Discriminant Analysis）、Isomap、MDS（Multi-Dimensional Scaling）等。
                5. 聚类算法
                    K-Means、DBSCAN、Spectral Clustering等。
                6. 关联规则算法
                    Apriori、Eclat等。
                7. 异常检测算法
                    k-NN、Isolation Forest、LOF等。
