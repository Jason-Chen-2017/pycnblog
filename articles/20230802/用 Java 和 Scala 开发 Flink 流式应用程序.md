# 1.简介
         

         2017年，Apache Flink™ 项目诞生于阿里巴巴集团。该项目是一个开源的分布式流处理框架，可以运行在实时、离线、批处理等各种环境中。它的独特之处在于其高性能的实时数据处理能力，能同时支持超高吞吐量的数据处理需求和较低延迟的响应时间。在云计算、大数据应用领域中，Flink 的广泛采用促进了其发展。本书就是基于 Apache Flink 进行深度解析，通过实践案例深入浅出地讲解 Flink 的基础知识和编程模型。
         
         在过去的十年间，由于人工智能、机器学习、云计算、大数据、IoT 技术的兴起，人们越来越多地从事服务端及微服务的开发工作。随着移动互联网、物联网、边缘计算、车联网等新兴技术的出现，Serverless 架构的普及使得前端工程师需要更多关注与参与到 Serverless 服务端的开发工作当中。Serverless 架构通过将服务器的运维任务交给第三方平台，让开发者只需要关注业务逻辑的实现。
         
         Serverless 模型极大地提升了软件开发效率，并降低了运营成本。在实际生产环境中，Flink 可以作为 Serverless 的计算引擎之一，提供统一的编程模型及高效的集群资源分配机制。相比于传统的单机部署模式，Flink 可以更加高效地利用资源，有效减少硬件投入，降低运营成本。通过本书，读者能够快速掌握 Flink 的相关技能和方法论，在实际生产环境中构建符合自身需求的流式应用程序。
         
         本书共分为六章，每章先介绍相应的 Flink 基本概念和术语，然后详细阐述该概念或术语的原理及应用场景。然后再结合示例代码和解释进行演示，最后还包括一些典型问题的解决方案，帮助读者在实际生产环境中上手实践。希望读者通过阅读本书，能够掌握 Flink 的各项基本技术和编程模型，能够在项目中应用 Flink 来实现对海量数据进行高速计算和实时分析，真正做到无人不知无人不晓。
