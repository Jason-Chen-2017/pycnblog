# 2. 核心概念与联系

2.1 计算机视觉技术概述
计算机视觉是人工智能的一个重要分支,旨在让计算机能够"看"和"理解"视觉信息,从而模拟人类视觉系统的功能。它涉及图像和视频的获取、处理、分析和理解等环节,广泛应用于医疗、安防、自动驾驶等领域。

2.2 智能医疗的内涵
所谓智能医疗,是指运用人工智能、大数据、物联网等前沿技术,实现医疗服务的智能化、精准化和个性化,提高医疗效率和质量,降低医疗成本。计算机视觉作为人工智能的重要分支,在智能医疗中扮演着关键角色。

2.3 计算机视觉与智能医疗的关系
计算机视觉技术可以帮助医疗行业实现多项关键功能,如医疗影像分析、病理诊断、手术导航等。通过对医疗图像/视频进行自动化处理和分析,计算机视觉能够提高诊断的准确性和效率,为医生提供更精准的决策支持,最终造福广大患者。因此,计算机视觉是智能医疗不可或缺的核心技术之一。
