## 4. 具体最佳实践

```python
# 构建医疗器械知识图谱的Python代码示例
import networkx as nx
import matplotlib.pyplot as plt

# 创建知识图谱
G = nx.Graph()

# 添加医疗器械实体及其属性
G.add_node('Device_A', product_name='Infusion Pump', manufacturer='ABC Corp', risk_level='High')
G.add_node('Adverse_Event_1', event_type='Overcurrent', reported_date='2022-03-15', severity='Serious')
G.add_node('Adverse_Event_2', event_type='Battery Failure', reported_date='2022-06-30', severity='Moderate')

# 添加实体之间的关系
G.add_edge('Device_A', 'Adverse_Event_1', relation='reported')
G.add_edge('Device_A', 'Adverse_Event_2', relation='reported')

# 可视化知识图谱
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True)
plt.show()
```

通过构建这样的医疗器械知识图谱,监管部门可以更好地理解设备的属性、不良事件的关联性,发现潜在的安全隐患,为后续的监管决策提供有力支撑。
