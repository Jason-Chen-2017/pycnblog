## 4.2 用户画像实例
以下是一个基于大型语言模型的用户画像构建实例:

```python
import spacy
from sklearn.cluster import KMeans

# 加载预训练的语言模型
nlp = spacy.load("en_core_web_lg")

# 提取用户行为数据的文本特征
user_texts = ["bought new laptop", "interested in digital marketing", "watched video on AI"]
user_features = [nlp(text).vector for text in user_texts]

# 利用聚类算法构建用户画像
kmeans = KMeans(n_clusters=3, random_state=0)
user_clusters = kmeans.fit_predict(user_features)

# 输出各个用户群体的特征
for i in range(3):
    print(f"Cluster {i}:")
    print(", ".join(user_texts[j] for j in range(len(user_texts)) if user_clusters[j] == i))
```
