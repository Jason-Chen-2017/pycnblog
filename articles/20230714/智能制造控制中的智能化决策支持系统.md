# 2.基本概念术语说明
## MECC
机械-电子-计算机协同控制(MECC)是一种利用现代电子技术提高制造过程效率的一种工业控制方法。它将机器，传感器，微控制器以及通过通信互联网实现的数据采集，处理和传输的功能集合在一起，通过分布式的控制系统，收集数据，进行预测和调整，确保整个制造过程能够高效运行，达到经济效益最大化。

## RNTS
实时网络优化调度(Real-time Network and Task Scheduling，简称RNTS)是一个面向实时任务调度的网络结构，它可以快速准确地根据多种调度目标对分布式任务进行最优调度，以保证实时的任务执行，有效利用资源。RNTS系统由任务调度模块、资源管理模块和网络控制模块组成。

## Data Driven Systems Control (DDS)
基于数据驱动的系统控制(Data Driven Systems Control，简称DDS)是一种运用机器学习、模式识别及统计学的方法，结合测量数据及其所产生的统计规律，建立模型，通过分析模型参数和特征，得到控制策略，进而进行控制系统的设计、优化及控制。

## Control System Engineering
控制系统工程(Control System Engineering，简称CSE)是指利用数理方程，图论，模拟，仿真等工具对控制系统进行工程建模，设计，测试，调试，改善，优化，运行等过程，达到控制系统的可靠，稳定，安全，可控的目的。

# 3.核心算法原理和具体操作步骤以及数学公式讲解
## MLE算法
MLE算法是Machine Learning Empowered Control（机器学习赋能控制）的缩写，即通过机器学习方法对控制问题进行建模和求解，从而找到一个全局最优的控制策略。
### 求解过程
1. 数据获取：获得传感器信息，物理参数，及其他相关信息。
2. 数据清洗：过滤掉无效或错误的数据，并进行标准化、归一化等处理。
3. 参数估计：假设模型参数服从正态分布，使用极大似然估计法或者最大似然估计法，计算出模型参数。
4. 模型构建：依据模型参数，构建动态系统的状态转移方程。
5. 控制策略计算：给定模型参数，确定控制策略。如：基于位置的PID控制策略。
6. 输出反馈：执行控制策略，输入给机器，改变机器运动方向，使得机器运行在合理范围内。
### 数学公式推导
MLE算法对于参数估计的假设条件为正态分布，所以可以使用最大似然估计方法对模型参数进行估计，如下所示：
$$p(    heta|D)=\frac{1}{Z}\prod_{i=1}^{N}f_i(    heta)^{\left[n_i+1\right]}{\left[\frac{m_i-1}{k_i}\right]^{-1/2}}\exp\left[-\frac{(n_i+1)s_{\bar i}^2}{2}\right]$$
其中，$    heta$是模型参数，$D$是观察到的训练数据，$N$是观察样本数目；$f_i(    heta)$表示第$i$个观测数据的概率密度函数，这里假设每个观测数据的概率密度都是一元正态分布；$n_i$是第$i$个观测数据的次数，$m_i$是平均次数，$k_i$是方差；$s_{\bar i}$是第$i$个观测数据标准差；$Z=\int_{-\infty}^{+\infty} \prod_{j=1}^{N} f_j(    heta)^{\left[n_j+1\right]}{\left[\frac{m_j-1}{k_j}\right]^{-1/2}} e^{-n_j s_{\bar j}^2/2} d    heta$表示似然函数值。

求解最大似然估计的结果可以得到模型参数的期望值，如下所示：
$$\hat{    heta}_{MLE} = \underset{    heta}{\arg\max}\prod_{i=1}^{N}f_i(    heta)^{\left[n_i+1\right]}{\left[\frac{m_i-1}{k_i}\right]^{-1/2}}\exp\left[-\frac{(n_i+1)s_{\bar i}^2}{2}\right]$$
## NN算法
NN算法是Neural Networks Empowered Control（神经网络赋能控制）的缩写，即通过神经网络（NN）对控制问题进行建模和求解，从而找到一个全局最优的控制策略。
### 求解过程
1. 数据获取：获得传感器信息，物理参数，及其他相关信息。
2. 数据预处理：将数据转换成适合用于神经网络的形式。
3. 模型训练：使用神经网络模型，对预处理后的数据进行训练，得到模型参数。
4. 控制策略计算：给定模型参数，确定控制策略。
5. 输出反馈：执行控制策略，输入给机器，改变机器运动方向，使得机器运行在合理范围内。
### 数学公式推导
NN算法的模型训练过程可以使用交叉熵损失函数进行训练，损失函数的值越小，代表着模型效果越好。
$$L(    heta)=-\frac{1}{N}\sum_{i=1}^{N}y^{(i)}log\left(f_    heta(x^{(i)})\right)+\lambda||    heta||^2$$
其中，$L(    heta)$是损失函数，$N$是训练数据量；$y^{(i)}$是第$i$个训练数据的标签值，$f_    heta(x^{(i)})$是输入向量$x^{(i)}$对应的输出值；$\lambda ||    heta||^2$是正则项，用来防止过拟合。

## LSTM算法
LSTM算法是Long Short Term Memory Empowered Control（长短期记忆神经网络赋能控制）的缩写，即利用LSTM对控制问题进行建模和求解，从而找到一个全局最优的控制策略。
### 求解过程
1. 数据获取：获得传感器信息，物理参数，及其他相关信息。
2. 数据预处理：将数据转换成适合用于LSTM的形式。
3. 模型训练：使用LSTM模型，对预处理后的数据进行训练，得到模型参数。
4. 控制策略计算：给定模型参数，确定控制策略。
5. 输出反馈：执行控制策略，输入给机器，改变机器运动方向，使得机器运行在合理范围内。
### 数学公式推导
LSTM算法的模型训练过程采用二分类问题，损失函数采用sigmoid函数进行计算，评价指标采用精确率和召回率。
$$Loss=\frac{1}{N}\sum_{i=1}^{N}[\sigma(-y^{(i)}(t))log\left(\sigma(h^{\prime}(x^{(i)};    heta)\right)+(1-\sigma(-y^{(i)}(t)))log\left(1-\sigma(h^{\prime}(x^{(i)};    heta)\right)]$$
其中，$h^{\prime}(x^{(i)};    heta)$表示LSTM模型的输出值，$\sigma(z)=\frac{1}{1+e^{-z}}$是sigmoid函数。

# 5.未来发展趋势与挑战
随着智能制造技术的不断发展，MECC、DDS和RNTS等智能制造控制方法逐渐成为工业控制领域的主流，各类控制系统也日臻完善，但仍存在许多问题需要解决，比如复杂的控制系统设计、自动化技术的革新、资源约束以及多任务协作的困难等。在这些问题的共同下，我国的制造业正在进入新的机遇期，如何有效、可靠、安全地控制生产设备，成为生产工艺、生产流程的关键问题。
