# 2.基本概念术语说明
## 2.1 AWS Blockchain 服务简介
AWS Blockchain 是一种完全托管的、安全可靠的区块链服务，提供高可用性的区块链网络和Solidity智能合约开发环境。该服务具有如下优点：

1. 完全托管：无需担心底层服务器硬件配置、维护或运维，只需要专注于业务逻辑的实现。
2. 安全可靠：Amazon 会提供高度保证的存储、处理、传输等服务，确保数据安全和机密性。
3. 可伸缩性：根据您的需求自动调配区块链网络的计算资源，确保网络能应对日益增长的数据量和用户请求。
4. 成本效益：AWS Blockchain 通过降低网络维护和托管成本、提供针对不同类型的客户定制的收费策略，帮助您节省大量的开支。
5. 易于使用：该服务提供图形化界面的便捷访问方式，使得开发者可以快速部署和开发区块链应用。
6. 可扩展性：你可以利用 AWS 的服务扩展性来满足你的需求。例如，你可以按需扩展区块链网络节点数量，提升性能或降低成本。
7. 多样性：AWS Blockchain 服务提供的网络架构可以选择多种网络拓扑结构，并支持联盟链、公有链或许多联盟模式。

## 2.2 一些重要的基本术语和概念
### 2.2.1 Ethereum Virtual Machine (EVM)
EVM 是区块链的虚拟机，是区块链上运行的智能合约的实际执行环境。在EVM中，所有的智能合约都是基于虚拟机实现的。智能合约通常由部署到区块链上的账户执行。每当发生特定事件时，EVM都会检查是否有相应的触发器，如果有则运行对应的智能合约。

### 2.2.2 Smart Contract（智能合约）
智能合约是一个用于管理数字资产的计算机协议，它基于分布式计算和密码学，并由区块链技术驱动。智能合约定义了合同条款、履行条件、违规惩罚措施等法律约束。任何有关商业活动的法律条文都可以转变为智能合约，并在区块链上自动执行。比如，在线购物网站会基于消费者行为生成一个智能合约，由区块链自动执行，确保消费者享受到全新购买体验。

### 2.2.3 Blockchain Network（区块链网络）
区块链网络是一个公共的、分布式的、点对点的去中心化网络，其中包含众多的节点（也称为矿工）以安全、可信的方式来存储、验证和传递区块链交易记录。区块链网络以去中心化的方式工作，这意味着所有参与者都独立地保持相同的副本。每个节点都有权利创建新区块，并添加到区块链中。网络中所有的节点都保持同步，不断确认并确认前序区块，从而形成一条记录整个分布式系统状态的链条。

### 2.2.4 Public and Private Networks （公有链和私有链）
区块链网络可以是公有的也可以是私有的。公有链允许任何人连接加入，所有参与者共享同一套技术基础设施；私有链由信任的实体成员组成，该实体拥有独自管理的基础设施、节点、服务和数据。对于私有链，没有任何机构能够控制网络，因此其安全性得到提升。

## 2.3 概念相关的问题解答
### 2.3.1 区块链网络架构类型有哪些？
区块链网络架构类型有两种：

1. 联盟链（Consortium Chain）：联盟链是指由多个参与者共同遵守共识规则的区块链网络。在这种架构下，只有经过认证的参与者才能加入区块链网络。该网络的所有节点都参与共识，也就是说，它们都存储了相同的区块链数据。联盟链架构的典型应用场景如银行间交易或跨境支付。
2. 私有链（Private Chain）：私有链是一个由单个组织或公司所建立的区块链网络。该网络由本地的服务器、数据库、应用程序和网络设备组成。私有链可用于保护企业数据的隐私，并可用于私有数据的处理。私有链架构的典型应用场景如银行或电子游戏领域。

### 2.3.2 什么是Solidity智能合约语言？
Solidity 是一种面向对象的编程语言，旨在实现智能合约。它是Ethereum虚拟机(EVM)的一个抽象层，让开发者可以非常容易地编写智能合约。Solidity 提供了一系列的关键字、运算符、语法和结构，使得开发者可以定义任意复杂的合约。

### 2.3.3 什么是DApp（分布式应用）？
DApp（Distributed Application）是基于区块链的去中心化应用程序。其定义为利用区块链技术构建的应用程序。DApp在本质上就是基于区块链的软件服务。DApp基于分布式账本、数据加密技术、智能合约等区块链技术构建。DApp属于去中心化的应用程序，可以被部署到不同的区块链网络，并通过互联网暴露给终端用户。DApp的特性主要有：

1. 用户友好：DApp的界面设计清晰简洁，操作简单，用户直观地了解它的功能。
2. 自动化执行：DApp中的智能合约都能自动执行，无需用户主动触发，自动完成智能合约流程。
3. 数据安全：由于区块链的去中心化特性，DApp可以确保数据的安全性。
4. 可追溯：DApp可以显示历史操作记录，记录所有用户操作，并为用户提供详细的反馈。

### 2.3.5 AWS Blockchain 服务目前支持的区块链网络有哪些？
AWS Blockchain 服务目前支持的区块链网络有以下几种：

1. Ethereum 区块链：AWS Blockchain 服务支持在Ethereum区块链上部署区块链网络。Ethereum 是一个开源的、基于智能合约的区块链平台，目前已经成为了区块链行业的领导者。
2. Hyperledger Fabric：Hyperledger Fabric是一个开源的、可插拔的区块链框架，适用于基于通道的业务场景。
3. Quorum：Quorum是一个由JP Morgan发起的开源项目，旨在创建一种新的去中心化的、企业级的联盟链解决方案。
4. Corda：Corda是一个基于区块链的分布式账本技术，旨在促进金融应用程序的部署、交付和管理。
