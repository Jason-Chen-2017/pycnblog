# 1.背景介绍


“心理准备”是技术面试中重要的一环。比如，面试前先做一道简短的自我介绍，询问应聘者对自己的看法、优缺点等方面的意见，或者给予面试官一些面试前的自我激励。这不仅能够让面试官更加了解面试者，也能帮助应聘者在面试过程中获取信息，从而调整面试策略以提高面试效果。但是，如何在面试过程中有效地进行“心理准备”，对于提升自信心、掌握技术、克服困难等，都具有非常重要的作用。
 
为了方便阅读和传播，本文将以面试者为中心，总结面试过程中的心理调适知识点。首先，我们对“心理准备”进行定义，然后讨论什么时候应该进行心理准备，何时应该放弃。接着，我们将结合常见的心理情景展开探讨，包括面试前期的紧张、疲惫、紧迫感、内心不安、紧张气氛、出错、失望等，以及面试过程中的内心冷静、平静、镇定、冷静等心理反应。最后，我们分享一些面试官可能会参考的心理准备经验和方法，并鼓励技术人才多关注自己与他人的关系。希望通过这些心理调适技巧，能够让技术面试者受益匪浅。 
 
# 2.核心概念与联系
## 2.1 什么是“心理准备”？
“心理准备”是指面试过程中进行的个人自我判断、状态监控、心理承受能力训练等活动，旨在增强候选人自信心、能力、协作精神、分析问题、解决问题的能力及动机。它可以促进面试官和面试者之间的沟通交流，还可以帮助候选人提升竞争力、塑造职业形象、优化职场生态。
 
在技术面试中，“心理准备”通常是由面试官主导的，目的是为面试者提供一个更加“安全”的环境，来保护面试者的自尊心、自信心和安全感。同时，“心理准备”也同样重要，因为它能够帮助面试者识别工作中的问题，帮助面试官评估面试者的潜力、更好地分配面试资源，并帮助面试者控制焦虑情绪。当候选人具备较好的心理素质时，他们往往更有可能得到更好的面试结果。
 
## 2.2 为什么要进行心理准备？
### 2.2.1 提升自信心
心理准备在面试前对技术人员的自信心和自我保护能力有很大的影响。尤其是在面试官针对个人长处和经历的筛选，面试者的自信心也会受到影响，有着直接的影响力。过早的进行“心理准备”可能会让面试者产生恐慌，因而降低面试的效率，甚至导致失败。因此，面试前要充分考虑面试者的心理状态、发挥特长、完善自我介绍等，提升候选人的自信心。
 
### 2.2.2 掌握技术
技术面试需要面试者能够快速理解面试题目，并且能够根据实际情况、知识储备、简历、面试经验，正确回答面试官提出的问题。如果没有足够的时间、充分的训练，则面试者可能会被面试官淹没、被迫害、丢掉工作等。因此，在技术面试中，对候选人“心理准备”很重要，它将对候选人的技术能力、解决问题的能力等各项技能进行考察。
 
### 2.2.3 克服困难
即使面试官面对面试者的优秀表现、十分重视候选人的自我介绍、多次邀请参加面试，但在面试过程中仍然容易出现一些误区，比如觉得候选人做不到、没有把握面试题、难以理解面试官的要求等。这时候就需要技术人员对此进行自我分析和总结，以及提升自身的团队合作能力、协助面试官处理信息、应对挫折，以减少或避免造成的伤害。
 
## 2.3 什么时候应该进行心理准备？
一般来说，技术面试中，“心理准备”最有效的时机是面试前两周。原因如下：

1. 技术面试需要时间——“心理准备”是整个面试过程中的关键环节，若花费太多时间准备，则面试官很难集中注意力在候选人身上，而导致面试时间过长；
2. “心理准备”需要专注——如果在心理上分散了注意力，则面试官很难全面评估候选人的综合素质；
3. “心理准备”需要细化——一份详细的“心理准备”计划可以帮助候选人在面试中有所反映，且易于落实。

在技术面试中，如果候选人能够在面试前对自我介绍清楚、真诚、负责任，并向面试官展示自己擅长的领域、学习能力、工作积极性，那么“心理准备”的效果将更加明显。不过，无论何时，只要候选人在“心理准备”中保持冷静、客观，充分准备，一定可以获得不错的面试效果。
 
## 2.4 什么时候不要进行心理准备？
以下情况建议不进行“心理准备”：

1. 在面试中，面试官已经非常熟悉候选人，而且已经对面试者的能力、品格等有了比较准确的判断。在这种情况下，“心理准备”其实没有任何意义，甚至会影响面试效果。
2. 当面试官和面试者都还处在正常的教育阶段，没有过多的事情可以交代、深入谈话。在这种情况下，“心理准备”反而不利于候选人的个人成长和能力提升。 
3. 如果候选人对面试官的个人品味、职业操守有质疑，认为面试官刻意刁难、攻击候选人，这时“心理准备”就显得没有必要了。毕竟，面试官的初衷是为了选出最佳的候选人，而不是利用候选人的弱点。
4. 如果候选人之前就曾接受过系统性的考验，不太可能再有时间参加面试，因此“心理准备”就显得多余了。
5. 面试官希望候选人完成的任务简单、单一，所以“心理准备”并不能突破这一限制。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
技术面试过程的关键之一就是面试官掌握候选人的技术能力。但这个能力如何表现出来呢？我们可以借鉴社会心理学中经典的E/I（外部）和I/E（内部）两种心理类型，分别对应技术面试者在面试中表现出的外部动机和内部驱动。
 
## 3.1 E/I
E（externality）指的是候选人对面试官表现出的某种性格外貌、行为举止等等，如神经质、贪婪、缺乏耐性等。
 
I（internality）指的是候选人由于个人长处或经验所具有的智力和理性水平，并对面试官可供选择的职位等有高度自信心和自尊心。
 
## 3.2 I/E转换的过程
1. 确定自我:首先，面试者应该确认自己的价值取向和身份，并给面试官一个正面的自我介绍。此时，候选人可以开始展现自己的个性和兴趣爱好，同时也可以与面试官分享自己的职业规划、追求梦想等事业方向。此时的自我表现程度可以用EI来衡量，E指的是脸部表情、姿态、语调、动作变化等表现，I指的是年龄、性别、家庭状况、文化背景、学习经历等个人信息。
  
2. 测试候选技能:当面试官开始调查候选人的工作经验、相关证书、语言水平、技能、兴趣爱好等信息后，候选人就可以展现自己的技能。这时，候选人只能通过个人陈述的方式展现自己的技能，通过这种方式面试官可以听到候选人有多么的专业、多么的积极，也可以知道候选人是否是一个热衷于分享知识的好学生。此时的自我表现程度可以用II来衡量，I指的是技能、专长，II指的是学历、学校、工作经验、证书、个人描述等。
  
3. 提出问题:候选人在向面试官提出问题时，可以尝试回避一些与技术无关的问题，比如生活、健康、职业规划等。这时，候选人已具备基本的技术技能，面试官需要以一种深入的思维方式来解答问题，才能准确地了解候选人是否适合这个岗位。此时的自我表现程度可以用IE来衡量，I指的是技术能力、问题解决能力、创新能力、逻辑推理能力，E指的是遇到的问题和困难。
  
4. 回答问题:面试官根据候选人自己的回答，从多个方面展现自己的能力。例如，面试官可以提出问题，询问候选人是否真诚待人，是否敢于探索新的技术领域，如果有机会，面试官可以邀请候选人去分享自己的科研成果，希望能够增进彼此的互动。此时的自我表现程度可以用EI来衡量，E指的是候选人是否展示出自己深厚的学识和专业知识，以及对自我介绍、问题的回答是否认真细致，I指的是面试官对候选人所说的自己是否认同和理解，是否支持他的发展方向。
 
5. 比较结果:面试官会根据候选人个人和相关信息、测试结果，最终决定面试结果。面试的最终结果，主要取决于面试官和候选人之间自我和对方的关系，以及面试官和候选人的沟通技巧。如果面试官的答辩方式与候选人的需求相符，候选人在表现上的突出优势，以及面试官和候选人的合作精神，都会对最终结果产生重大影响。
 
## 3.3 以E/I（外部/内部）模型解释技术面试者的行为特征
从上述分析可以发现，技术面试者在面试中有两个主要的心理特征：

1. EI模型：面试者展示出对面试官可取的表现，并且以面试官期待的方向去发展。

2. II模型：面试者展示出自己的专业技能，并且提出了令面试官满意的问题，等待面试官回答，以保证面试的顺利进行。

虽然，技术面试是个双向的过程，面试者既要展现自己的能力又要寻找职位，但实际上，技术人员只是其中一部分角色，即使是创新型企业里面的技术类岗位也是如此。

所以，除了以E/I（外部/内部）模型来理解技术面试者的行为特征外，还可以从其他角度来看待技术面试者：

1. TE模型：面试者试图通过模拟、测试等方式来获取反馈，最终达到模仿面试官的目的。

2. IT模型：面试者试图通过对新事物、产品的理解来判断自己的技能水平。

3. SC模型：面试者试图通过模糊技能和思维的方式，构建自己的假设世界，然后去逼近真实世界。
