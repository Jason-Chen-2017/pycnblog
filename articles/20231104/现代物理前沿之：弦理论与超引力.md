# 1.背景介绍


## 概述
弦理论（String Theory）最早由埃弗里德·哈斯金（Evariste Haussler）在1927年提出，是指用牛顿力学研究粒子束运动的相互作用，从而得出的关于宇宙中各种观测现象的理论。弦理论提出的特点就是它可以描述所有弯曲、旋转的真实现象。弦理论的主要观察对象是质点运动中的弧线或棱线。其支撑理论有微观物理学、量子力学和统计力学。
## 物理意义
19世纪末期，费米-狄拉克定律在牛顿力学的基础上发展形成的弦理论被认为是量子力学的反面。弦理论指出，宏观层面的费米-狄拉克定律和狭义相对论只允许处于静止平衡状态，但许多实际物理系统却可以非平衡地运动。这种“自由落体”现象既可以是单个粒子的运动，也可以是某种相互作用的集合现象，如铁磁耦合、弱核反应堆等等。因此，弦理论提供了一种新的途径来观察这些奇异性质的物理现象。弦理论的这种直接观察方式颠覆了古典力学的框架，并催生了物理学的新时代。


超引力（Supergravity）是弦理论的另一个分支，即考虑重力对物质运动的影响。由于重力场中的高速旋转导致加速度远大于引力场，因而称之为超引力。超引力的物理过程可以比照弦理论理解。弦螺旋运动只是物质运动的一个例子，实际上，超引力也出现在宇宙中的很多现象中，如星系团成长、黑洞的形成等。
## 大众认识
超引力是一个高度复杂的主题，它的范围很广，涉及到广泛的物理学领域。以下简要概括一些超引力相关的名词和观念。
### 形状与物理效应
超导材料(superconductor)和超导阴极材料(superoxide)是目前应用最广泛的超引力材料。它们具有透明、高电容性、高低温变化性和导电性，能够产生强大的电磁辐射，并能够自恢复到基态。另外，中子号（quark）和质子（proton）被证实存在着相互作用的超导效应。质子和中子引起的超导效应可产生弓形和螺旋型弥散现象；而中子和电子引起的超导效应则可产生杂乱流和粒子积聚。此外，不同物质的无色介电和有色介电效应也可以被利用作为超导物质的物理机制。
### 天体及星系团
超级大质量星体的形成和生命活动都是超引力的重要组成部分。这些对象都经历了巨大的磁场旋涡膨胀，使得他们的轨道发生扭曲。当银河系的一颗行星失去引力恢复的时候，就会引起银河系的崩溃。太阳是一个重要的超引力观察对象，它通过地球公转来获得引力，也通过自己的日食和行星的环绕而受到影响。
### 凌美珍惜
科学家们认为，超引力不仅仅是物质运动所带来的现象，而且还有其他深刻的物理含义。比如，超导材料中的纳米晶体管可能有助于传输信息、提升数据处理能力、构建集成电路等。伽利略的相对论、卢卡斯主张的时间不确定性、黑洞的形成等等，都可能与超引力密切相关。
### 不寻常的结构
超导体、超导金属、极限不稳定的原子核、引力波、量子飘浮、自旋本性子以及其他奇妙的物理现象，都可能与超引力息息相关。但是，要搞清楚超引力的真正本质还是需要对基础理论有深入的理解。
