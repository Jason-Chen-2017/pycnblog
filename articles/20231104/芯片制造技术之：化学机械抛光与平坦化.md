# 1.背景介绍


## 什么是化学机械抛光？
化学机械抛光（Chemical vapor deposition, CVD）是一种将金属粘土、氧气或其他物质液体在一个平面表面上进行流体注入的方法。该方法被广泛用于在硅片、电子件、印刷电路板等制造中实现对表面不良、缺陷和缺陷补偿的修复。人们常用化学机械抛光技术将无缺陷处的物理材料表面涂抹有色胶沉积上色，提高表面的清晰度、纹理饱和度，并产生优异的光照耐性和防护性能。这种方法也是制造玻璃、塑料和绝缘子等无纺锤、焊接需要的表面材料的主要方式之一。
## 为何要进行化学机械抛光？
由于硅和绝缘材料的特殊结构特性，使得它们易受外界环境影响而导致的缺陷增多，因此在工艺生产时需要进行处理，否则可能导致产品质量下降。利用化学机械抛光方法可以消除这些不良影响，从而使产品质量达到合格标准。此外，由于化学机械抛光技术十分简单便利，而且相比于其他加工工艺（如印刷工艺）成本低廉，所以具有很强的工程应用价值。
## 什么是化学平坦化？
化学平坦化（chemical curing process）是一种在研发生产过程中经过金属和非金属颗粒作用温升过程之后所形成的一种半固态状态。在当这个过程结束后，介乎前后固态和气态之间的分界带即被填满形成了一个有限尺寸的均匀、无缝、缺陷缺少的表面。这种表面既能够良好地反应介质中的活性元素，又不留任何裂痕、开裂现象。化学平坦化技术是为了解决硅片、绝缘片、液晶、玻璃、电子器件等软材料表面的缺陷问题，特别是在工艺生产方面。它的应用举凡电子屏显示器、射频照明管件、机器人的机身表面、汽车玻璃外壳、光刻胶等领域。

### CVD原理
1. 首先，制备好化学介质，如金属粘土、氧气或其他物质的溶剂；
2. 将介质及其溶剂浸泡在一定的时间，使浓度达到稳定状态；
3. 在给定的研磨速度、温度及风险容忍范围内，反复施加施加力矩，形成稳定的均匀颗粒状物料；
4. 将这些颗粒状物料铺平在表面待固化；
5. 当固化物料全部形成以后，就可以将介质倒入孔口，使其逆渗透进入玻璃瓶等物品；
6. 如果介质中的元素含量过高会导致材料融化，可在固化过程中加入氮气以稀释元素，或者在预固化时把表面缩小一些，让固化物质充分流向无缺陷处。
### 什么是激光光刻胶切削？
激光光刻胶切削(Laser Etching) 是一种精密光刻技术，它通过使用激光光刻机将电子信息转化为光电信号，再将光电信号转化为二极管形态的材料层，最后形成整体坚固且光亮度较高的材料。它是目前国际上应用最广泛的电子材料切削技术之一。  

在Laser Etching 的基础上，采用化学机械抛光技术还可以创造出由结构光所组成的光刻胶，这类材料具有较好的物理、电学和功能性能，但因材料大小、复杂度及需要高速工艺设备的增加，其制造工艺流程复杂、耗时长，且制作出的光刻胶材料质量不一定高。一般情况下，化学平坦化技术与激光光刻胶切削技术配合使用来制造光刻胶材料。

### 什么是石墨烯抛光？
石墨烯抛光（Graphite Deposition）是将氢氟化钠在水分子晶体中得到，石墨烯晶体的晶格结构，在化学平衡作用下形成的。其特点是无需等待温升，不需反复加热，因此表面摩擦紧密，具有优越的表面敏感性、强韧性及防腐蚀性，因而能高效、经济、精确地处理无缺陷处的物理材料。一般用于制造光刻胶制品、电子膜、LED灯等高分子及超高分子件。
