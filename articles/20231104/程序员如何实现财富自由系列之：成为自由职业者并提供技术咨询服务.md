# 2.核心概念与联系
首先，让我们搞清楚两个关键词：“自由”和“职业”。

**自由：**自由是指人的自主选择权，即个人或集体不受任何外界干涉而自己创造自己的生活方式和生活境界，包括工作、旅行、住宿、学习、艺术创作、个人价值观等。

**职业：**职业就是从事某种特定工作或职位，或者完成特定任务，并使之持续下去的生活方式和方式。例如，计算机工程师就是一名职业，他负责开发编写软件程序，帮助企业提升生产效率。

要实现自由和职业之间的平衡，就需要建立起职业生涯的完整且可控的体系，也就是所谓的“自由职业者”。

那么什么是“自由职业者”，他们能够实现什么样的目标？

自由职业者必须具备以下基本条件：

1. 自己闲暇时间能够投入（一般至少每周有一小时以上）。
2. 良好的职业操守、道德品质及受教育程度。
3. 有足够的专业技能知识，能够为社会创造价值。

实现财富自由的方法之一就是“自由职业”，这种制度赋予了人们自主选择的权利，因此可以选择从事自己感兴趣的工作，或选择停止工作或离开已有工作岗位，让自己做自己喜欢的事情。他们可以做自己热爱的事情、自己感兴趣的工作、付出自己认为合适的劳动。

## 一、创建职业生涯计划
第一步是创建一个职业生涯计划，制定具体的工作计划、目标、掌握技术和技能等，以及每个阶段应达到的要求。如图所示，包括：

1. 规划个人职业发展方向；
2. 确定个人发展路径，制定职业生涯规划；
3. 为个人职业发展规划个人发展目标；
4. 创建个人职业发展计划；
5. 设置关键目标，为职业生涯规划奠定基础。


## 二、规划个人职业发展方向
根据个人的兴趣、能力和长期目标，选择一条适合自己的职业发展道路，既考虑短期内的职业收益，也考虑长远发展规划。如图所示，包括：

1. 根据个人特点、目标和兴趣选择适合的职业方向；
2. 选择符合个人价值的职业，否则会掉队或失败；
3. 掌握自己熟练的技能和专业领域；
4. 不断提高自我完善能力和知识水平；
5. 欢迎别人改革，保持创新精神。


## 三、为职业生涯规划选择适合的工作
选择的工作一定要满足工作的稳定性、合理性、有利于个人发展的目标和职业发展方向，同时还要为个人创造效益，达成个人的职业目标。如图所示，包括：

1. 选取稳定的工作，抵御突发状况影响工作环境；
2. 从事与个人兴趣和能力相匹配的工作；
3. 对待雇主负责任，避免对公司造成损失；
4. 主动承担责任和义务，履行义务的结果并达到预期；
5. 不参加不必要的工作，降低对健康、财产、人际关系的影响。


## 四、设置好个人健康风险控制措施
健康是每个人都非常关注的因素，因此，为了防止发生意外伤害，需要妥善处理好个人健康状况，尤其是在职业生涯中。如图所示，包括：

1. 为自己进行健康检查；
2. 在工作中注意体态、饮食和睡眠；
3. 保持正常睡眠、运动和节食；
4. 使用戴口罩、护目镜和勤洗手；
5. 保持安全距离和遵守当地法律法规。


## 五、维护个人生活秩序
家庭是个人最重要的财产，其规模甚至可以超过个人职业生涯，因此，为家庭的稳定和幸福着想，需为之付出艰苦的努力。如图所示，包括：

1. 积极主动维护个人和家庭的独立和自主权；
2. 护理个人和家庭的财产；
3. 保持良好的工作和生活习惯；
4. 不依赖亲戚朋友关系；
5. 不仅要做好日常的物质和精神需求，还要关注家庭的健康和福祉。


## 六、扩大个人影响力
了解自己与他人的不同，是建立个人影响力的第一步。为了营造优良的工作氛围，你应该与同事建立良好的 rapport 和互动。你可以主动向他人倾诉你的感受，邀请他人加入你的项目，鼓励他人为你提供建议和意见。

你可以花时间认识其他志同道合的朋友，了解自己的优点和缺点，向别人介绍你的想法和建议，发表看法，并倾听别人的意见。这样你就逐渐地建立起与他人的关系，形成自己的网络，提升个人的影响力。
