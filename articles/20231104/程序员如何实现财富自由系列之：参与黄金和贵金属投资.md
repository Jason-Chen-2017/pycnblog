# 1.背景介绍


黄金和白银市场是全球最大的贵金属交易市场，国际上大概有十几亿美元左右的黄金和白银交易量占据着全球贵金属交易市场的绝大部分。作为一个技术人员，一般不太可能直接参与黄金和白银的交易，但可以通过做一些量化交易、选股策略等方式提高自己对黄金和白银市场的认识、了解。今天，我将和大家一起探讨一下参与黄金和白银投资有哪些方法、以及怎样参与其中，以期达到更高的收益。
# 2.核心概念与联系
首先，我们需要了解一些相关的核心概念，它们之间的关系如下图所示：


① 黄金价格指数（Gold Price Index）: 指数以国际货币基金组织发布的“全球金价指数报告”为依据，由专业机构评估每月黄金的买卖情况并对其进行综合排名，反映当前黄金市场的整体状况，可用来判断国内黄金市场的走势及运动方向。

② 黄金存款利率（Gold Deposit Yield）: 以年化利率形式表示，表示持有黄金的银行家通过存入黄金的方式获取的收益。

③ 沪深300指数：全球范围内上证、中证、沪深300指数成份股构成的大盘股票指数，其收益来自于国内上市公司持有的本产品股票的涨跌，该指数可广泛用于股市分析、投资决策、仓位管理。

④ 白银价格指数（Silver Price Index）: 指数以国际货币基金组织发布的“全球白银报告”为依据，由专业机构评估每月白银的买卖情况并对其进行综合排名，反映当前白银市场的整体状况，可用来判断国内白银市场的走势及运动方向。

⑤ 白银兑换黄金比率（Silver to Gold Ratio）: 表示单位1吨白银能换取多少克黄金。

⑥ 投资黄金和白银，最重要的是要找准黄金和白银的核心市场定位，才能取得成功。对于投资者来说，黄金和白银的区别主要在于：

1.价格：黄金更加透明、可靠，因此交易价格远高于白银；而白银则价格便宜、容易被偷窃，并且流通性较差。
2.货币属性：黄金拥有稳定的货币属性，可以固定兑换黄金，并且黄金的购买力比较强，可以长久地保存；而白银因为拥有较少的流通性、价格便宜、易腐蚀等特性，所以无法像黄金那样随意兑换。
3.周期性：白银存在着比较长的时间效应，周期性上也比较短；黄金则比较短时间周期性也比较长。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 黄金市场开拓者——布莱顿·皮尔森（Bloomberg Peirson）：
布莱顿·皮尔森是美国股票期货研究所的创始人，也是著名的黄金市场开拓者之一，他在1975年创立了雷曼兄弟公司，以保证他的好朋友布莱德利·诺瓦斯（Brad Novais）将黄金作为一种实际使用的货币来积累储备。虽然布莱顿·皮尔森在创业过程中很早就意识到了黄金和其他任何一种稳定货币一样，需要有固定的供应，但他坚信黄金可以解决黄金交易中的一切问题，包括贬值、印花税以及货币宽松等。

布莱顿·皮尔森首先确定了两种类型的黄金交易者，分别是那些相对低级别、具有经验的投资者、以及那些仅仅是一味追求金钱、不顾风险的散户。其中，前者具有实战经验，能够把握股票市场的规律，并且能够使用各种不同的工具来帮助自己进行交易，比如说趋势跟踪法、指数基金、多空组合、价差交易、长线保值等。后者则更多地依赖于个人直觉和感情，并且更注重个人能力而不是盈利能力。布莱顿·皮尔森建议投资者首先关注一波牛熊市，然后逐步减小仓位进入震荡市或向下反转后的均线金叉区域，这样可以获得长期的回报。

在黄金市场上，布莱顿·皮尔森提出了三条最基本的原则：

1.鼓励炒作：因为黄金相对其他任何一种稳定货币，它的价格仍然会发生变化，而炒作黄金的人往往没有意识到这一点，而且往往伴随着股市的过度炒作，这可能会导致股价的拉升，使得黄金交易者蒙受损失。所以，只有那些炒作黄金的人，才值得信任。

2.放弃短线交易：黄金市场的高抛低吸现象严重影响了短线交易。也就是说，短期内买入的黄金如果出现价差过大甚至形成死叉时，卖方往往会以极端手段将其砸盘。为了避免这种情况的发生，只有在长期趋势下才进行短线交易。

3.倍加注意大笔交易：很多投资者认为，向上突破，卖出黄金就是赚钱，但其实这是错的。在正常情况下，黄金价格在上涨的时候会迅速增加，但在某些特殊情况下，比如黄金贬值，它会下跌惨重。在这种情况下，向上突破，就会损失惨重。因此，投资者要特别注意购买黄金的金额，以免造成损失。另外，短期内大量持有黄金会导致其跌落到吸收价以下的程度。所以，只有合适的时机，才能密集买入黄金，并且保证长期的仓位保持在均线以上。

除此之外，布莱顿·皮尔森还提出了另一条黄金规则：“不要做无谓的炒作”。黄金交易者应该做到不炒作，只在必要的时候买进黄金。事实上，当时美国的房地产市场又一次证明了炒黄金的无用处。另外，为了防止黄金贬值带来的风险，投资者也应该关注黄金市场的长期趋势。布莱顿·皮尔森最后建议投资者应该勤奋储备黄金，并且购置一批含有一定折扣的黄金黄培股票。

## 3.2 白银市场的开拓者——哈里·费尔德曼（Harold Felderman）：
白银市场是在美国历史上第一次成为真正意义上的贵金属市场，它在短短的五六年间就成为美国股市上的龙头，几乎掀起了一阵似曾相识的浪潮。由于白银的不可替代性和稳定性，白银市场的交易者们一直以来都是一群在追逐利润的交易员，他们越来越多地参与白银的交易，制定了许多有效的策略，以此来证明自己的眼光、能力、经验和勤奋。但是，因为白银的货币属性和买卖的限制性，它对投资者的吸引力在很大程度上还是限于高级交易者的范围。

哈里·费尔德曼是位资深的白银交易者，他的交易策略涉及到黄金的配对交易、短线加长线套利、建仓策略等。但是，白银市场又不像黄金市场那么开放，所以这些交易策略只能由专业的交易员去设计、实施和执行。

哈里·费尔德曼将白银市场分为两类，第一类是基础设施类市场，即主要为黄金出口商提供白银市场的交易服务。第二类是跨境期货市场，主要基于交易翼品种白银市场的交易，也称为跨境远期市场。

他认为，白银市场的发展可以从三个方面入手。第一，降低中间价差，这是白银市场的主要矛盾。另外两个主要市场，黄金和纸，都已经成为中国经济增长和发展的主要支柱，且其价格具有决定性作用，因此，降低中间价差，会为整个金融市场减轻负担。第二，推动世界经济复苏，白银市场的流通性要求也越来越强，这会促进国际收支平衡，降低贸易顺差，减少原材料消耗。第三，缓解贫富差距，目前白银交易者面临着巨大的不平等，这也是白银市场的主要任务。

除了上述观点，哈里·费尔德曼还提出了一个准则，即“寻找趋势中游”，这个准则非常重要。为了尽快找到趋势中游，白银交易者需要采用一些经典的技术分析工具，如均线金叉、指标转向、价差加仓、波动指标等，并结合实际业务知识、经济数据和财务数据等进行调整，最终制订出符合市场要求的交易计划。

## 3.3 对中国白银市场投资的启示
投资者要特别注意中国白银市场的独特性。首先，中国经济的发展受到非洲裔和拉丁裔文化等因素的影响，其次，中国的央企大量介入白银市场，导致中国白银市场流动性弱。最后，中国近十年来一直在限制中国白银出口，这也将使得中国白银市场的流动性进一步下降。但是，中国白银市场依然具备蓬勃的发展潜力，尤其是在人口结构、工业化水平较高的国家，尤其需要青睐白银市场作为人民币国际化、民族国家开发的重要组成部分。因此，中国白银市场有很多投资机会。
