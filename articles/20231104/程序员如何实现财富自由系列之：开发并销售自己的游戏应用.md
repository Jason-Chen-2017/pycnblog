# 1.背景介绍


当今的互联网时代给予了个人不少便利，让个人变得更加独立，让各种各样的人都能享受到互联网带来的便利。近几年，网络游戏产业蓬勃发展，人们越来越重视网络游戏的娱乐性和快感，玩家也越来越多地喜欢网络游戏。然而，随着游戏行业的发展，游戏行业面临的风险也越来越高，这其中最突出的就是付费问题，这就意味着玩家在玩游戏过程中可能会有支出。游戏行业尤其是网络游戏行业的盈利模式还处于起步阶段，相对于实体游戏行业来说，盈利模式较为保守。因此，对于一些想要赚钱或者有收入渠道的游戏玩家来说，想通过游戏获利是一个比较难解决的问题。

# 2.核心概念与联系
首先，我将网络游戏分成两种类型——免费和收费。如下图所示：

1）免费游戏（Free-to-Play Game）: 是指玩家不需要支付任何费用就可以下载、安装、运行和玩游戏。最主要的好处是，玩家可以无限期地在线游玩，可以方便自己进行游戏个性化定制。但是缺点也很明显，玩家容易沉迷，无法投入真正有价值的游戏时间，并且很可能输光所有金币。

2）付费游戏（Paid-to-Play Game）: 是指玩家需要向游戏开发商购买游戏的特定的道具或服务才能完整地玩游戏。优点是能够玩到比免费游戏更多的内容，也可以多玩一些有价值的游戏内容。付费游戏的另一个优点是可以激励玩家去留存，并产生更多的游戏收入。但缺点也是有的，如果玩家付款不足，则会影响到游戏体验和游戏进度。另外，如果玩家过分依赖付费游戏的机制，则可能会失去游戏的吸引力。


其次，我将网络游戏分成两类——单机游戏和多人游戏，如下图所示：

1）单机游戏（Singleplayer Game）: 是指仅有一个玩家参与游戏。由于只能自己控制角色，所以只能看到角色的视角。单机游戏没有其他玩家的互动，只能靠自己一步步推进。比如，像《GTA5》、《魔兽世界》这样的游戏属于单机游戏；

2）多人游戏（Multiplayer Game）: 是指多个玩家可以一起参与游戏，可以看到同一时间内其他玩家的角色。多人游戏可以结合不同玩家的能力，共同探索相同的关卡。比如，像《坦克大战》、《梦幻西游》这样的网络游戏属于多人游戏。


第三，游戏中，资源有四种分类，分别是：金钱、技能、道具、经验。其中，金钱是最重要的一种资源，其次是技能，再者是道具，最后是经验。玩家可以通过资源获取来获得游戏中的奖励，也可以用于升级游戏中的各种技能和道具。因此，不同类型的资源所对应的任务也有所区别。

第四，网络游戏用户通常分为三种类型——新手、老手和专家，如下图所示：

1）新手：一般初级用户，对网络游戏了解甚少，对付费游戏比较抵触；

2）老手：一般高级用户，熟悉网络游戏规则，玩家积攒足够的游戏收入后，渴望更高级的游戏体验；

3）专家：是指玩家已经成为游戏的专家，他们具有丰富的知识和技能，对游戏充满激情，拥有完美的控制感。
