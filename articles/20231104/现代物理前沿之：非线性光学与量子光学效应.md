# 1.背景介绍


## 1.1 大背景介绍
非线性光学（Nonlinear Optics）和量子光学（Quantum Optics）是两个不同领域。其中非线性光学的目标是在传播过程中引入非线性干涉使传播路径发生扭曲、扩散和折射。在大多数情况下，这样的扭曲、扩散和折射会导致失真（deformation），而失真又可能会被用于其他目的，如物理学和医疗应用等。相比之下，量子光学则以其独特的特征——量子纠缠而闻名。这种交错的状态会造成极端的透镜效应，即电子间的相互作用产生离子间断结构，从而引起物质的粒子化。量子效应甚至可以在各种层次上施加影响，例如低维子宇（superconductivity）或超导体（supersymmetry）。量子效应是指量子态的特殊特征——本征态迁移到另一个本征态。因此，量子光学能提供对任意实验条件的全面、一致的测量结果。量子效应在自然界、工程科学、医疗保健、军事等多个领域都有重要的应用。

但是，由于量子效应的复杂性，无法直接用传统的摄影技术观察到量子效应。而现有的技术只是为了观察非线性光学效应，因此想要彻底理解量子效应需要更高级的技术。

 # 2.核心概念与联系
  ## 2.1 传播路径上的折射率与失真
  ### 2.1.1 折射率(Refractive Index)
  在传播途中，光在每一点会受到多个介质层的阻碍，这些介质层的阻力与角度和介质的粗糙度及折射率有关。
  如果某一点的折射率小于等于1，那么该点的光可以完全通过这个介质层，不会被阻挡；
  如果某一点的折射率大于1，那么该点的光就会被折射，折射出来的光会进入一个不同的介质层；
  如果某一点的折射率大于1并且介质还具有偏振性，那么该点的光就会被失焦。

  #### 理解折射率
  
  从左边开始看，光从红色的入射层射出来时，经过第一个红色介质层后，折射了出来，变蓝色；再经过绿色介质层，变成黄色；最后经过蓝色的观察层。
  
  根据光的守恒律，如果把红色、绿色、蓝色三种颜色的光同时加起来，那么最终的光强度应该等于它们各自的光强度之和。

  </div>
  
  
  
  因为每个介质层都能折射光子，所以光子的传播路径不止一条，会形成多条曲线，称为折射线。在每一条折射线上，光子的折射率是相同的。

  由图可知，当光子的波长长于折射率λ的一半时，它才会被双向传播，否则只能单向传播。
  
  对于平面介质而言，折射率定义为分子量与密度比，如下式所示：
  
  $$n = \sqrt{\frac{N}{\mathit{m}}}$$
  
  其中N为分子量，m为密度，n为折射率。
  
  ### 2.1.2 没有失真的传播
  当没有因介质而产生失真时，传播路径与折射率无关，它仅仅依赖于各个介质的折射率。例如，两根长度分别为d的直线天线，分别投射在两侧的物体上，则两根天线之间存在着一条直线，这条直线上的每一点都可以接收到传播方向的光。
  ```
  ●          ●
  |          |
  d--------∞●
             | 
             ▼
       h₀      h₁ 
  ```
  
  上述例子表示无失真的传播，折射率n并不参与传播路径的计算。
  
  ### 2.1.3 有失真的传播
  如果折射率较高的介质对光子进行了反射或者扩散，会导致失真。举例如下：
  ```
          ●
          |                         +-----------------------+
          |                        /|                       /|
          |         +--------------+   +-------------+    +-----------+
  ●       ↓↓↓     ↑↑↑          |   |            |    |           |  
  |        ∞●     |            r  |   |  r        t  |    |    y     |  
  |              ↓↓↓         n  |   |    k      v  |    |  ∩x      |  
  |             ∞●                  |   |            |    |           |  
  |                 ↓↓↓               |   |            |    |           |  
  |                d                   |   +------------+----+-----------+
                                           |                      ^
                                           +----------------------+|||
                                                                vv
                                                           mirrowed image
  ```
  
  假设光子的波长λ=632.8nm，入射角度θ=45°，入射方位为右斜角，相距90°，入射媒质的折射率为1.5。沿着入射方向的第一束光沿着一根直线传播，当抵达两束光的交汇处时，第二束光沿着曲线的方向传播。
  
  如上图所示，第一束光受到右边介质的1.5倍反射，因此失真了，就形成了一个镜像。第二束光沿着曲线的方向传播，再与第二个蓝色的方框相切，此时它重新与右边的介质层发生1.5倍的反射，也会形成镜像，但是角度会发生变化。
  
  因此，光子在传播过程中受到不同介质的折射率和非线性影响，可能造成折射线条变得尖锐、失真、偏离等现象。
  
  ### 2.1.4 混沌效应(Chaos Effect)
  混沌效应是量子光学的一个特征，它表现为随时间推进会发生诸如分裂、混合、折叠等不稳定现象。
  
  对传播过程中的量子态而言，它存在着很强的纠缠态特性，随着传播路径的变化，它的纠缠程度也会发生变化。例如，一条二极管上的反复载流会引起其纠缠模式的周期性分裂。
  
  混沌效应的出现取决于量子纠缠的随机性，也就是说，如果我们知道初始态和演化方程式，就可以预测随着时间的推移，系统的各种模式和行为都会发生变化。
  
  ## 2.2 量子纠缠
  ### 2.2.1 量子纠缠的基本原理
  量子纠缠指的是两个量子态之间存在着复杂的相互作用。
  
  量子纠缠的过程由以下三步构成：
  
  第一步：量子耦合

  二者是两个能量级别不同的粒子，它们通过共振过程生成一个带有能量的杂乱的量子态，该态同时包含两个粒子。
  
  第三步：非确定性漂移

  量子态中同时包含两个粒子，而且这两个粒子的轨道彼此不对称，这就导致了“混乱”的效果。
  
  第四步：反常相互作用

  相邻量子态之间能够产生一种正面的非线性相互作用。这种相互作用能够改变量子态的性质，并将它们转变为新的混合态。
  
  ### 2.2.2 量子纠缠的应用
  量子纠缠作为量子光学的一个重要分支，应用非常广泛。
  
  在超导材料领域，量子纠缠的研究有助于提升量子门性能，如量子辐射泵，量子纠缠材料。
  
  量子纠缠在微电子、高性能集成电路、信息通信、生物医学等领域都有广泛的应用。
  
  ### 2.2.3 量子纠缠在传播中的重要性
  通过量子纠缠，我们可以制备出具有超高热容性的高频电子管，通过电磁紧密结合的方式实现高频放大。
  
  另外，随着新型量子计算机的问世，我们可以利用量子纠缠的特性进行信息处理、计算。
  
  ## 2.3 非线性光学效应与量子光学效应
  ### 2.3.1 非线性光学效应
  非线性光学效应是指光在传播过程中发生的一种不可忽略的扭曲、失真和折射。
  
  常见的非线性光学效应包括：
  
  1. 偏振效应
   
   偏振效应是指光在非匀速传播过程中出现偏转。如鱼眼镜头、印刷上的一些字母书写方向的偏转、摩尔纹、振铃音、风景照片中的景物的摆动等。

  2. 倒置效应
   
   倒置效应是指物体对光的偏振作用，导致其镜像反射、旋转、投影后的图像有所差别。如雪松体中的光倒置、建筑物的倒影等。

   ### 2.3.2 量子光学效应
  量子光学效应指的是量子纠缠的一种特征，它在传播过程中起到无法估计的作用。
  
  一般来说，量子效应在不同的维度上都有着不同的作用。
  
  1. 二极管效应
   
   二极管的导通开关是量子光学效应的代表。
   
   二极管导通的时候，会释放一些精心设计的初态，然后在一定时间内迅速发生漫长的量子混合，形成一系列的子系统。这时候，子系统之间的相互作用产生离子间断结构，促使更多的高能级粒子聚集到一起。这是一种极端的情况，实际上是一种非凡的离子引起的现象。
   
   二极管导通时激发的态的大小决定了其粒子分数，而这个态的不同分数对应着不同的数值。
   
   
     <div align="center">
     </div>
   
  
  2. 氢原子云效应
   
   氢原子云在电离层内部形成，它通常呈现为微弱的光芒状，有着明显的色散。
   
   这一效应的原因在于氢原子与电子有一种强烈的吸引力。在正常的光谱范围内，氢原子能量约为-4.7eV，而电子的能量约为-3.6eV，两者差异相差近10倍。
   
   在电离层中，氢原子云的光滑形成了一种缺陷，使得微弱的电子无法穿透。这就导致了光芒的形成，这个效应被称作氢原子云效应。

  ## 2.4 非线性光学与量子光学效应的关系
  ### 2.4.1 非线性光学效应的产生方式
  
  非线性光学效应的产生主要有两种途径：
  
  1. 折射率
   
   光在传播途中，其折射率可以改变，导致其路径发生扭曲。

  2. 没有失真的传播
   
   如果没有因介质而产生失真，那么光的传播路径依赖于各个介质的折射率，不会因为任何失真而扭曲。
  
  ### 2.4.2 量子光学效应的产生方式
  
  1. 量子耦合
  
  2. 非确定性漂移
  
  3. 反常相互作用
  
  ### 2.4.3 量子光学效应如何影响非线性光学效应？
  
  除了量子纠缠导致的二极管效应和氢原子云效应外，量子光学还影响到其他维度上的非线性光学效应。
  
  在传播过程中，量子纠缠的存在会引入新的振荡、失真、偏振、混合效应，这些效应使得非线性光学效应发生扭曲、失真、变形等。
  
  除此之外，还有一些其他的相互作用也会影响非线性光学效应。比如，电子带隙、光带隙的交叉等。
  
  ### 2.4.4 量子光学效应对人类生活的影响
  
  虽然量子光学效应已得到广泛关注，但由于其复杂性和广泛的影响，直到最近才有相关的技术可供生产和使用。
  
  不过，依靠量子光学发展的各个领域，如航空航天、生物医学、材料、机器人、通信等，都是有潜在的好处的。
