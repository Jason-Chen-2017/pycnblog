# 1.背景介绍


对于一个企业来说，智能制造是其核心竞争力之一。相信随着信息技术的发展、智能化生产设备的不断普及、产业链上下游合作的深入，智能制造将会成为未来发展的主导力量。

但是，当前人工智能（AI）技术对智能制造的研发与应用存在着诸多短板。比如，传统的基于规则的生产模式依然占据主导地位；而在这一基础上建立起来的规则和流程仍然无法很好地应对需求的变化，导致效率低下、缺乏成本效益分析等问题。同时，由于制造的复杂性和不可预测性，智能制造还面临着从经济到社会的各方面的挑战。

因此，为了充分利用人工智能（AI）技术带来的巨大潜力，提升智能制造的整体效率、降低成本、提高质量、缩短响应时间，市场与政策双方都在积极探索如何打造更智能的工厂。

AI Mass，顾名思义就是用大数据、人工智能技术解决制造过程中的效率问题。它通过将复杂的制造过程从头到尾自动化，实现在线制造和可控的生产环境，真正实现“工厂聚焦”、“零工实现”。它的架构主要由三大模块构成：生产预测、生产规划、生产管理。如图1所示：

# 2.核心概念与联系
## 2.1 传统方法
制造企业往往需要按照一定的流程、标准和要求完成产品的制造，这种制造方式称为直通车制造或工具仓制造。直通车制造是指企业集中制造、装配、测试、安装所有部件后直接进行最终的产品投放。工具仓制造则是指企业将所有的制造设备、工具、材料、零件等都放在一个仓库内进行隔离，然后根据不同的工序进行流水线操作，以满足客户的个性化需求。

直通车制造存在两个弊端：

1.效率低下：由于工艺、流程和设备的限制，企业往往只能按照预先设计好的流程进行制造，无法实时调整需求，导致生产效率不高。

2.缺乏反馈机制：由于制程固定，无法及时获得制造过程中出现的问题并及时反馈给企业，导致生产出错率高且周期长。

工具仓制造存在两个弊端：

1.重复劳动：由于整个过程全部都是手动操作，生产效率很低，甚至可能因为人员疲劳、工作速度慢甚至造成人身伤害。

2.知识成本高：由于没有经验的知识盲点，很多工序需要重复的操作，知识消耗较多。

因此，目前制造企业采用的是混合制造的方法，结合直通车制造和工具仓制造的方式进行生产。

## 2.2 大模型
大模型的基本概念来源于统计学习和机器学习的研究。在大数据、机器学习和深度学习的驱动下，越来越多的人开始关注大数据的价值和应用。而大模型正是利用大数据提升生产效率、降低成本、优化产量和增加生命周期价值的一种有效方式。

大模型可以分为三个阶段：
- 首先，数据采集、清洗、标注、特征工程。大模型的训练数据一般是全量数据或增量数据，通过提取特征，能够帮助机器学习模型更准确地预测和分类数据。
- 其次，模型训练。大模型训练的目的不是去直接预测结果，而是通过对已知数据分布的假设建模，获取数据的概率模型。然后再根据实际情况对模型参数进行调整，使模型更符合实际。
- 最后，模型部署。大模型的部署通常是一个整体，涉及到对模型效果评估、异常检测、精细化控制、运维支持等环节，保证大模型能够快速、稳定、安全运行。

基于以上三个阶段，大模型被分为两类：
- 生成模型（Generative Model），通过监督学习、强化学习、遗传算法等算法对数据进行建模，能够生成类似真实数据的样本。
- 判别模型（Discriminative Model），通过无监督学习、分类算法等算法对数据进行建模，能够对样本进行分类，预测其所属的类别。

大模型在智能制造领域的应用也有两种形式。

第一种形式是生成式制造。生产线上的每个工序都是一个独立的生成模型，根据历史数据进行建模，能够生成符合制程的产品样本。这样就可以将每一个独立的产品样本拼装成最终的产品。

第二种形式是判别式制造。由于现有的产品配置方案无法很好地匹配新产品的需求，所以需要对产品进行定制，在之前的生产中学习借鉴，提升生产效率。而大模型就扮演了重要角色，辅助制造商明确自己想要的产品特色。
