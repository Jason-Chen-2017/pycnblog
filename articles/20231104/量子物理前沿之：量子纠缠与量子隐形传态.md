# 1.背景介绍


## 什么是量子纠缠？
量子纠缠(Quantum Entanglement)又称量子关联、量子耦合等，是指两个或多个量子体系相互作用而呈现出错综复杂的依赖关系的现象。而这些依赖关系反映了微观世界中真实存在的各种奇妙的现象，比如两颗子弹之间的弹性碰撞、电子与波粒二能级的相互作用、天体之间万有引力对撞等等，这些现象都是量子纠缠带来的结果。
## 为什么要研究量子纠缠？
随着近年来人工智能、机器学习、大数据等技术的飞速发展，科技水平的不断提高，生活环境的多样化、复杂化，人类也越来越需要解决多维度的问题。这些多维度的问题往往涉及到多个不同的变量之间复杂的依赖关系，而量子纠缠则是构建在古典力学基础上的一种抽象模型，可以描述和理解这些复杂的依赖关系。因此，研究者们围绕量子纠缠这一研究领域，开发和验证新的量子计算算法、方法、模型、技术。而量子纠缠所带来的便利性、普适性以及各种应用场景也正逐渐成为科学界共识。例如，它促进了计算机、通信、物理学、化学、生物学、医疗诊断等领域的突破性创新。
## 量子纠缠的分类
### 1.Classical-to-classical (CCC)纠缠
两类量子系统处于完全不同的时空环境中，它们之间没有任何手段可以直接通信。此种纠缠一般用相互作用很弱的核磁体作为媒介。CCC纠缠的两个系统都处于相互独立的状态，不依赖于相互作用，但通过交换信息也可以实现一些粗粒度的通信。目前来说，CCC纠缠的应用最广泛的领域就是量子通信。
### 2.Coherent-state exchange (CSE)纠缠
在相同的时间内，两个量子系统（一般为两个量子比特）通过光子交换态的方式实现了交流传输。其特点是只能进行极小规模的传输，且只允许局域范围的量子信息交换，且每一个传输的信息包只能被识别为特定单个比特的量子态，不能捕获整个系统的全局量子态。但是它的能量损失较低，因此可用于短距离的通信。
### 3.Superfluid-to-superfluid (SSF)纠缠
通常来说，在经过演化过程中，一种物质会发生融合，变成另一种物质。这种两个物质之间互相吸收或者产生耦合，使得它们之间更加紧密地结合，形成超纯的态，具有超导性。在这些超纯态之间，可能存在一种强大的能量张力，这种能量张力导致这两个超纯态之间能量的交换，从而形成了SSF纠缠。SSF纠缠的两个系统在同一个时空点上同时存在，彼此之间通过纠缠相互作用的方式相互作用。
