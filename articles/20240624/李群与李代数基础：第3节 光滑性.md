## 2. 核心概念与联系

### 2.1 李群的定义

李群（Lie Group）是一类特殊的群，它不仅满足群的公理，还具有连续性。具体来说，李群是一个集合 $G$，它满足以下条件：

1. **闭合性**：对于 $G$ 中的任意两个元素 $g_1$ 和 $g_2$，它们的乘积 $g_1 g_2$ 也在 $G$ 中。
2. **结合律**：对于 $G$ 中的任意三个元素 $g_1$、$g_2$ 和 $g_3$，都有 $(g_1 g_2) g_3 = g_1 (g_2 g_3)$。
3. **存在单位元**：存在一个元素 $e$ 在 $G$ 中，使得对于任意的 $g \in G$，都有 $g e = e g = g$。
4. **存在逆元**：对于 $G$ 中的任意元素 $g$，都存在一个元素 $g^{-1}$ 在 $G$ 中，使得 $g g^{-1} = g^{-1} g = e$。
5. **连续性**：对于 $G$ 中的任意两个元素 $g_1$ 和 $g_2$，它们的乘积函数 $g_1 g_2: G \rightarrow G$ 是连续的。

### 2.2 李代数的定义

李代数（Lie Algebra）是李群的一个代数结构，它由李群的元素导出。具体来说，李代数 $L$ 是由李群 $G$ 的所有元素导出的线性空间，并且满足以下条件：

1. **结合性**：对于 $L$ 中的任意三个元素 $[x, y, z]$，都有 $[x, y, z] = [y, z, x] + [x, z, y]$。
2. **雅可比恒等式**：对于 $L$ 中的任意三个元素 $x, y, z$，都有 $[x, [y, z]] + [y, [z, x]] + [z, [x, y]] = 0$。

### 2.3 李群和李代数的联系

李群和李代数之间存在密切的联系。具体来说，李群 $G$ 可以通过以下方式与李代数 $L$ 相关联：

1. **李括号**：对于李群 $G$ 中的任意两个元素 $x$ 和 $y$，它们的李括号 $[x, y] \in L$ 定义为 $[x, y] = x y^{-1} - y x^{-1}$。
2. **李括号的性质**：李括号满足结合性、雅可比恒等式和反交换性 $[x, y] = -[y, x]$。

## 3. 光滑性

### 3.1 光滑性的定义

李群的光滑性是指李群结构和其对应的李代数结构都是光滑的。具体来说，以下条件需要满足：

1. **李群的光滑性**：李群 $G$ 上的乘法、逆元和单位元运算都是光滑的。
2. **李代数的光滑性**：李代数 $L$ 上的李括号运算和数乘运算都是光滑的。

### 3.2 光滑性的性质

李群的光滑性具有以下性质：

1. **封闭性**：如果李群 $G$ 的子集 $H$ 是闭集，那么 $H$ 也是光滑子群。
2. **连通性**：如果李群 $G$ 是连通的，那么 $G$ 的任何光滑子群也是连通的。
3. **李代数的性质**：李代数 $L$ 的性质与李群 $G$ 的性质密切相关。例如，李代数 $L$ 的维度等于李群 $G$ 的维度。

### 3.3 光滑性的例子

以下是一些光滑李群的例子：

1. **实数加法群**：实数加法群 $\mathbb{R}$ 是一个光滑李群，其李代数也是实数加法群 $\mathbb{R}$。
2. **欧几里得空间中的旋转群**：三维欧几里得空间中的旋转群 $SO(3)$ 是一个光滑李群，其李代数是三维空间中的交叉积 $so(3)$。
3. **复数乘法群**：复数乘法群 $\mathbb{C}^*$ 是一个光滑李群，其李代数是复数乘法群的导代数 $\mathfrak{g}$。

```python
import numpy as np
import matplotlib.pyplot as plt

def rotation_matrix(theta, axis):
    """生成旋转矩阵"""
    axis = np.array(axis) / np.linalg.norm(axis)
    a = axis[0]
    b = axis[1]
    c = axis[2]
    t = np.sin(theta)
    c = 1 - np.cos(theta)
    return np.array([
        [a*a*c + b*b, a*b*t - c*c, a*c + b*t],
        [a*b*t + c*c, b*b*c - a*a, b*a - c*t],
        [a*c - b*t, b*t + c*a, c*c - a*a]
    ])

def plot_rotation(theta, axis):
    """绘制旋转矩阵对应的旋转效果"""
    fig = plt.figure()
    ax = fig.add_subplot(111, projection='3d')
    u = np.array([1, 0, 0])
    v = np.array([0, 1, 0])
    w = np.array([0, 0, 1])
    u_rot = rotation_matrix(theta, axis).dot(u)
    v_rot = rotation_matrix(theta, axis).dot(v)
    w_rot = rotation_matrix(theta, axis).dot(w)
    ax.quiver(0, 0, 0, u[0], u[1], u[2], color='r')
    ax.quiver(0, 0, 0, v[0], v[1], v[2], color='g')
    ax.quiver(0, 0, 0, w[0], w[1], w[2], color='b')
    ax.quiver(0, 0, 0, u_rot[0], u_rot[1], u_rot[2], color='r', alpha=0.5)
    ax.quiver(0, 0, 0, v_rot[0], v_rot[1], v_rot[2], color='g', alpha=0.5)
    ax.quiver(0, 0, 0, w_rot[0], w_rot[1], w_rot[2], color='b', alpha=0.5)
    plt.show()

# 绘制旋转效果
plot_rotation(np.pi/4, [1, 1, 1])
```
