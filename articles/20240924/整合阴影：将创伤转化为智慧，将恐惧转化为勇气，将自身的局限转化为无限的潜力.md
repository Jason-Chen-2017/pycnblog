#### 5.2. 源代码详细实现

以下是项目的核心代码实现：

```python
from flask import Flask, request, jsonify
from pymysql import connect
from queue import PriorityQueue
import json

app = Flask(__name__)

# 数据库连接配置
db_config = {
    'host': 'localhost',
    'user': 'root',
    'password': 'password',
    'database': 'task_scheduler'
}

# 创建数据库连接
def create_connection():
    conn = connect(**db_config)
    return conn

# 创建任务队列
def create_queue():
    return PriorityQueue()

# 添加任务到队列
def add_task(queue, task):
    queue.put(task)

# 执行任务队列
def execute_tasks(queue, conn):
    while not queue.empty():
        task = queue.get()
        execute_single_task(task, conn)

# 执行单个任务
def execute_single_task(task, conn):
    cursor = conn.cursor()
    cursor.execute(task['query'])
    conn.commit()
    cursor.close()

# 提供RESTful接口添加任务
@app.route('/tasks', methods=['POST'])
def add_task_api():
    data = request.get_json()
    queue = create_queue()
    add_task(queue, data)
    return jsonify({'status': 'success', 'message': 'Task added to queue.'})

# 提供RESTful接口获取任务列表
@app.route('/tasks', methods=['GET'])
def get_tasks_api():
    queue = create_queue()
    execute_tasks(queue, create_connection())
    tasks = list(queue.queue)
    return jsonify({'status': 'success', 'tasks': tasks})

if __name__ == '__main__':
    app.run(debug=True)
```

2. **任务队列**：使用`queue.PriorityQueue`创建一个优先级队列，用于存储和排序任务。
3. **任务添加**：提供一个`/tasks`接口，通过POST请求添加任务到队列中。添加的任务以字典形式存储，包含`query`字段，用于存储SQL查询语句。
4. **任务执行**：提供一个`/tasks`接口，通过GET请求获取任务队列，并依次执行队列中的任务。执行过程中，会将任务从队列中取出，并执行相应的SQL查询。
