#### 4.3 加密算法模型

加密算法模型用于保护数字版权信息，确保其在传输和存储过程中不被窃取或篡改。常见的加密算法包括对称加密和非对称加密。

**对称加密算法公式：**
$$
Encrypted\_Text = Encrypt(Key, Plain\_Text)
$$
$$
Decrypted\_Text = Decrypt(Key, Encrypted\_Text)
$$

其中，\( Encrypt \) 和 \( Decrypt \) 分别是对称加密和解密函数，\( Key \) 是加密密钥，\( Plain\_Text \) 是原始文本，\( Encrypted\_Text \) 是加密后的文本。

**非对称加密算法公式：**
$$
Encrypted\_Text = Encrypt\_Public(Key, Plain\_Text)
$$
$$
Decrypted\_Text = Decrypt\_Private(Key, Encrypted\_Text)
$$

其中，\( Encrypt\_Public \) 和 \( Decrypt\_Private \) 分别是非对称加密和解密函数，\( Key \) 是公钥和私钥对，\( Plain\_Text \) 是原始文本，\( Encrypted\_Text \) 是加密后的文本。

**举例说明：**
使用 RSA 算法进行非对称加密和解密：

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# 生成密钥对
key = RSA.generate(2048)
private_key = key.export_key()
public_key = key.publickey().export_key()

# 加密
cipher = PKCS1_OAEP.new(RSA.import_key(public_key))
encrypted = cipher.encrypt(b'版权保护是数字世界的核心挑战')

# 解密
cipher = PKCS1_OAEP.new(RSA.import_key(private_key))
decrypted = cipher.decrypt(encrypted)

print('原始文本:', decrypted.decode('utf-8'))
```

输出结果可能为：`版权保护是数字世界的核心挑战`
