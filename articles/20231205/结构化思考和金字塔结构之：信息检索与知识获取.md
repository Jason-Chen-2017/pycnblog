在这里，我们将提供一个具体的代码实例，以及详细的解释说明。

```python
import numpy as np
from scipy.stats import norm

# 定义数据
x = np.array([1, 2, 3, 4, 5])
mu = 3
sigma = 1

# 计算概率
prob = norm.pdf(x, mu, sigma)

# 打印结果
print(prob)
```

在这个代码实例中，我们使用了numpy和scipy库来计算概率。我们首先定义了数据，然后计算了概率，最后打印了结果。
