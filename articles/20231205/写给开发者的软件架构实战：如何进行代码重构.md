# 2.核心概念与联系

## 2.1 什么是代码重构
代码重构是一种改进现有代码结构和设计的技术，旨在提高代码的可读性、可维护性和可扩展性。通过代码重构，开发者可以更好地理解和维护代码，同时也可以提高代码的性能和质量。

## 2.2 重构的目的
重构的目的是提高代码的可读性、可维护性和可扩展性，以及提高代码的性能和质量。通过重构，开发者可以更好地理解和维护代码，同时也可以更好地发现和修复代码中的问题。

## 2.3 重构的类型
重构可以分为两类：语法重构和语义重构。语法重构是指对代码的语法结构进行改进，如修改变量名、函数名等。语义重构是指对代码的语义进行改进，如修改算法、数据结构等。
