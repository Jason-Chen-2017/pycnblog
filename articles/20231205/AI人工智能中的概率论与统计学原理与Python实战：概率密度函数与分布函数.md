## 3.1 概率论基础
### 3.1.1 概率的基本定义
概率是一个随机事件发生的可能性，通常用P表示。我们可以用以下公式来计算概率：

P(A) = n(A) / n(S)

其中，P(A)是事件A的概率，n(A)是事件A发生的方法数，n(S)是总方法数。

### 3.1.2 独立事件的概率
如果两个事件A和B是独立的，那么它们的发生不会影响彼此的发生。我们可以用以下公式来计算两个独立事件的概率：

P(A and B) = P(A) * P(B)

### 3.1.3 条件概率
条件概率是一个事件发生的可能性，给定另一个事件已经发生。我们可以用以下公式来计算条件概率：

P(A|B) = P(A and B) / P(B)

## 3.2 概率密度函数与分布函数
### 3.2.1 概率密度函数
概率密度函数是一个随机变量的概率分布的一种描述方法，它描述了随机变量在某个区间内的概率密度。我们可以用以下公式来计算概率密度函数：

f(x) = P(x <= X <= x + Δx) / Δx

### 3.2.2 分布函数
分布函数是一个随机变量的概率分布的另一种描述方法，它描述了随机变量在某个区间内的概率。我们可以用以下公式来计算分布函数：

F(x) = P(X <= x)

## 3.3 常见的概率分布
### 3.3.1 均匀分布
均匀分布是一种常见的概率分布，它的概率密度函数和分布函数如下：

f(x) = 1 / b - a
F(x) = (x - a) / (b - a)

指数分布是一种常见的概率分布，它的概率密度函数和分布函数如下：

def normal_probability_density_function(x, μ, σ):
