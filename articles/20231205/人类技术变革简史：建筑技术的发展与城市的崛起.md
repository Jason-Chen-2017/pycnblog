# 2.核心概念与联系

建筑技术的核心概念包括结构、材料、工艺和设计。结构是建筑物的基本组成部分，它决定了建筑物的稳定性和安全性。材料是建筑物的基本成分，它们决定了建筑物的性能和特性。工艺是建筑物的制造过程，它决定了建筑物的质量和效率。设计是建筑物的外观和内部空间的布局，它决定了建筑物的美观性和功能性。
