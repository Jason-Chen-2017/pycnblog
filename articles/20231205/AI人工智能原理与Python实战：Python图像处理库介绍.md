3. scikit-image：scikit-image是一个基于scikit-learn库的图像处理库，它提供了许多用于图像处理的功能，如滤波、边缘检测、图像合成等。scikit-image支持多种图像格式，如JPEG、PNG、GIF等。

4. matplotlib：matplotlib是一个用于Python的数据可视化库，它提供了许多用于绘制图像的功能。matplotlib支持多种图像格式，如JPEG、PNG、GIF等。
