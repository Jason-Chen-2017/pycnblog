# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来详细解释禅宗简化程序设计的具体实现。

例子：编写一个程序，输出1到100之间的所有奇数。

```python
def print_odd_numbers(n):
    for i in range(1, n + 1):
        if i % 2 != 0:
            print(i)

print_odd_numbers(100)
```

解释说明：

1. 我们首先定义了一个函数`print_odd_numbers`，它接受一个参数`n`，表示输出的最大数。
2. 我们使用`for`循环来遍历1到`n`之间的所有数。
3. 在循环中，我们使用`if`条件判断来检查当前数是否为奇数。如果当前数`i`除以2的余数不等于0，则表示`i`是奇数，我们就打印出当前数。
4. 最后，我们调用`print_odd_numbers`函数，并传入参数100，以输出1到100之间的所有奇数。
