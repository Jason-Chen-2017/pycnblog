我们可以使用以下代码来实现这个功能：

```python
import random

# 创建一个数据集合
data = [i for i in range(1000)]

# 将数据集合划分为5个部分
partitions = [data[i:i + 200] for i in range(0, len(data), 200)]

# 将5个部分存储在不同的节点上
nodes = ['node1', 'node2', 'node3', 'node4', 'node5']
for i, partition in enumerate(partitions):
    with open(f'{nodes[i]}/data.txt', 'w') as f:
        for item in partition:
            f.write(str(item) + '\n')
```

### 4.2 详细解释说明

1. 首先，我们创建一个数据集合，数据集合包含1000个元素。
2. 然后，我们将数据集合划分为5个部分，每个部分包含200个元素。
3. 最后，我们将这5个部分存储在不同的节点上，每个节点对应一个文件，文件名为`data.txt`。
