在上述代码中，我们使用了简洁、直接的算法的方式来实现程序的高效运行。我们首先使用 `for` 循环来实现简洁、直接的算法，然后使用 `printf` 函数来输出结果。

## 4.3 合理选择数据结构的代码实例

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node *next;
} Node;

int main() {
    Node *head = (Node *)malloc(sizeof(Node));
    Node *p = (Node *)malloc(sizeof(Node));
    head->data = 1;
    head->next = p;
    p->data = 2;
    p->next = NULL;
    printf("%d\n", head->data);
    printf("%d\n", p->data);
    free(head);
    free(p);
    return 0;
}
```

在上述代码中，我们使用了合理选择数据结构的方式来实现程序的高效运行。我们首先使用 `malloc` 函数来分配内存，然后使用链表来实现简洁、直接的算法，最后使用 `free` 函数来释放内存。
