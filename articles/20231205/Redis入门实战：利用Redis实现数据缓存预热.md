Redis（Remote Dictionary Server，远程字典服务器）是一个开源的高性能的key-value存储系统，它支持数据的持久化，可基于内存（in-memory）也可基于磁盘。Redis 提供多种语言的 API，包括：Ruby、Python、Java、C 和 C#，因此可以在任何语言中使用。

Redis 是一个使用 ANSI C 语言编写、遵循 BSD 协议的开源（GPL）、跨平台（Windows，Linux，Mac OS X，UNIX 等）的下键值存储数据库，它支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。

Redis 支持的数据类型包括字符串(string)、哈希(hash)、列表(list)、集合(sets)和有序集合(sorted sets)等。
