# 2.核心概念与联系

计算机图形学是一门研究计算机如何处理、生成和显示图形信息的科学。计算机图形学涉及到图形的表示、处理、存储、传输、显示等方面的问题。计算机图形学的主要内容包括：

1. 图形的表示：图形可以用点、线、面、体等几何形状来表示。图形的表示可以采用向量形式（如点、线、面、体的坐标）或者像素形式（如图像的像素值）。
2. 图形的处理：图形的处理包括图形的变换、投影、剪裁、合成等操作。图形的处理可以采用几何变换、矩阵变换、向量变换等方法。
3. 图形的存储：图形的存储可以采用内存存储、文件存储、网络存储等方式。图形的存储可以采用向量存储、像素存储、压缩存储等方法。
4. 图形的传输：图形的传输可以采用本地传输、网络传输、远程传输等方式。图形的传输可以采用向量传输、像素传输、压缩传输等方法。
5. 图形的显示：图形的显示可以采用显示器显示、打印机显示、投影显示等方式。图形的显示可以采用向量显示、像素显示、光栅显示等方法。

计算机图形学与动画的联系是，计算机图形学提供了动画的基础技术，动画是计算机图形学的一个重要应用领域。动画是一种以图像为基本单位，通过连续的图像播放来表现动态场景的艺术形式。动画可以采用2D或3D的方式进行表现。动画的主要内容包括：

1. 动画的表示：动画可以用图像、视频、动画片段等形式来表示。动画的表示可以采用帧序列、动画序列、动画片段等方式。
2. 动画的处理：动画的处理包括动画的播放、动画的控制、动画的编辑等操作。动画的处理可以采用播放器、控制器、编辑器等工具。
3. 动画的存储：动画的存储可以采用内存存储、文件存储、网络存储等方式。动画的存储可以采用帧存储、视频存储、动画存储等方法。
4. 动画的传输：动画的传输可以采用本地传输、网络传输、远程传输等方式。动画的传输可以采用帧传输、视频传输、动画传输等方法。
5. 动画的显示：动画的显示可以采用显示器显示、打印机显示、投影显示等方式。动画的显示可以采用帧显示、视频显示、动画显示等方法。
