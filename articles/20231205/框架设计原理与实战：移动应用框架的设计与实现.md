```python
import tkinter as tk

class MyButton(tk.Button):
    def __init__(self, master=None, **kwargs):
        super().__init__(master, **kwargs)
        self.configure(bg='blue', fg='white', relief='raised')

root = tk.Tk()
button = MyButton(root, text='Click me!', command=lambda: print('Button clicked!'))
button.pack()
root.mainloop()
```

### 4.2 数据存储的实例

以下是一个简单的数据存储实例：

```python
import sqlite3

def create_table(conn):
    cursor = conn.cursor()
    cursor.execute('''CREATE TABLE IF NOT EXISTS users
                      (id INTEGER PRIMARY KEY, name TEXT, age INTEGER)''')
    conn.commit()

def insert_data(conn, name, age):
    cursor = conn.cursor()
    cursor.execute('''INSERT INTO users (name, age) VALUES (?, ?)''', (name, age))
    conn.commit()

def query_data(conn):
    cursor = conn.cursor()
    cursor.execute('''SELECT * FROM users''')
    return cursor.fetchall()

conn = sqlite3.connect(':memory:')
create_table(conn)
insert_data(conn, 'Alice', 25)
insert_data(conn, 'Bob', 30)
data = query_data(conn)
print(data)
```

在这个实例中，我们使用SQLite数据库创建了一个用户表。我们定义了三个函数：`create_table`用于创建表，`insert_data`用于插入数据，`query_data`用于查询数据。然后，我们创建一个SQLite连接对象，并调用这些函数。最后，我们打印出查询结果。

### 4.3 网络请求的实例

以下是一个简单的网络请求实例：

```python
import requests

def get_data(url):
    response = requests.get(url)
    if response.status_code == 200:
        return response.json()
    else:
        return None

data = get_data('https://api.example.com/data')
print(data)
```

在这个实例中，我们使用`requests`库发送GET请求。我们定义了一个`get_data`函数，它接受一个URL参数，并发送GET请求。如果请求成功，我们解析响应体为JSON格式，并返回。否则，我们返回`None`。然后，我们调用`get_data`函数，并打印出结果。

以下是一个简单的定位实例：

```python
import geopy
from geopy.geocoders import Nominatim

def get_location():
    geolocator = Nominatim(user_agent="mobile")
    location = geolocator.geocode("New York")
    return location.latitude, location.longitude

latitude, longitude = get_location()
print(latitude, longitude)
```

在这个实例中，我们使用`geopy`库获取当前位置。我们定义了一个`get_location`函数，它使用Nominatim地理编码器获取当前位置的纬度和经度。然后，我们调用`get_location`函数，并打印出结果。
