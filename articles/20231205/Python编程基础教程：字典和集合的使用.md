## 1.5 字典和集合的具体代码实例

### 1.5.1 字典的代码实例

```python
# 创建一个字典
dict = {}

# 添加键值对
dict["key1"] = "value1"
dict["key2"] = "value2"

# 查找键值对
value = dict["key1"]
print(value)  # 输出：value1

# 删除键值对
del dict["key1"]

# 判断键是否存在
if "key1" in dict:
    print("键存在")
else:
    print("键不存在")
```

### 1.5.2 集合的代码实例

```python
# 创建一个集合
set = set()

# 添加元素
set.add("element1")
set.add("element2")

# 查找元素
element = set.find("element1")
print(element)  # 输出：element1

# 删除元素
set.remove("element1")

# 判断元素是否存在
if "element1" in set:
    print("元素存在")
else:
    print("元素不存在")
```
