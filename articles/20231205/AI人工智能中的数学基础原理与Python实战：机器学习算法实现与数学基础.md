# 2.核心概念与联系

在深入探讨机器学习算法之前，我们需要了解一些基本的概念和联系。以下是一些重要的术语及其定义：

- 数据集：一组包含多个样本的集合，每个样本都包含一组特征值和一个标签。
- 特征（Feature）：描述样本的属性，可以是数值、分类或者其他类型的数据。
- 标签（Label）：样本的预期输出，通常用于训练模型。
- 训练集：用于训练模型的数据子集。
- 测试集：用于评估模型性能的数据子集。
- 训练：通过对训练集进行迭代计算，更新模型参数以减少预测误差。
- 预测：使用训练好的模型对新数据进行预测。
- 损失函数（Loss Function）：用于衡量模型预测误差的函数。
- 梯度下降（Gradient Descent）：一种优化算法，用于最小化损失函数。
- 正则化（Regularization）：一种防止过拟合的方法，通过添加一个惩罚项到损失函数中。
