Python是一种流行的编程语言，它具有简洁的语法和强大的功能。在Python中，模块是代码的组织和重用的基本单元。模块可以包含函数、类、变量等，可以被其他模块导入和使用。在本文中，我们将讨论Python中的模块导入和定义的核心概念、算法原理、具体操作步骤以及数学模型公式。

# 2.核心概念与联系

## 2.1 模块的概念

在Python中，模块是一个包含Python代码的文件。模块可以包含函数、类、变量等，可以被其他模块导入和使用。模块的文件名后缀为.py。

## 2.2 模块的导入

模块导入是将一个模块的代码加载到当前的Python解释器中，使其可以被调用的过程。在Python中，使用import关键字进行模块导入。

## 2.3 模块的定义

模块的定义是将一段Python代码保存到一个文件中，使其可以被其他模块导入和使用的过程。在Python中，模块的定义通常包括函数、类、变量等。

# 4.具体代码实例和详细解释说明

## 4.1 模块导入的代码实例

```python
# 导入math模块
import math

# 使用math模块中的sqrt函数
print(math.sqrt(16))
```

## 4.2 模块定义的代码实例

```python
# 定义一个名为my_module的模块
# my_module.py

def add(x, y):
    return x + y

def sub(x, y):
    return x - y

# 在其他模块中导入和使用my_module
import my_module

# 使用my_module中的add和sub函数
print(my_module.add(1, 2))
print(my_module.sub(10, 5))
```
