# 4.具体代码实例和详细解释说明
在这里，我们将提供一个简单的代码实例，以帮助你更好地理解软件开发的过程。

假设我们要开发一个简单的计数器软件，用于计算用户输入的数字的总和。以下是代码实例：

```python
# 定义一个函数，用于计算用户输入的数字的总和
def calculate_sum(numbers):
    # 初始化一个变量，用于存储计算结果
    result = 0
    # 遍历所有数字
    for number in numbers:
        # 将数字加到结果中
        result += number
    # 返回结果
    return result

# 主函数
if __name__ == "__main__":
    # 创建一个列表，用于存储用户输入的数字
    numbers = []
    # 提示用户输入数字
    while True:
        # 获取用户输入的数字
        number = input("请输入一个数字（输入-1结束输入）：")
        # 将数字转换为整数
        number = int(number)
        # 如果数字为-1，则结束输入
        if number == -1:
            break
        # 将数字添加到列表中
        numbers.append(number)
    # 调用计算总和的函数
    result = calculate_sum(numbers)
    # 输出计算结果
    print("总和为：", result)
```

这个代码实例中，我们首先定义了一个`calculate_sum`函数，用于计算用户输入的数字的总和。然后，在主函数中，我们创建了一个列表来存储用户输入的数字，并使用一个`while`循环来获取用户输入的数字。当用户输入-1时，循环结束。最后，我们调用`calculate_sum`函数来计算总和，并输出结果。
