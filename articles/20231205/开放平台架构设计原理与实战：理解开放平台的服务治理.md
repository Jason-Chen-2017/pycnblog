```python
import time
import random

def sample():
    return random.random()

def statistic(data):
    avg = sum(data) / len(data)
    std = 0
    for i in range(len(data)):
        std += (data[i] - avg) ** 2
    std = std / len(data) ** 0.5
    return avg, std

def detect(data, threshold):
    avg, std = statistic(data)
    for i in range(len(data)):
        if data[i] > avg + threshold * std:
            return True
    return False

data = []
for i in range(100):
    data.append(sample())

if detect(data, 3):
    print("异常")
else:
    print("正常")
```

在这个代码实例中，我们首先定义了一个名为`sample`的函数，它用于生成随机数。然后，我们定义了一个名为`statistic`的函数，它用于计算出服务的平均值和标准差。最后，我们定义了一个名为`detect`的函数，它用于发现服务的异常情况。
