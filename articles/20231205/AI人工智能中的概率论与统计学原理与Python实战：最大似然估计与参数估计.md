## 4.1最大似然估计的Python代码实例
```python
import numpy as np

# 定义数据
x = np.array([1, 2, 3, 4, 5])

# 定义似然函数
def likelihood_function(theta, x):
    return np.sum(np.log(np.exp(theta * x)))

# 计算似然函数的极值
theta_hat = np.argmax(likelihood_function(theta, x))

# 输出结果
print("最大似然估计的参数估计为：", theta_hat)
```
在上述代码中，我们首先定义了数据`x`，然后定义了似然函数`likelihood_function`。接着，我们计算似然函数的极值，并输出结果。
