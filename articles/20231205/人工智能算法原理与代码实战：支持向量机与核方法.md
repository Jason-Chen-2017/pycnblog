# 2.核心概念与联系

## 2.1 支持向量机（Support Vector Machines，SVM）

支持向量机是一种用于分类和回归任务的算法，它的核心思想是通过在数据集中找到一个最佳的分隔超平面，将不同类别的数据点分开。支持向量机通过最大化间隔（Margin）来实现这一目标，间隔是指从最近的支持向量到分隔超平面的距离。支持向量是那些在分隔超平面附近的数据点，它们决定了分隔超平面的位置。

## 2.2 核方法（Kernel Methods）

核方法是一种用于处理高维数据的技术，它可以将数据映射到更高维的空间，以便更好地进行分类和回归。核函数（Kernel Function）是核方法的核心组成部分，它用于计算数据点之间的相似度。常见的核函数包括线性核、多项式核、高斯核等。通过使用核方法，我们可以将原始数据映射到更高维的空间，从而使得原本难以分类或回归的数据变得更加容易处理。

## 2.3 支持向量机与核方法的联系

支持向量机和核方法密切相关。支持向量机可以通过使用核方法来处理高维数据。通过将原始数据映射到更高维的空间，支持向量机可以更好地进行分类和回归任务。在实际应用中，我们通常会将支持向量机与不同类型的核函数结合使用，以便更好地处理数据。
