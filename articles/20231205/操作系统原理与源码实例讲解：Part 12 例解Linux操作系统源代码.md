# 2.核心概念与联系
操作系统的核心概念包括进程、线程、内存管理、文件系统、同步与互斥、调度策略等。这些概念是操作系统的基本组成部分，它们之间有密切的联系和关系。

进程是操作系统中的一个实体，它表示一个正在运行的程序的实例。进程有自己的资源、状态和生命周期。线程是进程的一个子集，它是进程内的一个执行单元。线程共享进程的资源，但具有独立的执行流程。内存管理负责分配和回收内存资源，确保程序的正确运行。文件系统用于存储和管理文件和目录，提供了持久化的数据存储服务。同步与互斥是操作系统中的一种资源访问控制机制，用于确保多个进程或线程之间的正确性和安全性。调度策略是操作系统中的一个重要组成部分，它决定了操作系统如何调度和分配资源。
