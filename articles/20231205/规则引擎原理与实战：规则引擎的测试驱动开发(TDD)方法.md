## 4.1 代码实例

以下是一个简单的规则引擎的代码实例：

```python
class RuleEngine:
    def __init__(self):
        self.rules = []

    def add_rule(self, condition, action):
        self.rules.append((condition, action))

    def execute(self, data):
        for condition, action in self.rules:
            if condition(data):
                action(data)

# 定义一个规则
def rule1(data):
    return data > 10

def rule2(data):
    return data < 5

# 创建一个规则引擎
engine = RuleEngine()

# 添加规则
engine.add_rule(rule1, lambda data: print(f"data is greater than 10: {data}"))
engine.add_rule(rule2, lambda data: print(f"data is less than 5: {data}"))

# 执行规则
engine.execute(15)
```

在这个代码实例中，我们创建了一个`RuleEngine`类，它可以用来定义、添加和执行规则。我们还定义了两个规则`rule1`和`rule2`，并将它们添加到规则引擎中。最后，我们执行了规则引擎，并输出了决策结果。
