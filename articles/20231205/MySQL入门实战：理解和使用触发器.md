# 4.具体代码实例和详细解释说明

以下是一个具体的MySQL触发器实例：

```sql
CREATE TRIGGER check_age
    BEFORE INSERT
    ON users
    FOR EACH row
    BEGIN
        IF NEW.age < 18 THEN
            SIGNAL SQLSTATE '45000'
            SET MESSAGE_TEXT = '用户年龄必须大于18岁';
        END IF;
    END;
```

在这个例子中，我们创建了一个名为check_age的触发器，它在INSERT事件发生时触发。触发器的代码检查用户的年龄是否大于18岁，如果不满足条件，则抛出一个错误信息。
