# 2.核心概念与联系

## 2.1大模型即服务（Model-as-a-Service，MaaS）

大模型即服务是一种基于云计算的服务模式，它允许用户通过网络访问和使用大型预训练模型，从而实现更高效、更便捷的人工智能应用开发。大模型即服务的核心优势在于它可以让用户无需自己训练大型模型，而是可以直接使用云端提供的预训练模型，从而大大降低了模型训练和维护的成本和复杂度。

6. 应用模型：将训练好的模型应用于实际的图像处理任务，并实现对图像的处理和分析。
