## 3.1 卡方检验
卡方检验是一种用于检验两个或多个分类变量之间是否存在相关关系的统计方法。卡方检验的基本思想是将两个或多个分类变量之间的观测值分类，然后计算每个类别的观测值与预期值之间的差异，并将这些差异的平方相加，得到一个卡方统计量。如果卡方统计量较大，则说明两个或多个分类变量之间存在相关关系。

卡方检验的公式为：

$$
X^2 = \sum_{i=1}^{r} \sum_{j=1}^{c} \frac{(O_{ij} - E_{ij})^2}{E_{ij}}
$$

其中，$X^2$ 是卡方统计量，$r$ 是分类变量的个数，$c$ 是每个分类变量的个数，$O_{ij}$ 是观测值，$E_{ij}$ 是预期值。

## 3.2 独立性检验
独立性检验是一种用于检验两个或多个变量之间是否存在相关关系的统计方法。独立性检验的基本思想是将两个或多个变量之间的观测值分类，然后计算每个类别的观测值与预期值之间的差异，并将这些差异的平方相加，得到一个独立性统计量。如果独立性统计量较大，则说明两个或多个变量之间存在相关关系。

独立性检验的公式为：

$$
X^2 = \sum_{i=1}^{r} \sum_{j=1}^{c} \frac{(O_{ij} - E_{ij})^2}{E_{ij}}
$$

其中，$X^2$ 是独立性统计量，$r$ 是分类变量的个数，$c$ 是每个分类变量的个数，$O_{ij}$ 是观测值，$E_{ij}$ 是预期值。

# 4.具体代码实例和详细解释说明
在这一部分，我们将通过一个具体的代码实例来演示如何使用Python进行卡方检验和独立性检验。

```python
import numpy as np
import scipy.stats as stats

# 创建一个2x2的混淆矩阵
data = np.array([[10, 20], [30, 40]])

# 计算卡方统计量
chi2, p, dof, expected = stats.chi2_contingency(data)
print("卡方统计量:", chi2)
print("p值:", p)
print("自由度:", dof)
print("预期值:", expected)

# 计算独立性统计量
independence, p, dof, expected = stats.chi2_contingency(data)
print("独立性统计量:", independence)
print("p值:", p)
print("自由度:", dof)
print("预期值:", expected)
```

在这个代码实例中，我们首先导入了numpy和scipy.stats库。然后，我们创建了一个2x2的混淆矩阵，并使用scipy.stats库中的chi2_contingency函数来计算卡方统计量和独立性统计量。最后，我们打印出了计算结果。
