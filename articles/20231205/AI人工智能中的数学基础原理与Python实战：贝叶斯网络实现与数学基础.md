首先，我们需要确定贝叶斯网络中的随机变量。在这个例子中，我们将有三个随机变量：$A$、$B$ 和 $C$。

接下来，我们需要确定随机变量之间的条件依赖关系。在这个例子中，我们将有以下条件依赖关系：

- $A$ 和 $B$ 之间有条件依赖关系
- $B$ 和 $C$ 之间有条件依赖关系

最后，我们需要确定每个随机变量的概率分布。在这个例子中，我们将有以下概率分布：

- $P(A) = 0.5$

现在，我们可以使用Python的`networkx`库来构建贝叶斯网络：

```python
import networkx as nx

# 创建贝叶斯网络
G = nx.DiGraph()

# 添加随机变量
G.add_nodes_from(['A', 'B', 'C'])

# 添加条件依赖关系
G.add_edges_from([('A', 'B'), ('B', 'C')])

# 添加概率分布
G['A']['probability'] = 0.5
G['B']['probability'] = 0.8
G['C']['probability'] = 0.7
```

接下来，我们可以使用Python的`numpy`库来计算给定条件下每个随机变量的概率分布。在这个例子中，我们将给定条件是$A = 1$：

```python
import numpy as np

# 计算给定条件下每个随机变量的概率分布
P_A_given_B = np.array([0.5, 0.8])
P_B_given_A = np.array([0.5, 0.8])
P_C_given_B = np.array([0.5, 0.7])

# 使用贝叶斯定理计算给定条件下每个随机变量的概率分布
P_B_given_A_1 = P_B_given_A[0] / P_A_given_B[0]
P_C_given_B_1 = P_C_given_B[0] / P_B_given_A_1
```

在这个例子中，我们可以看到给定条件是$A = 1$下，$B$ 的概率分布是$P(B|A=1) = 0.5$，$C$ 的概率分布是$P(C|B=1) = 0.7$。
