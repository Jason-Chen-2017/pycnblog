```python
            "time": data["time"],
            "power": data["power"]
        })
    return processed_data

processed_solar_data = process_solar_data(solar_data)
```
在这个代码中，我们定义了一个`process_solar_data`函数，它接收太阳能板的数据，并对数据进行处理。然后，我们将处理后的数据存储在`processed_data`列表中。
