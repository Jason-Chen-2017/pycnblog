# 创建一个线性方程组的矩阵

3. 然后，我们创建了一个线性方程组的向量 b，它是一个 2x1 向量。
4. 接下来，我们使用 numpy 库的 linalg.solve 函数来求解线性方程组的解。
5. 最后，我们输出解得的向量 x。
