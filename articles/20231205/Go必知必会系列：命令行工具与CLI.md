# 4.具体代码实例和详细解释说明

以下是一个 Go 语言的命令行工具示例：

```go
package main

import (
	"flag"
	"fmt"
	"os"
)

func main() {
	// 定义命令行参数
	flag.String("name", "world", "name to greet")
	flag.Parse()

	// 读取命令行参数
	name := flag.Arg(0)

	// 格式化输出
	fmt.Printf("Hello, %s\n", name)
}
```

在这个示例中，我们使用 `flag` 库来处理命令行参数，使用 `fmt` 库来格式化输出。首先，我们定义了一个名为 `name` 的命令行参数，并设置了默认值为 `"world"`。然后，我们使用 `flag.Parse` 方法来解析命令行参数。最后，我们使用 `fmt.Printf` 方法来格式化输出。
