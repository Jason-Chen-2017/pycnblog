                 

# 1.背景介绍

制造业是现代社会的核心产业，其在经济发展中的作用不可或缺。随着科技的不断发展，制造业也在不断发展和进步。在这个过程中，人工智能（AI）技术的应用也逐渐成为制造业的重要组成部分。
