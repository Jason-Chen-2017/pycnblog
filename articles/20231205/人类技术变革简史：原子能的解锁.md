# 2.核心概念与联系

原子能是指原子核内的能量，它是原子核的核心组成部分。原子核由一些子atomic nucleus组成，这些子atomic nucleus包含着原子核的所有能量和质量。原子核的能量来源于原子核内部的强力和弱力。强力是原子核内部的强大吸引力，它使得原子核紧密结合在一起。弱力是原子核内部的弱吸引力，它使得原子核之间相互作用。
