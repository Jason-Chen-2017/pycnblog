在本节中，我们将通过一个简单的排序算法——冒泡排序来展示禅宗哲学在计算机程序设计中的应用。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(arr))
```

在上述代码中，我们首先定义了一个冒泡排序函数`bubble_sort`，该函数接收一个数组作为输入，并返回排序后的数组。然后，我们创建了一个数组`arr`，并将其传递给`bubble_sort`函数，最后打印出排序后的数组。
