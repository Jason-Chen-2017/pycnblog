```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

选择排序的核心思想是在每次循环中找到最小的元素，并将其放到当前位置。我们可以看到，选择排序的实现方式非常简单，只需要两层循环即可。

在第一层循环中，我们从第一个元素开始，逐个遍历整个数组。在第二层循环中，我们从当前元素后面开始，找到最小的元素，并将其与当前元素交换。这样，每次循环后，整个数组的前n个元素就都是有序的。
