Redis是一个开源的高性能的key-value存储系统，它支持数据的持久化，可基于内存（in-memory）也可基于磁盘（Persistent）。Redis 提供多种语言的API，包括：Ruby、Python、Java、C、C++、PHP、Node.js、Go、C#、Objective-C等。Redis 支持通过TCP/IP协议进行网络访问，也可以使用Redis Cluster实现分布式集群。

Redis的核心特性有：数据结构的多样性、原子性操作、高性能、数据持久化、集群支持、可扩展性、高可用性等。Redis的数据结构包括：字符串(String)、哈希(Hash)、列表(List)、集合(Set)、有序集合(Sorted Set)、位图(BitMap)、 hyperloglog 等。

Redis的数据结构都支持原子性操作，即一次性完成，不会被打断。Redis的高性能是由于它的内存存储和非阻塞I/O模型的优势。Redis的数据持久化可以通过RDB（Redis Database）和AOF（Append Only File）两种方式实现。Redis Cluster是Redis的分布式集群解决方案，可以实现数据的分片和故障转移。Redis的可扩展性是指可以通过增加内存和CPU来提高Redis的性能。Redis的高可用性是指可以通过主从复制和哨兵（Sentinel）机制来实现Redis的高可用。
