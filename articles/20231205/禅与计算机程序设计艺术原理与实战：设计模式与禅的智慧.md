1. 单例模式的代码实例：

```python
class Singleton:
    _instance = None

    @classmethod
    def get_instance(cls):
        if cls._instance is None:
            cls._instance = cls()
        return cls._instance
```

在上述代码中，我们定义了一个单例类`Singleton`，通过`_instance`变量来保存类的唯一实例。通过`get_instance`方法，我们可以获取该实例。通过这种方式，我们可以确保在整个程序中只有一个`Singleton`实例，从而实现内存管理和性能优化。

2. 观察者模式的代码实例：

```python
class Observer:
    def update(self, subject):
        pass

class Subject:
    _observers = []

    def attach(self, observer):
        self._observers.append(observer)

    def detach(self, observer):
        self._observers.remove(observer)

    def notify(self):
        for observer in self._observers:
            observer.update(self)
```

在上述代码中，我们定义了一个观察者模式，包括`Observer`类和`Subject`类。`Observer`类表示观察者对象，通过`update`方法来更新自身的状态。`Subject`类表示主题对象，通过`attach`方法来添加观察者，通过`detach`方法来删除观察者，通过`notify`方法来通知所有观察者更新自身的状态。通过这种方式，我们可以实现数据同步和通信。
