# 2.核心概念与联系

- 标注数据（Labeled Data）：这是已经被人工标注的数据，用于训练模型。
- 未标注数据（Unlabeled Data）：这是未被人工标注的数据，用于帮助模型学习。
- 半监督学习模型（Semi-Supervised Learning Model）：这是一个可以利用标注数据和未标注数据进行训练的模型。

- 标注数据和未标注数据之间的相似性：半监督学习模型通过利用标注数据和未标注数据的相似性，从而提高模型的泛化能力。
