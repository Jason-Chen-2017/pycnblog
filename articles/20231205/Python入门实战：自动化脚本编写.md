## 4.1 文件操作示例
以下是一个简单的文件操作示例，用于读取一个文本文件并输出其内容：

```python
# 打开文件
file = open("example.txt", "r")

# 读取文件内容
content = file.read()

# 关闭文件
file.close()

# 输出文件内容
print(content)
```

解释说明：

1. 使用`open`函数打开文件，第一个参数是文件名，第二个参数是文件模式（"r"表示读取模式）。
2. 使用`read`函数读取文件内容，并将内容存储到`content`变量中。
3. 使用`close`函数关闭文件，释放系统资源。
4. 使用`print`函数输出文件内容。

## 4.2 数据处理示例
以下是一个简单的数据处理示例，用于读取一个CSV文件，将其内容转换为字典列表：

```python
import csv

# 打开文件
file = open("example.csv", "r")

# 创建CSV读取器
reader = csv.DictReader(file)

# 读取文件内容
data = [row for row in reader]

# 关闭文件
file.close()

# 输出文件内容
print(data)
```

解释说明：

1. 使用`import csv`语句导入CSV模块。
2. 使用`open`函数打开CSV文件，第一个参数是文件名，第二个参数是文件模式（"r"表示读取模式）。
3. 使用`csv.DictReader`函数创建CSV读取器，将文件内容转换为字典列表。
4. 使用`for`循环读取文件内容，将内容存储到`data`变量中。
5. 使用`close`函数关闭文件，释放系统资源。
6. 使用`print`函数输出文件内容。

## 4.3 网络爬虫示例
以下是一个简单的网络爬虫示例，用于抓取一个网页的内容并提取其链接：

```python
import requests
from bs4 import BeautifulSoup

# 发送请求
response = requests.get("https://www.example.com")

# 解析响应内容
soup = BeautifulSoup(response.text, "html.parser")

# 提取链接
links = soup.find_all("a")

# 输出链接
for link in links:
    print(link["href"])
```

解释说明：

1. 使用`import requests`语句导入requests模块。
2. 使用`import BeautifulSoup`语句导入BeautifulSoup模块。
3. 使用`requests.get`函数发送请求，将网页内容存储到`response`变量中。
4. 使用`BeautifulSoup`函数创建BeautifulSoup对象，将响应内容解析为HTML文档。
5. 使用`find_all`函数提取所有的链接元素。
6. 使用`for`循环输出链接。
