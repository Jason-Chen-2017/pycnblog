## 4.1 代码实例

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

arr = [64, 34, 25, 12, 22, 11, 90]
bubble_sort(arr)
print(arr)
```

1. 我们首先定义了一个名为 `bubble_sort` 的函数，它接受一个列表作为参数。
2. 我们使用了一个 `for` 循环来遍历列表的每个元素。
3. 在每个元素的前面，我们使用了另一个 `for` 循环来遍历列表的每个元素。
4. 我们使用了一个 `if` 语句来比较当前元素和下一个元素的大小。
5. 如果当前元素大于下一个元素，我们使用了元组赋值来交换它们的位置。
