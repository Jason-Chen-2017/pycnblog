## 4.具体代码实例和详细解释说明

### 4.1 数据库查询代码实例

```sql
-- 查询用户表中年龄大于30的用户
SELECT * FROM user WHERE age > 30;
```

### 4.2 数据库查询代码解释

1. SELECT * 是查询所有列。
2. FROM user 是查询用户表。
3. WHERE age > 30 是查询条件，年龄大于30。

### 4.3 数据库索引代码实例

```sql
-- 创建用户表的主键索引
CREATE INDEX idx_user_id ON user (id);

-- 删除用户表的主键索引
DROP INDEX idx_user_id ON user;
```

### 4.4 数据库索引代码解释

1. CREATE INDEX 是创建索引的语句。
2. idx_user_id 是索引名。
3. ON user 是查询表。
4. (id) 是查询列。
5. DROP INDEX 是删除索引的语句。

### 4.5 数据库排序代码实例

```sql
-- 查询用户表中年龄降序排列的用户
SELECT * FROM user ORDER BY age DESC;
```

### 4.6 数据库排序代码解释

1. SELECT * 是查询所有列。
2. FROM user 是查询用户表。
3. ORDER BY age DESC 是排序规则，年龄降序。
