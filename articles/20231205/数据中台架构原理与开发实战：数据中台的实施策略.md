                 

# 1.背景介绍

数据中台是一种架构，它的目的是为企业提供一种集成、统一管理、分享和可扩展的数据资源。数据中台可以帮助企业更好地管理数据，提高数据的可用性和可靠性，降低数据的成本。数据中台的核心是数据资源的集成、统一管理、分享和可扩展性。
