# 2.核心概念与联系

## 2.1 条件语句

条件语句是一种用于根据某个条件执行或跳过代码块的控制结构。在Go语言中，条件语句使用`if`关键字来表示。条件语句的基本格式如下：

```go
if 条件表达式 {
    // 执行的代码块
}
```

条件表达式可以是布尔值、数值类型或字符串类型。如果条件表达式的值为`true`，则执行代码块；否则，跳过代码块。

## 2.2 循环语句

循环语句是一种用于重复执行某段代码的控制结构。在Go语言中，循环语句使用`for`关键字来表示。循环语句的基本格式如下：

```go
for 初始化; 条件表达式; 更新 {
    // 循环体
}
```

初始化部分用于在循环开始时执行一次的初始化操作，条件表达式用于判断是否继续循环，更新部分用于在每次循环结束后执行的更新操作。

# 4.具体代码实例和详细解释说明

## 4.1 条件语句的实例

```go
package main

import "fmt"

func main() {
    age := 18
    if age >= 18 {
        fmt.Println("年龄大于等于18")
    } else {
        fmt.Println("年龄小于18")
    }
}
```

在这个实例中，我们定义了一个变量`age`，并使用条件语句判断其值是否大于等于18。如果`age`的值大于等于18，则输出"年龄大于等于18"；否则，输出"年龄小于18"。

## 4.2 循环语句的实例

```go
package main

import "fmt"

func main() {
    for i := 1; i <= 10; i++ {
        fmt.Println(i)
    }
}
```

在这个实例中，我们使用`for`循环遍历从1到10的数字，并在每次迭代时输出当前数字。

# 6.附录常见问题与解答

Q: 条件语句和循环语句有什么区别？

A: 条件语句用于根据某个条件执行或跳过代码块，而循环语句用于重复执行某段代码。条件语句使用`if`关键字，循环语句使用`for`关键字。

Q: 如何判断一个数是否为偶数？

A: 可以使用条件语句来判断一个数是否为偶数。例如，在Go语言中，可以使用以下代码来判断一个数是否为偶数：

```go
package main

import "fmt"

func main() {
    num := 10
    if num % 2 == 0 {
        fmt.Println("数是偶数")
    } else {
        fmt.Println("数是奇数")
    }
}
```

在这个实例中，我们定义了一个变量`num`，并使用条件语句判断其值是否为偶数。如果`num`的值为偶数，则输出"数是偶数"；否则，输出"数是奇数"。

Q: 如何实现一个简单的计数器？

A: 可以使用循环语句来实现一个简单的计数器。例如，在Go语言中，可以使用以下代码来实现一个简单的计数器：

```go
package main

import "fmt"

func main() {
    count := 0
    for i := 1; i <= 10; i++ {
        count++
        fmt.Println(count)
    }
}
```

在这个实例中，我们使用`for`循环遍历从1到10的数字，并在每次迭代时将`count`变量增加1。最终，`count`变量的值将为10。
