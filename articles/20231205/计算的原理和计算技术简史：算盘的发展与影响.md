# 4.具体代码实例和详细解释说明

以下是一个简单的算盘代码实例：

```python
class Calculator:
    def __init__(self):
        self.memory = []

    def input_number(self, number):
        self.memory.append(number)

    def input_operator(self, operator):
        self.memory.append(operator)

    def calculate(self):
        result = self.memory[0]
        for i in range(1, len(self.memory), 2):
            operator = self.memory[i]
            number = self.memory[i+1]
            if operator == '+':
                result += number
            elif operator == '-':
                result -= number
            elif operator == '*':
                result *= number
            elif operator == '/':
                result /= number
        return result

calculator = Calculator()
calculator.input_number(5)
calculator.input_operator('+')
calculator.input_number(3)
result = calculator.calculate()
print(result)  # 输出: 8
```

在这个代码实例中，我们定义了一个 `Calculator` 类，它有一个 `memory` 属性用于存储数字和运算符。我们通过调用 `input_number` 和 `input_operator` 方法来输入数字和运算符，然后调用 `calculate` 方法来进行计算。最后，我们输出计算结果。
