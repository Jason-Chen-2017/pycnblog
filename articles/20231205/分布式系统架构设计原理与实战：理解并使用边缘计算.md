# 2.核心概念与联系

在分布式系统中，我们需要关注以下几个核心概念：

1.分布式一致性：分布式系统中的多个节点需要保持一致性，即在任何情况下，所有节点都应该达成共识。

2.分布式事务：分布式系统中的事务需要跨多个节点进行处理，以确保数据的一致性和完整性。

3.分布式存储：分布式系统需要使用分布式存储来存储和管理数据，以实现高可用性和高性能。

4.边缘计算：边缘计算是一种新兴的计算模型，它将计算能力推向了边缘设备，以实现更高效的计算和更低的延迟。

在边缘计算中，我们需要关注以下几个核心概念：

1.边缘设备：边缘设备是具有计算能力的设备，如智能手机、IoT设备等。

2.边缘网络：边缘网络是边缘设备之间的网络，用于实现设备之间的通信和协同工作。

3.边缘计算模型：边缘计算模型是一种新的计算模型，它将计算能力推向了边缘设备，以实现更高效的计算和更低的延迟。
