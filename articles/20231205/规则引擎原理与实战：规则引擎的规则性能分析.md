### 1.6.1 规则引擎的代码实例

在这个部分，我们将提供一个简单的规则引擎的代码实例，用于演示规则引擎的基本功能。

```python
class RuleEngine:
    def __init__(self):
        self.rules = []
        self.facts = []

    def add_rule(self, rule):
        self.rules.append(rule)

    def add_fact(self, fact):
        self.facts.append(fact)

    def match(self, condition):
        for rule in self.rules:
            if rule.condition(condition):
                return rule
        return None

    def execute(self, action):
        rule = self.match(condition)
        if rule:
            rule.execute(action)

    def infer(self, condition):
        for rule in self.rules:
            if rule.infer(condition):
                return rule
        return None
```

- `match` 方法用于根据条件匹配规则。
