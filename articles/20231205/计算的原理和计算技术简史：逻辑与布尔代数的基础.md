### 1.3.1 布尔代数基本定理

布尔代数基本定理是布尔代数的核心概念，它有以下几个基本定理：

1. 交换律：a ∧ b = b ∧ a，a ∨ b = b ∨ a
2. 结合律：(a ∧ b) ∧ c = a ∧ (b ∧ c)，(a ∨ b) ∨ c = a ∨ (b ∨ c)
3. 分配律：a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)，a ∨ (b ∧ c) = (a ∨ b) ∧ (a ∨ c)
4. 对偶定理：a ∨ b = 非 a ∧ 非 b，a ∧ b = 非 a ∨ 非 b

### 1.3.2 布尔代数运算规则

布尔代数运算规则是布尔代数的核心概念，它有以下几个运算规则：

   | a | b | a ∧ b | a ∨ b | 非 a | 非 b | a → b | a ← b |

   | F | T |   F  |   F  |  T  |  F  |  T  |   F  |
   | F | F |   F  |   F  |  T  |  T  |  T  |   F  |

   其中，T 表示真，F 表示假。

2. 逻辑运算符：逻辑运算符是用来表示逻辑关系的符号，它有以下几个逻辑运算符：

   - 与（∧）：与运算符用来表示两个逻辑值之间的与关系。

   - 非（¬）：非运算符用来表示逻辑值的否定。
   - 箭头（→）：箭头运算符用来表示条件关系。
   - 左箭头（←）：左箭头运算符用来表示逆条件关系。

3. 逻辑表达式：逻辑表达式是用来表示逻辑关系的表达式，它可以使用逻辑运算符和逻辑值来构建。逻辑表达式的格式如下：

   - 复合逻辑表达式：复合逻辑表达式是由简单逻辑表达式和逻辑运算符组成的逻辑表达式，例如 a ∧ b、a ∨ b、a → b、a ← b 等。
