### 3.1 线性规划

线性规划是一种用于解决最优化问题的数学模型。线性规划问题可以用标准形式表示为：

$$
\begin{aligned}
\text{最小化/最大化} & \quad c^Tx \\
\text{subject to} & \quad Ax \leq b \\
& \quad x \geq 0
\end{aligned}
$$

其中，$c$是决策变量$x$的权重向量，$A$是决策变量$x$的约束矩阵，$b$是约束向量。

线性规划问题的解可以通过简单的算法，如基础推进法、双简化法等，得到。

### 3.2 约束优化

约束优化是一种用于解决约束条件下的最优化问题的数学模型。约束优化问题可以用如下形式表示：

$$
\begin{aligned}
\text{最小化/最大化} & \quad f(x) \\
\text{subject to} & \quad g(x) \leq 0 \\
& \quad h(x) = 0
\end{aligned}
$$

其中，$f(x)$是决策变量$x$的目标函数，$g(x)$是决策变量$x$的约束函数，$h(x)$是决策变量$x$的等式约束函数。

约束优化问题的解可以通过复杂的算法，如拉格朗日乘子法、内点法等，得到。

### 3.3 贝叶斯网络

贝叶斯网络是一种用于描述和推理概率关系的图模型。贝叶斯网络可以用以下形式表示：

$$
\begin{aligned}
P(G) & = \prod_{c \in C} P(c) \\
P(E | G) & = \prod_{e \in E} P(e | \text{pa}(e))
\end{aligned}
$$

其中，$G$是贝叶斯网络的结构，$C$是贝叶斯网络的条件节点，$E$是贝叶斯网络的边，$\text{pa}(e)$是边$e$的父节点。

贝叶斯网络的推理可以通过贝叶斯定理、条件概率、边的传播等方法进行。

### 4.1 线性规划示例

```python
import numpy as np
from scipy.optimize import linprog

# 定义决策变量
x = np.array([1, 2, 3])

# 定义权重向量
c = np.array([1, 2, 3])

# 定义约束矩阵
A = np.array([[1, 2, 3], [4, 5, 6]])

# 定义约束向量
b = np.array([10, 20])

# 使用linprog函数解决线性规划问题
result = linprog(c, A_ub=A, b_ub=b, bounds=[(0, np.inf), (0, np.inf), (0, np.inf)])

# 输出解决结果
print(result)
```

### 4.2 约束优化示例

```python
from scipy.optimize import minimize

# 定义决策变量
x = np.array([1, 2, 3])

# 定义目标函数
def f(x):
    return x[0] + x[1] + x[2]

# 定义约束函数
def g(x):
    return x[0] + x[1] - 5

# 定义等式约束函数
def h(x):
    return x[0] * x[1] - 4

# 使用minimize函数解决约束优化问题
result = minimize(f, x, constraints={'type': 'ineq', 'fun': g}, equality_constraints=[{'type': 'eq', 'fun': h}])

# 输出解决结果
print(result)
```
