# 4.具体代码实例和详细解释说明

在自由职业工作中，程序员需要掌握一些编程技能，例如Python、Java、C++等。以下是一个简单的Python代码实例，用于计算收入和支出：

```python
import datetime

class Income:
    def __init__(self, date, amount):
        self.date = date
        self.amount = amount

class Expense:
    def __init__(self, date, amount):
        self.date = date
        self.amount = amount

def calculate_balance(income_list, expense_list):
    balance = 0
    for income in income_list:
        balance += income.amount
    for expense in expense_list:
        balance -= expense.amount
    return balance

# 示例
income_list = [Income(datetime.date(2021, 1, 1), 1000), Income(datetime.date(2021, 1, 15), 1500)]
expense_list = [Expense(datetime.date(2021, 1, 5), 500), Expense(datetime.date(2021, 1, 10), 300)]

balance = calculate_balance(income_list, expense_list)
print(balance)  # 输出: 1200
```

在这个代码实例中，我们定义了两个类：Income和Expense，用于表示收入和支出。然后，我们定义了一个函数calculate_balance，用于计算收入和支出的总和。最后，我们创建了两个收入和两个支出的实例，并计算了总和。
