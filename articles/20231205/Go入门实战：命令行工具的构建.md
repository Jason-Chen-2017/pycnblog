## 4.具体代码实例和详细解释说明

在这个部分，我们将通过一个具体的命令行工具示例来详细解释Go语言的代码实现。

### 示例：文件大小统计工具

我们将构建一个命令行工具，用于统计文件夹下所有文件的大小。这个工具将接受一个文件夹路径作为参数，并输出文件大小的统计信息。

```go
package main

import (
	"flag"
	"fmt"
	"os"
	"path/filepath"
)

func main() {
	folderPath := flag.String("folder", ".", "文件夹路径")
	flag.Parse()

	err := filepath.Walk(*folderPath, func(path string, info os.FileInfo, err error) error {
		if err != nil {
			return err
		}

		if info.IsDir() {
			return nil
		}

		fileSize := info.Size()
		fmt.Printf("文件 %s 的大小为 %d 字节\n", path, fileSize)
		return nil
	})

	if err != nil {
		fmt.Println("错误：", err)
	}
}
```

### 代码解释

1. 首先，我们使用`flag`包来定义命令行参数。`flag.String`函数用于定义一个字符串类型的参数，其中`"folder"`是参数名称，`"."`是默认值，`"文件夹路径"`是参数描述。

2. 然后，我们使用`flag.Parse`函数来解析命令行参数。这个函数将解析后的参数值存储在`flag`包中。

3. 接下来，我们使用`filepath.Walk`函数来遍历文件夹下的所有文件。`filepath.Walk`函数接受一个文件夹路径和一个回调函数作为参数。回调函数将在每个文件和文件夹上调用，我们可以在回调函数中执行相关操作。

4. 在回调函数中，我们首先检查是否存在错误。如果存在错误，我们将其返回。

5. 然后，我们检查当前路径是否为文件夹。如果是文件夹，我们返回nil，表示当前路径已处理完成。

6. 如果当前路径不是文件夹，我们获取文件的大小信息，并输出文件路径和大小。

7. 最后，我们检查是否存在错误，如果存在错误，我们将其打印出来。
