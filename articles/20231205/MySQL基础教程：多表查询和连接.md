# 4.具体代码实例和详细解释说明
在MySQL中，我们可以使用JOIN子句来实现多表查询和连接。JOIN子句允许我们将多个表的数据组合在一起，并根据指定的条件进行连接。

例如，假设我们有两个表：客户表（Customer）和订单表（Orders）。客户表包含客户的名字和ID，订单表包含订单的ID和客户ID。我们可以使用JOIN子句来查询客户表和订单表的数据，并根据客户ID进行连接。

以下是一个使用JOIN子句实现多表查询和连接的例子：

```sql
SELECT c.name, o.order_id
FROM Customer c
JOIN Orders o
ON c.id = o.customer_id;
```

在这个例子中，我们使用JOIN子句将客户表和订单表连接在一起，并根据客户ID进行连接。我们选择了客户表中的名字字段和订单表中的订单ID字段，并将这些字段作为查询结果的列。
