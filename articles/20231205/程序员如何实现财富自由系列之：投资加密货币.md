## 4.1 代码实例

以下是一个使用Python编写的代码实例，用于交易加密货币：

```python
import requests
import json

# 设置API密钥
API_KEY = 'your_api_key'

# 设置加密货币名称
CRYPTO_CURRENCY = 'bitcoin'

# 获取加密货币价值
def get_crypto_value():
    url = f'https://api.coingecko.com/api/v3/simple/price?ids={CRYPTO_CURRENCY}&vs_currencies=usd'
    response = requests.get(url, headers={'x-rapidapi-key': API_KEY})
    data = json.loads(response.text)
    return data[CRYPTO_CURRENCY]['usd']

# 主函数
if __name__ == '__main__':
    crypto_value = get_crypto_value()
    print(f'加密货币价值：{crypto_value}美元')
```

1.设置API密钥：在使用API时，需要设置API密钥。在这个例子中，我们使用了CoinGecko API，并设置了API密钥。

2.设置加密货币名称：我们需要设置要交易的加密货币名称。在这个例子中，我们设置了加密货币名称为“bitcoin”。

3.获取加密货币价值：我们使用CoinGecko API获取加密货币的价值。我们发送一个GET请求，并将API密钥作为请求头。然后，我们解析响应数据，并返回加密货币的价值。
