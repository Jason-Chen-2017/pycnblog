# 4.具体代码实例和详细解释说明

## 4.1 创建一个简单的机器人类
```python
class Robot:
    def __init__(self):
        self.name = "Python机器人"

    def move(self, direction):
        print(f"{self.name} 向{direction}移动")

    def turn(self, angle):
        print(f"{self.name} 向{angle}度转向")
```

## 4.2 创建一个简单的机器人控制器
```python
import time

class RobotController:
    def __init__(self, robot):
        self.robot = robot

    def move_forward(self, distance):
        self.robot.move("前方")
        time.sleep(distance / 10)

    def turn_left(self, angle):
        self.robot.turn(angle)
        time.sleep(angle / 10)
```

## 4.3 创建一个简单的机器人控制程序
```python
if __name__ == "__main__":
    robot = Robot()
    controller = RobotController(robot)

    controller.move_forward(100)
    controller.turn_left(90)
```
