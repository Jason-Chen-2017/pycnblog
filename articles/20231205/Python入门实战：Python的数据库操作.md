## 1.6 Python数据库操作的代码实例

Python数据库操作的代码实例包括：

1. 数据库连接：

```python
import mysql.connector

# 创建数据库连接
db = mysql.connector.connect(
    host="localhost",
    user="root",
    password="password",
    database="test"
)
```

2. 创建数据表：

```python
# 创建数据表
cursor = db.cursor()
sql = "CREATE TABLE students (name VARCHAR(20), age INT)"
cursor.execute(sql)
```

3. 插入数据：

```python
# 插入数据
sql = "INSERT INTO students (name, age) VALUES (%s, %s)"
val = ("John", 20)
cursor.execute(sql, val)
db.commit()
```

4. 查询数据：

```python
# 查询数据
sql = "SELECT * FROM students"
cursor.execute(sql)
result = cursor.fetchall()
for row in result:
    print(row)
```

5. 更新数据：

```python
# 更新数据
sql = "UPDATE students SET age = %s WHERE name = %s"
val = (21, "John")
cursor.execute(sql, val)
db.commit()
```

6. 删除数据：

```python
# 删除数据
sql = "DELETE FROM students WHERE name = %s"
val = ("John")
cursor.execute(sql, val)
db.commit()
```
