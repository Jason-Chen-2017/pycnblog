### 4.1 进程的创建

在 C 语言中，可以使用 fork() 函数来创建新进程。fork() 函数会创建一个新进程，并将其与父进程进行分离。新进程可以继承父进程的部分信息，如程序代码、数据、文件描述符等。

以下是一个使用 fork() 函数创建新进程的代码实例：

```c
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>

int main() {
    pid_t pid = fork();

    if (pid == 0) {
        // 新进程
        printf("Hello, I am the child process!\n");
    } else {
        // 父进程
        printf("Hello, I am the parent process!\n");
    }

    return 0;
}
```

在上述代码中，我们首先调用 fork() 函数来创建新进程。如果 fork() 函数返回 0，则表示当前进程是新进程，我们可以在新进程中执行相关操作。否则，表示当前进程是父进程，我们可以在父进程中执行相关操作。

### 4.2 进程的终止

在 C 语言中，可以使用 exit() 函数来终止当前进程。exit() 函数会释放进程占用的资源，并通知父进程进程已经终止。

以下是一个使用 exit() 函数终止进程的代码实例：

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    printf("Hello, I am the parent process!\n");

    // 创建新进程
    pid_t pid = fork();

    if (pid == 0) {
        // 新进程
        printf("Hello, I am the child process!\n");
        // 终止进程
        exit(0);
    } else {
        // 父进程
        printf("Hello, I am the parent process!\n");
        // 等待子进程终止
        wait(NULL);
    }

    return 0;
}
```

在上述代码中，我们首先调用 fork() 函数来创建新进程。然后，如果当前进程是新进程，我们可以在新进程中执行相关操作。在新进程中，我们调用 exit() 函数来终止进程。如果当前进程是父进程，我们可以在父进程中执行相关操作。在父进程中，我们调用 wait() 函数来等待子进程终止。

1. Q: 进程和线程的区别是什么？
   A: 进程是操作系统中的一个基本单元，它是计算机程序在执行过程中的一个实例。进程有独立的地址空间，可以独立运行。线程是进程内的一个执行单元，它共享进程的资源，如内存空间、文件描述符等。线程之间可以并发执行，从而提高程序的执行效率。
2. Q: 进程和线程的优缺点是什么？
   A: 进程的优点是独立性强、资源隔离，进程之间相互独立，可以互相保护。进程的缺点是开销较大、切换开销较大。线程的优点是开销较小、并发度较高，线程之间共享进程的资源，可以减少内存开销。线程的缺点是线程间的同步问题较为复杂。
3. Q: 如何实现进程间通信？
   A: 进程间通信（IPC）是操作系统中的一个重要功能，它可以实现多个进程之间的数据交换。常见的进程间通信方法有：管道、命名管道、消息队列、信号量、共享内存等。这些方法可以根据不同的需求选择使用。
