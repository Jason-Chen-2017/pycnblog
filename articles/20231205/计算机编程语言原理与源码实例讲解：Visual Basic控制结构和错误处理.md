## 1.4 具体代码实例和详细解释说明

在这部分，我们将通过具体的代码实例来详细解释控制结构和错误处理的使用方法。

### 1.4.1 if-else语句的使用

```vb
Dim x As Integer
x = 10
If x > 5 Then
    Console.WriteLine("x 大于 5")
ElseIf x < 5 Then
    Console.WriteLine("x 小于 5")
Else
    Console.WriteLine("x 等于 5")
End If
```

在这个代码实例中，我们使用if-else语句来判断变量x的值是否大于5、小于5或等于5。如果x大于5，则输出"x 大于 5"；如果x小于5，则输出"x 小于 5"；如果x等于5，则输出"x 等于 5"。

### 1.4.2 循环语句的使用

```vb
For i As Integer = 1 To 10
    Console.WriteLine(i)
Next
```

在这个代码实例中，我们使用for循环来输出从1到10的数字。for循环的语法是For i = 初始值 To 终止值 步长 Next，其中初始值是循环开始的值，终止值是循环结束的值，步长是循环每次迭代的值。

### 1.4.3 try-catch语句的使用

```vb
Try
    Dim result = Convert.ToInt32(Console.ReadLine())
Catch ex As Exception
    Console.WriteLine("输入的值不是整数")
End Try
```

在这个代码实例中，我们使用try-catch语句来捕获输入值不是整数的异常。try语句用来定义可能出现异常的代码块，catch语句用来处理异常情况。在这个例子中，如果用户输入的值不是整数，则会捕获到异常，并输出"输入的值不是整数"。
