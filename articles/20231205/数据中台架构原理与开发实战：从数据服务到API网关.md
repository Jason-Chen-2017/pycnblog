假设我们有两个数据资源文件，分别是 employee.csv 和 department.csv，我们需要将这两个数据资源整合成一个新的数据资源文件，并将员工的部门信息添加到新的数据资源文件中。

首先，我们需要读取 employee.csv 和 department.csv 文件，并将其中的数据转换成 Python 中的字典对象。

```python
import csv

def read_csv(file_path):
    with open(file_path, 'r') as f:
        reader = csv.DictReader(f)
        return [row for row in reader]

employee_data = read_csv('employee.csv')
department_data = read_csv('department.csv')
```

接下来，我们需要将 employee.csv 中的员工信息与 department.csv 中的部门信息进行匹配，并将匹配到的部门信息添加到员工信息中。

```python
def match_department(employee, department):
    for d in department:
        if employee['department_id'] == d['department_id']:
            employee['department_name'] = d['department_name']
            break

for e in employee_data:
    match_department(e, department_data)
```

最后，我们需要将整合后的员工信息写入到一个新的数据资源文件中。

```python
def write_csv(file_path, data):
    with open(file_path, 'w') as f:
        writer = csv.DictWriter(f, fieldnames=data[0].keys())
        writer.writeheader()
        for row in data:
            writer.writerow(row)

write_csv('employee_department.csv', employee_data)
```
