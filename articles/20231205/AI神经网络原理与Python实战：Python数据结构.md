## 3.1 前向传播

前向传播是神经网络的一种计算方法，它通过从输入层到输出层传递信息，以计算输出。在前向传播过程中，每个神经元接收来自前一层神经元的输入，对其进行计算，并将结果传递给下一层。

在前向传播过程中，每个神经元的输出是由其输入和权重之间的乘积以及一个激活函数的应用得到的。激活函数是一个映射输入到输出的函数，它将神经元的输入映射到一个数值范围内。

## 3.2 损失函数

损失函数是一个数学函数，用于衡量神经网络的预测与实际值之间的差异。损失函数的目标是最小化这个差异，以便使神经网络的预测更接近实际值。

常用的损失函数有均方误差（Mean Squared Error，MSE）和交叉熵损失（Cross-Entropy Loss）。均方误差用于回归问题，它计算预测值与实际值之间的平均平方差。交叉熵损失用于分类问题，它计算预测值与实际值之间的交叉熵。

## 3.3 反向传播

反向传播是神经网络的一种训练方法，它通过计算每个神经元的梯度来优化神经网络的权重。在反向传播过程中，每个神经元的梯度是由其输入和权重之间的梯度以及激活函数的梯度得到的。

反向传播的目标是最小化损失函数，以便使神经网络的预测更接近实际值。为了实现这个目标，反向传播通过计算每个神经元的梯度来更新神经网络的权重。

## 3.4 优化算法

优化算法是一种数学方法，用于最小化一个函数。在神经网络中，优化算法用于更新神经网络的权重，以便最小化损失函数。

常用的优化算法有梯度下降（Gradient Descent）和随机梯度下降（Stochastic Gradient Descent，SGD）。梯度下降是一种迭代方法，它通过计算函数的梯度来更新函数的参数。随机梯度下降是一种随机梯度下降的变体，它通过随机选择样本来更新函数的参数。
