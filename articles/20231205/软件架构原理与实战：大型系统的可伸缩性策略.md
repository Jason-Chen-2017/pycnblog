```python
import hashlib
import threading

class DistributedHashTable:
    def __init__(self):
        self.hash_table = {}
        self.lock = threading.Lock()

    def put(self, key, value):
        with self.lock:
            hash_key = hashlib.sha256(key.encode()).hexdigest()
            self.hash_table[hash_key] = value

    def get(self, key):
        with self.lock:
            hash_key = hashlib.sha256(key.encode()).hexdigest()
            return self.hash_table.get(hash_key, None)
```

在上述代码中，我们实现了一个简单的分布式哈希表。`DistributedHashTable`类包含一个哈希表`hash_table`和一个锁`lock`。`put`方法用于将键值对存储到哈希表中，`get`方法用于从哈希表中查找键值对。我们使用了Python的`hashlib`库来计算哈希值，并使用了`threading.Lock`来实现线程安全。

通过这个简单的代码实例，我们可以看到如何实现一个基本的分布式哈希表，并使用锁来保证线程安全。这个实例可以作为大型系统的可伸缩性策略的起点，我们可以根据需要进一步优化和扩展这个实现。
