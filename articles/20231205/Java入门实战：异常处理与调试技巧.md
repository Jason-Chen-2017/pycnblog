## 1.2 Java异常处理的基本概念
Java异常处理的基本概念包括异常、异常类、异常处理器和异常处理策略等。

### 1.2.1 异常
异常是程序运行过程中发生的错误，可以是运行时错误（例如，数组越界错误）或者编译时错误（例如，变量未定义错误）。异常可以通过throw关键字抛出，并通过try-catch块捕获和处理。

### 1.2.2 异常类
异常类是Java中的一个特殊类型，它继承自Throwable类。异常类可以用来定义自定义异常，以便在程序运行过程中更好地处理异常情况。

### 1.2.3 异常处理器
异常处理器是Java中的一个特殊类型，它用于处理异常。异常处理器可以通过try-catch块捕获异常，并执行相应的处理逻辑。

### 1.2.4 异常处理策略
异常处理策略是Java中的一个重要概念，它决定了程序在发生异常时如何进行处理。异常处理策略可以是忽略异常、捕获异常、抛出异常等。

```java
    }
```


### 1.6.3 代码实例3：捕获异常
```java

    try {
        method1();
    } catch (MyException e) {
        e.printStackTrace();
    }
}
```

### 1.6.4 代码实例4：处理异常
```java
public void method3() {
    try {

    } finally {
        System.out.println("异常处理完成");
    }
}
```
