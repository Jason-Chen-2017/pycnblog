# 2.1 风格迁移的核心概念
在风格迁移中，我们需要两个图像：一张源图像（content image）和一张风格图像（style image）。源图像是我们要将风格应用到的图像，风格图像是我们要将风格从中提取出来的图像。我们的目标是将风格图像的特征（如颜色、纹理、边缘等）应用到源图像上，从而创造出一个新的艺术作品。

# 2.2 风格迁移与深度学习的联系
风格迁移是一种深度学习技术，它利用卷积神经网络（CNN）来提取图像的特征。CNN 是一种特殊的神经网络，它由多个卷积层、池化层和全连接层组成。卷积层用于提取图像的特征，池化层用于降低图像的分辨率，全连接层用于将提取出的特征映射到目标图像上。
