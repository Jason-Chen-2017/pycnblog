```python
import requests

def query_data(api_url, params):
    response = requests.get(api_url, params=params)
    if response.status_code == 200:
        return response.json()
    else:
        return None

data = query_data("https://api.example.com/data", {"key": "value"})
```

- 首先，我们导入了`requests`库，这是一个用于发送HTTP请求的库。

- 然后，我们定义了一个`query_data`函数，它接受两个参数：`api_url`和`params`。`api_url`是API的URL，`params`是API的参数。

- 在`query_data`函数中，我们使用`requests.get`方法发送一个GET请求，以获取API的数据。我们将`api_url`和`params`作为参数传递给`requests.get`方法。

- 如果请求成功（响应状态码为200），我们将返回API的数据，以JSON格式。否则，我们将返回`None`。

- 最后，我们调用`query_data`函数，并将API的URL和参数作为参数传递。我们将返回的数据存储在`data`变量中。
