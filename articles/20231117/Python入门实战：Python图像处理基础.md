## 概述
图像处理（Image Processing）是计算机视觉中非常重要的一环。它从不同角度观察、识别、分析、存储、处理和传输图像信息，通过图像处理技术可以对图像进行拍摄、采集、存储、检索、增强、分析和归纳提取其中的有效信息。Python在图像处理领域也占据着重要地位，许多优秀的图像处理库都是用Python编写而成。因此掌握Python图像处理技术是现代计算机视觉工程师必备技能之一。
本教程是作者结合个人经验，系统性地学习并使用Python进行图像处理的系列教程。希望通过本教程可以帮助读者了解Python在图像处理领域的作用及如何进行图像处理任务。通过本教程，读者将能够熟练掌握Python编程、掌握Python在图像处理领域的一些常用模块及库的使用方法，有能力实现基本的图像处理功能。
## 计算机图形学简介
图像是由像素组成的二维数组。数字图像由像素点阵组成，每一个像素点都是一个有颜色的点。图像分辨率越高，表示图像细节的数值就越多，每个像素点的信息量就越丰富；图像分辨率越低，则表示图像细节的数值就越少，每个像素点的信息量就越少。对于彩色图像来说，其颜色由红绿蓝三个基色叠加而成。
### 色彩空间与RGB
由于人眼对颜色的感知有限，所以图像中颜色的表达方式往往是采用加权组合的方式。在这个过程中，人们倾向于认为最亮或者最暗的那种颜色会影响我们的注意力。通常认为人眼对颜色的感知是存在上下界的，即最小的色度值为“黑”（0）对应于最暗，最大的色度值为“白”（1）对应于最亮。而更准确的说法应该是人眼对光谱范围的感知，因为不同的物体对于不同波长的光的反射特性不同。比如日光、水光等波长的反射特性不同，导致它们的色度值不同。如果不考虑光源特性，那么颜色就会存在上下界，通常使用RGB三原色来代表颜色。RGB三原色就是指红色、绿色、蓝色这三个色彩原件的色度组合，每个原件色度值的范围在0~1之间。RGB三原色的组合方式如图所示：
### 图片格式
在网络上传播时，常用的图片格式包括JPG、PNG、GIF等。其中JPG(Joint Photographic Experts Group)，一种适用于摄影或照片应用的高质量图像文件格式，JPG压缩率较高，但容量太大时，文件大小可能达到几百KB甚至MB。PNG(Portable Network Graphics)，也是一种图片格式，它具有高压缩比的优点，同时还支持透明度。GIF(Graphics Interchange Format)，一种动画图片格式，用于快速简单的实现简单动画效果。除此之外还有WebP格式等。
### RGB与HSV色彩模型
RGB颜色模型虽然直观，但缺乏层次感，对于图像处理来说比较不方便，于是便有了HSV(Hue Saturation Value)色彩模型。HSV色彩模型有两个特点，一是色相(H)，用角度度量，取值范围为0°～360°；二是饱和度(S)，用百分比度量，取值范围为0%～100%；三是明度(V)，用百分比度量，取值范围为0%～100%。通过色相和饱和度可以得到颜色的强度，而明度表示颜色的鲜艳程度。通过改变饱和度或明度，可以使图像变得模糊或丰满。HSV模型可以轻易的与其他颜色模型互转。如下图所示：
