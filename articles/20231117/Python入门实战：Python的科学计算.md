## 什么是Python？
Python是一个高层次、高级的编程语言，它广泛用于科学计算，数据分析，机器学习，Web开发，图形界面设计， scientific computing等领域。它的设计哲学强调代码简洁而可读性，是一个动态的数据类型、多种编程范式的语言，支持多线程和面向对象编程。
## 为什么要用Python？
- Python简单易学:Python学习起来相对来说比较容易，语法简单，且几乎所有平台都可以运行，很多熟练掌握Python的人员都可以在短时间内掌握。
- Python具有广泛的应用领域:Python具备了非常丰富的应用领域，包括但不限于科学计算、数据分析、机器学习、Web开发、图形界面设计、scientific computing等，只需要掌握Python的一小部分知识就可以在不同的领域有所作为。
- Python社区活跃:Python有着良好的社区氛围，拥有庞大的用户群体，相关的库和工具也十分丰富。而且由于Python是一个开源项目，因此有很多优秀的第三方库可以帮助进行快速开发。
- Python支持多种编程范式:Python支持多种编程范式，包括面向对象编程、命令式编程等，并且由于解释型语言天生具有动态语言的特征，可以轻松应对一些复杂的任务。

除了以上这些优点之外，Python还可以实现如下功能：

 - 爬虫和网络爬虫：通过编写爬虫脚本，我们可以自动地获取网页上的信息，并保存到本地数据库或文件中。通过Python，我们可以快速筛选出感兴趣的内容，提取想要的信息，从而实现数据的采集、清洗和处理。
 - 数据分析：Python的强大的数据分析功能可以实现对各种数据的快速分析、处理和可视化，使得数据更加清晰、可理解。
 - 游戏开发：Python提供了众多游戏引擎，可以用来开发丰富多彩的游戏内容。游戏制作也可以通过Python实现，其易用性和高效率使得游戏开发变得容易。
 - Web开发：Python提供了强大的Web框架，可以帮助开发者快速地开发出功能丰富的网站。比如Django，Flask等。同时，通过Python的第三方库，可以实现Web服务端的自动化运维。

总而言之，Python是一种简单而灵活的编程语言，适合于多种应用场景。作为一个学习计算机科学及相关领域的高级编程语言，Python可以成为科学计算、数据分析、机器学习、Web开发、游戏开发、深度学习、音视频处理等领域的通用编程语言。
# 2.核心概念与联系
## 基本数据类型
### 整数
整型也就是整数，又称整数类型，表示的是正或负整数，没有大小限制，可以使用下划线 '_' 表示数值的分隔符。示例如下：

```
>>> a = 1_234_567     # 使用下划线 "_" 分隔符表示数字的分界位置
>>> print(a)
1234567
```
Python 3 中，整数没有 long 类型，对于很大的整数，可以通过后缀 'L' 来表示。例如，如果超过 sys.maxsize (即 Python 的最大整数)，则会自动转换成长整数。

```
>>> b = 9223372036854775807   # 正常的 int 范围
>>> c = 9223372036854775808   # 会自动转成 long int
>>> d = 123_456_789_123_456_789
>>> e = 0b11111111_11111111_11111111_11111111    # 可以用二进制表示整数
>>> f = 0o777         # 可以用八进制表示整数
>>> g = 0x7FFFFFFFFFFFFFFF      # 可以用十六进制表示整数
```

### 浮点数
浮点数也就是带有小数部分的数值，由小数点 '.' 表示，可以使用下划线 '_' 表示数值的分隔符。示例如下：

```
>>> pi = 3.14159        # 定义浮点数
>>> tau = 2*pi          # 用 '*' 运算符进行乘法运算
>>> e = 2.7182818       # 以 e 表示自然常数
>>> one_over_sqrt_two = 1/2**0.5  # 使用 ** 运算符进行乘方运算
```

### 布尔值
布尔值类型只有两个值 True 和 False，一般用在条件判断和循环控制语句中。示例如下：

```
>>> flag = True     # 定义布尔值
>>> if flag:
    print('flag is true')  # 如果 flag 为 True，打印这句话
else:
    print('flag is false')  # 如果 flag 为 False，打印这句话

for i in range(5):            # for 循环
    if i % 2 == 0:             # 判断 i 是否为偶数
        continue              # 如果是偶数，跳过本次循环，继续执行下一次迭代
    else:
        print(i)               # 如果是奇数，打印 i
```

### 字符串
字符串类型是 Python 中最常用的类型，通常用单引号或双引号括起来的任意字符序列。字符串可以用加号 + 拼接、乘号 * 重复输出多个字符串。字符串类型有两种索引方式：

1. 元素索引（[]），通过索引来访问每个字符串的某个元素，索引从 0 开始，可以直接使用数字。
2. 切片索引（[:]），通过切片来访问一段连续的字符串。

示例如下：

```
>>> s = 'hello world!'           # 定义字符串
>>> len(s)                      # 获取字符串长度
12
>>> s[0]                        # 通过索引获得第一个元素
'h'
>>> s[-1]                       # 通过索引获得最后一个元素
'!'
>>> s[0:5]                      # 通过切片获得前五个字符
'hello'
>>> s[:5]                       # 省略第一个索引默认值为 0
'hello'
>>> s[6:]                       # 省略第二个索引默认值为字符串结束位置
'world!'
>>> s[-5:-2]                    # 通过切片获得倒数第五到倒数第二个字符
'dorl'
>>> s[:-2]                     # 省略第一个索引以获得整个字符串
'helloworl'
>>> str1 = "I'm Python"        # 定义字符串
>>> str2 = r"I\'m Python"      # 在字符串中添加反斜杠（\）表示原始字符串，防止转义
>>> str1[2]                     # 子串截取
"'m"
>>> '%d %.2f' % (3, 4.567)      # 将不同数据类型的值拼接为字符串
'3 4.57'
```

### None值
None 是 Python 中的一个特殊的数据类型，表示空值，赋值给变量时尤其有用，代表一个未知或无效的值。

```
>>> x = None                 # 定义空值
>>> y = abs(-1) if x is not None else 'default value'  # 根据 x 是否为空值决定输出
'default value'
```

### 元组
元组类型是不可变序列类型，包含若干逗号分割的值。元组可以被索引和切片，不能修改其值。元组的元素是按顺序排列的，不可改变，不能删除。示例如下：

```
>>> t = ('apple', 'banana', 'orange')    # 定义元组
>>> len(t)                                # 获取元组长度
3
>>> t[0]                                  # 元素索引
'apple'
>>> t[-1]                                 # 负数索引，从末尾开始计数
'orange'
>>> t[0:2]                                # 切片索引
('apple', 'banana')
>>> tuple([1,2,3])                         # 从列表创建元组
(1, 2, 3)
```

### 列表
列表类型是可变序列类型，包含若干逗号分割的值。列表可以被索引、切片、追加和插入元素，可以修改其值。列表的元素是按顺序排列的，可以改变，可以删除。示例如下：

```
>>> lst = [1, 2, 3]           # 创建列表
>>> len(lst)                  # 获取列表长度
3
>>> lst[0]                    # 元素索引
1
>>> lst[-1]                   # 负数索引，从末尾开始计数
3
>>> lst[0:2]                  # 切片索引
[1, 2]
>>> lst += ['new element']     # 添加元素
>>> lst.append(['another new element'])   # 使用 append() 方法添加另一个元素
>>> lst.insert(1, 'inserted')  # 插入元素
>>> lst[::-1]                 # 反转列表
['another new element', 'inserted', 'new element', 3, 2, 1]
>>> lst.reverse()             # 使用 reverse() 方法反转列表
>>> lst[::2]                  # 步长为 2 的切片
[1, 'inserted', 'new element', 2]
```

## 操作符与表达式
Python中有多种类型的操作符与表达式，我们将其归纳如下：

### 一元操作符

| 操作符 | 描述                                                         | 例子                          |
| ------ | ------------------------------------------------------------ | ----------------------------- |
| `+`    | 正号，当作无操作符使用时，返回对象 unchanged，适用于任何对象，但是如果操作数为 complex 对象，则返回复数单位元 | `print(+3)`<br>`complex(2)+2j`<br>`-(-3)` |
| `-`    | 负号，取反数，将对象变负，适用于数字，字符串，集合，列表和元组。 | `-3`, `'hello'[::-1]`         |

### 算术操作符

| 操作符 | 描述                                                         | 例子                            |
| ------ | ------------------------------------------------------------ | ------------------------------- |
| `+`    | 加法运算，用来连接或求和数字，也可以用于字符串，元组和列表的连接。 | `3 + 4` <br>`'Hello'+'World'`    |
| `-`    | 减法运算，用来减去数字，也可以用于字符串，元组和列表的切片。   | `5 - 2` <br> `'abcdefg'[3:5]`    |
| `*`    | 乘法运算，用来重复字符串或列表中的元素，或者计算矩阵乘积。     | `3 * 'abc'` <br>`[[1,2],[3,4]]@[[5,6],[7,8]]` |
| `/`    | 除法运算，用来计算数值的商，也可以用来切割字符串或列表的元素。 | `10 / 3` <br> `'abcde'[::2]`      |
| `%`    | 求模运算，返回除法运算后的余数。                               | `7 % 2`                         |
| `//`   | 取整除运算，返回商的整数部分。                                 | `10 // 3`                       |

### 比较操作符

| ------ | ------------------------------------------------------------ | ------------------------------ |
| `<`    | 小于关系运算符                                               | `1 < 2`                        |

### 逻辑操作符

| 操作符 | 描述                                                         | 例子                                   |
| ------ | ------------------------------------------------------------ | -------------------------------------- |
| `not`  | 逻辑非运算符，取反运算符，用于对表达式的逻辑值取反，可以用来实现真假表。 | `not True`                             |

### 位操作符

| 操作符 | 描述                                                         | 例子                              |
| ------ | ------------------------------------------------------------ | --------------------------------- |
| `&`    | 按位与运算符，将对应的二进位进行“与”运算，返回结果的每一位都是对应位上两边对应的二进位的 “与” 结果。 | `bin(3 & 5)<br>bin(3 & 6)`         |
| `\|`   | 按位或运算符，将对应的二进位进行“或”运算，返回结果的每一位都是对应位上两边对应的二进位的 “或” 结果。 | `bin(3 \| 5)<br>bin(3 \| 6)`       |

| `<<`   | 左移运算符，将数字的所有位向左移动指定的位数，低位溢出的位被舍弃，高位补零。 | `5 << 2`                          |
