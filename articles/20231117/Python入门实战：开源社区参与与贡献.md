# 2.核心概念与联系
## 2.1 Python简介
Python 是一种具有简单性、易用性、功能强大、适用于多种场景的高级编程语言。它具有丰富的数据结构、异常处理机制、函数式编程特性等特点。与其他编程语言如 Java、C++ 不同的是，Python 在最初设计之初就已经带有一个巨大的影响力。目前，Python 已成为主流的高级编程语言，几乎所有企业应用都采用了 Python 技术。
## 2.2 GitHub简介
GitHub是一个基于Web服务的软件版本控制仓库，可以托管各种版本的Git代码，并提供一个web界面用来访问和管理项目。
## 2.3 Linux简介
Linux（英语：/ˈlinuks/）是一个开源的类Unix操作系统（OS），是一个自由软件及UNIX操作系统内核。Linux 是由林纳斯·托瓦兹（<NAME>）在美国加利福尼亚大学伯克利分校开发，是第一个完全基于内核的自由软件/操作系统。它支持32位和64位硬件平台，并且其源代码开放透明，任何人都可以查看、修改或再发布。
## 2.4 Git简介
Git是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。它被GitHub、Bitbucket和 GitLab等多个知名代码托管网站广泛使用。
## 2.5 Git命令简介

| 命令    | 描述                                    |
|:--------:|:----------------------------------------|
| git clone| 从远程仓库克隆本地仓库                    |
| git push | 提交本地更改到远程仓库                    |
| git pull | 拉取远程仓库变动到本地仓库                |
| git fetch| 获取远程仓库最新提交                      |
| git status| 查看当前工作目录状态                     |
| git branch| 创建新分支或切换至已存在分支             |
| git checkout| 检出指定版本文件或切换分支              |
| git merge| 将指定分支合并到当前分支                 |
| git log   | 查看提交历史                            |
| git diff  | 比较不同提交之间的差异                   |
| git add   | 添加指定文件到暂存区                     |
| git rm    | 删除指定文件                           |
| git reset HEAD| 撤销暂存区文件的修改                     |
| git config| 配置Git                                       |
