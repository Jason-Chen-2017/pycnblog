# 2.核心概念与联系
## 2.1 数据类型介绍
首先，让我们先来了解一下什么是数据类型。数据类型是编程语言中用来描述数据的一个重要概念。在Python中，数据类型就是变量所存储值的类型。Python共有八种数据类型：

1.整数（integer）
- 整数就是没有小数点的数字，比如1，2，3，4等；
- 在Python中可以使用关键字`int`表示整数类型；

2.浮点数（floating point number）
- 浮点数就是带有小数点的数字，比如3.14，7.89等；
- 在Python中可以使用关键字`float`表示浮点数类型；

3.复数（complex number）
- 复数是一个由两个实部和一个虚部构成的数，可以用a + bj的形式表示，j表示纯虚部，也就是说j^2 = -1；
- 在Python中可以使用关键字`complex`表示复数类型；

4.布尔值（boolean value）
- 布尔值只有True和False两种值；
- 在Python中，可以直接使用True和False表示布尔类型的值；

5.字符串（string）
- 字符串是由零个或多个字符组成的序列，通常用单引号或双引号括起来，比如'hello'，"world"等；
- 在Python中，字符串类型是一种不可变序列，因此不能被修改，只能创建新的字符串对象；
- 字符串的拼接、截取、替换等操作都是支持的，而且还有很多内置的函数可以处理字符串。

6.列表（list）
- 列表是一种有序且可变的序列，可以存储任意数量和类型的元素；
- 可以通过索引来访问列表中的元素，还可以使用切片操作来获取子列表；
- 可以使用append()方法添加元素至末尾，也可以使用insert()方法插入指定位置；
- 使用pop()方法可以删除末尾的元素，也可以使用remove()方法删除指定元素。

7.元组（tuple）
- 元组与列表类似，不同之处在于元组是不可变的，即它的值一旦初始化后就不能再改变；
- 在定义元组时，如果只有一个元素，则必须加上逗号，否则括号会自动加上，比如(1)和(1,)。

8.字典（dictionary）
- 字典是一个无序的键-值对集合，它的键必须是唯一的，可以用于保存和查找各种信息；
- 通过键来访问字典的值，还可以使用get()方法设置默认值。

了解了这些数据类型之后，我们就可以开始认识Python的一些重要特性了。
## 2.2 内存管理机制
计算机程序运行过程中的数据都保存在内存中，当程序退出时，内存也随之消失。Python通过垃圾回收机制来自动释放不需要使用的内存，但对于某些特殊情况（例如内存占用过多），Python无法释放，这时候就需要手动进行内存管理了。
### 引用计数法
Python的内存管理机制一般采用引用计数法（reference counting）。这种方法的基本思路是跟踪内存中的对象，每当有一个变量指向一个对象时，引用计数器就加一，当变量离开作用域时，引用计数器就减一。当引用计数器变为零时，该对象的内存就会被释放。

但是，由于垃圾收集器的原因，在某些情况下，Python的引用计数可能出现问题。举个例子，如果两个变量都指向同一个列表，而这个列表是另一个变量的成员，那么当其中一个变量离开作用域时，列表的引用计数就会减一，但此时仍有另一个变量保持对其的引用，所以不会立刻释放内存。这种情况下，Python的垃圾收集器就要介入了。

### 标记-清除法
另一种常用的内存管理方式是标记-清除（mark and sweep）法。这种方法将所有的内存块都看作存活的，然后去扫描内存，把未被分配的内存块标记为垃圾，最后再一次性清理掉。

这种方法的缺陷是效率低下，每次清理时都要遍历所有已分配的内存，速度慢。

### 分代回收
除了引用计数法和标记-清除法，Python还有一种分代回收（generational collection）算法。这种算法把内存分成不同的世代，不同的世代按照生命周期长短，分到不同的内存池中，这样就可以避免老生死化的现象，提高性能。目前主流的解释器一般都采用这种方法来提高性能。

## 2.3 作用域规则
作用域（scope）是程序语言中一个重要的概念。作用域决定了变量的作用范围，某个变量是否可以在程序的任意地方被访问到。在Python中，作用域是静态的，即在编译阶段就确定好了。

在Python中，变量的作用域规则如下：

1.一个局部作用域总是嵌套在另一个较大的作用域内，比如一个函数内部；

2.如果在函数内部没有找到对应的名称，就会继续搜索上层作用域，直到找到为止；

3.如果在整个作用域内都找不到相应的名称，就会报错；

4.同一个作用域里面的名称不能重复；

5.不同作用域之间的名称可以相同，因为作用域之间没有联系。
