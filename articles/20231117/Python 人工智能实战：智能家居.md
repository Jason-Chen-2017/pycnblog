                 

# 1.背景介绍


## 智能家居产品和市场分析
### 智能家居市场概况
随着智能化、物联网等新兴技术的应用，智能家居行业也迎来了爆发式的发展。目前智能家居领域主要分为两个方向：一是客厅空调、电灯卫士，二是灯光照明、场景控制、智能窗帘、智能门锁、智能垃圾分类、智能客厅布置、智慧农业。由于市场需求巨大且多元，不同品牌的产品在同一领域内也具有相似性。例如空调产品各家厂商的型号、配置都不尽相同；门锁产品的安全性、空间开关率、防盗设计及防滑机制等参数也不相同。因此，如何准确地搭建智能家居解决方案和客户需要也是智能家居行业的一大难题。以下将介绍各大智能家居公司现有的产品和市场分析。

#### 小米智能家居
小米科技推出了中国第一款智能门锁——米家互联网门锁，可实现远程监控与开门控制，其中包括安防、体感人机交互、双路视频传输、自拍摄功能等。该产品已于2019年10月正式发布。小米还另外推出了一系列智能家居产品，如米家扫地机器人、米家厨房湿度传感器、米家智能净水器等。小米是全球最大的智能家居品牌之一，其产品已经形成了一个完整的生态系统。目前小米旗下的产品均面向国内消费者销售，但并没有像联想这样的公司大规模进入智能家居市场。

#### 荣耀智能家居
荣耀智能家居是一家华东区域的互联网企业，专注于智能家居领域。其前身为中兴通讯的飞利浦智能家居解决方案，基于行业领先的嵌入式技术、云计算平台和IoT技术。荣耀智能家居的产品主要集成在华为路由器上，提供完善的安全防护和网络管理能力，并通过优质的服务质量来确保用户的满意度。目前荣耀智能家居已成为我国智能家居领域的领头羊。目前，荣耀智能家居已经推出了包括荣耀手环、幻彩一夏、天猫精灵、我的生活、萤石云台等众多智能家居产品。除此之外，荣耀还推出了荣耀Vplay、三花智能马桶、睡莲智能蒸汽浴霸、多创智能电动净水器等多种高端智能家居产品。

#### 美的智能家居
美的集团为智能家居领域提供了一整套解决方案，包括智能电动门锁、智能窗帘、智能手套、智能健康管理、智能插座等产品，这些产品通过提升效率和降低成本，帮助客户节省时间和金钱。2019年4月，美国智能家居品牌美的宣布完成A轮5亿元融资，成为中国市值最高的智能家居公司之一。

### 智能家居行业应用案例

虽然智能家居市场蓬勃发展，但并非所有人都赞同将智能家居产品直接安装在家庭内部。作为普通消费者，除了享受到智能设备的便利之外，更希望能够通过简单地操作实现智能化，而不是依赖专业人员的指导。在日益增长的移动互联网时代，更多的人开始选择线上购物，在线播放电视剧、电影甚至网课。智能家居可以帮助人们完成这一步，让用户享受到智能化带来的便利。下面是几家知名企业在智能家居领域应用的一些案例。

#### 汉得信息
汉得信息位于广州，是国内目前规模较大的智能家居企业之一。公司以“为用户提供便捷、安全、智能的生活环境”为愿景，致力于打造具备品质、高效、经济、环保、可持续发展能力的智能家居生态系统。公司历经十余年发展历史，已建成覆盖全国多地的精细化数字化生活管理系统，为数百万家客户提供居家解决方案。其中，汉得信息自主研发的“南威视体验生活”产品可满足人们对生活、工作、学习、娱乐的需求，也可进行个人定制化配置，满足不同生活阶段的个性化需求。

#### 理工宝智能家居
理工宝智能家居于2015年11月份在广州落地，其目标是“打造中国智能家居产业领跑者”。公司拥有丰富的智能硬件产品开发经验，紧密合作伙伴关系，不断创新突破，以“运用科技赋能，实现绿色发展”为己任。截止2020年，公司已经与全国多个城市签署战略合作协议，并形成了中国领先的智能家居品牌。理工宝智能家居的智能音箱产品已在全国多个城市被应用，是广大消费者的首选。

#### 家得宝
家得宝位于广州，是一家集智能硬件、电子产品开发、数字营销、技术服务和品牌营销为一体的高科技公司。公司自成立以来，秉承“专注智能家居领域，打造国际顶尖品牌”的使命，致力于用专业化的解决方案，引领行业的发展方向。目前，家得宝集团共拥有多个项目，涉及智能家居领域多个行业，包括智能厨房、智能冰箱、智能洗衣机、智能电视、智能安防等。

以上都是智能家居行业里，比较有代表性的公司及产品。但实际上，智能家居行业还有很多其他的企业，如：中集家居、康佳智能、天翔智能等。以上只是举例，并非详尽无遗。对于那些没有落地的公司，需要考虑对这些公司的布局策略、产品优势、市场规模、竞争力等进行分析，进而决定自己的下一步发展方向。
