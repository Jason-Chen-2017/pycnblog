1. Python简介
    - Python是一种高级编程语言，拥有简洁、易懂、灵活的语法和强大的功能库。
    
2. Python独特的语法特征
    - 支持多种编程范式，面向对象、函数式编程、命令式编程都可以在其中实现。

3. Python的优点
    - 简单易学：Python具有简单易学、运行速度快的特点。

4. Python的缺点
    - 不适用于性能要求比较高的程序：Python不擅长执行计算密集型任务，如音视频编解码、图像处理、数据库查询等。

# 2.核心概念与联系
## 数据类型与变量
### 数据类型分类
- 整数型(int):表示整数值，例如: `a = 1`, `b= 100`
- 浮点型(float):表示小数值，例如: `c = 3.14`, `d = 2.718`
- 字符串型(str):用来存储文本信息，例如: `name = 'Alice'` 或 `"hello world"`
- 布尔型(bool):只有True或False两个值，表示真或假，例如: `flag = True` 或 `valid = False`
- 列表型(list):元素类型相同，有序集合，可以重复元素，索引从0开始，例如: `fruits = ['apple', 'banana', 'orange']`，或者使用 `list()` 函数创建空列表。
- 元组型(tuple):元素类型相同，有序集合，不可修改，不能添加或删除元素，索引从0开始，例如: `coordinates = (3.14, 2.718)`，或者使用 `tuple()` 函数创建空元组。
- 字典型(dict):键值对集合，无序但唯一，可以通过键获取对应的值，键类型必须是不可变类型，例如: `person = {'name': 'Alice', 'age': 25}` ，或者使用 `dict()` 函数创建空字典。

### 变量的声明
- 在Python中，变量名必须使用大小写英文字母、数字和下划线构成，且不能以数字开头；变量名应尽量简短，易于理解和记忆。
- 变量的赋值方式有两种，分别为直接赋值(=)和间接赋值(:=)。
  - 直接赋值(=`a = 1`):将右侧表达式的值赋给左侧变量。
  - 间接赋值(`a := 1`):将右侧表达式的值赋给左侧变量，同时返回右侧表达式的值。
