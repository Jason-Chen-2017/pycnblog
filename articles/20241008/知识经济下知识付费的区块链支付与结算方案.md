### 2.1 区块链技术

区块链技术是一种去中心化的分布式数据库系统，通过密码学确保数据的不可篡改性和安全性。它由多个区块组成，每个区块包含一定数量的交易记录。这些区块通过加密技术相互链接，形成一条链，因此称为“区块链”。

![区块链结构](https://example.com/blockchain_structure.png)

#### 区块链技术的特点
- 去中心化：区块链网络中的节点共同维护整个网络，不存在中心化的管理机构。
- 不可篡改性：一旦数据被记录在区块链上，就难以篡改，保证了数据的安全性和可信度。
- 安全性：通过密码学技术，如非对称加密和哈希函数，确保了区块链网络的安全。

### 2.2 智能合约

智能合约是基于区块链技术的一种自动化执行合约，通过预定义的逻辑规则自动执行。智能合约的代码通常存储在区块链上，当满足特定条件时，智能合约会自动执行相应的操作。

#### 智能合约的特点

- 可信性：智能合约的执行过程透明、公开，不可篡改，保证了合约的执行可信性。
- 安全性：智能合约基于区块链技术，具有去中心化和安全性的特点。

### 2.3 加密货币

加密货币是基于区块链技术的数字货币，如比特币、以太币等。加密货币通过去中心化的区块链网络进行交易，具有匿名性、安全性、可追溯性等特点。

```
// 哈希函数模型
string hashFunction(string data) {

}
```

举例说明：

假设我们要计算字符串“knowledge”的哈希值，使用SHA-256算法：

```
string hashValue = hashFunction("knowledge");
// 输出：e4d909c290d0fb1ca068ffaddf22cb3ef1a171dd3c28b93d728a8d7e7f4a7ba
```

哈希函数的特点是输入任意长度的数据，输出固定长度的哈希值，且具有抗碰撞性，即找到两个不同输入数据生成相同哈希值的难度极大。

非对称加密是加密货币交易中的核心技术，用于保障交易的安全。以下是一个简单的非对称加密模型：

```
// 非对称加密模型
class AsymmetricEncryption {
    public:
        string encrypt(string data, string publicKey);
        string decrypt(string data, string privateKey);
};

// 加密
string encrypt(string data, string publicKey) {
    // 使用公钥加密数据
    return encryptWithPublicKey(data, publicKey);
}

// 解密
string decrypt(string data, string privateKey) {
    // 使用私钥解密数据
    return decryptWithPrivateKey(data, privateKey);
}
```

举例说明：

假设Alice要向Bob发送一条加密消息，使用RSA算法：

1. Alice生成一对密钥（公钥和私钥）：
```
alicePublicKey = generatePublicKey();
alicePrivateKey = generatePrivateKey();
```

2. Alice使用Bob的公钥加密消息：
```
encryptedMessage = encrypt("Hello, Bob!", bobPublicKey);
// 输出：3c28d8a7f7e6d5c4b3a29182736c4a5b1a2938271a2b1c3d4e5f6g7h8i9j0k
```

3. Bob使用自己的私钥解密消息：
```

```

非对称加密的特点是加密和解密使用不同的密钥，公钥加密的数据只能用对应的私钥解密。

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract KnowledgePayment {
    // 定义知识产品结构
    struct KnowledgeProduct {
        string id; // 产品ID
        string title; // 产品标题
        uint256 price; // 产品价格
        address creator; // 产品创建者地址
    }

    // 定义支付事件
    event PaymentReceived(
        string id,
        string title,
        uint256 price,
        address payer
    );

    // 产品列表
    mapping(string => KnowledgeProduct) public products;

    // 用户余额
    mapping(address => uint256) public balances;

    // 添加新知识产品
    function addProduct(
        string memory id,
        string memory title,
        uint256 price
    ) public {
        require(products[id].id == "", "Product already exists");
        products[id] = KnowledgeProduct(id, title, price, msg.sender);
    }

    // 购买知识产品
    function buyProduct(string memory id) public payable {
        require(products[id].id != "", "Product not found");
        require(msg.value >= products[id].price, "Insufficient payment");

        // 转移加密货币给产品创建者
        payable(products[id].creator).transfer(msg.value);

        // 更新用户余额
        balances[msg.sender] += msg.value - products[id].price;

        // 触发支付事件
        emit PaymentReceived(id, products[id].title, products[id].price, msg.sender);
    }

    // 查询用户余额
    function getBalance() public view returns (uint256) {
        return balances[msg.sender];
    }

    // 提取余额
    function withdraw() public {
        uint256 balance = balances[msg.sender];
        require(balance > 0, "Insufficient balance");

        // 转移余额给用户
        payable(msg.sender).transfer(balance);

        // 更新用户余额
        balances[msg.sender] = 0;
    }
}
```
