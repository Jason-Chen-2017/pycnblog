#### 2.3.1 原理

- 将复杂问题分解为多个子问题；
- 递归地求解子问题，并保存已求解的子问题的解；
- 利用已求解的子问题的解求解原问题。

#### 2.3.2 模板

```
// 动态规划模板
int dp(int i, int j) {
    if (终止条件) {
        return 结果；
    }
    if (状态已计算) {
        return 已计算的状态；
    }
    // 分解子问题
    int a = dp(i + 1, j);
    int b = dp(i, j + 1);
    // 保存已计算的状态
    状态 = a + b；
    return 状态；
}
```

### 3.1 排序算法

排序算法是将一组数据按照特定顺序排列的算法。以下为常见的排序算法及其操作步骤：

#### 3.1.1 冒泡排序

```
// 冒泡排序
for (int i = 0; i < arr.length - 1; i++) {
    for (int j = 0; j < arr.length - i - 1; j++) {
        if (arr[j] > arr[j + 1]) {
            // 交换 arr[j] 和 arr[j + 1]
            int temp = arr[j];
            arr[j] = arr[j + 1];
            arr[j + 1] = temp；
        }
    }
}
```

#### 3.1.2 选择排序

```
// 选择排序
for (int i = 0; i < arr.length - 1; i++) {
    int minIndex = i；
    for (int j = i + 1; j < arr.length; j++) {
        if (arr[j] < arr[minIndex]) {
            minIndex = j；
        }
    }
    // 交换 arr[i] 和 arr[minIndex]
    int temp = arr[i];
    arr[i] = arr[minIndex];
    arr[minIndex] = temp；
}
```

#### 3.1.3 插入排序

```
// 插入排序
for (int i = 1; i < arr.length; i++) {
    int key = arr[i];
    int j = i - 1；
    while (j >= 0 && arr[j] > key) {
        arr[j + 1] = arr[j];
        j--；
    }
    arr[j + 1] = key；
}
```

### 3.2 查找算法

查找算法是在数据集合中查找特定元素的算法。以下为常见的查找算法及其操作步骤：

#### 3.2.1 顺序查找

```
// 顺序查找
int search(int[] arr, int target) {
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == target) {
            return i；
        }
    }
    return -1；
}
```

#### 3.2.2 二分查找

```
// 二分查找
int search(int[] arr, int target) {
    int low = 0；
    int high = arr.length - 1；
    while (low <= high) {
        int mid = (low + high) / 2；
        if (arr[mid] == target) {
            return mid；
        } else if (arr[mid] < target) {
            low = mid + 1；
        } else {
            high = mid - 1；
        }
    }
    return -1；
}
```

### 3.3 图算法

图算法用于解决图相关的问题。以下为常见的图算法及其操作步骤：

#### 3.3.1 深度优先搜索（DFS）

```
// 深度优先搜索
void dfs(int[] arr, int start) {
    visited[start] = true；
    for (int i = 0; i < arr.length; i++) {
        if (arr[start] == arr[i] && !visited[i]) {
            dfs(arr, i)；
        }
    }
}
```

#### 3.3.2 广度优先搜索（BFS）

```
// 广度优先搜索
void bfs(int[] arr, int start) {
    visited[start] = true；
    Queue<Integer> queue = new LinkedList<>();
    queue.offer(start)；
    while (!queue.isEmpty()) {
        int cur = queue.poll();
        for (int i = 0; i < arr.length; i++) {
            if (arr[cur] == arr[i] && !visited[i]) {
                visited[i] = true；
                queue.offer(i)；
            }
        }
    }
}
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明

在解决美团校招面试中的算法编程题时，掌握一定的数学模型和公式对于解题至关重要。以下为几个常见的数学模型和公式的详细讲解与举例说明。

### 4.1 概率论

#### 4.1.1 概率

概率是描述事件发生可能性的数值，取值范围在0到1之间。

#### 4.1.2 概率分布

概率分布是描述随机变量取值概率的函数，常见的有离散型概率分布和连续型概率分布。

**举例说明：**

设有两个随机变量X和Y，X的取值为0和1，P(X=0)=0.5，P(X=1)=0.5；Y的取值为1和2，P(Y=1)=0.6，P(Y=2)=0.4。

求P(X=0且Y=1)。

```
P(X=0且Y=1) = P(X=0) * P(Y=1) = 0.5 * 0.6 = 0.3
```

### 4.2 线性代数

#### 4.2.1 矩阵乘法

矩阵乘法是指两个矩阵相乘的结果是一个新的矩阵。

**举例说明：**

设有两个矩阵A和B：

```
A = | 1 2 |
    | 3 4 |

B = | 5 6 |
    | 7 8 |
```

求矩阵乘法AB的结果。

```
AB = | 1*5 + 2*7 1*6 + 2*8 |
      | 3*5 + 4*7 3*6 + 4*8 |

    = | 19 26 |
      | 43 58 |
```

#### 4.2.2 矩阵求逆

矩阵求逆是指求一个矩阵的逆矩阵。

**举例说明：**

设有矩阵A：

```
A = | 1 2 |
    | 3 4 |
```

求矩阵A的逆矩阵。

```
A的逆矩阵 = | 4 -2 |
            | -3 1 |

    = | 2/5 -1/5 |
      | -3/5 1/5 |
```

### 4.3 微积分

#### 4.3.1 导数

导数是描述函数在某一点处的斜率。

**举例说明：**

设有函数f(x) = x^2。

求f(x)在x=1处的导数。

```
f'(x) = 2x

f'(1) = 2 * 1 = 2
```

#### 4.3.2 积分

积分是求解曲线下的面积或函数的定积分。

**举例说明：**

设有函数f(x) = x。

求f(x)在区间[0, 1]上的定积分。

```
∫[0, 1] x dx = [1/2 * x^2] from 0 to 1

= 1/2 * 1^2 - 1/2 * 0^2

= 1/2
```

### 9.2 算法编程题常见问题

1. **如何实现一个排序算法？**

解答：常见的排序算法有冒泡排序、选择排序、插入排序等。以下是冒泡排序的实现示例：

```
// 冒泡排序
void bubbleSort(int[] arr) {
    int n = arr.length；
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交换 arr[j] 和 arr[j + 1]
                int temp = arr[j]；
                arr[j] = arr[j + 1]；
                arr[j + 1] = temp；
            }
        }
    }
}
```

2. **如何实现一个查找算法？**

解答：常见的查找算法有顺序查找、二分查找等。以下是二分查找的实现示例：

```
// 二分查找
int binarySearch(int[] arr, int target) {
    int low = 0；
    int high = arr.length - 1；
    while (low <= high) {
        int mid = (low + high) / 2；
        if (arr[mid] == target) {
            return mid；
        } else if (arr[mid] < target) {
            low = mid + 1；
        } else {
            high = mid - 1；
        }
    }
    return -1；
}
```
