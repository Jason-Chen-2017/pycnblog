```python
from rdflib import Graph, Namespace
from rdflib.plugin import DatasetParserFactory

# 定义命名空间
ns = Namespace("http://example.org/onto#")

# 创建一个空的Graph对象
g = Graph()

# 使用RDF数据文件加载知识图谱
parser_factory = DatasetParserFactory.for_iri("http://www.w3.org/1999/02/22-rdf-syntax-ns#")
parser_factory.create(g).parse("knowledge_graph.rdf")

# 添加知识图谱中的实体和关系
g.add((ns.gene, ns.type, ns.Entity))
g.add((ns.protein, ns.type, ns.Entity))
g.add((ns.gene, ns.has_protein, ns.protein))
g.add((ns.gene, ns.expression, ns.Expression))

# 保存知识图谱
g.serialize("knowledge_graph.nt", format="nt")
```

**代码解读**：

- **命名空间定义**：命名空间用于区分不同实体和关系的类别，避免命名冲突。
- **Graph对象创建**：创建一个空的Graph对象，用于存储知识图谱。

```python
import pandas as pd
```
