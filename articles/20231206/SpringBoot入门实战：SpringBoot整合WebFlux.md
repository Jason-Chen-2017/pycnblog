# 4.具体代码实例和详细解释说明

## 4.1 代码实例
以下是一个使用Spring Boot整合WebFlux的简单示例：

```java
@SpringBootApplication
public class WebFluxDemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(WebFluxDemoApplication.class, args);
    }
}

@RestController
class GreetingController {

    @GetMapping("/greeting")
    Mono<String> greeting(@RequestParam("name") String name) {
        return Mono.just("Hello, " + name + "!");
    }
}
```

## 4.2 详细解释说明
在这个示例中，我们创建了一个简单的Spring Boot项目，并添加了WebFlux依赖项。我们创建了一个名为`GreetingController`的控制器类，并使用`@RestController`注解进行标记。在控制器类中，我们使用`@GetMapping`注解定义了一个新的RESTful API端点`/greeting`。

当我们向`/greeting`端点发送GET请求时，`greeting`方法会被调用。这个方法接受一个名为`name`的请求参数，并使用`Mono.just`方法将一个`Mono`类型的值创建出来。在这个例子中，我们将`name`参数添加到字符串中，并将结果作为`Mono`类型的值返回。
