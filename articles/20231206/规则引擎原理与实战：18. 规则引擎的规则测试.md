# 4.具体代码实例和详细解释说明

以下是一个简单的规则测试代码实例：

```python
import rule_engine

# 定义规则
rule = "IF age > 18 AND gender = 'male' THEN eligible = true"

# 解析规则
parsed_rule = rule_engine.parse(rule)

# 执行规则
result = rule_engine.execute(parsed_rule, {'age': 20, 'gender': 'male'})

# 验证结果
if result['eligible']:
    print("规则执行成功")
else:
    print("规则执行失败")
```

在这个代码实例中，我们首先定义了一个规则，然后使用规则引擎的解析函数将规则解析成规则引擎可理解的格式。接下来，我们使用规则引擎的执行函数执行解析后的规则，并将事实作为参数传递给执行函数。最后，我们验证规则执行结果是否符合预期，并输出执行结果。
