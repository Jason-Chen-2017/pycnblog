## 4. 具体代码实例和详细解释说明

在Go语言中，错误处理的一个典型示例是文件操作。以下是一个读取文件的示例代码：

```go
package main

import (
    "fmt"
    "io/ioutil"
    "log"
    "os"
)

func main() {
    file, err := os.Open("test.txt")
    if err != nil {
        log.Fatal(err)
    }
    defer file.Close()

    content, err := ioutil.ReadAll(file)
    if err != nil {
        log.Fatal(err)
    }

    fmt.Println(string(content))
}
```

在这个示例中，我们首先尝试打开文件`test.txt`。如果打开文件失败，`os.Open()`函数将返回错误。我们可以通过检查返回值是否为`nil`来判断是否发生错误。如果返回值为`nil`，则表示没有发生错误；否则，返回值为错误对象。

我们可以通过调用`Error()`方法来获取错误信息。在这个示例中，我们使用`log.Fatal()`函数来输出错误信息，并终止程序执行。
