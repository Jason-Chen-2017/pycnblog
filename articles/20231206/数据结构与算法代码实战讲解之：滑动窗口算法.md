```python
def sliding_window(arr, k):
    window_sum = 0
    window_start = 0
    max_sum = float('-inf')

    for window_end in range(len(arr)):
        window_sum += arr[window_end]

        if window_end - window_start + 1 > k:
            window_sum -= arr[window_start]
            window_start += 1

        if window_end - window_start + 1 == k:
            max_sum = max(max_sum, window_sum)

    return max_sum
```

1. 初始化一个窗口，窗口的左边界和右边界都指向数组的第一个元素。
2. 对窗口内的数据进行操作，即将窗口内的元素加入到窗口和中，并计算窗口和的和。
3. 将窗口向前移动，以便在不同的位置进行操作。移动的方向是向右，移动的距离是1。
4. 重复步骤2和步骤3，直到窗口移动到数组的末尾。
5. 在窗口移动到数组的末尾后，比较窗口和的和与之前的最大和，更新最大和。
6. 返回最大和。
