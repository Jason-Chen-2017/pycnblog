# 2.核心概念与联系

## 2.1概率论与统计学基础

概率论是一门研究不确定性的数学学科，它研究如何计算事件发生的可能性。概率可以用来描述事件的不确定性，也可以用来描述数据的分布。统计学是一门研究如何从数据中抽取信息的学科，它研究如何从数据中估计参数，如何进行假设检验，如何进行预测等。概率论与统计学是机器学习的基础，它们提供了一种数学模型来描述数据和事件之间的关系。

## 2.2神经网络与深度学习基础

神经网络是一种由多个节点（神经元）组成的计算模型，每个节点都接收输入，进行计算，并输出结果。神经网络可以用来解决各种问题，如分类、回归、聚类等。深度学习是神经网络的一种更高级的技术，它可以通过多层次的神经网络来学习更复杂的模式。深度学习可以用来解决各种问题，如图像识别、语音识别、自然语言处理等。
