   ```

   ```

   1. 切换到主分支：
   ```
   git checkout master
   ```

   ```
   git merge new_branch
   ```
   3. 解决合并冲突：
   1. 找到冲突的文件：
   ```
   git diff master new_branch
   ```

   ```
   vim file.txt
   ```
   3. 提交修改：
   ```
   git commit -m "解决冲突"
   ```

   1. 切换到主分支：
   ```
   git checkout master
   ```
   2. 恢复到选择的版本：
   ```
   git reset --hard <commit_id>
   ```
   3. 删除不再需要的分支：
   ```
   git branch -d new_branch
   ```
