### 1.4.1 轨道计算代码实例

```python
import numpy as np

def calculate_orbit(initial_position, initial_velocity, time):
    G = 6.67430e-11  # Gravitational constant
    M = 5.972e24  # Mass of Earth

    r = initial_position
    v = initial_velocity
    t = time

    a = G * M / r**3
    omega = np.sqrt(a / r)

    x = r * np.cos(omega * t)
    y = r * np.sin(omega * t)
    vx = v * np.cos(omega * t)
    vy = v * np.sin(omega * t)

    return x, y, vx, vy
```
