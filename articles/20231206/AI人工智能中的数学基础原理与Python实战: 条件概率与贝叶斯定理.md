# 2.核心概念与联系

## 2.1条件概率

条件概率是概率论中的一个重要概念，它表示一个事件发生的概率，但是只考虑了另一个事件已经发生的情况。例如，如果我们知道一个人是男性，那么他的肺结核患者概率就会发生变化。条件概率可以表示为：

P(A|B) = P(A ∩ B) / P(B)

其中，P(A|B) 表示事件A发生的概率，已知事件B发生；P(A ∩ B) 表示事件A和事件B同时发生的概率；P(B) 表示事件B发生的概率。

## 2.2贝叶斯定理

贝叶斯定理是一种用于计算条件概率的数学公式，它可以帮助我们更好地理解事件之间的关系。贝叶斯定理可以表示为：

P(A|B) = P(B|A) * P(A) / P(B)

其中，P(A|B) 表示事件A发生的概率，已知事件B发生；P(B|A) 表示事件B发生的概率，已知事件A发生；P(A) 表示事件A发生的概率；P(B) 表示事件B发生的概率。

## 4.1条件概率实例

```python
import numpy as np

# 事件A和事件B的概率
P_A = 0.5
P_B = 0.6

# 事件A和事件B同时发生的概率
P_A_and_B = 0.3

# 计算条件概率
P_A_given_B = P_A_and_B / P_B

print("P(A|B) =", P_A_given_B)
```

在这个例子中，我们假设事件A和事件B的概率分别为0.5和0.6，事件A和事件B同时发生的概率为0.3。我们使用Python的NumPy库来计算条件概率。

## 4.2贝叶斯定理实例

```python
import numpy as np

# 事件A和事件B的概率
P_A = 0.5
P_B = 0.6

# 事件B发生的概率
P_B_given_A = 0.8

# 计算条件概率
P_A_given_B = P_B_given_A * P_A / P_B

print("P(A|B) =", P_A_given_B)
```

在这个例子中，我们假设事件A和事件B的概率分别为0.5和0.6，事件B发生的概率为0.8。我们使用Python的NumPy库来计算条件概率。
