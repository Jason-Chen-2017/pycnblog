1. 1956年，艾兹伯特·图灵（Alan Turing）提出了图灵测试，这是人工智能研究的起点。图灵测试是一种判断机器是否具有人类智能的方法，即如果一个人无法区分一个计算机和一个人类之间的对话，那么计算机就被认为具有人类智能。

2. 1960年代，人工智能研究开始兴起。在这一时期，人工智能研究者们主要关注的是如何让计算机模拟人类的思维过程，这一时期的人工智能研究主要集中在逻辑推理、知识表示和推理、自然语言处理等方面。

3. 1980年代，人工智能研究面临了一些挑战。这一时期的人工智能研究者们发现，让计算机模拟人类的思维过程并不是那么容易，因此人工智能研究开始转向机器学习和人工神经网络等方面。

4. 1990年代，人工智能研究开始取得了一些成功。这一时期的人工智能研究者们发现，让计算机学习从数据中提取信息并进行决策是一种有效的方法，因此人工智能研究开始集中在机器学习、数据挖掘和人工神经网络等方面。

5. 2000年代，人工智能研究取得了一些重大突破。这一时期的人工智能研究者们发现，让计算机理解自然语言、自主地决策以及与人类互动是人工智能的关键，因此人工智能研究开始集中在自然语言处理、机器学习和深度学习等方面。

6. 2010年代至今，人工智能研究正在不断发展。这一时期的人工智能研究者们发现，让计算机理解自然语言、自主地决策以及与人类互动是人工智能的关键，因此人工智能研究开始集中在自然语言处理、机器学习和深度学习等方面。

人工智能的发展历程表明，人工智能研究的目标是让计算机能够理解自然语言、学习从数据中提取信息、自主地决策以及与人类互动。人工智能的发展历程也表明，人工智能研究的方法和技术不断发展和进步，因此人工智能的发展历程是一个不断发展和进步的过程。
