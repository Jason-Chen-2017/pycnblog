# 2.核心概念与联系

在进入具体的框架设计原理之前，我们需要了解一些核心概念和它们之间的联系。这些概念包括：机器学习、人工智能、深度学习、神经网络、卷积神经网络、递归神经网络等。

机器学习是一种通过从数据中学习的方法，以便在未来的数据上进行预测或决策。机器学习可以分为监督学习、无监督学习和半监督学习三种类型。

人工智能是一种通过计算机程序模拟人类智能的技术。人工智能可以分为知识工程、机器学习和深度学习三种类型。

深度学习是一种通过神经网络模型进行机器学习的方法。深度学习可以进一步分为卷积神经网络、递归神经网络和自然语言处理等类型。

卷积神经网络是一种通过卷积层进行图像处理的深度学习模型。卷积神经网络可以应用于图像分类、目标检测、图像生成等任务。

递归神经网络是一种通过递归层进行序列处理的深度学习模型。递归神经网络可以应用于语音识别、文本生成、时间序列预测等任务。
