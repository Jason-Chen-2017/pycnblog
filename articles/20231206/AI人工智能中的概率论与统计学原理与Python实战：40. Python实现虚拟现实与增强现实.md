# 4.具体代码实例和详细解释说明

在这里，我们将提供一个简单的Python代码实例，用于生成随机点的分布：

```python
import numpy as np

# 生成随机点的数量
n = 1000

# 生成随机点的位置
x = np.random.normal(loc=0, scale=1, size=n)
y = np.random.normal(loc=0, scale=1, size=n)

# 绘制随机点的分布
import matplotlib.pyplot as plt
plt.scatter(x, y)
plt.show()
```

在这个代码实例中，我们首先使用`numpy`库生成了随机点的数量。然后，我们使用`np.random.normal`函数生成了随机点的位置，其中`loc`参数表示均值，`scale`参数表示标准差。最后，我们使用`matplotlib.pyplot`库绘制了随机点的分布。
