## 4.1 线性插值法的Python代码实例

```python
import numpy as np

# 定义已知的数据点
x = np.array([1, 2, 3, 4, 5])
y = np.array([1, 4, 9, 16, 25])

# 计算线性插值法的系数
m = (y[1] - y[0]) / (x[1] - x[0])
b = y[0] - m * x[0]

# 计算未知点的值
x_new = np.array([6, 7, 8, 9, 10])
y_new = m * x_new + b

print(y_new)
```

在这个代码实例中，我们首先定义了已知的数据点。然后，我们计算了线性插值法的系数。最后，我们使用这些系数来计算未知点的值。

## 4.2 多项式插值法的Python代码实例

```python
import numpy as np
from scipy.interpolate import lagrange

# 定义已知的数据点
x = np.array([1, 2, 3, 4, 5])

# 计算多项式插值法的系数

# 计算未知点的值

```
