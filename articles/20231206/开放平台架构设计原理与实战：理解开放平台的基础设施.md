# 2.核心概念与联系
在开发开放平台时，需要了解一些核心概念，如API、SDK、OAuth等。API（Application Programming Interface，应用程序编程接口）是开放平台与第三方应用之间的接口，用于实现数据交互和功能调用。SDK（Software Development Kit，软件开发工具包）是一种软件开发工具，包含了一系列的库和工具，帮助开发者更快地开发应用程序。OAuth是一种授权协议，用于允许用户授权第三方应用访问他们的资源。
