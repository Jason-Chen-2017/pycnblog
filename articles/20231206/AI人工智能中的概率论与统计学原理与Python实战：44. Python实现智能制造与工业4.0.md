## 3.1 贝叶斯定理
贝叶斯定理是概率论中的一个重要公理，用于计算条件概率。在智能制造与工业4.0中，我们可以利用贝叶斯定理来计算设备故障的概率。贝叶斯定理的公式为：

$$
P(A|B) = \frac{P(B|A) \times P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示条件概率，$P(B|A)$ 表示事件B发生的概率给事件A发生的条件下，$P(A)$ 表示事件A的概率，$P(B)$ 表示事件B的概率。

## 3.2 线性回归
线性回归是一种常用的统计学方法，用于预测一个变量的值。在智能制造与工业4.0中，我们可以利用线性回归来预测生产量、成本等变量的值。线性回归的公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

其中，$y$ 表示预测值，$x_1, x_2, ..., x_n$ 表示输入变量，$\beta_0, \beta_1, ..., \beta_n$ 表示回归系数，$\epsilon$ 表示误差。

## 3.3 支持向量机
支持向量机是一种常用的机器学习方法，用于分类和回归问题。在智能制造与工业4.0中，我们可以利用支持向量机来分类设备故障类型、优化生产流程等。支持向量机的公式为：

$$
f(x) = \text{sgn} \left( \sum_{i=1}^n \alpha_i y_i K(x_i, x) + b \right)
$$

其中，$f(x)$ 表示预测值，$x$ 表示输入变量，$y_i$ 表示标签，$K(x_i, x)$ 表示核函数，$\alpha_i$ 表示支持向量权重，$b$ 表示偏置。
