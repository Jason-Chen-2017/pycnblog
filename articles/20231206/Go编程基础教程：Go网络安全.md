## 3.3 数学模型公式详细讲解
在加密算法中，数学模型公式是加密算法的基础。以下是一些常见的加密算法的数学模型公式详细讲解：

1. AES加密：
   AES加密算法是一种基于替换、移位、选择和异或的加密算法。AES加密的数学模型公式如下：
   $$
   C = E_k(P) = P \oplus K
   $$
   其中，$C$ 是密文，$E_k$ 是加密函数，$P$ 是明文，$K$ 是密钥。
   AES加密的解密过程如下：
   $$
   P = D_k(C) = C \oplus K
   $$
   其中，$D_k$ 是解密函数。
2. RSA加密：
   RSA加密算法是一种基于数论的加密算法。RSA加密的数学模型公式如下：
   $$
   C = P^e \mod n
   $$
   其中，$C$ 是密文，$e$ 是公钥的指数，$P$ 是明文，$n$ 是公钥的模。
   RSA解密的数学模型公式如下：
   $$
   P = C^d \mod n
   $$
   其中，$d$ 是私钥的指数。
3. MD5加密：
   MD5加密算法是一种基于散列函数的加密算法。MD5加密的数学模型公式如下：
   $$
   H(M) = MD5(M)
   $$
   其中，$H$ 是哈希值，$M$ 是明文。

## 4.1 AES加密实例
以下是一个使用Go语言实现AES加密的代码实例：

```go
package main

import (
    "crypto/aes"
    "crypto/cipher"
    "crypto/rand"
    "encoding/base64"
    "fmt"
    "io"
)

func main() {
    key := []byte("1234567890abcdef")
    plaintext := []byte("Hello, World!")

    block, err := aes.NewCipher(key)
    if err != nil {
        panic(err)
    }

    ciphertext := make([]byte, aes.BlockSize+len(plaintext))
    iv := ciphertext[:aes.BlockSize]
    if _, err := io.ReadFull(rand.Reader, iv); err != nil {
        panic(err)
    }

    stream := cipher.NewCFBEncrypter(block, iv)
    stream.XORKeyStream(ciphertext[aes.BlockSize:], plaintext)

    fmt.Printf("Ciphertext: %x\n", ciphertext)
    fmt.Printf("Encrypted: %s\n", base64.StdEncoding.EncodeToString(ciphertext))
}
```

在上述代码中，我们首先定义了一个AES密钥和明文数据。然后，我们使用`aes.NewCipher`函数创建了一个AES加密对象。接着，我们使用`cipher.NewCFBEncrypter`函数创建了一个加密流对象，并使用密钥和初始向量进行加密操作。最后，我们将加密后的密文输出到控制台。

## 4.2 RSA加密实例
以下是一个使用Go语言实现RSA加密的代码实例：

```go
package main

import (
    "crypto/rand"
    "crypto/rsa"
    "crypto/x509"
    "encoding/pem"
    "fmt"
    "io/ioutil"
    "os"
)

func main() {
    privateKey, err := rsa.GenerateKey(rand.Reader, 2048)
    if err != nil {
        panic(err)
    }

    privatePEM := &pem.Block{
        Type:  "PRIVATE KEY",
        Bytes: x509.ExportPKCS1PrivateKey(privateKey),
    }

    err = ioutil.WriteFile("private.pem", pem.EncodeToMemory(privatePEM), 0600)
    if err != nil {
        panic(err)
    }

    publicKey := privateKey.PublicKey

    publicPEM := &pem.Block{
        Type:  "PUBLIC KEY",
        Bytes: x509.MarshalPKIXPublicKey(publicKey),
    }

    err = ioutil.WriteFile("public.pem", pem.EncodeToMemory(publicPEM), 0600)
    if err != nil {
        panic(err)
    }
}
```

在上述代码中，我们首先使用`rsa.GenerateKey`函数生成了一个RSA密钥对。然后，我们将私钥和公钥保存到文件中，以便在后续的解密操作中使用。

## 4.3 MD5加密实例
以下是一个使用Go语言实现MD5加密的代码实例：

```go
package main

import (
    "crypto/md5"
    "encoding/hex"
    "fmt"
    "io"
)

func main() {
    data := []byte("Hello, World!")

    hasher := md5.New()
    hasher.Write(data)

    fmt.Printf("Hash: %x\n", hasher.Sum(nil))
}
```

在上述代码中，我们首先定义了一个MD5哈希对象。然后，我们使用`Write`方法将明文数据写入哈希对象。最后，我们使用`Sum`方法获取哈希值，并将其输出到控制台。
