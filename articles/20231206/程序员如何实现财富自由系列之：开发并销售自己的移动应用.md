## 4.1 代码实例

我们将通过一个简单的计算器应用来详细解释如何开发移动应用的具体步骤。

```
import UIKit

class ViewController: UIViewController {

    @IBOutlet weak var resultLabel: UILabel!

    var result: Double = 0.0

    @IBAction func buttonPressed(_ sender: UIButton) {
        if let buttonText = sender.currentTitle {
            if buttonText == "=" {
                result = eval()
                resultLabel.text = String(result)
            } else {
                resultLabel.text = resultLabel.text! + buttonText
            }
        }
    }

    func eval() -> Double {
        let expression = resultLabel.text!
        let result = NSExpression(format: expression).expressionValue(withObject: nil, context: nil) as! Double
        return result
    }
}
```

在这个代码实例中，我们创建了一个简单的计算器应用。应用的界面包括一个结果标签，用于显示计算结果，以及一些按钮，用于输入数字和运算符。当用户点击等号按钮时，应用会计算表达式的结果并显示在结果标签中。

## 4.2 详细解释说明

在这个代码实例中，我们使用了Swift语言来开发计算器应用。首先，我们创建了一个ViewController类，它继承自UIViewController类。在ViewController类中，我们定义了一个resultLabel属性，用于存储计算结果，以及一个buttonPressed方法，用于处理按钮的点击事件。

在buttonPressed方法中，我们首先获取按钮的文本，然后判断按钮是否是等号按钮。如果是，我们调用eval方法来计算表达式的结果，并将结果显示在结果标签中。如果不是，我们将按钮文本添加到结果标签中。

在eval方法中，我们使用NSExpression类来计算表达式的结果。我们将表达式字符串传递给NSExpression的format参数，然后使用expressionValue方法来计算表达式的结果。最后，我们将结果返回给调用方。
