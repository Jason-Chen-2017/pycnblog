```rust
trait Sum {
    fn sum(&self) -> u32;
}

impl Sum for i32 {
    fn sum(&self) -> u32 {
        *self as u32
    }
}

impl Sum for u32 {
    fn sum(&self) -> u32 {
        *self
    }
}

fn main() {
    let nums = vec![1, 2, 3, 4, 5];
    let sum = nums.iter().sum();
    println!("The sum is: {}", sum);
}
```

在这个代码实例中，我们首先定义了一个trait `Sum`，它定义了一个名为`sum`的方法。然后，我们实现了`Sum`trait的两个实现，分别是`i32`和`u32`类型。最后，我们使用`iter()`方法创建了一个迭代器，并使用`sum()`方法计算了迭代器中所有元素的和。
