# 4.具体代码实例和详细解释说明
在开发软件工具时，可以通过编写代码来实现算法和功能。以下是一个简单的代码实例：

```python
def sort(arr):
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] > arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
sorted_arr = sort(arr)
print(sorted_arr)
```

在这个例子中，我们实现了一个简单的冒泡排序算法。首先，我们定义了一个名为 `sort` 的函数，该函数接受一个数组作为参数。然后，我们使用两层循环来比较数组中的每对相邻元素，并将它们进行交换，直到数组有序。最后，我们调用 `sort` 函数并将排序后的数组打印出来。
