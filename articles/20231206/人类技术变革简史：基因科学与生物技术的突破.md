# 2.核心概念与联系

在探讨基因科学与生物技术的突破之前，我们需要了解一些核心概念。

## 基因

基因是生物学中的一种遗传信息单位，它包含了生物体的遗传信息。基因由DNA（苷酸链）构成，DNA是生物体的遗传物质。每个基因都包含了一段特定的DNA序列，这个序列决定了基因所编码的特征。

## 遗传

遗传是生物体从父母中继承的特征。这些特征是通过基因传递的。每个生物体都有一定数量的基因，这些基因决定了生物体的特征，如颜色、身高、健康状况等。

## 基因组

基因组是一个生物体的所有基因的集合。基因组包含了生物体的所有遗传信息。人类基因组是一个例子，它包含了人类的所有基因。

## 基因工程

基因工程是一种生物技术，它允许人类修改生物体的基因组。通过基因工程，我们可以在基因组中添加、删除或修改基因，从而改变生物体的特征。

## 克隆

克隆是一种生物技术，它允许人类复制生物体。通过克隆，我们可以创建与原始生物体相同的生物体副本。

```python
def clone_gene(gene, num_copies):
    """
    Clone a gene.

    Args:
        gene: The gene to clone.
        num_copies: The number of copies to make.

    Returns:
        A list of cloned genes.
    """
    cloned_genes = []
    for _ in range(num_copies):
        cloned_genes.append(gene)
    return cloned_genes
```

这个函数接受一个基因和一个复制数量作为输入，并返回一个包含复制基因的列表。函数首先创建一个空列表，然后使用一个循环来复制基因。在每次循环中，函数将基因添加到列表中。最后，函数返回包含复制基因的列表。
