## 2.1 词向量

词向量是一种将词语表示为一个高维数学向量的方法，以便计算机可以对词语进行数学运算。词向量可以捕捉词语在文本中的语义和上下文信息，因此可以用于各种自然语言处理任务，如文本分类、情感分析、机器翻译等。

## 2.2 词汇表

词汇表是一种将词语映射到唯一标识符的数据结构，用于存储和管理词向量。词汇表可以将大量的词语映射到一个较小的向量空间中，从而减少计算复杂度和内存占用。

## 2.3 词嵌入

词嵌入是一种将词语表示为连续向量的方法，可以用于捕捉词语之间的语义关系。词嵌入可以通过使用神经网络来学习词语在文本中的上下文信息，从而更好地表示词语的相似性。

## 2.4 词性标注

词性标注是一种将词语映射到其语法类别的过程，用于捕捉词语在文本中的语法信息。词性标注可以用于各种自然语言处理任务，如命名实体识别、句法分析等。

## 2.5 命名实体识别

## 3.1 基于词频的统计方法

基于词频的统计方法是词向量技术的第一阶段，它通过计算词语在文本中的出现频率来表示词语的相似性。具体操作步骤如下：

1. 将文本分割为单词，并将每个单词映射到唯一的标识符。
2. 计算每个单词在文本中的出现频率。
3. 使用出现频率来表示词语的相似性。

数学模型公式为：

$$
\text{similarity}(w_1, w_2) = \frac{\text{count}(w_1, w_2)}{\text{count}(w_1) + \text{count}(w_2) - \text{count}(w_1, w_2)}
$$

其中，$\text{similarity}(w_1, w_2)$ 表示词语 $w_1$ 和 $w_2$ 之间的相似性，$\text{count}(w_1, w_2)$ 表示词语 $w_1$ 和 $w_2$ 在文本中的共同出现次数，$\text{count}(w_1)$ 和 $\text{count}(w_2)$ 表示词语 $w_1$ 和 $w_2$ 在文本中的出现次数。

## 3.2 基于上下文的统计方法

基于上下文的统计方法是词向量技术的第二阶段，它通过考虑词语在文本中的上下文来表示词语的相似性。具体操作步骤如下：

1. 将文本分割为单词，并将每个单词映射到唯一的标识符。
2. 计算每个单词在文本中的上下文信息。
3. 使用上下文信息来表示词语的相似性。

数学模型公式为：

$$
\text{similarity}(w_1, w_2) = \frac{\text{count}(w_1, w_2)}{\text{count}(w_1) + \text{count}(w_2) - \text{count}(w_1, w_2)}
$$

其中，$\text{similarity}(w_1, w_2)$ 表示词语 $w_1$ 和 $w_2$ 之间的相似性，$\text{count}(w_1, w_2)$ 表示词语 $w_1$ 和 $w_2$ 在文本中的共同出现次数，$\text{count}(w_1)$ 和 $\text{count}(w_2)$ 表示词语 $w_1$ 和 $w_2$ 在文本中的出现次数。

## 3.3 基于神经网络的词向量

## 4.1 基于词频的统计方法

```python
from collections import Counter

def word_frequency(text):
    words = text.split()
    word_count = Counter(words)
    return word_count

text = "I love you. You love me. We are family."
word_frequency(text)
```

## 4.2 基于上下文的统计方法

```python
from collections import defaultdict

def context_similarity(text):
    words = text.split()
    context = defaultdict(list)
    for word in words:
        for i in range(-2, 3):
            if i + len(word) >= 0 and i + len(word) < len(words):
                context[word].append(words[i + len(word)])
    return context

text = "I love you. You love me. We are family."
context_similarity(text)
```
