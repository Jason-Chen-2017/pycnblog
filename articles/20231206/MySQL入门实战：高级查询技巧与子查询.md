# 6.附录常见问题与解答

在使用MySQL的查询技巧和子查询时，可能会遇到一些常见问题。以下是一些常见问题及其解答：

1. 问题：如何使用LIMIT子句来限制查询结果的数量？

   解答：我们可以使用LIMIT子句来限制查询结果的数量，例如：

   ```sql
   SELECT * FROM employees LIMIT 10;
   ```

   在这个查询语句中，我们使用了LIMIT子句来限制查询结果的数量为10。

2. 问题：如何使用ORDER BY子句来排序查询结果？

   解答：我们可以使用ORDER BY子句来排序查询结果，例如：

   ```sql
   SELECT * FROM employees ORDER BY salary DESC;
   ```

   在这个查询语句中，我们使用了ORDER BY子句来排序查询结果，并指定了排序顺序为降序。

3. 问题：如何使用GROUP BY子句来分组查询结果？

   解答：我们可以使用GROUP BY子句来分组查询结果，例如：

   ```sql
   SELECT department, COUNT(*) FROM employees GROUP BY department;
   ```

   在这个查询语句中，我们使用了GROUP BY子句来分组查询结果，并计算每个部门的员工数量。
