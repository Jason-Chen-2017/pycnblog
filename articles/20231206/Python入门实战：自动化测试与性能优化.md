### 4.2 性能优化的具体代码实例

我们将通过一个简单的快速排序算法来进行性能优化。快速排序算法的主要思想是通过选择一个基准值，将数组分为两部分：一个大于基准值的部分和一个小于基准值的部分。然后递归地对这两部分进行排序。我们需要对快速排序算法进行优化，以提高其执行速度。

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

def optimized_quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return optimized_quick_sort(left) + middle + optimized_quick_sort(right)
```

在上述代码中，我们对快速排序算法进行了优化。优化的主要思想是将基准值选择为数组的中间元素，而不是随机选择一个元素。这样可以确保基准值的选择更加均匀，从而减少递归次数，提高执行速度。
