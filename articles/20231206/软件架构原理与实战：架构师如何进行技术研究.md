## 4.1 排序算法实例

例如，我们可以使用选择排序算法来对数据进行排序。选择排序算法的实现如下：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if arr[min_idx] > arr[j]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr
```

## 4.2 搜索算法实例

例如，我们可以使用二分搜索算法来查找数据。二分搜索算法的实现如下：

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

def adjacency_matrix(graph):
    n = len(graph)
    matrix = [[0] * n for _ in range(n)]
    for u in range(n):
