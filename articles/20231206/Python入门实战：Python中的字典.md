# 2.核心概念与联系

字典是一种数据结构，它可以存储键值对。在Python中，字典使用大括号{}来表示。每个键值对由冒号(:)分隔。例如，我们可以创建一个字典，将名称“John”映射到年龄25：

```python
person = {"name": "John", "age": 25}
```

在这个例子中，“name”和“age”是键，而“John”和25是相应的值。我们可以通过键来访问字典中的值。例如，我们可以通过`person["name"]`来获取名称“John”。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

Python字典的实现基于哈希表，哈希表是一种数据结构，它将键映射到其对应的值。哈希表使用哈希函数将键转换为索引，以便快速访问值。哈希函数将键转换为一个整数，该整数用于索引数组。这种转换使得我们可以在平均情况下以O(1)的时间复杂度访问值。

Python字典的实现使用了开放地址法（Open Addressing）来解决哈希冲突。当两个不同的键映射到相同的索引时，开放地址法会在数组中找到下一个空闲的索引，将值存储在那里。这种方法确保了字典的查找、插入和删除操作的平均时间复杂度为O(1)。

1.创建字典：我们可以使用大括号{}来创建一个空字典。例如，`person = {}`。

2.添加键值对：我们可以使用`[]`来添加键值对。例如，`person["name"] = "John"`。

3.访问值：我们可以使用`[]`来访问字典中的值。例如，`person["name"]`将返回“John”。

4.修改值：我们可以使用`[]`来修改字典中的值。例如，`person["name"] = "Jane"`。

5.删除键值对：我们可以使用`del`来删除字典中的键值对。例如，`del person["name"]`。
