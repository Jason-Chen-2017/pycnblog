## 6.2 问题2：如何处理缺失值？

答案：缺失值可以通过删除、填充、插值等方法进行处理。删除方法是直接删除缺失值的行或列，但是可能导致数据损失。填充方法是使用均值、中位数等统计值填充缺失值，但是可能导致数据偏差。插值方法是使用相邻值进行插值填充缺失值，但是可能导致数据不连续。
