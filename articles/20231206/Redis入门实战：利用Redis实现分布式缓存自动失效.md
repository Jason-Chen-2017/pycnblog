- Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。
- Redis 提供了多种语言的 API，如：Ruby、Python、Java、C、C++、PHP、Node.js、Go、Objective-C、C#、Perl 和 Lua。
- Redis 还支持发布/订阅(Pub/Sub)、监视(Watch)、事务(Transactions)和 Lua 脚本。
