# 2.核心概念与联系
在概率图模型中，我们主要关注的是随机变量之间的关系。概率图模型是一种图形模型，其中每个节点表示一个随机变量，每条边表示两个随机变量之间的关系。这种模型可以用来描述许多复杂的关系，例如人们之间的关系、物品之间的关系等。

概率图模型的核心概念包括：

1.随机变量：随机变量是一个可以取不同值的变量，其值的分布是随机的。

2.条件概率：条件概率是一个随机变量给定某个值时，另一个随机变量取值的概率。

3.条件独立性：条件独立性是指给定某个随机变量的值，另一个随机变量的值与该随机变量的值之间是独立的。

4.概率图：概率图是一个有向图，其中每个节点表示一个随机变量，每条边表示两个随机变量之间的关系。

5.贝叶斯定理：贝叶斯定理是概率图模型中的一个重要公式，用来计算条件概率。

1.计算条件概率：给定一个随机变量X的值x，我们可以计算另一个随机变量Y的条件概率P(Y|X=x)。这可以通过贝叶斯定理来计算：

$$
P(Y|X=x) = \frac{P(X=x|Y)P(Y)}{P(X=x)}
$$

2.贝叶斯定理：贝叶斯定理是概率图模型中的一个重要公式，用来计算条件概率。它可以表示为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}

3.计算条件独立性：给定一个随机变量X的值x，我们可以计算另一个随机变量Y是否与X独立。如果P(Y|X=x) = P(Y)，则Y与X是独立的。

4.概率图的构建：我们可以通过构建一个概率图来描述随机变量之间的关系。每个节点表示一个随机变量，每条边表示两个随机变量之间的关系。

    P_Y_given_X = np.sum(X * Y) / len(X)

    P_X_given_Y = np.sum(X * Y) / np.sum(Y)

    P_Y = np.mean(Y)

    else:
        return False
