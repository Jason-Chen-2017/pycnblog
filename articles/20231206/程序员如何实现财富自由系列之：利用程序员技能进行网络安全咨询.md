```python
import hashlib

def sha256_encrypt(data):
    sha256 = hashlib.sha256()
    sha256.update(data.encode('utf-8'))
    return sha256.hexdigest()

def rsa_encrypt(data, public_key):
    rsa = public_key.encrypt(data.encode('utf-8'), 65537)
    return rsa

def rsa_decrypt(data, private_key):
    rsa = private_key.decrypt(data, 65537)
    return rsa.decode('utf-8')
```

上述代码实例中，我们使用Python语言实现了SHA-256加密和RSA加密解密功能。SHA-256是一种常用的数字签名算法，RSA是一种常用的非对称加密算法。
