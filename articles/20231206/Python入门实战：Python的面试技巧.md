# 4.具体代码实例和详细解释说明
Python的面试技巧需要掌握一些具体的代码实例，如列表、字典、函数、类、模块等。以下是一些具体的代码实例和解释：

```python
# 列表
list1 = [1, 2, 3, 4, 5]
print(list1)

# 字典
dict1 = {"name": "John", "age": 30, "city": "New York"}
print(dict1)

# 函数
def greet(name):
    print("Hello, " + name)

greet("John")

# 类
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

person1 = Person("John", 30)
print(person1.name)

# 模块
import math
print(math.sqrt(16))
```
