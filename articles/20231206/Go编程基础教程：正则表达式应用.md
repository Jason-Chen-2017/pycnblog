# 4.具体代码实例和详细解释说明

在Go语言中，可以使用`regexp`包来实现正则表达式的功能。`regexp`包提供了对正则表达式的支持，可以用来实现字符串的匹配、替换、验证等操作。

以下是一个Go语言中使用正则表达式的简单示例：

```go
package main

import (
	"fmt"
	"regexp"
)

func main() {
	// 定义一个正则表达式
	pattern := `\d{3}-\d{2}-\d{4}`

	// 创建一个正则表达式对象
	regex := regexp.MustCompile(pattern)

	// 定义一个要匹配的字符串
	input := "123-45-6789"

	// 使用正则表达式对字符串进行匹配
	matches := regex.FindAllString(input, -1)

	// 输出匹配结果
	fmt.Println(matches)
}
```

在上面的示例中，我们首先定义了一个正则表达式`\d{3}-\d{2}-\d{4}`，它表示匹配三个数字、一个横杠、两个数字、一个横杠、四个数字的字符串。然后我们创建了一个正则表达式对象`regex`，并使用`FindAllString`方法对字符串进行匹配。最后，我们输出了匹配结果。
