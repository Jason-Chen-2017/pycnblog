在本节中，我们将通过一个简单的条形图的例子来解释数据可视化的具体代码实例。

```python
import matplotlib.pyplot as plt

# 创建一个条形图
plt.bar([1, 2, 3, 4, 5], [5, 10, 15, 20, 25])

# 设置图表标题和坐标轴标签
plt.title('Simple Bar Chart')
```

在这个例子中，我们使用了matplotlib库来创建一个简单的条形图。我们首先导入了matplotlib.pyplot模块，然后使用plt.bar()函数来创建一个条形图。plt.bar()函数接受两个参数：x和height。x参数是条形图的x坐标，height参数是条形图的高度。然后我们使用plt.title()、plt.xlabel()和plt.ylabel()函数来设置图表标题和坐标轴标签。最后，我们使用plt.show()函数来显示图表。

## 4.2 数据探索的具体代码实例

在本节中，我们将通过一个简单的数据清洗的例子来解释数据探索的具体代码实例。

```python
import pandas as pd

# 创建一个数据框
data = {'name': ['John', 'Jane', 'Alice', 'Bob'],
        'age': [20, 25, 30, 35],
        'gender': ['M', 'F', 'F', 'M']}
df = pd.DataFrame(data)

# 数据清洗
df['age'] = df['age'].astype(int)
df['gender'] = df['gender'].astype('category')

# 显示数据框
print(df)
```

在这个例子中，我们使用了pandas库来创建一个数据框。我们首先导入了pandas模块，然后使用pd.DataFrame()函数来创建一个数据框。data参数是一个字典，包含数据框的列名和值。然后我们使用df['age'].astype(int)来将'age'列转换为整数类型，使用df['gender'].astype('category')来将'gender'列转换为类别类型。最后，我们使用print()函数来显示数据框。
