### 4.4.决策规则构建

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 提取决策规则
rules = []
for index, row in data.iterrows():
    if row['age'] < 30:
        rules.append('If age is less than 30, then income is high.')
    else:
        rules.append('If age is 30 or more, then income is low.')

# 构建决策规则
from sklearn.feature_extraction.text import CountVectorizer

vectorizer = CountVectorizer()
X = vectorizer.fit_transform(rules)

from sklearn.naive_bayes import MultinomialNB

model = MultinomialNB()
model.fit(X, data['income'])
```

    occupation = strategy['occupation']
    return income, occupation
