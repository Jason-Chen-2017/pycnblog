                 

# 1.背景介绍

Redis是一个开源的高性能key-value存储系统，它支持数据的持久化，备份，重plication，集群等特性。Redis 支持各种类型的数据结构，如字符串(string)、哈希(hash)、列表(list)、集合(set)和有序集合(sorted set)等。Redis 还支持publish/subscribe、定时任务等功能。

Redis 是一个非关系型数据库，它的数据结构简单，性能出色，适合做缓存。Redis 的数据是在内存中存储的，所以它的读写速度非常快，远快于磁盘IO。Redis 的数据是持久化的，当Redis服务重启的时候，数据不会丢失。Redis 支持主从复制，即主节点可以同时为多个从节点提供数据，从节点是只读的。Redis 支持集群，即多个Redis节点组成一个集群，集群中的节点可以相互通信，实现数据的分布式存储和计算。
