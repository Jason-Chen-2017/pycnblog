# 2.核心概念与联系

在分布式缓存中，缓存预热是指在应用程序启动时，将缓存中的数据预先加载到内存中，以便在实际访问时，可以快速获取数据。缓存预热的主要目的是提高应用程序的性能和可用性。

缓存预热的策略主要包括：

- 预热策略：预热策略是指在应用程序启动时，如何选择要预热的数据。常见的预热策略有随机预热、热点预热等。
- 预热方法：预热方法是指在应用程序启动时，如何将数据加载到缓存中。常见的预热方法有批量预热、逐步预热等。
- 预热时间：预热时间是指应用程序启动时，预热数据所需的时间。预热时间可以通过调整预热策略和预热方法来优化。
