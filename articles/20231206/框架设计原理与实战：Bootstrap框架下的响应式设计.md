# 2.核心概念与联系

## 2.1响应式设计的核心概念

响应式设计是一种网页设计方法，它可以根据不同的设备和屏幕尺寸自动调整网页的布局和样式。这种设计方法的核心概念有以下几点：

1.流式布局：通过使用相对单位（如%、em、rem等）来设计网页布局，使得网页在不同的屏幕尺寸下都能保持良好的布局效果。

2.媒体查询：通过使用CSS媒体查询，可以根据用户的设备和屏幕尺寸来设置不同的样式规则。这样，当用户在不同的设备上访问网页时，网页会根据设备的特点自动调整布局和样式。

3.响应式图片：通过使用CSS的background-size属性和图片的原始尺寸信息，可以实现图片在不同设备和屏幕尺寸下的自适应显示。

## 2.2Bootstrap框架的核心概念

Bootstrap是一个开源的HTML、CSS和JavaScript框架，它提供了许多有用的组件和工具，帮助我们更快地开发响应式网页。Bootstrap的核心概念有以下几点：

1.基本组件：Bootstrap提供了许多基本的HTML和CSS组件，如按钮、表单、导航栏等，这些组件可以帮助我们快速构建网页布局和交互。

2.响应式布局：Bootstrap使用流式布局和媒体查询来实现响应式设计。通过使用Bootstrap的类和工具，我们可以轻松地实现不同设备和屏幕尺寸下的布局调整。

3.JavaScript扩展：Bootstrap提供了许多JavaScript扩展，如模态框、滚动条、拖拽等，这些扩展可以帮助我们实现更丰富的交互效果。
