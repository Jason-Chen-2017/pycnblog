### 4.1 一致性哈希实现

我们可以通过以下代码来实现一致性哈希：

```python
import hashlib

def hash(x):
    return hashlib.md5(x.encode('utf-8')).hexdigest()

def consistent_hash(x, servers):
    p = sum(hash(s) for s in servers)
    q = hash(x)
    return (p + q) % len(servers)

servers = ['server1', 'server2', 'server3']
x = 'data'

result = consistent_hash(x, servers)
print(result)
```

在上述代码中，我们首先导入了 hashlib 模块，然后定义了 hash 函数，用于计算哈希值。接着，我们定义了 consistent_hash 函数，用于实现一致性哈希。最后，我们通过将数据和服务器列表作为参数传递给 consistent_hash 函数，从而得到数据所在的服务器。
