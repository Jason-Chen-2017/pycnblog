## 1.1 数组的基本概念

数组是一种线性数据结构，它可以存储一组相同类型的元素。数组的元素可以通过下标（索引）进行访问和修改。数组的长度是固定的，一旦创建，就不能更改。

数组的基本概念可以通过以下几个核心概念来理解：

- 数组的元素类型：数组可以存储不同类型的元素，例如整数、字符串、对象等。
- 数组的长度：数组的长度是指数组中元素的个数。
- 数组的下标：数组的下标是从0开始的，用于唯一标识数组中的元素。
- 数组的访问和修改：通过下标，我们可以访问和修改数组中的元素。

## 1.2 数组的应用场景

数组的应用场景非常广泛，包括但不限于：

- 存储和处理大量相同类型的数据，例如数组可以用于存储学生成绩、员工工资等。
- 实现排序和搜索算法，例如数组可以用于实现冒泡排序、二分搜索等。
- 实现数据结构和算法，例如数组可以用于实现栈、队列、链表等。

## 1.7 数组的代码实例

以下是一个简单的数组代码实例：

```java
public class ArrayExample {
    public static void main(String[] args) {
        // 创建一个整数数组，长度为5
        int[] array = new int[5];

        // 访问数组元素
        int element = array[0];
        System.out.println("数组元素：" + element);

        // 修改数组元素
        element = 10;
        array[0] = element;
        System.out.println("修改后的数组元素：" + array[0]);

        // 删除数组元素
        array[0] = 0;
        System.out.println("删除后的数组元素：" + array[0]);

        // 插入数组元素
        array[0] = 100;
        System.out.println("插入后的数组元素：" + array[0]);

        // 排序数组元素
        Arrays.sort(array);
        System.out.println("排序后的数组元素：" + Arrays.toString(array));

        // 搜索数组元素
        int searchElement = 100;
        boolean found = false;
        for (int i = 0; i < array.length; i++) {
            if (array[i] == searchElement) {
                found = true;
                break;
            }
        }
        System.out.println("搜索结果：" + (found ? "找到" : "未找到") + " " + searchElement);
    }
}
```

在上面的代码实例中，我们创建了一个整数数组，并对数组进行访问、修改、删除、插入、排序和搜索等操作。
