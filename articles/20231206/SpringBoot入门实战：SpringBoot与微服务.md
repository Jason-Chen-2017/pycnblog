Spring Boot 是一个用于构建原生的 Spring 应用程序的快速开始点和 PaaS 平台。Spring Boot 旨在减少开发人员通过冗长的配置来配置 Spring 应用程序的时间和精力，从而使开发人员能够快速使用 Spring 构建原生的应用程序。

Spring Boot 的核心概念是“自动配置”，它通过使用 Spring 的依赖注入和组件扫描来自动配置 Spring 应用程序。这意味着开发人员不需要编写 XML 配置文件来配置 Spring 应用程序，而是可以通过简单的 Java 代码来配置 Spring 应用程序。
