## 3.1 概率论基础

概率论是一门研究不确定性的学科，它可以帮助我们更好地理解和处理不确定性。在概率论中，我们需要了解一些基本概念，如事件、样本空间、概率等。

### 3.1.1 事件

事件是概率论中的一个基本概念，它是一个可能发生或不发生的结果。例如，在一个硬币投掷实验中，事件可以是硬币正面或反面。

### 3.1.2 样本空间

样本空间是概率论中的一个基本概念，它是所有可能发生的事件集合。例如，在一个硬币投掷实验中，样本空间是{正面，反面}。

### 3.1.3 概率

概率是概率论中的一个基本概念，它是一个事件发生的可能性。概率通常用P表示，它的值范围在0到1之间。例如，在一个硬币投掷实验中，正面的概率是1/2，反面的概率也是1/2。

## 3.2 统计学基础

统计学是一门研究数据的学科，它可以帮助我们更好地理解和处理数据。在统计学中，我们需要了解一些基本概念，如样本、参数、统计量等。

### 3.2.1 样本

样本是统计学中的一个基本概念，它是从总体中随机抽取的一部分。例如，如果我们想要了解一个城市的平均年龄，我们可以从该城市的所有人中随机抽取一部分人来计算平均年龄。

### 3.2.2 参数

参数是统计学中的一个基本概念，它是一个总体的某个特征值。例如，在一个城市的平均年龄问题中，平均年龄就是一个参数。

### 3.2.3 统计量

统计量是统计学中的一个基本概念，它是一个样本的某个特征值。例如，在一个城市的平均年龄问题中，我们可以计算样本中的平均年龄，这就是一个统计量。

## 3.3 概率论与统计学的应用

在人工智能中，我们可以使用概率论与统计学来处理数据，进行预测和决策。例如，我们可以使用概率论来计算某个事件的可能性，使用统计学来计算某个参数的估计值。

## 4.1 概率论实例

### 4.1.1 硬币投掷实验

在这个实例中，我们将使用Python计算硬币投掷实验的概率。

```python
import random

def coin_toss(n):
    heads = 0
    tails = 0
    for _ in range(n):
        if random.random() < 0.5:
            heads += 1
        else:
            tails += 1
    return heads / n

print(coin_toss(1000))
```

在这个实例中，我们使用Python的random模块来生成随机数，然后计算硬币正面和反面的概率。

### 4.1.2 二项分布

在这个实例中，我们将使用Python计算二项分布的概率。

```python
import math

def binomial_probability(n, k, p):
    return math.comb(n, k) * (p ** k) * ((1 - p) ** (n - k))

print(binomial_probability(10, 5, 0.5))
```

在这个实例中，我们使用Python的math模块来计算二项分布的概率。
