$$

# 4.具体代码实例和详细解释说明
在这里，我们将给出一个具体的代码实例，以及对其的详细解释说明。

```python
import mysql.connector

# 创建数据库连接
db_connection = mysql.connector.connect(
    host="localhost",
    user="root",
    password="password",
    database="electronic_commerce"
)

# 创建数据库表
cursor = db_connection.cursor()
sql = """
CREATE TABLE IF NOT EXISTS products (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    price DECIMAL(10,2) NOT NULL,
    stock INT NOT NULL
)
"""
cursor.execute(sql)

# 插入数据
sql = "INSERT INTO products (name, price, stock) VALUES (%s, %s, %s)"
values = ("iPhone 11", 6999.00, 100)
cursor.execute(sql, values)

# 查询数据
sql = "SELECT * FROM products WHERE name = %s"
name = "iPhone 11"
cursor.execute(sql, (name,))
result = cursor.fetchone()
print(result)

# 更新数据
sql = "UPDATE products SET price = %s WHERE name = %s"
new_price = 7999.00
name = "iPhone 11"
cursor.execute(sql, (new_price, name))

# 删除数据
sql = "DELETE FROM products WHERE name = %s"
name = "iPhone 11"
cursor.execute(sql, (name,))

# 提交事务
db_connection.commit()

# 关闭数据库连接
db_connection.close()
```

在这个代码实例中，我们使用Python的mysql-connector库来连接MySQL数据库，并创建了一个名为`products`的表。我们插入了一条数据，然后查询了这条数据，接着更新了这条数据，最后删除了这条数据。整个过程中，我们使用了事务来确保数据的一致性。
