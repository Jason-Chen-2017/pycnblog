- 当收件人的邮箱地址为“example@example.com”时，发送一封邮件。
- 当收件人的邮箱地址为“example2@example.com”时，发送一封邮件。

代码实例如下：

```python
import smtplib

def send_email(recipient, subject, message):
    server = smtplib.SMTP('smtp.example.com')
    server.sendmail('sender@example.com', recipient, message)
    server.quit()

def main():
    recipient = 'example@example.com'
    subject = 'Hello'
    message = 'Hello, World!'
    send_email(recipient, subject, message)

if __name__ == '__main__':
    main()
```

在这个代码实例中，我们首先定义了一个`send_email`函数，用于发送邮件。然后，我们定义了一个`main`函数，用于触发邮件发送。最后，我们调用`main`函数来执行邮件发送任务。
