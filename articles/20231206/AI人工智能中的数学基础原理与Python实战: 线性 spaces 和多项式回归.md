## 2.1 线性空间

线性空间的一个重要特征是它的向量之间存在线性关系。这意味着，如果我们有两个向量，那么它们之间的关系可以用一个线性方程来表示。例如，如果我们有两个向量a和b，那么它们之间的关系可以用一个线性方程ax+by=c来表示，其中a和b是常数，c是一个数。

多项式回归是一种预测方法，它可以用来预测一个变量的值，根据其他变量的值。多项式回归是一种特殊类型的回归分析，它使用多项式函数来模拟数据的变化趋势。

多项式回归的核心思想是，通过将多个变量的值进行组合，我们可以更好地预测一个变量的值。例如，如果我们有一个变量x，我们可以通过将其与一个多项式函数进行组合来预测它的值。这个多项式函数可以是一个简单的二次函数，也可以是一个更复杂的多项式函数。

线性空间的基本操作包括向量的加法、乘法和线性组合。

- 向量的乘法：给定一个向量a和一个数k，它们的乘积可以通过将每个元素相乘来计算。例如，如果a=(a1,a2,...,an)和k，那么它们的乘积可以表示为(ka1,ka2,...,kan)。

- 线性组合：给定一个向量a和一个数k，我们可以通过将a和k相乘来得到一个新的向量。例如，如果a=(a1,a2,...,an)和k，那么它们的线性组合可以表示为(ka1,ka2,...,kan)。

### 3.1.3 数学模型公式

线性空间的数学模型可以通过以下公式来表示：

$$
ax+by=c

其中a和b是常数，c是一个数。

## 4.2 多项式回归

### 4.2.1 代码实例

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 定义输入变量X和输出变量Y
X = np.array([[1, 2], [2, 3], [3, 4]])
Y = np.array([3, 5, 7])

# 构建多项式回归模型
model = LinearRegression()
model.fit(X, Y)

# 预测输出变量的值
predicted_Y = model.predict(X)
print(predicted_Y)
```

### 4.2.2 解释说明

在这个代码实例中，我们首先导入了numpy库和sklearn库中的LinearRegression类。然后，我们定义了我们的输入变量X和输出变量Y。接下来，我们构建了我们的多项式回归模型，并使用fit方法来训练模型。最后，我们使用predict方法来预测我们的输出变量的值，并将结果打印出来。
