# 2.核心概念与联系

长短时记忆网络（LSTM）是一种特殊类型的递归神经网络（RNN），它可以处理长期依赖性问题，并在许多任务中取得了显著的成功。LSTM的核心概念是通过使用门（gate）机制来控制信息的流动，从而避免了传统RNN中的梯度消失和梯度爆炸问题。LSTM的主要组成部分包括输入门（input gate）、遗忘门（forget gate）和输出门（output gate），以及一个长期记忆单元（long-term memory cell）。
