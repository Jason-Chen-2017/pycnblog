# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来说明禅意面向对象设计的具体操作步骤。

例子：编写一个程序，输出1到100之间的所有奇数。

具体操作步骤如下：

1. 分析问题：我们需要输出1到100之间的所有奇数。
2. 抽象：我们可以将这个问题抽象成“输出1到n之间的所有奇数”的问题。
3. 模块化：我们可以将这个问题拆分成两个模块：“输出1到n之间的所有奇数”和“输出1到n之间的所有偶数”。
4. 简洁：我们可以将这个问题简化成“输出1到n之间的所有数”的问题。

具体代码实例：

```python
def print_odd_numbers(n):
    for i in range(1, n+1):
        if i % 2 != 0:
            print(i)

print_odd_numbers(100)
```
