## 4.1 实例背景

假设我们需要开发一个简单的温度转换程序，该程序可以将摄氏温度转换为华氏温度，并将华氏温度转换为摄氏温度。

## 4.2 实例代码

```python
# 摄氏温度转华氏温度
def celsius_to_fahrenheit(celsius):
    return (celsius * 9/5) + 32

# 华氏温度转摄氏温度
def fahrenheit_to_celsius(fahrenheit):
    return (fahrenheit - 32) * 5/9

# 主程序
if __name__ == '__main__':
    celsius = float(input('请输入摄氏温度：'))
    fahrenheit = celsius_to_fahrenheit(celsius)
    print('摄氏温度为：{:.2f}°C'.format(celsius))
    print('华氏温度为：{:.2f}°F'.format(fahrenheit))
```

## 4.3 实例解释说明

1. 定义摄氏温度转华氏温度的函数 `celsius_to_fahrenheit`，该函数接收摄氏温度作为参数，并将其转换为华氏温度。
2. 定义华氏温度转摄氏温度的函数 `fahrenheit_to_celsius`，该函数接收华氏温度作为参数，并将其转换为摄氏温度。
3. 在主程序中，使用 `input` 函数接收用户输入的摄氏温度，并将其转换为华氏温度。
4. 使用 `print` 函数输出摄氏温度和华氏温度的结果。
