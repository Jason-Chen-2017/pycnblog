3. 在`main`测试文件中，使用`testing.T`对象来检查测试结果。

以下是一个简单的测试示例：

```go
package main

import (
    "testing"
)

func TestAdd(t *testing.T) {
    result := Add(1, 2)
    if result != 3 {
        t.Errorf("Expected 3, got %d", result)
    }
}

func Add(a, b int) int {
    return a + b
}
```

在这个示例中，我们定义了一个名为`TestAdd`的测试用例，用于测试`Add`函数的正确性。我们使用`t.Errorf`函数来检查测试结果，如果测试结果不符合预期，我们将输出一个错误信息。
