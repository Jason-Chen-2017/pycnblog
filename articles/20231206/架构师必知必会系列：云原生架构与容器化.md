# 4.具体代码实例和详细解释说明

在本节中，我们将提供一个具体的容器化应用程序的代码实例，并详细解释其中的每个步骤。

## 4.1 创建Dockerfile文件

```
FROM python:3.7

WORKDIR /app

COPY requirements.txt .

RUN pip install -r requirements.txt

COPY . .

EXPOSE 8000

CMD ["python", "app.py"]
```

在这个Dockerfile文件中，我们首先指定了基础镜像为Python 3.7，然后设置了工作目录为/app。接着，我们将requirements.txt文件复制到容器内，并使用pip命令安装所有依赖关系。最后，我们将应用程序的源代码复制到容器内，并指定了启动命令为python app.py。

## 4.2 构建Docker镜像

```
docker build -t my-app .
```

在这个命令中，我们使用docker build命令构建Docker镜像，-t参数用于指定镜像的名称，my-app是镜像的名称，.是指定Dockerfile文件所在的目录。

## 4.3 运行Docker容器

```
docker run -p 8000:8000 my-app
```

在这个命令中，我们使用docker run命令运行Docker容器，-p参数用于指定容器内部的端口与主机端口的映射关系，8000:8000表示容器内部的8000端口映射到主机的8000端口。my-app是镜像的名称。
