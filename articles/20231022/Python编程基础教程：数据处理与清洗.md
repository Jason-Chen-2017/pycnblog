# 1.背景介绍


数据处理与清洗是指从原始数据中提取有效信息，转换成便于分析、建模等使用的形式，并对数据进行缺失值处理、异常值处理、同质性检测、重复值处理等预处理工作的过程。数据清洗的目标是使数据准确可靠，并且准备好用于后续分析工作。对于每一个数据科学家或分析人员来说，无论其是否熟悉数据处理和数据清洗技能，都是一个必备的技能。本教程将以Python语言为例，带领读者进行数据处理与清洗的基本知识学习与实践。
