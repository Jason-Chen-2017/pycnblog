# 1.背景介绍


## 什么是量子热力学？
量子热力学（quantum heat physics）是利用量子现象进行热力学的实验研究领域，属于宇宙学、化学、材料科学、工程学等多个学术领域。其研究对象是涉及电子、原子或分子行为的微观粒子，这些微观粒子在量子场中存在，随着温度的升高，它们不断膨胀或者收缩，从而形成各种状态，而宏观热力学则通过温度曲线展示宇宙微观物体的状态。
量子热力学所研究的内容包括：
* 热力学定律的普遍性质（温度传导、扩散、热容）是否依然适用于量子态？
* 在量子气体中，为什么可以保持恒定湍流速率而熵却不增加？
* 为什么量子运动不能像经典运动那样自然地转化为热运动？
* 为什么量子有极限热运动速度而无宇宙常数热容？
* 量子系统中的量子粒子之间是否存在相互作用的机制？如果有，该如何描述？
## 为什么需要量子热力学？
对于应用型计算平台，如量子计算机、纳米器件、智能手机等，对高性能、高效率、低功耗具有重要意义。而量子计算机需要高性能处理能力，还需要解决量子计算过程中遇到的各种量子效应，包括碰撞失误、测量误差、态矢关联等等，而这些量子效应可以归结到量子热力学的研究领域。因此，本文将介绍量子热力学与其相关的研究领域，并展望其在未来的发展方向。
# 2.核心概念与联系
## 热量与状态
热量是一个广泛的概念，它既可以指热，也可以指能量。在这里，我们将讨论热力学里的热量，即物质的特定状态下的所存储的能量。
在真空中，所有的物质都处于一种平衡状态，没有任何热量。当外界加上一定温度时，液体会逐渐变凝聚，表面发生融合现象，产生热量。若两物体（一方为固体，另一方为液体），它们具有相同的密度，且两物体都处于同一温度水平下，则两物体之间的摩擦力较小，不会使两物体界面产生热量。但如果两物体之间存在某种形式的潜在的相互作用，例如金属与氧气之间的相互作用，则两物体之间的摩擦力增大，可能导致他们之间的摩擦产生热量。这一过程又称为热传导（thermal diffusion）。
我们将真空条件下，两物体之间的温度差设置为$\Delta T$，即$T_0-T=T_0+\Delta T$。则$dQ/dT=\rho C_p\cdot \frac{q}{\sigma} \frac{\partial T}{\partial x}=k_B \frac{q}{\sigma}$。其中，$q$是能量，$\rho$是混合比，$C_p$是比热容，$\sigma$是介电常数，$x$代表距离。由此可知，当$\Delta T$足够大时，物体能够产生热量。但当$\Delta T$增大时，物体的状态也随之变化。这就是物质的不同状态之间的区别。
## 费米-狄拉克定律
费米-狄拉克定律（Fermi-Dirac equation）是热力学的一个基本定律。它描述了在一个封闭容器里面，具有热运动的微观粒子（费米子）的平均能量是由这个容器中所含粒子总数所决定的。
假设有一个封闭容器，里面存在两个等温的物质，大小分别为$a$和$b$，底部为$h$。上面的定理告诉我们，由宏观物理学来说，实际上只有一个温度$\bar{T}$。而在量子热力学的描述里，我们考虑的是一个带有微扰的系统，所以我们只能说到达某一个温度时，费米子的平均能量。如果是均匀分布，那么$\frac{a}{h}\left(\frac{\partial}{\partial T}\ln(e^{\frac{-\beta Q_{F}}{k_BT}}) + \frac{Q_F}{k_BT}\right)$，其中$\beta = \frac{1}{k_BT}$, $Q_F$为频率，则有$P(T) = e^{\frac{-\beta a^2 Q_F}{k_BT}}$。也就是说，宏观上平均有$a$个费米子和$b$个费米子，而在量子的描述下，我们只能考虑这种实际上的情况。
## 量子概率分布函数
在一个定态的量子系统中，所有可能的取值构成了一个希尔伯特空间。该希尔伯特空间的每个元素称为态矢，态矢可表示为一组复数$\alpha=(\alpha_r,\alpha_i)^T$，其中$\alpha_r$为实部，$\alpha_i$为虚部，称为本征振幅。态矢的希尔伯特空间的基是有限的，每个态矢都是基向量的线性组合，并且线性无关。每个态矢对应一个波函数，即一个矩阵元素，用来表示系统处于相应态矢的概率。量子概率分布函数（quantum probability distribution function）简记为$|\psi\rangle$。
假设系统处于态矢$\psi=\sqrt{\frac{m_\text{eff}}{\pi}}\sum_{\mu=-N}^{N}c_\mu e^{i\phi_\mu}$，其中$c_\mu$为概率分量，$\phi_\mu$为相位角。则该态对应的概率分布函数为
$$\langle \psi | \psi \rangle = |\left\lvert c_\mu \right\rangle|^2 = \left|\frac{m_\text{eff}}{\pi}\sum_{\mu=-N}^{N}|c_\mu|^2\right|^2,$$
即，$|\psi\rangle$在态矢积分后得到一个正方形矩阵。这是因为态矢积分表示为
$$\sum_{\mu=-N}^{N}\bra{\psi}\hat{c}_\mu\ket{\psi} = m_\text{eff}\int d\alpha\left\lvert\psi\right\rangle\left\lvert\alpha\right\rangle\left\langle\alpha|\psi\right\rangle.$$
该积分结果在每一个态矢处等于相应的本征振幅乘以对应的本征振幅的共轭元。由于每个态矢都是基向量的线性组合，并且线性无关，因此$|\psi\rangle$表示为一个关于基矢的矩阵。于是
$$|\psi\rangle = \sum_{\mu=-N}^{N}\left\lvert c_\mu\right\rangle\left\lvert\mu\right\rangle,$$
其中$\mu\in\left[-N,N\right]$。
上式给出了态矢积分所得矩阵的形式。不难看出，每个态矢对应一个本征振幅，而每个本征振幅又对应一个态矢，反过来也是成立的。于是，任何矩阵乘积$\sum_{\mu=-N}^{N}\ket{\mu}\bra{\nu}\otimes A_{\mu\nu}$，其中$A_{\mu\nu}$是单位阵，就是把矩阵乘积的基矢由$\mu,\nu$换成相应的本征矢$\lambda_\mu,\lambda_\nu$。换句话说，态矢的张成积分转换到了本征矢的张成积分。这就保证了任何矩阵乘积都可以通过积分将其表示为本征矢的张成积分。
## 热散射
热散射是量子热力学中一个重要的问题。热散射的产生原因主要是以下两个方面：第一，外部介质吸收和辐射；第二，物质与环境的相互作用。
热散射产生的原理类似于光的散射，都是因为微观粒子从初始状态进入某个临界点之后，不可能再回到原点。在过去的几十年里，一些先进的技术已经成功地探索了热散射的原理。当前的热力学实验方法主要集中在研究热散射与其他宏观热力学实验方法的相互关系上。
### 自由玻尔兹曼辛烷效应
在自由空间中，费米子与导带相互作用的结果是能量的均匀分配。在高温下，由于许多热运动带隙导致出现一定的概率丢失，称为自由玻尔兹曼辛烷效应（Franck-Condon effect）。如果两个相互作用的量子子系统之间存在着很强的相互作用，那么就会出现这种丢失。
### 费米-罗森海明堡金属线
通常来说，对于一般的热散射问题，主要依赖于相邻原子的重叠状态，即相邻原子处于同一态矢。由于受制于重叠原子的限制，在真空情况下热散射的能量损失往往很严重。在近期的一项实验中，引入了一颗气体层，将真空中的两个相邻原子隔开，则两个原子就可以形成两个单独的氢原子团。此时，由于两团氢原子不再相互重叠，从而摆脱了重叠原子的限制，可以实现更大的自由度。这样，相邻的氢原子团之间就可以存在相互作用的机会，因而可以形成一种新的态矢。
