# 1.背景介绍


## 游戏行业蓬勃发展
游戏行业在2019年已经成为国内最主要的娱乐产业之一，占据全国GDP的7%，比2018年增长了近5%。截至今年6月底，中国在线游戏出版总量超过5000万册，单季度创造利润逾230亿元。2021年，国家对游戏行业的支出预算高达2.7万亿元，比上一年增长26%，平均每款游戏将产生约100万美元的营收。据调查显示，全球游戏市场规模已经从上世纪70年代末到2018年前夕的2.7万亿美元，一路攀升到目前的顶峰，未来将继续保持这一强劲增长态势。

随着人们对游戏感情的释放，越来越多的人喜欢上了游戏这个品类。其中不乏一些精致的作品、看起来无比漂亮的界面、精妙的游戏机制，还有众多热炒的手游新品。而游戏开发者也变得越来越吃香。比如腾讯微信平台“农民工”成功让全网用户玩起了一起占领广州湛蓝的城堡、绝地求生爱好者玩起了吃鸡、任天堂 Switch 平台成功推出了经典游戏《王国之心》。

## 为什么要做游戏
游戏能够带给人们快乐、满足感和成就感，可以帮助人们解决生活中的各种问题，还能促进人们的职业发展。游戏也是一种趣味性很高的休闲方式，它可以帮我们发散注意力，提升精神层次，扩展视野，释放潜力。游戏对个人、家庭和社会都有着巨大的益处。

如果把游戏制作成收入源的话，那么它的收入模型可能会如下所示：

1. 免费游戏：顽固的竞争关系，消费者习惯于免费。这种模式吸引了大批忠实玩家，并获得了庞大的用户群体，但同时也产生了巨大的盈利空间。玩家花费时间、金钱和体验直接影响到了游戏制作者，他们可以通过游戏来赚取金钱。
2. 消费付费游戏：游戏制作者向消费者收取服务费用，提高用户留存率。不过，这种模式需要消耗掉游戏制作者的成本，并且不利于游戏的持续发展。
3. 通过游戏赢取积分：游戏中可以兑换、购买或者赢取虚拟货币，用于提升玩家的整体水平。这种模式吸引了更多的参与者，并且提供了一个积分系统，但玩家因为缺乏游戏激励机制而难以形成坚持玩下去的习惯。
4. 通过游戏赚取物质回报：游戏中设有各种奖励机制，比如免费道具、道具特效、特殊关卡等。这种模式吁求了游戏制作者投入更多资源，并通过激励机制吸引玩家，但是并不是所有玩家都喜欢这种模式，而且这个过程可能长期持续，会产生巨额的经济损失。

除了这些收入模型之外，游戏还可以开拓新的玩法，提升舒适度，增加社交互动性，丰富经营场景等。游戏是目前最流行的娱乐形式，占据了互联网时代的中心位置。因此，不管是电子竞技还是网络游戏，都会受到广泛关注和追捧。
