# 1.背景介绍


价值体系（Value System）是指人们在社会、经济、组织中塑造和体现的价值观念、目标、情感，是支撑人类社会进步、创新、成就卓越成功的关键所在。如此重要且繁琐的主题，一般人难以置信地需要长期坚持研究、探索和实践，充分吸收和传递才能达到真正意义上的通透和共鸣。但我国的价值体系建设也尚不健全，缺乏有力的引导、激励、培育和宣传机制，甚至还存在着“反智”现象。如何解决这一矛盾？既要提升社会各阶层对价值体系的认识，增强他们的参与性和责任感，又不能单纯倡议大家搞得太极端、偏激或盲目，更需要智慧、细致、务实的做法，有效统筹考虑价值赋能、保障价值实现、推动价值流转，提升社会福祉。本章试图通过分析价值体系的概念、基本理论和相关制度安排，阐述如何构建和完善中国特色的价值体系，使之成为人类社会进步的标志性举措，推动中国走向更美好的明天！

价值体系的概念源远而来，早在马克思主义哲学中就被提出。它定义为社会生产活动的目的和标准。它是基于人类普遍需求和利益的需求和要求，具有重要意义，其内涵主要体现在以下三个方面：

1. 公平分配：价值体系包括对所有人的平等权利和自由的承诺，让每个人都有权分享经济和社会资源；
2. 富足平安：价值体系要确保人民的幸福、安全和福祉，主张理想生活和追求平静、美好状态；
3. 个性解放：价值体系要允许个性自发、创造力得到发展，鼓励个人的发展及个人化的经营管理，促进个人自由和个人的幸福。

建立和完善中国特色的价值体系是一个十分复杂的任务。首先，要坚持以人为本的理念，以立德树人的初心为根本，着力培养和激发人类爱惜自己、珍视他人、追求美好生活的品质。其次，要坚持原则性和科学性，以社会责任和公正的原则为依归，正确处理社会风险和道德风险。第三，要充分调动人力资源、物质资源和精神资源，以互联网+价值链等新型产业形式，构建公平、开放、包容、可复制的价值网络。最后，要健全激励机制，形成持续的社会驱动力，激发起人才的积极性、创造性和主动性。

基于上述分析，本章将重点阐述以下五个方面的内容：

1. 价值体系理论
2. 价值体系实践方法论
3. 中国特色价值体系的建立与完善
4. 中国特色价值体系的发展路径规划
5. 价值观的社会流动与社会价值观的构建

# 2.核心概念与联系
## 2.1 价值观
价值观是指人们在一定的生活环境中所接受和遵循的一系列判断、价值观念、意愿、态度、行动准则或行为规范。价值观可以认为是人们行为的意志和原动力，它影响着个人、群体、国家乃至整个社会的命运。现代社会中，价值观的确十分重要。随着人类发展历史的进程，人类对社会、经济和文化等现实的认识加深，特别是在20世纪末，个性主义、理想主义、文化自觉、价值观念观念的兴起和衰落，以及科技进步和资本主义的催化下，价值观念正在发生变化，由精英主义向平庸主义、逆向歧途的方向演变，价值观念又重新回到集体主义的中心位置。

价值观指导着每个人对世界的看法，是影响人类生存与发展的关键因素。价值观也是社会进化的结果，它形塑着我们了解这个世界的方式，并以此影响我们的行为、选择、关注以及生存方式。不同种类的价值观都会影响到不同的人，比如西方的平庸、压迫、贫穷，以及东方的乐观、自由、社会主义。对社会进行价值判断是人们独立思维的前提条件，也因此导致社会成员之间的差异性很高。

价值观对个人、集体、组织、国家、社区的影响都是长久的，但是否能够真正融合在一起，成为长期稳定运行的社会秩序，这就成为了价值观的社会流动问题。衡量价值观的社会流动的指标主要有两个：第一，在一定时间内，从不同地域、不同文化、不同社会阶层的人口中获得的同样的信息，能否产生共鸣和认同？第二，价值观的社会流动如何影响个体的行为和选择？社会学者认为，如果个体的价值观念无法准确地代表整体价值观念，或者价值观念的张力过大，就会出现严重的偏见和歧视，并削弱社会的凝聚力。因此，在选取和宣传价值观时，应着力使价值观念平衡多元化，尤其是要保证不同阶层、不同领域、不同性格的人都能获得充分的发言权。

## 2.2 价值观的分类与区分
根据价值的客观性、主观性、可度量性和连续性四个维度，价值观可以分为客观价值观、主观价值观、可度量价值观和非连续价值观四种类型。客观价值观指的是直观和具体的，无需被任何个人或集体的经验、感受、能力、理性所支配。例如，仇恨、友爱、正义、快乐、幸福、真诚等都是客观价值观。主观价值观指的是高度抽象的、个人化的，经由个人的内心世界而存在。例如，一年四季听音乐、阅读、学习，追求自然、生命、爱情、平等、美丽等就是主观价值观。可度量价值观指的是能够量化衡量的，可以用数学计算的方式进行测算。例如，身体健康、财富自由、人格魅力、家庭 harmony、社会 equity等都是可度量价值观。非连续价值观是指不确定的、不稳定的，无法通过一个限定范围的时间长度来衡量的。例如，好奇、浪漫、梦想等就是非连续价值观。

由于各式各样的价值观形态相互交织、相互纠缠，所以有必要分辨清楚它们之间的界限。比如，并不是所有的价值观都是能够客观地、可度量地、可比较地量化的。比如，对于个体来说，我可以拥有热爱自己的一己之爱，也可以拥有对社会环境和生活水平的尽善尽美，但没有哪一种价值观是能够客观地、可度量地、可比较地量化的。同样，并不是所有的价值观都是可被客观地、可度量地、可比较地量化的。比如，其实很多人对爱情、友谊等非连续价值观并不陌生，但对其客观性、可度量性和可比较性却很模糊。因此，我们要把握价值观的本质，避免将其仅仅视作是价值指向或标准，而应将其作为一个具有广泛意义的现实问题，从而勇于去发现、思考和寻找其中的奥妙。
