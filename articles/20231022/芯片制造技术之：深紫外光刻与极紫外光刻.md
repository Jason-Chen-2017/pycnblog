# 1.背景介绍


在工业制造领域，需要高频、高精度地应用芯片刻蚀技术，主要分为两种方式：
- 深紫外光刻 (Deep UV Deposition)：利用光谱范围较宽，厚度较薄，透射性较强的深紫外光进行片刻。
- 极紫外光刻 (Infrared UV Deposition)：利用电气辐照度控制荧光带宽度，提升纳米芯片导热能力，降低设备成本的一种制备方法。

为了验证深紫外光刻（DUD）或极紫外光刻（IUD）技术的优越性，最近几年来，越来越多的研究人员对这两种技术进行了论证。

深紫外光刻技术早期应用于硬件生产环节，如打印件的模具、屏幕等，是计算机图形界面的标配。近年来，随着微处理器的采用，深紫外光刻技术也被应用到多种IC领域，如集成电路设计、内存芯片设计、半导体封装材料设计、非易失性存储器设计等方面。

与此同时，由于深紫外光刻技术的复杂性及其耗时、难度巨大的精神负担，普通企业往往难以接受甚至是望而生畏。因此，人们逐渐转向使用更多昂贵的高性能晶圆类别的芯片，如FPGA、ASIC等，但在深紫外光刻技术上，仍然有许多限制和不足。因此，新的科技革命似乎正在不断涌现——由像微瑞士这样的初创企业发起的“微芯”计划正在引爆芯片制造的新潮流，并将推动芯片制造技术走向深度、高精度。


# 2.核心概念与联系
## 深紫外光刻 (Deep Ultraviolet Deposition)
深紫外光刻是利用位于紫外线附近的深紫外线作为照射介质，以某种特殊的电子过程控制各层金属离子的位置，使得金属产生化学变化。特别适用于各种类型的金属，包括铜、镍、锡、钛、铝、硅等，并且具有自我修复功能。深紫外光刻技术允许在液氢或无机盐水环境中长期进行，且在过去几十年里已经成为制造IC芯片的主要手段。

## 极紫外光刻 (Infrared UV Deposition)
极紫外光刻是一种利用发光二极管阵列及激光照相技术，将纳秒尺寸的紫外光波长传递给多个探测器单元，根据成像结果来刻划电子元件的结构。这种方法可以显著提升制造产品的热敏性。目前，微型激光器、小型感光器，以及相关的激光切割技术已被应用于极紫外光刻技术的各个领域。近年来，随着传感器技术的进步，以超声波为基础的极紫外光刻技术正在蓬勃发展。

### 与深紫外光刻之间的区别
从基本原理上看，两者都是利用紫外线所发出的照射光束生成制造过程中的化学物质改变。但是，它们存在一些差异。首先，在制作过程中，深紫外光刻通常只考虑一个目标层次内的低层化学元素，而极紫外光刻则覆盖整个芯片表面。第二，深紫外光刻的方法依赖于特定电子设备如激光刻蚀器、偏振加速器等，而极紫外光刻通常采用半导体电子化学方法。第三，深紫外光刻对硬化、电镀等过程要求更高，容易受腐蚀、气味等因素影响；而极紫外光刻技术对熔点和厚度都没有太大限制，可以有效避免这些限制。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 深紫外光刻
### 材料准备
首先，准备一块特别厚的深紫外线色彩的液体，一般为稻米面、天鹅绒、玻璃瓶或者塑料桶等，用刀或笔轻轻划开溶于墙壁的硬币形状的微型瓷砖。然后将瓷砖放在一个干燥的浴室里面，等待适当的时间让其完全融化。用深度约为1cm的低功率筛选器取出瓷砖内含有的无机盐水，待其完全凝结成珠胶状后，即可进行深紫外光刻操作。

### 操作方法
在深紫外光刻之前，要做好几个必要的准备工作。首先，在对切片的金属粘结前，应先将垫板放置在切片的适当位置，否则会出现极细颗粒，影响分辨率。其次，在开始进行深紫外光刻的时候，可将底板平行移位，待金属沉积层完全落下，然后再反复摩擦，以消除一些残余颗粒，确保金属完全孤立。另外，对于镍镉反应器，还要注意做好增压。

#### 激光切片法
即在对切片的金属粘结前，将金属膜剪下来，用切片工具将其切成大小一致的小颗粒，将小颗粒按照顺序穿过底板，并贴上浓缩胶，然后启动照相机，通过每条激光线，对固定在小颗粒上的激光探测装置按顺序识别出每个小颗粒。由于每条激光线只能识别一个小颗粒，所以需要多条激光线才能正确切割整个切片。

#### 电子切片法
即将铜片直接与电子切片工具焊接，将切片完成后的金属粘结区域用高压锅烫成光滑模样，用浓缩胶将其贴在底板上。然后打开激光切割装置的开关，将固定在切片上的激光探测装置按顺序识别出每一个小颗粒，并对其数字化。通过数字标记法，即可将整个切片切割出来。电子切片法的切割速度快、成本低，但需要小心地固定金属，防止沉积在底板上。

## 极紫外光刻
极紫外光刻技术依赖于在表面层的高层金属雕刻纳米粒子电子，这种技术以纳米粒子结构作为实时的配位电子元件，通过对其电子结构的刻划与分层设计，实现纳米芯片的电镀、隔热和封装，并可通过不同级别的结构分层施加不同的性能。因此，极紫外光刻技术能够有效降低制造成本，降低技术门槛，以及减少重复工序。

### 原理和准备
#### 纳米粒子电子原理
在纳米芯片领域，各类纳米粒子结构电子元件的构造都基于纳米粒子的晶格结构。纳米粒子是电子学和量子物理学的重要研究对象，其周期在微米至毫米之间。纳米粒子体系由微晶核和微晶衬里组成，可以容纳数千或数万个电子。为了利用纳米粒子结构的特性，制造出具有尺寸合适、性能卓越的工艺品，各个领域的科学家均致力于寻找创新方法。

纳米粒子结构电子元件的结构是通过向晶粒的中心结构加上一定数量的电子结合点，由诸多电子排列在一起组成，实现极低的分散度。这种结构因其平坦、分层、结构紧凑等特征，在表面层的功能性作用不容忽视。如FET、BJT、MOSFET等电子器件都是以纳米粒子结构为基础的，通过对其结构的精确刻画和布局，设计出各种功能性电子元件。

#### 设备准备
首先，购买一套用于印制深紫外线色彩的液体，如稻米面、天鹅绒、玻璃瓶或者塑料桶等，准备一支具有小功率的激光或者微型激光器。

然后，制备一套基于小型感光器的能量扫描仪。这套设备需要连接导线和激光器，激光器只能通过导线送入感光器的光线进入感光探测单元，导线则用来将激光器与芯片间的距离固定住，并用于调节曝光速度和能量开平衡。

最后，进行结构设计和建立相关的芯片后，要加入高压锅（可将漆面浸湿、聚拢、镀层、压缩等），然后把芯片中的纳米粒子电子结构加注上极小的电极，如锂离子电池，待金属表面化学变化完成后，回到锅内烘干。

### 操作步骤
极紫外光刻技术最关键的是进行电子映射。即先将纳米粒子结构切割成单个电子元件，然后将该元件填充在导电材料内，用加热设备加压，使纳米粒子的电子熔在导电材料的液氢浴层中，使纳米粒子与导电材料间产生电子交换，从而形成电子网络。通过这种电子映射的方式，就能获得纳米粒子结构的电子层次设计。

在电子映射之后，就可以对电子结构进行掩盖，提升掩盖效率，降低曝光时间，以保证设备的稳定性和正常工作。掩盖时要仔细选择好掩盖位置，不要对表面过度施加掩盖，以免造成电子层级不连续、纳米粒子结构损失等问题。

然后，在掩盖完毕的电子结构上，加入锂离子电池，用多层导线拉低其浓度，保持电子结构稳定。此时，要通过不同级别的结构分层施加性能，如电镀层、隔热层、封装层，以达到合理的纳米芯片性能。
