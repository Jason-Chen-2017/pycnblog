# 1.背景介绍


## 大数据简介
大数据是指海量数据的集合。其产生的主要原因有两个：一是超大规模数据的产生，二是互联网+、移动互联网、物联网等新型的新技术带来的爆炸性的数据量、速度和复杂度。由于其高容量、多样性和异构性特征，使得数据存储、处理和分析等过程变得越来越复杂和繁琐，传统的数据库系统无法很好地处理这么庞大的信息量。为了有效地对这些海量数据进行收集、管理、分析和挖掘，数据采集、计算、分析、挖掘和管理等技术也应运而生。为了将海量数据转化为有价值的信息，大数据分析的工具、方法和流程也日益成熟。
在制造领域，通过各种各样的手段，可以收集到大量关于生产制造过程中的相关数据。例如，制造企业可能通过监控机器设备上的生产进度、利用电子计量仪表盘实时跟踪产量，或者采用电子温度计或烟感器实时测量出来的环境温度等方式来收集生产数据；而消费者则可以通过浏览购物网站、社交网络、搜索引擎来获取大量关于商品、服务和区域性产品的使用数据，这些数据对于营销、市场推广等都非常重要。同时，由于制造企业面临着更加严苛的质量标准要求，其收集到的生产数据往往都是缺乏真实、客观的评价和反映生产情况的数据，所以需要用大数据技术进行整合、分析和挖掘，从而提升企业的竞争力和品牌知名度。
## 智能数据应用架构简介
随着智能制造、智慧制造领域的迅速发展，如何有效地应用大数据技术及其所提供的巨大潜能对于制造业的发展至关重要。智能制造所需的核心技术是以人工智能（AI）为代表的机器学习、模式识别、图像处理等技术。对于这种技术的应用，传统的基于规则的系统设计显然不够灵活和准确，而基于数据驱动的系统设计正成为最佳选择。因此，智能数据应用架构应运而生。
智能数据应用架构包含三个层次：数据采集、数据存储、数据分析、数据挖掘和数据应用。其中，数据采集层负责将各种来源的大量数据按照一定规则统一整理并保存，包括原始数据、日志文件、历史数据、设备数据等；数据存储层负责对数据进行永久存储和长期保护，保证数据安全、可靠性和完整性；数据分析层包含数据预处理、数据清洗、数据转换、数据挖掘、数据挖掘结果的展示、应用系统的改进、新产品的开发等功能模块；数据挖掘层通过对现有数据进行挖掘，可以发现更多有价值的知识和信息；数据应用层是指将挖掘出的知识和信息用于生产系统、业务流程、管理决策等方面的应用。
基于智能数据应用架构，制造业可以充分运用大数据技术和自身的资源，结合人工智能、云计算、物联网等新兴技术，做到“一网通达”。通过大数据采集、分析、挖掘和应用，可以实现以下目标：

1. 全面的掌握生产和消费过程数据，优化生产过程和管理
2. 提升企业竞争力，降低成本，提升品牌影响力
3. 提高产品和服务质量，提升产品和服务供给效率
4. 改善生产环境，降低污染，提升环保水平

本系列教程的目标是帮助企业及个人理解大数据和智能数据应用架构。我们以制造领域作为案例研究，重点关注智能制造领域，涉及的主题有：

1. 大数据与智能制造产业的发展脉络及未来趋势
2. 大数据与智能制造领域的核心技术、架构及应用
3. AI算法、模式识别与图像处理在智能制造领域的应用及优势
4. 基于智能数据应用架构的制造企业运营管理模式、流程、平台、技术和基础设施建设
5. 制造业关键岗位的数据职业路径以及职业技能要求
