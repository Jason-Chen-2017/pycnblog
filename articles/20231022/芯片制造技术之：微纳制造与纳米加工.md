# 1.背景介绍


## 1.1 微纳(Mica)制造技术
微纳(Mica)是一种小尘土，微观结构紧密、没有沉淀、弹性较好、光透射性强，在空气中具有清洁特性。微纳与塑料、化纤维等有着不同的特性。微纳制造可以提高产品质量和性能，降低成本。微纳制造技术已经成为一种主流的新型产品制造技术。它的优点主要有以下几点：

1. 微纳材料的自回归能力很强，即使是在极端环境下也不会腐蚀变形。
2. 微纳材料的绝缘性很强，在金属介电体表面没有缺陷。
3. 微纳材料的大分子结构适合高温机械加工。
4. 微纳材料的开放性可以有效避免粘附在金属上而导致的切割损伤。
5. 微纳材料的厚度、孔径、平滑度都比较低，模拟软体或具有很好的抗拉行为。
6. 微纳材料价格比塑料更便宜，而且产地广泛。

由于微纳材料本身的特点和原理，微纳制造技术已经逐渐成为新时代的重要产品制造技术，并成为各种新产品的标配技术。比如苹果手机中的微晶屏，智能手表上的微导轨，笔记本电脑中的微软雕刻板，甚至还有一些小家电中使用的微漆。微纳制造技术已经成为消费品的“基石”。许多消费品牌已经开始尝试微纳制造来优化产品的结构设计，同时也帮助消费者节约了大量的材料成本。

但是微纳制造技术存在诸多局限性，其中最重要的问题就是制作难度太高，且耗费较长时间，通常需要通过精心设计才能达到较好的效果。因此，在该领域的科研工作者们要努力寻找一种新型的方法来改善这一现状。
## 1.2 纳米技术
纳米(Nanomaterials)与微纳不同，它是由纳米颗粒组成的一种微体系材料。它具有简单、非晶状结构、无机物质结构、无凝结、无杂质、自然排列、纳米级别的厚度、高分子聚合物。纳米技术的应用范围十分广泛，如半导体材料、电子设备、航天器、医疗设备等。纳米技术也是一种正在蓬勃发展的新型工艺技术。由于其结构简单、无机物质、无杂质、自然排列等特征，使得其能够用于各个领域。

在纳米技术出现之前，制造各种新材料都是依靠传统的喷涂方法。这种方法在制备玻璃、陶瓷、金属、合金、橡胶等材料方面都能得到很好的成效。但是这些方法都存在着一些缺陷，比如喷涂法的浪费、反复工序、耗时长、恶劣环境、错位装配等问题。因此，在纳米技术出现之后，便出现了一系列的纳米相关技术，例如微管、纳米粉末、纳米锭、纳米镜头等。

纳米技术利用纳米生物分子的特性，制成一种具有固定形状、分散、并行等特征的材料，极大的降低了制造复杂的难度。而这种材料的制造方式也十分简单，利用激光和微生物培养法等方法就可以进行快速生产。当然，纳米技术仍然处于初级阶段，但其带来的改变却非常迅速。

纳米技术相对于微纳制造技术的优势主要有三点：

1. 纳米材料具有纳米尺度的厚度，具有良好的聚焦，可以在相同的尺寸下制造出不同规格的材料。
2. 纳米材料的极大聚集，使得制造过程更容易控制。
3. 纳米材料的自我修复能力强，当发生塌陷、裂纹、开裂等问题时可以自行修复。

纳米技术引起了工业革命，使得人类的生活水平大幅提升。由于纳米技术的普及，航天飞机、医疗设备、半导体材料等领域的产品价格都大幅降低，整个社会都变得富裕起来。

但是，随着纳米技术的发展，纳米技术引起了一些新的问题，比如安全问题、使用限制等。为了解决这些问题，现有的一些工程技术被提出来，如激光电化学工程（LEC）、超纳米分析（SAM）等。这类工程技术目前还处于起步阶段，很多国家还处于禁用状态，因此，其技术成熟度仍然不够高。但是，随着技术的进步和人们对其安全性和使用限制的认识的增加，纳米技术将会成为一个越来越重要的科技发展方向。
