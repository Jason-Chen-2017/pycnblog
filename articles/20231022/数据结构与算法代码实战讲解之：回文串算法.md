# 1.背景介绍


回文串(palindrome)是指一个单词、短语、句子等具有相同读法的文字，在回环或倒转后依然还能继续读出原来的样子的字符串。回文串算法也称为回文对比算法或正向/逆向比较算法。一般来说，检测回文串的方法有两种：一种是暴力破解法，将所有可能的子序列都检查一遍；另一种是利用中心扩展法，通过将字符串向两端扩展，直到出现回文区域。本文采用第二种方法进行回文串识别和判断，首先介绍相关概念及其联系。
# 2.核心概念与联系
## 2.1 回文字符串
所谓回文串，就是指从左往右读和从右往左读都是一样的字符序列，比如："level", "noon"。
## 2.2 回文串的定义
### 2.2.1 模板字符串
回文串的定义可以用“从左往右读和从右往左读都是一样的字符序列”这个模板来表示，例如："level"就是回文串，"noon"也是回文串。
### 2.2.2 对称性
回文串一定是对称的。由两个方向读取同样的字符序列就构成了回文。
### 2.2.3 奇偶性
如果回文串长度为奇数，则中间某一位要么是回文中心（奇数），要么是对称轴（偶数）。若回文串长度为偶数，则中间没有奇偶分界线。
## 2.3 回文串的分类
### 2.3.1 回文对
定义：两个或多个连续的相同字符组成的串，这种串叫做回文对。比如：“rr”, “ooo”，“abcddcba”。
#### 2.3.1.1 回文对的性质
1）回文对一定是回文串。如果一个字符串不是回文串，它必定有且只有一个不为回文对的分割点，可以把该位置上的字符去掉，变成两个回文串，再把它们拼接起来即为该字符串的回文对。
2）回文对一定是回文阵列。即两个或多个不同字符的回文对中，如果有一个不是回文串，则整个数组也是无效的。
#### 2.3.1.2 回文对的分类
##### 2.3.1.2.1 满足一定的回文性质的回文对
- 奇回文对：一个回文串内部的任意字符都有奇数个，比如："abba"。
- 偶回文对：一个回文串内部的任意字符都有偶数个，比如："abcdcba"。
- 相邻重复的回文对：一个回文串内部的某个字符出现两次以上，且两次之间隔着至少三个不同的字符。如："abbaba"。
- 区间回文对：一个回文串的某个区间内，每个字符都出现偶数次。比如："ababa"中的"ab"和"ba"区间。
##### 2.3.1.2.2 不满足一定的回文性质的回文对
- 没有回文性质的回文对，如："acbbaca"。
- 有限回文对，如："aabbaa"。
- 不等长回文对，如："aaaaaaaaaaaaaaa"。
