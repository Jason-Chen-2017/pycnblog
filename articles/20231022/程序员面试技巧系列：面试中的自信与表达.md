# 2.核心概念与联系
## 自信
自信并不是单一的概念。它由两个方面组成：一是勇气和坚持，二是坦诚相待。一个人可以做很多事情，但只有持续不懈地去坚持，走出困境，他才能从容应对不确定的状况。而坦诚相待则是指面试官明白并诚恳地接受候选人的意见，从而鼓励他们不要执着于盲目猜测和局部优化，可以相互理解、互相支持。最后，勇气则是表现得机智、果敢、创造力十足。一个聪明的人会充满激情，但同时也不要太过悲观。比如，面试者遇到问题时，如果能冷静分析原因并主动寻求帮助，相当于给面试官提供了更多的机会，面试官也会觉得这位候选人更有建树和素质。

## 表达
表达是通过口头、书面、视觉或触觉等方式向面试者传达信息。它的作用包括准确传递信息、建立沟通渠道、消除面试官对候选人工作的质疑、确认候选人的真实情况、展现候选人的个性特征。表达的技巧和方法主要体现在以下几点：

1. 语言组织清晰简洁: 提供的信息越多、越准确，面试官就越信任候选人。所以在写作过程中要注意言之有物，用词准确无误，句子顺畅自然；
2. 注重主旨: 对候选人的主要问题陈述，一定要突出自己的见解，对整个过程进行攻读；
3. 使用简单易懂的语言: 消除面试官对语言的限制，尽量用简单的语言，表述清楚自己想说的话题；
4. 视听传达力: 通过拍摄像片、录音、视频等方式表达，能够增强表达的效果。
5. 保持镇定：面试过程中最重要的一点就是保持镇定。任何时候都不能心急，也不能情绪失控。每个面试官都希望面试者有一个积极和开放的心态，但是一旦出现紧张、冲突、矛盾的时候，就应该及时调整策略，让面试继续下去。

### 一、算法1：判断是否可以构成三角形
```python
def is_triangle(a, b, c):
    if a + b > c and b + c > a and c + a > b:
        return True
    else:
        return False
```
该函数用于判断三个边长a、b、c能否构成一个三角形。若能，返回True；否则，返回False。
说明：这个函数比较简单直接，即先判断a+b>c、b+c>a、c+a>b三条线段是否满足三角形的条件，如果都满足，则返回True；否则，返回False。

### 二、算法2：计算两点之间的距离
```python
import math
def distance(x1, y1, x2, y2):
    d = math.sqrt((x2 - x1)**2 + (y2 - y1)**2)
    return d
```
该函数用于计算两个点（x1,y1）和（x2,y2）之间的距离。这里利用了math库中的sqrt()函数求根号，然后返回距离d。

### 三、算法3：找出数组中重复的数字
```python
def find_duplicate(nums):
    duplicate = []
    for i in range(len(nums)):
        while nums[i]!= i:
            temp = nums[i]
            j = nums[temp]
            if j == temp:
                break
            elif j < len(nums)-1:
                nums[i], nums[j] = nums[j], nums[i]
            else:
                break
    for k in range(len(nums)):
        if nums[k]!= k:
            duplicate.append(nums[k])
    return duplicate
```
该函数用于查找数组nums中的重复数字。首先定义了一个空列表duplicate，用来存储重复的数字。遍历数组nums，对于每一个数字nums[i]，判断其是否等于i。如果等于i，则表示该数字是重复的，添加到duplicate列表中。如果不等于i，则表示当前数字和重复的另一个数字nums[j]交换位置后，会使该数字重复，则重复的数字肯定是nums[j]或者更大的某个数字。那么可以通过循环判断nums[j]是否大于等于len(nums)-1，如果大于等于len(nums)-1，则表示该数字不存在，否则，可以通过设置循环避免无限循环。循环结束后，duplicate列表中保存的都是重复的数字。
