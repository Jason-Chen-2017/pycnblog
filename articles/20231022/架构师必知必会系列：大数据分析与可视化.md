# 1.背景介绍


## 大数据的特点及优势
大数据（英语：big data）这个词汇从1996年由Google提出，它被定义为“指以超大规模的方式来处理、分析和挖掘大量的、复杂的数据。”换句话说，就是巨大的、海量的数据。由于数量庞大、结构复杂、采集方式多样，传统的数据库无法很好地管理和处理大型数据。因此，大数据具有非结构化、半结构化、高维、多样性、实时、分布式等特点。

根据国家统计局发布的数据显示，截至2017年末，全球人口超过了9.1亿，但仅占全球GDP的0.7%。而新兴市场的数据泵、电子商务平台、物联网、机器学习、大数据分析等技术蓬勃发展，正帮助企业在数字转型、升级换代中实现更加顺利的发展。

### 分析大数据常用的工具——Hadoop、Spark、Storm、Flink
如今，大数据已经成为企业数据存储、处理、分析的基础设施。分析大数据主要依靠四种开源框架——Hadoop、Spark、Storm、Flink——它们都有着不同的特性、优点和应用场景。以下通过对Hadoop、Spark、Storm、Flink进行阐述。

#### Hadoop
Apache Hadoop(简称HDFS)是一个开源的分布式文件系统，它允许集群中的节点存储和计算大数据集上的海量数据，而且它能够提供高容错性、高可用性的数据冗余和数据局部性功能。它可以用于处理来自网站、社交网络、广告点击日志、交易事务数据等不同来源的数据，并以多种形式进行分析。Hadoop MapReduce是 Hadoop 中最常用的编程模型，它基于 Hadoop 的 HDFS 将海量数据分割成离散的块，并将任务分配到节点上运行，同时也提供了包括排序、规约、过滤等一系列高级数据处理功能。此外，Hadoop Streaming 也可以用于快速处理实时数据流。

#### Spark
Apache Spark是一个开源的大数据计算框架，它基于内存计算和快速分布式数据处理能力。它的最大特点之一就是能够做到比Hadoop快上万倍，并且具有易于使用的API和丰富的算法库，能满足日益增长的数据量和海量计算需求。其编程模型类似于Hadoop MapReduce，不过它采用RDD（Resilient Distributed Datasets）作为内存数据结构，并支持多种语言，如Scala、Java、Python。Spark还提供Spark SQL和MLlib等组件，能够更方便地进行SQL查询和机器学习算法开发。

#### Storm
Apache Storm是一个开源的分布式实时计算系统，它是一个流处理器。它能够实时处理来自各种源头的事件流数据，然后将结果发送到指定的输出端。Strom运行在一个或多个服务器上，每个服务器可以处理TB甚至PB级别的数据。通过高吞吐量的硬件资源和容错机制，Storm可确保即使出现故障也不会丢失数据。Storm提供了一个简单且易于使用的编程模型，利用灵活的数据模型和反应式编程风格来处理实时的流数据。

#### Flink
Apache Flink 是一款开源的快速分布式数据处理框架，基于流处理和批处理的执行模式，能够对实时数据进行高效率地计算和分析。Flink 独特的特征是能够无缝地整合 Apache Hadoop、Apache Spark 和 Apache Storm，并兼容其他常用开源框架。Flink 的 API 可以简洁明了地表达数据流的变换逻辑。

### 可视化工具——Tableau、DatoIQ、QlikView、Microsoft Power BI、SAS Visual Analytics……
数据可视化是大数据分析不可缺少的一环。目前市面上有很多可视化工具供用户选择，例如Tableau、DatoIQ、QlikView、Microsoft Power BI、SAS Visual Analytics等。这些工具都可以帮助企业快速进行数据可视化，并得到直观的结果。以下通过对这些可视化工具进行分类和介绍。

#### Tableau
Tableau是一个商业智能数据可视化工具，它能够分析海量数据并形成直观和动态的图表、报告、仪表板。它既可以用作独立应用，也可以嵌入web页面、移动设备和服务器中，让用户访问数据。Tableau 具备强大的分析能力，可以同时分析多维数据，并将多个数据集呈现给用户，并根据用户的不同要求提供不同的视图。Tableau 的自动化分析和决策功能可以帮助企业快速发现隐藏的模式和趋势。

#### DatoIQ
DatoIQ是一个基于云服务的商业智能工具，它提供基于数据的见解、AI助理、分析功能、预测功能。它帮助企业分析、理解和预测业务数据，并生成直观的报表、图表、仪表盘。DatoIQ 提供基于角色的权限控制，让用户只看到他们需要看到的内容，进一步保护数据安全。DatoIQ 提供了更好的协同工作环境，让团队之间互相配合，共同完成复杂的数据任务。DatoIQ 的定制化工具包可以帮助企业快速获取见解，并有效降低企业内部成本。

#### QlikView
QlikView是一个数据可视化和分析工具，它可以帮助企业构建具有人类情感色彩的可视化报告，并帮助他们获得快速、准确的业务洞察力。QlikView 提供的数据连接能力，可以轻松地从各种数据源导入和合并数据，并进行筛选、聚合和分析。此外，QlikView 还提供了一套完整的生态系统，包括移动端、Web 前端、云端服务等。

#### Microsoft Power BI
微软Power BI是一个基于云服务的商业智能工具，它可以在任何地方访问，并且与所有主流应用程序集成，包括 Excel、Outlook、PowerPoint、Sharepoint等。Power BI 提供了丰富的可视化图表类型，并可以创建高度交互式的仪表板，帮助企业快速理解数据，从而实现决策优化。Power BI 提供了更强的自助服务能力，让广大的分析人员无需 IT 支持即可快速创建自己的仪表板，提升工作效率。

#### SAS Visual Analytics
SAS Viual Analytics(简称SASVA)是一个数据可视化工具，它提供基于专家设计的分析视图和仪表板模板，可以自动生成基于数据的可视化报告，并提供专业的建议、制定策略。SAS VA 可对多维数据集进行分析、比较、关联和关联分析，并生成丰富的可视化报告。SAS VA 还提供了数据的导入、清洗和转换功能，以及对数据进行编码、聚类、探索和可视化等分析功能。SAS VA 可帮助企业提升分析能力，并最大限度地释放数据价值。
