# 2.核心概念与联系
## 对象与类
Python面向对象编程（Object-Oriented Programming），简称OOP，是一种基于类的面向对象编程范式。面向对象编程提供了一种比过程化编程更加抽象和直观的编程方式。它将现实世界中的事物抽象为一系列对象的属性和方法，然后通过它们进行交互。每个对象都可以视为一个独立的实体，拥有自己的状态、行为和数据。

类（Class）是一个模板，用来创建一类对象的蓝图。在Python中，定义类使用class关键字，类的名字通常采用驼峰命名法，首字母大写。比如，我们定义了一个名为Person的类：

```python
class Person:
    pass
```

在这里，我们定义了一个空的Person类，它代表了一个人，但还没有具体的人在此类之上。接下来，我们可以给Person类添加属性、方法等来描述它所代表的真实对象。

举个例子，假设我们有一个人的类，其属性包括名字、年龄、体重、住址等，这些属性值由实例变量表示。另外，该类可能有一些动作，如说话、跑步、学习、工作等，这些方法由实例函数实现。例如，假设有一个人在这个世界上遭遇危险，他应该做什么？根据不同的情况，我们可能给他不同的反应。如下所示：

```python
class Person:
    
    def __init__(self, name, age):
        self.name = name   # 属性：姓名
        self.age = age     # 属性：年龄
        
    def run(self):        # 方法：跑步
        print("正在慢慢奔跑...")
        
    def speak(self):      # 方法：说话
        print("你好，我是%s!" % self.name)
        
p1 = Person("John", 25)    # 创建一个Person类的实例p1
p1.run()                  # 调用实例p1的run()方法
p1.speak()                # 调用实例p1的speak()方法
```

上面的代码定义了一个Person类，它有两个实例变量：姓名（name）和年龄（age）。它还定义了三个方法：跑步（run）、说话（speak）。在最后的代码中，我们创建了一个Person类的实例，并调用了它的run()方法和speak()方法。
