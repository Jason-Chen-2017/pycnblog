# 1.背景介绍


随着智能设备的普及和应用范围越来越广泛，物联网(IoT)已经成为一个相当热门的话题。其架构设计、协议规范以及应用场景也越来越丰富。物联网的边缘计算(Edge Computing)技术也逐渐被重视。借助边缘计算的特性，使得云端的计算能力能够更加集中地处理数据流，从而提升了整体效率。但同时也带来了一系列的挑战，比如如何选择合适的硬件设备、如何优化部署方案、如何保证安全性、如何进行资源管理等等。本文将从物联网和边缘计算两个角度出发，分别阐述这一主题的相关知识，并结合实际案例，给读者提供一些参考。
物联网是指由各种智能终端设备组成的网络，利用这些设备之间的互联、通信、信息采集与处理功能，实现数据的收集、存储、传输、分析、处理、控制等功能。物联网的主要特点包括智能化、标准化、可编程、动态更新等。其核心技术包括LPWAN、Zigbee、Wi-Fi、蓝牙、传感器、单片机、微控制器、路由器、NAS、服务器、数据库等。当前国内外已经涌现出很多物联网公司，如华为、小米、浪潮、格力电器等。随着智能终端设备的不断增加，这些公司正在围绕“海量、低时延、高实时”的关键技术，展开新的商业模式，希望通过“连接未来”实现真正的价值创造。
边缘计算作为物联网的重要组成部分之一，是一个离线的、分布式的计算平台。它可以在端侧设备上执行任务，同时减少用户的响应时间。在基于边缘计算的物联网系统中，各个节点之间的数据交换可以简化，降低了网络拥塞和传输时间。同时，边缘计算还可以有效解决传感器、传感器组合以及网络带宽等资源的缺乏问题。除此之外，边缘计算还具有安全、隐私、可靠、可扩展等特性，因此也受到广泛关注。
在物联网、边缘计算这两个领域的不断发展过程中，产生了很多技术上的问题与挑战。比如如何构建有效的物联网系统？如何进行可靠、可控的部署与管理？如何对传感数据进行统一的管理？如何兼顾性能、成本、功耗等方面的考量？如何应对不断增长的边缘计算计算资源？这些都是需要面临的问题。本文将尝试从以下几个方面，总结本次博文的学习笔记，并且分享相应的经验心得和建议：

1. 物联网基础知识
    本文首先回顾物联网的相关概念以及基本术语，然后详细讲解物联网的核心技术，如LPWAN、Zigbee、Wi-Fi、蓝牙、传感器、单片机、微控制器、路由器、NAS、服务器、数据库等。
