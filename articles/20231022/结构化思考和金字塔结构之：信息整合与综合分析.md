# 1.背景介绍


## 概述
互联网信息瞬息万变，时代的变化让人们越来越关注信息的价值、价谈信息的应用场景和功能特性。而信息获取和整合成为了日常工作中不可缺少的一环。如何从海量数据中提炼出有意义的知识，成为不同业务领域的优秀通才，成为行业里不可或缺的人才，成为真正的“智者”也是一个十分重要的研究课题。作为一个系统工程师，在处理和整合海量数据的过程中必然需要对各种信息的特性进行综合考虑和理解，以期达到事半功倍的效果。比如，一条新闻中既包括财经类的关键词又包括房地产类关键词，如何快速识别其中的信息关系、关联度以及长尾效应？针对这种复杂的需求，系统工程师往往需要掌握一些高级的技能，包括信息检索、文本挖掘、信息统计、知识抽取、数据可视化等。但同时，如何将这些技能有效运用于解决实际问题也至关重要。系统工程师可以运用自己的技能从不同角度探索信息，形成独特的思路，找到最佳路径，完成创新性的工作。因此，结构化思维是理解和应用信息技术的关键一环。它帮助系统工程师洞察复杂的信息环境，找出有效的整合策略，转化高纬度、多样化的数据为有价值的信息。

信息技术的发展不断促进着对人的要求的升级，而结构化思维则是一种辅助工具，更好地适应信息时代的人文科技思维，促进信息系统工程师对未来信息技术发展方向的规划和布局。结构化思维是一种方法论，用来帮助人们识别并管理复杂的信息。它不是单一的技能或工具，而是一种处理信息的方式和思维方式。应用结构化思维的方法包括信息分析、信息设计、信息通信、信息系统开发等。例如，在信息系统设计中，结构化思维被用来形成详尽的系统功能列表和需求文档。结构化思维还可以用来梳理复杂的过程及结果，形成有效的管理和协作机制。应用结构化思维技术可以实现组织目标的达成，降低成本和风险，提升工作效率。

## 金字塔结构的提出
结构化思维的理论基础是金字塔模型。金字塔模型由图形学家威廉·达芬奇在19世纪提出，他认为人类认知世界的方式可以被分解为一定层次的分析层。较高层的分析能够捕捉较小层的信息，然后反馈给较低层进行再加工。金字塔模型主要由四个阶段组成，即信息、主题、结构和意识。按照各自的权重顺序，金字塔的上部为信息层，中部为主题层，下部为结构层，上下的关系为意识层，左右为流向。金字塔模型主要用于分析复杂的问题和事务，把复杂的信息组织起来，从多个角度观察问题的结构。
