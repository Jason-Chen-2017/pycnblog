# 1.背景介绍


电子束光刻与多光子光刻是非常重要的光刻工艺，也是高精密电子设备及其数字集成电路制造、新能源传感器的制造等领域中的重要组成部分。本文主要讨论电子束光刻和多光子光刻技术在材料光刻工艺领域中一些基本理论、方法和工艺流程。
首先，介绍一下电子束光刻与多光子光刻的区别。电子束光刻的实质是将不同层次的晶体分层聚焊在一个平面上，通过激光照射施加电压使晶体中的不同层面之间的电子相互作用，从而实现电子束的形成，这种光刻工艺手段最早由梅克尔·约翰逊发现并提出，应用于广泛的电子学用品制造、光学显示器件的制造、显示屏幕的制造、激光打印机的制造等领域。后来发现，这种光刻工艺存在着缺陷，如晶体凹凸不平、缺乏良好的掩蔽性、晶体表面容易开裂、膨胀等。因此，出现了多光子光刻技术，这是一种利用微米波、红外线或激光等探测电磁波的远距离传播特性来实现晶体的光刻化。多光子光刻由于采用了一种更先进的光刻工艺，其制备速度、尺寸可控，能够实现高分辨率、高层次、高精度的光刻化，目前已成为高精密科技产品的标准光刻工艺。
# 2.核心概念与联系
## 2.1电子束光刻
电子束光刻的基本原理是通过在不同层面的晶体分层聚焊，使各层电子之间发生相互作用，从而形成一个电子束。这个过程可以使用激光或者声光共振的方法进行。电子束光刻的工艺流程包括：第一步是在晶圆体表面涂层薄膜，并对其进行掩膜处理，掩蔽掉低频噪声；第二步是利用激光照射，使晶圆体中的不同层面之间的电子相互作用，在此过程中会形成电子束；第三步是使用激光下攻击带（DLA）或者声光联合干涉法（SLS）将电子束固定在晶圆体表面，然后用高速摄像头拍摄，最后将图像数据转化为电路设计图。
电子束光刻的优点是具有良好的掩蔽性、耐候性强，可以保持晶体的结构完整性，缺陷是晶体大小受限于掩膜的大小、多数情况下晶体表面开裂程度较高、晶体凹凸不平、无法做到真空腔浓缩、各层面光通量差异大。一般来说，电子束光刻工艺耗时较长、使用激光资源少，但结果精度好。
## 2.2多光子光刻
多光子光刻的基本原理是利用微米波、红外线或者激光等探测电磁波的远距离传播特性，通过光波将晶体的各层面上的电子区分开来。多光子光刻的工艺流程包括：第一步是准备待光刻化晶体，用镀锌钴作为晶体表面活性剂；第二步是在晶体表面涂层白化层，并进行掩膜处理，避免光波在晶体表面折射；第三步是在晶体表面架设探测器，通过探测器反射回来的光波，就可以获得不同层面之间的光谱分布信息；第四步就是结合上一步的光谱数据，可以计算出晶体各层面的厚度信息。之后用此信息计算出各层面电子的位置分布，利用此位置分布信息，可以绘制出晶体的电路设计图。多光子光刻的优点是能够实现高精度、高层次、高分辨率的光刻化，能够制备微型的、轻巧的晶体电路板；缺陷是不适用于晶体完全无需干涉的情况、需要高昂的光学性能。
