# 1.背景介绍


“影响力”在企业文化中占有重要地位，也是衡量一个人的价值的关键指标。

随着互联网、智能手环、虚拟现实等新兴技术的兴起，互联网商业模式开始进入转型时期，企业需要创造更多新的价值。如何从传统的线下组织架构转变为网络虚拟社区，让个人能够快速融入信息、沟通协作，将成为至关重要的能力。

如何衡量、提升和激励员工的影响力，成为当前重要且紧迫的课题。现有的各种评价机制，如年终奖、绩效管理、晋升制度等，不能很好地反映员工真正的价值和工作表现。而面对新型网络虚拟社区带来的全新的组织架构和流程，如何通过合理的分配和激励机制来实现“高级知识生产”和“持续学习”，进一步激发员工的潜力，是需要我们进一步思考的问题。

本文将以“影响力”为中心，结合“人际关系”、“金字塔结构”、“共同目标”、“理想生活”四个方面，讨论当前影响力建设领域最热门的研究问题。基于此，作者将从以下几个方面进行阐述：

1. 影响力的定义及其内涵
2. 影响力的维度与层次

# 2.核心概念与联系
## 2.1 影响力的定义及其内涵
影响力是一个复杂的概念，它既包括感知到的那些物质或者行为带来的影响，也包括其他人的影响。影响力是通过相互依赖和互动形成的。比如，张三的名声就取决于李四的赞美，而李四的赞美又会受到周五的鼓舞。

影响力一般分为三个层次，即生理、社会、文化。生理层面的影响力体现在心灵身体发育，这是由于天赋的作用；社会层面的影响力体现在职场、家庭环境、企业声誉、声望等因素，是群体中多样性的凝聚力；文化层面的影响力体现在个人道德品质、良好的职业道德、爱国主义精神等社会规范，是个体的塑造和建构。

第一层次的影响力主要体现在职业或事务上的影响，比如说所在公司的股票收益、上司的赏识、培训班里获得的职称证书、主管部门的重视等。这些影响力具有客观性、可测量性、可追溯性，但由于不涉及个人情感，也难以激发个人的自我驱动力。

第二层次的影响力往往涉及个人的情感需求和热情，比如说对爱人的深情款待、对亲近的人的关怀、对朋友的帮助等。这种影响力具有主观性、具象性、可触达性、激励性。

第三层次的影响力更加超乎常理，涉及到个人的非正式角色和责任，比如说关切他人的隐私、参与共同的事业、担当企业使命等。这种影响力具有主观性、不可控性、无法量化、难以预测性，往往伴随着巨大的压力和风险。

总的来说，影响力除了具备基本特征外，还应兼顾不同层次之间的交互作用。只有充分考虑每一个人的影响才能有效地激发共鸣，营造出真诚的互动氛围。

## 2.3 影响力的形成过程
影响力的形成通常分为四步：认知、沟通、重视、付诸行动。

1. **认知**
   在这第一步中，影响力产生的根本原因就是人的潜意识——人们喜欢被别人理解。如果一个人能够清楚地知道自己拥有的财产和能力，并且可以用这些资源去做有益的事情，他们也许就会倍感荣耀。
2. **沟通**
   在第二步中，影响力形成的方式就是通过正向的、积极的沟通。正向的沟通可以促进彼此间的理解和信任，也可能通过倡导互相支持来提升自己的影响力。反之，负面的沟通则会削弱影响力，降低自我价值。
3. **重视**
   在第三步中，影响力由各种不同的因素引起。比如，一种优秀的教育、良好的管理或金钱支持都有助于人们获得认同和尊敬。因此，为了取得成功，人们需要花费大量的时间、金钱和情感投入。
4. **付诸行动**
   在第四步中，影响力的最终形态就是行为的结果。比如，在重要的考试上通过良好的成绩、组织活动得到别人的青睐，这些都使得人们获得成功。而失败者则会因为不懈的努力而丢掉生命中的希望。

## 2.4 对人际关系的影响力评估
### 2.4.1 影响力影响力发生的原因
影响力的发生往往与人际关系密切相关。具体来说，影响力首先是由感觉得到的，比如对于外界刺激或亲密关系的好奇、喜爱程度，或对于挚爱的人物或事物的关注，这种影响往往具有一定程度的自然性和主观性。但是，影响力也经常被外部条件或环境所影响，比如权威性的任职、威望、名气等，也可以导致影响力的形成。

### 2.4.2 影响力影响力评估的方法
影响力评估的方法有两种：
1. 直接影响力评估法（直IEF）
   此方法认为影响力的发生需要有明确的规则，并遵守这些规则。比如，如果两个人没有充分沟通，影响力就不会在两者之间建立。在直IEF中，主体只能根据规则判断被评估对象的影响力。例如，一位老师想要知道他学生的听课表现是否令人满意，但学生只能根据上课情况来评判。
2. 潜力影响力评估法（潜IEF）
   此方法侧重于分析被评估对象潜在的能力和机遇，看看自己是否能够利用这一点来增强影响力。潜IEF使用一系列的技巧来分析潜在的能力和机遇，并预测评估对象的潜力。例如，一位演员可能考虑一下自己是否会成为电影明星，然后决定拍摄一部影片，对这位演员来说，潜力IEF是一种更科学、更客观的方法。

### 2.4.3 影响力的评估标准
影响力的评估标准很多，例如社会地位、名誉、职业资格、健康状况、幸福感、情绪控制、人缘情谊等。不同的影响力评估方法使用的评估标准也不同，其中一些标准比如成就、声望、权威性等是不可替代的。

### 2.4.4 影响力的影响力评估效果
影响力的评估效果常常取决于以下三个方面：
1. 对象受众的特质和需求
   不同的受众对影响力的需求和期望存在差异，例如男性会更看重名誉、职业资格等评级标准，女性更注重亲密关系和亲情。
2. 评估对象作为个人还是团队
   如果评估对象作为个人，那么单一的行为就可能产生较大的影响力。但如果评估对象是一个团队，那么它需要把多个人的影响力综合起来才能产生显著的影响。例如，一个队长可能希望尽可能地提升所有队员的综合能力，而不是单独评判一名队员。
3. 评估方式的准确性
   影响力的评估往往采用多种方法，每个人可能都能给出不同的看法，因此，评估结果不能只凭直觉得出。但也应该注意的是，如何客观准确地衡量一个人的影响力至关重要，否则，评估结果可能会受到评估方法的影响。

## 2.5 对金字塔结构的影响力影响分析
金字塔结构是一个强大的工具，用于建构复杂的组织架构。它可以让管理人员很容易地看到各个层级的上下级关系，并掌握各个层级的工作任务。但是，管理人员很少会注意到金字塔结构在影响力建设中的作用。

金字塔结构的一个缺陷就是扁平化。虽然层级越低，上下级关系越近，但实际上，这种关系更多是在意识层面上，也就是说，即便是层级最低的一项任务，都会影响到全队甚至更远处的成员。因此，在管理中，需要做到“分而治之”，即把复杂的工作安排划分成可管理的小任务，然后再分配到各个成员手中。这样，各个成员就不会成为瓶颈，而是能发挥更大的作用。

### 2.5.1 金字塔结构对影响力影响的类型
金字塔结构对影响力影响的类型包括“扩张型影响力”、“集中型影响力”和“裂变型影响力”。
#### 2.5.1.1 扩张型影响力
扩张型影响力在管理上被广泛应用，它往往源于对某些人的过分重视。对于那些比较有才华的工作者，他们往往被高层的管理层所重视，不管他们的工作如何出色，但若跟普通员工一样，往往会得到重视。

扩张型影响力的典型例子是财务部副总裁戴安娜·哈迪。她是著名的世界顶级律师事务所Evercore Group的总裁，同时也是著名的旅游公司Hilton的董事长。她虽然是一个资深的金融家，但她的社会声誉却很高，所以她不断地被邀请参加一些有权势的活动，包括举办盛大的婚礼、举办宴会等。结果，很多公司都纷纷聘用她做副总裁，包括Wal-Mart、Disney、AT&T、Facebook等。

另一个典型的扩张型影响力例子是媒体大V陈冠希。陈冠希是2015年的中国首富，但她却被豆瓣CEO王石所捧为“美国第一富翁”。她在微博上连发十几条短短的微博，极大地刺激了许多人成为这个圈子的一员，甚至还有些人忍无可忍地问她：“你为什么要那么红？”

#### 2.5.1.2 集中型影响力
集中型影响力是指一个人被高度评价，他的任务和目标会得到高度重视。集中型影响力往往伴随着管理的个人目标，包括公司的营销或产品策略，或个人的事业发展。一个人的成功往往会被他身边的人所吸引，特别是在学校或领导岗位上。

集中型影响力的典型例子是前总统尼克松。尼克松曾是歌唱革命的先驱，曾是“星球大炮”的驻军，后来却被认为是特务的头号策动者。然而，尼克松在参与战争后，他的声誉逐渐得到提升，被视为当代“美国黄金帝国”的领袖。另一个典型的集中型影响力例子是著名的政治犯弗兰克·福山。福山在2000年至2008年期间频繁出演“布鲁斯大姐”电影，受到了全球关注。

#### 2.5.1.3 裂变型影响力
裂变型影响力是指个人的成功使得整个团队都受到重视。裂变型影响力往往伴随着突破、团队精神，以及精英队伍中的独特性。这种影响力使得人们的目光短浅，容易被他人牵着鼻子走。管理人员需要识别这种影响力的模式，并应用适当的沟通策略来鼓励团队的发展。

裂变型影响力的典型例子是克隆人。克隆人可以复制他人的性格、思维模式、竞争力、智慧和记忆力，使得一个人的成功会影响整个团队。克隆人已经在竞技比赛中扮演重要角色，包括在雪铁龙和皇冠发布会上给员工讲授比赛规则。另外，罗纳德·里根曾经是红衫军的一员，但在20世纪90年代却因表演滑雪技术而得罪了一位影星。
