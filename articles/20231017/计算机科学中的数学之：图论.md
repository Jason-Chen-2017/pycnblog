# 1.背景介绍


## 一、什么是图？
图（graph）是一种数据结构，它用来描述节点之间的关系。在计算机网络中，图被广泛应用于表示复杂系统中各个组件之间信息流动的情况。图可以用来表示现实世界中事物之间的联系，例如，电路图或地图就是图的一个例子。图还可以用来表示现实世界中的各种实体及其之间的关系，例如人际关系网络就是一个典型的图。

图的基本元素是“顶点”（vertex）和“边”（edge），顶点代表一个对象，边代表连接两个顶点的线条。在实际应用中，顶点可以具有属性值，边也可以具有权重值，这使得图成为网络结构、生物相关网络、商业关系网等各种复杂网络结构的有效描述工具。

常见的图结构有：无向图、有向图、加权图、带权图、环状图、二分图等。除了以上介绍的图结构外，还有一些其他特殊的图结构，如树、星形图等。

## 二、图的性质
### （一）无向图
无向图，又称简单图，是由无向边组成的图，即任意两个顶点间都存在一条边相连，但没有方向性。换句话说，无向图中不存在回路（cycle）。比如，银行里的交易网络、互联网上用户之间的社交网络等都是无向图。

无向图的性质：

1. 任何两顶点间都存在路径相连，即任意两个顶点间都可通过某条边到达。
2. 没有平行边对。即任意两个边的两端顶点不相同。
3. 没有自环。即某个顶点不可能同时和自己邻接。
4. 同一顶点不能有多个邻居。除非有多条边连向该顶点。

### （二）有向图
有向图，也称有向图或有向网，是由有向边组成的图，即每条边都有一个方向性，指向从其中一个顶点到另一个顶点。有向图可以用来表示在不同事件之间因果关系、事物的传递关系或者任务的优先级等。

有向图的性质：

1. 每个顶点至少有入度一条边，因为若没有入边，则无法到达这个顶点；每个顶点至少有出度一条边，因为没有出边意味着这个顶点不会影响其它顶点的状态。
2. 有向图中不存在回路（即不允许出现从顶点a经过顶点b再回到顶点a的情况），否则就会产生死循环。因此，有向图中一般不包含负权值的边。
3. 有向图可以是树，反之亦然。

### （三）加权图
加权图，也称赋权图或赋权网络，是指给图中的每条边分配了一个非负实数权值。由于权值越大的边会更重要，所以通过加权图可以找到重要的信息。比如，一张餐馆的菜单图就是一种加权图，表示了菜品的重要程度以及其相互之间的依赖关系。

加权图的性质：

1. 如果图中所有边的权值都是正数，那么它就是无向加权图。反之，如果图中有负权值边，那么它就是有向加权图。
2. 对于无向图，加权图的最小生成树算法是 prim 算法。
3. 对于有向图，加权图的最小生成树算法有两种，一种是基于 Prim-Jarnik 算法，另外一种是基于 Dijkstra 算法。

### （四）带权图
带权图，也称带权网络，是指给图中的每条边分配了一个实数权值，其值可以正、负或零。带权图可以用来刻画某些事件或过程的影响范围。例如，公路网络就属于带权图。

带权图的性质：

1. 在一个带权图中，边的权值总是非负的，因此图中的任意两顶点之间至少有一个负权值的边。
2. 当一个图的边的权值满足连续性递减时，它就是一个单调图。
3. 在一个带权图中，每个顶点最多只能到达一个其他顶点且权值为负，也就是说，它只能使某些路径权值变小而不能增加。

### （五）环状图
环状图，又称环图，是指任意一对顶点间存在一条环。环状图有时候也叫做“圈子”。环状图的特点是节点间存在着紧密联系，并且紧密的联系可能穿插着许多其他的关系。比如，互联网上的社交网络就是一种环状图。

环状图的性质：

1. 如果一个图是环状图，则必然存在着一组起始点和终止点。起始点是指某个节点到自身的距离为零的点，终止点是指某个节点到其余所有点的距离等于它的度数的点。
2. 如果一个图不是环状图，则一定不存在这样的一组起始点和终止点。
3. 环状图的中心有且只有一个。
4. 对于有向环状图，入度和出度分别等于顶点的度数。

### （六）二分图
二分图，也称匹配问题，是指图中任意两顶点都要么互不相连，要么互相连通，但不能同时相连。这种图称为二分图。二分图的关键特征是找出其中一组顶点，使得任意两组顶点都不相连，但如果将其中一组作为一个集合并去掉其中的所有边，剩下的图就是一个二分图。例如，股票市场上存在着的一组虚拟基金和对应实力股的二分图。

二分图的性质：

1. 对于一个二分图，任意两组顶点不可能全都相连，但可以互相连接。
2. 在一个二分图中，任意两组顶点不可能全都不相连，但至少有一个不相连的顶点。
3. 如果一个图是二分图，则必然可以找到一组最大匹配，这是指使图的任意两组顶点不相连的最大的匹配。
