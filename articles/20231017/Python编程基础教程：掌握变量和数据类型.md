# 2.核心概念与联系
## 变量
计算机程序的运行过程，离不开数据的存储、处理、运算。而在存储和处理数据时，就需要用到变量。变量是一个可以变化的量或物体。它代表了某一个内存位置，用来存放某个值的容器。

## 数据类型
变量的数据类型决定了该变量保存的值的类型和形式。不同的数据类型占用的内存空间大小不同。数据类型主要分为以下几类：
- 整型（int）:整数类型。通常情况下占用内存较小，一般都取值为正整数，也可取负整数、零。例如：-10，0，1000。
- 浮点型(float):浮点数类型。包括单精度float、双精度double。通常都是用十进制表示。例如：3.14，-2.7。
- 字符串(str)：字符串类型。由多个字符组成的序列，用单引号'或双引号"括起来。例如："hello world"，'Python'.
- 布尔类型(bool):布尔类型只有两个值True和False。当某表达式被求值时，如果结果为真，则返回True，否则返回False。例如：a > b。
- 列表(list):列表类型。是一种有序集合，可以包含不同的数据类型，可以按索引访问其中的元素，并且可以动态修改其长度。例如：[1, 'a', True]。
- 元组(tuple):元组类型。类似于列表，但是其内容不能修改。例如:(1, "a", False)。
- 字典(dict):字典类型。也是一种映射结构，用于存储键值对，其中值可以是不同的数据类型，通过键进行查找。例如：{'name': 'Alice', 'age': 25}。

## 符号的含义及其关系
- =：赋值运算符，将右侧的值赋给左侧的变量。如 a=b+c 将变量a的值设为变量b加上变量c的结果。
- +：加法运算符，用于数字相加。如 a+b 返回变量a的值加变量b的值。
- -：减法运算符，用于数字相减。如 a-b 返回变量a的值减去变量b的值。
- *：乘法运算符，用于数字相乘。如 a*b 返回变量a的值乘以变量b的值。
- /：除法运算符，用于数字相除。如 a/b 返回变量a的值除以变量b的值。
- //：地板除法运算符，用于整数除法。如 a//b 返回变量a的值向下取整除以变量b的值。
- %：取模运算符，用于取余数。如 a%b 返回变量a的值除以变量b的值的余数。
- ==：等于判断运算符，用于比较两边的值是否相等。如 a==b 返回True表示变量a的值等于变量b的值，否则返回False。
-!=：不等于判断运算符，用于比较两边的值是否不相等。如 a!=b 返回True表示变量a的值不等于变量b的值，否则返回False。
- <：小于判断运算符，用于比较两边的值是否小于。如 a<b 返回True表示变量a的值小于变量b的值，否则返回False。
- <=：小于等于判断运算符，用于比较两边的值是否小于等于。如 a<=b 返回True表示变量a的值小于等于变量b的值，否则返回False。
- >：大于判断运算符，用于比较两边的值是否大于。如 a>b 返回True表示变量a的值大于变量b的值，否则返回False。
- >=：大于等于判断运算符，用于比较两边的值是否大于等于。如 a>=b 返回True表示变量a的值大于等于变量b的值，否则返回False。
