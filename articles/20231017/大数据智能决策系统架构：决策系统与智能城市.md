# 1.背景介绍


人们生活在数字化、信息化社会，随着经济的发展、生产力的提升、生活消费水平的提高、环境污染的防治以及社会公众对互联网的需求越来越高，人们在选择、消费、娱乐等方面都面临着新的问题，决策系统应运而生。在互联网爆炸时代，人们享受着网络带来的便利，可以进行即时的、精准的、自动化的信息采集和分析，并基于这些信息制定出满意的购物方案，还可以提供个性化的咨询服务。然而，由于互联网的迅速发展、海量数据的涌入、复杂的信息依赖于各种决策因素之间的交互关系，给决策过程引入了很大的不确定性。如何建立一个具有智能化、高效率、可扩展的决策系统就成为当前研究的热点。智能城市是由数字化、智能化、综合化的四项特征构成的城市，其功能定位于提供全面、即时的、智能化的市民生活服务。通过对用户多元化信息及需求的收集、分析、理解、处理和应用，智能城市能够帮助人们快速、高效地找到满足自身需求的各种服务或商品，并形成独立的个性化推荐引擎。智能城市平台构建的基础就是智能决策系统。

# 2.核心概念与联系
本文将从两个视角介绍智能城市的决策系统架构：智能决策系统（Intelligent Decision System）以及基于知识的计算方法（Knowledge-based Computing）。首先介绍智能决策系统。
## 智能决策系统（Intelligent Decision System）
智能决策系统是指能够实现自动决策、辅助决策以及集成决策的决策支持系统，其具备以下几个主要特点：

 - 数据驱动型：智能决策系统对所有要处理的数据、信息进行跟踪、记录、分析、处理、储存、利用。
 - 模块化设计：智能决策系统可以分解为多个模块，分别处理不同的任务。
 - 组合式设计：智能决策系统可以进行不同模块的组合，形成更加智能的决策结果。
 - 实时更新：智能决策系统根据最新信息及时调整策略，确保决策的连续性、及时性及效果的稳定性。
 - 个性化设计：智能决策系统根据个人偏好、习惯或能力，对决策过程进行优化。

智能决策系统通常包括四个层次：输入层、规则层、推理层、输出层。其中，输入层负责获取待决策信息的获取、整理、存储；规则层负责定义抽取信息中关键词、类别和规则的识别和匹配；推理层采用所需的技术手段，对数据进行分析和预测，形成可信的结论或判断；输出层将得到的结论或判断反馈给决策者，最终达到决策目的。

本文所讨论的智能城市决策系统架构，就是基于这种四层结构的智能决策系统。
## 基于知识的计算方法（Knowledge-based Computing）
基于知识的计算方法是一种新型的计算机计算方法，它利用计算机对事物的知识和经验进行建模、表示、检索、推理、归纳、推断、优化、改进、总结、决策、管理和控制等，从而获得系统的智能行为。该方法的基本思路是把认识（知识）转变为计算机程序执行的指令，并让计算机自己根据已知信息、环境条件、知识库、规则以及学习过程进行推理、归纳、比较、分析、决策等，从而完成指定的目标。典型的基于知识的计算方法有：符号逻辑方法、规则系统、图形推理、神经网络、机器学习、贝叶斯概率、遗传算法、聚类分析、分类器方法、关联分析等。

基于知识的计算方法与智能决策系统之间存在密切的联系，因为基于知识的计算方法的目的是学习、应用、归纳、归纳、归纳、总结、模拟、演绎等，在此过程中得到的知识或经验就可以用于智能决策系统的决策支持。比如，基于统计方法、模式识别方法的图像识别、语音识别、文本处理，以及监督学习、非监督学习、半监督学习、强化学习等技术，都可以用于智能决策系统的开发。因此，了解智能决策系统的前置知识有助于理解基于知识的计算方法。
