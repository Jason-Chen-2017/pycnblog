# 2.核心概念与联系
## 什么是物联网？
物联网(IoT)，英文全称Internet of Things，它是一种将互联网技术、计算机技术、信息技术、经济学、管理学、社会学、心理学等多学科综合应用于物理世界的网络。它是指利用Internet、 wireless communication 和 sensor networks，使得各种物体或对象能够相互连接、交换信息，并实现对物体进行远程控制、数据采集、实时监测、远程操控的技术手段。其主要特点包括：

 - 大规模分布：物联网要面临越来越复杂的分布式计算和高带宽通信技术，能够提供海量数据。
 - 连接性：物联网连接着不同的设备、传感器、终端，能够实现远程传输、信息共享。
 - 智能化：物联网智能化的特征包括人工智能、机器学习、模式识别、语音识别、图像识别等技术。

## 如何应用物联网？
物联网可以用于多种场景，如：

 - 监控环境：通过多种传感器，例如温度计、震动传感器、光照传感器，可以监控房屋、楼宇、空气质量、健康状况、人流量等各个方面，进行及时预警，提升用户体验。
 - 自动化：通过各种智能家居设备，例如电风扇、冷氧机、扫地机器人、吸尘器等，可以实现洗衣服、扫地、制冷、清洁等重复性工作，提升用户效率。
 - 服务支持：物联网还可以用于医疗、安全、养老、环保、金融等领域，用于提供服务支撑、促进交流、提升效率。

## 为什么要做物联网应用开发？
物联网应用开发一直是热门话题，为什么要做这个开发呢？因为这样就可以在未来构建更加先进的智能设备，打造出让人惊叹的智能生活。如下图所示，物联网开发的四个阶段:

 - 数据采集阶段：通过不同类型的传感器收集和传输数据，形成原始的数据。
 - 数据处理阶段：对数据进行分析、过滤、归纳、转换、优化等处理，生成可供分析的结果数据。
 - 数据传输阶段：将经过处理的数据发送到后端服务器，实现数据持久化存储。
 - 数据展示阶段：将经过处理的数据进行可视化、图形展示、告警通知等，形成可以直观呈现的界面。
