# 1.背景介绍


提示工程(Prompt engineering)是一种语言学术语，指的是在学习某个主题时，使用富含提示、指导性信息的教学方式。由于每种主题的具体知识点众多，且难以全部覆盖到面面俱到的知识内容，所以，现代大学生和研究生都开始学习或转向研究更高级的学科领域时，往往需要掌握某种程度上的提示工程技巧才能做好笔记。

提示工程属于视觉呈现类学习方式，主要用于提高学生的英语水平、阅读能力及逻辑思维能力。其特点就是通过一些简单易懂的富有启发性的示例、故事、知识点等引导学生走进课堂，然后根据学生的实际反馈进行修正，从而达到有效提升学习效果的目的。

提示工程最常用的方法有四种：
- 用图形展示知识点
- 提供多个版本的练习
- 播放有声读物或互动视频
- 在课堂上使用板书工具记录知识点

这里我们先来介绍一下这些常用方法的基本知识。

# 方法一：用图形展示知识点
这是一种较为直观的方法，一般适合对已有的经典知识点进行深入阐述、扩展和实践。比如，我们可以制作一个多边形图形，每个多边形代表某个知识点或技能，并以箭头将图形连接起来。如下图所示：

# 方法二：提供多个版本的练习
这种方法比较流行，尤其是在考试中。一般在课程中设置几个具有挑战性的问题，然后给出不同的答案，让学生选择答案中的正确选项。这样可以增加学生对问题理解的深度。

例如，对于课堂上常见的“男女比例”问题，我们可以提供两种不同版本的测试题目：

1. 如果只是希望了解男女比例，那么答案可能就只有“男生占比3:1，女生占比1:3”。
2. 如果希望了解男女比例对社会经济发展的影响，我们可以说：“在美国，女性的数量占到总人口的24%左右，这说明女性越来越成为经济的主力军，她们支配着社会的最底层阶级，享受着较大的经济福利。”

这种方法既可鼓励学生独立思考，又可增加题目数量，加强学生对知识的理解。但同时需要注意控制题量，不能过多，否则会分散学生注意力。


# 方法三：播放有声读物或互动视频
这一方法属于教学生活中的趣味活动，尤其适合解决复杂或艰涩的知识点。通过讲解动画电影或小视频的方式，能很好的将知识融入到课堂氛围中。但是，这种方法往往受到学生注意力分散的影响，可能会导致学生误入歧途，从而影响学习效果。所以，这种方法通常只适用于单个课题，而不是整个课堂。


# 方法四：在课堂上使用板书工具记录知识点
这一方法在一定程度上也是一种提示工程，但它不仅限于单个课题上。它通过将所有知识点或技能以便于理解的形式在板子上进行摆放、标注，再加入一些练习，来帮助学生提高综合学习能力。具体方法如下：

1. 将知识点或技能按照重要程度排列整齐地分布在板子上。
2. 对每个知识点或技能进行简短的描述，并注明知识点、关键词或技能名称。
3. 使用锚点、任务图、流程图、决策树等图形工具进行辅助讲解。
4. 每个知识点后面都可以加入练习，以激发学生对知识理解的兴趣和求知欲。

这种方法虽然不是每个课题都适合采用，但它对于完成知识结构梳理、巩固知识点的关键、巩固复习知识点的作用尤为有效。而且，在此过程中，还能帮助学生提升自己的沟通表达能力、组织能力、分析能力、创新能力、团队协作能力等。

除此之外，提示工程还有很多其他方面的应用，比如：

- 在课堂上对概念和理论进行演绎式地阐述，可以激发学生对知识的理解、建立问题意识，提升学生的分析、表达能力。
- 通过实验性评估试题，来考察学生对各种知识点的掌握情况，判断是否需要对某些知识点进行补充讲解或重点强化。
- 通过提供拓展材料或科普资料，帮助学生理解复杂的学科理论，增强他们的科普兴趣。

最后，值得一提的是，提示工程作为一种高效的学习方式，其本质还是靠学生自己主动去发现问题、寻找答案，而非机械地死记硬背或者套用老师给的知识。因此，在使用提示工程时，应尊重学生自身的学习能力、学习节奏、反馈反馈的反映等，通过不断调整方式，增强学习效果。
