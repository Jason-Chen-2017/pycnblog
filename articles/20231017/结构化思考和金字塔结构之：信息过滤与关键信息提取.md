# 1.背景介绍

  
信息过滤（Information Filtering）和关键信息提取（Key Information Extraction）在实际应用中往往都是重点。随着互联网、移动互联网、物联网的发展，各种新型的信息来源越来越多，过滤和提取这些信息成为了一个重要的问题。作为信息处理的基石，信息过滤与关键信息提取理论已经成为很多学科的基础性理论。而且目前已经有一些关键技术被广泛应用到这些领域，例如，电子邮件过滤、微博监测、垃圾邮件过滤、网络入侵检测、语音识别、图像识别等等。因此，了解和掌握这些理论与技术，对于理解信息时代的数据产生、存储、处理、传播的方式，极为重要。
# 2.核心概念与联系  
## 2.1 定义
### 2.1.1 概念
**信息过滤(Information filtering)** 是指从一组无序的信息项中筛选出符合某种标准要求的项，并按照一定顺序输出。其核心功能是对无关的、不相关的信息进行筛选和排序，输出有效信息。如邮件过滤、垃圾邮件过滤等。

**关键信息提取(Key Information Extraction)** 是指从一段文本或文档中自动地抽取出有用的信息，使其成为关键点。关键信息提取的目的是为了能够快速准确地获取想要的信息，并帮助用户更加高效地利用信息资源。如通过搜索引擎搜索关键词获取相关文档，或者通过新闻头条、天气预报网站获取最新消息等。 

### 2.1.2 联系
**信息过滤与关键信息提取可以说是一类技术的集合。**由于它们的共同特点是从一组信息中抽取出有用信息并按一定顺序输出，所以可以认为它们具有相似的特性。因此，它们之间存在着很强的关联性。而这种关联性会影响到它们的技术发展方向、分析方法、实践效果等方面。如下图所示：

## 3.1 正则表达式
正则表达式（Regular Expression）是一个字符串匹配模式。它是一种文本模式描述语言，它能完成包括简单匹配、重复、选择、分组及更多复杂匹配方式在内的多种匹配任务。在信息过滤与关键信息提取过程中，正则表达式扮演了至关重要的角色。它提供了有效、简洁、高度可定制的过滤规则，可以帮助我们在海量信息中快速定位出那些符合我们的筛选条件的项。

### 3.1.1 简介
正则表达式是一个特殊的字符序列，它能帮助你方便、快捷地完成对字符串的各种匹配操作。它的基本语法规则非常简单，如\w表示任意单词字符、\d表示任意数字、\.表示任意英文句号、\*表示零个或多个前面的字符。正则表达式的匹配规则也是基于正则表达式的匹配结果来确定是否成功。比如，如果你希望匹配所有以“www”开头且长度为7-9之间的域名，那么你可以使用正则表达式\bwww[a-zA-Z0-9]{7,9}\.[a-z]+\b。

### 3.1.2 操作步骤
#### 3.1.2.1 找出所有符合条件的邮箱地址
假设要筛选所有的QQ邮箱，可以先尝试使用QQ的昵称注册的邮箱。假如你找不到这些邮箱的话，再考虑注册其他的邮箱。下面是找出所有符合条件的邮箱地址的方法：
1. 使用浏览器访问 QQ 的主页（http://mail.qq.com）。点击 “登陆” 链接，输入你的 QQ 账号密码。如果登录成功，你应该能看到你的收件箱。
2. 打开浏览器的开发者工具，切换到 Network 标签，刷新页面。
3. 在收件箱中找到任何一条邮件，然后鼠标右击，选择 “复制” 或 “拷贝链接”。
4. 将刚才的链接粘贴到记事本中，注意不要包含引号和空格。
5. 使用以下正则表达式匹配出所有QQ邮箱：\b[a-zA-Z0-9._%+-]+@qq\.com\b。替换掉“\b”表示单词边界，“.”表示任意字符，"+"表示至少出现一次，这里 \b[a-zA-Z0-9._%+-]+@qq\.com\b 表示以 "@qq.com" 结尾的所有由字母、数字、下划线、点、加号或减号组成的邮箱。
6. 使用 Python 或 Java 中的正则表达式包来进行正则表达式匹配。

#### 3.1.2.2 基于关键词查找微博热门话题
微博（Weibo，Chinese pronounced "wei bubble"）是中国最大的社会化媒体平台，由中国微软公司开发、拥有、运营。每天都有成千上万人使用微博进行生活更新、表达观点、参与活动。每一条微博都带有标签（hashtag）、图片、视频，甚至连地理位置也可能被记录。微博上的评论可以产生广泛而持久的影响力。因此，需要从微博中提取热门的话题，进行信息筛选。

下面介绍如何使用 Python 从微博 API 中提取热门话题：
1. 注册微博 API 账号，申请 APP_KEY 和 APP_SECRET。
2. 使用 Python 的 requests 模块，向微博 API 发起请求，获取微博热搜榜列表。
3. 解析 JSON 数据，提取 hashtags 字段中的数据。
4. 使用 Python 的 NLTK 库进行情感分析，筛选出含有褒义词或积极情绪的内容。
5. 将符合条件的内容保存到文件中，进行进一步分析。

#### 3.1.2.3 基于关键字过滤垃圾邮件
垃圾邮件（Spamming）是电子邮件中流通着的大量垃圾邮件，主要有两种形式：1）网页形式的垃圾广告；2）纯文字形式的恶意刷信或垃圾短信。垃圾邮件在很大程度上是因为发送者的质量较低或垃圾短信发送者的滥用造成的。下面介绍如何使用正则表达式过滤出垃圾邮件：
1. 使用编辑器打开待检测邮件，将其中包含的敏感信息删除，保留主要内容。
2. 根据电子邮件发送者的反馈，设置匹配规则。例如，可以匹配所有包含特定域名的邮件，或是匹配包含特定关键词的邮件。
3. 使用 Python 或其他编程语言编写脚本，实现邮件的检测。
