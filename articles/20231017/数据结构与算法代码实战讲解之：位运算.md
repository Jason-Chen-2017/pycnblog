# 2.核心概念与联系
## 位运算符简介
位运算符是一种特殊的运算符，用于对特定比特位进行操作。其功能包括按位与、按位或、按位取反、左移、右移等。位运算符在计算中一般都是逐位操作的，即一个操作对象会被拆分成若干个比特位，然后针对每一位进行操作。位运算符提供了一种快速有效的方法来进行算术和逻辑操作。

两个数字对应位相当于1时，结果位“1”，否则结果位“0”。也就是说，只要两个相应位有一个为1，结果就是1。例如，5 (0b101) 和 3 (0b011) 的按位或运算结果为 7(0b111)。

### 左移与右移
左移运算符（<<）将数字的各二进制位向左移动指定的位置，右移运算符（>>）则将数字的各二进制位向右移动指定的位置。移位操作可以用来扩大或缩小数字范围，使得更大的数值能够完整地表示出来。

对于一个数字来说，左移一位相当于乘以2，右移一位相当于除以2。因此，左移n位就等于乘以2的n次方，右移n位等于除以2的n次方。

例如，对于数字5 (0b101)，左移两位 (0b101 << 2 = 0b10100) ，右移一位 (0b101 >> 1 = 0b0101) 。

### 结合律
对于位运算符，结合律规则是相同优先级的运算符从左至右进行运算。比如，A & B | C，首先计算 A & B 的结果，再计算 A & B | C 的结果。如果改变计算顺序，比如 A & (B | C)，那么先计算 B | C 的结果，再计算 A & B 的结果。

## 位运算符在计算机中的实现
位运算符在计算机内部通常由指令或子程序来执行，而不是直接用编程语言实现。不同的编程语言对位运算符的实现也不同。这里我们给出几个常用的编程语言及其对应的位运算符的实现方式：

### C/C++
在 C/C++ 中，按位与、按位或、按位取反、左移、右移运算符分别对应 AND、OR、NOT、SAL、SAR 关键字。

### Java
Java 中的位运算符与 C++ 中的类似。

### Python
Python 没有内置位运算符，但是可以通过按位运算函数实现：bitwise_and()、bitwise_or()、bitwise_not()、left_shift()、right_shift() 函数。这些函数在 bitwise 模块里定义。

### JavaScript
JavaScript 不支持位运算符，但可以使用位运算函数来模拟它们。如，& 表示按位与函数，| 表示按位或函数，~ 表示按位取反函数，>>> 表示无符号右移函数，<< 表示左移函数。
