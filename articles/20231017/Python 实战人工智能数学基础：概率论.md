# 1.背景介绍


概率论是一门用于研究事件发生概率、不确定性的科学。在计算机领域，人工智能和机器学习等相关技术都涉及到大量的统计分析，包括机器学习中的数据预处理、参数估计、模型选择和分类、生成模型等等，而概率论也是这些统计分析的基础。概率论是统计学中一个重要分支，旨在研究随机事件发生的可能性。在本文中，我们将从基础知识入手，对概率论做一个全面的介绍。
概率论的起源可以追溯到古希腊哲学家贝叶斯（Bayes）的观点，他认为任何事物都有两种相互独立的状态——实验结果和样本空间。实验结果就是试验得到的结果，例如，抛掷一个硬币正面朝上的次数；而样本空间则是所有可能的结果组合，即硬币出现正面朝上或者反面朝上的两次。贝叶斯定理指出，给定某些样本空间的条件下，某事件的概率只与其发生的频率成正比，与其他条件无关。换句话说，当已知了某些条件时，某件事情发生的可能性就由该事情已经发生的次数占总体数量的比例决定。
概率论也称作“信息论”，它把整个世界看作由个别随机事件构成的集合，并通过信息论的发展发现了“客观世界”的稀缺性，进而形成了概率论作为数理工具的历史。概率论主要用来解决两个问题：一是描述信息的传递机制；二是利用数据对未来的预测和决策。概率论的研究早于其他的科学，因此，它的发展可以追溯至公元前500年左右。概率论是统计学的一个分支，与统计学不同的是，它注重观察者（人的主观意志）对随机事件的看法，所以一般学术界把概率论讨论的范围缩小为关于事件及其发生的假设。但随着信息技术的飞速发展，随着对世界的理解逐渐深入，概率论却成为了计算机科学、经济学、金融学、生物学、心理学、数学等诸多领域的基础。
# 2.核心概念与联系
## 2.1 概率
### 定义
在概率论中，概率（Probability）是介于0到1之间的数字，用来表示某个事件发生的可能性大小。通常用P表示。
### 常用记号
- P(A)：事件A发生的概率。
- P(A|B)：事件B已经发生的条件下，事件A发生的概率。
- P(A∩B):同时发生事件A和B的概率。
- P(A^C):事件A或事件C发生的概率。
- P(A^B):事件A和事件B各自发生的概率的和。
- P(AB)：事件A和事件B同时发生的概率。
- P(A+B):事件A和事件B任意发生一次的概率。
- P(S):随机变量X的分布函数。
### 求和律
P(AB)=P(A)+P(B)-P(A∩B)<=>P(A∪B)=P(A)+P(B)-P(A∩B)<=>P(AB)=min[P(A),P(B)]+max[P(A'),P(B')]-max[P(A'B),P(A'B')]<=>P(AB)=P(A)+P(B)-P(A')-P(B')+P(A'B)
### 乘积律
P(AB)=P(A∩B)*P(B/A)<=>P(AB)/P(A)=P(B/A)<=>P(AB)^2=P(A)*P(B)<=>P(AB)<=min[P(A),P(B)]*P(AB)<=max[(P(A)+P(B))/2]*P(AB)<=P(A'*B'+A'B'')<=>P(AB)>=P(A'*B'+A'B'')*(P(B')/P(A'))
### 分配律
P(A∪B)=P(A)+P(B)-P(A∩B)<=>P(A-B)=P(A)+P(B)-P(A∪B)<=>P(A-B)=-P(A'+B'+A'B')+P(A')+P(B')+P(A'B)
## 2.2 随机变量
### 定义
在概率论中，随机变量（Random Variable）是一个取值可以取到的数字、实数或离散的集合的符号。它代表了一件事物的样本空间，我们希望能够对这个样本空间进行分析，从而得出关于这件事物的信息。通常用X表示。
### 常用记号
- X:随机变量。
- x:随机变量的取值。
- p(x):随机变量取值为x的概率。
- f(x):概率密度函数。
- μ(X):期望值。
- σ(X):方差。
- ν(X):峰度。
- γ(X):偏度。
- 边缘分布：随机变量独立同分布，每个随机变量的概率密度函数都为常数。
- 可见分布：随机变量独立同分布，且每个随机变量的概率密度函数都可导。
## 2.3 联合分布
### 定义
在概率论中，联合分布（Joint Distribution）是描述多个随机变量间的关系的分布。它是一个表格形式的分布，其中每一行对应于随机变量的值，每一列对应于事件发生的可能性。它由概率密度函数描述，即对于任意一个联合分布p(x1,...,xn)，存在相应的概率密度函数f(x1,...,xn)。在概率论中，联合分布常被表示为p(x1,...,xn)。
## 2.4 概率分布函数
### 定义
在概率论中，概率分布函数（Probability Density Function，PDF）是一种定义概率密度的函数。在概率论中，如果随机变量X的概率密度函数为f(x)，那么X的概率分布函数就是满足如下条件的函数g(t)：

g(t) = int_{-\infty}^{+\infty} dx~ f(x) e^{-(x-t)^2 / 2σ^2}, (-\infty < t < \infty),

这里的int表示对x的上下限进行积分，dx是取integration unit的简写，σ^2表示X的标准差。

概率分布函数与概率密度函数类似，但是区别在于，概率分布函数表示取值落在某个区间内的概率，即积分。因此，可以根据概率分布函数计算某个事件发生的概率，如P(a<X<b)。

## 2.5 条件概率
### 定义
在概率论中，条件概率（Conditional Probability）是指在已知另外一个随机变量的情况下，另一个随机变量发生的概率。在条件概率中，假设有一个随机变量X和一个随机变量Y。假设Y由X给定的条件下发生的概率记为P(Y|X)，也就是说，事件Y在已经知晓了事件X发生的情况下发生的概率。
### 计算方法
在概率论中，条件概率是使用联合分布来计算的。给定事件X的某一具体值，则事件Y对应的联合分布第i行第j列元素的值等于P(Yi|Xi)，其中i、j分别表示Y的第i个取值和X的第j个取值。利用联合分布，就可以很方便地计算条件概率。

计算公式：
P(Y|X)=P(X,Y)/(P(X))

这里，P(X)是X的概率，P(X,Y)是X和Y的联合分布，/( )表示除法。

条件概率的几何解释：如果X和Y互相独立，即条件独立，则条件概率就等于X和Y的乘积。如果X和Y不独立，即条件独立性不成立，则条件概率就等于X在给定Y后，Y发生的概率除以X发生的概率。
