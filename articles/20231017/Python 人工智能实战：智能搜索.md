# 1.背景介绍


## 概述
当今互联网技术日新月异,在信息获取、共享、搜索引擎方面也产生了巨大的进步。而随着网络技术的不断发展，计算机对数据进行检索的方式越来越依赖于人工智能技术。其中，基于关键词搜索(Keyword Search)和语义理解(Semantic Understanding)等两大基础技术的智能搜索是十分重要的。本文将介绍Python实现智能搜索相关技术及其应用。

### Keyword Search
关键字搜索（keyword search）主要指通过搜索语句中的关键字找寻文档或者其他信息。通常情况下，关键词搜索通过索引建立关键词到文档的映射关系，根据这些映射关系对用户输入的查询进行处理，得到最匹配的搜索结果。此类搜索可以达到很高的准确率，但是往往需要人工维护索引，费时耗力，无法实时响应用户的查询请求。另外，这种搜索方式通常只能找到相似性较高的文档，不能完全匹配用户的需求。

### Semantic Understanding
语义理解（semantic understanding）主要指识别并理解文本中隐含的语义信息，从而能够提供给用户更多准确的搜索结果。例如，搜索引擎可以通过对用户的查询词和文档中存在的实体之间的语义关系进行分析，结合上下文信息提出更准确的搜索结果。此类搜索可以提升用户体验，实现信息检索的自然语言理解功能，提升搜索质量。

### 总结
关键字搜索与语义理解是实现智能搜索的一条主线。由于关键词搜索具有高度准确性，但是难以应付新的发展潮流；而语义理解属于人工智能的最新领域，目前还处于起步阶段。因此，如何结合两者的优点，更好地实现人机共同参与的智能搜索成为一个重要的研究课题。
