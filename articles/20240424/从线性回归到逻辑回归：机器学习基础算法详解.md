**From Linear Regression to Logistic Regression: A Detailed Explanation of Machine Learning Fundamentals**

**2. 核心概念与联系 Core Concepts and Connection**

Linear regression and logistic regression are both supervised learning algorithms, meaning they rely on labeled training data to learn patterns and relationships between input features and output targets. The primary difference between the two lies in their objectives:

* **Linear Regression**: Linear regression aims to predict continuous values (e.g., prices, temperatures) by modeling the relationship between input features and the target variable using a linear function.
* **Logistic Regression**: Logistic regression predicts binary outcomes (e.g., yes/no, 0/1) by modeling the probability of an event occurring given the input features using a logistic function.

Both algorithms can be viewed as special cases of generalized linear models, which extend traditional linear regression to handle non-normal distributions and non-linear relationships.

**3. 核心算法原理具体操作步骤 Core Algorithm Principles and Operation Steps**

### 3.1 Linear Regression

The goal of linear regression is to find the best-fitting linear equation that minimizes the sum of squared errors between predicted and actual values. The algorithm consists of the following steps:

1. Data preprocessing: Normalize and transform the data if necessary.
2. Model initialization: Initialize the model parameters (weights and bias).
3. Forward pass: Calculate the predicted values for each sample using the current model parameters.
4. Error calculation: Calculate the sum of squared errors between predicted and actual values.
5. Backward pass: Update the model parameters using gradient descent or another optimization algorithm.
6. Repeat steps 3-5 until convergence or a stopping criterion is reached.

### 3.2 Logistic Regression

The goal of logistic regression is to estimate the probability of an event occurring given the input features. The algorithm consists of the following steps:

1. Data preprocessing: Normalize and transform the data if necessary.
2. Model initialization: Initialize the model parameters (log odds ratio and intercept).
3. Forward pass: Calculate the log odds for each sample using the current model parameters.
4. Sigmoid transformation: Apply the sigmoid function to the log odds to obtain the probability estimates.
5. Loss calculation: Calculate the cross-entropy loss between predicted probabilities and actual labels.
6. Backward pass: Update the model parameters using gradient descent or another optimization algorithm.
7. Repeat steps 3-6 until convergence or a stopping criterion is reached.

**4. 数学模型和公式详细讲解举例说明 Mathematical Models and Formulas with Examples**

### 4.1 Linear Regression

The linear regression model can be represented as:

$$y = w^T x + b$$

where $y$ is the target variable, $w$ is the weight vector, $x$ is the feature vector, and $b$ is the bias term.

The cost function for linear regression is the mean squared error (MSE):

$$L(w, b) = \frac{1}{n} \sum_{i=1}^{n} (y_i - (w^T x_i + b))^2$$

### 4.2 Logistic Regression

The logistic regression model can be represented as:

$$p = sigmoid(z) = \frac{1}{1 + e^{-z}}$$

where $p$ is the probability of the event occurring, $z$ is the log odds, and $e$ is the base of the natural logarithm.

The cost function for logistic regression is the cross-entropy loss:

$$L(w, b) = -\frac{1}{n} \sum_{i=1}^{n} [y_i \ln(p_i) + (1-y_i) \ln(1-p_i)]$$

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# Generate some sample data
X = np.random.rand(100, 2)
y = 3 * X[:, 0] + 2 * X[:, 1] + np.random.randn(100)

# Create and train the model
model = LinearRegression()

# Make predictions
y_pred = model.predict(X)
```

```python
import numpy as np
from sklearn.linear_model import LogisticRegression

# Generate some sample data
X = np.random.rand(100, 2)
y = np.random.randint(0, 2, size=100)

# Create and train the model
model = LogisticRegression()
model.fit(X, y)

# Make predictions
y_pred = model.predict_proba(X)
```
