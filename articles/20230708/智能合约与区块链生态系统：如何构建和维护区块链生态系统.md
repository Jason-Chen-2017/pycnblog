# 2. 技术原理及概念

## 2.1. 基本概念解释

区块链是一个分布式数据库，通过加密和哈希算法实现不可篡改的数据存储。区块链上的每个区块都包含前一个区块的哈希值，通过连续计算，可以得到一个不可改变的哈希值。这种技术称为“区块链算法”。

智能合约是在区块链上运行的计算机程序，具有自主执行能力。它可以修改和执行合同条款，实现自动化的执行。智能合约的执行不需要人为干预，提高了工作效率。
