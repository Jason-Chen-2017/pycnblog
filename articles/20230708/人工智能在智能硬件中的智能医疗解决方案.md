4. 应用示例与代码实现讲解
----------------------------

4.1. 应用场景介绍

本文将介绍一种基于 CNN 的医疗图像识别系统，用于对脑部 CT 扫描图像进行分类和识别。该系统可以实现快速、准确地识别出是否存在异常，从而帮助医生实现对疾病的快速诊断。

4.2. 应用实例分析

假设某医院的一名医生需要对一批脑部 CT 扫描图像进行分类和识别，以确定是否存在异常。该医生可以在计算机上安装本文的智能医疗系统，并将一批脑部 CT 扫描图像输入到系统中。系统会将图像输入到 CNN 模块进行处理，然后输出分类结果。医生可以根据系统的输出结果，快速判断患者是否存在异常。

4.3. 核心代码实现

```python
import os
import numpy as np
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

# 加载数据集
mnist = keras.datasets.mnist.load_data()

# 数据预处理
(train_images, train_labels), (test_images, test_labels) = mnist.train_data, mnist.test_data
train_images = train_images.reshape((60000, 28, 28, 1))
test_images = test_images.reshape((10000, 28, 28, 1))

# 将图像数据输入到模型中
model = keras.models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

# 编译模型
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# 训练模型
model.fit(train_images, train_labels, epochs=5)

# 对测试集进行预测
test_loss, test_acc = model.evaluate(test_images)
print('Test accuracy:', test_acc)
```
