2.2.1 关键词抽取

关键词抽取是数据集标注中的第一步，它的目的是从原始文本中抽取出具有代表性的关键词或短语。常用的关键词抽取方法包括TF-IDF、TextRank和Word2Vec等。其中，TF-IDF是最常用的方法，它基于文档中所有关键词的平方和来计算关键词的重要性。代码实例如下：

```python
import numpy as np
import pandas as pd
from scipy.spatial import distance

def tfidf(text):
    import nltk
    from nltk.tokenize import word_tokenize
    from nltk.corpus import stopwords
    from scipy.spatial.distance import pdist

    # 去除停用词
    stop_words = set(stopwords.words('english'))
    words = [word for word in word_tokenize(text) if word not in stop_words]

    # 将词向量转换为稀疏矩阵
    vectors = [vector for word in words]
    matrix = pdist(vectors)

    # 计算平方和
    sum = 0
    for i in range(len(words)):
        for j in range(i+1, len(words)):
            sum += matrix[i][j]**2
    return sum


```
