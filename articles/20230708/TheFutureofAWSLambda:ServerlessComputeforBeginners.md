## 2.1. 基本概念解释

### 2.1.1. AWS Lambda 服务

AWS Lambda 是一项云上服务，提供给开发人员一个快速、简单、无责任的开发环境，让他们能够更加专注于业务逻辑的开发和部署。AWS Lambda 支持多种编程语言和运行时，能够帮助开发人员快速构建和部署代码，实现更加高效和灵活的云计算服务。

### 2.1.2. serverless 计算

serverless 是一种计算模式，让开发人员更加专注于业务逻辑的实现，而不必担心底层基础设施的运维。在 serverless 计算中，开发人员编写的代码会被打包成函数，运行在 AWS Lambda 服务上，实现更加高效和灵活的云计算服务。

### 2.1.3. API 网关

API 网关是一种中间件，用于管理和路由 API 请求，是 serverless 计算中非常重要的一部分。API 网关能够提供更加安全和高效的 API 访问方式，让开发人员更加专注于业务逻辑的实现。
