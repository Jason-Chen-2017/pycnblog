# 2. 技术原理及概念

2.1 基本概念解释

在机器翻译中,跨语言多语言交互是指在不同语言之间进行交互时,通过在不同语言之间建立交互,实现多语言之间的沟通。机器翻译中的跨语言多语言交互与多语言翻译是基于语言模型的,它将自然语言文本翻译成目标语言文本,并在目标语言文本中插入翻译结果。

实现跨语言多语言交互的关键是语言模型。语言模型是对自然语言文本进行建模,并可以预测目标语言文本的概
