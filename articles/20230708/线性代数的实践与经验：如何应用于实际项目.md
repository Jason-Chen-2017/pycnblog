2.2.1. 向量
------

在实际项目中，向量是一种常见的数据类型。向量可以用于表示用户信息、商品信息等。下面将介绍如何使用线性代数实现向量。

```python
import numpy as np

# 创建一个一维向量
v = np.array([1, 2, 3])

# 显示向量
print(v)
```

2.2.2. 矩阵
------

在实际项目中，矩阵是一种常见的数据结构。矩阵可以用于表示线性变换。下面将介绍如何使用线性代数实现矩阵。

```python
import numpy as np

# 创建一个二维矩阵
A = np.array([[1, 2], [3, 4]])

# 显示矩阵
print(A)
```

2.2.3. 线性变换
----------

在实际项目中，线性变换是一种常见的数学模型。线性变换可以将一个向量映射到另一个向量。下面将介绍如何使用线性代数实现线性变换。

```python
import numpy as np

# 创建一个二维矩阵
A = np.array([[1, 2], [3, 4]])
B = np.array([[1, 2], [3, 4]])

# 创建一个新的二维向量
C = A.dot(B)

# 显示结果
print(C)
```
