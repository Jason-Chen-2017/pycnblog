假设你是一位老年人，需要实时监测自己的健康状况，以便及时采取措施预防疾病。你可以穿戴一款可穿戴设备，该设备可以实时监测你的运动量、心率、体温等数据，并通过无线网络将数据传输至云端服务器进行处理。云端服务器在接收到数据后，可以形成对老年人健康状况的实时分析，为老年人提供个性化的健康服务。

### 核心代码实现

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

class Wearable设备:
    def __init__(self, sensor, server):
        self.sensor = sensor
        self.server = server

    def read_data(self):
        return self.sensor.read_data()

    def send_data(self, data):
        return self.server.send_data(data)

class Server:
    def __init__(self, data_存储):
        self.data_存储 = data_存储

    def receive_data(self, device):
        return device.send_data(device.read_data())

    def store_data(self, data):
        return self.data_存储.append(data)


# 运动员健康监测应用示例

sensor = Wearable设备('加速度计', Server('健康监测'))

data = sensor.read_data()
data = np.array(data).reshape(100)

server = Server('健康监测')

server.receive_data(sensor)
server.store_data(data)

# 绘制数据
plt.scatter(data)
plt.xlabel('时间')
plt.ylabel('数值')
plt.show()
```
