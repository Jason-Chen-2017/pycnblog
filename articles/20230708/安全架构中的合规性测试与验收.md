4.        应用示例与代码实现讲解

##4.1. 应用场景介绍

本实验针对常见的网络攻击场景：TOR（The Onion Router）攻击。TOR攻击是一种匿名网络通信，通过多径传播和时间回绕等方式绕过网络防火墙，实现通信。

##4.2. 应用实例分析

(1) 攻击场景：用户通过TOR客户端连接服务器，传输敏感信息。
(2) 数据包抓取：从TOR客户端抓取网络数据包。
(3) 统计分析：对抓取到的有效载荷数据进行统计分析，得到攻击成功率。

##4.3. 核心代码实现

```python
import random
import numpy as np
import matplotlib.pyplot as plt

def generate_packets(num_packets, num_有效载荷):
    packets = []
    for _ in range(num_packets):
        attack_packet = random.randint(0, 100)
        if attack_packet <= num_有效载荷:
            packets.append(attack_packet)
    return packets

def statistical_analysis(packets):
    num_attacks = 0
    for packet in packets:
        if packet > 0:
            num_attacks += 1
    attack_success_rate = num_attacks / len(packets)
    return attack_success_rate

def main():
    # 设定参数
    num_packets = 1000
    num_有效载荷 = 50

    # 数据包抓取
    packets = generate_packets(num_packets, num_有效载荷)

    # 统计分析
    attack_success_rate = statistical_analysis(packets)
    print(f"攻击成功率: {attack_success_rate}%")

if __name__ == "__main__":
    main()
```
