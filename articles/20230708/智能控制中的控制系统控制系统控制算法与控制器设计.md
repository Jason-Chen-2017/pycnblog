### 4.3. 核心代码实现

以 Python 为例，实现对智能音响、智能门锁和智能灯泡的控制，代码实现如下：

```python
import socket
import sys
import time

class SmartHomeController:
    def __init__(self,音响IP,门锁IP,灯泡IP):
        self.音响IP =音响IP
        self.门锁IP =门锁IP
        self.灯泡IP =灯泡IP

    def set音(self, value):
        print('set sound value to:', value)

    def set门锁(self, value):
        print('set door lock value to:', value)

    def set灯泡(self, value):
        print('set light bulb value to:', value)

    def play(self, value):
        print('play sound value to:', value)

    def turn_off(self, value):
        print('turn off all lights value to:', value)

    def get_sound_status(self):
        return 'is playing' if self.is_playing else 'is not playing'

    def get_do_status(self):
        return 'is locked' if self.is_locked else 'is not locked'

    def is_playing(self):
        return self.sound_status == 'is playing'

    def is_locked(self):
        return self.do_status == 'is locked'
```

以上代码中，SmartHomeController 类是控制系统的核心类，它包含了音响、门锁和灯泡的控制方法，以及获取控制系统状态的属性。set音、set门锁、set灯泡和 play 方法用于实现对音响、门锁和灯泡的控制，get_sound_status 和 get_do_status 方法用于获取控制系统状态，is_playing 和 is_locked 方法用于判断音响和门锁的状态。
