4.4. 代码讲解说明

在这里给出一个医学影像处理系统中的图像增强模块的代码实现，使用 OpenCV 库实现图像增强。

```python
import cv2
import numpy as np


def enhance_image(img):
    # 增强对比度
    img_enhanced = cv2.convertScaleAbs(img, alpha=1.5, beta=0)
    # 增强亮度
    img_enhanced = cv2.cvtColor(img_enhanced, cv2.COLOR_BGR2HSV)
    # 饱和化
    img_enhanced = cv2.cvtColor(img_enhanced, cv2.COLOR_HSV2BGR)
    # 归一化（0-1）
    img_enhanced = cv2.normalize(img_enhanced, None, 0.0, 1.0)
    return img_enhanced

```
