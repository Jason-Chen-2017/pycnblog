```python
import os
import sys
from nginx.ext import default_headers
from nginx.http import Request
from nginx.conf import global_conf
from nginx.events import events

class PyPyramid(object):
    def __init__(self):
        self.config = global_conf.load()
        self.certs = os.path.join(os.path.dirname(__file__),'ssl.crt')
        self.key = os.path.join(os.path.dirname(__file__),'ssl.key')
        self.py = sys.path.join(os.path.dirname(__file__), 'python')
        self.site_root = os.path.join(os.path.dirname(__file__),'site-root')
        self.cache_dir = os.path.join(self.site_root, 'cache')
        self.log_dir = os.path.join(self.site_root, 'log')

        if not os.path.exists(self.cache_dir):
            os.makedirs(self.cache_dir)

        if not os.path.exists(self.log_dir):
            os.makedirs(self.log_dir)

        nginx.conf.init(self.config)
        nginx.events.init(events)
        nginx.http.add_header(Request, default_headers)

    def run(self, handler):
        return nginx.http.add_header(handler, default_headers)
```

4.4. 代码讲解说明
首先，我们创建了一个名为`Config`的类，用于保存网站的配置信息。

* `CACHE_BACKEND`属性指定了使用的缓存后端，这里我们使用Django Redis作为缓存服务器。
* `CACHE_OPTIONS`属性用于设置缓存选项，包括缓存键的策略、缓存大小限制和自动清除策略等。
* `SITE_ROOT`属性指定了网站的根目录。
* `CACHE_DIR`属性指定了缓存目录，用于存放静态和静态资源文件。
* `STATIC_URL`和`STATIC_ROOT`属性指定了静态文件存储的目录。
* `LOG_DIR`属性指定了日志目录。

接着，我们创建了`MyConfig`类，用于读取和保存配置文件。

* `load_config`方法用于加载配置文件，并将配置信息保存到`self.config`变量中。
* `save_config`方法用于将配置文件保存到指定的文件路径。
