4.2. 应用实例分析

以下是一个 Python 代码示例，用于实现商品列表的功能：

```python
# -*- coding: utf-8 -*-

class Product:
    def __init__(self, name, price):
        self.name = name
        self.price = price

    def __str__(self):
        return f"{self.name} - {self.price}"

# 商品列表
products = [Product("iPhone 13", 10999), Product("iPhone 13 Pro", 11999), Product("iPhone 13 Ultra", 12999)]

# 输出商品列表
for product in products:
    print(product)
```

上述代码实现了商品类和商品列表类，其中商品列表类继承自商品类，重写了 __str__ 方法，用于打印商品列表对象。实现了商品列表功能后，可以通过调用 list() 方法获取商品列表，也可以通过 for 循环遍历商品列表。
