### 4. 应用示例与代码实现讲解

在这里，给出一个实际的文本摘要应用示例，以及实现该应用所需编写的代码。

应用示例：根据用户输入的关键词，提取摘要并输出。

```python
import spacy
from spacy.en_core_web_sm import ENT

nlp = spacy.load('en_core_web_sm')

def extract_summary(text):
    doc = nlp(text)
    summary = ''
    for para in doc.paragraphs:
        for sent in para.sentences:
            summary += sent.text.strip() +''
    return summary

user_input = input('请输入关键词：')
summary = extract_summary(user_input)
print('摘要：')
print(summary)
```

这段代码首先加载了 ENT 模型的自然语言处理核心为 Web Sm，然后使用 spacy 的 `nlp` 函数提取文本数据。接着，使用预处理函数对文本进行预处理，然后使用机器学习训练模型，最后对模型进行评估。最后，根据用户输入的关键词，提取文本的摘要并输出。
