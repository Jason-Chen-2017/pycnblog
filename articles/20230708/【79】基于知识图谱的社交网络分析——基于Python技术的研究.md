下面是一个基于知识图谱的社交网络分析的Python代码示例:

```python
import requests
from bs4 import BeautifulSoup
import json

# 请求原始数据
url = "https://example.com/data.json"
response = requests.get(url)
data = json.loads(response.content)

# 构建知识图谱
G = []
for user in data:
    for friend in user.get("friends", []):
        G.append({"node": user, "edge": friend})

# 社交网络分析
for user in G:
    print(f"User {user['node']}:")
    print("Friends:")
    for friend in user["edge"]:
        print(f"  {friend['node']}:")

# 结果可视化
import matplotlib.pyplot as plt

```
