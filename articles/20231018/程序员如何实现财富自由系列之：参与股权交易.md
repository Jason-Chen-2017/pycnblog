# 1.背景介绍


## 为什么要参与股权交易？
自从股市诞生以来，投资者人手聚集。越来越多的人开始把目光投向股票市场，开始了解股票背后的经济价值、公司运营模式、企业家精神等多种信息。那么为什么还会有如此多的人支持股票市场呢？这里有几个原因：
- 投资者的热情和专注力：股市是一个高度竞争性的市场，每天都有新的参与者涌现出来。在短短几年的时间里，这种热情是无法用语言形容的。一旦参与股票交易，就等于把自己的时间和资金投入到一种全新的市场中。不仅如此，很多股票投资者并没有买的起股票，也没有时间去研究市场机密。他们只是希望通过参与股票市场获得一些收益。
- 技术变革：随着互联网、移动支付等新兴技术的出现，人们对金融产品的认识发生了巨大的转变。越来越多的人开始关注这些科技工具的潜力，试图找到能够带来长期收益的投资项目。在这个过程中，股票市场似乎已经成为最好的选择。
- 汇率的影响：由于外汇储备量激增，导致国际收支更加紧张。为了保持经济运行正常，政府与央行开始实施更严格的货币政策。为抵御通胀、保持资本市场稳定，投资者需要了解市场的变化及相关法律法规。因此，多数股票投资者在熟悉股市规则、掌握技术分析方法后，会选择持有一些看好未来的股票。这样可以给自己一个很高的获利空间。
- 个人的需求：近几年来，股市发展迅速，成为年轻人的第一反应。人们对股市上涨、下跌的预期和担忧逐渐消失。股市的参与度越来越高，一方面是因为年龄的增长带来对生活方式的改变；另一方面是更多的人发现投资是件开心且有趣的事。由于大家对股市投资能力的要求越来越高，所以越来越多的聪明才智、聘请顶级经济学家、主导投资者组合，鼓励各类投资人参与其中。
## 什么是股权交易？
股权交易（Stock Option Trading），又称股票期权，是指利用证券市场提供的融资担保，根据所持有的证券（股票）及其配套债权，以期望收益最大化的方式进行的有偿投资行为。股权交易最早是由美国的证券商E*Trade公司于1979年提出，但实际应用却始于1993年。近几十年来，股权交易已成为一门独立的业务领域。
### 形式上：
股权交易一般分为期权和转托管两种形式。
- 期权：期权是一种具有时间限制和保证金率制度的放弃认购权的方式，其保证金率决定了交易双方交换的回报情况。股权交易主要采用的是看涨期权或看跌期权的交易方式，即根据所持有的股份数量所能取得的最大回报。
- 转托管：转托管是一种以债务担保的方式进行的有偿投资，主要包括有限责任公司或其他受托人将股份转让给受托人（交易商）进行交易，在委托人（投资人）死亡时无需承担任何风险。转托管交易方式通常由两方约定协议，设定认购条件、交割条件、转让价格、到期日等。
### 功能上：
股权交易的主要功能有三点：
- 投资收益的最大化：投资人通过购买或卖出证券、股份、债权而获得的收益是根据证券市场的变化及法律法规的规定确定的。股权交易往往比普通股票交易更容易实现收益最大化。
- 资金管理的便捷：股权交易可以在没有银行账户、信用卡或消费历史的情况下进行。只要有充足的现金，就可以进行各种交易。
- 个性化定制：股权交易具有高度个性化的特点，每个人根据对市场的理解和经验自行交易策略，能够给予投资者更高的回报。
