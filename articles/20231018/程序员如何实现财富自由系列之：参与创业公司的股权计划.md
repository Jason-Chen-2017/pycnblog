# 1.背景介绍


## 创业者的痛点
创业已经成为许多人梦想中的生活阶段。然而，面对种种艰难险阻、创业过程中诸如职场竞争、资金困难等各种问题，无论是创始人还是创业者都不得不面临更大的考验。
### 初创期的创业者
相较于经验丰富的技术人员或技术精英，初创时期创业者往往面临着诸如找不到合伙人的、急需资金的、没有人能够信任的情况。这些都是他们要克服的重重困难，但同时也是他们需要面对的一道风险。
创业初期，创业者可能会遇到一些棘手的问题，如将自己完全投入到一个陌生领域，如何运用互联网资源、团队管理能力、项目设计能力、营销推广能力等，让自己迅速的成长。
此外，由于创业初期还处在寻求创造性思维的阶段，所以也会经历早期的不确定性和矛盾冲突。为了解决这些问题，创业者们会依赖于网络，阅读各种相关书籍、学习新技能，提升自己的实力。
### 中长期的创业者
中长期的创业者一般是技术含量比较高的创业者。他们会有独特的见解和方法论，通过不断分析市场需求、制定产品路线、拓展市场占有率等，最终建立起独立且强大的企业。
作为技术人员，他们也将有机会结识并与其他技术人员进行交流和分享，进一步增强自身的技能。同时，他们还可以参与到各类商业活动中，比如举办一些本地或者国际的研讨会、促销自己的产品或者服务，从而为公司的长远发展提供助力。
### 灵活应变的创业者
最后，也是最重要的，灵活应变的创业者。他们能够根据市场的变化调整策略，利用互联网资源快速地找到新的市场，成功的转型。因此，他们的策略也可能发生变化，这也是创业者需要不断完善自我的方式。
## 创业者的目标
对于创业者来说，实现财富自由，最基本的要求就是拥有足够的收入。这是所有创业者的终极追求，也是他们可以达到的目的。
但是，追求收入本身并不是一件容易的事情，它需要时间、金钱、物质条件以及人脉资源的共同投入。这不仅需要付出巨额资金，而且还需要不断成长，不断的积累和总结经验。
因此，如何通过一份稳定的、可持续的股权激励计划帮助创业者实现财富自由是一个更加复杂的任务。
## 为什么选择股权激励计划？
股权激励计划是一个公益性组织，主要面向具有一定社会责任感的创业者，为创业者提供独特的经营建议、资源指导和业务培训，帮助创业者实现财富自由。
股权激励计划所做的事情，简单而言就是通过给予创业者股权，鼓励创业者创立商业模式、融资、扩张产品市场、扩展业务和开拓更大的市场。只要创业者愿意投资并且遵循规则，他就有可能获得一些回报。
## 如何决定创业者是否适宜参与股权激励计划？
首先，创业者应该对股权激励计划有很强烈的信心，因为它只提供些许建议，需要创业者自己去判断。其次，创业者需要证明自己的能力和能力范围，表现出的才华和综合素质。第三，创业者需要证明自己的项目能够带来商业价值。最后，创业者应当尽快认识到自己的状况，及时调整策略，确保每一笔投资都能产生实际的回报。
## 什么是股权激励计划？
股权激励计划是一个公益性组织，它由一群有经验的专业人士组成，负责为创业者提供独特的经营建议、资源指导和业务培训。
股权激励计划所做的事情，简单而言就是通过给予创业者股权，鼓励创业者创立商业模式、融资、扩张产品市场、扩展业务和开拓更大的市场。只要创业者愿意投资并且遵循规则，他就有可能获得一些回报。
股权激励计划的目的是帮助创业者实现财富自由，其具体形式分为如下四个方面：
### 1.股权激励方案
创业者购买股权激励计划资产包后，可以选择加入该计划的任何一种资产组合。这种资产组合是根据创业者的不同创业理念，进行选择的，如金融类资产组合、信息技术类资产组合、创业资本类资产组合等。
根据个人理解，一般来说，金融类资产组合包括公司债券、基金、贷款等；信息技术类资产组合包括基础设施设备、技术开发工具等；创业资本类资产组合包括科技创新资本、房地产租赁资本、新兴企业资本、旅游创业资本等。
其中，公司债券和基金类型资产组合，是最基础的一种资产组合，是创业者最容易接受的一种激励方式。不过，它们往往存在比较高的风险和不确定性，需要创业者充分考虑。
### 2.股权激励方案分配
创业者购买股权激励计划资产包后，可以选择享受股权激励计划资产包内的所有资产组合，也可以根据自己的创业理念，选择部分资产组合。购买不同的股权激励方案，可以覆盖不同的产品价值。例如，如果创业者属于金融类的资产组合，那么，他可以选择加入财务咨询类资产组合，获取有关金融、审计、财务、税务等方面的咨询服务。如果创业者属于创业资本类的资产组合，那么，他就可以选择加入房地产租赁资本类资产组合，获取基于共享经济模式的产品创新资本支持。
### 3.业绩奖励
股权激励计划将企业业绩视为决定股东激励的唯一标准，并设置业绩奖励制度。创业者可以不断提升产品或服务的质量和功能，并向股权激励计划提交经营业绩证明，该证明反映了创业者的创新能力、市场推广能力、合作意识和创业精神。业绩奖励将根据投资回报的大小来确定股权激励金额。
业绩奖励制度的另一好处是它使得创业者的业绩显著优于同行业的平均水平。这一机制鼓励创业者勇于尝试新事物，通过摒弃陈规、创新、品牌、理念，取得突破性的业绩成果。
### 4.终身年金
终身年金制度是另一种激励制度。它鼓励创业者长期坚持创业理念，培养个人的创业能力和创业精神，同时也为创业者创造一个长久的收入来源。终身年金制度中的年金年限一般为5至10年，每年都将按照预算百分比分红一次，直到创业者离世。
除了通过股权激励方案和终身年金制度，创业者还可以通过行业交流、活动支持、免费课程等方式提升自我能力和成长。
## 有哪些具体操作步骤呢？
下面，就来具体了解一下股权激励计划具体操作步骤。
### 第一步：注册并申请股权激励计划
1. 在线注册：访问股权激励计划官方网站（http://www.soe-inv.com/）创建账户，然后填写申请表格。申请表格包括身份信息、创业年限、期望投资额、财务状况、是否符合投资者准则等。
2. 面试：股权激励计划审核通过之后，会邀请您进行电话或视频面试。面试的内容通常包括以下几个方面：
   - 公司简介：通过介绍公司背景、发展历史、产品理念、核心价值观等内容，来对创业者的能力进行考察。
   - 投资分析：您会跟投资专业人士，一起探讨企业的战略定位、未来的发展方向、盈利模式、财务状况、有无可行的商业模式等，进一步考察创业者的财务知识和对企业的了解程度。
   - 概念验证：您将参与一项创业实习，具体项目将由创业者和您来定义，并向股权激励计划团队展示您的方案。
   - 股权激励方案：通过讨论，股权激励计划团队会给创业者提供一套股权激励方案，具体内容将根据创业者对企业的看法、资产配置等因素进行协商。
   - 获奖证书：创业者完成创业计划之后，可以向股权激励计划团队提出创业获奖奖项。
3. 资料审核：股权激励计划审核通过的资料将进入创业者档案库，资料包括个人信息、创业创造的证据、个人照片等。
### 第二步：参与创业计划
1. 创建公司网站：创业者创建网站是参与股权激励计划的必要前置条件。网站用于展示企业形象，并吸引投资人。
2. 设置门户支付宝账号：向股权激励计划申请并绑定银行卡，设置微信支付宝账号。
3. 招聘团队成员：为创业计划选聘员工，包括产品经理、项目经理、UI/UX设计师、前端工程师、后端工程师等。
4. 开拓市场：通过互联网、移动互联网、社交媒体等方式开拓产品的市场份额。
5. 提供顾问服务：向投资人提供专业的投资建议，并协助创业者开展业务。
### 第三步：进行商业计划
1. 确定商业模式：选择一个能够为公司带来商业价值的商业模式。
2. 获取资金：由创业者、股权激励计划和各大银行进行谈判，达成协议。资金由创业者支付部分现金和借款，部分由股权激励计划或银行进行投放。
3. 筹建团队：向投资人提供团队支持，筹措雇佣金。
4. 拓展业务：围绕产品或服务开展营销、广告、培训、咨询等方面的活动。
5. 完成项目：根据商业模式，创业者按计划进行上线运营，并提高营收。
### 第四步：终止投资
1. 完成商业计划：创业者完成商业计划后，股权激励计划将会向创业者提供退款。
2. 分配股权：股权激励计划将会给创业者分派股权激励资产，创业者根据投资结果选择转化为股票、债券、基金、贷款等不同类型的资产。
3. 缴纳年金：创业者退还股权激励资产，缴纳终身年金。
4. 交接股权：创业者签署协议交接股权，获得股权转让的权利。
5. 分红分成：股权激励计划将按照约定的方式分给创业者的股权分红。
