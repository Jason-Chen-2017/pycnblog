# 1.背景介绍


“参与互联网初创公司的股权计划”，通常简称“股权计划”。它是指创业者在具备一定规模、受到特定行业或领域的资源支持后，通过创造利润或赚取回报，筹集股份，从而得到持续投入或事业上升空间的资本运作模式。2007年，中国互联网初创公司（以下简称“互联网公司”）数量迅速激增，他们面临着很大的金融风险、成长瓶颈等问题。随着网络经济的火爆，越来越多的创业者选择互联网初创公司作为投资方向，这些公司往往具有较高的起步门槛、较强的自主性、较强的竞争力。但是，在过去的几年里，互联网公司面临着许多挑战，如破产风险、企业经营困难、税费负担过重、法律纠纷频发、投资者不景气等等。为了应对这一挑战，一些互联网初创公司也纷纷推出了“股权计划”，通过筹集股份的方式获得更多收益。

就像互联网公司一样，股权计划也是一种非同寻常的资本运作模式。一般来说，股权计划分为两种类型：
- 一类是与公司自身相关的股权计划——如通过参与分红、增资扩股等方式进行员工薪酬激励，或通过公开招股、定向增发或者转让股份的方式进行资金支持。
- 另一类是与个人相关的股权计划——如通过私募股权基金、债券投资计划、消费贷款股权投资等方式进行投资。

但无论是哪种类型的股权计划，它们的核心特征都是以创始人或管理层身份，用自己的金钱或时间，实实在在地为公司或个人提供资本运作空间。因此，股权计划对于创业者来说，是一个非常值得探索的领域。尤其是在创业初期，一项有效的股权计划能够极大地促进创业者的发展，并有助于降低创业者面临的风险和成本，帮助其进入良性循环。

本文将以参与互联网初创公司的股权计划为主题，讨论股权计划背后的概念和理念，并以案例具体阐述它的运作过程及其具体应用。
# 2.核心概念与联系
## 2.1 概念
### 股权计划
股权计划(equity plan)是一种有关创业者们共同分割公司股权，并且拥有自己事业上的事业优势并分享有限损失的实质化、规范化的财政制度。在股权计划实施过程中，各方面参与方都须遵守计划条款，否则则会面临严厉处罚。这既包括整个股东集团的股权协议条款，也包括公司董事会、监事会的决议。根据申请人在同等条件下获得的分红与股息，向股权持有人分配相应的权利，以激励股东投资和保障其权益的平衡。因此，股权计划又被称为“分红计划”、“奖金计划”、“盈利计划”或“惩罚计划”。

股权计划是在法律允许的范围内，由创业者与公司、家族、社区乃至教会等个人或组织共同组建的法律实体。创建股权计划，意味着创业者向第三方（或单位）申报参与其股权计划，承诺为其股权及劳动报酬分配额外收益；同时，第三方承诺按照创业者的指令进行投资活动，并建立计划净额的使用规则。参与股权计划的创业者必须遵守所设立的条款及条件，并接受税务和法律义务。除了分配可留存部分的现金收入，股权计划还可以将部分收益用作本金或创业资本进行再投资，并以此抵消或补偿创业者因创业失败而产生的损失。

### 股权结构
股权结构（equity structure）是指企业或机构拥有的直接股权及其他的所有者权益。股权结构包括企业股东对企业的持股比例、职能、控制权及企业发展目标的决定性作用。按照股权结构的不同，企业可以分为有限责任公司（ limited liability companies）、股份有限公司（limited partnerships）、有限合伙、私募股权公司、个人股权投资基金等多种形式。股权结构可以分为三大类:

第一类是有限责任公司（LLC）。这是股权结构中最基础的结构。LLC是指只有管理层及少量股东拥有股份，公司权力较小，企业运营活动较为简单。这种股权结构适用于任何规模的企业，尤其适用于创业型初创企业，但收益分配情况可能较为单一，无法满足创始人的需求。另外，LLC容易出现解体风险，因为管理层控制权过大，导致公司资产的掌控能力欠缺。

第二类是股份有限公司（LSP）。股份有限公司是指有限公司内股东与所有者间有两个以上持有股份，共同决策和运营公司的行为。所有者占股份的权力较小，不足以完全控制公司，但是可以通过公司章程限制其对某些行动的影响。股份有限公司适用于合伙企业、子公司、有限合伙或第三方管理下的公司。与LLC相比，股份有限公司没有管理层控制权，股东的投票权比LLC更大，而公司效率却要低于LLC。另外，股份有限公司也存在运营能力弱、风险较大的特点。

第三类是有限合伙（LP）。有限合伙是指由持股人约定的非营利组织，其成员人数一般少于50人。通常情况下，有限合伙由成员共同设立法人，拥有固定比例的股份。该比例通常为1/3～1/2，但也可能偏高或偏低。有限合伙的存在主要是为了协商管理上的便利，也方便成员之间进行资源共享。虽然成员可以根据自己的意愿退出股份合伙关系，但不影响总体股份的分配。

股权结构的不同决定了企业在运营中的不同特征。不同的股权结构对创业者的激励方式也有所差异。LLC可能更加注重个人积蓄，而股份有限公司则更注重团队合作，因为LLC只是公司的一种职能。除了向股东分配部分收益，股份有限公司还可以设立管理层薪酬福利基金，提高管理层的股东收益。有限合伙则更关注个人利益，对于所有者权益分配不太敏感。

## 2.2 基本原理
### 投资基金与分红派息
股权投资基金是指以基金的形式，由有经验且具有相关专业知识的投资顾问团队通过公开募集资金，吸引企业和个人的支持，通过对企业或个人股权的增持、减持和分红派息，实现企业或个人财富的增值。投资基金的运作需要制度上的规范，且收益较低，投资基金主承销商一般不会直接向投资者收取手续费或服务费，只能收取固定比例的费用，比如基金托管费等。然而由于基金发起人设置的策略并不能保证每次派息的一致性，因此基金的投资组合往往难以维持，投资者往往无法形成持久稳定的投资基金组合。

在中国，有的投资基金对股权投资基金给予过多关注，包括BATJ、ST，甚至出了一个国际版的杭州平安基金——平安证券通，其中透露有关基金运作的文章。但是目前国内的股权投资基金业界并未形成标准，没有统一的评级机制，市场空间正在逐渐被套牢。对于公众来说，了解一些国内的基金，尤其是股权投资基金，也是必要的。

#### 股权计划
股权计划是指通过基金或公司预先设立的计划，向私募基金或个人投资者分配股权，使投资者成为公司的股东或股东附属，从而获得股权增值的实证方式。股权计划由公司、个人或基金在设立时缔结书面协议，指定具体的分配办法，并委托相关的财务专业人员执行实际的分红派息。股权计划采用基金或公司与个人的集体认购的方法，由平台运作，平台通过发行股权证券或其他收益工具向投资者发放股权，或直接到股东所在地进行现场募集。个人的股权计划不需要取得法律授权。

股权计划一般分为以下四种类型：

第一类是基金投资计划。基金投资计划适用于个人的普通投资者，即其获得的投资收益为基金利润的4%-5%。它由基金管理人与私募投资者或个人签署协议，基金管理人收取管理费、托管费、分红费等费用，私募投资者为基金投资者发行股权证券或其他形式的权益货币。投资者可在证券交易所或交易所交易账户进行投资。

第二类是私募投资计划。私募投资计划是指由个人或机构（非基金管理人）设立，主张以其自身的资金或经营能力为基础，以小额、次级资产配置的方式向社会公众筹措资金，以满足其个人或机构对某种产品或服务的需求，或向特定对象募集资金。私募投资计划以个人或团体向发起者公布的条件下募集资金，并发布募集目标和收益计划，其资金获得归属权，但不享受公司所有者的股权激励。

第三类是创业公司自筹资金计划。创业公司自筹资金计划是指创业公司向投资人募集资金，然后按计划发放股权，创始人及其家庭也可获得一定比例的股权。创业公司自筹资金计划的收益取决于募集到的资金，并与创始人及其家庭分担相应的风险。

第四类是第三方个人股权投资计划。第三方个人股权投资计划是指以个人名义向管理人或雇主筹集资金，个人或机构将其拥有的股权认购给自己持有的基金或股份公司。该计划仅收取佣金或托管费。投资者不直接获取股权，需要跟踪基金或股份公司的动态才能知道自己是否得到投资的收益。

股权计划的优点是增强了创始人或企业的影响力和话语权，可以带动整个行业的发展。股权计划可以有效地解除个人或企业的雇佣关系，有效地实现投资者的自我激励。但是股权计划存在以下问题：

1. 由于基金的投资组合难以维持，投资者往往难以形成持久稳定的投资基金组合，并且基金的发起人往往设置不当的交易策略，使投资者远离投资。

2. 在发展初期，股权计划的运作方式仍需完善，特别是在考虑向市场上接纳新生的创业公司。

3. 股权计划还存在很多不确定性因素，如运作效率不高，资金来源不透明，风险控制能力弱，法律责任等，不利于创业者发展和走向成功。

### 分红派息制度
分红派息制度（dividend system）是指公司按规定的计划每年分派股权变动，并分派相应的现金或股份，这种制度实质上是对个人投资者给予信任、鼓励，以及鼓励企业扩大生产的一种制度安排。分红派息制度的目的是弥补股权发行的初始不足，通过对每年派出的股权进行价值评估，实现企业的持续增值，从而提高企业的利润。目前，分红派息制度主要应用于具有经营规模和特殊的创业目的的企业，如软件、房地产、游戏、农林牧渔等领域，均有分红派息制度。

分红派息制度的运作机制主要有四个方面：

1. 发行股权：公司通过筹资筹集资金，通过发行股权或其他权益货币的方式给予个人投资者作为股东的权利。

2. 拨付股利：公司定期将已发行的股权派给经资本或其他权益人，并在每年派息。股利的具体数额是根据公司存续期限和盈利能力确定的。

3. 质押贷款：公司通过质押的方式接受个人投资者的资金，并进行长期投资，为公司创造可持续的经济价值。质押贷款一般由公司内部的债务人发行，与股权结算方式类似，收取一定比例的息税。

4. 年终分红：公司将派出的股权按年度分派，根据所分配的股份大小给予分红。分红派息制度是鼓励企业发展和扩大生产的重要方式，是发达国家普遍实践的领域。

分红派息制度的优点是激励员工创造价值，长期坚持分红派息制度可以帮助企业实现持续盈利，创造更好的商业环境。但是分红派息制度也存在以下缺陷：

1. 分红派息制度依赖于股权的稀释和发放，这就使得公司获得的股权权益较少，企业的价值也不能真正体现出来。

2. 股权分散化：公司发行的股权一般较多，人才分布也不均匀，无法有效激励员工创造价值。

3. 无直接激励机制：公司不能将员工的实际努力与股权数量相匹配，进一步削弱了员工的动力。

4. 利润分配无法合理分配：目前股权分红制度下的利润分配仍然存在不足，有些项目存在加速晋升、疲劳裁员等现象。
