# 1.背景介绍


什么是面试心理调适？面试心理调适主要指的是面试前后，对候选人、面试官和面试场地三个角色的沟通、相互协作和心态平衡的过程。在这一过程中，面试官不仅要做到积极主动，更重要的是要通过自身能力及观察，能够及时发现问题和矛盾并及时解决，引导面试参与者走出思维惯性陷阱，改善自己的沟通技巧，充分发挥自己的优势。以下将从以下几个方面介绍面试心理调适的概念、机制、方法和技巧。

1.概念定义——面试心理调适
面试心理调适是指面试前后的相互沟通、协作、心态平衡过程。它是一个持续性的过程，整个过程应充满信任、包容、尊重和开放的气氛，面试官应当以诚恳的方式处理面试中出现的问题，同时注意以理服人的精神，让面试者自己反省、调整、提升。面试心理调适是指面试前后，候选人、面试官和面试场地之间的自然而然的互动，可以称之为“心灵上面的互动”，具有极强的社会意义，需要注重领导、同事间的了解、配合、沟通等各个环节的协调性与准确性。

2.机制设计——目标导向、倾听力、表达技巧、反馈回路
面试心理调适的机制包括目标导向、倾听力、表达技巧、反馈回路。
目标导向：面试心理调适的第一步便是明确目标，面试官需要着重于候选人在某项评判标准上的表现，但又不要偏离其真正的需求，目光放在长远考虑上。同时，面试者要清晰地知道自己应该达到的目标，同时面试者也需承认自己的不足。如：面试官问：“你希望你的职业生涯能走向何处？”候选人说：“我希望我的事业能给家庭带来好多收入。”
倾听力：面试过程中，双方都需要有耐心倾听对方的意见，这样才能建立起共鸣。在面试过程中，如果候选人表现出“不听话”的行为，则需要站出来控诉其不够负责任，否则会引起矛盾。因此，面试官应当在开始考核面试者之前给予候选人适当的倾听时间，同时还应告知对方如果选择不接受，可以适当推迟面试日期或重新安排面试。在这个过程中，候选人需要注意掌握自己的表情，尤其是注意反映出来的肢体语言，以此来判断是否是真诚的要求。
表达技巧：面试官在面试过程中，需要用简练、轻松的口头方式来回答候选人提出的各种问题。一般来说，应当先介绍自己的工作经验、学习经历以及个人特点，随后再回答具体的问题。对于一些比较复杂的问题，面试官可根据自己的擅长领域或技能，提供一些较为详实的答案或解决方案。通过声音的平静、顺畅、自然、深刻，及时传达候选人的需求和信息，有效地促进候选人和面试官的交流，提高他们之间的沟通技巧。
反馈回路：面试结束后，面试官会根据候选人反馈的情况，分析是否通过了面试。面试结束时，如果对方表示接受或比较满意，那么面试官就可以高兴地宣布面试结束；如果对方表示准备再次面试，那么就需要继续加强面试流程，比如收集更多信息、找寻面试机会，或组织一次新一轮的面试。反馈回路是指候选人给面试官的反馈、评价、建议都将得到及时的回应，并且以他们的理解准确反映出面试结果。因此，面试过程中，候选人需要积极参与，善于表达自己的想法和意见。
3.方法论——技术视角、实例研究、场景应用、模拟面试等
面试心理调适的方法论包括技术视角、实例研究、场景应用、模拟面试等。
技术视角：面试心理调适从技术的层面入手，围绕人类如何有效地进行沟通、管理、协调等不同领域的知识和技能，探讨面试中的心理问题、资源分配、沟通模式、工作压力、职业发展规划等问题。
实例研究：面试心理调适的方法是通过观察和分析面试中的实际例子，分析面试过程中面试官、面试者和被面试者之间的关系、相互之间所产生的影响和沟通效果，以此来帮助面试者构建自己的心理模型和优化技巧，从而达到心理层面的问题解决。
场景应用：面试心理调适从面试官角度出发，面试时的环境和人员条件对候选人的态度、能力、表现和成绩都有很大的影响。面试官的喜怒哀乐、厌恶嫉妒、自尊心、敏感度、安全感以及职业规划等都能影响候选人的面试表现。因此，面试者应该根据自己的实际状况，制定合适的调适策略，引导面试官创造性地处理面试问题。
模拟面试：面试心理调适也具有一种模拟性质，即可以采用“好运总会临"的说法，即使面试官和面试者都很失败，也不会因为这一点而让彼此完全崩溃。因此，面试者可以根据自己最初的面试计划进行模拟面试，测试自己的判断、沟通、协调、反馈等各方面的能力。
4.技巧总结——面试技巧、面试心理辅助工具、面试脚本、面试时间设置等
面试心理调适的技巧总结包括面试技巧、面试心理辅助工具、面试脚本、面试时间设置等。
面试技巧：面试心理调适是一门技艺，需要刻苦钻研才能达到不败之目的。面试技巧包括语气与表情、反应、表达、结构化问题、透析型问题等。良好的表达技巧、引导候选人正确回答、认真倾听面试官的问题、维护同僚关系、缓冲面试官的时间、引导面试官提出宝贵建议等都是应试技巧。
面试心理辅助工具：面试心理调适还可以使用一些面试心理辅助工具，如：观看电影、看书、虚拟沙龙、个人咨询等，通过引入具有营养的活动来激活候选人的思维。通过这些工具，面试官和面试者可以更好地了解面试者的行为模式和问题背景，从而更好地判断候选人是否真的具备相关能力和条件，以及如何帮助候选人突破瓶颈。
面试脚本：面试心理调适的最佳实践是制作面试脚本，它可以将面试官、候选人、面试流程、面试主题、时间安排等内容统一起来，方便面试官和面试者的配合。在制作面试脚本时，面试官需要确定好要考察的能力范围和能力考核标准，面试对象应该具备的品质和软技能，以及面试的环境和规则。在面试过程中，候选人应该时刻关注面试官的反馈，随时按面试官的要求进行回答，保持冷静、克制、节奏感和全神贯注，提升面试效率和成功率。
面试时间设置：面试心理调适的一个关键就是要尽量提早、准时进行面试，这对于候选人、面试官和面试场地的三方都非常重要。面试时间的设定，既要考虑面试官的工作量，也要考虑候选人的时间预算。充分利用定时机制和经历风险，逐步缩小面试的时间差距，将面试官的注意力集中在提升候选人能力和提供专业服务上。

未来发展——面试心理调适的未来发展方向
面试心理调适的未来发展方向包括：面试工具的升级、面试评估的改进、案例研究、学生学习研究、调剂式训练、企业培训、求职网站的改造等。

1.面试工具的升级——面试中使用的工具越来越智能化，面试时自动生成报告或问卷，更容易带来误解，因此需要引导面试官主动承担信息传递的责任。
2.面试评估的改进——由于当前的招聘市场竞争激烈，面试的有效性也日渐成为竞争热点，如何对面试的质量进行评估及其有效性也成为一个难题。在招聘中，企业需要考虑面试的效果、面试官的服务质量以及候选人的技能水平。
3.案例研究——面试心理调适作为一门技艺，面对复杂的案例，面试者需要了解该案例背后的商业逻辑、公司现状、政策法规和人文因素，并通过自己的研究提炼出问题的关键，从而有效解决面试中的问题。
4.学生学习研究——面试心理调适的理论基础是研究候选人在面试中遇到的种种问题，如何将面试技巧应用于日常生活中，将有助于学生学得扎实、进步。
5.调剂式训练——除了课堂上的教授、模拟面试外，面试心理调适也可以培养候选人在真实的面试环境中锻炼其调适能力。通过积累经验，面试者会有能力识别并解决面试过程中存在的困境，使其拥有更好的表现。
6.企业培训——面试心理调适的发展正在吸引越来越多的企业和学校的重视，如何将面试心理调适知识培养到企业内部的岗位管理者、招聘人员、HR人员、员工等岗位，也是面试心理调适的研究热点。
