# 1.背景介绍


什么是字典？字典是一种存储数据的方式，键值对（key-value pair）的数据结构。字典里的每个键都是唯一的，而且可以取任何值，因此，它类似于哈希表或者关联数组。

什么是集合？集合是一个无序不重复元素的集，可以使用大括号{}或set()函数创建。和列表一样，集合也支持索引、切片等操作。

Python中的字典与集合的用法相对来说比较简单，尤其是在进行数据处理方面，比起其他编程语言，它的灵活性和便利性更值得信赖。本教程旨在帮助初级开发人员快速理解并使用Python中字典和集合的基本功能，掌握字典和集合提供的各种方法，并且能够充分利用它们解决实际问题。

# 2.核心概念与联系
## 2.1 字典
### 2.1.1 定义
字典是一种存储数据的形式。字典由键-值对组成，每个键对应一个值，键是唯一的，不能重复。字典的格式如下所示：

```python
{ key1: value1, key2: value2,... }
```

其中，每一对键值对之间用冒号:隔开，每对键值对之间用逗号,隔开。当要访问某个键对应的的值时，只需要用对应的键就可以了。

字典是可变对象，所以可以在运行期间修改字典的内容。

### 2.1.2 操作方法
字典支持以下几种基本操作：

1. 插入（insert）：添加新的键值对到字典中；
2. 删除（delete）：从字典中删除指定的键值对；
3. 查找（lookup）：根据键查询相应的值；
4. 更新（update）：更新字典中指定键的值；
5. 遍历（traversal）：按顺序遍历整个字典的所有键值对。

所有的这些操作都可以使用字典的方法完成。

### 2.1.3 示例

创建一个字典

```python
person = {'name': 'John', 'age': 27, 'city': 'New York'}
```

向字典中插入新项

```python
person['phone'] = '123-456-7890'
```

获取字典中指定键对应的值

```python
print(person['name']) # output: John
```

删除字典中的某一项

```python
del person['city']
```

更新字典中指定键的值

```python
person['age'] = 28
```

遍历字典中的所有项

```python
for key in person:
    print('{} : {}'.format(key, person[key]))
```

输出结果为：

```
name : John
age : 28
phone : 123-456-7890
```

## 2.2 集合
### 2.2.1 定义
集合是一个无序不重复元素的集。集合使用大括号 {} 创建，并用逗号, 分割不同元素。集合中的元素不可改变，只能添加和删除。

### 2.2.2 操作方法
集合支持以下几种基本操作：

1. 添加（add）：将元素添加到集合中；
2. 删除（remove）：移除集合中的元素；
3. 查询（membership testing）：检查某个元素是否存在于集合中；
4. 大小（len）：返回集合的大小；
5. 清空（clear）：清空集合中的元素。

所有的这些操作都可以使用集合的方法完成。

### 2.2.3 示例

创建一个集合

```python
fruits = set(['apple', 'banana', 'cherry'])
```

向集合中添加元素

```python
fruits.add('orange')
```

从集合中删除元素

```python
fruits.remove('banana')
```

判断元素是否属于集合

```python
if 'apple' in fruits:
    print("Yes")
else:
    print("No")
```

输出结果为：

```
Yes
```

获取集合的长度

```python
length = len(fruits)
print(length) # Output: 3
```
