# 1.背景介绍


“Python 人工智能实战”系列将带领大家走进人工智能的大门，从基础到高级，从零入门到精通应用。本书采用逐章的方式进行编写，每章以一个完整案例或项目为主线，引导读者一步步实现自然语言处理、图像识别、语音合成等目标任务，同时也会讲述一些机器学习的基础知识、统计模型的建立及应用场景，帮助读者深刻理解人工智能背后的智慧。本书适用于对人工智能感兴趣、希望有所了解的人群，包括数据分析工程师、机器学习工程师、算法研究员、AI产品经理等。
