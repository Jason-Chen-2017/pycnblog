# 1.背景介绍


## 1.1 为什么要成为自由职业者
为了实现个人价值最大化、全人类福祉最大化，现代社会需要大量的人才积累，并通过激励机制赋予他们主动选择自己的工作领域、志向和职业道路。同时，一些人为了争取在所谓的“低技能时间”获得更多收入或财富，也会创办各种“兼职”，以获取社会声望和经济利益。而这一切都离不开自由职业者的到来。自由职业者即可以自己从事创新型工作，也可以按需外包，或者找别人订制职业计划。自由职业者可以参加社团活动、举行讲座、组织运动、协助企业创建商业模式等。自由职业者通过参与社会服务项目、社区管理、联合创作等方式，可以提升个人品质、培养社交能力、促进健康生活、增强自我信心、扩大影响力。这些都是个人奋斗的结果。因此，无论是大学毕业、年轻人进入职场、打算创业，还是老牌企业员工想走出舒适圈，都应该认真考虑加入一个自由职业者平台。
## 1.2 自由职业者的优势
### 1.2.1 个人价值最大化
自由职业者将个人价值最大化作为其工作目标，他们更注重的是自身对社会的贡献。在进行个人职业规划时，他们首先会思考自己的兴趣爱好、兴趣所在领域、兴趣所在子领域，然后选择自己热爱且感兴趣的工作。这样做的原因是：只有真正了解自己、真诚付出、乐于分享自己的兴趣才能让自己得到足够满足。
### 1.2.2 技术专长与成长空间
自由职业者往往具有高度的技术专长，可以掌握现代最先进的信息技术和管理技术，因此其获得的成功机率较高。自由职业者的成长空间更大，其技术能力、知识结构都可以不断升级和提升。例如，一位自由职业者学习编程语言Java已经超过十年，正在探索使用Python进行开发；另一位自由职业者已经成为一名拥有丰富经验的大数据分析师，还在学习Spark等新型计算框架。
### 1.2.3 全人类福祉最大化
因为自由职业者本身就享有更广泛的社会资源，所以其所得到的福利也是可观的。例如，自由职业者可以在户口、医疗保险、养老金等方面获得额外收入，还能享受免费体育锻炼和享受基本的医疗保健。在这样的福利背后，也存在着相应的隐形风险。但这种风险与一般聘用雇员所承担的风险不同。
### 1.2.4 激励机制与责任心
由于自由职业者的生活环境比较自由，往往没有固定期限和薪水，因此他们往往容易产生激励机制。例如，自由职业者可以给自己设定目标，比如每月完成多少任务或赚多少钱，并以此为导向选择自己的职业方向、技术专长、工作方法等。自由职业者也很重视自己的责任心，他们有责任为他人提供必要的生活保障和帮助，不会轻易出卖自己的才能。
### 1.2.5 财富自由
虽然自由职业者在大多数情况下只能获得一定的收入，但在某些情况下，他们甚至可以获得超出常人的财富。例如，一些企业为了获得市场份额，可能会鼓励或免费招聘自由职业者。另外，像马云这样的知名科学家也被称为“自由人”，在某些方面也属于享受特权。
## 1.3 如何成为自由职业者
对于刚刚踏入职场的大学生或刚毕业的青年人来说，成为自由职业者是一个比较难得的梦想。与很多初出茅庐的年轻人一样，他们可能不知道自己究竟想要什么，只是盲目地追求一切新鲜感，并且无法准确预测自己的职业生涯。所以，当碰到这种情况时，作为职场人的你一定要想一想：
- 第一步：学会识别自己。如果你只对某个领域感兴趣，那就不要着急加入太多的社群。如果你有兴趣并且已经具备了一定的工作能力，那么就可以先试着在该领域找到一个工作机会。如果还不能找到工作，你可以尝试自己创造或者在平台上寻找新的志同道合者。
- 第二步：做好准备。在决定加入某个社群之前，你需要做好充分的准备。首先，要清楚自己在哪个行业、领域、岗位上能达到目标，然后要熟悉自己想要从事的工作业务和流程。另外，要掌握网络和移动互联网的技巧，保持实时的沟通能力和应变能力，这样才能在工作中取得有效果。
- 第三步：了解规则。在寻找职业信息方面，你需要记住，有的职业信息网站需要缴纳费用，有的则不需要。根据自己的兴趣、能力、状况等因素选择合适的平台，以及合适的招募方式。这里推荐几个比较好的平台：
    - Glassdoor: 提供了许多职业信息，包括公司名称、职位描述、待遇福利、评级、评估等，以及针对不同的行业、位置和性别推出的职业调查问卷。可以帮助你选择最适合你的工作。
    - Indeed: 是美国最著名的工作搜索引擎，主要用于海外工作。你可以在上面搜索各个公司的职位信息，包括公司名称、职位描述、待遇福利、工作条件、岗位要求等。
    - Seek: 是一个由国际著名人士发起的职业数据库，你可以搜索到世界各地的职位信息，包括公司名称、职位描述、待遇福利、工作条件等。
    - Linkedin: 是一个海外人才交流社交平台，它连接了海外及国内的职业人才，提供各个领域的职业信息和招聘求职渠道。
- 第四步：调整心态。自由职业者是一种非常独立、自律的职业，因此你需要学会适应这种新的工作状态，调整心态，重新定义自己。如果你有责任心，不要以自我封闭的方式抵御压力，而是积极倾听别人的意见，相信自己能够成为更好的自己。同时，还要接受教育，了解社会、经济和法律方面的真相，这样才能更好的在职场中工作。
总的来说，成为一个自由职业者不是一件轻松的事情，它需要不断学习、研究、尝试、沉淀，还要不断成长。只要你足够努力，最终都会有突破性的进步。
