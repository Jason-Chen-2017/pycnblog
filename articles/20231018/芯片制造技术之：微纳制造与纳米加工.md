# 1.背景介绍


电子设计自动化(EDA)是一种新兴的、蓬勃发展的产业领域，它将计算机、材料科学、仪器工程与数学建模等多种学科的研究成果相结合，开发出了一整套的集设计、编程、测试于一体的工具链。随着近年来的高频集成电路（HIC）、低功耗计算设备（LED）等应用场景的不断增加，智能芯片制造技术也越来越重要。在这个行业中，我们通常把微型计算机（Microcomputer）、微控制器（Microcontroller）、微处理器（Microprocessor）等称为“微芯片”，它们的主要特点是规格小巧、封装紧密、工作电压较低，可以降低成本并满足对性能要求的高速响应时间。因此，它们被广泛应用在移动终端、嵌入式设备、汽车电子系统、航空航天等领域。目前，国内外有许多知名企业已经推出了基于微芯片的智能系统，如微软、苹果、谷歌等。
传统的制造方法依赖机械加工手段进行，而微型计算机则通过简单的编程指令控制生产线上的每个零件，可以极大地提升生产效率，而且由于不受外界因素的影响，不会出现烫伤、刮擦等危险性。但是，这种简单粗暴的编程方式存在着严重的缺陷，使得制造出来的芯片容易受到各种各样的问题，比如短路、漏电、损坏、超频等。为了解决这些问题，我们需要更进一步的发展方向——微纳制造与纳米加工。
微纳制造（Nanomanufacturing）是指利用低温微生物材料制造高分子芯片的新型工艺技术。在这一技术下，芯片的结构由纳米颗粒组成，其尺寸可以缩小到千厘米甚至微米。通过微纳技术，我们可以把传统的用铁、钢、铝或硅做的芯片尺寸缩小到分辨率可达万分之一的级别，从而实现高性能、高功率的功能。
纳米级制造（Nano Manufacture）具有以下几个显著特征：

1. 高度精确：纳米级制造实际上是在微观层面上对制造过程进行控制。微观层次上的细致控制能够在芯片的微观结构与功能之间建立起精确的匹配关系，并且在制备高性能、高功率芯片时提供高精度的结果。

2. 功能专用化：纳米级制造能够制备具有不同功能特性的芯片，包括但不限于显示屏、内存条、声音处理单元等。

3. 技术创新：纳米级制造还处于蓬勃发展阶段，其中一些先进制造技术已经突破了先前的瓶颈，如高可靠性纳米纤维聚氨酯组装、多环纳米复合材料等，还有未来可能带来全新技术发展的大大小小创新项目。

4. 经济价值：纳米级制造技术的制备成本要比传统工艺贵很多，但如果能够创造出优质且性能可控的芯片，其收益也会大幅度提升。

微纳制造与纳米加工也是一种创新的工艺技术，通过对微生物、物理、化学等方面的最新研究及技术革新，人们尝试制造出具有纳米级制造能力的、功能尺寸可压缩到近乎无形的芯片。在国际上，微芯片制造领域的技术水平处于世界领先水平，主要国家的财政支出也占据着巨大份额。但是，在国内，微芯片制造领域还处于起步阶段，相关教育培训和产业布局仍存在很大的优化空间。
微纳制造与纳米加工的基本原理和技术实现有很多，这里只做简单的介绍。
