# 1.背景介绍


光罩是电子元件的重要组成部分。由于光的反射特性，光栅结构中的光栅元件通常面临着将偏转角度或者其它量纲带来的折射效应影响到输出的光的频谱响应的困难。而光罩可以有效地解决这一问题，它在两极化导体的外层提供一个保护层，使得光栅元件的折射率降低，从而提高元件的精度和响应性能。
光罩的设计对电路系统的稳定性、灵敏度、速度、能耗以及容错性等方面的作用都十分关键。光罩能够减少电子束流失效的发生，保障高速集成电路的输出功率不受影响；同时，光罩也会在电路的各个节点进行隔离，使电流不能回流或随着时间改变方向，进一步增强了电路的安全性。光罩还能通过选择合适的阻抗匹配材料、选择更好的低温熔点等方式，对电路的输出表现产生积极的影响。因此，在实际的生产中，光罩的设计就显得尤为重要。
目前主流的光罩主要分为两个方向：圆形与方形。前者往往应用于晶体管、电池等应用，具有较大的面积开口，但是受到了金属膜的阻碍；后者则在IC、微波炉等多种设备中广泛应用，其小型的尺寸和平坦的结构，有利于结构、环境及技术要求的精确匹配。由于光栅结构中对偏转角度的影响，一般情况下，为了达到最佳的精度和准确性，都会采用环形、圆环型、具有斜向段的光罩结构。
# 2.核心概念与联系
## 2.1 光罩核心参数
光罩的主要参数有：
-   孔径（D）：表示光罩的孔径大小，单位为μm。孔径越大，所遮蔽的面积越大，传输的能量越高。常用取值范围为：10～200μm。
-   透过率（K）：表示光栅元件进入到光罩后的折射率。即光照射入光罩之后，再被反射光吸收，并转换为能量传递到下一级的光栅元件。它是一个介于0和1之间的实数，取值越接近1，则该元件对光线的影响就越弱。常用取值范围为：0.95～0.99。
-   轴距（S）：表示光栅元件中心距光罩轴线距离。单位为μm。轴距越大，光栅元件的折射率就会越低，即它的透过率将会降低。常用取值范围为：5～50μm。
-   阻抗（Y）：表示光栅元件与光罩之间的阻抗。阻抗越大，则光栅元件对光线的吸收能力越强，折射率越高。常用取值范围为：10MΩ～100MΩ。
## 2.2 光罩构成
光罩一般由以下三个主要部分组成：
-   一层成品陶瓷材料：即光罩的贴面部分。常用的成品陶瓷材料有石英、玻璃、铝箔等，有光滑、有光泽的材料都可以，但要有一定厚度。
-   二层材料：一般是玻璃、塑料、ABS、涂层玻璃层、LED、玻璃保护层等，这些材料可以是单层的也可以是多层的，有些光罩还包括塑料、ABS、膜板、绝缘隔离等多种材料，增加了额外的保护性能。
-   胶膜：光罩的外皮材料，是光罩的一层材料，也是光栅元件的外界，作为保护元件存在，防止光栅元件对电流的干扰和损害。一般来说，对于偏向晶体管或者电池等电子器件来说，胶膜就非常重要。
## 2.3 光罩性能指标
光罩的性能指标主要有：
-   孔径：即光罩的直径或半径，表示光栅元件与光罩的最短距离。其单位为毫米mm。通常认为，直径小于30mm的光罩性能较差，大于30mm~70mm的光罩性能逐渐提升，超过70mm的光罩一般认为不宜使用。
-   轴距：表示光栅元件中心距光罩轴线距离。单位为毫米mm。较小的轴距可以获得较高的透过率，因此可在成本、性能、空间占用等方面做出更好的权衡。
-   摩擦系数（friction cohesion）：表示光栅元件的摩擦系数，即通过光罩的间距能获得的抗力。摩擦系数越大，则光栅元件的折射率越低，折射损失的效应就越大。
-   折射率：表示光栅元件的透过率，就是经过光罩的元件在其线路上折射到其他媒质时，其向不同波长方向传播信号的能力。它是一个介于0和1之间的实数，取值越接近1，则该元件对光线的影响就越弱。通常认为，孔径小于70mm的光罩最好，以此类推，反之亦然。
-   电流密度：表示光罩中电流的分布密度。根据表面光通量、元件长度、光敏度的关系，可计算出。单位为安培/平方毫米。
-   阻抗匹配：表示光栅元件与光罩之间的阻抗匹配情况。若光栅元件的阻抗低于光罩的阻抗，则可通过调整光罩的轴距得到相应的改善。
## 2.4 光罩结构类型
光罩结构类型主要有：
-   环形结构：这是一种比较简单的结构，主要用于金属工艺中，其结构特点是金属盘在环形中，常见的实现方法为：基体用铜、铁钢、铝箔等材料包裹住，银轴线加工成圆形，然后用镀锌、锡、纸张等进行填充，最后与电子材料嵌入环形。这种结构最大的优点是，方便安装，价格便宜。缺点是，较为粗糙，容易受到损伤。
-   圆环型结构：这种结构在工艺上常见，例如圆筒状光栅的制作，而且容易在中心位置进行切割。其特点是光栅体由环状排列，周围有薄膜，很好的隔离了光源。这种结构一般比环形结构用得少一些，因为光栅在环形中只能看到一个面。缺点是在市场上成本较高。
-   拼接结构：拼接结构是一种复杂的结构，其特点是多个环形结构在一起，分别装配，形成一个整体。这种结构在各个环形之间建立了空间隔离，可以有效防止不同环形之间的相互干扰，并且可以很好的防止电流回流。缺点是结构复杂、结构密度高，故空间占用大。
## 2.5 光罩优化策略
光罩的优化策略主要有：
-   施工前检查：光罩的质量、机械性能、性能评测，确认光罩的性能指标达标，并做好相关准备工作。
-   确定光栅类型：光栅的选型应该考虑光栅的尺寸、重量、性能、功能、兼容性、功能性等因素。
-   根据光栅尺寸设定轴距：确定光栅的最佳尺寸和轴距，采用环形光罩时建议轴距小于30mm，并设置好半径范围。
-   检查电路参数：通过电路的测试和调试，确认电路连接没有问题，必要时可以调整电路。
-   修复破损地方：如果光罩出现破损，可以通过补丁的方法修复，但该方法需要使用不同的工具，费时且不易维护，不建议采用。
-   使用良好的光源：要使用某种光源来进行真实的测试，否则光罩的效果可能会变差。常见的光源如LED、白光等。
-   监控电流：测试前，应该注意光源和电路是否正常，及时发现电流的变化。
-   按孔径顺序施工：光罩的施工顺序应该以孔径的大小顺序，即先施工孔径最小的部分，再施工孔径第二小的部分，直至施工孔径最大的部分。
-   清洗干净：利用措施来清洗光罩，减少污染物的侵扰。如镀锌、洗净、在地里浇水。
## 2.6 光罩优化技术
光罩的优化技术主要有：
-   在光源外部加入镜像材料：在光源外部（如大气层），采用光的反射机制，使光线转换方向。这样，光罩就可以在外部接受更多的能量。
-   用闪烁屏蔽光线：在光源外面使用闪烁屏蔽光，使能量达到平衡，避免光栅元件的损伤。
-   使用三叠光罩：将一根光罩放在电路上，另一根光罩放置于电路的外部，再另外加一根光罩放在两根光罩的中间，通过不同的方向传播光线，提升电路的抗干扰能力。
-   采用高压水冷静压：高压水冷静压能够让电路处于严格的静电状态，不会受到波动的影响，从而保证光罩的稳定性。
-   使用维生剂：在电路连接线上加施压溶剂，降低电路的阻抗匹配情况。
-   使用电镀法：采用电镀的方法使光栅元件的高度和宽度均匀分布，从而减轻粘聚感，改善性能。
-   引导每一处高电压：在电路中引入印制有特殊标记的导线，使每一处电路连接点的电压都达到一个足够大的数值。
