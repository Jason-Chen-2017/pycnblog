# 1.背景介绍


薄膜沉积（Electrical insulation）是电子元件越来越多的原因之一。它与静止电流、电压、以及高频频率特性等相关。电子元件所需要的电介质不同于一般材料所需的介质，所以要用电介质包裹电子元件。目前市面上最普及的电介质主要有：海绵油、聚乙烯、硅橡胶、陶瓷、丁氧铂等。但这些电介质本身又会产生气体以及其他副产品，如水溶液、电极、氮气等，从而对环境造成影响。为了减少电介质对环境的污染，许多电子元件采用薄膜沉积。由于薄膜的厚度在微米到毫米之间，所以又称微透性材料。

在电子元件短路时，由于微透性材料厚度不大的缘故，会导致电流、电压以及高频频率特性的失真。为了避免发生电子短路，就必须考虑到电子元件在低频率下仍能正常工作。根据Kirschning效应（即电子密度偏离均匀状态下的现象），可以在器件内部布置较薄的微透性材料作为电子保护层。由于微透性材料能抑制电子的扩散，使得其周围电流不致增大，因而可以防止电子短路。

除此之外，还有其他一些材料也被用来解决薄膜沉积的问题。例如，锌、镍、铝等金属都可以很好地用于电子保护层的应用。

# 2.核心概念与联系
## 2.1 微透性材料
微透性材料是指厚度在微米到毫米之间的材料。这种材料能够抗蓝光、红外线、可燃冰、水蒸气的侵害。当电子带电状态发生变化时，微透性材料可以有效降低对电子的损伤。常用的微透性材料有金属材料、玻璃材料、塑料、木纺织品、人造革、非金属材料等。

## 2.2 薄膜沉积
电子的低频率特性及其与薄膜之间的关系，进而决定了电子元件的强度及其短路特性。虽然微透性材料有着良好的耐热性，但是其厚度通常不能满足电子需求。因此，电子元件的薄膜沉积现象就显现出来。如果将电子保护层厚度设置的过于厚，则会出现电子电压不稳定甚至电子短路。

电子元件薄膜沉积现象的一个重要原因就是电子密度偏离均匀状态，即Kirschning效应。这一现象表现为电子浓度随着衬底厚度的增大而逐渐降低，并且存在两种不同模式：衬底厚度变小，则电子浓度升高；而衬底厚度变大，则电子浓度减弱。在电子浓度远高于垒导层电流的情况下，再去涂覆衬底，便可能引起电子短路。

## 2.3 电子密度
电子密度反映了电子云密度分布的大小。低电子密度意味着越靠近半导体，电子越密集。高电子密度意味着越远离半导体，电子越分散。因此，电子密度的大小影响着电子密度偏离均匀状态的发生。

## 2.4 电介质
电介质是电子元件在空气中的隔离环境。介质分为四种类型：平衡介质、绝缘介质、共振介质、拼接介质。其中，平衡介质是指没有磁场和电场力的介质，是最容易受到环境的影响的。绝缘介质具有无量纲的绝缘性质，物体遇到它就会发生弯曲、变形或失焦，成为假想物质。共振介质是在紧密接触时，由多种物质组成的一种介质，会发生互相干扰，使电流波形混乱、漩涡等。拼接介质是由两种以上不同物质组成，物体具有多个部位，每一个都可能受到电磁力的影响。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 Kirschning效应
Kirschning效应是由电子密度偏离均匀状态（即电子浓度随衬底厚度的增大而逐渐降低）导致的。简单的说，如果半导体的衬底厚度太薄，那么电子浓度就会偏离均匀状态。电子密度偏离均匀状态可以迫使粒子密度变得更密或者更疏，从而影响电子的传输。因此，对于特别薄的衬底，才可能出现电子密度偏离均匀状态。为了避免这种现象的发生，即使制造者意识不到，也应该尽可能地使衬底薄。

## 3.2 Kirschning效应对电子元件的影响
电子元件薄膜沉积的主要原因是电子密度偏离均匀状态。薄膜的厚度要比电极短，衬底厚度与电流电压的变化关系紧密，因此，对电子密度的偏离可以直接影响电子元件的性能。根据Kirschning效应，制造商可以选择合适的薄膜厚度，来保证电子元件的电压电流性能。对于电子元件缺乏微透性材料的情况，也可以通过添加微透性材料来提高电子元件的抗噪声能力。

# 4.具体代码实例和详细解释说明
为了模拟Kirschning效应，这里给出一个例子。假设有一块微透金属薄膜，厚度为$d_m=0.5\ \mu m$，电流为$I_{leak}=500\ A$。另外假设在电流加上偏移后，衬底的电压为$U_{\text{gap}}=500\ V$，电容值为$C=0.1\ \mu F/cm^2$，计算时假设衬底的宽度为$W=5\ cm$，即$d_w=5\ cm$.

首先，将$I_{leak}$对里程长度进行单位换算，即$q=\frac{I}{A}=\frac{500}{(0.05)^2}\ =\ 31250\ \mu C/\mu m$，表示电流的磁场强度。计算得到$\rho_\text{el}=79\ \mu C/\mu m^3$，表示电子密度。这是一个恒定的值。

然后，根据Kirschning效应公式，计算衬底的电压变化：
$$\Delta U_{\text{gap}}=-\frac{\partial W}{\partial d_w}\cdot I_{leak}$$
约等于$-0.013\ V$。对照衬底厚度，这个电压偏差也不大。因此，假设电流衬底本身就是薄膜的源头，没有涂任何其他的材料。此时的电流密度为：
$$n_{\text{c}}=N\cdot q\cdot (1-f) + N\cdot q\cdot f$$
其中$N$是电流密度，$f$是屏蔽率。由于屏蔽率$f$的大小与衬底的厚度有关，与制造方法无关，因此可以忽略不计。取$N=50000\ A/\mu m^2$，那么：
$$\Delta U_{\text{gap}}\approx-\frac{\partial W}{\partial d_w}\cdot n_{\text{c}}$$
为了验证这个假设，只要找到一块厚度与$\Delta U_{\text{gap}}$之间的关系曲线即可。可以使用韦恩图来进行验证。韦恩图绘制了两变量的函数，横轴是衬底厚度，纵轴是$U_{\text{gap}}$偏移量。$U_{\text{gap}}$是将电流注入衬底后的电压。所以，电流与电压的关系曲线可以画在这样的图上，X轴代表衬底厚度，Y轴代表$U_{\text{gap}}$偏移量。



从图中可以看到，当衬底的厚度增加时，电流的电压越来越高。当衬底的厚度接近于电流电压的最大值时，即衬底的厚度足够大，以至于电流达到了最大值的情况，$U_{\text{gap}}$就达到了峰值。这时，电流的电压将再次偏离均匀状态，即偏离了$-\frac{\partial W}{\partial d_w}\cdot n_{\text{c}}$的距离。由于电流密度$n_{\text{c}}$的大小与衬底的厚度$d_w$有关，$d_w$的值是固定的，因此这个效应对其他的衬底尺寸而言也是一样的。由于薄膜的设计一般不是一个固定的值，而是依据制造工艺决定的，所以这种效应可能无法预测准确的结果。

另外，也可以从模拟实验中得到另一种观点。实际操作时，可以将一堆负电流注入微透金属薄膜，然后让其挤出。一般来说，当一切按设计要求施工完毕之后，当下流电压降到微亿级的时候，衬底的电压基本上就会降下来。这时候，电流与电压的关系曲线会呈现出一条平滑的曲线。因为实际电流可能远小于$I_{leak}$，所以这样的仿真演示并不一定能反映实际情况。

总结一下，Kirschning效应是一种随机而随着衬底厚度变化的现象，使得电流、电压以及高频频率特性等保持较高的一致性，从而降低电子元件短路的风险。在电子元件短路的情况下，可以通过添加微透性材料以及施工人员的注意力集中在电子元件薄膜设计和生产等方面来缓解这个问题。
