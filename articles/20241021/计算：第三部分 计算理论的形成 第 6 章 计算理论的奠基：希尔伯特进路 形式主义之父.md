### 第 6 章 计算理论的奠基：希尔伯特进路 “形式主义”之父

#### 1. 希尔伯特及其形式主义运动

## 1.1 希尔伯特的生平与贡献

大卫·希尔伯特（David Hilbert），1862年出生于德国的柯尼斯堡（今天的加里宁格勒），是20世纪最杰出的数学家之一。他的工作覆盖了数学的多个领域，包括几何学、数论、代数学和泛函分析。希尔伯特不仅以其深刻的数学贡献闻名，还以其在教育领域的贡献和对数学哲学的思考而受到尊敬。

希尔伯特的数学贡献涵盖了广泛的主题。他在几何学中引入了希尔伯特空间的概念，这个概念在现代数学和物理学的许多分支中都有应用。在数论中，他提出了著名的希尔伯特第十问题，该问题至今仍未解决。此外，他还提出了关于集合论和数学基础的一系列深刻问题，这些问题促使了数学逻辑和形式主义的发展。

在学术生涯中，希尔伯特担任了多个重要职位，包括哥廷根大学的教授和德国数学学会的主席。他的演讲和讲座影响了无数年轻数学家，其中包括了像库尔特·哥德尔和保罗·埃尔德舍尔德这样的杰出人物。

## 1.2 形式主义运动的兴起

形式主义是一种哲学和数学方法，它主张数学的真理是由符号系统中的证明得到的，而不是依赖于外部世界的物理实体。形式主义的兴起可以追溯到19世纪末和20世纪初，当时的数学界面临着一系列危机和挑战。

首先，数学基础的危机是一个主要推动力。在19世纪，集合论的发展揭示了许多数学问题中的矛盾和悖论，如罗素悖论。这些问题挑战了数学的基本原则，引发了关于数学真理和意义的深刻讨论。为了解决这些基础问题，数学家们开始寻求更加严格和形式化的数学方法。

其次，逻辑学的发展也为形式主义的兴起提供了动力。哥德尔的不完全性定理和罗素的类型论等成果表明，逻辑和数学之间存在深刻的内在联系。这些发现促使数学家们开始用逻辑的方法来构建数学系统，从而避免了基础危机带来的不确定性。

希尔伯特是形式主义运动的领军人物之一。他主张通过公理化的方法来建立数学系统，这种方法确保了数学定理的证明是严格和逻辑上无懈可击的。希尔伯特的公理化程序不仅是一种数学方法，也是一种哲学立场，它主张数学的真理是独立于物理世界的。

## 1.3 形式主义的基本原理

形式主义的基本原理可以概括为以下几点：

1. **符号系统**：形式主义将数学视为一个由符号和规则组成的系统。在这个系统中，符号代表数学概念，而规则定义了如何使用这些符号进行推理和证明。

2. **公理和规则**：形式主义的数学系统由一组公理和推导规则构成。公理是基本的不证自明的假设，而规则是用于从公理推导出更复杂命题的方法。

3. **证明的严格性**：形式主义强调证明的严格性，即证明过程中的每一步都必须基于公理或之前已证明的定理。这种严格的证明方法确保了数学结论的逻辑一致性和可靠性。

4. **独立于经验**：形式主义认为数学真理独立于外部世界的经验。数学命题的真假不是基于经验的验证，而是基于符号系统中的证明。

形式主义对数学的影响是深远和广泛的。它不仅为数学提供了更加严格和一致的基础，还为后来的计算机科学和逻辑学的发展提供了理论基础。形式主义方法促使数学家们将注意力从具体的数学问题转向更抽象的数学结构，从而推动了数学的抽象化和形式化进程。

#### 2. 形式化方法的引入

## 2.1 数学基础的危机

数学基础的危机是指19世纪末和20世纪初，数学界面临的关于数学逻辑一致性和真理性的挑战。这一危机主要源于集合论的发展揭示了许多数学问题中的悖论和矛盾，如罗素悖论和康托尔悖论。这些悖论表明，传统的集合论无法提供一致的基础来支持数学的所有分支。

罗素悖论是由伯特兰·罗素提出的，它揭示了集合论中的一个自相矛盾的情况。罗素悖论的核心是“集合的集合”的问题，即是否存在一个集合，它包含所有的集合，但不包含自身。罗素发现，如果我们假设存在这样一个集合，那么就会产生逻辑上的矛盾。

康托尔悖论是由乔治·康托尔提出的，它揭示了集合论的另一个悖论。康托尔悖论表明，对于任何集合S，都存在一个更大的集合S'，即S'包含了S的所有超集。这个悖论挑战了集合论的基本原则，特别是集合的无限性。

数学基础的危机引发了广泛的讨论和辩论。许多数学家开始寻找更加严格和一致的基础来重建数学，从而避免这些悖论的出现。这种需求催生了形式主义运动的兴起。

## 2.2 形式化方法的意义

形式化方法在数学基础危机中起到了关键作用。它通过将数学理论转化为一个由符号和规则组成的系统，从而避免了逻辑上的悖论和矛盾。形式化方法的意义主要体现在以下几个方面：

1. **提供一致的基础**：形式化方法通过公理和推导规则建立了数学系统的一致性。每个数学命题都必须基于公理或之前已证明的定理，从而确保了数学结论的逻辑一致性。

2. **消除模糊性**：形式化方法将数学表达为明确的符号系统，从而消除了传统数学表达中的模糊性。这种精确的符号表达使得数学推理更加清晰和准确。

3. **促进抽象化**：形式化方法鼓励数学家将注意力从具体的数学问题转向更抽象的数学结构。这种方法推动了数学的抽象化进程，促进了数学理论的发展。

4. **增强可靠性**：形式化方法通过严格的证明过程确保了数学结论的可靠性。每个数学定理的证明都必须遵循公理和推导规则，从而避免了错误和谬误的出现。

## 2.3 希尔伯特的公理化程序

大卫·希尔伯特是形式主义运动的领军人物之一，他提出了公理化程序作为建立数学基础的方法。希尔伯特的公理化程序包括以下几个步骤：

1. **选择研究对象**：首先，希尔伯特建议选择一个特定的数学领域或问题，例如几何学或数论。

2. **定义基本概念**：接下来，希尔伯特要求定义该领域中的基本概念，如点、线、数等。这些基本概念应该通过公理来定义，而不是依赖于直观的理解。

3. **建立公理系统**：然后，希尔伯特建议选择一组公理，这些公理应该是基本的不证自明的假设。这些公理构成了数学系统的核心。

4. **定义推导规则**：在公理系统的基础上，希尔伯特建议定义一套推导规则，用于从公理推导出更复杂的命题和定理。

5. **证明定理**：最后，希尔伯特要求使用推导规则从公理系统中证明定理。这些定理构成了数学理论的主体。

希尔伯特的公理化程序确保了数学系统的一致性和可靠性。通过将数学理论转化为一个形式化的系统，希尔伯特的方法为数学基础提供了坚实的基础，从而避免了逻辑上的悖论和矛盾。

总之，形式化方法的引入是为了应对数学基础的危机，它通过公理化和严格的证明过程为数学提供了更加一致和可靠的基础。希尔伯特的公理化程序是形式主义方法的一个重要实现，它对数学的发展产生了深远的影响。

#### 4. 希尔伯特的空间理论

## 4.1 希尔伯特空间的定义

希尔伯特空间是泛函分析中的一个基本概念，它是欧几里得空间在无穷维情形下的推广。希尔伯特空间是一种内积空间，它不仅具有向量空间的性质，还包括了内积运算和范数结构。

定义：一个实希尔伯特空间H是一个复数域上的内积空间，满足以下条件：

1. **完备性**：H是一个完备的度量空间，即每一个柯西序列都在H中收敛到某个极限。

2. **内积**：对于任意两个元素\( x, y \in H \)，存在一个内积运算，记作 \( \langle x, y \rangle \)，满足以下性质：
   - **对称性**：\( \langle x, y \rangle = \overline{\langle y, x \rangle} \)。
   - **线性性**：\( \langle ax + by, z \rangle = a\langle x, z \rangle + b\langle y, z \rangle \)，其中\( a \)和\( b \)是复数。
   - **正定性**：\( \langle x, x \rangle \geq 0 \)，且\( \langle x, x \rangle = 0 \)当且仅当\( x = 0 \)。

3. **范数**：对于一个元素\( x \in H \)，存在一个非负实数范数\( \|x\| \)，满足以下性质：
   - **正定性**：\( \|x\| \geq 0 \)，且\( \|x\| = 0 \)当且仅当\( x = 0 \)。
   - **齐次性**：\( \|ax\| = |a|\|x\| \)，其中\( a \)是复数。
   - **三角不等式**：\( \|x + y\| \leq \|x\| + \|y\| \)。

## 4.2 希尔伯特空间的性质

希尔伯特空间具有一系列重要的性质，这些性质使其在数学和物理学中有着广泛的应用。

1. **完备性**：希尔伯特空间的完备性保证了其中每个柯西序列都收敛到某个极限。这一性质使得希尔伯特空间中的函数和序列具有更好的分析性质。

### 7. 形式主义与直觉主义的比较

## 7.1 形式主义和直觉主义的基本观点

形式主义和直觉主义是20世纪数学基础研究领域中的两种主要哲学立场，它们在数学真理的本质和数学方法的合理性上有着根本的分歧。

**形式主义的基本观点**：
形式主义主张数学的真理在于符号系统中证明的有效性，而不是依赖于现实世界的实际对象。形式主义者认为，数学是一种逻辑结构，其基本元素是符号和规则。数学命题的真假由符号系统中的证明来确定，而不是通过经验验证。形式主义者强调数学证明的严格性，即每一步推理都必须基于已证明的定理或公理。

**直觉主义的基本观点**：
直觉主义则认为数学命题的真假依赖于我们对现实世界的直观理解。直觉主义者认为，数学的基础是直观的概念和过程，如自然数的概念和集合的形成。直觉主义者反对使用不可构造的抽象概念和证明方法，强调数学推理必须是可以实际验证和执行的。因此，直觉主义者通常只接受那些可以通过有限步骤构造出来的证明。

## 7.2 形式主义和直觉主义的优缺点

**形式主义的优点**：
1. **一致性**：形式主义通过严格的逻辑推导和公理系统确保了数学的一致性，避免了集合论中的悖论。
2. **普适性**：形式主义方法具有普适性，可以应用于广泛的数学领域和逻辑系统，为数学和计算机科学的发展提供了坚实的基础。
3. **严格性**：形式主义强调证明的严格性，使得数学推理更加可靠和严谨。

**形式主义的缺点**：
1. **复杂性**：形式化方法通常涉及复杂的符号系统和繁琐的证明过程，使得实际应用中难以实施。
2. **抽象性**：形式主义过于抽象，可能导致数学与现实世界的脱节。
3. **不可验证性**：形式主义中的某些证明可能过于复杂，以至于难以在实际中验证其正确性。

**直觉主义的优点**：
1. **直观性**：直觉主义强调直观的理解和实际的操作过程，使得数学更加贴近现实世界。
2. **可验证性**：直觉主义认为数学证明必须是可构造的，这意味着证明可以在实际中验证。
3. **实用性**：直觉主义方法在处理具体问题时往往更加直观和易于理解。

**直觉主义的缺点**：
1. **局限性**：直觉主义在处理抽象和复杂的数学问题时显得力不从心，难以提供统一的理论框架。
2. **不可靠性**：直觉主义依赖于个人的直观理解，这可能导致逻辑上的谬误和不一致。
3. **不普适性**：直觉主义方法通常难以应用于所有数学领域，其适用范围有限。

## 7.3 形式主义和直觉主义的关系

形式主义和直觉主义在数学基础的研究中有着复杂的关系。一方面，形式主义和直觉主义之间存在明显的对立和分歧。形式主义强调逻辑证明的严格性和符号系统的普适性，而直觉主义则强调直观理解和实际操作的合理性。另一方面，形式主义和直觉主义也存在互补性。

### 10. 形式主义案例分析

#### 10.1 某个具体形式化方法的介绍

在形式化方法中，B方法（B Method）是一个广泛应用的实例。B方法是一种基于数学模型的形式化软件开发方法，由Edmund M. Clarke等人于1980年代初期提出。B方法的主要目标是确保软件系统的正确性和可靠性，通过形式化的规格说明和验证来消除潜在的错误和缺陷。

B方法的核心思想是将软件系统规格说明转化为一个数学模型，然后使用形式化的验证工具来验证系统的行为是否符合预定的规格。B方法包括以下几个关键组成部分：

1. **抽象状态机（Abstract State Machines, ASM）**：ASM是一种用于描述软件系统行为的数学模型。它由状态、转移和操作组成，能够精确地描述系统的动态行为。ASM模型具有形式化的定义，使得其能够被形式化验证工具自动验证。

2. **规格说明语言**：B方法使用专门的规格说明语言（如B语言）来描述系统的规格。B语言是一种形式化的数学语言，支持状态、转移和操作的描述，使得规格说明具有高度的精确性和一致性。

3. **验证工具**：B方法的验证工具包括形式化验证器（如CBV和B-Verify），这些工具可以自动验证ASM模型是否满足预定的规格。验证工具使用逻辑推理和模型检查技术来确保系统的正确性。
