# 1.简介
  

## 1.1 什么是正交变换？
正交变换(Orthogonal Transformation)又称对角化变换、正交投影(Orthonormal Projection)，它是一种将一个二维或三维空间中的点映射到另一个正交坐标系下的过程。通过正交变换，可以将任意多边形转化成等腰直角三角形、长方形、圆形等形状，还可以将二维或三维空间中的曲线、面、体进行相似变换，实现复杂三维模型的切割、运动、缩放等操作。如下图所示：

## 1.2 为什么要用正交变换？
任何一个坐标系都可以看做是空间中某种特定形式的坐标变换，例如平面坐标系、直角坐标系、极坐标系、球坐标系等。不同坐标系之间的转换关系主要由坐标轴之间的关系确定，例如平面直角坐标系的两个坐标轴分别对应地心直径和第一象限坐标轴；而球坐标系则与笛卡尔坐标系的关系更加紧密，其由一个位置矢量和三个旋转矢量决定，相比其他几种坐标系更容易实现向量运算和计算。这些坐标系之间存在大量的相互转换关系，如何有效地利用这些转换关系进行计算，是一个值得探讨的问题。

正交变换的出现是为了降低坐标系间的转换难度，使得不同坐标系之间的运算更加简单，从而提高计算效率。正交变换在图像处理、模式识别等领域有着广泛应用，能够将复杂曲面、不规则体、三维模型等变换到二维或三维的图像上。正交变換的最主要作用是转换、裁剪、投影、压缩、旋转、扭曲等多种功能。

# 2.相关技术概念
## 2.1 坐标系
在正交变换中，需要建立两个坐标系，通常情况下是两个不同维度的空间坐标系。当一个坐标系固定时，另一个坐标系就可以通过对第一个坐标系进行正交变换得到。比如在二维空间中，第一个坐标系可以采用平行坐标系（直角坐标系），第二个坐标系可以采用极坐标系（极坐标系由极径和极角构成）。当两个坐标系满足正交性时，就可以将第一个坐标系中的点映射到第二个坐标系中去。

一般来说，正交坐标系可分为三种类型：
* 直角坐标系：这种坐标系称作平行坐标系。它的两条坐标轴是水平的、垂直的和斜的，任给任一点都可以用三个数字表示（x,y）=(r cosθ, r sinθ)。这种坐标系具有直观易懂的特点，并且在一定程度上还原了欧氏几何的一些特性。
* 椭圆坐标系：椭圆坐标系的两条坐标轴可以是同一直线上的。两个坐标轴之间存在弧度，并且是一比一。如同极坐标系一样，椭圆坐标系也可用两组参数来描述（a, b, θ）=(r cosθ, rsinθ,θ),其中θ表示的是椭圆的倾斜角。
* 射影坐标系：射影坐标系表示的一般是三维空间中的对象，其坐标轴一般是垂直于某个参考线的直线坐标轴。这样的一个坐标系就是射影坐标系。

## 2.2 射影变换
射影变换（Projection transformation），又叫投影变换、坐标变换（Coordinate transform），是在两个坐标系之间的一个空间变换。这个变换将三维空间中的物体投影到某个二维平面或者一张二维图片上。射影变换可以将三维空间中的点投影到平面上，即在前三个坐标轴上选取一个基准，然后将剩余的两个坐标轴对齐，这样就把一个三维空间中的点投影到了二维平面上。投影变换也可以反过来，即从二维平面或者一张二维图片投影到三维空间。投影变换可以用于光学仿真、标志物识别、运动物体跟踪、机器人导航、三维建模、三维打印、虚拟现实、摄影制作、图像处理以及医学影像分析等领域。

## 2.3 正交变换与射影变换的区别
正交变换与射影变换都是空间变换，只是它们的目标坐标系不同。对于二维空间，射影变换是将三维空间中的点投影到二维平面上，而正交变换则是把二维空间中的图形变换成直角坐标系下面的图形。因此，射影变换与正交变换的目的不同，但是它们的本质是相同的。 

另外，虽然二维和三维都可以使用正交变换和射影变换，但二维射影变换也可以看做是三维空间的正交变换，反之亦然。
