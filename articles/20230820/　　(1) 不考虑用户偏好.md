# 1.简介
  


互联网产品开发过程中，客户的反馈永远是重中之重，要想获得满意的用户反馈，我们需要在不考虑用户偏好的前提下设计产品。所谓不考虑用户偏好的产品就是功能完善、界面清晰、易于使用、安全性高等指标都很高。其设计理念主要围绕着以下五个方面：

1. 功能完善：不论是网站还是APP，都需要有丰富的功能和特色。这样才能吸引到更多的用户，提升产品的知名度。
2. 界面清晰：产品界面应该清晰美观，用户能够快速找到所需信息，并方便地进行操作。只有界面清晰，才能让用户从容不迫地使用产品。
3. 易于使用：界面功能模块化、操作便捷，用户只需简单的一步操作，即可完成相关任务，降低了用户的学习成本。
4. 安全性高：产品的安全性是所有产品都不能忽视的，尤其是在互联网时代。一款产品如果不能满足用户对其安全的需求，将会带来严重的安全隐患。
5. 用户体验好：产品的用户体验，也是一个重要的评判标准。好的用户体验可以说直接影响用户的使用频率和持续时间，进而促进销售额和盈利能力的增长。

为了实现以上五点要求，我们首先要了解这些产品的基本功能及特性，然后再根据实际情况制定相应的功能策略，来保证产品的最佳性能。同时，我们还要注意尽可能缩小产品的功能，确保功能的专注度，避免过度复杂化，增加用户学习难度，从而达到不考虑用户偏好的目的。



# 2.基本概念和术语说明

## 2.1 用户群组分类及特征

### 2.1.1 目标受众

　　1）高端市场（大中小企业）：由于高端市场的人力、资金、技术水平较高，所以各类服务都需要精心打造，以满足消费者的需求。例如手机品牌商，飞机品牌商，房屋装修企业等。

　　针对这些企业客户，通常都有着不同的购买决策流程和追求，不同用户群体有着不同的诉求。一些用户更侧重细节，希望有专业的客服人员为他们解决问题；另一些用户则更倾向于大局观，希望更加专业化的服务和价格优惠。

　　因此，针对不同的用户群体，公司可以选择不同的互动方式、针对性的营销方式、定价策略。高端市场的客户群体往往具有高度的责任感、追求卓越的品质以及独具魅力的个性。

　　例如：苹果手机业务是苹果公司的主要销售渠道，其目标用户主要是年轻消费者，需要掌握苹果产品的最新特性和功能，并且希望能够享受到即时的性能支持和故障补救。而对于一般年龄段的人群，苹byteimg远没有提供足够的特别优惠。

　　国内互联网电商平台拼多多也是高端市场客户群体的典型，其目标用户群体广泛且分布广泛，需求层次较为复杂，有着丰富的购物需求。

　　例如：拼多多是中国第一款商城级电商平台，占据了互联网电子商务市场绝大多数份额，拥有全球最大的线上货源资源。因此，拼多多用户除了具有超高的购物兴趣外，还需要承担起解决交易中的各种问题、评估商品质量、配送效率等一系列服务上的烦恼。

　　同样，对于房屋装修企业，高端市场客户群体则更加关注其品质、效果和投资回报。例如：华为在奥迪、宝马、诺基亚等车企领域，都是一家独大的行业龙头。

　　2）中端市场（政府/事业单位）：中端市场包括政府、事业单位以及其内部的办公室、机关等，它们对某些产品的使用比较特殊或有限。例如，文化用品市场，电子元器件市场，工厂管理软件市场，医疗设备市场等。

　　由于中端客户群体的消费能力相对较弱，相比高端市场的用户有着明显的需求落差。但是，中端客户群体又存在着大量的知识积累、技术含量较高、擅长沟通协作、工作效率高的特点。

　　因此，针对中端客户群体，公司需要基于自身业务特点、客户偏好构建符合自身需求的产品形态，并且保证核心功能的可用性。

　　例如：政务部门作为政府最常用的工作工具，对新闻舆情监测有着特殊需求。虽然媒体人、记者等普通人更关注政务部门提供的政策解读，但政府却更偏爱听取公众意见和建议。所以，我们可以设计出一款政务软件，专门用于监测和分析公共事务，并提供可供各部门互动的沟通平台。

　　同样，工厂管理软件市场也是中端客户群体的典型，对系统的日常运行有着直接的需求，需求量与数量都很大。所以，工厂管理软件应当具备较强的管理功能、直观的图表呈现、方便的操作手册。

　　例如：厨房装修软件是一款基于PC端的工厂管理软件，包括烹饪工艺、生产过程控制、现场指导、设备管理等功能。然而，中小型企业建设工厂规模一般，不适合安装PC服务器，只能采用移动终端版本。我们可以开发一种能够兼顾两者优点的产品，让工厂管理软件可以在小屏幕设备上运行，帮助中小型企业节省成本和提高效率。

　　最后，针对医疗设备市场的客户群体，由于需求特殊性较强，市场规模小，有着更加复杂的消费模式。所以，医疗产品的设计也需要考虑到用户的症状定位、相关护理知识、设备运用方法等方面。

　　例如：护肝保湿药品行业正蓬勃发展，希望能够满足中老年人的特殊需求。所以，医疗公司可以开发一款智能护肝保湿产品，通过机器人设备搭配专业护士和医生的技术团队，为消费者提供最科学的护肝保湿方案。

　　总结来说，由于市场细分化的特点，使得我们需要为不同的用户群体提供不同的产品形态，以实现不考虑用户偏好的需求。为此，我们需要定义清楚目标受众群体的身份特征和购买决策流程，以及针对不同角色设计不同的互动形式、营销方式、定价策略。
