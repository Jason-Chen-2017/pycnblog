# 1.简介
  

多视觉立体视觉(multi-view stereo vision)是指在同一个视野下同时捕捉多个不同视图图像中的目标，并利用其之间空间关系、相机内外参数等信息进行空间配准。它可以帮助机器识别自然环境中物体的空间分布、形状、运动轨迹及相互作用，具有广泛的应用领域。近年来，基于深度学习的多视觉立体视觉算法有了显著的发展。本文对现有的基于深度学习的多视觉立体视觉算法进行系统性的回顾，并分析其优缺点，总结其目前在不同任务上的研究进展。最后给出其未来的研究方向与挑战。


## 1.1 文章结构和组织方法
文章首先简要介绍相关理论基础知识，包括多视觉立体视觉的原理、主要算法及其特点、深度学习、多视角推理、空间配准等等。然后详细阐述具体算法的原理、特点及操作步骤。最后提出一些研究课题和挑战，并给出未来的研究方向。
