## Long Short-Term Memory (LSTM)
长短时记忆神经网络（Long Short-Term Memory Neural Network，LSTM）是一种能够有效记住并处理时间序列数据的神经网络模型，是一种特别适合于循环神经网络（Recurrent Neural Network，RNN）结构的循环神经网络模型中的一类。

它由Hochreiter 和 Schmidhuber在1997年提出，可以说是一种重要的模型，被广泛应用于如语言建模、图像分析等领域。通过LSTM可以学习到长期依赖关系，从而提升神经网络的学习能力。


# 2. 基本概念术语说明
## 时序数据及其特点
时序数据又称为时间序列数据或是时间记录数据，是一个关于一个或多个变量随时间变化情况的记录，其特征为顺序性、整齐性、连续性。例如：股票价格数据、气温数据、地震数据等都属于时序数据。在这样的数据中，每一条记录都是按照一定时间间隔采集到的，因此在时间上具有先后次序。由于数据之间存在相互依存关系，所以需要用模型去预测未来的变化趋势，并且这些模型通常都有很强的时序特性。

时序数据的特点：
1. 有时间性：每个观察事实都会随着时间的推移而增加、减少或变动。
2. 有顺序性：各个观察事实之间存在着相关性。
3. 有整齐性：数据采集的时间与单位是一致的。
4. 有重复性：观察事实可能出现重复。

## LSTM网络结构
LSTM网络结构包括输入门、遗忘门、输出门以及记忧更新单元（cell）。

1. 输入门：当LSTM要注意到某个特定的输入时，会向神经元引入新的信息，该层决定了那些神经元应该被激活，哪些不应该。这就像人的眼睛有视觉区域，只有进入了那个视觉区域的信息才会被接受。

2. 遗忘门：这个门负责抹掉一些记忆，使得之后的决策只基于当前的输入。

3. 输出门：这个门决定了一些信息应该如何被输出。

4. 记忆更新单元：LSTM的核心部分，它主要负责存储之前看到过的信息，以及对新输入的信息做出反应。LSTM的记忆单元包括两个部分：
    * 细胞状态（Cell state）：存储了当前时刻的长期信息。
    * 隐藏状态（Hidden state）：也是作为下一步计算输入的部分，也是训练过程中必须要优化的参数之一。

## 神经网络的优化目标
传统的机器学习方法通常采用损失函数最小化的方法求解神经网络参数。但是在时序数据预测任务中，由于模型对未来数据存在依赖关系，所以往往采用其他的优化目标，如预测准确率、损失函数等。
