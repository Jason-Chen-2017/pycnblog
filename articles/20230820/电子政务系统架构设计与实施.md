# 1.简介
  

电子政务系统（Electronic Public Administration System, EPS）是指由政府、政党、企业等参与者通过互联网或其他方式提供的在线服务，包括调查问卷、法律咨询、政策咨询、信访、报名咨询、投诉举报、信息发布、在线缴费、在线申领证件等功能。电子政务系统主要面向具有较高综合素质和信息化水平的群众群体，对实现共同富裕、促进社会治理、推动政治稳定具有重要作用。国内外多方面已经开发了不同类型的电子政务系统，例如中国地方志政务平台、北京市网上执业公正评议中心、广东省网上咨询平台等，这些系统都是为了解决当地政府管理政务的“一站式”需求而研发的。近年来，随着互联网技术的飞速发展，政务信息的快速更新、数据量的急剧膨胀、用户行为的不断变化以及社会的需求变迁，政务机关将信息公开和网上服务作为自己的战略发展方向，基于互联网和人工智能技术进行政务建设，逐步形成了一套完整的电子政务系统体系。
由于电子政务系统涉及的复杂性和系统的庞大规模，从服务对象定位到架构设计、模块功能划分、开发框架搭建、系统部署运维等整个流程，都需要具备相应的知识储备和技术能力，甚至要有跨界融合的才能。因此，成为一名优秀的电子政务系统工程师，不仅需要极强的编码能力和技术水平，还要求对信息技术、政务系统、计算机科学、互联网、管理学等多个领域有深入的理解，并且积极乐于分享和交流自己的经验、教训、感悟。本文旨在对电子政务系统相关技术进行全面描述并提出一些建设上的建议，希望能帮助读者更好地理解和掌握电子政务系统建设的相关技能和方法。
# 2.电子政务系统概述
## 2.1 电子政务系统定义
按照国家统一标准，电子政务系统是一个由政府、政党、企业等参与者通过互联网或其他方式提供的在线服务，包括调查问卷、法律咨��、政策咨询、信访、报名咨询、投诉举报、信息发布、在线缴费、在线申领证件等功能。电子政务系统主要面向具有较高综合素质和信息化水平的群众群体，对实现共同富裕、促进社会治理、推动政治稳定具有重要作用。电子政务系统应具有以下特点：
- （1）独立运行：电子政务系统是一个独立运行的系统，它与行政组织的日常工作过程分离，运行于政府网站之外；
- （2）覆盖全国各级政府：电子政务系统覆盖全国所有省、自治区、直辖市、计划单列市和特别行政区的政府部门，涵盖生活小区、工商企业、事业单位、农村集体经济组织、专业技术人员、个人和其他组织等；
- （3）信息时代：电子政务系统正在进入信息时代，利用数字化技术提升信息处理效率和服务品质；
- （4）民间参与：电子政务系统鼓励群众的参与，民间力量成为其基础和组成部分。
电子政务系统的构架可以分为三个层次：
- （1）应用级应用系统：应用级应用系统包括政务服务信息管理系统、信息采集系统、信息发布系统、办事工具系统、信息查询系统等，为政务机关提供了最基本的信息收集、整理、分析和呈现手段，为民众提供了最直接的服务；
- （2）政务内容服务系统：政务内容服务系统包括政策法规资讯发布系统、政务活动组织、参与监督检查、评价反馈等，是政务机关综合运用信息资源和人力资源进行政务内容服务的工具；
- （3）政务技术服务系统：政务技术服务系统包括信息采集、计算分析、数据存储、政务服务机器人等，是在政务服务体系中承担核心技术支撑的基础设施，是政务机关实现政务决策、调控、法律咨询、预警、事务审批等服务的重要工具。
## 2.2 电子政务系统运作流程
根据政府“一站式”管理的要求，一般情况下，政务信息通过综合网络、手机APP、微信公众号、微博客等渠道从政务机关系统导出，然后经过系统自动采集、分类、存储、过滤、转换、标注等处理，形成内部资料库，供政务机关后续操作。如此，政务机关就可以根据自身的业务需求，选择适用的政务内容服务系统，通过收集的、整理的、分析的政务信息进行政务内容服务。
## 2.3 电子政务系统角色、职责、任务和制度
电子政务系统建设过程中，涉及到多个角色、职责、任务和制度。其中，以下几类人员是电子政务系统建设的关键角色：
- （1）政府官员：该角色负责管理政务的各项事务，包括建立政务信息库、编制政务信息、编写政务规章、发布政务通告、履行政务承诺义务、服务热线、事务审批等；
- （2）公众部门：该角色负责向社会公众提供政务信息，包括向媒体报道政务动态、开展政务活动、收听政务新闻等；
- （3）第三方机构：该角色负责提供政务咨询、法律咨询等第三方服务，包括市场营销、人才推荐、法律咨询、社会公益、评估支持等；
- （4）电子政务系统开发人员：该角色负责构建、维护、运营电子政务系统，包括项目计划、需求调研、系统设计、实现、测试、部署、运维等；
- （5）IT技术人员：该角色负责使用IT技术搭建、运营电子政务系统，包括应用级开发、政务内容服务开发、政务技术服务开发等；
- （6）系统管理员：该角色负责电子政务系统的日常运维、管理、保障等工作，包括安全审核、服务器维护、数据备份、网络故障排除、数据查询等；
- （7）信息管理人员：该角色负责对政务信息进行收集、整理、分类、存储、过滤、转换、标注等处理，包括信息采集、信息验证、信息采购、信息核实等工作。
电子政务系统的建设过程中，还有一些制度约束需要遵守。例如，政府官员必须在法律允许范围内发布政务信息，公众部门应配合政府官员发布政务信息，第三方机构应履行公共利益及公共职能的责任。同时，电子政务系统开发、运营等工作也需要通过政府的相关部门进行统筹安排，确保政务信息的畅通无阻。
