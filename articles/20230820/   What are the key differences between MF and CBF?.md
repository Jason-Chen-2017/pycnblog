# 2.基本概念术语说明
首先，我们需要了解一下相关的基本概念和术语。什么是矩阵分解？什么是连续自回归模型（Collaborative Basis Function model, CBF）？为什么这两个名字这么有趣呢？下面让我们来详细介绍一下。
## 2.1 矩阵分解
矩阵分解，也叫奇异值分解（Singular Value Decomposition），是一个数学方法，它将一个矩阵分解成三个相互正交的矩阵的乘积：其中有一个矩阵是奇异矩阵（即只有实数元素且为零或非常小的值），另外两个矩阵则是其对应的向量。矩阵分解得到的三个矩阵满足如下关系：A=UΣV^T，其中A为待分解的矩阵，U为左奇异矩阵（对角线上为1，其他元素为0），Σ为奇异值矩阵（对角线上为奇异值，大小由最大的奇异值的绝对值决定），V为右奇异矩阵（对角线上为1，其他元素为0）。矩阵A可以由奇异值矩阵Σ、左奇异矩阵U、右奇异矩阵V的乘积表示：A=UVΣ。如图1所示，矩阵分解可以把任意一个矩阵A分解成三个矩阵U、Σ、V，使得矩阵A最好的表示为UΣV^T。

矩阵分解最早被用于图像处理，它把一幅图像分解成一组重要的方向和一组适应该方向的颜色。在文本处理中，它还可以用于主题建模、电影推荐、数据压缩、特征提取、图像修复等领域。矩阵分解已经成为许多计算机科学和应用领域的基础工具。
## 2.2 连续自回归模型（Collaborative Basis Function Model, CBF）
连续自回归模型（Collaborative Basis Function Model, CBF）是一种基于用户群体的推荐算法，它利用用户的点击行为和物品特征信息来预测用户对不同物品的兴趣程度。它假设用户对物品的兴趣随时间变化，并用多个线性模型描述这种动态变化。CBF模型本质上是一种线性模型，不同的是，它的基函数是多项式或者高斯径向基函数。CBF模型对用户和物品进行建模，使用隐含语义分析（Latent Semantic Analysis, LSA）的方法进行矩阵分解。LSA就是指根据物品间的语义关系，将它们映射到低维空间，而后对低维空间中的数据进行矩阵分解。由此，可以获得用户的潜在兴趣、物品之间的关系及物品的表达能力。如下图所示，CBF模型包括用户嵌入矩阵和物品嵌入矩阵。

CBF模型是一种改进的矩阵分解模型，主要的改进在于它的基函数，CBF模型的基函数不是简单的向量，而是多项式或者高斯径向基函数。通过使用不同的基函数，CBF模型可以捕捉到不同类型物品的表达特性，同时还可以考虑到物品的上下文环境。CBF模型在实际应用中表现出良好的效果，在很多领域中都有很好的性能。
## 2.3 为何这两个名字这么有趣？
为什么这两个名字这么有趣呢？简单来说，它们就是两类东西的代名词。想必每个人都听说过“奇异值分解”吧？这个名字牵扯到了一种矩阵运算的技巧——分解。在计算机视觉、生物信息学、信号处理、图像处理等领域，都是利用矩阵分解做一些有意思的事情。在推荐系统、社会网络分析、CTR预估、图像检索、文本分类、个性化推荐等方面，CBF模型也是常用的模型。这两者之间的差别只是形式上的不同，但背后的理论其实是相同的。那么，它们之间到底有什么共同之处呢？
我认为，共同之处有两个。第一，都是用来表达某种模式，都可以看作是一种矩阵分解的方法。第二，它们使用的基函数不一样，但效果却差不多。CBF模型更像是统计学习的一种特殊情况，它使用了复杂的基函数来描述用户和物品的关系。两种模型各有千秋，总体而言，如果没有特定的任务要求，一般情况下使用哪种模型并无太大影响。
