# 1.简介
  

动态规划（Dynamic Programming）是运筹学的一个重要分支，属于求策问题解决类。其特点在于它利用以前的结果对当前进行计算，从而避免重复计算，因此被认为是一种优化技术。动态规划可用于许多最优化问题，如求解最大流问题、最小费用流问题等。
蒙特卡罗方法（Monte Carlo Method），也属于运筹学的一类方法。它通过随机采样的方法来近似求解复杂的问题，常用于模拟系统行为及物理过程，如概率论、统计学、工程学等领域。其中，蒙特卡罗方法通常与数值计算结合起来，如运用模拟退火算法（Simulated Annealing）。
一般地，动态规划与蒙特卡loor方法可以说是运筹学的两个主要方法。由于动态规划和蒙特卡罗方法都具有很强的数学性质，并且有很多重叠的地方，所以可以将两者整合成一个更高层次的方法——近似算法（Approximation Algorithm）。近似算法结合了数值计算、近似计算以及随机化来解决一些经典的优化问题。
近几年，随着计算机科学的快速发展，动态规划和蒙特卡罗方法已经成为现实世界中使用的两种方法。然而，动态规划和蒙特卡罗方法仍然存在很多局限性。如何更好地理解并应用动态规划和蒙特卡罗方法则成为今后需要面对的问题。本文试图对动态规划和蒙特卡罗方法做出全面的阐述。希望能够从宏观角度、微观角度和应用角度三方面对他们做出解释。文章适合具有一定相关知识的人群阅读。
# 2.基本概念术语
## 概念
- 动态规划（Dynamic Programming）: 是指在重复子问题的情况下，用空间换取时间，从而减少计算量的方法。
- 无后效性(No backtrack): 在动态规划中，当子问题的解不受到已知子问题的影响时，称该问题具有无后效性。例如，当采用动态规划求解最长公共子序列问题时，某些子问题只与前几个字符相关，这样就不需要回溯。
- 有界性(Bounded): 当动态规划问题的子问题独立于其他子问题时，称该问题具有有界性。即每个子问题只能依赖于小范围内的变量。
- 滚动数组(Sliding Window Technique): 滑动窗口法是一种优化动态规划的有效手段。该方法允许我们将递归式存储为滚动数组，而不是创建完整的二维表格。
- 备忘录法(Memoization Technique): 将函数的返回值缓存起来，使得同样的参数值不需要再次计算。
- 分治策略(Divide and Conquer Strategy): 动态规划也可以采用分治策略，先拆分子问题，然后逐个求解。
## 术语
- $dp[i][j]$表示$i$天后第$j$种方式的收益。$i$取值从1到$n$，$j$取值从1到$k$。
- $state[i][j]$表示机器人在第$i$天持有硬币，并且已经开出了第$j$条消息后的状态。$i$取值从1到$n$，$j$取值从1到$m$。
- $coins[j]$表示第$j$种硬币面值的数量。
- $ways[i][j]$表示机器人在第$i$天持有硬币，并且已经开出了第$j$条消息后，选择第$i-1$天所有方式的组合中，第$j$种方式的收益的最大值。$i$取值从1到$n$，$j$取值从1到$m$。
- $\pi_{i-1}(j)$表示机器人在第$i$天持有硬币，并且已经开出了第$j$条消息后，选择第$i-1$天所有方式的组合中，第$j$种方式的收益的最大值对应的路径。$\pi_{i-1}$是一个二维数组，$i$取值从1到$n$，$j$取值从1到$m$。
- $p_j$表示第$j$条消息是否要送达。$1$表示要送达，$0$表示不要送达。
- $t_i$表示第$i$天发送第$j$条消息的时间。
- $T_i$表示第$i$天的时间。
- $\sum^{\infty}_j c_j t_i = T_i$表示每天只会发送一次。
- $C^{max}$表示最大储值金额。
