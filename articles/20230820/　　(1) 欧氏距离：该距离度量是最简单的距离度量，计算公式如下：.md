# 1.简介
  
:欧氏距离（Euclidean Distance）是一种常用的距离度量方法。它测量两个向量（或点）之间的“真实距离”，即两个点在多维空间中真正地形成的直线距离。
# 2.欧氏距离的几何意义:欧氏距离是指两点之间直线距离的一种度量方式，由毕达哥拉斯提出，属于度量的范畴。其数值表示的是对象到某点的“最短距离”。欧氏距离也可以用来衡量对象之间的差异，尤其是空间中的点、直线和曲面。当点、直线、曲面三者之间存在某种联系时，可以利用欧氏距离来测量这种联系。
# 3.欧氏距离的定义及性质：设点P（x1，y1），点Q（x2，y2）的坐标分别是x1, y1, x2, y2，则欧氏距离d(P, Q)可表示为：
# $$d(P, Q)=\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$$
# 欧氏距离是一个反映相似性的距离度量方式，当且仅当两点距离为零时，它们是同一个点。欧氏距离具有以下几个重要的性质：
# - 非负性：欧氏距离一定是非负的，即d(P, P)=0, d(P, Q)>0.
# - 对称性：对于任意的点P和Q，d(P, Q)=d(Q, P)。
# - 三角不等式：对于任意的点P、Q和R，如果d(P, R)≤d(P, Q)+d(Q, R)，则必有d(P, R)=d(P, Q),d(Q, R)=d(R, P).
# - 距离的三角性质：对于任意的点P、Q和R，如果d(P, Q)=d(P, R)+d(R, Q)，则有d(Q, R)=0。
# 根据欧氏距离的定义，我们可以得到关于欧氏距离的另一些直观理解。比如说，一条直线的斜率就是两个端点距离的比值。而一条线段的长度等于欧氏距离。因此，欧氏距离提供了一种直观的方法来衡量两个对象之间的距离关系，并赋予它们不同的含义。例如，在三维空间中，欧氏距离可以用于衡量三维物体之间的距离；在二维图像处理中，欧氏距离可以用来描述点之间的像素距离；在高维空间中，欧氏距离又可以用于衡量多样化的点之间的距离。
# (2) 欧氏距离计算代码实现：

```python
import math
def euclideanDistance(pointA, pointB):
    return math.sqrt((pointA[0]-pointB[0])**2 + (pointA[1]-pointB[1])**2)
    
pointA = [0, 0]   # point A coordinates (x1, y1) 
pointB = [3, 4]   # point B coordinates (x2, y2) 

distance = euclideanDistance(pointA, pointB)

print("The Euclidean distance between", pointA, "and", pointB, "is:", distance)    # Output: The Euclidean distance between [0, 0] and [3, 4] is: 5.0

```
