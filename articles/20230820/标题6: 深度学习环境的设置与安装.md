# 1.简介
  

深度学习的发展离不开开源框架、算法库和平台支持，在不同场景下，深度学习环境的搭建又经历了从个人电脑到服务器集群甚至云平台的不同流程。不同的深度学习任务对硬件的要求也不尽相同，因此我们应根据实际需求合理安排深度学习环境配置。本文将介绍如何设置并安装常用深度学习环境。
# 2.主要内容
## （一）安装CUDA
首先，要确认自己的显卡型号是否支持CUDA。可以通过在运行应用程序的时候，搜索"CUDA版本"检查结果是否显示出来。
## （二）安装cuDNN
随着深度学习的兴起，越来越多的应用需要使用GPU进行运算加速。但是，现阶段很多GPU厂商并没有向所有用户提供cuDNN开发包，这就使得很多用户无法运行需要cuDNN的深度学习程序。这里，我们通过源码编译的方式安装cuDNN。
首先，从NVIDIA官网下载cuDNN安装包，选择适用于对应版本的CUDA和操作系统的压缩包。然后，把压缩包解压到一个文件夹中。比如解压后的目录结构如下图所示：
接着，进入该文件夹，执行`build_win.bat`。这时，会出现一些菜单，选择`x64`，再选择`Release`，最后点击`Build cuDNN`。等待编译完成，出现“BUILD COMPLETE”的提示时，就可以关闭安装包。
## （三）安装TensorFlow
其次，安装Python语言环境。推荐安装Anaconda。Anaconda是一个开源的Python发行版本，基于Python3.7。可以直接下载安装包安装Anaconda。Anaconda安装成功后，默认自带Jupyter Notebook，是一个方便交互式编程的工具。
最后，将下载的安装包放在指定位置。打开命令行窗口，切换到解压安装包的文件夹中，输入以下命令：
```bash
pip install tensorflow-cpu*.whl # 安装TensorFlow CPU版本
```
等待安装完成即可。
## （四）安装PyTorch
其次，安装Python语言环境。同样，推荐安装Anaconda。安装完成后，默认自带Jupyter Notebook。
最后，将下载的安装包放在指定位置。打开命令行窗口，切换到解压安装包的文件夹中，输入以下命令：
```bash
pip install torch==1.7.0+cu110 torchvision==0.8.1+cu110 -f https://download.pytorch.org/whl/torch_stable.html # 安装GPU版本的PyTorch
```
等候安装完成即可。
## （五）安装MXNet
其次，安装Python语言环境。推荐安装Anaconda。安装完成后，默认自带Jupyter Notebook。
最后，将下载的安装包放在指定位置。打开命令行窗口，切换到解压安装包的文件夹中，输入以下命令：
```bash
pip install mxnet
```
等待安装完成即可。
## （六）安装其他依赖包
除了深度学习框架本身所需的依赖包外，还有一些其它依赖包需要安装才能运行深度学习模型。包括但不限于：
* Matplotlib
* NumPy
* Scikit-learn
* Pandas
* OpenCV

这些依赖包的安装方法与上述框架类似，可以使用pip命令进行安装。例如，Matplotlib的安装方法如下：
```bash
pip install matplotlib
```
