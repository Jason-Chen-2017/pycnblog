# 1.简介
  

自从上个世纪90年代末开始，物联网（IoT）已经成为越来越多人的关注热点。随着物联网的崛起，各大公司纷纷涌现，比如高通、三星、亚马逊等，它们通过提供各类硬件和服务，让消费者更方便地享受到物联网带来的便利。虽然目前市场上物联网相关公司相当丰富，但由于国内企业的垄断性格，在物联网领域仍然处于弱势地位。因此，为了获得更多的竞争优势，创新型的互联网企业——Dell应该选择将自己的产品和服务整合到物联网平台中，推出独有的物联网解决方案。

本文主要讨论以下两个方面：
1. Dell如何选择适合自己的IoT公司；
2. 为何要选择Dell作为自己的IoT公司？

# 2.背景介绍
## 2.1 IoT(Internet of Things)
物联网（Internet of Things，简称IoT）是一种通过互联网连接物品、设备和服务的网络，使得这些物品、设备和服务能够互联互通、交流共享信息、进行远程控制的技术。物联网可用于监测和管理城市中的居民、工厂的生产过程、环境中的污染源头、机器或产品的运行状态，甚至可以用来控制住宅的空调系统，甚至智能手机。

## 2.2 物联网行业结构
目前，物联网行业共分为三个层次：
1. 基础设施层：由供应商和采购商提供物联网硬件、固件和服务。如阿里巴巴的LinkBee、Intel的SensorGrid；
2. 服务层：提供物联网平台和解决方案。如IBM Watson、谷歌的Google Assistant；
3. 应用层：基于物联网的各种应用场景及落地案例。如智能城市、智能农业、智慧校园等。

## 2.3 目前的局限性
虽然物联网的发展势头很快，但由于国内企业的垄断性格，尤其是在智能硬件、软件、服务领域，它仍然处于弱势地位。随着美国的退出，智能手机和智能穿戴设备占据了越来越多的市场份额，但智能手表、智能电视、空调等传统设备仍然支配着市场主体。这就使得创新型的互联网企业——Dell应运而生。

# 3.基本概念术语说明
## 3.1 概念
- 数据中心：数据中心是指存储、处理数据的位置，一般在大城市或具有一定规模的地区，属于IDC（Internet Data Center）。
- 边缘计算：边缘计算是指云端的数据分析能力，通过边缘计算，用户可以快速对海量数据进行实时处理，提升响应速度。
- 物联网平台：物联网平台是指物联网的基础设施、服务和应用，如阿里云、腾讯云、百度云等。
- 物联网终端设备：物联网终端设备是指连接到物联网平台的终端设备，比如摄像头、微波炉、智能电视、智能手环、智能电动车、门锁、消防报警器、门禁卡、自动售货机等。
- 物联网智能应用：物联网智能应用是指基于物联网的各种应用场景及落地案例，如智能城市、智能农业、智慧校园等。

## 3.2 术语
- IaaS (Infrastructure as a Service):基础设施即服务，通过云服务商提供租户管理、虚拟化、网络、存储、安全、备份等基础设施能力的一种服务模式。
- PaaS (Platform as a Service):平台即服务，通过云服务商提供租户部署、开发、测试、维护Web应用和移动应用所需的一系列基础设施、中间件、工具等资源，同时也包括了一组完整的服务功能框架。
- SaaS (Software as a Service):软件即服务，通过云服务商提供租户使用的应用程序，包括办公软件、产品ivity工具、协同软件、CRM、SCM等等。
- BaaS (Backend as a Service):后端即服务，即把后端服务打包成一个API接口，并通过云服务提供商提供给客户调用，以此来降低IT研发与应用之间的绑定关系。
- FaaS (Function as a Service):函数即服务，指的是一种将应用程序逻辑或功能封装成可重用的模块的方式，帮助客户减少重复开发、提高开发效率、缩短开发周期、节省IT资源等。
- Edge computing：边缘计算，是利用服务器集群资源的边缘端（离设备近一些的地方）来执行任务的一种计算机技术。
- IoT platform：物联网平台，是由基础设施、协议、平台工具、应用开发者和最终用户组成的一个开放的生态系统。
- End device：终端设备，是指连接到物联网平台的终端设备，比如摄像头、微波炉、智能电视、智能手环、智能电动车、门锁、消防报警器、门禁卡、自动售货机等。
- Smart app：智能应用，是基于物联网的各种应用场景及落地案例，如智能城市、智能农业、智慧校园等。

## 4.1 选择IoT公司的标准
根据公司的服务范围、解决方案、研发团队、业务模式、产品策略等因素综合评判，确定该公司是否适合作为Dell的IoT公司。

**首先要考虑公司是否具备以下条件：**

1. 物联网公司：公司提供的物联网服务或产品一定要与物联网有关。比如，能够建立物联网网关、路由器、设备管理、数据传输、运维、数据采集等全套解决方案。
2. 有一定规模：公司的规模不能太小，不然可能会被边缘节点所淹没。
3. 产品力：公司的产品要能够产生实质性价值，而不是仅仅是套餐。比如，公司需要拥有完整的物联网解决方案。
4. 技术实力：公司需要有良好的科技实力，包括知识产权、工艺水平、工程实力、产品开发能力等。
5. 全球化布局：公司所在的国家或地区要具备较强的全球化发展能力。

**其次还要考虑公司的业务模式、合作伙伴以及用户群体等因素：**

1. 服务范围：公司的物联网产品应该覆盖领域广，才能吸引广大的用户群体。
2. 用户群体：物联网终端设备的消费者会越来越多，所以公司需要做好相应的宣传活动，吸引大量潜在用户。
3. 发展前景：公司的市场前景是否坚定，能否满足用户对物联网产品和服务的需求？

## 4.2 Dell选型流程
根据Dell自己在IoT领域的发展路线图，进行如下的选型流程：

1. 在云服务市场进行选择：首先对云服务市场了解一下，找到符合Dell业务发展目标和产品定位的公司，如AWS、Azure、Google Cloud Platform、AliCloud等。
2. 根据产品特色制定IoT服务策略：了解所选产品或公司的业务模式、目标客户群、产品特色及竞争对手，并制定合适的IoT服务策略。比如，Dell的IoT服务策略包括四个方面：一是设备制造：包括采购和自主开发产品；二是云服务：包括云服务商的选择和选型；三是服务支持：包括技术支持、售后服务等；四是联盟分享：包括宣传与宣扬，促进合作共赢。
3. 完善服务架构和技术栈：完善服务架构和技术栈，包括设备驱动程序、数据处理中心、数据仓库、网关中心、应用服务器、BI工具、安防、数字许可系统、业务数据集成等。
4. 研究并探索边缘计算：了解和尝试边缘计算，包括边缘节点计算资源、边缘设备通信、任务管理、机器学习、数据分析等。
5. 提升内部软实力：提升内部软实力，包括教育培训、薪酬福利、销售渠道、研发组织等方面。
6. 持续优化营销策略：持续优化营销策略，包括广告、展示、沟通、口碑、产品宣传等。

Q:为什么要选Dell作为自己的IoT公司？

A：根据Dell首席执行官兼CEO黄昱磊告诉笔者，Dell目前主要业务在于提供硬件、固件、云计算和软件等基础设施服务，其中硬件和固件服务在国内还没有完全占领市场，所以他们想用自己的专业技术、云计算和IT经验来改善这一点。另一方面，他们的解决方案有很多是和智能设备相关的，可以帮助消费者实现各种智能应用。Dell作为硬件、云计算和软件提供商，他们有足够的资源和能力来开发出符合国际标准的物联网解决方案。

Q:Dell如何选择适合自己的IoT公司？

A：Dell的选型流程中指出，首先要考虑公司是否具备以下条件：1. 物联网公司，2. 有一定规模，3. 产品力，4. 技术实力，5. 全球化布局。之后还要考虑公司的业务模式、合作伙伴以及用户群体等因素。Dell建议首先在云服务市场进行选择，在该市场中找到符合Dell业务发展目标和产品定位的公司，如AWS、Azure、Google Cloud Platform、AliCloud等。其次，通过研究公司的业务模式、用户群体、竞争对手，制定合适的IoT服务策略。之后，Dell可以选择在IT部门下设立IoT部门，专注于物联网的研发和部署工作。此外，Dell可以在各个行业中引进智能设备，试点部署在客户的家中，获取用户的反馈，并不断优化产品。最后，Dell可以不断投资和扩张自己的云服务商，争取更多的合作伙伴资源，共同打造出更强大的IoT生态系统。

Q:什么是边缘计算？如何实现？

A：边缘计算是一种利用服务器集群资源的边缘端（离设备近一些的地方）来执行任务的一种计算机技术。一般情况下，边缘计算由设备制造商、服务提供商或者系统集成商提供。边缘计算应用在智能城市、智慧校园、智慧农业等领域，通过在分布在多个地点的边缘节点上执行计算，避免传送数据和处理请求的网络传输延迟，提高响应时间和带宽利用率，并可降低本地设备的功耗。

Dell可以选择与现有的云服务商合作，完成边缘计算的方案设计、开发和测试，采用开源工具实现边缘节点的部署和管理。
