#### 5.3.1 用户登录与权限控制
```java
@PostMapping("/login")
public Result login(@RequestBody User user) {
    String username = user.getUsername();
    String password = user.getPassword();
    
    // 数据库查询用户信息
    User dbUser = userService.getByUsername(username);
    
    // 密码加盐哈希对比
    String saltPassword = SaltUtil.getSaltPassword(password, dbUser.getSalt());
    if (!saltPassword.equals(dbUser.getPassword())) {
        return Result.error("用户名或密码错误");
    }
    
    // 生成JWT令牌
    String token = JwtUtil.generateToken(dbUser);
    
    return Result.ok().put("token", token);
}
```

#### 5.3.3 考试管理模块
```java
@PostMapping("/exam/generate")
public Result generateExam(@RequestBody ExamConfigVO examConfigVO) {
    // 随机抽题组卷
    List<Question> questions = questionService.randomSelectQuestions(examConfigVO);
    
    // 生成试卷并保存到数据库
    String examId = examService.generateExam(questions, examConfigVO);
    
    return Result.ok().put("examId", examId);
}

@PostMapping("/exam/submit")
public Result submitExam(@RequestBody ExamSubmitVO examSubmitVO) {
    // 交卷计算得分
    ExamResult examResult = examService.judgeExam(examSubmitVO);
    
    // 保存考试结果到数据库
    examResultService.save(examResult);
    
    return Result.ok().put("score", examResult.getScore());
}
```
