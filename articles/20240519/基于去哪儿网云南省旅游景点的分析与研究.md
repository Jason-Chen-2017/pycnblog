## 4. 数学模型和公式详细讲解举例说明

### 4.1 TF-IDF

TF-IDF（Term Frequency-Inverse Document Frequency）是一种用于评估词语在文档集合中重要性的统计方法。

* **TF：** 词语在文档中出现的频率。
* **IDF：** 词语在文档集合中的逆文档频率，即包含该词语的文档数量的倒数的对数。

$$
TF-IDF(t, d) = TF(t, d) \times IDF(t)
$$

**举例说明：** 假设文档集合中有100篇文档，其中10篇文档包含词语“旅游”。则“旅游”的IDF为：

$$
IDF(旅游) = log(100 / 10) = 1
$$

### 4.2 余弦相似度

余弦相似度用于衡量两个向量之间的相似程度。

$$
similarity = \frac{\vec{a} \cdot \vec{b}}{||\vec{a}|| \times ||\vec{b}||}
$$

**举例说明：** 假设有两个景点向量：

* 景点A： [0.8, 0.6]
* 景点B： [0.7, 0.7]

则景点A和景点B的余弦相似度为：

$$
similarity = \frac{0.8 \times 0.7 + 0.6 \times 0.7}{\sqrt{0.8^2 + 0.6^2} \times \sqrt{0.7^2 + 0.7^2}} = 0.96
$$

## 5. 项目实践：代码实例和详细解释说明

### 5.1 数据获取

```python
import requests
from bs4 import BeautifulSoup

# 设置请求头
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/58.0.3029.110 Safari/537.3'
}

# 发送请求
response = requests.get('https://www.qunar.com/dest/yn/', headers=headers)

# 解析网页内容
soup = BeautifulSoup(response.content, 'html.parser')

# 提取景点信息
attractions = []
for item in soup.find_all('li', class_='item'):
    name = item.find('h2').text.strip()
    location = item.find('p', class_='des').text.strip()
    attractions.append({'name': name, 'location': location})

# 打印景点信息
print(attractions)
```

### 5.2 数据分析

```python
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.cluster import KMeans

# 读取景点数据
df = pd.read_csv('attractions.csv')

# 使用TF-IDF将景点描述转换为向量
vectorizer = TfidfVectorizer()
vectors = vectorizer.fit_transform(df['description'])

# 使用K-Means算法将景点聚类
kmeans = KMeans(n_clusters=5)
kmeans.fit(vectors)

# 打印聚类结果
print(kmeans.labels_)
```
