## 4.1 智能合约的代码实例
以下是一个简单的智能合约的代码实例：

```solidity
pragma solidity ^0.5.0;

contract SocialInsurance {
    uint public balance;

    event Transfer(address indexed from, address indexed to, uint value);

    function deposit() public payable {
        balance += msg.value;
    }

    function withdraw(uint amount) public {
        require(balance >= amount);
        balance -= amount;
        emit Transfer(msg.sender, address(0), amount);
    }
}
```

在这个代码实例中，我们定义了一个名为`SocialInsurance`的智能合约。智能合约有一个`balance`状态变量，用于存储智能合约的余额。智能合约还定义了一个`Transfer`事件，用于记录转账操作。

智能合约还定义了两个函数：`deposit`和`withdraw`。`deposit`函数允许用户向智能合约发送ETH，并将其添加到智能合约的余额中。`withdraw`函数允许用户从智能合约中提取金额，并记录转账操作。
