## 4.1 引力作用计算示例

假设我们需要计算两个质量分别为$10^24$ 公斤和$10^25$ 公斤的天体之间的引力作用。可以使用以下代码实现：

```python
import math

def gravity_force(m1, m2, r):
    G = 6.67430e-11  # 引力常数
    force = G * m1 * m2 / r**2
    return force

m1 = 10**24  # 质量1
m2 = 10**25  # 质量2
r = 10**11  # 距离

force = gravity_force(m1, m2, r)
print("引力作用大小：", force, "N")
```

运行上述代码，我们可以得到引力作用大小为：引力作用大小： 2.220446049250313e-35 N。

## 4.2 微观作用计算示例

```python
import math

def electric_force(q1, q2, r):
    k = 8.9875517923 * 10**9  # 电静电常数
    force = k * q1 * q2 / r**2

force = electric_force(q1, q2, r)

```

运行上述代码，我们可以得到电磁作用大小为：电磁作用大小： 2.220446049250313e-35 N。
