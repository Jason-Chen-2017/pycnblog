```python
import numpy as np

def high_phase_nonlinear_encrypt(data, key):
    nonlinear_func = np.vectorize(lambda x: x**3)
    encrypted_data = key * nonlinear_func(data)
    return encrypted_data

def high_phase_nonlinear_decrypt(encrypted_data, key):
    inverse_func = np.vectorize(lambda x: x**(-3))
    decrypted_data = key * inverse_func(encrypted_data)
    return decrypted_data

data = np.array([1, 2, 3, 4, 5])
key = 10
encrypted_data = high_phase_nonlinear_encrypt(data, key)
decrypted_data = high_phase_nonlinear_decrypt(encrypted_data, key)

print("Original data:", data)
print("Encrypted data:", encrypted_data)
print("Decrypted data:", decrypted_data)
```

在这个例子中，我们使用了立方函数作为高阶非线性函数，密钥为10。通过运算得到加密后的数据，然后使用逆函数得到原始数据。
