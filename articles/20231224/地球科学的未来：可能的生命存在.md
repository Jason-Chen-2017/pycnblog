## 2.地球科学模型的构建和优化

### 2.1 模型构建

```python
import scipy.optimize as opt

# 定义目标函数
def objective_function(params):
    temperature = params[0]
    pressure = params[1]
    return (data['temperature'] - temperature) ** 2 + (data['pressure'] - pressure) ** 2

# 初始化参数
initial_params = [np.mean(data['temperature']), np.mean(data['pressure'])]

# 优化参数
optimized_params = opt.minimize(objective_function, initial_params)

# 得到最优参数
temperature, pressure = optimized_params.x
```

### 2.2 模型优化

```python
# 使用随机梯度下降优化模型参数
def gradient_descent(learning_rate, num_iterations, initial_params):
    params = initial_params
    for _ in range(num_iterations):
        gradient = np.array([2 * (data['temperature'] - params[0]), 2 * (data['pressure'] - params[1])])
        params = params - learning_rate * gradient
    return params

# 优化参数
optimized_params = gradient_descent(learning_rate=0.01, num_iterations=1000, initial_params=np.array([np.mean(data['temperature']), np.mean(data['pressure'])]))

# 得到最优参数
temperature, pressure = optimized_params
```
