## 4.1 决策平面的代码实例

以下是一个简单的决策平面代码实例，用于表示和分析一项商业决策问题：

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建决策模型
G = nx.DiGraph()

# 添加决策节点
G.add_node("市场调查", type="决策节点")
G.add_node("产品设计", type="决策节点")
G.add_node("生产", type="决策节点")
G.add_node("销售", type="决策节点")

# 添加决策边缘
G.add_edge("市场调查", "产品设计", type="决策边缘")
G.add_edge("产品设计", "生产", type="决策边缘")
G.add_edge("生产", "销售", type="决策边缘")

# 绘制决策模型
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True, node_color=["red", "green", "blue", "yellow"])
plt.show()
```

在这个代码实例中，我们使用`networkx`库来构建和绘制决策模型。首先，我们创建了一个有向图`G`，并添加了四个决策节点：市场调查、产品设计、生产和销售。然后，我们添加了三条决策边缘，表示决策过程中的关系和依赖关系。最后，我们使用`matplotlib`库来绘制决策模型。

## 4.2 人工智能的代码实例

以下是一个简单的人工智能代码实例，用于进行文本分类任务：

```python
import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import Pipeline
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 训练数据和测试数据
X_train = ["这是一篇正面评论", "这是一篇负面评论", "这是一篇正面评论"]
y_train = [1, 0, 1]
X_test = ["这是一篇正面评论", "这是一篇负面评论"]
y_test = [1, 0]

# 构建文本分类模型
pipeline = Pipeline([
    ("vectorizer", CountVectorizer()),
    ("classifier", MultinomialNB()),
])

# 训练模型
pipeline.fit(X_train, y_train)

# 预测
y_pred = pipeline.predict(X_test)

# 评估模型
accuracy = accuracy_score(y_test, y_pred)
print("准确率：", accuracy)
```

在这个代码实例中，我们使用`sklearn`库来构建和训练一个文本分类模型。首先，我们创建了训练数据和测试数据，其中包含了正面和负面评论。然后，我们使用`Pipeline`类来构建一个文本分类模型，该模型包括`CountVectorizer`和`MultinomialNB`两个组件。接着，我们使用`fit`方法来训练模型，并使用`predict`方法来进行预测。最后，我们使用`accuracy_score`方法来评估模型的准确率。
