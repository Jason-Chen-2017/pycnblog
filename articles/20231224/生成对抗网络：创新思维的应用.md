                 

# 1.背景介绍

生成对抗网络（Generative Adversarial Networks，GANs）是一种深度学习的方法，它通过两个网络进行训练，一个生成网络（Generator）和一个判别网络（Discriminator）。这两个网络相互作用，生成网络试图生成逼真的样本，判别网络则试图区分这些生成的样本与真实的样本。这种竞争关系使得生成网络逐渐学会生成更逼真的样本，判别网络也逐渐更好地区分真实与假假的样本。
