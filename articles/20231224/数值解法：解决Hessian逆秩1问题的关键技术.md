# 2.核心概念与联系

为了更好地理解Hessian逆秩1问题，我们首先需要了解一些基本概念。

## 2.1 Hessian矩阵

Hessian矩阵是二阶导数矩阵，用于描述函数在某一点的曲线性。对于一个二次函数f(x) = (1/2)x^T H x + c，其二阶导数矩阵为Hessian矩阵。Hessian矩阵可以用来分析函数的最小、最大值以及梯度的方向。

## 2.2 秩

秩是线性代数中的一个重要概念，用于描述一个向量空间或矩阵的“稳定性”。秩是指向量空间中线性无关向量的最大数量。如果一个矩阵的秩小于它的纬度，则称该矩阵为逆秩不满的矩阵。

$$

2. 选择一部分奇异值以构建低秩矩阵：

$$

    return H + np.dot(H, H.T) / mu
