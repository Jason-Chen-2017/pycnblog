# 2. 核心概念与联系

泛函式分析的核心概念主要包括：

1. 函数空间：函数空间是指由一组满足某种条件的函数组成的集合，例如Lp空间、Sobolev空间等。
2. 泛函：泛函是将函数空间中的函数映射到数字空间中的映射。
3. 勾股不等式：勾股不等式是泛函式分析中的一个基本性质，它表示了函数的L2范数之和的最大值。
4. 泛函的性质：泛函可以具有线性性、连续性、凸性等性质。
5. 泛函的应用：泛函式分析在微积分、函数分析、线性算法、优化等领域中发挥着重要作用。
