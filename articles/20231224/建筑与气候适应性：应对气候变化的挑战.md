```python
def optimize_insulation(building, insulation_coefficient):
    original_heat_loss = building.calculate_heat_loss()
    new_insulation_coefficient = building.insulation_coefficient + insulation_coefficient
    building.insulation_coefficient = new_insulation_coefficient
    new_heat_loss = building.calculate_heat_loss()
    energy_saving = original_heat_loss - new_heat_loss
    return energy_saving
```

在这个代码实例中，我们首先定义了一个`optimize_insulation`函数，它接受一个`building`对象和一个`insulation_coefficient`值作为输入。这个函数的目的是通过调整建筑物的隔热性能来降低能源消耗。

首先，我们计算原始的热损失（`original_heat_loss`）。然后，我们将建筑物的隔热性能增加`insulation_coefficient`值，并计算新的热损失（`new_heat_loss`）。最后，我们计算能源节省量（`energy_saving`），并返回这个值。
