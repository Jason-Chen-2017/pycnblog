
```python

plt.ylabel('速度')
plt.title('实时监控')
plt.show()
```

### 4.1.3 预警

我们可以使用Python的scikit-learn库来进行预警：

```python
from sklearn.ensemble import RandomForestClassifier

# 训练预警模型
model = RandomForestClassifier()
model.fit(traffic_data[['time', 'speed', 'direction']], traffic_data['warning'])

# 预警

```
