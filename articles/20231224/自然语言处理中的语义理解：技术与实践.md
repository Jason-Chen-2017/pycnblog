### 4.1 词嵌入（Word Embedding）

我们可以使用Python的Gensim库来实现词嵌入。以下是一个简单的例子：

```python
from gensim.models import Word2Vec

# 训练一个简单的Word2Vec模型
model = Word2Vec([['apple', 'fruit'], ['banana', 'fruit'], ['apple', 'tasty']], min_count=1)

# 查看词嵌入
print(model.wv['apple'])
print(model.wv['banana'])
```

在这个例子中，我们训练了一个简单的Word2Vec模型，并查看了`apple`和`banana`的词嵌入。可以看到，`apple`和`fruit`之间的相似性得分较高，表示它们具有相似的语义信息。

### 4.2 语义角色标注（Semantic Role Labeling）

我们可以使用Python的spaCy库来实现语义角色标注。以下是一个简单的例子：

```python
import spacy

# 加载英文模型
nlp = spacy.load('en_core_web_sm')

# 文本
text = "John gave Mary a book."

# 进行语义角色标注
doc = nlp(text)

# 遍历文档中的实体和角色
for token in doc:
    print(f"{token.text}\t{token.dep_}\t{token.head.text}\t{token.head.dep_}")
```

在这个例子中，我们使用spaCy库进行英文文本的语义角色标注。可以看到，spaCy库可以识别出文本中的实体和角色，并将它们关联起来。

### 4.3 语义关系抽取（Semantic Relation Extraction）

我们可以使用Python的spaCy库来实现语义关系抽取。以下是一个简单的例子：

```python
import spacy

# 加载英文模型
nlp = spacy.load('en_core_web_sm')

# 文本
text = "John gave Mary a book."

# 进行语义关系抽取
doc = nlp(text)

# 遍历文档中的实体和关系
for ent1 in doc.ents:
    for ent2 in doc.ents:
        if ent1.text != ent2.text and ent1.head == ent2.head:
            print(f"{ent1.text}->{ent2.text}")
```

在这个例子中，我们使用spaCy库进行英文文本的语义关系抽取。可以看到，spaCy库可以识别出文本中的实体和关系，并将它们关联起来。

### 6.16 什么是过拟合？

过拟合是指模型在训练数据上的表现非常好，但在新的数据上表现不佳的现象。过拟合通常是由于模型过于复杂，导致对训练数据的拟合过于敏感，从而对新数据的泛化能力不利。

正则化是一种用于防止过拟合的方法，它通过在损失函数中添加一个惩罚项，限制模型的复杂度。正则化可以帮助模型在训练数据上保持良好的表现，同时在新数据上保持良好的泛化能力。

交叉验证是一种用于评估模型性能的方法，它涉及将数据分为多个子集，然后将模型训练和验证在不同子集上。通过交叉验证，可以更准确地评估模型的性能，并避免过拟合的风险。
