                 

# 1.背景介绍

边缘计算（Edge Computing）是一种新兴的计算模式，它将数据处理和分析功能从中心化的云计算环境移动到边缘设备（如智能手机、IoT设备等）。这种模式可以降低网络延迟、减轻云计算负载，并提高数据处理效率。然而，边缘计算也面临着严峻的安全和隐私挑战。

在边缘计算环境中，数据通常在多个设备之间传输和共享，这使得数据在传输过程中可能会泄露或被窃取。此外，边缘设备可能会受到攻击，攻击者可以篡改或伪造数据，从而影响系统的安全性和可靠性。因此，保障边缘计算环境的数据安全和隐私成为了一个重要的研究问题。

## 1.1 边缘计算的安全与隐私挑战

边缘计算的安全与隐私挑战主要包括以下几个方面：

1. 数据在传输过程中的泄露风险。在边缘计算环境中，数据通常需要在多个设备之间传输，这使得数据在传输过程中可能会泄露或被窃取。

2. 边缘设备的安全性。边缘设备可能会受到攻击，攻击者可以篡改或伪造数据，从而影响系统的安全性和可靠性。

3. 隐私保护。边缘计算环境中的数据通常包含敏感信息，如个人信息、健康数据等，因此需要保护数据的隐私。
