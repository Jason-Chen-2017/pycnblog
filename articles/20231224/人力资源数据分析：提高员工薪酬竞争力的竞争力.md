# 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的Python代码实例，展示如何使用pandas库进行人力资源数据分析。

```python
import pandas as pd

# 加载员工薪酬数据
data = pd.read_csv('employee_salary.csv')

# 数据预处理
data['salary'] = data['salary'].fillna(data['salary'].mean())

# 数据分析
average_salary = data['salary'].mean()
print('员工平均薪酬：', average_salary)

correlation = data[['salary', 'experience', 'education']].corr()
print('薪酬与经验年限的相关关系：', correlation['salary']['experience'])
```

在这个代码实例中，我们首先使用pandas库加载员工薪酬数据。然后进行数据预处理，填充缺失的薪酬数据。接着，我们计算员工平均薪酬，并计算薪酬与经验年限之间的相关关系。
