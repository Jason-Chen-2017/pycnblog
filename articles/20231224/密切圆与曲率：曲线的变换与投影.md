### 2.1 变换

变换是指在不同坐标系下表示相同曲线的过程。例如，我们可以将一个曲线从Cartesian坐标系转换到极坐标系或鸡尾酒坐标系等。变换可以是线性的，例如坐标系的转换，也可以是非线性的，例如在绘图过程中进行的缩放、平移、旋转等操作。

### 4.3 B-spline

假设我们有一个B-spline曲线的控制点：

\end{bmatrix}
$$

```python
import numpy as np
import matplotlib.pyplot as plt

def B(t, n, m):
    beta = np.ones(m + 1)
    for j in range(m):
        beta[j] *= (t - j)
        beta[j + 1] /= (j + 1)
    return np.dot(beta, np.eye(m + 1)[np.arange(m), :])

def curve(t, p, n):
    return np.sum([B(t, n, i) * p[i] for i in range(len(p))])

t = np.linspace(0, 1, 100)
p = np.array([[0, 0], [1, 0], [2, 2], [3, 2]])
n = 2

r = curve(t, p, n)
plt.plot(r[:, 0], r[:, 1])
plt.show()
```
