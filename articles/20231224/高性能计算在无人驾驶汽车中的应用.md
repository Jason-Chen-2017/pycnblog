## 4.2 机器学习代码实例

以下是一个简单的支持向量机代码实例，用于对数据进行训练和预测：

```python
import numpy as np
from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC

# 加载数据
iris = datasets.load_iris()
X = iris.data
y = iris.target

# 数据预处理
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 数据分割
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# 训练支持向量机
svm = SVC(kernel='linear')
svm.fit(X_train, y_train)

# 进行预测
y_pred = svm.predict(X_test)

# 评估模型
accuracy = np.mean(y_pred == y_test)
print('Accuracy: %.2f' % accuracy)
```

在这个代码实例中，我们首先使用 `sklearn` 库加载鸢尾花数据集。然后，我们对数据进行标准化处理，并将其分割为训练集和测试集。接着，我们使用支持向量机（线性核）训练模型，并对测试集进行预测。最后，我们使用准确率评估模型的性能。

## 4.3 实时决策代码实例

```python

import numpy as np

# 读取车辆信息
car_info = np.array([[100, 0, 100, 50, 0], [200, 0, 200, 100, 0]])

# 计算车辆距离
distances = np.linalg.norm(car_info[:, 1:5], axis=1)

# 判断是否需要避障

    print('Need to avoid obstacle')
else:
    print('No need to avoid obstacle')
```

在这个代码实例中，我们首先读取车辆信息，包括位置和速度。然后，我们计算车辆之间的距离。最后，我们判断是否需要避障，如果车辆距离小于50米，则需要避障。
