from sklearn.metrics.pairwise import cosine_similarity

# 文本内容列表

# 将文本内容转换为向量

# 计算文本之间的相似度

### 4.2 用户推荐算法实例

我们可以使用Python的NetworkX库来实现一个社交网络构建算法。首先，我们需要创建一个有向图，然后将用户之间的关注关系添加到图中。最后，我们可以使用度中心性指标来计算用户的权重，并将权重最高的用户推荐给用户。

```python
import networkx as nx

# 创建一个有向图
G = nx.DiGraph()

# 添加用户节点
users = ['user1', 'user2', 'user3']
G.add_nodes_from(users)

# 添加用户之间的关注关系
follows = [('user1', 'user2'), ('user2', 'user3')]
G.add_edges_from(follows)

# 计算用户的度中心性
centralities = nx.degree_centrality(G)

# 将权重最高的用户推荐给用户
recommended_user = max(centralities, key=centralities.get)
print(recommended_user)
```
