# 2.核心概念与联系
在深入探讨受限玻尔兹曼机的稳定性问题之前，我们需要了解一些基本概念。

## 2.1 玻尔兹曼机
玻尔兹曼机是一种概率模型，用于描述系统中各个状态之间的概率分布。它的核心思想是，通过调整状态之间的概率分布，可以实现对不同类型的数据进行建模和分析。玻尔兹曼机广泛应用于机器学习和人工智能领域，如自然语言处理、图像识别等。

## 2.2 受限玻尔兹曼机
受限玻尔兹曼机是一种特殊类型的玻尔兹曼机，它的状态空间是有限的。受限玻尔兹曼机通常用于处理有限数据集和有限状态空间的问题，如文本分类、图像识别等。受限玻尔兹曼机的主要优点是它的计算复杂度相对较低，易于实现和训练。
