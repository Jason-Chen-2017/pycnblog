# 2.核心概念与联系
边界填充是一种图像处理技术，它的目的是在图像的边界处填充像素值，以便在进行图像分类时不会丢失边界信息。边界填充可以通过以下几种方法实现：

1. 平滑填充：在边界处填充与周围像素值最接近的颜色。
2. 平均填充：在边界处填充周围像素值的平均值。
3. 反射填充：在边界处填充与对称轴相对应的像素值。
4. 随机填充：在边界处随机选择像素值。

边界填充与图像分类紧密相连，因为图像分类的质量取决于输入图像的质量。如果图像边界信息丢失或不完整，那么图像分类的准确率将下降。因此，在进行图像分类时，需要对图像进行边界填充处理，以确保输入图像的质量。

以下是一个平均填充的具体实现：

1. 读取输入图像。
2. 检测图像边界。
3. 遍历边界像素，计算周围像素值的平均值。
4. 在边界像素处填充计算出的平均值。
5. 保存填充后的图像。

数学模型公式为：

$$
P_{fill}(x, y) = \frac{\sum_{i=-r}^{r}\sum_{j=-r}^{r} P(x + i, y + j) \cdot (1 - \frac{|i| + |j|}{2r + 1})}{\sum_{i=-r}^{r}\sum_{j=-r}^{r} (1 - \frac{|i| + |j|}{2r + 1})}
$$

其中，$P_{fill}(x, y)$ 表示填充后的像素值，$P(x + i, y + j)$ 表示原图像中的像素值，$r$ 表示填充半径。
