```python

# 读取航空航天大数据
data = pd.read_csv('airline_data.csv')

# 数据清洗和预处理

data = data[data['age'] > 18] # 筛选年龄大于18岁的乘客

# 数据分析

average_miles = data['miles'].mean()

print('平均年龄：', average_age)
print('平均里程：', average_miles)
```

在这个示例中，我们首先使用Pandas库读取了一个航空航天大数据的CSV文件。然后，我们对数据进行了清洗和预处理，包括删除缺失值和筛选年龄大于18岁的乘客。最后，我们对数据进行了分析，计算了平均年龄和平均里程。
