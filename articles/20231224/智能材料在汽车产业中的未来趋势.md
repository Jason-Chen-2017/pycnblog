# 2. 核心概念与联系

## 2.1 智能材料的定义

智能材料是一种具有自主反应能力和自适应性能的材料，它们可以根据环境的变化或外部的刺激，实时调整自身的性能特征。智能材料的主要特点包括：

1. 自主反应能力：智能材料可以根据环境的变化或外部的刺激，实时调整自身的性能特征。
2. 自适应性能：智能材料可以根据不同的应用场景，调整自身的性能特征。

## 2.2 智能材料与传统材料的区别

智能材料与传统材料的主要区别在于其性能特征的自主性和自适应性。传统材料的性能特征是固定的，不能根据环境的变化或外部的刺激实时调整。而智能材料的性能特征可以根据环境的变化或外部的刺激实时调整，使得它们在某些应用场景下具有更高的性能。

## 2.3 智能材料与传统材料的联系

智能材料与传统材料之间存在着密切的联系。智能材料可以看作是传统材料的一种改进和升级。通过对传统材料的研究和改进，人们可以为智能材料设计更高性能、更高效率的性能特征。
