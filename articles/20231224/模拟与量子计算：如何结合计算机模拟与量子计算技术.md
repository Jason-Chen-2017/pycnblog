## 4.1计算机模拟代码实例
以下是一个简单的计算机模拟代码实例，用于计算一个物理系统中的能量状态。

```python
import numpy as np

def energy_state(position, velocity, mass, potential_energy_function):
    kinetic_energy = 0.5 * mass * velocity ** 2
    potential_energy = potential_energy_function(position)
    total_energy = kinetic_energy + potential_energy
    return total_energy

position = np.array([1.0, 2.0, 3.0])
velocity = np.array([1.0, -1.0, 0.0])
mass = 1.0
potential_energy_function = lambda x: x ** 2

energy = energy_state(position, velocity, mass, potential_energy_function)
print("Energy:", energy)
```

```python
# 量子测量
qc.measure([0, 1], [0, 1])  # 对两个量子比特进行测量

# 执行量子计算
backend = Aer.get_backend('qasm_simulator')
qobj = assemble(qc)
result = backend.run(qobj).result()

# 结果分析
counts = result.get_counts()
plot_histogram(counts)
```
