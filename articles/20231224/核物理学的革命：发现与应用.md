                 

# 1.背景介绍

核物理学是一门研究核子（原子核）的物理学科。核物理学的研究内容涉及核子的性质、运动、相互作用和稳定性等方面。核物理学的研究成果对于原子能、核能等新型能源的发展具有重要意义。

## 1.1 核物理学的发展历程

核物理学的发展历程可以分为以下几个阶段：

1. 19世纪初的诞生时期：在19世纪初，物理学家开始研究原子的性质。1911年，杰里·艾米斯（Ernest Rutherford）通过金属膜散射实验发现了原子核的存在。

2. 核膜散射实验：在1911年，杰里·艾米斯通过对金属膜的散射实验发现了原子核的存在。他发现大多数散射粒子来自于原子核，而不是电子。这一发现为后续的核物理学研究奠定了基础。

3. 核能的发现：在1932年，伊兹诺·罗斯姆（Isidor Isaac Rabi）发现了核磁共振，这一发现为核能的测量提供了理论基础。

4. 核反应的发现：在1938年，乔治·艾米斯（George Gamow）、马尔科·朗道（Marcel-Jean Chenot de Broglie）和伯努利·伯努利（George Eugene Uhlenbeck）一起提出了一种新的理论，即核反应可以通过碰撞产生新的核子。

5. 核�ission的发现：在1942年，乔治·艾米斯和罗伯特·奥姆（Robert Oppenheimer）通过实验发现了核分裂现象，这一发现为核能的发展提供了理论基础。

6. 核融合的发现：在1950年代，乔治·艾米斯和罗伯特·奥姆通过实验发现了核融合现象，这一发现为核能的发展提供了理论基础。
