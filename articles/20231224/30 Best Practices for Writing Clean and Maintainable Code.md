## 4.2 代码示例

以下是一个简单的Python代码示例，展示了如何编写干净、可维护的代码：

```python
def add(a, b):
    """
    添加两个数字
    """
    return a + b

def subtract(a, b):
    """
    减去两个数字
    """
    return a - b

def multiply(a, b):
    """
    乘以两个数字
    """
    return a * b

def divide(a, b):
    """
    除以两个数字
    """
    if b == 0:
        raise ValueError("Cannot divide by zero")
    return a / b
```
