# 3.2 机器学习 算法原理
机器学习 的核心算法原理包括：

- 线性回归：线性回归是一种简单的机器学习 算法，它使用线性模型来预测连续变量。线性回归的数学模型如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \ldots + \beta_nx_n + \epsilon
$$

其中 $y$ 是预测变量，$x_1, x_2, \ldots, x_n$ 是特征变量，$\beta_0, \beta_1, \beta_2, \ldots, \beta_n$ 是参数，$\epsilon$ 是误差项。

- 逻辑回归：逻辑回归是一种机器学习 算法，它使用逻辑模型来预测二值变量。逻辑回归的数学模型如下：

$$
P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \ldots - \beta_nx_n}}
$$

其中 $P(y=1|x)$ 是预测概率，$x_1, x_2, \ldots, x_n$ 是特征变量，$\beta_0, \beta_1, \beta_2, \ldots, \beta_n$ 是参数。

- 决策树：决策树是一种机器学习 算法，它使用树状结构来预测类别变量。决策树的数学模型如下：

$$
D(x) = \arg\max_c P(c|x)
$$

其中 $D(x)$ 是预测类别，$P(c|x)$ 是条件概率。

# 4.2 机器学习 代码实例
考虑一个简单的线性回归问题，用于预测员工薪资。假设我们有一个包含员工年龄和工作年限的表，我们可以使用以下代码实现线性回归：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 加载数据
data = np.loadtxt('employees.csv')

# 分割数据为特征和目标变量
X = data[:, 0:2]  # 年龄和工作年限
y = data[:, 2]    # 薪资

# 创建和训练线性回归模型
model = LinearRegression()
model.fit(X, y)

# 使用模型进行预测
predictions = model.predict(X)
```

这个代码将加载数据，将数据分割为特征和目标变量，创建和训练线性回归模型，并使用模型进行预测。

Q: Presto 和 Hadoop 有什么区别？
A: Presto 和 Hadoop 都是用于大数据处理的工具，但它们有一些主要的区别。首先，Presto 是一个查询引擎，它使用 SQL 语言进行查询，而 Hadoop 是一个分布式文件系统和数据处理框架。其次，Presto 专注于提供低延迟和高吞吐量，而 Hadoop 专注于处理大规模数据集。

A: Presto 和 Spark 都是用于大数据处理的工具，但它们有一些主要的区别。首先，Presto 是一个查询引擎，它使用 SQL 语言进行查询，而 Spark 是一个数据处理框架，它使用 Scala、Java 和 Python 语言进行编程。其次，Presto 专注于提供低延迟和高吞吐量，而 Spark 专注于处理大规模数据集和实时数据。
