# 2. 核心概念与联系

基因编辑技术的核心概念包括：基因组、基因编辑、基因修饰等。在本节中，我们将详细讲解这些概念以及它们之间的联系。

## 2.1 基因组

基因组是一个组织细胞的核内的DNA（分子）的全体遗传信息，它包括了所有的基因和非基因DNA序列。基因组是人类遗传的基本单位，它决定了一个生物的特征和性质。

## 2.2 基因编辑

基因编辑是一种能够精确地修改生物组织细胞基因序列的新兴科技，它具有广泛的应用前景和潜在的社会影响。基因编辑技术可以用于治疗疾病、改善生物资源的可持续性、提高农业产量等方面。

## 2.3 基因修饰

基因修饰是一种通过修改基因组来改变生物特征和性质的技术，它可以用于改善生物资源的可持续性、提高农业产量等方面。基因修饰技术与基因编辑技术有很大的区别，后者主要用于治疗疾病。
