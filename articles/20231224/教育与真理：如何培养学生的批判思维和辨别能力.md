# 2.核心概念与联系

## 2.1 批判性思维
批判性思维是指在思考过程中，通过对现有观点、信息和数据的分析、评价和反思，来达到自主判断的能力。这种思维方式需要学生能够对信息进行筛选、判断、评价和反思，从而形成自己的观点和判断。

## 2.2 辨别能力
辨别能力是指学生能够辨别信息的可靠性、准确性和有效性，并能够区分真理和错误。这种能力需要学生能够对信息进行分析、评价和判断，以便在面对各种信息时能够做出正确的决策和选择。

## 2.3 联系
批判性思维和辨别能力是相互联系的。批判性思维可以帮助学生更好地辨别信息的可靠性和准确性，而辨别能力又可以帮助学生更好地进行批判性思维，形成更加准确和合理的观点和判断。
