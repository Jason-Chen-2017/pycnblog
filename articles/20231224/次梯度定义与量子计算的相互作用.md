## 2.1 次梯度优化算法
次梯度优化算法是一种用于解决高维优化问题的方法，它主要应用于处理非凸、非线性、多模态等复杂问题。次梯度算法通过近似计算梯度信息，从而减少了计算量，提高了优化速度。常见的次梯度优化算法有随机梯度下降（SGD）、随机梯度上升（SGU）、随机梯度平均（SGA）等。

量子计算是一种利用量子力学原理实现计算的方法，它具有超越传统计算机的计算能力。量子计算的核心设备是量子比特（qubit），它可以存储和处理复杂的信息状态。量子计算的代表性算法有量子幂指数定理（QAOA）、量子支持向量机（QSVM）、量子神经网络（QNN）等。

# 定义次梯度优化算法
def times_gradient_optimizer(func, bounds, n_iter=100, n_queries=10):
    x = bounds[0]
    for _ in range(n_iter):
        grad = np.array([func(x + np.array([1e-4, 0])) - func(x - np.array([1e-4, 0])) / 2,
