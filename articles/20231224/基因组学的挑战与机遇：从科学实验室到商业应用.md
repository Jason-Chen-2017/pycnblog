## 4.1 基因组比对代码实例

```python
def needleman_wunsch(a, b):
    m, n = len(a), len(b)
    d = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1):
        for j in range(n + 1):
            if i == 0 or j == 0:
                d[i][j] = 0
            elif a[i - 1] == b[j - 1]:
                d[i][j] = d[i - 1][j - 1] + 1
            else:
                d[i][j] = max(d[i - 1][j], d[i][j - 1]) - 1
    align = []
    for i in range(m, -1, -1):
        for j in range(n, -1, -1):
            if i == m or j == n:
                break
            if a[i] == b[j]:
                align.append((a[i], b[j]))
            else:
                if d[i + 1][j] > d[i][j + 1]:
                    i += 1
                else:
                    j += 1
    return d, align
```

上述代码实现了Needleman-Wunsch算法，用于比对两个基因组序列。`a`和`b`是两个基因组序列，`d`是比对矩阵，`align`是比对结果。

## 4.2 基因组组装代码实例

```python
from collections import defaultdict

def de_bruijn_graph(reads, k):
    graph = defaultdict(list)
    for read in reads:
        for i in range(k - 1, len(read) - 1):
            node = read[i - (k - 1):i + 1]
            next_node = read[i + 1:i + k]
            graph[node].append(next_node)
    return graph

def de_bruijn_assembly(graph, k):
    paths = defaultdict(int)
    for node in graph:
        for path in graph[node]:
            paths[node + path] += 1
    for node in paths:
        if paths[node] == 0:
            paths.pop(node)
    return paths
```

上述代码实现了De Bruijn图组装算法，用于将基因组序列组装成基因和非基因区域。`reads`是基因组序列列表，`k`是K-mer长度。`graph`是De Bruijn图，`paths`是组装结果。

## 4.3 基因组变异检测代码实例

```python
def burrows_wheeler(s):
    table = [s[i:] + s[:i] for i in range(len(s))]
    return max(table, key=lambda x: x[-1:])

def crusoe(s, t):
    if len(s) != len(t):
        return False
    for i in range(len(s)):
        if s[i] != t[i]:
            return False
    return True

def variant_detection(reads, reference):
    variants = []
    for read in reads:
        forward_read = read[:len(read) // 2]
        reverse_read = read[len(read) // 2:]
        forward_consensus = burrows_wheeler(forward_read)
        reverse_consensus = burrows_wheeler(reverse_read)
        if crusoe(forward_consensus, reverse_consensus):
            continue
        variants.append((forward_read, reverse_read))
    return variants
```

上述代码实现了Burrows-Wheeler和Crusoe算法，用于检测基因组变异。`reads`是基因组序列列表，`reference`是参考基因组序列。`variants`是变异结果。
