# 4.具体代码实例和详细解释说明
以下是一个Python代码实例，用于进行K均值聚类分析：

```python
from sklearn.cluster import KMeans
import numpy as np

# 生成一组随机数据
X = np.random.rand(100, 2)

# 使用K均值聚类分析
kmeans = KMeans(n_clusters=2)
kmeans.fit(X)

# 获取聚类中心
centers = kmeans.cluster_centers_

# 获取聚类标签
labels = kmeans.labels_
```

在这个代码实例中，我们首先导入了KMeans聚类分析算法和numpy库。然后我们生成了一组随机数据，并使用K均值聚类分析对其进行聚类分析。最后，我们获取了聚类中心和聚类标签。
