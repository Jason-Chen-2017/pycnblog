# 2.核心概念与联系

在深度学习中，我们通常使用混淆矩阵来评估模型的性能。混淆矩阵是一个矩阵，用于显示预测结果与真实结果之间的关系。它可以帮助我们更好地理解模型的性能，并为后续的优化提供有益的指导。

混淆矩阵的主要组成部分包括：

- True Positives (TP)：正确预测为正的实例数
- False Positives (FP)：错误预测为正的实例数
- False Negatives (FN)：错误预测为负的实例数
- True Negatives (TN)：正确预测为负的实例数

这些组成部分可以用来计算一些重要的性能指标，如精度、召回率、F1分数等。
