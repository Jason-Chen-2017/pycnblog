# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释模型训练的过程。这个代码实例是一个简单的线性回归模型，它通过学习数据来预测目标变量。

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成数据
np.random.seed(0)
X = np.random.randn(100, 1)
Y = 1.5 * X + 0.8 + np.random.randn(100, 1)

# 初始化参数
theta = np.zeros(1)
alpha = 0.01

# 训练模型
for i in range(1000):
    h = X * theta
    loss = (1 / 200) * np.sum((h - Y) ** 2)
    gradient = (1 / 200) * np.sum(2 * (h - Y) * X)
    theta = theta - alpha * gradient

# 预测
X_test = np.linspace(-1, 1, 100)
h_test = X_test * theta

# 绘制
plt.scatter(X, Y)
plt.plot(X_test, h_test)
plt.show()
```
