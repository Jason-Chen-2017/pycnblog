# 4.具体代码实例和详细解释说明

代码实例：

```python
import numpy as np

def jacobian(f, x):
    n = len(x)
    J = np.zeros((n, n + 1))
    for i in range(n):
        J[i, :i] = np.eye(i)
        J[i, i + 1] = f.grad[i]
        J[i, :i] = f.hess[i, :i]
    return J

def f(x):
    n = len(x)
    return np.sum(x**2)

x = np.array([1, 2, 3])
J = jacobian(f, x)
print(J)
```

在这个代码实例中，我们定义了一个函数f，它接受一个n元素的向量x，并返回x的平方和。然后我们调用jacobian函数来计算雅可比矩阵J，其中J是一个(n, n + 1)的矩阵，它的第i列表示函数f在x的第i个元素处的雅可比矩阵。
