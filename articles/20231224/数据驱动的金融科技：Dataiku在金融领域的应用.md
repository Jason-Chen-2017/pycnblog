## 4.1 客户行为分析

### 4.1.1 数据清洗

```python
# 读取客户数据
customer_data = pd.read_csv('customer_data.csv')

# 填充缺失值
customer_data.fillna(method='ffill', inplace=True)

# 转换数据类型
customer_data['age'] = customer_data['age'].astype(int)
customer_data['income'] = customer_data['income'].astype(float)

# 标准化数据格式
customer_data['gender'] = customer_data['gender'].map({'M': 0, 'F': 1})
```

### 4.1.2 数据分析

```python
# 描述性统计
customer_data.describe()

# 图形绘制
plt.hist(customer_data['age'])
plt.show()

# 预测分析
from sklearn.linear_model import LogisticRegression
model = LogisticRegression()
model.fit(customer_data[['age', 'income', 'gender']], customer_data['purchase'])
```

### 4.1.3 模型构建

```python
# 特征选择
from sklearn.feature_selection import SelectKBest
model = SelectKBest(k=3)
model.fit(customer_data[['age', 'income', 'gender']], customer_data['purchase'])

# 模型选择
from sklearn.linear_model import LogisticRegression
model = LogisticRegression()
model.fit(customer_data[['age', 'income', 'gender']], customer_data['purchase'])

# 模型评估
from sklearn.model_selection import cross_val_score
scores = cross_val_score(model, customer_data[['age', 'income', 'gender']], customer_data['purchase'], cv=5)
print('Cross-validation scores:', scores)
```

### 4.1.4 模型部署

```python
# 模型保存
import joblib
joblib.dump(model, 'customer_model.pkl')

# 模型加载
model = joblib.load('customer_model.pkl')

# 模型预测
predictions = model.predict(customer_data[['age', 'income', 'gender']])
```
