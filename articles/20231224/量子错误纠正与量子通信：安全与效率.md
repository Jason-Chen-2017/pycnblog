# 2. 核心概念与联系

在本节中，我们将介绍一些关键概念，包括量子比特、量子错误纠正、量子通信以及与之相关的其他概念。

## 2.1 量子比特

量子比特（qubit）是量子计算中的基本单位，它可以表示为一个复数向量：

$$
| \psi \rangle = \alpha | 0 \rangle + \beta | 1 \rangle
$$

其中，$\alpha$ 和 $\beta$ 是复数，满足 $|\alpha|^2 + |\beta|^2 = 1$。量子比特与经典比特的主要区别在于，量子比特可以存储多种不同的状态，而经典比特只能存储0或1。

## 2.2 量子错误纠正

量子错误纠正（QEC）是一种用于纠正量子系统中错误的技术。在量子通信中，由于环境干扰和传输过程中的量子竞赛，量子比特可能会发生错误。量子错误纠正技术通过对量子系统进行检测和纠正，来控制错误率，从而保证量子通信的安全性。

## 2.3 量子通信

量子通信是一种利用量子物理原理实现信息传输的方法，其中最著名的是量子密钥分发（QKD）。量子密钥分发是一种基于量子物理原理的安全通信协议，它可以实现在不安全的通信环境下，实现安全可靠的信息传输。
