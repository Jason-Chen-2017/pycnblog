### 4.1代码实例

以下是一个简单的个性化饮食方案制定的Python代码实例：

```python
import math

def bmi(weight, height):
    return weight / height**2

def bmr(weight, height, age):
    return 88.362 + (13.397 * weight) + (4.799 * height) - (5.677 * age)

def energy(bmr, activity_level):
    return bmr * activity_level

def nutrient(energy, protein_ratio):
    return energy * protein_ratio

weight = 60
height = 1.75
age = 25
activity_level = 1.55
protein_ratio = 0.2

print("BMI:", bmi(weight, height))
print("BMR:", bmr(weight, height, age))
print("Energy:", energy(bmr(weight, height, age), activity_level))
print("Protein:", nutrient(energy(bmr(weight, height, age), activity_level), protein_ratio))
```
