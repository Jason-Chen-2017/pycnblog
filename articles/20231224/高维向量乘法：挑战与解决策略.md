# 2.核心概念与联系
在深入探讨高维向量乘法之前，我们需要了解一些基本概念。首先，我们需要了解什么是向量和向量乘法。向量是一个具有多个元素的有序列表，通常用大写字母表示。向量乘法是指将两个向量相乘得到一个新的向量。在低维空间中，向量乘法非常简单，只需要元素数量相同的两个向量相乘，然后求和。但是在高维空间中，向量乘法变得更加复杂。

高维空间指的是具有大量维度的空间，这种空间中的向量具有很多个元素。在高维空间中进行向量乘法时，我们需要考虑维数灾难、计算复杂度等问题。维数灾难是指当维度增加时，数据的稀疏性增加，导致模型的性能下降。计算复杂度是指在高维空间中进行向量乘法所需的计算资源和时间。因此，在高维向量乘法中，我们需要找到一种解决维数灾难和计算复杂度的方法，以实现高效的向量乘法。
