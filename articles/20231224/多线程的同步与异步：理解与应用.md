## 4.1 同步的代码实例

在本节中，我们将通过具体的代码实例来详细解释多线程的同步的应用。

```python
import threading
import time

class MyThread(threading.Thread):
    def __init__(self, name):
        threading.Thread.__init__(self)
        self.name = name

    def run(self):
        print(f'{self.name} is running')
        time.sleep(2)
        print(f'{self.name} is finished')

if __name__ == '__main__':
    t1 = MyThread('Thread-1')
    t2 = MyThread('Thread-2')

    t1.start()
    t2.start()

    t1.join()
    t2.join()
```

在上述代码中，我们创建了一个名为MyThread的类，它继承了threading.Thread类。在MyThread类的构造函数中，我们初始化了线程的名称。在MyThread类的run方法中，我们打印了线程的名称和当前时间，然后睡眠2秒，最后打印了线程的名称和当前时间。

在主函数中，我们创建了两个MyThread类的实例，分别名为t1和t2。然后我们启动了t1和t2两个线程。接着我们调用了t1.join()和t2.join()方法，使得主线程等待t1和t2两个线程结束后再继续执行。
