```python
import numpy as np
import matplotlib.pyplot as plt
```
## 4.2 定义函数
```python
def gradient(f, h=1e-5):
    dx = f[1] - f[0]
    dy = f[2] - f[1]
    return (f[1] - f[0]) / dx - (f[2] - f[1]) / dy

def laplace(f, h=1e-5):
    dx = f[1] - f[0]
    dy = f[2] - f[1]
    return (f[1] - f[0]) / dx / dy + (f[2] - f[1]) / dx / dy
```
## 4.3 创建数据
```python
x = np.linspace(0, 1, 100)
y = np.linspace(0, 1, 100)
X, Y = np.meshgrid(x, y)
f = np.exp(-(X**2 + Y**2))
```
## 4.4 绘制图像
```python
plt.figure()
plt.contour(X, Y, f)
plt.colorbar()
plt.show()
```
## 4.5 计算梯度
```python
grad = gradient(f)
plt.figure()
plt.contour(X, Y, grad)
plt.colorbar()
plt.show()
```
