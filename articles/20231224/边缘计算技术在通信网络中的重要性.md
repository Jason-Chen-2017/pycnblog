# 2.核心概念与联系

## 2.1 边缘计算与云计算的区别

边缘计算和云计算是两种不同的计算模型。边缘计算将计算能力移动到网络的边缘，使得数据处理和存储能够更加接近用户。而云计算则是将计算能力集中在数据中心，用户通过网络访问这些资源。

边缘计算的主要优势在于它可以降低网络延迟，提高计算效率，并减少数据传输成本。而云计算的优势在于它可以提供大规模的计算资源，并简化用户的管理和维护工作。

边缘计算和物联网密切相关。物联网是一种技术，它将传感器、设备和其他物理设备连接到互联网上，以实现数据的实时收集和传输。边缘计算可以在物联网设备上进行数据处理和存储，从而降低网络延迟，提高计算效率，并减少数据传输成本。
