# 2. 核心概念与联系

在开始探讨计算机视觉模型的可解释性和透明度之前，我们需要了解一些核心概念。这些概念包括：

- 计算机视觉：计算机视觉是一种人工智能技术，它旨在让计算机理解和处理图像和视频数据。这包括对象识别、场景分析、图像分类等任务。
- 深度学习：深度学习是一种机器学习方法，它基于人类大脑中的神经网络结构。深度学习模型可以自动学习从大量数据中抽取出的特征，从而进行图像分类、对象检测等任务。
- 可解释性：可解释性是指模型的输出可以被解释为人类可以理解的形式。这意味着我们需要能够理解模型是如何对输入数据进行处理和分析的，以及它们是如何到达某个结论的。
- 透明度：透明度是指模型的工作原理和决策过程是可以被观察和审查的。这意味着我们需要能够检查模型的决策过程，以确保它们是可靠的和公正的。
