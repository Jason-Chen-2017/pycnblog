# 4.1 代码实例：蒙特卡罗估计π的示例
在这个示例中，我们将使用蒙特卡罗方法来估计π。我们将通过生成大量的随机点并检查它们是否落在一个单位圆内来估计π。具体的代码实现如下：

```python
import random
import math

def monte_carlo_pi(n):
    inside_circle = 0
    for i in range(n):
        x = random.uniform(-1, 1)
        y = random.uniform(-1, 1)
        if math.sqrt(x**2 + y**2) <= 1:
            inside_circle += 1
    pi = (inside_circle / n) * 4
    return pi

n = 1000000
pi = monte_carlo_pi(n)
print("估计的π值：", pi)
```

在这个示例中，我们首先导入了`random`和`math`模块。然后，我们定义了一个`monte_carlo_pi`函数，该函数接受一个参数`n`，表示需要生成的随机实验的数量。在函数内部，我们通过生成大量的随机点并检查它们是否落在单位圆内来估计π。最后，我们调用`monte_carlo_pi`函数并传入参数`n`，并打印出估计的π值。
