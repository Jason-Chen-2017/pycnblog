# 1.背景介绍


在IT行业中，招聘信息量越来越大，职位需求也越来越复杂，使得很多技术人员因找不到好的工作而屡屡迷失方向。但实际上，如何更好地把握自己的职业生涯，为自己提供一个稳定的平台和价值体系，真正成为一名合格的技术人才，还是需要一些“技巧”的。这些“技巧”，就是本文所要探讨的面试技巧——自我反思、心态建设、应试教育、反馈及时性、简历编写技巧等。
其实，“自我反思”在所有人的职场生涯都非常重要。它的目的很简单——帮助自己成长，养成不断学习、提升自我的习惯。当遇到问题，看到机会的时候，首先应该做的是站起来反省，自己过去走过哪些弯路、为什么错过了什么样的机会，然后再行动起来，为明年找到更好的发展方向。正如盖茨在《管理精英》里说到的，每个人都需要经过自己的内心世界的检视，才能知道最适合自己的生活方式和发展方向。所以，“自我反思”是每位技术人都不可或缺的必修课。
# 2.核心概念与联系
## 2.1. 反馈与沟通
在IT行业，面试过程中经常出现的一种情况是，你面试完后，对方可能会给你反馈意见或者建议。比如，可能问起你的技术水平、项目经验、团队合作、工作积极性等。但是，通过反馈能获得的信息往往都是主观的、模糊的，甚至有时候根本就没有，因为这是在试图改变你，让你更好的投入到工作中去。那么，如何有效地收到和处理反馈？下面介绍几种方法。

### 2.1.1. 慎重对待反馈
在面试前多与同事交流，先和他们分享一下自己的想法，询问他们是否有意愿接受反馈。不要在面试前告诉对方，即便对方有意愿接受反馈，也会破坏面试氛围。因为，如果在面试过程中，反馈太多导致你紧张压力过大，甚至无法正确表达自己的想法和看法，这样的结果往往是严重的。而且，如果你跟对方承诺了一些东西，就算面试结束之后没得到回应，也需要勇于面对自己的不甘心，重新调整下节奏。

### 2.1.2. 充分利用反馈
反馈可以促进职业发展、激励个人能力的提高。但是，一味依赖反馈，并不能解决所有的问题。因此，当面试官反馈时，一定要对反馈进行分析和评估，只有充分考虑反馈的内容、反馈者的不同看法、自己的优缺点，才能找到解决问题的方法和方案。
另外，与面试官交流的时候，不要故意夸大自己的水平。比如，面试时你表现出很强的领悟能力，面试官可能会觉得你已经非常厉害，从而认为你“像外星人一样聪明”。但是，实际上，可能只是你领悟能力比一般人强而已。另一方面，面试官也希望你能够展示自己对某项知识和技术的理解程度，而不是单纯谈论自己技艺高超。

### 2.1.3. 不要盲目跟风
遇到强势面试官时，不要盲目跟风，而是先尽量低调处理。比如，你可以先私信面试官问候一下，然后继续安排日常工作。这既可以避免被“单刀直入”，还能保护自己及对方的安全。同时，还要仔细研究一下对方的面试方式，以及公司是否有针对性的培训。

## 2.2. 简历编写技巧
任何技术类岗位都需要一份优秀的简历。作为一个技术人员，简历是一个很重要的工具，它将你的 abilities 和 skills 提炼出来，同时也可以让雇主有个直观的了解。当然，也是你迈向更高职位的重要一步。下面介绍几种简历编写技巧。

### 2.2.1. 用词精准
简历的用词要精准，尤其是在面试中。比如，技术相关的职位，不要用“精通”“熟练”之类的词语，更加务实一点；专业术语要准确，不能用缩写或错别字；擅长的技术领域也要突出，不能搞得太过泛泛。除此之外，还有一些常用的表达方式，可以参考。

### 2.2.2. 语言表达清晰
简历的语言表达，要清晰准确、易懂。简历中的字数限制在1500-2000字之间，推荐使用PDF文件进行编辑。

### 2.2.3. 简历要结构化
简历的结构化包括个人信息、工作经历、教育背景、项目经验、证书、获奖证书等。各条信息之间有明确的界线，如工作经历和教育背景之间有一个简单的分隔符。简历应该有清晰的排版，并且可以打印输出。

### 2.2.4. 使用模板
可以选择有模板的简历，省去重复制作的时间。当然，也可以根据自己的需要制作简历。

## 2.3. 应试教育
应试教育是一个相当古老的学科，早在哲学、数学、物理学之前，就有应试教育这一概念。简单的说，应试教育就是通过考试的方式，来衡量一个人对某门课程的掌握程度、素质、水平。在这个过程中，考试以考查的方式呈现，考生通过答题、写作或语言表达，表现其对知识点的理解、分析和运用。因此，应试教育的目的是为了判断一个人知识和能力的水平，通过考核来对学习进行评估和改善。
面试的时候，应试教育往往成为最后的筛选环节。所以，提高应试技巧是非常必要的。下面介绍几个应试教育的技巧。

### 2.3.1. 应付考试中的倒计时
随着竞争日益激烈，面试成为求职中的重要环节。但是，由于笔试和面试本身都有着严苛的标准，因此，时间常常是最宝贵的资源。所以，应付考试中的倒计时，应该多花一些注意力。比如，在开始倒计时前，可以做好充足的准备，设置闹钟提醒自己。另外，在考试前，可以多花些时间复习。以免让人手忙脚乱，或者考试时间太短影响效率。

### 2.3.2. 抓住重点
在应试教育中，重点的意义在于你要快速解决某个问题，而非解决所有问题。所以，抓住重点，更容易取得好成绩。记住，只要能够解决关键的问题，其他的问题自然迎刃而解。

### 2.3.3. 记住亮点
对于一些技术含量比较高的问题，考试中出现的亮点往往是比较有启发性的。所以，记住考试中的亮点，对之后的学习、工作都会产生重大的影响。

### 2.3.4. 保持专注
在考试的时候，不要分心，专心于考试本身。不要花太多的时间在回答问题上，而是花更多的时间在思考上。很多人习惯用手机的互联网工具，而忽略了注意力和专注。

# 3.反思
面试过程中，除了考查知识技能，更重要的环节之一，是反思和总结。在面试中，可以和面试官一起，重新审视自己，收集对自己进行评价的材料，及时总结和修正自己的错误，提升自己对自己的认识和见识。通过反思，我们能够更全面地了解自己，增强自信，寻找到适合自己的发展方向，最终帮助自己更好地融入团队和社会。
