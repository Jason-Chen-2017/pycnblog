# 1.背景介绍


晶圆制造技术一直是电子行业不可缺少的一环。晶圆(Crystal)就是一种无色粒子晶体,它由晶体素构成,晶体素可以表现出许多物质的不同结构和特性。随着电子技术的不断发展,晶圆制造技术也在不断地进步,但为了使晶圆更容易制作、更容易切割、更好看、更耐用等优点,目前还没有一种特定的方法或工艺能够完全解决这一难题。

晶圆制造的关键在于精心设计,首先要考虑的是整个晶圆的性状,如晶体形态、结构、厚度等,其次要选择正确的加工工艺,最重要的是掌握设备的能力和技术。在完成晶圆设计之后,我们需要进行清洗,这是一种必要的过程,目的是为了去除晶圆上杂质、分离晶体,并且保护晶体不会腐蚀、温度升高或其他原因造成损坏。

清洗的主要目标是将晶体中的杂质和不必要的物质排除掉,常用的清洗方式包括磨光、腐蚀、冷风拧干、气流冲洗、液化等,然而这些方法都存在着各种缺陷和不足。为了达到最好的清洗效果,并避免一些因清洗方式导致的问题,还有一些专业的清洗工艺才会被应用。

去背面工艺是晶圆制造中一个重要的组成部分,它的目的就是从晶体中去除背面的层层包裹物,以此提高晶圆的耐用性、品质和易切割性。

由于各个国家的电路板制造企业都有自己独特的工艺流程,因此对于晶圆制造过程的理解不能局限于某一种工艺流程或者简单的做法。相反,了解制造过程中所有环节及其相互关联的知识,并把它们融汇贯通,才能使我们更好地理解和处理各项工程。

本文将通过浅显易懂的语言,详细阐述晶圆制造过程的核心概念、相关术语及其联系,并对晶圆清洗与去背面工艺进行科普。文章内容适合具有一定相关经验的技术人员阅读。

# 2.核心概念与联系
## 2.1 晶圆
晶圆又称晶体、石墨烯、晶石、结晶、晶体粒子,是指由无色晶体素组成的三维晶体,其构造与原子核反应的组合形成三种基本类型:立方晶体、环状晶体、棱晶体。每个晶体元素都是无色无结构的粒子,通过相互作用改变状态,产生所需的晶胞和图案,再结合在一起组成晶体。

根据晶体元素的结构特性和加工条件不同,可分为两种类型:正向晶体、负向晶体。正向晶体是指由金属、氮等丰富的原子形成的晶体,而负向晶体则是指由电子、硫酸盐等偏轻的原子形成的晶体。

## 2.2 回旋
回旋(Spin)是指晶体的角动量守恒律,即处于不同格点的两个原子的动量差等于自身动量的两倍。若有一个分子的动量守恒,该分子也具备回旋性。晶体的所有原子处于共轭关系,两个原子之间存在共轭分量,其大小等于它们自身相互作用的数乘积,称为本征值或本征频率。不同晶体格点上两个原子的动量、共轭和本征之间的关系决定了晶体的排列顺序,即形成各种不同的晶格模式,从而具有良好的自然颜色和定型特征。晶体的回旋能量等于其所有单电子自旋分量(占据能级)的相互作用的能量之和。

## 2.3 清洗
清洗(Cleaning),也称去除杂质,是指对一个晶体上的杂质进行去除、腐蚀、净化,使之成为清晰、纯净的晶体。现代晶圆制造技术的中心任务是在低温下有效地清洗晶体,并确保其清洁程度不受影响。常用的清洗手段有磨光、腐蚀、冷风拧干、气流冲洗、液化等。

磨光是利用电磁辐射来擦除晶体上杂质的一种方法。在晶圆面前施加足够强大的磁场,可以使晶体内部电子运动的方向发生颠倒,从而在短时间内清除晶体上的杂质。然而,由于晶体的大小、形状、摆放位置、加工环境等条件限制,磨光可能会引起晶体破碎、反射干涉等严重后果。另外,采用磨光的方法会损失部分晶体的分辨率,降低晶体的结构保真度。

腐蚀是指将晶体表面积分的杂质与无序的晶体层层堆叠在一起,从而形成像海绵一样的晶体表面。在晶圆上腐蚀主要依靠各种温湿度变化,增压、加热、风吹动等方法。

冷风拧干是通过对晶体上部过厚的层次进行热风拧紧,使这些层次不会脱落、破坏。由于冷风的热量交换带来的可燃性,使晶体表面的油脂容易蒸发,从而实现晶体表面清洁、细腻。

气流冲洗是指通过使晶体经过高速流动而对晶体中废料进行排放,从而将废料排除出晶体而达到清洗目的。然而,这种方法只能用于隔热保护性较差的晶体,且对速度要求较高。

液化是指通过将晶体表面和体底放置于液体中,使其渗透至各层层面的晶体层层之中。随着液化过程持续不断,晶体表面逐渐变干、变脆弱、变白,从而达到清洗目的。

## 2.4 去背面
去背面(Surface/Back Elimination)，简称BE，是晶圆制造工艺中的重要组成部分。BE是通过除去晶圆表面薄膜层(Backside Mask)的方式,消除晶圆表面层次,提高晶圆的厚度和稳定性。通常BE的主要方法有纯粹工艺或基于激光的方法。

BE的作用主要有三个方面。第一，可以降低晶圆表面电阻,减小晶圆开孔,抑制晶体腐蚀现象,保障晶圆的韧性和定型性能。第二，可以消除表面坏死物质、尘埃、飞沫等物质,改善晶圆的外观品质。第三，可以促进晶圆的自然分界、提高晶圆层次分明度,使其具有更高的厚度和韧性。

BE也可以采用多种方法,如电镀、洗涤、蜕石、偏振、模具疏通等。基于激光的方法,即利用光照的方式增厚晶圆表面薄膜层,使晶圆表面形成一层薄膜覆盖,然后通过控制光线波长,来除去或保持晶圆表面的薄膜层。

# 3.核心算法原理与操作步骤
## 3.1 腐蚀算法(Erosion Algorithm)
腐蚀(Erosion)是一种比较常见的图像平滑滤波算子。它可以用来模糊图像中的噪声、边缘等信息,从而提高图像的视觉质量和分析效果。在晶圆制造工艺中,由于晶体的加工工艺复杂,液化、电镀、模具疏通等各种方法都使得晶体表面产生各种层次的杂质,而腐蚀算法就扮演了重要角色。

腐蚀算法的基本思想是先对图像进行预处理,然后将图像灰度化,并对图像进行微小扰动。接下来进行腐蚀处理。第一次腐蚀是以一定的步长,将相邻像素中的最小值与当前像素进行比较,取较小者作为当前像素的值。第二次腐蚀时,步长逐渐减小,从而使图像中的细节更加清晰。经过两次或多次腐蚀后,得到的图像就非常平滑了。

在晶圆制造工艺中,可以使用腐蚀算法对晶圆表面进行清洗,使其更加平滑,从而降低晶体表面的坏死物质、尘埃、飞沫等物质,提高晶圆的整体外观品质。

## 3.2 模拟退火算法(Simulated Annealing Algorithm)
模拟退火算法(SA)是近年来一种基于概率论的优化算法。它采用模拟退火策略搜索全局最优解,属于启发式算法,不是一种全局最优求解方法。SA的基本思想是借助随机生成的方法,逐渐变换搜索空间中的参数,以期望找到更佳的解。

模拟退火算法可以对晶圆清洗工艺进行优化,从而改善清洗的效果。SA算法的工作原理是按照一定概率接受比当前最佳解更差的解,以期达到更快、更准确地搜索全局最优解。当搜索结束后,得到的结果可能不是全局最优的解,但其概率较低。

在晶圆制造工艺中,可以使用SA算法优化晶圆清洗工艺的效率,提高效益。SA算法的运行时间较长,因此需要对其调参、设置停止条件等,以达到优化清洗效果的目的。

## 3.3 流体模拟算法(Fluid Simulation Algorithm)
流体模拟算法(FSA)是一种利用计算机模型计算流体力学和动力学的数学方法。它可以模拟任意复杂的流体现象,包括粘性、液滴行为、沉积、流体运动、传播特性、压缩性、暂态性、热传导、自然抗力、渗透力、碰撞等。

在晶圆制造工艺中,可以使用流体模拟算法来研究晶体流动、分离、模塑等过程,从而推导出晶圆清洗工艺的理论依据。FSA算法可以在很短的时间内计算出给定结构的各种物理属性,因此有利于优化晶圆制造工艺。
