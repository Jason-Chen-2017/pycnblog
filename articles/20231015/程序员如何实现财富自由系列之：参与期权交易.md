# 1.背景介绍


## 什么是期权？
期权是一种风险投资工具，它提供给投资者一个选择权，允许他们在合约到期之前，根据自己的预期价格买入或卖出资产。相对于股票市场，期权交易具有更高的仓位要求，同时也意味着投资风险增加。

不同于现货期货交易，期权交易并不是以价格作为主要交换媒介，而是以权利金的方式进行交易。也就是说，投资者不用购买固定数量的证券，而是获得根据当时市场价值的权利金。期权是一个双向合约，由对方授权给投资者买入或卖出的权利，期权到期后，双方都要付费。期权的特点包括：

- 没有涨跌停板限制；
- 有隐含波动率，风险较低；
- 可用于多种行情，如股票、债券、商品期权等；
- 结算周期长，一般为1个月以上。

## 为什么要参与期权交易？
参与期权交易可以享受到很多优惠。首先，期权交易有很高的风险，投资回报率较低。如果没有风险控制能力，则风险会直接带来巨大的损失。其次，期权交易有更高的收益，因为收益取决于双方之间的博弈，而不是简单地以合约面值作为单位计算收益。第三，期权交易可以自主选择自己的持仓比例，可以减少交易成本。最后，期权交易可以在全球范围内流通，充分保障个人的权益。

# 2.核心概念与联系
## 期权定价公式
期权定价公式描述的是双边共担的权利义务，即期权双方各自承担的风险。公式为：

$C_t(S)=-\frac{e^{-rT}[(b-\frac{\delta}{2})S+\frac{\sigma^2T}{2}]}{\sqrt{T}}$

其中，C为期权合约的资金曲线，S为标的资产价格；

b为期权买方收到的权利金（保障金），即买方在收到期权时拥有的未付利息资金；

$\delta$为期权价格波动性，即每一次行权后的赔付比例；

$\sigma$为期权价格的幅度，即期权价格与正态分布的标准差的比值；

T为期权到期时间。

另外，期权定价还可以用波动率指数（VIX）作为参数。波动率指数是美国国债市场上根据历史波动率数据计算得出的一个指标，用来衡量国债价格的变动幅度。国债市场的波动率指数通常在波动率达到5%以上时表明市场大概率出现短期均值回归（市场宽幅震荡），上涨趋势收缩、下跌趋势加速。


$C_t(S)= - \frac{(b + r + (\frac{\delta}{2}-\gamma)\cdot VIX + \kappa_{v}\cdot \log S )}{(\frac{\sigma}{\sqrt{3}}\cdot \sqrt{T})} $



其中，

b表示信用评级公司提供给投资者的期限内承诺到期权的资金保障金额；

r表示期权的无风险利率，可根据交易所调整；

$\delta$表示期权价格的跳空比例，期权收盘价高于最低估计价时，买方可以通过跳空的方式获利，反之亦然；

$\gamma$表示负的惩罚项，如果期权价格落在合理估计范围外，实际价差超过了$\delta/2$，则会对买方惩罚；

$\kappa_{v}$表示向下估计和向上估计两个估计价之间的比例系数，通常取值在0.98左右；

$VIX$表示波动率指数，可通过网页或应用程式获取；

$\log S$表示标的资产的对数价值。


## 看涨期权和看跌期权
看涨期权是期权交易中较为常用的一种类型。根据期权的特性，买方愿意支付合理的期权费用，就能够以更优惠的价格获得期权证券的实物价值。这种情况下，买方将获得以其投入的资金为保障的更多的利润。

而看跌期权则相反，它提供了另一种选择。看跌期权只能在期权期限内买进，因此，看跌期权是一种保守的行为。只有买方能够承受一定程度的损失，才会决定是否继续维持该方向的期权。

## ATM（ATM神奇奶酪）定价方法

ATM（ATM神奇奶酪）定价方法是一种以当前价格为基础，设置一个价格触发点（PTM）的期权交易方式。PTM是指某个特定的价格水平，当买方的最优策略执行时，以此价格抵扣掉这部分额度。按照价格触发点进行交易的最大好处是避免了买方损失过多。当然，缺点也是有的，例如，价格触发点可能与市场其他波动变化形成冲突，难以找到符合条件的价格触发点。

ATM定价公式：

$C_t(S) = -\frac{(b+r)(S-\lambda)^+}{\delta}\text{ if } S > PTM; C_t(S) = \frac{-b-r}{(\delta-\theta)}S^+ + b\text{ otherwise }\tag{1}$ 

在公式(1)中，$S^\pm$表示市场处于压力阶段还是支撑阶段，当市场处于支撑阶段时记作$(+)$,当市场处于压力阶段时记作$(-)$. 当期权到期时，如果期权合约面值$C_t(S)$大于等于$PTM$,则执行买方最优策略；否则，执行卖方最优策略，按照相反方向进行交易。

公式(1)中，我们假设期权的价格从$PTM$开始升值，并且我们知道初始的$PTM$的值，则公式(1)可以有效地避开波动率下跌的影响，从而保证利润最大化。另外，公式(1)中的$\lambda$是对应于$PTM$值的一个值。
