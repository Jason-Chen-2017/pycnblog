# 1.背景介绍


在进入数据的科研、应用及商业等各个环节之后，传统的数据分析服务将无法应对海量、多样化的数据。因此，如何从数据源头（即数据采集层）解决数据治理与合规性，成为企业面临巨大的技术、管理、法律、法规等方面的挑战。“数据治理”作为企业内外信息流动的枢纽，也是推动数据价值的关键节点。如何通过有效管控，保障数据安全、完整性、可用性以及处理过程中的隐私风险等，成为运用大数据技术带来的新一代信息革命的一项重要课题。

数据治理是指对数据的收集、存储、分类、使用、共享、管理等全生命周期进行规范的管理，目的是确保数据的正确性、完整性、可用性以及处理过程中的隐私风险等。数据治理的目标既要满足信息需求，也要保障数据安全、稳定运行，以防止数据泄露、个人信息侵犯等风险。

数据合规是指数据的获取、使用者身份确认、授权、以及数据的共享、公开、监督等行为都受到相关法律法规约束，并符合相应的数据使用协议。数据合规意味着信息能够准确且可靠地传递给需要它的人，不受到欺诈、非法使用等风险。

现实世界的数据管理是一件艰巨而复杂的工作。无论是互联网、移动互联网、物联网还是传感网、工业制造等，大数据技术都正在助力创新型经济，更好地服务人类社会。如何加强数据治理，成为企业综合实践中的一项重要技能，才能取得更好的效果。

本系列文章将讨论数据治理与合规性的原理、方法、技巧、工具以及适用场景，阐述企业中如何建立数据治理机制，达成数据治理与合规性的共识，并提出数据治理相关的行业政策建议。希望通过本系列文章，帮助读者深入理解数据治理与合规性，为其打下坚实的基础，实现数据价值最大化。
# 2.核心概念与联系
## 数据管理
数据管理是指对数据的收集、存储、分类、使用、共享、管理等全生命周期进行规范的管理。数据管理是通过技术手段及制度流程来控制和促进组织间信息的共享和使用，以达到优化信息资源和服务质量，并减少数据的流失、腐败等隐患，提升数据价值和信息价值的过程。
## 数据治理
数据治理是对数据的获取、使用者身份确认、授权、以及数据的共享、公开、监督等行为都受到相关法律法规约束，并符合相应的数据使用协议。数据治理的目标既要满足信息需求，也要保障数据安全、稳定运行，以防止数据泄露、个人信息侵犯等风险。数据治理是为了有效地管理数据，保证数据准确、完整、可用、可用性以及处理过程中个人隐私权的保护。

数据治理体系包括三大支柱功能：主体责任、法律法规与规范、流程治理。其中，主体责任是指保证个人信息主体的合法权益；法律法规与规范是指对数据管理进行必要的法律法规要求和规范约束；流程治理则是指完善的数据管理制度，确保数据流转的合法、有效、透明、可追溯。

## 数据资产
数据资产是指与数据管理相关的各种资产、文件、制度、标准、手册、办证等制度化的流程和服务，如数据资产管理计划、数据处理标准、数据资料库、数据质量审核、数据安全管理规章等。数据资产可以分为两个部分，第一部分为数据相关的技术资产，例如大数据处理平台、数据传输网络、云计算服务器等；第二部分为法律法规的法律文书资产，包括数据安全法律法规、电信、通讯、信息内容等方面的法律法规。

数据资产管理是对数据的管理过程中所产生的数据资产的整体把握、使用情况的统计分析、数据治理与合规性的评估、管理制度的改善、数据管理审计的执行等工作，是数据治理中不可或缺的一环。

## 数据分类
数据分类是对来源不同的原始数据按照其特征和目的，按数据内容、结构、时间、空间等维度对数据进行整理归类的过程。数据分类的作用主要有：

1. 将不同类型的数据相互区别开来，避免混淆。
2. 提高数据处理效率，缩短数据的响应时间。
3. 对数据进行规范化、标准化、清洗等处理，降低数据孤岛化、数据违规概率。
4. 为数据提供业务价值，发现隐藏的信息。

## 数据隐私
数据隐私是指敏感、私密和有关个人生活信息的数据，在社会生活、经济活动和交易过程中，可能会被非法使用、泄漏、滥用甚至披露给他人。数据隐私通常存在以下几种形式：

1. 静态数据隐私：指数据没有发生变化，仅反映某种状态，如住址、手机号码、姓名等；
2. 动态数据隐私：指数据随着时间变动，表现出新的特征，如职业、年龄、生日、婚姻状况等；
3. 群体数据隐私：指由多个数据主体共同组成的总体数据，如家庭住址、联系方式等；
4. 跨境数据隐私：指跨境电子商务数据、个人信息等。

数据隐私的保护是人们在使用数据时对个人信息的保护，包括数据的安全性、真实性、合法性、可用性和持续性。数据隐私保护制度应遵循《中华人民共和国信息安全法》和相关国家数据安全法律法规，不得泄露、篡改数据、滥用数据、侵害数据主体合法权益。

数据隐私与数据安全的区别：数据安全是指保障信息资产、信息系统和人员的机密性、完整性、可用性和可靠性。一般情况下，数据安全保障对个人隐私的保护比数据隐私保护更加重要。数据隐私是指数据中可能含有个人信息的内容，个人数据包括直接可识别个人的相关信息，如姓名、出生日期、住址、个人身份信息等。
