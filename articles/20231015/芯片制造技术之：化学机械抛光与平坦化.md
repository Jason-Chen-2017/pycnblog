# 1.背景介绍


芯片制造是我们作为一个企业都十分关注的一个方面，而这方面的技术发展也给我们带来了许多机遇。随着信息技术的发展，现在我们可以通过智能手机、平板电脑等等获得海量的数据，对现实世界进行大数据的分析处理。芯片制造作为一项重要的技术，其起步阶段往往较晚，但是随着人工智能、云计算等技术的不断发展，芯片制造在近几年的发展可以说已经到达了蓬勃发展的状态。不少国内外知名公司均已布局芯片制造领域。比如华为、联发科、爱立信、高通、苹果、微软、特斯拉、中兴等等。

然而，芯片制造技术又是一个非常复杂的领域，需要依赖工程技术人员高度的技术能力，而这些能力却大多受到一些束缚。比如，目前国内外绝大多数企业的芯片制造都是以工艺流程为主导，制程人员只能简单地执行简单的操作，缺乏更高级的知识。另外，由于这些制造工艺的特殊性，工艺流程往往不可复制，导致效率低下、成本高昂、产出不佳。因此，如何提升芯片制造工艺流程的可复制性和自动化程度就成为一个必然的方向。

其中，化学机械抛光与平坦化是芯片制造领域里最基础的两个环节。首先要理解的是，什么是化学机械抛光？为什么要用它？它的作用又是怎样的？还需了解一下，什么是平坦化？平坦化又是为了解决什么问题？它具体实现的方式又是什么？这一系列的过程将帮助我们进一步了解这个领域的关键技术。

# 2.核心概念与联系
## 2.1 化学机械抛光
化学机械抛光（Chemical mechanical polishing, CAMP）是指利用化学加工技术在金属表面、天窗、玻璃等表面或喷涂在导电材料上对物质进行物理破坏，从而使设备的电性能变得更好，更耐用的方法。通过这种技术，可以降低电池电压，缩短寿命，降低能耗。

化学机械抛光包括以下三个主要过程：

1. 清洗法：此过程中先对设备表面和基层施加刮除剂，然后用污染掩膜覆盖设备，随后在设备上快速移动机械操纵设备，使其对表面进行物理化学清洗，对污染部分进行摧毁，最后对未经过清洗的污染部分进行重组。
2. 涂胶法：这是一种常用的前期处理方式。用剂化学合成各种各样的物质，将它们嵌入到设备的表面或基层，并逐渐侵蚀设备的表面，消除杂质。在设备表面留下的残余物质，就称为胶体。胶体的形状和大小与抛光原件的形状及功能密切相关，而且胶体的生长速度很快，所以很多设备都采用了这种清洗和修复的方式，以尽可能的减轻设备损伤。
3. 固化法：此时，用中空或者绝缘子孔作承载物，使之固定在设备上，防止其粘结，从而达到一次性完成整个过程。通常情况下，前两者属于温度控制的抛光过程，而后者则是高速稳定的过程。

通过以上三个主要过程，可以把设备表面物质切割为细小颗粒，同时还可以在每个细小颗粒中插入专门设计的感应体系，这样就可以收集到该颗粒表面的各种离子。这样的离子对于降低设备电压、提高其寿命和有效保存功率都有很大的作用。

## 2.2 作用
化学机械抛光主要用来降低电池电压、缩短寿命、降低能耗。其中降低电压和降低能耗是其显著优点。虽然降低电压和降低能耗对于人们来说都是一件值得欣慰的事情，但在实际应用中，仍然存在着很多隐患。比如，由于抛光过程中容易造成设备焊接、折叠等不良现象，甚至会造成设备严重故障。另外，化学机械抛光还会产生热、光泽等辐射，可能会对环境造成危害，增加排放、污染等风险。因此，在保证设备安全和性能的前提下，每当有设备需要抛光时，应该优先考虑这一项技术。

## 2.3 关键环节
- **清洗法**是使机身的表面受到一定的污染，并将它们清除掉的过程。
- **涂胶法**主要用于处理设备的表面，主要是通过一些不易腐蚀的物质，对电路区域进行物理处理，使之远离杂质并降低设备损伤，提高设备的操控精度和性能。
- **固化法**是使化学机械抛光具有长期稳定性的方法，需要仔细设计并进行数量上的投资。

## 2.4 抛光原件
化学机械抛光的前提条件就是选择适宜的抛光原件，即将电路结构中的杂质转移到设备表面。不同的抛光原件在不同的条件下，所产生的力有所不同，并且在不同的环境中能够产生不同的效果。常见的抛光原件有：

- **磨砂原件**：由砂石、石英、或其他不含任何化学成分的原料做成，基本没有任何质量和电荷，但具有良好的抗震性能。
- **树脂原件**：树脂是一种非金属矿物质，具有强烈的抗冲击性和弯曲变形能力，在抛光过程中具有良好的兼容性。
- **塑料原件**：塑料制品具有较高的硬度，能够克服摩擦，提供较好的操控精度。
- **氧化物原件**：用于替代硫酸钾等高敏电介质。
- **硅酸盐原件**：用于补偿设备表面缺陷，尤其是在设备制造中，有些工艺步段中会将这一部分原件涂在沉积层、组件表面等位置，以提高整机的抗腐蚀性和抗碎性能。
