# 2.核心概念与联系
## 概念解析
### 什么是智能制造？
智能制造是指通过智能技术实现的制造过程中的各个环节的自动化，包括生产线的组装、质量管理、物流跟踪、风险管控、工时控制等。
根据对智能制造的定义，主要包括以下几个方面：

1. 生产线组装自动化

   自动化生产线组装是智能制造的一个重要组成部分。自动化生产线组装系统能够精准的根据设计图纸，按照设定的标准生产制品件，并能够进行快速、准确的整合，从而缩短产品开发时间，提升生产力水平。

2. 物料供应链管理

   通过精细化的物料采购、仓储、运输网络，以及现代信息技术手段，智能制造可以大幅降低成本，提高企业的利润空间。

3. 产品质量管理

   在智能制造中，产品质量管理将被部署到制造环节。在质量不达标的情况下，智能制造系统能够预测故障原因、调整生产流程、节省成本和维修费用。

4. 风险管理

   对工厂和运输设施进行定期的安全评估，通过技术手段及时发现可疑事件，减轻企业的安全风险。

5. 工时控制

   为保证生产效率，工人需要保持必要的睡眠时间，而智能制造可以检测工作环境变化，根据工人的个人情况，调节工作时间。

总之，智能制造是利用人工智能技术来改善生产经营方式，提升生产效率、降低成本、优化产品质量，并防范工厂和产品的安全风险的一种产业。

### 有哪些类型的人工智能技术？
目前，人工智能技术已经成为许多行业的热点。其中，制造业也处于迅速发展阶段。智能制造领域，通常有三种类型的人工智能技术：智能控制、智能数据处理、智能协同。

#### （一）智能控制
智能控制是指通过计算机控制机器人、机器人控制机器人，让机器人实现各种自动化功能。典型的智能控制技术有深度学习技术、强化学习技术、模糊逻辑技术、遗传算法技术等。

#### （二）智能数据处理
智能数据处理是指通过计算机分析数据，生成符合人类认知习惯的结果，例如图像识别、语音识别、文本处理等。该技术可以帮助解决复杂的业务问题，提升工作效率。

#### （三）智能协同
智能协同是指多个智能体（机器人）之间的互动，共同完成复杂任务，例如人机交互、无人机协同等。与其它两种技术不同的是，智能协同可以同时处理多个任务，提高协同的效率。

### 如何应用智能制造技术？
一般来说，智能制造可以分为三个阶段：建模阶段、部署阶段、应用阶段。

- 建模阶段

  在该阶段，智能制造人员会分析用户需求，提取产品特性和规格，然后基于计算机模型建立产品的自动化系统。构建模型需要考虑的因素有：生产工艺、原材料库存、供应链动态、工时约束等。

- 部署阶段

  在该阶段，会选择一家制造商或供应商，与其合作，以获取对产品的研发生产和运输方面的支持。除了提供基本的技术支持外，还会帮助企业规划和开展研发计划，跟踪设备的质量，并管理设备的采购、安装和保养。

- 应用阶段

  在该阶段，产品制造的整个流程都由人工智能驱动，例如：制造、质检、发货。相关部门也会把注意力转移到服务上，例如：售后服务、支持客户，减少人工服务的投入。
