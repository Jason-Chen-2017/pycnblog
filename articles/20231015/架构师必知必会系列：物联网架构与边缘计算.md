# 1.背景介绍



2019年，智能设备、机器人、电子产品等新型产业爆发式增长，物联网、移动互联网、大数据、区块链等技术领域正在崛起，其影响力已经远超过互联网。作为行业的引领者和领跑者，物联网行业正在颠覆传统经济和商业模式，成为经济全球化和产业升级的新窗口。如今，物联网从应用层到基础设施层都处于蓬勃发展阶段，为企业提供从智能化到云端一体化的整体解决方案。但同时，IoT边缘计算也在逐步被提出，它与传统的云计算相比，将智能终端设备、环境资源、用户行为等信息进行智能汇聚、分析和处理，并通过边缘节点或网关连接到云端，实现数据的传输及云端计算。边缘计算带来了极高的效率和低延迟，为人们生活中的众多场景提供了服务，但同时也面临着各类复杂的技术挑战。

本专栏的作者清华大学电子信息系研究生陈益均，现任华为公司技术经理、CTO兼SOHO中国区总裁。他历任阿里巴巴集团、小米科技、腾讯、百度、华为、美团点评等知名公司的高级技术专家，曾就职于国内顶尖IT企业，负责相关产品的研发、架构设计和推广工作。具有丰富的物联网、大数据、区块链项目开发和落地经验，对物联网、边缘计算技术有深刻的理解和实践能力。他认为，物联网与边缘计算是互相促进的两个分支，必须协同创新、共同发展，才能实现物联网时代的真正的跨越。因此，他以《架构师必知必会系列：物联网架构与边缘计算》为题，撰写了一篇具有深度和思考的专业技术博客文章。

# 2.核心概念与联系
## 2.1 什么是物联网（IoT）？

物联网（Internet of Things，简称IoT），是一个由物理对象、传感器、终端设备、网络以及云计算组成的生态系统，能够收集、存储、处理、交换和通信各种形式的非结构化、弱ly链接的资料。它可以帮助人与物之间建立持续的信息连接，获取信息并实现远程控制。利用物联网，能够实时监测、管理、调控和优化各种物理过程和实体。简而言之，物联网就是把互联网的基础上，引入了物理世界的“智能”设备，使得互联网能够为物理世界中的所有实体提供服务。目前，物联网已经成为人们生活的一部分，如智能手机、电冰箱、电视机、穿戴式设备等。

## 2.2 为什么要搭建物联网边缘计算平台？

物联网的革命性变革和创新，给我们带来了巨大的商业价值和社会效应，但也给企业带来了复杂的技术问题和挑战。如前所述，物联网采用的主要技术包括传感器网络、云计算、IoT平台等，这些技术在边缘计算平台上同样适用。边缘计算可以为企业节省大量的服务器和带宽开销，降低成本，实现资源的高度利用率，帮助企业实现更高的生产效率和成本效益。另外，边缘计算平台还可以为企业提供更安全、可靠和更快速的数据传输，打通物联网与云端之间的鸿沟，保障物联网产品的正常运作。此外，边缘计算平台还可以助力企业将更多的功能和知识下移至边缘，赋予物联网产品更强的响应速度和自主学习能力，让智能设备具备“惰性”，为用户提供个性化、精准的服务。

## 2.3 物联网与边缘计算有何不同？

一般来说，物联网和边缘计算都是用于处理信息的一种方式。两者的核心区别在于物联网关注整个物理空间的连接，而边缘计算则着重于处理局部的感兴趣信息。两者的特色如下：

1. 物联网关注整个物理空间的连接

   物联网需要从多个终端设备收集、处理、储存、传输和分析数据。例如，智能摄像头可以捕捉到摄像头前面的人脸、声音、行为轨迹等信息，并实时传输给云端，进行分析和处理。所以，物联网能够构建一个分布式的网状网络，将物理世界中每一个实体的感兴趣信息收集、传递、分析和处理起来。例如，当一只鸟走近某个空闲的停车位，该停车位的物联网终端设备就会接收到信息，然后将信息发送给云端，在云端运行相应的算法，决定是否准许鸟通过。

2. 边缘计算关注局部的感兴趣信息

   边缘计算处理的都是局部的信息，如路牌信息、物流信息、广告推送、用户习惯信息等。例如，智能城市中的垃圾分类系统可以根据用户的行为习惯和位置信息，结合历史上收费站点的停车记录、道路交通情况、地质风险等多种因素，得出符合用户实际需求的垃圾分类结果。此外，边缘计算平台还可以通过云端和边缘端之间的双向连接，将云端的分析结果实时推送给本地边缘节点，实现设备实时更新、信息共享和智能决策。

3. 物联网设备比较昂贵，边缘计算设备较便宜

   在工业领域，物联网设备价格相对昂贵，而边缘计算设备相对便宜。例如，智能空调每加仑电力消耗约3万元，而相对应的千兆双模光猫则只需几十元。而对于个人用户来说，边缘计算平台不需要购买昂贵的服务器、配套的硬件，就可以享受到物联网带来的服务。

综上，物联网和边缘计算都有自己的优势，并且都必须协同发展，才能推动物联网时代的到来。
