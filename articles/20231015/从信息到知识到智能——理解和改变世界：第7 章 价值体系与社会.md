# 1.背景介绍


价值体系在当代文明史上扮演着重要角色，它起到了一个载体作用，使得人类认识世界、组织生活、作出决策具有共同的参照系。价值观既可以源于个人的天赋品质也可来自于精神层面的内化和形成。这种价值体系已经成为现代社会中不可分割的一部分。每个个体都通过自己的价值观来看待世界，并据此做出取舍和决策，进而影响整个社会的发展方向。因此，价值的内涵及其产生过程、意义的界定成为衡量价值体系是否健全、有效、稳定的重要指标之一。
 
作为学科名词，价值体系可以定义为以人的身份进行研究的跨学科领域。它的发展经历了从认识论到政治哲学再到社会学的演变。现代价值体系所要解决的主题是一个中心主题——人生。在这个主题中，价值观是如何塑造的？价值观对人的自由、尊严、责任和权力权威的约束又是什么？这些问题的答案将直接影响到全球的命运。
 
# 2.核心概念与联系
## 价值观——观念或信仰
价值观，是指一种客观存在的关于某些特定事物的看法或态度。价值观可能是直观的，例如爱、厌恶、友好等，也可能是潜意识的，例如主张激励员工、停止利用腐败行为等。不同的价值观者往往倾向于不同而互相矛盾，但往往也有相通之处，比如热爱生命的快乐主义者一般也会对追求完美的人格保持警惕。基于这样的原因，价值观往往不受个人偏好或利益的影响，并且会被多种因素影响，包括社会环境、文化传统、个性习惯等。因此，价值观是难以摆脱随时变动、复杂多样、冲突不断的影响。
 
## 潜在价值——非物质的价值感知
潜在价值是指个人对某种特定事物的可能性的一种认识。潜在价值所反映出的不是物质上的东西，而是对一个人或一个群体生命意义的认识。潜在价值由内心信念和价值导向行为共同组成，其理想状态就是在完成生命功能后能够带来长远的福祉。潜在价值是为了实现个人目的而认识世界的一种能力。潜在价值理论认为，个人对某件事物的期望值应该与其潜在价值成正比。因此，个人应该不断寻找新的契机来培养自己的潜在价值。
 
## 自由意志——自我选择的能力
自由意志是指人的主观意愿，是指人可以在短时间内或长时间内不受他人的支配而行事的能力。人的自由意志有三方面特征，即选择自由、权力支配自由、让步自由。人的自由意志往往随着年龄的增长和环境的变化而发生巨大的变化。人的自由意志可以是自觉的，也可以是被压抑的。例如，孩子刚刚出生的时候往往无法立刻决定自己该走向哪条道路，也没有什么理由将控制权移交给父母。随着经验的积累，人们逐渐获得越来越强烈的自我意识，并开始更加自觉地为自己选择道路。然而，人们常常会被人性的恶习所左右。

## 人生目标——生命意义的觉醒
人生目标是指人们追求永久幸福和幸福生活的目标。在现代社会，人们普遍需要一种方式来认识并达成自己的人生目标。人生目标可以是宗教的、哲学的、社会学的，甚至是个人的、公司的。目标的设定往往引起人们的思维和行为转变，改变人们的生活方式和价值观。但是，人生目标应当具备明确的目的性和长远的方向，且必须是符合实际情况的。

## 生命权益——包括财富、尊严、名誉、身体健康、安全、幸福和安宁等
生命权益包括生命的物质基础、生活方式和生存条件，以及充满激情、梦想、挑战和创造力的人生活动。这些权益构成了人们日常生活的基本需要。生命权益以人为本，由个人掌握并履行。生命权益是每一个个体的核心需求，也是社会的基础。在现代社会，个人必须具备适用于不同收入水平的基本生活保障。例如，对于低收入者来说，可提供贫困救助；而对于中产阶级来说，则需要有税收减免政策。

## 价值理念——广义的社会共识和共识意识
价值理念是指多元价值观念的总称，是指在大众传播、公共讨论中形成的社会共识。价值理念一般包括各种人生观、伦理理念、价值观等。由于价值理念的宽泛性和包容性，它不仅可以塑造现实生活，而且还具有全局意义。比如，20世纪90年代的启蒙思潮，是以启蒙派价值为代表的社会共识。另外，无论国籍、民族、区域还是阶级，每个人都有自己的价值观念，都认为自己生活在一个有关自我的共同体里，因此就必须遵守共同的价值理念。
