# 1.背景介绍


## 概念介绍
大数据时代正在到来，作为一个领域，它涵盖了各种各样的新兴信息技术、服务模式、新的数据收集方法、大规模数据的处理和分析能力、以及业务模式上的全面变革。在不断增长的数据量和复杂度下，新的需求也越来越多，为此，制造业已经成为大数据领域的一个重要市场。随着数字经济的崛起，制造业的规模和结构正在逐渐向数字转型，大数据在制造业中的应用也越来越广泛。因此，理解和掌握大数据与制造业的关系至关重要。

在本文中，我们将探讨大数据与制造业的相互作用及其对社会的影响。我们将从以下几个方面阐述这一点：

⒈ 数据采集方法
⒉ 数据处理方法
⒊ 计算平台建设
⒋ 模式创新
⒌ 成果落地应用

## 基本概念
### 数据采集方法
在企业中进行大数据应用，通常需要收集海量数据并对其进行整合、清洗、转换等处理。数据采集的方法分为：

1. 传感器采集法：指通过自动化或半自动化设备，定期或实时地采集、监测、记录企业或产品的相关信息。例如，可收集工厂设备的数据，包括工人产量、机器运行状态、工艺流程和生产情况等。

2. 文件传输采集法：指通过网络上传输的文件获取企业或产品的相关信息。例如，可以通过FTP协议或SFTP协议上传工厂生产过程中的图片文件，然后利用图像识别技术对其中的信息进行提取、归纳、分析。

3. 数据库采集法：指采用数据库的存储机制和查询功能，基于已有的历史数据进行分析和挖掘，从而得出企业或产品的相关信息。例如，可以从ERP系统、SCM系统、财务系统等数据库中获取企业财务数据，再结合其他信息进行分析。

### 数据处理方法
对于数据采集到的数据，需要进一步加工、整理、分析等处理，才能得到最终的分析结果。主要包括：

1. 数据清洗：指对原始数据进行缺失值处理、异常值处理、重复值处理、格式转换等操作，最终输出高质量、规范的数据集。

2. 数据分析：指对清洗后的数据进行统计分析、多维数据分析、因子分析等，通过对数据的呈现、模式、关联、特征进行分析，找出数据背后的规律、趋势、价值。

3. 机器学习：指运用机器学习算法对数据进行分析，找出数据中的隐藏模式和规律，实现预测和决策。例如，可以使用分类算法对工厂的生产状况进行预测；也可以使用聚类算法对客户群体进行划分，根据不同群体提供不同的解决方案。

4. 可视化：指通过图形化的方式，直观地呈现数据，让不同部门的同事都能够快速理解和分析数据。例如，可以采用报表形式，将数据呈现给各级管理层，以及工厂各个环节的负责人，以帮助他们更好地掌握生产状况和执行任务。

### 计算平台建设
对于制造业来说，由于信息量很大，而且存在大量的异构数据，如何有效地进行数据仓库建设、数据湖建设、数据中台建设、实时计算平台建设等是一个重大课题。

1. 数据仓库：指存储企业中产生的各种类型的数据，用于支持企业的决策、风险管控、经营改善、业务发展和产品开发。数据仓库的构建一般分为三步：首先确定数据源头（即来自哪些来源的数据），然后定义数据模型、建立完整性约束条件、最后加载数据。

2. 数据湖：指储存和处理企业上百个甚至上千个数据源的分布式存储系统，可支撑企业快速响应变化的商业环境。数据湖的设计原则主要有三点：数据去中心化、数据集中化和数据湖内数据共享化。

3. 数据中台：指构建一个由不同业务部门共同协作的基于云计算平台的数据中台。数据中台的设计原则一般有四点：数据安全、数据集成、数据价值共享、数据治理。

4. 实时计算平台：指根据大数据的实时特征和实时要求，构建实时计算平台。实时计算平台的设计原则一般有五点：低延迟、高吞吐、弹性扩展、自动容错、高可用性。

### 模式创新
制造业也在进行数据驱动的创新，探索更多的模式，创造更多的价值。如：
1. 供应链管理：制造业中的物流管理、供应链协同等，都可以借助大数据的力量，提升效率，降低成本。
2. 电子商务：电子商务中用户行为数据、商品物流信息、订单交易数据等，都可以用于提升客户满意度、促进销售增长，改善市场竞争力。
3. 新零售：新零售模式下，用户消费习惯和产品质量的变化，都需要大数据和人工智能的辅助。
4. 服务外包：服务外包模式是一种以服务提供商为主导，服务者提供解决方案的一种工作方式。服务外包模式下，服务提供商的技能水平参差不齐，数据成为了评估技能水平的最佳依据。

### 成果落地应用
制造业的创新往往带来大量的利润和红利，但是这种创新往往会引发一些问题。如：

1. 数据孤岛：数据孤岛指由于企业的数据只能存在于少数数据源中，无法有效整合。数据孤岛还可能导致分析结果的局限性，影响企业决策的准确性和公正性。

2. 行业壁垒：制造业存在众多企业利益，因此企业之间存在巨大的差距。制造业壁垒还会阻碍数据创新和进步。

3. 强制推行：企业往往存在着各种政策压力，必须严格遵守制定的标准和流程。制造业强制推行会引发不可抗力，影响创新创造的积极性。

综上所述，了解大数据和制造业的关系，理解其背后的机制、原理、法则，对于科技工作者来说尤为重要。只有充分理解其局限性和发展方向，才能够避免踩到“假山空”或者“两山相见”的坎。
