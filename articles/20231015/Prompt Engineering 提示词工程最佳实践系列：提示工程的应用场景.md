# 1.背景介绍


提示词工程（Prompting Engine）是一种自动生成语言模型所需的上下文信息的方法。由于现实中存在诸多任务都需要用到生成语言模型，因此，提出了一种高效有效的解决方案——提示词工程。通过自动生成预训练好的提示词模型，可以在生成语言模型前引入文本结构化的信息，提升生成质量并节省资源。提示词工程的应用范围十分广泛，主要用于中文、英文、日语等自然语言处理领域。在搜索引擎、知识图谱、自动问答、对话系统、评论观点抽取、机器翻译、机器人回复等方面均有重要的应用价值。
提示词工程有什么优势？提示词工程的优势在于：
- 提供结构化信息可以避免生成偏差，提升生成效果；
- 通过引入有意义、相关且重要的信息，可以使模型生成更准确、客观、更有信息量；
- 可以降低数据集成成本，大幅度减少标注成本；
- 在做分类、回归任务时也可以提升精度。
# 2.核心概念与联系
提示词工程共分为三层结构，包括：全局层、局部层和领域层。其中，全局层是在全局语境下生成，例如一些关键词、实体等；局部层是指在特定片段或子句下的生成，它会融合全局层的信息以形成更加完整的句子或者短语；领域层则是在一个领域（如电影、音乐、科技等）内的生成，它可以基于特定领域的语义进行生成。
下面我们结合案例分析一下这三层之间的关系。
## 案例分析：零度制简化
假设有一个数字“1234”，零度制表示法为：“十二点三四”，而将零度制简化为最简形式“十二三四”的过程就是典型的提示词工程的一个应用场景。提示词工程首先把整个数字作为全局层信息，然后再根据局部层的位置进行上下文切割和替换，得到最终简化后的形式。
### 全局层信息
“1234”作为全局层信息，即整个数字1234。
### 局部层信息
对于零度制简化来说，除了要考虑数字1234外，还需要考虑当前数字1的位置。如果当前数字1在奇数位，则不需要进行任何转换；如果当前数字1在偶数位，则可以通过“零”字消除其位置，得到“十二三四”。因此，在数字1的位置上，有两个局部层信息。第一个局部层信息是“第……位”，第二个局部层信息是“点零”。
### 领域层信息
由于这个任务涉及到的领域范围非常广，所以无法详细讨论。这里仅给出一个例子——在电影中简化零度制。电影中通常会有四种符号表示零度制，分别为：零、一、两、无。因此，在电影中简化零度制可以基于电影的类型、情节、语言特色等进行判别。
综上所述，该案例的全局层只有数字1234，局部层有第几位和点零两类，并且没有需要具体的上下文信息。但是，通过给出特定场景的参考信息，提示词工程可以帮助生成模型提供更加准确、完整的描述信息，从而提升生成语言模型的效果。
