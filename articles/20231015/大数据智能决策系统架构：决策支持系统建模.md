# 1.背景介绍


## 概述
决策支持系统（Decision Support System, DSS）是指基于信息技术、管理科学、人工智能等综合性知识的人机结合的信息系统，其目标在于促进管理决策活动，使决策更加科学、透明、有效、及时并提升决策效率。DSS通过收集、整理、分析、整合数据资源、运用人工智能、规则引擎等技术手段对各种复杂的数据进行快速分析、处理、归纳和推理，最终得出有价值的信息并作出决策建议。DSS的应用范围广泛，涉及银行、保险、制造业、教育、电信、金融、医疗、公共政策、交通等各个领域。近年来，随着大数据的快速增长、深度普及和日益成为主要经济社会生产力，基于大数据的决策支持系统已成为管理人员必备的工具。但是，如何构建和部署高质量的决策支持系统却面临着巨大的挑战。本文将系统阐述大数据智能决策系统架构，主要包括决策支持系统的概念、分类、设计方法、以及关键技术、组件和框架。 

## 定义
决策支持系统，又称决策支持工具、决策支持应用或决策支持服务，是一个基于计算机技术和自然语言处理的应用软件，它利用计算机和人工智能技术从大数据资源中抽取信息，辅助管理者做出决策。这种应用软件的目标是基于历史数据、统计数据、经验数据、现实世界数据，或者基于客户反馈、市场调研、产品销售记录等各种形式的数据，基于决策模型，进行精准的预测、判断和决策，为管理者提供科学、高效、可靠、及时的决策支撑。

## 类别
根据其功能特点和应用领域，决策支持系统可以分为三类：

1. 数据仓库决策支持系统: 数据仓库决策支持系统（Data Warehouse Decision Support System, DW-DSS）是一个集成数据仓库、多维分析以及数据库分析能力、人工智能和业务规则引擎等技术为一体的系统，能够帮助组织提高决策能力、改善决策过程、减少风险、提升效益。DW-DSS通常包括元数据描述、事实表、维度表、计算视图、机器学习算法以及决策规则等组成部分。此类系统适用于对事务性和历史性较强、多样性较高、依赖时间序列数据的决策需要。

2. 复杂事件决策支持系统: 复杂事件决策支持系统（Complex Event Decision Support System, CEDS）是指在特定场景下进行复杂事件处理和评估的一种系统。CEDS 可用于分析和预测发生在组织外部的复杂事件，并根据分析结果采取相应的行动。CEDS 将复杂事件的整个过程，包括触发、传播、监控、分析、反应等环节，考虑在其中作用一系列因素，如政策法规、操作技术、人员掌握情况等，并最终给出决策建议。

3. 工业控制系统决策支持系统: 工业控制系统决策支持系统（Industrial Control System Decision Support System, ICS-DSS）是指基于复杂工业系统运行模式和条件的实际情况，对控制系统的性能进行综合评价、预测和优化的一种系统。ICS-DSS 的核心技术是模型化、仿真、控制、自适应、优化以及自动化，目的是为了实现对工业控制系统的智能化管理。ICSDSS 的目标是在不牺牲系统安全性、效率、精确性的前提下，最大限度地提升管理效率和效果。
