# 1.背景介绍


## 1.1 什么是图论？
图（graph）是由顶点和边组成的集合。顶点称为节点或顶点、点；边代表两顶点之间连接的线段或直线，它通常表示某种关系。例如，一张无向图中的一条边代表两个顶点间存在一条边，有向图中则用箭头表示方向。图可以用来描述各种复杂结构、模式及其之间的联系。图论是计算机科学的一个分支，与数据结构与算法密切相关。图论经常用于网络路由、电路设计、通信网络设计、生物信息学等领域。

## 1.2 为什么需要图论？
图论是一门十分重要的数学科目，是理解复杂系统的一把利器。在现代计算机科学中，图论应用十分广泛，包括网络路由、网络优化、图形图像处理、文本分析、生物信息学、信号处理、金融市场分析、控制系统、心理学等。图论也可以用在很多其它领域，如图像识别、数据库查询、生物统计、路径规划等。

图论主要研究如何从复杂的网络中发现隐藏的模式、识别重要节点，还可以用于计算网络效应、评估社会网络结构、预测政权更迭、测定交通拥堵、探测食源分布等等。同时，图论还给出了许多数学上的精巧结论及证明方法，这些知识对于构建高效的机器学习模型、优化电路设计、规划交通流量都很有帮助。

## 2.核心概念与联系
## 2.1 概念
### 2.1.1 图
图G=(V,E)是一个包含n(n≥0)个顶点集V和m(m≥0)条边的有穷集合。每个顶点v∈V对应一个元素e(v)和相应的函数f(v)，称为顶点属性或者度量。顶点e(v)代表着顶点v所代表的实体或对象，比如图中一个人或结点。函数f(v)可以定义v到其他所有顶点的距离，也称为度序列。度序列是一个整数序列，其中第i个数记录的是顶点v的度，记做d(v)。对于任意顶点u∉V，若存在一条路径p=vu-...-w，其中v,w∈V且pi+1=wi，则称该路径为u关于v的路径。路径长度path length(u,v)等于路径上边的数量。

### 2.1.2 邻接矩阵
邻接矩阵是一种表示图的方式。矩阵的每一行代表一个顶点，列代表它的相邻顶点。如果两个顶点之间有边相连，那么就将对应的元素置1，否则置0。
$$A=[a_{ij}]$$

其中a_{ij}代表顶点Vi和Vj之间的边，如果Vi和Vj之间没有边相连，那么a_{ij}=0; 如果Vi和Vj之间有边相连，那么a_{ij}=1.

### 2.1.3 有向图
有向图G=(V,E)是一个包含n个顶点集V和m条边的有穷集合，其中每条边(u,v)∈E都有一个方向。有向图中，边(u,v)表示从顶点u指向顶点v，并且有方向。通常用符号<u,v>来表示一条有向边(u,v)。

### 2.1.4 简单图与不相交子图
设G=(V,E)为一个图，如果对任意两个顶点u,v∈V和任意的三个顶点s,t,x∈V满足:

1. (u,v)∉E 或 (v,u)∉ E
2. (s,t),(t,x)不是同一条边

那么我们说G是一个简单图。设G的一个不相交子集S={(u_1,u_2,...u_k)|1<=i<=k<=|V|}，其中每一个u_i∈U(V,E)。那么S就是G的一个不相交子图。

### 2.1.5 完全图
设G=(V,E)为一个图，当且仅当存在顶点u,v∈V使得(u,v)∈E。那么G就是一个完全图。

### 2.1.6 子图
设G=(V,E), H=(V',E')为两个图。如果H的所有顶点都是G的顶点，并且H的所有边都是G的边，那么H就是G的一个子图。

### 2.1.7 核与生成子图
设G=(V,E)为一个图。G的核是指G中所有的极小子图。G的生成子图是指G中任意添加一条边而仍然是连通的子图。

### 2.1.8 邻接表
邻接表是一种实现图数据的常用方式。在邻接表中，图用一张链表数组表示。数组的下标i表示顶点的编号，每一个元素都是一个链表，保存了与该顶点相邻的所有顶点的编号。这样的话，查找任意两个顶点之间的边只需访问它们所在的链表就可以完成。另外，为了避免同向边重复存储，邻接表一般只存储非重边。

## 2.2 联系
### 2.2.1 路径与路径长度
路径是图的一种重要的性质，也是图论最基本的研究对象。设G=(V,E)是一个有向图。若存在一条从顶点s到顶点t的路径，即存在一个顶点序列<s,v_1,...,v_k,t>, 其中v_i,i=1,2,...,k∈V，并且(vi,vi+1)∈E, i=1,2,...,k-1。那么路径(s,t)就被称为G的一个路径。路径长度path length(s,t)表示从顶点s到顶点t的最短路径，记做l(s,t)。

### 2.2.2 连通性与弱连通性
设G=(V,E)是一个图。若G中任意两点之间都有路径，则称G是连通的，记做C(G)=1；否则，称G是不连通的，记做C(G)=0。设G'=(V',E')是一个图，其中V'=<v_1,...,v_m>, m≤|V|, E'=<e_1,...,e_n>, n≤|E|，且E'中没有自回路或平行边。如果从任意两个顶点u,v∈V'都存在着一条路径(u,v)，则称G'为弱连通的。

### 2.2.3 强连通分量与割点
设G=(V,E)是一个连通图。G的强连通分量是一个子集T'，它是G的一个非空真子集，而且G中存在着至少两个顶点u,v∈T',使得从u到v存在路径，但不存在其他任何顶点属于T'。T'就是G的一个强连通分量。

设G'=(V',E'), V'<V, E'<E，且G'是一个连通图。G'的一个割顶点是一个顶点v∉V', 它不是G的任何顶点，并且不能通过去掉它和它所关联的边而得到一个不连通的子图。对于给定的割点v, 可以找到一个割顶点u∈V', 使得(u,v)∈E'. 这两个顶点构成了一个割。G的割集是指G中所有割。
