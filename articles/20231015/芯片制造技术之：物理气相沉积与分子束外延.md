# 1.背景介绍


电子芯片(Chip)是集成电路的一种制造工艺。一个完整的电子芯片通常由微型功能块或称为晶体管单元组成，这些晶体管单元被堆叠在一起形成逻辑电路板，并通过一个集成电路互连器连接到主板上的其他部件上。在制作晶圆电路时，设备制造者需要注意解决物理气相沉积（PS）和分子束外延（PE）的问题。PS问题是指晶圆电路中物质的浓度高于其表面张力所能够承受的极限，导致PS凝胶或者合金表面无法覆盖整个晶圆层。PE问题则是指电路板上某些区域由于太阳光照射导致的自由空间传播，从而导致信号损失或者信号反馈过程异常，严重地影响设计的功能。

# 2.核心概念与联系
## 物理气相沉积(Physical Segregation)
物理气相沉积（PS）是指物质的浓度高于其表面张力所能够承受的极限，导致PS凝胶或者合金表面无法覆盖整个晶圆层。因此，物理气相沉积会阻碍电路中的信号传递，进而影响整机的性能。如图1所示，在制造晶圆电路时，如果晶圆表面存在高浓度的物质如冷极、湿极、氢化氮等，电路就容易受到PS影响，而这种情况往往发生在制作高低温电压时，因为这些高浓度的物质导致电路层之间的PS现象尤为明显。

## 分子束外延(Particle Elongation)
分子束外延（PE）是指电路板上某些区域由于太阳光照射导致的自由空间传播，从而导致信号损失或者信号反馈过程异常，严重地影响设计的功能。PE主要是由于摩尔体晶体管所含有的电子在不同温度下的受热程度不同而引起的。如果晶体管内部存在异常大的电子，或者有一部分电子分子束膨胀，就会产生波纹的效应，进而影响电路的功能。如图2所示，当电子布置的地方离晶圆边缘过近的时候，就会出现分子束外延。

## PS/PE模型
为了更好的理解PS/PE问题，科学家们提出了PS/PE模型，该模型可将晶圆内的晶体管进行分类，并用理论公式计算出每种晶体管在特定温度下的有效工作温度（eta）、电流密度（j）和设备发散指数（D）。根据这个模型，设计人员可以对每种晶体管类型的电路板进行调整，降低其PS/PE现象，从而提升设备的功耗和稳定性。

## 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
一般来说，制造电子芯片时需要完成以下几步：

1. 晶圆模具选型和布局设计
2. 晶圆材料和导电层的选择
3. 晶体管的设计
4. 集成电路的设计
5. 集成电路封装和测试验证

其中，第3步和第4步是最重要的两步。这两步涉及到电路设计的关键环节。

电路设计的核心思想是利用高性能集成电路单元构建出电路板，在构建电路板的过程中，晶体管的PS/PE问题往往成为难点。制作电路板时，设计人员可以采取以下措施：

1. 用各种各样的导电层掩盖晶圆表面，保证物质和电路之间的区别。
2. 在导电层的四周留空出一定的间隙，避免晶圆中物质的干扰。
3. 将晶圆中高浓度的物质如冷极、湿极、氢化氮等排除掉，不至于影响晶圆中所有晶体管的正常工作。
4. 使用可逆变复印件（MIG）结构，实现晶圆单元自毁保护。

还有一些特殊情况下还需要考虑其他措施，比如在制作集成电路的时候，还需要考虑集成电路的隔离和设计电源电路。但总的来说，制作晶圆电路的基本流程就是这样。

对于PS问题，最简单的做法就是不要让晶圆表面有高浓度的物质，只要保持晶圆的浓度在合理范围即可，也可以用氩气(CaCO3)填充或冻结住晶圆。对于PE问题，可以使用扩散性材料如PTA来缓解。
