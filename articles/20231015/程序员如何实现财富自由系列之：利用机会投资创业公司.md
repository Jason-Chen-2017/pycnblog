# 1.背景介绍


近几年，随着互联网金融、网络游戏等技术的飞速发展，新经济领域已经成为人们的必经之路。而大量的创业者开始面临难以企及的财务困境，因此越来越多的人选择了投资机会投资创业公司。但投资创业公司到底能否实现财务自由呢？在投资过程中会遇到哪些坎险？如何才能找到那些值得关注的创业公司并投资进去？这就需要一个技术专家深入分析此类问题，为投资者提供有用的信息。
本系列文章将从“投资创业公司”这一切入点，逐步向读者介绍投资创业公司的基本原理、核心概念和操作流程、投资中常见的风险、如何发现值得投资的公司、投资收益如何计算、投资成功后的收益分配方式，以及投资创业公司的未来发展方向与展望。希望通过阅读本系列文章，能够帮助读者更好的理解投资创业公司这一过程。
本文将以创业公司的“起源”—互联网金融为切入点，介绍投资创业公司的各种相关知识点。而后继续介绍一家互联网金融公司——米粉钱包的诞生背景、核心理念和投资策略，并结合技术层面的角度阐述其投资逻辑和细节，最后介绍一些投资者的心声和投资感受。希望能够通过这样一个有趣的故事，引导读者从入门到精通投资创业公司的各个环节，提高对投资这个“大道”的理解能力和掌控力。
# 2.核心概念与联系
## 2.1 投资创业公司
投资创业公司作为新经济中的一种新型创业模式，可以说是“知识界泡沫”的最早发端。早期的创业企业往往只是小项目，如手机制造商、家电制造商、农产品批发市场等等；而后来，更多的创业者开始进入创业公司的领地，大公司或政府部门也加入其中，成为了创业的先锋，如腾讯、阿里巴巴、美团、拼多多等。
但是，这些创业公司的募资均来自于“风险投资”，即以极低的起投金额购买风险资产，投资者往往无法预测投资组合的收益率，不知道投资到什么时候才有回报，容易被大的投资集体套牢。因此，如何保障创业公司的资本运作，抵御投资的风险，成为当下很多创业者的痛点。
投资创业公司的定义就是“以创业的方式进行投资”。因此，本文将以创业公司作为切入点，介绍其相关的知识点。
## 2.2 机会投资
所谓“机会投资”，就是从市场机遇出发，寻找具有巨大潜力且缺乏历史做基础的企业，买入股票进行长线投资，以期获取远超市场平均利润的回报。
“机会投资”这一概念的提出源于企业经营管理的一贯原则——不要等到市场成熟时再投资，而应该在竞争激烈、客户资源紧张、市场形势恶化时及时买入。
虽然市场上的大多数企业都遭遇了短期内的失败，但是在大周期轮动中仍然能够获得足够的优势。另一方面，由于创业公司的“小幅投入”，资金短期内很容易转化为实物价值。
“机会投资”不仅能获得高回报，而且能够获得较长的持续时间。在互联网金融领域，这种方式占据主导地位，绝大多数创业者都选择投资创业公司。
## 2.3 创业公司的投资要求
投资创业公司一般都有以下几个要素：
- 资金：创业公司通常都会通过募集阶段筹集一定数量的资金，用于承接前期的融资需求。而后期的资金主要来源于天使投资或创业者个人投入。
- 投资目标：创业公司的投资目标往往很明确，如盈利增长、规模扩大、品牌推广等。一般情况下，创始人的理想是通过自己的努力实现财务自由。
- 时间：创业公司的发展时间一般较短，往往在半年至一年之间完成，需要勤奋和坚韧。
- 成功标准：创业公司成功的关键取决于创始人的能力、智慧和执行力。但一般来说，成功的创业公司需要具备以下特点：
  - 创始人的努力：创业初期的努力可能并非大不了的，但一旦进阶到了某个阶段，创始人往往要承担起越来越重的责任。
  - 没有垄断地位：创业公司的存在不意味着占有独特的市场份额，而是在竞争中生存。
  - 有净盈利能力：创业公司必须能够产生长期稳定的净利润。
- 管理模式：创业公司往往采用市场化运作模式，通过战略部署、管理层筹划、分包合作等方式将资金和资源调配给合适的岗位。不过，由于管理复杂性、职业经验缺乏、资源分布散乱等因素，创业公司的管理经常会出现问题。
- 投资风险：投资创业公司的风险主要来自于投资公司的控制力度，以及创始人的投资决策。投资者应注意保持开放心态，尤其要注意创业公司的投资政策变化、信用状况变化、行业变革等事件。
## 2.4 创业公司的融资情况
目前，创业公司的融资模式多种多样，包括股权众筹、货币基金、债券、房地产等。下面我将以“腾讯视频”的融资情况为例，介绍创业公司的融资情况。
### 腾讯视频的融资情况
腾讯视频是中国最大的视频直播平台，其创始人李彦宏曾经透露过他的计划：通过海外的视频网站得到成熟的用户群体，然后将视频网站的内容做成视频直播服务，从而帮助用户通过互联网观看自己喜欢的电视节目。
当时的李彦宏虽然很看好腾讯的发展潜力，但考虑到腾讯内部的政治斗争正在升级，加上美国对中国视频网站的审查、法律风险等因素，导致腾讯将投资的优先级降低。
腾讯一直在寻求外部资金支持，包括腾讯和其它创业公司的股权投资、房地产开发、技术合作等，但仍然无法取得突破。最终，腾讯视频未能获得足够的资金，只能靠内部人士出资来维持运营。
### 小结
创业公司可以说是“知识界泡沫”的最早发端，但在创业公司进入市场之后，管理的复杂性和内部矛盾让创业公司面临巨大的风险。投资者应当珍惜创业公司的资金、市场资源，充分关注创业公司的投资情况，并认真研判创业公司的未来发展。
