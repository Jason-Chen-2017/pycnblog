# 1.背景介绍


## 1.1什么是影视制作？
电影、电视剧、动画、漫画的创作者称为影视制作。简单的说，就是把想法用电脑软件或者DVD动画软件制作成影片或视频，制作出来之后发布到网络上，让观众可以观看和下载，获得一定的收益。

## 1.2为什么要参与影视制作？
现在很多人都喜欢看电影、看美剧。但其实如果不出门看一下自己的生活的话，很难理解为什么需要去参加一些短时间内的电影节目。所以，对自己的生活习惯、观点进行改革开放，把自己融入世界的进程，也是非常必要的。

作为一个程序员，如果你也喜欢编程，那么在工作中接触到了影视制作这个领域，那就可以考虑加入到这个行列中来。即使没有什么经济收入，也能通过参与到电视剧、电影等相关项目的制作中，提升自己的能力。

## 1.3电影和电视剧的制作过程及不同之处
电视剧和电影的制作流程基本相同。下面简单地介绍一下两者的不同之处。

### 1.3.1电视剧制作
电视剧制作的核心流程包括：
- 摄制：通过摄像机拍摄场景、人物、镜头等，摄制人员还应当准备好反差较大的光源、助手、机器，以及精湛的音效调音师。
- 配音：配音是指由演员扮演角色，将情感表达给观众听。配音师则是负责将演员的声音呈现出艺术效果的专业人才。
- 预告片：电视台根据预告片的内容制作片尾曲和片头，并检查片子是否符合播放要求。预告片一般需要两周左右的时间才能完成。
- 拍摄：包括选取场景、配置摄像机、选择道具、拍摄手法等。
- 播放：在电视台的播放场所安排片子、播放时长、准备投放设备，播放音轨等。播放后观众会评价影片的整体效果和故事性。
- 发行：电视台根据播映情况决定是否实施制片费用，并且检查电视节目是否具有打斗、暴力、色情、粗俗、色情、过于激烈的元素。
- 分销：电视台和其他分销商进行分成制作后的资源，并交付给广大消费者。

### 1.3.2电影制作
相比电视剧，电影制作的流程更加复杂，更需要投入更多的人力、物力和金钱。

下面简单描述电影制作的流程：
- 搜集资料：收集电影的剧本、镜头、特殊效果、编辑、制片等资料。电影制作通常需要四五个月才能完成。
- 摄制：整个制作过程从摄像、后期、特效、制作等环节都需要高科技的设备支持。
- 导演：导演的作用主要是统筹整个电影制作流程，审查影片结构、剪辑剧本，监督导演。他还需负责将故事的情节讲述清楚，并协调各方面的配合。
- 主演：对于电影来说，主演至关重要。每部影片都得有一个鲜明的主角，而这项任务就由演员完成。每个演员的表演力、演技、气质都会影响电影的成败。
- 编剧：编剧的任务是负责对影片进行编辑，删减冗余内容，为影片提供主题线索，提高整体效果。
- 场记：每个演员的出镜时间都不同，所以需要编排好剧中的场次，确保所有演员均能充分发挥才能达到既吸引人又精彩的效果。
- 故意错漏：故意的漏洞可能导致整个故事出现偏离。编辑人员也应该注意到这一点，做到不留一点瑕疵。
- 后期：完成主角剧照、特效处理、颜色修饰、天幕布景、文字等各种后期处理。需要专业人员参与配合，时常保持跟踪。
- 导演美术：涉及电影的主题和风格，需要的画面和素材都可以在这里找到。除了画面外，他们还需要负责照顾电影院的环境、对电影室及屏幕进行维护，确保每部影片都能够正常播放。
- 拍摄：将导演制作好的素材精心录制成一段段真实的镜头，拍摄者需要精湛的技术和功底。录像仪器、摄像头、配乐设备、单反相机、洗印设备等都要有专业的知识和设备才能胜任此项工作。
- 录像制作：每部影片都需要经过精心的记录，导演们需要为影片选取和使用最佳的音响、胶片、光盘等，以保证影片的完整和准确。
- 分级制：经历了多年的制作过程，某些影片可能会被制作成特殊版本，例如《悬崖上的金鱼》、《大卫贝克汉姆》等。这一步是由制片公司根据国家政策规定分级制的方案进行的。
- 发行：影片所有的前期工作都已经完成，最后一步就是向观众推出最新的电影。这其中还需要分期发行，比如第一期只推出几十万张票，第N期再补上几百万张。

# 2.核心概念与联系
## 2.1数字化的进程与历史回忆
数字化是一个长期的历史进程。从纸张记录到计算机的出现，数字化的进程有许多的阶段。数字化的进程与历史回忆息息相关。下面介绍几个关键的历史节点。

### 2.1.1第一个数字计算机——ENIAC
ENIAC是一个巨型的二进制计算机，由英国科学家艾隆・图灵提出。它采用双稿纸机、六条晶体管和皮尔逊电磁转移计数方法（Pentium）运算。它是一个非常复杂的电子计算机，可以进行矩阵乘法、计算圆周率、搜索引擎、人工智能等各种复杂运算。

### 2.1.2世界第一个web浏览器——NCSA Mosaic
Mosaic是第一个web浏览器，它诞生于美国宾夕法尼亚大学。Mosaic的第一个版本是在1993年发布的，它的功能比较简单，后来逐渐增加了一些特性。

### 2.1.3计算机辅助设计——CGI脚本
CGI（Common Gateway Interface，通用网关接口），是一套服务器端技术规范，它定义了Web服务器与浏览器之间的数据交换方式。Web服务器可以通过CGI脚本动态生成Web页面。

### 2.1.4互联网的兴起——ARPANET
ARPANET是一个早期的互联网，它位于美国旧金山的阿帕网上。它主要用于军用通信，但也提供了一些研究和试验平台。

### 2.1.5互联网的普及与应用——WWW
WWW（World Wide Web，全球互联网）的诞生标志着互联网的爆炸式增长。它是一个采用HTTP协议的基于超文本的网络服务，所有信息都通过超链接链接起来。

## 2.2计算机图形学与建模技术
计算机图形学是计算机科学的一个分支，它研究如何利用计算机进行视觉、图像、虚拟现实、动画、可视化等三维信息的渲染与计算。它围绕着两个主要的技术——图像处理与建模技术展开。

### 2.2.1图像处理技术
图像处理技术主要研究如何识别、处理、存储、显示和传输数字图像。它包括：
- 数码相机与摄像头
- 图像压缩技术
- 图像处理算法
- 图像模式识别

### 2.2.2建模技术
建模技术旨在使用计算机建立模拟对象、系统和空间，并对其进行建模、分析、设计、实施和测试。它包括：
- CAD（Computer Aided Design，计算机辅助设计）
- 数值模拟
- 可视化与可编程渲染技术
