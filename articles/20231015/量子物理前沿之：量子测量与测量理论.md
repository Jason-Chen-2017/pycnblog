# 1.背景介绍


“量子”这个词语近年来备受关注，各种关于量子的科普书籍、期刊如雨后春笋般冒出来，人们对这个“神奇的存在”的探索已经进入了高潮。随着量子技术的迅速发展，人们对量子物理学的理解也日渐深入，这其中就包括量子测量理论（quantum measurement theory）这一重要分支。

量子测量理论是指利用量子力学来研究信息传输、信息处理及信息存储等量子过程的本质。它从观察量子态和测量结果之间建立起量子信道的基础，并运用统计物理的方法求解量子信道的各种性质。量子测量理论是量子技术发展的一个里程碑性理论，其开创性成果被誉为“量子计算机”的基石。

量子测量是对“量子态”进行测量的一门学术研究。量子态通常是一个具有两个以上特征的物体或系统在某个特定的时刻的量子比特的纠缠态（superposition）。在量子计算中，量子态经历了各种纠缠转换，而量子测量则用于检测、分析和处理这些纠缠。许多量子算法和量子计算模型都是通过对量子态的测量来实现。

量子测量理论与物理学的其他领域如电动力学、化学、生物学不同，它处于复杂的量子力学研究范围之外。因此，对于非专业人员来说，学习量子测量理论仍然需要一定的认识基础和能力提升。


# 2.核心概念与联系
## 2.1 量子态
量子态可以理解为一种纠缠态，即处于两个或多个量子态之间并不稳定地相互作用下所形成的混合态。通常情况下，一个量子态是由一组具有微弱张力的粒子构成的，这些粒子本身又高度耦合，无法单独存在；但若把具有相同粒子数量的不同粒子组合起来，就构成了一幅宏观上的分布态。

量子态描述了物质世界的“自然”状态，所有物体都处于量子态中，比如一个刚好跳跃了一下的小球，则该小球所处的位置、速度、加速度、动能等都是固定的，但它们同属于量子态。

## 2.2 量子态转移
量子态转移是指两个量子态之间的某种变换。在任何时刻，一个量子态只能处于纯态、单重态或多重态中的一种，一个态只能从另一态转变到自己的态，这是因为根据量子力学的基本假设，一个量子态只能由两个或者更多能量不同的粒子相互作用所形成。而态的转移就是两种不同的态由于特定的因素（可能是外界影响）而发生变化的过程。

量子态转移通常可以分为两类：量子门操作和非线性操作。量子门操作指的是直接作用在量子态上的操作，即一些特定的操作对量子态产生了直接的影响，但这种影响并不会引起量子态的混合。比如：Pauli门、Hadamard门、S-门等。而非线性操作则更为复杂，可能会导致量子态的混合。比如：Werner状態的生成与湮灭，以及测量等。

## 2.3 测量
测量是指从已知的量子态中提取信息的过程。一个量子系统的测量通常包括两个阶段：准备和测量。

准备阶段:首先制备好测量所需的各种设备，如光源、偏振器、读数器等。然后让系统处于准备好的态，即将其全部纠缠在一起。准备好的态可以是任意一段时间内系统处于的态，也可以是一定时间后的量子态。

测量阶段:然后使用测量工具对系统的量子态进行测量。测量工具的工作方式一般是利用信息论的观点，即从量子态的空间或时序上寻找特定模式（比特串）来获得信息。当测量完成后，得到的测量结果会告诉我们系统处于何种态。比如，假设我们想要测量一个含有两个量子比特的量子系统。若测量结果为[01]或[10]，则系统的状态必然在第一个量子比特处为|0>态，第二个量子比特处为|1>态，这时便可确定系统处于那些态中。

量子测量理论中的测量方法主要有两种，即变分法与密度矩阵法。

### （1）变分法
变分法是量子力学中常用的测量方法。它涉及测量对象的参数方程，将量子系统在不同参数值下的量子态分解成一系列矩阵构成的波函数向量。在各个参数值的波函数向量之间做积分，就可以得到给定的参数值对应的测量结果。对于一个含有n个量子比特的量子系统，变分法所得的测量结果会对应着一个n位字符串，每个字符串分别对应着两个量子态中的一个量子比特。比如，假设测量得到的结果为"00010110",则该系统的态可能为 "|psi_1><psi_2|" 或 "psi^+*psi' |psi>|psi'",其中 psi 和 psi' 分别代表两个量子态，<psi| 和 |psi'> 分别代表系综矢和克崔奎矢表示的向量投影。

变分法测量的优点是简单易行，只要知道系统的量子态即可，不需要进行严格的算符演算。但是，对于一个含有n个量子比特的量子系统，每进行一次测量就会消耗O(n^2)的时间，这对于实际应用来说显然是不可接受的。而且，即使采用精确的微扰实验，测量得到的结果也是不可避免地受到噪声的影响。因此，变分法在实际应用中得到的效果往往很差。

### （2）密度矩阵法
密度矩阵法是量子力学中另一种常用的测量方法。它的基本思想是基于统计力学的观点，通过求取系统处于不同态的概率分布来确定系统的状态。

将系统的任意一段时间内的量子态视作一个概率分布矩阵，即P = rho^(t) = \sum_{x,y} p_i(x, y)\left|x\right><y|，其中rho^(t)表示系统处于态x的概率分布。那么，可以通过求解由rho^(t)决定的量子态概率分布方程，来得到系统的整体状态。具体而言，我们定义测量结果为 mi=f(|m>)，其中 m 为已知量子态，而 f 是从密度矩阵导出测量结果的函数。可以证明，对于一个含有n个量子比特的量子系统，测量结果 mi 会对应着一个 n 维空间中的 n 个坐标。通过调整 mi 中的坐标，就能改变系统的整体状态。

密度矩阵法的优点是计算量小，效率高，不需要进行严格的算符演算，测量结果精确无误。但其局限性在于，只能确定系统处于某一特定的态，而不能完全解读系统当前处于哪种态。而且，由于测量量子态所产生的纠缠，密度矩阵法所得到的量子态实际上只是局部的混合态，而不是真正意义上的量子态。

综上所述，量子测量理论涵盖了量子态的概念、量子态转移、量子态的测量、测量的分类以及两种常用的测量方法——变分法和密度矩阵法。对于量子技术的发展而言，测量理论与量子通信的重要性日益凸显，是构建量子通信系统的关键。
