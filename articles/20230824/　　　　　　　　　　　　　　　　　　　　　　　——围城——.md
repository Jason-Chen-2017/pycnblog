# 1.简介
  

围城(Pugong)是个很火的游戏。它的出处就像魔兽世界一样，是在魔兽争霸3中出现的一种野怪兽。它看起来十分的凶猛，攻击力也相当高。游戏中的围城与普通的野怪兽不同，它对敌方的进攻方式更加“狡猾”。比如说，它会第一时间进行先攻，然后再采用围城的方式进行攻击。围城还会随着敌人的血量逐渐减少而收缩，并且能够“压制”单位，使其无法行动，直到自己的体力耗尽。此外，围城在攻击时还会不断产生干扰波，令敌人感觉到恐惧。因此，围城在魔兽世界中已经占据了一定的市场。

围城也是魔兽世界里的一款神奇宝藏游戏。通过收集经验点，玩家可以增加他们的强度、速度、生命值等属性。这种能力可以让玩家作为一个英雄保护周围的人民和平息战争。另外，玩家可以通过自己的技能和道具获得各种好处，比如恢复自身或他人生命值、减速、冻结敌人等。这些成就可以在游戏中的榜单上展示出来。这些因素使得围城游戏在过去几年里快速发展起来，在全球范围内引起了轰动。但同时，围城游戏也存在一些问题。如今围城已经成为一款比较耗时的游戏，而且随着游戏时间的推移，玩家积累的经验和荣誉也越来越多。所以，我们需要开发一款新的围城游戏，来改善用户的游戏体验和游戏性。
# 2.游戏规则
围城游戏是由两名玩家控制的一个团队模式。玩家可以选择攻击某个敌人（单人）或者进攻（两个人一起攻击）。围城的目标就是抓住对手的每一个兵刃，并用自己的兵刃将其打死。对于普通的普通兵刃来说，围城提供了五种不同的攻击模式，每个攻击模式都可以造成不同的伤害。攻击模式包括：

1. 普通攻击（普通攻击）：普通攻击是最简单的攻击模式。它直接对敌人造成等同于自身攻击力的伤害。

2. 重装步枪（Mortar）：重装步枪可以对多数兵刃造成较大的伤害，但对特种兵刃效果不佳。

3. 油门炮（Turret）：油门炮可以释放出一系列的连续子弹，造成较大数量的伤害。但是攻击距离只有自己一人可以射击。

4. 自爆炸（Self-Destruct）：自爆炸可以令敌人变形，并对附近的敌人造成较大伤害。

5. 撞斗士（Shock Trooper）：撞斗士可以对敌人的肚脐造成剧烈的冲击，但对普通兵刃无效。

除此之外，围城还具有一项特殊技能，叫“嫁祸”，这个技能可以让队友被围城所杀。其他的玩法也不胜枚举。围城的目标就是抓住对手的每一个兵刃，用自己的兵刃将其打死。每回合玩家只能选择一件事情进行。比如，攻击，防守，或者使用技能。除了普通攻击，其他的攻击模式都有自己的特点，且都要在攻击距离内才能发动。由于攻击方式的复杂性，围城游戏也有许多的策略。比如，不同兵刃的攻击角度不同，敌人可能会根据不同的攻击模式调整反应，甚至会陷入混乱。最后，围城游戏没有规则限制，可以随心所欲地玩耍。

# 3.设计方案
围城游戏是一个多人在线的角色扮演类游戏。因此，我们需要设计一个可以在网页端运行的基于WebRTC技术的多人游戏系统。游戏系统的主要功能如下：

1. 游戏互动性：游戏系统应该能够提供丰富的游戏玩法，玩家可以通过拖拽旋钮进行攻击，而不是输入键盘指令。

2. 网络同步：为了实现游戏互动性，游戏系统需要与服务器保持实时通信，使得双方的状态及行为能够同步。

3. 用户界面美观：游戏系统的UI设计应该清晰美观，有图标化风格。

4. 技能系统：游戏系统需要有一个完整的技能系统，玩家可以使用技能提升自己的属性和能力。

5. 声音效果：游戏系统需要有音效系统，让游戏场景更生动。

6. 分布式计算：游戏系统需要支持分布式计算，让游戏中的决策过程能够更快响应。

总的来说，游戏系统应该具有以下几个特点：

1. 使用户能够在网页端玩到魔兽世界中的经典游戏。

2. 提供丰富的游戏体验，包括操作简便、游戏性强，并能在短时间内获取高分。

3. 为用户提供最完备的特性，包括全面可用的技能系统、独有的声音效果系统，以及低延迟的网络通信机制。
