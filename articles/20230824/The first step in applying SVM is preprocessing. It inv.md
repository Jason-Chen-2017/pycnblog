# 1.简介
  

SVM (Support Vector Machine) 是一种二类分类模型，属于监督学习方法中的一种。它的主要目的是在高维空间中找到一个超平面，将数据分割开来，其中间存在一些间隔边界（margin）。SVM 的核函数的作用就是能够将输入空间进行非线性转换，使得训练样本可以划分为多个互相不完全重合的子空间，从而实现对复杂模式的建模。SVM 在某些特定情况下也可用于回归问题。

今天要写的是SVM预处理阶段的介绍，它包括了数据的标准化、降维、离群点检测等过程。SVM作为一款机器学习方法，其预处理环节非常重要，对于准确地构建模型至关重要。因此，掌握SVM预处理技巧对于机器学习领域的开发者来说尤为重要。

# 2.基本概念术语
## （1）特征向量
设X是一个数据集，X = {x(1), x(2),..., x(N)}, X是一个d维实数向量空间。那么，X中的每个元素都是一个特征向量，即：

x(i) = [xi1, xi2,..., xid]

其中，xij是第i个样本的第j个特征值，d是特征空间的维度。

## （2）特征空间
设X是一个数据集，X = {x(1), x(2),..., x(N)}。那么，特征空间是指满足约束条件的集合：

F = {f|x^T * f >=1 for x in X}  

这里，f∈Rn, R是实数空间，x^T是x的转置，>=1表示右半部分的大于等于号，即：

f ≥ 0, ∀f

约束条件保证了特征空间是一个超平面的集合，所有样本都被分成两类，并且每类之间存在一定的距离。

## （3）线性可分
如果存在一个超平面φ，使得所有的xi∈C1和xj∈C2，i!=j，那么称数据集X是线性可分的。

## （4）支持向量
对于给定的超平面φ，如果存在某个样本点xi使得φ(xi)≥1，则该样本点称作支持向量。注意，支持向量一般不是最小的，但它们构成了核函数的支撑集，所以对最终结果影响很大。

# 3.核心算法原理及操作步骤
## （1）数据标准化
数据标准化是指将数据变换到同一个尺度上，通常缩放到[0,1]或[-1,1]区间内。具体方法是在每一列上计算平均值和方差，然后用以下公式进行标准化：

Xi = (Xi - μ)/σ

μ: 每列的均值  
σ: 每列的标准差

比如说，假如有一组数据如下所示：

	X1	X2	X3
	-1	0	2
	0	1	1
	2	3	0

经过标准化后，得到的数据如下所示：

	X1	X2	X3
	-1	0	1
	0	0	0.707107
	1	1	0
