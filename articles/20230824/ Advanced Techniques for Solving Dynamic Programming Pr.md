# 2.前言

动态规划（Dynamic Programming）是一种在线性时间内解决很多决策问题的有效算法。它的运行时间依赖于问题的输入大小，但它的计算时间却不随着输入的增加而指数增长，因此在实际应用中被广泛使用。然而，很多动态规划问题都具有“子问题重叠”（sub-problem overlapping）的特性，即一个较小的问题的解可以由该问题的所有更小的问题的解得到。这种问题被称作是“重叠子问题”。

与传统的“分治法”（Divide and Conquer）相比，动态规划通常需要更多的内存，但它可以在线性的时间内解决大量的问题。同时，动态规划往往容易写成递归形式，而且它适用于许多类别的问题。

尽管动态规划算法的名称和概念已经成为研究热点，但其真正影响力却始于几十年前的《格雷戈里序列和编辑距离》一书。它给出了一个完善的框架，描述了如何通过递归解决这样的问题。后续的算法探索基于动态规划的算法变体，如贪婪搜索、回溯、序列划分等。

虽然动态规划具有强大的计算能力，但要确保正确使用，仍然需要精心设计。本文提供了一些提升算法效率、降低内存占用的技巧。本文的结构如下所示。

2.2 阅读建议

本文涉及的算法知识比较复杂，故建议对照相关书籍进行学习。

- “*Introduction to Algorithms”（CLRS）：作为动态规划算法的入门书。全面介绍动态规划问题的定义、分类、模型、P、NP、近似算法、启发式策略等；提供许多可视化示例。
- “The Algorithm Design Manual”（Skiena）：是一本引导读者设计自己的算法的书。第五版更新，加入了更多内容，包括动态规划设计模式、适应性搜索、贪婪算法、分层搜索、动态编程与矩阵乘法等。
- “Grokking DP Patterns”（Russel）：以实践方式介绍动态规划的八种常见模式。从简单到困难，每种模式都有清晰的解释。
