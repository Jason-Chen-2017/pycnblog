# 1.简介
  

## 1.1 研究背景及意义
传感器跨越领域的分布式监测网络已经成为遥感卫星图像获取的一种新方式。随着这一领域的成熟，如何在地面卫星图像和海陆两大卫星图像之间建立一致的索引也成为一项重要任务。许多方法试图解决这一难题，但通常都依赖于地球物理模型、光学信息或其他特征来衡量相似性。然而，由于卫星本身具有不同时间、空间分辨率等特性，其特征又无法直接用来计算相似性。因此，需要考虑其他办法来更好地捕捉卫星之间的相似性。

Attention-based leakage model（ABLM）是一种跨模态远程感知检索模型，它利用注意力机制学习到地面图像和海陆两大卫星图像之间的隐式关系并进行相关联。通过利用基于注意力机制的特征融合策略，ABLM可以将两者的信息融入到一起，从而可以有效地匹配海陆两大卫星图像与其对应地面图像。其主要特点如下：

1. **特征提取模块**：提取出图像特征并通过卷积神经网络（CNNs）进行特征降维。
2. **注意力机制模块**：对特征提取后的高级特征进行注意力学习，用以选择重要的特征子集。
3. **特征融合模块**：通过注意力权重对特征进行加权融合，得到匹配的结果。

## 1.2 系统组成与工作流程
ABLM模型由三个模块组成：特征提取模块、注意力机制模块和特征融合模块。

特征提取模块包括三层卷积神经网络（CNNs），分别提取海陆两大卫星图像的高级特征。第1层卷积核大小为7x7，第二层卷积核大小为5x5，第三层卷积核大小为3x3。每层输出的特征图尺寸为[N/64,H/32,W/32]，其中N表示通道数目，H,W分别表示高宽。另外，在第3个最大池化层之前增加了BatchNormalization层，进行数据标准化。海陆两大卫星图像都经过相同的处理过程。

注意力机制模块采用序列到序列（Seq2seq）结构，其中输入是海陆两大卫星图像经过特征提取模块提取出的高级特征，输出是相应地面的特征。具体地，在注意力编码层，通过双向LSTM生成编码向量，用以捕获局部的特征信息；在注意力解码层，通过单向LSTM生成注意力权重矩阵，用以分配权重给不同的特征子集；最后，再根据权重矩阵与特征矩阵相乘得到最终的融合特征。

特征融合模块则把海陆两大卫星图像的特征与其对应的地面图像进行融合，并得到匹配的结果。在特征融合模块中，首先对海陆两大卫星图像的特征进行L2归一化后求平均值得到最终的特征向量；然后，结合地面图像的特征使用FC层进行非线性变换；最后，使用softmax分类器得到最终的预测标签。

整个模型的训练过程中，首先使用海陆两大卫星图像的匹配集训练注意力机制模块；然后，使用匹配集、验证集、测试集对整个模型进行微调，直至收敛。此外，还可以加入数据增强、正则化等技术来进一步提升模型的性能。

# 2.主要术语
## 2.1 特征提取模块
### 2.1.1 基本概念
卷积神经网络（Convolutional Neural Networks，CNNs）是一类深度学习技术，旨在从输入的多维数据中提取高阶特征，被广泛应用于图像、文本、音频等领域。一般来说，CNNs由卷积层、池化层、全连接层以及激活函数构成。卷积层用于检测图像中的特定模式（如边缘检测、颜色特征等），池化层则用于降低纬度。全连接层则用于对抽象特征进行处理，激活函数用于控制神经元的输出范围。

### 2.1.2 模块概述
ABLM模型的特征提取模块由三层CNNs构成。第一层的卷积核大小为7x7，步长为2，第二层的卷积核大小为5x5，步长为2，第三层的卷积核大小为3x3，步长为1。每层输出的特征图尺寸为[N/64,H/32,W/32]，其中N表示通道数目，H,W分别表示高宽。第二层之后的特征图经过Batch Normalization。

## 2.2 注意力机制模块
### 2.2.1 基本概念
注意力机制（Attention）是指模型能够关注某些特定的输入，并倾向于产生较大的输出。注意力机制可用于推断信息，提高模型效率，改善机器学习结果。常用的注意力机制有全局注意力和局部注意力两种。全局注意力可以使得模型注意到整体的上下文信息，局部注意力则只关注局部的上下文信息。Attention-based leakage model（ABLM）就是利用局部注意力进行特征融合。

### 2.2.2 模块概述
ABLM模型的注意力机制模块是一个序列到序列（Seq2seq）结构。在注意力编码层，海陆两大卫星图像的高级特征通过双向LSTM生成编码向量。每个向量代表了一张图像的重要信息。在注意力解码层，通过单向LSTM生成注意力权重矩阵，用于分配权重给不同的特征子集。最后，根据权重矩阵与特征矩阵相乘得到最终的融合特征。

## 2.3 特征融合模块
### 2.3.1 基本概念
特征融合（Fusion）是指将多个特征合并到一起以实现更好的学习效果。常用的特征融合方法有多种，如拼接、加权和平均等。ABLM模型的特征融合模块，是在两个特征向量之间进行拼接，形成一个新的特征向量，作为海陆两大卫星图像匹配的结果。

### 2.3.2 模块概述
ABLM模型的特征融合模块包括三个部分：海陆两大卫星图像特征向量的拼接、基于全连接层的非线性变换和softmax分类器。拼接的方法是直接将海陆两大卫星图像的特征向量拼接起来，然后使用softmax分类器进行分类。非线性变换的方法是使用全连接层进行非线性变换，然后得到预测的输出结果。
