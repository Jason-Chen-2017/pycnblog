# 1.简介
  

## 1.项目背景
在信息化时代，组织管理模式、流程和工具正在发生着重大的转变。越来越多的非营利组织（NPOs）选择了更加开放包容的管理方式，并将精力集中于使命驱动型的服务倾斜。然而，当下这些组织面临新的挑战，比如效率低下、缺乏透明度以及缺乏改进能力。如何让NPOs实现更高质量的服务？如何提升员工满意度？如何确保非营利组织能够顺利运作？

Global Giving Machine(GGM)就是为了解决上述问题而诞生的。它是一个平台，提供了一个全面的管控系统，不仅可以满足非营利组织的管理需求，还可以从社会经济角度对其进行评估、规划和预测。GGM采用模块化设计，具有众包、财务、信息、团队建设等功能，帮助NPOs实现资源的最大化利用。通过个性化服务和可视化展示，GGM可以帮助组织者了解成员偏好，及时调整策略，以达到更好的结果。

## 2.项目目标
1.降低组织运营成本
- 增强组织管理和服务能力
- 提升组织的社会责任感和公益心

2.提升服务质量
- 实施预算优化机制
- 降低组织间互相竞争程度
- 为成员提供更优质的服务

3.提升成员参与度
- 使用个性化服务和提升积极性
- 更好地激发员工的创造力和个性

## 3.项目构想

### 1.运营管理层
运营管理层由具有管理经验的IT专业人员组成，负责推进GGM的运营工作，包括组织目标设置、项目跟踪、数据分析、决策支持等工作。运营管理层主要承担以下任务：
- 监控成员行为和反馈信息，形成数据；
- 建立数据仓库，对所有的数据进行集中管理；
- 建立规范化的工作制度，确保运营工作的有效执行；
- 发行会刊物，及时发布新产品、新活动以及更新；
- 制定目标管理计划，指导各项工作的有效实施；
- 负责组织内部管理，确保组织的运行和管理符合最佳管理标准。

### 2.财务分析部
财务分析部由一群财务专业人士组成，他们对组织收支情况、活动预算进行审计、分析和预测，并对管理层的预算要求进行协调。财务分析部主要承担以下任务：
- 通过预算管理工具，对成员、团队和机构进行现金流预测；
- 对公司总资产、净利润、每股盈利、利润分配结构进行分析，发现管理层没有考虑到的问题；
- 将分析结果通过报告和图表呈现出来，帮助管理层制定决策；
- 进行运营成本的分析，将成本的贡献、结构以及影响者梳理清楚；
- 根据业务需要进行自身的管理工作，包括决策支持、执行管理和服务保障。

### 3.项目管理部
项目管理部由一群项目管理专业人士组成，他们对组织的各项业务活动进行详细策划、组织实施、项目管理和信息共享。项目管理部主要承担以下任务：
- 智能识别和匹配需要的资源，配置人员和资源，保证业务目标的实现；
- 建立内部培训系统，促进员工的知识水平和职业道德；
- 开发和维护内部数据库，记录各种信息；
- 组织成员进行各种训练和交流，提升技能、提高效率；
- 提供最新的信息和最新技术，使组织保持领先地位。

### 4.信息部
信息部由一群信息专业人士组成，他们对成员的满意度、工作状态、工作满意度等信息进行收集、整理、分析和呈现。信息部主要承担以下任务：
- 收集、整理、分析和呈现成员的信息，形成报表和数据；
- 形成系统化的工作信息管理体系，确保信息的准确、完整和及时；
- 借助互联网和微信等技术手段，提升组织的网络状况，并使员工及时获取最新信息；
- 将内部信息进行有效的传播，鼓励员工积极分享自己的故事、经验以及感受。

### 5.培训部
培训部由一群教育专业人士组成，他们对组织的管理理念、业务模式、操作方法等进行教授，并对员工进行专业培训。培训部主要承担以下任务：
- 负责组织内部培训，向新员工介绍组织的运营理念、业务模式和操作方法；
- 组织举办各类培训课程，提升员工的专业水平；
- 建立学习交流平台，促进员工之间的交流，共享知识；
- 以优秀的管理者形象，塑造员工的工作态度和价值观，激发员工的自我驱动力。

### 6.设计团队
设计团队由一群视觉设计专业人士组成，他们负责GGM的产品设计，包括网站设计、APP界面设计、海报设计等。设计团队主要承担以下任务：
- 负责网站和APP的设计，为组织呈现最专业、最吸引人的形象；
- 合理布局页面元素，使界面简洁易用；
- 研究并采用用户研究方法，收集用户的反馈信息，优化产品；
- 深入理解用户习惯，提升产品的可用性和可用性。
