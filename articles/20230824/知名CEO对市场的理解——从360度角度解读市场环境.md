# 1.简介
  

自从互联网泡沫破灭后，近几年经济飞速增长，经济危机也逐渐消退。随着社会的进步，电子商务、社交媒体、智能手环、无人驾驶等新兴产业蓬勃发展。但同时也带来了新的挑战。

无论是供应链金融，还是社会信用体系的转型升级，都需要一个系统的解决方案。越来越多的人对如何在这个过程中找到合适的发展方向感到迷茫。但是越来越多的创业者和领袖开始站出来，说服投资者加入他们的阵营，试图将未来的制造业变成自动化和数字化。人们不断寻找新的突破口，重塑商业模式，实现自己的价值。

今天我想和大家一起分享的是一位颇具影响力的企业家对市场的理解。他曾经历了什么样的困境？如何正确的面对竞争？从360度角度，全方位地解读市场环境。
# 2.核心观点
## 2.1为什么要关注市场?
首先，市场不是静止不动的。它是一个动态变化的过程。市场变化的原因很多，比如新产品的出现、客户需求的增加、外部环境的变化等等。所以，只看表面的数字并不能准确反映市场的真实情况。

其次，经济学告诉我们，要想成功，就要把握时机，市场总是充满着机会。市场有着巨大的潜力，我们必须谨慎行动，尤其是在关键时刻。

第三，市场非常复杂。涉及许多利益相关者，品牌、客户、供应商、竞争者等。每个参与者都面临着自己的需求、目标和愿景。如果忽略这些因素，就很难判断市场的走向。

综上所述，了解市场对我们每一个人的生活和工作都至关重要。只有全面把握市场，才能更好地做出判断，提高我们的能力，开拓新的市场。
## 2.2三类企业家
### 2.2.1信息企业家（Information Executives）
信息企业家关注的是信息的获取、分析和整合。他们的首要任务是搜集、整理、分析、汇总有价值的信息。信息源包括网络、媒体、销售人员、同事等等。

他们对信息采取“眼光宽阔”，不局限于某些特定领域。他们喜欢接触不同部门的信息资源，并持续不断地学习、研究，提升自己的能力。例如，亚马逊的联合创始人杰夫·贝索斯就是一位信息企业家。

信息企业家通过收集信息，形成具有洞察力的思维方式。他们擅长利用信息快速地发现新机会、做出决策。比如，电子商务网站阿里巴巴公司的副总裁刘强东是一位典型的信息企业家。他会分析市场的发展趋势，根据消费者的需求进行规划和布局，开拓新的业务领域。

### 2.2.2商业决策者（Business Decision Makers）
商业决策者追求的不是单纯的利益最大化，而是以客观的视角，基于信息的商业判断。他们对市场上的事物保持敏锐的洞察力，并结合历史数据、现状以及未来的发展趋势，作出明智的决定。

商业决策者的角色主要是执行，而不是创造，因此通常不会参与到事务的组织和管理中。他们往往善于观察、分析、比较，并得出一定的判断。例如，任正非就属于这一类的职业。

商业决策者把市场信息分为几个层级。第一层是商业趋势，第二层是竞争环境，第三层是客户需求，第四层是公司战略，第五层是团队精神。

商业决策者提出的决策应该有理性、可操作性，并且能够回答“为什么”、“怎么办”、“应该怎么做”等系列的问题。他们可以帮助公司领导者制定新的业务计划，或调整公司的经营策略。比如，拉里·佩奇就担任过三星手机的董事长兼总经理。

### 2.2.3资源开发者（Resource Developers）
资源开发者追求的是增值服务。他们认为市场可以提供给客户丰富的各种服务，而不需要为此付出太多的代价。

资源开发者经常会与商业决策者合作，探讨如何最有效地运用市场资源来赢得更多的用户。例如，苹果公司的市场营销总监蒂姆·库克就属于这一类人。他会在各种不同渠道积极扩大产品的知名度，扩大宣传面积，以吸引目标用户。

资源开发者也有自己的独特优势。比如，Facebook的CEO扎克伯格就是一位资源开发者。他的顽强不息的努力促使这个社交网站获得快速的发展。

综上所述，信息企业家、商业决策者、资源开发者都是投资者在创业过程中不可或缺的角色。对市场的正确认识、全面掌握各类信息，将直接影响企业的生存、发展、成功。
# 3.1对市场的定义
市场是一个买卖双方互相博弈的交易平台。任何个体，只要有足够的钱或者其他东西作为筹码，就可以进入市场参与竞争。每个人在市场中都存在着两种行为，即做出决策和接受信息。

做出决策，就是在市场中选择自己认为有价值的东西，做出购买或投资决策。接受信息，是指从别处获得的知识、信息、消息。市场中的信息，既可以是公开的信息，也可以是隐私信息，或者是公司内部的管理消息。

由此可见，市场是一个不断更新的信息平台。只要有意愿，任何人都可以在市场中提供自己能够提供的价值，与别人分享自己的成功经验。

# 3.2市场的组成
市场的构成主要有两个，即生产者和消费者。其中，生产者产生商品，如房屋、手机、飞机等；消费者则消费商品，如购买商品、使用服务、担保质量等。

市场的形成又可以分为三个阶段，即供需关系阶段、价格制定阶段和竞争阶段。在供需关系阶段，消费者的数量开始急剧增加，由于不确定性的原因，导致供需不平衡。在价格制定阶段，企业根据需求、供应商的供应情况、市场的运行情况等条件，确定适当的价格水平。在竞争阶段，为了取得更高的利润，各企业开始进行竞争。

市场的其他构成还有产品、服务、渠道、媒体等。其中，产品是消费者购买的对象，如手机、相机、汽车等；服务是生产者向消费者提供的产品功能，如影像传输、支付处理、网络游戏等；渠道是用于传递信息的媒介，如电视、报刊等；媒体是信息的消费者，如新闻网站、微博客等。

# 3.3市场的结构
市场的结构，有不同的分类方法。最简单的分类法是基于效用函数的方法，即每种商品都有自己的收益和风险。效用函数可以表示商品的获利能力和损失风险。另一种分类法是基于市场份额的方法，即按照占据的份额大小，将市场分为垄断、合作、第三方三种类型。

# 3.4市场的路径
市场的路径，即消费者的购买行为的演变过程。市场路径包括两个方面，即供需曲线和趋势。

供需曲线显示了两种商品之间价格之间的关系。一旦价格低于供应，就会出现供需不足，此时购买者就会减少；而当价格高于供应，就会出现供需超载，此时购买者就会增加。供需曲线最显著的特征是商品的买方和卖方的数量对比发生变化。

趋势是指市场对于商品的需求和供给都在发生变化，但是还没有达到临界点的状态。在这种情况下，商品的价格会慢慢趋于稳定。通常来说，趋势的长度依赖于商品的价格和供需之间的关系。

# 3.5市场的本质
市场的本质是双方博弈的结果。当双方都接受市场的存在，且认同市场的本质时，才可能形成真正的共赢。任何一个人，不管是在生产者、消费者、管理者还是其他参与者，都要关注自己的利益，而不是被市场左右。

市场的本质是流通资本的博弈，是资本和时间的凝聚。流通资本的大小决定着市场的重要性。当流通资本很小时，流通资本的力量就会弱化；当流通资本达到一定程度时，就会对市场产生重大作用。

# 4.1竞争情景
关于市场的理解，一个最重要的概念是竞争情景。竞争情景描述了当前市场的竞争环境，以及未来的发展方向。

在信息阶段，信息企业家开始追求新的市场信息，寻找新的机会。他们可以研究财经新闻、技术文档、分析报告，看是否有新的信息可以帮助他们提高竞争力。在商业决策阶段，商业决策者则更加关注市场的外在环境，选择适合自己公司的竞争环境。

在资源开发阶段，资源开发者关注资源的分配。它们知道如何运用生产者的资源，改善生产过程，提升产品质量，吸引更多的用户。每个人都希望自己的工作得到重视，而不仅仅是赚钱。

在未来的发展方向，各类企业家都会面临新的挑战，如信息时代的停滞、市场的转型、资本的迁移等。但与此同时，竞争也正在发展壮大。

# 5.1市场现状和未来展望
市场的现状：
1、流通资本呈现初级趋势。中国有超过八成的流通资本分布在地区经济较为落后的区县，三农地区的流通资本有限，甚至还有不少流向国企的情况。

2、消费需求持续扩大。今年前十个月，全国300多个城市的年轻人正式踏入工作岗位，这股消费需求一直未衰，占GDP的1/4左右。

3、三大运营商“主攻”增长。中国移动、中国电信、中国联通三大运营商在过去三年连续盈利，分别创下11年、11年、9年的纪录。

市场的未来展望：
1、人工智能时代到来，机器学习成为重要科技。随着机器学习的普及和技术进步，未来会出现越来越多的产品和服务受到机器学习的驱动。

2、农业4.0时代到来，智慧农业不再只是为农民谋福利，成为各地农民土地的保障和服务方式。

3、人口红利期逐渐结束，经济将迎来高速增长期。
