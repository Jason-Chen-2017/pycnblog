# 2.基本概念术语说明
---
## （1）什么是机器学习？
机器学习是一门具有广泛 applications 的学科，它使计算机可以自动学习并改善它的性能，以更好地解决一般的任务和重复出现的问题。它所涉及到的主要是以下三个方面：
- 数据：机器学习所需的输入数据通常都是非结构化的，也就是说，没有事先定义好的表格形式。机器学习 algorithms 将这些数据转换成可用于分析的形式，因此可以用于预测和分类任务。
- 算法：机器学习 algorithms 是一种能够从数据中学习并推导出规律的工具。根据数据的特点、大小、质量，以及目标，机器 learning algorithms 分别有不同的实现方法。例如，linear regression 可以用来回归问题，clustering algorithms 可以用来聚类问题，neural networks 可以用来分类问题。
- 模型：机器学习 models 则是指由数据及 algorithms 所产生出的结果，包括线性回归模型、决策树模型、随机森林模型、支持向量机模型等。models 包含了许多参数，通过参数调优的方式，可以通过优化目标函数来提升 models 的能力。
因此，机器学习是一门研究如何从数据中获取知识、建立模型，并利用模型对新的输入数据进行预测或分类的学科。  

## （2）什么是深度学习？
深度学习是机器学习的一个子分支，是指在机器学习过程中，用多层神经网络来学习数据的表示，并使用这种表示来解决各种复杂的学习问题。深度学习依赖于大量的训练数据，可以处理高维、非结构化数据，并可以构建非常复杂的模型。深度学习的主要研究对象是图像、文本、声音和视频等复杂的数据，可以自动提取高级特征，从而使得机器学习算法更具一般性和普适性。深度学习包括卷积神经网络 CNN、循环神经网络 RNN 和递归神经网络 LSTM。

## （3）什么是增强学习？
增强学习 (Reinforcement Learning) 是一种机器学习方法，用于解决复杂的任务，它强调 agents 在环境中学习的能力。agents 通过 trial-and-error 方式，不断试错、不断调整策略，最终达到最佳的效果。它的特点是通过反馈机制来学习，agent 必须学会在一个环境中做出最好的选择，即使这个环境可能永远不会终止。所以，增强学习可以看作是一套强化学习的形式。其算法如 Q-learning、SARSA、 actor-critic 等。

## （4）什么是弱化学习？
弱化学习 (Reinforcement Learning with Weak Priorities) 是增强学习的一种变体，主要区别是，在增强学习中，agent 在执行每一个 action 时都会得到奖赏，而在弱化学习中，agent 只在达到某种目标后才会得到奖赏。弱化学习的目的是更好的容忍延迟、忍受困难，并最大化长远利益。其算法如 CEM、 DQN 等。

## （5）什么是强化学习？
强化学习 (Reinforcement Learning with Strong Priorities) 是指，agent 在环境中学习时，以一定的目标为导向，整合知识、策略和经验，在执行 actions 时不断修正策略，以期达到最大化长远利益。强化学习的特点是注重效益最大化，试图找到全局最优的解决方案。其算法如 DDPG、 A3C 等。  

## （6）什么是马尔可夫决策过程？
马尔可夫决策过程 (Markov Decision Process, MDP) 是一种强化学习的基本框架。MDP 中，存在一组状态 s ，表示当前 agent 的位置；存在一组动作 a ，表示 agent 对状态的决策；存在一个 transition function P(s'|s,a)，表示状态转移概率；以及一个 reward function R(s,a)。其中，状态转移函数表示在状态 s 采取动作 a 后的下一个状态 s'；奖励函数表示在状态 s 下执行动作 a 后的奖励。MDP 可以定义为一元系统，也可以定义为多元系统。

## （7）什么是贝叶斯学习？
贝叶斯学习 (Bayesian Learning) 是一种机器学习方法，它假设 agent 的行为是由一个先验概率分布 (prior distribution) 决定的。由于这个先验分布可能会存在错误，agent 需要利用数据及先验分布来更新自己的 belief distributions。其算法如 Gibbs sampling、 variational inference 等。

## （8）什么是样本复杂度控制？
样本复杂度控制 (Sample Complexity Control) 是为了解决学习过程中的样本偏差问题。当样本数量太少时，无法很好地估计真实模型，容易陷入局部最小值或震荡。当样本数量太多时，会导致计算量太大，难以有效地进行求解。因此，在样本数量足够时，我们要控制学习算法的复杂度，防止过拟合。样本复杂度控制的方法有加法噪声项 (additive noise terms)、核函数 (kernel functions)、拉普拉斯先验 (Laplace prior)、稀疏矩阵分解 (sparse matrix decomposition) 等。

## （9）什么是无监督学习？
无监督学习 (Unsupervised Learning) 是机器学习中的一个分支。在该方法中，agent 并不需要标注数据，只要有一个无监督学习算法即可对数据进行聚类、划分、降维等操作。无监督学习方法可以帮助我们发现数据中隐藏的模式，发现数据中的结构信息，进一步分析数据。其算法如 k-means 聚类、隐马尔可夫模型 HMM、 PCA、ICA 等。
