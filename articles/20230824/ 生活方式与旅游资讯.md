# 1.简介
  
  
# 在过去的一段时间里，随着互联网的蓬勃发展，消费者的购物需求量激增，导致了旅游行业的爆炸式发展，在全球各个角落都产生了新的需求，成为当今世界经济发展的一个热点。旅游行业的规模不断扩大，带动着经济的发展，人们也对旅游需求逐渐增加的需求。  
而对于一般人来说，旅游不仅仅局限于观光或者外出，更多的是为了探索自然资源，满足精神需求。所以，旅游信息的获取是一个至关重要的问题。尤其是在互联网的发达过程中，如何更有效地获取旅游资讯是非常有必要和重要的。  
针对这一问题，本文就基于国内外一些主流旅游网站，以及旅游资讯提取技术的研究成果，分析并总结一下如何提高旅游资讯获取效率的方法和手段。  
# 2.相关概念术语说明  
## 概念介绍
**搜索引擎：**通过关键词搜索的方式找到目标网页，用来获取海量、丰富的网络信息，是目前最流行的获取信息的方式之一。

**信息提取（Information Extraction）:** 即从大量文本中自动抽取特定特征作为输出，用于文本分类、情感分析、意图识别、事件识别等应用领域，是信息检索的一种重要方法。

**数据挖掘（Data Mining):** 从大量数据的集合中找寻具有显著性质的模式或规律性，建立预测模型来描述数据之间的联系，是很多数据科学、机器学习任务的核心。

**爬虫（Crawler）:** 通过访问网站来收集大量网页上的信息，是一种快速、便捷的获取信息的方式。

**网页结构化信息：**网页上的文本、图片、视频、音频等多媒体内容经过提取、分析、组织后形成结构化的文字、图片、视频、音频等形式的信息，是获得更详细、更准确的内容信息的关键。

**微博客、微信公众号、新闻网站、论坛评论等社交媒体信息：**以短平快的速度发布海量内容，传播知识和观念的媒介，影响力极大，可以进行大量的有价值的信息获取。

**网络游戏资源：**游戏中的素材、道具、场景等都是网络上免费下载的，并根据玩家的游戏情况进行游戏体验评估和改进，是参与到游戏开发及制作中不可或缺的一环。

## 术语说明
**API(Application Programming Interface)：**应用程序编程接口，它是一些预先定义的函数、过程或者类的集合，提供应用程序与系统之间进行信息传递和通信的手段。

**HTML/XML/JSON：**Hypertext Markup Language / Extensible Markup Language / JavaScript Object Notation 是三种主要的数据交换格式，主要用于不同平台之间的数据交换。

**NLP(Natural Language Processing)：**自然语言处理是指人工智能领域的研究，旨在使电脑“理解”、“生成”和“运用”人类语言、文本、图像等符号。

**NLU(Natural Language Understanding)：**自然语言理解，它是指让计算机能够“读懂”人类的语言，并且用计算机程序做出相应的回应。

**RE(Regular Expression)：**正则表达式是一种用来匹配字符串的强有力的工具，它可以帮助您方便地检查一个字符串是否符合某种模式。

**SPA(Single Page Application)：**单页面应用，也叫前端路由应用，是一种运行于浏览器端的Web应用架构。它将所有功能都集成到同一个页面中，在无需重新加载页面的情况下，实现用户与应用之间互动。

**URL(Uniform Resource Locator)：**统一资源定位符，它是互联网上用来表示网络资源所在位置的字符串，通常由协议、域名、端口号、路径和参数组成。

**XML(eXtensible Markup Language)：**可扩展标记语言，是一种对各种数据结构进行编码的语言，它被设计用来传输和存储复杂的、层次化的、结构化的文档。
