# 1.简介
  

1、计算机视觉（Computer Vision）:从图像的摄影、拍照到医疗诊断等多个领域，都需要大量地运用计算机视觉技术来进行相关的分析处理。随着深度学习的发展，人们越来越关注如何利用机器学习技术实现计算机视觉的一些功能。计算机视觉技术是指能够让电脑“看”的能力，并在此基础上进行图像识别、图像理解、目标跟踪等高级分析处理。

2、基于深度学习的计算机视觉任务主要包括：图像分类（Image Classification），物体检测（Object Detection），图像分割（Image Segmentation）。而对于目标检测（Object Detection）来说，其可以应用于行人重识别、车辆跟踪等多种应用场景中。

3、最近十年来，基于卷积神经网络（Convolutional Neural Network，CNN）的目标检测模型取得了不错的效果。

4、为了解决目标检测中的两个主要问题——边界框回归（Bounding Box Regression）和类别预测（Category Prediction），提出了R-CNN和Fast R-CNN等两类模型。前者只需一次性对整个图片的特征进行提取，速度快但准确率低；后者则通过RoI pooling层，分别对每一个候选区域进行特征提取，再使用线性整流单元（ReLU）将其转换为置信度（confidence score），进一步提升模型的准确率。

5、基于深度学习的目标检测技术还面临着两个主要挑战：计算复杂度的增加以及训练数据不足的问题。近几年来，由于图像采集的数量增加、成本下降、计算性能提升，人们越来越多地采用弱监督学习的方式进行目标检测任务的训练。

6、YOLO (You Only Look Once)是一个用于实时目标检测的轻量级模型，它与其它目标检测模型相比，具有快速推理时间和较高的精度。YOLO模型只有一次完整的卷积网络推理，并且没有任何像素点组合或先验框生成过程，因此速度很快。同时，YOLO模型自带预训练权重，不需要额外的数据训练，使得它在目标检测方面具有一定的泛化能力。

7、深度学习方法在计算机视觉领域已经得到广泛的应用。但是，这些方法并不能解决所有问题，特别是在目标检测方面。最近，一组研究人员提出的端到端目标检测框架EfficientDet已经成功地解决了这一难题。其主要思想是借鉴了Mask R-CNN的设计模式，在网络结构上结合了Faster RCNN的锚点机制和FPN机制，使用多尺度特征图及更高分辨率的输出图像进行预测。

# 2.深度学习常用库
在深度学习中，我们通常会使用以下几个常用的库：
1. TensorFlow : TensorFlow 是由Google开发的开源机器学习框架，是目前最流行的深度学习库。TensorFlow支持多种编程语言，如Python、C++、Java、GoLang等，可以帮助用户快速搭建深度学习模型。
2. Keras : Keras是一个基于Theano或者TensorFlow之上的高级神经网络API，它提供简单易用的接口，使得我们可以方便地构建各种类型的神经网络，例如卷积神经网络(Convolutional Neural Networks， CNNs)，循环神经网络(Recurrent Neural Networks，RNNs)，递归神经网络(Recursive Neural Networks，RNNs)。Keras可以让你快速完成从数据加载，预处理，微调模型，到部署的整个流程，非常适合机器学习初学者。
3. PyTorch : PyTorch是Facebook AI Research团队开源的一个基于Torch的机器学习框架。PyTorch相比于TensorFlow，在Python API和运行效率上都有所提高。相比Keras，PyTorch更适合图像处理和自然语言处理领域的深度学习项目。
4. MXNet : Apache MXNet是一种分布式、动态规划的深度学习框架。MXNet可以帮助用户创建、定制化深度学习系统，用于训练、评估、调优和部署强大的模型。MXNet使用符号式定义，可以自动进行数值求解，有效地节省内存。它的优势是速度快，并且可以利用GPU加速计算。
5. Caffe : Caffe是一个深度学习框架，它基于纯C++编写。其提供了多种模块化组件，使得用户可以灵活地组装和搭建不同的神经网络。Caffe的优势在于容易安装和配置，它是目前最流行的深度学习框架。
