# 1.简介
  


作为一名软件工程师，一定要对自己所学知识及技能有足够的理解和掌握。阅读一些经典著作能够帮助我更加全面地了解计算机科学、网络、算法等新技术的概念、原理和应用，能够指导我的工作和职业规划。所以，在阅读完本书之后，不但能掌握较好的编程技术，而且也能对自己的编程能力有一个全面的认识和把握。同时，还可以用编程的方式思考一些经典的问题，如“如何设计一个系统”、“如何高效处理海量数据”、“如何解决并发问题”。学习这些经典的计算机基础知识，对提升自身的综合素质、培养自己的分析问题和解决问题的能力都有很大的帮助。

阅读《Effective Java》之前，读者需要具备扎实的Java编程基础。比如：熟悉面向对象编程、链表、数组、集合框架、异常处理机制、多线程编程、反射机制等。另外，在阅读Java相关的书籍时，也应当注意选择那些内容易于理解，且能给实际工作中所需的知识提供很好的实践支持的书籍。对于初学者来说，《Head First Java》或许是一个不错的选择；而对于已经有一定经验的Java开发者来说，则应该选取一些经典的教程或项目作为练习项目。当然，无论哪种情况，阅读时要能够耐心、仔细和系统地阅读，并快速消化其中的精华。

## 本书的写作动机

虽然计算机科学的所有分支都伴随着软件工程师，但是作为工程师的我们，一直缺乏一个能系统阐述计算机科学知识的方法。并且，即使知道这些知识的原理和操作方法，我们仍然缺乏一种全新的视角，能将这些知识应用到实际工作中去，甚至能带来个人价值。所以，我们编写了一本能够帮助软件工程师进行系统性学习、掌握计算机科学的原理和方法的书籍——《Effective Java》。

《Effective Java》一书既不是普通的技术书籍，也不是普通的教程或者培训课程。它是一个由作者亲自总结的、独到的参考手册，提供了一些从实践中学习知识的方法。它通过介绍和分析最佳实践，并基于这些最佳实践，为开发人员提供可行的建议。书中的很多内容都是作者经过研究发现的，包括各种编码规范、设计模式、并发编程等方面的最佳实践。这使得读者能够在短时间内掌握最重要的计算机科学概念、原理和方法，并真正运用于工作中。

由于《Effective Java》这本书很难写，所以目前还不能免费提供下载。但希望读者们多多关注，在实践中磨炼自己的编程技能。
