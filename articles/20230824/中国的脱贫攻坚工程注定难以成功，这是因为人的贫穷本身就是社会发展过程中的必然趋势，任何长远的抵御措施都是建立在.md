# 1.简介
  

中国的人均GDP已经从9.5万美元上升到12.7万美元，这对世界经济发展具有非常重要的意义。但是对于一个国家来说，如何在长期保持或改善人均收入水平，是一个永恒的话题。
近年来，中国正处于“脱贫攻坚”的关键时期。许多人担忧中国的“脱贫攻坚”将面临生存危机、人口老龄化等全方位挑战。事实上，只要解决了制度性因素（如房地产、教育、医疗、养老保险等）、政治体制改革、财政政策调整等方面的问题，就一定能够实现“脱贫攻坚”。
不少评论者指出，“脱贫攻坚”的政策目标设得太高，目前只有两条路可走：一条是以计划经济为主、引进外资、提高劳动力技术等为辅，另一条则是以市场化的方式加快转型，逐步推动社会的公平正义。因此，并没有出现严格的“优胜劣汰”的局面，也没有完全消除贫困。因此，要真正采取“脱贫攻坚”，还需要长期艰苦奋斗。
本文主要阐述中国“脱贫攻坚”的演变及其失败原因，讨论如何通过制度创新、政策扶持等方式实现“脱贫攻坚”工程，并总结提炼出“中国的“脱贫攻坚”工程应该具备的特征”。
# 2.中国的“脱贫攻坚”是怎么回事？
## 2.1 背景介绍
“脱贫攻坚”是中共十八届三中全会确立的政府工作报告之一。根据中央的部署，从2015年至今，习近平提出“三个定位”，即富裕起来的、吃饭起来的、穿衣起来的。以富裕起来的这个定位为导向，中央决定全面推进“脱贫攻坚”，成为我国的第一把旗帜。到了2018年底，中国经济遥遥领先世界，人均GDP已经达到12.7万美元，而世界银行预测到2025年全球GDP将超过17万亿美元。因此，近几年来，中国追赶甚至超越了一些发达国家。但在过去几年里，由于经济发展取得阶段性成果，人均GDP增速放缓，再加上经济结构性问题带来的贫富差距持续扩大的压力，使得人们对中国经济发展的关注、关切不断增加。而作为首当其冲的“脱贫攻坚”首先就成为众多关注热点。
## 2.2 “脱贫攻坚”为什么不能成功？
### 2.2.1 “脱贫攻坚”定义不清晰
“脱贫攻坚”被设计用来描述整个中共领导集团为了推动国民经济发展、实现“两个一百年”奋斗目标，在减轻贫困、促进就业、提高民生等方面采取的一系列举措。在一定程度上，“脱贫攻坚”可以视作经济体制改革、政策调整和社会责任感的体现，但它的目标范围也很广泛。例如，经济发展带来的收入分配不平等，既与国家能力的提升息息相关，又与税费等生活成本息息相关；教育、医疗、养老保障等福利政策以及推动城镇化等城市治理政策都有着举足轻重的作用。总之，“脱贫攻坚”还涉及一系列的其他议题，如环境保护、人权保护、规则转型、投资引导等等。
但是，当前关于“脱贫攻坚”的研究往往忽略了“脱贫攻坚”定义的具体含义，导致一些不同观点存在争议。据笔者所知，以下几种观点是最普遍的：

1. 在“脱贫攻坚”工程中，包括财政收入、财政支出、政府投入等项目，都需要得到有效安排和保障。此外，为了更好地实现“脱贫攻坚”，还需继续推动绩效管理制度的完善，让各类政府职能部门成为公开、透明、问责的机制。
2. 在“脱贫攻坚”工程中，需要搞活基础建设。基建工程作为基础设施保障，发挥着举足轻重的作用。同时，修建物业公交站点、高铁站点等，也可以为居民提供便利。
3. 虽然经济结构性问题是影响中国“脱贫攻坚”工程不可避免的挑战，但是是否“左”可以成为问题的关键。如果在“脱贫攻坚”工程中设置了错的目标，如以养老金替代医疗保障、继续建设高价值工程，那么“左”势力势必介入，对相关决策产生影响。
4. 从表面看，“脱贫攻坚”工程主要靠政策扶持、财政拨款等方式来完成。但是，中央下达的各种补助措施，比如“促进企业破产率降低”、“鼓励新设小微企业”等，实际上也存在一些问题，尤其是在“税负过重”的问题上。由此可见，当前关于“脱贫攻坚”的研究仍存在一些局限性。

### 2.2.2 制度性因素、体制性约束无法根治贫困
在讨论“脱贫攻坚”工程的可行性之前，笔者认为，制度性因素、体制性约束同样是造成中国“脱贫攻坚”工程失败的主要原因。这一因素主要表现在三个方面：

1. 制度性因素。制度缺失导致农村地区贫困潜伏已久，一些农村落后单位存在财政困难、社会矛盾激化、法律矛盾频发、行政压力大等问题。另外，制度性歧视也是一个重要的制度性因素。过去十几年，随着中央的政策性引导，农村阶级普遍存在阶级固化倾向，贫富分化加剧，导致农村阶层劳动力比例偏高，农村贫困人口占比过高，农村经济日益落后。
2. 体制性约束。经济体制改革力度有限，政策执行力度有限，结果造成贫困问题比想象的更严重。近年来，为了解决经济发展中遇到的种种问题，各地纷纷启动“脱贫攻坚”工程，提升基层公共服务和群众基础设施的水平。然而，这些工程有些时候效果欠佳、反而使情况恶化。另外，政策偏向也影响着“脱贫攻坚”工程的有效性。在一些农村，财政补贴过多、农村公共服务投入过少、农村基础设施的建设过少等等，都会导致贫困问题更加突出。
3. 智力不足。中央政府官员众多、智慧资源丰富，但作为决策者的他们无法像普通民众一样，总是站在智力绝顶的高度来决策，而且往往缺乏战略思维能力。在“脱贫攻坚”工程中，官员们往往更加依赖政府政策宣传系统，“沉默不说，走火入魔”，而不是真正理解问题，因此难以有效防止和化解贫困问题。

### 2.2.3 投资约束条件恶化“脱贫攻坚”效果
相较于制度性因素、体制性约束，投资约束条件的限制则更容易导致“脱贫攻坚”工程的失败。目前，中国股市、房地产市场、土地市场都处于高速发展状态，海外投资进入中国也日渐增加，“谈钱伤身”，致使中央政府对外投资、对内投资政策执行力度相对弱化，引发了“脱钩”、“夹击”、“反哺”等问题，造成经济结构性问题的恶化。另外，为了解决投资者的合规、融资成本、风险承受能力等问题，银监会和央行也在推动地方政府进行信贷和融资政策的改革。但是，由于自身条件所限，地方政府也无法有效制衡国家对内对外投资的影响。
## 2.3 中国的“脱贫攻坚”工程应该具备什么特征？
### 2.3.1 有序且有力的政府决策
“脱贫攻坚”工程应当具备自上而下的政府决策能力，首先是实施方案，然后制定政策措施，然后分批次、跨区域、多层次地展开执行。至于具体操作措施的可行性和效益，则还需要结合科技进步、产业升级、产业结构、财富增长、民生改善、社会公平等多个方面才能评估和衡量。同时，在实施“脱贫攻坚”工程的过程中，政府要保持开放的态度，依托法律、法规、标准、机制、手段、工具及其他参与者的支持，坚持真诚合作互利，深化各方合作，做到科学合理、精准施策、依法行政。
### 2.3.2 透明公开的信息公布
“脱贫攻坚”工程实施之后，信息公布应当尽可能完整、详细、时效、及时。政府信息公布应当对公众开放，公布的内容应当做到权威、客观、易理解、易操作、及时。公众需要知道的信息应当包括：各级党组织活动情况、政府动态、财政收入及支出项目、政府投入项目及结果、基建项目实施进度及成效、社会公共服务开展情况、民生改善情况等。同时，政府也应该积极听取公众的意见建议，及时行政、法规的修改和完善。
### 2.3.3 科技助推产业结构升级
“脱贫攻坚”工程的一个重要标志是科技和产业的飞跃。以至于生产经营方式和制度性规范发生变化，对社会、经济、文化、卫生等方面都产生深远影响。因此，“脱贫攻坚”工程也必须强调产业结构的升级，实现产业整合升级，推动生产力、生产关系、资本市场化、制度性规范改革。
### 2.3.4 时刻保持警惕、开拓创新
“脱贫攻坚”工程是一场面临严峻挑战的全面性工程，存在着巨大的挑战和风险。因此，“脱贫攻坚”工程也应当时刻保持警惕、开拓创新，促进经济发展与社会进步、促进国际合作与交流。在这方面，政府应该做到：

1. 提升国家治理水平、树立正确的治理观念、强化监督检查、完善社会监督体系。
2. 增强国际竞争力、提升国际合作水平，在国际事务中发挥示范效应。
3. 发展新兴产业、加强国际沟通合作，推动国际公平正义。
4. 提升政府决策与执行效率，健全决策机制、改善决策制度。
5. 深化制度建设，充分利用社会资源和人力，开拓市场化改革的空间。
6. 积极推进新旧模式转换，扩大对外开放。
