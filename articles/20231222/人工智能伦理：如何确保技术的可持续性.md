## 3.1 线性回归

线性回归是一种简单的监督学习算法，用于预测连续型目标变量。它假设目标变量与输入特征之间存在线性关系。线性回归的数学模型公式如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是目标变量，$x_1, x_2, \cdots, x_n$ 是输入特征，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是权重参数，$\epsilon$ 是误差项。

具体的操作步骤如下：

1. 数据预处理：对输入数据进行清洗和标准化处理。
2. 训练模型：使用梯度下降算法优化权重参数，以最小化误差项。
3. 评估模型：使用测试数据评估模型的性能。

## 3.2 逻辑回归

逻辑回归是一种简单的分类算法，用于预测二分类目标变量。它假设目标变量与输入特征之间存在线性关系。逻辑回归的数学模型公式如下：

$$
P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n)}}
$$

其中，$y$ 是目标变量，$x_1, x_2, \cdots, x_n$ 是输入特征，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是权重参数。

具体的操作步骤如下：

1. 数据预处理：对输入数据进行清洗和标准化处理。
2. 训练模型：使用梯度下降算法优化权重参数，以最大化目标变量的概率。
3. 评估模型：使用测试数据评估模型的性能。

支持向量机是一种强大的分类和回归算法，可以处理线性不可分和高维数据。它通过找到支持向量来将数据分割为多个区域，从而实现分类或回归。支持向量机的数学模型公式如下：

## 3.4 决策树

决策树是一种强大的分类算法，可以处理复杂的数据关系。它通过构建一颗树来表示不同的决策规则，从而实现分类。决策树的数学模型公式如下：

$$
f(x) = \left\{
\begin{aligned}
& g_1(x), & \text{if } x \in D_1 \\
& g_2(x), & \text{if } x \in D_2 \\
& \cdots \\
& g_n(x), & \text{if } x \in D_n
\end{aligned}
\right.
$$

其中，$f(x)$ 是目标变量，$g_1(x), g_2(x), \cdots, g_n(x)$ 是决策规则，$D_1, D_2, \cdots, D_n$ 是决策树的节点。

具体的操作步骤如下：

1. 数据预处理：对输入数据进行清洗和标准化处理。

随机森林是一种强大的分类和回归算法，可以处理高维数据和复杂的数据关系。它通过构建多个决策树并进行投票来实现分类或回归。随机森林的数学模型公式如下：

## 4.1 线性回归示例

### 4.1.1 数据准备

首先，我们需要准备一个线性回归示例的数据集。我们将使用一个简单的线性关系：$y = 2x + 3 + \epsilon$，其中$\epsilon$是随机噪声。

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成数据
np.random.seed(0)
x = np.linspace(-1, 1, 100)
y = 2 * x + 3 + np.random.normal(0, 0.1, 100)

# 绘制数据
plt.scatter(x, y)
plt.xlabel('x')
plt.ylabel('y')
plt.show()
```

### 4.1.2 线性回归模型

接下来，我们将使用线性回归模型来预测目标变量$y$。

```python
from sklearn.linear_model import LinearRegression

# 训练模型
model = LinearRegression()
model.fit(x.reshape(-1, 1), y)

# 预测目标变量
y_pred = model.predict(x.reshape(-1, 1))

# 绘制预测结果
plt.scatter(x, y, label='真实值')
plt.plot(x, y_pred, color='red', label='预测值')
plt.xlabel('x')
plt.ylabel('y')
plt.legend()
plt.show()
```

### 4.1.3 模型评估

最后，我们将使用均方误差（MSE）来评估模型的性能。

```python
from sklearn.metrics import mean_squared_error

# 计算均方误差
mse = mean_squared_error(y, y_pred)
print(f'均方误差：{mse}')
```
