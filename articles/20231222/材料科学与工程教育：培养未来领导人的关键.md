# 2.核心概念与联系

## 2.1 材料科学与工程的核心概念

材料科学与工程的核心概念包括：

1. 材料的性质和性能：材料的微观结构、宏观性能、稳定性、可 recycling 性等方面。
2. 材料的制造与处理：包括材料的制造、加工、修复、再利用等方面。
3. 材料的应用与服务：材料在各种工业和科技领域的应用，以及材料在不同环境中的服务性能。

## 2.2 材料科学与工程与其他学科的联系

材料科学与工程与其他学科之间的联系如下：

1. 与物理学的联系：材料科学与工程研究材料的微观结构和宏观性能，与物理学在研究材料的基本性质和性能方面有着密切的联系。
2. 与化学学的联系：材料科学与工程研究材料的组成、结构和性能，与化学学在研究材料的组成和性能方面有着密切的联系。
3. 与生物学的联系：材料科学与工程研究生物材料和生物接口的性能，与生物学在研究生物材料和生物接口方面有着密切的联系。
4. 与工程学的联系：材料科学与工程研究材料的制造、加工、修复和再利用，与工程学在研究材料制造和加工方面有着密切的联系。
