# 2.核心概念与联系
在图形学中，矩阵是一种用于表示向量的数据结构，矩阵可以用来表示变换、光照、纹理映射等。初等矩阵是指具有特定形式和特定属性的矩阵，例如单位矩阵、对称矩阵、正交矩阵等。初等矩阵在图形学中具有以下核心概念与联系：

1. 变换：变换是指将一个坐标系从另一个坐标系中转换到另一个坐标系的过程。在图形学中，变换主要包括平移、旋转、缩放、斜切等。初等矩阵可以用来表示这些变换，通过将矩阵乘以向量，可以实现向量在不同坐标系中的表示。

2. 光照：光照是指光线在物体表面发生的反射、折射、吸收等过程。在图形学中，光照主要包括环境光、点光源、平行光等。初等矩阵可以用来表示光照效果，通过将矩阵乘以向量，可以实现光线在不同坐标系中的表示。

3. 纹理映射：纹理映射是指将二维图像映射到三维模型表面的过程。在图形学中，纹理映射主要包括平面映射、环绕映射、重复映射等。初等矩阵可以用来表示纹理映射，通过将矩阵乘以向量，可以实现纹理在不同坐标系中的表示。

# 6.附录常见问题与解答
1. 初等矩阵与高级矩阵的区别是什么？

初等矩阵是指具有特定形式和特定属性的矩阵，例如单位矩阵、对称矩阵、正交矩阵等。高级矩阵是指不具备这些特定形式和特定属性的矩阵。

2. 初等矩阵在图形学中的应用范围是什么？

初等矩阵在图形学中的应用范围包括变换、光照、纹理映射等。

3. 如何计算初等矩阵的逆矩阵？

对于单位矩阵、对称矩阵、正交矩阵等初等矩阵，其逆矩阵可以通过简单的公式计算。对于其他初等矩阵，需要使用矩阵分解、矩阵求逆等方法计算逆矩阵。

4. 如何使用初等矩阵实现坐标系转换？

使用初等矩阵实现坐标系转换主要包括以下步骤：

1. 构建初等矩阵，例如旋转矩阵、平移矩阵、缩放矩阵等。

2. 将初等矩阵与向量进行乘法，得到转换后的向量。

3. 将转换后的向量用于后续的图形学计算。
