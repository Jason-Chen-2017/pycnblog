# 4.具体代码实例和详细解释说明

在这一节中，我们将通过一个具体的代码实例来演示如何处理缺失值、重复值和错误值。

## 4.1 缺失值处理的代码实例

### 4.1.1 数据集

我们使用以下数据集来演示缺失值处理的代码实例：

```python
import pandas as pd

data = {
    'name': ['Alice', 'Bob', 'Charlie', 'David'],
    'age': [25, 30, 35, 40],
    'salary': [70000, 80000, None, 100000]
}

df = pd.DataFrame(data)
```

### 4.1.2 缺失值处理

我们使用均值方法来填充缺失值：

```python
df['salary'].fillna(df['salary'].mean(), inplace=True)
```

### 4.1.3 结果

填充后的数据集如下：

```python
    name  age  salary
0  Alice  25  70000.0
1   Bob  30  80000.0
2 Charlie  35  85000.0
3  David  40 100000.0
```
