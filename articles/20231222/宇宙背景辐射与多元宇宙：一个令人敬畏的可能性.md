# 2.核心概念与联系
# 2.1宇宙背景辐射
宇宙背景辐射（CMB）是大气分子在宇宙早期的热膨胀过程中发出的微波辐射。它是现代宇宙学的一个关键证据，因为它提供了关于宇宙起源和宇宙大规模结构的重要信息。CMB 的发现得到了诺贝尔奖的荣誉，它的发现者是阿尔弗雷德·潘德拉（Arno Penzias）和罗伯特·沃尔夫（Robert W. Wilson）。

CMB 的温度为2.725 K（大约为-270.45 ℃），这是宇宙的最低温度。CMB 的分辨率非常高，这意味着它在宇宙中的分布非常均匀。CMB 的观测结果可以通过WMAP（Wilkinson Microwave Anisotropy Probe）和PLANCK（HFI）等天文望远镜得到。

$$

# 4.具体代码实例和详细解释说明
# 4.1宇宙背景辐射的计算
在这个例子中，我们将计算宇宙背景辐射在不同观测角度下的强度。我们将使用以下Python代码：

```python
import numpy as np
import matplotlib.pyplot as plt

def compute_cmb_intensity(theta):
    I0 = 2.0 * np.pi * (2.725 * 10**(-3))**4
    I = I0 * (0.5 + 0.5 * np.cos(2 * theta))
    return I

theta = np.linspace(0, np.pi, 100)
intensity = [compute_cmb_intensity(theta[i]) for i in range(len(theta))]

plt.plot(theta, intensity)
plt.xlabel('Angle (radians)')
plt.ylabel('Intensity')
plt.title('CMB Intensity vs. Angle')
plt.show()
```

这个代码首先导入了Python的NumPy和Matplotlib库。然后定义了一个函数`compute_cmb_intensity`，用于计算宇宙背景辐射在不同观测角度下的强度。接着，我们将`compute_cmb_intensity`函数应用于一个从0到$\pi$（180度）的等距角度数组，并将结果绘制在图像中。

        universe = np.random.rand(universe_size)
