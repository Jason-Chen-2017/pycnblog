                 

# 1.背景介绍

随着数据规模的不断增加，传统的计算机视觉模型已经无法满足需求。传统的计算机视觉模型通常是基于卷积神经网络（CNN）的，它们的主要缺点是无法捕捉到远程的关系和局部的关系，以及对于图像中的不连续的变化也不够灵活。因此，需要一种新的机制来解决这些问题。

在2017年，一篇论文《Squeeze-and-Excitation Networks: Achieving Efficient Representations with Squeezing and Attention Mechanisms》提出了一种新的机制——注意机制（Attention Mechanisms），它可以帮助模型更好地捕捉到远程的关系和局部的关系，以及更好地处理不连续的变化。从此，注意机制成为了计算机视觉领域的热门话题。
