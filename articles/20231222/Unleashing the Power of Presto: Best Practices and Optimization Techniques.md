                 

# 1.背景介绍

Presto 是一个高性能、分布式的 SQL 查询引擎，由 Facebook 开发并开源。它可以在大规模的数据集上执行交互式查询，具有高吞吐量和低延迟。Presto 的设计目标是提供一个可扩展的、高性能的查询引擎，用于处理大规模数据集和复杂的查询工作负载。

Presto 的核心组件包括查询引擎、查询计划器和连接器。查询引擎负责执行查询，查询计划器负责生成查询计划，连接器负责与数据源进行通信。Presto 支持多种数据源，如 HDFS、S3、Cassandra、HBase 和 MySQL。

Presto 的优势在于其高性能和灵活性。它可以在大规模数据集上执行交互式查询，并且可以轻松扩展到多个节点。此外，Presto 支持多种数据源，使得数据分析师可以在一个平台上查询来自不同数据源的数据。
