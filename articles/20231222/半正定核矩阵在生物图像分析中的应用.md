# 2.核心概念与联系

## 2.1 正定核矩阵

正定核矩阵（Positive definite core matrix，P-CM）是指一个实对称矩阵A，满足对于任何向量x，x^T * A * x > 0。其中x^T是向量x的转置，*表示矩阵乘法。正定核矩阵具有以下性质：

1. 正定核矩阵是对称的，即A = A^T。
2. 正定核矩阵的所有特征值都是正数。
3. 正定核矩阵可以用于计算距离、相似性等，因为它可以保证计算结果的正定性。

## 2.2 半正定核矩阵

半正定核矩阵（Half-positive definite core matrix，HPD-CM）是指一个实对称矩阵A，满足a_{ij} >= 0，其中i <= j。半正定核矩阵是正定核矩阵的一种特殊情况，它仅包含半正定元素。半正定核矩阵具有以下性质：

1. 半正定核矩阵是对称的，即A = A^T。
2. 半正定核矩阵的所有特征值都是非负数。
3. 半正定核矩阵可以用于计算距离、相似性等，因为它可以保证计算结果的半正定性。
