在本节中，我们将通过一个具体的代码实例来说明曼哈顿距离在物理学中的应用。

```python
def manhattan_distance(point1, point2):
    x1, y1 = point1
    x2, y2 = point2
    distance = abs(x1 - x2) + abs(y1 - y2)
    return distance

point1 = (1, 2)
point2 = (3, 4)

result = manhattan_distance(point1, point2)
print(result)
```

在上述代码中，我们定义了一个名为`manhattan_distance`的函数，该函数接受两个点的坐标作为输入，并计算它们之间的曼哈顿距离。在此示例中，我们将两个点的坐标分别设为（1，2）和（3，4），并调用`manhattan_distance`函数计算它们之间的曼哈顿距离。最后，我们将计算结果打印出来。
