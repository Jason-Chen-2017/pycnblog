## 4.1 松弛定义的代码实例

我们将通过一个简单的最短路径问题来展示松弛定义的代码实例。假设我们有一个有向图$G = (V, E)$，其中$V$是顶点集合，$E$是边集合。我们要求找到图$G$中从顶点$s$到顶点$t$的最短路径。

我们可以使用松弛定义来解决这个问题。我们可以设定一个允许的误差范围$\epsilon$，然后找到一个满足$d(s, t) \leq d^*(s, t) + \epsilon$的解，其中$d(s, t)$是从顶点$s$到顶点$t$的实际最短路径长度，$d^*(s, t)$是从顶点$s$到顶点$t$的最短路径长度。

```python
import heapq

def dijkstra(graph, start, end, epsilon):
    dist = {v: float('inf') for v in graph}
    prev = {v: None for v in graph}
    dist[start] = 0
    pq = [(0, start)]

    while pq:
        _, u = heapq.heappop(pq)
        if u == end:
            break
        for v, w in graph[u].items():
            if dist[v] > dist[u] + w + epsilon:
                dist[v] = dist[u] + w + epsilon
                prev[v] = u
                heapq.heappush(pq, (dist[v], v))

    return dist, prev

graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'C': 2, 'D': 5},
    'C': {'D': 1},
    'D': {}
}
start = 'A'
end = 'D'
epsilon = 1
dist, prev = dijkstra(graph, start, end, epsilon)
print(dist)
```

在这个代码实例中，我们使用了松弛定义来解决最短路径问题。我们设定了一个允许的误差范围$\epsilon = 1$，然后找到一个满足$d(s, t) \leq d^*(s, t) + \epsilon$的解。
