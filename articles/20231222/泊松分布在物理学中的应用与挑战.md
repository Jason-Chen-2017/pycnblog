# 4.具体代码实例和详细解释说明

```python
import numpy as np
import matplotlib.pyplot as plt

def poisson_pdf(x, lambda_):
    return (lambda_ ** x) * np.exp(-lambda_) / np.math.factorial(x)

def poisson_pmf(x, lambda_):
    return poisson_pdf(x, lambda_)

lambda_ = 3
x = np.arange(0, 10, 0.1)
pdf = np.array([poisson_pdf(i, lambda_) for i in x])

plt.plot(x, pdf)
plt.xlabel('x')
plt.ylabel('PDF')
plt.title('Poisson PDF')
plt.show()
```
上述代码实例中，我们首先定义了泊松分布的概率密度函数 `poisson_pdf` 和概率质量函数 `poisson_pmf`。然后，我们设定了参数 $\lambda = 3$，并计算了在 $0 \leq x < 10$ 的概率密度函数值。最后，我们使用 `matplotlib` 绘制了概率密度函数曲线。
