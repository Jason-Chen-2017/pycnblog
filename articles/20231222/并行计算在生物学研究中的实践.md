# 4.具体代码实例和详细解释说明
以下是一个基因组比对算法的具体代码实例和详细解释说明：

```
def needlman_wunsch(seq1, seq2):
    m = len(seq1)
    n = len(seq2)
    score = [[0 for j in range(n+1)] for i in range(m+1)]
    traceback = [[0 for j in range(n+1)] for i in range(m+1)]
    for i in range(m+1):
        for j in range(n+1):
            if i == 0:
                score[i][j] = -j
                traceback[i][j] = ''
            elif j == 0:
                score[i][j] = -i
                traceback[i][j] = ''
            elif seq1[i-1] == seq2[j-1]:
                score[i][j] = score[i-1][j-1] + 1
                traceback[i][j] = traceback[i-1][j-1] + seq1[i-1]
            else:
                score[i][j] = max(score[i-1][j], score[i][j-1]) - 1
                if score[i-1][j] > score[i][j-1]:
                    traceback[i][j] = traceback[i-1][j]
                else:
                    traceback[i][j] = traceback[i][j-1]
    align1 = traceback[m][n]
    align2 = align1[::-1]
    return align1, align2, score[m][n]
```

这个函数实现了Needlman-Wunsch算法，用于基因组序列比对。输入两个序列seq1和seq2，输出两个对齐结果align1和align2以及比对得分。具体实现过程如下：

1. 初始化一个m*n的二维数组score，用于存储比对得分；初始化一个m*n的二维数组traceback，用于存储比对过程中的信息。

2. 遍历seq1和seq2中的每个字符，根据字符相等情况更新score和traceback数组。

3. 当seq1[i-1] == seq2[j-1]时，说明这两个字符相等，比对得分加1；当seq1[i-1] != seq2[j-1]时，比对得分减1。

4. 当比对得分最大时，选择相应的对齐方向。

5. 最后返回两个对齐结果align1和align2以及比对得分。

Q：并行计算与分布式计算有什么区别？
A：并行计算和分布式计算都是高性能计算的一种方法，但它们的区别在于并行计算是同时处理多个任务，而分布式计算是将任务分布在多个计算节点上。并行计算通常用于处理大规模、复杂的问题，而分布式计算用于处理分布在多个计算节点上的问题。
