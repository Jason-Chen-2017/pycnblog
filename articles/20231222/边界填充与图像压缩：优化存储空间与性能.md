### 3.1 边界填充算法原理

边界填充算法的核心思想是通过填充图像边界来减少图像文件的大小。边界填充算法可以将图像边界填充为一定的颜色或模式，从而减少图像文件的大小。边界填充算法通常包括以下步骤：

1. 读取输入图像文件；
2. 检测图像边界；
3. 填充图像边界；
4. 保存填充后的图像文件。

### 3.2 边界填充算法具体操作步骤

边界填充算法的具体操作步骤如下：

1. 读取输入图像文件，获取图像的宽度、高度和像素值。
2. 遍历图像的每个像素点，检测像素点是否位于图像边界。
3. 如果像素点位于图像边界，则将其像素值填充为指定的颜色或模式。
4. 保存填充后的图像文件。

### 3.3 边界填充算法数学模型公式

边界填充算法的数学模型可以通过以下公式表示：

$$
f_{fill}(x, y) =
\begin{cases}
    c, & \text{if } (x = 0 \lor x = w - 1 \lor y = 0 \lor y = h - 1) \\
    f(x, y), & \text{otherwise}
\end{cases}
$$

其中，$f_{fill}(x, y)$ 表示填充后的图像像素值，$c$ 表示填充颜色，$f(x, y)$ 表示原始图像像素值，$w$ 表示图像宽度，$h$ 表示图像高度，$x$ 表示横坐标，$y$ 表示纵坐标。

import cv2

def boundary_fill(image, fill_color):
    rows, cols = image.shape[:2]
    filled_image = np.zeros((rows, cols, 3), dtype=np.uint8)
    for row in range(rows):
        for col in range(cols):
            if (row == 0 or row == rows - 1 or col == 0 or col == cols - 1):
                filled_image[row, col] = fill_color
            else:
                filled_image[row, col] = image[row, col]
    return filled_image

# 读取输入图像文件
