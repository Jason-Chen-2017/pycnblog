## 2.1 基因组

基因组是一个组织或细胞的全部 DNA 质的集合，包括所有的基因和非基因区域。基因组是遗传信息的载体，包含了组织或细胞所有的基因。基因组的组成单位是核苷酸，主要包括脱氢胺基（A）、脱氢葡萄糖基（T）、脱氢腺苷基（C）和脱氢腺唑基（G）。

```python
    expression_data = f.readlines()

import pandas as pd

expression_df = pd.DataFrame(expression_data, columns=['基因 ID', '表达水平'])
expression_df['表达水平'] = expression_df['表达水平'].astype(float)

# 基因表达分析
from scipy import stats

# 计算基因表达的均值和方差
mean_expression = expression_df['表达水平'].mean()
variance_expression = expression_df['表达水平'].var()

# 使用 t 检验对基因表达进行分析
t_statistic, p_value = stats.ttest_ind(expression_df['表达水平'], expression_df['表达水平'])

# 输出结果
print(f'基因表达均值：{mean_expression}')
print(f'基因表达方差：{variance_expression}')
print(f't 统计量：{t_statistic}')
print(f'p 值：{p_value}')
```
