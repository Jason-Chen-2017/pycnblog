# 2.核心概念与联系

生成模型的故障分析主要涉及以下几个核心概念：

1. 模型过拟合：模型过拟合是指模型在训练数据上的表现非常好，但在新的、未见过的数据上表现很差的现象。过拟合会导致模型在实际应用中的性能非常差。

2. 模型复杂性：模型复杂性是指模型中参数的数量或结构的复杂程度。模型过于复杂可能导致训练速度慢、过拟合等问题。

3. 训练速度慢：训练速度慢是指模型在训练过程中需要很长时间才能收敛到一个满意的结果。

4. 模型泄漏：模型泄漏是指模型在训练过程中不应该具有的信息，但是却被模型学到了。这种信息可能会导致模型在实际应用中表现不佳。

5. 模型稳定性：模型稳定性是指模型在不同的数据集、不同的参数设置下的表现是否稳定。

6. 模型可解释性：模型可解释性是指模型的决策过程是否可以被人类理解和解释。
