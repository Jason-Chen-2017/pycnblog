## 3.3 数学模型公式详细讲解

### 3.3.1 线性回归

线性回归（Linear Regression，LR）是一种常用的机器学习算法。线性回归的主要公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是目标变量，$x_1, x_2, \cdots, x_n$ 是自变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。

### 3.3.2 逻辑回归

逻辑回归（Logistic Regression，LR）是一种常用的机器学习算法。逻辑回归的主要公式为：

$$
P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \cdots - \beta_nx_n}}
$$

其中，$P(y=1|x)$ 是目标变量，$x_1, x_2, \cdots, x_n$ 是自变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

### 3.3.3 支持向量机

支持向量机（Support Vector Machine，SVM）是一种常用的机器学习算法。支持向量机的主要公式为：

$$
\min_{\omega, b} \frac{1}{2}\|\omega\|^2 \\
s.t. \quad y_i(\omega \cdot x_i + b) \geq 1, \quad i = 1, 2, \cdots, n
$$

其中，$\omega$ 是权重向量，$b$ 是偏置项，$y_i$ 是目标变量，$x_i$ 是自变量。

### 3.3.4 决策树

决策树（Decision Tree）是一种常用的机器学习算法。决策树的主要公式为：

$$
\text{if } x_1 \text{ is } A_1 \text{ then } y = f_1 \\
\text{else if } x_2 \text{ is } A_2 \text{ then } y = f_2 \\
\vdots \\
\text{else if } x_n \text{ is } A_n \text{ then } y = f_n
$$

其中，$x_1, x_2, \cdots, x_n$ 是自变量，$A_1, A_2, \cdots, A_n$ 是条件，$f_1, f_2, \cdots, f_n$ 是目标变量。

### 3.3.5 随机森林

随机森林（Random Forest）是一种常用的机器学习算法。随机森林的主要公式为：

$$
\hat{y} = \frac{1}{K} \sum_{k=1}^K f_k(x)
$$

其中，$\hat{y}$ 是预测值，$K$ 是决策树的数量，$f_k(x)$ 是第$k$个决策树的预测值。

### 3.3.6 梯度下降

梯度下降（Gradient Descent）是一种常用的优化算法。梯度下降的主要公式为：

$$
\omega_{t+1} = \omega_t - \eta \nabla_{\omega} L(\omega_t)
$$

其中，$\omega_{t+1}$ 是更新后的权重，$\omega_t$ 是更新前的权重，$\eta$ 是学习率，$\nabla_{\omega} L(\omega_t)$ 是损失函数的梯度。

## 4.1 线性回归

### 4.1.1 数据收集与预处理

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler

# 加载数据

# 划分训练集和测试集

X_train = scaler.fit_transform(X_train)
X_test = scaler.transform(X_test)
```

```python
from sklearn.linear_model import LinearRegression

# 创建模型
model = LinearRegression()

# 训练模型
model.fit(X_train, y_train)
```

### 4.1.3 模型应用与优化

```python
# 预测
y_pred = model.predict(X_test)

# 评估
from sklearn.metrics import mean_squared_error

mse = mean_squared_error(y_test, y_pred)
print('MSE:', mse)
```

## 4.6 梯度下降

### 4.6.1 数据收集与预处理

```python
# 加载数据
data = pd.read_csv('data.csv')

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(data.drop('target', axis=1), data['target'], test_size=0.2, random_state=42)

# 标准化
scaler = StandardScaler()
X_train = scaler.fit_transform(X_train)
X_test = scaler.transform(X_test)
```

### 4.6.2 模型选择与训练

```python
# 定义损失函数
def loss(y_true, y_pred):
    return np.mean((y_true - y_pred) ** 2)

# 定义梯度下降函数
def gradient_descent(X, y, learning_rate, num_iterations):
    weights = np.zeros(X.shape[1])
    for _ in range(num_iterations):
        y_pred = np.dot(X, weights)
        gradients = 2 * (y - y_pred) * X
        weights -= learning_rate * gradients
    return weights

# 训练模型
weights = gradient_descent(X_train, y_train, learning_rate=0.01, num_iterations=1000)
```

### 4.6.3 模型应用与优化

```python
# 预测
y_pred = np.dot(X_test, weights)

# 评估
mse = mean_squared_error(y_test, y_pred)
print('MSE:', mse)
```
