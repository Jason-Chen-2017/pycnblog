## 1.背景介绍

Java 反射的核心是 `java.lang.reflect` 包，包含了 `Field`、`Method`、`Constructor` 等类，用于表示类、接口、构造函数、方法、变量等元素。通过这些类，我们可以在运行时动态地访问和操作这些元素。

Java 反射的主要应用场景包括：

- 框架和库的开发，如 Spring、Hibernate、JPA 等。
- 动态代理、AOP 等。
- 序列化、反序列化等。
- 测试、工具等。
