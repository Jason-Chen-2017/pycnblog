# 2.核心概念与联系

对偶基是一种线性代数方法，它可以用于优化问题、有限元分析、数值解析等领域。在物理模型中，对偶基可以用于表示物理量的变换、求解部分差方方程以及求解偏微分方程等。对偶基的核心概念包括：

2.对偶空间：对偶空间是原始空间的对偶空间，用于表示物理量的变换。例如，在有限元分析中，原始空间是元空间，对偶空间是强形状函数空间；在偏微分方程求解中，原始空间是函数空间，对偶空间是弱形状函数空间。

d.通过求解对偶问题方程组，得到对偶解。
