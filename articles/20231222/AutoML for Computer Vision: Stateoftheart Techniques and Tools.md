# 2.核心概念与联系

## 2.1 机器学习（ML）

机器学习是一种通过从数据中学习模式的方法，以便进行预测或决策的方法。机器学习算法可以分为两类：

- 监督学习：在这种类型的算法中，模型通过观察已知输入和输出数据来学习。这种类型的算法通常用于分类和回归任务。
- 无监督学习：在这种类型的算法中，模型通过观察输入数据本身来学习。这种类型的算法通常用于聚类和降维任务。

## 2.2 深度学习（DL）

深度学习是一种通过神经网络进行机器学习的方法。神经网络是一种模拟人脑神经元的计算模型，由多个节点和它们之间的连接组成。这些节点称为神经元或神经网络中的单元。神经网络可以学习从大量数据中提取特征，并使用这些特征进行预测或决策。

深度学习的一个主要优势是它可以处理大规模数据集，并且在许多应用中表现出色，例如图像识别、自然语言处理等。

## 2.3 计算机视觉（CV）

计算机视觉是一种通过计算机处理和理解图像和视频的科学和技术。计算机视觉的主要任务是通过从图像中提取特征来识别和分类对象。这种技术在许多应用中得到了广泛的应用，例如人脸识别、自动驾驶汽车、医疗诊断等。

计算机视觉的主要任务包括：

- 图像处理：这是一种通过从图像中提取特征的方法来识别和分类对象的方法。
- 图像分类：这是一种通过观察已知输入和输出数据来学习的方法，以便进行预测或决策的方法。
- 对象检测：这是一种通过观察输入数据本身来学习的方法，以便进行预测或决策的方法。

## 2.4 AutoML

AutoML是一种通过自动化机器学习模型的过程来构建高性能模型的方法。AutoML已经成为了机器学习社区的热门话题，主要是因为它能够帮助用户快速构建高性能的机器学习模型。

AutoML可以分为两类：

- 自动特征工程：这是一种通过自动化选择和提取特征的方法来构建高性能模型的方法。
- 自动模型选择：这是一种通过自动化选择和优化模型的方法来构建高性能模型的方法。

深度学习是一种通过神经网络进行机器学习的方法，而AutoML是一种通过自动化机器学习模型的方法来构建高性能模型的方法。这意味着深度学习是一种机器学习方法，而AutoML是一种用于构建高性能模型的方法。
