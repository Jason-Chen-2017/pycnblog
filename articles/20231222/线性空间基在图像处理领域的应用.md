## 3.1 线性空间基的构造
线性空间基的构造主要包括以下步骤：
1. 选择一组线性无关的向量集合，如原始图像的像素值、边缘信息、纹理信息等。
2. 对选定的向量集合进行正交化处理，以确保其线性无关。
3. 对正交化后的向量集合进行归一化处理，以确保其单位长度。
4. 将归一化后的向量集合作为线性空间基，用于表示图像的特征和结构。

## 3.2 图像压缩
图像压缩是图像处理领域中的一个重要技术，其主要目标是将原始图像的大量信息压缩为较小的尺寸，以实现存储和传输的便捷性。线性空间基可以用来实现图像的压缩，具体操作步骤如下：
1. 根据原始图像的特征和结构，构造一组线性空间基。
2. 对原始图像进行线性空间基展开表示，即将原始图像表示为线性空间基的线性组合。
3. 对线性空间基展开表示的系数进行压缩处理，以减少存储和传输的数据量。
4. 将压缩后的系数与线性空间基相乘，得到压缩后的图像。

## 3.3 图像恢复
图像恢复是图像处理领域中的一个重要技术，其主要目标是将损坏的图像信息恢复为原始状态，以实现图像的清晰度和质量。线性空间基可以用来实现图像的恢复，具体操作步骤如下：

2. 对损坏的图像进行线性空间基展开表示，即将损坏的图像表示为线性空间基的线性组合。
3. 根据线性空间基展开表示的系数，恢复原始图像的信息。

        if dot_product != 0:
            alpha = dot_product / np.linalg.norm(basis[i])
            basis[j] -= alpha * basis[i]

# 归一化处理
for i in range(len(basis)):
