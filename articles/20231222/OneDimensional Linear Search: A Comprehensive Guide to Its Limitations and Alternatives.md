# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
一维线性搜索算法的核心思想是逐个检查数组中的每个元素，直到找到目标元素或者数组末尾。具体的操作步骤如下：

1. 从数组的第一个元素开始检查。
2. 将当前元素与目标元素进行比较。
3. 如果当前元素与目标元素相等，则返回当前元素的下标。
4. 如果当前元素与目标元素不相等，则将当前元素向后移动一个位置，并重复步骤2-4。
5. 如果数组末尾的元素与目标元素不相等，则返回-1，表示目标元素不在数组中。

一维线性搜索的时间复杂度为O(n)，其中n是数组的长度。这是因为在最坏情况下，需要检查整个数组的所有元素。因此，在大数据集合中，一维线性搜索的性能可能不佳。

    for i in range(len(arr)):

            return i
    return -1
