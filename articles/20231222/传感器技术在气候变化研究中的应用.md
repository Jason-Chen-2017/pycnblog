# 4.具体代码实例和详细解释说明

## 4.1 温度测量代码实例
```python
import time
import Adafruit_DHT

# 设置传感器类型和测量位置
SENSOR = Adafruit_DHT.DHT22
PIN = 4

# 测量温度和湿度
humidity, temperature = Adafruit_DHT.read_retry(SENSOR, PIN)

# 打印测量结果
if temperature is not None:
    print('Temperature: {0:0.1f}C'.format(temperature))
    print('Humidity: {0:0.1f}%'.format(humidity))
else:
    print('Failed to read sensor')
```
上述代码实例使用Python语言编写，通过Adafruit_DHT库进行温度和湿度的测量。代码首先导入时间和Adafruit_DHT库，然后设置传感器类型和测量位置。接着，通过Adafruit_DHT.read_retry()函数进行温度和湿度的测量，并打印测量结果。
