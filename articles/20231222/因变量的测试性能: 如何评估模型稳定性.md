# 2.核心概念与联系
在进行因变量的测试性能评估之前，我们需要了解一些核心概念。首先，我们需要了解什么是因变量（dependent variable）和什么是自变量（independent variable）。因变量是受影响的变量，而自变量是影响因变量的变量。在实际应用中，我们通常关注因变量的变化，以评估模型的性能。
