# 4.具体代码实例和详细解释
以下是一个简单的监控系统的具体代码实例和详细解释：

```python
import json
import requests
from collections import defaultdict

# 数据收集
def collect_data():
    url = 'http://example.com/data'
    response = requests.get(url)
    data = json.loads(response.text)
    return data

# 数据处理
def process_data(data):
    cleaned_data = []
    for item in data:
        if item['type'] == 'log':
            cleaned_data.append({
                'timestamp': item['timestamp'],
                'level': item['level'],
                'message': item['message']
            })
    return cleaned_data

# 数据存储
def store_data(data):
    db = defaultdict(int)
    for item in data:
        db[item['level']] += 1
    return db

# 数据分析
def analyze_data(data):
    total = sum(data.values())
    return {
        'total': total,
        'percentage': {k: v / total * 100 for k, v in data.items()}
    }

# 数据展示
def show_data(data):
    print('Total:', data['total'])
    print('Percentage:')
    for k, v in data['percentage'].items():
        print(f'{k}: {v:.2f}%')

# 主函数
def main():
    data = collect_data()
    cleaned_data = process_data(data)
    stored_data = store_data(cleaned_data)
    analyzed_data = analyze_data(stored_data)
    show_data(analyzed_data)

if __name__ == '__main__':
    main()
```

上述代码实例主要包括以下部分：

1. 数据收集：使用`requests`库发送HTTP请求，获取数据。

2. 数据处理：将获取到的数据进行清洗，只保留日志类型的数据。

3. 数据存储：使用`collections`库的`defaultdict`类型存储数据。

4. 数据分析：计算总数和百分比。

5. 数据展示：使用`print`函数输出结果。
