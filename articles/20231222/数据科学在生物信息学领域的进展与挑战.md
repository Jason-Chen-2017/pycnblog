                 

# 1.背景介绍

生物信息学是一门研究生物科学领域数据的科学。随着生物科学领域产生大量数据，如基因组数据、蛋白质结构数据、生物路径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径
