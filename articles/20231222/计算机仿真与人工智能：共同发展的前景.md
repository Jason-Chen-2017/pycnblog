## 4.1 模拟优化

以下是一个简单的模拟优化算法的Python代码实例：

```python
import numpy as np

def objective_function(params):
    # 定义目标函数
    return np.sum(params**2)

def simulated_annealing(initial_params, temperature, cooling_rate, num_iterations):
    current_params = initial_params
    current_value = objective_function(current_params)
    best_params = current_params
    best_value = current_value

    for _ in range(num_iterations):
        new_params = current_params + np.random.randn(*current_params.shape) * temperature
        new_value = objective_function(new_params)

        if new_value < best_value or np.random.rand() < np.exp((best_value - new_value) / temperature):
            best_params = new_params
            best_value = new_value

        current_params = new_params
        temperature *= cooling_rate

    return best_params

initial_params = np.random.randn(5)
temperature = 100
cooling_rate = 0.99
num_iterations = 1000

optimized_params = simulated_annealing(initial_params, temperature, cooling_rate, num_iterations)
```

在这个例子中，我们定义了一个简单的目标函数，即参数的平方和。然后，我们使用模拟优化算法来优化这个目标函数。算法的主要步骤包括初始化参数、生成模拟数据、评估结果、更新参数和重复这些步骤。
