### 3.1.1 随机变量

随机变量是从一个概率空间（sample space）上定义的函数，它将随机事件映射到实数域上。随机变量可以分为两类：离散随机变量和连续随机变量。离散随机变量的取值是离散的，即只能取有限个或无限个但只有有限多个不同的值。连续随机变量的取值是连续的，即可以取有限个或无限个不同的值。

### 3.1.2 随机事件

随机事件是一个子集（子空间）的概率空间，它表示某个特定的结果或结果集合。随机事件可以分为两类：独立事件和相依事件。独立事件之间的发生不会影响彼此的概率，而相依事件之间的发生会影响彼此的概率。

### 3.1.3 概率度量

概率度量是一个函数，它将随机事件映射到一个概率值（0到1之间的数字）上。概率度量可以用来描述随机事件的可能性和发生的频率。

### 3.1.4 期望

期望是一个随机变量的数学期望，它表示随机变量的平均值。期望可以用来描述随机变量的中心趋势和预测值。

### 3.1.5 方差

方差是一个随机变量的数学方差，它表示随机变量的波动程度和不确定性。方差可以用来描述随机变量的稳定性和可预测性。

## 4.1 随机过程的Python代码实例

在这个例子中，我们将实现一个简单的离散随机过程模型，用来描述硬币投掷的结果。

```python
import numpy as np

# 定义硬币投掷的随机过程模型
def coin_toss():
    # 定义硬币面值
    sides = ['Heads', 'Tails']
    # 定义硬币投掷的概率度量
    probabilities = [0.5, 0.5]
    # 生成硬币投掷的结果
    result = np.random.choice(sides, p=probabilities)
    return result

# 测试硬币投掷的随机过程模型
for i in range(10):
    print(coin_toss())
```

在这个例子中，我们首先导入了numpy库，然后定义了一个硬币投掷的随机过程模型，该模型包括硬币面值、硬币投掷的概率度量和生成硬币投掷的结果。最后，我们使用for循环测试了硬币投掷的随机过程模型，并打印了10次硬币投掷的结果。

## 4.2 人群流动模型的Python代码实例

在这个例子中，我们将实现一个简单的连续人群流动模型，用来描述人群在一条直线上的运动和分布。

```python
import numpy as np

# 定义人群流动的随机过程模型
def random_walk():
    # 定义人群流动的步长
    step_length = 1
    # 定义人群流动的概率度量
    probabilities = [0.5, 0.5]
    # 初始化人群的位置
    position = 0
    # 生成人群流动的结果
    for i in range(10):
        direction = np.random.choice(['left', 'right'], p=probabilities)
        if direction == 'left':
            position -= step_length
        else:
            position += step_length
    return position

# 测试人群流动的随机过程模型
for i in range(10):
    print(random_walk())
```

在这个例子中，我们首先导入了numpy库，然后定义了一个人群流动的随机过程模型，该模型包括人群流动的步长、人群流动的概率度量和生成人群流动的结果。最后，我们使用for循环测试了人群流动的随机过程模型，并打印了10次人群流动的结果。
