## 3.1 线性回归
线性回归是一种常见的机器学习算法，用于预测连续变量的值。它的基本思想是根据一组已知的输入和输出数据，找到一个最佳的直线，使得这个直线可以最好地拟合这组数据。线性回归的数学模型如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$是输出变量，$x_1, x_2, \cdots, x_n$是输入变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$是参数，$\epsilon$是误差。

线性回归的具体操作步骤如下：

1. 收集数据：首先，我们需要收集一组已知的输入和输出数据。
2. 计算参数：根据这组数据，我们可以使用最小二乘法来计算参数$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$。
3. 使用参数拟合数据：使用计算出的参数，我们可以得到一个直线，这个直线可以用来预测输出变量的值。

## 3.2 逻辑回归
逻辑回归是另一种常见的机器学习算法，用于预测二值变量的值。它的基本思想是根据一组已知的输入和输出数据，找到一个最佳的分割面，使得这个分割面可以最好地分割这组数据。逻辑回归的数学模型如下：

$$
P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n)}}
$$

其中，$P(y=1|x)$是输出变量的概率，$x_1, x_2, \cdots, x_n$是输入变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$是参数。

逻辑回归的具体操作步骤如下：

1. 收集数据：首先，我们需要收集一组已知的输入和输出数据。
2. 计算参数：根据这组数据，我们可以使用最大似然法来计算参数$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$。
3. 使用参数预测输出变量的值：使用计算出的参数，我们可以得到一个分割面，这个分割面可以用来预测输出变量的值。

## 4.1 线性回归实例
我们将通过一个简单的例子来演示如何使用线性回归算法来预测房价。我们的输入变量是房屋面积，输出变量是房价。我们的数据如下：

| 面积 | 价格 |
| --- | --- |
| 50 | 30000 |
| 80 | 45000 |
| 100 | 60000 |
| 120 | 75000 |
| 150 | 95000 |

我们可以使用Python的scikit-learn库来实现线性回归算法。首先，我们需要导入库和数据：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

X = np.array([50, 80, 100, 120, 150]).reshape(-1, 1)
y = np.array([30000, 45000, 60000, 75000, 95000])
```

接下来，我们可以创建一个线性回归模型，并使用模型来拟合数据：

```python
model = LinearRegression()
model.fit(X, y)
```

最后，我们可以使用模型来预测新的房价：

```python
print(model.predict([130]))
```

## 4.2 逻辑回归实例
我们将通过一个简单的例子来演示如何使用逻辑回归算法来预测顾客是否会购买产品。我们的输入变量是年龄和收入，输出变量是是否购买。我们的数据如下：

| 年龄 | 收入 | 购买 |
| --- | --- | --- |
| 20 | 30000 | 0 |
| 30 | 40000 | 1 |
| 40 | 50000 | 1 |
| 50 | 60000 | 1 |
| 60 | 70000 | 0 |

我们可以使用Python的scikit-learn库来实现逻辑回归算法。首先，我们需要导入库和数据：

```python
import numpy as np
from sklearn.linear_model import LogisticRegression

X = np.array([20, 30, 40, 50, 60]).reshape(-1, 1)
Y = np.array([0, 1, 1, 1, 0])
```

接下来，我们可以创建一个逻辑回归模型，并使用模型来拟合数据：

```python
model = LogisticRegression()
model.fit(X, Y)
```

最后，我们可以使用模型来预测新的购买决策：

```python
print(model.predict([35, 55000]))
```
