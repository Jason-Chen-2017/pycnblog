```python
import numpy as np

def iterative_method(f, df, x0, steps):
    for i in range(steps):
        x1 = x0 - f(x0) / df(x0)
        x0 = x1
    return x1

def f(x):
    return x**2 - 5*x + 4

def df(x):
    return 2*x - 5

x0 = 1
steps = 100
result = iterative_method(f, df, x0, steps)
print("Result: ", result)
```

在这个例子中，我们使用了一种称为牛顿法的一般迭代法。牛顿法是一种求解方程组的方法，它通过在每一次迭代中使用函数的梯度来逼近解决方案。在这个例子中，我们使用了Python的NumPy库来计算梯度，并使用了一个简单的函数来表示方程组。
