# 2.核心概念与联系

数据清洗的核心概念包括：

1. 数据质量：数据质量是指数据的准确性、完整性、一致性、时效性和可靠性等方面的度量。数据清洗的目的就是提高数据质量，以便更好地支持数据分析和决策。

2. 数据清理：数据清理是指对原始数据进行检查、修正和删除不准确、不完整、重复或错误的数据。数据清理的主要手段包括数据校验、数据纠正、数据去重和数据删除等。

3. 数据转换：数据转换是指将原始数据转换为适合分析的格式。数据转换的主要手段包括数据类型转换、数据格式转换、数据单位转换和数据编码转换等。

4. 数据整理：数据整理是指对原始数据进行排序、分组、过滤和聚合等操作，以便更好地支持数据分析。数据整理的主要手段包括数据排序、数据分组、数据过滤和数据聚合等。

5. 数据校验：数据校验是指对原始数据进行检查，以确保数据满足一定的规则和约束条件。数据校验的主要手段包括范围校验、格式校验、唯一性校验和完整性校验等。

6. 数据质量评估：数据质量评估是指对数据质量进行系统性评估，以确定数据是否满足预期的质量标准。数据质量评估的主要方法包括数据质量指标、数据质量报告和数据质量审计等。

这些核心概念之间存在着密切的联系，数据清洗是一个紧密相连、相互影响的过程。例如，数据清理和数据校验是数据质量的重要组成部分，数据转换和数据整理是数据分析的重要支持，数据质量评估是数据清洗过程的反馈和优化的基础。因此，在进行数据清洗时，需要综合考虑这些概念和联系，以确保数据的准确性、完整性、一致性、时效性和可靠性等方面的质量。
