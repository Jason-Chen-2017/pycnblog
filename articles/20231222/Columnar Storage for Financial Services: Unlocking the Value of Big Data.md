# 2.核心概念与联系
## 2.1列式存储的基本概念
列式存储是一种数据库存储技术，它将数据按照列而非行进行存储。这种存储方式可以提高数据压缩率、加速数据查询和分析。列式存储技术主要适用于大数据环境，特别是涉及到大量数据和复杂查询的场景。
