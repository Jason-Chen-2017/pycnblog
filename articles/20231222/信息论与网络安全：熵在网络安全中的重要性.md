## 1.2 核心概念与联系

### 1.2.1 信息熵

信息熵是信息论中的一个重要概念，用来度量信息的不确定性。信息熵的公式为：

$$
H(X)=-\sum_{i=1}^{n}P(x_i)\log_2 P(x_i)
$$

其中，$X$ 是一个随机变量，取值为 $x_1, x_2, \dots, x_n$ 中的一个；$P(x_i)$ 是 $x_i$ 的概率。

信息熵的单位是比特（bit），表示信息的最小单位。信息熵的大小决定了信息的不确定性：

- 当信息熵较大时，信息的不确定性也较大，说明信息中包含的信息量较多；
- 当信息熵较小时，信息的不确定性较小，说明信息中包含的信息量较少。

### 1.2.2 条件熵

条件熵是信息论中的一个概念，用来度量已知某个条件下的信息不确定性。条件熵的公式为：

$$
H(X|Y)=-\sum_{i=1}^{n}\sum_{j=1}^{m}P(x_i, y_j)\log_2 P(x_i|y_j)
$$

其中，$X$ 和 $Y$ 是两个随机变量，$x_i$ 和 $y_j$ 是它们的取值；$P(x_i|y_j)$ 是 $x_i$ 给定 $y_j$ 时的概率。

条件熵的大小决定了已知某个条件下信息的不确定性：

- 当条件熵较大时，已知某个条件下信息的不确定性较大，说明已知条件下信息中包含的信息量较多；
- 当条件熵较小时，已知某个条件下信息的不确定性较小，说明已知条件下信息中包含的信息量较少。

### 1.2.3 互信息

互信息是信息论中的一个概念，用来度量两个随机变量之间的相关性。互信息的公式为：

$$
I(X;Y)=\sum_{i=1}^{n}\sum_{j=1}^{m}P(x_i, y_j)\log_2 \frac{P(x_i, y_j)}{P(x_i)P(y_j)}
$$

其中，$X$ 和 $Y$ 是两个随机变量，$x_i$ 和 $y_j$ 是它们的取值。

互信息的大小决定了两个随机变量之间的相关性：

- 当互信息较大时，两个随机变量之间的相关性较大，说明它们之间存在一定的信息传递关系；
- 当互信息较小时，两个随机变量之间的相关性较小，说明它们之间的信息传递关系较弱。

熵在网络安全中有着重要的应用价值。例如，在密码学中，熵用来度量密码的强度；在数据传输中，熵用来度量数据的完整性；在网络安全中，熵用来度量系统的安全性和可靠性。

### 1.4.1 信息熵计算

```python
import numpy as np

def entropy(prob):
    return -np.sum(prob * np.log2(prob))

prob = np.array([0.1, 0.3, 0.2, 0.4])
print("信息熵:", entropy(prob))
```

### 1.4.2 条件熵计算

```python
import numpy as np

def conditional_entropy(prob, condition_prob):
    return -np.sum(prob * np.log2(prob / condition_prob))

prob = np.array([0.1, 0.3, 0.2, 0.4])
condition_prob = np.array([0.6, 0.2, 0.1, 0.1])
print("条件熵:", conditional_entropy(prob, condition_prob))
```
