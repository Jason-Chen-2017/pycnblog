# 2.核心概念与联系

计算几何的核心概念包括点、线、多边形、凸包、凸包算法等。这些概念是计算几何算法的基础，同时也是计算几何问题的核心内容。

## 2.1 点、线、多边形

在计算几何中，点表示二维空间中的一个坐标，线表示两个点连接的直线，多边形表示由多个点连接的闭合曲线。

### 2.1.1 点

点在计算几何中表示为二维空间中的一个坐标（x, y）。例如，点A可以表示为（1, 2），点B可以表示为（3, 4）。

### 2.1.2 线

线在计算几何中表示为两个点连接的直线。例如，线AB可以表示为点A和点B的连线，其中点A表示为（1, 2），点B表示为（3, 4）。

### 2.1.3 多边形

多边形在计算几何中表示为由多个点连接的闭合曲线。例如，一个三角形可以表示为点A、点B和点C连接形成的闭合曲线，其中点A表示为（1, 2），点B表示为（3, 4），点C表示为（5, 6）。

## 2.2 凸包、凸包算法

凸包是计算几何中一个重要的概念，它是一个多边形的一种特殊形式，其内部包含所有点，而外部不包含任何点。凸包算法是计算几何中一个重要的问题，用于求解给定多边形的凸包。

### 2.2.1 凸包

凸包在计算几何中表示为一个多边形，其内部包含所有点，而外部不包含任何点。例如，一个矩形可以看作是一个凸包，而一个五边形可能不是一个凸包。

### 2.2.2 凸包算法

凸包算法是计算几何中一个重要的问题，用于求解给定多边形的凸包。凸包算法可以分为两种类型：一种是基于分割的算法，另一种是基于点的算法。

基于分割的算法将多边形分为多个子多边形，然后检查每个子多边形是否是凸包。如果是，则将其加入到凸包中。这种方法通常需要较多的计算资源和时间。

基于点的算法将给定多边形的点一一检查，然后将其加入到凸包中。如果一个点不在凸包内，则将其从凸包中移除。这种方法通常更高效，但可能需要较多的空间资源。

def is_point_in_polygon(point, polygon):
    is_in = False
    for i in range(len(polygon)):
        x1 = polygon[i][0] - point[0]
        y1 = polygon[i][1] - point[1]
        x2 = polygon[(i + 1) % len(polygon)][0] - point[0]

        cross_product = x1 * y2 - x2 * y1
        if cross_product != 0:

    return is_in

Q: 计算几何中的向量叉积有哪些应用？

A: 计算几何中的向量叉积有多个应用，例如：

1. 判断两个线段是否相交：通过计算两个线段的向量叉积，可以判断它们是否相交。

Q: 计算几何中的点对距离有哪些类型？
