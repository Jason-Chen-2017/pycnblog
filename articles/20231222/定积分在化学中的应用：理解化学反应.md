$$

$$

其中，$k$ 是常数。

2. 积分的常数积分规则：

$$
\int_{a}^{b} cf(x) dx = c \int_{a}^{b} f(x) dx

其中，$c$ 是常数。

3. 积分的反对称性：

$$
\int_{a}^{b} f(x) dx = -\int_{b}^{a} f(x) dx

$$
\int_{a}^{b} f(x) dx = \int_{a}^{c} f(x) dx + \int_{c}^{b} f(x) dx

```python
import matplotlib.pyplot as plt

# 定义化学反应速率函数
def reaction_rate(t):
    return -0.5 * np.exp(-t)

# 定义反应产物质量函数
def product_mass(t):
    return np.trapz(reaction_rate, t)

# 生成时间数组

# 计算反应产物质量
m_B = product_mass(t)

# 绘制反应产物质量与时间关系
plt.plot(t, m_B)
plt.xlabel('Time (s)')
plt.ylabel('Product Mass (g)')
plt.title('Product Mass vs Time')
plt.show()
```

1. 首先，我们导入了 `numpy` 和 `matplotlib.pyplot` 库，用于数值计算和图形绘制。
2. 然后，我们定义了化学反应速率函数 `reaction_rate`，它是一个指数函数。
3. 接着，我们定义了反应产物质量函数 `product_mass`，它使用 `numpy.trapz` 函数进行数值积分。
4. 我们生成了时间数组 `t`，从 0 到 10，步长为 100。
5. 我们计算了反应产物质量 `m_B`，并使用 `matplotlib.pyplot` 库绘制了反应产物质量与时间的关系。
