```python
import numpy as np

class GravitySimulator:
    def __init__(self, mass1, mass2, G):
        self.mass1 = mass1
        self.mass2 = mass2
        self.G = G
        self.r = np.random.rand(3)
        self.v = np.random.rand(3)
        self.t = 0

    def update(self, dt):
        r_diff = self.r - self.mass2
        r_mag = np.linalg.norm(r_diff)
        F = self.G * self.mass1 * self.mass2 / r_mag**2
        a = F / self.mass1
        self.v += a * dt
        self.r += self.v * dt
        self.t += dt

    def run(self, T):
        for t in np.arange(0, T, 0.1):
            self.update(0.1)
            print(self.r, self.v, self.t)
```

这个代码实例定义了一个`GravitySimulator`类，用于模拟球体在引力场中的运动。`GravitySimulator`类的构造函数接受两个质量和引力常数作为输入参数，并初始化球体的位置和速度。`update`方法用于更新球体的位置和速度，`run`方法用于运行仿真过程，并输出球体的位置、速度和时间。

通过运行这个代码实例，我们可以观察球体在引力场中的运动，并分析其轨迹和速度变化。这个代码实例的应用可以帮助我们理解引力的作用，并为更高级的物理仿真提供基础。
