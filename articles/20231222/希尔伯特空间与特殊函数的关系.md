希尔伯特空间是一个内积空间，其基本概念和性质如下：

# 4.1 希尔伯特空间的实现
在Python中，可以使用numpy库来实现希尔伯特空间。以下是一个简单的希尔伯特空间实现示例：

```python
import numpy as np

def inner_product(u, v):
    return np.dot(u, v)

def norm(u):
    return np.linalg.norm(u)

def distance(u, v):
    return norm(u - v)

u = np.array([1, 2])
v = np.array([3, 4])

print("内积:", inner_product(u, v))
print("范数:", norm(u))
print("距离:", distance(u, v))
```
