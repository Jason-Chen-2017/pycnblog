# 2.核心概念与联系
## 2.1 分块矩阵
分块矩阵是将原矩阵划分为若干个较小的矩阵块（称为分块）组成的矩阵。这些矩阵块可以是方形矩阵或非方形矩阵，可以是方阵或矩阵。常见的分块矩阵包括：

- 上三角分块矩阵
- 下三角分块矩阵
- 对角线分块矩阵
- 上下三角分块矩阵
- 全矩阵分块矩阵

分块矩阵的主要优势在于，它可以将大矩阵划分为多个较小矩阵，从而使得矩阵计算可以并行进行，提高计算效率。

## 2.2 高性能分块矩阵操作
高性能分块矩阵操作是一种针对分块矩阵进行计算的方法，它通过将分块矩阵操作并行化，提高了计算效率。这种方法主要应用于线性代数计算、线性方程组求解、奇异值分解、奇异值分解等。
