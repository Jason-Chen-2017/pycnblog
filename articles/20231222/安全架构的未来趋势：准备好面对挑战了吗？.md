# 2. 核心概念与联系

安全架构是一种系统性的安全保障措施，它的核心概念包括：

1. 安全性：信息系统在各种安全风险面前具有足够的保护能力。
2. 可靠性：信息系统在故障或攻击后能够继续运行，并能够及时恢复。
3. 可扩展性：安全架构能够适应不断变化的安全需求和环境。

# 4. 具体代码实例和解释

以下是一个简单的密码认证示例：

```python
import hashlib

def hash_password(password):
    return hashlib.sha256(password.encode()).hexdigest()

def verify_password(password, hashed_password):
    return hash_password(password) == hashed_password
```

在这个示例中，我们使用了SHA-256算法对密码进行了加密，并实现了密码验证功能。
