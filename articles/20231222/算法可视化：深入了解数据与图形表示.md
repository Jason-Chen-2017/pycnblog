### 4.1 排序算法的可视化

我们将实现一个简单的冒泡排序算法的可视化。冒泡排序算法的基本思想是通过多次交换相邻的元素，将最小的元素逐步冒泡到数组的前端。

以下是冒泡排序算法的Python实现：

```python
import matplotlib.pyplot as plt
import numpy as np

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

def plot_bubble_sort(arr):
    fig, ax = plt.subplots()
    ax.plot(arr, 'o')
    ax.set_xlabel('Index')
    ax.set_ylabel('Value')
    plt.show()

if __name__ == '__main__':
    arr = np.random.randint(0, 100, size=10)
    print('Original array:', arr)
    sorted_arr = bubble_sort(arr)
    print('Sorted array:', sorted_arr)
    plot_bubble_sort(arr)
```

在上述代码中，我们首先导入了matplotlib.pyplot和numpy库。然后定义了一个`bubble_sort`函数，用于实现冒泡排序算法。接着定义了一个`plot_bubble_sort`函数，用于可视化冒泡排序算法。最后，在`if __name__ == '__main__'`块中，我们生成了一个随机的整数数组，并使用`bubble_sort`函数对其进行排序。最后，调用`plot_bubble_sort`函数对排序前后的数组进行可视化。
