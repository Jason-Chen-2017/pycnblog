首先，我们需要导入 Arrow 库：

```python
import arrow
```

接下来，我们需要创建一个 Arrow 数据表：

```python
data = arrow.Table.from_pydict({
    'age': [1, 2, 3, 4, 5],
    'salary': [1000, 2000, 3000, 4000, 5000]
})
```

接下来，我们需要计算这个数据集中的平均薪资：

```python
average_salary = data.aggregate(arrow.agg.mean('salary'))
```

最后，我们需要打印出计算结果：

```python
print(average_salary)
```

输出结果为：

```

```
