# 2.核心概念与联系

在交通系统中，排队论主要关注车辆在交通信号灯、道路岔口、公共交通站等位置的排队现象。这些现象会导致车辆的流动速度减慢，导致拥堵。排队论可以帮助我们理解这些现象的原因，并提供有效的解决方案。

## 2.1 核心概念

- **队列（Queue）**：队列是排队论的基本概念，是一种先进先出（FIFO，First In First Out）的数据结构。在交通系统中，队列表示车辆在某个位置等待的集合。
- **服务速率（Service Rate）**：服务速率是交通系统中车辆被处理（例如通过信号灯或者进入道路）的速度。
- **平均排队时间（Average Waiting Time）**：平均排队时间是车辆在队列中等待的平均时间。
- **系统吞吐量（System Throughput）**：系统吞吐量是在单位时间内处理的车辆数量。

## 2.2 联系

排队论在交通系统中的应用，可以帮助我们理解交通拥堵的原因，并提供有效的解决方案。通过分析车辆在不同位置的排队现象，我们可以了解交通系统的瓶颈和矛盾，并采取相应的措施进行改进。
