## 4.2 损失函数示例

在本示例中，我们将使用Python和Pytorch来实现均方误差（MSE）损失函数。首先，我们需要导入所需的库：

```python
import torch
```

接下来，我们需要定义真实值和模型预测值：

```python
ground_truth = torch.tensor([[1, 2], [3, 4]])
prediction = torch.tensor([[1.5, 1.8], [3.1, 3.9]])
```

现在，我们可以使用均方误差（MSE）损失函数计算损失值：

```python

    error = ground_truth - prediction
    return torch.mean(error ** 2)

loss = mse_loss(ground_truth, prediction)
print(loss)
```

在这个示例中，我们首先定义了真实值`ground_truth`和模型预测值`prediction`。然后，我们定义了一个`mse_loss`函数，该函数使用均方误差（MSE）损失函数计算损失值。最后，我们调用`mse_loss`函数并打印了损失值`loss`。
