```python
import numpy as np

def calculate_motivation(goal_value, challenge_degree, success_feeling):
    return goal_value * challenge_degree * success_feeling

def calculate_effect(time, effort, ability):
    return time * effort * ability

goal_value = 10
challenge_degree = 5
success_feeling = 8
time = 10

```

接下来，我们设定了一组变量，分别表示目标价值、挑战度、成功感、时间、投入和能力。然后，我们调用了`calculate_motivation`和`calculate_effect`函数，计算出学习动力和学习效果的值。
