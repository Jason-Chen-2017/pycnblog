## 2.1 元素特性

软件测试是一种验证软件系统是否满足需求和规范的过程。它旨在发现和修复缺陷，确保软件系统的质量和可靠性。软件测试可以分为以下几种类型：

1. 单元测试：测试软件系统的单个组件或函数。
2. 集成测试：测试多个单元组件的交互和集成。
3. 系统测试：测试整个软件系统的功能和性能。
4. 验证与验证：测试软件系统是否满足业务需求和规范。

    def test_multiply(self):
        calculator = Calculator()

    def test_divide(self):
        calculator = Calculator()
        self.assertEqual(calculator.divide(6, 3), 2)

    unittest.main()
