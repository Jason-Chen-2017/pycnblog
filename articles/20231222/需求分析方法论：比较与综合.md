```python
def manage_requirements_changes(requirements, changes):
    for change in changes:
        old_requirement = requirements[change.old_id]
        new_requirement = change.new_requirement
        requirements[change.new_id] = new_requirement
        old_requirement.status = 'obsolete'
```
