# 4. 具体代码实例和详细解释说明

以下是一个简单的Python代码实例，用于监控MySQL数据库的性能指标：

```python
import mysql.connector
import time

# 连接MySQL数据库
conn = mysql.connector.connect(
    host='localhost',
    user='root',
    password='password',
    database='test'
)

# 获取数据库性能指标
def get_performance_metrics():
    cursor = conn.cursor()
    cursor.execute('SHOW GLOBAL STATUS')
    rows = cursor.fetchall()
    metrics = {}
    for row in rows:
        metrics[row[0]] = row[1]
    return metrics

# 主程序
if __name__ == '__main__':
    while True:
        metrics = get_performance_metrics()
        print(metrics)
        time.sleep(60)
```

这个代码实例主要包括以下几个步骤：

1. 使用`mysql.connector`库连接到MySQL数据库。

2. 定义一个`get_performance_metrics`函数，用于获取数据库性能指标。这里使用`SHOW GLOBAL STATUS`命令获取全局性能指标。

3. 在主程序中，不断地调用`get_performance_metrics`函数获取性能指标，并打印到控制台。
