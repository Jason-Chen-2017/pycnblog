# 2.核心概念与联系

在本节中，我们将介绍一些核心概念，包括词嵌入、视频处理、帧和序列等。

## 2.1 词嵌入

词嵌入是一种将词语或短语转换为连续的数字表示的技术。这些数字可以用于计算机算法进行处理。词嵌入通常使用一种称为神经网络的机器学习技术来创建。神经网络可以学习从大量文本数据中提取的词汇表示，并将这些表示转换为连续的数字向量。这些向量可以用于计算机算法进行处理，例如文本摘要、情感分析、机器翻译等。

## 2.2 视频处理

视频处理是一种将视频数据转换为其他形式的过程。视频数据通常包括视频帧和音频信号。视频帧是视频流中的单个图像，可以用来表示视频中的动态场景。视频处理可以包括一些常见的操作，例如裁剪、旋转、调整亮度和对比度、添加滤镜等。

## 2.3 帧

帧是视频流中的单个图像，可以用来表示视频中的动态场景。帧通常以连续的顺序呈现，以创建动画或视频效果。帧的速度通常以帧率（frames per second，FPS）表示，表示每秒钟显示的帧数。

## 2.4 序列

序列是一系列相互关联的元素的集合。在视频处理中，序列通常用于表示视频帧之间的关系和依赖关系。序列可以用于表示视频帧之间的时间顺序、空间关系等。
