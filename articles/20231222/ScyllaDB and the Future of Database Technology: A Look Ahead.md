# 4. Code Examples and Detailed Explanations
In this section, we will provide code examples and detailed explanations of how to use ScyllaDB to perform various operations.

## 4.1. Creating a Table
To create a table in ScyllaDB, you can use the following CQL (Cassandra Query Language) command:

```
CREATE TABLE users (
  id UUID PRIMARY KEY,
  name TEXT,
  age INT,
  email TEXT
);
```

This command creates a table called `users` with four columns: `id`, `name`, `age`, and `email`. The `id` column is the primary key, which means that it will be used to index and store the data.

## 4.2. Inserting Data
To insert data into the `users` table, you can use the following CQL command:

```
INSERT INTO users (id, name, age, email) VALUES (uuid(), 'John Doe', 30, 'john.doe@example.com');
```

This command inserts a new row into the `users` table with the specified values for the `id`, `name`, `age`, and `email` columns.

## 4.3. Querying Data
To query data from the `users` table, you can use the following CQL command:

```
SELECT * FROM users WHERE name = 'John Doe';
```

This command selects all columns from the `users` table where the `name` column is equal to `'John Doe'`.

## 4.4. Updating Data
To update data in the `users` table, you can use the following CQL command:

```
UPDATE users SET age = 31 WHERE id = uuid();
```

This command updates the `age` column to `31` for the row with the specified `id`.

## 4.5. Deleting Data
To delete data from the `users` table, you can use the following CQL command:

```
DELETE FROM users WHERE id = uuid();
```

This command deletes the row with the specified `id` from the `users` table.
