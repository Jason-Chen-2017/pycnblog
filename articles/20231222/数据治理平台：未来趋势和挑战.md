1. **数据清洗**：

假设我们有一个包含名字、年龄和性别的数据集，我们需要对这个数据集进行清洗。首先，我们需要检查数据中是否有缺失值，如果有，我们需要进行缺失值的处理。然后，我们需要检查数据中的性别是否只包含两种可能的值（男、女），如果不是，我们需要进行性别类型转换。最后，我们需要检查数据中的年龄是否是整数，如果不是，我们需要进行年龄格式转换。

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 检查缺失值
missing_values = data.isnull().sum()

# 处理缺失值
data = data.fillna(data.mean())

# 检查性别类型
gender_values = data['gender'].unique()
if len(gender_values) != 2:
    data['gender'] = data['gender'].map({'男': 'male', '女': 'female'})

# 检查年龄格式
age_values = data['age'].unique()
if not all(isinstance(age, int) for age in age_values):
    data['age'] = data['age'].astype(int)
```

2. **数据安全加密**：

假设我们需要对一个文本文件进行加密，我们可以使用AES加密算法来实现。首先，我们需要生成一个密钥，然后我们需要对文本文件进行加密，最后我们需要将加密后的文件保存到磁盘上。

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# 生成密钥
key = get_random_bytes(16)

# 加密文件
with open('data.txt', 'rb') as f:
    data = f.read()
cipher = AES.new(key, AES.MODE_EAX)
ciphertext, tag = cipher.encrypt_and_digest(data)

# 保存加密文件
with open('data.enc', 'wb') as f:
    f.write(cipher.nonce)
    f.write(tag)
    f.write(ciphertext)
```

```python
import re

# 读取文件
with open('data.txt', 'r') as f:
    data = f.read()

# 脱敏姓名
name_pattern = re.compile(r'\b(.*?)\b')
names = name_pattern.findall(data)
data = name_pattern.sub('[REDACTED]', data)

# 脱敏身份证号码
id_card_pattern = re.compile(r'\b\d{6}(?=\d{4})\d{2}\d{2}(?=\d{4})\d{2}\d{2}\b')
id_cards = id_card_pattern.findall(data)
data = id_card_pattern.sub('[REDACTED]', data)
```
