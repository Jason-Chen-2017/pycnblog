## 2.1 泊松分布

泊松分布是一种概率分布，用于描述一种独立且等概率发生的随机事件发生的次数。泊松分布的概率密度函数为：

$$
P(X=k)=\frac{\lambda^k e^{-\lambda}}{k!}
$$

其中，$k$ 是随机变量的取值，$\lambda$ 是参数，$e$ 是基数。

泊松分布的主要特点是：

1. 随机变量$X$的取值为非负整数；
2. 随机变量$X$的期望为$\lambda$，方差为$\lambda$；
3. 随机变量$X$的概率密度函数为上式所示。

```python
import matplotlib.pyplot as plt

# 数据预处理
data = np.random.poisson(100, 1000)

# 泊松分布模型构建
lambda_hat = np.mean(data)
poisson_dist = np.random.poisson(lambda_hat, 1000)

# 预测
predicted_data = np.random.poisson(lambda_hat, 1000)

# 分析
accuracy = np.sum(predicted_data == data) / len(data)
print("预测准确性：", accuracy)

# 可视化
plt.hist(data, bins=30, density=True, alpha=0.5, label="原始数据")
plt.hist(poisson_dist, bins=30, density=True, alpha=0.5, label="泊松分布模型")
plt.hist(predicted_data, bins=30, density=True, alpha=0.5, label="预测结果")
plt.legend()
plt.show()
```
