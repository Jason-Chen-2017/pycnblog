代码实例：

```python

import matplotlib.pyplot as plt

# 定义生长率函数
def growth_rate(t):
    return np.exp(-t)

# 计算生长率的导数
derivative = np.gradient(growth_rate(np.linspace(-10, 10, 100)))

# 绘制生长率和其导数
plt.plot(np.linspace(-10, 10, 100), growth_rate(np.linspace(-10, 10, 100)), label='Growth Rate')
plt.plot(np.linspace(-10, 10, 100), derivative, label='Derivative')
plt.legend()
plt.show()
```

这个代码实例中，我们首先定义了生长率函数`growth_rate(t)`。然后，我们使用`np.gradient()`函数计算生长率的导数。最后，我们使用`matplotlib.pyplot`库绘制生长率和其导数的曲线图。
