# 2.核心概念与联系
# 2.1.缓存概述
缓存是一种数据存储技术，用于提高应用程序性能。缓存通常存储在内存中，因为内存访问比磁盘访问快得多。缓存可以减少数据库查询，从而减少磁盘 I/O 操作。缓存还可以减少网络延迟，因为它可以将数据存储在应用程序之间的距离近处。

缓存有几种类型，例如：

- 内存缓存：内存缓存存储在应用程序内存中。内存缓存访问快，但容量有限。
- 磁盘缓存：磁盘缓存存储在磁盘上。磁盘缓存容量大，但访问慢。
- 分布式缓存：分布式缓存存储在多个服务器上。分布式缓存可以提高可用性和性能，但复杂度高。

缓存有几种实现方式，例如：

- 基于键的缓存：基于键的缓存使用键（key）存储数据。键是唯一标识数据的字符串。
- 基于对象的缓存：基于对象的缓存使用对象存储数据。对象是数据的实例。

缓存有几种缓存策略，例如：

- 最近最少使用（LRU）策略：LRU 策略将最近最少使用的数据替换为最近使用的数据。LRU 策略可以提高缓存命中率。
- 最近最久使用（LFU）策略：LFU 策略将最近最久使用的数据替换为最近使用的数据。LFU 策略可以提高缓存命中率。
