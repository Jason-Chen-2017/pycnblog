# 4.具体代码实例和详细解释说明
在这里，我们将给出一个简单的Python代码实例，用于计算排队系统中的平均排队时间和平均排队长度。

```python
import numpy as np

def queue_time(arrival_rate, service_rate, service_time):
    num_customers = 0
    queue_length = 0
    queue_time = 0

    while num_customers < 1000:
        if np.random.rand() < arrival_rate:
            num_customers += 1
            queue_length += 1
        if num_customers > 0 and np.random.rand() < service_rate:
            num_customers -= 1
            queue_length -= 1
            queue_time += service_time

    return queue_time / num_customers

arrival_rate = 0.1
service_rate = 0.2
service_time = 1

average_queue_time = queue_time(arrival_rate, service_rate, service_time)
print("Average queue time: ", average_queue_time)
```

这个代码实例中，我们首先导入了numpy库，然后定义了一个函数`queue_time`，用于计算排队系统中的平均排队时间和平均排队长度。在函数中，我们使用了随机数生成器来模拟人或物的到达和服务。然后，我们使用了一个while循环来模拟1000个人或物的到达和服务。最后，我们使用了平均排队时间和平均排队长度的公式来计算平均排队时间。
