### 3.1.1 物料传输算法

物料传输算法是用于实现物料在流水线工程中的自动传输的算法。常见的物料传输算法有以下几种：

```python
import threading

class MaterialTransport:
    def __init__(self, material):
        self.material = material
        self.transport_lock = threading.Lock()

    def transport(self):
        with self.transport_lock:
            print(f"Material {self.material} is transporting.")

material_transport = MaterialTransport("clay")
material_transport.transport()
```
