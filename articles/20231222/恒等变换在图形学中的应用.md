$$
\begin{bmatrix} x' \\ y' \\ 1 \end{bmatrix} = \begin{bmatrix} 1 & 0 & t_x \\ 0 & 1 & t_y \\ 0 & 0 & 1 \end{bmatrix} \begin{bmatrix} x \\ y \\ 1 \end{bmatrix}

$$

### 3.3 缩放变换

## 4. 具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来说明恒等变换的实现。我们将使用 OpenGL 库来实现一个简单的旋转变换。

```cpp
#include <GL/glut.h>

void display() {
    glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);
    glLoadIdentity();
    glTranslatef(0.0f, 0.0f, -5.0f);
    glRotatef(360.0f, 1.0f, 1.0f, 1.0f);
    glBegin(GL_QUADS);
    glColor3f(1.0f, 0.0f, 0.0f);
    glVertex3f(-0.5f, -0.5f, 0.0f);
    glVertex3f(0.5f, -0.5f, 0.0f);
    glVertex3f(0.5f, 0.5f, 0.0f);
    glVertex3f(-0.5f, 0.5f, 0.0f);
    glEnd();
    glFlush();
}

int main(int argc, char** argv) {
    glutInit(&argc, argv);
    glutCreateWindow("Rotate Transform");
    glutDisplayFunc(display);
    glutMainLoop();
    return 0;
}
```

在这个代码实例中，我们使用了 `glLoadIdentity` 函数来加载单位矩阵，这个矩阵表示恒等变换。然后，我们使用了 `glTranslatef` 函数来实现平移变换，接着使用了 `glRotatef` 函数来实现旋转变换。最后，我们使用了 `glBegin` 和 `glEnd` 函数来绘制一个正方形。
