# 2.核心概念与联系
自动化测试工具的核心概念包括：

1. **测试框架**：测试框架是自动化测试的基础，它提供了一种结构化的方法来组织和执行测试用例。

2. **测试库**：测试库是一组预先编写的测试用例，可以直接使用或者作为参考。

3. **测试工具**：测试工具是一种软件，可以帮助开发人员自动化地执行测试用例，并记录测试结果。

4. **测试平台**：测试平台是一种环境，可以用来运行和管理自动化测试工具和测试用例。

5. **测试报告**：测试报告是自动化测试的结果，可以帮助开发人员了解测试的结果和问题。
