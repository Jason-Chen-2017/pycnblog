# 2.核心概念与联系
电磁波谱是指电磁波在不同频率和波长上的分布。电磁波是光、电场和磁场的结合体，它们可以在空间中传播。电磁波谱可以分为许多区域，如微波、可见光、红外等。这些区域分别对应不同的通信技术和应用。

在电信通信中，电磁波谱的主要应用有以下几个方面：

1. 无线通信：无线通信技术是指不需要物理媒介的通信技术，而是通过电磁波在空气中传输信息。无线通信技术的主要优势是灵活性和便利性，因此在现代社会中得到了广泛应用。

2. 有线通信：有线通信技术是指需要物理媒介的通信技术，如电话、电报、电线电报等。有线通信技术的主要优势是稳定性和信号质量。

3. 广播电视：广播电视技术是一种通过电磁波在空气中传播音频和视频信号的技术。广播电视技术的主要优势是覆盖范围和实时性。

4. 卫星通信：卫星通信技术是一种通过卫星在空中传播电磁波的通信技术。卫星通信技术的主要优势是覆盖范围和信道资源。
