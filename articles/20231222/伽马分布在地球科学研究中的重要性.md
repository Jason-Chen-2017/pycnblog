## 2.1 伽马分布简介

伽马分布（Gamma distribution）是一种连续概率分布，用于描述实值随机变量的分布。它的名字来源于古希腊神话中的伽马神。伽马分布在许多科学领域都有应用，包括地球科学、生物统计学、金融市场等。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 伽马分布的概率密度函数

伽马分布的概率密度函数（PDF）为：

$$
f(x; \alpha, \beta) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}
$$

其中，$\alpha$ 和 $\beta$ 是分布的参数，$\Gamma(\alpha)$ 是$\alpha$阶的伽马函数。

## 3.2 伽马分布的累积分布函数

伽马分布的累积分布函数（CDF）为：

$$
F(x; \alpha, \beta) = \frac{1}{\Gamma(\alpha)} \int_0^x t^{\alpha-1} e^{-\beta t} dt
$$

## 3.3 伽马分布的期望和方差

伽马分布的期望为：

$$
E[X] = \frac{\alpha}{\beta}
$$

伽马分布的方差为：

$$
Var[X] = \frac{\alpha}{\beta^2}
$$

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的例子来演示如何使用Python编程语言计算伽马分布的概率密度函数和累积分布函数。

```python
import numpy as np
from scipy.stats import gamma

# 设置参数
alpha = 2
beta = 3
x = np.linspace(0, 10, 100)

# 计算概率密度函数
pdf = gamma.pdf(x, alpha, beta)

# 计算累积分布函数
cdf = gamma.cdf(x, alpha, beta)

# 绘制PDF和CDF
import matplotlib.pyplot as plt

plt.figure(figsize=(12, 6))
plt.subplot(1, 2, 1)
plt.plot(x, pdf, label='PDF')
plt.xlabel('x')
plt.ylabel('f(x)')
plt.legend()

plt.subplot(1, 2, 2)
plt.plot(x, cdf, label='CDF')
plt.xlabel('x')
plt.ylabel('F(x)')
plt.legend()

plt.show()
```
