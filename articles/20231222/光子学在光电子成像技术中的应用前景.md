# 2. 核心概念与联系

在光电子成像技术中，光子学的核心概念主要包括：

2. 光子的相互作用：光子可以与物质粒子（如电子）相互作用，这种作用可以分为吸收、散射、反射等。在光电子成像技术中，光子与物质粒子之间的相互作用是成像过程中的关键步骤。
