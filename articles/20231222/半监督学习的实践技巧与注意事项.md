# 2. 核心概念与联系

1. 有监督学习：有监督学习是指在训练过程中使用标签数据进行模型训练的学习方法。常见的有监督学习算法包括线性回归、逻辑回归、支持向量机等。

2. 无监督学习：无监督学习是指在训练过程中不使用标签数据进行模型训练的学习方法。常见的无监督学习算法包括聚类、主成分分析、自组织特征分析等。

3. 半监督学习：半监督学习是指在训练过程中使用有限标签数据和无标签数据进行模型训练的学习方法。半监督学习可以将无监督学习和有监督学习的优点相结合，提高模型的准确性和泛化能力。

半监督学习的联系包括：

1. 半监督学习可以利用有限的标签数据进行模型训练，从而提高模型的准确性。
