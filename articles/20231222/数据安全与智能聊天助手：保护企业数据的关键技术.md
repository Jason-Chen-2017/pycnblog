## 3.1 RSA算法

RSA算法是一种公钥加密算法，可以用于实现数据安全。RSA算法的核心思想是使用两个不同的密钥进行加密和解密：公钥用于加密数据，私钥用于解密数据。

RSA算法的具体操作步骤如下：

1. 生成两个大素数p和q，然后计算n=p*q。
2. 计算φ(n)=(p-1)*(q-1)。
3. 选择一个随机整数e，使得1<e<φ(n)，并且gcd(e,φ(n))=1。
4. 计算d=mod^{-1}(e^{-1}modφ(n))，其中mod^{-1}表示模反转。
5. 公钥为(n,e)，私钥为(n,d)。

## 3.2 AES算法

AES算法是一种对称加密算法，可以用于实现数据安全。AES算法的核心思想是使用一个密钥进行加密和解密。

AES算法的具体操作步骤如下：

1. 扩展密钥：将输入密钥扩展为128位（对应于128位AES）或256位（对应于256位AES）。
2. 加密：对于每个数据块，执行以下操作：
   - 加密：使用扩展密钥和数据块进行加密。
   - 混淆：对加密后的数据块进行混淆。
   - 扩展：对混淆后的数据块进行扩展。
   - 选择：从扩展中选择一部分位作为输出。
3. 解密：对于每个数据块，执行以下操作：
   - 选择：从输入中选择一部分位。
   - 扩展：对选定位进行扩展。
   - 混淆：对扩展后的位进行混淆。
   - 解密：使用扩展密钥和混淆后的位进行解密。

def is_prime(n):
    if n <= 1:

    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

def mod_inverse(a, m):
    for x in range(1, m):
        if (a * x) % m == 1:
            return x
    return None

def rsa_key_generation(p, q):
    n = p * q
    phi_n = (p - 1) * (q - 1)
    e = random.randint(1, phi_n - 1)
    g = gcd(e, phi_n)
    d = mod_inverse(e, phi_n)
    return (n, e, d)

def rsa_encrypt(m, e, n):
    return (m ** e) % n

def rsa_decrypt(c, d, n):
