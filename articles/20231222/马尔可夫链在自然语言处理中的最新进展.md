# 4.具体代码实例和详细解释说明

在这里，我们将提供一个简单的Python代码实例，展示如何使用有限马尔可夫链来构建一个简单的语言模型。

```python
import numpy as np

# 词汇表
vocab = ['I', 'love', 'natural', 'language', 'processing']

# 转移矩阵
transition_matrix = np.zeros((len(vocab), len(vocab)))

# 计算转移概率
for i in range(len(vocab)):
    for j in range(len(vocab)):
        transition_matrix[i][j] = (vocab.count(vocab[j]) - i) / float(vocab.count(vocab[i]))

# 生成文本
start_word = 'I'
current_word = start_word
while current_word != '':
    next_word = np.random.choice(vocab, p=transition_matrix[vocab.index(current_word)])
    print(current_word)
    current_word = next_word
```

这个简单的代码实例首先定义了一个词汇表`vocab`，然后创建了一个转移矩阵`transition_matrix`。接下来，我们计算了转移概率，并使用了一个随机选择的过程来生成文本。
