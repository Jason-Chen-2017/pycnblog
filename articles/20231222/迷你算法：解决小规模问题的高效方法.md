# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的迷你算法实例为例，来展示迷你算法的具体实现和解释。

## 4.1 问题描述
给定一个小规模的整数数组，找出其中的最大值和最小值。

## 4.2 算法实现
```python
def find_max_min(arr):
    if not arr:
        return None, None
    max_val = arr[0]
    min_val = arr[0]
    for num in arr[1:]:
        if num > max_val:
            max_val = num
        elif num < min_val:
            min_val = num
    return max_val, min_val
```

## 4.3 解释说明
这个算法的核心思路是通过遍历数组中的每个元素，来找到最大值和最小值。首先，我们检查输入数组是否为空，如果为空，则返回None，表示无法找到最大值和最小值。接着，我们将数组的第一个元素赋值给max_val和min_val。然后，我们遍历数组中的其他元素，如果当前元素大于max_val，则更新max_val的值；如果当前元素小于min_val，则更新min_val的值。最后，我们返回最大值和最小值。
