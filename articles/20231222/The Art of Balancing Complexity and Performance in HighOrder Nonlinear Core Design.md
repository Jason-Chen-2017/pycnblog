## 4.1 代码实例

```python
import numpy as np

def matrix_multiply(A, B):
    n = A.shape[0]
    C = np.zeros((n, n))
    for i in range(n):
        for j in range(n):
            for k in range(n):
                C[i, j] += A[i, k] * B[k, j]
    return C
```

在这个代码实例中，我们首先导入了NumPy库，然后定义了一个名为`matrix_multiply`的函数，该函数接受两个矩阵A和B作为输入，并返回它们的乘积C。
