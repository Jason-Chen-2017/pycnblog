# 2.核心概念与联系

## 2.1条件熵定义

条件熵是信息论中的一个概念，它可以用来度量一个随机变量给定某个条件下的不确定性。条件熵的定义如下：

$$
H(Y|X) = -\sum_{x\in X} P(x|Y) \log P(x|Y)
$$

其中，$H(Y|X)$ 表示随机变量 $Y$ 给定随机变量 $X$ 的条件熵，$P(x|Y)$ 表示随机变量 $X$ 给定随机变量 $Y$ 的概率分布。

from sklearn.metrics.pairwise import cosine_similarity

def compute_condition_entropy(user_features, item_features, user_item_ratings):
    # 计算用户与项目之间的相似度
    similarity = cosine_similarity(user_features, item_features)
    
    # 计算用户对项目的评价
    ratings = user_item_ratings
    
    # 计算用户对项目的条件熵
    condition_entropy = -np.sum(ratings * np.log(ratings + np.finfo(float).eps))
    
    return condition_entropy

# 数据预处理
user_features = np.array([[1, 0, 0], [0, 1, 0], [0, 0, 1]])
item_features = np.array([[1, 0], [0, 1], [1, 0]])
user_item_ratings = np.array([[3, 4, 5], [2, 3, 4], [1, 2, 3]])

# 计算条件熵
condition_entropy = compute_condition_entropy(user_features, item_features, user_item_ratings)
print("条件熵:", condition_entropy)
