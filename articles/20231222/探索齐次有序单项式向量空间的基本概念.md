## 3.1 齐次有序单项式向量空间的定义

定义1（齐次有序单项式向量空间）：一个向量空间V是一个齐次有序单项式向量空间，如果它的基元都是齐次有序单项式。

其中，齐次有序单项式是指形如x^a*y^b*z^c...的表达式，其中x、y、z...是变量，a、b、c...是非负整数，并且对于任意两个齐次有序单项式p和q，有p≺q if deg(p) < deg(q)或者deg(p) = deg(q)且对于所有非零项i，p_i≺q_i，其中deg(p)表示p的最高项指数，p_i和q_i分别是p和q的度量为i的项。

## 3.2 齐次有序单项式向量空间的基本性质

定理1（齐次有序单项式向量空间的线性无关性）：如果向量v = Σc_ip_i是一个齐次有序单项式向量，其中c_i是系数，p_i是齐次有序单项式，并且对于所有非零项i，c_i≠0，那么v是线性无关的。

证明：假设v是线性相关的，那么存在一个非零整数k，使得Σc_ip_i^k = 0。由于p_i^k是齐次有序单项式，因此存在一个度量为k的非零项q_k，满足q_k≺p_i^k。然而，由于c_i≠0，我们有c_iq_k≠0，这与Σc_ip_i^k = 0矛盾。因此，v是线性无关的。

定理2（齐次有序单项式向量空间的维数）：如果向量空间V是一个齐次有序单项式向量空间，并且V的基元是{p_1, p_2, ..., p_n}，那么V的维数dim(V) = n。

证明：由于每个基元p_i都是齐次有序单项式，因此它们是线性无关的（根据定理1）。因此，我们可以将V表示为一个直和的基元p_i的线性组合。由于p_i是齐次有序单项式，它们的度量不同，因此它们是线性独立的。因此，dim(V) = n。

输出：一个齐次有序单项式向量空间V。

## 4.1 代码实例

```python
from sympy import symbols, powers

# 定义变量
x, y, z = symbols('x y z')

# 生成齐次有序单项式向量空间V
V = [powers(x, i) + powers(y, j) + powers(z, k) for i in range(1, 4) for j in range(1, 3) for k in range(1, 2)]

# 计算基础集合B
B = []
for v in V:
    if v not in B:
        B.append(v)

# 计算维数dim(V)
dim_V = len(B)

# 打印结果
print('基础集合B:', B)
print('维数dim(V):', dim_V)
```

在这个代码实例中，我们首先定义了三个变量x、y和z。然后，我们使用了一个三层嵌套的for循环来生成一个齐次有序单项式向量空间V。在这个向量空间中，每个元素都是一个由x、y和z变量组成的齐次有序单项式。接下来，我们计算了基础集合B，并将其添加到B中。最后，我们计算了维数dim(V)，并将其打印出来。
