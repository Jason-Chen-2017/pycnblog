- **强大的关系表示**：图数据库可以很好地表示复杂的关系，这使得它非常适合用于社交网络、知识图谱等应用。

## 4.具体代码实例和详细解释说明

### 4.1 RethinkDB代码实例

以下是一个使用RethinkDB存储和查询实时数据的代码实例：

```python
from rethinkdb import RethinkDB

# 连接RethinkDB
r = RethinkDB()

# 创建数据库
db = r.db('mydb')

# 插入数据
db.table('mytable').insert({'name': 'John', 'age': 30}).execute()

# 查询数据
result = db.table('mytable').filter(lambda row: row('age') > 25).run()

# 打印结果
print(result.pluck('name').run())
```

在这个代码实例中，我们首先连接到RethinkDB，然后创建一个名为`mydb`的数据库。接着，我们使用`table`方法插入一条数据，并使用`filter`方法查询年龄大于25的数据。最后，我们使用`pluck`方法打印结果。

### 4.2 图数据库代码实例

以下是一个使用Neo4j图数据库存储和查询数据的代码实例：

```python
from neo4j import GraphDatabase

# 连接图数据库
driver = GraphDatabase.driver('bolt://localhost:7687', auth=('neo4j', 'password'))

# 创建会话
session = driver.session()

# 插入数据
session.run("CREATE (a:Person {name: $name, age: $age})", name='John', age=30)

# 查询数据
result = session.run("MATCH (a:Person) WHERE a.age > 25 RETURN a.name", name='John', age=30)

# 打印结果
for record in result:
    print(record['name'])
```

在这个代码实例中，我们首先连接到Neo4j图数据库，然后创建一个会话。接着，我们使用`CREATE`语句插入一条数据，并使用`MATCH`和`WHERE`语句查询年龄大于25的数据。最后，我们使用`RETURN`语句打印结果。
