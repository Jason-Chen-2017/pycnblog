```sql

```

接下来，我们可以使用以下SQL命令来创建一个表：

```sql

  id INT PRIMARY KEY,
  name VARCHAR(255),
  email VARCHAR(255)
);
```

现在，我们可以使用以下SQL命令来插入一条记录：

```sql
INSERT INTO mydb.users (id, name, email) VALUES (1, 'John Doe', 'john@example.com');
```

最后，我们可以使用以下SQL命令来查询记录：

```sql
SELECT * FROM mydb.users;
```
