# 4. 具体代码实例和详细解释说明
在这里，我们以一个简单的知识抽取示例来详细解释代码实例和解释说明：

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.tag import pos_tag

# 文本数据
text = "John is a student at the University of California, Berkeley."

# 文本预处理
tokens = word_tokenize(text)

# 实体识别
tagged = pos_tag(tokens)

# 关系抽取
entities = ["John", "student", "University of California, Berkeley"]
relations = ["is a"]

# 知识存储
knowledge_base = {"John": "student", "University of California, Berkeley": "University"}

# 知识抽取
for entity, relation in zip(entities, relations):
    for token, tag in tagged:
        if token == entity:
            knowledge_base[entity] = relation
```

在这个示例中，我们首先导入了nltk库，并使用word_tokenize函数将文本数据分词。接着，使用pos_tag函数对分词后的文本进行词性标注。然后，我们定义了实体和关系列表，并将它们存储到知识库中。最后，通过遍历实体和关系列表，我们将关系存储到知识库中。
