以下是一个简单的K均值聚类算法实现的例子：

```python
from sklearn.cluster import KMeans
import numpy as np

# 假设我们有一个用户数据集，包括年龄、性别、购买记录等特征
data = np.array([[25, '男', 1], [30, '女', 0], [28, '男', 1], [32, '女', 0], [26, '男', 0], [31, '女', 1]])

# 使用K均值算法对数据集进行聚类

# 获取聚类结果
labels = kmeans.labels_

# 打印聚类结果
print(labels)
```
