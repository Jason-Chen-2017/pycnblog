## 4.2 信号处理算法实例

我们将通过一个简单的信号滤波示例来解释信号处理算法原理。在这个示例中，我们将使用Python编程语言来实现信号滤波算法。

```python
import numpy as np
import matplotlib.pyplot as plt

def filter_signal(signal):
    # 对信号进行滤波
    filtered_signal = np.fft.ifft(np.fft.fft(signal) / np.abs(np.fft.fft(signal)))
    return filtered_signal

signal = np.random.rand(100, 100)
filtered_signal = filter_signal(signal)

plt.imshow(filtered_signal, cmap='gray')
plt.show()
```

在这个示例中，我们首先导入了Python的NumPy和Matplotlib库。然后，我们定义了一个名为`filter_signal`的函数，该函数用于对信号进行滤波。在这个函数中，我们使用了Python的NumPy库中的`np.fft.ifft`和`np.fft.fft`函数来实现信号滤波。最后，我们使用Matplotlib库来绘制滤波后的信号。
