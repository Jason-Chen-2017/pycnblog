```python
import threading
import time

class TrafficLight:
    def __init__(self):
        self.state = "red"
        self.lock = threading.Lock()

    def change_state(self):
        with self.lock:
            if self.state == "red":
                self.state = "green"
            elif self.state == "green":
                self.state = "yellow"
            elif self.state == "yellow":
                self.state = "red"

def traffic_light_thread():
    traffic_light = TrafficLight()
    while True:
        traffic_light.change_state()
        time.sleep(1)

if __name__ == "__main__":
    threading.Thread(target=traffic_light_thread).start()
```

在上述代码中，我们定义了一个`TrafficLight`类，该类包含一个`state`属性和一个`lock`属性。`state`属性用于存储交通信号灯的当前状态，可以取值为`red`、`green`或`yellow`。`lock`属性用于保护`state`属性的并发访问。

`change_state`方法用于更新交通信号灯的状态。该方法使用`with`语句来获取`lock`属性的锁定，确保在更新状态时不会出现并发问题。

`traffic_light_thread`函数用于创建一个交通信号灯的线程，该线程不断更新交通信号灯的状态。

`if __name__ == "__main__":`语句用于确保`traffic_light_thread`函数只在主程序中执行。

通过上述代码实例，我们可以看到独立化处理技术在交通运输领域的应用，可以通过将控制权转移给分散的节点，提高系统的灵活性和可靠性。
