## 4.1 生物引擎

生物引擎的一个简单实现如下：

```python
import numpy as np

def activate_gene(gene, activator):
    return gene + activator

def repress_gene(gene, repressor):
    return gene - repressor

gene = np.array([1, 2, 3])
activator = np.array([1, 2, 3])
repressor = np.array([1, 2, 3])

activated_gene = activate_gene(gene, activator)
repressed_gene = repress_gene(gene, repressor)

print("Activated gene:", activated_gene)
print("Repressed gene:", repressed_gene)
```

在这个例子中，我们首先导入了numpy库，然后定义了两个函数：`activate_gene`和`repress_gene`。这两个函数分别用于激活和抑制基因。接下来，我们定义了一个基因数组`gene`，以及激活和抑制序列`activator`和`repressor`。最后，我们调用了这两个函数，并打印了激活和抑制后的基因数组。
