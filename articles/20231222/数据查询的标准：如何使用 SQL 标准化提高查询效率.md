# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明如何使用 SQL 标准化来提高查询效率。

假设我们有一个销售订单表，表名为 `sales_orders`，其中包含以下列：

1. `order_id`：订单 ID，主键
2. `customer_id`：客户 ID
3. `order_date`：订单日期
4. `total_amount`：订单总金额

我们需要查询出每个客户的总订单金额。我们可以使用以下 SQL 查询语句：

```sql
SELECT customer_id, SUM(total_amount) AS total_amount
FROM sales_orders
GROUP BY customer_id;
```

这个查询语句使用了 `GROUP BY` 子句来分组数据，从而避免了使用多个子查询。此外，我们使用了 `SUM` 函数来计算每个客户的总订单金额。
