## 3.1 数组与链表

数组是一种连续的内存分配方式，其中元素的访问速度非常快。链表则是一种连续的内存分配方式，其中元素的访问速度相对较慢。

### 3.1.1 数组

数组是一种固定大小的数据结构，其中元素的访问速度非常快。数组的元素可以是任意类型的数据，例如整数、字符串等。数组的声明和初始化如下：

```python
# 声明一个整数型数组
arr = [1, 2, 3, 4, 5]

# 声明一个字符串型数组
str_arr = ["hello", "world", "!"]
```

数组的访问速度非常快，因为它们的元素是连续存储在内存中的。例如，访问数组中的第一个元素只需要读取内存中的一个地址，即可获取到该元素的值。

### 3.1.2 链表

链表是一种动态大小的数据结构，其中元素的访问速度相对较慢。链表的元素是以链接的方式存储在内存中的，每个元素都包含一个指向下一个元素的指针。链表的声明和初始化如下：

```python
# 声明一个整数型链表
class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

# 初始化一个链表
head = Node(1)
head.next = Node(2)
head.next.next = Node(3)
```

链表的访问速度相对较慢，因为它们的元素是以链接的方式存储在内存中的，需要逐个访问以获取到某个元素的值。

## 3.2 排序算法

排序算法是一种常用的数据处理方法，用于将一组数据按照某个特定的顺序进行排序。以下是一些常见的排序算法：

### 3.2.1 冒泡排序

冒泡排序是一种简单的排序算法，其基本思想是通过多次比较相邻的元素，将较大的元素向后移动，以达到排序的目的。冒泡排序的时间复杂度为O(n^2)，其中n是数据的个数。

### 3.2.2 快速排序

快速排序是一种高效的排序算法，其基本思想是通过选择一个基准元素，将较小的元素放在基准元素的左侧，较大的元素放在基准元素的右侧，然后递归地对左侧和右侧的子数组进行排序。快速排序的时间复杂度为O(nlogn)，其中n是数据的个数。

### 3.2.3 归并排序

归并排序是一种高效的排序算法，其基本思想是将一个大的排序问题分解为多个小的排序问题，然后将这些小的排序问题合并为一个大的排序问题。归并排序的时间复杂度为O(nlogn)，其中n是数据的个数。

## 3.3 搜索算法

搜索算法是一种常用的数据处理方法，用于在一组数据中查找满足某个条件的元素。以下是一些常见的搜索算法：

### 3.3.1 二分搜索

二分搜索是一种高效的搜索算法，其基本思想是将一个大的搜索问题分解为多个小的搜索问题，然后将这些小的搜索问题合并为一个大的搜索问题。二分搜索的时间复杂度为O(logn)，其中n是数据的个数。

### 3.3.2 深度优先搜索

深度优先搜索是一种搜索算法，其基本思想是从搜索树的根节点开始，逐层向下搜索，直到搜索树中的叶节点为止。深度优先搜索的时间复杂度为O(n)，其中n是数据的个数。

### 3.3.3 广度优先搜索

广度优先搜索是一种搜索算法，其基本思想是从搜索树的根节点开始，逐层向下搜索，直到搜索树中的叶节点为止。广度优先搜索的时间复杂度为O(n)，其中n是数据的个数。

## 4.1 冒泡排序

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

在上述代码中，我们首先定义了一个名为`bubble_sort`的函数，其参数为一个需要排序的数组`arr`。然后，我们通过两层for循环来实现冒泡排序的算法。在内层for循环中，我们将当前元素与下一个元素进行比较，如果当前元素大于下一个元素，则交换它们的位置。这样，每次循环都会将一个最大的元素移动到数组的末尾，直到整个数组被排序为止。

## 4.2 快速排序

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)
```

在上述代码中，我们首先定义了一个名为`quick_sort`的函数，其参数为一个需要排序的数组`arr`。然后，我们通过递归来实现快速排序的算法。首先，我们检查数组的长度是否小于等于1，如果是，则直接返回数组。然后，我们选择数组的第一个元素作为基准元素`pivot`。接着，我们将数组中的其他元素分为两个子数组：一个是小于基准元素的子数组`left`，另一个是大于或等于基准元素的子数组`right`。最后，我们通过递归地对`left`和`right`子数组进行快速排序，并将它们与基准元素`pivot`组合在一起，得到排序后的数组。

## 4.3 归并排序

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]
    return merge(merge_sort(left), merge_sort(right))

def merge(left, right):
    result = []
    while left and right:
        if left[0] < right[0]:
            result.append(left.pop(0))
        else:
            result.append(right.pop(0))
    result.extend(left)
    result.extend(right)
    return result
```

在上述代码中，我们首先定义了一个名为`merge_sort`的函数，其参数为一个需要排序的数组`arr`。然后，我们通过递归来实现归并排序的算法。首先，我们检查数组的长度是否小于等于1，如果是，则直接返回数组。然后，我们将数组分为两个子数组：一个是数组的左半部分`left`，另一个是数组的右半部分`right`。接着，我们通过递归地对`left`和`right`子数组进行归并排序，并将它们合并在一起，得到排序后的数组。最后，我们通过递归地对`left`和`right`子数组进行归并排序，并将它们合并在一起，得到排序后的数组。
