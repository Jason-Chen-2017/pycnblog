# 4. 具体代码实例和详细解释说明

在这里，我们将给出一个具体的物流路径规划算法的代码实例，并进行详细解释说明。

```python
import numpy as np

def dijkstra(graph, start):
    n = len(graph)
    dist = [np.inf] * n
    prev = [None] * n
    dist[start] = 0
    unvisited = set(range(n))
    while unvisited:
        u = min(unvisited, key=lambda i: dist[i] if i in unvisited else np.inf)
        unvisited.remove(u)
        for v, weight in graph[u].items():
            old_dist = dist[v]
            dist[v] = min(dist[v], dist[u] + weight)
            if old_dist > dist[v]:
                prev[v] = u
    return dist, prev
```

这段代码实现了迪杰斯特拉算法，用于求解物流路径规划问题。首先，我们导入了numpy库，用于数值计算。然后，我们定义了一个`dijkstra`函数，接受一个图和一个起点作为输入，返回距离和前驱表。在函数内部，我们初始化距离和前驱表，将起点的距离设为0，其他距离设为无穷大。然后，我们创建一个未访问集合，包含所有顶点。接下来，我们开始从起点开始扩展，直到所有顶点都被访问为止。在扩展过程中，我们更新距离和前驱表，并找到最短路径。最后，我们返回距离和前驱表。
