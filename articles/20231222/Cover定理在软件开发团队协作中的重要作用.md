# 2.核心概念与联系

## 2.1 Cover定理简介

Cover定理是一种信息论定理，它可以用于评估一个随机变量的熵，从而衡量信息的不确定性。Cover定理的基本思想是，通过对一个随机变量的分布进行估计，可以得到该随机变量的熵。Cover定理的数学表达式如下：

$$
H(X) = -\sum_{x \in X} p(x) \log p(x)
$$

其中，$H(X)$ 表示随机变量 $X$ 的熵，$p(x)$ 表示随机变量 $X$ 取值 $x$ 的概率。

```python
import numpy as np

def cover_definition(probabilities):
    """
    Calculate the entropy of a random variable using Cover definition.
    """
    entropy = -np.sum(probabilities * np.log2(probabilities))
    return entropy

# Example usage
probabilities = np.array([0.1, 0.3, 0.2, 0.4])
entropy = cover_definition(probabilities)
print(f"Entropy: {entropy}")
```
