# 2. 核心概念与联系

在了解患者组织学在诊断和研究方面的关键角色之前，我们需要了解一些关键概念。

## 2.1 患者组织学

患者组织学是一种非营利性组织，其成员主要是患病者和他们的家属。这些组织通常旨在提供信息、资源和社区，以帮助患者更好地管理他们的疾病。这些组织可以是针对特定疾病的，也可以是针对特定病种或病人群体的。

诊断是医生根据症状、检查结果和其他信息确定患者所有患病的原因。诊断是医疗系统的核心部分，因为它为治疗和研究提供了基础。

## 2.3 研究

研究是通过科学方法来探索和解决问题的过程。在医疗领域，研究通常涉及诊断、治疗和预防。
