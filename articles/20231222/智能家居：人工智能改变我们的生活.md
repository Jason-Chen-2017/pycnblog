# 4.具体代码实例和详细解释说明
# 4.1 智能灯泡控制示例

以下是一个智能灯泡控制的Python代码示例：

```python
import requests
import json

# 设备信息
device_info = {
    "device_id": "123456",
    "device_type": "light",
    "device_ip": "192.168.1.100"
}

# 控制指令
control_command = {
    "command": "turn_on",
    "brightness": 50
}

# 发送控制指令
response = requests.post("http://{device_ip}/api/control".format(**device_info), json=control_command)

# 解析响应
response_data = json.loads(response.text)
print(response_data)
```
