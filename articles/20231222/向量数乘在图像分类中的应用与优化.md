## 3.1 向量数乘的定义

向量数乘是一种在向量空间中进行的操作，它可以用来计算两个向量之间的内积。给定两个向量 $a$ 和 $b$，向量数乘可以表示为：

$$
c = a^T b
$$

其中，$c$ 是向量数乘的结果，$a^T$ 是向量 $a$ 的转置。

## 3.2 向量数乘的性质

向量数乘具有以下性质：

1. 交换律：对于任意向量 $a$ 和 $b$，有 $a^T b = b^T a$。
2. 分配律：对于任意向量 $a$、$b$ 和 $c$，有 $(a + b)^T c = a^T c + b^T c$ 和 $k(a^T b) = (ka)^T b$，其中 $k$ 是一个标量。
3. 非负性：对于任意向量 $a$ 和 $b$，如果 $a$ 和 $b$ 的元素都非负，则 $a^T b \geq 0$。
