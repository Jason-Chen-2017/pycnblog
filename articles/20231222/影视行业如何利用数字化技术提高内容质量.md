## 4.1 制作技术
### 4.1.1 3D技术
#### 4.1.1.1 代码实例
```python
import bpy

# 创建一个立方体
bpy.ops.mesh.primitive_cube_add(location=(0,0,0))

# 设置立方体的材质
material = bpy.data.materials.new(name="MyMaterial")
material.use_nodes = True
nodes = material.node_tree.nodes
links = material.node_tree.links

# 添加一个颜色节点
color_node = nodes.new(type="ShaderNodeValToRGB")
color_node.inputs["Color"].default_value = (1, 0, 0)

# 添加一个输出节点
output_node = nodes.new(type="ShaderNodeOutputMaterial")
output_node.location = color_node.location
links.new(color_node.outputs["Color"], output_node.inputs["Surface Color"])

# 给立方体应用材质
cube = bpy.data.objects["Cube"]
cube.data.materials.append(material)
```
#### 4.1.1.2 解释说明
上述代码首先创建了一个立方体，然后设置了立方体的材质，最后给立方体应用了材质。

