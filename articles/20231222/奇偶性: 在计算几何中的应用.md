### 1.3.1 求解多边形的面积

在计算几何中，求解多边形的面积是一个非常重要的问题。奇偶性可以用于判断多边形是否为凸多边形。如果多边形是凸多边形，那么它的面积可以通过Heron公式求解。Heron公式是一种用于求解三角形面积的公式，它可以用于求解任意一个三角形的面积。在求解多边形面积时，我们可以将多边形拆分为多个三角形，然后通过Heron公式求解每个三角形的面积，最后将所有三角形的面积相加得到多边形的面积。

### 1.3.2 判断两个线段是否相交

在计算几何中，判断两个线段是否相交是一个重要的问题。奇偶性可以用于判断线段的交点是否在同一侧。如果线段的交点是在同一侧，那么线段就不会相交。具体的操作步骤如下：

1. 计算两个线段的交点。
2. 计算两个线段的奇偶性。
3. 如果两个线段的奇偶性相同，那么线段的交点是在同一侧，因此线段不会相交。
4. 如果两个线段的奇偶性不同，那么线段的交点是在不同的侧，因此线段可能会相交。

2. 计算基准点到多边形其他点的奇偶性。
3. 如果基准点到多边形其他点的奇偶性都是一样的，那么基准点是在多边形内部。
4. 如果基准点到多边形其他点的奇偶性不一致，那么基准点可能不在多边形内部。

### 1.4.1 求解多边形的面积

```python
def heron(a, b, c):
    s = (a + b + c) / 2
    area = (s * (s - a) * (s - b) * (s - c)) ** 0.5
    return area

def area(points):
    n = len(points)
    area = 0
    for i in range(n):
        x1, y1 = points[i]
        x2, y2 = points[(i + 1) % n]
        area += heron(abs(x1 - x2), abs(y1 - y2), ((x1 - x2) ** 2 + (y1 - y2) ** 2) ** 0.5)
    return area

points = [(0, 0), (2, 0), (1, 2)]
print(area(points))
```
