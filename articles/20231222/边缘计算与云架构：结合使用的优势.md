# 2.核心概念与联系

首先，我们需要了解边缘计算和云计算的基本概念。

## 2.1 边缘计算

边缘计算是一种计算模式，将数据处理和分析从中央集中式数据中心移动到边缘设备，如传感器、摄像头和其他智能设备。这种模式的主要优势在于可以减少数据传输延迟、降低网络负载、提高数据安全性和隐私保护。

边缘计算可以应用于各种场景，如智能城市、智能制造、智能能源等。例如，在智能城市中，边缘计算可以用于实时监控交通状况、环境质量、安全设施等，从而提高城市管理效率。

## 2.2 云计算

云计算是一种基于互联网的计算资源提供方式，允许用户在需要时轻松访问和使用计算资源和数据存储。云计算可以提供各种服务，如计算服务、存储服务、数据库服务等。

云计算的主要优势在于可以降低运营成本、提高资源利用率、提供灵活的扩展能力。云计算可以应用于各种场景，如电子商务、在线教育、游戏等。例如，在电子商务中，云计算可以用于处理大量订单、用户数据、商品信息等，从而提高商业运营效率。

## 2.3 边缘计算与云计算的联系

边缘计算与云计算的结合使用，可以充分发挥各自的优势，提高整体效率。具体来说，边缘计算可以处理大量实时数据，降低网络延迟，提高数据安全性和隐私保护；而云计算可以提供大规模的计算资源和数据存储，提供灵活的扩展能力。

边缘计算与云计算的结合使用，可以应用于各种场景，如智能城市、智能制造、智能能源等。例如，在智能能源中，边缘计算可以用于实时监控能源消耗、生成、传输等，从而优化能源分配；而云计算可以用于处理大量能源数据、提供远程控制服务等。
