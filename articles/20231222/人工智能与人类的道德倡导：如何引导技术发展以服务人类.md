## 4.1 道德判断算法实例

以下是一个简单的道德判断算法实例，用于判断一段文本是否包含敏感词汇。

```python
import re

def is_sensitive_word(text):
    sensitive_words = ['敏感词1', '敏感词2', '敏感词3']
    for word in sensitive_words:
        if re.search(word, text):
            return True
    return False
```

在此代码中，我们首先定义了一些敏感词汇，然后通过正则表达式来检查文本中是否包含敏感词汇。如果包含，则返回 `True`，表示存在道德问题；否则返回 `False`。

## 4.2 道德优化算法实例

以下是一个简单的道德优化算法实例，用于将敏感词汇替换为星号。

```python
def replace_sensitive_word(text):
    sensitive_words = ['敏感词1', '敏感词2', '敏感词3']
    for word in sensitive_words:
        text = text.replace(word, '*' * len(word))
    return text
```

在此代码中，我们首先定义了一些敏感词汇，然后通过 `replace` 函数来将文本中的敏感词汇替换为星号。这样可以在保持文本信息完整性的同时，避免泄露敏感信息。
