### 4.1 数据采集示例

数据采集示例：使用传感器采集机器的温度、速度、力等状态信息。

```python
import time
import board
import adafruit_si7021
import adafruit_motor_ds3277

# 温度传感器
i2c = board.I2C()
si7021 = adafruit_si7021.SI7021(i2c)

# 速度传感器
speed_sensor = adafruit_motor_ds3277.DS3277(i2c)

# 力传感器
force_sensor = adafruit_motor_ds3277.DS3277(i2c)

# 获取温度、速度、力的值
temperature = si7021.temperature
speed = speed_sensor.speed
force = force_sensor.force

print("Temperature: {:.2f}°C".format(temperature))
print("Speed: {:.2f} RPM".format(speed))
print("Force: {:.2f} N".format(force))
```
