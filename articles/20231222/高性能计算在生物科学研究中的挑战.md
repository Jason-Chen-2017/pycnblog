6. 根据标记的行和列，构建出两个序列之间的对齐关系。

# 4.具体代码实例和详细解释说明

在这里，我们将给出一个序列对齐算法的具体代码实例和详细解释说明。

```python
def needman_wunsch(seq1, seq2):
    len1, len2 = len(seq1), len(seq2)
    score_matrix = [[float('-inf')] * (len2 + 1) for _ in range(len1 + 1)]
    for i in range(len1 + 1):
        score_matrix[i][0] = 0
    for j in range(len2 + 1):
        score_matrix[0][j] = 0
    for i in range(1, len1 + 1):
        for j in range(1, len2 + 1):
            match = seq1[i - 1] == seq2[j - 1]
            score_matrix[i][j] = max(score_matrix[i - 1][j - 1] + match,
                                     score_matrix[i - 1][j] - 1,
                                     score_matrix[i][j - 1] - 1)
    traceback = [['' for _ in range(len2 + 1)] for _ in range(len1 + 1)]
    i, j = len1, len2
    while i > 0 or j > 0:
        match = seq1[i - 1] == seq2[j - 1]
        if i > 0 and j > 0:
            if match:
                traceback[i][j] = traceback[i - 1][j - 1] + match
            else:
                traceback[i][j] = '*'
        elif i > 0:
            traceback[i][j] = traceback[i - 1][j] - 1
        else:
            traceback[i][j] = traceback[i][j - 1] - 1
        if traceback[i][j] == '*':
            i -= 1
            j -= 1
        else:
            i -= 1
            j -= 1
    align1, align2 = '', ''
    for i, j in enumerate(traceback[len1]):
        if j == '*':
            break
        align1 += seq1[len1 - i - 1]
        align2 += seq2[len2 - i - 1]
    return align1, align2, score_matrix[len1][len2]

seq1 = 'ACTG'
seq2 = 'ACGT'
align1, align2, score = needman_wunsch(seq1, seq2)
print('Alignment 1:', align1)
print('Alignment 2:', align2)
print('Score:', score)
```

这个代码实例实现了 Needleman-Wunsch 算法，它是一种用于序列对齐的算法。算法的输入是两个序列，输出是两个序列之间的对齐关系和得分。具体来说，代码首先定义了一个 `needman_wunsch` 函数，然后实现了算法的核心逻辑。最后，调用了这个函数来对两个序列进行对齐，并打印了对齐结果。
