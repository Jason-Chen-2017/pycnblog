## 4.1 代码实例：光纤传输系统

我们以一个简单的光纤传输系统为例，来展示准粒子在光子电子技术中的应用。

import matplotlib.pyplot as plt

# 定义光纤传输系统的参数
n0 = 1.0  # 核心光纤的折射率
n1 = 1.5  # 胶体光纤的折射率

# 计算单位长度的光纤的损耗

# 绘制光信号和传输后的光信号

plt.plot(t, s_transmitted, label='Transmitted Signal')
plt.legend()
plt.show()
