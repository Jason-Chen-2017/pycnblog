## 1.1 背景介绍

决策支持系统（Decision Support System，DSS）是一种利用计算机和数据库技术为管理者提供有关决策过程中的信息和支持的系统。决策支持系统的主要目标是帮助管理者更好地理解复杂问题，并在做出决策时更有针对性地选择行动。

社交网络分析（Social Network Analysis，SNA）是一种研究人际关系和交流的方法，通过分析人际关系网络中的结构、动态和属性来理解人们之间的关系、交流和影响力。社交网络分析在决策支持系统中具有广泛的应用，例如市场营销、政府政策、公共卫生、金融服务等领域。

在决策支持系统中，社交网络分析可以帮助我们回答以下问题：

- 哪些人具有较高的影响力？
- 如何找到关键的联系人或组织？
- 如何识别和预测潜在的社会动荡或市场趋势？
- 如何优化组织内部的沟通和协作？

为了更好地应用社交网络分析在决策支持系统中，我们需要掌握其核心概念、算法和技术。在下面的部分中，我们将详细介绍这些内容。

## 1.2 核心概念与联系

在进入具体的算法和技术之前，我们需要了解一些核心概念，包括节点、边、网络、中心性、度、 Betweenness Centrality 等。

### 1.2.1 节点（Node）和边（Edge）

节点（Node）是社交网络中的基本元素，表示人、组织或其他实体。边（Edge）表示节点之间的关系或连接。一个简单的社交网络可以用一个节点集合和边集合来描述，其中节点集合是一个无序集合，边集合是一个有序二元组集合。

### 1.2.2 网络（Network）

网络（Network）是节点和边的组合，可以用图（Graph）来表示。图是一个有穷集合V的集合，其中V是一个无序集合，V中的元素称为节点（Vertex），集合E是V x V的有序二元组的集合，E中的元素称为边（Edge）。

### 1.2.3 中心性（Centrality）

中心性（Centrality）是衡量节点在社交网络中的重要性的一个指标，通常包括度中心性（Degree Centrality）、路径中心性（Path Centrality）和 closeness 中心性（Closeness Centrality）等。中心性可以帮助我们识别社交网络中的关键节点。

### 1.2.4 度（Degree）

度（Degree）是节点与其他节点的连接数，用于衡量节点在社交网络中的活跃程度。度中心性是根据节点的度来衡量节点在社交网络中的重要性。

### 1.2.5 Betweenness Centrality

Betweenness Centrality 是一种路径中心性，用于衡量节点在所有可能路径中的占比。一个节点的 Betweenness Centrality 越高，表示它越容易成为不同组件之间的桥梁，因此越重要。
