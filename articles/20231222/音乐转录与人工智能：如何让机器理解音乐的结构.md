# 2. 核心概念与联系

在进行音乐转录的人工智能研究之前，我们需要了解一些核心概念和联系。

## 2.1 音乐谱面

音乐谱面是音乐作品的图形表示，包括音符、弦、节奏等元素。音乐谱面可以用来表示音乐作品的结构和特点，也可以用来指导音乐家演奏。

## 2.2 音符

音符是音乐谱面中最小的音乐单位，表示音乐作品中的音高和音长。音符可以分为多种类型，如整音符、半音符、小节符等。

## 2.3 弦

弦是指音乐作品中的音乐器琴弦等字符，表示音乐作品中的音乐器和音乐器的弦位。

## 2.4 节奏

节奏是音乐作品中的节拍和拍子等元素，表示音乐作品的时间结构和节奏特点。

## 2.5 人工智能与音乐转录
