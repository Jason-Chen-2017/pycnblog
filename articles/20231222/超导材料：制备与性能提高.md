                 

# 1.背景介绍

超导材料是一类具有超导性的材料，它们在零温度下可以导电，而不会产生电阻。这种现象在一般的金属中是不可能的，但是在某些特殊的条件下，如低温、高压或者特殊的化学组成，这些材料可以展示出超导性。超导材料的发现和研究对于现代物理学、电子学和材料科学的发展具有重要的影响力。

超导材料的发现可以追溯到1911年，当时荷兰物理学家Heike Kamerlingh Onnes在研究氢的液体时，发现了超导现象。他发现当氢液被冷却到绝对零度时，它可以导电，而不会产生电阻。这一发现为后来的超导材料研究奠定了基础。

到目前为止，已经发现了几种不同类型的超导材料，如氧化钠超导、钼超导和高氧化铬超导等。这些材料在低温下的超导性表现出不同的性能特征，并且在不同的应用领域具有不同的重要性。
