# 2.核心概念与联系

## 2.1 量子错误纠正（Quantum Error Correction, QEC）
量子错误纠正是一种方法，用于在量子计算机上检测和纠正错误。量子计算机的主要特点是它们使用的是量子比特（qubit），而不是传统的二进制比特（bit）。由于量子比特的特性，它们很容易受到环境干扰和其他因素的影响，这可能导致计算结果的错误。

量子错误纠正通过将量子比特组合成多量子比特（logical qubit）来实现错误检测和纠正。这种组合方式允许量子计算机检测和纠正错误，而不会损失计算能力。量子错误纠正的一个重要优点是它可以提高量子计算机的可靠性和准确性。

## 2.2 量子神经网络（Quantum Neural Networks, QNN）
量子神经网络是一种利用量子计算机进行神经网络计算的方法。它们可以处理大规模的数据集，并且在某些问题上表现出更高的准确性和速度。量子神经网络通常由一组量子层组成，每个量子层包含多个量子神经元（qubit）。这些量子神经元可以进行线性混合和非线性激活函数运算，从而实现神经网络的计算。

量子神经网络的一个主要优点是它们可以处理大规模的数据集，并且在某些问题上表现出更高的准确性和速度。然而，量子神经网络也面临着一些挑战，其中之一是量子错误纠正。
