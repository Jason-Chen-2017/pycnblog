# 4.具体的代码实例

在本节中，我们将通过一个简单的数据清洗示例来解释数据清洗的概念和方法。

## 4.1 数据清洗示例

假设我们有一个客户信息表，包括客户姓名、客户年龄、客户电话和客户地址等属性。部分客户信息如下：

| 客户姓名 | 客户年龄 | 客户电话 | 客户地址 |
| --- | --- | --- | --- |
| 张三 | 25 | 13888889999 | 北京市 |
| 李四 | 30 | 13888889999 | 上海市 |
| 王五 | 35 | 13888889999 | 北京市 |
| 赵六 | 40 | 13888889999 | 上海市 |
| 张三 | 30 | 13888880000 | 北京市 |
| 李四 | 35 | 13888880000 | 上海市 |
| 王五 | 40 | 13888880000 | 北京市 |
| 赵六 | 45 | 13888880000 | 上海市 |

从上述客户信息表中可以看出，客户姓名为张三、李四、王五和赵六的记录重复出现了，这会影响数据的准确性和完整性。为了解决这个问题，我们可以对数据进行清洗，将重复记录删除或修正。

### 4.1.1 数据清洗示例代码

```python
import pandas as pd

# 读取客户信息表
df = pd.read_csv('customer_info.csv')

# 删除重复记录
df.drop_duplicates(inplace=True)

# 保存清洗后的客户信息表
df.to_csv('customer_info_cleaned.csv', index=False)
```

在上述代码中，我们首先使用pandas库读取客户信息表，然后使用drop_duplicates()方法删除重复记录，最后使用to_csv()方法保存清洗后的客户信息表。
