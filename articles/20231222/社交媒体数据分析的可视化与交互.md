## 4.1 数据收集与预处理
我们将使用Python的Tweepy库来收集微博数据，并使用Pandas库来进行数据预处理。

```python
import tweepy
import pandas as pd

# 设置API密钥和访问密钥
consumer_key = 'your_consumer_key'
consumer_secret = 'your_consumer_secret'
access_token = 'your_access_token'
access_token_secret = 'your_access_token_secret'

# 设置API对象
auth = tweepy.OAuthHandler(consumer_key, consumer_secret)
auth.set_access_token(access_token, access_token_secret)
api = tweepy.API(auth)

# 收集微博数据
tweets = api.search(q='社交媒体', count=100, lang='zh')

# 数据预处理
data = [[tweet.id, tweet.text, tweet.created_at, tweet.user.followers_count] for tweet in tweets]
df = pd.DataFrame(data, columns=['id', 'text', 'created_at', 'followers_count'])
df.to_csv('weibo_data.csv', index=False)
```

## 4.2 数据分析
我们将使用Python的Scikit-learn库来进行数据分析。

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.decomposition import PCA

# 数据加载
df = pd.read_csv('weibo_data.csv')

# 文本特征提取
vectorizer = TfidfVectorizer(stop_words='english')
X = vectorizer.fit_transform(df['text'])

# 主成分分析
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)

# 数据可视化
df_pca = pd.DataFrame(X_pca, columns=['PC1', 'PC2'])
df_pca['id'] = df['id']
df_pca['created_at'] = df['created_at']
df_pca['followers_count'] = df['followers_count']
df_pca.to_csv('weibo_pca.csv', index=False)
```

我们将使用Python的Matplotlib库来进行可视化。
