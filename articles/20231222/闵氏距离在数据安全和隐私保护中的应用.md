# 2.核心概念与联系

## 2.1 闵氏距离的定义

闵氏距离（Manhattan Distance）是一种度量两点距离的方法，通常用于二维或三维空间中。它的定义为：两点之间曼哈顿距离的计算方式为：
$$
d = |x_1 - x_2| + |y_1 - y_2|
$$
其中，$(x_1, y_1)$ 和 $(x_2, y_2)$ 是两个点的坐标，$|x_1 - x_2|$ 和 $ |y_1 - y_2|$ 分别表示横坐标和纵坐标的绝对差值。

## 4.1 计算闵氏距离的Python代码实例

```python
def manhattan_distance(point1, point2):
    x1, y1 = point1
    x2, y2 = point2
    return abs(x1 - x2) + abs(y1 - y2)

point1 = (1, 2)
point2 = (4, 6)
print(manhattan_distance(point1, point2))
```

在上述代码中，我们定义了一个名为`manhattan_distance`的函数，该函数接受两个点的坐标作为输入，并返回它们之间的闵氏距离。然后，我们定义了两个点`point1`和`point2`，并调用`manhattan_distance`函数计算它们之间的闵氏距离。

Q: 闵氏距离与欧几里得距离有什么区别？

A: 闵氏距离是在二维或三维空间中，曼哈顿距离的计算方式为：两点之间曼哈顿距离的计算方式为：$$
d = |x_1 - x_2| + |y_1 - y_2|
$$
而欧几里得距离是在二维或三维空间中，计算方式为：$$
d = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}
$$
闵氏距离只考虑横纵坐标的绝对差值的和，而欧几里得距离则考虑横纵坐标的差值的平方和的平方根。闵氏距离更适合在曼哈顿空间中进行计算，而欧几里得距离更适合在欧几里得空间中进行计算。
