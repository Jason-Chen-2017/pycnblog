# 2.核心概念与联系

在进行机器学习任务时，我们通常需要对数据进行二分类，即将数据划分为两个不同的类别。线性可分是一种用于描述这种二分类问题的概念，它指的是在特征空间中，两个类别之间是否存在一个线性分隔面（hyperplane），使得所有属于同一类别的样本点都在该分隔面的一侧，而属于另一类别的样本点都在另一侧。

线性可分的核心概念包括：

1. 特征空间：特征空间是指由输入特征组成的多维空间，每个维度对应于一个特征。在机器学习中，我们通常使用高维特征空间来表示数据。

2. 线性分隔面：线性分隔面是指在特征空间中，由一组线性相关的向量构成的面。线性分隔面可以用矩阵表示，其中每一列向量对应于一个特征。

3. 线性可分：如果在特征空间中，两个类别之间存在一个线性分隔面，则称这个问题为线性可分的。

线性可分与其他机器学习概念的联系：

1. 支持向量机（SVM）：支持向量机是一种用于解决线性可分问题的算法，它的核心思想是找到一个最大化间隔的线性分类器，使得该分类器同时满足所有训练样本点的约束条件。

2. 逻辑回归：逻辑回归是一种用于解决线性可分问题的算法，它通过最大化似然函数来学习模型参数，从而实现样本的分类。

3. 线性判别分类（LDA）：线性判别分类是一种用于解决线性可分问题的算法，它通过找到一个最大化类别间距离，最小化内部距离的线性分类器来学习模型参数。
