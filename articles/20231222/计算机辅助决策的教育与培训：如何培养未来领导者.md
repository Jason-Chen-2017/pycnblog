# 2.核心概念与联系

在本节中，我们将介绍计算机辅助决策的核心概念，以及与其他相关领域的联系。

## 2.1 计算机辅助决策的核心概念

### 2.1.1 决策支持系统（DSS）

决策支持系统（DSS）是一种帮助用户在不确定环境下做出更好决策的系统。DSS 通常包括以下组件：

- 数据库：存储有关问题的数据
- 数据处理和分析工具：用于处理和分析数据
- 模型：用于描述问题和解决方案的数学模型
- 用户界面：用于用户与系统交互

### 2.1.2 预测决策支持系统（PDSS）

预测决策支持系统（PDSS）是一种特殊类型的 DSS，它旨在预测未来事件。PDSS 通常包括以下组件：

- 数据库：存储有关问题的数据
- 数据处理和分析工具：用于处理和分析数据
- 预测模型：用于预测未来事件的数学模型
- 用户界面：用于用户与系统交互

### 2.1.3 人工智能（AI）

人工智能（AI）是一种利用计算机程序模拟人类智能的技术。AI 可以用于解决各种问题，包括计算机辅助决策。AI 的主要技术包括：

- 机器学习：机器学习是一种通过学习从数据中提取规律的方法。机器学习可以用于预测、分类、聚类等任务。
- 深度学习：深度学习是一种通过神经网络学习的方法。深度学习可以用于图像识别、自然语言处理等任务。
- 规则引擎：规则引擎是一种通过规则来描述问题和解决方案的方法。规则引擎可以用于知识表示、推理等任务。
