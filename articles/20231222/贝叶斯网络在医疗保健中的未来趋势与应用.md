# 2. 核心概念与联系

## 2.1 贝叶斯网络的基本概念

贝叶斯网络是一种有向无环图（DAG），其节点表示随机变量，边表示变量之间的条件依赖关系。贝叶斯网络可以用来表示一个条件独立性结构，即给定其他变量的条件下，某些变量之间是独立的。

贝叶斯网络的一个重要特点是它可以用来表示概率分布，即给定一个条件下的概率分布，可以得到一个完整的概率分布。这使得贝叶斯网络成为一种强大的推理工具，可以用来推理新的信息，预测未来的情况，甚至用来做决策。
