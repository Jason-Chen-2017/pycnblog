# 4. 具体代码实例和详细解释说明
在这一节中，我们将通过一个具体的代码实例来详细解释Presto的核心算法原理和操作步骤。

## 4.1 代码实例
```
SELECT a.column1, b.column2
FROM table1 a
JOIN table2 b
ON a.column1 = b.column2
WHERE a.column1 > 1000
GROUP BY a.column1
ORDER BY COUNT(*) DESC
LIMIT 10;
```

## 4.2 详细解释说明
1. 首先，我们使用SELECT子句指定要查询的列。
2. 然后，我们使用FROM子句指定表1作为查询的来源。
3. 接下来，我们使用JOIN子句将表1与表2进行连接，根据两个表的相同列进行连接。
4. 接着，我们使用WHERE子句筛选出列1大于1000的记录。
5. 然后，我们使用GROUP BY子句对结果进行分组，根据列1进行分组。
6. 接下来，我们使用ORDER BY子句对分组后的结果按照计数 Desc 进行排序。
7. 最后，我们使用LIMIT子句限制返回结果的数量为10。
