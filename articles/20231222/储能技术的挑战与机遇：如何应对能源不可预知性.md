# 2.核心概念与联系

## 2.1 储能技术概述
储能技术是指能源储存和发电设备的技术，主要包括电容器、电池、流动化学能源（FES）、热储等。这些技术可以将能源存储在不同形式的设备中，以便在需要时提供电力供应。

## 2.3 储能技术与能源不可预知性的关系
储能技术可以应对能源不可预知性，通过将能源存储在设备中，在需要时提供电力供应，缓冲能源价格波动，提高能源供应的可靠性和可预测性。因此，储能技术在应对能源不可预知性方面具有重要意义。
