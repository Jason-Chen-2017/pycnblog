## 2. 核心概念与联系

### 2.1 独立同分布的定义

独立同分布（Independent and Identically Distributed, IID）是一种数据分布，它表示数据之间相互独立且具有相同的分布。更正式地说，如果随机变量X1、X2、...、Xn是独立同分布的，那么对于任何的实数a1、a2、...、an和b，有：

P(X1 ≤ a1, X2 ≤ a2, ..., Xn ≤ an) = P(X1 ≤ a1) * P(X2 ≤ a2) * ... * P(Xn ≤ an)

其中，P(Xi ≤ ai)是随机变量Xi在ai以下的概率，i = 1, 2, ..., n。
