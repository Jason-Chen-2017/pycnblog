## 4.1 Splunk 代码实例
在这里，我们将给出一个 Splunk 代码实例，用于查询日志数据。

```
index=main sourcetype=access_log | stats count by source
```

这段代码的意思是：从名为 main 的索引中，类型为 access_log 的日志数据中，统计 source 字段的计数。
