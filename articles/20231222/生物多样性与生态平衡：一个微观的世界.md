
```python
model.compile(optimizer='adam', loss='mse', metrics=['accuracy'])
```

最后，我们需要训练我们的模型。我们将使用一个生态数据集来训练模型。

```python
model.fit(x_train, y_train, epochs=10, batch_size=32)
```
