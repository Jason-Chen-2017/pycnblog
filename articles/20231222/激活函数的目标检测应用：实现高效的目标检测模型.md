# 4.具体代码实例和详细解释说明

在Python中，使用Pytorch实现目标检测模型的激活函数如下所示：

```python
import torch
import torch.nn as nn

class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.conv1 = nn.Conv2d(3, 64, 3, padding=1)
        self.relu = nn.ReLU()

    def forward(self, x):
        x = self.conv1(x)
        x = self.relu(x)
        return x

# 创建模型实例
model = Net()

# 定义输入数据
input = torch.randn(1, 3, 32, 32)

# 进行前向传播
output = model(input)
print(output)
```

在上述代码中，我们定义了一个简单的目标检测模型，其中包含一个卷积层和一个ReLU激活函数。通过调用模型的`forward`方法，我们可以对输入数据进行前向传播，得到模型的输出。
