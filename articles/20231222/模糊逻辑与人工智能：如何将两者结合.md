### 1.模糊集的表示和处理

模糊集可以用以下数学模型公式表示：

$$
\mu_A(x) = \begin{cases}
0, & \text{if } x \leq a \\
\frac{x - a}{m - a}, & \text{if } a < x < m \\
\frac{b - x}{b - m}, & \text{if } m \leq x \leq b \\
1, & \text{if } x > b
\end{cases}
$$

其中，$\mu_A(x)$ 是模糊集 $A$ 的元素 $x$ 的度量值，$a$ 和 $b$ 是模糊集 $A$ 的支持域，$m$ 是模糊集 $A$ 的中心值。

模糊集的常见操作包括：

- 模糊合并：给定两个模糊集 $A$ 和 $B$，求出一个新的模糊集 $C$，使得 $C$ 的度量值在 $A$ 和 $B$ 的度量值之间取得平衡。
- 模糊差分：给定两个模糊集 $A$ 和 $B$，求出一个新的模糊集 $C$，使得 $C$ 的度量值在 $A$ 和 $B$ 的度量值之间有差异。

其中，$R(x, y)$ 是模糊关系 $R$ 的元素 $x$ 和 $y$ 之间的度量值，$a$ 和 $b$ 是模糊关系 $R$ 的支持域，$m$ 是模糊关系 $R$ 的中心值。

def triangular_fuzzy_set(x, a, m, b):
    if x <= a:
        return 0
    elif a < x < m:
        return (x - a) / (m - a)
    elif m <= x <= b:
        return (b - x) / (b - m)
    else:
        return 1

def fuzzy_union(a, b):
    a_union_b = np.zeros_like(a)
    for i in range(a.shape[0]):
        a_union_b[i] = (a[i] + b[i]) / 2
    return a_union_b

def fuzzy_intersection(a, b):
    a_intersection_b = np.zeros_like(a)
    for i in range(a.shape[0]):
        a_intersection_b[i] = a[i] * b[i]
    return a_intersection_b

b = np.array([0.4, 0.3, 0.2, 0.1, 0.0])
m = 0.3
