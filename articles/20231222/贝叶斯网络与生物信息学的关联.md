# 2.核心概念与联系

## 2.1 贝叶斯网络

贝叶斯网络是一种有向无环图（DAG），其节点表示随机变量，边表示变量之间的条件依赖关系。贝叶斯网络可以用来表示一个概率模型，该模型描述了随机变量之间的条件独立性和条件概率关系。

贝叶斯网络的一个关键特点是它可以通过对条件独立性进行条件化来简化概率计算。这使得贝叶斯网络在处理大规模的随机变量集合时具有很大的优势。

生物信息学是一门研究生物数据的计算科学领域。生物信息学的主要目标是研究生物数据，以揭示生物过程的机制和功能。生物信息学的应用范围广泛，包括基因功能预测、蛋白质结构预测、病理生物学等。
