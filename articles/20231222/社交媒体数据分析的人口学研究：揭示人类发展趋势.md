# 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的Python代码实例，用于从微博API中获取数据，并进行基本的分析。

```python
import requests
import json

# 获取微博API访问令牌
access_token = 'your_access_token'

# 获取用户关注列表
url = 'https://api.weibo.com/2/friendships/friends.json'
params = {'access_token': access_token}
response = requests.get(url, params=params)
data = json.loads(response.text)

# 提取用户信息
users = []
for user in data['users']:
    users.append({
        'id': user['id'],
        'screen_name': user['screen_name'],
        'gender': user['gender'],
        'province': user['province']
    })

# 计算用户性别分布
gender_count = {'male': 0, 'female': 0}
for user in users:
    gender_count[user['gender']] += 1

# 计算用户地理分布
province_count = {}
for user in users:
    province = user['province']
    if province in province_count:
        province_count[province] += 1
    else:
        province_count[province] = 1

# 打印结果
print('用户性别分布:', gender_count)
print('用户地理分布:', province_count)
```

这个代码实例首先从微博API中获取用户关注列表，然后提取用户信息，计算用户性别分布和用户地理分布。最后，将结果打印出来。
