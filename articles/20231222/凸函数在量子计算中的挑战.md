# 2.核心概念与联系

## 2.1 凸函数

凸函数是一种在整个定义域内具有最小值的函数，对于任意给定的两个点，其对应的斜率都是非负的。在数学上，如果给定一个向量空间V和其对应的内积$\langle \cdot, \cdot \rangle$，一个函数$f: V \rightarrow \mathbb{R}$被称为凸函数，如果对于任意给定的$x, y \in V$和$0 \leq t \leq 1$，都有$f(tx + (1-t)y) \leq tf(x) + (1-t)f(y)$。

## 2.2 量子计算

量子计算是一种利用量子比特（qubit）和量子门（quantum gate）的计算方法，它在处理一些特定类型的问题时具有显著优势。量子比特不同于经典比特，因为它可以处于多个状态同时，这使得量子计算能够同时处理多个问题。

## 2.3 量子优化

量子优化是一种利用量子计算机解决经典优化问题的方法。优化问题是寻找满足一组约束条件的最优解的问题。量子优化通过将问题状态编码到量子状态中，并利用量子门和量子计算机来解决这些问题。

    return -(x[0]**2 + x[1]**2)

    return np.array([x[0] + x[1] - 1])

# 将量子电路编译并运行

result = qobj.run().result()
