## 1.4 具体代码实例和详细解释说明

在这里，我们将给出一个具体的并行计算代码实例，并详细解释说明其工作原理。

```python
import numpy as np
from multiprocessing import Pool

def compute_pi(n):
    h = 1.0 / (2 * n)
    sum = 0.0
    for i in range(n):
        sum += (np.random.rand() * 2 - 1) * (np.random.rand() * 2 - 1) * h
    return sum

def parallel_compute_pi(num_processes, num_iterations):
    with Pool(num_processes) as pool:
        results = pool.map(compute_pi, [num_iterations] * num_processes)
    return np.sum(results)

if __name__ == "__main__":
    num_processes = 4
    num_iterations = 10000
    pi = parallel_compute_pi(num_processes, num_iterations)
    print(f"Parallel compute pi: {pi}")
```

这个代码实例使用Python的`multiprocessing`库来实现并行计算。`compute_pi`函数用于计算单个迭代的π值，`parallel_compute_pi`函数则使用`Pool`类的`map`方法将`compute_pi`函数应用于`num_iterations`次，并将结果累加得到最终的π值。

在`if __name__ == "__main__":`块中，我们设置了4个处理器（`num_processes = 4`）和每个处理器迭代10000次（`num_iterations = 10000`）。最终，我们将得到一个近似的π值。

这个代码实例展示了如何使用并行计算来提高计算效率，从而更快地解决复杂的物理学问题。
