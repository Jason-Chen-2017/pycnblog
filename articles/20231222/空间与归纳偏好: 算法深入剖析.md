1. 问题分解：将问题分解为更小的子问题。
2. 解决方案存储：将子问题的解决方案存储在内存中。
3. 子问题组合：将存储在内存中的解决方案组合成原问题的解决方案。

$$
T(n) = O(n \log n)
$$

1. 问题分解：将问题分解为更一般的问题。
2. 递归求解：通过递归的方式得到更一般问题的解决方案。

首先，我们来看一个简单的阶乘计算算法：

```python

    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)
```

这个算法的时间复杂度为$O(n)$，因为它需要递归地计算$n$个阶乘。

### 3.2 空间优化

接下来，我们将通过空间优化技术来优化上述算法。我们将使用一个列表来存储已经计算过的阶乘值，以减少重复计算：

```python

    if n == 0:
        return 1
    elif n in memo:
        return memo[n]
    else:
        memo[n] = n * factorial_space_optimized(n - 1, memo)
        return memo[n]
```

```python

    result = 1
    for i in range(1, n + 1):
        result *= i
    return result
```
