# 2.核心概念与联系

内积是一种数学概念，它在线性代数中表示为两个向量之间的乘积。在图像处理中，内积主要用于计算两个向量之间的点积。点积是一种常用的向量乘积，它可以用来计算两个向量之间的夹角、长度等信息。

在图像处理中，内积主要用于以下几个方面：

1. 图像特征提取：通过计算图像中不同像素点之间的相似性，可以提取图像的特征信息。
2. 图像分类：通过计算图像特征向量之间的距离，可以对图像进行分类。
3. 图像识别：通过计算图像特征向量与标签向量之间的相似性，可以对图像进行识别。
