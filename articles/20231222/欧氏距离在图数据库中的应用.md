## 2.2 欧氏距离

欧氏距离是一种度量空间中两点距离的方法，通常用于二维或三维空间中。欧氏距离的公式为：

$$
d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}
$$

其中，$(x_1, y_1)$ 和 $(x_2, y_2)$ 是两个点的坐标。

在图数据库中，欧氏距离可以用于计算两个节点之间的距离，从而实现节点相似性的检测、聚类分析、路径查找等应用。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的例子来演示欧氏距离在图数据库中的应用。我们将使用Python的NetworkX库来创建图数据库，并使用NumPy库来计算欧氏距离。

```python
import networkx as nx
import numpy as np

# 创建图数据库
G = nx.Graph()

# 导入数据
nodes = [('A', 10, 20), ('B', 30, 40), ('C', 50, 60)]
edges = [('A', 'B'), ('B', 'C')]

# 添加节点和边
for node, attr in nodes:
    G.add_node(node, **attr)
for edge in edges:
    G.add_edge(edge[0], edge[1])

# 映射节点
x_coords = [node[0][1] for node in G.nodes(data=True)]
y_coords = [node[0][2] for node in G.nodes(data=True)]

# 计算距离
distances = []
for i in range(len(x_coords)):
    for j in range(i+1, len(x_coords)):
        dx = x_coords[i] - x_coords[j]
        dy = y_coords[i] - y_coords[j]
        distance = np.sqrt(dx**2 + dy**2)
        distances.append((i, j, distance))

# 打印距离
for i, j, distance in distances:
    print(f"Distance between node {i} and node {j}: {distance}")
```

在这个例子中，我们首先创建了一个图数据库，然后导入了数据。接着，我们为每个节点分配了一个二维空间中的坐标，并使用欧氏距离公式计算两个节点之间的距离。最后，我们打印了距离结果。
