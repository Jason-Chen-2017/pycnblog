# 4.具体代码实例和详细解释说明
在这里，我们将给出一个简单的电力网络规模扩展的算法实例，并详细解释其过程。

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建一个有向无权网络
G = nx.DiGraph()

# 添加节点
G.add_node(1)
G.add_node(2)
G.add_node(3)

# 添加边
G.add_edge(1, 2)
G.add_edge(2, 3)

# 绘制网络图
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True)
plt.show()
```

在这个实例中，我们首先导入了networkx和matplotlib.pyplot这两个库，networkx用于创建和操作网络，matplotlib.pyplot用于绘制网络图。然后我们创建了一个有向无权网络G，添加了三个节点1、2、3，并添加了两条边1-2和2-3。最后，我们使用nx.spring_layout()函数布局节点，并使用nx.draw()函数绘制网络图，并显示出来。
