```python
class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, value):
        if not self.head:
            self.head = Node(value)
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = Node(value)

    def print_list(self):
        current = self.head
        while current:
            print(current.value, end=' ')
            current = current.next
        print()

# 创建一个链表并添加元素
ll = LinkedList()
ll.append(1)
ll.append(2)
ll.append(3)

# 打印链表
ll.print_list()
```

在上述代码中，我们首先定义了一个链表节点类`Node`，它包含一个值和一个指向下一个节点的指针。然后我们定义了一个链表类`LinkedList`，它包含一个头节点和两个方法：`append`和`print_list`。`append`方法用于向链表中添加元素，`print_list`方法用于打印链表中的元素。最后，我们创建了一个链表实例，添加了三个元素，并打印了链表。

```python
import numpy as np

# 数据集
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 6, 8, 10])

# 计算权重
learning_rate = 0.01
n_iterations = 1000
weights = np.zeros(1)

for _ in range(n_iterations):
    predictions = X * weights
    errors = y - predictions
    weights -= learning_rate * X.T.dot(errors)

# 打印权重
print(weights)
```

在上述代码中，我们首先定义了一个线性回归数据集，包括一个特征矩阵`X`和一个目标变量向量`y`。然后我们定义了一个学习率`learning_rate`和迭代次数`n_iterations`。接着，我们使用梯度下降法计算权重`weights`。最后，我们打印了权重。

```python

def square(x):
    return x * x

if __name__ == '__main__':
    num_processes = 4
    pool = Pool(num_processes)

    data = [1, 2, 3, 4, 5]
    results = pool.map(square, data)

```

在上述代码中，我们首先定义了一个`square`函数，它接收一个参数并返回其平方。然后我们使用`multiprocessing`库创建了一个进程池`pool`，指定了4个进程。接着，我们使用`map`函数将`data`列表中的元素并行地传递给`square`函数，并获取结果列表`results`。最后，我们打印了结果。

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# 生成密钥
key = get_random_bytes(16)

# 生成加密对象
cipher = AES.new(key, AES.MODE_ECB)

# 加密数据
data = b'Hello, world!'
encrypted_data = cipher.encrypt(data)

# 解密数据
decrypted_data = cipher.decrypt(encrypted_data)

print(decrypted_data.decode())
```

在上述代码中，我们首先通过`Crypto`库导入了AES加密功能。然后我们生成了一个16字节的随机密钥`key`。接着，我们使用`AES.new`函数生成了一个加密对象`cipher`，并设置了ECB模式。接着，我们使用`encrypt`函数将`data`数据加密，并将结果存储在`encrypted_data`变量中。最后，我们使用`decrypt`函数将`encrypted_data`数据解密，并将结果打印出来。
