## 2.1 图 State 压缩（Graph State Compression, GSC）

1. 图的邻接矩阵（Adjacency Matrix）：邻接矩阵是一种用于表示图的数据结构。它是一个 n×n 的矩阵，其中 n 是图中节点的数量。矩阵的每一行和每一列都对应于图中的一个节点。矩阵的每一个元素 a[i][j] 表示节点 i 和节点 j 之间的边的权重。

2. 图的邻接表（Adjacency List）：邻接表是一种用于表示图的数据结构。它是一个节点到节点的映射，其中键是节点，值是该节点的邻接节点集合。

# 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来演示 GSC 算法的实现。

```python
import networkx as nx

# 创建一个有向图
G = nx.DiGraph()

# 添加节点
G.add_node("A")
G.add_node("B")
G.add_node("C")
G.add_node("D")

# 添加边
G.add_edge("A", "B", weight=1)
G.add_edge("B", "C", weight=2)
G.add_edge("C", "D", weight=3)
G.add_edge("A", "D", weight=4)

# 对图进行压缩
compressed_G = nx.graph_compression.greedy_compression(G)

# 查找最短路径
shortest_path = nx.shortest_path(compressed_G, "A", "D", weight="weight")
print(shortest_path)
```

在这个代码实例中，我们首先创建了一个有向图，并添加了节点和边。接着，我们对图进行了压缩，并使用了 GSC 算法来查找最短路径。
