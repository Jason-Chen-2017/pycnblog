                 

# 1.背景介绍

SQL注入是一种常见的网络攻击手段，它利用用户输入的数据来构造恶意的SQL语句，从而欺骗数据库服务器执行非法操作。这种攻击方法通常用于获取受保护的数据，篡改数据，或者对数据库服务器进行拒绝服务（DoS）攻击。

SQL注入攻击的核心在于攻击者通过构造特殊的输入值，让数据库服务器执行未经授权的操作。这种攻击通常发生在网站或应用程序中，其中用户输入的数据被直接嵌入到SQL语句中，从而导致SQL注入。

以下是一个简单的SQL注入示例：

```python

password = input("请输入您的密码：")

cursor.execute(sql_query)
```
