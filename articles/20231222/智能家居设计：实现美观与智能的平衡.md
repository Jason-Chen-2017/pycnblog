## 4.1 智能家居设备连接示例

```python
import paho.mqtt.client as mqtt

def on_connect(client, userdata, flags, rc):
    print("连接状态：" + str(rc))
    client.subscribe("smart_home/light")

def on_message(client, userdata, msg):
    print(msg.topic + " " + str(msg.payload))

client = mqtt.Client()
client.on_connect = on_connect
client.on_message = on_message
client.connect("mqtt.eclipse.org", 1883, 60)
client.loop_forever()
```

在这个示例代码中，我们使用了Paho MQTT库来实现智能家居设备的连接。首先，我们导入了Paho MQTT库，并定义了连接状态的回调函数on_connect和消息接收的回调函数on_message。然后，我们创建了一个MQTT客户端对象client，并设置了连接状态的回调函数和消息接收的回调函数。最后，我们使用client.connect()方法连接到MQTT服务器，并使用client.loop_forever()方法开始循环监听消息。
