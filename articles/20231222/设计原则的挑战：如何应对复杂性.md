## 4.具体代码实例和详细解释说明

在这里，我们将提供一些具体的代码实例，以展示如何应用上述原理和方法。

### 4.1 并行处理示例

```python
import multiprocessing as mp

def square(x):
    return x * x

def main():
    pool = mp.Pool(4)
    results = pool.map(square, range(10))
    print(results)
    pool.close()
    pool.join()

if __name__ == '__main__':
    main()
```

在这个示例中，我们使用了Python的multiprocessing库实现了并行处理。我们将一个列表中的元素平方并返回结果。通过设置池的大小，我们可以控制并行度。

### 4.2 分布式处理示例

```python
import requests

def fetch_data(url):
    response = requests.get(url)
    return response.text

def main():
    urls = ['http://example.com/data1', 'http://example.com/data2', 'http://example.com/data3']
    results = []
    with mp.Pool(4) as pool:
        results = pool.map(fetch_data, urls)
    print(results)

if __name__ == '__main__':
    main()
```

在这个示例中，我们使用了Python的requests库和multiprocessing库实现了分布式处理。我们将一个列表中的URL地址传递给池，池将这些任务分配给多个进程处理。每个进程负责从指定的URL中获取数据。通过设置池的大小，我们可以控制分布式度。
