# 4.2 Aster Numerix的具体代码实例
以下是一个使用Aster Numerix进行线性回归分析的具体代码实例：

```sql
-- 创建一个表
CREATE TABLE housing (
    id INT,
    price INT,
    size INT,
    bedrooms INT,
    year INT
);

-- 插入数据
INSERT INTO housing VALUES
    (1, 300000, 1200, 3, 1990),
    (2, 400000, 1500, 4, 2000),
    (3, 500000, 2000, 5, 2010),
    (4, 600000, 2500, 5, 2020);

-- 使用线性回归算法进行预测
SELECT id, price, size, bedrooms, year,
    price_predicted AS predicted_price
FROM housing
CROSS JOIN Numerix.LinearRegression(
    price AS target,
    size,
    bedrooms,
    year,
    'Learn',
    'Auto'
) AS regression
ORDER BY predicted_price DESC;
```
