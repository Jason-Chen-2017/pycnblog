# 4.具体代码实例和详细解释说明
在这里，我们将提供一个具体的代码实例，以便帮助读者更好地理解分布式计算在气候变化研究中的应用。

```python
from mpi4py import MPI
import numpy as np

# 初始化MPI
comm = MPI.COMM_WORLD
size = comm.Get_size()
rank = comm.Get_rank()

# 生成气候数据
data = np.random.rand(1000)

# 划分任务
chunk_size = int(data.shape[0] / size)
start = chunk_size * rank
end = start + chunk_size

# 执行任务
if rank == 0:
    result = np.zeros(size)
    for i in range(size):
        chunk_data = data[start:end]
        chunk_result = np.sum(chunk_data)
        result[i] = chunk_result
else:
    chunk_data = data[start:end]
    chunk_result = np.sum(chunk_data)
    comm.Send(chunk_result, dest=0)

# 收集结果
if rank == 0:
    for i in range(1, size):
        result[i] = comm.Recv()
    print("Result:", result)
```

这个代码实例使用了MPI库来实现分布式计算。首先，我们初始化MPI并获取计算节点的数量和编号。然后，我们生成一个气候数据数组，并将其划分为多个子任务。接下来，我们使用MPI的Send和Recv函数来执行任务并收集结果。最后，我们将结果打印出来。
