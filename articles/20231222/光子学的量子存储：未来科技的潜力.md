# 2. 核心概念与联系

## 2.1 光子学基础

光子学是一门研究光子的科学。光子是光的基本单位，它是一个无量纲的量子粒子，具有波特性和粒子性。光子可以通过光源生成，通过光学和电子元件传输，通过光电转换器将其转换为电信号。光子学的核心概念包括：

- 光子的产生、传播和干扰
- 光子的分辨率和噪声
- 光子的存储和重构

## 2.2 量子存储基础

量子存储是一种基于量子力学的存储技术，它利用量子比特（Qubit）来存储信息。量子比特不同于传统的二进制比特，它可以存储两种不同的状态：0、1，也可以存储其他混合状态。量子存储的核心概念包括：

- 量子比特的创建和操作
- 量子比特的存储和重构
- 量子比特的错误纠正

## 2.3 光子学量子存储的联系
