                 

# 1.背景介绍

人类对宇宙的探索始于古代的哲学家们对天体的观察和猜测。随着科学的发展，人类逐渐将宇宙的观念从神话和神学转向科学。在20世纪，随着天体物理学、天文学和宇宙学的发展，人类对宇宙的认识得到了巨大的进步。

在20世纪60年代，宇宙学家莱布尼茨（Edwin Hubble）的观测证实了大宇宙观念：宇宙中的星系相互运动，这些运动的速度和方向相同，这意味着宇宙中存在着一个中心，这个中心被称为大气球。此外，莱布尼茨还发现了宇宙中的恒星轨道斜率，这意味着宇宙中存在着遥远的恒星。

随着科学技术的不断发展，人类对宇宙的探索也不断扩展。在20世纪70年代，美国的阿波罗宇宙探险队成功地探测到月球，这是人类对宇宙的第一次实际探索。在20世纪80年代，美国的维克特伯特宇宙探险队成功地探测到麦克唐纳恒星系统，这是人类对遥远星球的第一次探测。

在20世纪90年代，人类对宇宙的探索得到了新的突破。美国的海伯特宇宙望台成功地观测到了遥远的恒星，这是人类对宇宙的第一次观测遥远恒星的成功。此外，在20世纪90年代，美国的迪士尼宇宙观成功地观测到了遥远的星球，这是人类对宇宙的第一次观测遥远星球的成功。

在21世纪，人类对宇宙的探索得到了更大的进步。在2009年，美国的帕劳顿宇宙望台成功地观测到了遥远的星球，这是人类对宇宙的第一次观测遥远星球的成功。此外，在2015年，美国的新天地望台成功地观测到了遥远的星球，这是人类对宇宙的第一次观测遥远星球的成功。
