在行为主义中，我们可以通过设计奖励和惩罚机制来塑造孩子的正确行为。例如，我们可以设计一个奖励系统，当孩子完成某个任务时，给予他们奖励，例如颁发薪金或奖金。同时，我们也可以设计一个惩罚系统，当孩子做出不良行为时，给予他们惩罚，例如扣除薪金或奖金。

```python
def reward(child, task, reward):
    if child.complete_task(task):
        child.receive_reward(reward)
        return True
    else:
        return False

def punish(child, task, punishment):
    if child.do_wrong_thing(task):
        child.receive_punishment(punishment)
        return True
    else:
        return False
```
