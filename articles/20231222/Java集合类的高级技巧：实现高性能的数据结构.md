## 4.具体代码实例和详细解释说明

在本节中，我们将通过具体的代码实例来详细解释Java集合类的使用方法和优化方法。

### 4.1代码实例

我们以ArrayList和HashMap为例，来展示Java集合类的具体代码实例。

```java
import java.util.ArrayList;
import java.util.HashMap;
import java.util.Map;

public class Main {
    public static void main(String[] args) {
        // 创建一个ArrayList对象
        ArrayList<Integer> list = new ArrayList<>();
        // 添加元素
        list.add(1);
        list.add(2);
        list.add(3);
        // 遍历元素
        for (int i : list) {
            System.out.println(i);
        }

        // 创建一个HashMap对象
        Map<String, Integer> map = new HashMap<>();
        // 添加键值对
        map.put("one", 1);
        map.put("two", 2);
        map.put("three", 3);
        // 遍历键值对
        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            System.out.println(entry.getKey() + ":" + entry.getValue());
        }
    }
}
```

### 4.2详细解释说明

在上述代码实例中，我们首先创建了一个ArrayList对象，然后添加了三个整数元素，并遍历了这些元素。接着，我们创建了一个HashMap对象，添加了三个键值对，并遍历了这些键值对。

通过这个代码实例，我们可以看到Java集合类的使用方法和优化方法。例如，我们可以使用ArrayList来实现顺序访问的集合，使用HashMap来实现键值对的集合。
