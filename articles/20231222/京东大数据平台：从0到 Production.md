                 

# 1.背景介绍

京东大数据平台是京东公司在2010年左右开始建设的，目的是为了解决京东公司在大数据处理方面的需求。京东公司是中国最大的电商平台，每天处理的数据量非常大，需要一套高效的大数据处理平台来支撑其业务发展。

京东大数据平台涵盖了京东公司的所有业务，包括电商、物流、金融、游戏等各个领域。平台的核心功能包括数据收集、存储、处理、分析和可视化等。京东大数据平台的构建和运营是京东公司高级管理层重视的一个重要项目，平台的建设和运营也是京东公司高管的一个重要工作内容。

京东大数据平台的核心架构包括数据收集、数据存储、数据处理、数据分析和数据可视化等五个部分。数据收集部分负责从各个业务系统中收集数据，数据存储部分负责存储收集到的数据，数据处理部分负责对存储的数据进行处理，数据分析部分负责对处理后的数据进行分析，数据可视化部分负责将分析结果以图表、图片等形式展示给用户。

京东大数据平台的核心技术包括Hadoop、Spark、Flink、Hive、Presto、Impala、Kafka、Storm等。这些技术都是大数据处理领域的主流技术，京东公司在大数据处理方面有着丰富的实践经验，并在大数据处理领域有着一定的技术领导力。
