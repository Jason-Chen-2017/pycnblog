# 2.核心概念与联系

## 2.1 向量范数

向量范数是一种用于度量向量长度的量度。在数学中，向量范数通常定义为向量的模（即向量的长度）的绝对值。常见的向量范数有两种，即欧几里得范数（Euclidean norm）和曼哈顿范数（Manhattan norm）。

### 2.1.1 欧几里得范数

欧几里得范数（L2范数）是对一个向量的模的绝对值，可以通过以下公式计算：

$$
\| \mathbf{v} \|_2 = \sqrt{\sum_{i=1}^{n} v_i^2}
$$

其中，$\mathbf{v}$ 是一个 $n$ 维向量，$v_i$ 是向量的第 $i$ 个元素。

### 2.1.2 曼哈顿范数

曼哈顿范数（L1范数）是对一个向量的模的绝对值，可以通过以下公式计算：

$$
\| \mathbf{v} \|_1 = \sum_{i=1}^{n} |v_i|
$$

其中，$\mathbf{v}$ 是一个 $n$ 维向量，$v_i$ 是向量的第 $i$ 个元素。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来展示如何使用向量范数计算高维数据的距离。我们将使用 Python 编程语言和 NumPy 库来实现这个功能。

```python
import numpy as np

def euclidean_distance(v1, v2):
    return np.sqrt(np.sum((v1 - v2) ** 2))

def manhattan_distance(v1, v2):
    return np.sum(np.abs(v1 - v2))

v1 = np.array([1, 2, 3])
v2 = np.array([4, 5, 6])

print("欧几里得距离:", euclidean_distance(v1, v2))
print("曼哈顿距离:", manhattan_distance(v1, v2))
```

在这个代码实例中，我们首先导入了 NumPy 库，然后定义了两个函数 `euclidean_distance` 和 `manhattan_distance`，分别用于计算欧几里得距离和曼哈顿距离。接着，我们定义了两个向量 `v1` 和 `v2`，并使用我们定义的两个函数计算它们之间的距离。最后，我们输出了结果。
