# 4.具体代码实例和详细解释说明
在这里，我们将通过一个简单的最短路问题来展示随机算法的具体实现。我们将使用一个简单的有向图来表示城市之间的距离关系，并使用随机算法来找到一条最短的旅行路线。

```python
import random

def random_algorithm(graph, start_node):
    visited = set()
    path = []
    current_node = start_node
    path.append(current_node)
    visited.add(current_node)

    while True:
        neighbors = graph[current_node]
        if not neighbors:
            break

        next_node = random.choice(neighbors)
        if next_node not in visited:
            visited.add(next_node)
            path.append(next_node)
            current_node = next_node

    return path

graph = {
    'A': ['B', 'C'],
    'B': ['D', 'E'],
    'C': ['F'],
    'D': [],
    'E': ['F'],
    'F': []
}

start_node = 'A'
path = random_algorithm(graph, start_node)
print(path)
```

在这个代码实例中，我们首先定义了一个简单的有向图，表示城市之间的距离关系。然后，我们定义了一个`random_algorithm`函数，它使用随机算法来找到一条最短的旅行路线。最后，我们调用这个函数并输出结果。
