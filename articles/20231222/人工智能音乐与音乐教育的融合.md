# 4.具体代码实例和详细解释说明

## 4.1 音乐创作的代码实例

在这个例子中，我们将使用Python的`numpy`库来生成一段随机音乐。

```python
import numpy as np

# 定义音乐规则
notes = ['C', 'C#', 'D', 'D#', 'E', 'F', 'F#', 'G', 'G#', 'A', 'A#', 'B']
rhythms = ['1/4', '1/2', '1/8', '1/16']

# 生成随机音乐
music = []
for i in range(16):
    note = np.random.choice(notes)
    rhythm = np.random.choice(rhythms)
    music.append((note, rhythm))

# 输出音乐
for note, rhythm in music:
    print(f'{note} {rhythm}')
```

## 4.2 音乐分析的代码实例

在这个例子中，我们将使用Python的`music21`库来分析一段音乐作品。

```python
from music21 import converter, stream

# 加载音乐作品
file = 'example.mid'
stream = converter.parse(file)

# 分析音乐作品
notes = [note.pitch for note in stream.notes]
rhythms = [note.startTime for note in stream.notes]

# 输出音乐特征
for note, rhythm in zip(notes, rhythms):
    print(f'Note: {note}, Rhythm: {rhythm}')
```

## 4.3 音乐推荐的代码实例

在这个例子中，我们将使用Python的`scikit-learn`库来实现基于内容的音乐推荐。

```python
from sklearn.neighbors import NearestNeighbors
import pandas as pd

# 加载音乐数据
data = pd.read_csv('music_data.csv')
features = data[['danceability', 'energy', 'valence']]

# 训练推荐模型
model = NearestNeighbors(metric='cosine', algorithm='brute')
model.fit(features)

# 推荐音乐作品
query = np.array([0.7, 0.8, 0.6])
distances, indices = model.kneighbors(query, n_neighbors=5)

# 输出推荐音乐作品
for distance, index in zip(distances[0], indices[0]):
    print(f'Distance: {distance}, Index: {index}')
```
