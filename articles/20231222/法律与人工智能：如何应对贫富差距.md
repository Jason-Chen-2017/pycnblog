# 4.具体代码实例和详细解释说明
以下是一个简单的Python代码示例，展示了如何使用线性规划算法来优化资源分配：

```python
from scipy.optimize import linprog

# 资源需求和供应
resources_demand = [10, 5]
resources_supply = [20, 10]

# 权重
weights = [1, 2]

# 线性规划问题
minimize_resources = [0, 0]
for i in range(len(resources_demand)):
    minimize_resources[i] = -resources_demand[i]

# 优化
result = linprog(minimize_resources, [resources_supply], [-weights], bounds=[[0, None], [0, None]])

print(result)
```

这个代码示例使用了scipy库中的linprog函数来解决一个简单的线性规划问题。这个问题涉及到资源需求和供应，以及资源的权重。通过优化这个问题，我们可以确保资源的分配更加公平和有效。
