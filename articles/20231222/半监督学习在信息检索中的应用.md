# 2.核心概念与联系

## 2.1 半监督学习
半监督学习是一种机器学习方法，它在训练数据中同时包含有标注数据和无标注数据。有监督学习只依赖于标注数据进行学习，而无监督学习只依赖于无标注数据进行学习。半监督学习则利用有限的标注数据来指导学习，同时利用无标注数据来扩大训练数据集，从而提高学习效果。

```python

from sklearn.pipeline import Pipeline

# 使用有监督学习训练一个文本分类器

pipeline.fit(X_train_labeled, y_train_labeled)

# 使用无监督学习对训练数据中的文本进行聚类

# 将聚类结果作为无标注数据

# 将有标注数据和无标注数据结合起来，形成一个混合数据集
```
