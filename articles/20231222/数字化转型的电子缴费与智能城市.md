# 4.具体代码实例和详细解释说明
## 4.1 电子缴费的具体代码实例
以下是一个简单的电子缴费示例代码：

```python
import requests

url = 'https://api.example.com/payment'
data = {
    'user_id': '123456',
    'password': 'password',
    'amount': '100',
    'payment_method': 'alipay'
}

response = requests.post(url, data=data)

if response.status_code == 200:
    print('Payment successful')
else:
    print('Payment failed')
```
