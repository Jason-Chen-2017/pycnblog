$$

我们的任务是找到这个函数的最大值。我们可以使用一个简单的随机搜索算法来解决这个问题。首先，我们需要定义一个随机搜索算法的函数：

```python
import numpy as np

def random_search(f, bounds, n_iter):
    x_best = None
    f_best = -np.inf

    for _ in range(n_iter):
        x = np.random.uniform(*bounds)
        f_val = f(x)

        if f_val > f_best:
            x_best = x
            f_best = f_val

    return x_best, f_best
```

接下来，我们可以使用这个随机搜索算法来解决我们的优化问题：

```python
def f(x):
    return -x**2

```
