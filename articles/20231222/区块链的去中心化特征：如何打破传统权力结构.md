### 3.1哈希函数

哈希函数（hash function）是区块链技术的一个核心算法原理。哈希函数是一个从输入空间到输出空间的函数，它可以将任意长度的输入转换为固定长度的输出。在区块链技术中，哈希函数主要用于实现数据的完整性和不可篡改性。

哈希函数的主要特点包括：

1. 单向性：哈希函数是不可逆的，即从输出空间到输入空间是不可能的。

2. 碰撞性：哈希函数可能存在碰撞，即不同的输入可能产生相同的输出。

3. 扩展性：哈希函数可以将输入空间扩展到输出空间，从而实现数据的完整性和不可篡改性。

在区块链技术中，常用的哈希函数有SHA-256和KECCAK。这些哈希函数具有较高的安全性和效率，并被广泛应用于区块链技术的数据存储和交易验证。

### 4.1创建区块链

在创建区块链之前，我们需要安装一些依赖库，如Web3.js、truffle等。这些库将帮助我们实现区块链技术的各种功能。

首先，我们需要安装Web3.js库：

```
npm install web3
```

接下来，我们可以创建一个简单的区块链：

```javascript
const Web3 = require('web3');

const web3 = new Web3('http://localhost:8545');

const createBlockchain = async () => {
  const blockchain = [];
  web3.eth.getBlockNumber().then(async (blockNumber) => {
    for (let i = blockNumber - 1; i >= 0; i--) {
      const block = await web3.eth.getBlock(i);
      blockchain.push(block);
    }
    console.log(blockchain);
  });
};

createBlockchain();
```

上述代码首先导入Web3库，然后创建一个Web3实例。接下来，我们定义一个`createBlockchain`函数，该函数通过调用`web3.eth.getBlockNumber`获取区块链的高度，并通过循环获取每个区块。最后，我们调用`createBlockchain`函数来创建区块链。

### 4.2创建智能合约

在创建智能合约之前，我们需要安装Truffle框架：

```
npm install -g truffle
```

接下来，我们可以创建一个简单的智能合约：

```solidity
pragma solidity ^0.5.12;

contract SimpleContract {
  uint256 private value;

  event ValueChanged(uint256 indexed oldValue, uint256 newValue);

  modifier onlyOwner() {
    require(msg.sender == owner, "Only the owner can call this function.");
    _;
  }

  address public owner;

  constructor() public {
    owner = msg.sender;
  }

  function setValue(uint256 newValue) public onlyOwner {
    require(newValue > value, "New value must be greater than the current value.");
    value = newValue;
    emit ValueChanged(value, newValue);
  }

  function getValue() public view returns (uint256) {
    return value;
  }
}
```

上述代码首先定义一个Solidity版本，然后定义一个名为`SimpleContract`的智能合约。该合约包含一个`uint256`类型的私有变量`value`，一个`event`类型的`ValueChanged`事件，一个`onlyOwner`模ifier，一个`address`类型的公共变量`owner`，一个构造函数和两个公共函数`setValue`和`getValue`。`setValue`函数通过`onlyOwner`模ifier限制调用者必须是合约的拥有者，并检查新值是否大于当前值。`getValue`函数是只读的，用于获取`value`的值。

### 4.3部署智能合约

接下来，我们可以通过Truffle框架部署智能合约：

```javascript
const Web3 = require('web3');
const web3 = new Web3('http://localhost:8545');
const SimpleContract = require('./build/contracts/SimpleContract.json');

const deployContract = async () => {
  const accounts = await web3.eth.getAccounts();
  const contract = new web3.eth.Contract(SimpleContract.abi);

  const gasEstimate = await contract.deploy({ data: '0x' + SimpleContract.bin }).estimateGas();
  const gasPrice = await web3.eth.getGasPrice();
  const value = web3.utils.toWei('1', 'ether');

  const transaction = {
    from: accounts[0],
    gas: gasEstimate,
    gasPrice: gasPrice,
    data: '0x' + SimpleContract.bin,
    value: value,
  };

  const result = await web3.eth.sendTransaction(transaction);
  console.log('Contract deployed at:', result.contractAddress);
};

deployContract();
```
