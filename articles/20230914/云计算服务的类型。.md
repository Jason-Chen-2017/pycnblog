# 1.简介
  

云计算(Cloud Computing)是一个利用Internet连接网络的计算资源，按需提供给用户的一种服务。它主要基于网络服务、平台服务、软件服务、存储服务、数据服务等的联合组合，包括：基础设施即服务（IaaS）、平台即服务（PaaS）、软件即服务（SaaS）。云计算服务提供了一种经济可行的新型计算模式。随着互联网的普及，云计算已经成为当今生活不可或缺的一部分。
云计算服务可以分为三个层级：IaaS、PaaS、SaaS。
## IaaS层级
基础设施即服务(Infrastructure as a Service，IaaS)，也称基础设施即服务商（IaaS Provider），它是在线服务，用户不需要管理服务器硬件和操作系统，只需要简单配置就能使用云上的服务器资源。在IaaS层级中，服务商提供硬件资源、网络连接、存储设备、操作系统镜像等，开发者可以使用这些资源快速部署虚拟机或其他云主机。这种云上资源的共享和虚拟化使得开发者无需购买自己的服务器硬件，从而降低了成本和投入。
例如：亚马逊AWS、微软Azure、谷歌Google Cloud Platform等都是属于IaaS的云计算服务提供商。

## PaaS层级
平台即服务(Platform as a Service，PaaS)，也称开发平台即服务商（Platform Provider），是指开发者可以在云上构建应用程序，而不需要关心底层的基础设施或服务器配置等。PaaS在这一层级的服务会提供完整的平台环境，包括运行环境、编程语言、框架、工具等。开发者只需要关注业务逻辑、数据库、日志、监控等相关技术细节即可。
例如：IBM Bluemix、Amazon Web Services Elastic Beanstalk、Google App Engine等都是属于PaaS的云计算服务提供商。

## SaaS层级
软件即服务(Software as a Service，SaaS)，也称软件即服务商（Service Provider），是指云上提供各种软件应用，包括办公、生产、法律、教育、金融等领域的应用。SaaS在这一层级的服务一般都可以直接用于日常工作。用户无需安装、更新、配置软件，只需登录到相应的网站或客户端应用，就可以使用软件服务。
例如：SalesForce、Microsoft Office 365、Zoom等都是属于SaaS的云计算服务提供商。

# 2.基本概念术语说明
云计算服务包括以下两个方面：
- 服务类型(Service Types): IaaS、PaaS、SaaS
- 服务供应商(Service Providers): AWS、Azure、Google、IBM等

另外，还有一些基本的术语：
- 提供商(Provider): 云计算服务提供商，如AWS、Azure、Google、IBM等
- 区域(Region): 云计算服务的地理位置，如东京、硅谷、华盛顿
- Zones: 一个区域下可能存在多个可用区（Availability Zone），每一个可用区都有独立的电源、网络、机房等，可用区之间可以通过专用网络相连。
- 服务产品(Services): 用户可用的云计算服务产品，如EC2、VPC、ElastiCache等
- 模板(Templates): 是预配置好的某种类型的云主机。模板保存好了软件环境、组件安装等，用户通过模板创建云主机后，可以直接使用已安装好的软件进行应用开发。
- API(Application Programming Interface): 是计算机软件组件之间的接口，由应用程序调用其中的函数，来实现功能。
- CLI(Command Line Interface): 命令行界面，用于用户跟机器进行交流，与GUI类似，但操作更加简单。
- GUI(Graphical User Interface): 图形用户界面，是通过图形的方式向用户呈现信息，方便用户进行交互。
- SDK(Software Development Kit): 软件开发套件，是提供给开发者使用的API集合。
