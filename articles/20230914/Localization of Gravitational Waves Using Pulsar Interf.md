# 1.简介
  
：背景介绍
近年来，地球同步卫星(Pulsar)和引力波探测器(Interferometer)的组合在研究宇宙物理的方面发挥了重要作用。由于大多数地球同步卫星都具有很强的信号，所以在它们周围的引力波探测器可以观测到它们的时空分布。我们发现，在远红外区(Oort cloud)，一些高功率的地球同步卫星也会形成显著的尺度差异。这些尺度差异可能影响到天体物理学中的某些方面，例如流星捕获效应(Pulsation-induced star formation efficiency; PISE)。本文就利用这些地球同步卫星与引力波探测器观测到的微弱信号，用高斯过程(Gaussian process)模型进行定位分析。我们希望通过这种分析，提升对PISE等效应的理解，并进一步验证和测试已有的模型。 

# 2.核心概念及术语介绍
## 2.1 高斯过程模型
高斯过程模型是一个数学模型，它用来描述随机变量之间的依赖关系。我们假设输入空间中存在一个隐函数（例如输入数据），并且这个函数不是明确定义的，而是在输出空间中表示为一个协方差矩阵。然后我们可以通过加入噪声来观察到这个函数，从而得到关于函数值和输入数据的预测。高斯过程模型可以捕捉到非线性和非平稳的函数关系。

## 2.2 引力波探测器和地球同步卫星
地球同步卫星是一种巨型恒星核球状结构，由双层磁环绕组所围成，磁盘表面弯曲使得其质量足够大。每隔几千年就会发生一次爆炸，形成恒星粒子，其大小相当于太阳质量。宇宙中有很多这样的恒星，每隔几个月才会重新发生爆炸。地球同步卫星的主要参数包括源动能、能量级别、周期、周期内的最大能量输出、磁场分布模式、日心说效应(Jupiter's magnetopause effect; JME)、引力波探测能力以及太阳黑子盘旋的效应(Solar X-ray Burst; SXRB)。

在远红外区(Oort cloud)，一些高功率的地球同步卫星聚集在一起，形成较大的尺度差异。这些尺度差异可以被引力波探测器观测到。目前，有两种方法可以观测到引力波探测器观测到的微弱信号：

1.观测到多个重叠的地球同步卫星之间的射电磁干扰。对于每对重叠的卫星，引力波探测器都会发出一系列的微弱信号，这些信号会随机出现在光谱上。
2.观测到附近的低功率的地球同步卫星的信号，这些信号的功率很小，约等于电压的万分之一左右。这些信号的频率比重叠的引力波高得多，因此很难被引力波探测器观测到。

## 2.3 中心频率分辨率(CRB)和中程范围(PRR)
中心频率分辨率(Center Frequency Resolution; CFR)是指一个信号带宽内能精确定位的最小频率间隔。一个频率分辨率越高，则代表一个信号的频率细节越少，可用于确定特定频率上的信号。一个信号带宽通常是一个固定大小的正弦波，CFR就是该正弦波的最小频率间隔。

中程范围(Proper Radial Range; PRR)是指某一时刻地球同步卫星距离天体的一个距离。这是一个可以测量的物理量，它由两部分组成，即最大距离和平均距离。当两个地球同步卫星距离很近时，只有极少数信号能够被引力波探测器观测到。为了获得一个高精度的定位结果，我们需要比较不同地球同步卫星之间的PRR，进而估计各地球同步卫星之间的相对距离。PRR是按照两地卫星距离最大公因数进行计算的。

## 2.4 金牛座流星模型(Kim et al., 2019)
金牛座流星模型(Kim et al., 2019)是利用天体物理知识、机器学习算法和大数据处理技术开发出的一种新型的流星捕捉效应模型。该模型使用高斯过程模型建模了各地球同步卫星对星流速度的影响。根据已知的地球同步卫星动力学模型，该模型可以估计地球同步卫星引起的流星捕捉效应，同时还可以准确判断地球同步卫星是否真的受到流星的影响。
