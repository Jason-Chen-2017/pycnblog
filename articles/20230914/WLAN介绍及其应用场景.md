# 1.简介
  

WLAN(Wireless Local Area Network)即无线局域网，它是利用无线电波传播的传输媒体技术，将两台计算机或其他设备连接成一个广域网。WLAN的特点是低时延、高速率、短距离、跨越性强、节能性好、可靠性高、灵活性强等优点。WLAN目前在家庭、商业、科研、医疗领域等众多应用领域得到广泛应用。

# 2.基本概念术语说明
## 1.WLAN网络结构
WLAN网络由许多工作站节点(station)组成，节点之间通过无线信道进行通信。节点间的数据传输采用CSMA/CA协议，即载波监听多路访问/串行通信协议。该协议要求每台主机都要预先监听信道，避免冲突产生。WLAN中工作站间数据交换主要采用共享加密密钥的方案，即所有节点采用同一种安全加密方式进行通信。根据使用范围不同，WLAN可以分为以下三种类型：

1. Infrastructure WLAN: 此类WLAN网络通常部署在工控机房或其它室内场所。无线电信号被卫星接收机拦截后再转发到中心基站上，这样可以增加系统抗干扰能力。此类网络数据传输速度可以达到几兆字节/秒，适用于企业内部办公、生产等要求较高的应用。

2. Ad-Hoc WLAN: 此类WLAN网络部署在用户周边，与其它用户不共用物理信道资源，网络由多个独立的节点构成。这种网络由于规模小，距离远，传输速率较慢，但由于使用不同的信道资源，因此存在较高的信噪比，适合即时通信、远程控制等简单应用场景。

3. PAN (Personal Area Network): 此类WLAN网络部署在个人住宅、办公楼、学校等室外空间，可以广播或点对点地与多台主机进行通信。PAN网络一般支持随身携带设备，可以在需要的时候随时搜索附近的热点，实现个人之间的即时通信。

## 2.WLAN常用协议
WLAN主要采用IEEE 802.11标准协议作为传输层协议，该标准协议包括如下几个部分：

1. MAC层: 定义了信道划分、接入策略、地址管理、QoS等机制。

2. PHY层: 提供了信号调制、解调、频谱划分、加解密功能。

3. RTS/CTS协议: 为保证数据的完整性和顺序性，提供了发送端数据包的重传机制。

4. PWRMGMT协议: 支持AP的功率控制策略。

5. IBSS协议: 直接向所有的STA发送无线信息，没有独立的信道。

## 3.WLAN标准
目前WLAN协议的国际化趋势导致很多国家和地区已经制定了自己的WLAN标准，例如美国的802.11a/b/g/n，中国的Wi-Fi、Infiniband等。这些规范中的一些参数值如速率、距离、功耗、安全等都具有很大的实践意义，并且可以通过它们来评价WLAN性能。另外，不同的WLAN标准还可能兼容或融合，共同组成一个大的WLAN系统。

## 4.WLAN安全性
WLAN采用无线信道进行数据传输，使得WLAN本身具有很强的安全性。但是由于WLAN是一个开放的环境，任何人都可以安装无线AP，使得WLAN网络存在安全漏洞。针对此问题，IEEE在MAC层引入RSN（Robust Security Network）标准，通过密码学和认证机制确保WLAN的安全。RSN旨在提升WLAN的安全性，通过强大的加密和认证机制来保护WLAN用户的信息安全。对于未经授权的用户，可以设置加密策略，防止他们的流量被窃取或篡改。另外，WLAN运营商也会提供专门的安全产品来增强WLAN网络的安全性。
