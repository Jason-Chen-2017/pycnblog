# 1.简介
  

自我价值评估(Self-esteem assessment)是一个很重要的领域，也是衡量一个程序员成熟度、能力水平的一个有效指标。评估的方法很多，有主观性质的自我评估方式，比如看重个人职业规划、项目经验、工作态度等；也有客观性质的技术能力评估，比如看重编程语言、数据库技术、系统设计等方面的知识和技能。不同的评估方法对于评估的结果也不同，主观评估需要个人对自己的认识和理解；而客观评估则依赖于工程实践或项目实施的过程中，通过可靠的数据计算出来的统计数字。本文将结合自己多年的工作经历，从两个方面阐述程序员自我价值评估的方法及其效果。希望能够对读者有所启发，帮助读者更准确地评估自己的优劣，并在今后进一步提升自己。
# 2. 定义和历史回顾
## 2.1 自我价值评估的定义
自我价值评估是一种基于客观标准的职业技能考核方式。它通过对个人能力和专业技能的定量分析和综合评价，提供给个人一个比较客观的评判依据。20世纪80年代末至90年代初，国际上流行开展自我价值评估调查，主要是针对青少年学生进行，目的是为了给他们提供职业发展方向上的建议。
自我价值评估又称职业评估、能力评估或技能评估。它是指通过计算机模型和工具，根据社会规范、工作表现、个人特点、风险承受力和职业发展方向等客观因素，对某个人的潜在职业或个人能力水平进行客观的评估，并对其做出个人认为具有价值的参考。一般情况下，对个人自我价值评估的要求是客观、公正、科学，不应过分强调个人情感和个性。一般认为，由于自我价值评估是一个技能考核方式，因此其结果不应该被用来作为个人发展的唯一标准。相反，自我价值评估可以帮助企业、政府部门、管理人员或HR人员了解员工的长处和短处，帮助个人更好地定位自己的价值，同时也促使个人多加锻炼、学习、充实自己，提升职场竞争力。
## 2.2 自我价值评估的历史回顾
### 2.2.1 古代的自我评估方法
自然界中存在着一些非神灵的存在，它们具有各种各样的能力，包括智力、体力、美学、聪明才智等等。它们可以在一定条件下对人的能力进行评估。例如亚里士多德提出的“智商测试”就是这样一种测验方法。亚里士多德以雅利安人智力的突飞猛进而闻名，他认为人的智商可以通过智力的开发来提高，所以他设立了这个测试，即若一个人能够做出两道题目（每道都考察某些难以在短时间内解决的具体问题），他就可以得知自己智商的程度。
### 2.2.2 近代的自我评估方法
约翰·威廉姆斯·麦肯锡提出了著名的“象限图”，来刻画个人的能力和潜力。麦肯锡认为每个人都有两种可能性，分别是他们具备某种天赋，或者没有这种天赋。他把这种两极化的能力分布比作象限图中的四个区域，上半部分的人拥有高度的技能和独到之处，落入较低的层次；下半部分的人则没有特殊能力，但却有丰富的潜力，能发掘自身的长处，寻求新的突破。
麦肯锡的“象限图”理论在近代影响非常大，许多研究者都把它作为一种自我评估的理论基础。如法国科学家尼尔·弗莱彻将人们的能力分为三类：显性能力、隐性能力和创造力。显性能力表现在与生俱来或天生具有的能力，如视力、听觉、嗅觉、味觉、肢体运动、记忆力、计算能力等；隐性能力则是与这些能力类似，但不能完全控制的能力，如想象力、直觉、情绪表达、逻辑思维、文字处理、团队协作等；创造力则是超越了人类认知范畴的能力，如音乐创作、美术创新、工程设计、体育活动、营销策划、理财投资、电脑编程等。
### 2.2.3 信息技术革命后兴起的自我评估需求
随着信息技术的发展，新的管理模式和服务模式出现。传统的管理理念和方法受到了冲击。曾经对人才的管理通常只注重教学能力、办公能力等技能。而在新的管理模式中，更多的任务由计算机软件替代，这让评估人员必须考虑计算机技能的掌握情况。由于计算机的普及性，新兴的技术掌握者也越来越多。因此，计算机技能的评估成为公司、组织关注的热点话题。当下，各大公司纷纷开始启动自我评估服务，以改善员工培训、晋升过程，并为招聘决策提供更加客观、全面的判断。
### 2.2.4 流行病的影响
自我评估的出现引发了一系列的研究热点。由于长期处于劳动力供需关系的均衡状态，大众对自己是否具备良好的职业生涯追求一直在增长。不过，随着人口的急剧增加，出现了对劳动力价格的攀升、失业率的上升以及医疗保障不足等问题，导致许多人纷纷减少职业生涯规划。因此，为了让那些倾向于自我评估的年轻人有所选择，许多公司都推出了相关的行业榜单，展示有一定职业技能的优秀人才。但是，随着市场的发展，各种软件、服务以及设备日益普及，许多公司的评估服务也越来越契合流行病的需要，导致它也变得越来越火爆。

## 2.4 自我价值评估的方法
自我价值评估方法种类繁多，大致可以分为以下几种：

1. 主观性质的自我评估——这是最为常用的自我评估方法。主观性质的自我评估属于一种依靠个人的直觉判断和判断力。通常，通过问卷调查、焦点小组讨论等形式进行。这种方法虽然客观性不强，但可以更直观地看到个人的优缺点，因而有助于了解个人的需求、价值取向、职业志向等。但由于个人的主观判断容易受到环境、家庭、亲友等因素的影响，因此它的评价结果往往不能直接代表一个人的能力和潜力。此外，由于评估对象群体庞大且多元，主观性质的自我评估结果往往无法形成统一的标准。

2. 客观性质的自我评估——客观性质的自我评估属于由计算机模型和工具计算出来的能力评估。例如，通过编程语言、数据结构、算法等方面能力的掌握情况进行评估，或者通过自身项目、职业生涯、竞赛、挫败经验等方面实际案例进行评估。这种方法结合了多种数据指标，既有客观性，又有可重复性。但同时，这种评估方式也存在着很大的局限性。首先，由于个人的能力水平具有不确定性，因此它不能完全客观地反映一个人的真实水平。其次，由于个人的行为可以塑造自己的价值观、理念、世界观，因此评估的结果往往与个人的实际发展息息相关。再者，这种评估方式并不能准确地反映一个人的全部潜能。最后，由于评估的对象群体庞大且多元，因而结果的可信度也存在一定程度的质疑。

3. 混合型的自我评估——由于各个方面的能力都可以影响个人的职业成功，因此，混合型的自我评估方法既有主观性质的自我评估，也有客观性质的自我评估。它既能客观评估个人的能力，又能主观评估个人的潜力和追求。这种方法结合了不同类型数据的优势，既能够客观衡量一个人的能力，又能够直观、精准地发现一个人的优缺点、价值取向、职业志向。但是，这种方法也存在着不可避免的复杂性。

4. 成熟度模型——为评估某个职业人员的成功或失败，学校或培训机构都会制订一个职业成熟度模型。它强调个人在该职业领域的实际能力、适应能力、学习能力、社交能力、责任心、品格等多个方面的综合能力。这种方法采用模型的方式，更为客观准确。但是，成熟度模型本身也存在着局限性。首先，虽然它对个人能力的评估客观有效，但它无法真正深入理解个人的思维、感情、工作习惯和生活方式，只能作为一种职业发展指导工具。其次，成熟度模型无法保证一定精确度，而且还受到评估对象的个人品质、认知水平等因素影响。

总的来说，各类自我评估方法之间存在着交叉互相补充的关系，选择恰当的评估方法，才能更好地了解自我，找到自我价值最大化的关键。
