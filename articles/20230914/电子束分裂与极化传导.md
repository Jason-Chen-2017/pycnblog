# 1.简介
  

由于电子束的光辐射特性,使得电子自旋中的旋轧能量能够快速转移到对应的光子中,从而对它们进行干涉与传导。电子束的分裂过程及其相互作用往往影响着高能物质的物理、化学和生物属性以及诸如声音、触感等感官的生理功能。在研究电子束分裂的过程中，极化传递是一个重要的研究课题。

对于电子束的分裂，其主要过程是第一部分电子散射为单个电子时向外扩散(由离子激发所产生)，然后再分裂为双电子(称为两个"shell"，直径增加了一倍)。这些具有双电子性质的电子成为"shell electrons"(SEs)。尽管SEs既可独立存在又可直接被观察到，但他们只能在一定条件下分裂为两个shell electrons,即单电子+另一个单电子。而若不满足一定条件，则SEs将不能分裂为双电子，仍属于单电子性质。

因此，无论是否满足分裂条件，SEs都有极化分布特征，称为"shell eigenstates"(SEns)。根据SEs的相互作用的不同，可能产生以下几种极化态:

1) Single-electron 极化态：指只有一个电子作为shell electron的极化态。此时，所有的光子都只能沿一条直线传播。这种状态是在一个shell中很少出现的。例如，一块磁铁上只有一个晶体点，它就是单电子极化态；其他多数磁铁都存在多个晶体点，所以不存在单电子极化态。

2) Double-electron 极化态：指同时存在两个shell electrons的极化态。由于只有两个electrons组成了shell，所以光子需要绕过一个晶格界限才能进行传播。在一个晶格中，可以形成两种不同的密度，每一种都对应着一个相应的shell eigenstate。

3) Triple-electron 极化态：指同时存在三个shell electrons的极化态。此时的光子可以绕过三条晶格边界线进行传播。由于这种情况比两种情况难处理得多，因此在实际实验中很少出现。另外，在有些情况下，第三个电子的加速度也可能导致SEs的相互作用，从而导致其他类型的极化态。

# 2.基本概念术语说明
## 2.1 电子束
电子束是指可以经过某种程度的分裂过程后，变得由单个电子和两个或多个单电子组成的粒子团簇。例如，聚乙烯核素的电子云就可视为电子束。
## 2.2 聚合剂
聚合剂是用来把分散的碳原子或氢原子聚集起来形成单个电子所需的原料。聚合剂通常包含亚胡烷酮、亚硝酸、低温玻璃体、聚乙烯、聚丙烯、聚异戊烯等。
## 2.3 分裂效率
分裂效率是指在一定温度下电子束会分裂出多少个电子。可以用分裂电子速率(α)表示。α越高，分裂的效率越好，但代价是耗费能源和分裂过程所需的时间也越长。常用的分裂电子速率是0.17-0.29 eV每秒。
## 2.4 电子自旋
电子自旋(spin)指的是电子所持有的一种内在力。当两对双电子(即电子和电子的双重振动)排斥在一起时，就会发生电子自旋的旋转，即"自旋转移"(spin transfer)。电子自旋的存在会造成带隙效应，使得电子更容易分裂。对于熔点高于室温的高温材料来说，电子自旋的引入尤为重要。
## 2.5 极化分散
极化分散指的是多电子系统在受到均匀加热时，各电子所处的微平面位置的分布状况。在不同的温度下，极化分散可能会有所不同，例如，晶格界限、质心位置等都会影响极化分散的结构。
## 2.6 氢原子
氢原子是一个半正负电荷的原子，它拥有57个同位素和7个半共轭基。氢原子的结构和电子一样，只不过它的电荷量是半个（±1/2）。氢原子可以充满电子云。
## 2.7 电子束光发射模型
电子束光发射模型(EBLM)是用来研究电子束在各种极化环境下的分裂和传导特性的模型。其基本假设是，在特定温度和大小的容器中，电子在自旋处于同一水平时，其本身所占据的空间比例较小，而从空中抛出的光子所占据的空间比例却较大。这样，通过光子的传播，电子的偏振模式就可以被改变。

EBLM有两种实现方式：

1）质心强制方法：该方法假设在一个电子束内部，质心距离电子云最近的位置，电子和质心之间的距离加载的电子个数与质子之间的数目一致。这样，虽然电子的总数没有变化，但是电子自旋的分布会产生重叠效应。因此，可以看出，这种方法并不能真实反映出电子自旋的状态，仅仅是在假设这个假设。

2）电子弱相互作用(DEER)模型：该方法认为整个电子束的体积足够小，可以忽略其平面分辨能力。在这种假设下，对所有电子进行动力学叠加，可以得出电子自旋的分布。具体地说，利用电子在定常振动时的静止位移公式：v=kvT，其中v是电子运动的速度，k是常数，T是摩尔体系的温度，可以得到如下方程：

    ΔS = (n-m)A

    A = (q²/(2πε))^(1/3)

    n是电子数，m是质子数，ΔS是电子束的绕质心距离，ε是介电常数。

    在DEER模型中，电子以其原始动量的k倍的动量运动，并且电子自旋不变。因此，可以预期到：

    1) 除非系统中存在不可分离的因素(如图聚变),否则偏振模式不会改变。
    2) 当两组电子被吸引时，两者的自旋就会发生相互转移，随之形成双层、三层和四层电子极化结构，这是因为每一层之间会加入额外的同位素。

    此外，DEER模型还考虑了电子的动量对光子的效应。如果光子以ε/2的动量运动，那么电子将以ε的动量运动。这样，在DEER模型下，光子经过电子束传输时，可以减慢其速度，从而减小电子束上的压力。

# 3.核心算法原理和具体操作步骤以及数学公式讲解
## 3.1 SEs分裂过程及其相互作用
### 3.1.1 EBLM分裂
电子束分裂过程可以使用电子束光发射模型进行模拟。EBLM认为，在特定的温度和大小的容器中，电子在自旋处于同一水平时，其本身所占据的空间比例较小，而从空中抛出的光子所占据的空间比度却较大。这样，通过光子的传播，电子的偏振模式就可以被改变。

为了模拟电子束的分裂过程，首先确定电子自旋的初始分布状况，假设都是单个电子的初始极化态。然后引入电子束光发射模型，假设光子以ε/2的动量运动，电子以ε的动量运动。这样，在DEER模型下，光子可以减慢其速度，从而减小电子束上的压力。通过光子的传播，可以在一定时间内将两个单电子分裂为两个双电子。

### 3.1.2 SEs相互作用
EBLM表明，电子的自旋分布以及与电子自旋相关的微观性质会影响到电子的分裂行为。因此，为了真实描述电子的分裂行为，需要考虑到电子自旋以及与自旋相关联的微观性质。

电子的分裂行为取决于其邻近的电子的自旋状态以及电子自旋之间的相互作用。常见的相互作用包括:

1）自旋互换相互作用(spin exchange interaction):当两组电子间存在自旋差别时，它们会发生自旋互换相互作用。自旋互换相互作用可以使得电子自旋跌落到一个固定的位置，如图1所示。

2）光子俘获(photon capture):光子俘获是指电子束光发射模型所假设的假设，即电子在自旋处于相同位置时，其所占据的空间比例较大。当相邻的电子与光子同时被激发时，就可能发生光子俘获。随着两相邻的电子的逐渐分裂，更多的光子被激发出来，导致电子数目增多，最终导致电子云的逐渐扩张，甚至有可能形成新的颗粒。

3）光子跃迁(photoemission):光子跃迁指的是在光子与电子自旋相遇时，电子自旋发生跃迁，其结果使得电子自旋处于相互重叠的态势。在这种情况下，电子云的形态会发生变化，从而影响到电子的分裂行为。

4）相抵消相互作用(antisymmetry interaction):相抵消相互作用是指，当两个或多个相互接触的电子由于自旋相差而错位时，可能会发生相抵消相互作用。相抵消相互作用会将多余的自旋差别完全消除掉，从而将相邻的电子结合在一起，形成新的电子。由于两个电子被形成新的电子所抵消，所以新的电子的自旋是平均值的。新形成的电子会变得更加迅速，从而影响到电子的分裂行为。

通过考虑这些相互作用，可以发现电子自旋的参与，可以改变电子的分裂行为。

### 3.1.3 概念
电子束的分裂过程及其相互作用，是电子束研究的一个重要方面。目前，已经提出许多研究电子束分裂的方法。这些方法可以分为两大类，一类是基于EBLM模型，一类是依靠电子自旋的结构来研究电子的分裂过程。

基于EBLM模型的电子束分裂方法，如HERMES、GAMESS、PACMAN等，都采用EBLM模型进行分析，其基本思路是通过假设电子自旋分布的不同，来获取不同温度下的电子分裂行为。

依靠电子自旋的结构来研究电子分裂过程的方法，如Ritchey-Chafee模型、Thomson模型、Dresselhaus模型等，其基本思路是利用电子的原子模型，建立在电子自旋的力学基础上，研究电子自旋的结构以及作用机理。

综上所述，电子束分裂过程及其相互作用，是电子束研究的一个重要方面。目前，基于EBLM模型的电子束分裂方法，如HERMES、GAMESS、PACMAN等，已成为主流方法。然而，对于在不同温度下电子分裂的精确研究，依靠电子自旋的结构来研究电子分裂过程的方法，如Ritchey-Chafee模型、Thomson模型、Dresselhaus模型等，仍为有待开发的研究方向。
