# 1.简介
  

技术领域是一个充满了复杂性的产物，使得很多技术人员因无法掌握某些细枝末节而感到束手无策。作为技术人员自身水平的不足，也给一些技术领域的管理者带来了巨大的压力。越是高级、越是顶尖的技术人员，越需要有效的技术管理才能保障其职业成长和利益的最大化。
本文主要讨论的是如何成为优秀的技术管理人员。首先，我们从技术人员的角度出发，分析技术管理者应该具备哪些素质；然后，以工程项目管理为例，阐述怎样在项目团队中担任技术管理角色，真正做到以人为本，把控项目的成功率；最后，提供一些技巧，帮助技术管理者更加主动地、果断地采用新技术、解决问题，提升个人能力和影响力。


# 2.什么是技术管理？
技术管理就是指对技术产品及服务的开发、维护、管理、优化等方面的专业人员。它是一种技术人员的职业，也是掌握复杂信息技术（包括计算机系统、网络通信、数据库、编程语言等）的专业人员的必修课。简单的说，技术管理就是借助信息技术工具、理论方法、流程规范、组织结构、专业知识、工具、经验积累等优势，以项目管理的原则和方法，指导、组织和协调相关部门，管理、控制、监督公司或项目的技术运作，实现组织目标的任务。

技术管理通常由项目管理和信息系统两个部分组成。项目管理则强调预算、计划、管理和协同等项目活动，将项目从需求提升到执行，确保项目成功。信息系统则涉及的信息管理、安全管理、法律管理、供应链管理、服务管理等方面，旨在对信息技术系统进行完整、准确、科学地管理，确保技术服务的可用性、可靠性和持续性。

总结一下，技术管理者就是一个能管理复杂信息技术的专业人士，具有项目管理和信息系统管理两个重要环节，能够更好地管理企业或者组织的技术发展方向、资源配置和投入，并通过技术管理方式实现产品或服务的开发、迭代、交付、部署、维护、监控、扩展等全过程的管控。


# 3.技术管理者的基本素质
1.承担责任心
作为技术管理人员，必须充分了解自己所在项目的上下游业务需求、产品形态、市场竞争对手、客户群体、开发人员、测试人员、实施人员、其他协作者、第三方服务商等情况，在做决定时站在全局考虑、既考虑到长远利益，又考虑到短期利益。在处理日益复杂的工作环境时，必须保持高度的责任心，能够清晰地认识和定义工作目标、制定方案、推进工作，团队内部也要树立共识，达成一致意见。

2.灵活机动
作为技术管理者，必须敢于突破常规，跳出舒适区，快速响应变化，快速学习新知识，培养创新能力。在管理中，应善于识别并利用机会点，发现问题，主动寻求支持，帮助团队成员解决难题。在工作中，也应逐渐转变工作方式，关注细节，根据情况灵活调整任务分工和执行时间表，不断探索新的技术和管理方法。

3.技术综合素质
技术管理者的专业素养不可谓不重要，但更重要的是技术知识储备。在工作中，技术管理者必须掌握项目管理、IT管理、网络管理、业务流程、营销管理等多种专业技术，同时要掌握基本的数据分析、数据可视化、文档编写、沟通表达等技能。此外，技术管理者还要善于利用现有的工具、平台和资源，协同外部资源完成各种项目。

4.历史经验积累
作为技术管理者，首先要具备丰富的工作经验和个人能力。技术管理者需要熟练使用各种技术软件，具备基本的编程能力，对工程技术有深刻的理解。同时，他还要有良好的沟通和协调能力，在团队之间、内部和外部都要有较强的沟通能力，能够有效地向各个层次的人员传递信息。

5.勤奋努力
技术管理者一定要善于投入时间，主动把控自己的工作节奏和精力分配，尤其是在面临新信息技术、新领域、新挑战的情况下，更要时刻保持警惕和敏锐的头脑，多下功夫研究新技术，掌握最新研究成果，开拓创新思维，努力提升个人能力。

6.坚守正道
作为技术管理者，必须严格遵守企业内部的工作纪律，诚信守誓，坚持原则，脚踏实地，尊重别人的劳动成果，并且建立起科学的评价机制。在团队里，大家要共同关心下属，不断提升自己的工作效率和绩效，不断修正工作上的偏差和不足。
