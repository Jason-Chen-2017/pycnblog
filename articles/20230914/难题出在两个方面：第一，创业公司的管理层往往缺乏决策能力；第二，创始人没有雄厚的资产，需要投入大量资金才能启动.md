# 1.简介
  

前几天，深圳的一家创业公司发布了他们的第一个产品“智能手表”，售价只有99元，但是很快就风靡全国。但是，随着时间的推移，该公司却逐渐被大家所遗忘。因为很多人认为，创业公司缺乏决策能力导致无法快速启动项目，管理层的能力又不足以支撑公司快速扩张，最后陷入了失败的境地。那么，到底什么是创业公司的管理层的决策能力？如何提升管理层的决策能力？带着这些问题，我将结合自己的研究经历、学习成果和专业背景，阐述一下为什么说“管理层缺乏决策能力”以及如何通过学习和实践提升管理层的决策能力。

首先，我想强调一下：创业公司一般都处于一个相对封闭的状态，这意味着管理层只能听命于外部因素，无法获得足够的资源进行独自决策。同时，创始人的资产也会成为限制管理层能力的瓶颈，创业者通常没有足够的个人能力独自承担起公司的管理责任。因此，提升管理层的决策能力需要我们花费较多的时间和精力，并不是一蹴而就的。

因此，解决这个难题最根本的办法就是培养管理层的自主性，允许他们独立做出决定。如果可以找到自己擅长的领域，给予充分关注，并且制定一些明确的决策过程和标准，就能够在较短的时间内取得突破。另外，增强创始人的信心也是重要的。创业公司的失败往往是由于创始人没有正确的方向，或者没有足够的资产支持公司持续运营，因此，培养创始人的坦诚沟通能力也非常重要。

# 2.概念术语
为了更好地理解管理层缺乏决策能力的问题，让我们先了解一下相关的概念术语：

1.创业公司：指的是企业的创立初期阶段，作为其快速发展的一个重要环节，由创始人、联合创始人、法人代表、董事、监事等人员组成。

2.管理层：指由管理人员掌控公司事务和决策的人群。

3.决策能力：指管理者拥有的决策能力，包括信息获取能力、决策分析能力、决策执行能力和控制意识。

4.团队协作：指管理者能够有效地利用资源，提高工作效率的能力。

5.激励机制：指管理者设定的奖惩制度，使得管理者能够获得积极的社会回报。

# 3.核心算法
根据上面的概念，可以总结出两个关键点：第一，管理层缺乏决策能力。第二，创业者没有足够的资产支持公司的持续发展。因此，提升管理层的决策能力应该从以下几个方面入手：

1.建立清晰的决策流程：指清楚地定义公司的目标和市场需求，明确自己的工作职责，并制订明确的行动计划，以便管理层能够集中精力实现目标，确保结果能够达到预期。

2.建立可靠的行动计划：指设置具有持久性的目标，制订紧急事件应急计划，准备必要的物质条件，制订安全保障措施，确保各项资源能够得到最大限度的利用。

3.建立正向激励机制：指管理者给予各级主管，员工的正向激励，鼓励他们按照公司政策、规章制度和管理制度履行职务，建立健康的管理关系，树立企业形象，使其产生积极的社会责任感。

4.加强管理人员之间的沟通联系：指通过不同渠道及时、准确地传达信息，增强管理人员之间的沟通互动，降低管理层与员工的矛盾，防止出现僵局和纠纷。

5.提升管理层的决策水平：指采用新型的管理方法，集中化管理、信息化管理、战略性管理，通过公开、透明、问责制、专业化管理等方式，进一步提升管理层的决策能力，更好地发挥管理功能。

# 4.实际操作
1.建立清晰的决策流程

为了更好的服务客户，公司的目标和市场需求应该清晰，而且应该落实到细节上。比如，对于智能手表，公司的目标是将人体的行为记录下来，并且提供个性化的个性化智能体验。那么，管理层应该设计如下的决策流程：

1）收集信息：确定客户需求，收集相关的数据，如用户反馈、竞品分析、市场占有率分析等。

2）拆解问题：根据客户需求，将问题拆解成多个小问题，然后逐一解决。

3）制订方案：将解决方案制订出来，并逐步完善，确认可行性。

4）评估方案：对方案进行评估，确定是否可行，以及是否存在风险。

5）实施方案：将方案实施到生产线或工厂，并跟踪运行情况。

6）监督方案：定期检查运行状况，确保没有出现任何问题。

7）改进方案：根据产品实践、市场反馈、内部数据等，不断优化方案，确保满足需求。

2.建立可靠的行动计划

作为一家有限公司，公司管理层通常没有足够的财力来完成所有的工作，因此，必须制订必要的物质基础，保证运行良好。比如，可以准备相应的设备、办公室配套设施、交通工具等，作为公司的固定资产。管理层还要制订紧急事件应急计划，提前做好备份，并安排保密工作。

3.建立正向激励机制

除了奖励员工外，管理层还可以鼓励员工按时按质量履行职务。比如，管理层可以给予赢得年终奖、团队活动奖励、生日礼物等激励，鼓励员工踊跃参加团队活动。

4.加强管理人员之间的沟通联系

管理人员之间一定要保持良好的沟通交流习惯，一定不能出现同志间的矛盾冲突，也要严格遵守公司的管理纪律。比如，每周至少召开一次会议，每个月发布年度目标，明确各项工作职责，以及分配任务的权利与义务。

5.提升管理层的决策水平

管理层的决策水平越高，公司的整体效益就越好。因此，提升管理层的决策能力可以通过如下的方式实现：

1）制定绩效管理制度：通过绩效考核、薪酬福利等方式，提升管理层的绩效管理能力。

2）采用新型管理方法：例如，采用决策支持系统（Decision Support System, DSS），引入计算机科学、统计学等方法，辅助管理者进行决策。

3）公开信息：鼓励管理层分享管理知识、制度、实践，促进管理改革，保障公司信息公开。

4）建立专业管理团队：创建专门管理人员的队伍，主导公司管理，不受管理层过多干涉。

综合以上四个方面，如果我们能创造性地应用这些方法，将帮助管理层的决策能力得到有效提升。

# 5.未来发展趋势
从这次的学习和研究过程中，我发现管理层的决策能力是一个困扰创业公司管理层的重要问题。特别是在创业公司刚刚起步的时候，还没有比较成熟的管理理论基础，管理层的决策能力尤其重要。因此，未来的管理层决策能力建设，还需要更多的研究和实践，否则，仍然有可能面临缺乏决策能力的危机。

当然，管理层自主性的培养还需要时间，我们还需要借鉴国际上的管理模式和管理理论，用以提升管理层的决策能力。目前，国内一些公司已经开始试点这种管理模式，但是，仍然还处于早期阶段。我们需要持之以恒地努力，才能让管理层决策能力真正有所提高。
