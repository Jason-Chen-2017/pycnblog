# 1.简介
  

## 概述
海思980处理器是华为公司的第二代高端处理器，也是首个采用了“90nm芯片”这一新型材料工艺的处理器，可以说它代表了华为从工艺到产品质量的一次革命性转变。本文将介绍海思980处理器的主要特性、设计原理、架构设计及其典型应用场景等，并对其进行深入剖析。文章既不局限于技术层面，也要结合实际业务需求和所处行业环境，真正地为读者呈现出完整的产品生态链。同时本文还要重点阐述在应用该处理器时需要注意的问题及相关解决方案，力争让大家通过阅读此文，对海思980处理器有一个全面的认识。
## 作者简介
李云迁，国内著名芯片架构师，具有多年科研经验。他对华为高端服务器、网络设备的建设以及芯片架构等领域均有丰富的经验。2017年，李云迁获得华为科技创新奖，是华为集成电路部门最具影响力的一位奖项获得者之一。

    * 是否支持CUDA编程
    * 是否可用于ASIC开发

    * 是否支持网络计算
    * 是否支持CPU负载和服务器资源隔离
  - （2）解答FAQ
    * CUDA编程是一种基于NVIDIA的异构计算编程模型，支持GPU和CPU混合编程；
    * ASIC开发是利用半自动生成的硬件布线图，将软件指令转换为片上可编程逻辑（PLA）输出，完成集成电路的设计。华为目前已有两个独立的ARM处理器平台，可以做为参考。但是，对于980系列处理器来说，还有非常多的挑战值得期待。
    * 由于980系列处理器引入了Cortex A53/A73等多核架构，因此支持多核编程，但不支持分布式计算。
    * CPU负载和服务器资源隔离是华为自主研发的最佳实践。980系列处理器尚无法完全隔离，这仍然是一个有待进一步完善的技术。
    * 在目前的版本中，980系列处理器只适用于服务器领域，不能直接用于消费电子领域，例如手机、手表、穿戴设备等。除此之外，还有很多其它应用场景等待探索。
