# 2.基本概念术语说明
## 2.1 5G概述
&emsp;&emsp;5G是指第三代移动通信技术的统称。2018年1月，根据欧洲电信联盟（ETSI）发布的第五代移动通信技术（第五代5G）的正式命名，于2019年2月正式启用。它是将通信频率提高至约5G赫兹，扩展了整个通信系统的范围。其最大特征包括新一代的无线技术、宽带接口、高速传输能力、超高速率的计算能力和极大的数据处理能力。除此之外，还包括新的业务模式、数字经济和社会服务等方面的变化。

&emsp;&emsp;5G将影响到许多领域，其中包括无人驾驶汽车、智能家居、智慧城市、互动娱乐、医疗健康、教育科技、金融支付、农业耕地、环保环境等。

## 2.2 5G系统架构

&emsp;&emsp;5G系统架构由两层结构组成，即核心层和边缘层。核心层由若干个基站组成，这些基站分散分布在不同的地方，提供高带宽、低时延的无线连接；边缘层则为用户提供计算、存储、显示、定位等应用服务。

### 2.2.1 核心层

&emsp;&emsp;核心层的功能是建立无线连接，通过各种技术手段将终端节点连接到无线网络上，实现数据的传输、可靠性保证和高速传输。在核心层，5G基站可以提供如下服务：

 - 数据连接：包括覆盖范围扩大、无线传输容量提升和基站数量增加三个方面。覆盖范围扩大，基站数量从几十个扩大到了数千个，可以提供更多的频率带宽组合，提升无线连接的速度和性能；无线传输容量提升，从毫米波、窄带到高带宽，基站可以适应不同连接场景，提供更好的通信质量和响应时间；基站数量增加，可以满足增长的网络需求。

 - 服务质量保证：基站在覆盖范围内依托传输、压缩、调制等技术手段，保证数据可靠性和传输效率，保障应用服务的正常运行。同时，基站还需要考虑设备开销、网络条件、恶劣天气、设备故障等各种环境因素，确保服务质量。

 - 综合布置：基站需要能够承载各种类型的终端节点，为终端设备提供定制化的服务，包括数据处理、高速交换、卫星定位、视频处理、云计算、机器学习等。

 - 安全防护：由于5G是全新的无线技术，安全问题也是一个重要的研究课题。基站要充分利用业界的最新技术，加强安全防护措施，比如采用先进的身份认证方法、网关鉴权系统、集中管理的加密机制等。

### 2.2.2 边缘层

&emsp;&emsp;边缘层提供计算、存储、显示、定位等应用服务，帮助终端节点完成复杂的任务。边缘层的功能可以划分为如下几个方面：

 - 计算、存储：主要包括终端设备中的计算和存储设备，提供高性能的计算资源和持久化存储空间，为终端设备提供更丰富的功能。

 - 显示、定位：包括终端设备中的屏幕、摄像头和麦克风，用来呈现视频或图像、接收和处理来自外部世界的信息，如位置信息、语音指令等。

 - 互联互通：包括终端设备之间的互联互通，让设备之间的数据、消息可以快速、安全地传递。

 - 平台互联：包括终端设备和不同平台之间的互联互通，如手机、平板、电视、穿戴设备等。

 - 远程控制：通过互联网、Wi-Fi、蓝牙等无线技术，终端设备可以远程控制和协同其他终端设备。

 - 服务生态：包括行业内的开放生态系统，让终端设备可以连接到各类互联网服务，提升应用服务的能力和流畅度。

## 2.3 5G技术特点

&emsp;&emsp;5G技术的特点主要包括以下几个方面：

 - 增强的高频带宽：5G无线技术的关键是使用超高频率的基带信号，可以在几毫秒的时间内传输大量的数据。因此，5G将占用双频段的全部频谱资源，使得运营商能够以前所未有的速度、规模和带宽进行通信。

 - 更复杂的连接机制：5G使用了复杂的连接机制，使得终端设备能够支持多种应用场景，同时仍然保持高速连接，并可靠传输数据。

 - 复杂的应用模式：5G提供了更灵活的业务模式，使得终端设备可以针对特定任务进行优化配置。例如，应用可以根据场景、需求、状况进行自适应调度和流控，从而实现资源的最优分配和优化利用。

 - 新型个人计算设备：5G将使终端设备中的计算设备从传统的CPU和内存转变为功能更加强大的芯片阵列，实现更快的计算处理能力。

## 2.4 关键组件分析

&emsp;&emsp;5G的关键组件有多种类型，包括基站、网关、终端设备、控制单元、管理服务器、虚拟化平台等。每个组件都具有独特的功能，并且与其他组件密切相关。下面介绍一些关键组件的功能、特性和应用场景。

### 2.4.1 网关

&emsp;&emsp;网关就是5G核心组件中的一种，它的作用是建立无线连接、可靠传输数据、提供各类服务。网关通常位于边缘层，提供计算、存储、显示、定位等应用服务。在5G网络中，网关通常包含如下功能：

  - 可靠传输：网关提供高可靠的传输能力，确保数据最终可达终端设备。

  - 数据处理：网关的运算能力可以提高，通过各种处理手段将上层数据转换为下层数据，实现终端设备之间的通信。

  - 服务组合：网关通过将多个服务组合起来，提供统一的业务接口，方便终端设备的调用。

### 2.4.2 终端设备

&emsp;&emsp;终端设备是5G网络的主要用户，它主要用于承载应用，包括各种计算、存储、显示、定位、信息采集等。终端设备通常包含如下功能：

  - 应用接口：终端设备提供统一的应用接口，包括接口规范、协议栈和API等，方便网关和上层应用调用。

  - 计算、存储：终端设备中的计算和存储设备，提供高性能的计算资源和持久化存储空间，为应用提供更多的功能。

  - 用户界面：终端设备中的屏幕、摄像头和麦克风，用来呈现视频或图像、接收和处理来自外部世界的信息。

  - 通信模块：终端设备具有低功耗、高速、稳定的无线通信模块，能够快速和可靠地进行数据传输。

### 2.4.3 管理服务器

&emsp;&emsp;管理服务器是一个重要的后台服务器，用于维护和管理5G网络，并收集和分析网络状态信息。管理服务器的作用主要包括如下方面：

  - 动态管理：管理服务器能够实时地监测和管理5G网络，并能够响应网络事件，调整策略和参数，确保网络运行的稳定和安全。

  - 技术支持：管理服务器可以提供技术支持，帮助运营商解决技术问题、疑难问题。

### 2.4.4 虚拟化平台

&emsp;&emsp;虚拟化平台是5G技术的关键组件之一，它负责向边缘设备提供虚拟的计算、存储、显示、定位、互联互通等功能。虚拟化平台的功能可以划分为如下几个方面：

  - 系统管理：虚拟化平台负责虚拟化资源的管理，包括创建、删除、调整虚拟机、配置网络等。

  - 系统计费：虚拟化平台可以提供计费功能，包括按需计费和包年包月计费两种方式。

  - 服务组合：虚拟化平台可以通过各种技术手段，将多个服务组合在一起，为终端设备提供统一的业务接口。

## 3.1 5G网元设计及关键技术

&emsp;&emsp;5G的关键技术是采用毫米波和厘米波技术结合的信息传输技术，构建覆盖全球的高速基站。华为5G使用的技术主要有以下几种：

 - 频分多址技术（FDD）：主要是基于话音时分复用技术，将同样的信号通过多个角度在多个方向传播。

 - 时变频分多址技术（TDD）：主要是利用每个子帧上发送不同的时间偏移，来传输同样的数据包。

 - 大规模无线接入系统（BSAS）：是一种高容量、低成本的无线通信系统，可以提供海量的设备接入和快速的数据传输。

 - 小区内网（EMS）：通过减少小区间距离，提高数据传输的速度和可靠性。

 - 分布式频谱隔离（DSIC）：将不同频段的用户数据分别传输到多个子频道，防止不同频段数据相互干扰。

 - 中心频率网格（CN）：将整颗小区分割为若干个频道，形成共用的频谱资源池，可以避免小区内频谱碎片，提高了小区间的可用性。

 - 基站自适应调度（BSA）：根据用户的特征选择性地分配小区资源，增强用户体验，提升用户满意度。

 - 超大功率毫米波（HSS）：是一种重大突破，利用高度集中和具有极高强度的单波束的毫米波激光器，可以将户内信息传输速度提升到每秒几兆字节以上。

&emsp;&emsp;在5G的网元设计过程中，还涉及到以下关键技术：

 - 小区设计：确定小区的大小，选择合适的布局和类型。

 - 微动漫：采用微波技术来降低网络功耗。

 - 无线电调制解调器：主要用于接收和发送小区间数据的信号。

 - 矩阵编解码器：用于将原始数据转换为数字信号。

 - 微动漫传输技术：以微弱的信号传输微视频，可以有效降低系统耗电。

## 3.2 基站集中调度技术

&emsp;&emsp;集中调度是指5G基站在小区中集中部署并统一调度小区资源，从而避免频繁的小区间调度，提高了资源的利用率。

 - 专用功率控制：为了避免与用户干扰，5G基站一般采用专门的功率控制模式。

 - 资源预留策略：在小区内预留一定的带宽，避免与其他应用发生冲突。

 - 邻区检测与应急应变：当邻区出现拥塞、功率超限、业务回落等情况时，基站可以及时介入调配。

## 3.3 其他关键技术

&emsp;&emsp;除了上面提到的关键技术外，华为5G还包括以下几个关键技术：

 - AI人工智能：通过科技手段改善基站性能和增强用户体验。

 - 边缘计算：提升终端设备的计算性能。

 - 智能云：提供云计算服务，将大量计算资源和存储资源映射到云端，实现应用部署和更新自动化。

 - 移动边缘计算：利用弱网条件下的资源调配技术和方法，提升终端的使用体验。

 - 人脸识别：通过人脸识别技术，可以精准识别用户身份，提高用户满意度。

## 3.4 常见应用场景

&emsp;&emsp;5G的应用场景主要分为以下几类：

 - 网络视频传输：视频会议、短视频、直播、视频云等。

 - 地理位置服务：提供智能巡检、轨迹跟踪、地图导航、导航提示等功能。

 - 物联网服务：涵盖智能电表、智能照明、智能交通、智能厨房等场景。

 - 智能家庭：通过安防监控、智能视频装饰、智能家居助手、智能监控等功能。

 - 智能出行：通过智能网约车、出租车、地铁、共享单车等功能。

 - 企业级大数据服务：提供高性能、大规模数据处理、机器学习能力，为商业提供支持。
