# 1.简介
  

Facebook为其用户提供了一个免费的“广告词”计划工具。该计划工具可以帮助用户找到品牌关键词、形成短期、中期、长期关键词竞争力等相关数据。本文将从以下几个方面详细介绍该工具的功能及使用方法：

1. 如何创建账户并登录？
2. 如何设置关键词？
3. 关键词优化过程中的注意事项？
4. 为什么要关注展示量与点击率？
5. 使用该工具时需要注意哪些细节？

整个文章共分为两部分，第一部分主要介绍“如何创建账户并登录”，第二部分主要介绍“关键词优化过程中的注意事项”。

# 3.如何设置关键词？
完成登录之后，就要开始设置关键词了。进入关键字设置页面，如下图所示：
这个页面上，左边有一个“Add New Keyword”按钮，可以直接添加新的关键词；右边则是已经添加的关键词列表，每个关键词都有对应的说明、选取的查询量、预算等参数，这里需要先选取关键词，然后再设置其查询量和预算。
首先，选择一个适合你的广告形式的类型，比如信息流、社交媒体、移动应用等。然后，根据自身的需求和对不同词之间的竞争力的理解，选择一些比较相近的词，这些词会影响到你的排名结果。最后，设置一个不错的查询量和预算，比如每天或每周的点击次数、花费的金额等。
至于关于关键词的数量，也是需要考虑的因素之一。一般来说，选择几十个关键词就足够了，但如果想要提高排名效率的话，建议添加更多的相关性较低的关键词，这样可以提高被竞争对手抢走的概率。
设置好关键词之后，就可以点击“Generate Results”按钮生成结果了。点击“Download Report”按钮，就可以下载当前设置下的关键词报告。
# 4.关键词优化过程中的注意事项？
通过设置好的关键词，Facebook Adwords 关键字生成器就可以生成一份报告，其中包括搜索量、排名、CPC预算、广告效果指标等信息。但是，这份报告是否能带给我们改进呢？这正是关键词优化过程中需要注意的地方。
首先，如何确定最优的关键词组合？不同的关键词组合可能会产生不同的排名结果，但它们的性能指标都是基于相同的优化目标——增加搜索量、提升排名、降低CPC。所以，关键词的组合和相关性也是需要结合实际情况去判断的。
其次，如何调整关键词的权重？Facebook Adwords提供了一套强大的规则引擎，可以自动调整关键词的权重，使得优化目标更加精准。举个例子，如果有一个关键词的性能表现比另一个关键词好很多，那么它的权重就会更高，反之亦然。因此，只需仔细查看生成的关键词报告，找出那些权重有待提高的关键词，再相应地调整他们的权重。
第三，优化目标的范围应该多宽？关键词优化最终的目的是提升广告转化率，但它仅仅是一个起点。为了达到最佳效果，还需要考虑其他相关领域的优化措施。比如，在线销售渠道的优化、社交网络营销的优化、内容营销的优化等。因此，关键词优化只是局部的，全局的优化才是关键。
第四，如何设置预算？衡量一个广告主的预算不是件容易的事情。因此，还是建议以自己认为合理的预算为基准，然后尝试各种实验来验证预算的有效性。同时，也不要盲目追求完美的预算，而应尽可能地设定一个大致的预算，然后根据实验的结果进行微调。
最后，关键词优化结果的有效性如何评估？最终的结果往往是反映了预算分配、关键词组合以及关键词权重的综合效果。因此，需要在实际操作中进行迭代，逐步提升关键词的整体效果。
# 5.为什么要关注展示量与点击率？
阅读过上述的内容，有没有想到这样一个问题：为何要关注展示量与点击率？这一切背后的原因又是什么？接下来，我将通过两个小例子，为大家提供一些个人感觉。
第一个例子：房子投资者的生活经历。假设你是一个房产投资者，正在寻找一个适合自己的房子。那么，你如何才能做出决定呢？通常来说，这里有几个指标可以参考：价格、位置、建筑风格、外墙装修程度、内饰设计、配套设施、开放空间等。每一个因素都可以影响最终的购买决定。
比如，你希望能够获得好的物业环境、邻居比较多、朝南的房子、房屋干净整洁、楼层高等等，那么这些因素就很重要。你可以选择一套学校区的住宅作为入口，然后通过研究其周围的环境、同学们的生活、学校的历史发展等因素，找出一个价格适宜的房子。
第二个例子：SEOer的工作压力。假设你是一个SEOer，正在为某个网站进行优化。那么，你如何才能找到网站上的关键词呢？关键词就是影响搜索结果排序的一个重要因素。它的作用包括让网页能够呈现给用户足够多的重要信息，提升其知名度，帮助网站吸引新用户等。因此，关键词优化也是SEO的一部分。
如果你是一名SEOer，那么你肯定需要知道哪些技巧、策略以及工具，才能更好地提升网站的关键词排名。阅读完以上两个小例子，你或许会有新的启发，或许会发现新的问题，或许会重新认识到“如何利用自己的能力、资源、知识为他人创造价值”的重要性。
