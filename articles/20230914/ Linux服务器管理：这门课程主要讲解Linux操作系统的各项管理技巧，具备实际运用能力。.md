# 4.为什么要选取Linux操作系统？

Linux操作系统是一个开放源码的类UNIX操作系统，它的优点很多，包括：

1. 大量免费的软件可供下载，可以满足广大网民的各种需求。
2. 基于社区支持，可靠可信赖。
3. 良好的性能表现，尤其是在服务器领域，稳定性、可靠性是最重要的。
4. 内核高度模块化设计，各个模块之间松耦合。
5. 系统安全性高，无需担心黑客攻击。
6. 支持多种编程语言，可以运行各类应用程序。

虽然Linux已经成为最流行的操作系统，但它的市场份额却相对来说小得多。国内外很多知名互联网公司、大型科技企业均使用了Linux操作系统，比如阿里巴巴、腾讯、百度、京东、网易等等。
