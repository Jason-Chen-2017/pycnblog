# 2.基础语法
## 2.1 Hello World!程序
首先，让我们来看一个最简单的程序——Hello World！程序。

```python
print("Hello World!")
```

这是一段打印“Hello World!”到屏幕上的简单程序。

### 运行方式
要运行这段程序，只需在Python环境中打开命令行窗口（Windows下按Win+R键调出运行框输入cmd回车），然后将以上代码复制粘贴到命令行窗口，按下Enter键即可运行。如果成功运行，屏幕上会显示Hello World!字样。

### 注释
一般情况下，程序中的每一行都应该添加注释，方便别人理解和修改代码。在Python中，单行注释以#开头；多行注释可以使用三个双引号"""或单引号'''括起来的区域表示。例如：

```python
# This is a comment line
'''
This is a multi-line comment block.
It can span multiple lines and can be nested.
'''
```

## 2.2 变量与赋值
接下来，我们来看一下Python语言中的变量和赋值。

### 创建变量
在Python中，可以使用关键词`var = value`创建一个新的变量`var`，并将值`value`赋给它。

示例如下：

```python
name = "John"    # 变量"name"存放着字符串"John"
age = 30         # 变量"age"存放着整数30
is_student = True   # 变量"is_student"存放着布尔值True
```

### 修改变量的值
可以通过重新给变量赋值的方式改变变量的值。

```python
num = 10      # 变量"num"存放着数字10
num += 5      # num现在等于15
num -= 3      # num现在等于12
num *= 2      # num现在等于24
```

注意：当试图修改变量的类型时（如将整数变成字符串），Python会报错。因此，需要格外小心。

### 在同一行定义多个变量
在Python中，可以在同一行中定义多个变量，但需要按照标准格式：

```python
a, b = 10, 'hello'     # 此处定义了两个变量"a"和"b"，分别存放着数字10和字符串"hello"
x, y, z = [1, 2, 3]    # 此处定义了三个变量"x", "y", 和 "z"，它们分别存放着数字1、2和3
```

如果试图在相同的赋值语句中定义不同类型的变量，则会报错。

```python
# 错误示范: 在同一行同时定义了数字和字符串类型的变量
num1 = age = str(25)        # 会报错，因为age已经是一个变量名了
```

正确的方式是在不同的赋值语句中定义不同类型的变量。

```python
# 正确示范: 分别定义数字和字符串类型的变量
num1 = 10                  # 变量"num1"存放着数字10
age = str(25)              # 变量"age"存放着字符串"25"
```

### 类型转换
除了显式地将变量转换为指定的数据类型之外，还可以通过类型转换函数来完成。这些函数通常以`int()`、`float()`、`str()`等形式出现，它们能够将某种类型的数据转换成另一种类型的数据。

```python
num = 10           # num存放着数字10
text = str(num)    # 将num转换为字符串后存放在变量"text"中
number = float(text)  # 将字符串"text"转换为浮点数存放在变量"number"中
```

注：不要使用过于灵活的类型转换，可能会导致程序行为不明确。

## 2.3 基本运算符
有了变量之后，就可以对其进行各种运算了。在Python中，主要的运算符有：

### 算术运算符

|运算符|描述|实例|
|-|-|-|
|+|加法|`x + y`|
|-|减法|`x - y`|
|\*|乘法|`x * y`|
|/|除法|`x / y`|
|%|取模（返回余数）|`x % y`|
|**|指数（求幂）|`x ** y`|

### 比较运算符

|运算符|描述|实例|
|-|-|-|
|==|等于|`x == y`|
|!=|不等于|`x!= y`|
|>|<|`x > y`|
|<|>||`x < y`|
|>=|<=|`x >= y`|
|<=|>=|`x <= y`|

### 逻辑运算符

|运算符|描述|实例|
|-|-|-|
|and|与（仅当两个条件均满足时，才返回True）|`x and y`|
|or|或（只要其中有一个条件满足，就返回True）|`x or y`|
|not|非（对条件取反）|`not x`|

### 赋值运算符

|运算符|描述|实例|
|-|-|-|
|=|简单赋值|`x = 5`|
|+=|增量赋值|`x += 3`|
|-=|减量赋值|`x -= 3`|
|\*=|乘量赋值|`x *= 3`|
|/=|除量赋值|`x /= 3`|
|%=|模量赋值|`x %= 3`|
|**=|指数量赋值|`x **= 3`|
