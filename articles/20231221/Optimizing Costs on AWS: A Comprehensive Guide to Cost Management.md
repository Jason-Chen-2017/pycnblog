# 4.具体代码实例和详细解释说明

在 AWS 上优化成本，我们可以使用以下代码实例和详细解释说明：

1. **实例类型**：

```python
import boto3

ec2 = boto3.resource('ec2')

instances = ec2.instances.filter(
    Filters=[
        {
            'Name': 'instance-type',
            'Values': ['t2.micro', 't2.small', 't2.medium', 't2.large']
        }
    ]
)

for instance in instances:
    print(instance.id, instance.instance_type)
```

2. **存储类型**：

```python
import boto3

s3 = boto3.resource('s3')

buckets = s3.buckets.all()

for bucket in buckets:
    for obj in bucket.objects.all():
        print(obj.key, obj.storage_class)
```

3. **数据库类型**：

```python
import boto3

rds = boto3.resource('rds')

db_instances = rds.db_instances.all()

for db_instance in db_instances:
    print(db_instance.db_instance_class)
```

4. **网络类型**：

```python
import boto3

ec2 = boto3.resource('ec2')

vpcs = ec2.vpcs.all()

for vpc in vpcs:
    print(vpc.id, vpc.cidr_block)
```

5. **计费模式**：

```python
import boto3

aws_billing = boto3.client('aws_billing')

billing_summary = aws_billing.get_cost_and_usage(
    TimePeriod='P1',
    Currency='USD'
)

for item in billing_summary['ResultsByService']['ResultsByInstance']:
    print(item['ServiceCode'], item['InstanceId'], item['AverageCost'])
```
