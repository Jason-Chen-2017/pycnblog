## 4.1 家具搬移算法实现

```python
import numpy as np

class FurnitureMoving:
    def __init__(self, weight, force):
        self.weight = weight
        self.force = force

    def moving(self, distance):
        acceleration = self.force / self.weight
        time = distance / acceleration
        return time

furniture = FurnitureMoving(100, 1000)
time = furniture.moving(10)
print("家具搬移所需的时间为：", time)
```

详细解释说明：

1. 首先，导入 numpy 库，用于数学计算。

5. 创建一个 `FurnitureMoving` 对象，表示一个重量为 100 公斤，应用于搬移的力为 1000 牛顿的家具。
6. 调用 `moving` 方法，计算家具搬移所需的时间，并打印结果。

    def __init__(self, current, voltage):
