## 4.2 构建购买行为概率矩阵

根据收集到的客户购买行为数据，我们可以构建购买行为概率矩阵。以下是一个简单的Python代码实例：

```python
import numpy as np

# 收集客户购买行为数据
data = [
    ['A', 'B', 'C'],
    ['B', 'C', 'D'],
    ['C', 'D', 'E'],
    ['D', 'E', 'F'],
    ['E', 'F', 'G'],
    ['F', 'G', 'H'],
]

# 构建购买行为概率矩阵
prob_matrix = np.zeros((len(data), len(data[0])))
for i, row in enumerate(data):
    for j, item in enumerate(row):
        prob_matrix[i, j] = item

print(prob_matrix)
```
