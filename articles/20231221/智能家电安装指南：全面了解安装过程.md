# 4.具体代码实例和详细解释说明
在这里，我们将以一个简单的智能家电示例进行说明。假设我们需要编写一个智能灯泡控制程序，以便根据用户的使用习惯自动调整亮度和颜色。以下是一个简单的Python代码实例：

```python
import numpy as np
from sklearn.linear_model import LogisticRegression

# 加载数据
data = np.loadtxt('smart_light_data.txt', delimiter=',')
X = data[:, :-1]  # 输入变量
y = data[:, -1]   # 目标变量

# 训练模型
model = LogisticRegression()
model.fit(X, y)

# 预测
def predict(input_data):
    input_data = np.array(input_data).reshape(1, -1)
    return model.predict(input_data)

# 使用示例
input_data = [23, 72, 45]  # 温度、湿度、亮度
predicted_color = predict(input_data)
print('预测的颜色：', predicted_color)
```

在这个示例中，我们首先加载了智能灯泡的训练数据，并将其分为输入变量和目标变量。然后，我们使用LogisticRegression模型进行训练。在预测阶段，我们定义了一个`predict`函数，用于根据输入数据进行预测。最后，我们使用了一个示例输入数据，并根据预测结果得出了预测的颜色。
