以下是一个量子随机 walk 的代码实例：

```python
import numpy as np
import qiskit

# 初始化一个量子状态
qc = qiskit.QuantumCircuit(2)
qc.h(0)

# 选择一个邻居状态
np.random.seed(0)
neighbor = np.random.randint(0, 2)

# 将量子状态移动到邻居状态
if neighbor == 0:
    qc.cx(0, 1)
else:
    qc.x(1)

# 绘制量子电路
qiskit.Aer.get_backend('qasm_simulator').run(qc).result().get_counts()
```
