## 4.1 代码实例

以下是一个使用Apache Flink实现实时营销分析的代码实例：

```python
from flink import StreamExecutionEnvironment

env = StreamExecutionEnvironment.get_execution_environment()

# 定义数据源
data_source = env.from_elements([('user1', 1), ('user2', 1), ('user3', 1)])

# 定义数据处理函数
def process_data(value):
    return value[1] + 1

# 将数据处理函数应用于数据源
result = data_source.map(process_data)

# 将结果输出到控制台
result.print()

env.execute("real-time_marketing_analysis")
```
