# 2.核心概念与联系

## 2.1 量子比特（Qubit）

量子比特（qubit）是量子计算机中的基本单元，它是传统比特（bit）的量子版本。量子比特可以存储0和1的信息，但与传统比特不同的是，量子比特可以同时存储0和1，这使得量子计算机具有超越传统计算机的计算能力。

量子比特可以用纯态和混合态来描述。纯态量子比特可以表示为一个向量，该向量的长度为2，表示量子比特可以存储0和1的信息。混合态量子比特则是纯态量子比特的概率组合，表示量子比特处于多种不同状态的情况。

## 2.2 纠缠

纠缠是量子计算机中的一个重要概念，它是量子比特之间的一种相互作用。纠缠可以用于实现量子计算机中的并行计算，这使得量子计算机能够解决一些传统计算机无法解决的问题。

纠缠可以通过量子门实现，如肖尔迪门和托尼门。肖尔迪门可以用于实现两个量子比特之间的纠缠，而托尼门可以用于实现三个或多个量子比特之间的纠缠。

## 2.3 叠加态

叠加态是量子计算机中的另一个重要概念，它是量子比特处于多种不同状态的情况。叠加态可以用于实现量子计算机中的并行计算，这使得量子计算机能够解决一些传统计算机无法解决的问题。

叠加态可以通过量子门实现，如叠加门和门门。叠加门可以用于实现量子比特处于0和1状态的叠加，而门门可以用于实现量子比特处于多种不同状态的叠加。
