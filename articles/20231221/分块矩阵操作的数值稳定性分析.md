# 2.核心概念与联系

在进行分块矩阵操作之前，我们需要了解一些基本概念。首先，我们需要了解什么是矩阵，以及如何将矩阵分块。

## 2.1 矩阵

矩阵是一种二维数组，由行和列组成。一个矩阵可以表示为 $A = (a_{ij})_{m \times n}$，其中 $a_{ij}$ 是矩阵的元素，$m$ 和 $n$ 是矩阵的行数和列数。

## 2.2 分块矩阵

分块矩阵是将矩阵划分为多个子矩阵的矩阵。一个分块矩阵可以表示为 $A = \begin{pmatrix} A_{11} & A_{12} \\ A_{21} & A_{22} \end{pmatrix}$，其中 $A_{ij}$ 是子矩阵，$i,j = 1,2$。

矩阵加法和乘法是分块矩阵操作中最基本的操作。对于矩阵加法，我们可以将两个矩阵相加，得到一个新的矩阵。对于矩阵乘法，我们需要将两个矩阵的行列相乘，得到一个新的矩阵。

### 3.1.1 矩阵加法

对于两个矩阵 $A = (a_{ij})_{m \times n}$ 和 $B = (b_{ij})_{m \times n}$，它们的和可以表示为 $C = A + B = (c_{ij})_{m \times n}$，其中 $c_{ij} = a_{ij} + b_{ij}$。

### 3.1.2 矩阵乘法

对于两个矩阵 $A = (a_{ij})_{m \times n}$ 和 $B = (b_{ij})_{n \times p}$，它们的乘积可以表示为 $C = AB = (c_{ij})_{m \times p}$，其中 $c_{ij} = \sum_{k=1}^{n} a_{ik}b_{kj}$。
