## 4.3 图像分类代码示例

以下是一个简单的图像分类代码示例，它使用了支持向量机（SVM）算法：

```python
from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# 加载数据集
iris = datasets.load_iris()
X = iris.data
y = iris.target

# 数据预处理
scaler = StandardScaler()
X = scaler.fit_transform(X)

# 训练测试数据集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 训练SVM模型
svm = SVC(kernel='linear')
svm.fit(X_train, y_train)

# 预测
y_pred = svm.predict(X_test)

# 评估模型
accuracy = accuracy_score(y_test, y_pred)
print('Accuracy:', accuracy)
```

在这个代码示例中，我们首先使用`sklearn.datasets.load_iris()`函数加载了一个名为`iris`的数据集。然后，我们使用`sklearn.model_selection.train_test_split()`函数将数据集划分为训练集和测试集。接着，我们使用`sklearn.preprocessing.StandardScaler()`函数对数据集进行标准化处理。最后，我们使用`sklearn.svm.SVC()`函数创建了一个SVM模型，并使用`fit()`方法对训练集进行训练。在训练完成后，我们使用`predict()`方法对测试集进行预测，并使用`accuracy_score()`函数计算模型的准确率。
