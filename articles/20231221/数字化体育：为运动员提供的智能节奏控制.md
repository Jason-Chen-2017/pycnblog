## 4.1 运动数据采集代码实例

在这个代码实例中，我们使用 Python 编程语言来实现运动数据采集。我们使用 Pygame 库来收集加速度计、磁场传感器、陀螺仪等运动数据。

```python
import pygame
import time

# 初始化 Pygame
pygame.init()

# 创建一个窗口
window = pygame.display.set_mode((400, 300))

# 创建一个表示加速度计的文本框
accelerometer = pygame.text.Text(font=pygame.font.Font(None, 30), text="")

# 创建一个表示磁场传感器的文本框
magnetometer = pygame.text.Text(font=pygame.font.Font(None, 30), text="")

# 创建一个表示陀螺仪的文本框
gyroscope = pygame.text.Text(font=pygame.font.Font(None, 30), text="")

# 主循环
while True:
    # 处理事件
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            pygame.quit()

    # 更新文本框的内容
    accelerometer.update(f"Accelerometer: {time.time()}")
    magnetometer.update(f"Magnetometer: {time.time()}")
    gyroscope.update(f"Gyroscope: {time.time()}")

    # 绘制文本框
    window.fill((255, 255, 255))
    window.blit(accelerometer.render(), (10, 10))
    window.blit(magnetometer.render(), (10, 50))
    window.blit(gyroscope.render(), (10, 90))

    # 更新窗口
    pygame.display.flip()

    # 延时
    time.sleep(0.1)
```
