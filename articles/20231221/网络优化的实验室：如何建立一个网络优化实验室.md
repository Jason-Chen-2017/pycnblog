### 4.1.1 token bucket 算法实例

token bucket 算法是一种流量控制算法，用于限制发送数据的速率。它的原理是将发送数据的速率限制在一个固定的速率内，通过一个令牌桶来实现。令牌桶中的令牌表示可以发送的数据量，每隔一段时间从令牌桶中取出一个令牌，然后发送一包数据。如果令牌桶中没有令牌，则需要等待，直到令牌桶中有令牌再发送数据。

```python
import time

class TokenBucket:
    def __init__(self, rate, bucket_size):
        self.rate = rate
        self.bucket_size = bucket_size
        self.tokens = bucket_size
        self.last_refill_time = time.time()

    def refill(self):
        current_time = time.time()
        elapsed_time = current_time - self.last_refill_time
        self.tokens = min(self.tokens + elapsed_time * self.rate, self.bucket_size)
        self.last_refill_time = current_time

    def send_packet(self):
        if self.tokens > 0:
            self.tokens -= 1
            return True
        else:
            return False

# 使用 token bucket 算法实例
bucket = TokenBucket(rate=10, bucket_size=100)
for i in range(100):
    if bucket.send_packet():
        print("发送数据成功")
    else:
        print("发送数据失败，需要等待")
```

### 4.1.2 leaky bucket 算法实例

leaky bucket 算法是一种流量控制算法，用于限制发送数据的速率。它的原理是将发送数据的速率限制在一个固定的速率内，通过一个漏斗来实现。漏斗中的水表示可以发送的数据量，每隔一段时间从漏斗中流出一定量的水，然后发送一包数据。如果漏斗中没有水，则需要等待，直到漏斗中有水再发送数据。

```python
import time

class LeakyBucket:
    def __init__(self, rate, bucket_size):
        self.rate = rate
        self.bucket_size = bucket_size
        self.water = bucket_size
        self.last_refill_time = time.time()

    def refill(self):
        current_time = time.time()
        elapsed_time = current_time - self.last_refill_time
        self.water = min(self.water + elapsed_time * self.rate, self.bucket_size)
        self.last_refill_time = current_time

    def send_packet(self):
        if self.water > 0:
            self.water -= 1
            return True
        else:
            return False

# 使用 leaky bucket 算法实例
bucket = LeakyBucket(rate=10, bucket_size=100)
for i in range(100):
    if bucket.send_packet():
        print("发送数据成功")
    else:
        print("发送数据失败，需要等待")
```
