                 

# 1.背景介绍

曼哈顿距离，也被称为人工距离或Taxicab geometry，是一种在二维平面上的距离度量方法。它从一个点到另一个点的距离是垂直和水平方向上的距离之和。在许多实际问题中，曼哈顿距离是一个很有用的度量标准。例如，在地图导航中，曼哈顿距离可以用来计算两个地点之间的步行距离；在数据库中，曼哈顿距离可以用来计算两个数据记录之间的距离，以实现数据聚类和搜索优化；在物流和运输中，曼哈顿距离可以用来计算货物运输的成本。
