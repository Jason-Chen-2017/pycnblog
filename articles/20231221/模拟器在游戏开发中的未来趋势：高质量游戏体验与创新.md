```python
class GameSimulator:
    def __init__(self, game_state):
        self.game_state = game_state

    def step(self, action):
        next_state = self.game_state.apply_action(action)
        reward = self.game_state.reward(action)
        done = not self.game_state.is_episode_finished()
        return next_state, reward, done
```

这个简单的模拟器代码实例定义了一个`GameSimulator`类，该类包括一个构造函数`__init__`和一个`step`方法。构造函数用于初始化模拟器，并传入游戏的当前状态。`step`方法用于模拟游戏的下一步行为，包括执行一个动作，获取该动作的奖励和是否结束游戏的信息。
