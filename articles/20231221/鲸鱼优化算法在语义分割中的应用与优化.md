语义分割是计算机视觉领域的一个重要任务，它涉及到将图像中的各个像素点分配到不同的类别，以表示这些像素所代表的物体、场景或其他信息。语义分割的应用非常广泛，包括自动驾驶、医疗诊断、地图生成等等。

传统的语义分割方法主要包括基于手工特征的方法和基于深度学习的方法。基于手工特征的方法需要人工提取图像中的特征，如边缘、纹理等，然后根据这些特征进行分类。这种方法的主要缺点是需要大量的人工工作，并且对于复杂的图像场景，手工提取的特征可能不够准确。

基于深度学习的方法则是利用深度学习模型自动学习图像中的特征，并根据这些特征进行分类。这种方法的优点是不需要人工工作，并且可以处理复杂的图像场景。但是，深度学习模型的训练过程是一种迭代过程，需要大量的计算资源和时间。

为了解决这些问题，近年来研究者们开始关注优化算法在语义分割中的应用。优化算法可以帮助我们更有效地训练深度学习模型，从而提高模型的性能。其中，鲸鱼优化算法是一种非常有效的优化算法，它在语义分割任务中取得了显著的成果。

```python
import cv2
import numpy as np

def load_data(data_dir):
    images = []
    labels = []
    for filename in os.listdir(data_dir):
        img = cv2.imread(os.path.join(data_dir, filename))
        img = cv2.resize(img, (224, 224))
        img = np.expand_dims(img, axis=0)
        img = img / 255.0
        images.append(img)
        label = np.zeros((224, 224))

        labels.append(label)
    return images, labels
```
