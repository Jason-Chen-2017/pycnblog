# 4.1 生产线自动化
在这个例子中，我们将使用Python编程语言和ROS（Robot Operating System）库来实现生产线自动化。

首先，我们需要安装ROS库：

```bash
$ sudo apt-get install ros-melodic-python
$ sudo apt-get install ros-melodic-rospy
$ sudo apt-get install ros-melodic-geometry-msgs
```

接下来，我们创建一个名为`production_line.py`的Python文件，并编写以下代码：

```python
#!/usr/bin/env python

import rospy
from geometry_msgs.msg import Twist

class ProductionLine:
    def __init__(self):
        rospy.init_node('production_line', anonymous=True)
        self.pub = rospy.Publisher('/turtle1/cmd_vel', Twist, queue_size=10)
        self.rate = rospy.Rate(10)

    def move(self):
        while not rospy.is_shutdown():
            twist = Twist()
            twist.linear.x = 0.5
            twist.angular.z = 0
            self.pub.publish(twist)
            self.rate.sleep()

if __name__ == '__main__':
    try:
        line = ProductionLine()
        line.move()
    except rospy.ROSInterruptException:
        pass
```

这个代码定义了一个名为`ProductionLine`的类，它继承自ROS的`Node`类。在`__init__`方法中，我们初始化了ROS节点并创建了一个发布者对象`pub`，用于发布控制命令。在`move`方法中，我们发布一个前进速度为0.5的`Twist`消息，使机器人前进。

为了运行这个程序，我们需要在终端中运行以下命令：

```bash
$ rosrun melodic production_line.py
```
