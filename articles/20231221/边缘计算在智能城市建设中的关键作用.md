                 

# 1.背景介绍

智能城市是指通过大数据、人工智能、物联网等技术手段，将城市发展为一种更加智能化、可持续的生活环境。智能城市的建设需要涉及到多个领域的技术，包括通信技术、计算技术、感知技术、存储技术等。边缘计算是一种计算技术，它将计算能力推向设备边缘，使得数据处理能力更加分布式，实时性更加强。在智能城市建设中，边缘计算的关键作用有以下几点：

1.1 提高计算能力和实时性

在智能城市中，数据来源于各种设备，如摄像头、传感器、车载设备等。这些设备产生的数据量巨大，传输到中心服务器进行处理的话，会导致网络负载过大，计算能力不足，同时实时性也会受到影响。边缘计算可以将计算能力推向设备边缘，让数据在设备本身或者 nearby 的边缘设备进行处理，从而提高计算能力和实时性。

1.2 降低网络延迟和带宽需求

在传统的中心化计算模型中，数据需要通过网络传输到中心服务器进行处理，这会导致网络延迟和带宽需求很高。边缘计算可以将计算能力推向设备边缘，让数据在设备本身或者 nearby 的边缘设备进行处理，从而降低网络延迟和带宽需求。

1.3 保护隐私和安全

在智能城市中，个人隐私和数据安全是一个重要问题。边缘计算可以将数据处理能力推向设备边缘，让数据在设备本身或者 nearby 的边缘设备进行处理，从而减少数据传输，降低泄露隐私的风险。

1.4 支持实时应用

在智能城市中，许多应用需要实时性很高，如智能交通、智能能源等。边缘计算可以让数据在设备本身或者 nearby 的边缘设备进行处理，从而实现实时应用。

1.5 降低成本

边缘计算可以将计算能力推向设备边缘，降低中心服务器的负担，从而降低成本。
