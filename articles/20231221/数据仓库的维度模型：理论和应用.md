# 2.核心概念与联系
维度模型是一种用于描述数据仓库中数据结构的方法。它将数据分为两个部分：维度和度量。维度是用于描述数据的属性，度量是用于描述数据的值。维度模型可以帮助我们更好地理解和分析数据，因为它可以将复杂的数据关系简化为更简单的形式。

维度模型的核心概念包括：

1. 维度（Dimension）：维度是数据仓库中数据的属性，用于描述数据的特征。维度可以是任何东西，例如时间、地理位置、产品、客户等。

2. 度量（Measure）：度量是数据仓库中数据的值，用于描述数据的数值。度量可以是任何东西，例如销售额、利润、客户数量等。

3. 维度层次（Hierarchy）：维度层次是维度中的一种结构，它可以将维度分为多个级别。例如，产品可以分为品类、品牌、产品，时间可以分为年、季度、月、日等。

4. 维度模型（Star Schema）：维度模型是数据仓库中的一种结构，它将维度和度量组合在一起，形成一个星形图案。

维度模型与其他数据模型的联系包括：

1. 关系模型：关系模型是数据库中的一种结构，它将数据分为多个表，通过关系来描述数据之间的关系。维度模型与关系模型的区别在于，维度模型将数据分为两个部分：维度和度量，而关系模型将数据分为多个表。

2. 雪花模型：雪花模型是数据仓库中的一种结构，它将维度层次扩展到多个维度之间。维度模型与雪花模型的区别在于，维度模型将维度层次组合在一起，形成一个星形图案，而雪花模型将维度层次扩展到多个维度之间。
