# 2. 核心概念与联系
## 2.1 矩阵基本概念
矩阵是由行和列组成的数字元素的二维数组。矩阵可以进行加法、减法、数乘等基本运算，还可以与向量进行乘法运算。矩阵在线性代数中扮演着重要的角色，用于表示线性方程组、线性变换等。

## 2.2 特征值与特征向量的定义
给定一个方阵A，如果存在一个非零向量x，使得Ax = λx（其中λ是一个数值），则称λ为矩阵A的特征值，向量x为特征向量。特征值和特征向量的关系是有密切联系的，它们可以描述矩阵的性质和特点。

# 3. 核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 求特征值的方法
### 3.1.1 特征值定义
设A是n×n的方阵，如果存在一个n维向量x，使得Ax = λx，则λ是矩阵A的特征值。

### 3.1.2 特征值的计算方法
1. 求A的特征值，可以使用特征值定理（也称为特征值方程）：
$$
\text{det}(A - λI) = 0
$$
其中，det表示行列式，I是单位矩阵。

2. 求解特征值方程得到的多项式方程组的根，即可得到矩阵A的特征值。

## 3.2 求特征向量的方法
### 3.2.1 特征向量定义
设A是n×n的方阵，λ是A的特征值，如果存在一个n维向量x，使得Ax = λx，则向量x是矩阵A的特征向量。

### 3.2.2 特征向量的计算方法
1. 给定一个特征值λ，可以得到一组线性方程组：
$$
(A - λI)x = 0
$$
2. 求解这组线性方程组的解，即可得到矩阵A的特征向量。

# 4. 具体代码实例和详细解释说明
在本节中，我们将通过一个具体的例子来说明如何计算矩阵的特征值和特征向量。

## 4.1 代码实例
```python
import numpy as np

A = np.array([[4, -2], [-2, 4]])

# 计算特征值
def calc_eigenvalues(A):
    eigenvalues, eigenvectors = np.linalg.eig(A)
    return eigenvalues, eigenvectors

eigenvalues, eigenvectors = calc_eigenvalues(A)

# 输出结果
print("特征值:", eigenvalues)
print("特征向量:", eigenvectors)
```
## 4.2 解释说明
1. 首先，我们使用numpy库中的`linalg.eig()`函数计算矩阵A的特征值和特征向量。
2. `linalg.eig()`函数内部会调用特征值定理和特征向量计算方法，得到矩阵A的特征值和特征向量。
3. 最后，我们将计算结果打印输出。
