# 4.具体代码实例和详细解释说明

## 4.1 数据清洗的代码实例

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.dropna()  # 删除缺失值
data = data[data['age'] > 0]  # 删除年龄小于等于0的记录
data = data.replace(to_replace='', value=None)  # 将空字符串替换为None
data = data.replace(to_replace='unknown', value=None)  # 将unknown替换为None
```

## 4.2 数据校验的代码实例

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 数据校验
def check_age(x):
    if x < 0 or x > 120:
        return False
    return True

data['age'].apply(check_age)
```

## 4.3 数据纠正的代码实例

```python

```
