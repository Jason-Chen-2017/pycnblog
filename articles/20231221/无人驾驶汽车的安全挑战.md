```python
    # 获取摄像头帧

    # 显示摄像头帧

    # 等待键盘输入

    # 如果按下'q'键，则退出循环

        break

# 关闭摄像头窗口
cv2.destroyAllWindows()
```

```python
import cv2

# 使用Sobel滤波器对灰度图进行边缘检测
sobel_x = cv2.Sobel(processed_gray, cv2.CV_64F, 1, 0, ksize=5)
```

在这个代码实例中，我们使用OpenCV库的Sobel函数来对灰度图进行边缘检测。同时，我们使用np.sqrt函数来计算边缘梯度，并使用cv2.Canny函数来对边缘梯度进行处理。最后，我们使用cv2.imshow函数来显示边缘检测结果，并使用cv2.waitKey函数来等待键盘输入。如果按下'q'键，则退出循环。最后，我们使用cv2.destroyAllWindows函数来关闭摄像头窗口。
