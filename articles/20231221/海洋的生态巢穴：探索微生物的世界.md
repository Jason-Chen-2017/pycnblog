```python
import numpy as np
import pandas as pd
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

# 加载数据
data = pd.read_csv('microbe_data.csv')

# 数据预处理
scaler = StandardScaler()
data_scaled = scaler.fit_transform(data)

# 使用KMeans算法进行聚类
kmeans = KMeans(n_clusters=3)
kmeans.fit(data_scaled)

# 输出聚类结果
print(kmeans.labels_)
```

在这个例子中，我们首先使用pandas库加载了微生物数据，然后使用StandardScaler库对数据进行归一化处理。接着，我们使用KMeans算法对微生物进行聚类，并输出了聚类结果。这个简单的例子展示了如何使用Python和Scikit-learn库对海洋微生物数据进行分析。
