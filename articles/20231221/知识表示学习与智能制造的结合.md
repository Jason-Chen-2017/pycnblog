# 2.核心概念与联系

## 2.1 知识表示学习（Knowledge Representation Learning，KRL）

知识表示学习是一种通过学习自动构建知识表示的方法，它的主要目标是从数据中学习出知识，并将其表示成计算机可以理解和处理的形式。知识表示学习的主要任务包括：

- 概念学习：学习出概念的定义，如人、动物、植物等。
- 关系学习：学习出关系的定义，如父母、子女、同伴等。
- 规则学习：学习出规则的定义，如如果A则B，否则C。

知识表示学习的主要方法包括：

- 规则学习：通过学习规则来描述事物之间的关系。
- 案例学习：通过学习案例来描述事物之间的关系。
- 图结构学习：通过学习图结构来描述事物之间的关系。

## 2.2 智能制造（Intelligent Manufacturing）

智能制造是一种通过运用人工智能、自动化、网络化、数字化等技术，以提高生产效率、降低成本、提高产品质量的制造方法。智能制造的核心是通过大数据、人工智能等技术，实现智能化的生产流程和制造过程。智能制造的主要特点包括：

- 自主化：制造过程具有自主性，可以自主地完成任务。
- 智能化：制造过程具有智能性，可以智能地处理问题。
- 网络化：制造过程具有网络性，可以通过网络进行协同工作。
- 数字化：制造过程具有数字性，可以通过数字技术进行优化。
