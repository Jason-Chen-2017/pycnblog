# 4.具体代码实例和详细解释说明
# 4.1 数据收集与传输
在这个例子中，我们使用了一些传感器来收集气温、湿度、光照等数据，并使用了Zigbee协议进行数据传输。以下是一个简单的Python代码实例：

```python
import time
from zigbee import Zigbee

# 初始化Zigbee设备
zigbee = Zigbee()
zigbee.connect()

# 收集数据
while True:
    temp = zigbee.get_temperature()
    humidity = zigbee.get_humidity()
    light = zigbee.get_light()
    print("温度: {}, 湿度: {}, 光照: {}".format(temp, humidity, light))
    time.sleep(60)
```

