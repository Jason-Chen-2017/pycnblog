3. 查询区间：对于每个查询的区间，需要在哈希表中查找与其相交的区间。查询操作主要包括以下步骤：

   - 使用哈希值作为索引，遍历哈希表中对应的桶，查找与当前区间相交的区间。

4. 删除区间：对于每个需要删除的区间，需要将其从哈希表中删除。删除操作主要包括以下步骤：
   - 根据区间的起始点和终点计算其哈希值；
   - 使用哈希值作为索引，将区间从哈希表中对应的桶中删除。

数学模型公式详细讲解：

对于一个给定的区间（start，end），其哈希值可以使用以下公式计算：

$$
hash(start, end) = start \times P + end \times Q \mod M
$$

其中，P、Q、M 是三个大素数，满足：

$$
P \times Q \equiv 1 \mod M
$$

这种哈希函数的选择可以确保哈希表具有良好的负载因子和低的冲突概率。

# 4.具体代码实例和详细解释说明
以下是一个使用 Python 实现的简单示例，演示了如何在哈希表中实现区间算术的基本操作：

```python
class Interval:
    def __init__(self, start: int = 0, end: int = 0):
        self.start = start
        self.end = end

    def __hash__(self):
        return hash((self.start, self.end))

    def __eq__(self, other):
        return self.start == other.start and self.end == other.end

class IntervalHashTable:
    def __init__(self):
        self.table = {}

    def insert(self, interval: Interval):
        if interval.start < interval.end:
            self.table[hash(interval)] = interval
        else:
            raise ValueError("Invalid interval")

    def query(self, interval: Interval):
        hashes = [hash(interval)]
        for start, end in self.table.keys():
            if start < end:
                if interval.start <= start and end <= interval.end:
                    hashes.append(hash((start, end)))
        for hash_key in set(hashes):
            if hash_key in self.table:
                return self.table[hash_key]
        return None

    def remove(self, interval: Interval):
        if interval.start < interval.end:
            del self.table[hash(interval)]
        else:
            raise ValueError("Invalid interval")
```

在上述代码中，我们首先定义了一个 `Interval` 类，用于表示一个区间。然后定义了一个 `IntervalHashTable` 类，用于实现基本的插入、查询和删除操作。通过使用哈希表来存储和查询区间，我们可以在平均时间复杂度为 O(1) 的情况下完成这些操作。
