## 4.1 数据中心的边缘计算代码实例

以下是一个简单的数据中心的边缘计算代码实例：

```python
import multiprocessing

def compute(data):
    # 计算任务的具体实现
    result = data * data
    return result

if __name__ == '__main__':
    data = [1, 2, 3, 4, 5]
    pool = multiprocessing.Pool(processes=4)
    results = pool.map(compute, data)
    print(results)
```

在这个代码实例中，我们使用了Python的multiprocessing库来实现数据中心的边缘计算。我们将计算任务分解为多个子任务，并将这些子任务分配给多个进程进行处理。最后，我们将结果列出来。

## 4.2 分布式存储代码实例

以下是一个简单的分布式存储代码实例：

```python
import os

def save_block(block, file_name):
    # 存储块的具体实现
    with open(file_name, 'ab') as f:
        f.write(block)

if __name__ == '__main__':
    data = [1, 2, 3, 4, 5]
    file_name = 'data.bin'
    for i in range(len(data)):
        block = data[i].to_bytes(4, 'big')
        save_block(block, file_name)
```

在这个代码实例中，我们使用了Python的os库来实现分布式存储。我们将数据存储分解为多个块，并将这些块存储在文件中。最后，我们将文件名列出来。
