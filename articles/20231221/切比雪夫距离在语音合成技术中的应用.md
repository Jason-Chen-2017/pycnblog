# 2.核心概念与联系

## 2.1 切比雪夫距离简介

切比雪夫距离（Chebyshev distance）是一种度量距离，用于衡量两个数值之间的差异。它的定义为：

$$
d_C(x, y) = \mathop{\max}\limits_{i=1,2,\cdots,n} \left|\frac{x_i - y_i}{x_i}\right|
$$

其中，$x = (x_1, x_2, \cdots, x_n)$ 和 $y = (y_1, y_2, \cdots, y_n)$ 是两个n维向量，$x_i$ 和 $y_i$ 分别是向量x和向量y的第i个元素。

切比雪夫距离的特点是它对最大差异值不敏感，即使在数据集中存在噪声、异常值或者极值，切比雪夫距离也能保持稳定。因此，它在语音合成技术中具有很高的应用价值。

import numpy as np

def chebyshev_distance(x, y):
    if len(x) != len(y):
        raise ValueError("x and y must have the same length")
    max_diff = np.max(np.abs(x - y) / np.maximum(1e-10, x))
    return max_diff

distance = chebyshev_distance(x, y)
print("Chebyshev distance:", distance)
