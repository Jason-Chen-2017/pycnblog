条件半正定（Conditionally Negative Definite, CND）是一种用于解决数值分析中的优化问题的方法。它主要应用于处理那些在某些区域内具有多个局部最优解的问题。条件半正定方法的核心思想是通过构建一组条件半正定的函数来描述问题空间，从而找到问题的全局最优解。

条件半正定方法的研究起源于1960年代，主要由俄罗斯数值分析学者V.V.Adianovsky和他的学生开发。自那时以来，这一方法在数值优化、控制理论和机器学习等领域得到了广泛应用。

## 2.1.条件半正定函数

条件半正定函数是一种特殊的函数，它在某个子集上是半正定的，即在该子集内部的任何点上都满足f(x)≥0，但在该子集外部的某些点上可能仍然满足f(x)<0。这种半正定性质使得条件半正定函数可以用来描述问题空间中的多个局部最优解。

## 2.2.条件半正定区域

条件半正定区域是一种特殊的区域，它由一组条件半正定函数共同构成。在该区域内，所有的条件半正定函数都是半正定的，因此该区域内的任何点都是问题的局部最优解。通过对条件半正定区域进行探索和优化，我们可以找到问题的全局最优解。

## 2.3.条件半正定方程组

条件半正定方程组是一种特殊的方程组，其解可以被描述为条件半正定区域。通过解析和数值方法解决条件半正定方程组，我们可以得到问题的解。

- 条件半正定函数的定义：

$$

其中，$D$ 是问题空间的子集，$P$ 是正定矩阵，$q$ 是向量。

    return np.sum(x) == 0

    return x

    fx = cnd_half_def_func(x)
