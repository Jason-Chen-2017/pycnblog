## 4.1 环境污染预测算法代码实例

```python
import numpy as np
import matplotlib.pyplot as plt

def predict_pollution(data):
    x = data[:, 0]  # 时间
    y = data[:, 1]  # 污染指数
    model = np.polyfit(x, y, 1)  # 拟合线性模型
    p = np.poly1d(model)  # 得到线性模型
    y_pred = p(x)  # 预测污染指数
    return y_pred

data = np.loadtxt('pollution_data.txt', delimiter=',')
y_pred = predict_pollution(data)
plt.plot(x, y, label='实际污染指数')
plt.plot(x, y_pred, label='预测污染指数')
plt.legend()
plt.show()
```

在这个代码实例中，我们使用了线性回归模型来预测环境污染的发展趋势。首先，我们导入了numpy和matplotlib库，然后定义了一个`predict_pollution`函数，该函数接受污染数据作为输入，并使用线性回归模型进行预测。最后，我们使用matplotlib绘制了实际和预测的污染指数，以便于比较。
