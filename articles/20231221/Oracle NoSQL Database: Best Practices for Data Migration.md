# 4.具体代码实例和详细解释说明

在这里，我们将提供一个具体的代码实例，展示如何在 Oracle NoSQL Database 中进行数据迁移。这个例子将演示如何使用 Data Pump 工具将数据从一个 Oracle 关系数据库迁移到 Oracle NoSQL Database。

```
# 导出源数据库中的数据
expdp directory=DP_DIR dumpfile=expdp.dmp logfile=expdp.log grant_options=y

# 导入目标数据库中的数据
impdp directory=DP_DIR dumpfile=expdp.dmp logfile=impdp.log grant_options=y
```

在这个例子中，我们首先使用 `expdp` 命令导出源数据库中的数据，然后使用 `impdp` 命令导入目标数据库中的数据。在导入数据过程中，我们可能需要使用 `grant_options=y` 选项，以便授予目标数据库中的对象权限。
