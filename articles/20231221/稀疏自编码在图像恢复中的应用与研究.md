# 2.核心概念与联系

稀疏自编码技术是一种深度学习方法，它通过学习编码器和解码器来实现压缩和恢复。在图像恢复中，稀疏自编码技术可以有效地恢复损坏的图像，提高图像质量，降低计算成本。

稀疏自编码技术的核心概念包括：

1. 稀疏表示：稀疏表示是指在特定基础上，信号只有很少的非零分量。例如，在wavelet基础上，很多实际应用中的信号都可以用很少的非零分量来表示。

2. 自编码器：自编码器是一种深度学习模型，它包括编码器和解码器两部分。编码器将输入信号压缩为低维的特征表示，解码器将这些特征表示恢复为原始信号。

3. 稀疏自编码器：稀疏自编码器是一种特殊的自编码器，它利用输入信号的稀疏特性，将输入信号压缩为低维的稀疏特征表示，然后通过解码器恢复为原始信号。

稀疏自编码技术与图像恢复之间的联系在于，图像信号具有稀疏特性，因此可以使用稀疏自编码技术来实现图像恢复。在图像恢复中，稀疏自编码技术可以有效地恢复损坏的图像，提高图像质量，降低计算成本。
