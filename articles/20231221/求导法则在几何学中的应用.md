## 3.1 面积的计算
在几何学中，求导法则可以用于计算各种形状的面积。例如，对于一个圆形的面积，我们可以使用求导法则来计算其面积。

圆形的面积公式为：$$ A = \pi r^2 $$

其中，$A$ 是面积，$r$ 是圆的半径。通过求导法则，我们可以计算圆形的面积。

## 3.2 体积的计算
求导法则在几何学中也可以用于计算体积。例如，对于一个圆柱体的体积，我们可以使用求导法则来计算其体积。

圆柱体的体积公式为：$$ V = \pi r^2 h $$

其中，$V$ 是体积，$r$ 是圆柱体的半径，$h$ 是圆柱体的高度。通过求导法则，我们可以计算圆柱体的体积。

## 3.3 曲率的计算
求导法则在几何学中还可以用于计算曲率。例如，对于一个圆形曲线的曲率，我们可以使用求导法则来计算其曲率。

圆形曲线的曲率公式为：$$ C = \frac{1}{r} $$

其中，$C$ 是曲率，$r$ 是圆形曲线的半径。通过求导法则，我们可以计算圆形曲线的曲率。

```python
import math

def circle_area(radius):
    return math.pi * radius ** 2

def cylinder_volume(radius, height):
    return math.pi * radius ** 2 * height

def circle_curvature(radius):
    return 1 / radius

# 测试
radius = 5
height = 10

print("圆形面积：", circle_area(radius))
print("圆柱体体积：", cylinder_volume(radius, height))
print("圆形曲线曲率：", circle_curvature(radius))
```

在这个代码示例中，我们首先导入了`math`模块，用于获取π的值。然后，我们定义了三个函数：`circle_area`、`cylinder_volume`和`circle_curvature`，分别用于计算圆形面积、圆柱体体积和圆形曲线曲率。最后，我们测试这些函数，并输出结果。
