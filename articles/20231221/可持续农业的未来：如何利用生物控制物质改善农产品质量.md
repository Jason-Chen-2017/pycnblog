```python
import numpy as np
import matplotlib.pyplot as plt

def growth_model(t, X, r, K):
    dXdt = r * X * (1 - X / K)
    return dXdt

def euler_method(t, X, h, r, K):
    for _ in range(int(t / h)):
        X = X + h * growth_model(t, X, r, K)
        t += h
    return X

t = np.arange(0, 100, 0.1)
X0 = 0.1
r = 0.1
K = 10

X = euler_method(t, X0, 0.1, r, K)
plt.plot(t, X)
plt.xlabel('Time')
plt.ylabel('Population')
plt.title('Growth Model of Sweet Pepper')
plt.show()
```

在这个代码实例中，我们首先导入了numpy和matplotlib库，用于计算和绘图。然后，我们定义了一个生长模型函数`growth_model`，用于计算生物过程的变化。接着，我们使用了Euler方法来求解生长模型，并绘制了生长曲线。通过观察生长曲线，我们可以看到胺激素对甜甜菜生长速度的影响。
