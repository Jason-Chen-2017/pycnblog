## 4.1 AES加密和解密
以下是一个使用Python和AES加密和解密数据的简单示例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad

# 生成AES密钥
key = get_random_bytes(16)

# 生成AES块加密器
cipher = AES.new(key, AES.MODE_CBC)

# 加密数据
data = b"Hello, World!"
ciphertext = cipher.encrypt(pad(data, AES.block_size))

# 解密数据
plaintext = unpad(cipher.decrypt(ciphertext), AES.block_size)

print(plaintext.decode())
```

在这个示例中，我们首先导入了AES加密器、随机字节和填充实用程序。然后，我们生成了一个16字节的AES密钥，并创建了一个AES块加密器。接下来，我们使用加密器加密了一个字符串，并将其存储在`ciphertext`变量中。最后，我们使用加密器解密`ciphertext`，并将其解压缩为原始数据。

## 4.2 SNMP查询
以下是一个使用SNMP查询网络设备的简单示例：

```python
import snmp

# 连接到设备
session = snmp.Session('127.0.0.1', community='public')

# 发送SNMP查询
response = session.send('1.3.6.1.2.1.1.1.0')

# 解析响应
oid = response[0]
value = response[1].prettyPrint()

print(f"{oid}: {value}")
```

在这个示例中，我们首先导入了SNMP库。然后，我们使用`snmp.Session`类连接到一个设备，并使用公共社区字符串进行身份验证。接下来，我们使用`session.send`方法发送一个SNMP查询，并将其存储在`response`变量中。最后，我们使用`response[1].prettyPrint()`方法将响应解析为人类可读的格式，并将其打印出来。
