# 2. 核心概念与联系

## 2.1 密切圆

密切圆是指在平面上两条不相交的曲线在相交点上的切线。在几何中，密切圆是一种特殊的外接圆，它通过两条曲线的相交点和曲线的切点。密切圆的概念在古典几何学中已经有很长时间了，但是在计算机图形学中，密切圆的计算和应用得到了更加深入的研究。

密切圆在计算机图形学中的一个重要应用是曲线和曲面的近似。通过计算多个曲线在相交点上的密切圆，可以得到一个近似曲面，这种近似方法在许多计算机图形学应用中得到了广泛使用。

## 2.2 曲率

曲率是指在平面上一个曲线在某一点的弧度。曲率是几何和分析几何中的一个基本概念，它有许多应用，例如在计算机图形学中用于曲线和曲面的建模和处理。

曲率在计算机图形学中的一个重要应用是曲线和曲面的描述和处理。通过计算曲线在某一点的曲率值，可以得到曲线的形状信息，从而进行更精确的图形处理。此外，曲率还在机器学习和计算机视觉等领域发挥着重要作用，例如在对象检测、图像分割和自动驾驶等方面。

    tangent_vector_norm = np.linalg.norm(tangent_vector)
    return np.linalg.norm(tangent_vector) / tangent_vector_norm

def tangent_circle_center(x1, y1, x2, y2):

    y_diff = y1 - y2
