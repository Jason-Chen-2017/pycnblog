在本节中，我们将通过一个具体的代码实例来详细解释迁移学习在图像超分辨率恢复中的应用。我们将使用Python和Pytorch来实现这个任务。

### 4.1 代码实例
```python
import torch
import torchvision
import torchvision.transforms as transforms
import torch.nn as nn
import torch.optim as optim

# 加载预训练模型
model = torchvision.models.resnet50(pretrained=True)

# 修改模型输入和输出层
model.fc = nn.Linear(2048, 3)

# 加载低分辨率图像数据集
transform = transforms.Compose(
    [transforms.Resize((224, 224)),
     transforms.ToTensor(),
     transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5))])

dataset = torchvision.datasets.ImageFolder(root='./data', transform=transform)

# 定义损失函数和优化器
criterion = nn.MSELoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 训练模型
for epoch in range(10):
    running_loss = 0.0
    for i, data in enumerate(dataset, 0):
        inputs, labels = data

        optimizer.zero_grad()

        outputs = model(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()

        running_loss += loss.item()
    print('Epoch: %d, Loss: %.3f' % (epoch + 1, running_loss / len(dataset)))

# 使用模型进行超分辨率恢复
def super_resolve(lr_img):
    lr_img = torch.unsqueeze(torch.unsqueeze(lr_img, 0), 0)
    lr_img = transforms.Resize((224, 224))(lr_img)
    lr_img = torch.unsqueeze(torch.unsqueeze(lr_img, 0), 0)
    lr_img = torch.unsqueeze(lr_img, 0)

    with torch.no_grad():
        output = model(lr_img)

    output = torch.squeeze(output, 0)
    output = torch.squeeze(output, 0)
    output = torch.squeeze(output, 0)

    return output
```

在这个代码实例中，我们首先加载了一个预训练的ResNet50模型，并将其输入和输出层进行了修改。接着，我们加载了一组低分辨率图像数据集，并对其进行了预处理。然后，我们定义了损失函数和优化器，并使用了一个循环来训练模型。在训练过程中，我们使用了一组已知的高分辨率图像来进行监督学习，以便更好地学习到目标任务的特点。

最后，我们使用了训练好的模型进行超分辨率恢复。具体来说，我们首先将输入的低分辨率图像resize到了模型输入的大小，然后使用模型进行预测。最后，我们对预测结果进行了解压缩，得到了高分辨率的恢复图像。
