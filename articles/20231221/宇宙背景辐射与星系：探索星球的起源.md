```python
import numpy as np
import matplotlib.pyplot as plt

def integrate(f, a, b):
    return np.trapz(f, a, b)

def main():
    # 模拟宇宙背景辐射的谱线
    f = np.random.rand(1000)
    a = np.linspace(0, 10, 1000)

    # 计算微波辐射的谱线
    i = integrate(f, a, a + 1e-6)

    # 绘制谱线图
    plt.plot(a, f, label='Spectrum')
    plt.plot(a, i, label='Integrated Spectrum')
    plt.xlabel('Frequency')
    plt.ylabel('Intensity')
    plt.legend()
    plt.show()

if __name__ == '__main__':
    main()
```
这个代码实例使用了NumPy和Matplotlib库来模拟宇宙背景辐射的谱线，并使用了NumPy的trapz函数来计算谱线的积分。最后，使用了Matplotlib库来绘制谱线图。
