# 4.具体代码实例和详细解释说明

## 4.1 数据存储

```python
import pandas as pd

# 创建一个数据框
data = {'name': ['John', 'Alice', 'Bob'],
        'age': [25, 30, 35],
        'gender': ['M', 'F', 'M']}
df = pd.DataFrame(data)

# 将数据存储到CSV文件中
df.to_csv('data.csv', index=False)
```

## 4.2 数据清洗

```python
# 读取CSV文件
df = pd.read_csv('data.csv')

# 删除缺失值
df = df.dropna()

# 将数据类型转换为字符串
df['name'] = df['name'].astype(str)
```

## 4.3 数据转换

```python
# 创建一个新的数据框
df_new = pd.DataFrame(columns=['name', 'age', 'gender', 'sex'])

# 将数据转换为新的数据框
df_new['name'] = df['name']
df_new['age'] = df['age']
df_new['gender'] = df['gender']
df_new['sex'] = df['gender'].map({'M': '男', 'F': '女'})
```

## 4.4 数据分析

```python
# 计算平均年龄
average_age = df['age'].mean()

# 计算男女比例
male_ratio = df['gender'].value_counts(normalize=True)

# 打印结果
print('平均年龄：', average_age)
print('男女比例：', male_ratio)
```
