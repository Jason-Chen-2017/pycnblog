## 4.1 社交网络的构建和分析

我们将使用Python的NetworkX库来构建和分析社交网络。首先，我们需要导入所需的库：

```python
import networkx as nx
import pandas as pd
import numpy as np
```

接下来，我们需要从CSV文件中加载用户的信息，并将其转换为Graph对象：

```python
# 加载用户信息
user_data = pd.read_csv('user_data.csv')

# 创建Graph对象
G = nx.Graph()

# 添加节点
G.add_nodes_from(user_data['user_id'])

# 添加边
G.add_edges_from(user_data[['user_id', 'followed_id']])
```

现在，我们可以对社交网络进行分析了。例如，我们可以计算每个节点的度中心性：

```python
# 计算度中心性
degree_centrality = nx.degree_centrality(G)

# 打印度中心性
print(degree_centrality)
```

或者，我们可以计算每个节点的Betweenness Centrality：

```python
# 计算Betweenness Centrality
betweenness_centrality = nx.betweenness_centrality(G)

# 打印Betweenness Centrality
print(betweenness_centrality)
```
