                 

# 1.背景介绍

计算流体动力学（Computational Fluid Dynamics，简称CFD）是一门研究流体的动力学行为的科学和技术。它结合了数值解算、数学模型、计算机科学和物理学等多个领域的知识和方法，以求解流体中的各种力和速度等量度。CFD 在各个行业领域中具有广泛的应用，如航空、汽车、能源、化学、环境等。

CFD 的核心概念包括流体的物理现象、数值解算方法和计算模型。流体的物理现象包括流体的动力学和热力学行为、流体与界面的相互作用等。数值解算方法包括有限元方法、有限差分方法、有限卷积方法等。计算模型则包括基础流体模型、混合流体模型、多相流体模型等。
