# 2.核心概念与联系
## 2.1 Altibase简介
Altibase 是一款高性能的关系型数据库管理系统（RDBMS），它具有低延迟、高可用性、高可扩展性和强大的数据处理能力。Altibase 使用了一种称为 In-Memory 数据库（IMDB）的技术，该技术将数据存储在内存中，从而实现了极快的数据访问速度。此外，Altibase 还支持多种数据类型，如时间序列数据、图形数据等，使其在人工智能和大数据分析领域具有广泛的应用。
