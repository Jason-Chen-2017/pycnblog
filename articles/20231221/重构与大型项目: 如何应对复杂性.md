我们可以通过以下步骤对这个程序进行重构：

1. 将计算器程序拆分成多个类，每个类负责一个计算任务。
2. 将这些类组合在一起，形成新的计算器程序。

具体的代码实例如下所示：

```
#include <iostream>
using namespace std;

class Calculator {
public:
    int add(int a, int b) {
        return a + b;
    }

    int sub(int a, int b) {
        return a - b;
    }

    int mul(int a, int b) {
        return a * b;
    }

    int div(int a, int b) {
        return a / b;
    }
};

int main() {
    int a, b;
    cout << "Enter two numbers: ";
    cin >> a >> b;
    Calculator calculator;
    cout << "Addition: " << calculator.add(a, b) << endl;
    cout << "Subtraction: " << calculator.sub(a, b) << endl;
    cout << "Multiplication: " << calculator.mul(a, b) << endl;
    cout << "Division: " << calculator.div(a, b) << endl;
    return 0;
}
```

通过这个例子，我们可以看到重构技术可以帮助我们将复杂的代码简化为更简单的代码，提高代码的可维护性和可读性。
