# 2.核心概念与联系

## 2.1 IoT

IoT（Internet of Things）是一种通过互联网连接物理设备的技术。IoT可以让物理设备通过网络互相交流信息，实现设备的智能化，提高设备的利用效率，实现设备的可持续发展。IoT的核心技术包括：

1. 物联网设备：物联网设备是通过网络连接的物理设备。物联网设备可以是传感器，可以是摄像头，可以是智能灯泡，等等。

2. 物联网通信：物联网通信是物联网设备通过网络互相交流信息的技术。物联网通信可以是无线通信，可以是有线通信，可以是混合通信，等等。

3. 物联网平台：物联网平台是物联网设备通过网络互相交流信息的平台。物联网平台可以是云平台，可以是边缘平台，可以是混合平台，等等。

4. 物联网应用：物联网应用是通过物联网设备、物联网通信、物联网平台来实现的应用。物联网应用可以是智能家居，可以是智能城市，可以是智能交通，等等。
