# 2. 核心概念与联系

## 2.1 用户生命周期
用户生命周期（User Lifecycle）是指从用户首次接触到企业产品或服务，直到用户离开的整个过程。用户生命周期可以分为以下几个阶段：

1. 潜在客户（Prospect）：这是用户首次接触企业产品或服务的阶段，用户还没有成为真正的客户，但他们可能在未来成为客户的潜在来源。
2. 新客户（New Customer）：这是用户成为真正客户的阶段，用户开始使用企业的产品或服务。
3. 活跃客户（Active Customer）：这是用户积极使用企业产品或服务的阶段，用户可能会对企业产品或服务进行评价和推荐。
4. 潜在流失客户（At-risk Customer）：这是用户开始减少使用企业产品或服务的阶段，可能会在未来离开的客户。
5. 流失客户（Churned Customer）：这是用户已经离开企业的阶段，用户不再使用企业产品或服务。

## 2.2 社交媒体营销
社交媒体营销是利用社交媒体平台（如Facebook、Twitter、Instagram等）进行营销活动的方式。社交媒体营销的目的是建立和维护与用户的关系，以便更好地推广产品和服务。
