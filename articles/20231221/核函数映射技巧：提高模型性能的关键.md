# 2.核心概念与联系

核函数映射技巧是一种将高维数据映射到低维空间的方法，从而减少计算复杂度和提高模型性能。核函数映射技巧主要包括以下几个概念：

- 核函数：核函数是一种用于计算两个数据点之间距离的函数。常见的核函数包括欧氏距离、余弦距离、高斯核函数等。
- 核映射：核映射是将高维数据映射到低维空间的过程。通过核映射，我们可以将高维数据转换为低维数据，从而减少计算复杂度。
- 核函数的选择：核函数的选择对于核映射的效果非常关键。不同的核函数会导致不同的映射结果，因此在实际应用中需要根据具体问题选择合适的核函数。

3. 求核矩阵的特征值和特征向量。通过对核矩阵进行特征分解，我们可以得到低维数据的表示。
4. 将低维数据用于模型训练和预测。通过将高维数据映射到低维空间，我们可以减少计算复杂度，从而提高模型性能。

Q: 核函数映射技巧与SVM有什么区别？
