## 1.2 核心概念与联系

### 1.2.1 矩阵

### 1.2.3 矩阵的乘法

矩阵的乘法是将一矩阵的每一行看作是另一个矩阵的列向量，然后按照元素的乘积和求和的规则计算得到新的矩阵。矩阵的乘法是不交换律的。

矩阵的行列式是一个数值，可以用来判断矩阵是否可逆。如果矩阵的行列式不为零，则该矩阵是可逆的；否则，该矩阵是非可逆的。

import numpy as np

def matrix_inverse(A):
    n = A.shape[0]
    B = np.identity(n)
    for i in range(n):
        for j in range(n):
            B[i][j] = (A[i][j] * (np.linalg.det(A[i:n, j:n]) * np.linalg.det(A[:i, :j])) / (np.linalg.det(A)))
    return B

A = np.array([[1, 2], [3, 4]])
A_inv = matrix_inverse(A)

```

import numpy as np

def matrix_inverse(A):
    n = A.shape[0]
    B = np.identity(n)
    for i in range(n):
        B[i] = np.linalg.solve(A[i:n, :], A[i, :])

import numpy as np

def matrix_inverse(A):
    n = A.shape[0]
    B = np.identity(n)
    for i in range(n):
        max_abs = max(abs(A[i:n, i:n]))
        for j in range(n):
            if abs(A[i][j]) >= max_abs:
                A[i], A[j] = A[j], A[i]
                B[i], B[j] = B[j], B[i]
        if A[i][i] == 0:
            raise ValueError("Matrix is not invertible")
        A[i] /= A[i][i]
        B[i] /= A[i][i]
        for j in range(n):
            if j != i:
                A[j] -= A[i][j] * A[i]
                B[j] -= A[i][j] * B[i]
    return B

A = np.array([[1, 2], [3, 4]])
A_inv = matrix_inverse(A)
print(A_inv)
```
