# 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来解释微分在图像处理中的实际应用。

```python
import numpy as np
import cv2
import matplotlib.pyplot as plt

# 加载图像

# 高斯滤波
def gaussian_blur(img, sigma):
    kernel = cv2.getGaussianKernel(5, sigma)
    blurred = cv2.filter2D(img, -1, kernel)
    return blurred

# 梯度计算
def gradient(img):
    grad_x = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)
    grad_y = cv2.Sobel(img, cv2.CV_64F, 0, 1, ksize=3)
    grad = np.sqrt(grad_x**2 + grad_y**2)
    return grad

# 边缘检测
def edge_detection(img):
    laplacian = cv2.Laplacian(img, cv2.CV_64F)
    return laplacian

# 显示图像
def show_image(img):
    plt.imshow(img, cmap='gray')
    plt.show()

# 高斯滤波
img_blur = gaussian_blur(img, 1)

# 梯度计算
img_grad = gradient(img_blur)

# 边缘检测
img_edge = edge_detection(img_grad)

# 显示图像
show_image(img_edge)
```

在这个代码实例中，我们首先加载了一个灰度图像，然后使用高斯滤波来减少噪声。接着，我们使用梯度计算来计算图像的梯度，最后使用拉普拉斯滤波来检测边缘。最后，我们使用matplotlib来显示处理后的图像。
