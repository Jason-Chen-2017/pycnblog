## 4.具体代码实例和详细解释说明

在这里，我们以一个常见的算法题为例，进行具体的代码实例和解释说明：

### 题目：求两个有序数组的交集

给定两个有序数组 `nums1` 和 `nums2`，返回两个数组的交集。数组中的元素是唯一的。

示例：
```
输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2]
```

解题思路：
1. 创建一个结果数组 `result` 来存储交集元素。
2. 使用两个指针 `i` 和 `j` 分别遍历 `nums1` 和 `nums2`。
3. 当 `nums1[i]` 等于 `nums2[j]` 时，说明这个元素是交集中的元素，将其加入到 `result` 中，并分别增加 `i` 和 `j` 的值。
4. 如果 `nums1[i]` 小于 `nums2[j]`，则增加 `i` 的值；如果 `nums1[i]` 大于 `nums2[j]`，则增加 `j` 的值。
5. 当 `i` 或 `j` 到达数组的末尾时，遍历结束。

代码实现：
```python
def intersection(nums1, nums2):
    i, j = 0, 0
    result = []
    while i < len(nums1) and j < len(nums2):
        if nums1[i] == nums2[j]:
            result.append(nums1[i])
            i += 1
            j += 1
        elif nums1[i] < nums2[j]:
            i += 1
        else:
            j += 1
    return result
```

在这个例子中，我们使用了两个指针的方法来解决问题，时间复杂度为 O(m+n)，空间复杂度为 O(min(m, n))。
