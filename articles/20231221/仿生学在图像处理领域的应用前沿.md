## 4.1 代码实例
```python
import numpy as np
import cv2
import matplotlib.pyplot as plt

# 读取图像

# 模仿生物视觉系统的光侦响应
def light_response(img):
    return cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 模仿生物视觉系统的图像处理
def image_processing(img):
    return cv2.GaussianBlur(img, (5, 5), 0)

# 模仿生物视觉系统的信息传输
def information_transfer(img):
    return cv2.resize(img, (200, 200))

# 主程序
def main():
    # 读取图像

    # 模仿生物视觉系统的光侦响应
    img_gray = light_response(img)

    # 模仿生物视觉系统的图像处理
    img_processed = image_processing(img_gray)

    # 模仿生物视觉系统的信息传输
    img_transfer = information_transfer(img_processed)

    # 显示图像
    plt.imshow(img_transfer)
    plt.show()

if __name__ == '__main__':
    main()
```
## 4.2 详细解释说明
这个代码实例主要包括以下几个步骤：

1. 读取图像：使用OpenCV库的`imread`函数读取图像，并将其存储为变量`img`。
2. 模仿生物视觉系统的光侦响应：使用OpenCV库的`cvtColor`函数将原始图像从BGR格式转换为灰度格式，并将其存储为变量`img_gray`。
3. 模仿生物视觉系统的图像处理：使用OpenCV库的`GaussianBlur`函数对灰度图像进行高斯模糊处理，并将其存储为变量`img_processed`。
4. 模仿生物视觉系统的信息传输：使用OpenCV库的`resize`函数将处理后的灰度图像缩放到200x200的大小，并将其存储为变量`img_transfer`。
5. 显示图像：使用Matplotlib库的`imshow`函数显示处理后的图像，并使用`show`函数显示图像。
