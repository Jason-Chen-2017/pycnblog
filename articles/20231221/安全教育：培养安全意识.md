# 4. 安全教育的具体代码实例和解释

以下是一个简单的Python代码实例，用于计算MD5哈希值：

```python
import hashlib

def md5(data):
    md5 = hashlib.md5()
    md5.update(data.encode('utf-8'))
    return md5.hexdigest()

data = "Hello, World!"
print(md5(data))
```

这个代码实例中，我们首先导入了`hashlib`模块，然后定义了一个`md5`函数，该函数接收一个字符串参数`data`，并使用`md5`哈希算法计算其MD5哈希值。最后，我们使用`print`函数打印了计算结果。
