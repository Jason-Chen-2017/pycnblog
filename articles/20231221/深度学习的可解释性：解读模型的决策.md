# 2.核心概念与联系
## 2.1 可解释性与解释性
可解释性是指模型的决策过程可以被人类理解和解释的程度。解释性是指模型的决策过程可以被模型本身或其他算法解释出来的程度。可解释性和解释性之间存在密切的关系，但它们不同。

## 2.2 解释性与诊断
解释性是指模型的决策过程可以被模型本身或其他算法解释出来的程度。诊断是指通过分析模型的错误决策，以及通过可视化方法展示模型的决策过程。解释性与诊断之间存在密切的关系，但它们不同。

## 2.3 解释性与可视化
解释性是指模型的决策过程可以被模型本身或其他算法解释出来的程度。可视化是指将模型的决策过程以图形或其他形式展示给人类的过程。解释性与可视化之间存在密切的关系，但它们不同。
