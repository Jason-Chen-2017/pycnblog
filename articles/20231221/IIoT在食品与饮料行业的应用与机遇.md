## 4.1 实时监控生产线

### 4.1.1 传感器数据收集

首先，我们需要收集传感器数据。以下是一个简单的Python代码实例，用于收集传感器数据：

```python
import time
import random

def collect_sensor_data():
    while True:
        temperature = random.uniform(20, 30)
        humidity = random.uniform(40, 60)
        pressure = random.uniform(900, 1000)
        timestamp = time.time()
        data = {
            'temperature': temperature,
            'humidity': humidity,
            'pressure': pressure,
            'timestamp': timestamp
        }
        print(data)
```

### 4.1.2 数据传输与存储

接下来，我们需要将收集到的传感器数据通过网络传输到服务器，并存储到数据库中。以下是一个简单的Python代码实例，用于将传感器数据存储到数据库中：

```python
import requests

def store_sensor_data(data):
    url = 'http://your_server_url/store_sensor_data'
    headers = {'Content-Type': 'application/json'}
    response = requests.post(url, json=data, headers=headers)
    if response.status_code == 200:
        print('Data stored successfully.')
    else:
        print('Failed to store data.')
```

### 4.1.3 数据分析与展示

最后，我们需要对收集到的传感器数据进行分析，并展示出来。以下是一个简单的Python代码实例，用于对传感器数据进行分析并展示出来：

```python
import matplotlib.pyplot as plt

def analyze_sensor_data(data):
    temperatures = [data['temperature'] for data in data]
    humidities = [data['humidity'] for data in data]
    pressures = [data['pressure'] for data in data]
    timestamps = [data['timestamp'] for data in data]

    plt.plot(timestamps, temperatures, label='Temperature')
    plt.plot(timestamps, humidities, label='Humidity')
    plt.plot(timestamps, pressures, label='Pressure')
    plt.xlabel('Time')
    plt.ylabel('Value')
    plt.legend()
    plt.show()
```
