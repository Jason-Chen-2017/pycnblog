## 4.1 Virtuoso 的具体代码实例

假设我们有一个关系数据库，其中包含一张员工表，其结构如下：

```sql
CREATE TABLE employee (
    id INT PRIMARY KEY,
    name VARCHAR(255),
    age INT,
    salary DECIMAL(10, 2)
);
```

我们希望将这个表集成到一个数据仓库中，并实现一些基本的查询功能。 以下是一个简单的 Virtuoso 代码实例，用于实现这个任务：

```python
import virtuoso

# 连接到 Virtuoso 数据库
conn = virtuoso.connect("localhost:1111", "dbname=mydb")

# 创建一个新的表，用于存储员工数据
conn.execute("CREATE TABLE employee_warehouse (id INT, name VARCHAR(255), age INT, salary DECIMAL(10, 2))")

# 将员工表中的数据插入到新表中
conn.execute("INSERT INTO employee_warehouse (id, name, age, salary) SELECT id, name, age, salary FROM employee")

# 实现一个查询功能，用于获取员工的平均薪资
def average_salary(conn):
    result = conn.execute("SELECT AVG(salary) FROM employee_warehouse")
    return result.fetchone()[0]

# 调用查询功能
print(average_salary(conn))

# 关闭连接
conn.close()
```

在这个代码实例中，我们首先连接到 Virtuoso 数据库，然后创建一个新的表 `employee_warehouse` 用于存储员工数据。 接着，我们将员工表中的数据插入到新表中。 最后，我们实现了一个查询功能，用于获取员工的平均薪资，并调用这个查询功能。
