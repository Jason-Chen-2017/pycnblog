Apache Nifi 是一个流处理框架，它可以实现数据流的创建、管理和监控。它可以处理大规模的数据流，并提供了一种简单的方法来构建、扩展和管理数据流管道。Nifi 使用一个名为“流处理”的概念来描述数据流的处理。流处理是一种在数据流中执行操作的方法，例如过滤、转换、聚合等。Nifi 提供了一种名为“流处理节点”的组件，这些节点可以在数据流中执行各种操作。

Nifi 的设计目标是提供一个可扩展、可配置的流处理框架，可以处理大规模的数据流。Nifi 的核心组件是流处理节点，它们可以在数据流中执行各种操作，例如过滤、转换、聚合等。Nifi 提供了一种名为“流处理关系”的组件，这些关系可以用来描述数据流如何在流处理节点之间流动。Nifi 还提供了一种名为“流处理报告”的组件，这些报告可以用来监控数据流的性能。
