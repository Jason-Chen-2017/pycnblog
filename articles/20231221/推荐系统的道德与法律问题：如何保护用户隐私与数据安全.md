## 2.1 推荐系统的定义和类型

推荐系统是一种基于数据挖掘、机器学习和人工智能技术的系统，它的主要目标是根据用户的需求和兴趣，为其提供个性化的产品和服务建议。推荐系统可以根据不同的特点和应用场景，分为以下几类：

2. 基于行为的推荐系统：这种推荐系统通过分析用户的浏览、购买和点赞等行为数据，为其提供与其行为相关的建议。例如，电商推荐系统、电影推荐系统等。

## 2.2 推荐系统中的数据和特征

在推荐系统中，数据是构建推荐模型和生成推荐结果的基础。推荐系统通常涉及到以下几种类型的数据：

1. 用户数据：包括用户的基本信息（如用户ID、年龄、性别等）、用户的行为数据（如浏览记录、购买记录、点赞记录等）和用户的社交关系数据（如好友关系、关注关系等）。

## 2.3 推荐系统中的评估指标和优化目标

在推荐系统中，评估指标是用于衡量推荐系统的性能的标准，优化目标是用于指导推荐系统的改进和优化的目标。常见的推荐系统评估指标有：

## 3.1 基于协同过滤的推荐算法

基于协同过滤（Collaborative Filtering）的推荐算法是一种基于用户行为数据的推荐方法，它的核心思想是通过分析用户的共同行为，为用户推荐与其相似的项目。基于协同过滤的推荐算法可以分为以下两种类型：

1. 基于用户的协同过滤（User-Based Collaborative Filtering）：这种方法通过分析用户之间的共同行为，为用户推荐与其相似的项目。具体的操作步骤如下：

   a. 构建用户相似度矩阵：将用户和项目分为多个类别，为每个类别构建一个相似度矩阵，以便于计算用户之间的相似度。
   b. 计算用户之间的相似度：使用欧氏距离、皮尔逊相关系数等方法，计算用户之间的相似度。
   c. 筛选出与目标用户相似的用户：根据用户之间的相似度，筛选出与目标用户相似的用户。
   d. 为目标用户推荐项目：根据筛选出的用户的行为数据，为目标用户推荐项目。

2. 基于项目的协同过滤（Item-Based Collaborative Filtering）：这种方法通过分析项目之间的共同行为，为用户推荐与其相似的项目。具体的操作步骤如下：

   a. 构建项目相似度矩阵：将用户和项目分为多个类别，为每个类别构建一个相似度矩阵，以便于计算项目之间的相似度。
   b. 计算项目之间的相似度：使用欧氏距离、皮尔逊相关系数等方法，计算项目之间的相似度。
   c. 筛选出与目标项目相似的项目：根据项目之间的相似度，筛选出与目标项目相似的项目。
   d. 为目标用户推荐项目：根据筛选出的项目的行为数据，为目标用户推荐项目。

import numpy as np
import pandas as pd
from scipy.spatial.distance import cosine

# 构建用户相似度矩阵
def build_user_similarity_matrix(user_matrix):
    user_similarity_matrix = pd.DataFrame(index=user_matrix.index, columns=user_matrix.index)
    for i in user_matrix.index:
        for j in user_matrix.index:
            if i == j:
                user_similarity_matrix.loc[i, j] = 0
            else:
                user_similarity_matrix.loc[i, j] = 1 - cosine(user_matrix.loc[i], user_matrix.loc[j])
    return user_similarity_matrix

# 筛选出与目标用户相似的用户
def filter_similar_users(user_similarity_matrix, target_user):
    similar_users = user_similarity_matrix.loc[target_user].sort_values(ascending=False)
    similar_users = similar_users[similar_users > 0]
    return similar_users

# 为目标用户推荐项目
def recommend_items(user_matrix, similar_users, top_n):
    user_item_matrix = pd.DataFrame(index=user_matrix.index, columns=user_matrix.columns)
    for user in similar_users.index:
        user_item_matrix += user_matrix.loc[user].sort_values(ascending=False)
    user_item_matrix = user_item_matrix.sort_values(ascending=False)
    return user_item_matrix.iloc[:top_n]

# 测试基于用户的协同过滤推荐算法

target_user = "user1"
top_n = 5
similar_users = filter_similar_users(user_matrix, target_user)
recommended_items = recommend_items(user_matrix, similar_users, top_n)
print(recommended_items)
