# 2.核心概念与联系

## 2.1 量子通信的核心概念

1. **量子比特（Qubit）**：量子比特是量子信息处理中的基本单位，它可以存储为0、1或者同时存储为0和1。这种特性使得量子比特具有超越经典比特的并行处理能力。
2. **量子叠加**：量子叠加是指量子系统可以存在多种状态同时，这种现象被称为纵向叠加。纵向叠加使得量子比特可以同时表示多种信息，从而实现并行处理。
3. **量子纠缠**：量子纠缠是指两个或多个量子系统之间的相互作用，使得它们的状态相互依赖。量子纠缠使得量子系统之间可以实现无缝连接，从而实现信息传输的安全性。
4. **量子密钥分发（QKD）**：量子密钥分发是量子通信中的核心技术，它利用量子物理现象实现安全的信息传输。通过量子密钥分发，两个远程用户可以共享一个安全的密钥，用于加密和解密信息。

## 2.2 卫星通信的核心概念

1. **地球轨道卫星（GEO）**：地球轨道卫星是一种绕地球轨道运行的卫星，它可以覆盖整个地球表面，提供高速、低延迟的通信服务。
2. **低轨道卫星（LEO）**：低轨道卫星是一种绕地球轨道运行的卫星，它的轨道高度较低，可以提供更快的通信速度，但覆盖范围较小。
3. **卫星通信网**：卫星通信网是一种利用卫星进行通信的网络系统，它可以实现高速、稳定的网络连接。

## 2.3 量子通信在卫星通信中的联系
