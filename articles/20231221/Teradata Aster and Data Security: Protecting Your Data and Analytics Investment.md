# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个简单的代码示例来说明Teradata Aster中的数据安全功能。假设我们有一个包含客户信息的表，我们想要对这个表进行加密。以下是一个简单的SQL查询，它将对表中的所有数据进行AES加密：

```sql
CREATE TABLE customer_encrypted
(
    id INT PRIMARY KEY,
    name VARCHAR(255),
    email VARCHAR(255),
    address VARCHAR(255)
)
WITH DATA ENCRYPTION
(
    ENCRYPTION_METHOD = AES,
    ENCRYPTION_KEY = 'your_encryption_key'
);

INSERT INTO customer_encrypted
SELECT id, name, email, address
FROM customer;
```

在这个示例中，我们首先创建了一个名为`customer_encrypted`的新表，并指定了数据加密方法和密钥。然后，我们使用INSERT INTO语句将原始客户表中的数据插入到新表中，同时对数据进行加密。
