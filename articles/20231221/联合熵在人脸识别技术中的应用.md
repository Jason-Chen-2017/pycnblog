# 2.核心概念与联系

## 2.1 信息熵

信息熵是信息论中的一个重要概念，用于衡量一个随机变量的不确定性。信息熵可以用来衡量一个文本数据集的纯随机性，也可以用来衡量一个特征的重要性。信息熵的公式为：

$$
H(X) = -\sum_{i=1}^{n} P(x_i) \log_2 P(x_i)
$$

其中，$H(X)$ 是信息熵，$X$ 是一个随机变量，$x_i$ 是 $X$ 的取值，$P(x_i)$ 是 $x_i$ 的概率。

## 2.2 联合熵

联合熵是信息熵的一种拓展，用于衡量多个随机变量之间的不确定性。联合熵的公式为：

$$
H(X_1, X_2, ..., X_n) = -\sum_{i=1}^{n} P(x_i) \log_2 P(x_i)
$$

其中，$H(X_1, X_2, ..., X_n)$ 是联合熵，$X_1, X_2, ..., X_n$ 是 $n$ 个随机变量。
