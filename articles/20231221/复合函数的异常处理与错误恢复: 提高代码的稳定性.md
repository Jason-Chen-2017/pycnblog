# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个具体的代码实例来详细解释异常处理和错误恢复的实现过程。

## 4.1 代码实例
假设我们有一个复合函数，它接收一个整数参数，并返回该整数的平方。如果参数为负数，则返回-1。代码如下：

```python
def square(x):
    if x < 0:
        return -1
    else:
        return x ** 2
```

在这个代码实例中，我们需要处理以下异常和错误情况：

- 如果参数不是整数，则抛出TypeError异常。
- 如果参数为负数，则返回-1。

## 4.2 异常处理实现
为了处理上述异常情况，我们可以在函数中添加try-except语句来捕获TypeError异常，并在参数不是整数时抛出异常。代码如下：

```python
def square(x):
    if not isinstance(x, int):
        raise TypeError("参数必须是整数")
    elif x < 0:
        return -1
    else:
        return x ** 2
```

在这个代码实例中，我们首先检查参数x是否是整数，如果不是，则抛出TypeError异常。然后，我们检查参数x是否为负数，如果是，则返回-1。最后，如果参数x不为负数，则返回x的平方。

## 4.3 错误恢复实现
为了实现错误恢复，我们可以在函数中添加try-except语句来捕获异常，并在参数为负数时进行回滚恢复。代码如下：

```python
def square(x):
    try:
        if not isinstance(x, int):
            raise TypeError("参数必须是整数")
        elif x < 0:
            return -1
        else:
            return x ** 2
    except TypeError as e:
        print(e)
        return -1
```

在这个代码实例中，我们首先尝试执行参数检查和计算平方的操作。如果参数不是整数，则抛出TypeError异常，并在except语句块中捕获异常，并打印异常信息，然后返回-1。
