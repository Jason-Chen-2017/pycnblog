### 4.1 依赖解析示例

假设我们有以下句子：

```
John gave Mary a book.
```

我们可以使用依赖解析算法来识别句子中的词与词之间的关系。例如，我们可以使用NLTK库来进行依赖解析：

```python
import nltk
nltk.download('punkt')
nltk.download('averaged_perceptron_tagger')
nltk.download('universal_tagset')

sentence = "John gave Mary a book."
tokens = nltk.word_tokenize(sentence)
pos_tags = nltk.pos_tag(tokens)
dependency_parse = nltk.DependencyParse(pos_tags, nltk.RegexpParser('SBAR|VB|NN|PRP|NP'))
```

这将生成以下依赖解析树：

```
(S
  (SBAR
    (PRP John)
    (VP
      (VBD gave)
      (NP
        (PRP Mary)
        (PP
          (IN to)
          (NP
            (DT a)
            (NN book)
          )
        )
      )
    )
  )
)
```
