## 4.具体代码实例和详细解释说明

### 4.1 加密算法实例
以ECDSA为例，我们来看一个具体的加密算法实例。

```python
import os
from ecies import Encryptor

# 生成一对公私钥
private_key = os.urandom(32)
public_key = Encryptor.generate_key(private_key)

# 使用公钥加密数据
message = b'hello world'
encrypted_data = Encryptor.encrypt(public_key, message)

# 使用私钥解密数据
decrypted_data = Encryptor.decrypt(private_key, encrypted_data)

print(decrypted_data.decode())
```
