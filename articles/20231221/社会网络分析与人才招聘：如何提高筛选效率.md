## 3.3数学模型公式详细讲解

### 3.3.1度（Degree）

度是节点的直接关系数，表示节点与其他节点之间的关系数。公式为：

$$
D(v) = |E(v)|
$$

### 3.3.2 Betweenness Centrality

Betweenness Centrality 是节点在网络中的中心性指标，表示节点在网络中的重要性。公式为：

$$
BC(v) = \sum_{s \neq v \neq t} \frac{\sigma_{st}(v)}{\sigma_{st}}
$$

其中，$\sigma_{st}$ 是从节点 $s$ 到节点 $t$ 的总路径数，$\sigma_{st}(v)$ 是从节点 $s$ 到节点 $t$ 的路径中经过节点 $v$ 的路径数。

### 3.3.3 Closeness Centrality

Closeness Centrality 是节点在网络中的中心性指标，表示节点与其他节点的平均距离。公式为：

$$
CC(v) = \frac{N-1}{\sum_{u \neq v} d(u,v)}
$$

其中，$N$ 是节点总数，$d(u,v)$ 是节点 $u$ 和节点 $v$ 之间的距离。

# 4.具体代码实例和详细解释说明

## 4.1代码实例

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建社交网络
G = nx.Graph()

# 添加节点
G.add_node("A")
G.add_node("B")
G.add_node("C")
G.add_node("D")

# 添加边
G.add_edge("A", "B")
G.add_edge("A", "C")
G.add_edge("B", "C")
G.add_edge("C", "D")

# 计算度
degrees = nx.degree(G)
print("度:", degrees)

# 计算 Betweenness Centrality
betweenness = nx.betweenness_centrality(G)
print("Betweenness Centrality:", betweenness)

# 计算 Closeness Centrality
closeness = nx.closeness_centrality(G)
print("Closeness Centrality:", closeness)

# 绘制社交网络
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True, node_color='skyblue', edge_color='gray')
plt.show()
```

## 4.2详细解释说明

1. 首先，导入 networkx 和 matplotlib.pyplot 库。

2. 创建一个社交网络，使用 nx.Graph() 函数。

3. 添加节点，使用 G.add_node() 函数。

4. 添加边，使用 G.add_edge() 函数。

5. 计算度，使用 nx.degree() 函数。

6. 计算 Betweenness Centrality，使用 nx.betweenness_centrality() 函数。

7. 计算 Closeness Centrality，使用 nx.closeness_centrality() 函数。

8. 绘制社交网络，使用 nx.draw() 函数。
