边是图数据库中的另一个基本元素，它表示实体之间的关系。每个边都有一个唯一的ID，以及一组属性。边可以连接两个或多个节点，形成图。

## 4.具体代码实例和详细解释说明

### 4.1 图数据库的具体代码实例

#### 4.1.1 使用Python的Neo4j库实现图数据库

```python
from neo4j import GraphDatabase

# 连接图数据库
db = GraphDatabase.driver("bolt://localhost:7687", auth=("neo4j", "password"))

# 创建节点
with db.session() as session:
    node = session.run("CREATE (a:Person {name: $name})", name="Alice")
    print(node)

# 创建关系
with db.session() as session:
    session.run("MATCH (a:Person), (b:Person) WHERE a.name = $name1 AND b.name = $name2 CREATE (a)-[:KNOWS]->(b)", name1="Alice", name2="Bob")
    print(session.run("MATCH (a:Person)-[:KNOWS]->(b:Person) RETURN a.name, b.name"))

# 关闭图数据库连接
db.close()
```
