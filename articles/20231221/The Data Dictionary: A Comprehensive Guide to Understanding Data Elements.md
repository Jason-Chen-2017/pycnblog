```python
data_dictionary1 = {
    "name": "age",
    "type": "int",
    "definition": "the number of years since birth",
    "description": "age is a positive integer",
    "format": "yyyy-mm-dd",
    "range": "0-120",
    "related": "gender"

    "name": "height",
    "type": "int",
    "definition": "the number of meters",
    "description": "height is a positive integer",
    "format": "yyyy-mm-dd",
    "range": "0-250",
    "related": "weight"
}

def dictionary_merge(data_dictionary1, data_dictionary2):
    data_dictionary = data_dictionary1.copy()
    data_dictionary.update(data_dictionary2)
    return data_dictionary

merged_data_dictionary = dictionary_merge(data_dictionary1, data_dictionary2)
print(merged_data_dictionary)
```


未来，数据字典的发展方向主要体现在以下几个方面：

1. 智能化：随着人工智能技术的发展，数据字典的智能化将成为主流。例如，通过自然语言处理技术，可以将自然语言描述的数据元素自动转换为数据字典中的数据元素；通过机器学习技术，可以自动发现和提取数据字典中的关键信息，以提高数据的可用性和价值。

2. 集成：随着数据技术的发展，数据字典的集成将成为主流。例如，可以将不同数据源的数据字典集成到一个统一的数据字典中，以便于管理和应用；可以将不同领域的数据字典集成到一个统一的数据字典中，以便于协同和共享。

3. 可视化：随着可视化技术的发展，数据字典的可视化将成为主流。例如，可以将数据字典中的数据元素以图表、图形、地图等形式展示，以便于理解和分析；可以将数据字典中的关系以流程图、实体关系图等形式展示，以便于理解和管理。

1. 数据字典和数据模型的区别是什么？
