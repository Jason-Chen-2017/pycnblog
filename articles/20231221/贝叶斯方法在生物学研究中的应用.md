# 2.核心概念与联系

1. 先验概率：这是我们对某个事件发生的初始概率估计。
2. 后验概率：这是我们根据新的数据更新的概率估计。
3. 条件概率：这是某个事件发生的条件下，另一个事件发生的概率。
4. 边缘概率：这是某个事件发生的概率，不考虑其他事件。

```python
prior_A = 0.5
```

```python

```

