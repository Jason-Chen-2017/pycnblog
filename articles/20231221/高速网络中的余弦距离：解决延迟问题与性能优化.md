# 2.核心概念与联系

## 2.1 余弦距离

余弦距离是一种度量，用于衡量两个向量之间的相似性。它是通过计算两个向量之间的内积和两个向量的长度来得到的。公式如下：

$$
cos(\theta) = \frac{a \cdot b}{\|a\| \cdot \|b\|}
$$

其中，a和b是两个向量，\|a\|和\|b\|是向量a和向量b的长度，a·b是向量a和向量b的内积。余弦距离的范围为[-1, 1]，其中1表示两个向量完全相似，-1表示两个向量完全不相似，0表示两个向量完全不相关。

import numpy as np

def cosine_similarity(a, b):
    a_norm = np.linalg.norm(a)
    b_norm = np.linalg.norm(b)
    dot_product = np.dot(a, b)
    return dot_product / (a_norm * b_norm)

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
print(cosine_similarity(a, b))
