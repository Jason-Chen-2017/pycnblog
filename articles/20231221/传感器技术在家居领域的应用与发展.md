# 4. 传感器技术在家居领域的具体代码实例和解释

传感器技术在家居领域的具体代码实例主要包括：温度传感器、湿度传感器、光照传感器、空气质量传感器、人体活动传感器等。

## 4.1 温度传感器

温度传感器是用于测量室内温度的设备。例如，使用DHT11温湿度传感器获取室内温度信息的代码实例如下：

```python
import Adafruit_DHT

humidity, temperature = Adafruit_DHT.read_retry(Adafruit_DHT.DHT11, 4)
if temperature is not None:
    print('Temperature: {0:0.1f}C'.format(temperature))
else:
    print('Failed to read temperature')
```

## 4.2 湿度传感器

湿度传感器是用于测量室内湿度的设备。例如，使用DHT11温湿度传感器获取室内湿度信息的代码实例如下：

```python
import Adafruit_DHT

humidity, temperature = Adafruit_DHT.read_retry(Adafruit_DHT.DHT11, 4)
if humidity is not None:
    print('Humidity: {0:0.1f}%'.format(humidity))
else:
    print('Failed to read humidity')
```

## 4.3 光照传感器

光照传感器是用于测量室内光照强度的设备。例如，使用BH1750光照传感器获取室内光照信息的代码实例如下：

```python
import Adafruit_BMP.BMP085 as BMP085

bmp = BMP085.BMP085()
lux = bmp.read_lux()
print('Light intensity: {0:0.1f} lux'.format(lux))
```

## 4.4 空气质量传感器

空气质量传感器是用于测量室内空气质量的设备。例如，使用CCS811空气质量传感器获取室内空气质量信息的代码实例如下：

```python
import time
import board
import adafruit_ccs811

ccs = adafruit_ccs811.CCS811(board.SCL, board.SDA)

while True:
    eco2 = ccs.eco2
    tvoc = ccs.tvoc
    print('eCO2: {0:0.0f} ppm, TVOC: {1:0.0f} ppb'.format(eco2, tvoc))
    time.sleep(1)
```

## 4.5 人体活动传感器

人体活动传感器是用于测量用户运动状态的设备。例如，使用ADXL345三轴加速度计传感器获取用户运动状态信息的代码实例如下：

```python
import Adafruit_ADXL345

adxl = Adafruit_ADXL345.ADXL345()

while True:
    x, y, z = adxl.reading
    print('Acceleration: X: {0:0.0f} g, Y: {1:0.0f} g, Z: {2:0.0f} g'.format(x, y, z))
    time.sleep(0.1)
```
