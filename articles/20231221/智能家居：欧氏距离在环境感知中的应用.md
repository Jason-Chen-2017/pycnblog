# 2.核心概念与联系

## 2.1欧氏距离

欧氏距离是一种度量空间中两点之间距离的方法，它可以用来衡量两个向量之间的相似性。在环境感知技术中，我们可以将环境数据看作是一个多维向量，通过计算这些向量之间的欧氏距离，我们可以实现数据的聚类和分类。

欧氏距离的公式为：

$$
d(x, y) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)^2}
$$

其中，$x$ 和 $y$ 是两个向量，$n$ 是向量的维度，$x_i$ 和 $y_i$ 是向量的各个元素。

环境感知技术是一种通过设备和传感器实时收集环境数据的技术，它可以实时获取家居内部的环境信息，如温度、湿度、空气质量等。通过环境感知技术，我们可以实现家居的智能化管理，提高家居的安全性和节能效率。

            category = j
