# 2. 核心概念与联系

## 2.1 混合现实（Mixed Reality，MR）

混合现实是一种将现实世界和虚拟世界相结合的技术，它可以让我们在现实世界中看到虚拟世界的对象，并与虚拟世界的对象进行互动。混合现实可以分为三个主要类型：

1. 增强现实（Augmented Reality，AR）：将虚拟对象Overlay在现实世界中，让我们看到现实世界和虚拟世界的结合。例如，Google Glass等智能眼镜。
2. 虚拟现实（Virtual Reality，VR）：将现实世界中的对象替换为虚拟世界中的对象，让我们完全进入虚拟世界。例如，Oculus Rift等VR头盔。
3. 增强虚拟现实（Mixed Reality，MR）：将现实世界和虚拟世界的对象相结合，让我们看到现实世界和虚拟世界的结合。例如，Microsoft HoloLens等混合现实眼镜。

## 2.2 智能家居

智能家居是一种利用人工智能、互联网、大数据等技术的家居设备和系统，它可以让我们的生活更加舒适、高效和安全。智能家居通常包括以下几个方面：

1. 智能家电：通过网络连接家电，实现遥控、定时、智能调度等功能。
2. 智能家居安全：通过摄像头、感应器等设备，实现家庭安全监控和报警。
3. 智能家居控制：通过智能门锁、智能窗帘、智能灯泡等设备，实现家居环境的自动控制。
4. 智能家居娱乐：通过智能音响、智能电视等设备，实现音乐、视频等娱乐服务。
