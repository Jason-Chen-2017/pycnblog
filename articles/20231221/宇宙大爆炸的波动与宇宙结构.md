# 4.具体代码实例和详细解释说明
在这里，我们将提供一个简单的Python代码实例，用于模拟宇宙波动的产生和传播。这个代码实例使用了NumPy库来生成随机波动，并使用了Matplotlib库来可视化波动的传播。

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机波动
wave = np.random.randn(1000, 1000)

# 模拟宇宙的扩张过程
for _ in range(100):
    wave = np.fft.fft2(wave)
    wave = np.fft.ifft2(wave)

# 可视化波动的传播
plt.imshow(wave, cmap='hot', origin='lower')
plt.colorbar()
plt.show()
```

这个代码实例首先使用NumPy库生成了一个1000x1000的随机波动数组，然后使用了FFT（快速傅里叶变换）和IFFT（逆傅里叶变换）来模拟宇宙波动的产生和传播。最后，使用了Matplotlib库来可视化波动的传播。
