# 2.核心概念与联系

在本节中，我们将详细介绍人脸识别技术的核心概念和联系。

## 2.1 人脸图像

人脸图像是人脸识别技术的基础，它是通过摄像头或其他设备捕捉的人脸的二维或三维图像。人脸图像可以分为以下几种类型：

- 正面人脸图像：正面人脸图像是指摄像头捕捉到的人脸正面图像，它是人脸识别技术中最常见的类型。
- 侧面人脸图像：侧面人脸图像是指摄像头捕捉到的人脸侧面图像，它是人脸识别技术中较少见的类型。
- 三维人脸图像：三维人脸图像是指通过三维摄像头捕捉到的人脸三维图像，它是人脸识别技术中较新的类型。

## 2.2 人脸特征

人脸特征是人脸识别技术的关键，它是人脸图像中的一些特定特点，例如眼睛、鼻子、嘴巴等。人脸特征可以分为以下几种类型：

- 外观特征：外观特征是指人脸图像中的表面特点，例如眼睛的颜色、鼻子的大小和形状等。
- 结构特征：结构特征是指人脸图像中的空间关系，例如眼睛与鼻子之间的距离、嘴巴与鼻子之间的距离等。
- 光照特征：光照特征是指人脸图像中的光照条件，例如光线的方向、光线的强度等。

## 2.3 人脸识别算法

人脸识别算法是人脸识别技术的核心，它是用于提取人脸特征并进行比对的计算机程序。人脸识别算法可以分为以下几种类型：

- 基于特征的算法：基于特征的算法是指通过对人脸图像进行特征提取并进行比对的算法，例如主成分分析（PCA）、线性判别分析（LDA）等。
- 基于深度的算法：基于深度的算法是指通过对人脸图像进行深度学习并进行比对的算法，例如卷积神经网络（CNN）、递归神经网络（RNN）等。

## 2.4 人脸数据库

人脸数据库是人脸识别技术的支持，它是存储人脸图像和对应人脸特征的大型数据库。人脸数据库可以分为以下几种类型：

- 公开人脸数据库：公开人脸数据库是指可以公开访问的人脸数据库，例如美国国家人脸识别数据库（NFID）、中国人脸识别大数据库（CFDD）等。
- 私有人脸数据库：私有人脸数据库是指不能公开访问的人脸数据库，例如企业内部的人脸识别数据库、政府部门的人脸识别数据库等。
