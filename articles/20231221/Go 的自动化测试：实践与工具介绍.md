# 4.1 Go 语言自动化测试的具体代码实例

以下是一个 Go 语言自动化测试的具体代码实例：

```go
package main

import (
	"fmt"
	"testing"
)

func Add(a, b int) int {
	return a + b
}

func TestAdd(t *testing.T) {
	result := Add(2, 3)
	if result != 5 {
		t.Errorf("expected %d, got %d", 5, result)
	}
}
```

在上述代码中，我们首先定义了一个名为 Add 的函数，用于实现整数加法。然后，我们编写了一个名为 TestAdd 的测试函数，用于验证 Add 函数的正确性。最后，我们使用 go test 命令运行了测试。

在上述代码中，我们首先导入了 fmt 和 testing 包，这两个包分别提供了格式化输出和自动化测试的功能。然后，我们定义了一个名为 Add 的函数，用于实现整数加法。接着，我们编写了一个名为 TestAdd 的测试函数，用于验证 Add 函数的正确性。在 TestAdd 函数中，我们调用了 Add 函数，并检查了结果是否与预期一致。如果结果与预期不一致，则使用 t.Errorf 函数记录错误信息。最后，我们使用 go test 命令运行了测试。
