                 

# 1.背景介绍

自动化机器学习（AutoML）是一种通过自动化机器学习模型的过程来实现无代码的机器学习平台的技术。这种技术使得数据科学家和非专业人士可以轻松地构建、训练和部署机器学习模型，从而降低了机器学习的门槛。

自动化机器学习的发展受益于多种因素，包括数据的增长、计算资源的可用性以及机器学习算法的复杂性。在过去的几年里，数据量不断增加，计算资源变得更加便宜和可用，这使得机器学习变得更加普及。同时，机器学习算法变得越来越复杂，需要专业知识和经验才能正确地选择和调整。这使得自动化机器学习成为一个迫切的需求。

自动化机器学习的目标是自动化地选择合适的算法、调整参数、选择特征和训练模型。这使得数据科学家和非专业人士可以轻松地构建、训练和部署机器学习模型，从而降低了机器学习的门槛。
