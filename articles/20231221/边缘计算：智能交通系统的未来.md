## 3.2 数学模型公式

在交通信号灯的智能控制中，我们可以使用以下几个数学模型公式来描述决策模型：

1. 支持向量机（SVM）：
$$
\min_{w,b} \frac{1}{2}w^T w + C \sum_{i=1}^n \xi_i \\
s.t. \quad y_i(w^T \phi(x_i) + b) \geq 1 - \xi_i, \xi_i \geq 0, i=1,2,...,n
$$

2. 随机森林（RF）：
$$
\hat{f}(x) = \frac{1}{m} \sum_{j=1}^m f_j(x)
$$

其中，$w$ 是支持向量机的权重向量，$b$ 是偏置项，$C$ 是正则化参数，$\xi_i$ 是松弛变量，$y_i$ 是样本的标签，$x_i$ 是样本的特征向量，$\phi(x_i)$ 是特征映射，$m$ 是随机森林中树的数量，$f_j(x)$ 是第 $j$ 个树的预测值。

## 4.1 数据收集

我们首先需要收集交通情况的实时数据，例如车辆数量、车速、车流量等。我们可以使用Python的OpenCV库来获取摄像头的帧，并通过计算帧中的车辆数量和车速来获取交通情况的实时数据。

```python
import cv2
import numpy as np

# 获取摄像头的帧
cap = cv2.VideoCapture(0)

# 循环获取帧
while True:
    ret, frame = cap.read()
    if not ret:
        break

    # 计算车辆数量
    cars_count = np.sum(frame[:, :, 0] > 0)

    # 计算车速
    frame_width = int(cap.get(cv2.CAP_PROP_FRAME_WIDTH))
    frame_height = int(cap.get(cv2.CAP_PROP_FRAME_HEIGHT))
    fps = int(cap.get(cv2.CAP_PROP_FPS))
    car_speed = frame_width * frame_height * fps / 1000

    # 显示帧
    cv2.imshow('frame', frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# 释放摄像头
cap.release()
cv2.destroyAllWindows()
```
