1. 梯度下降法代码实例：

```python
import numpy as np

def gradient_descent(f, grad_f, x0, lr=0.01, max_iter=1000):
    x = x0
    for i in range(max_iter):
        grad = grad_f(x)
        x = x - lr * grad
        if i % 100 == 0:
            print('iter: {}, f(x): {}'.format(i, f(x)))
    return x
```

上述代码实例中，`f`是目标函数，`grad_f`是目标函数的梯度，`x0`是初始解，`lr`是学习率，`max_iter`是最大迭代次数。

```python
import numpy as np

def newton_method(f, grad_f, hess_f, x0, lr=0.01, max_iter=1000):
    x = x0
    for i in range(max_iter):
        hess = hess_f(x)
        dx = -np.linalg.inv(hess) * grad_f(x)
        x = x - lr * dx
        if i % 100 == 0:
            print('iter: {}, f(x): {}'.format(i, f(x)))
    return x
```

上述代码实例中，`f`是目标函数，`grad_f`是目标函数的梯度，`hess_f`是目标函数的二阶导数（Hessian矩阵），`x0`是初始解，`lr`是学习率，`max_iter`是最大迭代次数。
