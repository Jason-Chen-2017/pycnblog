# 2.核心概念与联系

零代码技术是一种基于配置和拖拽的软件开发和集成方法，它的核心概念包括：

1. 模块化设计：零代码技术将软件功能分解为多个模块，每个模块都具有明确的功能和接口。这样，用户可以通过简单地组合和配置这些模块，实现软件开发和集成。

2. 数据流和连接器：零代码技术通过数据流和连接器来实现不同模块之间的数据交换和对接。数据流是用于传输数据的通道，连接器则是用于连接数据流和模块的桥梁。

3. 配置和拖拽：零代码技术通过配置和拖拽来实现软件开发和集成。用户可以通过拖拽模块到工作区域，并通过配置参数和属性来实现功能的扩展和定制。

4. 无需编程知识：零代码技术不需要用户具备编程知识，因此更易于学习和使用。

通过这些核心概念，零代码技术实现了跨业务的融合和对接，从而帮助企业和组织更好地满足客户需求，提高业务效率，降低成本。
