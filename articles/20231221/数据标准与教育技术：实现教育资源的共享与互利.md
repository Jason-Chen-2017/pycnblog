### 1.4.1 数据标准化示例

假设我们有一些课程数据，格式如下：

```
{
  "course_id": "1",
  "course_name": "数学",
  "course_format": "PDF"
}
```

我们需要将这些数据转换为XML格式。以下是一个简单的Python代码实例，展示了如何实现这个转换：

```python
import xml.etree.ElementTree as ET

def course_to_xml(course):
    course_element = ET.Element("course")
    course_element.set("id", course["course_id"])
    course_element.set("name", course["course_name"])
    course_element.set("format", course["course_format"])
    return course_element

course = {
  "course_id": "1",
  "course_name": "数学",
  "course_format": "PDF"
}

course_xml = course_to_xml(course)
ET.dump(course_xml)
```

### 1.4.2 数据清洗示例

假设我们有一些课程数据，格式如下：

```
[
  {
    "course_id": "1",
    "course_name": "数学",
    "course_format": "PDF"
  },
  {
    "course_id": "2",
    "course_name": "英语",
    "course_format": "unknown"
  }
]
```

我们需要将这些数据进行清洗，将不明确的格式转换为确定的格式。以下是一个简单的Python代码实例，展示了如何实现这个清洗：

```python
def clean_course_format(course):
    formats = {
        "PDF": "text/pdf",
        "PPT": "application/vnd.ms-powerpoint",
        "MP4": "video/mp4"
    }
    course["course_format"] = formats.get(course["course_format"], "unknown")
    return course

courses = [
  {
    "course_id": "1",
    "course_name": "数学",
    "course_format": "PDF"
  },
  {
    "course_id": "2",
    "course_name": "英语",
    "course_format": "unknown"
  }
]

cleaned_courses = [clean_course_format(course) for course in courses]
print(cleaned_courses)
```

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

def encrypt_course_data(course_data):
    key = get_random_bytes(16)
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(course_data.encode())
    return {
        "key": key,
        "ciphertext": ciphertext,
        "tag": tag
    }

course_data = [
  {
    "course_id": "1",
    "course_name": "数学",
    "course_format": "PDF"
  }
]

encrypted_course_data = encrypt_course_data(b"".join(course_data))
print(encrypted_course_data)
```
