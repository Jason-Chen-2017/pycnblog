## 3.2 数据清洗的数学模型公式

在本节中，我们将详细讲解数据清洗中的数学模型公式。

### 3.2.1 数据缺失值处理

1. 平均值填充：
$$
x_{fill} = \bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i
$$

2. 中位数填充：
$$
x_{fill} = \text{中位数}
$$

3. 模数填充：
$$
x_{fill} = \text{mod}(x_i)
$$

### 3.2.2 数据重复值处理

1. 去重：
$$
x_{unique} = \{x_1, x_2, \dots, x_n\}
$$

2. 筛选：
$$
x_{filter} = \{x_i | x_i \neq x_j, \forall j \neq i\}
$$

### 3.2.3 数据错误值处理

1. 异常值检测：
$$
x_{outlier} = \{x_i | |x_i - \bar{x}| > k \times \text{IQR}\}
$$

2. 极值检测：
$$
x_{extreme} = \{x_i | x_i < a \text{ 或 } x_i > b\}
$$

### 3.2.4 数据类型转换

1. 整型转浮点型：
$$
x_{float} = x_i \times 1.0
$$

2. 字符转整型：
$$
x_{int} = \text{int}(x_i)
$$

### 3.2.5 数据单位转换

1. 秒转毫秒：
$$
x_{ms} = x_i \times 1000
$$

2. 米转英尺：
$$
x_{ft} = x_i \times 0.3048
$$

### 3.2.6 数据格式转换

1. CSV转TXT：
$$
x_{txt} = \text{CSV} \rightarrow \text{TXT}
$$

2. TXT转JSON：
$$
x_{json} = \text{TXT} \rightarrow \text{JSON}
$$

## 4.1 数据清理

### 4.1.1 数据缺失值处理

```python
import numpy as np
import pandas as pd

# 创建一个数据集
data = {'name': ['Alice', 'Bob', 'Charlie'],
        'age': [25, np.nan, 30],
        'gender': ['F', 'M', 'M']}
df = pd.DataFrame(data)

# 检测数据中的缺失值
print(df.isnull().sum())

# 填充数据中的缺失值
df['age'].fillna(df['age'].mean(), inplace=True)
print(df)
```

### 4.1.2 数据重复值处理

```python
import numpy as np
import pandas as pd

# 创建一个数据集
data = {'name': ['Alice', 'Bob', 'Alice'],
        'age': [25, 30, 35]}
df = pd.DataFrame(data)

# 检测数据中的重复值
print(df.duplicated().sum())

# 删除数据中的重复值
df.drop_duplicates(inplace=True)
print(df)
```

### 4.1.3 数据错误值处理

```python
import numpy as np
import pandas as pd

# 创建一个数据集
data = {'name': ['Alice', 'Bob', 'Charlie'],
        'age': [25, 'error', 30]}
df = pd.DataFrame(data)

# 检测数据中的错误值
print(df.isnull().sum())

# 修正数据中的错误值
df['age'] = df['age'].replace('error', np.nan)
df['age'].fillna(df['age'].mean(), inplace=True)
print(df)
```
