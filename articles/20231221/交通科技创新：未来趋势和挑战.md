其中，$I(x,y)$表示原图像的灰度值，$S(x,y)$表示滤波后的灰度值，$f(x,y)$表示原图像的灰度值，$C(x,y)$表示滤波核，$w_k(x,y)$表示滤波核的权重，$d_k(x,y)$和$e_k(x,y)$表示滤波核在x和y方向的偏移量。

$$
$$
s.t. \sum_{t=1}^{T} \sum_{n=1}^{N} a_{tn} \times x_{tn} \leq A_t

$$
其中，$x_{tn}$表示在时间t和路段n的流量，$c_{tn}$表示在时间t和路段n的成本，$a_{tn}$表示在时间t和路段n的容量，$A_t$表示在时间t的总容量，$b_{tn}$表示在时间t和路段n的需求，$B_t$表示在时间t的总需求。

```python
from scipy.optimize import linprog

def traffic_planning(demand, capacity):
    A = np.ones(len(demand))
    b = demand
    A_eq = capacity
    b_eq = np.zeros(len(capacity))
    bounds = [(0, 1) for _ in range(len(demand))]
    constraints = [{'type': 'ineq', 'fun': lambda x: np.sum(A * x) - A_eq},
                   {'type': 'eq', 'fun': lambda x: b - A * x},
                   {'type': 'ineq', 'fun': lambda x: b_eq - A_eq * x}]
    result = linprog(b, A_ub=A, bounds=bounds, constraints=constraints, method='highs')
    return result.x
```
