# 2.核心概念与联系

## 2.1 什么是禁忌搜索

禁忌搜索（Tabu Search，TS）是一种基于本地搜索的优化算法，它可以用于解决复杂的优化问题。TS通过在搜索空间中移动搜索点，逐步找到最优解。在搜索过程中，TS使用一个禁忌列表（Tabu List）来记录已经访问过的搜索点，以避免回溯和循环搜索。

## 2.2 禁忌搜索在图像处理中的应用

禁忌搜索在图像处理中的应用主要包括以下几个方面：

1. 图像分割：通过禁忌搜索优化图像分割的目标函数，以实现自动分割和识别图像中的区域和对象。
2. 边缘检测：通过禁忌搜索优化边缘检测的目标函数，以实现自动识别图像中的边缘和线条。
3. 图像压缩：通过禁忌搜索优化图像压缩的目标函数，以实现高效的图像压缩和传输。

**Q：什么是禁忌值？**

A：禁忌值是指在禁忌列表中的值，它表示一个已经访问过的搜索点。在搜索过程中，如果生成的候选搜索点的值与禁忌列表中的值相同，则不能被选中。

**Q：如何设置禁忌列表的大小？**

A：禁忌列表的大小可以根据问题的复杂性和搜索空间的大小来设置。一般来说，较大的禁忌列表可以避免回溯和循环搜索，但也会增加内存占用和计算开销。

**Q：如何设置终止条件？**

A：终止条件可以是搜索时间、迭代次数等。一般来说，可以根据问题的特点和应用场景来设置终止条件。
