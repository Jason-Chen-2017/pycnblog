# 2.核心概念与联系

## 2.1 查准率（Precision）
查准率是一种度量方法，用于衡量在一个给定类别中选出正确的数量。在医疗领域中，查准率可以用来衡量一个医疗诊断或治疗方法的准确性。查准率的计算公式为：

$$
Precision = \frac{True Positives}{True Positives + False Positives}
$$

其中，True Positives（TP）表示正确预测的数量，False Positives（FP）表示错误预测的数量。查准率的范围在0到1之间，值越接近1，表示预测的准确性越高。

## 2.2 查全率（Recall）
查全率是另一种度量方法，用于衡量在一个给定类别中选出了多少正确的数量。在医疗领域中，查全率可以用来衡量一个医疗诊断或治疗方法的效果。查全率的计算公式为：

$$
Recall = \frac{True Positives}{True Positives + False Negatives}
$$

其中，True Positives（TP）表示正确预测的数量，False Negatives（FN）表示错误预测的数量。查全率的范围在0到1之间，值越接近1，表示预测的效果越高。

## 2.3 查准率与查全率的联系
查准率和查全率是两个相互独立的指标，它们可以通过以下公式得到：

$$
F1 = 2 \times \frac{Precision \times Recall}{Precision + Recall}
$$

其中，F1是F1分数，是查准率和查全率的调和平均值。F1分数的范围在0到1之间，值越接近1，表示预测的准确性和效果越高。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来解释查准率和查全率的计算过程。假设我们有一个医疗诊断方法，它对于100个患者进行了诊断，其中：

- 正确诊断的患者数量为80个
- 错误诊断的患者数量为10个
- 未诊断的患者数量为10个

我们可以通过以下代码来计算查准率和查全率：

```python
true_positives = 80
false_positives = 10
false_negatives = 10

precision = true_positives / (true_positives + false_positives)
recall = true_positives / (true_positives + false_negatives)

print("Precision:", precision)
print("Recall:", recall)
```

输出结果：

```
Precision: 0.8947368421052632
Recall: 0.8947368421052632
```

从输出结果中可以看出，查准率和查全率的值都是0.8947，表示医疗诊断方法的准确性和效果较高。
