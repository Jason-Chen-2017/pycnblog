## 1.4 具体代码实例和详细解释说明

### 1.4.1 代码实例

```python
import math

def entropy(probabilities):
    entropy = 0
    for probability in probabilities:
        entropy -= probability * math.log2(probability)
    return entropy

probabilities = [0.3, 0.3, 0.4]
print("熵权:", entropy(probabilities))
```

### 1.4.2 详细解释说明

在这个代码实例中，我们首先导入了 `math` 模块，并定义了一个名为 `entropy` 的函数，该函数用于计算熵权。在函数中，我们首先初始化一个变量 `entropy` 为0，然后遍历 `probabilities` 列表中的每个概率值，并计算其对应的熵权。最后，我们打印了熵权的结果。

在这个代码实例中，我们使用了 `math.log2` 函数来计算自然对数的二对数，并将其乘以概率值。通过这样的计算，我们可以得到系统的熵权。
