```dockerfile

EXPOSE 3000

COPY app.js /app.js
COPY package.json /package.json

RUN npm install

CMD node app.js
```

在这个 Docker 文件中，我们使用了 Aerospike 官方镜像，并将应用程序的代码和依赖项复制到镜像中。然后，我们使用 `npm install` 命令安装依赖项，并使用 `node app.js` 命令启动应用程序。

## 4.2 Kubernetes 部署文件示例

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: aerospike-deployment
spec:
  replicas: 3
  selector:
    matchLabels:
      app: aerospike
  template:
    metadata:
      labels:
        app: aerospike
    spec:
      containers:
      - name: aerospike
        image: your-docker-image-url
        ports:
        - containerPort: 3000
```

在这个 Kubernetes 部署文件中，我们定义了一个名为 `aerospike-deployment` 的部署，包括 3 个副本。我们使用了 `selector` 字段来匹配标签为 `app: aerospike` 的 Pod，并使用 `template` 字段定义了 Pod 的模板。在 `containers` 字段中，我们定义了一个名为 `aerospike` 的容器，使用了我们之前构建的 Docker 镜像，并暴露了端口 3000。
