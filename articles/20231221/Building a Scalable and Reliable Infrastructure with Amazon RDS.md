                 

# 1.背景介绍

Amazon RDS (Relational Database Service) 是一种托管的数据库服务，可以帮助您在 AWS 上轻松部署、运行和可扩展关系数据库。它提供了完全托管的数据库服务，使您能够专注于构建应用程序，而不需要担心数据库的运行和维护。在本文中，我们将讨论如何使用 Amazon RDS 构建可扩展且可靠的数据库基础设施。
