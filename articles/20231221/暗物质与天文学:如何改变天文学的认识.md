# 4.具体代码实例和详细解释说明

在这里，我们将提供一个简单的Python代码实例，用于计算星系之间的引力。

```python
import math

def gravity(m1, m2, r):
    G = 6.67430e-11
    return G * m1 * m2 / r**2

m1 = 1e12  # mass of star 1 in kg
m2 = 1e12  # mass of star 2 in kg
r = 1e20  # distance between stars in meters

gravity_force = gravity(m1, m2, r)
print(f"The gravitational force between the two stars is {gravity_force} N")
```

这个代码首先导入了`math`模块，然后定义了一个`gravity`函数，用于计算两个星体之间的引力。接着，我们设定了星体1和星体2的质量和距离，并使用`gravity`函数计算引力。最后，我们打印出结果。
