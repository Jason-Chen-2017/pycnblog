from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB
from sklearn.metrics import accuracy_score

data = pd.read_csv('environment_sentiment.csv')

### 4.2关键词提取实例

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.feature_extraction.text import TfidfTransformer

# 数据加载
data = pd.read_csv('environment_sentiment.csv')

# 数据预处理
vectorizer = TfidfVectorizer(stop_words='english')
X = vectorizer.fit_transform(data['text'])

# 关键词提取
transformer = TfidfTransformer()
X_tfidf = transformer.fit_transform(X)

# 关键词提取
vocabulary = vectorizer.get_feature_names_out()
weights = X_tfidf.toarray()
keywords = [(word, weight) for word, weight in zip(vocabulary, weights.flatten())]
keywords = sorted(keywords, key=lambda x: x[1], reverse=True)

# 打印关键词
for keyword, weight in keywords[:10]:
    print(keyword, weight)
```

在上述代码中，我们首先使用TfidfVectorizer对舆情数据进行了特征提取。然后使用TfidfTransformer对特征向量进行了转换，得到关键词和其对应的权重。最后，将关键词和权重排序，并打印出前10个关键词。

### 4.3社交网络分析实例

```python
import networkx as nx
import matplotlib.pyplot as plt

# 数据加载
data = pd.read_csv('environment_sentiment.csv')

# 数据预处理
vectorizer = TfidfVectorizer(stop_words='english')
X = vectorizer.fit_transform(data['text'])

# 网络建立
G = nx.Graph()
for i in range(len(X)):
    G.add_node(i, text=data['text'][i])

# 添加边
for i in range(len(X)):
    for j in range(i+1, len(X)):
        if nx.has_edge(G, i, j):
            continue
        similarity = nx.cosine_similarity(X[i], X[j])
        if similarity > 0.5:
            G.add_edge(i, j, weight=similarity)

# 网络可视化
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True, node_color='skyblue', node_size=3000, font_size=10)
plt.show()
```

在上述代码中，我们首先使用TfidfVectorizer对舆情数据进行了特征提取。然后使用networkx库构建了一个社交网络，其中节点表示舆情数据，边表示相似性。最后，使用matplotlib库对社交网络进行可视化。
