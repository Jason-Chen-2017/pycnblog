# 2.核心概念与联系

## 2.1 欧氏距离的定义

在欧几里得几何中，欧氏距离是两点之间的最短距离，它可以用来衡量两个点之间的距离。在n维空间中，给定两个点A=(a1, a2, ..., an)和B=(b1, b2, ..., bn)，它们之间的欧氏距离可以通过以下公式计算：

$$
d(A, B) = \sqrt{(a_1 - b_1)^2 + (a_2 - b_2)^2 + ... + (a_n - b_n)^2}
$$

## 2.2 欧氏距离在图像处理中的应用

在图像处理中，欧氏距离可以用来衡量两个像素点之间的距离，从而进行像素值的差异判断。例如，在图像压缩和噪声除去等方面，我们可以通过计算像素点之间的欧氏距离来衡量它们之间的相似性，从而进行相似性判断和压缩处理。

## 2.3 欧氏距离在图像识别中的应用

在图像识别中，欧氏距离可以用来衡量两个特征向量之间的距离，从而进行特征匹配和分类识别。例如，在人脸识别等方面，我们可以通过计算两个人脸特征向量之间的欧氏距离来判断它们是否属于同一个人，从而进行人脸识别。
