数学模型公式详细讲解：

在边界计算中，我们可以使用以下数学模型公式来描述边缘设备上的数据处理和分析：

1. 滤波算法：滤波算法可以用来减少数据噪声，从而提高数据质量。滤波算法的数学模型公式如下：

$$
y(t) = \frac{1}{T}\int_{t-T}^{t} x(\tau) d\tau
$$

其中，$y(t)$ 是滤波后的数据，$x(t)$ 是原始数据，$T$ 是滤波窗口。

2. 聚类算法：聚类算法可以用来分类数据，从而进行数据挖掘。聚类算法的数学模型公式如下：

$$
\min \sum_{i=1}^{k} \sum_{x \in C_i} d(x,\mu_i)
$$

其中，$k$ 是聚类数量，$C_i$ 是聚类$i$，$\mu_i$ 是聚类$i$的中心，$d(x,\mu_i)$ 是数据$x$与聚类中心$\mu_i$之间的距离。

# 定义MQTT回调函数
def on_connect(client, userdata, flags, rc):
    print("Connected with result code "+str(rc))

# 定义MQTT消息回调函数
def on_message(client, userdata, msg):
    print(msg.topic+" "+str(msg.payload))

# 创建MQTT客户端
client = mqtt.Client()

# 设置回调函数
client.on_connect = on_connect
client.on_message = on_message

# 连接MQTT服务器
client.connect("localhost", 1883, 60)

# 启动MQTT客户端
client.loop_start()

# 发布MQTT消息
client.publish("edge/data", np.random.rand(1).tolist())
