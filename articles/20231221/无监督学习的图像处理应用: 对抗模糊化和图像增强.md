无监督学习是一种机器学习方法，它不需要预先标注的数据来训练模型。相反，它利用未标注的数据来自动发现数据中的结构和模式。在图像处理领域，无监督学习算法可以用于图像分类、聚类、降噪、增强和对抗模糊化等任务。在本文中，我们将讨论无监督学习在对抗模糊化和图像增强应用中的表现。

## 4.2图像增强代码实例
```python
import numpy as np
import cv2
import matplotlib.pyplot as plt

# 加载原始图像

# 调整亮度
image_bright = cv2.convertScaleAbs(image, alpha=1.5, beta=0)

# 调整对比度
image_contrast = cv2.equalizeHist(image)

# 调整色彩
image_color = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)

# 显示原始图像和增强图像
plt.subplot(1, 3, 1)
plt.imshow(image, cmap='gray')
plt.title('Original Image')

plt.subplot(1, 3, 2)
plt.imshow(image_bright, cmap='gray')
plt.title('Brightness Enhanced Image')

plt.subplot(1, 3, 3)
plt.imshow(image_contrast, cmap='gray')
plt.title('Contrast Enhanced Image')

plt.subplot(1, 3, 4)
plt.imshow(image_color, cmap='RGB')
plt.title('Color Enhanced Image')

plt.show()
```
在上述代码中，我们首先加载原始图像，然后使用OpenCV库调整亮度、对比度和色彩。最后，我们使用matplotlib库显示原始图像和增强图像。
