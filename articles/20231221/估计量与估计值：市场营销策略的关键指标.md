```python
import pandas as pd
import numpy as np
from scipy.stats import zscore

# 加载客户价值数据
data = pd.read_csv('customer_value.csv')

# 数据清洗
data = data.drop_duplicates()
data = data.fillna(0)

# 数据分析
mean = data['value'].mean()
median = data['value'].median()
std = data['value'].std()
quartile = data['value'].quantile([0.25, 0.75])

# 客户价值分段
bins = [0, quartile[0], quartile[1], np.inf]
labels = ['低价值客户', '中价值客户', '高价值客户']
data['price_segment'] = pd.cut(data['value'], bins=bins, labels=labels)

# 客户价值筛选
high_value_customers = data[data['price_segment'] == '高价值客户']
loyal_customers = data[data['loyalty'] == '忠诚客户']
recommend_customers = data[data['recommend'] == '推荐客户']

# 客户价值优化
high_value_customers['value'] = high_value_customers['value'] * 1.1
loyal_customers['value'] = loyal_customers['value'] * 1.1
recommend_customers['value'] = recommend_customers['value'] * 1.1
```

在这个代码实例中，我们首先使用 pandas 库加载客户价值数据，然后使用 drop_duplicates() 函数删除重复数据，使用 fillna() 函数填充缺失数据。接着，我们使用 scipy.stats.zscore 库计算客户价值的平均值、中位数、方差、分位数等。然后，我们使用 pd.cut() 函数将客户价值分成三个段，分别为低价值客户、中价值客户、高价值客户。接着，我们使用筛选条件对客户进行筛选，筛选出高价值客户、忠诚客户、推荐客户等。最后，我们对客户价值进行优化，将高价值客户、忠诚客户、推荐客户的价值进行加权增加。
