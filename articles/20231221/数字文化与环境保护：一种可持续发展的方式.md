```python

data = pd.read_csv('environment.csv')
```

```python

data = data[data['temperature'] < 50]
data = data[data['humidity'] < 100]
data = data[data['pressure'] > 900]
```

## 4.3 数据集成

我们将使用Python的scikit-learn库来进行数据集成。首先，我们需要将多个数据集合并在一起：

```python
from sklearn.datasets import concatenate

data2 = pd.read_csv('environment2.csv')
data3 = pd.read_csv('environment3.csv')

data = concatenate([data1, data2, data3])
```

## 4.4 数据挖掘

我们将使用Python的scikit-learn库来进行数据挖掘。首先，我们需要选择一个数据挖掘算法，如决策树：

```python
from sklearn.tree import DecisionTreeClassifier

clf = DecisionTreeClassifier()
clf.fit(data[['temperature', 'humidity', 'pressure']], data['air_quality'])
```
