## 4.1 代码实例

```python
from pachyderm.dsl import Pipeline
from pachyderm.dsl import Read
from pachyderm.dsl import Write

pipeline = Pipeline("my_pipeline")

input_data = Read(
    pipeline=pipeline,
    source="input_data",
    file_glob="*.csv"
)

output_data = Write(
    pipeline=pipeline,
    source=input_data,
    file_glob="*.csv",
    destination="output_data"
)

pipeline.run()
```

在这个代码实例中，我们首先导入了Pachyderm的DSL和API。然后，我们创建了一个名为`my_pipeline`的数据处理流程。接下来，我们使用`Read`函数来读取数据，`Write`函数来写入数据。最后，我们使用`run`函数来执行数据处理流程。
