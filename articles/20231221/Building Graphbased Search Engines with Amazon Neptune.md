## 4.1 创建图形数据库

首先，我们需要创建一个图形数据库。我们可以使用 Amazon Neptune 提供的 API 来创建图形数据库。以下是创建图形数据库的代码示例：

```python
import boto3

client = boto3.client('neptune')

response = client.create_graph(
    GraphName='my-graph',
    Engine='neptune',
    Schema='my-schema'
)
```

在这个代码示例中，我们使用 `boto3` 库来创建一个名为 `my-graph` 的图形数据库。我们指定了 `Engine` 为 `neptune`，表示使用 Neptune 引擎。我们还指定了 `Schema` 为 `my-schema`，表示使用自定义的数据模式。

## 4.2 加载图形数据

接下来，我们需要加载图形数据到图形数据库中。我们可以使用 Amazon Neptune 提供的 API 来加载图形数据。以下是加载图形数据的代码示例：

```python
import json

with open('data.json', 'r') as f:
    data = json.load(f)

client = boto3.client('neptune')

for node in data['nodes']:
    response = client.run_graph_query(
        GraphName='my-graph',
        Query='CREATE (n:Node {id: {id}, name: {name}})'.format(**node)
    )

for edge in data['edges']:
    response = client.run_graph_query(
        GraphName='my-graph',
        Query='CREATE ()-[:{relationship}]->()'.format(**edge)
    )
```

在这个代码示例中，我们首先从一个 JSON 文件中加载图形数据。我们使用 `json` 库来加载 JSON 文件。然后，我们使用 `boto3` 库来加载图形数据到图形数据库中。我们使用 `run_graph_query` 方法来执行创建节点和边的查询。

## 4.3 执行图形查询

最后，我们需要执行图形查询来获取满足条件的节点和边。我们可以使用 Amazon Neptune 提供的 API 来执行图形查询。以下是执行图形查询的代码示例：

```python
client = boto3.client('neptune')

response = client.run_graph_query(
    GraphName='my-graph',
    Query='MATCH (n)-[r]->(m) WHERE n.name = {name} RETURN n, r, m'.format(name='Alice')
)

for record in response['data']['records']:
    node = record['fields']['n']
    relationship = record['fields']['r']
    node2 = record['fields']['m']
    print('Node:', node)
    print('Relationship:', relationship)
    print('Node2:', node2)
```

在这个代码示例中，我们使用 `run_graph_query` 方法来执行图形查询。我们使用 `MATCH` 子句来查找满足条件的节点和边。我们使用 `WHERE` 子句来筛选满足条件的节点。我们使用 `RETURN` 子句来获取满足条件的节点和边。
