## 3.1 微波辐射的基本概念和公式
微波辐射是指频率在10GHz到300GHz之间的电磁波。在宇宙背景辐射中，微波辐射的主要特征是它的温度分布和波长分布。微波辐射的温度分布可以通过以下公式来描述：

$$
I(\lambda) = \frac{2hc^2}{\lambda^5} \frac{1}{e^{hc/\lambda kT} - 1}
$$

其中，$I(\lambda)$ 是辐射强度，$h$ 是弦理常数，$c$ 是光速，$\lambda$ 是波长，$T$ 是温度，$k$ 是布尔常数。

## 3.2 宇宙背景辐射的传播模型
宇宙背景辐射的传播模型主要包括两部分：一是光线的传播模型，二是光辐射的散射和吸收模型。光线的传播模型可以通过辐射transfer函数来描述，光辐射的散射和吸收模型可以通过散射和吸收 coefficent 来描述。

$$
\frac{dI}{d\tau} = -I + S
$$

其中，$I$ 是辐射强度，$S$ 是源强度，$\tau$ 是光辐射的传播距离。

## 3.3 宇宙光学的模拟方法
宇宙光学的模拟方法主要包括两种：一是光线追踪模拟，二是辐射transfer函数模拟。光线追踪模拟是通过跟踪光线的传播过程来模拟宇宙光学现象，它的主要优点是能够准确地描述光线的传播过程，但其主要缺点是计算量较大。辐射transfer函数模拟则是通过使用辐射transfer函数来描述光辐射的传播过程，它的主要优点是计算量较小，但其主要缺点是无法准确地描述光线的传播过程。

```python
import numpy as np
from astropy.cosmology import Planck13

def compute_cmb_intensity(l):
    h = 6.62607004e-34  # J s
    c = 299792458  # m/s
    k = 1.38064852e-23  # J/K
    T_CMB = 2.725  # K

    wl = l * 1e-3  # convert to micron
    I = 2 * h * c**2 / (wl**5) * 1e**23  # erg/s/cm^2/sr/micron
    I /= np.exp(h * c / (wl * k * Planck13.T)) - 1
    return I

def compute_cmb_power_spectrum(l):
    I = compute_cmb_intensity(l)
    return I**2

l = np.linspace(1e-4, 1e-1, 1000)
cmb_power_spectrum = compute_cmb_power_spectrum(l)

import matplotlib.pyplot as plt

plt.plot(l, cmb_power_spectrum)
plt.xlabel('Wavenumber ($\mu m^{-1}$)')
plt.ylabel('Power Spectrum')
plt.show()
```
