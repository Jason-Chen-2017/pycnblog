# 4.具体代码实例和详细解释说明
以下是一个具体的数据地图代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成一组随机数据
data = np.random.normal(loc=0.0, scale=1.0, size=1000)

# 计算数据的均值和标准差
mean = np.mean(data)
std = np.std(data)

# 绘制数据分布图
plt.hist(data, bins=30, density=True)
plt.plot([mean - 3*std, mean + 3*std], [0.05, 0.95], 'r')
plt.show()
```

上述代码首先导入了 numpy 和 matplotlib.pyplot 这两个库，然后生成了一组随机数据。接着计算了数据的均值和标准差，并使用 matplotlib.pyplot 库绘制了数据分布图。最后使用了红色直线表示了数据的均值和标准差。
