# 2. 核心概念与联系

## 2.1 光伏材料的基本概念

光伏材料是指能够将光能转化为电能的材料，常见的光伏材料有单晶体光伏材料（如单晶光伏板）和多晶光伏材料（如多晶光伏板）。光伏材料的主要组成部分是光伏细胞，光伏细胞由光伏材料组成，其中包括阳极、阴极和光伏材料等部分。光伏细胞的工作原理是通过光伏材料将光能转化为电能，从而实现电力输出。

## 2.2 光伏材料与光伏电池的关系

光伏材料是光伏电池的核心组成部分，光伏电池是将光伏能量转化为电能的设备。光伏电池由光伏材料、电路板、护理材料等组成，其中光伏材料是光伏电池的核心部分，负责将光能转化为电能。因此，光伏材料的效率对光伏电池的效率具有重要影响。
