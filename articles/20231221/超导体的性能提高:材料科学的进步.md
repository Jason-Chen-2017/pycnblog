# 2. 核心概念与联系
# 2.1 超导体的基本概念

超导体是指在零温度下可以导电的材料。在超导体中，电子的运动不受抗阻，因此电阻为零。超导体的发现可以追溯到1911年，当时荷兰物理学家赫尔曼·杜姆（Heike Kamerlingh Onnes）在实验中发现了氢化锂（LiH）的超导性。随后，杜姆还发现了氢（H2）、氢化碳（CH）、氢化氧化物（CO）等其他材料的超导性。

超导体的性能特征包括：

1. 零电阻性：在零温度下，超导体可以导电，但电阻为零。
2. 高电导率：超导体的电导率远高于普通导体。
3. 低损耗：由于超导体的电阻为零，因此在导电过程中不会产生损耗。
4. 高温稳定性：超导体的超导性只在非常低的温度下才会出现，因此需要维持低温环境以保持其超导性。

# 2.2 超导体的类型

根据超导体的性质，可以分为以下几类：

1. 类一超导体：这类超导体的超导性仅在特定的磁场下才会出现，如氢化锂（LiH）。
2. 类二超导体：这类超导体的超导性不受磁场的影响，如氢化铅（Pb）。
3. 类三超导体：这类超导体的超导性仅在特定的压力下才会出现，如氢化锂酸钙（Li2O·Pb）。

# 2.3 超导体的应用

超导体在各种领域中有广泛的应用，如：

1. 超导磁共振成像（MRI）：通过将超导电线放入人体内部，可以产生强大的磁场，从而实现对内部组织的细致扫描。
2. 超导电机：由于超导体的低损耗和高电导率，因此可以用于制作高效的电机，如磁 levitation train 。
