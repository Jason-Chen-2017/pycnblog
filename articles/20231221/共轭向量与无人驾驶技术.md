## 3.1 共轭向量的定义

给定两个向量 $a$ 和 $b$，它们的共轭向量 $r$ 可以通过以下公式计算：

$$
r = a \times b
$$

其中，$a \times b$ 表示向量 $a$ 和 $b$ 的外积。

## 3.2 共轭向量的计算

计算共轭向量的步骤如下：

1. 首先，计算向量 $a$ 和 $b$ 的外积。外积的计算公式为：

$$
a \times b = (a_y b_z - a_z b_y) \mathbf{i} + (a_z b_x - a_x b_z) \mathbf{j} + (a_x b_y - a_y b_x) \mathbf{k}
$$

其中，$\mathbf{i}$、$\mathbf{j}$ 和 $\mathbf{k}$ 是三个基向量，$a_x$、$a_y$、$a_z$ 和 $b_x$、$b_y$、$b_z$ 是向量 $a$ 和 $b$ 的分量。

2. 将外积结果赋值给共轭向量 $r$。

在无人驾驶技术中，共轭向量可以用于解决以下问题：

import numpy as np

def cross_product(a, b):
    return np.cross(a, b)

def corres_vector(a, b):
    return cross_product(a, b)
