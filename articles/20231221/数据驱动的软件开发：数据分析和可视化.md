## 4.2 数据清洗
我们将使用Python的pandas库来清洗和处理数据。

```python
import pandas as pd

# 读取CSV文件
data = pd.read_csv('sales.csv')

# 将日期列转换为日期类型
data['date'] = pd.to_datetime(data['date'])

# 将日期列设置为索引
data.set_index('date', inplace=True)
```
