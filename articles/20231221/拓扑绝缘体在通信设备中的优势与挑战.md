## 4.1 代码实例：拓扑绝缘体信号传输线路设计

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置线路长度和电容
L = 1000
C = 1e-9

# 设置输入信号
t = np.linspace(0, 1, 1000)
Vin = np.sin(2 * np.pi * 1e6 * t)

# 计算信号传输速度
v = 1 / np.sqrt(L * C)

# 计算信号传输距离
distance = v * t

# 计算输出信号
Vout = Vin * np.exp(-distance / C * v)

# 绘制输入输出信号波形
plt.figure()
plt.plot(t, Vin, label='Input Signal')
plt.plot(t, Vout, label='Output Signal')
plt.legend()
plt.show()
```
