# 2.核心概念与联系
无序单项式向量空间（Randomized Sparse Vectorspace, RSVS）是一种基于随机生成的稀疏向量的空间表示方法，它可以在高维空间中构建稀疏向量，从而减少计算复杂度和提高识别准确率。RSVS的核心概念包括：

1. 向量空间：向量空间是一个包含向量的线性空间，向量之间可以进行加法和数乘运算。在人脸识别中，向量空间可以用来表示人脸特征。

2. 稀疏向量：稀疏向量是指在高维空间中，只有很少的元素具有非零值。稀疏向量可以减少计算量，提高识别效率。

3. 随机生成：RSVS使用随机生成的方法来创建稀疏向量，这可以减少计算复杂度，提高识别准确率。

4. 高维空间：人脸特征通常存在高维空间中，因此需要使用高维空间的表示方法来表示人脸特征。

Q：无序单项式向量空间在实际应用中的局限性有哪些？

A：无序单项式向量空间在实际应用中的局限性主要包括：

1. 计算复杂度较高：由于无序单项式向量空间使用了高维空间和稀疏向量，因此计算复杂度较高，可能影响实时性能。

3. 对于部分人脸特征不易识别：无序单项式向量空间可能对于部分人脸特征（如老年人或皮肤颜色不均匀的人脸）的识别效果不佳。
