## 1.2 核心概念与联系

在进行二次型解析之前，我们需要了解以下几个核心概念：

1. 二次方程组：二次方程组是指包含两个不等式的数学方程，通常表示为：

$$
\begin{cases}
    ax + by + c = 0 \\
    dx + ey + f = 0
\end{cases}
$$

其中，$a, b, c, d, e, f$ 是已知常数。

2. 二次型：二次型是指一个形如 $ax^2 + bx + c$ 的方程，其中 $a, b, c$ 是已知常数。

3. 解析几何：解析几何是一门数学分支，主要研究的是几何图形在坐标系中的表示和性质。

接下来，我们将讨论如何通过二次型解析来解决二次方程组的问题。

       ax + by + c = 0 \\
