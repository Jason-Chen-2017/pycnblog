### 4.1.1 物联网算法实例

在智能水网中，我们可以使用物联网技术实现水资源设施的远程监控、控制。以下是一个简单的物联网算法实例：

```python
import time
from pymodbus.client.sync import ModbusTcpClient

def read_registers(address, count):
    client = ModbusTcpClient("192.168.1.100")
    client.connect()
    data = client.read_holding_registers(address, count)
    client.close()
    return data.registers

def main():
    while True:
        water_level = read_registers(0x0001, 2)
        print("水位：", water_level[0], water_level[1])
        time.sleep(5)

if __name__ == "__main__":
    main()
```
