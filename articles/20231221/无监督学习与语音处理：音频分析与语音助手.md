## 4.1 语音信号处理

### 4.1.1 滤波

滤波是一种常见的语音信号处理方法，它通过对语音信号进行滤波来消除噪声和干扰。常见的滤波方法包括低通滤波、高通滤波、带通滤波、带阻滤波等。以下是一个使用Python的Numpy库实现低通滤波的代码示例：

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成一段语音信号
fs = 16000
T = np.linspace(0, 1, fs, endpoint=False)
f = 440
x = np.sin(2 * np.pi * f * T)

# 设置滤波器的中心频率和带宽
fc = 220
bw = 100

# 计算滤波器的权重
alpha = 1.0 / (1 + (T / bw - fc / bw) ** 2)

# 进行低通滤波
y = np.convolve(x, alpha[::-1])

# 绘制原始语音信号和滤波后的语音信号
plt.plot(T, x, label='Original')
plt.plot(T, y, label='Filtered')
plt.legend()
plt.show()
```
