## 2.核心概念与联系

### 2.1量子光学基础

量子光学是量子物理学和光学的结合，研究光子在量子态下的行为特性。量子光学的核心概念包括：

- 光子：光子是光的基本单位，是光的量子化形式。光子具有波特性和粒子特性，可以看作是一种能量的量子化。
- 光子的生成、传输、检测、处理等。
- 光子纠缠：光子纠缠是指两个或多个光子之间的相互作用，使其量子态相互依赖。

### 2.2纠缠的核心概念

纠缠是量子信息处理中的一个重要概念，它可以实现多个量子比特之间的相互作用，提高量子计算和量子通信的性能。纠缠的核心概念包括：

- 量子比特：量子比特是量子计算中的基本单位，它可以表示两个状态：0和1。量子比特的特点是可以存储和处理多个状态，具有超越经典计算的处理能力。
- 纠缠状态：纠缠状态是指多个量子比特之间的相互依赖关系，使它们的状态相互影响。
- 纠缠操作：纠缠操作是指在量子比特之间进行的相互作用操作，使它们形成纠缠状态。

### 2.3纠缠在光子级别的应用

```python
def prepare_photon_states(photons):
    return [prepare_photon_state(photon) for photon in photons]
```
