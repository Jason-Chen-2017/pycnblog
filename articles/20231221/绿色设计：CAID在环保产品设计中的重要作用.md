### 4.1 代码实例

```python
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
from sklearn.cluster import KMeans

# 加载数据
data = pd.read_csv('env_product_data.csv')

# 数据预处理
scaler = StandardScaler()
data_scaled = scaler.fit_transform(data)

# 数据分析
pca = PCA(n_components=2)
data_pca = pca.fit_transform(data_scaled)

# 聚类分析
kmeans = KMeans(n_clusters=3)
data_clusters = kmeans.fit_predict(data_pca)

# 结果输出
print('聚类结果：', data_clusters)
```

### 4.2 详细解释说明

这个代码实例主要包括以下几个步骤：

1. 加载数据：从 CSV 文件中加载环保产品设计相关的数据，并将其存储在变量 `data` 中。
2. 数据预处理：使用 `StandardScaler` 进行数据标准化处理，将数据缩放到 [-1, 1] 范围内。
3. 数据分析：使用主成分分析（PCA）进行数据分析，将原始数据降维到两个维度，并将结果存储在变量 `data_pca` 中。
4. 聚类分析：使用 K 均值聚类算法进行聚类分析，将数据分为三个簇，并将结果存储在变量 `data_clusters` 中。
