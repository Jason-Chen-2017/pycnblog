# 4.具体代码实例和详细解释说明

以求导法则的常数法则为例，我们来看一个具体的代码实例：

```python
import sympy as sp

x = sp.Symbol('x')
f = 3*x**2 + 2*x + 1
c = 2

df = sp.diff(f*c, x)
print(df)
```

输出结果：

```
6*x + 2
```

这个代码实例中，我们首先导入了sympy库，然后定义了待求导函数f和常数c。接着，我们使用sp.diff()函数进行求导，得到了结果6*x + 2。
