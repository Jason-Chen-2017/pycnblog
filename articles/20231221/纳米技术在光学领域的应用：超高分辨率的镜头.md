## 4.1 代码实例

我们以一个简单的纳米光学镜头设计为例，来详细解释其设计和制造过程。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义纳米光学镜头的参数
focal_length = 50  # 焦距（以米为单位）
aperture_diameter = 10  # 光圈直径（以米为单位）
sensor_width = 24  # 传感器宽度（以毫米为单位）
sensor_height = 16  # 传感器高度（以毫米为单位）

# 计算镜头的分辨率
resolution = focal_length / aperture_diameter
print("镜头的分辨率：", resolution, "像素/毫米")

# 计算镜头的视场
field_of_view = np.arctan(sensor_width / (2 * focal_length)) * 180 / np.pi
print("镜头的视场：", field_of_view, "度")

# 绘制镜头的光圈曲线
theta = np.linspace(0, field_of_view, 100)
r = aperture_diameter / (2 * np.tan(theta * np.pi / 180))
plt.plot(r, theta)
plt.xlabel("光圈直径（米）")
plt.ylabel("视场（度）")
plt.title("纳米光学镜头的光圈曲线")
plt.show()
```

## 4.2 代码解释

上述代码首先导入了numpy和matplotlib.pyplot两个库，用于进行数值计算和绘图。然后，我们定义了纳米光学镜头的参数，包括焦距、光圈直径、传感器宽度和传感器高度。接下来，我们计算了镜头的分辨率和视场，并将其打印出来。最后，我们绘制了镜头的光圈曲线，以可视化镜头的光学性能。
