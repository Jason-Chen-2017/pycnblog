# 4. 具体代码实例和详细解释说明

以下是一个简单的Python代码实例，用于实现基础设施布局的优化。

```python
import numpy as np
from scipy.optimize import linprog

# 目标函数
def objective_function(x):
    return np.sum(x)

# 约束条件
def constraint1(x):
    return np.sum(x * 0.5) - 10

def constraint2(x):
    return np.sum(x * 1.5) - 20

# 优化模型
model = linprog(objective_function, bounds=[(0, 1)], constraints=[{'type': 'ineq', 'fun': constraint1},
                                                                    {'type': 'ineq', 'fun': constraint2}])

# 求解
result = model.solve()

# 输出结果
print("基础设施布局：", result.x)
```

在这个例子中，我们使用了线性规划算法来优化基础设施布局。目标函数是基础设施的数量，约束条件是基础设施的总消耗。通过优化模型，我们可以得到一个满足约束条件的基础设施布局，同时最小化基础设施的消耗。
