# 2.核心概念与联系
## 2.1 椭圆曲线
椭圆曲线是一个二次曲线，定义为：$$y^2 = x^3 + ax + b$$，其中a和b是常数。椭圆曲线是一个有限数量的点集，可以用于生成一个群。椭圆曲线群是一个非常有趣的数学结构，它具有许多令人印象深刻的数学性质。

## 2.2 点加法
在椭圆曲线上，我们可以定义一种特殊的点加法。给定两个点P和Q，我们可以计算它们的和S，记作P+Q。点加法的规则是：

- 如果P≠Q，则S在曲线上，且P+Q=S。
- 如果P=Q，则S在曲线上，且P+P=S。
- 如果P=O（椭圆曲线上的起点），则P+Q=Q。

## 2.3 椭圆曲线密码学
椭圆曲线密码学是一种基于椭圆曲线的密码学技术，主要包括以下几个方面：

- 椭圆曲线加密：使用椭圆曲线生成密钥对，进行加密和解密操作。
- 数字签名：使用椭圆曲线生成密钥对，进行数字签名和验证操作。

1. 随机选择一个整数k，使得k是n的倍数。
2. 计算x=k^2%n和y=k*(x^3+a*x+b)%n。
3. 返回点(x,y)。

## 3.3 椭圆曲线加密
椭圆曲线加密主要包括以下步骤：

1. 生成密钥对：使用椭圆曲线参数生成一个私钥pri和公钥pub。
2. 加密：选择一个随机整数k，计算密文c=(pub+k*G)，其中G是椭圆曲线上的基点。
3. 解密：使用私钥pri，计算明文m=(c-pri*G)。

## 3.4 椭圆曲线数字签名
椭圆曲线数字签名主要包括以下步骤：

1. 生成密钥对：使用椭圆曲线参数生成一个私钥pri和公钥pub。
2. 签名：使用私钥pri，计算签名sig=(m+pri*G)。
3. 验证：使用公钥pub，计算验证值ver=(sig-m+pub)。如果ver=G，则签名是有效的。
