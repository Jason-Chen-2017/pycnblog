## 4.1 智能水泵

### 4.1.1 传感器数据收集

```python
import time
import requests

# 设置传感器数据收集地址
sensor_data_url = "http://sensor.example.com/data"

# 设置智能水泵控制地址
water_pump_control_url = "http://water_pump.example.com/control"

while True:
    # 发送请求获取传感器数据
    response = requests.get(sensor_data_url)
    sensor_data = response.json()

    # 解析传感器数据
    temperature = sensor_data["temperature"]
    humidity = sensor_data["humidity"]
    soil_humidity = sensor_data["soil_humidity"]

    # 根据传感器数据调整水量
    if temperature < 25 and humidity < 60 and soil_humidity < 30:
        water_volume = 100
    elif temperature < 25 and humidity < 60 and soil_humidity >= 30:
        water_volume = 75
    elif temperature >= 25 and humidity < 60 and soil_humidity < 30:
        water_volume = 50
    else:
        water_volume = 0

    # 发送请求调整水泵水量
    requests.post(water_pump_control_url, json={"water_volume": water_volume})

    # 等待一段时间后再次获取传感器数据
    time.sleep(60)
```
