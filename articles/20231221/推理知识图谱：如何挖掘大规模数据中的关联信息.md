接下来，我们将创建一个简单的知识图谱。我们将使用`pandas`库来创建一个数据框，其中包含实体、关系和实例。

```python
import pandas as pd

data = {
    'entity': ['Alice', 'Bob', 'Charlie'],
    'relation': ['teacher', 'student', 'teacher'],
    'instance': ['Alice', 'Bob', 'Charlie']
}

df = pd.DataFrame(data)
```

现在，我们将使用`networkx`库来构建一个图，其中包含实体和关系。

```python
import networkx as nx

G = nx.DiGraph()

for entity, relation, instance in zip(df['entity'], df['relation'], df['instance']):
    G.add_edge(entity, instance, relation=relation)
```

最后，我们将使用`networkx`库来查询图中的实体和关系。

```python
def query(entity, relation):
    return [instance for instance, attr in G.in_edges(entity, data=True) if attr['relation'] == relation]

students = query('Alice', 'student')
teachers = query('Alice', 'teacher')
```

在这个简单的示例中，我们创建了一个知识图谱，其中包含三个实体（Alice、Bob和Charlie）和三个关系（teacher、student和teacher）。我们使用`networkx`库来构建一个图，并使用`query`函数来查询图中的实体和关系。
