## 3.3 数学模型公式详细讲解

点互信息可以用以下公式表示：

$$
PII(A,B) = \int_{t_1}^{t_2} A(t) \cdot B(t) dt
$$

其中，$PII(A,B)$ 表示点互信息，$A(t)$ 和 $B(t)$ 分别表示数据点 A 和 B 在时刻 t 的位置信息。$t_1$ 和 $t_2$ 分别表示计算点互信息的时间范围。

```python
import numpy as np

def PII(A, B, t1, t2):
    dt = np.arange(t1, t2)
    A_t = A * dt
    B_t = B * dt
    PII = np.trapz(A_t * B_t, dt)
    return PII

A = np.array([1, 2, 3])
B = np.array([4, 5, 6])
t1 = 0
t2 = 10

PII_result = PII(A, B, t1, t2)
print("点互信息结果：", PII_result)
```
