### 4.1整数乘法

整数乘法是元素乘法中最基本的实现之一。我们可以使用位级乘法来实现高效的整数乘法。以下是一个简单的整数乘法实现：

```python
def multiply_int(a, b):
    result = 0
    while b > 0:
        if b & 1:
            result += a
        a <<= 1
        b >>= 1
    return result
```

在上述代码中，我们使用了位运算来实现整数乘法。通过不断左移和右移位，我们可以计算出两个数的乘积。这种实现方式具有较高的效率，因此在大数据场景下可以实现高效的计算。
