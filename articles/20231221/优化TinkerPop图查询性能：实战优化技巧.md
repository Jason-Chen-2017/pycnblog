# 2.核心概念与联系

在了解如何优化TinkerPop图查询性能之前，我们需要了解一些核心概念。这些概念包括：

- **节点（Node）**：节点是图形数据中的基本元素。它们表示实体，如人、地点或物品。
- **边（Edge）**：边是连接节点的连接。它们表示关系，如人与人的关系或地点之间的距离。
- **图（Graph）**：图是节点和边的集合。它们可以表示复杂的数据结构，例如社交网络或地理信息系统。
- **TinkerPop**：TinkerPop是一种用于处理图形数据的查询语言。它提供了一种简洁的方式来表示和查询图形数据。

在优化TinkerPop图查询性能时，我们需要了解一些核心算法原理。这些算法包括：

- **图遍历**：图遍历是一种用于访问图中所有节点和边的算法。它可以通过多种方式实现，例如深度优先搜索（DFS）或广度优先搜索（BFS）。
- **中心性度量**：中心性度量是一种用于衡量节点在图中的重要性的度量。它可以通过多种方式计算，例如度（Degree）、 Betweenness Centrality 或 Closeness Centrality。
- **图匹配**：图匹配是一种用于找到图中满足特定条件的子图的算法。它可以通过多种方式实现，例如最大独立集（Maximum Independent Set）或最小覆盖集（Minimum Vertex Cover）。
