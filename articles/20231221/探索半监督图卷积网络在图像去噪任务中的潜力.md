# 3.3数学模型公式详细讲解
假设我们有一个包含 $n$ 个像素的图像，有标签数据集 $D_l = \{ (x_i, y_i) \}_{i=1}^{|D_l|}$，无标签数据集 $D_u = \{ x_j \}_{j=1}^{|D_u|}$。其中 $x_i$ 和 $x_j$ 是图像数据，$y_i$ 是有标签数据的标签。

我们定义一个 Half-SQUARE 网络 $f_{\theta}$，其中 $\theta$ 是网络参数。网络的目标是最小化以下损失函数：

$$
L(\theta) = \alpha L_{l}(\theta) + \beta L_{u}(\theta)
$$

其中 $L_{l}(\theta)$ 和 $L_{u}(\theta)$ 分别是有标签和无标签数据的损失函数，$\alpha$ 和 $\beta$ 是正规化参数。

有标签损失函数 $L_{l}(\theta)$ 可以使用交叉熵损失函数表示为：

$$
L_{l}(\theta) = -\frac{1}{|D_l|} \sum_{i=1}^{|D_l|} [y_i \log f_{\theta}(x_i) + (1 - y_i) \log (1 - f_{\theta}(x_i))]
$$

无标签损失函数 $L_{u}(\theta)$ 可以使用对偶对数损失函数表示为：

$$
L_{u}(\theta) = -\frac{1}{|D_u|} \sum_{j=1}^{|D_u|} \log \frac{1}{Z_j} \sum_{c=1}^{C} e^{f_{\theta}(x_j)^T w_c}
$$

其中 $Z_j$ 是归一化因子，$C$ 是类别数量。

通过优化上述损失函数，我们可以得到 Half-SQUARE 网络的参数 $\theta$。
