## 4.1 数据质量

以下是一个数据清洗的Python代码实例：

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 数据检查

```

在上述代码中，我们首先使用pandas库加载数据，然后通过dropna()函数来删除缺失值，接着通过apply()函数来修复错误值，最后通过astype()函数来转换数据类型。
