## 4.1代码实例

以下是一个简单的Table Store数据库迁移代码实例：

```python
import pandas as pd
import psycopg2

# 连接源数据库
source_conn = psycopg2.connect(host='source_host', port='source_port', dbname='source_db', user='source_user', password='source_password')
source_cur = source_conn.cursor()

# 连接目标数据库
target_conn = psycopg2.connect(host='target_host', port='target_port', dbname='target_db', user='target_user', password='target_password')
target_cur = target_conn.cursor()

# 读取源数据库中的数据
source_data = pd.read_sql('SELECT * FROM source_table', source_conn)

# 将源数据分成多个小的表
table_size = 10000
chunks = [source_data[i:i+table_size] for i in range(0, source_data.shape[0], table_size)]

# 导入目标数据库中的数据
for chunk in chunks:
    chunk.to_sql('target_table', con=target_conn, if_exists='append', index=False)

# 关闭数据库连接
source_cur.close()
source_conn.close()
target_cur.close()
target_conn.close()
```

## 4.2详细解释说明

这个代码实例中，我们首先连接到源数据库和目标数据库，然后读取源数据库中的数据。接着，我们将源数据分成多个小的表，并将这些表一次性地导入到目标数据库中。最后，我们关闭数据库连接。
