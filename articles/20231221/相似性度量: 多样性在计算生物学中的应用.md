## 4.具体代码实例和详细解释说明

### 4.1代码实例

```python
import numpy as np

# 定义两个样本
x = np.array([1, 2, 3])
y = np.array([4, 5, 6])

# 计算两个样本的长度
norm_x = np.linalg.norm(x)
norm_y = np.linalg.norm(y)

# 计算两个样本的内积
inner_product = np.dot(x, y)

# 计算相似性度量
similarity = inner_product / (norm_x * norm_y)

print("相似性度量:", similarity)
```

1. 首先，我们导入了 numpy 库，因为我们需要使用 numpy 库来计算向量的长度和内积。
2. 然后，我们定义了两个样本 `x` 和 `y`。
3. 接下来，我们计算了两个样本的长度。长度可以通过 numpy 库的 `linalg.norm()` 函数来计算。
4. 然后，我们计算了两个样本的内积。内积可以通过 numpy 库的 `dot()` 函数来计算。
5. 最后，我们计算了相似性度量。相似性度量可以通过将两个样本的长度和内积计算结果插入相似性度量的公式中，并计算得到相似性度量值。
