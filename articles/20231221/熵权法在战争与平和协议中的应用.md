熵权法的核心算法原理是基于信息论的熵计算。熵可以通过以下公式计算：

$$
H(X) = -\sum_{i=1}^{n} P(x_i) \log_2 P(x_i)
$$

其中，$H(X)$ 是熵值，$n$ 是事件数，$P(x_i)$ 是事件 $x_i$ 的概率。

```python
import numpy as np
import scipy.stats as stats

def entropy(probabilities):
    return -np.sum(probabilities * np.log2(probabilities))

def weighted_entropy(probabilities, weights):
    return np.sum(weights * entropy(probabilities))

def calculate_entropy(relations, weights):
    probabilities = np.array(relations) / np.sum(relations)
    return weighted_entropy(probabilities, weights)

relations = [10, 5, 3, 2]

print("Entropy:", entropy)
```
