```python

    {'feature1': 1, 'feature2': 2, 'feature3': 3},

    {'feature1': 3, 'feature2': 1, 'feature3': 2}
]
```

```python
from sklearn.preprocessing import MinMaxScaler

# 模型构建
svd = TruncatedSVD(n_components=2)

# 模型训练
svd.fit(data_scaled)

# 模型测试
data_reduced = svd.transform(data_scaled)
```
