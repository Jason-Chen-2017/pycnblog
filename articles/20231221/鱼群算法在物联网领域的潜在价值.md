# 2.核心概念与联系
# 2.1.鱼群算法的基本概念

鱼群算法是一种基于自然鱼群行为的优化算法，它是一种分布式的、动态的、自适应的优化算法。鱼群算法的核心思想是通过模拟鱼群中的鱼的行为，来解决复杂的优化问题。

鱼群算法的主要组成部分包括：

1. 鱼群：鱼群是算法中的主要组成部分，它由多个鱼组成。每个鱼都有自己的位置、速度和方向。
2. 鱼的行为：鱼的行为包括 swimming、separation、alignment 和 cohesion。这些行为分别对应于鱼在水中的移动、避免相互碰撞、向同伴靠近和向群体中心靠近。
3. 目标函数：目标函数是算法中的主要组成部分，它用于评估鱼群的优化效果。目标函数可以是任意的优化问题，包括物流、生产、金融等等。
