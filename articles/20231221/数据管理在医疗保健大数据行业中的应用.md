2. 数据存储：

```python
import sqlite3
import pandas as pd

conn = sqlite3.connect('medical_data.db')
cursor = conn.cursor()

sql = 'CREATE TABLE IF NOT EXISTS medical_data (id INTEGER PRIMARY KEY, name TEXT, age INTEGER, gender TEXT)'
cursor.execute(sql)

data = {'name': ['John', 'Jane'], 'age': [25, 30], 'gender': ['M', 'F']}
df = pd.DataFrame(data)
df.to_sql('medical_data', conn, if_exists='append', index=False)

conn.close()
```

3. 数据清洗：

```python
import pandas as pd

data = {'name': ['John', 'Jane', None], 'age': [25, 30, None], 'gender': ['M', 'F', None]}
df = pd.DataFrame(data)

df['name'].fillna('Unknown', inplace=True)
df['age'].fillna(df['age'].mean(), inplace=True)
df['gender'].fillna('Unknown', inplace=True)

print(df)
```

4. 数据分析：

```python
from sklearn.linear_model import LinearRegression

X = df[['age', 'gender']]
y = df['name']

model = LinearRegression()
model.fit(X, y)

print(model.score(X, y))
```

5. 数据安全：

```python
import hashlib

key = b'my_secret_key'
iv = hashlib.sha256(key).digest()

plaintext = b'my_secret_message'
ciphertext = hashlib.aes.encrypt(plaintext, key, iv)

print(ciphertext)
```

6. 数据隐私保护：

```python
import re

data = {'name': ['John', 'Jane'], 'age': [25, 30], 'gender': ['M', 'F']}
df = pd.DataFrame(data)

df['name'] = df['name'].apply(lambda x: re.sub(r'[A-Za-z]', '*', x))
df['age'] = df['age'].apply(lambda x: '*' * (len(str(x)) - 2))
df['gender'] = df['gender'].apply(lambda x: '*' * len(x))

print(df)
```
