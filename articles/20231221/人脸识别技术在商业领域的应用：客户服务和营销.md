# 2.核心概念与联系

## 2.1 人脸识别技术的基本概念

1. 一对一匹配：将一张人脸与数据库中的其他人脸进行比较，以确定是否匹配。
2. 一对多匹配：将一张人脸与数据库中的多个人脸进行比较，以确定其中的匹配。
3. 多对多匹配：将多个人脸之间进行比较，以确定它们之间的关系。
