# 2.核心概念与联系
在讨论保护世界遗产之前，我们需要了解一些关键的概念和联系。这些概念包括世界遗产、文化多样性、建筑遗产、保护和传承等。

## 2.1 世界遗产
世界遗产是指全球范围内具有 outstanding value 的建筑物、地点和区域。这些遗产可以是文化、历史、科学或艺术等方面的表现。例如，世界遗产列表包括古埃及的埃及文化遗产、罗马帝国的遗产、中国的古迹和古迹群等。

## 2.2 文化多样性
文化多样性是指不同文化之间的差异和多样性。这些差异可以是语言、宗教、传统、艺术、建筑等方面的表现。文化多样性是人类社会的一个重要特征，也是世界遗产的一个重要组成部分。

## 2.3 建筑遗产
建筑遗产是指具有历史、文化或艺术价值的建筑物。这些建筑物可以是古老的、历史的或具有特殊功能的。例如，建筑遗产可以是古埃及的金字塔、罗马的古墓、中国的古寺庙等。

## 2.4 保护和传承
保护和传承是保护世界遗产的两个关键词。保护是指对遗产的物理和法律保护，例如建筑物的修复、保护区域的设立等。传承是指对遗产的文化和知识传承，例如教育、研究、展览等。
