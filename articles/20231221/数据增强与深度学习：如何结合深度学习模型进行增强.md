## 3.1 数据生成

数据生成是通过随机生成新的数据样本来扩展数据集的方法。例如，通过噪声添加、图像旋转、翻转等。数据生成的数学模型公式如下：

$$
y = x + n
$$

$$
x' = rotate(x, \theta)
$$

$$
x'' = flip(x)
$$

其中，$y$ 是生成后的数据样本，$x$ 是原始数据样本，$n$ 是噪声，$\theta$ 是旋转角度，$x'$ 是旋转后的数据样本，$x''$ 是翻转后的数据样本。

## 3.2 数据变换

数据变换是通过对现有数据进行修改来创建新的数据样本的方法。例如对图像进行灰度转换、调整对比度、锐化等。数据变换的数学模型公式如下：

$$
x' = g(x)
$$

其中，$x'$ 是变换后的数据样本，$x$ 是原始数据样本，$g(x)$ 是变换函数。

## 3.3 数据混合

数据混合是通过将多个数据样本组合在一起来创建新的数据样本的方法。例如图像剪切和粘贴、颜色替换等。数据混合的数学模型公式如下：

$$
x' = \alpha x_1 + (1 - \alpha) x_2
$$

其中，$x'$ 是混合后的数据样本，$x_1$ 和 $x_2$ 是原始数据样本，$\alpha$ 是混合系数。

## 3.4 数据剪裁

数据剪裁是通过对图像进行剪裁操作来创建新的数据样本的方法。例如随机剪裁、中心剪裁等。数据剪裁的数学模型公式如下：

$$
x' = crop(x, (u, v, w, h))
$$

其中，$x'$ 是剪裁后的数据样本，$x$ 是原始数据样本，$(u, v, w, h)$ 是剪裁区域的坐标和大小。

## 4.1 数据生成

```python
import numpy as np
import cv2
import random

def add_noise(image):
    noise = np.random.normal(0, 0.05, image.shape)
    return image + noise

def rotate(image, angle):
    return cv2.rotate(image, cv2.ROTATE_90_CLOCKWISE)

def flip(image):
    return cv2.flip(image, 1)

image = add_noise(image)
image = rotate(image, 45)
image = flip(image)
```

## 4.2 数据变换

```python
import cv2
import random

def grayscale(image):
    return cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

def adjust_contrast(image, factor):
    alpha = np.clip(factor - 0.5, 0, 1)
    return cv2.convertScaleAbs(image, alpha=alpha)

def sharpen(image):
    kernel = np.array([[-1, -1, -1], [-1, 9, -1], [-1, -1, -1]])
    return cv2.filter2D(image, -1, kernel)

image = grayscale(image)
image = adjust_contrast(image, 1.5)
image = sharpen(image)
```
