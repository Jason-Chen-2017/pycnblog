## 2.1 Table Store 的基本概念

Table Store 是一种基于列存储的数据库技术，它将数据按列存储在磁盘上，而不是传统的行存储方式。这种存储方式可以减少磁盘的随机访问，提高数据的压缩率，从而提高查询性能。
