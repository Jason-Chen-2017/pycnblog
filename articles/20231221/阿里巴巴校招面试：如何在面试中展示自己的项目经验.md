## 4.1 代码实例
以下是一个简单的代码实例，它实现了一个简单的排序算法——冒泡排序。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

## 4.2 代码解释
上述代码实现了一个简单的冒泡排序算法。冒泡排序算法的基本思想是通过多次遍历数组，将较大的元素逐步向后移动，直到数组排序为止。

在上述代码中，`bubble_sort`函数接受一个整数数组`arr`作为输入，并返回一个排序后的数组。函数中的两个`for`循环分别实现了数组的遍历和元素的交换。在内层`for`循环中，如果当前元素`arr[j]`大于下一个元素`arr[j+1]`，则交换它们的值。这个过程会一直持续到数组排序为止。
