### 4.1.1 随机生成

以下是一个简单的随机生成音效的Python代码实例：

```python
import random
import pygame

# 加载音频片段库
sound_library = ['sound1.wav', 'sound2.wav', 'sound3.wav']

# 随机选择音频片段
selected_sound = random.choice(sound_library)

# 加载选定的音频片段
sound = pygame.mixer.Sound(selected_sound)

# 调整音量和时长
sound.set_volume(0.5)
sound.set_duration(500)

# 播放音效
sound.play()
```

在这个例子中，我们首先导入了`random`和`pygame`库。然后，我们加载了一组音频片段，并随机选择了一个音频片段。接着，我们加载选定的音频片段，调整了音量和时长，并播放了音效。
