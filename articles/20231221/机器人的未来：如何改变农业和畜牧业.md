## 4.1 农机机器人的代码实例

以下是一个简单的农机机器人的代码实例，用于完成农业生产中的种植任务。

```python
import numpy as np
import cv2

def plant_seeds(field, seeds):
    for seed in seeds:
        field[seed['y'], seed['x']] = 1
    return field

def water_plants(field, water_level):
    for y in range(field.shape[0]):
        for x in range(field.shape[1]):
            if field[y, x] == 1:
                field[y, x] += water_level
    return field

def harvest_crops(field):
    harvested_crops = []
    for y in range(field.shape[0]):
        for x in range(field.shape[1]):
            if field[y, x] == 2:
                harvested_crops.append((y, x))
                field[y, x] = 0
    return harvested_crops

# 初始化种植场地
field = np.zeros((100, 100), dtype=int)

# 种植种子
seeds = [{'x': 50, 'y': 50}, {'x': 50, 'y': 60}]
field = plant_seeds(field, seeds)

# 给植物水
water_level = 0.1
field = water_plants(field, water_level)

# 收获作物
harvested_crops = harvest_crops(field)
print(harvested_crops)
```

在这个代码实例中，我们首先导入了numpy和cv2库，然后定义了三个函数：`plant_seeds`、`water_plants`和`harvest_crops`。`plant_seeds`函数用于种植种子，`water_plants`函数用于给植物水，`harvest_crops`函数用于收获作物。然后，我们初始化了一个100x100的种植场地，种植了两个种子，给植物水，并收获了作物。

## 4.2 畜牧机器人的代码实例

以下是一个简单的畜牧机器人的代码实例，用于完成畜牧生产中的饲养任务。

```python
import numpy as np

def feed_animals(animals, feed_level):
    for animal in animals:
        animal['weight'] += feed_level
    return animals

def monitor_health(animals):
    healthy_animals = []
    for animal in animals:
        if animal['health'] == 'healthy':
            healthy_animals.append(animal)
    return healthy_animals

# 初始化畜牧动物
animals = [{'name': 'cow1', 'weight': 500, 'health': 'healthy'},
           {'name': 'cow2', 'weight': 500, 'health': 'healthy'}]

# 饲养畜牧动物
feed_level = 10
animals = feed_animals(animals, feed_level)

# 监控畜牧动物的健康状况
healthy_animals = monitor_health(animals)
print(healthy_animals)
```

在这个代码实例中，我们首先导入了numpy库，然后定义了两个函数：`feed_animals`和`monitor_health`。`feed_animals`函数用于饲养畜牧动物，`monitor_health`函数用于监控畜牧动物的健康状况。然后，我们初始化了两头健康的畜牧动物，饲养了畜牧动物，并监控了畜牧动物的健康状况。
