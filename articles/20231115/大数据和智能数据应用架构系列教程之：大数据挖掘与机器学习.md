                 

# 1.背景介绍


## 数据挖掘(Data Mining)概述
数据挖掘，也称为数据分析、数据 mining，是从大量数据中提取有价值的信息，并对其进行整合、评估和挖掘的过程。数据挖掘的目的是发现隐藏在数据中的模式、关联、规律以及知识。主要利用计算机的方法实现，利用信息检索、分类、聚类、预测等方法从海量的数据中发现有效的信息，为企业制定决策提供指导。数据挖掘技术是基于海量数据的统计分析和模式识别的一种快速而有效的手段。数据挖掘的研究和开发目前处于蓬勃发展阶段。它已经成为高科技、新兴产业、金融服务领域的基础性技术和应用工具。数据挖掘技术是人工智能的一个重要组成部分，是智能的数据处理的基础。同时，随着互联网、移动计算平台和云计算技术的发展，数据挖掘正在成为更多领域的基础性技术。
## 大数据概述
大数据是指分布式存储、处理、分析和挖掘海量数据的技术和方法。数据数量已超过了传统数据库管理系统能够处理的范围。如今，在世界范围内，有越来越多的人把注意力放在互联网、移动通讯和物联网上收集的数据及其分析处理上。这些数据存在众多的特征：大量、高维、异构、动态、海量等。因此，数据探索和挖掘技术必须具备高效处理大数据、提升数据分析的速度和准确率，成为当前数据中心和产业界所需的新型技术。此外，为了利用数据挖掘技术解决复杂的问题，需要大数据平台能够支持高度并行化的处理能力、能够有效地存储和处理海量数据。
## 智能数据应用架构概述
智能数据应用架构（IDRAC）是指一种用于设计、部署、运行和管理智能数据应用的技术框架。它包括大数据系统、大数据分析平台、大数据处理平台、应用层平台、应用组件库和服务体系结构等。IDRAC包含了一系列标准组件，可实现大数据应用的各个层面，包括数据采集、存储、处理、分析、展示、输出和监控。IDRAC可以根据不同应用场景、数据规模和服务级别进行配置优化，保证应用的高可用性、扩展性、弹性和安全性。根据应用需求，通过适配器接口对接各种第三方数据源和服务，实现数据统一、共享和管理。IDRAC的目标是通过对大数据分析平台、大数据处理平台、应用组件库和应用服务体系结构的集成，实现数据驱动业务的自动化、智能化、可观测性和稳定性。
