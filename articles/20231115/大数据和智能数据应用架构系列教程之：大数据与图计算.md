                 

# 1.背景介绍



1999年，IBM公司发布了第一张PC机，它完全是一个小型计算机，处理能力只有几百万条指令，并且很贵。直到20世纪90年代初期，互联网的普及才让大数据分析成为现实，人们逐渐产生了对大数据的需求，并开始从事相关的研究工作。

1990年，美国斯坦福大学的加布里埃尔教授在其博士论文中提出了“超链接网络”（hyperlink network）这一概念。1994年，他在一篇题为《The Structure and Function of Complex Networks》的文章中首次提出用“节点”、“边”和“属性”三个基本要素构成复杂网络。

1997年，Google首次公开了一种叫做PageRank的算法。2002年，Facebook也推出了一种名为Social Graph的应用。随后，英国剑桥大学的Guy Amato教授，还有斯坦福大学的Alexa Ranking，都陆续提出了基于图论的新型搜索引擎算法。这些改进让大数据分析和挖掘变得更加“聪明”、“神秘”。

2006年，谷歌宣布收购图搜（GraphSearch），这是一家专门针对海量图数据进行查询的公司，旨在帮助用户快速找到感兴趣的信息。

时至今日，由于大数据爆炸性增长、云端计算技术飞速发展、智能手机、互联网的迅速普及，以及计算机硬件性能不断提升等因素的影响，使得数据处理、分析和挖掘正在经历一场“信息革命”。正如Wikipedia文章所言，“大数据时代已经开启，下一个五年或十年将是巨大的财富和社会变革的关键时刻。”因此，对于如何有效地运用大数据技术来解决复杂的实际问题，以及如何利用大数据进行高效的决策支持，以及如何对海量图数据进行快速检索，有着极为重要的意义。

2014年，图计算领域的两位大牛——Google Brain团队成员沙利文和Facebook员工马拉松（Mark Salgos）联合推出了一套名为“facebook的页相似性建模”（graph similarity modeling for facebook pages）的研究报告。这项研究旨在通过对多种关系类型的组合和拓扑结构的探索，建立起更有效的大规模图数据相似度计算的方法，并用于建议用户关注的好友。

2016年，由云栖社区提供赞助，阿里巴巴集团主办的“图计算与推荐系统研讨会”在北京举行。本系列教程的主要讲者就将围绕以上突破性技术发展方向，为读者呈现一系列的有关图计算和推荐系统的应用场景。其中包括但不限于电影推荐、婚恋网站匹配、基于地理位置的交通推荐、微博、微信群组等方面。

# 2.核心概念与联系

首先，我们需要了解一下图计算的基本概念和术语。

1.图（Graph）：图是由顶点（vertex）和边（edge）组成的网络结构。图的定义非常宽泛，通常包含点、线、面等所有可以被观察到的对象。例如，图可以用来表示复杂系统中的物理连接、组织结构、社交网络、经济关系、股市交易网络、金融交易网络等等。图还可以用来表示各种对象之间的关系，例如，人员间的通信联系、实体间的产品销售关系、知识图谱中的词语和实体间的关联等等。图可以用来表示各种实体之间的关系、某一类对象的状态变化等，是一种多样化的网络结构。

2.图的基本术语：
- 顶点（Vertex）：图中的每个点称为顶点。顶点也可以称为结点、节点或者实体。每个顶点都有一个唯一标识符，可用于识别该顶点。
- 边（Edge）：边是图中的连接两个顶点的线段，称为边。一条边连接着两个顶点，且两个顶点之间存在一条边。边可以表示两种关系：
   - 无向边（Undirected Edge）：当一条边连接着顶点A和顶点B时，则称为无向边，即(A, B)和(B, A)都是一条无向边。无向边仅保留了指向性质，没有方向性。
   - 有向边（Directed Edge）：当一条边的指向是从顶点A指向顶点B，则称为有向边，即只保留了方向性，而不保留指向性质。有向边代表从一个节点到另一个节点的方向。
- 路径（Path）：一条路径就是一个顶点序列，它通过某些边的序列，形成了一个图的一条边。路径上的顶点和边可以互换。
- 连通性（Connectivity）：如果从图中的任意一点出发，都可以访问图中所有其他顶点，则称该图是连通的，否则，称该图是不连通的。

3.节点属性（Node Attribute）：图中的每一个顶点可以有一些属性，这些属性可以用来表示该顶点的特征。节点属性可以是标量值（如年龄、性别、学历等）或者非标量值（如文本、图像、视频等）。

4.边属性（Edge Attribute）：图中的每一条边可以有一些属性，这些属性可以用来表示该边的特征。边属性可以是标量值或者非标量值。

5.权重（Weight）：边可以有一个权重，表示连接它的两个顶点之间的距离、强度等。权重的范围可以是任意的实数。

6.子图（Subgraph）：由一些顶点和边组成的子图称为子图。

通过上述的基本术语，我们可以了解到图的基本结构和属性。接下来，我们再回过头来看看图计算的应用场景。

1.图数据库

图数据库（Graph Database）是一个存储与处理图数据的NoSQL数据库。图数据库中的数据结构是图，可以用来存储各种复杂的网络数据，同时它提供了一系列高级的分析功能，比如路径查询、聚合查询、关联查询、最短路径查询等等。图数据库特别适合复杂网络数据的存储、索引和查询。图数据库中的API提供了灵活的查询语言，支持声明式查询和事务机制。

2.推荐系统

推荐系统（Recommendation System）的目标是给用户推荐他们可能感兴趣的内容。推荐系统采用用户画像、行为数据和上下文环境作为输入，输出是一个个与用户兴趣相似的推荐列表。推荐系统是个大数据和机器学习的热门话题，它的重要性已经被广泛认可。当前，许多互联网公司都在采用推荐系统来优化营销效果、提升用户体验、降低广告投放成本等。

3.网络攻击检测与防御

网络攻击检测与防御（Network Intrusion Detection & Prevention）是一种对网络安全事件进行预警和应急响应的系统。通过对网络流量、日志和其他相关数据进行分析，能够找出异常的行为模式和攻击手法。网络攻击检测与防御通过提高网络安全防护水平和流程改善网络运营效率，可以显著降低企业的损失。

4.智能交通与路况管理

智能交通与路况管理（Intelligent Transportation and Traffic Management）是智能交通和路况的集成系统，利用地理位置、天气状况、道路网络等信息为驾驶员提供精准、可靠、实时的交通信息。通过智能调度、语音助手和智能地图，智能交通系统能够为乘客提供实时的路况指示、轨迹纠偏、交通事件预警等服务。

5.生物信息分析与网络分析

生物信息分析与网络分析（Bioinformatics and Network Analysis）是利用生物信息学方法对人类基因组、蛋白质组学网络、免疫网络、信号转导网络等网络进行分析。通过利用统计学方法、图论方法、机器学习方法等技术，生物信息分析与网络分析可以发现复杂系统的特性、规律和功能，具有广阔的应用前景。

6.传播学与互动计算

传播学与互动计算（Epidemiology and Interactivity Computing）可以用于研究复杂的网络动态、传染病爆发的传播过程，以及人类行为对网络结构、功能的影响。通过对人类行为的观测，传播学与互动计算可以揭示人类生活中的群体行为模式、规律，并洞察互联网在人类健康、科技发展、经济发展中的作用。
