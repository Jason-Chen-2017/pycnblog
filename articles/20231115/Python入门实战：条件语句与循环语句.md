# 2.核心概念与联系
## 条件语句
条件语句（if-elif-else）是指根据不同的条件选择执行不同代码的语句。如果条件成立，则执行第一个代码块；如果不成立，则根据条件是否满足其他的情况，选择执行第二个或第三个代码块。如以下代码所示：
```python
if condition:
    # 执行该代码块
elif other_condition:
    # 执行该代码块
else:
    # 执行该代码块
```
## 循环语句
循环语句（for-while）是指按照顺序重复执行某段代码的语句。如果要执行的代码块中有多个语句，就需要使用循环语句，否则只需一次执行即可。如以下代码所示：
```python
for variable in sequence:
    # 执行该代码块
   ...
```
```python
while condition:
    # 执行该代码块
   ...
```
## range()函数
range()函数是一个内置的迭代器，可以生成一系列整数序列。比如，range(n) 生成一个从0开始小于n的整数序列。如果给定两个参数 m 和 n ，range(m, n) 将生成一个从 m 开始小于 n 的整数序列。如果第三个参数 s 指定了一个增量值，例如 range(m, n, s)，它将生成一个从 m 开始小于 n 的整数序列，每隔 s 个元素生成一个值。此外，还可以使用反向切片（[::-1]）生成整数序列的逆序。如以下代码所示：
```python
range(stop)              # 从0开始，小于stop的整数序列
range(start, stop[, step])    # 从start开始，小于stop的整数序列，步长为step
range(n)[::-1]           # 反向切片，生成长度为n的整数序列的逆序
```
## break语句
break语句用来结束当前循环体，即停止执行后续代码块，直接跳至下一个语句。如以下代码所示：
```python
for i in range(1, 10):
    print(i)
    if i == 7:
        break
print("执行结束")   # 此时输出“1”-“6”，然后跳过“7”，继续执行下面的代码
```
## continue语句
continue语句用来跳过当前循环体中的剩余语句，直接进入下一次循环。如以下代码所示：
```python
for i in range(1, 10):
    if i == 7:
        continue
    print(i)
print("执行结束")   # 此时只输出“1”-“6”，因为循环到了“7”时自动跳过
```
## pass语句
pass语句是空语句，表示什么都不做。通常作为占位符，在实际编写代码时可用于填充某个位置但暂时不需要实现的代码。如以下代码所示：
```python
def myfunc():
    pass    # 函数定义但是没有任何功能
```
