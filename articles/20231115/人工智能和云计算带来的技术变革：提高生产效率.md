                 

# 1.背景介绍


## 人工智能（Artificial Intelligence）简介
　　人工智能（AI）是指由计算机所模拟实现的智能功能、包括推理、决策等能力的一系列科技领域。它涉及认知、理解、学习、计划、运用这些能力解决问题，或者帮助智能体实现自我改进。2017年，全球IT产业协会发布2017年世界人工智能报告，统计显示，人工智能领域在国际上发展迅速、规模巨大，预计将在2025至2030年达到顶峰。

　　近几年来，人工智能在各个行业、领域均呈现爆炸性发展势头，据IDC数据显示，2019年全球AI产品市场规模估值达到了20.2万亿美元，这个估值超过了整个移动互联网、电信、医疗健康、金融服务和制造业的规模总和，也是第一家突破千亿美元的AI公司估值。截止2018年底，谷歌、亚马逊、微软、Facebook、苹果、IBM、英特尔、甲骨文、雅虎、腾讯、京东、Uber等AI企业累计营收已超4000亿美元。

　　基于人工智能技术的各项创新正在改变着我们的生活、工作方式、社会关系、生产制造等方方面面，其应用范围之广、影响之深远非同小可。这其中，主要有三个方面的变化正在逐渐成为行业热点，分别是：

　　1、智能交通领域的自动驾驶汽车；

　　2、虚拟现实技术的增强现实AR/VR头戴设备；

　　3、医疗行业的机器人和诊断系统。

　　为了能够顺利应对这种巨大的技术变革，各相关行业部门都在努力寻找新的技术发展方向，尤其是对于智能交通领域，目前最重要的技术之一就是自动驾驶汽车，它的出现标志着汽车行业进入了一个全新的时代。

## 云计算（Cloud Computing）简介
　　云计算是一种通过网络将计算资源或存储空间开放给用户进行大规模数据处理、分析和计算的技术。目前，云计算的主要框架分为公有云、私有云和社区云三种类型，根据不同的场景选择适合自己的类型。公有云一般是由政府、大型机构或云服务提供商等直接提供的云平台，有完整的数据中心设施和可靠的网络连接；私有云则是企业内部或外部的个人或组织购买使用的云平台，拥有独立的数据中心和网络连接；社区云则是提供多租户共享的云计算环境，服务对象包括公共研究、学术研究、企业、大型机构和个人。

　　云计算已经从单纯的“计算”逐渐演变成一种更复杂的服务。作为计算资源的托管者和使用者，云计算的利用率越来越高，云计算平台的容量越来越大。例如，阿里云、微软Azure、百度云、腾讯云等都提供各类基础服务，包括服务器、网络、数据库、安全、容器等等。2018年，IDC预测，到2025年，全球云计算服务市场规模将占GDP的6%，云计算覆盖率将达到70%以上。

　　随着云计算的普及和发展，新的技术也被引入其中。智能交通领域的自动驾驶汽车是一个重大项目，需要大量的测试和验证。如何让汽车在复杂道路上安全快速地识别和避障，如何让车辆绕过障碍物并正确进入车道，如何降低驾驶员对车辆性能的依赖程度，这些都是需要高端人才掌握的新技术。另外，医疗行业的机器人和诊断系统也经历了一段艰难的过程，它需要解决诊断速度慢、操作费用高、耐药性、误诊率高等痛点，而这些恰好是需要机器人来解决的问题。
