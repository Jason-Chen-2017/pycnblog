                 

# 1.背景介绍


晶圆的制造方式主要分为高压、低温和超高温三种方法。而晶圆的平整度则取决于晶圆的不同物性。所谓平整度就是指晶粒之间距离的均匀程度。平整度越高，晶粒之间的距离就越小，晶圆就越饱满；反之，则表示晶粒之间的距离越大，晶圆就越精致。在晶圆中，晶粒的位置都比较密集，因此平整度是衡量晶圆表面的质量的重要指标。相比之下，晶圆表面的非晶体材料质量往往会影响到晶圆的性能。因此，晶圆的平整度对于晶圆的制造和应用至关重要。
# 2.核心概念与联系
## 2.1 平整度（Crystallography Intervals）
平整度或称晶圆间距或晶系等距，是指晶粒之间平均距离，它是一个用来描述晶圆结构的基本参数。一般认为，晶圆的平整度越高，晶粒之间的距离就越小，晶圆就越饱满；反之，则表示晶粒之间的距离越大，晶圆就越精致。

平整度可由以下的几个方面衡量：

1. 平均晶体质心偏离理想平面的距离：即一个晶体质心到两个对称平面的距离的平均值，通常约等于半径的倒数，也称为Noll指数。
2. 晶粒尺寸：每个晶粒的尺寸大小不一样，有的小一些，有的大一些。
3. 晶粒内角度：晶体中每两对相邻晶粒的夹角大小也不同。
4. 晶圆孔隙宽度：处在晶粒外层的空隙大小，也就是晶圆孔隙。
5. 晶圆层厚度：晶圆中最薄的一层厚度。

## 2.2 反射率与反射强度
反射率又称折射率，它是物理学上用于描述物质的一种属性，在光学中用以描述感光材料反射光的能力。其单位为百万分之一，通常用希腊字母ν表示，代表某种介电介质中的折射率系数。当反射率超过一定阈值时，可发生反射。显然，反射率越大，就可以获得更多的收益。在晶圆结构的制作中，由于晶粒内部存在多个离子，如果将这些离子全部转化为等能级，那么晶圆的反射率就会大幅下降。所以，晶圆的反射率往往要高于其原材料的反射率。
## 2.3 晶体的可靠性及可维护性
可靠性主要表现为晶体的抗破坏力和抗腐蚀能力。而抗破坏力是指晶体经受的挤压能量和冲击能量，其单位为焦耳，通常用希腊字母ε(r)表示。常用的抗破坏力衡量标准包括低温电泼频率下的峰值反应强度和施加在晶体的冲击下产生的微弱微扰动。抗腐蚀力是指晶体因腐蚀而失去的粒子数。常用的抗腐蚀力衡量标准包括质量的增长和大小的减少。

可维护性是指晶体在使用过程中出现任何错误或者故障的概率。可靠性和可维护性共同决定了晶体产品的品质。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 计算平整度的两种方法
### （1）统计法（Statistical Method）
统计法是基于晶粒的空间分布情况来测定平整度的一种方法。它根据晶圆中晶粒的大小分布情况，通过统计其密度曲线，然后求解该曲线的峰值，从而得到晶圆的平均晶粒大小。由此可以计算出晶圆的平均晶粒大小，再根据晶粒大小之间的差异，计算出平均晶粒大小的标准差，最后得出晶圆的平均晶粒平整度，即由平均晶粒大小标准差计算出的距离。

假设晶圆的晶粒密度函数为 $g(R)$，$R$ 表示晶粒在晶圆上的平均距离。假设晶圆内含有 $n$ 个晶粒，那么 $R_{min}$ 和 $R_{max}$ 分别表示其中的两个最小和最大晶粒的平均距离。已知晶体平均晶粒大小 $a$ ，根据统计法可以计算出晶圆平均晶粒大小为：
$$A=\frac{4}{\pi}\int_0^{2\pi}\sum^n_{i=1}{n-1}g(\frac{\Delta R_i^2}{2})d\Omega=\frac{4}{\pi}(na)^2$$
其中 $\Delta R_i^2=\left[(R_i-R_j)^2+a^2\right]^{1/2}$, $\Omega$ 是截面积元。

由于各晶粒的平均距离间隔可能较大，因此实际上测定的平整度 $I$ 会比这个结果更准确。为了获得真实的平整度，需要采用实际的晶圆数据进行模拟分析。

### （2）数值法（Numerical Method）
数值法也是用于计算晶圆平整度的一种方法。它首先选定一系列的晶体参数，如晶粒的数量，晶粒的大小范围，晶粒的大小分布，晶圆形状和尺寸，将晶圆结构按不同晶体参数进行计算，将晶圆中的晶粒分成不同大小，并计算各个晶粒的平均大小。利用这些平均大小之间的差异，计算出平均晶粒大小的标准差。最后根据标准差来计算晶圆的平均晶粒平整度。

实际上，晶圆结构的平整度还可以用更复杂的数学模型来刻画。但由于各种晶体参数的多样性，各种模型的参数依赖关系不同，难以统一，因此我们通常只采用统计法或数值法来测量晶圆的平整度。

## 3.2 计算平整度的数学模型
### (1) Bowman-Mauguin 等价圆环模型
Bowman-Mauguin 等价圆环模型建立在晶格图象的基础上，对晶圆的各个参数和晶粒间的距离之间的关系做出了很好的抽象。Bowman-Mauguin 模型认为，晶圆结构由一系列截面环构成，每条截面环与晶圆的某一点的连线平行，且所有截面环的圆环均等长。换言之，整个晶圆看作是一个等长的圆环组成的等价集合。因此，可以在晶圆上定义多个不同的圆环，它们彼此重叠，但不相交。晶圆平整度等于多个等价圆环的总宽度除以平均半径，即
$$\Gamma=\frac{W_{\rm total}}{R}$$
其中 $W_{\rm total}$ 是所有圆环的总宽度。

### (2) Zemax 等值圆环模型
Zemax 等值圆环模型建立在晶格图象的基础上，其特点是在某些特殊情况下，这种圆环模型可以给出比其他模型更精确的晶圆平整度。Zemax 等值圆环模型认为，晶圆中晶粒的距离大小关系满足 Bowman-Mauguin 模型的条件，但是相邻晶粒的角度不等。换言之，只要晶粒间的角度都足够接近，即使圆环之间无重叠，整个晶圆仍然可以等效为一系列等值圆环。

Zemax 等值圆环模型认为，各等值圆环的圆心是由晶圆平均晶粒中心向外推进一定距离得到的。因此，各等值圆环的圆心与晶圆边界相切，且彼此不会相互重叠，因此将整个晶圆变为一系列等值圆环的行为称为等值剖面（Equi-Sectional）。考虑到晶体上可能会存在孔隙、缺陷、污染等杂质，在设计时应避免忽视其形态。因此，通过设计合理的晶体结构，可以使得晶圆表面具有良好的平整度。

### (3) Kirchhoff 圆环模型
Kirchhoff 圆环模型与 Zemax 等值圆环模型类似，都是建立在晶格图象的基础上，但不同的是，Kirchhoff 圆环模型考虑了晶圆内各个相邻的晶粒之间的角度。它假定晶体中存在着一组有限度的交叉连接，而这些交叉连接造成的导热作用能够使晶粒之间的距离更加均匀。同时，Kirchhoff 圆环模型还考虑了晶圆边缘和高温区域内的散热条件，可以有效地限制晶粒的空间分散。

Kirchhoff 圆环模型认为，晶圆上的晶粒之间没有明显的孔隙，并且晶圆内部的每个闭合区域都由一段圆环来界定。该模型用一个函数 $\psi$ 来刻画每个晶粒之间的角度和距离关系，其形式如下：
$$\psi(\theta)=\sin^m(\theta)\cos^n(\theta),\quad \theta \in [0,\pi]$$
$\psi$ 函数的表达式可以看作是关于角度 $\theta$ 的一次多项式函数。因此，我们可以考虑不同的值域上的不同多项式，如 $\psi(\theta)=\sin(\theta)$ 或 $\psi(\theta)=\tan^k(\theta)$ 。

为了描述晶体结构，Kirchhoff 圆环模型引入了一组交叉连接，这些连接建立在晶粒之间的空间分散中，表示了晶体的各个区域之间的联系。每个交叉连接都会引起另一个晶粒之间的距离差异，因此晶粒之间的距离会呈现不同模式的波动。因此，Kirchhoff 圆环模型考虑了晶体中交叉连接的数目，不同值的连接会导致不同形状的晶体。

### (4) Dunham-Steinhauser 平行圆环模型
Dunham-Steinhauser 平行圆环模型与 Kirchhoff 圆环模型类似，也是建立在晶格图象的基础上。Dunham-Steinhauser 模型提出了一个新的观点，即晶圆结构可以被看作是一系列平行的圆环构成的集合。每个圆环都与晶圆的某个点相切，两者之间的距离恰好相同，因此，从任意一个点入手，通过绕圆环中心旋转的方式，就能看到整个晶圆。平行圆环模型不需要考虑晶粒之间存在的角度关系，因此可以得到精确的晶圆平整度。

与 Kirchhoff 圆环模型不同，Dunham-Steinhauser 模型不需要考虑晶体内部的交叉连接，因此不能把晶体划分成不同的区域。但是，与 Kirchhoff 圆环模型不同，Dunham-Steinhauser 模型所计算出的晶圆平整度直接与晶圆的形状相关，而与晶体的具体参数无关。

## 3.3 晶圆表面清洁技术
晶圆表面清洁技术是为了改善晶圆表面质量和规整度而使用的技术。目前常用的晶圆表面清洁技术包括水润法、氮化法、乙烯腔清洗、冷脉缓冲及绝缘技术。

### （1）水润法
水润法是一种常用的晶圆表面清洁技术。它利用特殊的溶液来涂抹晶圆表面，达到清洁晶圆表面的效果。

### （2）氮化法
氮化法利用氮气和水进行表面修补，属于平整度增强或变形增强类的方法。氮化法对晶圆表面的效果非常敏感，具有广泛的应用领域。

### （3）乙烯腔清洗
乙烯腔是一种特殊的含氮化工序，它的应用十分广泛。由于晶圆表面具有强烈的膨胀性，因此常用乙烯腔进行表面精细化处理，以提升晶圆表面的平整度。

### （4）冷脉缓冲及绝缘技术
冷脉缓冲是一种采用冷凝油加热或喷涂来使晶圆表面更加平滑，使晶圆平整度更高。冷脉缓冲技术适用于晶圆表面缺乏自然亲和力的问题。

绝缘技术是利用金刚石、玻璃钢等微小材料，将晶圆表面嵌入某种绝缘体系中，使晶圆表面具有完整封闭的特性。虽然绝缘技术对晶圆表面的清洁效果不尽人意，但它可以在一定程度上增强晶圆的抵抗力。
