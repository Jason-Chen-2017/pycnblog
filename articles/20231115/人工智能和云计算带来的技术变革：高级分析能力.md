                 

# 1.背景介绍


## 1.1 概念定义
什么是人工智能（Artificial Intelligence）？它的概念可以从三个层面进行分类：
* 自然语言处理(Natural Language Processing, NLP)：以人类的语言形式进行信息处理，如语言理解、语音识别、文本生成等。
* 机器学习(Machine Learning): 通过训练数据对计算机提取规律并解决问题，如图像识别、语音识别、推荐系统等。
* 深度学习(Deep Learning): 对多层神经网络进行训练，通过多个隐层来逼近复杂函数，如自动驾驶、图片搜索、视频分析等。
云计算(Cloud Computing)：通过网络连接计算机，共享资源，实现资源池化、弹性伸缩、按需付费等功能。基于云平台搭建的人工智能应用需要在本地端进行编程开发，然后将应用部署到云端运行。
## 1.2 历史回顾及现状
人工智能研究历史可追溯到上个世纪六七十年代，当时蒂姆·伯纳斯-李（Donald Bernstein）提出了“认知机”的概念，提出用机器模拟人的学习过程。两年后，麻省理工学院的约翰·格雷厄姆·阿特金森（John Graves）发表了第一篇关于“深度学习”的论文，提出神经网络的概念，正式开启了人工智能领域的新纪元。
与此同时，互联网的蓬勃发展引起了产业界的广泛关注。2006年，亚马逊推出AWS云服务，成为市值排名第一的云计算提供商，为企业提供了大量的云服务产品。2014年，微软宣布与AWS合作为公有云服务提供商，随之而来的是Azure、Google Cloud Platform和IBM Cloud Platform等。
截至目前，人工智能领域已经进入了一个新的阶段——云计算时代。通过把大数据、人工智能和云计算结合起来，实现了更高的效率、更快的响应时间和更低的成本。这也为企业搭建自己的人工智能平台提供了一条道路。

## 1.3 大数据与人工智能的结合
云计算的发展为企业搭建人工智能平台提供了一条新路，但是如何利用大数据的价值却仍然是一个重要的问题。首先要区分大数据与数据科学，前者指海量的数据，后者指采用数据科学方法处理这些数据的过程。其次，大数据时代还存在许多问题，比如安全问题、数据质量问题、数据的分布式存储问题、数据实时获取问题、数据采集、数据清洗、数据加工、数据传输、数据集成、数据分析等流程不断被提出新的技术问题。另外，数据科学中常用的统计学、机器学习、深度学习等概念也是应对这一切变化做出的反映。因此，如何有效整合大数据、人工智能和数据科学，使得云计算和大数据得到更好的协同作用，实现真正的智慧与商业价值，也是人工智能技术发展的方向。

# 2.核心概念与联系
## 2.1 概念
### 2.1.1 符号主义与局部主义
符号主义认为，符号系统就是人脑中用来思考的那些符号结构，它控制着我们的行为、认识、思维和行为方式。符号系统只能运作在特定环境下才能发挥作用，而且在很多方面都处于劣势。所以说符号主义有一个最大的缺陷，就是过分强调所谓的客观真理，而忽略了主观意识、情绪、直觉等人类活动的因素。局部主义则认为符号系统其实是局部的，并且是最基本的功能单元，一个完整的符号系统需要由各种功能单元组合而成。所以局部主义认为，只有符号系统才能让我们更好的理解客观世界，无法脱离符号系统，就没有意义了。总体来说，局部主义容易误导一些观点，认为所有事物都能被符号系统所呈现，但事实上每个系统都有局限性。
### 2.1.2 知识表示与语义网络
所谓知识表示，即将不同来源、不同形式的信息组织成统一的、结构化的形式，并对其进行抽象、归纳、概括、关联等处理，使得信息变得更易于处理、记忆和交流。所谓语义网络，就是利用各种关系来描述实体间的相互作用，构建出具有共同主题的多种视角、多种意义的网络。
### 2.1.3 图谱与链接规则
所谓图谱，就是对知识进行结构化、可视化的一种方式，包括实体、属性、关系三大组成部分。所谓链接规则，就是决定两个实体是否相关联的方法。一个实体可能属于不同的类别，那么不同的类别之间的关系如何处理呢？
### 2.1.4 模型与实例
所谓模型，就是对实际问题的一个假设或者对未知问题的一个抽象，用于刻画现实世界中的某种现象或现象模式，模型本身是模糊的，并不一定精确。所谓实例，就是从模型中抽象出来的具体事例。一个模型可以对应多个实例，一个实例也可以对应多个模型。
### 2.1.5 知识库与知识工程
所谓知识库，就是存储、管理、检索、整理、利用、应用、传播、交流和扩展知识的集合体，它是一个庞大的、动态的、全面的、跨越时空的知识集合。所谓知识工程，就是构建知识库的技术与方法学。知识工程的方法论分为理论基础、工程手段、实践方法三个层次。
### 2.1.6 推理与计算
所谓推理，就是从已知事实推导出其他事实的过程。所谓计算，就是用计算机对逻辑关系进行自动推理，能够根据输入的数据生成相应输出结果的过程。
### 2.1.7 规则与统计
所谓规则，就是按照一定的规范和逻辑来进行操作的指令。所谓统计，就是对数据进行计数、分析和预测的一系列方法。统计学是人工智能的一种主要方法学，通过数据处理、描述统计规律、判断规律性质、对比不同场景下的统计规律，使得机器具有分析、预测和决策等能力。
### 2.1.8 知识融合与自动驾驶
所谓知识融合，就是利用多个知识源或模型对事件进行分析和推理，产生有意义的结论或判断的过程。所谓自动驾驶，就是通过计算机处理图像、声音、位置、速度、卫星图像、传感器信息、交通状态等信息，结合计算和分析，能够自动地制定、执行和改善行车路线，减少人力参与、提升效率的一种技术。
