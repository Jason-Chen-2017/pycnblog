                 

# 1.背景介绍


## 智能大数据的定义、概念及特点
### 智能大数据的概念
“智能”一词被频繁使用，其内涵可以从三个方面进行理解：
- 自动化：人类在这个过程中能够自己完成重复性任务，使得人工劳动变得低效而效率高；
- 数据驱动：利用大量数据，机器学习和统计方法能够分析出规律和模式，实现对现实世界的建模和预测；
- 新兴技术：互联网、云计算、移动互联网等新兴技术的出现，给予了人们巨大的创造力空间。
基于以上三个特点，特别是在新兴技术如人工智能、大数据、云计算、物联网的快速发展下，智能大数据成为未来时代的主要驱动力之一。智能大数据具有以下几个特征：

1. 多维度信息：通过互联网、传感器、GPS、摄像头等收集到的海量数据，形成了不同维度的信息。因此，这一特点促进了数据的挖掘、整合和分析过程。

2. 非结构化数据：非结构化数据指的是在不同格式或结构的数据中，没有统一的标准，难以处理和存储。需要人为地对数据进行清洗、转换和规范化，才能获得结构化数据。

3. 海量数据：海量数据使得数据分析工作变得复杂且耗时长。如何有效地分析海量数据、提取有效信息以及建立分析模型，将成为智能大数据的关键。

4. 快速增长：智能大数据正在产生的数据量正在以每天千倍的速度增长。如何有效地管理和处理海量数据并加快数据分析速度，也将成为重点关注的问题。

### 智能大数据解决方案概述
智能大数据解决方案一般由四个层次构成：基础设施层、数据采集层、数据处理层、数据应用层。各层之间的关系如下图所示：


#### （一）基础设施层
基础设施层主要包括数据中心、网络、服务器、集群、软件组件等硬件设备和软件资源，为智能大数据提供服务的关键环节。这一层提供数据的收集、传输、存储和处理的能力。

#### （二）数据采集层
数据采集层包括各种各样的数据源，例如各类网站日志、业务数据、用户行为数据、微博评论数据、电子邮箱、社交媒体数据等等。这些数据源经过清洗、转换、规范化后得到结构化的、可查询的、可分析的数据库。这一层的主要作用是将各种来源的数据进行汇总、整理、归档和过滤，形成高质量、易于使用的大型数据仓库。

#### （三）数据处理层
数据处理层包括数据分析、挖掘、报告、决策支持等模块。它首先对数据进行清洗、转换、规范化，然后对数据进行分析，发现潜在的价值所在，并做出相应的结论和建议。这一层的核心任务就是从大数据中找出有价值的模式、热点事件、异常检测、风险评估等，并根据结果做出决策支持，提供给用户直观、准确、及时的反馈。

#### （四）数据应用层
数据应用层是数据最终呈现和运用的地方，是数据终端用户看到的最重要的一环。它包括BI（Business Intelligence，商业智能）、推荐系统、搜索引擎、广告营销等各个环节。通过数据应用层的呈现和功能，向最终用户提供多种形式、多层次、多领域的价值洞察。
