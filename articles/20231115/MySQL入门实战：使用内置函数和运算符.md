# 2.核心概念与联系
## 2.1 函数
函数就是为了完成特定任务而预先定义好的代码片段。它接受输入参数,经过计算后输出一个结果值。通过使用函数可以提高编程效率,减少代码重复度并改善程序结构。下面列出了MySQL支持的常用函数列表:

- 日期时间函数
  - NOW() 返回当前日期和时间
  - CURDATE() 返回当前日期
  - CURTIME() 返回当前时间
  - DATE_ADD() 对日期进行加减运算
  - DATEDIFF() 返回两个日期之间相差的天数
  - DATE_FORMAT() 将日期格式化成指定的字符串
  - TIMESTAMP() 返回当前时间戳
- 数学函数
  - ABS() 返回绝对值
  - CEIL() 向上取整
  - FLOOR() 向下取整
  - RAND() 生成随机数
  - ROUND() 四舍五入
  - SIGN() 判断正负号
- 字符串函数
  - ASCII() 返回ASCII码
  - CONCAT() 拼接字符串
  - INSERT() 插入子串
  - LCASE() 小写化字符串
  - LENGTH() 获取字符串长度
  - LOCATE() 查找子串位置
  - LOWER() 小写化字符串
  - LPAD() 在左侧填充字符串
  - REPEAT() 重复字符串
  - REPLACE() 替换子串
  - RIGHT() 从右边截取字符串
  - RPAD() 在右侧填充字符串
  - SOUNDEX() 返回SOUNDEX编码
  - SUBSTR() 提取子串
  - UPPER() 大写化字符串
- 条件函数
  - IFNULL() 如果表达式为NULL则返回指定值
  - CASE... WHEN... ELSE... END 执行不同条件下的操作
  - COALESCE() 如果任何一个表达式不为空,则返回第一个非空表达式的值
  - NULLIF() 比较两值是否相同,如果相同则返回NULL
- 聚集函数
  - COUNT() 统计个数
  - MAX() 求最大值
  - MIN() 求最小值
  - SUM() 求和
  - AVG() 求平均值
- 分析函数
  - STDDEV() 求标准方差
  - VARIANCE() 求方差
- 信息函数
  - DATABASE() 当前数据库名称
  - USER() 当前用户名称
  - SCHEMA() 当前模式名称
  - CONTEXT_INFO() 查询上下文信息

函数的作用主要如下:

1. 提供便利。通过函数可以方便地实现一些复杂的功能,如日期和时间处理,计算,加密等功能。
2. 提高性能。很多函数都有优化实现,具有更快的执行速度。同时,数据库优化工具也可以利用这些函数的特点对查询进行自动优化。
3. 简化代码。通过函数可以避免重复编写相同的代码,降低编程难度。
