                 

# 1.背景介绍


计划制定与行动安排是一个过程。它既涉及到管理层的工作，也涉及到企业技术人员的创新能力、执行能力和管理技能。在计划制定的过程中，管理层需要整合各方资源，将各种信息、数据、知识、经验等资源综合成一个具有指导意义的计划，并与企业战略方向相结合，进行分步、精益、及时、有效的决策，然后根据计划的实施情况调整工作任务，使计划落地实施，确保组织的运行效益最大化。在这其中，首先要明确企业需求，了解客户诉求，制定正确的战略规划，设立战略目标，分析市场环境、竞争对手以及自身业务形态，提出符合客户期望的方案，制定细致的计划细则，落实具体行动纲领，推进各项工作的顺利实施，这些就是计划制定与行动安排的关键环节。

而为了实现这一目标，企业管理者需要掌握一系列的分析、设计、管理方法和工具。例如，如何制定战略计划，如何对方案进行分析、综合、筛选，如何设计预算、开支计划，如何实施方案，如何跟踪和评估实施效果？还有如何做好风险管控，建立起完善的内部控制体系？结构化思维和金字塔结构可以帮助企业管理者快速定位和解决疑难问题，有效处理复杂的信息，达成预期目标。

因此，结构化思考和金字塔结构是一套科学的方法论，它不仅适用于计划制定与行动安排，还可应用于任何信息处理、决策和战略部署相关的问题，如财务分析、人事管理、产品开发、项目管理等。

# 2.核心概念与联系
## 2.1 什么是结构化思考
结构化思考是一种方法论，其核心思想是将复杂问题转化为易于处理和理解的形式。

结构化思考的一个重要特点是关注“解决”而不是“描述”，即将问题分解为一系列小问题，逐步找出根本原因，最终得出答案。结构化思考有如下几个优点：

1. 可重复性：结构化思考的每一步都可以用“步骤-结果”的方式复述，可以对同样的问题反复试验验证；
2. 透明性：结构化思考的每一步都很清楚，易于观察和验证，容易发现问题所在；
3. 智能化：结构化思考能够准确识别、消除矛盾和不确定性，将复杂问题转换成可以轻松办结的小问题；
4. 效率化：结构化思考每一步都切中要害，一步到位，不需要太多的辅助工具和技巧；
5. 提升动力：结构化思考的每一步都提醒着自己，完成了多少，才可以接着下一步。

## 2.2 什么是金字塔结构
金字塔是由圆柱状以上的结构，上面只有最高层，向上依次延伸的一系列金字塔形层，它的高度决定着信息的重要程度，越往上就越重要。

金字塔结构的基本思路是，将复杂事物拆解为简单组块或子问题，从而获得更多的注意力。结构化思考借鉴了金字塔结构的原理，将复杂的计划问题化简为一系列小问题，逐层解决，逐渐得到答案。金字塔结构又被称为“三个步骤法”，包括：

1. 第一步：理解业务和机会；
2. 第二步：梳理假设和条件；
3. 第三步：制定方案与实施步骤。

## 3.1 制定战略计划
制定战略计划是一个关键环节，主要涉及以下四个方面：

1. 战略分析：对目前的业务和机遇，结合公司的发展战略，形成战略规划和目标。
2. 商业模式分析：分析公司当前的商业模式，评估是否仍然适合经营，考虑在新的商业模式中添加或减少某些功能。
3. 市场分析：分析市场环境、竞争对手、消费者群体、客户价值及客户需求，以此为基础分析公司在市场中的战略位置，确定公司应处在的全球竞争地位。
4. 技术方案：根据业务分析和商业模式分析的结果，制定相关的技术方案，包括品牌策略、营销方式、用户画像、留存策略、网络营销、广告投放、外部技术支持、核心技术等。

## 3.2 对方案进行分析、综合、筛选
对方案进行分析、综合、筛选可以根据公司实际情况和需求，识别其优劣，形成一份完整的商业方案。

1. 方案调研：收集竞品、客户反馈等商业信息，与公司业务线、技术团队沟通，分析竞品的优缺点，以及客户对不同产品/服务的认识和喜好。
2. 比较方案：比较所有竞争对手的方案，判断哪种方案更适合公司业务，以及哪些方面存在改进空间。
3. 流程设计：按照公司流程标准化的要求，制订公司流水线，对商业方案进行细化。
4. 数据分析：通过数据分析，探索和洞察商业现象背后的驱动因素，找到优质客户，验证自己的业务理念。

## 3.3 设计预算、开支计划
设计预算、开支计划将预算与开支进行匹配，根据预算内投入产出的比例，设置相应的收益分析。

1. 设置预算范围：制定一个健康的预算范围，便于公司随时管理和调整。
2. 制定预算结构：根据年度、季度、月度、周边、营销策略等，设计预算结构，如固定费用、变动费用、增长性收入、投资回报等。
3. 制定费用计划：考虑收益分析和收益预测，设置核心营销渠道的费用预算，以及各项增值服务的费用预算。
4. 审查预算：公司预算应当和实际情况保持一致，否则可能导致项目延期或变成亏损。
5. 执行开支计划：将预算分摊给各个部门，每个部门负责度量和控制自己的开支。

## 3.4 实施方案
实施方案分为两个阶段：

1. 实施前准备：公司的内部管理人员应制定实施方案，形成各项工作职责和时间表。
2. 实施方案：检查实施方案的内容、条款，与各项人员核对无误后，启动实施流程，根据流程监督实施进度。

## 3.5 跟踪和评估实施效果
跟踪和评估实施效果是计划制定与行动安排的最后一步。其目的是获取实施成功的证明，确认效果，对策划方案进行必要的改进。

1. 实施控制：对实施的结果，进行真实记录和统计，确定公司的实际成果。
2. 运营评估：对新功能、增值服务的效果进行客观评估，以便决定是否继续实施。
3. 成果回顾：检视公司近几年的运营成果，总结经验教训，总结实施方案的弊端和改进方向。

## 3.6 做好风险管控
做好风险管控需要建立完整的风险管控体系，防止出现意外事件影响组织的正常运作。

1. 风险评估：制定风险管理标准，分析现有的风险，提出应对策略，并制定定期检测和风险评估制度。
2. 内部控制：建立内部工作流程和规范，强化员工承担的风险管理责任。
3. 外部监控：设置外部监控机制，及时掌握外界动态，采取有效措施响应突发事件。
