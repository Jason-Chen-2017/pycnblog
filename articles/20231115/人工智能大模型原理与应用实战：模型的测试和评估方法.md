                 

# 1.背景介绍



2017年是中国模式经济学的重要研究年份之一。2017年9月28日，北京大学、清华大学、中科院计算所和英特尔举行了首届中国模式经济学会议。会议由“模式经济学视角下的智慧投资新秀”(模式经济学系列报告)主办，“如何评价模型？机器学习模型及其评估”(“模式经济学视角下的智慧投资”系列主题报告)协办，分为2个论坛，每场论坛邀请了一批模式经济学方面的优秀学者做分享。在这次颁奖典礼上，华东师范大学、北京交通大学等七所重点高校和研究机构都以不同形式出席，“模式经济学+人工智能”会议也成为引领智慧投资的重要阵地。

本文作者，杨云龙，也是模式经济学界和机器学习界的一名顶尖科学家。他是多领域重量级专家，现任美国斯坦福大学教授，国防高级研究计划局科研顾问，中科院计算所博士后，曾任Facebook产品设计经理。杨云龙先生在人工智能领域耕耘十余年，曾经担任微软亚洲研究院（MSRA）研究员、旷视科技研究院智能营销组负责人，之后又多年担任首席经济学家、研究员，现任美国伊利诺伊大学香槟分校助理教授。

2019年5月14日，美国商务部召开科技督导小组座谈会，讨论虚拟现实、人工智能、区块链和其他数字化技术发展对商业竞争的影响。中国模式经济学会联合国际会计事务专家团队，提出了“区块链与模式经济学”主题的论文集征集工作。国际会计专家王森春、彭康宇、陈金铠、何新衡、傅盛、赵军、钟荣、严复、李嘉欣，也分别提交了关于区块链与模式经济学的论文。

作为国内外多领域最前沿研究者之一，杨云龙先生的理论基础十分扎实，文章不仅全面准确的阐述了模式经济学和人工智能的相互关系，还通过丰富的案例、模型和试验，力求探索到底人工智能带来的新经济学特性会带来怎样的社会、经济效益。文章既具有新颖而独特的分析视角，又兼具深刻而通俗的语言风格，极易阅读。

本文围绕模式经济学与人工智能的关系，介绍并验证了模式经济学在人工智能领域的两个重要研究方向，即基于群体决策的智能投资与基于序列建模的智能制造。首先，基于群体决策的智能投资(Group-Decision-Making Intelligent Investment)是指利用人工智能技术从历史数据中学习，提升对未来市场发展趋势的预测能力，并据此做出适应性投资决策。与传统的精准投资不同的是，这种群体决策的智能投资认为，投资者的行为是具有不确定性的，并用机器学习的方法来预测未来市场的走向，帮助投资者更好地把握节奏、规避风险，获得理想的投资收益。同时，在传统的银行间同业拆放市场中，基金经理需要花费大量的时间去判断市场走势、抓住机会、利用手段提升收益率，但通过利用机器学习的方式，可以提升基金经理决策的效率。

第二，基于序列建模的智能制造(Sequence-Based Intelligent Manufacturing)则是指利用人工智能技术开发出新型的生产方式，帮助企业降低成本、提升竞争力。基于序列建模的智能制造可以预测出大量产线中的缺陷，并将它们用于改进生产流程，提升生产效率。机器学习技术已经在许多领域得到应用，比如基于位置数据的图像识别、基于文本数据的自然语言处理、基于结构数据的物理仿真等。通过引入机器学习算法，可以自动化生产过程、提升生产质量和工艺水平。

基于群体决策的智能投资与基于序列建模的智能制造，是人工智能和模式经济学结合的一个有趣的新趋势。机器学习和模式经济学可以结合起来，构建一个“大脑”型的模式经济学模型，这对于未来的商业发展意义非凡。我们期待着这两项重要研究工作的进一步进展。
