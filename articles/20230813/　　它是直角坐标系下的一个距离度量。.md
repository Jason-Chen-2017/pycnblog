# 1.简介
  
　　
　　平面几何中，最简单且最重要的是关于距离的问题。对于两个点，需要计算它们之间的距离。
　　在平面上，一条直线与另一条直线相交只有两种情况：没有交点、有一个交点、有无数个交点。而直角坐标系则给出了一种特殊的距离计算方法——毕达哥拉斯定理。所谓毕达哥拉斯定理就是直角坐标系下两条直线上的任一点到这两条直线的距离都是一个常数。因此，当在直角坐标系中做直线方程两端点之差，就可以直接得到该点到直线的距离值，而不需要求出交点位置，也不需要进行比较运算，这就是毕达哥拉斯定理的精妙所在。本文将从基本概念术语开始，对毕达哥拉斯定理及其应用进行介绍，然后给出一个Python实现的例子。最后，还会讨论未来的发展方向以及一些实际的应用。
# 2.基本概念及术语　　定义：毕达哥拉斯定理（Pythagorean theorem）是一种直角坐标系下两条直线的距离加起来等于斜边的比值定理。直角坐标系是一个三维空间中所有直线的基准系统，每一个点都是由直角坐标系中的两个参数——x轴上的一个参数表示直线上的投影长度，y轴上的另一个参数表示投影高度——来唯一确定。用这个坐标系可以很容易地计算任意两点之间的距离。其一般形式如下：

$$a^2+b^2=c^2$$

其中，$a$, $b$, $c$ 分别为任意两个点的横纵坐标。由毕达哥拉斯定理可知：

$$\frac{a}{c}=\sqrt{\frac{c^2-b^2}{a^2}}=\sqrt{\frac{(c-\frac{b}{2})^2+(c+\frac{b}{2})^2}{(a-\frac{b}{2})^2+(a+\frac{b}{2})^2}}=\sqrt{\frac{c^2-(b/2)^2+(b/2)^2}{a^2-(b/2)^2+(b/2)^2}}$$

即：$c$ 为任意两点的斜边的长度，$a$ 和 $b$ 分别为斜边两端点到直线的垂线，角度为$90^\circ$ 或 $\pi/2$。

更直观地说，如果把两个点连接成直线，那么斜边即为两点间的距离；如果沿着这条直线延长两点，那么沿着这两点的法向量的投影即为斜边的一半；两个投影的合计长度即为斜边的另一半；由于平面上所有的直线均满足毕达哥拉斯定理，因此任意两个点之间的距离都是相同的。

通过在坐标系中找到两点并画出一条直线，就能够通过计算斜边来确定两点之间的距离。如图1所示，用坐标系的横轴表示直线的两个参数——水平距离h，竖直距离k——，那么两个点（P，Q）距离PQ的距离可以用毕达哥拉斯定理算出来，公式为：
