# 1.简介
  

平均绝对误差(Mean Absolute Error)或称MAE,又称作“回归问题中的均方误差”、“回归性能指标”。它是一个用来量化预测结果与实际情况偏差程度的非负值指标，它是被广泛使用的机器学习模型评估标准之一。

平均绝对误差常用在回归任务中，即预测连续型变量的值的问题。该指标可以直观地反映预测精度的好坏。当MAE较小时，代表模型预测精度较高；当MAE较大时，代表模型预测精度较差。对于某些场景下，需要知道一个模型是否出现过拟合现象（overfitting）。

方差(Variance)，也称“回归问题中的方差”，用来衡量模型的预测值的波动程度。方差越小，代表模型的预测值越稳定，这往往意味着模型的预测能力更强。

方差与偏差的区别：
- 偏差(Bias): 模型预测值与真实值之间的差距。它会随着样本规模的增加而减小。偏差决定了模型训练过程中的系统atic error，模型的泛化能力。
- 方差: 模型的预测值的变化范围。方差会随着样本规模的增加而增大。方差代表了模型的随机性，它反映了模型的不确定性。

基于以上两个概念，我们可以将回归问题分成两类：
1. 用标签值直接预测连续值，比如房屋价格预测、销售额预测等问题。这种情况下，我们希望模型能够准确地估计每个输入变量对应的输出变量。此时，目标函数一般采用平方误差损失函数，即计算预测值与真实值之间差的平方值，然后取平均值作为目标函数值。

2. 用标签值预测分类值，比如垃圾邮件分类。这种情况下，我们希望模型能够正确地分类输入数据，此时通常采用交叉熵损失函数。

总体来说，MAE和方差都是评估回归模型好坏的重要指标。然而，在实际应用中，它们仍有一定的局限性。比如，MAE仅能衡量相对误差大小，而不能直观显示每个标签变量的预测值偏离程度，因此并不适用于分析多标签问题。另外，当样本量不足或者存在异常值时，MAE及其变种可能无法给出良好的模型效果。因此，在这些情况下，我们应该结合其他指标进行综合评估。


# 2.基本概念与术语说明
## 2.1 MAE 平均绝对误差
### 2.1.1 概念
平均绝对误差(Mean Absolute Error)或称MAE,又称作“回归问题中的均方误差”、“回归性能指标”。它是一个用来量化预测结果与实际情况偏差程度的非负值指标，它是被广泛使用的机器学习模型评估标准之一。

平均绝对误差常用在回归任务中，即预测连续型变量的值的问题。该指标可以直观地反映预测精度的好坏。当MAE较小时，代表模型预测精度较高；当MAE较大时，代表模型预测精度较差。对于某些场景下，需要知道一个模型是否出现过拟合现象（overfitting）。

### 2.1.2 计算方法
MAE的计算方法如下：
$$MAE=\frac{1}{n}\sum_{i=1}^n|y_i-\hat y_i|$$
其中$y_i$为真实值，$\hat y_i$为模型预测值，$n$表示样本数量。

### 2.1.3 使用场景
- 用于预测连续型变量的值。
- 可以直观地反映预测精度的好坏。当MAE较小时，代表模型预测精度较高；当MAE较大时，代表模型预测精度较差。
- 可用于分析单标签问题，但不适用于多标签问题。

### 2.1.4 优缺点
#### 2.1.4.1 优点
- 简单易于理解，计算方便，计算速度快。
- 不容易受到异常值的影响，鲁棒性高。
- 适用于回归问题，包括分类问题。

#### 2.1.4.2 缺点
- 对异常值敏感，对极端值不太敏感。
- 在不同的场景下，表现力不同。

## 2.2 方差 Variance
### 2.2.1 概念
方差(Variance)，也称“回归问题中的方差”，用来衡量模型的预测值的波动程度。方差越小，代表模型的预测值越稳定，这往往意味着模型的预测能力更强。

### 2.2.2 计算方法
方差的计算方法如下：
$$Var=\frac{1}{n}\sum_{i=1}^{n}(y_i-\mu)^2$$
其中$y_i$为样本值，$\mu$为样本均值，$n$为样本数量。

### 2.2.3 使用场景
- 可以衡量模型的预测值波动范围。
- 模型的预测能力更强，误差更小。
- 对异常值敏感，对极端值不太敏感。

### 2.2.4 优缺点
#### 2.2.4.1 优点
- 提供了一种对数据分布的深刻认识，可以帮助我们确定模型的可靠性。
- 通过方差可以得到模型的偏差，从而判断模型的拟合度。

#### 2.2.4.2 缺点
- 需要用到样本均值，可能受到异常值的影响。
- 无法直观呈现每个标签变量的预测值偏离程度。
