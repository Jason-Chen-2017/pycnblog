# 2.基本概念与术语介绍
## 2.1 NumPy简介
NumPy（Numerical Python）是一个用Python编写的科学计算包，其设计目的是提供一个快速便捷的方式处理多维数组和矩阵。它提供了包括线性代数、傅里叶变换、随机数生成等在内的许多数值计算函数库。主要特性如下：

1. 数据结构：NumPy中的数据结构是ndarray，一种同构的、大小可变的数组。可以轻松地将不同类型的数据合并到一起，并且支持高效矢量化运算符，使得复杂的操作如索引、切片和迭代都变得简单易行。
2. Broadcasting：NumPy的广播机制允许在算术运算中向 arrays 之间执行“模式匹配”，从而使它们兼容的同时执行操作。这种机制既适用于相同形状的输入数组，也适用于输入数组与标量之间的运算。
3. 便利的统计函数：NumPy包含各种统计函数，如 mean()、std() 和 var() ，这些函数能够快速地计算多维数组或矩阵的均值、标准差和方差。此外，NumPy还提供了许多其他统计函数，如 corrcoef() 和 cov() 。
4. 灵活的索引方式：NumPy支持多种索引方式，包括整数、布尔型、整数数组和切片，这些索引方式可以快速地访问和修改数组元素。此外，NumPy还支持级联索引，这使得一次操作多个数组元素成为可能。
5. 内存连续性：NumPy中的所有数组都存储在一个 contiguous block of memory 中，因此访问速度非常快。
6. 线性代数运算库：NumPy提供了一个矩阵类 Matrix 来进行矩阵运算。这个矩阵类继承自 ndarray ，同时也实现了许多矩阵运算方法。
7. FFT（快速傅里叶变换）：NumPy提供对快速傅里叶变换（FFT）的支持。这个功能可以使用 fft() 函数或者 fft2() 函数调用。
8. C/C++扩展模块：NumPy拥有一个称为 numpy.dual 模块的子模块，该模块为两个NumPy接口提供一致的访问。在某些情况下，你可以使用该模块避免纠结于不同的API。
9. 广泛的应用领域：除了科学计算之外，NumPy也被广泛应用于信号处理、图像处理、生物信息学、天文学等领域。
## 2.2 基础知识点
以下是一些基础知识点：

- 标量(Scalar)：只有单个数值的数组叫做标量。
- 向量(Vector)：一组有序的、按照排列顺序的一系列标量组成的数组叫做向量。
- 矩阵(Matrix)：二维矩形表格，每一行与每一列都由若干个元素组成的数组叫做矩阵。
- 张量(Tensor)：高阶的数组形式，比如说三维空间中的图像，四维空间中的视频流，五维空间中的立体声场等都是张量。
- dtype：数据类型，描述数组元素的存储和处理方式。
- 轴(Axis)：数组的一个维度，比如说矩阵中，横坐标轴就是第1个轴，纵坐标轴就是第2个轴。
- shape：数组的维度。
- size：数组元素的总数量。
- stride：数组中指针移动的距离。
- reshape():改变数组的shape，即改变数组的维度。
- transpose():转置数组的维度。
- slicing：取数组的一部分，切片。
- indexing：根据索引位置获取数组中的特定元素。
- universal function(ufunc): 一种对数组的运算。
## 2.3 安装与配置
### 2.3.1 安装
#### 在Windows上安装
如果你的电脑运行的是Windows操作系统，你可以选择下载适合你的版本的预编译好的NumPy安装包：https://www.lfd.uci.edu/~gohlke/pythonlibs/#numpy 或直接从官网下载安装程序安装。
#### 在Linux或Mac上安装
如果你正在使用Linux或Mac操作系统，建议使用Anaconda或Miniconda，并通过conda命令安装NumPy：
```bash
$ conda install numpy
```
### 2.3.2 配置
通常情况下，只需要按默认配置就可以了。如果你遇到了问题，你可以查看相关文档或反馈到社区求助。
