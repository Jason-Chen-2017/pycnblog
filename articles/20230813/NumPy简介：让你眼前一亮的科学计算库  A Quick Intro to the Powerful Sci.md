# 2. 基本概念及术语
## 1.什么是NumPy？
NumPy，读作“num-plus”，是Python生态系统中的一个开源项目。它是一个用于科学计算的第三方库，可提供多种高级数据结构（如N维数组），以及用于对数组进行各种运算的函数。

与类似的数据结构如列表（list）、元组（tuple）不同，NumPy的数组（array）具有以下两个显著特征：

1. **内存连续存储**：数组中的元素在内存中是相邻存放的，可以直接通过指针来访问；而对于列表来说，则需要逐个元素地寻址，速度较慢。

2. **类型固定**：每个数组元素都必须是相同的数据类型。例如，不能将整数数组与浮点数组混合起来。

除此之外，NumPy还提供了一些其它功能，包括数组的形状（shape）、数据类型（dtype）、线性代数、随机数生成、FFT、向量化运算、文件输入输出等。总的来说，NumPy是一个高性能的矩阵运算工具包，也是Python生态系统中的优秀工具。

## 2. ndarray类
ndarray（读作"n-dimensional array"）是NumPy中一个重要的数据结构。它是一个通用的多维数组对象，其中各元素可以保存不同的数字类型（即dtype）。ndarray可以使用Python列表或嵌套列表来初始化。

ndarray的重要属性有以下几个：

- shape：数组的维度大小，即每一维上的元素数量。
- dtype：数组元素的类型，例如int32、float64等。
- ndim：数组的维度数量。
- size：数组元素的个数。

除了这些属性之外，ndarray还提供了很多实用方法，比如：

- astype()：将数组元素转换成指定的类型。
- copy()：拷贝数组，产生一个副本。
- reshape()：改变数组的维度。
- transpose()：转置数组。
- squeeze()：压缩掉长度为1的轴，即把多余的轴去掉。
- ravel()：展平数组。

这些方法有助于完成常见的数组操作，如数组的转置、压缩、展开等。

## 3. 多维数组与标量
在Numpy中，所有元素都是以数组形式存在的。举例来说，如果要表示一个二维的矩形区域（如5行4列），那么这个矩形区域就是由多个元素构成的一个二维数组。而单个的数字叫做标量（scalar）。

Numpy的API设计者对这一点也很重视，他们认为标量的概念不太好理解。所以，为了统一两者之间的概念，Numpy区分了数组与标量的概念。如果数组只有一个元素，那么它就是标量。反之，数组就是具有多个元素的集合。

## 4. 数组运算的四则运算规则
数组运算的四则运算规则与标量运算相同，但是要注意数组与数组之间只能进行对应位置的运算。具体如下所示：

1. 加法运算：对应位置的元素相加。
2. 减法运算：对应位置的元素相减。
3. 乘法运算：对应位置的元素相乘。
4. 除法运算：对应位置的元素相除。
5. 取模运算：对应位置的元素取模（取余数）。
6. 比较运算：返回True或者False，代表该位置元素是否满足比较关系。
