# 1.简介
  

“红黑树” 是一种平衡二叉搜索树（Binary Search Tree），对每个节点增加一个额外的标记 bit（color）使得它可以区分为红色或黑色节点。在插入、删除等操作中通过旋转和重新着色的方式保持这个平衡状态。“红黑树” 的目的是为了解决一些平衡二叉查找树（BBST）的不足，例如自平衡的需要消耗大量的CPU资源和内存空间；迭代和遍历的时间复杂度都比较低，因此也适合应用于一些高性能要求的场合。
## 2.基本概念术语说明
- 节点 Node: 树中的数据结构单元，每个节点具有父亲、子女指针、关键字（key）值和额外的颜色属性（Red 或 Black）。在红黑树中，一个节点可能有两个子女节点。
- 红链接 Red link: 从一个节点指向它的红色子女节点。
- 黑链接 Black link: 从一个节点指向它的黑色子女节点。
- 根节点 Root node: 树中的最顶层节点。
- 叶节点 Leaf node: 没有子女的节点。
- 路径 Path: 在树中从一个节点到另一个节点的一系列节点。
- 高度 Height: 一条路径上各个节点之间的距离。树的高度是指根节点到最远叶节点的最长路径长度。
- 森林 Forest: 由多棵互不相交的树组成的一个集合。
- 子树 Subtree: 一棵树中某个节点下的所有节点构成的子树。
- 外部节点 External node: 不在任何子树中的节点。
- 内部节点 Internal node: 在树中不是根或叶节点的节点。
- 双倍结点 Doubled node: 根节点下最多有一个红色节点和两个黑色节点，这种节点被称作“双倍结点”。
- 左旋 Left rotation: 通过对某些节点的重新塑造，将一个子树的根节点上移至另一个子树的左边。
- 右旋 Right rotation: 通过对某些节点的重新塑造，将一个子树的根节点下移至另一个子树的右边。
- 插入 Insertion: 将一个新元素添加到树中，并保持树的平衡性。
- 删除 Deletion: 从树中移除一个元素，并保持树的平衡性。
- 替代 Substitution: 使用一个不同的值替换掉某个节点。
- 上溢 Overflow: 当某个节点的子女个数超过限制时触发的异常情况。
- 下沉 Underflow: 当某个节点的子女个数太少时触发的异常情况。
- 比较 Comparison: 对两者之间的值进行比较。
- 折半 Searching: 以折半的方式探测某个值所在的位置。
- 查找 Lookup: 检索某个值的过程。
- 赋值 Assigning: 为某个变量分配内存。
- 分配 Allocating: 创建或者申请内存空间。
- 复制 Copying: 创建新的副本。
- 函数 Function: 执行特定任务的代码块。
- 参数 Parameter: 传递给函数的参数。
- 返回 Return value: 函数执行完毕后返回的值。
- 递归 Recursion: 调用自己以完成特定任务的代码。
- 栈 Stack: 数据结构，先进后出，后进先出，类似堆栈。
- 队列 Queue: 数据结构，先进先出，先进者先出队。
- 链表 Linked List: 数据结构，物理存储中按顺序存放多个元素。
- 数组 Array: 数据结构，物理存储中连续排列的多个元素。
- 深度 Depth: 从根节点到某一节点所经过的边的数量。
- 广度 Breath: 从某一节点到所有叶节点所经过的边的数量。
- 哈希 Hash: 用唯一标识符表示的数据结构。
- 最大值 Maximum Value: 数值中取最大的值。
- 最小值 Minimum Value: 数值中取最小的值。
- 大端模式 Big Endian: 高位字节排放在内存的起始地址处，低位字节排放在内存的结束地址处。
- 小端模式 Little Endian: 低位字节排放在内存的起始地址处，高位字节排放在内存的结束地址处。
- 轮询 Polling: 主动询问计算机是否有数据可读或可写的行为。
- 缓存 Cache: 主存中的临时存储器。
- 循环 Loop: 在代码中重复执行的语句块。
- 中断 Interrupt: 打断正在运行的程序执行流程的信号。
- 处理器 Processor: 可以理解为计算设备，执行指令和操作数据的单元。
- 寻址 Addressing: 操作系统根据内存地址定位数据或指令的过程。
- 寻回 Retries: 尝试执行失败的系统调用的次数。
- 文件 File: 系统内用于存储和管理数据的基本单位。
- 目录 Directory: 用于存储文件名的文件夹。
- 文件系统 FileSystem: 操作系统提供的文件组织方式。
- 文件格式 Format: 文件中保存数据的组织方式。
- 虚拟内存 Virtual Memory: 提供比实际内存更大的逻辑存储空间。
- IO 输入/输出 Input/Output: 信息输入计算机和输出设备的过程。
