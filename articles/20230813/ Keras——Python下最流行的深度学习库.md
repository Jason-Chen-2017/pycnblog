# 1.简介
  

Keras（读音类似kee-er），是一个基于Theano或者TensorFlow之上的一个高级神经网络API。它可以运行在Theano或TensorFlow之上，能够快速便捷地进行模型训练、评估和推断等。Keras提供了一系列高层次的抽象，使得开发者能够快速搭建复杂的神经网络。同时，它也提供友好的API接口和丰富的文档支持，方便使用者进行深度学习的研究和应用。
其主要特性如下：

1. 框架独立性：Keras可以运行于多个后端引擎，包括Theano、TensorFlow、CNTK和Caffe，并可通过设置环境变量切换使用哪个引擎。
2. 模型构建模块化：Keras拥有模块化的API，允许用户方便地构建复杂的神经网络。
3. 数据处理模块化：Keras对数据预处理过程进行了高度模块化的支持，包括图像数据的增强、归一化、序列特征的编码等功能。
4. 可视化工具：Keras可以很容易地将训练过程中的数据可视化，以帮助用户理解模型结构及优化效果。
5. 模型保存与加载：Keras提供了模型的保存与加载功能，用户可以灵活地选择保存哪些信息，从而实现模型的热启动和冷启动。
6. 迁移学习：Keras提供了迁移学习的功能，用户可以利用已有的模型参数快速训练新的模型，节省了训练时间。
7. 拓展性：Keras具有良好的拓展性，提供了许多模型层、激活函数、优化器等模块供用户使用。
# 2.基本概念及术语
## 2.1 基本概念
深度学习（deep learning）是机器学习的一个分支，它涉及到人工神经网络（artificial neural network，ANN）的发展。

深度学习的关键是使用多层非线性变换对输入的数据进行逐层抽象，并学习到数据的模式，从而使计算机具有识别、理解和模仿人类智能的方式。

深度学习的基本概念如下：

1. 模型：深度学习模型是指用来做一些特定的任务的函数集合。典型的深度学习模型可以分成两大类，即有监督学习模型（supervised learning model）和无监督学习模型（unsupervised learning model）。目前绝大部分的深度学习模型都属于有监督学习模型。例如图像分类、文字识别、语音合成、手写体识别等。

2. 数据集：深度学习需要大量的数据用于训练模型，这些数据被组织成为样本（sample）组成的数据集（dataset）。

3. 特征：数据集中每个样本都是由多个特征（feature）组成的。通常来说，深度学习模型会自动学习到有效的特征，而不是手工设计特征。

4. 标签（Label）：每条样本都有一个对应的标签（label），即目标变量，用来区分样本是否满足某种条件。标签一般用于监督学习模型训练，用来评价模型的好坏。

5. 损失函数（Loss function）：在深度学习模型训练过程中，用损失函数衡量模型预测结果与真实值之间的差距。

6. 优化器（Optimizer）：在深度学习模型训练过程中，优化器就是决定更新模型权重的方法。

7. 训练（Training）：深度学习模型通过迭代更新模型的参数，以提升模型的性能。在训练过程中，模型根据训练数据不断调整参数，使得损失函数的值降低。

8. 测试（Testing）：在训练完成之后，如果要评估模型的性能，就需要测试模型。测试过程中，模型根据测试数据计算出预测值的损失，用来衡量模型的准确率。

9. 超参数（Hyperparameter）：超参数是在模型训练前指定的参数，用来控制模型的训练过程，如学习率、正则化系数、批量大小等。超参数的选择非常重要，往往直接影响模型的收敛速度、泛化能力、精度、资源消耗等。

## 2.3 Python API
Keras除了提供命令行界面外，还提供了Python API。

以下是Keras的几个主要模块：

1. keras.models：该模块包含用于构建，编译，训练，评估和预测深度学习模型的类。

2. keras.layers：该模块包含用于构建深度学习模型各个层的类。

3. keras.activations：该模块包含各个激活函数的实现，可以通过激活函数对象对层的输出施加非线性转换。

4. keras.callbacks：该模块包含用于定义回调函数的类，这些函数可以在训练过程中定期执行特定操作。

5. keras.datasets：该模块包含用于加载常用的数据集的函数。

6. keras.utils：该模块包含用于建立模型和数据处理的实用函数。
