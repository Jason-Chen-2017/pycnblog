# 1.简介
  

# 你好，我是XX先生/小姐，很高兴认识到你对AI领域的研究以及创新有浓厚兴趣，正在努力拓展自己的专业技能。我向你请教一下关于你的上家公司的一些情况。
# 2.面试背景介绍
请你回忆一下自己做的第一份职业规划，你曾经计划着进入一个行业，到底是什么样的一个行业呢？这个行业的发展前景如何？你期待从事这个行业的主要工作是什么？
# 3.职业规划
你希望从事的职业方向包括但不限于以下几个方面：AI、机器学习、大数据、图像识别、语音识别、自然语言处理、数据库设计、Web开发、移动应用开发等。而在这些领域中，你最感兴趣的是哪个？
# 4.现状及过往经历
目前，你所在的行业还处于起步阶段，没有多大规模的企业可以依赖与支持，所以，你选择做这个行业的主要工作。你曾经做过的项目中，你觉得最有成就的一项是？
# 5.兼职培训
你比较喜欢兼职培训这种方式，每周都能收到海量的信息，让你快速学会新知识。你参加过几次兼职培训课堂吗？如果参加过，那一天对你来说有什么意义？
# 6.个人能力优势
你可以说说自己的个人优势和潜力，比如：擅长与客户沟通，精益求精，有良好的沟通技巧；熟悉各类编程语言，掌握常用的开发工具；有较强的独立解决问题能力，能够突破既有限制，创造新的商业价值。
# 7.推荐一位同学
据我所知，你之前在某家公司担任CTO（首席技术官）。他是你当初选择进入AI领域时刻被迫转型的大boss。他曾经提出了一个“A轮带动B轮”的战略，并且成功的推进了AI产业的发展。他也非常重视与员工之间的合作，并且帮助他们成长成职场精英。
因此，你应该以他为目标进行追踪了解，这位同学曾经做过什么有意义的事情，他最近有什么新的消息或产品等。
# 8.结论
你很乐意成为一名AI科研人员或工程师，但首先，你需要明白自己真正想要从事的方向以及自己有哪些优势才能更有价值地参与到AI的发展中来。你在兼职培训方面的表现给你留下了深刻的印象，这也能让你看到很多有趣的AI相关的内容。
最后，我要祝愿你们，在新的一年里，都能充分发挥自己的优势，成为AI领域的领军人物！也祝愿所有看过我的文章的人能从中受益！
