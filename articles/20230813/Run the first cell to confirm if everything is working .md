# 1.简介
  

什么是Colaboratory？

Colaboratory（“colab”）是一个可以运行Python脚本的云端服务器，可以在浏览器中编辑、执行、分享Python代码。它是由Google Research开发的基于Web的免费Jupyter笔记本环境，可用来编写和执行机器学习、数据处理、统计建模、AI和科学计算等项目。该服务于研究人员、教育者、工程师、学生、实验室和研究机构提供了一个高效的开发工具。

本文主要讲述如何在Colab上创建一个Jupyter Notebook并运行Python代码。

注意：请确保您已安装Chrome浏览器或其他支持的浏览器，且已经连接到互联网。

## 条件语句

### if…elif…else语句

if语句是条件判断的关键字，根据判断条件的成立与否执行相应的分支语句。

```python
num = 3
if num > 0:
    print('Positive')
elif num == 0:
    print('Zero')
else:
    print('Negative')
```

这里例子中的`if`语句判断变量`num`的值是否大于零，如果大于零则输出字符串“Positive”，否则判断变量`num`的值是否等于零，如果等于零则输出字符串“Zero”，否则判断变量`num`的值是否小于零，如果小于零则输出字符串“Negative”。

### for循环

for循环是迭代语句，用于遍历序列中的每个元素。

```python
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)
```

这里例子中的`for`语句依次对列表`fruits`中的每一项进行遍历，并将当前项赋值给变量`fruit`，然后打印出当前项的值。

### while循环

while循环也是迭代语句，但是其条件表达式不确定。

```python
n = 0
while n < 5:
    print(n)
    n += 1
```

这里例子中的`while`语句首先初始化变量`n`的值为零，然后判断变量`n`的值是否小于五，如果小于五则会继续执行循环体，并打印出变量`n`的值，随后再次对变量`n`进行递增操作，这样会一直循环到变量`n`的值大于或等于五。
