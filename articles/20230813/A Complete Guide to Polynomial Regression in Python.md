# 2 基本概念术语说明
## 2.1 什么是多项式回归？
多项式回归（Polynomial Regression）是一种用来描述由不同输入变量决定的输出变量间关系的模型。它是在统计学和计算机科学中使用的一种回归分析方法。

在多项式回归中，利用最高次多项式来拟合数据，使得输出结果更加符合真实的样本。通过选择最佳的多项式次数，从而可以将输入数据映射到输出数据上。

多项式回归在数据集中的作用主要有两个方面。首先，它可以提供数据的非线性拟合能力，能够找到数据中隐藏的模式；其次，它还可以产生预测值，预测值有利于对未知数据进行预测。

## 2.2 多项式回归的工作原理
多项式回归的基本想法是在给定数据点集中的点到极小距离处的一个连续曲线来拟合数据集。这里的“点”可能代表某个输入变量的值或者某个输出变量的值，而“曲线”则对应于函数形式。

为了表示函数形式，我们使用多项式来描述函数的形状，也就是说，多项式回归模型由一系列次数不同的多项式构成。

具体地，假设数据点集$\left\{(x_i, y_i)\right\}_{i=1}^n$满足独立同分布，且服从均值为$\mu$、方差为$\sigma^2$的正态分布。那么多项式回归模型可以表示为：
$$y=\theta_0+\theta_1 x+\theta_2 x^2+...+\theta_k x^k+e,$$
其中，$\theta=(\theta_0,\theta_1,\cdots,\theta_k)$为待求参数向量，$k$为多项式的最高次数，$e$是误差项。

误差项$e$代表模型预测值与真实值之间的误差，损失函数（loss function）定义为
$$L(\theta)=\frac{1}{2n}\sum_{i=1}^{n}(h_{\theta}(x_i)-y_i)^2,$$
其中，$h_{\theta}(x)$是多项式函数，等于$\theta_0+\theta_1 x+\theta_2 x^2+...+\theta_k x^k$.

因此，多项式回归的目标就是寻找最佳的参数向量$\theta$，使得损失函数达到最小。损失函数的最小值对应于使得拟合误差最小的参数向量，可以通过梯度下降算法或者其他优化算法来求解。

## 2.3 为何使用多项式回归？
使用多项式回归的原因有以下几点：
1. 拥有较强的预测力。多项式回归模型的优点之一就是它具备较强的预测力，因为它可以提供非线性拟合能力，能够对数据集中潜藏的模式进行有效的描述。因此，当数据存在复杂的非线性关系时，多项式回归模型可以帮助我们发现它们。

2. 具有较高的可靠性。一般来说，数据规模越大，越容易出现过拟合的问题。但是，多项式回归可以避免此类问题发生。这是因为多项式回归没有像线性回归那样的局部极值问题。这就保证了多项式回归的预测结果具有高度的可靠性。

3. 方便解释。多项式回归模型的表达式简单易懂，而且易于使用，可以帮助我们直观地理解数据。

## 2.4 多项式回归模型的缺陷
虽然多项式回归模型的表现不错，但仍然有一些局限性和缺陷，包括：
1. 模型复杂度。多项式回归模型往往会比线性回归模型更复杂，尤其是在多项式次数较高的时候。这主要是因为多项式回归模型使用了更多的特征，参数更加多，导致模型变得更加复杂。

2. 准确率低。多项式回归模型在很多情况下并不能很好地拟合数据，尤其是对于非线性数据集。这是因为多项式回归模型往往无法很好地拟合非线性数据，只能在一定程度上拟合。此外，对于数据集中的噪声也不是很敏感。

3. 对异常值的敏感度不足。对于异常值（outlier），多项式回归模型并不能很好地抗衡。此外，对于离群点（extreme value point）也是如此。

4. 参数估计不确定。多项式回归模型的参数估计不够精确。原因是参数估计本质上是无穷逼近问题，其结果存在较大的不确定性。

## 2.5 多项式回归模型的类型
多项式回归模型主要有两种类型：
1. 线性回归模型。线性回归模型意味着我们的目标函数是一个线性函数。即，假设$h_{\theta}(x)$只依赖于输入变量$x$，不依赖于任何其他变量。此时，多项式回归模型的表达式可以简化为
$$y=\theta_0+\theta_1 x.$$

2. 多元回归模型。多元回归模型意味着我们的目标函数依赖于多个输入变量。通常情况下，目标函数是一个多项式函数，即，$h_{\theta}(x)$可以是一个多项式函数。此时，多项式回归模型的表达式可以简化为
$$y=\theta_0+\theta_1 x+\theta_2 x^2+...+\theta_k x^k.$$

## 2.6 多项式回归模型的优点
1. 可解释性强。多项式回归模型的表达式比较简单，易于理解。因此，它非常适合用来解释数据，检查数据是否符合模型预期，以及对数据产生预测。

2. 模型鲁棒性高。多项式回归模型具有较高的鲁棒性，因为它不受过拟合的影响，而且它的损失函数是平滑的。这使得多项式回归模型在处理非线性关系的数据集时具有很好的效果。

3. 可选择性强。多项式回归模型的最高次方次项允许我们对数据的局部模式进行建模，从而发现数据中隐藏的信息。

4. 计算高效。多项式回归模型的训练时间和内存开销都很少。这使得它可以在大数据集上运行。
