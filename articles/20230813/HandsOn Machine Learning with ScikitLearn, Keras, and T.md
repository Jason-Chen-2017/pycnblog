# 1.简介
  

机器学习（Machine learning）是指利用数据来训练模型，使计算机能够从中提取有效信息，并做出预测或决策，而不是依赖于规则或者启发式的方法。在过去几年里，机器学习领域取得了显著的成果，特别是在视觉、文本、音频、图像等多种领域，得到广泛应用。然而，传统的基于规则或者启发式的方法在解决一些现实的问题时效率不高，导致效率低下或者无效。因此，出现了许多基于机器学习的新方法和算法，这些方法既可以处理复杂的数据集，又可以高效地处理大型数据量。本书提供了可靠的机器学习工具和技术，能够帮助开发人员和数据科学家更好地理解和应用机器学习。作者首先会介绍机器学习的基本概念、术语及其重要性。然后会介绍机器学习过程中经常使用的算法，以及如何使用Scikit-learn、Keras和TensorFlow进行实现。最后，还会给出实际案例，展示如何使用机器学习工具构建分类器、回归模型，以及处理文本、时间序列数据，以及一些最佳实践。读者将学到一些快速入门的机器学习技巧，并且了解更多关于机器学习的知识。此外，通过阅读本书，读者将了解到机器学习的最新进展，以及该领域所涉及到的各个方面所用的机器学习技术和工具。
