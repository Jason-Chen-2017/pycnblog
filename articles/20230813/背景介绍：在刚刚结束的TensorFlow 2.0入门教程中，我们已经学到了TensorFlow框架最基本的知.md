# 2.基本概念术语说明

## Tensor
Tensor是由多维数组组成的数据类型。在机器学习的研究中，通常把数据表示为张量。张量指的是具有特定形式的多维矩阵，其元素可以是数字、符号、字符串等。每个张量都有一个唯一标识符，并且可以通过坐标的方式来访问其中的元素。张量的维数称为秩，张量的每个轴上的分量也称为轴向量。一般来说，张量的秩和轴向量数量是相同的。例如，一个三维的张量的秩为3，每个轴上有三个轴向量。

在TensorFlow中，张量主要用于存储和处理数据，比如机器学习模型的输入输出，图像数据，文本数据等。张量也可以被视为矢量化的矩阵，这意味着张量的所有元素可以批量运算，可以加快运算速度。张量还提供了一些特殊的运算功能，比如矩阵分解、张量积等。

## Operation
Operation是对张量进行各种运算操作得到的新张量，包括线性代数运算、矩阵乘法、softmax分类、卷积等。操作可以看作是对张量进行一系列变换，最终得到新的张量。操作的结果可以保存在变量中供后续使用，也可以直接作为输出返回给调用者。

## Session
Session是TensorFlow中用来执行计算图的一个上下文管理器。当需要运行图时，应该先创建会话，再通过该会话来执行图。

## Variable
Variable是可以在计算过程中被更新的值。它类似于普通的变量，但可以参与到训练中，并根据反向传播算法自动更新。Variable可以保存模型参数、中间计算结果等。

## Gradient
Gradient是损失函数对模型参数的偏导数，它是优化算法的目标函数。在训练过程中，梯度下降算法会不断调整模型的参数，使得损失函数最小，直到模型收敛。

## AutoGraph
AutoGraph是TensorFlow 2.0新增的特性，它可以将Python代码转换成图模式，并在图模式中运行，这样就可以实现动态图的效果。

## Eager Execution
Eager Execution是在TensorFlow 2.0默认使用的模式，它允许直接在Python环境中执行计算图，不需要构建图。这样做有助于调试程序，并提供直观的感受。

## Module
Module是TensorFlow 中用于模块化编程的机制，它可以将计算图划分成多个子图，从而更好地控制内存占用。

## Layer
Layer 是TensorFlow中提供的一套高级API，它提供大量的预定义的神经网络层，可以快速构建复杂的神经网络。

## Estimator
Estimator 是TensorFlow中提供的一套高级API，它提供了一个标准流程，可以简化模型的训练过程。
