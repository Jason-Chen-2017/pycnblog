# 1.简介
  

根据《中国图书出版行政管理体制改革方案》，1997年《中华图书报》在中国共产党领导下恢复活跃，并在1998年4月创刊。近几年来，我国图书出版行政管理体制的建设取得了迅速发展。新世纪图书出版领域迎来了新的发展机遇。因此，为了推动中国图书出版事业的发展，我国图书出版管理部门组织制定了《中国图书出版行政管理体制改革方案》。本方案明确提出，从2008年至2020年，我国图书出版管理部门将全面实施绩效评价体系、分类分级制度、主管部门垂直化管理、责任制约机制、严格审批制度等方面的改革。其中，绩效评价体系的建立是最重要的一项改革，它涉及到各级各类单位的评估工作。随着绩效评价体系建设的不断完善，图书出版领域将形成更科学有效的管理体系。

# 2.分类分级制度
分类分级制度是指编制图书的基础。它的目的是通过对图书的各项属性进行归纳总结，使读者在阅读时更快找到自己需要的信息，并帮助图书管理员加强图书库的整体掌控能力。分类分级制度的建立，既要考虑图书分类的合理性，又要保证各分类下的图书数量均衡。分类分级的基本原则是“主体-客体-主题”，即“人-物-事”。目前中国图书出版行政管理部门已实施了五种分类标准，分别是按内容类型、业务范围、使用目的三个维度划分；按图书主题和图书时代划分；按社会经济效益划分；按图书类型划分；按作者类型划分。除此之外，一些国家还实行了基于图书特色或风格的二级分类法。

# 3.绩效评价体系
绩效评价体系是指建立符合中国绩效评价的评价规范，评价对象包括出版单位和图书管理员。评价中心制定了一套图书管理绩效指标，出版单位每年或每季度向评价中心提交其编辑、校对、销售、采购、信息化、统计等工作的实际效果，评价中心按一定的办法对这些数据进行综合分析，并根据评价结果提供给出版单位建议。这种评价制度能够对出版单位的图书管理工作做出及时反馈，提高工作质量和效率，并保障出版单位对图书的持续供应。

# 4.主管部门垂直化管理
图书管理部门的垂直化管理，意味着将不同的业务环节统筹到同一个主管部门，并且设置不同级别的主管人员，形成独立的管理团队。这对于企业和图书管理部门都有好处。首先，降低管理成本和管理难度，减少不必要的干预和职权划分。其次，简化了审核、库存、流通等环节的流程，利于促进各方之间的协调配合，有效防止腐败行为蔓延。再次，为各主管部门之间互相了解和沟通奠定基础，培养起沟通文化和服务意识，提升工作效率和团队凝聚力。

# 5.责任制约机制
图书管理部门制定相关管理规范后，如何建立责任制约机制？责任制约机制是指对图书管理部门的工作任务进行认真细致地规划，明确各项任务的职责主体和责任范围，并对各类人员进行责任划分，形成一套工作计划和流程安排。其主要任务包括制订工作计划、落实工作规范、监督检查、问责纠正，以确保图书管理工作按要求开展、有效执行。
