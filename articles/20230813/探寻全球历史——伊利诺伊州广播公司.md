# 1.简介
  

在《伊利诺伊州广播公司》这个传奇的美国电视台的故事里，有许多令人着迷的细节值得我们一读。作为一个世界上最具影响力的商业电视台，它的基础设施和资金实力都使它保持了巅峰不衰的地位。它是一个互联网时代的现代化先锋、颠覆性的创新者和转型期的创业者。我们今天来谈论它的历史。
## 一、前言
　　伊利诺伊州广播公司（Elizabeth Tonty Broadcast Company）是美国第一家百万级收入的美国电视频道。其办公地址位于北卡罗来纳州的伊利诺伊市，是一个知名的业界精英团队的聚集点。该公司建立于1919年，主要经营影视娱乐产品，并拥有自己的影院群落。由于其优秀的形象和长久的历史，加之良好的商业模式，使得它成为美国电视业界的领头羊。
　　如今，伊利诺伊州广播公司已经成为全球最大的百万级收入电视台，每月产生超过1亿美元的收入。在过去几十年里，它一直处于全国版权保护局的监控下，受到当地政府严厉打击，但同时也逐渐走出这一困境。其影响力不断扩大，其海外影响也日益扩大。总体来说，伊利诺伊州广播公司是一个富有成效且具有典范意义的企业。
　　然而，本文所讨论的只是单个电视台的历史，却忽略了这家公司的全球影响。历史可以用时间轴的方式呈现，从伊利诺伊州广播公司的创立开始，到其主导全球内容领域的关键时刻——《卫报》，再到它的崛起与衰落……如果我们回溯这些关键节点，我们将看到伊利诺伊州广播公司对于当代电视业的重要贡献。
　　下面我们就此诸多历程展开我们的探索。
# 2.基本概念术语说明
## 2.1 直觉
　　当您第一次听说伊利诺伊州广播公司的时候，或许会感觉像是在自我介绍。其名字让人耳熟能详，似乎与众不同。但是随着时间的推移，我们越来越多地了解到，伊利诺伊州广播公司其实不是那么容易被认识的名字。它曾经是一个古老的电视台，但是为了应对当下的新型媒体环境，它在不断发展壮大中发生了巨大的变化。
　　因此，为了更好地理解该公司及其发展历史，我们首先需要搞清楚一些基本概念。
### 2.1.1 TV
　　“TV”(television)是指利用接收、显示以及控制信息的技术，向观众提供连续、动态的图像和声音信息的广播电视系统。在电视业界，俗称“康健”(health-oriented)、“时尚”(fashionable)、“爱情”(romantic)、“动作”(action-packed)等，虽然这些标签并不能完全准确反映出电视的发展趋势，但是它们给人的印象是电视技术的进步带来了新的价值观。它们包括生活、工作、娱乐和科技四种价值层次。通过不同的价值层次，人们能够获得适合自己需求的内容。
### 2.1.2 大画廊
　　电视具有“大画廊”(mass gallery)的特征。一部电视的观赏时段大致为24小时。观看的感官享受是无穷的，并且随着时间的推移，观众也愈来愈多，对这一片土地的习惯也越来越习惯。因此，大画廊也是一种能力。这种能力给电视制造者提供了巨大的生产力。每个画面都是非常有创造性的，而且能够精心设计。对大画廊来说，想把所有东西都呈现在观众面前，那显然是不可能的。因此，大画廊除了具有大画廊的艺术追求外，还需兼顾品质、时效性、经济性。
### 2.1.3 时代
　　所谓时代，就是指一种社会制度或者文化传统，其发展往往引起社会结构的重大变革，带来重大变革后果。这个过程可能会持续很久，并且不可预测。所以，所谓的“突变”，既有科技的进步带来的技术革命，也有更广阔的文化视野带来的价值观变革。伊利诺伊州广播公司在1927年由音响制造商琼斯（Pecsi）与莱昂哈德（Lloyd Heard）共同创建。他们希望通过其自主创造的产品，改善世界。电视业的创始人约翰·W·伯默（John Backman）早在19世纪末期就提出了“大革命”的概念，其目标是在电视制造业发展壮大之后，通过持续的技术革命、产品升级和文化转型，最终建立起新兴的电视产业。这其中有一个关键节点，就是1933年10月22日，《卫报》杂志创刊。
　　该报在创刊两周后便荣登纽约时报的首页。它介绍了自己如何开拓了一个新的电视新闻业，并向读者提供了非常有趣的、不可替代的体验。它不仅改变了当时电视新闻的领域，也改变了读者的阅读方式，影响了整个社会。《卫报》也因此得以成为全美最畅销的时评报纸，并成为国际电视新闻的中心。
　　不过，另一方面，《卫报》的创刊对于电视业的发展有着至关重要的作用。这是因为它为伊利诺伊州广播公司的建设提供了良好的舞台，它借助新闻的平台、宣传的形式以及播放的模式，在媒体的介入下，利用大众传播手段进行宣传，使得电视业取得了一定的影响力。在这个过程中，它逐渐形成了自己的品牌。在经历了惊涛骇浪的创刊之初，《卫报》还是一家小型的小报纸，它的影响力并不大。然而，随着时间的推移，它逐渐成长为全国性报纸，它的影响力也越来越大。
### 2.1.4 信息时代
　　“信息时代”(information age)，也叫“互联网+时代”。是指从1990年代到2010年代，全球信息通信技术和电信网络快速发展。信息技术的飞速发展促进了新媒体的发展，而在这个过程中，电视则占据着举足轻重的位置。在信息时代，电视所扮演的角色，就像新闻一样，必须进行信息的深度整合，并且采用大众化的消费方式。
　　信息时代的到来使得电视行业获得新的机遇，它带来了巨大的红利。可以说，伊利诺伊州广播公司在20世纪末期，迎来了第一次变革，实现了重新定义自己的使命。它从电视新闻转向了在线教育、社交网络以及多媒体产品的新兴领域，创造了无可替代的商业价值。
### 2.1.5 文化与意识形态
　　电视制作及其所传达的消息，既取决于电视的制作者，也取决于观众的接受和喜好。这就涉及到对电视制作和观众接受的文化、意识形态等因素的影响。
　　1927年，伯默与莱昂哈德决定创立电视公司，他们创作了第一部完整的电视节目——《电影通告栏》。这部节目创造了纪录片制作的新流派。这件事导致了电影产业的重大变化。
　　第二个重要影响因素是伯默本人对科幻小说的兴趣。他本人对科幻小说有极大的热情，并且鼓励自己参加科幻写作比赛，通过自己的创造性活动来推动科幻小说的发展。伯默后来成为了著名科幻作家。
　　第三个影响因素是欧洲启蒙运动。由于西方的文化影响，伊利诺伊州广播公司在创立之初就注重文化。它在设计电视产品的同时，还要保证这一产品的适应性。因此，它通过各种手段，宣传伯默的代表作——《阿凡达》(Avatar)。
　　第四个影响因素是邵氏体系。19世纪末期，伯默提出了邵氏体系的概念，认为“知识的源泉并非天赋而是通过长期的沉淀而得到的”。因此，伊利诺伊州广播公司在设计电视产品的同时，也要承担知识产权保护的责任。
　　第五个影响因素是电影制作者对费城电影节的支持。20世纪，伊利诺伊州广播公司参与了费城电影节。这项活动吸引了大量的观众，而且还创造了新的文化气息。另外，费城电影节的成功与当时电视业的火爆相辉映，也促使伊利诺伊州广播公司在创立之前与费城电影节建立了联系。
　　总之，伊利诺伊州广播公司从1927年开始，创立伯默、莱昂哈德、费城电影节、《卫报》等，积累了丰富的经验，并通过持续的创新，最终建立起了一家大型的电视公司，并发展成为全球最大的百万级收入电视台。
# 3.核心算法原理和具体操作步骤以及数学公式讲解
　　我们接下来将讨论一下伊利诺伊州广播公司的运营机制。首先，我们要了解一下电视的操作方式，主要分为以下三类:

　　　　1.普通频率调频器TV,例如爱立信的PAL和SECAM频率调频器，由主要靠收费的方式进行播出。

　　　　2.高清晰度产品High Definition Televisions (HDTV),这种产品一般采用广角幕和镭射屏幕，技术含量高，但却无法完全克服信噪声的影响，容易出现明显的“黑边”现象。

　　　　3.超高清晰度产品Ultra High Definition Televisions (UHDTV),采用的技术是真正的4K制作，分辨率提高到了3840*2160,但价格相对较高。


        其次，我们来介绍一下电视的营运过程。

　　　　1.选址和布局

        在开始启动电视业务之前，我们需要找到合适的地方，选择好的安装位置能够增加用户的观影体验。伊利诺伊州广播公司的选址条件非常好，坐落在伊利诺伊市的中心，离主要的公共交通设施较近，有近距离的饭店、咖啡馆等服务区。

        2.固定投资

        当确定了电视项目之后，我们需要进行固定投资，即确定其初始投资金额。伊利诺伊州广播公司的固定投资周期为三个月，每月产生的利润大约在2.5万美元左右。

        3.选片

        在选片阶段，我们需要确定电视节目的类型，包括主題片和访谈片两种类型。主題片的类型可以是各类经典电影、电视剧；访谈片则侧重于听众的个人感受、议题，一般是有针对性的问题。

        4.制作电视节目

        通过观看大量主題片和访谈片，我们可以发现不同类型的电视节目都有其独特的风格。因此，制作电视节目需要仔细考虑节目的构思、内容创意和方向，并结合相关的人物、组织等进行创作。

        5.线路和产品定价

        根据节目类型和地域情况，我们可以选择不同的产品定价策略，比如：普通频率调频器的价格大约在2000~4000美元/套；高清晰度产品和超高清晰度产品的价格也存在差异，一般高清晰度产品价格在4000～6000美元/套，超高清晰度产品的定价比较高。

        6.维护、升级和维修

        电视一直都是我们的第二生命，其生死存亡都与我们息息相关。我们需要时刻关注电视的维护状态，每年都会有许多维护人员上门，对电视进行必要的维护，确保它持续运行正常。

        7.股东分配

        每个出资的股东都将获得一定份额的股权。伊利诺伊州广播公司共有十几个股东，并没有任何上市公司的独家股权。因此，股东们在产生利润的同时，也会从公司获得分红。

        8.版权及销售

        在电视的出售环节，需要注重产品的保密性，这样才不会造成损失。伊利诺伊州广播公司的电视节目通常属于盈利性产品，所以一般会与某个电视台合作，通过商业授权获得授权，然后再卖给大众。
