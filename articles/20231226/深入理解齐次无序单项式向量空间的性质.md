# 2. 背景介绍

为了更好地理解 HPVS，我们首先需要了解一些基本概念，如向量空间、多项式和齐次多项式。

## 2.1 向量空间

向量空间是一个包含向量的集合，这些向量可以通过线性组合和标量乘法得到。线性组合是将两个向量相加，而标量乘法是将一个标量乘以一个向量。向量空间必须满足以下几个条件：

1. 向量的加法是关于向量的闭操作，即对于任何两个向量 a 和 b，a + b 也是向量空间中的一个向量。
2. 标量乘法是关于向量的闭操作，即对于任何向量 a 和标量 α，α * a 也是向量空间中的一个向量。
3. 向量空间中的每个向量都有其负向量，即对于任何向量 a，存在一个负向量 -a，使得 a + (-a) = 0，其中 0 是向量空间的零向量。

一些常见的向量空间包括：

- 实数域下的 n 维欧几里得空间（R^n）
- 复数域下的 n 维欧几里得空间（C^n）

## 2.2 多项式

多项式是一个数学函数，它可以表示为一组终止的有限项的线性组合。每个项都是一个变量的乘积，其中变量可以是实数、复数或其他数字类型。多项式通常用符号 x、y、z 等表示，其中 x、y、z 是变量。

例如，对于三元多项式 f(x, y, z)，它可以表示为：

f(x, y, z) = 2x^3 + 3xy^2 + 4z^5

## 2.3 齐次多项式

齐次多项式是一种特殊类型的多项式，其所有项的指数相等。对于一个 n 元齐次多项式，它可以表示为：

f(x_1, x_2, ..., x_n) = a_0x_1^d_1 * x_2^d_2 * ... * x_n^d_n

其中 a_0 是一个常数，d_1, d_2, ..., d_n 是非负整数，满足 d_1 + d_2 + ... + d_n = d。

例如，对于一个三元齐次多项式 f(x, y, z) 的 d = 2 的例子，它可以表示为：

f(x, y, z) = ax^2y^2 + by^2z^2 + cx^2z^2
