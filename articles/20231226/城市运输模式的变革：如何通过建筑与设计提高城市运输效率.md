### 3.1 建筑空间效率

建筑空间效率是指建筑空间内的利用率。我们可以使用以下公式来计算建筑空间效率：

$$
\text{空间效率} = \frac{\text{建筑面积}}{\text{总面积}} \times 100\%
$$

### 3.2 公共交通效率

公共交通效率是指公共交通系统中的人流量与车辆数量之间的关系。我们可以使用以下公式来计算公共交通效率：

$$
\text{公共交通效率} = \frac{\text{人流量}}{\text{车辆数量}}
$$

### 3.3 非动力运输效率

非动力运输效率是指非动力运输方式（如步行、骑自行车等）在城市中的利用率。我们可以使用以下公式来计算非动力运输效率：

$$
\text{非动力运输效率} = \frac{\text{非动力运输人数}}{\text{总人数}} \times 100\%
$$

### 4.1 计算建筑空间效率

我们首先需要计算一个建筑的空间效率。我们可以使用以下代码来计算建筑空间效率：

```python
import numpy as np

def building_space_efficiency(total_area, built_area):
    efficiency = (built_area / total_area) * 100
    return efficiency

total_area = 10000  # 总面积，单位：平方米
built_area = 8000  # 建筑面积，单位：平方米

efficiency = building_space_efficiency(total_area, built_area)
print("建筑空间效率：{:.2f}%".format(efficiency))
```

```python
def public_transport_efficiency(passenger_flow, vehicle_num):

    return efficiency

```
