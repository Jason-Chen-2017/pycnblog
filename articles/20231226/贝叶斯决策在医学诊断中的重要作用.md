# 4. 贝叶斯决策在医学诊断中的具体代码实例和解释

以下是一个具体的代码实例，用于演示贝叶斯决策在医学诊断中的应用：

```python
import numpy as np

# 构建条件概率模型
p_d = np.array([0.9, 0.8, 0.7, 0.6])
p_nd = np.array([0.1, 0.2, 0.3, 0.4])

# 计算条件概率
def calc_conditional_probability(p_d, p_nd, symptom):
    p_d_given_symptom = (p_d * (1 - symptom)) / (1 - p_nd)
    p_nd_given_symptom = (p_nd * symptom) / p_d
    return p_d_given_symptom, p_nd_given_symptom

# 选择最佳决策
def bayes_decision(p_d_given_symptom, p_nd_given_symptom, loss_function):
    decision = np.argmax(p_d_given_symptom * (1 - loss_function) + p_nd_given_symptom * loss_function)
    return decision

# 损失函数
loss_function = 0.2

# 症状
symptom = 0.5

# 计算条件概率
p_d_given_symptom, p_nd_given_symptom = calc_conditional_probability(p_d, p_nd, symptom)

# 选择最佳决策
decision = bayes_decision(p_d_given_symptom, p_nd_given_symptom, loss_function)

print("最佳决策：", decision)
```

在这个代码实例中，我们首先构建了一个条件概率模型，然后计算了条件概率，最后根据计算出的条件概率和损失函数选择了最佳的决策。
