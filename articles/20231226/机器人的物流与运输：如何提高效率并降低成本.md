## 4.1 代码实例

以下是一个简单的机器人在物流与运输中的代码实例：

```python
import numpy as np
import rospy
from geometry_msgs.msg import Twist

class RobotController:
    def __init__(self):
        rospy.init_node('robot_controller', anonymous=True)
        self.publisher = rospy.Publisher('/cmd_vel', Twist, queue_size=10)
        self.rate = rospy.Rate(10)
        self.speed = 0.5
        self.angle = 0.0

    def move(self, direction):
        if direction == 'forward':
            linear_vel = self.speed
            angular_vel = 0.0
        elif direction == 'backward':
            linear_vel = -self.speed
            angular_vel = 0.0
        elif direction == 'left':
            linear_vel = 0.0
            angular_vel = -self.speed
        elif direction == 'right':
            linear_vel = 0.0
            angular_vel = self.speed
        else:
            linear_vel = 0.0
            angular_vel = 0.0

        cmd_vel = Twist()
        cmd_vel.linear.x = linear_vel
        cmd_vel.angular.z = angular_vel
        self.publisher.publish(cmd_vel)
        self.rate.sleep()

if __name__ == '__main__':
    try:
        robot_controller = RobotController()
        while not rospy.is_shutdown():
            direction = raw_input('Enter direction (forward/backward/left/right): ')
            robot_controller.move(direction)
    except rospy.ROSInterruptException:
        pass
```

上述代码实例主要包括以下几个部分：

1. 导入必要的库：通过`import numpy as np`和`import rospy`导入必要的库。
2. 定义机器人控制类：通过`class RobotController:`定义机器人控制类，主要包括初始化节点、发布者、速度、角速度等属性。
3. 定义移动方法：通过`def move(self, direction):`定义移动方法，主要包括根据方向设置线速度和角速度，并发布移动命令。
4. 主程序：通过`if __name__ == '__main__':`定义主程序，主要包括创建机器人控制对象，并实现根据用户输入的方向进行移动的功能。
