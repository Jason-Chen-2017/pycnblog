## 2.1 切比雪夫距离的定义

切比雪夫距离是一种度量两个概率分布之间距离的方法，它通过比较两个分布在某个特定空间上的分布密度函数来计算。具体来说，切比雪夫距离定义为：

$$
d_{C}(P, Q) = \max_{x \in \mathcal{X}} |p(x) - q(x)|
$$

在这里，我们以 Python 语言为例，给出一个计算切比雪夫距离的简单代码实例。

```python
import numpy as np

def chernoff_distance(p, q):
    """
    Calculate the Chernoff distance between two probability distributions p and q.
    """
    max_distance = 0
    for x in np.arange(0, 1, 0.1):
        distance = abs(p[int(x * 10)] - q[int(x * 10)])
        if distance > max_distance:
            max_distance = distance
    return max_distance

# Example usage
p = np.array([0.1, 0.2, 0.3, 0.4])
q = np.array([0.2, 0.1, 0.35, 0.15])
distance = chernoff_distance(p, q)
print("Chernoff distance:", distance)
```
