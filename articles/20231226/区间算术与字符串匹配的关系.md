## 3.1 区间算术基础
区间算术主要关注于处理有界数列的运算，如求和、差等。例如，给定一个有界数列 {a1, a2, ..., an}，我们可以计算其前缀和、后缀差等。

### 3.1.1 前缀和
前缀和是指在数列中，从第一个元素开始累加到当前元素的和。例如，给定一个数列 {a1, a2, a3, a4}，其前缀和为 {a1, a1+a2, a1+a2+a3, a1+a2+a3+a4}。

### 3.1.2 后缀差
后缀差是指在数列中，从最后一个元素开始累加到当前元素的差。例如，给定一个数列 {a1, a2, a3, a4}，其后缀差为 {a4-a3, a4-a2, a4-a1}。

### 3.1.3 区间和
区间和是指在数列中，从一个元素开始累加到另一个元素的和。例如，给定一个数列 {a1, a2, a3, a4}，其区间和为 {a2-a1, a3-a2, a4-a3}。

## 3.2 字符串匹配基础
字符串匹配则关注于在给定的字符串中查找子字符串的问题。例如，给定一个字符串 S 和一个子字符串 T，我们需要判断 T 是否是 S 的子串。

### 3.2.1 暴力法
暴力法是一种简单的字符串匹配方法，它通过逐个比较字符串中的每个字符来判断子字符串是否匹配。例如，给定字符串 S = "abcdefg" 和子字符串 T = "abcd"，通过逐个比较字符串中的每个字符，我们可以发现 T 是 S 的子串。

### 3.2.2 字符串哈希
字符串哈希是一种高效的字符串匹配方法，它通过计算字符串的哈希值来判断子字符串是否匹配。例如，给定字符串 S = "abcdefg" 和子字符串 T = "abcd"，通过计算字符串的哈希值，我们可以发现 T 是 S 的子串。

### 3.2.3 KMP算法
KMP算法是一种高效的字符串匹配方法，它通过预处理子字符串的next数组来减少不必要的比较。例如，给定字符串 S = "abcdefg" 和子字符串 T = "abcd"，通过预处理T的next数组，我们可以快速判断T是否是S的子串。

### 3.2.4 Rabin-Karp算法
Rabin-Karp算法是一种高效的字符串匹配方法，它通过计算字符串的哈希值来判断子字符串是否匹配。例如，给定字符串 S = "abcdefg" 和子字符串 T = "abcd"，通过计算字符串的哈希值，我们可以发现 T 是 S 的子串。

```python
def prefix_sum(nums):
    prefix_sums = [0] * (len(nums) + 1)
    for i in range(1, len(nums) + 1):
        prefix_sums[i] = prefix_sums[i - 1] + nums[i - 1]
    return prefix_sums

def suffix_diff(nums):
    suffix_diffs = [0] * (len(nums) - 1)
    for i in range(len(nums) - 1):
        suffix_diffs[i] = nums[i] - nums[i + 1]
    return suffix_diffs

def interval_sum(nums, left, right):
    if left > right:
        return 0
    if left == 0:
        return prefix_sum(nums)[right + 1]
    return prefix_sum(nums)[right + 1] - prefix_sum(nums)[left]
```
