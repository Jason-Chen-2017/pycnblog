在本节中，我们将通过一个具体的代码实例来详细解释 FaunaDB 的工作原理。

假设我们需要使用 FaunaDB 创建一个包含名字、年龄和职业的数据库。以下是创建这个数据库的具体代码实例：

```
// 创建数据库
CREATE DATABASE people;

// 创建数据集
CREATE COLLECTION people.names;

// 插入数据
INSERT people.names {
  data: {
    name: "John Doe",
    age: 30,
    occupation: "Software Engineer"
  }
};

// 查询数据
GET people.names {
  data: {
    name: "John Doe"
  }
};

// 更新数据
UPDATE people.names {
  data: {
    name: "John Doe",
    age: 31,
    occupation: "CTO"
  }
};

// 删除数据
DELETE people.names {
  data: {
    name: "John Doe"
  }
};
```

在这个代码实例中，我们首先创建了一个名为 "people" 的数据库，然后创建了一个名为 "names" 的数据集。接下来，我们插入了一个新的数据记录，包括名字、年龄和职业。然后，我们查询了这个数据记录，更新了年龄和职业，并最后删除了这个数据记录。
