## 4.1 分布式计算的具体代码实例

### 4.1.1 分布式计算的具体代码实例

一个简单的分布式计算的具体代码实例是计算大整数的和。假设我们有一个大整数，它的值为1000000000000000000，我们需要计算这个整数的和。我们可以将这个整数划分为多个部分，分布在多个计算节点上，然后分配任务给各个计算节点，让它们计算各自的部分和，最后通信交换结果，得到最终的和。

```python
import multiprocessing as mp

def sum_part(start, end):
    result = 0
    for i in range(start, end):
        result += i
    return result

if __name__ == '__main__':
    num_processes = 4
    part_size = 1000000000000000000 // num_processes
    processes = []
    result = 0
    for i in range(num_processes):
        start = i * part_size
        end = (i + 1) * part_size
        p = mp.Process(target=sum_part, args=(start, end))
        processes.append(p)
        p.start()
    for p in processes:
        p.join()
        result += p.result()
    print('Sum:', result)
```
