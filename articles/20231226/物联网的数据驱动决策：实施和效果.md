# 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的Python代码实例，用于预测气候变化的趋势。这个代码实例使用了线性回归算法，并使用了NumPy和Matplotlib库进行数据处理和可视化。

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# 加载数据
data = np.loadtxt('temperature.txt')
x = data[:, 0]  # 年份
y = data[:, 1]  # 气温

# 数据预处理
x_train = np.array([[x[i]] for i in range(len(x))]).reshape(-1, 1)
y_train = np.array(y).reshape(-1, 1)

# 模型构建
model = LinearRegression()
model.fit(x_train, y_train)

# 预测和可视化
x_test = np.array([2020, 2021, 2022, 2023, 2024, 2025])
y_test = model.predict(x_test.reshape(-1, 1))

plt.figure(figsize=(10, 6))
plt.plot(x, y, 'o', label='原始数据')
plt.plot(x_test, y_test, '-', label='预测结果')
plt.xlabel('年份')
plt.ylabel('气温')
plt.title('气温预测')
plt.legend()
plt.show()
```

这个代码实例首先加载了气温数据，然后对数据进行了预处理，接着使用线性回归算法构建了模型，并对模型进行了预测。最后，使用Matplotlib库对预测结果进行了可视化。
