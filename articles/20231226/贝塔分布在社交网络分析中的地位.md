# 2.核心概念与联系

## 2.1 贝塔分布

贝塔分布是一种连续概率分布，定义在0到1的区间内。它的概率密度函数（PDF）为：

$$
f(x; \alpha, \beta) = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha) \Gamma(\beta)} x^{\alpha - 1} (1 - x)^{\beta - 1}
$$

其中，$\alpha$ 和 $\beta$ 是贝塔分布的参数，$\Gamma$ 是伽马函数。

贝塔分布具有许多有趣的性质，例如，它可以用于建模概率，描述不确定性，进行随机采样等。

## 2.2 社交网络分析

社交网络分析涉及到的主要任务包括：

- 节点（或 vertices）：社交网络中的基本组成部分，可以表示为人、组织、设备等。
- 边（或 edges）：连接节点的关系或连接。
- 网络（或 network）：节点和边的集合。

社交网络分析的主要任务包括：

- 节点重要性评估：评估节点在网络中的重要性，例如通过中心性、度中心性、 Betweenness Centrality 等指标。
- 社交网络聚类：发现网络中的子网络或组件，例如通过基于结构的聚类、基于过程的聚类等方法。
- 社交网络可视化：将网络的结构和信息以可视化的方式表示，以帮助人们更好地理解和分析网络。

```python
G.add_edge('A', 'C')
G.add_edge('B', 'C')

# 计算节点的度
degrees = dict(G.degree())

# 计算节点的重要性分数
importances = {}
for node, degree in degrees.items():
    alpha = (degree + 1) / 2
    beta = ((len(G.nodes()) - degree) + 1) / 2
    importance = beta.pdf(degree / len(G.edges()))
    importances[node] = importance

# 绘制节点重要性分数
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True)
nx.draw(G, pos, node_color=[importances[node] for node in G.nodes()], cmap=plt.cm.viridis)
plt.show()
```
