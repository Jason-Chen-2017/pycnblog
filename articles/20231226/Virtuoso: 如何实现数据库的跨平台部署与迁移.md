## 1.关系型数据库的具体代码实例和详细解释说明

关系型数据库的具体代码实例和详细解释说明如下：

1. 创建一个表：

```sql
CREATE TABLE employee (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT,
    salary DECIMAL(10,2)
);
```

这个代码创建了一个名为`employee`的表，其中包含`id`、`name`、`age`和`salary`这四个字段。`id`字段是主键，其他字段都是普通字段。

2. 插入一条记录：

```sql
INSERT INTO employee (id, name, age, salary) VALUES (1, 'John Doe', 30, 5000.00);
```

这个代码插入了一条记录到`employee`表中，其中`id`字段的值为1，`name`字段的值为`John Doe`，`age`字段的值为30，`salary`字段的值为5000.00。

3. 查询记录：

```sql
SELECT * FROM employee WHERE age > 25;
```

这个代码查询了`employee`表中年龄大于25的记录。

4. 更新记录：

```sql
UPDATE employee SET salary = 5500.00 WHERE id = 1;
```

这个代码更新了`employee`表中`id`字段值为1的记录的`salary`字段的值为5500.00。

5. 删除记录：

```sql
DELETE FROM employee WHERE id = 1;
```

这个代码删除了`employee`表中`id`字段值为1的记录。

## 2.图形型数据库的具体代码实例和详细解释说明

图形型数据库的具体代码实例和详细解释说明如下：

1. 创建一个图形：

```graphql
CREATE GRAPH graph1 {
    node1:Node {
        id: 1,
        name: "John Doe",
        age: 30
    }
    node2:Node {
        id: 2,
        name: "Jane Smith",
        age: 28
    }
    edge1:Edge {
        source: node1,
        target: node2,
        relationship: "friend"
    }
}
```

这个代码创建了一个名为`graph1`的图形，其中包含两个节点`node1`和`node2`，以及一个关系`edge1`。

2. 查询图形：

```graphql
SELECT * FROM graph1 WHERE relationship = "friend";
```

这个代码查询了`graph1`图形中关系为"friend"的记录。

3. 更新图形：

```graphql
UPDATE graph1 SET node2.age = 29 WHERE node1.id = 1;
```

这个代码更新了`graph1`图形中`node1`的`id`字段值为1的记录的`node2`的`age`字段的值为29。

4. 删除图形：

```graphql
DELETE FROM graph1 WHERE relationship = "friend";
```

这个代码删除了`graph1`图形中关系为"friend"的记录。
