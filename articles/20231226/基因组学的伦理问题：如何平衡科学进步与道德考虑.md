## 3.1 基因组比对算法

基因组比对算法是一种常用的基因组学算法，用于比较两个基因组之间的相似性。这些算法通常基于Needleman-Wunsch或Smith-Waterman算法，这些算法可以找到两个序列之间的最佳局部对齐。需要注意的是，这些算法的时间复杂度通常很高，因此需要进行优化。

### 3.1.1 Needleman-Wunsch算法

Needleman-Wunsch算法是一种全局对齐算法，它可以找到两个序列之间的最佳全局对齐。算法的主要步骤如下：

1. 初始化一个矩阵，矩阵的行和列分别对应于两个序列中的每个氨基酸或核苷酸。
2. 计算矩阵中每个单元格的得分，得分是基于两个序列中相应氨基酸或核苷酸之间的相似度。
3. 对矩阵进行动态规划，找到最佳的全局对齐。

### 3.1.2 Smith-Waterman算法

Smith-Waterman算法是一种局部对齐算法，它可以找到两个序列之间的最佳局部对齐。算法的主要步骤如下：

1. 初始化一个矩阵，矩阵的行和列分别对应于两个序列中的每个氨基酸或核苷酸。
2. 计算矩阵中每个单元格的得分，得分是基于两个序列中相应氨基酸或核苷酸之间的相似度。
3. 对矩阵进行动态规划，找到最佳的局部对齐。

### 3.1.3 数学模型公式

Needleman-Wunsch和Smith-Waterman算法的数学模型公式如下：

Needleman-Wunsch：
$$
S_{ij} = \max\left\{0, S_{i-1,j-1} + s(a_i, b_j), S_{i-1,j} + \delta_1, S_{i,j-1} + \delta_2\right\}
$$

Smith-Waterman：
$$
S_{ij} = \max\left\{0, S_{i-1,j-1} + s(a_i, b_j), \max_{k=1,...,i-1} S_{k,j-1} + \delta_1, \max_{k=1,...,j-1} S_{i-1,k} + \delta_2\right\}
$$

其中，$S_{ij}$是矩阵中第$i$行第$j$列的得分，$a_i$和$b_j$是两个序列中的相应氨基酸或核苷酸，$s(a_i, b_j)$是氨基酸或核苷酸之间的相似度，$\delta_1$和$\delta_2$是Gap Penalty，用于惩罚Gap的开放和闭合。

## 3.2 基因组变异检测算法

基因组变异检测算法是一种常用的基因组学算法，用于检测基因组中的变异。这些算法通常基于Burrows-Wheeler Transform（BWT）和后续的压缩文本匹配技术，如BM tagging和BMAligner。需要注意的是，这些算法的准确性和敏感性通常受到变异类型和基因组序列质量的影响。

### 3.2.1 Burrows-Wheeler Transform

Burrows-Wheeler Transform（BWT）是一种用于转换基因组序列的技术，它可以将原始序列转换为一个新的序列，这个新序列更容易进行压缩。BWT的主要步骤如下：

1. 创建一个矩阵，矩阵的行分别对应于原始序列中的每个子序列。
2. 对矩阵进行转置。
3. 取矩阵的最后一行。

### 3.2.2 BM tagging

BM tagging是一种基于BWT的压缩文本匹配技术，它可以用于检测基因组中的变异。算法的主要步骤如下：

1. 对BWT后的序列进行压缩文本匹配。
2. 找到匹配的子序列，并检查它们是否与原始序列相同。

### 3.2.3 BMAligner

BMAligner是一种基于BWT的压缩文本匹配技术，它可以用于检测基因组中的变异。算法的主要步骤如下：

1. 对BWT后的序列进行压缩文本匹配。
2. 找到匹配的子序列，并检查它们是否与原始序列相同。

### 3.2.4 数学模型公式

BWT的数学模型公式如下：
$$
BWT(S) = T
$$

其中，$S$是原始基因组序列，$T$是BWT后的序列。

    score = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            match_score = (a[i - 1] == b[j - 1]) * match_score
            score[i][j] = max(0, score[i - 1][j - 1] + match_score,
                              score[i - 1][j] + gap_penalty,
                              score[i][j - 1] + gap_penalty)
    alignments = [['' if score[i][j] == 0 else a[i - 1] for j in range(n)] for i in range(m)]

def bwt(s):
    rev = s[::-1]
    table = ['' for _ in range(len(s))]
    for i in range(len(s)):
        table[i] = rev[i:] + '$'
    table.sort()
    return ''.join(table[1:])

def bm_tagging(s, bwt_table):
    n = len(s)
    tags = ['$'] * (n + 1)
    for i in range(n - 1, -1, -1):
        tags[i] = bwt_table[i + 1][-1]
        while bwt_table[i][-1] != bwt_table[i + 1][-1]:
            i += 1
    return tags

def bma_aligner(s, t, bwt_table, tags):
    n, m = len(s), len(t)
    alignments = [['' for _ in range(m + 1)] for _ in range(n + 1)]
    for i in range(n):
        for j in range(m):
            if s[i] == t[j]:
                alignments[i + 1][j + 1] = alignments[i][j] + s[i]
            elif tags[i + 1] == tags[j + 1]:
                alignments[i + 1][j + 1] = alignments[i][j] + s[i]
            else:
                alignments[i + 1][j + 1] = alignments[i][j + 1]
    return alignments
