细胞死亡是生命延长的一个关键因素，因为细胞死亡可以清除损坏的细胞，从而保护组织和系统的健康。细胞死亡的数学模型可以用卵巢死亡模型来描述。卵巢死亡模型的概率密度函数为：

其中，$t$ 表示时间，$N_0$ 表示初始细胞数量，$\mu$ 表示细胞死亡率。

```python
import numpy as np

def poisson_pmf(k, lambda_):
    e = np.exp(-lambda_)
    fact = np.math.factorial
    return (lambda_**k) / (fact(k) * e)

lambda_ = 10
k = 5
print(poisson_pmf(k, lambda_))
```
