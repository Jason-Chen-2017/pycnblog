# 2.核心概念与联系

## 2.1 玻尔兹曼机（Boltzmann Machine）
玻尔兹曼机是一种生成模型，它可以用于学习和生成高维数据。玻尔兹曼机由一组随机布尔变量组成，这些变量可以分为可观测变量和隐变量。可观测变量表示输入数据，隐变量表示模型内部的状态。玻尔兹曼机的目标是学习一个高度非线性的概率分布，使得生成的数据尽可能接近输入数据。

## 2.2 深度玻尔兹曼机（Deep Boltzmann Machine）
深度玻尔兹曼机是玻尔兹曼机的扩展，它在玻尔兹曼机的基础上增加了多层结构。深度玻尔兹曼机可以学习更复杂的概率分布，从而更好地处理高维数据。深度玻尔兹曼机可以应用于多种任务，包括图像处理、语音识别、自然语言处理等。

## 3.1 深度玻尔兹曼机的结构
深度玻尔兹曼机由多个隐藏层和输入层组成。输入层的节点表示输入数据，隐藏层的节点表示模型内部的状态。每个节点都有一个激活值，表示该节点是否被激活。激活值由一个sigmoid函数计算，如下式所示：

$$
a_i = \frac{1}{1 + e^{-z_i}}
$$

其中，$a_i$ 是激活值，$z_i$ 是节点的总输入。

### 3.2.1 前向传播阶段

1. 初始化输入层的激活值。
2. 计算第一层隐藏层的激活值。
3. 计算第二层隐藏层的激活值。
4. 重复步骤2-3，直到所有隐藏层的激活值都被计算出来。
