## 4.1 Python代码实例

我们将使用Python编程语言来实现Hessian矩阵的计算。以下是一个简单的Python代码实例，它计算了一个二变量函数的Hessian矩阵：

```python
import numpy as np

def f(x, y):
    return x**2 + y**2

def compute_hessian(f):
    x = np.array([0, 1, 2, 3])
    y = np.array([0, 1, 2, 3])
    H = np.zeros((4, 4))
    for i in range(4):
        for j in range(4):
            H[i, j] = f(x[i], y[j]) * (x[i]**2 + y[j]**2)
    return H

H = compute_hessian(f)
print(H)
```
