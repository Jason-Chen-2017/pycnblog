# 2.核心概念与联系

在介绍贝叶斯方法在人脸识别中的优势之前，我们需要了解一些核心概念。

## 2.1 贝叶斯定理
贝叶斯定理是贝叶斯方法的基础，它描述了如何根据现有的信息更新概率分布。给定一个事件A和B，贝叶斯定理表示：

P(A|B) = P(B|A) * P(A) / P(B)

其中，P(A|B)是条件概率，表示在已知B发生的情况下，A发生的概率；P(B|A)是联合概率，表示在已知A发生的情况下，B发生的概率；P(A)和P(B)是边缘概率，表示A和B分别发生的概率。

## 2.2 贝叶斯优势
贝叶斯方法的优势在于它可以根据新的信息动态更新模型，从而更好地处理不确定性。传统的机器学习方法通常需要预先设定模型结构，而贝叶斯方法可以通过设定先验分布来表示模型不确定性，并通过观测数据来更新后验分布。这使得贝叶斯方法更适合处理复杂和不确定的问题。
