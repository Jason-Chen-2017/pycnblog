# 2.核心概念与联系

在量子调控研究中，我们主要关注以下几个核心概念：

1. **量子系统**：量子系统是一种由量子粒子组成的系统，这些粒子遵循量子 mechanics 的规则。量子系统可以是原子、分子、子原子等。

2. **量子态**：量子态是量子系统的一个描述，它可以表示为一个向量。量子态可以是纯量子态或混合量子态。

3. **量子操作**：量子操作是对量子态进行的变换，它可以是单位性操作或非单位性操作。量子操作可以通过量子门实现。

4. **量子门**：量子门是一种对量子态进行特定变换的操作，例如 Hadamard 门、Pauli-X 门、Pauli-Z 门等。

5. **量子调控**：量子调控是一种对量子系统进行控制的方法，它可以通过对量子门进行调整来实现不同的量子操作。

这些概念之间的联系如下：

- 量子系统通过量子门进行操作，这些操作可以实现不同的量子态变换。
- 量子调控可以通过调整量子门来实现对量子系统的精确控制。
- 量子调控研究的目标是找到如何使量子系统实现特定的行为，以及如何在实际应用中实现这些行为。
