# 2.核心概念与联系
凸集分离定理是一种用于解决优化问题的方法，它主要关注于将一个多边形划分为多个子多边形，使得每个子多边形都在凸集中。这种方法可以用于解决各种优化问题，如线性规划、非线性规划、机器学习等。

凸集分离定理的核心概念包括：

1. 凸集（Convex Set）：一个集合S是凸的，如果对于任何两个点a,b在S中，它们的中间点m（a,b）也在S中。
2. 多边形（Polygon）：一个有限个点的凸集。
3. 凸集分离定理（Convex Hull Separation Theorem）：给定一个多边形P和一个凸集C，如果P不完全包含在C中，则存在一个点x在P中，使得x不在C中。

具体操作步骤：

1. 输入一个多边形P和一个凸集C。
2. 找到多边形P中与凸集C最接近的点。
3. 将这个点作为中心，以凸集C为半径，以点为圆心，构造一个圆。
4. 将圆与多边形P的边界交叉点作为圆心，以凸集C为半径，构造一个圆。
5. 将两个圆的交点作为子多边形的顶点，将多边形P划分为多个子多边形。
6. 判断每个子多边形是否在凸集C中，如果不在，则找到该子多边形中与凸集C最接近的点，并将该点作为中心，以凸集C为半径，构造一个圆，将圆与子多边形的边界交叉点作为圆心，以凸集C为半径，构造一个圆，将两个圆的交点作为子多边形的顶点，将多边形P划分为多个子多边形。
7. 重复上述步骤，直到所有子多边形都在凸集C中。

1. 凸集C的定义：C = {x | Ax <= b}
