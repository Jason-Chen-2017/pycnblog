## 2.1 共轭向量的定义与性质

共轭向量的定义如下：

给定一个n维复数向量a = a1, a2, ..., an，其共轭向量的定义为a* = a1, a2, ..., an，其中ai* = ai，i为虚数单位。

共轭向量具有以下性质：

1. 内积：给定两个共轭向量a*和b*，它们的内积定义为a*·b* = a1*b1 + a2*b2 + ... + an*bn，其中a1*b1 + a2*b2 + ... + an*bn是实数。

2. 外积：给定两个共轭向量a*和b*，它们的外积定义为a* x b* = a1*b2 - a2*b1 + a3*b4 - a4*b3 + ... + an*bn*，其中a1*b2 - a2*b1 + a3*b4 - a4*b3 + ... + an*bn*是实数。

3. 正交：给定两个共轭向量a*和b*，如果a*·b* = 0，则a*和b*是正交的。

4. 单位：给定一个共轭向量a*，如果a*·a* = 1，则a*是单位向量。
