Impala是一种高性能的SQL查询引擎，可以在Hadoop生态系统中进行快速查询。Impala是Cloudera开发的，可以与Hadoop Distributed File System (HDFS)、HBase、Parquet等存储系统集成。Impala支持SQL查询，可以在大数据集上进行实时查询和分析。
