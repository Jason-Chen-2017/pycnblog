# 4.具体代码实例和详细解释说明

## 4.1问题描述
编写一个函数，实现两个有序数组的合并。合并后的数组应该保持有序。

## 4.2代码实例
```python
def merge_sorted_arrays(arr1, arr2):
    i = 0
    j = 0
    result = []
    while i < len(arr1) and j < len(arr2):
        if arr1[i] < arr2[j]:
            result.append(arr1[i])
            i += 1
        else:
            result.append(arr2[j])
            j += 1
    result.extend(arr1[i:])
    result.extend(arr2[j:])
    return result
```

## 4.3解释说明
上述代码实现了两个有序数组的合并。首先，我们使用两个指针 i 和 j 分别指向两个数组的开头。然后，我们逐一比较两个数组的元素，将较小的元素加入结果数组。当一个数组遍历完毕后，我们将剩余的元素加入结果数组。最终，我们返回合并后的结果数组。
