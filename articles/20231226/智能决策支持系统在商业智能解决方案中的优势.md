                 

# 1.背景介绍

商业智能（Business Intelligence，BI）是一种利用数据、工具、技术和最佳实践来提高企业竞争力的方法。商业智能的目的是将数据转化为有价值的信息，以支持企业的决策和竞争优势。智能决策支持系统（Intelligent Decision Support System，IDSS）是一种利用人工智能技术来自动化决策过程的系统。在商业智能解决方案中，智能决策支持系统的优势如下：

1.1 提高决策效率
1.2 降低决策风险
1.3 提高决策质量
1.4 提供实时决策支持
1.5 支持跨部门协作
