光子学，也被称为量子光学，是一门研究光的量子特性的科学。它研究光的行为和光子（光的基本单位）的性质，包括光的传播、散射、折射、反射、吸收和发射等。光子学的研究内容广泛，涉及到光子的产生、传播、相互作用、光子的辐射和吸收等。

在过去的几十年里，光子学已经取得了巨大的进展，它已经成为了现代物理学的一个重要分支。光子学的研究对于许多科学领域的发展起到了重要的作用，如物理学、化学、生物学、天文学等。

```python
import numpy as np
import matplotlib.pyplot as plt

# 电磁波的振动
def electric_field(t, amplitude, frequency, phase):
    return amplitude * np.cos(2 * np.pi * frequency * t + phase)

# 磁场的振动
def magnetic_field(t, amplitude, frequency, phase):
    return amplitude * np.cos(2 * np.pi * frequency * t + phase)

# 时间数组
t = np.linspace(0, 1, 1000)

# 电场和磁场的振动
E = electric_field(t, 1, 5, 0)
B = magnetic_field(t, 1, 5, 0)

# 绘制电场和磁场的振动
plt.plot(t, E, label='Electric Field')
plt.plot(t, B, label='Magnetic Field')
plt.legend()
plt.show()
```
