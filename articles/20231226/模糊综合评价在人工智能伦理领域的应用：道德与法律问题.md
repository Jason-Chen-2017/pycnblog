# 4.具体代码实例和详细解释说明

代码实例：

```python
import numpy as np
from skfuzzy import control as ctrl

# 构建模糊集
x = ctrl.Antecedent(np.arange(0, 101, 1), 'x')
y = ctrl.Consequent(np.arange(0, 101, 1), 'y')

# 构建模糊关系
r1 = ctrl.Rule(ctrl.IF_Expr(x['x'] <= 50, 'x <= 50'), ctrl.THEN_Expr(y['y'] == 0))
r2 = ctrl.Rule(ctrl.IF_Expr(50 < x['x'] <= 100, '50 < x <= 100'), ctrl.THEN_Expr(y['y'] == 1))

# 构建模糊规则
k = ctrl.ControlSystem([r1, r2])

# 构建模糊度量
b = ctrl.ControlSystemAntecedent(k, 'b')

# 评估
result = k.aggregate(x, b)

print(result)
```

在这个代码实例中，我们首先构建了一个模糊集，包括输入变量 $x$ 和输出变量 $y$。然后我们构建了两个模糊关系，分别对应于输入变量 $x$ 的不同范围。接着我们构建了一个模糊规则，用于描述输入变量 $x$ 与输出变量 $y$ 之间的关系。最后我们使用模糊综合评价算法来评估输入变量 $x$ 的不确定性。
