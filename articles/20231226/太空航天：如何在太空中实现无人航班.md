```
import numpy as np

def calculate_position(velocity, time):
    position = velocity * time
    return position

def calculate_direction(angle):
    direction = np.array([np.cos(angle), np.sin(angle)])
    return direction

def calculate_trajectory(position, direction, time):
    trajectory = position + direction * time
    return trajectory

position = np.array([0, 0])
velocity = np.array([1, 0])
angle = np.radians(45)
time = 10

position = calculate_position(velocity, time)
direction = calculate_direction(angle)
trajectory = calculate_trajectory(position, direction, time)

print("Position: ", position)
print("Direction: ", direction)
print("Trajectory: ", trajectory)
```

在这个代码实例中，我们首先导入了numpy库，然后定义了三个函数：calculate_position、calculate_direction和calculate_trajectory。这三个函数分别计算位置、方向和轨迹。接着，我们定义了位置、速度、角度和时间的变量，并调用了这三个函数来计算位置、方向和轨迹。最后，我们打印了这三个变量的值。
