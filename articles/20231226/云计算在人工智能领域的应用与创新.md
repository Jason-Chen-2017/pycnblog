### 4.4 模型部署

#### 4.4.1 模型部署

使用 TensorFlow Serving 部署卷积神经网络（CNN）模型的代码实例：

```python
import grpc
from tensorflow_serving.apis import predict_pb2
from tensorflow_serving.apis import prediction_service_pb2
from tensorflow_serving.client import PredictRequest
from tensorflow_serving.client import PredictResponse

# 创建 PredictRequest
request = PredictRequest()
request.model_spec.name = 'my_model'
request.model_spec.signature_name = 'predict'
request.inputs['input'].CopyFrom(predict_pb2.Input(dtype=predict_pb2.DT_FLOAT, shape=[1, 28, 28, 1]))

# 连接 TensorFlow Serving
with grpc.insecure_channel('localhost:8500') as channel:
    stub = prediction_service_pb2.beta_create_PredictionService_stub(channel)
    response = stub.Predict.request(request)

# 解析 PredictResponse
output = response.outputs['output'].float_val[0]
```

详细解释说明：

- 首先，我们导入 TensorFlow Serving 的 gRPC 客户端。
- 然后，我们创建一个 PredictRequest，设置模型名称和签名名称，并设置输入数据。
- 接着，我们连接到 TensorFlow Serving。
- 最后，我们调用 Predict 方法发送请求，并解析 PredictResponse 中的输出数据。
