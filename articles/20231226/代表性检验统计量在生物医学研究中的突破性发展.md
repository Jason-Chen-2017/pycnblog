$$

$$

```python
import numpy as np
import scipy.stats as stats

# 生成两个独立样本
np.random.seed(0)
sample1 = np.random.normal(loc=100, scale=15, size=100)
sample2 = np.random.normal(loc=105, scale=15, size=100)

# 计算两个样本的均值和方差
mean1 = np.mean(sample1)
mean2 = np.mean(sample2)
var1 = np.var(sample1)
var2 = np.var(sample2)

# 计算t统计量
t_statistic = (mean1 - mean2) / np.sqrt((var1 / 100) + (var2 / 100))

# 计算p值
p_value = 2 * (1 - stats.t.cdf(np.abs(t_statistic), df=198))

print("t统计量:", t_statistic)
print("p值:", p_value)
```

详细解释说明：

1. 首先，我们使用numpy生成两个独立样本，这两个样本的均值分别为100和105，方差分别为15^2。
2. 然后，我们计算两个样本的均值和方差。
3. 接下来，我们计算t统计量，使用样本均值和方差计算。
4. 最后，我们计算p值，使用scipy.stats.t.cdf计算。
