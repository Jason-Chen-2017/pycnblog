```python
def delete(self, data):
    if self.head is None:
        return
    if self.head.data == data:
        self.head = self.head.next
        self.size -= 1
        return
    current = self.head
    while current.next is not None:
        if current.next.data == data:
            current.next = current.next.next
            self.size -= 1
            return
        current = current.next
```

## 6.1 数据流与队列的区别

数据流和队列都是用于存储数据的数据结构，但它们之间有一些重要的区别。数据流是一种无限序列，它可以被视为一种可处理的资源，而队列是一种有限的数据结构，它用于存储一组数据项。数据流的插入、删除、查找等操作是在数据被处理的同时进行的，而队列的插入、删除、查找等操作是在数据被处理之后进行的。

## 6.2 数据流与链表的区别

数据流和链表都是用于存储数据的数据结构，但它们之间也有一些重要的区别。链表是一种线性数据结构，它用于存储一组数据项，每个数据项都有一个指向下一个数据项的指针。数据流是一种无限序列，它可以被视为一种可处理的资源，而不是一组数据项。数据流的插入、删除、查找等操作是在数据被处理的同时进行的，而链表的插入、删除、查找等操作是在数据被处理之后进行的。
