## 4.2 数据挖掘在弹性网络安全中的代码实例

### 4.2.1 网络安全事件挖掘的代码实例

```python
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

# 加载数据
data = pd.read_csv('network_security_events.csv')

# 数据预处理
X = data['event_description']

# 特征提取
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(X)

# 聚类分析
k = 3
model = KMeans(n_clusters=k, init='k-means++', max_iter=100, n_init=1)
model.fit(X)
labels = model.predict(X)

# 聚类评估
score = silhouette_score(X, labels)
print('Silhouette Score:', score)
```

### 4.2.2 网络安全风险挖掘的代码实例

```python
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.decomposition import LatentDirichletAllocation
from sklearn.metrics import adjusted_rand_score

# 加载数据
data = pd.read_csv('network_security_risks.csv')

# 数据预处理
X = data['risk_description']

# 特征提取
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(X)

# 主题模型
n_components = 5
model = LatentDirichletAllocation(n_components=n_components, max_iter=100, learning_method='online', learning_offset=50)
model.fit(X)

# 主题模型评估
terms = model.components_
terms = [vectorizer.get_feature_names_out()[index] for index in terms]
score = adjusted_rand_score(terms, labels)
print('Adjusted Rand Score:', score)
```
