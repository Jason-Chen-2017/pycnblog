```python
import time
import random
from datetime import datetime

class SocialMediaMarketing:
    def __init__(self, platform, content_list):
        self.platform = platform
        self.content_list = content_list
        self.publish_time_list = []

    def publish_content(self):
        for content in self.content_list:
            publish_time = datetime.now()
            self.publish_time_list.append(publish_time)
            print(f"{self.platform}发布内容：{content}，发布时间：{publish_time}")

if __name__ == "__main__":
    platform = "Twitter"
    content_list = ["今天是个美好的日子", "学习Python编程语言", "Python编程语言的应用"]
    marketing = SocialMediaMarketing(platform, content_list)
    marketing.publish_content()
```

在这个代码实例中，我们首先定义了一个`SocialMediaMarketing`类，该类包括一个构造函数和一个`publish_content`方法。构造函数中，我们初始化了平台和内容列表，并定义了一个用于存储发布时间的列表。`publish_content`方法中，我们遍历内容列表，并将当前时间作为发布时间添加到列表中，然后将内容和发布时间打印出来。

在主程序中，我们创建了一个`SocialMediaMarketing`对象，并调用`publish_content`方法来发布内容。
