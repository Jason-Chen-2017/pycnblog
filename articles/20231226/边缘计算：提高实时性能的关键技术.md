                 

# 1.背景介绍

边缘计算是一种新兴的计算模型，它将数据处理和分析推向了边缘设备，例如智能手机、IoT设备、传感器等。这种模型的出现是为了解决云计算的一些问题，例如高延迟、高带宽、数据安全等。边缘计算可以帮助我们更快地处理数据，提高实时性能，并且可以减少数据传输成本。

## 1.1 背景介绍

### 1.1.1 云计算的局限性

云计算是一种基于互联网的计算模型，它将数据存储和处理推向云端服务器，从而实现了资源共享和计算能力的集中化。虽然云计算有很多优点，如便宜、可扩展、易于使用等，但是它也有一些局限性：

1. 高延迟：由于数据需要通过网络传输到云端服务器，因此会产生较高的延迟。这对于实时性应用来说是一个问题。
2. 高带宽：数据传输需要消耗较高的带宽，这会增加成本。
3. 数据安全：数据在传输过程中可能会泄露或被篡改，这会影响数据安全。

### 1.1.2 边缘计算的诞生

为了解决云计算的上述问题，边缘计算诞生了。边缘计算将数据处理和分析推向了边缘设备，从而实现了更快的响应速度、更低的延迟、更高的带宽和更好的数据安全。边缘计算可以应用于各种领域，例如智能城市、智能交通、智能医疗等。
