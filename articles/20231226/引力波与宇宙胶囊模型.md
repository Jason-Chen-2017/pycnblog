# 4.1引力波检测
在这个例子中，我们将使用 Python 和 NumPy 库来实现一个简单的引力波检测程序。这个程序将观测一个星系的运动，并检测是否存在引力波。

```python
import numpy as np

# 观测到的星系的运动
star_motion = np.array([0.1, 0.2, 0.3])

# 预期的星系运动
expected_motion = np.array([0, 0, 0])

# 计算引力波的大小
gravity_wave_size = np.linalg.norm(star_motion - expected_motion)

# 判断是否存在引力波
if gravity_wave_size > 0.1:
    print("存在引力波")
else:
    print("不存在引力波")
```
