# 2.核心概念与联系

无人机气候变化研究的核心概念包括：

- 无人机：无人机是一种由无人驾驶系统控制的飞行器，通常用于拍摄影像、传感器数据采集、物流运输等应用。在气候变化研究中，无人机主要用于采集大气参数数据，如温度、湿度、风速、大气压力等。
- 气候变化：气候变化是地球大气层和海洋表面温度的长期变化，主要由人类活动引起，如碳排放、土壤污染等。气候变化可能导致海平面上升、极地冰川融化、极地温度升高等严重影响，对人类生活和生态系统产生严重威胁。
- 无人机气候变化研究：无人机气候变化研究是一种利用无人机技术为气候变化研究提供数据支持的方法，包括无人机飞行控制、数据传输、数据处理和分析等。

无人机气候变化研究与气候变化研究、无人机技术、气象观测等领域密切相关。无人机气候变化研究可以为气候变化研究提供实时、高精度的气候数据，有助于科学家更好地理解气候变化的规律和趋势，从而制定更有效的应对措施。
