泊松分布是一种概率分布，用于描述一定时间内发生的独立事件数量的分布。泊松分布的概率密度函数为：

$$

import matplotlib.pyplot as plt

event_type = 'stock_price_change'

# 计算事件发生的平均次数
stock_price_changes = np.random.normal(0, 1, time_interval)

# 使用泊松分布概率密度函数计算事件发生的概率
lambda_value = average_change * time_interval
x = np.arange(0, time_interval * 2 + 1)
poisson_probability = np.exp(-lambda_value) * (lambda_value ** x) / np.math.factorial(x)

# 绘制泊松分布概率密度函数
plt.plot(x, poisson_probability)
plt.xlabel(event_type + ' count')
plt.ylabel('Probability')
plt.title('Poisson Distribution for ' + event_type)
plt.show()
