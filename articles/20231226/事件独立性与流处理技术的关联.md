### 4.1 事件独立性的Python代码实例
在Python中，我们可以通过以下代码来实现事件独立性的判断：

```python
import numpy as np

def is_independent(A, B):
    P_A = np.mean(A)
    P_B = np.mean(B)
    P_A_B = np.mean(A * B)
    if P_A_B == P_A * P_B:
        return True
    else:
        return False
```

在上面的代码中，我们首先导入了numpy库，然后定义了一个名为is_independent的函数。这个函数接受两个参数A和B，分别表示事件A和事件B。接着，我们计算了事件A和事件B的概率P_A和P_B，以及事件A和事件B同时发生的概率P_A_B。最后，我们通过判断P_A_B是否等于P_A * P_B来判断事件A和事件B是否独立。
