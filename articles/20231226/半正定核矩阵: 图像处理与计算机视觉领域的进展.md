## 1.1 背景介绍

半正定核矩阵是一种特殊的实数矩阵，其元素为正数且对角线上的元素为0。这种矩阵在图像处理和计算机视觉领域中具有广泛的应用，如图像滤波、图像重建、图像识别等。

半正定核矩阵的元素可以通过以下公式计算：

$$
H_{ij} = \frac{1}{(i + j + 1)^2} \quad \text{for} \quad i, j \geq 0
$$

其中，$H_{ij}$ 表示矩阵的第 $i+1$ 行第 $j+1$ 列的元素。

## 1.4 具体代码实例和详细解释说明

### 4.1 代码实例

```python
import numpy as np

def hilbert_matrix(n):
    H = np.zeros((n, n))
    for i in range(n):
        for j in range(n):
            H[i, j] = 1 / ((i + j + 1) ** 2)
    return H

n = 5
H = hilbert_matrix(n)
print(H)
```

在 `for` 循环中，我们使用公式计算每个元素的值：
