## 3.3 数学模型公式详细讲解

智能制造的数学模型公式如下：

1. 数据收集和处理：

$$
y = \beta_0 + \beta_1 x_1 + \cdots + \beta_n x_n + \epsilon
$$

其中，$y$ 是 dependent variable，$x_1, \cdots, x_n$ 是 independent variables，$\beta_0, \cdots, \beta_n$ 是 regression coefficients，$\epsilon$ 是 error term。

2. 数据分析和预测：

$$
\hat{y} = \sum_{i=1}^n w_i y_i
$$

其中，$\hat{y}$ 是 predicted value，$w_i$ 是 weight，$y_i$ 是 observed value。

3. 智能决策和自动化：

$$
\min_{w} \sum_{i=1}^n (y_i - \hat{y}_i)^2
$$

其中，$w$ 是 weight vector，$y_i$ 是 observed value，$\hat{y}_i$ 是 predicted value。
