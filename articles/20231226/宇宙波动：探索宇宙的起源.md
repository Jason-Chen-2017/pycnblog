                 

# 1.背景介绍

宇宙波动（Cosmic Microwave Background，简称CMB）是指宇宙最早时期的一种热辐射，它是现代宇宙学的一个关键证据。根据大气热辐射定理，宇宙波动是由于宇宙初期的热辐射在宇宙大气中的散射所产生的。这种辐射在大气分离的时期（大约400000年后）成为微波辐射，这就是我们现在所观测到的宇宙波动。

宇宙波动的发现是现代宇宙学的一个重要里程碑，它证实了大气热辐射定理，并为我们提供了关于宇宙起源和宇宙结构的重要信息。在1964年，美国科学家阿罗兹·卢兹勒（Arno Penzias）和罗伯特·沃尔特（Robert W. Wilson）在新泽西州的希腊神庙（Greek Orthodox Cathedral）附近的天文望远镜观测到了这种微波辐射，并将其作为宇宙波动的发现。

宇宙波动的观测结果表明，宇宙起源时的温度约为2.7K（-270.45度），这是现代宇宙学的一个关键证据。此外，宇宙波动的观测结果还表明了宇宙的平均密度和结构，这为我们研究宇宙起源和宇宙演化提供了关键信息。
