## 4.具体代码实例和详细解释说明

### 4.1 DBMS的代码实例

在这里，我们以Python的SQLite库为例，展示如何实现简单的数据库管理系统。

```python
import sqlite3

# 创建数据库
conn = sqlite3.connect('test.db')

# 创建表
cursor = conn.cursor()
cursor.execute('''CREATE TABLE IF NOT EXISTS students
                  (id INTEGER PRIMARY KEY, name TEXT, age INTEGER, score REAL)''')

# 插入数据
cursor.execute('''INSERT INTO students (name, age, score)
                  VALUES ('Alice', 20, 85.5)''')

# 查询数据
cursor.execute('''SELECT * FROM students''')
print(cursor.fetchall())

# 更新数据
cursor.execute('''UPDATE students SET score = 90 WHERE id = 1''')

# 删除数据
cursor.execute('''DELETE FROM students WHERE id = 1''')

# 提交事务
conn.commit()

# 关闭数据库
conn.close()
```

### 4.2 数据挖掘的代码实例

在这里，我们以Python的scikit-learn库为例，展示如何实现简单的数据挖掘。

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import accuracy_score

# 加载数据
iris = load_iris()
X, y = iris.data, iris.target

# 数据预处理
scaler = StandardScaler()
X = scaler.fit_transform(X)

# 特征选择
X = X[:, :2]

# 训练模型
clf = DecisionTreeClassifier()
clf.fit(X, y)

# 模型评估
y_pred = clf.predict(X)
print('Accuracy:', accuracy_score(y, y_pred))
```
