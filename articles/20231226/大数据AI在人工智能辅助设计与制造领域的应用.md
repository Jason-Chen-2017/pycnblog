## 2.1 大数据

大数据是指由于互联网、物联网、移动互联网等新兴技术的发展，产生的数据量巨大、多样性丰富、实时性强的数据。大数据具有以下特点：

### 4.1.2 数据清洗

```python
# 去除缺失值
design_data = design_data.dropna()
manufacturing_data = manufacturing_data.dropna()
quality_data = quality_data.dropna()

# 纠正异常值
Q1 = design_data.quantile(0.25)
Q3 = design_data.quantile(0.75)
IQR = Q3 - Q1
design_data = design_data[~((design_data < (Q1 - 1.5 * IQR)) | (design_data > (Q3 + 1.5 * IQR))).any(axis=1)]
```
