# 4.具体代码实例和详细解释说明
## 4.1 智能合约代码实例
以下是一个简单的智能合约代码实例，使用Solidity编写：

```
pragma solidity ^0.5.0;

contract SimpleContract {
    uint public number;

    function setNumber(uint _number) public {
        number = _number;
    }

    function getNumber() public view returns (uint) {
        return number;
    }
}
```

上述智能合约定义了一个名为`SimpleContract`的合约，包含一个公共的uint类型的变量`number`，以及两个函数`setNumber`和`getNumber`。`setNumber`函数用于设置`number`变量的值，`getNumber`函数用于获取`number`变量的值。

## 4.2 大数据处理代码实例
