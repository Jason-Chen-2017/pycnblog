## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的优化问题来展示如何使用蜻蜓优化算法。假设我们要求最小化以下目标函数：

$$
f(x) = (x - 3)^2 + (y - 1)^2
$$

其中，$x$ 和 $y$ 是问题的变量。我们可以使用以下 Python 代码来实现蜻蜓优化算法：

```python
import numpy as np
import matplotlib.pyplot as plt

def objective_function(x):
    return (x - 3) ** 2 + (y - 1) ** 2

def firefly_algorithm(n_fireflies, n_iterations, n_dimensions):
    # 初始化蜻蜓群
    fireflies = np.random.rand(n_fireflies, n_dimensions)
    fitness = np.array([objective_function(firefly) for firefly in fireflies])

    # 评估蜻蜓的 FITNESS
    for t in range(n_iterations):
        for i in range(n_fireflies):
            for j in range(n_fireflies):
                # 更新蜻蜓的位置和发光强度
                if fitness[j] < fitness[i]:
                    beta0 = 1
                    gamma = 1
                    r = np.linalg.norm(fireflies[i] - fireflies[j])
                    theta = np.arccos(np.dot(fireflies[i], fireflies[j]) / (np.linalg.norm(fireflies[i]) * np.linalg.norm(fireflies[j])))
                    alpha = np.random.rand()
                    fireflies[i] = fireflies[i] + beta0 * np.exp(-gamma * r**2) * np.cos(theta) * (fireflies[j] - fireflies[i]) + alpha * np.random.randn(n_dimensions)

                # 更新蜻蜓的适应度
                fitness[i] = objective_function(fireflies[i])

    # 返回最佳解和最佳适应度
    best_firefly = fireflies[np.argmin(fitness)]
    best_fitness = np.min(fitness)
    return best_firefly, best_fitness

n_fireflies = 50
n_iterations = 100
n_dimensions = 2
best_firefly, best_fitness = firefly_algorithm(n_fireflies, n_iterations, n_dimensions)
print("最佳解: ", best_firefly)
print("最佳适应度: ", best_fitness)
```

在上述代码中，我们首先定义了目标函数 `objective_function`，然后实现了蜻蜓优化算法的主要逻辑。在算法的主体部分，我们首先初始化了蜻蜓群，然后进行了多次迭代，以更新蜻蜓的位置和发光强度。最后，我们返回了最佳解和最佳适应度。
