## 3.3 数学模型公式详细讲解

1. 线性回归：

   - 模型公式：$$ y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon $$
   - 最小二乘法：$$ \min_{\beta_0,\beta_1,\cdots,\beta_n}\sum_{i=1}^n(y_i - (\beta_0 + \beta_1x_{i1} + \beta_2x_{i2} + \cdots + \beta_nx_{in}))^2 $$

2. 逻辑回归：

   - 模型公式：$$ P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - \cdots - \beta_nx_n}} $$
   - 最大似然估计：$$ \max_{\beta_0,\beta_1,\cdots,\beta_n}\prod_{i=1}^nP(y_i=1|x_i)^{y_i}P(y_i=0|x_i)^{1-y_i} $$

3. 支持向量机：

   - 模型公式：$$ y = \text{sgn}(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon) $$
   - 最大间隔：$$ \max_{\beta_0,\beta_1,\cdots,\beta_n}\min_{x\in\mathcal{X}}\min_{y\in\mathcal{Y}}||\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon - y|| $$

4. 决策树：

   - 模型公式：$$ y = \begin{cases} g_1(x) & \text{if } x \in R_1 \\ g_2(x) & \text{if } x \in R_2 \\ \vdots & \vdots \\ g_m(x) & \text{if } x \in R_m \end{cases} $$
   - 信息增益：$$ G(S,A) = IG(S) - \sum_{v\in\mathcal{V}(A)} \frac{|S_v|}{|S|}IG(S_v) $$

5. 随机森林：

   - 模型公式：$$ y = \frac{1}{K}\sum_{k=1}^Kg_k(x) $$
   - 减少过拟合：$$ \min_{\beta_0,\beta_1,\cdots,\beta_n}\sum_{k=1}^K\sum_{i=1}^n(y_{ik} - (\beta_0 + \beta_1x_{ik1} + \beta_2x_{ik2} + \cdots + \beta_nx_{ikn}))^2 $$

6. 神经网络：

   - 模型公式：$$ y = f_{\theta}(x) = \sigma(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon) $$
   - 梯度下降：$$ \min_{\theta}\sum_{i=1}^nL(y_i,f_{\theta}(x_i)) $$

# 4.具体代码实例和详细解释说明

## 4.1 线性回归

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 1)
y = 3 * x.squeeze() + 2 + np.random.randn(100, 1)

# 训练模型
model = LinearRegression()
model.fit(x, y)

# 预测
x_test = np.array([[0.5], [0.8], [1.0]])
y_pred = model.predict(x_test)

# 绘图
plt.scatter(x, y, color='red')
plt.plot(x, model.coef_[0] * x.squeeze() + model.intercept_, color='blue')
plt.show()
```

## 4.3 支持向量机

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.svm import SVC

# 生成数据
np.random.seed(0)
x = np.random.rand(100, 2)
y = np.where(x[:, 0] > 0.5, 1, -1) + np.random.randint(-1, 2, 100)

# 训练模型
model = SVC(kernel='linear')
model.fit(x, y)

# 预测
x_test = np.array([[0.4, 0.6], [0.7, 0.4], [0.9, 0.2]])
y_pred = model.predict(x_test)

# 绘图
plt.scatter(x[:, 0], x[:, 1], c=y, cmap='viridis')
plt.plot(model.support_vectors_[:, 0], model.support_vectors_[:, 1], color='red')
plt.show()
```
