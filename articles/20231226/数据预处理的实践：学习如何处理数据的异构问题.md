# 2.核心概念与联系
异构数据问题主要体现在以下几个方面：

- 数据类型不同：例如，文本数据、图像数据、音频数据等。
- 数据结构不同：例如，结构化数据（如表格数据）、非结构化数据（如文本数据）、半结构化数据（如HTML数据）等。
- 数据来源不同：例如，数据库、文件、Web、Sensor等。

为了处理异构数据问题，我们需要掌握以下几个核心概念：

- 数据清洗：包括数据缺失值处理、数据噪声去除、数据重复值处理等。
- 数据转换：包括数据类型转换、数据格式转换、数据单位转换等。
- 数据归一化：包括数据值范围归一化、数据分布归一化、数据维度归一化等。
- 数据融合：包括数据特征融合、数据模型融合、数据来源融合等。
