### 4.1 数据清洗代码实例

以下是一个简单的数据清洗代码实例，用于处理环境监测数据中的缺失值：

```python
import numpy as np
import pandas as pd

# 读取环境监测数据
data = pd.read_csv('env_monitoring_data.csv')

# 处理缺失值
data['temperature'].fillna(value=np.mean(data['temperature']), inplace=True)
data['humidity'].fillna(value=np.mean(data['humidity']), inplace=True)
data['pressure'].fillna(value=np.mean(data['pressure']), inplace=True)

# 保存清洗后的环境监测数据
data.to_csv('env_monitoring_data_cleaned.csv', index=False)
```

详细解释：

1. 读取环境监测数据，使用 `pandas` 库读取 CSV 格式的环境监测数据。
2. 处理缺失值，使用 `fillna` 方法填充缺失值，将缺失值替换为对应列的均值。
3. 保存清洗后的环境监测数据，使用 `to_csv` 方法将清洗后的环境监测数据保存为 CSV 格式的文件。
