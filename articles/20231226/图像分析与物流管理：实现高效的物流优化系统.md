## 4.1 代码实例：货物检验

在这个代码实例中，我们将使用Python编程语言和OpenCV库来实现货物检验的功能。具体代码实例如下：

```python
import cv2
import numpy as np

# 读取图像文件

# 对图像进行预处理
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
blur = cv2.GaussianBlur(gray, (5, 5), 0)

# 对图像进行边缘检测
edges = cv2.Canny(blur, 50, 150)

# 对边缘进行填充
kernel = np.ones((3, 3), np.uint8)
filled = cv2.dilate(edges, kernel, iterations=1)

# 对填充后的边缘进行轮廓检测
contours, hierarchy = cv2.findContours(filled, cv2.RETR_TREE, cv2.CHAIN_APPROX_SIMPLE)

# 绘制轮廓
cv2.drawContours(img, contours, -1, (0, 255, 0), 2)

# 显示图像
cv2.imshow('Goods', img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

在这个代码实例中，我们首先读取了货物检验的图像文件，然后对图像进行了预处理，例如将图像转换为灰度图，对图像进行高斯模糊处理。接着，我们对图像进行了边缘检测，然后对边缘进行了填充。最后，我们对填充后的边缘进行了轮廓检测，并将轮廓绘制在原图像上。最后，我们显示了处理后的图像。
