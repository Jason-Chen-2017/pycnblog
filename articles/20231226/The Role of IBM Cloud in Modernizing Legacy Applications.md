                 

# 1.背景介绍

背景介绍

随着业务需求的不断变化和技术的不断发展，企业需要不断更新和优化其信息技术基础设施以满足业务需求。这种更新和优化过程被称为“现代化”或“现代化”。现代化涉及到许多方面，包括数据中心的更新、应用程序的重构以及整个技术基础设施的优化。

在过去的几十年里，许多企业使用了“传统”或“遗留”应用程序来支持其核心业务流程。这些应用程序通常是基于旧版技术栈的，如COBOL、PL/I、JCL等，运行在传统的大型数据中心上。这些应用程序已经经过了数十年的使用，但它们的维护和扩展已经变得越来越困难，因为它们的技术基础已经过时，人力成本也很高。

因此，企业需要对这些遗留应用程序进行现代化，以便在竞争激烈的市场环境中保持竞争力。现代化遗留应用程序的主要目标是提高其性能、可扩展性和可维护性，同时降低运行成本。这可以通过多种方式实现，例如重构应用程序代码、将其迁移到云计算环境中或将其与新技术结合。

在这篇文章中，我们将讨论IBM云在现代化遗留应用程序方面的角色。我们将讨论如何使用IBM云服务来现代化遗留应用程序，以及这种现代化方法的优缺点。我们还将讨论如何在IBM云环境中部署和管理现代化后的应用程序，以及如何确保这些应用程序的安全性和可靠性。
