# 4.具体代码实例和详细解释说明
# 4.1 Concepts 的代码实例
以下是一个使用 Concepts 的代码实例：

```cpp
#include <iostream>

template <typename T>
concept int_type = std::integral<T> && std::signed<T>;

int main() {
    int a = 10;
    if (a is int_type) {
        std::cout << "a is an integer" << std::endl;
    }
    return 0;
}
```

# 4.2 Coroutines 的代码实例
以下是一个使用 Coroutines 的代码实例：

```cpp
#include <iostream>

int main() {
    int a = 0;
    auto gen = std::coroutine<int()>::from_promise([&](std::promise<int>& p) {
        p.set_value(a++);
    });
    std::cout << gen() << std::endl;
    std::cout << gen() << std::endl;
    return 0;
}
```

# 4.3 Module 的代码实例

```cpp

```

# 4.4 Ranges 的代码实例
