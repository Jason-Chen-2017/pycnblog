数学模型公式详细讲解：

在这一部分，我们将详细讲解一种用于评估人机交互系统效果的数学模型，即用户满意度（User Satisfaction, US）模型。用户满意度模型是一种用于评估人机交互系统效果的数学模型，它可以帮助我们更好地理解人机交互系统的效果。

用户满意度模型的公式如下：

$$
US = \frac{\sum_{i=1}^{n} (S_{i} - M_{i})}{n}
$$

其中，$S_{i}$ 表示用户对第 $i$ 个特征的满意度，$M_{i}$ 表示用户对第 $i$ 个特征的期望满意度，$n$ 表示特征的数量。

通过使用用户满意度模型，我们可以评估人机交互系统的效果，并优化它们。

代码实例：

我们将设计一个简单的在线购物系统，该系统允许用户查看商品信息，添加商品到购物车，并完成购买。

```html
  margin: 10px;
}
.product img {
  width: 100px;
  height: 100px;
}
.product button {
  margin-top: 10px;
}
</style>
</head>
<body>

<div class="product">
  <h2>Product 1</h2>
  <p>Price: $10</p>
  <button onclick="addToCart(1)">Add to Cart</button>
</div>

<div class="product">
  <h2>Product 2</h2>
  <p>Price: $20</p>
  <button onclick="addToCart(2)">Add to Cart</button>
</div>

<script>
function addToCart(productId) {
  alert("Product " + productId + " added to cart");
}
</script>

</body>
</html>
```
