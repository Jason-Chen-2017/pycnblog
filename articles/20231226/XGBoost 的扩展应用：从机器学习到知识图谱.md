首先，我们需要将数据集转换为一个可以用于训练 XGBoost 模型的格式。这通常包括将数据集分为特征和标签两部分，并将特征编码为数值。

接下来，我们可以使用 XGBoost 库来训练一个模型。以下是一个简单的代码示例：

```python
import xgboost as xgb

# 准备数据
X_train, y_train, X_test, y_test = ... # 从数据集中获取特征和标签

# 训练模型
model = xgb.XGBClassifier()
model.fit(X_train, y_train)

# 预测
predictions = model.predict(X_test)
```

在这个示例中，我们使用了 XGBClassifier 类来训练一个分类模型。这个模型可以用于预测一个实体与另一个实体之间的关系。
