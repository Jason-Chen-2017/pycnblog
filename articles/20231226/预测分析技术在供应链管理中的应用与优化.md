## 1.时间序列分析的主要方法和公式

时间序列分析的主要方法包括：

1. 移动平均（Moving Average, MA）
2. 指数移动平均（Exponential Moving Average, EMA）
3. 差分（Differencing）
4. 季节性分解（Seasonal Decomposition）
5. 自动差分（Auto-Differencing）
6. 趋势分解（Trend Decomposition）

时间序列分析的主要公式包括：

1. 移动平均的公式为：$$ MA_t = \frac{1}{w} \sum_{i=0}^{w-1} X_{t-i} $$
2. 指数移动平均的公式为：$$ EMA_t = \alpha X_t + (1-\alpha) EMA_{t-1} $$
3. 差分的公式为：$$ \Delta X_t = X_t - X_{t-1} $$
4. 季节性分解的公式为：$$ S_t = X_t - T_t $$
5. 自动差分的公式为：$$ \nabla X_t = X_t - X_{t-1} $$
6. 趋势分解的公式为：$$ T_t = \beta_0 + \beta_1 t + \epsilon_t $$

## 2.机器学习的主要算法和公式

机器学习的主要算法包括：

1. 线性回归（Linear Regression, LR）
2. 逻辑回归（Logistic Regression, LR）
3. 决策树（Decision Tree, DT）
4. 支持向量机（Support Vector Machine, SVM）

机器学习的主要公式包括：

1. 线性回归的公式为：$$ y = \beta_0 + \beta_1 x_1 + \cdots + \beta_n x_n + \epsilon $$
2. 逻辑回归的公式为：$$ P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + \cdots + \beta_n x_n)}} $$
3. 决策树的公式为：$$ \text{if } x_1 \leq \text{split}_1 \text{ then } \cdots \text{ else } \cdots $$
4. 支持向量机的公式为：$$ \min_{\mathbf{w},b} \frac{1}{2} \mathbf{w}^T \mathbf{w} \text{ s.t. } y_i (\mathbf{w}^T \mathbf{x_i} + b) \geq 1, i=1,\cdots,n $$

## 3.深度学习的主要算法和公式

深度学习的主要算法包括：

1. 卷积神经网络（Convolutional Neural Network, CNN）
2. 循环神经网络（Recurrent Neural Network, RNN）
3. 自然语言处理（Natural Language Processing, NLP）

深度学习的主要公式包括：

1. 卷积神经网络的公式为：$$ y = \text{softmax}(Wx + b) $$
2. 循环神经网络的公式为：$$ h_t = \text{tanh}(W_{hh} h_{t-1} + W_{xh} x_t + b_h) $$
