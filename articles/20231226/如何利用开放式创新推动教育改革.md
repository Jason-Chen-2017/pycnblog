# 2.核心概念与联系

## 2.1 开放式创新

开放式创新是一种新型的创新模式，它通过将创新过程打开给更多人，提高了创新的效率和质量。开放式创新的核心思想是将创新过程与网络社区的知识和人才结合，通过大众参与和协作，实现创新的目的。

开放式创新的主要特点是：

- 开放性：开放式创新将创新过程打开给更多人，让他们参与到创新中来。
- 协作性：开放式创新通过网络和其他技术手段，让不同的人和组织协作，共同创造价值。
- 分享性：开放式创新鼓励分享知识和资源，让更多人可以利用这些资源来创造更多的价值。
