# 2.核心概念与联系
拓扑相变的核心概念包括：拓扑状态、相变点、拓扑逆变等。这些概念在物理学、数学和计算机科学中都具有重要意义。

## 2.1 拓扑状态
拓扑状态（topological state）是指系统在不同条件下具有不同拓扑结构的状态。拓扑状态可以被描述为不变的量，它们在系统的能量、秩序和相互作用发生变化时保持不变。拓扑状态的一个典型例子是电子系统中的顶点玻色子（Dirac fermion），它们具有不可观测的拓扑数（topological number），使其在某些条件下保持稳定。

## 2.2 相变点
相变点（phase transition point）是系统在不同条件下拓扑状态之间转换的临界点。在相变点处，系统的能量、秩序和相互作用之间的变化导致拓扑状态的转换。相变点的一个典型例子是超导体中的超导相变点，在这个点上，超导体从普通状态转换到超导状态。

## 2.3 拓扑逆变
拓扑逆变（topological transition）是指系统在相变点附近发生的拓扑状态转换。拓扑逆变通常涉及到系统的能量、秩序和相互作用之间的平衡，这种平衡使得系统在相变点附近的拓扑状态发生转换。拓扑逆变的一个典型例子是迪拉克玻色子（Dirac fermion）在磁场应用时的拓扑逆变，这种逆变导致玻色子的拓扑数发生变化。
