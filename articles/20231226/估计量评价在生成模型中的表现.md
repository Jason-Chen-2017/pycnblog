# 2. 核心概念与联系

在本节中，我们将介绍生成模型、估计量以及它们之间的关系。

## 2.1 生成模型

生成模型是一类生成模型，它们的目标是生成新的数据样本。这些模型可以用于文本生成、图像生成、音频生成等任务。生成模型的主要组件包括：

1. 编码器：将输入数据编码为低维的向量表示。
2. 解码器：根据编码器的输出生成新的数据样本。

生成模型的表现取决于它们的设计和训练。为了评估生成模型的表现，我们需要一种方法来衡量模型在生成新数据时的性能。这就是估计量的作用。

## 2.2 估计量

估计量是一种用于评估模型性能的量度。在生成模型中，估计量可以用于评估模型的质量、稳定性、可解释性等方面。不同的估计量可能会对模型的表现产生不同的影响，因此选择合适的估计量至关重要。

## 2.3 生成模型与估计量之间的关系

生成模型和估计量之间的关系可以通过以下几点来概括：

1. 估计量用于评估生成模型的表现。
2. 选择合适的估计量可以帮助提高生成模型的性能。
3. 生成模型的设计和训练可能会影响估计量的选择和使用。
