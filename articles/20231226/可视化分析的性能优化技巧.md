## 3.1 算法优化

### 3.1.1 分治法

分治法是一种递归地将问题拆分成子问题，然后解决子问题并将结果合并的算法。分治法的时间复杂度通常为O(nlogn)，空间复杂度为O(logn)。

具体操作步骤：

1. 将数据分成若干个子集。
2. 对于每个子集，递归地应用分治法。
3. 将子集的结果合并为最终结果。

数学模型公式：

$$
T(n) = 2T(n/2) + O(n)
$$

### 3.1.2 动态规划

动态规划是一种递归地解决问题的算法，通过将问题拆分成子问题并解决子问题，然后将子问题的结果存储在一个表格中，以便后续使用。动态规划的时间复杂度通常为O(n^2)，空间复杂度为O(n^2)。

具体操作步骤：

1. 创建一个表格，用于存储子问题的结果。
2. 对于每个子问题，递归地应用动态规划。
3. 将子问题的结果存储在表格中。
4. 从表格中获取最终结果。

数学模型公式：

$$
dp[i] = max(dp[i-1], dp[i-2] + a[i])
$$

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]
    return merge(merge_sort(left), merge_sort(right))

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```
