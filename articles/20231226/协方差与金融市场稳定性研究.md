# 2.核心概念与联系
协方差是一种度量两个随机变量之间线性相关性的统计量。它可以用来衡量两个随机变量的变化程度是否相同，以及它们之间是否存在某种联系。在金融市场中，协方差可以用来度量不同金融市场之间的相关性，从而帮助我们更好地理解和预测市场波动。

协方差的计算公式为：
$$
\text{Cov}(X,Y) = E[(X - \mu_X)(Y - \mu_Y)]
$$
其中，$X$ 和 $Y$ 是两个随机变量，$\mu_X$ 和 $\mu_Y$ 是它们的均值。

协方差的值范围为 $-1 \leq \text{Cov}(X,Y) \leq 1$。当协方差等于1时，表示两个随机变量是完全正相关的；当协方差等于-1时，表示两个随机变量是完全负相关的；当协方差等于0时，表示两个随机变量是无相关的。

import numpy as np

# 假设X和Y是两个随机变量，它们的值分别为：

# 计算X和Y的均值

# 计算协方差

print("协方差：", cov_XY)
