# 2.核心概念与联系

在本节中，我们将介绍数据质量的核心概念和联系。

## 2.1 准确性

准确性是指数据是否正确地反映了实际情况。准确性是数据质量的一个关键要素，因为只有准确的数据才能支持正确的决策。

## 2.2 可靠性

可靠性是指数据是否可以依赖地用于决策和运营。可靠性是数据质量的另一个关键要素，因为只有可靠的数据才能支持稳定的决策和运营。

## 2.3 一致性

一致性是指数据是否在不同来源和时间点之间保持一致。一致性是数据质量的另一个关键要素，因为只有一致的数据才能支持一致的决策和运营。

## 2.4 及时性

及时性是指数据是否在所需时间内得到更新和提供。及时性是数据质量的另一个关键要素，因为只有及时的数据才能支持及时的决策和运营。

## 2.5 完整性

完整性是指数据是否缺失重要信息。完整性是数据质量的另一个关键要素，因为只有完整的数据才能支持全面的决策和运营。

## 2.6 有用性

有用性是指数据是否能够满足用户需求。有用性是数据质量的另一个关键要素，因为只有有用的数据才能支持有效的决策和运营。

## 4.1 数据清洗的具体代码实例

### 4.1.1 数据校验的具体代码实例

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 数据校验
def check_data(data):
    for col in data.columns:
        if data[col].isnull().sum() > 0:
            print(f'{col} 列存在缺失值')

check_data(data)
```

### 4.1.2 数据转换的具体代码实例

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 数据转换
def convert_data(data, col, old_value, new_value):
    data[col] = data[col].replace(old_value, new_value)
    return data

data = convert_data(data, 'gender', 'male', 'female')
```

### 4.1.3 数据填充的具体代码实例

```python
import pandas as pd
import numpy as np

# 加载数据
data = pd.read_csv('data.csv')

# 数据填充
def fill_data(data, col, value):
    data[col] = data[col].fillna(value)
    return data

data = fill_data(data, 'age', np.mean(data['age']))
```

### 4.1.4 数据删除的具体代码实例

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 数据删除
def delete_data(data, col, value):
    data = data[data[col] != value]
    return data

data = delete_data(data, 'gender', 'male')
```

### 4.1.5 数据归一化的具体代码实例

```python
import pandas as pd
import numpy as np

# 加载数据
data = pd.read_csv('data.csv')

# 数据归一化
def normalize_data(data, col, min_val, max_val):
    data[col] = (data[col] - min_val) / (max_val - min_val)
    return data

data = normalize_data(data, 'age', np.min(data['age']), np.max(data['age']))
```

### 4.2.3 一致性的具体代码实例

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 一致性
def consistency(data):
    return (data == data.groupby('user_id').mean()).mean()

consistency(data)
```

### 4.2.4 及时性的具体代码实例

```python
import pandas as pd
import datetime

# 加载数据
data = pd.read_csv('data.csv')

# 及时性
def timeliness(data, time_col, time_threshold):
    late_data = data[data[time_col] > time_threshold]
    return (data.shape[0] - late_data.shape[0]) / data.shape[0]

time_col = 'last_update'
time_threshold = datetime.datetime.now() - datetime.timedelta(days=7)

timeliness(data, time_col, time_threshold)
```

### 4.2.5 完整性的具体代码实例

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 完整性
def completeness(data, col):
    return (data[col].isnull().sum() == 0) / data.shape[0]

col = 'age'
completeness(data, col)
```

### 4.2.6 有用性的具体代码实例

```python
import pandas as pd

# 加载数据
data = pd.read_csv('data.csv')

# 有用性
def usefulness(data, target_col):
    return (data[target_col] != 0).mean()

target_col = 'salary'
usefulness(data, target_col)
```
