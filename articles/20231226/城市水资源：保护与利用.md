# 2.核心概念与联系

在研究城市水资源保护与利用时，我们需要了解以下几个核心概念：

- 水资源：水资源是指能够满足人类生活、经济发展和生态平衡需要的所有水体。水资源可以分为：
  - 表面水资源：包括河流、湖泊、海岸等水体。
  - 地下水资源：包括地下水流、水库等水体。
- 城市水资源：城市水资源是指城市内部的水资源，包括：
  - 废水：城市居民生活、工业和农业等活动产生的废水。
  - 雨水：城市雨水收集和利用。
  - 地下水：城市地下水资源的保护与利用。
- 水资源保护与利用：水资源保护与利用是指通过合理的水资源管理、开发、利用和保护措施，确保水资源的可持续性和可靠性。
