### 2.1 凸函数的定义和性质

凸函数是一种特殊的函数，它在其域内具有最小值，并且在其域内具有单调性。更正式地说，对于一个实值函数f(x)，如果对于任何x1、x2在其域内，且0≤λ≤1时有f(λx1+(1-λ)x2)≤λf(x1)+(1-λ)f(x2)，则f(x)是一个凸函数。

凸函数具有以下性质：

1. 凸函数在其域内具有最小值。
2. 凸函数在其域内具有单调性。
3. 凸函数的梯度是单调递增的。
4. 凸函数的Hessian矩阵是非负的。

### 4.1 代码实例1：使用凸函数解决最小化问题

以下是一个使用凸函数解决最小化问题的代码实例：

```python
import numpy as np

def f(x):
    return x**2

def gradient(x):
    return 2*x

def update(x, alpha):
    return x - alpha * gradient(x)

x = np.random.rand()
alpha = 0.1
for i in range(1000):
    x = update(x, alpha)
print(x)
```
