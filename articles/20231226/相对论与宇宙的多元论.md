# 2.核心概念与联系
# 2.1 相对论
相对论是以艾萨克·埃尔辛斯泰勒（Albert Einstein）于1905年发表的两篇论文为起点的。相对论的核心概念是空间和时间的相对性，即空间和时间的关系取决于观察者的运动和位置。相对论将重力视为曲空间时间的结果，而不是力的作用。

相对论的主要贡献包括：

1. 时间膨胀：在引力场中，时间会变慢。这意味着靠近重力源（如星球或黑洞）的物体会经历时间更慢的流逝。

2. 长度变形：在引力场中，物体的长度会变化。靠近重力源，物体会变短，而靠远，物体会变长。

3. 质量变化：在引力场中，物体的质量会变化。靠近重力源，物体的质量会增加，靠远，物体的质量会减少。

4. 光的弯曲：光在引力场中会被弯曲。这意味着在靠近重力源的地方，星空会变得扭曲。
