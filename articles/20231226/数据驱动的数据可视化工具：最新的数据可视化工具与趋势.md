## 4.1Python的matplotlib库

matplotlib是一个流行的Python数据可视化库，它提供了丰富的图表类型和自定义选项，可以用于创建静态和动态图表。以下是一个使用matplotlib库创建简单柱状图的代码实例：

```python
import matplotlib.pyplot as plt

# 创建数据
data = {'北京': 1200, '上海': 1500, '广州': 1800, '深圳': 2000}

# 创建柱状图
plt.bar(data.keys(), data.values())

# 设置图表标题和坐标轴标签
plt.title('中国四大城市GDP对比')
plt.xlabel('城市')
plt.ylabel('GDP')

# 显示图表
plt.show()
```

## 4.2Python的pandas库

pandas是一个流行的Python数据分析库，它提供了强大的数据处理功能，可以用于数据预处理和数据分析。以下是一个使用pandas库读取CSV文件、预处理数据并进行统计分析的代码实例：

```python
import pandas as pd

# 读取CSV文件
df = pd.read_csv('data.csv')

# 预处理数据
df['price'] = df['price'].astype(float)
df['area'] = df['area'].astype(int)

# 进行统计分析
mean_price = df['price'].mean()
median_price = df['price'].median()
std_price = df['price'].std()

print('均值：', mean_price)
print('中位数：', median_price)
print('标准差：', std_price)
```
