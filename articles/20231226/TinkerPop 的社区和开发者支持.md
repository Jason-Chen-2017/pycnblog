                 

# 1.背景介绍

TinkerPop是一个开源的图数据处理框架，它提供了一种标准的API，以便在不同的图数据库之间进行交互和操作。TinkerPop的目标是提供一种通用的图数据处理模型，以便在不同的应用场景中使用。TinkerPop的核心组件包括Gremlin和Blueprints，后者是一个接口规范，用于定义图数据库的基本功能。TinkerPop还提供了一些扩展功能，如TinkerGraph和JanusGraph。

TinkerPop的社区和开发者支持非常丰富，它有一个活跃的开发者社区，提供了许多资源和工具，以帮助开发者更好地使用和理解TinkerPop。这篇文章将介绍TinkerPop的社区和开发者支持的各个方面，包括文档、教程、论坛、社交媒体和开发者工具。
