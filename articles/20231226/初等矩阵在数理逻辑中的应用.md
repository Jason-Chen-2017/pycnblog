# 2.核心概念与联系

## 2.1 初等矩阵

初等矩阵是指在行列式不变的前提下，可以通过初等矩阵变换得到的矩阵。初等矩阵可以通过以下四种基本变换得到：

1. 交换两行或两列。
2. 将某一行或列的所有元素乘以一个非零常数。
3. 将某一行或列的所有元素加上或减去另一行或列的某个元素。
