### 代码实例

```python
def generate_waste_data():
    waste_data = np.random.randint(100, 1000, size=(100, 4))
    return waste_data

def process_waste(waste_data):
    T = 37
    O2 = 21
    S = 0.9
    k1 = 0.01
    k2 = 0.02
    P = k1 * T * O2 * S
    Q = k2 * T * O2 * S
    return P, Q

waste_data = generate_waste_data()
P, Q = process_waste(waste_data)
print("废弃物处理的产量: ", P)
print("废弃物处理的消化量: ", Q)
```

### 解释说明

1. 生成废弃物数据：通过 `numpy` 库生成随机的废弃物数据，数据包括废弃物的种类、组成成分等。
2. 处理废弃物：通过设定温度、氧浓度、盐浓度等条件，调用 `process_waste` 函数进行废弃物处理，计算废弃物处理的产量和消化量。
3. 输出结果：输出废弃物处理的产量和消化量。
