## 4.1 数据清洗

### 4.1.1 数据剥离

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 去重
data = data.drop_duplicates()

# 分离
data = data.drop(['column1', 'column2'], axis=1)

# 合并
data = pd.concat([data, pd.read_csv('data2.csv')], axis=1)
```

### 4.1.2 数据转换

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 类型转换
data['column1'] = data['column1'].astype('float')

# 单位转换
data['column2'] = data['column2'].mul(1000)

# 格式转换
data['column3'] = data['column3'].str.strip()
```

### 4.1.3 数据填充

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 默认填充
data['column1'].fillna(0, inplace=True)

# 预测填充
data['column2'].fillna(data['column2'].mean(), inplace=True)

# 插值填充
data['column3'].fillna(method='ffill', inplace=True)
```

### 4.1.4 数据删除

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 过滤
data = data[data['column1'] > 0]

# 筛选
data = data[data['column2'] == 'x']

# 排除
data = data.drop(['column3'], axis=1)
```

### 4.2.2 数据比较

```python
import pandas as pd

# 读取数据

data2 = pd.read_csv('data2.csv')

# 相等比较
if data1.equals(data2):
    print('数据相等')
else:
    print('数据不相等')

# 相似度比较

    print('数据相似')
```
