## 4.1 Mesos代码实例

以下是一个简单的Mesos代码实例：

```python
from mesos import MesosExecutor

class MyExecutor(MesosExecutor):
    def __init__(self, task, task_id, slave_id, master_info, framework_info):
        super(MyExecutor, self).__init__(task, task_id, slave_id, master_info, framework_info)

    def register(self):
        self.register_task(self.task)

    def launch(self):
        self.launch_task(self.task)

    def stop(self):
        self.stop_task(self.task)

if __name__ == '__main__':
    executor = MyExecutor(task, task_id, slave_id, master_info, framework_info)
    executor.run()
```

在上面的代码中，我们定义了一个名为`MyExecutor`的类，它继承了`MesosExecutor`类。然后，我们实现了`register`、`launch`和`stop`方法，从而让`MyExecutor`类可以与Mesos框架进行通信。最后，我们创建了一个`MyExecutor`实例，并调用`run`方法来启动执行器。

## 4.2 Docker代码实例

以下是一个简单的Docker代码实例：

```python
from docker import Client

client = Client()

def build_image(image_name, build_args):
    build_options = {'rm': True}
    client.build(build_args, image_name, **build_options)

def run_container(image_name, container_args):
    container = client.create_container(image_name, **container_args)
    client.start(container)

if __name__ == '__main__':
    build_image('my_image', {'path': 'Dockerfile'})
    run_container('my_image', {'command': 'my_command'})
```

在上面的代码中，我们首先导入了`Client`类，然后创建了一个客户端实例。然后，我们定义了`build_image`和`run_container`函数，从而让我们可以构建镜像和运行容器。最后，我们调用`build_image`和`run_container`函数来构建镜像和运行容器。
