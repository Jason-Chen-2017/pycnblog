```python
import networkx as nx

# 创建一个有5个节点和5个边的网络

G.add_edges_from([(1, 2), (1, 3), (2, 4), (3, 4), (4, 5)])

# 检测网络的连通性
is_connected = nx.is_connected(G)
print("网络是否连通：", is_connected)

# 检测网络中是否存在循环
is_cycle = nx.is_cycle(G)
print("网络中是否存在循环：", is_cycle)

# 检测网络是否是二部图
is_bipartite = nx.is_bipartite(G)
print("网络是否是二部图：", is_bipartite)
```

在这个代码实例中，我们首先导入了`networkx`库，并创建了一个有5个节点和5个边的网络。然后我们使用`nx.is_connected`函数来检测网络的连通性，使用`nx.is_cycle`函数来检测网络中是否存在循环，使用`nx.is_bipartite`函数来检测网络是否是二部图。
