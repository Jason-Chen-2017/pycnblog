$$
P(O|H) = \prod_{t=1}^{T} a_t P(o_t|h_t)

其中，$O$ 是观测序列，$H$ 是隐状态序列，$a_t$ 是转移概率，$P(o_t|h_t)$ 是发射概率。

其中，$f(x)$ 是输出函数，$\omega$ 是权重向量，$\phi(x)$ 是特征映射，$b$ 是偏置项。

## 4.1 实体识别

实体识别的一个常见实现是基于规则引擎的实体识别。以下是一个简单的Python代码实例：

```python
import re

def entity_recognition(text):
    # 定义实体规则
    rules = [
        (r'\b[A-Z][a-z]*\b', 'PERSON'),
        (r'\b[0-9]{4}\b', 'DATE'),
        (r'\b[A-Za-z0-9_-]+@[A-Za-z0-9_-]+\.[A-Za-z]{2,}\b', 'EMAIL'),
    ]

    # 匹配实体
    entities = []
    for rule, entity_type in rules:
        entities.extend(match.group(0) for match in re.finditer(rule, text))

    return entities

text = "John Doe was born on 1990-01-01. His email is john.doe@example.com."
text = entity_recognition(text)
print(text)
```

上述代码实例定义了一些实体规则，例如人名、日期、电子邮件地址等。然后，通过正则表达式匹配文本中的实体，并将实体类型标记为实体ID。最后，输出识别出的实体。

def relation_extraction(text, entities):
    # 定义关系规则
    rules = [
        (r'\b[A-Z][a-z]* was born on \b', 'BIRTH_DATE'),
        (r'\b[A-Z][a-z]* has an email \b', 'EMAIL'),

上述代码实例定义了一些关系规则，例如出生日期、电子邮件地址等。然后，通过正则表达式匹配文本中的实体和关系，并将关系类型标记为关系ID。最后，输出识别出的关系。
