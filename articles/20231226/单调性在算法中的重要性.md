## 3.1 排序算法
排序算法是一种常见的算法，它的目的是将一个数据集按照某种顺序进行排序。排序算法可以根据不同的顺序进行分类，例如非递减顺序、非递增顺序、逆序等。在这里，我们主要关注非递减顺序和非递增顺序的排序算法。

### 3.1.1 非递减顺序排序算法
非递减顺序排序算法的目的是将一个数据集按照非递减顺序进行排序。这种排序算法的核心思想是将数据集中的元素按照其值进行比较，并将较小的元素放在前面，较大的元素放在后面。

非递减顺序排序算法的一个典型例子是冒泡排序（Bubble Sort）。冒泡排序的基本思想是将数据集中的元素依次进行比较和交换，直到所有元素都到达正确的位置为止。冒泡排序的时间复杂度为O(n^2)，其中n是数据集的大小。

### 3.1.2 非递增顺序排序算法
非递增顺序排序算法的目的是将一个数据集按照非递增顺序进行排序。这种排序算法的核心思想是将数据集中的元素按照其值进行比较，并将较大的元素放在前面，较小的元素放在后面。

非递增顺序排序算法的一个典型例子是选择排序（Selection Sort）。选择排序的基本思想是将数据集中的元素依次进行选择和交换，直到所有元素都到达正确的位置为止。选择排序的时间复杂度为O(n^2)，其中n是数据集的大小。

# 4.具体代码实例和详细解释说明
在这里，我们将通过一个具体的代码实例来说明单调性在算法中的应用。我们将选择一个常见的排序算法——冒泡排序，并编写一个Python程序来实现它。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个程序中，我们首先定义了一个名为`bubble_sort`的函数，它接受一个整数列表`arr`作为输入参数。然后，我们使用两个嵌套的`for`循环来实现冒泡排序的过程。在内部循环中，我们将当前元素与下一个元素进行比较，如果当前元素大于下一个元素，则交换它们的位置。这个过程会一直持续到所有元素都到达正确的位置为止。

通过这个例子，我们可以看到单调性在算法中的应用。在这个例子中，我们的目标是将一个整数列表按照非递减顺序进行排序。通过使用冒泡排序算法，我们可以确保输入和输出之间存在单调性，即排序后的列表中的元素始终保持不变的顺序。
