## 4.3 协同过滤算法实现

现在我们可以实现一个基于协同过滤的推荐系统。协同过滤算法的原理是根据用户的历史行为（观看历史）来推荐类似的影视内容。

```python
from collections import defaultdict

def collaborative_filtering(user_history, movie_data):
    user_similarity = defaultdict(dict)
    for movie in movie_data:
        if movie["title"] not in user_history:
            continue
        for other_movie in movie_data:
            if other_movie["title"] not in user_history:
                continue
            if movie["category"] == other_movie["category"]:
                user_similarity[movie["title"]][other_movie["title"]] = 1
            else:
                user_similarity[movie["title"]][other_movie["title"]] = 0

    user_preference = defaultdict(dict)
    for movie in movie_data:
        user_preference[movie["title"]] = 0

    for movie in movie_data:
        if movie["title"] not in user_history:
            continue
        for other_movie in movie_data:
            if other_movie["title"] not in user_history:
                continue
            if movie["title"] == other_movie["title"]:
                continue
            if user_similarity[movie["title"]][other_movie["title"]] == 1:
                user_preference[other_movie["title"]] += user_preference[movie["title"]]
            else:
                user_preference[other_movie["title"]] += user_preference[movie["title"]] / user_similarity[movie["title"]][other_movie["title"]]

    return [movie for movie in sorted(user_preference.items(), key=lambda x: x[1], reverse=True)[:5]]

recommended_movies = collaborative_filtering(user_history, data)
print(recommended_movies)
```

上述代码实现了一个基于协同过滤的推荐系统，根据用户的观看历史和喜好，推荐了5部类似的影视作品。具体的推荐算法实现过程如下：

1. 首先，我们定义了一个用户行为数据处理函数，用于处理用户的观看历史和喜好。
2. 接下来，我们实现了一个协同过滤算法，根据用户的观看历史来推荐类似的影视内容。具体来说，我们首先计算了影视内容之间的相似度，然后根据相似度来调整用户对其他影视内容的评分，最后按照调整后的评分排序，推荐出5部最相似的影视作品。
