## 1.数据清洗

### 1.1 数据去重

数据去重是一种常见的数据清洗方法，用于删除数据中重复的记录。常见的数据去重算法包括：

1.基于哈希表的去重算法：将数据中的每一条记录存储到哈希表中，如果哈希表中已经存在相同的记录，则删除该记录。
2.基于排序的去重算法：将数据按照某个字段进行排序，然后删除相邻重复记录。

### 1.2 数据填充

数据填充是一种常见的数据清洗方法，用于填充数据中缺失的值。常见的数据填充算法包括：

1.基于平均值的填充算法：将缺失值替换为相同字段中的平均值。
2.基于中位数的填充算法：将缺失值替换为相同字段中的中位数。
3.基于最大值和最小值的填充算法：将缺失值替换为相同字段中的最大值或最小值。

### 1.3 数据格式转换

数据格式转换是一种常见的数据清洗方法，用于将数据中的不同格式转换为统一格式。常见的数据格式转换算法包括：

1.字符串到数字的转换：将字符串类型的数值转换为数字类型。
2.数字到字符串的转换：将数字类型的数值转换为字符串类型。
3.日期到时间戳的转换：将日期类型的时间转换为时间戳类型。

### 1.4 数据标记

数据标记是一种常见的数据清洗方法，用于将数据中的不规范值标记为异常值。常见的数据标记算法包括：

1.基于正则表达式的标记算法：将不规范的值替换为异常值，通过正则表达式匹配。
2.基于字符串长度的标记算法：将不规范的值替换为异常值，通过字符串长度判断。


### 1.2 数据填充

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 填充缺失值
data['age'].fillna(data['age'].mean(), inplace=True)

# 保存填充后的数据
data.to_csv('data_filled.csv', index=False)
```

### 1.3 数据格式转换

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 转换日期格式
data['date'] = pd.to_datetime(data['date'])

# 转换时间戳格式
data['timestamp'] = data['date'].apply(lambda x: x.timestamp())

# 保存转换后的数据
data.to_csv('data_transformed.csv', index=False)
```

### 1.4 数据标记

```python
import pandas as pd
```
