# 2.核心概念与联系
## 2.1 向量与向量的关系
在几何几何中，向量是表示一个方向和长度的量。向量可以用一个点表示，该点的位置表示向量的起点，向量的方向和长度可以通过向量的斜率和长度来描述。在计算机图形学中，向量通常用于表示物体的位置、方向和旋转。在机器学习中，向量通常用于表示数据的特征。

## 2.3 余弦与夹角的关系
余弦是一个数学概念，它可以用来描述两个向量之间的关系。余弦的定义是：余弦=向量A的长度*向量B的长度/向量A和向量B之间的内积的绝对值。余弦的范围是[-1,1]，当余弦为1时，表示两个向量是完全相同的，当余弦为-1时，表示两个向量是完全相反的，当余弦为0时，表示两个向量是完全相互垂直的。

```python
import numpy as np

def dot_product(vector_a, vector_b):
    return np.dot(vector_a, vector_b)

def magnitude(vector):
    return np.linalg.norm(vector)

def cosine_similarity(vector_a, vector_b):
    dot = dot_product(vector_a, vector_b)
    mag_a = magnitude(vector_a)
    mag_b = magnitude(vector_b)
    return dot / (mag_a * mag_b)

vector_a = np.array([1, 2, 3])
vector_b = np.array([4, 5, 6])
print(cosine_similarity(vector_a, vector_b))
```
