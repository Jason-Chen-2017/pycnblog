# 4. 具体代码实例和解释

在这里，我们以一个简单的传感器数据预处理和特征提取的Python代码实例为例，进行详细解释：

```python
import numpy as np
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA

# 加载传感器数据
data = pd.read_csv('sensor_data.csv')

# 去噪
data['soil_humidity'] = data['soil_humidity'].rolling(window=5).mean()
data['temperature'] = data['temperature'].rolling(window=5).mean()
data['light'] = data['light'].rolling(window=5).mean()

# 填充缺失值
data['soil_humidity'].fillna(method='ffill', inplace=True)
data['temperature'].fillna(method='ffill', inplace=True)
data['light'].fillna(method='ffill', inplace=True)

# 归一化
scaler = StandardScaler()
data[['soil_humidity', 'temperature', 'light']] = scaler.fit_transform(data[['soil_humidity', 'temperature', 'light']])

# 特征提取
pca = PCA(n_components=2)
principal_components = pca.fit_transform(data[['soil_humidity', 'temperature', 'light']])
principal_df = pd.DataFrame(data=principal_components, columns=['principal_component_1', 'principal_component_2'])

```

在这个代码实例中，我们首先使用Pandas库加载传感器数据，然后使用rolling函数进行移动平均去噪，使用fillna函数填充缺失值，使用StandardScaler库进行归一化，最后使用PCA（主成分分析）进行特征提取。
