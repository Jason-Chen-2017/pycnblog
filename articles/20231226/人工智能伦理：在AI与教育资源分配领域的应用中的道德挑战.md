## 4.1 线性回归示例

我们将通过一个简单的线性回归示例来解释线性回归算法的原理和操作步骤。

```python
import numpy as np

# 生成一组随机数据
X = np.random.rand(100, 1)
y = 3 * X + 2 + np.random.randn(100, 1)

# 使用最小二乘法求解线性回归模型
theta = np.linalg.inv(X.T @ X) @ X.T @ y

# 预测
X_new = np.array([[0.5]])
y_pred = X_new @ theta
```

在这个示例中，我们首先生成了一组随机的X和y数据，其中y是X的线性变换加上噪声。然后，我们使用最小二乘法来求解线性回归模型的参数θ。最后，我们使用求解出的参数θ来预测新的X值对应的y值。
