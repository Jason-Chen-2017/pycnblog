```python
import numpy as np

# 定义基本函数
def f(x):
    return x**2

def g(x):
    return np.sin(x)

# 定义组合方法
def h(x, y):
    return x + y

# 设置参数
x = np.array([1, 2, 3, 4])

# 计算复合函数
y = h(f(x), g(x))

print(y)
```

在这个例子中，我们定义了两个基本函数`f(x)`和`g(x)`，并使用了顺序组合方法`h(x, y)`来组合它们。最后，我们使用了`numpy`库来计算复合函数的值。
