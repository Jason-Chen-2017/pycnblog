# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的例子来展示决策平面在环境保护领域的应用。假设我们需要优化一个城市的能源消耗和碳排放，目标是降低碳排放，同时保证能源需求的满足率。我们可以建立以下目标函数和约束条件：

目标函数：

$$
\min_{E,P} -E + 0.9P \\
s.t. E \geq 0, P \geq 0
$$

约束条件：

$$
\begin{aligned}
&E \leq 1000 \\
&P \leq 500 \\
&E + P \leq 1500
\end{aligned}
$$

其中，$E$ 表示能源消耗，$P$ 表示碳排放。我们可以使用Python的PuLP库来解决这个问题：

```python
from pulp import *

# 创建优化问题
prob = LpProblem("Environment_Protection", LpMinimize)

# 创建决策变量
E = LpVariable("Energy", lowBound=0)
P = LpVariable("Pollution", lowBound=0)

# 建立目标函数
prob += -E + 0.9*P, "Objective"

# 建立约束条件
prob += E <= 1000, "Constraint_1"
prob += P <= 500, "Constraint_2"
prob += E + P <= 1500, "Constraint_3"

# 求解问题
prob.solve()

# 输出结果
print("Status:", LpStatus[prob.status])
print("Energy:", E.varValue)
print("Pollution:", P.varValue)
```

运行这个代码，我们可以得到以下结果：

```
Status: Optimal
Energy: 1000.0
Pollution: 500.0
```

这表示在满足能源需求的同时，我们可以降低碳排放的量。
