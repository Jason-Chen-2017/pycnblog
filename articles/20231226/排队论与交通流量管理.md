# 2.核心概念与联系

在排队论中，队列是一个最基本的概念，它表示一个人或物在某个服务点等待服务的集合。交通拥堵可以理解为一种特殊类型的队列，其中车辆在道路上等待交通信号灯的绿灯信号。

排队论中的一些核心概念如下：

1. 服务速率（Service Rate）：服务点为车辆提供服务的速率，单位为车辆/时间单位。
2. 到达率（Arrival Rate）：车辆在道路上到达交通信号灯的速率，单位为车辆/时间单位。
3. 队列长度（Queue Length）：队列中等待服务的车辆数量。
4. 平均等待时间（Average Waiting Time）：车辆在队列中等待服务的平均时间。

def MM1_queue_analysis(lambda_arrival, mu_service):
    """
    M/M/1队列模型的性能分析
    :param lambda_arrival: 到达率
    :param mu_service: 服务速率
    :return: 平均等待时间、队列长度
    """
    p = lambda_arrival / mu_service
    L = lambda_arrival / (mu_service - lambda_arrival) if mu_service > lambda_arrival else np.inf
    W = L * p
