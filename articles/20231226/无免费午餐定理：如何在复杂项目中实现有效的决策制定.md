我们可以使用Python的PuLP库来解决这个问题。首先，我们需要安装PuLP库：


然后，我们可以编写如下代码来解决这个问题：

```python
import pulp

# 创建一个线性规划问题
prob = pulp.LpProblem("NoFreeLunch", pulp.LpMaximize)

# 创建变量
x1 = pulp.LpVariable("Task1", lowBound=0)
x2 = pulp.LpVariable("Task2", lowBound=0)
x3 = pulp.LpVariable("Task3", lowBound=0)

# 添加目标函数
prob += 1 * x1 + 2 * x2 + 1 * x3, "ResourceTotal"

# 添加约束条件
prob += x1 + 2 * x2 + x3 <= 4, "ResourceConstraint"

# 解决问题
prob.solve()

# 输出结果
print("Task1:", x1.varValue)
print("Task2:", x2.varValue)
print("Task3:", x3.varValue)
print("ResourceTotal:", pulp.value(prob.objective))
```

通过运行这个代码，我们可以得到以下结果：

```
Task1: 2.0
Task2: 0.0
Task3: 1.0
ResourceTotal: 5.0
```

这意味着我们应该执行任务1和任务3，并且最终可以获得5个资源。
