## 4.1 量子模拟的代码实例

我们来看一个简单的量子模拟代码实例，它使用量子计算机来解决一个简单的量子化学问题：两个原子之间的相互作用。

```python
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 初始化量子比特和量子门
qc = QuantumCircuit(2, 2)
qc.h(0) # 对第一个量子比特应用H门
qc.cx(0, 1) # 对第一个量子比特和第二个量子比特应用CX门

# 将量子电路编译为可执行的量子电路
qc = transpile(qc, baseline_gate_error=0.001, layout='static_2d')
qobj = assemble(qc)

# 使用量子计算机执行量子电路
simulator = Aer.get_backend('qasm_simulator')
result = simulator.run(qobj).result()

# 度量量子状态
counts = result.get_counts()
print(counts)
```

在这个代码实例中，我们首先导入了QuantumCircuit、Aer、transpile和assemble等库。然后，我们初始化了两个量子比特，并对第一个量子比特应用了H门，对第一个量子比特和第二个量子比特应用了CX门。接着，我们将量子电路编译为可执行的量子电路，并使用量子计算机执行量子电路。最后，我们度量量子状态，并打印出度量结果。
