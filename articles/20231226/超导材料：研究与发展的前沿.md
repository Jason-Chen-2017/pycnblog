                 

# 1.背景介绍

超导材料是一类具有极高电导率的材料，它们在零温度下可以导电，而不会产生电阻。这种特性使得超导材料在许多领域具有重要应用，例如量子计算、超导电机、磁共振成像（MRI）等。在过去几十年里，研究人员一直在努力寻找新的超导材料和提高现有超导材料的性能。

超导材料的发现可以追溯到1911年，当时荷兰物理学家Heike Kamerlingh Onnes在研究液体氢时发现了超导现象。随后，他通过对氢和其他元素进行实验发现，低温下氢和氧的混合物也具有超导性。然而，直到1986年，美国科学家Jerome Isaac Friedman、Henry W. Kendall和Richard E. Taylor发现了第一个高温超导材料——cuprate。这一发现催生了超导材料研究的新兴时期，并引起了广泛关注。
