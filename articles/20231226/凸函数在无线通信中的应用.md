## 2.1 凸函数的定义

凸函数是一种在数学中的一种特殊函数，它在整个定义域内具有最小值。形式上，一个函数f(x)在区间[a, b]上是凸函数，如果对于任何x1, x2在[a, b]内和其对应的取值y1 = f(x1), y2 = f(x2)，满足以下不等式成立：

f(λx1 + (1 - λ)x2) ≤ λy1 + (1 - λ)y2

其中λ ∈ [0, 1]。
