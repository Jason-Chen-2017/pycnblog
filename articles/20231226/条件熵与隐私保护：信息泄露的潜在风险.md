# 2.核心概念与联系

## 2.1 熵与条件熵
熵是信息论中的一个重要概念，用于衡量一种随机事件的不确定性。熵越高，事件的不确定性就越大。在信息论中，熵的公式为：

$$
H(X)=-\sum_{x\in X}P(x)\log_2P(x)
$$

条件熵是基于给定某一条件的熵，用于衡量在已知某个条件时，剩余未知信息的不确定性。条件熵的公式为：

$$
H(Y|X)=-\sum_{x\in X}\sum_{y\in Y}P(x,y)\log_2P(y|x)
$$

## 2.2 互信息与条件互信息
互信息是信息论中的一个概念，用于衡量两个随机变量之间的相关性。互信息的公式为：

$$
I(X;Y)=\sum_{x\in X}\sum_{y\in Y}P(x,y)\log_2\frac{P(x,y)}{P(x)P(y)}
$$

条件互信息是基于给定某一条件的互信息，用于衡量在已知某个条件时，剩余未知信息之间的相关性。条件互信息的公式为：

$$
I(Y;X|Z)=-\sum_{z\in Z}\sum_{x\in X}\sum_{y\in Y}P(x,y|z)\log_2P(x|z)
$$

## 4.1 计算熵和条件熵
```python
import numpy as np

def entropy(p):
    return -np.sum(p * np.log2(p))

def conditional_entropy(p_xy, p_x, p_y):
    return -np.sum(p_xy * np.log2(p_y))
```

def mutual_information(p_xy, p_x, p_y):
    return entropy(p_xy) - conditional_entropy(p_xy, p_x, p_y) - conditional_entropy(p_xy, p_y, p_x)

def conditional_mutual_information(p_xyz, p_xz, p_yz, p_x, p_z):
