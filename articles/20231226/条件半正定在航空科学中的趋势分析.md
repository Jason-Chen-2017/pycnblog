## 3.1 条件半正定的定义
对于一个线性系统，如果存在一个正定矩阵$P$和一个正实数$\omega$，使得系统的谱半正定性满足以下条件：
$$
\begin{aligned}
&(1) \quad Re(\lambda_i) < \omega, \quad i = 1,2,\cdots,n \\
&(2) \quad Re(\lambda_i) \leq -\omega, \quad i = n+1,n+2,\cdots,2n \\
\end{aligned}
$$
则称该系统是条件半正定的。

## 3.2 条件半正定的检测
要检测一个线性系统是否是条件半正定的，可以使用以下算法：

1. 计算系统的谱半正定性矩阵$T$。
2. 计算矩阵$T$的特征值$\lambda_i$。
3. 根据谱半正定性矩阵$T$的特征值$\lambda_i$判断系统的稳定性。

具体的算法实现如下：

```python
import numpy as np

def condition_half_definite(A, omega):
    n = A.shape[0]
    T = A.T @ A + omega * np.eye(n)
    lambda_values, v = np.linalg.eig(T)
    is_condition_half_definite = np.all(np.real(lambda_values) < omega)
```
