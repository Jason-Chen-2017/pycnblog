$$
f(x) = \frac{\Gamma(\alpha_1 + \alpha_2)}{\Gamma(\alpha_1)\Gamma(\alpha_2)} x^{\alpha_1 - 1}(1 - x)^{\alpha_2 - 1}
$$

其中，$\Gamma(\cdot)$ 是伽马函数，$\alpha_1$ 和 $\alpha_2$ 是贝塔分布的自由度参数。

## 1. 使用Python计算贝塔分布的PDF和CDF
我们可以使用Python的`scipy.stats`模块来计算贝塔分布的PDF和CDF。以下是一个示例代码：

```python
import numpy as np
from scipy.stats import beta

# 设置自由度参数
alpha1 = 5
alpha2 = 7

# 设置取值点
x = np.linspace(0, 1, 100)

# 计算PDF
pdf = beta.pdf(x, alpha1, alpha2)

# 计算CDF
cdf = beta.cdf(x, alpha1, alpha2)

# 绘制PDF和CDF
import matplotlib.pyplot as plt

plt.figure(figsize=(10, 6))
plt.plot(x, pdf, label='PDF')
plt.plot(x, cdf, label='CDF')
plt.legend()
plt.show()
```

在上述代码中，我们首先导入了`numpy`和`scipy.stats`模块。然后，我们设置了贝塔分布的自由度参数`alpha1`和`alpha2`，并创建了一个取值点数组`x`。接着，我们使用`beta.pdf`函数计算贝塔分布的PDF，并使用`beta.cdf`函数计算贝塔分布的CDF。最后，我们使用`matplotlib.pyplot`模块绘制了PDF和CDF的图像。

## 2. 使用Python计算贝塔分布的期望和方差
我们可以使用Python的`scipy.stats`模块来计算贝塔分布的期望和方差。以下是一个示例代码：

```python
import scipy.stats as stats

# 设置自由度参数
alpha1 = 5

# 计算期望
```

在上述代码中，我们首先导入了`scipy.stats`模块。然后，我们设置了贝塔分布的自由度参数`alpha1`和`alpha2`。接着，我们使用`stats.beta.mean`函数计算贝塔分布的期望，并使用`stats.beta.var`函数计算贝塔分布的方差。最后，我们打印了结果。
