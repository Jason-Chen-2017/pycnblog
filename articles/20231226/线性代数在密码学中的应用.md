## 4.4 AES加密算法

```python
import numpy as np

def aes_encrypt(key, plaintext):
    ciphertext = np.dot(key, plaintext)
    return ciphertext

def aes_decrypt(key, ciphertext):
    plaintext = np.dot(np.linalg.inv(key), ciphertext)
    return plaintext

key = np.array([[4, 1], [3, 2]])
plaintext = np.array([[1, 2], [3, 4]])

ciphertext = aes_encrypt(key, plaintext)
print(ciphertext)

plaintext_decrypt = aes_decrypt(key, ciphertext)
print(plaintext_decrypt)
```

在这个代码实例中，我们使用了numpy库来实现AES加密算法。我们定义了两个函数`aes_encrypt`和`aes_decrypt`，它们分别实现了AES加密和解密过程。我们使用了`np.dot`函数来计算矩阵乘法，并使用了`np.linalg.inv`函数来计算逆矩阵。
