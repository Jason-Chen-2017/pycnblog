## 3.1 信息度

信息度是一种衡量网络安全状况的指标，它可以用来衡量网络中信息的不确定性。信息度的公式为：

$$
I(X) = -\sum_{i=1}^{n} P(x_i) \log_2 P(x_i)
$$

其中，$X$ 是一个随机变量，$x_i$ 是 $X$ 的取值，$P(x_i)$ 是 $x_i$ 的概率。

## 3.2 熵

熵是一种衡量网络安全状况的指标，它可以用来衡量网络中信息的不确定性和随机性。熵的公式为：

$$
H(X) = -\sum_{i=1}^{n} P(x_i) \log_2 P(x_i)
$$

其中，$X$ 是一个随机变量，$x_i$ 是 $X$ 的取值，$P(x_i)$ 是 $x_i$ 的概率。

## 3.3 条件熵

条件熵是一种衡量网络安全状况的指标，它可以用来衡量给定某个条件下网络中信息的不确定性和随机性。条件熵的公式为：

$$
H(X|Y) = -\sum_{i=1}^{n} P(x_i|y_i) \log_2 P(x_i|y_i)
$$

其中，$X$ 和 $Y$ 是两个随机变量，$x_i$ 和 $y_i$ 是 $X$ 和 $Y$ 的取值，$P(x_i|y_i)$ 是 $x_i$ 给定 $y_i$ 的概率。

## 3.4 互信息

互信息是一种衡量网络安全状况的指标，它可以用来衡量两个随机变量之间的相关性。互信息的公式为：

$$
I(X;Y) = \sum_{i=1}^{n} P(x_i,y_i) \log_2 \frac{P(x_i,y_i)}{P(x_i)P(y_i)}
$$

其中，$X$ 和 $Y$ 是两个随机变量，$x_i$ 和 $y_i$ 是 $X$ 和 $Y$ 的取值，$P(x_i,y_i)$ 是 $x_i$ 和 $y_i$ 的联合概率，$P(x_i)$ 和 $P(y_i)$ 是 $x_i$ 和 $y_i$ 的单变量概率。

def entropy(data):
    label_counts = np.bincount(data.ravel())
    probabilities = label_counts / label_counts.sum()
    return -np.sum(probabilities * np.log2(probabilities))

# 计算条件熵

    condition_counts = np.bincount(condition.ravel())
    probabilities = condition_counts / condition_counts.sum()

    label_probabilities = label_counts / label_counts.sum()
    return -np.sum(probabilities * np.log2(probabilities)) - np.sum(probabilities * np.log2(label_probabilities))

    joint_counts = np.bincount(data[condition].ravel())
    single_counts = np.bincount(data.ravel())
    return -np.sum(joint_counts * np.log2(joint_counts)) - np.sum(single_counts * np.log2(single_counts)) + np.sum(joint_counts * np.log2(joint_counts / single_counts))

# 计算信息度
def information_degree(data):
    return entropy(data)

# 计算网络安全性能指标
information_degree = information_degree(data)
entropy = entropy(data)
