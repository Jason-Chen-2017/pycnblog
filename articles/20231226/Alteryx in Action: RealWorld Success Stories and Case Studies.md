# 2.核心概念与联系
# 2.1 Alteryx Analytics
Alteryx Analytics 是 Alteryx 的核心产品，它是一种基于云的数据整合和分析平台，可以帮助用户快速创建数据分析流程、可视化报告和应用程序。Alteryx Analytics 提供了丰富的数据源支持、强大的数据处理功能和高度可扩展的架构，使用户可以轻松地将数据整合、清洗、分析和可视化。

# 2.2 Alteryx Connect
Alteryx Connect 是 Alteryx 的另一款产品，它是一种基于云的数据同步和集成服务，可以帮助用户实现数据源之间的自动同步和集成。Alteryx Connect 支持多种数据源，包括 Salesforce、Marketo、HubSpot 等 CRM 系统，以及 Google Analytics、Adobe Analytics 等网站分析系统。

# 2.3 联系关系
Alteryx Analytics 和 Alteryx Connect 之间的联系关系是：Alteryx Connect 提供数据同步和集成服务，以便用户可以将数据源与 Alteryx Analytics 连接起来。这样，用户可以在 Alteryx Analytics 中直接访问和处理这些数据源，从而实现更高效、更智能的数据分析。
