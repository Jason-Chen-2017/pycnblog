# 2.核心概念与联系

在深入学习机器学习的基本原理之前，我们需要了解一些核心概念和联系。这些概念和联系包括：

1. 数据：数据是机器学习的基础，它是从实际场景中收集、整理和处理得到的信息。数据可以是结构化的（如表格数据、关系数据库）或者非结构化的（如文本、图像、音频、视频等）。
2. 特征：特征是数据中用于描述事物的属性或特点。在机器学习中，特征通常用于描述数据样本，以便于模型进行学习和预测。
3. 标签：标签是数据样本的预期输出或结果。在监督学习中，标签是用于训练模型的关键信息，模型的目标是学习如何根据输入的特征预测正确的标签。
4. 模型：模型是机器学习的核心组件，它是用于将输入特征映射到预期输出的函数或算法。模型可以是线性的（如线性回归）或非线性的（如支持向量机、决策树等）。
5. 损失函数：损失函数是用于衡量模型预测与实际标签之间差异的指标。损失函数的目标是使模型的预测更接近实际的标签，从而使模型的性能得到提高。
6. 优化算法：优化算法是用于更新模型参数以最小化损失函数的方法。常见的优化算法包括梯度下降、随机梯度下降、Adam等。
7. 评估指标：评估指标是用于衡量模型性能的标准。常见的评估指标包括准确率、召回率、F1分数等。
