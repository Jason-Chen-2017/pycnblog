## 4.1 温度传感器数据分析

假设我们有一台智能空调，它可以收集家庭成员在家中的温度数据。我们可以使用以下代码来分析这些数据，以便帮助家庭成员更好地控制空调：

```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression

# 加载温度数据
data = pd.read_csv('temperature_data.csv')

# 数据预处理
data['date'] = pd.to_datetime(data['date'])
data['day'] = data['date'].dt.day
data['month'] = data['date'].dt.month
data['year'] = data['date'].dt.year

# 分析温度数据
X = data[['day', 'month', 'year']]
y = data['temperature']

# 使用线性回归模型进行分析
model = LinearRegression()
model.fit(X, y)

# 预测今天的温度
today = pd.DataFrame({'day': [1], 'month': [1], 'year': [2021]})
print("今天的温度预测:", model.predict(today))
```

在这个代码实例中，我们首先使用pandas库加载温度数据，然后使用numpy库对数据进行预处理。接着，我们使用scikit-learn库的线性回归模型进行温度数据的分析。最后，我们使用模型预测今天的温度。

## 4.2 心率传感器数据分析

假设我们有一台智能心率监测设备，它可以收集家庭成员在睡眠时的心率数据。我们可以使用以下代码来分析这些数据，以便帮助家庭成员了解自己的健康状况：

```python
import numpy as np
import pandas as pd
from sklearn.cluster import KMeans

# 加载心率数据
data = pd.read_csv('heart_rate_data.csv')

# 数据预处理
data['date'] = pd.to_datetime(data['date'])
data['day'] = data['date'].dt.day
data['month'] = data['date'].dt.month
data['year'] = data['date'].dt.year

# 分析心率数据
X = data[['day', 'month', 'year', 'heart_rate']]

# 使用KMeans聚类算法进行分析
model = KMeans(n_clusters=3)
model.fit(X)

# 分析结果
print("心率分类结果:", model.labels_)
```

在这个代码实例中，我们首先使用pandas库加载心率数据，然后使用numpy库对数据进行预处理。接着，我们使用scikit-learn库的KMeans聚类算法进行心率数据的分析。最后，我们输出心率分类结果，以便家庭成员了解自己的健康状况。
