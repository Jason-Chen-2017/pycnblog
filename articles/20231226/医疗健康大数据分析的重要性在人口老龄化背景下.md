# 3.3 数学模型公式详细讲解
在医疗健康大数据分析中，常用的数学模型包括：线性回归模型、逻辑回归模型、决策树模型、支持向量机模型、聚类分析模型等。这些数学模型的公式如下：

1. 线性回归模型：$$ y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon $$
2. 逻辑回归模型：$$ P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - ... - \beta_nx_n}} $$
3. 决策树模型：通过递归地对数据集进行分割，将其划分为多个子节点，使得各子节点内部数据的纯度达到最大。
4. 支持向量机模型：通过寻找最大化满足约束条件下，分类器在支持向量所构成的hyperplane上的边际值，从而找到最佳的分类器。
5. 聚类分析模型：通过寻找数据集内部的簇，使得各簇内部数据的相似度达到最大，各簇间的相似度达到最小。
