# 2. 核心概念与联系

Java集合类的核心概念包括：

1. 集合接口：Collection、Set、List、Queue等。
2. 集合实现类：ArrayList、LinkedList、HashMap、HashSet等。
3. 集合工具类：Collections、Arrays等。

这些概念之间的联系如下：

1. 集合接口定义了集合类的基本功能，如添加、删除、查询等。
2. 集合实现类实现了集合接口的方法，提供了具体的数据结构和算法实现。
3. 集合工具类提供了一些静态方法，用于操作集合对象，如排序、反转、并集等。

# 4. 具体代码实例和详细解释说明

以下是一个简单的ArrayList实例：

```java
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<Integer> list = new ArrayList<>();
        list.add(1);
        list.add(2);
        list.add(3);
        System.out.println(list);
    }
}
```

这个实例中，我们创建了一个Integer类型的ArrayList对象，然后使用add方法将3个整数添加到列表中。最后，我们使用println方法输出列表。
