```python
data = pd.read_csv('supply_chain_data.csv')

# 数据清洗
data = data.dropna()

# 数据转换
data['total_cost'] = data['cost'] * data['quantity']

# 数据集成
data = data.groupby(['product_category', 'supplier']).agg({'total_cost': 'sum'})
```

## 4.2 供应链可视化的具体操作步骤

### 4.2.1 数据可视化

```python
import matplotlib.pyplot as plt

# 条形图
data.groupby('product_category')['total_cost'].sum().plot(kind='bar')
plt.show()

# 折线图
data.groupby('product_category')['total_cost'].resample('M').mean().plot()
plt.show()

# 散点图
plt.scatter(data['product_category'], data['total_cost'])
plt.show()

# 柱状图
data.groupby('supplier')['total_cost'].sum().plot(kind='bar')
plt.show()

# 饼图
data.groupby('product_category')['total_cost'].sum().value_counts().plot(kind='pie')
plt.show()
```
