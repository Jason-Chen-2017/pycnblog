# 2.核心概念与联系

## 2.1 欧氏距离

欧氏距离是一种度量空间中两点之间距离的方法，它是基于向量的差的长度。给定两个向量a和b，欧氏距离（Euclidean Distance）可以通过以下公式计算：

$$
d(a, b) = \sqrt{\sum_{i=1}^{n}(a_i - b_i)^2}
$$

其中，n是向量a和向量b的维数，$a_i$和$b_i$分别是向量a和向量b的第i个元素。

## 2.2 生物序列

生物序列是生物学中的基本单位，包括DNA、RNA和蛋白质序列。这些序列由一系列酸基（DNA和RNA）或氨基酸（蛋白质）构成。生物序列存储了生物功能和生物过程的信息，因此研究生物序列的相似性和差异性对于理解生物过程和生物系统非常重要。

2. 计算序列之间的差异，即插入、删除和替换操作的数量。
3. 使用欧氏距离公式计算序列之间的距离。

$$
d(a, b) = d(a, b-1) + 1 \\

```python
def euclidean_distance(a, b):
    n = len(a)
    distance = 0
    for i in range(n):
        distance += (a[i] - b[i]) ** 2
    return distance ** 0.5

def edit_distance(a, b):
    if not a or not b:
        return 0
    if a == b:
        return 0
    m, n = len(a), len(b)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1):
        for j in range(n + 1):
            if i == 0:
                dp[i][j] = j
            elif j == 0:
                dp[i][j] = i
            elif a[i - 1] == b[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]
            else:
                dp[i][j] = 1 + min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1])
    return dp[m][n]

a = ['A', 'C', 'T', 'G']
b = ['A', 'T', 'C', 'G']
print(euclidean_distance(a, b))
print(edit_distance(a, b))
```

# 6.附录常见问题与解答

1. Q: 欧氏距离和汉明距离有什么区别？
A: 欧氏距离是基于向量的差的长度，而汉明距离是基于二进制表示的差的计数。欧氏距离可以用于比较任意维度的向量，而汉明距离主要用于比较二进制序列。
2. Q: 欧氏距离是否适用于比较不同长度的序列？
A: 欧氏距离不适用于比较不同长度的序列。在比较不同长度的序列时，可以使用其他距离度量，如汉明距离或编辑距离。
3. Q: 欧氏距离是否能捕捉到序列之间的结构相似性？
A: 欧氏距离主要捕捉到序列之间的点对点差异，而不能直接捕捉到序列之间的结构相似性。为了捕捉到结构相似性，可以使用其他方法，如序列对齐或结构比较。
