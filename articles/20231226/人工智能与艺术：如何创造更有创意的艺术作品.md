## 4.1 图像生成示例

我们将通过一个简单的图像生成示例来说明如何使用人工智能技术创造艺术作品。这个示例使用Python编程语言和TensorFlow框架来生成一幅随机颜色的点阵画。

```python
import numpy as np
import matplotlib.pyplot as plt
import tensorflow as tf

# 生成随机颜色的点阵画
def generate_random_image(width, height, num_colors):
    # 生成随机颜色
    colors = np.random.randint(num_colors, size=(height, width, 3))
    # 生成随机点
    points = np.random.randint(0, width, size=(height, width))
    # 绘制点
    plt.imshow(colors[points])
    plt.show()

# 设置参数
width = 500
height = 500
num_colors = 10

# 生成图像
generate_random_image(width, height, num_colors)
```

在这个示例中，我们首先导入了必要的库，包括numpy、matplotlib和tensorflow。然后定义了一个名为`generate_random_image`的函数，该函数接受宽度、高度和颜色数量作为参数，并生成一幅随机颜色的点阵画。最后，我们设置了一些参数，并调用该函数来生成图像。

## 4.2 音乐生成示例

我们将通过一个简单的音乐生成示例来说明如何使用人工智能技术创造音乐作品。这个示例使用Python编程语言和tensorflow框架来生成一段随机音乐。

```python
import numpy as np
import tensorflow as tf

# 生成随机音乐
def generate_random_music(duration, tempo, num_notes):
    # 生成随机节奏
    rhythm = np.random.randint(0, tempo, size=(duration,))
    # 生成随机音调
    pitches = np.random.randint(0, num_notes, size=(duration,))
    # 生成音乐
    music = tf.signal.convolve(rhythm, pitches)
    return music

# 设置参数
duration = 100
tempo = 100
num_notes = 12

# 生成音乐
music = generate_random_music(duration, tempo, num_notes)
```

在这个示例中，我们首先导入了必要的库，包括numpy和tensorflow。然后定义了一个名为`generate_random_music`的函数，该函数接受持续时间、节奏和音符数量作为参数，并生成一段随机音乐。最后，我们设置了一些参数，并调用该函数来生成音乐。
