## 4.1 人力资源管理
### 4.1.1 对教师资源进行评估
```python
import numpy as np

# 教师资源的评分
scores = np.array([80, 85, 90, 95])

# 教师资源的使用量
usage = np.array([20, 25, 30, 35])

# 教师资源的分配量
allocation = np.zeros(len(scores))

# 教师资源的总量
total = np.sum(usage)

# 对教师资源进行评估和分配
for i in range(len(scores)):
    allocation[i] = scores[i] * usage[i] / total

print(allocation)
```
