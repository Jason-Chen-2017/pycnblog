# 2. 核心概念与联系

## 2.1 边缘计算

边缘计算（Edge Computing）是一种计算模式，将数据处理和应用程序从中心化的服务器移动到边缘设备（如路由器、交换机、传感器等）。这种模式可以在数据产生的地方进行实时处理，减少数据传输和计算延迟。边缘计算具有以下特点：

1. 低延迟：通过在边缘设备上进行实时处理，降低了数据传输和计算延迟。
2. 高效：通过在边缘设备上进行预处理，减少了中心化服务器的负载，提高了整体处理效率。
3. 安全：通过在边缘设备上进行处理，降低了数据泄露和盗用的风险。

## 2.2 卫星观测

卫星观测是一种从地球表面观测到地球的方法，通过卫星传感器收集到的数据，可以得到地球的各种信息，如地形、地貌、气候、资源等。卫星观测技术广泛应用于地球科学、气候变化、资源调查、国防和地图制作等领域。

在这里，我们以一个简单的卫星图像处理示例进行说明。我们将使用Python编程语言，以及OpenCV库进行图像处理。

然后，编写以下代码：

```python
import cv2
import numpy as np

# 加载卫星图像

# 将图像转换为灰度图像
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 对灰度图像进行二值化处理
_, binary_image = cv2.threshold(gray_image, 127, 255, cv2.THRESH_BINARY)

# 对二值化图像进行腐蚀处理，以去除噪声
kernel = np.ones((5,5), np.uint8)
eroded_image = cv2.erode(binary_image, kernel, iterations=1)

# 对腐蚀后的图像进行膨胀处理，以恢复原始特征
dilated_image = cv2.dilate(eroded_image, kernel, iterations=1)

# 显示处理后的图像
cv2.imshow('Processed Image', dilated_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

在这个示例中，我们首先使用OpenCV库加载卫星图像，然后将其转换为灰度图像，并对其进行二值化处理。接着，我们对二值化图像进行腐蚀处理，以去除噪声，并对腐蚀后的图像进行膨胀处理，以恢复原始特征。最后，我们显示处理后的图像。
