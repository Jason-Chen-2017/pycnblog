## 4.1 核反应的代码实例

在本节中，我们将介绍一个简单的核反应模拟程序。这个程序将模拟稳定核子的分裂过程。

```python
import numpy as np

def reaction_rate(cross_section, flux, activation):
    return cross_section * flux * activation

def reaction(cross_sections, fluxes, activations):
    return np.sum([reaction_rate(cross_section, flux, activation) for cross_section, flux, activation in zip(cross_sections, fluxes, activations)])

cross_sections = [0.1, 0.2, 0.3]
fluxes = [10, 20, 30]
activations = [0.5, 0.4, 0.3]

Q = reaction(cross_sections, fluxes, activations)
print("核反应的产量：", Q)
```
