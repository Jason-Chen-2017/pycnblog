```python

    for K in K_list:
        for k in K:
            if k not in K_fusion:
                K_fusion.append(k)
    return K_fusion

K_A = ['k_A1', 'k_A2', 'k_A3']
K_B = ['k_B1', 'k_B2', 'k_B3']
K_C = ['k_C1', 'k_C2', 'k_C3']

K_fusion = fusion([K_A, K_B, K_C])

```

接下来，我们定义了三个节点的知识空间，分别为 $K_A$、$K_B$ 和 $K_C$。然后，我们调用 `fusion` 函数，将这三个节点的知识空间作为输入，得到融合后的知识空间 $K_{fusion}$。
