### 4.1 智能合约的具体代码实例

以下是一个简单的智能合约的代码实例：

```
pragma solidity ^0.5.0;

contract MedicalFinance {
    address public owner;
    uint public fundBalance;

    event FundsTransferred(address from, uint amount);

    constructor() public {
        owner = msg.sender;
        fundBalance = 0;
    }

    function transferFunds(address payable to, uint amount) public {
        require(msg.sender == owner);
        require(amount <= fundBalance);

        fundBalance -= amount;
        to.transfer(amount);

        emit FundsTransferred(msg.sender, amount);
    }
}
```

3. 在构造函数中，我们初始化智能合约的状态。我们设置拥有者为创建智能合约的地址，资金余额为0。
