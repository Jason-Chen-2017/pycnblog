## 4.1 IaC代码实例

我们将使用Ansible作为IaC的实现工具，来定义和配置Hadoop集群。以下是一个简单的Ansible Playbook示例：

```yaml
- hosts: hadoop-nodes
  become: true
  vars:
    hadoop_version: "3.2.0"
  tasks:
    - name: Install Hadoop
      ansible.builtin.package:
        name: "hadoop-${hadoop_version}"
        state: present

    - name: Configure Hadoop
      ansible.builtin.copy:
        src: hadoop-env.sh.j2
        dest: /etc/hadoop/hadoop-env.sh
        owner: hadoop
        group: hadoop
        mode: "0755"
```

在这个示例中，我们使用Ansible定义了一个Hadoop集群的基础设施，并使用变量`hadoop_version`来描述Hadoop的版本。

我们将使用Python作为元素特性的实现工具，来描述和管理Hadoop集群的数据。以下是一个简单的Python示例：

```python
import os

class HadoopCluster:
    def __init__(self, name, nodes):
        self.name = name
        self.nodes = nodes

    def describe(self):
        return {
            "name": self.name,
            "nodes": self.nodes
        }

hadoop_cluster = HadoopCluster("my_hadoop_cluster", ["node1", "node2", "node3"])
```

在这个示例中，我们使用Python类`HadoopCluster`来描述Hadoop集群的数据，并使用属性`name`和`nodes`来描述集群的名称和节点。
