# 2. 核心概念与联系

## 2.1 低代码平台

低代码平台（Low-Code Platform）是一种软件开发平台，它使用图形化的工具和预定义的代码片段来简化软件开发过程，使非专业的用户能够快速地构建软件应用。低代码平台通常包括以下组件：

- 图形化设计器：用于创建用户界面和逻辑流程的工具。
- 数据库管理器：用于管理应用程序的数据和数据源。
- 代码生成器：根据用户的设计自动生成代码。
- 部署和管理工具：用于部署和维护应用程序。

低代码平台的主要优势在于它们降低了软件开发的门槛，使得更多的人可以快速地构建软件应用，从而提高了软件开发的效率和降低了成本。

## 2.2 社区支持与资源

社区支持与资源是低代码平台的重要组成部分，它为用户提供了各种形式的帮助，包括在线论坛、文档、教程、视频、工具和插件等。社区支持与资源的主要优势在于它们提供了实用的信息、经验和建议，帮助用户解决问题、学习新技术和分享经验。
