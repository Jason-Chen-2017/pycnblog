## 1.4 具体代码实例和详细解释说明

### 1.4.1 函数导数的Python实现

在Python中，可以使用NumPy库来计算函数导数。以下是一个简单的示例：

```python
import numpy as np

def func(x):
    return x**2

x = np.array([1, 2, 3, 4, 5])
dx = np.array([0.01, -0.01, 0.01, -0.01, 0.01])

df = (func(x+dx) - func(x)) / dx
print(df)
```

在这个示例中，我们定义了一个简单的函数`func`，然后使用NumPy库计算其导数。可以看到，导数的计算结果与函数的斜率相符。

### 1.4.2 泰勒展开的Python实现

在Python中，可以使用SymPy库来计算泰勒展开。以下是一个简单的示例：

```python
import sympy as sp

x = sp.symbols('x')
f = x**3

taylor_expansion = sp.taylor(f, x, 0, 4)
print(taylor_expansion)
```

在这个示例中，我们使用SymPy库计算函数`f`的泰勒展开。可以看到，泰勒展开的结果与函数的近似值相符。
