# 2. 核心概念与联系

在研究新型抗菌药物时，我们需要了解一些核心概念和联系，包括：

- 抗菌药物：抗菌药物是一种药物，可以杀死或抑制细菌的生长和繁殖。
- 肺炎病毒：肺炎病毒是一种病毒，可以导致肺炎和其他肺部疾病。
- 抗菌药物耐药：抗菌药物耐药是指病毒或细菌对抗菌药物的抵抗力增强，从而使抗菌药物无法有效治疗。
- 新型抗菌药物：新型抗菌药物是一种新型的抗菌药物，可以有效治疗耐药病毒和细菌。
