```python
import time

class TemperatureController:
    def __init__(self, setpoint):
        self.setpoint = setpoint
        self.current_temperature = 0

    def read_temperature(self):
        # 读取传感器数据
        self.current_temperature = 20
        return self.current_temperature

    def control_heater(self, on):
        # 控制加热器
        if on:
            print("Turn on heater")
        else:
            print("Turn off heater")

    def control_loop(self):
        while True:
            current_temperature = self.read_temperature()
            if self.current_temperature < self.setpoint:
                self.control_heater(True)
            elif self.current_temperature > self.setpoint:
                self.control_heater(False)
            time.sleep(60)

if __name__ == "__main__":
    controller = TemperatureController(25)
    controller.control_loop()
```

在这个代码实例中，我们定义了一个`TemperatureController`类，它包括一个`setpoint`属性（设定温度），一个`current_temperature`属性（当前温度），一个`read_temperature`方法（读取传感器数据），一个`control_heater`方法（控制加热器），以及一个`control_loop`方法（控制循环）。

在`control_loop`方法中，我们通过一个无限循环来实现温度控制。我们首先读取传感器数据，然后根据当前温度和设定温度来控制加热器。如果当前温度低于设定温度，我们将开启加热器；如果当前温度高于设定温度，我们将关闭加热器。
