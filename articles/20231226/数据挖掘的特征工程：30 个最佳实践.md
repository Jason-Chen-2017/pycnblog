# 2.核心概念与联系

在进入具体的实践之前，我们需要了解一些核心概念：

1. **特征（Feature）**：特征是数据集中的一个变量，用于描述样本的某个方面。例如，在人口统计数据中，年龄、性别、收入等都可以视为特征。

2. **特征工程（Feature Engineering）**：特征工程是指根据业务需求和数据特点，对原始数据进行预处理、转换、创建新特征等操作，以提高数据挖掘算法的性能和准确性。

3. **特征选择（Feature Selection）**：特征选择是指根据某种评估标准，从原始特征集中选择出一些具有较高相关性的特征，以减少特征的数量并提高算法性能。

4. **特征提取（Feature Extraction）**：特征提取是指通过对原始数据进行某种转换或压缩操作，生成一组新的特征，以捕捉数据中的更多信息。

5. **特征转换（Feature Transformation）**：特征转换是指将原始特征进行某种数学变换，以改善算法性能或使其适应特定算法的需求。

6. **模型评估（Model Evaluation）**：模型评估是指根据某种评估标准，评估数据挖掘算法的性能，以选择最佳的算法或特征组合。
