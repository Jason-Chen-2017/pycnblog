# 2. 背景介绍

全文搜索的核心是能够快速、准确地查找大量文本数据中的关键字或短语。为了实现这一目标，我们需要对文本数据进行索引。索引可以提高搜索速度，但也会增加存储和维护成本。因此，在设计全文搜索系统时，我们需要权衡索引的精度和性能。

ArangoDB 支持文本搜索，但它的实现并不是基于传统的全文搜索引擎，如 Elasticsearch 或 Solr。相反，ArangoDB 使用自己的算法来实现文本搜索。在本文中，我们将详细介绍 ArangoDB 中的文本搜索算法，并提供一些实际的代码示例。
