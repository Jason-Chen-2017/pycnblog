# 2.核心概念与联系

## 2.1 熵定义与性质

熵是信息论中的一个重要概念，它用于衡量一个随机变量的不确定性。熵的定义为：

$$
H(X)=-\sum_{x\in X} P(x) \log P(x)
$$

其中，$X$ 是一个有限随机变量，$P(x)$ 是$x$ 的概率。

熵的性质如下：

1. 非负性：对于任何随机变量，其熵不小于0。
2. 极大化性：对于一个确定的随机变量，其熵为0；对于一个均匀分布的随机变量，其熵为最大。
3. 增加性：对于两个独立的随机变量，其熵是独立的。

## 2.2 熵与网络安全

熵与网络安全之间的关系主要表现在以下几个方面：

1. 密码学中，熵用于衡量密码强度。密码的熵越高，表示密码的不确定性越大，安全性越高。
2. 信息隐私保护中，熵用于衡量隐私保护的水平。隐私的熵越高，表示隐私信息的不确定性越大，保护水平越高。
3. 网络安全中，熵用于衡量系统的不确定性。系统的熵越高，表示系统的不确定性越大，安全性越低。

def entropy(probabilities):
    return -sum(p * math.log2(p) for p in probabilities if p > 0)

# 密码强度评估
password = "Aa1!"
characters = [c for c in password]

password_entropy = entropy(probabilities)

print("System entropy:", system_entropy)
