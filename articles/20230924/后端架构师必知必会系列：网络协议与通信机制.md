# 1.简介
  

## 一、课程介绍
“后端架构师”这个岗位的名称，听起来很高大上，其实真不那么高大上。你只要从事网络编程相关工作，就能够胜任这个职位，而不需要担心它的吉庆，因为你已经在不断地学习，不断地进步中。

因此，如果你想成为一个合格的后端架构师，了解网络协议和通信机制，尤其是HTTP/HTTPS和TCP/IP协议栈，那你需要阅读本专栏的内容。在本篇教程中，将详细介绍网络协议及其在应用程序中的作用，并提出一些后端开发者应当具备的基础知识。文章主要包括如下几个方面：

1. 了解网络模型、协议和端口号的区别；
2. 理解socket的原理和使用方法；
3. 分析应用层协议如HTTP/HTTPS等实现原理；
4. 详细介绍HTTP协议各版本的特性；
5. TCP/IP协议的体系结构，以及常用的传输控制协议（TCP）和互联网协议（IP）。
