# 1.简介
  

云计算是一种新兴的计算模式，它利用分布式计算资源、存储空间、网络带宽、服务器等的资源池进行快速部署和自动化运营。通过将核心业务应用部署在云上，可以降低成本、提高效率、节约IT资源。同时，也会减少本地硬件投入，满足多变的业务需求。基于公有云、私有云或混合云（公有云与私有云相互连接）构建的系统架构主要包括以下四个层次：
- 基础设施层：该层提供计算资源、存储资源、网络资源、安全保障、服务管理等基础服务。
- 服务层：该层向用户提供完整的业务解决方案，包括业务计算、数据存储、网络通信、系统集成、用户接口等功能模块。
- 平台层：该层作为云服务供应商的服务控制台，负责产品的开发、测试、上线、运维和售卖等全生命周期管理工作。
- 应用层：该层实现用户实际需要的业务功能，采用云服务或第三方服务的形式，可按需购买，降低了IT支出及成本。
目前，国内外很多公司都已经建立起自己的云平台，并且推出了各种各样的云产品。云计算的初衷是为了更有效、更便宜地使用 IT 资源，并帮助企业实现业务快速迭代，创造更多价值。
# 2.云计算基本概念术语说明
## （一）云计算的定义、特征与优势
云计算（Cloud Computing）的英文名称为“The Internet as a Service”，它是一种新的计算模型，允许用户不受限制地访问、使用计算机系统、存储设备和网络等计算资源。2006年美国科技期刊WIRED发表了一篇名为“The Age of the Cloud”的文章，阐述了云计算的目标、定义、特性及优势。其中，云计算的定义如下：
> Cloud computing is a model for enabling convenient, on-demand access to a shared pool of configurable computing resources (e.g., networks, servers, storage), without ownership, management, and control.
云计算通过共享的、可配置的计算资源池，为用户提供简单、按需的网络服务，并避免了对硬件设施的物理访问权限和日益增长的维护费用。
其特性主要体现在以下几个方面：
### 1.弹性可伸缩
云计算为用户提供了高度弹性的计算能力，用户只需要按照实际需要付费，而不需要事先计划好所有资源量。当某一个项目运行出现性能瓶颈时，可以通过扩容增加计算资源，进而满足更多用户的需求。此外，云计算还支持按需付费，即按实际使用的计算能力收费，无需一次性付费就能使用大量的计算资源。
### 2.按使用付费
云计算按使用的方式支付费用，而不是一次性预付费用，这使得用户只需要按照实际使用量付费，并且不必担心资源闲置或浪费的问题。另外，云计算也具有按使用时间计费的优势，用户只需要为实际使用时间支付费用即可。
### 3.灵活组合
云计算允许用户根据自己的业务需要，快速组建不同类型的计算资源池，以满足各种不同的应用场景需求。用户只需要按需购买所需的计算资源，再通过标准的接口或SDK调用，就可以获得所需的计算资源。因此，云计算赋予了用户高度灵活的组合能力。
### 4.可扩展性
云计算的可扩展性高，用户能够轻松增加或减少计算资源量，而不影响其他用户的服务。因此，云计算可以很好的满足大型公司或政府部门的海量计算需求。
### 5.服务广泛
云计算已经成为各行各业中的流行词汇，遍及银行、电信、零售、医疗、科研、教育、游戏、媒体、金融等领域。而且，随着互联网的飞速发展，越来越多的人开始使用手机、平板电脑、智能手机等移动终端，越来越多的个人和小型企业开始迁移到云端。因此，云计算的广泛应用及普及程度正在逐渐加快。
## （二）云计算相关术语
在学习云计算之前，首先要了解一些相关的术语和概念。这些术语和概念的含义将对后面的学习有所帮助。
### 1.云计算模型
云计算的模型有多种，如公有云、私有云、混合云、社区云、私有云混合云等。其中，公有云指的是由一家实体通过公开的网络连接提供的云计算服务，比如亚马逊AWS、微软Azure等。私有云是指一家实体拥有自己的服务器集群和存储系统，并且通过内部的网络连接提供计算服务。混合云是指结合了公有云和私有云的一种云计算模型，其中私有云又可以细分为私有云计算、私有云存储、私有云网络等，私有云混合云则是私有云的组合。社区云是由云服务提供商通过自己的社交网络形成的云计算平台。
### 2.虚拟机
虚拟机（Virtual Machine，VM）是一个软件模拟的物理硬件。它通常被用来创建和运行多个操作系统实例。每个VM都有自己独立的操作系统、应用程序、文件系统和网络接口，并且可以在任意数量的PC主机上运行。VM提供了比传统物理硬件更高的计算性能、可靠性和可用性。
### 3.虚拟化技术
虚拟化技术是云计算的一个重要组成部分。它使用户能够在实际使用中获得接近于真实环境的效果。虚拟化技术可以将物理硬件资源抽象成逻辑资源，并将它们映射到一系列虚拟机上，实现虚拟机之间的快速切换。当前的虚拟化技术有Xen、KVM、Docker、OpenStack等。
### 4.云计算框架
云计算框架是一个软件开发包，用于构建、管理和部署云计算平台。它包括计算、网络、存储、安全、监控、运维、计费、报警、规则引擎、流量控制等相关组件。目前，最主流的云计算框架有OpenStack、Kubernetes、Mesos等。
### 5.云计算中心
云计算中心（Cloud Center）是云计算的一个主要组件，它负责云资源的分配、整合并调度。云计算中心一般由一系列的资源池组成，如服务器池、存储池、网络池等。云计算中心还提供相应的服务，如计算、存储、网络等，包括公有云服务、私有云服务、混合云服务等。
### 6.云计算平台
云计算平台（Cloud Platform）是云计算的一项重要特征，它是云计算的软件、硬件和服务三者的集合。云计算平台一般包括云控制器、云管控器、云编排器、云调度器、云存储、云网络、云安全、云监控、云备份等多个组件。云计算平台是一个统一的系统，包含了云计算的所有功能。
### 7.云计算商店
云计算商店（Cloud Store）是一个云计算平台，它提供了云计算相关产品、服务的销售和购买，让用户能够按需购买所需的云服务。目前，云计算商店上有非常多的云服务商，涉及各种类型的云服务，比如公有云、私有云、容器云、托管云、Saas云等。
### 8.云计算市场
云计算市场（Cloud Market）是云计算的一个重要平台，它提供了云服务的评估、选择、购买等流程。云计算市场是云服务提供商进行宣传、广告宣传、展示产品和服务的地方。云计算市metry上还有各种促销活动，例如免费试用、折扣、返点等。
