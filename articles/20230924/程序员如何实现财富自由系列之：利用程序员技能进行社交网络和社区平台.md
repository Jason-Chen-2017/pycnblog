# 3.基本概念术语说明
## 1.计算机网络
计算机网络（英语：computer network）是一个广义的计算机科学领域术语，通常指的是将多个网络设备连接起来的计算机系统。它由节点和链路组成，节点可分为主机、路由器、交换机等不同类型。通过通信链路，能够实现不同节点之间的数据交换、计算处理等。其中，路由器起到路由作用，负责把收到的分组转发到相应的网络接口。计算机网络中的协议是指控制计算机网络中所有节点之间信息传输的规则和机制。目前，计算机网络技术已经成为信息技术发展的一个重要分支，并得到广泛应用。常用的计算机网络协议有TCP/IP协议、IPv6协议等。

## 2.HTTP协议
超文本传输协议（Hypertext Transfer Protocol， HTTP）是一种用于从WWW服务器或其他网站服务器传输超文本文档到本地浏览器的协议。简单的说，HTTP协议是Web通信的基石，通过HTTP，浏览器可以请求访问Web页面，从而浏览网站的内容。由于HTTP协议简单易用，使得它被广泛使用。HTTP协议包括请求报文和响应报文，在请求报文中，客户端发送请求信息，如请求行、请求头部、空行和请求数据。在响应报文中，服务器返回请求的资源，如状态码、响应头部、空行和响应实体。常用的HTTP方法有GET、POST、PUT、DELETE等。

## 3.TCP/IP协议
TCP/IP协议族是互联网上使用的传输层协议，它涵盖了Internet的主要协议。TCP/IP协议族提供的服务主要有三个：1、域名系统（DNS）：它是因特网的一套命名系统，用于将域名转换为IP地址。2、网际协议（IP）：它是网络互连的基础，负责传递数据包。3、传输控制协议（TCP）：它是一种面向连接的、可靠的、基于字节流的传输层协议，用于在两个计算机之间建立可靠的连接，提供传输服务。TCP协议提供可靠的、按序到达的数据流，并且通过校验、丢弃重复数据、超时重传、拥塞控制等方式确保数据完整性。4、用户数据报协议（UDP）：它是一种无连接的协议，提供不可靠的传输服务，即不保证数据传输的可靠性，适用于对可靠性要求不高的场合。许多高级的应用程序都采用了TCP/IP协议族，例如WWW、邮件、文件传输、远程登录等。

## 4.Socket通信模型
Socket（套接字）也是计算机网络通讯的一个抽象概念。它是一组接口函数的集合，用来执行网络程序间的通信过程。其通信机制是基于标准的BSD Socket API，由两端的进程间的通信 socketpair() 创建，传输可靠的消息，并不会受中间节点影响。Socket可以用来构建基于不同协议的应用层网络程序，比如TCP/IP、UDP、HTTP等。Socket通信模型由四个要素构成：1、一组地址端口号：标识唯一网络接口；2、通信模式：描述传输类型，数据报还是连接模式；3、传输方式：描述消息是否可靠；4、效率：描述通信的吞吐量、延迟和容量。

## 5.Web开发语言
Web开发语言一般指脚本语言、标记语言和编程范型，用于实现动态网站的开发。它们共同构成了网站开发的三大阵营：JavaScript、Java、PHP。其中，JavaScript 是当今最流行的脚本语言，广泛应用于网站开发中。PHP 是基于服务器端脚本语言，是一种开源的通用计算机脚本语言。它是一种支持各种功能的脚本语言，可以嵌入到HTML中，与服务器端代码共同运行。Java 因为性能高、安全可靠等优点逐渐占据市场主流。

## 6.常用数据结构与算法
- 数据结构
  - 数组
    - 描述：数组是一组相同类型元素的顺序集合。
    - 操作：添加、删除元素、查找元素、修改元素。
    - 示例：[1, 2, 3, 4, 5]、[“apple”, “banana”, “orange”]、[true, false, true, false]。
  - 链表
    - 描述：链表是物理存储单元上非连续、非顺序的方式存储数据元素，每个元素只存放数据值和指向下一个元素的指针，没有其他元素的信息。
    - 操作：添加、删除元素、查找元素。
    - 示例：单向链表、双向链表、循环链表。
  - 栈
    - 描述：栈是一种特殊的线性表，只允许在一端插入和删除数据元素，另一端只能读取数据元素。栈先进后出（LIFO），也就是栈顶元素优先出栈。
    - 操作：压栈（push）、弹栈（pop）。
    - 示例：括号匹配、进制转换。
  - 队列
    - 描述：队列是一种特殊的线性表，只允许在一端插入元素，另一端删除元素，先进先出（FIFO）的特点。
    - 操作：入队（enqueue）、出队（dequeue）。
    - 示例：打印机任务队列、TCP协议传输队列。
  - 散列表
    - 描述：散列表是一种键值对集合，它根据关键字的值直接进行访问。
    - 操作：添加、删除元素、查找元素。
    - 示例：存储数字和字符串、表示图片的哈希值。
  - 树
    - 描述：树是一种抽象数据类型，它由n(n>=1)个有限结点组成一个具有层次关系的集合。树可以有零个结点，即树为空，也可以有多个结点，即树为不空。
    - 操作：遍历树、求树的高度、判断是否是满二叉树、判断是否是完全二叉树。
    - 示例：二叉树、森林。
  - 堆
    - 描述：堆是一种特殊的树，它是一个完全二叉树或者是近似完全二叉树，并且除了最底层外，其他各层上的各节点都是一个最大堆（堆序最大的元素排在前面）。
    - 操作：插入元素、删除元素、取最大元素、调整堆。
    - 示例：优先级队列、堆排序。
  - 图
    - 描述：图是由边和顶点组成的结构，它描述了一组对象的相互联系和关系，图可以表示为节点的集合以及在图中存在的边的集合。
    - 操作：创建图、添加边、删除边、路径查找、最小生成树。
    - 示例：网络、社交网络、推荐系统。
- 算法
  - 搜索算法
    - 描述：搜索算法（又称图论算法）是指从指定节点（初始节点、目标节点、起始状态、终止状态等）开始，在图或图形结构中通过特定路径发现特定目标的计算过程。
    - 操作：深度优先搜索、广度优先搜索、A*算法。
    - 示例：DFS、BFS、A*。
  - 分治算法
    - 描述：分治法，也叫分而治之，指将一个复杂的问题分成两个或更多的相同或相似的子问题，递归解决这些子问题，然后再合并其结果，得出原问题的解。
    - 操作：快速排序、归并排序、快速幂运算。
    - 示例：快速排序、归并排序、矩阵乘法。
  - 排序算法
    - 描述：排序算法是一种对记录的重新排序的方法，目的是将一串记录按照一定顺序排列成一个序列。
    - 操作：选择排序、冒泡排序、插入排序、希尔排序、归并排序、快速排序、桶排序、计数排序、基数排序。
    - 示例：选择排序、冒泡排序、插入排序、希尔排序、归并排序、快速排序、桶排序、计数排序、基数排序。
  - 回溯算法
    - 描述：回溯算法（又称枚举搜索）是一种用于确定给定状态空间的所有可能路径的搜索算法。
    - 操作：N皇后、八皇后、跳跃游戏、九宫格。
    - 示例：N皇后、八皇后、跳跃游戏、九宫格。
