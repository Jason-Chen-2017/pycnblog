# 1.简介
  

知识管理是企业最基础也是重要的环节之一。企业要想在竞争中脱颖而出、独立创造价值，首先就需要掌握企业中的知识资源并有效地进行处理、利用和传播。然而，如何系统化整理企业的知识资源、分配合适的知识赋权，建立起健全的信息共享机制成为企业知识管理的难点和核心。
由于知识是企业生存的根本，也因此知识管理是企业的生命线。它不仅对公司的发展、职业发展、产品及服务提供指导作用，同时也影响到企业对外形象、品牌、文化的塑造，提升企业竞争能力和经济效益。如何运用信息技术、制定信息政策、主动倾听市场需求、及时反馈信息并做好售前支持等方面，都离不开一流的知识管理能力。
因此，结构化思考和金字塔结构，是信息管理领域中的两个基本模式。结构化思维的关键是按照不同层次、不同视角、不同形式将各种数据组织成一个个模块或数据库，并能够有效地进行检索、分类、归档、描述、分析、总结、分享和保存。信息资产的分级管理就是基于这种思路的典型应用。
金字塔的最上端为元数据层，主要保存数据的结构、属性、含义等相关信息；下层为特征层，保存数据的实际值、编码方式、缺失率等相关信息；中间层为事实层，保存真实世界的数据、事件和意向等信息；最底层为观念层，主要保存知识论述、经验和规则等文档。通过这种层次化的思维，可以很好地实现信息的收集、整理、存储、分析、传递和销毁。而知识管理，则是在这样一个金字塔结构的基础上，构建起完整的知识体系。
该专题的文章，试图从信息管理和知识管理两个方面，为读者呈现一种全新的思维方法——结构化思考和金字塔结构。所涉及的内容包括如下几个方面：

1) 背景介绍
信息管理与知识管理的定义及其区别。

2) 基本概念术语说明
信息，信息系统，知识，知识系统，元数据，特征数据，事实数据，观念数据，信息资产，知识资产，元数据层，特征层，事实层，观念层，信息资产管理，知识资产管理，信息资产生命周期，知识资产生命周期。

3) 核心算法原理和具体操作步骤以及数学公式讲解
信息系统建模的方法。

4) 具体代码实例和解释说明
