# 1.简介
  

最近几年，芯片行业迎来了蓬勃发展时期，各路厂商纷纷推出了高性能芯片、智能手机等产品。无论是做AI系统还是处理器等硬件，都离不开芯片的制造与设计。但如何合理地布局电子材料和物质，才能确保芯片的成熟稳定运行，成为提升产业链效率的关键环节。近年来随着工艺技术的飞速发展，基于化学反应堆的新型超导材料已经出现，但在实际应用中，由于结构复杂、耗能高、耐磨性差、控制难度大等缺点，仍处于理论建模阶段。本文将从化学气相沉积（Chemical Vapor Deposition，CVD）及其发展过程，热量传递及其影响因素，逐步阐述相关知识，并展示在APS-CM（Advanced Photon Source Construction Manual）标准下基于膜镀锌的高效无缝衔接电路板制作方法。

# 2.化学气相沉积简介
在化学气相沉积（Chemical vapor deposition，简称CVD），是一种用光刻胶或者其他材料涂层进行的光刻化工过程。主要用于加工复杂形状、丰富颜色的高分子材料，如晶体、石英、金属、化合物、复合材料等。它在表面覆盖有液态金属或其他材料的情况下进行，利用低温、高压下的液态气流进行化学键、层岩分离，释放由配位有机物生成的气体，从而制成高分子层块。一般来说，从晶体到石墨烯至单层芯片所需的尺寸都可以用化学气相沉积技术制成。

在CVD过程当中，首先需要选择适合材料的液态气，一般采用硫酸钾、氢氧化钠或氮气作为气源。然后对材料进行表面清洗、掏空以及用纱布包裹起来，使之能够吸附气体，保持湿润状态。在保证表面稳定后，就可以在上述物料上施加不同厚度的层岩材料，使其获得足够的液态气蒸腾作用。层岩材料的厚度由不同的研究人员确定，通常为100~500um，一般在500um以下就可以完全地融化整个液体，达到良好的化学键分离及层分离能力。在层岩处结合碘纳或者酵母酶，经过一定时间即可产生高分子晶体架，这就是完成的化学气相沉积过程。

如下图所示，原始材料经过化学气相沉积过程，可以得到一层层岩分离后的晶体结构，每一层都具有特定功能属性，如导电、导热等。每一层岩都会释放出由配位有机物生成的气体，根据气体的成分、大小、功能特性以及接触面积大小，我们可以利用不同的晶体管电路进行制作，以实现所需功能的实现。


如上图，原材料经过氢气充填和对表面进行层岩分离，然后分别对晶体上的不同部分进行低溢射电极照射，形成两个电极之间的空间隔离。再使用研磨电极将晶体与金属组织隔离，然后利用双向接头电极连接，实现高层级嵌套晶体功能。这样就完成了一个电路板的制作流程。

# 3.技术特点
- 可重复性：采用不同的层岩材料、电极类型、厚度及层数，都可得出同样大小、功能的晶体管电路。
- 耗能低：无需复杂的加工工艺，只需简单的一二次方程即可制作。
- 耐磨性强：化学氢气的低温和高压，使得材料能够完整地融化，保证了晶体的耐磨性。
- 低成本：不需要复杂的化学设备，仅使用电动工具即可完成。

# 4.热量传递及其影响因素
除了电极隔离、电气连接等电路部件外，还存在着一个比较微观的物理过程——热量传递。其中最重要的原因就是化学气相沉积所释放出的热能，它被用来对材料的结构进行微调，使之适应功能需求，进而实现高效、精准的功能。热量的传递会引起材料的膨胀、变形、缩短长度、增加高度、产生变化的厚度，同时也会改变材料的摩擦力和粘结性。因此，在设计功能复杂的大型晶圆阵列电路板时，需要考虑到物理、化学和生物学的相互作用，以尽可能地减小热量传递对结构性能的影响。

一般来说，在CVD过程之后，可以有两种类型的热能传递发生。第一种是固体层间的热传递，主要是由气体在氢气作用下产生的热能通过固体层之间的挤压、聚集、流动所引起。在低电压下，热能容易集中，此时材料表面的凝胶层即受到较大的压力，并且层间产生的热浪效果很好；在高电压下，由于材料表面的气孔过窄，热浪影响并不明显，所以在许多晶圆电路板制造过程中都采用这种方式进行热力传递。

第二种是液体层间的热传递，它主要是由于氢气和水混合而产生的热能传输过程。如果在热交换设备上增设热流来传输热能，则可提升电路板的电容性能和过载能力。然而，这种热传导的方法要求安装跨越多个层的交叉电路，且当交叉层之间的宽度超过100nm时，便会引入较大噪声。另外，在低电压下，由于氢气和水的气相，使得材料内部的密度大于表面密度，热流只能通过一定范围的区域，但是在高电压下，由于水分子的光滑性，使得热流分布更广泛，其效率要高于固体层间热传递。

为了尽可能降低热量传递对晶圆电路板的性能影响，我们需要对液体层间的热传递采用膜镀锌的方法。这种方法利用高层级膜、镀锌、铁电极等来制成一个金属环保剂，以固化在特定位置上的液体，从而屏蔽层间的热传递带来的影响。膜镀锌的方法虽然需要高温、高压下专门的制备设备，但是它的成本非常低廉，而且在膜的选择上也有一些规律可循。

# 5.膜镀锌法与高效无缝衔接电路板制作
膜镀锌法为一种可重复性高、耗能低、耐磨性强、低成本、无缝衔接的晶圆电路板制作方法。它的基本过程是把一个材料上精心布局的锌粒子，通过施加压力来固定在材料的层边缘，然后在其上层盖上低温、高压的氢气，这样就会形成一种介质，这就可以将不同的材料堆叠在一起。通过膜镀锌法，可以制作出一系列的功能完整的晶圆电路板。

例如，一个膜镀锌法的衔接电路板的布局如图所示。


1. 第一层（P）：电极的定位和膜的掺杂，可以加快衔接速度。
2. 第二层（N-PT）：由于氢气的低压和高温，可以降低衔接热量，使得低电压下能够直接进行接连。
3. 第三层（SiO2-Ni）：镀锌电极与锡箔电极的混合，使得电流密度更高。
4. 第四层（Ni-B）：B层与Ni层的相互靠近，可以提高电流。
5. 第五层（PT-Au）：由于Au层的厚度较薄，可以降低电流损失。
6. 第六层（Pb）：最后一层是过度膜镀锌法的结束，该层的材料中含有许多微粒子，可以促进电流的流动。

这种结构特别适合于用于制作高效、精准的晶圆电路板。由于晶圆电路板通常具有多个入口、出口，膜镀锌法可以有效地实现无缝衔接。而且，膜镀锌法的制作方法不需要任何专门的电路设计技能，只需要简单的化学修饰和掌握一些高温、高压的气候条件，就可以轻松地完成设计和制作。
