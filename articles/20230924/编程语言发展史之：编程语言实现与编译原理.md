# 1.简介
  

## 一、编程语言概述
计算机程序设计语言（Programming Language）是一种用来控制电子计算机或其上的机器进行高速数据处理的编程语言，是人们为了解决自然界和现实世界中复杂问题而创造的一种新工具。
计算机程序的结构由语句组成，一条语句可以是一个赋值语句，一个条件判断语句或者循环语句。语句组合在一起构成了完整的程序。计算机程序的目标就是给出输入，经过各种运算得到输出，最终达到指定效果。
程序语言是程序设计者与计算机之间交流的语言，它定义了程序的语法结构和语义规则，并提供了对各种数据类型、表达式、变量等的支持。
编程语言是人类为了解决日益复杂的计算任务，建立更强大的计算模型所制定的工具箱。一般来说，编程语言分为：解释型语言和编译型语言。解释型语言是在运行时将源代码解释为机器指令；而编译型语言则是在源码编译期间，将源代码转换为机器指令。目前主流的编程语言包括C、Java、Python、JavaScript、Go、Swift、Rust等。
## 二、编程语言分类
### （一）命令式语言
命令式语言（Imperative language），又称声明式语言，指的是使用语句来指导计算机完成各种任务的语言。命令式语言采用的是命令/赋值模型，即通过改变状态或变量的值，来实现某种功能。它的特征是提供了诸如赋值、条件判断、循环、函数调用等执行流程控制语句。
代表性命令式语言有SQL、MATLAB、Lisp等。
### （二）函数式语言
函数式语言（Functional language），指的是用函数的方式来编程。函数式编程通过数学上的函数抽象机制，避免使用变量来存储状态，进而提高代码的可读性和易维护性。函数式语言的特点是没有副作用，函数的返回值只用于计算结果。在函数式语言中，所有状态都被显式地编码为参数，通过函数参数传递。
代表性函数式语言有Haskell、ML、Erlang等。
### （三）逻辑式语言
逻辑式语言（Logical language），又称集合语言，是一种基于集合论的语言。它在逻辑上推导出表达式之间的关系，从而使得复杂的计算过程可以被形式化地描述。
逻辑式语言最早由美国卡尔·皮查伊（Kurt Pike）于20世纪70年代发明。代表性逻辑式语言有Prolog、Mercury、Oz等。
### （四）面向对象语言
面向对象语言（Object-oriented programming language），又称面向对象编程语言，是一种通过类和对象的方式来编程的编程语言。它强调对象之间的通信和协作，鼓励代码重用和模块化，提升代码的灵活性和扩展能力。
代表性面向对象语言有C++、Java、Smalltalk等。
### （五）事件驱动语言
事件驱动编程语言（Event-driven programming language），也称异步编程语言，指的是程序中的各个组件之间交互少量消息的方式来驱动程序执行。它不依赖顺序执行的单线程模型，提升程序的响应速度。
代表性事件驱动语言有CSP、Erlang、Clojure等。
### （六）并行编程语言
并行编程语言（Parallel programming language），是指多核、分布式系统及高性能计算领域中的编程语言。这些语言的出现旨在提升计算机硬件性能，改善并发、分布式、高性能计算方面的应用。
代表性并行编程语言有OpenMP、MPI、CUDA等。
