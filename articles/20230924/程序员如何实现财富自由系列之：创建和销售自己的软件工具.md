# 1.简介
  

程序员总是被称作“码农”，但其实，任何从事计算机编程工作的人都可以被称为“程序员”。近几年，随着互联网行业的蓬勃发展，越来越多的程序员开始关注并创造出属于自己的产品或服务。通过软件开发、产品设计、网站制作、游戏开发等方式，程序员们已经在不断创造新的价值，帮助更多的人解决实际问题。

然而，要想真正掌握程序员所谓的“财富自由”，需要更加锻炼的能力和知识。程序员面临的最大挑战就是如何摆脱困境，实现自我价值的发现和创造。因此，如何让自己变得更加独立、强大，成为一个优秀的程序员，是一个值得研究的问题。

为了帮助读者更好地理解程序员如何实现财富自由，笔者将介绍程序员应该具备哪些基本素质，以及如何建立自己的技能树，打造自己的高收入创业平台。这套系列的文章将逐渐更新，力争为程序员实现财富自由提供一线指导。

# 2.核心概念
## 2.1 为什么需要“财富自由”？
个人认为，“财富自由”这个词汇是由托马斯·伯林创造的。他在其名著《社会契约论》中提出，“财富”不是商品或者劳动果实，而是一种基于关系的、权利与义务的东西。他认为，现代经济主要依赖的是交易，而市场上的每个参与者都必须遵守价值观、意志与道德约束，“人们的努力仅仅是让他们达到某种目的，但同时也正在为这种目的创造价值”。

基于这个认识，“财富自由”的关键在于，如何摆脱市场经济对人的束缚，实现自己的价值创造。伯林认为，个人拥有的财产，在很大程度上取决于个人选择和努力。正如慈禧太后去征服西方，但是她没有坐飞机的权限一样。

程序员面临的最大问题便是如何摆脱市场经济的枷锁，实现自我的价值发现和创造。程序员从事编程工作，其核心任务就是编写程序。无论是企业还是个人，为了赚钱，就必须有能力和资源创造价值，否则根本就无法生存。

## 2.2 为什么需要“技能树”？
为了实现“财富自由”，程序员首先要建立自己的技能树。

“技能树”一词最早由大卫·麦克唐纳引入，指的是计算机科学领域的一类树状结构——“技能的层次体系”。它显示了程序员具备的不同技能的等级关系，以及各项技能之间的相互依赖性。

每一个层级代表着一种抽象概念，比如“编程语言”、“数据结构和算法”、“软件工程”等。每个概念又细分为具体的技能，比如“C语言”、“SQL”、“Linux系统”等。这种层次化的结构可以帮助程序员快速学习新技术和掌握相应的技能。

## 2.3 “建立自己的高收入创业平台”的定义
通过建立自己的技能树，程序员可以利用自己的专长和经验，帮助更多的人实现财富自由。但是，真正的财富自由还需要很多条件，包括：

1. 有足够的能力去创造自己的价值；
2. 有能力购买和分享高质量的技术资源；
3. 拥有一个稳定的工作岗位和薪水。

如果程序员能够把这些条件付诸实践，建立起自己的高收入创业平台，那么他就可以在市场上获得成功。

笔者希望借用教育经济学的说法，用“建立自己的高收入创业平台”这句话来描述程序员寻找个人成功的道路。如今，创业是一个风靡一时的词汇，很多程序员却认为自己无法独当一面，只能从事职业中低层次的工作。

这是因为，“初级”程序员往往具有较好的编程基础，对计算机科学理论比较了解，但是却缺乏精益求精的能力，无法从事复杂的项目开发。“高级”程序员通常擅长面向对象编程、Web开发、数据库设计、算法优化等领域，但缺乏商业思维、市场营销技巧等非计算机专业相关的能力。

而程序员成功的关键不在于自己是否“高级”，而在于他们能够凭借自己的专业能力，打造自己的高收入创业平台。
