# 2.投资加密货币概述和相关术语
## 加密货币的产生
加密货币（cryptocurrency）是一种基于密码学和电子技术的数字货币系统，通过互联网计算机网络进行交易。它是一种完全去中心化的虚拟货币，用户可以拥有它而不需要依赖实体银行或支付机构的运营。这种数字货币可以解决传统货币体系存在的一些问题。

1990年，比特币诞生。比特币是第一个实现加密货币概念的全球性项目，其独创性在于通过分布式哈希表（blockchain）来记录所有交易信息，确保交易过程的真实性和透明性。

2008年，中本聪发现了SHA-256哈希函数，并创造性提出了一个新的加密货币协议——“工作量证明（proof of work）”机制，使得比特币区块链具备难题求解能力。

2013年，维基解密发布了一个重要消息——中本聪获得了比特币首枚硬币的记账权，此后该币值持续走高。

2017年，加密货币市值已超过15万亿美元，而其规模也远超过实体商品和金融机构。截至2021年7月，整个市场上加密货币的总计数量已经超过1.7万亿美元，占比达到60%左右。

2021年7月，美国纽交所预估，全球加密货币市场规模将达到4000亿美元，目前排名第一的莱特币正在快速扩张。

加密货币是一个新兴领域，它吸引着世界各地的投资者、程序员、学生、研究者、律师等，从而推动了这项技术的进步与革命。

## 加密货币的类型
加密货币分为两大类：

1. 消费类加密货币（如比特币、莱特币等）：消费类加密货币允许个人或者组织直接消费，并能够防止双重支付。

2. 服务类加密货币（如以太坊、EOS、NEM等）：服务类加密货币为个人和组织提供数字服务，比如为网页浏览提供加密连接，提供数字身份认证服务，或提供云计算、物联网、供应链等服务。

# 3.投资加密货币策略原理和实践方法
## 投资加密货币的目的和意义
加密货币是一种新型的数字货币，它具有很多特性促进了它的流通和使用，并且容易被黑客攻击、交易，因此成为大众的安全感和热爱。但是，由于加密货币涉及巨大的风险，一般人很难投资，只能依靠专业的量化投资者才能收益。同时，加密货币市场上没有做空的操作，因此加密货币被认为是一种高风险的投资品种。

## 投资加密货币的策略
### 预测市场趋势
1. 预测市场：加密货币市场前景如何？

2. 反映现实情况：分析加密货币市场的最新交易信息、报价变化、政策变化，推断市场的方向。

### 获取足够的资金
1. 提供充足的初始资金：保证金、定期存款。

2. 找到合适的投资标的：选择特定的加密货币作为投资标的，比如比特币、莱特币、以太坊等。

### 控制风险
1. 避免高波动率投资：防范常见的“五日无量跌停”，做好买卖的心理准备，合理安排投资时间和金额。

2. 进行风险管理：了解加密货币投资的规律和原理，善用账户余额，正确分配资金，节制仓位。

### 执行策略
1. 确定交易时间：精准预测市场的方向和波动率，选定固定的交易时间。

2. 进行交易：使用智能交易机器人进行自动化交易，判断出入市的交易信号，根据信号执行交易操作。

## 投资加密货币的策略优劣
### 优点：

1. 价格便宜：相对于实体货币，加密货币的价格远远低于实体货币，并且数量较少。

2. 不受制于实体银行：加密货币的使用不会受到实体银行的限制，因此可以让普通消费者更加自由地购买加密货币。

3. 有利可图：加密货币市场一直处于蓬勃发展阶段，随时可能出现大幅增长。在充分预测市场的情况下，可以轻松获利。

4. 安全可靠：由于采用分布式存储的技术，加密货币交易过程相对简单，可以保证数据的完整性，不会受到任何形式的恶意攻击。

### 缺点：

1. 投资门槛较高：相对实体货币来说，加密货币的交易量、使用门槛都比较高。

2. 监管成本高：加密货币市场的法律环境不好，很多国家和地区禁止实体投资加密货币。

3. 数据安全性差：加密货币市场的数据安全性较差，很多交易信息可能被黑客入侵、泄露。

4. 投资周期长：加密货币市场仍然处于刚需期，因此投资周期较长。

# 4.Python量化交易平台搭建
## QUANTAXIS量化框架简介
QUANTAXIS量化框架由两部分组成：

1. QAData：负责数据获取模块，主要用于获取历史数据和实时数据；

2. QAMarket：负责数据清洗、标准化、存储模块，主要用于整理和保存数据。

QUANTAXIS量化框架可以满足程序员快速开发量化交易平台的需求，帮助企业对外开放自己的量化交易接口。

## 安装QUANTAXIS量化框架

首先需要安装anaconda，python3.x版本，jupyter notebook，最后通过命令行安装quantaxis。
```
conda install anaconda -y #安装anaconda python3.x
pip install jupyterlab
pip install quantaxis
```
然后运行jupyter notebook打开notebook，在第二个cell中输入以下代码，查看是否安装成功：
```
import QUANTAXIS as QA

```
