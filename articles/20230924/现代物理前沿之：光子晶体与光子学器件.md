# 1.简介
  

## 1.1 概述
“光子”这一词汇是最早被发现并在1900年左右由爱因斯坦提出来的，其发明人海森堡·约翰逊曾经说过："存在一个信念，即电磁波是自然界里某种离散的能量传输媒介。"然而随着二十世纪末期的科技革命的到来，光子这个概念已经从简单的波动中升温到可以用来干预、制造和控制整个人类社会的一股重要能量。但是光子本身究竟是什么？它有哪些特性？又该如何利用？
## 1.2 发展脉络
光子的探索始于费米的微观世界，也就是狭义相对论的一个小时。他的目的是通过理论模型将宏观物质的运动规律揭示出来。19世纪末，康奈尔·约翰·爱因斯坦创立了热力学。当时物理学家们普遍认为，原子的运动是无序的，只是静止不动，除非加上外界的诱惑才会发生物体的移动。因此，为了找寻真正的宏观世界，爱因斯坦建立了狭义相对论。狭义相对论是根据量子力学的基本假设建立起来的，认为空间和时间都是量子。因此，物体静止不动的时候，处于某一时刻的两个相互垂直的原子之间的相对运动是不能用时空穆勒（时空上的守恒定律）来描述的，只能用一个希尔伯特空间来描绘，但对于一个物质来说，这种希尔伯特空间所包含的信息量却比相互垂直的原子还要多。爱因斯坦的工作让后人得出结论，宇宙是由物质组成的，而在宇宙中，任何运动都可以看做是由粒子对称性决定的。
到了第二个世纪，当时的物理学家们仍然认为宇宙是由物质组成的，但是需要确认的还有另一个问题——为什么有那么多的原子同处一个地方？物理学家们发现，是因为宇宙是一个奇点。奇点的概念与宇宙的存在方式息息相关。如果有一个位置无法把所有可能的事件都纳入考虑范围，那么它就是个奇点。于是，费米的理论走向了原子世界。费米指出，如果存在一个看不见的虚妄世界，使得人的认识停留在认识宇宙的奇点上，那么一定有某种特殊的东西在支配着世界。随着对奇点的分析，科学家们开始发现奇点周围蕴藏着一个有机的生命，它应该给予他们更深远的理解。因此，弗朗西斯·培根和丘利·费米合著了一本名叫《奇点看待世界》的书，其中涉及到费米的两个观点：第一，宇宙是由不确定性、不确定性再度引导和不完全性的原因所创造出来的；第二，宇宙是由奇点所构成。在费米的看来，宇宙可以分成三个层次：最高级的就是奇点，那里有超乎一切想象的东西；中间层次是宇宙的中心，既有物质又有意识，而宇宙的意识是由四维时空的众多粒子构成的；最低级的就是物质。
费米的理论再度拓宽了人类的认识，它由无限的空间和时间组成。他的天才之处就在于将宇宙的所有不确定性都降到了微观的程度。它揭示了宇宙中的不完美，进一步促使大家开始思考：是否可以找到一种方式，使得宇宙的某些方面可以制造出奇迹。
## 1.3 暂存的问题
不过，作为一个物理学家，爱因斯坦并没有结束他的探索。反而相对论给世人留下了深刻的影响。由于狭义相对论过于简单，很多学生认为它是错误的。实际情况是，狭义相对论只是满足了微观的要求，而非宏观的真实需要。狭义相对论并没有揭示宏观物理的所有规律。比如，它虽然将相对论运动学的基本框架给予了解释，但是却没有展开广义相对论的阐述，而且还忽视了其它许多相对论的细节。在现代科技的影响下，相对论已被很多学者提倡，尤其是当今物理学家。与此同时，关于现代物理学的研究越来越火热，各路领军人物纷纷着手推陈出新。今天的主要问题是：如何使得光子的基本概念得到深入地剖析，而不是只是堆砌谬误？为什么当前许多热门理论能够得到实践的验证，但是光子学却还有很多疑问？最后，如何充分地利用光子学发展出来的这些知识？
