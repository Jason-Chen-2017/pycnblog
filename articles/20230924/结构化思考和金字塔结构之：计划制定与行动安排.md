# 1.简介
  

当今社会是一个复杂的时代，对于企业、个人而言，都需要进行一系列的管理活动，从创业到市场营销、产品开发等等，这些活动都要求从各种角度出发，把握时代的趋势，提升工作效率，从而取得成功。其中计划制定的重要性不容忽视。本文将介绍如何进行计划制定，涉及的内容包括什么是计划、计划目的、计划时间、计划范围、计划对策、计划影响、计划结果等等。计划制定有哪些要素？又该如何进行计划制定呢？作者将详细介绍相关内容。
# 2.计划的定义与意义
计划，是指用来实现某种目标或达成某种任务的有序实践过程。在企业管理中，计划可以分为长远、中期、短期三个层次，它们之间存在差异性，具体如下图所示：

- 长远计划，通常由企业年度或者更长时间内的目标来确定。例如：设立一个新的企业，则可能设立年度规划；建立一个新产品，则可以制定长远目标并提出计划；推动某项政策的实施，则应制定长远目标。
- 中期计划，主要用于改善现有产品或服务的质量、可靠性或性能。它一般是在长远计划的基础上，按照短期目标的优先级，根据实际情况确定，如：每季度制定销售目标、改进生产效率、减少运输成本。
- 短期计划，是指一次性或短促的活动。它一般侧重于完成当前目标，如：每周开一次会议；分配几个关键任务；提醒关注新的领域。

计划的目的是为了实现一定目标、达成某个目标或任务。其意义就是把目标、计划及其影响转化为实际可执行的方案、措施及步骤。
# 3.计划目的与目标设置
计划的目的，即为了达到什么目的或获得什么结果，计划的目标往往不仅限于企业内部的事宜，还可以是面向外部的行为，如：筹备一个会议、打造一个产品、宣传某方面的理念、寻找合作伙伴。

目标设置有助于明确自己的目标，帮助自己制定有利于自己的计划，也能促进团队成员间的沟通协调。目标设置可以采用下列四个步骤：
1. 制订预期
首先，对公司、部门或者个人的情况进行全面分析，明确预期结果。目标需要设定一些指标，比如销售额、新增客户、营业收入、税前利润等等，并且采用量化的方法去衡量他们的变化。

2. 确认目标
然后，把预期结果进行排序，确定目标。目标设置首先需要考虑事业部的优先级，因为只有事业部方可最终决定目标的实现。目标不能太宽泛，否则难以落实；目标也不能太具体，否则无法确定方向；目标应该和预期结果相符，但不宜过多。

3. 制定计划
最后，通过制定具体的计划，把目标细化、分解、描述清楚。计划的设计应当清晰准确，即使有突发事件发生也可以及时调整。计划务必包括项目的开始日期、结束日期、角色、职责、资源、范围、交付物和各项信息。

4. 审核与奖励
除了制定计划外，目标设置还需要审核和奖励。目标设置得益于人员对计划制定的熟悉程度。对目标设置具有正面的评价、奖励才能激励员工认真履行目标。
# 4.计划时间、范围、对策
计划的时间、范围、对策可以体现在以下三个方面：

1. 计划的开始日期
计划的开始日期，是指计划具体实施之前的一个最佳机会，要早到好处多多。开始日期最好早些，可以让更多的人参与进来，形成共识，也便于提前做好准备。当然，这也需要公司有足够的资源支持才行。

2. 计划的范围
计划的范围，即指计划实施的对象、范围，要详尽准确。计划的范围不宜太小，且不能超出本单位的资源承受能力。计划的范围应包括涉及到的资源、业务范围、角色、主要时间节点、分解方式、组织架构、风险控制策略等等。

3. 计划的对策
计划的对策，即指计划实施的方式、方法、步骤，每个阶段都要有具体的对策。对策既要直接行动，又要有时限限制和反馈机制，以期得到及时的反馈。对策可以是日常工作中的具体措施，也可以是流程文档、指令手册、工作表格等。
# 5.计划影响与结果
计划的影响，是指计划实施后带来的不同程度的变化。计划影响的形态可以包括产品、项目、财务、人力资源、市场营销、法律、环境等多个方面。计划影响可以有积极的因素和消极的因素，也存在积极、稳定的影响和消极、倒霉的影响。

计划的结果，是指计划达成之后的实际效果。结果的衡量标准，通常包括效果、效率、效益、成本、持续性等多个方面。计划结果应注重数字，也应避免偏离目标太远。
# 6.计划的有效性检验
计划的有效性检验是指验证计划是否真正落地、按时完成。有多种有效性检验方法，可以包括定量分析、定性分析、审计分析、访谈评估、会议记录等。其中定性分析和定量分析最为有效，但也应注意监控计划的持续性和透明度。
# 7.未来发展趋势与挑战
计划制定对于企业而言至关重要。计划制定对于提升企业竞争力、增强员工激情等有着十分重要的作用。不过，随着经济的发展、竞争的加剧，企业计划制定的效率及成果也在逐渐下降。计划制定本身的局限性越来越突显，很可能会成为“管理的灾难”——发展中“恶性循环”。

因此，目前的企业更注重“长”计划而不是“短”计划，而短计划正在走向历史的终点。在这种情况下，长远计划制定的重要性会越来越凸显。计划制定要以一种“结构化”的方式，综合考虑长远、中期、短期计划，以及如何结合科技、产业、人的各个方面，充分发挥计划制定的潜能。
