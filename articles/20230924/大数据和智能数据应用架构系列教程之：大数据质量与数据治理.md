# 1.简介
  

## 一、课程目标与要求
为期6周的《大数据和智能数据应用架构系列教程》系列教程，目的是让更多的人能够快速掌握大数据的基础理论、技能和最佳实践方法。
本次教程《大数据和智能数据应用架构系列教程之：大数据质量与数据治理》的主要目标如下：

1. 满足企业对大数据技术领域的需求，包括数据采集、存储、计算、分析、应用等方面的知识和能力；

2. 能帮助企业更好地管理、运用大数据资源，提升业务价值并减少成本；

3. 为深入理解大数据系统和框架提供一个切入点；

4. 可用于实际工作中。

## 二、教材简介
《大数据和智能数据应用架构系列教程》由阿里云技术专家张波主编，主要内容围绕大数据技术、架构、开发、部署和管理等多个方面进行深入剖析，旨在让读者不仅具备扎实的数据处理、统计、分析、建模等技能，而且还能根据实际场景解决问题。涉及的内容包括：

1. 数据采集、清洗、收集：从各个渠道获取海量数据，整合成结构化、半结构化或非结构化数据，确保数据质量；

2. 数据存储与查询：选择高效可靠、成本低廉的数据存储方案，构建统一的数据模型和访问接口，提供便利的数据查询功能；

3. 数据计算与分析：掌握海量数据的分布式运算、并行计算等技术，有效利用集群资源，实现数据快速分析，建立高质量的数据模型；

4. 机器学习：运用机器学习算法进行数据分析，发现新模式，预测结果，提升业务效果；

5. 海量数据计算平台设计：了解大数据计算平台的体系架构和关键技术，设计适合自己的数据计算平台架构；

6. 数据治理：如何通过数据质量、数据分类和数据使用管理等方式，保障大数据平台的健康稳定运行，避免出现数据泄露、数据安全隐患等风险；

7. 大数据生态环境搭建：介绍云计算、大数据、容器技术、消息队列等技术，以及如何整合到数据治理的各个环节中，助力企业实现数字化转型升级。

## 三、教材总结
本课从数据采集、清洗、收集、数据存储、查询、计算、分析、机器学习、海量数据计算平台设计、数据治理、大数据生态环境搭建等多方面对大数据技术和架构进行全面剖析。本课的核心主题是数据质量与数据治理，本课主要的学习目标如下：

1. 学会应用开源工具ELK（Elasticsearch、Logstash、Kibana）进行数据采集、清洗、收集和日志分析；

2. 了解HDFS、Hive、Spark等技术，掌握这些技术对大数据处理的不同方式和优劣势；

3. 掌握HiveSQL，了解SQL语言的应用和扩展；

4. 了解MapReduce、Hadoop Streaming、Storm、Flink等流计算框架，掌握这些技术的应用原理；

5. 学会使用机器学习算法，通过海量数据挖掘出有价值的商业价值；

6. 学习大数据生态，了解Hadoop生态、云计算、大数据技术发展历史及趋势；

7. 具备构建大数据平台能力，了解各类大数据组件的功能、配置、使用方法；

8. 掌握数据治理的各种方式，以此保障大数据平台的安全稳定运行；

9. 了解数据治理相关的法律法规，如何落实数据治理策略和流程，提升公司竞争力。

通过这门课，读者可以获得以下能力和知识：

1. 在大数据计算平台上，了解HDFS、Yarn、Hive、Impala、Zookeeper、Kafka、Flume等相关技术，了解它们之间的联系与区别；

2. 能够正确使用SQL语言进行Hive数据库的查询、分析、聚合、分组等操作，掌握SQL语句优化的方法；

3. 掌握Hadoop生态中的重要模块如Hadoop Distributed File System（HDFS）、Yet Another Resource Negotiator（YARN）、MapReduce、Streaming、Hive等，能够灵活选择不同的计算框架解决问题；

4. 掌握基于机器学习算法的数据分析方法，掌握常用的机器学习算法如逻辑回归、决策树、随机森林、朴素贝叶斯等，掌握特征工程的方法进行特征提取；

5. 掌握大数据生态，了解大数据技术的发展历程，掌握国内外顶级公司和机构对大数据技术的应用经验；

6. 掌握数据治理的原则、方法和工具，能够充分认识数据安全、隐私保护、合规要求、数据价值、处理效率、成本、数据共享等各方面，合理设计数据治理策略；

7. 了解现代互联网公司面临的大数据技术发展趋势，包括数据的收集、存储、计算、分析、应用等方面所带来的挑战和机遇；

8. 具备构建大数据平台能力，具备大数据应用开发、测试、部署和维护的相关能力；

9. 对数据治理和数据分类有深刻的理解和把握，能够将数据资源管理得当，确保数据安全、隐私保护、合规要求得到落实。
