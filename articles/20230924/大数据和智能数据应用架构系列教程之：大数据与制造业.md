# 1.简介
  

## 1.1 大数据的产生背景及意义
2010年左右，随着互联网的兴起，网络购物、短信、社交媒体等新型业务模式的出现，使得用户不断地产生海量的数据，这些数据极大的激增了计算机数据的存储和处理能力。但是由于数量巨大的数据带来的计算资源不足问题，很快就面临海量数据的存储、分析、处理和通信问题。为了解决这个问题，在2011年，Google发布了MapReduce框架，用于分布式处理海量数据。
但是随后出现的大数据、云计算、机器学习等概念，更是给传统IT行业注入了新的活力。

2012年阿里巴巴集团董事长马云提出“互联网+”，推动公司通过整合商业、金融、零售等领域的信息技术，逐步实现互联互通。阿里巴巴提出基于云计算平台的“互联网+”战略，将“信息技术”的发展引向更高维度。2013年，淘宝、京东等电子商务平台相继崛起。截止到今年5月，全球第三方支付平台累计超过20亿元人民币的交易额，使得线上购物、在线支付、数字化支付成为新的商业模式。

2014年中国移动互联网的普及推动了移动应用市场的快速扩张。国内第三方支付公司当当、财付通等也纷纷布局移动支付领域。中国电信推出了中国移动互联网的云服务平台，通过手机数据流量出口，实现了全国范围内的移动互联网覆盖。

2015年，微信的红包功能试点成功，微信支付横空出世。截至2017年9月，微信全网用户规模达到2.65亿，日均新增用户1.2亿。而支付宝则是2017年年报季度盈利108.1亿元，同比增长162%，净利润同比增长158%。

2016年11月，拼多多宣布正式向消费者开放支付宝钱包，这也标志着支付宝已经从电子商务平台走向了完整的金融支付平台。

随着大数据的流行，越来越多的行业都加入到了大数据与智能应用的平台中，如互联网金融、电子商务、智能制造、智慧城市、智能政务等。从技术层面看，如何有效地运用大数据进行优化、改进以及保障各行业的发展，是各企业面临的共性难题。

## 1.2 智能制造业背景及发展趋势
### 1.2.1 智能制造业概述
智能制造是指利用计算机技术提升生产效率和降低成本，主要应用于智能化的生产环节。目前，智能制造可以分为机械制造、生物制造、自动化手段、虚拟现实、光电子、量子技术等多个产业领域。

目前，智能制造技术的应用主要有以下几个方面：
- 模块化加工技术：智能制造是将复杂的零件拆分为小片，分别组装成不同的产品，这种模块化设计是提高产品精度和满足客户需求的关键。
- 工艺智能技术：智能制造能够识别、改善或预测工艺过程中的失误，提高工艺精度、效率和质量。
- 材料智能技术：智能制造能够根据产品需求，选择合适的材料和工艺来制造产品，帮助企业降低成本并节省能源。
- 智能系统技术：智能制造还可以让机器更聪明，自动识别和完成工作流程，减少人的干预，提高制造效率。
- 产品供应链管理：智能制造要为客户提供更高品质的产品，需要采用全面的供应链管理方法。
- 生产控制系统：智能制造对生产过程进行远程监控、控制和优化，提高整体生产效率。

### 1.2.2 智能制造业发展趋势
2010年，全球智能制造产业投资规模达2.7万亿美元，占比超过30%；2011年至2015年，产业结构经历了由钢铁、建筑、农业等基础行业向其他领域转移；2015年至2018年，AI（人工智能）创新成果带动制造业进入新的发展阶段，促进了产业升级；2018年，智能制造产业持续增长，投资规模达4.3万亿美元，占比超过35%。

2019年，我国智能制造领域初具雏形，预计2020年我国智能制造市场规模将达到6000亿元。此外，近期我国半导体、芯片、芯片组、装备制造等多个领域的智能化转型正在加速推进，致力于实现由产业投资向核心技术转型。

2020年，我国制造业投资达到9.5万亿元，同比增长16.8%；智能制造产业投资达到12.1万亿元，同比增长12.9%。

## 1.3 大数据与智能制造的结合
大数据技术的应用也成为智能制造技术的一项重要支撑，其中包括如下三种应用方式。
### 数据采集与预处理
第一类是从现有数据源采集、清洗、归档、转换数据并产生中间结果，再进行分析、挖掘、搜索及分类，以找出数据中的规律及模式，并据此作出决策。

第二类是分析海量数据，从数据中提取有价值的信息，并将其转化成有用的知识或模型，为人们提供决策支持。

第三类是结合不同设备的数据，包括图像数据、激光雷达数据、无线信号数据、温度数据等，通过算法进行智能化分析和建模，辅助制造过程提升效率、降低成本，提升制造质量。
### 数据驱动的智能化管理
第四类是通过大数据处理、挖掘、分析和预测，实现对生产过程及其设备的智能化管理，优化生产方式、降低成本、提升效率。

该类应用目前在智能制造领域尤为火爆，包括集成车间管理、精益制造、工厂协同智能化、远程监控与控制、精准零配送、机器人技术、自主驾驶等。

### 数据采集与智能制造结合
综上所述，大数据与智能制造结合可分为两类：

第一类是基于大数据的智能化管理方案，如集成车间管理、精益制造、工厂协同智能化、远程监控与控制、精准零配送、机器人技术、自主驾驶等。

第二类是采集、处理、分析数据的应用，如图像数据、激光雷达数据、无线信号数据、温度数据等，通过算法进行智能化分析和建模，辅助制造过程提升效率、降低成本，提升制造质量。
