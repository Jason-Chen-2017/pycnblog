# 1.简介
  

## 1.1电子商务(E-commerce)
电子商务(E-commerce)是一个利用计算机网络、网页技术、在线交易及相关服务来实现互联网用户与商家之间在线交流、销售商品和提供消费者服务的一门新兴产业领域。

根据美国劳工部对1990年到2007年间全美年轻男性人口中购买商品而支付的金额的调查显示，这个行业是经济增长最快的行业之一。据此，我国将电子商务推向纵横百万的网市场，创造了世界上最庞大的“消费者资本”。中国是电子商务蓬勃发展的国家之一，其商业模式如今已成为行业标杆。

电子商务的特点主要体现在以下几方面：

1. 通过互联网进行销售：电子商务采取的是网络化零售的方式，顾客可以在线下单、付款、收货等整个过程都在网上完成，不依赖于店铺的实体门店。

2. 提供多样化产品：电子商ocop拥有庞大且丰富的产品资源，包括图书、音乐、数码产品、日用品等各个领域的产品。顾客可以从不同的商城购买产品，同一时间可享受到不同产品的优惠。

3. 灵活定价机制：顾客在购物过程中可以按实际需要调整价格，也可按特定优惠条件购买。由于网购具有更低的运费成本和即时到账的优势，顾客在选择电子商务购物方式时一般不会忽视运费。

4. 极速支付和快速结算：电子商务采用先付款后付货的方式，顾客在网上完成支付、配送等环节都可以享受到极速结算。一般情况下，一笔订单提交后三小时内即可确认完成，效率非常高。

5. 可靠售后服务：顾客在使用电子商务过程中如果遇到问题，可直接与商家取得联系，快速解决问题。通过网络支付渠道，保证了交易的顺畅。

综合以上特点，可以认为电子商务已经成为商业活动发展的重要组成部分，正在引领着人们对网上的消费方式的转变。随着“互联网+”的发展，电子商务将会越来越受到人们的欢迎。

## 1.2电子商务平台（e-commerce platform）
电子商务平台是在线上交易市场的基础设施，为卖家和买家提供了信息发布、交易管理、物流配送、促销宣传、客户服务等多种服务，是电子商务的一个重要衍生品。电子商务平台为电子商务公司提供了一个共享的应用环境，使得所有参与者（卖家、买家、顾客）都能享受到共同的服务。目前，电子商务平台主要由三个主要部分构成，即网站建设、订单处理系统、营销工具。

网站建设是电子商务平台的基础平台，它可以帮助卖家建立一个专属的网站，展示产品、提供售卖信息、维护信誉档案。网站的功能可以分为三大类：产品展示、购物流程、客户服务。

订单处理系统负责存储、处理和跟踪客户所下的订单。它包括订单提交、确认、发货、售后等一系列的流程，确保每笔订单在整个流程结束之后都是准确无误的。

营销工具则是电子商务平台的关键支撑工具，包括促销推广、搜索排名等。其中，搜索排名工具可以提升网站的知名度，提高品牌影响力。

# 2.基本概念术语说明
## 2.1电子商务平台（E-commerce Platform）
电子商务平台（E-commerce Platform），是指在线交易的基础设施，包含网站建设、订单处理系统和营销工具，为卖家和买家提供了信息发布、交易管理、物流配送、促销宣传、客户服务等多种服务。

电子商务平台通常由三个主要部分组成：网站建设、订单处理系统、营销工具。网站建设负责维护卖家的专属网站，它可以帮助卖家展示产品、提供售卖信息、维护信誉档案；订单处理系统负责存储、处理和跟踪客户所下的订单，它包括订单提交、确认、发货、售后等一系列的流程，确保每笔订单在整个流程结束之后都是准确无误的；营销工具则是电子商务平台的关键支撑工具，包括促销推广、搜索排名等，它可以帮助卖家提升网站的知名度、提高品牌影响力。

## 2.2Django框架
Django是一个Python Web框架，被称为“全栈式Web应用框架”，它是一个开放源代码的Web应用框架，由Python语言编写，以BSD许可证授权。Django是由吉多·范罗苏姆和罗伯特·康伯巴斯基开发的，旨在降低Web开发难度。Django最初是作为Yahoo!的内部工具而诞生的，但后来逐渐流行开来，成为Web开发者和企业的必备选择。
