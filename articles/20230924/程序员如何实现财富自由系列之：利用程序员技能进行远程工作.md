# 2.什么是远程工作？为什么要进行远程工作？为什么很多企业还不适合远程工作？
远程工作（Remote Working）是指一个人可以在异地或者在线地点工作，而不是依靠一处设施完成工作任务。传统上，对于企业而言，在线办公环境往往无法满足员工的需求，所以在90年代末期，一些大型公司开始转型，开始尝试在线办公。但随着移动互联网的普及，在线办公已经逐渐被淘汰，但仍有不少企业坚持在线办公，原因大概是因为在线办公需要依赖视频会议工具或远程协作软件，这些软件对于初级用户来说太过复杂。
现在，越来越多的企业开始采用远程工作方式，许多知名公司，如谷歌、微软、亚马逊等，甚至像苹果、Facebook这样的科技企业也开始尝试远程工作。远程工作模式能够让员工享受到较好的工作环境，不用担心在家里翻墙、连接速度慢等问题。同时，远程工作对员工的身体健康也是有好处的，他们可以从事自己喜欢的工作，并且不必担心长时间坐着。远程工作的兴起也促进了人才的分布式流动，有效利用了组织的资源。
但需要注意的是，虽然远程工作很高产，但也存在一些弊端。比如，由于不再需要固定场所，导致在工作中可能会产生分心、疲劳，甚至产生紧张情绪；同时，远程工作对日常生活中的交流、沟通和协调也会产生一定困难。另外，远程工作容易引起社交恐慌、家庭矛盾等问题。因此，目前，在企业内推广远程工作，尤其是在互联网公司，仍然是一个很小的比例，并不是所有人都适合远程工作。
# 3.为什么程序员在远程工作上需要具备特殊技能？有哪些具体技能需要学习？
远程工作对于程序员而言，除了熟悉网络相关技能外，还需要具备以下几个方面的技能：
1. 智能指导（Intelligent Instruction）——远程工作模式下，每个人的工作习惯和流程都不同。面对各种信息、任务和需求，程序员需要有自己的方法、技巧和处理流程，才能快速准确的把控工作节奏。
2. 技术理解（Technical Understanding）——远程工作需要知识和技能的沉淀。很多软件工具、编程语言、平台等都是需要在实际应用中熟练掌握的。
3. 信息协作（Information Collaboration）——远程工作模式下，员工需要跟踪其他同事的工作进度和情况，并提出建议和意见。因此，程序员需要掌握远程协作软件的使用方法，包括邮件交流、文件共享、任务管理等。
4. 业务表达（Business Communication）——远程工作模式下，程序员需要用英语、母语等方式与团队成员之间进行沟通。因此，程序员需要懂得如何说话、表情、发音、语句间的语调、语法、措辞等。
5. 适应力（Adaptability）——远程工作模式下，程序员的工作环境可能会发生变化。例如，一天中的某段时间没有网络连接，或者当需要连续工作时，只有手机连接。因此，程序员需要具有良好的工作耐性和容错意识。

# 4.如果你不是程序员，希望通过学习程序员的远程工作技能来达到财富自由，你应该如何入门？
首先，了解一下什么是编程。编程是一种自然语言的符号化，计算机能识别的机器语言。程序员利用编程技能，开发计算机软件，完成各种工作，比如编写游戏、网站、APP等。如果你不是程序员，或许可以通过观看一些编程电影或听一些编程相关的播客，了解一些编程的基础知识。比如，《黑客与画家》、《编码》、《重构》、《代码整洁之道》、《程序员的呐喊》等。
然后，接触一些远程工作的案例。通过观看一些公众号、博客，或者参加一些编程聚会，了解远程工作的方式和相关的技能。另外，也可以在Github上搜索关键字“remote work”，查看远程工作相关的开源项目。
最后，可以浏览国外著名远程工作者的个人主页，了解他的经历、技能和过往成功。比如，<NAME>、<NAME>、<NAME>等。他们分享的经验和教训，既可以反映现实，也可以提供借鉴。
通过阅读、观看、聆听，相信你能学到一些远程工作的技能，并有所收获。
# 5.有了这些知识和技能，我们还可以如何提升自己的职业生涯？
程序员在远程工作上的职业生涯，可以分为不同的阶段。我认为，最有利于提升职业生涯的阶段，就是“成长型”。“成长型”的远程工作人员，需要扎实的技术基础，具备专业知识，能够解决突发事件、抓住战略机遇，并拥有丰富的人际关系。除了技术技能外，还需要在日常生活中养成良好的工作习惯，保持积极的态度，在团队中与其他程序员和领导建立深厚的感情联系，尤其是在面对职场竞争时，更需要勇敢冒险。

## Github上关注度最高的项目：TeamViewer Remote Desktop Access
TeamViewer是一个跨平台的远程控制软件，功能强大且价格便宜。它支持Windows、Mac OS、Linux、iOS、Android等多个平台，可在多台电脑之间进行远程控制。而且，免费版还能用于商业用途。
TeamViewer Remote Desktop Access (RDA)是一款TeamViewer的客户端，针对远程办公场景，提供更加便捷的远程控制体验。通过RDA，你可以轻松地访问远程桌面、应用程序，并能接收到来自远程计算机的声音、图像、键盘输入。它的使用界面类似Teamspeak、Skype等常用的远程协作软件。
