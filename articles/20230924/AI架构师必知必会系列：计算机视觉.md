# 1.简介
  

计算机视觉(Computer Vision)是指让电脑“看”或识别图像、视频中的信息的技术。在传统的人类视觉中，我们的眼睛可以直观地看到图像及其中的物体信息，但对于计算机而言，通过摄像头捕获到的图像只是数字信号。因此，计算机视觉系统需要对原始图像进行处理，提取图像中的有用信息并将其呈现给程序员、用户、或者其他的一些机器的处理。例如，在搜索引擎上，我们输入查询关键字之后就立刻得到相关图片、视频或网页结果，这些结果都是经过计算机视觉处理后得到的；人脸识别系统使用计算机视觉技术来进行面部识别，能够检测到面部特征并实时分析。计算机视觉已经成为人工智能领域的热点，越来越多的企业都采用了计算机视觉技术作为核心技术来开发智能产品和服务。

随着移动互联网的蓬勃发展，计算机视觉技术也受到了广泛关注。近年来，随着智能手机的普及和应用飞速增长，人们逐渐将目光投向了“超级摄像机”，也期待着摄像头变得更加智能化、高效能。

为了帮助广大的AI架构师更好地掌握计算机视觉技术，华南理工大学电子信息工程学院于今年7月推出了一套AI架构师必知必会系列课程《AI架构师必知必会系列：计算机视觉》。本课程的目标是通过“动手实践”的方式，介绍计算机视觉领域常用的算法和技术，帮助架构师系统性地学习和掌握计算机视觉技术。课程共分七章，分别介绍计算机视觉的基本原理、数据表示、视觉特征提取、目标检测、图像检索、视频跟踪、自然语言理解等内容。

# 2.概念术语介绍
## 2.1 图片与三维坐标系
首先，我们需要了解一下什么是图片、图象，以及如何使用三维坐标系来描述二维平面的位置关系。

### 2.1.1 图片
图片（Image）是由像素构成的矩阵，每一个像素代表图像中的一个点，每个像素点都有一个颜色值。通常情况下，图像可以是黑白灰度图像、彩色图像或者深度图像。如果是深度图像，则每个像素点都有一个对应的距离值，这个距离值反映了该像素位置到摄像头或相机所在的位置的距离。


上面是一个典型的黑白图像，它的尺寸是200x200个像素，每一个像素点的颜色值是黑或白。如果是彩色图像，则每个像素点还对应着三个颜色值（红绿蓝）。最后，如果是深度图像，则每个像素点还对应着一个距离值，单位是米或厘米。

### 2.1.2 三维坐标系
三维坐标系（Three-dimensional coordinate system），又称为三维空间或三维坐标系，是一个直角坐标系，它在三维空间中用来描述三维对象的位置、方向和距离。它的表示方法可以分为笛卡尔坐标系和球面坐标系两种。其中，笛卡尔坐标系是通过长度、宽度、高度三种直线表示三维空间，与之对应的球面坐标系是通过球心、半径、方位角来表示三维空间。

在三维坐标系中，原点（origin）一般被定义在坐标系中心，坐标轴一般被定义在三个轴线上，如图所示：

## 2.2 概率图像模型
概率图像模型（Probabilistic Image Model, PIM）是一种用于描述真实世界图像的模型，它假设每张图像都是从一定数量的相似模型中生成的，并且这些模型共享某些共同的特征，并且还有一个潜在的生成过程。PIM有以下几个主要组成部分：

1. 模型参数：模型参数包括模型的总数n和每个模型的参数θ，θ表示模型的某种形态、纹理和颜色。不同的模型具有不同的参数，它们之间存在相似性和差异性。

2. 隐变量：隐变量代表了图像数据不易直接获得的信息，比如边缘、噪声、模糊等。在实践中，隐变量通常是通过估计或优化模型参数来得到的。

3. 生成过程：生成过程是一个隐变量到模型参数的映射，它把生成模型的输出描述为随机变量，使得模型参数分布的不同值能导致相同的输出。生成过程决定了模型参数是如何影响生成图像的质量的。

4. 似然函数：似然函数衡量的是数据集P出现某个特定图像的可能性，也可以认为是模型参数θ的后验概率分布。

概率图像模型可以分为以下几种类型：

* 有限集合模型：这种模型只能生成有限数量的模型，即n是有限的，但这种模型的数量仍然有限，不能很好地描述复杂的场景。

* 混合模型：这种模型的n可以无限接近无穷大，但是生成的图像数量往往比较少，因为不同模型之间的相似性和差异性使得图像之间存在一定的相关性。

* 统计模型：这种模型通过统计方法求解模型参数的最大似然估计，并且参数估计值可以认为是生成图像的平均值。虽然这种方法得到的结果较精确，但是计算量很大。

* 分层模型：这种模型利用先验知识、标签信息以及语义信息来分层建模图像。由于模型参数的数量随着分层的增加而增加，因而参数估计的代价也随之增加。

## 2.3 图像金字塔
图像金字塔（Image pyramid）是将一张图像按照不同尺度分割成不同分辨率的小块，并根据不同尺度上的信息，组合成原来的图像。

图像金字塔结构由若干个不同尺度的图像组成，不同尺度的图像之间依次缩小，直至最小尺度的图像。如下图所示：
