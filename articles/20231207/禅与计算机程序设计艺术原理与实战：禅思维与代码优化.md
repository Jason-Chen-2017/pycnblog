# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明禅思维与代码优化的应用。我们将实现一个最长公共子序列的算法。

```python
def longest_common_substring(s1, s2):
    n = len(s1)
    m = len(s2)
    dp = [[0] * (m + 1) for _ in range(n + 1)]
    max_len = 0
    for i in range(1, n + 1):
        for j in range(1, m + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                max_len = max(max_len, dp[i][j])
            else:
                dp[i][j] = 0
    return max_len
```

在这个代码实例中，我们首先定义了一个`longest_common_substring`函数，它接受两个字符串`s1`和`s2`作为输入。我们使用动态规划算法来解决最长公共子序列问题。我们首先创建了一个`dp`数组，用于存储子问题的状态值。然后，我们遍历`s1`和`s2`中的每个字符，并根据字符是否相等来更新`dp`数组中的值。最后，我们返回最长公共子序列的长度。
