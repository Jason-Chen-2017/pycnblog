# 4.具体代码实例和详细解释说明

在本节中，我们将通过详细的代码实例来解释Lua表和元表的概念和操作。

## 4.1 创建表

```lua
table = {}
```

在这个代码实例中，我们创建了一个新的表，并将其赋值给变量`table`。这个表是一个空表，没有任何元素。

## 4.2 添加元素

```lua
table["key"] = "value"
```

在这个代码实例中，我们将一个元素添加到表中。这个元素的键是`"key"`，值是`"value"`。

## 4.3 删除元素

```lua
table["key"] = nil
```

在这个代码实例中，我们将一个元素从表中删除。这个元素的键是`"key"`，值是`nil`。

## 4.4 查找元素

```lua
value = table["key"]
```

在这个代码实例中，我们从表中查找一个元素。这个元素的键是`"key"`，值是`value`。

## 4.5 遍历表

```lua
for key, value in pairs(table) do
    print(key, value)
end
```

在这个代码实例中，我们遍历了表中的所有元素。我们使用`for key, value in pairs(table) do`来遍历表中的所有元素，并将每个元素的键和值打印出来。
