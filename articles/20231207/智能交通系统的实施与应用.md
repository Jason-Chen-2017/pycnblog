                 

# 1.背景介绍

智能交通系统是一种利用信息技术、通信技术、电子技术、控制技术、计算机技术等多种技术为交通系统提供智能化、自主化、可靠性、高效性、环保性等特征的交通系统。智能交通系统的发展是为了应对交通拥堵、交通安全、交通环境污染等问题，为提高交通效率、减少交通拥堵、提高交通安全、减少交通污染、减少交通成本、提高交通服务质量等目标而努力发展。

智能交通系统的主要组成部分包括交通信息收集系统、交通信息处理系统、交通信息传播系统、交通控制系统、交通应用系统等。

交通信息收集系统主要包括交通信息传感器、交通信息采集设备、交通信息传输设备等。交通信息传感器用于收集交通信息，如车辆速度、车流量、车辆位置等；交通信息采集设备用于对交通信息进行处理，如对车辆速度进行加速度测量、对车流量进行统计等；交通信息传输设备用于将交通信息传输到交通信息处理系统中，如通过无线通信技术进行传输。

交通信息处理系统主要包括交通信息处理软件、交通信息处理硬件等。交通信息处理软件用于对交通信息进行处理，如对车辆速度进行分析、对车流量进行预测等；交通信息处理硬件用于对交通信息进行处理，如对车辆速度进行加速度测量、对车流量进行统计等。

交通信息传播系统主要包括交通信息传播设备、交通信息传播通道等。交通信息传播设备用于将交通信息传播到交通用户中，如通过电视、电话、互联网等进行传播；交通信息传播通道用于将交通信息传播到交通用户中，如通过电视、电话、互联网等进行传播。

交通控制系统主要包括交通控制软件、交通控制硬件等。交通控制软件用于对交通控制进行控制，如对车辆速度进行控制、对车流量进行控制等；交通控制硬件用于对交通控制进行控制，如对车辆速度进行控制、对车流量进行控制等。

交通应用系统主要包括交通应用软件、交通应用硬件等。交通应用软件用于提供交通应用服务，如提供交通路线规划服务、提供交通实时信息服务等；交通应用硬件用于提供交通应用服务，如提供交通路线规划服务、提供交通实时信息服务等。
