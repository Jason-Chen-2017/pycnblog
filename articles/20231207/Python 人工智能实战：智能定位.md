## 4.具体代码实例和详细解释说明

在这里，我们将提供一个简单的 Python 代码实例，用于计算两点之间的距离：

```python
import math

def distance(x1, y1, x2, y2):
    return math.sqrt((x2 - x1)**2 + (y2 - y1)**2)

x1, y1 = 0, 0
x2, y2 = 1, 1

print(distance(x1, y1, x2, y2))
```

在这个代码中，我们首先导入了 math 模块，然后定义了一个 distance 函数，用于计算两点之间的距离。接着，我们定义了两个点的坐标，并调用 distance 函数来计算距离。最后，我们打印出计算结果。
