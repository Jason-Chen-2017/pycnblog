### 1.4.1 MySQL数据库操作库的代码实例

以下是一个使用MySQL数据库操作库的代码实例：

```python
import mysql.connector

# 建立数据库连接
connection = mysql.connector.connect(
    host="localhost",
    user="your_username",
    password="your_password",
    database="your_database"
)

# 创建游标对象
cursor = connection.cursor()

# 执行查询
query = "SELECT * FROM your_table"
cursor.execute(query)

# 获取查询结果
results = cursor.fetchall()

# 关闭游标对象
cursor.close()

# 关闭数据库连接
connection.close()
```

在这个代码实例中，我们首先建立了一个数据库连接。然后，我们创建了一个游标对象，并使用该对象执行了一个查询。最后，我们获取了查询结果，并关闭了游标对象和数据库连接。
