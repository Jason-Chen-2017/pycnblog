# 2.核心概念与联系

在主动学习中，学习算法可以选择性地请求对未知类别的标签。这与传统的监督学习中的学习方法不同，其中标签是预先给定的。主动学习可以提高学习效率，因为学习算法可以根据其需求选择最有价值的样本进行学习。

大模型在主动学习中的应用，可以帮助学习算法更有效地选择样本，从而提高学习效率和准确性。大模型可以通过对数据进行预测，为学习算法提供有关样本的信息，从而帮助学习算法更有效地选择样本进行学习。
