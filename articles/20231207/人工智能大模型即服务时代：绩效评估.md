# 4.具体代码实例和详细解释说明

在大模型即服务时代，我们需要一种更加高效、准确的方法来评估大模型的绩效。这种方法的具体代码实例如下：

```python
from sklearn.metrics import precision_score, recall_score, f1_score

# 假设我们有一个预测结果的列表
predictions = [1, 0, 1, 1, 0]

# 假设我们有一个真实结果的列表
true_labels = [1, 1, 0, 0, 0]

# 计算精度
precision = precision_score(true_labels, predictions)
print("Precision:", precision)

# 计算召回率
recall = recall_score(true_labels, predictions)
print("Recall:", recall)

# 计算F1分数
f1 = f1_score(true_labels, predictions)
print("F1 Score:", f1)
```

在这个代码实例中，我们使用了`sklearn`库中的`precision_score`、`recall_score`和`f1_score`函数来计算精度、召回率和F1分数。我们将预测结果和真实结果作为输入，并将计算结果打印出来。
