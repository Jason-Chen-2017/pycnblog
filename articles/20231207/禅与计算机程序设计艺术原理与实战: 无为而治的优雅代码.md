```python
def longest_common_subsequence(s1, s2):
    dp = [[0] * (len(s2) + 1) for _ in range(len(s1) + 1)]
    for i in range(len(s1)):
        for j in range(len(s2)):
            if s1[i] == s2[j]:
                dp[i+1][j+1] = dp[i][j] + 1
            else:
                dp[i+1][j+1] = max(dp[i+1][j], dp[i][j+1])
    return dp[-1][-1]
```

在这个例子中，我们使用动态规划来解决“最长公共子序列”问题。我们首先创建了一个dp数组，用于存储子问题的最优解。然后，我们遍历字符串s1和s2的每个字符，如果当前字符相等，我们更新dp数组的值；否则，我们选择两个子问题的最大值。最后，我们返回dp数组的最后一个元素，即最长公共子序列的长度。

```python
def activity_selection(activities):
    activities.sort(key=lambda x: x['start'])
    selected_activities = []
    current_end_time = -float('inf')
    for activity in activities:
        if activity['start'] > current_end_time:
            selected_activities.append(activity)
            current_end_time = activity['end']
    return selected_activities
```

在这个例子中，我们使用贪心算法来解决“活动选择”问题。我们首先将活动按照开始时间进行排序。然后，我们遍历所有活动，如果当前活动的开始时间大于当前结束时间，我们将当前活动加入到选择的活动列表中，并更新当前结束时间。最后，我们返回选择的活动列表。

```python
def fast_power(x, n):
    if n == 0:
        return 1
    elif n % 2 == 0:
        return fast_power(x * x, n // 2)
    else:
        return x * fast_power(x * x, (n - 1) // 2)
```

在这个例子中，我们使用分治算法来解决“快速幂”问题。我们首先判断n是否为0，如果是，我们返回1；否则，我们判断n是否为偶数，如果是，我们将问题分解为快速幂的两倍；否则，我们将问题分解为快速幂的(n-1)//2次方。然后，我们递归地解决子问题，直到n为0为止。最后，我们返回最终的结果。
