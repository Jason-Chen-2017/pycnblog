在了解加密货币的核心算法原理之后，我们需要看一些具体的代码实例。

## 4.1 创建钱包

创建钱包需要生成一对公钥和私钥。以下是一个使用Python的PyCryptoDome库创建钱包的示例代码：

```python
from Crypto.PublicKey import ECC

# 生成一对公钥和私钥
key = ECC.generate(curve="P-256")
public_key = key.public_key().export_key()
private_key = key.export_key()
```

## 4.2 创建交易

创建交易需要使用钱包的私钥签名交易，并使用公钥验证交易的有效性。以下是一个使用Python的PyCryptoDome库创建交易的示例代码：

```python
from Crypto.PublicKey import ECC
from Crypto.Signature import DSS
from Crypto.Hash import SHA256

# 使用私钥签名交易
hash_obj = SHA256.new(b"交易数据")
signature = DSS.new(private_key, 'fips-186-3', hash_algorithm=SHA256)
signature.sign(hash_obj)

# 使用公钥验证交易的有效性
public_key = ECC.import_key(public_key)
try:
    public_key.verify(hash_obj, signature)
    print("交易有效")
except ValueError:
    print("交易无效")
```
