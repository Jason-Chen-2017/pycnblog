# 4.具体代码实例和详细解释说明

在讨论框架设计原理之前，我们需要看一些具体的代码实例。这些代码实例可以帮助我们更好地理解框架设计的原理和实战。

例如，我们可以看一个简单的框架实例：

```python
class Framework:
    def __init__(self):
        self.core = Core()
        self.plugins = []

    def add_plugin(self, plugin):
        self.plugins.append(plugin)

    def run(self):
        self.core.run()
        for plugin in self.plugins:
            plugin.run()

class Core:
    def run(self):
        print("Core is running...")

class Plugin:
    def run(self):
        print("Plugin is running...")

framework = Framework()
framework.add_plugin(Plugin())
framework.run()
```

在这个例子中，我们创建了一个框架类，它包含一个核心部分和一个插件系统。框架类的`add_plugin`方法可以让我们添加插件，框架类的`run`方法可以让我们运行框架和插件。
