                 

# 1.背景介绍

Redis是一个开源的高性能的key-value数据库，它支持数据的持久化，可基于内存（Volatile）或磁盘（Persistent）。Redis 提供多种语言的 API。

Redis 是一个使用 ANSI C 语言编写、遵循 BSD 协议的开源 ( BSD ) 软件。Redis 的根源是 Redis 数据结构服务器，Redis 可以用作数据库、缓存和消息队列。Redis 提供多种语言的 API，包括：C、Ruby、Python、Go、Java、PHP、Node.js、Perl、R、C#、C++、Lua、Objective-C、Swift 和 JavaScript。

Redis 支持数据的持久化，可以将它们保存在磁盘上，以便在没有 Redis 实例运行的情况下对数据的操作和查询。Redis 允许用户在配置文件中指定保存数据的磁盘路径。

Redis 提供了多种数据结构：字符串 (strings)、哈希 (hashes)、列表 (lists)、集合 (sets) 和有序集合 (sorted sets)。每种数据结构的命令集都与 Redis 的其他数据结构集合相互独立。
