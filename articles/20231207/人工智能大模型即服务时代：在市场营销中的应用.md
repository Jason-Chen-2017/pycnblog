3. **广告投放优化**：

```python
from scipy.optimize import minimize

# 定义优化目标函数
def objective_function(x):
    return x[0] * x[1] + x[2]

# 定义约束条件
def constraint_function(x):
    return x[0] + x[1] - 100

# 初始化变量
x0 = [10, 10, 10]

# 使用优化算法进行广告投放优化
result = minimize(objective_function, x0, constraints=[{'type': 'eq', 'fun': constraint_function}])

# 获取优化结果
print(result.x)
```
