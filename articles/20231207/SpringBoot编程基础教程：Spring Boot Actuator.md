Spring Boot Actuator 是 Spring Boot 的一个核心组件，它提供了一组用于监控和管理 Spring Boot 应用程序的端点。这些端点可以用于获取应用程序的元数据、性能指标、错误信息等信息。

Spring Boot Actuator 的核心功能包括：

- 提供一组用于监控和管理应用程序的端点，如 /actuator/health、/actuator/metrics、/actuator/info 等。
- 提供一组用于管理应用程序的操作，如 /actuator/shutdown、/actuator/reset 等。
- 提供一组用于监控和管理数据库的端点，如 /actuator/db/* 等。
- 提供一组用于监控和管理缓存的端点，如 /actuator/cache/* 等。
