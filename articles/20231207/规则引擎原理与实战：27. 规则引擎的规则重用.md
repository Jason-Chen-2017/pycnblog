# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明规则引擎的规则重用技术。

假设我们有以下三个规则：

1. 如果客户的年龄大于30岁，则提供优惠券。
2. 如果客户的年龄大于40岁，则提供优惠券。
3. 如果客户的年龄大于50岁，则提供优惠券。

我们可以将这三个规则组合成一个规则集，以实现更高效的规则执行。

```python
# 定义规则集
rules = [
    (lambda age: age > 30, '提供优惠券'),
    (lambda age: age > 40, '提供优惠券'),
    (lambda age: age > 50, '提供优惠券')
]

# 定义客户的年龄
customer_age = 45

# 遍历规则集，执行规则
for rule in rules:
    if rule[0](customer_age):
        print(rule[1])
```

在这个代码实例中，我们首先定义了一个规则集，包含三个规则。然后，我们定义了一个客户的年龄，并遍历规则集，执行每个规则。如果规则条件满足，则执行规则动作。
