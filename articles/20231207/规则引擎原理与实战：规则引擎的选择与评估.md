## 3.2 规则引擎的执行过程
规则引擎的执行过程包括以下步骤：
1. 从规则库中加载规则。
2. 从工作内存中加载事实。
3. 根据规则库中的规则，对事实进行检查和处理。
4. 根据检查和处理结果，更新工作内存中的事实。
5. 重复步骤3和4，直到所有规则都被检查和处理。

# 4.具体代码实例和详细解释说明

```python
# 定义规则库
rules = [
    {
        "condition": "age >= 18",
        "action": "grant_access"
    },
    {
        "condition": "age < 18",
        "action": "deny_access"
    }
]

# 定义工作内存
facts = {
    "age": 20
}

# 定义规则引擎核心
def execute_rules(rules, facts):
    for rule in rules:
        if eval(rule["condition"]):
            rule["action"]()

# 执行规则引擎
execute_rules(rules, facts)
```
