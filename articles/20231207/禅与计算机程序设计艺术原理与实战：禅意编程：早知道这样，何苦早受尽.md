# 4.具体代码实例和详细解释说明

为了更好地理解禅意编程的实际应用，我们将通过一个具体的代码实例来解释禅意编程的实际应用。

假设我们需要编写一个程序，用于计算两个整数的和。我们可以使用以下的代码实现：

```python
def add(a, b):
    return a + b

if __name__ == '__main__':
    a = 10
    b = 20
    print(add(a, b))
```

在这个代码实例中，我们首先定义了一个名为`add`的函数，用于计算两个整数的和。然后，我们在主程序中调用了这个函数，并将两个整数的值传递给这个函数。最后，我们使用`print`函数输出了计算结果。

这个代码实例的时间复杂度为O(1)，空间复杂度为O(1)。这意味着，无论输入的整数值有多大，程序的运行时间和占用内存都是固定的。这就是禅意编程的精益求精的思想。
