优化算法的具体操作步骤如下：

1. 初始化决策变量 x 和目标函数值 f(x)。
2. 对决策变量 x 进行迭代更新，以找到使目标函数值最小的决策组合。
3. 对迭代更新后的决策变量 x 进行收敛判断，以确定是否满足停止条件。
4. 如果满足停止条件，则输出最优决策组合 x 和最优目标函数值 f(x)。否则，返回第2步。

    return x - 1

x = 0

while True:
    x_new = x - 0.1
