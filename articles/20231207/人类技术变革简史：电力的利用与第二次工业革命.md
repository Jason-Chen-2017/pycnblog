# 4.具体代码实例和详细解释说明

在第二次工业革命中，具体代码实例可以用来说明如何使用电力来驱动机器和设备。以下是一个简单的代码实例：

```python
import time

def motor_on(voltage, current):
    power = voltage * current
    print("Motor is on")
    time.sleep(1)
    print("Motor is off")

def motor_off(voltage, current):
    power = voltage * current
    print("Motor is off")
    time.sleep(1)
    print("Motor is on")

if __name__ == '__main__':
    voltage = 12
    current = 5
    motor_on(voltage, current)
    motor_off(voltage, current)
```

在这个代码实例中，我们定义了两个函数：`motor_on`和`motor_off`。`motor_on`函数用来启动电机，`motor_off`函数用来关闭电机。我们使用`voltage`和`current`变量来表示电压和电流。我们使用`power`变量来计算功率。我们使用`time.sleep`函数来模拟电机的运行时间。
