# 2.核心概念与联系
在线广告是指在互联网上展示的广告，包括图片、文字、视频等多种形式。广告技术则是用于优化广告投放、展示和收集数据的技术。

在线广告的核心概念包括：
- 广告发布方：即广告主，通常是企业或个人，希望通过在线广告来提高品牌知名度和销售额。
- 广告投放方：即广告网络，通常是互联网公司，负责将广告发布方的广告投放到合适的网站或应用程序上。
- 广告展示方：即广告主机，通常是网站或应用程序，负责将广告投放方的广告展示给用户。

广告技术的核心概念包括：
- 广告投放策略：即如何将广告发布方的广告投放到合适的广告展示方上。
- 广告展示策略：即如何将广告投放方的广告展示给用户。
- 广告数据收集：即如何收集广告的展示和点击数据，以便进行广告效果的分析和优化。
