# 2.核心概念与联系

虚拟现实（VR）和增强现实（AR）是两种不同的技术，但它们之间存在密切的联系。虚拟现实是一个完全虚构的环境，用户可以通过特殊的设备进入这个环境，与虚拟物体和场景进行互动。增强现实则是将虚拟元素与现实环境相结合，让用户在现实世界中与虚拟物体进行互动。

虚拟现实（VR）的核心概念包括：

- 虚拟现实环境：用户可以进入的虚构的环境。
- 虚拟物体：用户可以与之互动的虚构物体。
- 输入设备：用户与虚拟环境进行互动的设备，如手柄、头戴式显示器等。
- 输出设备：用户可以看到虚拟环境的设备，如VR头盔、声音等。

增强现实（AR）的核心概念包括：

- 现实环境：用户所处的现实环境。
- 虚拟元素：用户可以与之互动的虚构物体。
- 输入设备：用户与虚拟环境进行互动的设备，如手势识别、手柄等。
- 输出设备：用户可以看到虚拟元素的设备，如AR显示器、声音等。

虚拟现实和增强现实的联系在于它们都涉及到虚拟元素与现实环境的结合。虚拟现实完全是虚构的环境，而增强现实则将虚拟元素与现实环境相结合，让用户在现实世界中与虚拟物体进行互动。
