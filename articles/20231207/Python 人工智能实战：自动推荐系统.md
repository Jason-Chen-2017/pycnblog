# 2.核心概念与联系

在自动推荐系统中，我们需要关注以下几个核心概念：

1. 用户：用户是系统的主体，他们的行为和兴趣是推荐系统的核心数据来源。
2. 物品：物品是用户所关注的对象，例如商品、视频、音乐等。
3. 用户行为：用户行为是用户与物品的互动，例如购买、收藏、点赞等。
4. 推荐：推荐是将物品推送给用户的过程，目的是让用户更好地满足需求。

```python
import numpy as np
from scipy.spatial.distance import cosine

def calculate_similarity(user_behavior):
    user_behavior_matrix = np.array(user_behavior)
    similarity_matrix = 1 - cosine(user_behavior_matrix)
    return similarity_matrix

def recommend(user_id, user_behavior, similarity_matrix, top_n):
    user_behavior_vector = np.array(user_behavior[user_id])
    similarity_vector = similarity_matrix[user_id]
    top_n_items = np.argsort(similarity_vector)[-top_n:]
    return top_n_items

user_behavior = {
    1: [(1, 1, 'buy', '2021-01-01'), (1, 2, 'buy', '2021-01-02'), (1, 3, 'buy', '2021-01-03')],

    3: [(3, 1, 'buy', '2021-01-01'), (3, 2, 'buy', '2021-01-02'), (3, 3, 'buy', '2021-01-03')],
}

similarity_matrix = calculate_similarity(user_behavior)
top_n = 2
recommended_items = recommend(1, user_behavior, similarity_matrix, top_n)
print(recommended_items)
```
