```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)

arr = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
print(quick_sort(arr))
```

在这个代码实例中，我们可以看到快速排序算法的实现。我们可以看到，这个算法的时间复杂度为O(nlogn)，空间复杂度为O(logn)。同时，我们可以看到这个算法的代码是可读的，并且可以轻松地进行修改和优化。
