                 

# 1.背景介绍

Redis是一个开源的高性能的key-value数据库，它支持数据的持久化，可基于内存（Redis）或磁盘（AOF）。Redis 提供多种语言的 API。Redis 可以作为数据库，缓存和消息队列的替代方案。

Redis 是一个使用 ANSI C 语言编写、遵循 BSD 协议的开源（GPL）、跨平台（Windows，Linux，Mac OS X，UNIX 等）的高性能 key-value 数据库，它支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。

Redis 提供多种语言的 API，包括：Ruby、Python、Java、C#、Go、Node.js、PHP、Perl、Lua、C、C++、R、Haskell、F#、Elixir 和 Swift。
