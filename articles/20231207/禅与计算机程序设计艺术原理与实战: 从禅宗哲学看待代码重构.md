# 4.具体代码实例和详细解释说明
在这里，我们将通过一个简单的代码重构示例来说明上述算法原理和操作步骤：

原始代码：
```python
def add(a, b):
    c = a + b
    return c

def subtract(a, b):
    c = a - b
    return c

def multiply(a, b):
    c = a * b
    return c

def divide(a, b):
    c = a / b
    return c
```

重构后代码：
```python
def calculate(a, b, operation):
    if operation == 'add':
        return a + b
    elif operation == 'subtract':
        return a - b
    elif operation == 'multiply':
        return a * b
    elif operation == 'divide':
        return a / b
    else:
        raise ValueError('Invalid operation')
```

通过将原始代码中的四个函数合并为一个函数，我们可以提高代码的模块化和封装性，同时减少代码的重复和冗余。这种重构方法有助于提高代码的可维护性和可扩展性。
