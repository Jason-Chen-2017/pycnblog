## 4.4 线性规划

我们可以使用Python的pulp库来进行线性规划。例如，我们可以使用pulp的LpProblem类来创建线性规划问题，并使用LpMinimize类来设置目标函数，最后使用solve方法来求解问题。

```python
import pulp
import numpy as np

# 创建线性规划问题
problem = pulp.LpProblem('Health Allocation Problem', pulp.LpMinimize)

# 创建变量
x = pulp.LpVariable('x', lowBound=0)

# 设置目标函数
problem += x

# 添加约束条件
problem += x <= 100

# 求解问题
problem.solve()

# 输出结果
print(x.value())
```
