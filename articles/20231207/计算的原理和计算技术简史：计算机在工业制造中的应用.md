```python
import numpy as np

def production_plan(demand, resource):
    plan = np.zeros(resource.shape)
    for i in range(resource.shape[0]):
        plan[i] = min(demand[i], resource[i])
    return plan

# 示例使用

resource = np.array([150, 250, 350])
plan = production_plan(demand, resource)
print(plan)
```

在示例使用部分，我们创建了一个示例的生产需求和生产资源，并将其传递给`production_plan`函数，然后将生产计划矩阵打印出来。
