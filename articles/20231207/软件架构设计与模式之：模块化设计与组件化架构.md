## 1.4 具体代码实例和详细解释说明

### 1.4.1 模块化设计的具体代码实例

以一个简单的计算器程序为例，我们可以将其划分为多个模块，如：

- 输入模块：负责获取用户输入的数字。
- 计算模块：负责执行计算操作，如加法、减法、乘法、除法等。
- 输出模块：负责输出计算结果。

具体代码实例如下：

```python
# 输入模块
def get_input():
    num1 = input("请输入第一个数字：")
    num2 = input("请输入第二个数字：")
    return num1, num2

# 计算模块
def compute(num1, num2, op):
    if op == "+":
        return float(num1) + float(num2)
    elif op == "-":
        return float(num1) - float(num2)
    elif op == "*":
        return float(num1) * float(num2)
    elif op == "/":
        return float(num1) / float(num2)
    else:
        return "错误的运算符"

# 输出模块
def output(result):
    print("计算结果为：", result)

# 主函数
def main():
    num1, num2 = get_input()
    op = input("请输入运算符：")
    result = compute(num1, num2, op)
    output(result)

if __name__ == "__main__":
    main()
```

### 1.4.2 组件化架构的具体代码实例

以一个简单的文件上传程序为例，我们可以将其划分为多个组件，如：

- 文件选择组件：负责选择要上传的文件。
- 文件上传组件：负责将文件上传到服务器。
- 上传进度组件：负责显示上传进度。

具体代码实例如下：

```python
# 文件选择组件
from tkinter import filedialog

def select_file():
    file_path = filedialog.askopenfilename()
    return file_path

# 文件上传组件
import requests

def upload_file(file_path, url):
    with open(file_path, "rb") as f:
        files = {"file": f}
        response = requests.post(url, files=files)
        return response.text

# 上传进度组件
import time

def show_progress(progress):
    print("上传进度：", progress, "%")

# 主函数
def main():
    file_path = select_file()
    url = "http://example.com/upload"
    response = upload_file(file_path, url)
    show_progress(response)

if __name__ == "__main__":
    main()
```
