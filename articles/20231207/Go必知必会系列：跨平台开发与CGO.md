# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释CGO的使用方法。

假设我们有一个C语言函数，它可以将两个整数相加：

```c
int add(int a, int b) {
    return a + b;
}
```

我们可以创建一个Go程序，调用这个C语言函数，并将结果返回给Go程序：

```go
package main

import "C"

func main() {
    a := 10
    b := 20
    c := C.add(C.int(a), C.int(b))
    fmt.Println(int(c))
}
```

在这个Go程序中，我们首先引用了C语言代码和库。然后，我们调用了C语言函数`add`，并将结果返回给Go程序。最后，我们将结果打印出来。

当我们运行这个Go程序时，它将输出`30`，这就是两个整数`10`和`20`的和。
