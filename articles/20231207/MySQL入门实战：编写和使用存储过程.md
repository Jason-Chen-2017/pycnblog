以下是一个具体的存储过程代码实例：

```sql
CREATE PROCEDURE get_employee_info (IN emp_id INT)
BEGIN
    SELECT * FROM employees WHERE employee_id = emp_id;
END;
```

在这个例子中，我们创建了一个名为get_employee_info的存储过程，它接受一个整数参数emp_id。在存储过程中，我们使用SELECT语句来查询员工信息，并将结果返回给调用者。

详细解释说明：

- BEGIN和END关键字用于定义存储过程的开始和结束。
- SELECT语句用于查询员工信息，并将结果返回给调用者。
