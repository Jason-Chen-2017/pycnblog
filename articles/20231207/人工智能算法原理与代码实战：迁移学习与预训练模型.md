# 2.核心概念与联系

## 2.1 迁移学习

迁移学习是一种机器学习技术，它可以帮助我们在一个任务上学习的模型在另一个任务上进行迁移，从而减少学习时间和资源消耗。迁移学习可以分为两种类型：一种是基于特征的迁移学习，另一种是基于模型的迁移学习。

基于特征的迁移学习是一种将学习到的特征用于另一个任务的方法。例如，我们可以在一个任务上学习一个特征表示，然后将这个特征表示用于另一个任务。这种方法的优点是简单易用，缺点是可能会丢失一些任务特定的信息。

基于模型的迁移学习是一种将学习到的模型用于另一个任务的方法。例如，我们可以在一个任务上训练一个神经网络模型，然后将这个模型用于另一个任务。这种方法的优点是可以保留任务特定的信息，缺点是可能会增加计算复杂度。

## 2.2 预训练模型

预训练模型是一种深度学习技术，它可以帮助我们在一个大规模的数据集上进行预训练，然后在一个小规模的数据集上进行微调，从而提高模型的性能。预训练模型可以分为两种类型：一种是自监督预训练，另一种是监督预训练。

自监督预训练是一种通过自动编码器等方法将大规模无标签数据用于预训练的方法。例如，我们可以将大规模的图像数据用于自监督预训练，然后将这个预训练模型用于小规模的分类任务。这种方法的优点是可以利用大规模无标签数据进行预训练，缺点是可能会丢失一些任务特定的信息。

监督预训练是一种通过回归、分类等方法将大规模标签数据用于预训练的方法。例如，我们可以将大规模的语音数据用于监督预训练，然后将这个预训练模型用于小规模的语音识别任务。这种方法的优点是可以利用大规模标签数据进行预训练，缺点是可能会增加计算复杂度。
