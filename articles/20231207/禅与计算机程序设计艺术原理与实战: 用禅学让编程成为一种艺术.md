1. **递归实例**：

```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)
```

在这个递归实例中，我们可以看到递归的思想在计算阶乘时的应用。我们通过将问题分解为子问题，并将子问题的解决方案存储在一个表格中，以便在后续的计算中重用。

2. **动态规划实例**：

```python
def coin_change(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0

    for i in range(amount + 1):
        for coin in coins:
            if i >= coin:
                dp[i] = min(dp[i], dp[i - coin] + 1)

    return dp[amount]
```

在这个动态规划实例中，我们可以看到动态规划的思想在计算硬币组合时的应用。我们通过将问题分解为子问题，并将子问题的解决方案存储在一个表格中，以便在后续的计算中重用。

3. **贪心算法实例**：

```python
def knapsack(items, capacity):
    items = sorted(items, key=lambda x: x[1]/x[0], reverse=True)

    result = []
    current_weight = 0

    for item in items:
        if current_weight + item[0] <= capacity:
            result.append(item)
            current_weight += item[0]

    return result
```

在这个贪心算法实例中，我们可以看到贪心算法的思想在背包问题时的应用。我们通过在每个步骤中选择最佳解决方案来解决问题。
