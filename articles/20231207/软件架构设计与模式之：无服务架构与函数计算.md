# 2.核心概念与联系

- 服务（Service）：无服务架构中的基本组成单元，是一个独立的、可扩展的、可独立部署和维护的应用程序组件。
- API（Application Programming Interface）：服务之间的通信接口，用于实现服务之间的数据交换和调用。
- 微服务（Microservice）：无服务架构中的服务化设计模式，将应用程序划分为多个小型服务，每个服务都是独立的、可扩展的、可独立部署和维护的。
- 函数（Function）：函数计算中的基本组成单元，是一个独立的、可扩展的、可独立部署和维护的计算任务组件。
- 计算任务（Compute Task）：函数计算中的基本工作单元，是一个需要完成的计算任务。
