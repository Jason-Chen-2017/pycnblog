# 2.核心概念与联系

## 2.1 冗余

冗余是指在系统中添加多余的组件，以便在某些组件失效时，其他组件可以继续提供服务。冗余可以分为热备、冷备和活备三种类型。热备是指在线备份，即备份和主备一起工作；冷备是指离线备份，即备份不与主备一起工作；活备是指主备一起工作，并在主备之间进行负载均衡。

## 2.2 负载均衡

负载均衡是指将请求分发到多个服务器上，以便在系统的负载均衡。负载均衡可以分为硬件负载均衡和软件负载均衡两种类型。硬件负载均衡是指使用专门的硬件设备进行负载均衡；软件负载均衡是指使用软件进行负载均衡。

## 2.3 容错

容错是指系统在出现故障时，能够自动恢复并继续工作。容错可以分为故障捕获、故障恢复和故障预防三种类型。故障捕获是指系统能够捕获故障信息；故障恢复是指系统能够自动恢复并继续工作；故障预防是指系统能够预防故障发生。
