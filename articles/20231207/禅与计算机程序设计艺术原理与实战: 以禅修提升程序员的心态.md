动态规划是一种计算方法，可以帮助程序员更好地解决问题。动态规划的基本思想是将问题分解为更小的子问题，然后递归地解决这些子问题。

例如，我们可以使用动态规划来解决最长公共子序列问题。最长公共子序列问题是指在两个字符串中找出最长的公共子序列。我们可以使用动态规划来解决这个问题，如下所示：

```python
def longest_common_substring(s1, s2):
    m = len(s1)
    n = len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    max_len = 0
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
                max_len = max(max_len, dp[i][j])
            else:
                dp[i][j] = 0
    return max_len
```

在上面的代码中，我们定义了一个名为longest_common_substring的函数，该函数接受两个字符串参数s1和s2。我们使用一个二维数组dp来存储子问题的结果。我们遍历字符串s1和s2，并根据字符是否相等来更新dp数组。最后，我们返回最长公共子序列的长度。

## 4.3贪心算法实例

贪心算法是一种计算方法，可以帮助程序员更好地解决问题。贪心算法的基本思想是在每个步骤中选择最佳解，然后递归地解决问题。

例如，我们可以使用贪心算法来解决最接近目标值的k个数问题。给定一个数组arr和一个目标值target，我们需要找到距离target最近的k个数。我们可以使用贪心算法来解决这个问题，如下所示：

```python
def closest_k_numbers(arr, target):
    arr.sort()
    k = 5
    min_diff = float('inf')
    closest_numbers = []
    for num in arr:
        diff = abs(num - target)
        if diff < min_diff:
            min_diff = diff
            closest_numbers = [num]
        elif diff == min_diff:
            closest_numbers.append(num)
    return closest_numbers[:k]
```

在上面的代码中，我们定义了一个名为closest_k_numbers的函数，该函数接受一个数组参数arr和一个目标值target。我们首先对数组arr进行排序。然后，我们遍历数组arr，并计算每个数与目标值的差值。我们维护一个最小差值和最接近目标值的k个数。最后，我们返回这k个数。

例如，我们可以使用分治算法来解决乘法表问题。给定一个整数n，我们需要输出一个大小为n*n的乘法表。我们可以使用分治算法来解决这个问题，如下所示：

```python
def multiply_table(n):
    if n == 0:
        return []
    else:
        top_half = multiply_table(n-1)
        bottom_half = [[n*i + j for j in range(n)] for i in range(n)]
        return top_half + bottom_half
```

在上面的代码中，我们定义了一个名为multiply_table的函数，该函数接受一个整数参数n。如果n等于0，我们返回一个空列表。否则，我们递归地调用multiply_table函数，并将结果组合成一个乘法表。
