人工智能（Artificial Intelligence，AI）是计算机科学的一个分支，研究如何让计算机模拟人类的智能。人工智能的目标是让计算机能够理解自然语言、学习、推理、解决问题、识别图像、语音识别、自主决策等。人工智能的发展将有助于提高生产力、提高生活质量、解决社会问题等。
