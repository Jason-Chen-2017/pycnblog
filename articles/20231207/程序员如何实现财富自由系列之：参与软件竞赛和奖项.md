# 4.具体代码实例和详细解释说明
在参与软件竞赛和奖项时，我们需要编写具体的代码实例来解决问题。以下是一个简单的代码实例，用于解决排序问题：

```python
def sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [5, 2, 9, 1, 3]
print(sort(arr))
```

在这个代码实例中，我们使用了冒泡排序算法来解决排序问题。冒泡排序算法的时间复杂度为O(n^2)，空间复杂度为O(1)。
