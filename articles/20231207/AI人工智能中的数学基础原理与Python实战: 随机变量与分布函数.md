# 2.核心概念与联系
随机变量是一个随机事件的结果，它可以取任意的值。随机变量的值是不确定的，因此我们需要一种方法来描述它们的分布。分布函数是随机变量的一个重要特性，它描述了随机变量的取值概率。

分布函数是一个从实数域到[0,1]的函数，它的定义域是随机变量的所有可能取值。对于每个随机变量的取值，分布函数返回一个概率值，表示该值的出现概率。

## 3.1 均匀分布
均匀分布是一种简单的分布函数，它描述了随机变量在一个固定范围内的均匀分布。均匀分布的概率密度函数为：

$$
f(x) = \frac{1}{b-a}
$$

其中，$a$ 和 $b$ 是随机变量的范围，$x$ 是随机变量的取值。

在Python中，我们可以使用`numpy`库来生成均匀分布的随机变量：

```python
import numpy as np

# 生成均匀分布的随机变量
x = np.random.uniform(a, b, size)
```

## 3.2 指数分布
指数分布是一种常见的分布函数，它描述了随机变量的指数分布。指数分布的概率密度函数为：

$$
f(x) = \frac{1}{\beta} e^{-\frac{x-\mu}{\beta}}
$$

其中，$\mu$ 是随机变量的期望值，$\beta$ 是随机变量的方差。

在Python中，我们可以使用`scipy`库来生成指数分布的随机变量：

```python
import scipy.stats as stats

# 生成指数分布的随机变量
x = stats.expon(scale=beta)
```

## 3.3 正态分布
正态分布是一种非常重要的分布函数，它描述了随机变量的正态分布。正态分布的概率密度函数为：

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$\mu$ 是随机变量的期望值，$\sigma$ 是随机变量的标准差。

在Python中，我们可以使用`numpy`库来生成正态分布的随机变量：

```python
import numpy as np

# 生成正态分布的随机变量
x = np.random.normal(loc=mu, scale=sigma, size)
```
