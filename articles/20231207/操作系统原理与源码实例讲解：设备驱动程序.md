具体代码实例：

```c
#include <linux/module.h>
#include <linux/kernel.h>
#include <linux/init.h>

static int __init my_init(void)
{
    printk(KERN_INFO "Hello, World!\n");
    return 0;
}

static void __exit my_exit(void)
{
    printk(KERN_INFO "Goodbye, World!\n");
}

module_init(my_init);
module_exit(my_exit);
```

详细解释说明：

在这个代码实例中，我们创建了一个简单的模块，用于打印“Hello, World!”和“Goodbye, World!”。这个模块的核心功能是在模块初始化时打印“Hello, World!”，在模块卸载时打印“Goodbye, World!”。

在这个代码实例中，我们使用了`module.h`、`kernel.h`和`init.h`这三个头文件。`module.h`是Linux内核中的模块头文件，用于定义模块的接口。`kernel.h`是Linux内核中的核心头文件，用于定义内核的基本数据类型和宏。`init.h`是Linux内核中的初始化头文件，用于定义模块的初始化和卸载接口。

在这个代码实例中，我们使用了`module_init()`宏来定义模块的初始化函数，`module_exit()`宏来定义模块的卸载函数。`module_init()`宏用于告诉内核，当模块被加载时，需要调用`my_init()`函数。`module_exit()`宏用于告诉内核，当模块被卸载时，需要调用`my_exit()`函数。

在这个代码实例中，我们使用了`printk()`函数来打印消息。`printk()`函数是Linux内核中的打印函数，用于将消息打印到内核日志中。
