# 2.核心概念与联系

在本节中，我们将介绍贝叶斯定理的核心概念，包括条件概率、事件、条件事件、事件的独立性、贝叶斯定理的公式等。

## 2.1 条件概率

条件概率是一个事件发生的概率，给定另一个事件已经发生。例如，如果我们知道一个人是男性，那么这个人的概率是否会患上心脏病？这个概率就是条件概率。

## 2.2 事件

事件是一个可能发生的结果。例如，一个人是男性或女性，这两个结果就是事件。

## 2.3 条件事件

条件事件是一个事件发生的条件，给定另一个事件已经发生。例如，一个人是男性，给定这个人已经患上了心脏病。

## 2.4 事件的独立性

事件的独立性是指两个事件发生的概率不受另一个事件发生的影响。例如，一个人是男性和一个人患上心脏病是两个独立的事件。

## 2.5 贝叶斯定理的公式

贝叶斯定理的公式是：P(A|B) = P(B|A) * P(A) / P(B)。这个公式表示条件概率P(A|B)，给定条件事件B发生，事件A的概率。
