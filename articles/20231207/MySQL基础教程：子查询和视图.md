## 1.1 子查询的基本概念

子查询是MySQL中一种特殊的查询语句，它可以将一个查询语句嵌套在另一个查询语句中。子查询可以用于查询数据库中的数据，并将查询结果作为另一个查询语句的一部分。子查询可以用于查询数据库中的数据，并将查询结果作为另一个查询语句的一部分。

子查询的基本语法如下：

```sql
SELECT column_name(s)
FROM table_name
WHERE column_name operator (value or subquery);
```

在这个语法中，`column_name`是要查询的列名，`table_name`是要查询的表名，`operator`是比较运算符，`value`是要比较的值，`subquery`是子查询。
