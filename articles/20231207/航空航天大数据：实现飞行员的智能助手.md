在数据预处理阶段，我们需要对飞行数据进行清洗和转换。这可以通过以下代码实现：

```python
import pandas as pd
import numpy as np

# 读取飞行数据
data = pd.read_csv('flight_data.csv')

# 数据清洗
data = data.dropna()

# 数据转换
data['speed'] = data['speed'] * 1.60934

# 数据归一化
data = (data - data.min()) / (data.max() - data.min())
```
