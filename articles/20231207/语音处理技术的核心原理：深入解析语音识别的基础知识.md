# 2. 核心概念与联系

1. 语音信号：语音信号是人类发出的声音，它是由声波组成的。声波是空气中的压力波，它们的振动速度为340米每秒。语音信号的主要特征包括频率、振幅和时间等。

2. 语音特征：语音特征是用于描述语音信号的一些数值特征。语音特征可以分为时域特征和频域特征两种。时域特征描述语音信号在时间域的变化，如振幅和时间等。频域特征描述语音信号在频域的变化，如频谱等。

3. 语音识别：语音识别是将语音信号转换为文本信息的过程。语音识别可以分为两个阶段：语音信号的预处理和语音特征的提取。语音信号的预处理是将语音信号转换为计算机可以理解的形式，如采样、量化等。语音特征的提取是将语音信号转换为数值特征的过程，如MFCC、LPCC等。

4. 语音识别算法：语音识别算法是用于将语音特征转换为文本信息的方法。语音识别算法可以分为两种：基于模型的算法和基于规则的算法。基于模型的算法使用统计学习方法来学习语音信号的模式，如HMM、GMM等。基于规则的算法使用人工规则来描述语音信号的模式，如DTM、HMM等。

5. 语音识别模型：语音识别模型是用于实现语音识别算法的计算机程序。语音识别模型可以分为两种：基于规则的模型和基于模型的模型。基于规则的模型使用人工规则来描述语音信号的模式，如DTM、HMM等。基于模型的模型使用统计学习方法来学习语音信号的模式，如HMM、GMM等。
