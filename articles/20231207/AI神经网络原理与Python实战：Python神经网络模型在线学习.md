在过去的几十年里，人工智能研究得到了很多进展，但是在1990年代末和2000年代初，人工智能研究遭到了一些挫折。这是因为人工智能研究人员试图通过创建复杂的规则来解决问题，而这种方法在许多情况下并不是最佳的。
