```python
def fibonacci(n):
    if n == 1:
        return 1
    elif n == 2:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

def main():
    n = int(input("请输入要计算的斐波那契数列的第几个数："))
    result = fibonacci(n)
    print("斐波那契数列的第{}个数为：{}".format(n, result))

if __name__ == "__main__":
    main()
```

在这个代码实例中，我们编写了一个斐波那契数列的计算程序。我们使用了递归的方法来计算斐波那契数列的第n个数。我们可以看到，代码的简洁性、可读性和可维护性都得到了很好的保障。
