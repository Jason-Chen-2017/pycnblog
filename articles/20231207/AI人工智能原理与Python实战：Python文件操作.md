# 4.具体代码实例和详细解释说明
以下是一个使用Python文件操作读取文本文件的代码实例：

```python
import os

# 打开文件
file = open('example.txt', 'r')

# 读取文件内容
content = file.read()

# 关闭文件
file.close()

# 打印文件内容
print(content)
```

这个代码首先使用`open()`函数打开名为"example.txt"的文本文件，以只读模式（'r'）打开。然后使用`read()`方法读取文件内容，并将其存储在`content`变量中。最后，使用`close()`方法关闭文件，以释放系统资源。最后，使用`print()`函数打印文件内容。
