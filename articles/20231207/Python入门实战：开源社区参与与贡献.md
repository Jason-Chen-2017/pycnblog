```python
def calculator(num1, num2, operator):
    if operator == '+':
        return num1 + num2
    elif operator == '-':
        return num1 - num2
    elif operator == '*':
        return num1 * num2
    elif operator == '/':
        return num1 / num2
    else:
        return 'Invalid operator'

num1 = 5
num2 = 3
operator = '+'
result = calculator(num1, num2, operator)
print(result)
```

4.1.2 排序算法：排序算法是一种常见的Python代码实例，它可以用来对数据进行排序。排序算法的代码实例如下：

```python
def sort(arr):
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] > arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
sorted_arr = sort(arr)
print(sorted_arr)
```
