# 4.具体代码实例和详细解释说明

在本节中，我们将提供一个具体的 Spring Boot 单元测试代码实例，并详细解释其中的每个部分。

```java
import org.junit.Test;
import org.junit.runner.RunWith;
import org.springframework.boot.test.context.SpringBootTest;
import org.springframework.test.context.junit4.SpringRunner;

@RunWith(SpringRunner.class)
@SpringBootTest
public class MyTest {

    @Test
    public void testAdd() {
        int a = 1;
        int b = 2;
        int expected = 3;
        int actual = a + b;
        if (expected == actual) {
            System.out.println("测试通过");
        } else {
            System.out.println("测试失败");
        }
    }
}
```

在上述代码中，我们创建了一个名为 MyTest 的测试类，并使用 @Test 注解将其标记为测试方法。我们还使用 JUnit 编写了一个测试用例，该用例涉及到两个整数的加法运算。我们预期的结果是 3，实际结果是 3，因此测试通过。
