```python
import hashlib
import hmac
import base64

def encrypt_data(data, key):
    # 使用AES算法对数据进行加密
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(data)
    return cipher.nonce + tag + ciphertext

def decrypt_data(ciphertext, key):
    # 使用AES算法对数据进行解密
    cipher = AES.new(key, AES.MODE_EAX, nonce=ciphertext[:16])
    data = cipher.decrypt_and_verify(ciphertext[16:])
    return data

def generate_hmac(data, key):
    # 使用HMAC算法对数据进行签名
    digest = hmac.new(key, data, hashlib.sha256).digest()
    return base64.b64encode(digest)

def verify_hmac(data, signature, key):
    # 使用HMAC算法对数据进行验证
    digest = hmac.new(key, data, hashlib.sha256).digest()
    if base64.b64encode(digest) == signature:
        return True
    else:
        return False
```

在这个代码实例中，我们使用了AES算法对数据进行加密和解密，以及HMAC算法对数据进行签名和验证。AES算法是一种对称加密算法，可以用于保护数据在传输和存储过程中的安全性。HMAC算法是一种基于哈希函数的数字签名算法，可以用于保护数据在传输过程中的完整性和身份验证。
