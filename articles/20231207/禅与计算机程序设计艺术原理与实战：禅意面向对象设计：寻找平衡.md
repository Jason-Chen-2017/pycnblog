# 2.核心概念与联系

禅意面向对象设计的核心概念是将禅意哲学与面向对象编程相结合。禅意哲学强调直接体验现实，而不是依赖理论和概念。而面向对象编程则是一种编程范式，将程序划分为多个对象，每个对象都有其自己的属性和方法。

在禅意面向对象设计中，我们需要将禅意哲学的思想应用到编程过程中。这意味着我们需要直接体验程序的运行过程，而不是依赖于代码的细节。同时，我们需要将面向对象编程的思想应用到禅意哲学中，将程序划分为多个对象，每个对象都有其自己的属性和方法。
