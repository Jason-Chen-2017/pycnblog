## 4.具体代码实例和详细解释说明

在学习Python网络安全与防御技术时，我们需要看到和编写一些具体的代码实例，以便更好地理解和掌握这些技术。以下是一个简单的Python网络安全与防御代码实例：

```python
import hashlib

def md5_hash(data):
    md5 = hashlib.md5()
    md5.update(data.encode('utf-8'))
    return md5.hexdigest()

if __name__ == '__main__':
    data = 'Hello, World!'
    print(md5_hash(data))
```

在上述代码中，我们使用Python的hashlib模块来计算数据的MD5哈希值。MD5哈希值是一种常用的数据校验方法，用于确保数据的完整性和安全性。
