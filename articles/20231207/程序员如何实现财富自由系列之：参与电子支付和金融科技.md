# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的电子支付示例来展示如何实现电子支付的核心功能。

```python
import hashlib
import hmac
import os
import base64

# 生成密钥
key = os.urandom(16)

# 生成支付信息
message = "支付信息"

# 生成数字签名
signature = hmac.new(key, message.encode(), hashlib.sha256).digest()

# 编码数字签名
encoded_signature = base64.b64encode(signature)

# 发送支付信息和数字签名
data = message + "|" + str(encoded_signature, "utf-8")

# 接收方验证数字签名
received_signature = data.split("|")[1]
decoded_signature = base64.b64decode(received_signature)

# 验证数字签名的完整性
verified_signature = hmac.new(key, message.encode(), hashlib.sha256).digest()

if verified_signature == decoded_signature:
    print("数字签名验证通过")
else:
    print("数字签名验证失败")
```

在这个示例中，我们使用了HMAC-SHA256算法来生成和验证数字签名。首先，我们生成了一个密钥，然后生成了一个支付信息。接着，我们使用HMAC-SHA256算法对支付信息进行签名，并将签名编码为base64格式。最后，我们将支付信息和编码后的签名发送给对方。对方接收后，将签名解码并使用相同的HMAC-SHA256算法对支付信息进行验证，以确认签名的完整性。
