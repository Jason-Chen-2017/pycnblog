# 4.具体代码实例和详细解释说明
在这里，我们将提供一个具体的代码实例，以及对其解释的详细说明。

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建一个简单的社交网络
G = nx.Graph()
G.add_nodes_from(['Alice', 'Bob', 'Charlie', 'David'])
G.add_edges_from([('Alice', 'Bob'), ('Alice', 'Charlie'), ('Bob', 'Charlie'), ('Charlie', 'David')])

# 计算每个节点的度
degree_distribution = nx.degree_distribution(G)
print(degree_distribution)

# 绘制度分布图
plt.bar(degree_distribution.keys(), degree_distribution.values())
plt.xlabel('Degree')
plt.ylabel('Frequency')
plt.title('Degree Distribution')
plt.show()
```

在这个代码实例中，我们使用Python的networkx库来创建一个简单的社交网络。我们首先创建一个空的图对象，然后使用`add_nodes_from`和`add_edges_from`方法来添加节点和边。接下来，我们使用`nx.degree_distribution`方法来计算每个节点的度，并将结果打印出来。最后，我们使用`matplotlib.pyplot`库来绘制度分布图。
