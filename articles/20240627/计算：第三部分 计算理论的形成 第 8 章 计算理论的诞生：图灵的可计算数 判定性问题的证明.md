## 2. 核心概念与联系

### 2.1 图灵机

图灵机是一种抽象的计算模型，由图灵在1936年提出。图灵机由一个有限状态机、一个带读写头的无限长纸带以及一系列的规则组成。图灵机能够通过一系列的读写操作，在纸带上移动头，并根据当前的状态和符号进行计算。

### 2.2 可计算数

可计算数是指那些可以通过图灵机进行计算或判断的数。一个数如果能够被图灵机在有限时间内接受或拒绝，那么它就是一个可计算数。

    def __init__(self, states, alphabet, transition_function, initial_state, initial_symbol, final_state):
        self.states = states

        self.transition_function = transition_function
        self.current_state = initial_state

        transition = self.transition_function.get((self.current_state, self.current_symbol))
        if transition is None:
            return False  # 无对应转移规则
        self.current_state, self.current_symbol, direction = transition
        if direction == 'R':
            self.current_symbol = self.tape.pop(0)
            self.tape.append('B')
        elif direction == 'L':
            self.current_symbol = self.tape.pop()
            self.tape.insert(0, 'B')
        return True

    (0, 'B'): (0, 'B', 'R'),
    (0, '0'): (1, '0', 'R'),
    (1, '0'): (1, '0', 'R'),
    (1, '1'): (2, '1', 'R'),
    (2, '1'): (2, '1', 'R'),
    (2, 'B'): (0, 'B', 'L'),
}

# 创建图灵机实例

    states=[0, 1, 2],

    transition_function=transition_function,
    initial_state=0,

    final_state=0

while tm.step():
    pass
