# 2.基本概念术语说明
## 2.1 R语言概述
R (The R Project for Statistical Computing)是一种开源的统计分析、绘图及数据操控环境。它是GNU S实现的自由软件，由数学家Edward Tufte先生于1995年在纽约的梅陇大学开发出来。R是一个强大的命令行界面和脚本语言环境，提供很多统计、绘图以及数据操控功能，通过提供简单而易用的语言构造，可以快速进行数据处理、分析和建模工作。

## 2.2 数据类型及相关概念
### 2.2.1 数据类型
数据类型指的是数据的属性（质料、大小、结构等），其分类主要基于存储的特点，按照数据在计算机中的表示形式不同分为以下几类：

1. 标称型数据：又叫定型变量，数据取值只有两种或两种以上，例如性别、颜色、级别等。

2. 有序型数据：又叫顺序变量，具有一定顺序性，数据间存在上下限之分。如分数、成绩等。

3. 标量型数据：也叫单个数值变量，是一个数字或一个字符型的值。如身高、体重、年龄、姓名等。

4. 数组型数据：多维度数据，通常表现为二维或三维矩阵。

5. 矩阵型数据：是一种特殊的数组型数据，每个元素都是相同的数据类型。

6. 数据框型数据：是一种比较复杂的数据结构，将多个变量按列排列，每个变量都有自己的名称和数据类型。

### 2.2.2 维度
维度(Dimension)是指数据的多少个属性，即数据集里所含有的变量个数。数据集的维度可以从数据结构来定义，对于矩阵型数据，其维度一般为2或3；对于数据框型数据，则是指有多少个变量。

## 2.3 可视化简介
数据可视化(Data Visualization)是利用信息图象对数据进行清晰地呈现和理解。数据可视化既要有创造性又要简洁直观，能够让人更容易理解数据的规律，对分析结果产生积极的影响。数据可视化的目标就是用图形符号来显示数据，帮助人们发现数据之间的联系和联系。

数据可视化的类型主要分为四种：
- 静态图表(Static Charts): 在报告或网络文档中使用的图表，需要非常精确的坐标轴刻度，适合较小数量的数据，如条形图、折线图、柱状图等。
- 演示图表(Infographics): 是采用动画技术将数百万个数据点动态地投射到屏幕上的图片。这种方法突出数据的变化、关系和分布特征，而非静态图像。
- 交互式图表(Interactive Charts): 可以与用户互动的图表，允许用户探索和分析数据，并且可以根据数据的变化实时更新。例如：棒棒糖分析软件的销售额走势图表就是典型的交互式图表。
- 富媒体可视化(Rich Media Visualizations): 使用视频、音频、图像、文本等来增强数据可视化的内容，通过对数据的呈现方式进行整体设计，可以提供更具说服力的洞察和分析。
