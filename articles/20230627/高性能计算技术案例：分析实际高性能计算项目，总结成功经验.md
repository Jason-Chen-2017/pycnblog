4.2. 应用实例分析
以下是一个使用TensorFlow实现的高性能计算示例。以大规模线性回归问题为例，问题描述为：给定一组特征，预测一个连续的输出值。
```
import tensorflow as tf
from tensorflow import keras

# 读取数据
data = keras.datasets.load_iris(return_X_y=True)

# 数据预处理
data = data.astype('float32') / 255.0
data = (data - 0.5) * 4
data = (data + 3) * 8

# 划分训练集和测试集
train_data = data[:int(data.shape[0] * 0.8)]
test_data = data[int(data.shape[0] * 0.8):]

# 构建模型
model = keras.models.Sequential([
    keras.layers.Dense(10, input_shape=(4,), activation='relu'),
    keras.layers.Dense(1)
])

# 编译模型
model.compile(optimizer='adam',
              loss='mse',
              metrics=['accuracy'])

# 训练模型
model.fit(train_data, epochs=50, batch_size=1, validation_split=0.1)

# 评估模型
test_loss, test_acc = model.evaluate(test_data)

# 使用模型进行预测
predictions = model.predict(test_data)
```
