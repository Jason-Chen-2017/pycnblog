        self.last_data = data
        self.last_time = time.time()

    def check_error(self):
        self.last_data = None
        self.last_time = None

    def update(self):
        if not self.last_data:
            print("Sensor not working!")
            return
        data = self.last_data
        self.last_data = None
        self.last_time = time.time()

        if self.last_time - self.last_data > 100:
            self.read_data()

本实例中使用的传感器数据采集类（Sensor）负责从各种传感器中获取数据，并将其发送到数据存储系统中。该类的方法有：
