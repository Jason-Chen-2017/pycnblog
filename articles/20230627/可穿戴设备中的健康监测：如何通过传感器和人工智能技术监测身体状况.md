### 4.3. 核心代码实现

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

class WearableData:
    def __init__(self, device, sensors, data_length):
        self.device = device
        self.sensors = sensors
        self.data_length = data_length

    def sync(self):
        for sensor in self.sensors:
            self.device.send(sensor, [0] * self.data_length)

    def measurement(self):
        for sensor in self.sensors:
            value = self.device.read(sensor)
            self.sensors.append(value)

    def prepare_data(self):
        data = []
        for sensor in self.sensors:
            data.append(self.device.read(sensor))
        return np.array(data)

    def train_model(self, model):
        X = self.prepare_data()
        y = np.array([self.device.read(sensor) for sensor in self.sensors])
        model.fit(X, y)
```
### 4.4. 代码讲解说明

上述代码实现了一个简单的可穿戴设备，包括心率传感器、血压传感器和运动传感器等。在测量数据时，通过`send`函数将数据发送至设备，然后通过`read`函数读取传感器数据。在数据处理时，使用`prepare_data`函数对传感器数据进行处理，包括数据清洗和数据采样等。在模型训练时，使用`fit`函数对数据进行训练，并返回模型的训练结果。
