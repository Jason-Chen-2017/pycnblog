4.3. 核心代码实现

这里以 Python 和 PyTorch 为例，实现一个简单的机器学习算法，用于预测用户购买某个商品的概率。

```python
import torch
import torch.nn as nn
import torch.optim as optim

class MLP(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super(MLP, self).__init__()
        self.fc1 = nn.Linear(input_dim, hidden_dim)
        self.fc2 = nn.Linear(hidden_dim, output_dim)

    def forward(self, x):
        out = torch.relu(self.fc1(x))
        out = self.fc2(out)
        return out

# 训练模型
model = MLP(input_dim, hidden_dim, 1)
criterion = nn.BCELoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# 训练数据
train_x = torch.randn(1000, input_dim)
train_y = torch.randn(1000, 1)

# 训练模型
for epoch in range(100):
    for inputs, targets in zip(train_x, train_y):
        outputs = model(inputs)
        loss = criterion(outputs, targets)
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
```

4.4. 代码讲解说明

上述代码实现了一个 MLP 模型，该模型可以用于预测用户购买某个商品的概率。其中，`input_dim` 表示输入数据的特征维度，`hidden_dim` 表示模型的隐藏层维度，`output_dim` 表示输出的预测维度。

首先，使用 PyTorch 的 `nn.Module` 类，定义一个包含一个线性层和一个全连接层的 MLP 模型。其中，线性层用于将输入数据转化为模型的输入，全连接层用于输出模型的预测结果。

接着，在 `forward` 方法中，实现输入数据的 forward 计算过程，并返回模型的输出。

最后，在训练模型时，使用随机梯度下降 (SGD) 算法来更新模型的参数，以最小化损失函数。
