4. 应用示例与代码实现讲解

4.1. 应用场景介绍

蒙特卡罗方法可以用于各种随机化模拟，如模拟股票价格、模拟天气变化、模拟生物进化等。

4.2. 应用实例分析

假设我们要模拟股票价格的随机波动。我们可以使用蒙特卡罗方法来生成一组股票价格，然后根据生成的价格计算出平均值和方差，从而得到概率分布。

4.3. 核心代码实现

```python
import random
import numpy as np

# 生成随机数
rand_num = random.randint(0, 1)

# 生成一组股票价格
prices = [rand_num for _ in range(1000)]

# 计算平均值和方差
mean = np.mean(prices)
std = np.std(prices)

# 输出结果
print("平均值: ", mean)
print("方差: ", std)
```

4.4. 代码讲解说明

上述代码使用 Python 语言实现了蒙特卡罗方法，用于生成一组随机股票价格并计算平均值和方差。
