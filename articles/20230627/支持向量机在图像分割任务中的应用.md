支持向量机是一种监督学习分类算法，其主要思想是将数据映射到高维空间，通过二元变量（0/1）表示数据中的正面/负面实例，并利用核函数将数据映射到高维空间，实现数据之间的分类。

支持向量机需要一个训练数据集，用于训练分类器模型。数据集中的样本包括训练数据和测试数据，其中训练数据用于训练模型，测试数据用于评估模型的准确率。

4.4. 代码实现
------------

代码实现如下所示：
```python
import numpy as np
import pandas as pd
import os
import sys
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from sklearn.metrics import classification_report
import matplotlib.pyplot as plt

# 读取数据集
iris = load_iris()

# 对数据集进行归一化处理
scaler = StandardScaler()
iris_scaled = scaler.fit_transform(iris.data)

# 将数据集拆分为训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(iris_scaled.toarray(), iris.target, test_size=0.3)

# 建立支持向量机模型
clf = SVC()
clf.fit(X_train, y_train)

# 对测试集进行预测
y_pred = clf.predict(X_test)

# 输出分类结果
print(classification_report(y_test, y_pred))

# 输出模型训练结果
print(clf.score(X_test, y_test))
```
