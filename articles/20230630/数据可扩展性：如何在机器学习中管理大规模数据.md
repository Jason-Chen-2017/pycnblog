4.3. 核心代码实现
下面是一个使用Hadoop MapReduce对数据进行分布式处理的Python代码实现:

```python
import pandas as pd
from pyspark import SparkConf, SparkContext

# 读取数据
data = pd.read_csv('input.csv')

# 分布式处理
conf = SparkConf().setAppName("Data Processing")
sc = SparkContext(conf=conf)

# 分解数据
data = data.map(lambda row: (row[0], row[1]))
data = data.reduceByKey((lambda x, y: x + y))

# 输出结果
data.write.mode('overwrite').csv('output.csv', test_mode=True)
```
