```
import numpy as np

class CoreModule:
    def __init__(self, input_size, hidden_size, output_size):
        self.weights1 = np.random.randn(input_size, hidden_size)
        self.bias1 = np.zeros((1, hidden_size))
        self.weights2 = np.random.randn(hidden_size, output_size)
        self.bias2 = np.zeros((1, output_size))

    def forward(self, X):
        self.z1 = np.dot(X, self.weights1) + self.bias1
        self.a1 = np.tanh(self.z1)
        self.z2 = np.dot(self.a1, self.weights2) + self.bias2
        self.a2 = self.softmax(self.z2)
        return self.a2

    def softmax(self, X):
        exp_X = np.exp(X)
        return exp_X / np.sum(exp_X, axis=1, keepdims=True)
```
4. 代码讲解说明
以上代码实现了一个简单的神经网络模型，该模型包含输入层、隐藏层和输出层。输入层接受客户的ID和信用评级等信息，隐藏层则对输入层的信息进行加权和激活，输出层输出客户的信用评分。其中，`weights1`和`bias1`分别表示输入层和隐藏层中权重和偏置的值，`weights2`和`bias2`同理。

在`forward`函数中，首先将客户的ID和信用评级等信息输入到输入层中，并计算出对应的隐藏层的输入值。接着，将隐藏层的输入值代入一个多层感知器中，并计算出对应的输出值。最后，使用`softmax`函数将输出值转化为概率分布，从而得到客户的信用评分。
