4.3. 核心代码实现

以下是一个简单的计算机视觉应用示例，用于在视频中查找人脸。代码主要涉及OpenCV库、numpy库以及Python编程语言。

```python
import cv2
import numpy as np

# 加载视频
cap = cv2.VideoCapture("example.mp4")

# 循环读取视频
while True:
    ret, frame = cap.read()

    # 转换为灰度图像
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

    # 检测人脸
    faces = cv2.findContours(gray, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

    # 在视频上绘制矩形框
    for (x, y, w, h) in faces:
        cv2.rectangle(frame, (x, y), (x + w, y + h), (0, 255, 0), 2)

    # 显示视频
    cv2.imshow("Video", frame)

    # 按 'q' 键退出循环
    if cv2.waitKey(1) & 0xFF == ord("q"):
        break

# 释放资源
cap.release()
cv2.destroyAllWindows()
```
