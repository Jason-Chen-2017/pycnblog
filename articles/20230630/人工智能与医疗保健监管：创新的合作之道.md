4.4. 代码讲解说明

下面是一个简单的 Python 代码，可以实现医疗数据的分析和挖掘，以获得有价值的信息。
```python
import pandas as pd
import numpy as np

# 读取医疗数据
hospital_data = pd.read_csv('hospital_data.csv')

# 定义变量
df = pd.DataFrame(hospital_data)

# 计算平均值
mean = df.mean()

# 计算标准差
std = df.std()

# 输出结果
print('平均值:', mean)
print('标准差:', std)
```
