3.2. 核心模块实现
--------------------

C++11 和 C++14 标准库包含了许多核心模块，例如：

* std::vector：一种动态数组，支持向量和迭代器。
```
#include <vector>

int main() {
    std::vector<int> vec = {1, 2, 3, 4, 5};
    for (const auto& e : vec) {
        std::cout << e <<'';
    }
    return 0;
}
```

* std::map：一种键值对容器，支持快速查找和插入。
```
#include <map>

int main() {
    std::map<std::string, int> map = {
        {"key1": 1, "key2": 2},
        {"key3": 3, "key4": 4},
        {"key5": 5, "key6": 6}
    };
    map["new_key"] = 6;
    for (const auto& e : map) {
        std::cout << e.first <<'' << e.second <<'';
    }
    return 0;
}
```

* std::unordered\_map：一种键值对容器，支持快速查找和插入，并提供了更好的读性能。
```
#include <unordered_map>

int main() {
    std::unordered_map<std::string, int> map = {
        {"key1": 1, "key2": 2},
        {"key3": 3, "key4": 4},
        {"key5": 5, "key6": 6}
    };
    map["new_key"] = 6;
    for (const auto& e : map) {
        std::cout << e.first <<'' << e.second <<'';
    }
    return 0;
}
```

* std::vector\_concept：一种概念，用于定义模板的要求。
```
#include <vector_concept>

int main() {
    std::vector<int> vec = {1, 2, 3, 4, 5};
    for (const auto& e : vec) {
        std::cout << e <<'';
    }
    return 0;
}
```

* std::is\_same：一个函数，用于比较两个类型的相似性。
```
#include <is_same>

int main() {
    int a = 1;
    int b = 2;
    if (is_same(a, b)) {
        std::cout << "a and b are equal" << std::endl;
    } else {
        std::cout << "a and b are not equal" << std::endl;
    }
    return 0;
}
```
