## 2. 技术原理及概念

2.1. 基本概念解释

关系型数据库 (RDBMS) 是一种按照行顺序存储数据的数据库。在 RDBMS 中，数据以表的形式进行组织，其中表由行和列组成。每个行包含一个或多个数据列，每个列包含一个数据类型和一个或多个数据值。

关系型数据库使用 SQL (结构化查询语言) 语句对数据进行查询和操作。SQL 是一种用于管理关系型数据库的标准语言。它允许用户创建、查询、更新和删除数据库中的数据。
