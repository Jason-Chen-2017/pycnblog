# 2.计算机视觉（CV）
## 2.1 场景理解模块（Scene understanding module）
### 2.1.1 VGGNet-19
VGGNet-19，第一个提出并公布了基于CNN的深度学习模型架构的网络。它由多个卷积层组成，每层都有多个卷积核。卷积核个数逐渐增加，并且每次都采用最大池化的方式下采样特征图，使得特征图越来越小，并减少参数数量。最后再连接一个全连接层。它的结构如下所示： 


通过重复堆叠同样的结构，可以产生不同的网络结构，例如VGG-11和VGG-16等。在CIFAR-10数据集上，VGGNet-19可以达到92%的测试准确率。

### 2.1.2 GoogleNet
GoogleNet，2014年被提出并被证明比AlexNet更有效。它将多个卷积层通过inception block堆叠，每个block中含有多个卷积层。每个inception块的前两个卷积层的通道数相差不大，以保证一定程度上的信息流通；中间的卷积层只有一个输出通道，以减少计算量；后面的1x1卷积层用于降维，确保各个卷积层输出的通道数相同。它的结构如下图所示：


GoogleNet在ILSVRC-2014图像识别挑战赛上取得了最高奖项。

### 2.1.3 ResNet
ResNet，2015年被提出，在解决梯度消失和梯度爆炸的问题的基础上提出了残差学习。它将残差单元引入网络结构，目的是增加非线性变换层带来的耦合能力，提升网络的表达能力。它的结构如下图所示：


ResNet在多个任务上取得了最好的结果，包括ImageNet分类、物体检测、人脸识别。

## 2.2 检测和分割模块（Detection and segmentation module）
### 2.2.1 Faster R-CNN
Faster R-CNN，2015年被提出，是一种多阶段目标检测方法。它首先使用卷积神经网络（CNN）提取图像特征，然后将得到的特征输入到两个不同但相关的模块中。第一个模块生成候选区域（Region of Interest），第二个模块使用预训练的卷积神经网络进一步分类和回归这些候选区域。它的结构如下图所示：


Faster R-CNN在PASCAL VOC 2007和2012上取得了state-of-the-art的性能。

### 2.2.2 Mask R-CNN
Mask R-CNN，是在Faster R-CNN的基础上添加了一个分割模块，能够直接输出目标的边界框、类别标签及像素级的掩膜掩码。它的结构如下图所示：

### 2.2.3 YOLO
YOLO，2016年被提出，是一个单独分支的卷积神经网络。它不需要使用FC层或者RNN，因此速度很快。它的主要思想是用预定义的anchor box和滑动窗口对输入图片进行预测，通过判断预测框和真实框之间的IOU大小来进行评估。它的结构如下图所示：

## 2.3 文本理解模块（Text understanding module）
### 2.3.1 Recurrent Neural Network with Long Short-Term Memory (LSTM)
LSTM，是一种递归神经网络，可以保留之前的信息。它接收输入序列的序列数据，通过隐藏状态和记忆细胞控制其输出，可以利用上下文信息进行预测。它的结构如下图所示：


LSTM在很多自然语言处理任务上都取得了最好效果。

### 2.3.2 Convolutional Neural Networks for Sentence Classification
卷积神经网络（CNNs）对于文本分类任务来说很受欢迎，如IMDB电影评论分类任务。它的工作流程如下：

1. 将文本转换为固定长度的向量表示形式（embedding）。
2. 通过卷积层提取句子特征。
3. 使用池化层对句子特征进行局部汇总。
4. 通过全连接层进行分类。

CNNs能够自动提取词语级别的特征。
