# 1.简介
  

“How to Become an Entrepreneur?” 是一本由麻省理工大学林耀宏教授撰写的一本经典的商业管理、投资分析、创新模式、创业指导和个人成功法则等领域的指南。作为一名专业的技术人员、产品经理或项目经理，需要学习并掌握如何才能成为一个卓越的企业家。这也是本书的作者受到社会各界追捧的原因之一。本书的作者将其五年的时间的独自创业经验及对互联网的热衷，透过对商业、创业、管理的理解和研究，系统整合了创业、发展及管理的各个方面，并且结合创业公司的经验，帮助读者树立起正确的创业观念，更好地做出决定、开拓事业、展望未来。而在介绍完商业、投资、创新、创业等各个方面之后，还附有作者独到的个人成功指南。
# 2.作者简介
林耀宏，麻省理工大学商学院博士，现任麻省理工科技工程师（CTO），曾任阿里巴巴集团创始人兼CEO、百度创始人，并担任多家顶尖公司的副总裁。他研究方向包括管理科学、组织行为学、竞争机制、商业模式等，是美国最具影响力的经济学家和管理学专家。在此期间，他通过持续不断的创业实践，取得丰硕的财富，成为世界顶尖创业者之一。林耀宏深知作为企业家的角色至关重要，所以他十分重视个人奋斗的能力，对于新生事物、管理创新的精益求精，以及心胸宽阔的解决问题的能力，都十分赞赏。他对创业成长的道路上，存在着一定的阻碍和风险，但他仍然坚信，只要我们肯付出努力，克服困难，把这些困难消除掉，就一定可以实现创业的目标。
