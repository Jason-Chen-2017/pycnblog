# 2. 机器学习与深度学习概述

## 2.1 概念
### 定义
**机器学习**(Machine Learning) 是一类计算机科学研究方法，它是一种基于数据、模式、算法的系统化学习方法，使得计算机具有自适应性和灵活性，并快速处理复杂的数据、任务或者场景。机器学习方法从训练集中学习规律，从而对输入数据预测相应的输出结果。

**深度学习**(Deep Learning) 是一类人工智能的技术，它利用多层神经网络结构，使用各种优化手段，训练得到多个深度神经网络，通过组合这些不同的神经网络，可以解决复杂的问题。

## 2.2 发展历史
### 早期机器学习
在1959年，罗宾·加西亚·马尔可夫提出的"贝叶斯统计学"被认为是西蒙曼语的一个里程碑。他提出的贝叶斯估计方法直接应用于工程实践当中，其表现优异。1962年，卡内基梅隆大学的史玫柏和李·埃弗莱特（Leon Effiancio）发现两个相互独立的事件之间存在强大的关联关系。他们因此提出了著名的"感知器"（perceptron）模型，用来表示和分类输入变量。

1980年，麻省理工学院的李·萨莫（Lisa Sorower）提出了"支持向量机"（support vector machine，SVM），这是一种二类分类模型。他成功地将非线性分类任务转化成线性问题，通过求解凸优化问题求得最优解。1995年，同济大学的周志华教授发现，支持向量机算法能够有效地处理高维空间中的非线性数据，并且可以在样本容量很小的情况下仍然表现良好。

随着统计学、信息论、最优化理论、人工神经网络、模式识别和图像处理等学科的不断突破，机器学习已经成为当今最热门的研究方向之一。

### 深度学习
深度学习的前身是神经网络。1943年，罗伯特·皮茨（Ralph McCulloch）及其同事提出的神经网络模型，是深度学习的起源。1986年，约翰·欧文（John O.E.Collins）及其同事证明了多个感知器可以组成高度非线性的函数。1987年，李·贺东（T<NAME>hong）、谢益辉（<NAME>）、陈天奇（Chien-Shiung Chan）及其同事提出了深层网络结构，推动了深度学习的发展。

近几年，随着深度学习的火热，机器学习变得越来越关注性能、效率和泛化能力，而深度学习则是一种新的学习方法，它具有较好的理论基础、高效的计算能力和广泛的适用范围，被认为是一种跨越式增长的技术。

## 2.3 分类
机器学习的基本分类有监督学习、半监督学习、无监督学习和强化学习等。其中，监督学习指的是使用已标注的数据来训练学习模型，学习过程中需要监督学习者提供有关数据的正确标签或结果。例如，可以训练一个分类器，将输入的图片分类为猫或狗；也可以训练一个回归器，根据房屋的平面图面积和位置，预测其价格。

半监督学习则是在有限的标注数据情况下，结合少量的大量数据来训练学习模型。该数据既有有用的信息，又有噪声。例如，可以使用聚类算法将图像中的对象分为若干个类别，然后使用这些类的信息去标记缺失的标记数据。此外，半监督学习还可以根据已有的知识建立先验知识，对数据进行分类或回归。

无监督学习则是指没有标签的数据，仅依靠特征提取技术进行数据聚类。例如，可以使用聚类算法自动发现数据之间的联系，并进行分类或回归。

强化学习则是一种机器学习问题，即让机器在不断试错的过程中学习到最佳策略。它的特点是反馈机制，以获得最大的奖励，而不是像其他机器学习方法那样只认识到目标。

除此之外，还有聚类、推荐系统、生成模型等其他类型的机器学习问题。
