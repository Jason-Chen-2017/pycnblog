# 1.简介
  

光刻胶（PCB）是数字化设计中最常用的成果之一，在制作电路板、IC、器件、模块等高频设备时作为主导材料。
PCB的制造涉及到对亚太技术、工艺、设计、制造、测试等领域的跨界能力，同时也需要特定的技能和工具。本文将详细介绍常见光刻胶工艺过程中两极效应产生原因、表现形式以及对应的防御措施。
光刻胶的制造过程包括了烧结、腐蚀、电泳、预处理、镀锌、冷削、切割、焊接、镭射、组装和测试等七个环节，而制品质量也直接影响最终的产品性能。
而对于光刻胶工艺中的两极效应来说，主要表现为：
- 反向聚焊(Backlash)
- 滞后(Lagging)
两者均会导致焊盘表面出现较大的积孔或凹陷，最终导致产品制造过程的减速或产品性能下降。
光刻胶工艺中的两极效应并非一成不变，其原因仍然是相关环节存在缺陷。因此，为了更好的保护客户，制造商需要有针对性的调整工艺流程，提升产品质量，降低两极效应对生产效率的影响。
# 2.相关概念
## 2.1.反向聚焊(Backlash)
反向聚焊是指由两个电极、多层金属或塑料组成的结构发生短时间内相互排斥作用而导致的电流回流。这一现象通常被称为“开关后反弹”，意思是当某种情况下的交流信号输入电路后，另一种情况下的信号输出出现反转现象。这种现象的出现通常是由于在不同驱动电压的作用下，双极微粒或多极粒体之间所形成的场力大小差异导致的。
反向聚焊现象一般存在于采用铜箔、镍箔或氢箔制成的金属电极之间，甚至可以跨越多个纤维层，如铝合金或绝缘材料等。它还可能发生在由于导通线路上的失真、喷头阻抗等导致的导通效应。
反向聚焊现象在整个制造工艺过程中都有重要的影响，其严重程度远比毛刺、刮花更加危险。因为这些现象往往会引起制造机械精度不稳定、制品温度过热、工作效率下降甚至产品缺陷。但在处理反向聚焊现象时，可以通过一些方法来避免或者减少。
## 2.2.滞后(Lagging)
滞后是指多个电极之间的气流方向偏离平均值，导致有的电流慢慢回流，有的快得没有回流速度。这种现象主要出现在两条或多条电话电缆上。它的产生往往是由于导电材料的导电率不够，导致电流传递的时间过长，使得电流处于不稳定状态。
虽然滞后现象不能完全消除，但是通过适当的配套措施比如降低功率、使用铁芯或强化导电材料等，可以缓解大部分情况。但由于工艺复杂度的增加、设备的改动以及人为因素的干扰，滞后现象仍然是造成制造过程畸变、物料移位、缺陷扩散和产品毛坯化的重要原因之一。
# 3.核心算法原理和具体操作步骤以及数学公式讲解
光刻胶的制造过程包括烧结、腐蚀、电泳、预处理、镀锌、冷削、切割、焊接、镭射、组装和测试等七个环节。下面将介绍其中三个环节：预处理、镀锌和组装。
### 3.1.预处理
预处理是指光刻胶制造前的准备工作，主要目的是去除胶粒、渣土、气泡等杂质，提高光刻胶的整理、涂层的成熟速度，进而加快光刻胶的制作。
具体操作步骤如下：
1. 脱水处理：将光刻胶浸在室温环境里，通过蒸馏或蒸发的方式将渣土去除掉。
2. 分层、去除杂质：通过分层的方法将胶粒从表面和内部分离出来，然后去除杂质，如虫子、泥沙、氨气等。
3. 消除小颗粒：用小型照明弹簧移除胶粒的残余，使胶粒均匀分布。
4. 清洗液进行干燥：通过微波炉或风扇将胶粒洗净，并在相应的过滤液中进行冲洗。
5. 去加熱：将胶粒添加适量的热量，将其从冰层中熄掉。
6. 放置干燥阴影：将将胶粒放置在干燥的环境中，以减少吸收微生物的可能性。
7. 投放胶层：将烧完的胶层放在适当的温度、湿度和条件下，待胶层融合。
### 3.2.镀锌
镀锌是一个精心设计的过程，通过分解离子、挤出高锰酸盐、破碎、增锌、晾晒、过滤、混合等一系列操作，才可能得到产品的最终结果。其中，增锌和晾晒、过滤是最关键的环节。
具体操作步骤如下：
1. 分解离子：首先将离子分解为聚丙烯、苯丙烯、乙二醇和铬，并使用离子注入试剂调节离子分解速率。
2. 挤出高锰酸盐：然后将挤出高锰酸盐。挤出高锰酸盐的方法有两种：一种是将高锰酸盐和普通锌离子一起分解，另一种是将高锰酸盐分解为普通锌离子和异铬离子，异铬离子具有很好的挤出效果。
3. 破碎：将挤出的高锰酸盐和矿石、杂质一起分解。
4. 增锌：将破碎后的高锰酸盐和普通锌离子混合，增锌至一定含量，并加入碱性化合物，如三氟甲烷、四氟化碳钠等。
5. 晾晒：将增锌后的高锰酸盐晾晒至室温，待高锰酸盐完全结晶化。
6. 过滤：将晾晒后的高锰酸盐和油墨一起过滤，过滤后残余的高锰酸盐及碱性化合物可用于进一步的合成。
7. 混合：将滤出的高锰酸盐及碱性化合物混合，并经过筛选后获得晶粒。
### 3.3.组装
组装的目的就是把光刻胶的晶粒结合到一起，组成可用的PCB板、集成电路、电容器件、印刷电路板、电阻器件等各种应用产品。组装一般要经历以下几个环节：

1. 滤网：滤网是组装PCB板、集成电路等产品的一道重要环节，用于控制光滑度、透视度、凹凸程度以及其他属性。
2. 压型：压型是用来固定材料到铝箔上的。材料需要预先用压电钻或锤压平，压型就相当于这个过程。
3. 滤制：滤制是将材料经过涂层后表面的锡层和铜芽完全溶解，然后使用面包膜或其它材料，将材料膨胀并定位在铝箔上。
4. 嵌贴：嵌贴是将锡化膜嵌入铝箔之后的最后一步，目的是让材料固定住，同时减轻板材或电路器件重量。
5. 海绵编织：海绵编织就是将不同颜色的材料通过海绵串联起来，然后进行切割、焊接等工序，最终完成PCB板的制作。
6. 检查、测试：检查和测试是在组装完毕之后，对产品进行功能测试、安全测试以及其它测试，确认是否满足设计要求。
总而言之，光刻胶的制造过程要保证产品的质量，做到物料的精确处理、安全、经济、快速，否则两极效应就会对产品造成损害。为了尽量减少两极效应对产品的影响，制造商应该注意以下几点：
- 在环节之间引入合理的暂停时间，保证工作质量；
- 将各个环节做好措施，减少风险隐患；
- 对光刻胶工艺人员的培训和管理，提高技艺水平，建立起专业化管理机制；
- 为产品制造商提供一系列检测工具，帮助他们监控工艺流程和产品质量。
