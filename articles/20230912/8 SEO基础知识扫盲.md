# 2.SEO理论基础
## 2.1 概念和术语
- **SEO**：Search Engine Optimization，搜索引擎优化，指的是通过网络从事内容营销活动，提升在网络搜索引擎结果中的排名，增加网站流量并提高访问量，从而实现网站建站目的。
- **爬虫**：一种自动获取网页上的信息并保存到数据库或文件的程序。
- **URL**：Uniform Resource Locator，统一资源定位符，用来表示互联网上某个资源的位置。
- **网站结构**：由不同的HTML文档组成的网站，各HTML文档之间可能存在跳转关系，也可能没有。
- **关键词**：用来描述客户需求的单词或短语，通常用在搜索引擎中查找用户所需的内容，网站需要注意的是关键字的选择要符合用户的诉求。
- **链接**：两个网页之间相互关联的超文本标签，指向其他网站或当前网站的特定页面。
- **权重：**关键字排名靠前的原因。权重越高，代表该关键字出现的次数越多，同时被该页面的其他内容重复使用，给其排名的可能性就越大。
- **SEO算数运算公式**：根据搜索引擎算法计算出来的一个页面得分数值。得分越高，代表页面质量越好，对于用户来说，页面质量好意味着更好的流量转化率。一般情况下，其公式如下：
    - S = (1 – R/m + C) x P
    - R 是页面的pagerank值，pagerank是一个衡量网页被查询多次的重要指标；
    - m 是网页包含的单词数量；
    - C 是页面上关键字的密度系数；
    - P 是页面的平均质量。
- **归因分析**：将访客行为和受益分析，用来判断和分析不同原因对网站访问产生的影响，帮助企业明白用户为什么访问他们的网站、他们喜欢什么、不喜欢什么，最终追踪问题根源，改进产品和服务，提高品牌价值。
- **反向链接**：每个网页都在一定程度上会被其他网站引用，反向链接就是指一个网站上的所有链接，通过这些链接可以找到这个网站上的所有页面。反向链接的好处在于可以发现网站中存在的问题、热门内容和市场机会。
