# 1.简介
  

## 一、系列主题
### “工程”或“科学”科普类图书：

1. 《微积分》—— 微积分是工程和科学中非常基础的课程。在学生毕业之后，还可以继续深造，研究更复杂的微积分概念。
2. 《线性代数》—— 线性代数可以理解为向学生揭示矩阵变换的过程。对于学生的过渡阶段非常重要。
3. 《力学》—— 可以帮助学生了解物体运动中的无摩擦假设和牛顿第二定律。
4. 《博弈论》—— 博弈论可以用来预测不同的游戏结果。对于学生的职业生涯发展至关重要。
5. 《概率论》—— 概率论通过对一些事件发生的可能性进行分析。对于学生的职业生涯发展至关重要。

### 计算机类图书：

1. 《算法导论》—— 从最基本的算法设计到复杂的排序算法，适合作为算法功底的提升工具书。
2. 《计算机网络》—— 本书从物理层到应用层，详细阐述了计算机通信协议的各个方面。可以作为高级工程师的必备读物。
3. 《深度学习》—— 深度学习技术目前正在火爆发展。本书深入浅出地介绍了深度学习的各种原理。
4. 《编译原理》—— 编译器的构造方法和过程都可以在这本书中找到。适合作为编程语言和系统结构的基础知识。
5. 《数据库系统实现》—— 本书结合了相关的数据库技术知识，可以帮助学生更好的理解数据库的工作原理。

### 经济管理类图书：

1. 《金融风险管理》—— 这是一本关于金融市场风险管理的经典著作。可以教给学生掌握风险管理的关键要素和方法。
2. 《工商企业管理》—— 这是一本在海量工商企业数据上进行统计分析并得出可操作的建议的理论性著作。适合作为新手财务经理的参考。
3. 《管理行为学》—— 管理行为学着重于行为理论与实践之间关系，尤其是在组织管理过程中的复杂性。适合作为大学生及以上专业的导师用书。
4. 《产权保护与法律》—— 本书探讨了产权的基础理论、法律规范以及宪法赋予产权者的特殊义务。适合作为法律硕士及以上学历的人士。
5. 《商业模式生成》—— 本书通过丰富的案例，系统讲述了商业模式的形成过程。适合作为创业公司及初创团队的研究阅读材料。

### 生活类图书：

1. 《亲子课堂》—— 由许多年轻小朋友编写而成的一本儿童教育类图书。鼓励孩子树立正确的道德观念和良好的家庭关系。
2. 《古董饰品精选集》—— 提供了经过艺术鉴赏和考究的古董饰品的优美插画。通过展览可以让人们回味历史的魅力。
3. 《饮食健康指南》—— 这是一本关于个人饮食健康、营养搭配、减肥瘦身的方法的医学教科书。适用于所有想提高饮食健康水平的消费者。
4. 《文化经典阅读》—— 此书收集了一批代表作品，作者以独特的方式呈现文化精神。值得学生花时间阅读。
5. 《新闻研究》—— 研究报告、调查报告等资讯系类的书籍多是美国人写的。但是这些书籍在中国也有很好的译本。这本书的内容就是一个例子。

## 二、系列介绍
《多项式与有理函数》系列是由中科院数学与系统科学国家重点实验室主办，华东师范大学出版社联合出版的系列图书。该系列围绕多项式和有理函数的基本理论和应用展开。包括多项式分裂、递推、特征根、高阶、极限运算、代数闭包等基础理论；有理函数的表示、运算、性质、求根、变分、积分、极限等；组合数、群论、图论、数论、随机数、模运算等专题。书中所有内容均采用公式、图示、表格的形式，可有效满足专业人员的学习需求。
