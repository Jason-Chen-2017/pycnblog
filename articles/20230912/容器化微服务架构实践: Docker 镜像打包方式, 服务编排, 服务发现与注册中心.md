# 1.简介
  

本文主要对Docker、K8s等容器编排技术进行深入探索与实践，通过实际案例，阐述如何运用Docker镜像打包的方式实现微服务的自动化构建、编排和部署，同时提升微服务架构的可靠性、可用性、弹性伸缩、可观测性以及降低复杂度。
