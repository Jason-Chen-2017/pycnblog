# 1.简介
  

随着科技发展的迅速、产业的飞速发展和经济效益的提高，智能出租车（Intelligent Taxi）已经成为轰动一时的新兴消费需求。虽然目前智能出租车领域还处于起步阶段，但各个厂商已经在其产品或服务上全面拥抱自动驾驲技术，并尝试着打造出一系列具有先进功能和性能的智能出租车。本文从物理层面的功能和性能角度，介绍一下智能出租车背后的底层基础设施——车身（Vehicle Body）、电子系统（Electronic System）、驱动系统（Driver System）和通信系统（Communication System）。
# 2. 基本概念术语
## 1. 车身(Vehicle Body)
智能出租车的车身结构及其结构特征，可以分为四部分：

- 中央控制器：车身中心位置的一座中央控制台，可以根据用户的需要进行各种控制；
- 后视镜：可以看到前方道路的一对透视镜，能够直观地看到车辆的正前方区域；
- 主体结构：包括车门、方向盘、安全带等主要部件组成；
- 尾部附属装置：包括车灯、雨刷、遥控器等。

## 2. 电子系统(Electronic System)
智能出租车的电子系统由多个子系统构成，包括照明系统、汽车内饰显示系统、导航系统、车辆控制系统、CAN总线通信系统、车载存储系统、车载云平台系统、车位管理系统、安防监控系统等。其中，车辆控制系统是智能出租车的核心功能之一，负责完成路线规划、驾驶状态监测、方向识别、速度估计、加速减速、刹车转向以及其他车辆控制相关的任务。

智能出租车的电子系统有三个重要的特点：

1. 模块化设计：智能出租车电子系统由多个不同模块构成，可以根据需要安装新的功能，提升智能出租车的性能；
2. 多模式协同：智能出租车的不同系统之间通过网络连接，实现多模式协同，同时提升系统的处理能力和安全性；
3. 大数据分析：智能出租车的电子系统可以通过大数据采集、分析和预测，实现车辆的精确控制，降低人为因素影响。

## 3. 驱动系统(Driver System)
驱动系统是智能出租车最关键的部件之一，它由两部分组成，分别是司机发动机与方向识别系统，以及驾驶手势系统与仪表系统。司机发动机系统的作用是将车辆的电力转换为动力，以满足用户的行驶需求；方向识别系统的作用是识别驾驶人的当前航向，从而使车辆适时停下，避免发生碰撞；驾驶手势系统的作用是控制车辆的行驶，以使驾驶者的操作准确无误；仪表系统则是车辆运行的仪表盘，用于显示车辆的运行状态、里程、油耗、温度、速度等信息。

## 4. 通信系统(Communication System)
智能出租车的通信系统由车载终端设备、车载云平台系统、车载摄像头、导航设备、外部设备等组成。车载终端设备的作用是提供连接到车辆的入口，接收和处理用户的输入指令；车载云平台系统的作用是整合车辆的传感器、雷达、激光雷达、相机、GPS等设备，并与车载终端设备进行联动；车载摄像头的作用是捕获车辆前方环境，用于导航、辅助驾驶；导航设备的作用是提供实时、高精度的导航信息，帮助车辆快速准确定位；外部设备的作用是与智能出租车的硬件系统互联，实现信息的交换。

## 5. 综述
智能出租车的底层基础设施可以概括为车身、电子系统、驱动系统、通信系统。车身作为智能出租车的最基本组成部分，承担着重要的制动、安全和操控功能；电子系统则主要由控制系统、显示系统、导航系统等组成，完成路线规划、驾驶状态监测、方向识别、速度估计、加速减速、刹车转向、遥控器操作等功能；驱动系统负责发动机、方向识别、手势控制、仪表系统的功能；通信系统则负责连接终端设备、云平台系统、车载摄像头、导航设备、外部设备等，实现信息的传输。因此，智能出租车的基础设施构建好之后，才能真正实现功能的实现。
