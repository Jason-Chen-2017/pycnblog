# 2.主要内容
## 2.1 A股股票不能做保本的原因分析

### （1）债务链问题

股票市场存在着债务链问题。所谓债务链问题就是，一家公司的流动负债越多，它就越难清偿。也就是说，一家企业的债务越多，如果没有足够的资金进行清偿，那么它的偿债能力就会越差。债务链问题对中小企业尤其重要，因为它们一般都比较脆弱。因此，随着债务规模的增加，不少小企业的债务迟迟无法得到解决，甚至还被恶性的融资陷入困境。

在A股市场，每个公司都有一个存款保险基金（CDP），专门用来应对各种资产的损失，如旗下的银行、企业、员工、股东等。这家基金由多方共同拥有，每月收取的费用由六个月的年化利率计算。如果某个公司出现严重亏损，没钱偿还公司债务，那么基金就会承担巨大的责任。而且，基金虽然独立运营，但实际控制的是整个公司，包括董事会、监事会、总裁、高管等，因而形成一种既垄断又无孔不入的局面。

### （2）政策干预问题

A股市场存在着政策干预问题。比如，市场普遍倾向于刺激炒作，这时，机构可能会提前布局，给予大量融资，扭曲市场竞争机制，损害散户的信心，严重破坏股市的正常运行。另一个例子是房地产相关政策调整，导致房价上涨，同时带动股价上涨，进一步刺激房企炒作。

### （3）行业供需关系问题

A股存在着行业供需关系问题。比如，电子元件行业需求旺盛，但在过去几年里，它的供给却大幅缩水。这种现象表明，制造业和服务业的需求量之间存在矛盾，即供给侧结构性壁垒正在形成，这直接影响着股市走势。

### （4）企业自身问题

A股存在着大量的企业自身问题。比如，有的企业为了减轻成本，把营收做空，这种做法往往伴随着很高的成本，造成营销上的不当动向，继而损害股民利益。还有的企业为了保持利润增长速度，采取短期行为而不是长期积蓄，导致利润下滑，继而损害股民利益。

## 2.2 为什么A股股票不能做保本

以下是一些导致A股股票不能做保本的原因：

①债务链问题。A股股票市场存在着债务链问题，一家公司的流动负债越多，它就越难清偿，企业债务将会一直累积下去，不可能完全用以支付股息或偿还贷款，因此A股股票无法做保本；
②政策干预问题。A股存在着政策干预问题，例如市场普遍倾向于刺激炒作、房地产政策调整等，这些政策往往冲击股市正常运行，引起股价暴涨和融资危机，继而影响股民利益；
③行业供需关系问题。A股存在着行业供需关系问题，比如制造业供应充足，但需求却大幅缩水，这些状况导致供求关系失衡，影响股市走势；
④企业自身问题。A股存在着大量的企业自身问题，如营收做空、毫无长期预算的创业公司等，这些问题直接威胁到股民利益，使得A股股票无法做保本。

以上四种原因的综合表现，导致A股股票不能做保本。

## 2.3 为什么A股股票可以做保本

A股股票可以做保本的几个原因如下：

①时间限制。很多企业已经进入低净值阶段，财报已经审计完毕，因此可以实施股权激励，以维持现金流。因此，A股股票可以做保本，并在一定时期内取得最大回报；
②风险可控。很多企业是小企业，银行等机构的揽储和贷款利率也相对较高，因此风险可控；
③独立团队。股权激励需要大量的筹资支撑，因此需要由独立的团队来设计和实施激励措施；
④强调风险管理。有些企业选择在股权激励中加入风险控制机制，以减轻市场风险，提高企业经营能力。

综上所述，A股股票可以做保本。下面我们就详细阐述一下A股股票保本理财的过程。

# 3.具体操作步骤

## 3.1 准备工作

- 检查自己是否符合缴纳社保条件，缴纳社保包括养老保险、医疗保险和失业保险，具体条件可参考各地区政府要求。

- 确定风险偏好。选择适合自己的抗风险方式，例如选择分散风险、温和低波动策略还是保守高波动策略。

- 了解所投资产品的特点和收益率。了解其风险特性，例如是否具有较高的历史回报率，能够长期保持稳定盈利能力，而且不会因政策调整或行业变迁而大幅改变其表现。

- 认识储蓄和投资的区别。储蓄意味着用来购置保险，投资则是买卖特定产品或基金的行为。

- 提前准备储备资金。

## 3.2 确定适合自己的保本理财计划

理财产品的选择根据您的风险偏好、储备资金和资产配置情况进行，通常可以分为三类：

- 投资型理财产品。顾名思义，投资型理财产品是以投资理财为主，投资于股市，涉及到股票、债券、货币基金、现金等多个资产类别。投资型理财产品的优点是收益率高，风险低，适合长期持有，且不需要投入过多的精力和物质。

- 保本型理财产品。保本型理财产品则不同于投资型理财产品，它旨在提供稳定的年度收益率，可以长期保持现金流，并且会按照政策的要求提前赎回或退出资产。保本型理财产品的特点是在收益率方面超过投资型理财产品，同时由于采用保本理财模式，风险也会降低。

- 普通型理财产品。普通型理财产品不属于前两者，是指对不同类型资产均采用风险管理模式，例如可开展企业年金、公积金、存款、贷款等保本方式，但是风险和收益比一般会低一些。

## 3.3 怎么选股

根据个人喜好、技能水平和研究，选择一批适合您投资的股票。这里建议您首先关注公司业绩好、盈利能力强、基本面优秀、估值合理、可靠经纪人的股票。这样可以尽快抓住波动周期的特征，并控制投入。

另外，也可以考虑买入某只股票后，观察其公司的分析师对其股价走势和财报的评价，判断其是否存在市场震荡的可能性。

## 3.4 如何分析股票

要做好对标基准的分析，首先要有个清晰的目标。如果你知道明天股票的价格和估值指标，那就可以根据这个目标判断今日股价的走势。

然后，对待每只股票，务必注意以下三个指标：

- 上涨趋势。上涨趋势的定义是当前股价上涨速度快于下跌，而且上涨幅度大。

- 下跌趋势。下跌趋势的定义是当前股价下跌速度快于上涨，而且下跌幅度大。

- 沪深300指数。沪深300指数是一个大盘指数，代表全球上证和深圳主板交易所上市股票的平均价格水平。如果某个股票的涨幅超过了沪深300指数的涨幅，那么我们就可以认为它在上涨趋势中。反之，如果股票的跌幅超过了沪深300指数的跌幅，那么我们就可以判断它在下跌趋势中。

股票分析的第二步是研究它的业绩表现，包括营业收入、利润、营业利润、净利润、股东权益等。要找出业绩比较突出的公司，分析其成长性。还有，还要看它的前景。如，市盈率、市净率、总资产周转率等指标。

第三步是研究它的资产配置，包括流动资产、固定资产、无形资产、长期负债等。为了避开虚假的泡沫，多关注长期负债的情况。还有，要注意流动性，看它是否具有较高的流动性。如果流动性较差，要考虑是否适合投资。

最后，还要研判它的未来趋势。如，市场整体的情绪如何，价格的持续走势如何，业绩会不会出现明显的下滑？如果下滑的幅度大，是否会影响到您投资的结果？

## 3.5 建立保本理财模型

在决定投资之前，应该制定自己的理财计划，包括收益率、风险系数、期限等。其中，收益率取决于您设置的目标，风险系数取决于你的储备资金和风险偏好，期限取决于你投资的能力和资金分配。

设立保本理财计划的方式有两种，即按揭和保本。按揭是指由保险公司发放到期保单，投资者以此为基础按揭到期的资金进行投资，从而收取一定的期望收益率，这种方式的风险较大，效率较低。

而保本类型则是依据公司财报或者其他资料确认其资产负债率（资本成本/净利润），然后将其中约定的比例分配给本人和家庭成员，以此建立长期的收益预期，这种方式的收益较高，风险较低。当然，为了达到安全的目的，也需要对保险公司、投资者、家属、亲戚等人员加以保密，防止他们拒绝你的申请。

import pandas as pd
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt


def daily_returns(df):
    """计算日收益率"""
    df['daily_return'] = (np.log(df['Close']) - np.log(df['Close'].shift(-1))) * 100
    return df[['Date', 'daily_return']]


def cumulative_returns(df):
    """计算累计收益率"""
    df['cum_return'] = (1 + df['daily_return']).cumprod() - 1
    return df[['Date', 'cum_return']]


def sharpe_ratio(df):
    """计算夏普比率"""
    avg_annualized_return = annualized_return(df)
    rf_rate = 0.01  # 无风险利率为1%
    std_dev = df['daily_return'].std()

    sr = (avg_annualized_return - rf_rate) / std_dev
    return round(sr, 4)


def information_ratio(df):
    """计算信息比率"""
    t_value = stats.t.ppf(0.95, len(df))  # 设置置信水平为95%
    mean = df['daily_return'].mean()
    std_dev = df['daily_return'].std()

    ir = mean / (std_dev / t_value ** 0.5)
    return round(ir, 4)


def max_drawdown(series: pd.Series):
    """计算最大回撤"""
    dates = series.index[::-1]
    peak = float(series[dates].max())
    trough = float(series[:peak][::-1].idxmin())
    mdd = ((peak - trough) / peak) * 100

    return round(mdd, 4)


def annualized_return(df):
    """计算年化收益率"""
    years = int((pd.to_datetime(df['Date'][0]) -
                pd.to_datetime(df['Date'][-1])).days / 365)
    avg_daily_return = df['daily_return'].mean()

    ann_r = (1 + avg_daily_return) ** (1 / years) - 1
    return round(ann_r * 100, 4)


def rolling_beta(df):
    """滚动计算股票收益的贝塔系数"""
    closes = df['Close']
    returns = np.log(closes)
    betas = []

    for i in range(len(closes)):
        if i < 252:
            rets = returns[i:]
        else:
            rets = returns[i - 252:i]

        x = rets[:-1]
        y = rets[1:]

        slope, intercept, r_value, p_value, std_err = stats.linregress(x, y)
        beta = slope

        betas.append(beta)

    df['Beta'] = pd.Series(betas).rolling(window=20).mean().fillna(method='bfill')
    df['Alpha'] = df['daily_return'] - df['Beta'] * df['Close']

    return df[['Date', 'Close', 'Beta', 'Alpha']]


def plot_data(df, title='', width=10, height=5, legend=True, color='b'):
    """绘制收益率曲线图"""
    fig, ax = plt.subplots(figsize=(width, height))

    ax.set_title(title)
    ax.plot(df['Date'], df['cum_return'], label='Cumulative Return', lw=1, alpha=0.7, c=color)
    ax.plot(df['Date'], df['alpha'], ls='--', label='Alpha', lw=1, alpha=0.7, c='k')
    ax.axhline(y=0, xmin=0, xmax=1, linestyle='-', lw=1, c='gray')
    ax.grid('on')

    if legend:
        ax.legend()


def draw_graph(df, stocks, risk_free=0.01):
    """绘制收益率曲线图"""
    df = df.loc[:, ['Date', 'Open', 'High', 'Low', 'Close', 'Adj Close', 'Volume']]
    df = daily_returns(df)
    df = df.dropna()

    cum_df = cumulative_returns(df)
    ann_ret = annualized_return(df)
    sharp_rat = sharpe_ratio(df)
    inf_rat = information_ratio(df)
    mddl = max_drawdown(df['cum_return'])

    print(f'Annualized Return:\n{ann_ret}%\n\nSharpe Ratio:\n{sharp_rat}\n\nInformation Ratio:\n{inf_rat}\n\nMax Drawdown:\n{mddl}%\n')

    for name in stocks:
        stock_df = df[(df['ticker'] == name)].drop(['ticker'], axis=1)

        roll_beta_df = rolling_beta(stock_df)
        alpha = roll_beta_df['Alpha'].iloc[-1]
        beta = roll_beta_df['Beta'].iloc[-1]

        graph_df = roll_beta_df[[col for col in list(roll_beta_df) if col not in ['Date', 'Close']]]
        graph_df['cum_return'] = cum_df['cum_return']

        plot_data(graph_df, f'{name} ({alpha:.4f}, {beta:.4f})',
                  legend=False, color='#ff6a00')

    plt.show()


if __name__ == '__main__':
    df = pd.read_csv('AAPL_2019.csv')
    draw_graph(df, ['AAPL', 'MSFT', 'AMZN', 'GOOG', 'TSLA'])
