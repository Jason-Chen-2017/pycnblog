# 1.简介
  

李石祥(李斌)，Wolai Inc的创始人、CEO兼COO，他2007年加入中国移动集团担任CEO一职。

2019年11月2日，李石祥接受了采访，并发表了题为《李石祥：数字孪生时代，智能工业赋能大脑功能革命 - Wolai Inc.》的演讲。

李石祥：欢迎来到Wolai Inc！很高兴您能参加我在Wolai会议上的演讲。首先，我想向大家分享一下关于“智能工业”这个词汇的一些历史知识。

数字孪生（Digital Human）这个术语最早是在日本由芝加哥大学的Daiichi Ogawa教授提出来的，并在20世纪80年代被应用于军事领域。它主要意味着人工智能系统能够像动物一样操控自然界的物体，同时也在不断探索另一个可能性——能够让机器感知、理解、运用和控制身体的能力。这种能力使得数字孪生可以从事各种各样的工作，例如训练、治疗、驾驶、交通导航、翻译等。

20世纪90年代末期，美国国家科学基金委员会主办的“超级计算机计划”项目将人工智能方面的研究重点转移到了大脑科学上。这场具有里程碑意义的科研活动几乎占据了整个科技行业的中心地位。在这一过程中，人们对如何开发能够理解、记忆、学习、决策和执行复杂任务的数字系统的需求越发强烈。由于数字系统的计算速度比传统机械系统快很多，因此它们可以实现某些实用的任务。例如，随着新技术的不断进步，虚拟现实技术已经在人类视觉、听觉、嗅觉等方面实现了前所未有的突破。

2010年，清华大学的邱锡鹏教授领导的“大脑在线计划”项目开展了一项颠覆性的研究，探索利用脑电图技术实现精确、连续的大脑信号记录。通过利用高频、低功率的脑电波，他成功捕捉到大脑中神经元之间的交流关系，从而实现全脑层次的图像采集。然而，该项目后来因为危及肖尔法尼亚大学的政治筹款导致停滞，直到2014年才重新启动。

时至今日，数字孪生（Digital Human）已经成为一种正在蓬勃发展的重要分支产业，影响着人类的生活和社会行为。除了军事领域，数字孪生还广泛用于医疗保健、食品安全、环境监测、远程医疗、家庭管理、商业服务、智能机器人、虚拟现实等领域。在人工智能领域，李石祥认为“智能工业”是一个广阔的领域，涵盖了多种技术和应用场景，包括智能穿戴、智能城市、智慧医疗、智能制造、智能服务等，但仍处于起步阶段，面临着巨大的挑战。

# 2.数字孪生在医学中的应用
先谈谈“智能医疗”，为什么“智能”这个词最近似乎变得越来越热门呢？李石祥介绍，目前医学界尚处于“智能药物+机器人+AI”的转型时期。

李石祥说：“在过去的几十年里，我们一直在尝试利用新技术来改善患者的生存质量，提升他们的医疗效益。这种技术通常被称为“智能化”或“智能医疗”。比如，在2013年，欧洲联合国开发出了一种基于红细胞的基因检测方法，其目的是为了检测是否存在疾病导致的遗传缺陷。随后，在德国、日本、英国等其他国家，人们也纷纷投入大量资源进行“智能医疗”试点，期望为病人的生命健康提供更好的保障。但是，“智能医疗”的发展却遇到了一些严重的问题。比如，通过基因检测、视觉或听觉诊断的“智能医疗”方法很难覆盖所有的疾病，因为它们需要耗费大量的时间和金钱。此外，当前的“智能医疗”产品往往无法满足个体差异性或客户群体的特点，造成医疗费用高昂或服务质量差。因此，我们看到了更多的投入，希望找到一种更优秀的方法来解决这些问题。”

“智能医疗”对患者来说应该是一种简单有效的方案，只要患者服用这个产品，就可以获得良好的治愈效果。但对于患者来说，并不是每个人都适应这种方法，并且它的运作过程对患者来说也是隐私的。另外，这种解决方案只能针对少数特定类型的疾病，并不能完全解决人类疾病的普遍问题。因此，李石祥建议，在未来的“智能医疗”解决方案中，应该包含包括基因检测、诊断、治疗等流程的全链路优化，而不是仅靠单一的“智能”工具。

下一步，李石祥提到在“智能制造”领域也有着相当的影响力。目前，智能制造的发展方向之一就是利用机器人手臂、足端、甚至自主意识来帮助生产者完成复杂的工业生产任务。在这个过程中，物料、设备、零部件、工艺等环节都会由机器人自动化的完成，从而提升生产效率、缩短生产时间，降低成本。但同时，如果出现问题，机器人也会通过分析、监控、反馈等方式快速响应和恢复，避免机器人对人类的主观失误导致的错误。因此，“智能制造”并非一蹴而就，需要在生产环节的各个环节融入“智能”组件，才能真正实现“智能”工厂。

# 3.智能手术与自动化手术的区别
在今天这个智能化、数字化、网络化的世界，手术还是最受欢迎的高科技活动之一。每年约有三亿例的手术在医院进行，其中大部分都是通过计算机化技术进行的。那么，“智能”手术与普通的手术有什么不同呢？李石祥介绍，“智能”手术的关键在于两点：“快速准确”以及“即时可见”。

李石祥介绍，“智能”手术采用“机器人-病人-镜头”的工作模式，首先由机器人进入病房进行指导，然后病人可以直接在镜头前进行手术。这样做可以显著提高手术的效率和安全性。同时，“智能”手术也可以通过实时跟踪、分析图像、语音数据等手段，帮助病人准确掌握手术进度。

除此之外，“智能”手术还可以在手术前预估出可能出现的问题，甚至能够根据病人的身体状况和行为进行优化调整，从而减少出错风险。目前，科学家正在开发新型的无人机手术系统，通过无人机对人体内部进行全切割、颗粒试探、局部浸润等手术技术。

最后，李石祥认为“智能”手术还可以应用在多种不同的场景中，如医疗器械、养老金领取、政务咨询、房地产销售等领域。“智能”手术会彻底改变手术的模样，提升手术的安全性、经济性、舒适性、便利性，为患者带来全新的体验。

# 4.“数字孪生”对大脑功能的影响
数字孪生作为一个从海量数据的处理入手，突破了我们认识大脑的上限，为我们提供了更加丰富、更加复杂的大脑功能。它可以让我们在看不到或者摸不到的情况下，做出许多之前仅仅依赖视觉、听觉的判断、决策和行为，甚至还可以进行实时的脑电波跟踪、同步绘图、情绪识别、语言交流、个性化编程、学习和记忆等。

李石祥提到，“数字孪生”对大脑功能的影响还不止这些，例如通过智能运动控制、运动训练、语音助手等将大脑功能释放出来，可以帮助我们获取更多的乐趣、减少压力、提高工作效率、解放双手，甚至还能够影响大脑神经元的工作状态，产生新的神经性行为。

# 5.未来发展方向
李石祥认为，未来“智能工业”的发展方向分为两个方面：第一，脑电信号的深度测量和分辨。当前的脑电信号收集的方式存在着技术瓶颈，无法达到深度测量的要求。因此，未来的脑电信号的搜集、分析、存储和传输技术的研发，将极大地推动脑电生理领域的发展。第二，在大规模的脑电信号数据基础上，智能人工智能的模型构建、训练、部署和应用将成为推动“智能”大脑功能应用的关键环节。

李石祥强调，当前“智能”大脑功能的应用还处于初期阶段，还需要不断努力，还需要深刻反省我们的教育方式、娱乐方式和消费习惯等方面的问题。因此，未来“智能”工业的发展不会停止，而是源源不断地吸收各种新技术、新方法和新产品，为人类的健康、幸福和福祉不懈努力。
