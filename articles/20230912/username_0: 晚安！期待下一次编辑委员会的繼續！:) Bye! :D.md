# 1.简介
  


昨天，GitHub搞了个活动——《开源中国周刊》，让大家把自己开源项目的一年以来的经验、感悟等等分享出来，希望通过这个平台能够激发更多的人参与到开源社区，推动开源事业的发展。看着看到许多优秀的技术文章和开源项目，我心里一片欢喜之情。不过我更关心的是这些开源项目背后的技术理念和价值观如何影响了他们的发展，在国内的商业应用是否也出现了一些不错的成果？当然，作为一名程序员，我更关注这些技术人员所遵循的编程规范和编码习惯。因此，今天给大家带来的正是这个话题。

首先，先给大家介绍一下我自己吧。我是一个CTO，虽然做的是技术工作，但是对AI、大数据、互联网和其他新兴领域都充满激情。我从计算机科学出生，对软件开发和系统架构都很感兴趣。我曾经是一名创业者，曾经也是老牌互联网公司的CTO。不过那时候互联网还比较小，技术和工具知识比较少。所以，我在学习过程中，主要是从各种开源项目和博客中获取知识和方法论。

在这次的活动中，我想邀请一些开源项目的作者来分享自己的经验。同时，我也将介绍几个我觉得值得关注的开源项目。最后，我还会和大家探讨一下这些开源项目背后的技术理念和价值观，并且试图找出这些理念和价值观对今天国内的商业应用的影响。因此，如果您愿意接受这样的采访，请不要犹豫，马上填写下面的表格，让我们一起见证这个“圆点”系列活动的圆满结束。期待您的加入！

# 2.采访对象与项目介绍

请选择以下3个项目中的1-3个进行面试。


   PaddlePaddle是一个中国领先的开源机器学习框架。它具有完善的深度学习模型库和丰富的API接口，可以快速实现模型的训练、预测和部署。PaddlePaddle的主要特性包括支持单机CPU、GPU及分布式多机多卡等多种硬件环境，具备良好的Python API开发体验，支持广泛的任务类型，如图像分类、序列模型、推荐系统等。

   
   DeepCTR是一个基于深度学习的CTR预估模型库，目前已支持多种模型结构（如DCN、PNN等）、多种激活函数（如ReLU、Sigmoid等）、多种评估指标（如AUC、LogLoss等）。通过统一的输入输出格式，提供不同的样本处理方式、特征工程方式、模型训练方式和其它辅助工具，帮助用户快速搭建CTR预估模型。


   Ant Design Pro是一个基于React和umi的企业级中后台前端/设计解决方案。它的界面采用了阿里妈妈MUXD设计语言，提供了中台管理、流程配置、表单页、列表页、统计报表等功能，帮助企业快速完成复杂业务功能。同时也提供了可扩展性，可以用于构建不同业务类型的中后台系统，比如电商、餐饮、运营管理等。
