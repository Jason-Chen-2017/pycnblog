# 2.基本概念

## 2.1 无线电

无线电是指利用无线电波进行的电磁感应传输的一种广播电信系统。它是利用电磁波在空间和时间上发生作用而形成的电磁场，包括高频率(FM)、超低频率(AM)和短波(SWL)。它的特点是具有广播、多普勒效应、弱空间辐射、无反射性、不可分辨性等，广泛应用于广播、通信、导航、医疗、儿童教育、军事等领域。

无线电技术有全向、半导体、非谐振荡等多个分类，目前市面上存在的无线电产品非常多。比如：微波收音机、卫星接收机、激光雷达、天线组网器材、功放、用作声控装置的手机、WiFi、蓝牙、Zigbee等。

## 2.2 无线电产品

无线电产品是指由无线电设备及组件、配套软硬件组合而成的完整的无线电系统，可实现无线电技术的应用。无线电产品一般有三种类型：固定无线电站设备，移动无线电终端设备及模块，以及数字化无线电终端设备及模块。

### 2.2.1 固定无线电站设备

固定无线电站设备是指具有固定坐标，由专业工程师设计、安装、调试和维修的专用设备。由单台设备组成的固定无线电站一般只能容纳少量用户，通常设计了多路复用的话输出功率也可增大。

固定无线电站设备的典型代表有微波收音机、智能功放、红外发射管、微波炉、微波炮等。

### 2.2.2 移动无线电终端设备及模块

移动无线电终端设备及模块属于特殊的无线电终端产品。它由单个或多个无线电接口卡、配套软件、配套硬件构成。功能相对简单，但耗电量极低，方便随身携带。常见的移动无线电终端设备及模块有手机、平板电脑、电视机顶盒、数码相机、耳麦盒子、GPS定位器、望远镜等。

### 2.2.3 数字化无线电终端设备及模块

数字化无线电终端设备及模块是一种由计算机芯片、无线电模块及嵌入式软件、硬件组成的综合性终端设备。功能复杂，集成度高，通过网络连接到无线电网络，能够实时接收、处理、分析无线电信号。常见的数字化无线电终端设备及模块有路由器、智能手机、PC、无人机、车载系统、GPS导航系统等。

# 3.核心算法原理

基于以下几个要素，完成控制无线电产品的操作:

1. 操作对象：固定无线电站、移动无线电终端设备及模块、数字化无线电终端设备及模块；
2. 操作设备：无线电发送设备、无线电接收设备、无线电控制系统、无线电交换设备等；
3. 控制参数：发射功率、接收灵敏度、传送距离、工作模式、调制方式、变幻速率、混杂效果等；
4. 命令输入：键盘、遥控器、手机APP、笔记本电脑等；
5. 输出显示：显示屏、声音、蜂鸣器、LED灯等。

## 3.1 固定无线电站

固定无线电站是指专门用于发射无线电的设备，具有专门设计的结构和设备，并经过专业的维护。因其结构紧凑、设备少，故其受电力供应能力限制，能够满足相对较少的人口密度。固定无线电站的应用范围及收益最高，且通常运营时间比较长，不适合个人和小型企业用户。

操作对象：固定无线电站

操作设备：无线电发送设备

控制参数：发射功率、变幻速率、混杂效果

命令输入：按键、遥控器

输出显示：无

控制流程：

1. 配置无线电发送设备的参数：设置发射功率、变幻速率、混杂效果。
2. 使用遥控器或按键启动发送设备工作。
3. 通过遥控器或按键进行参数调整。
4. 按下停止按钮停止发送。

## 3.2 移动无线电终端设备及模块

移动无线电终端设备及模块是一种特殊的无线电终端产品，由单个或多个无线电接口卡、配套软件、配套硬件构成。功能相对简单，但耗电量极低，方便随身携带。它采用了单模技术，即仅有一个信号输出端口。移动无线电终端设备及模块可以单独或组合使用。

操作对象：移动无线电终端设备及模块

操作设备：无线电发送设备、无线电接收设备、无线电控制系统

控制参数：发射功率、接收灵敏度、传送距离、工作模式、调制方式

命令输入：手机APP

输出显示：显示屏

控制流程：

1. 安装配套软件：下载安装手机APP，配置WIFI/蓝牙账号密码等。
2. 配置无线电发送设备的参数：设置发射功率、工作模式、调制方式。
3. 通过手机APP控制无线电发送设备工作。
4. 使用遥控器或按键进行参数调整。

## 3.3 数字化无线电终端设备及模块

数字化无线电终端设备及模块是一种由计算机芯片、无线电模块及嵌入式软件、硬件组成的综合性终端设备。功能复杂，集成度高，通过网络连接到无线电网络，能够实时接收、处理、分析无线电信号。它的特点是结合数字信号处理和无线电系统技术，能有效提升无线电系统的接收、处理、输出性能，实现更快、精准的控制和监测功能。

操作对象：数字化无线电终端设备及模块

操作设备：无线电发送设备、无线电接收设备、无线电控制系统、无线电交换设备

控制参数：发射功率、接收灵敏度、传送距离、工作模式、调制方式、变幻速率、混杂效果

命令输入：手机APP

输出显示：显示屏

控制流程：

1. 配置无线电发送设备的参数：设置发射功率、变幻速率、混杂效果。
2. 建立网络连接：连接至无线电交换设备，获得IP地址等。
3. 安装配套软件：下载安装手机APP，配置WIFI/蓝牙账号密码等。
4. 配置无线电接收设备的参数：设置接收灵敏度、工作模式、调制方式。
5. 使用手机APP控制无线电接收设备工作。
6. 修改无线电信号传输路径：选择不同信号源、转发点等。
7. 测试无线电系统的工作状态。
