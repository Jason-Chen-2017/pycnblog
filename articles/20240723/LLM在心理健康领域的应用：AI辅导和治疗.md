### 5.2 源代码详细实现

以下是一个使用BERT模型进行心理健康辅导的Python代码实现：

```python
import torch
from transformers import BertForSequenceClassification, BertTokenizer
from flask import Flask, request, jsonify

# 加载BERT模型和分词器
model = BertForSequenceClassification.from_pretrained('bert-base-uncased', num_labels=2)
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')

app = Flask(__name__)

# 定义API接口
@app.route('/psychology_chat', methods=['POST'])
def psychology_chat():
    data = request.get_json()
    text = data['text']
    
    # 对输入文本进行分词
    inputs = tokenizer.encode(text, return_tensors='pt')
    
    # 将分词结果输入模型进行情感分类
    with torch.no_grad():
        outputs = model(inputs)
        logits = outputs.logits
        label = torch.argmax(logits, dim=1).item()
        
    # 输出情感分类结果
    return jsonify({'label': label})

if __name__ == '__main__':
    app.run(debug=True)
```

代码实现主要包括以下几个步骤：

1. **模型加载**：使用BERT模型和分词器进行加载，准备情感分类任务。
2. **API接口**：定义API接口，接收用户输入的自然语言文本，进行情感分类，并返回分类结果。
3. **情感分类**：使用分词器对输入文本进行分词，输入到BERT模型进行情感分类，输出分类结果。
4. **结果返回**：将情感分类结果封装为JSON格式，返回给用户。
