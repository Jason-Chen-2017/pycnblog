# 2. 技术原理及概念

## 2.1. 基本概念解释

物联网是指通过信息传感设备，实现对物品和环境的智能感知和信息交互。物联网的核心技术包括传感器技术、网络通信技术、数据存储技术和应用软件等方面。

云计算是一种计算模式，通过网络将计算资源集中管理，实现大规模的计算能力。云计算的核心技术包括虚拟化技术、分布式系统、数据存储和计算资源调度等方面。

容器化技术是一种轻量级的虚拟化技术，可以将应用程序及其依赖关系打包成独立的可移植的容器运行时环境。容器化技术的核心技术包括Docker、Kubernetes等技术。
