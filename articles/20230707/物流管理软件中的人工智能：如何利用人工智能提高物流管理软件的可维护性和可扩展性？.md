### 2.5. 数学公式

在机器学习算法中，常用的数学公式包括：

1) 平均准确率（Mean Accuracy）：指预测准确的样本数占总样本数的比例。

2) 召回率（Recall）：指实际为阳性样本的样本数占总实际为阳性样本的样本数的比例。

3) 精确率（Precision）：指预测为阳性样本的样本数占总预测为阳性样本的样本数的比例。

4) F1 值（F1-Score）：综合考虑精确率、召回率和准确率，是衡量模型性能的最佳指标。

### 2.5. 数学公式

在机器学习算法中，常用的数学公式包括：

1) 平均准确率（Mean Accuracy）：指预测准确的样本数占总样本数的比例。公式为：

平均准确率 = (TP) / (TP + FP)，其中TP为真正为阳性样本的样本数，FP为真正为阴性样本的样本数。

2) 召回率（Recall）：指实际为阳性样本的样本数占总实际为阳性样本的样本数的比例。公式为：

召回率 = TP / (TP + FN)，其中TP为真正为阳性样本的样本数，FN为真正为阴性样本的样本数。

3) 精确率（Precision）：指预测为阳性样本的样本数占总预测为阳性样本的样本数的比例。公式为：

精确率 = TP / (TP + FN)，其中TP为真正为阳性样本的样本数，FN为预测为阴性样本的样本数。

4) F1 值（F1-Score）：综合考虑精确率、召回率和准确率，是衡量模型性能的最佳指标。公式为：

F1 值 = 2 * Precision * Recall / (Precision + Recall)
