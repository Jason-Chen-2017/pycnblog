模型量化的代码实现通常使用编程语言完成,例如Python等。下面给出一个Python的代码实例,用于计算欧式期权的价格:

```
import numpy as np
from scipy.stats import option

S = 50    # 标的资产价格
K = 100    # 行权价格
T = 1     # 期权到期时间(以年为单位)
r = 0.05   # 无风险利率

d1 = (np.log(S / K) + (r + 0.5 * T * r) * T) / (np.sqrt(T * (S / K)) + 1)
d2 = d1 - T * r

P = option.BlackScholes(S, K, T, r, d1, d2)

print(P.price)
```
