### 第一步：导入必要的库和工具

```python
import pandas as pd

```

### 第二步：数据预处理

```python
# 读取数据
iris = load_iris()

# 将数据分为训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(iris.data, iris.target, test_size=0.2)

# 将数据转换为NumPy数组
X_train = X_train.astype('float32')
X_test = X_test.astype('float32')
```

### 第三步：特征提取

```python
# 从X_train中提取特征
features = []
for feature in X_train.columns:
    features.append(X_train[feature].mean())

# 从X_test中提取特征
features_test = []
for feature in X_test.columns:
    features_test.append(X_test[feature].mean())
```

### 第四步：模型选择

```python
# 选择聚类算法
kmeans = KMeans(n_clusters=3)

# 训练模型
kmeans.fit(X_train)
```

### 第五步：模型评估

```python
# 评估模型
from sklearn.metrics import silhouette_score

```
