### 2.2.1. 医疗影像识别

医疗影像识别是人工智能医疗的核心技术之一，其目的是让计算机通过对医学影像的分析，实现自动识别和定位疾病部位、细胞变化等，从而提高医学影像的准确度和医生诊断的效率。

```python
import numpy as np
import cv2

# 读取医学影像
img = cv2.imread('medical_image.jpg')

# 定义病理性区域范围
threshold = 0.6

# 创建掩膜，用于寻找病理性区域
mask = cv2.threshold(img, threshold, 255, cv2.THRESH_BINARY)[1]

# 查找病理性区域
ret, thresh = cv2.threshold(mask, threshold, 255, cv2.THRESH_BINARY)
contours, hierarchy = cv2.findContours(thresh, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

# 绘制轮廓
img = cv2.drawContours(img, contours, -1, (0, 0, 255), 1)
```
