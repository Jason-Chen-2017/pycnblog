2.2.4. 代码实例和解释说明

以下是一个简单的Thrust代码示例，用于计算向量的长度：
```c++
#include <thrust/container/transform_ algorithms.hpp>

using namespace std;
using namespace thrust;

int main() {
    vector<int> vec = {1, 2, 3, 4, 5};
    double len = vector<int>::length(vec);
    cout << "Vector length: " << len << endl;
    return 0;
}
```
在Thrust中，使用`length`函数可以轻松地获取向量的长度。
