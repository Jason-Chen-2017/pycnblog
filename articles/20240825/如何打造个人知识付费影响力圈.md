### 5.2 源代码详细实现

```python
# 定义函数计算影响力
def calculate_influence(data):
    # 计算总得分
    total_score = sum(data['得分'])
    
    # 计算影响力
    influence = total_score / len(data)
    
    return influence

# 调用函数计算影响力
influence = calculate_influence(data)

# 打印影响力
print("影响力得分：", influence)
```

最后，我们调用函数`calculate_influence`计算影响力得分，并打印结果。
