### 4.2 公式推导过程

用户兴趣模型可以通过贝叶斯定理进行推导。假设用户对某个兴趣点的兴趣概率为P(Interest|User Behavior)，其中Interest表示兴趣点，User Behavior表示用户的行为数据。根据贝叶斯定理，有：

P(Interest|User Behavior) = P(User Behavior|Interest) * P(Interest) / P(User Behavior)

其中，P(User Behavior|Interest)表示在给定兴趣点的情况下，用户的行为数据的概率；P(Interest)表示兴趣点的概率；P(User Behavior)表示用户的行为数据的概率。

首先，我们需要定义用户A的行为数据集合User Behavior A，以及商品B的兴趣点Interest B。然后，我们需要计算P(User Behavior A|Interest B)、P(Interest B)和P(User Behavior A)。

通过收集用户A的历史行为数据和商品B的属性数据，我们可以计算出上述三个概率值。最终，我们可以得到用户A对商品B的兴趣概率P(Interest B|User Behavior A)。根据这个概率值，我们可以决定是否向用户A推荐商品B。

import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB

# 读取用户行为数据
user_behavior = pd.read_csv('user_behavior.csv')

# 构建词袋模型
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(user_behavior['text'])

# 分割训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, user_behavior['label'], test_size=0.2, random_state=42)

# 使用朴素贝叶斯模型进行分类
model = MultinomialNB()
model.fit(X_train, y_train)

# 预测测试集
y_pred = model.predict(X_test)

# 评估模型性能
print("Accuracy:", model.score(X_test, y_test))
