2. 学生在课程学习过程中，其成绩等信息也被实时记录在区块链上。

### 5.2 源代码详细实现

在本项目中，我们创建了一个名为`DegreeCertificate.sol`的智能合约，用于生成和验证学历证书。

```solidity
pragma solidity ^0.8.0;

contract DegreeCertificate {
    struct Student {
        string id;
        string name;
        uint256 avgScore;
        uint256 creditHours;
    }

    mapping(string => Student) public students;

    function createStudent(
        string memory id,
        string memory name,
        uint256 avgScore,
        uint256 creditHours
    ) public {
        require(students[id].id == "", "Student already exists");
        students[id] = Student(id, name, avgScore, creditHours);
    }

    function verifyStudent(string memory id) public view returns (bool) {
        Student memory student = students[id];
        require(student.id != "", "Student does not exist");

        bool passed = (
            student.avgScore >= 80 &&
            student.creditHours >= 30
        );

        return passed;
    }
}
```

### 5.3 代码解读与分析

- `createStudent`函数用于创建学生的学籍信息。
- `verifyStudent`函数用于验证学生的学历证书。
