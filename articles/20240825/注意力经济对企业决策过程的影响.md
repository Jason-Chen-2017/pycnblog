## 1. 背景介绍

在当今的信息化社会中，企业面临着海量的信息输入，如何在众多的信息中筛选出有价值的内容，成为决策过程中的关键。注意力经济，作为一种经济学理论，它描述了人类注意力的稀缺性及其分配方式，这对于企业如何有效地进行决策具有重要的启示。

### 注意力经济的概念

注意力经济是指由于注意力的稀缺性，企业或个人为了获取更多的信息或资源，需要投入更多的注意力。在注意力经济中，注意力的分配是动态的，取决于信息的价值以及获取信息的成本。

### 注意力经济与企业决策

企业决策是一个复杂的过程，涉及到信息收集、分析、评估和选择等多个环节。在信息过载的背景下，如何有效地管理和分配注意力，对于提高决策效率和质量至关重要。

import numpy as np

# 定义信息价值函数
def value_function(info):
    if info == 'A':
        return 100
    elif info == 'B':
        return 50
    else:
        return 200

# 定义获取成本函数
def cost_function(info):
    if info == 'A':
        return 20
    elif info == 'B':
        return 10
    else:
        return 30

def attention_allocation(info_list):
    values = [value_function(info) for info in info_list]
    costs = [cost_function(info) for info in info_list]
