### 2.1. 量子传感原理

下面是一个简单的量子传感器应用示例，实现量子态的制备与测量：

```python
from qiskit import QuantumCircuit, Aer, execute
from qiskit.visualization import plot_bloch_multivector
import numpy as np

# 创建量子电路
qc = QuantumCircuit(1)

# 量子态制备
qc.h(0)

# 量子测量
qc.measure_all()

# 执行量子电路

plot_bloch_multivector(qc, title='Quantum State')
```
