Python 是一种解释型、面向对象的高级编程语言，由Guido van Rossum于1989年创建，是一种多范式编程语言。它被设计用来有效地编写可读性强、结构紧凑、执行效率高、并具有动态类型系统和自动内存管理等特征的脚本或应用程序。同时支持多种编程范式，包括面向过程、命令式、函数式和面向对象。Python还适用于科学计算、Web开发、网络数据处理、人工智能、嵌入式应用和自动化运维等领域。

## 数据类型与运算符
### 数据类型
Python 中的数据类型分为以下几类：

1. 数字类型: int(整数), float(浮点数), complex(复数)。

2. 字符串类型: str(字符串)。

3. 布尔类型: bool(布尔值)。

4. 序列类型: list(列表)，tuple(元组)，range(范围)。

5. 字典类型: dict(字典)。

6. set 类型: set(集合)。

通过type()函数可以查看变量所属的数据类型。
```python
a = 1 # int
b = 3.14 #float
c = True #bool
d = "hello" #str
e = [1, 2, 3] #list
f = (4, 5) #tuple
g = range(6) #range
h = {"name": "Alice", "age": 20} #dict
i = {1, 2, 3} #set
print(type(a)) #输出int
print(type(b)) #输出float
print(type(c)) #输出bool
print(type(d)) #输出str
print(type(e)) #输出list
print(type(f)) #输出tuple
print(type(g)) #输出range
print(type(h)) #输出dict
print(type(i)) #输出set
```

## 流程控制语句
流程控制语句主要有条件语句if...else、循环语句for...in、while、break和continue。

### if...else语句
if 语句允许根据一个表达式的真假来决定执行的代码块。如下例所示，当 x 为负时，输出 "x is negative"; 如果 x 为零或正数，则输出 "x is positive".

```python
x = -5
if x < 0:
    print("x is negative")
elif x == 0:
    print("x is zero")
else:
    print("x is positive")
```

### for...in语句
for...in 语句用来遍历可迭代对象（如列表、字符串、元组）中的元素。如需指定步长，可使用第三个参数。

```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
```

### while语句
while 语句是一种循环语句，在给定的判断条件为真时，才会继续执行循环体。

```python
count = 0
while count < 5:
    print("The count is:", count)
    count += 1
```

### break语句
break 语句终止当前循环，转到循环后的语句执行。

```python
count = 0
while True:
    print("The count is:", count)
    count += 1
    if count >= 5:
        break
```

### continue语句
continue 语句跳过当前循环块中的剩余语句，直接进入下一次循环。

```python
count = 0
while count < 5:
    count += 1
    if count == 3:
        continue
    print("The current count is:", count)
```
