# 2.核心概念与联系
## Excel基础知识
### Excel简介
Excel（Officie Automation Exel，简称OAExel），是微软于2007年推出的一款免费的电子表格软件。它具有图表统计、插图、表单计算、数据验证、填充色彩、打印、保护、数据透视表等功能，能帮助用户进行数据的筛选、分析、汇总和报表的生成。
### Excel工作原理
- Excel文件是由多个工作表组成的。
- 每个工作表由若干行和若干列组成，每一个单元格都有一个值。
- 用户可以通过不同的方式输入数据，包括文本、数字、日期、百分比、货币等。
- 不同的数据类型会影响到单元格的显示效果。
- 在工作表中可以对数据进行筛选、排序、格式设置等操作，并可以通过公式应用到其他单元格。

## Word基础知识
### Word简介
Word 是 Microsoft Office 的一个应用程序，用来创建和编辑纯文本文件，可以支持多种语言和多种排版样式，非常适合编写专业文档、书籍或是管理大型信息资料。
### Word工作原理
- Word是一个多窗体的程序。
- 窗口主要分为两大部分：“视图”区和“编辑”区。视图区是整个窗口的外观；编辑区则是主要用来编辑文档的内容。
- 要创建一个新文档或者打开现有的文档，只需双击Word应用程序图标即可。

## Python基础知识
Python是一种易于学习的高级编程语言，它拥有很强大的类库和丰富的第三方工具包，可以用来进行各种各样的开发工作。
### Python简介
- Python 是一种面向对象的动态编程语言，支持多种编程范式。
- Python 拥有简洁而优美的语法风格，可读性强，适用于各种应用领域。
- Python 支持动态类型，也允许声明变量的类型。
- Python 是一个开源项目，其社区庞大且活跃。
