# 2.核心概念与联系
## 2.1 聚类
### 2.1.1 概念
聚类（Clustering）是一种无监督学习方法，它将具有相似性的数据集划分为多个子集，每个子集代表一个族群或类别。在聚类中，数据对象通常是向量形式，例如图像的像素值或文本的词频统计。聚类的目的是使得具有相似属性的数据归属于同一个子集，避免数据的混淆，提高数据的可视化能力和分析效果。
### 2.1.2 分类
按照根据输入数据类型和输出结果，可将聚类任务分为三种：
- 密度聚类：输入是距离矩阵或样本，输出是集合的划分。
- 分层聚类：输入是层次结构数据，输出是多个层次结构，每个层次代表不同的类别。
- 联合聚类：输入是多个数据集合，输出是共同的划分。

## 2.2 降维
### 2.2.1 概念
降维（Dimensionality Reduction）是指通过某种方式对数据进行压缩，使得数据呈现出较低维度的表示形式，但仍保留原数据的信息。降维的目的是为了简化数据，以便更好地进行数据的可视化、分析、处理。
### 2.2.2 分类
降维任务可以分为两类：
- 可视化降维：希望利用可视化的方式直观地展示数据分布情况，如散点图、箱形图、热力图等；
- 特征选择降维：希望基于某些特性来选择出一些特征，将这些特征整合到一起。如PCA、ICA、LDA、Sparse Autoencoder、t-SNE、UMAP等。

## 2.3 分类
### 2.3.1 概念
分类（Classification）是一种无监督学习方法，它将数据划分为不同的类别或簇。在分类中，数据对象的属性可能包括类别、颜色、大小、形状、位置等。它的目标是从给定的训练数据中找到一个模型，该模型能够对新的、未知的、带有属性的数据进行正确的分类。分类的结果可以用来确定待预测数据的类别标签或概率分布。
### 2.3.2 分类
按照根据输入数据类型和输出结果，可将分类任务分为四种：
- 多类分类：输入是实例的特征向量和类别标记，输出是各个类的概率分布。
- 二类分类：输入是实例的特征向量和类别标记，输出是只有两个类别的概率分布。
- 回归分类：输入是实例的特征向量和连续变量，输出是连续变量的值。
- 标注分类：输入是序列数据，输出是标签序列。
