# 2.核心概念与联系
## 2.1 if语句
if语句（英语：if statement）是一种用于条件判断的控制结构，其一般形式如下所示：
```python
if condition:
    # do something when condition is true
```
其中condition表示一个布尔表达式，如果该表达式的值为True，则执行缩进块里的代码；否则，不执行代码。

### 2.1.1 else子句
如果if语句的条件表达式为False，则对应的else语句会被执行，例如：
```python
a = input("请输入第一个数字：")
b = input("请输入第二个数字：")
c = a + b
if c > int(b):
    print("%d 比 %d 大"%(int(a),int(b)))
else:
    print("%d 小于等于 %d"%(int(a),int(b)))
```
上述代码输入两个整数，并求它们之和，然后比较和第二个整数的大小关系，并打印结果。如果第一个数字比第二个数字大，则显示“a 比 b 大”；否则，显示"a 小于等于 b”。

### 2.1.2 elif子句
当if和else组合使用时，我们可以添加多个elif子句，即表示若前一个条件都不满足的时候，才去尝试下一个条件。举例如下：
```python
score = int(input("请输入您的成绩："))
if score >= 90:
    grade = "A"
elif score >= 80 and score < 90:
    grade = "B"
elif score >= 70 and score < 80:
    grade = "C"
elif score >= 60 and score < 70:
    grade = "D"
else:
    grade = "F"
print("您的成绩为", score, "分, 等级为", grade)
```
以上代码通过用户输入的分数，根据分数自动计算成绩对应的等级。如果分数>=90，则等级为A；如果90>分数>=80，则等级为B；以此类推。

### 2.1.3 使用嵌套if语句
在一个if语句内部还可以包含其他的if或elif语句，称为嵌套if语句。例如：
```python
num = int(input("请输入一个数字："))
if num <= 0:
    print("无效输入！")
else:
    if num == 1:
        print("这个数字很简单")
    elif num == 2 or num == 3:
        print("这个数字很容易")
    elif num <= 10:
        for i in range(2, num):
            if num % i == 0:
                break
        else:
            print("这个数字是质数")
    else:
        print("这个数字太大了")
```
以上代码接收一个输入数字，首先判断是否为负值，若为负值则输出“无效输入”，否则继续判断数字大小：
- 如果数字小于等于0，输出“无效输入”；
- 如果数字等于1，则输出“这个数字很简单”；
- 如果数字等于2或者等于3，则输出“这个数字很容易”；
- 如果数字小于等于10，则判断它是否为质数，如果不是质数，则把2到数字-1中所有的因数都试除，只要有一个除得出来，则该数字不是质数；
- 如果数字大于10，则输出“这个数字太大了”。
