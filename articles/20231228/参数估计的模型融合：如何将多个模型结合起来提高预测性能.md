# 2.核心概念与联系

模型融合是一种将多个模型结合起来的方法，以提高预测性能的方法。这种方法的核心思想是，将多个模型的预测结果进行融合，从而获得更准确的预测结果。模型融合可以分为两种类型：参数估计融合和结构融合。

参数估计融合是一种将多个模型的参数进行融合的方法。这种方法的核心思想是，将多个模型的参数进行加权求和，从而获得一个新的模型。这种方法的优点是简单易行，但其缺点是需要对模型的参数进行手动调整。

结构融合是一种将多个模型的结构进行融合的方法。这种方法的核心思想是，将多个模型的结构进行组合，从而获得一个新的模型。这种方法的优点是可以自动学习模型的结构，但其缺点是需要对模型的结构进行手动调整。
