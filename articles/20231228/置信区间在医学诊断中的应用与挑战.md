## 3.3 置信区间的数学模型公式

在本节中，我们将详细讲解置信区间的数学模型公式。

### 3.3.1 标准误差置信区间

标准误差置信区间是一种常见的置信区间计算方法，用于估计一个参数的取值范围。其公式如下：

$$
CI = \bar{x} \pm Z \times \frac{s}{\sqrt{n}}
$$

其中，$\bar{x}$ 是样本平均值，$Z$ 是Z统计量（通常取为1.96或1.645），$s$ 是样本标准差，$n$ 是样本大小。

### 3.3.2 t统计量置信区间

t统计量置信区间是另一种常见的置信区间计算方法，用于估计一个参数的取值范围。其公式如下：

$$
CI = \bar{x} \pm t \times \frac{s}{\sqrt{n}}
$$

其中，$\bar{x}$ 是样本平均值，$t$ 是t统计量（取决于样本大小和置信度），$s$ 是样本标准差，$n$ 是样本大小。

其中，$\bar{x}$ 是样本平均值，$Z$ 是Z统计量（通常取为1.96或1.645），$s$ 是样本标准差，$n$ 是样本大小。

## 4.1 代码实例1：计算标准误差置信区间

在本代码实例中，我们将计算一个药物对疾病的有效性的标准误差置信区间。

```python
import numpy as np

# 样本数据
data = np.array([10, 12, 14, 16, 18])

# 计算样本平均值
sample_mean = np.mean(data)

# 计算样本标准差
sample_std = np.std(data)

# 计算样本大小
sample_size = len(data)

# 选择Z统计量（例如1.96）
Z = 1.96

# 计算标准误差
std_error = sample_std / np.sqrt(sample_size)

# 计算置信区间
confidence_interval = sample_mean - Z * std_error

print("置信区间：", confidence_interval)
```

在这个代码实例中，我们首先导入了numpy库，然后定义了一个样本数据列表。接着，我们计算了样本平均值、样本标准差和样本大小。然后，我们选择了一个Z统计量（例如1.96），并计算了标准误差。最后，我们计算了置信区间，并输出了结果。
