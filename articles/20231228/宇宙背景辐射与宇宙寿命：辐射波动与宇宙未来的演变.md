## 4.1 观测宇宙背景辐射的代码实例

以下是一个观测宇宙背景辐射的代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义观测设备的参数
wavelength = np.linspace(1e-3, 1, 1000)  # 微波波长区域
flux_density = 2.725 * 10**(-9) * (wavelength/1e-3)**3  # 辐射强度

# 绘制辐射强度与波长的关系
plt.plot(wavelength, flux_density)
plt.xlabel('Wavelength (m)')
plt.ylabel('Flux Density (W/m^2/Hz)')
plt.title('CMB Flux Density vs Wavelength')
plt.show()
```

在这个代码实例中，我们首先导入了numpy和matplotlib.pyplot这两个库，然后定义了观测设备的参数，包括微波波长区域和辐射强度。最后，我们绘制了辐射强度与波长的关系。
