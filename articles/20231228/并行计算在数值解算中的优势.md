1. 数据并行：数据并行是指在多个处理器上同时处理相同数据的不同部分。例如，在计算一个大矩阵的迹时，可以将矩阵划分为多个子矩阵，然后在多个处理器上同时计算每个子矩阵的迹，最后将结果汇总起来。
2. 任务并行：任务并行是指在多个处理器上同时执行不同任务。例如，在求解一个系统的方程组时，可以将问题划分为多个子问题，然后在多个处理器上同时解决这些子问题，最后将结果汇总起来。

我们以一个简单的矩阵乘法问题为例，来展示并行计算在数值解算中的优势。

```python
import numpy as np
from multiprocessing import Pool

def matrix_mul(A, B):
    return np.dot(A, B)

if __name__ == '__main__':
    A = np.random.rand(1000, 1000)
    B = np.random.rand(1000, 1000)

    with Pool(4) as pool:
        result = pool.apply_async(matrix_mul, (A, B))
        result = result.get()

    print(result)
```

在这个代码实例中，我们使用了Python的multiprocessing库来实现并行计算。我们将矩阵A和矩阵B划分为多个子矩阵，然后在多个处理器上同时计算每个子矩阵的乘积，最后将结果汇总起来。
