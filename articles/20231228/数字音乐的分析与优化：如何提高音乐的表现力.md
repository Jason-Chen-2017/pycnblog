# 4.具体代码实例和详细解释说明

以下是一个使用Python实现的音频信号的压缩和恢复的代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.fft import fft

# 生成一个随机音频信号
fs = 1000
T = 1 / fs
t = np.arange(0, 1, T)
x = np.sin(2 * np.pi * 440 * t) + np.sin(2 * np.pi * 880 * t)

# 对音频信号进行傅里叶变换
X = fft(x)

# 对傅里叶变换结果进行截取和压缩
N = 100
X_compressed = X[:N]

# 对傅里叶变换结果进行逆变换
x_reconstructed = fft.ifft(X_compressed)

# 绘制原音频信号和重构音频信号的波形
plt.plot(t, x, label='Original')
plt.plot(t, x_reconstructed.real, label='Reconstructed')
plt.legend()
plt.show()
```

这个代码实例首先生成了一个随机音频信号，然后对音频信号进行了傅里叶变换，接着对傅里叶变换结果进行了截取和压缩，最后对傅里叶变换结果进行了逆变换，从而实现了音频信号的压缩和恢复。
