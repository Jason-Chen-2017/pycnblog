以下是一个简单的代码优化实例：

```python
# 原始代码
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

# 优化后代码
def factorial(n):
    cache = {0: 1}
    def inner(n):
        if n not in cache:
            cache[n] = n * inner(n-1)
        return cache[n]
    return inner(n)
```

```python

def get_weather(city):
    response = requests.get(f'http://api.example.com/weather/{city}')
    return response.json()

# 优化后代码
import requests
import cache

def get_weather(city):
    cache_key = f'weather_{city}'
    if cache.exists(cache_key):
        return cache.get(cache_key)
    response = requests.get(f'http://api.example.com/weather/{city}')
    data = response.json()
    cache.set(cache_key, data)
    return data
```

在这个实例中，我们将原始的HTTP请求优化为了使用缓存的HTTP请求，从而减少了请求次数。
