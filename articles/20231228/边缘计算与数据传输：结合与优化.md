# 2. 核心概念与联系

## 2.1 边缘计算

边缘计算是一种在设备上进行数据处理和计算的技术，它的主要特点是：

- 降低数据传输延迟：边缘计算可以将数据处理推向设备本身，从而减少数据传输的延迟。
- 降低带宽需求：边缘计算可以减少数据传输的量，从而降低带宽需求。
- 提高系统实时性：边缘计算可以使系统更加实时，因为数据处理发生在设备本身，而不是在云端。
- 提高系统可扩展性：边缘计算可以使系统更加可扩展，因为设备之间可以相互协同，共同完成任务。

## 2.2 数据传输

数据传输是将数据从一个设备传送到另一个设备的过程。数据传输可以通过各种方式进行，如无线通信、有线通信等。数据传输的主要特点是：

- 数据传输延迟：数据传输延迟是指从发送设备到接收设备的时间。数据传输延迟取决于数据传输速度和距离。
- 数据传输带宽：数据传输带宽是指数据传输过程中可以传输的数据量。数据传输带宽取决于传输媒介和传输速度。
- 数据传输实时性：数据传输实时性是指数据传输过程中的时间紧迫性。数据传输实时性取决于传输速度和延迟。
- 数据传输可扩展性：数据传输可扩展性是指数据传输过程中的扩展能力。数据传输可扩展性取决于传输媒介和传输速度。
