## 2.1 自动驾驶级别

自动驾驶技术可以分为五个级别，从0级（完全人工驾驶）到4级（完全无人驾驶）。根据不同的级别，自动驾驶汽车的安全性要求也会有所不同。

- 级别0：全人工驾驶，无自动驾驶功能。
- 级别1：驾驶助手，辅助驾驶，但驾驶人仍需保持全身心注意。
- 级别2：自动驾驶在特定条件下，例如高速公路。
- 级别3：自动驾驶在特定条件下，但不需要驾驶人在车内。
- 级别4：完全无人驾驶，无需驾驶人。

```python
import numpy as np

class TrafficLight:
    def __init__(self, color):
        self.color = color

    def change_color(self):
        self.color = 'green' if self.color == 'red' else 'red'

class Car:
    def __init__(self, speed):
        self.speed = speed

    def accelerate(self):
        self.speed += 1

    def decelerate(self):
        self.speed -= 1

def traffic_light_control(traffic_light, car):
    if traffic_light.color == 'green' and car.speed > 60:
        car.decelerate()
        traffic_light.change_color()
    elif traffic_light.color == 'red' and car.speed < 30:
        car.accelerate()
        traffic_light.change_color()

traffic_light = TrafficLight('red')
car = Car(30)
traffic_light_control(traffic_light, car)
```
