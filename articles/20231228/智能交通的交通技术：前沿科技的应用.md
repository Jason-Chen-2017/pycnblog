                 

# 1.背景介绍

智能交通是一种利用信息技术、通信技术、电子技术、自动化技术、机械电气技术等多种技术为交通运输系统创造智能化、可控制、可见化、可扩展的交通运输系统而产生的新型交通运输方式和技术。智能交通的核心是通过大数据、人工智能、云计算等前沿科技的应用，实现交通运输系统的智能化、可控制、可见化、可扩展。

智能交通技术的发展已经进入了关键时期，它不仅是国家战略和城市发展的重要组成部分，还是全球化和资源紧缺的重要反应。智能交通技术的发展需要大量的技术人才和资源投入，同时也需要政策支持和社会认可。
