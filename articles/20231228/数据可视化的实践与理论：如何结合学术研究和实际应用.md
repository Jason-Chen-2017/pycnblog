# 4.具体代码实例和详细解释说明

## 4.1 条形图

条形图是一种常用的数据可视化图表类型，用于表示数据的分布和关系。条形图可以是水平的（如条形图）或垂直的（如柱状图）。条形图可以是单一的（如单个条形）或多个的（如多个条形）。

以下是一个简单的Python代码实例，使用matplotlib库创建一个柱状图：

```python
import matplotlib.pyplot as plt

# 数据
categories = ['A', 'B', 'C', 'D', 'E']
values = [10, 20, 30, 40, 50]

# 创建柱状图
plt.bar(categories, values)

# 设置图表标题和坐标轴标签
plt.title('柱状图示例')
plt.xlabel('分类')
plt.ylabel('值')

# 显示图表
plt.show()
```

## 4.2 饼图

饼图是一种常用的数据可视化图表类型，用于表示数据的比例和占比。饼图可以是三维的（如饼图）或二维的（如环形图）。饼图可以是单一的（如单个饼图）或多个的（如多个饼图）。

以下是一个简单的Python代码实例，使用matplotlib库创建一个饼图：

```python
import matplotlib.pyplot as plt

# 数据
sizes = [10, 20, 30, 40, 50]
slices = ['A', 'B', 'C', 'D', 'E']

# 创建饼图
plt.pie(sizes, labels=slices, autopct='%1.1f%%')

# 设置图表标题和坐标轴标签
plt.title('饼图示例')

# 显示图表
plt.show()
```

## 4.3 散点图

散点图是一种常用的数据可视化图表类型，用于表示数据的关系和相关性。散点图可以是二维的（如散点图）或三维的（如三维散点图）。散点图可以是单一的（如单个散点）或多个的（如多个散点）。
