# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的文本分析案例为例，展示如何使用Python编程语言和NLTK自然语言处理库，对一段神话与传说文本进行分析。

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
from nltk.probability import FreqDist

# 文本内容
text = "神话与传说是古人用言语、艺术表现的信仰，它们体现了古人对自然、生活、道德等方面的认识和感受。"

# 分词
tokens = word_tokenize(text)

# 去除停用词
stop_words = set(stopwords.words('english'))
filtered_tokens = [w for w in tokens if not w in stop_words]

# 词频分析
fdist = FreqDist(filtered_tokens)

# 打印词频分布
print(fdist)
```

在这个例子中，我们首先导入了nltk库和相关的模块。然后，我们定义了一个文本内容，并使用nltk库的`word_tokenize`函数进行分词。接着，我们使用nltk库的`stopwords`函数获取了停用词列表，并使用列表推导式筛选出非停用词的分词结果。最后，我们使用nltk库的`FreqDist`函数进行词频分析，并使用`print`函数打印出词频分布。
