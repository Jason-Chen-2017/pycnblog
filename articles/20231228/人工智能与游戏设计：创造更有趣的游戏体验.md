# 2.核心概念与联系
在游戏设计中，人工智能主要用于创造更有智能性和复杂性的非人角色（NPC），使得玩家在游戏中可以与更有智能的对手或伙伴互动。人工智能在游戏中的核心概念包括：

1. **决策**：人工智能系统需要在游戏过程中进行决策，以便在不同的情境下采取不同的行动。
2. **学习**：人工智能系统可以通过学习来提高其在游戏中的表现，以便在与玩家互动时更有效地适应不同的情况。
3. **感知**：人工智能系统需要对游戏环境进行感知，以便获取有关游戏状态和环境的信息。
4. **行动**：人工智能系统需要对游戏环境进行行动，以便实现其决策的目标。

这些概念之间的联系如下：感知、决策、行动构成了人工智能系统在游戏中的基本行为循环，而学习则是人工智能系统在游戏中提高表现的关键。
