# 4. 具体代码实例和详细解释说明

在这里，我们以一个简单的Python代码实例来展示大数据分析在旅游团队建设中的应用：

```python
import pandas as pd
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

# 加载数据
data = pd.read_csv('customer_data.csv')

# 数据预处理
data = data.drop_duplicates()
data = data.fillna(0)
data = data.astype(int)

# 聚类分析
kmeans = KMeans(n_clusters=3)
kmeans.fit(data)
labels = kmeans.predict(data)

# 评估聚类效果
score = silhouette_score(data, labels)
print('聚类效果：', score)
```

在这个代码实例中，我们首先使用pandas库加载客户数据，然后对数据进行清洗和预处理。接着，我们使用sklearn库中的KMeans算法进行聚类分析，并获取聚类结果。最后，我们使用silhouette_score函数评估聚类效果。
