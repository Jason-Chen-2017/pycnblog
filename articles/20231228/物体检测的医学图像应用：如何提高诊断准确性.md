深度学习技术在医学图像的物体检测中主要使用卷积神经网络（CNN）算法。CNN算法是一种特殊的神经网络，它可以自动学习从图像中抽取出的特征，从而实现对医学图像的高精度检测。

具体操作步骤如下：

2. 训练卷积神经网络：使用大量的医学图像数据进行训练，以便于网络学习出特征。

3. 进行物体检测：使用训练好的卷积神经网络对新的医学图像进行检测，以便于诊断疾病。

数学模型公式详细讲解：

卷积神经网络主要包括以下几个部分：

- 卷积层：卷积层是卷积神经网络的核心部分，它可以自动学习从图像中抽取出的特征。卷积层使用卷积核（filter）来对输入的图像进行卷积操作，以便于提取特征。卷积核是一种权重矩阵，它可以学习从图像中抽取出的特征。

- 激活函数：激活函数是卷积神经网络中的一个关键组件，它可以使网络具有非线性性。常见的激活函数包括sigmoid、tanh和ReLU等。

- 池化层：池化层是卷积神经网络中的另一个重要组件，它可以用来减少图像的分辨率，以便于提高训练速度和减少计算量。池化层使用最大池化或平均池化来对输入的图像进行操作。

- 全连接层：全连接层是卷积神经网络中的最后一个部分，它可以将输入的特征映射到输出层。全连接层使用权重矩阵来对输入的特征进行操作，以便于得到最终的检测结果。

```python
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense
```

```python
model = Sequential()
model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(224, 224, 3)))
model.add(MaxPooling2D((2, 2)))
model.add(Conv2D(64, (3, 3), activation='relu'))
model.add(MaxPooling2D((2, 2)))
model.add(Conv2D(128, (3, 3), activation='relu'))

model.add(Flatten())
model.add(Dense(512, activation='relu'))
model.add(Dense(1, activation='sigmoid'))
```
接下来，我们需要编译模型：
```python
model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])
```

接下来，我们需要训练模型：
