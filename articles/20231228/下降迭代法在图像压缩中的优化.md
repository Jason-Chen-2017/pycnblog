# 2.核心概念与联系

下降迭代法（Iterative Relaxation Method）是一种迭代优化算法，它通过在图像数据上进行多次迭代计算，逐步近似解压缩后的图像。下降迭代法主要包括两种类型：一种是热门下降迭代法（Hot Pursuit），另一种是基于下降迭代法的图像压缩（IRC）。

热门下降迭代法是一种基于下降迭代法的图像压缩方法，它通过在图像数据上进行多次迭代计算，逐步近似解压缩后的图像。热门下降迭代法的主要优点是它可以实现高压缩率和高质量压缩，但其主要缺点是它需要较长的计算时间。

基于下降迭代法的图像压缩（IRC）是一种基于下降迭代法的图像压缩方法，它通过在图像数据上进行多次迭代计算，逐步近似解压缩后的图像。基于下降迭代法的图像压缩的主要优点是它可以实现高压缩率和较短的计算时间，但其主要缺点是它需要较高的计算资源。
