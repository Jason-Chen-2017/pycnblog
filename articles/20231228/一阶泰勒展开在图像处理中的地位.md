### 3.1 核心算法原理

### 3.2 具体操作步骤

1. 对于给定的函数f(x)，计算其在x0处的导数f'(x0)。
2. 计算积分的近似值，即一阶泰勒展开的近似值。
3. 将积分的近似值与原函数f(x)进行比较，以评估一阶泰勒展开的近似准确度。

### 3.3 数学模型公式详细讲解

一阶泰勒展开的数学模型公式为：

$$
f(x) \approx f(x_0) + f'(x_0)(x - x_0)
$$

其中，f(x)是原函数，x0是近似点，f'(x0)是原函数在x0处的导数。

def first_taylor_expansion(f, x0, h=1e-6):
    f_x0 = f(x0)
    f_prime_x0 = (f(x0 + h) - f(x0 - h)) / (2 * h)
    return f_x0 + f_prime_x0 * (x - x0)
