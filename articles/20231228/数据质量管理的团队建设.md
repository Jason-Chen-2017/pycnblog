### 4.1 数据清洗

以下是一个简单的数据清洗代码实例：

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.dropna(subset=['age'])  # 删除年龄为空的记录
data = data[data['age'] > 0]  # 删除年龄为负的记录
data = data[data['age'] < 150]  # 删除年龄超过150的记录
data = data.replace(to_replace='', value=None)  # 将空字符串替换为None
data = data.replace(to_replace='unknown', value=None)  # 将'unknown'替换为None
```

在这个代码实例中，我们首先使用pandas库读取数据，然后对数据进行清洗。具体来说，我们首先删除年龄为空的记录，然后删除年龄为负或超过150的记录，最后将空字符串和'unknown'替换为None。
