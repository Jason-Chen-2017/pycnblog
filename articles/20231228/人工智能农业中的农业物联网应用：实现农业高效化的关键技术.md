## 4.1 示例：智能灌溉系统
智能灌溉系统是农业物联网应用的一个典型示例，它通过对土壤湿度的实时监测，实现农田的智能灌溉。

### 4.1.1 传感器接收端代码
```python
import time
import RPi.GPIO as GPIO
from grove import GrovePiSensor

# 初始化传感器
GPIO.setmode(GPIO.BOARD)
soil_moisture = GrovePiSensor(GPIO.BOARD.GPIO_PIN_11)

# 传感器数据采集
def collect_data():
    while True:
        soil_moisture.read()
        time.sleep(1)
```
### 4.1.2 云计算平台代码
```python
import requests

# 数据传输
def send_data(data):
    url = 'https://api.example.com/soil_moisture'
    headers = {'Content-Type': 'application/json'}
    response = requests.post(url, data=data, headers=headers)
    return response.json()

# 数据解码
def decode_data(data):
    soil_moisture = data['soil_moisture']
    return soil_moisture
```
### 4.1.3 智能决策模块代码
```python
import numpy as np

# 智能决策
def make_decision(soil_moisture):
    threshold = 0.6
    if soil_moisture < threshold:
        return 'irrigate'
    else:
        return 'no_irrigate'
```
### 4.1.4 主程序代码
```python
if __name__ == '__main__':
    data = collect_data()
    response = send_data(data)
    soil_moisture = decode_data(response)
    decision = make_decision(soil_moisture)
    print(f'决策结果：{decision}')
```
在这个示例中，我们首先通过传感器接收端收集到土壤湿度数据，然后将数据传输到云计算平台，在云计算平台上对数据进行解码，并通过智能决策模块实现智能决策。最后，将决策结果输出。
