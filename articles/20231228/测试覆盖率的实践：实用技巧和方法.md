$$
Coverage = \frac{CoveredLines}{TotalLines}
$$

其中，$CoveredLines$ 表示被测试代码中被覆盖的代码行数，$TotalLines$ 表示被测试代码中的总代码行数。

$$
ConditionCoverage = \frac{CoveredConditions}{TotalConditions}
$$

其中，$CoveredConditions$ 表示被测试代码中被覆盖的条件数，$TotalConditions$ 表示被测试代码中的总条件数。

```python

        return x + y
    else:
        return x - y
```

我们可以使用Python的`coverage`库来计算代码覆盖率。首先，我们需要安装`coverage`库：

```bash
pip install coverage
```

然后，我们可以使用`coverage`库来计算代码覆盖率，如下所示：

```python
import coverage

cov = coverage.Coverage()
cov.start()

        return x + y
    else:
        return x - y

cov.stop()
cov.save()
cov.report()
```
