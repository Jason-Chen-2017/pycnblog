## 3.1 机器学习在医疗诊断中的应用
机器学习在医疗诊断中的应用主要是将机器学习算法应用到医疗数据中，以提高诊断准确性和降低诊断成本。常见的机器学习算法有：

- 逻辑回归：用于二分类问题，可以用来预测患者是否患上某种疾病。公式为：
$$
P(y=1|x) = \frac{1}{1+e^{-(\beta_0+\beta_1x_1+\cdots+\beta_nx_n)}}
$$
- 支持向量机：用于多分类问题，可以用来预测患者患病的类型。公式为：
$$
f(x) = \text{sgn}(\alpha_0 + \alpha_1K(x_1,x) + \cdots + \alpha_nK(x_n,x))
$$
- 随机森林：用于回归问题，可以用来预测患者病情发展的趋势。公式为：
$$
\hat{y} = \frac{1}{m}\sum_{i=1}^m f_i(x)
$$
其中，$f_i(x)$ 是由随机森林中的第 i 棵树生成的预测值。

## 3.2 深度学习在金融风险评估中的应用
深度学习在金融风险评估中的应用主要是将深度学习算法应用到金融数据中，以提高风险预测准确性和降低风险评估成本。常见的深度学习算法有：

- 卷积神经网络：用于图像分类问题，可以用来预测股票价格的涨跌。公式为：
$$
y = \text{softmax}(Wx + b)
$$
- 循环神经网络：用于时间序列预测问题，可以用来预测货币汇率。公式为：
$$
h_t = \text{tanh}(W_{hh}h_{t-1} + W_{xh}x_t + b_h)
$$
- 自编码器：用于降维问题，可以用来预测企业财务风险。公式为：
$$
\min_E \min_{G} \lVert x - G(E(x)) \rVert^2
$$
其中，$E$ 是编码器，$G$ 是解码器。

import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def cost_function(y, y_hat):
    return -np.sum(y * np.log(y_hat) + (1 - y) * np.log(1 - y_hat)) / len(y)

def gradient_descent(X, y, learning_rate, num_iters):
    m, n = X.shape
    weights = np.zeros((n, 1))
    for _ in range(num_iters):
        y_hat = sigmoid(X.dot(weights))
        dw = (y - y_hat).dot(X.T).dot(y_hat) / m + (1 - y_hat) * (1 - y) / m
        weights -= learning_rate * dw
    return weights

X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
