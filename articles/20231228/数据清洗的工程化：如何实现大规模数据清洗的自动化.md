# 4.具体代码实例和详细解释说明
## 4.1 数据整理
```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 数据整理
data = data.dropna(subset=['age']) # 删除年龄缺失值
data = data[data['age'] > 0] # 删除年龄为0的值
data = data.astype(int) # 将所有列转换为整型
```
## 4.2 数据校验
```python
# 数据校验
data = data[data['age'] > 0] # 检查年龄是否大于0
data = data[data['age'] < 150] # 检查年龄是否小于150
```
## 4.3 数据修复
```python
# 数据修复
data['age'].fillna(data['age'].mean(), inplace=True) # 填充年龄缺失值
```
## 4.4 数据过滤
```python
# 数据过滤
data = data[data['age'] < 60] # 去除年龄大于等于60的值
```
