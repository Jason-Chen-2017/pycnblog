# 2.核心概念与联系
在计算几何中，我们需要处理的数据通常是高维的。高维空间是指有多个维度的空间，这些维度可以是实数或复数。高维空间的一个基本特征是，它的维数越高，我们对其的直观理解就越困难。因此，在处理高维空间问题时，我们需要找到一种抽象的方法来表示和操作这些空间。
