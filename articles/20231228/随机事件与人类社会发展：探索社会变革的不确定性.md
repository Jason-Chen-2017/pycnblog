# 4.具体代码实例和详细解释说明
以下是一个使用蒙特卡洛方法进行随机事件分析的具体代码实例：

```python
import random

def random_event(probability):
    return random.random() < probability

def simulate(trials, probability):
    successes = 0
    for _ in range(trials):
        if random_event(probability):
            successes += 1
    return successes / trials

trials = 10000
probability = 0.5
successes = simulate(trials, probability)
print(f"Successes: {successes}")
```
