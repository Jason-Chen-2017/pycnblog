## 4. 具体代码实例和详细解释说明

在这里，我们以 Python 语言为例，使用 scikit-learn 库来实现凸集分离定理的图像分割。

```python
from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# 加载鸢尾花数据集
data = datasets.load_iris()
X = data.data
y = data.target

# 将数据集分为两个类别
X1 = X[y == 0]
X2 = X[y == 1]
y1 = [0] * len(X1)
y2 = [1] * len(X2)

# 将两个类别的数据集分割为训练集和测试集
X1_train, X1_test, y1_train, y1_test = train_test_split(X1, y1, test_size=0.2, random_state=42)
X2_train, X2_test, y2_train, y2_test = train_test_split(X2, y2, test_size=0.2, random_state=42)

# 使用支持向量机进行分类
clf = SVC(kernel='linear')
clf.fit(X1_train, y1_train)
y_pred1 = clf.predict(X1_test)
accuracy1 = accuracy_score(y1_test, y_pred1)

clf.fit(X2_train, y2_train)
y_pred2 = clf.predict(X2_test)
accuracy2 = accuracy_score(y2_test, y_pred2)

print('准确度：', accuracy1, accuracy2)
```

在这个代码实例中，我们首先使用 scikit-learn 库加载了鸢尾花数据集，然后将数据集分为两个类别。接着，我们将这两个类别的数据集分割为训练集和测试集。最后，我们使用支持向量机进行分类，并计算出准确度。
