```python
import numpy as np

def ice_sheet_melt(T, h, dt):
    """
    极地冰川融化模型

    Parameters:
    T (numpy.ndarray): 冰川表面温度
    h (numpy.ndarray): 冰川高度
    dt (float): 时间步长

    Returns:
    numpy.ndarray: 冰川融化率
    """
    # 计算冰川融化率
    melt_rate = np.where(T > 0, h * dt, 0)

    return melt_rate
```

这个函数接受冰川表面温度、冰川高度和时间步长作为输入参数，并返回冰川融化率。具体实现如下：

1. 首先，我们使用 `numpy.where` 函数来判断冰川表面温度是否大于0。如果大于0，则表示冰川正在融化，此时将冰川高度与时间步长相乘，得到融化率；如果小于等于0，则表示冰川未融化，此时融化率为0。

2. 最后，我们将融化率作为函数的返回值。
