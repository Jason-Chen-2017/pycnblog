# 2. 核心概念与联系

在全球供应链中，海运和航空是两个最关键的环节。海运主要用于运输大量的商品，如矿产、原材料和制造品，而航空则主要用于运输人员和敏感的、高价值的货物。这两个环节之间存在着紧密的联系，它们共同构成了全球供应链的生命线。

在优化全球供应链时，我们需要关注以下几个核心概念：

1. 运输成本：运输成本是全球供应链的关键因素之一，它包括运输费用、仓储费用、保险费用等。降低运输成本可以提高企业的盈利能力。

2. 运输时间：运输时间是全球供应链的另一个关键因素，它决定了商品从生产地到消费地的时间。缩短运输时间可以提高供应链的敏捷性和可靠性。

3. 环境影响：全球供应链在运输过程中产生了大量的碳排放和其他环境污染，这对环境和人类健康产生了严重影响。因此，在优化全球供应链时，我们需要关注环境因素。
