## 4.1 数据隐私保护

```python
import hashlib

def anonymize(data):
    for key, value in data.items():
        if isinstance(value, str):
            data[key] = hashlib.sha256(value.encode()).hexdigest()
    return data
```

在这个代码示例中，我们使用了SHA-256算法对敏感信息进行了加密，从而保护了数据隐私。
