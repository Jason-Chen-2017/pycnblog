```python
def calculate_heart_rate(bpm):
    heart_rate = (60 * bpm) / (60 * 1000)
    return heart_rate

def calculate_blood_pressure(sbp, dbp):
    blood_pressure = (sbp + dbp) / 2
    return blood_pressure

bpm = 75
sbp = 120
dbp = 80

heart_rate = calculate_heart_rate(bpm)
blood_pressure = calculate_blood_pressure(sbp, dbp)

print("Heart rate: ", heart_rate)
print("Blood pressure: ", blood_pressure)
```

这个代码首先定义了两个函数：`calculate_heart_rate` 和 `calculate_blood_pressure`。然后，它们分别使用了心率计算公式和血压计算公式来计算心率和血压。最后，它们将计算结果打印出来。
