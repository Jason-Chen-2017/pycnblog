# 2.核心概念与联系
在了解迁移过程之前，我们需要了解一些关键概念。传统数据库通常包括关系数据库和文档数据库。关系数据库使用表格结构存储数据，而文档数据库使用文档结构存储数据。ArangoDB是一个多模型数据库，它支持这两种数据模型，并且还支持图形数据模型。

## 2.1 关系数据库
关系数据库是一种基于表的数据库，其中数据以表格形式存储。每个表包含一组相关的数据，这些数据通过一组关系来连接。关系数据库通常使用SQL进行查询和操作。

## 2.2 文档数据库
文档数据库是一种基于文档的数据库，其中数据以文档形式存储。每个文档是一个独立的数据结构，可以包含多种数据类型，如字符串、数字、列表等。文档数据库通常使用NoSQL技术进行查询和操作。

## 2.3 ArangoDB
ArangoDB是一个多模型数据库，它支持文档、关系和图形数据模型。这意味着ArangoDB可以处理关系数据库和文档数据库的数据，并且还可以处理图形数据。ArangoDB使用AQL（ArangoDB查询语言）进行查询和操作。
