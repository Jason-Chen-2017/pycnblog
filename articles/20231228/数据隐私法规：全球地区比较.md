## 2.核心概念与联系

在全球范围内，数据隐私法规的核心概念主要包括：

1. 个人信息：个人信息是指能够单独识别某个特定个人的任何信息。这包括但不限于姓名、身份证号码、住址、电话号码、电子邮件地址、身份证明文件、健康记录、生物特征信息等。

3. 数据泄露：数据泄露是指因为组织或个人未能采取适当措施保护个人信息而导致个人信息被未经授权访问、传播或滥用的事件。
4. 数据保护权：数据保护权是指个人对于他们的个人信息所具有的权利，包括但不限于权利要求组织或个人停止处理个人信息、删除个人信息等。

# 生成混淆向量
def generate_noise(data, noise_level):
    noise = np.random.normal(0, noise_level, data.shape)
    return data + noise

# 移除混淆向量
def remove_noise(data, noise_level):
    return data - np.random.normal(0, noise_level, data.shape)

# 测试混淆器

noise_level = 1

print("Noisy data:", noisy_data)

print("Clean data:", clean_data)
