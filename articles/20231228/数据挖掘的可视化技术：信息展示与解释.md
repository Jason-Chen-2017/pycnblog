# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来展示如何使用Python的matplotlib库来实现数据可视化。

```python
import matplotlib.pyplot as plt
import numpy as np

# 创建一组随机数据
x = np.random.rand(100)
y = np.random.rand(100)

# 创建一个散点图
plt.scatter(x, y)

# 添加标签和标题
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.title('Scatter plot example')

# 显示图表
plt.show()
```

在这个例子中，我们首先导入了matplotlib和numpy库。然后，我们创建了一组随机数据，并使用scatter函数来创建一个散点图。接着，我们使用xlabel、ylabel和title函数来添加标签和标题。最后，我们使用show函数来显示图表。
