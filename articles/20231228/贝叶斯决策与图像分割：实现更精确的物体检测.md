# 2.核心概念与联系

## 2.1 贝叶斯决策理论

贝叶斯决策理论是基于贝叶斯定理的决策理论，它提供了一种基于概率模型的决策规则。贝叶斯决策理论的核心思想是将不确定性模型为概率分布，并根据这些分布进行决策。

贝叶斯决策理论的主要思路如下：

1. 对于每个可能的决策选项，建立一个概率模型，描述不同决策选项的结果。
2. 根据这些概率模型，计算每个决策选项的期望收益。
3. 选择那些收益最高的决策选项。

在计算机视觉中，贝叶斯决策理论可以用于解决多种问题，包括图像分割和物体检测。

## 3.1 贝叶斯决策理论的数学模型

贝叶斯决策理论的数学模型主要包括以下几个部分：

1. 观测值：观测值是决策过程中的输入，可以是图像数据、特征向量等。
2. 决策选项：决策选项是决策过程中的输出，可以是物体分类、物体位置等。
3. 概率模型：概率模型描述了不同决策选项的结果，可以是观测值给定决策选项的概率分布，也可以是决策选项给定观测值的概率分布。
4. 损失函数：损失函数描述了决策选项与实际结果之间的差异，用于计算决策选项的收益。

贝叶斯决策理论的目标是找到一种决策策略，使得预期收益最大。预期收益可以通过以下公式计算：

$$
E[U(d)] = \sum_{y} P(y|d)U(d,y)
$$

其中，$E[U(d)]$表示预期收益，$P(y|d)$表示给定决策选项$d$时，实际结果为$y$的概率，$U(d,y)$表示决策选项$d$和实际结果$y$之间的收益。
