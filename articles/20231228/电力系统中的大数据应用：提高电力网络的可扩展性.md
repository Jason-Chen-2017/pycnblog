# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的电力需求预测示例来展示大数据技术在电力系统中的应用。

1. 数据收集：从电力公司提供的API获取电力需求数据。
2. 数据存储：将数据存储到HDFS中。
3. 数据清洗：使用Apache Nifi将数据从API获取到HDFS。
4. 数据分析：使用Apache Spark计算电力需求的平均值、最大值、最小值等统计信息。
5. 数据挖掘：使用Apache Spark MLlib构建线性回归模型，预测未来的电力需求。

具体代码实例如下：

```python
from pyspark.sql import SparkSession
from pyspark.ml.regression import LinearRegression

# 创建SparkSession
spark = SparkSession.builder.appName("PowerDemandPrediction").getOrCreate()

# 读取数据
df = spark.read.json("hdfs://localhost:9000/power_demand.json")

# 数据预处理
df = df.select(df["date"], df["power_demand"])
df = df.withColumn("date", df["date"].cast("int"))
df = df.withColumn("power_demand", df["power_demand"].cast("double"))

# 数据分析
avg_power_demand = df.select(df.agg({"power_demand": "avg"})).collect()[0][0]
max_power_demand = df.select(df.agg({"power_demand": "max"})).collect()[0][0]
min_power_demand = df.select(df.agg({"power_demand": "min"})).collect()[0][0]

# 数据挖掘
linear_regression = LinearRegression(featuresCol="date", labelCol="power_demand")
model = linear_regression.fit(df)
predictions = model.transform(df)

# 输出预测结果
predictions.select("date", "power_demand", "prediction").show()
```
