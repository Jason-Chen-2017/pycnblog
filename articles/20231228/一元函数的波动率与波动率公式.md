# 4.具体代码实例和详细解释说明

## 4.1 波动率公式实例

```python
import numpy as np

# 生成一组随机数据
np.random.seed(0)
X = np.random.normal(0, 1, 1000)

# 计算波动率
mean = np.mean(X)
variance = np.var(X)
sigma = np.sqrt(variance)
print("波动率:", sigma)
```

上述代码首先生成了一组随机数据，然后计算了该数据的均值、方差和波动率。最后，输出了波动率的值。
