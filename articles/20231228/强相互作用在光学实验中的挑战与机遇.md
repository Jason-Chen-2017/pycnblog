## 4.3 光学模拟与计算代码实例

```python
import numpy as np
import scipy.integrate as spi
import matplotlib.pyplot as plt

def dydt(y, t, gamma, omega0):
    dy = np.array([y[2], -gamma * y[1] - omega0**2 * y[0]])
    return dy

def runge_kutta(y, t, h, gamma, omega0):
    k1 = h * dydt(y, t, gamma, omega0)
    k2 = h * dydt(y + k1 / 2, t + h / 2, gamma, omega0)
    k3 = h * dydt(y + k2 / 2, t + h / 2, gamma, omega0)
    k4 = h * dydt(y + k3, t + h, gamma, omega0)
    return y + (k1 + 2 * k2 + 2 * k3 + k4) / 6

gamma = 0.1
omega0 = 1
t = np.linspace(0, 10, 1000)
y = np.array([0, 0, 1])

for t_step in t:
    y = runge_kutta(y, t_step, 0.01, gamma, omega0)

plt.plot(t, y[0])
plt.xlabel('Time')
plt.ylabel('Displacement')
plt.title('Damped Oscillator')
plt.show()
```

上述代码实例中，我们定义了一个谐振模拟问题，并使用Runge-Kutta方法进行解析解。我们绘制了谐振的振幅与时间的关系。
