## 4.1求解方程的代码实例

我们来看一个求解一元方程的代码实例：

```python
import numpy as np

def f(x):
    return x**2 - 4

def df(x):
    return 2*x

def newton_raphson(x0, tol=1e-6, max_iter=100):
    x = x0
    for _ in range(max_iter):
        dx = f(x) / df(x)
        x -= dx
        if abs(dx) < tol:
            break
    return x

x0 = 1
root = newton_raphson(x0)
print("方程的根为：", root)
```

在这个例子中，我们使用牛顿法来求解一元方程$x^2 - 4 = 0$的根。我们首先定义了函数$f(x)$和其导数$df(x)$，然后使用牛顿法来求解方程的根。
