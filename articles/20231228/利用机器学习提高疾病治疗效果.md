# 2. 核心概念与联系

在开始讨论如何利用机器学习提高疾病治疗效果之前，我们需要了解一些核心概念。

## 2.1 机器学习

机器学习是一种自动学习和改进的方法，它可以帮助计算机程序自动学习和改进，以解决复杂的问题。机器学习的主要任务是通过学习从数据中得出规律，从而进行预测和决策。

机器学习可以分为两个主要类别：

1. 监督学习：在监督学习中，程序通过学习从标签好的数据中得出规律，然后进行预测和决策。监督学习可以进一步分为多种类型，如分类、回归、支持向量机等。
2. 无监督学习：在无监督学习中，程序通过学习从未标签的数据中得出规律，然后进行预测和决策。无监督学习可以进一步分为多种类型，如聚类、降维、主成分分析等。

医学图谱是一种用于存储和管理医学知识的数据库。医学图谱中存储了大量的医学知识，包括疾病的表现、症状、发病率、治疗方法等。医学图谱可以帮助医生更好地诊断和治疗疾病。
