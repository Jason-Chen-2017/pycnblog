# 2.核心概念与联系

## 2.1 贝叶斯网络

贝叶斯网络是一种有向无环图（DAG），其节点表示随机变量，边表示变量之间的条件依赖关系。贝叶斯网络可以用来表示一个有限的随机事件集合之间的条件依赖关系，这些事件可以被表示为随机变量。

贝叶斯网络的一个重要特点是它可以用来计算概率分布。给定一个贝叶斯网络和一个事件的条件概率，我们可以计算出这个事件在网络中的概率。这是因为贝叶斯网络可以用来表示一个条件独立性，即给定其他变量的值，任意两个变量之间的概率是独立的。

## 2.2 社交网络

社交网络是一种网络结构，其中节点表示人或组织，边表示之间的关系。社交网络可以用来表示人们之间的社交关系，如朋友、家人、同事等。社交网络还可以用来表示组织之间的关系，如公司之间的合作关系、政府机构之间的联系等。

社交网络的一个重要特点是它可以用来表示人们之间的关系。这使得社交网络成为一种强大的工具，可以用来分析人们的行为、信息传播和社会动态。
