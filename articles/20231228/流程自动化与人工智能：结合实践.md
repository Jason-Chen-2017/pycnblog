## 4.1 流程自动化的代码实例

## 4.2 人工智能的代码实例

我们将通过一个简单的文本分类任务来展示人工智能的代码实例。在这个例子中，我们需要将一段文本分类为正面或负面。

```python
import numpy as np
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC
from sklearn.pipeline import Pipeline

# 训练数据
train_data = ['我非常喜欢这个产品', '这个产品非常糟糕', '我非常满意这个服务', '这个服务非常糟糕']

# 标签
train_labels = [1, 0, 1, 0]

# 创建一个管道，包括TF-IDF向量化和SVM分类器
pipeline = Pipeline([
    ('tfidf', TfidfVectorizer()),
    ('svm', SVC())
])

# 训练模型
pipeline.fit(train_data, train_labels)

# 测试数据
test_data = ['我非常不满意这个产品', '这个产品非常棒']

# 预测标签
predicted_labels = pipeline.predict(test_data)

# 输出预测结果
for data, label in zip(test_data, predicted_labels):
    print(f'文本：{data}，预测标签：{label}')
```

在这个例子中，我们首先创建了一个训练数据集和标签。然后，我们创建了一个管道，包括TF-IDF向量化和SVM分类器。接着，我们使用管道训练模型，并使用训练好的模型对测试数据进行预测。最后，我们输出预测结果。
