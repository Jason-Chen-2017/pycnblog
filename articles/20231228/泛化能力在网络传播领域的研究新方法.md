## 4.3 个人隐私泄露的预防

我们将通过一个简单的数据脱敏示例来展示如何使用泛化能力预防个人隐私泄露。首先，我们需要收集一些敏感信息，例如姓名、电话号码等。然后，我们可以使用脱敏技术来保护这些敏感信息。

```python
import re

def anonymize(info, patterns):
    for pattern in patterns:
        result = re.sub(pattern, "", info)
        if result:
            return result
    return info

info = "姓名：张三，电话号码：13800138000"
patterns = [r"姓名：(.+?),", r"电话号码：(.+?)$"]
print(anonymize(info, patterns))  # "，13800138000"
```
