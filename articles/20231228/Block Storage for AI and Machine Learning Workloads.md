# 2.核心概念与联系

## 2.1 块存储的基本概念

块存储是一种存储技术，它将数据以固定大小的块（通常为4KB或512字节）存储在存储设备上。块存储通常用于存储文件系统、数据库和其他类型的数据。块存储可以是本地的（如硬盘驱动器），也可以是远程的（如网络附加存储，NAS）。

## 2.2 AI和机器学习工作负载的特点

AI和机器学习工作负载通常涉及大量的数据处理和存储。这些工作负载通常需要高速访问和高吞吐量的存储，以及对数据的并行处理。此外，这些工作负载还需要对数据进行持久化存储，以便在不同的计算节点之间进行分布式处理。
