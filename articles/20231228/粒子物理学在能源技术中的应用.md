```python
import numpy as np
import matplotlib.pyplot as plt

def core_reaction(N0, t, lambda_core):
    N = N0 * np.exp(-lambda_core * t)
    return N

N0 = 1000000
t = np.linspace(0, 100, 1000)
lambda_core = 0.001

N = core_reaction(N0, t, lambda_core)

plt.plot(t, N)
plt.xlabel('Time (s)')
plt.ylabel('Core Substance (#/m^3)')
plt.title('Core Reaction')
plt.show()
```

在这个代码中，我们首先导入了`numpy`和`matplotlib.pyplot`库。然后，我们定义了一个`core_reaction`函数，用于计算核反应堆中核子数量的变化。在这个函数中，我们使用了一个指数函数来描述核子数量的变化，其中`N0`是初始核子数量，`t`是时间，`lambda_core`是核反应率。

接着，我们设定了一些参数，如初始核子数量、时间范围、核反应率等。然后，我们调用了`core_reaction`函数，计算了核子数量的变化。最后，我们使用`matplotlib.pyplot`库绘制了核子数量的变化曲线。

```python
import matplotlib.pyplot as plt

    return I

plt.plot(t, I)
plt.xlabel('Time (s)')
plt.ylabel('Radiation Intensity (W/m^2)')
plt.title('Radiation Protection')
plt.show()
```

接着，我们设定了一些参数，如初始辐射强度、辐射源与观测点的距离、时间范围、辐射强度的衰减常数等。然后，我们调用了`radiation_protection`函数，计算了辐射强度的变化。最后，我们使用`matplotlib.pyplot`库绘制了辐射强度的变化曲线。

接着，我们设定了一些参数，如初始CO2浓度、容积、时间范围、CO2浓度的增加常数等。然后，我们调用了`ccs_technology`函数，计算了CO2浓度和容积的变化。最后，我们使用`matplotlib.pyplot`库绘制了CO2浓度和容积的变化曲线。
