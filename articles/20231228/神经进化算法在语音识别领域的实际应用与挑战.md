# 2.核心概念与联系

## 2.1 语音识别
语音识别是将人类语音信号转换为文本信息的过程，可以分为两个主要阶段：语音特征提取和语音模型识别。语音特征提取通过对原始语音信号进行处理，得到一组代表语音特性的特征向量。语音模型识别则是根据这些特征向量，通过各种语音模型（如Hidden Markov Model、深度神经网络等）来识别出对应的文本信息。
