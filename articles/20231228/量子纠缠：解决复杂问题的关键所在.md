# 2. 核心概念与联系

## 2.1 量子纠缠的定义与特点

量子纠缠是指两个或多个量子系统之间的一种特殊相互作用，使得它们的量子状态不再是单独存在，而是形成一个共同的量子状态。量子纠缠具有以下特点：

1. 非局部性：量子纠缠不受空间距离的影响，即使两个量子系统之间的距离非常远，也可以保持纠缠状态。
2. 超越本征值：量子纠缠的系统无法通过本征值来描述其性质，因为纠缠状态的本征值不再是单个量子系统的和，而是一个更高维的多态性空间。
3. 不确定性：量子纠缠使得量子系统之间的信息传递变得非常快速和高效，这导致了量子系统之间的相互作用的不确定性。

## 2.2 量子纠缠的历史与发展

量子纠缠的历史可以追溯到1935年，当时爱因斯坦和波尔多夫发表了一篇论文，提出了“莫斯科实验”的想法。后来，1982年，柏拉图大学的研究人员成功地实现了莫斯科实验，这是量子纠缠的实验性验证。

随着量子信息处理技术的发展，量子纠缠在量子计算、量子通信和量子测量等方面发挥了重要作用。目前，量子纠缠已经成为现代物理学和计算机科学的一个热门研究领域。
