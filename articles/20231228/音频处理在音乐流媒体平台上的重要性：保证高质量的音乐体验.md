# 2.核心概念与联系

在音频处理技术中，核心概念包括音频信号、音频压缩、音频解码、音频编码、音频处理、音频恢复、音频混音等。这些概念之间存在密切的联系，如下所示：

1. 音频信号：音频信号是人类听觉系统能够感知的信号，通常以波形表示。音频信号的主要特征包括频谱、时域特性和相位信息。音频信号的处理是音频处理技术的基础。

2. 音频压缩：音频压缩是将原始音频信号压缩为较小的文件大小，以降低存储和传输的开销。音频压缩技术主要包括损失型压缩和无损压缩。损失型压缩会损失部分音频信息，而无损压缩则能够完全保留原始音频信息。

3. 音频解码：音频解码是将压缩后的音频文件解码为原始的音频信号。音频解码技术需要结合音频压缩技术，以确保解码后的音频信号与原始音频信号一致。

4. 音频编码：音频编码是将原始音频信号编码为可存储和传输的文件。音频编码技术需要结合音频压缩技术，以确保编码后的文件大小与压缩后的文件大小一致。

5. 音频处理：音频处理是对原始音频信号进行各种操作，如滤波、增益、平衡等，以改善音频质量或实现特定的音频效果。音频处理技术可以分为数字音频处理和模拟音频处理。

6. 音频恢复：音频恢复是对损坏的音频信号进行恢复，以提高音频质量。音频恢复技术主要包括噪声除去、丢失恢复等。

7. 音频混音：音频混音是将多个音频信号混合在一起，以创建新的音频效果。音频混音技术主要包括音频合成、音频调节、音频均衡等。
