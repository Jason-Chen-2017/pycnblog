# 2.核心概念与联系

在数字经济中，核心概念包括：

1. 数字货币：数字货币是一种电子货币，利用数字技术进行交易。数字货币的主要特点是无需中央银行发行，不需要物理货币，可以在线上线下进行交易。

4. 数字合约：数字合约是指在数字经济中所使用的合约，如智能合约、数字资产合约等。数字合约可以自动执行，降低了交易成本。
5. 数字身份：数字身份是指在数字经济中所拥有的身份，如数字身份证、数字签名、数字证书等。数字身份可以确保数字经济的安全性。

4. `function set(uint x) public {`：定义一个公共函数`set`，接收一个`uint`类型的参数`x`，不需要返回值。

5. `storedData = x;`：将`x`的值赋给`storedData`。

6. `function get() public view returns (uint) {`：定义一个公共函数`get`，不需要接收参数，返回类型为`uint`。

4. 数字合约与传统合约的区别：数字合约是在数字经济中所使用的合约，如智能合约、数字资产合约等，而传统合约则是传统经济中所使用的合约，如购销合约、贷款合约等。
5. 数字身份与传统身份的区别：数字身份是在数字经济中所拥有的身份，如数字身份证、数字签名、数字证书等，而传统身份则是传统经济中所拥有的身份，如身份证、驾驶证、户口本等。
