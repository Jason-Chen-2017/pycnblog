## 3.1 数据清洗

### 3.1.1 规则引擎算法

```python
import re

def clean_data(data, rule):
    for key, value in rule.items():
        pattern = re.compile(value)
        data[key] = pattern.sub(value, data[key])
    return data

rule = {
    'email': r'(\w+)@(\w+)\.(\w+)',
    'phone': r'\(?\d{3}\)?[-.\s]?\d{3}[-.\s]?\d{4,5}'
}

data = {
    'email': 'test@example.com',
    'phone': '123-456-7890'
}

cleaned_data = clean_data(data, rule)
print(cleaned_data)
```
