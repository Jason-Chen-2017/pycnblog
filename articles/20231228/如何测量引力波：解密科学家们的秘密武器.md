                 

# 1.背景介绍

引力波是来自于震动宇宙的波动，它们是由于大型物体（如星系碰撞、黑洞形成和星体晨曦）产生的。引力波是光和其他波动不同的，因为它们不受物质吸收和散射的影响。这使得它们成为观察宇宙最靠谱的方法之一，尤其是在观察那些光无法到达的事件时。

在2015年，两个在美国的科学家团队——洛杉矶的Caltech和华盛顿的MIT——共同发现了引力波。这是一项历史性的发现，为科学家们研究宇宙的内部结构和物理现象提供了全新的视角。

测量引力波的关键在于能够检测到这些波动的极小的能量。这需要一种非常敏感的仪器，名为LIGO（Laser Interferometer Gravitational-Wave Observatory，激光干涉引力波观测室）。LIGO由美国国家科学基金会（NSF）资助，由Caltech和MIT共同运营。
