代码实例：分布式哈希表

```python
import hashlib

class DistributedHashTable:
    def __init__(self):
        self.nodes = {}

    def put(self, key, value):
        node_id = self.hash(key)
        if node_id not in self.nodes:
            self.nodes[node_id] = []
        self.nodes[node_id].append((key, value))

    def get(self, key):
        node_id = self.hash(key)
        if node_id not in self.nodes:
            return None
        for k, v in self.nodes[node_id]:
            if k == key:
                return v
        return None

    def hash(self, key):
        return hashlib.sha256(key.encode()).hexdigest()

```

1. 首先，我们定义了一个`DistributedHashTable`类，它包含一个`nodes`字典，用于存储数据。

2. 然后，我们实现了`put`方法，它用于将数据存储到分布式哈希表中。首先，我们通过`hash`方法计算数据的哈希值，并将其作为节点ID使用。如果节点ID不存在，我们创建一个新的列表并将其添加到`nodes`字典中。然后，我们将数据存储到列表中。

3. 接下来，我们实现了`get`方法，它用于从分布式哈希表中获取数据。首先，我们通过`hash`方法计算数据的哈希值，并将其作为节点ID使用。如果节点ID不存在，我们返回`None`。然后，我们遍历节点中的所有数据，并检查是否存在与给定键匹配的数据。如果存在，我们返回该数据，否则返回`None`。

4. 最后，我们实现了`hash`方法，它用于计算数据的哈希值。我们使用了SHA256算法来计算哈希值。
