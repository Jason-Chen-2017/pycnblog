### 4.1 代码实例：图像压缩

在本节中，我们将通过一个具体的代码实例来详细解释高性能计算在图像压缩中的应用。

```python
import numpy as np
import cv2
import h5py

# 读取图像

# 图像压缩
def compress_image(img, ratio):
    # 计算压缩后的图像大小
    new_size = (int(img.shape[0] / ratio), int(img.shape[1] / ratio))
    # 创建一个新的图像数组
    compressed_img = np.zeros(new_size, dtype=np.uint8)
    # 压缩图像
    for i in range(new_size[0]):
        for j in range(new_size[1]):
            # 获取原始图像的像素值
            pixel_value = img[i * ratio, j * ratio]
            # 设置压缩后的像素值
            compressed_img[i, j] = pixel_value
    return compressed_img

# 保存压缩后的图像
with h5py.File('compressed_lena.h5', 'w') as f:
    f.create_dataset('compressed_img', data=compress_image(img, 2))
```

在上述代码中，我们首先读取了一个图像，并将其转换为灰度图像。接着，我们定义了一个`compress_image`函数，该函数接受一个图像和一个压缩比例作为输入，并返回一个压缩后的图像。在该函数中，我们首先计算压缩后的图像大小，并创建一个新的图像数组。接着，我们遍历原始图像的每个像素值，并将其设置为压缩后的像素值。最后，我们将压缩后的图像保存到一个高性能存储中。
