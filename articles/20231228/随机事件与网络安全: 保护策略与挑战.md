## 4.1 密钥生成

在对称加密算法中，密钥是加密和解密过程中的关键部分。随机事件可以用于生成密钥，从而保证算法的安全性。以下是一个具体的代码实例和详细解释：

```python
import os
import hashlib

def generate_key(key_length):
    # 生成随机数
    random_number = os.urandom(key_length // 2)
    # 计算哈希值
    hash_value = hashlib.sha256(random_number).digest()
    # 组合密钥
    key = random_number + hash_value
    return key

# 生成一个128位的密钥
key = generate_key(128)
print("生成的密钥:", key)
```

在上述代码中，我们首先使用`os.urandom()`函数生成一个随机数，然后使用`hashlib.sha256()`函数计算随机数和其他参数的哈希值，最后将随机数和哈希值组合成一个完整的密钥。

## 4.2 伪随机序列生成

```python
import random

def generate_pseudo_random_sequence(sequence_length):
    # 生成伪随机序列
    pseudo_random_sequence = [random.randint(0, 255) for _ in range(sequence_length)]
    return pseudo_random_sequence

# 生成一个1000个字节的伪随机序列
pseudo_random_sequence = generate_pseudo_random_sequence(1000)
print("生成的伪随机序列:", pseudo_random_sequence)
```

在上述代码中，我们使用`random.randint()`函数生成一个范围在0到255之间的随机整数，然后将这些随机整数组合成一个伪随机序列。
