# 2.核心概念与联系

## 2.1 线性方程组
线性方程组是一种包含多个方程的数学问题，每个方程都是线性方程。线性方程的通用形式为：

$$
a_1x_1 + a_2x_2 + \cdots + a_nx_n = b
$$

其中，$a_i$ 和 $b$ 是常数，$x_i$ 是未知量。

线性方程组可以用矩阵形式表示为：

$$
\begin{bmatrix}
a_11 & a_12 & \cdots & a_1n \\
a_21 & a_22 & \cdots & a_2n \\
\vdots & \vdots & \ddots & \vdots \\
a_m1 & a_m2 & \cdots & a_mn
\end{bmatrix}
\begin{bmatrix}
x_1 \\
x_2 \\
\vdots \\
x_n
\end{bmatrix}
=
\begin{bmatrix}
b_1 \\
b_2 \\
\vdots \\
b_m
\end{bmatrix}
$$

其中，$A$ 是方程组矩阵，$X$ 是未知量向量，$B$ 是常数向量。

## 2.2 线性方程组的解
线性方程组的解是指找到一个向量 $X$，使得方程组成立。线性方程组的解可以有多个，也可以没有解，或者无限多个解。

## 2.3 线性方程组解决器
线性方程组解决器是一种软件或算法，用于计算线性方程组的解。线性方程组解决器可以根据不同的算法和数据结构实现，如直接法、迭代法和分解法等。

## 3.1 高斯消元法
高斯消元法是一种直接法，用于解决方程组 $AX = B$。其主要步骤如下：

1. 将方程组中的每个变量都表示成最高项。
2. 通过行交换（如果必要）使得方程组中的每个变量的系数的绝对值最大的行在方程组的上方。
3. 使用行减法（或加法）消去每一列中除了第一列外其他变量的系数。
4. 重复步骤3，直到第一列中的所有系数都为0。
5. 如果方程组有解，则将第一列中的常数项除以第一列中的第一个系数，得到第一列中的解。
6. 使用列减法（或加法）消去第一列中的系数，并将其他列中的常数项除以相应的系数，得到其他列中的解。

迪欧尔法是一种迭代法，用于解决方程组 $AX = B$。其主要步骤如下：

1. 选择一个初始猜测 $X_0$。

Eigen是一个C++库，提供了一系列的数据结构和算法来处理矩阵和线性代数问题。Eigen库支持高效的矩阵运算、线性方程组解和奇异值分解等功能。

## 4.2 Eigen库的使用方法
要使用Eigen库，首先需要在项目中包含Eigen库的头文件。在C++代码中，可以通过以下方式包含Eigen库：

```cpp
#include <Eigen/Dense>
```

接下来，我们可以通过以下代码创建一个线性方程组的实例，并使用Eigen库的解决方法来解决它：

```cpp
#include <iostream>
#include <Eigen/Dense>

int main() {
    // 定义方程组矩阵和常数向量
    Eigen::MatrixXd A(3, 3);
    A << 1, 2, 3,
         2, 1, 1,
         1, 2, 2;

    Eigen::VectorXd b(3);
    b << 1,
         2,
         3;

    // 使用Eigen库的解决方法解决线性方程组
    Eigen::VectorXd x = A.colPivHouseholderQR().solve(b);

    // 输出解决结果
    std::cout << "x = " << x << std::endl;

    return 0;
}
```

在上述代码中，我们首先定义了一个3x3的方程组矩阵 `A` 和一个3元素的常数向量 `b`。然后，我们使用Eigen库提供的 `colPivHouseholderQR` 方法来解决线性方程组，并将结果存储在向量 `x` 中。最后，我们输出解决结果。
