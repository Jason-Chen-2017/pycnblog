# 2.核心概念与联系

合规性法规违反行为是指企业和组织在处理数据时违反合规性法规的行为。这些行为可能包括但不限于：

1. 处理个人信息时未获得同意的情况。
2. 未按照法规对个人信息进行保密处理的情况。
3. 未按照法规对个人信息进行删除或更正的情况。

识别和处理合规性法规违反行为的策略主要包括以下几个方面：

1. 数据标记化：将数据标记为合规性法规要求的不同类别，以便于识别和处理违反行为。
2. 数据审计：对数据处理流程进行审计，以确保遵循合规性法规。
3. 违反行为识别：通过机器学习算法识别违反合规性法规的行为。
4. 违反行为处理：根据违反行为的类别和严重程度，采取相应的处理措施。

3. 将标签赋给相应的数据。

数据审计是对数据处理流程进行审计的过程。这可以通过以下步骤实现：

1. 确定需要审计的数据处理流程。
2. 根据合规性法规的要求，为数据处理流程创建审计规则。
3. 通过审计规则，对数据处理流程进行审计。
