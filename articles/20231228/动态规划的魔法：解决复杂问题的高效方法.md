                 

# 1.背景介绍

动态规划（Dynamic Programming，DP）是一种解决复杂问题的高效方法，它主要应用于求解最优化问题，通常可以将复杂问题拆分成多个相互依赖的子问题，并将解决过程分步骤进行，以达到减少冗余计算的目的。动态规划的核心思想是“分而治之”，将大问题拆分成小问题，然后将小问题的解组合成大问题的解。

动态规划的应用范围广泛，包括但不限于：最短路问题、最长子序列问题、背包问题、编辑距离问题等等。动态规划在实际应用中具有很高的价值，因为它可以帮助我们找到问题的最优解，并且算法的时间复杂度通常是可接受的。

## 1.1 动态规划的基本概念

动态规划（Dynamic Programming）：一种解决最优化问题的方法，通过将问题分解为相互依赖的子问题，并将子问题的解组合成大问题的解，从而避免重复计算。

最优子结构（Optimal Substructure）：一个问题的最优解包含其子问题的最优解。

无后效性（Overlapping Subproblems）：已解决的子问题不会再次出现。
