# 4.具体代码实例和详细解释说明

## 4.1并行计算的具体代码实例

在本节中，我们将通过一个简单的矩阵乘法示例来介绍并行计算的具体代码实例。

```
#include <stdio.h>
#include <omp.h>

int main() {
    int n = 4;
    double a[n][n], b[n][n], c[n][n];
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            a[i][j] = (i + 1) * (j + 1);
        }
    }
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            b[i][j] = (i + 1) * (j + 1) * 10;
        }
    }
    #pragma omp parallel for shared(a, b, c) private(i, j)
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            c[i][j] = 0;
            for (int k = 0; k < n; k++) {
                c[i][j] += a[i][k] * b[k][j];
            }
        }
    }
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            printf("%f ", c[i][j]);
        }
        printf("\n");
    }
    return 0;
}
```

在上述代码中，我们使用了OpenMP库来实现矩阵乘法的并行计算。通过`#pragma omp parallel for`指令，我们可以将矩阵乘法的计算分配给多个线程进行并行处理。

## 4.2并行计算的详细解释说明

在上述代码中，我们首先定义了一个4×4的矩阵a和b，并将其初始化为一些数值。然后，我们使用`#pragma omp parallel for`指令将矩阵乘法的计算分配给多个线程进行并行处理。在并行区域内，我们使用了共享变量a、b和c，并将局部变量i和j设置为私有。通过对矩阵a和b的元素进行遍历，我们可以计算矩阵c的元素。最后，我们将矩阵c的元素打印出来。
