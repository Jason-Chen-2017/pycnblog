### 4.1 代码实例

```python
from cloud_service import CloudService

def combine_cloud_service(cloud_services):
    combined_cloud_service = CloudService()
    for cloud_service in cloud_services:
        combined_cloud_service.add_service(cloud_service)
    return combined_cloud_service

cloud_services = [
    CloudService('aws', 'ec2'),
    CloudService('azure', 'vm'),
    CloudService('aliyun', 'ecs')
]

combined_cloud_service = combine_cloud_service(cloud_services)
```

在这个代码实例中，我们首先导入了 `cloud_service` 模块，然后定义了一个 `combine_cloud_service` 函数，该函数接收一个包含多个云服务的列表，并将这些云服务组合成一个新的云服务。然后我们创建了一个包含三个云服务的列表，分别是 AWS EC2、Azure VM 和 AliCloud ECS。最后，我们调用 `combine_cloud_service` 函数，将这些云服务组合成一个新的云服务，并将其赋值给 `combined_cloud_service` 变量。

这个代码实例展示了如何在多云环境中实现可组合扩展性，通过组合不同的云服务和资源，实现应用程序和数据的扩展和迁移。
