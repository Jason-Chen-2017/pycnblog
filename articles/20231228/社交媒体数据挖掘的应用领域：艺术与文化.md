# 4.具体代码实例和详细解释说明
在这里，我们将提供一个简单的Python代码实例，用于分析艺术作品的评价。这个例子使用了scikit-learn库中的K-均值算法来对艺术作品进行分类。

```python
from sklearn.cluster import KMeans
import pandas as pd

# 加载数据
data = pd.read_csv('art_data.csv')

# 提取特征
features = data['features']

# 使用K-均值算法进行分类
kmeans = KMeans(n_clusters=3)
kmeans.fit(features)

# 分类结果
labels = kmeans.predict(features)

# 打印结果
print(labels)
```
