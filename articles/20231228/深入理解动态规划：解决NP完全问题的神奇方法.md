                 

# 1.背景介绍

动态规划（Dynamic Programming, DP）是一种解决决策过程中子问题相互依赖的方法。它主要应用于求解最优化问题，如最长公共子序列、最长递增子序列、0-1背包问题等。动态规划的核心思想是将大问题拆分成小问题，然后将小问题的解组合成大问题的解。这种方法可以避免多次计算相同的子问题，从而提高算法的效率。

动态规划的应用范围广泛，但它并不能解决所有的最优化问题。特别是当问题规模较大时，动态规划的时间复杂度可能会非常高，甚至超过暴力搜索法。因此，在实际应用中，我们需要根据具体问题的特点选择合适的算法方法。
