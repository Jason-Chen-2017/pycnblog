# 2.核心概念与联系
自动驾驶技术可以分为五个层次：0、1、2、3和4。其中，自动驾驶层次0表示完全没有自动驾驶功能，即传统的人类驾驶；层次1表示有限的自动驾驶功能，例如电子稳定程度控制（ESC）和自动刹车；层次2表示有限的自动驾驶功能，例如自动巡航和自动停车；层次3表示高级自动驾驶功能，即人类驾驶员可以在需要的时候接管驾驶；层次4表示完全的自动驾驶功能，即无需人类驾驶员的干预。

机器人学是一门研究机器人如何在未知环境中行动和交互的学科。机器人学的核心概念包括感知、理解、决策和行动。在自动驾驶领域，机器人学的应用主要体现在以下几个方面：

1. 感知：自动驾驶系统需要通过摄像头、雷达、激光雷达等传感器获取环境信息，以便进行后续的决策和行动。
2. 理解：自动驾驶系统需要对获取到的环境信息进行处理，以便将其转化为机器可理解的形式。
3. 决策：自动驾驶系统需要根据环境信息和驾驶规则进行决策，以便实现安全和高效的驾驶。
4. 行动：自动驾驶系统需要根据决策执行驾驶操作，以便实现目标。
