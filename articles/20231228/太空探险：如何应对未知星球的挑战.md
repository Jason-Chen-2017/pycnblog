## 4.具体代码实例和详细解释说明
4.1 太空探险代码实例
以下是一个简单的太空探险代码实例，用于模拟火箭轨迹计算：

```python
import numpy as np

def calculate_trajectory(initial_velocity, time, gravity):
    """
    计算火箭轨迹
    """
    position = initial_velocity * time
    velocity = initial_velocity - gravity * time
    return position, velocity

initial_velocity = 10000  # 初始速度，单位：米/秒
time = 300  # 时间，单位：秒
gravity = 9.8  # 重力，单位：米/秒^2

position, velocity = calculate_trajectory(initial_velocity, time, gravity)
print(f"火箭轨迹：位置 {position} 米，速度 {velocity} 米/秒")
```

4.2 代码解释
上述代码实例主要包括以下几个部分：

1. 导入numpy库，用于数值计算。
2. 定义一个函数`calculate_trajectory`，用于计算火箭轨迹。该函数接受初始速度、时间和重力作为输入参数，并返回火箭的位置和速度。
3. 设置初始速度、时间和重力的值。
4. 调用`calculate_trajectory`函数，并打印火箭轨迹的位置和速度。
