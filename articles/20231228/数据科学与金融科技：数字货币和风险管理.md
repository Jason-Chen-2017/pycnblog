### 3.1.1 加密算法

加密算法是数字货币的核心部分，用于保护交易和存储数据的安全。例如，比特币使用SHA-256算法来生成Proof of Work，这是一种摘要算法，可以生成一个固定长度的哈希值。

SHA-256算法的基本步骤如下：

1. 将输入数据分为多个块。
2. 对每个块进行摘要计算。
3. 对摘要进行迭代处理。
4. 得到最终的哈希值。

### 3.1.2 分布式共识算法

分布式共识算法是数字货币的核心部分，用于确保网络的一致性和安全性。例如，比特币使用Proof of Work算法来实现分布式共识。

Proof of Work算法的基本步骤如下：

1. 矿工竞争解决一个难度函数。
2. 解决难度函数的矿工获得新的区块。
3. 新的区块加入区块链。
4. 网络达成共识。

        hash = hashlib.sha256(data.encode('utf-8') + str(nonce).encode('utf-8')).hexdigest()
        if hash[:4] == '0000':
            break
        nonce += 1
    return nonce
