# 4.具体代码实例和详细解释说明

```python
import matplotlib.pyplot as plt

# 数据
categories = ['A', 'B', 'C', 'D', 'E']
values = [10, 20, 30, 40, 50]

# 创建条形图
plt.bar(categories, values)

# 设置图表标题和坐标轴标签
plt.title('数据可视化-条形图')
plt.xlabel('分类变量')
plt.ylabel('连续变量')

# 显示图表
plt.show()
```

这个代码实例首先导入了Matplotlib库，然后定义了数据，包括分类变量和连续变量。接着，使用了`plt.bar()`函数创建了一个条形图，并设置了图表标题和坐标轴标签。最后，使用了`plt.show()`函数显示了图表。

通过这个代码实例，我们可以看到数据可视化技术的实际应用，并了解如何使用Python和Matplotlib库实现数据可视化。
