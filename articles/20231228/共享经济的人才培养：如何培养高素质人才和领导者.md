具体代码实例如下：

1.资源分配算法的代码实例：

```python
def resource_allocation(resources, demands):
    allocation = {}
    for resource in resources:
        if resource in demands:
            allocation[resource] = min(demands[resource], resources[resource])
    return allocation
```

2.平台化算法的代码实例：

```python
def platformization(supply, demand):
    platform = {}
    for item in supply:
        if item in demand:
            platform[item] = min(supply[item], demand[item])
    return platform
```

3.创新算法的代码实例：

```python
def innovation(innovations, profits):
    innovation_result = {}
    for innovation in innovations:
        if innovation in profits:
            innovation_result[innovation] = max(innovations[innovation], profits[innovation])
    return innovation_result
```
