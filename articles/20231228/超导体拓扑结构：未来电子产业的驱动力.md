
### 1.4.2 导入库和定义量子电路

```python
import qiskit as qs
from qiskit import QuantumCircuit

# 定义量子电路
qc = QuantumCircuit(2, 2)
```

### 1.4.3 初始化量子比特和设置初始状态

```python
# 初始化量子比特
qc.initialize([1, 0], range(2))
```

### 1.4.4 应用量子门

```python
# 应用量子门
qc.h(0)
```

### 1.4.5 实现量子纠缠

```python
# 实现量子纠缠
qc.cx(0, 1)
```

### 1.4.6 度量量子比特

```python
# 度量量子比特
qc.measure([0, 1], [0, 1])
```

### 1.4.7 运行量子电路

```python
# 运行量子电路
simulator = qs.Aer.get_backend('qasm_simulator')
job = qs.execute(qc, backend=simulator, shots=1000)
result = job.result()
```

### 1.4.8 输出结果

