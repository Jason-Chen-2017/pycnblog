## 4.1 积分

```python

import matplotlib.pyplot as plt

def f(x):
    return x**2

a = 0
b = 2
dx = 0.1

for x in np.arange(a, b, dx):
    area += f(x) * dx

print("Area:", area)

plt.plot(np.arange(a, b, dx), [f(x) for x in np.arange(a, b, dx)])

```

```python
    return y - x**2

    return -2*x

y = np.zeros(len(x))

for i in range(1, len(x)):
    y[i] = y[i-1] + df(x[i-1], y[i-1], y[i] - y[i-1]) * (x[i] - x[i-1])

```
