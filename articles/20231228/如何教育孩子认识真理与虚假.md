# 2.核心概念与联系

在教育孩子认识真理与虚假时，我们需要关注以下几个核心概念：

1. **信息质量**：信息的准确性、可靠性、完整性和时效性等方面。
2. **信息判断**：人们如何对信息进行判断和分辨，以及如何识别虚假信息和谎言。
3. **信息传播**：信息如何传播和传播的速度和范围，以及如何控制信息传播。
4. **信息应用**：信息如何应用于日常生活和决策，以及如何避免信息导致的误导和损失。

这些概念之间存在着密切的联系，信息质量对信息判断的准确性有很大影响，信息判断对信息应用的正确性也很重要。同时，信息传播和信息应用也是相互影响的。因此，在教育孩子认识真理与虚假时，需要全面地考虑这些概念和联系。

    if n <= 1:
