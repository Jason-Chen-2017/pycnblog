我们可以使用Python的NLTK库来构建知识图谱。具体代码如下：

```python
import nltk
from nltk.entity.wordnet import WordNetLemmatizer

# 初始化WordNetLemmatizer
lemmatizer = WordNetLemmatizer()

# 构建实体关系
entities = [
    ("人脸识别", "技术"),
    ("知识图谱", "技术"),
    ("人脸识别", "应用"),
    ("知识图谱", "应用"),
]

# 将实体关系存储到知识图谱中
for entity, relation in entities:
    nltk.corpus.wordnet.synsets(lemmatizer.lemmatize(entity))
    nltk.corpus.wordnet.synsets(lemmatizer.lemmatize(relation))
```
