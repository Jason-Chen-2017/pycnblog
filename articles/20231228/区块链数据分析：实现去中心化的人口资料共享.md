# 2.核心概念与联系
区块链是一种去中心化的分布式数据存储和传输技术，它通过将数据存储在多个节点上，并通过加密算法来保证数据的完整性和安全性。区块链的核心概念包括：

- 区块：区块链是由一系列区块组成的，每个区块包含一定数量的交易数据。
- 链：区块之间通过哈希链接在一起，形成一个有序的链。
- 共识机制：区块链网络中的节点通过共识机制来达成一致，确保数据的完整性和一致性。

人口资料共享通过区块链技术实现的过程中，需要考虑以下几个方面：

- 数据存储：将人口资料存储在区块链上，以实现去中心化的数据存储和传输。
- 数据安全：通过加密算法来保证数据的安全性，防止数据泄露和篡改。
- 数据访问：实现人口资料的安全访问和共享，以满足不同的应用需求。

```python
from web3 import Web3
from sha3 import keccak256
import json

# 连接到Ethereum网络
w3 = Web3(Web3.HTTPProvider("http://localhost:8545"))

# 创建一个新的账户
account = w3.eth.accounts[0]

# 创建一个新的合约
contract = w3.eth.contract(address="0x751f5EcCb848996446525297884584e8906a44e9", abi=[...])

# 将人口资料存储在区块链上
def store_population_data(population_data):
    # 将人口资料编码为JSON格式
    encoded_data = json.dumps(population_data).encode("utf-8")
    
    # 计算哈希值
    hash = keccak256(encoded_data)
    
    # 创建一个新的交易
    tx = contract.functions.storeData(encoded_data).buildTransaction({
        "gas": 1000000,
        "gasPrice": w3.eth.gasPrice,
        "from": account.address,
    })
    
    # 发送交易
    signed_tx = w3.eth.account.signTransaction(tx, account.privateKey)
    w3.eth.sendRawTransaction(signed_tx.rawTransaction)

# 存储人口资料示例
population_data = {
    "name": "张三",
    "age": 25,
    "gender": "男",
}
store_population_data(population_data)
```
