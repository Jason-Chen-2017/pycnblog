                 

# 1.背景介绍

可再生能源（Renewable Energy）是指来自新鲜、可再生、无需消耗的自然资源的能源。这些能源包括太阳能、风能、水能、地热能、生物质能、地球能等。在全球范围内，可再生能源已经成为应对气候变化和保护环境的关键手段之一。

在国际领域，各国政府和国际组织已经采取了许多措施，以促进可再生能源的发展和应用。例如，世界气候组织（UNFCCC）的Paris协议目标是限制全球温度上升到2摄氏度以下，以促进可再生能源的使用。此外，国际金融机构如世界银行和亚洲发展银行也开始投资于可再生能源项目。

在国内，中国政府也采取了多项政策措施，以支持可再生能源的发展。例如，中国发展研究中心（中国政府机构）发布的《中国可再生能源发展规划（2016-2030年）》，明确了到2020年可再生能源总能耗占比达到15%的目标。此外，国家发展和改革委员会还发布了《可再生能源发展规划（2016-2030年）》，明确了到2020年可再生能源总能耗占比达到15%的目标。
