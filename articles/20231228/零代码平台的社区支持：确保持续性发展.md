## 4.1 拖放和点击操作的代码实例

在这个例子中，我们将实现一个简单的拖放和点击操作的代码实例。

```python
from tkinter import Tk, Canvas, Button

class DragDropApp:
    def __init__(self, master):
        self.master = master
        self.canvas = Canvas(master, width=400, height=300)
        self.canvas.pack()
        self.button = Button(master, text="Drag me")
        self.button.pack()
        self.button_id = self.canvas.create_window(200, 200, window=self.button)

    def on_drag_start(self, event):
        self.drag_item = event.widget
        self.drag_x = event.x
        self.drag_y = event.y

    def on_drag_motion(self, event):
        dx = event.x - self.drag_x
        dy = event.y - self.drag_y
        self.canvas.move(self.drag_item, dx, dy)

    def on_drag_end(self, event):
        self.drag_item = None

    def on_click(self, event):
        print("You clicked at", event.x, event.y)

    def run(self):
        self.master.bind("<ButtonPress-1>", self.on_drag_start)
        self.master.bind("<B1-Motion>", self.on_drag_motion)
        self.master.bind("<ButtonRelease-1>", self.on_drag_end)
        self.canvas.bind("<Button-1>", self.on_click)
        self.master.mainloop()

root = Tk()
app = DragDropApp(root)
app.run()
```

在这个例子中，我们创建了一个简单的拖放和点击操作的应用程序。用户可以通过拖动按钮来移动它，同时可以通过点击按钮来显示其坐标。

## 4.2 数据存储和管理的代码实例

在这个例子中，我们将实现一个简单的数据存储和管理的代码实例。

```python
import json

class DataManager:
    def __init__(self):
        self.data = {}

    def save_data(self, key, value):
        self.data[key] = value
        with open("data.json", "w") as f:
            json.dump(self.data, f)

    def load_data(self):
        with open("data.json", "r") as f:
            self.data = json.load(f)

    def delete_data(self, key):
        if key in self.data:
            del self.data[key]
            with open("data.json", "w") as f:
                json.dump(self.data, f)

data_manager = DataManager()
data_manager.save_data("name", "John")
data_manager.load_data()
data_manager.delete_data("name")
```

在这个例子中，我们创建了一个简单的数据存储和管理的应用程序。用户可以通过调用`save_data`方法来保存数据，通过调用`load_data`方法来加载数据，通过调用`delete_data`方法来删除数据。
