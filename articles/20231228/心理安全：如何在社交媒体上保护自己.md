### 4.1 垃圾邮件过滤算法实例

我们可以使用Python的scikit-learn库来实现基于机器学习的垃圾邮件过滤算法。首先，我们需要准备一个数据集，包括一些标签为垃圾邮件的邮件和标签为正常邮件的邮件。然后，我们可以使用scikit-learn库中的`CountVectorizer`类来将邮件转换为词袋模型，并使用`TfidfVectorizer`类来将邮件转换为TF-IDF模型。最后，我们可以使用`SVC`类来训练一个支持向量机分类器，并使用`predict`方法来预测邮件是否为垃圾邮件。

```python
from sklearn.feature_extraction.text import CountVectorizer, TfidfVectorizer
from sklearn.svm import SVC
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 准备数据集
spam_data = [
    ("这是一个垃圾邮件", "spam"),
    ("这是一个正常邮件", "normal"),
    # ...
]

# 将邮件转换为词袋模型
vectorizer = CountVectorizer()
X = vectorizer.fit_transform([email[0] for email in spam_data])

# 将邮件转换为TF-IDF模型
tfidf_vectorizer = TfidfVectorizer()
Y = tfidf_vectorizer.fit_transform([email[1] for email in spam_data])

# 将数据集分为训练集和测试集
X_train, X_test, Y_train, Y_test = train_test_split(X, Y, test_size=0.2, random_state=42)

# 训练支持向量机分类器
clf = SVC(kernel='linear', C=1)
clf.fit(X_train, Y_train)

# 预测邮件是否为垃圾邮件
predictions = clf.predict(X_test)

# 计算准确率
accuracy = accuracy_score(Y_test, predictions)
print("Accuracy: {:.2f}".format(accuracy * 100))
```
