# 2.核心概念与联系
## 2.1 泰勒展开
泰勒展开是一种用于逼近函数的方法，它可以将一个函数在某个点附近展开为一个包含多项的表达式。泰勒展开的基本思想是利用函数的导数信息来逼近函数。给定一个函数f(x)和一个点a，泰勒展开可以写为：

$$
f(x) \approx f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \frac{f'''(a)}{3!}(x-a)^3 + \cdots + \frac{f^{(n)}(a)}{n!}(x-a)^n
$$

其中，f'(a)、f''(a)、f'''(a)、...表示函数f(x)在点a的第一、第二、第三、...阶导数，n是泰勒展开的阶数。泰勒展开可以用来逼近函数在某个点的值以及函数在该点附近的弧度。

def taylor_expand(f, a, x, n=2):
    f_prime = lambda x: f(x + a) - f(a)
    f_double_prime = lambda x: f_prime(x + a) - f_prime(a)
    taylor_expand = f(a) + sum([f_double_prime(i) / (i * (i - 1)) * (x - a) ** i for i in range(1, n + 1)])
    return taylor_expand

n = 2
