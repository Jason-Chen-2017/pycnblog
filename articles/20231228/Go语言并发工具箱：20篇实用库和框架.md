## 4.1 Goroutine的使用示例
```go
package main

import (
	"fmt"
	"time"
)

func main() {
	// 创建一个goroutine
	go func() {
		fmt.Println("Hello, Goroutine!")
	}()

	// 主goroutine等待
	var input string
	fmt.Scanln(&input)
}
```

## 4.2 Channel的使用示例
```go
package main

import (
	"fmt"
	"time"
)

func main() {
	// 创建一个channel
	ch := make(chan string)

	// 在一个goroutine中发送数据
	go func() {
		ch <- "Hello, Channel!"
	}()

	// 在主goroutine中接收数据
	data := <-ch
	fmt.Println(data)

	// 主goroutine等待
	var input string
	fmt.Scanln(&input)
}
```
在上面的代码示例中，我们创建了一个Channel，它可以用来实现并发编程中的同步和通信。在一个Goroutine中，我们发送了一个字符串“Hello, Channel!”到Channel中。在主Goroutine中，我们接收了这个字符串并打印了它。

## 4.3 Context的使用示例

```go
	"context"
	"fmt"
	"time"
)

	// 在一个goroutine中执行一个任务

		}

	// 在主goroutine中取消任务
	ctx.Done()

	// 主goroutine等待

	fmt.Scanln(&input)

```
在上面的代码示例中，我们创建了一个Context，它可以用来实现并发编程中的错误处理和取消。在一个Goroutine中，我们使用select语句来监听Context的取消事件。在主Goroutine中，我们取消了Context，这会导致Goroutine中的任务被取消。
