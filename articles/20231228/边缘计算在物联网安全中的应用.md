# 2.核心概念与联系
## 2.1 边缘计算简介
边缘计算（Edge Computing）是一种新兴的计算技术，它将数据处理和应用程序从中心服务器移动到边缘设备（如路由器、交换机等），从而实现更快的响应时间和更高的效率。边缘计算可以应用于各种场景，如智能城市、智能制造、自动驾驶等。

## 2.2 物联网安全
物联网安全是指在物联网环境中保护设备、数据和系统免受未经授权的访问和攻击的过程。物联网安全涉及到设备安全、数据安全、通信安全等方面。

## 2.3 边缘计算在物联网安全中的应用
边缘计算可以在物联网安全中发挥着重要作用，主要表现在以下几个方面：

1. 减少通信量，减轻中心服务器的负载，从而提高系统安全性。
2. 将敏感数据处理在边缘设备上，降低数据泄露的风险。
3. 通过边缘计算实现实时监控和快速响应，提高物联网安全的有效性。

## 6.1 边缘计算与云计算的区别
边缘计算和云计算的主要区别在于数据处理和存储的位置。边缘计算将数据处理和应用程序移动到边缘设备，而云计算将数据处理和存储移动到中心服务器。
