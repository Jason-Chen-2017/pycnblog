# 2.核心概念与联系

在时间序列分析中，我们需要了解以下几个核心概念：

1. **时间序列数据**：时间序列数据是按照时间顺序排列的数据点。它们通常用于表示某个变量在不同时间点的值。

2. **趋势**：时间序列数据的趋势是指数据点在长期观察下的变化规律。趋势可以是上升、下降或者平稳。

3. **季节性**：季节性是指时间序列数据在特定时间间隔内出现周期性变化的现象。例如，商业销售数据通常会出现季节性变化，因为人们在某个季节购买商品的频率会有所不同。

4. **周期性**：周期性是指时间序列数据在较长时间间隔内出现循环变化的现象。周期性通常比季节性长。

5. **分析方法**：时间序列分析方法包括观察、描述性分析、模型构建和预测等。这些方法可以帮助我们更好地理解时间序列数据的特征和规律。

6. **可视化**：可视化是将数据以图形方式呈现给用户的方法。可视化可以帮助用户更直观地理解数据。

   答案：模型性能可以通过误差、R²值、MAPE等指标来评估。误差是用于衡量预测值与实际值之间差异的指标，例如均方误差、均方根误差等。R²值是用于衡量模型预测能力的指标，范围在0到1之间。MAPE是用于衡量预测误差的相对指标，范围在0到100%之间。
