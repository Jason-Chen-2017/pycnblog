# 2. 核心概念与联系
协方差矩阵是一种用于描述多个变量之间关系的统计学方法。它是一种矩阵，包含了每对变量之间的协方差。协方差矩阵可以帮助我们理解变量之间的线性关系，并用于进行数据预处理、特征选择和机器学习模型训练等多种应用。

协方差矩阵的核心概念包括：

1. 协方差：协方差是一种度量两个随机变量线性关系的量。它表示两个变量的变化趋势是否相同。协方差的正值表示两个变量的变化趋势相同，负值表示两个变量的变化趋势相反。协方差的值为0表示两个变量之间没有线性关系。

2. 方差：方差是一种度量随机变量离散程度的量。它表示随机变量在一个给定分布下的平均值与观测值之间的差异。方差的正值表示随机变量离散程度较大，负值表示随机变量离散程度较小。方差的值为0表示随机变量的所有观测值与平均值相同。

3. 协方差矩阵：协方差矩阵是一种矩阵，包含了每对变量之间的协方差。协方差矩阵可以帮助我们理解变量之间的线性关系，并用于进行数据预处理、特征选择和机器学习模型训练等多种应用。
