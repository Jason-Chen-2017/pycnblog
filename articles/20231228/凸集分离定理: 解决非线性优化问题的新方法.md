# 2.核心概念与联系
凸集分离定理是一种新的优化方法，它可以将一个复杂的非线性优化问题分解为多个简单的线性优化问题。这种方法的核心概念是凸集（Convex Hull）和分离（Separation）。

凸集是指一个集合中的每个点都在其自身的凸包内。例如，在二维平面上，一个圆是一个凸集，而一个椭圆则不是。凸集具有很多优秀的数学性质，如凸集的任何两个点都可以连接成一条凸线，凸集的任何子集都是凸集等。

分离是指将一个集合分为两个互不相交的部分。在凸集分离定理中，我们需要将一个凸集分离成两个不相交的子凸集。这种分离方法可以用来解决非线性优化问题，因为它可以将一个复杂的问题分解为多个简单的问题。
