## 3.1 整数因式分解

整数因式分解是数论中的一个基本问题，即将一个整数拆分成其不同的因子的乘积。通常，我们可以使用初等变换来解决这个问题。

算法原理：

1. 从给定整数中找到一个最小的质数因子。
2. 将这个质数因子除去，然后重复第一步，直到剩下的整数为1。

具体操作步骤：

1. 从给定整数中找到一个最小的质数因子。
2. 将这个质数因子除去，然后重复第一步，直到剩下的整数为1。

数学模型公式：

$$
n = p_1^{e_1} \times p_2^{e_2} \times \cdots \times p_k^{e_k}
$$

其中，$p_1, p_2, \cdots, p_k$ 是质数，$e_1, e_2, \cdots, e_k$ 是大于0的整数。

## 3.2 线性同余方程

线性同余方程是数论中的一个基本问题，即在模$m$下，找到满足方程$ax \equiv b \pmod{m}$的整数解。通常，我们可以使用初等变换来解决这个问题。

算法原理：

1. 将方程两边除以$gcd(a, m)$。
2. 将得到的方程的左侧替换为$a'x$，右侧替换为$b' = b \bmod{m}$。
3. 如果$a'$与$m$互质，则可以直接求解$x$。否则，可以通过扩展欧几里得算法求解$x$。

具体操作步骤：

1. 计算$gcd(a, m)$。
2. 将方程两边除以$gcd(a, m)$。
3. 将得到的方程的左侧替换为$a'x$，右侧替换为$b' = b \bmod{m}$。
4. 如果$a'$与$m$互质，则可以直接求解$x$。否则，可以通过扩展欧几里得算法求解$x$。

数学模型公式：

$$
ax \equiv b \pmod{m}
$$

## 3.3 欧几里得算法

欧几里得算法是数论中的一个基本算法，用于计算两个整数的最大公约数。通常，我们可以使用初等变换来优化欧几里得算法。

算法原理：

1. 使用扩展中国余数定理（EGBD）求解两个整数的最小公倍数。
2. 将最小公倍数除以两个整数得到的最大公约数。

具体操作步骤：

1. 使用扩展中国余数定理（EGBD）求解两个整数的最小公倍数。
2. 将最小公倍数除以两个整数得到的最大公约数。

数学模型公式：

$$
\gcd(a, b) = \max_{i \in [1, n]} \{x_i\}
$$

其中，$x_i$ 是满足方程$ax_i + by_i = \text{lcm}(a, b)$的整数解。

# 4.具体代码实例和详细解释说明

在这里，我们将给出一个Python实现整数因式分解的代码实例，并详细解释其工作原理。

```python
def prime_factors(n):
    factors = []
    i = 2
    while i * i <= n:
        if n % i:
            i += 1
        else:
            n //= i
            factors.append(i)
    if n > 1:
        factors.append(n)
    return factors

n = 100
factors = prime_factors(n)
print(factors)
```

这个代码实现了一个名为`prime_factors`的函数，用于求解给定整数的因式分解。函数的核心逻辑如下：

1. 初始化一个空列表`factors`，用于存储因式分解结果。
2. 初始化一个变量`i`，用于遍历整数`n`的所有可能因子。
3. 使用一个`while`循环，直到`i * i > n`。在每一次迭代中，检查`n`是否能被`i`整除。如果能被整除，则将`i`作为一个因子添加到`factors`列表中，并将`n`除以`i`。如果不能被整除，则将`i`增加1。
4. 如果`n`大于1，则将`n`作为最后一个因子添加到`factors`列表中。
5. 最后，返回`factors`列表。

在这个实例中，我们调用了`prime_factors`函数，将整数`n`设置为100，并打印了因式分解结果。输出结果为`[2, 2, 5, 5]`，表示100的因式分解为2^2 * 5^2。
