# 2.核心概念与联系
## 2.1 量子光学基础
量子光学是一门研究量子系统与光的相互作用的科学。它的基本概念包括：

- 光子：光子是光的基本单位，它是一个能量量子。光子的能量可以表示为 $E = h\nu$，其中 $h$ 是平行四元体，$\nu$ 是光频率。
- 波函数：波函数是描述量子系统状态的函数，它可以通过概率密度函数来描述。波函数可以表示为 $\psi(\textbf{r},t)$，其中 $\textbf{r}$ 是位置向量，$t$ 是时间。
- 量子态：量子态是一个系统在量子力学中的描述方式，它可以是纯量子态或混合量子态。纯量子态可以表示为 $|\psi\rangle$，混合量子态可以表示为 $\rho$。

## 2.2 光学镜头基础

- 光学元件：光学元件是用于调节光线方向和大小的设备，常见的光学元件有镜头、 objectives 和 eyepieces。
- 感光元件：感光元件是用于将光线转换为电信号的设备，常见的感光元件有CCD和CMOS。
