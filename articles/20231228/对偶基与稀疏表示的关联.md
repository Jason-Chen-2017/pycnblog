## 3.1对偶基的定义与性质

对偶基是一种线性代数概念，它可以用来描述一个向量或矩阵的子空间。对偶基的定义如下：

对于一个给定的向量集合V，如果存在一个线性独立的向量集合W，使得V的每个向量都可以唯一地表示为W中某个向量的线性组合，那么W就是V的对偶基。

对偶基的性质如下：

1.对偶基中的向量是线性独立的。

2.对偶基中的向量可以唯一地表示给定向量集合中的每个向量。

3.对偶基中的向量的个数等于给定向量集合中的维数。

## 3.2对偶基在稀疏表示中的应用

在稀疏表示中，对偶基被用于构建稀疏矩阵的基础向量，从而实现稀疏表示的效果。具体的操作步骤如下：

1.首先，将稀疏矩阵的非零元素提取出来，构成一个新的向量集合V。

2.然后，通过Gram-Schmidt正交化算法，将向量集合V转换为线性独立的向量集合W，即W是V的对偶基。

3.最后，将对偶基W中的向量作为稀疏矩阵的基础向量，进行稀疏矩阵的存储和计算。

```python
import numpy as np

# 定义一个稀疏矩阵
A = np.array([[0, 0, 3, 0],
              [0, 0, 0, 4],
              [0, 5, 0, 0],
              [6, 0, 0, 0]])

# 提取非零元素构成向量集合V
V = A.flatten()

# 通过Gram-Schmidt正交化算法，将向量集合V转换为线性独立的向量集合W
W = np.linalg.qr(V.reshape(-1, 1))[1]

# 将对偶基W中的向量作为稀疏矩阵的基础向量，进行稀疏矩阵的存储和计算
H = np.hstack([W, np.eye(W.shape[1])])

# 输出稀疏矩阵A和对偶基矩阵H
print("稀疏矩阵A:\n", A)
print("对偶基矩阵H:\n", H)
```
