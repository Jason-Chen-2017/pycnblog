# 2. 核心概念与联系

## 2.1 贝塔分布的定义

贝塔分布的概率密度函数（PDF）定义为：

$$
f(x; \alpha_1, \alpha_2) = \frac{\Gamma(\alpha_1 + \alpha_2)}{\Gamma(\alpha_1) \Gamma(\alpha_2)} x^{\alpha_1 - 1} (1 - x)^{\alpha_2 - 1}
$$

其中，x ∈ [0, 1]，α1 和 α2 是贝塔分布的自由度参数。Γ函数是一个连续的增长函数，满足 Γ(n) = (n-1)! ，n 是正整数。

## 2.2 贝塔分布的性质

1. 贝塔分布是一个连续的概率分布。
2. 贝塔分布的期望（ Expectation）为：

$$
E(X) = \frac{\alpha_1 + \alpha_2 - 2}{\alpha_1 + \alpha_2 - 1}
$$

3. 贝塔分布的方差（Variance）为：

$$
Var(X) = \frac{2 \alpha_1 \alpha_2}{(\alpha_1 + \alpha_2 - 1)(\alpha_1 + \alpha_2 + 1)}
$$

4. 贝塔分布的模型参数可以通过最大似然估计（MLE）方法进行估计。

## 3.1 贝塔分布的参数估计

在实际应用中，我们需要根据数据来估计贝塔分布的参数 α1 和 α2。我们可以使用最大似然估计（MLE）方法来进行参数估计。

假设我们有一组观测数据 x1, x2, ..., xn，其中 xi ∈ [0, 1]。我们可以根据数据来估计贝塔分布的参数 α1 和 α2 如下：

1. 计算数据中的成功次数 S 和失败次数 F：

$$
S = \sum_{i=1}^n I(x_i) \\
F = \sum_{i=1}^n I(1 - x_i)
$$

其中，I(x) 是指示函数，如果 x 为真则返回 1，否则返回 0。

2. 根据成功次数 S 和失败次数 F来估计贝塔分布的参数 α1 和 α2：

$$
\hat{\alpha_1} = S + 1 \\
\hat{\alpha_2} = F + 1
$$

## 3.2 贝塔分布的概率累积函数（CDF）

贝塔分布的概率累积函数（CDF）定义为：

$$
F(x; \alpha_1, \alpha_2) = \frac{\Gamma(\alpha_1 + \alpha_2)}{\Gamma(\alpha_1) \Gamma(\alpha_2)} \int_0^x t^{\alpha_1 - 1} (1 - t)^{\alpha_2 - 1} dt
$$

通过计算上述积分，我们可以得到贝塔分布的CDF。

# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来演示如何使用贝塔分布进行概率分布建模。

```python
import numpy as np
from scipy.stats import beta

# 生成一组随机数据
np.random.seed(42)
n = 1000
x = np.random.beta(5, 5, size=n)

# 计算贝塔分布的参数
alpha1, alpha2 = beta.fit(x, quantile=[0.5, 0.5])

# 绘制贝塔分布
x_plot = np.linspace(0, 1, 100)
pdf = beta.pdf(x_plot, alpha1, alpha2)
plt.plot(x_plot, pdf)
plt.show()
```

在上述代码中，我们首先生成了一组随机数据 x，并使用 scipy 库中的 `beta.fit` 函数来估计贝塔分布的参数 α1 和 α2。然后，我们绘制了贝塔分布的概率密度函数（PDF）。
