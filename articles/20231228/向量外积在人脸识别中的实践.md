# 2.核心概念与联系

## 2.1 向量外积简介

向量外积（Vector Product），也称为叉积（Cross Product），是在三维空间中两个向量所产生的向量。它表示两个向量在空间中的旋转角度和方向。向量外积的结果是一个垂直于两个原向量平面的向量。

在数学上，向量外积的定义如下：

$$
\mathbf{a} \times \mathbf{b} = \begin{vmatrix}
\mathbf{i} & \mathbf{j} & \mathbf{k} \\
a_1 & a_2 & a_3 \\
b_1 & b_2 & b_3
\end{vmatrix}
$$

其中，$\mathbf{a} = (a_1, a_2, a_3)$ 和 $\mathbf{b} = (b_1, b_2, b_3)$ 是两个三维向量，$\mathbf{i}$, $\mathbf{j}$, $\mathbf{k}$ 是单位向量，$\begin{vmatrix} \end{vmatrix}$ 表示行列式。

## 2.2 向量外积在人脸识别中的应用

在人脸识别中，向量外积可以用于提取人脸特征，从而提高识别准确率。具体来说，我们可以将人脸图像表示为三维向量，然后计算其之间的向量外积，从而得到一个新的特征向量。这个新的特征向量可以用于训练深度学习模型，以实现更高的识别准确率。

import cv2
import numpy as np

def preprocess(image):
    # 裁剪人脸区域
    face_cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    faces = face_cascade.detectMultiScale(gray, scaleFactor=1.1, minNeighbors=5, minSize=(30, 30))
    for (x, y, w, h) in faces:
        face = gray[y:y+h, x:x+w]
        break
    # 调整大小
    face = cv2.resize(face, (100, 100))
    # 归一化
    face = face / 255.0
    return face

def vector_product(a, b):
    a = np.array(a).reshape(3, 1)
    b = np.array(b).reshape(3, 1)
    return np.cross(a, b)

def extract_features(image):
    image = preprocess(image)
    features = []
    for y in range(image.shape[0]):
        for x in range(image.shape[1]):
            pixel = image[y, x]
            gradient_x = np.array([1, 0])
            gradient_y = np.array([0, 1])
            gradient_x_pixel = np.dot(gradient_x, pixel)
            gradient_y_pixel = np.dot(gradient_y, pixel)
            feature = np.array([pixel[0], gradient_x_pixel, gradient_y_pixel])
            features.append(feature)
    return np.array(features)
