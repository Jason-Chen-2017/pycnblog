## 2.核心概念与联系

### 2.1 杰卡德距离

杰卡德距离（Jaccard Distance）是一种用于度量两个集合之间的相似性的度量标准。它通过计算两个集合的交集和并集的大小来衡量它们之间的差异。公式如下：

$$
Jaccard(A, B) = \frac{|A \cap B|}{|A \cup B|}
$$

其中，$A$ 和 $B$ 是两个集合，$|A \cap B|$ 表示 $A$ 和 $B$ 的交集的大小，$|A \cup B|$ 表示 $A$ 和 $B$ 的并集的大小。

### 2.2 区块链社区

区块链社区是指一组参与区块链网络的节点和参与者组成的社会系统。在区块链网络中，每个节点都会维护一份完整的区块链数据，并与其他节点进行同步。同时，参与者可以在社区中发起交易、智能合约执行等操作，以实现去中心化的数据共享和交易。

    return intersection / union

def main():
    # 示例节点数据

    similarity = jaccard_similarity(node1['contract'], node2['contract'])
    print(f'Contract similarity: {similarity}')

if __name__ == '__main__':
    main()
