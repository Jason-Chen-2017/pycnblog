# 2.核心概念与联系

## 2.1 量子比特和量子错误
量子比特（qubit）是量子计算机中的基本单位，它可以表示为0、1或两者的叠加状态。由于量子比特的特性，它们在处理信息时比经典比特更容易受到干扰和错误的影响。量子错误是指在量子计算过程中由于环境干扰、操作误差等原因导致量子比特状态的变化。

## 2.2 量子错误纠正
量子错误纠正技术的目标是在量子计算机中检测和纠正量子错误，以提高计算结果的准确性。量子错误纠正可以分为两类：一是基于测量的量子错误纠正，二是基于非测量的量子错误纠正。基于测量的量子错误纠正通常涉及将量子比特测量为经典状态，然后根据测量结果进行纠正。基于非测量的量子错误纠正则是在不进行测量的情况下对量子状态进行纠正。
