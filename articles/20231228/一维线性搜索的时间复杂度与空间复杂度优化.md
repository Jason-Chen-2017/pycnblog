1. 从数组的第一个元素开始比较。
2. 如果当前元素与查找关键字相等，则返回当前元素的下标。

## 4.具体代码实例和详细解释说明

以下是一个一维线性搜索的具体代码实例：

```python
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1
```

在这个代码实例中，我们定义了一个名为linear_search的函数，它接受一个数组arr和一个目标元素target作为参数。函数通过遍历数组中的每个元素，并比较当前元素与目标元素之间的关系。如果当前元素与目标元素相等，则返回当前元素的下标。如果遍历完整个数组仍未找到目标元素，则返回-1。
