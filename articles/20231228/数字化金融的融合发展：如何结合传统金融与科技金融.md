## 4.1 线性回归模型实例

以下是一个简单的线性回归模型实例：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据集
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 6, 8, 10])

# 模型训练
model = LinearRegression()
model.fit(X, y)

# 预测
X_new = np.array([[6]])
y_predict = model.predict(X_new)
print(y_predict)
```

在这个实例中，我们使用了sklearn库中的LinearRegression模型，对线性回归模型进行了训练和预测。

## 4.2 逻辑回归模型实例

## 4.5 神经网络模型实例

以下是一个简单的神经网络模型实例：

```python
import numpy as np
from sklearn.neural_network import MLPClassifier

# 数据集
X = np.array([[1, 0], [2, 0], [3, 0], [4, 1], [5, 1]])
y = np.array([0, 0, 0, 1, 1])

y_predict = model.predict(X_new)
print(y_predict)
```

在这个实例中，我们使用了sklearn库中的MLPClassifier模型，对神经网络模型进行了训练和预测。
