```python
import numpy as np
import matplotlib.pyplot as plt

# 定义无人机轨迹数据
x = np.linspace(0, 10, 100)
y = np.sin(x)

# 定义轨迹跟踪算法
def track(x, y):
    # 使用移动平均算法对轨迹信息进行滤波处理
    x_filtered = np.convolve(x, np.ones(3, dtype='int') / 3, mode='valid')
    y_filtered = np.convolve(y, np.ones(3, dtype='int') / 3, mode='valid')
    return x_filtered, y_filtered

# 调用轨迹跟踪算法
x_filtered, y_filtered = track(x, y)

# 绘制轨迹图像
plt.figure()
plt.plot(x, y, label='Original Track')
plt.plot(x_filtered, y_filtered, label='Filtered Track')
plt.legend()
plt.show()
```

这个代码实例中，我们首先定义了无人机轨迹数据，然后定义了一个简单的移动平均滤波算法，并将其应用于轨迹数据中。最后，我们绘制了原始轨迹和滤波后的轨迹图像，以便进行可视化比较。
