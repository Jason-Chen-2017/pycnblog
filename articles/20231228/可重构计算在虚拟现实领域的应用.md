# 2.核心概念与联系

可重构计算（Reconfigurable Computing）是一种可以根据应用需求动态调整结构和性能的计算方法，它主要包括以下几个核心概念：

1.硬件可重构：硬件可重构是指可以根据应用需求动态调整硬件结构和性能的计算机系统。通常，硬件可重构包括程序可重构（FPGA）和硬件可重构（ASIC）两种形式。程序可重构通常采用门逻辑结构实现，可以在运行时动态调整；硬件可重构通常采用定制芯片结构实现，可以在设计时动态调整。

2.软件可重构：软件可重构是指可以根据应用需求动态调整软件结构和性能的计算机程序。通常，软件可重构包括编译时可重构和运行时可重构两种形式。编译时可重构通常采用多种实现方案并在编译期间选择最佳实现；运行时可重构通常采用动态调整算法和数据结构实现。

3.可重构计算架构：可重构计算架构是一种可以支持硬件可重构和软件可重构的计算机架构。通常，可重构计算架构包括程序可重构（FPGA）、硬件可重构（ASIC）和软件可重构（编译时可重构、运行时可重构）三种形式。

在虚拟现实领域，可重构计算可以为用户提供更加实时、准确、高质量的虚拟现实体验。例如，可重构计算可以用于实时处理用户的交互操作，实时生成高质量的3D图像、音频和其他感官刺激，实时调整虚拟现实场景以满足用户的需求。
