# 2.核心概念与联系

## 2.1 凸函数

凸函数是一种特殊的函数，它在其定义域内具有最大值，并且在任何一点的梯度都指向函数的凸区域。形式上，一个函数f(x)在域D上是凸的，如果对于任何x1、x2在D中，和任何0≤λ≤1，都有f(λx1+(1-λ)x2)≤λf(x1)+(1-λ)f(x2)。

## 2.2 博弈论

博弈论研究多个玩家在做决策时如何取得最大利益。博弈可以被表示为一个游戏，其中每个玩家都有一组可能的行动，每个行动都有一个对应的结果。博弈论的一个重要概念是纯策略和混策略。纯策略是玩家在游戏中选择一个固定的行动，而混策略是玩家根据对手的行动选择行动。

## 2.3 凸博弈

凸博弈是一种特殊的博弈，其中每个玩家的利益可以表示为一个凸函数。在凸博弈中，玩家的最优策略可以通过求解凸函数的极值来找到。这种方法被称为凸性结果。

## 3.2 求解凸函数的极值

求解凸函数的极值可以通过多种方法来实现，例如梯度下降、新凯撒法等。这里我们介绍一种常用的求解凸函数极值的方法：梯度下降。

梯度下降是一种迭代的优化算法，它通过不断地更新参数来逼近函数的极值。具体来说，梯度下降算法的步骤如下：

1. 初始化参数值。
2. 计算梯度。
3. 更新参数值。
4. 重复步骤2和步骤3，直到收敛。

在求解凸函数极值时，梯度下降算法的表达式为：

$$
x_{k+1} = x_k - \alpha \nabla f(x_k)
$$

其中，$x_k$ 表示第k次迭代的参数值，$\alpha$ 表示学习率，$\nabla f(x_k)$ 表示第k次迭代的梯度。
