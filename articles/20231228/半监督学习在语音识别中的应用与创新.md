# 4. 具体代码实例和解释

在这里，我们以一个简单的半监督学习语音识别示例为例，展示半监督学习在语音识别中的具体代码实例和解释。

```python
import numpy as np
from sklearn.mixture import GaussianMixture
from sklearn.datasets import make_blobs

# 生成语音数据和标签
X, y = make_blobs(n_samples=1000, centers=2, cluster_std=0.6, random_state=0)
y[np.random.rand(*X.shape) > 0.5] = -1

# 使用GaussianMixture进行半监督学习
gmm = GaussianMixture(n_components=2, random_state=0).fit(X)

# 预测未标注数据的标签
y_pred = gmm.predict(X)

# 计算准确率
accuracy = np.mean(y_pred == y)
print("Accuracy: %.2f" % (accuracy * 100))
```

在这个示例中，我们首先生成了语音数据和标签，其中有一部分数据是未标注的。然后，我们使用`GaussianMixture`进行半监督学习，并预测未标注数据的标签。最后，我们计算准确率，以评估模型的性能。
