                 

# 1.背景介绍

宇宙背景辐射（CMB，Cosmic Microwave Background），也被称为微波背景辐射，是宇宙最早时期形成的微波光辐射。它是现代宇宙物理学的一个关键证据，也是大气中微波传播的一个重要干扰。在1964年，阿姆斯特朗·哥贝尔（Arno Penzias）和罗伯特·沃尔夫（Robert W. Wilson）在新泽西州的 Хо伯特塔（Hobart, New Jersey）观测到这种微波辐射，为宇宙背景辐射的发现奠定了基础。

宇宙背景辐射的温度约为2.725K（公平的说，这个温度并不是一个稳定的数字，而是一个在不断变化的数字，但在整体上保持稳定），这是宇宙最早时期形成的光辐射，是宇宙的“初代光”。在大气中，这些微波是由于气体分子的振动和转动所吸收和散射而被掩盖的。因此，只有在大气高层或从太空观测时，我们才能看到这些微波。
