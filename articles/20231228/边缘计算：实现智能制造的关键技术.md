2. 降低网络负载：边缘计算可以将部分数据处理任务从中心化服务器移动到边缘设备，从而降低网络负载，提高系统吞吐量。

4. 提高系统可扩展性：边缘计算可以通过将计算能力分布在边缘设备上，实现系统可扩展性，满足不同规模的智能制造需求。

# 2.核心概念与联系

## 2.1 边缘计算定义

边缘计算（Edge Computing）是一种新兴的计算技术，它将计算能力从中心化的数据中心移动到边缘设备，如物联网设备、智能手机、车载电子等。边缘计算可以实现数据处理的分布式、智能化和实时性，从而提高系统性能和数据安全性。

## 2.2 边缘计算与云计算的区别

边缘计算与云计算是两种不同的计算技术。云计算是一种中心化的计算模式，它将计算能力集中在数据中心，通过网络提供服务给边缘设备。边缘计算则将计算能力分布在边缘设备上，实现数据处理的分布式、智能化和实时性。

边缘计算与云计算的主要区别在于：

1. 计算能力的分布：边缘计算将计算能力分布在边缘设备上，云计算将计算能力集中在数据中心。

2. 数据处理的实时性：边缘计算可以实现数据处理的实时性，云计算的实时性受网络延迟和数据中心负载的影响。

3. 数据安全性：边缘计算可以提高数据安全性，因为敏感数据可以在边缘设备上处理。云计算的数据安全性受数据传输和存储的风险影响。

## 2.3 边缘计算与物联网的联系

边缘计算与物联网密切相关。物联网是一种技术，它将物理设备通过网络连接起来，实现设备之间的数据交换和控制。边缘计算可以在物联网设备上实现数据处理，从而提高系统性能和数据安全性。

边缘计算与物联网的主要联系在于：

1. 数据处理的分布式：边缘计算将数据处理分布在物联网设备上，实现数据处理的分布式。

2. 实时性：边缘计算可以实现物联网设备的数据处理实时性，从而提高系统响应速度。

3. 数据安全性：边缘计算可以提高物联网设备的数据安全性，因为敏感数据可以在设备上处理。
