# 4.具体代码实例和详细解释说明

## 4.1 条形图实例

以下是一个简单的条形图实例：

```python
import matplotlib.pyplot as plt

data = [10, 20, 30, 40, 50]
categories = ['A', 'B', 'C', 'D', 'E']

plt.bar(categories, data)
plt.show()
```

在这个实例中，我们使用了matplotlib库来绘制一个简单的条形图。`categories`变量表示数据的分类，`data`变量表示数据的值。`plt.bar()`函数用于绘制条形图，`plt.show()`函数用于显示图像。

## 4.2 折线图实例

以下是一个简单的折线图实例：

```python
import matplotlib.pyplot as plt

data = [1, 2, 3, 4, 5]

plt.plot(data)
plt.show()
```

在这个实例中，我们使用了matplotlib库来绘制一个简单的折线图。`data`变量表示数据的值。`plt.plot()`函数用于绘制折线图，`plt.show()`函数用于显示图像。

## 4.3 饼图实例

以下是一个简单的饼图实例：

```python
import matplotlib.pyplot as plt

data = [10, 20, 30, 40, 50]
categories = ['A', 'B', 'C', 'D', 'E']

plt.pie(data, labels=categories)
plt.show()
```

在这个实例中，我们使用了matplotlib库来绘制一个简单的饼图。`categories`变量表示数据的分类，`data`变量表示数据的值。`plt.pie()`函数用于绘制饼图，`plt.show()`函数用于显示图像。
