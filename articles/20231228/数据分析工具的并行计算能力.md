# 2.核心概念与联系

## 2.1 并行计算的基本概念

并行计算是指同时处理多个任务，以提高计算效率。并行计算可以分为两种：

1. 并行计算：多个处理单元同时处理不同的任务，以提高计算效率。
2. 并行处理：同一个处理单元同时处理多个任务，以提高计算效率。

并行计算的主要特点包括：

- 并行度：并行计算中，处理单元的数量。
- 数据分区：并行计算中，数据被划分为多个部分，每个处理单元处理一部分数据。
- 通信：并行计算中，处理单元之间需要进行通信，以交换信息。
