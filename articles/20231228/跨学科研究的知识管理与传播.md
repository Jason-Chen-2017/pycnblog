## 4.1 知识管理代码实例

以下是一个简单的知识管理系统的代码实例，使用Python编程语言和Flask web框架实现。

```python
from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///knowledge.db'
db = SQLAlchemy(app)

class Knowledge(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(100), nullable=False)
    content = db.Column(db.Text, nullable=False)

@app.route('/knowledge', methods=['GET', 'POST'])
def knowledge():
    if request.method == 'GET':
        knowledge_list = Knowledge.query.all()
        return jsonify([{'id': k.id, 'title': k.title, 'content': k.content} for k in knowledge_list])
    elif request.method == 'POST':
        data = request.get_json()
        new_knowledge = Knowledge(title=data['title'], content=data['content'])
        db.session.add(new_knowledge)
        db.session.commit()
        return jsonify({'id': new_knowledge.id, 'title': new_knowledge.title, 'content': new_knowledge.content})

if __name__ == '__main__':
    app.run(debug=True)
```

上述代码实例实现了一个简单的知识管理系统，包括知识的收集、存储、查询和应用。用户可以通过POST请求将知识资源发布到系统中，同时系统也提供了GET请求查询所有知识资源的接口。

## 4.2 知识传播代码实例

以下是一个简单的知识传播系统的代码实例，使用Python编程语言和SMTP邮箱发送邮件实现。

```python
import smtplib
from email.mime.text import MIMEText

def send_email(subject, content, to):
    sender = 'your_email@example.com'
    password = 'your_password'
    receiver = to

    msg = MIMEText(content)
    msg['Subject'] = subject
    msg['From'] = sender
    msg['To'] = receiver

    server = smtplib.SMTP('smtp.example.com', 587)
    server.starttls()
    server.login(sender, password)
    server.sendmail(sender, [receiver], msg.as_string())
    server.quit()

subject = '知识传播示例'
content = '这是一个知识传播示例，请查收。'
to = 'recipient@example.com'

send_email(subject, content, to)
```

上述代码实例实现了一个简单的知识传播系统，通过发送邮件将知识传播给指定的受众。用户可以通过调用send_email函数将知识内容发送给指定的邮箱地址。
