```python
import time
import requests

# 设备的API地址
device_api = "http://device_api"

# 设备的ID
device_id = "device_id"

# 获取设备的运行状况
def get_device_status(device_id):
    url = f"{device_api}/status/{device_id}"
    response = requests.get(url)
    return response.json()

# 主程序
if __name__ == "__main__":
    while True:
        # 获取设备的运行状况
        status = get_device_status(device_id)
        # 判断设备的运行状况
        if status["status"] == "fault":
            # 发送故障通知
            send_fault_notification(status)
        time.sleep(60)
```

在这个代码实例中，我们首先导入了`time`和`requests`模块，然后定义了一个`get_device_status`函数，该函数用于获取设备的运行状况。接着，我定义了一个`send_fault_notification`函数，该函数用于发送故障通知。最后，我定义了一个主程序，该程序不断获取设备的运行状况，判断设备的运行状况，如果设备发生故障，则发送故障通知。
