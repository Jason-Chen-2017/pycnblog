# 2.核心概念与联系

1. 气体传感器：用于检测空气中的污染物，如二氧化碳、二氧化硫、氮氧化物等。
2. 微处理器：用于收集、处理和分析气体传感器的数据，并根据需要调节车内环境。
3. 控制系统：用于根据微处理器的指令自动调节车内环境，如调节空气流量、调节温度和湿度等。
