                 

# 1.背景介绍

自动锚点分割（Automatic Time Series Slicing, ATSS）是一种针对时间序列分割任务的突破性算法，它能够自动地识别并分割时间序列数据中的不同阶段，从而提高了时间序列分析和预测的准确性。ATSS的核心思想是通过自适应地学习锚点（即时间序列中具有代表性的数据点），从而实现对时间序列数据的更精确的分割。

在过去的几年里，时间序列分析和预测已经成为数据科学和人工智能领域的一个重要研究方向，它在金融、气象、医疗等领域具有广泛的应用价值。然而，传统的时间序列分割方法往往需要人工设定锚点，这会导致较高的成本和较低的准确性。ATSS则通过自动地识别和分割时间序列数据的不同阶段，从而实现了更高的准确性和更低的成本。
