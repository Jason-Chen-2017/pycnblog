# 2.核心概念与联系

## 2.1 马氏距离

马氏距离（Mahalanobis distance）是一种度量两个向量之间距离的方法，它考虑了向量之间的方差，从而更准确地表示两个向量之间的相似性。马氏距离的公式如下：

$$
D = \sqrt{(x_1 - y_1)^2 / \sigma_1^2 + (x_2 - y_2)^2 / \sigma_2^2 + \cdots + (x_n - y_n)^2 / \sigma_n^2}
$$

其中，$x_i$ 和 $y_i$ 分别是向量 $x$ 和向量 $y$ 的第 $i$ 个元素，$\sigma_i^2$ 是向量 $x_i$ 和向量 $y_i$ 的方差。
