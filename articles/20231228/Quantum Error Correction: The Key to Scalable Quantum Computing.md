# 2.核心概念与联系
# 2.1量子位和量子错误
量子位（qubit）是量子计算机中的基本单元，它可以表示为一个复数向量。与经典位不同，量子位可以同时存在多个状态。然而，量子位也比经典位更容易受到外界干扰和稳定性问题的影响，这导致了量子错误的产生。

量子错误可以是由于量子位的稳定性问题引起的，例如过度震荡（decoherence），或由于量子操作的不准确性引起的，例如量子门的误操作。这些错误可能会导致量子计算机的输出结果不准确，从而影响其计算能力。

# 2.2量子错误纠正
量子错误纠正是一种用于检测和纠正量子错误的技术。其主要目标是提高量子计算机的稳定性和可靠性，从而提高其计算能力。量子错误纠正技术可以分为两类：基于测量的纠正（measurement-based error correction）和基于重传的纠正（retransmission-based error correction）。

基于测量的纠正通常涉及将量子位转换为经典位，然后对其进行测量和检测错误。基于重传的纠正则是通过重复发送量子信息来提高信息传输的可靠性。
