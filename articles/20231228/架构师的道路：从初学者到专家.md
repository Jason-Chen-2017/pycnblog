## 4.具体代码实例和详细解释说明

### 4.1 代码实例

在这里，我们以一个简单的大数据计算任务为例，来展示代码实例。这个任务是计算一个大数据集中的平均值。

```python
from pyspark import SparkContext

sc = SparkContext("local", "Avg")

data = sc.textFile("hdfs://localhost:9000/data.txt")

values = data.map(lambda line: int(line))

avg = values.reduce(lambda x, y: x + y) / values.count()

print(avg)
```

### 4.2 详细解释说明

这个代码实例使用了Spark框架来计算一个大数据集中的平均值。首先，我们创建了一个SparkContext对象，并指定了本地文件系统的HDFS地址。然后，我们使用textFile函数读取数据文件，并将其转换为RDD。接着，我们使用map函数将每一行数据转换为整数。最后，我们使用reduce函数计算平均值，并将其打印出来。
