## 4.具体代码实例和详细解释说明

### 4.1 订单管理智能合约示例

以以太坊平台为例，下面是一个简单的订单管理智能合约示例：

```solidity
pragma solidity ^0.5.0;

contract OrderManagement {
    struct Order {
        uint id;
        uint price;
        address buyer;
        address seller;
        bool isPaid;
    }

    Order[] public orders;

    function createOrder(uint _price, address _buyer, address _seller) public {
        uint id = orders.length;
        orders.push(Order({
            id: id,
            price: _price,
            buyer: _buyer,
            seller: _seller,
            isPaid: false
        }));
    }

    function payOrder(uint _id) public {
        Order storage order = orders[_id];
        require(order.isPaid == false, "Order has already been paid");
        order.isPaid = true;
    }
}
```

上述智能合约示例包括以下几个函数：

1. `createOrder`：创建订单，将订单信息存储到`orders`数组中。
2. `payOrder`：支付订单，将订单状态从`isPaid`为`false`更改为`true`。

### 4.2 付款管理智能合约示例

以以太坊平台为例，下面是一个简单的付款管理智能合约示例：

```solidity
pragma solidity ^0.5.0;

contract PaymentManagement {
    address public payee;
    uint public amount;

    function setPayee(address _payee) public {
        require(msg.sender == payee, "Only payee can set new payee");
        payee = _payee;
    }

    function pay(uint _amount) public {
        require(msg.sender == payee, "Only payee can pay");
        require(_amount <= amount, "Insufficient balance");
        amount -= _amount;
        payee.transfer(_amount);
    }
}
```

上述智能合约示例包括以下几个函数：

1. `setPayee`：设置付款收款人，只有当前付款收款人可以设置新的付款收款人。
2. `pay`：支付金额，只有当前付款收款人可以支付。
