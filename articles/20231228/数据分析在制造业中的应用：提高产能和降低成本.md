```python
import pandas as pd

    '产品生命周期': [1, 2, 3, 4, 5]
```

## 4.2 数据预处理

接下来，我们需要对数据进行清洗和预处理。这包括去除缺失值、数据类型转换、数据归一化等。

```python
# 去除缺失值
df = df.dropna()

# 数据类型转换
df['生产需求'] = df['生产需求'].astype(int)
df['市场需求'] = df['市场需求'].astype(int)
df['产品生命周期'] = df['产品生命周期'].astype(int)

from sklearn.preprocessing import MinMaxScaler

scaler = MinMaxScaler()
df[['生产需求', '市场需求', '产品生命周期']] = scaler.fit_transform(df[['生产需求', '市场需求', '产品生命周期']])
```

```python
# 导入线性回归模型
from sklearn.linear_model import LinearRegression

# 选择线性回归模型
model = LinearRegression()
```
