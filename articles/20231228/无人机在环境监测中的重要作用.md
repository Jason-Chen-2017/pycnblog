无人机控制代码实例（使用Python语言）：

```python
import time
import dronekit

# 连接无人机
drone = dronekit.connect('127.0.0.1:14550', wait_ready=True)

# 设置飞行路线
waypoints = [(100, 100), (200, 200), (300, 300)]

# 飞行
drone.simple_takeoff(altitude=100)
time.sleep(10)
drone.simple_go_to(waypoints)
```

传感器数据收集代码实例（使用Python语言）：

```python

import dronekit

# 连接无人机
drone = dronekit.connect('127.0.0.1:14550', wait_ready=True)

# 获取传感器数据

pressure = drone.observable_property.pressure

# 打印传感器数据

print("湿度：", humidity)
print("气压：", pressure)
```
