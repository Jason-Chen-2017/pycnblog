## 4.1 异常检测

我们将通过一个简单的异常检测示例来详细解释异常检测的实现过程。

### 4.1.1 统计方法

我们将通过计算数据的平均值和标准差来识别异常情况。例如，假设我们有一个数据集，包括正常数据和异常数据，如下：

```python
data = [10, 12, 12, 13, 12, 11, 14, 13, 15, 100]
```

我们可以计算数据的平均值和标准差，如下：

```python
import numpy as np

data = np.array(data)
average = np.mean(data)
std_dev = np.std(data)

print("平均值:", average)
print("标准差:", std_dev)
```

通过计算平均值和标准差，我们可以识别异常情况。例如，如果一个值的绝对值大于两倍的标准差，则可以认为该值是异常值。

```python
threshold = 2 * std_dev

for value in data:
    if abs(value - average) > threshold:
        print(value, "是异常值")
```

### 4.1.2 机器学习方法

我们将通过使用决策树算法来识别异常情况。例如，假设我们有一个包含正常数据和异常数据的数据集，如下：

```python
data = [[10, 12], [12, 12], [12, 13], [12, 12], [11, 14], [13, 13], [15, 100]]
labels = ['正常', '正常', '正常', '正常', '异常', '异常', '异常']
```

我们可以使用决策树算法来训练模型，然后使用训练好的模型来识别异常情况。

```python
from sklearn.tree import DecisionTreeClassifier

X = np.array(data)
y = np.array(labels)

clf = DecisionTreeClassifier()
clf.fit(X, y)

for value in X:
    prediction = clf.predict([value])
    print(value, "是", prediction[0], "类别")
```
