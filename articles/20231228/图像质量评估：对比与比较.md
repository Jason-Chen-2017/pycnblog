# 2. 核心概念与联系

在图像质量评估中，我们需要关注以下几个核心概念：

1. 图像质量指标：图像质量指标是用于评估图像质量的数值量度，例如均方误差（MSE）、结构相似性指数（SSIM）等。

2. 图像压缩：图像压缩是指将图像数据压缩为较小的大小，以便于存储和传输。图像压缩技术包括失真压缩和无失真压缩，其中失真压缩通常采用波形压缩（JPEG）技术，无失真压缩通常采用波形变换（JPEG2000）技术。

3. 图像传输：图像传输是指将图像数据从一台设备传输到另一台设备。图像传输技术包括无线传输和有线传输，其中无线传输通常采用Wi-Fi和蓝牙技术，有线传输通常采用USB和HDMI技术。

4. 图像处理：图像处理是指对图像数据进行处理，以便于改善图像质量或实现特定的应用目的。图像处理技术包括图像增强、图像压缩、图像识别等。

5. 图像识别：图像识别是指将图像数据转换为数值数据，以便于计算机进行处理和分析。图像识别技术包括边缘检测、对象检测、目标识别等。

在这些核心概念之间，我们可以看到一些联系和关系。例如，图像质量指标可以用于评估图像处理和压缩技术的效果，图像传输技术可以用于实现图像处理和识别技术的应用，图像识别技术可以用于实现图像处理和传输技术的优化和改进。因此，在研究图像质量评估时，我们需要关注这些核心概念之间的联系和关系，以便于更好地了解图像质量评估的重要性和应用价值。
