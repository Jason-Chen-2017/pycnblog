# 4.具体代码实例和详细解释说明

以下是一个使用半监督KMeans算法处理气候数据的具体代码实例：

```python
import numpy as np
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

# 加载气候数据
data = np.loadtxt('climate_data.txt')

# 标准化气候数据
scaler = StandardScaler()
data = scaler.fit_transform(data)

# 选择一部分已知标签的数据作为初始聚类中心
initial_centers = data[:5]

# 使用半监督KMeans算法进行聚类
kmeans = KMeans(n_clusters=3, init=initial_centers)
kmeans.fit(data)

# 输出聚类结果
labels = kmeans.labels_
print(labels)
```

在这个代码实例中，我们首先加载了气候数据，然后使用标准化方法对数据进行预处理。接着，我们选择了一部分已知标签的数据作为初始聚类中心，并使用半监督KMeans算法进行聚类。最后，我们输出了聚类结果。
