教育数据分析的核心概念包括：

1.数据清洗：对教育数据进行清洗和预处理，以消除噪声和错误数据，提高数据质量。

2.数据分析：对清洗后的教育数据进行分析，以挖掘隐藏的知识和规律。

3.模型构建：根据数据分析结果，构建教育数据分析模型，以支持教师职业发展的决策。

4.模型评估：对构建的模型进行评估，以确保其准确性和可靠性。

5.模型应用：将模型应用于实际教育场景，以实现教师职业发展的目标。

在教育数据分析中，我们可以使用以下数学模型公式：

1.均值（Mean）：对一组数据的和除以数据的个数。

$$
Mean = \frac{\sum_{i=1}^{n} x_i}{n}
$$

2.中位数（Median）：将数据排序后，中间值。

3.方差（Variance）：对一组数据的平均值与其平均值的差的平均值。

$$
Variance = \frac{\sum_{i=1}^{n} (x_i - Mean)^2}{n}
$$

4.标准差（Standard Deviation）：方差的平方根。

$$
Standard\ Deviation = \sqrt{Variance}
$$

在教育数据分析中，我们可以通过以下具体代码实例和详细解释说明：

1.使用Python的Pandas库进行数据清洗和预处理。

```python
import pandas as pd

# 读取教育数据
data = pd.read_csv('education_data.csv')

# 数据清洗
data = data.dropna()  # 删除缺失值
data = data[data['score'] > 0]  # 删除分数为0的记录
```

2.使用Python的Scikit-learn库进行数据分析和模型构建。

```python
from sklearn.linear_model import LinearRegression

# 数据分析
X = data[['age', 'gender', 'score']]
y = data['salary']

# 模型构建
model = LinearRegression()
model.fit(X, y)
```

3.使用Python的Scikit-learn库进行模型评估。

```python
from sklearn.metrics import mean_squared_error

# 模型评估
y_pred = model.predict(X)
mse = mean_squared_error(y, y_pred)
print('Mean Squared Error:', mse)
```

4.使用Python的Scikit-learn库进行模型应用。

```python
# 模型应用
age = 30
gender = 'male'
score = 80
salary = model.predict([[age, gender, score]])
print('Predicted Salary:', salary)
```
