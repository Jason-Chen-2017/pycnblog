## 4.1 代码实例

我们以一个简单的流处理程序为例，该程序可以实时计算数据流中的平均值。

```python
import numpy as np

def process_data(data_stream):
    # 初始化累积和和计数器
    cumulative_sum = 0
    count = 0
    # 遍历数据流
    for data in data_stream:
        # 更新累积和和计数器
        cumulative_sum += data
        count += 1
        # 计算平均值
        average = cumulative_sum / count
        # 输出平均值
        print("Average: ", average)

# 生成一个数据流
data_stream = np.random.randn(1000)
# 调用流处理程序
process_data(data_stream)
```

## 4.2 详细解释说明

1. 首先，我们导入了numpy库，该库可以用于生成和处理数据流。
2. 然后，我们定义了一个名为`process_data`的函数，该函数接收一个数据流作为参数。
3. 在函数内部，我们初始化了累积和和计数器，分别用于存储数据流中的总和和数据个数。
4. 接着，我们遍历了数据流，对于每个数据，我们分别更新累积和和计数器。
5. 然后，我们计算了平均值，即累积和除以计数器。
6. 最后，我们输出了平均值。
7. 最后，我们生成了一个数据流，并调用了流处理程序进行处理。
