## 4.4 降低运输成本

我们将使用一个简单的粒子群优化算法来降低运输成本。首先，我们需要导入所需的库：

```python
import random
```

接下来，我们需要定义粒子群优化算法的函数：

```python
def particle_swarm_optimization(population, fitness_function, w, c1, c2, num_iterations):
    for _ in range(num_iterations):
        for i in range(len(population)):
            r1, r2 = random.random(), random.random()
            if random.random() < c1:
                population[i][0] = population[i][0] + r1 * (personal_best[i][0] - population[i][0])
            if random.random() < c2:
                population[i][0] = population[i][0] + r2 * (global_best[0] - population[i][0])
            if fitness_function(population[i][0]) > fitness_function(personal_best[i][0]):
                personal_best[i] = population[i]
            if fitness_function(population[i][0]) > fitness_function(global_best[0]):
                global_best = population[i]
    return global_best
```

接下来，我们需要初始化粒子群：

```python
population = [random.randint(0, 100) for _ in range(10)]
```

接下来，我们需要进行粒子群优化算法循环：

```python
w = 0.7
c1 = 1.5
c2 = 1.5
num_iterations = 100
global_best = max(population)
personal_best = [x for x in population]
for i in range(num_iterations):
    new_population = []
    for _ in range(len(population) // 2):
        particle = particle_swarm_optimization(population, evaluate, w, c1, c2, 10)
        new_population.append(particle)
    population = new_population
```
