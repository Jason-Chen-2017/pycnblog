# 4.具体代码实例和详细解释说明

```python
from oraclenosql import OracleNoSQL

# 创建OracleNoSQL对象
nosql = OracleNoSQL()

# 设置数据源和目标数据库信息
nosql.set_source("source_host", "source_port", "source_user", "source_password")
nosql.set_target("target_host", "target_port", "target_user", "target_password")

# 设置数据迁移方法和顺序
nosql.set_method("full")
nosql.set_order("sequential")

# 启动数据迁移过程
nosql.start()

# 监控数据迁移过程
while nosql.is_running():
    print(nosql.status())

# 结束数据迁移过程
nosql.stop()
```

在这个代码实例中，我们首先导入了OracleNoSQL模块，然后创建了一个OracleNoSQL对象。接着，我们设置了数据源和目标数据库信息，以及数据迁移方法和顺序。最后，我们启动了数据迁移过程，并监控了迁移过程中的状态。
