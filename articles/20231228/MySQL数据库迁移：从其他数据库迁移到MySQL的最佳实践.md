## 4.1 代码实例

```
# 导入数据库库
import mysql.connector

# 连接源数据库
source_db = mysql.connector.connect(
    host='source_host',
    user='source_user',
    password='source_password',
    database='source_database'
)

# 连接目标数据库
target_db = mysql.connector.connect(
    host='target_host',
    user='target_user',
    password='target_password',
    database='target_database'
)

# 创建目标数据库表
cursor = target_db.cursor()
cursor.execute('''
    CREATE TABLE target_table (
        id INT PRIMARY KEY,
        name VARCHAR(255),
        age INT
    )
''')

# 导入数据
import pandas as pd
source_data = pd.read_sql('SELECT * FROM source_table', source_db)
target_data = pd.read_sql('SELECT * FROM target_table', target_db)
source_data.to_csv('source_data.csv', index=False)
target_data.to_csv('target_data.csv', index=False)

# 导入源数据到目标数据库
source_data.to_csv('source_data.csv', index=False)
target_data.to_csv('target_data.csv', index=False)

# 关闭数据库连接
source_db.close()
target_db.close()
```

## 4.2 解释说明

在这个代码实例中，我们首先导入了数据库库，并连接了源数据库和目标数据库。然后，我们创建了目标数据库表。接着，我们使用pandas库导入了数据，并将数据导入到目标数据库。最后，我们关闭了数据库连接。
