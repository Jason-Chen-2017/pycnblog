### 4.1.1建立制造线的数字模型

```python
class ManufacturingLine:
    def __init__(self, name, capacity):
        self.name = name
        self.capacity = capacity
        self.status = 'idle'
        self.products = []

    def start(self):
        self.status = 'running'

    def stop(self):
        self.status = 'idle'

    def produce(self, product):
        if self.status == 'running' and product not in self.products:
            self.products.append(product)
            return True
        else:
            return False
```
