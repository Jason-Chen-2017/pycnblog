```python
import networkx as nx
import matplotlib.pyplot as plt

# 构建社会网络图
G = nx.Graph()
G.add_edge("Alice", "Bob")
G.add_edge("Alice", "Charlie")
G.add_edge("Bob", "Charlie")

# 识别信息传播路径
path = nx.shortest_path(G, source="Alice", target="Charlie", weight="weight")
print(path)

# 评估信息可靠性

print(centrality)
```

在这个代码实例中，我们首先构建了一个简单的社会网络图，其中包括三个人（Alice、Bob和Charlie）。接下来，我们识别了信息传播路径，即从Alice到Charlie的路径。最后，我们评估了信息可靠性，通过计算每个人的度中心性。
