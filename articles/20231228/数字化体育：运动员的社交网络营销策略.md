## 4.1 知名度计算代码实例

以下是一个计算运动员知名度的Python代码实例：

```python
import numpy as np

def calculate_famous(followers, likes, comments):
    alpha = 0.5
    beta = 0.3
    gamma = 0.2
    famous = alpha * followers + beta * likes + gamma * comments
    return famous

```
