# 4. 具体代码实例和解释

在这里，我们以一个简单的情感分析案例为例，展示如何使用Python编程语言和NLTK库进行情感分析。

```python
import nltk
from nltk.sentiment import SentimentIntensityAnalyzer

# 初始化情感分析器
sia = SentimentIntensityAnalyzer()

# 定义一篇文章
article = "今天的股市表现非常好，投资者对未来的期待非常高。"

# 对文章进行情感分析
sentiment = sia.polarity_scores(article)

# 打印情感分析结果
print(sentiment)
```

上述代码首先导入了nltk库，并初始化了一个情感分析器。然后定义了一篇文章，并对文章进行情感分析。最后打印了情感分析结果。
