## 4.1 代码实例

以下是一个简单的代码实例，用于说明软件工程人才培养的具体方法。这个例子是一个简单的排序算法，使用了插入排序的方法。

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

## 4.2 详细解释说明

这个代码实例使用了插入排序的算法，它是一种简单的排序算法，具有较好的稳定性和时间复杂度。插入排序的主要思想是将一个记录插入到已排好的有序的子列中，从而得到新的有序的子列。

在这个例子中，我们首先定义了一个名为`insertion_sort`的函数，它接受一个列表`arr`作为输入参数。然后，我们使用了一个`for`循环来遍历列表中的每个元素。在每一次循环中，我们首先将当前元素存储在一个名为`key`的变量中，然后使用一个`while`循环来比较`key`与之前元素的大小。如果`key`小于当前元素，我们就将当前元素向后移动一位，直到找到正确的位置。最后，我们将`key`插入到正确的位置。
