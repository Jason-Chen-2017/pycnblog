在这里，我们以一个简单的航空航天大数据处理示例来展示如何使用Python进行数据清洗、数据分析和数据可视化。

## 4.1 数据清洗

```python
import pandas as pd
import numpy as np

# 加载数据
data = pd.read_csv('flight_data.csv')

# 处理缺失值
data['departure_delay'] = data['departure_delay'].fillna(data['departure_delay'].mean())

# 去除噪声
data['air_time'] = data['air_time'].apply(lambda x: x if x > 0 else np.nan)
data = data.dropna()

# 数据转换
data['air_time'] = np.log(data['air_time'])
```

## 4.2 数据分析

```python
# 计算平均值
average_delay = data['departure_delay'].mean()

# 计算方差
variance_delay = data['departure_delay'].var()

# 计算相关性
correlation = data['air_time'].corr(data['departure_delay'])
```

## 4.3 数据可视化

```python
import matplotlib.pyplot as plt

# 直方图
plt.hist(data['air_time'], bins=20)
plt.xlabel('Air Time (hours)')
plt.ylabel('Frequency')
plt.title('Air Time Distribution')
plt.show()

# 散点图
plt.scatter(data['air_time'], data['departure_delay'])
plt.xlabel('Air Time (hours)')
```
