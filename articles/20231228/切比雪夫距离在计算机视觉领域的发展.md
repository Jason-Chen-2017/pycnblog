# 2.核心概念与联系

## 2.1 切比雪夫距离定义

切比雪夫距离是一种度量距离，用于衡量两个特征之间的最大差异。它的定义如下：

$$
C_{ij} = \max_{k=1,2,...,n} |x_i^k - x_j^k|
$$

其中，$C_{ij}$ 是切比雪夫距离的表示，$x_i$ 和 $x_j$ 是两个特征向量，$x_i^k$ 和 $x_j^k$ 是特征向量 $x_i$ 和 $x_j$ 的第 $k$ 个元素，$n$ 是特征向量的维度。

## 4.1 代码实例

```python
import numpy as np

def chebyshev_distance(x_i, x_j):
    n = len(x_i)
    max_diff = 0
    for k in range(n):
        diff = abs(x_i[k] - x_j[k])
        if diff > max_diff:
            max_diff = diff
    return max_diff

x_i = np.array([1, 2, 3])
x_j = np.array([4, 5, 6])
C_ij = chebyshev_distance(x_i, x_j)
print("切比雪夫距离：", C_ij)
```
