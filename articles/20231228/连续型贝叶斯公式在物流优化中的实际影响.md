# 4.具体代码实例和详细解释说明

在这里，我们提供一个简单的Python代码实例，展示了如何使用连续型贝叶斯公式在物流优化中进行运输时间预测。

```python
import numpy as np
import scipy.integrate as spi
from scipy.stats import norm

# 先验分布参数
alpha = 1.0
beta = 1.0

# 观测函数参数
sigma = 0.5

# 观测数据
observations = np.array([2.0, 2.5, 3.0, 3.5, 4.0])

# 计算后验分布
def posterior_distribution(alpha, beta, sigma, observations):
    # 计算先验分布
    prior_mean = alpha / (alpha + beta)
    prior_variance = (alpha * beta) / ((alpha + beta) ** 2)
    
    # 计算观测函数
    likelihood_mean = np.mean(observations)
    likelihood_variance = sigma ** 2
    
    # 计算后验分布
    posterior_mean = (prior_mean * prior_variance + likelihood_mean * likelihood_variance) / (prior_variance + likelihood_variance)
    posterior_variance = (prior_variance * likelihood_variance) / (prior_variance + likelihood_variance)
    
    return posterior_mean, posterior_variance

# 主程序
if __name__ == '__main__':
    posterior_mean, posterior_variance = posterior_distribution(alpha, beta, sigma, observations)
    print('后验均值:', posterior_mean)
    print('后验方差:', posterior_variance)
```

在这个代码实例中，我们首先定义了先验分布参数和观测函数参数。然后，我们使用观测数据计算后验分布。最后，我们打印了后验均值和方差。
