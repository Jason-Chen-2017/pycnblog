# 2.核心概念与联系
排队论是一门研究在有限资源的情况下，多个请求或任务如何排队和被处理的学科。排队论的核心概念包括：

- 系统类型：电商系统通常被分为三类：M/M/1、M/M/c、M/G/1。其中，M表示指数分布的随机事件，M/M/1表示单个服务器处理请求，M/M/c表示多个服务器并行处理请求，M/G/1表示单个服务器处理不同时长的请求。
- 性能指标：排队论中的性能指标包括平均等待时间、平均吞吐率、平均响应时间等。
- 队列模型：排队论中的队列模型包括：FIFO（先来先服务）、LIFO（后来先服务）、PRIORITY（优先级服务）等。
