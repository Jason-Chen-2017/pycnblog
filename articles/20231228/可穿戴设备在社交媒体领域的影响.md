#### 3.1.1 数据预处理

数据预处理是对原始数据进行清洗和转换的过程，以便于后续的分析和应用。常见的数据预处理方法包括：

- 缺失值处理：当数据中存在缺失值时，可以使用各种方法来填充或删除缺失值，例如均值填充、删除缺失值等。

- 数据归一化：数据归一化是将数据转换到同一范围内的过程，可以使得不同单位的数据能够进行比较和分析。常见的归一化方法包括最小-最大归一化、标准化等。

#### 3.1.2 数据清洗

数据清洗是对原始数据进行纠正和过滤的过程，以便于后续的分析和应用。常见的数据清洗方法包括：

- 出liers检测：出liers是数据中异常值的称呼，它们可能会影响数据的分析结果。可以使用各种检测方法来检测和删除出liers，例如IQR方法、Z分数方法等。
- 数据纠正：当数据中存在错误值时，可以使用各种纠正方法来修正错误值，例如温度纠正、光线纠正等。

数据转换是将原始数据转换为有意义信息的过程。常见的数据转换方法包括：

- 数据压缩：数据压缩是将原始数据压缩为更小尺寸的过程，可以减少数据传输的开销。常见的压缩方法包括丢失压缩、无损压缩等。

#### 4.1.2 数据清洗

```python
# 检测出liers

Q3 = data.quantile(0.75)
IQR = Q3 - Q1

# 删除出liers
data = data[(data >= (Q1 - 1.5 * IQR)) & (data <= (Q3 + 1.5 * IQR))]
```
