## 1.1 无线通信技术的发展

无线通信技术的发展可以分为以下几个阶段：

1. **第一代无线电通信**（1980年代）：这一阶段的技术主要用于语音通信，通信速度较低，不支持数据传输。

2. **第二代无线电通信**（1990年代）：这一阶段的技术主要用于语音和数据通信，通信速度较高，支持移动电话和数据传输。

3. **第三代无线电通信**（2000年代）：这一阶段的技术主要用于语音、数据和视频通信，通信速度较高，支持网络浏览和多媒体传输。

4. **第四代无线电通信**（2010年代）：这一阶段的技术主要用于高速网络访问和实时视频传输，通信速度较高，支持移动互联网和云计算。

5. **第五代无线电通信**（2020年代）：这一阶段的技术主要用于超高速网络访问和实时视频传输，通信速度极高，支持物联网和人工智能。

## 1.2 无线通信技术在医疗服务中的应用

无线通信技术在医疗服务中的应用主要包括以下几个方面：

1. **远程医疗诊断和治疗**：通过无线通信技术，医生可以在远程监控患者的生理数据，如心率、血压、血糖等，并根据生理数据提供个性化的诊断和治疗建议。

2. **电子病历系统**：通过无线通信技术，医生可以在线查阅患者的病历，实现病历的共享和协作。

3. **医疗设备远程维护**：通过无线通信技术，医疗设备可以实时向维修中心报告故障信息，以便及时进行维护。

4. **医学影像传输**：通过无线通信技术，医学影像可以实时传输给其他医生进行诊断和治疗。

5. **医疗保健信息管理**：通过无线通信技术，医疗保健信息可以实时传输和共享，实现信息化管理。

        self.device_address = device_address
        self.ble = BLE(self.device_address)

    def run(self):
        while True:
            heart_rate = self.ble.read_heart_rate()
            print("Heart rate: {}".format(heart_rate))
            time.sleep(1)

        super(Doctor, self).__init__()

            print("Doctor: Patient's heart rate is {}".format(heart_rate))
            time.sleep(1)

    doctor = Doctor(doctor_device_address)
