## 4.1 简单气候模型（SAM）
以下是一段简单气候模型（SAM）的Python代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义微积分方程
def sam(T, dt):
    dx = 100
    dy = 100
    T_new = np.zeros((dy, dx))

    for y in range(1, dy - 1):
        for x in range(1, dx - 1):
            T_new[y][x] = T[y][x] + dt * (T[y + 1][x] - 2 * T[y][x] + T[y - 1][x] + T[y][x - 1] - T[y][x + 1]) / 6

    return T_new

# 初始温度分布
T = np.zeros((101, 101))
for y in range(101):
    for x in range(101):
        T[y][x] = np.sin(np.pi * x / 50) * np.sin(np.pi * y / 50)

# 时间步长
dt = 0.1

# 模型迭代
for _ in range(1000):
    T = sam(T, dt)

# 绘制温度分布
plt.imshow(T, cmap='hot', extent=[0, 50, 0, 50])
plt.colorbar()
plt.show()
```

在这个代码实例中，我们使用了Python的NumPy和Matplotlib库来实现简单气候模型（SAM）。我们首先定义了微积分方程，然后使用了初始温度分布来初始化模型。接着，我们使用了时间步长来迭代模型，最后绘制了温度分布。
