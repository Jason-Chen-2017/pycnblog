r = sr.Recognizer()

    print("请说话")
    audio = r.listen(source)

try:
    print("你说的是：" + r.recognize_google(audio))
except sr.UnknownValueError:
    print("알 수 없는 값")

    print("错误；{0}".format(e))

```python
import datetime

def set_alarm(time):
    current_time = datetime.datetime.now()
    alarm_time = current_time.replace(hour=time.hour, minute=time.minute)
    while alarm_time > current_time:
        current_time += datetime.timedelta(seconds=1)
    print("闹钟已设置为：" + alarm_time.strftime("%Y-%m-%d %H:%M"))

set_alarm(datetime.time(8, 30))
```

在这个代码实例中，我们使用了`datetime`库来实现设置闹钟功能。首先，我们定义了一个`set_alarm`函数，这个函数接受一个`time`参数，表示闹钟的时间。接着，我们获取当前时间，并将闹钟时间与当前时间进行比较。如果闹钟时间已经过去，我们将当前时间加上一个`timedelta`对象，表示每秒增加一秒。最后，我们将闹钟时间输出到控制台。
