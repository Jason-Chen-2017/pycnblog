# 2.核心概念与联系

## 2.1 区块链基础概念

区块链是一种分布式、去中心化的数据存储和交易技术，它由一系列交易组成的“区块”构成。每个区块包含一组交易，并与前一个区块通过一个唯一的哈希值连接。这种连接方式使得区块链具有不可篡改的特性，因为任何尝试更改一个区块都将破坏整个链条。

区块链还具有去中心化的特点，因为它没有单一的中心化管理体系。相反，区块链由多个节点组成，这些节点负责验证和存储交易。这种去中心化结构使区块链更加安全和可靠，因为它减少了单点故障和数据篡改的风险。

## 3.1 区块链算法原理

区块链算法的核心是一种称为“共识算法”的机制，它用于确定哪些交易是有效的，并将它们添加到区块链中。最常用的共识算法是“工作量证明”（Proof of Work，PoW）和“权益证明”（Proof of Stake，PoS）。

在PoW算法中，节点需要解决一些数学问题，以证明他们的交易是有效的。解决这些问题需要大量的计算资源，因此只有那些投入了足够的计算资源的节点才能成功获得奖励。这种机制有助于防止恶意节点篡改区块链。

在PoS算法中，节点的权益决定了他们的交易是否有效。权益通常基于节点拥有的数字资产，因此那些拥有更多资产的节点更有可能成功获得奖励。这种机制有助于防止恶意节点篡改区块链。

    def create_block(self, proof, previous_hash):
        block = {
            'index': len(self.chain) + 1,
            'timestamp': time.time(),
            'proof': proof,
            'previous_hash': previous_hash
        }
        self.chain.append(block)
        return block

    def get_last_block(self):
        return self.chain[-1]

    def hash(self, block):
        block_string = json.dumps(block, sort_keys=True).encode()
        return hashlib.sha256(block_string).hexdigest()

    def proof_of_work(self, last_proof, block_data):
        proof = 0
        while self.valid_proof(last_proof, proof, block_data) is False:
            proof += 1
        return proof

    def valid_proof(self, last_proof, proof, block_data):
        if self.hash(last_proof, block_data) == block_data['proof']:
            return True
        return False

    def valid_chain(self, chain):
        last_block = chain[-1]
        current_index = len(chain) - 1

        while current_index > 0:
            block = chain[current_index]
            block_data = {
                'index': block['index'],
                'timestamp': block['timestamp'],
                'proof': block['proof'],
                'previous_hash': block['previous_hash']
            }

            if self.hash(block) != block_data['previous_hash']:
                return False

            current_index -= 1

        return True

    def add_block(self, proof, previous_hash):
        new_block = self.create_block(proof, previous_hash)
        self.chain.append(new_block)
        return new_block
