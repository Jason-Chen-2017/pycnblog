## 4.1 随机过程代码实例
以下是一个简单的随机过程代码实例，它描述了一个掷骰子的随机过程：

```python
import random

def roll_dice():
    return random.randint(1, 6)

for i in range(10):
    print(roll_dice())
```

在这个代码实例中，我们首先导入了`random`模块，然后定义了一个`roll_dice`函数，该函数通过`randint`函数生成一个1到6的随机整数，模拟掷骰子的过程。接着，我们通过一个`for`循环，调用`roll_dice`函数并打印出结果，模拟了掷骰子的10次操作。

## 4.2 量子计算代码实例
以下是一个简单的量子计算代码实例，它描述了一个量子位的基本操作：

```python
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

qc = QuantumCircuit(1)
qc.h(0)
qc.measure_all()

simulator = Aer.get_backend('qasm_simulator')
t_qc = transpile(qc, simulator)
qobj = assemble(t_qc)
result = simulator.run(qobj).result()
counts = result.get_counts()

plot_histogram(counts)
```

在这个代码实例中，我们首先导入了`qiskit`模块，然后定义了一个`QuantumCircuit`对象，并对量子位进行了`h`门（ Hadamard门 ）的操作，然后对所有量子位进行了测量。接着，我们通过`Aer`模块获取了`qasm_simulator`后端，对量子电路进行了转换和汇编，然后通过`run`方法运行量子程序，获取结果。最后，通过`plot_histogram`方法绘制结果的直方图。
