## 4.1Python数据处理
Python是一种流行的数据处理语言。以下是一个简单的Python数据处理代码实例：

```python
import pandas as pd

# 导入数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.dropna()

# 数据转换
data['age'] = data['age'].astype(int)

# 数据聚合
data['total_income'] = data['income'] + data['salary']

# 数据减少
data = data[['name', 'age', 'total_income']]
```

在这个代码实例中，我们首先导入了pandas库，然后导入了CSV文件，接着对数据进行了清洗、转换、聚合和减少等操作。
