# 2.核心概念与联系

数字孪生的核心概念包括：

- 数字模型：数字孪生的基础，是对现实世界对象或系统的数字表示。
- 实时监控：通过数字模型实现对现实世界对象或系统的状态和行为的实时监控。
- 预测分析：通过数字模型对现实世界对象或系统的未来状态和行为进行预测分析。
- 虚拟实验：通过数字模型对现实世界对象或系统进行虚拟实验，以优化其设计和运行。

在交通运输行业中，数字孪生可以应用于以下方面：
