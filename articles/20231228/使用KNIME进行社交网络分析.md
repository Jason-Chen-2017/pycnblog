# 2. 核心概念与联系

社交网络是由人们之间建立的关系和互动组成的网络。社交网络可以用图论来描述，其中节点表示人或组织，边表示人或组织之间的关系或互动。

社交网络分析的核心概念包括：

1. 节点（Node）：节点表示社交网络中的实体，如人、组织等。
2. 边（Edge）：边表示节点之间的关系或互动。
3. 度（Degree）：节点的度是与其相连的边的数量。
4. 组件（Component）：组件是网络中连通的最大子集，即任何两个节点之间都存在一条路径。
5. 桥梁（Bridge）：桥梁是两个连通组件之间的边。
6. 循环（Cycle）：循环是节点序列中每个节点的度大于1的闭路。
7. 中心性（Centrality）：节点的中心性是其在社交网络中的重要性，可以通过度、短路径数、 closeness 等指标计算。
8. 社群（Community）：社群是网络中密集连接的节点集合。

## 3.1 构建社交网络

要在KNIME中构建社交网络，首先需要创建一个节点表示人或组织，并创建边表示他们之间的关系或互动。可以使用KNIME中的“创建节点”（Create Nodes）节点和“创建边”（Create Edges）节点实现这一功能。

## 3.2 计算节点度

节点度是与节点相连的边的数量。可以使用KNIME中的“计算度”（Calculate Degree）节点计算节点度。该节点将输入节点列表和边列表，并输出每个节点的度。

## 3.3 计算中心性

节点中心性是节点在社交网络中的重要性，可以通过度、短路径数、 closeness 等指标计算。可以使用KNIME中的“计算中心性”（Calculate Centrality）节点计算节点中心性。该节点支持多种中心性指标，如度中心性、短路径中心性和 closeness 中心性。

## 3.4 发现社群

社群是网络中密集连接的节点集合。可以使用KNIME中的“发现社群”（Find Communities）节点发现社群。该节点支持多种社群发现算法，如模组算法、快速重新排序算法等。
