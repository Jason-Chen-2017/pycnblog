### 1. 计算用户平均心率

**题目：** 根据用户连续运动过程中记录的心率数据，计算用户平均心率。

**输入：** 

```python
heart_rate_data = [150, 155, 160, 165, 170]
```

**输出：**

```python
average_heart_rate = 160.0
```

**解析：** 

```python
def calculate_average_heart_rate(heart_rate_data):
    total_heart_rate = sum(heart_rate_data)
    average_heart_rate = total_heart_rate / len(heart_rate_data)
    return average_heart_rate

heart_rate_data = [150, 155, 160, 165, 170]
average_heart_rate = calculate_average_heart_rate(heart_rate_data)
print(average_heart_rate)
```

### 2. 运动数据聚类分析

**题目：** 根据用户连续运动过程中记录的心率数据，将用户分为不同的运动群体。

**输入：**

```python
heart_rate_data = [150, 155, 160, 165, 170, 145, 152, 158, 168, 180]
```

**输出：**

```python
clusters = {
    'cluster_1': [150, 155, 160, 165],
    'cluster_2': [170, 145, 152, 158],
    'cluster_3': [168, 180]
}
```

**解析：** 

```python
from sklearn.cluster import KMeans

def cluster_heart_rate_data(heart_rate_data, num_clusters):
    kmeans = KMeans(n_clusters=num_clusters, random_state=0).fit(heart_rate_data.reshape(-1, 1))
    labels = kmeans.predict(heart_rate_data.reshape(-1, 1))
    clusters = {}
    for i, label in enumerate(labels):
        if label not in clusters:
            clusters[label] = []
        clusters[label].append(heart_rate_data[i])
    return clusters

heart_rate_data = [150, 155, 160, 165, 170, 145, 152, 158, 168, 180]
clusters = cluster_heart_rate_data(heart_rate_data, 3)
print(clusters)
```

### 3. 运动数据异常值检测

**题目：** 根据用户连续运动过程中记录的心率数据，检测并去除异常值。

**输入：**

```python
heart_rate_data = [150, 155, 160, 165, 170, 450, 152, 158, 168, 180]
```

**输出：**

```python
cleaned_heart_rate_data = [150, 155, 160, 165, 170, 152, 158, 168, 180]
```

**解析：**

```python
import numpy as np

def remove_outliers(heart_rate_data, z_threshold=3):
    mean_hr = np.mean(heart_rate_data)
    std_hr = np.std(heart_rate_data)
    z_scores = [(x - mean_hr) / std_hr for x in heart_rate_data]
    cleaned_data = [x for x, z in zip(heart_rate_data, z_scores) if np.abs(z) <= z_threshold]
    return cleaned_data

heart_rate_data = [150, 155, 160, 165, 170, 450, 152, 158, 168, 180]
cleaned_heart_rate_data = remove_outliers(heart_rate_data)
print(cleaned_heart_rate_data)
```
