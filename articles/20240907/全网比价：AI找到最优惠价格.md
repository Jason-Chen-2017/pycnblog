#### 3.1 数据清洗

**题目：** 编写一个函数，实现以下功能：给定一个包含商品价格的列表，去除重复的价格项，并将价格统一格式化为两位小数。

```python
def clean_prices(prices):
    # 请在这里实现函数
    pass

prices = [100.0, 200.0, 100.0, 300.0, 200.0]
clean_prices(prices)
```

**答案：** 

```python
def clean_prices(prices):
    unique_prices = list(set(prices))
    formatted_prices = [round(price, 2) for price in unique_prices]
    return formatted_prices

prices = [100.0, 200.0, 100.0, 300.0, 200.0]
print(clean_prices(prices))  # 输出：[100.0, 200.0, 300.0]
```

**解析：** 该函数首先使用 `set` 函数去除重复的价格项，然后使用列表推导式将价格统一格式化为两位小数。

#### 3.2 比价分析

**题目：** 编写一个函数，实现以下功能：给定两个商品的价格列表，返回两个商品的最小价格差。

```python
def compare_prices(price_list1, price_list2):
    # 请在这里实现函数
    pass

price_list1 = [100.0, 200.0, 300.0]
price_list2 = [150.0, 250.0, 350.0]
min_difference = compare_prices(price_list1, price_list2)
print(min_difference)  # 输出：50.0
```

**答案：**

```python
def compare_prices(price_list1, price_list2):
    merged_prices = price_list1 + price_list2
    merged_prices.sort()
    min_difference = merged_prices[-1] - merged_prices[0]
    return min_difference

price_list1 = [100.0, 200.0, 300.0]
price_list2 = [150.0, 250.0, 350.0]
min_difference = compare_prices(price_list1, price_list2)
print(min_difference)  # 输出：50.0
```

**解析：** 该函数首先将两个价格列表合并，然后对合并后的价格列表进行排序，最后返回最大价格与最小价格的差值。
