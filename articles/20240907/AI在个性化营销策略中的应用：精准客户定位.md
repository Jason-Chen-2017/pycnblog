##### 二、算法编程题库

###### 3. 客户数据预处理

**题目：** 假设你有一个包含客户信息的CSV文件，请编写一个Python脚本进行数据预处理，包括数据清洗、去重和格式转换。

**答案：** 

```python
import pandas as pd

def preprocess_data(file_path):
    # 读取CSV文件
    df = pd.read_csv(file_path)
    
    # 数据清洗
    df = df.dropna()  # 删除缺失值
    df = df[df['Age'] > 0]  # 过滤不合法的年龄值
    
    # 去重
    df = df.drop_duplicates()
    
    # 格式转换
    df['DateOfBirth'] = pd.to_datetime(df['DateOfBirth'])
    df['Age'] = df['DateOfBirth'].apply(lambda x: int((pd.Timestamp.now() - x).days / 365))
    
    return df

# 调用函数
df = preprocess_data('customer_data.csv')
```

**解析：** 该脚本使用Pandas库对客户数据进行了清洗、去重和格式转换，以便后续分析。

###### 4. 构建用户画像

**题目：** 假设你有一个用户行为数据集，请编写一个Python脚本构建用户画像，并输出每个用户的主要特征。

**答案：**

```python
import pandas as pd
from sklearn.cluster import KMeans

def build_user_profile(data_path, n_clusters=5):
    # 读取数据
    df = pd.read_csv(data_path)
    
    # 数据预处理
    df = preprocess_data(df)
    
    # 特征工程
    features = df[['Age', 'Income', 'Spending']]
    
    # K-means聚类
    kmeans = KMeans(n_clusters=n_clusters)
    df['Cluster'] = kmeans.fit_predict(features)
    
    # 输出用户画像
    for i in range(n_clusters):
        cluster_df = df[df['Cluster'] == i]
        print(f"Cluster {i}:")
        print(cluster_df.describe())
    
# 调用函数
build_user_profile('user_behavior_data.csv')
```

**解析：** 该脚本使用K-means聚类算法对用户行为数据进行聚类，构建用户画像，并输出每个用户集群的主要特征。
