## 1. 纺织机械创新背景

纺织机械创新是现代纺织工业发展的重要推动力。18世纪末，英国纺织工人詹姆斯·哈格里夫斯发明了珍妮纺纱机，大大提高了纺纱效率。随后，查尔斯·阿克莱特发明了水力纺纱机，将纺纱过程机械化，进一步推动了纺织工业的进步。

## 4. 代码实例与解析

### 1. 纱线牵伸系统算法实例

```python
def extend_spiral(spiral: List[int]) -> List[int]:
    extended_spiral = []
    for direction in spiral:
        if direction == 0:  # 向右
            extended_spiral.append(extended_spiral[-1] + 1)
        elif direction == 1:  # 向下
            extended_spiral.append(extended_spiral[-1] - 1)
        elif direction == 2:  # 向左
            extended_spiral.append(extended_spiral[-1] - 1)
        elif direction == 3:  # 向上
            extended_spiral.append(extended_spiral[-1] + 1)
    return extended_spiral
```
