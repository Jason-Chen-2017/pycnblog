### 算法编程题库

**算法编程题 1：** 设计一个算法，判断一个人体增强设备的通信是否正常。

**题目描述：** 给定一系列通信数据，判断一个人体增强设备是否在正常通信。通信正常的标准是：数据包必须按照一定的顺序传输，并且数据包的传输时间间隔不得超过一定阈值。

**输入：** 通信数据数组，每个数据包包含数据内容和传输时间。

**输出：** 如果通信正常，返回“通信正常”；否则，返回“通信异常”。

**示例：**

```plaintext
输入：[[1, 1], [2, 2], [3, 3], [4, 4]]
输出：“通信正常”

输入：[[1, 1], [2, 3], [3, 3], [4, 4]]
输出：“通信异常”
```

**答案：**

```go
package main

import "fmt"

func isCommunicationNormal(data [][]int) string {
    for i := 1; i < len(data); i++ {
        if data[i][1]-data[i-1][1] > 1000 {
            return "通信异常"
        }
    }
    return "通信正常"
}

func main() {
    data1 := [][]int{{1, 1}, {2, 2}, {3, 3}, {4, 4}}
    fmt.Println(isCommunicationNormal(data1)) // 输出：“通信正常”

    data2 := [][]int{{1, 1}, {2, 3}, {3, 3}, {4, 4}}
    fmt.Println(isCommunicationNormal(data2)) // 输出：“通信异常”
}
```
