##### 4. 实现一个函数，判断一个字符串是否是回文字符串。

**题目：** 编写一个函数 `isPalindrome(s string) bool`，判断一个字符串 `s` 是否是回文字符串。

**答案：**

```go
func isPalindrome(s string) bool {
    for i, j := 0, len(s)-1; i < j; i, j = i+1, j-1 {
        if s[i] != s[j] {
            return false
        }
    }
    return true
}
```

**解析：** 该函数使用两个指针 `i` 和 `j` 分别从字符串的两端开始，逐步向中间移动，比较对应位置的字符是否相同。如果所有对应位置的字符都相同，则字符串是回文字符串。

##### 5. 实现一个函数，找出字符串中的所有子串。

**题目：** 编写一个函数 `findAllSubstrings(s string) []string`，找出字符串 `s` 中的所有子串。

**答案：**

```go
func findAllSubstrings(s string) []string {
    var subs []string
    n := len(s)
    for i := 0; i < n; i++ {
        for j := i + 1; j <= n; j++ {
            subs = append(subs, s[i:j])
        }
    }
    return subs
}
```

**解析：** 该函数使用两层循环，外层循环 `i` 从 0 到 `n-1`，内层循环 `j` 从 `i+1` 到 `n`。对于每个 `i` 和 `j` 的组合，截取字符串 `s[i:j]` 并添加到结果数组 `subs` 中。
