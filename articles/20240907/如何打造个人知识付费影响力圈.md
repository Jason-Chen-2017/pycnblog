在这个算法编程题中，我们将设计一个简易的在线课程系统，用于管理课程、学生和教师信息。我们将实现以下功能：

1. **添加课程**：允许添加新的课程。
2. **删除课程**：允许删除已存在的课程。
3. **查看课程**：显示所有课程的列表。
4. **添加学生**：允许添加学生到课程。
5. **删除学生**：允许从课程中删除学生。
6. **查看学生**：显示某个课程的学生的列表。

我们将使用Python语言来实现这个系统，并将课程、学生和教师的信息存储在字典中。以下是具体的代码实现和解析：

#### 代码实现

```python
class OnlineCourseSystem:
    def __init__(self):
        self.courses = {}  # 存储课程信息，键为课程ID，值为课程详情
        self.students = {}  # 存储学生信息，键为学生ID，值为学生姓名
        self.teacher = {}   # 存储教师信息，键为教师ID，值为教师姓名

    def add_course(self, course_id, course_name, teacher_id):
        # 添加新的课程
        self.courses[course_id] = {
            'name': course_name,
            'students': [],
            'teacher_id': teacher_id
        }
        # 将教师与课程关联
        if teacher_id in self.teacher:
            self.teacher[teacher_id]['courses'].append(course_id)
        else:
            self.teacher[teacher_id] = {
                'name': self.teacher.get(teacher_id, {}),
                'courses': [course_id]
            }

    def delete_course(self, course_id):
        # 删除课程
        if course_id in self.courses:
            del self.courses[course_id]
            # 更新教师信息
            if 'courses' in self.teacher[course_id]:
                self.teacher[course_id]['courses'].remove(course_id)

    def list_courses(self):
        # 显示所有课程的列表
        return [course['name'] for course in self.courses.values()]

    def add_student(self, student_id, student_name, course_id):
        # 添加学生到课程
        if student_id in self.students:
            self.students[student_id].append(course_id)
        else:
            self.students[student_id] = [course_id]
        # 将学生添加到课程的学生列表中
        if course_id in self.courses:
            self.courses[course_id]['students'].append(student_id)

    def delete_student(self, student_id, course_id):
        # 从课程中删除学生
        if student_id in self.students:
            self.students[student_id].remove(course_id)
        # 更新课程的学生列表
        if course_id in self.courses:
            self.courses[course_id]['students'].remove(student_id)

    def list_students(self, course_id):
        # 显示某个课程的学生的列表
        if course_id in self.courses:
            return [self.students[student_id] for student_id in self.courses[course_id]['students']]
        else:
            return []
```

4. **删除课程**：`delete_course` 方法用于删除课程，并更新教师的课程列表。
