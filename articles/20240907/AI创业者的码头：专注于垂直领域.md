##### 1. LeetCode 605. 种花问题

**题目：** 假设你有一个长度为 n 的花坛，要种花，你需要先在花坛的每块土地上种上花，然后按下述方式浇水：如果相邻的两块土地种有花，则可以浇水。不能两种不同的花种在一块土地上，相邻的土地不能留空。给定一个 n 整数数组 flowerbed 表示花坛初始状态，其中 0 表示未种植的土壤，1 表示种植的土壤。能否在不破坏现有花朵的情况下种植更多的花朵？可以则返回 true，否则返回 false。

**代码：**

```python
class Solution:
    def canPlaceFlowers(self, flowerbed: List[int], n: int) -> bool:
        length = len(flowerbed)
        if length < 2:
            if flowerbed[0] == 0 and n == 0:
                return True
            else:
                return False
        else:
            for i in range(length):
                if flowerbed[i] == 0:
                    if i == 0 and flowerbed[i+1] == 0:
                        flowerbed[i] = 1
                        n -= 1
                    elif i == length-1 and flowerbed[i-1] == 0:
                        flowerbed[i] = 1
                        n -= 1
                    elif flowerbed[i-1] == 0 and flowerbed[i+1] == 0:
                        flowerbed[i] = 1
                        n -= 1
            return n <= 0
```

**解析：** 该代码使用了一个简单的遍历算法，逐个检查每个未种植的土壤，并判断是否可以种植花朵。如果可以，就将该土壤设置为已种植，并减少需要种植的花的数量。

##### 2. LeetCode 435. 无重叠区间

**题目：** 给定一个区间数组 intervals ，其中 intervals[i] = [starti, endi] ，返回需要更改的最小数量，以便使得数组中之间的区间无重叠。

**代码：**

```python
class Solution:
    def minInterval(self, intervals: List[List[int]]) -> List[List[int]]:
        intervals.sort(key=lambda x: x[1])
        left, right = 0, 0
        ans = []
        while left < len(intervals):
            if intervals[left][1] <= intervals[right][0]:
                ans.append(intervals[left])
                right = left
            left += 1
        return ans
```

**解析：** 该代码首先对区间数组按结束时间进行排序，然后使用两个指针 left 和 right 分别表示当前已选择的区间。遍历数组时，如果当前区间的结束时间小于或等于 right 指针所指向的区间的开始时间，说明当前区间与已选择的区间不重叠，可以将当前区间添加到答案中，并将 right 指针移动到当前区间的位置。最后返回答案数组。
