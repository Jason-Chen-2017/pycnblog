## 二、AI在旅游业中的算法编程题

### 1. 编写一个基于K-means算法的旅游景点聚类程序，要求输入一组旅游景点的地理位置坐标，输出景点聚类结果。

```python
import numpy as np
from sklearn.cluster import KMeans

def k_means_clustering(data, k):
    kmeans = KMeans(n_clusters=k, random_state=0).fit(data)
    labels = kmeans.predict(data)
    return labels

# 示例数据：一组旅游景点的地理位置坐标
data = np.array([[40.7128, -74.0060], [34.0522, -118.2437], [41.8781, -87.6298], [37.7749, -122.4194]])
k = 2

# 聚类结果
labels = k_means_clustering(data, k)
print("景点聚类结果：", labels)
```
