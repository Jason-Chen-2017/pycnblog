```python
def logical_inference(prems, conclusion):
    for prem in prems:
        if conclusion.startswith("not"):
            if prem.startswith("and"):
                return False
            else:
                return True
        else:
            if prem.startswith("and"):
                return True
            else:
                return False

prems = ["P and Q", "not Q"]
conclusion = "not P"
result = logical_inference(prems, conclusion)
print(result)  # 输出 True
```

**解析：** 这个程序使用规则匹配的方法，实现了基于逻辑运算的自然语言推理。

**题目2：实现一个语义相似度度量系统**

**题目描述：** 编写一个程序，实现一个基于 Word2Vec 的语义相似度度量系统，能够计算两个文本的语义相似度。

**答案：**

```python
from gensim.models import Word2Vec

def semantic_similarity(text1, text2):
    model = Word2Vec.load("word2vec.model")
    words1 = text1.split()
    words2 = text2.split()
    sum_similarity = 0
    for word1 in words1:
        for word2 in words2:
            if word1 in model.wv and word2 in model.wv:
                sum_similarity += model.wv.similarity(word1, word2)
    return sum_similarity / (len(words1) * len(words2))

text1 = "I love to eat pizza."
text2 = "Pizza is my favorite food."
similarity = semantic_similarity(text1, text2)
print(similarity)  # 输出相似度值
```

**解析：** 这个程序使用 Gensim 库中的 Word2Vec 模型，计算两个文本的语义相似度。
