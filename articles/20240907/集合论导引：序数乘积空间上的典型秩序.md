##### 1. 如何判断一个序列是否是可数序列？

**题目：** 编写一个函数，判断给定的序列是否是可数序列。

**答案：** 可数序列是指可以与自然数集合建立一一对应关系的序列。我们可以通过判断序列中的元素是否可以按顺序与自然数集合中的元素一一对应来判断序列是否是可数序列。

```python
def is_countable(seq):
    n = 1
    for item in seq:
        if item != n:
            return False
        n += 1
    return True
```

**解析：** 这个函数通过遍历序列中的元素，检查每个元素是否与自然数集合中的元素一一对应。如果发现某个元素不对应，则返回 `False`。如果遍历完整个序列，则返回 `True`。

##### 2. 求两个序列的笛卡尔积

**题目：** 编写一个函数，求两个序列的笛卡尔积。

**答案：** 笛卡尔积是指将两个序列中的每个元素都与对方的每个元素组合，形成一个新序列。

```python
def cartesian_product(seq1, seq2):
    result = []
    for item1 in seq1:
        for item2 in seq2:
            result.append((item1, item2))
    return result
```

**解析：** 这个函数通过嵌套循环遍历两个序列中的元素，将每个元素的组合添加到结果序列中。最后返回结果序列。
