#### 算法编程题解析

##### 1. 判断二叉树是否对称

**题目：** 给定一个二叉树，判断它是否对称。

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def isSymmetric(root: TreeNode) -> bool:
    def check(left, right):
        if left is None and right is None:
            return True
        if left is None or right is None:
            return False
        if left.val != right.val:
            return False
        return check(left.left, right.right) and check(left.right, right.left)

    return check(root.left, root.right)
```

**答案解析：**
此题利用递归判断二叉树的左右子树是否对称。递归函数`check(left, right)`检查左右子树的值是否相等，并递归地检查左右子树的左右子节点和右子树的左右子节点。

##### 2. 最长公共前缀

**题目：** 编写一个函数来查找字符串数组中的最长公共前缀。

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s):
            if prefix[i] != s[i]:
                break
            i += 1
        prefix = prefix[:i]
    return prefix
```

**答案解析：**
此题通过遍历字符串数组，逐个比较字符串的前缀，更新最长公共前缀。每次比较时，从第一个字符串开始，逐个字符比较，直到找到不同的字符为止，此时更新最长公共前缀。
