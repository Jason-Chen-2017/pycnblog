#### 算法编程题库

##### 4. 请实现一个文本相似度比较算法。

**题目描述：** 给定两段文本，实现一个算法计算它们的相似度。

**算法：** 使用余弦相似度计算文本的相似度。

**代码示例：**

```python
from collections import Counter

def text_similarity(text1, text2):
    # 计算两段文本的词频向量
    vec1 = Counter(text1.split())
    vec2 = Counter(text2.split())
    
    # 计算余弦相似度
    dot_product = sum(vec1[word] * vec2[word] for word in vec1)
    mag1 = sum(v**2 for v in vec1.values())**0.5
    mag2 = sum(v**2 for v in vec2.values())**0.5
    
    similarity = dot_product / (mag1 * mag2)
    return similarity

text1 = "I love programming and learning new things."
text2 = "I enjoy coding and exploring technology."

print(text_similarity(text1, text2))
```

**解析：** 余弦相似度是一种衡量两向量夹角的余弦值的相似度度量方法。通过计算两向量的点积和各自模长的乘积，可以得到它们之间的相似度。

##### 5. 请实现一个基于TF-IDF的文本分类算法。

**题目描述：** 给定一组文档和预定义的类别，使用TF-IDF算法将文档分类到相应的类别。

**算法：** 使用TF-IDF算法计算文档特征向量，并将其与类别特征向量进行匹配，实现分类。

**代码示例：**

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB

# 文本数据
documents = ["I love programming", "I enjoy coding", "I prefer learning new things"]
labels = ["positive", "positive", "negative"]

# 训练集和测试集划分
X_train, X_test, y_train, y_test = train_test_split(documents, labels, test_size=0.2, random_state=42)

# 创建TF-IDF向量器
vectorizer = TfidfVectorizer()

# 转换为TF-IDF特征向量
X_train_tfidf = vectorizer.fit_transform(X_train)
X_test_tfidf = vectorizer.transform(X_test)

# 创建分类器
classifier = MultinomialNB()

# 训练分类器
classifier.fit(X_train_tfidf, y_train)

# 预测测试集
predictions = classifier.predict(X_test_tfidf)

# 评估分类效果
print(classifier.score(X_test_tfidf, y_test))
```

**解析：** TF-IDF（Term Frequency-Inverse Document Frequency）是一种衡量文本中词语重要性的指标。通过TF-IDF向量器，可以将文本数据转换为特征向量，然后使用分类器进行分类。
