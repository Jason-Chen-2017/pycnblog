## 算法编程题一：基于深度学习的图像分类

**题目描述：** 编写一个深度学习算法，实现图像分类功能。

**答案解析：** 下面是一个简单的卷积神经网络（CNN）实现图像分类的示例：

```python
import tensorflow as tf
from tensorflow.keras import datasets, layers, models

# 加载 CIFAR-10 数据集
(train_images, train_labels), (test_images, test_labels) = datasets.cifar10.load_data()

# 数据预处理
train_images, test_images = train_images / 255.0, test_images / 255.0

# 构建卷积神经网络
model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

# 添加全连接层
model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10))

# 编译模型
model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])

# 训练模型
model.fit(train_images, train_labels, epochs=10, batch_size=64)

# 评估模型
test_loss, test_acc = model.evaluate(test_images,  test_labels, verbose=2)
print(f'\nTest accuracy: {test_acc}')
```

**解析：** 该示例使用 TensorFlow 和 Keras 框架构建了一个简单的卷积神经网络，用于分类 CIFAR-10 数据集。通过训练和评估，我们可以得到模型的准确率。

## 算法编程题二：基于脑电信号的特征提取

**题目描述：** 编写一个算法，实现基于脑电信号的特征提取。

**答案解析：** 脑电信号的特征提取通常涉及信号处理和模式识别技术。以下是一个基于傅里叶变换进行特征提取的示例：

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.fft import fft

# 生成模拟脑电信号
fs = 1000  # 采样频率
t = np.linspace(0, 1, fs)
x = np.sin(2 * np.pi * 5 * t) + 0.5 * np.sin(2 * np.pi * 10 * t)

# 傅里叶变换
fft_x = fft(x)
freq = fs * np.linspace(0, 0.5, len(x) // 2)

# 绘制频谱图
plt.plot(freq, np.abs(fft_x[:len(freq) // 2]))
plt.title('Spectrum of the signal')
plt.xlabel('Frequency (Hz)')
plt.ylabel('|X(freq)|')
plt.grid()
plt.show()
```

**解析：** 该示例使用 NumPy 和 SciPy 库生成了一段模拟脑电信号，并使用傅里叶变换提取其频率特征。通过绘制频谱图，我们可以观察到信号中的不同频率成分。
