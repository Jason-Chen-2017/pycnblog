2. **边界正则性：** 子集在点集拓扑空间中的边界点构成的集合是闭集。

**解析：** 实数子集的正则性是指子集在拓扑空间中的形态，是否满足内部点和边界点的集合都是闭集。这是判断子集正则性的重要依据。

```python
def is_regular_subset(subset, space):
    # 判断内部点构成的集合是否为闭集
    internal_points = set()
    for point in subset:
        if point not in space:
            internal_points.add(point)
    if not space.is_closed(internal_points):

    # 判断边界点构成的集合是否为闭集
    boundary_points = set()
    for point in subset:
        if point not in space:
            boundary_points.add(point)
    if not space.is_closed(boundary_points):
        return False

    return True
```
