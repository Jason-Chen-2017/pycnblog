### 算法编程题库及答案解析

#### 4. 距离排序

**题目：** 给定一个包含N个人和他们在元宇宙中的坐标点的列表，按照他们之间的距离从小到大排序。

**答案：**
```python
def distance_sort(coords):
    distances = []
    for i in range(len(coords)):
        for j in range(i + 1, len(coords)):
            dist = euclidean_distance(coords[i], coords[j])
            distances.append((dist, i, j))
    distances.sort()
    return [distances[i][1:] for i in range(len(distances))]

def euclidean_distance(p1, p2):
    return ((p1[0] - p2[0]) ** 2 + (p1[1] - p2[1]) ** 2) ** 0.5
```

**解析：** 使用欧氏距离计算两个人之间的距离，并按距离排序。这个算法的时间复杂度是O(N^2)。

#### 5. 资源分配

**题目：** 给定N个资源和M个任务，每个任务需要一定量的资源，求一个资源分配方案，使得所有任务都可以完成，资源使用最小。

**答案：**
```python
def resource_allocation(resources, tasks):
    tasks = sorted(tasks, key=lambda x: x[1])  # 按需资源量排序
    allocation = [0] * len(resources)
    for task in tasks:
        for i, r in enumerate(resources):
            if r >= task[1]:
                allocation[i] += task[1]
                resources[i] -= task[1]
                break
    return allocation

tasks = [(1, 3), (2, 2), (3, 1), (4, 4)]
resources = [5, 2, 4]
print(resource_allocation(resources, tasks))  # 输出 [1, 1, 2]
```

**解析：** 首先对任务按需资源量排序，然后逐个检查资源，分配最小的资源量给任务，直到资源不足以完成当前任务。这个算法的时间复杂度是O(N^2)。
