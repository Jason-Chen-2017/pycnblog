#### 二、算法编程题库

##### 4. 编写一个算法，计算知识付费平台上用户的平均课程完成率。

**题目描述：** 给定一个用户课程完成情况的列表，编写一个函数计算平均课程完成率。

**输入：** `[0.2, 0.5, 0.8, 1.0, 0.3]`

**输出：** `0.5`

**答案：**

```python
def average_completion_rate(completions):
    total_completed = sum(completions)
    average_rate = total_completed / len(completions)
    return average_rate

completions = [0.2, 0.5, 0.8, 1.0, 0.3]
print(average_completion_rate(completions))
```

**解析：** 该算法通过计算完成课程的总时长与课程总数之比，得到平均完成率。

##### 5. 编写一个算法，根据用户兴趣推荐相似的课程。

**题目描述：** 给定一个用户兴趣标签列表和一个课程标签列表，编写一个函数计算两个列表的相似度，并返回相似度最高的课程。

**输入：** `user_interests = ["编程", "人工智能"]`，`course_tags = [["编程", "前端"], ["人工智能", "深度学习"]]`

**输出：** `1`（表示第一门课程与用户兴趣相似度最高）

**答案：**

```python
def similarity_score(user_interests, course_tags):
    max_score = 0
    max_course = None
    for course, tags in enumerate(course_tags):
        course_score = 0
        for tag in user_interests:
            if tag in tags:
                course_score += 1
        if course_score > max_score:
            max_score = course_score
            max_course = course
    return max_course

user_interests = ["编程", "人工智能"]
course_tags = [["编程", "前端"], ["人工智能", "深度学习"]]
print(similarity_score(user_interests, course_tags))
```

**解析：** 该算法通过计算用户兴趣标签与课程标签的交集，得到相似度分数，并返回相似度最高的课程。
