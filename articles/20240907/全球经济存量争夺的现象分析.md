#### 1. 货币兑换的最小交易次数

**题目描述：** 给定一个数组，表示不同货币的面值，以及一个目标金额。求使用这些货币兑换目标金额所需的最小交易次数。

**示例：** 输入：[1, 2, 5]，目标金额：10。输出：4。

**算法解析：** 可以使用动态规划的方法求解。定义一个二维数组 `dp[i][j]` 表示使用前 `i` 种货币兑换金额为 `j` 的最小交易次数。状态转移方程为：

```python
dp[i][j] = min(dp[i-1][j], dp[i-1][j-c[i]] + 1)
```

其中，`c[i]` 表示第 `i` 种货币的面值。

**Python 代码实现：**

```python
def minTransactionTimes(c, target):
    n = len(c)
    dp = [[0] * (target + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for j in range(target + 1):
            dp[i][j] = dp[i-1][j]
            if j >= c[i-1]:
                dp[i][j] = min(dp[i][j], dp[i][j-c[i-1]] + 1)
    return dp[n][target]
```

#### 2. 最小化物流成本

**题目描述：** 给定一个物流网络，包含若干城市和道路，以及每条道路的运输成本。求从起点城市到终点城市的最低物流成本。

**示例：** 输入：城市列表 `[A, B, C, D]`，道路列表 `[['A', 'B', 3], ['B', 'C', 1], ['C', 'D', 2]]`。输出：3。

**算法解析：** 可以使用 Dijkstra 算法求解。首先构建一个邻接表表示物流网络，然后使用 Dijkstra 算法逐步扩展，记录每个城市的最低物流成本。

**Python 代码实现：**

```python
from collections import defaultdict

def minLogisticsCost(roads, n, start, end):
    graph = defaultdict(list)
    for u, v, cost in roads:
        graph[u].append((v, cost))
        graph[v].append((u, cost))
    dist = [float('inf')] * n
    dist[start] = 0
    visited = [False] * n
    for _ in range(n):
        min_dist = float('inf')
        min_idx = -1
        for i in range(n):
            if not visited[i] and dist[i] < min_dist:
                min_dist = dist[i]
                min_idx = i
        visited[min_idx] = True
        for v, cost in graph[min_idx]:
            dist[v] = min(dist[v], dist[min_idx] + cost)
    return dist[end]
```
