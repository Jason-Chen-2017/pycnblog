**1. 给定一个整数数组，找出数组中的众数。**

**代码：**

```python
def majorityElement(nums):
    count = 0
    candidate = None
    for num in nums:
        if count == 0:
            candidate = num
        count += (1 if num == candidate else -1)
    return candidate
```

**2. 给定一个整数数组，找到最小的 k 个数。**

**代码：**

```python
def findKthLargest(nums, k):
    nums.sort(reverse=True)
    return nums[k-1]
```

**3. 给定一个字符串，找出最长的公共前缀。**

**代码：**

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s) and prefix[i] == s[i]:
            i += 1
        prefix = prefix[:i]
    return prefix
```
