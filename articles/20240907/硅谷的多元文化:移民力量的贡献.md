#### 算法编程题

**1. 给定一个字符串数组，返回其中不同的元素数量。**

**题目描述：** 编写一个函数 `uniqueElements(arr: List[str]) -> int`，返回字符串数组 `arr` 中不同的元素数量。

**示例：**
```plaintext
Input: ["apple", "banana", "apple", "orange"]
Output: 3
```

**答案：**
```python
def uniqueElements(arr):
    return len(set(arr))

# 示例调用
print(uniqueElements(["apple", "banana", "apple", "orange"]))  # 输出: 3
```

**2. 设计一个算法来查找一个字符串中的最长公共前缀。**

**题目描述：** 编写一个函数 `longestCommonPrefix(strs: List[str]) -> str`，返回给定字符串数组 `strs` 的最长公共前缀。

**示例：**
```plaintext
Input: ["flower", "flow", "flight"]
Output: "fl"
```

**答案：**
```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    
    prefix = strs[0]
    for s in strs[1:]:
        while not s.startswith(prefix):
            prefix = prefix[:-1]
            if not prefix:
                return ""
    return prefix

# 示例调用
print(longestCommonPrefix(["flower", "flow", "flight"]))  # 输出: "fl"
```
