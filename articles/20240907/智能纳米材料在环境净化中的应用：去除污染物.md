#### 题目1：基于纳米材料吸附性能的水质监测

**题目描述：** 编写一个程序，利用纳米材料的吸附性能监测水中的重金属离子。给定一个包含重金属离子浓度的数组，编写程序计算去除重金属离子后水质的改善程度。

**输入：** 重金属离子浓度数组。

**输出：** 水质改善程度（百分比）。

**示例：**
```
输入：[5, 10, 3, 8, 2]
输出：50%
```

**参考代码：**

```python
def water_quality_improvement(concentrations):
    # 假设纳米材料吸附能力为50%
    adsorption_rate = 0.5
    original_sum = sum(concentrations)
    after_adsorption = [concentration * adsorption_rate for concentration in concentrations]
    improved_sum = sum(after_adsorption)
    improvement = (original_sum - improved_sum) / original_sum * 100
    return f"{improvement:.2f}%"

concentrations = [5, 10, 3, 8, 2]
print(water_quality_improvement(concentrations))
```

    degradation_rate = catalyst_efficiency
    after_degradation = [concentration * (1 - degradation_rate) for concentration in concentrations]
    degradation_sum = sum(after_degradation)
