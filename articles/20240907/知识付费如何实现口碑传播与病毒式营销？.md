**3. 编程题：**

**题目3：** 编写一个函数，计算病毒式营销中用户增长的数量。

**答案3：** 假设每个用户平均能够带来 `k` 个新用户，初始用户数为 `n`，则用户增长的数量可以使用以下递推公式计算：

```python
def virus_growth(n, k, days):
    users = [n]
    for _ in range(days):
        new_users = users[-1] * k
        users.append(users[-1] + new_users)
    return users[-1]
```

**解析3：** 此函数模拟病毒式营销中用户数量的增长，其中 `n` 是初始用户数，`k` 是每个用户平均带来新用户的数量，`days` 是模拟的天数。返回的是第 `days` 天的用户总数。

**4. 编程题：**

**题目4：** 编写一个函数，计算病毒式营销活动的覆盖范围。

**答案4：** 覆盖范围可以通过计算在给定天数内用户增长的总数量来估算：

```python
def coverage(n, k, days):
    return sum(virus_growth(n, k, 1) for _ in range(days))
```

**解析4：** 此函数计算在给定天数内病毒式营销活动的覆盖范围，`n` 是初始用户数，`k` 是每个用户平均带来新用户的数量，`days` 是活动持续的天数。覆盖范围是每天增长的用户数量之和。
