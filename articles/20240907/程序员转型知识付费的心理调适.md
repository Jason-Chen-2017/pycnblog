#### 算法编程题库

**1. 如何实现一个函数，计算字符串中重复字符的个数？**

```python
def count_repeated_chars(s):
    # 你的代码实现
    pass
```

**答案：**

```python
def count_repeated_chars(s):
    char_count = {}
    for char in s:
        if char in char_count:
            char_count[char] += 1
        else:
            char_count[char] = 1
    return sum(char_count.values())
```

**2. 如何实现一个函数，找出字符串中第一个只出现一次的字符？**

```python
def find_first_unique_char(s):
    # 你的代码实现
    pass
```

**答案：**

```python
def find_first_unique_char(s):
    char_count = {}
    for char in s:
        if char in char_count:
            char_count[char] += 1
        else:
            char_count[char] = 1
    for char in s:
        if char_count[char] == 1:
            return char
    return None
```
