##### 5. 如何评估广告投放效果？

**题目：** 请简要介绍一种评估广告投放效果的方法，并解释其指标。

**答案：** 一种评估广告投放效果的方法是使用广告投放效果指标，如点击率（Click-Through Rate，CTR）、转化率（Conversion Rate，CVR）、投资回报率（Return on Ad Spend，ROAS）等。

**指标：**

1. 点击率（CTR）：广告被点击的次数与广告展示次数的比值，用于衡量广告的吸引力。
2. 转化率（CVR）：广告带来的实际转化次数与广告点击次数的比值，用于衡量广告的有效性。
3. 投资回报率（ROAS）：广告带来的收益与广告投放成本的比值，用于衡量广告的经济效益。

**举例：**

```python
def calculate_ctr(clicks, impressions):
    return clicks / impressions

def calculate_cvr(conversions, clicks):
    return conversions / clicks

def calculate_roas(revenue, cost):
    return revenue / cost
```
