#### 算法编程题

**1. 无重复字符的最长子串**

**题目描述：** 给定一个字符串 s ，请你找出其中不含重复字符的最长子串的长度。

**示例：** s = "abcabcbb"，返回 3 。无重复字符的最长子串是 "abc"，其长度为 3。

**答案：** 

```python
def lengthOfLongestSubstring(s: str) -> int:
    left = 0
    max_len = 0
    seen = set()
    for right, c in enumerate(s):
        while c in seen:
            seen.remove(s[left])
            left += 1
        seen.add(c)
        max_len = max(max_len, right - left + 1)
    return max_len
```

**解析：** 这个问题要求考生能够利用滑动窗口的方法，动态调整窗口大小，找到无重复字符的最长子串，展现出算法思维和编程能力。

**2. 有效的数独**

**题目描述：** 判断一个 9x9 的数独是否有效。只需要根据以下规则，验证已经填入的数字是否有效。

* 数字 1-9 在每一行只能出现一次。
* 数字 1-9 在每一列只能出现一次。
* 数字 1-9 在每个以粗实线分隔的 3x3 宫内只能出现一次。

**示例：** 输入：board = \[ \[5, 3, 0, 0, 7, 0, 0, 0, 0\], \[6, 0, 0, 1, 9, 5, 0, 0, 0\], \[0, 9, 8, 0, 0, 0, 0, 6, 0\], \[8, 0, 0, 0, 6, 0, 0, 0, 3\], \[4, 0, 0, 8, 0, 3, 0, 0, 1\], \[7, 0, 0, 0, 2, 0, 0, 0, 6\], \[0, 6, 0, 0, 0, 0, 2, 8, 0\], \[0, 0, 0, 4, 1, 9, 0, 0, 5\], \[0, 0, 0, 0, 8, 0, 0, 7, 9\] \]

输出：true

**答案：**

```python
def isValidSudoku(board):
    rows = [set() for _ in range(9)]
    cols = [set() for _ in range(9)]
    boxes = [set() for _ in range(9)]

    for i in range(9):
        for j in range(9):
            num = board[i][j]
            if num != '.':
                num = ord(num) - ord('0')

                if num in rows[i] or num in cols[j] or num in boxes[i//3*3 + j//3]:
                    return False

                rows[i].add(num)
                cols[j].add(num)
                boxes[i//3*3 + j//3].add(num)

    return True
```

**解析：** 这个问题要求考生能够通过遍历数独矩阵，利用哈希集合来判断是否存在重复数字，展现出对数组和哈希集合的熟练应用。
