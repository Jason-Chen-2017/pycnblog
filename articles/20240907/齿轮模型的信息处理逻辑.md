```python
def read_gear_data(file_path):
    with open(file_path, 'r') as file:
        lines = file.readlines()
    
    gears = []
    for line in lines:
        gear_data = line.strip().split(',')
        gear_id, gear_size, gear_tooth_count = map(int, gear_data)
        gears.append({'gear_id': gear_id, 'gear_size': gear_size, 'gear_tooth_count': gear_tooth_count})
    
    return gears
```

**解析：** 这个函数首先读取文件中的数据，然后通过格式转换将每一行的数据解析为字典，并存储在一个列表中。这个列表最终被返回作为输入处理的结果。
