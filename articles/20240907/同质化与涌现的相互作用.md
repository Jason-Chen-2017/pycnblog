**1. 编写一个算法，计算字符串的同质化程度。**

**题目：** 编写一个函数 `calculate_heterogeneity(input_string: str) -> float`，计算输入字符串的同质化程度。同质化程度越高，返回值越接近 1。

**答案：** 可以使用集合交集的方法计算同质化程度。

```python
def calculate_heterogeneity(input_string: str) -> float:
    words = input_string.split()
    word_set = set(words)
    intersection_size = len(word_set.intersection(set(words)))
    return 1 - (intersection_size / len(words))

input_string = "apple orange banana apple orange"
print(calculate_heterogeneity(input_string))
```

**2. 编写一个算法，根据用户互动数据生成产品或服务的推荐列表。**

**题目：** 编写一个函数 `generate_recommendation(user_interactions: List[List[str]]) -> List[str]`，根据用户互动数据生成产品或服务的推荐列表。推荐列表应包含用户可能感兴趣的产品或服务。

**答案：** 可以使用协同过滤算法生成推荐列表。

```python
from collections import defaultdict

def generate_recommendation(user_interactions: List[List[str]]) -> List[str]:
    user_rating = defaultdict(set)
    for user, interactions in enumerate(user_interactions):
        for interaction in interactions:
            user_rating[user].add(interaction)

    similarity_matrix = []
    for i in range(len(user_rating)):
        row = []
        for j in range(len(user_rating)):
            intersection_size = len(user_rating[i].intersection(user_rating[j]))
            row.append(intersection_size)
        similarity_matrix.append(row)

    recommendation_list = []
    for user, interactions in enumerate(user_rating):
        scores = []
        for i, interaction in enumerate(user_rating):
            if i == user:
                continue
            score = similarity_matrix[user][i]
            scores.append(score)
        scores /= sum(scores)
        recommendation_list.append(scores)

    return recommendation_list

user_interactions = [["apple", "orange", "banana"], ["orange", "apple", "kiwi"], ["apple", "orange"]]
print(generate_recommendation(user_interactions))
```
