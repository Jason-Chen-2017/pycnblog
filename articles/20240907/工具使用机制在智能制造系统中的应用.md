### 算法编程题库

### 1. 编写一个算法，用于计算智能制造系统中工具的使用效率。

**题目：** 请编写一个算法，用于计算智能制造系统中工具的使用效率。工具的使用效率定义为工具实际运行时间与工具计划运行时间的比值。

**输入：**
- 工具的计划运行时间（单位：分钟）
- 工具的实际运行时间（单位：分钟）

**输出：**
- 工具的使用效率（保留两位小数）

**示例：**
```plaintext
输入：
计划运行时间：300
实际运行时间：250

输出：
使用效率：0.83
```

**答案：**
```python
def calculate_usage_efficiency(planned_runtime, actual_runtime):
    efficiency = actual_runtime / planned_runtime
    return round(efficiency, 2)

# 示例
print(calculate_usage_efficiency(300, 250))
```

### 2. 编写一个算法，用于优化智能制造系统中的工具调度。

**题目：** 请编写一个算法，用于优化智能制造系统中的工具调度。假设系统中有多个工具需要同时执行任务，但每个工具的执行时间不同，要求算法能够使所有工具的总执行时间最小。

**输入：**
- 工具数组（每项为一个元组，包含工具的名称和执行时间）

**输出：**
- 优化后的工具调度顺序

**示例：**
```plaintext
输入：
[['ToolA', 10], ['ToolB', 20], ['ToolC', 15]]

输出：
[['ToolC', 15], ['ToolA', 10], ['ToolB', 20]]
```

**答案：**
```python
from heapq import heappush, heappop

def optimize_tool_scheduling(tools):
    # 按执行时间升序排序
    sorted_tools = sorted(tools, key=lambda x: x[1])
    # 使用最小堆进行调度优化
    min_heap = []
    for tool in sorted_tools:
        heappush(min_heap, tool)
    
    optimized_order = []
    while min_heap:
        optimized_order.append(heappop(min_heap))
    
    return optimized_order

# 示例
print(optimize_tool_scheduling([('ToolA', 10), ('ToolB', 20), ('ToolC', 15)]))
```

### 3. 编写一个算法，用于预测智能制造系统中工具的故障率。

**题目：** 请编写一个算法，用于预测智能制造系统中工具的故障率。故障率定义为工具在特定时间段内的故障次数与总运行时间的比值。

**输入：**
- 工具的故障记录数组（每项为一个元组，包含故障发生的时间和故障类型）
- 预测时间段

**输出：**
- 工具的故障率（保留两位小数）

**示例：**
```plaintext
输入：
[['2023-01-01 10:00', 'TypeA'], ['2023-01-02 12:00', 'TypeB'], ['2023-01-03 16:00', 'TypeA']]
预测时间段：'2023-01-01 00:00' 至 '2023-01-07 23:59'

输出：
故障率：0.22
```

**答案：**
```python
from datetime import datetime

def predict_fault_rate(fault_records, start_date, end_date):
    start = datetime.strptime(start_date, '%Y-%m-%d %H:%M')
    end = datetime.strptime(end_date, '%Y-%m-%d %H:%M')
    total_runtime = (end - start).total_seconds()
    
    fault_count = 0
    for record in fault_records:
        fault_time = datetime.strptime(record[0], '%Y-%m-%d %H:%M')
        if start <= fault_time <= end:
            fault_count += 1
    
    fault_rate = fault_count / total_runtime
    return round(fault_rate, 2)

# 示例
fault_records = [['2023-01-01 10:00', 'TypeA'], ['2023-01-02 12:00', 'TypeB'], ['2023-01-03 16:00', 'TypeA']]
start_date = '2023-01-01 00:00'
end_date = '2023-01-07 23:59'
print(predict_fault_rate(fault_records, start_date, end_date))
```
