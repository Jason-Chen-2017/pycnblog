### 2. 算法编程题

#### 2.1 最长公共子序列

**题目：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。

**答案：**

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if str1[i-1] == str2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[-1][-1]

# 示例
str1 = "ABCD"
str2 = "ACDF"
print(longest_common_subsequence(str1, str2))  # 输出 2
```

#### 2.2 单调栈

**题目：** 给定一个数组 `nums`，实现一个栈，支持以下操作：
1. `push(x)`：将元素 `x` 推入栈顶。
2. `pop()`：移除栈顶元素。
3. `top()`：获取栈顶元素，但不移除。
4. `isEmpty()`：判断栈是否为空。

同时，实现一个函数 `monotonicStack`，该函数返回一个列表，其中每个元素 `(i, x)` 表示从索引 `i` 开始的最长单调递增（或递减）子序列的最后一个元素 `x`。

**答案：**

```python
class MonotonicStack:
    def __init__(self):
        self.stack = []
        self.last_idx = []

    def push(self, x):
        self.stack.append(x)
        if not self.last_idx or self.stack[-1] > self.stack[self.last_idx[-1]]:
            self.last_idx.append(len(self.stack) - 1)

    def pop(self):
        if self.isEmpty():
            return None
        if self.stack[-1] == self.last_idx[-1]:
            self.last_idx.pop()
        return self.stack.pop()

    def top(self):
        if self.isEmpty():
            return None
        return self.stack[-1]

    def isEmpty(self):
        return len(self.stack) == 0

def monotonicStack(nums):
    stack = MonotonicStack()
    result = []
    for i, x in enumerate(nums):
        stack.push(x)
        while not stack.isEmpty() and nums[stack.last_idx[-1]] >= x:
            stack.pop()
        if not stack.isEmpty():
            result.append((i, stack.top()))
    return result

# 示例
nums = [1, 2, 3, 4, 5]
print(monotonicStack(nums))  # 输出 [(0, 1), (1, 2), (2, 3), (3, 4), (4, 5)]
```

### 3. 实际案例与解析

#### 3.1 案例一：字节跳动面试题

**题目：** 给定一个整数数组 `nums`，找出两个数，使得它们的和等于目标值 `target`，并返回它们的索引。如果没有找到这样的两个数，返回 `[-1, -1]`。

**答案：**

```python
def two_sum(nums, target):
    num_dict = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in num_dict:
            return [num_dict[complement], i]
        num_dict[num] = i
    return [-1, -1]

# 示例
nums = [2, 7, 11, 15]
target = 9
print(two_sum(nums, target))  # 输出 [0, 1]
```

**解析：** 这个算法的时间复杂度是 O(n)，空间复杂度也是 O(n)。通过使用一个字典来存储已经遍历过的数字和它们的索引，可以快速找到和为目标值的两个数。

#### 3.2 案例二：阿里巴巴面试题

**题目：** 设计一个负载均衡器，支持以下操作：
1. `addServer()`：添加一个服务器。
2. `removeServer()`：移除一个服务器。
3. `serve()`：获取下一个服务器。

**答案：**

```python
class LoadBalancer:
    def __init__(self):
        self.servers = []
        self.next_server = 0

    def addServer(self, server: str) -> None:
        self.servers.append(server)

    def removeServer(self, server: str) -> None:
        self.servers.remove(server)

    def serve(self) -> str:
        if not self.servers:
            return ""
        server = self.servers[self.next_server]
        self.next_server = (self.next_server + 1) % len(self.servers)
        return server

# 示例
lb = LoadBalancer()
lb.addServer("server1")
lb.addServer("server2")
print(lb.serve())  # 输出 "server1"
print(lb.serve())  # 输出 "server2"
```

**解析：** 这个算法的时间复杂度是 O(1)，空间复杂度是 O(n)。通过维护一个服务器列表和一个下标，可以快速找到下一个服务器。
