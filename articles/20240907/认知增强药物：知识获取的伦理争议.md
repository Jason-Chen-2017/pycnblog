**二、算法编程题库**

**1. 设计一个算法，用于评估一个人使用认知增强药物后的表现。**

**题目描述：**

编写一个函数 `evaluatePerformance`, 它接受一个表示认知能力测试分数的列表 `scores` 和一个表示使用认知增强药物前后的分数差异的列表 `differences`，并返回一个列表，其中每个元素表示一个人使用药物后的表现评分。评分由原始分数加上药物效果（正差异增加，负差异减少）计算得出。

**输入格式：**
- `scores`（列表）：一个人的连续认知能力测试分数列表，例如 `[85, 90, 88, 92]`。
- `differences`（列表）：药物使用前后的分数差异列表，例如 `[3, -2, 0, 5]`。

**输出格式：**
- 返回一个列表，其中每个元素是使用药物后的评分，例如 `[88, 88, 88, 97]`。

**示例：**

```python
scores = [85, 90, 88, 92]
differences = [3, -2, 0, 5]
evaluatePerformance(scores, differences) # 应返回 [88, 88, 88, 97]
```

**答案解析：**

```python
def evaluatePerformance(scores, differences):
    performance = []
    for score, difference in zip(scores, differences):
        performance_score = score + difference
        performance.append(performance_score)
    return performance

# 测试
scores = [85, 90, 88, 92]
differences = [3, -2, 0, 5]
print(evaluatePerformance(scores, differences)) # 应输出 [88, 88, 88, 97]
```

**2. 设计一个算法，用于分析一组人使用认知增强药物的效果。**

**题目描述：**

编写一个函数 `analyzeEffects`, 它接受一个表示不同人使用认知增强药物后表现的列表 `performances`，并返回一个字典，其中每个键是一个人的名字，值是他们药物使用后的平均表现评分。

**输入格式：**
- `performances`（列表）：一组人的药物使用后的表现评分列表，例如 `[88, 90, 87, 95]`。

**输出格式：**
- 返回一个字典，其中每个键是一个人的名字，值是他们药物使用后的平均表现评分。

**示例：**

```python
performances = [88, 90, 87, 95]
analyzeEffects(performances) # 应返回 {'Person1': 88.0, 'Person2': 90.0, 'Person3': 87.75, 'Person4': 95.0}
```

**答案解析：**

```python
def analyzeEffects(performances):
    effects = {}
    for i, performance in enumerate(performances):
        name = f'Person{i+1}'
        effects[name] = performance
    average_effects = {name: sum(values) / len(values) for name, values in effects.items()}
    return average_effects

# 测试
performances = [88, 90, 87, 95]
print(analyzeEffects(performances)) # 应输出 {'Person1': 88.0, 'Person2': 90.0, 'Person3': 87.75, 'Person4': 95.0}
```
