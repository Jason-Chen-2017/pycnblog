### 算法编程题库与源代码实例

#### 1. 利用K-means算法对员工进行聚类分析
**题目：** 编写一个K-means算法的代码，用于对员工数据进行分析和聚类。
**源代码：**

```python
import numpy as np
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

# 假设员工数据为二维数组，其中第一列表示工作时长，第二列表示工作满意度
data = np.array([[10, 0.8], [20, 0.7], [30, 0.6], [40, 0.9], [50, 0.5]])

# 使用KMeans算法进行聚类，假设要分成2个簇
kmeans = KMeans(n_clusters=2, random_state=0).fit(data)

# 输出聚类结果
print("聚类结果：", kmeans.labels_)

# 绘制聚类结果
plt.scatter(data[:, 0], data[:, 1], c=kmeans.labels_, cmap='viridis')
plt.show()
```

**解析：**
- **数据预处理：** 将员工数据转化为数组格式。
- **模型训练：** 使用KMeans算法进行聚类，并输出聚类结果。
- **可视化：** 使用散点图展示聚类结果。

#### 2. 利用线性回归预测员工流失
**题目：** 编写一个线性回归的代码，用于预测员工流失风险。
**源代码：**

```python
import numpy as np
from sklearn.linear_model import LinearRegression
import matplotlib.pyplot as plt

# 假设员工流失风险数据为二维数组，其中第一列表示工作时长，第二列表示工作满意度
data = np.array([[10, 0.8], [20, 0.7], [30, 0.6], [40, 0.9], [50, 0.5]])
labels = np.array([0, 0, 0, 1, 1])

# 使用线性回归算法进行预测
regressor = LinearRegression()
regressor.fit(data, labels)

# 输出预测结果
print("预测结果：", regressor.predict([[60, 0.3]])))

# 绘制回归结果
plt.scatter(data[:, 0], labels, color='red', label='员工流失风险')
plt.plot(data[:, 0], regressor.predict(data), color='blue', label='线性回归线')
plt.xlabel('工作时长')
plt.ylabel('工作满意度')
plt.legend()
plt.show()
```

**解析：**
- **数据预处理：** 将员工流失风险数据转化为数组格式。
- **模型训练：** 使用线性回归算法进行预测，并输出预测结果。
- **可视化：** 使用散点图和回归线展示预测结果。
