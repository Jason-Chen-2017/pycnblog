**解题思路：** 使用费曼提问法，将问题分解为以下几个步骤：

* **第一步：定义问题**：我们需要实现一个函数，计算整数数组中两个相邻元素的最大差值。
* **第二步：理解问题**：确保我们理解了函数的输入和输出。输入是一个整数数组 `arr`，输出是数组中两个相邻元素的最大差值。
* **第三步：编写代码**：根据理解，编写函数代码：

```go
func maxDiff(arr []int) int {
    maxDiff := arr[1] - arr[0]
    for i := 2; i < len(arr); i++ {
        diff := arr[i] - arr[i-1]
        if diff > maxDiff {
            maxDiff = diff
        }
    }
    return maxDiff
}
```
