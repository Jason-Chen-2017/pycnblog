### 4. 如何使用Weaver模型进行文本生成？

**题目：** 编写一个程序，使用Weaver模型生成一段指定主题的文本。

**答案：** 

```python
import requests

def generate_text(topic):
    url = "https://api.weavermodel.com/generate"
    payload = {
        "topic": topic
    }
    response = requests.post(url, json=payload)
    return response.json()["text"]

topic = "人工智能的未来发展"
text = generate_text(topic)
print(text)
```

**解析：** 该程序使用requests库向Weaver模型API发送请求，传递指定主题，并接收生成的文本。
