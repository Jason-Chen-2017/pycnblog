### 算法编程题库

#### 4. 给定一个用户行为数据集，编写一个函数，计算每个用户的行为数量。

**答案：** 

```python
def count_user_actions(user_actions):
    user_counts = {}
    for action in user_actions:
        user_counts[action['user_id']] = user_counts.get(action['user_id'], 0) + 1
    return user_counts

# 示例数据
user_actions = [
    {'user_id': 1, 'action': 'click'},
    {'user_id': 2, 'action': 'view'},
    {'user_id': 1, 'action': 'view'},
    {'user_id': 3, 'action': 'click'},
    {'user_id': 2, 'action': 'click'},
]

# 调用函数
print(count_user_actions(user_actions))
```

**解析：** 该函数通过遍历用户行为数据集，使用字典统计每个用户的行为数量，返回一个字典，其中键为用户ID，值为行为数量。

#### 5. 给定一个广告点击率数据集，编写一个函数，计算每个广告的平均点击率。

**答案：**

```python
def calculate_average_click_rate(ad_data):
    ad_click_rates = {}
    for ad in ad_data:
        clicks = ad['clicks']
        impressions = ad['impressions']
        if impressions > 0:
            ad_click_rates[ad['ad_id']] = clicks / impressions
    return ad_click_rates

# 示例数据
ad_data = [
    {'ad_id': 1, 'clicks': 50, 'impressions': 1000},
    {'ad_id': 2, 'clicks': 30, 'impressions': 800},
    {'ad_id': 3, 'clicks': 20, 'impressions': 500},
]

# 调用函数
print(calculate_average_click_rate(ad_data))
```

**解析：** 该函数遍历广告数据集，计算每个广告的平均点击率（点击次数/展示次数），并将结果存储在字典中，返回包含平均点击率的字典。
