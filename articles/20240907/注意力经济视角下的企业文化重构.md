### 1. 字符串匹配算法

**题目：** 实现一个字符串匹配算法，找出字符串s中的所有出现t的子串。

**答案：** 使用KMP算法：

```python
def KMP(s, t):
    def compute_lps(t):
        lps = [0] * len(t)
        length = 0
        i = 1
        while i < len(t):
            if t[i] == t[length]:
                length += 1
                lps[i] = length
                i += 1
            else:
                if length != 0:
                    length = lps[length - 1]
                else:
                    lps[i] = 0
                    i += 1
        return lps

    lps = compute_lps(t)
    i = j = 0
    while i < len(s):
        if t[j] == s[i]:
            i += 1
            j += 1
        if j == len(t):
            print("找到子串", s[i - j:i])
            j = lps[j - 1]
        elif i < len(s) and t[j] != s[i]:
            if j != 0:
                j = lps[j - 1]
            else:
                i += 1
```

### 2. 最长公共子序列

**题目：** 给定两个字符串，找出它们的最长公共子序列。

**答案：** 使用动态规划：

```python
def LCS(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```
