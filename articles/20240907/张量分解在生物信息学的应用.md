**1. 实现CP分解的Python代码**

**答案：** 

```python
import numpy as np

def cp_decomposition(A, rank):
    """
    CP分解
    A: 高维张量
    rank: 分解的秩
    """
    U, S, V = np.linalg.svd(A, full_matrices=False)
    U = U[:, :rank]
    S = S[:rank]
    V = V.T[:rank]
    return U * np.diag(S) * V

A = np.array([[1, 2], [3, 4], [5, 6]])
rank = 2
print(cp_decomposition(A, rank))
```

import numpy as np

def tucker_decomposition(A, r1, r2, r3):
    """
    Tuck分解
    A: 高维张量
    r1, r2, r3: 分解的秩
    """
    Q, R = np.linalg.qr(A)
    B = np.dot(Q.T, A)
    C = np.dot(B, Q)
    U, S, V = np.linalg.svd(C, full_matrices=False)
    U = U[:, :r3]
    S = S[:r3]
    V = V.T[:r3]
    return U * np.diag(S) * V

A = np.array([[1, 2], [3, 4], [5, 6]])
r1 = 2
r2 = 2
r3 = 2
print(tucker_decomposition(A, r1, r2, r3))

### 2. 张量分解的常用方法

**题目：** 张量分解有哪些常用方法？

**答案：** 常见的张量分解方法包括CP分解（标准正交分解）、DCC分解（双中心化标准正交分解）、PCA（主成分分析）、NMF（非负矩阵分解）和Tucker分解等。

**解析：**

1. **CP分解（Canonical Polyadic Decomposition）**：CP分解是一种将张量分解为几个低维张量的方法，通常用于降维和去噪。

2. **DCC分解（Double Centered Canonical Decomposition）**：DCC分解是CP分解的变体，通过双中心化操作来提高分解的稳定性。

3. **PCA（Principal Component Analysis）**：PCA是一种常用的降维方法，通过计算数据的协方差矩阵并找到最大的特征值对应的特征向量，来降低数据的维度。

4. **NMF（Non-Negative Matrix Factorization）**：NMF是一种无监督学习算法，通过将高维数据分解为几个非负低维矩阵的乘积，来提取数据中的非负特征。

5. **Tucker分解（Tucker Decomposition）**：Tucker分解是一种将高维数据分解为几个低维张量乘积的方法，通常用于降维和特征提取。

**示例代码：**
