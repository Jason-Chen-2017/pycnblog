### 3. AI 大模型在心理健康领域的算法编程题

#### **面试题 3：**
设计一个算法，用于分析一段文字，判断其中是否包含消极情绪。

**输入：** 一段文字字符串

**输出：** 消极情绪的判定结果（是/否）

**代码示例：**

```python
import re

def contains_negativity(text):
    # 定义包含消极情绪的词汇
    negative_words = ["伤心", "失望", "痛苦", "绝望", "愤怒"]

    # 使用正则表达式匹配文本中的消极情绪词汇
    for word in negative_words:
        if re.search(re.escape(word), text):
            return "是"
    return "否"

# 测试代码
text = "我今天去面试，感觉很不顺利，心情很低落。"
result = contains_negativity(text)
print(result)  # 输出："是"
```

**解析：**
该算法通过定义一个包含消极情绪词汇的列表，使用正则表达式在输入的文本中匹配这些词汇。如果找到任何匹配项，则判定文本中包含消极情绪，并返回“是”，否则返回“否”。这个简单的算法可以作为 AI 大模型在心理健康领域情感分析任务的一个基础实现。
