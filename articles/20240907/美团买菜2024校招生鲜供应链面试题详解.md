#### 4. 算法与编程

**题目：** 编写一个 Go 语言函数，计算两个整数之间的最大公约数（利用辗转相除法）。

**答案：**

```go
package main

import "fmt"

func gcd(a, b int) int {
    for b != 0 {
        a, b = b, a%b
    }
    return a
}

func main() {
    fmt.Println(gcd(24, 18)) // 输出 6
}
```

**解析：** 通过循环逐步减小两个数之间的差值，直到其中一个数为 0，此时另一个数即为最大公约数。
