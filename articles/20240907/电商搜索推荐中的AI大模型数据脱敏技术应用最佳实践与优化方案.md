### 算法编程题库

#### 1. 编写一个函数，实现将姓名中的敏感字符替换为星号。

**题目：** 编写一个函数 `maskName(name string) string`，将输入的姓名中的敏感字符替换为星号。

**答案：**

```python
def maskName(name):
    masked_name = ""
    for c in name:
        if c.isalpha():
            masked_name += "*"
        else:
            masked_name += c
    return masked_name
```

#### 2. 编写一个函数，实现将电话号码中的敏感部分用星号遮挡。

**题目：** 编写一个函数 `maskPhoneNumber(phone_number string) string`，将输入的电话号码中的敏感部分用星号遮挡。

**答案：**

```python
def maskPhoneNumber(phone_number):
    masked_phone_number = ""
    for i, c in enumerate(phone_number):
        if i >= 3 and i < len(phone_number) - 4:
            masked_phone_number += "*"
        else:
            masked_phone_number += c
    return masked_phone_number
```
