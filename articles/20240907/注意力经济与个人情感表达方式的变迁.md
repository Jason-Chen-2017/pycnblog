#### 三、算法编程题库

**1. 编写一个基于词典的情感分析函数。**

**题目描述：** 给定一个包含情感词典的文本，编写一个函数判断其情感极性。

**答案：** 

```python
# 情感词典
emotion_dict = {
    'happy': 1,
    'sad': -1,
    'angry': -1,
    'surprise': 1,
    'disgust': -1
}

def sentiment_analysis(text):
    text = text.lower()  # 将文本转换为小写
    words = text.split()  # 分割文本
    sentiment = 0
    for word in words:
        if word in emotion_dict:
            sentiment += emotion_dict[word]
    return 'positive' if sentiment > 0 else 'negative'

text = "I am very happy to see you!"
print(sentiment_analysis(text))  # 输出：positive
```

**2. 编写一个基于机器学习的情感分析函数。**

**题目描述：** 使用 sklearn 库训练一个情感分析模型，对给定的文本进行情感极性判断。

**答案：**

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import make_pipeline

# 训练数据
X_train = ["I love this movie!", "This is a terrible movie!", "I am feeling happy.", "I am feeling sad."]
y_train = ["positive", "negative", "positive", "negative"]

# 创建 pipeline
pipeline = make_pipeline(TfidfVectorizer(), MultinomialNB())

# 训练模型
pipeline.fit(X_train, y_train)

# 测试数据
text = "I hate this movie!"
predicted_sentiment = pipeline.predict([text])[0]

print(predicted_sentiment)  # 输出：negative
```
