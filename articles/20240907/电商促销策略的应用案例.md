#### 1. 订单优惠计算

**题目：** 设计一个订单优惠计算系统，实现根据订单金额和优惠规则计算最终支付金额的功能。

**答案：**

```go
package main

import (
    "fmt"
)

func calculateDiscount(totalAmount float64, discounts []DiscountRule) float64 {
    discountedAmount := totalAmount
    for _, discount := range discounts {
        if discountedAmount >= discount.MinimumAmount {
            discountedAmount -= discount.Amount
        }
    }
    return discountedAmount
}

type DiscountRule struct {
    Amount        float64
    MinimumAmount float64
}

func main() {
    totalAmount := 1500.0
    discounts := []DiscountRule{
        {Amount: 100.0, MinimumAmount: 1000.0},
        {Amount: 200.0, MinimumAmount: 2000.0},
    }

    finalAmount := calculateDiscount(totalAmount, discounts)
    fmt.Printf("最终支付金额: %.2f\n", finalAmount)
}
```

**解析：** 该程序定义了一个 `calculateDiscount` 函数，用于根据订单金额和优惠规则计算最终支付金额。通过遍历优惠规则，对符合条件的订单金额进行折扣计算。

#### 2. 限时抢购

**题目：** 设计一个限时抢购活动系统，当库存不足时，限制用户参与。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

func main() {
    stock := 100
    for {
        if stock > 0 {
            fmt.Println("库存还剩：", stock)
            stock--
        } else {
            fmt.Println("库存不足，活动结束。")
            break
        }
        time.Sleep(1 * time.Second)
    }
}
```

**解析：** 该程序模拟了一个简单的限时抢购活动。每次循环都检查库存是否大于 0，如果大于 0，则打印库存并减一；如果库存不足，则结束活动。

#### 3. 购物积分

**题目：** 设计一个购物积分系统，用户购物后根据消费金额获得相应积分，并实现积分兑换功能。

**答案：**

```go
package main

import (
    "fmt"
)

var points int

func purchase(amount float64) {
    points += int(amount * 1.5)
    fmt.Printf("购买成功，获得积分：%d\n", points)
}

func exchange(points int) {
    if points >= 1000 {
        points -= 1000
        fmt.Println("兑换成功，剩余积分：", points)
    } else {
        fmt.Println("积分不足，无法兑换。")
    }
}

func main() {
    purchase(500.0)
    purchase(300.0)
    exchange(points)
}
```

**解析：** 该程序模拟了一个简单的购物积分系统。用户每次购物都会获得相应积分，积分可以用来兑换商品。

#### 4. 优惠券使用

**题目：** 设计一个优惠券使用系统，用户可以在购物时使用优惠券，并计算最终的支付金额。

**答案：**

```go
package main

import (
    "fmt"
)

func applyCoupon(total float64, couponCode string) float64 {
    discount := 0.0
    switch couponCode {
    case "ABC123":
        discount = 0.1
    case "XYZ456":
        discount = 0.2
    }
    return total - (total * discount)
}

func main() {
    total := 1000.0
    couponCode := "ABC123"
    finalAmount := applyCoupon(total, couponCode)
    fmt.Printf("使用优惠券后，最终支付金额: %.2f\n", finalAmount)
}
```

**解析：** 该程序定义了一个 `applyCoupon` 函数，用于根据优惠券代码计算折扣金额。用户可以在购物时使用优惠券，并计算最终的支付金额。

#### 6. 限时折扣

**题目：** 设计一个限时折扣促销策略，在特定时间段内，对指定商品进行折扣销售。

**答案：**

```go
package main

import (
    "fmt"
    "time"
)

func checkDiscount(itemID int, startTime, endTime time.Time) float64 {
    currentTime := time.Now()
    if currentTime.After(startTime) && currentTime.Before(endTime) {
        return 0.1 // 10% 折扣
    }
    return 0.0
}

func main() {
    itemID := 1
    startTime := time.Now().Add(-24 * time.Hour)
    endTime := time.Now().Add(24 * time.Hour)

    discount := checkDiscount(itemID, startTime, endTime)
    fmt.Printf("折扣率: %.2f\n", discount)
}
```

**解析：** 该程序定义了一个简单的限时折扣促销策略。在指定时间段内，对指定商品进行 10% 的折扣销售。

#### 7. 购物车优惠

**题目：** 设计一个购物车优惠系统，根据购物车中的商品数量和金额，为用户提供相应的优惠。

**答案：**

```go
package main

import (
    "fmt"
)

func calculateCartDiscount(cartItems []CartItem) float64 {
    totalAmount := 0.0
    for _, item := range cartItems {
        totalAmount += item.Price * item.Quantity
    }

    if totalAmount >= 500 {
        return 0.1 * totalAmount
    }
    return 0.0
}

type CartItem struct {
    ProductID int
    Price     float64
    Quantity  int
}

func main() {
    cartItems := []CartItem{
        {ProductID: 1, Price: 100.0, Quantity: 2},
        {ProductID: 2, Price: 200.0, Quantity: 1},
    }

    discount := calculateCartDiscount(cartItems)
    fmt.Printf("购物车优惠金额: %.2f\n", discount)
}
```

**解析：** 该程序定义了一个购物车优惠系统。根据购物车中的商品数量和金额，为用户提供相应的优惠。

#### 8. 积分兑换商品

**题目：** 设计一个积分兑换商品系统，用户可以使用积分兑换指定商品。

**答案：**

```go
package main

import (
    "fmt"
)

var products = map[int]Product{
    1: {Name: "手机", Points: 1000},
    2: {Name: "耳机", Points: 500},
}

type Product struct {
    Name  string
    Points int
}

func exchangePoints(points int, productID int) bool {
    if product, exists := products[productID]; exists && points >= product.Points {
        products[productID].Points = 0
        return true
    }
    return false
}

func main() {
    points := 1000
    productID := 1

    if exchangePoints(points, productID) {
        fmt.Println("兑换成功")
    } else {
        fmt.Println("兑换失败")
    }
}
```

**解析：** 该程序定义了一个积分兑换商品系统。用户可以使用积分兑换指定商品，兑换成功后，该商品在系统中标记为已兑换。

**题目：** 设计一个折扣叠加系统，当订单满足多个折扣条件时，可以为用户提供叠加优惠。

**答案：**

```go
package main

import (
    "fmt"
)

func applyDiscounts(total float64, discounts []Discount) float64 {
    for _, discount := range discounts {
        if total >= discount.MinimumAmount {
            total -= discount.Amount
        }
    }
    return total
}

type Discount struct {
    Amount        float64
    MinimumAmount float64
}

func main() {
    total := 1500.0
    discounts := []Discount{
        {Amount: 100.0, MinimumAmount: 1000.0},
        {Amount: 200.0, MinimumAmount: 2000.0},
    }

    finalAmount := applyDiscounts(total, discounts)
    fmt.Printf("叠加优惠后，最终支付金额: %.2f\n", finalAmount)
}
```

**解析：** 该程序定义了一个折扣叠加系统。当订单满足多个折扣条件时，可以为用户提供叠加优惠。
