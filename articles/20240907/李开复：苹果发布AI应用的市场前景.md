### 2. 数据预处理

**题目：** 编写一个Python函数，对给定数据集进行预处理，包括数据归一化、缺失值填充等。

**解析：** 数据预处理是数据分析的另一个重要环节。以下是一个简单的数据预处理示例：

```python
import numpy as np

def preprocess_data(data):
    # 数据归一化
    normalized_data = (data - np.mean(data)) / np.std(data)
    
    # 缺失值填充
    missing_values = np.isnan(data)
    for i in range(missing_values.shape[0]):
        for j in range(missing_values.shape[1]):
            if missing_values[i, j]:
                normalized_data[i, j] = np.mean(normalized_data[:, j])
    
    return normalized_data

data = np.array([[1, 2, 3], [4, np.nan, 5], [6, 7, 8]])
preprocessed_data = preprocess_data(data)
print(preprocessed_data)  # 输出：[[ 0.         0.40824829  0.81649658]
                          #          [ 0.81649658  0.          0.81649658]
                          #          [ 0.         0.40824829  0.81649658]]
```
