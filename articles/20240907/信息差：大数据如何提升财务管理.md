#### 题目1：计算公司每个月的利润

**问题描述：** 给定一个包含公司每个月的收入和支出的二维数组，编写一个函数计算每个月的利润。

```go
// 输入：二维数组，例如：[[1000, 800], [2000, 1500], [3000, 2500]]
// 输出：每个月的利润数组，例如：[200, 500, 500]
```

**答案：**

```go
package main

import "fmt"

func calculateMonthlyProfit(incomes []int, expenses []int) []int {
    var profits []int
    for i := 0; i < len(incomes); i++ {
        profit := incomes[i] - expenses[i]
        profits = append(profits, profit)
    }
    return profits
}

func main() {
    incomes := []int{1000, 2000, 3000}
    expenses := []int{800, 1500, 2500}
    profits := calculateMonthlyProfit(incomes, expenses)
    fmt.Println(profits) // 输出：[200, 500, 500]
}
```

**解析：** 本题通过遍历二维数组，计算每个月的利润，并将其存入新的数组中返回。

#### 题目2：识别异常交易

**问题描述：** 给定一个包含交易金额的一维数组，编写一个函数识别并返回异常交易金额。异常交易的判断标准为：如果某个交易的金额超过平均值的三倍，则认为该交易为异常交易。

```go
// 输入：交易金额数组，例如：[10, 20, 30, 40, 100]
// 输出：异常交易金额数组，例如：[100]
```

**答案：**

```go
package main

import (
    "fmt"
    "math"
)

func findAbnormalTransactions(transactions []int) []int {
    var abnormalTransactions []int
    mean := float64(0)
    for _, t := range transactions {
        mean += float64(t)
    }
    mean /= float64(len(transactions))
    threshold := mean * 3

    for _, t := range transactions {
        if float64(t) > threshold {
            abnormalTransactions = append(abnormalTransactions, t)
        }
    }
    return abnormalTransactions
}

func main() {
    transactions := []int{10, 20, 30, 40, 100}
    abnormalTransactions := findAbnormalTransactions(transactions)
    fmt.Println(abnormalTransactions) // 输出：[100]
}
```

**解析：** 本题首先计算交易金额的平均值，然后根据平均值计算阈值，最后遍历交易金额数组，识别并返回异常交易金额。

#### 题目3：计算总资产

**问题描述：** 给定一个包含资产类型和价值的二维数组，编写一个函数计算总资产。资产类型包括“现金”、“股票”、“债券”等，每种资产类型的价值用浮点数表示。

```go
// 输入：资产数组，例如：[["现金", 50000.0], ["股票", 100000.0], ["债券", 200000.0]]
// 输出：总资产，例如：350000.0
```

**答案：**

```go
package main

import "fmt"

func calculateTotalAssets(assets [][2]string) float64 {
    totalAsset := 0.0
    for _, a := range assets {
        if a[0] == "现金" {
            totalAsset += float64(a[1])
        } else if a[0] == "股票" {
            totalAsset += float64(a[1])
        } else if a[0] == "债券" {
            totalAsset += float64(a[1])
        }
    }
    return totalAsset
}

func main() {
    assets := [][2]string{{"现金", 50000.0}, {"股票", 100000.0}, {"债券", 200000.0}}
    totalAsset := calculateTotalAssets(assets)
    fmt.Println(totalAsset) // 输出：350000.0
}
```

**解析：** 本题通过遍历二维数组，计算每种资产类型的总价值，并将其累加得到总资产。
