#### 1. 如何评估促销活动对销售额的影响？

**题目：** 如何评估电商促销活动对销售额的实际影响？

**答案：** 要评估促销活动对销售额的影响，可以采取以下步骤：

1. **数据收集：** 收集促销前、促销中和促销后的销售额数据。
2. **控制变量法：** 保证促销活动期间，其他影响销售额的因素（如广告投放、新品发布等）保持不变。
3. **A/B 测试：** 对不同用户群体进行 A/B 测试，比较促销活动前后的销售额变化。
4. **统计分析：** 采用统计方法（如 t 检验、方差分析等）分析促销活动对销售额的影响。

**代码示例：**

```python
import pandas as pd
from scipy import stats

# 假设 df 是包含销售额数据的 DataFrame，其中 'sales' 列表示销售额，'is_promotion' 列表示是否为促销期间
promotion_sales = df[df['is_promotion'] == True]['sales']
non_promotion_sales = df[df['is_promotion'] == False]['sales']

# 检验促销活动对销售额的 t 检验
t_stat, p_value = stats.ttest_1samp(promotion_sales, non_promotion_sales.mean())

# 输出结果
print(f"T-statistic: {t_stat}, P-value: {p_value}")
```
