#### 面试题

#### 1. 科技股投资的指标分析

**题目：** 请解释市盈率、市净率和每股收益（EPS）等指标在科技股投资中的意义。

**答案：** 

- **市盈率（P/E Ratio）：** 反映股票价格与每股收益的比值，用于评估股票是否被高估或低估。高市盈率可能表示股票价格高估，低市盈率可能表示股票价格低估。
- **市净率（P/B Ratio）：** 反映股票价格与每股净资产的比值，用于评估股票的估值是否合理。高市净率可能表示股票价格高估，低市净率可能表示股票价格低估。
- **每股收益（EPS）：** 每股收益是公司净利润与发行在外普通股总数的比值，用于衡量公司盈利能力。

**解析：** 理解这些指标对于分析科技股的估值和盈利能力至关重要。

#### 算法编程题

#### 4. 股票买卖的最佳时机

**题目：** 给定一个整数数组 `prices`，其中 `prices[i]` 是第 `i` 天每股第 `i` 只股票的价格。设计一个算法来找出第 `i` 天你所能获取的最大利润。你可以无限次地买卖股票，但不得同时持有多只股票（你必须在再次购买前卖出当前的股票）。

**示例：**

```
输入：prices = [7,1,5,3,6,4]
输出：5
解释：在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
     注意利润不能是 6-1 = 5，因为在这个情况下，没有交易发生，所以利润为 0 。
```

**答案：**

```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_price = float('inf')
        max_profit = 0
        for price in prices:
            if price < min_price:
                min_price = price
            else:
                profit = price - min_price
                max_profit = max(max_profit, profit)
        return max_profit
```

**解析：** 通过记录最低购买价格和计算利润差值，可以找到最大利润。

#### 5. 投资组合收益风险分析

**题目：** 假设您有一份包含 `n` 只股票的投资组合，每只股票的价格和收益风险如下表：

| 股票 | 价格 | 收益风险 |
|------|------|----------|
| A    | 100  | 0.1      |
| B    | 150  | 0.15     |
| C    | 200  | 0.2      |

设计一个算法计算以下指标：

1. 组合的预期收益
2. 组合的收益风险
3. 组合的最坏情况损失（即最小收益）

**答案：**

```python
import numpy as np

prices = [100, 150, 200]
risks = [0.1, 0.15, 0.2]

# 1. 预期收益
expected_return = sum(p * r for p, r in zip(prices, risks))

# 2. 收益风险
return_variance = sum(r**2 for r in risks)
risk_return = expected_return / np.sqrt(return_variance)
return_std_dev = np.sqrt(return_variance)

# 3. 最坏情况损失
worst_case_loss = min(p * (1 - r) for p, r in zip(prices, risks))

print("预期收益:", expected_return)
print("收益风险:", return_std_dev)
print("最坏情况损失:", worst_case_loss)
```

**解析：** 通过计算每只股票的预期收益、收益风险和最坏情况损失，可以全面评估投资组合的风险和收益。
