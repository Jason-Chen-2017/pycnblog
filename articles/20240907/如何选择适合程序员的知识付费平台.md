**题目 2：** 腾讯面试题：请实现一个函数，找出数组中重复的数字。

**答案：** 可以使用哈希表或排序等方法找出重复数字。

**源代码实例（哈希表）：**

```go
func findRepeatNumber(nums []int) int {
    m := make(map[int]int)
    for _, num := range nums {
        if _, exists := m[num]; exists {
            return num
        }
        m[num]++
    }
    return -1
}
```

**题目 3：** 字节跳动面试题：请实现一个函数，判断一个字符串是否是回文串。

**答案：** 可以使用双指针法判断字符串是否回文。

**源代码实例：**

```go
func isPalindrome(s string) bool {
    n := len(s)
    left, right := 0, n-1
    for left < right {
        if s[left] != s[right] {
            return false
        }
        left++
        right--
    }
    return true
}
```
