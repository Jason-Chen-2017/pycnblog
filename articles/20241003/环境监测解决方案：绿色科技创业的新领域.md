### 4. 数学模型和公式 & 详细讲解 & 举例说明

#### 环境质量指数（EQA）计算模型

环境质量指数（EQA）是衡量环境质量的重要指标，其计算公式如下：

$$
EQA = \frac{1}{N} \sum_{i=1}^{N} \frac{Q_i}{Q_{max}}
$$

其中，$N$为评价指标数量，$Q_i$为第$i$个评价指标的得分，$Q_{max}$为第$i$个评价指标的最大得分。

#### 举例说明

假设某地区环境监测包括三个指标：空气质量指数（AQI）、水质指数（WI）和噪声指数（NI）。各指标的得分和最大得分如下表：

| 指标    | 得分   | 最大得分 |
| ------- | ------ | -------- |
| AQI     | 50     | 100      |
| WI      | 70     | 100      |
| NI      | 30     | 100      |

根据公式计算EQA：

$$
EQA = \frac{1}{3} \times \left( \frac{50}{100} + \frac{70}{100} + \frac{30}{100} \right) = 0.6
$$

EQA值为0.6，表明该地区环境质量处于中等水平。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 数据预处理
def preprocess_data(data):
    # 清洗数据
    data = data.dropna()
    # 去噪
    data = data.rolling(window=3).mean()
    return data

# 数据分析
def analyze_data(data):
    # 计算环境质量指数
    eqa = np.mean(data)
    # 预警
    if eqa > 0.7:
        print("环境质量预警：环境质量较差，需加强治理。")
    else:
        print("环境质量良好。")
    return eqa

# 可视化展示
def visualize_data(data):
    plt.plot(data)
    plt.xlabel("时间")
    plt.ylabel("指标得分")
    plt.title("环境质量指数变化趋势")
    plt.show()

# 主函数
def main():
    # 读取数据
    data = pd.read_csv("environment_data.csv")
    # 数据预处理
    data = preprocess_data(data)
    # 数据分析
    eqa = analyze_data(data)
    # 可视化展示
    visualize_data(data)

if __name__ == "__main__":
    main()
```

4. 可视化展示：将环境质量指数变化趋势以图表形式展示。
