## 3. 核心算法原理 & 具体操作步骤

### 常见排序算法
- 冒泡排序：$$\text{for i in range(len(arr)): for j in range(len(arr) - i - 1): if arr[j] > arr[j + 1]: arr[j], arr[j + 1] = arr[j + 1], arr[j]}$$
- 快速排序：$$\text{def quicksort(arr): if len(arr) <= 1: return arr pivot = arr[len(arr) // 2] left = [x for x in arr if x < pivot] middle = [x for x in arr if x == pivot] right = [x for x in arr if x > pivot] return quicksort(left) + middle + quicksort(right)}$$

### 常见查找算法
- 二分查找：$$\text{def binary_search(arr, target): low = 0 high = len(arr) - 1 while low <= high: mid = (low + high) // 2 if arr[mid] == target: return mid elif arr[mid] < target: low = mid + 1 else: high = mid - 1 return -1}$$

### 动态规划
- 最长公共子序列：$$\text{def longest_common_subsequence(X, Y): m = len(X) n = len(Y) dp = [[0] * (n + 1) for _ in range(m + 1)] for i in range(1, m + 1): for j in range(1, n + 1): if X[i - 1] == Y[j - 1]: dp[i][j] = dp[i - 1][j - 1] + 1 else: dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]) return dp[m][n]}$$

### 5.2 源代码详细实现和代码解读

以下是一个简单的线性回归模型的实现：

```python
import numpy as np

def linear_regression(X, y):
    """
    线性回归模型实现
    """
    # 添加常数项
    X = np.concatenate((np.ones((X.shape[0], 1)), X), axis=1)
    
    # 计算参数
    theta = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)
    
    return theta

# 生成模拟数据
np.random.seed(0)
X = np.random.rand(100, 1)
y = 2 + 3 * X + np.random.randn(100, 1)

# 训练模型
theta = linear_regression(X, y)

# 输出参数
print("参数：", theta)
```

### 5.3 代码解读与分析

- `linear_regression`函数实现线性回归模型，输入为自变量矩阵`X`和因变量向量`y`。
- 通过`np.concatenate`函数添加常数项，使得`X`矩阵变为 $(1, n)$ 的形式。
- 使用`np.linalg.inv`函数计算逆矩阵，`X.T.dot(X)`表示自变量矩阵的转置与自身相乘。
- 使用`dot`函数计算逆矩阵与自变量矩阵的转置的乘积，得到参数`theta`。
