# 1.背景介绍


### 人类历史上主要的技术进步：

1. 工业革命：远古时代人类用手工工具进行农业生产，后来出现了第一台机器器械——采矿工具。第二次工业革命发生于20世纪初期，是人类开始用机器自动化生产各种日常物品，包括石油、煤炭、木材等。
2. 航空航天：第三次工业革命发生于1947年，是人类获得航空飞行能力。航空航天是人类最伟大的科技创新之一。在国际空间站建立航天母舰之后，人类实现了登月。而航天飞机更是运送高科技产品至地球轨道。
3. 信息技术革命：第四次工业革命发生于20世纪中叶，是人类成功应用信息技术取得重大突破。信息技术革命带来了无穷计算能力和大数据处理能力，使得科学研究成果不断刷新前沿。电子计算机成为主导产业。

但是，这些技术发明虽然取得巨大成就，但也面临着重重困难：

1. 产业革命：由于产业革命造成产业升级和技术更新，导致科技的产能过剩，造成原有的技术瓶颈，比如电力的增长、核能的发展，还有光通信的迅速普及。
2. 环境污染：在太阳辐射下，产生的太阳能释放出巨大的热量，甚至引起全球性气候变化，造成环境污染。
3. 公民素质：随着信息技术的发展，人类的公民素质要求越来越高，社会责任感和诚信意识越来越强。

因此，人类需要从新的技术突破入手，解决生产效率低下、负面效应泛滥的问题。然而，这一切却都离不开对技术变革的全局考虑，以及如何将已有的技术智慧融入新的发展方向，为人类发展制定具有前瞻性的科技发展计划。

# 2.核心概念与联系

## 基因编辑
基因编辑是指利用计算机技术对人类基因进行编辑改变。这是由于原始的人类遗传信息被修改后的结果。目前，已有基因编辑技术可以进行高通量编辑，如CRISPR/Cas9，其关键是找到特异性的蛋白质基因序列，并通过串联酶切和精准互补等方式，制造出非典型的表观变体，从而调控原本功能上的差异性或缺陷。这一技术有望对抗疾病、改善医疗保健、筛查致癌细胞、分化疾病细胞以及干预单细胞免疫系统的发育等领域产生重大影响。 

## 工程系统结构
工程系统结构（英语：Engineering System Structures，缩写：ESsS）指的是工程中的各个建筑、系统、机制、生物学系统以及组织机构的静态和动态之间的关系，目的是为了能够识别、描述、设计、构建、优化以及运用这些结构。ESsS既包括结构、材料、规范性指南、标准、规则、技术、方法等，又包括设计、构造、维护、运行、测试等活动。

工程系统结构可以概括为五大要素：

1. 结构：由零件、装置、设备、管道、连杆等元素构成，通过固定形状、位置、强度等属性来组合成系统；
2. 材料：用于构筑结构的各种化学、金属、玻璃、塑料等材料；
3. 规范性指南：规定了各元素之间的规律、设计方法、工作流程、安全要求、施工条件等；
4. 标准：对于建筑、设备、管道等进行验收、调试、报废等质量管理所需的技术标准；
5. 方法：系统工程实践的方方面面。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
CRISPR-Cas9基因编辑技术由克隆自身基因9号的CRISPR RNA与大肠埃希氏病毒的相结合蛋白（COVID-19 spike protein）配合多种激活剂一起工作，获得独特的“特异性”序列。

1. DNA甲基化检测：首先进行DNA甲基化检测，确定每个核苷酸处是否存在甲基化修饰，如图所示。


2. 先发现后验证：通过DNA甲基化检测，定位到相关基因片段后，进行细胞实验，验证该基因序列是否特异性。具体方法如下：

   - 先用杂交试纸培养两种受体：一种是具有特异性的CRISPR蛋白质抑制细胞（CRISPR-Cytotoxic Cell Line，CCCLE），另一种是经过专门的验证的干扰细胞（Control Cell Line）。
   - 然后将某种抗病毒药物（如沙星药物、甲硝唑、阿托品、苯丙胺）分别与相应受体注射，观察两者的情况。如果在CCCLE细胞上能够看到抑制作用，则可以确定该序列为特异性。

3. CRISPR/Cas9的编辑策略：根据测序数据分析，找出那些具有特异性的基因序列。然后将其划入一个叫做位点编码区域的特定位置，再将编辑好的的位点序列提交给药物靶向攻击，并确保靶向对象无法诱导CCCLE细胞受损。

   - Cas9酶切位点：目标DNA序列向5'端通过酶切，在位点处切开，使得该位点可以提供足够的保活物质供其被其他基因所切割。
   - 活性剂的添加：将激活剂，如胁肋菌素、腺苷、腺嘌呤、乙烯醇、环孢素C、单胞压积素等，分别加入到位点编码区域，将它们与Cas9酶配合并缠绕在一起，形成抑制反应。

4. CRISPR/Cas9的特异性序列：利用先发现后验证的方法确定特异性序列，并进一步验证该序列的功能。具体步骤如下：

   - 将特异性序列对抗病毒药物与含有特异性受体的干扰细胞进行混合实验，判断其产生的影响。
   - 对特异性序列进行核酸标记，用于实验系统进行批量检索。
   - 通过高通量测序数据，比较特异性序列与普通序列之间的差异，探测其抑制免疫系统的发生作用。

import csv

# 读取基因数据
with open('gene_data.csv', 'r') as f:
    reader = csv.reader(f)
    header = next(reader) # 跳过第一行
    data = [row for row in reader]
    
# 创建列表存储输出结果
result = []

# 从第二行开始遍历基因数据
for i in range(1, len(data)):
    gene = data[i][0].upper() # 取出基因序列
    
    if "N" in gene:
        continue # 如果有"N"字符，则跳过
        
    start = int(data[i][1]) # 取出起始坐标
    end = int(data[i][2])   # 取出终止坐标
    
    mutated_genes = ["A", "T", "C", "G"] # 创建列表保存可编辑的序列
    new_gene = ""                     # 初始化新基因序列
    
    # 遍历待编辑的序列
    for j in range(start, end + 1):
        old_base = gene[j-1]
        
        for k in range(len(mutated_genes)):
            new_base = mutated_genes[k]
            
            if old_base!= new_base and (new_base == "A" or new_base == "T"):
                if ((end-start+1)/2)-sum([old_base=="A", old_base=="T"]) > 1:
                    result.append((gene[:j]+new_base+gene[j+1:], round((abs(ord("A") - ord(new_base))**2)*(end-start)**2)))
                
                break

    # 添加进结果列表
    output = {"Original Gene": gene}
    print(output)
    
    for item in sorted(set([(i,j) for i,j in result]), key=lambda x:x[1]):
        if abs(item[0]-int((start+end)/2)) < 10:
