# 1.背景介绍


## 一、什么是图神经网络？
图神经网络（Graph Neural Networks）是一种用于处理图数据的非线性模型。一般来说，图数据由节点（node）和边（edge）组成，节点代表实体或事件，边代表节点之间的关系或相互作用。图神经网络可以对节点及其之间的连接进行特征学习、分类、聚类、预测等任务。

早在2017年，谷歌团队提出了图卷积网络（Graph Convolutional Network）并命名为“GraphConvNets”。该网络结构能够学习到节点间的空间关联信息、全局特征、以及局部路径上的特征。然而，随着该领域越来越火热，新的研究不断涌现出来，比如更复杂的网络结构、多模态融合、更高效的训练方法、模型压缩等。近几年来，由于图数据本身复杂多变的特点，越来越多的研究人员将注意力放在如何处理图数据的非线性特性上。因此，本文将从以下几个方面系统阐述图神经网络的相关理论知识、最新进展、应用和挑战。

## 二、图神经网络模型简介
### （1）图表示法
图是由节点(Node)和边(Edge)组成的集合，其中每个节点与其他节点通过一条或多条边相连，构成一个有向图或者无向图。如下图所示，图中包括三个节点A、B、C，以及四条边AB、BC、AC、BA。图的邻接矩阵表示形式是一个$n\times n$的矩阵，每行对应于一个节点，第i个元素的值为1代表节点i和其他节点有连接；否则，值为0。如此构成的邻接矩阵是一个方阵，当n较大时，其空间复杂度较高。另一种图表示方式是采用邻接表的方式，即每一个节点都存储它直接相连的节点列表。这种表示方式具有更好的空间复杂度，因为每个节点仅需要存储它的邻居节点的指针，而不用存储完整的邻接矩阵。


### （2）图神经网络基本框架
基于图神经网络的深度学习主要分为两步：特征学习和分类。

**1.特征学习**：特征学习旨在从图数据中自动地学习到节点的特征表示。假设图中每个节点的特征都可以用向量f表示，那么特征学习就是要找到一种映射函数h：$\mathcal{H}\rightarrow \mathbb{R}^d$，使得对于任意节点i和j，h(f_i)≈h(f_j)，其中≈表示差异较小。

例如，图卷积网络（Graph Convolutional Networks）的设计思路就是先对每个节点的邻居做卷积操作，再做池化操作得到每个节点的新特征表示。

**2.分类**：分类则是根据 learned features 来完成。由于特征的维度可能很高，难免存在维度过低的问题。为了解决这个问题，通常会选择一些降维的方法，如PCA、SVD等，但这些方法往往会丢失掉重要的全局信息。因此，需要进一步考虑是否可以通过引入额外的全局信息来增强图神经网络的表达能力。常用的全局信息包括节点之间的结构、图的拓扑结构以及全局性质，比如图中的所有节点处于同一层级或属于同一类别。图注意力机制（Graph Attention Networks，GAT）便是考虑全局信息的一个有效方案。

综上，图神经网络的基本框架为：Learned Features -> Classification。
