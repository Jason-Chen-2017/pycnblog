# 1.背景介绍


在现代社会，人们已经习惯了数字化的信息生产方式，如电子邮件、网络聊天、微博、微信等，也积累了海量数据，这种大数据的分析能力可以让我们洞察社会运行的本质，掌握我们的经济命运和社会生活规律。但是，这些信息的价值观念却无法捕捉到商业模式的发展机会和创新潜力，形成了新的商业领域——数字经济（Digital Economy）。这个新的经济领域涵盖了人工智能、机器学习、大数据、云计算、物联网、区块链等诸多前沿技术。2017年全球科技领域的全景图揭示了数字经济对经济和社会的深远影响。图中显示，消费者正在成为最大的市场，全球55%的消费者每年花费超过40亿美元，而创造收入的能力则由数字经济带来的分散式商业模式来填补。另外，数字经济还将推动智能制造、智慧城市、人工智能交通等重大产业的发展。因此，对于任何一个技术领域来说，数字经济都将成为重要的驱动力。

然而，如何利用数字经济的发展，为社会解决各种复杂且艰难的社会问题，并通过商业模式的创新提升经济效益，是一个更加复杂的课题。对于创新型企业而言，数字技术赋予其巨大的商业价值；但对于一些传统行业部门来说，他们正在被数字技术所左右，这些部门可能正缺乏新的创新能力。如何利用数字经济提供的巨额利润实现商业模式的创新，成为构建新型国家、新兴行业、新商业模式的关键之一。

在过去的几十年里，人类一直在面临着两大历史性的技术变革：从工业革命到互联网时代，再到移动互联网和虚拟现实时代。在这三次技术革命中，人类的技术都取得了显著进步，但都没有彻底颠覆旧有的生产关系。在一次又一次的技术革新中，人类通过探索未知领域，创造出了全新的生产方式。如今，全世界的技术创新正在以全新的姿态影响着社会，而这种影响力正逐渐扩大、深化，且越来越为人们所认同。

2017年1月，在美国加州大学洛杉矶分校，MIT等国际顶级研究机构举办了第七届“人类技术变革博览会”，这是继五年前“科技、管理、产品”三个领域展开合作之后，又一次以“科学、科技、科技”为主题的研讨会。在这一期间，来自世界各地的著名科学家，工程师和领导者分享了当前最热门的技术前沿以及未来技术革命对人类社会的影响。会议上发布的“Global Technology Radar”报告直指“数字经济”的新机遇，充满激情的嘉宾们抛砖引玉，鼓吹“增强现实（Augmented Reality）”、“智能机器人”、“虚拟现实”等热门技术，争论不休。无论是谁站台，都会受到高超的解读、批评和意见，让听众有机会结识新朋友、讨论热点话题、看清问题所在。

2017年的会议上，特别邀请到来自德国柏林大学、英国剑桥大学、印度南加尔理工大学、俄罗斯莫斯科大学等诸多顶尖科学院校的嘉宾，一起探讨数字经济对人类社会的影响。与会嘉宾的分享内容涵盖了机器学习、人工智能、云计算、区块链等多个方面的最新研究进展。与会者们认为，数字经济正在带来一种全新的生产方式，它为个人和组织提供了能够影响世界的工具，并逐渐改变了生产关系和工作方式。此外，数字经济将通过全新的商业模式推动产业升级、提升经济效益，并为企业创造新的发展空间。此外，数字经济的发展也将推动人工智能、智慧城市和智能交通等重大产业的发展。而对于传统行业部门来说，虽然数字技术已经使得一些行业部门的生产效率得到提高，但它们仍面临着许多挑战，比如缺乏有效的创新机制和资源投入。

综上所述，数字经济的崛起或许会带来新的生产力，但同时也将带来新的商业模式、社会角色甚至政治局面。当这种新的商业模式被低估、被忽视时，或许就会成为社会的巨大灾难。为保护数字经济不被忽视、落井下石，科学家、工程师、企业家需要发掘新型的生产关系，用新经济的观点看待经济现象，以创新和发明的方式解决社会问题。“人类技术变革简史：从虚拟现实的崛起到增强现实的应用”通过对数字经济的宏观、微观、商业模式三个方面详尽介绍，向读者阐释了数字经济的产生、功能、发展和未来发展方向。
