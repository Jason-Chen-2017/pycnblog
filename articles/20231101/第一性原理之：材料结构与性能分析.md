# 1.背景介绍


## 1.1 什么是材料结构？
材料结构，又称“材料组成”，指的是由材料元素构成的整体形状、结构及其在温度、压力、流量等多种环境条件下的行为特征。它是对材料存在的性质、运动特性、结构特性及其变化规律的描述。
## 1.2 为什么需要分析材料结构？
- 可以更好地理解材料运动与结构的关系。
- 通过材料结构的研究可以探索材料的制造过程和应用机理。
- 了解材料的性能优劣以及转化工艺、加工工艺的设计原则。
- 进而可以提高产品和服务的品质、降低成本。
## 1.3 材料结构的分类
按照材料的分类，通常可分为以下几类：
### （一）金属结构
包括绝缘子孔、超导体层等；
### （二）晶体结构
包括称谓晶体（Crystal），临界晶体（Austenite），水晶体（Silicon），石英晶体（Gold）等；
### （三）半导体结构
包括硅、锗、钨、铜等；
### （四）微结构
包括液晶，蛋白质层，磁性物质等；
### （五）结构合成
指具有不同结构功能的不同器件、固体混合物或金属缺陷共同作用的结构，如耦合器、聚合剂、贮藏剂等。
## 1.4 如何进行材料结构分析
材料结构分析包括计算法（也可以用实验方法）、图像法、统计方法等。计算法主要采用模拟退火算法、随机蒙特卡洛方法等计算手段，图像法则可依靠显微观察和实验数据采集的方式获得结构信息，统计方法则可利用数据的分析工具对材料结构有更全面的认识。
# 2.核心概念与联系
## 2.1 胞的基本结构
- **晶胞**：晶格结构中各个原子的位置关系是最简单的，而晶胞是其中的一个具有多种周期性的正方形胞，具有整体晶格的周期性。
- **涡旋胞**：涡旋胞也被称为石墨烯，是一个晶格的反演图，其原子通常排列为一个平面，组成两个方向的旋涡，通过旋涡的交替变化来表现出晶格的周期性。
- **超胞**是指由多个无序小颗粒构成的，包含完整晶格单元、沟槽等零维点的多种结构，具备极高的能量密度。在晶胞与超胞之间存在着一个重要的区别：超胞能够实现精确的宏观结构刻画能力。
## 2.2 基本原理及原子构成
**原子结构**: 原子包括电子和质子两类，一般情况下，电子占据费米面积较大的部分，而质子占据费米面积较小的部分。为了使材料的结构可以顺利发挥作用，需要保证原子之间的距离足够近，这样才能够构建起较复杂的晶体、微颗粒的结构。因此，原子结构一定程度上决定了材料的结构与性能。
**基本原理**: 主要有两种基本原理：即带隙假设与约化假设。
- **带隙假设**：认为材料结构由无限多个层次化的层堆组成，每一层都有固定的厚度（带隙）。在每一层内，分散在各种原子上，经过无限长的链状排列，形成无限长的层间纽结。由于带隙的存在，使得每个原子都会紧邻其他原子。在每一层的末端，还会有一系列的键环（钛基），通过键环能使相邻层的原子互联。总之，如果没有带隙假设，那么原子之间就会存在着极大空隙，破坏整个结构。
- **约化假设**：认为所有的物质都可以被各种不同构型的原子所构成，而且这些原子之间会相互配合共同组成各自不同的结构。就像核反应堆一样，根据反应物质的不同，可以形成不同的材料结构。例如，当热力驱动时，反应物质会被氢原子吸收，然后形成具有有机林、石棉、海绵等不同的结构。因此，这也是基于约化假设进行材料结构分析的关键所在。
## 2.3 计算原理
**金属结构和晶体结构的性质差异**
在金属结构与晶体结构上，可以发现一些结构性质的差异。由于金属结构和晶体结构的区别，在计算方面存在一些差异。比如，在金属结构中，能量较弱，几乎不依赖于温度变化。但在晶体结构中，能量随温度的升高而增加，受到温度影响更大。另外，在金属结构中，能够用分数表示原子数，因为每种元素都有无限的分量。但是，在晶体结构中，只能用整数表示原子数，因为原子的个数总是有限的。所以，对于金属来说，采用计算比较简单，只需要求取费米面积，而晶体结构则要求考虑更多的因素。
**如何计算金属结构中的元件数目**
首先，我们要确定研究材料的金属性质。接下来，我们需要把研究材料的各个元件坐标转换到直角坐标系中，这样才能方便计算。
### 方法一：基于基本的三维坐标计算法
这一方法是从各个周期性点的坐标及周期性距离，利用最小图像原理（Minimum Image Principle，简称MICP）将每一种情况的位置关系重新投影回二维平面上。具体步骤如下：
1. 计算每个周期性点的位置坐标。
2. 把所有周期性点分成两组，每一组只有两种情况下，即用圆心和两个边分别进行置换，得到新的位置关系。
3. 选取所有周期性点中，哪些可以产生最小的 MICP 误差。
4. 在这些点周围找到一个距离最近的一个周期性点，如果有一个距离更短，则记录这个距离作为最小的 MICP 误差。
5. 对每一个周期性点，重复以上步骤，直到没有更多可以替换的点出现。
6. 重复第 4 和 5 步，直到所有周期性点都已处理完成。
7. 汇总所有可能的情况，并统计其中满足 MICP 误差条件的情况，得到不同元件的数量，再乘上一个倍数，便得到最终结果。
### 方法二：基于化学反应网络理论的计算法
这种方法主要用于计算单原子团的分子结构。其基本思路是采用化学反应网络理论中的约化原理。具体步骤如下：
1. 用标准化学键找到一个原子团的平均分子排列。
2. 将整个反应网络分成两部分，一部分为研究目标的部分，另一部分为固定住的部分。
3. 从固定住的部分中删除那些在反应过程中不会发生变化的原子，把其余的原子归类到几个层次上去。
4. 每一层都由若干个原子组成，通过图论的算法计算出任意一个原子团的排列。
5. 根据化学反应网络的路径理论，利用最短路径的方法找出一个原子团的最短路径长度。
6. 当研究目标的原子团的排列符合某种分布的时候，可以通过计数获得相应的元件数目。
7. 如果想了解每一层的内部结构，还可以在第一步的基础上，找到不同层次之间的键长、键错、键隙等参数，进而对各个原子进行更精细的分析。
## 2.4 性能指标及评判准则
在材料结构分析过程中，常用的性能指标有晶体的温度对比和压强的对比。其他的性能指标还有结构的倾向性和局部阻止率。评判准则则以重整化学反应机理为例，通过该机理预测材料的性能。
