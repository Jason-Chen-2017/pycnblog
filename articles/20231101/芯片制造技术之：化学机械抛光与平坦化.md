# 1.背景介绍


## 概述
芯片的制造技术是人类历史上最大规模科技投入的一项。从诞生到今日，芯片制造已经经历了多个阶段，比如晶圆厂、导电工艺、焊接、摩擦、组装、测试、封装、安装、调试等等，而近几年来由于通用集成电路（IC）的出现和应用广泛，芯片制造技术也越来越火热。
除了上述的环节，芯片制造还涉及到很多其他问题，如电源、驱动电路、封装材料、组装工具等。在各个环节中都存在着各种不同的工艺和技术，这使得芯片制造技术研究、开发变得复杂而繁琐。其中，电子芯片的制造技术尤其复杂，主要包括电压冲击敏感、光刻胶纳米损伤、反向物性、集成测试、制程设计、集成电路封装、冲刷、改装、漏电、布局等方面。
无论是电子芯片的制造，还是普通的半导体芯片制造，都需要对某些关键性能或特性进行优化，提高产品的可靠性和价格效益。在20世纪70年代末和80年代初，科学家们发现许多不同的化学物质能够影响电子设备的性能。其中，硫酸盐是一种重要的组分，尤其适用于电子器件的制造。它能够改变电子管道内的水素杂质，产生一种特殊的电流，这称为“电压冲击”。而硅铁矿石（Silicon Manganese Oxide, SiMOX）则能够引起三极管（J-fet）的失配，并产生额外的电压，进而影响电子器件的性能。
另外，光刻胶纳米结构对电子器件的保护效果更好，因为光刻胶纳米结构中含有大量的氧化剂，可以抵消硫酸盐带来的电压冲击影响。然而，为了制造出高效的光刻胶纳米结构，需要考虑多种技术细节，如电子制备、光刻过程的控制、晶粒的选取、温度调控等等。而这些技术又往往依赖于许多工程实践的知识和经验，这就使得普通科研工作者很难掌握整个流程。因此，如何建立一个整体的流程，有效地利用现有的知识和方法，成为芯片制造领域的重要课题。
## 化学机械抛光与平坦化
化学机械抛光（Chemical Mechanical Polishing, CMP），又称化学加工精细技术，是一种采用化学方法对物料进行精心加工的方法。其基本过程是把金属、杂质或者其他不良材料（非金属损坏的表面）放在铝锭上，在铝锭表面的喷涂物层上加入一定的溶液、盐和碱等物质，然后将铝锭捏合，即使化学变化导致表面微缩，也能保持铝锭的结构不变，这样就能够达到物料的精确清洗。
抛光过程中，在铝锭表面的喷涂物层上的物质称为“净化剂”，它的作用是使得铝锭表面发生自由裂开，从而产生光泽，实现材料的整洁、色泽和抹平，使其具有良好的物理性状。同时，化学机械抛光还会发生渗透、腐蚀和烧结，使铝锭内部产生大量的气孔、空穴等坏死物质，如果能够对这些坏死物质进行去除、修复和化学处理，就能够降低电路元件的损耗。
### 原理
化学机械抛光的原理简单来说就是将物料放入一张铝锭上，通过喷涂作业使铝锭表面微缩、发生自由裂开，产生一定数量的化学净化剂，然后通过对气孔、空穴等物质的清除与修复，使铝锭表面发生腐蚀、干燥、烧结，在一定条件下，就可以得到清洁而完整的物料，同时也能起到抹平、刮削、光泽的作用。
图1 晶圆厂生产的电压表面稳定剂，具有“看起来像锭状，实际却是‘野蛮’”的特点，而且其表面上长满了细小的核孔、野蛮肉眼无法察觉的细胞、磨损残余等坏死物质，是一种比较容易出现的坏死物质。
### 操作步骤
1.收集待抛光物料（如电压表面稳定剂）；

2.选择或订购符合要求的抛光设备（如晶圆厂、研磨设备）；

3.用标准化的喷涂作业标准把物料表面打理匀后，倒入装有抛光液的灯笼，利用低温快速打光（约10秒左右）；

4.重复步骤3，直至获得足够数量的化学净化剂；

5.将物料移至表面平滑、洁净且无任何杂质的状态，等待10~15分钟即可，这一步是为了使化学净化剂与被抛物质之间形成互补作用；

6.将物料放在表面碱性液、真空、水分严重缺乏的环境中静置10~15分钟，待物料呈浅黄色状态时再将物料放在手术台上做切割、挖掘、清洗操作（取决于剪切处有无腐蚀），直到物料呈红色、干燥且有所缩紧为止；

7.将物料重新装入硅胶垫、镀锌板或者光剂胶垫内，并打开设备盖，等待10分钟以上的固化时间，使其完全化学处理；

8.将物料放入塑料袋，确认物料没有受损或失焊，然后转移至专业塑料模具或加工工业企业进行后续加工；

9.完成产品的制造，并根据客户的要求进行适当调整。

### 模型与计算
CMP的主要优点是简单方便、可靠性高，但其同样也存在一些局限性，比如抛光物料的特性、特定工艺条件下的性能等。因此，对电子设备的制造及其他工程领域的电路制造，应该考虑一些更专业的技术和方法，如预制芯片、反相工艺、表面工程、结构设计、设计优化、创新工艺等。

模型可以用来描述化学机械抛光技术的整个过程，如图2所示。抛光液由漆状颗粒聚拢成颗粒状的化学净化剂，这些颗粒在被放入要处理的物料表面时发生自由裂开，产生吸附力，将其中含有的化学元素浓缩到铝锭表面。随着自由裂开的持续时间增加，更多的化学元素就被释放出来，铝锭表面呈现光泽、平滑、清爽的效果，从而达到了抛光物料的清洗目的。

计算公式如下：

$$V_{out}=\frac{I^{2}\cdot R_L}{d}=IR_{\text {leak }}+I_{\text {diff }}+\mu\cdot I_{\text {rad }}$$

其中$V_{out}$代表输出电压，$I$代表输入电流，$R_{\text {leak}}$代表漏电阻，$I_{\text {diff }}$代表电子-介质之间的差异电流，$\mu$代表玻尔兹曼辛亥常数，$I_{\text {rad }}$代表远离光源的射线电流。

我们假设以电压表面稳定剂（如图1所示）作为抛光物料，光电耦合时光照射面积为A，晶圆厂所在位置为P，晶圆厂设计功率为W，$V_{\text {sat }}$代表表面稳定剂的饱和电压，$t_{\text {leak }}$代表漏电时间，则可以用以下公式计算退耦电压：

$$U_{th}=AP_{\parallel }\cos \theta-AP_{\perp }-\frac{\mu WA}{\epsilon } V_{\text {sat }} e^{-t_{\text {leak }}}$$

其中$\theta$代表光束与晶圆管轴线夹角，$\epsilon$代表介质厚度，$\theta=0^\circ $时退耦电压最小。$e^{-t_{\text {leak }}}$代表电流逐渐减弱的效应。
