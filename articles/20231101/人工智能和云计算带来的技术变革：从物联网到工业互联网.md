# 1.背景介绍


物联网（IoT）和云计算已经成为当今社会中最热门的两个领域之一，其带来的影响无疑给我们的生活、工作和产业带来了新的机遇和挑战。但是，为了适应这种变化，作为技术专家，我们需要对人工智能和云计算这些新兴技术有更深入的理解，并加以应用。因此，本文将对人工智能和云计算技术的发展进行一个综述性的介绍。


人工智能和云计算的定义及历史由来
　　“人工智能”（Artificial Intelligence，AI）的概念最初出现于1956年的麻省理工学院（MIT）的一个研究小组，他们提出了三层人工智能模型：机器人、人类专家系统和人工神经网络。20世纪60年代末期，深蓝计算机公司首席执行官罗伯特·麦卡锡（Russell McKenzie）把人工智能定义为“智能机器与自主决策”。但是，随着人工智能的发展，人们逐渐认识到，这种定义过于狭隘，缺乏通用性，而且还不够准确。2015年，美国国家科学技术委员会（NSTC）发布了“通用人工智能”（U.S.-wide Artificial Intelligence）的白皮书。其中提出了人工智能定义：“以计算机或机器的方式实现高度智能化的功能和行为。”如此，“人工智能”一词才得以被广泛接受。

　　1966年，约翰·列文顿（John Lennon）提出“信息处理器”概念，将电脑视为一种信息处理器，具有能够接收、存储和转化输入信息、输出结果的能力。但是，它并没有涉及到任何形式的智能，它只是一种可以运行程序的工具。

　　1980年，马修·阿克莱姆（Malcolm Ackerman）提出“计算机程序”这一概念，指的是一段按照特定规则编写的用来完成特定任务的代码。不过，这个概念还是太抽象，不易于讨论具体的问题。

　　1987年，斯坦福大学计算机科学教授史密斯（Stephen Smilgart）和他的学生一起提出“人工智能”概念，并认为其需要包括三个方面：（1）感知（perception）——让计算机理解外界世界；（2）推理（inference）——让计算机从知识库中学习；（3）决策（decision-making）——让计算机做出合理而明智的决策。

　　1987年，雅达利（Jadair）、西蒙弗德（Simon Ford）和卡内基梅隆大学的盖茨（Grace Hopper）三位教授共同提出“人工智能”的概念。它们认为，在未来，人工智能将由两大支柱构成：一是图灵机（Turing machine），二是基于模式识别的学习（pattern recognition）。

　　1987年至今，人工智能已经逐渐形成了一个完整的体系，其中包含的领域非常丰富，比如语音识别、图像识别、自然语言处理、机器翻译等等。

　　20世纪90年代后期，随着科技飞速发展，人工智能领域也开始发生重大变革。近几年，人工智能的发展主要依赖于三个关键因素：数据、算法和计算资源的增长。据统计显示，2013年全球每年产生的数据量是前十的财富之一，占全球GDP的1/3左右。由于人工智能技术的应用越来越广泛，所以数据量的增长使得人工智能技术的研发速度远远快于经济规模。同时，计算能力的提升也促进了人工智能技术的发展。


　　云计算的定义与发展过程
　　“云计算”（Cloud computing）是指利用网络基础设施的资源、服务、平台、应用和商业模式，通过网络进行信息、计算和数据的传输。传统的计算机系统主要集中在单个中心数据中心，这就造成了数据中心的高昂费用、不稳定、安全隐患等问题。因此，云计算作为一种新型的计算模型，意味着将部分服务器、存储设备、计算资源等分布在不同地区，通过网络相互连接，实现资源共享、服务交付、按需扩缩容等功能。

　　2006年，亚马逊在中国举办的AWS(Amazon Web Services)正式发布，宣布它将提供公共云服务，即云计算服务。公共云是指供所有人使用、免费提供的云计算服务，用户只需要注册就可以获得相应的资源。

　　2008年，谷歌宣布开放自己的云计算服务Google Cloud Platform(GCP)，即谷歌开发的全球范围的云计算服务，用户可以通过该平台轻松创建、访问和管理虚拟服务器、存储设备、数据库以及相关应用软件。

　　2013年，微软宣布与亚马逊、谷歌和其他云服务提供商合作，将Azure云平台整合到其内部业务，打造全面覆盖混合云、本地数据中心和公有云的Azure Stack。

　　2014年，IBM宣布正在将其企业级私有云部署到多个全球数据中心，以扩展IBM在全球范围内的云计算服务。

　　2015年，Redhat宣布将Red Hat OpenShift部署到IBM、Microsoft Azure和Amazon Web Services上的公共云平台上，用于支持容器应用的自动化部署、运行和管理。

　　2015年，Docker宣布推出云服务Docker Hub，以便云厂商可以在线管理用户的公有云和私有云的镜像仓库。

　　2016年，微软宣布推出Microsoft Azure Cosmos DB云数据库服务，可在任意规模的云中提供高可用性、高吞吐量、快速查询的文档型数据库。

　　2016年，Facebook宣布在全球各地建立数千个节点，以提供安全、可靠的云计算服务。

　　2017年，OpenStack社区宣布推出兼容Kubernetes的项目OpenShift Origin V3.9，可帮助公有云、私有云及本地数据中心构建和部署应用程序。
