# 1.背景介绍


## 1.1 计算的定义及其历史

计算(computing)，英文翻译为“计算”，是指利用计算机系统进行数学、逻辑判断、文字处理、数据存储、图形制作等多种计算活动的能力。为了计算而产生的设备包括算盘、扫瞄仪、电子秤、计算器、图像处理器、随机数发生器、编程语言、数据仓库、数据库系统等。在计算出现之前，人们依赖手工操作数学、逻辑判断和文字记录的过程进行计算活动。随着计算技术的发展，新型的计算装置如微型计算机、个人电脑、手机、电视机等已成为生活中的重要组成部分。

计算之所以能够实现各种计算任务的同时，也带来了新的问题。随着计算技术的发展，出现了一些很大的科技革命性的创新。计算技术的革命性的进步不仅带动了经济领域，而且也对社会各个领域产生了巨大的影响。其中最具代表性的就是人工智能领域。

### 1.1.1 人工智能简介

人工智能（Artificial Intelligence，AI），又称机器智能、通用智能或通才智能，它是由外界环境输入的数据经过学习与自我修正得到的一种智能。它是通过模拟人的大脑神经网络来完成复杂的决策和认知功能。人工智能的研究目标是开发具有智能的机器，让它们像人类一样思维、行动、学习和创造。因此，它是科学研究、工程应用和社会服务的重要领域。

现代人工智能的主要研究内容分为三大类：搜索、推理和学习。

- 搜索：人工智能可以解决在宇宙中寻找特定信息的问题，例如无人驾驶汽车、机器人导航、机器翻译、图像识别等。搜索可以包括全局搜索、局部搜索、规划搜索、知识库查询等。
- 推理：人工智能可以从事文本理解、图像分析、语音理解、知识表示等领域，包括逻辑推理、计算推理、概率推理、强化学习、监督学习、非监督学习等。
- 学习：人工智能可以从事知识学习、规则学习、模式识别、强化学习、自适应控制等领域。学习的目的是使得智能体能够自动地学习、改善和优化它的行为，促进其智能性。

### 1.1.2 人工智能的挑战

1956年，美国物理学家艾伦·图灵提出了一个著名的组合猜想——图灵机，其机械构造简单、结构清晰、可编程、能解决很多复杂计算问题，其关键特征就是具有任意存储、任意转移和交替执行的状态转移函数。图灵机被认为是计算机的雏形，虽然它没有维纳斯·冯·诺依曼结构，但它启发了计算机的发明。

1970年代末期，在阿姆斯特丹大学的图灵奖颁奖典礼上，图灵、阿兰·图灵、考夫曼、艾伦·麦卡洛克和约翰·麦卡洛尔等人相继获得图灵奖。这些人对计算机科学的发展起到了重要的作用，其影响远远超过了当时的计算机产业。

然而，人工智能领域的发展却遇到严重困难。首先，计算能力的需求并未放缓，虽然计算能力越来越快、便携、大量集中，但是仍然存在极高的价格水平，这就导致了总体成本上升。此外，技术壁垒较低、算法封闭、数据缺乏等因素限制了人工智能的发展，也促使许多人担心人工智能会走向失败。

另外，由于人工智能涉及到高维数据的运算、无法获取真实世界的数据，计算机学术界也一直处于劣势地位。随着生物医疗、卫星航拍、大规模数据计算、地图导航等应用场景的出现，需要大规模、分布式的人工智能技术。而传统的单机计算机资源有限，更难满足这些需求。

### 1.1.3 人工智能的机遇

随着机器学习、强化学习等技术的发展，人工智能的研究已经成为一个新兴的科研方向。虽然人工智能领域的研究仍然面临着许多技术瓶颈，但是在新一轮的科研热潮下，人工智能的研究将会持续蓬勃发展。

目前，人工智能正在对整个经济社会产生着重大影响，其对社会、经济和产业的影响都很广泛。在这个过程中，人工智能技术将会从两个方面取得长足的进步。第一，它将赋予普通民众更多的职责和选择权。第二，它将使得整个社会的生产力和效率得到大幅度提高。

在这场变革中，人工智能领域要面对的挑战还有很多。人工智能需要能够与人们互动，需要能够理解用户的意愿，还需要能够在快速变化的市场环境中迅速做出反应。同时，为了使得人工智能技术真正落地，还需要承担相应的法律义务、管理压力、技术壁垒等问题。
