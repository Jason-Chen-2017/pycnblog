# 1.背景介绍


建筑技术是现代文明的重要组成部分，也是解决日常生活问题、促进社会和谐共处的一项重要工具。建筑技术的发展离不开工业革命、信息化革命、城市运动等变革性的历史进程。本书将从建筑技术的历史、发展脉络、作用领域及其变化等方面对建筑技术进行全面的系统性阐述。  
# 2.核心概念与联系
## 2.1 建筑技术的产生与发展
- 古希腊时期：古希腊的建筑文明逐渐成为世界上最先进的技艺。它引入了多种形式的建筑材料和装饰品，并广泛运用不同的工程手段创造出不同形式的建筑。古希腊的城邦都有高度的建筑文化，而且它的建筑也带有浓郁的哲学意味。
- 中世纪：古典建筑、罗马尼亚城堡、塞耳河德式建筑、北欧的贵族墅、修道院、教堂、寺庙和宫殿等，这些都是中世纪初期的建筑风格。随着中世纪的结束，启蒙思想家们对物质文明的重视和对人类社会的需要，促使社会发展了技术革命。在中世纪末至现代，技术革命推动了建筑技术的发展，特别是建筑设计和建筑施工技术的发展，并催生了一批新兴产业如手工艺、纺织制作、服装鞋帽、电子产品、金融投资、纸浆制造等。
- 工业革命：在工业革命之前，建筑设计与施工都是由专门的专业人才完成。如古代的雕刻者，画家，瓷匠等；而在工业革命之后，各个行业的人员都有机会参与到建筑设计的工作中来。工业革命后期的建筑设计更注重对空间结构的布局、功能的优化以及美观的审美效果，并通过工程手段来达到更高的设计水准。因此，建筑设计成为当时主要专业之一。
- 信息化革命：1970年代，计算机技术发展、互联网技术普及，带动了全球范围内的科技革命。信息化革命使得建筑设计与施工得到迅速发展。由于数字化、自动化和云计算的影响，建筑设计与施工领域发生了前所未有的变革，包括数字建模、虚拟现实、增强现实等技术的应用。同时，建筑行业向服务业转型，在向住宅建设和商业业主提供更好的建筑服务。
- 城市运动：到20世纪90年代末，科技革命和经济危机使得社会生产力急剧下降，房地产价格快速上涨。随着90年代末的美国经济衰退，美国的房价越来越高，越来越多的人迁往其他国家。由于这场房地产热潮，许多人积极探索利用建筑技术来解决他们的居住问题。1990年，艾伦·凯恩斯、卡尔·巴斯、哈里·丹尼斯、约翰·洛克菲勒、唐纳德·特朗普等人，联合起来组建了“富人都想买的房子”团体，并举办了房屋展览会。90年代末，美国迈入了一个金融危机的时代，很多人失去了工作，为了保住家产，还要把房子租给中介。
## 2.2 建筑技术的应用领域
建筑技术的应用领域主要包括住宅、商业、公共建筑、学校、医疗卫生机构、教育中心、交通运输设施、公园和景区等领域。
## 2.3 发展历程
### 古代建筑
古代建筑最早源自于地域的特定。古埃及的石窟建筑、古埃及神庙、古巴比仑阿提目拉的城堡建筑、古希腊的雕塑、亚美尼亚的雕像、古罗马的建筑、古埃及的建筑、古印度的沙漠建筑、古希腊的战争花园、古埃及的圆形建筑、古埃及的女神宫殿，都是早期古代建筑的代表。
### 中世纪建筑
中世纪初期的建筑包括古典建筑、罗马尼亚城堡、塞耳河德式建筑、北欧贵族墅、修道院、教堂、寺庙和宫殿。这些建筑曾经很受欢迎，被认为具有独具一格的艺术魅力，但随着文明的进步，也渐渐走向破旧、败落。而到了20世纪初，这些建筑已经逐渐被淘汰或边缘化。
### 工业革命时期建筑
在工业革命之前，建筑都是专门的专业人才完成的，例如雕刻者、画家、瓷匠等。工业革命之后，建筑领域也开始跟上形势。19世纪70年代以后，世界进入了新的技术革命，这时，建筑行业开始成为一个重要的领域。新的科技革命给建筑业带来了前所未有的变革，各种新技术的使用促使建筑设计与施工领域发生了巨大的变化。如数字建模、虚拟现实、增强现实等技术的应用，使得建筑行业进入了一个全新的阶段。
### 信息时代建筑
20世纪90年代以后，随着科技革命和经济危机的加剧，国际金融危机爆发，全球房地产市场崩溃，使得美国房价飙升，全球房地产市场快速发展。这是一个非常重要的历史节点，也是美国建筑学界和专业人员的奋斗目标。1990年，美元指数飙升到接近2万5，而房价暴涨到了每平米30万美元以上。由于这个情况，许多人开始考虑利用建筑技术来解决他们的居住问题。1990年，艾伦·凯恩斯、卡尔·巴斯、哈里·丹尼斯、约翰·洛克菲勒、唐纳德·特朗普等人，联合起来组建了“富人都想买的房子”团体，并举办了房屋展览会。90年代末，美国迈入了一个金融危机的时代，很多人失去了工作，为了保住家产，还要把房子租给中介。由于房价高企，许多中产阶级人士选择购买第二手房，甚至冒着生命危险的风险购买豪华豪宅。到了2000年代，人们开始觉得建筑科技正在改变美国社会。
### 建筑技术发展历程总结
- 古希腊时期（前7千年）：古希腊建筑演绎出独具魅力的艺术风格，成为世界上最先进的技艺。它的建筑材料和装饰品在世界范围内流传，古希腊城邦都有高度的建筑文化，而且它的建筑也带有浓郁的哲学意味。
- 中世纪时期（5千—2千年）：中世纪的建筑曾经很受欢迎，被认为具有独具一格的艺术魅力，但随着文明的进步，也渐渐走向破旧、败落。
- 工业革命时期（2千—70年代）：在工业革命之前，建筑设计与施工都是由专门的专业人才完成的。工业革命后期，建筑设计更注重对空间结构的布局、功能的优化以及美观的审美效果，并通过工程手段来达到更高的设计水准。
- 信息时代（90年代）：20世纪90年代以后，由于科技革命和经济危机的加剧，房价暴涨到了每平米30万美元以上。建筑技术的发展对全球的房地产市场、住房需求和房地产业态的变化都起到了重要的作用。
## 2.4 建筑技术的作用领域及其变化
### 住宅建筑领域
建筑可以帮助人们获得住宅，从而解决日常生活中的实际需求。建筑对于人们的居住环境的影响，直接决定着人类社会的平等和自由。目前，中国、日本、韩国、泰国、台湾、香港、澳门等国家都在建设超过1000亿平方米的住宅。
### 商业建筑领域
商业建筑能够实现企业的经营目的，并吸引外界资金的注入。企业通过建筑可以更好地定位自己的位置，将更多的精力放在满足客户的需求上，提升效率。另外，商业建筑还能提供企业的隐私和安全感，增强人们的商业信心。
### 公共建筑领域
公共建筑是人类社会赖以存在的基础设施，没有公共建筑，就无法维持正常的社会秩序。公共建筑可以吸引游客、开发商业机会、提升交通条件、促进公众参与、营造良好的居住氛围，从而提高社会的幸福感。公共建筑也具有社会责任感和公共利益，是普通居民的共同财富。
### 学校建筑领域
学校建筑能够帮助学生提升竞争能力、增强个人能力。学校建筑能够提升学生的综合素质、塑造大学生的气质。另外，学校建筑还能培养学生的基本素质，鼓励学生树立正确的价值观，培养他们的独立思考能力。
### 民政建筑领域
民政建筑是社会的基石，也是政府机关为维护公共利益而建立的制度。民政建筑能够让人们享有和平、民主的生活。民政建筑还可以减少暴力犯罪、促进社会稳定，防止社会冲突升级。
### 学术建筑领域
学术建筑用于提升科研人员的研究能力、开发创新思维。建筑设计可以提高科研团队的整体协调性、透明度和凝聚力，并且有助于促进科研活动的顺利开展。
### 服务业领域
建筑对于提供各种服务来说也十分重要。比如，医院、学校、警察局、便民服务中心、物业管理公司、电影院、旅馆、餐厅、咖啡厅等。建筑不仅能提升服务的品质和用户体验，而且还能让公司保持运营的连续性。
### 交通运输领域
交通运输业是我国的支柱产业之一。建筑能够引导车辆到达目的地，降低交通事故的发生。建筑也有助于促进交通网络的畅通，提升交通管理的效率。
### 展览馆、博物馆、体育场馆、博览园林、游乐园、风景名胜、旅游景点等。建筑也用于展示旅游产品、传播旅游知识，促进旅游业的发展。
## 2.5 建筑技术的发展趋势
建筑技术的发展趋势总体呈现出复合发展模式：
- 1、需求驱动：在当今信息技术的发展、用户习惯的变化、社会的不断变迁、经济的不断增长和成熟等方面，都要求建筑设计和建筑施工的技术水平不断提高。随着人口的增加和城市的扩张，以及外部的技术革命的影响，建筑技术将成为未来社会不可替代的重要力量。
- 2、智能、生物、多样性：建筑技术正在从单一的形式转向多样化的形式。智能建筑、生物建筑、多样化建筑三者密切相关。智能建筑指的是建筑装置、智能设备的集成、应用和管理，可以实现自动化、人工智能和机器人控制，提升建筑性能和节能效率；生物建筑又称人工景观，采用植物生命生长发展作为设计原则，通过生物工程、人工制药、造纸技术、玻璃技术、锻造技术等手段，构造出适应自然的植物、动物、鱼类、植物、建筑结构等场景，吸引眼球，增加购买决策的可信度；多样化建筑除了具备智能、生物的特性之外，还有多层次的空间规划、美学语言、建筑学理论、装饰艺术、规划设计等方面。
- 3、协同互联：当前，建筑行业的协同互联是一个新兴的现象。不同项目之间的合作往往能够产生更好的效果。智能化建筑、环保建筑、免疫建筑、绿色建筑、新能源建筑、多孔建筑、智慧光伏、人工智能等多方面在建筑技术的融合与整合。
- 4、服务经济：建筑技术的出现、应用以及持续的发展都会推动我国建筑业的转型。服务业是建筑行业未来发展的一个重要方向。建筑作为一种新型的服务承载平台，将成为提供专业、优质的服务的必要元素。
