# 1.背景介绍


## 一、数字医疗的兴起——从20世纪70年代末至今
在20世纪70年代末，美国经济危机蔓延到全球，“新冠”病毒席卷全球，医疗行业也被危机所冲击。由于当时世界医疗卫生组织（WHO）尚未形成国际制度，各国之间难以合作，因此各国医院只能单打独斗，无力应对危机。而在这个背景下，越来越多的国家和地区开始了数字化医疗的尝试，比如美国通过互联网、医疗器械和设备在线销售，让临床医生可以在线上跟踪患者病情并实时诊断，而印度则推出了用数字化方式进行血液检查的系统。到了90年代末，随着中国的崛起，数字化医疗迎来蓬勃发展。2015年，中国政府启动了人民的共同富裕工程——共建共享，开启了数字医疗的大门。截止到今年7月，中国已经建立了超过30万个独立的数字医疗平台，覆盖全国、全市、全省乃至全国每个村、镇、街道。截至目前，全国已有超过6000万患者接受过数字化的医疗服务。
图1 中国各地区医疗器械互联网平台数量排名  
  
可以看到，数字医疗的发展如火如荼，但是仅仅靠技术突破并不能带来真正的改变。原因在于，医疗的核心问题还是老百姓的医疗需求，即使做出了一些技术上的突破，仍然需要社会的支持、政策的落实和更加开放的心态，才能够让人们慢慢地习惯数字医疗。这一点，数字化医疗已经有了很好的证明。在这段时间里，全球多个国家都出现了类似的情况，数字医疗在推动健康领域的发展方面发挥了重要作用。

## 二、健康管理的普及——在科技与政策相辅相成的双向循环中推进
进入21世纪之后，数字医疗已经成为医疗行业的一个重要组成部分。然而，如果仅仅靠数字医疗就能解决所有的医疗问题，那势必会造成严重的资源浪费。因此，需要有更多的机制来促进健康管理的信息化进程，提升健康管理的透明度、问责性、参与性、可及性和可预测性。也就是说，不仅要加强数字医疗的基础设施建设，而且要结合健康管理的信息化进程，充分发挥数字化工具的价值。
图2 中国医疗健康管理信息化现状  
  
“十三五”期间，中国的健康管理信息化工作取得重大进展，主要包括：

1. 统一身份认证管理平台建设。2015年底，国家医疗信息中心将统一身份认证（U Authentification）管理平台部署到北京。其目的是建立统一的医疗机构登记，统一门户网站登录认证，简化患者、医生、护士、药剂师等的登录注册流程，提高工作效率。同时，还可以通过统一身份认证管理平台快速实现数据共享，实现患者之间的实时交流。截止目前，已有超过10万家医疗机构注册、认证，其中超过60%拥有网络门户网站。

2. 护理知识问答中心建设。2016年，中国第一个版本的护理知识问答中心（PHAC）于广州正式启动。该平台是以患者咨询和就医为核心，整合全国各地不同专业护士群体的最佳实践经验，为患者提供集远程诊断、疾病筛查、医疗建议为一体的综合服务。同时，平台通过社区互助的方式促进自我学习，形成定期问答交流会，帮助护士群体解决生活中的实际问题。截止目前，平台已经形成医疗咨询服务收入约50亿元，其中咨询收入占比20%，信息流通能力强，覆盖范围广，用户质量良好。

3. “上海疾病预防控制中心”建设。2017年，“上海疾病预防控制中心”（SHBDC）宣布启动。该中心将成为目前国内规模最大的医疗卫生服务平台，打造国际一流的医疗卫生服务体系。目标是建立完整的“从患者到护士再到国家的全生命周期健康管理”解决方案，包括就诊流程优化、疾病筛查和诊疗建议、远程监控和指导、患者满意度评价和改善计划、健康档案、住院治疗、退休转福、重症监护等各项服务功能。截止目前，SHBDC已经建立了相关产业链，包括生物制品、医疗设备、健康管理、影像影像、健康教育、养老康复、慈善捐赠等领域的布局，确保“上海成为中国疾病预防控制最具代表性的城市”。

4. 健康管理市场建立。2018年，“中国医疗健康管理市场”（CHMM）正式成立。这是由健康IT界人士联合主办的一项开放型行业创新创业项目，旨在搭建一个由医疗机构、医生、药剂师、患者四方共同参与的、面向全国各地的医疗健康管理信息化平台。平台将围绕“医疗健康管理、健康公共服务、医疗管理咨询、医疗保障体系建设、智慧医疗健康服务、产业转型升级”五大领域，构建医疗健康管理信息化生态圈，以实现服务实体和个人、行业和市场两级的融合。截止目前，平台已经在京津冀、沿海、长三角、珠三角、环渤海等区域布局，覆盖数百家医院、诊所和公立医院等全国范围内的医疗机构。

总之，在科技与政策相辅相成的双向循环中推进，数字医疗和健康管理信息化的发展具有重要的历史意义。数字医疗可以有效地促进健康信息化进程的发展，让医生、患者、药企和公众互联互通，提升医疗服务质量；健康管理信息化则有利于促进健康管理的专业化、规范化、透明化和服务化，提升健康服务效率和公众满意度。
