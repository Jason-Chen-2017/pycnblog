# 1.背景介绍


青铜器时期（公元前234-公元前79年）是人类文明发生里程碑式的时期，大约四千多年前，玻璃、石英、陶瓷、骨头、铁和黄金在当时已经开始进入工业化生产领域并得到应用。青铜器作为一种高强度、耐磨性、耐用度高且不易腐蚀的金属制品，被视作人类进步的阶梯，并且是远古文明的标志。

自从234年到79年，这一段历史被称之为“青铜时代”。从这段历史中我们可以得出几个关键的观点：

1.全球化及文艺复兴带来的经济社会文化转型。这一波文化运动带来的改变显著地促进了科技革命，带来了全球化。其中最重要的是印刷术的发明。它使人类的信息交流变得更加有效率。

2.科技革命也带来了生产力的飞跃。在这个过程中，人们创造出各种各样的金属制品，如贝壳、玉石、矿石、宝石、翡翠等。

3.科技革命给人类的生活带来了巨大的变化。比如，数码技术让人们的生活发生了翻天覆地的变化。从早期的扑克牌到现如今的智能手机、电子游戏、VR头盔、无人机等，我们都可以看到由科技驱动的个人升级换代。

因此，作为一个技术人，如果想要对接触到的技术进行深入研究并能够写好一篇专业的技术文章，那么需要先了解一下青铜时代这段历史。

# 2.核心概念与联系
我将把“青铜”作为整本书的中心主题。“青铜”指的是至少从公元前234年到公元前79年，这一段时间里出现的三种主要金属——石英、陶瓷、黄金。

对于青铜来说，它的三个特征是“硬度”、“柔韧性”和“低温耐久性”，这三者相互关联，共同塑造了它独特的魅力。所以，我们首先介绍一下这些概念以及它们之间的关系。

## 硬度（Hardness）
硬度是指金属粘结时所需的摩擦力，通常用单位体积上的硬度来衡量。硬度越强，表面粘结的机会就越多，表面越稳定，但也更容易沾到其他材料上导致断裂。有的金属刚开始制造时硬度较弱，但随着它的使用和制品的使用，它的硬度逐渐提升，而到了后期制成的成品，它的硬度往往达到一个比较稳定的水平。目前，国际标准化组织将硬度分为“软度”（Softness）和“硬度”两个级别。目前世界上非常著名的五种金属，如铜、铝、钒、铬、镍，都是软度为0，硬度分别为16.5、14.3、13.5、11.3、10.6，很难找到软度为1的金属。

## 柔韧性（Elasticity）
柔韧性是指金属的抗拉、抗压能力。金属具有良好的刚度，可以在一定范围内施加压力而不会变形或起伏。但是，如果是抗拉的金属，就会承受较大的推力。比如，铁锈或者苹果皮，虽然强度很高，但是只能承受较小的推力。因此，柔韧性是衡量金属是否坚固、韧性的重要参数。

## 低温耐久性（Low Temperature Stability）
低温耐久性是指金属在低于绝对零度（-273.15°C）时，是否仍然保持耐用的属性。低温耐久性意味着金属在低温下不会起爆炸、烧坏或熔融。因此，低温耐久性也是衡量金属是否耐用、质量可靠的重要参数。在国际标准化组织中，低温耐久性通常用单位体积下的温度为-273.15°C下的耐久性来衡量。但由于不同的材料存在着不同的温度耐久性曲线，所以该值并非普遍适用。

根据青铜的三个特性，我们可以总结一下它们之间的关系：硬度影响着柔韧性，柔韧性影响着低温耐久性，而低温耐久性又影响着硬度。比如，如果硬度越高，柔韧性就越强，并且还能承受较大的推力；反过来，低温耐久性越好，就越容易使用，而且其硬度也要相应提高。此外，如果硬度和低温耐久性都较高，则还可以通过增加硬度和降低温度来增强硬度，提高重量和防止损坏。
