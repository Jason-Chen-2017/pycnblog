# 1.背景介绍


结构化思考（Structured Thinking）可以说是现代信息时代重中之重。随着互联网的普及和信息处理速度的提高，越来越多的人需要在短时间内收集、整理、理解大量的信息，而如何快速有效地获取、整理、分析、应用这些信息变得越发重要。但是，一般人往往会遇到两个难题：第一，缺乏对信息的全景式的把握；第二，缺乏对信息的系统性的思考能力。因此，结构化思考就是为了解决这个问题而产生的一种新方式——将复杂的知识和信息按照逻辑、层次、主题等等框架结构进行分类、归类，并运用所学的分析技能、方法论以及知识体系，将事物的联系紧密地组织起来，从而更好的掌握和分析这些信息，从而达到全面而深刻的认识。
结构化思考的重要体现就是将复杂的信息和知识通过所谓的“金字塔”（pyramid）形态来进行分类和呈现。一般来说，金字塔由上至下分为四个层级：主题（Top），背景（Background），主干（Main），支撑（Support）。下面是一个简单的金字塔示意图：
图1：简单金字塔示意图
由于结构化思考的巨大作用，人们已经很久没有机会静下心来慢慢研究某个特定领域的知识了。从古至今，无数的大师们都在研究“为什么？”、“怎么办？”，但是只有少数人能够抓住“如何做？”这三个要点。结构化思考正是在如此需要帮助的大环境下产生的。
但是，在实际应用过程中，由于知识的复杂性和多样性，很多时候只是知道它的位置信息还不够，还需要进行进一步的阅读、思考和总结才能成为真正的专家。另外，结构化思考通常只适用于一定的思想高度，越是底层的东西越容易被遗忘，所以也不能单独应用于所有的场景。因此，在本文中，我希望通过介绍一些基本的结构化思考原理和相关的方法论，结合生活中的例子，教授大家如何用结构化思考的方法打磨自己的思维、认识和信息组织能力。
# 2.核心概念与联系
下面，我将首先简要介绍一下结构化思考的一些基础概念和联系。
## 2.1 “金字塔”（Pyramid）与信息组织
结构化思考的关键在于理解“金字塔”（pyramid）这一最常用的工具。“金字塔”是由上至下的五个层级组成，分别为：主题层（Topic），背景层（Background），主干层（Main），支撑层（Support）。主题层代表着当前关注的主要事情或主题，背景层提供了比主题层更宏观的视角，而主干层则主要集中展示重要信息，支撑层则提供一些支撑信息。如下图所示：
图2：“金字塔”（pyramid）的层次结构
“金字塔”这种信息组织模式有什么用呢？“金字塔”的基本特点是“简单到复杂”，即在一个层级里，只有最重要的信息或数据才会出现。每一层级都会包含上一层所有数据的细节信息，这样就可以让读者看到最新的、重要的、具体的消息。比如，电视台在播出新闻的时候一般会优先播放重要的、正在发生的事件，其次再播放其他热门的事件。这也是“金字塔”的一个重要用途。

举个例子来说，以图2中的“金字塔”为例，如果想要了解某个事件背后的故事，只需关注“背景层”即可。因为该层主要提供背景信息，包括事件的初始情况、发展过程、及其影响。而对于那些新闻报道，该层的信息就会比较具体，其中就可能涉及到政策法规、采取的行动，以及舆论反映。当我们想要了解某件具体的事情时，我们可以跳过一部分“金字塔”，直接进入“主干层”。“主干层”主要是用来展示最核心的、最主要的信息。由于“金字塔”层次分明，所以读者可以通过对比来判断哪些层次的信息值得关注，以及各层信息之间的相互关系。

## 2.2 “金字塔”（Pyramid）与习惯与风格
“金字塔”（pyramid）这一工具还可以被用来形象地描绘人的思维习惯和行为风格。例如，你可以把你的工作习惯看作是“金字塔”的主题层，你的生活习惯则是支撑层。比如，你可能会认为在工作中应该更重视创新，生活中则更重视传统。而另一方面，如果你认为在工作中应该注重效率，生活中则需要花更多的时间陪家人。这两者的差别反映了你的不同想法，它们也会对你后续工作和生活产生影响。

同样的，“金字塔”也可以用来形象地表示某个群体的性格特点。比如，如果你发现自己喜欢讨论现实主义、情绪激昂的新闻，那么这可能就是受精神文明、幽默感影响的结果。当然，“金字塔”并不是绝对的、完美的，它只是用来描述人的思维方式和行为风格。
