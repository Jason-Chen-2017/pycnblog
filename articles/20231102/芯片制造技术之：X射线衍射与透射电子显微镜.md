# 1.背景介绍



X射线衍射（X-ray scattering）指的是用X射线击中金属表面而发生的一系列反射、折射现象，其产生的能量可用于各种应用，如光纤传输、射出辐射、聚焦等。除了自然界常用的高能物质外，人工制成的X射线聚合物也会产生衍射效应，例如核反应堆喷气背包中的硝酸盐反射性材料就属于这一类型。 X射线衍射常与电子显微镜（Electron Microscopy, EMC）相关联，但其实它们之间的关联并不直接，只是因为其不同特性导致可以用不同的技术观察到。 EMC 是一款高分子摄影测序仪，通过显微镜下蚀测量或照相质谱的方式记录电子的位置分布，根据这些信息可以诊断某些组织的结构。所以，EMC 可以用来检查衍射物质的结构和功能。


为了更好地了解X射线衍射，我们首先需要了解一下X射线的基本知识。X射线具有电磁波的性质，可以由束流或连续发射方式发出，也可以由多条束流叠加形成波长较宽的吸收点阵。它可以穿透金属介质，在非金属物质中产生强烈的粒子反射及散射作用。X射线的传播速度接近于声速，在某些情况下可以直接穿过晶体，进入晶格内部。对于高能物质如核反应堆喷气背包中的硝酸盐、氢化卤素、三水素等，X射线在入射物质发生衍射时将释放巨大的能量。



电子显微镜的原理简单来说就是利用超精细结构（X射线集束）刻蚀微观颗粒，使得从样品上采集到的信号成为图像的一部分，并最终实现全息影像的制作。电子显微镜技术和X射线衍射技术密切相关，这是因为电子显微镜通常配备有大功率的反射镜头，能够在一定程度上捕获不同能级的光子，而这种捕获能力正是X射线衍射所依赖的能量来源。当被衍射物质上浮微粒时，其能量由X射线所吸收、转换后再放回原处。这种能量的传播路径称为电子漫游轨迹（Electron Trajectory），当电子漫游轨迹与晶体层的平行层接触时，就会在晶体层留下一个电子谱图标记（称为电子轨迹图）。通过电子显微镜的观察者，可以在晶体层看到相应的电子轨迹图，从而进行结构分析、功能鉴定等。X射线衍射与电子显微镜的结合，提供了一种既快速又准确的材料分析手段。

# 2.核心概念与联系

## 2.1 衍射概念

衍射是指通过传导性光子与电子的相互作用而发生的一种现象，这种现象使某种物质受到某种波长或电压的干涉而呈现出衍射状况，包括短波衍射、中波衍射、长波衍射。通常情况下，衍射是由于弱、偏离等原因所引起的，它的产生往往是由几种原子共同作用而发生的，衍射的物质可以是一种粒子，如质子、中子或离子，也可以是一种构件，如电子芯片等。衍射反映了材料与其外界环境之间的相互作用关系。

## 2.2 衍射过程

X射线的衍射过程一般分为以下四个步骤：

1． 入射光线的收集与混合。入射X射线经过非共线空间中的空间区传播，在空间中的某一点突破非共线状态，被散射器捕捉到，经过衍射腔和衍射瓣的聚焦，形成一个光子束流，随后被收集器收集。

2． 撞击、损伤和衰变。散射器对着X射线束流前进，遇到激发元件后会发生撞击，激发元件损坏或者失灵时可能会出现衰变，其后果是使散射光束发生旋转、模糊、退散、失真等现象。

3． 相空间的平移。散射光束通过某些晶粒之后，传播到其他位置，这叫做相空间的平移，光子本身不动，只改变了它们的路径。

4． 散射到目标物质。散射光束在晶体层中自由穿行，遇到物质表面就会发生反射，在衍射光栅内产生光子辐射，在目标物质表面形成衍射电子轮廓。

## 2.3 衍射相关术语

### 2.3.1 中波衍射

中波（μwave，约等于1.1 µm，即微米）是目前可见光最宽的波长。它通常由外层环境所产生的光子束进行散射而发出，远处的反射衬底甚至可以做成星云状。由于中波天文特征尺度小，因此可以用来确定星系的大小和运动方向，有广泛应用。同时，中波还有个特别的名字——黑洞吸收线，这个特别的名字表明中波长的衍射衰减截止频率较低，可以完全阻碍到光子束的继续传播。

### 2.3.2 长波衍射

长波（γwave，约等于1064 nm）是波长最小的可见光波长，它的衍射效果可以到很低的能级，且反射效果也十分猛烈，是大多数物质的标准波长。对于一些高能物质如核反应堆喷气背包中的硝酸盐、氢化卤素、三水素等，长波衍射的能量远比中波衍射的能量要大很多。

## 2.4 X射线与电子显微镜

X射线的衍射过程直接影响了电子显微镜的性能。通常情况下，电子显微镜主要用于显微测距、结构分析和DNA测序等方面的研究。电子显微镜可以发现各种生物体的亚基的原子分子结构，探索发现非金属的结构以及分子翻译过程中的分子轨迹。但是，对于衍射物质的结构分析，还存在一些技术上的障碍。

1．衍射电子的迸发时间。电子显微镜的探测速度一般为每秒钟1000次扫描，因此迸发的速度应该尽可能快。同时，电子的衍射波长越短，其迸发时间就越长，这就要求衍射物质的表面特征不能太复杂，否则会影响整个衍射过程的时间。另外，由于电子吸收较慢，因此存在一定的局限性，只能发现那些局部的衍射特征，而不能观察全局。

2．衍射物质的退散情况。电子显微镜只能记录原子分子的结构，并不能看见退散情况。也就是说，它只能观察电子能量在一定程度上的逃逸，但无法观察退散的电子。对于一些衍射物质，比如金属，电子显微镜并不能完全捕捉到衍射物质的全部结构。

3．衍射物质的作用方式。X射线衍射可以是主动的，即原子分子的衍射过程由X射线主导；也可以是被动的，即原子分子的衍射过程由其本身的电子能量产生。当X射线衍射是主动生成的时候，我们就可以用X射线照射进行测量。当X射线衍射是被动生成的时候，我们就需要考虑如何破坏、阻止衍射的过程。

4．衍射物质的构成。衍射物质通常是由一系列的原子和其他电子构成。与X射线相比，电子显微镜主要用于捕捉单个原子的结构，因此对于各种复杂的衍射物质，电子显微镜的识别能力就比较差。
