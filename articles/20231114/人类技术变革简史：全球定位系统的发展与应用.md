                 

# 1.背景介绍

  
随着科技的飞速发展，科学技术的快速进步与产业的日益壮大带动了人类社会的科技文明程度的提高，使得“智慧”这个词开始成为一种主流话语，甚至成为了一项基本生活要素。而在当代，智能手机、互联网、电子消费品、无人机等一系列科技产品及服务正在蓬勃发展中，它们对人类生存环境、社会经济活动、健康和医疗等方面都产生着重要影响。其中，人类能够精确地知道自己当前所在位置的信息被称为“全球定位”。
目前全球定位已经成为众多行业的标配，比如汽车、航空航天等领域，并且随着传感器的普及及技术能力的不断提升，全球定位系统也越来越智能，能够提供更加精准的位置信息。但是，由于技术水平、工程规模、运算量等因素的限制，目前全球定位系统仍然存在一些不足之处。因此，如何利用全球定位进行有效率地运用将是一个值得深入探讨的话题。本文试图从全球定位系统的历史发展过程和其发展趋势角度出发，阐述全球定位系统的特性、优点和局限性，以及如何利用它解决实际问题。  
# 2.核心概念与联系  
## 2.1 全球定位系统的定义
“全球定位系统（Global Positioning System, GPS）”是指能够精确确定人或移动物体的坐标位置的一套系统。它由一系列卫星固定在陆地上，通过接收无线信号、电磁波或其他能量形式来获取地球自转动的卫星，并根据运动学原理测算地球上各种天体的相对位置，然后根据地球椭圆轨道模型对卫星轨迹进行修正，最后将得到的精确位置数据反馈给用户。它的主要功能包括：

1. 定位：精确定位人或者移动物体的经纬度、高度信息。
2. 追踪：跟踪移动物体的位置变化，可以用于车辆、船舶、飞机的自动巡航、轨道监控等。
3. 导航：提供出行方向和距离的指示，可用于智能交通系统、地图制作、出租车路线规划等。
4. 智能交通：为汽车提供最佳驾驶路径和估计路况，基于车辆的定位信息可预知前方道路情况。

## 2.2 全球定位系统的分类
GPS系统按定位方法、定位精度、定位范围、定位时延、使用的设备种类及技术实现方式分为如下几种类型：

1. 单一卫星定位系统：指仅使用一个卫星进行定位，主要应用于大气层外的小区、军事区、道路等环境。如美国国家卫星导航中心的Galileo系统。
2. 组合卫星定位系统：指使用多个卫星进行定位，主要应用于城市周边的区域。如欧洲的米特兰系统、中国的北斗系统等。
3. 离线定位系统：指使用卫星接收器作为基站，无需连接网络即可进行定位。如欧盟的卫星导航定位系统、中国的基站定位系统等。
4. 直接定位系统：指不需要卫星接收器直接进行定位，只需要外部输入基准星坐标就可以进行定位。如英国的QZSS系统。
5. 行星跟踪系统：指使用轨道上的行星进行定位。如欧洲的伊利湖系统。
6. 综合定位系统：指将多个定位系统的定位数据融合，提升定位精度。如中国的多GNSS混合定位系统。
7. 补充定位系统：指与真实卫星距离较远时，采用静态卫星定位数据对真实卫星进行补充，提升定位精度。如俄罗斯的ORB系统。
8. 增强定位系统：指改善卫星捕获图像、处理速度、导航性能等，提升定位精度。如美国的WAAS系统。
9. 其他定位系统：如高效定位系统、微型定位系统、移动基站定位系统等。

## 2.3 全球定位系统的发展过程
### 2.3.1 GPS初期阶段——早期GPS原型（Rover Project）
19世纪末到20世纪初，英国的<NAME>以及他的同事们刚刚研制成功了一架实验性的卫星车Rover，能够在短时间内进行定位和轨迹推导。但因为缺乏可靠的通信网络，最终未能进入商用化，这段历史称为“初期GPS”。

20世纪20年代末，欧洲各国开始逐渐尝试开发自己的GPS系统，但由于技术水平参差不齐、设备昂贵、通信链路复杂、维护费用高等原因，始终没有取得突破性进展。直到20世纪50年代末，美国人Sir <NAME>首次提出“超级卫星轨道（GPS satellite system）”概念，试图利用超过八颗GPS卫星构建一个综合的、能覆盖整个地球的大型卫星轨道，这才获得了令人瞩目且持久的成功。

20世纪60年代初，中国第一颗GPS卫星GSAT-100在西安登月成功后，引起了国际的极大关注。此后，北京、长沙、珠海等地陆续建设更多的GPS卫星，扩大了覆盖面积。2008年6月，美国的GPS卫星GEO-60完成低轨道升空，是迄今为止中国、欧洲、日本三大GPS卫星轨道上的第四颗卫星。随后，中国又建成全国第一个GPS全国区域定位系统（GNSS）基地——华北五省区——作为全国唯一的独立 GNSS 发射系统，中国则成为了目前世界上唯一实现了具有完整导航功能的全国性定位系统。

### 2.3.2 GPS第一阶段——原创GPS系统（Personal GPS）
原创GPS系统(Personal GPS)主要由一个卫星进行定位，可以满足大部分居民的基本需求。20世纪80年代中期，英国、德国等国兴建的欧洲第一座卫星接收站EUREFON-AVS-1与欧洲载人航天中心一起开启了原创GPS项目。该项目的目标是为了开发一款可以满足个人使用需求的全球定位系统，其设计原理是借助卫星的广播信号进行定位。

20世纪90年代末，原创GPS项目后续开放，英国、法国、荷兰、比利时、瑞士等国还各自投资开发了一款Personal GPS系统。如英国的PulseStar GPS、瑞士的FlyMaster GPS、比利时的Galileo GPS、法国的Gaia GPS、荷兰的GALILEO2 GPS等等。但这些系统都只能局限于特定领域，缺乏通用的处理能力。

2010年，英国的Satellogic公司推出一款名为Orbital的Personal GPS系统，改造了原有的原创GPS系统，成功进行了商业化。这款新产品的定位精度超过9米，可以满足居民的日常用途定位需求。此后，Orbital又升级为OSRION定位系统，但其定位精度只有5米左右。

### 2.3.3 GPS第二阶段——中国卫星定位系统（China GNSS）
中国卫星定位系统(China GNSS)是在2010年3月，由中国国家科委主办的首个卫星定位系统，也是中国第一个具备完整导航功能的GNSS定位系统。该系统由一颗运载火箭所发射的MEO-1卫星搭载，利用高分辨率宽带卫星导航系统进行定位。

2011年，中国卫星定位系统上了新的楼台，成为中国卫星导航定位系统(CNS)的总部所在地。随着卫星数量的增加、质量的提升，CNS已具备支撑和扩展全国GNSS发射系统的条件。2013年，CNS又升级为支援地面基站定位、衛星定位和偏航导航的超星全球定位系统（Globalstar）。超星、中星、北斗、Inmarsat、NOAA等国际导航组织纷纷加入超星星系导航系统，共同促进中国的GNSS系统建设。

2015年5月20日，中国卫星定位系统正式向公众推出，并且全面开放。同时，为了吸引更多的企业和运营商使用该系统，2016年，中国卫星导航设备巡礼协会宣布批准100余家运营商享受权益福利，推出各种产品和服务。

2018年，中国卫星定位系统建成后的第一周，当地已接到超过2.7亿人的定位请求，累计服务近千万人次，为中国GNSS系统的应用提供了巨大的便利。

## 2.4 全球定位系统的特性、优点和局限性
### 2.4.1 全球定位系统的特性
1. 高度集中的定位原理：全球定位系统的定位原理是使用基准卫星和其它卫星之间的相对距离，结合空间曲率、重力加速度、星历参数等手段计算出卫星的真实位置。因此，定位精度高度依赖于基准卫星和其它卫星之间的距离。
2. 灵活性高：全球定位系统的灵活性很强，可以用于不同类型的定位任务，如对战机、汽车、飞机等。它具备一定的容错能力，能够适应多种场景下的定位要求。
3. 可靠性高：全球定位系统的定位功能基本无毒害，可以信赖，适用于各种环境下，如危险环境、干旱环境、高山、雪山、雾霾环境等。
4. 时延性低：全球定位系统的定位时延一般在几百微秒到几秒之间，即使是快速移动的物体也能快速定位。这与卫星接收器的速度有关，通常为几兆赫兹到几十兆赫兹。
5. 覆盖广：全球定位系统覆盖范围非常广泛，可以在任何时空提供精准的位置信息。覆盖范围涉及全球，但主要用于北半球和亚洲地区。
6. 全球定位网络：全球定位网络是一个庞大的数据网络，主要包括8组GPS卫星和GLONASS卫星组成，还有超过400台GPS接收机、600台GLONASS接收机。其中，8组GPS卫星分布于不同区域，每组5颗以上，组数不定。GLONASS系统占地面轨道相对GPS卫星的优势，可提供高精度轨道及卫星同步信号。
7. 空间上精密的定位模型：全球定位系统采用的定位模型是空间上精密的模型，包括重力加速度、曲率、太阳蝎、卫星轨道、地球扰动等。该模型可以精确判断卫星的真实位置。
8. 大容量存储器：全球定位系统每天产生大量的定位数据，可用于存储和分析，形成数据分析工具。
9. 服务更优质：全球定位系统的所有基础设施均由国际标准组织承担，保证服务的质量、稳定性和可靠性。
10. 技术实力：全球定位系统涉及到的科学技术和工程技术实力极为丰富。例如，中国卫星导航设备巡礼协会承担的GNSS规范、中央认证中心负责认证相关技术，与苏州天鼎通信等第三方硬件厂商合作的定向激光系统等都是全球定位系统不可或缺的关键环节。

### 2.4.2 全球定位系统的优点
1. 准确性高：全球定位系统能够提供高度精准的定位信息，适用于各种场景下，如地震、火灾、暴风雪等突发事件。
2. 安全性高：全球定位系统能够在各种场景下提供高安全性，能够保护生命安全和财产安全。例如，在火灾发生时，可以第一时间发布警报，避免生命财产损失；在突发情况下，可以优先避让、返航，减少人员伤亡。
3. 效率高：全球定位系统的定位能力极高，可满足各种应用场景，尤其适用于军事、地勘、航空、交通、卫星导航等领域。它提供快速、及时的位置信息，满足各种应用需求。
4. 用户友好性高：全球定位系统的用户接口简单易懂，容易上手。它提供简明清晰的操作界面，帮助普通人学习使用。
5. 价格低廉：全球定位系统的价格并不高昂，只需要几百元一颗卫星，适用于家庭、商业用途。
6. 使用方便：全球定位系统的所有硬件、软件和支持设施都比较成熟，安装部署方便，使得普通人能够在任意时间、任何地方进行位置定位。

### 2.4.3 全球定位系统的局限性
1. 隐私泄漏：全球定位系统能够获取大量的数据，存在数据隐私问题。对于个人信息，可以通过加密等方式进行保护。对于敏感区域，可以使用“匿名化”的方式进行处理，使敏感数据无法被公开。
2. 成本高：全球定位系统的成本相对较高，主要取决于几百元一颗卫星的价格。但是，由于发展速度和技术水平的提升，越来越便宜。
3. 功耗高：全球定位系统的功耗相对较高，每颗卫星的耗电量约为15瓦，超出平均值。此外，由于运算量大，需要高速处理器和内存才能运行。
4. 单点故障：全球定位系统的设计初衷是具有冗余结构，避免单点故障。但随着增发和更新，仍有可能出现单点故障。所以，在实际使用过程中，需要注意系统的可用性。
5. 成本高：全球定位系统的成本是基于一批卫星而言的，包括单卫星、组合卫星、组合系统等。每颗卫星的发射成本约为2000～5000美元，每台接收机的价格约为5000～20000美元，因此价格昂贵。
