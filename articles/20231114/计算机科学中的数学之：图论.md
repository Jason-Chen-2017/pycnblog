                 

# 1.背景介绍


图（Graph）是一种重要的数据结构。它可以用来表示对象之间的关系、经济网络、交通网络等复杂的结构，并用图论的方法进行分析、理解和处理。它由两部分组成，节点（Node）和边（Edge），构成一个节点集合V和边集合E。每个节点v∈V代表一个对象或事物，每个边e=(u,v)∈E表示连接两个节点的边，其中u和v是节点的邻居。
在实际应用中，图可以用于解决很多领域的问题，如推荐系统、网络爬虫、知识图谱、人际关系网络分析、生态系统研究、计算广告优化等。而图论的相关理论、算法和技术也越来越火热，给它的研究带来了新的契机。因此，掌握图论的基础理论与算法技巧对于学习、研究和实践图论很有帮助。
图论中最基本的概念是“路径”（Path）。一条路径是一个无向的边序列，它将一个节点集里面的任意两个点相连，且满足不重复的约束条件。例如，在图6-1中，存在着一条从结点1到结点5的路径，它可以分解成5条边分别是(1,2),(2,3),(3,5)，也可以简化成四条边(1,2),(2,4)(4,5)。

6-1.图的表示方法
图的表示有三种主要方法：
- 矩阵表示法：图G=(V,E)可通过邻接矩阵A[i][j]=w(i,j), w(i,j)=1表示边ij存在，否则为0来表示。邻接矩阵的缺陷是不便于表达边权重，且无法表达无向图。
- 列表表示法：图G=(V,E)可通过一个二维数组adj[][], adj[i][j]存储与节点Vi相连的节点Vj，如果边ij存在则值为vj，否则为-1。列表表示法易于实现，适用于稠密图，但难以表达边权重。
- 字典表示法：图G=(V,E)可通过一个字典dict{node:neighbours}来表示，其中key为节点，value为该节点相连的节点组成的列表。
以上三种表示方法各有优缺点，在不同的情况下选取合适的表示方法有助于理解、分析图论中的数据。本文采用列表表示法。

# 2.核心概念与联系
## 2.1 图的定义
图G = (V, E)是由顶点集V和边集E组成的一个有向、无圈图。顶点集V中的元素称为顶点（Vertex），边集E中的元素称为边（Edge）。图中的每条边都有一个起点和终点，并有一个方向，通常表示为(u, v)表示以顶点u为起点，以顶点v为终点的有向边。
## 2.2 图的性质
### 2.2.1 有向图
有向图：是指边具有方向性，即边指向其始点。在有向图中，对于任意顶点u和v，都有相应的出度和入度。出度d_out(u)表示以顶点u作为头的边的个数；入度d_in(u)表示以顶点u作为尾的边的个数。定理：在有向图中，任一顶点的出度等于到达该顶点的出边的数目，任一顶点的入度等于从该顶点到达的入边的数目。

### 2.2.2 无环图
无环图：是指图中不存在回路，即不存在从某个顶点到达自身的边。判定无环图的两种方法：
- 使用DFS或者BFS遍历图，若能完整访问所有顶点，则说明无环；否则，有回路。
- 判断一个有向图是否有回路，可以使用拓扑排序的思想，先对有向图进行拓扑排序，然后判断每个顶点的入度和出度是否相同，若相同，则说明无回路；否则，有回路。

### 2.2.3 无向图
无向图：边无方向。设G=(V, E)是一个无向图，若(u, v)∉E，则称边(u, v)互补边。对于无向图，任意两个顶点之间都存在两条对应的边，在构造无向图时需注意避免创建互补边。无向图的边数是偶数。

### 2.2.4 树图
树图：是无向连通图，其中任意两个顶点之间仅有一条简单路径相连。通过边来描述树的结构，因此树图也可以看作是一种无向图。树图的边数是奇数。

### 2.2.5 有根树
有根树：树中至少有一个节点，称为树根。有根树的定义可以有多种形式，以下是其中一种：对有向树来说，只有源顶点（源点）没有父亲，其他节点都有一个唯一的父亲；对于无向树来说，任何一个顶点都有一个唯一的双亲。无根树只能是树形结构，不能直接应用于图论。

### 2.2.6 子图
子图：设G=(V, E)是一个图，H=(W, F)是一个子图，则G的所有顶点都属于W，F的所有边都属于E。即G中的顶点子集V和边子集E都属于H。当W为空集时，称为一个空图。

### 2.2.7 割
割：在无向图G=(V, E)中，令S为顶点集合，T为V-S的集合，如果T中每个顶点都可以从某些顶点到达另外一些顶点，并且这些顶点都属于S，那么称S是割，G中称T为被割的顶点。

### 2.2.8 意义顶点与独立集
意义顶点：设G=(V, E)是一个图，若从G的一部分顶点到达另一部分顶点，则称那些可以到达的顶点是图G的意义顶点。
独立集：设G=(V, E)是一个图，S为任意集合，如果G中没有回路的子图，则称集合S是G的独立集，反之则不是。

### 2.2.9 森林
森林：是指有多个互不相交的树的集合。森林中的每棵树均是一颗无根树，且它们共享一些顶点，但是其他顶点是孤立的。森林可以用来描述非常复杂的结构，比如社交网络中的关系网、组织结构的层级结构、产品间依赖关系的网络等。

## 2.3 图的分类
根据图的邻接矩阵或邻接表来进行分类。

### 2.3.1 完全图
完全图：是指图中任意两个顶点之间都有边。例如，在有n个顶点的图中，有n*(n-1)/2条边。因此，完全图一定是连通的。

### 2.3.2 简单图
简单图：是指图中任意两个顶点之间只有一条简单路径相连。例如，一个有向图中不存在顶点v和u同时到达顶点s的路径，则称该图为简单图。同样地，一个无向图中不存在顶点v和u同时到达顶点s的路径，则称该图为简单图。

### 2.3.3 权值图
权值图：是指图的边上带有权值的图。

### 2.3.4 平面图
平面图：是指边都是直线的图。

### 2.3.5 稀疏图
稀疏图：是指图中边的数量远小于顶点的数量的图。

### 2.3.6 无向图
无向图：是指图中每条边都有方向的图。

### 2.3.7 连通图
连通图：是指图中任意两个顶点之间都有路径相连的图。

### 2.3.8 弱连通图
弱连通图：是指图中某些顶点之间存在路径相连的图，但是还有某些不连通的区域。

### 2.3.9 非连通图
非连通图：是指图中所有顶点都不是连通的图。

### 2.3.10 二部图
二部图：是指在一个无向图中，所有的边都分为两类，每类边至多出现一次。换句话说，它是两个子图的和。
