                 

# 1.背景介绍


如何构建符合业务需求的人际关系网络，是企业管理者面临的一项重要课题。在实际工作中，许多企业会选择使用复杂的人际关系系统架构设计方案，如矩阵式、星型或雷达型网络。然而，这种模式并不能完全解决企业的实际需求，需要考虑多种因素，包括个人职能的匹配、信息沟通效率、资源交流等方面。为了提升业务能力、更好地管理团队成员之间的关系、优化资源分配，我们需要进行有效的人际关系构建，并且该网络架构能够具备一定弹性、灵活性。因此，结构化思维可以帮助我们开发出具有竞争力、满足业务需求的人际关系网格。

人际关系是一个涉及到互动的双向关系，比如亲密、合作、疏离等。结构化思维就是指按照预先设定的标准，将人际关系划分成若干层次的金字塔架构，从而达到从不同层级到整体的全面视图。这种思维方式的应用主要是通过对人的属性、角色、技能、目标等进行分析，从而形成具有影响力的连接图谱。通过构建如此人际关系网格，可以有效组织团队成员间的协同工作、相互促进、以及员工培养。结构化思维还可以用于招聘、薪酬福利、晋升等领域，提供高质量、客观的职业发展参考。

# 2.核心概念与联系
## 2.1 基本概念
### 2.1.1 概念
- 顶层(Top)：指最外侧的组织，通常是指公司。
- 底层(Bottom)：指最里侧的组织，通常是指部门或者个人。
- 上层(Upper)：指直接隶属于上一级组织的下属组织。
- 下层(Lower)：指直接拥有下属组织的上级组织。
- 领导（Leader）：指负责组织管理方向、控制工作流程、发号施令的个人或小组。
- 关键人物(Key Personnel)：指至关重要的员工，其在组织中的作用决定了组织的成功与否。
- 优势资源(Advantageous Resources)：指能够产生重大价值的资源。
- 任务承接者(Task Acceptor)：指能够胜任某项任务的人员。
- 潜在问题(Penetrating Problems)：指由于组织人手不足、人员素质较低或制度导致的问题。
- 工作单元(Working Units)：指根据工作职责、功能不同，划分出来的不同的工作小组或团队。
- 能力(Capability)：指员工在工作中所担负的职责能力，包括专业知识、技能、经验、业务理论和素质、理解力、处理问题的能力、沟通协调能力、学习和创新能力、团队精神和激励机制等。
- 冲突(Conflict)：指两个或多个组织成员之间存在着的不正当、不平等、权利纠纷或道德问题。
- 资源(Resource)：指能够产生经济价值或者社会价值的活动、事物、制品、服务或资产。
- 关系(Relationship)：指两个或多个组织成员之间的上下级、上级、下属、附属、合作者等关系。
- 边界(Boundary)：指管理、信息或资源边界线。

### 2.1.2 关系类型
- 朋友关系：通常在两个单位之间可以认识，而且彼此熟悉且深知对方的业务。
- 同事关系：指两个单位有一定职务相同的中层人员。
- 供应商关系：指各单位之间可以互通有无的信息，互助合作。
- 服务关系：指两个单位直接建立联系，提供服务。
- 同侪关系：指单位内部的人员之间互惠互利、平等交流。
- 客户关系：指单位与单位之间的购买关系，并由相关人员在一定程度上进行管理。
- 合作关系：指单位之间共同开展某些项目，或交换某些产品或服务。
- 撤单关系：指两个单位之间的某些交易关系被终止。
- 利益关系：指两个单位之间或同一单位内的人员之间的利益纠纷。
- 归属感：指个人对自己的归属感，比如自身的身份、资产、影响力或地位。
- 信号情报：指个人对某个主题或组织有特殊兴趣或意义。
- 喜好关系：指个人对特定产品或服务具有极大的热情或依赖。

## 2.2 金字塔架构
### 2.2.1 金字塔模型

1. 第一层（最高层）: 通常是企业组织的最高决策层。此层的主要工作是制定战略、规划长期发展，以及创造条件促使企业实现增长。这一层中往往会出现一个或几个领导人物，其通常拥有非常强的领导才能，并有着领导全公司的超级优势。
2. 第二层（高层）: 此层一般由管理层、财务、人力资源、行政、市场营销、技术支持等部门组成，此层工作的主要内容是协调各个子部门的日常工作，同时提出建议、制定计划，并对相关工作做出反馈。
3. 第三层（中层）: 中层通常包括研发、业务、营销、信息技术、生物医药、供应链管理、法律等部门，这些部门扮演了支撑企业发展的关键角色，往往拥有多名关键领导。
4. 第四层（基层）: 基层主要包括生产、销售、仓储、采购、零售、人力资源、制造、物流、质控、工程建设、水利建设等部门，在企业发展的各个环节起到了至关重要的作用。

### 2.2.2 金字塔原则
1. 识别上下级关系: 金字塔结构通常呈现一幅等级关系的树状图。每一个层级都只会处于下层之上，只有那些直接上司有资格填补空缺。
2. 通过事实证明而不是利益博弈: 在金字塔结构中，成功往往不是凭借个人私利的简单复制，而是基于事实的支撑。任何层级的改革都必须确保它符合上一级、下一级及整个组织的共同利益。
3. 设定边界: 边界越宽则越容易出现腐败行为。将主要资源集中在核心部门，避免产生不必要的不良影响。设置适当的评估标准，识别和消除薄弱环节。
4. 透明度: 金字塔结构必须是清楚易懂的，所有层级都须参与，且每个层级都必须知道自己工作的目的。
5. 制度化管理: 层级制度化代表着一种规范化的、制度化的管理方法，其目的是为了有效地分散权力，降低复杂度。同时，层级制度化也是防止组织腐败的有效措施。
