# 2.核心概念与联系
## 什么是库？
在计算机科学中，库（Library）是指提供某种功能或实现特定逻辑的代码集合。库可以用来解决某些重复性的任务或者是简化编程过程。例如，`math`库提供了基本的数学运算函数，`image`库提供图像处理相关的函数，`http`库用于构建Web服务等。

## 为什么要用库？
通过库，我们可以快速地完成程序开发，避免重复造轮子，节约时间和资源。正因为如此，越来越多的人开始使用开源库来解决日益增长的工程问题。比如，Python的生态圈里就有很多强大的库，包括数据分析、机器学习、Web框架、网络爬虫等。

所以，如果想成为一个全栈工程师，掌握一定的编程技能和使用的库至关重要。

## Go的库

除此之外，还有大量的第三方库也在蓬勃发展。它们都是基于Go语言实现的，并且都遵循Go的规范。第三方库可以帮我们节省很多时间和精力。因此，了解和掌握Go语言的库，也是非常必要的。

## 示例：三角形的面积计算
假设我们有一个三角形的三个边长，分别为a、b、c。则该三角形的面积S等于斜边两边之比：

```go
import (
    "math"
)

func area(a float64, b float64, c float64) float64 {
	s := (a + b + c) / 2      // 斜边
	area := math.Sqrt(s * (s - a) * (s - b) * (s - c))    // 面积
    return area
}
```

说明：
- `math/sqrt`库用于计算平方根。
- `(s*(s-a)*(s-b)*(s-c))`为海伦公式。
- 函数`area()`接收三个参数，返回面积。

# 4.具体代码实例和详细解释说明
我们可以继续使用上述例子，实现判断一个给定数是否是素数。

```go
package main

import (
    "fmt"
)

// 判断一个数是否是素数
func isPrime(n int) bool {
    if n <= 1 {
        return false
    } else if n == 2 || n == 3 {
        return true
    } else if n%2 == 0 || n%3 == 0 {   // 排除特殊情况
        return false
    }

    i := 5
    for i*i <= n {        // 只需要检查从5到√n的数字
        if n%i == 0 || n%(i+2) == 0 {
            return false
        }
        i += 6               // 每次加6，跳过偶数
    }
    return true
}

func main() {
    fmt.Println("判断7是否为素数:", isPrime(7))     // true
    fmt.Println("判断19是否为素数:", isPrime(19))    // false
}
```

说明：
- 通过函数`isPrime()`判断一个整数是否为素数。
- 如果某个整数n不是质数，那么它一定可以表示为6k ± 1形式，其中k为任意整数。
- 我们可以通过判断两个条件：第一个是n不能被2整除；第二个是n不能被3整除。
- 此外，还需要遍历从5到√n的奇数，查看是否存在能够整除n的因子。
