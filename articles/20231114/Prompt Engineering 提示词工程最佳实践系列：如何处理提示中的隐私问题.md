                 

# 1.背景介绍




## 一、什么是提示词？

提示词（Prompt）是在AI系统生成文本时，会给出一些关键字或者模板，引导用户完成输入，帮助用户更准确地描述自己的需求，从而可以提高生成的文本的质量和易读性。提示词就是系统预设好的文本输入的框架，可以通过不同的提示词来指定用户所要表达的内容。比如，机器翻译系统的提示词是“请您用中文或英文进行翻译”，对话系统的提示词是“说点什么吧”等。

由于提示词本身也是一个输入项，当用户实际输入的内容跟提示词不符时，就会造成信息遗漏或者错误，甚至导致结果出现偏差。因此，作为系统设计者和开发者，我们需要对提示词相关的安全问题有清晰的认识，并通过措施来保障用户隐私信息的安全。


## 二、为什么需要保障提示词中的隐私问题？

提示词是一种在AI系统中用来引导用户完成输入的框架，但其本身也是信息输入的一部分，如果缺乏足够的保护措施，可能会对用户隐私产生危害。举个例子，假设一个面向公众的对话系统，它使用提示词“你好，我是XX公司的XX产品，可以为您做些什么？”，用户可能对此完全无感，因为他只知道这个消息的框架，却无法得知真正想要什么，因此，在这种情况下，很容易就泄露了隐私信息。

另外，在提示词中，也存在着其他隐私数据，如用户姓名、联系方式、个人信息等。这些数据对于保护用户的隐私非常重要，如果能够有效地保护它们，将极大地鼓励用户主动透露自己的个人信息，促进自我探索，提升产品的便利性和吸引力。


## 三、提示词中的隐私问题

提示词中存在以下隐私问题：

### （1）恶意广告

提示词中可能会放置恶意广告，造成恶意流量投放。例如，某台视频网站的AI助手可能会提供短视频编辑服务，而其提示词就可能包括“快速创作时尚小视频”。恶意广告可能诱导用户点击链接下载软件，从而获得广告收入；或将购物链接贴到提示词上，骗取用户的信任和付费意愿，导致欺诈交易。为了避免这一问题，建议AI平台在提供提示词的时候，应当充分注意控制广告联盟的影响，及时检测广告效果并进行调整。

### （2）低级语言

提示词通常使用较低级的语言，可能会泄露用户的个人信息。例如，对于某电商网站，提示词可能使用生僻的汉语，而用户却可能不会理解其含义，因此，将商品名称用汉语拼音写在提示词中可能引起误会。为了提高提示词的易懂性，建议尽量采用英语或法语的形式。

### （3）默认推销

提示词往往具有高度营销效果，例如某搜索引擎会根据提示词推荐相关的搜索结果，所以，不宜过于简洁地填写提示词，而应该添加有意义的信息。否则，会造成误导，影响用户体验。

### （4）不准确表达需求

提示词中往往会包含不准确的描述，如某在线客服系统可能会询问用户“请您告诉我您的具体要求”，而不是“请输入你的需求”，这样可能导致用户不明白自己想要什么。为了避免出现这种情况，建议合理安排提示词，并为用户提供清晰明了的文字描述。

### （5）逃避风险

提示词本身并非全无风险，但是，当用户不仔细阅读时，可能会忽略或怀疑其中隐私数据，从而造成风险。因此，我们需要在提示词的提示语中，包含清晰的风险提示，例如“请不要向陌生人分享该账号密码等敏感信息”。

除了上述隐私问题之外，提示词还有以下安全风险：

### （6）盗用提示词

攻击者可以使用虚假的账户或伪装身份发布虚假的广告，将自己的隐私信息植入到提示词中，通过不正当的方式获取用户的信任和关注，从而达到攻击的目的。为了避免这种行为，建议AI平台建立可信赖的第三方审核机制，确保用户对广告信息的真实性和准确性负责。

### （7）恶意回报

恶意广告和虚假宣传可能损害用户的积极性和自尊心，进一步导致用户选择退出产品或服务，造成系统损失。为了防止恶意广告带来的损失，建议在展示广告前，充分考虑用户的隐私权，避免向用户索要个人信息。
