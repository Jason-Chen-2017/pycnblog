# 2.核心概念与联系
## 2.1 什么是变量？
计算机中存储信息的最小单位叫做“bit”，一个bit只能表示0或1两个状态，而一般的变量却可以表示各种不同的类型和大小的信息，比如整数、浮点数、字符串等等。在计算机程序中，为了方便程序员管理复杂的数据，可以使用“变量”对不同的数据进行命名，并赋予它们一个特定的用途和含义。在Python中，变量是一个非常重要的概念，也是最基础的一个概念。它的定义形式如下：
```python
variable_name = value
```
其中，`variable_name`是变量的名字，通常用下划线或者驼峰命名法（首字母小写，后续单词每个字母的首字母大写）；`=`是赋值符号，用于将右侧的值赋给左侧的变量。`value`是变量所持有的具体值。
## 2.2 数据类型
在计算机中，每种数据都有自己的特殊符号来表示，这些符号统称为“数据类型”。计算机程序需要知道数据类型才能正确处理它，否则就无法正常运行。Python支持的数据类型有以下几类：
- 整型：int，包括正整数、负整数和零
- 浮点型：float，包括有理数和无理数
- 布尔型：bool，只包含True和False两种值
- 字符串型：str，由任意数量的Unicode字符组成，用单引号、双引号、三重单引号括起来
- 列表型：list，由一系列按顺序排列的元素组成，用方括号括起来，元素之间用逗号分隔
- 元组型：tuple，同样也是一系列按顺序排列的元素，但是不可修改，用圆括号括起来，元素之间用逗号分隔
- 集合型：set，类似于列表但元素没有重复，用花括号括起来，元素之间用逗号分隔
- 字典型：dict，也称为关联数组（associative array），由键-值对组成，用花括号括起来，键和值的分隔符号就是冒号(:)
除以上五种数据类型外，还有一些其他数据类型，如文件、类、对象等，但在Python中，对它们的支持还不是很完善。因此，了解基本的变量和数据类型对于学习Python是至关重要的。
## 2.3 变量的分类
变量按照定义方式可以分为两类：
- 局部变量（Local Variable）：函数内部定义的变量，只能在函数体内访问，离开函数就失去作用，没有全局变量那样的作用域限制
- 全局变量（Global Variable）：定义在函数外部，所有函数都可以访问，拥有全局作用域，可以在整个程序中使用
根据变量的作用范围，可以把变量分为四种：
- 本地变量：函数体内定义的变量，函数调用结束时自动销毁，具有块级作用域
- 参数传递：函数调用时将实际参数值传进函数内部的临时变量，具有块级作用域
- 全局变量：直接在模块级别定义的变量，可以被任何函数、模块访问，具有全局作用域
- 模块变量：模块内定义的变量，只对当前模块有效，如果其他模块导入这个模块，则无法访问，具有全局作用域
## 2.4 变量的生命周期
由于计算机内存的限制，在一定时间内，一个变量所占用的内存空间不能超过某一阈值，超出这个阈值就需要回收内存。不同的数据类型，生命周期又有所不同：
- 整数、浮点型：值不会改变，生命周期始终保持在某个特定的数字范围内，不会超出范围。一般情况下，变量的生命周期可以简单认为是该变量在程序中的寿命，即从创建直到消亡。
- 布尔型：值也不会改变，只会在创建时被初始化。
- 字符串型：Python中的字符串属于不可变类型，值创建后便不可更改，除非重新分配新的内存空间。生命周期随着程序的执行而变化，不同位置的相同字符串名，其生命周期也可能不同。
- 列表、元组、集合、字典：这几种数据类型的值在创建时就会固定下来，不会改变，只有当引用变量指向另一个不同的值时，变量才会重新指向新的内存地址。生命周期取决于数据的长度和元素的变化频率。
# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 数字运算
Python可以支持四种基本运算符：加减乘除、整除、余数、自增自减运算符。
### 3.1.1 加法
Python中的加法运算符为`+`，例如：
```python
print(2 + 3) # output: 5
```
### 3.1.2 减法
Python中的减法运算符为`-`，例如：
```python
print(7 - 3) # output: 4
```
### 3.1.3 乘法
Python中的乘法运算符为`*`，例如：
```python
print(2 * 3) # output: 6
```
### 3.1.4 除法
Python中的除法运算符为`/`，如果是整除，结果会向下取整，例如：
```python
print(9 / 3) # output: 3.0
```
如果要计算商和余数，可以使用求模运算符`%`。注意，求模运算符返回的是第一个数除以第二个数的余数，而不是求第二个数除以第一个数的商，所以前面的例子应该这样写：
```python
a = 9 // 3    # a = floor(9/3)
b = 9 % 3     # b = 9 mod 3
print(a, b)   # output: 3 0
```
或者使用`divmod()`函数，它返回一个包含商和余数的元组：
```python
quotient, remainder = divmod(9, 3)
print(quotient, remainder)  # output: (3, 0)
```
### 3.1.5 求幂运算
Python中求幂运算可以通过`**`运算符实现，例如：
```python
print(2 ** 3)  # output: 8
```
### 3.1.6 自增自减运算符
Python提供了两个运算符来对变量进行原子性的增量和减量操作，分别为`+=`和`-=`，例如：
```python
x = 2
y += x  # y = y + x
print(y)  # output: 4
```
```python
x -= 1  # x = x - 1
print(x)  # output: 1
```
注意，`+=`和`-=`运算符只能与整数类型相结合，对其他类型数据进行操作时可能会报错。
## 3.2 比较运算符
Python提供比较运算符用于判断两个表达式的值是否相等、不等于、大于、大于等于、小于、小于等于。比较运算符包括：
### 3.2.1 == 和 is 的区别
在Python中，==和is的区别主要在于对象的标识是否相同，二者都可以用于判断两个变量是否相等，但二者的区别如下：
- `==`：表示值相等，比较两个对象是否具有相同的数值、字符串、列表等等。
- `is`：表示对象的标识相同，比较两个对象的ID是否相同。
用一句话总结，如果想判断两个变量是否具有相同的内容，应该优先考虑用`==`运算符。如果只是想判断两个对象是否同一个对象，可以用`is`运算符。
### 3.2.2!=、>、>=、<、<= 的区别
Python中，!=表示不等于，>表示大于，>=表示大于等于，<表示小于，<=表示小于等于。这些运算符的语法与C语言中的对应运算符一致。
### 3.2.3 in 和 not in 的区别
Python中的in和not in运算符用于判断一个值是否存在于某容器内，其语法如下：
```python
value in container
```
这里，`container`可以是列表、元组、集合、字典等任意序列容器。和C语言不同的是，Python中的in运算符还支持字符串的成员测试。
```python
s = 'hello world'
if 'l' in s:
    print('found it!')
else:
    print("didn't find it")
```
输出：
```
found it!
```
用not in也可以判断一个值是否不在某容器内：
```python
value not in container
```
和in相反，not in返回布尔值True或False，根据条件决定后续程序的执行。
### 3.2.4 and 和 or 的区别
Python中，and和or的优先级相同，但作用不同。and的短路求值规则表示，如果第一个表达式为False，则不再检查第二个表达式，直接返回False；或如果第一个表达式为True，则忽略第二个表达式，直接返回True。or的短路求值规则表示，如果第一个表达式为True，则忽略第二个表达式，直接返回True；若第一个表达式为False，则不再检查第二个表达式，直接返回False。and和or均可用于布尔运算。
## 3.3 赋值运算符
Python提供多个赋值运算符，包括：
### 3.3.1 简单的赋值运算符 =
Python中的赋值运算符为`=`，用于将右侧的值赋给左侧的变量，例如：
```python
x = 2
y = x
print(y)  # output: 2
```
此处，变量`y`已经被赋予了一个初始值，所以等效于：
```python
x = 2
y = 2
print(y)  # output: 2
```
### 3.3.2 交换赋值运算符 <==>
Python还提供交换赋值运算符`<=>`，其作用是交换两个变量的值，并返回交换后的值，语法如下：
```python
<left variable> <==> <right variable>
```
例如：
```python
x = 2
y = 3
print(x <==> y)  # output: 3
print(y <==> x)  # output: 2
```
```python
x = "abc"
y = "def"
print(x <==> y)  # output: def
print(y <==> x)  # output: abc
```
交换赋值运算符可以用于交换两个变量的值，并不需要第三个变量，因此对于赋值语句来说，其优势在于可以简化代码。
### 3.3.3 可选地使用多个赋值
Python允许将多个变量一次性地赋予多个值，这在赋值较多变量或值得时候非常方便，例如：
```python
a, b, c = 1, 2, 3
print(a, b, c)  # output: 1 2 3
d = e = f = g = h = i = j = k = l = m = n = o = p = q = r = s = t = u = v = w = x = y = z = 1
print(w, x, y, z)  # output: 1 1 1 1
```
### 3.3.4 混合使用赋值运算符
Python允许混合使用赋值运算符，包括加法赋值运算符`+=`、减法赋值运算符`-=`、乘法赋值运算符`*=`、除法赋值运算符`/=`等。这些运算符可以方便地对变量进行运算和赋值操作。例如：
```python
x = 2
x *= 3   # x = x * 3
print(x)  # output: 6
x /= 2   # x = x / 2
print(x)  # output: 3.0
```
