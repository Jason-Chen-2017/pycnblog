                 

# 1.背景介绍


## 人工智能的发展历史回顾
​    在现代社会里，人工智能已经成为不可或缺的一部分，它不仅对我们生活中的方方面面产生了深远影响，也影响着我们的工作、学习、社会活动甚至艺术创作等各个领域。从人工智能的发明到今天，它的发展过程经历了三个阶段：蒸蒸日上、繁荣昌盛、衰落颓败。

- **蒸蒸日上期**

  信息革命时期。互联网的普及推动了人类的知识产权和生产力的飞速发展，人工智能研究、开发与应用迎来蓬勃发展时期。人工智能可以自动进行数据分析、图像识别、语音识别、自然语言处理等众多领域的任务。

- **繁荣昌盛期**

  中国特色的工业革命推动了人工智能研究与应用的高速发展，最著名的是通过深度学习（Deep Learning）算法提升计算机视觉、自然语言理解能力，从而实现图像分类、视频分析、文本生成、机器翻译、人脸识别、姿态估计、语音合成等一系列应用。

- **衰落颓败期**

  2017年乔布斯向人们展示了iPhone X和Apple Watch之后，在全球范围内人工智能的研发投入下滑严重，2019年“世界经济奇迹”即将到来的时候，人工智能的发展势头已然变缓，有关部门正在加强监管，政府开始全面禁止利用人工智能技术进行违法犯罪，未来人工智能的发展前景出现危机。


## 大型生物医学图像数据库构建

随着医疗大数据的迅速增长、深度学习技术的广泛应用，越来越多的医学图像数据集、标注工具和工具链被开发出来。为了促进医学图像数据分析的发展，目前主要用于医学图像的数据源包括免费开放共享的生物医学图像数据库，包括以下几类：

1. 公共数据集。

2. 公立医院提供的公共医学影像数据库。

3. 第三方提供的生物医学图像数据库。

4. 从网上爬取的生物医学图像数据。

5. 模拟器生成的医疗模拟图像数据。

目前我国已建成的免费开放共享的生物医学图像数据库有：

1. 中山大学中华神经精神病学中心共享医学影像资源库

2. 河南省内江市盆地人民医院共享医学影像资源库

3. 浙江省杭州市萧山区天目山妇幼保健院医学影像资源库

4. 北京市平谷区佩奇胡同新华路医院共用镜数据集

5. 华东师范大学中山医院普通外科医疗影像数据集

目前建成的公立医院提供的公共医学影像数据库包括：

1. 河北省廊坊市安次区敦煌医院共享医学影像资源库

2. 河南省三门峡市湖滨区三门峡医院共享医学影像资源库

3. 广西壮族自治区南宁市青秀区广西医科大学公共影像资源库

4. 湖南省株洲市攸县黄金医院医学影像资源库

5. 黑龙江省哈尔滨市道里区吴桥医院医学影像资源库

除此之外，第三方提供的生物医学图像数据库还有：

1. 美国国家乳腺癌协会 (National Cancer Institute) - National Cancer Electronic Collaborative (NCEC) SHARED MRI Database. 

2. 爱丽丝梅西教授团队提供的肿瘤图像共享数据库 “Prostate Cancer Image Share”. 

 

这些医学影像数据库都提供了医疗图像数据的基础设施支持，包括数据下载，数据存储，数据管理，数据分发，图像预处理，图像特征提取，图像表示学习，图像分类，模型训练等模块。而这些医学图像数据都是公开可用的，用户可以在没有权限或者限制的情况下自由使用。因此，如何建立一个大型生物医学图像数据库并提供生物医学图像相关的科研、开发和应用服务是关键。
