# 2.核心概念与联系
## 2.1.云计算（Cloud Computing）
云计算是一种将本地计算资源或数据中心连接到网络的计算服务。它利用分布式、动态资源池的方式，能够按需获取计算能力，实现“按使用付费”的计费模式。其基本特征包括：

- 分布式：云计算通过将用户的数据、应用和计算基础设施分散地储存于不同的地点，可以让用户拥有可靠的计算资源。

- 动态资源池：云计算通过快速扩张与收缩计算资源的方法，有效利用资源，提升效率。

- 按需获取计算能力：云计算允许用户根据需要购买计算能力，不需要预先支付总容量，降低成本，节约开支。

- “按使用付费”：云计算采用按使用付费的方式，用户只需要按实际使用量付费即可，同时云计算平台会自动优化资源配置，保证计算资源的稳定性。

云计算的优势主要体现在以下方面：

1. 降低成本

   云计算的另一个优势就是降低了成本。无论是在硬件上购置服务器还是软件上托管云服务，都可以在大幅度降低运行成本。当公司的业务量增加时，用户就可以享受到云服务的免费试用或购买服务的折扣。

2. 提升灵活性

   在云计算中，用户只需要按使用量付费即可，就算某些时候业务量骤减，也可以按照实际使用量付费。这就为用户提供了更多的灵活性，可以随时调整自己的计算资源。

3. 灵活应对突发状况

   当突发事件发生时，云计算平台可以立即快速扩展资源，满足用户的业务需求，提供更好的服务质量。

4. 降低IT风险

   用户不再需要关注底层服务器的维护、安全等问题，而是可以专注于业务，充分利用云计算平台提供的服务。

## 2.2.云存储（Cloud Storage）
云存储是指把数据放到一系列的云端服务器上进行管理，让用户直接通过网络访问这些数据。云存储可分为静态存储、动态存储、流媒体存储三种类型。其中，静态存储又分为基于文件系统的云盘和基于对象存储的云存储，动态存储是指实时上传、下载数据的存储服务；流媒体存储则是对流媒体内容的存储。

### 静态存储
静态存储是指将用户的文件或数据存储到云端服务器中，并通过网络访问，实现文件的传输和存储。它的特点是简单易用、高可靠性、低成本、支持多种编程语言、开发工具包丰富。例如，亚马逊S3、微软Azure Blob、Google Cloud Storage都是典型的静态存储产品。

#### 文件系统存储
基于文件系统的云盘的特点是支持多种编程语言、开发工具包丰富，适用于小型文件共享网站、个人私密照片空间等场景。但是，由于每个用户都需要独立的账号和权限管理，管理起来较为复杂，同时也存在数据冗余、备份难度大的问题。因此，目前的文件系统云盘还不能真正成为企业级解决方案。

#### 对象存储
基于对象存储的云存储是指将用户的文件或数据存储在云端服务器中，并给予唯一标识符(Object ID)，通过网络访问。它具有高可靠性、低延迟、易扩展性、按需付费等优点。对象存储可以实现文件、图片、视频等数据的存储、检索、搜索功能，应用领域广泛。亚马逊的S3、微软Azure Blob、百度BOS、腾讯COS、金山云KS3、网易云音乐云音频、Tencent COS、网盟对象存储、Ucloud UFile、AWS S3、Aliyun OSS、Qiniu Kodo等都是典型的对象存储产品。

### 动态存储
动态存储是指实时上传、下载数据的存储服务。它一般由云计算提供商和客户端(如浏览器、应用程序)共同提供。当用户需要上传数据到云端时，可以实时将数据上传到云端服务器，并通过网络访问。与静态存储相比，动态存储具有更快响应速度、低延迟、高容量、低成本等优点。例如，亚马逊的Glacier、Google的Firebase Storage、微软的Azurite、腾讯的TCAP、金山的KS3MS、网易云音乐的NOS等都是典型的动态存储产品。

### 流媒体存储
流媒体存储是指对流媒体内容的存储。它可以帮助用户实时观看直播电视或点播视频。云存储可以将原始视频或音频编码、转封装后，保存至云端服务器中，并实时上传至客户的浏览器或设备上进行播放。流媒体存储产品可以使客户快速接入高清视频、超高速率流媒体等资源。例如，腾讯的CMKE、网易云音乐的Cloud Live、新浪微博的直播CDN等都是典型的流媒体存储产品。

## 2.3.云数据库（Cloud Database）
云数据库是指通过云端服务器实现的关系数据库服务，它存储着海量的数据，可以方便的访问、分析和处理。云数据库可分为托管型数据库和专业型数据库两种类型。托管型数据库是指云计算服务商负责提供、运维数据库服务，例如亚马逊的RDS、微软的SQL Azure、Google的Cloud SQL等；专业型数据库则是指云计算服务商推出面向行业应用的数据库产品，例如微软的Azure Cosmos DB、百度的表格存储、腾讯的TDSQL、阿里云的文档数据库、华为云的对象存储等。

### 托管型数据库
托管型数据库的特点是简单易用、弹性伸缩、自动备份、降低运营成本、提供全球化服务。由于云计算服务商控制服务器，用户不必担心数据安全、性能及可用性问题。它可以满足各种规模的应用场景，并且价格便宜廉，对于中小型企业来说是个不错的选择。国内外知名的云厂商目前提供的托管型数据库产品，如亚马逊的RDS、微软的SQL Azure、Google的Cloud SQL等，均属于此类。

### 专业型数据库
专业型数据库的特点是提供行业内最佳的服务质量和性能，包括更高的并发读写请求、更完善的安全防护、更低的运营成本、更便捷的运维管理等。例如，微软Azure Cosmos DB、百度的表格存储、腾讯的TDSQL、阿里云的文档数据库、华为云的对象存储等产品属于此类。由于具备更强的性能及更高的可用性，这些数据库产品可以满足对高吞吐量、高并发访问要求的应用场景。

1.云存储

云存储(Cloud storage)是把数据存储到一系列的云端服务器上进行管理，让用户直接通过网络访问这些数据。主要有两种存储方式：静态存储和动态存储。

2.静态存储

静态存储是指将用户的文件或数据存储到云端服务器中，并通过网络访问，实现文件的传输和存储。主要有两种形式：文件系统存储和对象存储。

文件系统存储：基于文件系统的云盘的特点是支持多种编程语言、开发工具包丰富，适用于小型文件共享网站、个人私密照片空间等场景。目前的文件系统云盘还不能真正成为企业级解决方案。

动态存储是指实时上传、下载数据的存储服务。其一般由云计算提供商和客户端(如浏览器、应用程序)共同提供。当用户需要上传数据到云端时，可以实时将数据上传到云端服务器，并通过网络访问。主要有三种形式：块存储、文件存储、消息队列存储。

块存储：块存储是指通过云端服务器实现的块级存储，即用户可以上传任意大小的数据块。它适合上传超大文件、视频等。云计算服务商提供的块存储服务主要有亚马逊的Glacier、Google的Firebase Storage、微软的Azurite、腾讯的TCAP、金山的KS3MS等。

文件存储：文件存储是指通过云端服务器实现的云端文件存储，即用户可以上传任意大小的文件。云计算服务商提供的文件存储服务主要有微软的Azure File、网易云音乐的NOS等。

消息队列存储：消息队列存储是指通过云端服务器实现的消息队列存储，即用户可以实时上传、下载消息队列中的数据。云计算服务商提供的消息队列服务主要有阿里云的MNS、腾讯云的CMQ等。
