                 

# 1.背景介绍


人类的认知过程，往往是由感官、视觉、触觉、味觉等不同感知系统相互作用和协作共同完成的。当一个事物发生时，经过众多感官体系的交流，信息会被加工处理形成对外界的呈现，再通过语言的发出转换为信息。这是一个复杂而多层次的过程，每个人的认识都不尽相同。如何准确、全面地捕获这种交流信息，并把握其中的逻辑关系，成为许多领域的研究热点。结构化思考就是从大局出发，运用自己的知识和智慧，把复杂的信息和事件分解为一些易于理解和分析的组块，进而找寻其中的关键因素、原因、结果之间的联系和演化规律。
在生活中，我们遇到的各种事件其实都可以视作一系列的因果关系的组合。比如，我们开车上班时遇到路口拥堵情况，自然就会想到这可能与拥堵车辆密度有关。那么，有哪些因素可能导致车辆拥堵呢？可能会包括交通设施的老化、车辆的废气排放、司机对车道的畸变程度、驾驶技巧和意识。一旦找到了这些因素的影响因子，就可以按照结构化的方式组织起来，找寻它们之间所起到的作用关系，找出最根本的原因和结果。而结构化思考的核心方法就是用因果图进行因果分析。

# 2.核心概念与联系
## 2.1 什么是结构化思考？
结构化思考（Structured Thinking）是指将复杂的问题或问题解决方案分解为简单的组块，并通过各个组块的联系和关联分析出产生原因和结果的最终目的。该方法是基于模式观念提出的，是一种“以抽象为基础，由问题驱动直到问题的终极解决”的方法。它旨在应用模式的思维方式来分析问题，从而找出关键因素、原因、结果之间的联系和演化规律。
## 2.2 为什么要使用结构化思考？
在日常生活中，因果关系就像磁铁一样，任意两个相互作用和影响的元素之间都会存在着一定的关联性。因此，借助结构化思考，我们可以有效地找寻问题的关键原因，掌握问题的走向，从而准确预测问题的发展方向。同时，结构化思考也是对复杂现象进行概括和总结的有效工具。此外，还能发现问题的新模式、新趋势，为进一步改善问题的解决方案奠定坚实的理论基础。
## 2.3 如何应用结构化思考？
使用结构化思考主要需要遵循以下步骤：

⒈ 将复杂问题划分为不同的组块，找出其之间的关联关系；

⒉ 通过各个组块的关联关系来分析为什么某个组块具有更大的影响力；

⒊ 找出最终原因和结果；

⒋ 提炼出新的假设和预测。

具体操作步骤如下：

### （1） 将复杂问题划分为不同的组块
将复杂问题分解为多个组块的方法也称为“事物拆分法”，即将问题按需细化，逐步缩小，最终得到一个简单的模型。一般来说，我们可以尝试将问题分解为如下几类：

⒈ “基本因素”：影响问题发展的一个最小单元，如个人能力水平，环境条件，土地价格，劳动强度等；

⒉ “中间因素”：既不是基本因素又不是结果变量，但是在问题的发展过程中起到重要作用的因素，如技术优势，法律限制，市场份额，竞争者影响等；

⒊ “结果变量”：指出影响问题最终结果的要素，如收入、利润、贷款、投资回报等。

### （2） 通过各个组块的关联关系分析为什么某个组块更具备更大的影响力
确定各个组块之间的关联关系需要注意以下几点：

⒈ 反映出一个因果关系：当两个组块存在某种相关性的时候，就表明了因果联系。如劳动强度与收入之间的关系，竞争者影响与生产效率之间的关系，法律限制与企业盈利之间的关系等；

⒉ 情况与原因的影响：当某个组块扮演的是“状态”角色，即一个较大的变化会影响其他所有组块的时候，表明了“状态”与“原因”之间的影响关系；

⒊ 在一定范围内对比效果：当某个组块处于一定范围内，它的影响力越来越弱，但仍能够产生影响的时候，表明了“均衡”的影响关系；

⒋ 在一定范围内的分配效应：当某个组块处于一定范围内，在其他组块的影响下，可以产生相对较大的影响时，表明了“分配”的影响关系。

### （3） 找出最终原因和结果
分析完各个组块的关联关系之后，就可以找出整个问题的根源和结果。通常情况下，结果变量越大，对整个问题造成的影响就越大，我们可以利用这种模式来判断问题是否已经解决，即问题的主要矛盾是否已经解决。如果没有解决，则可以通过分析原因变量来判断目前存在的矛盾。

### （4） 提炼出新的假设和预测
借助结构化思考的模式匹配分析方法，我们可以逐渐把握关键因素、原因、结果之间的演化规律。根据这种分析方法，我们可以针对目前存在的困境，提炼出一些新的假设或者预测。这样做有几个好处：

⒈ 有助于科学化思维：通过科学化的假设，我们可以更好的预测问题的解决路径。

⒉ 可以增强认识水平：通过新的假设和预测，我们可以更加深入地探索问题，弥补认识上的不足。

⒊ 有助于创新决策：有了新的认识，我们可以重新定义问题，找出新的办法，从而寻求创新发展的可能性。
