## 背景介绍

### SpringBoot 简史

SpringBoot 是由 Pivotal 团队于 2014 年 6 月发布的全新产品，基于 Spring Framework 5.0 和 Tomcat 8 等技术栈，提供快速创建基于 Spring 的应用的开发框架。它通过启动器（Starter）、自动配置（AutoConfiguration）、嵌入式 Servlet 容器等特性，极大地降低了 Java 开发的复杂性和门
