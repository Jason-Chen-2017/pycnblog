# 1.简介
  

## 1.1 通信工程与通信网络工程是什么？
通信工程（Communication Engineering）是研究如何将信息从一个节点传输到另一个节点的工程。通信网络工程（Communications Network Engineering）是研究电信、光缆、互联网等计算机网络的设计、制造、优化及管理的一门学科。

通信工程的主要任务是研究各种通信系统及其性能、可靠性、安全性、经济性、成本、规模化等特性，并设计和分析它们的构造、结构、运作方式、控制方法、电路布线方法等。通信工程涉及的信息处理、信号处理、通信系统设计、电磁场与电流传播、网络调制、无线通信、信息传输、终端设备、网络管理、性能评估、电力系统建设、经济可行性评价等多个方面。

通信网络工程的任务是为了建立和维护电信、光缆、互联网等通信网络，并且根据需求进行实施。通信网络工程涉及通信网络基础理论、技术、系统、模式、方法、工具、标准及其应用。通信网络工程的目标是能够实现跨越频段、城市、地区和国家的低时延、高带宽、高可靠、高容量的信息服务，同时满足用户不同业务需要、应用特点和合法性要求。

## 1.2 本专业的学生在校期间能获得哪些职业培训？
- 学生职业建议书：一般是由各高校的院系或导师组织给予的职业建议书。它一般具有参考意义，可为学生做好职业发展规划提供依据。
- 中国职业资格认证中心有关培训课程：专业的相关职业指导课程可以帮助学生提升职业技能、了解相关职业的工作要求，掌握求职中的关键能力，以及改善职场环境。
- 中国计算机教育中心有关计算机技术职业培训课程：对于掌握通信工程所需的计算机语言、数据结构、算法等知识，具有很大的吸引力。

## 1.3 为什么要招收本专业的学生？
通信工程作为高校综合教育的一项重要分支，是大学综合素质教育的一部分。相比其他分支，通信工程更注重综合能力培养，对个人因素、社会因素、国际因素等多方面因素都有较强的影响，因此招聘这种专业的学生显得尤为重要。

随着我国通信网络发展蓬勃，特别是5G、WLAN、移动通信、边缘计算等新兴技术的发展，通信工程也应当跟上时代的步伐，以保证人才的竞争力不断提高。如果要成为一名优秀的通信工程师，必须具备良好的专业知识、技能、团队精神和科研经历，积极参加国内外知名大学的教学课程、参加国际会议、从事科研项目、竞赛，努力为国家创造新的经济价值。所以，招收本专业的学生，是为了使本专业达到世界一流水准，发挥本专业的优势作用，并增强专业的竞争力和国际竞争力。

## 1.4 选择本专业有什么优势？
- 拥有全面的知识面：通信工程是一门涉及信息交换、物理层、数据链路层、网络层、传输层、应用层等多个领域的学科，而这些领域的基础知识和理论都非常复杂。这也是为什么很多人选择了通信工程专业。
- 发达国家的需求：通信工程正在成为发达国家最需要的技能之一。中国在通信工程方面长期处于弱势地位，但随着我国数字经济的发展，需求量日益增加，需要在通信工程这个行业中发挥自己的主动性，将通信工程带入到与人工智能、生物医疗等相关的领域。
- 涉及科学领域：通信工程属于非常重要的科学方向，近几年来，由于通信领域技术的飞速发展，通信工程与微电子、纳米技术、芯片、激光、超声、红外、雷达、GPS、无线通信、雾霾等领域的合作也逐渐形成。
- 优质就业机会：由于通信工程的人才分布广泛，且国家鼓励留学回国留职，所以具有一定市场竞争力。一旦找到一个适合自己的职业，就可以获得丰厚的经济利益，享受到美好的生活。
- 全球顶尖学府：通信工程学科还在迅速发展，而且越来越依赖计算机科学、经济学、数学、物理学、统计学等基础学科，因此被誉为“计算机通信领域的通用专业”。另外，通信工程学科在世界范围内也有很大的影响力。
