# 1.简介
  
:Bioinformatics（生物信息学）是指利用计算机系统、数据库和算法对分子生命科学中的各种现象进行研究、分析及分类的领域。它从细胞结构、蛋白质组成、基因序列等多个角度对生命体进行研究，对了解人类疾病原因、行为调节、宿主适应性等方面提供宝贵的遗传信息。其目标就是通过技术手段建立可靠、准确、有效的生物信息学模型，用于预测及辅助诊断各种生物性疾病。

生物信息学的应用领域非常广泛，涉及到基础生物学、遗传学、分子生物学、细胞生物学、微生物学、信号 transduction（包括神经系统）、心理生物学、医学实验室、制药、基因工程等多种分支领域。而作为一个热门的学科，生物信息学也在不断地更新迭代。其最新的发展趋势是基于现代机器学习算法、大数据处理与存储技术、高性能计算能力的结合，构建复杂的生物信息分析平台和生物信息网络。

因此，掌握生物信息学知识对于具有较强医学、生命科学、数学、统计、计算机、工程等综合素养的人士来说都是一项必备技能。与此同时，还需熟悉相关的工程实践和临床实验流程，并具有扎实的数学、统计、计算机、编程基础才能真正把握生物信息学的精髓。所以，希望读者能够真正关注和关注生物信息学领域的最新进展，为他人提供更加全面的生物信息学知识。

2.基本概念术语说明：
# 1. 什么是序列？

   序列即一串字符或者数字构成的有序集合，它可以用于表征任何客观事物的一切性质，比如单个核苷酸的序列、染色体的序列、氨基酸链的序列、基因的序列等。一般情况下，相同物种的不同个体之间，各自的序列就会发生变化，但却属于同一种生物的类型。
   
# 2. 为什么要进行序列比对？

   序列比对是生物信息学的一个重要任务，因为序列信息往往是物种相异的不同个体之间共享的特性，而其具体的数值变化却可以通过比较来进行探索。如果没有序列比对手段，就无法对两条序列之间的差异进行归纳、理解和确认。
   
# 3. 活性序列（Protein Sequence）与非编码序列（Nucleotide sequence)
   
   活性序列是指由一系列氨基酸组成的蛋白质序列，每一个氨基酸都对应一个特定的功能作用，按照不同的分类标准可以分为两大类：蛋白质序列与蛋白质结构序列；后者包括蛋白质结构图、二维碳水化合物图（DSSP）、三维蛋白质晶体结构图。非编码序列是指DNA或RNA分子中除ATG外的其他所有碱基对。
   一般来说，编码DNA和RNA由5'端的ATG开始，通过ATCG-结尾的四环磷酸转移ases修饰结构，最后被连接起来。
   
# 4. 大分子（Protein Molecules）与小分子（Small Molecules）
   
   大分子即含有丝分裂所需的核糖体和糖腺嘧啶的分子，如人体组织中的维生素A、C、E、K、L等。它们由多种功能酶参与稳定生成，并参与了细胞内的各种信息传递过程。

   小分子，又称分子气体、细胞载体（Cellular Organelle），如溶酶、修饰酶、辅酶等无机化合物，主要负责细胞间调控。
   
# 5. 结构比对（Structural Alignment）
   
   结构比对是将两个序列或两组分子的结构进行对比，以确定它们的相似性和联系，通过结构对比可以判断两段DNA序列之间是否存在复制事件、折叠现象、突变、翻译异常等。
   
   在实际工作中，结构比对通常会采用三种不同的方法，分别为全局比对、局部比对、结构直方图（Structure Histograms）。
   
    - 全局比对，是指两序列共有的区域或对应位置进行对比，这种方法具有最高的准确率，但计算量较大，时间复杂度为O(n^2)。

    - 局部比对，是指只比较不同区域的序列，这种方法的计算量较少，但准确率较低，时间复杂度为O(nm)。

    - 结构直方图，是指将两序列的结构特征映射到坐标轴上，然后将坐标轴上的点分布做成图像，这样就可以直接看出哪些位置的对应关系较强。这种方法的计算量很小，速度快，但是对称性较弱，不能反映出有向性和距离等关系。

# 6. 信息检索（Information Retrieval）
   
   信息检索是生物信息学的一项重要课题，它的目的就是通过搜索引擎等方式，在海量的数据中找到自己需要的信息。例如，给定一条序列，可以通过序列比对的方法找出它的近邻序列（Neighborhood Sequence）；再如，可以通过蛋白质结构图、相互作用图等方式找到相似性较高的蛋白质；再如，可以通过基因注释的方式找到相关的疾病。
   
# 7. DNA序列处理工具

   由于DNA序列具有一定的复杂度和规模，所以目前人们已开发出相应的序列处理工具。其中包括比对、重建、变异检测、甲基化、结构分析、代谢机制分析、抗原识别等等。
   
  # 8. 结构预测工具
   
   结构预测的目的是预测蛋白质的三维结构，并由此推测其功能和活性。目前，蛋白质结构预测工具主要分为三种，即深度学习方法、传统机器学习方法、结构搜索方法。
   
  # 9. 可视化工具
   
   通过可视化工具，生物信息学人员可以对结构数据进行建模、分析和展示。一些典型的可视化工具有Chimera、PyMOL等。
   
  # 10. 协同分析工具
   
   协同分析工具，是指利用多组数据进行共同分析，从而提升数据的整体认识。它主要应用在复杂生物学网络、复杂系统中，比如复杂多样性、转座子调控、基因表达调控等领域。
