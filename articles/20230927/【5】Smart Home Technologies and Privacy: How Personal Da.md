# 1.简介
  

智能家居一直以来都被视为一个热门话题。不仅如此，近年来智能家居领域也产生了很多新的技术，比如无线开关、智能门锁、智能摄像头等等。越来越多的人开始关注这个领域，并且提出了许多新的需求，比如隐私保护、效率提升、节能降低等等。这项研究的目的就是通过对智能家居技术及其应用现状、新兴技术的出现、隐私保护理念的形成以及新的用户需求进行系统性地分析，从而揭示智能家居的特性及背后的社会影响因素，并指明未来的发展方向。  

# 2.Smart Home Technologies 
## Smart Switches and Controllers
智能开关和控制器（Smart Switch）是智能家居的一大类产品。它们的主要功能是控制各种设备的开启或关闭，通常安装在房间的不同位置，通过网络连接到智能设备，实现自动化的控制。据统计，截至2017年底，全球已有超过6亿个智能开关和控制器被制造。其中，中国产的智能开关控制器数量占到了整个市场的三分之一。根据全国智能开关设备交易平台——艾瑞智联的数据显示，截至2017年底，全国累计生产的智能开关和控制器共计50万台，约占全国智能设备总数的7%左右。



目前，市面上主要有两类智能开关控制器：
   * 第一代型号，早期采用装有触摸屏的LCD显示屏的控制按钮，如图所示，这种类型的控制器一般都具备较好的使用效果。
   * 第二代型号，采用电子开关，具有低功耗、微重量级、便携性强等特点。它们包括传统的电力开关、模拟开关和数字开关。其中，数字开关又可以划分为插座型号和插卡型号。插座型号一般配套有带有钥匙孔的负反馈按钮，通过按压按键可以控制电器的打开和关闭；插卡型号则相对简单一些，直接用硬件开关接在电源线上就可控制设备的开关状态。

## Smart Thermostats and Controls
智能终端配套设施（Smart Terminal Auxiliary Equipment，STEAE）是智能家居中的一大类产品。它们由多个控制器和终端设备组成，包括智能空调、家用厨房电器、智能生活小夜灯等。这些产品通过网络连接到智能设备，实现远程控制。目前，市面上已有的智能终端配套设施还有：
   * 智能窗帘、天花板遮阳伞、智能窗户隔断
   * 智能布艺台灯、智能衣柜
   * 智能健康监测仪、智能水质检测仪
   * 智能冰箱、智能洗衣机、智能电动打蛋器
   * 智能路由器、智能定时开关、智能抽湿器
   * 智能清洁扫拖机器人、智能室内净化器、智能汽车充电桩
   * 智能门锁、智能运动门禁卡、智能安防监控系统等

其中，智能空调的销售额同比增长31%，占到历年销售总额的10.8%，成为2018年全球第三大智能家居市场主体。目前，智能家居领域还存在着许多的技术难题。比如，如何降低风险，减少耗能，保证安全？如何提升用户满意度？如何让消费者快速适应新变化？这些都是很重要的课题。


# 3.Privacy Protection 

数据隐私保护的重要性已经得到越来越广泛的重视。在当下个人信息收集和处理各项技术的发达背景下，人们越来越意识到自己的个人数据已经成为个人生活不可缺少的一部分。随着信息技术的发展，数据的获取、存储、使用、共享等过程也越来越复杂。因此，保障个人信息的安全、隐私与个人自由是当前和今后发展的重点。

## How personal data is used in smart home technologies?

首先，我们需要明确一点，由于智能家居一直以来都被视为一个热门话题，但真正谈到数据的使用时，更多的是论述“Who”、“What for”、“Why”三个方面的问题，而不是“How”。换句话说，要了解智能家居中个人数据是如何使用的，就要回归到两个大的方面：一是用户，二是服务提供商（即制造者）。下面我们就分别从这两个方面入手，看看个人数据是如何在这两个角色中流通、使用、共享的。

### User side - Personal data collection and usage behavior

**Personal data collection**: 智能设备通常会采集许多种类的个人数据。除了自身产品相关的数据，还包括用户习惯、偏好、配置信息等等。例如，通过智能摄像头捕获用户的面部表情、声音等信息，再将这些信息上传到云端，用于帮助用户进行活动识别。智能设备同时也会利用用户的手机、WiFi、蓝牙等通讯方式收集用户的社交媒体账号、邮箱地址等敏感信息，用于进行人脸识别、精准广告推送等用途。另外，智能设备还可以通过语音助手收集用户的指令、指令的执行结果等信息。以上，智能设备收集到的所有个人数据，都属于用户的个人敏感数据。

**Data use and sharing**: 在用户的日常生活中，智能设备会通过各式各样的方式收集个人数据。例如，通过语音助手与用户沟通、与上下班时间同步播放节目、通过摄像头拍照记录住户的生活轨迹。更有甚者，智能设备可能会在用户不知情的情况下，在用户不易察觉的情况下将个人数据上传到云端，用于营销推广。所以，对于个人数据的使用行为，保障用户的个人数据安全和隐私，是智能家居技术设计者和制造者需要考虑的问题。

**Personal data processing**: 数据收集、使用和分享之后，智能设备就会对用户的数据进行处理。这一步是智能技术制造商和服务提供商应该重点考虑的环节。由于收集的个人数据可能包括用户的敏感信息、私密信息、用户上传的文件等，所以在数据处理过程中务必要注意保护个人信息的完整性和可用性。对于无法完全控制的数据，例如敏感信息，智能设备应该通过加密、匿名化等方式，尽最大努力保障个人数据的安全和隐私。

**Personal data security**: 除了数据收集、使用、处理过程中，用户自己也有可能泄露自己的个人信息。智能设备制造商和服务提供商为了保障用户的个人信息安全，可以在整个用户使用过程引入相应的安全措施，包括但不限于：要求用户设置密码、验证身份、手机绑定、数字证书等。

**User satisfaction**: 用户对智能设备及其使用情况的满意度也是衡量智能家居技术成功与否的一个重要指标。智能设备制造商和服务提供商应该持续不断地收集用户的反馈意见，并持续改进产品，提升用户的满意度。

### Service provider side - Personal data storage and handling

**Data storage and protection**: 智能设备的制造商和服务提供商也要对用户的数据进行安全存储。虽然数据传输过程本身是安全的，但为了避免数据丢失、泄漏、泄露等问题，数据处理、存储和传输都需要符合相关法律规定。数据存储通常采用云计算的形式，位于云端服务器上，只向授权人员、法律实体和事先指定的合作伙伴开放。另外，智能设备的制造商和服务提供商还可以选择是否将用户的敏感信息存放在专用数据库中，这样就可以满足信息安全的要求。

**Data handling policies and practices**: 为了更好地保障数据安全和隐私，智能设备的制造商和服务提供商需要制定详细的数据处理政策和实践规范。这些政策规范包括但不限于：数据分类标准、数据保留策略、数据删除政策、数据传输协议和加密方案等。除了保障用户的数据安全，智能设备制造商和服务提供商还可以积极参与公众舆论，收集用户的声音、观点、意见，并将之用于数据开发、政策制定和产品改进等工作。

## Privacy Concerns Faced by Smart Home Developers

**Data breaches and attacks**: 数据安全始终是个问题。由于智能设备的数据收集、使用、处理、存储等流程均是基于个人信息，所以任何一方在这方面出现问题都会引起严重的安全问题。智能设备制造商和服务提供商需要认真响应国家的相关规定，加强安全管理，防止个人信息被窃取、滥用。同时，智能设备的制造商和服务提供商也要积极参与公众舆论监督，防范违规违法的行为，提高用户的知晓度和认可度。

**Privacy risks to consumers**: 另一方面，智能设备制造商和服务提供商还要考虑到隐私权的侵犯问题。为了满足用户的隐私权利，智能设备制造商和服务提供商应该努力降低个人数据的使用范围、保护数据安全，并提供详尽的隐私政策和用户协议。智能设备制造商和服务提供商应避免收集、使用、处理个人数据所导致的用户隐私权益受损。同时，为了提升用户的隐私权益，智能设备制造商和服务提供商应鼓励用户主动保护自己的数据。
