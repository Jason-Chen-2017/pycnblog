# 1.简介
  

在上个世纪的九十年代末期，由于美国政府的“9·11恐怖袭击事件”，带来了严重的社会经济影响。2001年之后，美国加强了反恐战争，并实行一系列政策以抵御此次袭击。2017年，特朗普上台后，宣布退出修订后的《国家安全战略》，并重新成为美国最高领导人的最高授权，采取更激进的反恐策略，令全球陷入恐慌状态。然而，美国民众却越来越多地认为，警察应当充分信任。那么，美国民众是如何看待警察的呢？以下我将从事实说起，试图揭示美国民众对警察的真正态度。
# 2.相关知识背景
## 2.1 恐怖袭击
1991年11月1日夜晚，三名美国士兵突然袭击了埃塞俄比亚驻美大使馆，造成100余人死亡、四千余人受伤。
这个“11·11”恐怖袭击是为了向苏联等轴心国国民暴露其核武器技术，制裁恐怖主义。它引发了包括全球范围内反恐战争、欧洲自由党、意大利自由党、西班牙自由党在内的各国政治力量进行的激烈论战。20世纪初，世界各国均陷入经济危机、贫困潮、种族冲突、社会动荡、暴力事件的严重局面。
## 2.2 《国家安全战略》
在1991年10月，当时担任美国总统的奥巴马签署了《国家安全战略》，该法案旨在确立美国政府作为美国人民的核心利益和国家安全的基本权力。其中保障公民权利、财产权、隐私权和安全权利的观念至关重要。美国进入防务前沿，积极应对全球恐怖活动，取得了重大的成果。但是，在“9·11恐怖袭击”发生两周之后，美国政府才终于下决心结束这种“空前的军备竞赛”。
1997年，特朗普上台后，新任总统取消了《国家安全战略》，并发布了新的战略法律。到2001年，新版《国家安全战略》对美国国内外环境的影响开始显现出来。
2003年，美国商务部长麦凯恩提出“空袭美国”（Airstrikes Against America）的倡议，表明美国国家安全战略的失败。2005年，美国总统罗斯福在一次记者招待会上声称，美国应当继续支持恐怖主义，美国要大幅提高对付各种恐怖组织的能力。但随着两党关系的恶化，这场讨论变得难以实现。
## 2.3 美国警察队伍的发展
美国民众对于警察队伍一直存在不满情绪，原因很多，但其中一个重要原因就是警察的严厉程度过低。在“9·11恐怖袭击”之后，许多美国人都认为，美国警方的工作效率较低，且随意性很高。美国联邦警察局系统已经连续五年内平均每天花费超过8万美元用于职权保护、巡逻、协调、搜查、搜索、引渡、惩处犯罪。从2008年开始，联邦警察局规模扩大，将警察数量从20万增加到了70万。虽然人们普遍认为这是美国警方的巨大进步，但也有一些美国人认为，警察系统过于庞大可能会导致效率下降，甚至出现警察滥用执法权的情况。

另外，在国内，各地相继爆发反恐事件，警方频繁应对，给警方的工作压力也越来越大。据估计，从2014年至今，美国陆地警察共有70万人，海岸警察共有2.5万人。海外警察占到了整个警察队伍的7%左右。因此，特别是在国际政治舞台上，美国警方的声望和威望是可以肯定的。
# 3.美国民众对警察的看法
## 3.1 对警察的评价
美国民众对警察的信任一直是一个难题。很多美国人认为，警察应当被视为能够保护公民的最佳工具；而另一些人则认为，警察没有足够的能力、能力不足或错误的决定可能导致重大灾难。虽然这两种看法都有其道理，但它们却不能完全否定警察工作者的能力。

2011年的一份研究发现，美国约有四分之一的公民表示，他们对警察的信任程度非常高。同时，有五成的人认为警察应当信任，这反映出美国社会的公平、正义感以及警务力量的雄厚威望。

根据《麻省理工评论》的一项调查显示，美国民众对警察的满意度只有五分之一不到。五分之二到六分之一的民众对警察有些担忧，认为他们过于依赖警察保护自己的财产，不具有足够的自主权。还有七分之一到八分之一的民众对警察比较宽容，认为警察的职责应该由他们自己承担。

尽管美国民众对警察的满意度普遍偏低，但这一现象并非意味着警察的普遍失误。美国历史上曾经多次发生警方滥用执法权等问题，但这些问题均得到了警方的高度重视，而且得到了民众的普遍赞扬。这反映出美国社会对政府公共事务有高度认可的氛围。

## 3.2 关于警察的一些误解
美国的警察队伍长期处在艰苦创业、改革创新和机遇改变等三个阶段，它的成熟历程经历了政治，经济，文化，科技，法律和社会等多重因素的作用，包括经济增长、金融市场的发展、经济结构的调整、环境污染的控制等。

近年来，美国警察队伍的形象和德勤斯基氏在《超级合伙人》中的所作所为已经完全不同。前者曾是牢狱区中流浪的恐怖分子，现已成为有着超高绩效的公检法一体化力量。这也从侧面反映出美国警察队伍的发展速度和绩效，以及对公众的尊重。

另外，媒体还时常谈及美国的警察队伍，强调警察对国民生命安全的重要性。但是，正如加州大学伯克利分校的教授莱昂内尔·格林尼治说的那样：“一个国家如果没有良好警察队伍，就不可能建立起真正的保护人民的社会。……警察机构对一个国家来说往往是国内最大的潜在威胁。”美国警方无疑对社会、对公民负有不可推卸的责任。

## 3.3 有些美国人甚至认为，美国警察队伍就像中国的公安队一样，存在着贪婪、腐败、猎奇、残酷等种种弱点。
实际上，美国警察队伍没有贪污腐败，大多数都是受过良好教育的优秀公民。他们经常受到社会各界的热烈欢迎，并得到公众的广泛认可，被誉为民主公正的职业角色标。

美国警察队伍经常培训新人，并且收缴旧车、工具等，以防止新人上当受骗。除了训练，美国警察队伍还要维护其既有的纪律和执法水平，保持警察队伍顺畅、稳定，以及警察人员的技能水平、办案能力、应变能力等优势。在这方面，美国警察队伍遵循的是“两个凡是”原则，即对所有现役警官严守纪律，严禁贪污、滥用职权，对违反条例或命令的行为进行严厉惩戒。

对于许多美国人来说，美国警察队伍过于薄弱或者缺乏专业技能是一种误解。其实，美国警察队伍的综合素质有待于提高。在战争中，许多警察在生死线外、荣辱行间捍卫自己的权益，也常常因为身体、健康状况等原因而难以服从命令。此外，警察队伍内部也存在着各种派系斗争，例如警察与黑人，警察与白人，警察与党派。所以，要想让美国警察队伍变得更加强大，就需要不断完善其工作机制、扩大执法规模，以及引入更先进的管理方式。
