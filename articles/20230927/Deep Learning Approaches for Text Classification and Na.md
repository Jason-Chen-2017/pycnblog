# 2.文本分类（Text Classification）
## 2.1 基本概念及分类模型介绍
文本分类就是给定一个文本输入，对其所属的类别或者分类标签进行预测的问题。一般来说，文本分类可以分为两大类：
- 一是多标签分类：一种文本可以同时属于多个类别中。例如新闻文章的主题标签就是多标签分类。
- 二是单标签分类：一种文本只能属于其中一个类别中。例如垃圾邮件分类。
文本分类的方法可以根据文本特征、词汇和句法结构等信息进行分类，不同的方法又可以分为有监督分类、半监督分类、强化学习分类等。
## 2.2 有监督文本分类方法
### 2.2.1 朴素贝叶斯法（Naive Bayes）
朴素贝叶斯法是基于贝叶斯定理与特征条件独立假设的概率分类方法。它认为特征之间相互独立，因此朴素贝叶斯法适用于文本分类。朴素贝叶斯法的基本思想是假设所有特征之间都是相互独立的。对于给定的一组特征向量x，它被认为是在类别c上的第i个文档，那么P(c|xi)就等于先验概率p(c)乘以条件概率p(xi|c)，其中p(c)表示所有文档中类的先验概率，p(xi|c)表示在类别c上，第i个特征出现的概率。
### 2.2.2 SVM支持向量机法（SVM with kernel）
支持向量机（Support Vector Machine, SVM）是一种二类分类模型，由Vapnik于1992年提出，其特点是通过寻找间隔最大的线性分类边界或超平面，最大限度地将两类样本完全分开。朴素贝叶斯法假定所有的特征之间是相互独立的，而SVM采用的是核函数，也就是将原始特征空间映射到高维空间后，再用线性判别式将数据投影到低维空间，从而达到非线性划分的效果。常用的核函数包括线性核函数、高斯核函数、多项式核函数和径向基函数。
### 2.2.3 决策树法（Decision Tree）
决策树是一种基本的分类与回归模型，它能够做出预测、分类和回归。决策树模型本身是一个if-then规则集合，它由根节点、内部节点和叶子节点构成。决策树模型利用决策树的学习策略，构建一系列的测试，并反复迭代，不断缩小训练集的范围，最终输出分类结果。决策树算法非常有效，能够快速准确地解决分类和回归问题。决策树的工作原理如下图所示：
### 2.2.4 KNN近邻法（K-Nearest Neighbors, KNN）
KNN是一种基于最近邻的机器学习分类算法。该算法认为距离近的样本应当属于同一类，距离远的样本则不同类。KNN算法实现起来简单，且易于理解，但缺点是速度慢，无法直接处理高维特征空间。
## 2.3 无监督文本聚类方法
### 2.3.1 LDA潜在狄利克雷分配法（Latent Dirichlet Allocation，LDA）
LDA是一种无监督学习方法，它用来对文本数据进行聚类。其基本思路是先根据文本数据生成共现矩阵，然后进行变分推断得到话题分布，再对每篇文档进行话题分配，最后将文档分配到各个对应的话题中。LDA通过直接学习各个文档的隐含主题，使得文本数据成为具有自然语言意义的结构化数据。
### 2.3.2 Hierarchical clustering层次聚类法
层次聚类法是一种无监督聚类方法，它利用合并、分割、重塑等操作对数据集进行层次划分，最后形成若干个簇。Hierarchical clustering是一种常用的聚类方法，其基本思想是按照样本之间的距离关系进行层次划分，距离较近的样本放在一起，这样就可以形成一颗层次树，而层次树的每个节点代表一簇，从上往下进行合并，直到所有样本都在某一个节点中。
