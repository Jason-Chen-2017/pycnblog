## 5. 项目实践：代码示例和详细解释

为了进一步阐述这一想法，让我们创建一个将食物知识图谱与元宇宙结合的示例项目。在这个项目中，我们将创建一个名为“FoodGenie”的应用程序。用户可以使用FoodGenie创建自己的元宇宙厨房空间，根据他们的饮食偏好和健康目标生成自定义菜单。

以下是一个简单的代码示例：

```python
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression

# 加载数据集
data = pd.read_csv("food_data.csv")

# 将数据集拆分为训练集和测试集
train_data, test_data = train_test_split(data, test_size=0.2)

# 使用逻辑回归模型创建分类器
classifier = LogisticRegression()
classifier.fit(train_data.drop("target", axis=1), train_data["target"])

# 用测试集评估分类器
accuracy = classifier.score(test_data.drop("target", axis=1), test_data["target"])
print(f"模型准确率：{accuracy:.2f}")
```
