4.2.1. 音乐创作

假设有一个AI音乐创作者，他希望通过AI技术来生成新的曲子。我们可以使用Python中的PyAudio库来实现音频播放和录制。具体步骤如下：

1. 安装PyAudio库
2. 创建一个Python脚本，并导入需要的库
3. 调用PyAudio库中的start()函数来启动录制
4. 在录制过程中，使用Python中的time模块来控制录制的时间长度
5. 录制完成后，使用Python中的sys模块中的exit()函数来关闭录制

下面是一个简单的示例代码：
```python
import numpy as np
import pyaudio
import sys

CHUNK = 1024
SAMPLE_RATE = 44100
RECORD_SECONDS = 5

audio = pyaudio.PyAudio()

stream = audio.open(format=pyaudio.paInt16,
                channels=1,
                rate=SAMPLE_RATE,
                input=True,
                frames_per_buffer=CHUNK)

frames = []

for i in range(0, int(SAMPLE_RATE / CHUNK * RECORD_SECONDS)):
    data = stream.read(CHUNK)
    frames.append(data)

stream.stop_stream()
stream.close()
audio.terminate()

print('Done recording.')
```
这个示例代码中，我们使用了PyAudio库来实现音频播放和录制。在录制过程中，我们使用了time模块来控制录制的时间长度，使用sys.exit()函数来关闭录制。
