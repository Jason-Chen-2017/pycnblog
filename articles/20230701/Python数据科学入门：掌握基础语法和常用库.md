2. 技术原理及概念
---------------------

2.1. 基本概念解释
Python 中的数据结构有列表、元组和字典。其中，列表和元组是可变的，而字典是唯一的。

2.2. 技术原理介绍:算法原理，操作步骤，数学公式等
Python 中的列表和元组都有独特的索引和切片功能。列表索引下标从 0 开始，而元组索引下标从 1 开始。此外，Python 中还提供了切片功能，可以对列表和元组进行切片操作。

2.3. 相关技术比较
Python 中的列表和元组在功能上比较接近，但在性能上有一定的差异。在比较中，列表的性能稍高于元组，而元组的性能要高于列表。
