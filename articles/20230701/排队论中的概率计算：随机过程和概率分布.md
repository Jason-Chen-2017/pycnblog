2. 技术原理及概念
---------------------

2.1. 基本概念解释
排队论中，我们主要关注等待时间和可用服务时间。等待时间是指顾客到达设施后到开始享受到服务之间的时间，而可用服务时间则是设施可以提供给顾客服务的时间。

* 这是一个等可能过程，所有的顾客到达设施后都享有相等的等待时间和可用服务时间。
* 这是部分公平的计数原理，不同的顾客具有不同的等待时间和可用服务时间。
* 这是最公平的计数原理，所有的顾客都具有相同的等待时间和可用服务时间。

2.3. 相关技术比较
我们可以根据等待时间和可用服务时间对排队论进行分类。

* 计数原理：
   - 等可能计数原理：所有顾客具有相同的等待时间和可用服务时间。
   - 部分公平计数原理：不同的顾客具有不同的等待时间和可用服务时间。
   - 公平计数原理：所有的顾客都具有相同的等待时间和可用服务时间。
* 数学公式：
   - 计数原理：
      - 平均等待时间 = 总等待时间 / 顾客数
      - 平均可用服务时间 = 总可用服务时间 / 顾客数
   - 等可能计数原理：
      - 每个顾客到达设施后，等待时间的概率为 1/n
      - 每个顾客到达设施后，可用服务时间的概率为 1/m
   - 部分公平计数原理：
      - 每个顾客到达设施后，等待时间的概率为 p1，可用服务时间的概率为 q1
      - 顾客到达设施后，总共的等待时间记为 T，则 T = p1 * T1 + q1 * T2
      - 顾客到达设施后，总共的可用服务时间记为 U，则 U = p1 * U1 + q1 * U2
   - 公平计数原理：
      - 每个顾客到达设施后，等待时间的概率为 p，可用服务时间的概率为 q
      - 顾客到达设施后，总共的等待时间记为 T，则 T = p * T1 + q * T2
      - 顾客到达设施后，总共的可用服务时间记为 U，则 U = p * U1 + q * U2
