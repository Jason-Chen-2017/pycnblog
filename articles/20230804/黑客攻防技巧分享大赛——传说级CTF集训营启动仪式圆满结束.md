# 1.简介
         
2019年，国外技术大牛们联合举办了一场令国内少部分人目瞩的“黑客攻防技巧分享大赛”———创客马拉松（CreatorMarathon）活动，目标是邀请全球最顶尖的科技精英、创新者及学生共同参与到一个跨界的攻防比赛中。该活动于2月份在西班牙首尔举行，吸引了来自世界各地的精英和学生共计超过1万人参加，历时两个月，获得极高关注度和赞誉。“黑客攻防技巧分享大赛”活动帮助了国内业内领先的黑客大佬们与优秀的黑客才俊共事并互相学习，用技巧解决实际问题，促进了知识的交流与共享，推动了信息安全领域的革命性变革。
         在活动结束之际，我作为此次活动的举办方，看到了经过一个月的筹备工作和几千人的参与，广大的参赛人员积极踊跃报名参加了这场赛事。看到如此热烈的参与感激不尽，作为国家队长，作为战略级专家，我当然不能落下手。我要用自己的专业经验、丰富的社区经验，结合自身对信息安全的理解和对创客文化的浓厚兴趣，为国内顶尖的创客培养更多的训练师和高素质的黑客才俊，推动黑客文化的繁荣和创新的发展。
         为了让更多创客参与到赛事中来，现场组织了主题分享会，邀请了业内领域的创新者、前沿科技人员和白帽子等参与到赛事中与众多的同龄人的分享互动。
         由于时间仓促，内容不够充分，所以希望通过这篇文章可以给国内的创客们带来一点收获。
         # 2.背景介绍
         “黑客攻防技巧分享大赛”（Creator Marathon）活动由创客马拉松（CreatorMarathon）活动发起，创客马拉松是一款基于竞技体验的虚拟创客社区，提供全面、深入的技术教育及个人成长能力训练。
         此次活动是一次创客竞技赛事，重视技术能力，用研人员的独特视角与技能培养方式将创客们带到了前所未有的高度。
         本次“黑客攻防技巧分享大赛”活动的主要内容如下：
         - 没有指定题目，大家自由组队参与，用自己的喜好、兴趣、技能或成长路线组建队伍；
         - 采用创造性的方式，将攻防技巧分享给大家，围绕主题进行展开；
         - 每位选手将在5-7分钟左右的时间里，展示自己的团队和解决方案，并向全场展示自己的攻防技巧；
         - 以奖牌机制为主，奖励优胜者。
         # 3.基本概念术语说明
         ## CTF(Capture the Flag)简称网关测试。
         简而言之，CTF (Capture the Flag)，即网络游戏中的网络安全竞赛。一般情况下，CTF 通过给予参赛者若干FLAGS，同时围绕着这些FLAG的安全设置，在规定时间段内完成任务，赢得比赛的胜利。比赛往往采用现实赛制或者虚拟赛制。
        ### flag
        网关通常由5个字符组成，用来提交与服务绑定相关的数据，也称作flag。例如：FLAG{123ab}，这个flag就是一条数据。它起到验证参赛者身份、判断作品是否符合要求、寻找正确答案的作用。

       ### pwn(Personal Web Challenge)
       Personal Web Challenge，亦被称为简易Web挑战赛，是一个颇受欢迎的网络游戏，由法国计算机系的一支学生团队开发，目标是通过一步步的复杂的题目，逐渐熟悉Web应用和安全攻防技巧。

       ### nmap(Network Mapper)
       Nmap是一款开源的网络扫描器工具，用于网络发现和安全审查。其功能强大、灵活性高，能够快速准确地检测服务器上运行的各种服务、主机、路由器、端口的开放情况，并且提供完善的报告功能，十分适合作为网络攻击、安全审计、管理运维、设备维护、系统监控等方面的工具。

       ### XSS(Cross Site Scripting)
       Cross-site scripting(XSS) 是一种网站应用程序安全漏洞，攻击者可以在网站上注入恶意脚本代码，从而影响网站的用户体验，达到持久控制用户浏览器的目的。

       ### SQL injection
       SQL injection，也称为SQL注入，是一种在WEB应用中，恶意攻击者利用输入欺骗服务器执行恶意SQL命令的攻击方法。

       ### DOS(Denial of Service)攻击
       Denial of service attack，缩写为DoS攻击，又称拒绝服务攻击、丢包攻击，是一种利用网络拥塞或资源耗尽攻击的方式，企图使目标网络或系统停止正常服务。

       ### DLP(Data Leakage Prevention)
       数据泄露保护（Data Leakage Prevention，DLP）是指通过有效控制和减小对外泄露数据的范围、程度和速度，预防企业数据泄露风险，并支持合规运营的过程。DLP可为企业节省大量资金及维护资源，提升企业的整体竞争力。

       ### CSRF(Cross-Site Request Forgery)攻击
       CSRF（Cross-Site Request Forgery，跨站请求伪造），也叫做“假冒注册页面”或Session Riding，是一种挟制用户在当前已登录的状态下，以受信任用户的名义，非法利用被授权访问本网站的网站功能，实现转账、购买商品或进行其他恶意操作的一种攻击方式。

      ## Web安全
      网站的安全本身其实已经不是什么新鲜事物了，但是如果没有好的措施去保障网站的安全的话，很容易就会被黑客攻陷。所以在举办活动之前，大家都需要了解一些网站的安全攻防技巧，防范常用的攻击手段，并提高自己的信息安全意识。
    ## 技术篇
     ### Burpsuite
     Burpsuite是一款开源的Web应用安全测试工具，可用于渗透测试、安全评估、漏洞分析、攻击搏击和penetration testing，它可以通过截取、修改、重放HTTP请求等方式来诊断、攻击、修改网站的安全漏洞。

     ### sqlmap
     Sqlmap是一款开源的自动化数据库取证工具，它是为了方便安全研究人员以及管理员从中央数据库和其他潜在风险数据库中自动提取数据的工具。sqlmap使用了内置的 payload 与工序生成、注入数据，检测错误输出、超时错误、SQL错误的技术，并且提供了一个有效的手段来检测和利用 SQL 注入漏洞。

     ### Metasploit Framework
     MetaSploit框架是一款针对Ruby编程语言的开放源码渗透测试平台，它主要面向于红队和蓝队，以模块化的方式提供了多种利用手段。它具有反射型、基于插件的攻击模式，并且支持多种数据库、中间件和操作系统等多个平台。

     ### Wireshark
     Wireshark是一款自由软件，它是用于网络封包分析的捕获与显示工具，也是一种协议分析工具。它能帮助您跟踪流经网络的通信数据包，提取协议数据单元（PDU），评估网络性能，还可以用于嗅探和修改网络数据包。

     ### Kali Linux
     Kali Linux是一套 Linux 发行版，由 Offensive Security 创建，旨在为黑客和渗透测试者提供一个开放的平台，包括 Web 应用安全测试的重要工具。它包含许多用于渗透测试的工具，包括Burp Suite、NMAP、Sqlmap、Metasploit Framework、Wireshark、Katoolin等。

     ### IcedID
    IcedID 是一个由德国汉森堡大学设计的轻量级密码破解软件。IcedID 使用硬件接口和标准技术来解密存储在磁盘上的文件，目前已支持Windows、Linux和Mac OS X平台。

    ### sqlninja
    sqlninja是一款开源的无线安全工具，它基于Kali Linux，针对的是Android移动设备，通过分析Wireless Traffic获取密码、加密的SQL查询语句等信息。
