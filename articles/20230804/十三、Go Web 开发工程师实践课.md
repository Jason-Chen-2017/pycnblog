# 1.简介
         
1. Go 是由 Google 开发并开源的一种静态强类型、编译型编程语言，它是高性能、轻量级、原生支持并行计算的语言，其设计思想受到 CSP(Communicating Sequential Processes)、Erlang、Haskell 和 Lisp 的影响。它的主要特性包括内置垃圾回收器、自动内存管理、GC、类型安全、反射、结构化语法等，使得其具有卓越的执行效率、适应性及稳定性，同时又易于学习和使用。

         2. Go 语言作为目前最火爆的开发语言之一，它能够帮助快速开发出健壮、可扩展、容错性强、易维护的代码。Go 语言特有的简单性和灵活的组合方式，也促进了其成为云计算、微服务、分布式系统等领域的一流语言。除此之外，Go 语言在国际标准化组织 ISO/IEC 国际标准化组织 (ISO/IEC JTC1/SC39) 中得到了 ISO/IEC 认证，也被视作是现代编程语言中的佼佼者。

         3. 本课程主要从以下三个方面教授 Go Web 开发的知识技能：基础知识、实践项目和优化技巧。通过本课程，学生可以掌握 Go Web 开发的各项基本技术，能够独立完成 Web 服务端的开发。

         # 2.内容概览
         1. Go Web 开发环境搭建
         2. Web 服务端的 HTTP 协议和路由实现
         3. 模板引擎的使用（如 html/template）
         4. 基于 ORM 框架 GORM 进行数据库操作
         5. 安全防范和访问控制
         6. 错误处理和日志记录
         7. RESTful API 的设计和实现
         8. WebSocket 协议和应用
         9. 单元测试和集成测试
         10. Docker 容器化部署
         11. 线上性能调优及常见性能问题定位方法
         12. 项目中常用的优化技术
         13. 网站迁移至云服务器上的注意事项
