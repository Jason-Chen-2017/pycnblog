# 1.简介
         
 ## 什么是云计算？ 
          云计算（Cloud Computing）是一种利用互联网技术、计算机网络及存储技术，将用户数据的存储和计算资源池化的新型信息技术服务模式。云计算通过网络将远程服务器、存储空间、数据库、计算能力等IT资源通过网络相互连接，为用户提供按需获取的资源服务，实现信息化资源共享，能够有效降低数据中心建设成本、提升服务质量、节约IT支出。 
        
          ## 什么是云计算防御？
          在云计算平台上运行的应用或服务都是多种形式的，它们由不同的开发者、运维人员和系统管理员共同完成部署、运维、管理，并不总是容易被检测到。为了避免攻击者滥用这些应用或者服务而造成严重的安全事故，需要在云计算平台上进行相应的安全防御措施。云计算防御可以分为静态检测、动态检测、无线防护三大类。 
        
          ### 静态检测
          静态检测即对运行中的程序或系统进行扫描，检测是否存在恶意行为，但这种检测方式耗时长、易受误报、误报率高，因此一般用于实时监控、预警、过滤。静态检测主要包括机器学习算法、行为分析、日志审计、入侵检测等。 

          ### 动态检测
          动态检测即实时跟踪和分析系统的运行状态、访问记录、网络流量、网络流量特征、系统调用等数据，基于这些数据，能够识别出异常的活动指标，从而发现攻击者的入侵企图。动态检测又可细分为：
           - **主机入侵检测系统（HIDS）**：HIDS（Host-based Intrusion Detection System），顾名思义，就是根据主机的行为特征进行入侵检测。HIDS能够实时收集主机的数据，如文件、网络活动、进程创建等，并分析其特征向量，判断其中是否存在可疑的攻击行为，是一种硬件加速的入侵检测系统。目前主流的HIDS产品有Bro，Snort和OSSEC等。
           
           - **网络入侵检测系统（NIDS）**：NIDS（Network-based Intrusion Detection System），顾名思义，就是根据网络中传播的流量特征进行入侵检测。NIDS能够在检测流量过程中，实时捕获流量中的恶意数据包，并分析其特征向量，判断其中是否存在可疑的攻击行为。目前主流的NIDS产品有IDS，IPS等。
           
           - **端点检测与响应（EDR）**：EDR（Endpoint Detection and Response），是一种与NIDS结合的网络安全产品。它的主要功能是在网络入侵检测基础上，集成了一系列的网络安全工具，用于辅助快速响应攻击、收集信息、违规行为的整体过程。目前主流的EDR产品有Symantec Endpoint Protection和Trend Micro Control Manager等。
           
           - **云安全管控（CSP）**：云安全管控，它是一个云服务提供商（Microsoft Azure、Amazon Web Service等）提供的安全解决方案，包括云漏洞扫描、安全事件通知、安全态势感知、安全基线验证、安全控制管理等功能。云安全管控能够在云上识别攻击行为、管理漏洞、触发威胁、实施自动化响应等，是一种基于云的安全解决方案。
       
          ### 无线防护
          无线防护是一种基于智能手机、蓝牙、WIFI等无线通讯技术的安全保障手段，是防范网络攻击的重要途径之一。无线防护的核心是通过各种设备的认证、加密、威胁模型、危险行为检测、网络控制、行为管理等，对各类威胁进行检测和控制。目前主流的无线防护设备有多普达SoftTrack、Linksys Wi-Fi、TP-Link TD-W8980等。

          ## 为何要做云计算防御？ 
          在云计算平台上运行的应用或服务都是多种形式的，它们由不同的开发者、运维人员和系统管理员共同完成部署、运维、管理，并不总是容易被检测到。为了避免攻击者滥用这些应用或者服务而造成严重的安全事故，需要在云计算平台上进行相应的安全防御措施。做好云计算防御，可以保护用户的数据隐私和业务利益，保障云计算平台的稳定、可靠、安全。 
          当然，做好云计算防御还可以增加IT团队的工作压力，需要多方面考虑包括法律法规、安全行业经验、技术水平、产品策略、组织文化等因素，才能确保企业在云计算领域的安全。因此，云计算防御是一个复杂的工程，需要众多人的协作，才能实现企业级的云计算防御体系。
