# 1.简介
         
　　“企业架构”（Enterprise Architecture）是一种企业整体性结构设计方法，它是指企业组织在其生命周期中的规划及管理方针。该方法主要包括业务流程、组织机构、人力资源、信息系统、基础设施、战略决策支持等所有领域的设计。如今，“企业架构”已成为高管们关注的重点之一。然而，对于大多数公司来说，“企业架构”实现起来往往非常困难，原因如下:
          1.组织架构不匹配
         　　作为一个复杂的社会现象，企业的组织结构也经常发生变化。从最初的少量的大型科研团队逐渐扩展到现在的复杂的董事会、CEO办公室、多个公司，甚至多个国家的多个公司组成的联合体。因此，企业内部各个层级的组织架构不断向更加复杂化的方向演变。
          2.需求不统一
         　　企业面临着各种各样的业务需求，这些需求既包括自身产品及服务的开发，也包括来自外部的投资及合作。因此，如何满足不同类型、层级、能力范围内的需求成为一个极具挑战的问题。
          3.未知的复杂性
         　　复杂性源于多种因素，例如历史、文化、法律、地理等多方面的原因。因此，“企业架构”的实施通常被认为是复杂的工程任务。
          4.管理力量缺乏
         　　“企业架构”是一个高度管理性的工作，涉及到多方面的人员协调配合，需要有效的管理队伍参与。但是，目前绝大多数企业还没有形成足够的管理层级，尤其是独立的、有专业知识的人才。因此，企业架构实施者往往依赖于外部的咨询顾问团队、内部的跨职能团队或兼职人员完成一些管理上的事务。
          
       　# 2.企业架构的定义
       　　所谓“企业架构”，其实就是企业各部门的功能分工与职责范围进行协调的结果，其中包括了整体的管理目标、资源结构及各部门的信息共享机制。企业架构师可以从不同视角对企业的资产负债表、利润分配表、生产经营效率、人力资源投入、合同履行情况等进行全面分析，并制定相应的决策支持计划。架构师在制定计划时应考虑到企业的所有方面，不仅要考虑业务领域，还应该考虑组织管理、财务运营、法律遵守、管理风险、安全管理等一系列领域。
        
       　# 3.企业架构的作用
       　　1.提升企业竞争力
       　　企业的竞争力取决于其独特的创新模式、绩效和效益。企业架构能够帮助企业找到新的增长点，优化各项经营管理方式，激励员工加强综合素质，实现企业长期的持续发展。
        　　2.改善企业运行环境
       　　企业的运行环境包括各类资本、技术、资产、设备及其它各种资源的配置、利用以及产出效率。通过正确的企业架构，企业可以有效地利用资源，节约资源，提升竞争力。
        　　3.降低企业成本
       　　企业的成本主要包括物料采购、固定资产折旧、人力成本、材料成本、工期、管理费用、利息支出、折旧等。企业架构的建立可以降低企业的成本，提高企业的竞争能力。
        　　4.促进企业的成功
       　　企业的成功可以通过改善公司业务模式、客户满意度、竞争优势等方面得到提升。企业架构师可以为企业提供战略意图、价值观、管理理念、目标导向等重要的价值导向。
