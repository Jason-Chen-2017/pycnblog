         ## 2.1 小程序概述
         小程序是一种全新的连接用户与服务的方式，由微信提供支持。它可以让用户用最简单的方法获取所需的内容，同时无需下载安装即可运行，彻底解决了 APP 的分发难题。目前小程序已经覆盖企业、政府、医疗等多个领域，且以微信支付等支付方式为代表的第三方支付成为小程序不可或缺的一部分。
         
         小程序的主要特征包括以下几个方面：
         1.开放性：小程序的代码完全开源，没有审查和封锁限制，因此任何开发者都可以基于微信的基础设施进行二次开发，进一步释放了微信生态系统的活力。
         2.跨平台：用户可以在微信、百度、头条等多个平台上获得同样的应用体验，开发者只需要发布一次，就可以发布到多个平台上，实现跨平台能力。
         3.免费使用：小程序免费发布、免费下载、免费更新，用户不必担心设备兼容性问题。
         4.安全性：小程序采用 HTTPS 协议传输数据，数据加密传输，安全可靠。开发者可以通过接口调用的方式直接与用户的个人信息进行交互。
         5.便捷：小程序开发者不需要掌握复杂的技术，只需使用熟悉的前端语言即可快速完成小程序开发，从而打通小程序用户线下的各种渠道。

         ### 2.3 生命周期与生命周期函数
         小程序的生命周期可以粗略分成两个阶段：创建和激活。创建阶段表示小程序被加载，可以理解为注册过程，是小程序进入运行状态的标志；而激活阶段则是在页面真正显示之前，用户可见的状态。这里的生命周期函数主要包括如下四个：
         * onLaunch（小程序启动）：监听小程序初始化，全局只触发一次。通常用于加载应用自身的初始数据。
         * onShow（小程序展示）：小程序启动或从后台进入前台显示时触发，全局会触发多次。通常用于刷新当前页面数据。
         * onHide（小程序隐藏）：小程序从前台进入后台时触发，全局只触发一次。通常用于保存当前页面数据。
         * onError（小程序错误）：小程序发生脚本错误或者 api 调用失败时触发，全局只触发一次。可用来提示用户错误详情。

         ### 2.4 数据存储
         在微信小程序中，提供了三种数据存储方案：本地存储、缓存存储和云端存储。其中，本地存储只能用于单个用户数据存储，不能存储大批量的用户数据；缓存存储可用于临时数据存储，每次小程序启动都会清空，适合临时数据；云端存储则可以存储大量的用户数据，但需要开发者购买服务器资源。

         1. 本地存储：本地存储只能用于单个用户的数据存储，虽然可以持久化数据，但是占用空间过大可能导致用户的空间占用过高，因此不推荐使用。

           ```javascript
           wx.setStorage({
             key: 'key', // 存储键名
             data: 'value' // 要存储的 value
           })
           
           wx.getStorage({
             key: 'key',
             success(res){
               console.log(res.data) // 输出 {key: "value"}
             }
           })
           ```

         2. 缓存存储：缓存存储用来缓存临时数据，只有缓存中的数据才可以保留下来。当缓存被清空后，数据也就无法再读取。缓存存储可以将临时数据存储在内存中，可以有效减少客户端硬件的内存消耗。

            ```javascript
            wx.getStorage({
              key: 'key',
              success(res){
                console.log(res.data) // 如果有缓存数据，输出 {key: "value"}
              },
              complete(){
                wx.setStorageSync('key', 'value') // 缓存数据，下次启动时可直接读取
              }
            })
            
            wx.clearStorage() // 清空所有缓存数据
            ```

         3. 云端存储：云端存储是一个付费服务，提供数据存储的高性能和高可用。微信小程序官方提供了第三方数据存储服务 wetool 和云开发，前者提供了数据库、文件存储、云函数等功能，后者提供了数据增删改查、用户鉴权、实时同步等功能。

         4. 常见问题与解答
          问：是否可以分享小程序给朋友？
          答：是的，小程序可以分享给微信好友、QQ好友、朋友圈、小程序卡片等。也可以通过其他渠道进行分享。小程序分享的链接默认带有当前的小程序参数，可以方便用户进入指定位置，比如分享到群聊时，可以打开该群聊对应的小程序；分享到朋友圈时，可以看到对应的小程序卡片。
