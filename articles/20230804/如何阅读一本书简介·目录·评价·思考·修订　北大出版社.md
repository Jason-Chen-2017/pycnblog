# 1.简介
         
19世纪末到20世纪初期，由于对信息技术、工程技术等的快速发展，“阅读”已成为生活中不可或缺的一部分。人们越来越依赖阅读，把时间花费在学习、研究、创作上，逐渐形成了一种理性主义的习惯。
         
         在当今这个信息化时代，我们每天都面临着海量的信息，知识的快速膨胀，眼花缭乱的需求让我们无暇顾及其他事务，唯独缺少好的阅读来满足自己的需要。好读的书籍可以帮助我们理解某个主题，提高我们的效率，增强自我思维。当然，阅读也有其局限性。书籍不一定适合每个人的情况，有的则需要反复阅读才能融会贯通。
         
         本书是由海归知名教育出版社倡议，5岁起即可阅读的图书，它将教你如何通过系统阅读、分析、综合阅读来获取知识，提升你的认知水平。
         
         本书共分五章，主要内容如下：
         
         - 一、阅读须知
         - 二、划定目标
         - 三、规划阅读过程
         - 四、建立阅读兴趣
         - 五、专注于阅读效果
         
         
         2、关于阅读的一切
         阅读一件事情，既要有目的又要脚踏实地，不能盲目跟随流行趋势而迷失方向。作为一个非虚构类文学爱好者，阅读带给我的启发多过感动。
        
         阅读能够帮助我们从不同角度审视事物，洞察问题的本质。把握历史脉络、社会现实、现代文明，甚至哲学思想。阅读既可拉近我们心灵距离，也能促进思维融通，提高我们的能力。
        
         阅读并不是一件轻松的事情，而是需要坚持不懈、不断学习、细致入微的。如果只是被动地阅读，很容易陷入“读什么而停留在那里”，如果不抓住阅读的重点，就无法全面了解一项技能或理论。同时，阅读需要耐心，不能贪多嗦少，不然将导致疲劳，最终难以获得精彩的阅读体验。
         
         阅读一本书，应当根据自己的年龄、经验、兴趣、态度、目的等因素来决定阅读哪些章节，遇到什么时候该停下来，因为不停地阅读反而会毫无收获。阅读过程中还应该注意控制自己手中的信息源，避免走马观花，专注于一项任务或领域的细节，这样才不会错过重要的知识。
         
         除了“明知故问”，还有一条更务实的做法是“读万卷书、行万里路”。只有充分调研、阅读，我们才能洞悉这个世界，寻找生命的意义。
         
         3、目标与步骤
         普通的学习或工作学习，往往是件枯燥乏味、低效费时的活。在阅读这项技能上，除了用正确的方式读外，还有很多可以借鉴的方法。本书将教会你如何建立一套完整的阅读习惯，提升你读书的能力。首先，我们要定义阅读的目标。
         
         “阅读一本书”是最常见的阅读目标，但其实“阅读一本书”只是一个抽象的目标。如何更具体地确定目标？也就是说，你希望从书中学到什么？例如，你可能想知道作者的观点、故事背景、技巧、方法，或者学习他/她关注的主题，如科学、社会学、语言文字、艺术等。
         
         定义好目标之后，就可以制定计划。阅读计划是指对一本书进行系统阅读的步骤，即怎样分配时间、内容和顺序，以达到提高阅读质量、提升记忆力、塑造个人品牌的效果。具体步骤如下：
         
         - 分享：首先要与作者保持良好的沟通，分享自己对这本书的看法和感受。一方面，能够让作者和读者更快捷地交流；另一方面，也可以加深互相了解，为接下来的阅读打下基础。
         - 准备：确定好阅读的时间安排、目标页码、阅读材料等。一般来说，推荐一周内完成一次阅读，这样可以使自己有充裕的时间进行反省和总结。但是，不同的书籍长度不同，建议根据自己的实际情况调整阅读时间。
         - 整理：选取适合自己的笔记工具，记录书中的重点、知识点、生词、句子，便于后续查找。
         - 分析：阅读完毕后，应该清楚地体会到这本书的优点和缺点。如果你发现这本书中有任何漏洞、错误或偏颇之处，也应该及时进行回应，以提升自己阅读能力。
         - 核对：在阅读结束前，再次核对自己的记录，确保没有遗漏、没有错别字、没有语法错误。核对结束后，可以尝试将自己所学的内容与相关专业背景进行比较，重新梳理知识框架，提升整体认识。
         
         以上的阅读方式仅仅是一例，不可能涵盖所有阅读技巧。因此，我们更关心的是如何在阅读中获得自信和成长，而不是贪求功利。
         
         4、兴趣培养
         阅读是一件享受且有趣的事情。对一般的读者来说，兴趣往往来源于他们的生活经历、与朋友的交往、所在的行业、兴趣爱好等。但是对于有“读万卷书、行万里路”的意识的人来说，阅读自然是最有效的学习方法。
         
         如果您已经拥有了一定的专业技能，并且能够吸引到同类的书籍，那么阅读这项技能就非常重要了。通过阅读，我们可以了解到其他学科的研究成果，获得新的思考方式。更重要的是，阅读可以让我们面对真实世界、改善我们的社会关系、解决实际问题。
         
         有了兴趣后，我们还需要保持好奇心，不断地挖掘新鲜的知识，保持一颗敬畏之心。与此同时，我们还应该以开放包容的态度对待读到的内容，尊重不同文化背景、阶层、年龄层之间的差异，不要局限于某种形式。
         
         5、专注阅读
         当我们开始阅读一本书时，应该立刻投入，并且全神贯注。不要在感觉到厌烦的时候停下来，要随时牢记自己阅读的目的。阅读一本书并不等于仅仅阅读，更多的应该是在阅读的过程中，去体会、去理解、去思考、去创造。
         
         阅读一本书的过程也不止是听、读、背、默。它包含了不同形式的阅读活动，包括观赏、思考、提问、写作、游戏、做选择……通过提炼、复述、批判性地分析，我们可以获取更多的信息。当我们对某一个观点、命题产生疑惑时，可以通过反馈、批判来证伪、辩护。
         
         把注意力集中在阅读的过程，不要去娱乐、socialize，尤其要避免进入情绪的漩涡，避免过多地对自己或他人计较。保持专注阅读，可以帮助我们保持理性、冷静、有条理的思维模式，从而更好地理解事物。
         
         最后，除了坚持阅读，还有许多其他的方法可以帮助我们更有效地学习。比如，可以查阅参考资料，借助互联网、工具、课堂等平台进行线上学习，或参与到各种活动中，帮助自己建立专业知识体系。
         
         6、未来展望
         “阅读一本书”并不意味着一蹴而就，它只是一种学习方式。在阅读过程中，我们可以通过自己的方式理解信息，提升自身的能力。未来，我们需要继续努力，不断探索、完善阅读技能。在取得长足进步的同时，我们也应该记得，阅读并不只是为了获取知识，也有助于我们培养独立思考能力、沟通表达能力、解决问题能力等。
