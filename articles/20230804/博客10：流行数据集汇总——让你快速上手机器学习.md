         # 2.数据集分类
         数据集按训练数据规模分为小数据集、中等数据集、大数据集三种类型。其中小数据集主要用于实验验证，比如线性回归算法、K近邻算法等；中等数据集一般是一些基础数据集，如MNIST手写数字数据库、CIFAR-10图片分类数据库；大数据集则是海量数据集合，如ImageNet图像识别数据库等。
         ### 小数据集（如IRIS）
         #### Iris数据集
          Iris数据集是最简单的小数据集之一，共有150条记录，包含三个类别（山鸢尾、变色鸢尾、维吉尼亚鸢尾），每个样本的特征有四个，即花萼长度、宽度、花瓣长度、花瓣宽度。这是一种经典的线性回归数据集，可以用来测试机器学习算法性能。
            ```python
            import numpy as np
            from sklearn.datasets import load_iris

            iris = load_iris()
            X = iris.data     # 特征值
            y = iris.target   # 标签值

            print("Feature matrix shape:", X.shape)
            print("Target vector shape:", y.shape)

            print("First five rows of feature matrix:")
            print(X[:5])

            print("First five labels:")
            print(y[:5])
            ```
            输出结果如下:
            ```
            Feature matrix shape: (150, 4)
            Target vector shape: (150,)
            First five rows of feature matrix:
            [[5.1 3.5 1.4 0.2]
             [4.9 3.  1.4 0.2]
             [4.7 3.2 1.3 0.2]
             [4.6 3.1 1.5 0.2]
             [5.  3.6 1.4 0.2]]
            First five labels:
            [0 0 0 0 0]
            ```
         ### 中等数据集（如MNIST）
         #### MNIST数据集
          MNIST数据集是最常用的计算机视觉数据集之一，由美国国家标准与技术研究院（National Institute of Standards and Technology）开发，用于手写数字识别任务。该数据集包含60,000张训练图像和10,000张测试图像，每张图像都是28x28像素的灰度图，共有十个分类。

         ### 大数据集（如ImageNet）
         #### ImageNet数据集
          ImageNet数据集是目前最大的图像识别数据集之一，包含约14万张训练图像和约5万张测试图像，每张图像都有固定的大小224x224像素，属于中等规模。该数据集包含超过一千类的物体，且训练数据包含220万张图像。
