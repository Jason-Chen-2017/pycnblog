# 1.简介
         
20世纪70年代末80年代初，欧洲经济危机发生后，许多企业家、投资者、学者纷纷从传统的价值投资转向市场中无风险套利、单边赌博等新的投资方式，创造出了一种叫做“无风险套利”（hedge funds）的投资模式。
         1987年，美国经济学家约翰·罗宾逊提出了一个著名的证券分析方法——“巴特拉法则”，他认为任何两个资产之间的价格关系都可以用一条直线来表示，称为“巴特拉曲线”。这个发现极大的推动了股票市场的走向。
         1994年，香港证监会发布了一项决议，规定股市交易金额不得超过资本净额的四分之一。在此之前，很多投资者对超额配售（overpriced sale）或者短期内暴涨暴跌的股票持批评态度，担心损失惨重。
         2007年，美国证券交易委员会（SEC）正式启动了制定反垄断指导方针的工作。该委员会指出，金融科技领域的企业由于获得巨额利润，其竞争优势也越来越弱化。因此，为了促进竞争及降低成本，各国政府在执法监管方面采取积极举措，其中最重要的一条就是保护知识产权。同时，各地监管部门也根据国家财政收入、商品和服务质量、法律要求、商业模式等综合因素，对各类金融企业进行分类管理。
         2013年，英国政府启动“GDPR”(General Data Protection Regulation)立法，将“一般数据保护规则”引入市场经济。尽管GDPR的条款非常复杂，但它对个人信息和其他隐私数据的保护具有根本性影响。
         在当今社会，互联网已经成为每一个人的生活不可缺少的组成部分。然而，如何保护个人信息并维持网络安全一直是个难题。因此，谷歌、Facebook、亚马逊等公司通过加密传输技术、防火墙和VPN等安全措施，极大程度上保障了用户的数据安全。但是，即使是在高度数字化的互联网时代，犯罪分子仍然可以窃取用户信息，甚至破坏网络的正常运作。为此，一些国家和组织开始探索新的解决方案。
         日本、德国、意大利、中国等国家及机构已经形成了一些新的制度框架和标准，试图通过技术手段或法律法规的方式，保障个人信息安全。其中，日本推行的GDPR就是一例。美国、欧盟、加拿大等一些国家也提出了一些关于保护个人信息的法律。但同时，法律总体上还是有很大的模糊性，容易被滥用。为了更有效地保障个人信息的安全，研究者们借鉴了加密技术、数据分类、黑客攻击、云计算和区块链等新兴技术。他们提出了一种新的框架——“Logarithmic portfolios framework”，其主要思想是：通过优化组合中的持仓比例，来平衡市场风险和投资回报。
         # 2.基本概念术语说明
         ## 2.1 组合优化
         “组合优化”是指在一个给定的限制条件下，找到一种投资策略，使得在某个特定时点收益最大化。例如，假设一个人要决定买入哪些股票，那就可以把这个人视为一个组合优化模型的求解者。组合优化模型的目标是找出一组股票的配置，能够使得这个组合的总收益最大化。不同的模型对组合的定义可能不同，比如，某些模型只考虑固定收益率的股票；另外一些模型允许加入非固定收益率的股票，如债券等。
         ## 2.2 Logarithmic portfolios framework
         “Logarithmic portfolios framework”是由<NAME>等人于2016年提出的一种新的投资组合优化模型。它的基本思想是，把一个单独的资产视为一个资产组合的第一个资产，然后依次添加第二个资产，每个资产的权重都是1/k，其中k是指资产的顺序。这样构造的资产组合是一个等权组合，且第一个资产的权重是1/2。
         当资产组合的容量为n时，资产i的权重是1/ni。根据这一模型，我们可以通过设置一系列的收益率目标，来优化资产组合的权重，从而达到最佳的投资效果。
         这种模型的一个重要特性是，它把每个资产的权重都看作是相同的，这就保证了资产之间所占份额的均匀性。虽然资产间的收益率存在差异，但它们之间的绝对差距不会影响最终结果。
         ## 2.3 Portfolio risk and return
         无论何时，一个投资组合的风险或收益都可以用以下形式表示：
        R = Σiwi x Ri - Σiwc xi
        r = ln(R/Rf), where Ff is the risk-free rate
        
        w: weights of the portfolio, i ∈ [1, n]
        R: total return of the portfolio (return on investment minus transaction costs).
        r: excess return of the portfolio relative to a benchmark such as the risk-free rate.
        
        上式左侧表示组合的总收益，右侧表示组合的夏普比率（Sharpe ratio）。上式中，wi是资产i的权重，Ri是资产i的预期收益率，xci是资产i的超额收益率。
        一旦知道了所有资产的预期收益率和超额收益率，就可以计算组合的风险和收益。由于组合的权重是相同的，所以组合的风险等于每个资产的风险的和。而且，由于组合的风险等于各资产风险之和，所以夏普比率也是各资产超额收益率之和除以各资产风险之和的自然对数。夏普比率是一个测度组合表现优劣的可靠指标。
        ## 2.4 Transaction cost
        涉及资产组合的任何交易都需要付出一定成本。这包括印花税、佣金、信用卡费用、支付宝手续费、托管费等。transaction cost可以用T表示：
        T = Σtci * wi, tci ∈ [1, c], ci is the cost per share or transaction fee for each asset in the portfolio.
        投资组合优化模型的一个关键参数是tci，它代表每笔交易的交易费用。交易费用应该在整个组合内保持一致，否则组合的效用就会受损。
