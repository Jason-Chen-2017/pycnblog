# 1.简介
         
4月底参加了王者荣耀联赛北京冬奥会首个比赛的实况直播，感觉真的是非常惊艳！为了让更多的人知道这个消息，特地采访了几个不同领域的优秀好友，做了一系列的互动。本期节目邀请到“创造营”成员王烁，李明轩，张钧甯，小林杰和刘梦如，他们都是不错的好友，分别从事音视频相关的工作。
         
         今天小编和大家聊聊：“请问你们朋友们看什么书，听什么歌？”让我们一起感受一下身边人各自的爱好吧。
        
         “为什么要跟大家分享一下自己的爱好呢？”小编觉得，作为一个技术人，“分享”这样的行为无疑是最有意义也最有价值的。能帮助到别人的那种亲切感真的很重要。希望大家能够欣赏到自己的爱好，从中获得收获，共同进步。
         
        # 2、关于我的爱好
        
        小编：“我喜欢看一些编程方面的书籍，例如算法导论、数据结构、计算机网络等。还有些技术类的综合性书籍，例如架构思维、编程珠玑、设计模式、代码整洁之道等。对于设计领域的书籍，像《设计原则》、《用户体验要素》、《交互式设计四策略》这些我都会比较关注。
        
        除了技术书籍外，我还喜欢听音乐。平时喜欢听流行音乐，有时候会听古典音乐。热爱各种电子音乐，最近比较喜欢海顿竞技场中的吉他音效。听过的歌曲多半是来自于纯正爱尔兰、英伦等文化，但我对任何一种音乐都不会完全陌生。例如，喜欢听海顿的《猫》、莫扎特的《第九交响曲》、邓丽君的《告白气球》。
        
        除了音乐，我也喜欢看电影。以前比较喜欢老片子，现在看日剧、韩剧、泰剧、欧美剧已经习惯了。最近一直看一部老船戏《蝴蝶效应》。也尝试了解日本动画、动画片的制作过程及风格。所以说，不管你喜欢什么类型的内容，都可以通过电视、书籍、电影等形式给予自己不同的沉浸感。
        
        在职业方面，我喜欢学习新知识并且有一颗充满求知欲的心态。兴趣使然，我也喜欢从事各种领域的创新工作。例如，在工作之余，我喜欢阅读历史文献、旅游、饮食。特别是旅游，我相信每个人的心灵都饱受其所。
        
        最后，在生活中，我也很享受创造力。无论是用手机记录美好瞬间、还是编织快乐幻想，我都觉得享受其中。因此，无论是看书、听歌、写代码，还是画画、DIY，我都十分向往。我希望我的朋友们可以欣赏到我的爱好，从而提升自身的能力，实现人生目标。
