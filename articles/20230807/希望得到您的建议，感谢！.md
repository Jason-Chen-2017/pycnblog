# 2.基本概念介绍

## 2.1 数据

机器学习的关键在于构建一个能够从输入数据中学习并预测输出的模型，这个模型建立在众多已知数据的基础上。这些数据通常被称为“样本”（Sample），每个样本都有一个特定的输入值（称为“特征”（Feature）或“属性”（Attribute）），并对应一个输出值（称为“标签”（Label））。

## 2.2 特征

特征一般是指对输入数据进行抽象表示的方法，它可以是连续的也可以是离散的。对于连续变量，可以直接使用；对于离散变量，需要先进行编码转换。例如，假设有一条人的年龄作为输入特征，可以直接用这个数字表示；而性别则需要先把男女编码为0和1两个值，再用这个二进制值表示。

## 2.3 标签

标签是一个目标值，用于描述输入数据对应的真实值。比如，某个图像识别任务的标签可能是人脸、汽车或狗等；某股票价格预测任务的标签可能是股票的涨跌幅度等。

## 2.4 模型

模型是根据数据特征和标签训练出来的一系列规则，用来对新的数据进行预测。不同的模型有着不同的预测能力，但它们都具有以下共同特点：

1. 在学习过程中可以自动调整参数。
2. 可以处理不同类型的特征，如数值型、文本型、图像型等。
3. 对异常值不敏感。
4. 有能力处理多维数据。

## 2.5 训练过程

模型的训练往往分成两个阶段：

1. “学习”阶段：根据给定的数据集，模型试图找到最佳的表示形式和拟合参数。
2. “推断”阶段：应用训练好的模型对新数据进行预测。

# 3.机器学习模型介绍

## 3.1 分类

分类是最常用的一种机器学习任务。它由输入空间X和输出空间Y组成，输入X映射到输出Y的一个函数f(x)。分类器通常用于对离散的值进行分类，如是否为正例或负例。目前常用的分类器有：

1. k近邻算法（kNN）
2. 决策树
3. 朴素贝叶斯
4. 支持向量机
5. 逻辑回归
6. 神经网络

## 3.2 聚类

聚类是将相似对象或事件集合起来，形成组的过程。聚类的目的是使得相似的事物聚集在一起，而不像分类一样，把所有事物分成完全不同且彼此之间没有任何联系的类。聚类算法通常包括凝聚层次聚类、流形学习、层次聚类和基于密度的聚类等。目前常用的聚类算法有：

1. K均值聚类
2. DBSCAN
3. 层次聚类
4. 谱聚类

## 3.3 生成式模型

生成式模型是指以数据生成的方式来建模，通过观察数据的分布来预测未知数据。常见的生成式模型有：

1. 高斯混合模型
2. 混合模型
3. 泊松潜在分布
4. 局部加权回归

## 3.4 判别式模型

判别式模型是以数据之间的依赖关系来建模。它主要是由条件概率分布p(y|x)或者后验概率分布p(x|y)定义的，通过学习从输入空间X到输出空间Y的映射来进行分类、回归或其他预测任务。常见的判别式模型有：

1. Naive Bayes模型
2. 最大熵模型
3. 逻辑回归模型
4. 最大 margin 策略
5. 线性判别分析

## 3.5 迁移学习

迁移学习是指借助源域数据来提升目标域模型的性能。它可以克服来自不同领域的数据分布差异带来的问题。迁移学习技术有两种：

1. 共享特征提取：即两个领域使用相同的特征提取器，仅在顶层网络结构上进行微调。
2. 参数微调：即在两个领域共享权重参数，仅在输出层进行微调。
