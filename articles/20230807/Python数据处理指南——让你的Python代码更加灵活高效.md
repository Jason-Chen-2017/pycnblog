         # 2.基本概念及术语
         　　2.1 数据结构
         　　数据结构（Data Structure）又称数据组织形式，它是计算机存储、管理、运算、显示数据的方式。分为顺序表、链接表、树形结构、图状结构、集合结构、哈希表等多种数据结构。
         　　2.2 文件I/O
         　　文件I/O(File Input/Output) 是指输入/输出设备与信息之间的交换过程。即数据的输入与输出是存储在磁盘或其他永久性存储设备上的文件。
         　　2.3 算法
         　　算法（Algorithm）是指用来完成特定任务的一组指令、指令序列或者计算方法。算法代表着对客观事物的精确描述和对计算机的控制逻辑的指令集体系。
         　　2.4 模块化编程
         　　模块化编程（Modular Programming）是一种编程方法，它将一个大的系统分成不同的子系统或模块，并分别设计、编译、测试各个子系统。
         　　2.5 函数式编程
         　　函数式编程（Functional programming）是编程范式，它把函数视作第一等公民，并且是数学和抽象计算的主要手段。函数式编程强调程序要只关注结果，而不是过程。
         　　2.6 命令式编程
         　　命令式编程（Imperative programming）是一种编程范式，它指定程序应该如何更改程序状态，并依据具体命令执行一系列的更新。
         　　2.7 面向对象编程
         　　面向对象编程（Object-Oriented Programming，简称OOP）是一种编程模型，它将程序的状态（数据）和行为（函数）封装进对象里，并通过消息传递来进行通信和协作。
         　　2.8 对象
         　　对象（Object）是一种抽象概念，是指在运行时创建的各种实例，由属性和方法组成，每个对象都是一个独立的个体，可以通过消息发送给另一个对象，从而实现互动。
         　　2.9 方法
         　　方法（Method）是某个对象的行为，它定义了一个对象在特定的条件下能够做出的动作。方法就是对象所拥有的操作。
         　　2.10 属性
         　　属性（Attribute）是某些值的名称、特征、性质等，它可以赋予对象独有的特征和属性。属性的值可以随时被修改。
         　　2.11 语句
         　　语句（Statement）是指一段代码片断，它可以是一条完整的表达式或者是一些赋值、条件判断、循环语句等。
         　　2.12 程序
         　　程序（Program）是一系列语句的集合，它用于完成某个特定任务。程序由函数、模块、变量、运算符和数据类型构成。
         　　2.13 可变对象与不可变对象
         　　可变对象（Mutable object）可以修改自己的内容；不可变对象（Immutable object）则不能被修改。
         　　2.14 数组
         　　数组（Array）是多个相同类型的元素按照一定顺序排列在一起的集合。数组的长度固定，且所有元素均为同一类型。
         　　2.15 列表
         　　列表（List）是数组中的一种，是一种动态集合。列表可以改变大小，可以按索引访问元素，还可以组合不同类型的数据。
         　　2.16 元组
         　　元组（Tuple）与列表类似，但是元素不能再被修改。
         　　2.17 字典
         　　字典（Dictionary）是一种无序的键值对的集合。字典中的每一项由两个元素组成，第一个元素是键，第二个元素是值。键必须是唯一的，值可以重复。
         　　2.18 迭代器
         　　迭代器（Iterator）是一个可用来访问集合元素的对象。
         　　2.19 生成器
         　　生成器（Generator）是一种特殊的迭代器，它一次返回一个结果，并在每次调用next()的时候生成下一个结果。生成器可以用for...in循环来迭代。
         　　2.20 正则表达式
         　　正则表达式（Regular Expression）是一种匹配字符串模式的工具。
         　　2.21 JSON
         　　JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，它采用键值对形式存储数据。
         　　2.22 XML
         　　XML（Extensible Markup Language）是一种标记语言，用来标记和定义网络上的数据。
         　　2.23 CSV
         　　CSV（Comma Separated Values，逗号分隔值）是一种常用的纯文本格式，它是由一组用逗号分隔的值（通常是数字、字符或文本字符串）组成的数据记录。
