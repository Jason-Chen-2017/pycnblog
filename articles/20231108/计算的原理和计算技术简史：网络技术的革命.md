# 1.背景介绍


计算技术作为人类社会发展到今天的基础设施之一，其核心功能就是用计算机进行信息处理、数据处理和决策。而这一重要的应用场景也导致了计算技术从单纯的存储和处理信息的工具，逐渐演变成信息通信领域的一个基础设施，成为人类进入现代化社会不可或缺的一部分。
随着互联网的普及和信息技术的飞速发展，计算技术已不再仅局限于特定领域。计算机技术的发展使得各行各业都能接受数字化的数据输入，并通过计算和分析实现精准的反应。因此，计算技术正在经历从硬件设备到服务器端软件，再到应用软件接口层面的进化过程。

20世纪80年代初，美国政府曾经对全球网络建设进行过几次大规模投资，但由于缺乏专业的管理人员和技术支持，很难实施真正有效的网络建设项目。随着互联网的蓬勃发展，计算机科学家和工程师开始意识到网络技术的强大威力。他们借助电子技术、信息论、信号处理等方法开创了一个全新的网络建设时代。

20世纪90年代中期以来，新一轮的网络建设热潮席卷了全球各个角落。许多国家迅速转型为数字经济体系中的重点企业，涌现出了一批拥有丰富网络技能的人才。同时，美国政府也对网络建设领域进行持续的投入，投资国际级的高科技网络设备、网络服务、云计算等。

2000年左右，随着中国互联网的崛起，计算技术在中国也同样受到重视。截至目前，中国有超过6亿网民、覆盖超过8亿人口，是世界上互联网用户最多的国家。从2017年中国数字经济启动以来，中国已经成为全球第一大互联网市场。中国是构建数字经济的关键所在，也是许多国际领先的大国之一。

然而，随着网络技术的日益发展，出现了一些重大的突破性变化。例如，2008年，Facebook发布的“扎克伯格计划”，将通过建立具有强大影响力的社交媒体平台，获得巨额的商业收益。另一个重大的突破则是在2013年，Google推出基于云计算的搜索引擎，使得人们可以轻松地浏览各种类型的文件、网页，以及其他需要快速查找的资源。这些突破性变化都充分证明了网络技术的巨大潜力。

20世纪末，随着全球产业结构的全面转型，计算技术也正在转向新的方向。2010年底，麻省理工学院的<NAME>教授发表了一篇题为“计算的终结”的文章，提出了计算终结的观点。他认为，随着物联网、工业机器人的发展，以及人工智能、深度学习、区块链等新兴技术的普及，计算技术会遇到越来越严峻的挑战。

# 2.核心概念与联系
## 2.1计算技术的定义
计算技术（Computing technology）由两大支柱组成：运算技术和存储技术。运算技术利用电脑中集成的各种电路逻辑单元和控制装置，按一定顺序和规律对数据进行运算；存储技术负责数据的保存、检索、传输和保护。计算技术的特征是能够按照指令进行自动、高效、准确的执行，并产生计算结果。除此之外，计算技术还包括相关软硬件产品、编程语言、网络协议、标准、协议和方法。


## 2.2网络技术的特点
网络技术的特点主要有以下几方面：
### 共享资源
网络技术依赖于共同的网络资源，如互联网连接的路由器、交换机等设备。不同的网络设备之间共享相同的通信信道，这样就可以实现不同设备之间的通信传输。无论是内容服务还是信息服务，网络技术都可以满足不同用户的需求，使各类服务用户享受到更高质量的服务。
### 分布式计算能力
网络技术可以在分布式环境下运行，这意味着任何一台服务器都可以独立计算任务。因为所有的计算任务都由多个服务器共同完成，所以用户可以获得更快的响应速度和更可靠的服务。此外，网络技术也可以扩展到大规模的集群系统，提供更高的可用性。
### 网络技术的延迟性
网络技术通常存在较高的延迟性，原因主要有两个方面：
- 网络本身的特性
网络技术的传输方式往往采用非对称的形式，即某些节点比其他节点传输信息的速度要慢。因此，当一个消息从发送端发送到接收端时，消息会经过多个路由节点，这个过程可能需要耗费很多时间。另外，网络技术又经常发生冲击波，意味着网络的某些节点会成为瓶颈，无法承载流量。
- 服务质量保证
为了保证服务的质量，网络技术会采用各种手段来检测和控制传输的质量，比如超时重传、流控机制、拥塞控制算法等。

## 2.3网络技术发展历史
### 2.3.1信息革命时期
信息革命时期是网络技术发展的早期阶段。在信息革命之前，主要依靠各个城市之间的边界线进行信息传递，并且每隔一段时间就要搞一次大范围的重大战役。在这种情况下，使用传统的物理线缆或电话线缆进行信息的传递，由于速度慢，很容易出现网络堵塞的问题。因此，在信息革命时期，人们对网络技术的需求激增，开始寻找新的方式来实现信息的共享、传递、处理和存储。

### 2.3.2互联网的出现
互联网的诞生是网络技术的重要里程碑事件。互联网是由一系列的计算机网络设备相互链接而成的庞大网络，用来连接因特网上的不同网络设备。它既可以作为信息共享平台，也可作为信息源和信息接收者。早期的互联网主要由多个大学、科研机构、公司、政府等组织共同开发维护。其中，美国政府的NSFNET（美国国防部网络）是互联网的先驱，其后来成为TCP/IP协议族的基础。

### 2.3.3万维网的出现
万维网（World Wide Web，WWW）是第一个用于普及网络信息技术的网络服务，它由超文本标记语言HTML、互联网扩展Markup Language（IML）、Cascading Style Sheets（CSS），动态脚本语言JavaScript、基于HTTP协议的Web服务器组成。由于网络技术的发展，WWW越来越成为人们获取、浏览、发布信息、协作工作、购物消费的主要渠道。
