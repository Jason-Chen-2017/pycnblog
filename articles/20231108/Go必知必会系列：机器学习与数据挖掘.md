# 2.核心概念与联系
## 概念理解
首先，了解机器学习、数据挖掘相关的概念对了解和应用技术有非常大的帮助。下面给出一些关键词的基本概念和联系：
- **特征工程（Feature Engineering）**：特征工程是指利用原始数据构造新的、有效的、有意义的特征变量，用于训练模型进行预测或分类。包括特征选择、特征变换、缺失值处理等。
- **监督学习（Supervised Learning）**：监督学习是机器学习中的一个分支，它假设输入和输出之间存在某种关系，即输出是由输入决定的。监督学习的任务就是找到这种关系，也就是学习到输入和输出之间的映射关系。分类、回归和聚类都是属于监督学习的范畴。
- **无监督学习（Unsupervised Learning）**：无监督学习是指没有明确的输出结果，只知道输入的情况下进行学习，目的是发现数据内隐藏的结构。比如，聚类就是一种无监督学习的方法。
- **半监督学习（Semi-supervised Learning）**：在有部分标签数据的情况下训练模型，用以解决样本不均衡的问题。例如，有部分样本标记清楚，而另一部分没有标记，则可以使用半监督学习方法来训练模型。
- **增量学习（Incremental Learning）**：通过不断收集新的样本数据，在不断更新模型参数，从而实现模型的持续学习。增量学习通常采用迭代式的学习过程，每次只用较少量的数据进行一次迭代，逐渐增加模型的规模和能力。
- **欧几里得距离（Euclidean Distance）**：欧几里得距离是空间中两个点距离的一种度量。它是一个从直角坐标系到平面直角坐标系的度量方式。
- **聚类（Clustering）**：聚类是无监督学习的一种方法，用来将相似的数据集划分为多个簇，使每个簇内部的数据相似度最大化，不同簇的数据相异度最小化。
- **邻居（Neighborhood）**：如果一个样本点被划分到某个簇中，那么该样本点的邻域（Neighborhood）内的其他样本点也被划分到这个簇中。
- **样本（Sample）**：机器学习中的样本指的是数据的单个实例，可以是某个物品、用户、评论或者其他事务。
- **标记（Label）**：在监督学习过程中，标签就是用来表示样本与其对应的真实结果。对于分类问题来说，标签可以是离散的或者连续的；对于回归问题来说，标签就是目标变量的值。
- **超参数（Hyperparameter）**：超参数是机器学习算法的模型参数，如贝叶斯估计中的先验分布的参数。它们不是根据训练数据自动确定，需要人工指定。
