# 1.背景介绍


## 时序数据的特点
时序数据通常由固定频率的数据采集记录得到，其特点如下：

1. 不间断：每个样本的时间戳都对应于一个确定的绝对时间或相对时间；
2. 可变：每天、周、月、年等不同时间周期的数据都会发生变化；
3. 有序：可以用先后顺序排列，前一个样本与当前样本之间的时间差不超过某个固定的时间间隔；
4. 不可预测：每个样本的值都不是独立生成的，会受到历史数据影响。例如股票价格、气温变化、市场指数波动、系统性能指标等。
5. 复杂性：时序数据往往具有多个维度，比如物理量、观察值、预测值等，要想分析并处理这些复杂性数据，就需要进行多维数据处理。
## 时序数据的常见形式
最简单的时序数据形式就是一系列的连续数字，每个数字代表着某个特定时刻的时间步长内某个观察变量的值。如：

2, 7, 12, 15, 9, 5, 1, -2, 4, -3, -6

这样的一串数字序列，就构成了单个时间序列。当然，时序数据还包括更多的形式，例如：

1. 数据整合与合成：原始数据可能包含不同来源、不同形式的数据，需要进行整合和合成才能获得结构化的信息；
2. 异常检测：时序数据中的异常值往往具有特殊意义，如财务数据、运营数据等。通过识别异常值，我们可以发现隐藏在数据中的模式，从而找出潜在的商业机密，甚至对政策制定产生重要影响；
3. 预测和监控：在时序数据中，我们可以进行回归分析、ARIMA（自回归移动平均）、VAR（VECTOR AUTOREGRESSION）等模型预测变量之间的关系，并根据预测结果对系统进行优化或调整；
4. 滤波和重构：时序数据经过一些处理之后可能会丢失一些信息，或者得到一些降维后的特征向量。这时候，我们就可以利用滤波和重构技术对数据进行修复，提高分析效果。
5. 分类和聚类：对于时序数据，我们可以将其划分成几个子集，然后基于不同的特性进行分类和聚类。如，我们可以将不同时间段的股价数据按时间段划分成几个子集，然后基于不同子集进行多元分析，找出不同风格的股票组合。
## 时序数据的基本任务
时序数据的基本任务有以下几种：

1. 抽取：获取某个时间段的数据，如日线、周线、月线等；
2. 合并：将不同数据源的数据合并成同一个时间序列；
3. 转换：将不同时间单位的数据转换成统一的时间单位；
4. 清洗：去除噪声、缺失值、异常值等；
5. 编码：将文本数据编码为数值数据；
6. 切片：获取某个时间范围内的序列；
7. 计算统计特征：统计时序数据的最大值、最小值、均值、方差、标准差、偏度、峰度等；
8. 模型构建：训练时间序列模型，如ARIMA、LSTM、GRU、Kalman Filter等；
9. 模型应用：使用训练好的模型对新数据进行预测或分析。
# 2.核心概念与联系
## 术语解释
- **时间(time)：**表示某一时刻的时间点或持续时间。在时间序列分析领域，通常把连续的时间点称作时刻，而离散的时间点或间隔称作间隔。
- **时间跨度(time span)：**在时间序列分析中，指的是时间的一个范围。比如，两个时刻的差距叫做时间跨度。
- **时间序列(time series)：**在时间序列分析中，时间序列是一个或一组随时间而变换的数据点的集合。它由若干个时刻和相应的观察值组成。比如，股票的收盘价、期货的中间价、传感器测量值的温度等。
- **时间间隔(time interval)：**在时间序列分析中，表示两个时间点之间的时间差。时间间隔一般指的是单位时间长度的倍数。比如，日线的时间间隔就是1天。
- **时间序列分析方法(time series analysis method)：**包括趋势检验法、AIC、BIC、卡尔曼滤波、ARIMA模型、马尔可夫链蒙特卡罗方法、隐马尔科夫模型、贝叶斯方法等。
## 基本假设
- 线性时间系列：时间序列是一条随机游走的过程，即任一时刻$t_i$处的值只依赖于它之前的一个时刻$t_{i-1}$的值，且该值不能被无限早的时刻影响。
- 平稳时间序列：时间序列平稳（白噪声）或平缓地变化，即任意一个小的时间间隔内的随机游走曲线看起来都相同。
- 滞后时间序列：时间序列有滞后性，即时间序列的最后几个观察值会受到更早时刻观察值的影响。
- 有效观察次数：一个时间序列的有效观察次数是指这个时间序列能够准确反映出系统行为所需的最小观察次数。
