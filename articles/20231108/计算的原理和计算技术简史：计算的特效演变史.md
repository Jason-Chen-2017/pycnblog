# 1.背景介绍


随着技术的不断进步，计算领域也在不断追赶人类在各个领域的飞跃。作为计算机发明者之一杰克逊·戈尔曼（JohnGraham）在1946年开创了计算这个行业。但是，计算行业远没有达到像图形图像、电子游戏这样的高科技行业那样的成熟阶段。因此，计算的理论基础还是比较薄弱的。本文将从计算的历史脉络出发，详述数字计算机的发展过程，从晶体管时代到今天都有哪些重大的突破性技术革新。另外，本文还将讨论计算对社会的影响，比如计算技术可以让个人生活发生翻天覆地的变化。  
# 2.核心概念与联系
“计算”一词的产生有很长的历史渊源。最早是由皮尔逊·约翰·麦卡锡（PierreMcCurry）提出的。他将数据视作能被计算机所识别的信息，并将其转化为能够执行某种特定任务的指令。约翰·范布伦特（JohanFredkin）、亚历山大·汉密尔顿（ArthurHimmler）等计算机科学家基于计算理念，提出了很多重要的概念如编程语言、自动机、组合逻辑、图灵机等，都源自于此。  
1947年，布莱恩·柯林斯（BertrandMclinnies）提出了人工智能的概念，并在此基础上提出了三层机器学习的框架，即输入层、输出层和处理层。处理层负责对输入信息进行处理，如模式识别、决策分析、归纳推理等；而输入层则负责接收外部输入信息，输出层则负责提供给外界反馈信息。这个理论后来被称为哥德尔功绩定理（Godel'sIncompletenessTheorem）。  
1948年，艾伦·图灵（ErnestE.Turing）用“图灵测试”衡量计算机的能力。这个测试向计算机出一组简单的问题，希望它能准确回答是否正确。之后，人们便将这个测试运用于实际应用中，证明图灵机比人类更聪明。  
1957年，丹尼斯·沃尔特·皮凯蒂（DonaldWinstonPike）提出了著名的冯诺依曼结构。这个结构指的是计算机由五大部件组成：运算器、存储器、控制器、输入设备、输出设备。其中，运算器负责执行算术运算，存储器负责保存数据，控制器负责调控整个系统，输入设备负责获取信息，输出设备则负责显示结果。  
1964年，爱迪生发表了著名的“电子计算机原理”。这篇文章主要讨论了二进制编码和运算过程。另外，当时的计算机性能也相对差一些。  
1965年，约瑟夫·冯·诺伊曼（VonNeumannJr.vonNeumann）发明了“图灵机”，这是一种抽象的计算模型，其设计目标是让机器执行一般的计算任务。该模型的特点是只有两种状态——“1”和“0”，运行速度快，但缺少可编程性。  
1968年，丹尼斯·里奇（DanRitchie）提出UNIX操作系统，使得计算机拥有了自己的文件系统、命令解释器等功能，使计算机的功能越来越强大。  
1970年，托马斯·阿兰基米德（ThomasAlanKay）和约翰·波卡尔（JohnBarsky）开发了图灵完备的语言，将图灵机推广到任意计算模型中。同时，贝尔实验室的克劳迪娜·莫斯利（BellLabsMarvinMoss）建立了首个商用微型计算机—TRS-80。  
1974年，威廉姆·李德（WilliamLoyd）和罗伯特·麦克斯韦（RobertMontyCarlo）发明了FORTH语言，其是一种堆栈式机器码编程语言，具有极小的内存占用率，是最早实现编译器的语言之一。  
1977年，巴斯德·安德烈欧·西蒙（BorisE.SchmidtSmith）发明了LISP语言，其是在图灵完备编程模型上的方言。  
1978年，康雷尔·麦克唐纳（KurtCalvin）提出分布式计算的概念，将多个计算机连接起来，并让它们协同工作，完成复杂的计算任务。  
1979年，肖尔斯·霍夫曼（JeanH.Huffman）发明了哈夫曼编码，采用二叉树的方式对信息进行压缩，压缩后的信息可以更好地存储和传输。  
1981年，艾伦·库珀（AronUllman）发明了抽象代数和集合理论，并对计算机科学产生深远影响。
