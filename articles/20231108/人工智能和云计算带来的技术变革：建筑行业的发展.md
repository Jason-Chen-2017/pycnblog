# 1.背景介绍



随着“互联网+”、智能城市、智慧农业等高科技产品的不断涌现，以及经济飞速发展的时代背景下，建筑领域也进入了全新的技术发展阶段。2019年，我国建筑业总产值突破千亿美元，实现了12%左右的增长率，占GDP比重超过8%。由于技术飞速进步的同时，建筑产业的创新能力也在不断提升。

根据国家统计局2017年的估算数据，2025年我国建筑企业将达到1.2万个以上，规模将达到3.3万亿元以上，这些数字仍然令人鼓舞，而这也带动着建筑领域的技术革命。根据新浪财经预计，2025年，中国制造业新增就业岗位将达到1.1万万个左右。也就是说，每一个新增的就业岗位都将带动制造业生产的能力增加百倍，对整个产业链产生更加惊天的效应。

随着人工智能、大数据等技术的应用普及开来，建筑行业的机器人化、自动化和智能化正在成为可能。在这个过程中，建筑行业将逐渐从以前单纯的拆装建材向更加智能化、精益化方向迈进。

2017年，谷歌宣布其AlphaGo战胜围棋世界冠军，这标志着人工智能的雄风已经席卷建筑领域。据最新消息显示，阿里巴巴集团主导推出的无人机一直在助力房地产行业做更多的事情，打造成爆炸性的优质住宅。同时，贝壳找房也利用人工智能技术为业主提供更加精准的房源推荐。随着中国制造业的蓬勃发展，中国建筑工地正在走向自动化，甚至成为未来建筑工地的标配。

2018年，腾讯宣布其微信支付宝平台助力小微商户入驻线上，这也是建筑业自发追赶互联网金融潮流的一种方式。为了让更多的企业参与到新一轮的智能建筑建设中来，建筑行业正逐步实施模式转型，走向更多的全场景智能化。

综合来看，建筑业面临着技术革命、产业升级、创新驱动、社会变革的诸多挑战。需要科研人员和工程师结合实际情况研究和攻克其中一些难题，把建筑业的发展变成一个高效、可持续的、科技强、绿色环保的行业。否则，随着技术的发展和应用，建筑业的未来仍将是一个迷茫的地方。因此，我国建筑行业要始终坚持以人为本，用科技赋能社会变革，促进建筑业发展，推动建筑业的科技创新和产业变革，并充分发挥示范引领作用。
