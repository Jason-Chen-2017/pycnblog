# 1.简介
  

Arm 公司是一家全球知名芯片设计、制造和生态系统企业，其 ARM 处理器系列由高端、中端和旗舰级产品组成。其 Mali 系列芯片由高性能图形处理单元（GPU）、机器视觉处理单元（VPU）、网络处理单元（NPU）以及安全计算单元（SCU）四个产品系列组成。这些系列可用于许多高层级应用场景如移动设备、影音娱乐、电子游戏、互联网服务等。
近年来，ARM 处理器领域的进步引起了硅谷的极大关注。截至目前，Mali-T880、Adreno 640、Mali-G76、Mali-G77 和 Adreno 655 GPU 都已经正式进入了市场销售。同时，英伟达也宣布推出三星独有的 Arm Mali-G78 MP12 处理器，并表示将在 12 个月内向市场推广该处理器。
本文主要讲述了英伟达推出三星 Arm Mali-G78 MP12 处理器的基本情况、核心概念及其特性，还有三星特有的特性，最后对它未来的发展展望做出展望。希望通过这个专题报道，能够帮助读者理解英伟达最新技术动态，更好地把握时代脉动，为自己的工作、学习和生活提供更多的参考价值。
# 2.基本概念术语说明
## 2.1 Arm 公司
### 2.1.1 Arm 概况
Arm 是一家全球性的芯片设计、制造、分销和生态系统服务企业，其历史可追溯到 1969 年贝尔实验室创立，随后逐渐发展成为全世界最大的科技企业之一。Arm 作为一家全球性公司，拥有强大的研发能力和精湛的工程技艺，通过其各项业务，从基础芯片制造、授权分销、第三方芯片验证、全球合作伙伴关系管理等各个环节为客户提供优质服务。截止 2018 年底，Arm 在全球拥有约 1.7 万家客户，其中超过七成以上都是国际化的消费者。

### 2.1.2 Arm 产品目录
Arm 提供着各种级别的核心处理器系列和相关配套硬件产品，包括图形处理器系列 Mali、机器视觉处理器 Vivante、网络处理器 Nvidia 旗下 Kepler 网络处理器系列、独立网络处理器 Arm AMBA NXP 和 Arm Semihosting 等，总体产品种类覆盖了图形图像、计算机通信、机器视觉、神经网络、多媒体、传感器、认证、工控等多个领域。其中图形处理器 Mali 为 Android 系统的基础架构，被广泛应用于手机、平板电脑、穿戴设备、云平台等行业领域。Mali 的 GPU、VPU、NPU 和 SCU 产品系列在图像处理、机器视觉、AI 推理等领域均占据重要地位。


图：Arm 公司相关产品结构图

Arm 还针对不同的终端用户群体进行定制开发，如嵌入式系统的专用处理器、消费级笔记本的 CPU 或微控制器产品、台式机服务器的主板产品等。除此之外，Arm 还与博世合作，推出基于 Arm 架构的 Synopsys FPGA。Synopsys FPGA 是高端设计和实现专用的逻辑功能块阵列，可快速构建复杂且高速的数据处理系统。Synopsys 的首席执行官陈越男先生表示，Synopsys FPGA 可为客户提供高端的电路级设计能力，而且价格相对于其他常规开发平台来说较低廉。

## 2.2 Mali 系列
### 2.2.1 Mali-G76
Mali-G76 是英伟达第一款开源图形处理器，是英伟达 2013 年发布的第一款 GPU，性能强劲。其架构也十分简单，采用 CISC（精简指令集计算机）架构，在性能、功耗和价格之间取得了很好的平衡。但是由于采用精简指令集，无法充分发挥高端芯片的性能，只能满足一些特定的使用场景。其外观采用全金属结构，具有独特的绚丽图案，画面质感超越任何竞争对手。


图：Mali-G76 芯片架构示意图

Mali-G76 在 2015 年便问世，它的主要特点是速度快、价格低，支持 OpenGL ES 2.0、OpenGL ES 3.0 和 WebGL，可以支持 Android 操作系统上的运行，也适用于 Linux、Windows、Mac OS X 等系统平台。不过，由于采用精简指令集，无法发挥出强大的性能，因此其算力仍处于比较初级的水平。

### 2.2.2 Mali-G77
Mali-G77 继承了 Mali-G76 的架构，引入了新特性。引入 WGP 架构，更好地利用了多核 CPU 资源，为 ARM Mali-G77-powered 手机提供了更流畅的视觉效果。而且，Mali-G77 引入了类似 Adreno 的 TMU 架构，使得它可以在手机上支持更复杂的任务。在同类产品中，Mali-G77 比较擅长处理视频渲染、3D 渲染等需求。


图：Mali-G77 与 Mali-T880 芯片架构不同之处

然而，即使是在 Intel 的三条龙组合之中，Mali-G77 也不及 Mali-T880，尤其是在深度学习领域。因此，英伟达在 2017 年宣布将停止为 Mali-G77 手机发布更新。此后，只有 Mali-G76 可以在 Android 系统上获得支持。

### 2.2.3 Mali-G78
英伟达近日宣布推出了三星自主研发的基于 Arm Mali-G78 MP12 处理器，这也是他们继 Mali-G76 以来推出的第二款独有芯片。Mali-G78 继承了 Mali-G77 的架构，同时兼顾速度和图形质量两个方面，支持 Vulkan、OpenCL 和 OpenGL 等图形 API。Mali-G78 也可以在运行高端芯片的手机上获得媲美顶级 GPU 的性能表现。Mali-G78 通过调优性能、优化功耗以及新增特性，更加贴近市场需求，并提升其图形性能。


图：Mali-G78 芯片架构示意图

英伟达最早于 2016 年就推出了第一个有四颗流派的 Mali-400 MP4 系列芯片。其后推出了 Mali-G76 和 Mali-G77 两款独立的图形处理器。Mali-G78 则是作为三星 Galaxy S10+、S10e、Note 10+、S10 系列等新一代手机的处理器。在这里，三星做出了巨大贡献，将三星自家的独特技术结合到了 Arm Mali 系列中。三星基于 Mali-G78 进行了内部优化，使得图形性能和功耗都有所提升，并提升了整体体验。

 ## 2.3 独有特色
 ### 2.3.1 Arm Mali-G78 MP12 核心特点
  - 自研设计，类似 Zynq 系列
  - 支持高效率网络协议如 TCP/IP、UDP/IP
  - 内置 2D 纹理处理单元 TMU
  - 使用最新一代的 A78 处理器
  - 支持 Vulkan、OpenCL 和 OpenGL 等图形 API
  - 运算单元支持加速加密、压缩、数据转发和 SIMD 指令集
  - 支持 “Lazy Litmus” 编译技术
  - 利用 HEVC 和 HDR10 编码技术
  - 可用于移动、桌面和嵌入式设备
  - 性能与功耗均优于 Mali-G76 和 Mali-G77
### 2.3.2 三星特色
#### 2.3.2.1 画中画（Picture-in-Picture）技术
三星 Galaxy S10+ 升级版支持的画中画技术，可以显示用户当前正在观看的内容的实时画面，用户无需退出当前应用即可继续观看，屏幕分辨率会根据用户的操作自动调整。


图：三星 Galaxy S10+ 画中画模式

#### 2.3.2.2 应用框架
三星 S Pen 笔记本上嵌入了完整的 Arm Linux 发行版，这意味着你可以安装所有常用的应用程序，并且与你的 Android 设备共存。此外，S Pen 可以使用额外的设备输入，例如带触摸屏的按钮、滑动球或圆柱形方向盘。


图：三星 S Pen 上嵌入的完整 Arm Linux 发行版

#### 2.3.2.3 更快的启动时间
三星 Galaxy S10+ 更快的启动时间让它在手机中的位置显得尤为突出。它的处理器有着性能高速的性能，可以为您快速打开应用和游戏。而且，除了省电外，它还支持更多的 Android 特性，比如快速响应的通知栏、划词搜索等。
