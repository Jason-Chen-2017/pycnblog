# 1.简介
  

## 1.1编程语言介绍
编程语言（programming language）是人们用来进行计算机编程的工具。编程语言定义了程序编写的方式、程序执行的过程以及各种数据类型及其处理方式。现如今，几乎所有主要的编程语言都已经出现并得到广泛应用，包括C、Java、Python等。常用的编程语言还有JavaScript、PHP、Swift等。

在编写程序时，程序员需要选择一种编程语言并采用它的语法规则，然后通过编译器或解释器将源代码转换成机器语言，才能运行在计算机系统上。目前，世界上存在多种编程语言，但并不是每一种语言都是最好的选择。不同语言之间存在着语言之间的差异性、运行效率上的差异、支持库和框架的差异、社区支持力度等方面的差别。因此，正确选取适合自己的编程语言至关重要。

## 1.2编程语言分类
根据编程语言的设计目标、开发环境、面向对象特性、编程范式、编译性质、运行性能、适用性、学习难度、生态系统支持度、工具支持度、热点技术变化情况等因素，可以把编程语言分为不同的类型。这些类型按照易学、易用、强类型、跨平台、高性能、标准化程度、并行性等属性可以划分如下：

1. 易学语言（Interpreted Languages）
    - Python、Ruby、Perl、JavaScript、Tcl、Smalltalk、Erlang、Haskell
    
2. 编译型语言（Compiled Languages）
    - C、C++、Objective-C、Fortran、Scala、Java
    
3. 静态类型语言（Statically Typed Languages）
    - Haskell、ML、F#
    
4. 动态类型语言（Dynamically Typed Languages）
    - JavaScript、Python、Ruby、Perl
    
5. 命令式语言（Imperative Languages）
    - SQL、HTML、XML、Groovy、Bash、Lua
    
除了以上分类外，还有基于actor模型的并发编程语言、脚本语言、解释型语言、标记型语言、Domain Specific Language（DSL）等。

# 2.基本概念术语说明
## 2.1计算机基础知识
计算机是由运算器、控制器、存储设备、输入/输出设备组成。其中，运算器负责对数字信号进行加减乘除、移位等运算；控制器负责指令的接收、顺序执行、错误恢复、中断处理等功能；存储设备负责保存数据和程序；输入/输出设备负责信息的输入输出。

计算机系统通常由主板、主芯片、硬盘、显卡、鼠标、键盘等组成，其中，主板即主处理器插槽，通常是集成电路板，用于连接各个部件；主芯片通常是一个小型计算机芯片，是整个计算机的运算核心；硬盘则是一个长期存储数据的介质，存储着操作系统、应用程序、文档、游戏等文件；显卡则是显示屏幕、声音、图像的设备；鼠标和键盘则是人机交互的接口。

计算机网络就是指将多个计算机设备连接到一起，使得它们可以通过通信互相传递信息、协作工作、共享资源等，从而实现计算、通信、娱乐等功能。

## 2.2编程语言概念
### 2.2.1编程语言层次结构
程序设计语言的层次结构一般由四个层次组成，分别是表达式层次、语句层次、声明层次、函数层次。

**表达式层次**是最低的层次，包括变量、常量、字面值、算术运算符、逻辑运算符、关系运算符、条件运算符等。

**语句层次**包括赋值语句、控制语句（循环语句、选择语句）、输入输出语句、函数调用语句等。

**声明层次**包括变量声明、常量声明、函数声明、类声明等。

**函数层次**则是最高的层次，由若干个函数组成一个程序。

程序设计语言的各层次的嵌套关系决定了程序的结构，而层次结构中的每个层次又对应于特定程序设计语言的语法结构。例如，表达式层次对应的语言有A+B*C、if (X>Y) {return A;}等。
