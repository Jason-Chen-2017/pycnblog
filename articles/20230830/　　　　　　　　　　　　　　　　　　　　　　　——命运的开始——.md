# 1.简介
  

“命运”是心灵、生意或职业上的必然事件。你是否曾经思索过命运，在某个点上需要做出抉择，却发现事情并没有按照你的预期走下去？也许你有过类似的烦恼，只是你没有找到原因或者解决办法。为了避免这种情况的发生，本文将从一个普通人的角度谈起，为那些因着迷茫而产生焦虑的朋友们提供一些必要的帮助和指引。本文并不会涉及太多高深的理论，只会借助一些生活中的例子，来告诉大家如何克服充满自我期望和自以为正确的焦虑感。

# 2.关于命运
首先，什么是命运呢？简单地说，命运就是一个突如其来的宇宙变化，天翻地覆、星辰变幻之类的。它影响到我们的很多方面，比如工作、婚姻、旅行、人生方向等。但是要想更深入地了解这个话题，需要先了解一下“随机性（randomness）”这一概念。

随机性是数学的一个分支，它研究的是物质世界中各种无序现象，包括但不限于运动、旋转、化学反应、重力作用等等。由于存在随机性，一切事物都有可能以各种方式演化，所以它被称为“随机性”。例如，我们说“如果今天下雨了，明天一定会下雨”，这句话中的“今天”和“明天”都是随机变量。另外，随机变量还可以由另一个随机变量经过一定处理得到。因此，随机性的影响非常广泛。

既然随机性是数学的一项重要分支，那么对于命运，就不能再用枯燥乏味的语言描述了。我们可以举个例子来说明。

假设我们住在某家公司里，负责公司内部的管理工作。该公司有10个人，他们平均每周都会召开例会，每个月会制定年度计划。假设今年刚刚到年终，临近发布，你作为其中一名员工，准备给自己列个计划。通常情况下，公司内部会根据年度计划分配任务，同时保持良好的工作态度。然而，由于管理人员都有各自的工作需求和工作时间安排，导致每个人的任务计划可能会有所偏差，导致出现下面这样的情况：

- A：“听说昨天晚上你跟同事吃饭，一起去游玩吗？”
- B：“我也要去参加聚会，不过我不知道你们要去哪里……”
- C：“你们公司有‘七夕’活动吧？要不要我来……”
- D：“最近我们很忙，有空的话，一起去散步吗？”
- E：“你还有什么计划吗？最近一直都是工作状态。”
- F：“下周五要开会讨论一下春节后的事情……”
- G：“你好久没回邮件了，有什么事吗？”
- H：“最近在看电影，有什么推荐的吗？”
- I：“你在公司工作时间有要求吗？”
- J：“我今天下午十点钟回公司，希望你们早点休息……”

如此种种，每个人的工作计划都无法按时完成。你越是看到这种情况，就越觉得有些沮丧，甚至害怕。为什么？因为这些工作计划都不是预先规划好的，而是在发生突发状况时才确定的，这就给公司带来巨大的压力和困难。最后，工作计划本身就是一种随机变量，它的变化又会影响到其他的随机变量。因此，你也许根本找不到自己理想中的工作，只能眼睁睁看着自己的工作计划在遭遇挫折。

那么，如何克服“命运”带来的巨大冲击？其实并不难。首先，要做到“沿着舒适圈往外生长”，不要受命运的牵绊。其次，不要任由命运的摆布。尤其是不要盲目乐观，把信念当作最坏的打算。第三，勇敢面对现实，对风险有自知之明，面对压力，主动承担责任。最后，不要妄自菲薄，寻求他人帮助。通过树立正向的价值观和责任心，大家都有机会成功，这也是每个成功者共同的经验之谈。

# 3.焦虑的来源
我们已经了解了什么是命运，以及为什么它会影响到人的行为。下面，我们来看看焦虑是如何产生的，以及如何通过一些简单的生活小技巧来减轻焦虑的影响。

## 3.1 内心恐惧症（Anxiety Disorder）
焦虑是人类社会普遍存在的正常现象，是人类高度易感、高度敏感的心理疾病。其原因主要有三种：

1. 想当然地认为自己能力强、条件优秀，或认为自己很聪明、很厉害；
2. 被生活所左右，包括情绪、欲望、家庭、朋友等等；
3. 缺乏理性思维能力，容易陷入思维误区，造成认识错误、判断失误、行动失误、决策失误等问题。

## 3.2 自我保护机制
焦虑主要是由于不知道应该如何面对命运的影响，因此，我们需要养成自我保护意识，不要让这种恐惧感引诱我们做出错误的决定。

- 建立安全感：摒弃对未知的恐惧，习惯冒险，适应变化。
- 拓宽视野：增强独立思考和分析能力，发现更多可能性。
- 控制情绪：降低情绪负担，避免自我暴露。
- 把握平衡：调整自己的欲望和目标，逐渐形成满意的心态。

## 3.3 消除焦虑的方法
减少内心的恐惧，拒绝对未知的恐惧，积极面对所有事情，持续学习，提升自我保护能力，并让生活不断美好！
