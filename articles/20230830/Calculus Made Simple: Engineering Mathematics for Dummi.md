# 3. 微积分入门
## 一、微积分概述
### 1. 微积分的定义
微积分，中文译名为微分运算，是利用导数的概念，探究函数表达式在坐标点上的变化规律，即从某点出发通过一阶导数、二阶导数等途径，寻找变化最快的方向，以此确定最优路径。它是利用微分方程来研究质量、温度、距离、动量、能量、电阻率、压强、密度、速度、物体表面弯曲程度等物理量随时间、空间等变量的变化规律的数学方法。

### 2. 微积分的组成
微积分由五种元素组成：
1. 函数：是一类特殊的连续统函数。它可以表示现实世界中的任何事物。一般来说，函数由自变量和因变量两部分组成，其中自变量是表示范围或者横轴，而因变量是表示纵轴。在微积分中，函数通常被表示成一个变量对另一个变量的映射关系。
2. 活跃函数：也称为定义域为实数的函数，它是在某个区间内取值的函数。活跃函数是由一些指定的点组成的区域所形成的集合，这些点满足函数的表达式，即存在对应的函数值。
3. 导数：是测量一个函数的斜率，是导数的概念。微积分的导数是函数在某一点的切线相对于该点的斜率，也称为该函数的一阶导数。导数可用来分析函数值随自变量变换的变化情况，从而推断其数学形式和趋势。导数所描述的是导数指向的方向，而不是其速率。导数的值决定了函数在这一点的变化率。
4. 积分：是测量函数的梯度，是积分的概念。积分是将曲线沿着某一方向上的分段曲线的长度之和作为总长的过程。积分值是一个函数在某个区域上的平均值。
5. 方程：是用来描述变化的函数。微积分所研究的问题都是关于变量随着自变量(或参数)变化的函数、方程、曲线、空间等的研究。

### 3. 微积分的应用
微积分的应用领域非常广泛，可以应用到很多领域，如：

1. 微分方程：微分方程是微积分的一个重要工具。微分方程描述了变量在平面或曲面上的曲率。它们有助于了解自变量的变化对因变量的影响。
2. 概率论和统计学：微积分的数学理论可以用于分析随机事件的发生，并把随机事件的结果转化为有用的信息。例如，利用微积分计算正态分布的概率密度函数。
3. 物理学：微积分的理论可以用于研究牛顿力学、运动学、热学、电磁学、传播学、光学等各种物理现象的数学模型。
4. 工程学：工程学中常用的微积分方法包括：正交分解法、向量场和拉普拉斯方程、偏微分方程。工程问题的研究通常借助微积分可以找到解析解的近似。
5. 经济学：经济学中微积分的应用主要有生产函数和动态预期模型的建立。动态预期模型可以用来估计变化的趋势，并对未来的预测做出合理的判断。
6. 数学：微积分对于一些复杂的数学问题提供了一种抽象的方法。如，利用多元微积分求解具有多个变量的方程，或计算曲线在一点处的切线或法向量。

## 二、微积分基本概念
### 1. 符号的含义
首先，我们需要搞清楚微积分中使用的一些符号的含义。

1. x：表示自变量。
2. f(x): 表示函数。
3. y=f(x): 表示函数图像。
4. g(x), h(x), k(x)，l(x)，... : 表示其他函数。
5. ∫：表示积分符号。
6. ∇：表示nabla符号。
7. d：表示微元符号。
8. ϕ(x): 表示指标函数。
9. [a,b]: 表示区间。

### 2. 函数
#### （一）函数的定义
函数是一种特殊的连续统函数，它可以在一段连续的自变量区间上取值，且函数值随自变量变化的曲线、线段或直线。函数的形式：
y = f(x)。其中，y是函数的表达式，x是自变量，即输入到函数中的参数。函数表达式用一个符号表示，这个符号就是函数的名称。

#### （二）函数的性质
1. 函数值：一个函数的每一个定义域上的点都对应着唯一的函数值。
2. 折叠性：若y = f(g(x))，则有y = f(h(z)), z属于函数g的定义域。
3. 可加性：两个函数的和等于其任一函数的和，即：
y = f(x)+g(x) = (f+g)(x)
4. 分配性：乘法与加法的分配律：
y = f(x) + g(x) = (f+g)(x)
y = f(x) * g(x) = fg(x)
