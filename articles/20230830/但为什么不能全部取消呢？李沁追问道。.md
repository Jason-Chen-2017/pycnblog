# 1.简介
  

近年来，随着经济形势不断发展、社会不平等加剧、人们生活水平提高、科技进步加快、全球化进程加速等诸多原因，国际金融危机给予了世界各地的金融企业以及个人巨大的冲击，引起了许多人的关注。
近日，南方多地爆发了经济严重衰退、房价上涨、失业率上升、贫富分化加剧等广泛而复杂的现象。在此背景下，越来越多的人对各种各样的收入分配制度产生了质疑，并对其根本原因进行寻找。比如，是否存在一种有效的收入分配方式？这种制度设计是否会带来更好的福利、公平性以及健康的社会环境？本文试图通过探讨中国目前的收入分配制度及其弊端，揭示其根源和未来的发展方向，并期待通过理论分析和实证研究，推动中国经济的转型发展。
# 2.背景介绍
20世纪70年代末，英国曾经实行过几次反垄断调查，结果显示，主要垄断行业中存在着诸如假冒伪劣产品、虚增产量等问题。1980年，美国国务院颁布了一部关于打破垄断、规范经济的《联邦贸易法》，之后，美元开始大幅贬值，日本也陆续实施贸易管制，其后果很不乐观。2008年12月至2009年5月，阿尔都塞·奈特迪斯·戴维森(<NAME>)出任美国总统，紧接着，他宣布采取行政手段进行反垄断调查。
这些反垄断法律及其运作方式，为各国政府提供了一个解决垄断问题的思路和途径。但是，中国共产党在这一领域始终没有取得预想的成功，因为它把反垄断当成政治斗争的工具而不是根本解决之道。自从2001年中共十五大上通过“积极应对新形势、扩大开放和改革发展”的纲领以来，反垄断事宜的处理从“压倒一切”的策略变成了“排除一切”，甚至出现了一些明显违背政治本身的做法。
1994年，中国共产党第十三次全国代表大会选举产生的党的十二大报告指出，“在国际市场、区域经济、金融体系和外交关系等新形势下，需要加强反垄断法律工作，制定相关法律依据和规定，确保整个社会的经济活动具有公正、透明、包容的特征”。1997年至今，两会召开四届，反垄断的主题已经成为政策界的热点话题，2007年修改后的《反垄断法》正式生效。
# 3.基本概念术语说明
收入分配制度包括三个层面：计划层面、市场层面、分配层面。
计划层面指的是由计划者（即国家）所设定的收入分配方案。在计划层面，按固定比例或者目标收入水平对总收入进行分配；计划可以是长期的，也可以是短期的。例如，国家或集体可以根据每个人的年龄、职业、教育程度等条件确定每个人的基本收入和福利状况。
市场层面指的是由市场主体自发形成的市场秩序所形成的收入分配模式。在市场层面，市场主体可以自由选择自己的居住地、职业、消费习惯、投资意愿等条件，以及提供商品和服务的价格、供应商以及消费者数量等信息，然后由这些条件对不同种类的收入进行分配。在这个过程中，市场主体往往充分尊重消费者的权益，试图创造一个公平竞争、共享的市场秩序。
分配层面指的是由具体的个人、组织和群体在收入和福利上的选择行为所形成的收入结构。在分配层面，收入分配以个体的实际情况为基础，体现为个人账户和家庭支出、医疗保险、养老保障等方面的开支。分配层面还涉及到房地产市场、土地使用权、税收筹划、婚姻匹配等其他方面的问题。
# 4.核心算法原理和具体操作步骤以及数学公式讲解
# 4.1核心算法
宏观经济理论对收入分配制度的描述一般采用“公平竞争、有限 government”模型，即认为有限政府为了公平竞争而设置收入分配制度。该模型认为，政府可以通过增加财政补贴、税收减免、分派优惠金、鼓励企业布局等方式来增加公民的收入，从而形成公平竞争。而在具体的操作中，一些发达国家实行的较为激进的收入分配制度，比如，西欧、北美地区的收入分配制度往往都属于政府收入占比过高、低薪职位高昂的“皇帝新衣”。另一些发展中国家的收入分配制度则比较合理，比如，中国实行的完全垄断的制度即使在计划经济时代也是一个巨大的倒退。
中国长期以来一直坚持计划经济体制，在计划层面上，政府通过各种手段鼓励和限制非公经济的发展，如支持低收入阶层创业、福利留成、农村土地承包经营权等。虽然计划体制存在很多弊端，但它也是实行公平竞争、鼓励创新、促进就业的重要制度安排。在市场层面上，市场主体能够自由选择自己的收入分配方式，比如，房地产商可以在售楼处设立自己的房源评估标准，并通过高招标出售住宅；个人则可以选择不同的职业，以便得到不同的待遇。在分配层面上，个人根据自己的实际情况选择不同的收入项目，包括个人所得税、养老保险、医疗保险、教育费用等。这些都是中国实行公平竞争、有限政府的根本要素。
# 4.2具体操作步骤
1、设置收入分配基准。中国长期实行计划经济，其收入分配基准常常是居民一次性缴纳的税负，即公积金余额与年终奖金之和，国家通过拨付个人所得税的方式来支付基本开销。因此，从长远来看，该计划经济体制将继续保持高度的税负率，这是当前中国经济发展的基本矛盾。

2、落实收入差距补偿机制。在计划经济时期，国家对财政赤字进行累计上调，用于资本项目的扩张，其结果导致“超税费”现象蔓延。2005年，中国的财政赤字达到24万亿元，到了2009年的峰值时达到45万亿元。为了平衡这一赤字，中国政府推出了“地方财政费征收补贴”的政策，旨在降低财政支出并提高企业收入。截止到2015年底，地方财政已支付给企业的超过1200亿元的财政补贴，总计财政支出的98%以上来自地方财政。同时，在财政刺激的同时，还实行了国企改革、央企分拆等一系列经济政策，助力缓解财政困境。

3、优化国有资本市场规则。中央与地方之间存在巨大的资本流通和调配过程，而国企存在着严重的资本依赖，因此，国企需要遵守市场规则。在这一方面，中国政府推出了许多措施，包括建立以股份制为核心的国企改革、建立国有资本市场信息公开制度、开展国有资本监控与管理、健全股权激励机制等。这些政策都旨在尽可能减少国企的对外投资，促进国企在资本市场中享有主导地位。

4、健全转移支付机制。由于中国长期以来财政依赖国企，国企大量向非国有经济转移资本，使得这些部门得不到足够的资金支撑。2007年后，中国政府推出了大量转移支付机制，允许企业向企业或个人发放跨省、跨市、跨国等各种形式的资金，用于改善职工和学生生活、重振国际声誉、防止金融风险等方面的需求。这些机制已经成为促进中国经济发展的重要组成部分。

5、发展适度的国际合作。中国在全面开放、国际化进程中积极参与，建立与亚洲国家的合作关系，同邻国分享中国的先进技术，促进区域经济发展。中国与欧盟、日本、韩国、印度等国家签署了多项协议，合作共赢，共同应对全球性挑战。中国共产党在建设国家主权力量方面也积极倡议，推行国家—主权分离制度。

6、鼓励创新。长期以来，中国在提高经济效率、促进创新、防止腐败等方面都做出了积极的贡猎，取得了令人瞩目的成果。新的产业革命正在兴起，无论是信息技术、汽车、电子技术、装备制造还是传统制造业的升级，都面临着前所未有的机遇。中国政府积极鼓励和支持创新，尤其是在基础科学、工程技术、医疗器械领域。

7、积极构建新型工业体系。中国长期以来实行“计划经济”和“国有垄断”，制约了创新和发展。在这种情况下，中国必须主动让位于创新驱动的市场经济，积极参与到制造业的改革中来。在这个方面，中国政府推出了“产业联合体”等一系列政策，鼓励企业联合起来，共同致力于新工业的开发和创新。

8、完善政府收入来源，提高财政绩效。中央政府持续注资、投放基金、搞补贴，改善就业形象，这是为了提高政府绩效，促进经济增长。然而，目前中央政府的财政支出仍然偏低，且占GDP的比重过低。同时，地方财政有限的政策导向也影响了地方政府的收入分配决策。因此，中国政府提出了减轻财政负担、增加财政收入、提高财政绩效的目标。其中，减轻财政负担最重要。
# 4.3数学公式讲解
公平竞争模型中，公共物品的最大获利者通常被称为“超级用户”。公共物品可以是时间、空间、金钱等任何可被利用的资源。“超级用户”可以通过两种方式获得收益：一种是直接获利，即通过使用公共物品来获取直接收益，另一种是间接获利，即通过出售公共物品的部分或者全部收益作为个人获利。公共物品的收益越大，则“超级用户”越受益。公平竞争模型认为，如果一个国家的公共物品（如土地、房屋、环保设施、交通工具等）分布均匀，则所有“超级用户”都将获得同等的收益；如果一个国家的公共物品分布不均匀，则所有“超级用户”都会获得比平均水平略高的收益。公平竞争模型给出公共物品分配问题的递归方程，即：
P = S(1 - a) + Sa*ra
其中，Pi为第i个“超级用户”的初始收益，Si为第i个“超级用户”购买公共物品的初始成本，ai为第i个“超级用户”对公共物品的最佳承受能力，ri为公共物品的实际产值。公共物品的平均产值Ra=∑pi/Ni，Ni为分配的总数量。
公平竞争模型给出了递归方程，但它忽视了生产公共物品的“超级用户”，即“公共品厂商”。因此，为了使公共物品在所有“超级用户”之间公平地分配，公共品厂商必须考虑所有“超级用户”的消费能力、购买意愿、和消费水平。在公共品厂商的指导下，公共物品的分配可以产生良好的效益，同时满足不同“超级用户”的收益要求。
# 5.未来发展趋势与挑战
收入分配制度是影响中国经济发展和社会进步的关键制度。它的实践方式一直是中国经济发展的一个难解难分的话题。计划经济的弊病在很长的一段时间内都未能得到根本解决，社会主义市场经济的发展又使得计划经济重新成为现实。如何破解计划经济中的巨大矛盾，推动中国经济的现代化转型，需要由中国共产党和政府团结起来共同努力。
基于公平竞争模型，我们可以对中国的收入分配制度进行分析和理论阐述。基于公平竞争模型，我们可以发现以下六个弊端。
第一，“市场中心效应”与“行政干预过度”。中国在1997年后陆续实行了“税前投资扩张”的政策，即鼓励企业进行准入审批，以获取税前投资。然而，“税前投资”仅占到公共财政收入的5%左右，相当于企业的净资产比例较小。另外，在这样的政策下，中央政府却以各种名目牵扯企业高价进入市场，有损于公共利益。因此，如果政府不采取切实措施防止“税前投资扩张”的滥用，将会带来严重的问题。
第二，“政府部门利益过度集聚”与“市场有效性弱化”。在计划经济体制下，政府通过各种手段鼓励和限制非公经济的发展，如支持低收入阶层创业、福利留成、农村土地承包经营权等。然而，这些举措只能局限于政府主导的经济领域，并不能真正推动经济走向平等。同时，国家财政面临税负累积、投资困难、就业岗位稀缺等问题。因此，如果政府部门不采取切实措施实现公平竞争、创新、可持续发展，将会带来政绩的倒退。
第三，“收入不平等陷阱”与“全面落实平等原则”。中国实行了计划经济体制长达两千多年，但国民收入仍然存在巨大不平等。长期以来，中国政府以各种名目压榨工人、官员，造成城乡差距的拉大，导致就业和收入不平等的严重化。同时，计划经济的收入分配制度使得地方政府通过掠夺国有土地和土地增值税等手段获取巨额利润，又把国有经济转嫁到地方，形成“垄断国企”的弊端。因此，如果政府不采取措施解决收入不平等、消除全面落实平等的历史遗留问题，将会导致改革的方向失误。
第四，“创新杠杆效应”与“行政导向过度”。中国政府在20世纪90年代初试验过新农村经济，并启动了城镇化进程。然而，这一尝试后果不堪设想。随着城镇化进程的加速，大量外来劳动力涌入城市，并且原有的户籍制度无法有效管理这些人口，这直接威胁着社会的发展。同时，由于严格的户籍制度，许多人原有的劳动力只能流动到接受系统培训的农村，并没有实现真正的职业化。因此，如果政府不采取切实措施推动创新型经济发展，将会产生严重的外部性效应。
第五，“资本市场规则失灵”与“政府行为低效率”。2008年金融危机后，中国政府通过扩大再贷款、发行更多货币和创造更多金融创新，来刺激经济增长。然而，近些年，中国政府却陷入金融泡沫化的怪圈，且投资国企无法有效保障公司收益，致使国企出现亏损，同时不利于引进优质技术、产业链和人才。因此，如果政府不采取措施确保资本市场的正常运行，则会影响经济的快速发展。
第六，“利率攀升、居民收入水平下降”与“制度性不公平”。2008年金融危机之后，由于国内货币贬值的过速，尤其是对股市、债市的过度宽松，导致各类投资和贷款都遇到问题。同时，居民的收入水平下滑，导致消费和储蓄的不平衡。这一事件发生后，中国政府以“紧缩”为名，紧缩财政，随后货币一直保持在较低水平。为了解决这一问题，中国政府推出了应对货币贬值的多种措施，如降息、降准等，但是没有真正落实公平准入制度。因此，如果政府不采取措施推动中长期利率下调、解决收入不平等，将会导致制度性不公平和社会不满。
以上六个弊端表明，中国的收入分配制度存在着系统性的弊端。如何进一步加强公共品的分散性，推动创新型经济的发展，充分利用和保护市场主体的资源，是实现中国经济现代化的关键任务。
# 6.附录常见问题与解答
1、如何正确认识和理解“计划经济”和“市场经济”？
1.1 中国政府的目标
中国政府要抓住经济复苏的机遇，推动经济转型，实现可持续发展。

1.2 计划经济的弊端
计划经济体制存在以下五大弊端：
- 市场中心效应：计划经济有利于“行政干预”，有时候会导致“市场失灵”，导致收入分配不合理、政策不畅通、生产力过剩、效率低下等问题。
- 行政干预过度：计划经济鼓励生产领域的垄断，导致生产力过剩，结果必然是“经济停滞”，市场化改革不彻底、投资过度等问题。
- 政府部门利益过度集聚：计划经济鼓励企业开拓“新市场”，通过投资项目“创业”，但这些举措只能局限于政府主导的经济领域，并不能真正推动经济走向平等。
- 投资过度：计划经济鼓励企业投资创新项目，造成大量资金投入，资本不平等，并且投资项目往往不具备可持续发展的属性，过度扩张会导致全社会的不公平、贫富分化、劳动力成本过高等问题。
- 收入不平等陷阱：计划经济存在社会不公平，长期以来，中国政府以各种名目压榨工人、官员，造成城乡差距的拉大，导致就业和收入不平等的严重化。

为了克服计划经济的弊端，中国政府推出了一系列的政策，包括税前投资扩张、紧缩财政、降息降准、“地区双轨制”等。

1.3 市场经济的优势
市场经济体制有以下四大优势：
- 普遍的竞争性：市场经济有利于解决经济发展中的不平等问题。在计划经济时期，人们习惯于以计划为导向，人为地束缚了经济的活力；在市场经济时期，人们逐渐形成了主动性，享受到真正的竞争。
- 适应性：市场经济的公平性使得人们可以选择自己的消费行为，充分利用市场资源，创造就业机会。在计划经济时期，人的收入只能受计划制度的控制。
- 可持续发展：市场经济能够促进经济持续增长，并且不会因为某些因素的变化而突然崩溃掉。在计划经济时期，由于过度生产、需求缺乏弹性，生产力过剩、生产要素短缺、外部性等问题频繁发生，经济发展的方向不确定。在市场经济时期，新的消费需求不断涌现，经济的整体效率也在不断提高。
- 透明性：市场经济拥有良好的信息透明度，所有人的交易行为都能在公开的市场上进行，使得公众能够了解真实的情况。在计划经济时期，很多信息往往被政府操纵，无法真正了解经济发展的真相。

2、什么是“公共品”？
2.1 公共品定义
公共品是指可供所有人平等享用的资产、服务和资讯等。公共品的分类一般包括以下几类：
- 自然资源：包括土地、水利工程、交通设施、电力、天然气等。
- 文化传统：包括艺术品、科学技术、文学艺术、古建筑、古陶瓷等。
- 劳动力资源：包括公厕、公共医院、职业训练场、购物中心等。
- 知识产权：包括专利、商标、版权、美术作品等。
- 金融资产：包括银行、信托、保险等。

2.2 公共品分类
中国有两种类型的公共品：内需品和外需品。
- 内需品：指政府及政府部门提供的物质上的好处，由政府及政府部门来支付。包括土地、水利工程、交通设施、电力、天然气、教育、文化传统等。
- 外需品：指公民及公民个人提供的物质上的服务，由公民个人或公民单位来支付。包括人力、食物、住房、医疗保健、文娱用品等。

3、什么是“公平竞争”？
3.1 公平竞争的含义
公平竞争是指企业、个人或其他组织之间的竞争条件完全相同。公平竞争意味着市场上的所有参与者，都要公平地受到别人的资源、知识、利益的不平等待遇，即所有人都能分享同等的经济利益，并且在市场上只能获得真正公平的竞争。

3.2 公平竞争的必要性
公平竞争是解决收入分配问题的必要条件。公平竞争的实现，意味着人们的生活水平提高、国家的经济增长，以及各国之间的经济关系的正常化。

3.3 公平竞争的原理
“公平竞争”模型认为，在一个竞争性市场中，如果所有企业、个人或其他组织都遵循“买者多，卖者少”的原则，所有的资源、知识、利益都可以获得公平分享，就形成了公平竞争。如果某个企业或其他组织违背了这个原则，那么该企业或其他组织就必须按照市场的要求以自己的方式提供相应的资源、知识、利益，才能获得优先的市场份额。
“公平竞争”模型的原理是基于两个基本假设：
- 所有参与者，都以自己作为唯一的生产者和消费者，不存在任何其他的生产者或消费者。
- 所有企业、个人或其他组织都遵循同等的竞争条件，他们生产的产品、服务、资讯等，以及享有的待遇与其他竞争者相等。
