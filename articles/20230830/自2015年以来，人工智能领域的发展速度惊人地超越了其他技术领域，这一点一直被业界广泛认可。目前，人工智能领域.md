# 2.NLP—自然语言处理(Natural Language Processing)。
NLP是人工智能的一个子领域，涵盖了如语言模型、词嵌入、机器翻译、文本摘要、自动问答、情感分析、文本分类等方面。其核心任务是在海量的数据中提取出有意义的信息，并进行有效的组织、检索、理解、表达等。
NLP背后的技术包括：特征抽取、序列建模、语言模型、主题模型、信息检索等。
2.1特征抽取——使用NLP对文本进行分析、提取特征。
所谓“特征”就是指对输入文本的分析结果，例如提取出关键词、词性标签、句法结构等。通过对文本进行分析，可以更好地理解文本，完成后续的分析任务。
常用特征抽取方法包括：
- TF-IDF（Term Frequency - Inverse Document Frequency）
- Word Embedding
- Doc2Vec、BERT等

2.2序列建模——训练模型预测未知文本。
文本表示学习（Text Representation Learning）是利用文本的潜在特性将文本转化成稠密向量的过程。深度学习模型的训练目标是通过训练数据（即文本）学习到数据的分布，从而将未知文本映射到高维空间中，实现文本的编码和分类。
常用的文本表示模型有：
- CNN-LSTM
- GPT-2、Transformer-XL等

2.3语言模型——基于历史文本生成新文本。
语言模型本质上是一个计算概率P(w)，即给定前面的一些字符，下一个可能出现的字符是什么？语言模型能够帮助模型理解上下文，并且生成新的语句或文本。它可以用于生成文本、翻译、对话系统等。
- 马尔可夫链蒙特卡洛模型（Markov Chain Monte Carlo Model）
- N-Gram模型
- 概率图模型（Probabilistic Graphical Model）
- 条件随机场（Conditional Random Field）
- VAE、GAN等

2.4主题模型——探索文本内在结构。
主题模型的主要目的是从文本集合中发现隐藏的主题，并对每一个主题赋予一个简短的描述性词汇。它可以帮助用户快速理解文本内容、整合不同文档的内容、找到相关文本、分析热点话题等。
常用主题模型方法包括：
- Latent Dirichlet Allocation (LDA)
- Non-negative Matrix Factorization (NMF)
- Hierarchical Clustering
- LSA、SVD等

2.5信息检索——帮助用户找寻信息。
信息检索的核心任务是按照用户需要找到相关的信息。基于文本匹配的方法，包括布尔检索、向量空间模型（Vector Space Model）、排序算法等。
信息检索往往配合搜索引擎一起使用，并提供丰富的查询方式，比如关键字、短语、过滤器、排序标准等。

2.6机器翻译——自动将一种语言转换为另一种语言。
机器翻译是指使用计算机软件实现人与机器之间互相传递文字的过程。传统的机器翻译方法包括手工翻译、规则引擎、统计语言模型等。近年来，深度学习模型受到了越来越多关注，因为它们可以提升准确率并改善翻译效果。
常用深度学习机器翻译模型有：
- Seq2Seq（Sequence to Sequence）模型
- Transformer模型
- Attention Is All You Need模型
# 3.CV—计算机视觉(Computer Vision)。
CV是指利用数字图像处理技术从图像、视频中识别、理解并做出决策的一门学科。该领域的核心任务包括物体检测、跟踪、分割、姿态估计、三维重建、图像风格迁移、目标识别等。
CV背后的技术包括：特征提取、优化算法、卷积神经网络、循环神经网络等。
CV领域的应用范围很广，包括但不限于物体检测、行人重识别、交通场景理解等。

3.1特征提取——识别图片中的对象、场景、属性等。
特征提取通常通过机器学习算法从图像或者视频中提取图像特征，比如人脸检测、图像分类、目标检测等。目前流行的特征提取方法有HOG（Histogram of Oriented Gradients），SIFT（Scale-Invariant Feature Transform），CNN（Convolutional Neural Network）。

3.2优化算法——利用深度学习提升图像处理性能。
由于传统图像处理算法存在效率瓶颈，如特征匹配算法复杂度高、内存消耗大等，因此深度学习的优化算法应运而生。目前，最流行的优化算法是Adam（Adaptive Moment Estimation）、Adagrad（Adaptive Gradient）、SGD（Stochastic Gradient Descent）。

3.3卷积神经网络——构建图像理解模型。
卷积神经网络（CNN）是一种深度学习模型，它在图像处理中起着至关重要的作用。CNN通过多个卷积层和池化层，提取图像的局部特征，形成一个特征图，再通过全连接层输出最终结果。

3.4循环神经网络——解决时序数据问题。
循环神经网络（RNN）是一种深度学习模型，它可以捕获时间上的依赖关系，适用于序列数据，比如文本、音频、视频。它通过反向传播算法来更新参数，通过隐藏状态记忆之前的输入信息。

3.5目标检测——识别物体位置及类别。
目标检测是CV领域的基础技能。当物体出现在图像中，如何确定它的位置、大小、方向等信息，这是计算机视觉中最基础的问题之一。目前，比较流行的目标检测方法有YOLO、SSD、RetinaNet、FPN等。

3.6图像分割——把图像分割成若干个区块。
图像分割是指把图像划分成不同颜色区域，并给每个区域赋予不同的标签，这些标签用于进一步的处理或分析。图像分割是计算机视觉中经典的任务，涉及图像分析、图像检索、图像修复等众多领域。

3.7姿态估计——估计物体的旋转角度和平移程度。
姿态估计是指根据图像或视频中物体的二维或三维坐标，计算物体在空间中的姿态、位置等信息。姿态估计在多视角监控、增强现实、人机交互等领域有着广泛的应用。

3.8三维重建——还原原始图像中物体的三维形状。
三维重建是指还原真实世界中物体的三维形状，包括物体尺寸、姿态、纹理等。三维重建技术在虚拟现实、AR/VR、医学影像诊断等领域有着广泛的应用。

3.9目标识别——识别图像中的特定目标。
目标识别是指识别特定物体是否出现在图像中，并将其定位。目标识别在人脸识别、安防领域有着广泛的应用。
# 4.RL—强化学习(Reinforcement learning)。
强化学习是机器学习领域的一种机器动作学派，强调训练智能体（agent）以最大化累积奖励（cumulative reward）的过程。通过对环境和行为之间的交互，来选择动作使得长期收益最大化。

深度强化学习是强化学习的一种变体，通过深度学习的方式训练智能体，并结合深度学习的神经网络结构，来提升学习效率、降低样本效率和泛化能力。

RL领域的核心算法有Q-Learning、SARSA、Actor-Critic、PG、DQN、A3C、DDPG等。

4.1Q-Learning——实现动态规划。
Q-Learning（Q-table）是一种值函数形式的强化学习算法，它使用Q表格来存储每一个状态动作对的价值，然后基于当前状态采取某一动作，然后得到的奖励，更新Q表格中对应的值。

4.2SARSA——利用TD-error更新Q表格。
SARSA（State-Action Reward State-Action）是一种Q-Learning的变体，它直接对Q表格进行更新，而不是像Q-learning那样通过递推的方式。

4.3Actor-Critic——结合策略梯度和值函数梯度。
Actor-Critic（Policy Gradient）是一种结合策略梯度和值函数梯度的强化学习方法，它同时训练智能体的策略网络和值网络，并通过策略网络来选取动作，通过值网络来评估每一个动作的价值。

4.4PG——采用随机梯度下降法进行策略更新。
PG（Policy Gradient）是一种策略梯度算法，它采用随机梯度下降法（Stochastic Gradient Descent，SGD）更新策略网络的参数，以减少策略网络的损失。

4.5DQN——利用神经网络进行深度强化学习。
DQN（Deep Q Network）是深度强化学习的一种算法，它利用神经网络拟合价值函数，并通过Experience Replay来减少样本效率，通过目标网络来避免样本偏差。

4.6A3C——采用异步更新策略网络参数。
A3C（Asynchronous Advantage Actor Critic）是异步的深度强化学习算法，它采用多线程更新策略网络参数，并通过异构体系结构训练智能体，来达到更好的学习效果。

4.7DDPG——结合DDPG算法和Actor-Critic方法。
DDPG（Deep Deterministic Policy Gradient）是一种深度强化学习算法，它结合DDPG算法和Actor-Critic方法，利用目标网络来缓解Exploration-Exploitation tradeoff，并同时训练策略网络和值网络。
