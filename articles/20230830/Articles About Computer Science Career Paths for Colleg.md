# 2.为什么不用计算机专业？
首先，你需要明白的一点是，即使你没有获得任何计算机科学相关的职称，计算机科学也是一个很好的兴趣爱好。如果你只是喜欢编程，那就可以不去念计算机科学专业。这不代表你可以不懂计算机科学。如果你真的不希望自己成为计算机科学家，也不要指责别人嘲笑你。毕竟还有其他的领域比计算机科学更有意义。

其次，计算机科学学科是如此多样化，以至于几乎每一个分支都有自己的独特的风格和适合的人群。这就意味着，如果你打算在某个领域钻研，比如网络安全、区块链，那你还是得花时间把这门课学透彻。否则，如果你仅仅是一个计算机菜鸟，即使成功进入某个行业，也只能作为一个上班族而已。

再次，计算机科学还有很长的路要走。即使你已经有了一定水平，未来仍然充满着各种挑战和机遇。比如，移动互联网和物联网的革命，全新的计算模型和工具的出现等等。虽然计算机技术日新月异，但它的核心原理都是不变的。

最后，如果你是一个非常老练的程序员，对计算机科学了解甚少，那么如果你不早点转向其他领域，你的职业生涯还会更加丰富多彩。

综上所述，不管你是否真的想要做计算机科学家，只要你想做一些让人开心的事情，并且对计算机科学有很强烈的兴趣，就可以继续选择它。正如《失控》电影里的小丑一样，如果你能认清现实世界的真相，学会乐观面对，也许你会发现美好。

# 3.在刚刚走上研究生第一步时，怎么决定要不要做研究？
研究生的第一步是选择研究方向。这是一个重要的决定，因为它关系到你之后的整个职业生涯。下面给出四条建议：

1) 找到你热爱的工作领域。

2) 和老板一起磨合，找到研究兴趣所在。

3) 搜集研究领域的最新研究成果，关注热点新闻。

4) 在不同的机构之间做交叉学科研究。

第1条建议比较直观，就是说找到你热爱的工作领域。举例来说，如果热爱音乐，你可以去研究音乐作曲、编曲、制作器材、声音效果，或者是电子音乐创作。如果你热爱艺术，可以从多媒体设计开始，或是绘画。如果热爱哲学，可以研究伦理学、经济学、社会学等方面的研究。计算机科学同样如此。

第2条建议是和老板一起磨合的。如果没有老板支持，或是老板不赏识你的研究，你可能会感到束手无策。尽量找个老板在你周围，向他请教，看看他是否支持你的研究。如果他支持，你就可以开始行动了。不过，真正的成功往往来自于你的努力和运气。

第3条建议主要是为了获取信息。对每个人来说，知道最新研究成果是件有益的事。在不同学术机构之间做交叉学科研究同样很重要。举例来说，如果你的研究兴趣是机器学习，可以去参加会议或是期刊，阅读那些关于机器学习的论文。也可以看看其他的学科，如心理学、语言学等。通过阅读这些信息，你就可以了解该研究领域的最新进展。当然，在信息过载时，最好注意注意自己的兴趣。

第4条建议则是在多个学科之间进行研究。不同学科之间有着千丝万缕的联系，如果不是合作的关系，你可能很难找到自己热爱的领域。另外，如果一个人对某个领域很熟悉，他很可能知道另一些相关领域的研究情况。因此，有的时候一个人可以先在自己熟悉的领域做一些尝试，然后再扩展到其他领域。

最后，选择研究方向的过程并非一蹴而就，有时你还需要做一番艰苦卓绝的准备。在开始之前，你需要找准自己的定位和目标。只有找到正确的方向，才能够充分发挥自己的才能。

# 4.把握专业前景和资源优势，如何利用好这些优势？
研究生期间，你需要把握住很多事情的权衡。一方面，你必须谨慎地行动，免得出现失败的情况；另一方面，你还需要和老板搭建良好的沟通渠道，共同推进项目。下面，我推荐三种方法：

1）利用系统资源优势。首先，你需要确认自己的系统资源是否足够。不同的大学提供的各类资源也不同，比如计算机网络中心可以提供专属服务器、加速网络服务，图书馆提供了丰富的参考书籍等等。因此，在行动前，一定要检查自己有没有充分的资源。

2）积极主动地寻找导师。导师是研究生的精神支柱，也是你的榜样。他/她会帮助你理解研究主题，提出各种疑问，并且鼓励你承担更多的工作。你可以在导师的指导下做一些基础性的工作，或者加入一些志愿者团队，帮助他人解决困难的问题。

3）充分利用时间。研究生有时需要处理繁重的任务，这就需要及时的关注消息。你可以订阅一些论坛、邮件列表，了解不同学科的动态。并定期主动回应老板的询问，和导师保持联系。在完成某个任务后，回顾一下结果，总结经验教训，分享知识与经验。这样，你就可以帮助他人避免踌躇，更好地掌握自己的研究领域。

# 5.如何才能找到自己的方向和理想？
一般来说，计算机科学的学生们都认为自己的未来取决于自己对某一特定领域的兴趣。有很多方法可以帮助你找到自己的方向。

1）做好计划。计划可以帮助你规划自己的职业生涯。你应该列出一份详细的计划，包括每天的计划、周末的计划、有限的时间做什么、如何度过寒暑假、读哪些书籍、如何充实生活等。

2）与身边的人建立联系。社交圈子中存在很多有影响力的计算机科学家，他们可以告诉你一些最新的技术进展，并提供一些建议。邀请他们来分享你的想法，还可以听听他们的故事。

3）投身开源社区。开源社区有很多活跃的开发者，你也许可以结识一些志同道合的朋友，并协助开发新的产品。

4）参加本地的学习俱乐部。如果所在城市有计算机科学相关的组织，你也可以考虑加入其中。这些组织通常都会组织一些有价值的学习交流活动。

总之，不管你选择哪一条路，都需要找到自己的理想和激情。如果你真的想要在某个领域钻研，就需要紧跟前沿，投身其中，勇敢追求成功。
