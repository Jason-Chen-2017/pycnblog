# 2.计算机视觉（CV）
## 2.1 什么是计算机视觉
计算机视觉(Computer Vision，CV) 是指让电脑“看到”、理解并处理图像或视频信息的一门学科。它涉及图像采集、处理、分析、显示、存储以及智能系统的研发。在CV领域，主要研究如何提取有效的图像特征，同时对图像进行识别、理解、跟踪、检测、分割、编辑等处理。
### CV的主要研究方向包括：
* 图像表示与理解：对图像进行特征提取、描述、分析、检索，建立图像数据库、检索系统、图像检索方法，以及基于深度学习的方法，能够理解图像内容。
* 图像拼接与修复：将多幅图像合成成一个完整的图像，或对图像进行修复、去噪、增强等操作。
* 图像分类与检测：对图像进行自动分类，按照一定的规则和标准划分出图像类别；检测对象及其位置，例如人脸、车辆、道路标志等。
* 视频分析与处理：对视频中的图像进行分析、处理，提取有效的信息和数据。
* 智能系统：构建具有视觉感知能力的智能体，如自动驾驶汽车、人脸识别系统、遥感图像理解系统等。

## 2.2 CV的应用场景
1. 目标检测：图像里通常存在很多物体需要定位和检测，比如目标人脸、交通标志、物体等。目标检测就是识别出这些目标的坐标位置，可以用于识别交通工具的位置、识别抓拍到的人脸、识别机器人产出的垃圾、监测某些行业里的安全事件发生。 

2. 图像分类：图像分类就是根据图像的内容自动划分为不同的类别，比如一张猫的照片可以划分为一类，一张狗的照片可以划分为另一类。图像分类可以帮助生产制造等部门将大量图像分类整理归类，便于管理、维护、加工。

3. 图像分割：图像分割就是将一个复杂的图像分割成若干个子图像，通常用颜色、纹理、形状等区分子区域。图像分割应用最广泛的是医疗图像中的肿瘤切除、自动驾驶汽车的路线规划、高分辨率影像的精细化处理。 

4. 目标跟踪：目标跟踪就是在连续帧图像中识别和跟踪目标的运动轨迹。目标跟踪可以用于复杂环境下的视频监控、视频剪辑等应用场景，实现实时监控、视频创作。

5. 文字识别：传统的文字识别技术需要大量的人力、时间和硬件投入，而随着深度学习技术的发展，可以利用大数据进行训练，实现准确率更高的文字识别。例如，OCR技术可以用来扫描文档、票据等信息，为银行提供服务。

6. 图像超分辨率：图像超分辨率(super resolution, SR)是指将低分辨率图像恢复为高分辨率图像的技术。SR在工业、娱乐、图像教育等领域都有很大的应用前景。SR可以改善图像质量、增加图像的真实感。

## 2.3 CV常用的模型
1. 卷积神经网络(CNN): CNN是深度学习领域里最常用的模型之一，是一种适用于图像识别、计算机视觉、模式识别任务的神经网络。CNN采用卷积层、池化层、全连接层组成，能从原始图像中学习到有效的特征表示，因此在图像分类、目标检测、语义分割等任务上有很好的效果。

2. 循环神经网络(RNN): RNN是一类序列模型，是深度学习领域里最常用的模型之一。它能够对一段序列数据建模，并且能够记忆之前出现过的数据。在自然语言处理、视频分析、音频处理等任务上有非常好的效果。

3. 深度信念网络(DBN): DBN是一个无监督、生成模型。它的基本假设是通过不断地学习马尔可夫链来寻找数据的有意义的结构。DBN可以在许多领域应用，包括图片风格转移、音频生成、语音识别等。

4. GAN(Generative Adversarial Networks): GAN是一种通过训练两个神经网络相互竞争的方式生成新样本的无监督学习模型。可以生成图像、视频、声音等。GAN在图像合成、图像增强、视频生成、人脸生成、风格迁移、数据增强等方面有广泛的应用。

5. 对抗神经网络(GANs): 对抗神经网络(GANs)是通过训练两个神经网络相互博弈的方式来解决问题。它的特点是可以生成训练数据集中不存在的样本，这种模型能够在神经网络中找到具有共同特性的模式，并创造出新的、具有代表性的模式。对抗神经网络已被证明在图像、视频生成、文本生成、推荐系统等领域都有着卓越的效果。
