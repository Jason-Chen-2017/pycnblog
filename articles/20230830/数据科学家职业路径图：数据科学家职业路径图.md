# 1.简介
  

在日益复杂的产业链中，数据科学家正成为一个重要角色。作为一种重要工具、技术以及业务，数据科学的应用范围涵盖了计算机视觉、自然语言处理、生物信息学、金融风险管理等众多领域。数据科学家的需求也越来越强烈，但目前还没有明确的数据科学家职业规划和路径。为此，笔者创建了《数据科学家职业路径图》，基于业界认可的数据科学家技术路线图，对数据科学家职业规划进行结构化梳理，并用可视化的方式呈现出来。
该图共分为三个层次：入门、精通和专家。每一层次包含若干技能要求和职位要求。通过该职业路径图，可以帮助读者了解自己适合从事哪个层级的工作，并且可以将工作重点放在自己擅长的领域，提升自己的职场竞争力。
另外，《数据科学家职业路径图》还包括最新的高薪薪酬福利、学习建议、相关推荐阅读等。在向应届生推广《数据科学家职业路径图》时，数据科学家们会受到鼓励和关怀，提升职业能力和个人价值。
# 2. 1-入门（初级）
## 2.1 技能要求
1. 有基本的编程能力，如掌握Python、R、SQL等至少一种编程语言；
2. 对数据有基本的了解，能够清楚地理解数据的定义、特征、维度、分布及属性，并有充足的分析和建模能力；
3. 具备数据理解、预处理、探索、统计建模、模型评估、可视化和报告的能力；
4. 掌握数据采集方法，掌握数据获取方式和相关工具的使用，如爬虫、API、数据库连接等；
5. 具有良好的沟通能力和团队合作精神；
## 2.2 职位要求
1. 有需求分析、调研和设计数据的经验，能够独立完成数据产品开发过程，包括探索性数据分析、用户画像、风险评估、预测模型、机器学习等；
2. 在机器学习、深度学习、模式识别等方面有丰富的经验；
3. 熟练掌握PyTorch/TensorFlow/scikit-learn等主流深度学习框架和算法库；
4. 有过项目管理或敏捷开发经验者优先；
5. 配有一定的数据库开发能力、数据分析能力、运营能力、市场营销能力；
# 3. 2-精通（中级）
## 3.1 技能要求
1. 深入理解机器学习模型和算法，对特征工程、数据增强、模型选择、超参数优化、模型评估、错误诊断、模型解释等有全面的理解和实践；
2. 了解复杂的数据集，掌握大型数据集的性能优化技巧，包括数据采样、特征工程、特征转换、归一化等；
3. 能够利用统计学、机器学习、数据科学的方法进行有效的分析和预测；
4. 掌握常用的模型，如线性回归、逻辑回归、决策树、随机森林、聚类、KNN、SVM等；
5. 掌握时间序列分析、文本分析、图像处理、网络爬虫等领域的技术；
6. 具备数据仓库开发和ETL开发经验；
7. 善于利用数据实现商业价值，可以根据分析结果制定决策；
## 3.2 职位要求
1. 有自我驱动力，对新鲜事物有浓厚兴趣，能快速学习新知识和技能，能够解决复杂的问题；
2. 能够利用数据深刻洞察公司和业务的发展方向，提供产品改进建议、产品策略建议、项目启动指导；
3. 掌握数据挖掘、大数据处理、分布式计算等技术，有数据科学家的头衔；
4. 有优秀的英文口语和表达能力，掌握多种语言，包括英语、中文、俄语等；
5. 年龄不限；
6. 提供1年以上相关工作经历；
# 4. 3-专家（高级）
## 4.1 技能要求
1. 对整个机器学习流程有完整的认识，能够掌握不同机器学习算法的优缺点，并结合实际场景做出正确的选择；
2. 拥有更高级的模型设计、超参优化、模型评估等技能，能够进行深入的模型分析、数据分析、系统优化等工作；
3. 理解复杂的经济、金融、政策、法律、社会、生物等现象背后的因果关系，能够将数据科学方法应用到实际世界中，解决复杂的问题；
4. 熟悉NLP、CV、RL、GCN等领域，能够快速的转变和应用到实际工作中；
5. 有业内大牛级别的讲解、研究、评估、总结能力，能将自己所学、所见所闻、所感所得形成通俗易懂的文字，达到高水平的交流；
6. 具有独立工作、学习能力，能够高度专注于某项工作，并坚持不懈地加以输出；
7. 有意愿通过职业培训取得更多专业知识和技能提升；
8. 有雄心勃勃的追求，拥抱变化、不断学习，保持开拓进取的态度，并且拥有激情澎湃的劲头和热忱；
9. 能够以专业的口吻阐述自己的观点、方案、看法，能够让别人听得懂；
10. 年龄不限；
## 4.2 职位要求
1. 年龄35岁以下；
2. 掌握机器学习、深度学习、计算机视觉、自然语言处理、图神经网络等多个领域的相关知识，以及相关框架和工具的使用；
3. 对研究生阶段研究方向有独到的见解，并且有很强的动手能力、创造能力和团队精神；
4. 具有极强的研究能力、解决问题的能力，能够进行严谨的论证和研究，并且能够写出具有说服力的科普文章；
5. 对国际标准化组织、行业标准组织和公司内部的标准化及标准制定有着浓厚的兴趣；
6. 负责数据分析、模型构建、数据管理、标准化工作，积累较丰富的知识和经验；
