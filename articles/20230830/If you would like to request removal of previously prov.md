# 2.基本概念术语说明
## 定义
隐私权：一个人对自己之外的他人的隐私所持有的特定权利，包括对自己的个人信息的收集、使用、保存、披露、传输、公开展示等。

个人信息：指涉自然人的各种辨识资料，包括名字、生日、地址、电话号码、邮箱地址、居住地、工作单位、教育经历、职业、身份证号码、银行账号等，仅指可用于识别个人的确定范围内的信息。

个人信息保护：一切关于保障个人信息安全的法律规范和规则。包含以下五个部分：

1. 定义
2. 获得
3. 使用
4. 存储
5. 共享

个人信息保护机关：负责统筹协调国内多个省份、自治区、直辖市的公安、检察院、法院等部门，并依照相关法律法规的规定，制定、发布、管理和维护个人信息保护规定和机制的机构。

数据主体：指个人直接或间接控制或者以影响为目的取得个人信息的自然人。

个人信息共享：由数据主体向第三方提供、接收、转让其个人信息，或从第三方获取其个人信息的行为。

个人信息主体权利：个人对自己的信息享有的所有权利，包括获取、 rectification、 erasure、 restriction、 data portability、 objectionable processing等。

个人信息委托：指一方指定另一方代表其履行个人信息主体的各项权利，比如使用其名称、标识符和照片来识别数据主体，或者根据授权委托其共享数据。委托合同中通常会约束委托人处理个人信息的自由裁量权。

## 个人信息分类
### 普通个人信息
通常是指个人直接或间接能够提供的有关本人身份的信息，包括但不限于姓名、出生日期、性别、住址、电话号码、邮箱地址、通信联系信息、婚姻状况、国籍、民族、注册资金、收入、教育程度、职业、职称、行业、家庭住址、家庭成员姓名、家庭成员联系方式、工作单位及职务、个人网络档案、社交关系、宗教信仰、政治面貌、健康状况、疾病史、死亡事故史等。

### 关联个人信息
指涉两人以上之间存在密切关系、共同生活中的个人信息，包括但不限于夫妻间的通信联系信息、子女间的亲情关系、兄弟姐妹间的友谊关系、同事间的工作关系、合伙关系中各方的个人信息等。

### 精准个人信息
指能够唯一和直接与某一特定业务相关联的信息，包括手机号、身份证号、银行卡号、IP地址、Cookie ID、其他个人辨识信息等。

### 不可跨越地域个人信息
指涉地理位置范围较广、涉及跨区域的数据收集、使用、共享等个人信息，包括但不限于居住地、工作单位所在地等。

### 业务相关个人信息
指在业务中产生和使用的个人信息，包括用户注册资料、浏览记录、搜索记录、购买记录、运营分析结果等。

### 超链接/引用信息
指网页中包含的其他网站（或资源）的URL、超链接文本、图片描述信息，这些信息指向了数据主体的个人信息。

### 其他个人信息
指与登录、银行、保险、信贷、征信、知识产权等业务无关的信息，诸如血型、民族、生育年龄、个人爱好、个人习惯、传单上的联系方式、物流跟踪记录、个人照片等。
