# 1.背景介绍


## 1.1 什么是芯片制造？
芯片制造，即从零件到芯片制造，包括材料、电路设计、封装、测试、调试、生产和销售等各个环节。其主要目标是利用材料、工艺、方法和工具对新型高性能的电子设备进行研制和开发，并将其制成具有一定规格和性能要求的产品。为了提高生产效率、降低成本、保障质量，计算机、通信、自动化领域一直都在进行芯片制造的尝试。但对于现阶段，国内外同类制造的企业仍然较少，缺乏共同的技术和管理规范，制造流程也存在很大的不确定性和缺陷。因此，如何建立一套科学合理、规范化的芯片制造体系，成为一项重要的战略任务。
## 1.2 为什么要研究物理气相沉积？
物理气相（简称气相），是真空中的一种物态。由于空气中的氧气，溶于水分，而水分又可形成气团状结构，凝结成气团。人的身体则作为容器，填满汁液，形成一个有机的气团。这种气团就是我们平时所说的气候。如此庞大的气团，如何通过控制流动方向、阻力大小和温度变化来调控，以及如何将它固定在一个特定位置？这一切都离不开物理气相沉积与分子束的运动。
物理气相沉积与分子束的运动，已经成为我国国际学术界探索的一个热点课题。根据物理气相的物理特性，研究者们认为：“物理气相”是带有潜在反馈机制的复杂网络。这些网络可以调节流动方向、阻力大小、温度变化及物质运输，影响大气物理过程的关键参数，可被广泛应用于各种领域。“物理气相沉积”通过物理上精心设计的分子束，可以形成封闭的带状气团、旋转的喷流风场、涡旋的重力场以及排布的湍流波、洋流暴雨、太阳光谱等。有了物理气相的基本理解，就可以更好地认识与改善当今全球产业链中物理气相的运用。
## 1.3 分子束外延的定义与作用
分子束外延（Molecular Pore Penetration，简称MPP）是指由微观颗粒组成的表面液滴通过微生物分子相互作用而向周围组织逼近的现象。从结构上看，分子扩散层是微粒形成的空间，一般由吸附在微粒的分子进行水溶液相互作用而形成。分子扩散层由于不同分子的自主复制，构成一个很大的网络，使得扩散层的形态发生变化，甚至可出现上千倍数量级的分子并行扩散。因此，扩散层的形态决定了微粒的运动范围，分子扩散层之间的微粒能量交换和导热作用会促进微粒的运动。由于分子扩散层与表面之间存在微妙的相互作用，分子扩散层所包围的微粒逐渐向扩散层的一侧推进，途经扩散层就会发生“分子束外延”。MPP与表面液滴的密度及表面的摩擦力密切相关，其中某些情况下，甚至直接导致了表面破坏或损伤。例如，在临床应用中，MPP可引起癌细胞疾病的发展，甚至会导致肿瘤的发生。
# 2.核心概念与联系
## 2.1 气相沉积与分子束外延模型
根据分子扩散力学，分子扩散层由具有不同空间坐标轴的不同尺寸的微粒组成，每一个微粒都具有空间分布不均匀且动态演化不规则的特性。由于不同的微粒间存在相互作用，因而形成了一张“物理网络”，从微粒的角度看，相互作用的微粒在空间上形成“分子束”，随着时间的推移，分子束会膨胀变长，而最终演化到形成“气相”的整体，气相由数十亿到数百万微米大小，如图1所示。

对于分子扩散层中的每一张分子束，它的宽度将随着时间的推移逐渐增加，直到其长度达到特定值，称为分子束膨胀区（Pores of Diffusion）。在分子扩散层内部，存在一系列的“吸附区”，在这里，微粒只能通过相互作用来产生连锁反应，才可能吸附在一起。吸附区由微粒包裹着，并且微粒会根据自身的位置，与附着在该区域的其他微粒产生相互作用。吸附后的微粒将迅速向各自的吸附点移动，形成一个明显的迂回，称为“迂回中心”。类似的迂回行为也可以在气团中看到。通过分析分子束的构成方式和动力学特性，可以得到一些物理上的直观感受。例如，在气团中，每一条分子束有着统一的轨迹，随着时间的推移，分子束会形成悬浮气流，这也是为什么当地人不会掉进水里。
## 2.2 MPP模型中的主要参数
### （1）非晶面区（Bulk-like Area）
MPP模型假定了分子扩散层是一个典型的菲涅尔（Fisherian）结构，其核心区称为“非晶面区”。这种结构源于微粒的分子结构、自旋状态和运动状态的叠加，这三者在一定程度上可以作用在分子扩散层中，形成一张微小的菲涅尔势场。该模型的创立是基于实验的发现：在微透镜下，微粒的微观空间能够在很短的时间内自由跃迁、旋转和变幻，这就给予分子扩散层以超越晶体结构的动力学行为。但是在实践中，非晶面区远远没有达到可以模拟分子扩散的程度。因此，需要结合实际的实验数据，来建立非晶面区的参数模型。
### （2）松弛系数（Hooke’s Constant）
松弛系数，也叫温度系数，是一个衡量微粒表面张力与摩擦力的重要参数。微粒表面张力与摩擦力都是依靠微观尺度下的材料相互作用的结果。在分子扩散层中，微粒表面张力通常依赖于原子核的静电势，摩擦力则取决于分子的磁场。同时，在整个扩散层内部存在不同的吸附点，使得微粒之间存在多样的相互作用，所以微粒表面张力、摩擦力、电荷势能的不同组合，都会影响微粒运动的方向、速度、角速度、张力和加速度等。松弛系数便是用来描述微粒的这些相互作用关系的重要参数。
### （3）表面温度（Surface Temperature）
表面温度也叫表面制冷温度，指的是微粒液滴表面温度。在常温下，分子表面不会融化，但是在某些特定的条件下，分子表面将会冷却，形成微粒的液滴表面。表面温度决定了分子扩散层中微粒的运动速度、大小、摩擦力和碰撞力。通常，表面温度越高，微粒液滴表面就会越容易冷却，而分子的运动速度、大小也就相应减小。
## 2.3 MPP模型计算技术及实验验证
### （1）计算技术
MPP模型是一个复杂的理论模型，为了求解其中的参数，需要进行大量的计算。目前，MPP模型的计算技术主要有以下几种：
1. 基于统计模拟的方法：这种方法模拟微粒、吸附点、分子扩散层及表面上的所有相互作用，以估计出每个分子扩散层的参数。通过对不同条件下的模拟，可以计算出不同条件下的MPP模型参数。
2. 基于数值模拟的方法：这种方法采用计算机程序来模拟微粒、吸附点、分子扩散层及表面上的所有相互作用，然后求解出MPP模型参数。目前，最常用的数值模拟方法有时间积分法（Time Integration Method，TIM）和经典欧拉方法（Classic Euler Method，CEM）两种。
3. 通过实验测量的方法：MPP模型的参数可以通过对实验进行数据收集和处理，来获得。例如，在液滴界面温度为常温的常见实验条件下，可以通过测量表面温度来获取微粒表面热传导系数。在材料表面附近实验测量表面热传导系数，可以获得摩擦力系数；在分子表面附近，实验测量的大小电荷势能可以获得电荷势能系数。
### （2）实验验证
对于MPP模型，需要建立符合实际情况的实验平台，才能对其进行验证。例如，在很多国家，试验室的设施还比较落后，所以在实验上可能会遇到困难。另外，MPP模型仅仅局限于理论和物理层面，而无法精确预测实验结果。因此，需要结合很多方面，比如微生物学、材料科学、力学、计算机科学、统计学等，共同构建起一个完备的MPP模型的验证平台。
