# 1.背景介绍


电子器件制造中涉及到薄膜(silicon-based)、氮化物(nitride)等多种材料的制备，而使得成品表面存在各种沉积物质，这些沉积物质不仅对成品的性能影响巨大，而且会影响产出的成品质量。对于电子器件而言，由于其特殊结构及大面积占用空间，薄膜材料与其他分层材料的结合还会对其结构紊乱及功能失调产生影响。
例如：嵌入式系统中的硅基微型芯片，可以从厚度700~900nm的Ni-Cd/Si晶体结构中获得；
激光器件中的沉积物如PZT/CVD、MoS2/Ag/Sm等，可在晶格和半导体间形成开孔状态或短路连接；
集成电路IC中的绝缘子、电极/印制层等材料，都需要通过锻造、烧焊、冲压、浸泡等工艺得到，并可能受到多种温度、湿度、缺陷等因素影响。
因此，如何有效地管理电子器件的沉积物与温度、湿度、缺陷，实现长期稳定又保证产品质量成为一个值得关注的问题。
# 2.核心概念与联系
## 2.1.化学气相沉积
现代电子器件的制备过程中，由于层次复杂的设计，不同层之间的气相互作用，尤其是负离子气相变，对电子器件有着十分重要的作用。负离子气相变化是指由正离子组成的气体和负离子构成的气体发生化学反应，释放出新的气相子。由于引起负离子气相变化的物质包括温度较低的氢、氦、氮等，不同物质的气相变化情况也不一样。对于硅钙、镍钴等金属，层之间的温度、湿度均较高，使得他们之间的气相变化和蒸汽化非常缓慢，通常不会引起电子器件沉积。但对于硅氧化物(Silicon Oxide, SOI)，含有相对较少的正离子和较多的负离子，它呈现了一个介于金属与氧化物之间，温度比较高的状态，层之间的热流也比较大，故而对SOI的掺杂会引起电子器件的沉积，特别是在硅表面上。
一般来说，硅O2和GaO在表面方向上化学反应较少，基本不改变原有的结构，但在掺杂层中则可以显著改变电子器件的结构和性能。如GaO掺杂在金刚石墨烯、镁磝胞中，可以生成金刚石墨烯/镁磝胞超导片，极大的降低了电容工艺的难度。硅O2和GaO之间的化学反应则表现出明显的热力效应，能够加快固化过程，提高产品的尺寸。
除了反应速度差异外，硅氧化物还有着其独特的热容性，热容体内的溶解氢离子数量随温度的升高而逐渐增多，所以在许多热闹的环境中硅氧化物也容易被击穿，导致电子器件的破坏。为了保护电子器件的安全，人们利用表面的化学渗透率、复合材料的制作方法、冷却工艺以及隔热、防腐蚀等方法，有效地控制电子器件的气相变化，抑制其沉积。
## 2.2.电子相关物质沉积
电子器件制备过程中，由于制备精度的限制，电子会留在电子部件的表面，而不会充满整个材料。在制备过程中，电子会嵌入到各种物质中，例如半导体、硅基金属、二氧化硅、金刚石、冰点材料等等，形成一种特殊的表面附着模式。这些附着物一般称为“电子相关物质”（Electronically Related Materials）。由于各类电子相关物质的特性和结构不同，使得它们之间还存在着复杂的相互作用，在电子器件制备中可能引起沉积。
电子相关物质中，硅基材料和铝基材料会受到表面电子的束缚，形成受体状粒子，形成电子关联层，在相同尺寸下，电子相关层越厚，材料的电子表面附着效果越好。在这个阶段，出现了以硅基材料为代表的Ni-Cd/Si/Ga-In系结构，成为电子器件的最常用的结构。此外，Au-Ir/Co/Zn等材料也具有很强的电子介电能力，在某些设备上也可以作为电子滤波层。
另外，其他材料，如电镜材料、颗粒状物料、金属氧化物等，也可能会产生电子相关物质。这些电子相关物质与电子器件的功能密切相关，且存在着巨大的化学能量，这就需要进一步的掌握和利用，才能有效防止电子器件的损伤。
## 2.3.薄膜材料的影响
电子器件制备过程中，牵涉到的材料类型繁多，而且成本昂贵，无法一次购买完整。电子器件的制备流程往往依赖于熔炼、压制、冷却等一系列工序，而这些过程都会引入一些临时使用材料，如镍钴铜镁酚、银白锇、硫酸石墨、硫化橡胶、氯化钠等，这些材料在制备过程中都会产生沉积。近年来，随着科技水平的提升，人们逐渐意识到，薄膜材料对电子器件的制备有着决定性的作用，甚至是关键性的作用。
所谓薄膜材料，是指带有一定厚度的微米级，如0.25um，甚至更小的材料，比如铛，它的功能类似于金属纳米管，可以形成接地线，但是由于没有实际物理结构，不会像真空管那样承受大面积流量，所以便于在高频电场下工作。由于薄膜材料在制备过程中所产生的物质变化很小，所以制备速度比其他材料快很多。
在电子器件制备中，一般使用10nm以下的微米薄膜，因为在该范围内，电子与棱针头的距离可以忽略不计。这样就可以用精简后的电子管道代替之前的庞大的机械电子流水线，缩短制备时间，节省经费。同时，由于薄膜材料的特性，使得它们在制备过程中很难产生沉积，所以在电子器件制备中非常重要。
