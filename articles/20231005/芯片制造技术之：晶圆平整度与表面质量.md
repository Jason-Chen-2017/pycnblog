# 1.背景介绍


晶圆是常用的工艺件类产品，其重要性不言而喻。晶圆的制造过程主要由研磨、分拣、组装、烧结、热处理、冲击成型等几个主要环节组成。因此，了解晶圆的制作原理至关重要。晶圆的制造过程由晶体管工艺、制造工艺、电子工艺、物理和化学工艺共同组成。其中，晶体管工艺占据主导地位，影响着晶圆的质量、表面积、精度、纵横比、耐腐蚀能力等关键性能指标。
晶圆的平整度指的是晶体管之间的空间分布程度，包括线宽、线距、线厚、距离等方面的信息，可以直接反映晶圆的晶圆度。晶圆的平整度越高，则说明晶圆的结构越稳定；晶圆的平整度越低，则说明晶圆的结构越不稳定。而晶圆的表面质量与晶圆度直接相关，越是平整的晶圆，其表面质量也就越高。
# 2.核心概念与联系
## 2.1 晶圆度及其计算方法
晶圆度又称晶球度，是描述晶圆的一种性能指标。它反映了晶圆内单个晶粒之间的距离、数量、角度关系以及不同位置上的晶粒间的摩擦力分布情况等因素的综合结果，用来测量晶圆的表面质量和结构稳定性，并用来评价晶圆性能。晶圆度值通常取决于晶圆表面晶粒大小、晶圆形状、晶圆厚度、晶向（轴）的粗细程度、材料的类型、温度、表面加工方法等多种因素。计算晶圆度的方法通常如下所示：
1. 晶体中晶粒的大小：晶圆度与晶体中晶粒的大小息息相关，一般来说，晶体中晶粒的平均尺寸越小，晶圆度就越高。但需要注意的是，不同晶体、不同晶圆度下，晶粒的大小会发生变化，因此，真实的晶圆度值往往会受到不同的因素的影响。例如，金属晶体、镁晶体、青铜晶体、锰晶体晶粒尺寸都非常小，而镍、铅、锌等金属晶体的晶粒尺寸却较大。
2. 晶体的晶圆形状：晶体的晶圆形状决定了晶圆的外观及晶圆度的计算方法。典型的晶体晶圆形状可以是斜椭圆或曲面，如贝壳、炭黑等；也可以是斜圆或椭圆形状，如石英、铁钢、冷轧等。在上述晶体的情况下，计算晶圆度的方法有两种：
   - 方法一：对于斜椭圆形状的晶体，计算时可以用晶体中晶粒的密度来估计晶圆度，即通过每平方米内的晶粒数目来求出晶圆度，此方法被称为密度法。
   - 方法二：对于曲面形状的晶体，计算晶圆度需借助探测器工具或模拟仿真软件进行数学模拟，依靠拉丝宽度等参数来估计晶圆度，此方法被称为探测法。
3. 晶体厚度：晶体厚度越厚，晶粒越小，晶圆度也就越高。但是由于不同晶体厚度下晶粒的大小可能不同，导致计算结果误差较大。另外，晶体厚度也对晶圆度的准确性产生影响，在厚度相同时，晶圆度也会出现波动。
4. 晶向（轴）的粗细程度：晶向（轴）的粗细程度越粗，晶圆度就会越高。但是，晶向粗糙度也会带来明显的误差。
5. 材料类型：不同的材料会对晶圆度有较大的影响，如，镁、铅、锌晶体的晶粒相对更小，因此晶圆度会低于其他晶体，而石墨烯、绝缘材料、电介质等材料的晶粒则会相对更大，晶圆度会高于其他晶体。
6. 温度：由于晶圆的组成晶粒在温度作用下会发生热运动，而晶粒的热运动将会增加晶粒间摩擦力，导致晶圆度减少。当晶体达到一定温度时，晶粒之间摩擦力减弱，晶圆度升高。但是，由于不同材料、不同工艺条件、不同温度等因素的影响，实际的晶圆度值会随时间变化。
7. 表面加工方法：表面加工方法会直接影响晶圆度的计算结果，如，对于经过精心加工的晶圆表面，晶粒间的摩擦力会增强，从而使晶圆度降低，或者晶圆度高的晶体表面经过自然光照射后可能会发生变色，导致晶圆度的降低或提高。
## 2.2 表面质量
表面质量是描述晶圆表面质量的一个重要性能指标。晶圆表面质量通常用以克为单位表示，越高代表晶圆的质量越好。表面质量值是对晶圆的组成元素进行微观分析后得到的结果，体现了晶圆表面组织的各项性能。表面质量计算方法主要基于晶体学知识、晶体结构、表面层的形态学特征以及加工工艺等。表面质量值的计算公式如下所示：
$$\sigma=\frac{m}{A}$$
- $\sigma$：表面质量
- $m$：晶圆表面离表面最近的点的高度，$m=f(r)$
- $A$：晶圆表面积
- $r$：晶圆半径

表面质量越高，晶圆的质量越好。但是，由于表面质量仅代表晶圆表面质量的一部分信息，因此还需要考虑其他方面的因素，如：
1. 层数：晶圆表面有多个层，表面质量高的晶圆，其单个层面的质量可能较高，而层的总体质量可能偏低。
2. 多孔材料：晶圆表面可能具有多孔结构，表面质量较高的晶圆其单个孔的质量可能较高，而总体质量可能偏低。
3. 内部材料：晶圆内部可能存在浆膜、填充物、颗粒物等物质，这些物质的质量会影响晶圆表面质量。
4. 表面形态：晶圆表面形态复杂，表面质量的确定可能比较困难。

综上，晶圆表面质量是一个综合性的性能指标，需要综合考虑以上因素，才能得出准确的表面质量数据。
