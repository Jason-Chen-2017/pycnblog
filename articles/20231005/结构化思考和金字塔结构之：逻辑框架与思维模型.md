# 1.背景介绍


结构化思考（structured thinking）是一个重要的思维方法，它借助于拆分和分类已知信息，运用逻辑顺序、步骤和关系，从而得出对问题的处理或决策方法或判断，并将其组织成“结构”形式的知识，形成新的观点、经验或概念。在实际生活中，结构化思考的方式应用非常普遍，如对问题的分析和解决过程，产品设计，项目管理等。
结构化思考和金字塔结构（Pyramid of Reasoning）是一种理论型思维模式，主要用于理解复杂问题和取得解决方案。在金字塔结构中，主要有“推理”、“比较”和“归纳”三个层次，每一个层次都由上而下逐步细化。推理层关注如何从较小的事物中得到更大的知识，包括从知识的最小单位到整体知识，也就是通过重复、实践、归纳和总结等方式深入理解事物背后的原因和规律。比较层考虑不同选择对结果产生的影响，并进行取舍，包括直觉、经验、逻辑、道德判断等。归纳层则是从多个不同源头获得的信息汇总，形成对问题的新看法或更好的答案，是最后一步完成各种思考结果的过程。
对于结构化思考和金字塔结构，本文将详细阐述其基本概念和思想。在后面的章节中，我们会根据这一理论重新梳理现有的学习材料，尝试发现不同主题和相关理论之间的关联关系，进而寻找新的突破口。希望通过阅读本文，能够帮助读者了解结构化思考和金字塔结构的概念和内涵，掌握此类理论的应用及创新思路。
# 2.核心概念与联系
## 2.1 结构化思考
结构化思考指的是运用拆分、分类、逻辑顺序、步骤和关系等手段，从已经存在的、复杂的事物中，提炼抽象意义最强的关键要素，将其组合成逻辑上的整体，进而寻找其解决或决策的方法或判断，并将其呈现为结构化的形式。因此，结构化思考旨在达到以下目标：
1. 能够将复杂问题转变为简单易懂的表述；
2. 将原本混杂无章的问题变得清晰易懂；
3. 提供新的解决办法或策略；
4. 提升问题解决的效率、准确性或可信度；
5. 提高解决问题能力，提升个人成长。

结构化思考和金字塔结构，是在目前研究热潮下的产物。结构化思考是对分析问题、求解问题、系统设计、领域建模等方面已有理论的延申和扩展，是对人的认知发展方向和工作方法的一次革命。结构化思考的前身，来自1978年诺贝尔信息处理奖获得者马尔科姆·格雷戈兹的著作《为什么说人们容易迷失？》。他把日常生活中的各种问题划分为七个阶段：分析、归纳、判决、情感、计划、执行和思考，而通过分析阶段得到的结果，可以作为后续的归纳和判决阶段的输入。在七个阶段中，归纳阶段是结构化思考的核心。
## 2.2 金字塔结构
金字塔结构也称作“推理金字塔”或者“推理环”，是结构化思考的一种逻辑框架，被广泛用于复杂问题的分析和解决。它的基本思路是将世界上所有可能的事物，按重要程度大小排列，从“物”开始，逐步向上分级。该结构共分为三层：推理层、比较层和归纳层。每一层的内容都比上一层略微宽松一些，但又是连贯的、完整的。
- **推理层**：推理层用于找出问题的答案，可以分为“假设”、“观察”、“数据”等方法。
- **比较层**：比较层试图通过分析不同方案的优劣，确定更好的方案。
- **归纳层**：归纳层将不同信息整合起来，按照规律、原理或假定，予以综合，得出最终的答案。

推理层包括：
- 假设：人们往往先提出假设，然后验证假设的真伪。
- 观察：观察者通过实地观察，识别问题所在，并提出自己的见解。
- 数据：通常采用数据驱动的方式，通过统计分析的方法，对问题进行探索。

比较层包括：
- 直觉：直接从直觉判断的好坏。
- 经验：接受一些经验教训。
- 逻辑：基于推理规则的判断。
- 道德判断：基于社会道德标准的判断。

归纳层包括：
- 概念：人们会不断的发现和创造新的概念。
- 模型：构建模型并验证。
- 总结：透过已有信息的总结，寻求新的突破。

简言之，金字塔结构的思想就是：
> 找到问题的症结，深刻理解其根源；对不同方案做客观分析，发现其优劣；归纳经验、知识、理论和模型，创造新的模型或思维方式，来提供新的方案。
