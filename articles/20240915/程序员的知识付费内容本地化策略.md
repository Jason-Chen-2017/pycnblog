### 5.2 源代码详细实现

以下是一个简单的Python示例代码，用于提取文档内容并进行本地化：

```python
import os
import subprocess

def translate_text(text, target_language):
    # 使用Google翻译API进行翻译
    command = f"curl -X POST 'https://translate.googleapis.com/translate_a/single?client=gtx&sl=auto&tl={target_language}&dt=t&q={text}'"
    result = subprocess.run(command, stdout=subprocess.PIPE, text=True)
    return result.stdout

def localize_document(document_path, target_language):
    # 读取文档内容
    with open(document_path, 'r', encoding='utf-8') as file:
        content = file.read()

    # 翻译文档内容
    translated_content = translate_text(content, target_language)

    # 保存翻译后的文档
    output_path = os.path.splitext(document_path)[0] + f".{target_language}"
    with open(output_path, 'w', encoding='utf-8') as file:
        file.write(translated_content)

# 使用示例
localize_document("example.md", "zh-CN")
```

### 5.3 代码解读与分析

上述代码用于将Markdown文档翻译成指定语言。主要步骤包括：

1. **读取文档内容**：使用`open`函数读取文档内容。
2. **翻译文档内容**：使用`curl`命令调用Google翻译API进行翻译。
3. **保存翻译后的文档**：将翻译后的内容保存到新文件中。
