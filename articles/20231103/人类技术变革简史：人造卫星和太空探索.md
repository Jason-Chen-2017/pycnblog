# 1.背景介绍


人类从远古到今天，一直在不断推动科技进步。一百多年前，人们发现了“原子弹”。也就是说，如果利用化学武器制造出原子的能量并将其释放出来，它可以对全宇宙进行毁灭性打击。这是一项不可思议的发明，被称为“宇宙之火”。五六十年后，苏联发现了“苏-2”号卫星，用科学技术进行太空飞行，最终成为俄罗斯的霸主。上个世纪八十年代末，美国在阿波罗登月成功地将第一颗人造卫星送入轨道，这是近代最伟大的科幻作品之一。
在过去的三十多年里，人类通过技术的发展以及社会的进步，创造了诸如航天飞机、汽车、电脑等等各种新事物，也面临着各种各样的挑战。例如，中国要建设新的高铁，不得不考虑用火箭火花点燃森林，或者为了避免危险而使用核弹；印度的农业生产面临着环境污染、水土流失的问题；俄罗斯要建造核反应堆，又不得不考虑环境污染、核辐射危害等因素。这些挑战已经成为科技领域的永恒难题，也因此催生了许多新领域的发展。
二十一世纪的今天，人工智能、互联网、区块链、物联网等新兴技术的快速发展，让人类的生活和工作越来越依赖于计算机程序的计算能力，极大地提升了我们的生活质量和效率。其中，人造卫星在人类信息技术发展史上的重要地位和作用尤其突出。人造卫星可以帮助我们探索宇宙，也可以给我们带来经济利益，甚至改变世界历史进程。
在人造卫星的历史上，自从古埃及人驾驶火车时，就一直在利用火箭做实验。但是，古埃及火车缺乏动力，需要采用其他运载方式才能持续航行，所以才有了“火车道”。之后，希腊人改用助推器作为火车载重装置，以此来驱动火车在陆地上行驶。随着航空运输技术的发展，许多人开始寻求通过无人机或航天器进行高速、低成本、高度安全的空间探测。直到最近，我们才真正意识到“航天”的威力。
那么，人造卫星到底为什么能帮助我们探索宇宙？人造卫星的主要功能之一就是探测。与传统的卫星不同，人造卫星能够探测到非常遥远的位置，而且具有巨大的探测范围。可以想象，如果在宇宙中存在着类似于太阳这样的巨型天体，那么人造卫星的探测范围可能比整个银河系还要广阔。另外，人造卫星也不受地球引力的影响，具有极大的空间自由度。另外，人造卫星通常都配备了各种探测设备，比如望远镜、激光雷达、红外探测器等等。因此，利用人造卫星可以获取有关宇宙的各种信息。
总结一下，人造卫星已经成为一种引人注目的科技创新领域。它的历史虽然很短暂，但在今天仍然是巨大的科技宝库，有着极大的应用价值。我们应该把握好科技的演进方向，将人的才华和智慧投入到科技创新中，而不是重复沉没在历史的垃圾堆里。
