# 2.核心概念与联系

## 2.1 概念

- **人工智能（Artificial Intelligence）**：指的是机器拥有自主性、智能性，且可以通过感知、思考、学习来完成某些特定任务的能力。它可以进行精确的分析、决策和执行，并通过高度的自我学习来改善性能。
- **机器学习（Machine Learning）**：是人工智能的一个领域，它以数据为基础，利用计算机的算法和统计模型来训练自己进行一些预测和决策。机器学习方法的目标是开发一种算法，使计算机能够自我学习，从数据中提取信息并做出有效的决策。
- **深度学习（Deep Learning）**：是机器学习的一类技术，它的特点是由多层复杂的神经网络组成，能够学习数据的抽象特征，而非像其他机器学习方法那样依赖于显式的规则或假设。深度学习能够处理高维数据，并且可以在多种任务中取得最佳效果。
- **强化学习（Reinforcement Learning）**：是机器学习中的一个子领域，其研究目标是在特定环境中智能地选择行为，以最大化奖励。强化学习的核心是一个智能体（Agent），它接收周围环境的状态（State），然后根据其动作（Action）在环境中采取反馈（Reward）。智能体根据这些反馈来选择最佳的动作。
- **监督学习（Supervised Learning）**：是机器学习中的一种学习方式，它通过已标注的数据来训练模型，目的是在给定输入条件下预测输出结果。监督学习有两种模式：一是回归（Regression），二是分类（Classification）。
- **无监督学习（Unsupervised Learning）**：是机器学习中的另一种学习方式，它不需要任何已标记的数据。无监督学习的目标是在未知的数据集中发现隐藏的结构。无监督学习算法包括聚类（Clustering）、降维（Dimensionality Reduction）、关联（Association Analysis）等。
- **半监督学习（Semi-Supervised Learning）**：是一种特殊的监督学习方法，其中有部分数据已经被标注，但大部分数据没有被标注。半监督学习的目标是在数据量较小的情况下，仍然可以训练出有效的模型。
- **元学习（Meta-Learning）**：是机器学习的一种学习范式，它将机器学习技术应用到其它机器学习算法之上。元学习算法尝试基于已有的经验，训练模型生成新任务的学习策略。
