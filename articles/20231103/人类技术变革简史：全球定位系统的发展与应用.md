# 1.背景介绍

  
随着科技的飞速发展，我们的生活越来越多样化、智能化，但同时也带来了新的问题：如何让人们更好地生活、工作、学习和交流？例如，怎样帮助个人更有效地解决日常生活中的各项问题，如用手机与他人视频通话、参加兴趣组活动、寻找工作、购物等；怎样让城市更美好、安全、便利、经济运行良好；怎样提升个人生活品质、减少压力、改善工作效率；怎样建立知识网络、促进人际关系、提供优质服务；如何在中国共产党领导下，推动科技发展。这些问题的关键都在于人类如何“定位”自己。如何准确且实时地发现自己的位置并移动到周围适当的位置，成为当前的一项基本技术问题。
全球定位系统（Global Positioning System，GPS）是目前应用最广泛的定位技术之一。它可以精确定位用户所在的地理位置，为各种需要导航功能的应用提供了重要依据。通过对卫星的遥感接收、导航算法与配套导航设备的完善，GPS可以实现高度精确的定位。但是，GPS本身并不完美，存在着众多缺陷，其中包括误差、分辨率、可靠性、数据处理能力、穿透、用户隐私保护等问题。因此，为了更好地利用GPS技术，引进智能定位技术成为当务之急。
国际标准组织（International Organization for Standardization，ISO）制定了一系列标准，即全球定位系统规范（GNSS），旨在统一GPS与其他定位系统的技术标准，促进它们之间的互操作性。另外，美国商务部与欧盟共同开发出了另一款定位系统——北斗卫星定位系统（BeiDou Satellite Navigation）。随着人们对GPS的需求日益增加，全球定位系统市场的规模也越来越大。但由于其复杂的特性及缺点，很难形成完整的标准体系，导致国内的技术研究工作面临巨大的困难。因此，基于ISO/IEC 19118标准的《人类技术变革简史》的主题报告系列便产生了。
# 2.核心概念与联系  
本文将重点分析全球定位系统（GPS）的发展、应用及其所存在的问题。首先，我们要理解GPS的概念和功能。GPS由两部分组成，即卫星传输链路和定位系统。其中，卫星传输链路负责将卫星的传播信号编码，并通过无线电波发送到空间站上进行轨道同步。定位系统则根据接收到的卫星传播信号和解码得到的导航信息，计算出用户的实际位置。定位系统主要用于定位车辆、船舶、航空器等移动物体或接收器的精确位置。GPS系统的主要特点包括：高精度、高灵敏度、高可靠性、低误差、高分辨率、数据处理能力强。其次，我们还要了解GPS与其他定位技术的区别。GPS具有全球性、固定频率、高分辨率、实时性和穿透性等特点，可以用于导航、航空导航、环境监测、疾病预防、跟踪等多个领域。其他定位技术一般采用模糊定位、定位基站或者移动台部署的方式，或者需要基于信号雷达技术的雷达型定位方式，或采用基于移动充电指示灯技术的WIFI/蓝牙型定位方式。不同定位技术之间的差异主要在于精度、分辨率、成本和应用场景等方面。第三，还要了解GPS的发展历程及其应用领域。从GPS第一次在1986年被发明至今，它的发展经历了十几年的时间，主要包括以下几个阶段：
- GPS原型阶段：1979年，Moore与Rivest等人在卡纳维拉国家大学(Canada)进行了第一版设计，但由于种种原因没有获得成功。
- GPS设计阶段：1986年，NASA、NOAA与JPL联合创立了GPS卫星定位系统（GLONASS）项目。该系统基于土耳其Sputnik1号卫星系统，在欧洲发射通信。1988年，首个商用的GPS终端诞生。1993年，第一个国际标准GNSS规范诞生。
- GPS应用阶段：1997年，GPS正式进入用户消费者应用阶段，覆盖了全球超过90%的人口。2007年，美国召开的GPS国际会议宣布，GPS正式为个人和商业领域的许多应用提供服务。
- GPS更新阶段：2006年至今，GPS系统每隔10至15年就会发布一次升级版，包括新的卫星系统、新的导航算法、新的数据处理技术和设备等。目前，世界上已经有超过1亿颗GPS卫星，平均每天都会发送超过百万条定位数据。
GPS的应用领域非常广泛，涉及医疗保健、教育、金融、交通、军事、地图制作、出租车导航、车辆控制、零售店定位、景点探索、社交网络、视频游戏、移动互联网、航空导航、气象观测、地震预警、环境保护、测绘、风险评估等领域。除了以上应用领域外，GPS还被应用在很多军工、环保、农业、石油勘探、林业、化肥、电子政务、智能建筑、汽车科技等领域。
而GPS所存在的问题也很多，比如误差、分辨率、可靠性、数据处理能力、穿透、用户隐私保护等问题。其中，误差指的是GPS定位结果与真实位置之间的距离。分辨率表示GPS定位精度。可靠性代表定位数据的有效性。数据处理能力代表GPS系统能够处理多少数据量。穿透通常发生在某个地方的信号强度较弱，而用户却仍然不能得到准确的位置信息。用户隐私保护问题可能使一些人无法直接享受GPS技术带来的便利。因此，为了提高GPS的应用范围，更好地保障个人信息的安全，提升GPS的标准化水平，国际标准组织正在努力推动相关标准的制定。
