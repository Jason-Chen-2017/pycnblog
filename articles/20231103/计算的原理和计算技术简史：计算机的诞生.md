# 1.背景介绍


计算（Computing）这个词一直伴随着人类的发展，始于人工智能领域。通过计算，可以解决很多复杂的问题。早在几千年前，远古的人类就已经用火堆煤气对阴暗地带进行探险、开发工具等。之后，人们制造出了第一台数字电子表计算器，帮助人类解决了复杂的问题。比如：利用算盘计算圆周率、求解约瑟夫环问题、算术运算、图形处理、数据处理等等。

由于大型机并不是每个人都能够轻易掌握，因此人们又基于微型计算机的想法设计了个人计算机（PC）。这些小型计算机能够运行各种各样的应用软件，极大地提高了人们的工作效率和生活质量。在20世纪70年代末，第一台个人计算机IBM PC诞生。在PC出现之前，计算技术基本上是靠纸和笔来进行的。

1956年，美国物理学家丹·卡门曼第一次向公众展示了自己构造的原子模型——艾普曼体系，显示了电子云的存在。它开启了人们对原子和原子核以及它们之间相互作用有关的科学研究。20多年过去了，人们终于搞清楚了很多有关原子的基本知识，并且制成了比较完整的材料——玻璃体、金属，还有某些特殊形状的材料，比如分子表面、水滴、膜蛋白、核反应堆、超新星爆炸产生的碎片……这一切的基础都是由人工确定的原子模型构建而成。

1961年，两位科学家艾伦·米勒和马修·麦克卢汉联合创立了微型计算机和互联网，开拓了计算机科学研究的新方向。他们将计算视为一种智力活动而不是像机械一样工具性的活动，并将其与实际生活紧密结合。微型计算机意味着更小的体积和高速的处理能力，使得它可以作为个人电脑、小型嵌入式系统或者作为服务器来运作。同时，互联网为信息服务的发展提供了新的模式。这个时期的计算技术被称为计算机时代。

从物理学到材料科学再到人工智能，计算技术不断进步。例如：信息论、密码学、网络、机器学习、量子计算、深度学习、区块链、云计算等等，都是计算技术的重要组成部分。物理学和材料科学的进展促使人们认识到可以利用计算技术进行数值模拟和建模，从而解决一些复杂的数学问题。近年来，云计算领域正在蓬勃发展，它利用分布式计算资源实现超大规模的运算任务，同时降低运算成本。

2000年代后半叶，随着个人电脑的发展，越来越多的人开始接受智能手机、平板电脑等智能设备。智能手机上的应用程序越来越丰富，使得人们的工作和生活变得越来越智能化。虽然智能手机带来的便利和娱乐，使得人的工作效率得到大幅提升，但是同样也带来了一系列的安全隐患，导致更多的病毒感染、个人信息泄露等问题。为了防止这种问题，2010年发布的移动支付支付卡业务正逐渐成为行业的热点。它的安全性和可靠性保证了消费者的信任。

如今，智能手机已经成为我们生活中的必需品。同时，在“远程办公”、“无人驾驶汽车”、“智慧城市”、“医疗健康”等方面，计算技术也扮演着越来越重要的角色。这些发展给我们带来了巨大的商业价值和社会影响力。随着计算技术的不断进步，我们也在探索计算之外的可能。



# 2.核心概念与联系
## 2.1计算
计算就是用计算机程序进行自动或半自动的、重复性的、计算性的工作。计算的对象通常是一个个的数字、符号、数据或指令。计算以算法形式指定，通过执行程序，输出结果。计算的目的有很多，包括创建数字模型、发现隐藏信息、改善产品性能、解决复杂的数学问题、验证假设、分析数据等等。

计算的三个阶段:
- 1940s-1950s 符号逻辑和数字逻辑
- 1950s-1960s 数据表示、运算和存储器组织
- 1960s至今 流水线、矿机、GPU、超级计算机等通用计算机
