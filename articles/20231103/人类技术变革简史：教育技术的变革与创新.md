# 1.背景介绍


## 1.1 概述
“人类技术变革”这个词语指的是通过对制造、交流、存储、处理、控制等方面的各种信息设备和技术进行逐步改善或革命的方式，使得我们的生活更加便利、高效、智能化。它涵盖了计算机技术、通信技术、物联网技术、生物技术、光学技术等所有能够影响人的生活的领域。其中最重要的一个领域就是教育技术。
近几年来，随着国内外教育产业蓬勃发展，教育技术也经历了一场技术革命。如今，以学生为中心的数字教育正在成为真正的主流，也是学习的前沿。同时，笔者认为，教育技术永远是一个“纸老虎”，只是时间的问题。
对于教育技术的变革而言，从古到今都面临着巨大的挑战。今天我们就来梳理一下教育技术的历史，探寻它的起源与演变。
## 1.2 教育技术的发展历史
### 1.2.1 中国的教育技术革命
#### 1.2.1.1 雅塔改进计划与传统教科书
19世纪70年代中期，中国开始实行科举制度。但是这种制度实在是太麻烦了。于是，科举教官为了弥补手中的军队，不惜向一些贫苦农民索取钱财，为这些农民开办学堂。这样做导致了贫苦农民的劣迹昭然若揭。因此，当时由清政府筹建了一个名叫雅塔的教育部。雅塔改进计划旨在“改善普通工商户的教育条件”，目的是想通过对农村贫困儿童的教育工作提高他们的能力，以达到“养懒汉，富乡镇”的目标。该计划于1905年正式启动。由于该计划直接影响到农村教育，所以这段历史被称为“雅塔改革史”。

1907年，中央政府下令停止科举考试，此后，雅塔改革计划逐渐淡出视线。

1909年，全国各地农村中小学生陆续受到初等教育，也积极参与到公立学校教育中来。到1912年，全国已有1万多个初等、高等学校，数千万孩子受到良好教育。但由于经费上的限制，很多农村贫困学生仍然无法上学。

1913年，中央政府颁布了《第一修正案》，将教育权力下放给社会。对贫苦农民实施奖学金制度。与此同时，国家也开始实行免费教育。

#### 1.2.1.2 第一次鸦片战争与鸦片危机

1920年，中日甲午战争爆发，日本侵略扩张，许多中国知识分子感到担忧。因此，他们组织起来，反对“满洲国”的滥用祖国资源，号召“停止内战，废除帝制”。

1925年，满洲国在日本的支持下，借口侵华，炮制了一套侵略中国的大计。这支由“阴谋集团”组成的军队，宣称“我一定要把日本占领者打败”。结果，日本军队胜利，但“阴谋集团”的卑鄙行为却得到了执行。这场引发中国“鸦片危机”的事件，标志着“内战”及其敌人的覆灭。

#### 1.2.1.3 “辛亥革命”与康乾主义
1911年10月8日，发生在中国广州的一场规模浩大的群众运动，被称为“五四”运动。社会主义的旗帜笼罩了中国。社会主义派别的领袖康有为，早年投身革命活动，受到社会舆论的高度关注，鼓吹毛泽东的“新民主主义”，并提出了“共同纲领”。他站在学生的立场上，反对“汉奸卖国”、要求“统一”。学生们响应党的号召，拥护共产主义。但是，“五四”运动虽然取得了一些成果，但它触犯了国际国内法律，也遭到了当时的政府的严厉批评。

1912年10月，党的元老陈独秀发表了重要讲话，针对“五四”运动的种种错误，呼吁停止暴力革命，坚持新民主主义社会制度。他以“康乾主义”的名义，宣布“抗日胜利”、“北伐完成”，推翻清政府，结束了内战，建立了新的国家。

### 1.2.2 第二次鸦片战争与内战的结束
#### 1.2.2.1 “一二·九四八”与两广事变
1931年，国民政府迁往广州，中国革命风潮席卷整个东南亚。

1948年的“一二·九四八”事件，是中国近代史上最大的内战，超过半数中国人死于非命。从1949年10月至1951年3月，“一二·九四八”爆发了三次全面内战，是中国历史上规模最大的内乱。

孙文领导的民族解放战争，把广东、江西、湖南、四川等十三个省份划入自己的疆土。当时，解放军只有广州、上海、福州三地，其他地方的解放军根本就不管。孙文指挥的“中国军民反法西斯战争”从1949年11月开始，一直到1950年7月止，所到之处，仅关乎自己宣传“中国不可战胜”，实际上根本没有什么战斗。此时的广东、江西、湖南、四川等省，还保留着封建余毒。孙文说过：“闽粤苏三省而已。”

1949年10月底，孙文率领的国民党反动派攻入广州市，杀害了共产党驻广州代表马英九、陈潭秋等五十多人，并占领了广州市区。孙文迫使陈潭秋将自己控制的军队撤离广州，返回延安，孙文便率领他的部队返回故乡，继续参加同盟会组织的“二·七大会议”。

陈潭秋等人组织了叛乱团伙——“黑道团伙”，企图篡夺共产党的领导地位，实行“皇汉政策”。

1950年3月14日晚，“二·七大会议”召开。陈潭秋等五百余人举行起哄，朝鲜劳动党领袖金泽涛、高岗，日本在野党代表黄兴和托洛茨基，致电陈潭秋，要他与共产党作坚决斗争，撕毁“二·七大会议”协议。陈潭秋等不听劝阻，继续捏造“恩准”、“行动方案”，企图阻碍“二·七大会议”的通过。

孙文任命的中共临时政府由陈潭秋等人操控，他们对广州、上海、福州三地实行包围割据政策，并任命刘少奇为总书记。陈潭秋等人被逼自首，叛军占领了上海，保安派驻守福州、广州，以及赣南、吉林等地。在蒋介石和王明等人的策动下，“一二·九四八”变成一场惊天动地的大屠杀。

#### 1.2.2.2 “一二·九五○”与大跃进和文化大革命
“一二·九五○”之后，中国陷入了经济危机。国民政府不得不实行“共产风”，鼓励企业、公社、大专院校等私营部门进行“大跃进”运动，以大炼钢铁、推行食品高指标、修缮公路为先例。结果，中国面临饿死人的局面。

在大跃进失败之后，国民党政府加紧镇压民间反抗运动，甚至连“反修防修”、“总结经验”的宣传也全面展开。这是自“五四”运动以来中国民间反抗运动史上的第一次大规模反政府示威。1966年，广州被公安部查封。

在那个年代，中国是一个多党政治制度，遍布派别林立。无论是左翼文艺青年群体，还是中间势力（国民党和共产党），都在面临政治生态的变化。

在这种情况下，发生了著名的“文化大革命”。1967年5月，广州市委书记陈伯达发动了“破四旧”、整风运动，开展对所谓“修正主义”的批判，批斗知识分子。他说：“要在党内树立马列主义信条”。陈伯达命令全市大搞修旧改革，包括用农药代替砖头，在道路上铺设蜡像，强迫戴上脚镣，禁止开展“三面红旗”集会。他还命令各单位“严格落实党内精神路线”，要求大家注意文明礼貌。当年6月2日，“文化大革命”开始。

“文化大革命”的目的就是要打倒一切反动势力，夺回冤屈于集体、个人之下的人民，恢复和发扬民族优良传统。它一开始，就对社会上一切看不顺眼的东西，包括“修正主义”、“资本主义复辟”，进行了揪斗。当时，造反派挨家挨户检查斗，屠夫无情，扬眉吐气，造成很大震撼。

“文化大革命”中，中国共产党被迫暂停一切组织活动，只留下一个党支部，镇压反革命言论。不久，被镇压的人数接连增加。

1972年，中共总书记江泽民接任广州市委书记。江泽民宣布中国共产党、中央人民政府决心彻底破除一切形式的封建迷信，决心实行马克思主义。

1976年10月，广州爆发“八一”事件，这是一个倡导非暴力反歧化，反对党政干预民间活动的游行示威活动。“八一”运动结束之后，粤港澳三地陆续复工，国民经济恢复了发展势头。

### 1.2.3 第三次鸦片战争与邓小平理论和“反右派”运动
1976年后，中国共产党领导下发生了惊天大地的三次内乱。这三次内乱分别是：1978年的“文化大革命”；1989年的“六四”天安门事件；2003年的“一二九”事件。这三次内乱的原因都跟“文化大革命”一样，都是对共产党的政策实行错误的斗争。

这次“文化大革命”结束后，邓小平发表了重要的“邓小平理论”。他在1989年10月1日的讲话中指出，“党和国家领导制度确立以后，出现了一系列严重问题，特别是执政党内的资产阶级与地主阶级的矛盾日益尖锐，阶级斗争日益激烈。”他还主张，“社会主义制度的任务不能只限于实现公有制，必须把私有制、自由竞争和个体经济纳入制度的轨道。”

邓小平理论一出，“文化大革命”就很快被否定。到了1989年，由于社会矛盾激化，周恩来提出了“退守台湾”的想法，在“九一八”后，海峡两岸局势开始紧张。

1989年12月，周恩来接见港媒。他提出的“退守台湾”的想法，得到国际社会热烈响应。2001年，美国总统奥巴马访华时，提出“继续开放六四学生领袖李文亮的学术自由”等要求，并主张尊重两岸公民在台湾的正常生活。台湾当局立即释放李文亮，并许诺给予其学术上的平等权利。

2003年12月，李文亮在台湾病逝。这次“一二九”事件，标志着邓小平理论和“六四”天安门事件已经彻底破产。2003年10月2日，中央纪委决定对“闹事者”进行猥亵、殴打和监视居住十八年的惩戒。这被称为“六四事件”。

从1978年“文化大革命”到“一二九”事件，“四人帮”在全党范围内掀起全面文化大革命。这个“四人帮”并不是一伙假“毛派”，而是中共高层对毛泽东的最凶恶的继承者。

“反右派”运动是指1989年10月，中共中央发布“关于继续开展大批判、批评周恩来的《关于当前政策的几点意见》”一文。“文革”期间，批斗“现行反革命”的状况愈演愈烈。1991年5月，周恩来发表了“关于反右派运动的讲话”。

1992年2月8日，中共中央、国务院作出《关于召开中央全面依法治国若干问题的座谈会的通知》，提出“全面依法治国”、“实行党的领导”的新时代任务。

1992年4月1日，中共中央、国务院决定将《关于当前政策的几点意见》作为“反右派”运动的“导火索”。这件事成为“四人帮”在全党的惯用手段。

1992年6月，中共中央在北京召开“反右派”总结大会，宣布：“十一届三中全会以来，党的建设取得重大成功。不仅学术界在批判资产阶级自由化、修正主义问题上取得了很大进展，文艺界也在不断加强对这一问题的批判。”

1992年12月，中共十一届四中全会通过的《中共中央关于当前政策的决定》，提出了“以邓小平理论为指导，认真贯彻党的路线、方针、政策、对待一切非宪法性文件和现行反革命分子的态度，不断巩固党的执政基础和制度建设，全面推进国家治理体系和治理能力现代化，始终坚持四项基本原则，坚持改革开放方向，实行改革开放、人民民主、法治的总设计师和总裁─副总裁负责人身份”，成为新时代中国特色社会主义的建设指南。

“文化大革命”、“反右派”运动、“六四”事件、“一二九”事件，既是中国共产党的历史又是中国历史的事件。它们影响到中国的教育、政治、经济、文化和社会，展开了世界历史的进程。
