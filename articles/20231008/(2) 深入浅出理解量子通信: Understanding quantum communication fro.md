# 1.背景介绍


## 1.1 什么是量子通信？
量子通信是一个利用量子通信系统进行信息传输、交流、沟通的过程。它可以实现两端之间直接相互传递信息，还可以使得双方的处理和分析能力相对单纯的电磁通信更强大。而人类目前所熟知的电信就是一种经典通信方式，而另外一种可被视作量子通信的现象则是太空中地球之间的信息传输。这其中就涉及到了两个主要的量子通信系统，它们分别是量子网络和量子比特通信。
## 1.2 为什么要做量子通信研究？
### 1.2.1 突破电磁时代的信息壁垒
地球自转轴，太阳与月亮距离之类的基本物理规律已经无法通过光传播途径进行测量。这个信息壁垒导致了社会生活的极速发展，世界各国的经济迅速发展，而地球上的各种组织却无力应对这种突如其来的信息革命。因此需要有新的通信技术来解决这一难题。
### 1.2.2 提高通信效率
量子通信系统的实现，能够达到实质上一个比一个单纯的电磁通信系统的通信效率高出很多。此外，将复杂的计算任务转变为简单的纠缠计算，也可以有效地降低通信成本。另一方面，量子通信可以在更短的时间内完成某些复杂的任务，例如物理实验。
### 1.2.3 在高度不确定性下进行通信
当某个信息源的信号不可预测时（即存在一定程度的随机性），可以通过量子通信的编码技术来降低传输错误的可能性。同时，通过量子通信系统的纠错机制，还能在出现错误信号时快速恢复到正常通信状态。
## 1.3 量子通信系统的组成
根据量子通信系统的不同层级结构，分为集线器、半导体、量子引擎、量子通信基站、量子信道等几个部分。
### 1.3.1 集线器
集线器是指将多条线路连接起来的设备，也就是将几个电话线或电报线绕成一个环状，这样就可以把多个电话连接起来。它是最底层的通信系统，主要用来实现两台计算机之间的数据传输。集线器在低频段也有较好的性能，但一般只用于局域网通信，并不主流。
### 1.3.2 半导体
半导体又称硅，是一种具有活性、储存、传送功能的金属材料。每一个量子通信都离不开半导体。量子通信中的多个纠缠器件中都含有硅。这些硅介于纠缠管与量子引擎之间，具有极大的物理尺寸，而且还具有良好的抗干扰性。所以，量子通信系统中通常至少包含一些硅片。
### 1.3.3 量子引擎
量子引擎由量子位移引擎和量子控制单元组成。每个量子引擎都可以处理许多量子比特。量子位移引擎会将量子态从一个量子比特移动到另一个量子比特。量子控制单元则是进行量子门操作的单元，包括NOT门、Hadamard门、CNOT门等。不同类型的量子引擎还具有不同的特性。比如，色光刻蚀光量子引擎可以实现超精密的通信。
### 1.3.4 量子通信基站
量子通信基站（Quantum Communications Base Station）是通过信号处理技术将无线电信号转换为量子波，再通过光电耦合技术将量子波传输到接收端。它可以接收和发送信号，并将信号翻译成特定的编码，方便接收者解码。
### 1.3.5 量子信道
量子信道是指量子通信过程中用于传输、传播和接收量子信息的媒介。它包括光信道、无线信道和电信道等。量子通信基站作为中心节点负责管理整个系统的通信工作。每一条信道都要遵循固定的规则，才能保证数据的传输顺畅。
## 1.4 量子通信的分类
由于不同类型的量子通信系统都具有不同的特性，所以量子通信的分类也有不同的标准。按照主要的量子通信系统可以分为两种类型：
- Classical: Classical 通信系统的特点是全双工，通信速度快，成本低，且易于部署，但是同时也容易受到干扰。它们基于电磁波传输，属于静态信道。Classical 通信系统的应用包括数据传输、电话、视频会议、电子邮件等。
- Quantum: Quantum 通信系统的特点是可靠性高，同时又不需要任何外部设施支持。它们基于量子力学中的纠缠、同步、混合等特点，属于动态信道。而在实际使用中，这种信道往往采用弱化后的量子纠缠通信。比如说，如果两个节点之间没有通信信道，他们之间只能用其他方法进行通信，那对于信息传输的可靠性就无从谈起。因此，Quantum 通信系统需要有能够保护通信过程免受干扰的措施。因此，Quantum 通信系统的应用包括地球与地球之间的通信、星际通信、量子加密、量子计算等。
## 1.5 量子通信的标准
为了衡量量子通信的效果，需要有一个标准来评估它的性能。目前国际上有很多标准和准则，但仍然没有统一的标准。因此，我们可以自己制定一套量子通信的标准。下面列举一些常用的标准：
- TBF: Transmission bit rate。传输速率，单位bps。它表示某种通信模式下单个比特的传输速率。通常情况下，TBF越高，通信的可靠性就越好。
- EER: Energy efficiency ratio。能效比，它衡量的是用功效率和运行效率。用功效率与运行效率相比，前者代表了通信终端设备的运行效率，后者代表了通信线路中纠缠效率。EER越大，意味着通信的可靠性越高。
- FBER: Frame error rate。帧错误率，它衡量的是通信链路中传送的完整信息帧数占总传输帧数的百分比。FBER越小，意味着通信的可靠性越高。
- BER: Bit error rate。比特错误率，它衡量的是传输过程中发生错误比特的数量占所有比特的数量的百分比。BER越小，意味着通信的可靠性越高。
## 1.6 小结
量子通信是利用量子通信系统进行信息传输、交流、沟通的过程。虽然它可以使用电磁信号进行通信，但其性能远远优于单纯的电磁信号。根据量子通信系统的组成，可以划分为集线器、半导体、量子引擎、量子通信基站、量子信道五个部分。在标准的量子通信评价标准中，有TBF、EER、FBER、BER四项。希望本文能给读者提供关于量子通信的一些基础知识。
