# 1.背景介绍


“巫师陵阙”（ Wizard's tomb）是神话中的一个传说，主要讲述的是古代的一个修道院，里面有许多神秘的魔术师在修炼自己的魔法技巧，但由于某些原因，魔术师们无法从修道院出来。而当修道院被发现后，便有些神秘的巫师来到这里进行调查。为了证明自己没有伪装，巫师们在这里向神秘的人士宣告他们的死亡，并且赠送了一本神秘书籍，其中记载着各种奇怪的魔术知识。从而使得许多人相信，神秘的魔术师们已经毁灭了整个修道院。但是很多人却不相信，因为这本神秘书籍只是暗示，并非是真实存在。其实，当时许多巫师并不是一无所获，他们找到了一些不为人知的秘密道具，如强力的小型陷阱、纤细的魔灵手套等等，并从这些物品中获取到了丰富的信息。另外，还有一些年轻的天才学生也在这个修道院学习，他们努力的探索着这个陷阱之下隐藏的玄机。所以，很多人的疑惑就变成了，既然神话中出现了这样的巫师陵阙，那么现实中是否也有同样的情景呢？

## “巫师陵阙”相关概念
- **神秘的魔术师**：这里指的是那些为了获取神秘信息而聚集在一起的大魔术师。他们通过神秘的书籍或记录来传达智慧。
- **巫师陵阙**：这里指的是一个神秘的修道院，里面有许多神秘的魔术师在学习神秘技能，但由于某些原因，他们无法离开。当神秘的魔术师试图逃离时，会被一些异端魔术师抓住，导致死亡。
- **神秘的书籍**：这个书籍一般是一个秘密手册，记载着各种奇怪的魔术技巧。但是，在“巫师陵阙”的描述里，可能并不会提及这种书籍，因为它作为一种不可缺少的元素，是不可或缺的。
- **神秘的陷阱**：这个陷阱往往非常隐蔽，只有极少数的高级魔术师才能找到。他们会使用一些特别的物质或魔术方法，诸如旋风卷轴、诡异的血腥符文等，使陷阱不易被发现。
- **死亡宣告**：这里指的是那些企图证明自己没有伪装，而宣告自己死亡的巫师们。他们通常会用一些神奇的符咒，比如誓言、抽象想象、召唤出的末日、无法言喻的美女面孔等等，向神秘的魔术师宣告自己已经死去。
- **各种奇怪的魔术知识**：这里指的是那些掌握在巫师们手中的神秘魔术秘方。这些秘方会给予神秘的魔术师们强大的能力。
- **巫师学生**：这里指的是那些年轻的天才学生，他们在学习神秘魔术，希望能够找到隐藏在这个陷阱里的秘密。
