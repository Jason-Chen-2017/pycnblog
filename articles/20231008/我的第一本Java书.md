# 2.核心概念与联系

1.Java虚拟机（JVM）：Java运行在虚拟机上，它负责字节码的执行，并将其翻译成实际的机器指令。

2.类加载器（ClassLoader）：Java中的类加载器负责从文件系统或者网络上加载Class文件，转化为可以被虚拟机执行的内存中的Class对象。

3.面向对象编程：Java是一种纯粹的面向对象编程语言，具有封装、继承和多态等特征。

4.异常处理机制：Java提供了try-catch结构来处理异常。

5.多线程机制：Java提供对线程的支持，允许多个任务同时执行。

6.反射机制：Java通过反射机制可以调用类的私有方法和变量。

7.垃圾回收机制：Java的垃圾回收机制自动管理内存，有效地防止内存泄漏。

8.语法糖：Java拥有一些语法糖来简化代码编写。

9.集合框架：Java提供了丰富的集合类来帮助开发者进行数据存储和访问。

10.注解机制：Java通过注解机制可以在编译期间插入一些运行时信息，例如日志记录。

## 数据类型

Java语言有八种基本的数据类型：byte（字节型），short（短整型），int（整型），long（长整型），float（单精度浮点型），double（双精度浮点型），boolean（布尔型），char（字符型）。

```java
//声明变量
byte b = 1; 
short s = 2; 
int i = 3; 
long l = 4L; //使用后缀L表示long类型
float f = 5.0F; //使用后缀F表示float类型
double d = 6.0;
boolean flag = true;
char c = 'A';

System.out.println(b); //输出1
System.out.println(s); //输出2
System.out.println(i); //输出3
System.out.println(l); //输出4
System.out.println(f); //输出5.0
System.out.println(d); //输出6.0
System.out.println(flag); //输出true
System.out.println(c); //输出A
```

Java中声明变量不需要指定数据类型，因为Java的编译器会根据变量的赋值方式推断出变量的数据类型。但是如果需要显式指定数据类型，可以使用关键字“final”来修饰常量值。另外，使用关键字“auto”也可以实现类型推断。

```java
final int a = 10; //a是一个常量值
auto x = "hello"; //x是一个字符串类型
final auto y = new ArrayList<>(); //y是一个ArrayList对象
```

Java除了基本数据类型，还包括数组、枚举、类、接口、注解、泛型等数据类型。

## 运算符

Java语言共有以下运算符：

1.算术运算符：加号“+”，减号“-”，乘号“*”，除号“/”，模运算符“%”，自增“++”，自减“--”。
2.关系运算符：等于号“==”，不等于号“!=”，大于号“>”，小于号“<”，大于等于号“>=”，小于等于号“<=”。
3.逻辑运算符：与(&&)，或(||)，非(!)。
4.位运算符：按位与(&)，按位或(|)，按位异或(^)，按位取反(~)以及左移(<<)右移(>>)运算符。
5.条件运算符：三元运算符“? :”。
6.赋值运算符：简单的赋值(=)，组合赋值（+=,-=,*=,/=,%=,&=,|=,^=）以及复合赋值运算符（+=,-=,*=,/=,%=,&=,|=,^=）
