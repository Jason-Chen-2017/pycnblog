# 1.背景介绍


## 什么是Patent？
Patent（专利），是一种权利的一种法律术语，指国际上或某个国家颁布、同意实施或许可使用某种专利方法、设备或过程的一种权力，其目的在于保护创造者及他人的版权和专利权，并防止其他人复制、仿造、盗用和侵犯版权或专利权所得的经济利益或不利影响。
一般来说，每一个专利申请都包括两个文件：专利申请书和专利诉讼状。其中，申请书主要负责描述被发明的新产品、新工艺或者新技术，并且需要提交到专利局评审认证；而诉讼状则承担起对专利权的原告之间的争议解决问题、调解纠纷的作用。因此，专利的申请、撤销、变更等一系列环节都是由专利部门进行仲裁调查和管理。
除了专利申请外，还有一种被称为商标的类似权利。它同样具有版权意义，但拥有者为企业或组织，并且可以用于商业活动中的某些方面，例如商标注册、商标展示、广告宣传等。商标的申请、撤销等也是由专利局进行仲裁调查和管理的。
综上所述，Patent是一个涉及版权与专利权的复杂法律关系，涉及政府部门的职能部门划分，及相关部门制度安排、监管措施等情况。本文所涉及到的知识自动化领域的自动提取与知识抽取技术，对于该领域的研究、应用和发展都具有重要的参考价值。
## 为什么要做Patent知识抽取？
随着互联网的普及、科技的迅速发展、金融的崛起，很多企业通过网络向全世界推出产品、服务和公司。这些产品、服务和公司往往都带来巨大的市场营收和社会效益。然而，如今来看，由于专利法的存在，企业、团体和个人之间因“版权”或“专利”问题引起的纠纷频发。一旦出现纠纷，纳税损失高昂且难以弥补，给企业带来的经济损失更是难以估量。因此，对于企业而言，如何有效地运用专利法，发展自己的产品或服务，成为一个关键的问题。
为了应对这种专利纠纷问题，本文提出了一个新的知识抽取任务——Patent知识抽取。它的目标就是从专利文本中抽取出其中的知识信息，进而分析、预测和处理这些知识，从而减少纠纷发生、降低纳税损失、促进企业发展。
# 2.核心概念与联系
## 词汇表
- *Patent*：专利，也叫做“无形资产”。是一种权利的一种法律术语，指国际上或某个国家颁布、同意实施或许可使用某种专利方法、设备或过程的一种权力。其目的在于保护创造者及他人的版权和专利权，并防止其他人复制、仿造、盗用和侵犯版权或专利权所得的经济利益或不利影响。
- *Knowledge*：有关专利中的知识，通常指与创造性工作有关的知识、理论、原理、方法、工艺、技能、教授、专家、经验和经历，以及新技术、产品和服务的研制方法、成果及其运用的基础知识、经验、技能、工具等。
- *Extraction*：从专利文本中提取知识的过程。
- *Information*：信息，是指数据的符号表示，在计算机中通常是以二进制或十六进制的方式存储。
- *Machine Learning*：机器学习，是一种统计模式识别技术，是利用计算机及相关算法从数据中发现有意义的模式或规律，并进行预测、决策和分类的一类人工智能的技术。
- *Deep Learning*：深度学习，又称为深层神经网络，是一类用多层神经网络构建深度结构来学习特征、模型和参数的机器学习技术。
- *Classification*：分类，是在给定一组输入的数据集合时，将每个数据项分配给一个离散的输出类别的过程。
- *Prediction*：预测，是指基于已知的输入变量、算法和数据集，对某一未知的数据项产生输出值的过程。
- *Generation*: 生成，指根据既定的模板和语料库生成特定类型文档或片段的能力。

## 技术架构

- 首先，原始的专利文本会转换成可以输入到机器学习模型的训练数据格式。最常见的是 Bag-of-Words 模型，即将所有的词汇按照出现次数进行计数，然后将这些词频向量作为模型的输入。
- 之后，经过特征工程的模型将利用训练数据进行训练，最终得到一个可以对新专利文本进行分类的模型。不同的模型可以采用不同的算法，例如支持向量机、决策树、随机森林等。
- 当模型训练完成后，就可以对任意的新专利文本进行分类，并从中抽取出与创造性工作有关的知识。
- 根据分类结果，可以使用不同的方式处理知识信息，例如基于规则的、基于神经网络的、基于图数据库的等。如果选用基于图数据库的方法，就需要有一个图数据库作为知识存储介质。

## 主要挑战
Patent知识抽取面临的主要挑战有以下几点：
1. 复杂的专利法律关系。由于专利法本身的复杂性，不同人群可能在不同的领域内使用相同的专利，甚至对某一领域使用专利授权另一领域使用。所以，如何准确理解和区分不同专利权的内容和限制，确保知识的准确抽取是一个重要的挑战。
2. 专利语料库的积累。专利知识的真正价值在于能够从这些语料中提炼出共性、关联、概率、原因、机制、性能和规律等关键信息，这需要大量专利知识的积累。当前，只有少量专利作者和专利代理人参与了专利知识的生产，但随着社会的发展，这一现象必将继续扩大。
3. 专利文本的质量。专利文本的质量直接影响着知识的抽取效果，因为专利文本涉及专利权的权利义务关系，如专利法的公平原则、专利权的灵活运用、专利权获得者的权利义务关系等。对专利文本的质量的监控和管理也是一个重要的挑战。
4. 大量的知识的需求。随着社会的发展，更多的人将渴望获得更多的知识，例如技能、工具、经验、经历、技术等。如何让客户快速获得这些信息，并获得满足，是技术的更大挑战。
