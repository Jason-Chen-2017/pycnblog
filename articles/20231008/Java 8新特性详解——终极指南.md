# 1.背景介绍


Java 8（以下简称Java 8）是一个快速的开发、运行和部署现代应用程序的语言。它包含了许多新的功能特性，使得开发者能够编写出更加高效、可读性好并且易于维护的代码。本文将从基础知识开始，全面地讲解Java 8新特性，包括Lambda表达式、Stream API、Optional类等最值得关注的特性。同时还会对比Java 7和Java 8之间的差异，提供建议帮助读者进行决定。

## Java 8为什么突然火爆？

2014年10月23日，OpenJDK项目发布了Java SE 8。作为Java SE 系列的最新版本，Java 8带来了众多新特性，其中包括Lambda表达式、Stream API、函数式接口、接口默认方法、Date/Time API、注解处理器等。这些特性不仅对Java开发者来说非常实用，而且在企业级开发中也扮演着重要角色。因此，Java 8迅速成为主流开发语言，其热度不断上升。

Java 8的发布促使一批技术人员纷纷涌向Java开发领域，他们有的热衷于研究Java 8新特性，有的开始使用Java 8开发应用程序；而另一些技术人员则抱持着更加保守的态度，等待Java 8发布之后再去尝试使用新特性。本文将从两个角度分析Java 8为什么如此火爆：其一，Java 8所带来的革命性变革如何激发起众多热衷于探索新特性的技术人才；其二，通过对比Java 7和Java 8之间的差异，为读者提供参考建议，帮助读者做出适合自己的判断。

## 为什么需要学习Java 8？

Java 8为Java开发人员提供了多个新特性，比如Lambda表达式、Stream API、Function接口、Date/Time API、注解处理器等，这些特性能够提升编程效率并降低代码复杂度。本文重点讨论的就是Java 8的新特性，但除了学习Java 8之外，理解它们背后的原理也是很有必要的。学习Java 8后，可以掌握Java 8的一些核心机制及其实现方式，更好地理解各种设计模式、工具类及框架的运作机制。所以，无论读者是否准备好完全投入到Java 8的学习和应用中，都应该花一些时间来阅读、理解Java 8的新特性，至少能明白它们为什么这么火爆、解决了哪些实际问题、又有哪些未来趋势。

## 如何学习Java 8？

想要全面地学习Java 8，建议可以按照下面的学习路线图进行：

- 首先，通读Java 8官方文档中的“New and Noteworthy”章节，了解Java 8新特性的概览、使用方法、设计思想及其背后的原理。
- 然后，结合Java 8教程、书籍或视频课程，系统地学习各个新特性。这里推荐《Java 8 in Action》一书（英文版）。
- 如果希望进一步深入学习Java 8，也可以查看官方示例代码，理解各种新特性的实际应用场景。
- 在学习过程中，记得留心一些常见问题的解答。一般来说，Java 8的相关问题可以在stackoverflow.com上搜索并找到答案。
- 浏览过一遍Java 8的新特性之后，就可以选择性地学习Java 8的一些扩展库，比如Reactive Streams规范、Java 9 modularity、并发改进、JVM性能优化等。
- 当然，还有其他很多其它方面需要学习，不过这一切都是建立在基础知识的基础上的。
