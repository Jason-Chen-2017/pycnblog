# 1.背景介绍


“工作”这个词语的创造者伽拉·莱恩·科斯勒在其著作《莫顿问答》中提到：“工作只不过是人类为了生活而做的一系列机械活动。”但是随着生产力的发展，工厂、办公室、学校等日常的工作也越来越少，成为一种“消磨时间”的“游戏”，已经不再是一个富足的生存方式。现代社会所拥有的精神财产已经远超“奢侈消费”的物质层面。因此，“工作”的定义已经发生了变化。

既然“工作”已经成为一种消耗资源的行为，那么人们应该如何度过自己的余暇时光呢？所以有必要重新审视“工作”这个概念，真正意义上的工作应该是生活的另一种形式。

在新的工作观念下，“工作”的目标从“劳动”转向了“玩耍”，“娱乐”。个人的时间应该从工作中拿出更大的份额来沉浸在各种各样的娱乐活动之中。在这样的观念下，“工作”变成了一个工具，可以帮助人们更好地享受生活，放弃单调乏味的工作。而“游戏”和“消遣”的定义也都可以在这样的视角下重新界定。

在这个阶段，随着互联网、智能手机的普及和发展，人们越来越容易满足于智能化产品，“上班”成为了人们“刷微博”、“看短视频”、“打游戏”的重要方式。“工作”逐渐失去了意义。但是，在这个过程中，还有一些工作相关的工作可以完成，比如：

1. 负责促进内部团队合作和外部沟通
2. 管理组织内的人事关系
3. 招聘、培训新人
4. 处理重大事务
5. 抓住人才需求

而这些工作本身却并不是生活必须的，它们只是个人生活中的小块时间，不需要刻意追求。因此，“工作”的终结就可以延后至“玩耍”阶段，只要把时间花费在自己喜欢的娱乐活动和互联网上，“工作”自然就会变得没有意义。

# 2.核心概念与联系
## 2.1 生活的四个维度
在《The End of Work and the Beginning of Play》的开头，作者讨论了“工作”的三个层次：“职业层次”（professional）、“社会层次”（social）、“个人层次”（personal），并提出了他对“工作”这个概念的理解。这里我将从生活的四个维度出发，谈一下作者认为“工作”应该具备哪些特征。

- 时间维度 - “工作”的主题就是时间，这也是为什么作者说“工作”定义发生变化的原因。工作从需要消耗大量的时间中获得舒适感，消解人类的精力。
- 消息维度 - 在过去的生活里，“工作”被理解为向老板索取报酬，或者用工作来换取福利。但现在，消息和信息正在推动着我们的生活，“工作”的内容正在发生改变。
- 空间维度 - 工作场所成为人们生活不可或缺的一部分。随着城市和国家建设的扩张，我们的生活由很多的地方组成，工作场所将越来越多。
- 价值维度 - 一项工作给予人的什么样的价值呢？我认为，工作除了提供收入外，更多的是让我们有机会认识新的朋友，结交志同道合的伙伴，还可以体验不同的文化、活动。

基于以上四个维度，作者希望“工作”具备以下特性：

1. 更加丰富多彩的生活方式 - 人们现在的生活已经离不开工作。但是，除了职场、家庭和日常的生活之外，人们又渐渐发现了其他维度上的工作，比如体育运动、音乐表演、绘画、编程、设计、旅游、摄影等。
2. 个性化的生活建议 - 不管你是在干什么领域，“工作”其实都是个人对自己的一种服务。你可以主动发起协作项目、寻找志同道合的伙伴，也可以利用“工作”带给你的特权、帮助解决难题。
3. 全面的知识结构 - 在这个知识爆炸的时代，工作只是用来完成任务，而且收益极低。在如今的时代，知识无处不在，你可以找到全方位的学习资源。工作中最宝贵的可能就是独处的时间，你可以从书籍、播客、文档、课程、讲座等获取知识。
4. 艺术化的工作环境 - 许多优秀的设计师、建筑师、音乐ians、科技记者、艺术评论家都会选择从事一些艺术类的工作，比如绘画、雕塑、声乐、舞蹈。你可以把你的才能发挥到极致。
5. 理想中的世界 - 如果人们对工作和工作中获得的价值的期望达到了一个新高度，那么就一定会看到越来越多的“无聊”工作出现。但如果工作的效率不高，你可能会发现生活无法继续维持，甚至有些时候会陷入倦怠、迷茫和厌倦。因此，要认识到工作并非“目的”而是一种“手段”，不要抗拒、批评它。

最后，我要感谢作者的出色的观点和论述，希望能够引起广泛的思考。
