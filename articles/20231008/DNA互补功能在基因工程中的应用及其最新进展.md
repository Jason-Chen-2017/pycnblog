# 1.背景介绍


## 基因工程
基因工程（Gene Engineering）是利用基因编辑技术对生物体进行改造，改变它的基因组，从而实现人体组织结构的改变、生理特征的改善或疾病的治疗等目的的工程活动。基因工程技术有助于医药制品的研发、卫生保健产品的开发、新型疾病的诊断和治疗、宇航事业的飞行器设计、疫苗的制备、食品科技的生产、航天器建设等领域。
## DNA互补功能
DNA互补功能是指通过基因工程手段将特定基因引起的DNA形态变异，引起特定化合物或蛋白质分子，从而影响生物的正常功能或产生新的功能。DNA互补功能包括两类：基因调控互补（GCI）和轨道调控互补（TCI）。其中，基因调控互补是指影响DNA的三联胞结构中某一特定位置上的核苷酸的基序与一定程度上相互配合。该位置可以是基因组某个特定区域，也可以是靶向或者目标性基因组的特定区域；轨道调控互补则是指通过基因工程技术，改变特定染色体的染色质结构，使该染色体能够与其他染色体的特定区域的染色质序列相互作用。目前，基于DNA互补功能的基因工程技术已成为临床诊断和医学研究的重要手段。
# 2.核心概念与联系
## 二级结构

二级结构是指蛋白质的主要三维结构，包括核苷酸的排列方式以及氨基酸与蛋白质间的联系。二级结构的关键是氨基酸的排列顺序，即链主元结构。蛋白质结构由四个阶段组成：α 链，β 链，3D 槽，和键区。其中，α 和 β 链之间的核酸距离为 30~50 碱基，而 3D 槽之间的核酸间距则更长，通常为 60 ~ 80 碱基。二级结构存在许多循环区域，如启动子、单复聚体、α 和 β 链等，它们共同作用构成蛋白质结构。
## 多肽结构

多肽结构（polysome structure），也称为杂合二级结构（multimers of secondary structures），是指一种在某些基因组中存在的亚型、编码区和蛋白质之间的互作关系。它是由多个亚型、编码区和蛋白质组成的一个集合，每个亚型、编码区或蛋白质都由一或几个微小的多肽（polysome）包围着。多肽结构发生于不同亚型、编码区和蛋白质之间的相互作用。多肽结构使得同一个蛋白质具有不同表达状态和功能。多肽结构对基因工程技术的研究非常重要。例如，多肽结构可用于研究异源之间的基因表达调节，或对疾病的耐药性进行研究。
