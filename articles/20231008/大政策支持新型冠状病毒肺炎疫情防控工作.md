# 1.背景介绍


## 1.1 新型冠状病毒(COVID-19)简介
新型冠状病毒（Coronavirus）是一种致命的病毒。它是一种高等单细胞免疫系统（HCoV）的最初宿主，主要感染人类鼻咽喉呼吸道，继而造成了全球性流行。目前已知有两个亚型的COVID-19，即2019-nCov和SARS-CoV-2。
截至2020年4月7日，我国累计报告新型冠状病毒肺炎确诊病例已达到23528例，疑似病例13334例。当地卫生部门公布数据显示，我国境外输入的病例中，境外输入的病例86%为武汉华南海鲜城市输入的病例。
## 1.2 2019-nCov新冠病毒对中国社会产生的影响
2019-nCov在中国造成严重的经济损失、人员伤亡和生命财产损失，对国家和民族都产生了巨大的灾难性影响。从2020年初开始，社会各阶层对疫情反应强烈，开始思考和行动。2020年3月1日，习近平总书记在政府工作报告中提出“新冠肺炎疫情防控工作要坚定不移坚持党中央决策部署，统筹疫情防控和经济社会发展”等重要指示。为此，国家主管部门开展了包括疫苗接种、社区管理、灾后恢复重建等方面的政策举措，并对外发布了《新冠肺炎疫情防控工作要点》等文件。

2020年4月1日，四川省遂宁市政府、成都市政府、贵州省黔南州市政府等共同印发了《关于加强新型冠状病毒感染的肺炎疫情防控工作实施方案》，其提出的实施思路主要围绕以下几个方面：

1. 坚持科学防控：从建立健全疫情预警机制、建立信息共享机制、建立基本医疗物资保障制度、确保公共卫生资源充足等角度，严格落实和执行规范要求；

2. 深化卫生保健体系建设：围绕新冠肺炎疫情防控联合抗击肺炎战役、全面推进基础医学教育和重点精准医学工程、重大公共卫生事件应急响应等领域，不断完善公共卫生资源；

3. 提升科技水平：通过加强相关技术研发、新药研发和工程投入，加快推广新药和疫苗，形成有效的医疗救治能力；

4. 优化生活方式：着力改善人们日常生活方式，促使群众更多参与公共卫生活动，做到人人都是吃得起热闹的食客；

5. 构建服务体系：通过建立感染科普、防疫知识普及、保险助学金等多渠道服务，提供专业的防疫服务，满足人民群众的自我保护需求。

经过十八届三中全会后，习近平总书记高度重视疫情防控工作，亲自指导并指挥地方各级政府加强疫情防控工作，提出“各项措施均应立足当前，统筹考虑全局、协调应对，切实保障人民群众生命安全”，并明确了“坚持稳中求进取、精准防控”的方针。

## 1.3 SARS-CoV-2新冠病毒对世界其他国家的影响
### 1.3.1 欧洲
欧洲疫情爆发时期，西班牙曾被称为“黑马之都”。在该国发生大规模的感染事件，促使欧盟政府采取不同措施，以减缓疫情蔓延。2003年，欧盟正式批准了SARS病毒的全球免疫接种计划，成为国际上最早进行免疫接种的人类免疫病毒研究所。截止2020年3月31日，欧洲累计报告新冠病毒感染者已超过58万例，占全球感染人数的近七分之一。美国也接连发生多起新冠肺炎病例。

### 1.3.2 中东
中东（阿拉伯半岛，意译：巴勒斯坦）在SARS疫情爆发时期，由于受到严重饥饿、贫困和戴口罩的影响，导致死亡率激增。然而，随着世代人的文明程度越来越高，人口向北迁移，生活条件得到改善，因此中东的疫情出现了转折点。2003年，根据临时协议，美国与巴林签订了一系列援助协议，帮助巴勒斯坦人民接种SARS病毒。2004年9月，苏联官方宣布解散所有由世界银行资助的用于抗击和消除SARS的基金项目，并将资金移交给其他机构。2004年11月，世卫组织接到联合国指派任务，确定于11月13日进行中东集团峰会。2005年6月17日，联合国秘书长古特雷姆在首次联合国大会上演讲时表示，“如果没有共同努力，我们将无法战胜病毒”。2005年11月，苏联政府决定与哈萨克斯坦、阿富汗、塔吉克斯坦、土库曼斯坦、叙利亚、伊朗、科威特、沙特和委内瑞拉组建新型冠状病毒防疫合作网络（PADON）。2006年5月，苏联举行金融危机，直接经济损失超过200亿美元。尽管国际上曾有多轮谴责和禁令，但2006年11月11日，联合国秘书长古特雷姆宣布：“通过在中东的工作，我们证明了世界范围内共同的力量。”之后，多个国家相继加入了该网络，并承诺每年向新冠肺炎病毒免费提供经费。

### 1.3.3 非洲
非洲国家中的塞尔维亚、赞比亚和津巴布韦在1993年爆发并持续了十几年的SARS疫情，虽然在当地得到了控制，但是伴随着各方面因素的变化，2020年6月，仍有少数国家出现了人传人现象。在这些国家中，新冠病毒感染者人数逐步增加。

### 1.3.4 亚太地区
在亚太地区的多个国家，已经出现了广泛流行的SARS-CoV-2。例如，日本在2003年爆发SARS后，大规模感染导致国内死亡人数激增。日本政府表示，当前局势危急，正在积极应对，严守国门，保持正常的商业运营。然而，由于日本人口结构和资源有限，长期的医疗设施建设尚不足，因此当前无法很好地处理各种人群的感染。另外，日本政治文化的复杂性以及高度依赖外汇储备、国际贸易和金融的国家形势，也使得日本政府陷入了巨额财政压力。此外，日本有大量的人口居住在偏远地区，且在财政收入水平较低的情况下，这些地区的防疫措施也面临着巨大挑战。
