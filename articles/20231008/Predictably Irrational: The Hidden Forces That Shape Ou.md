# 1.背景介绍


人类一直以来都生活在一个信息时代，我们的一举一动、所接受的信息的多少与质量，影响着我们的行为。越来越多的人开始从各种渠道获取大量的资讯信息，不管是新闻、期刊、网络，还是各大平台上的社交媒体内容，都极其丰富、精准。过去几年，随着科技进步和互联网的飞速发展，人们对信息的收集、整理、处理、利用已经成为一门独立的技术。而由于个人因素的干扰，导致了人类的各种偏执性思维、认知和行为。这本书是一部解释人类为什么会有某些偏执的书籍。

在这本书中，作者Richard Gere尝试从人类社会的角度分析人的心理和行为背后的原因，探讨了让人感到有些“难以置信”的现象背后隐藏着的复杂的机制。他认为当前流行的一些看起来很简单或理所当然的做法背后，其实蕴藏着深刻的意义。

作者试图通过研究人的各种决策和行为，揭示出让人不安、甚至焦虑的那些心理现象背后，所隐藏的支配自我的想法和心理特质。这本书能够给读者提供一系列有关人性的真知灼见。

无论是医生还是工程师，都应该阅读这本书，并了解自己的决定背后，正在发生什么样的影响。这可以帮助我们更好的预测自己的行为，并且更加充分地利用我们的优势资源。

2.核心概念与联系
# 解决问题：当面临困境时，我们如何找到解决之道？—— 这是Gere提出的第一个观点，这个观点是建立在前面的三个观点之上。它强调人类是为了解决各种问题而诞生的，因此只有正确的预判，才能找准问题的本质，然后提升我们的能力。只有这种预判才能找到适合自己的解决办法。

# 目的性：为何要做某事或者做某种行为？—— Gere认为这是影响我们判断行为的第一个机制。决定权力的权利，最终取决于我们为实现某个目标而付出的努力和认识。

# 基于直觉的行为模式：通过直觉来判断行为的模式。—— 在大多数情况下，人类的决策是由内在的直觉驱动的。这也是为什么很多人在没有被明确指导的时候，表现出迷失方向、缺乏自控力的行为。

# 情绪效应：情绪的产生和持续性会影响人的决策。—— Gere认为情绪是一个重要的因素，因为情绪会影响人的决策过程，并导致人们出现不理智、冲动等行为。

# 感知-决策联合效应：通过两个相互作用系统之间的相互作用，人类就能形成有效的决策。—— 感知-决策的连贯性可以促使人们在任何情况下作出最佳的决策。

# 认知惩罚机制：人们往往会在错误的决策下受到惩罚。—— 作为一种心理现象，这种机制也存在于社会生活中。犯错、违反规则都会带来惩罚，但其结果可能不同。

3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
# 模型1：潜意识中的信息过滤器
潜意识中的信息过滤器：人们往往喜欢阅读经典小说，他们会把自己置身于虚幻世界中，过滤掉非虚构的内容，只留下精神层面的东西。同样，许多人在潜意识里就有信息过滤器。这个信息过滤器的工作原理是，在我们看似专注于一件事情的同时，忽略其他的杂音，只关注与任务相关的讯息。Gere把这个模型命名为“潜意识中的信息过滤器”。

# 模型2：梦境中的快乐幻想
梦境中的快乐幻想：在梦境中，我们往往会发现自己的童年故事、亲情怀旧、友情浓郁等情绪的回忆。作者认为，这些情绪的产生往往源自于人们对过去的回忆。梦境中的快乐幻想并不是偶然的，而是有意识地通过我们与过去的关联来形成的。Gere把这个模型称为“梦境中的快乐幻想”。

# 模型3：系统1（激活）和系统2（防御）
系统1（激活）和系统2（防御）：当我们遇到重要的事情时，我们首先会倾向于做出积极的反应。但当情况变得危急或不再重要时，我们可能会做出消极的反应。Gere把这种现象叫做“激活-防御”系统。

# 模型4：随机化机制
随机化机制：一项事件发生的概率并不会完全等于1，因此，我们经常无法完全预料所有可能的结果。随机化机制通过模拟事件的不确定性，来解决这一问题。Gere把这种机制称为“随机化机制”。

# 模型5：长期记忆
长期记忆：人类学习到的知识和经验，往往比我们自己所知的要久远。古埃及人就曾经有这样的记忆。Gere把这种记忆叫做“长期记忆”。

# 模型6：倒置的解决方案
倒置的解决方案：由于人类习惯于从结果的角度看待问题，因此，解决问题往往从反方向进行，即先考虑问题的原因。Gere将这种方式称为“倒置的解决方案”。

4.具体代码实例和详细解释说明
# 实例1：亚历山大·欧文的决策模型
亚历山大·欧文是一位美国电气工程师，他曾经作为一名项目经理参加过工程竞赛，被人誉为“世界第一创业家”。经过两次比较艰苦的竞争，他最终获得成功。但是，在获胜之后，他却非常焦虑，担心自己未来的命运。他并不知道应该怎么办才好，于是他开始思考。他开了一个头脑风暴，列出了他要做哪些决策，然后列出了他以往的成功经验。但是，这种方法太过简化，无法真正评估他对每一个决策的决定性影响。所以，他邀请了卡罗尔·马克斯（Kam Karst）来给他一个更为科学的决策模型。

经过仔细研究，亚历山大·欧文发现，他的决策模型可以分为四个部分。第一部分是看清世界和环境的因素。第二部分是突破局限，开拓新的市场。第三部分是尊重大众的意见。第四部分是保持开放的心态，创造共赢的局面。Gere使用这个模型分析亚历山大·欧文的决策，发现他的行为具有高度的随机性，并且无法预测他将面临哪种结果。

# 实例2：迈克尔·杰克逊的抗焦虑心理学实验
迈克尔·杰克逊是一位培训师，他的一天通常分为五个阶段。第一阶段是学习，他会听讲座、上课，阅读有关他所学领域的书籍。第二阶段是探索，他会出外旅游、参加研讨会、钓鱼等活动，以寻求不同的刺激。第三阶段是计划，他会制定日程表、规划行动方案。第四阶段是执行，他会按照计划进行行动，并且不断调整策略以达到更高的效益。最后一阶段是总结，他会对自己过去一段时间的表现进行总结、反思，并设法改善他的工作方式。但是，这些都是理性且有计划的活动。在第四阶段，迈克尔·杰克逊的头脑有时候出现空白状况，这让他十分焦虑，担心自己还是否能够正常工作。

为了缓解这种心理障碍，迈克尔·杰克逊找了两个心理学家帮忙。第一个心理学家叫做卡莫拉·海塞尔，他告诉迈克尔，如果他不想被伤害，那么可以通过“抵制行为”来控制自己的情绪。第二个心理学家叫做尼古拉斯·塔勒布，他建议他可以暂时冷静下来，从容面对自己的挣扎。在迈克尔·杰克逊坚持完这个实验后，他感觉情绪平复了许多，而且也终于安心工作了。
