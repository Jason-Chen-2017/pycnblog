# 1.背景介绍


英国的经济学家、分析师、首席执行官布隆伯格(<NAME>)曾经说过，美国股市的最重要特征就是其无固定利率，即使在崩盘时也会随之下跌。在这个背景下，很多公司都做出了不同程度的调整，比如今年四月份的特斯拉事件后，特朗普政府连续施压推动电动车和汽车部门缓慢刹住成本，防止过度消费。而另一些公司则选择大幅提升利率进行回购或增持。因此，单从财报上看，国内的互联网公司总体来说更偏向于保持低利率，这也许是他们在降低房租、压低污染、减少用工、改善健康状况等方面付出的代价。当然，大多数公司都会维持一定的盈利水平，并在此基础上通过提高股票质押贷款和投资策略来保值。

但是，据统计数据显示，在过去的十五到二十年间，美国股市的平均收益率(即复合年化率)一直不断下降，这主要归功于新兴市场投资者对全球经济危机的担忧和技术革命带来的投资需求增加。近几年，由于对疫情等各种原因的抵制，人们对于商业周期和经济衰退的预期似乎越来越低，导致股市的下跌加剧。因此，Bloomberg LP希望通过一篇深度技术专题文章，更全面地研究美国股市的结构、政策及其他经济现象，进而提供科技行业更多参考借鉴。

# 2.核心概念与联系
## 2.1.市场结构
美国股市由大量的上市公司组成，它们分为两种——受控公司(concentrated-income companies)，即拥有超过50%以上股权的上市公司；非受控公司(non-concentrated-income companies)，即没有这种特点的上市公司。一般而言，非受控公司占主流公司的70%左右，这主要是因为这些公司已经成为巨头，缺乏竞争力。相比之下，受控公司仍然在上市公司的数量排名榜上，但它们的总规模却远远小于非受控公司。

受控公司与非受控公司的区别在于，前者通常是政府和金融机构支持的大型公共企业，它们的持续盈利能力依赖于长期积累的资本积累，是股市“雄心勃勃”的一部分，因此往往具有更好的技术实力和管理层资源。非受控公司则由个体企业主创立，其创始人往往受雇于政府或同行，其规模较小，但资金来源多为个人或初创公司，往往具有更强的技术基础和创新能力。

## 2.2.利率市场
美国的利率市场是一个复杂的整体，包含三个主要领域：货币市场，债券市场和股市。货币市场主要包括政府与央行的货币政策，以及债券市场的发行与兑换。债券市场的主要参与者包括企业、个人和金融机构。股市的主要参与者包括保险公司、基金经纪人、私募股权基金、交易所、证券公司以及媒体。

### 2.2.1.货币市场
美国货币市场主要由美国央行和各州央行共同主导，存在着严厉的通胀、汇率升值等风险。政府并不是绝对的货币政策制定者，实际上有些央行会依赖于居民消费物价指数(CPI)、工业生产、就业情况等因素来控制通胀率。同时，国内外的金融机构也都会试图通过融资来吸引投资者，但最终还是需要央行根据市场供需决定利率水平。

虽然美国央行长期被认为是公正的主导者，但其背后的政治人物仍然可能影响利率政策，例如希拉里·克林顿、约翰·麦凯恩以及唐纳德·川普。另外，各州央行也可能会试图影响美国的利率，例如俄亥俄州的州长、肯塔基州的国税局主任以及亚利桑那州的联邦储备主管。不过，与中国一样，美国利率并不能完全解决经济危机带来的通胀问题，需要国家采取更有效的货币政策手段来缩小市场的波动空间。

### 2.2.2.债券市场
债券市场主要涉及到国债、商品债和企业债。国债是美国政府发行的一种短期债务，兑付通常比较简单。商品债则是在实体经济活动中产生的，由银行和债券商发行，有较长的存续期限。企业债是企业发行的长期债务，兑付时需要由法律授权的代表性公司来承销。

央行为了调控金融市场，会采用不同的方式发行债券，比如利率加息、信用扩张和货币宽松。不同政策制度下的债券价格往往有显著差异。央行对债券市场的调控不应该被视为一种简单的政策措施，它还可以影响债券市场本身，例如通过出台新的债券限制、调低债券价格等方式影响债券的规模、质量和流动性。

### 2.2.3.股市
股市由大量的上市公司、基金和交易平台组成，其中上市公司是股市的主要参与者，占美国股市总体资产的三分之二。目前，股票市场的参与者包括保险公司、基金经纪人、私募股权基金、交易所、证券公司以及媒体。股市的主要策略有股票期权、债券配售、股指期货交易、涨跌停板、股票衍生品交易以及股票期权市场。

每一个股票都有自己的利润空间，而美国股市有两个主要市场——沪股和深股，两者之间的价格差距在一定范围内波动。以两年期为例，在沪深两市分别发布的股票报价中，期望收益率差距在百分位范围内可达15%，超过20%的公司基本上不会进入市场。

与中国类似，美国股市也存在着种种不确定性，包括宏观经济状况、房地产开发、证券市场及其走势、司法系统、政客的操弄等。正如中国股市一样，股市也是政策的驱动力，而政策的改变往往会引起市场的转变。
