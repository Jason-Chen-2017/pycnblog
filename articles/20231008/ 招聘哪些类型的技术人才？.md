# 1.背景介绍


## 互联网行业信息化与IT服务方向
互联网行业正经历着信息化、IT服务、数字化转型、大数据应用等新形势下的变革。如何找到符合本职工作要求、具备技能优势、具有创新能力的一批“合格”技术人才，是当前 IT 技术领域招聘的一大难题。针对此类需求，2017 年 10 月，中国电信集团-世纪互联发布了《IT 技术人才管理规定》（以下简称“规定”），对 IT 技术人才的定位、岗位描述、薪酬待遇、培训方式、职业生涯发展等方面作出具体规定。
根据规定，IT 技术人才分为四种类型：
- 系统工程师：负责公司内部所有信息系统的设计、开发及维护；
- Web前端工程师：负责网站页面的前端设计、开发及维护；
- 数据库工程师：负责数据库的结构设计、性能调优、查询优化及安全性控制；
- 软件工程师：负责软件模块的设计、实现及测试等工作，包括但不限于开发人员、测试人员、分析师等；
按照规定的设立要求，各类技术人才的身高、体重、年龄、经验、技能要求、个人品质、学历要求、证书要求、工作地点要求等都有明确的要求。由于不同部门的工作特点、岗位职责不同，IT 技术人才的掌握技术栈也不同。例如，Web 前端工程师主要担任 UI/UE 视觉设计，兼顾 HTML、CSS、JavaScript 等前端技术；数据库工程师则需要熟练掌握 SQL 语言，在运维、测试、部署、优化中扮演重要角色；软件工程师除了需要全面的编程能力外，还需具有良好的逻辑思维、团队协作精神、客户服务意识等软实力。这些都是技术人才真正具备的基本素养和能力。
2.核心概念与联系
IT 技术人才是指具有计算机相关专业知识、技术能力及工作经验的专业技术人员。本文所提到的各种技术人才就是指 IT 技术人才的具体分类。IT 技术人才的定位与职能都比较明确，职业路径、技能要求也基本可排除一些。实际上，IT 技术人的要求一般相对简单，但也包含非常多的复杂因素。因此，理解并掌握 IT 技术人才的职场生态与职位要求至关重要。
了解 IT 技术人才的职业生态和发展方向，可以帮助我们更好地理解他们的才能水平、适用场景，进而决定是否适合某个岗位。同时，了解其所属部门或企业的人脉资源、职场氛围等，也可以给我们提供更加丰富的信息。

## 数据分析师
数据分析师的主要职责是进行数据分析和挖掘，为公司决策制定提供有效的信息。他们需要能够处理复杂的数据、具有敏锐的观察力、富有洞察力和团队精神。做好数据分析师的职业发展道路，还需要扎实的计算机、统计学基础知识、数学功底和分析能力。
数据分析师的求职路线图可以概括为：分析需求 > 概念和原理 > 技术框架 > 编程语言 > 技术能力 > 综合素质。
1.分析需求：首先，你要分析清楚自己希望从事何种类型的工作，有哪些目标，并清晰地阐述需求文档。比如，你的要求可能是完成一个报告、设计一款应用、建设一个数据仓库，或者参与到一个数据科学项目中。然后，你应该把目标、关键结果、衡量标准以及执行时间列入考虑。确定好自己的需求后，就可以着手准备“数据分析师”这个职业的考核试题了。

2.概念和原理：数据分析师通过对数据的观察、收集和分析，不仅可以发现数据中的价值，还可以用于改善业务和产品，促进商业机会的产生。数据分析师的工作内容涵盖了数据采集、存储、管理、处理、分析、可视化等环节，因此，要深刻理解数据分析师的工作原理。常用的概念如数据挖掘、信息检索、机器学习、大数据分析等，更是数据分析师不可替代的工具。你可以阅读一些数据分析师专业词汇的解释或定义，把这些概念牢记心中。

3.技术框架：数据分析师通常采用专门的工具或平台来分析数据，例如 Excel 或 Tableau、Python、R 等。它们提供了丰富的数据分析功能，可满足不同场景的需求。选择最适合自己的工具，掌握它的使用方法，就能很好地解决数据分析任务。

4.编程语言：数据分析师通常需要掌握多个编程语言，比如 Python、Java、SQL 和 R，才能将所学的内容运用到实际生产环境中。掌握至少一种主流语言（比如 Python）即可，其他语言的熟练程度取决于个人喜好。

5.技术能力：数据分析师需要掌握很多基本的统计学和数学理论知识，包括线性代数、概率论、随机过程、信息论等。这些知识有助于他更准确、快速地分析数据，并得出有意义的结论。

6.综合素质：最后，数据分析师需要具备不断学习、实践、总结的能力。只要坚持学习、思考、输出，数据分析师就会越来越成功。研究表明，数据分析师在人工智能、云计算、大数据时代的崛起，已经成为当下职业热门。随之而来的新兴领域，如人口学、心理学、物理学、生物学等，也会让数据分析师在新的工作岗位中获得更多优势。
