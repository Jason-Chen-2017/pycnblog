
作者：禅与计算机程序设计艺术                    

# 1.简介
  
  
据统计显示，全球有近50万种职业存在，但只有2-3种职业可以被称为“技术类”。那么技术类职位的招聘要求又都有哪些特征呢？本文将从人才需求、技术能力、工作环境三个角度来分析技术类职位的招聘要求。  

首先，我们来看一下什么是“技术类”职业。什么样的事情属于技术类，就要视公司对它的定位、业务模式、产品形态等多方面因素而定。比如在电子行业，汽车制造业、航空运输业和通信设备制造业等就是属于技术类职业；而医疗健康领域则主要包括医生、护士、药剂师等职业。从范围上来说，技术类职业一般分为以下几类：  

- 计算机、网络和系统工程师（Computer Network and System Engineer）  
- 数据分析师（Data Analyst）  
- 数据库管理员（Database Administrator）  
- 前端开发（Front End Developer）  
- 撰写编辑（Copywriter）  
- 网络管理员（Network Administrator）  
- 硬件工程师（Hardware Engineer）  
- 技术经理（Technical Manager）  
- Web开发（Web Developer）  
- IT项目管理（IT Project Manager）  
- 云计算工程师（Cloud Computing Engineer）  
- 测试工程师（Test Engineer）  
- 嵌入式工程师（Embedded Systems Engineer）  
- 软件工程师（Software Engineer）  
- AI算法工程师（AI Algorithm Engineer）  

在这些职业中，有很多要求的相同点，比如工作年限、学历要求、工作地点、薪酬待遇等。然而，也存在着一些差异。比如前端开发工程师通常会有HTML、CSS、JavaScript基础要求，掌握一定的计算机应用基础知识。后端开发工程师需要熟悉至少一种编程语言，如Java或Python，了解数据库模型及SQL语法。数据库管理员需要理解数据库结构，并熟练掌握SQL语言。同时，还有一些职业只是为了加分，如云计算工程师、AI算法工程师等，职位名称里面的关键字往往暗示着职业所需技能。总之，“技术类”职位的招聘要求都比较复杂，涵盖了软实力、硬实力、语言和工具、平台、工作方式等多个维度。不同职业对候选人的要求也是不同的。这些复杂性带来的一个结果就是，即使求职者表现出较高的综合素质，也很难保证得到所有技能均达标的好工作机会。因此，技术类职位的招聘者应当对自己的综合素质要求更高一些，在确保能够胜任的前提下，选择最适合自身需求的职业。  

# 2. 人才需求  
## 2.1 学历要求  
技术类职业普遍需要大专以上学历，理论基础扎实。本科及以上学历表示申请者在理论水平方面有自己独到的见解。硕士及以上学历则需要更高的专业能力。同时，技术类的职位越高级，对应的技术能力要求也越高。例如，计算机科学专业的学生需要熟悉网络安全、分布式计算、分布式存储等方面的研究。因此，如果你的目标是技术类职业，学历要求的最低要求应该是本科学历。另外，对于一些高级技术人员的招聘，可能会更偏向于学历、成长路径和相关工作经验的匹配程度。  

## 2.2 工作年限要求  
技术类职业的平均工作年限可在2到4年之间，取决于工作职责的复杂性和工作地点的位置。笔者建议公司在面试时提醒候选人不要局限于某个特定的岗位，同时鼓励他们尝试新鲜领域，因为不同职位的要求和发展路径各不相同。另外，不同的企业在技术类职位的要求上往往有所区别，比如有的公司强调技术实践经验、有的公司注重个人技术能力。总之，要根据具体情况确定工作年限的上限值。  

## 2.3 语言要求  
技术类职位的语言要求可分为两种：  

- 英语要求：技术类职位的招聘信息都会提供英语能力测试，要求候选人通过听说读写和表达能力考核。笔者认为这种要求十分必要，因为软件开发与技术解决方案普遍使用英文进行沟通。  
- 中文要求：在部分大型国企，如中国移动、腾讯、阿里巴巴等，技术类职位还会要求候选人拥有中文技能，这是由于技术类职位的竞争激烈，语言上的差异非常重要。除此之外，笔者还建议在设计岗位描述时，把要求清晰、明确，并提供相应的培训班、语言学习课程。  

## 2.4 技能要求  
技术类职位的基本要求是具备基本的计算机技术能力，比如熟练掌握常用的数据结构和算法，以及熟悉常用的操作系统、数据库、网络协议等。在不同技术类职位，还有不同要求的技能。比如前端开发工程师需要掌握HTML/CSS/JavaScript等技术，具备响应式布局、数据交互和动画效果等能力。后端开发工程师需要具备至少一种主流编程语言和数据库技术，如Java、Python、MySQL、PostgreSQL等。数据库管理员需要对关系数据库和SQL语言有一定了解，以及能有效处理复杂查询。这些要求往往与工作职责紧密结合。因此，在面试时，公司可以进一步要求候选人评估自己的技能和经验是否符合要求。  

# 3. 技术能力  
## 3.1 计算机技术能力  
技术类职位要求候选人具有基本的计算机技术能力，包括数据结构、算法、操作系统、数据库等。对于初级职位，只需要候选人基本掌握常用数据结构和算法即可；对于高级职位，可以要求候选人具有更深厚的技术积累，如机器学习、深度学习、图算法、数据挖掘等。除了计算机相关的技能，作为技术经理的候选人还应当具有优秀的团队协作能力、协商和表达能力、分析问题的能力、创新精神等。  

## 3.2 编程语言要求  
不同的技术类职位的要求会有所差异，但基本上都会有至少一种主流编程语言要求。如后端开发工程师必须掌握至少一种主流的脚本语言，如Python、PHP、Ruby等，前端开发工程师则需要掌握HTML/CSS/JavaScript等技术。如果在设计职位描述时，务必为候选人清楚地阐述需求，并指导他们去学习相应的语言。 

## 3.3 平台和工具要求  
技术类职位的平台要求往往与所选职业息息相关，比如前端开发工程师通常需要熟练掌握Web框架如React、Angular等。后端开发工程师往往需要掌握服务器端编程语言如Java、Python等，以及数据库技术如MySQL、MongoDB等。数据库管理员需要了解各种数据库及SQL语句，以及它们的优缺点、性能特性、架构设计等。而移动应用开发工程师往往会要求具有iOS或Android开发的能力，这得益于苹果和谷歌在移动平台领域的持续投入。  

## 3.4 操作系统要求  
技术类职位通常会要求候选人对操作系统、虚拟化、容器化等相关技术有一定了解。操作系统往往直接影响到软件运行的效率，并会影响到系统的稳定性和可用性。虚拟化和容器技术有助于降低软件部署的时间和成本，提升软件的易维护性和扩展性。不过，在设计岗位描述时，可以对候选人描述其在特定操作系统环境下的工作经验，并引导他们学习操作系统相关的技术。  

# 4. 工作环境  
## 4.1 办公环境要求  
技术类职位的办公环境要求往往是最让人头疼的一环。办公环境往往决定着候选人是否能够按时完成任务。比如，后端开发工程师需要有一间独立的办公室，配有显示器、键盘和鼠标；前端开发工程师可能需要一间集中办公室，配有屏幕放映，以及协同办公工具如飞书、钉钉、微信等。另外，要求候选人有良好的社交能力，能够和其他部门的人员建立良好的沟通渠道，避免冲突发生。  

## 4.2 学习能力要求  
技术类职位的学习能力对候选人也是至关重要的。除了刻苦钻研理论外，技术类职位的学习氛围还有另一层意义。对候选人而言，不仅需要对自己的专业知识有极高的理解，更要具备学习能力。面试官常常喜欢问候选人哪些方面能够胜任，这反过来又可以促进候选人对自己的专业知识和技能产生兴趣。另外，还可以通过参加训练班、语言培训课程等方式，帮助候选人在关键时刻快速适应新的工作环境。  

# 5. 未来发展方向与挑战  
技术类职位的发展壮大速度日益加快。随着云计算、大数据、物联网、人工智能等新技术的出现，无论是技术类职位还是非技术类职位都处于爆发期。如果公司愿意接受这样的挑战，它的招聘能力和组织能力就会显著增强。未来，更多的技术类职位都会由机器自动化师、云计算工程师、UI/UX设计师等担任主攻方向。职位的薪酬福利也会呈现出越来越多的差异化，而非技术类职位的薪酬待遇仍然保持低廉。因此，在招聘技术类职位时，公司需要持续投入资源，保持更新的眼光，力争达到前列。
