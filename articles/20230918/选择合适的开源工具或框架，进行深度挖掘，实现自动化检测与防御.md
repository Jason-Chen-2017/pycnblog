# 2.基本概念术语说明
## 2.1 什么是机器学习？为什么要用它？
“机器学习”（英语：Machine Learning）是指计算机根据输入数据（比如图像、文本、音频、视频等），利用算法（如统计建模、强化学习、神经网络等）自我学习，对未知数据进行预测和分析的一门技术。机器学习可以用于监督式学习、无监督式学习、半监督式学习和强化学习，适用于多个领域，包括但不限于图像识别、文本处理、生物信息、自然语言处理等。

用“机器学习”技术主要有如下三个优点：

1. 模型训练效率高

   传统上，人们需要耗费大量的时间和金钱来设计机器学习模型。而借助机器学习，只需提供少量数据，就可以自动地训练出一个模型，大大节省了时间成本。

2. 模型泛化能力强

   通过训练好的模型，可以对未知数据进行预测和分析，极大地提升了模型的泛化能力。换句话说，机器学习模型能够更好地处理和理解复杂的数据，且易于更新和迭代。

3. 可解释性好

   在实际场景中，我们往往需要判断不同模型的结果是否可信。而借助机器学习，我们可以获取到模型内部的参数和结构信息，并对模型进行解释，更容易确定模型的可靠程度。
   
## 2.2 Tensorflow 是什么？
TensorFlow是一个开源的深度学习框架，专注于实现通用型计算图功能。它的特点是简单灵活，可以运行于多种平台上，包括手机、服务器、笔记本电脑、云端等。在处理文本、图像、语音、视频等多种数据时，TensorFlow都非常擅长。除此之外，TensorFlow还有一系列的特性，例如支持分布式计算，异构计算集群，自定义层等等。
## 2.3 Tensorflow Object Detection API 是什么？
Tensorflow Object Detection API (TF-ODAPI) 是 TensorFlow 的官方库，旨在帮助开发者训练、评估和部署目标检测模型。它可以让用户轻松构建、训练并部署基于 TensorFlow 框架的目标检测模型。TF-ODAPI 将深度学习与目标检测技术紧密结合，支持包括物体检测、图像分类和空间框回归在内的多个任务。其包括预训练模型、训练脚本、导出脚本、配置文件以及示例数据集等，使得用户可以很方便地训练、测试并部署自己的目标检测模型。
## 2.4 目标检测
目标检测是指通过计算机视觉技术从图像中检测和识别目标对象并给出相应位置和大小的技术。目标检测的核心功能是从图像中识别出多个感兴趣区域，再将这些区域与已知目标类别进行匹配，最后输出识别出的目标实例及其位置信息。其目的是为了将图像中的目标特征、边界框及其类别信息等提取出来，并能对它们进行有效地组织、管理、存储和检索。

## 2.5 深度学习
深度学习是机器学习的一个分支，由多层神经网络组成，能够实现高度抽象和学习图像、声音、文本等各种数据的特征表示，并最终达到对输入数据的精确分析、推理和预测。深度学习的关键技术包括卷积神经网络、循环神经网络、递归神经网络、自编码器、变压器网络、长短期记忆网络等。

## 2.6 为何要使用 TF-ODAPI 来进行目标检测
目前，目标检测的主流方法主要有两种：基于传统的分类器（如 SVM 和 CNN）和基于检测器的模型（如 Faster RCNN 和 YOLO）。然而，两者都各有利弊。基于传统分类器的方法速度快，但只能识别固定的目标类别；而基于检测器的方法能够识别任意形状和大小的目标，但速度较慢。所以，目标检测领域的研究者们正在寻找一种既能满足需求，又能兼顾准确率、速度和资源消耗的完美方案。

Tensorflow Object Detection API （TF-ODAPI） 正是为了解决这个难题而诞生的。它提供了一个统一的接口，可以应用在不同的任务上，比如图像分类、物体检测、实例分割等。它是一个开源项目，由 Google 提供维护，其项目地址为 https://github.com/tensorflow/models/tree/master/research/object_detection 。

而使用 TF-ODAPI ，你可以：

1. 使用现有的模型进行快速实验，验证你的想法；
2. 用自己的数据训练模型，进一步提升效果；
3. 将模型部署到移动设备、服务器和浏览器上，快速响应用户的需求；
4. 与其他开源项目无缝集成，方便迁移学习和模型组合。
