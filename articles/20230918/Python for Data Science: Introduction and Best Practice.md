# 2.Python基本语法介绍
## 2.1 Python安装及环境配置
首先需要下载并安装Python3.x版本。推荐使用Anaconda，这是一款基于Python的数据科学分析包管理系统，可以轻松安装Python及多个数据科学分析库。Anaconda安装成功后即可打开命令提示符窗口输入python命令，进入交互模式，即输入print("Hello World")打印“Hello World”信息。

```python
conda create -n myenv python=3 # 创建虚拟环境myenv，指定Python版本为3.x
activate myenv # 激活虚拟环境myenv
deactivate # 退出虚拟环境
```
如果已经成功创建了虚拟环境，那么激活环境后，可以通过pip命令安装所需的第三方库，例如numpy、pandas、matplotlib等。如果需要退出虚拟环境，则输入deactivate命令。

接下来，我们用Python解释器自带的turtle模块绘制一只乌龟：

```python
import turtle 

t = turtle.Turtle() 
for i in range(5): 
    t.forward(100)
    t.left(90)
    
turtle.done() # 停顿显示
```

运行该程序后会弹出乌龟画板，然后循环向前走100像素并左转90度，最终停留在原地，并不会消失。可以根据自己的需求调整参数，绘制出不同的图形。
