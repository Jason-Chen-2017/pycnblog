# 1.简介
  


线性代数(Linear Algebra)是一门十分重要的数学学科。它涉及到向量空间、矩阵运算、特征值与特征向量的概念、概率论、离散数学等多个领域。但由于其广泛的应用价值，使得学过线性代数的人员一般都具有极高的求知欲望。因此，本文不打算全面系统地对线性代数做详细阐述，而是着重介绍它的几个核心概念及其在科技行业中的实际应用。

# 2.线性方程组与列空间
## 2.1 线性方程组
对于一个由m个n维列向量组成的系数矩阵A和m个向量b，称A*x=b为线性方程组。若存在n个实数，使得Ax=b，则说方程组AX=B是可解的。方程组中出现的变量称为未知元或未知数。当线性方程组可以唯一确定一组n维向量x时，称线性方机构是可消元的。对于可消元的线性方程组，用列向量表示法，把方程组写成如下形式：

    A * X = B    (1)

其中，A是一个m行n列矩阵，X是一个n行1列矩阵，而B是一个m行1列矩阵。

## 2.2 求解线性方程组的方法

### 2.2.1 初等行变换法

对于线性方程组AX=B，如果存在一系列行交换和列交换，使得A1X=B1且A2X=B2，则称两者等价。初等行变换法就是通过一系列初等行变换，把方程组左侧的矩阵A进行相应的变换，直至将它转换为上三角阵或下三角阵。

设初等行变换后的矩阵R为[I|r]，其中I是单位矩阵，r为任意m行n列矩阵。如果存在n-1个非零元素，使得r第i行对应于r的第i+1列上的元素均为零，则称这个r为满秩矩阵。设A=[A1|A2]，则方程组AX=B等价于方程组ARX=BR，其中R为满秩矩阵。

那么，如何利用初等行变换法求解线性方程组呢？方法如下：

1. 判断A是否可解，即是否存在n个非零元素，使得r的第i行对应于r的第i+1列上的元素均为零。如果不存在这样的n个元素，说明不可解，返回无穷解。
2. 如果存在n个非零元素，则把方程组左侧的矩阵A划分成两个矩阵A1和A2，A1和A2的行数相同，其列数也相同，分别对应于方程组左侧的矩阵A的左上角和右下角。例如，A=[a b;c d;e f]，则A1=[a b];A2=[c d;e f]。
3. 对A1进行初等行变换，直至A1变为上三角矩阵。
4. 把方程组ARX=BR中的A取为A2，把B取为方程组AX=B的右端的值。
5. 对方程组ARX=BR进行初等行变换，直至找到满秩矩阵Q和它的伴随矩阵S，满足AR=QS，此时的Q是一个正交矩阵，S是一个上三角矩阵。
6. 根据满秩矩阵Q和它的伴随矩阵S，即可求出方程组AX=B的解。

注意：初等行变换法得到的解可能不是唯一的，只能保证解存在且是唯一的。

### 2.2.2 高斯消去法

高斯消去法是一种常用的求解线性方程组的方法。该方法是通过对矩阵A进行初等行变换，将矩阵A化为上三角矩阵，然后对矩阵A进行相乘，得到一个上三角矩阵A'。利用初等行变换法，把矩阵A进行相应的变换，直至将它转换为上三角阵，可得：

    AX = B   --->     [ I | r ] * [ x1 ] = [ b1 ]
                     [ 0 | t ]           [ b2 ]
    R*Z + S*T       [ 0 | u ]
                             [ -t'*u / s ]  <-- (1)

其中，A为任意m行n列矩阵，B为任意m行1列矩阵，I为单位矩阵，r为m-1行n列矩阵，t为m-1行k列矩阵，s为非零实数。方程组AX=B可以化为两个方程组R*Z=b1, S*T=-t'*u/s, Z, T 是未知数，r, s是任意的实数值。

对于线性方程组AX=B来说，不一定每次都能够得到满秩矩阵。但是，只要对AX=B使用高斯消元法，就可以保证方程组至少有一个解，并且解存在且唯一。高斯消元法的具体步骤如下：

1. 将方程组AX=B的左边矩阵A看作一个增广矩阵，将右边的矩阵B看作增广矩阵的最右端的一列。
2. 从第1行开始，把每一行的第j个元素看作一个初等因子，如果这一元素的值为0，则不进行任何操作；否则，将这一元素所在的行的所有元素都减去第一个初等因子除以该元素所属的位置的元素，使得该元素的值为1。这样一来，左边矩阵A就变成了上三角矩阵。
3. 在第2步之后，所有第一列的元素都已经变成1，即为单位阵。然后，对每一列的其他元素，进行同样的处理，直至第j列的元素都为0或1。
4. 最后，对第j列的元素进行置换，使得第j列上的所有元素都位于左边的矩阵的第j列。
5. 使用第2步中得到的初等因子，计算方程组的解。

注意：高斯消元法得到的解可能不是唯一的，只能保证解存在且是唯一的。
