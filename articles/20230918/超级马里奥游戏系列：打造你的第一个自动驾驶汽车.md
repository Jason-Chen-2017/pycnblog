# 1.简介
  

## 1.1 什么是自动驾驶汽车？
自动驾驶汽车（Autonomous Vehicle，AV），英文缩写为 AUTOMATED VEHICLE，是指由机器驱动的具有完整自主驾驶能力的车辆。它可以自己决定何时、何地行驶，并能够适应不同的道路条件及环境，实现对人的高度关注。这一切都依赖于激光雷达、巡航雷达、摄像机、激光循迹传感器等传感器的协同工作，通过信号识别系统和信息处理系统，让汽车以最佳方式导航并行驶。2021年，全球销量超过7亿辆，世界领先的自动驾驶汽车制造商包括特斯拉、Tesla、BMW和亚马逊的无人驾驶车队。
## 1.2 为什么要学习自动驾驶？
- 享受生活中不可或缺的一部分——你可以开车骑单车、乘坐公共交通、观看电影、打网球、游泳……
- 可以节省时间——提高工作效率，比如打车、搭地铁、上班、做功课等，都可以自动完成，节省宝贵的时间。
- 有更多的选择——在人口密集地区，随处可见的自动驾驶汽车，将成为生活必需品。
- 安全、舒适、经济——由于汽车本身不得不承载极其重的负荷，因此越来越多的公司和个人，开始着力减少汽车的占用空间，改装为自动驾驶汽车。这也促使各个领域的人们投入到自动驾驶汽车技术的研究开发中来，进而帮助提升社会的福祉。
## 1.3 如何学习自动驾驶？
自主学习的方法有很多种，其中一种就是阅读相关专业书籍和论文，跟着作者一步步的实践，了解自动驾驶的基本原理、关键技术、应用场景，以及各种算法与框架的运作流程。
当然，更加有效的方法是参加线下或者线上的课程教授，大家一起交流、讨论、分享自己的体验和心得，互相学习。
无论采用哪种方法，学习自动驾驶，首先得知道它的历史。也就是说，要先弄清楚自动驾驶汽车的产生背景和定义，以及为什么要发明这一类新型交通工具。
# 2.基本概念术语说明
## 2.1 概念和术语
**定义1**
- **终端控制系统**：终端控制系统（TCS）是一个硬件和软件系统，用于接收车辆控制指令并执行它们。它通常由一组控制器、处理器、内存和通信子系统构成，这些组件分布在多个设备之间，并将指令从一个设备传输至另一个设备。
- **自动驾驶汽车控制系统(ADCS)**：ADCS 是一台由系统工程师、设计人员、编码人员、测试人员以及维护人员组成的集体协作型工作室，负责整个汽车系统的自动化控制，包括雷达、电脑视觉、激光雷达、GPS、行人检测和自适应巡航系统。

**定义2**
- **基于模型的控制**：基于模型的控制（Model-based control）是指使用物理或虚拟模型作为输入，进行系统预测和决策。
- **基于轨迹的控制**：基于轨迹的控制（Trajectory based control）是指根据车辆当前位置和速度、预期目标点、障碍物位置等，计算出下一个目标位置和速度。
- **路径规划**：路径规划（Path planning）是指根据各种约束条件（如车辆状态、驾驶策略、运动限制）以及目标点，输出具有高准确度的合理路径。

**定义3**
- **红外避障**：红外避障（Infrared obstacle avoidance）是指车辆利用红外遮挡传感器来检测障碍物，通过红外发射器发出信号，检测是否有障碍物靠近车辆。
- **超声波避障**：超声波避障（Ultrasonic obstacle avoidance）是指车辆利用超声波传感器来检测障碍物，通过超声波发射器发出信号，检测障碍物距离。
- **巡航模拟器**：巡航模拟器（Path following simulator）是指模拟车辆自主巡航的过程，通过输出模拟数据，模拟车辆的行为。
- **姿态估计**：姿态估计（Pose estimation）是指确定汽车或其他物体在空间中的位姿。
- **深度估计**：深度估计（Depth estimation）是指确定场景中某个点的距离程度。
- **前视摄像头**：前视摄像头（Front camera）是指用来观察车辆前方环境的摄像头。
- **雷达**：雷达（Radar）是一种通过反射式或非反射式的方式来测量距离、方向、速度和其它环境参数的一台装置。

## 2.2 信号及传感器
**信号**
- **CAN信号**（Controller Area Network，控制器局域网）：CAN 网络是一套总线标准，该协议主要用于在车辆之间、计算机与设备之间、现场电源管理设备之间或控制器与其它控制装置之间传递数据。
- **RS-485信号**（Serial Peripheral Interface，串行外围接口）：RS-485 是一种二线制同步传输标准，允许不同芯片之间的串行通信。它可以在短距离（几十米）范围内实现较好的通信性能。

**传感器**
- **摄像头**（Camera）：摄像头是一种能够捕获信息的器件，它由图像传感器、镜头、光学元件、数字处理单元和显示器组成。
- **激光雷达**（Laser radar）：激光雷达（LiDAR）是一种通过在发射一段长宽不等的激光束并接收返回信号来获取三维空间信息的装置。

## 2.3 控制算法
**典型控制算法**
- **PID控制器**（Proportional–Integral–Derivative controller）：PID控制器是一种常用的回环控制算法，它通过积分误差、微分误差和静态误差的相互作用，计算调节输出电压。
- **MIMO（Multiple Input Multiple Output）控制算法**（Multiple-Input Multiple-Output Control System，多输入多输出控制系统）：MIMO 控制系统通常由多个输入/输出（I/O）单元、传感器和控制器模块组合而成。
- **HMM（Hidden Markov Model）建模控制算法**（Hidden Markov Model Based Control System，隐马尔科夫模型控制系统）：HMM 是一种统计方法，可以用来描述和分析随机生成的序列，属于监督学习范畴。

**其他控制算法**
- **最大熵控制算法**（Maximum Entropy Control Algorithm，最大熵控制算法）：最大熵控制算法（MEC）是一种求解控制问题的优化方法，被广泛用于高纬度控制系统的设计与分析。
- **鲁棒控制算法**（Robust Control Algorithm，鲁棒控制算法）：鲁棒控制算法是一种容错控制算法，它通过对控制系统的非线性特性进行分析，设计出一个稳健、可靠的控制系统。
- **最小绝对值法**（Minimum Absolute Value Method，最小绝对值法）：最小绝对值法（SAM）是一种基于相对熵的控制算法，它考虑了系统状态变量与输入变量之间的相互影响，同时适用于多维、非线性系统。
