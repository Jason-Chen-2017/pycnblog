# 2. 实体识别技术简介
实体识别（Entity Recognition）是从文本中抽取出具有意义的实体并进行分类、标记、分析和归类的方法。其主要功能包括信息提取、文本分类、情感分析、事件关系提取等。实体识别是一种自然语言处理中的常用任务，它可以对用户输入的文本进行初步的分析，找出其中实体及相关词汇，用于后续的任务处理。实体识别主要分为命名实体识别（Named Entity Recognition，NER）、指代消歧、共指消岐、语义角色标注、实体关系抽取六个子任务。本节将对实体识别技术进行简单的介绍。
# 命名实体识别(NER)
命名实体识别（Named Entity Recognition，NER），又称为“专名标识”，是指从给定的文本中抽取出具体的实体名称、类型及属性。例如，在一段话中可能包含有人名、地名、机构名等，这些具体的实体会被NER模型识别出来。NER有多种模型，包括规则-based方法、基于统计学习方法、神经网络方法等。其中基于统计学习方法的代表性有CRF、HMM、BiLSTM-CRF等。实体识别的目的是为了方便数据的整理、分析、挖掘。此外，实体识别的结果还可以用来训练机器学习模型对实体的关系进行建模，进而用于后续的任务处理。
# 3.实体识别系统设计原理
实体识别系统一般由实体识别器、特征工程器、分类器、实体关系检测器五个模块组成。它们的工作流程如下图所示：

①实体识别器：是用于从文本中抽取实体的组件，它的输入是一段文本序列，输出是该文本中包含的实体及其对应位置。目前最主流的实体识别器包括CRF、HMM、BiLSTM-CRF等。

②特征工程器：对抽取出的实体进行特征工程处理，生成特征向量表示。特征工程器可以基于规则或统计方法对实体进行自动化的特征提取。

③分类器：对特征向量进行分类，得到每个实体的标签。分类器可以采用SVM、神经网络、决策树等方法。

④实体关系检测器：用于抽取文本中两个实体之间的关系。目前最主流的实体关系抽取器包括基于规则的算法和基于神经网络的算法。
