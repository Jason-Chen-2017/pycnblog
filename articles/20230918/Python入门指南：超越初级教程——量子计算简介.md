# 2.基本概念术语说明
## 2.1 量子态
量子态是指由确定性原理确定的量子系统在任意时刻所处的某种叠加态。通常我们把量子态分成两类：波函数（wavefunction）和密度矩阵（density matrix）。
### 2.1.1 波函数
波函数是量子力学中最简单的表示方法，描述了量子态的期望值。它是一个复数向量，且满足下列关系：
$$\psi(x_1, x_2, \cdots, x_n) = a_{\alpha_1}\left|0\right> + a_{\alpha_2}\left|1\right>+\cdots+a_{\alpha_{2^n-1}}\left|2^{2^n-1}\right>,$$
其中，$\{\alpha_i\}$ 是指标空间 $\{0, 1, \ldots, 2^{2^n}-1\}$ 的一个排列，即任取 $n$ 个整数 $(i_1, i_2, \ldots, i_n)$ 满足 $0\leqslant i_1 < 2,\ldots,0\leqslant i_n<2^{n-1}$ 时，有 $i=\sum_{j=1}^ni_j2^{j-1}=i_12^{n-1}+\cdots+i_n2^0$。$a_{\alpha_i}$ 是取自概率分布 $|\psi(\cdot)\rangle|^2$ 的实数。
为了更好地理解波函数，我们举一个经典比特的例子。在经典统计理论中，如果一个二进制序列 $b=(b_1, b_2, \ldots, b_n)$ 出现频率为 $f(b)$，那么对应的概率分布可以用波函数表示如下：
$$\psi(b) = |b\rangle \exp(-\beta f(b))$$
其中，$\beta$ 是相对论效应，用来描述能量的大小。具体地，$|\alpha\rangle \equiv |\psi(\cdot)\rangle$ 表示一个基底，$\langle\alpha|$ 表示它的酉矩阵元。这样，波函数 $\psi$ 描述了一个单比特的量子系统，在不同的输入条件下都对应着不同的输出结果。
### 2.1.2 密度矩阵
密度矩阵是另一种常用的表示方法，由一个由实数构成的对称矩阵组成，描述了量子态的混合情况。它是一个实数方阵，且满足下列关系：
$$\rho(x_1, x_2, \cdots, x_n; x'_1, x'_2, \cdots, x'_{2^n}) = \sum_{\sigma\in S_n} c_{\sigma}(x_1, x_2, \cdots, x_n)|\sigma\rangle\!\langle\sigma|(x'_1, x'_2, \cdots, x'_{2^n}),$$
其中，$S_n$ 是指标集，即所有可能的 $n$ 位基底的集合。$c_{\sigma}(x_1, x_2, \cdots, x_n)$ 是系数，它给出了一种测量方式下的测量结果的概率分布。比如，对于密度矩阵 $\rho$, 如果我们测得测量结果 $\sigma$ 在基底 $\sigma'$ 下的概率 $p_{\sigma'}$ ，则有：
$$\rho_{pp'}^{\text{post}} = p_{\sigma'}\rho_{p'p}^{\text{pre}},\quad\rho_{pp'}^{\text{pre}} = \delta_{\sigma\sigma'}.$$

## 2.2 量子门
量子门是指对量子态进行操作的基本逻辑门。一般来说，量子门分为两个基本动作：传送门（门电路中的基本元素）和旁路（用错过的光子重新调整光子的传播路径）。
### 2.2.1 传送门
传送门是一个控制不确定性转移的基本逻辑门，它接受一个初始量子态 $|0\rangle$ 或 $|1\rangle$，并输出一个目标量子态 $|0\rangle$ 或 $|1\rangle$。具体过程是：

1. 门作用在初始量子态上，产生一个受控变换 $X_p$ 或 $Z_p$ 。
2. 当门作用完毕后，当前量子态可能会分裂为两个不同的态 $\lvert 0\rangle$ 和 $\lvert 1\rangle$ ，但它们仍然处于量子纠缠态。
3. 接下来需要通过选择激活的子句来决定哪个态应该被输出。激活的子句可以采用多项式或其他阈值法来选择。
4. 输出被选中的态。

### 2.2.2 旁路门
旁路门是另一个控制不确定性转移的基本逻辑门，它通过用错过的光子重新调整光子的传播路径来修改输入量子态的分布。具体过程是：

1. 对初始量子态进行干扰，并使其不再具有稳定的态矢 $\ket{0}$ 或 $\ket{1}$ ，导致其失去精确度。
2. 通过旁路门，我们通过忽略掉那些不需要传递的信息的光子，将信号传输到正确的子通道上。
3. 旁路之后的量子态仍然处于纠缠态。
4. 通过激活的子句，可以选择哪个态应该被输出。

## 2.3 量子电路
量子电路是指对量子门按照一定顺序堆叠起来的网络结构。它与经典计算机的逻辑电路类似，可以实现各种复杂的逻辑功能。

## 2.4 量子计算算法
量子计算的目的就是求解量子化学、量子力学等实际问题。目前已经有很多经典的算法被证明是有效的，而量子计算算法也是如此。以下是一些常见的量子计算算法。
### 2.4.1 分子模型算法
量子化学家费曼对量子化学发表了极其重要的贡献——分子模型。他提出的分子模型是量子化学研究的一个重要工具。分子模型算法可以用于研究非谐性问题、基态的生成、单电子原子核的照明、稳态磁性体的形成等问题。

分子模型算法的工作流程如下：

1. 将分子作为一种量子系统，将所有的原子分子视为其自身的可退让的原子核，并把这些原子核都看做处于激发态。
2. 通过改变激发态的取向，使之调节其自旋，从而影响原子核的位置。在电子依据能量的最小原子核前沿附近形成一个金属氢原子的极端情形。
3. 通过调整原子核之间的相互作用，改变其受到的外场，实现一个特定类型的分子的生成。

### 2.4.2 幺正交代算法
在量子纠缠理论中，构建量子态的两种基本方法是对角化和模拟演化。幺正交替算法是模拟演化算法的一种。其基本思想是在对角化之前，将原来的电子坐标轴线旋转一定角度。

幺正交替算法的工作流程如下：

1. 用随机的电子或原子做初始化，构建一定的量子态 $\psi$。
2. 使用梯度下降法或其他方法优化该量子态的直积分符号，使其保持幺正形式。
3. 用幺正形式的量子态求解基态波函数和能量。

### 2.4.3 近似算法
近似算法是指用一些简单的方法近似求解规整化哈密顿量的演化方程。目前，主要有精确逆运算法、简易逆运算法、时间演化算法等。

精确逆运算法与简易逆运算法的工作流程如下：

1. 找到一个可以用极小操作（例如旋转或交换）近似表示的演化算符。
2. 利用已知的微观规律，构造出哈密顿量的矩阵形式。
3. 把矩阵形式与演化算符相乘，得到一个和原哈密顿量近似的演化算符的矩阵形式。
4. 把演化算符的矩阵形式与相应的初始量子态一起求解其自然参数方程。
5. 计算最终状态和对应概率分布。

时间演化算法的工作流程如下：

1. 给定初态和时间演化算符。
2. 根据演化算符的时间依赖性，构造一个时间演化算符的平方。
3. 固定演化算符的平方的一阶导数，得到其二阶导数。
4. 对该二阶导数作泰勒展开，并假设离散化误差小于某个阈值，使得原演化算符的误差可以通过对该导数的近似来减小。
5. 从初态出发，依据时间演化算符和时间演化算符的平方，用一些递推的方法来近似计算量子态的演化。

# 3. 应用案例：量子密码学
量子密码学是利用量子力学的特性来实现传统加密算法的无损耗、快速、可靠的特性。目前，量子密码学研究取得了一定的进展。其中，最著名的有BB84量子通信协议、NMR量子变分分析、Topological Quantum Key Distribution等。下面我们以BB84量子通信协议为例，介绍如何利用量子密码学实现信息安全。
## 3.1 BB84量子通信协议
BB84量子通信协议是量子通信协议的代表。它是美国爱立信公司提出的通信协议。其基本思想是利用量子纠缠效应，在信道上进行双边测量，实现高速、可靠、无损耗的通信。

BB84量子通信协议的工作流程如下：

1. Alice和Bob各准备一个具有两个贝叶斯基态的量子系统。
2. Alice首先发送一个带噪声的量子比特串给Bob。
3. Bob接收到消息后，根据自己的测量结果，采取适当的动作来恢复带噪声的比特串。
4. Bob发送一个受保护的密钥给Alice。
5. Alice解码得到密钥。
6. Alice和Bob之间的通信就完成了。

Alice和Bob共享一个两比特量子系统，分别被标记为$\vert 00\rangle$和$\vert 11\rangle$。两个量子比特均被赋予不同的错误概率。其中，$\vert 00\rangle$比特被赋予0错误概率，$\vert 11\rangle$比特被赋予1错误概率。Alice和Bob在信道上进行双边测量，测量结果是否正确，以恢复带噪声的比特串。为了保证通信的可靠性，Alice和Bob之间还共享一个被动的接收器。当Alice或者Bob出现故障时，接收器会检测到这一情况，并停止通信。

以上是BB84量子通信协议的基本思想。通过测量量子纠缠效应，双边测量，实现高速、可靠、无损耗的通信。同时，还共享了信道，保证通信的可靠性。
