# 1.简介
  

## 概述
神经网络（Neural Network）是一个用于分类、回归或者预测的计算模型，由输入层、隐藏层和输出层组成。它由一个或多个隐藏层（也称为神经元层）组成，每一层又包括多个节点（或称神经元）。每个隐藏层都由前一层的输出通过激活函数（activation function），转换成后面的层的输入。输出层则将上一层的输出做分类、回归或者预测。
## 发展历史
1943年，约瑟夫·麦卡洛克（Ronald McCulloch）提出了神经网络这一概念。1949年，罗纳德·费尔班（R<NAME> Bell）开始开发神经网络的硬件设备。1957年，罗伯特·李宏基（Robert Lighthill）首次提出“人工神经网络”（Artificial Neural Networks，ANN）这个术语。1958年，约翰·布莱克（John McCulloch）、詹姆斯·麦卡洛克（James McCulloch）等人合著了一本书——“模糊科学和认知过程”，对人工神经网络的发展作出了重要贡献。在这之前，人们普遍认为人脑只能处理简单的信息。然而，随着时间的推移，人们发现人脑具有高度的灵活性，并且可以处理复杂的信号信息。因此，人们开始研究能够模拟人的这种神经系统。当时，提出的一些想法包括：

1. 大脑中存在许多不同的区域，每一个区域负责不同功能的处理；
2. 每个区域之间可以通过神经连接相互联系；
3. 信号传递需要一定的时间延迟，否则信息容易丢失；
4. 不同的区域可能对同一刺激产生不同反应。

为了解决这些问题，人们提出了许多不同的神经网络模型。其中最知名的就是基于“感觉-动作”关系的动态系统模型——模拟人类大脑的工作原理。随着时间的推移，不同的模型逐渐演变，包括：

1. 对证据的显著性评价——Hebbian学习
2. 使用无监督学习方式训练——自组织映射网络
3. 模仿生物行为——Hopfield网络
4. 通过环境影响学习——基于遗传算法的进化算法
5. 建模时考虑时空相关性——卷积网络
6. 将计算机视觉技术应用于神经网络——卷积神经网络CNN

## 特征
- 非线性结构：神经网络通常由若干个隐藏层构成，每层由若干个神经元构成，每个神经元的输出都会影响下一层的输入。因此，神经网络有着非线性的特征，从而可以处理各种复杂的数据。
- 层级结构：每一层之间都是全连接的，前一层中的任意一神经元都直接与后一层中的所有神经元相连，这样就保证了特征的全局有效性。
- 参数共享：由于每一层都与前一层完全连接，因此参数共享是一种常用的手段，使得模型参数规模变小，训练速度加快。
- 可微分特性：因为神经网络是基于误差逼近的，所以它具备良好的可微分特性，可以被用来进行机器学习、优化以及统计分析等领域。
- 自动学习：神经网络可以在数据中自我学习，并对数据的分布和模式做出自适应调整，从而在不需人工参与的情况下做出很好的预测。
- 高容量：神经网络通常都具有较高的容量，能够处理多维度、高维度的输入数据，同时保持较高的精度。
