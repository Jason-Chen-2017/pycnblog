# 1.简介
  

数据结构(Data Structure)是指相互之间存在一种或多种关系的数据元素的集合。它是计算机存储、组织、管理数据的方式，可以用来高效地处理复杂的信息。根据存储空间的需求及访问时间的要求，系统总体上可以划分为若干个不同的抽象数据类型，每个抽象数据类型都由若干个具体数据对象组成，并通过一些逻辑关系将这些对象联系起来。数据结构就是数据的组织形式、存储方式和访问方法。

数据结构的分类和常用数据结构的讲解，本文不会详细阐述。读者可自行了解相关概念。下面重点讨论面试中常见的数据结构算法，包括栈（Stack）、队列（Queue）、链表（Linked List）、树（Tree）、图（Graph）。并且根据实际工作经验进行详实的讲解和分析，帮助读者更好地理解相应的数据结构及其在实际应用中的作用。

2.栈 Stack
栈是一种线性表数据结构，只允许在表尾添加或者删除元素，遵循后进先出(Last-In-First-Out)原则。栈具有以下几个主要特性:

1. push() 操作：在栈顶插入一个新的元素
2. pop() 操作：删除栈顶元素，同时返回被删除的元素值
3. peek() 操作：查看栈顶元素的值，但不对栈做任何修改
4. isEmpty() 操作：判断栈是否为空
5. size() 操作：获取栈中元素个数

堆栈应用：比如程序调用栈、表达式求值运算符优先级栈、编译器中的函数调用栈等。


栈操作的实现通常基于数组或者链表。下面从代码实现层面和应用层面分别介绍栈的四个基本操作以及应用场景。

# 1.入栈 push() 操作
push() 操作是在栈顶插入一个新元素到栈内。常见的代码实现如下所示：

```java
public void push(int item){
    stack[++top] = item; // top是栈顶指针，++top表示下标向后移动一步指向新插入的元素
}
```

push() 操作的时间复杂度是 O(1)，因为 push() 操作只涉及栈顶指针的简单赋值操作，无需循环迭代查找位置，因此执行速度非常快。

# 2.弹栈 pop() 操作
pop() 操作是删除栈顶元素并返回该元素的值。常见的代码实现如下所示：

```java
public int pop(){
    if (isEmpty()){
        throw new RuntimeException("Stack is empty!");
    }
    return stack[top--]; // --top表示下标向前移动一步指向栈顶元素
}
```

pop() 操作的时间复杂度是 O(1)。

# 3.查看栈顶元素 peek() 操作
peek() 操作用于获取栈顶元素的值而不删除它。常见的代码实现如下所示：

```java
public int peek(){
    if (isEmpty()){
        throw new RuntimeException("Stack is empty!");
    }
    return stack[top];
}
```

peek() 操作的时间复杂度也是 O(1)。

# 4.判空 isEmpty() 操作
isEmpty() 操作用于判断栈是否为空。常见的代码实现如下所示：

```java
public boolean isEmpty(){
    return top == -1; // top == -1 表示栈为空
}
```

isEmpty() 操作的时间复杂度是 O(1)。

# 5.获取栈大小 size() 操作
size() 操作用于获取栈中元素个数。常见的代码实现如下所示：

```java
public int size(){
    return top + 1; // top 是栈顶指针，size() 操作返回 top 下标对应的元素
}
```

size() 操作的时间复杂度是 O(1)。

# 6.应用示例：进制转换
进制转换一般都是采用栈来实现。比如十进制数 123 可以转换成二进制数 1111011。我们可以按位将十进制数 123 从右至左压入栈，然后依次弹出数字，最后得到转换后的结果。下面是一个 Java 代码实现：

```java
public static String decimalToBinary(int num) {
    StringBuilder sb = new StringBuilder();
    while (num > 0) {
        sb.append(num % 2); // 余数为 0 或 1，则入栈
        num /= 2;           // 将结果整除为 2 以便计算下一个数字
    }
    return sb.reverse().toString(); // 把栈中的数字连接成字符串并反转顺序
}
```

以上代码的运行时，会把整数转换成二进制数，并逆序输出结果。例如，对于输入参数 123，输出为 "1111011"。
