# 2. 基本概念术语说明
## 2.1 Excel 概览

Microsoft Excel 是微软公司推出的电子表格程序。它具有完整的数据处理功能，能够打开、编辑、分析数字文件，并提供大量的图表、统计工具，同时支持数据的批注和交互式注释。在日常工作和学习中，Excel 可用于数学方面的计算、文字处理、统计分析、财务分析、制图、数据挖掘、可视化和数据建模等各个方面。

## 2.2 数据类型

Excel 有以下几种数据类型：

1. 文本（Text）

文本数据类型用于存储字符串信息，如姓名、地址、职称等。这种类型的数据通常以单元格中的字母、符号、数字形式出现。

2. 数字（Number）

数字数据类型用于存储数值信息，如整数、小数、百分比等。这种类型的数据通常以单元格中的阿拉伯数字或其他符号形式出现。

3. 日期（Date）

日期数据类型用于存储日期和时间信息。这种类型的数据通常以单元格中的日期值或日期字符串形式出现。

4. Boolean（Boolean）

布尔数据类型用于存储逻辑型信息，即真或假的信息。这种类型的数据通常以单元格中的 TRUE 或 FALSE 形式出现。

5. Error（Error）

错误数据类型表示发生了错误。这种类型的数据通常显示为单元格中的 #ERR 这样的文字。

6. 数组（Array）

数组数据类型表示多个值的集合。这种类型的数据会创建多列、多行的数据。

7. 公式（Formula）

公式数据类型用以表示公式。公式可以由其他单元格中的值、运算符、函数等构成。

## 2.3 表单ulas

公式是 Excel 中最重要的功能之一。公式实际上就是基于已有的单元格计算得到的值。例如，A1 单元格的值为 5，那么它的下一个单元格 B1 可以设置为 =A1+3，这时当打开 B1 时就会计算出该值为 8。如果再创建一个新的公式 C1=B1-2，那么当打开 C1 时就会计算出该值为 6。公式不仅可以对单元格进行运算，还可以从不同的数据源读取值并生成新的数据。例如，可以从一个外部文档或数据库中读取值，然后将其赋给某个单元格。

## 2.4 工作表Worksheets

工作表是 Excel 中的一个主要单位。工作表中的每一行和每一列都是独立的单元格，称作单元格。每个工作表可以划分为不同的工作簿。在同一个 Excel 文件中可以创建多个工作表。可以通过不同的方式查看工作表，如按顺序排列、选择性展开等。

## 2.5 工作簿workbook

工作簿是一张 Excel 表，它包括一个或多个工作表。一个 Excel 文件可以包含多个工作簿。除了普通的工作表外，还有可发布表、图表工作表、打印预览工作表、查询结果工作表、定制函数库等。

## 2.6 页表Page Setup

页表设置项让你可以自定义 Excel 页面的大小、纸张方向、边距、字体、颜色等。你可以指定打印多少份，每份的数量、页眉和页脚的位置、缩放比例等。页表设置可以让你方便地打印报告、电子表格、大量的图表、电子病历等。

## 2.7 插入插入选项Insert

插入选项允许你从不同的数据源插入数据，比如从外部文件或数据库导入数据。除了插入数据外，还可以在表格中插入图像、视频、注释、形状等。你可以在当前单元格、另一个单元格或者其他位置插入数据。

## 2.8 数据验证Data Validation

数据验证规则用于确保数据有效性。它可以对输入的数据值进行检查，确保其满足一定条件，否则会导致程序运行错误。例如，你可以限制某个单元格只能输入数字、限制日期范围、限制文本长度等。数据验证还可以减少因用户输入错误而导致的数据出错、数据冲突、系统故障等问题。

## 2.9 分页分页选项Pages

分页选项可以控制 Excel 文件的分页方式，例如，是否在打印时分页，每页多少行或者列。通过分页，可以把数据分成几个较小的文件，便于管理和传输。分页也可以加快 Excel 文件的打开速度。
