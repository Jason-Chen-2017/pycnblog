# 1.简介
  

## 1.1 投资的意义
“投资”这个词汇最早源自于西塞罗的《伦敦帝国》（L'École des Métiers），这是一部关于欧洲贵族的著作。在这部小说中，塞皮尔斯（Sir Isaac Stern）一边盛赞纽曼（Napoleon）的威望、金钱、豪放、文化，另一边又惊叹到底层工人的勤奋、乐观主义。他的结论是，“人的贫困和短缺是人类历史上最大的病症之一”，而“只有极少数精英阶层可以摆脱贫困，取得成功”。因此，要想摆脱贫穷、超越平庸，投资就是唯一出路。投资给予我们更大的财富自由，让我们获得终身的幸福和满足，最终塑造了社会形态和经济基础。

近几年，随着互联网行业崛起，无数炙手可热的技术创业公司逐渐冒出来，带来巨大的金融价值和资本红利。如何从0到1，以最快的速度获得投资回报，成为技术的炼金术，也成为很多投资者研究技术公司长久持有、盈利的关键。

技术企业不仅需要技术，而且还需要对市场、对投资者和对公司的理解。作为一名技术人员，你需要熟悉领域内的基本概念和术语，掌握一种编程语言或者工具，学会运用它解决实际的问题，能够给出准确的商业方案。

作为资深技术人才，对于技术投资的一切疑问都值得思考，因为它们往往关乎生命中的重大选择，而这些选择所涉及的风险往往决定了你的生命走向。如果你不了解这些基本知识和技能，那么就可能错失良机。

## 1.2 技术股的特点
技术股的特点主要有以下五方面：

1. 发行规模：一般来说，技术股的发行规模都很大，比如阿里巴巴集团、腾讯、百度、滴滴等一线互联网公司都是大型公司，具有海量的资金需求；
2. 技术影响力：技术股的盈利模式和员工技术水平，往往决定于公司是否拥有独特的技术优势，以及该公司对整个行业的布局及影响力；
3. 信息时代特征：技术股通常具备“高增长率”和“快速反应性”等特征，能够快速追赶市场变化，为客户提供新的服务或产品；
4. 深度价值取向：技术股往往将重点放在一些领域，比如人工智能、区块链、物联网、虚拟现实、AR/VR、5G、量子计算等；
5. 上市前景多元：技术股的发行量非常大，并且变化多端，因此在上市前景也是十分多元的，你可以找到很多种类的公司，包括刚刚成立的、上市排名前几的、亏损严重的、下跌明显的。

## 1.3 技术投资的含义
技术投资（Technology investing）指的是利用科技进行金融投资活动。简单来说，就是利用科技手段开发新技术，创造新产品，开拓新的产业，从而实现资本市场的优化配置，通过对资本市场不同玩法的综合利用，更好地发展和提升企业的竞争力，达到企业增长和收益最大化的目的。

与传统的金融投资相比，技术投资的一个显著特征是投资者需要大量的时间和经验才能参与其中，以及对待投资人和管理层的特殊要求。这些要求使得技术投资呈现出更多的不确定性和复杂性，并引入了一定的风险。但这一切的背后，都体现了投资者对新兴事物的敏锐洞察能力。

基于此，技术投资者除了关注长期的经济增长之外，还需要考虑其短期效应。例如，一款新技术可能会在短时间内改变某些行业的格局，导致投资者在短期内获得相应的回报，甚至超过了其长期平均回报。技术投资还可以帮助企业发现新的市场和客户，促进科技创新、制定创新政策，并更有效地吸引和保留高技能人才。

最后，技术投资者也需要加强对信息化、云计算、大数据、人工智能、区块链等新技术的应用。随着技术革命的发展，技术投资将扮演越来越重要的角色，成为国家政策制定、经济发展和社会变迁的支柱之一。

# 2. 技术概念及技术选股原则
## 2.1 技术概念
1. 概念：

- 机器学习(Machine Learning)：机器学习是指计算机通过训练算法来模仿人的学习行为，并从数据中推导出规律，从而对未知数据进行预测的一种方法。机器学习有三大功能：
- （1）特征工程：通过分析、处理、转换、抽取数据的过程，生成用于机器学习的输入特征，如文本、图像、视频等。
- （2）模型训练：通过对特征工程之后的数据进行训练，生成模型参数，如神经网络的参数等。
- （3）模型预测：利用已经训练好的模型对新输入数据进行预测，输出结果。

- 深度学习(Deep Learning):深度学习是指利用多层结构的神经网络对大量数据进行高效训练，从而对未知数据进行预测的一种机器学习算法。其主要特点有四：
- （1）非线性激活函数：深度学习网络通常采用ReLU函数作为非线性激活函数，避免了传统神经网络中的sigmoid、tanh等非线性激活函数的饱和特性。
- （2）权值共享：相较于传统的神经网络，深度学习网络使用权值共享减少了模型参数数量，同时又保证了模型性能。
- （3）梯度裁剪：为了防止梯度爆炸和梯度消失，在训练过程中，通过设置阈值限制梯度大小，防止模型过拟合。
- （4）批归一化：为了使各层的输入分布一致，使用Batch Normalization对输入数据进行归一化处理，同时减轻网络的过拟合问题。

## 2.2 技术选股原则
1. 选股原则：价值驱动。通过“孔雀”策略和“乌鸦”策略，在长期价值和短期收益之间寻找平衡。
2. 价值驱动：通过分析价值、对照竞争对手、挖掘潜在市场价值、评估投资风险、构建投资组合，选择出价值突出的技术股票。
3. “孔雀”策略：对技术股及其相关性、市场占有率、管理层能力、盈利模式等进行调查分析，选出业内领先技术股。
4. “乌鸦”策略：选择出市场表现较好的少数股票，获取短期价值回报。
5. 挑选指标：根据上市公司的管理层、技术能力、研发能力、盈利能力等因素对股票进行筛选。
