# 2.基本概念术语说明
## 2.1 TensorFlow
TensorFlow是一个开源机器学习框架，可以用来进行实时的数值计算、数据建模、深度学习和图形处理等。它的主要特点包括：

1. 数据流图（Data Flow Graphs）：使用数据流图可以高效地表示和执行各种机器学习算法。它可以自动地把不同的算子连接成一个计算图，并将数据流动的方式编码进图中的边缘。数据流图使得编写、优化和部署机器学习算法变得非常简单。

2. 自动微分（Automatic Differentiation）：通过自动微分，TensorFlow可以利用链式法则快速、准确地求导数。此外，它还可以使用计算图中的反向传播算法（Backpropagation）实现高效的训练过程。

3. GPU支持：TensorFlow可以在NVIDIA CUDA GPU上运行，并自动利用GPU资源加速运算。

4. 跨平台：TensorFlow可以运行在Linux、Windows和macOS等不同操作系统平台上，并且支持Python、C++和Java编程语言。

5. 易用性：TensorFlow提供了易用的API接口和丰富的文档，使得初学者无需深入了解底层机制即可快速上手。

## 2.2 Tensorflow vs PyTorch
### 2.2.1 Tensorflow
Tensorflow是一个基于数据流图（data flow graph）的开源机器学习库，用于高性能的数值计算。由Google主导开发，得到了广泛的应用，包括：图像分类、文本分析、声音识别、自然语言处理、推荐系统、无人驾驶、医疗健康等。
### 2.2.2 PyTorch
PyTorch是基于Torch张量库的一个开源机器学习库，其功能强大且易于使用。相比于TensorFlow，PyTorch更适合于构建复杂的神经网络，但学习曲线较低。Torch的作者斯坦福大学的<NAME>教授曾说过：“两者之间的差距正在缩小，PyTorch现在成为当前最热门的深度学习框架。”Torch实现了卷积神经网络、循环神经网络等神经网络结构，提供良好的灵活性和高性能，是深度学习领域目前主流的工具。

## 2.6 深度学习的基本方法
深度学习的基本方法包括：
1. 监督学习：深度学习的监督学习就是指给定输入和正确输出的数据集，通过训练模型以学习数据的规律，这种方式能够极大地提升机器学习的效率。监督学习的典型任务有回归问题和分类问题，其中回归问题是预测连续变量的值，而分类问题是预测离散变量的值。
2. 无监督学习：深度学习的无监督学习就是指没有给定正确输出的情况下，尝试从数据中学习隐藏的模式和结构。无监督学习的典型任务有聚类问题、关联分析问题、降维问题。
3. 强化学习：深度学习的强化学习是指在与环境互动过程中，通过选择行为来最大化长期收益。强化学习可以看作是一种特定的机器学习问题，它涉及到一组智能体（Agent）与环境互动，通过学习获得奖励并根据策略调整自己的行为，以达到最大化累计奖励的目的。
4. 迁移学习：深度学习的迁移学习是指利用已有的模型来解决新问题。迁移学习借鉴了现有模型的学习能力，通过改变新任务的设置来训练模型，而不是从头开始训练模型。
