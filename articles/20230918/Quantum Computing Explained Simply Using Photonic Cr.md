# 2.基本概念
## 2.1 引言

量子计算（Quantum Computing）是指利用现代物理定律研究的一种新型计算方式。它不同于经典计算方式的二进制表示法（Binary Representation），采用的是用希腊字母组成的超对称多体量子系统。这种计算方式背后的动机是对于经典计算的局限性提出质疑，通过量子技术打破经典计算机的限制，达到超越经典计算机的计算高度。

量子计算的理论基础可以归纳为三个方面：
1. 量子叠加：指量子现象（如电子或粒子）的存在可以叠加产生一种多层级的影像，称为量子叠加态。量子计算利用这种特性进行计算。
2. 量子通信：量子通信利用量子通信的特性——不确定性和冗余信息——来传输任意规模的信息。
3. 量子随机算法：量子计算提供了一个量子随机算法的概念。这个算法由量子算法和量子电路组成，能够在不经意间掩盖出计算过程中的错误，使得对信息的处理变得更加复杂。

## 2.2 量子位、量子态和量子门

### 2.2.1 量子位（Qubit）

量子位（Qubit）是量子计算的最小单元，通常是一个比特。每个量子位都有一个两种可能的状态——0和1。两个不同的量子位之间有三种可能的组合方式——可以相互垂直排列，也可以通过贝尔-门（Bell-state）产生相互激励的效果。通过测量可以得到0或1的结果。量子位的结构类似于二进制位。

### 2.2.2 量子态（Quantum State）

量子态（Quantum State）是指系统中所有量子位的集合，既包括基态（Ground State）又包括其他的非基态。除了基态，其他的非基态也是构成量子态的基本组成单元。一个物理系统可能处于多种不同的量子态，不同的量子态代表着不同的系统状态。不同量子态下的物理系统也有所不同。

### 2.2.3 量子门（Quantum Gate）

量子门（Quantum Gate）是指用来控制量子系统的运作的逻辑门。量子门有许多种类型，如三角形门、矩形门、酉门、U门等。目前一般采用的量子门都是基本门，基本门是在量子门中最简单的门。

## 2.3 光子晶体架（Photon crystal）

光子晶体架（Photon crystal）是指由许多离散的光子点阵成的一块凹面玻璃，它是量子计算的计算资源。我们可以把它看成是由多个量子位组成的一个量子计算机。

## 2.4 量子电路（Quantum Circuit）

量子电路（Quantum Circuit）是量子计算的一种实现方式。量子电路中有一系列的量子门连续作用，将初始态（输入）转换为目的态（输出）。量子电路的作用是为了解决实际问题而设计的。它是为了制造量子计算机和量子通信设备等量子系统的关键组件。

## 2.5 量子算法（Quantum Algorithm）

量子算法（Quantum Algorithm）是指用来处理和演化量子态、从而解决某些问题的计算过程。其包括两个步骤：编码和译码。编码阶段是将信息转换为量子态的过程；译码阶段则是从量子态恢复信息的过程。编码和译码之间的界线并不是绝对的，这是由于编码过程中可能会丢失或者损坏信息。为了正确地做出判断，需要有译码步骤。

## 2.6 量子信息（Quantum Information）

量子信息（Quantum Information）是指利用量子计算和量子通信对信息进行传递、存储、处理和演化的方法。它是一种信息编码和传送方式。量子信息属于量子通信范畴。

# 3.量子计算的基本概念

## 3.1 量子态

任何可以被观察到的量子世界都是由量子态决定的。因此，理解量子态的结构、特性和行为至关重要。量子态由两个部分组成：基态（Ground State）和杂项（Excited States）。

### 3.1.1 基态（Ground State）

量子态的基态（Ground State）是指系统的最小正能量状态，它是一个平衡态，任何其他的可能状态都要比基态低。没有能量的分子无法与基态发生共振。根据不同的物理原理，我们可以预测哪些东西会在系统的基态发生作用，例如费米子、粒子、量子黑洞、甚至是宇宙的边缘。当然，我们可以人为改变量子态的基态。

### 3.1.2 潜在态（Occupied States）

除去基态外，还有一些其它状态被称为潜在态（Occupied States）。一个态可以是已占据的（Occupied）、自由的（Unoccupied）或震荡的（Traced out）。已占据的态是指那些只在某个特定位置上有正电荷的态，此时不会发生任何作用。自由的态可以被认为是还没有占据的态，可以被任然处于基态的热量。震荡的态没有任何正电荷。通常情况下，我们只能观测到已占据的态，因为我们只能从这些态中进行观测。

### 3.1.3 非振幅（Anti-Frequency）

当存在时间的演化时，系统可以转变成另一种态。若两个态之间存在差别，那么它们之间的能量差距就会增加。这一点可以通过研究系统中非振幅的大小来表征。系统中具有较高非振幅的态往往是异常态（Abnormal States），它不能再通过能量简洁形式描述了。非振幅可以通过谱分析等方法进行研究。

### 3.1.4 玻色子（Boson）

玻色子（Boson）是物理系统中所含有的粒子。一个处于玻色子态的物理系统被称为玻色子系统。一般来说，玻色子态是密封的（Closed），而且只有一个。玻色子系统可以在其基态产生作用，这被称为电磁辐射玻色子理论。

## 3.2 量子门

量子门（Quantum Gates）是指用来控制量子系统的运作的逻辑门。量子门有两种类型：

### 3.2.1 基本门

基本门是量子计算中最简单和最基本的门。基本门由电路模型和门电路模型表示。如下图所示，基本门由三个元素组成，分别是输入门、逻辑门、输出门。输入门负责引入能量，并使量子位从输入态变为工作态；逻辑门负责对其中的能量进行运算；输出门负责将能量释放出来，并将量子位从工作态变为输出态。一个量子电路可以由多个基本门相连，实现任意计算。



### 3.2.2 可逆门

可逆门（Reversible Gates）是指可以将量子门还原为其输入态的量子门。如果一个量子门可逆，那么它就必须满足以下条件：

1. 没有内阻（Decoherence）
2. 有某种可逆运算（Unitary Operation）

可逆门有很多种类型，例如恒等门、Pauli门、Hadamard门、S-门、CNOT门、SWAP门、T-门、Toffoli门等。

## 3.3 量子电路

量子电路（Quantum Circuits）是量子计算的一种实现方式。量子电路中有一系列的量子门连续作用，将初始态（输入）转换为目的态（输出）。量子电路的作用是为了解决实际问题而设计的。它是为了制造量子计算机和量子通信设备等量子系统的关键组件。

### 3.3.1 准备工作

我们需要对某种形式的物理系统构建相应的量子电路，然后演化该量子电路上的量子态，最终使其达到期望的目标。

### 3.3.2 执行顺序

量子电路执行顺序是指量子门的应用顺序。一个量子电路的执行顺序决定了量子系统的状态演化的顺序。量子电路一般按照固定的步骤进行，如图所示。




## 3.4 量子算法

量子算法（Quantum Algorithms）是指用来处理和演化量子态、从而解决某些问题的计算过程。其包括两个步骤：编码和译码。编码阶段是将信息转换为量子态的过程；译码阶段则是从量子态恢复信息的过程。编码和译码之间的界线并不是绝对的，这是由于编码过程中可能会丢失或者损坏信息。为了正确地做出判断，需要有译码步骤。

### 3.4.1 编码

量子算法的编码（Encoding）是指将信息转换为量子态的过程。编码可以采用各种编码方案。常用的编码方案有 amplitude encoding、bit string encoding 和 phase kickback encoding。 amplitude encoding 是基于信号强度的编码方案。 bit string encoding 是基于比特串的编码方案。 phase kickback encoding 是基于相位反馈的编码方案。 

### 3.4.2 译码

量子算法的译码（Decoding）是指从量子态恢复信息的过程。常用的译码方案有 amplitude decoding 和 measure based decoding 。amplitude decoding 是基于信号强度的译码方案。 measure based decoding 是基于测量的译码方案。 

### 3.4.3 量子信道

量子信道（Quantum Channel）是指利用量子信息交换来进行信息传输的过程。量子信道分为单向信道和双向信道。单向信道指的是发送者只能接收，而接收者只能发送消息。双向信道指的是双方都可以收发消息。

## 3.5 量子信息

量子信息（Quantum Information）是指利用量子计算和量子通信对信息进行传递、存储、处理和演化的方法。它是一种信息编码和传送方式。量子信息属于量子通信范畴。

### 3.5.1 量子纠缠（Quantum Entanglement）

量子纠缠（Quantum Entanglement）是指两个量子系统在相互作用下具有相同或很接近的态，并且可以通过某种方式联系起来。在某些情况下，两个量子系统可以不经过任何干扰直接通信。量子纠缠可以是弱纠缠或强纠缠。

### 3.5.2 量子编码（Quantum Encoding）

量子编码（Quantum Encoding）是指通过一种特殊的方式将比特流编码成量子态。一个有效的量子编码方案应该允许消息在传输过程中保持不变。

# 4.量子计算的基本原理

## 4.1 测量（Measurement）

测量（Measurement）是量子计算的基本操作之一。测量操作是指从量子态中取出特定的量子态。即选择一个特定的量子态，并使之脱离当前的量子态。之后，系统就会混合成一个单独的“粒子”（Particle），这一过程就是测量。由于测量过程会随机丢弃信息，所以测量的结果不一定是确定的。量子位的测量值称为测量结果（Measurement Result）。

### 4.1.1 测量结果的统计分布

量子态的每种可能的结果都可以由概率来表示。我们可以使用概率分布函数来刻画测量结果的统计分布。统计分布函数是指给定系统的某种状态后，系统的各个可能状态出现的概率。概率分布函数可以用来描述任意一段时间内，量子系统的不同状态出现的频率。

### 4.1.2 测量门

测量门（Measurement Gate）用于测量量子系统，它将量子态转换为类ical状态。测量门由两部分组成，第一部分是所要检测的量子位，第二部分是测量结果的存储位。在测量完毕后，量子位将置零。

### 4.1.3 量子测量

量子测量（Quantum Measurement）是指利用量子门操作来测量量子系统的基本操作。量子测量可以定义为求取“期望值”或“投影值”。

## 4.2 量子通道

量子通道（Quantum Channels）是利用量子信息交换来进行信息传输的过程。量子信道分为单向信道和双向信道。单向信道指的是发送者只能接收，而接收者只能发送消息。双向信道指的是双方都可以收发消息。

### 4.2.1 流形码

流形码（Polar Codes）是一种具有两个可能结果（0或1）的编码方案。流形码的出现是为了克服量子计算机上“冯诺依曼”编码器所面临的困境。流形码的优点是容错能力高，且通信时间短。

## 4.3 測量

量子測量（Quantum Measures）是指利用量子测量来获得系统的统计分布信息的计算方法。其目的是为了确定量子系统的量子态是处于什么状态，以及量子态处于该状态的概率。

## 4.4 混合（Entanglement）

混合（Entanglement）是指两个量子系统在相互作用下具有相同或很接近的态，并且可以通过某种方式联系起来。在某些情况下，两个量子系统可以不经过任何干扰直接通信。量子纠缠可以是弱纠缠或强纠缠。

### 4.4.1 强纠缠

强纠缠（Strong Entanglement）是指两个量子系统完全耦合在一起，可以直接进行通信。这要求两个量子系统必须经过某种场的作用才能相互作用。例如，在宇宙量子力学中，两个处于激发态的粒子可以直接相互作用，也可以通过某种场的作用相互作用。强纠缠可以一定程度上避免量子通信网络的堵塞。

### 4.4.2 弱纠缠

弱纠缠（Weak Entanglement）是指两个量子系统相互作用，但是没有严格的耦合关系。这就要求两个量子系统必须经过某种场的作用才能相互作用，然而这种场必须不严重影响两个量子系统的独立性。例如，两个处于未激发态的粒子可以是弱纠缠的。弱纠缠可以在一定程度上减少量子通信的资源需求。
