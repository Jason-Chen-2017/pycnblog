### 第1章：群论的基本概念

#### 1.1 群的定义与性质

群论是数学中研究一组元素及其组合规则的理论。一个群是由一组元素组成的集合，这些元素满足特定的组合规则，即封闭性、结合律、单位元和逆元。具体来说，一个群 \(\mathbb{G}\) 应满足以下性质：

1. **封闭性**：对于群中的任意两个元素 \(a\) 和 \(b\)，它们的组合 \(a \cdot b\) 也应该属于该群。
2. **结合律**：对于群中的任意三个元素 \(a\)、\(b\) 和 \(c\)，组合的顺序不影响结果，即 \((a \cdot b) \cdot c = a \cdot (b \cdot c)\)。
3. **单位元**：群中存在一个特定的元素 \(e\)，使得对于群中的任意元素 \(a\)，都有 \(a \cdot e = e \cdot a = a\)。
4. **逆元**：对于群中的任意元素 \(a\)，存在一个元素 \(a^{-1}\)，使得 \(a \cdot a^{-1} = a^{-1} \cdot a = e\)。

例如，整数加法构成一个群，其中 \(e = 0\)，\(a^{-1}\) 是 \(a\) 的相反数。同样，整数乘法也构成一个群，其中 \(e = 1\)，\(a^{-1}\) 是 \(a\) 的倒数（对于非零整数而言）。

#### 1.2 子群与商群

子群是指一个群内部的另一个子集，它本身也满足群的定义。例如，在一个整数加法群中，所有偶数的集合构成一个子群。

商群则是通过一个群对某个子群的“除法”操作得到的。具体来说，给定群 \(\mathbb{G}\) 和其子群 \(\mathbb{H}\)，可以将 \(\mathbb{G}\) 中每个元素与 \(\mathbb{H}\) 中的某个元素对应，形成一个新的集合，这个新的集合构成一个商群。商群的概念在理解群的结构和性质中非常重要。

#### 1.3 群的同态与同构

群的同态是指两个群之间存在一种映射，这种映射保持群的结构。换句话说，如果存在一个函数 \(f: \mathbb{G} \rightarrow \mathbb{G'}\)，使得对于群中的任意两个元素 \(a\) 和 \(b\)，都有 \(f(a \cdot b) = f(a) \cdot f(b)\)，那么这个函数就是一个同态。

同构是同态的一种特殊情况，即两个群之间存在一个双射的同态。如果两个群之间存在同构，那么这两个群在数学意义上是完全相同的，只是元素的具体表示不同。

#### 1.4 群的代表理论与线性群

群的代表理论是研究如何将一个群表示为线性变换的集合。在群的代表理论中，线性群是一个非常重要的概念。线性群是指所有线性变换构成的群，这些线性变换保持了向量空间的性质。

线性群在群论和物理学中都有广泛的应用。例如，在量子力学中，物理系统的对称性通常用群表示，而线性群则提供了描述这些对称性的数学工具。此外，线性群在计算机科学中也有重要应用，例如在图像处理和机器学习等领域。

#### 3.1 简单对称性与舞蹈编排

简单对称性是最基本的对称性形式，包括镜像对称和轴对称等。在舞蹈编排中，简单对称性可以通过以下方式运用：

1. **镜像对称**：通过身体部位的镜像反射来创造对称动作。例如，一个舞者可以面对观众，同时他的左右手和左右脚分别做相同的动作，形成镜像对称。
2. **轴对称**：通过一个中心轴将动作分为两部分，两部分对称。例如，一个舞者可以沿着身体的中轴线旋转，使得身体的一部分向上运动，另一部分向下运动，形成轴对称。

#### 3.2 旋转对称性与舞蹈编排

旋转对称性是指一个动作可以通过旋转某个角度后与自身重合。在舞蹈编排中，旋转对称性可以通过以下方式运用：

1. **旋转动作**：舞者可以围绕自己的中心点旋转，形成旋转动作。例如，一个舞者可以旋转180度或360度，使得动作与自身重合。
2. **多边形舞姿**：通过舞者的位置和动作形成多边形的旋转对称。例如，四个舞者可以分别站在一个正方形的四个角上，他们的动作形成一个正方形的旋转对称。

#### 3.3 对称性与舞蹈节奏

舞蹈节奏是舞蹈中的重要元素，对称性可以用来创造有趣的节奏效果。以下是对称性与舞蹈节奏的几种运用：

1. **同步节奏**：通过对称性使舞者的动作与音乐节奏同步。例如，舞者可以同时抬起手臂或腿部，形成一个同步的节奏动作。
2. **交替节奏**：通过对称性使舞者的动作交替出现，形成交替节奏。例如，舞者可以交替进行抬手和抬腿的动作，形成一个交替的节奏。

#### 3.4 对称性与空间布局

对称性还可以用来设计舞蹈的空间布局，创造独特的视觉效果。以下是对称性与空间布局的几种运用：

1. **对称舞台布局**：通过舞台的空间布局形成对称效果。例如，舞台可以分为两个对称的部分，舞者可以在两个部分中分别进行对称动作。

#### 附录 A：群论基础公式与符号说明

- **群的定义**：设 \(G\) 为一个非空集合，\(a, b \in G\)，若 \(a \cdot b \in G\)，则称 \(G\) 对运算 \(\cdot\) 是封闭的。设 \(e\) 为 \(G\) 中的一个元素，如果对于任意的 \(a \in G\)，都有 \(a \cdot e = e \cdot a = a\)，则称 \(e\) 为 \(G\) 的单位元。如果对于任意的 \(a \in G\)，都存在 \(a^{-1} \in G\)，使得 \(a \cdot a^{-1} = a^{-1} \cdot a = e\)，则称 \(a^{-1}\) 为 \(a\) 的逆元。
- **子群**：设 \(H\) 为 \(G\) 的一个子集，若 \(H\) 本身也是一个群，则称 \(H\) 为 \(G\) 的子群。
- **商群**：设 \(H\) 为 \(G\) 的一个子群，\(G/H\) 表示 \(G\) 对 \(H\) 的商群，其元素为 \(G\) 中与 \(H\) 中元素同余的等价类。

```python
import numpy as np

# 定义旋转群
class RotationGroup:
    def __init__(self, angle):
        self.angle = angle

    def __mul__(self, other):
        return RotationGroup(self.angle + other.angle)

    def __repr__(self):
        return f"Rotation({self.angle} degrees)"

# 定义舞蹈动作类
class DanceAction:
    def __init__(self, rotation_group):
        self.rotation_group = rotation_group

    def perform(self):
        print(f"Executing a {self.rotation_group.angle} degree rotation.")

# 演示旋转对称性舞蹈
def demonstrate_rotation_symmetry():
    rotation_180 = RotationGroup(180)
    rotation_360 = RotationGroup(360)

    action_180 = DanceAction(rotation_180)
    action_360 = DanceAction(rotation_360)

    action_180.perform()
    action_360.perform()

# 执行演示
demonstrate_rotation_symmetry()
```

- `RotationGroup` 类表示一个旋转群，具有角度属性。它实现了 `__mul__` 方法，用于实现旋转群的组合规则。
- `DanceAction` 类表示一个舞蹈动作，具有旋转群属性。它实现了 `perform` 方法，用于执行舞蹈动作。
