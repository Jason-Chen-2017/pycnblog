### 4.2 公式推导过程

首先，我们定义查询向量 $Q$、键向量 $K$ 和值向量 $V$ 分别为：

$$
Q = [q_1, q_2, ..., q_n]
$$

$$
K = [k_1, k_2, ..., k_n]
$$

$$
V = [v_1, v_2, ..., v_n]
$$

然后，我们定义注意力分配函数 $f(Q, K, V)$ 为：

$$
f(Q, K, V) = \text{softmax}(WQ^K)

其中，$W$ 是一个可训练的权重矩阵。

### 5.2 源代码详细实现

```python
import tensorflow as tf
import numpy as np
import matplotlib.pyplot as plt

# 定义查询向量、键向量和值向量
Q = tf.random.normal([3, 10])
K = tf.random.normal([3, 10])
V = tf.random.normal([3, 10])

# 定义权重矩阵 W
W = tf.random.normal([10, 10])

# 定义注意力分配函数
def attention(Q, K, V, W):
    # 计算查询向量与键向量的内积
    scores = tf.matmul(Q, K, transpose_b=True)
    
    # 应用softmax函数，得到注意力分配结果
    scores = tf.nn.softmax(scores)
    
    # 计算注意力加权值
    output = tf.matmul(scores, V)
    
    return output

# 应用注意力策略
output = attention(Q, K, V, W)

# 可视化输出结果
plt.scatter(range(output.shape[0]), output.numpy())
plt.xlabel('Query Index')
plt.ylabel('Attention Score')
plt.show()
```

- **权重矩阵 (W)**：用于计算查询向量与键向量的内积，得到注意力得分。
- **注意力分配函数**：通过softmax函数计算注意力得分，然后对值向量进行加权求和，得到最终输出。

在代码中，我们首先生成随机查询向量、键向量和值向量，然后定义权重矩阵。接着，应用注意力策略，计算注意力得分并可视化输出结果。
