### 4.2 公式推导过程

1. **财务模型**：

   - 收入计算公式：$R_t = R_0 \times (1 + r)^t$
   - 支出计算公式：$C_t = C_0 \times (1 + c)^t$
   - 财务安全评估公式：$SA = \frac{R_t - C_t}{C_0}$

2. **健康管理模型**：

   - 健康评估指标：$HI = \frac{BMI + HbA1c + BP + Age}{4}$

3. **技术传承模型**：

   - 知识库构建公式：$KB = \{K_1, K_2, ..., K_n\}$，其中$K_i$为第$i$个知识点。

4. **兴趣爱好模型**：

   - 兴趣评估公式：$I_i = \frac{A_i}{B_i}$，其中$A_i$为第$i$个兴趣点的得分，$B_i$为第$i$个兴趣点的满分。
   - 兴趣爱好推荐公式：$R_i = \sum_{j=1}^{m} w_j \times I_j$，其中$w_j$为第$j$个兴趣点的权重。

5. **职业规划模型**：

   - 职业评估公式：$CA = \frac{SA_i + HI_i + TB_i}{3}$

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 财务模型
def financial_model(years_to_retirement, annual_income, annual_expenses, investment_rate, inflation_rate):
    r = (1 + investment_rate) / (1 + inflation_rate) - 1
    R = annual_income * (1 + r)**years_to_retirement
    C = annual_expenses * (1 + inflation_rate)**years_to_retirement
    SA = (R - C) / C
    return R, C, SA

# 健康管理模型
def health_model(BMI, HbA1c, BP, Age):
    HI = (BMI + HbA1c + BP + Age) / 4
    WS = (HI - 35) / 35
    return HI, WS

# 技术传承模型
def knowledge_model(knowledge_nodes, knowledge_edges):
    KG = (knowledge_nodes, knowledge_edges)
    KB = {i: [] for i in knowledge_nodes}
    for edge in knowledge_edges:
        KB[edge[0]].append(edge[1])
        KB[edge[1]].append(edge[0])
    return KG, KB

# 兴趣爱好模型
def hobby_model(scores, max_scores):
    I = {i: scores[i] / max_scores[i] for i in scores}
    R = sum(w * I[i] for i, w in sorted(I.items(), key=lambda x: x[1], reverse=True))
    return I, R

# 职业规划模型
def career_model(finance_score, health_score, tech_score):
    CA = (finance_score + health_score + tech_score) / 3
    PS = (CA - 0.8) / 0.8
    return CA, PS
