### 4.1 数学模型构建

假设公司有n个员工，每个员工对技术泄露的风险贡献为r_i（i=1,2,...,n）。则总风险可以表示为：

\[ R = \sum_{i=1}^{n} r_i \]

### 4.2 公式推导过程

首先，我们定义员工的风险贡献为：

\[ r_i = P(A_i) \cdot C(A_i) \]

其中，\( P(A_i) \) 表示员工i发生技术泄露的概率，\( C(A_i) \) 表示员工i的技术贡献。

### 5.2 源代码详细实现

```python
import numpy as np
from sklearn.tree import DecisionTreeClassifier

# 假设数据集为5个员工的风险贡献
data = np.array([[0.2], [0.3], [0.1], [0.15], [0.15]])

# 使用决策树分类器进行风险评估
clf = DecisionTreeClassifier()
clf.fit(data, data)

# 输出每个员工的风险等级
for i, risk in enumerate(data):
    pred = clf.predict([risk])
    print(f"员工{i+1}的风险等级：{pred[0]}")
```

代码中，我们首先导入所需的库，然后定义一个5×1的数据集，表示5名员工的风险贡献。接着，我们使用决策树分类器对数据集进行训练，并输出每个员工的风险等级。
