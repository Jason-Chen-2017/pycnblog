### 3.1 基础集合操作

1. **并集**：给定两个集合A和B，其并集A∪B是由属于A或属于B的元素组成的集合。
   $$ A \cup B = \{ x | x \in A \text{ 或 } x \in B \} $$
2. **交集**：给定两个集合A和B，其交集A∩B是由同时属于A和B的元素组成的集合。
   $$ A \cap B = \{ x | x \in A \text{ 且 } x \in B \} $$
3. **补集**：给定一个集合A，其补集A'是由不属于A的元素组成的集合。
   $$ A' = \{ x | x \notin A \} $$
4. **子集**：给定两个集合A和B，A是B的子集，当且仅当A中的所有元素都属于B。
   $$ A \subseteq B \text{ 当且仅当 } \forall x (x \in A \Rightarrow x \in B) $$

### 3.2 类的操作

1. **类与集合的转换**：给定一个类C，可以通过引入新的集合D来表示C，其中D中的元素是C中的所有元素。
   $$ D = \{ x | x \in C \} $$
2. **类与集合的关系**：给定两个集合A和B，其类C是由属于A且属于B的元素组成的类。
   $$ C = \{ x | x \in A \text{ 且 } x \in B \} $$
3. **类的并集**：给定两个类C1和C2，其并集C1∪C2是由属于C1或属于C2的元素组成的类。
   $$ C1 \cup C2 = \{ x | x \in C1 \text{ 或 } x \in C2 \} $$
4. **类的交集**：给定两个类C1和C2，其交集C1∩C2是由同时属于C1和C2的元素组成的类。
   $$ C1 \cap C2 = \{ x | x \in C1 \text{ 且 } x \in C2 \} $$
5. **类的补集**：给定一个类C，其补集C'是由不属于C的元素组成的类。
   $$ C' = \{ x | x \notin C \} $$
6. **类的子集**：给定两个类C1和C2，C1是C2的子集，当且仅当C1中的所有元素都属于C2。
   $$ C1 \subseteq C2 \text{ 当且仅当 } \forall x (x \in C1 \Rightarrow x \in C2) $$

  $$ \emptyset = \{ \} $$

- **单元素集合**：单元素集合包含一个唯一的元素，用{a}表示。
  $$ \{ a \} $$

- **集合的元素**：给定集合A，集合A中的元素可以用符号x ∈ A表示。
  $$ x \in A $$

- **集合的子集**：集合A是集合B的子集，表示为A ⊆ B，如果A中的所有元素都属于B。

集合的基本运算包括并集、交集、补集和子集等。这些运算在NBG公理集合论中有着重要的地位，以下是对这些运算的详细讲解：

- **幂集**：给定一个集合A，其幂集P(A)是A的所有子集构成的集合。
