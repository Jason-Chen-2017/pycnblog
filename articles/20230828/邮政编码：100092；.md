# 1.简介
  

“邮政编码”是一个国家标准单位，由六个字母（区、县、乡、村、街道、社区）组成。它用来唯一地标识一个地址。在中国，由于长期实行计划生育政策，所以大多数农村居民户籍中的城镇住址经常出现不明确、错误甚至不存在的问题。作为有效的管理工具，邮政编码能够帮助解决这些问题。通常情况下，邮政编码需要包含以下几方面的信息：
# 省/直辖市名称（2字母代码）
# 市/区名称（2-3字母代码）
# 区划级代码（3位数字）
# 街道或乡镇名称（最多四个汉字）
# 村委会名称（最多两三个汉字）
# 在中国，目前普遍采用的是“6位编码”方法。该方法包括省份、城市、区县、乡镇、村庄和街道等信息，按照层级结构进行分割，每个级别的代码总共6位。例如，广东省深圳市南山区龙岗镇坪洲村的邮政编码可以用“6位编码”表示为：518035。

然而，随着社会发展，这种简单粗暴的方法已经无法满足现代生活方式下的需求了。比如，城市之间的距离越来越远，很多城市的邮编只能使用“普通话”字符来书写，并且每一个字都是一种组合，阅读起来费时费力。另外，由于历史原因和地理位置的影响，一些地区存在多个邮政编码系统，各自使用自己的编码规则和编码标准。

为了适应这个变化过程，《中国邮政编码分类规定》（邮编编制法）于2017年颁布实施。该规定将邮政编码分为六种类别，分别对应于：
# （1）前三位表示省份、直辖市、自治区、特别行政区和澳门特别行政区。
# （2）后四位表示市、区县、县级市政区和县级市、州级市的郊区郊县、区、乡镇、街道办事处、群众团体和宗教活动场所。
# （3）中间两位表示乡镇和街道办事处。
# （4）第五位及第六位表示村委会、社区和民族乡。

因此，按照新的编码分类规定，“100092”应该属于第一种类别，即前三位表示省份、直辖市、自治区、特别行政区和澳门特别行政区。

# 2.概念术语说明
## 2.1 中国邮政编码（邮编）
“邮政编码”是指按照中国的邮编编制法将地址符号化的一种符号。地址符号化就是把地址信息按照一定形式编码转换成一串字母、数字和符号的字符串。地址符号化完成后，就可以在全球范围内任意传输、传递、查询或存储。“邮政编码”用于识别地址、联系电话、银行账号、身份证号码等。中国从古至今，一直沿袭传统邮编的编制方法，但随着时间的推移，这种简单的编制方法已经无法满足发展的需要。因此，到了20世纪80年代末，中国政府决定对中国的邮编编制提出新的要求，于1982年颁布了《邮政编码工作条例》。该条例制定了国务院关于邮政编码分类的分类法和规定。

## 2.2 “邮政编码”与“邮编”
“邮政编码”是指符合中国邮政编码系统中各个级别的编码规则，按顺序排列的6位数字字母序列。由6位字母或数字组成。“邮编”是指根据邮政编码编制法编制的邮政编码的别称。
