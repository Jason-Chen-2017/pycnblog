# 2.5G基础理论和关键技术
## 2.1 什么是5G
5G是一种新兴的无线电信技术，其主要特点是带宽比之前使用的移动互联网、固定互联网、大容量宽带等都要高得多。这使得5G能够突破传统的单向带宽限制，在短时间内同时传输更多的数据。它可以让服务提供商和终端设备之间建立极快的连接、实现低延迟高带宽的通讯，使得设备间、系统间以及用户之间的信息交流速度得到提升。而5G也具有如下几个主要特征：

1. 全球覆盖：通过第三代移动通信技术（5G）打通全球各个角落，实现超大规模的网络覆盖。
2. 大规模数据处理：支持高吞吐量的数据处理能力，满足不断增长的数据需求。
3. 可靠性与安全性：保证核心网络的可靠性与安全性，防止恶意攻击和病毒感染。
4. 灵活性与开放性：采用统一的物理信道结构，支持不同业务类型和应用程序，充分释放了网络资源的灵活性。
5. 智能计算：5G系统可以智能地处理复杂的计算任务，为人类社会和经济活动提供新的机遇。

## 2.2 5G的主要组成部分

1. 移动性：使用极低功耗的多小时即可切换到5G模式，具有良好的低延迟、高带宽、超低时延的特性，可以用作各种短视频、电视剧、视频会议等高带宽娱乐应用，以及各类智能穿戴产品的上下班导航。
2. 用户体验：5G网络建立的过程中，终端设备需要进行协调优化，确保用户的流畅和优质体验，包括：降低网络拥塞造成的丢包率、提升视频质量、优化画面渲染、改善屏幕刷新频率等。
3. 灵活部署：5G网络可以灵活地部署在不同的网络环境中，从而实现资源共享，包括：高速率、低延迟、广域网、WAN等多种方式，满足不同业务场景下的需求。
4. 多功能通信：5G网络可以支持各种业务，如音视频通话、汽车远程驾驶、航空航天通信、智能住宅管理、城市运输、工业制造、环保监测、医疗健康等，实现高度集成化、高效率的协同工作。
5. 数据分析与挖掘：5G网络可以提供海量数据的存储、分析、挖掘，实现对大数据、IoT数据的收集、处理和分析，助力数据驱动的创新与产业升级。

## 2.3 5G关键技术
### 2.3.1 新一代基站
5G的第二代技术主要是小区小基站(Macro Base Station MBS)，即小型基站。目前，中国已建成超过3亿米级小基站。小基站所搭载的主动转发系统和天线布局，就像是在移动电话上一样，能够进行高度优化的多址接入。虽然小基站部署密度不够精细，但其成本相对较低，同时也可以实现区域范围内的资源共享。
