# 1.简介
  

InfoQ中文社区是一个完全由程序员组成的开源项目，主要关注于中、高级开发人员及云计算领域的技术分享。它以“分享，交流，学习”为核心理念，通过提供专业化、具有价值的技术文章，与国内外技术大牛进行深入交流，促进技术共同进步。 InfoQ中文社区提供了丰富的技术内容，涵盖Java、Python、数据库、运维等领域的技术文章。InfoQ中文社区以开源协作的方式实现其社区的技术力量，鼓励和支持所有爱好者通过参与到社区建设中来。InfoQ中文社区现已形成了良好的品牌声誉，同时也得到了各大公司和政府部门的关注。
## 为什么要建立InfoQ中文社区？
InfoQ中文社区始终坚持以“分享，交流，学习”为核心理念，是一个基于开放社区的技术社区。它的诞生离不开国内外的技术社区的强烈需求，结合InfoQ的独特品牌形象以及定位，以及InfoQ的内容体系的丰富性和更新频率，在国内推出了中文版的InfoQ社区。通过打造一个易于上手的技术社区，通过分享优质文章，帮助国内技术人才快速成长并互相促进，提升国内技术氛围，希望能够创造一个平等、自由、公正的技术环境。
## InfoQ中文社区的愿景
- 提供最全面的中/高级技术文章。
- 通过经验类文章，加深技术人的知识和理解。
- 将用户群体的诉求转化为技术人员的需求。
- 深受广大技术从业者欢迎。
- 鼓励和支持所有爱好者参与社区建设。
## 沟通方式
InfoQ中文社区始终坚持以“分享，交流，学习”为核心理念，因此我们有不同的沟通渠道：
- QQ群（591979055）：该QQ群将为 InfoQ 中文社区的活跃成员提供热心服务。加入后，可以获得较快的响应速度，也可以与其他成员建立更进一步的沟通关系。
- 微信公众号（infoqchina）：微信公众号是 InfoQ 中文社区的官方账号，你可以关注该账号获取最新信息。
- Twitter：Twitter是国内知名的微博平台，可以关注 InfoQ 中文社区的官方账号（@infoqcn）。在该账号下，你可以看到InfoQ中文社区的相关消息、文章。
- GitHub：InfoQ中文社区的源代码仓库托管在 GitHub 上，你可以访问 https://github.com/infoqgz to get more information about the project and find how you can help us improve it!
- Blog：InfoQ中文社区还有一个独立的博客页面，提供更多的技术文章。地址为：https://www.infoq.cn/chinese-infoq-community
- Mailing list：InfoQ中文社区还有一个邮件列表，你可以订阅 infoqchina@gmail.com 来接收 InfoQ 中文社区的最新消息。
