# 1.简介
  

## 什么是独立向量分析（Independent Vector Analysis, IVA）？
IVA是一个音频信号处理领域的著名方法。它的主要思想是将两个或多个声源混合到一起，然后提取出每个声源的单独分离出来。所以我们可以把它看做一种分离声源的方法。

## 我为什么要用IVA？
首先，IVA可以分离出多个声源，这一点在现实中是很有用的。比如在游戏中我们可以利用IVA去除多人同时说话时产生的干扰。其次，IVA可以帮助我们找到某些隐藏的声音信息，比如人类共鸣声、狗叫声等。最后，IVA还可以帮助我们捕捉特定的声音效果，比如电视广播中的杂音、爆竹声等。

## 为什么IVA不是那么普遍？
首先，IVA需要更多的数据才能得到较好的结果。IVA往往只能从无噪声的情况下分离出声源。如果环境噪声比较大，或者有干扰环境导致无法分清声源，IVA的分离质量就可能会受到影响。另外，目前很多应用场景还没有充分探索到IVA的极限，因此还存在一些瑕疵。例如，IVA不能用于训练模型进行声源分类。

# 2.背景介绍
## 声源分离原理及其局限性
声源分离指的是把不同声源的混合信号分离成各自单独的信号。而声源分离又可以分成几种类型：

1. 基于时间的分离

   这种方法是通过对混合信号的时间谱分析，来确定声源的位置和强度。时间谱分析最早是用来研究声音传播途径而发明的。根据混合信号的频谱，可以计算出每个声音源的强度分布。在时间谱图上，就可以看到不同的频率下的激励分布。通过分析不同频率下的激励分布，就可以发现声音源的位置和强度。但时间谱分析有一个缺陷就是要获取整个混合信号的全部信息，当声源个数越多的时候，就需要更多的采样点才能满足要求，而且每秒的采样点也不可能达到数百万。时间谱分析由于获取整体信息，所以只能用于短时间内的声源识别。

2. 基于空间的分离

   这种方法是在空间频率范围内进行分析，通常会采用傅立叶变换FT变换的方式，将混合信号变换到频率域，然后分析每个声音源的频谱分布，通过功率谱比值或者峰值对比，就可以定位出不同声音源的位置和强度。但是，空间频率范围分析的局限性在于只能用于单声道信号的情况。当混合信号具有多个声道的时候，空间频率分析就无能为力了。另外，空间频率范围分析是没有物理意义的，只是为了求某个频率下信号的幅值而设定的。

3. 混合通道分离

   在某些实际应用场景中，通常会存在三个以上声源混合的情况。这是由于人的耳蜗和麦克风的特性决定的。对于三个以上声源混合的情况，可以通过数字音频混合技术混合多个声音信号后再进行分离。这种方法可以有效地消除噪声和混响，还可以实现一定程度的信号增益和声源分离。然而，数字音频混合技术仍然有待商榷，因为混合过程存在着较大的失真和失真�enuation问题。另外，由于采用多个声道的混合，数字音频混合技术往往会造成误分割，使得声源分离的结果出现错误。

4. 全局参数化分离（Global parameterizing separation, GPS）

   随着多种混合声源越来越多，声源分离面临新的挑战。GSP就是在这种情况下诞生的。GSP的基本思想是通过最大似然估计方法来拟合混合参数模型，使得声源分离能够最大程度地抓住目标特征。GSP的主要思路是建立一个参数化的混合模型，包括声源参数和混合参数。声源参数表示了声源的数量和位置，而混合参数则表示了混合系统的动态参数。通过优化声源参数，就可以获得多源分离的结果。但是，GSP的优化问题往往十分复杂，因此GSP的性能一直都不是很理想。

## Blind source separation(BSS)分类及其优点
### 混合声源分离
这里的混合声源分离是指把两种或两种以上的声源混合起来，然后通过算法找到其中每种声源的单独信号。典型的例子包括立体声混合、多声道混合、具有线性相关性的混合等。

BSS的优点如下：

1. 不需要知道声源的准确参数。通过分离混合信号得到的声源的位置和强度，以及其他相关参数，都是经过数字模拟计算而得出的。
2. 可以处理大量混合信号。Blind source separation可以处理信号从很短的持续时间到长时间不断混合的情况。通过BSS可以更好地理解混合环境，并找出其中包含的声源。
3. 可适应各种混合条件。在BSS中，算法不需要知道混合时所使用的算法。也就是说，即使混合系统采用非标准方法混合音乐、交通噪声等，也可以通过Blind source separation进行分离。
4. 有利于通信系统的研究。BSS可以帮助研究人员了解传播路径和信噪比。

### 普通空间频率范围分离
这里的普通空间频率范围分离是指直接分析信号的频率域，从而确定每个声源的位置和强度。典型的例子包括傅里叶变换、时频分解、多普勒效应等。

BSS的优点如下：

1. 算法简单、高效。通过直接分析信号的频率域，BSS算法可以快速分离出声源，而且运行速度快。
2. 灵活性强。BSS可以在不同频率范围内进行分离，可以处理各种混合条件。
3. 无需声源参数。BSS不需要知道声源的准确参数，只需要分析信号的频率域就可以得到声源的位置和强度。
4. 对噪声敏感。BSS算法对噪声敏感，不会引入任何噪声。

### 固定通道分离
这里的固定通道分离是指把信号投影到固定维度，如时间或空间坐标轴，然后进行傅里叶变换分解，求取相应的频谱或幅值。典型的例子包括卡尔曼滤波器、小波分解、矩阵倒谱、汉明窗等。

BSS的优点如下：

1. 分离精度高。通过固定通道分离，可以获取足够的分离精度。
2. 数值稳定。固定通道分离算法得到的结果可靠，容易处理异常数据。
3. 实现简单。固定通道分离算法实现简单，只需要数学知识即可。
4. 适应任意带宽。固定通道分离算法对任意带宽都可以工作。
