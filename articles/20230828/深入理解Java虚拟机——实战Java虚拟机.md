# 1.简介
  

---
Java是一种面向对象的编程语言，拥有功能强大的类库，成为目前最流行的编程语言之一。为了更高效地开发和运行应用程序，Sun公司推出了Java虚拟机(JVM)，它是一个可以执行Java字节码的虚拟机。JVM允许在各种平台上运行相同的代码，从而实现“一次编写，到处运行”的跨平台能力。虽然Sun公司将其开源，但由于其复杂性和开放程度，导致很多公司和开发者对JVM的了解存在误区。因此，本书试图通过系统的介绍Java虚拟机的基本知识和原理，帮助读者系统、全面的掌握JVM。

# 2.为什么要读这本书？
---
如果你希望系统、全面的掌握JVM，或者你有以下需求：

 - 对JVM的机制不了解；
 - 想学习JVM的工作原理；
 - 需要深刻理解JVM中的关键组件及其作用；
 - 在日常开发中遇到了一些性能瓶颈或其他疑难问题需要解决。
 
那么你需要阅读这本书。该书从JVM的内存布局、类加载机制、垃圾回收算法、动态类型、并发编程等方面，系统介绍JVM的核心机制。文章从JVM历史演变，解释JVM技术演进的动力，介绍OpenJDK和OpenJDK HotSpot等JVM实现，并详细阐述JVM的启动过程。书中还覆盖了运行时数据区，堆空间管理，栈空间管理，方法调用、异常处理，线程管理等方面。最后还会对HotSpot虚拟机进行深入剖析，分析JVM的具体实现，并通过实际例子和分析，帮助读者真正理解JVM的实现细节。

当然，本书的目的是为读者提供一本完整的JVM权威教程，涉及的内容也比较广泛。如果只是作为参考，那么也可以作为一个补充材料来快速了解JVM。
