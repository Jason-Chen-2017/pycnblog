# 1.简介
  

## 概念定义
在中国，邮政编码（postal code）也称邮政编制码、邮政区号或者邮政编码系统是用来标识邮政地址的一种信息符号。它是一个唯一的12位数字字母序列，其中第1到第6位表示省级行政区划代码（直辖市用两位表示），第7到第12位代表一个城镇或乡镇的庄/村的街道门牌号码。

## 发展历程及特点
古代中国没有统一的 postal code system ，各地均根据自己的风俗习惯、特有的区域规划，创造了各自的邮政编码系统。不同的地方使用不同的编码标准，如浙江省使用的是“三级”邮编——前两位代表省份，中间一位代表城市，最后四位代表区县；而上海、广东等直辖市则只使用“二级”邮编，前两位代表省份，后四位代表区县。如今邮政编码系统已形成全国共识，各地均使用“五级”邮编——前两位代表省、市、自治区、计划单列市或特别行政区的代码，中间三位代表乡镇、街道或其他城乡单位的标识，最后四位表示小区、单元、住宅楼、大厦等的编号。目前全国范围内共计18个不同级别的邮编制度，在一定程度上已经成为中国的地理位置编码方案。

一般来说，邮政编码不仅可以用来方便快捷地寻找生活物件，还可以作为证明身份、文件归属等方面的重要参考依据。但随着网络、移动支付等新兴技术的发展，越来越多的人开始把注意力集中到纸质信件上面。因此，对于保障个人信息安全和资产交易权益，政府正在加强对邮政编码的管理，确保邮政编码属于个人隐私，从而逐步减少对邮政编码的依赖。同时，为了更好地推动邮政编码科技的发展，国家也提出了相应的标准、规范以及保护措施。

# 2.Postal Code分类
根据邮政编码的层次结构，我们可以将其分为以下几种类型：

1. 二级邮编（又称“区号”）: 使用这种类型的邮政编码，只需要填写前两位即可。例如：山西省太原市的邮政编码为：101090，黑龙江省齐齐哈尔市的邮政编码为：230104。
2. 五级邮编: 这种邮政编码有较高的信息复杂度。包括两个编码区域——省级、市级、乡镇级和村级编码区域，以及三个编制区域——镇、村、组。这种邮编具有很强的空间、功能、效率、速度、节约电费、折扣优惠的特点。目前世界上很多国家都采用这种邮编，例如日本、韩国、台湾、香港、澳门等。例如，香港特别行政区的邮政编码是：HGK 10218，台湾省的邮政编码是：310012。
3. “编制区”邮编：这类邮政编码相对于传统的邮编，有着更高的可靠性和完整性。一般情况下，“编制区”邮编的注册由当地的地方当局或组织负责。并不需要居民在网上提交申请，就可以获得相关邮政编码。例如，澳大利亚、菲律宾等地就使用“编制区”邮编作为国际收发邮件、速递服务的重要载体。
4. 直筒“区号”邮编：这种邮政编码是最早出现的。它主要用于社区团购、外卖、网上商城等场景。这种邮编的命名方式比较简单，只需要填写后六位，便于收件人快速识别。例如，宁波市海曙区的邮政编码为：330100，海南省临高县的邮政编码为：462010。
5. 海关申报编码：这种邮政编码是为了满足海关对收发货物的要求而设立的。这些编码按规定的格式采用字母数字组合，以区分收发货人的身份信息，并进行各种监管、清关工作。海关会通过邮政系统获取该编码进行查询、验证，并对收货货物进行检查、评估。

# 3.Postal Code解析规则
按照中国邮政编码系统的规则，每一级的编码由一至多个字符表示。汉字、英文字母、数字混合组成，长度不限。除最后一级外，每一级编码都由“-”隔开。所有的编码都不包含空格、点号、斜线等特殊符号。下表给出了中国常见的邮政编码的解析规则。

| 编码层级 | 描述          | 解析规则                                  | 示例                                    |
|---------|---------------|------------------------------------------|----------------------------------------|
| 一级     | 直辖市         | 两位数字                                   | 01           /   20            |
| 二级     | 省级市（直辖市）   | 两个英文字母+四位数字                     | 京 /   淮                      |
| 三级     | 城市           | 两个英文字母+三位数字                     | 北京 /  苏州                   |
| 四级     | 区县           | 26个省级直辖市或36个省级非直辖市、市辖区级的直辖市、非直辖市的市、区县选择的不同，总共38个区县       | （内蒙古）包头市区 / 云南省德宏傣族景颇族自治州瑞丽市      |
| 五级     | 乡镇乡 / 街道乡   | 在一、二级基础上，添加乡镇的字样，然后在其后添加四至七个字母或数字，取决于所处的街道的长度。在乡镇的第一个字母之后，必须加上“乡”，第二个字母后面可以是“镇”，也可以是“乡”。如果街道名中已经含有“镇”字样，则须删除掉。例如：天津市东丽区孝陵卫街道三里坊二号路10号 / 重庆市渝北区沙坂坡镇赤土坎村 / 四川省乐山市沿河街道红星村 |
