## 二、相关概念及术语
### 2.1 Plotly
#### 2.1.1 Plotly简介
Plotly是一个基于web的开源可视化库，主要用于可视化数据的分布、变化趋势、模式识别等。其主要特点如下：

1. 可视化：Plotly提供了丰富的图表类型，包括散点图、折线图、条形图、直方图、地图、气泡图等。你可以自由选择所需的图表类型，调整样式参数，满足多种场景下的需求。
2. 直观：Plotly的图表设计采用专业的可视化理论，可轻松看出数据分布的特征及规律。无论是对比还是分布图，都可以使用颜色、大小、形状等属性直观呈现数据之间的联系。
3. 交互式：Plotly提供了多种交互方式，如滑动缩放、标注点、交叉拖放等，使得用户可以灵活地操作数据。同时，Plotly还提供了高级分析功能，允许你绘制数据热力图、矩阵图、箱线图等。
4. 支持移动端：Plotly支持手机、平板电脑、智能显示器、VR设备等多个平台，无缝适配各种屏幕尺寸和分辨率。
5. 插件生态系统：Plotly提供了丰富的插件生态系统，你可以搜索到各种各样的图表插件，比如Sankey图、嵌入式图表等。这些插件都可以免费下载或购买，帮助你进一步丰富你的可视化作品。
6. 数据接口：Plotly提供了一个Python API，你可以用Python脚本调用API完成数据可视化工作。除了官方的Python库外，还有一些第三方库也能够帮助你实现更多复杂的可视化效果。
7. 支持多种编程语言：Plotly目前支持R、JavaScript、MATLAB、Julia、Swift、C++等多种编程语言。通过官方的API，你可以快速地把可视化效果部署到各种环境下。

#### 2.1.2 Plotly术语
* **Figure**：绘图对象，用于承载可视化信息。
* **Trace**：Trace对象，用于承载可视化数据。每个图表可能由多个Trace对象组合而成。
* **Data**：数据源，用于存储图表所需的数据。
* **Layout**：布局设置，用于设置图表整体的样式。
* **Axis**：坐标轴设置，用于设置图表坐标轴的显示范围、刻度值等。
* **Annotation**：注解设置，用于添加注释、标注等内容。
* **Colorscale**：色带设置，用于设置颜色的取值范围及相邻颜色的渐变情况。
* **Marker**：标记设置，用于设置图表中的数据点样式。
* **Font**：字体设置，用于设置字体样式。
* **Dash Patterns**：虚线设置，用于设置线型。
* **Updatemenus**：更新菜单设置，用于控制图表的更新频率和更新方式。
* **Sliders**：滑块设置，用于控制图表的显示范围。
* **Frames**：帧设置，用于对动画效果进行配置。
* **Angular Axes**：角度轴设置，用于创建三维图表。
