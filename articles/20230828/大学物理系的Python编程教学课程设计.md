# 1.简介
  

## 概述

这是一门关于用Python进行工程物理科学计算的课程。本课程不涉及太多高深的物理知识，只会涉及一些基础的物理常识。相信通过这个课程学习的人也不会觉得枯燥乏味，只会轻松愉快。

## 主要内容

* Python语言入门
    * 安装Python环境
    * 数据类型、变量、运算符、控制语句
    * 函数定义、调用和模块导入
* NumPy库
    * 创建数组
    * 使用NumPy数组进行计算
    * 生成随机数组并对其进行处理
* Matplotlib库
    * 绘制图像
    * 使用颜色设置线条图例等功能
* Scikit-learn库
    * 数据集分割
    * 模型拟合
    * 模型评估
* TensorFlow库
    * 深度学习模型搭建
    * 模型训练及评估
