# 1.简介
  

在中国，知识产权保护是一个充满了挑战的事情。近年来，随着知识产权法律法规的不断更新、监管力度的加强以及社会的消费升级需求的提升，知识产权保护也越来越受到社会各界的关注。同时，由于传统商业模式的缺陷，知识产权市场仍然处于萎靡状态。因此，如何通过合理的政策配置，建立知识产权保护机制，并对知识产权者提供公平的、透明的、可控的服务，已经成为共同面临的问题。

本文将从知识产权的定义、保护原则、市场竞争规则以及知识产权保护市场的整体状况等方面介绍知识产权保护领域的基本情况。随后，将结合实际案例，阐述知识产权保护机制的构建、保障和运营策略，并分析当前国内外相关的知识产权保护事件，评估其成效与效果，给出建议。最后，结合政府的宏观调控政策，展望未来国家层面的知识产权保护机制建设。

# 2.知识产权的定义、保护原则和市场规则
## 2.1 知识产权的定义
“知识产权”是指作品（包括文本、图片、音频、视频、软件、数据库、绘画作品等）的创作者所拥有的财产权益，而这些作品成为特定经济利益群体的商品、服务或收入来源的依据。按照马克思主义哲学观点，知识就是人类的智慧，是客观存在并且只能由人类才能认识、继承、开发和利用的价值。任何知识产权都是为了达成某种经济利益或社会价值的创造性发明，它具有不可分割性、自主权和不可撤销性。知识产权属于所有权，是创作者对作品进行创造的独占财产，不因创作的完成而转移。

## 2.2 知识产权保护原则
* 原创性：只要是作者创造，就没有版权；对他人的成果，除了作者同意，其他人都无权复制、再创作。
* 知情权：所有权人对他人作品的所有权、使用权或者其他权利享有知情权。
* 使用限制：禁止盗用、非法转让、隐匿使用和侵犯著作权。
* 尊重价值判断：尊重作品的价值判断和创新意识，对不同价值的作品予以区别对待。
* 可维权：任何人均可向权利人举证自己的权利，请求恢复被侵权作品。

## 2.3 知识产权市场规则
1. 保护原创性：对创造性的研究成果，不得作为市场上的商品卖掉。
2. 保护重要作品：重要作品的创作者必须获得知识产权保护，否则，他/她不得购买该作品。
3. 保护版权主张人：知识产权所有者必须获得被申请的作品的所有权，或者请求其他权利人授权其使用其作品。
4. 优先保护版权：公众认为重要的作品拥有最高的版权权益，并且可以依法行使版权。
5. 保护中长期价值：如果一个作品曾经在商业上获利，那么它的版权义务结束后，其价值就减少了，也就无须担保了。
6. 保护相对完整的作品：不可能保护绝对的全文检索能力，但可以在大量使用之后保护重要作品。
7. 活跃的市场环境：需要密切关注并参与知识产权保护活动的人员以及区域的变化。
8. 严厉打击盗版行为：对有损害版权的盗版行为，应严惩凶手。

## 2.4 知识产权保护市场的整体状况
目前，知识产权保护市场已经形成了一个庞大的生态系统，涵盖多个环节及其参与者。

* 制度方面
  * 各级部门、政府部门逐步制定相应的知识产权法律法规。
  * 地方政府普遍推进知识产权法律法规的落实工作。
  * 有关机构如媒体、出版社、图书馆、网吧等也纷纷制定知识产权宣传、管理等方面的政策。
* 组织结构方面
  * 发达国家的知识产权局负责知识产权事务的日常管理和协调。
  * 在一些发达国家，如美国、英国等，还存在着知识产权监管机构。
  * 许多大学、科研机构、公司、政府部门都会开展知识产权保护宣讲会、培训讲座等活动。
* 产品和服务方面
  * 知识产权的产品和服务如教育软件、医疗器械、数字出版物等已经遍布于生活领域。
  * 政府部门积极布局知识产权领域，推动各类创新项目的研发与落地。
  * 电影、音乐、游戏等艺术类作品的创作者们也要求能够保护自己的创作。
* 技术工具方面
  * 在知识产权领域有非常成熟的技术保护工具，如版权保护软件、版权认证工具、专利交易平台等。
  * 浏览器插件可以自动识别网站是否存在侵权信息，并根据用户的选择采取不同的措施，如弹窗警告、跳转链接等。
* 环境方面
  * 大型跨国企业、创业公司在知识产权保护领域投资很大，尤其是在新兴市场。
  * 许多大学的开设的“创新创业计划”，吸引着大批创业者的加入。

# 3.当前国内外相关的知识产权保护事件
## 3.1 最近十年的知识产权保护事件
### 3.1.1 中国共产党创建者马列宁格勒佛教寺院倒塌
马列宁格勒佛教寺院于1978年5月1日在中国共产党的领导下倒塌。为此，红卫兵集体上街抗议，要求修复寺庙墓园，拆除窝棚，烧毁寺庙遗址等。
当时，中共曾多次强调知识产权的重要性，要求全党共同保护这类珍贵的民族文化遗产。1982年国务院颁布了《关于鼓励创新和推进科技产业发展若干政策的决议》，其中就明确规定：“凡是实施侵害著作权、商标权、专利权、版权等知识产权的行为，或者产生侵害著作权、商标权、专利权、版权等知识产权的风险的，由有关部门给予行政处罚，给予违法罚款。”。但是，对马列宁格勒佛教寺院的国际声誉和影响力，仍然表示诧异。
1993年，马列宁格勒佛教寺院对外宣布将关闭，国际上不少媒体报道称，世界上很多国家都发现了这个美丽的寺庙，表示了对该文化遗产的敬仰和钦佩。中国政府还出台了一系列政策鼓励公民对侵犯知识产权行为提起诉讼。
2008年，香港一代英雄张铭宇在马列宁格勒佛教寺院的墓前高喊“共和国的光荣！”，悼念该寺庙。随后，多国政客发表演说鼓舞士气，声援中国共产党人。2010年，马列宁格勒佛教寺院又发生火灾，令当时的中共总书记胡锦涛极为愤怒。当天下午，他亲笔写信给国务院，要求严惩此案。
2013年，总书记提出的《关于严肃处理侵害知识产权行为问题的通知》对此案作出了最终判决，并于2014年1月1号正式发布。一审中，有关部门指控寺院倒塌和火灾两件事情的嫌疑人黄先韵系明星的私服出现了问题，于是在未经证实的情况下，将案件移交给法院审理。法院认为，黄先韵系因明星私服问题购买的产品的授权已终止，故侵权行为不存在，对他的个人权利不承担任何责任。判决书规定，任何单位、个人不得实施侵害他人知识产权行为，并对被告人已收回侵权作品的权利追究其法律责任。
虽然当时的中国共产党还是历来重视知识产权，而且宣扬、保护知识产权的主张没有改变，但是，这场运动之后，随着执政党内部改革，知识产权保护方式的转变显现出来。

### 3.1.2 唐山大地震
唐山大地震于2008年10月3日晨发生，震中距今约1万公里。位于四川省北部、贵州省东南部、云南省西南部、甘肃省东北部、青海省西北部的广大地区和地理位置发生交通线路及通信网络瘫痪，多条铁路和航空线路均无法通车，酒店、旅游业遭受重创。此次震波带来的经济损失是巨大的。据媒体报道，中国官方至今未向国际社会做出回应。
2010年，国务院召开第八届人大一次会议，以依法对唐山大地震及政府职能部门职责作出更严格的监督检查，包括清理问题矛盾、完善工作制度、开展反腐倡廉工作。2011年，国务院决定，根据国家有关法律法规的规定，对中共中央纪委发出通知，撤销对唐山大地震的纪律处分决定，恢复唐山地区正常国内秩序。
2012年初，中共中央、国务院又召开了全国三次知识产权工作座谈会，会议提出，推进健全知识产权领域的政策导向，健全知识产权保护体系，突破信息网络时代的知识产权保护壁垒。2013年4月，在京召开的第三届中国知识产权大会上，习近平主席题词赞扬知识产权领域取得的辉煌成果，强调知识产权保护具有十分重要的社会公共利益。

### 3.1.3 “郭文贵”事件
“郭文贵”是一个2013年底在新浪微博上发表自杀言论的用户。他发表的内容主要是：上周末，我和一个朋友在高铁上听到了郭威这样的话："你的前途就像一把钥匙，我的生命就像一段插曲，打开它，我就可以逆袭成功……"这条微博引发热议，被部分人转发到微信，并被删除。网友爆料，之前几乎同一时间，另一个新浪微博账号也发表类似的言论，引发热议。其中一名叫做“陈启元”的网友用自己的微博回复了一条话：郭文贵是我三好学生，是他俩的未来，我希望永远和他保持联系……而他的博文似乎没有显示出这种情感。目前，消息还未完全解读出来，有必要全盘调查一下。
2014年5月28日，新浪微博张铁志在其微博下评论，认为郭文贵的自杀构成严重侵犯了他人合法权益，这是一项侵权行为。他说：“在互联网上发表侮辱、诽谤、淫秽等不实信息，造成的后果严重、昭然若揭，也引起了广泛关注。”在微博上公布了李彦宏、蔡奇、马云等高官及媒体人的相片，揭露郭文贵在私生活中的丑陋、残忍、邪恶。
2014年6月10日，新浪微博张铁志在其微博下转发，称“郭文贵的账号出现了恶意刷屏的现象”，并呼吁网民配合政府部门进行核查调查。截至6月15日，网络上持续流传郭文贵微博张铁志的删帖、评论等信息。
2014年8月27日，中共中央、国务院办公厅、新闻办联合印发《关于严肃查处郭文贵先生等伪装“郭威”而发表严重侮辱诽谤言论的通知》。《通知》明确指出，郭文贵先生及其影响力范围广泛的个人及团体，利用网络谋取不正当利益，制作虚假宣传谎言等行为，严重侵犯其合法权益，必须依法严肃查处。
2014年9月3日，国家有关部门立即启动全面调查。2014年11月2日，国家有关部门对郭文贵先生作出了解释，并退赔其因公殉职所得税。2015年1月23日，国家有关部门宣布，撤销对郭文贵先生的追究。
2015年3月1日，福建省政府书记处书记王超华率队到郭文贵家中探访。王书记介绍，郭文贵生于一九五六年，曾任上海市青浦县委书记，参与组建城市经济特区。抗战爆发后，郭文贵任河南省安阳市经济委员会副主任兼住房保障处处长。抗战胜利后，郭文贵率部投奔解放军。后又兼任上海市市长、国家工商总局局长。
2015年7月2日，党中央、国务院发出《关于严肃查处郭文贵先生等伪装“郭威”而发表严重侮辱诽谤言论的紧急通知》。《通知》强调，郭文贵先生及其影响力范围广泛的个人及团体，利用网络谋取不正当利益，制作虚假宣传谎言等行为，严重侵犯其合法权益，必须依法严肃查处，采取取缔、关闭、涉嫌渎职等有效措施。
2015年11月25日，中共中央纪委对郭文贵先生严肃查处情况作出答辩，指出，郭文贵先生的行为构成严重侵害他人合法权益，且将侮辱、诽谤、淫秽言论刊登在他人微博、微信公众号等平台上，情节较为严重，需要依法严肃处理。对于这一事件，郭文贵先生既没有过错也没有违法行为，必须严肃追究其法律责任，严格依法惩戒，坚决制止，切实保障公民合法权益，维护社会公平正义，将郭文贵先生列入相关职务人员黑名单，并依法依规采取法律手段。
