# 2.基本概念术语说明
## 2.1 概念和术语
- AI（Artificial Intelligence）：人工智能的缩写，泛指计算机智能的某一领域。它是由一系列的研究和发明所构成的科学，可以让计算机具有智能、自动化、自我学习、决策支持等功能，促进人类智慧的增长。
- ML（Machine Learning）：机器学习的英文名称，是一类人工智能技术，涉及通过训练计算机对数据的分析和预测，以此来改善系统行为、进行控制或预测结果的准确性。
- DL（Deep Learning）：深度学习的英文名称，一种以人工神经网络为代表的机器学习方法，也是现代机器学习的主要研究领域之一。深度学习可以对输入的数据进行非线性变换，使得模型能够基于数据的复杂结构进行抽象，并能够学习到丰富的模式，有效地进行预测和决策。
- NLP（Natural Language Processing）：自然语言处理的缩写，是指利用计算机科学、统计学、信息论、自动机理论、人工智能理论以及语言学等学科的方法，对文本、语音、图片、视频等形式的语言进行转换、分析、储存、归纳、组织和表达的一门技术。
- CV（Computer Vision）：计算机视觉的英文名称，是指研究计算机如何获取、处理、存储以及解释图像、声音和视频等图形、声音、影像等媒体信息的理论、方法、技术及应用的一门学科。
- RS（Recommender Systems）：推荐系统的英文名称，是指借助人工智能技术帮助用户选择感兴趣的信息而制定的推荐系统模型。它通过分析用户偏好的历史记录、商品特征、上下文环境等信息，为用户提供个性化的商品推荐。
- CNN（Convolutional Neural Networks）：卷积神经网络的简称，是一类深度学习模型，特别适用于处理二维图像数据，是近些年非常火爆的图像识别技术。CNN 的卷积层和池化层构成了特征提取器，可以提取局部、全局及整体特征，再通过全连接层输出最终的分类或回归结果。
- RNN（Recurrent Neural Networks）：循环神经网络的简称，是一类深度学习模型，特别适用于处理序列数据，如文本、语音、时间序列等。RNN 通过将上一次计算的状态反馈给下一次计算，可以记住之前发生过的事情并影响当前的计算。
- LSTM（Long Short-Term Memory）：长短期记忆神经网络的缩写，是一种特殊的 RNN，可在内存中保存并维护信息传递过程中的中间状态。LSTM 在一定程度上能够克服门限反向传播算法中梯度消失的问题。
- GRU（Gated Recurrent Unit）：门控循环单元的缩写，是一种特殊的 RNN，与 LSTM 有很多相似之处。GRU 可以在计算过程中保持记忆细胞和遗忘细胞之间的平衡，从而改进 RNN 的性能。
- SVM（Support Vector Machine）：支持向量机（SVM）的英文名称，是一个著名的二分类和多分类的机器学习算法，可以有效地解决分类、回归和异常检测等问题。它是核函数的集合，通过最大化间隔边界或最近邻边界的距离，从而找到最优解。
- KNN（K-Nearest Neighbors）：K近邻算法的缩写，是一种简单有效的非线性分类算法。该算法会把测试样本划分成不同的类别，对于每个测试样本，选取其最近的 k 个邻居，然后根据这 k 个邻居所属的类别决定测试样本的类别。
- PCA（Principal Component Analysis）：主成分分析的英文名称，是一种无监督、无标记的降维技术，通过分析数据集的协方差矩阵或者相关系数矩阵，找寻其中的低纬空间。PCA 会找到数据集中最大方差的方向，并将其他方向上的投影压缩到这个方向上。
- KDTree（k-Dimensional Tree）：KD树的英文名称，是一种数据结构，它是一种对数据点进行快速检索的空间索引数据结构。KD树的构造分两步：第一步是递归地建立两个子结点，从根节点开始。第二步是分别对两个子结点的数据点进行排序，按照第 i 个坐标轴的大小排列。通过这样的排序，就可以把数据集切割成两个子集，并找到他们的中点作为分割点。
- TF-IDF（Term Frequency-Inverse Document Frequency）：TF-IDF 的英文名称，即 Term Frequency-Inverse Document Frequency。它的目的是通过词频(Term Frequency)和逆文档频率(Inverse Document Frequency)两个指标评估一个词是否重要。词频表示某个词在一篇文章中出现的次数，逆文档频率表示单词很少出现在整个文档库中的可能性。TF-IDF 是一种常用的文本挖掘技术，能有效地筛选出关键词、提高分类效果。
