# 2.基本概念及术语
## 2.1.什么是黑客攻击？
黑客攻击（Hacking），英文名称为hacker，是指利用计算机系统漏洞对网络、服务器等设备进行恶意控制和数据窃取，并通过一定手段获取系统的管理权限的行为。黑客攻击是计算机安全领域的一个重要方面，它影响着公司、政府、学校甚至普通百姓的正常工作和生活。它的目的就是为了获取机密文件、破坏网络设施、对用户进行非法侵入、泄露隐私信息等，这也使黑客攻击成为社会经济活动中的一项重要基础设施。
## 2.2.什么是社会工程学攻击？
社会工程学攻击（Social Engineering Attack）是黑客攻击中的一种常用手段。所谓社会工程学攻击，是指黑客攻击人员通过一些技巧把对方诱骗进入指定联系人的情境，让目标人物采取某种行动或者做出某些举动。比如，当黑客攻击者通过电子邮件、短信或其他方式给目标企业发送恶意的招聘信息时，往往可以获得对方青睐。此外，也存在一些基于社交媒体的攻击方式，比如使用社交工程学攻击向某公司发布虚假广告信息，通过吸引受害者的注意力来获取他们的认同感和支持。社会工程学攻击的前提是合法的渠道。通常情况下，一般只有良好的管理者才会允许使用社会工程学攻击。而对于那些明知故犯或者不明真相的黑客，就可能会被利用来盈利或者获利。所以，在社会上要小心对待那些可能导致你的财产损失的人！
## 2.3.什么是漏洞（Vulnerability）？
漏洞（Vulnerability）是指计算机系统或者网络中存在的一些弱点，这些弱点可以通过攻击者利用它们实施恶意行为。例如，网站登录页面上的密码泄露，存在数据库表结构和SQL注入漏洞，服务器上的反垃圾邮件功能缺陷等等，都是容易被攻击者利用的漏洞。漏洞可以是显著的、隐蔽的，也可以是比较复杂的，并且在不同系统中都可能存在相同的漏洞。
## 2.4.什么是敌我识别（Adversarial Reconnaissance）？
敌我识别（Adversarial Reconnaissance）是指黑客攻击者获取他人信息的一项技术，如收集个人身份信息、浏览记录、社交关系、信用卡号码、银行账号、住址等。通过这一技术，黑客攻击者可以挖掘到有价值的信息，包括组织架构、人员信息、关键项目、人脉关系等。这可以帮助他人了解他人想要什么，从而针对性地策划、投放攻击工具。
## 2.5.什么是边界网关协议（Border Gateway Protocol，BGP）？
边界网关协议（Border Gateway Protocol，BGP）是互联网边界路由器之间通信协议，它用于路由选择，将流量导向最近的可用路由。通过BGP，互联网上的所有路由器都会保持及时的、准确的路由表，以最低的延迟和最佳带宽传送网络流量。BGP最大的优点就是保证了互联网上流量的快速、可靠的传输。但是，它也是黑客攻击者分析网络的又一利器，因为黑客可以通过BGP的相关数据，分析整个互联网的数据流向、规模以及运营商等信息。
## 2.6.什么是中间人攻击？
中间人攻击（Man-in-the-Middle attack）是指攻击者在两端系统之间插入第三方代理， intercepts and alters network communication between the two systems to redirect or tamper with the traffic flow. It is a common type of hacking where an attacker poses as both the victim and the server by intercepting and modifying messages sent from the client machine to the server machine, without their knowledge. A man-in-the-middle attacker can steal sensitive information like passwords or credit card details, as well as browse records, emails and other private data transmitted through the internet. The most effective ways to defend against this kind of attack are:
* Use secure VPN tunnels when connecting to remote servers
* Regularly update software and antivirus programs on your computer
* Perform regular system backups and monitoring for intrusion detection tools
