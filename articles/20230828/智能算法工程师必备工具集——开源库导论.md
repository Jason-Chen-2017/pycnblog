# 2.Python基础
## 2.1 Python语言特性
### 2.1.1 变量
```python
a = 1
b = 'hello'
c = True
d = [1, 2, 3]
e = {'name': 'Alice', 'age': 25}
f = None
```
可以看到，在Python中，变量类型并不固定，只要赋予不同的赋值方式即可，不同类型的变量可以相互转换。如`a`为整型，则可以把它赋给另一个变量作为浮点型，反之亦然；`b`、`d`、`e`均为可变类型（Mutable）的变量，即可以添加或者删除元素；而`f`为NoneType的单个对象，不能更改。

还有一种特殊的变量——集合(set)，跟字典类似但不允许有重复值。可以通过set()函数或{}创建。

除此之外，还存在元组tuple、列表list、字符串str、数字int、布尔bool等，这几个基本类型都可以直接使用，不需要任何声明。

### 2.1.2 数据结构
#### 2.1.2.1 列表 List
列表是最常用的序列数据结构，可以用来存储多个值的集合。它的定义形式如下：
```python
my_list = [1, "apple", False, ["orange"]]
```
列表中的每个元素都有序排列，通过索引值可以访问对应的元素。列表支持以下几种操作：

1. `len()` 函数用于获取列表的长度。
2. `append()` 方法用于在列表末尾追加元素。
3. `insert()` 方法用于在指定位置插入元素。
4. `pop()` 方法用于删除指定位置的元素，默认弹出最后一个元素。
5. `remove()` 方法用于删除第一个出现的指定元素。
6. `sort()` 方法用于排序。
7. `reverse()` 方法用于逆序。

列表还有切片操作，通过下标来访问元素的子集。

#### 2.1.2.2 元组 Tuple
元组是一个不可变的序列数据结构，定义形式如下：
```python
my_tuple = (1, "apple", False)
```
元组中的元素也有序排列，但不能修改。元组支持切片操作。

#### 2.1.2.3 字典 Dict
字典（Dictionary）是另一种映射类型的数据结构，可以用来存储键值对（Key-Value Pair）。其中，键（Key）必须是不可变类型（比如整数、浮点数、字符串、元组），而值（Value）可以是任意类型。

字典的定义形式如下：
```python
my_dict = {"name": "Alice", "age": 25, "city": "Beijing"}
```
字典的每一个键值对用冒号(:)分割，键和值中间用等号(=)表示，不同键值对之间用逗号(,)隔开。

字典支持以下几种操作：

1. `keys()` 方法用于返回字典所有键的集合。
2. `values()` 方法用于返回字典所有值得集合。
3. `items()` 方法用于返回字典的所有键值对的集合。
4. `get()` 方法用于根据键查找对应的值。
5. `in` 操作符用于判断某个键是否在字典中。
6. `update()` 方法用于更新字典。

字典还有其他的方法，包括迭代、弹性大小调整、清空、合并、拷贝等。

#### 2.1.2.4 集合 Set
集合（Set）也是一种数据结构，可以用来存储一组不重复的值。集合的定义形式如下：
```python
my_set = {1, "apple", False}
```
集合的元素不重复且无序排列，因此不能通过索引访问元素。但是，可以使用for循环来遍历集合中的元素。

集合的其他操作与列表、字典相同，如：

1. add()方法添加元素。
2. remove()方法删除元素。
3. union()方法返回两个集合的并集。
4. intersection()方法返回两个集合的交集。
5. difference()方法返回两个集合的差集。

### 2.1.3 if...else语句
if...else语句用于条件控制，语法如下：
```python
if condition:
    # do something
elif other_condition:
    # do some else
else:
    # do default action
```
如果表达式condition成立，则执行第一个块内的代码；否则检查第二个条件other_condition，如果它成立，则执行第四个块内的代码；否则执行第三个块内的代码。

### 2.1.4 for...in语句
for...in语句用于遍历序列（List、Tuple、Dict、Set等）或迭代器对象，语法如下：
```python
for variable in sequence:
    # do something with the element of the sequence/iterator
```
其中variable代表每次迭代获得的元素，sequence代表待遍历的序列或迭代器。for...in语句一般配合range()函数一起使用，用来产生一个整数序列。

```python
for i in range(10):
    print(i)
```

上述代码输出1到9。

### 2.1.5 while...else语句
while...else语句用于条件循环，直到表达式condition不再满足为止，语法如下：
```python
while expression:
    # do something repeatedly until the expression becomes false
else:
    # execute this block after the loop has been exhausted without encountering a break statement
```
如果表达式expression成立，则执行第一个块内的代码；否则执行第二个块内的代码。

### 2.1.6 函数 Function
函数是组织好的，可重复使用的，用来实现特定功能的代码段。函数通常由输入参数（input parameter）、输出结果（output result）以及函数体（function body）三部分构成。函数的定义格式如下：
```python
def function_name(parameter1, parameter2,...):
    """docstring"""
    # code blocks that implement the functionality of the function
    
```
函数的输入参数可以有多个，但是只能有一个返回值，返回值的类型可以根据实际需要设定。函数可以通过return关键字返回结果，也可以不返回结果，这种情况下，返回值为None。

函数的文档字符串（docstring）用于描述函数的功能。

函数调用时，传入相应的参数即可，函数内部的代码块将被自动执行。

### 2.1.7 模块 Module
模块是面向对象的程序的基本单位。模块可以包含函数、类、常量、变量和其他模块。模块在文件中定义，后缀名为`.py`。模块的导入与导包（import/export）是组织代码的重要方式。

模块的导入格式如下：
```python
import module_name as alias_name
```
这里的module_name表示要导入的模块名，alias_name表示模块别名。如果没有设置别名，则默认与模块名相同。

模块的导包格式如下：
```python
from module_name import *
```
这样就可以直接使用该模块中的函数或类。

# 3.NumPy库
## 3.1 NumPy简介
NumPy（Numeric Python）是一个用于数组计算的 Python 库。主要提供了矩阵运算、线性代数、随机数生成等功能。

NumPy 也叫做 Numpy，是一个开源的 Python 科学计算扩展包。Numpy 是 Python 的一个扩展程序库，支持 large array 和 linear algebra on arrays，同样也提供了矩阵运算、随机数生成等数学功能。

NumPy 支持 C、C++、Fortran 编程语言，其接口统一纯 Python API，方便用户进行数组运算。

NumPy 在2006年由初识洛伦兹（Lawrence Livermore National Laboratory）开发，目的是为了解决Python在科学计算方面的缺陷。

## 3.2 NumPy基本使用
### 3.2.1 创建数组
NumPy 提供了多种创建数组的方式：
- 使用已有的数据创建数组（Array creation using existing data）：从现有的数组或列表创建数组，例如，从列表创建一个二维数组。
- 从数组的元素构造一个新数组（Constructing an array from elements of another array）：创建一个新的数组，初始化为另一个数组的元素。
- 根据数组的范围来创建数组（Creating arrays based on ranges）：根据数组的范围创建数组，例如，创建从0到9的整数序列数组。
- 通过用特定值填充数组来创建数组（Creating arrays by filling it with specific values）：创建一个全为指定值的数组。
- 从随机分布中创建数组（Creating arrays from random distributions）：通过给定的概率密度函数，从指定的随机分布中创建数组。

```python
import numpy as np

arr = np.array([1, 2, 3])   # create a rank 1 array
print("Array is:", arr)

arr = np.array([[1,2], [3,4]])   # create a rank 2 array
print("Array is:\n", arr)

# Create a 3D array from a list of lists of lists
lst = [[[1,2],[3,4]],[[5,6],[7,8]]]
arr = np.array(lst)
print("3D Array is:\n", arr)

# Use the zeros() method to create an array of all zeroes
arr = np.zeros((2,2))
print("\nZeroes Array is:\n", arr)

# Use the ones() method to create an array of all ones
arr = np.ones((2,2), dtype=int)    # specifying datatype
print("\nOnes Array is:\n", arr)

# Using arange() method to generate sequences of numbers
arr = np.arange(start=1, stop=5, step=2)      # start inclusive and end exclusive
print("\narange Array is:", arr)

# Generate random number between 0 and 1 with uniform distribution
arr = np.random.rand(2,2)       # shape determines dimensions
print("\nRandom Array is:\n", arr)

# Generating random integers
arr = np.random.randint(low=1, high=5, size=(2,2))     # low inclusive and high exclusive
print("\nRandom Integer Array is:\n", arr)
```
