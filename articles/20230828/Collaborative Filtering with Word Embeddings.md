# 1.简介
  
：基于用户兴趣及其交互行为的协同过滤推荐系统，可利用用户输入的信息进行有效推荐。
# 2.基本概念：
    # User-based CF：对用户之间的相似度进行建模，将兴趣向量表示成用户对物品（商品）的评分矩阵。
    # Item-based CF：对物品之间的相似度进行建模，将兴趣向量表示成物品的特征矩阵，并用物品之间的距离衡量相似度。
    # Latent factors：隐性因子，在协同过滤算法中，主要处理两个用户、两个物品间的差异性。
    # Neighborhood of users and items：邻域用户和邻域物品指的是与目标用户或物品有某种共同兴趣的人群。
    # Preference profile：偏好表达，是用户对某个特定物品的程度打分。
    # Rating matrix：评分矩阵是表示用户偏好的一种矩阵，它由三个部分组成：用户ID、物品ID、对应评分值。
    # TF-IDF：Term Frequency-Inverse Document Frequency，是一种用于信息检索与文本挖掘的加权统计方法。
# 3.核心算法：
    ## SVD（Singular Value Decomposition）
    Singular Value Decomposition(SVD)是一个矩阵分解的方法，它可以将一个矩阵分解成三个矩阵的乘积，其中任意两个矩阵都没有重复元素。SVD的基本思想是将待求的矩阵A从物品空间转换到用户空间，即A_hat = U * Sigma * V^T。U是用户矩阵，V是物品矩阵，Sigma是奇异值矩阵，它是一个对角阵，每一对角元的值都是各个奇异值的平方根，负号表示排除这些奇异值中的正交部分。Sigma的大小决定了多少维度被保留，比如k=10则保留前10个奇异值对应的维度。此外，还可以应用L2归一化对奇异值矩阵Sigma进行正则化，得到：A_hat = (U * L2) * Lambda * V^T。Lambda是对角阵，每一对角元的值都是各个奇异值的倒数，负号表示这些奇异值是需要被截断的。
    
    通过SVD，可以计算出物品之间的余弦相似度，并根据相似度进行推荐。

    ## Nonnegative Matrix Factorization （NMF）
    NMF是一个矩阵分解的方法，它可以将一张庞大的矩阵分解成较小的几个奇异值矩阵的乘积，然后再恢复原来的矩阵。与SVD不同的是，NMF不要求奇异值矩阵Sigma为对角阵，也允许每个元素为负值。所以，NMF可以使用L1惩罚项正则化奇异值矩阵，使得奇异值保持非负值。
    
    NMF的主要思路是：找到用户矩阵和物品矩阵使得它们的重构误差最小化，这是通过设置损失函数来实现的。损失函数的计算涉及两个矩阵之间差距的度量，在这里，一般使用Frobenius norm。所以，NMF就可以用来发现用户的偏好和物品的特点。

    ## Word Embedding
    Word Embedding，中文叫词嵌入，是通过神经网络学习到的一套映射关系，将文本转化为固定长度的向量。对于每个词汇，词嵌入都可以获得一个唯一的高维空间中的坐标，这使得不同的词汇在这种高维空间中能够更加接近。词嵌入可以用来解决很多自然语言处理的问题，例如：词的相似度计算、文本聚类、文本分类等。
