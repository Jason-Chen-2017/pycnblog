# 2.基本概念
## 2.1 矩阵
矩阵（Matrix）是一种二维表格形式的数据结构，由若干个相同数据类型的元素排成规定的行和列的顺序构成。因此，矩阵是一个 n 阶方阵，其中 n 为矩阵的维数（行数或列数）。每个元素称为矩阵的一个元素，对应于矩阵中的一个坐标。如下图所示：

矩阵 A 的每行代表矩阵中一个观察者，每列代表了一个事件或变量，相应位置上的元素则记录了该观察者在该事件或变量上的观测值。比如，对于动物数据分析，矩阵 A 可以包含不同的动物，每行代表一个动物，每列代表一个观测量（如体重、长度、身高），相应位置上的元素则记录了各动物在这些观测量上的观测值。

## 2.2 稀疏矩阵
稀疏矩阵（Sparse Matrix）是指矩阵中的元素大多为零，其余元素很少出现非零值，即矩阵中存在许多值为零的元素。稀疏矩阵在很多实际应用场景下都有着广泛的应用。比如，在图像处理、文本分类、推荐系统等领域，图像和文本都是非常稀疏的，而在语义处理、生物信息分析等领域，有关基因表达、蛋白质结构的矩阵也往往是稀疏矩阵。 

稀疏矩阵有两种主要的类型：密集型稀疏矩阵和随机型稀疏矩阵。密集型稀疏矩阵（Dense Sparse Matrix）就是指矩阵中大部分元素都为零，只有极少数非零元素。随机型稀疏矩阵（Random Sparse Matrix）则是指矩阵中所有元素都不为零，但存在一些元素的值较小或较大。

## 2.3 度矩阵
当某个元素取值为 0 时，它就被称作是“度为 0”的元素，它的度为 0，意味着这个元素是没有对应的特征值的，所以可以看做是无用的特征。反之，如果某个元素的绝对值大于零，则它被称作是“度不为 0”的元素，它的度不为 0，意味着这个元素是有用的特征。这样就可以把矩阵分成两个子矩阵——具有非零度的子矩阵和度为 0 的子矩阵。

![image-2.png](attachment:image-2.png)

通过计算矩阵的不同子矩阵的范数，可以了解矩阵中非零元素的分布情况。范数又称矩阵的“尺度”，用于衡量矩阵的大小。最常用的几种范数包括 1 范数、2 范数、F 范数等。1 范数（Manhattan norm）衡量的是矩阵中绝对值之和，也就是说，把矩阵的所有元素都加起来，再求和。2 范数（Euclidean norm）衡量的是矩阵中元素平方和的平方根。

度矩阵（Degree Matrix）是指用来描述矩阵某些特征向量的角度的矩阵。例如，对于矩阵 A ，其第 i 个特征向量可以表示成 Av = (a1, a2,..., an)，而度矩阵 D = diag(ai) 则为：

D = [[a1, 0, 0], [0, a2, 0], [0, 0, an]]  

D 中的每一个对角元 ai 表示第 i 个特征向量的长度（对应于矩阵 Av 的范数），其绝对值越大，则表示该特征向量所包含的方向性越多。换句话说，特征向量越长，则表示其包含的信息越丰富。
