# 1.简介
         
　　随着人类社会对人身安全的关注，越来越多的人们对智能安全设备、产品的需求增加，而当前最流行的智能安防系统包括了视频监控、人脸识别等功能，其中人脸识别技术占据了绝大的市场份额。近年来，由于人脸识别技术已经取得重大突破，使得它成为了一种新兴的、高效且实用的安全系统保护手段。其本质就是通过摄像头拍摄到的图像或视频数据进行人脸识别，确定人物是否真实存在。如今，已有很多基于人脸识别的智能安防系统在实际生产中得到广泛应用。本文将探讨基于人脸识别的智能安防系统的发展现状、应用前景及其面临的挑战，并将结合作者自己的个人经验，阐述一些经典的基于人脸识别的智能安防系统的优缺点，指出这些系统需要进一步完善的地方。
# 2.人脸识别系统概述
　　人脸识别系统由三部分组成：摄像头（Video Camera）、图像采集模块（Image Acquisition Module）、人脸识别模块（Face Recognition Module）。摄像头负责获取图像信息，图像采集模块主要用于对图像进行处理，并输出预期的图像结果。人脸识别模块则根据图像采集模块的结果进行人脸特征提取和比较，从而判断出图像中是否存在人脸。

人脸识别系统通常分为两个层级结构，一是基于静态图像的人脸识别系统，二是基于实时视频流的人脸识别系统。

总体上，基于静态图像的人脸识别系统主要采用传统的方法，如Haar特征和机器学习分类器；而基于深度学习的人脸识别系统一般采用卷积神经网络或者人工特征点检测器。而实时视频流的人脸识别系统则主要采用滑动窗口检测、CNN框架检测、深度学习检测等方式。

人脸识别系统可以分为两种类型，一是单一类型，二是混合型。单一类型人脸识别系统仅仅针对某一特定目标人物进行识别，比如特定款手机的识别；而混合型人脸识别系统既可以识别特定目标人物也可以识别非特定目标人物，比如安防系统。
