# 1.简介
         
移动支付是一个颠覆性的创新领域，由一个公司（微信）推出第一个移动支付服务微信支付，到百度、支付宝、银联等支付机构的广泛落地，已经成为全球货币化社会不可或缺的一部分。移动支付的快速发展，也引起了移动支付行业的激烈竞争。然而，如何更好地发展移动支付市场、提升用户体验并使之能够产生长远影响力，则是移动支付领域经久不衰的话题。本文将着重介绍移动支付行业的市场竞争策略。
# 2.移动支付行业的市场结构
移动支付行业的经济规模占全球GDP的比例从2017年的5.1%降至2021年的1.2%，其规模将近三分之二以上来自中国。相比于固定电话支付、互联网支付，移动支付在传统支付方式上的劣势正在逐步显现。首先，由于人们普遍对电话的依赖程度高、同时移动通信网络连接速度也较慢，打电话是一种昂贵的消费行为，所以在移动支付领域中，用户习惯了在线支付，通过扫描二维码完成付款。其次，移动设备硬件配置较低，对系统性能要求较高，导致与传统支付方式一样，需要较多耗电量的手机APP。再者，移动支付的主要应用场景是在线交易，但实时交易会带来较大的运营成本，且交易额往往较固定电话支付高很多，因此实时交易越来越少出现。第三，目前全球移动支付平台数量仍处于较小水平，有些平台仅提供简单易用的支付功能，如支付宝，但对于复杂的业务场景，则需要依赖第三方支付机构的支付服务。总结来说，移动支付行业的市场结构特征为：

①移动支付的用户群较固定电话支付、互联网支付用户群更大、经济规模占全球GDP比例更高；
②移动支付平台数量有限，且各平台的功能及规模尚未统一，存在着功能差异、服务差异；
③移动支付领域的创新能力比较薄弱，主要靠服务商的市场占有率驱动；
④移动支付领域存在着长期竞争态势，市场份额与用户习惯、设备性能、软件质量等因素高度相关；
# 3.移动支付的竞争环境
移动支付市场竞争非常激烈，主要体现在以下几个方面：

①全球化：随着全球产业链的整合，移动支付在国际竞争中的主导地位日益加强。中国是全球移动支付领域的重要参与者，其手机App的知名度超过日本、欧洲等国家，同时，国内的移动支付领域有着巨大的自主权、主导权。
②垂直细分领域：移动支付领域的细分领域也在不断增多，包括支付通、滴滴等子领域，这极大的增加了竞争的难度。例如，滴滴的“打车”服务可以让用户享受到生活上的便利，但滴滴的整个生态系统并没有形成一个完整的、独立的竞争优势，这就使得其它类型的支付产品很难进入同类竞争。
③服务品牌市场：移动支付领域也是由服务商主导的市场。由于服务的品牌化特性，移动支付领域的平台和服务商都会在品牌推广上投入更多资源。移动支付服务的品牌价值越高，越容易被接受，从而实现垄断地位。在微信支付之后，支付宝的注册用户数已达到近亿，其注册用户所对应的支付宝账户占全球支付宝总用户数的35%，这已经成为中国支付宝的主流竞争者，但这也是一种“另类”竞争方式，因为它的服务层次不够完备，无法完全满足移动支付领域的需求。
④技术革命：移动支付领域的技术革命正在推动着这个行业的进步，如新一代的端到端加密技术、区块链技术等。虽然这些技术的进步带来了新的市场机遇，但由于国内市场发育不良、技术封锁等原因，使得这些技术在中国的应用还比较困难。
# 4.移动支付的市场竞争策略
移动支付行业的竞争策略有两种：

①垄断策略：通过控制手段压制竞争对手，例如垄断所有移动支付平台、垄断所有支付服务费用、压制特定的支付渠道。这种策略的缺点是不能充分发挥移动支付平台的资源、覆盖用户的多元需求。
②市场化策略：建立自己的支付系统，在服务上寻找优势，吸引用户。这种策略的优点是充分利用资源，吸引大量用户。但由于创新、竞争的局面，单独建设支付系统难度较大，市场化策略还需要持续跟踪发展方向，才能获利更多。
在我看来，目前中国的移动支付市场还处于初创阶段，所以采用垄断策略可能是最佳选择。因为移动支付行业是由服务商垄断，各个平台都受制于同样的服务条件，如果想获得收入，只能靠自身服务质量的提升。如果以后移动支付的竞争模式发生变化，可以考虑尝试市场化策略，探索新的突破口。
# 5.总结
本文以微信支付作为案例，阐述了移动支付行业的市场结构、竞争环境和市场竞争策略。移动支付是高速发展的新兴领域，创新能力、行业规模和竞争环境都处于全球领先地位。因此，要想在这个领域取得突破性的进步，就需要引入新的理念、策略和工具，开拓新市场。
