# 1.简介
         
随着互联网及相关技术的发展，视频在社交、娱乐领域日渐成为信息传递、流通的重要方式之一。传统上，视频文件的分类往往依赖于人工标注或自动化标记工具。然而，采用这种方法进行视频分类，会带来以下不足：
（1）视频数量庞大，手动标记耗时且易出错；
（2）难以捕捉到视频的全局信息；
（3）缺乏系统性，分类标签之间存在层级、关联、多样性等复杂关系；
因此，基于半监督学习的方法应运而生。它的基本思想是在训练数据充足的情况下，通过对无标注数据进行学习，提取视频的全局信息并进行分类。
本文将从视频分类的基础知识、半监督学习的主要思路、分类算法的详细介绍和实现、半监督学习在实际应用中的展望和实践三个方面来阐述半监督学习在视频分类任务中的研究进展。
# 2.视频分类的基础知识
## 2.1 视频分类简介
视频分类（Video classification）一般指根据视频特征和内容进行智能视频分析和智能推荐，将不同类别的视频划分为不同的类型或主题，方便用户查找感兴趣的内容。在之前的研究中，视频分类一直都是一个热门话题，可以说是近几年计算机视觉领域的一个热门方向，但是由于视频数据的多样性、规模、复杂性、变化速度、真实场景复杂性等诸多因素的影响，仍然存在很多难点、挑战。
### 2.1.1 数据集
视频分类的数据集通常包括两部分，即视频和标签。视频可以是来自于视频网站或者应用、手机相册、手持摄像头等各种设备采集到的，标签则对应的是视频所属的类别。目前，学术界和工业界已经开发出了丰富的视频分类数据集。其中最为著名的有YouTube-8M、Sport-1M、HMDB-51、UCF101等。这些数据集包括大量的高清视频、标注好的视频类别标签、视频描述信息等。
### 2.1.2 模型框架
视频分类模型一般由三部分组成，即特征提取模块、分类器模块和视频表示模块。特征提取模块主要负责从视频中提取特征，如光流、颜色等，它可以帮助提升模型的性能。分类器模块负责把特征映射到视频类别空间，如图像分类中使用的softmax函数，它可以使用一系列的类别标签进行训练。视频表示模块则是用来刻画视频整体特征的，如CNN卷积神经网络结构，它将不同帧特征通过时间维度进行拼接，生成视频表示向量。整个模型的输出是每个视频的类别概率分布。

目前，主流的视频分类模型大致可分为两类，分别为特征提取和分类模块。特征提取模块一般有两种形式，一种是用底层卷积神经网络如AlexNet、VGG等进行特征提取，另一种是用预训练的深度学习模型如ResNet、Inception等直接进行特征提取。分类器模块一般用分类器如softmax或支持向量机SVM等进行训练。

针对同一个目标检测任务，不同的特征提取方法和分类器方法都会有很大的差异。如图1所示，特征提取方法可以分为单阶段、两阶段和三阶段。如SPPNet、CornerNet、FCOS、TFA、SMDLNet等都是用单阶段特征提取方法。两阶段特征提取方法可以分为滑动窗口和检测区域回归，如FRCNN、YOLOv1、SSD、FasterRCNN等都是使用两个阶段特征提取方法。三阶段特征提取方法可以分为定位、分割和匹配，如MaskRCNN、Libra R-CNN、Panoptic Segmentation等都是使用三阶段特征提取方法。

在分类方法方面，包括线性分类器、多层感知机（MLP）、卷积神经网络（CNN）、循环神经网络（RNN）、注意力机制（AM）等。其中，CNN方法又可细分为目标检测方法、多标签分类方法、行为识别方法等。如Faster RCNN、YOLO、SSD、RetinaNet等都是用于目标检测任务的CNN分类方法。

同时，不同的分类方法也会对模型的性能产生较大的影响。如图2所示，线性分类器可以达到较好的性能，但其对于类间关系的建模能力弱。多层感知机（MLP）、CNN、RNN都可以在保持一定计算效率的前提下，增加模型的表达能力。

针对不同的视频分类任务，通常会选择不同的模型架构和训练策略。在Kinetics 400数据集上的训练策略通常分为warmup、noam、lr scheduler、cosine annealing scheduler、mixup、cutmix、label smoothing、negative sampling、random erasing、batch normalization、dropblock、dense connection等，其中比较典型的如warmup、lr scheduler、cutmix、label smoothing等。
