# 1.简介
         
在现代社会，人们生活节奏日益加快，而健康又是每一个人的不可或缺的权利，人们需要在工作、学习、生活中不断提升自己的身体素质。然而，由于疾病多发、自然因素导致的健康问题持续蔓延，使得各国都面临着“医疗危机”的问题。在当前医疗危机的背景下，美国近几年来也面临着各种挑战和困难，比如说建立综合性的医疗保健体系、提高医患纠纷解决效率、改善药品管理和投资策略等方面的问题。因此，为了应对医疗危机带来的挑战，美国政府在2017年建立了国会立法机构——健康和人类服务部(NCC)——旨在推进医疗危机的应对。根据国际上已有的经验和模式，该部制定了一系列相关法律法规来规范美国的医疗保健体系建设。本文将从“NCC”和“综合性医疗保健体系”两个视角，探讨美国医疗保健体系建设的一些典型案例，并进行详细阐述。
# 2.基本概念及术语说明
## 2.1 NCC (National Committee for Coordination)
在美国，“NCC”（National Committee for Coordination）是一个政府部门，其主要职责是制定相关政策，支持和促进医疗保健领域的科学和制度创新。其主要的目标有：维护医疗保健系统的正常运转；提升医疗服务质量；促进药物研究和开发；促进健康数据共享；提升医疗保健效率；增强国民健康意识和能力；维护医疗保健秩序。

## 2.2 综合性医疗保健体系
综合性医疗保健体系是指由不同类型和层次的医疗保健系统组成的完整的医疗保健系统体系。该体系主要包括以下三个层次：

⒈ 普通疾病与慢性病管理体系；
⒉ 重症病例管理体系；
⒊ 卫生防护措施、基层卫生和安全管理体系。

综合性医疗保健体系具有如下优点：

1.科学化、结构化：综合性医疗保健体系由多种类型医疗保健系统组成，各个系统之间互相独立，系统内的功能高度分离，系统间采用互联网技术实现信息共享，保证全方位的治疗覆盖。

2.专业化、人性化：综合性医疗保健体系由专门的医疗专家团队来负责各个层级的医疗工作，主导建立全球统一的医疗保健体系。

3.健康满意度：综合性医疗保健体系确保人群享受到充分有效的医疗保健服务，通过客观、公正、可预测的方式满足人的需求，提升人们的健康满意度。

综合性医疗保健体系建设涉及到的关键环节和要素主要包括：

⒈ 组织管理体制：各级政府组织和人员的培训、岗位任免、薪酬福利、资格认证、轮岗制度、责任划分、职责范围等，保证综合性医疗保健体系的顺利实施。

⒉ 人才培养、专家培训：各级政府、医院、学校建立健康管理专业、学科博士后及工程技术人员培训中心，确保人才力量的合理布局和发挥作用。

⒊ 技术创新：医疗保健行业的技术革命正在席卷各个领域，综合性医疗保健体系建设必须抓住机遇，积极参与科技革命，推动医疗保健服务和效率水平的提升。

⒌ 资源整合与共享：医疗保健资源包括药物、医疗器械、诊疗设备等，通过国际标准、组织交流、网络连接等方式，把资源整合到一起，更好地满足不同人的需求。

⒍ 政策制订与指导：综合性医疗保健体系的运作需要建立健康管理部门的管控机制，严格按照职业健康行为要求管理医疗机构，建立医疗体系风险评估和风险管理制度，健全完善的政策制订和执行机制。

⒎ 制度保障：综合性医疗保健体系的建立和运行需要相应的法律法规、规章制度和执业规范，并且还需要对财政、金融、人力、技术等各方面进行整合配套。

综合性医疗保健体系建设将成为各个国家的共同努力方向，推动我国医疗保健事业的发展与繁荣。
# 3.案例介绍
根据“NCC”的建设目标，以及“综合性医疗保健体系”的定义，我们可以总结出以下几个典型案例：

## 3.1 Datica医疗保健的创新与应用
Datica医疗保健是美国一家为全球客户提供医疗保健产品和服务的科技公司，创立于2008年。该公司通过开发医疗保健平台、应用和服务，帮助消费者在线快速找到所需的医疗产品。该公司业务范围遍布全球，且产品面向所有类型的医疗保健需求。截止目前，其产品已经遍布医院、社区、零售店、居家医疗等多个细分市场，提供了多种类型和价格标签的健康保健产品。例如，其专注于儿童医疗健康管理的ProvidenceCare产品，就是为残障宝宝及其家庭成员提供智能手机随时随地检查、测量、推荐药物和保健方案的解决方案。

Datica医疗保健的成功离不开其团队的努力。首先，他们的专家团队深耕医疗保健领域，为客户提供独特的产品和服务。其次，Datica医疗保健运用新型的科技手段，不断扩大自己的规模和影响力。Datica医疗保健成立初期建立起“医疗云计算”平台，为医疗机构提供“一次性医疗卡片”及医疗记录管理系统，帮助其实现数据共享、信息分析及决策支持，是当时美国“医疗危机”的重要“插曲”。第三，Datica医疗保健运用行业的先进制造能力，研发先进的医疗产品和服务，如图形显示、自动化设备和机器学习算法，帮助用户获得更多健康和幸福感。第四，Datica医疗保健也致力于教育界的医疗健康教育。为开展全国性的医疗健康教育活动，Datica医疗保健利用自身的优势力量和资源，如宣传媒体、网络平台、物流网络、物业服务、社区服务等，带领医疗保健教育倡议得到社会广泛关注和响应。

综上，Datica医疗保健的建设既是为了让世界各国的消费者享受到医疗保健的便利，也是为了营造一个健康良好的国际医疗环境，丰富国民的医疗保健需求。值得注意的是，Datica医疗保健虽然处于早期阶段，但其产品和服务的销售额占全美比例较高，且Datica医疗保健一直保持着稳定的盈利模式。因此，在短期内，Datica医疗保健可能会面临企业经营上的困境，但长远看，他们仍将继续努力，确保健康良好的国际医疗环境。

## 3.2 Blue Cross Blue Shield的综合性医疗保健体系建设
美国的卫生保健领域发展较晚，在美国之前的欧洲和日本均有类似体系的组织存在，但在美国却完全没有。美国是全球第二大医疗保健消费国，而美国卫生保健协会(Healthcare Associates Inc.)成立于1903年，其第一任主席Henry Ford是美国首位获得公爵勋章的国会议员。

1937年，在卢沟桥七彩原子弹爆炸、八月苏修内战结束、国会正式通过223号公报等一系列政治事件之后，“Blue Cross Blue Shield of America”（以下简称“BCBSA”）由美国最早的公立卫生保健组织诞生，打破了欧洲、日本等其他国家的垄断地位。BCBSA的主要任务是：

1. 提供全民健康服务；
2. 为公共卫生服务和健康教育提供必要保障；
3. 促进公共服务的共享与发展。

BCBSA建成于20世纪40年代末，是美国国内第一个公立卫生保健机构。2016年，BCBSA依据其“全民健康”的使命，推出了“医疗保障计划”，这是美国公共卫生服务的重要组成部分。截至2019年底，BCBSA在全国拥有13万多名员工，其中有3千多名卫生保健专业人员，覆盖范围涵盖多个疾病和特殊人群，包括孤老、残障人士、老年人、青少年以及慢性病患者。

BCBSA推行的“医疗保障计划”包括：

1. 医疗支援计划（Medicare Part A）：为低收入者提供基本医疗保障，包括基本检查、X光、MRI、核磁共振等项目；
2. 医疗捐助计划（Medicaid）：为老年人提供基本医疗保障；
3. 医疗救助计划（Medigap）：为全民提供特定健康项目，如失眠、残疾、糖尿病等救助；
4. 医疗保险（Insurance）：提供保险来源，覆盖疾病、残疾、精神疾病和老年人等方面的医疗服务；
5. 综合医疗服务（Co-pay）：与医院合作，为预付费服务提供基本医疗保障；
6. 医疗互助计划（Health Care Co-Op）：与其他社区和组织合作，为乡村和边缘社区提供基本医疗保障。

BCBSA建立的综合性医疗保健体系包括：

1. 医院治疗系统：分为大型医院、小型诊所、专科医院等，提供各类医疗服务，如肿瘤和慢性病治疗、口腔内科治疗、心脏病和胃肠病防护、牙科治疗、儿科治疗、精神科治疗、康复治疗等；
2. 康复医院：为老人、残疾人和精神病患者提供康复咨询、康复治疗、康复教育等服务；
3. 社区卫生服务中心：分散在美国各州、省份的中心区域，为社区居民提供新闻和健康咨询、计划服务等；
4. 个性化服务：提升公共服务的有效性，通过个人化服务为患者提供个性化服务，如精准医疗和分娩指导；
5. 治疗纠纷处理中心：提供直接向医疗机构及其服务人员解决治疗纠纷、解答疑问、帮助患者就医等。

综合性医疗保健体系构建，不仅能够为公众提供可靠的医疗服务，而且能够最大程度地提高各项服务的质量和效率。同时，它还可以促进医疗保健领域的专业化程度提升，人才培养。
