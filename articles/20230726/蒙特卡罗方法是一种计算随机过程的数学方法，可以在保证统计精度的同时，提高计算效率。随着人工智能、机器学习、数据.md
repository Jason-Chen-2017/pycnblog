# 1.简介
         
蒙特卡罗方法（Monte Carlo method）是一种用于解决数值模拟问题的概率分析的方法。它是基于重复试验、随机抽样、理论误差估计、方差分析及数值积分等概念。它的主要优点是可以准确而有效地模拟随机系统的行为，具有广泛的应用前景。
蒙特卡罗方法由美国数学家、物理学家罗森·卡尔纳普克于1947年提出，并被称为“随机实验方法”或“统计研究方法”。卡尔纳普克将其作为“模拟计算机的有效工具”，并借用他的一句名言——“不要去寻找有意义的数学问题，而要尽可能找寻不确定性。”因此，蒙特卡罗方法既可以用来求解复杂的系统，也可以用来做理论研究。
蒙特卡罗方法的基本原理是利用随机数生成器（Random Number Generator，RNG），通过对某一过程或模型进行多次采样（即随机变量的独立重复采样），从而获得该过程或模型的各种性质的近似值。这种通过采样产生的近似值，通常可以更加真实地反映实际的系统行为。
# 2.概念术语说明
## 2.1 随机变量及分布函数
在讨论蒙特卡罗方法之前，需要先了解一些基本的随机变量及其分布函数的概念。
### 二元随机变量及分布函数
设X是一个随机变量，它具有两个可能取值的集合{x1, x2,..., xn}，其中xn = X。记x1 ≤... ≤ xn，则称此随机变量为有序随机变量。一个二元随机变量X的分布函数F(x)就是x给定时的概率，即P(Xn = xi)。若Xn服从某个概率分布λ(xi)，则对应于该分布函数的取值为λi(xi)，i=1,2,...n。例如，假设X为抛硬币结果，有时记为X~B(p)，即正面朝上的概率为p。则分布函数为:
F(H) = P(Xn = H) = p if Xn = H else (1-p)
### 一维离散随机变量及分布律
设X为一个一维离散随机变量，其取值为整数k={0,1,...,N-1}，N为观测次数。记以Nk为单位时间内发生的事件个数为Ni，i=0,1,...,N-1。则概率密度函数f(x)表示事件发生在第x个位置所需的单位时间。例如，某段时间内匀速旋转的磁盘可能会具有均匀分布的序列。这种情况下，f(x)=1/N。
## 2.2 概率分布
在讨论蒙特卡罗方法时，还会涉及到大量关于概率分布的概念。本小节将介绍一些基本的概念。
### 连续型随机变量
如果随机变量X的分布函数F(x)存在且可导，则称X为连续型随机变量；否则，称X为离散型随机变量。如果随机变量X为连续型随机变量，则X的分布函数F(x)是概率密度函数，定义如下：
F(x) = Prob[X<=x]，即随机变量X落在区间(a,b]中的概率。
一般来说，如果连续型随机变量X的取值能够按照一定规律变化，比如X在[a, b]区间内随机跳跃的随机过程，那么相应的分布函数F(x)是一个连续函数，即F(x)不是关于x唯一的表达式。但对于均匀分布的随机变量X，其分布函数恒等于1/(b-a), a<x≤b。
### 离散型随机变量
如果随机变量X的取值集合{x1, x2,..., xm}是有限的，则称X为离散型随机变量。离散型随机变量的分布函数是概率分布表，即F(x)为概率，表示随机变量X的值等于x的概率。离散型随机变量X的分布函数必然是非负的。
