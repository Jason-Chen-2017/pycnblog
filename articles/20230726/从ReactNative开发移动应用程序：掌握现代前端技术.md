# 1.简介
         
React Native是一个用于编写跨平台移动应用的框架，由Facebook开发。它的主要优点就是可以快速开发出功能完整、可发布的应用。本专栏将教会读者从基础知识入手，学习React Native开发移动应用的相关知识。本专栏适合零基础开发人员，让他们能够快速上手React Native进行开发。
