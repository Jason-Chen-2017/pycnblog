# 1.简介
         
随着高性能计算、云计算、机器学习、互联网技术等技术的广泛应用，“数据驱动”应用和“快速迭代”开发模式已成为现代软件开发的主流。同时，越来越多的创新企业也选择使用这些技术。此外，C++作为最受欢迎的编程语言，已经成为开发人员的首选。而在近几年，C++已经推出了两个大的版本——C++11和C++14。两者的新特性、改进和语法差异非常之大，也吸引了广泛的开发者的关注。
这次分享将详细介绍C++11和C++14在现代编程语言的开发实践。这两项技术的出现与发展对于当前C++编程语言的发展来说是一个重要的里程碑。它带来了诸如完善的类模板系统、lambda表达式、函数对象、constexpr关键字、移动语义、右值引用、decltype、可变参数模板、统一初始化语法等一系列特性，极大地提升了编程语言的能力，降低了开发难度和错误率。
# 2.基本概念术语说明
## 2.1 C++ 11
C++11是ISO/IEC 14882:2011中定义的新的C++标准，主要包括如下特征：
- 异常处理机制
- Rvalue References（右值引用）
- 默认模板参数
- 强类型别名(aliases)
- 初始化列表(Initialization Lists)
- nullptr关键字
- 删除的运算符=delete
- Range-based for循环
- lambda表达式
- 透明指针
- alignas
- 线程支持(thread support library)
- 模板的static成员函数和外部链接
- uniform initialization语法(brace initialization)
- 特殊成员函数(Special Member Functions)
- noexcept异常说明符
- 继承构造函数
- decltype Type traits(type_traits header)
- fold expression（折叠表达式）
- inline命名空间(inline namespace)
- final关键字
- override关键字
- nullptr关键字
- constexpr作用域
- 统一初始化语法(Uniform Initialization Syntax)
- atomic操作
- 通用引用(Universal References)
- explicit关键字
- class-enum 转换
- 概念常量(concept constants)
- 静态assert关键字
- inline变量
- 类模板实参Deduction
- 函数式编程工具库
- std::string及std::wstring支持unicode
- initializer_list容器
- 模板参数pack(Parameter packs)
- 属性(Attributes)
- nested scope(嵌套作用域)
- 元编程(Metaprogramming)
- 属性(attributes)
- constexpr函数
- explicit conversion functions
- 混入类型(mixin types)
- 函数引用(Function References)
- rvalue arrays(右值数组)
- alignmentSpecifier
- 原生类型(native types)
- 结构化绑定(structured binding)
- 固定宽度整数类型(fixed width integer types)
- 条件编译(Conditional Compilation)
- static assertions(静态断言)
## 2.2 C++14
C++14 是ISO/IEC 14882:2014中定义的第二个C++标准，主要包括如下特征：
- 不完全类型(Narrowing Conversions)
- 括号初始化(Bracket Initialization)
- 返回类型decltype(Decltype Auto)
- Lambda捕获(Lambda Capture)
- 常量表达式(Constant Expressions)
- 初始化语句范围(Init-statement-range)
- 复合字面量(Compound Literals)
- 一致的转发(Consistent Forwarding)
- 文件作用域导入声明(File-scope Imports Declarations)
- 可变参数宏(Variadic Macros)
- 用户定义字面量(User-Defined Literals)
- 并行算法库(Parallel Algorithms Library)
- [[deprecated]]属性
- 对象池(Object Pools)
- if constexpr
- new-expression requirements
- variable templates
- null pointer constant (nullptr)
- relaxed constexpr
- 用户自定义字面量
- 折叠表达式(Fold expressions)
- 有限自动机(Finite State Machines)
