4.3. 核心代码实现

4.3.1. 最短路算法实现

最短路算法实现主要涉及两个步骤：一是构建图中所有边的最短路径，二是计算出图中所有边的最小费用。

```python
import networkx as nx

def shortest_path(G, source, target):
    dist = float('inf')
    path = None

    for edge in G.edges(data=True):
        dist, path = edge[0], edge[1]

        if dist < 0 and dist < path:
            continue

        if dist < 0:
            break

        if dist == 0 and edge[1]!= target:
            path = edge[1]

    return dist, path

def min_cost(G, source, target):
    dist = float('inf')
    path = None

    for edge in G.edges(data=True):
        dist, path = edge[0], edge[1]

        if dist < 0 and dist < path:
            continue

        if dist < 0:
            break

        if dist == 0 and edge[1]!= target:
            path = edge[1]

    return dist, path

4.3.2. FC算法实现

FC算法实现与最短路算法类似，只是输入数据为边的费用，而非路径信息。

```python
import networkx as nx

def min_cost_fc(G, source, target):
    dist = float('inf')
    path = None

    for edge in G.edges(data=True):
        dist, path = edge[0], edge[1]

        if dist < 0 and dist < path:
            continue

        if dist < 0:
            break

        dist = edge[0][1]  # 这里我们使用边的费用作为代价
        path = edge[1]

    return dist, path
```

在实际网络环境中，需要使用`networkx`库来构建网络图，使用`shortest_path`函数来计算图中边的最短路径，使用`min_cost_fc`函数来计算图中边的最小费用。

```python

    G[edge][source], G[edge][target] = 1, 1

接下来，使用`shortest_path`函数计算图中边的最短路径：

```python

```

```python

```
