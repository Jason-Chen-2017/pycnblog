2. 技术原理及概念

2.1. 基本概念解释

社交网络分析（Social Network Analysis，简称SNA）是一种研究社交网络中节点和关系之间数学模型的分支。在SNA中，节点表示个体，关系表示个体之间的联系。通过对网络结构的分析，可以发现节点之间的相互作用、联系和模式，从而为数据挖掘提供新的视角。

2.2. 技术原理介绍：算法原理，操作步骤，数学公式等

2.2.1. 层次聚类分析

层次聚类分析（Hierarchical Clustering，HC）是社交网络分析中一种常用的聚类算法。它根据节点之间的距离（如距离公式、权重等）将其划分为一定数量的簇。通过调整聚类参数，可以控制聚类的层次结构，如：自定义聚类数、聚类层次等。

2.2.2. 社交网络分析核心算法——有向图分析

有向图分析（Directed Graph Analysis，DAGA）是社交网络分析中的另一个核心算法。有向图分析将网络中的节点和边划分为有向图，然后利用有向图的特性，如：路径、边权、节点嵌入等，研究节点之间的关系和网络结构。

2.2.3. 社交网络分析模型——社交网络挖掘

社交网络挖掘（Social Network Mining，简称SNM）是将社交网络作为数据源，利用各种分析技术对网络结构进行挖掘和分析，以发现节点和关系之间的规律。常见的社交网络挖掘模型包括：基于用户画图的模型（User-centric Approach）和基于节点嵌入的模型（Node-centric Approach）等。
