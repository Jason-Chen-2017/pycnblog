4.3. 核心代码实现

以 Python 语言为例，实现一个简单的数据挖掘项目，主要包括数据预处理、特征选择和模型训练等步骤。

```python
import pandas as pd
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# 读取数据
data = load_iris()

# 将数据集分为训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(data.data, data.target, test_size=0.2)

# 特征选择
features = ['petal_length', 'petal_width','sepal_length','sepal_width']

# 创建分类器
clf = KNeighborsClassifier()

# 训练模型
clf.fit(X_train[features], y_train)

# 测试模型
y_pred = clf.predict(X_test[features])

# 计算准确率
accuracy = accuracy_score(y_test, y_pred)
print("Accuracy: ", accuracy)
```

代码讲解说明
-----------------

上述代码实现了一个简单的数据挖掘项目，主要涉及数据预处理、特征选择和模型训练等步骤。

首先，使用 Pandas 库读取数据，并使用 train\_test\_split() 函数将数据集分为训练集和测试集。

接着，使用 Scikit-learn 库中的 KNeighborsClassifier 类创建一个分类器对象，并使用 fit() 函数对模型进行训练，使用 predict() 函数对测试集进行预测。最后，使用 accuracy\_score() 函数计算模型的准确率。

此代码实现的数据挖掘项目非常简单，但它演示了数据挖掘的基本原理和流程，以及如何使用机器学习算法对数据进行分类和预测。
