4. 应用示例与代码实现讲解
----------------------------

4.1. 应用场景介绍

本文将介绍如何利用图表对数据库中的数据进行可视化处理，以提高数据的可视化和理解。

4.2. 应用实例分析

假设我们有一个名为 `students` 的数据表，其中包含 `id`、`name`、`age` 和 `gender` 四个字段。我们可以利用图表来查看学生的分布情况。

```sql
SELECT * FROM students;

# 查询结果连接到 Python 对象

students_data = your_table.select('id', 'name', 'age', 'gender');
```

4.3. 核心代码实现

```python
import mysql.connector
import matplotlib.pyplot as plt

# 连接到数据库
cnx = mysql.connector.connect(user='your_username', password='your_password', host='your_host', database='your_database')
cursor = cnx.cursor()

# 查询数据库中的所有数据

students_query = ("SELECT * FROM students")
cursor.execute(students_query)
students_data = cursor.fetchall()

# 将查询结果连接到 Python 对象
students_data = your_table.select('id', 'name', 'age', 'gender');

# 绘制柱状图
df = students_data
ggplot(df, aes(x=age)) + 
    geom_bar(stat = 'identity', color = 'lightblue') + 
    labs(title = 'Student Distribution') + 
    ggtitle('Student Distribution')
```
