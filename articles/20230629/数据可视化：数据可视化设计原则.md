4.4. 代码讲解说明
--------------

下面是一个简单的数据可视化代码实现，展示了一个金融领域的股票价格数据。
```python
import pandas as pd
import matplotlib.pyplot as plt

# 读取数据
df = pd.read_csv('stock_price.csv')

# 处理数据
df = df[['Date', 'Close']]

# 可视化数据
plt.figure(figsize=(14, 7))
plt.plot(df['Date'], df['Close'])
plt.title('Stock Price')
plt.xlabel('Date')
plt.ylabel('Price (USD)')
plt.show()
```
