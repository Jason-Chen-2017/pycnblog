# 2.核心概念与联系
在深入探讨操作系统原理与源码实例之前，我们需要了解一些基本概念：
- **进程（Process）**：进程是操作系统中最小的独立运行单位，它包括程序及其数据集合。每个进程都有自己独立的内存空间和资源。
- **线程（Thread）**：线程是进程内部的一个执行单元，它可以并发执行不同任务。线程共享进程的内存空间和资源。
- **同步（Synchronization）**：同步是指多个线程之间相互协调执行的过程。通过同步机制，多个线程可以安全地访问共享资源。
- **异步（Asynchronous）**：异步是指多个任务之间没有顺序关系或者没有等待关系的执行过程。异步任务可以在不阻塞其他任务的情况下完成。
- **锁（Lock）**：锁是一种同步原语，用于控制对共享资源的访问。锁可以确保在某个时刻只有一个线程能够访问共享资源。
- **信号量（Semaphore）**：信号量是一种计数锁，它允许多个线程并发访问共享资源。信号量可以用来控制对某些资源的访问次数限制或者等待队列长度限制。
