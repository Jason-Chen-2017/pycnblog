                 

# 1.背景介绍

计算机编程语言的链接器与加载器是编译型编程语言中的重要组成部分，它们负责将编译后的目标代码转换为可执行文件，并在运行时加载和执行这些文件。链接器（Linker）和加载器（Loader）之间存在一定的联系，但它们也有一些区别。

链接器主要负责将多个对象文件（Object File）或库文件（Library File）合并成一个可执行文件，解决符号引用和符号定义之间的关系，并解决内存地址等问题。链接器通常在编译阶段进行，生成可执行文件或共享库文件。

加载器则是在程序运行时负责将可执行文件加载到内存中，为程序提供所需的资源，并初始化程序环境。加载器还负责管理内存空间、处理动态链接和调用相关函数等任务。加载器通常在程序启动阶段进行。
