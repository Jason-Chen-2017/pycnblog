# 2.核心概念与联系
## 2.1矩阵分解
矩阵分解是一种数学方法，用于将一个矩阵划分为多个较小的矩阵之和或积。这种方法主要应用于处理高维数据集，以减少计算复杂性并提取有意义的信息。常见的矩阵分解方法包括奇异值分解（SVD）、非负奇异值分解（NMF）等。

## 2.2降维
降维是一种数学方法，用于将高维数据映射到低维空间中，以简化数据表示和处理。降维可以减少计算复杂性并提高计算效率，同时也可以保留数据中的关键信息。常见的降维方法包括主成分分析（PCA）、线性判别分析（LDA）等。
