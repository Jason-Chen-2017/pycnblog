# 2.核心概念与联系
## 2.1概率论与统计的区别
概率论和统计分析是两个相互独立且相互补充的科学领域。概率论主要研究事件发生的可能性以及其他随机现象，而统计分析则主要研究从大量数据中抽取信息并进行描述和预测。

## 2.2随机变量、事件和条件概率等基本概念
### 随机变量：在某种事件发生时产生不确定结果的变量称为随机变量。例如：抛硬币所得到头或尾等结果就是随机变量。
### 事件：在某种情况下可能发生或不发生的结果集合称为事件。例如：抛硬币得到头或尾都是事件。
### 条件概率：给定某一事件已经发生，另一事件发生的可能性称为条件概率。例如：给定硬币已经被抛出，再次抛出硬币得到正面（头）或反面（尾）等可能性都是条件概率。
### 独立事件：若两个或多个事件之间没有任何关联，那么它们之间就是独立事件。例如：抛硬币两次得到正面（头）或反面（尾）时，每次抛硬币都是独立事件。
