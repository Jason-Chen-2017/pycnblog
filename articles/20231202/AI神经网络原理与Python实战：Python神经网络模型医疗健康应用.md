# 2.核心概念与联系
## 2.1神经元与层次结构
一个神经网络由多个相互连接的节点组成，每个节点称为“神经元”或“单元”。这些节点分为输入层、隐藏层和输出层三个部分。输入层负责接收输入数据；隐藏层进行数据处理；输出层提供最终结果。每个节点都有一个权重向量，用于计算其输出值。
## 2.2激活函数
激活函数是神经网络中的关键组成部分，它控制了神经元的输出值。常见的激活函数有sigmoid函数、tanh函数和ReLU函数等。激活函数使得神经网络具有非线性性质，从而能够学习复杂的模式。
## 2.3损失函数
损失函数是衡量模型预测误差的标准，通过优化损失函数来调整模型参数以获得更好的预测效果。常见的损失函数有均方误差（MSE）、交叉熵损失（Cross-Entropy Loss）等。损失函数使得训练过程具有目标性，从而能够逐步提高模型性能。
