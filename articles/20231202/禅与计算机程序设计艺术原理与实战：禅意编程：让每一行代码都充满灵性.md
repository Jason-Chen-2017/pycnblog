问题：实现一个简单的冒泡排序算法。

代码实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
result = bubble_sort(arr)
print(result)
```

解释说明：

- 首先，我们定义了一个名为`bubble_sort`的函数，它接受一个列表作为输入参数，表示需要排序的数组。
- 然后，我们获取数组的长度，并使用两个嵌套的循环来实现冒泡排序的过程。外部循环用于遍历数组的每个元素，内部循环用于比较相邻的元素，并交换它们的位置，如果它们的顺序不正确。
- 最后，我们调用`bubble_sort`函数，将需要排序的数组作为输入参数，并将排序后的结果打印出来。
