Python是一种强大的编程语言，它具有简洁的语法和易于学习。在Python中，字典是一种特殊的数据结构，用于存储键值对。类则是面向对象编程中的基本概念，用于创建和定义对象的蓝图。在本文中，我们将深入探讨Python中的字典和类，并揭示它们之间的联系以及如何使用它们来解决实际问题。

# 2.核心概念与联系
# 2.1字典
字典（Dictionary）是Python中一个非常重要的数据结构，它可以存储键值对（key-value pairs）。每个键值对由一个唯一的关键字（key）和一个值（value）组成。关键字必须是不可变类型（如整数、浮点数、字符串等），而值可以是任何类型。例如：
```python
person = {'name': 'John', 'age': 30, 'city': 'New York'}
```
在这个例子中，'name'、'age'和'city'都是关键字，而'John'、30和'New York'都是值。通过使用关键字作为索引，我们可以轻松地访问或修改字典中的值。例如：
```python
print(person['name'])  # 输出: John
person['age'] = 31    # 更新年龄为31岁
```
