# 2.核心概念与联系
## 2.1概率论与统计学的基本概念
### 2.1.1概率
概率是一个事件发生的可能性，通常表示为0到1之间的一个数值。例如，抛硬币的正面朝上的概率为0.5。
### 2.1.2随机变量
随机变量是一个可以取多个值的变量，每个值都有相应的概率。例如，抛硬币得到正面或反面的结果就是一个二元随机变量。
### 2.1.3期望值和方差
期望值是随机变量取值平均数，用于衡量随机变量取值集合整体位置。方差是一种度量数据分布离散程度的指标，用于衡量随机变量取值集合离散程度。
## 2.2信号处理中probability和statistics联系
在信号处理中，我们经常需要对信号进行分析、滤波、重构等操作。这些操作往往涉及到概率论和统计学知识。例如：
- **估计**：我们需要根据观测数据估计未知参数；
- **检验**：我们需要判断某个假设是否成立；
- **预测**：我们需要根据历史数据预测未来数据；
