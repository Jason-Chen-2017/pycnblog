# 4.具体代码实例和详细解释说明

在这部分，我们将通过一个具体的代码实例来详细解释规则引擎的执行流程和算法原理。

```python
# 定义规则库
rules = [
    {
        "condition": "age >= 18",
        "action": "grant_access"
    },
    {
        "condition": "age < 18",
        "action": "deny_access"
    }
]

# 初始化工作内存
work_memory = {"age": 20}

# 执行决策
def execute_decision(rules, work_memory):
    for rule in rules:
        if eval(rule["condition"]):
            return rule["action"]
    return None

# 更新工作内存
def update_work_memory(action, work_memory):
    if action == "grant_access":
        work_memory["access_granted"] = True
    elif action == "deny_access":
        work_memory["access_denied"] = True

# 主函数
def main():
    action = execute_decision(rules, work_memory)
    update_work_memory(action, work_memory)
    print(work_memory)

if __name__ == "__main__":
    main()
```

在这个代码实例中，我们定义了一个规则库，包括两个规则。然后，我们初始化了工作内存，包括一个年龄的属性。接着，我们执行了决策，根据规则库和工作内存来选择最佳的决策。最后，我们更新了工作内存，根据执行的结果来选择最佳的决策。
