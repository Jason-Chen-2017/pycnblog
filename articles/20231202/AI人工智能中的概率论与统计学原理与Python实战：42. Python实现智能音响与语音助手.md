# 2.核心概念与联系
# 2.1概率论与统计学基础知识
## 2.1.1概率论基础知识
概率论是一门研究随机事件发生的可能性和相关性的科学。它主要包括几个基本概念：事件、样空间、事件空间、事件的互斥性、独立性等。在AI人工智能中，概率论被广泛应用于各种场景，如推荐系统、搜索引擎排名等。
## 2.1.2统计学基础知识
统计学是一门研究从数据中抽取信息并进行推断的科学。它主要包括几个基本概念：数据集、变量、参数估计、假设检验等。在AI人工智能中，统计学被广泛应用于各种场景，如机器学习算法选择、模型评估等。
# 2.2语言模型与NLP（自然语言处理）基础知识
## 2.2.1语言模型简介
语言模型是一种预测下一个词或短語序列出现概率的模型。它主要包括几个基本概念：上下文窗口、条件独立性等。在AI人工智力中，语言模型被广泛应用于各种场景，如自动完成功能等。
## 2.2.2NLP基础知识
自然语言处理（NLP）是一门研究让计算机理解和生成人类自然语言的科学。它主要包括几个基本任务：词汇处理（tokenization）、句法分析（parsing）、命名实体识别（named entity recognition）等。在AI人工智力中，NLP被广泛应用于各种场景，如情感分析功能等。
