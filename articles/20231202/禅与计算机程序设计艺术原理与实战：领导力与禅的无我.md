## 4.1 代码实例：求两个数的最大公约数

我们可以使用辗转相除法来求两个数的最大公约数。辗转相除法的算法步骤如下：

1. 如果a为0，则返回b；
2. 如果b为0，则返回a；
3. 如果a大于b，则将a除以b，并返回辗转相除法的递归调用；
4. 如果a小于b，则将b除以a，并返回辗转相除法的递归调用。

以下是一个Python的代码实例：

```python
def gcd(a, b):
    if a == 0:
        return b
    if b == 0:
        return a
    if a > b:
        return gcd(a % b, b)
    else:
        return gcd(a, b % a)

a = 12
b = 18
print(gcd(a, b))
```

在这个代码实例中，我们首先定义了一个名为gcd的函数，它接受两个参数a和b。然后我们使用辗转相除法的算法步骤来计算a和b的最大公约数。最后，我们输出了a和b的最大公约数，即6。
