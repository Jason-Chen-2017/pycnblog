# 2.核心概念与联系
在本文中，我们将关注以下几个核心概念：
- **人工智能大模型**：指一种通过深度学习或其他机器学习方法训练出来的神经网络模型，可以处理复杂问题并提供准确的预测和建议。
- **服务**：指一个可以被其他系统或应用程序调用并获取资源或功能的软件组件或平台。
- **实体决策**：指在特定场景下采取针对性行动的过程，包括识别目标、评估选项、分析风险和制定计划等环节。
