# 2.核心概念与联系
在深度学习中，神经网络是最重要的组成部分。一个神经网络由多个节点（或神经元）组成，这些节点之间通过权重连接起来。每个节点接收输入信号并根据其权重和偏置值对信号进行处理，然后将结果传递给下一个节点。这个过程会在多个层次上重复，从而形成了所谓的“深”结构。

深度学习主要包括以下几个方面：
- 卷积神经网络（Convolutional Neural Networks, CNNs）：适用于图像处理任务；
- 循环神经网络（Recurrent Neural Networks, RNNs）：适用于序列数据处理任务；
- 自然语言处理（Natural Language Processing, NLP）：适用于文本分析任务；
- 生成对抗网络（Generative Adversarial Networks, GANs）：适用于生成新数据任务；
- 变分自动编码器（Variational Autoencoders, VAEs）：适用于降维和生成新数据任务；等等。
