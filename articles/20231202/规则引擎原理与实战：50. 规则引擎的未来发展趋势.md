## 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的代码实例来说明规则引擎的工作原理：

```python
from jython.rules import *

# 定义规则
rule1 = Rule("rule1", "当温度高于30度时，开启空调")
rule2 = Rule("rule2", "当温度低于15度时，关闭空调")

# 定义条件
condition1 = Condition("condition1", "温度高于30度", lambda temperature: temperature > 30)
condition2 = Condition("condition2", "温度低于15度", lambda temperature: temperature < 15)

# 定义动作
action1 = Action("action1", "开启空调", lambda: air_conditioner.start())
action2 = Action("action2", "关闭空调", lambda: air_conditioner.stop())

# 添加条件和动作到规则
rule1.add_condition(condition1)
rule1.add_action(action1)
rule2.add_condition(condition2)
rule2.add_action(action2)

# 加载规则和知识表
rule_engine = RuleEngine()
rule_engine.load_rules(rules)
rule_engine.load_knowledge(knowledge)

# 规则匹配
matched_rules = rule_engine.match_rules(knowledge)

# 规则执行
for rule in matched_rules:
    rule.execute(knowledge)

# 输出决策结果
print(knowledge)
```

在这个代码实例中，我们首先定义了两个规则（rule1和rule2），以及它们的条件（condition1和condition2）和动作（action1和action2）。然后，我们加载了规则和知识表，并进行规则匹配和规则执行。最后，我们输出了决策结果。
