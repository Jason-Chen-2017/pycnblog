```python
import queue

class TaskScheduler:
    def __init__(self):
        self.tasks = queue.Queue()

    def add_task(self, task):
        self.tasks.put(task)

    def get_task(self):
        return self.tasks.get()
```

在这个代码实例中，我们使用了Python的queue模块来实现一个先来先服务（FCFS）的任务调度策略。当我们添加一个任务时，它会被放入队列中，当我们获取一个任务时，它会从队列中取出第一个任务。
