# 2.核心概念与联系

在本节中，我们将介绍异常检测的核心概念和联系，包括异常检测的定义、类型、方法和评估指标。

## 2.1 异常检测的定义

异常检测是一种监督学习任务，旨在识别数据中的异常点。异常点通常是与大多数数据点的特征或行为有显著差异的数据点。异常检测在许多领域具有重要应用价值，例如金融、医疗、生产等。

## 2.2 异常检测的类型

异常检测可以分为两类：无监督异常检测和半监督异常检测。

- 无监督异常检测：在这种方法中，我们没有任何关于异常数据的标签信息。我们只能根据数据的特征来判断是否为异常点。
- 半监督异常检测：在这种方法中，我们有一小部分异常数据的标签信息。我们可以利用这些标签信息来帮助训练模型，以提高异常检测的性能。

## 2.3 异常检测的方法

异常检测的方法包括以下几种：

- 统计方法：如Z-score、IQR等。
- 机器学习方法：如SVM、决策树、随机森林等。
- 深度学习方法：如卷积神经网络、循环神经网络等。

## 2.4 异常检测的评估指标

异常检测的评估指标包括以下几种：

- 准确率：异常点预测正确的比例。
- 召回率：预测为异常的异常点的比例。
- F1分数：准确率和召回率的调和平均值。
