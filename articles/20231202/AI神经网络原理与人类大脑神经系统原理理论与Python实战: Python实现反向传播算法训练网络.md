# 2.核心概念与联系
## 2.1 AI 与深度学习
AI（Artificial Intelligence）是计算机科学领域的一个分支，旨在让计算机模拟人类的智能行为。深度学习（Deep Learning）是一种 AI 技术，它基于神经网络进行数据处理和模式识别。深度学习可以自动从大量数据中提取特征并进行预测或分类任务，因此被广泛应用于图像识别、自然语言处理等领域。
## 2.2 神经网络与人类大脑
神经网络是一种由多个节点（神经元）组成的图形结构，每个节点都有输入和输出信息。这些节点通过有向边连接在一起，每条边都有一个权重值。当输入数据流向输出层时，这些权重会根据前向传播规则相加并进行激活函数处理后得到最终结果。同时，损失函数会根据预测结果与真实结果之间的差异来计算误差值；然后使用反向传播规则更新权重值以减小误差值；最终达到最小化损失函数目标并得到最佳参数设置。
## 2.3 Python & TensorFlow & Keras
Python是一种高级编程语言，拥有简单易读的语法和强大的库支持。TensorFlow是Google开发的开源机器学习库之一；Keras则是TensorFlow上运行的高级API框架，它提供了许多便捷功能以简化深度学习模型构建和训练过程。在本文中我们将使用Python+Keras+TensorFlow来实现反向传播算法训练网络示例代码。
