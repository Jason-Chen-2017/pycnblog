# 2.核心概念与联系
## 2.1 AI神经网络基础知识
AI神经网络是一种由多层感知器组成的计算模型，它可以自动学习从数据中抽取特征，并使用这些特征进行预测或分类任务。这些感知器通过权重连接起来形成一个图形结构，这个图形结构被称为神经网络。每个节点（也称为单元或神经元）接收输入信号并根据其权重和偏置对信号进行处理，然后将处理后的信号传递给下一个节点。这个过程被称为前向传播。在训练过程中，神经网络会根据错误调整它们的权重和偏置以便更好地预测或分类任务。
## 2.2 人类大脑神经系统基础知识
人类大脑是一个复杂且高度组织化的生物电路系统，由数十亿个neuron（神經元）组成。每个neuron都包含着输入端（dendrites）、主体（soma）和输出端（axon）三部分组成部分。neuron之间通过synapses（同apses）相互连接，synapses上有一个称为weight（权重）的参数来表示信息传递强度。大脑内部存在许多不同层次和类型的neuron群集，这些群集被称为brain regions（大脑区域）或brain networks（大脑网络）。大脑区域或网络之间通过white matter tracts（白质脉管）相互连接。大脑区域或网络负责执行各种功能任务，如感觉、思考、记忆等等。
