推荐系统的一个典型的数学模型是基于协同过滤的推荐模型。基于协同过滤的推荐模型可以用以下公式来表示：

$$

    keywords = []

    results = []
    for keyword in keywords:
        if query in keyword:
            results.append(keyword)
    return results
