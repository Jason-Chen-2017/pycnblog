### 3.1 设置断点

设置断点是Go语言调试的基础。以下是设置断点的具体操作步骤：

1. 在需要设置断点的代码行前面添加`debug.Break()`函数。例如：

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
    debug.Break() // 设置断点
    fmt.Println("Hello, Go!")
}
```

2. 启动Go语言的调试器，并加载需要调试的程序。

3. 运行程序，直到遇到设置的断点。

## 4.具体代码实例和详细解释说明

以下是一个具体的Go语言调试代码实例，以及详细的解释说明：

```go
package main

import "fmt"

func main() {
    a := 10
    b := 20
    c := a + b
    fmt.Println("a + b =", c)
    debug.Break() // 设置断点
    c = a - b
    fmt.Println("a - b =", c)
}
```

在这个代码实例中，我们设置了一个断点，以便在运行时暂停执行。当程序运行到断点时，调试器会暂停执行，并显示当前变量的值。在这个例子中，我们可以看到`a`的值为10，`b`的值为20，`c`的值为30。
