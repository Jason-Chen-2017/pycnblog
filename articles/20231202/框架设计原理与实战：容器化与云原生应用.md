云原生应用是一种新型的应用程序，它们是为云平台而设计的，可以在云平台上运行和扩展。这种应用程序通常使用容器化技术来部署和管理，并且可以自动地在云平台上扩展和缩放。云原生应用的主要优点是它们可以更加灵活地运行和扩展，并且可以更好地利用云平台的资源。

## 4.1 容器化代码实例

我们可以使用Docker来实现容器化。以下是一个简单的Dockerfile示例：

```Dockerfile
# 使用一个基础镜像
FROM python:3.7

# 安装一些依赖项
RUN pip install numpy pandas

# 复制代码文件
COPY app.py .

# 设置运行命令
CMD ["python", "app.py"]
```

这个Dockerfile中，我们首先使用了一个基础镜像（python:3.7）。然后，我们使用了RUN命令来安装一些依赖项（numpy pandas）。接着，我们使用了COPY命令来复制代码文件（app.py）。最后，我们使用了CMD命令来设置运行命令（python app.py）。

## 4.2 云原生应用代码实例

我们可以使用Kubernetes来实现云原生应用。以下是一个简单的Kubernetes Deployment示例：

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: my-app
spec:
  replicas: 3
  selector:
    matchLabels:
      app: my-app
  template:
    metadata:
      labels:
        app: my-app
    spec:
      containers:
      - name: my-app-container
        image: my-app-image
        ports:
        - containerPort: 80
```

这个Kubernetes Deployment中，我们首先定义了一个名为my-app的Deployment。然后，我们设置了3个副本。接着，我们设置了一个选择器，用于选择匹配的Pod。最后，我们设置了一个Pod模板，包含一个名为my-app-container的容器，使用名为my-app-image的镜像，并且在80端口上开放了访问。
