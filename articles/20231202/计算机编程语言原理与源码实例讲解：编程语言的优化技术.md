```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

在这个实例中，我们实现了一个简单的冒泡排序算法。冒泡排序算法的时间复杂度为O(n^2)，其中n是输入的大小。我们可以通过优化算法来降低时间复杂度。

一种优化方法是将每次循环的范围缩短，这样可以减少不必要的比较和交换操作。我们可以通过在每次循环后检查是否发生了交换来实现这一点：

```python
def optimized_bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
```

在这个优化后的实例中，我们通过在每次循环后检查是否发生了交换来减少不必要的比较和交换操作。这样可以降低时间复杂度为O(n^2)到O(n)。

## 4.3 并行优化实例

以下是一个简单的求和算法的实例，我们将通过这个实例来详细解释并行优化原理：

```python
def sum(arr):
    total = 0
    for num in arr:
        total += num
    return total
```

在这个实例中，我们实现了一个简单的求和算法。求和算法的时间复杂度为O(n)，其中n是输入的大小。我们可以通过优化并行计算来降低时间复杂度。

一种优化方法是将数据分割为多个部分，然后将这些部分分配给多个处理器进行并行计算。我们可以通过将数据分割为多个部分并将这些部分分配给多个处理器来实现这一点：

```python
import multiprocessing

def parallel_sum(arr):
    chunk_size = len(arr) // multiprocessing.cpu_count()
    pool = multiprocessing.Pool()
    partial_sums = pool.starmap(sum, [arr[i:i+chunk_size] for i in range(0, len(arr), chunk_size)])
    total = sum(partial_sums)
    pool.close()
    pool.join()
    return total
```

在这个优化后的实例中，我们将数据分割为多个部分并将这些部分分配给多个处理器进行并行计算。这样可以降低时间复杂度为O(n)到O(n/p)，其中p是处理器的数量。
