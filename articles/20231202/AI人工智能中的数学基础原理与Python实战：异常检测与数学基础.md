### 3.1 线性判别分析（LDA）

线性判别分析（LDA）是一种用于异常检测的线性模型。LDA的目标是找到一个线性分类器，可以将数据分为两个类别：正常类和异常类。LDA的数学模型如下：

$$
w = \frac{S_{w}^{-1}(S_{b} - \mu_{b}\mu_{b}^{T})}{\mu_{w}^{T}S_{w}^{-1}(\mu_{b} - \mu_{w})}
$$

其中，$S_{w}$ 是正常类的协方差矩阵，$S_{b}$ 是异常类的协方差矩阵，$\mu_{w}$ 是正常类的均值向量，$\mu_{b}$ 是异常类的均值向量。

LDA的具体操作步骤如下：

1. 计算正常类和异常类的均值向量。
2. 计算正常类和异常类的协方差矩阵。
3. 计算线性判别分析向量。
4. 使用线性判别分析向量对数据进行分类。

### 3.2 支持向量机（SVM）

支持向量机（SVM）是一种用于异常检测的非线性模型。SVM的目标是找到一个非线性分类器，可以将数据分为两个类别：正常类和异常类。SVM的数学模型如下：

$$
f(x) = \text{sgn}(\sum_{i=1}^{n}\alpha_{i}y_{i}K(x_{i}, x) + b)
$$

其中，$K(x_{i}, x)$ 是核函数，$y_{i}$ 是标签，$\alpha_{i}$ 是支持向量的权重。

SVM的具体操作步骤如下：

1. 计算数据的标签。
2. 计算核函数。
3. 使用支持向量机对数据进行分类。

### 4.1 线性判别分析（LDA）

```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
from sklearn.datasets import make_classification

# 生成数据
X, y = make_classification(n_samples=1000, n_features=20, n_informative=2, n_redundant=10, random_state=42)

# 训练LDA模型
lda = LinearDiscriminantAnalysis()
lda.fit(X, y)

# 预测
pred = lda.predict(X)
```

### 4.2 支持向量机（SVM）

```python

# 训练SVM模型
```
