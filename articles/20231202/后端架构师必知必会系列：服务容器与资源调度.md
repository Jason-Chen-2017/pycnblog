## 4.1 代码实例

```python
import heapq

def resource_scheduler(resources, demands):
    # 创建一个优先级队列
    priority_queue = []

    # 将资源和需求添加到优先级队列中
    for resource, demand in zip(resources, demands):
        heapq.heappush(priority_queue, (-demand, resource))

    # 初始化一个资源分配列表
    resource_allocation = []

    # 循环分配资源
    while priority_queue:
        # 获取资源和需求
        demand, resource = heapq.heappop(priority_queue)

        # 分配资源
        resource_allocation.append(resource)

        # 更新需求
        demands[resource] -= demand

        # 如果需求还没有满足，则将需求添加回优先级队列
        if demands[resource] > 0:
            heapq.heappush(priority_queue, (-demands[resource], resource))

    return resource_allocation
```

这个代码实例是一个简单的资源调度算法，它使用了优先级队列来实现资源的分配。

首先，我们创建了一个优先级队列，并将资源和需求添加到优先级队列中。然后，我们初始化了一个资源分配列表。

接下来，我们循环分配资源。在每次循环中，我们从优先级队列中获取资源和需求。然后，我们分配资源，并更新需求。如果需求还没有满足，我们将需求添加回优先级队列。
