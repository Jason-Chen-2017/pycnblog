# 2.核心概念与联系
## 2.1 AI神经网络基础概念
### 2.1.1 什么是神经网络？
神经网络是一种由多层节点组成的计算模型，每个节点都接受输入信号并根据其权重和偏置进行计算。这些计算结果被传递给下一层节点，直到最后一层产生输出结果。神经网络通过训练来学习从输入到输出之间的关系。
### 2.1.2 什么是深度学习？
深度学习是一种特殊类型的机器学习方法，它使用多层次结构的神经网络进行自动化学习。深度学习可以处理更复杂的数据结构和任务，例如图像、语音和自然语言处理等。深度学习已被应用于许多领域，包括图像识别、语音识别、机器翻译等。
## 2.2 人类大脑神经系统基础概念
### 2.2.1 大脑结构与功能
大脑是人类身体中最复杂且高级的组织之一，主要负责控制身体活动、感知环境、思考和记忆等功能。大脑由两个半球组成：左半球负责语言、逻辑思维等左侧功能；右半球负责视觉、空间关系等右侧功能；两个半球之间通过前枢纤维连接起来共同协调整身体运动和感知环境等功能。
### 2.2.2 大脑神经元与信息传递
大脑中每个细胞称为“神经元”或“neuron”（也称为“单元”）,它们相互连接形成复杂网状结构,这就形成了大脑内部所有信息传递路径,每条路径都由若干个相互连接起来的细胞组成,这些细胞可以传递电信号或化学信号,从而实现大脑内部各部分之间的信息交流和协同工作
