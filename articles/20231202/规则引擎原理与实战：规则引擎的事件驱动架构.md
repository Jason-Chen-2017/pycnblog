```python
import threading
import time
from queue import Queue

# 定义事件类
class Event:
    def __init__(self, name, data):
        self.name = name
        self.data = data

# 定义事件处理器类
class EventHandler:
    def __init__(self, name):
        self.name = name

    def handle_event(self, event):
        print(f"{self.name} is handling event {event.name}")

# 定义规则引擎类
class RuleEngine:
    def __init__(self):
        self.event_queue = Queue()
        self.handlers = []

    def add_event_handler(self, handler):
        self.handlers.append(handler)

    def start(self):
        while True:
            event = self.event_queue.get()
            for handler in self.handlers:
                handler.handle_event(event)

# 定义事件生成器类
class EventGenerator:
    def __init__(self, event_queue):
        self.event_queue = event_queue

    def generate_event(self):
        event = Event("event1", {"data": "data1"})
        self.event_queue.put(event)

# 主程序
if __name__ == "__main__":
    event_queue = Queue()
    event_generator = EventGenerator(event_queue)
    rule_engine = RuleEngine()
    event_handler = EventHandler("event_handler")

    rule_engine.add_event_handler(event_handler)
    threading.Thread(target=event_generator.generate_event, args=()).start()
    rule_engine.start()
```

上述代码实例中，我们定义了以下几个类：

2. `EventHandler` 类：用于定义事件处理器，包括事件处理器的名称和处理事件的逻辑。
3. `RuleEngine` 类：用于定义规则引擎，包括事件队列、事件处理器列表等。
4. `EventGenerator` 类：用于定义事件生成器，包括事件队列、事件生成逻辑等。

在主程序中，我们创建了一个事件生成器和一个规则引擎实例，并将事件处理器添加到规则引擎中。然后，我们启动事件生成器和规则引擎，使其可以处理事件。
