# 2.核心概念与联系
## 2.1微服务架构概述
微服务架构是一种新兴的软件架构风格，它将单个应用程序分解为小型服务，这些服务可以独立部署、运行和扩展。每个微服务都是完全自治的，可以通过网络进行通信。这种设计方式有助于提高系统的灵活性、可扩展性和稳定性。

## 2.2Spring Boot简介
Spring Boot是一个开源框架，它使得构建独立的、生产就绪的Spring应用程序变得简单。Spring Boot提供了许多工具和功能来帮助开发人员快速创建、配置和部署Spring应用程序。它还提供了许多预先配置好的依赖项，以便开发人员可以专注于编写业务逻辑而不需要关心底层技术细节。
