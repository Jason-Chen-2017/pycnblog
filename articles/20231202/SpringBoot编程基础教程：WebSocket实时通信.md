# 2.核心概念与联系
## 2.1 WebSocket简介
WebSocket是一种基于TCP协议的全双工通信协议，它允许客户端和服务器之间建立持久化连接，从而实现实时数据传输。WebSocket比传统HTTP协议更高效，因为它避免了不必要的连接重建和数据传输延迟。此外，WebSocket还支持二进制数据传输，使其在处理大量数据时更加高效。

## 2.2 Spring Boot简介
Spring Boot是一个用于构建Spring应用程序的框架，它提供了许多便捷的功能，包括自动配置、依赖管理、嵌入式服务器等。Spring Boot使得开发人员可以快速地创建可扩展的企业级应用程序，而无需关注底层细节。同时，Spring Boot也支持WebSocket协议，使得开发人员可以轻松地集成WebSocket功能到他们的应用程序中。

## 2.3 WebSocket与Spring Boot之间的关系
Spring Boot为开发人员提供了一种简单且高效的方式来集成WebSocket功能到他们的应用程序中。通过使用Spring Boot框架和相关组件（如Stomp协议），开发人员可以轻松地创建WebSocket服务器和客户端应用程序。此外，Spring Boot还提供了丰富的API和工具来帮助开发人员处理WebSocket连接、消息传输等任务。总之，Spring Boot为开发人员提供了一种简单且高效的方式来集成WebSocket功能到他们的应用程序中。
