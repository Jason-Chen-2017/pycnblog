# 2.核心概念与联系
## 2.1概率论与统计学基础知识
### 2.1.1概率论基础知识
概率论是数学分支之一，主要研究事件发生的可能性和相关概念。概率是一个数值，表示事件发生的可能性。它通常取值于0到1之间，表示事件发生的可能性。例如：抛硬币的正面朝上（正面）出现的概率为0.5（或50%）；同时两个硬币正面朝上出现的概率为0.25（或25%）。
### 2.1.2统计学基础知识
统计学是一门研究从数据中抽取信息并进行推理和预测的科学。统计学主要研究样本和总体之间的关系以及样本数据如何影响总体参数估计和假设检验结果。例如：从一个总体中抽取一个样本并对样本数据进行分析以得出关于总体参数估计或者对某种假设进行检验等结果。
