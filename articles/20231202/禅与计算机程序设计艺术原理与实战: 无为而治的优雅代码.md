## 3.1动态规划算法原理
动态规划（Dynamic Programming）是一种求解最优解问题的方法，它通过分步递推求解子问题得到最终结果。动态规划主要包括三个步骤：初始化、状态转移方程和终止条件。动态规划可以应用于各种类型的问题，如最长公共子序列（Longest Common Subsequence, LCS）、0-1背包问题（0-1 Knapsack Problem）等。下面我们详细讲解动态规划算法原理及其应用示例：
### 3.1.1初始化阶段
在初始化阶段，我们需要为每个子问题创建一个状态表格dp[i][j]（其中i表示当前字符串第i个字符，j表示目标字符串第j个字符）。同时，我们还需要定义一个变量n表示字符串长度（n=len(s)）。初始化完成后，我们将dp[i][0]置为0（因为目标字符串第一个字符必须是空字符串）：
$$dp[i][0] = 0$$
### 3.1.2状态转移方程阶段
接下来我们需要定义状态转移方程来描述从当前状态到下一个状态之间的关系：
$$dp[i][j] = max(dp[i-1][j], dp[i-1][j-1] + f(s_i, t_j))$$其中f(s_i, t_j)表示将当前字符串第i个字符添加到目标字符串第j个字符所形成的新子序列长度；如果两个字符相同则增加1；否则不增加长度。这里使用max函数选择两种情况下较大值作为当前状态值dp[i][j]。同时注意边界条件：如果当前位置已经遍历完成（即$ i = n $或$ j = n $），那么dp[n][n]就是最终结果；否则需要继续进行递归求解直至所有位置都被遍历完毕。### 3.1.3终止条件阶段终止条件是指何时停止递归求解并返回结果：当$ i = n $或$ j = n $时停止递归求解并返回结果；否则继续进行递归求解直至所有位置都被遍历完毕。总结一下动态规划三个步骤：初始化阶段：定义状态表格dp[][]并初始化各元素；状态转移方程阶段：根据当前状态得出下一个状态；终止条件阶段：确定何时停止递归求解并返回结果。### 3.2贪心算法原理
