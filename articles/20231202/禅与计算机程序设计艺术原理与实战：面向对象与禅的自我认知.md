## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的例子来说明面向对象编程的具体实现。我们将实现一个简单的动物类和子类的例子。

### 4.1动物类

首先，我们需要创建一个动物类，该类包含一个名为`name`的属性和一个名为`speak`的方法。

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name} says hello!")
```

在这个类中，我们定义了一个`__init__`方法，该方法用于初始化对象的属性。我们还定义了一个`speak`方法，该方法用于输出动物的名字和一个默认的问候语。

### 4.2子类

接下来，我们需要创建一个子类，该类继承自动物类，并实现自己的`speak`方法。

```python
class Dog(Animal):
    def speak(self):
        print(f"{self.name} says woof!")
```

在这个子类中，我们继承了动物类的`speak`方法，并实现了自己的`speak`方法。当我们创建一个`Dog`对象并调用其`speak`方法时，它将输出该动物的名字和一个默认的汪汪声。

### 4.3测试代码

最后，我们需要创建一个测试代码，用于测试我们的类和方法。

```python
dog = Dog("Buddy")
dog.speak()  # Output: Buddy says woof!
```

在这个测试代码中，我们创建了一个`Dog`对象，并调用其`speak`方法。当我们运行这段代码时，输出结果为：Buddy says woof!
