# 2.核心概念与联系
## 2.1 随机变量与事件
随机变量是一个数值函数，它可以取不同的值，每个值都有相应的概率。事件是随机变量可能取得的某个特定值所对应的结果集合。例如，在一个简单的硬币投掷游戏中，随机变量可以表示硬币正面或反面出现次数；事件则可以表示硬币正面出现一次或多次。

## 2.2 条件概率与独立性
条件概率是指给定某个事件已经发生了（称为条件），另一个事件发生的概率。独立性是指两个事件之间没有任何关联或者关联度很低（即知道其中一个事件发生了不会影响另一个事件发生的可能性）。例如：在抛硬币两次游戏中：抛出第一次硬币正面（H）或反面（T）后，第二次硬币正面或反面出现的条件概率都是0.5；而第一次和第二次硬币抛出之间是独立的（即知道第一次结果不会影响第二次结果）。
