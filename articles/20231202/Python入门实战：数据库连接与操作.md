## 1.5 Python数据库连接与操作的代码实例

以下是一个Python数据库连接与操作的代码实例：

```python
import sqlite3

# 建立数据库连接
conn = sqlite3.connect('example.db')

# 创建游标对象
cursor = conn.cursor()

# 执行SQL命令
cursor.execute('SELECT * FROM table_name')

# 获取查询结果
results = cursor.fetchall()

# 处理查询结果
for row in results:
    print(row)

# 关闭数据库连接
conn.close()
```
