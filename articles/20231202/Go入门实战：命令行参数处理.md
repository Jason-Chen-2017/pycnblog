以下是一个简单的Go程序示例，展示了如何处理命令行参数：

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    args := os.Args
    if len(args) < 2 {
        fmt.Println("No arguments provided.")
        os.Exit(1)
    }

    for i := 1; i < len(args); i++ {
        fmt.Printf("Argument %d: %s\n", i, args[i])
    }
}
```

在这个示例中，我们首先获取命令行参数`os.Args`，然后检查参数数量。如果参数数量小于2，我们打印一条错误消息并退出程序。接下来，我们遍历`os.Args`数组，并按顺序打印每个参数。

3. Q: 如何处理命令行参数？
