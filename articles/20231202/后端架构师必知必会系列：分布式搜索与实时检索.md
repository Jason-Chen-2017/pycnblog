# 2.核心概念与联系

在分布式搜索与实时检索中，有几个核心概念需要我们了解：

1. 分布式系统：分布式系统是一种由多个计算节点组成的系统，这些节点可以在不同的地理位置，使用不同的硬件和软件。这种系统可以提供高可用性、高性能和高可扩展性。

2. 搜索引擎：搜索引擎是一种用于查找和检索信息的系统，它可以在大量数据源中查找相关的信息，并将其排序和返回给用户。

3. 分布式搜索：分布式搜索是指在分布式系统中进行搜索的技术，它可以在多个数据源和计算节点上进行搜索，并将结果聚合和返回给用户。

4. 实时检索：实时检索是指在数据更新后立即更新搜索结果的技术，它可以确保用户在搜索时获取最新的信息。

5. 索引：索引是搜索引擎中的一种数据结构，它用于存储和管理搜索引擎中的数据。索引可以加速搜索过程，并提高搜索的效率和准确性。

6. 查询：查询是用户向搜索引擎提出的请求，它可以是关键词查询、范围查询、排序查询等多种形式。
