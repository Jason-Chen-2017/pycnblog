# 2.核心概念与联系
## 2.1条件概率
条件概率是一个随机事件发生的可能性，给定另一个事件已经发生的情况下。例如，如果我们知道某个人是女性，那么他/她患上癌症的可能性会增加多少？这就是条件概率问题。

## 2.2贝叶斯定理
贝叶斯定理是一种推理方法，它允许我们根据现有信息更新我们对未知事实的信念。贝叶斯定理可以帮助我们计算条件概率，从而更好地预测和决策。
