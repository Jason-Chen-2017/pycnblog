                 

# 1.背景介绍

在AI和人工智能领域，概率论和统计学起着重要的作用。我们可以将问题建模并使用概率论和统计学方法来解决问题。 Naive Bayes, 决策树, 神经网络等算法通常使用数据进行构建。在Deep pursuit和Reinforcement learning中，我们都使用==偏概率==方法来构建模型。
## 1.1.概率论的发展
- 古代：古希腊では奇数は定数として考慮され、チェパスは概率木に従事した
- 被发现充满图案的推理过程，被称为的“观测模式”
- 当时的科学家们使用 statistics 和概率论来研究我们可以有信任的看到特定事件发生的比例
- 1585年：马尔科夫，创造出了一种可提供有关事件发生的概率的方法
- 中世纪：弗兰克 Hand 开发了随机应用的第一项系统
- 以上都是所谓的得 climbed up habit 的滴。
