## 4.1 示例应用：天气预报应用

我们将开发一个简单的天气预报应用，该应用可以获取用户的位置信息，并显示当前天气情况。

### 4.1.1 代码实例

```python
import requests
from bs4 import BeautifulSoup

# 获取用户的位置信息
def get_location():
    # 使用IP地址获取用户的位置信息
    ip_address = requests.get('http://ip.taobao.com/').text
    location = requests.get(f'http://ip.taobao.com/{ip_address}').text
    return location

# 获取天气预报
def get_weather(location):
    # 使用API获取天气预报
    api_key = 'your_api_key'
    url = f'http://api.openweathermap.org/data/2.5/weather?q={location}&appid={api_key}'
    response = requests.get(url).json()
    weather = response['weather'][0]['description']
    return weather

# 主函数
def main():
    location = get_location()
    weather = get_weather(location)
    print(f'当前天气：{weather}')

if __name__ == '__main__':
    main()
```

### 4.1.2 代码解释

- `get_location()`函数用于获取用户的位置信息，通过IP地址获取用户的位置。
- `get_weather()`函数用于获取天气预报，通过API获取当前位置的天气情况。
- `main()`函数是程序的主函数，负责调用上述两个函数，并输出当前天气情况。
