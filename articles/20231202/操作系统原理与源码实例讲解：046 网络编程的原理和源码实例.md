                 

# 1.背景介绍

网络编程是一种开发应用程序，旨在在端到端之间进行数据交换的编程。 这是与对邻近的服务器进行通信的内部通信相反的远程通信。网络编程涉及在客户端和服务端应用程序之间建立网络通信通道的方法。这些应用程序使用服务器和客户端端口来实现各种网络活动，如打开服务器、使用HTTP请求进行网络通信和关闭会话。

网络编程涉及TCP/IP套接字通信。套接字是用于网络传输数据的极小模块。套接字可以将数据发送或接收到TCP/IP可路由地址。TCP/IP用于通过独立路由器或其他网络设备传输数据的协议。
