# 2.核心概念与联系
## 2.1 AI神经网络原理与人类大脑神经系统原理联系
### 2.1.1 AI神经网络原理简介
AI神经网络是一种由多个节点组成的复杂系统，它们之间有权重和偏置。这些节点可以通过输入层、隐藏层和输出层进行组织。每个节点都接收来自前一层的信号并对其进行加权求和，然后使用激活函数对结果进行非线性变换。最终得到的结果被传递给下一层或作为输出结果返回给用户。
### 2.1.2 人类大脑神经系统原理简介
人类大脑是一个复杂而高度连接的神经系统，由数十亿个神经元（也称为neuron）组成。每个神经元都有许多输入连接和一个输出连接，这些连接被称为synapses或突触。当一个电信号（即action potential或动作潜力）到达某个neuron时，它会根据该neuron上所有synapse上存储的权重值对信号进行加权求和，然后通过激活函数对结果进行非线性变换。最终得到的结果被传递给下一个neuron或导致动作响应（如身体运动）返回给用户。
### 2.1.3 AI神经网络与人类大脑相似之处：
- **同构**：两者都由多个节点组成并且每个节点都有自己独特的权重值；
- **异构**：两者都包含不同类型的节点（如感知器/感知单元/感知元素/感知单元/感知单位/感知器/感知器/感知器/感知器/感知器/感知器）；
- **非线性**：两者都使用激活函数对信号进行非线性变换；
- **学习**：两者都可以通过训练数据更新权重值以改善预测性能；
