                 

# 1.背景介绍


编译器是由程序设计语言的符号、语法和语义规则构成的编译辅助系统。编译器通常有前端和后端两个部分，前端负责理解源代码的结构和语法，将其翻译成一种中间表示，并在后端执行操作；后端执行操作将中间表示翻译成目标执行代码。本文主要讨论目标代码在不同平台之间的可移植性问题。

目标代码的可移植性包括编译结果的效率、目标机器数量和性能差异等多个因素。在不同平台之间的可移植性有很多种方法，例如对目标代码的不同平台的实现进行分离，或在目标代码中加入目标定义的类型。在不同平台之间的可移植性问题可能分为很多子问题，因此需要结合不同方法来解决这些问题，并考虑不同平台的性能差异。
