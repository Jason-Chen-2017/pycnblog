# 2.核心概念与联系
循环神经网络（RNN）是一种特殊的神经网络结构，它可以处理序列数据。RNN通过在隐藏层中保持状态来记住之前的输入信息。这使得RNN能够在处理长序列数据时避免信息丢失问题。另外，RNN还可以通过更新隐藏状态来适应不同类型的输入数据。这使得RNN成为处理自然语言、音频和视频等序列数据的首选方案。
