## 3.1 概率论基础
概率论是一门研究随机事件发生概率的科学。在AI人工智能中，我们需要使用概率论来处理随机事件，以便更好地理解数据之间的关系和模式。

### 3.1.1 概率的基本定义
概率是一个随机事件发生的可能性，通常用P(E)表示，其中P表示概率，E表示事件。

### 3.1.2 概率的计算方法
1. 直接计数法：直接计算所有可能的结果，然后计算满足条件的结果的比例。
2. 定义域法：将事件空间划分为两个部分，满足条件的部分和不满足条件的部分，然后计算满足条件的部分的比例。
3. 比例法：将事件空间划分为两个部分，满足条件的部分和不满足条件的部分，然后计算满足条件的部分的比例。

### 3.1.3 概率的几种性质
1. 非负性：概率的值必须大于等于0。
2. 完整性：概率的和必须等于1。
3. 交换性：对于两个事件A和B，有P(A∩B)=P(B∩A)。

## 3.2 统计学基础
统计学是一门研究如何从数据中抽取信息的科学。在AI人工智能中，我们需要使用统计学来处理数据，以便更好地理解数据之间的关系和模式。

### 3.2.1 统计学的基本概念
1. 样本：从总体中随机抽取的一组数据。
2. 总体：所有满足某一特定条件的事件或数据的集合。
3. 参数：总体的某个特征值。
4. 统计量：样本的某个特征值。

### 3.2.2 统计学的基本方法
1. 描述性统计：用于描述数据的方法，如计算平均值、中位数、方差等。
2. 推断统计：用于从样本中推断总体特征的方法，如计算置信区间、检验假设等。

### 3.2.3 常用的统计学公式
1. 平均值：(x1+x2+...+xn)/n
2. 中位数：对于有序数据，中位数是数据集中间的数值。
3. 方差：(x1-mean)^2+(x2-mean)^2+...+(xn-mean)^2/n
4. 标准差：方差的平方根。

## 4.1 概率论的Python实例
```python
import random

# 直接计数法
def direct_count_law(total, satisfied):
    return satisfied / total

# 定义域法
def domain_law(total, satisfied):
    return satisfied / total

# 比例法
def ratio_law(total, satisfied):
    return satisfied / total

# 测试
total = 1000
satisfied = 800

print("直接计数法:", direct_count_law(total, satisfied))
print("定义域法:", domain_law(total, satisfied))
print("比例法:", ratio_law(total, satisfied))
```

## 4.2 统计学的Python实例
```python
import numpy as np

# 计算平均值
def mean(data):
    return np.mean(data)

# 计算中位数
def median(data):
    return np.median(data)

# 计算方差
def variance(data):
    return np.var(data)

# 计算标准差
def stddev(data):
    return np.std(data)

# 测试
data = np.array([1, 2, 3, 4, 5])

print("平均值:", mean(data))
print("中位数:", median(data))
print("方差:", variance(data))
print("标准差:", stddev(data))
```
