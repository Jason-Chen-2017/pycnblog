# 2.核心概念与联系
原子能是指一种能量来源，它来自于原子核的分裂或融合。原子核是原子的核心部分，由一些子atomic的核子组成。原子核分裂和融合是原子能的核心概念，它们都会释放大量的能量。

原子核分裂是指原子核内部的一些核子分离，从而释放能量。原子核融合是指原子核内部的一些核子相互结合，从而释放能量。原子核分裂和融合都会释放大量的能量，这种能量可以用来驱动各种设备和机器。

原子能的发展历程可以分为以下几个阶段：

1. 原子核发现：在20世纪初，科学家发现了原子核的存在，这是原子能的发展的基础。

2. 原子核分裂：在20世纪初，科学家发现了原子核分裂的方法，这是原子能的第一次应用。

3. 原子核融合：在20世纪中叶，科学家发现了原子核融合的方法，这是原子能的第二次应用。

4. 原子弹发明：在20世纪中叶，科学家利用原子核分裂和融合的方法发明了原子弹，这是原子能的第一次大规模应用。

5. 原子能发电：在20世纪后期，科学家开始研究如何利用原子核分裂和融合的能量来发电，这是原子能的第二次大规模应用。
