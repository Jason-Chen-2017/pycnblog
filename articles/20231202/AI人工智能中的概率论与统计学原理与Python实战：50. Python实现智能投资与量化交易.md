# 2.核心概念与联系
在进入具体内容之前，我们需要了解一些关键概念：
- **机器学习**：机器学习是一种自动化方法，它允许计算机从数据中自动学习和预测未来结果。它是人工智能领域的一个重要部分。
- **深度学习**：深度学习是一种特殊类型的机器学习方法，它使用多层神经网络来处理复杂问题。深度学习已被证明对于图像识别、语音识别等任务非常有效。
- **回归**：回归是一种预测方法，它试图预测一个变量的值基于其他变量的值。在金融领域中，回归可以用于预测股票价格、汇率等变量。
- **分类**：分类是一种预测方法，它试图将数据点分为不同的类别或组合。在金融领域中，分类可以用于判断某个股票是否会上涨或下跌、判断某个企业是否会破产等任务。
- **贝叶斯定理**：贝叶斯定理是概率论中一个重要原则，它描述了如何根据新信息更新现有知识。贝叶斯定理在机器学习和人工智能领域具有广泛应用。
- **Markov链**：Markov链是一种随机过程模型，它描述了随时间推移状态转移的概率性质。Markov链在金融市场预测中具有广泛应用，例如股票价格预测、汇率预测等任务。
