# 2.核心概念与联系
## 2.1概率论与统计学基本概念
### 2.1.1随机事件
随机事件是指无法预测确切结果的事件，其结果只能通过某种方法得到估计或推测。例如：抛硬币的结果（正面或反面）、抽卡牌（红桃或黑桃）等。
### 2.1.2样本空间
样本空间是所有可能发生的结果组成的集合，表示了一个随机事件可以取得哪些值。例如：抛硬币两次得到正面或反面（HH,HT,TH,TT）组成样本空间。
### 2.1.3事件
事件是样本空间中某一子集的集合，表示某一特定结果发生或不发生。例如：抛硬币两次得到正面（HH,HT,TH）组成一个事件。
### 2.1.4条件概率
条件概率是已知某个事件A发生时，另一个事件B发生的概率。记作P(B|A)，表示当A已经发生时，B还有多大可能性会发生。例如：已知第一张卡片是红桃时，第二张卡片也是红桃的条件概率为0.5。
### 2.1.5独立性
两个或多个事件之间独立意味着它们之间没有任何关联关系，其发生或不发生对于彼此都没有影响。例如：抛硬币两次获得正面和抽卡牌获得红桃这两个事件相互独立。
