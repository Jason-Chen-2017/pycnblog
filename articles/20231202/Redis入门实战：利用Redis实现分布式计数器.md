                 

# 1.背景介绍

Redis是一个开源的高性能key-value存储系统，它支持数据的持久化，可基于内存（Redis）或磁盘（Redis-Disk）。Redis 提供多种语言的 API。

Redis 是一个使用 ANSI C 语言编写、遵循 BSD 协议的开源 ( BSD ) 软件。 Redis 的根目录下包含了完整的源代码。

Redis 是综合性的内存数据库管理系统，它支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。

Redis 提供了多种语言（如：Ruby、Python、Go、Java、PHP、Node.js、C#等）的 API。
