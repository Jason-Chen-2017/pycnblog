# 2.核心概念与联系
## 2.1 AI与深度学习
深度学习是一种特殊类型的机器学习方法，它主要基于人工神经网络（ANN）来模拟大脑思维方式进行问题解决。深度学习可以处理复杂问题并从大量数据中自动学习出模式和规律。AI则是指通过计算机程序模拟或扩展人类智能的技术范畴。因此，深度学习是AI领域中非常重要且广泛应用的技术之一。
## 2.2 神经元与层次结构
一个简单的神经元包含输入、输出和权重三个部分：输入为信号源；权重为信号传递时所需要调整的参数；输出为信号接收端。在一个完整的神经网络中，每个层次都有不同数量和布局的神经元组成。通常情况下，第一层称为输入层、最后一层称为输出层；之间的层称为隐藏层（如果存在）。每个连接都有一个权重值表示强度或相关性；这些权重值会随着训练而更新以达到最佳效果。
## 2.3 激活函数与损失函数
激活函数是控制输出值变化范围和速率的函数；损失函数衡量预测结果与真实结果之间差异大小（误差）；损失函数越小表示预测结果越准确。常见激活函数有sigmoid、tanh等；常见损失函数有均方误差（MSE）、交叉熵等。激活函数对于避免过拟合至关重要；损失函数对于评估模型性能至关重要。
