# 2.核心概念与联系
## 2.1概率论基础
概率论是数学统计学中的一个分支，研究事件发生的可能性。概率可以用来描述事件发生的可能性、相互依赖关系等。主要包括：
- 定义：事件A发生的概率P(A)表示事件A在所有可能结果中出现的次数占总次数之比。
- 几种基本定理：包括总体定理、贝叶斯定理等。
- 独立性：两个或多个事件之间没有任何相互依赖关系，即知道其中一个发生或不发生对另一个事件发生或不发生的概率都不会产生影响。
- 条件独立性：当给定某些条件时，两个或多个事件之间具有独立性。例如，当天气晴天时，雨和雪是独立的；当天气阴天时，雨和雪是相互依赖的。
- 随机变量：随机变量是一个函数从一组输入值到一组输出值（称为范围）上映射。随机变量可以用来描述某种现象在不同情况下取得不同值得程度。主要包括离散型随机变量和连续型随机变量两类。离散型随机变量只能取有限个值（如硬币投掷结果）；连续型随机变量可以取无限个值（如温度）。常见类型包括均匀分布、指数分布、正态分布等。
- 期望：期望是随机变量取值平均值所代表的期望值（即期望），用于衡量某种现象在长期观测下取得平均值得程度。例如，硬币投掷结果中正面朝上和反面朝上各出现一半时，硬币正面朝上出现的期望为0.5；若硬币正面朝上出现3/4时，则硬币正面朝上出现的期望为0.75等等。期望也被称为“预期”或“平均”等词汇表示意义相似但含义略有差异（如预期往往强调对未来进行猜测而非历史数据推导）
