## 1.4 具体代码实例和详细解释说明

### 1.4.1 代码实例

本节将给出一个简单的图算法的代码实例，即最短路径算法的实现。

```python
import heapq

def dijkstra(graph, start):
    distances = [float('inf')] * len(graph)
    distances[start] = 0
    pq = [(0, start)]
    while pq:
        current_distance, current_vertex = heapq.heappop(pq)
        if current_distance > distances[current_vertex]:
            continue
        for neighbor, weight in graph[current_vertex].items():
            distance = current_distance + weight
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))
    return distances
```

### 1.4.2 详细解释说明

上述代码实例是一个简单的最短路径算法的实现，它使用了堆优先队列（heapq）来实现。具体的解释说明如下：

1. 初始化距离数组：距离数组用于存储每个顶点到起始顶点的最短距离，初始值为无穷大。
2. 初始化堆优先队列：堆优先队列用于存储当前最短距离和对应的顶点。
3. 遍历堆优先队列：遍历堆优先队列，找到当前最短距离和对应的顶点。
4. 更新距离：如果当前顶点的距离小于距离数组中的值，则更新距离数组。
5. 更新堆优先队列：更新堆优先队列，以便在下一次遍历时可以找到更短的距离。
6. 返回距离数组：返回距离数组，其中每个元素表示从起始顶点到对应顶点的最短距离。
