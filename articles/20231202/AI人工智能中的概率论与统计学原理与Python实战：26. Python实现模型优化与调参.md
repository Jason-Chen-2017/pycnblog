# 2.核心概念与联系
在进入具体内容之前，我们需要先了解一下相关的核心概念：
- **模型优化**：指在训练过程中通过调整网络结构、更新算法等手段来提高模型性能。
- **调参**：指通过对网络结构、学习率等超参数进行微调来提高模型性能。
- **搜索空间**：指所有可能取值范围的集合。在模型优化和调参中，搜索空间包括网络结构、学习率等各种超参数。
- **评估标准**：用于衡量模型性能的标准或指标，如准确率、F1分数等。
- **交叉验证**：一种验证方法，通过将数据集划分为多个子集（训练集、验证集、测试集）来评估模型性能。
