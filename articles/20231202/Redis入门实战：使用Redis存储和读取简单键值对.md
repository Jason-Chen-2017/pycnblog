Redis（Remote Dictionary Server）是一个开源的高性能的键值对存储系统，由Salvatore Sanfilippo开发。它支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。Redis的数据结构支持简单的字符串（string）、哈希（hash）、列表（list）、集合（set）和有序集合（sorted set）等。Redis 支持提供数据备份、给定键的时间到期（key expiration）和自动删除键的功能。

Redis 和关系型数据库（RDBMS）之间的一个主要区别是，Redis是内存中的数据库，而关系型数据库是基于磁盘的。Redis 的数据结构支持键值对存储，而关系型数据库的数据结构支持表（table）。Redis 是一个单线程的数据库，而关系型数据库是多线程的。Redis 是一个开源的数据库，而关系型数据库可以是开源的（如 MySQL 、PostgreSQL ）或者商业的（如 Oracle 、Microsoft SQL Server ）。
