# 2.核心概念与联系

## 2.1 内存模型

内存模型是计算机编程语言中一个重要概念，它描述了程序在执行过程中如何访问和操作内存。内存模型规定了多个线程之间如何同步访问共享变量，以及如何处理缓存一致性等问题。通过了解内存模型，我们可以更好地控制程序的执行流程，避免数据竞争和其他并发相关问题。

## 2.2 并发

并发是指多个任务同时进行的情况，这些任务可以在单个处理器或多个处理器上运行。在计算机编程语言中，并发是一个重要的特性，它允许我们利用多核处理器提高应用程序的性能。然而，并发也带来了新的挑战，例如数据竞争、死锁等问题需要我们进一步研究和解决。
