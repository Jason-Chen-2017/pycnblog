## 4.1 规则引擎的代码实例

### 4.1.1 规则定义

```python
from jython.rules.core import Rule

class MyRule(Rule):
    def __init__(self):
        super(MyRule, self).__init__()
        self.name = "my_rule"
        self.condition = "event == 'start'"
        self.actions = ["print('started')"]

    def execute(self, context):
        for action in self.actions:
            eval(action, context)
```
