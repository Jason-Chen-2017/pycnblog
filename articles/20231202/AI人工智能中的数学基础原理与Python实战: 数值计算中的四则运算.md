# 4.具体代码实例和详细解释说明

在Python中，我们可以使用内置的数学库来实现四则运算。以下是具体的代码实例和解释：

```python
import math

# 加法
def add(a, b):
    return a + b

# 减法
def subtract(a, b):
    return a - b

# 乘法
def multiply(a, b):
    return a * b

# 除法
def divide(a, b):
    return a / b

# 测试代码
a = 10
b = 5

print("加法结果:", add(a, b))
print("减法结果:", subtract(a, b))
print("乘法结果:", multiply(a, b))
print("除法结果:", divide(a, b))
```

在这个代码中，我们使用了Python的内置数学库来实现四则运算。我们定义了四个函数（add、subtract、multiply和divide）来实现加法、减法、乘法和除法的运算。然后，我们使用测试代码来验证这些函数的正确性。
