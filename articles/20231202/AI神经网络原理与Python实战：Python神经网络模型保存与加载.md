## 创建神经网络模型

首先，我们需要创建一个简单的神经网络模型。在本例中，我们将使用Keras库来创建一个简单的神经网络模型。

```python
import numpy as np
from keras.models import Sequential
from keras.layers import Dense

# 创建神经网络模型
model = Sequential()
model.add(Dense(32, input_dim=784, activation='relu'))
model.add(Dense(10, activation='softmax'))

# 编译模型
model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])
```

在这个例子中，我们创建了一个简单的神经网络模型，它有一个输入层和一个输出层。输入层有784个节点，输出层有10个节点。我们使用了ReLU激活函数和softmax激活函数。

## 保存神经网络模型

接下来，我们需要将神经网络模型保存到磁盘上。

```python
# 保存模型
save_model(model, 'model.pkl')
```

在这个例子中，我们使用`save_model`函数将神经网络模型保存到磁盘上。

## 加载神经网络模型

最后，我们需要从磁盘上加载神经网络模型。

```python

```

在这个例子中，我们使用`load_model`函数从磁盘上加载神经网络模型。
