```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

## 4.2 详细解释说明

在这个代码实例中，我们使用了插入排序算法来对一个数组进行排序。插入排序算法的时间复杂度为O(n^2)，空间复杂度为O(1)。

在这个算法中，我们首先从第二个元素开始，将其与前面的元素进行比较。如果当前元素小于前一个元素，我们将前一个元素向后移动，直到找到正确的位置，然后将当前元素放入该位置。这个过程会重复进行，直到所有元素都被排序。
