# 2.核心概念与联系
Quasar框架是基于Vue.js的，因此需要对Vue.js有一定的了解。Vue.js是一个进化迅速的JavaScript框架，它采用了MVVM（Model-View-ViewModel）模式来组织数据和UI视图之间的关系。这种模式使得视图和数据之间保持同步，并且可以轻松地进行单向数据绑定。

Quasar框架扩展了Vue.js的功能，提供了许多有用的工具和组件来帮助开发者更快地构建跨平台应用程序。这些工具包括：
- 自动化构建系统：通过Webpack等工具自动化构建项目，减少手动配置的时间和精力；
- 跨平台支持：通过使用Cordova或Electron等技术实现Android、iOS、Windows等多个平台的支持；
- 丰富的组件库：提供了大量预制好的UI组件，如按钮、输入框、弹出层等；
- 强大的状态管理：通过Vuex等工具实现状态管理，让应用程序更容易维护和扩展；
- 集成第三方服务：如Google Maps API等第三方服务集成到项目中；
- 高性能优化：通过各种优化手段提高应用程序性能。
