# 2.核心概念与联系

在讨论迁移学习之前，我们需要了解一些基本的概念。首先，我们需要了解什么是神经网络，以及它与人类大脑神经系统的联系。

神经网络是一种模拟人类大脑神经系统的计算模型，它由多个相互连接的节点组成，这些节点称为神经元或神经网络。神经网络可以学习从数据中提取特征，并使用这些特征来预测或分类数据。

人类大脑神经系统是一种复杂的计算机，它可以处理大量的信息并进行高度并行的计算。神经网络试图模拟这种计算方式，以便在处理大量数据时更有效地进行计算。
