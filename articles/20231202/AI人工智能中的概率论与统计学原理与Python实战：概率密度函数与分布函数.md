```python
import numpy as np

def pdf_uniform(x, a, b):
    return 1 / (b - a) if a <= x <= b else 0

def cdf_uniform(x, a, b):
    return (x - a) / (b - a) if a <= x <= b else 0
```

## 4.2正态分布的概率密度函数和分布函数

```python
import numpy as np
from scipy.stats import norm

def pdf_normal(x, mu, sigma):
    return norm.pdf(x, loc=mu, scale=sigma)
```
