1.4.2 自然语言处理代码实例：我们可以通过使用Python的NLTK库来实现自然语言处理任务，如文本分类、文本摘要等。例如，我们可以使用NLTK来实现一个简单的文本分类任务，如下所示：

```python
import nltk
from nltk.corpus import stopwords
from nltk.tokenize import word_tokenize
from nltk.stem import PorterStemmer

# 加载数据
data = nltk.corpus.stopwords.words('english')

# 预处理数据
def preprocess(text):
    tokens = word_tokenize(text)
    stemmer = PorterStemmer()
    stemmed_tokens = [stemmer.stem(token) for token in tokens]
    return stemmed_tokens

# 训练模型
classifier = nltk.NaiveBayesClassifier.train(
    [(preprocess(text), category) for (text, category) in data]
)

# 预测
predicted_category = classifier.classify(preprocess(test_text))
```
