                 

# 1.背景介绍

Redis是一个开源的高性能key-value存储系统，它支持数据的持久化、重plication、集群等特性。Redis 提供了多种语言的 API，包括 Ruby、Python、Java 和 PHP。Redis 可以用作数据库、缓存和消息队列。

Redis 是一个使用 ANSI C 语言编写的 open-source（开源）、无License（没有许可证）的内存级数据结构存储系统，它支持数据的持久化、复制、簇状等特性。Redis 提供了多种语言的 API，包括 Ruby、Python、Java 和 PHP。Redis 可以用作数据库、缓存和消息队列。
