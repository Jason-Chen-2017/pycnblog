# 2.核心概念与联系
在进入具体内容之前，我们需要了解一下数据中台的核心概念：
- **数据湖**：是指一个大型、分布式的存储系统，用于存储各种类型的结构化和非结构化数据。它允许用户轻松地查询、分析和可视化这些数据。
- **元数据**：是指关于数据库或其他信息系统的信息。元数据包括了对象名称、类型、大小等属性信息。
- **计算引擎**：是指负责执行计算任务的组件。它可以处理各种类型的计算任务，如查询、分析、预测等。
- **流水线**：是指一系列相互依赖的任务或操作序列。在数据中台中，流水线用于实现不同阶段之间的逻辑连接和控制流程。
- **监控与预警**：是指对系统性能、资源使用情况等方面进行实时监控和预警机制的设置。通过监控与预警，企业可以更快地发现问题并采取相应措施进行解决。
