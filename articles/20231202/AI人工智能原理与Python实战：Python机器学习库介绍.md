人工智能（Artificial Intelligence），是一门跨学科的研究学科，研究自然界的智能集聚体之一人，如能否独立或与人类与互动并拥有一些人的智能能力，如信息处理、学习和传导。人工智能在人类体裁内存储和处理信息，以及处理复杂三维物体的能力等方面发展得越来越强大，并且这一发展可以继续持续下去。

尽管 AI 被广泛应用于全球范围内的信息技术和电子技术企业的产品品种和设备、医学、生物、金融、国防、有能源、 wiped aria ,机器语音用、无人驾驶汽车用、空间探测器、机器翻译用、教育、社会的各个领域的各种操作与安全管理业务 etc。AI 也不是一项普通的科学技术，而是一种渐变的研究。

随着计算机科学的研究，作为人工智能的核心部分的机器学习也得到了迅速的发展和崛起。

机器学习是一种算法。在 Python 中的 Python 原生库，如 Numpy，Scikit-Learn 和 TensorFlow，使得用户可以进行模型的训练和预测。在这些库中的大多数算法是来自于以下几个主要领域之一：

1.形象
2.回归
3.挖掘
4.神经

Python 涉及机器学习的理论知识应该足够深入，这是我们接下来要逐步深入讨论的内容。
