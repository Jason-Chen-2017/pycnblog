Spring Boot 是一个用于构建新的 Spring-based 应用程序以及 existing Spring 项目的快速开箱即用的框架。

将 Spring Boot 与 Spring Cloud 一起使用，则可以实现创建分布式系统的全功能支持。

基本上，Spring Boot 就是 Spring Cloud 的组成部分。我们部署 Spring Cloud 项目时，能够发现 Spring Boot 是暗示 Spring Cloud 编译器依赖云架构所需的子依赖。

```

        System.out.println("访问地址：http: //localhost:80" +
            "/测试");
    }
}
```
