# 4.具体代码实例和详细解释说明
在参与软件竞赛和奖项时，我们需要编写一些具体的代码实例。以下是一个简单的排序算法的实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(arr))
```

上述代码实现了一个冒泡排序算法，它的时间复杂度为O(n^2)，空间复杂度为O(1)。通过这个简单的例子，我们可以看到如何编写程序并解决问题。
