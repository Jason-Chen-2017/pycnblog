## 4.1 决策规则的代码实例
以下是一个简单的决策规则的代码实例：
```python
class DecisionRule:
    def __init__(self, condition, action):
        self.condition = condition
        self.action = action

    def execute(self, context):
        if self.condition(context):
            self.action(context)
```
在这个代码实例中，我们定义了一个 `DecisionRule` 类，它有一个条件和一个动作。条件用于判断是否满足决策规则，动作用于执行决策规则。

```python
class InferenceEngine:
    def __init__(self, rules):
        self.rules = rules

    def execute(self, context):
        for rule in self.rules:
            rule.execute(context)
```
在这个代码实例中，我们定义了一个 `InferenceEngine` 类，它有一组规则。规则用于描述推理过程，并根据这些规则生成推理结果。
