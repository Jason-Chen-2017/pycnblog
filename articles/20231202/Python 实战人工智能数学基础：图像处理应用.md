# 2.核心概念与联系
在进入具体内容之前，我们需要先了解一下图像处理中的一些基本概念：
- **图像**：是由数字信号组成的二维矩阵数据结构，每个元素称为**像素**（picture element）或**点**（pixel）。每个像素都有其对应的颜色值（RGB值）和亮度值（灰度值）。
- **图像处理**：是指对图像进行各种数学运算和变换以改变其特征或性质的过程。常见的图像处理操作包括滤波、边缘检测、形状识别等。
- **计算机视觉**：是一种通过计算机程序对视觉信息进行分析和理解的技术。计算机视觉主要包括图像处理、特征提取、模式识别等环节。
- **深度学习**：是一种基于神经网络模型的机器学习方法，它可以自动学习从大量数据中抽取出高级表达形式并进行预测或决策。深度学习已经成为现代计算机视觉系统中最重要且最活跃的研究领域之一。
