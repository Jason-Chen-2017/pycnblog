# 2.核心概念与联系

在讨论身份认证和授权的过程之前，我们需要了解一些核心概念。

## 2.1 身份认证

身份认证是确认用户是谁的过程。在互联网应用程序中，身份认证通常是通过用户名和密码的组合来实现的。当用户输入正确的用户名和密码时，应用程序会验证这些信息，并确认用户的身份。

## 2.2 授权

授权是确定用户可以执行哪些操作的过程。在互联网应用程序中，授权通常是通过角色和权限的组合来实现的。当用户被分配了特定的角色和权限时，应用程序会根据这些信息来确定用户可以执行哪些操作。

身份验证流程是身份认证和授权的整个过程。在身份验证流程中，用户首先需要通过身份认证来确认他们的身份，然后根据授权来确定他们可以执行哪些操作。
