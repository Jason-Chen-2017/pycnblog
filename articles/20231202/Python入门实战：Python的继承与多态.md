## 1.1 Python的继承与多态的基本概念

### 1.1.1 继承

继承是面向对象编程中的一种重要概念，它允许一个类从另一个类继承属性和方法。在Python中，我们使用`class`关键字来定义类，并使用`:`符号来继承父类的属性和方法。

例如，我们可以定义一个`Animal`类，并定义一个`Dog`类继承自`Animal`类：

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print("Animal speaks")

class Dog(Animal):
    def speak(self):
        print("Dog barks")

# 创建一个Dog对象
dog = Dog("Buddy")

# 调用Dog类的speak方法
dog.speak()  # 输出: Dog barks
```

在这个例子中，`Dog`类继承了`Animal`类的`speak`方法，但我们可以重写该方法以实现特定的行为。

### 1.1.2 多态

多态是面向对象编程中的另一个重要概念，它允许一个对象在不同的情况下表现出不同的行为。在Python中，我们可以通过继承和重写方法来实现多态。

继续上面的例子，我们可以定义一个`Cat`类，并实现其`speak`方法：

```python
class Cat(Animal):
    def speak(self):
        print("Cat meows")

# 创建一个Cat对象
cat = Cat("Whiskers")

# 调用Cat类的speak方法
cat.speak()  # 输出: Cat meows
```

在这个例子中，`Cat`类继承了`Animal`类的`speak`方法，但我们重写了该方法以实现特定的行为。这就是多态的体现。
