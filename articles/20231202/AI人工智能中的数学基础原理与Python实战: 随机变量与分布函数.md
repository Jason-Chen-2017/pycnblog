# 2.核心概念与联系

## 2.1随机变量

随机变量是一个可以取任意值的变量，其值的分布是随机的。随机变量可以分为两类：离散型随机变量和连续型随机变量。离散型随机变量的取值是有限的或可数的，例如骰子上的点数。连续型随机变量的取值是连续的，例如人体的身高。

## 2.2分布函数

分布函数是描述随机变量可能取值的概率的函数。给定一个随机变量X，其分布函数F(x)定义为：

$$
F(x) = P(X \leq x)
$$

其中，P(X ≤ x)是随机变量X的取值小于或等于x的概率。

## 3.1常见的随机变量分布

### 3.1.1均匀分布

均匀分布是一种连续型随机变量的分布，其分布函数为：

$$
F(x) = \begin{cases}
0, & x < a \\
\frac{x - a}{b - a}, & a \leq x < b \\
1, & x \geq b
\end{cases}
$$

其中，a和b是均匀分布的参数，表示随机变量的取值范围。

### 3.1.2指数分布

指数分布是一种连续型随机变量的分布，其分布函数为：

$$
F(x) = 1 - e^{-\frac{x}{\beta}}
$$

其中，β是指数分布的参数，表示随机变量的平均值。

### 3.1.3正态分布

正态分布是一种连续型随机变量的分布，其分布函数为：

$$
F(x) = \frac{1}{\sqrt{2\pi\sigma}} \int_{-\infty}^{x} e^{-\frac{(t-\mu)^2}{2\sigma^2}} dt
$$

其中，μ和σ是正态分布的参数，表示随机变量的均值和标准差。

## 3.2生成随机变量的方法

### 3.2.1采样方法

采样方法是通过从随机变量的分布中抽取样本来生成随机变量的值。例如，我们可以从均匀分布中抽取样本来生成均匀分布的随机变量。

### 3.2.2算法方法

算法方法是通过使用随机算法来生成随机变量的值。例如，我们可以使用随机数生成器来生成随机变量的值。

## 4.1生成均匀分布的随机变量

我们可以使用Python的numpy库来生成均匀分布的随机变量。以下是一个生成均匀分布的随机变量的Python代码实例：

```python
import numpy as np

# 生成均匀分布的随机变量
a = 0
b = 1
x = np.random.uniform(a, b, 1000)

# 打印随机变量的值
print(x)
```

在上述代码中，我们首先导入了numpy库，然后使用numpy的random.uniform函数来生成均匀分布的随机变量。函数的参数a和b分别表示随机变量的取值范围。最后，我们使用print函数来打印随机变量的值。

## 4.2生成指数分布的随机变量

我们可以使用Python的scipy库来生成指数分布的随机变量。以下是一个生成指数分布的随机变量的Python代码实例：

```python
import numpy as np
from scipy.stats import exponweib

# 生成指数分布的随机变量
beta = 1
x = exponweib.rvs(beta, 1000)

# 打印随机变量的值
print(x)
```

在上述代码中，我们首先导入了numpy和scipy库，然后使用scipy的stats.exponweib模块来生成指数分布的随机变量。函数的参数beta表示指数分布的参数。最后，我们使用print函数来打印随机变量的值。

## 4.3生成正态分布的随机变量

我们可以使用Python的scipy库来生成正态分布的随机变量。以下是一个生成正态分布的随机变量的Python代码实例：

```python
import numpy as np
from scipy.stats import norm

# 生成正态分布的随机变量
mu = 0
sigma = 1
x = norm.rvs(mu, sigma, 1000)

# 打印随机变量的值
print(x)
```

在上述代码中，我们首先导入了numpy和scipy库，然后使用scipy的stats.norm模块来生成正态分布的随机变量。函数的参数mu和sigma分别表示正态分布的均值和标准差。最后，我们使用print函数来打印随机变量的值。
