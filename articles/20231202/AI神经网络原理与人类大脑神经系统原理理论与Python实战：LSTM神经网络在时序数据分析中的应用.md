# 2.核心概念与联系
## 2.1 LSTM神经网络基础知识
LSTM（Long Short-Term Memory）是一种特殊类型的递归神经网络（RNN），它可以更好地处理长期依赖关系和时序数据。LSTM单元包含输入门、遗忘门和输出门三个部分：当前状态、隐藏状态和输出值。这些门控制信息流动方向和速度，从而有效地捕捉长期依赖关系。

## 2.2 RNN与LSTM的区别与联系
RNN（Recurrent Neural Networks）是一种循环结构的神经网络，它可以通过循环连接层次来处理序列数据。然而，传统的RNN存在“梯度消失”或“梯度爆炸”问题：随着时间步数增加，梯度会逐渐减小或变得非常大，导致训练难以收敛或不稳定。相比之下，LSTM通过引入门机制来解决这个问题：每个门都有自己独立的计算公式和权重参数集合，从而更好地控制信息流动方向和速度。因此，LSTM在处理长期依赖关系和时序数据方面比传统RNN更加强大。
