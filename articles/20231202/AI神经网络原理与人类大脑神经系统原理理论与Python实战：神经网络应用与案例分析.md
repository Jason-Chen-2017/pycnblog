# 2.核心概念与联系
## 2.1 AI与神经网络的关系
AI是一种计算机科学技术，旨在让计算机模拟人类智能。它包括多种方法和技术，如规则引擎、专家系统、遗传算法等。神经网络是AI中一个重要的方法之一，它试图通过模仿人类大脑中神经元之间的连接和信息处理方式来解决问题。因此，AI可以看作是一个更广泛的领域，而神经网络则是其中一个具体方法。
## 2.2 人类大脑与神经网络的联系
人类大脑是一个复杂且高度并行的信息处理系统。大脑内部有约100亿个神经元（neurons）和许多更小的胞质结构（glial cells）组成。这些神经元相互连接形成了大脑内部复杂但有序的信息传递路径。每个神经元都有输入端（dendrites）和输出端（axon）：输入端接收来自其他神经元或感官器官的信号；输出端向其他神經元或动作器官发送信号。当某个特定信号达到某个阈值时，该神經元会激活并发射电化学信号（action potentials）到下游连接上去；同时该激活也会被传递给与该激活相关联的记忆库存器结构进行持久化保存；当下游连接到达足够强度时也会被激活并发射电化学信号到下游继续传播；最终所有相关联激活都会被集合起来进行整合处理并产生最终决策结果返回给感知层进行执行动作或反馈给上层进行调整参数优化等等...
