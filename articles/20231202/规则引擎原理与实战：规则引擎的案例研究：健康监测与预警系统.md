```python
# 定义规则
rules = [
    {
        "condition": re.compile(r"^(20|[3-9]\d)\d{2}-(0[1-9]|1[0-2])-(0[1-9]|[12][0-9]|3[01])$"),
        "action": "发送预警通知"
    },
    {
        "condition": re.compile(r"^(20|[3-9]\d)\d{2}-(0[13-9]|1[0-2])-31$"),
        "action": "发送预警通知"
    }
]

# 定义数据
data = [
    "2022-01-01",
    "2022-02-29",
    "2022-03-01"
]

# 遍历数据
for d in data:
    # 匹配规则
    for rule in rules:
        if rule["condition"].match(d):
            # 执行规则
            print(rule["action"], d)
            break
```

在这个代码实例中，我们首先定义了一组规则，每个规则包括一个条件和一个动作。条件是一个正则表达式，用于匹配日期字符串。动作是一个字符串，表示执行的操作。

然后，我们定义了一组数据，每个数据都是一个日期字符串。我们遍历每个数据，并尝试匹配每个规则的条件。如果条件匹配成功，我们执行规则的动作。

在这个例子中，我们的规则是根据日期来发送预警通知的。如果日期符合规则的条件，我们将执行发送预警通知的操作。
