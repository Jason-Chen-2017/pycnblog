# 2.核心概念与联系
## 2.1 强化学习基础
强化学习是一种机器学习方法，它通过与环境进行交互来获取反馈并优化行为。在强化学习中，智能体与环境进行交互以完成任务。智能体可以从环境中接收奖励信号，并根据这些信号调整其行为策略。强化学习的目标是找到一个策略，使得智能体在执行任务时获得最大的累积奖励。

## 2.2 价值函数基础
价值函数是强化学习中一个重要概念，用于衡量某个状态或行为的优劣。价值函数表示从当前状态开始执行某个策略时，预期获得的累积奖励总和。具体来说，价值函数V(s)表示从初始状态s开始执行策略π时预期获得的累积奖励总和：
$$
V^{\pi}(s) = E_{\pi}[G_t|S_t=s] = E_{\pi}[\sum_{k=0}^{\infty}\gamma^k R_{t+k+1}|S_t=s]
$$
其中：
- Gt表示从当前时刻t开始计算到终止时刻T-1之间所有时刻的累积奖励总和：$G_t = \sum_{k=0}^{\infty}\gamma^k R_{t+k+1}$；
- Rtk表示在时刻tk后续动作所获得的奖励；
- $\gamma$（0 < $\gamma$ <= 1）是折扣因子，用于控制未来奖励对当前决策权重分配；
- $E_{\pi}[\cdot]$表示按照策略π采样期望值；
- $S_t$表示在时刻t所处的状态；
- $R_{t+k+1}$表示在时刻tk后续动作所获得的奖励；
- k代表迭代次数（从0开始计数）。
