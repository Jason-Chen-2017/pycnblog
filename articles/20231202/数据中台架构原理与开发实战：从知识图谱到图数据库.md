# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释知识图谱和图数据库的开发和实战应用。

假设我们要构建一个知识图谱，用于表示一个电影的信息，如电影名称、导演、演员、类型等。我们可以使用以下代码来实现这个知识图谱：

```python
from rdflib import Graph, Namespace, Literal

# 定义一个名称空间
ns = Namespace("http://example.com/movie#")

# 创建一个图数据库
g = Graph()

# 添加实体
g.add((ns.movie, ns.title, Literal("Titanic")))
g.add((ns.movie, ns.director, Literal("James Cameron")))
g.add((ns.movie, ns.actor, Literal("Leonardo DiCaprio")))
g.add((ns.movie, ns.type, Literal("Drama")))

# 保存图数据库
g.serialize(format="turtle", destination="movie.ttl")
```

在这个代码中，我们使用了`rdflib`库来构建一个知识图谱。我们首先定义了一个名称空间，然后创建了一个图数据库。接着，我们添加了实体、关系和属性信息到图数据库中，并将图数据库保存到文件中。

接下来，我们可以使用以下代码来查询这个知识图谱：

```python
from rdflib import Graph

# 加载图数据库
g = Graph().load("movie.ttl", format="turtle")

# 查询电影的信息
query = """
SELECT ?movie ?title ?director ?actor ?type
WHERE {
  ?movie ns:title ?title .
  ?movie ns:director ?director .
  ?movie ns:actor ?actor .
  ?movie ns:type ?type
}
"""

# 执行查询
results = g.query(query)

# 输出查询结果
for row in results:
  print(row)
```

在这个代码中，我们使用了`rdflib`库来查询知识图谱。我们首先加载了图数据库，然后使用`SELECT`语句来查询电影的信息。最后，我们输出查询结果。
