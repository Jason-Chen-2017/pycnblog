## 3.1 数据清洗与预处理
数据清洗是数据科学家的重要任务之一，它涉及到数据的缺失值处理、数据类型转换、数据归一化等操作。以下是一个简单的数据清洗示例：

```python
import pandas as pd
import numpy as np

# 读取数据
data = pd.read_csv('data.csv')

# 处理缺失值
data = data.fillna(data.mean())

# 类型转换
data['age'] = data['age'].astype('int')

# 归一化
data['age'] = (data['age'] - data['age'].mean()) / data['age'].std()
```
