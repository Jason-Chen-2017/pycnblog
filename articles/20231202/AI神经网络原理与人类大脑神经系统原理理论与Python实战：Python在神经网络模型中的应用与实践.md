# 2.核心概念与联系
## 2.1 AI神经网络原理与人类大脑神经系统原理理论的关系
AI神经网络原理与人类大脑神经系统原理理论之间存在密切联系。AI神经网络借鉴了人类大脑的结构和功能，以实现自主学习和决策。同时，研究人员也试图利用这些知识来改进AI技术，使其更加接近于或超越了人类智能水平。
## 2.2 AI神经网络基本组成部分：输入层、隐藏层和输出层
- **输入层**：负责接收数据并将其转换为适合处理的格式。
- **隐藏层**：包含多个节点（单元），每个节点都有一个权重向量和偏置值。这些参数决定了节点对输入数据的响应程度。通过多次迭代计算得到最终预测结果。
- **输出层**：负责产生预测结果并将其返回给用户或下游应用程序。
