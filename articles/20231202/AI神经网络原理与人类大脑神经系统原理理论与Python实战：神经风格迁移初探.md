# 2.核心概念与联系
## 2.1 AI神经网络原理与人类大脑神经系统原理理论的联系
### 2.1.1 人类大脑神经系统原理
人类大脑是一个复杂而高度组织化的生物电路系统，由数十亿个神经元组成。每个神经元都有输入和输出端，可以接收来自其他神经元的信号并进行处理。这些信号通过连接到其他神經元的纤维传递给其他部分大脑或身体部位。大脑通过这种方式进行思考、记忆、感知等功能。
### 2.1.2 AI神经网络原理
AI神经网络是一种模拟大脑工作方式的计算机程序结构。它由多层节点组成，每个节点都接收输入信号并对其进行处理后产生输出信号。这些节点之间通过连接传递信息。AI神經網絡可以用來學習從數據中抽取特徵並進行預測或決策作業。
