# 4.具体代码实例和详细解释说明

在开发软件工具时，你需要编写一些具体的代码实例。以下是一个简单的代码实例：

```python
def sort(arr):
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] > arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
print(sort(arr))
```

这个代码实例是一个简单的冒泡排序算法，用于对一个数组进行排序。在这个例子中，我们定义了一个名为`sort`的函数，它接受一个数组作为参数，并返回一个排序后的数组。我们创建了一个名为`arr`的数组，并将其传递给`sort`函数，然后打印出排序后的数组。
