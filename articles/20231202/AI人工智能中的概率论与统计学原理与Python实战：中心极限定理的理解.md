$$

```python
import numpy as np
from scipy.stats import norm

# 生成一个大样本
n = 1000

# 计算随机变量的期望值和标准差

sigma = np.std(X_n)

# 将随机变量标准化
Z = (X_n - mu) / (sigma * np.sqrt(n))

# 使用中心极限定理的数学公式计算概率
probability = norm.cdf(Z)

print("中心极限定理的概率:", probability)
```
