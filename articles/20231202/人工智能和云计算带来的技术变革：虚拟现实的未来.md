```python
# 设置屏幕尺寸

# 设置背景颜色
background_color = (255, 255, 255)
screen.fill(background_color)

# 绘制图像
pygame.draw.circle(screen, (0, 0, 255), (400, 300), 100)

# 处理用户输入
for event in pygame.event.get():
    if event.type == QUIT:
        pygame.quit()

# 更新屏幕
pygame.display.flip()

# 循环运行
while True:
    pass
```

代码还包括一个事件循环，用于处理用户输入。如果用户点击窗口关闭按钮，代码将结束程序。最后，代码使用`pygame.display.flip`函数更新屏幕，并进入一个无限循环，以便程序一直运行。
