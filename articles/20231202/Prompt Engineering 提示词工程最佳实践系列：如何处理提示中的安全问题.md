# 2.核心概念与联系
在开始探讨如何处理提示中的安全问题之前，我们需要先了解一下相关的核心概念：
- **自然语言处理（NLP）**：NLP是计算机科学领域中一个研究领域，旨在让计算机理解、生成和翻译人类语言的能力。通常情况下，NLP涉及到文本分析、信息抽取、情感分析等任务。
- **提示工程（Prompt Engineering）**：提示工程是一种设计方法，主要针对AI模型进行指导和优化。通过设计合适的输入提示，可以帮助模型更好地完成任务并达到预期效果。
- **安全性**：安全性是保护数据和隐私免受未经授权访问或损坏的能力。在使用AI技术时，确保数据和隐私得到充分保护是非常重要的。
