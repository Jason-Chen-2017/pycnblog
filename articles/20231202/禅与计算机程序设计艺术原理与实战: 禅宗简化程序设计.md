假设我们需要编写一个程序，计算一个数组中所有数字的和。我们可以按照以下步骤进行操作：

1. 首先，我们需要对问题进行分析，发现问题的内在特点是需要遍历数组中的每个数字，并将其加到一个总和中。
2. 然后，我们需要将问题分解为更简单的子问题，即遍历数组中的每个数字。
3. 接下来，我们需要根据禅宗思想，对问题的子问题进行解决。我们可以使用循环来遍历数组中的每个数字，并将其加到一个总和中。
4. 最后，我们需要将问题的子问题的解决方案组合成一个完整的解决方案，即计算数组中所有数字的和。

具体代码实例如下：

```python
def sum_array(arr):
    total = 0
    for num in arr:
        total += num
    return total

arr = [1, 2, 3, 4, 5]
result = sum_array(arr)
print(result)  # 输出: 15
```
