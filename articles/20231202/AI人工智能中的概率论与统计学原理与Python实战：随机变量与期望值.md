# 2.核心概念与联系
## 2.1 随机变量
随机变量是一个可以取多个不同值的变量，每个值都有相应的概率。在AI人工智能领域，我们经常需要处理大规模数据集或复杂模型中包含许多随机因素的问题。为了解决这些问题，我们需要了解如何描述和分析随机变量所代表的信息。

### 2.1.1 离散型随机变量
离散型随机变量只能取有限个不同的数值。例如：抛硬币得到正面还是反面；抽卡牌得到红色还是黑色等等。离散型随机变量通常使用泊松分布、二项分布等来描述其分布特征。

## 2.2 期望值（Expectation）
期望值是一个数字，表示某个事件发生时所产生的平均效果或影响力（即对某种结果进行预测）,也称为“期望”或“预期”（Expected value）,它可以看作是一种“预测”或者“估计”方法,主要应用于对未来事件进行预测
