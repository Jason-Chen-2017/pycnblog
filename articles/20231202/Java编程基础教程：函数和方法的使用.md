# 2.核心概念与联系
## 2.1 函数与方法的区别
在Java中，函数和方法是相互联系的两个概念。函数是一种计算机程序的组成部分，它接受输入参数（如变量或数据）并返回一个输出结果。而方法则是类中特定功能的实现细节，它由一组语句组成并且可以被调用。简单来说，函数是对外暴露给其他代码的接口，而方法则是对内部实现细节的封装。

## 2.2 函数与子程序之间的关系
子程序（Subroutine）是一种特殊类型的函数，它不需要返回值但可以接收参数。在Java中，子程序通常用于处理复杂任务或逻辑操作。例如：计算两个整数之和、比较两个字符串等等。子程序可以被调用多次，从而提高代码的重复利用性和可维护性。
