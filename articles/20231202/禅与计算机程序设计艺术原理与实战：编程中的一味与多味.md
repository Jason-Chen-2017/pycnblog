# 2.核心概念与联系

在《禅与计算机程序设计艺术原理与实战：编程中的一味与多味》一书中，作者提出了一种新的编程思维方式，即“一味与多味”。这种思维方式强调在编程过程中，程序员需要保持一种专注的心态，将自己的注意力集中在当前的任务上，而不是分散在多个任务之间。

这种思维方式与心理学中的“专注”和“分散”两种思维模式有密切的联系。当我们处于专注状态时，我们能够更好地集中注意力，更快地完成任务，而且更容易避免错误。而当我们处于分散状态时，我们的注意力会被多个任务所分散，这会导致我们的工作效率下降，错误率上升。
