# 2.核心概念与联系

数据湖是数据中台架构的核心组成部分，它是一种存储大量结构化和非结构化数据的仓库。数据湖可以存储来自各种数据源的数据，如关系数据库、非关系数据库、文件系统、Hadoop分布式文件系统（HDFS）等。数据湖的特点是灵活性、可扩展性和易用性，它可以帮助企业更好地管理和分析大量的数据资源。
