## 4.1 代码实例

```python
# 定义规则库
rules = {
    "rule1": "if $flow > 1000 then alert('网络流量过高')",
    "rule2": "if $security_level < 5 then alert('网络安全风险')",
    "rule3": "if $error_count > 3 then alert('网络故障')"
}

# 创建工作内存
work_memory = {
    "flow": 1200,
    "security_level": 4,
    "error_count": 4
}

# 实现规则引擎引擎
def rule_engine(rules, work_memory):
    for rule_name, rule in rules.items():
        # 匹配规则
        if eval(rule):
            # 执行规则
            print(rule_name + " triggered")

# 调用规则引擎
rule_engine(rules, work_memory)
```

然后，我们创建了一个工作内存，用于存储5G网络中的相关信息。这些信息包括网络流量、网络安全、网络故障等信息。
