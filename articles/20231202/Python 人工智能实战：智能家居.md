# 4.具体代码实例和详细解释说明
在这里，我们将通过一个简单的例子来说明如何使用 Python 实现智能家居的设备控制功能。

```python
import RPi.GPIO as GPIO
import time

# 设置 GPIO 引脚
GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)  # 设置 GPIO 引脚 17 为输出

# 控制灯的开关
def light_control(state):
    if state == 1:
        GPIO.output(17, GPIO.HIGH)  # 打开灯
    else:
        GPIO.output(17, GPIO.LOW)  # 关灯

# 主程序
while True:
    # 读取用户输入
    user_input = input("请输入灯的状态（1：开，0：关）：")

    # 控制灯的状态
    if user_input == "1":
        light_control(1)
    elif user_input == "0":
        light_control(0)
    else:
        print("输入错误，请重新输入")

    # 延迟一段时间
    time.sleep(1)
```

在这个例子中，我们使用了 Python 的 GPIO 库来控制 Raspberry Pi 上的一个 LED 灯。用户可以通过输入 1 或 0 来控制灯的开关。
