## 1.2 Python标准库核心概念与联系
Python标准库是指Python语言内置的一组模块和函数。这些模块和函数可以帮助开发者更快地完成各种任务，而无需自己编写底层代码。以下是一些核心概念：

- **模块**：一个模块是一个包含多个函数、类或变量的文件。在Python中，我们可以通过导入模块来使用其内容。例如：`import math` 可以导入`math`模块中定义的所有函数和常量。
- **函数**：函数是一个可执行代码片段，可以接受输入参数并返回结果。在Python中，我们可以通过定义函数来创建自己的功能逻辑。例如：`def add(x, y): return x + y` 定义了一个名为`add`的函数，该函数接受两个参数并返回它们之和。
- **类**：类是一个用于创建对象实例化对象（也称为“实例”）的蓝图或模板。在Python中，我们可以通过定义类来创建自己的数据结构和功能逻辑。例如：`class Person:` 定义了一个名为`Person`的类，该类表示人员信息及相关操作方法（如获取姓名、年龄等）。
- **变量**：变量是存储值（如整型、浮点型、字符串等）或引用（如列表、字典等）的容器。在Python中，我们可以通过赋值操作来创建变量并存储值或引用。例如：`x = 1000000000000.5678945321453245324532453245324532453245324532453245324 `定义了一个名为x的浮点型变量并赋予其值1e11（即十亿）加上小数部分（精度较高）；同样地我们也可以直接赋予其他数据类型值（如整型、字符串等）；还有就是引用数据类型（列表、字典等）也被视为变量；同时也有全局变量与局部变量之分；还有静态变量与动态变量之分；还有不同作用域范围内不同形式存在于不同环境下不同形式出现且具备不同特性属性上下文环境状态机制所带来复杂性问题等等复杂性问题需要深入理解才能真正掌握
