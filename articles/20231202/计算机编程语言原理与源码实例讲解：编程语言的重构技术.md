```python
def semantic_analysis(ast_tree):
    for node in ast.walk(ast_tree):
        if isinstance(node, ast.Name):
            if not node.parent.parent:
                print(f"Variable {node.id} is not defined")
        elif isinstance(node, ast.Assign):
            if not node.target:
                print(f"Assignment target is not defined")
```

在这个代码实例中，我们首先导入了ast模块，然后定义了一个semantic_analysis函数，该函数接受一个抽象语法树作为参数，并进行语义分析。我们通过遍历抽象语法树的每个节点，并根据节点的类型来检查变量是否被定义，以及赋值目标是否被定义。
