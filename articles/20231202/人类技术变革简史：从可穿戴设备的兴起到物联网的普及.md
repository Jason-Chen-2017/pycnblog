## 1.1 可穿戴设备的兴起

可穿戴设备的兴起是21世纪初的一个重要技术变革。这些设备可以直接戴在人体上，如手表、眼镜、耳机等，为用户提供实时的信息和服务。可穿戴设备的出现为人类的生活带来了更加方便、智能化的体验。

可穿戴设备的发展历程可以追溯到20世纪90年代，当时的个人数字助手（PDA）就是一种可穿戴设备。然而，直到2000年代末，可穿戴设备才开始迅速发展。

2009年，苹果公司推出了第一代iPhone，这是可穿戴设备的一个重要里程碑。iPhone的成功证明了人们对于手机的需求，同时也为可穿戴设备的发展提供了新的动力。

2013年，苹果公司推出了第一代Apple Watch，这是第一个真正意义上的智能手表。Apple Watch可以与用户的手机进行通信，同时还具有许多独立功能，如健康监测、通知推送等。

import CoreMotion

class ViewController: UIViewController {
    var motionManager: CMMotionManager?
    var heartRate: Double?

    override func viewDidLoad() {
        super.viewDidLoad()
        motionManager = CMMotionManager()
        motionManager?.startActivityUpdates(to: OperationQueue.main) { (activity) in
            if let activity = activity {
                self.heartRate = activity.heartRate
                self.updateUI()
            }
        }
    }

    func updateUI() {
        if let heartRate = heartRate {
            let label = UILabel(frame: CGRect(x: 0, y: 0, width: 100, height: 20))
            label.text = "Heart Rate: \(heartRate)"
            self.view.addSubview(label)
        }
    }
}
```

import CoreMotion
import Alamofire

class ViewController: UIViewController {
    var motionManager: CMMotionManager?
    var heartRate: Double?

    override func viewDidLoad() {
        super.viewDidLoad()
        motionManager = CMMotionManager()
        motionManager?.startActivityUpdates(to: OperationQueue.main) { (activity) in

                self.heartRate = activity.heartRate
                self.sendHeartRateToServer()
            }
        }
    }

            let parameters: [String: Any] = ["heartRate": heartRate]
            Alamofire.request("http://example.com/api/heartRate", method: .post, parameters: parameters, encoding: JSONEncoding.default).responseJSON { (response) in

                case .success:
                    print("Heart rate sent to server")
                case .failure(let error):
                    print("Error sending heart rate to server: \(error)")
                }
            }
        }
    }
}
```
