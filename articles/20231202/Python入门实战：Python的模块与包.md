## 1.1 Python模块与包的基本概念

### 1.1.1 模块（Module）

在Python中，一个模块就是一个`.py`文件。通过使用`import`关键字，我们可以将该文件中的函数、类或变量导入到当前的脚本或交互环境中使用。例如：
```python
import math
print(math.sqrt(4)) # 输出：2.0
```
在这个例子中，我们导入了内置的`math`模块并调用了其中的`sqrt()`函数。这样做有助于避免重复编写相同功能的代码，提高代码的可维护性和可读性。
