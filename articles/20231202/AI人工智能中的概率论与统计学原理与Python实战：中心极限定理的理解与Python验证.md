# 2.核心概念与联系
## 2.1概率论与统计学的基本概念
### 2.1.1概率
概率是衡量事件发生可能性的度量标准，通常表示为一个数值范围0到1之间的小数。例如，抛硬币的正面朝上的概率为0.5（或50%），而撞车死亡的概率则远高于这个值。

### 2.1.2随机变量
随机变量是一个可以取多种不同值得变量，每个值都有相应的概率。例如，掷骰子所得点数就是一个随机变量，它可以取值为1、2、3、4、5或6，每个值出现的概率相同（即1/6）。

### 2.1.3期望和方差
期望是随机变量取值平均值所对应于某一特定参考点（如均值）之间距离平均值所做出的总和除以该总和所作出来得到期望结果；方差则是期望偏差平方后求和再除以总数得到结果。例如：对于一个正态分布随机变量X,其期望E(X) = mu,方差Var(X) = sigma^2,标准差Std(X) = sqrt(sigma^2).
