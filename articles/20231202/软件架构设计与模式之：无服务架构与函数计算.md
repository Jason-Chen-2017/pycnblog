                 

# 1.背景介绍

无服务架构（Microservices Architecture）是一种新兴的软件架构设计模式，它将应用程序划分为多个小型服务，每个服务都是独立的、可扩展的、可独立部署和维护的。这种架构的出现主要是为了解决传统的单体应用程序在性能、可扩展性、可维护性等方面的局限性。

函数计算（Function Computing）是一种基于无服务架构的应用程序开发模式，它将应用程序拆分为多个小型函数，每个函数都是独立的、可扩展的、可独立部署和维护的。这种模式的出现主要是为了解决传统的单体应用程序在性能、可扩展性、可维护性等方面的局限性。
