                 

# 1.背景介绍

全球定位系统（Global Positioning System，GPS）是一种使用卫星导航技术的导航系统，由美国国防部开发并维护。它可以为地面接收器提供位置、速度和时间信息。

GPS 系统由24个卫星组成，每个卫星都在高空环绕地球。这些卫星携带着精确的时钟和定位传感器，可以向地球表面发射信号。当接收器接收到来自多个卫星的信号时，它可以计算出自身的位置、速度和时间。

GPS 技术的发展历程可以分为几个阶段：
1. 1973年，美国国防部开始研究全球定位系统的概念和设计。
2. 1978年，第一个测试卫星上线，初步验证了 GPS 技术的可行性。
3. 1983年，GPS 正式上线，但仅限于美国军方使用。
4. 1995年，美国政府决定将 GPS 技术公开给民用市场。
5. 2000年代后期至今，随着 GPS 技术的不断发展和改进，其应用范围逐渐扩大到各个领域。
