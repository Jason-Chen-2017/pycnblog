# 2.核心概念与联系
## 2.1正态分布
正态分布是一种连续的概率分布，其概率密度函数为：
$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$
其中，$\mu$ 是均值，$\sigma$ 是标准差。正态分布的特点是：
1. 数据分布是对称的，左右两侧相对于均值对称。
2. 数据分布是连续的，没有跳跃性。
3. 数据分布是单峰的，峰值为均值。
正态分布在实际应用中非常广泛，特别是在处理连续型数据时，如高温、低温、体重等。正态分布的出现可以归因于多种因素的相互作用，使得数据分布达到平衡状态。

## 2.2中心极限定理
中心极限定理是概率论与统计学中的一个基本定理，它描述了样本均值在大样本数量下的分布特征。中心极限定理的主要内容是：
对于任意随机变量$X$，当样本数$n$足够大时，样本均值$\bar{X}$的分布接近正态分布。
中心极限定理的证明需要使用欧拉积分公式、欧拉方程等高级数学知识，但在实际应用中，我们可以直接利用中心极限定理来进行数据分析和预测。
