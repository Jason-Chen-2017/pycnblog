Redis是一个开源的高性能的key-value存储系统，它支持数据的持久化，可基于内存（Redis）或磁盘（Redis-Persistent）。Redis 是一个开源的使用 ANSI C 语言编写、遵循 BSD 协议的高性能的 key-value 存储系统。Redis 可以在内存中存储数据，通过 Redis Cluster 提供了数据的高可用性，也可以将数据保存在磁盘中，实现数据的持久化。
