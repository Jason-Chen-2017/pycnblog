Python 是一种流行的编程语言，它具有简洁的语法和强大的功能。在实际应用中，我们经常需要处理文件操作和异常处理。本文将详细介绍 Python 文件操作与异常处理的核心概念、算法原理、具体操作步骤以及数学模型公式。同时，我们还将通过代码实例来详细解释这些概念和操作。

## 1.1 Python 文件操作基础
Python 提供了丰富的文件操作功能，可以方便地读取和写入文件。在进行文件操作之前，我们需要导入 os 模块，该模块提供了与操作系统互动的各种方法。

```python
import os
```

### 1.1.1 打开文件
要打开一个文件，我们可以使用 `open()` 函数。该函数接受两个参数：第一个参数是文件名，第二个参数是打开模式（'r'表示只读模式，'w'表示写入模式等）。成功打开文件后，我们可以通过返回值（一个 file 对象）来访问该文件的内容。

```python
file = open('example.txt', 'r') # 打开 example.txt 文件并设置为只读模式
```
### 1.1.2 读取文件内容
要读取一个已经打开的文件的内容，我们可以使用 `read()` 方法。该方法不需要传递任何参数，会返回整个文件的内容。如果只想读取部分内容，可以传递一个整数参数指定偏移量（从0开始计算）和长度（字节）。如果指定了偏移量和长度，则会从指定偏移量处读取指定长度的内容；否则会从当前位置处读取剩余部分内容。如果已经到达末尾或者没有剩余部分内容了，则会返回空字符串 ''。如果没有传递任何参数或者传递了负值或者超出范围的值，则会抛出 ValueError 异常。如果已经关闭了该 file 对象或者其他错误发生了（如磁盘满等），也会抛出 IOError 异常。最后记得关闭已经完成阅览或写入工作后的 file object,否则资源不释放,造成资源浪费,甚至影响系统性能稳定性甚至安全性等多种问题,所以最好养成习惯关闭所有file object
