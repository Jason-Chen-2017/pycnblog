### 4.1 代码实例

```python
class Node:
    def __init__(self, name):
        self.name = name
        self.children = []

    def add_child(self, child):
        self.children.append(child)

    def __str__(self):
        return self.name

def build_mind_map(root, nodes):
    for node in nodes:
        root.add_child(node)
        build_mind_map(node, nodes)

def main():
    root = Node("主题")
    nodes = [
        Node("子节点1"),
        Node("子节点2"),
        Node("子节点3"),
    ]
    build_mind_map(root, nodes)
    print(root)

if __name__ == "__main__":
    main()
```

在这个代码实例中，我们定义了一个`Node`类，用于表示思维导图中的节点。每个节点都有一个名称和一个子节点列表。我们还定义了一个`build_mind_map`函数，用于构建思维导图。

在`main`函数中，我们创建了一个根节点，并创建了三个子节点。然后，我们调用`build_mind_map`函数来构建思维导图。最后，我们打印出根节点，以便我们可以看到思维导图的结构。
