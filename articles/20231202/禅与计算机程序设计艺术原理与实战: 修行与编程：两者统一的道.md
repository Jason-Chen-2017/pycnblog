## 3.1 选择排序算法

选择排序算法是一种简单的排序算法，它的核心思想是在每次迭代中选择最小的元素，并将其放在当前位置。选择排序算法的时间复杂度为O(n^2)，其中n是数组的长度。

### 3.1.1 算法原理

选择排序算法的核心思想是在每次迭代中选择最小的元素，并将其放在当前位置。具体操作步骤如下：

1. 从数组中选择第一个元素，记录其下标。
2. 从第一个元素后面的元素中选择最小的元素，并将其与第一个元素交换位置。
3. 重复第二步，直到数组中所有元素都被排序。

## 4.1 代码实例

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```
