# 2.核心概念与联系
## 2.1概率论
概率论是数学的一个分支，主要研究事件发生的可能性和相关概念。在AI人工智能中，我们需要对大量数据进行分析和预测，因此需要掌握概率论的基本知识。

### 2.1.1事件、样本空间、事件空间
- 事件：具体发生或不发生的某种情况。例如：抛硬币得到正面（H）或反面（T）。
- 样本空间：所有可能结果构成的集合。例如：S = {H, T}。
- 事件空间：所有可能发生的事件组成的集合。例如：E = {HH, HT, TH, TT}。
### 2.1.2条件概率、独立性、贝叶斯定理
- 条件概率：给定某个事件B发生，另一个事件A发生的概率。表示为P(A|B)。
- 独立性：两个事件A和B之间没有任何关联，即A发生不会影响B发生或不发生。表示为P(A∩B) = P(A) * P(B)。
- 贝叶斯定理：给定已知某个事件B发生，求出另一个事件A发生的概率P(A|B) = P(A∩B) / P(B)。通常用于解决条件依赖问题。
## 2.2统计学
统计学是一门应用数学科学，主要研究从观察数据中抽取信息并进行推断和预测的方法和理论基础。在AI人工智能中，我们需要对大量数据进行分析和预测，因此需要掌握统计学的基本知识。
