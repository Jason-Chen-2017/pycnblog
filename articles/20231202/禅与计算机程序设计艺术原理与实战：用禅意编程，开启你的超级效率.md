## 4.具体代码实例和详细解释说明

以下是一个计算两个数之和的程序的具体代码实例：

```python
def add(A, B):
    return A + B

A = 1
B = 2
S = add(A, B)
print(S)
```

这个程序的解释说明如下：

- 首先，我们定义了一个名为`add`的函数，它接受两个参数A和B，并返回它们之和。
- 然后，我们将变量A的值设置为1，变量B的值设置为2。
- 接下来，我们调用`add`函数，将变量A和变量B作为参数传递给它，并将返回值赋给变量S。
- 最后，我们使用`print`函数输出变量S的值。
