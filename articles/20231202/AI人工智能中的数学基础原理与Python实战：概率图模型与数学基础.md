# 2.核心概念与联系

- 随机变量：随机变量是一个可能取多个值的变量，每个值都有一个概率。
- 条件概率：给定某些事件已经发生，另一个事件发生的概率。
- 概率分布：概率分布是一个随机变量的概率值的函数。
- 独立性：两个事件之间没有任何关系，发生的概率与另一个事件发生的概率之间没有关联。
- 条件独立性：给定某些条件，两个事件之间没有关系。

概率图模型可以用来表示随机变量之间的关系，这些关系可以是线性或非线性的。概率图模型可以用来表示随机变量之间的条件独立性，这有助于我们更好地理解数据之间的关系。
