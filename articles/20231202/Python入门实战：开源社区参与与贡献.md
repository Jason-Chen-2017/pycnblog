## 2.核心概念与联系
### 2.1 Python基础知识
- **变量**：变量是存储数据值的内存空间名称。在Python中，变量没有声明类型，可以动态更改类型。例如：`x = 10` 表示整数类型；`x = "Hello, World!"` 表示字符串类型；`x = [1, 2, 3]` 表示列表类型等。
- **数据结构**：数据结构是组织数据及相关操作方法的抽象框架。在Python中主要包括列表（List）、元组（Tuple）、字典（Dictionary）和集合（Set）等数据结构。例如：列表可以通过索引访问元素；元组则是不可变的列表；字典是键值对映射关系；集合则是无序且唯一元素集合等。
- **函数**：函数是一个代码块，可以接受输入参数并返回输出结果。在Python中定义函数时需要使用关键字def进行定义并指定参数及返回值类型等信息。例如：`def add(a, b): return a + b` 定义了一个加法函数；`def greet(name): print("Hello,", name)` 定义了一个打印问候语函数等。
- **面向对象编程**：面向对象编程（Object-Oriented Programming, OOP）是一种编程范式，将软件系统划分为多个对象组成并规定它们之间相互作用的方式来实现软件功能模块化设计和复杂度控制等目标。在Python中实现面向对象编程需要创建类及其实例化对象并调用其方法等操作步骤。例如：创建一个Person类并实现getName()方法获取姓名信息；创建一个Car类并实现drive()方法启动车辆运行等操作步骤都属于面向对象编程范式下完成工作内容所需要执行的操作步骤之一或多个之一所需执行的操作步骤所需执行的操作步骤所需执行
