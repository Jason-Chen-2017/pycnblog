## 4. 具体代码实例和详细解释说明

以下是一个Go语言测试示例：

```go
package main

import (
    "testing"
)

func TestAdd(t *testing.T) {
    result := Add(1, 2)
    if result != 3 {
        t.Errorf("Expected 3, got %d", result)
    }
}

func Add(a, b int) int {
    return a + b
}
```

在这个示例中，我们创建了一个名为`main_test.go`的测试文件，定义了一个名为`TestAdd`的测试函数。这个测试函数调用了`Add`函数，并使用`t.Errorf()`方法来验证结果是否为3。

要运行这个测试，可以使用`go test`命令：

```
$ go test
ok      main    0.001s
```

如果测试失败，将输出错误信息：

```
$ go test
--- FAIL: TestAdd (0.001s)
    main_test.go:7: Expected 3, got 4
FAIL
```

这个示例展示了Go语言测试框架的基本用法。通过编写和运行测试用例，开发者可以发现并修复代码中的错误，从而提高软件的质量。
