## 1.1 Python字符串基础知识

在Python中，字符串是由一系列字符组成的序列。字符串可以包含文本、数字、特殊字符等各种元素。Python使用单引号（'）或双引号（"）来定义字符串。例如：
```python
str1 = 'Hello, World!'
str2 = "This is a string."
```
在Python中，还可以使用三重引号（''' ''''或""""）来定义多行字符串。例如：
```python
multiline_str = '''This is a multiline string.
It can span multiple lines.'''
```
