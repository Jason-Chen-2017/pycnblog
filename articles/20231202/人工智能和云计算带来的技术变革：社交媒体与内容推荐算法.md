## 4.1 基于内容的推荐

以下是一个基于内容的推荐算法的Python代码实例：

```python
import numpy as np
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 文本数据
texts = [
    "这是一个关于人工智能的文章",
    "这是一个关于云计算的文章",
    "这是一个关于社交媒体的文章"
]

# 文本特征提取
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(texts)

# 文本相似度计算
similarity = cosine_similarity(X)

# 推荐相关文章
recommended_article = np.argmax(similarity[0])
print("推荐的文章是：", texts[recommended_article])
```

from scipy.sparse import csr_matrix
from scipy.sparse.linalg import svds

# 用户-物品矩阵

    [1, 0, 0],
    [0, 1, 0],
    [0, 0, 1]
])

# 用户-用户相似度矩阵
S = csr_matrix([
    [1, 0.5, 0.5],
    [0.5, 1, 0.5],
    [0.5, 0.5, 1]
])

# 协同过滤推荐
U, Sigma, Vt = svds(A.dot(S), k=2)
recommended_item = np.dot(U, Vt)
print("推荐的物品是：", np.argmax(recommended_item))
