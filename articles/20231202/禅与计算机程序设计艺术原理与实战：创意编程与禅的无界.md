## 3.1递归
递归是一种非常重要的算法思想，它允许函数在其自身定义中被调用多次以求解问题。递归可以帮助我们简化问题并提供更清晰易懂的代码结构。例如：求Fibonacci数列第n项值：f(n) = f(n-1) + f(n-2)；f(0) = 0, f(1) = 1, n >= 0.递归公式为f(n) = baseCase + f(n-1),其中baseCase表示基础情况下返回值（即当n=0或n=1时）,f(n-1)表示递归调用函数时返回值（即当前函数对应参数减少一个时）,其中baseCase表示基础情况下返回值（即当n=0或n=1时）,f(n-1)表示递归调用函数时返回值（即当前函数对应参数减少一个时）.使用Python语言实现如下:```python def fibonacci(num): if num <= 0: return None elif num == 1: return [0] elif num == 2: return [0, 1] else: result = [None] * num result[0], result[1] = (0, 1) for i in range(2, num): result[i] = result[i - 2] + result[i - 1] return result ```
