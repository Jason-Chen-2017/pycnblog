                 

# 1.背景介绍

软件架构（Software Architecture）是构建软件系统时的一个核心过程，它影响各种可观察到和不可观察到的方面，如广泛的行为、应用方法上的约束以及对特定的能力支持需求。

软件架构原理涉及到软件系统中各种系统结构和结构来源、子系统、组件、模块、对象、数据、信息和控制的组织。这一系统架构的 sophistication 很大，较为复杂，但同样不紧密的与其具体的软件实现是相关的。
