## 4.1 比特币挖矿的代码实例

比特币挖矿的代码实例可以使用 Python 编程语言来编写。以下是一个简单的比特币挖矿代码实例：

```python
import hashlib
import time

def mine_block(previous_block):
    nonce = 0
    while True:
        hash = hashlib.sha256(previous_block.encode('utf-8') + str(nonce).encode('utf-8')).hexdigest()
        if hash[:4] == '0000':
            break
        nonce += 1
    return hash, nonce

previous_block = '0' * 64  # 上一个区块的哈希值
hash, nonce = mine_block(previous_block)
print('Mined block with hash:', hash)
print('Nonce:', nonce)
```

在这个代码实例中，我们首先导入了 hashlib 和 time 模块。然后，我们定义了一个 `mine_block` 函数，它接受一个参数 `previous_block`，即上一个区块的哈希值。在函数内部，我们使用一个 `while` 循环来计算哈希值，直到哈希值的前四位为 '0000'。当满足这个条件时，我们将哈希值和非零值返回。

最后，我们使用一个示例的上一个区块的哈希值来调用 `mine_block` 函数，并打印出挖矿的哈希值和非零值。
