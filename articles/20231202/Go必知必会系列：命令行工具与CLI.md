# 4.具体代码实例和详细解释说明

## 4.1代码实例
以下是一个简单的Go命令行工具示例：

```go
package main

import (
	"fmt"
	"os"
	"bufio"
	"strings"
)

func main() {
	reader := bufio.NewReader(os.Stdin)
	fmt.Print("请输入要查找的字符串：")
	input, _ := reader.ReadString('\n')
	input = strings.TrimSpace(input)

	count := 0
	for _, word := range strings.Fields(input) {
		if strings.Contains(word, "Go") {
			count++
		}
	}

	fmt.Printf("包含'Go'的单词数量：%d\n", count)
}
```

## 4.2详细解释说明
上述代码实例是一个简单的Go命令行工具，用于统计输入文本中包含“Go”的单词数量。具体解释如下：

1. 导入必要的包，包括fmt、os、bufio和strings。
2. 创建一个bufio.Reader对象，用于读取标准输入（os.Stdin）。
3. 提示用户输入要查找的字符串，并使用bufio.Reader的ReadString()方法读取用户输入的文本。
4. 使用strings.TrimSpace()函数去除输入文本中的空格。
5. 使用strings.Fields()函数将输入文本分割为单词列表。
6. 遍历单词列表，使用strings.Contains()函数检查每个单词是否包含“Go”，并统计结果。
7. 使用fmt.Printf()函数输出结果。
