# 2.核心概念与联系

在讨论贝叶斯网络之前，我们需要了解一些基本概念：

1. **随机变量**：随机变量是一个可能取多个值的变量，每个值都有一个概率。
2. **条件概率**：给定某些事件发生的条件下，另一个事件的概率。
3. **贝叶斯定理**：给定已知某个事件发生的条件下，另一个事件的概率。
4. **概率图模型**：一种用于表示随机变量之间关系的图，每个节点表示一个随机变量，每条边表示一个条件依赖关系。

贝叶斯网络是一种概率图模型，它使用贝叶斯定理来描述随机变量之间的条件依赖关系。在贝叶斯网络中，每个节点表示一个随机变量，每条边表示一个条件依赖关系。通过计算条件概率，我们可以预测未知变量的值。
