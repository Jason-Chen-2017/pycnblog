## 4. 具体代码实例和详细解释说明

在这个部分，我们将通过一个具体的代码实例来说明提示工程的过程。

假设我们需要设计一个文本摘要生成的任务，我们可以使用以下代码实现：

```python
import torch
from transformers import BertTokenizer, BertModel

# 设置输入提示
input_prompt = "请将以下文本内容简要概括为一句话："

# 加载模型和标记器
tokenizer = BertTokenizer.from_pretrained('bert-base-chinese')
model = BertModel.from_pretrained('bert-base-chinese')

# 将输入提示转换为输入序列
input_ids = torch.tensor(tokenizer.encode(input_prompt, add_special_tokens=True)).unsqueeze(0)

# 通过模型进行前向传播
outputs = model(input_ids)

# 提取输出序列
output_ids = outputs[0][0]

# 将输出序列转换为文本
output_text = tokenizer.decode(output_ids.tolist()[0])

print(output_text)
```

在这个代码中，我们首先设置了输入提示，然后加载了模型和标记器。接着，我们将输入提示转换为输入序列，并通过模型进行前向传播。最后，我们提取输出序列，将输出序列转换为文本，并打印输出结果。
