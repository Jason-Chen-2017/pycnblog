## 4.1 最大似然估计
```python
import numpy as np

# 定义随机变量的概率密度函数
def pdf(x, mu, sigma):
    return 1 / (np.sqrt(2 * np.pi * sigma**2)) * np.exp(-(x - mu)**2 / (2 * sigma**2))

# 定义随机变量的似然函数
def likelihood(x, mu, sigma):
    return pdf(x, mu, sigma)

# 定义随机变量的最大似然估计函数
def mle(x_list):
    n = len(x_list)
    x_mean = np.mean(x_list)
    x_var = np.var(x_list)
    return x_mean, np.sqrt(x_var)

# 测试最大似然估计
x_list = [1, 2, 3, 4, 5]
x_mean, x_var = mle(x_list)
print("x_mean =", x_mean)
print("x_var =", x_var)
```
在这个代码示例中，我们定义了一个随机变量的概率密度函数、似然函数和最大似然估计函数。然后我们测试了这些函数，并得到了x_mean和x_var的估计值。

## 4.3 卡方检验
```python
import numpy as np
from scipy.stats import chi2_contingency

# 定义卡方检验数据
def chi2_data(x, y):
    n = len(x)
    chi2 = np.sum((x - y)**2 / y)
    return chi2

# 定义卡方检验函数
def chi2_test(x, y):
    chi2 = chi2_data(x, y)
    df = len(x) - 1
    p = chi2_contingency(x, y)[1]
    return chi2, df, p
```
在这个代码示例中，我们定义了一个卡方检验数据和卡方检验函数。然后我们测试了这些函数，并得到了chi2、df和p的估计值。

## 4.4 朴素贝叶斯分类器
```python
import numpy as np
from sklearn.naive_bayes import GaussianNB

# 定义训练数据
```
