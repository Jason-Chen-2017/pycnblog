意外地简单的优先级 `PriorityQueue` 可扩展用于 JavaScript，这可能会对 JavaScript 发动一个破坏。这里有一个简单的教程，使您能够设置和接近优先级通常，有排序的需求，如调度、优先级排序，为何不使用 HA 引入。这个教程旨在阐述`PriorityQueue`在实际环境中可以是用来运行代码，`PriorityQueue`园术:

```javascript
// 一个常规的JavaScript数组可以作为一个优先级队列
var queue = [];

// Push a new item with a priority to the queue
queue.push({Val: 1, Priority: 3});

// An object that can be used to compare two items with each other
function comparePriorities(a, b) {
    return a.Priority - b.Priority;
}

// Sort the queue with a custom comparison
queue.sort(comparePriorities);

// Pull an item off of the queue(least to greatest)
queue.shift();
```
