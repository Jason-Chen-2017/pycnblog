# 4.具体代码实例和详细解释说明

在本节中，我们将提供一个具体的代码实例，并详细解释其中的步骤。

```python
import torch
from transformers import AutoTokenizer, AutoModelForSeq2SeqLM

# 加载预训练模型和标记器
tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")
model = AutoModelForSeq2SeqLM.from_pretrained("bert-base-uncased")

# 定义输入文本
input_text = "自动摘要技术在各个领域的应用广泛，但其准确性和效率是否能够满足实际需求，这是一个值得深入探讨的问题。"

# 将输入文本转换为输入序列
input_ids = tokenizer.encode(input_text, return_tensors="pt")

# 将输入序列输入到模型中
outputs = model(input_ids)

# 提取生成的摘要
summary_ids = outputs.decoder_contributions[0, -1, :]

# 将生成的摘要解码为文本
summary_text = tokenizer.decode(summary_ids.tolist()[0])

# 输出生成的摘要
print(summary_text)
```

在上述代码中，我们首先加载了预训练的模型和标记器。然后，我们定义了一个输入文本，并将其转换为输入序列。接着，我们将输入序列输入到模型中，并提取生成的摘要。最后，我们将生成的摘要解码为文本，并输出结果。
