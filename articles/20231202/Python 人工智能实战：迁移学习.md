# 2.核心概念与联系
迁移学习的核心概念包括源域、目标域、特征表示、模型迁移和学习策略等。源域和目标域是迁移学习中最基本的概念。源域是已经有标签的数据集，而目标域是需要预测的数据集。特征表示是将输入数据映射到一个高维空间的过程，模型迁移是将源域模型应用于目标域数据的过程，学习策略是迁移学习中的核心组成部分，它决定了如何将源域模型与目标域数据结合使用。

$$
