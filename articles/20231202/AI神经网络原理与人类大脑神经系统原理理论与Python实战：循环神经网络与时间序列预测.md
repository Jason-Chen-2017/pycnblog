                 

# 1.背景介绍

**人工 neural networks （ANN）** 是人工智能（AI）领域的研究之一，旨在学习模式并去做决策。这一方法基于计算机的人工神经元 （ReLU），这些神经元与人脑中的神经元有所类似，这就是为什么我们将其称为“人工神经网络”的原因。我们将通过本文为人工神经网络引入基本考虑限制，并提出一种 AI 神性胞本的计算模式。
