静态代码分析是一种不需要运行代码的代码审计方法。通过对代码进行自动化检查，静态代码分析可以发现潜在的安全漏洞、性能瓶颈和逻辑错误。静态代码分析可以通过检查代码的结构、语法和逻辑来发现问题。

混合代码分析是一种将静态代码分析和动态代码分析相结合的代码审计方法。通过将静态代码分析和动态代码分析相结合，混合代码分析可以发现潜在的安全漏洞、性能瓶颈和逻辑错误。混合代码分析可以通过检查代码的结构、语法、逻辑和行为来发现问题。
