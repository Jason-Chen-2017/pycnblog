```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
result = selection_sort(arr)
print(result)
```

在函数内部，我们首先获取数组的长度，并使用两个循环来实现选择排序算法。外部循环用于遍历数组的每个元素，内部循环用于找到最小元素的索引。如果找到了更小的元素，我们将其索引与当前元素进行交换。
