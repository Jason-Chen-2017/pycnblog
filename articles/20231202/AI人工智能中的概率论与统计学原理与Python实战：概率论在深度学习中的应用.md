# 3.1概率论基本概念的数学模型公式
事件：一个可能发生或不发生的结果。

样本空间：所有可能发生的事件的集合。

概率：事件发生的可能性，通常用P表示。

条件概率：给定某个事件已经发生，另一个事件发生的概率。

独立事件：两个事件发生的概率与它们发生的顺序无关。
