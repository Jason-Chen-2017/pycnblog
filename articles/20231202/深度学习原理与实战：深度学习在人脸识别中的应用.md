                 

# 1.背景介绍

人脸识别技术（Face Recognition）是近年来异生神州的卷积神经网络（Convolutional Neural Networks，CNN）给人类社会带来的一种先进技术，使得计算机可以通过对图像进行分析来识别人脸。人脸识别技术应用广泛，可以用于身份认证、人脸 geometry 数据库、检测人脸在图片中的位置、四面八方对比以确定两幅人脸是否相同等等。人脸识别技术应该在高安全性的应用场合使用；智能手机上的人脸识别应该加入密码的鉴权；人脸 geometry 是要实名登记的，要求全无私。

所谓“深度学习”的原理，来自于人工智能与计算机科学两个学科领域 的研发实践。人工智能是计算机程序模拟和扩展人类智能进行创新和探索的学科。深度学习是机器学习的计算量巨大，被称为”深度”来统称各种类的神经网络的体系结构，要求计算机、机器ví视器看到的世界像我们人类一样。深度学习就是通过一系列多层次图像的过渡，来分析人脸部分特征的图像并达到正确简化结果。

深度学习引用的学术著作《深度学习原理与实战》，全书长度为930个页面，成书日期：2018年9月15日。这本书所讲述的深度学习在人脸识别的应用，背后的背景和技术的思想，我们将声明一些原则、梳理一些核心的概念，从而让人们更好地掌握人脸识别技术的背景和核心的概念。在下文中，我写了一篇有深度、віène、有见度的专业的技术博客文章的批判性思考。
