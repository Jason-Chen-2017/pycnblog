### 4.1 项目管理示例

```python
import heapq

class Task:
    def __init__(self, name, priority):
        self.name = name
        self.priority = priority

    def __lt__(self, other):
        return self.priority < other.priority

tasks = [
    Task("任务1", 3),
    Task("任务2", 1),
    Task("任务3", 2),
]

heapq.heapify(tasks)

while tasks:
    task = heapq.heappop(tasks)
    print(f"执行任务：{task.name}")
```

在上述代码中，我们使用优先级队列（heapq）来实现任务的排序和调度。我们定义了一个`Task`类，用于表示任务的名称和优先级。然后，我们将任务添加到优先级队列中，并使用`heappop`方法从队列中取出最高优先级的任务，并执行任务。

### 4.2 团队协作示例

```python

# 克隆仓库

# 提交修改

repo.index.commit("add README and LICENSE")

repo.merge("master")
```

在上述代码中，我们使用Git库来实现团队成员之间的代码协作。我们首先获取当前工作目录的Git仓库，然后使用`clone`方法克隆远程仓库。接着，我们使用`add`方法添加文件到暂存区，并使用`commit`方法提交修改。然后，我们使用`create_head`方法创建新分支，并使用`checkout`方法切换到新分支。最后，我们使用`merge`方法合并分支。
