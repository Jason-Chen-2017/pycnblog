# 4.具体代码实例和详细解释说明
在这一部分，我们将通过一个具体的代码实例来说明Python音频处理库的使用方法。

```python
import numpy as np
import scipy.signal as signal

# 读取音频文件
audio_file = 'audio.wav'
audio_data = np.fromfile(audio_file, dtype=np.int16)

# 采样率
sample_rate = 44100

# 滤波
filtered_data = signal.lfilter(b=[1], a=[1], x=audio_data, fs=sample_rate)

# 保存滤波后的音频文件
filtered_audio_file = 'filtered_audio.wav'
np.save(filtered_audio_file, filtered_data)
```

在这个代码实例中，我们首先导入了numpy和scipy.signal库。然后，我们读取一个音频文件，并将其转换为numpy数组。接着，我们设置了采样率，并使用scipy.signal库中的lfilter函数进行滤波。最后，我们将滤波后的音频数据保存到一个新的音频文件中。
