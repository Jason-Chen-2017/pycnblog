### 4.1 数据清洗

```python
import pandas as pd
import numpy as np

# 读取数据
data = pd.read_csv('data.csv')

# 去除噪声
data = data.dropna()

# 填充缺失值
data['age'] = data['age'].fillna(data['age'].mean())

# 去除重复数据
data = data.drop_duplicates()

# 转换数据类型
data['gender'] = data['gender'].astype('category')

# 保存数据
data.to_csv('data_cleaned.csv', index=False)
```
