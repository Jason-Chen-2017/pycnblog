```python
# 规则引擎的代码实例
from rule_engine import RuleEngine

# 创建规则引擎实例
rule_engine = RuleEngine()

# 创建规则
rule1 = {
    "condition": "age >= 18",
    "action": "grant_loan"
}
rule2 = {
    "condition": "age < 18",
    "action": "deny_loan"
}

# 添加规则到规则引擎
rule_engine.add_rule(rule1)
rule_engine.add_rule(rule2)

# 设置数据
data = {
    "age": 20
}

# 执行规则引擎
rule_engine.execute(data)
```

在上面的代码实例中，我们创建了一个简单的规则引擎，并添加了两个规则。我们设置了一个数据字典，并执行了规则引擎。规则引擎会根据数据和规则的条件来匹配和执行规则。
