# 2.核心概念与联系
## 2.1正态分布的定义与特点
正态分布是一种连续性概率分布，其出现频率最高于均值附近，逐渐减少到两个尾部。正态分布有以下几个主要特点：
1. 对称性：正态分布在均值附近具有对称性，左右两侧都有相同数量的数据点。
2. 完全确定性：正态分布只需要知道均值和标准差即可完全确定其形状和参数。
3. 无穷延伸：正态分布是一个无穷小区间内的连续函数，所以它没有边界。
4. 密度函数：正态分布具有一个唯一且单调递增的密度函数f(x)。
5. 累积函数：正态分布具有一个唯一且单调递增的累积函数F(x)。
6. 标准化变换：通过标准化变换（z-score），可以将任何正态随机变量转换为标准正太随机变量N(0,1)。这使得我们可以利用标准正太随机变量的知识来处理各种不同类型的问题。
7. 高斯定理：如果n个独立同质随机变量X1, X2, ..., Xn从N(μ,σ^2)发生，那么它们之和S=X1+X2+...+Xn也会遵循N(nμ, nσ^2)规律。这意味着当样本大小足够时，我们可以使用平均值来估计总体参数。
8. 最大似然估计（MLE）：对于给定样本数据集D={x_i} (i=1,...,n) , MLE估计器θ_MLE = argmax p(θ|D) = argmax p(D|θ)p(θ)/p(D),其中p(θ|D), p(D|θ), p(θ), p(D)是条件概率、后验概率和先验概率等概率密度函数表达式；MLE方法在大样本情况下具有较好效果；但在小样本情况下可能存在偏差问题（如贝叶斯方法）；另外MLE方法需要对参数进行初始化（如牛顿法）或迭代更新（如EM算法）等优化策略；最后需要检验MLE估计器是否收敛或者是否存在局部极大值问题等问题；总之MLE方法需要结合实际情况进行选择和优化。
