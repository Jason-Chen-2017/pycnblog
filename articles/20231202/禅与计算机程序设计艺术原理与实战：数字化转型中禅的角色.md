# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个简单的例子来帮助读者更好地理解禅与计算机程序设计的联系。

例子：编写一个程序，输出1到100之间的所有奇数。

```python
# 定义一个函数，用于输出1到100之间的所有奇数
def print_odd_numbers(start, end):
    for i in range(start, end + 1):
        if i % 2 != 0:
            print(i)

# 调用函数，输出1到100之间的所有奇数
print_odd_numbers(1, 100)
```

解释：

- 首先，我们定义了一个函数`print_odd_numbers`，该函数接受两个参数：`start`和`end`，用于表示输出范围。
- 然后，我们使用`for`循环遍历从`start`到`end`的所有数字。
- 在循环中，我们使用`if`语句判断当前数字是否为奇数（即当前数字除以2的余数不等于0），如果是，则输出当前数字。
- 最后，我们调用`print_odd_numbers`函数，并传入参数1和100，以便输出1到100之间的所有奇数。
