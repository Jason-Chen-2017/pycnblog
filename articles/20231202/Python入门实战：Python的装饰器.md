                 

# 1.背景介绍

Python装饰器是一种用于增强函数功能的编程技巧。它可以让我们在不修改函数源代码的情况下，为函数添加新的功能。这种技术非常有用，因为它可以让我们更容易地重用和扩展现有的代码。

Python装饰器是一种高级编程技术，它允许我们在不修改函数源代码的情况下为其添加新功能。这使得我们可以更轻松地重用和扩展现有的代码。

## 2.核心概念与联系
装饰器是Python中一个高级编程技术，它允许我们在不修改函数源代码的情况下为其添加新功能。这使得我们可以更轻松地重用和扩展现有的代码。装饰器通过将一个函数作为参数传递给另一个函数来创建新的函数对象，该对象包含了原始函数及其附加行为。

### 2.1 装饰器定义方式
Python中有两种主要类型的装饰器：内置装饰器和自定义装饰器。内置装饰器是Python标准库提供的一些预先定义好的装饰器，如@property、@classmethod等。自定义装饰器则是由开发人员自己编写并定义的decorator函数，通常接受一个被修改（或“被包裹”）的目标函数作为参数并返回一个新的包含附加行为（或“增强”）目标函数后面调用时执行额外操作）decorator function对象。
