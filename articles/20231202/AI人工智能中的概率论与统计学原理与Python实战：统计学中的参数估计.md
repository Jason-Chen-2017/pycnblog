# 2.核心概念与联系

在本节中，我们将讨论概率论与统计学原理的核心概念，以及它们之间的联系。

## 2.1 概率论

概率论是一种数学方法，用于描述不确定性。它可以帮助我们理解事件发生的可能性，从而更好地做出决策。概率论的基本概念包括事件、概率、随机变量等。

### 2.1.1 事件

事件是概率论中的基本概念，它是一个可能发生或不发生的结果。事件可以是一个简单的结果，如抛硬币的结果，也可以是一个复杂的结果，如篮球比赛的结果。

### 2.1.2 概率

概率是概率论中的一个重要概念，它用于描述事件发生的可能性。概率通常用一个数字来表示，范围在0到1之间。0表示事件不可能发生，1表示事件必然发生。

### 2.1.3 随机变量

随机变量是概率论中的一个重要概念，它是一个可以取不同值的变量。随机变量可以是连续的，也可以是离散的。连续的随机变量可以取任意的值，而离散的随机变量只能取有限的值。

## 2.2 统计学

统计学是一种用于分析数据的方法，它可以帮助我们找出数据中的模式和趋势。统计学的基本概念包括数据、统计量、分布等。

### 2.2.1 数据

数据是统计学中的基本概念，它是一组数字或符号，用于描述事物的特征。数据可以是连续的，也可以是离散的。连续的数据可以取任意的值，而离散的数据只能取有限的值。

### 2.2.2 统计量

统计量是统计学中的一个重要概念，它是用于描述数据的一些特征的数字。统计量可以是描述性的，也可以是性质的。描述性的统计量用于描述数据的特征，如平均值、中位数等。性质的统计量用于描述数据的分布，如方差、标准差等。

### 2.2.3 分布

分布是统计学中的一个重要概念，它是用于描述数据的分布的方法。分布可以是连续的，也可以是离散的。连续的分布可以用概率密度函数来描述，而离散的分布可以用概率质量函数来描述。

## 2.3 概率论与统计学之间的联系

概率论与统计学之间有很强的联系。概率论可以帮助我们理解事件发生的可能性，而统计学可以帮助我们分析数据。在统计学中，我们经常需要使用概率论的概念，如概率和期望。

## 4.1 最大似然估计的具体代码实例

```python
import numpy as np

# 生成数据
np.random.seed(0)
x = np.random.normal(loc=0, scale=1, size=100)
y = 2 * x + np.random.normal(loc=0, scale=1, size=100)

# 定义似然函数
def likelihood(theta, x, y):
    return np.sum((y - theta * x) ** 2)

# 最大似然估计
theta_ml = np.argmin([likelihood(theta, x, y) for theta in np.linspace(-5, 5, 100)])
print("最大似然估计的参数值为：", theta_ml)
```
