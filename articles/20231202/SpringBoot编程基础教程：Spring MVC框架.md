# 2.核心概念与联系

在本节中，我们将介绍 Spring Boot 和 Spring MVC 的核心概念，以及它们之间的联系。

## 2.1 Spring Boot 的基本概念

Spring Boot 是一个用于构建 Spring 应用程序的快速开始点，它提供了一种简化的配置，以便快速开始构建 Spring 应用程序。Spring Boot 的目标是让开发者专注于编写代码，而不是花时间在配置和设置上。

Spring Boot 提供了许多有用的功能，例如自动配置、嵌入式服务器、数据访问、缓存、会话管理、定时任务等等。这使得开发者可以更快地构建和部署 Spring 应用程序。

## 2.2 Spring MVC 的基本概念

Spring MVC 是 Spring 框架的一个核心组件，它提供了一个用于处理 HTTP 请求和响应的框架。Spring MVC 使得开发者可以更轻松地构建 Web 应用程序，因为它提供了许多有用的功能，例如数据绑定、模型驱动编程、异常处理等等。

Spring MVC 的核心组件包括：

- 控制器（Controller）：处理 HTTP 请求并生成响应的组件。
- 模型（Model）：用于存储和传递应用程序数据的组件。
- 视图（View）：用于生成 HTTP 响应的组件。

## 2.3 Spring Boot 和 Spring MVC 的联系

Spring Boot 和 Spring MVC 之间的关系是：Spring Boot 是 Spring 框架的一个子集，它包含了 Spring MVC 的所有功能。这意味着当你使用 Spring Boot 时，你也可以使用 Spring MVC。

```java
@Controller
public class HelloController {

    @GetMapping("/hello")
    public String hello(Model model) {
        model.addAttribute("message", "Hello World!");
        return "hello";
    }
}
```

在上述代码中，我们创建了一个名为 `HelloController` 的类，并使用 `@Controller` 注解将其标记为一个控制器。我们还定义了一个名为 `hello` 的方法，并使用 `@GetMapping` 注解将其标记为一个 GET 请求映射。

# 6.附录常见问题与解答

在本节中，我们将回答一些常见问题。

Q：什么是 Spring Boot？
A：Spring Boot 是一个用于构建 Spring 应用程序的快速开始点，它提供了一种简化的配置，以便快速开始构建 Spring 应用程序。

Q：什么是 Spring MVC？
A：Spring MVC 是 Spring 框架的一个核心组件，它提供了一个用于处理 HTTP 请求和响应的框架。

Q：Spring Boot 和 Spring MVC 有什么关系？
A：Spring Boot 是 Spring 框架的一个子集，它包含了 Spring MVC 的所有功能。这意味着当你使用 Spring Boot 时，你也可以使用 Spring MVC。

Q：如何创建一个新的 Spring Boot 项目？
A：要创建一个新的 Spring Boot 项目，你可以使用 Spring Initializr 网站（https://start.spring.io/）来生成一个新的项目。

Q：如何配置项目的依赖关系？
A：要配置项目的依赖关系，你可以在项目的 `pom.xml` 文件中添加依赖项。

Q：如何创建一个新的控制器类？
A：要创建一个新的控制器类，你可以创建一个新的 Java 类，并使用 `@Controller` 注解将其标记为一个控制器。

Q：如何配置控制器类的方法？
A：要配置控制器类的方法，你可以使用各种注解，如 `@RequestMapping`、`@GetMapping`、`@PostMapping` 等，来映射 HTTP 请求。

Q：如何创建一个新的模型类？
A：要创建一个新的模型类，你可以创建一个新的 Java 类，并使用 `@Entity` 注解将其标记为一个实体类。

Q：如何配置模型类的属性？
A：要配置模型类的属性，你可以使用各种注解，如 `@Column`、`@NotNull`、`@Size` 等，来定义属性的约束。

Q：如何创建一个新的视图？
A：要创建一个新的视图，你可以创建一个新的 HTML 文件，并将其放在 `src/main/resources/templates` 目录下。

Q：如何配置视图的引擎？
A：要配置视图的引擎，你可以在项目的 `application.properties` 文件中添加相关的配置。

Q：如何运行项目？
A：要运行项目，你可以使用 Spring Boot 提供的 `spring-boot-maven-plugin` 插件来运行项目。
