## 漏洞扫描示例

以下是一个使用Nmap进行漏洞扫描的示例：

```python
import subprocess

def scan_vulnerability(ip, port):
    command = f"nmap -sT -p {port} {ip}"
    result = subprocess.check_output(command, shell=True)
    return result

ip = "192.168.1.1"
port = "80"
result = scan_vulnerability(ip, port)
print(result)
```

在这个示例中，我们使用Python的subprocess模块执行Nmap命令，并将扫描结果打印到控制台。

## 密码破解示例

以下是一个使用John the Ripper进行密码破解的示例：

```python
import subprocess

def crack_password(password_file, wordlist_file):
    command = f"john --wordlist={wordlist_file} {password_file}"
    result = subprocess.check_output(command, shell=True)
    return result

password_file = "password.hash"
wordlist_file = "wordlist.txt"
result = crack_password(password_file, wordlist_file)
print(result)
```

在这个示例中，我们使用Python的subprocess模块执行John the Ripper命令，并将破解结果打印到控制台。

## 网络监控示例

以下是一个使用Wireshark进行网络监控的示例：

```python
import subprocess

def start_monitoring(interface):
    command = f"wireshark -i {interface}"
    subprocess.Popen(command, shell=True)

interface = "eth0"
start_monitoring(interface)
```

在这个示例中，我们使用Python的subprocess模块执行Wireshark命令，并启动网络监控。
