## 2.1 概率图模型

概率图模型（Probabilistic Graphical Model，PGM）是一种用于表示概率分布的图形表示方法，它将概率分布表示为一个有向无环图（DAG）。每个节点表示一个随机变量，每条边表示一个条件独立关系。概率图模型可以用来表示复杂的概率关系，并且可以用于进行推理和预测。

## 2.2 条件独立性

条件独立性是概率图模型的核心概念之一。条件独立性是指两个随机变量在给定其他变量的条件下，它们之间的概率分布是独立的。在概率图模型中，每条边表示一个条件独立关系，即给定其他变量，两个相连的节点之间的概率分布是独立的。

## 2.3 贝叶斯定理

贝叶斯定理是概率图模型的另一个核心概念。贝叶斯定理是一种用于计算条件概率的公式，它可以用来计算给定某个事件发生的条件下，另一个事件的概率。在概率图模型中，贝叶斯定理可以用来进行推理和预测。

$$

其中，$P(A|B)$ 表示给定事件 $B$ 发生的条件下，事件 $A$ 的概率；$P(B|A)$ 表示给定事件 $A$ 发生的条件下，事件 $B$ 的概率；$P(A)$ 表示事件 $A$ 的概率；$P(B)$ 表示事件 $B$ 的概率。

## 4.1 使用Python的pgmpy库构建概率图模型

我们可以使用Python的pgmpy库来构建概率图模型。pgmpy是一个用于构建和操作概率图模型的库，它提供了一系列的函数和类来帮助我们构建和操作概率图模型。

以下是一个使用pgmpy库构建概率图模型的示例：

```python
from pgmpy.models import BayesianModel
from pgmpy.factors.discrete import DiscreteFactor

# 创建随机变量
A = DiscreteFactor('A', card=2, values=[0, 1])
B = DiscreteFactor('B', card=2, values=[0, 1])
C = DiscreteFactor('C', card=2, values=[0, 1])

# 创建条件独立关系
model = BayesianModel([(A, B), (A, C)], [A, B, C])

# 添加条件概率表
model.add_evidence(A, 1)
model.add_evidence(B, 1)

# 计算条件概率
print(model.get_evidence(C, 1))
```

在上述代码中，我们首先导入了pgmpy库，并创建了三个随机变量：$A$、$B$ 和 $C$。然后，我们创建了一个条件独立关系，即$A$ 与 $B$ 之间是条件独立的，$A$ 与 $C$ 之间是条件独立的。接下来，我们添加了条件概率表，并计算了条件概率。

from pgmpy.factors.discrete import DiscreteFactor
