规则是规则引擎的基本组成部分，它定义了在特定条件下需要执行的操作。规则通常由条件部分和操作部分组成，条件部分用于判断是否满足规则执行的条件，操作部分用于执行相应的操作。

## 2.2 事件

事件是规则引擎中的触发器，它表示某个特定的发生情况。当事件发生时，规则引擎会根据规则的条件部分判断是否满足执行规则的条件，从而触发相应的操作。

## 2.3 决策

# 定义规则

    {
        "condition": "supplier_stock < 1000",
        "action": "notify_supplier"
    },
    {
        "condition": "transport_time > 7",
        "action": "notify_transport_company"

    "supplier_stock": 900,
    "transport_time": 8
}

# 执行决策
engine.execute(event)
