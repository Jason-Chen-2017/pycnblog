# 2.核心概念与联系
## 2.1面向对象编程（Object-Oriented Programming, OOP）
面向对象编程是一种软件设计方法，它将问题划分为“对象”（Object），每个对象都包含数据（Attributes）和操作（Methods）。这种方法使得代码更加模块化、可重用性强且易于维护。主要概念包括：类（Class）、对象（Object）、属性（Attributes）和方法（Methods）等。
## 2.2类与对象
类是一种蓝图或者说模板，用于创建具有相同特征和行为的对象。而对象则是类实例化后生成的具体实体。每个对象都有其独立的内存空间来存储其属性和方法。通过创建不同类型的对象，可以轻松地处理复杂问题并组织代码结构。
## 2.3属性与方法
属性是类或者对象所拥有的特征值或变量；方法则是类或者对象所具备的功能行为或动作。通过定义合适的属性和方法，可以使代码更加易于理解与维护。同时也可以提高代码复用性与灵活性。
