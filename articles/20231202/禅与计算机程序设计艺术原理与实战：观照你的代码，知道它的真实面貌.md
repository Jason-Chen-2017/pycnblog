### 3.1 动态规划的基本概念

动态规划（Dynamic Programming，简称DP）是一种优化问题的解决方法，它通过将问题分解为子问题，并将子问题的解存储在一个表格中，从而避免了多次计算相同的子问题。动态规划的核心思想是“分而治之”，即将一个复杂的问题分解为多个简单的子问题，然后将子问题的解组合起来得到原问题的解。

1. 确定dp数组的大小：根据问题的具体情况，确定dp数组的大小。dp数组的大小通常与问题的输入相关。
2. 初始化dp数组：根据问题的具体情况，初始化dp数组。初始化的方式可以是设置所有元素为0，或者设置某些元素为特定值。
3. 计算每个状态的值：根据问题的具体情况，计算每个状态的值。计算的方式可以是递归的，也可以是迭代的。
4. 填充dp数组：根据问题的具体情况，填充dp数组。填充的方式可以是递归的，也可以是迭代的。

假设我们需要计算一个数列的最大和，数列的元素为a1, a2, ..., an。我们可以使用动态规划的方法来解决这个问题。

### 4.2 代码实例

以下是一个Python的代码实例，用于计算两个字符串的最长公共子序列：

```python
def longest_common_subsequence(s1, s2):
    m = len(s1)
    n = len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```
