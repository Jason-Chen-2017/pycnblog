## 4.2 特征提取

我们将通过一个简单的Sobel边缘检测操作来演示特征提取的具体实现。

```python
import cv2
import numpy as np

# 读取图像

# 转换为灰度图
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 使用Sobel算子检测边缘
sobelx = cv2.Sobel(gray, cv2.CV_64F, 1, 0, ksize=5)
sobely = cv2.Sobel(gray, cv2.CV_64F, 0, 1, ksize=5)

# 计算梯度的平方和
gradient = np.sqrt(sobelx**2 + sobely**2)

# 显示边缘图
cv2.imshow('edges', gradient)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

在这个代码中，我们首先使用`cv2.imread`函数来读取图像。然后，我们使用`cv2.cvtColor`函数来转换图像为灰度图。接着，我们使用`cv2.Sobel`函数来计算图像的梯度。最后，我们使用`cv2.imshow`函数来显示边缘图。
