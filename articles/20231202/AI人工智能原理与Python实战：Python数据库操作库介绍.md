# 2.核心概念与联系
在进入具体内容之前，我们需要了解一些关键概念：
- Python：一种高级编程语言，拥有简单易学、强大功能等优点。
- AI：人工智能是指通过计算机程序模拟人类智能的科学研究领域。
- ML：机器学习是一种自动化方法，通过计算机程序从数据中自动发现模式或规律。
- 数据库：存储结构化数据的仓库，可以对其进行查询、更新等操作。
- Python数据库操作库：提供接口来访问不同类型数据库（如MySQL、PostgreSQL等）并执行相关操作的软件包。
