                 

# 1.背景介绍

禅的本质是一种对现实的直接体验。表示的术语可以是“直觉”、“直接经验”或“直觉”。禅是详细分析问题的自然产物，它不可能独立于问题或实例的CONTEXT之外存在。 禅更适合反映补 ful状态的本质，而非具体的附加内容；它是人性的正面表现，成为剑指情绪和行为方式的直接体验，与方法间的分析和论诉无关。 然而，禅并不是偏好我一个人或一些人的感受，它是另一个获取状态的方式。它强调迁移到不平坦的平面，通过直接体验来驱动与世界无关的流畅。在计算机程序设计的范围内，我，这个痴迷于编程的学生，了解了禅的卷本。在我卖掉大部分的财产后，我踏入米拉斯大河的尽头，感受当下与几何的关联，感受逻辑判定与物质世界的最后与我无关的印象。我甚至开始感受直觉，因为督促、偏见和情绪对我没有作用。我踏入不确定的状态，学习如何在静止的基础上eland logic并使“求精”得以正式启动。
