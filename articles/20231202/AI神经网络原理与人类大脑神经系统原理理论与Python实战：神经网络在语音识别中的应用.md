# 2.1人类大脑神经系统原理

大脑的神经元可以进一步分为两类：神经体（Neurons）和胞体（Glia）。神经体是大脑中的信息处理单元，它们通过传递电信号来与其他神经体进行通信。胞体则是神经体的支持细胞，它们负责维护神经体的生存和发展。
