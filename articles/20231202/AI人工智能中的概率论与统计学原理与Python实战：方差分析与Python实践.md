# 2.核心概念与联系
## 2.1概率论
概率论是数学的一个分支，主要研究事件发生的可能性和相关概念。在AI人工智能中，我们需要对数据进行预测和判断，因此需要了解概率论的基本概念和公式。

### 2.1.1事件、样本空间、事件空间
- **事件**：一个实验或观察结果可以出现多种不同的结果集合称为事件。例如：抛硬币的结果有两种：正面（H）或反面（T）。
- **样本空间**：所有可能发生的结果构成一个集合称为样本空间。在硬币举例中，样本空间为{H, T}。
- **事件空间**：所有可能发生的事件构成一个集合称为事件空间。在硬币举例中，事件空间为{H, T}。
### 2.1.2条件概率、独立性、贝叶斯定理
- **条件概率**：给定某个事件B已经发生，另一个事件A发生的概率称为条件概率P(A|B)。通常表示为P(A|B)=P(A∩B)/P(B)。
- **独立性**：两个或多个事件之间没有任何关联时，它们被认为是独立的。例如：抛硬币和掷骰子两个实验是相互独立的。
