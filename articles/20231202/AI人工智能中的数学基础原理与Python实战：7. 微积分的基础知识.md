# 2.核心概念与联系

## 2.1 积分

积分是微积分的一个重要概念，它是对一个函数在一个区间上的面积的计算。积分可以用来计算曲线下的面积、曲线上的长度、曲线的曲率等。在机器学习中，积分是解决许多问题的关键技术，例如梯度下降法、回归分析等。
