以下是一个简单的反射技术的代码实例：

```java
import java.lang.reflect.Method;

public class ReflectionExample {
    public static void main(String[] args) throws Exception {
        // 获取一个动态代理对象
        Method[] methods = ReflectionExample.class.getDeclaredMethods();
        Object proxy = Proxy.newProxyInstance(ReflectionExample.class.getClassLoader(),
                methods,
                new InvocationHandler() {
                    @Override
                    public Object invoke(Object proxy, Method method, Object[] args) throws Throwable {
                        // 在这里可以对方法的调用进行拦截和修改
                        return method.invoke(ReflectionExample.this, args);
                    }
                });

        // 使用反射技术调用程序的方法和属性
        Method method = ReflectionExample.class.getDeclaredMethod("sayHello");
        method.invoke(proxy);
    }

    public void sayHello() {
        System.out.println("Hello, World!");
    }
}
```

上述代码实例中，我们首先获取了一个动态代理对象，并使用了一个InvocationHandler来拦截和修改程序的调用。然后，我们使用反射技术调用了程序的方法和属性。
