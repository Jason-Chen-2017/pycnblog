## 2.1 什么是数据挖掘？
数据挖掘（Data Mining）是指从海量、复杂、不规则的数据集中抽取有价值信息或知识的过程。这些信息或知识可以帮助企业做出更明智、更准确的决策。通常情况下，这些信息或知识被嵌入到大量无结构化或半结构化的记录中，需要使用专门的算法和技术来找出这些信息或知识所在。
