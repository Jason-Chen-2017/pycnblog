# 2.核心概念与联系
在Go语言中，GUI开发主要依赖于两个核心库：`html/template`和`github.com/golang/freetype`。

`html/template`库提供了用于创建和渲染HTML模板的功能。通过使用这个库，我们可以轻松地创建具有交互性和动态内容的GUI界面。

`github.com/golang/freetype`库则提供了用于处理和渲染SVG图像的功能。通过使用这个库，我们可以轻松地在GUI界面中添加图像、图标等元素。
