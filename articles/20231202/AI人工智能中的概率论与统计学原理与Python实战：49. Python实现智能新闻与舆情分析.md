# 2.核心概念与联系
在进行智能新闻与舆情分析之前，我们需要了解一些基本概念：
- **数据**：数据是信息的集合，可以是数字、文本、图像等形式。在新闻与舆情分析中，数据可以包括新闻报道、评论文章、社交媒体内容等。
- **特征**：特征是描述数据的属性或特点。例如，新闻报道可能包含标题、摘要、作者等信息；评论文章可能包含作者、发表时间等信息；社交媒体内容可能包含发布者、发布时间等信息。
- **训练集**：训练集是用于训练机器学习模型的数据集合。它由输入（特征）和输出（标签）组成，其中输入是已知的特征值，输出是根据这些特征值进行预测或分类的结果。在新闻与舆情分析中，训练集可以包括已知为正面或负面新闻报道或评论文章的样本。
- **测试集**：测试集是用于评估机器学习模型性能的数据集合。它由输入（特征）组成，但没有对应的输出（标签）。在新闻与舆情分析中，测试集可以包括未知类别或主题的新闻报道或评论文章样本。
- **模型**：模型是一个数学函数或算法，用于描述数据之间存在关系并进行预测或分类任务。在智能新闻与舆情分析中，模型可以使用各种方法进行构建和优化，例如支持向量机（SVM）、逻辑回归、随机森林等方法。
