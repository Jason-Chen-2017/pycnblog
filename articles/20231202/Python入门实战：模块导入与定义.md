# 2.核心概念与联系
## 2.1 什么是模块？
在Python中，模块是一个包含多个变量、函数或类的文件。通过导入这个文件，我们可以在当前脚本中使用其内容。模块通常以`.py`结尾，并且可以包含多个类或函数。每个Python文件都是一个独立的模块，可以被其他程序引用和使用。

## 2.2 什么是导入？
导入是指从其他文件中引用某些功能或变量到当前脚本中去使用它们。通过导入一个模块，我们可以访问该模块定义的所有功能和变量。Python提供了`import`关键字来实现这一功能。例如：`import math`表示从math库中引入所有功能和变量；`from math import pi`表示从math库引入pi变量；`import math as m`表示从math库引入所有功能和变量并给予别名m；`from math import sqrt as s`表示从math库引入sqrt函数并给予别名s等等。
