from tensorflow.keras.layers import Dense, Conv2D, Flatten, Dropout, MaxPooling2D
from tensorflow.keras.models import Sequential

# 定义模型
model = Sequential()

# 添加卷积层
model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))

# 添加池化层
model.add(MaxPooling2D((2, 2)))

# 添加扁平层
model.add(Flatten())

# 添加全连接层
model.add(Dense(128, activation='relu'))

# 添加Dropout层
model.add(Dropout(0.5))

# 添加输出层
model.add(Dense(10, activation='softmax'))

# 编译模型
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# 训练模型
model.fit(x_train, y_train, epochs=10, batch_size=32)

首先，我们导入了所需的库。然后，我们定义了一个Sequential模型，这是一个线性堆叠的神经网络模型。

接下来，我们添加了卷积层、池化层、扁平层、全连接层和Dropout层。这些层可以帮助我们学习特征、减少过拟合和提高模型的准确性。

然后，我们添加了输出层，这个输出层可以用于预测图像的类别。我们使用了softmax激活函数，因为这是多类分类问题的常用激活函数。

接下来，我们使用了Adam优化器来编译模型。这是一种常用的优化器，它可以自动调整学习率。我们使用了交叉熵损失函数，因为这是多类分类问题的常用损失函数。

最后，我们使用了训练数据来训练模型。我们设置了10个训练轮次和32个批次大小。
