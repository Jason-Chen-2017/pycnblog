                 

# 1.背景介绍

Redis（Remote Dictionary Server）是一个开源的高性能的key-value存储系统，由Salvatore Sanfilippo开发。Redis支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。Redis不仅仅支持简单的key-value类型的数据，同时还提供list、set、hash和sorted set等数据结构的存储。

Redis支持数据的备份，即master-slave模式的数据备份，以及全量复制和增量复制等方式进行数据备份。Redis还支持Pub/Sub模式，可以实现消息通信。

Redis是一个使用ANSI C语言编写、遵循BSD协议的开源软件（ BSD Licensed Open Source Software）。Redis的核心团队由Italian NoSQL Group（意大利NoSQL组）组成。
