# 2.核心概念与联系
## 2.1 Phoenix框架简介
Phoenix是一个基于Elixir语言的Web框架，它使用OTP库来构建可扩展且易于维护的应用程序。Phoenix提供了强大的功能，如路由、控制器、视图等，使得开发者可以快速地构建复杂的Web应用程序。同时，Phoenix还支持实时通信功能，使得开发者可以轻松地集成聊天室、推送消息等功能。

## 2.2 Elixir语言简介
Elixir是一种动态类型函数式编程语言，它基于Erlang虚拟机（BEAM）运行。Elixir具有轻量级线程模型、分布式原生支持等特点，使得其在并发处理方面表现出色。此外，Elixir也具有强大的错误处理机制和自动垃圾回收功能，使得开发者可以更关注业务逻辑而非底层细节。

## 2.3 OTP库简介
OTP（Open Telecom Platform）是Erlang/OTP系统中最重要的组件之一。OTP提供了许多预先构建好的软件组件和工具库，这些组件可以帮助开发者快速构建分布式系统。OTP包含了许多关键功能模块，如进程管理、错误处理、日志记录等，这些功能模块都是基于Erlang虚拟机（BEAM）运行的。
