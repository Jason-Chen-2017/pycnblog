                 

# 1.背景介绍

回文串是指一个字符串读起来和反读起来是一样的，例如“abba”、“level”等。回文串在字符串处理、模式匹配等领域具有重要的应用价值。回文串的判断问题是一种经典的字符串问题，其核心是判断一个字符串是否为回文串。

回文串的判断问题可以分为两种：完全回文串和半回文串。完全回文串是指一个字符串的前半部分与后半部分相同，如“abba”、“level”等。半回文串是指一个字符串的前半部分与后半部分相同，但可能有一个或多个字符不同，如“abba”、“level”等。

回文串的判断问题可以采用多种不同的算法来解决，例如KMP算法、Z算法等。在本文中，我们将主要介绍回文串的判断问题的两种常见算法：Manacher算法和Rabin-Karp算法。

## 2.1 Manacher算法
Manacher算法是一种高效的回文串判断算法，其核心思想是利用动态规划和对称性特征来判断一个字符串是否为回文串。Manacher算法的时间复杂度为O(n)，其中n是字符串的长度。

Manacher算法的核心步骤如下：
1. 将输入字符串的每个字符都加上一个特殊字符，使得字符串变成一个回文串。例如，将字符串“abba”变为“^#a#b#a#$”。
2. 根据字符串的对称性特征，将字符串划分为多个回文子串，并计算每个回文子串的最大长度。例如，将字符串“^#a#b#a#$”划分为“^#”、“a#b#a#”、“#$”等回文子串，并计算每个回文子串的最大长度。
3. 根据最大长度信息，判断输入字符串是否为回文串。例如，根据“^#a#b#a#$”的最大长度信息，可以判断输入字符串“abba”是回文串。

# 4.具体代码实例和详细解释说明

## 4.1 Manacher算法实现
```python
def manacher(s):
    if not s:
        return []
    t = "#"
    for c in s:
        t += c + "#"
    n = len(t)
    p = [0] * n
    c = 0
    for i in range(1, n - 1):
        if i < c + p[c]:
            p[i] = min(p[2 * c - i], c + p[c] - i)
        while t[i - p[i] - 1] == t[i + p[i] + 1]:
            p[i] += 1
        if i + p[i] > c + p[c]:
            c = i
    return p
```
