## 3.1贝叶斯网络的基本概念
贝叶斯网络是一种有向无环图（DAG），用于表示随机变量之间的条件依赖关系。贝叶斯网络的核心概念是条件依赖性和条件独立性。

### 3.1.1条件依赖性
条件依赖性是指一个随机变量与另一个随机变量之间的关系。在贝叶斯网络中，条件依赖性是通过有向边来表示的。有向边表示从一个随机变量到另一个随机变量的依赖关系。

### 3.1.2条件独立性
条件独立性是指一个随机变量与另一个随机变量之间的关系。在贝叶斯网络中，条件独立性是通过无向边来表示的。无向边表示两个随机变量之间的关系是独立的。

## 3.2贝叶斯网络的算法原理
贝叶斯网络的算法原理是基于贝叶斯定理的。贝叶斯定理是一种概率推理方法，用于计算条件概率。贝叶斯定理的公式是：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

在贝叶斯网络中，我们可以使用贝叶斯定理来计算条件概率。

```python
import networkx as nx
import numpy as np

# 创建贝叶斯网络
G = nx.DiGraph()

# 添加节点
G.add_node('A')
G.add_node('B')
G.add_node('C')

# 添加边
G.add_edge('A', 'B')
G.add_edge('B', 'C')

# 计算条件概率
P_A = np.array([0.5, 0.5])
P_B_given_A = np.array([0.8, 0.2])
P_C_given_B = np.array([0.7, 0.3])

P_B = np.array([0.5 * P_A[0] * P_B_given_A[0] + 0.5 * P_A[1] * P_B_given_A[1],
                0.5 * P_A[0] * P_B_given_A[1] + 0.5 * P_A[1] * P_B_given_A[0]])

P_C = np.array([0.5 * P_B[0] * P_C_given_B[0] + 0.5 * P_B[1] * P_C_given_B[1],
                0.5 * P_B[0] * P_C_given_B[1] + 0.5 * P_B[1] * P_C_given_B[0]])
```
