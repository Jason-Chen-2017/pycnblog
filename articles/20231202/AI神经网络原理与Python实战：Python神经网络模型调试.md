# 2.核心概念与联系
## 2.1神经网络基础知识
- 输入层：数据进入神经网络的第一层。
- 隐藏层：数据通过输入层后进入隐藏层，每个节点都会对数据进行处理并传递给下一层。
- 输出层：最后一个阶段，生成预测结果或决策。
- 权重：连接不同节点之间的数值参数。权重可以通过训练来调整和优化模型性能。
- 激活函数：控制输出值范围和形状的函数，常见激活函数有sigmoid、tanh和ReLU等。
- 损失函数：衡量预测结果与真实结果之间差异的指标，常见损失函数有均方误差（MSE）和交叉熵损失（Cross Entropy Loss）等。
- 反向传播：根据损失函数对权重进行微调的算法。通过计算前向传播得到预测结果后，从输出端向前向回传错误信息并更新权重值。这个过程被称为反向传播（backpropagation）。
## 2.2 Python中的深度学习库TensorFlow与Keras简介
TensorFlow是Google开发的开源深度学习框架，提供了许多高级API来简化深度学习任务的实现和优化。Keras是一个高级Neural Networks API，运行于TensorFlow上，允许快速原型设计和构建深度学习模型。Keras提供了简单易用且强大功能的抽象层次结构以及灵活且可扩展的底层接口，使得创建复杂模型变得容易且高效。本文将主要基于Keras库来演示如何使用Python实现神经网络模型调试。
