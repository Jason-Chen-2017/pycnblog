# 2.核心概念与联系
## 2.1概率论与统计学的基本概念
### 2.1.1随机事件
随机事件是指在某个实验或过程中可能发生或不发生的事情。例如：抛硬币、抽卡等。
### 2.1.2样本空间
样本空间是所有可能结果构成的集合，表示所有可能性。例如：硬币抛出结果为正面或反面组成的集合。
### 2.1.3事件
事件是样本空间中的一个子集，表示某种结果发生或不发生。例如：硬币抛出结果为正面组成的子集。
### 2.1.4概率
概率是一个事件发生的可能性，通常表示为0到1之间的一个数值。它反映了某个事件在所有可能性中出现的频率。例如：硬币正面出现的概率为0.5（假设硬币均衡）。
## 2.2朴素贝叶斯算法基本概念
### 2.2.1条件依赖性与条件独立性
条件依赖性：两个变量相互依赖，即其中一个变量对另一个变量有影响；条件独立性：两个变量相互独立，即其中一个变量对另一个变量无影响。朴素贝叶斯算法利用条件独立性来简化问题复杂度，提高计算效率。
