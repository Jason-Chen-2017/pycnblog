# 2.核心概念与联系
大模型即服务（Model-as-a-Service，MaaS）是一种将大型人工智能模型作为服务提供给用户的方式。这种方式使得用户无需自己构建和训练大型模型，而是可以通过API或其他接口直接使用这些模型。这种方式有助于降低用户的成本和技术门槛，从而提高用户体验。

# 训练模型
