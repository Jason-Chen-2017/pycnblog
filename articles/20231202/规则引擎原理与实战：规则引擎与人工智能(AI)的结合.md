# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来演示规则引擎的使用方法。假设我们有一个简单的规则库，用于判断一个人是否满足某个条件。

```python
rules = [
    {
        "condition": "age >= 18",
        "action": "grant_access"
    },
    {
        "condition": "age < 18",
        "action": "deny_access"
    }
]
```

我们可以使用以下代码来实现规则引擎的基本功能：

```python
def execute_rules(rules, data):
    for rule in rules:
        if eval(rule["condition"]):
            return rule["action"]
    return None

data = {"age": 20}
result = execute_rules(rules, data)
print(result)  # Output: "grant_access"
```

在这个例子中，我们首先定义了一个`execute_rules`函数，用于执行规则库中的规则。我们遍历规则库中的每个规则，并使用`eval`函数来评估规则的条件。如果条件满足，我们就执行相应的操作并返回结果。如果所有规则都不满足，我们返回`None`。

我们创建了一个名为`data`的字典，用于存储需要处理的数据。然后我们调用`execute_rules`函数来执行规则库中的规则，并将结果打印出来。
