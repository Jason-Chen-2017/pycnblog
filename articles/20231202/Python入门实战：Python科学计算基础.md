### 1.4.1 排序算法实例
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

arr = [64, 34, 25, 12, 22, 11, 90]
bubble_sort(arr)
print("排序后的数组为：")
for i in range(len(arr)):
    print("%d" % arr[i])
```
这个代码实例是一个冒泡排序算法的实现，它可以用来对一个数组进行排序。

### 1.4.2 搜索算法实例
```python
def binary_search(arr, x):
    low = 0
    high = len(arr) - 1
    mid = 0

    while low <= high:
        mid = (high + low) // 2

        if arr[mid] < x:
            low = mid + 1
        elif arr[mid] > x:
            high = mid - 1
        else:
            return mid

    return -1

arr = [2, 3, 4, 10, 40]
x = 10

result = binary_search(arr, x)

if result != -1:
    print("元素找到，位置为:", str(result))
else:
    print("元素不存在")
```
这个代码实例是一个二分搜索算法的实现，它可以用来在一个有序数组中查找一个特定的元素。

### 1.4.3 分治算法实例
```python
def divide_and_conquer(arr, low, high):
    if low >= high:
        return

    mid = (low + high) // 2
    divide_and_conquer(arr, low, mid)
    divide_and_conquer(arr, mid + 1, high)

    merge(arr, low, mid, high)

def merge(arr, low, mid, high):
    n1 = mid - low + 1
    n2 = high - mid

    L = [0] * (n1)
    R = [0] * (n2)

    for i in range(0, n1):
        L[i] = arr[low + i]
    for j in range(0, n2):
        R[j] = arr[mid + 1 + j]

    i = 0
    j = 0
    k = low

    while i < n1 and j < n2:
        if L[i] <= R[j]:
            arr[k] = L[i]
            i += 1
        else:
            arr[k] = R[j]
            j += 1
        k += 1

    while i < n1:
        arr[k] = L[i]
        i += 1
        k += 1

    while j < n2:
        arr[k] = R[j]
        j += 1
        k += 1

arr = [1, 3, 5, 7, 9, 2, 4, 6, 8, 0]
divide_and_conquer(arr, 0, len(arr) - 1)
print(arr)
```
这个代码实例是一个分治算法的实现，它可以用来对一个数组进行排序。

### 1.4.4 动态规划算法实例
```python
def fibonacci(n):
    if n < 0:
        print("输入错误")
    elif n == 1:
        return 0
    elif n == 2:
        return 1
    else:
        a, b = 0, 1
        for i in range(2, n):
            a, b = b, a + b
        return b

print(fibonacci(9))
```
这个代码实例是一个斐波那契数列的动态规划算法的实现，它可以用来计算斐波那契数列的第n项。
