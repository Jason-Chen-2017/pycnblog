# 2.核心概念与联系

## 2.1响应式设计的核心概念

响应式设计是一种网页设计方法，它可以让网页在不同的设备和屏幕尺寸上保持良好的显示效果。这是通过使用CSS和HTML5的媒体查询来实现的。媒体查询可以根据设备的屏幕尺寸、分辨率、方向等特征来应用不同的样式表。这样，我们可以为不同的设备提供不同的样式，从而实现响应式设计。

## 2.2Bootstrap框架的核心概念

Bootstrap是一个基于HTML、CSS和JavaScript的前端框架，它提供了许多有用的组件和工具，可以帮助我们快速开发网页。Bootstrap的核心概念包括：

- 基本HTML和CSS结构：Bootstrap提供了一系列的HTML和CSS类，可以帮助我们快速构建网页布局和组件。
- 响应式设计：Bootstrap的设计原则是“移动优先”，即首先考虑在移动设备上的显示效果。通过使用Bootstrap的响应式工具类，我们可以轻松实现响应式设计。
- 组件和插件：Bootstrap提供了许多有用的组件，如导航栏、按钮、表格等。此外，Bootstrap还提供了许多插件，如日历、图片轮播等。
- JavaScript：Bootstrap提供了一些JavaScript组件，如模态框、弹出框等。这些组件可以帮助我们实现更丰富的交互效果。

```css
@media (max-width: 767px) {
  /* 在屏幕宽度小于或等于767像素时应用的样式 */
}
```

在Bootstrap框架中，我们可以使用`col-*`类来实现响应式布局。这些类的基本语法如下：

```css
.col-xs-* { /* 在所有设备上的样式 */ }
.col-sm-* { /* 在屏幕宽度大于767像素的设备上的样式 */ }
.col-md-* { /* 在屏幕宽度大于992像素的设备上的样式 */ }
.col-lg-* { /* 在屏幕宽度大于1200像素的设备上的样式 */ }
```

其中，`*`表示一个数字，表示列的宽度。例如，`col-sm-3`表示在屏幕宽度大于767像素的设备上，该列的宽度为3列。

## 4.1代码实例

以下是一个简单的Bootstrap响应式设计的代码实例：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <title>Bootstrap响应式设计实例</title>
  <!-- 引入Bootstrap基本文件 -->
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>
</head>
<body>

<div class="container">
  <div class="row">
    <div class="col-xs-12 col-sm-6">
      <h2>屏幕宽度小于或等于767像素时的显示效果</h2>
      <p>在屏幕宽度小于或等于767像素的设备上，该列的宽度为12列。</p>
    </div>
    <div class="col-xs-12 col-sm-6">
      <h2>屏幕宽度大于767像素时的显示效果</h2>
      <p>在屏幕宽度大于767像素的设备上，该列的宽度为6列。</p>
    </div>
  </div>
</div>

</body>
</html>
```

在上述代码中，我们首先引入了Bootstrap的基本文件，包括CSS文件、JavaScript文件和图标文件。然后，我们使用Bootstrap的基本HTML和CSS结构来构建网页布局。最后，我们使用Bootstrap的响应式设计工具类`col-xs-12`和`col-sm-6`来实现不同设备的显示效果。

## 4.2详细解释说明

在上述代码中，我们首先引入了Bootstrap的基本文件，包括CSS文件、JavaScript文件和图标文件。然后，我们使用Bootstrap的基本HTML和CSS结构来构建网页布局。最后，我们使用Bootstrap的响应式设计工具类`col-xs-12`和`col-sm-6`来实现不同设备的显示效果。

- `col-xs-12`：在所有设备上，该列的宽度为12列。
- `col-sm-6`：在屏幕宽度大于767像素的设备上，该列的宽度为6列。

通过使用这些工具类，我们可以实现不同设备的显示效果。当屏幕宽度小于或等于767像素时，两个列会水平排列。当屏幕宽度大于767像素时，两个列会垂直排列。
