# 2.核心概念与联系
## 2.1 Python数据类型简介
Python语言提供了多种内置的数据类型，包括整数、浮点数、字符串、列表、元组、字典等。每种数据类型都有其特定的属性和方法，可以用来处理不同类型的数据。下面我们将详细介绍这些数据类型及其应用场景。
### 2.1.1 整数（int）：
整数是一种无小数部分的数字，可以是正数或负数。例如：3, -5, 0。整数可以用于计算和存储整个数值信息。
### 2.1.2 浮点数（float）：
浮点数是一种带小数部分的数字，例如：3.14, -5.67, 0.001。浮点数可以用于计算需要精确到小数点后几位的信息。
### 2.1.3 字符串（str）：
字符串是由一个或多个字符组成的序列，例如："Hello", "World", "Python".字符串可以用于存储和处理文本信息。
### 2.1.4 列表（list）：
列表是一种可变序列集合，可以包含任意类型的对象。例如：[1, "Hello", [3, "World"]].列表可以用于存储和处理复杂结构的信息。
### 2.1.5 元组（tuple）：
元组是一种不可变序列集合，也可以包含任意类型的对象。例如：(1, "Hello", (3, "World")).元组与列表相似，但不能修改其内容。元组主要用于存储不需要修改的信息集合。
### 2.1.6 字典（dict）：
字典是一种键值对映射关系集合，例如{"name": "John", "age": 30}.字典主要用于存储并查询关联关系信息集合。
