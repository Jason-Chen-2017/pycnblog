# 2.核心概念与联系

## 2.1 任务

任务是需要执行的操作，它可以是计算任务、数据处理任务或者业务任务等。任务可以是同步的或者异步的，同步任务需要等待任务完成后才能继续执行，异步任务可以在任务执行过程中继续执行其他任务。

## 2.2 任务调度器

任务调度器是负责调度任务的组件，它可以根据任务调度策略来调度任务。任务调度器可以是中心化的或者分布式的，中心化的任务调度器是一个单点，它可以接收所有任务的请求，分布式的任务调度器是一个集群，它可以根据任务调度策略来调度任务。

## 2.3 任务调度策略

任务调度策略是任务调度器使用的算法，它可以根据任务的特点来调度任务。任务调度策略可以是基于资源的策略、基于时间的策略或者基于需求的策略等。

## 2.4 任务调度器组件

任务调度器组件是任务调度器的组成部分，它可以包括任务调度器的控制器、任务调度器的存储器和任务调度器的监控器等。
