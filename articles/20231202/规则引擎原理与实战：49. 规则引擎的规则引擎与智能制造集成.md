## 4.1 规则引擎的代码实例

以下是一个简单的规则引擎的代码实例：

```python
class RuleEngine:
    def __init__(self, rules):
        self.rules = rules

    def process(self, input_data):
        for rule in self.rules:
            if rule.condition(input_data):
                return rule.action(input_data)
        return None
```
