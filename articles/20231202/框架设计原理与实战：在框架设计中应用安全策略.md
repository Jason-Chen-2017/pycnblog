```python
def validate_input(input_data):
    # 使用正则表达式对输入数据进行验证
    pattern = re.compile(r'^[a-zA-Z0-9]+$')
    if pattern.match(input_data):
        return True
    else:
        return False

def handle_request(request_data):
    # 对请求数据进行验证
    if validate_input(request_data):
        # 处理请求
        # ...
    else:
        # 返回错误信息
        return "Invalid input"
```

在这个代码实例中，我们首先定义了一个 `validate_input` 函数，该函数使用正则表达式对输入数据进行验证。然后，我们在 `handle_request` 函数中调用 `validate_input` 函数，以确保请求数据是有效的和安全的。
