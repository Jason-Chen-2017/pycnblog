# 2.核心概念与联系
## 2.1 强化学习基础概念
强化学习是一种动态决策过程，其目标是让代理（agent）在环境中取得最大收益。在这个过程中，代理会接收环境的反馈信息并根据所学到的知识选择合适的行动。强化学习主要包括四个组成部分：状态空间（state space）、动作空间（action space）、奖励函数（reward function）和策略（policy）。

### 2.1.1 状态空间
状态空间是指代理在环境中可以取得的所有可能状态集合。每个状态都包含了当前环境的所有相关信息，用于帮助代理做出决策。例如，在一个自动驾驶汽车问题中，状态空间可以包括车速、路况、天气等因素。

### 2.1.2 动作空间
动作空间是指代理可以执行的所有可能行为集合。每个行为都会导致环境从一个状态转移到另一个状态。例如，在同样的自动驾驶汽车问题中，动作空间可以包括加速、减速、刹车等操作。

### 2.1.3 奖励函数
奖励函数用于评估代理在环境中取得收益的方式。通常情况下，奖励函数会根据当前状态和执行的行为给出正负值奖励 feedbacks,从而激励或惩罚代理采取不同行为策略 policy,s..例如,在游戏中,玩家获胜时获得正奖励,失败时获得负奖励;而在自动驾驶问题中,安全驾驶获得正奖励,不安全驾驶获得负奖励..
