假设我们有一个简单的规则库，用于判断一个用户是否满足购买某个产品的条件：

```

rule2: if age < 18 then can_buy = false
```

我们的工作内存包括以下信息：

```
age = 20
balance = 1200
```

根据这些信息，我们可以执行以下步骤：

1. 加载规则库：从文件中加载规则库。
2. 初始化工作内存：创建一个空的工作内存，包括age和balance等信息。
3. 执行规则：根据规则库和工作内存中的信息来执行规则。
4. 更新工作内存：根据规则的执行结果，更新工作内存中的信息。

具体的代码实现如下：

```python
# 加载规则库
rules = load_rules_from_file("rules.txt")

# 初始化工作内存
work_memory = {"age": 20, "balance": 1200}

# 执行规则
for rule in rules:
    if rule.match(work_memory):
        rule.execute(work_memory)

# 更新工作内存
can_buy = work_memory["can_buy"]
```
