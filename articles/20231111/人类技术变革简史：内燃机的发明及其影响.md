                 

# 1.背景介绍


在这个物质极度缺乏的时代，工程师们不得不寻求新的解决方案来帮助人类生存。直到上个世纪70年代末，发明了第一个可以自给自足，可运行于空间环境中的制动系统——内燃机。1967年，蒸汽机正式问世。而在十多年后的今天，内燃机已经成为人类最重要的能源之一，无人驾驶汽车、电动车和空中交通工具都依赖它。本文将从历史角度出发，全面探讨内燃机的发明、应用和影响。
# 2.核心概念与联系
## 2.1 概念定义
### 2.1.1 发动机结构
根据国际标准，汽油机发动机结构分为两个层次：排气装置、主动驱动装置。
#### 排气装置
由一堆排气管组成的装置负责处理气体流动、阻力、液体的抽丝、排尘，并释放排气管道上的气体，使排气管道保持清洁，消除空气泡沫。排气管道也是一种过滤器，只有经过过滤的气体才能进入主动驱动装置。
#### 主动驱动装置
主动驱动装置又称发动机舱（Engine Cylinder）或压缩机舱（Compressor Cylinder），由密封的外壳、发动机、转子、传动带、控制阀门、助力剂，以及其它零件组成。发动机通过固定在外壳下的转子旋转产生推力，传递给齿轮和风扇。压缩机舱中还包括压气机、减压阀、压缩机、排气阀、喷射室等设备，用来将热源、煤气、蒸汽和其他形式的燃料加工成高效的蒸汽或原油。
### 2.1.2 特性
#### 可控性
内燃机具有超强的可控性，能够持续工作很长时间。根据国际标准，一个家庭用品级的内燃机每天能够运转1000小时以上。这意味着它能够在需要时即刻起动，并在需要时立刻停止工作。
#### 重复利用率高
由于燃料循环利用率较高，相比于其它形式的燃料，内燃机更适合于长期供应性生产。特别是在高度人为浓缩的环境中，内燃机有较好的循环利用率。在国际标准中，一般认为燃烧一升油所需的时间大约为80秒左右。因此，如果要连续生产一年的油，只需按时喷射就可以实现1年的供应。而且由于主动驱动装置的节能特性，因此采用该种方式生产的燃料可以节省大量能源。
#### 节能效果优秀
由于主动驱动装置的节能特性，因此其可运行时间比重庆热水器、苏州冷却塔等传统的热交换器的总时间短很多。同时，压缩机排放的热量和二氧化碳也远远少于热交换器。因此，在一定程度上能够达到节能降低耗费的效果。
#### 耐火性
在不太大的范围内，内燃机虽然不能完全克服废气排放的问题，但由于其发动机和压缩机结构的特征，可以有效地防止火灾的发生。
#### 易维护性
在各方面条件允许的情况下，内燃机可以在短时间内就进行维修。不过，在国际标准中，关于内燃机的维修保养问题，仍存在一些疑问。目前，没有完全符合国际标准要求的内燃机的售后服务机制，因此还有待进一步研究。
#### 小型化设计
内燃机被设计为一种轻型化产品，这意味着它可以安装在所有大小、形状的汽车、飞机上，并且可以使用各种不同的燃料，如汽油、柴油、天然气等。
## 2.2 发明过程
1967年，蒸汽机问世。蒸汽机采用外壳金属材料和传动带，将液态燃料引入车体。在国际标准组织的指导下，美国空军的科罗拉多航空兵首次使用了蒸汽机。
### 2.2.1 蒸汽机机械构造
蒸汽机结构与普通汽缸差异很大。首先，蒸汽机的发动机采用气冷式发动机，即底部有气缸，在离心作用下生成推力；其次，蒸汽机有四个独立运行的马达，每一个马达既负责驱动引擎的前进，又负责通过双闸门把进料汽体送入到引擎的缸体内；再次，引擎是一个大约2.5英寸的铝片状的无烟囱的，可紧急提升的斜坡；最后，蒸汽机的座舱则是一种特殊的箱型座舱，舱体上贴有橡胶保护层。
### 2.2.2 蒸汽机燃料加工方法
蒸汽机的燃料加工是整个过程的关键环节。蒸汽机的加热模式为核聚变燃料（如低温燃料、高温燃料等）燃烧，核燃料的表面表面积较薄，易熔化，所以必须在加热器外部加盖以免熔融殆尽，然后逐渐输送至发动机内，使其内部结构达到一定温度，这样才能够启动引擎加速工作。对于辛苦的燃料工人来说，这种工作实在太辛苦了。因此，蒸汽机必须设计出一种精密的加工方法。根据蒸汽机加工工艺的不同，通常有以下三种加工方法：
- 方法一是锅炉加热法：在锅炉上加入蒸汽加热，放入稀释液和高原原油，由高原原油与蒸汽结合，产生蒸汽团。这种方法在提高蒸汽机效率方面有重要意义。它的基本工作流程如下：
  1. 熔炼：在锅炉中冷却液和原油，使其表面熔化。
  2. 提油：在蒸汽温度达到一定值之后，投放高原原油，使其凝固。
  3. 拌匀：经过一段时间的加热，低温燃料团就会被完全熔化，形成蒸汽团。
  4. 输送：在锅炉外侧的发动机内侧接入涡轮盘，使蒸汽团与引擎的间隙充满液体。
  5. 加速：蒸汽团注入引擎，引擎加速运行。
- 方法二是蒸馏法：在低温压力下加热，利用蒸汽与高压气液混合，形成蒸汽团。蒸馏法对提高蒸汽机效率的贡献主要在于可以通过增加蒸馏温度来提高蒸馏速度。
  - 在蒸汽机外部加入多个较大的钢缆、金属夹子等锻造材料，锻造出较厚、较薄的壳，将加热工艺加入其中。
  - 将锻造材料稍作切削，加入压力容器、高原原油、低压气体，并在相应的温度下加热。
  - 通过送至发动机外部后调整压力，使蒸馏出的蒸汽团在满足发动机需求的情况下，形成较厚的缸体。
  - 当发动机运行时，通过闸门自动提升蒸馏材料的压力，使蒸馏出的蒸汽团在引擎制动、润滑流动、减震作用下均匀分布于缸体各处。
  - 此方法除了提高蒸汽机效率，还避免了燃料燃烧带来的损失，而且在部分环境中也可以工作良好。
- 方法三是卷积燃烧法：采用专用的微波爆破机对燃料进行加工，在锅炉里先进行熔炼，然后向内注入微波炉，经过多次微波冲击，燃烧开，产生卷积液。当卷积液被输送至发动机内时，其混合物会与发动机各部件的连接件发生反应，引发发动机转速提升，这是卷积燃烧法的关键性技术。
