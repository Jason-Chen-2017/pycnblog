                 

# 1.背景介绍


## 数据治理是什么？
数据治理（Data Governance）是指组织在运用数据分析、挖掘、归纳、整理等技术获取价值之前对数据的收集、处理、存储、使用、共享、流动、传输、保护、授权管理的过程及其结果。它以数据为中心，旨在保证数据的真实性、准确性、完整性、可靠性、可用性及相关利益相关者的权利和义务的有效性，从而为业务决策提供科学、可信的依据。数据治理的目标包括数据质量（Data Quality）的维护，数据安全性（Data Security）的保障，数据价值的提升和数据资源的分配。
数据治理是一个综合性的活动，它涵盖了整个数据生命周期内的数据管理活动，包括数据采集、建设、整合、分析、加工、应用、存储、交付、销毁等多个方面。其中最重要的是如何做好数据收集、存储、处理、使用、共享以及流转等环节中的相关流程、规范、制度、工具和服务。
## 为什么要进行数据治理？
随着互联网信息技术的迅速发展，数字化进程的不断推进，海量数据被大量生产、产生、流通，它们不断成为新的商业机遇和金融资产，同时也对个人生活、工作和社会产生巨大的影响。因此，越来越多的人开始意识到，保障个人、企业和国家的经济、社会和文化权益对保障人类发展至关重要。同时，为了确保信息安全和隐私，保障数据的价值和完整性，更好地促进经济发展，政府部门也在不断增加对数据的监管力度。
数据治理对于保障个人、企业和国家的经济、社会和文化权益至关重要，但同时也是解决数据价值和完整性问题的有效途径之一。它可以增强信息系统的整体稳定性、数据安全性和数据价值的一致性，降低系统故障风险，防止腐败、非法活动和数据滥用。数据治理通过建立健全的数据管理机制、制定数据分类、使用规则、数据标签、数据共享协议和数据隐私保护法律法规，有效提升数据价值、保护个人隐私和数据安全。
## 数据治理的主要职责与目标
数据治理的主要职责有以下几点：

1. 支撑组织的长远发展目标：从组织的角度看，数据治理旨在实现数据价值的最大化，为组织创造价值并增强竞争能力；

2. 提升组织的信息技术水平：数据治理能够帮助组织构建信息技术平台，提升信息技术水平，降低成本；

3. 提高组织的工作效率：数据治icality能够提高组织的工作效率，降低生产效率损失，优化资源配置，提升工作质量；

4. 提升组织的创新能力：数据治理能提升组织的创新能力，促进组织内部的协作和创新，为业务决策提供科学、可信的依据；

5. 建立信息化程度评估体系：数据治理能够建立信息化程度评估体系，帮助组织评估自身的信息化程度，识别信息化过程中的重点领域，找出改善方向；

6. 促进组织内外沟通协调：数据治理能够促进组织内外沟通协调，开展有效沟通，搭建共赢的团队合作氛围。

数据治理的主要目标有以下几点：

1. 规范数据采集、存储、处理、使用、共享、流转等环节中的流程、规范、制度、工具和服务：数据治理通过制定相关政策法规、规范数据采集、存储、处理、使用、共享、流转等环节中各方的行为规范、程序、制度、工具和服务，为数据采集、存储、处理、使用、共享、流转提供合理、可控的服务环境；

2. 明确数据价值和数据安全的关系：数据治理通过明确数据价值和数据安全之间的关联和区别，帮助理解个人信息和其他敏感信息的价值，防范欺诈、非法活动、泄露等风险，保障个人数据和其他敏感数据安全；

3. 维护数据质量，确保数据价值最大化：数据治理需要持续关注和维护数据质量，确保数据价值最大化，为数据价值管理提供科学、可行的决策支持；

4. 营造信息技术环境，增强信息系统的整体稳定性：数据治理通过制定规范、制度、流程、工具、方法和服务，让信息技术人员充分参与，增强信息系统的整体稳定性，保持信息服务的连贯性和统一性；

5. 建立数据治理档案，保障数据价值的历史记录和完整性：数据治理需要建立数据治理档案，全面记录和保存数据价值的历史记录和完整性，促使管理层重视数据价值的历史积累，为未来的合规要求提供依据。
## 什么是数据策略？
数据策略（Data Strategy）是指一个组织或个人在特定的时间段内所主张的对市场、技术和产品的总体战略、策略及特定目标的决策集合。它由战略决策与执行决策构成，战略决策即所希望达到的目标和愿景，执行决策则指将战略决策转化为具体措施、计划、决策或政策。数据策略是管理科学研究领域的一个重要分支，旨在发掘人们的需求，理解客户的痛点，开发行业特定、时尚的解决方案，并且给予数据决策者设计一套完整的决策工具箱。
数据策略有助于组织成功地吸引、保留和留住有价值的、独特的、具有竞争力的客户。数据策略能够帮助组织在不同阶段制定针对性的策略，实现数据收益最大化。数据策略能够帮助组织决定采用何种技术方案，开发哪些数据驱动型产品或服务，引导公司致力于扩大收入来源。数据策略能够帮助组织设定目标并制定战略，为市场和销售策略制定指南针。数据策略对于每个行业都是非常重要的，无论是在IT、医疗、金融、房地产还是旅游等行业都需要制定数据策略。
## 数据治理与数据策略之间的关系
数据治理和数据策略之间存在密切的联系，因为数据策略是关于战略决策的，而数据治理则是一个综合性的管理决策过程，包括数据收集、处理、存储、使用、共享、流动、传输、保护、授权管理等环节。数据治理是一个为所有相关角色—包括管理层、人员、承包商、合作伙伴、供应商—提供服务的全过程，既涉及管理层的决策，又依赖于外部数据服务供应商和合作伙伴的贡献。数据治理的目的是实现“价值”最大化，为组织的长远发展、提升信息技术能力和营销绩效奠定坚实的基础。
