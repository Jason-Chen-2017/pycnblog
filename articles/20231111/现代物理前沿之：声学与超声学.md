                 

# 1.背景介绍


声学和超声学是近几年来发展最快的两门物理学科。它们的兴起是由于电磁波的发明与应用，以及人类对空气中高频杂波的关注。由于声音具有空间、时间和温度三种自然属性，因而声学和超声学都是广义的物理学科。而且由于它们的普及程度还不够，因而还有很长的一段路要走，我们还需要在更多领域探索更加复杂的物理性质。

20世纪初期，在美国的Edison实验室里，为了研究高频声源所产生的微弱辐射，科学家们发现了一种新的可辨识信号。这个信号可能来自于气体或氢气的泄露，也可能是由细小颗粒在空气中的自由扩散所形成的。这种新型信号被命名为“高频透镜效应”(High-frequency aperture effect)。他们把这个信号的来源归咎于雷达前面的光圈。随着时代的发展，人们越来越重视高频信息的处理。1975年，日本大学生佐野将日文翻译为英文发表了一篇文章，题目叫做“超声波大覚書”，意思就是“超声波的梦幻书”。他认为超声波是一种一种信息传播方式，在两千多年前就已经存在。

20世纪80年代末，随着医疗卫生和环保等领域的需要，超声技术得到了越来越多的人们的重视。同时，随着人类活动范围的扩展，新的需求也呼唤着传统的物理学知识和技术的升级。20世纪90年代，美国科学家艾登·阿普尔曼提出了著名的”异相干涉实验”，其目的是通过制造声音的微弱变化从而了解不同频率之间的耦合作用。这个实验的主要思想是通过反复冲击两个固定点，使得激发者和受激者之间形成复杂的混乱，从而让人们能够窥探到微观物理层面的各种复杂现象。随后，日本纳田修造推出了第一台超声望远镜。

2010年，武汉市民肖杰举办“火神庙论坛”，宣扬超声和声学相关的科技工作。当天，人们围绕着超声波的应用领域展开讨论。大家表示：希望通过技术手段降低疾病患病率，解决环境污染问题。2011年，全国各省都组织了超声波技术培训。

2016年，世界首例成功手术治愈恶性蚊子感染已成为超声疾病的一次革命性的突破。

2018年，日本“2018年度超声技术会议”在京召开，来自亚洲各国的领军人物，如韩国前总统金正恩、马来西亚前总统李克强等，共同主持了该会议。研讨会上，“超声波：真理还是幻觉”的主题激发了众多年轻人的思考，大家都在思考如何运用超声技术解决疾病、改善居住环境、保护珍贵的古迹等领域的问题。

但是，超声技术也面临着巨大的挑战。首先，作为“超越现实”的先锋技术，它带来的问题很多，比如安全隐患、诱导犯罪、遗漏病理条件、误诊等等。其次，超声技术本身的复杂性使得它的性能并不是永恒的，过分依赖它会导致患者误诊、出现各种各样的副作用。最后，超声技术有可能被用于造假、炒作、欺骗，成为许多国际、国内的禁忌。所以，更好的研究方法是建立在对其功能、原理和实际效果的深刻理解基础上的，这是超声技术作为一门独立的学科，才能真正解决一些根本性的问题。

为了给读者提供一个直观的认识，下面我们对声学和超声学进行比较。
# 2.核心概念与联系
## 情景一：水声

20世纪中期，物理学家赫兹·爱因斯坦探究声波的传播特性。他发现了一个奇妙的现象——“声线振动定律”（sound wave vibration law）。这一定律告诉我们，声波在空间中传播时，都会发生振动。也就是说，当一个声波从某个方向传播时，它一定会引起其他方向的声波振动。因此，声音的振动对我们人类的生活来说无疑是至关重要的。

早在19世纪50年代，孔雀翅缘由一个天才少年设计，将波长达到约1米的珠宝戴在嘴巴上。因而，后来被称为“第一张天才人像”的照片就留存下来了。这一切都是因为它产生的音乐中充满了无限可能。

如今，声音是一种非常重要的信号。我们的生活离不开声音，不管是在家里、在车站乘车，还是进行社交。所以，声学一直是研究的一个热点。


## 情景二：超声波

20世纪初期，科学家们发现一种新的信号——高频信号。这些信号来自于外界的热量、动力或者太阳光，由于它们的高频特性，使得它们很难被检测和记录。而这时，物理学家们把注意力转向一种新的信号——超声波。

超声波通常比声波短，只有几十毫秒，却可以穿透很薄的介质。它与声波一样，具有空间、时间、温度三个自然属性。其传播方式和声波不同，其传播过程是由电磁波驱动的。电磁波在探测器中会被作用在一个折射率不断变化的介质中，随着能量的吸收和放射，电场就会形成。这个电场再与另一个折射率不断变化的介质相碰撞，就会产生一种特殊的电流。通过这个电流，可以把传播途经的所有介质中的能量收集起来。

高频信号主要是由两种来源：气体泄露和气液混浴。气体泄露主要发生在靠近水平面的热源处，例如房间温度较高，以及大气中辐射带的传播。气体泄露导致的高频信号会发生聚集，形成噪声。气液混浴则是由于海水波浪的影响，导致的空间传播的高频信号。

通过这种高频信号，可以测量密度、速度、温度、剂量等高维度的信息。而此时的超声技术已经具备了应用的条件。

超声波技术的发展，促进了空间信息技术的革命。在人工智能、计算生物学、材料物理等领域都取得了突出的成果。由于它独特的高频特性，可以帮助我们进行精准地定位、识别、跟踪等任务。超声波技术的应用已经成为不可或缺的一项技术。

所以，超声波技术不仅是一个独立的学科，更是一个连接多个领域的交叉学科。它所涉及到的领域包括电子学、光学、力学、工程学、心理学等。在这些领域的协作下，超声技术正在创造新的知识和产品，比如医疗和护理领域的超声诊断、三维建模、体育运动训练领域的远距离战术、精密加工领域的光刻胶打印。这些技术不断刷新着人们对信息传输、生产和利用的理解。
