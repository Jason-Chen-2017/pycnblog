# 2.核心概念与联系
## 深度学习简介
深度学习是机器学习的一个分支，它利用多层神经网络对数据进行处理，使计算机能够从原始数据中学习出有效的特征表示，进而完成任务或实现预测。深度学习可以理解为一个由多个层组成的巨型神经网络，其中每一层都是一个简单的神经元网络，通过反向传播算法训练这些网络来优化模型参数，最终达到学习数据的规律性，实现数据的高效分类、预测等功能。
深度学习框架主要包括以下几类：
1. 卷积神经网络CNN（Convolutional Neural Network）: 是最常用的深度学习框架之一，通过对输入图片中的像素点进行局部化卷积操作，提取图像特征。该框架在图像分类、目标检测、语义分割等方面有着重要作用。
2. 激活函数AF（Activation Function）: 是指每个神经元单元的输出值计算时使用的非线性函数，如Sigmoid、tanh、ReLU等。不同的激活函数会影响神经网络的拟合能力，使得神经网络具有更强大的表达力。
3. 梯度下降法GD（Gradient Descent）: 用于更新神经网络权重参数，使其能够更好地拟合输入样本的标签。梯度下降法采用误差反向传播算法，利用梯度下降的方式不断调整权重参数，最终使得模型参数逼近全局最优解。
4. 循环神经网络RNN（Recurrent Neural Network）：是一种特殊的神经网络结构，能够解决序列数据的建模问题。RNN能够记录并利用先前时间步的信息，能够捕捉序列数据中长期依赖关系。
5. 生成式 adversarial networks GANs（Generative Adversarial Networks）：是深度学习的一个新兴研究领域，其利用生成器网络生成假样本，让判别器网络判断真伪，从而提升生成模型的能力。
6. 自编码器 AE（Autoencoder）：是一个无监督的深度学习模型，其学习数据的内部结构，用于数据压缩、去噪和异常检测等方面。
7. 递归神经网络 RNN （Recursive Neural Network）：是另一种常用深度学习模型，其具有记忆性，能够解决序列数据建模的问题。
## 深度学习框架使用流程
### 1. 数据准备阶段
首先需要准备好训练集、测试集、验证集以及数据转换后的格式（如图片转化为矩阵）。然后加载数据，按照比例划分训练集、测试集、验证集，并进行预处理。
### 2. 模型构建阶段
根据深度学习框架的不同，确定使用的模型类型，如常见的卷积神经网络CNN、循环神经网络RNN、生成式adversarial networks GANs、自编码器AE、递归神经网络RNN。
### 3. 模型编译阶段
编译模型，指定优化器、损失函数、评估指标等。
### 4. 模型训练阶段
在编译好的模型上，进行模型的训练，同时设定训练次数、批次大小、学习率等参数。
### 5. 模型保存及加载阶段
在模型训练结束后，保存模型的参数值，用于预测或者微调模型。此外，也可以直接加载已经训练好的模型，用于预测或者微调。
