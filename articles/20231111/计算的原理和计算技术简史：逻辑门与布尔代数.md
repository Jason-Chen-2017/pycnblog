                 

# 1.背景介绍


计算机由二进制表示的数字组成，每个数字被称作“比特”(bit)。早期计算机只能处理整数。随着科技的进步，人们发现可以用小数表示，因此出现了浮点数、复数等数字形式。这样一来，计算机可以进行运算。
为了能够对任意大小的数字进行有效的处理，计算机还必须具备“算术运算”能力。而最初的人类解决这个问题的方法就是手工计数。但是数学家们发现可以用数字来表示真值表中的基本元素——变量与值的关系。从而得出了一系列的逻辑门、逻辑表达式以及逻辑运算符。这些逻辑门与逻辑表达式可以帮助人们对数字进行有效的计算。因此，人们创造了新的计算方法，即“计算机代数”，基于逻辑代数中使用的逻辑门及其运算规则来制造计算机。


## 2.核心概念与联系
### 2.1 逻辑门
“逻辑门”是指一个或多个输入信号经过一定逻辑运算后产生输出信号。逻辑门具有明确的输入、输出和功能，并可用于构造复杂的电路。以下是常用的逻辑门类型:
- AND门(AND gate)
- OR门(OR gate)
- NOT门(NOT gate)
- NAND门(NAND gate)
- NOR门(NOR gate)
- XOR门(XOR gate)
- XNOR门(XNOR gate)
- Buffer(Buffer)


如图所示，AND门将两个输入信号与“与”操作，输出1；OR门将两个输入信号与“或”操作，输出1；NOT门将输入信号取反输出；NAND门将两个输入信号与“非且”操作，输出1；NOR门将两个输入信号与“非或”操作，输出1；XOR门将两个输入信号与“异或”操作，输出0或1；XNOR门将两个输入信号与“同或”操作（即两者都是1或者都不是1），输出1。

“组合逻辑电路”是一种通过串联各种逻辑门实现不同功能的电路。如下图所示：



如图所示，一个典型的组合逻辑电路包括多个输入端、输出端以及逻辑门组成。当输入信号与相应的逻辑门相连接时，就能形成电路。组合逻辑电路的输入、输出和逻辑门可以简单地分为三种类型：
- 布尔输入：只允许0或1两种信号输入。
- 数字输入：可以输入任意实数。
- 数字输出：以某种方式生成一个实数作为输出。

### 2.3 时序逻辑电路
“时序逻辑电路”是指采用同步时钟控制信号来驱动各个逻辑门工作的电路。时序逻辑电路的关键是保持信号稳定，在规定的时间周期内完成其工作。如下图所示：


如上图所示，一个典型的时序逻辑电路包括多个输入端、输出端、逻辑门组成以及同步时钟信号。输入信号通过同步时钟信号，触发逻辑门的工作，逻辑门根据输入信号的变化，决定输出信号的状态。时序逻辑电路的输入、输出和逻辑门可以简单地分为三种类型：
- 异步输入：无需时钟信号驱动。
- 同步输入：需要时钟信号驱动。
- 时序输出：按一定时间间隔改变状态，在固定时间范围内生成连续的信号流。
