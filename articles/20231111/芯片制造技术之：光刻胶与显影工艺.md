                 

# 1.背景介绍


光刻胶(gel electrophoresis)和显影(fluorescence luminescence)技术是所有电子设备制造的基本方法。基于二维平面上的光刻胶晶体结构，使得电子可以保持固定，在不破坏材料的情况下表现出不同颜色或其它任何光谱特征。在电子学、物理、化学、生物学等多个学科中都有研究，其中又以光刻胶与显影技术最为重要。

显影法与光刻胶法之间的界限越来越模糊，甚至出现了“混合”的说法。在实际应用中，两种技术的结合应用也逐渐成为主流，这是因为其特点各有千秋。光刻胶法用于精确的成型和结构分析，而显影法则可以用来进行结构化的控制和反应性测试。同时，光刻胶法还能够提供更高的处理能力和更长的寿命。但是，显影法的实时性较差，而光刻胶法的制品质量及效率往往要高于显影法。另一方面，光刻胶的制作难度比较低，且精度可控，适合制作原件及样品。所以，为了提升产品的精度和产能，大多数公司都将两种技术结合起来用。

光刻胶在电子学领域有着广泛的应用，应用到各种电子产品上如LED、IC、印刷电路板、智能手机等。显影是由光刻胶晶体中的特定电子被激发而产生的光波，它的主要作用有两类：一是表现结构的特性，如铁电镜；二是辅助诊断和改善产品质量。显影技术的研究已经深入到了分子层面的，主要包括结构工程、化学工程、生物工程、材料工程、机械工程等。如分子束锁定的实现、显影管道的设计、显影模组的制造等都是其研究方向。在中国，由于地理位置的原因，显影技术研究力度远不及西方，尤其是德国、日本等亚洲国家。因此，中国的光刻胶工艺需要更多的基础知识、技术能力、人才支撑才能走向更高的水平。

# 2.核心概念与联系
## 2.1 核心概念
光刻胶(Gel Electrophoresis,GE)：简称GE，又称沉积电镜法，是一种在微孔、颗粒状的晶体上施加离心力并通过排列这些离心面与其他离心面之间形成的相互作用，将无序的液体聚集到有序的球状孔中，从而产生有组织的、均匀分布的庞大体系。光刻胶制备通常采用微透明的G-C乙烯(AgCl)溴液，一般采用2%~10%溴酸钠(NaOH)溶润，并加热至97~100摄氏度，使得溴培过程完全完成后再进行后续的光刻胶步骤。光刻胶的优点是其高分子密度，具有良好的分散性、形态清晰、吸收性、耐腐蚀性、免疫及抗菌性能，并能在实验室环境下高速制备。

显影(Fluorescence Luminescence,FL)：又称荧光增强法，即在液体中加入化学激素，对其不同色光的吸收情况进行测定，得到其荧光信号，然后根据这一信号，计算其对应的光功率值，再经过某种放大装置反映出来。通过显影法可观察物质的结构性质、电性状态、功能性质、反应性质等，已成为目前电子设备制造过程的关键技术之一。

显影法与光刻胶法之间的界限越来越模糊，其实它们的区别很简单，光刻胶用硅钢离心，液化成分对其进行加熟、离子交换，最后形成晶体。晶体堆积后，电子膨胀变浓，而电子也形成稳定的电子气囊。晶体与电子共同组成电子气囊，然后在电子气囊中进行高温离心电镀，电子经过电离过程分解成为光子。晶体上的电子受到聚集和扩散作用，形成晶体带隙，晶体带隙中的电子由于大小不一样，形成不同颜色的光子，最后像荧光一样混杂在液体表面。

显影法与光刻胶法之间的界限越来越模糊，很多文章将光刻胶和显影的方法混淆。其实，它们只是利用了不同的物理原理而已，但为了取得更好的性能和效益，它们之间的结合应用是不可或缺的。在应用中，光刻胶法用于精确的成型和结构分析，而显影法则可以用来进行结构化的控制和反应性测试。同时，光刻胶法还能够提供更高的处理能力和更长的寿命。但显影法的实时性较差，且制品质量及效率往往要高于光刻胶法。所以，为了提升产品的精度和产能，大多数公司都将两种技术结合起来用。

## 2.2 联系：GE与FL的联系
光刻胶、显影的本质上都是将无序、混沌的原子液体结晶成有序、密集的晶体。GE和FL的关系，既可以说GE就是FL的前身，也可以说GE是FL的衍生产品。由于前者应用的更广泛，所以更易于理解。这里以GE+FL的步骤作为举例，阐述它们之间的联系。

1.步骤一：沉积成型
首先，利用DE-BEAD等技术将电子液体沉积成形状类似晶格的团簇。

2.步骤二：离心增殖
对晶体团簇进行离心增殖，将其中的微孔填充为电极，制备有机物、细胞。

3.步骤三：电镀、建立带隙
在晶体表面电极之间建立层层的电场，促使电子膨胀。随后在晶体上电镀，将电子与离子混合，形成晶体带隙。

4.步骤四：光刻
在晶体带隙中增强色素和吡啶酮，使得晶体的特定电子被激发，形成与晶体中原有的电子数量相当的光子。

5.步骤五：照射
照射不同色素的晶体区域，获得不同光强的荧光信号。

6.步骤六：放大
将荧光信号转化为电压信号，并通过放大装置反映出来。
