                 

# 1.背景介绍


## 1.1 为什么参加天使轮？
创业是一个漫长且艰难的过程，越是在创业初期越需要勇气、理性、毅力。为了在这个过程中不断成长、壮大、找到自己的定位，很多人都会选择加入一些名校或顶级高校的网络教育培训计划。这些教育项目主要帮助学生提升自身的职场技能，包括沟通能力、分析能力、执行力等软实力。但是也正如同所有的投资一样，这样的训练机构只是提供了一种形式上的优惠券而已。真正的收益来源还是要靠个人的努力与付出。因此，现在更多的人加入一些专注于自我发展的创业项目中，如个人微信公众号，公开课，或者基于个人兴趣的线上活动，这些活动一般都提供各种各样的折扣优惠，促进个人的个人开发能力提升。其中比较有名的就是京东零售的“高佣王”活动，相信很多朋友都听说过这个活动吧。
## 1.2 “天使投资”的定义及其分类
所谓“天使投资”，最早起源于1987年马歇尔·鲍威尔博士在斯坦福大学的一次演讲。在他的演讲中，鲍威尔认为“成功的企业家都是由一个人决定的”。因此，“天使投资”的目的是寻找并招聘具有独创能力的个人，帮助其打造独特的产品或服务。随着互联网的发展，随着更多创业者逐渐走向独立，硅谷创业者也纷纷加入到“天使投资”的行列。
“天使投资”的分类也很丰富。按照当时的定义，“天使投资”分为以下几种类型：
### A类：具有创新性的创业者，将个人打造成为独特产品或服务的顶尖企业家。他们的独创性可以从产品、服务、团队等多个方面体现出来。像Facebook、LinkedIn、Uber等都是属于这一类的典型代表。
### B类：具有高远景、长期规划的创业者，通过长期合作伙伴关系积累资源、扩大影响力。这类投资人通常都具有很强的商业嗅觉，了解市场趋势，同时也对客户服务意识很强。像Stripe、Eventbrite、Zapier、Brave Browser等都是属于这一类的典型代表。
### C类：既不是A类也不是B类，一般更偏向于技术性的创业者，将理论知识转化为实际产品、服务，或提升自己在行业内的知名度。这类投资人更看重在市场中获得的利润，其收益来源可能是线下、线上、人脉关系、投资机会、股权激励等多种形式。像Airbnb、Buffer、Lyft、Slack等都是属于这一类的典型代表。
总结一下，“天使投资”可以帮助企业家在短时间内建立起独特的品牌、产品或服务，建立初期可能需要较大的资金投入，但随着时间的推移其收益也随之增长。而且这种类型的投资往往不是立刻就能带来巨额回报的，需要持续不断的不断投资才能带来足够的回报。
