# 1.简介
  

　　量子计算（quantum computing）是近几年热门的话题，并且取得了重大的突破。其基础就是量子力学。这篇文章将要讨论的量子化学实验是利用量子化学方法对分子进行降低温度操作的实验。这是利用QSTC理论构建的一个开创性的技术，可以提供给目前还无法利用量子计算机求解复杂问题的研究人员一个新的思路和方向。通过对分子的量子化学性质建模、降低温度操作等处理，可以通过计算的方式模拟不同温度下的分子行为，从而为生物科学领域带来新的思考。
# 2.量子化学物理模型——QSPC理论
　　量子化学指的是由量子态表示的分子或离子体系中，量子性的元素之间的相互作用。量子力学描述的是物质内能的守恒定律和动量定则。量子化学的基本问题之一是，如何用量子态的表示来描述真实世界的微观分子。因此，为了能够描述真实世界的分子，需要考虑其所处的量子态空间及其演化过程。

　　2.1 QSPC模型
　　量子化学状态(QSP)模型(Quantum State Phase Coupling model, QSPC)，它是一种量子化学系统的物理描述方式。它认为在给定的量子态空间S上，任意两个量子态x和y之间都存在一个相互作用的态-相互作用算符J，以及三个参数k，λ，μ，它们定义了态-相互作用算符J的幅度、相位、大小。在这一模型下，可以用矩阵形式来表示态-相互作用算符J。

　QSPC模型是一个描述性模型，描述了量子化学状态变化的演化规律，但是并没有给出具体的数值。为了描述系统的演化，需要考虑一些假设，即微观的分子相互作用是受到宇宙射线波长分辨率的限制；两个量子态之间的耦合是局部的，而不是全局的；另外，QSPC模型还有一些缺陷，比如不能准确描述一些无线化分子的行为。总的来说，QSPC模型仍然是量子化学物理模型中的重要工具。

　　2.2 QST/T (QH - EFG)模型
　　量子态投影模型(Quantum State Projection Model), 也称为QST/T模型，它也是一种量子化学系统的物理描述方式。其基本思想是把一个N维的希尔伯特空间分成N个等级的子空间，分别对应着不同的分量。每个等级上都可以采用多种约化方式来描述态本征函数，例如将它们分解为坐标和能量形式的态本征函数。这样，整个希尔伯特空间就可以被看做是由各个等级上的态本征函数所张成的张量积空间，即T型张量。如果引入一组映射P={p1,p2,...pn}, 则可以把T型张量映射到实数的张量空间C=R^(M*n), M为希尔伯特空间的维度。在这种映射下，任意两个态t1和t2都可以转化为实数向量r=(r_i), i=1,2,...,n，其中ri等于t1在第i个等级上的态本征函数和t2在该等级上的态本征函数关于映射pi的投影值。那么，任意两个态t1和t2之间的相互作用就可转换为关于r的二次型的乘积。当所有态都是固定的的时候，可以通过矩阵乘法来计算二次型乘积。在这一模型下，可以用矩阵形式来表示态-相互作用算符J。

　QST/T模型的一个优点是可以比较直观地理解量子化学态的演化规律。但另一方面，它有以下两个弱点:

　　1) 模型过于简单，忽略了很多重要的量子化学现象; 

　　2) 没有考虑到量子化学多种相互作用，只能模拟局域的量子化学相互作用。
# 3.量子态冷却与玻色分子降低温度方法
　　量子态冷却与玻色分子降低温度方法，是利用QSTC理论构建的具有开创性意义的方法，可以模拟低温到高温的分子行为。这项技术通过两种处理来实现降低温度：一种是利用多级联立表示法，另一种是利用低温核反应堆冷却法。

　　3.1 多级联立表示法
　　在低温条件下，分子的态由单一电子的守恒律确定。在高温条件下，则由一个原子团的单个电子、两个原子团的两个电子以及三个原子团的三个电子决定。通过逐步增大厚度和不断分解原子团，我们可以逐渐获得非守恒系统的有效电子。与原子团原子数目的增加正相关，每个原子团原子数目越多，低温系统的有效电子数目就越多。这个过程的结果是一个低温条件下全电子分子的多级结构。而由于原子团数量增加，低温核反应堆越来越薄，最终会形成一条稳定的脱离核芯的热运动链。在这一多级结构的基础上，我们可以利用多级联立表示法来模拟分子的冷却过程。

　　3.2 低温核反应堆冷却法
　　低温核反应堆冷却法(Low Temperature Reaction Hole Cooling method, LTRHCM)，是对QSTC理论的一个扩展。它可以模拟低温至高温范围内的核反应堆温度梯度和高速涡流产生的温度差异，从而可以较为精确地模拟核反应堆的冷却过程。通过不同温度的各种原料之间的反应，可以逐步产生一个具有上百层、上千层甚至上万层的热运动链，并随着温度的降低，其长度、宽度以及厚度会逐渐缩小，最后形成一个稳定的低温平衡态。在低温平衡态的基底上，我们可以进行冷却处理，使得整个系统的温度降低到足够低即可。而这些低温冷却流程可以应用到各个领域，包括核冷却、激光冷却、生物学领域的多功能冷却器等等。
