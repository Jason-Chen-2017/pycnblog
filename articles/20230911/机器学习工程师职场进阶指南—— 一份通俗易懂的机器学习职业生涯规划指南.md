
作者：禅与计算机程序设计艺术                    

# 1.简介
  

机器学习工程师（ML Engineer）是一个高端技术职位，由数据科学家、统计学家和工程师组成。其工作重点是构建、训练及优化机器学习模型，利用计算机视觉、自然语言处理、推荐系统等技术解决日益增长的业务需求。在这方面，他们能够处理海量的数据，并利用强大的计算能力进行高性能的分析。
此外，ML Engineers还需要掌握一定的编程语言、软件工具链、数据库系统等技能，因此在ML职业道路上更有诸多挑战。本文将帮助读者理解机器学习工程师的职业目标、需求、技能要求，以及不同阶段所需的个人职业发展路径。阅读本文档后，读者可以清晰地认识到如何在现实中取得成功，实现自己的职业梦想。
# 2.概述
作为机器学习工程师，你可以从事以下任一领域：

1. 数据科学：数据科学家负责管理和分析公司内或外部的数据，包括原始数据、结构化数据、非结构化数据以及各种类型的数据。他们通过抽象出有意义的信息，以可视化的方式呈现给团队成员，以达到更好的理解和决策的目的。
2. 算法工程师：算法工程师通常承担一些复杂的数学或计算机科学问题，比如优化、统计建模、信息检索、图像识别等。他们必须具有扎实的数学基础，同时也要善于应用机器学习的原理和方法。
3. 软件工程师：软件工程师主要负责开发机器学习模型的内部组件，如算法、数据预处理、特征提取等。他们必须熟悉常用的编程语言，如Python、Java、C++，并了解常用机器学习框架和库，如TensorFlow、PyTorch、Scikit-learn等。
4. 系统工程师：系统工程师负责构建、运行、维护整个机器学习平台的各个组件。他们必须理解机器学习的整个生命周期，包括数据收集、数据预处理、特征工程、模型训练、模型评估、线上运维等环节，并能与团队成员合作搭建整体的系统架构。
5. 管培生：管培生职责是在企业中带领机器学习工程师完成从数据分析到项目实施的全流程，包括模型选择、模型训练、超参数调优、部署到生产环境等。他们需要跟踪项目的进度、总结经验教训、改进产品质量、管理人力资源，以及创新并推广新的技术。

以上只是机器学习工程师可能参与的不同领域。根据个人喜好，你可以从多个方向中选取自己感兴趣的某个领域加入到机器学习工程师的队伍中。在这个过程中，你可能会遇到各种挑战，如时间压力大、软 skills 缺失、技术栈不匹配等。为了克服这些困难，下面我们逐一来看一下这些挑战。
# 3.时间压力大
首先，时间是一件很宝贵的资源。工作时间越长，你的职业生涯规划就越容易落空。对于一名有着丰富经验的机器学习工程师来说，每天要花费至少8小时的时间，才能保证工作效率，而其中的大部分时间都耗在沟通协调上。这就要求工程师必须投入大量的时间精力来减少沟通上的差异。所以，在决定加入机器学习工程师的行列之前，建议先充分准备自己的简历，做好技术面试、面向应届生的自我介绍等工作。这样可以降低招聘方对工程师的期望，避免出现在职位描述里面的“具有超强的分析能力”等词。
# 4.软 skills 缺失
第二，软 skills 是指与职位相关的技能、知识、能力。这些软 skills 在工作中起着重要作用，但却往往被忽略或者被低估。Soft skills 可以使一个工程师更具吸引力、更容易被雇佣。举例来说，工程师往往会喜欢与人打交道，但是如果没有适当的沟通能力，就无法做到全身心投入到工作中。同样，机器学习工程师也需要擅长沟通，可以说是至关重要的软 skills 。
# 5.技术栈不匹配
第三，工程师们需要在不同的技术栈之间进行切换。不同技术栈的要求和技能点可能有很大差别。举例来说，要求计算机视觉工程师掌握 Tensorflow 和 Keras 的 API ，而要求 NLP 工程师则需要掌握 PyTorch 的 API 。这些技术栈之间的切换让工程师更加擅长一种技术，但同时也增加了他们的挑战。
# 6.误判风险
第四，职业生涯中，错过良机也许是最大的风险。一旦确定进入机器学习工程师的行列，工程师们就会面临巨大的挑战——没有固定的工作模式和薪水，每天都有巨大的工作压力。当出现失误时，工程师们往往会陷入混乱甚至犯罪行为。所以，在决定加入机器学习工程师的行列之前，一定要认真思考自己的情况，以免陷入危险境地。
# 7.展望未来
最后，工程师们正在走向成长期。除了掌握机器学习领域的知识之外，工程师还需要进一步锻炼自己。通过不断学习，工程师们可以迅速适应和学习新的技术，并掌握快速有效的方法来提升工作效率。而在新的挑战中，工程师们还需要保持冷静、理性，不断寻找新鲜血液，才有可能走向更好的成功。
综上所述，机器学习工程师的职业生涯规划其实非常复杂。虽然每个人的经历、阅历都不同，但很多经验值得借鉴。在成为一名优秀的机器学习工程师之前，建议首先充分准备自己的简历，做好技术面试、面向应届生的自我介绍等工作，并集中精力攻克几个难题，确保职业生涯顺利。
