# 1.简介
  

搜索引擎优化（SEO）是指通过对网站内容的技术分析、关键词排名及页面权重的提升，使其在web搜索结果中的排名靠前，从而达到网站流量的最大化。这项工作由搜索引擎软件工程师完成。本文将介绍一些个人认为最重要的13个SEO技巧，帮助大家快速入门并掌握SEO工作。

2.SEO是什么？
SEO，即Search Engine Optimization，中文意思为“搜索引擎优化”，是一种根据网络受众需求进行网站内容制作、组织和呈现，以改善用户体验和提高竞争力的方式，通过互联网对外输出营销信息的行为。它的目标就是让更多的人找到所需的信息、获得所需的服务或解决相关问题。简单的说，SEO就是让你站上SEO榜单。所以，如果你想让自己的网站在搜索引擎上排名更靠前，就需要刻苦学习并实践SEO。

3.SEO的优势
SEO具有以下几个优点：
- 搜索引擎更容易找到您的内容；
- 更多的搜索者会访问您的网站，提高了网站的流量；
- 用户更喜欢使用您的网站，从而增加了网站的信任度；
- 您的网站可能被邀请加入到其他网站的目录中，带来更多的收益。

4.SEO的主要任务
了解SEO主要任务：
SEO的主要任务包括：网页内容策划、关键字规划、网页编码、网站链接建设、搜索引擎设置、收录优化、公共关系管理等。其中，关键词规划和网页编码属于基础性工作，其余均可以有效提升SEO效果。以下是该博客介绍的所有SEO技巧。

5.1.标题标签
SEO第一条建议是要用恰当的标题标签来呈现网页的主题和含义。好的标题标签应该能够反映网页的内容，同时还应当简短扼要，便于搜索引擎收录。下面是使用7种标题标签的案例：

1) H1标题 - 在网页的开头使用，并且重要程度应当是最高的，比如页面的名字、产品名称等。
```html
<h1>你的网站主页</h1>
```
2) H2标题 - 可以用来表示网页中的重要组成部分，如页面的子标题、特定分类页面等。
```html
<h2>关于我们</h2>
```

3) H3标题 - 用于显示网页内的小节标题。
```html
<h3>联系方式</h3>
```

4) H4标题 - 使用较少，一般不推荐使用。

5) H5标题 - 使用较少，一般不推荐使用。

6) H6标题 - 使用较少，一般不推荐使用。

7) 无序列表标题 - 如果要给无序列表加标题，可以把标题放在第一个列表项前面。如下所示：

```html
<ul>
    <li><strong>故事背景：</strong>...</li>
    <li><strong>展开过程：</strong>...</li>
    <li><strong>展望未来：</strong>...</li>
</ul>
```


注：其实H1到H6标题都可以应用在网页的任何地方，不过H1标题只应出现一次，H2标题到H6标题则可以在同一级标题内继续使用。

6.2.URL优化
SEO的第二个建议是要确保网站的URL清晰易懂，准确地反映网页的内容。好的URL设计能够增加网站的点击率、排名效率，也会影响SEO收录。下面是使用七个元素来优化URL的案例：

1) 中文URL
网站的中文域名可以让网民更直观地了解到网址的含义，因此非常值得注意。例如，我的网站的中文域名是cn.guangxu.me。

2) 不要使用动词
使用动词来命名文件和目录名称没有明显的意义，可能会造成用户困惑。例如，应该使用“about”来代替“introduction”、“services”来代替“products”。这样做不会妨碍搜索引擎收录，但会降低用户的兴趣。

3) 添加相关性描述
添加相关性描述可以为搜索引擎提供更丰富的信息，可以帮助网民更快地理解网页的内容，从而提高搜索引擎的排名。例如，“公司介绍”比“介绍”更容易被搜索引擎理解。

4) 避免使用数字区段
URL中不要出现连续的数字，尤其是在URL中作为参数传递的情况下，例如“http://www.example.com/news/2019/08”。这样会降低搜索引擎的抓取效率。

5) 清除重复的内容
在URL中不要出现相同的内容，例如不要出现两个相似的关于某个产品的信息页面。这样可以减少重复内容的分散效应，提高搜索引擎的抓取效率。

6) 用相关的英文单词拼写
URL可以使用相关的英文单词拼写，例如，可以使用“service”而不是“services”来代替“产品”，可以让网民更容易记住网站地址。

7) 添加查询字符串
添加查询字符串可以向搜索引擎传递额外的参数，有助于网页的优化和排名。例如，“http://www.example.com?utm_source=google&utm_medium=cpc&utm_campaign=new"。

注：URL不能太长，否则可能会影响搜索引擎的收录速度，甚至无法正常显示。另外，不同的搜索引擎对URL长度的限制也不同，因此不能盲目缩短URL。


# 7.元数据
SEO的第三个建议是提供足够充分的元数据，对网页进行分类、描述、标记。通过提供好的元数据，可以帮助网页被检索到，提高网站的排名。下面是设置网页元数据的几种方法：

1) 设置网页标题和描述
设置网页标题和描述能够帮助网页被搜索引擎检索到，同时也可给读者提供一个更直观的概览。例如，我的博客的首页标题是“郭旭伟的博客 | 郭旭伟的专栏 | 郭旭伟的随笔”，描述是“郭旭伟，一名计算机科学与技术专业学生，博客专栏记录了自己的学习经历、心路历程以及教学心得。”

2) 为网页添加描述信息
添加描述信息可以帮助网页被搜索引擎展示，也能加强网站的内容传达力。例如，在每一篇博文的开头都添加作者、发布时间、阅读次数等信息，能够帮助读者更好地了解这篇博文的内容。

3) 使用标签、类别和关键词
为网页添加标签、类别和关键词，能够帮助网页更好地被搜索引擎收录，并提高网站的排名。标签和类别可以帮助读者发现相关的网页，关键词可以提高网站的关键词排名。例如，我写过两篇关于SEO的博文，它们都被设置了“seo”、“seo技巧”、“how to improve seo”三个标签。

# 8.内容建设
SEO的第四个建议是要经常更新、编辑网站内容，提高网站质量。好的内容建设可以为搜索引擎提供更准确的网页内容，增强网站的知名度和可信度。下面是一些内容建设的建议：

1) 选择精彩的内容
选择精彩的内容对于SEO来说至关重要，它可以吸引网民、留住老访客、促进销售转化率，并帮助网站保持竞争力。因此，务必密切关注热点事件、新闻动态和行业趋势，充分利用媒体资源，创造独特的内容。

2) 提供有价值的内容
提供有价值的，而不是无用的内容。如果内容毫无价值，或者已经很久没更新了，那搜索引擎很可能不会收录。因此，需要选择适合网站定位和用户群体的内容，充分利用内容平台的功能，开发创作能力。

3) 选择突出的图片
选择突出的图片，可以增强网页的吸引力。好的视觉效果可以突出图片中的内容，令读者产生欣赏之情。另一方面，正确使用图片也能提升网站的品牌价值。

4) 创建个人品牌
创建个人品牌，可以让读者更容易被搜索引擎识别，促进销售转化率。因此，需要为自己打造一个看起来独特、具有自信的形象，展示出诚实、冷静、负责任的态度。

# 9.链接建设
SEO的第五个建议是建立健康的网站连接，提高网站的整体效率。好的链接建设可以帮助网站排名更靠前，提高网站的营收率。下面是一些链接建设建议：

1) 检查死链
检查死链可以帮助网站建立良好的链接结构，帮助网站建立外链互动，提升网站的流量和点击率。不过，如果死链数量过多，可能意味着网站的质量下降，搜索引擎将会删除该网站的索引。因此，需要提醒自己定期查看网站的外链情况。

2) 使用nofollow属性
使用nofollow属性可以告诉搜索引擎不要追踪该外链的网页。nofollow属性可以防止网页上的链接被搜索引擎收录，但又能保留网站的正常流量。

3) 把内容和链接放在一起
把内容和链接放在一起，可以让读者在网站上获得更多信息，并快速理解链接的含义。因此，一定要把信息和导航放在一起，在链接的后面加上相关描述文字。

4) 使用友好的链接文本
使用友好的链接文本可以帮助搜索引擎理解链接的目的，提高外链的点击率。不过，良好的链接文本也应该遵循SEO规范，避免使用“click here”、“go to”等内容。

5) 让网页内容脱离上下文
让网页内容脱离上下文，可以提升网站的排名效率。例如，如果网站的所有页面都只有一条新闻，而新闻正文之间存在不相关的内容，搜索引擎将无法正确收录。因此，不要让内容和上下文隔断。

# 10.外部推广
SEO的第六个建议是通过外部推广，吸引更多的用户。好的外部推广能够帮助网站提升网站的流量和排名，获取更多的收益。下面是一些外部推广的建议：

1) 维护良好的外部链接库
维护良好的外部链接库，可以帮助网站持续推送高质量的文章和内容。因此，需要积极筛选外部推广的优质资源，并确保这些资源都是经过审核的。

2) 选择可信度高的推广渠道
选择可信度高的推广渠道，可以帮助网站获得更大的曝光率，增加网站的流量。比如，Facebook、Twitter、YouTube、Blogger、Pinterest、LinkedIn等社交媒体都可以有效推广网站。

3) 使用社交分享按钮
使用社交分享按钮可以帮助读者分享网站内容，促进网站的流量。但是，不能滥用社交分享功能，过度分享会降低网站的知名度和点击率。

4) 使用Google AdWords、百度推广、微信公众号推广等工具
使用AdWords、Baidu、微信等工具可以帮助网站实现更精准的外部推广，增加网站的收入。但是，请注意用法规范，不要贪图便宜。

# 11.内部链接建设
SEO的第七个建议是通过内部链接建设，扩大网站的边界和影响力。好的内部链接建设能够为网站创造更多的机会，鼓励用户参与网站互动，提升网站的整体活力。下面是一些内部链接建设的建议：

1) 使用分类目录和层次结构
使用分类目录和层次结构，可以让网站内容更加容易被用户发现。因此，应该为网站的不同内容区域建立独立的分类系统，并适时更新和维护。

2) 为所有内容页添加隐私声明
为所有内容页添加隐私声明，可以帮助网站用户更好地保护自己的数据安全。而且，隐私声明可以帮助读者更好地了解网站的政策法律约束。

3) 实施反垃圾邮件策略
实施反垃圾邮件策略，可以帮助网站过滤垃圾邮件，减少垃圾邮件对网站的影响。但是，请注意不要滥用反垃圾邮件机制，反垃圾邮件机制本身也是一门学问。

4) 使用RSS订阅功能
使用RSS订阅功能，可以及时获取最新的网站内容，并实时通知读者。但是，请注意订阅频率的控制，防止订阅者的反感。

# 12.投放关键词
SEO的第八个建议是投放适合的关键词，通过关键词的覆盖率和排序，来提升网站的排名。下面是一些关键词投放的建议：

1) 使用短语和一句话总结关键词
使用短语和一句话总结关键词，可以有效提升关键词的点击率。例如，“网站优化，谷歌搜索引擎排名”比“网站优化+谷歌搜索引擎排名”更能抓住用户的 attention。

2) 将网站描述和关键词放在一起
将网站描述和关键词放在一起，可以帮助读者更好地理解搜索结果，并决定是否购买商品。因此，描述和关键词的组合，应当搭配使用。

3) 保持谨慎选择关键词
保持谨慎选择关键词，可以提高排名效率。因此，请依据网站的定位和用户群体的需求，选择适合的关键词。同时，请注意控制关键词数量，避免出现“太多了，没人看！”的情况。

4) 通过SEM、PPC、SEO反馈来调整关键词
通过SEM、PPC、SEO反馈来调整关键词，可以帮助网站更准确地为用户提供内容。但是，请注意不要过度调整，以免影响关键词的排名。

# 13.收录优化
SEO的第九个建议是通过收录优化来提升网站的收录率。好的收录优化能够帮助网站更快地被搜索引擎收录，提升网站的排名。下面是一些收录优化的建议：

1) 减少打开网页的请求数
减少打开网页的请求数可以减少服务器资源占用，提高网站的响应速度。因此，应该在服务器端进行压缩、缓存、合并等优化。

2) 使用sitemap来提升收录速度
使用sitemap来提升收录速度，可以帮助搜索引擎更快速收录网站内容。但是，请注意控制sitemap的大小，防止搜索引擎扫描超时。

3) 设置robots.txt文件
设置robots.txt文件可以帮助网站更好地管理自己的爬虫行为。但是，请注意控制文件内容，防止误导搜索引擎。

4) 利用插件和模板来提升收录速度
利用插件和模板来提升收录速度，可以帮助搜索引擎快速识别网页的结构，提高收录效率。但是，请注意保证插件和模板的安全性，防止攻击者对网站进行篡改。
