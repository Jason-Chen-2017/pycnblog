# 1.简介
  

随着时代的进步和科技的飞速发展，中国社会发生了巨大的变化。不同于古代中央集权的国家，现代中国是一个平等、自由的多元主义国家。然而，也正如世界历史上其他国家一样，中国存在着长期的政治斗争。每一个朝代都面临着许多重大的政治问题。因此，我们在此通过阅读前人的研究成果，对中国历代政治得失进行梳理。通过这项研究，希望能够帮助我们理解当前中国所面临的严峻形势，提升我们应对国内外形势的判断力、洞察力和勇气。

# 2.定义
“政治”（zhèng zhì）是指国家的统治方式、社会制度及各种规则的总称。在中国传统的政治观念中，“政治”可以分为“政府政治”和“宪政政治”。“政府政治”是指对政府权力的垄断和支配；“宪政政治”则是要求各方面遵守共同的法律。古代的中国只有“政府政治”，即中央政府独自领导一切事务并对百姓负责，而现代中国则同时具备“政府政治”和“宪政政治”，包括自治区政府和市级政府掌握着部分行政权力，一些地区则实行“地方自治”。

“政治得失”（zhèng zhì dàile）指某一时期或者某一政党的政治行为对社会造成的影响。政治得失具有时间性、地域性、政治性、阶层性等特征，主要包括以下六种类型：
* 时期性政治得失：指不同时期政治制度、意识形态对社会的影响。从战争年代到改革开放时期，中国都经历了民主化浪潮、计划经济转向市场经济的过程，产生了不同的政治得失。
* 地域性政治得失：指城乡差异对社会的影响。在农村，农民因地籍流动受到的冲击较小，但因户籍制度导致的贫富分化则十分突出。在城市，则存在住房保障和医疗卫生保障等福利制度，不少人口过多的地区社会财富结构不平衡。
* 政治性政治得失：指不同政治制度或意识形态对社会的影响。如市场经济体制下的新旧政治制度之争，或不同政治倾向对社会的影响。
* 阶层性政治得失：指不同社会阶层对社会的影响。如知识分子群体对社会的支配作用较弱，社会底层民众的积极参与则会影响他们的生活。
* 社会性政治得失：指不同社会结构对社会的影响。如不同群体的声音和需求可能会影响社会的走向。
* 文化性政治得失：指不同文化对社会的影响。如儒家思想强调仁德修身，道德修养的重要性被低估，促使社会出现很多偏激的现象。

# 3.历史进程回顾
## 中国古代政治制度
中国古代封建社会的政治制度，有三种主要形式。第一是君主制，即天子独裁，所拥有的封邑称臣，君主为国家之主，是最高政治权力的代表。第二是伯氏政治，即大家族庶出的政治权力体制，由人民选举产生，称“大夫相国”。第三是士大夫政教合一的政治制度。其中，士大夫是指孔子、荀子、墨子、韩非子、苏轼等儒家圣贤，包括公羊、盐铁、工商、游魂、隋文、李白等著名诗人和词人。他们颂扬孔孟之道，反对王侯将相，维护儒家正统思想，旨在实现个人奉献、集体努力、国家繁荣昌盛。

## 明清两朝政治制度
明代军事专制主义（也称军功专制主义）、官僚主义制度，加之当时的种种外来因素，导致明朝初年军阀混战、官僚主义泛滥。军阀割据、士绅擢作、官商勾结、贪污腐败横行，严重阻碍了社会秩序的稳定。因此，明政府不得不削减财政收入，停息赋税，加强征税，实施整党（改组），修缴治安费，甚至动用巡警，严厉打击异己。明政府采取了几种办法，包括“皇亲国戚”，实行阶级斗争，“为民除害”，责令扫除滥觞，严禁演艺活动；而且还建立了一套严格的司法制度，凡有异议都须通过法律途径解决。

隋文帝之后，江南党派纷纷起兵，帝党欲维持自己的霸权，使其无法有效治理国家。直到唐朝李煜所创设的全民政治，试图改变明末的弊端，为满汉、汉唐三朝后来者开辟道路。但是，唐朝开国初期的政治腐败也引发了国人的愤恨，曹操惨死、刘禹锡牺牲，李世民革命失败、封建贵族暴力滋长，这使得李唐之间的阵营出现分裂，李唐三代都面临着严峻的政治斗争。

清末民主化浪潮席卷中国，推翻满清政府，实现政治上的“变局”，颠覆了国民党的统治，建立了新的政治制度——一方面是各党派之间进行竞争，另一方面是一批英才优秀的知识分子利用社会资源，推动政治改革，形成了新的社会结构。康有为、梁启超、林则徐、鲁迅等成为时代的榜样，他们继承先人的政治理想，自觉抱负的是推动中国走向更好的政治发展方向。

## 清末新政
清末新政是清政府对“满蒙豪族主义”、“汉武帝主义”、“汉魏玄宗主义”、“邕江密谋”、“鹿邑荆轲”、“倭寇威胁”等国民党势力的一次有效行动。为了消灭异己，清政府采取了以下策略：
* 民主改革：清政府开放党禁，实行言论自由，允许不同党派人士在官方平台上交流意见。
* 政党改革：清政府主张政党轮替，打破长达半个世纪的单一党制，确立民主宪政的政治制度。
* 经济改革：清政府大幅度增加财政收入，平反冤假错案，依法处罚特务和匪徒。
* 外交策略：清政府从拉美西方接收很多外交机构，如尼日利亚驻华使馆、肯尼亚驻华使馆等，建立了一系列的国际联系。

不过，由于社会进步的压力，新政还是被广泛认为是野心家的“大倒退”，导致经济、政治、军事等方面的空虚和混乱，最终酿成了血雨腥风的大屠杀。

# 4.北京奥运会历史演变
1988年，中国举办第七届北京奥运会。这是国际足联1980年代初推举中国举办的举办大型赛事，因为奥运会是世界上规模最大的自然竞赛，是亚洲、非洲、欧洲、北美和南美等地区最重要的比赛。

1980年，北京奥运会于1980年7月31日在中国北京举行。这是首次由中国主办，也是第一届。

1984年，第五届北京奥运会在北京举行。这是中国举办的第三个综合性比赛，也是中国举办的第一个国际性赛事。1984年12月31日，北京队取得了在奥运会上的冠军奖牌，这是中国自1908年至1988年举办的第一届冠军奖牌。

第八届北京奥运会于2008年5月28日在北京举行。这是中国举办的第四个综合性比赛，也是中国举办的最后一届国际性赛事。

1988年和2008年的北京奥运会都是中国举办的第一届和最后一届冠军奖牌。两届奥运会的筹办者都不是中国政府，却都被中国政府授予了政府奖章。

# 5.中国农业现状与发展

## 一、中国农业发展的历程

### 早期阶段
甲午战争（1066年－1072年）期间，中国由于战火频仍，农田锈蚀，农产品无法出口，农民滞留在边境。西周初年（公元前501~前439年），龙族人接受汉族人的带领，组建了为数不少的雇佣农民。公元前386年（周武王时）的一段时期，郡国的土地贵族控制了农业，只能依靠卖田补贴来获取粮食，缺乏耕种能力。此时，农民已经独立出来，开始拥护周朝贵族的改良政策。

西汉末年（公元前221~前177年），日本和越南入侵中国，不少城市被围困。太平天国运动之后，清政府派遣军队镇压叛乱，导致全国范围内反清复明、清兵入关，农业生产得到恢复。民国初年，北京政府实行计划经济，由国家直接给予农民的补贴，连年扩大了农业投入。

19世纪初，中国的农业占GDP的1/3左右，产量仅次于钢铁，在短时期内发展缓慢，主要依靠耕种和手工业。鸦片战争后，许多人担心中国的粮食供应不足，于是提倡捐献给沦陷区的穷苦百姓。

### 近代阶段
辛亥革命（1911年）推翻了旧的君主专制制度，建立了共和国，结束了周朝的一党专政。这是农业改革的起点。

1912年改革开放，农业生产开始蓬勃发展。第一次世界大战爆发，欧美农民的反抗、恐怖活动愈演愈烈。

大跃进（1936年），为了克服自然资源的限制，中国大规模的反右倾运动，打压知识分子，废除市场管理体制，实行严格的物价管制，对农民进行猛烈的阶级斗争。

文革（1966年）结束了共和制度，开启了新时期的改革开放，培育了互联网、移动通讯、工业化、城市化进程。

2008年奥运会后，随着经济的发展，农业产业蓬勃活跃。

2014年至今，随着中美贸易摩擦加剧，农业产品价格走势愈演愈烈。

### 发展阶段划分
中国农业从早期阶段到近代阶段分为两个阶段：

* 农业落后阶段：指在东周初年就开始农业发展的阶段，生产水平低下，主要靠手工业、采棉织布、简单加工等手段，农民的年龄也较为年轻。
  * 长城镇
  * 糯米
  * 棉花
  * 小麦
  * 玉米
  * 稻米
  * 大豆

* 农业兴盛阶段：从辛亥革命到文革结束的二十世纪五十年代，经济的快速发展，使农业的生产能力超过了国家的需要，出现了多种农产品的丰富，农民的生活水平提高，品质也有了显著的提高。
  * 城镇化
  * 高产农作物
  * 豆类
  * 花类
  * 生猪
  * 淡水鱼
  * 红薯
  * 山药

## 二、农业的资源配置情况

中国农业的主要资源主要是土地。中国有14亿平方公里耕地，约占全球耕地总量的1/10。农业耕地分布在黄河流域、长江流域和珠三角地区，尤其是在长江、黄河两条大河流域，适宜农业生产。

中国农业用地的大小与产量成正比，耕地面积一般以千亩到万亩不等。其中，田边地、磨石地、平原地三大类占约70%，余下三大类约占20%-30%，并以边远地区为主。

土壤质量：农田的土壤有涝、湿、干、粘、砂四类，其中砂类占75%以上。耕地的土壤，主要是含有钙质化合物，土壤中含有的污水、水分不断积聚，故农业作物在充分光照条件下，适宜于保存、晒干。

气候条件：中国北方气候温暖湿润，有利于大豆生长，其他农作物也有适应性。长江流域的乾旱湿润带来清澈的阳光，气候宜人，耕作条件好。此外，由于中国生物多样性的保护和农业资源的多样性，使中国农业发展面临各种环境质量、生物多样性、经济效益、人口数量等多方面挑战。

## 三、中国农业主要产业分类及现状

中国农业主要产业主要是以下几类：

1. 辛酉革命后期到鸦片战争期间农业的发展。
2. 鸦片战争后期到元代农业的发展。
3. 元代到明代农业的发展。
4. 明代到清代农业的发展。
5. 清代至今的农业发展。

### 辛酉革命期间农业发展

在辛酉革命后期，农业已进入一个相对发达的阶段，农村贫困、土地兼并、生产资料日益贵重，农产品普遍价廉物美。

主要农作物有大棉、小麦、大豆、玉米、青菜等，主要经济作物有高粱、大麦、玉米等，主要农副产品有油料、棉纱、糖料等。

辛酉革命期间，中国的城市人口以农民为主，人均耕地数量在五六千亩左右。辛酉以后，在几千年的社会主义建设中，农村人口的增长被压缩，土地的流失越来越严重，农民的身心不健康等问题日益突出。农业生产总值有所下降，但其它农产品仍保持较高的产品价值。

### 鸦片战争期间农业发展

鸦片战争（1840年）之后，清政府以武力镇压了日本侵略，但此时却没有改变农业的发展方向。

鸦片战争之后，清政府因经济、政治原因，并不愿意农业的发展。农业生产处于停滞状态，主要经济作物如高粱、大麦等价廉物美，主要农副产品有牲畜肉、油料、柴火等。

鸦片战争之后，有些人私生子嗣大量出现，把农民作为寡头，不让农民有健康的工作、居住、消费。许多农民由于没有钱、没有福利，无法靠农业收入维持生活。

鸦片战争期间的农业，主要经济作物有高粱、大麦、玉米等，主要农副产品有油料、棉纱、糖料等。

### 元代到明代农业发展

元代到明代，中国农业得到发展。

元代至明代，主要农作物有大棉、小麦、大豆、玉米等，主要经济作物有高粱、大麦、玉米等，主要农副产品有油料、棉纱、糖料等。

1905年，蔡京、张之洞率领五万农民离开山东回乡，创建了中国第一个县乡农业专门委员会。到了1912年，农业生产实现了初步发展，开始渐入佳境。

1927年，国民政府发明了轮船，使海岸上的农民更方便地运输粮食。

1949年，中国共产党在延安发动“一二·九”事件，鼓动农民们对国民党的投降，要求实现“赤化”。

1949年至1978年，农业开始进入高速发展时期，粮食生产量大增，主要农作物高产，工业品占总产值的比例逐年提高。

### 明代到清代农业发展

明代到清代，农业发展呈现出国际比较优势。

明代以后，中国的农业开始走向了一个新阶段，开始利用新技术、新农具进行高科技的加工、农产品的开发和产品出口，农业生产力明显超过当时国家需要。

明代末年，黄花、豆苗等外来物种生长的广泛出现，并成为明末的主要经济作物。

光绪、袁世凯等大臣通过改革开放，对农业的发展作出重大贡献，中国农业生产力由最初的1/3逐渐提高到1/2。

在中国农业史上，明代后的农业政策主要是“三农”政策，即土地流转、农田整顿、农业科技。明末以来，生产资料、经济作物、农业要素的增加，使农业政策不断完善、优化，农业发展取得巨大成功。

至清代末，农业生产呈现出历史性的爆发性增长，工业品占总产值的比例持续下降。

### 清代至今的农业发展

清代以来，中国农业实现了大规模的现代化，农业发展取得巨大成就。

从1912年辛亥革命到1949年五四运动，中国的农业普遍落后，主要经济作物仍然有高粱、大麦，但大田已经出现，品种纷繁，主要农副产品以棉纱、油料为主。

1950年，中国开始实行计划经济，开始了现代化的农业发展。农业生产率由最初的20%以上逐渐下降到10%-15%。

1952年，恢复了人民公社体制，使农村经济得到初步的发展。

1964年，广东省的粮食生产率由最初的60万斤/亩下降到70万斤/亩。

在中国农业史上，农业的发展具有五个阶段：

1. 辛酉革命后期到鸦片战争期间农业的发展。
2. 鸦片战争后期到元代农业的发展。
3. 元代到明代农业的发展。
4. 明代到清代农业的发展。
5. 清代至今的农业发展。
