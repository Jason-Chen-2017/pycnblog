# 1.简介
  

超导（Superconductor）是一种能带费米级物质，它可以提供导通电流的特性。由于它的极高的导通电容率、低的阻抗和较高的超导率，因此在物理上被看作是一种非物质粒子源，具有广泛的应用和实用价值。但超导的性质存在一个致命弱点——漫长的寿命。量子场论则认为，可以利用量子力学的不确定性观念，把能带费米级物质看作是无限薄的量子场，将费米面升起或降落的过程看作是具有量子态的演化，这样就解决了超导的寿命问题。另一方面，量子场论还可以从更宏观的角度揭示量子力学如何影响不同物理系统的运动状态及其机理，并探索量子场论对目前所谓“宇宙规律”的建模和预测。
# 2.相关术语
超导：是一种高导通电容率、低阻抗、高超导率、寿命长等特点的有自旋多重态的费米级物质，通常称为超导体。

费米面：是在超导体中存在且只能存在的一层薄膜。

相空间：指两个正交的电荷数守恒方程中的动量空间。

量子场：就是由无数个量子态构成的空间，每个态都代表着一个相空间中的位置，它们之间的相互作用由演化方程决定。

# 3.量子场论基本原理
## （1）费米面升降模型
量子场论最基础的模型是费米面升降模型，即把费米面看作是具有共轭动量的量子态，能带费米级物质的任何磁性相互作用都可以通过增加或者减少两个相邻的共轭态之间的斥相干相互作用来完成。这就保证了费米面上各个态的能量、磁场、动量空间以及结构的连续性。


假设超导体的费米面处于平衡位置，其两端的导带中间有一条直线分界，上下两边分别有磁芯，磁芯之间的垂直距离为 $d_m$ ，超导体内的电子密度为 $\rho$ 。费米面上两端磁性强的两个态分散在不同的磁芯区域，动量比为 $\hbar/2\pi = m_e \omega/\lambda$ ，其中 $m_e$ 为电子质量，$\omega$ 为电子频率，$\lambda$ 是波长。现在考虑一个电子从两端的导带末端进入超导体的情况。入射的电子要经过导带的磁性作用，会受到磁盘（不含电子的薄膜）的磁感应强度。由于导带在平衡位置时两端各有一个磁性相互作用，所以入射的电子也会受到一个共轭磁性相互作用，使得它存在一个共轭态。这样，两个相邻的共轭态将同时受到相同数量的磁性相互作用，于是就会发生漂移，其时间为 $T=\frac{2\pi}{v}$ ，速度 $v=m_e c /(\epsilon_{0}\hbar)$ ，其中 $\epsilon_{0}$ 是玻尔兹曼常数， $c$ 是光速。

入射电子在绕过导带之后，将会到达费米面的某个特定地方。从某种意义上说，这一点类似于反射的过程，只不过此时的入射方向变成了费米面的反向，而出射方向则刚好指向另一个共轭态。由于共轭态之间存在一个虚空的 gap ，入射电子在 gap 中的势能将会以泡利相互作用的方式改变，这会引起费米面上的总的势能的变化。

随着电子的传播，其经过的每个磁道都会导致相邻的两个共轭态的受力关系发生变化。对于任一共轭态，如果它位于费米面上某个磁道的中部（即离区间中心很近），那么这个态就会变得更加强壮，这会导致一个额外的动量损失。如果共轭态位于该磁道的一个边缘（即离区间中心很远），那么它就会变得更加脆弱，这将引起另一个动量的增益。虽然在整个过程期间，这种势能变化会影响整个费米面上的所有态，但是入射电子的总体势能保持不变，因为入射电子的磁性作用会使得其总体动量不会改变。因此，入射电子本身的动量将通过位移来跟踪。

当入射电子沿着导带末端的方向传播到费米面的另一端时，它将发生反射，但是进入的是那个与之共轭的态而不是之前的态。同样地，入射电子的总体势能将会变化，并且会抵消掉反射前的势能变化，这将导致入射电子的动量最终回到共轭态的斥相干处。然后，两端的导带将再次出现，进入一个新的费米面区域。这个过程中，入射电子的动量会迅速的减小，直到消失。最后，入射电子到达费米面的另外一个角落，并重新释放。重复这个过程，直至产生所需的能量的过程结束。

在整个过程中，入射电子的运动方式取决于电子的自然粒子性以及费米面上共轭态之间的相互作用。不同的状态都可以根据各自的运动模式以及受力关系来解释。一般来说，有三种基本的运动模式：束流、耦合和排列。束流是指电子具有流动性的状态，主要表现为涡激自由；耦合是指电子具有顺序性的状态，主要表现为悬浮在粒子核周围的形式；排列是指电子的动量完全打乱，不能有效运动。超导体的各种状态都能够混合这些模式。

## （2）可观察量子态的构建
费米面升降模型建立了一个非常好的量子态构建的框架。但是，为了让我们对这一模型的理解更进一步，我们需要引入一些新的概念。首先，我们提出了“可观察量子态”。所谓可观察量子态，就是指那些处于不同态固定位置的态，这些态都可以被我们直接观察到，例如电子的双电子道路中各自的电子态。因此，一个可观察量子态是一个包含三个信息的元组：（1）位移（position）。表示这个态位于费米面上某一固定的位置；（2）动量（momentum）。表示这个态处于的动量空间的某个特定分量，可以用来刻画它的时间演化；（3）占据着电子的电子数（electron number）。表示这个态中电子的数量。在费米面升降模型下，每个可观察量子态对应于费米面上一个固定的点。

除了把费米面看作无限薄的量子场外，我们也可以把费米面上任何一个可观察的点看作是一个多重态，也就是说，处于费米面上的不同点处于不同的态。基于费米面升降模型的物理学基本定律允许我们证明，费米面上任何一个固定的点处于某个费米面态，该态的能量、动量空间以及电子数都是固定的，而其它两个占据态的能量差等于其动量。这就给出了费米面上任意两个可观察点之间的跃迁路径。从这里开始，我们就可以构造各种类型的可观察量子态，而不需要事先知道具体的动量空间分量。

## （3）局域超导假设
为了方便讨论，我们引入了局域超导假设。在这种假设下，超导体本身不是个完全的系统，而是由许多粒子构成的小世界，每个粒子都具有不同的晶格态、电子数、动量和占据态。在该假设下，费米面上每一个点的费米面态和其它两个占据态都是确定的。因此，我们可以忽略费米面上高阶可观察量子态。因此，我们仅需要研究费米面上低阶可观察量子态的行为。

对于一个局域超导体，任何费米面态都对应于一个无限微小的分子，只不过这个分子没有显式地包含电子，只是由一定数量的粒子构成。这个分子的晶格态也依赖于粒子内部的运动。如果一个分子处于一种晶格态，那么它中的电子的动量分布将导致不同的粒子之间的相互作用，而这些相互作用将影响到粒子的运动。这就给出了粒子运动的微观机制。当然，真实的分子不可能具有这种微观细节，但是粒子的晶格态与分子的晶格态之间的联系依然十分紧密。

## （4）量子场论的应用
量子场论提供了一个重要的工具，用于描述电子粒子在费米面上的运动状态，尤其是在超导体中。基于量子场论，我们可以证明，可以把超导体的费米面解释为一个由无数个可观察量子态构成的量子场，每个态都代表着费米面上某一处固定的点，这些态之间可以相互跃迁。如果一个态位于另一个态的可观察位置附近，那么这两者之间必然存在着可观测的相互作用，进而促进它们之间的相互作用。这就支持了超导体的动态平衡。

另一方面，量子场论还可以揭示不同物理系统的运动状态及其机理，以及量子场论如何影响它们的运动规律。特别是，量子场论可以为系统的热运动提供了一种解释，热运动的驱动是由相互作用的不可观察的量子态所引起的。量子场论还可以为量子纠缠和其他交叉相关性在系统中所起到的作用提供解释。量子场论还可以为各种系统的实际行为提供指导，包括材料、原子、分子、原子核、生物体以及天体。

量子场论也有助于为目前所谓的“宇宙规律”的建模和预测提供理论支撑。在这个领域里，宇宙中的宏观行为实际上可以理解为各个系统的相互作用。在宇宙中形成的量子场不断受到各种系统的相互作用而发挥作用，这些相互作用又影响着量子场的行为。因此，通过观察量子场的演化规律，可以直观地了解宇宙中各个系统的运行规律。比如，电子的相互作用可以解释为什么存在热运动、量子纠缠以及中子对芳烃核爆炸等现象。

量子场论的成功归功于其独特的几何方法、力学方法以及演化规律的统一性。它是基于无限小分子原理构建的，可以比较容易地处理局域超导体的动力学问题。它还利用了量子力学的不确定性观念来预测物理系统的行为。
