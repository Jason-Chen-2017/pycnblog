# 2.概率论基础
## 2.1 贝叶斯定理
贝叶斯定理(Bayes' theorem)又称作贝叶斯法则或 Bayes' law。它是关于事件的条件概率的定律。在概率论和统计学中，给定某些条件时，随机变量 X 的概率分布 P(X)，可以用来描述观察到某个值 x 时，其对应的条件概率分布 P(x|C)。该条件概率分布可以根据已知的各种条件的结果来估计未知事物的结果。假设某个人生于年龄 n 年的人群中，人口比例为 p，那么在这些人的条件下，死亡的概率为：

P(D=yes|n,p)=p*q^n*(1-q)^(n-1)

其中，q 为患病的概率，也就是假定存在一个检测准确率为 q 的测试。

贝叶斯定理是由拉普拉斯在19世纪提出的。他关于盒子里的球，已知其颜色，问“除去黄色的球后剩下的白色和黑色的各有多少？”这一问题的答案是：

(1) 已知颜色是白色的，剩余的其他颜色分别有 a/(a+b) 和 b/(a+b) 个；
(2) 已知颜色是黑色的，剩余的其他颜色分别有 c/(c+d) 和 d/(c+d) 个。

通过这两个假设，利用贝叶斯定理，可以求得总体中白色球和黑色球的比例：

P(A|B)=P(B|A)*P(A)/P(B) 

其中 A 表示球是白色的还是黑色的，B 表示除去了黄色的球之后，白色球和黑色球各占多少。通过贝叶斯定理的证明，知道两者之间的关系，就可以推断出除去黄色后的白色和黑色球的比例。

## 2.2 概率分布
概率分布是概率论的一个重要概念。随机变量 X 有若干取值的集合 V，记作 X={v1,v2,...,vn}。根据概率论，对于随机变量 X 来说，其取值为 v 的概率分布可以表示为函数 Pr(X=v) 。概率分布函数与离散型随机变量相关联，而连续型随机变量则没有概率密度函数。对于连续型随机变量 X ，其概率密度函数为：

Pr(X∈[a,b]) = ∫_{a}^{b} f(x)dx

式中，f(x) 是 X 在 [a,b] 区间上连续的概率密度函数，即 ∀x∈[a,b],0<=f(x)<=1 。

## 2.3 期望、方差、协方差、马氏距离
### 2.3.1 期望
对于一个连续型随机变量 X ，如果对任意实数 a≠b，有：

E(af(x)+bf(y)) = a*E(f(x))+b*E(f(y))

则称 af(x) 和 bf(y) 是独立的，否则，称它们不独立。随机变量的期望 E(X) 可以表示为：

E(X) = sum_v{vPr(X=v)}

### 2.3.2 方差
对于一个随机变量 X ，如果对任意实数 a≠0，有：

Var(af(x)) = a^2*Var(f(x))

则称 af(x) 和 X 之间是相互独立的，否则，称它们有关联。随机变量 X 的方差 Var(X) 可表示为：

Var(X) = E((X-E(X))^2) = E(X^2)-[E(X)]^2

### 2.3.3 协方差
对于两个随机变量 X 和 Y ，如果对任意实数 a≠0，有：

Cov(af(x),bf(y)) = abCov(f(x),f(y))

则称 af(x) 和 bf(y) 是独立的，否则，称它们有关联。随机变量 X 和 Y 的协方差 Cov(X,Y) 可表示为：

Cov(X,Y) = E[(X-E(X))(Y-E(Y))]

### 2.3.4 马氏距离
对于两个向量 u=(u1,u2,...,uk) 和 v=(v1,v2,...,vk) ，定义其马氏距离 d(u,v) 为 sqrt((u1-v1)^2+(u2-v2)^2+...+(uk-vk)^2 ) 。如果对任意实数 k，有：

1. d(u,v)>0
2. d(u,u)=0
3. d(u,v)=d(v,u)

则称 u 和 v 是正交的，否则，称它们不正交。

## 2.4 均匀分布
均匀分布是指所有可能的值都有相同的概率，记作 U(a,b) ，其中 a 和 b 分别为均匀分布的上下边界，且满足 a ≤ b 。对于连续型随机变量 X ，若其概率密度函数为：

f(x) = {1}/{b-a},   a ≤ x ≤ b

则称 X 服从均匀分布，记作 U(a,b) 。当 a=−∞ 或 b=∞ 时，X 不再是连续型随机变量。当 a=b 时，X 服从恒等分布。

## 2.5 指数分布
指数分布是一种在一定区间内取值较小，而离开该区间后，概率增长很快的概率分布，记作 Exp(λ)。其概率密度函数为：

f(x;λ) = {λ}{e^-{λ|x|}},   x>0

式中，λ > 0 为参数，e 为自然常数。当 λ < 0 时，指数分布退化成泊松分布。

## 2.6 对数正态分布
对数正态分布是一种双峰分布，分布曲线呈现两个尖峰，峰值高度不同，形状和位置都可变。其概率密度函数如下：

f(x;μ,σ) = (1/{2π}|σ|)exp(-{(ln(x)-μ)^2}/{2σ^2})

μ 和 σ 分别为平均值和标准差，当 μ = ln(θ) 时，称之为标准对数正态分布。当 θ > 0 时，X 服从对数正态分布，记作 LN(θ)。
