# 1.简介
  

目前市场股票交易已经成为中国互联网金融的主流业务之一，占据着金融交易量的85%以上，尤其是在互联网、移动支付、IoT、区块链等新兴技术的驱动下，股票的交易频繁、速度快、金额庞大，已经成为整个社会经济活动中不可缺少的一部分。但是在股票市场中，并不是所有人的交易行为都符合大众认可的交易规则，比如那些常被称作牛熊线、短平仓和反弹行情的震荡行情，他们对股价的影响力究竟有多大，实际上还有哪些潜在风险值得我们关注？下面就让我们一起探讨一下。

# 2.背景介绍
“股市大戏”又一次到来，霸道总裁耶伦宣布了截然不同的重磅信号：“疯狂吞噬”（Crash）。即使是长期炒股的人也明白，疯狂吞噬行情是没有好日子过的！市场迅速回暖，赚钱效应不断强化，全球股市急转直下，财务危机依旧深重，中国股市连续四个交易日下跌超过20%，如果不能及时发现、跟踪、分析这股票的震荡情况，势必会让股民损失惨重。

但是，如何通过正确的方法判断出个股是否存在震荡、如何从震荡中获利是每一个追逐高收益的投资者都需要面对的问题。要想把握个股的真实走势，首先要理解个股震荡背后的核心概念——技术指标。

## 2.1 技术指标
“技术指标”通常指的是通过对一段时间内股价波动的研究，来确定该股是否在出现震荡行情。所谓技术指标，就是以股价走势图绘制出的一些曲线或技术形态，用以检测其震荡性和趋势性，比如趋势线、三角形底分型、K线死叉反转等。基于技术指标，可以快速、准确地判断出股价走势的走向、规律，从而预测出未来股价走势的变化，并做出相应的调整。以下是广泛使用的技术指标：

1. 移动平均线MA(n)：
移动平均线指标（Moving Average）是为了寻找一定长度周期内价格平均值的一种技术分析工具。它是一种统计分析的方法，计算某一特定时间段的股价的移动平均值，用于观察、分析和预测股价在变动中的 tendency和 momentum。一般情况下，移动平均线由n条线组成，每日更新一条线，显示股票的平均买入价、卖出价、持有价以及当日的最高价/最低价。移动平均线对价格变动的敏感度较高，是一种比较灵活的分析工具。

2. 指数平滑移动平均线EMA(n):
指数平滑移动平均线（Exponential Moving Average，简称EMA），是一种改进的移动平均线。它利用前一日的指数加权平均数来计算当天的股价平均值，其计算公式如下：EMA_i=2/(n+1)*X_i+(1-2/(n+1))*EMA_(i-1)，其中：X_i表示第i日收盘价；EMA_i表示第i日的EMA值；n表示平滑参数。参数n越小，拟合程度越高，但计算量越大；参数n越大，拟合程度越低，但误差越小。

3. 布林带Bollinger Bands:
布林带由两条轨道线组成，一顶部称为上轨线，另一根称为下轨线。两条线之间的空间称为箱体。箱体外侧的部分称为离群点，是指超出正常范围的波动。下轨线负责表示股价处于均值位置附近，上轨线则负责表示股价处于均值位置远离。在箱体内的部分称为中间线，表示股价处于趋势阶段。股价在趋势阶段通常具有较大的波动幅度，但是随着震荡行情的到来，箱体外的离群点会越来越密集。

4. K线死叉反转三角形底分型：
K线死叉反转三角形底分型是一类复杂反转信号。其由三根K线构成，包括第一根最高价线、第二根最低价线、第三根当前价格线，底部交叉成交盘，上涨趋势逆转形成第三根最高价线，再次交叉破位，下跌趋势反转形成第二根最低价线，最后再次交叉成交盘。形成此形态的主要原因是K线上方的短期内反转及下方的长期内反转。
