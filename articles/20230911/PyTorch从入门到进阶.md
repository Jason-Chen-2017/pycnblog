# 3.基础知识
## 3.1 Python语言基础
Python是一门简单易学的编程语言，具有良好的可读性和简洁语法，是数据科学、机器学习等领域的首选语言。本章节主要介绍一些Python语言的基本概念和语法规则。
### 3.1.1 变量赋值和打印
在Python中，您可以在内存中创建变量，并赋予不同类型的值。下面给出了一个简单的例子：
```python
x = 10    # 创建一个整型变量x并赋值为10
y = 'hello'   # 创建一个字符串变量y并赋值为'hello'
print(x)      # 输出变量x的值
print(y)      # 输出变量y的值
```
在这个例子中，我们分别创建了整数变量`x`和字符串变量`y`，并将它们的值分别设置为10和'hello'。接着，我们使用`print()`函数输出变量`x`和`y`的值。由于Python是一种动态类型语言，因此不需要指定变量的类型。当您运行这段代码时，输出应该如下所示：
```
10
hello
```
### 3.1.2 数据类型
在计算机编程语言中，数据类型指的是存储值的种类。在Python中，有多种数据类型，包括整数、浮点数、布尔值、字符串、列表、元组、集合等。其中，最常用的就是整数和浮点数。下面给出一些关于不同数据类型的例子：
```python
a = 10         # 整型变量
b = 3.14       # 浮点型变量
c = True       # 布尔型变量
d = "hello"    # 字符串型变量
e = [1, 2, 3]  # 列表型变量
f = (1, 2, 3)  # 元组型变量
g = {1, 2, 3}  # 集合型变量
```
除了以上数据类型之外，还有诸如NoneType、NotImplementedType等内置数据类型。
### 3.1.3 条件判断和循环语句
条件判断语句和循环语句可以帮助您控制程序的流程。下面给出一些示例：
```python
if x > y:
    print('x is greater than y')
elif x == y:
    print('x and y are equal')
else:
    print('x is smaller than y')
    
for i in range(5):
    print(i)
    
while x < 10:
    x += 1
    if x % 2 == 0:
        continue
    print(x)
```
第一个条件判断语句比较两个变量的值，并根据比较结果输出信息。第二个循环语句实现了一个简单的计数功能，用于输出0到4之间的数字。第三个循环语句用于实现偶数输出的功能，每次循环都会增加变量`x`的值，直到`x`等于或超过10。在每轮循环中，我们都检查是否满足`x`为偶数的条件，如果满足，就跳过当前轮循环，否则，就会输出`x`。
