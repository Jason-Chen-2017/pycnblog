# 1.简介
  

## 1.背景介绍
在过去的几年里，市场上已经出现了很多与股票相关的互联网平台，如雪球、新浪财经等。越来越多的人开始关注股票市场的投资动向，同时也受到各类私募基金的青睐。市场上存在着越来越多的“炒股热”，越来越多的人购买股票作为一种获得经济利益的方式。而私募基金则一直没有停止的火热。私募基金的投资模式就是以低价买入股票，然后通过分红等机制持续获利，以期达到长久稳定的收益。但是，目前国内私募基金的参与门槛较高，投资者不仅需要良好的风险意识，而且还要具备很高的理财意识。

市场研究机构发布了一项调查报告指出，全球范围内的私募基金总资产规模超过3万亿美元，其中超过三分之一的资产是由中国人开设的。因此，中国是当前私募基金的最大最活跃地区。特别是近年来，随着中国经济的发展，中国的经济总量越来越大，并且金融业也逐渐成为中国经济的一部分，加之世界经济的转变，中国的投资环境也发生了变化。因此，对于中国的私募基金来说，拥有相对较大的资产规模，并实施定期返还投资收益是一个比较理想的选择。

然而，这种情况下，如何利用私募基金合法化的优势，发展出私募基金的赚钱能力，却是一个值得探索的课题。在这个背景下，就产生了本文所要阐述的问题。

## 2.基本概念术语说明
私募股权基金（Private Equity Fund）或称私募基金，是指非营利组织中基金管理人向其个人会员发放股份证券而组成的私募基金。主要成员有高级管理人员、顾问及基金公司工作人员等。私募基金与传统的公募基金相比，其运作方式更接近于私人企业经营，基金的管理人员的个人偏好往往决定了该基金的运作方向和策略。私募基金建立的初衷是为了扩大企业内部的资本运作空间，有效地实现个人理财目标。因此，私募基金的理财产品往往具有较强的增长性、适度可控性，并被设计为由少数精英投资者共同管理。私募基金由于其独特的运作方式，使其与一般的公募基金有着不同的运作模式。

### 2.1 发行方式
私募股权基金的发行方式通常分为两步，第一步是发行准备期，即公司筹备期，这时公司主要人员筹集资金、进行准备。第二步是公开发行，一般在六个月后公开发行。公司的所有权掌握在股东手中，只是向基金管理人发放股票，所以不存在任何公司所有制上的限制。私募股权基金的持有人可以自由退出基金，获得退休金或年金保障。

### 2.2 投资目标
私募股权基金的投资目标主要是为基金管理人提供投资资金，该基金将根据管理人的个人意愿和决策，按规定方式筹集资金。因此，私募股权基金的投资策略有两个维度，一个是管理层的决策，另一个是股权激励制度。管理层的决策主要包括如下四种方式：

1. 直接推荐投资人：私募基金的管理层会与投资人直接进行交流，讨论交易计划和方法，并与投资人签署协议。
2. 内部投资组合：私募基金的管理层根据对客户的需求进行分析，创建符合自身条件的投资组合。
3. 外部投资人参与：私募基金的管理层可能会聘请外部投资人参与私募基金的建设与运营过程。
4. 资本运作模型：私募基金的管理层可能采用资本运作模型，即私募基金与资本经纪人合作，帮助公司筹资运作。

股权激励制度主要包括：

1. 激励设定：私募基金的管理层经常会设定基金投资者的预期收益率。这可以引导基金投资者设定自己的投资目标，为投资行为提供指导。
2. 分红方式：基金管理人有权从投资者那里收取分红。分红方式可以采取定期分红、透明分红、商业模式分红等。
3. 年费设置：私募基金经常设定年费，向投资者分享管理费用的补偿。年费的大小一般为1%至2%，具体还要视基金管理人的资历和经验而定。
4. 流动性管理：私募基金经常会在股票、债券、现金等多个资产之间进行流动性管理。流动性管理可以通过设定溢价、抗风险策略等来优化投资结果。

### 2.3 公司治理结构
私募股权基金的公司治理结构主要有以下三个方面：

1. 董事会制度：私募股权基金只有一名董事长，他负责基金运作管理、向投资者分配股权，并制定基金运作的具体方案。
2. 管理层制度：私募基金的管理层由基金投资经理、基金销售人员、财务官员、风险管理人员等组成，他们分管不同职责。
3. 监事会制度：私募股权基金的监事会由基金的独立顾问组成，他们不直接参与基金运作，但对基金的经营状况进行监督。

### 2.4 运作方式
私募股权基金的运作方式主要分为两种，一种是跟踪式运作方式，另一种是指数化投资方式。

1. 跟踪式运作方式：跟踪式运作方式是指基金投资者买入和卖出单位成本价的固定比例，比如每当股票价格上涨10%，基金便按照相同比例卖出股票，每当股票价格下跌10%，基金便按照相同比例买入股票。这种方式具有较高的准确性，但交易成本高昂，交易次数少，操作难度大，且容易被割韭菜。
2. 指数化投资方式：指数化投资方式是指基金投资者买入与指数基金一致的股票组合，指数基金的表现受全体投资者的关注，因此基金投资者以指数基金表现为参考进行投资。这种方式相对简单，但存在着投资风险。
