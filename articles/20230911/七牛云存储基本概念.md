# 1.简介
  

## 什么是七牛云存储？
七牛云存储（Qiniu Cloud Storage）是基于七牛云平台构建的海量、安全、低成本的云端数据存储服务，提供包括图片、音频、视频、文档在内的多种媒体文件的存储、下载及处理。其独有的七牛云 CDN 服务、可靠的数据传输网络、免费的文件处理接口等产品特性吸引着越来越多用户加入这一领域。而七牛云存储也以其独有的商业模式，将用户的存储空间和带宽合法化、真正实现“按需付费”。 

## 为什么要选择七牛云存储？
七牛云存储具备以下五个优点：

1. 安全性：七牛云存储采用 HTTPS + AES-256 加密方式保障数据的安全，通过校验段签名保证数据完整性。此外，七牛云存储还提供独有的防盗链机制，能够有效防止数据的泄露和恶意攻击。

2. 性能卓越：七牛云存储具有最高的上传速度和下载速度，平均每秒约 7MB 的上传速度和 250KB/s 的下载速度，同时也支持 HTTPS 大文件分片断点续传。

3. 价格经济：七牛云存储的计费模式完全遵循阿里云 oss 对象存储的计费模式。即按流量包或请求包计费，按 GB 或 MB 的存储容量计费，并提供免费套餐。相比于其他对象存储服务，七牛云存储的定价更加友好、简单。

4. 方便快捷：七牛云存储提供了多个 SDK 和 API ，让开发者可以快速集成到自己的应用中，使得开发变得简单又便捷。

5. 弹性扩容能力：七牛云存储可根据业务需要随时增加容量和节点，无限扩展存储空间。而且在地区分布上也是分散的，单个区域的延迟都较低。

综上所述，七牛云存储是一款具备强大功能且价格适宜的云端存储服务。并且，作为国内领先的互联网公司之一，七牛云存储背后有着知名的科技企业运营团队，并在持续创新中不断完善自身服务。因此，七牛云存储是值得考虑的选择！

## 七牛云存储的基本概念和术语
### 数据存储(Object Storage)
七牛云存储提供了对象存储功能，即存储各种类型的数据如：图片、音频、视频、文档、文本等。不同于其他厂商的平台，七牛云存储没有专门的目录结构。一个 bucket 可以存储无限数量的 object。object 是七牛云存储中的最小存储单元，具有唯一的 key 。bucket 中的 object 可以通过 HTTP、HTTPS 访问。

### 账户管理(Account Management)
七牛云存储的所有资源均以账户为单位进行计费。每个账户下可以创建多个 bucket ，每个 bucket 下可以存储多个 object 。其中，accessKey 和 secretKey 分别对应了身份标识和密钥，用于进行认证和授权。

### 桶(Bucket)
bucket 是七牛云存储中最大的存储逻辑单元。每个 bucket 中可以存放多个对象（object）。bucket 命名全局唯一，七牛云存储会自动分配相应的域名（BucketURL）。bucket 还可以配置访问权限（权限控制），以及配额限制（空间和流量控制）。

### 域名(Bucket URL)
bucket 的 BucketURL 是七牛云存储对外暴露的唯一地址。开发者可以通过该地址对 bucket 中的数据进行读写访问。BucketURL 的形式一般为 `http://<你的bucket名>.qiniudn.com/`。访问该地址可以直接访问到 bucket 中的所有对象。

### 上传(Upload)
七牛云存储采用 RESTful API 进行对象上传。客户端可以通过上传请求向服务器提交待上传数据，七牛云存储会自动切割成大小一致的块（block），并保存到多个服务器上，实现分布式存储。对象上传完成后，七牛云存储会返回一个唯一的资源定位符（key），该 key 可供其他用户下载。

### 下载(Download)
客户端通过获取对象的下载链接（URL）即可实现对 object 的下载。七牛云存储提供 HTTP/HTTPS 协议的下载链接。开发者也可以通过七牛云存储提供的 SDK 将 object 下载到本地。

### 断点续传(Resumable Upload)
当网络连接异常或者中断导致上传失败时，七牛云存储提供断点续传机制，从上次中断的位置继续上传。即使上传失败，七牛云存储也会将已经上传成功的块保存到服务器上，以保证数据的完整性。

### 哈希值(ETag)
七牛云存储为每个上传的 object 生成一个唯一的 Hash 值，称为 ETag。ETag 可用来校验上传的内容是否完整，也可用于识别重复上传的对象。

### 数据处理(Data Processing)
七牛云存储提供对象级别的图像处理（如裁剪、缩放、旋转、锐化等）、文字水印、视频截图、音频转换等功能。这些功能通过数据处理 API 提供给用户，支持同步和异步两种模式。同步模式指的是数据处理请求由用户主动发起，七牛云存储收到请求后立即执行处理，然后返回处理结果；异步模式则是在后台执行数据处理任务，处理完成后向用户推送通知。

### 流量统计(Traffic Statistics)
七牛云存储提供实时的流量统计功能，展示当前账号的总流量、入口流量和出口流量。另外，还提供存储空间报表和流量报表，帮助管理员了解当前存储空间的使用情况、流量消费趋势等信息。

### CDN(Content Delivery Network)
七牛云存储还提供可选的 CDN 服务。CDN 是一种透明代理服务，它能够缓存用户访问的热点内容，提升用户访问速度。CDN 节点部署在遍布全球的机房，可根据各地负载情况智能调度数据传输路径，最大限度降低整体的响应时间。

### 备份策略(Backup Policy)
七牛云存储提供异地备份策略，即将数据自动复制到离自己最近的另一个可用区，避免因地域原因造成的系统故障。
