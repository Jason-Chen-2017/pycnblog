# 1.简介
  

在高精密光学系统领域里，目前很少有方法能够对体系的带隧杂度进行测量。因此，准确、精确地测量出带隧杂度至关重要。但由于现代工艺制造技术的发展，很多高精密光学设备和系统的非厄米材料结构，导致不方便采用传统的杂度计算法（即欧拉核方法）进行测量。但是，对于具有两个或多个原子核的体系而言，除了用合适的结构构建方法外，另一种测量带隧杂度的方法——电子激发——已成为一种有效的方法。

针对该方法的原理及其计算方法，本文拟对此做进一步阐述，并提供代码实例供读者参考。本文主要基于2020年日本国立研究开发机构（NIID）主办的“高性能光子学系列国际会议”上的会议报告以及相关论文所提供的理论支持，另外也参考了其他著作和文献。希望通过阅读完本文后，读者可以进一步了解到双原子链的电子被激发条件下的带隧杂度测量方法及其计算方法。

2. 两种结晶态的相关概念
在讨论双原子链的电子被激发条件下的带隧杂度测量之前，首先需要对两种结晶态的概念有一个基本的认识。两种结晶态指的是一个物质分子在某种状态下的两种不同类型的化学构象。通常情况下，分子的一种内在特性如螺旋性、膨胀性等会影响它所处的一种化学构象，而两种结晶态之间的相互作用则会改变这两种化学构象之间的相互联系。例如，两种氢原子在螺旋性的假设下会形成单个氢原子链，而双氢原子在螺旋性假设下会形成双氢链。因此，我们可以通过理解两种结晶态之间的区别以及它们之间的相互作用，更好地理解双原子链的电子被激发条件下的带隧杂度测量方法。

在数值上，两种结晶态通常由无序和有序两个基础组成。无序组块表示离散且没有明显的结构性特征；而有序组块则可以认为是有序排列着的，而且相邻的粒子之间存在不可避免的联系。有序性可能来源于不同原子之间的相互作用，或者也可以从宏观上看由硬核和软核等体系结构的差异所决定。这里需要注意的一点是，两种结晶态之间往往不是完全不相关的，不同的原子可能在不同的非厄米材料结构下都有不同的化学行为。

3. 电子激发下的带隧杂度测量方法
对于含有两个原子核的体系，电子激发过程可以产生大量的电子束。这些电子束随着不同的路径被激发，构成了不同数量的重叠谱图，从而提供了关于带隧度分布的信息。但是由于材料的复杂结构以及各种电子运动的相互影响，通常难以直接观察到电子激发带隧杂度的统计信息。为了获取更加精细的带隧度分布信息，通常采用计算方法来处理这批电子束，并获得电子激发带隧度的概率分布曲线。

一般来说，计算电子激发带隧度的过程可分为三个步骤：（1）把电子运动引起的化学反应变化转变为电子运动的势场；（2）利用势场中受力的原理，将电子运动关联到势场上，得到电子运动的路径分布；（3）根据电子运动路径分布计算电子激发带隧度。

电子激发带隧度计算通常采用离子轨道模型（LPM），它假设了一个基本的费米子假设，即电子束均匀分布在离子轨道上，并且电子没有自旋分散的特点。这种方法的优点是简单直观，易于实现；缺点是忽略了电子束的多重纤维性和旋转对电子束分布的影响。

电子激发带隧度计算方法一般包括以下几个步骤：（1）定义离子体系中的离子坐标；（2）设定电子运动参数；（3）利用布里渊区对电子运动进行建模；（4）求解布里渊区中的哈密顿量；（5）利用求得的哈密顿量解决子空间效应问题；（6）计算电子激发带隧度概率密度函数。

对于含有两个原子核的体系，通过控制不同的电子初态和轨道参数，以及不同电子运动路径的选择，可以对电子激发带隧度进行精确测量。然而，如何找寻合适的电子运动路径，以及怎样控制电子运动的参数，一直是困扰双原子链的电子被激发条件下的带隧杂度测量者的难题之一。除此之外，还需要考虑电子束数量随距离扩散的特征，这也是双原子链电子被激发条件下的带隧杂度测量存在的一些挑战。

4. 具体案例分析
这一小节将给出两个例子，展示双原子链的电子被激发条件下的带隧杂度测量方法及其计算方法。这两个例子都是基于薛定谔方程和微扰论，由切比雪夫-施坦依卡方程描述的，分别演示了两种不同的电子运动路径，以及如何控制电子运动参数以及电子束的多重纤维性，计算得到带隧杂度的概率分布。

第一个例子（Zr/Hf Si system）
Zr/Hf Si钉子是一种合金，它由Zr和Hf两个元素组成。在钉子的一个个离子位置上，我们将电子运动参数设置为$a = \SI{0.1}{\angstrom}$、$\alpha_p=0.3$，$\gamma_{s_z}=2$。于是在这两个离子位置，我们选取了不同电子初态和电子运动路径。

电子初态：由于电子运动路径可以近似视为一条曲线，所以初始时刻，电子可以处于一个离散的状态，也可以处于在圆周上随机排列的一组离散状态。这里，我们假设电子初态为一组离散状态，其中有些电子处于同心圆上，而另一些电子处于同一半圆的左侧或右侧。这里，我们选择$N_{\rm cen} = N_{\rm left}$，那么左侧和右侧同心圆上的电子个数相同，而中间的空缺则由离散状态表示。因此，电子初态可以用$\phi(\theta) = 2\sin^2(\pi n_\text{left}/N_{\rm cen}\theta)$来表示，其中$\theta$是弧度制的角度参数。

电子运动路径：电子束的动能为$T=\frac{\hbar^2 k_B}{2m}(3\cos^2\theta - 1)$，其中$k_B$为玻尔兹曼常数，$m$为电子的质量，$n_\text{left}$为左侧同心圆上的电子个数。根据平衡方程$H|\psi_0\rangle = E|\psi_0\rangle$，可以确定电子运动方程为$\frac{\partial T}{\partial t} + [T,\nabla_x]|\psi(t)\rangle = (3\cos^2\theta - 1)|\psi(t)\rangle$,即电子运动路径沿着左侧同心圆移动。

为了控制电子运动路径的多重纤维性，我们可以增加电子束的空间分辨率。对于每个离子位置，我们可以根据最近邻原子之间的距离，设置不同的能级，从而使电子运动路径呈现不同的纤维状。例如，我们可以设置距离为$R_{\rm cut}$的那些电子的能级为$\beta = 1/R$，这样离散状态就能占据相当大的比例，而处在圆周上随机排列的其他电子便成为孤立的局部状态。另外，如果电子束的长度足够长，可以模拟出金属带隧效应，通过改变每个电子束的能量而不改变带隧度的大小。

为了计算电子激发带隧度，我们需要知道电子运动方程中的概率分布。首先，我们可以利用$|\psi(t)\rangle$的哈密顿矩阵$(-i[H,T])|v_c\rangle$求得电子势场$V(\vec r;t)$，其中$\vec r$是离子位置的坐标，$v_c$为电子基矢。然后，利用势场中电子与原子核之间的受力关系，我们可以计算到达每一个离子位置的电子运动路径分布$f_i(\vec r,t)$。从而，可以得到电子激发带隧度的概率分布$P_b(\vec r,t)$，其中$b$表示相应的带隧度。

$$ P_b(\vec r,t) = \sum_{ij} f_j(\vec r,t) V^{*}_i(\vec r;t) V^{\dagger}_{ij}(\vec r;t), i,j=1,2 $$

第二个例子（ZnO two electrons）
ZnO是一种单晶有机氯化钙，它是一个比较简单的纳米结构。我们假设ZNO钙分子的基态为两个粒子同时处于激活态，分别位于A和C原子核，它们具有很好的耦合性。

电子初态：由于电子初态只能是激活态，所以我们只需要选择一个电子初始化。我们选取$|\uparrow\rangle$初态，即只有一个原子核处于激活态。

电子运动路径：为了使电子运动路径呈现多重纤维性，我们可以使用多种电子运动路径。最简单的就是将电子运动路径分割为两个子集，一个子集对应于原子核A的两个离子位置，另一个子集对应于原子核C的两个离子位置。在电子运动方向上，电子在A原子核上的概率分布可以用电子基矢$|\downarrow\rangle$表示，而在电子运动方向上，电子在C原子核上的概率分布可以用电子基矢$|\uparrow\rangle$表示。对于离子位置，我们可以设置不同的能级，使得电子运动路径呈现不同的纤维状。

为了控制电子运动路径的多重纤维性，我们可以考虑加入更多电子。例如，对于原子核A，我们可以在A和B离子位置之间加入电子，并将它们视为孤立的局部状态。对于原子核C，我们可以在C和D离子位置之间加入电子，并将它们视为孤立的局部状态。

为了计算电子激发带隧度，我们需要知道电子运动方程中的概率分布。首先，我们可以利用$|\psi(t)\rangle$的哈密顿矩阵$(-i[H,T])|v_c\rangle$求得电子势场$V(\vec r;t)$，其中$\vec r$是离子位置的坐标，$v_c$为电子基矢。然后，利用势场中电子与原子核之间的受力关系，我们可以计算到达每一个离子位置的电子运动路径分布$f_i(\vec r,t)$。从而，可以得到电子激发带隧度的概率分布$P_b(\vec r,t)$，其中$b$表示相应的带隧度。

$$ P_b(\vec r,t) = \sum_{ij} f_j(\vec r,t) V^{*}_i(\vec r;t) V^{\dagger}_{ij}(\vec r;t), i,j=1,2 $$

5. 未来的发展方向
双原子链的电子被激发条件下的带隧杂度测量方法是目前在高精密光学系统领域里的一项热门课题。它既有理论意义又有实际应用价值。虽然目前的方法仍处于早期阶段，但已经取得了一定的成果。作为一种实用的测量方法，双原子链的电子被激发条件下的带隧杂度测量尚需进一步完善和优化。

未来，在方法的设计、计算方面还有许多工作要做。目前，电子运动模型较为简单，也容易受到噪声影响。而电子束多重纤维性的模拟方法还无法完全匹配真实情况，尤其是在厚度很大时。更加关键的是，即使通过对比实验表明了电子被激发条件下的带隧杂度的规律，但是如何自动化、精确地测量出带隧杂度依然是个棘手的问题。因此，未来一段时间将继续努力探索这个领域，提升双原子链的电子被激发条件下的带隧杂度测量方法的准确性和有效性。
