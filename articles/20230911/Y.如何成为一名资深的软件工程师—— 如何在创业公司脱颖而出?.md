# 1.简介
  

　　软件工程师（Software Engineer）是一种与计算机科学、信息技术、经济学等相关专业相互联系但又不同于其他职业的人。他们从事设计、开发、测试、维护和优化应用软件、网站或其他软件系统的任务。无论其目的如何，软件工程师都致力于提升产品质量、改善用户体验、提高企业竞争力、节省资源并实现业务目标。一般认为软件工程师具有高度的专业知识、解决复杂问题的能力、较强的沟通协调能力、能够有效管理他人的技能以及适应新环境的能力。

　　很多创业公司都会招聘软件工程师，但很少有成功创业的公司培养出像样的软件工程师队伍。很多创业公司刚开始兴起时，并不愿意投入太多人力、资金和时间去培养软件工程师，因为对他们来说，做好自己的产品和服务比培养一个完整的软件工程师队伍更重要。然而随着创业公司的发展，一些公司积极寻找软件工程师来为他们提供咨询、设计、开发、测试、运维等一系列服务。

　　软件工程师是创业公司不可或缺的一部分。通过全方位了解客户需求、提升竞争力和效益，软件工程师可以帮助创业公司实现商业模式的转型、扩张，以及赢得竞争者的青睐。为此，本文将分享我多年经验的四个创业公司的经验教训，以及如何成为一名资深的软件工程师，让您在创业公司里取得成功！

　　在开始之前，首先需要说明的是：本文不是一份手册，也没有过多的文字罗列，而是希望作者可以用一种比较清晰的语言呈现给大家有关软件工程师所需的各种技能和知识。为了避免篇幅过长，本文不会对每个领域都详细阐述各项技术，而只会试图从多个视角和角度探讨这些技术到底是什么、为什么要用、如何实践等。因此，建议读者根据自己对软件工程师的要求，结合实际情况，自行阅读相应的书籍、文章或者参阅其他资料。

 

# 2.创业公司的软件工程师应具备什么技能？

　　①项目管理能力：创业公司都处于初创期，重在快速验证市场需求和营收，项目管理能力至关重要。一般来说，项目管理者要掌握包括项目启动、规划、分配资源、监控进度、控制风险、回顾检视、总结反思等流程。
　　②编程语言和工具熟练程度：因为软件工程师的工作主要涉及编写软件的代码，因此必须掌握至少一种编程语言和至少一种常用的集成开发环境 (IDE)。比如，对于 Java 或 Android 程序员，可以使用 IntelliJ IDEA；对于 Python 或 Ruby 程序员，可以使用 PyCharm 或 Sublime Text；对于 C++ 或.NET 程序员，可以使用 Visual Studio 。另外，还需要掌握其他软件工程常用的工具，如单元测试框架 JUnit、日志记录库 log4j、构建工具 Maven 和 Ant 等。

　　③团队精神：创业公司往往是一个小圈子，因此，软件工程师们之间要有良好的团队精神。团队精神即使不是一名优秀的软件工程师都不可或缺，创业公司必须要树立一个积极向上、诚信友爱的氛围。要懂得尊重他人，尤其要尊重那些持怀疑态度的员工。

　　④文档和沟通能力：软件工程师每天都要与多个人员进行交流，并且需要面对不同的语言和背景。因此，除了基本的英文和编程语言，还需要掌握一定的文档写作技巧。同时，还要懂得倾听和理解别人的意见，并把自己的看法表达出来。

　　⑤软件架构和设计能力：在复杂的软件工程中，软件工程师往往被派到项目组之外，独立完成软件架构设计和模块设计等工作。所以，软件工程师必须善于分析并解决实际问题，有强烈的逻辑思维能力、分析能力、设计能力。在做架构设计时，还需要懂得建立团队共识和决策机制，并做好决策记录。
　　⑥敏捷开发能力：软件工程师必须掌握敏捷开发方法，并在项目开发和测试阶段严格遵循流程和规范。不仅要做到产品经理和设计师的设计需求直接影响编码实现，还要知道软件工程的演进过程及如何从上一个版本演变到下一个版本。

　　⑦数据结构和算法基础：软件工程师需要掌握数据结构和算法的基础知识，包括数据结构的选择、性能分析、排序、搜索等算法的实现。如果工作中遇到问题不能立即解决，则可以向专业的算法工程师学习。

　　⑧系统和网络安全：软件工程师应当有系统安全意识和防火墙意识。系统安全意识包括防止攻击、访问控制、数据加密、漏洞修复等安全技术，防火墙意识则指要时刻注意周边的网络攻击和外部威胁，采取相应的防护措施。

　　⑨设计模式和项目管理工具：软件工程师要了解常见的软件工程模式、设计模式，并将其融会贯通，融入自己的工作方式中。Project Management Tool 是用来管理项目的工具，包括禅道、Jira、Redmine 等。这些工具可以帮助项目管理者管理项目并提升工作效率。
