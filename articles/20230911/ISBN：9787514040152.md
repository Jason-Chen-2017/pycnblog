# 1.简介
  

机器学习（英语：Machine learning）是一门多领域交叉学科，涉及概率论、统计学、优化、信息论、 Decision theory等多个学科。是人工智能的核心技术。它的研究重点是设计和开发 algorithms ，利用数据或经验对系统做出反馈，从而使计算机系统可以自动 learn 和 improve 。本书作者对“机器学习”这一术语进行了全面的阐述。他通过系统的学习过程，清晰地阐述了机器学习的历史脉络、技术基础、应用场景和发展方向。并结合实际案例，让读者对机器学习的理论知识和实践能力有一个全面的认识。本书具有理论性很强、实用价值高、广泛适用性的特点。
# 2.导读
“机器学习”作为一个高技术含量的词汇，背后的知识面广阔、涉及面面俱到。阅读本书之前，需要对其中的一些基本概念有所了解。 

首先，机器学习这个名词是由西瓜书（Sammon and Garcia 1998）提出的，而它是从数据中学习和改进，以便于分析、预测或者控制某种系统行为的一种技术。因此，“机器学习”一词往往会被用来形容能够通过训练算法发现数据的模式并应用于其他任务的技术。

其次，机器学习可以分成两类，即监督学习和非监督学习。

监督学习是指输入数据既包括有标记的数据集，又包括没有标记的数据集，也就是说，输入数据的正确输出已经给定。机器学习模型则根据这些数据集来学习如何对新的未知数据进行分类、回归等预测。例如，手写识别就是监督学习的一个典型应用场景。

非监督学习是指输入数据仅包括没有标记的数据集，无标签数据通常是非常杂乱无章、难以划分的数据。此时，机器学习模型需要自己去发现数据的结构、规律，然后基于这种结构、规律将数据分类、聚类等。例如，聚类就是非监督学习的一个典型应用场景。

第三，关于数据集的组织形式，除了监督学习和非监督学习之外，还有强化学习（Reinforcement Learning）。这是因为在强化学习中，环境给予的奖赏不仅仅取决于当前的状态，还取决于以前的状态、动作和影响。因此，需要记录所有历史状态、动作、奖赏等信息，并根据这些信息来确定下一步应该采取的行动。

第四，机器学习技术总体上可分为四个方面，包括监督学习、非监督学习、决策树、支持向量机（SVM）、神经网络。

监督学习最主要的特点是输入数据包括有标记的数据集，同时也包括无标记的数据集。因此，监督学习方法通常都会构建分类器或者回归器。分类器用于对输入数据进行分类，回归器用于对输入数据进行连续值预测。

非监督学习与监督学习相比，最大的区别在于输入数据仅包括无标记的数据集。因此，非监督学习方法通常都采用聚类、降维、模式识别、关联分析等手段来发现数据中的结构和规律。

决策树是一种常用的机器学习技术。它是一个树形结构，每一层的节点表示某个属性的特征，而分支则表明属性的不同取值对应的子区域。因此，决策树可以直观地表达出数据中的复杂的非线性关系。

支持向量机（SVM）是另一种常用的机器学习技术。它是一种二类分类器，可以用来学习线性可分的情况。它通过找到满足 margin 的分割超平面来实现对输入数据的分类。

神经网络（Neural Network）是一种多层次的神经元网络，它可以模拟生物神经系统的工作方式。它可以处理非线性关系、自我纠错、并行计算、自适应权值更新等特性。

第五，机器学习技术能够有效解决很多现实世界的问题。

例如，图像识别、垃圾邮件过滤、序列标注、病毒检测、金融风险管理、人脸识别、新闻推荐等。这些问题一般都是非监督学习、监督学习、决策树、SVM和神经网络组合来解决的。

第六，机器学习是一项长期的技术。

随着深度学习、集成学习、人工神经网络等新兴技术的出现，机器学习正在进入一个新的阶段——万物皆可学习。机器学习将会逐渐发展成为一个综合性的科技领域。随着人们对其理解的加深和实践水平的提升，机器学习将变得更加智能、精准、和有效。
