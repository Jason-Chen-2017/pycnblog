# 1.简介
  


## GitHub简介
GitHub是一个面向开源及私有软件项目的代码托管平台服务。它是基于git版本控制系统的一个web应用程序。GitHub帮助人们管理包括代码、项目文档、任务跟踪、wikis等在内的复杂内容，可以作为一个项目的协作工具，使不同的开发者之间可以一起合作完成工作。目前很多优秀的开源项目都选择了GitHub进行代码托管。GitHub在国内已成为非常流行的IT社区，因为GitHub为程序员提供了方便快捷的分享和协作方式。

2017年GitHub就成为了程序员必不可少的工具之一。去年GitHub被微软收购后，虽然免费提供公共仓库，但是还有一些付费的功能需要额外付费才能使用，而且一直没有很好的替代Git。因此，国内的程序员或技术人员很早便开始使用GitHub，并且越来越多的人开始关注GitHub。所以，学习GitHub对于个人发展、职场发展以及面试中的技巧性题目都有着重要的作用。

3.为什么要学习GitHub？

2017年GitHub被微软收购后，GitHub又一次进入了程序员的视野中，此次微软决定对GitHub不断投入资源和技术支持。其最大的原因在于GitHub帮助人们进行开源项目的协作。GitHub将软件代码变成一个整体，它让更多的人参与进来，从而形成了一个更加美丽和高效的开源生态系统。另外，GitHub是一个专门针对程序员设计的网站，包括用户界面友好、易用性强、功能完整、代码质量高、支持版本控制系统、拓展性强等诸多优点。

GitHub对于个人发展和职场发展都是有利的。由于GitHub为全球程序员提供了优秀的资源和交流平台，使得程序员可以在这个平台上找到志同道合的伙伴。通过参与到开源项目的开发中，可以学习到前人的经验，并结识到志同道合的朋友。

同时，GitHub也是一个极具潜力的市场。由于GitHub对创新和成长环境有着高度的认可度，企业、组织和科研机构等相关部门都在蓄意开辟新的商业模式，尤其是使用GitHub进行大规模开源软件开发的公司。不仅如此，越来越多的企业已经考虑到GitHub带来的商业价值，甚至搭建自己的内部代码托管平台，甚至还在尝试将GitHub用于其自有的产品中。因此，学习GitHub是个人发展、职场发展、和未来的发展方向都有助益的。

总而言之，学习GitHub能够帮助个人提升能力、增加实践经验、获得动力、寻找工作机会、参与到开源项目的开发中，还有很多其他的方面。

4.什么时候适合使用GitHub？

学习GitHub之前，先要明确自己希望使用的Git平台。GitHub和其他代码托管平台一样，都有自己的Git服务器端。如果不打算把代码迁移到GitHub上，那么推荐使用其他代码托管平台，例如码云、Gitlab或者是Bitbucket等。

1）个人项目：如果你只是在学习GitHub，或者你想开始一个开源项目，那么你可以创建自己的公开仓库。

2）小型团队项目：如果你所在的团队成员较少，并且你对Git的操作比较熟练，那么可以使用GitHub作为你的代码仓库。

3）大型团队项目：如果你所在的团队成员较多，并且希望避免分散，那么你需要使用代码托管平台来管理你的项目。

4）需要代码共享：如果你希望与他人分享你的项目代码，或者你需要与他人协作开发某个项目，那么GitHub就是最佳的选择。

5）开源项目：如果你正在参与一个开源项目的开发，那么你只需了解一下GitHub，然后你就可以随时提交你的代码，并获取其他开发者的反馈。

6) 需要帮助：如果你需要寻找协助解决代码中的问题，或者你需要进行代码审查或复核，那么GitHub是一个非常好的平台。
