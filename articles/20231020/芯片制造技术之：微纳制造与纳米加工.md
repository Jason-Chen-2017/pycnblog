# 1.背景介绍


## 什么是芯片？为什么要做芯片？
电子器件、集成电路(IC)、芯片的出现促进了科技发展。从十六世纪末到二十世纪初，科学家们发现了很多能够提升计算机性能和效率的方法，譬如采用晶体管集成电路等。但随着信息技术的发展，需要解决的问题越来越复杂，为了更好地应对需求，工程师们开发出了新的电子器件，如LED、微处理器、传感器、驱动器等等，这些新型的电子器件带来了巨大的市场效益。此外，随着电子技术的飞速发展，单个芯片的面积也越来越小，纳米级、微型的单片机已经成为一种新型的电子装备。因此，在这个时代，芯片制造似乎也成为一个重要的科研方向。
## 为什么要做芯pix制造呢？
在今天的消费电子领域，除了具备良好的消费性外，还有一个非常重要的用途就是作为基础设施的组成部分，无论是在互联网、智能手机、航空航天等领域，都将芯片视作一个重要组件。因此，做芯片制造并不是一件简单的事情，它需要有较高的专业知识、丰富的经验、严谨的工作态度、强烈的创意欲望和持续不断的改进能力。只有把握好它的精髓，才能打造一款上乘的芯片，为人类共同的利益做出贡献。
## 微纳制造与纳米加工
微纳和纳米，都是指微小而功能健康的物质，微纳制造指的是利用微生物、氢等微型材料合成微型设备，比如乙醇键盘、摄像头、激光打印机；而纳米加工则通过冷冻、低温或者其他特殊工艺过程把微型的材料制成厚实的物体，即纳米。如摩尔定律、计算机功耗降低等，是两种截然不同的产业模式，其特性又有许多区别。那么，微纳制造和纳米加工又有何不同呢？我们来看下图：
微纳制造：使用微型材料、微生物、氢等等来制造小型设备，如氧化锌键盘、激光打印机、微型磁铁等，具有单一用途和可穿戴性优点，可以应用于某些特定领域或应用场景中。

纳米加工：利用冷冻、低温或者其他特殊工艺过程把微型的材料制成厚实的物体，即纳米，如电脑组件、路由器、耳机、显卡、嵌入式产品等。具有较高的集成度、精密度、耐腐蚀性、复原性、对温度变化敏感等特点。由于纳米结构的厚度很大，所以可以应用于核武器等安全关键系统。

二者各有优劣，怎么选择呢？首先，对于绝大多数普通人来说，微纳制造比较容易理解和接受，因为它满足日常生活中的需求，而且可以快速搭建起来，价格也便宜。但如果想做出一些高端外观的产品，如高端显示器、智能手表、照相机、机器人等，就需要进行一些比较高级的设计了，这种情况下，就需要转向纳米加工了。另外，纳米制品的大小比微型制品小得多，所以其成本也会高一些，但由于其结构的厚度和强度，也更难破坏，所以还是值得投资的。
