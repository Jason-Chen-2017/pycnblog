# 1.背景介绍


## 一、目标层级化现象与影响
在管理决策中，一般采用层级化的方式进行决策，即上至总目标、下到具体工作目标。管理者将目标分解成更小粒度的子目标并按顺序排列起来，逐步达成目标。如大目标是建立企业的营收增长百分比，细则包括：
- 提高产品质量，从而提高收入增长百分比；
- 拓展业务范围，从而增加运营成本；
- 为员工提供更多的培训机会，提升工作能力。
这样，管理者先着力于提高产品质量的目标，再依据产品质量，调整其他目标的优先级。这就是目标层级化的过程。目标层级化可让目标管理更容易形成一个清晰的任务目标，且能够给出明确的指导方向。
目标层级化对工作效率也有显著的促进作用。一方面，层级化的目标可以减少层与层之间的沟通交流，降低管理难度；另一方面，层级化的目标又能激励团队成员自我要求，使其具有动手能力、自主性，从而取得更好的工作效果。
## 二、如何设置目标层级化？
层级化的目标设置方法与多元化产品开发的方法类似。首先，确定目标的主要目标。其次，分析主要目标的内在联系，揭示它们之间存在的规律关系或相互依赖关系。然后，分析内在联系，把它们转化成为相应的具体目标。最后，根据各个具体目标的重要程度，安排它们在层级顺序中的位置。例如，“建立企业的营收增长百分比”作为一级目标，其下设的三项具体目标分别是：提高产品质量，拓展业务范围，为员工提供更多的培训机会。以上三个目标可以按照重要性递进的顺序放在一级目标的下方。层级化目标设置可以有效地组织工作目标、促进信息沟通、提升工作效率等。
## 三、目标层级化的优点和局限性
### 1.优点
目标层级化的好处主要体现在以下几方面：
- 可协同管理：层级化目标使得不同部门的目标相互支撑，从而可以更快地实现目标的达成；
- 有助于提高工作效率：层级化的目标对工作流程的制定有所帮助，可以大大简化工作过程，节省时间、消耗资源；
- 可以促进信息沟通：层级化的目标可以有效地组织目标任务，并在各层之间传递信息；
- 有利于目标的目标驱动：层级化的目标设置可以激励员工独立完成目标，并提升工作绩效。
### 2.局限性
但是，层级化目标也存在一些局限性。一是层级化的设置不一定适合所有企业，另一方面，层级化设置可能导致目标滞后，导致效益下滑。此外，层级化设置还可能会过度细化目标，导致目标管理变得复杂。因此，为了达到最佳的目标管理效果，企业需要多种因素综合考虑，充分利用人力、物力、财力资源。因此，建立一套全面的目标层级化规范并落实到工作岗位上，是成功的关键。
