# 2.核心概念与联系
## 2.1 一份好的简历应该什么样子
首先，一份好的简历应该简洁明了，突出重点并快速反应应聘者的求职意向。其次，应该让应聘者了解到自己擅长的技能，以及公司所需要的专业知识。再者，推荐应聘者能加强相关的基础知识学习，以及对工作中用到的知识点进行深入理解。最后，还要注重于实习经历、项目经验等各个方面，充分体现应聘者的综合能力。如果将这几点做好，那么简历就会成为应聘者很好的榜样。
## 2.2 为什么要写好简历
简历是一种很好的展示个人能力以及履历的方式，能够吸引到更多的机会和资源。因此，好的简历能够帮助求职者快速地定位到符合自己要求的工作岗位，并顺利进入到该公司。所以，写好简历，是提升求职效率的有效手段。
## 2.3 如何优化简历结构及内容
简历结构上需要满足求职者需求，从头到尾都包含有丰富的个人信息，但也不能过于简单，简历中也不能太过庞大。结构上要突出个人经历、技能特长和教育经历三个大的模块，而且每个模块之间要有清晰的界限。模块内部则是按优先级排列，要求应聘者描述自己擅长的技能及领域。对于教育经历模块，应聘者可以根据自己的实际情况进行删减或补充，这样才能真正体现自己的能力。
## 2.4 有哪些工具或技巧可以帮助写好简历
写简历是一项耗时且费力的任务，可以通过一些工具或技巧来帮助简历的制作更加容易、准确。比如，可以通过简历生成器、Word模板等工具快速制作个人信息表格；也可以采用手机端的简历编辑软件或网站；还可以利用在线简历分享平台发布简历，获取志同道合的好友帮忙修改；另外，应聘者也可以使用其他技术交流平台如微信群、QQ群等进行沟通。总之，写简历不是一件易事，只要坚持努力，最终都会有所收获。

## 3.1 SIFT算法
SIFT（Scale-Invariant Feature Transform）算法是由周志华教授提出的，主要用来检测图像特征。它的处理流程如下图所示：


SIFT算法包含三个阶段，第一阶段是尺度空间金字塔(scale space pyramid)，用于建立图像不同尺度的特征。第二阶段是关键点检测，通过高斯差分滤波器计算梯度幅值和方向角度，选取响应强的点作为关键点。第三阶段是特征描述，将每个关键点对应的特征向量转换成一个定长的向量。

作者认为，SIFT算法的实现复杂度比较高，而且关键点检测过程需要依赖高斯差分滤波器，这使得它在速度和精度上都有缺陷。为了降低计算复杂度和提高准确性，作者设计了一个快速算法KLT(Kernel Local Tracker)。KLT算法通过建立高斯核函数来代替高斯差分滤波器，提高了关键点检测的速度。同时，还将匹配过程改进为局部匹配，解决了关键点检测的盲目性。下面小编用数学语言描述一下作者所述的KLT算法的具体操作步骤以及数学模型公式。

## 3.2 KLT算法
### 3.2.1 模板匹配
模板匹配算法是一种在两张图片或者多张图片中查找相似区域的算法。一般来说，模板匹配算法包括两个部分：搜索和匹配。搜索就是从一副待检索图像中找出一个与模板最匹配的位置，而匹配就是判断两张图像是否真的拥有相同的结构。模板匹配算法一般分为三种类型：
1. 自相关算法：自相关算法是指把待查样本点（模板）分布在图像中任意位置，然后在图像的周围区域滑动一定的步长来对比待查样本点与图像的像素值的差别，从而确定图像中的目标区域。自相关算法最大的问题是在图像边缘处无法找到足够一致的模板，影响匹配结果的稳定性。
2. 平移不变性：平移不变性是指图像中的目标区域即使发生平移，它的匹配结果也不会变化。
3. 尺度不变性：尺度不变性是指图像中的目标区域的大小不变，而只是改变它的形状。
### 3.2.2 KLT算法
KLT算法（Kernel Local Tracker）是作者设计的一种快速模板匹配算法。与传统的模板匹配算法一样，KLT算法的目的是寻找一张图片中的目标区域。但是与传统的模板匹配算法不同的是，KLT算法采用高斯核函数来代替高斯差分滤波器，以提高关键点检测的速度。具体的操作步骤如下：

1. 使用窗口来框定图像中的目标区域，选择高斯核半径r。
2. 对窗口中的每一个像素点，计算其与邻域内的像素值的差别（公式见下文）。
3. 如果某个像素的差别超过一个阈值T，则认为这个像素是一个关键点，记为KP。
4. 在窗口外寻找另一个目标区域，并重复步骤2-3直到所有的关键点被确认。
5. 应用RANSAC算法来找到目标区域的局部最优。

其中，公式表示当前像素点的差别为：

δ = (I(i+1,j)-I(i,j))^2 + (I(i,j+1)-I(i,j))^2 - 2*g(i+1,j)*g(i,j) 

其中，I(i,j)是当前像素的值，(i+1, j), (i,j+1)是邻域内的相邻像素，g(i+1,j), g(i,j)是高斯核函数。g(x,y)=exp(-r^2/2*(x^2+y^2))，r为选择的高斯核半径。

KLT算法的速度快且准确率高，同时又不需要对图像的平移和尺度进行特殊处理，因而可以在图像中任意位置搜索模板。

## 3.3 LBP算法
LBP算法（Local Binary Pattern，局部二值模式）是一种用在图像二值化中的人工神经网络方法，用于检测图像的局部模式。它与SIFT算法的功能类似，都是用于特征点检测。与SIFT算法一样，LBP算法也是基于特征点检测的，但它的计算代价更小，所以速度更快。下面小编用数学语言描述一下LBP算法的具体操作步骤以及数学模型公式。
