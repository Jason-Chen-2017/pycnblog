# 1.背景介绍


“世界上最昂贵的材料”（费米氏定律）认为，物质自然界中拥有绝对价值的只有两个元素：费米子和夸克，但它们都是无法制成工艺品或可以轻易被复制的。相反，由人类掌握的技能、组织和工程实践所产生的低纳米级材料才是真正的商品，而这些材料的制造技术依赖于化学的智慧、手段和设备。化学机械抛光技术（Chemical Mechanical Artillery）与其他现代电子芯片制造技术一样，也属于一种“旧瓶装新酒”，在过去几十年间被广泛应用到各种电子电路、集成电路、半导体等各个领域。今年下半年，芯片制造技术在国内迎来了一个全新的发展阶段——“电芯”（IC Design），这个阶段包括了最新世纪交叉学科，如材料科学、机械设计、信息技术、生物工程、通信工程、生态环境、人工智能、量子计算、机器学习、深度学习、量子纠缠、超导、传感器与认知、芯片组装等多个学术研究领域。虽然芯片制造技术依然依赖化学的一些原理和方法，但是由于其成本和高时延要求，它们已经越来越少地被用于制造消费电子产品，而更多地被用来创造“电芯”产品。近几年来，越来越多的企业和个人开始关注电芯片的研发。因此，制作具有无可比拟性的高性能处理能力和扩展性的芯片已成为许多企业、政府部门、军事部门和科研机构的核心竞争力，也是人才培养和产业革命的关键。本文将从化学机械抛光技术出发，为读者提供电芯领域的知识指导。
# 2.核心概念与联系
化学机械抛光技术（CMA）是利用化学反应的方法制备平坦型、微孔型或卷曲型的化合物阵列作为芯片厚度（p/e值）的一种应用技术。通常情况下，CMA 是用在多种工艺领域，例如微电子设备、集成电路、显示屏、玻璃制品、硅酸盐膜、金属制品、磁性材料、光刻胶、电池、耐膜材料等方面。CMA 原理主要由四个要素组成：激发剂、介质、分离剂和溶剂。
- 激发剂: 一般采用六氟乙烯（HFC-62）、六氟醇（HFE-64）、三氟甲烷（CFC-12）、三氟乙烷（CCl_4）等可溶性气体和气体混合物。
- 介质: 是化学物质、染料、溶剂和精华在短时间内反应而得出的化学产物。介质主要由碳、氮、硫、氢、钾、铊等元素组成。
- 分离剂: 一般采用碱或酸溶液来分离激发剂与介质之间的杂质。
- 溶剂: 是进一步分离介质中的重水、气泡、沉淀和残余物质的物质。
CMA 技术可以应用于各种工艺领域，例如芯片制造、纳米材料、石墨烯、太阳能板等领域。它的主要优点包括：
- 可以制备小尺寸、高度集中的芯片，具有极佳的可靠性和灵活性；
- 不受电容器的限制，在过渡期可以维持高产出率；
- 可降低芯片制造成本，提升成果转化效率；
- 有利于提升产能效益和经济效益；
- 能实现对外开放的产品模式。
同时，CMA 也可以被应用于现代信息技术领域，例如网络安全、智能网关、云计算、边缘计算、手机芯片等。在信息技术中，CMA 的运用有利于提升性能、可靠性和成本效益，有效减少重复制造的损耗。CMA 还能够满足信息系统的高速传输、高容量存储需求。
# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## （一）模糊切割法：什么是模糊切割法？如何进行模糊切割？
模糊切割法，又称为投影模糊切割法（Projected Ablation Technique），是一种用于制备高纤维度的化合物阵列的电镀工艺技术。它可以用来制备不同形状的平坦型或微孔型电子元件、集成电路（IC）和印刷电路板。模糊切割法分为模糊切割和模糊填充两种操作过程。
### 模糊切割（Ablation Cutting）
模糊切割是在原有的电子学结构基础上加入化学胶水，然后进行电镀切割。由于电子学结构具有良好的功能性、封装性、隔热性及适应性，故在电镀过程中可以使微电子片段聚集成簇，并有助于保持微电子片段的稳定性。但是如果微电子片段聚集在一起就会导致电镀厚度过小，甚至导致电子层中缺陷。所以需要通过电镀前的模糊切割过程来解决这一问题。模糊切割有两种基本方式，即层切割和轴切割。
#### 1.层切割
层切割即按照一定层数，把杂质物质从电子学结构中移走。把电子学结构切割为若干层，每一层上都加上一个特定的化学胶，形成具有不同浓度的化学胶胶水，使得该层上的电子学分子被带走，并且逐渐把电子学结构中的杂质物质带走。
#### 2.轴切割
轴切割即按照一定方向，把杂质物质从电子学结构中移走。首先在结构两端（表面）加上多个电子隔板，然后用沉积液把微电子片段分散到周围空间，这样就可以逐渐把电子学结构中的杂质物质带走。
### 模糊填充（Ablation Filling）
模糊填充是一种在模糊切割的基础上增加电子粘结材料的电镀工艺技术。经过模糊切割后，会留下微电子层，这部分微电子会聚集在一起，形成单一的电子单元。为了使微电子层保持平滑，可以用高纯度、高摩尔数的电子粘结材料做为电镀材料，让电子粘结到微电子层的表面，形成一个完整的电子单元。电子粘结材料可以是小颗粒状的、菱形的、环状的或者蛋白状的。
### 切割过程
模糊切割包括以下几个主要步骤：
- 提炼杂质：先对待切割的电子学材料进行剥离，挖除杂质物质，使杂质物质从电子学结构中完全移走。
- 化学加工：加入适量的化学胶和清洁剂对切割过后的杂质物质进行处理，消除多余的化学物质，保持切割过后的电子学材料具有较高的纤维度。
- 电镀加工：在离子激发下对切割过后的杂质物质加热到合适的电流，使切割过后的电子学材料及微电子聚集，形成平坦型、微孔型、卷曲型的电子元件。
- 水平切割：把切割完成后的电子学材料，通过制作电子密封层，做成不同厚度的微电子片段。不同的厚度的微电子片段堆叠在一起，就成为了最终的模糊切割完成的电子元件。
- 垂直切割：在模糊切割过程中还可以通过垂直切割的方法，把电子学结构的杂质物质都移走。由于微电子片段之间距离有限，垂直切割是为了缩短电子之间的距离，减少微电子片段之间的冲击力。
- 回切：在切割过程中，如果出现电子学材料断裂，或电子学结构本身存在缺陷，则需要用附着剂或者其他介质来回切补齐缺失的部分，确保整体电子学结构的完好性。
## （二）小批量多步切削法：什么是小批量多步切削法？如何进行小批量多步切削？
小批量多步切削法，也叫分块切削法，是一种较为复杂的、高度自动化的切削方法。它允许在单次切削中，对许多不同形状的小型电子元件进行切削，从而节省切削过程中的人工劳动和资源消耗。小批量多步切削法由若干小批量的切割计划组成，每一次切割计划都会切除某些特定形状的电子元件。在整个切削流程结束之前，所有的小型电子元件都不会完整的切割完成。每个切割计划都是一个独立的工作流，允许实时跟踪和监控切削过程的进度。
### 小批量切削（Microbatching）
小批量切削是指在连续切削过程中，采用多项切削计划。在单次切削过程中，对许多不同形状的小型电子元件进行切削，而不是一次切除一个电子元件。这种切削方法可以在短时间内完成大量的电子元件的切割工作，节约了人工切削的时间和资源。
### 多步切削（Multi-pass Technique）
多步切削是一种高度自动化的切削过程。在不同形状的电子元件中，每一次切削都会考虑很多因素，包括电压电流、位置和形状。多步切削技术根据不同形状的电子元件及其特点，制定了不同的切削顺序，从而可以获得不同程度的切割精度。不同的切削顺序意味着不同的切削速度、宽度、厚度和剪切倾向。多步切削法在优化切削精度的同时，还可以节省切削时的生产成本。
### 切削过程
小批量多步切削法包括以下几个主要步骤：
- 零件定位：在预先设计好的切割工作中心中找到待切割零件的准确位置。
- 零件预加工：将零件进行精细的加工，以保证零件的完美切割。
- 初始切削计划：按照实际情况，制定初始切削计划，确定切割工具、切削工艺参数等。
- 连续切削：连续切削一直进行，直到所有零件均完成切割。每个切削计划都会切除某些特定形状的电子元件。
- 数据分析：分析切削结果，识别出哪些零件切割效果不佳，针对性调整切削计划。
- 最终修饰：根据切削结果进行最终的修正，消除切割过程中的瑕疵，使得电子学元件的大小、形状和位置达到一致。
## （三）高速抛光技术：什么是高速抛光技术？如何进行高速抛光？
高速抛光技术，也称为动态光学、极化光学（Field Light Technique）、超声波抛光技术（Ultrasound Foil Technology）等，是一种用于制备高纤维度、高精度电子芯片的技术。它可以实现对硅材料、石墨烯、太阳能板等材料的高速抛光，这种技术可以应用于电子电路、集成电路、显示屏、嵌入式系统、元件、电池、耐膜材料等领域。抛光工艺包括电阻式抛光、电容式抛光、场效应式抛光等，不同的抛光工艺都可以实现不同类型的平坦电子元件的制备。目前，国内外还存在许多关于高速抛光技术的研究和进展。随着相关领域的发展，高速抛光技术也正在成为制备高纤维度、高精度电子芯片的新引擎。
### 电阻式抛光
电阻式抛光是一种实现高度精度和快速制备的轻量化高速抛光技术。它是最早采用高速抛光技术的材料，主要用于制备微电子芯片的电子电路和集成电路。电阻式抛光技术的原理是将一种有机化学胶（称为抛光胶）或者其他表面活性化学反应生成的化学物质，用电阻器进行固定。电阻器具有电荷聚变作用，在固定后产生离子之间的相互作用，从而产生长宽不同，大小不同，颜色不同的电子束，这些电子束击穿电阻器中的电荷，并吸收在其内部释放的信号。
### 电容式抛光
电容式抛 LIGHT Capillary Throwing) 又称精谱光纤抛光，是一种利用电容引起场效应的高速抛光技术。它不需要使用任何干扰剂，在表面电子纳米管中的存在电子的位置，引入一个电容，使得材料能够受到微观影响。此时，引起的场效应作用使得电子纳米管中的电子受到紫外线影响而被切割，从而实现材料的高速抛光。电容式抛光技术已经得到了广泛的应用，在制备微电子芯片时，已经成为一个重要的组成部分。
