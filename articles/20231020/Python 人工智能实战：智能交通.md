# 2.核心概念与联系
## 2.1 数据类型及对应处理方式
- **结构化数据**（Structured Data）：结构化数据的特点是每个值都有一个清晰的定义和明确的含义。它往往带有表格或者关系数据库的结构化特性。例如，信用卡交易历史记录就是结构化数据，包含了买卖双方的身份信息、交易日期、金额、商品种类、交易凭证号码等字段。
- **非结构化数据**（Unstructured Data）：非结构化数据往往难以直接处理，需要进行一定的数据预处理才能得到有意义的结果。主要包括文本数据、音频、视频、图像、电子邮件、社交网络、电子商务网站的网页等。例如，网页上的文本、评论、图片、视频等就是非结构化数据。
- **半结构化数据**（Semistructured Data）：半结构化数据具有多样性的结构，同时还有部分数据缺失或异常。例如，XML和JSON数据都是半结构化数据。
## 2.2 基本机器学习术语
- **输入变量**（Input Variable）：模型的输入变量一般包括特征、属性、特征向量或者样本。这些变量用于描述某个事物的特征或状态，用于表示模型的训练和测试数据。
- **输出变量**（Output Variable）：模型的输出变量通常是一个预测变量，用于描述某个事物的变化或行为。例如，预测销售额的模型可能有一个输出变量“销售额”，根据给定的销售额、价格等输入特征，模型可以预测出相应的销售量。
- **特征**（Feature）：特征是指对输入变量的一个测量或描述。特征向量由多个特征组成，向量中的每一个元素代表了一个输入变量的值。特征向量用于表示输入变量之间的联系，是机器学习模型中非常重要的一环。
- **标签**（Label）：标签是在训练数据中存在的真实值，用以标记训练样本。标签是机器学习中非常重要的概念之一，其作用是使得模型能够正确的拟合数据，并得到一个可以用来预测的输出。
- **假设空间**（Hypothesis Space）：假设空间是由所有可能的函数或模型组成的集合。在许多情况下，假设空间很大，难以找到全局最优解。因此，我们通常采用启发式的方法，搜索局部最优解。
- **训练集**（Training Set）：训练集是包含训练数据的集合，是模型训练过程中的一部分。模型通过训练集学习到输入和输出之间的映射关系，使得在其他数据下也可以预测出相应的输出结果。
- **测试集**（Test Set）：测试集是用于评估模型性能的集合。模型在训练过程中通过测试集来确定模型的泛化能力。如果模型在测试集上的表现不好，则可以通过调整模型参数或重新训练模型来提升模型性能。
- **泛化误差**（Generalization Error）：泛化误差是衡量模型的泛化能力的一种指标。泛化误差可以用来衡量模型的过拟合、欠拟合或无偏误差。过拟合的模型不能很好地泛化新数据，而欠拟合的模型在训练集上表现优异，但在测试集上性能却不佳；无偏误差的模型会低估测试误差，导致过高的泛化误差。
## 2.3 普通最小二乘法与岭回归
普通最小二乘法（Ordinary Least Squares Regression，简称OLS），也叫做最小平方法，是一个简单而直观的线性回归方法。OLS方法利用最小化残差的平方和作为损失函数，通过求解这个损失函数极小值的过程获得回归系数。岭回归（Ridge Regression）是OLS的改进版本，在求解过程中加入了对角权重矩阵，使得只有少数变量的权重不为零。
## 2.4 核函数与支持向量机
核函数（Kernel Function）是一种将原始特征空间映射到高维空间的函数，它的本质是计算两个输入向量的相似度。核函数的目的是扩充输入空间的维度，使得支持向量机能够利用非线性关系来进行分类。支持向量机（Support Vector Machine，SVM）是一种监督学习算法，它利用非线性分类函数对数据进行间隔最大化的分割。
## 2.5 深层神经网络与递归神经网络
深层神经网络（Deep Neural Network，DNN）是机器学习中应用较多的深度学习模型，它由多层感知器组成，每一层都由若干个神经元组成。DNN可以在输入层、隐藏层和输出层之间加入更多的隐含层，使得模型的表达力更强。递归神经网络（Recurrent Neural Network，RNN）是一种特殊的深度学习模型，它通过循环连接的方式处理序列数据，并能够记住之前发生的事件。
## 2.6 遗传算法与进化策略
遗传算法（Genetic Algorithm）是一种模拟自然界进化的算法，它利用随机生成的初始种群逐渐演变为优秀的适应者，最终形成一个较优的解决方案。进化策略（Evolution Strategy，ES）是遗传算法的一个变体，它通过反复试错的方式寻找最优解。
## 2.7 决策树与随机森林
决策树（Decision Tree）是一种流行的机器学习算法，它通过树状结构的分枝，一步步的划分数据，直至叶子节点的分类结果。随机森林（Random Forest）是基于决策树的集成学习算法，它在构建每棵树时采用了随机选择的特征和切分点，以降低模型的方差，增强模型的鲁棒性。
## 2.8 蒙特卡洛模拟与贝叶斯统计
蒙特卡洛模拟（Monte Carlo Simulation）是一种广泛运用的数值计算技术，它利用随机数生成一系列可能的结果，并统计各种可能结果出现的概率。贝叶斯统计（Bayesian Statistics）是一种关于概率的理论，它利用先验概率和后验概率以及条件概率来计算联合概率。
