# 1.背景介绍

虽然期权交易所平台对期权的价格波动率较高，但由于需求增加以及美元贬值引发的利差收益减少，期权市场仍然受到广泛关注。那么，作为一名技术专家、程序员和软件系统架构师，如何参与到期权交易平台的行列？又该如何参与并积极提高交易成功率呢？

# 2.核心概念与联系
## 什么是期权？

作为一名期权交易者，首先应该清楚什么是期权。期权是一项法律赋予的权利工具，它规定了双方在特定时间内（通常是几个月甚至几年）能够取得特定价格的特别待遇。期权由两方面组成：被保护人和保障人。被保护人的权利被授予第三方——保障人——来执行。目前，在中国，保障人往往为券商，即将受益人收取一定比例的股息。

期权可以分为看涨期权（Call Option）和看跌期权（Put Option），如下图所示：

* 看涨期权：期权保障人承诺以保底价或溢价的方式向被保护人支付的加上固定成本的现金。如果在期限到达之前被保护人的价格超过了发起人设定的涨幅，则可以购买到期权证券。被保护人按照约定的价格向保障人缴纳保证金，保障人收到后，按照约定的时长归还债务，并按同等份额分派利息。
* 看跌期权：期权保障人承诺以保底价或溢价的方式向被保护人支付的加上固定成本的负债，并愿意按照约定的价格付出的代价来赔偿被保护人。如果在期限到达之前被保护人的价格低于了发起人设定的跌幅，则可以购买到期权证券。被保护人按照约定的价格向保障人缴纳折价准备金，保障人收到后，按照约定的时长不偿还债务，并按同等份额扣除利息。

## 为什么要参与期权交易？

一般而言，参与期权交易有很多好处。

1. 有爆仓风险，利用期权策略，可以获得比较好的收益率回报；
2. 可以减少交易成本，节省资金；
3. 增强个人能力，提升金融知识水平，积累实战经验；
4. 有丰厚的收益，能够短时间内积累相当于人民币十万左右的收益；
5. 有更多的选择，比如高息期权、跨市场交易等。

除了这些正向收益，参与期权交易还会有以下一些不利因素：

1. 不可预测性，持仓的风险和收益都是在不断变化的；
2. 对投资品种敏感度高，每一个品种都需要考虑手续费，套保风险；
3. 投机性很强，容易产生心理上的波动，出现亏损的情况；
4. 交易时间段太短，影响交易效率。

但是总体来说，期权交易是一项比较复杂的理财产品，也是最有吸引力的金融投资。因此，作为一名期权交易者，必须有较强的知识背景和实战能力，同时对期权市场的走势非常敏锐，具备极大的耐心和坚韧精神。
