# 1.背景介绍


　　对于世界上所有的人来说，都无法完全摆脱经济学的影响。不论从事哪种职业、生活在什么样的国家、经历过怎样的磨难，在最后的一生中，都离不开经济学。没有经济学作为支撑，人的生活就无法正常运行。作为“阶级斗争”的产物，经济学理论是影响一切的枢纽，它左右着每个人的人生轨迹。从古至今，经济学的理论贯穿于人类历史和现实，它呈现了自然界、社会、制度等众多方面的相互作用。其基本原理是什么？如何应用？在一个开放、竞争的市场环境下，如何进行有效的决策？这些问题，如果不能正确理解，那么就可能付出沉重的代价。

　　20世纪70年代末，美国经济学家威廉·马歇尔提出的“古典均衡”理论，成为众多经济学家的共识。它认为，“古典均衡”状态指的是一种静态的、无外部干扰的经济秩序，此时的经济活动可以被看作是不断地进行的斗争——越是面临困境、压力越大的各个阶层之间就越要竞争，最终形成一种合作关系。为了达到这种均衡状态，政府和市场都需要高度的协调配合，既不能过度干预市场，也不能过度依赖政府，而应该充分尊重个人权利、保障自主选择的权利。当经济处于“古典均衡”时，个人的行为就不会产生积极或消极的影响，社会的运作也将朝着正确方向演进。

　　“古典均衡”理论曾经成为经济学的一个基础性概念，其认知理论基础在于对经济现象及其原因的广泛研究。同时，随着近些年来经济学理论的日益复杂化、理论实践的不断完善，“古典均衡”理论的许多原则也逐渐被新的经济观点所代替。不过，由于其仍然具有现实意义，“古典均衡”理论和其他经济学理论并不断相互激荡。然而，“古典均衡”理论始终被一些学者视为一种“终极理论”，意即只有在这个状态下才能真正解决所有实际问题，而实际上，在市场经济条件下，各种问题都还有待解决。所以，如何找到一条可行的路径，走出“古典均衡”陷阱，以及如何更好地应用经济学原理，是当前经济学研究者应当继续关注的课题。

　　20世纪90年代中期以后，随着信息技术的飞速发展和大数据应用的兴起，计算机科学和统计学逐渐成为经济学领域的重要组成部分。特别是在中国这样的大国，经济学家们发现数字技术在促进生产力升级、经济增长和社会发展方面的潜力已经越来越强。利用计算机模拟和计算技术，经济学家们开始认识到信息技术对经济发展的直接影响。他们设计出了一系列的模型，用于描述信息技术对经济的影响。如信息经济学（IE）模型、凯恩斯全球贸易模型、奥巴马全面建设小康社会模型、网络效应模型等。这些模型揭示了信息技术带来的巨大变化。例如，通过互联网技术，经济活动的参与者可以获得更多的信息，从而实现信息披露、流通和共享，使得经济活动更加透明、公平、多元。

　　另外，随着产业结构变革、服务业的普及，传统经济学家也开始重新审视经济领域的弊端。在这种情况下，新型经济学家试图寻找新的方法来解释经济现象，探索经济制度设计与管理中的新模式。美国经济学家拉尔森·萨缪尔森提出的“新古典经济学”试图构建一个基于市场的经济系统，其理念认为，传统经济学是借助政府规定的政治制度来规范经济活动，因此会导致资源配置效率低下、社会风险加剧、产业结构失衡等弊端。他提出“非市场”范式，将计划经济的观念和市场经济的观念分开，允许企业在自己的内部进行自由选择，从而避免了政府管控带来的弊端。

　　然而，如何应用经济学原理，走出“古典均衡”陷阱，以及如何更好地解决实际问题，则是更加重要的问题。如何利用经济学的理论来引导行动，是值得我们思考的问题。
