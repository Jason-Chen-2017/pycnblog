# 2.核心概念与联系
## 什么是Python？
Python 是一种高级编程语言，被设计用来进行可移植性编程、交互式编程、面向对象编程、动态编程和并行计算。Python 的语法简洁而直观，能够有效提升程序员的工作效率，被广泛用于科学计算、Web开发、自动化运维、网络爬虫、人工智能等领域。

## Python和其他语言的比较
下表列出了 Python 和其他编程语言之间的主要区别：

1. 易学易学：Python 比较容易学习，因为它是一种简单的、易于理解的语言，并且拥有丰富的第三方库。
2. 自由开源：Python 是一个开源项目，它遵循 GPL(GNU General Public License)协议，你可以随意免费使用该语言。
3. 性能好：Python 是一种解释型语言，它的运行速度很快，可以胜任那些需要快速执行的任务。
4. 可移植性好：Python 支持多种平台，你可以在 Linux、Windows、OS X 等不同平台上使用 Python。
5. 大数据支持：Python 提供强大的第三方库，可以方便地处理大数据，包括统计分析、数据挖掘、金融建模等。
6. 社区活跃：Python 有着丰富的第三方库和大型的用户群体，其中包括很多优秀的 AI 和数据科学家。
7. 解释器友好：Python 可以直接在命令行中执行，并且提供丰富的交互式环境，可以直接输出结果。

## 数据结构
数据结构是计算机中组织、存储和处理数据的形式。数据结构是建立在计算机硬件和操作系统基础上的抽象概念，通过对数据元素之间的关系进行分类，数据结构可以极大地提升计算机运算效率、降低资源消耗、提升数据访问速度和存储容量。

Python 中常用的数据结构如下：

1. List: 列表，它是一种有序集合，可以按索引访问元素，每个元素可以存放任意类型的数据，可以动态调整大小；
2. Tuple: 元组，它是一种有序集合，和列表类似，但是不可修改；
3. Set: 集合，它是一个无序集合，元素之间不允许重复，不能有相同的值，可以使用add()方法添加元素，或者使用花括号{}创建集合；
4. Dictionary: 字典，它是一种键值对的集合，键必须唯一，值可以重复，通过键可以获取对应的值，可以通过key=value的方式添加或者更新元素。

## 模块
模块是一系列Python代码的集合，它们一般以.py 为后缀名。模块的作用主要是实现某一功能或解决某个问题，比如，你就可以把一些日常经常要用到的函数、类封装成一个模块，然后导入到当前程序中调用。在Python中，模块就是包含了Python代码的文件。

## 函数
函数是指接受输入参数，执行特定功能的代码片段，并返回输出结果。函数通常分为内置函数和自定义函数两种。

1. 内置函数：Python 语言提供了许多内置函数，可以直接使用，例如 print() 函数用于打印输出，input() 函数用于接收用户输入，len() 函数用于获取对象的长度。
2. 自定义函数：也可以自己定义函数，函数的定义格式如下：

   ```python
   def function_name(parameter):
       # 函数体
       return result
   ```

   - 参数：函数的参数可以是零个或多个，多个参数之间用逗号隔开，每个参数都有一个默认值，可以省略，如果没有默认值则必须加上。
   - 返回值：函数的返回值是函数执行结束之后得到的结果，函数执行完毕后，可以通过 return 语句返回结果给调用者，调用者可以通过接收这个返回值的方式获取函数的执行结果。

## 异常处理
异常处理是程序运行过程中发生错误时，捕获错误信息并作出相应的处理的过程。在 Python 中，可以使用 try...except...finally 来进行异常处理。try 子句用来检测错误，如果没有错误发生，则执行 except 子句中的代码；如果错误发生，则执行对应的 except 子句中的代码。最后，如果没有错误发生，则执行 finally 子句中的代码。

```python
try:
    # 可能产生异常的代码
except ExceptionName as e:
    # 如果产生了ExceptionName类型的异常，则把其绑定到变量e
    # 此处可以自定义对该异常的处理方式
else:
    # 没有产生异常的时候执行的代码
finally:
    # 不管是否异常都会执行的代码
```
