# 2.基本概念术语说明
## 2.1 数据可视化概述
数据可视化（Data Visualization）是将数据转化成具有图像特征的信息的过程，主要目的是为了更直观、更生动地展示信息。数据可视化是指通过不同形式的图表、图形、图像、地图、比例尺、树形图、气泡图、雷达图等手段，将复杂的数据变成易于理解、直观的可视化形式。它通过对数据的分析、描述和表达，帮助人们理解数据背后的意义和含义，提高数据的分析效率和数据获取准确性。
数据可视化通常包括三个方面:
- 可视化对象选择: 首先要明确可视化对象的类型、属性及维度，并针对该对象制作相应的图表或图像。
- 可视化方法选择: 根据数据的分析要求和目的，选择最合适的方法绘制数据图表。
- 颜色主题选择: 在绘制图像时，选择不同的颜色主题，增强可视化效果。
## 2.2 数据类型与常用统计图表
数据类型主要包括四种: 量值型、分类型、时序型、坐标系型。
### （1）量值型
- 折线图(Line Chart)：用来表示某变量随时间变化的趋势，比如房价的价格变化曲线、股票的收益率变化曲线。
- 曲线图(Curve Chart)：也是一种折线图，但当变量的取值范围较广时，使得其图形更加光滑。
- 柱状图(Bar Chart)：用来显示一组离散的、相同数量级的变量，按照不同的值排列在一条竖直方向上的一系列图形。
- 饼图(Pie Chart)：用来显示分类数据中各个部分占据的比例，占比越大的部分则对应着越丰富的颜色填充。
### （2）分类型
- 雷达图(Radar Chart)：用来表示多变量之间的相关关系。
- 仪表盘图(Dial Chart)：用来表示不同维度的度量结果。
- 象形图(Pyramid Chart)：用来表示不同维度的比例。
### （3）时序型
- 分时图(Slopegraph)：一种用来显示事件发生的时间顺序或次数的图表。
- 棒形图(Bullet Graph)：用来显示一组数据中存在偏差、缺失值或者异常值的情况。
- 时间序列图(Time Series Chart)：用来显示随时间变化的数值或数据点。
### （4）坐标系型
- 地图(Map)：用来显示不同位置的分布区域。
- 热力图(Heat Map)：用来表示矩阵中的数据。
- 空间直角坐标系(Scatter Plot)：用来显示二维或者三维数据。
- 平行坐标系(Parallel Coordinates)：用来比较多组数据之间的差异。
- 椭圆坐标系(Ellipse Coordinate System)：用来比较两个以上的维度间的相关性。
## 2.3 Matplotlib库简介
Matplotlib是Python的一个开源绘图库。它提供了极其详细的API接口，可以方便地创建各种各样的可视化图形，包括折线图、条形图、饼图等。Matplotlib的主要功能有：
- 创建直方图、散点图、折线图等图形
- 支持中文文本的支持
- 支持自定义图表风格和颜色
Matplotlib库的安装和使用方法非常简单。下面简单介绍一下安装和使用方法。
## 安装
Windows平台用户下载并安装Anaconda集成环境。如果没有安装Anaconda，可直接下载安装Python+Matplotlib。
```
pip install matplotlib
```
Linux/MacOS平台用户安装pip之后执行以下命令即可：
```
sudo pip install matplotlib
```
## 使用
### 加载Matplotlib库
导入Matplotlib模块如下：
```
import matplotlib.pyplot as plt
```
调用plt对象的函数完成图表绘制。
### 数据准备
这里我们随机生成了10个整数作为数据进行可视化。
```
data = [7, 5, 9, 3, 6, 1, 8, 2, 4, 10]
```
### 基本折线图绘制
这里创建一个基本的折线图，纵轴是数据的个数，横轴是每个数据的数值。
```
plt.plot(range(len(data)), data) # 生成数据列表
plt.show() # 绘制图表
```
