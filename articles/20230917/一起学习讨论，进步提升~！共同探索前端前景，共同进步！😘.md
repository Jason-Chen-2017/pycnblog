# 2.基本概念术语说明

## 2.1 HTML

HTML (Hypertext Markup Language) 是用于创建网页的标准标记语言。它包括了文本内容（如 paragraphs，headings 和 lists）、链接（internal and external）、图像、表格、视频、音频等各种媒体元素，以及一些特殊的标签如 `<html>`,`<title>` 和 `<body>` 。HTML 使用 ASCII 或 Unicode 字符集编码，并使用元素来描述文档中的结构。HTML 中所有的标签都是成对出现的，由尖括号包围，比如 `<div>`, `<p>` ，`<a>` 。

## 2.2 CSS 

CSS (Cascading Style Sheets) 是层叠样式表，它定义了网页的视觉效果。CSS 描述了网页元素应该如何显示，如颜色、字体、边框样式、位置、大小、透明度等属性。CSS 使用层叠机制，即样式可以继承自父元素、祖先元素或通过上下文关系（例如层叠级别）被指定给子元素。CSS 提供了许多样式选择器来控制页面的各个元素，使得页面更加容易维护和修改。

## 2.3 JavaScript

JavaScript 是一种动态脚本语言，它允许网页的内容、行为和动画交互。它主要用于实现浏览器端的各种功能，如表单验证、图形渲染、数据存储、AJAX 请求等。JavaScript 的语法类似 Java 和 C++ ，但有些细微差别。JavaScript 可以与 HTML 和 CSS 混合使用，实现复杂的动态效果。

## 2.4 HTTP

HTTP (HyperText Transfer Protocol) 是用于网络传输的协议，它属于客户端-服务器模型。它规定了客户机（如浏览器）如何向服务器请求资源、以及服务器响应返回信息时的规则。HTTP 通过 URL 来标识互联网上的资源，因此 HTTP 只支持静态页面的访问，对于服务端的处理只能通过其他的协议如 WebSocket、Server-sent events 或 RESTful API 来实现。

## 2.5 DOM

DOM (Document Object Model) 是 W3C 组织推荐的处理可扩展置标语言的标准接口。它定义了操作 HTML 及 XML 文档的标准方法，包括操纵节点、获取属性及设置样式等。DOM Level 1 定义了最基础的方法，Level 2 增加了针对 Range 和 TreeWalker 的方法，Level 3 增加了针对 Event、Web Components 和 File APIs 的方法。

## 2.6 AJAX

AJAX (Asynchronous JavaScript And XML) 是一种Web开发技术，它允许在不重新加载整个页面的情况下，根据用户的操作，局部地更新某些部分的内容。AJAX 技术通过 XMLHttpRequest 对象实现异步通信，并通过 DOM 操作实现页面的局部刷新。

## 2.7 浏览器内核

浏览器内核 (Browser Engine) 是指负责控制web页面内容展现、跑javascript程序的核心组件，其核心功能包括：

1. 用户界面。
2. 渲染引擎。
3. JS interpreter。
4. 数据存储。

不同的浏览器厂商会开发自己的浏览器内核，但它们之间往往存在较大的差异性。不同内核的特点可能会影响到web页面的渲染效率、稳定性和安全性。

目前，主要的浏览器内核有Google Chrome的Blink内核，Mozilla Firefox的Gecko内核，Apple Safari的WebKit内核以及Opera Software的Presto内核。

## 2.8 浏览器渲染过程

浏览器渲染过程如下图所示：


1. 当浏览器接收到 HTML 文件时，解析器（parser）开始读取文件，构建 DOM（Document Object Model）。
2. 根据 CSS 文件，生成 CSSOM（CSS Object Model），计算出元素的具体样式值。
3. 将 DOM 和 CSSOM 合并为 Render Tree。
4. 对 Render Tree 中的每个节点进行布局，确定其在屏幕上显示的坐标。
5. 调用 GPU 来绘制 Render Tree。
6. 显示渲染结果。
