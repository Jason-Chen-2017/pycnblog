# 1.简介
  

在C++语言中，为了方便编码和减少内存占用，STL提供了一种新的机制——引入命名空间(namespace)，使得某些标准库中的名字可以直接访问而不需要加上前缀。这种机制一般称为“using声明”，其语法形式如下：using namespace 名字空间名；例如，using namespace std;就可以把std命名空间中的所有名称都导入到当前作用域，使得不再需要加上前缀。但是这种机制会带来一些潜在的风险，比如命名冲突等。所以在编写C++代码时，使用命名空间是个不错的方式。
# 2.基本概念术语说明
# 2.1 标准库
在C++编程中，通常都会依赖于标准库提供的各种功能、类、函数等，这些功能、类、函数统称为标准库（standard library）。比如，iostream头文件就是C++标准库中最基础的文件输入输出流类。标准库由三种主要部分组成：基础库（如iostream），工具库（如容器、算法等），第三方库。下面简单介绍一下标准库的一些术语。
# 2.1.1 命名空间
命名空间（namespace）是C++的一个重要概念，它提供了一种结构化的方法来组织代码，并避免同名实体之间的名称冲突。每个命名空间都是一个全局作用域，其中的变量名、类型名、函数名等都可以互相独立。命名空间的目的是用来解决名称冲突的问题，通过在不同的命名空间中定义相同的名称，可以保证各个命名空间中的实体不会冲突。最常用的命名空间是std，即标准库。
# 2.1.2 头文件
在使用C++进行编程时，需要导入或链接一个或多个头文件（header file）才能使用库中的函数或者其他数据结构。头文件实际上就是预编译好的源码文件，它的内容被处理后形成有效的代码。对于每一个库，都有一个对应的头文件，开发者只要在程序中包含这个头文件，就可以使用库中的函数。头文件中通常包括库的相关定义信息，例如函数原型、宏定义、常量定义等。
# 2.1.3 模块（module）
在C++17版本之前，模块（module）是另外一种组织库的方式。它的主要意义是将库划分为更小的逻辑单元，每个单元可以独立编译，也可以集成为一个完整的库。由于模块功能还处于试验性阶段，目前很多编程环境还不支持，本文不做讨论。
# 2.1.4 STL
C++ Standard Template Library（STL）是C++的一系列模板技术的集合，包含了诸如数组、链表、队列、堆栈、散列表等算法和数据结构。虽然STL已经成为C++编程领域中非常通用的工具，但它的使用也需要对C++的一些基本概念有一定的了解。下面简单介绍一下STL的一些术语。
# 2.1.4.1 迭代器
迭代器（iterator）是STL中的一个重要概念。迭代器是一种对象，它能够遍历容器内的元素，并且允许对元素进行操作。迭代器能够帮助我们在容器中安全、高效地移动。迭代器又分为输入迭代器和输出迭代器两类，它们分别用于读取元素和写入元素。输入迭代器只能读元素，不能写元素，输出迭代器只能写元素，不能读元素。
# 2.1.4.2 容器
容器（container）是STL中最重要的数据结构之一。容器是指能够存放数据的结构。在C++中，STL提供了丰富的容器，包括序列式容器、关联式容器和容器适配器。序列式容器是指顺序存储方式的容器，例如vector、deque、list等。关联式容器是指关键字-值存储方式的容器，例如set、map等。容器适配器则是指包装另一个容器的接口的容器。
# 2.1.4.3 函数对象
函数对象（function object）是STL中的另一个重要概念。函数对象是一个类，它重载了()运算符，因此可以像函数一样作为参数传递给算法。函数对象一般用于回调函数的传递。
# 2.2 using声明
using声明的形式如下：using namespace 名字空间名；using声明一般出现在源文件的开头，用于引入某个命名空间下的所有名称。using声明可以让程序的编写变得更加简单、优雅。比如，假设我们想实现一个求两个数的最大值的函数。如果没有using声明，那么我们可能这样实现：

```c++
int max(int a, int b) {
    return (a > b? a : b);
}
```

但是这样做有一个缺点：如果其他地方也使用了max函数，就会导致命名冲突。为了避免这一问题，我们可以使用using声明：

```c++
using namespace std; // 引入命名空间std
int main() {
    int a = 10, b = 20;
    cout << "Max of " << a << " and " << b << " is: " << max(a,b) << endl;
    return 0;
}
```

在这里，我们首先申明了using namespace std语句，然后调用函数max，同时传入两个整数参数a和b。因为max是在命名空间std中定义的，所以我们可以直接调用，而无需指定命名空间。此外，cout和endl语句也在命名空间std中定义。这样，我们的程序就实现了求最大值的目的。
