# 1.简介
  

云计算是一种新的计算模式，它将基础设施、应用软件及服务等在线资源通过网络提供给最终用户。云计算平台作为服务提供者的角色，向最终用户提供可弹性扩展、按需付费的计算资源，并提供各种软件、数据库、中间件等计算服务。利用云计算平台可以实现大规模分布式计算任务，提升计算效率，降低成本。

1996年，美国的哈佛大学计算机科学系学生蒂姆·伯纳斯-李 (<NAME>) 提出了云计算的概念。随后，大量公司、学术界、政府机构等推出了基于云计算的服务，例如亚马逊 Web 服务 (AWS)、微软 Azure、谷歌云平台 (Google Cloud Platform)，IBM SoftLayer 和 Oracle Cloud Infrastructure 等。

在云计算出现之前，中心化的计算方式依赖于硬件设备部署到数据中心中，如服务器、存储设备。而到了云计算时代，中心化的管理被转变成分布式的方式。目前，大多数的云计算平台都支持多种类型的计算资源（如虚拟机、容器、数据库），包括 CPU、内存、存储、网络等。这些计算资源可以动态分配和调度，既可实现自动扩展，又可节省成本。而且，由于云计算服务和管理系统可以按需访问，所以它可以更好地满足用户需求，使得资源利用率达到最优。

2.基本概念术语说明
## IaaS（Infrastructure as a Service）
基础设施即服务 (IaaS) 是云计算的一个重要组成部分。顾名思义，IaaS 提供的是计算资源、网络资源和存储资源等基础设施的服务，最终用户可以使用这些资源构建自己的计算环境。例如 Amazon 的 EC2 (Elastic Compute Cloud)、微软 Azure 的 Virtual Machines、 Google Cloud 的 Compute Engine 等都是 IaaS 服务。

## PaaS（Platform as a Service）
平台即服务 (PaaS) 是基于 IaaS 之上的一层抽象，它提供了开发环境、测试环境和生产环境等运行环境的服务。用户只需要简单配置就可以快速部署、测试和发布应用程序。比如，Red Hat OpenShift 或 IBM BlueMix App Resource 可以提供 PaaS 服务。

## SaaS（Software as a Service）
软件即服务 (SaaS) 是云计算的最后一层抽象，它提供了一整套完整的业务应用系统，最终用户只需要在浏览器上登录网站或移动设备 APP ，就可使用该系统提供的所有功能。比如，Facebook 的 WhatsApp、Google 的 Gmail、微软的 Office 365 就是 SaaS 服务。

## 混合云
混合云是指采用多种不同类型的云服务商的组合，以此实现数据的集成、共享和分析。典型的混合云场景有企业内部云、私有云和公有云三大类。企业内部云即在内部部署一整套的 IT 运维工具和框架，对公司内外网进行统一管理；私有云则是在第三方数据中心部署独立的云计算服务，租用自有的数据中心资源；公有云则是托管于云服务提供商的云服务，一般情况下，它是为大型组织或商业客户提供的，它通常包含众多的特性，如弹性扩展、按需计费、安全、高可用等。

## 数据中心
数据中心 (Data Center) 是指用来存储、处理和传输数据的场所。一般来说，一个数据中心由多个服务器、存储设备、交换机等组成，并且还要连接着电力、网络和其它设施。

## 超融合云
超融合云是指通过高度集成的多个云计算平台，打通各个分支之间的信息，实现不同业务领域、不同地区的数据、计算、存储等资源的相互协同和整合，实现云端的异地多活能力。
