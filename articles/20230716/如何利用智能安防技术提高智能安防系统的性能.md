# 2.基本概念术语说明
首先，了解相关的基本概念和术语可以帮助我们更好的理解智能安防系统。这里给出一些基础的定义：

1. 物理层(Physical Layer): 是指电气信号在物理信道上传输过程中的表示、转换和调制，负责传送数据或者信息。物理层是指在两个节点间传输数据或者信息的过程中，要经过各种传输媒介和器件的处理，例如网络媒介和电缆，以及光纤。
2. 数据链路层(Data Link Layer): 数据链路层是一个面向连接的网络层协议，它利用尽可能短的报文段（即包）来传送数据。它的任务是使源主机和目的主机之间的数据交换变成透明的，对传输错误进行恢复和重新排序，并保证数据可靠地传输。数据链路层使用环形缓冲区来存储传输的数据，并确保数据的完整性、顺序性和时序性。
3. 网络层(Network Layer): 网络层提供了端到端的、点到点的、可靠的数据传递服务。网络层主要功能包括寻址（地址解析协议ARP），路由（路由选择协议RIP、OSPF、BGP等），拥塞控制和流量管理。
4. 传输层(Transport Layer): 传输层提供的是进程到进程的通信，负责建立可靠的、双向通信通道。它是面向连接的，因此需要建立连接后才能传输数据。传输层通常采用虚电路协议TCP或面向记录的协议UDP。
5. 会话层(Session Layer): 会话层用来建立、维护和管理传输层实体之间的通信。它建立传输层实体之间的会话，包括建立同步和关闭连接。
6. 表示层(Presentation Layer): 它提供不同数据格式之间的转换。例如，字符编码、数据压缩等。
7. 应用层(Application Layer): 它是用户直接使用的高层接口。例如，浏览器、文件传输协议FTP、电子邮件系统SMTP等。

## 3.1 动态自适应域划分法(DAFD)

动态自适应域划分法(Dynamic Adaptive Field Division, DAFD)是一种机器学习方法，用于图像分割。该方法由Jianming Liu教授于2002年提出，其特点是能够自动优化图像分割区域。DAFD算法的具体流程如图所示: 

![image.png](https://cdn.nlark.com/yuque/__latex__/d97b8bf4566d96a19a20757687d9659a.svg)

- 输入图像: DAFD采用二值图像作为输入。
- 阈值确定: DAFD首先确定图像的全局阈值，一般取Otsu阈值。
- 分割区域划定: DAFD根据图像的局部相似度对图像的背景区域进行分割。
- 更新阈值: 根据分割区域的不连续程度和局部相似度对图像的全局阈值进行更新。

## 3.2 K-means聚类法

K-means聚类法(K-Means Clustering Algorithm)是一种典型的无监督学习算法，用于对数据进行聚类。该算法可以将n个样本点划分成k个组。K-means聚类的基本步骤如下：

1. 选取初始均值向量。
2. 对每个样本点，计算距离该点最近的均值向量。
3. 将每个样本点分配到距其最近的均值向量所在的组。
4. 根据新的群集中心重新计算均值向量。
5. 重复步骤2-4，直至均值向量不再变化或满足指定终止条件。

K-means聚类法的优缺点如下: 

1. K-means聚类法简单有效，且易于实现。
2. 需要指定k值，若k值过小，聚类效果不佳；若k值过大，计算量太大。
3. K-means聚类法对初始值敏感。
