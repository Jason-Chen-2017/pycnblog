# 2.基本概念术语说明
## 数据可视化简介
数据可视化（Data Visualization）又称信息可视化，它通过图形化的方式呈现海量数据中的信息。它是一种实用的分析工具，可以突出重点、揭示趋势，把复杂的信息表现得简单易懂。数据可视化的基本过程包括以下几个步骤：

1. 数据导入与处理：需要准备好分析所需的原始数据，并对数据进行清洗、分析和统计。
2. 数据转换：将原始数据转化成合适的数据模型，如树状图、方框图等。
3. 数据可视化设计：选择合适的可视化方式进行呈现，例如散点图、条形图、饼图等。设置不同维度的颜色、透明度、大小等属性，使得图形更具识别性。
4. 数据可视化输出：将可视化结果输出到文件、屏幕或网页，供分析人员查看。

数据可视化常用技术主要包括：

1. 可视化编程语言：包括Python、JavaScript、Java、C++、HTML、CSS、SQL等。
2. 可视化库：包括matplotlib、ggplot、Seaborn、D3.js等。
3. 前端框架：包括Bootstrap、jQuery、AngularJS、React等。
4. 服务器端框架：包括Flask、Django等。

## 数据图表介绍
数据图表（Data Charts）是一种用来显示和理解数据的图形化形式。数据图表能够很直观地反映数据之间的联系、空间分布和相关关系。数据图表有助于快速发现模式、预测趋势、总结商业洞察等。数据图表包括柱状图、折线图、饼图、堆积图、热力图、雷达图、等值线图等。

数据图表分类：

1. 分析类图表：用于描述数据的聚集分布、分散分布、离群点、数据差异、局部极大值、局部最小值、异常值等特性；

2. 交互类图表：具有点击、悬停、缩放、筛选等交互行为的图表；

3. 比较类图表：用于比较多个组别的数据之间的关系；

4. 概念类图表：通过特定的主题或故事元素展现数据之间的联系；

5. 信息类图表：通过色彩编码、多维度展现数据之间的联系、空间分布。

## 数据探索和清理
首先，要对数据进行探索和清理，才能找到其中的模式和趋势。数据探索是指通过各种方法获取数据，了解数据集的内容、概括信息，以便掌握数据背后的含义和规律。数据清理是指从原始数据中删除数据噪声、缺失值、重复值等信息，然后得到干净、整齐、可分析的数据。一般情况下，数据清理需要依据业务背景、数据质量、数据安全等因素进行处理。

## 数据转换和可视化设计
转换和设计阶段主要分为以下几步：

1. 数据转换：主要用于处理数据的问题。例如，将数据转化成图表显示的格式。数据转换的主要方式有按列还是按行、是否对数据进行聚合、排除空白、是否对时间序列数据进行切割等。

2. 可视化设计：用于设计数据可视化的外观、感觉和效果。设计可视化的目的就是为了突出数据中最重要的信息，同时还需要遵循一些设计原则，如清晰、美观、易读、有意义等。可视化的种类有图表、地图、平面布局、空间立体等。

## 具体代码实例和解释说明
- 数据可视化介绍
  - Matplotlib是Python中的一个著名的绘图库，其可实现数据的视觉化。Matplotlib提供了大量的图形对象供用户使用，如折线图、散点图、柱状图、饼图、等等。通过简单的调用即可轻松实现数据的可视化。
  - Seaborn是一个基于matplotlib构建的可视化库，该库提供简洁、直观且具有吸引力的统计可视化。Seaborn对Matplotlib进行了更进一步的封装，使得可视化的操作更加简单。
  
```python
import matplotlib.pyplot as plt
import seaborn as sns
sns.set_style("whitegrid") # 设置Seaborn风格为格子状
tips = sns.load_dataset('tips') # 加载示例数据集
plt.figure(figsize=(12,8)) # 设置画布尺寸
sns.barplot(x='day', y='total_bill', hue='sex', data=tips) # 画直方图
plt.show() # 显示图片
```

- 使用 Seaborn 画直方图

```python
import numpy as np
import pandas as pd
from scipy import stats
import matplotlib.pyplot as plt
import seaborn as sns

# 生成数据
np.random.seed(123)
data = np.random.normal(loc=0, scale=1, size=100)
df = pd.DataFrame({'data': data})
print(df.head()) 

# 柱状图
sns.distplot(data, hist=True, kde=False, bins=int(180/5), color='darkblue', 
             hist_kws={'edgecolor':'black'},
             kde_kws={'linewidth': 4})  
plt.xlabel('Value')  
plt.ylabel('Density')  
plt.title('Histogram of data');  

# 核密度估计
sns.distplot(data, hist=False, rug=True, label="Data points", color="red")   
plt.legend();    

# QQ plot 
stats.probplot(data, dist="norm", plot=plt);    
plt.title("QQ Plot");     
```
