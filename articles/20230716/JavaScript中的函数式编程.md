函数式编程是一种编程范式，它通过将运算过程视为数学上的函数来进行抽象。函数式编程的特点是只关注输入数据和输出结果而忽略中间变量。

随着Web应用日益复杂，移动端和前端开发领域崛起，越来越多的人开始倾向于使用函数式编程来编程。前端工程师通常会选择较偏向面向对象的方式来编程，而非函数式编程。但是，因为函数式编程非常适合解决问题，在许多方面都可以实现更高效和更易维护的代码，因此也越来越受到青睐。

JavaScript 是目前最热门的语言之一，也是使用最广泛的客户端编程语言。很多开源库如 Underscore、Lodash、Ramda 和 RxJS 都是基于函数式编程思想设计出来的。本文将以 JavaScript 为主要平台介绍函数式编程的相关知识。
# 2.基本概念术语说明
## 什么是函数？
函数是指可以重复使用的一个或者多个语句集合。比如，你经常看到这样一段代码：
```javascript
function sayHello() {
  console.log("hello world");
}
sayHello(); // output: hello world
```
上面这是一个函数定义。函数名是`sayHello`，括号内没有参数，只是一条打印语句。调用函数时，后面跟着一个空括号 `()` 。

函数也可以接收参数。比如：
```javascript
function sum(a, b) {
  return a + b;
}
console.log(sum(2, 3)); // output: 5
```
上面这个函数接受两个参数，并返回它们的和。

还有一个重要的概念是，作用域（scope）。函数内部声明的变量仅在该函数内部有效，其他地方不能访问这些变量。函数外部可以通过闭包或立即执行函数表达式来创建新的作用域，从而访问这些变量。
## 什么是算子？
算子就是一元操作符或二元操作符，它对一个值进行操作，并返回一个新值。比如，加法运算符 (+)，减法运算符 (-)，乘法运算符 (*)，除法运算符 (/)等。

在函数式编程中，算子被用来表示计算规则。比如，map 方法用于遍历数组并生成一个新的数组；filter 方法用于过滤数组元素并生成一个新的数组；reduce 方法用于对数组元素进行聚合操作。
## 什么是纯函数？
纯函数是指无论其输入怎样，其输出总是相同的函数。换句话说，纯函数不会产生任何可观察到的副作用，它只是根据输入计算输出。

纯函数的一个优点是计算速度快，因为不用保存状态，不需要考虑函数的执行顺序，所以当传入相同的参数时，它的输出必定一致。

纯函数还有另一个好处，就是可以作为参数传递给其它函数。因为没有副作用，所以函数之间可以互相独立地测试，也可以在没有全局变量的情况下进行测试。

最后，纯函数始终保持不变性，也就是说，如果一个函数对于同样的输入永远返回相同的值，那么这个函数就是一个“幺半群”。
## 什么是高阶函数？
高阶函数就是能够接收函数作为参数或者返回值的函数。举个例子，Array.prototype.sort() 方法就是一个典型的高阶函数。

Array.prototype.sort() 会对数组进行排序，但它不是一个纯粹的数组操作函数。它首先需要一个比较器函数，用于对数组中的每两个元素进行比较，然后按升序还是降序排列，最后将结果存入数组。

由此可见，高阶函数真正强大的地方就在于，它允许我们编写紧凑的、便于组合的函数式代码，而不是编写大量的单个函数。
