# 2.基本概念术语说明
## 2.1. 人工智能与智能安防系统
人工智能(Artificial Intelligence, AI)是一门多领域的科学研究，涉及计算机科学、数学、生物学、心理学等多个领域。而智能安防系统则是指利用人工智能技术建立的控制和保护环境的计算机网络系统，用来辅助人类进行日常活动和生活。

## 2.2. 人工智能关键技术
1. 机器学习
机器学习是指让计算机学习从数据中获取的模式，并据此做出相应的预测或决策。它属于人工智能的一种算法类型，是一种以训练数据为基础，根据输入数据的反馈，改善自身模型的方式。

2. 深度学习
深度学习是基于机器学习和神经网络的技术。深度学习的重点在于解决海量数据下的模型训练和参数优化问题。通过堆叠多层神经元，能够学习复杂的数据表示形式，能够处理图像、文本等非结构化数据。

3. 数据挖掘
数据挖掘是指从大量的无结构数据中提取有效的信息，并进行分析、建模和总结，从而对现实世界进行理解、洞察、预测和描述。数据挖掘是人工智能的一个重要分支。

4. 强化学习
强化学习是指智能体通过不断试错，以期望的状态作为奖励，将学到的经验转化为行为策略。这种学习方式可以使智能体在不同的情况下作出不同的决策，达到最优的效果。

5. 模糊系统
模糊系统是指由不同感官、多种动作、外观、语言等输入引起的响应结果。模糊系统使得智能体具备某些特定的功能。如人类的视觉、听觉、触觉都可以看做是模糊系统的一部分。

## 2.3. 智能安防系统中的应用对象
智能安防系统是利用人工智能技术建立的控制和保护环境的计算机网络系统，它可以用于许多领域，如公共安全、环境保护、交通设施安全、智慧城市等。

目前智能安防系统主要应用于以下四个领域：

1. 公共安全领域
公共安全领域是指专门针对公共场所的安全保障，主要应用机器学习算法和传感器网络。例如，智能出租车、智能交通灯、智能停车场、智能火灾报警系统等。

2. 环境保护领域
环境保护领域是指通过对土壤、水体、植被、森林等自然资源的测绘、采集、检测、监测等，建立与资源密切相关的环境状况数据库，对环境状况进行预测、监测、控制，提升整体绿色建筑的效益。智能水资源管理、智能空气资源管理、智能光伏电站、智能电网等都是典型的环境保护应用。

3. 交通设施安全领域
交通设施安全领域是指利用各种感知、认知、传感技术，通过人工智能技术分析交通信息，识别异常场景，实时识别交通违法犯罪，控制交通流量，提升交通事故的侦破能力。主要技术有车辆监控、车辆定位、交通信号识别、行人轨迹跟踪、车祸风险预警等。

4. 智慧城市领域
智慧城市领域是指通过利用大数据、云计算、人工智能等新技术，构建覆盖全球的智能系统，对城市进行智能化管理、智能化调度、智能化开发、智能化运营。例如，智能停车管理系统、智能电网管理系统、智能供水管理系统、智能出租车管理系统、智能垃圾收集、智能垃圾分类、智能指路系统、智能驾驶辅助系统等。

## 2.4. 智能安防系统的主要组成部分
在智能安防系统的设计过程中，一般按照如下三个步骤进行：

1. 数据采集与处理：智能安防系统需要从各种信息源获得包括图像、视频、文本、音频等各种媒体数据，对原始数据进行数据采集、存储和处理，最终得到分析和处理后的信息数据。

2. 信息采集与分析：信息采集与分析是指从设备上传输过来的各种数据包括位置信息、人员信息、事件信息等，进行信息采集、聚合、分析，形成信息数据。信息数据可以进行分析判断出一些潜在危险因素，例如发生火灾、警车碰撞、路面塌陷、摔倒等情况。

3. 决策与控制：决策与控制是指采用机器学习算法进行分析和处理后的信息数据进行决策，产生相应的行为策略。行为策略一般包括怎么去应对这些危险因素，比如转向、减速、避开等。通过此类行为策略，控制设备的行为，从而达到保障公共场所的安全目的。
