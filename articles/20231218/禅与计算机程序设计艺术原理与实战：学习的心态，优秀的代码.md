## 2. 代码实例解释

以下是我们实现的冒泡排序算法：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个代码实例中，我们可以看到我们使用了两个for循环来实现冒泡排序算法。在第一个for循环中，我们遍历整个数组，而在第二个for循环中，我们遍历未排序的部分数组。通过这种方式，我们可以逐个比较数组中的元素，并将较大的元素移动到数组的末尾。
