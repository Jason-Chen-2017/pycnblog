# 2.核心概念与联系

## 2.1 禅学的基本概念

禅学是一种哲学思想，主要关注于人类的心灵和精神世界。禅学的核心概念包括：

- 直指人心：禅学认为，人类的心灵世界是最真实的，因此禅学的目标是帮助人们直接看到自己的心灵，达到心灵的平静和智慧。
- 无言传心：禅学认为，真正的智慧和心灵感受是无法用语言表达的，因此禅学的方法主要是通过无言的行动来传达心灵感受。
- 一箭双雕：禅学认为，真正的智慧和心灵感受是不可言说的，因此禅学的目标是帮助人们在一瞬间看到自己的心灵，达到心灵的平静和智慧。
