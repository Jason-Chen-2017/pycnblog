## 1.1 Spring Boot 简介

Spring Boot 是一个用于构建 Spring 应用程序的优秀框架，它提供了许多有用的功能，包括 WebSocket 支持。Spring Boot 使得构建 Spring 应用程序变得简单而高效，因为它自动配置 Spring 应用程序，并提供了许多预配置的 starters。

Spring Boot 的核心原则是：

- 提供一些自动配置，以便快速启动应用程序。
- 不需要 XML 配置文件，而是使用 Java 配置。
- 提供了许多预配置的 starters，以便快速构建应用程序。

## 1.2 WebSocket 简介

WebSocket 是一种基于 TCP 的协议，它允许客户端和服务器全双工地建立持久性连接，以实现实时通信。WebSocket 使得实时通信变得简单而高效，因为它不需要经过 HTTP 请求/响应循环，而是直接建立连接。

WebSocket 的核心概念包括：

- 连接：WebSocket 连接是一种持久性的、双向的连接。
- 消息：WebSocket 使用消息进行通信，消息可以是文本、二进制数据或者其他类型。
- 协议：WebSocket 使用特定的协议进行通信，例如：ws 或 wss。
