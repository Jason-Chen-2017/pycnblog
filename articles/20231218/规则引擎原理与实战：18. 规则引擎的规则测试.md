# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的代码实例来说明规则引擎的规则测试。

## 4.1 代码实例

```python
from rule_engine import RuleEngine
from rule_engine.rule import Rule

# 创建一个规则引擎实例
engine = RuleEngine()

# 创建一个规则
rule = Rule(
    name="is_adult",
    condition="age >= 18",
    action="return True"
)

# 添加规则到规则引擎
engine.add_rule(rule)

# 设置事实
engine.set_fact("age", 20)

# 执行规则
result = engine.execute("is_adult")

print(result)  # True
```

在这个代码实例中，我们创建了一个简单的规则引擎实例，并添加了一个规则`is_adult`。这个规则的条件是`age >= 18`，当这个条件满足时，结果部分将返回`True`。我们设置了一个事实`age`的值为20，并执行了`is_adult`规则。结果为`True`，表示这个规则在这个事实下执行正确。

## 4.2 详细解释说明

在这个代码实例中，我们首先导入了`RuleEngine`和`Rule`类，并创建了一个规则引擎实例`engine`。然后我们创建了一个规则`is_adult`，其中`name`为规则名称，`condition`为条件部分，`action`为结果部分。我们将这个规则添加到规则引擎中，并设置了一个事实`age`的值为20。最后，我们执行了`is_adult`规则，并输出了结果。
