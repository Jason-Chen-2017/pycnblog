## 4.1 简单的推荐系统实现
```python
from sklearn.metrics.pairwise import cosine_similarity

def recommend(user_id, user_ratings, movies):
    user_similarities = {}
    for other_id in user_ratings:
        if other_id != user_id:
            user_similarities[other_id] = cosine_similarity(user_ratings[user_id], user_ratings[other_id])
    similarities = {}
    for movie_id in movies:
        similarities[movie_id] = sum(user_similarities[other_id][movie_id] for other_id in user_similarities)
    return sorted(similarities.items(), key=lambda x: x[1], reverse=True)
```
这个简单的推荐系统实现使用了协同过滤算法，通过计算用户之间的相似性，从而为用户推荐相似的电影。

## 4.2 简单的搜索引擎优化实现

```python
def seo(content):
    keywords = ["电子商务", "程序员", "电子商务技术"]
    for keyword in keywords:
        content = content.replace(keyword, f"<strong>{keyword}</strong>")
    return content
```
这个简单的搜索引擎优化实现通过将关键词包装为`<strong>`标签，从而提高关键词的重要性，以便搜索引擎更容易找到这些关键词。
