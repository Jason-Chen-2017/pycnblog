```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个代码实例中，我们可以看到冒泡排序算法的简洁、高效、直接和自然的表现。冒泡排序算法的时间复杂度为O(n^2)，空间复杂度为O(1)，并且它的实现过程中避免了中间结果和临时变量的使用，同时也避免了人为的操作和干预。

具体来说，冒泡排序算法的实现过程中，我们首先遍历数组中的每个元素，然后比较相邻的两个元素，如果后一个元素比前一个元素大，则交换它们的位置。这个过程重复进行，直到数组中的所有元素都被排序为止。这个算法的直接和自然的表现在于它的实现过程中避免了中间结果和临时变量的使用，同时也避免了人为的操作和干预。
