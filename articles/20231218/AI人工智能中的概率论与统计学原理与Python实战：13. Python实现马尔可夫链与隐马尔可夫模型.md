## 1.2 核心概念与联系

### 1.2.1 马尔可夫链

马尔可夫链是一种随机过程，它的未来状态只依赖于当前状态，而不依赖于过去状态。换句话说，马尔可夫链是一种依赖于当前状态的随机过程。

例如，假设我们有一个三个状态的马尔可夫链：A -> B -> C。如果我们从状态A开始，那么下一步我们只能转向状态B，如果我们从状态B开始，那么下一步我们只能转向状态C。这个例子说明了马尔可夫链的基本特征：未来状态只依赖于当前状态。

### 1.2.2 隐马尔可夫模型

隐马尔可夫模型（Hidden Markov Model，HMM）是一种有限自动机，它的状态和输入之间存在一定的概率关系。隐马可夫模型的特点是：

1. 系统的内部状态是隐藏的，不能直接观测到。
2. 系统的输入和输出是可观测的。
3. 系统的状态转移和输出生成是根据概率规则的。

例如，假设我们有一个两个状态的隐马尔可夫模型：A -> B。这个隐马尔可夫模型的状态是隐藏的，我们只能通过观测输入和输出来推测系统的状态。当我们从状态A开始，如果我们观测到某个输入，那么我们可以推测系统迁移到状态B。

马尔可夫链和隐马尔可夫模型之间的联系在于它们都是基于概率论的随机过程。马尔可夫链是一种纯粹的随机过程，其状态是可观测的。隐马尔可夫模型是一种混合随机过程，其状态是隐藏的，需要通过观测输入和输出来推测。

在人工智能领域，马尔可夫链和隐马尔可夫模型的应用是相互补充的。例如，在语言模型中，我们可以使用马尔可夫链来描述词汇之间的关系，但是这样的模型无法处理词汇的起伏。为了解决这个问题，我们可以使用隐马尔可夫模型来描述词汇的起伏，同时保留了词汇之间的关系。
