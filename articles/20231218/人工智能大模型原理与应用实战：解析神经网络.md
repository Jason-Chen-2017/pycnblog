# 2.核心概念与联系

在本节中，我们将介绍神经网络的基本概念，包括：

- 神经网络的基本结构
- 神经元（neuron）和层（layer）
- 激活函数（activation function）
- 损失函数（loss function）
- 反向传播（backpropagation）

## 2.1 神经网络的基本结构

神经网络是一种由多层节点相互连接的计算模型，每个节点都表示为一个神经元。神经网络的基本结构包括输入层、隐藏层和输出层。

- 输入层：包含输入数据的节点，这些节点通常是已知的。
- 隐藏层：位于输入层和输出层之间的层，可以有一个或多个。这些层通常用于处理输入数据，以便在输出层进行更高级的处理。
- 输出层：包含输出结果的节点，这些节点通常是未知的。

## 2.2 神经元和层

神经元是神经网络的基本构建块，它们接收输入信号，进行处理，并输出结果。神经元通常由权重和偏置参数组成，这些参数在训练过程中会被更新。

层是一组相邻的神经元，它们共享相同的输入和输出。层之间通过权重矩阵相互连接，这些权重矩阵用于传播输入信号到输出信号。

## 2.3 激活函数

激活函数是神经网络中的一个关键组件，它用于将输入信号转换为输出信号。激活函数通常是非线性的，这意味着它们可以在输入信号的基础上产生新的输出信号。常见的激活函数包括：

- 步函数（step function）
-  sigmoid 函数（sigmoid function）
-  hyperbolic tangent 函数（hyperbolic tangent function，tanh）
-  ReLU 函数（Rectified Linear Unit，ReLU）

## 2.4 损失函数

损失函数用于衡量神经网络的预测与实际值之间的差异。损失函数通常是一个非负值，小的损失值表示预测结果与实际结果更接近。常见的损失函数包括：

- 均方误差（mean squared error，MSE）
- 交叉熵损失（cross-entropy loss）
- 平均绝对误差（mean absolute error，MAE）

## 2.5 反向传播

反向传播（backpropagation）是一种优化神经网络权重的方法，它通过计算输出层和隐藏层之间的梯度来更新权重。反向传播的主要步骤包括：

1. 前向传播：从输入层到输出层传播输入数据，计算每个神经元的输出。
2. 计算损失：使用损失函数计算神经网络的预测与实际值之间的差异。
3. 后向传播：从输出层到输入层传播损失，计算每个神经元的梯度。
4. 更新权重：使用梯度下降法（gradient descent）更新神经元的权重和偏置。
