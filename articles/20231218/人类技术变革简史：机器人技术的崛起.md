## 4.2 基于模型的控制实例

### 4.2.1 简单的PID控制器

```python
class PIDController:
    def __init__(self, Kp, Ki, Kd):
        self.Kp = Kp
        self.Ki = Ki
        self.Kd = Kd
        self.integral = 0
        self.last_error = 0

    def control(self, error, dt):
        self.integral += error * dt
        derivative = (error - self.last_error) / dt
        output = self.Kp * error + self.Ki * self.integral + self.Kd * derivative
        self.last_error = error
        return output
```

### 4.2.2 PID控制器的解释

- 首先，我们定义了一个PID控制器类，包含Kp、Ki、Kd三个参数以及积分值和前一次误差。
- 然后，我们在类的 `control` 方法中实现了基于PID算法的控制逻辑，计算输出值并更新积分值和前一次误差。

## 4.3 基于学习的控制实例

### 4.3.1 简单的Q学习实例

```python
import numpy as np

class QLearningAgent:
    def __init__(self, actions, alpha, gamma, epsilon):
        self.actions = actions
        self.alpha = alpha
        self.gamma = gamma
        self.epsilon = epsilon
        self.q_table = np.zeros((len(actions), len(actions)))

    def choose_action(self, state):
        if np.random.uniform(0, 1) < self.epsilon:
            return np.random.choice(self.actions)
        else:
            return np.argmax(self.q_table[state])

    def learn(self, state, action, reward, next_state):
        best_next_action = np.argmax(self.q_table[next_state])
        td_target = reward + self.gamma * self.q_table[next_state][best_next_action]
        self.q_table[state][action] += self.alpha * (td_target - self.q_table[state][action])
```

### 4.3.2 Q学习实例的解释

- 首先，我们定义了一个Q学习代理类，包含动作集、学习率、折扣因子和贪婪度。
- 然后，我们在类的 `choose_action` 方法中实现了动作选择策略，根据贪婪度选择是否随机选择动作。
- 最后，我们在类的 `learn` 方法中实现了Q学习算法，根据TD目标值更新Q值表。
