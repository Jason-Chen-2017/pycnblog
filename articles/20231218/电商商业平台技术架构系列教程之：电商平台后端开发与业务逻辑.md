## 4.1 用户身份验证

在实现用户身份验证时，我们可以使用Python的hashlib库来实现密码加密。以下是一个简单的示例代码：

```python
import hashlib

def hash_password(password):
    salt = hashlib.sha256(password.encode()).hexdigest()
    return salt

password = "123456"
hashed_password = hash_password(password)
print(hashed_password)
```

在这个示例代码中，我们使用了SHA-256算法对用户输入的密码进行加密，并将加密后的密码返回。
