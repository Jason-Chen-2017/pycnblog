# 4.具体代码实例和详细解释说明

在本节中，我们将通过具体的代码实例来详细解释自然发展算法的实现过程。

## 4.1 简单的自然发展算法实例

我们来看一个简单的自然发展算法实例，即寻找一个列表中的最大值。

```python
def find_max(lst):
    max_val = lst[0]
    for x in lst:
        if x > max_val:
            max_val = x
    return max_val
```

在这个实例中，我们首先设定了一个初始条件，即列表中的第一个元素为最大值。然后，我们通过遍历列表中的所有元素，找到最大值。这个算法的时间复杂度为$O(n)$，其中$n$是列表的长度。

## 4.2 自然发展算法的拓展

我们可以将自然发展算法拓展到更复杂的问题上，例如寻找多个最大值或寻找最小值。以下是一个寻找多个最大值的例子：

```python
def find_max_values(lst):
    max_vals = []
    max_val = lst[0]
    for x in lst:
        if x > max_val:
            max_val = x
            max_vals = [x]
        elif x == max_val:
            max_vals.append(x)
    return max_vals
```

在这个实例中，我们首先设定了一个初始条件，即列表中的第一个元素为最大值。然后，我们通过遍历列表中的所有元素，找到最大值。当找到新的最大值时，我们更新最大值并重置最大值列表。这个算法的时间复杂度仍然为$O(n)$。
