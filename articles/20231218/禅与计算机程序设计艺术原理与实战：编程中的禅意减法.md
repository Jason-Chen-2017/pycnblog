# 4.具体代码实例和详细解释说明

## 4.1 代码实例

以下是一个简单的Python代码实例，用于从一个列表中删除重复的元素：

```python
def remove_duplicates(lst):
    result = []
    for item in lst:
        if item not in result:
            result.append(item)
    return result

lst = [1, 2, 2, 3, 4, 4, 5]
print(remove_duplicates(lst))
```

## 4.2 详细解释说明

在这个代码实例中，我们定义了一个函数`remove_duplicates`，它接受一个列表作为输入，并返回一个不包含重复元素的新列表。在函数内部，我们创建了一个空列表`result`，用于存储不重复的元素。然后，我们遍历输入列表`lst`，并检查每个元素是否已经在`result`列表中。如果没有，我们将其添加到`result`列表中。最后，我们返回`result`列表。
