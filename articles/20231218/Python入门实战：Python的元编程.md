# 4.具体代码实例和详细解释说明

以下是一个具体的Python代码实例，它使用了Python的元编程技术来实现一个简单的动态类创建和修改功能：

```python
class DynamicClass:
    def __init__(self, name):
        self.name = name

    def __getattr__(self, item):
        if item == 'age':
            return 25
        else:
            raise AttributeError(f'{self.name} has no attribute {item}')

    def __setattr__(self, key, value):
        if key == 'name':
            raise AttributeError(f'{self.name} cannot modify name')
        else:
            super().__setattr__(key, value)

    def __delattr__(self, item):
        if item == 'name':
            raise AttributeError(f'{self.name} cannot modify name')
        else:
            super().__delattr__(item)

obj = DynamicClass('Alice')
print(obj.name)  # Alice
print(obj.age)   # 25
obj.age = 30     # 修改age属性
print(obj.age)   # 30
del obj.age      # 删除age属性
```

在这个代码实例中，我们定义了一个名为`DynamicClass`的类，它使用了`__getattr__`、`__setattr__`和`__delattr__`这三个特殊方法来实现动态属性访问、修改和删除功能。通过这些特殊方法，我们可以在运行时动态地添加、修改和删除类的属性。
