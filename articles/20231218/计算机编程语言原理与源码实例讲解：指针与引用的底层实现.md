# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个具体的代码实例来讲解指针和引用的底层实现。

## 4.1 指针的具体实例
```c
#include <stdio.h>

int main() {
    int a = 10;
    int *p = &a;
    printf("a = %d\n", a);
    printf("p = %p\n", p);
    printf("*p = %d\n", *p);
    return 0;
}
```
在上面的代码实例中，我们创建了一个整型变量a，并创建了一个指向整型变量的指针p。然后，我们将p指向a的内存地址，并使用printf函数来打印a、p和*p的值。从输出结果中可以看出，a的值为10，p的值为内存地址，*p的值为10，这表明p指向了a的内存地址，并且可以访问a的值。

## 4.2 引用的具体实例
```c++
#include <iostream>

int main() {
    int a = 10;
    int &ref = a;
    printf("a = %d\n", a);
    printf("ref = %d\n", ref);
    return 0;
}
```
在上面的代码实例中，我们创建了一个整型变量a，并创建了一个引用int型变量的引用ref。然后，我们将ref指向a，并使用printf函数来打印a和ref的值。从输出结果中可以看出，a的值为10，ref的值为10，这表明ref指向了a，并且可以访问a的值。
