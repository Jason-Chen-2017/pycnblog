## 4.1 Future的具体代码实例

```dart
void main() {
  Future<int> future = Future<int>(() {
    return compute(add, 1, 2);
  });

  future.then((int result) {
    print(result);
  });
}

int add(int a, int b) {
  return a + b;
}
```

在这个代码实例中，我们创建了一个Future对象，并传入一个回调函数。在回调函数中，我们执行了一个异步任务，即计算两个数的和。在异步任务完成后，我们调用Future对象的then方法，传入一个回调函数，并在回调函数中执行某个函数。
