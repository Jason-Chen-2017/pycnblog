## 1.2 核心概念与联系

在本节中，我们将介绍分布式缓存的核心概念和联系，包括：

1. 缓存一致性
2. 缓存穿透
3. 缓存雪崩
4. 缓存击穿

### 1.2.1 缓存一致性

缓存一致性是指缓存和原始数据源之间的数据一致性。在分布式缓存中，我们需要确保缓存和原始数据源之间的数据一致性，以避免数据不一致的情况。

### 1.2.2 缓存穿透

缓存穿透是指在缓存中查询不到的数据被查询时，会导致请求直接访问原始数据源，导致原始数据源的压力增加。

### 1.2.3 缓存雪崩

缓存雪崩是指在缓存过期时间过短或者缓存服务器宕机时，大量的请求会同时访问原始数据源，导致原始数据源的压力增加。

### 1.2.4 缓存击穿

缓存击穿是指在缓存过期时间短，并发量大的情况下，缓存过期的同时，大量的请求会同时访问原始数据源，导致原始数据源的压力增加。
