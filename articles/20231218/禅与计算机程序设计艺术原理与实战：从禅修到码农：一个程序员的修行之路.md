```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print("排序前的数组:", arr)
print("排序后的数组:", bubble_sort(arr))
```

通过实践可以发现，冒泡排序算法的时间复杂度为O(n^2)，空间复杂度为O(1)。
