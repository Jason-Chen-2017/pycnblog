```python
class RuleEngine:
    def __init__(self):
        self.rules = []

    def add_rule(self, rule):
        self.rules.append(rule)

    def execute(self, facts):
        for rule in self.rules:
            if rule.condition(facts):
                rule.action(facts)
```

在这个规则引擎中，我们可以使用以下规则来处理商品信息：

    def action(self, facts):
