# 4.具体代码实例和详细解释说明
# 4.1 代码实例
以下是一个Python脚本编程的代码实例，用于实现文件复制功能：

```python
import os
import shutil

def copy_file(src, dst):
    if os.path.exists(dst):
        os.remove(dst)
    shutil.copy(src, dst)

src = '/path/to/source/file'
dst = '/path/to/destination/file'
copy_file(src, dst)
```

# 4.2 代码解释
以下是代码的详细解释：

- **导入模块**。导入os和shutil模块，用于实现文件操作功能。
- **定义函数**。定义copy_file函数，用于实现文件复制功能。
- **检查目标文件是否存在**。使用os.path.exists函数来检查目标文件是否存在，如果存在则使用os.remove函数来删除目标文件。
- **复制文件**。使用shutil.copy函数来复制源文件到目标文件。
- **调用函数**。调用copy_file函数，将源文件复制到目标文件。
