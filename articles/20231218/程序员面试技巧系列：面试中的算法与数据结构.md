### 4.1 冒泡排序

冒泡排序是一种简单的排序算法，它可以用来对数据进行排序。冒泡排序的原理是：通过多次遍历数据，将较大的元素向后移动，将较小的元素向前移动，最终实现数据的排序。

以下是冒泡排序的具体代码实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 4.2 选择排序

选择排序是一种简单的排序算法，它可以用来对数据进行排序。选择排序的原理是：通过多次遍历数据，将最小的元素放在最前面，将最大的元素放在最后面，最终实现数据的排序。

以下是选择排序的具体代码实例：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

### 4.3 插入排序

插入排序是一种简单的排序算法，它可以用来对数据进行排序。插入排序的原理是：将数据分为两部分，已排序部分和未排序部分。从未排序部分中取出一个元素，将其插入到已排序部分中的正确位置，直到所有元素都被排序。

以下是插入排序的具体代码实例：

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i-1
        while j >=0 and key < arr[j]:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr
```

### 4.4 希尔排序

希尔排序是一种简单的排序算法，它可以用来对数据进行排序。希尔排序的原理是：将数据分为多个组，每个组中的元素间距离相等。通过多次遍历数据，将元素按照组的顺序进行排序，最终实现数据的排序。

以下是希尔排序的具体代码实例：

```python
def shell_sort(arr):
    n = len(arr)
    gap = n//2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j-gap] > temp:
                arr[j] = arr[j-gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr
```

### 4.5 快速排序

快速排序是一种高效的排序算法，它可以用来对数据进行排序。快速排序的原理是：选择一个元素作为基准，将所有小于基准的元素放在基准的左边，将所有大于基准的元素放在基准的右边，最终实现数据的排序。

以下是快速排序的具体代码实例：

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr)//2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

### 4.6 归并排序

归并排序是一种高效的排序算法，它可以用来对数据进行排序。归并排序的原理是：将数据分为多个子问题，然后将子问题的解组合成原问题的解。归并排序的时间复杂度是 O(nlogn)。

以下是归并排序的具体代码实例：

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr)//2
    left = arr[:mid]
    right = arr[mid:]
    return merge(merge_sort(left), merge_sort(right))

def merge(left, right):
    result = []
    while left and right:
        if left[0] < right[0]:
            result.append(left.pop(0))
        else:
            result.append(right.pop(0))
    result.extend(left)
    result.extend(right)
    return result
```
