## 4.1 代码示例
```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

## 4.2 解释说明
上述代码是快速排序算法的一个实现。快速排序是一种常用的排序算法，它的时间复杂度为O(nlogn)。通过观察这个算法的运行过程，我们可以看到它的核心思想是通过选择一个基准元素，将数组分为两部分，一部分小于基准元素，一部分大于基准元素，然后递归地对这两部分进行排序。

通过直接体验和观察，我们可以看到这个算法的运行过程非常清晰，它的时间和空间复杂度也是可控的。此外，这个算法是稳定的，这意味着它能够保持输入的顺序不变。
