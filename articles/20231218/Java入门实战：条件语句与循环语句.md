# 2.核心概念与联系

## 2.1 条件语句

条件语句是一种用于根据某个条件的值来执行不同代码块的语句。在Java中，条件语句主要包括if语句、switch语句和conditional-operator（条件运算符）。

### 2.1.1 if语句

if语句是Java中最基本的条件语句，它用于根据某个条件的值来执行不同的代码块。if语句的基本语法如下：

```
if(condition){
    // 执行的代码块
}
```

condition是一个布尔表达式，如果其值为true，则执行代码块；如果其值为false，则不执行代码块。

### 2.1.2 switch语句

switch语句是一种用于根据某个变量的值来执行不同代码块的语句。switch语句的基本语法如下：

```
switch(variable){
    case value1:
        // 执行的代码块
        break;
    case value2:
        // 执行的代码块
        break;
    // ...
    default:
        // 执行的代码块
}
```

variable是一个变量，根据其值来执行不同的代码块。case后面的value是一个常量表达式，用于匹配变量的值。如果变量的值与某个case的value匹配，则执行对应的代码块。如果没有匹配的case，则执行default后的代码块。

### 2.1.3 conditional-operator（条件运算符）

条件运算符是一种用于根据某个条件的值来执行不同代码块的语句。条件运算符的基本语法如下：

```
condition ? expression1 : expression2
```

如果condition的值为true，则执行expression1；如果condition的值为false，则执行expression2。

## 2.2 循环语句

循环语句是一种用于重复执行某个代码块的语句。在Java中，循环语句主要包括for语句、while语句和do-while语句。

### 2.2.1 for语句

for语句是Java中一种用于重复执行某个代码块的语句，它可以用来遍历数组或其他集合。for语句的基本语法如下：

```
for(initialization; condition; iteration){
    // 执行的代码块
}
```

initialization是用于初始化循环变量的表达式，condition是用于判断循环是否继续的条件表达式，iteration是用于更新循环变量的表达式。

### 2.2.2 while语句

while语句是Java中一种用于重复执行某个代码块的语句，它根据某个条件的值来判断循环是否继续。while语句的基本语法如下：

```
while(condition){
    // 执行的代码块
}
```

condition是一个布尔表达式，如果其值为true，则执行代码块；如果其值为false，则不执行代码块。

### 2.2.3 do-while语句

do-while语句是Java中一种用于重复执行某个代码块的语句，它根据某个条件的值来判断循环是否继续。do-while语句的基本语法如下：

```
do{
    // 执行的代码块
}while(condition);
```

condition是一个布尔表达式，如果其值为true，则执行代码块；如果其值为false，则不执行代码块。

3. 如果循环继续，则执行代码块；否则，不执行代码块。

# 4.具体代码实例和详细解释说明

## 4.1 条件语句的具体代码实例和详细解释说明

### 4.1.1 if语句的具体代码实例

```java
int x = 10;
if(x > 5){
    System.out.println("x大于5");
}else if(x == 5){
    System.out.println("x等于5");
}else{
    System.out.println("x小于5");
}
```

在这个代码实例中，我们使用if语句来判断变量x的值。如果x的值大于5，则输出"x大于5"；如果x的值等于5，则输出"x等于5"；如果x的值小于5，则输出"x小于5"。

### 4.1.2 switch语句的具体代码实例

```java
char c = 'a';
switch(c){
    case 'a':
        System.out.println("c等于a");
        break;
    case 'b':
        System.out.println("c等于b");
        break;
    default:
        System.out.println("c不等于a或b");
}
```

在这个代码实例中，我们使用switch语句来判断变量c的值。如果c的值等于'a'，则输出"c等于a"；如果c的值等于'b'，则输出"c等于b"；如果c的值不等于'a'或'b'，则输出"c不等于a或b"。

### 4.1.3 条件运算符的具体代码实例

```java
int x = 10;
String result = (x > 5) ? "x大于5" : "x小于或等于5";
System.out.println(result);
```

在这个代码实例中，我们使用条件运算符来判断变量x的值。如果x的值大于5，则result的值为"x大于5"；否则，result的值为"x小于或等于5"。

## 4.2 循环语句的具体代码实例和详细解释说明

### 4.2.1 for语句的具体代码实例

```java
for(int i = 0; i < 10; i++){
    System.out.println("i的值为：" + i);
}
```

在这个代码实例中，我们使用for语句来遍历变量i的值。从0开始，小于10，每次增加1。如果i的值小于10，则输出"i的值为：" + i；否则，不输出。

### 4.2.2 while语句的具体代码实例

```java
int i = 0;
while(i < 10){
    System.out.println("i的值为：" + i);
    i++;
}
```

在这个代码实例中，我们使用while语句来遍历变量i的值。如果i的值小于10，则输出"i的值为：" + i；然后，i的值增加1。如果i的值小于10，则继续执行循环；否则，不执行循环。

### 4.2.3 do-while语句的具体代码实例

```java
int i = 0;
do{
    System.out.println("i的值为：" + i);
    i++;
}while(i < 10);
```

在这个代码实例中，我们使用do-while语句来遍历变量i的值。首先，输出"i的值为：" + i；然后，i的值增加1。如果i的值小于10，则继续执行循环；否则，不执行循环。

# 6.附录常见问题与解答

## 6.1 常见问题

1. 条件语句和循环语句的区别是什么？

   条件语句是根据某个条件的值来执行不同代码块的语句，而循环语句是重复执行某个代码块的语句。

2. 如何判断一个变量的值是否在某个范围内？

   可以使用if语句来判断一个变量的值是否在某个范围内。例如：

   ```
   if(x >= 10 && x <= 20){
       // 执行的代码块
   }
   ```

3. 如何实现一个无限循环？

   可以使用while语句或do-while语句来实现一个无限循环。例如：

   ```
   while(true){
       // 执行的代码块
   }
   ```

   ```
   do{
       // 执行的代码块
   }while(true);
   ```
