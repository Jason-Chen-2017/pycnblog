		+ 半监督学习：半监督学习是指一个机器能够从部分标注的数据中学习出知识的技术。
	* 深度学习：深度学习是指一个机器能够从大量数据中自主地学习出高级特征的技术。深度学习可以被分为以下几个类别：

4. 人工智能应用：人工智能应用是指一种能够让机器具备智能行为的应用。人工智能应用可以被分为以下几个类别：
