### 4.1单例模式实例

单例模式是一种确保一个类只有一个实例的模式。这种模式可以用于实现资源共享和避免资源浪费。下面是一个简单的单例模式实例：

```python
class Singleton:
    _instance = None

    def __new__(cls, *args, **kwargs):
        if not cls._instance:
            cls._instance = super(Singleton, cls).__new__(cls, *args, **kwargs)
        return cls._instance
```

在这个实例中，我们定义了一个类`Singleton`，它使用`__new__`方法来确保只有一个实例。`__new__`方法在实例化对象时调用，它可以用来控制对象的创建。

```python

        pass

    def get_order(self, order_id):
```

在这个实例中，我们定义了两个服务`UserService`和`OrderService`。这两个服务分别负责处理用户和订单相关的操作。这种架构风格可以让每个服务独立部署，从而实现更好的可扩展性和可维护性。
