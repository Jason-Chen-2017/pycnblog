## 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来详细解释设备的申请与释放的过程：

```c
#include <stdio.h>
#include <semaphore.h>

sem_t printer;

void print(int id) {
    printf("Process %d is printing\n", id);
    sem_post(&printer);
}

void copy(int id) {
    sem_wait(&printer);
    printf("Process %d is copying\n", id);
    print(id);
}

int main() {
    sem_init(&printer, 0, 1);

    pid_t pid1, pid2;
    pid1 = fork();
    if (pid1 == 0) {
        copy(1);
    } else {
        pid2 = fork();
        if (pid2 == 0) {
            copy(2);
        } else {
            wait(NULL);
            wait(NULL);
        }
    }

    return 0;
}
```

在这个代码实例中，我们使用了信号量来实现设备的申请与释放。信号量是一种同步原语，它可以用来实现资源的互斥和同步。在这个例子中，我们使用了一个信号量来表示打印机的设备资源。

当进程1和进程2需要使用打印机时，它们 respective地调用`sem_wait(&printer)`来请求打印机的资源。如果打印机已经被其他进程使用，`sem_wait`会阻塞，直到打印机被释放。当进程1和进程2释放打印机后，它们 respective地调用`sem_post(&printer)`来释放打印机的资源。
