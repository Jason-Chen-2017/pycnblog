## 4.1 代码实例

我们将通过一个简单的代码实例来详细解释设计模式的使用。假设我们需要编写一个程序来计算两个数的和、差、积和商。我们可以使用面向对象编程的思想，将这些计算表示为类和对象。

```python
class Calculator:
    def add(self, a, b):
        return a + b

    def subtract(self, a, b):
        return a - b

    def multiply(self, a, b):
        return a * b

    def divide(self, a, b):
        if b == 0:
            raise ValueError("Cannot divide by zero")
        return a / b
```

在这个代码实例中，我们首先定义了一个名为Calculator的类，该类包含四个方法：add、subtract、multiply和divide。这些方法分别实现了两个数的和、差、积和商的计算。

接下来，我们可以创建一个Calculator对象，并使用它的方法来计算两个数的和、差、积和商。

```python
calc = Calculator()
print(calc.add(10, 5)) # 15
print(calc.subtract(10, 5)) # 5
print(calc.multiply(10, 5)) # 50
print(calc.divide(10, 5)) # 2.0
```

通过这个具体的代码实例，我们可以更好地理解设计模式的使用。
