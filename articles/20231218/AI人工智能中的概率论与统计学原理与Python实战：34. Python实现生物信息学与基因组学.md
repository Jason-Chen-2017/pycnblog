                 

# 1.背景介绍

生物信息学（Bioinformatics）是一门研究生物学信息的科学。生物信息学的主要目标是研究生物数据，如基因组序列、蛋白质结构和功能、生物路径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径径
