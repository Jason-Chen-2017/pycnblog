## 4.1 分治法实例

假设我们需要计算一个数列的和，这个数列的长度是2的幂次方。我们可以使用分治法来解决这个问题。

```python
def sum_of_sequence(sequence):
    if len(sequence) == 1:
        return sequence[0]
    else:
        mid = len(sequence) // 2
        left_sum = sum_of_sequence(sequence[:mid])
        right_sum = sum_of_sequence(sequence[mid:])
        return left_sum + right_sum
```

在这个例子中，我们将一个长度为2的幂次方的数列分解为两个长度为1的数列，然后递归地计算这两个数列的和，最后将两个数列的和相加。

## 4.2 动态规划实例

假设我们需要计算一个字符串的最长回文子串。我们可以使用动态规划来解决这个问题。

```python
def longest_palindrome_substring(s):
    n = len(s)
    dp = [[False] * n for _ in range(n)]
    max_length = 1
    start = 0
    for i in range(n):
        dp[i][i] = True
        for j in range(i - 1, -1, -1):
            if s[i] == s[j] and i - j <= 2:
                dp[j][i] = True
                if max_length < i - j + 1:
                    max_length = i - j + 1
                    start = j
    return s[start:start + max_length]
```

在这个例子中，我们将一个字符串分解为多个子串，然后递归地计算每个子串是否为回文串，最后将所有子串的回文串组合成最长回文串。

## 4.3 贪心算法实例

假设我们需要计算一个数列的最大子数列和。我们可以使用贪心算法来解决这个问题。

```python
def max_subarray_sum(sequence):
    max_sum = sequence[0]
    current_sum = sequence[0]
    for i in range(1, len(sequence)):
        if current_sum <= 0:
            current_sum = sequence[i]
        else:
            current_sum += sequence[i]
        max_sum = max(max_sum, current_sum)
    return max_sum
```

在这个例子中，我们将一个数列分解为多个子数列，然后递归地计算每个子数列的和，最后将所有子数列的和组合成最大子数列和。

```python
def all_subarray_sums(sequence):
    result = []
    for i in range(len(sequence)):
        for j in range(i + 1, len(sequence) + 1):
            subarray = sequence[i:j]
            result.append(subarray)
    return result
```

在这个例子中，我们将一个数列分解为多个子数列，然后递归地计算每个子数列的和，最后将所有子数列的和组合成所有子数列和。
