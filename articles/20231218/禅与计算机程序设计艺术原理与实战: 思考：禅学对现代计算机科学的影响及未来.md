## 4.1 动态规划算法实例

考虑一个经典的背包问题：给定一个物品集合和一个背包的容量，如何将物品放入背包以使得背包的重量最大化？

```python
def knapsack(items, capacity):
    n = len(items)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if items[i - 1][1] <= w:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - items[i - 1][1]] + items[i - 1][0])
            else:
                dp[i][w] = dp[i - 1][w]

    return dp[n][capacity]

items = [(1, 1), (2, 3), (4, 4)]
capacity = 5
print(knapsack(items, capacity))
```

在上述代码中，我们使用了动态规划算法来解决背包问题。通过将问题分解为多个子问题，并将子问题的解存储在一个表格中，我们可以得到问题的最优解。

## 4.2 贪心算法实例

考虑一个经典的活动选择问题：给定一个活动集合，每个活动有一个开始时间和一个结束时间，如何选择一个最大的活动集合，使得活动之间不会相互冲突？

```python
def activity_selection(activities):
    activities.sort(key=lambda x: x[1])
    selected_activities = []
    current_end_time = 0

    for activity in activities:
        if activity[0] >= current_end_time:
            selected_activities.append(activity)
            current_end_time = activity[1]

    return selected_activities

activities = [(1, 4), (2, 5), (3, 6), (5, 7), (8, 9)]
print(activity_selection(activities))
```

在上述代码中，我们使用了贪心算法来解决活动选择问题。通过在每个决策中选择最佳选择，我们可以逐步优化整个问题。

## 4.3 分治算法实例

考虑一个经典的排序问题：给定一个数组，如何将数组排序？

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]

    left = merge_sort(left)
    right = merge_sort(right)

    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])

    return result

arr = [38, 27, 43, 3, 9, 82, 10]
print(merge_sort(arr))
```

在上述代码中，我们使用了分治算法来解决排序问题。通过将问题分解为多个子问题，并将子问题的解组合在一起，我们可以得到整个问题的解。
