# 2.核心概念与联系

## 2.1 正则表达式的基本概念

正则表达式（Regular Expression）是一种用于匹配字符串的模式，它由一系列字符组成，这些字符可以表示字符串中的具体内容、结构或关系。正则表达式可以用来匹配、替换、提取和验证字符串中的模式。

## 2.2 正则表达式的核心组成部分

正则表达式的核心组成部分包括：

2.2.1 字符集：用来匹配一组字符，如[a-z]表示匹配小写字母。

2.2.2 字符类：用来匹配特定类型的字符，如\d表示匹配数字。

2.2.3 量词：用来匹配一定数量的字符，如*表示匹配0个或多个字符。

2.2.4 组：用来匹配一组子表达式，如(a|b)表示匹配a或b。

2.2.5 引用：用来匹配之前定义的子表达式，如\1表示匹配第一个组。

2.2.6 分组：用来匹配一组字符并将其作为一个整体，如(a+)表示匹配一个或多个a。

2.2.7 贪婪模式：用来匹配尽可能多的字符，如.*表示匹配任意多的字符。

2.2.8 懒惰模式：用来匹配尽可能少的字符，如.*?表示匹配任意少的字符。

2.2.9 回溯：用来匹配之前的内容，如(a.*?b)\1表示匹配a和b之间的内容。

2.2.10 修饰符：用来修改正则表达式的匹配行为，如i表示忽略大小写。

## 2.3 正则表达式与正则引擎的关系

正则表达式与正则引擎之间的关系可以理解为“模式与执行器”的关系。正则表达式是一种描述字符串模式的方法，而正则引擎则负责根据这些模式来匹配、替换、提取和验证字符串中的内容。正则引擎通过解析正则表达式并根据其规则来执行匹配操作，从而实现了对字符串的处理。

# 4.具体代码实例和详细解释说明

## 4.1 正则表达式的基本使用

```python
import re

# 匹配字符串中的数字
pattern = r'\d+'
string = '1234567890'
match = re.match(pattern, string)
if match:
    print(match.group())
```

## 4.2 正则表达式的替换

```python
import re

# 将字符串中的数字替换为星号
pattern = r'\d+'
replacement = '*'
string = '1234567890'
new_string = re.sub(pattern, replacement, string)
print(new_string)
```

## 4.3 正则表达式的提取

```python
import re

# 提取字符串中的所有邮箱地址
pattern = r'[\w\.-]+@[\w\.-]+'
string = 'Please contact me at admin@example.com or support@example.net'
match = re.findall(pattern, string)
print(match)
```

## 4.4 正则表达式的验证

```python
import re

# 验证密码是否符合规则
pattern = r'^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)[a-zA-Z\d]{8,}$'
string = 'Password123'
if re.match(pattern, string):
    print('密码符合规则')
else:
    print('密码不符合规则')
```

5. 正则表达式的复杂度主要包括时间复杂度和空间复杂度。
