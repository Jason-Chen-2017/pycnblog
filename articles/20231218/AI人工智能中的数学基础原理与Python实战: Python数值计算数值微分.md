## 4.1 数值计算代码实例

我们来看一个简单的数值积分计算的代码实例：

```python
import numpy as np

def f(x):
    return np.exp(-x**2)

def simpson(a, b, n):
    h = (b - a) / n
    x = np.linspace(a, b, n + 1)
    y = f(x)
    return (h / 3) * (y[0] + 4 * np.sum(y[1::2]) + 2 * np.sum(y[2::2]))

a = 0
b = 1
n = 1000
print(simpson(a, b, n))
```

在这个代码实例中，我们使用了Simpson积分法来计算函数f(x)在区间[a, b]上的积分。Simpson积分法是一种高精度的数值积分方法，它通过将区间划分为多个小区间，并对每个小区间内的函数值进行求和来计算积分。

## 4.2 数值微分代码实例

我们来看一个简单的数值微分计算的代码实例：

```python
import numpy as np

def f(x):
    return np.exp(-x**2)

def derivative(f, x, h=1e-5):
    return (f(x + h) - f(x)) / h

x = 0
print(derivative(f, x))
```

在这个代码实例中，我们使用了前差差分法来计算函数f(x)在x处的微分。前差差分法是一种简单的数值微分方法，它通过将函数在x处的值与函数在x+h处的值进行差分来计算微分。
