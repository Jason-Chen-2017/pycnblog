# 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的推荐系统的代码实例，以及其详细解释说明。

```python
import numpy as np
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity
from sklearn.preprocessing import MinMaxScaler

# 加载数据
data = pd.read_csv('data.csv')

# 数据预处理
scaler = MinMaxScaler()
data_scaled = scaler.fit_transform(data)

# 计算相似度矩阵
similarity_matrix = cosine_similarity(data_scaled)

# 推荐用户的前5个最相似的项目
user_id = 1
similar_items = np.argsort(-similarity_matrix[user_id])[:5]
print(similar_items)
```

这个代码实例中，我们使用了`sklearn`库中的`cosine_similarity`函数，计算了用户1的项目与其他所有项目之间的相似度。然后，我们根据相似度排序，选取了用户1的前5个最相似的项目作为推荐。
