## 1.2 人工智能与机器学习的核心概念

人工智能（AI）是一种使计算机能够像人类一样思考、学习和决策的技术。人工智能可以分为以下几个方面：

1. **知识工程：**这是一种将人类的知识编码到计算机中的方法。

2. **规则引擎：**这是一种使计算机根据一组规则进行决策的方法。

3. **模式识别：**这是一种使计算机从数据中识别模式的方法。

4. **神经网络：**这是一种使计算机模拟人类大脑的方法。

机器学习（ML）是一种使计算机从数据中学习出规律的技术。机器学习可以分为以下几个类型：

1. **无监督学习：**这是一种使计算机从未标记的数据中学习出规律的方法。

2. **有监督学习：**这是一种使计算机从标记的数据中学习出规律的方法。

3. **半监督学习：**这是一种使计算机从部分标记的数据中学习出规律的方法。

4. **强化学习：**这是一种使计算机从环境中学习出行为的方法。
