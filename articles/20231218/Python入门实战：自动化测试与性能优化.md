## 4.1 单元测试的代码实例

以下是一个简单的单元测试代码实例：

```python
import unittest

class TestCalculator(unittest.TestCase):

    def test_add(self):
        self.assertEqual(calculator.add(1, 2), 3)

    def test_subtract(self):
        self.assertEqual(calculator.subtract(5, 3), 2)

if __name__ == '__main__':
    unittest.main()
```

在这个例子中，我们使用Python的`unittest`库来编写单元测试。我们定义了一个`TestCalculator`类，并在该类中定义了两个测试用例`test_add`和`test_subtract`。这两个测试用例使用`assertEqual`方法来验证计算器的加法和减法功能是否正确。

## 4.3 性能测试的代码实例

以下是一个简单的性能测试代码实例：

```python
import time
import requests

def test_response_time():
    start_time = time.time()
    response = requests.get('https://api.example.com/users')
    end_time = time.time()
    response_time = end_time - start_time
    self.assertLess(response_time, 1)

if __name__ == '__main__':
    test_response_time()
```

在这个例子中，我们使用Python的`time`和`requests`库来编写性能测试。我们定义了一个`test_response_time`函数，该函数通过发送HTTP请求来测试API的响应时间。然后，我们使用`assertLess`方法来验证响应时间是否小于1秒。

## 4.4 性能优化的代码实例

以下是一个简单的性能优化代码实例：

```python
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def optimize_calculator():
    calculator = {'add': add, 'subtract': subtract}
    return calculator

if __name__ == '__main__':
    optimized_calculator = optimize_calculator()
    print(optimized_calculator['add'](1, 2))
    print(optimized_calculator['subtract'](5, 3))
```

在这个例子中，我们定义了一个`optimize_calculator`函数，该函数返回一个优化后的计算器。我们使用`add`和`subtract`函数来实现加法和减法功能，这些函数相对于原始的`calculator.add`和`calculator.subtract`函数更高效。然后，我们使用`print`函数来输出优化后的计算器的加法和减法结果。
