## 2.1 禅宗的基本概念

禅宗是一种宗教和哲学思想，起源于中国。禅宗的核心概念包括：

- 无思念（無思念，Wú Simìng）：无思念是指在行动时完全放下思念，只专注于当前的行动。这种状态下，人们可以更好地感受到当前的瞬间，更好地理解自己和周围的事物。
- 直指人心（直指人心，Zhí Zhí Rén Xīn）：直指人心是指通过禅宗的哲学思想和实践方法，可以帮助人们直接触及自己内心的真实本性。
- 一切皆空（一切皆空，Yī Qiǎng Měi Kōng）：一切皆空是指所有的事物都是相对的，没有绝对的存在。这种观念可以帮助人们放下自己的偏见和限制，更好地理解世界和自己。

## 4.1 代码实例：求两个整数的最大公约数

假设我们需要设计一个程序，用于计算两个整数的最大公约数。这个问题可以通过辗转相除法（Euclidean algorithm）来解决。辗转相除法是一种常用的算法，用于计算两个整数的最大公约数。

以下是辗转相除法的具体实现：

```python
def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b)
```

在这个代码实例中，我们定义了一个名为 `gcd` 的函数，它接受两个整数 `a` 和 `b` 作为输入参数。如果 `b` 等于 0，则返回 `a`；否则，返回 `gcd(b, a % b)`。
