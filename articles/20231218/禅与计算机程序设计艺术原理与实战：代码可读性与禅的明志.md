```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

这个排序算法是一种简单的排序算法，它的核心思想是通过多次遍历数组，将较大的元素逐步移动到数组的末尾，从而实现排序。
