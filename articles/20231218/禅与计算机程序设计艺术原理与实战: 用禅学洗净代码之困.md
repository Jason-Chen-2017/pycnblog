## 4.1 代码实例

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

从上面的代码实例中，我们可以看出，冒泡排序算法的核心思想是通过多次交换相邻的元素，将较大的元素逐步冒泡到数组的后面。这种方法的时间复杂度为O(n^2)，空间复杂度为O(1)。
