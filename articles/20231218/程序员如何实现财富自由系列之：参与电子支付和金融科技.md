3. 2000年代：移动互联网的蓬勃发展，移动电子支付逐渐成为主流。

## 3.1 加密算法

加密算法是电子支付中非常重要的一种技术，它可以确保支付过程中的数据安全。常见的加密算法有：

1. 对称加密：使用同一个密钥进行加密和解密的加密算法，如DES、3DES等。
2. 非对称加密：使用不同密钥进行加密和解密的加密算法，如RSA、DH等。

### 3.1.1 RSA算法

RSA算法是一种非对称加密算法，它的核心思想是使用两个不同的密钥进行加密和解密。RSA算法的具体步骤如下：

1. 生成两个大素数p和q，然后计算n=p*q。
2. 计算φ(n)=(p-1)*(q-1)。
3. 选择一个随机整数e（1<e<φ(n)，且与φ(n)互质），然后计算e的逆元d（modφ(n)）。
4. 公钥为(n、e)，私钥为(n、d)。
5. 对于加密，将明文m加密为ciphertext c，使用公钥（n、e）进行加密。
6. 对于解密，将加密后的ciphertext c解密为明文m，使用私钥（n、d）进行解密。

### 3.1.2 AES算法

AES算法是一种对称加密算法，它使用同一个密钥进行加密和解密。AES算法的具体步骤如下：

1. 选择一个密钥key和一个块大小block size。
2. 将明文分为多个block，然后对每个block进行加密。
3. 对于每个block，使用密钥key进行加密。
4. 将加密后的block组合成加密后的明文。

## 4.1 RSA算法实例

### 4.1.1 生成两个大素数

```python
import random

def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def generate_prime():
    while True:
        p = random.randint(100, 2**16)
        if is_prime(p):
            return p

p = generate_prime()
q = generate_prime()
```

### 4.1.2 计算n、φ(n)、e、d

```python
n = p * q
phi_n = (p - 1) * (q - 1)

def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

def extgcd(a, b):
    if b == 0:
        return 1, 0
    else:
        x, y = extgcd(b, a % b)
        return y, x - y * (a // b)

e = random.randint(1, phi_n - 1)
   
while gcd(e, phi_n) != 1:
    e = random.randint(1, phi_n - 1)

d = extgcd(e, phi_n)[0]
```

### 4.1.3 加密和解密

```python
def encrypt(m, e, n):
    return (m ** e) % n

def decrypt(c, d, n):
    return (c ** d) % n

m = 123
c = encrypt(m, e, n)
print("加密后的明文：", c)

m_dec = decrypt(c, d, n)
print("解密后的明文：", m_dec)
```

## 4.2 AES算法实例

### 4.2.1 加密

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

key = get_random_bytes(16)
iv = get_random_bytes(16)

cipher = AES.new(key, AES.MODE_CBC, iv)
m = b'hello world'
c = cipher.encrypt(m)
print("加密后的明文：", c)
```

### 4.2.2 解密

```python
cipher_dec = AES.new(key, AES.MODE_CBC, iv)
m_dec = cipher_dec.decrypt(c)
print("解密后的明文：", m_dec)
```
