Redis（Remote Dictionary Server）是一个开源的高性能的键值存储系统，由Salvatore Sanfilippo在2009年开发。Redis支持数据的持久化，不仅仅提供简单的键值存储。它的关键功能包括数据的排序（string, list, sets, sorted set），键的自增、自减操作，定时任务等。Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。

Redis 是一个使用 ANSI C 语言编写、遵循 BSD 协议、支持网络、可基于内存的 key-value 存储数据库的一种。Redis 的主要特点是简单，快速，可靠。Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，当重启的时候加载进行使用。
