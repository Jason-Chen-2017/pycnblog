### 3.1.1 数据去重
数据去重是对数据资源进行去重的过程，主要包括以下步骤：
1. 读取数据资源；
2. 创建一个空列表；
3. 遍历数据资源，将每个元素添加到列表中；
4. 遍历列表，将重复的元素删除；
5. 返回列表。

数学模型公式：
$$
R = \{x \in S | \forall y \in S, x \neq y\}
$$

## 4.1 数据清洗案例
### 4.1.1 数据去重
```python
def remove_duplicates(data):
    result = []
    for item in data:
        if item not in result:
            result.append(item)
    return result

data = [1, 2, 2, 3, 4, 4, 5]
print(remove_duplicates(data))
```
输出结果：
```
[1, 2, 3, 4, 5]
```

### 4.1.2 数据过滤
```python
def filter_data(data, condition):
    result = []
    for item in data:
        if condition(item):
            result.append(item)
    return result

data = [1, 2, 3, 4, 5]
even_numbers = filter_data(data, lambda x: x % 2 == 0)
print(even_numbers)
```
输出结果：
```
[2, 4]
```

### 4.1.3 数据填充
```python
def fill_data(data, default_value):
    result = {}
    for key, value in data.items():
        result[key] = value if value is not None else default_value
    return result

data = {'a': 1, 'b': None, 'c': 3}
print(fill_data(data, 0))
```
输出结果：
```
{'a': 1, 'b': 0, 'c': 3}
```
