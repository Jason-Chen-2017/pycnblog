                 

# 1.背景介绍


在过去的几年里，全球股市已经由一帮炒房客、保险套利者、套牢狗屎者、炒币之徒等无脑群体取代了年轻、聪明的IT从业人员。无论是“马云买房”还是“黄金叠加购物”，现在无处不在的都是机器人、算法驱动的“白嫖”。这给创业者或技术精英们带来巨大的冲击，但同时也引入了新的机遇——可以从事期权交易。

期权（Options）是一个合约，规定了某种商品的买方与卖方之间的某种权利义务关系。它是一种风险控制工具，能够最大化投资者的收益。在期权交易中，以买入或卖出某只债券为例，买方可以通过向期权持仓人付费得到更高价值的证券，而卖方则可获得利息。期权的价格并非基于市场价值，而是基于证券的预期收益率和其他相关信息。通过行使期权所得的差额即为收益。

对于传统的股票交易来说，如果市场上某个公司正在发生剧烈的走势，那么通常只有暴涨暴跌时才会交易。相反，期权交易则是设定预期收益率，比如说3%，期权持仓人如果短期内可以获利超过3%，那么他们就会买进该股票。但是期权交易不是真正的交易，而只是一种合约上的权利义务关系，需要借助数字货币进行交易，而且期权并不是天然的开放式市场，需要个人或机构根据不同的风险承受能力进行选择。因此，相对于传统股票交易来说，期权交易更多的是一种投资方式。

不过，由于期权交易涉及较高的风险，投资者需要谨慎交易，特别是在股市大幅下跌或者亏损时，期权交易可能面临较高的风险。另外，由于期权是一种时间敏感型产品，过去的交易往往无法适应未来的环境，可能会出现赔钱的情况。因此，成功参与期权交易还需要投资者有较强的独立判断力和风险意识。

# 2.核心概念与联系
## 期权合约
期权合约是指由标普500指数（S&P 500）认可并发行的美国国内股票发行人授权的，以国内收益为基础，以标准化的波动率的形式授予持有期权者的权利。这类合约的发行量一般远远大于普通股票。

在中国，期权合约也被称为“认购期权”，是指受国家法律保护，由股东在指定时间内以固定价格购买具有给定的溢价权的权利，用来换取远期无风险套利的权利。认购期权通常包括认购权和申报权两部分，申报权的存在是为了确保认购权有效。

每一个期权合约都有一个唯一的标识符号，在中国习惯用期权代码表示，如"000300-C-3200-P"代表中国平安银行连续三个月认购权，持有期权将对平安银行股票承担溢价3.20%。

## 期权结构
期权合约可以分为以下几个主要组成部分：

1. 主动方：期权合约的主体，它是承诺发行人提供的权利；
2. 被动方：期权合约的客体，它是承诺给主动方的权利；
3. 条件：合约生效的基础条件；
4. 交割规则：如果合约发生了交割，主动方或被动方的相应份额将按照一定比例分派给持有人；
5. 生命周期：合约在市场上流通的时间长度；
6. 演变期：在期权合约上线市场时，可以选择在初始时间段开放给持有人竞价交易的期限，只有在演变期结束后方可最终确认交易结果。

例如，双边认购期权的主动方就是指认购期权合约的发行人，它的认购权力通常是免费的；被动方是指发行人给予的股票折价差额，也就是期权合约在执行时承诺给主动方的权利；条件一般包括发行人给予的折价幅度、时间范围等；交割规则一般是主动方认购权的当天交易，被动方将按照一定比例分派给持有人；生命周期一般是一年或三年；演变期一般是1个月至6个月。

## 期权的属性
在交易期间，期权有如下几个主要属性：

1. 可交易性：合约认可的交易类型有认购权、认沽权、衍生权，也就是主动方的证券的买卖；
2. 方向性：期权合约是一个双边的关系，分别是主动方承诺给被动方的证券，也反过来被动方承诺给主动方的权益；
3. 投资金额：期权合约的投资金额一般较小，通常只有几百美元左右；
4. 成交量：期权合约的成交量较少，一般都是几千笔/秒左右，主要因为合同数量有限；
5. 操作复杂度：期权合约涉及到多种复杂的法律法规，需要各方签订严格的协议，需要对标的证券做充分的了解和认识，以及熟练的操作技巧。

## 期权的优势
相对于其它期货合约来说，期权交易具有如下几个优点：

1. 没有涨跌限制：期权合约没有涨跌限制，由主动方在确定条件下发行合约，被动方即可按照条件是否满足认购并行销，实现对股票市场的主动管理；
2. 可以高频交易：期权合约可以高频交易，一旦条件满足，立刻就能成交，在节假日大宗交易时，不足以带来损失；
3. 允许交易不稳定品种：期权合约不仅可以在股票市场主动管理，也可以用于管理债券、贵金属等其他不涉及股票市场的品种，且成交手续费低廉；
4. 有投资者保证：期权合约没有交易保证金或交易成本，而且保证了每个投资者的利益。
