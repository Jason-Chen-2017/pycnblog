                 

# 1.背景介绍


## 数据中台介绍
数据中台，也称数据管道中心，是一个企业级的数据服务平台，它由数据采集、数据存储、数据分析、数据展示、数据安全等多个环节组成。数据中台使得公司内部数据能够有效地进行整合、提炼、计算、传输、分析和呈现，通过统一的平台实现了数据共享、数据应用、数据驱动的价值实现。

在“数据分割治理”等国内外管理理论中，很多人认为数据应属于战略资源，只有一个数据中心才能实现数据共享，因此，数据中台往往被定义为“集成战略资源的集合”。而实际上，数据中台的边界不应该限制于单一数据仓库或集中式数据中心，其核心概念也超越了传统意义上的企业IT架构。

数据中台作为一种架构模式，虽然定义繁琐，但它的核心目标仍然很清晰——为企业提供数据服务，解决数据孤岛和信息孤岛问题。下面我们结合国际金融危机的背景，来讨论一下数据中台的定义和角色。

## 国际金融危机背景下的数据中台定义及角色
### 数据中台的定义
国际金融危机期间，全球经济陷入严重衰退，资本市场崩盘，大量国际金融企业遭遇困境。为了应对金融危机带来的冲击，各国政府都在积极寻求资本投放方式的创新，其中以英国为代表，推出了“数字货币”、“区块链”等新的金融工具，并通过数字货币交易所进行资本注入。这些金融工具引起了全球金融市场的注意。

数字货币、区块链等新型金融工具为国际金融市场带来了巨大的变革性冲击，但同时也带来了前所未有的财富创造效应。由于这些工具涉及到高门槛的加密算法、极速的交易速度和海量用户的参与，使得其利用率较低，且容易受到攻击和泄露。因此，如何提升资产管理水平、降低风险、实现长久稳定运行成为当务之急。

近年来，越来越多的国家和地区开始推行各种形式的数字化改革，如智能手机普及、网络游戏爆发、人工智能、数字货币、区块链等。数字化转型正导致数据成为重资产、新绩效和新增长点。

数据中台可以理解为一类新的商业模式，旨在提升数据集成能力和满足企业的业务需求。它可以定义为“一个基于云计算、大数据和机器学习技术体系构建的全生命周期数据服务平台”，包括三个关键要素：数据采集、存储、处理和共享；数据分析、挖掘、可视化；数据服务，即数据中台为用户提供数据的获取、分析和服务。数据中台将数据从源头采集、汇总、计算、存储和整合，形成了完整、结构化的知识库，为企业提供丰富的业务数据和服务支持。

据国际银行业协会统计，截至2017年底，全世界有29个主要金融机构建立了数据中台，其中包括欧洲央行、美国证券交易委员会（SEC）、日本三菱信託、中国工商银行、德国支付宝、法国巴黎银行、瑞士苏宁易购、瑞典Swedbank、香港星展银行、澳门兴业银行、韩国KB국민은행等。

### 数据中台的角色
数据中台主要负责数据的采集、存储、处理和共享，包括数据采集平台、数据仓库、离线数据库、实时计算集群、ETL工具、数据交换平台和数据接口。同时，数据中台还需要具备数据分析、挖掘、可视化能力，包括数据质量保障、数据建模、数据流动管理、数据发现与分析、智能运营等功能模块。此外，数据中台还需具备数据服务能力，包括数据接入、数据服务、数据产品化和数据流通等方面。
