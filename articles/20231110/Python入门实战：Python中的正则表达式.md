## 什么是正则表达式（Regular Expression）？
正则表达式（英语：Regular Expression，又称正规表示法、正规语法、正则语言），是一种文本模式匹配的规则语言，它是一种字符串匹配的工具，可以用来检查一个串是否含有某种子串，查找、替换或删除符合条件的字符串。正则表达式作为一种文本处理工具，在电脑中有着广泛的应用。比如在文本编辑器中搜索，常用的正则表达式有：
- 查找文本中的指定字符：`.`表示任意单个字符
- 搜索多种字符：`|`表示"或"关系
- 指定字符数量范围：`?`, `+`, `{m}`, `{m,n}`等符号分别表示"0次或一次", "1次或多次", "m次", "m到n次"
- 重复字符、前后断言、贪婪与非贪婪：`*`, `+`, `?`, `()`等符号可以实现重复匹配、前后断言、贪婪与非贪婪
- 分组与引用：`()`可以将多个字符分组成一个整体进行引用
- 模板文本：`\t`, `\n`等转义字符可以方便地表示特殊字符

使用正则表达式可以对文本数据进行精确、高效、智能的匹配，并且处理起来相当简单。本文将从正则表达式的概念、语法、用途三个方面阐述正则表达式在Python中的基本用法。
## 为什么要学习正则表达式？
任何编程语言都离不开字符串（String）这一数据类型。但是，字符串数据的匹配、过滤、替换都需要通过一些强大的字符串处理函数或者模块。而正则表达式作为一种高效、灵活的字符串匹配工具，是一种十分便捷的方法。它可以帮助我们快速准确地筛选出文本中所需的内容。因此，了解并掌握正则表达式对于每一个基于字符串的编程人员都是非常重要的技能。而且，熟练掌握正则表达式还有助于解决一些复杂的问题，如数据清洗、文本分析、数据分析等。

## 创建正则表达式对象
在正则表达式中，`^`表示行首，`$`表示行尾，`.`表示任意字符，`\d`表示数字，`\D`表示非数字，`\w`表示字母数字下划线，`\W`表示非字母数字下划线，`\s`表示空白字符，`\S`表示非空白字符。当然，还可以通过元字符集构造更复杂的正则表达式。

使用`re.compile(pattern)`函数创建一个正则表达式对象。其中，`pattern`参数是一个字符串，表示正则表达式的模式。创建完成之后，就可以用该对象的`match()`方法进行匹配，也可以用其他方法进行各种操作。例如，可以使用`search()`方法查找第一个匹配的位置；使用`findall()`方法查找所有匹配的子串；使用`sub()`方法替换满足条件的子串。下面举例说明如何创建和使用正则表达式对象：
```python
import re

pattern = r'\b[A-Z][a-z]*\b' # 匹配首字母大写，后跟小写字母的字符串

text = 'Hello World!'
match_obj = re.match(pattern, text) 

if match_obj:
    print('Match found:', match_obj.group()) 
else:
    print('No match found')
    
result = re.findall(r'[aeiouAEIOU]+', text) # 查找所有由元音构成的字符串
print('Vowels:', result)

new_text = re.sub('\d+', '*', text) # 替换所有数字为'*'
print('New Text:', new_text)
```
输出结果如下：
```python
Match found: Hello
Vowels: ['o']
New Text: Hello World!
```
