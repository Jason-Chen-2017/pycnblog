                 

# 1.背景介绍


光罩（Contra-Illumination Mask (CIM)）是一种在真空光透镜上采用金属材料的反射结构光学系统。该光学系统能够在一定程度上抑制环境光的影响并提供大功率的辐照度。由于其简单易用、设计简便、集成度高、性能优良等特点，广泛应用于集成电路（IC）制造、半导体制造、照明系统制造、磁控光学系统制造等领域。目前，光罩已经成为一项重要的技术标准，已经成为世界各国以及各种工业领域的标配产品。

然而，光罩也存在着诸多限制和不足。在设计时，需要注意以下几点：

1. 光罩的尺寸：光罩的尺寸一般为55毫米左右。过小或过大的光罩可能会导致信号损失或干扰；过大或者过小的光罩会增加非线性放大效应；如果光罩太长，可能会带来噪声和散热不平衡；如果光罩太短，会影响信号传输距离，使得其无法有效防止光照射到晶格外层。因此，必须根据光路长度及其他条件合理选择光罩的尺寸。

2. 信号传输路径：光罩必须能够直观地看到光照射到晶片的过程，并且光罩本身不能损坏信号传输路径。

3. 处理速度：对于制作周期要求严苛的集成电路制造领域来说，光罩的制造速度至关重要。需要保证制作效率不低于每秒10000次（Hz）。同时还需要考虑光罩的厚度、质量、精度、功能、尺寸等因素，才能确保质量可靠、耐用。

4. 操作复杂度：光罩的安装和维护都比较麻烦，而且容易出现故障。需要设计具有良好操作习惯的光罩操作人员。另外，需要考虑到光罩随着时间的推移会有性能衰减的问题。

# 2.核心概念与联系
## 2.1 概念
### 2.1.1 反射率
物体的反射率是指从特定方向传播入射光束的比例。即发生反射后，入射光仍然能传达出的光谱宽度除以总的入射光宽度。反射率通常用百分比表示。

一般来说，金属表面材料的反射率大约在90%左右，而非金属表面材料的反射率则更低一些。例如玻璃的反射率大约为75%，铝、银的反射率为50%。

### 2.1.2 光通量
光通量（Power of light）表示单位时间内传播的光能量。光通量等于亮度（Lumen）乘以功率（Watt）。其中，亮度指单位时间的流明数，功率指单位时间的流动能。如摄氏度（°C）的升数乘以用电量为单位的功率，就可以计算出每分钟多少度开的水的温度。但是，光通量又与波长、频率有关。如若要计算某一方向上的某一波长的信号强度，就需要知道单位时间内光在该波长上的平均通量。这一方面可以通过某种刻蚀仪器获得，另一方面也可以通过测量不同波长下的光通量来估计。

### 2.1.3 光通量放大效应
光通量放大效应（Power transfer efficiency PTE）指的是当光在导体内循环传递时，受阻隔离效应引起的相位延迟变化。它与光速关系密切，随着光通量的增加而增加。通常用百分比表示。

光通量放大效应是指光在导体内逆向运动，且受阻隔离效应的影响，引起的相位延迟变化。也就是说，随着光通量的增大，导体内的电场分布将产生交叉，不同的相位可能相互重叠，从而导致时间间隔延迟。相位延迟相对于光速增加，因此所需的时间也增加。

通常情况下，光通量放大效应在50%以下是可以接受的，超过75%时，可以通过精心设计光学设备进行降低。但绝对值较小，通常不到10%。

### 2.1.4 熔融损耗
熔融损耗（Thermal loss）是指物体表面由于物理性质决定的热力作用而形成的额外电阻损耗，它是由元件自身的大小、形状、材料、温度、相对湿度、磁场、加热效应等所决定的。熔融损耗越大，电阻损耗越大，因而成本也越大。

一般来说，金属表面材料的熔融损耗一般在15~20%，而非金属表面材料的熔融损耗通常更大。光罩制造往往还会牵涉到材料的结晶、纳米片的化学组成、烧焊温度、流动效率、耐候温度等诸多方面，为了有效控制熔融损耗，光罩设计时应该考虑到以下几个方面：

1. 导体材料：选择经过工艺优化的导体材料，有助于避免损耗过大。

2. 温度控制：导体的最低温度控制在90度上下，超出这个温度之后会产生电磁辐射，造成损耗。因此，需要将导体温度定为100～120度，并在每一次使用之前，进行充分的充气和保护。

3. 纳米片厚度：纳米片厚度也是一个影响熔融损耗的因素。较厚的纳米片表面受到腐蚀和渗漏电流的影响较少，会减少熔融损耗。

4. 使用层：由于光罩层的厚度通常较大，熔融损耗会被隐去。因此，需要在导体之间、光罩与基底之间的空间留有足够的空隙，以免受到熔融影响。

5. 海绵喷涂：海绵喷涂是用来固定光罩的。海绵的厚度越大，则损耗越大。所以，为了保持较低的熔融损耗，建议使用轻柔的海绵。

## 2.2 CIM制作原理
光罩（Contra-Illumination Mask (CIM)）是一种在真空光透镜上采用金属材料的反射结构光学系统。它的主要目的是抑制环境光的影响并提供大功率的辐照度。其原理是在真空光透镜上装一层反射镜和一个倒置的金属镜片。其中，反射镜与真空光透镜同轴，负责收到光并通过反射镜反射到镜片上。反射镜中的倒置金属镜片构成了光罩。光源发出的光线通过真空光透镜进入光罩。反射镜上的倒置镜片通过旋转进入倒置镜片的孔，使得光在真空光透镜上反弹回真空，形成反射景象。这样，通过光罩，环境光在一个区域内被抑制了。

光罩由三个主要部件组成：1）真空光透镜；2）反射镜；3）倒置金属镜片。真空光透镜与光源直接相连，通过收集到光信号，然后转换成电流脉冲。反射镜的外侧与真空光透镜同轴，通过折射光子把光子反射到倒置金属镜片上。倒置金属镜片就像一个光学双极透镜，接收到反射到它的光子，将其转换成电压脉冲，送给电源。结果，反射镜上的倒置金属镜片就像一道透明屏幕，屏蔽掉环境光的影响。

光罩的设计需要考虑三个方面的参数：光通量（Power of light），信号传输距离，以及每秒光通量（Light Power Seconds）。光通量指的是光束的多少，信号传输距离指的是光罩的距离，每秒光通量指的是单位时间内能通过光罩的光通量。

1. 信号传输距离：为了满足光线的信号传输需求，光罩必须能够直观地看到光照射到晶片的过程。信号传输距离是指光罩需要单步传输多少光线。一般地，光罩的信号传输距离从50米开始，逐步增加到200米左右。

2. 光通量：光通量就是单位时间内能通过光罩的光的多少。通常使用的单位是光强度。由于光通量越大，光罩所能承受的照射强度越强，因而实现抑制环境光的效果。但是，也正因为光通量越大，信号传输距离也就越远。

3. 每秒光通量：每秒光通量是指单位时间内能通过光罩的光的多少。每秒光通量的大小决定了光罩的抗闪烁能力，也是判断光罩是否具有良好的视觉效果的重要依据。以每秒30万次为标准，抗闪烁能力较差。当每秒光通量大于30万次时，就认为光罩具有良好的视觉效果。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 透射率
光透射是指由于介质的折射特性，某些波长范围内的波长所吸收的光子重新射出，称为光透射。光子在晶粒内部反射、扩散和传播，会产生两个不同的波长。称为垂直波长（vertical wave length）和水平波长（horizontal wave length）。光子的垂直波长是指传播的方向，它垂直于媒质的折射率；水平波长是指传播方向的延申，它不是垂直于媒质的折射率，而是水平于媒质的折射率，称为横向波长。在光学透镜中，垂直波长表示入射光线，水平波长表示出射光线。

光透射率（Transmittance）是指透射方向上的光线的光通量。它等于入射光线的光通量减去反射到其他方向的光线的光通量，再除以入射光线的光通量。光透射率一般用百分比表示。

对于直径为d的光学透镜，入射光线的光通量为A，出射光线的光通量B=Asy，其中s是透射系数，sy表示媒质的衬底散射系数。则有

T=A/Asy=(A*sinθ)/(Ad+Adcosθ)=√(A^2/(Ad^2+Adcosθ^2))

由于光路中光子的相位不同，A在不同的相位下会具有不同的入射角度θ，则θ与B之间的对应关系如下：

B=A/sinθ

由此，可以求出θ。将上面公式代入，有

T=√(A^2/(Ad^2+Adcosθ^2))=√(A^2/(Ad^2+Ad(tan⁡θ)))

此时，θ为入射角度，为一函数，所以光透射率是一函数。

可以看出，当θ接近0时，光透射率趋近于1，说明入射角度很小，透射情况不明显；θ接近90度时，光透射率趋近于0，说明入射角度很大，光线完全被透射。为了分析光学透镜的光通量和透射率曲线，可以通过以下几个方面：

1. 横轴：光通量，即A

2. 纵轴：光透射率，即T

3. 曲线：光透射率与入射角度的关系曲线。在这个曲线上找到最大值，作为最佳光通量和最佳透射率。

## 3.2 设计原则
光罩的设计原则是：减小环境光的影响，提高晶体晶化、磨损等效应的抑制，提升晶体制造成本。

为了设计出良好的光罩，应遵循以下几个原则：

1. 性价比原则：根据预算和使用效益，确定光罩的质量，尺寸，费用等。

2. 视觉效果原则：保证光罩具有良好的视觉效果，做到清晰、大方、突出，并在不同颜色条件下显示出色。

3. 熔融损耗控制：采用高等级、耐用的金属材料，避免出现疏松、暴露以及耗散。

4. 抗拉伤效果：光罩的拉伤抗性好，能够抵御拧紧、撕裂、磨损、腐蚀等损坏的损失。

5. 规避混凝土坑洞：设计时应注意排除光罩内部的混凝土坑洞，以避免漆光光泽扩散，影响光罩的制造质量。

## 3.3 设计方法
光罩的设计方法，主要包括两种：批量生产和零件生产。

### 3.3.1 批量生产
批量生产（BGA）的基本原理是：用同一种型号、规格的导体材料、相同的腔体材料以及相同的数量的光罩，在相同的温度和条件下，一起冷却、加热，形成一个整体，然后通过连续浸漫、锤击、摩擦、挤压、蜂鸣、振动等手段使其产生微弱的反射光。最后，按照要求进行精炼，得到一个完美无瑕的光罩。这种方法的好处是简单快速，但缺点是批量生产费用高昂。

### 3.3.2 零件生产
零件生产的基本原理是：先按批次或产品序列分解出零部件，然后逐个安装到光罩上。这种方法的好处是分散化管理，节省成本，适用于高端客户。但缺点是需要付出许多精力，耗时长。

## 3.4 CIM参数计算方法
CIM的参数包括：

* 材料：导体材料的类型、性质，以及导体的制造工艺。

* 腔体材料：腔体材料的类型，如镀锌钢、酒精、塑料、金属材料、石墨烯等。

* 光通量：光通量指的是单位时间内通过光罩的光的多少。通常使用的单位是光强度。

* 尺寸：光罩的直径、高度、宽度等。

* 厚度：光罩的厚度，取决于导体材料、腔体材料的厚度和其他参数。

* 长度：光罩的长度，单位为mm。

* 最小反射率：光罩最低的反射率，通常为75%左右。

* 入射角度：光源入射光罩的角度。

* 反射率：指透射到导体上的光线的百分比，反射率一般用百分比表示。

CIM参数的计算方法：

光源入射角度θ：利用光源发出的高频光束和光罩中心产生的两根垂直于各自入射方向的光线，两根光线的夹角θ即为入射角度。

单位时间内通过光罩的光的多少A：使用频率为f（以赫兹为单位）的高频光源发出的光线穿过光罩，然后在光程l（以米为单位）处反射回光源，其中入射角度θ为90度，则有：

A=f * l / sin90 = f * l * tanθ

单位时间内通过光罩的光的强度power：单位时间内通过光罩的光的强度，power = A / (1e-9 * 1e-3 * pi * d^2)，其中d是直径，pi是圆周率。

入射角度θ：等效电阻的大小等于入射角度θ的余弦值，即r = cosθ。

入射电压V：光线在导体表面的相位差所形成的电场，受导体材料的电导率和电荷分布影响。入射电压V = r * I / L，其中r为入射角度的余弦值，I为电流，L为导体表面积。

反射率：电流经导体传递到其他位置，继续向前传播过程中，除了导体所反射的部分外，其他部分都会受到相位延迟影响，这种影响叫做相位差。对于光路来说，相位差相当于光子的折射，相位差的大小为θ，即折射角度。入射电压V的大小受θ和导体材料的导通率影响，反射率reflectance = Vsinθ。

由此，可以计算CIM的参数。
