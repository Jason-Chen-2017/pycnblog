# 2.核心概念与联系
## 2.1 数据类型与变量
- int（整数）：
在计算机内部，整数使用补码表示法，也就是用一个正负号来表示这个数字是否为正数或负数。整数类型的变量通常用于存储和计算数字，也可以参与数值计算。例如：a = -3，b = 9，c = a + b。
- float（浮点数）：
浮点数类型用于表示小数。它的大小与所占内存大小无关。浮点数类型的变量通常用于表示小数和比较小的数字。例如：x = 3.14，y = 1.0e-3。
- bool（布尔值）：
布尔值只有两种取值True和False，可以用来做条件判断和逻辑运算。它可以用and、or、not三个关键字来进行逻辑运算。例如：flag = True and False。
- str（字符串）：
字符串是由多个字符组成的一段文字。字符串类型的变量可以存储、修改文本信息。字符串可以使用单引号''或双引号""括起来，也可以使用三引号'''或"""多行输入。例如：name = 'Alice'，address = "123 Main St"。
- list（列表）：
列表是一个元素的集合。列表中的每个元素可以不同类型，同样可以嵌套其他列表或者字典。列表可以使用方括号[]表示。例如：[1, 'Hello', [2.3, True]]。
- tuple（元组）：
元组是一个不可变序列，即其元素不能修改。元组可以使用圆括号()表示。例如：('apple', 1) 。
- set（集合）：
集合是一个无序不重复元素的集合。集合使用花括号{}表示。例如：{1, 2, 3}。
- dict（字典）：
字典是一个键值对的无序集合。字典使用大括号{}表示，其中每个键值对之间用冒号:分隔。字典中的每个键值对可以根据需求指定任意的对象作为值。字典可以用来保存和组织数据。例如：{'name': 'Alice', 'age': 25}。
## 2.2 条件语句与循环结构
- if条件语句：
if条件语句用于执行条件判断，如果满足条件，则执行相应的代码块。它的语法形式如下：
```python
if condition:
    # 若条件为True时执行的代码块
else:
    # 若条件为False时执行的代码块
```
比如，要实现一个判断年龄是否大于18岁的程序，可以这样写：
```python
age = input("请输入你的年龄:")
if age > 18:
    print("恭喜你！你已经成年了！")
else:
    print("你还不到18岁，还未成年哦~")
```
- for循环语句：
for循环语句用于遍历列表、元组、字符串或其他可迭代对象，每次遍历时，获取列表、元组或字符串中的一个元素，并执行代码块。它的语法形式如下：
```python
for variable in sequence:
    # 对sequence中的每一个元素，执行代码块
```
比如，要打印出[1, 2, 3]中的所有数字，可以这样写：
```python
my_list = [1, 2, 3]
for num in my_list:
    print(num)
```
- while循环语句：
while循环语句类似于for循环语句，但是只会执行代码块，直到指定的条件为假。它的语法形式如下：
```python
while condition:
    # 当condition为True时，执行代码块
```
比如，求1+2+...+n的值，可以通过如下方式实现：
```python
sum = 0
i = 1
while i <= n:
    sum += i
    i += 1
print(sum)
```
## 2.3 函数定义与调用
函数是用于封装特定任务的子程序。它可以提高代码的复用性，降低代码的复杂度。在Python中，函数的定义使用def关键字，函数名后面跟一对圆括号，然后是函数体，最后以冒号结尾。

```python
def function_name():
    # 函数体
```

函数的调用也很简单，直接使用函数名称即可，并在圆括号中传入参数（如果有）。

```python
function_name()
```

### 参数传递
在Python中，函数的参数传递有两种方式：位置参数和关键字参数。

#### 位置参数
位置参数指的是按照顺序把参数传入函数。这种方式在调用函数的时候，需要保证函数定义的形参列表中的参数数量和实际传入的参数数量相同。

```python
def add(a, b):
    return a + b

result = add(1, 2)    # 相当于add(1, b=2)，这里不需要设置b，因为默认值为0
print(result)          # 返回结果为3
```

#### 关键字参数
关键字参数指的是按照参数的名字把参数传入函数。这种方式在调用函数的时候，只需按照函数定义的形参列表中的参数名，把参数传入即可。如果函数定义的形参列表中已经有了一个参数名，那么可以在调用该函数的时候，省略掉这个参数名，直接传入对应的值。

```python
def person(name='Alice', age=25):
    print('Name:', name)
    print('Age:', age)

person()                  # 使用默认参数
person(age=30)            # 使用关键字参数
person('Bob')             # 不使用关键字参数，按顺序传入参数
```

#### 混合参数
Python允许函数同时采用位置参数和关键字参数。

```python
def func(*args, **kwargs):
    pass
```

* `*args` 是可变参数，它收集位置参数，以元组的方式传入，如 `func(1, 2, 3)`；
* `**kwargs` 是关键字参数，它收集关键字参数，以字典的方式传入，如 `func(name='Alice', age=25)`。
