                 

# 1.背景介绍


近年来，随着海量数据、高计算能力和强大的算力支撑，人工智能(AI)正在改变各个行业的运作方式。在这样的背景下，智能生成能源（IGE）领域迎来了巨大的变革。IGE是智能生成能源领域，由智能电网和智能制冷系统组成，通过利用大数据的分析和预测功能，提升机房用电效率、降低能源成本等，促进经济社会发展。截止目前，国内有超过7万家IGE企业获得了政府或企业授信，拥有超过50亿度电。

2019年底，中国联通与集团公司联合举办“机器学习大模型”首届全国线上论坛。来自世界顶级会议及知名院校的嘉宾将分享行业最新技术、关键突破、创新产品等内容，展示各界对人工智能、大数据、云计算、物联网、工业互联网的期待与热情。

从国内来看，山东金马互联网科技有限公司———山东省数据中心有限责任公司、山东易网网络科技股份有限公司、山东医药生物工程集团股份有限公司、山东雅戈尔电气有限公司、山东省核电研究院股份有限公司、山东电子机械化公司、山东瑞威电器股份有限公司、山东核能信息技术股份有限公司、山东聚福环境技术有限公司、山东长青环保科技股份有限公司、山东济川新材料股份有限公司、山东宏达电子股份有限公司、山东莱美特种材料股份有限公司、山东芝麻孵化器有限公司———都取得了阶段性的成果。

2019年，在这里我想向大家介绍一下智能生成能源的一些典型应用场景，包括风电场、太阳能电池板、太阳能电池柜、风力涡轮机、燃气发电场等。为了更好的阐述应用案例，以下内容仅针对几个主要的应用领域进行简要阐述。

3.云计算平台能源管理及精准调度

    IGE的核心问题之一是如何为客户提供最佳的能源管理服务。目前国内IGE企业使用的管理工具主要是电表监控系统，其只能快速发现能源系统的问题，但无法对系统内部产生的负载进行精细化管理。如何根据市场变化及消费者需求，智能化地分配能源资源并进行精准管理是一个很重要的课题。山东聚福环境技术有限公司基于区块链的分布式数据库技术打造了一套能够实时跟踪客户电力消耗、能源管理、用户健康状况等信息的系统。该系统支持多种模糊查询、智能推荐、语音交互、历史轨迹回顾等多维度管理功能，有效降低能源成本、提升运行质量。

4.风电场智能管理

    在风电场中，存在很多智能化运维难题。比如如何减少环保污染、精确响应供需双方需求，如何提升空气清洁效率、如何优化运行维护成本？基于先进的机器学习和计算机视觉技术，山东雅戈尔电气有限公司开发了一种智能检测方案。该方案利用移动摄像头采集和分析风速、风向、湿度等信息，自动识别异常情况并做出反应，对风场进行实时掌握，并通过站点管理解决人员安排及过程改善。

5.燃气发电场智能智能网关

    发电量激增后，如何实现燃气供应链管理的精准化、高效率，成为越来越多企业所面临的课题。山东济川新材料股份有限公司基于深度学习技术设计了一种边缘计算平台，为客户提供一站式的设备远程监控、报警和管理服务。该平台采用智能网关对设备进行集中管理，确保能源安全、降低成本，同时结合智能分析、AI协同等综合化技术帮助客户精准定位和排除故障。

6.太阳能电池板智能协同管理

    根据智能电网的规定，企业不得以任何手段实施跨区间的拆分电动车充电。山东宏达电子股份有限公司及其他合作伙伴在完成了第一版基础设计、技术开发后，转向到了集成商业模式设计，开发具有大规模布局意义的完整项目。通过制造商业智能化解决方案，该公司将推动全球绿色能源发展，实现环保可持续发展目标。

未来发展趋势
除了这些主流应用场景外，还有许多相关领域的应用案例还在蓬勃发展。例如，智能控制系统，无人驾驶汽车、环保输送等。总体而言，IGE的发展仍然处于蓬勃发展时期，行业也在加速变化中，因此需要更多创新者共同努力，通过共享知识、数据和技术的方式，创造价值。
