                 

# 1.背景介绍


## 大数据的三维特征
由于数据量的激增、传播速度的加快和产业变革的需要，现在已成为社会的重要资源。随着互联网、云计算、移动互联网等技术的发展，数据不断地产生、流动、存储、处理并呈现出全新的三维特征，而这些特征又会影响到我们的日常生活。

首先，数据集中蕴藏着丰富的价值。比如，从航空、金融、保险、交通等行业的数据可以帮助我们掌握机票、交易等信息；再比如，从医疗、健康、养老等行业的数据则提供给医生、病患和社区卫生工作者更加便捷有效的信息服务；最后，在电子商务、物流、零售等行业的数据也被用于分析市场、进行商品推销、优化营销策略。因此，只有充分利用数据中的信息和价值，才能真正实现对人的全面支配。

其次，数据是时代的产物。在过去的20年里，科技飞速发展，人类所创造的全部财富都源于数据产生。数据已经成为经济、文化、政治、教育等领域的基础设施。而现在，数据正在催生新的技术革命，如人工智能、机器学习、大数据分析、虚拟现实、区块链等。

第三，数据产生的社会效应将越来越多地影响我们的生活。据预测，未来两到五年内，数据将产生超过百亿美元的价值。此外，数据还将重塑各行各业的生产方式、组织结构、价值观、工作方式，促进经济全球化。

## 智能大数据定义
智能大数据（Artificial Intelligence + Big Data），是指利用新一代信息技术、包括云计算、大数据、智能机器人、自然语言处理、图像识别、自动决策支持系统等，对海量、高维、复杂的原始数据进行存储、提取、汇总、分析和处理，从而构建具有高度智能的知识引擎、决策引擎和预测引擎，应用于人工智能领域，实现数据驱动、决策助力、预测制定、预警监控、风险管理等各项智能应用场景。

## 智能大数据关键技术
- 数据采集和存储：包括不同类型的数据源、海量数据采集、数据规范化、数据仓库建设、数据质量保证等。
- 数据分析：包括数据挖掘、推荐系统、文本挖掘、图像识别、智能搜索、无监督学习、半监督学习、迁移学习等。
- 数据处理：包括数据清洗、数据合并、数据转换、数据关联、数据降维等。
- 模型训练及部署：包括深度学习、支持向量机、随机森林、神经网络等算法模型的训练、优化、调优、模型集成和部署等。
- 数据接口开发：包括数据接入、API开发、可视化工具开发等。
- 数据质量评估及控制：包括数据质量标准、数据质量检查、数据质量评估、数据质量反馈及控制等。

## 智能大数据应用场景
智能大数据主要应用于如下场景：
- 数据分析：根据大数据做精准的营销决策、产品设计、投资决策、政策规划、风险控制、客户群体划分、运营优化、病人诊断、投诉处理、产品质量控制等。
- 风险控制：通过大数据收集的各种数据资源对矿山安全、公共安全、法律和贸易等方面的风险进行监控、评估、预警、分析和管理，提升矿山整体安全性。
- 实体识别：通过对大量无序和杂乱的数据进行语义理解、实体抽取、知识图谱构建、知识库查询、事件分析、情感分析、意图识别、聚类分析等，形成行业级的知识图谱，用于智能化实体识别、智能问答、智能助理等任务。
- 主题模型：通过大数据挖掘、统计方法等对海量文本或语料进行主题分析、关键词提取、文档分类，形成主题模型，实现智能文本挖掘、智能广告投放、智能内容推荐等功能。
- 图像识别：通过深度学习、图像处理等技术，结合海量的图像数据及其相应的标签，建立智能图像识别、智能视频分析、智能血管检测、智能安防、智能无人机等功能。
- 垂直领域：通过对个性化需求和业务模式进行深度挖掘、洞察、分析和挖掘，形成新一代智能大数据服务体系，具备垂直领域特色和核心竞争力。
