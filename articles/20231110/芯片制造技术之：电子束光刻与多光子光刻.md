                 

# 1.背景介绍


目前，我们都知道，传统的集成电路制造方法是将微小的电子元件拼装在一起组成逻辑电路，然后由逻辑器件来实现各种功能。这种做法虽然简单但效率不高，而且集成电路的复杂性无法满足需求的快速增长。而随着芯片工艺的进步和应用的广泛化，越来越多的人开始转向人工智能+机器学习等新型技术，寻找更加自动化、智能化的制造模式。但是，单纯靠人工智能还远远不能完全解决制造上的困难，而对于电子材料的制备过程也需要新的技术手段。这两年来，由于芯片工艺方面的革命性变化，传统的电子束光刻（EDT）与多光子光刻（MLM）技术逐渐被取代或淘汰。为了解决这些技术所带来的不便和限制，很多企业都已经开始探索人工智能+机器学习+材料科技的结合，研发出了一些高精度、低噪声、低功耗、高品质的新型制造技术。今天，我就先从这两种新型制造技术入手，来谈论一下它们分别适用的领域、优缺点以及未来发展方向。
# EDT(Electron Beam Tapering)：电子束削
电子束光刻技术的特点是使用一定的刀头或开口刃，将一块电阻面板剪切成一个或多个平行且尺寸一致的电子束，然后对电子束进行电流注入以制造导电材料。EDT可将庞大的电阻面板分割成很小的单元，因而制造出的导电材料的厚度比整块电阻面板低得多，其性能也受到限制。EDT技术的优点是工艺简单、生产周期短，但制作时间长；缺点是材料的导通性差、晶体层出现电气短路、导电能力弱、易受高温的影响；另外，EDT容易产生高温过膜层、开窗效应。因此，现在很少再采用EDT制造导电材料。
# MLT(Multi-Laser Melting)：多光子冷焊
多光子光刻技术是一种使用多个激光对金属材料进行冷焊的方法，在表面附近形成的多光子现象中，光子不断扩散并与金属界面发生反射，导致材料逐渐升华或熔融。通过控制光速大小和功率，可以制造出具有良好导电性、多层级结构的导电材料。多光子光刻技术的优点是导电性强、耐腐蚀性好、晶体层厚度不易减少、制作精度高；缺点是生产成本高、加工时间长、传动力较弱；另外，多光子光刻技术曾经被用于制造绝缘子材料、传感器件、纳米管、硅基液晶。
# 概括
目前，主要的制造方法仍然是EDT和MLT两种，但是两者存在明显不同，EDT是剪切电阻面板成一个电子束，而MLT则是用多个激光对金属材料进行冷焊。两种技术都有自己独特的优点，但是面临同样的问题——材料导通性差、缺乏定制性和控制力，同时需要付出高额的生产成本和耗时。所以，未来很可能会出现一种全新的方法——混合型材料制造，它既可以利用原有的电子束光刻技术生成特殊的导电材料，也可以结合MLT技术生成高性能、高导电性的导电材料。
