                 

# 1.背景介绍


## 概述
近几年随着人工智能的火热，各行各业都在大力借助计算机科技进行高速发展。
从词、短语、句子、文档等各种形式的数据到图片、视频、音频等各类媒体数据，数据的产生、存储、分析、呈现都离不开计算机及相关硬件设备的支持。
由于这种数据的多样性、快速增长、分布广泛，使得传统信息技术与计算技术无法应对其需求，而人们便萌生了技术创新、数据处理方法的转型之愿望。
这种计算机科学研究领域之所以能够蓬勃兴起、蓬勃发展，关键在于算法与技术的进步。
而对于计算科学的分类，按宗教和社会科学的看法，则可分为基础科学、应用科学、工程科学三个层次，并逐步细化到自然科学。

基础科学层包括计算机理论、电子工程、通信工程、数学、物理学、化学、生物学等方面；
应用科学层包括图像处理、语音识别、机器学习、推荐系统、搜索引擎、视频监控、金融分析等方面；
工程科学层包括软件开发、算法设计、数据库管理、系统结构与控制、系统集成、网络安全、高性能计算等方面。

当代计算科学的核心是算法与技术，它决定了信息技术的进步方向。
根据计算机科学的历史发展过程，可以把计算科学划分成三个阶段：
（1）早期科技进步期，主要侧重于研究信息处理机的制造、加工、储存和传输，如蒸汽机、电报机、印刷机等；
（2）中期科技成熟期，主要指数学、物理学、生物学和工程学领域的研究成为主流，如图灵机、冯诺依曼结构、莱昂哈德结构、哈佛大学等；
（3）后期科技革命期，主要是由计算机算法及程序变得主导地位，如阿尔法狗、人工神经网络、深度学习、自然语言处理、统计学习等。

而对于计算科学的不同研究领域来说，如自然语言处理、计算机视觉、图像处理、模式识别、语音识别等等，又可以分为以下三种类型：
（1）算法研究，通过抽象化及形式化对计算机执行的基本运算规则进行研究；
（2）数据驱动，基于一些具有代表性的数据集合，利用统计分析方法或数据挖掘技术，构建数学模型，得到一个算法或模型参数，用于预测或解决某一类问题；
（3）理论研究，建立起对问题的一般性理解，即如何进行有效的数据收集、信息编码、学习、分析和决策。


## 计算机历史演变
### 第一代计算机
1945年，美国贝尔实验室的皮姆-科特勒和斯图尔特-卡内尔合著的“计算机”一书，启发了全世界范围内的科学家共同研究计算机的发明创造。这本书首次提出了计算机编程的概念，也正式将“计算机科学”作为整个科学界的重要分支，被认为是计算机领域的百科全书。

1947年，美国国防部研制出第一台电子计算机ENIAC。这是世界上第一个完整的数字计算机，它的指令长度只有56位，芯片尺寸仅仅只有21×27毫米，但是却可以执行乘法、除法、加减、跳转、输入输出等复杂操作。

1951年，冯诺依曼教授提出了一个著名的麻烦：如果让计算机真正理解人类的语言，那么将会出现“语法丧失”的问题。因为电脑只能识别二进制信号，因此需要先将文本翻译成计算机所能识别的数字信号，也就是人类用语中的单词、短语、语句等要素都需要转换成机器指令才能被计算机处理。

1952年，图灵机问世，它为计算机界带来了革命性的变化。它具有即时执行、非凡的处理能力，能够进行任意的计算。它还具有完备的存储器，可以无限存储数字、符号、字符串等数据，同时可以连接多个外部设备，实现交互式计算。

### 第二代计算机
1955年，日本，贝尔实验室的谢尔曼·布兰德尔一起创建了著名的UNIX操作系统。它是一个支持多用户的多任务操作系统，最初只支持诊断、远程登录、命令输入、打印输出等功能。但很快，UNIX便迅速成为计算机界最重要的系统之一。

1956年，麻省理工学院的艾伦·图灵和戴明·摩尔一起提出了著名的图灵测试，目的是证明“一切都能用图灵机解决”，但该测试因其严苛的要求而受到挫折。

1961年，摩尔推出了电子计算机ENIAC，它是世界上第一个真正意义上的商用计算机，售价高达1,000美元。

1966年，英国皇家工程学院的约翰·凯奇、弗朗西斯·福特、沃尔特·皮凯克、比尔·吉布森、威廉·麦席尔、威廉·李嘉图等一批科学家一起设计了著名的EDSAC计算机，它是世界上第一个真正意义上的多道程序系统计算机。它拥有二十五六个芯片，每块芯片都具有56位的处理能力。

至此，计算机领域正式进入第三代时代。在这一阶段里，计算机已经完全脱离了科学研究的范畴，变成了高度集成的通用工具，供广大的工业、商业及政府部门使用。


### 中国计算机历史
中国在计算机领域也获得了崛起的机遇。中国自八十年代末起推动计算机产业发展，取得了举足轻重的作用。

#### IBM System/360
IBM公司在1962年推出了第一款商用计算机System/360。它主要用于超级计算机的研制、生产和销售。

#### Apple Macintosh
1984年，苹果公司推出第一台个人计算机Apple Macintosh，可惜后来它因故宣告破产。

1985年，联想公司推出了300 series、1200 series、400 series、500 series等系列PC产品，这些产品都是为笔记本电脑市场量身定做。这些产品拥有多核CPU、大容量内存、USB接口、可安装的外设、采用TFT显示屏、集成音响系统。

1987年，华硕推出了第一台商用的服务器计算机——ASUS P5K Deluxe Edition，价格不菲，不过好在配置很高。

#### Windows NT/9x
微软公司推出Windows NT和Windows 9x。前者是一款多任务操作系统，后者是一款商用计算机操作系统。两者经历了不同的发展历程。NT直到Windows 2000才引入图形界面，而Windows 9x则直到Windows Millennium Edition才彻底退出历史舞台。
