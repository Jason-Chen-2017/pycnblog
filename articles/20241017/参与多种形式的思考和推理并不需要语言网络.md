   非语言思考和推理可以分为以下几种类型：

   - **听觉思考**：通过听觉信息进行思考，如声音、音乐、语言等。
   - **触觉思考**：通过触觉信息进行思考，如触感、温度、压力等。
   - **运动思考**：通过运动信息进行思考，如动作、姿态、运动轨迹等。

3. **非语言思考和推理的特点**

   - **直观性**：非语言思考和推理能够直接反映事物的本质和规律，具有较高的直观性。

def collect_info(preprocessed_data):

    }
    
    return collected_info
