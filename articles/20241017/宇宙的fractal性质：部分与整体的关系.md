#### 2.1 fractal的定义与特性

fractal，即分形，是一种具有无限层次结构的几何形状。分形的基本特性包括自相似性、无限层次结构和复杂性与简单性。

1. **自相似性**：分形具有局部与整体之间的相似性，即无论观察的尺度大小，分形的基本结构保持不变。例如，一个树的分枝和整个树的结构在形态上具有相似性。

2. **无限层次结构**：分形具有无限层次结构，意味着在分形的每一个局部中，都可以找到与整体相似的结构。分形可以不断地被分割和放大，形成无限的层次。

3. **复杂性与简单性**：分形既具有复杂性，又具有简单性。虽然分形的结构看起来非常复杂，但它们的生成规则却非常简单。例如，分形集可以通过简单的迭代过程生成。

#### 2.2 fractal的数学描述

fractal的数学描述主要包括fractal维数、分形集和迭代函数系统（IFS）。

1. **fractal维数**：fractal维数是描述分形几何特性的重要指标。它介于传统维数（如一维、二维、三维）之间，反映了分形的复杂程度。常见的fractal维数包括Hausdorff维数、容量维数和信息维数等。

2. **分形集**：分形集是一种特殊的集合，其内部结构具有fractal性质。常见的分形集包括科赫曲线、康托尔集、分形树等。

3. **迭代函数系统（IFS）**：迭代函数系统是一种生成分形的数学工具。通过定义一组迭代函数，这些函数反复迭代，生成具有fractal性质的图形。IFS广泛应用于分形图形的生成，如分形树、分形雪花等。

#### A.1 midpoint displacement算法

midpoint displacement算法是一种用于生成分形图形的经典算法，常用于生成分形树和分形山脉等。该算法的基本思想是：从初始图形开始，每次将每个顶点的位置按照一定的规则进行偏移，直到满足终止条件。

1. **算法步骤**：

   - 初始化：选择一个初始图形，如一个正方形。
   - 偏移：对于每个顶点，将其位置向相邻顶点的中点进行偏移。偏移量可以通过以下公式计算：
     \[
     (x_i', y_i') = \left(\frac{x_j + x_k}{2}, \frac{y_j + y_k}{2}\right)
     \]
     其中，\( (x_i, y_i) \) 是顶点的原始位置，\( (x_j, y_j) \) 和 \( (x_k, y_k) \) 是相邻顶点的位置。
   - 递归：对每个顶点的新位置进行递归，继续进行偏移操作，直到满足终止条件（如达到预设的层数或偏移量小于某一阈值）。

2. **伪代码**：

   ```python
   def midpoint_displacement(graph, depth, threshold):
       if depth == 0 or is_small_enough(graph, threshold):
           return graph
       else:
           for vertex in graph.vertices():
               new_vertex = midpoint(vertex, graph.get_adjacent_vertices(vertex))
               graph.move_vertex(vertex, new_vertex)
           return midpoint_displacement(graph, depth - 1, threshold)
   ```

#### A.2 迭代函数系统（IFS）算法

迭代函数系统（Iterated Function System，IFS）算法是一种强大的分形生成算法，通过迭代一组变换函数生成分形图形。IFS算法可以生成各种复杂形状的分形，如分形雪花、分形曲线和分形表面等。

1. **算法步骤**：

   - 初始化：选择一个初始点或初始图形。
   - 迭代：将初始点或初始图形映射到一组变换函数中的每个函数，得到新的点或新的图形。迭代过程持续进行，直到满足终止条件（如达到预设的迭代次数或分形图形的尺寸达到某一阈值）。
   - 生成分形：将迭代过程中生成的所有点或图形合并，形成最终的分形图形。

2. **数学描述**：

   假设IFS由一组变换函数 \( f_1, f_2, \ldots, f_n \) 组成，每个变换函数将一个点映射到另一个点。对于初始点 \( P_0 \)，经过一次迭代后得到的新点 \( P_1 \) 可以表示为：
   \[
   P_1 = f_i(P_0)
   \]
   其中，\( i \) 是随机选择的。

3. **伪代码**：

   ```python
   def IFS(point, transformations, iterations):
       current_point = point
       for _ in range(iterations):
           transformation = random_choice(transformations)
           current_point = transformation(current_point)
       return current_point
   ```
