## 2. 核心概念与联系

### 2.1 环代数

环代数是抽象代数的一个分支，它研究带有乘法和加法的抽象结构。环代数的核心概念包括：

- 环：一个集合R，以及两个二元运算+和*，满足以下性质：
  1. $(R, +)$ 是一个交换群。
  2. $(R, *)$ 是一个结合律。
  3. 存在一个元素0，使得对于任意的元素a属于R，有 $a + 0 = a$。
  4. 对于任意的元素a和b属于R，存在一个元素$-a$，使得 $a + (-a) = 0$。
  5. 存在一个元素1，使得对于任意的元素a属于R，有 $1 \cdot a = a$。
- 环同态：一个函数f：R→S，满足以下性质：
  1. $f(a + b) = f(a) + f(b)$。
  2. $f(a \cdot b) = f(a) \cdot f(b)$。
- 环同构：两个环R和S，如果存在一个环同态f：R→S，且f是双射，则称R和S是环同构的。

### 2.2 张量代数

张量代数是环代数的一个推广，它研究的是多线性映射。张量代数的核心概念包括：

- 张量：一个n维数组，其中n是自然数。
- 张量积：两个张量的张量积是一个新的张量，其元素是原两个张量对应元素的乘积。
- 张量积的运算规则：张量积满足结合律、分配律等运算规则。

### 2.3 路代数

路代数是环代数的一种推广，它研究的是路上的代数结构。路代数的核心概念包括：

- 路上的环：一个图G，以及两个运算+和*，满足以下性质：
  1. $(G, +)$ 是一个交换群。
  2. $(G, *)$ 是一个结合律。
  3. 存在一个元素0，使得对于任意的元素a属于G，有 $a + 0 = a$。
  4. 对于任意的元素a和b属于G，存在一个元素$-a$，使得 $a + (-a) = 0$。
  5. 存在一个元素1，使得对于任意的元素a属于G，有 $1 \cdot a = a$。
- 路同态：一个函数f：G→H，满足以下性质：
  1. $f(a + b) = f(a) + f(b)$。
  2. $f(a \cdot b) = f(a) \cdot f(b)$。
- 路同构：两个路上的环G和H，如果存在一个路同态f：G→H，且f是双射，则称G和H是路同构的。
