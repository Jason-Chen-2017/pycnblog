2. 技术原理及概念

2.1. 基本概念解释

API 网关是一个中央化的服务,用于处理和路由来自客户端的请求。它允许开发人员将不同的服务组合在一起,并提供一个统一的方式来访问它们。API 网关可以提供多种功能,如安全身份验证、请求转发、负载均衡和访问控制等。

无服务器架构是一种应用程序构建方法,它使用云服务来提供计算能力。在这种架构中,开发人员不需要管理服务器或存储设备,而是使用云服务提供商的云服务来构建应用程序。这种架构可以提供更高的可扩展性和灵活性,同时还可以节省开发人员的成本。
