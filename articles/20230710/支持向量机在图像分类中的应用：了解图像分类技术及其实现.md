### 2.1. 基本概念解释

支持向量机是一种二分类的监督学习算法，其基本思想是通过将数据映射到高维空间来寻找最优的超平面，从而实现对数据的分类。在图像分类中，超平面可以看作是图像特征空间的投影，其任务是最小化同一类样本与不同类样本之间的距离。

```python
import numpy as np
import matplotlib.pyplot as plt
from skimage import io
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# 加载数据
iris = load_iris()

# 数据预处理
scaler = StandardScaler()
iris = scaler.fit_transform(iris)

# 数据划分
iris_train, iris_test = train_test_split(iris, test_size=0.2, n_informative_features=2)

# 建立支持向量机模型
clf = SVC(kernel='rbf', C=1)
clf.fit(iris_train, iris_train.target)

# 进行预测
iris_pred = clf.predict(iris_test)

# 计算准确率
accuracy = accuracy_score(iris_test.target, iris_pred)

print('支持向量机在图像分类中的准确率：', accuracy)
```

### Q:

在支持向量机训练过程中，为什么会出现负的权重值？

A:

在支持向量机训练过程中，可能会出现负的权重值。这是因为支持向量机在训练过程中，会通过计算样本到超平面的距离，来更新模型的参数。如果某个样本到超平面的距离为负数，则说明该样本与当前超平面相反，即该样本应该属于错误的类别。因此，在支持向量机训练过程中，负的权重值表示当前超平面对该类别的样本的拟合效果较差，需要对模型的参数进行调整。
