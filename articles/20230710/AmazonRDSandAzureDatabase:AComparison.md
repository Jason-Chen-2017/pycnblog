2. 技术原理及概念
---------------------

### 2.1. 基本概念解释

2.1.1. 关系型数据库

关系型数据库 (RDBMS) 是一种数据存储和管理的传统方式,以表和行的方式存储数据,支持 SQL 查询语言。是当前最流行的一种数据库类型。

### 2.2. 分布式数据库

分布式数据库是指将数据分布在多个物理或逻辑服务器上的数据库,可以提高数据库的可靠性、可扩展性和性能。常见的分布式数据库有 Hadoop、Zookeeper 等。

### 2.3. 数据库引擎

数据库引擎是数据库管理和操作的软件,负责控制数据库的物理结构和逻辑结构,并提供 SQL 查询语言。常见的数据库引擎有 MySQL、Oracle、Microsoft SQL Server、PostgreSQL、Java Database Connectivity (JDBC) 等。

### 2.4. 数据库管理系统

数据库管理系统 (DBMS) 是用于管理数据库的软件,负责控制数据库的物理结构和逻辑结构,并提供数据库引擎。常见的数据库管理系统有 MySQL Workbench、Microsoft SQL Server Management Studio、PostgreSQL Management Console、JDBC 等。
