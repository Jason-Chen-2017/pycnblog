## 4.2. 应用实例分析

假设某医院计划对患者进行定期体检，通过收集患者的病史、生活方式等数据，运用机器学习算法对患者的健康状况进行预测，并生成健康风险评估报告。

## 4.3. 核心代码实现

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.tree import DecisionTreeClassifier
from sklearn.linear_model import LinearRegression

# 读取数据
df = pd.read_csv('patient_data.csv')

# 提取特征
features = ['age', 'gender', 'height', 'weight', 'family_history','lifestyle']

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(df[features], df['target'], test_size=0.2)

# 特征标准化
scaler = StandardScaler()
X_train = scaler.fit_transform(X_train)
X_test = scaler.transform(X_test)

# 模型选择
model = LinearRegression()

# 训练模型
model.fit(X_train[features], y_train)

# 预测
y_pred = model.predict(X_test[features])

# 评估模型
rmse = np.sqrt(np.mean(np.power((y_test - y_pred), 2))
print("Root Mean Squared Error (RMSE):", rmse)

# 应用实例

# 假设某医院计划对患者进行定期体检

# 收集患者数据
patient_data = pd.DataFrame({
    'age': [30, 40, 50, 60],
    'gender': [1, 1, 0, 0],
    'height': [170, 180, 190, 200],
    'weight': [60, 70, 80, 90],
    'family_history': [1, 2, 1, 2],
    'lifestyle': [1, 1, 0, 0],
    'target': [0, 1, 0, 1]
})

# 预测
y_pred = model.predict(patient_data[features])

# 输出结果
print("预测结果：")
print(patient_data['target'])
```
