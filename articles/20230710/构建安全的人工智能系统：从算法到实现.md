### 2.4. 代码实例和解释说明

在这里，我们提供一个使用Python语言实现神经网络模型的例子。我们使用Keras框架构建了一个简单的神经网络模型，并使用数据集对其进行训练和测试。

```python
# 导入需要的库
import numpy as np
import keras
from keras.datasets import load_iris
from keras.models import Sequential
from keras.layers import Dense, Dropout

# 加载iris数据集
iris = load_iris()

# 将数据集划分为训练集和测试集
train_data, test_data = iris.train_test_split(iris.data, iris.target)

# 创建神经网络模型
model = Sequential()
model.add(Dense(5, input_shape=(iris.data.shape[1],), activation='relu'))
model.add(Dropout(0.2))
model.add(Dense(3, activation='softmax'))

# 编译模型
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# 训练模型
model.fit(train_data, epochs=5)

# 对测试集进行预测
predictions = model.predict(test_data)
```
