# 2. 技术原理及概念

## 2.1. 基本概念解释

无监督学习是一种不需要人工标注的数据学习方法，它通过自然数据内部的信息来提取特征并建立模型。无监督学习算法可以分为两大类：基于距离的算法和基于密度的算法。

## 2.2. 技术原理介绍: 算法原理，具体操作步骤，数学公式，代码实例和解释说明

### 2.2.1 基于距离的算法

- 相似度（Distance）是一种常用的度量两个向量之间距离的方法，如欧几里得距离（Euclidean Distance）。
- 基于距离的算法通过计算样本之间的距离来发现相似的特征，然后将相似的样本进行聚类。
- 常见的基于距离的算法包括K-Means、DBSCAN、高斯混合模型（Gaussian Mixture Model，GMM）等。

```python
from sklearn.cluster import KMeans
import numpy as np

# 生成随机数据
X = np.random.rand(100, 10)

# 计算距离
distances = np.sqrt(np.sum((X[:, None, :] - X[None, :, :]) ** 2, axis=2))

# 聚类
kmeans = KMeans(n_clusters=3)
kmeans.fit(distances)
```

### 2.2.2 基于密度的算法

- 密度（Density）是一种度量连续数据集中某种属性值的测量值，如每只羊的体重。
- 基于密度的算法通过计算样本所属聚类的概率来发现相似的特征，然后将相似的样本进行合并。
- 常见的基于密度的算法包括潜在狄利克雷分配（LDA）、高斯混合模型（GMM）等。

```python
from sklearn.mixture import GaussianMixture
import numpy as np

# 生成随机数据
X = np.random.rand(100, 10)

# 计算概率
probabilities = X / np.sum(X)

# 合并
gmm = GaussianMixture(n_components=1)
gmm.fit(probabilities)
```

## 2.3. 相关技术比较

| 技术 | 基于距离的算法 | 基于密度的算法 |
| --- | --- | --- |
| 算法原理 | 计算样本之间的欧几里得距离或曼哈顿距离 | 计算样本所属聚类的概率 |
| 操作步骤 | 1. 计算样本之间的距离 <br> 2. 计算每个样本的聚类概率 <br> 3. 根据概率进行特征选择 | 1. 计算样本所属的聚类 <br> 2. 根据概率对相似样本进行合并 |
| 数学公式 | 欧几里得距离：D(X) = (1/2) * Σ[(xi - xj) ^ 2] <br> 曼哈顿距离：d(X) = |xi - xj| <br> 聚类概率：P(xi) = (1/N) * ∑[p(xis) * exp(-gamma * d(Xis))] |
| 代码实例 |
