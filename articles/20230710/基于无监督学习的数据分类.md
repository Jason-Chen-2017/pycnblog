2.2.4. 代码实例和解释说明

代码实例如下：

```python
import numpy as np
import pandas as pd
from sklearn.cluster import KMeans
from sklearn.decomposition import DBSCAN

# 数据预处理
data = pd.read_csv('data.csv')
data = data.dropna()
data = (data - 0.1) / 2

# 特征提取
features = []
for col in data.columns:
    features.append(col)

# 数据划分
X = data.drop('cluster', axis=1)
y = data['cluster']

# 模型训练
kmeans = KMeans(n_clusters_per_class=1)
kmeans.fit(X)

# 模型测试
accuracy = kmeans.score(X, y)

# 模型优化
dbscan = DBSCAN(eps=2)
clusters = dbscan.fit_predict(X)
```

该代码使用 scikit-learn 库中的 KMeans 和 DBSCAN 算法来实现基于无监督学习的数据分类。其中，KMeans 算法用于聚类，DBSCAN 算法用于降维。在代码中，首先对数据进行预处理，然后对数据进行特征提取，接着将数据分为训练集和测试集，并使用 KMeans 算法对数据进行聚类，最后使用 DBSCAN 算法对聚类后的数据进行降维。同时，代码中还使用了准确率作为模型优化的指标。
