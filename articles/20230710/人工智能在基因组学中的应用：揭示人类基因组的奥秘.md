## 4.3. 核心代码实现

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# DNA sequence comparison
def dna_sequence_comparison(sample1, sample2):
    distance = np.linalg.norm(sample1 - sample2)
    similarity = distance / (sample1.shape[0] + sample2.shape[0])
    return similarity, distance

# Predicting the function of a new sequence
def predict_function(sequence):
    function = ''
    for base in sequence:
        if base == 'A':
            function += 'DNA'
        elif base == 'C':
            function += 'RNA'
        elif base == 'G':
            function += 'DNA'
        else:
            function += base
    return function

# Show the function prediction
new_sequence = 'CGAGCGATCG'
predicted_function = predict_function(new_sequence)
print('The predicted function of', new_sequence, 'is', predicted_function)

# Example usage
sample1 = np.array([1, 2, 3, 4, 5])
sample2 = np.array([6, 7, 8, 9, 10])
similarity, distance = dna_sequence_comparison(sample1, sample2)
predicted_function = predict_function(new_sequence)
print('The similarity is', similarity)
print('The distance is', distance)
print('The predicted function is', predicted_function)
```

- dna_sequence_comparison 函数接受两个 DNA 序列样本，并计算它们之间的距离和相似度，然后返回两个值：相似度和距离。
- predict_function 函数接受一个 DNA 序列样本，并预测其功能。我们使用一个循环来遍历每个碱基，如果碱基是 A，则将其添加到功能字符串中，如果碱基是 C 或 G，则将其添加到功能字符串中。最后，我们返回功能字符串。
- show_function 函数用于展示给定的 DNA 序列的预测功能。这里我们使用的新序列是CGAGCGATCG，我们调用 predict_function函数预测其功能，然后将其打印出来。
