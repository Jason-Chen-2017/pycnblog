## 2.2. 技术原理介绍：算法原理，具体操作步骤，数学公式，代码实例和解释说明

### 2.2.1. 什么是BSD许可协议？

BSD（Berkeley Software Distribution，Berkeley软件分发）许可协议是一种开源的、兼容BSD（Berkeley自由软件授权）协议的授权方式。允许用户自由地修改和重新分发所获得的源代码，包括在源代码中添加新的代码和修改现有的代码。

### 2.2.2. BSD许可协议的授权方式

BSD许可协议有3种授权方式：

1. Attribution-ShareAlike-NoDerivs：允许用户自由地修改和重新分发所获得的源代码，但不允许用户在修改后的代码中添加新的代码。
2. Attribution-ShareAlike：允许用户自由地修改和重新分发所获得的源代码，包括在修改后的代码中添加新的代码。
3. Attribution-NoDerivs：允许用户自由地修改和重新分发所获得的源代码，但不允许用户在修改后的代码中添加新的代码。

### 2.2.3. BSD许可协议的数学公式

假设用户从源代码中获取了k个函数，那么用户需要为这些函数分别提供k个附加声明。附加声明必须以源代码中的原有声明为基础，且允许用户添加、修改或删除声明。

然后，定义一个函数`calculate_average(int *positions, int n)`，用于计算移动平均值。

```arduino
        sum += positions[i];
        average += sum / (2 * i + 1);
    }
    return average;
}
```
接着，定义一个函数`get_location(int i)`，用于获取指定位置的经纬度位置。
```arduino

```

