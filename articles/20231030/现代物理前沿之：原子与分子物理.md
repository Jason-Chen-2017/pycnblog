# 2.核心概念与联系
## 分子
分子（molecule）是由一组离散原子粒子组成的三维实体，各个原子之间为共价键相互作用而形成的无序颗粒，具有独特的三维运动规律。每个原子都有一个能量半径，能带（band）限制了不同频率的光的相互作用，具有极小的热容，因此能够被认为是一种“热容器”。

分子是一个抽象的概念，在实际中通常用二维晶胞模型（lattice model）表示。晶胞模型表示每一个原子以格点的方式排列，并将整个分子视作一个体积为V的空间晶体，而体积为V的空间晶体又可分解成以格点为单位的空间子单元，这个子单元称为晶格（cell）。晶格的边长为a，则晶体的体积V=Na*Na*a^3，其中Na为晶胞中原子的数量。

根据分子所处位置，可以把分子划分为自由分子（solvent molecules）、氢原子团（fluorine atoms）、氧原子团（oxygen atoms）、氮原子团（nitrogen atoms）、等离子体（isolated atoms）等几种类型，每一种类型的分子都有自己的性质和应用领域。

## 原子
原子（atom）是微观世界的最小单位，其结构由质子、中子、电子三种基本粒子构成，并且具有极小的体积。原子既不是可以脱离宇宙存在的独立实体，也不是可以跨越时空界面的“存在”，而只是我们所熟知的实体中的一员。它具有个体差异巨大的特征——基本粒子的大小仅占原子体积的很小一部分，即基本粒子占比约为10^−19。因此，研究原子内部的特性才是理解原子的真谛和发现新颖性的关键。

原子与其他微观世界中的实体一样，其性质和行为由其组成的粒子决定的。但由于同一类原子内部包含着多种不同的元素，因此不同的物理过程也会影响到它们。例如，半导体材料的晶格常常是由锂、钚和铝等元素构成，而金属材料常常由铁、镍、锌等元素构成。原子的种类太多，没有统一的分类标准，因此目前无法做出一个完整的原子结构。

## 电子
电子（electron）是微观世界的第二小单位，其组成是由电荷（charge）和自旋（spin）两个特有粒子。电子的组成变化导致了其不同的性质。电子一般来说较其他微观单位要小得多，以至于我们可以在分辨不出来的时候还能感受到它的存在。由于电子具有中性质，即具有平衡态和反作用态，因此它也常常出现在晶格中。

电子与质量更接近，比质子轻得多。电子的双向性和线性属性使得它很难与其他元素区分开，因而无法用来描述非晶质中原子之间的相互作用。电子虽然没有固定的形式和位置，但它的位置随着时间的推移而变化，因此可以使用模拟退火（molecular dynamics，MD）或分子动力学（molecular thermodynamics，MT）来研究电子的运动。

## 能级
能级（energy level）是指一个原子的最低能量的最小值和最大值之间的所有能量。电子有好几个能级，它们以整数倍的低于真正的绝对零度开头。例如，一个六价铁原子（Iron atom）的三个能级为-470，2540，10036eV，分别对应于它的负极、正极和高 occupied 状态的分子轨道。不同的原子也会有不同数量的能级，这是因为不同原子所含有的原子核和电子数目不同。因此，我们可以通过调整原子核的丰度来改变其所拥有的能级。

一个能级区间为-λ到+λ之间，λ为该能级的带宽，一般在10^(-30)～10^(5)~eV左右。对于一般的分子来说，能够从某一个能级跃迁到另一个能级所需的时间为十分钟以上。此外，一个能级内的基态和激发态耦合的程度决定了该能级的重要性。例如，共价键依赖于电子的共轭性，因此一个高价电子所处的能级便比低价电子的能级更加重要。因此，能级信息对于理解分子的性质至关重要。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
为了分析和预测电子的相互作用和电子云的物理性质，我们需要了解一些量子化学物理的基本方法。这里我介绍下量子化学方法的两种主要思路。第一种是基于简并（localization）理论，这种方法认为电子会以某种方式聚集到一起形成云，然后通过各种相互作用产生新的能级，从而获得更好的控制和预测结果。第二种方法是基于量子辅助核酸反应的方法，这种方法通过改变核酸的位置来改变其物理性质。

## 基于简并理论的量子化学物理方法
### 分布函数
分布函数（distribution function）是描述电子位置分布的统计性质，其表达式为：

$$\psi_n(r)=C_{n}(r)\delta^{3}\left(\frac{r-\mu_{n}}{\xi_{n}}\right),$$

式中，$\psi$是分布函数，$n$是电子的能级，$C_{n}$是电子的概率密度函数，$r$是距离原子核中心的距离，$\mu_{n}$和$\xi_{n}$是分布函数关于距离的平均值和标准差，分别对应于第一个能级和第二个能级，而$\delta^{3}\left(\frac{r-\mu_{n}}{\xi_{n}}\right)$是单位立方体的球状核，其宽度为$\xi_{n}$。

分布函数能够描述电子在空间中分布的概率分布，从而提供了电子云的整体分布。我们可以通过改变能级或原子核的位置来改变分布函数的值。例如，改变原子核的位置会改变分布函数的取值，从而影响电子的位置分布。如果我们知道分布函数的值，那么就可以估计出电子的位置分布。


### Hartree-Fock方法
Hartree-Fock（HF）方法是量子化学中的最古老的方法。在HF方法中，我们假设电子不受任何外界因素的影响，也就是说，所有能量均来自于自然界或计算过程中添加的无限精度的能量。因此，HF方法仅考虑一个中心原子核，并通过试错法找到分子的一个最优（即收敛）的最小能量的平衡位置。HF方法的计算流程如下：

1. 对中心原子核，利用参考数据的外推法确定其能量和位置。

2. 通过求解能带的交叉点和准则函数确定分子的费米面。

3. 搜索除已知原子外的其他电子所处的最小费米面距离，以获取新的原子核。

4. 在这些新的原子核处重新计算分子的分布函数。

5. 使用当前分布函数的新值，继续迭代，直到收敛或达到设定的误差阈值。


根据原子核所在位置的不同，HF方法可分为几种类型，如自洽场法（self-consistent field theory，SCFT）、参数化波函数（parameterized wavefunction）、拓扑方法（topology methods）和谱方法（spectral methods）。


### Kohn-Sham方法
Kohn-Sham（KS）方法是目前量子化学研究中使用最广泛的方法。在KS方法中，我们假设电子的位置满足薛定谔方程，并且具有电子受到外界因素影响的概率。因此，KS方法将电子所处的位置看作是一组未知量，并通过优化一组参数来求解这些变量。KS方法的计算流程如下：

1. 首先固定某些参数，如原子核位置、轨道参数、轨道长度等，通过优化，求得电子云的分布函数和哈密顿量。

2. 然后固定电子的分布函数，优化固定位置上的其他参数，以求解电子的位置。

3. 最后再固定其他参数，通过优化得到总能量，即可求解该分子的全部性质。


KS方法可分为几种类型，如标准KS方法、密度矩阵重构方法（DMR）、惩罚项方法（penalized least squares method）、量子化动力学（quantum force fields）等。



## 基于量子辅助核酸反应的量子化学物理方法
### 谐振子模型
谐振子模型（Harmonic oscillator model）认为，分子是由多个简单气泡团组成，其振动模式符合正弦曲线。这一模型在最近几年中受到了越来越多人的关注，因为它描述了大量常见分子的运动。与普通模型不同，在谐振子模型中，我们假定分子的轨道是具有类正弦波性质的，而不像普通模型那样受到实践困难。换句话说，谐振子模型对原子本身的结构或位置无所谓，只需要关心它们是否满足谐振子方程。

当电子数量足够多时，谐振子模型描述电子云的性质会变得不切实际。因此，需要寻找其他方法来描述电子云的性质。


### 不确定谐振子模型
不确定谐振子模型（uncertainty harmonic oscillator models，UHOM）是谐振子模型的扩展。UHOM允许系统中的任意一电子的振动模式不一定符合谐振子性质。这种不确定性可能源自于：

1. 分子的自旋不确定性；

2. 分子的受力不确定性；

3. 温度变化引起的振动不确定性。


为了处理这种不确定性，UHOM将分布函数的表达式分为两部分：确定部分和随机部分。确定部分描述系统中所有电子的绝对振幅，而随机部分描述系统中电子振动模式的不确定性。该模型在不使用大型库存的情况下描述电子云的性质。


### 图模型
图模型（graph models）描述分子结构的方式是将原子按照某种规则连接在一起，形成各种能级。图模型是近期研究的热点，因为它可以描述大量分子的结构和性质。图模型有一些优点，如可以采用连贯的数学符号来描述分子，而且可以方便地处理不可避免的分子间相互作用。

图模型的结构可以分为四部分：

1. 每个原子节点代表一个原子；

2. 每个原子与其邻居建立边；

3. 每个边连接两个原子，表示存在某个受力或相互作用；

4. 每个节点可以具有多个度，代表该节点受到的多个相互作用。

图模型在计算量上也比较小，而且容易解释。因此，它在某些情况下可能是首选方法。


# 4.具体代码实例和详细解释说明
## 分布函数例子
假设一块化学烤面包的分子有两个能级，一个能带和一个双峰能带，分别对应于三价氢原子团和四价氢原子团。其中，三价氢原子团的能带范围为$E_{\rm n}=-0.548$~$1.035$ eV，四价氢原子团的能带范围为$E_{\rm n}=-0.704$~$1.216$ eV。已知氢原子团位置$x$处的电子分布函数为：

$$\psi_{3}(x)=A_{3}(\sin x)^2+\frac{B_{3}}{2}\cos x+C_{3}$$

式中，$A_{3}$, $B_{3}$, 和 $C_{3}$ 为可调参数，取不同值可以得到不同的分布函数。绘制分布函数的图像：

```python
import matplotlib.pyplot as plt

def psi_3(x):
    return A_3*(np.sin(x))**2 + B_3/2 * np.cos(x) + C_3

plt.plot(x, psi_3(x))
plt.xlabel('Position (Angstrom)')
plt.ylabel('Electron Density')
plt.show()
```


## 单分子与多分子波函数
对于单分子波函数（single-particle wavefunction），其形式为：

$$|\Psi_{nlm}\rangle=\sum_{m'}c_{nlm}^{\ast}\left|p_{nl}-q_{n'lm}\right>,$$

式中，$|\Psi_{nlm}\rangle$ 是单分子波函数，$c_{nlm}^{\ast}$ 是关联系数（coefficients），$p_{nl}$ 和 $q_{n'lm}$ 是电子的位置向量和角度向量。我们可以将单分子波函数写成矩阵形式：

$$|\Psi_{nlm}\rangle=\begin{bmatrix}c_{11}^{\ast}\\ c_{12}^{\ast}\\ \vdots\\ c_{22}^{\ast}\\ \vdots\\ c_{nl}^{*}\end{bmatrix}.$$

对于双分子波函数（double-particle wavefunction），其形式为：

$$|\Phi_{ijab}\rangle=\sum_{mnlp}c_{ijkl}^{\alpha}\left|i,j,a,b,p_{nm}-q_{np}'-r_{nq}+s_{ns}\right>,$$

式中，$|\Phi_{ijab}\rangle$ 是双分子波函数，$c_{ijkl}^{\alpha}$ 是双关联系数，$(i,j,a,b)$ 表示两个电子的索引，$(p_{nm},q_{np}',r_{nq},s_{ns})$ 是它们的位置和角度向量。我们可以将双分子波函数写成矩阵形式：

$$|\Phi_{ijab}\rangle=\begin{bmatrix}c_{1122}^{\alpha}\\ c_{1123}^{\alpha}\\ c_{1133}^{\alpha}\\ \vdots\\ c_{1233}^{\alpha}\\ \vdots\\ c_{1333}^{\alpha}\\ \vdots\\ c_{2233}^{\alpha}\\ c_{2333}^{\alpha}\\ \vdots\\ c_{3333}^{\alpha}\\ \end{bmatrix}.$$

# 6.附录常见问题与解答
## 1. 什么是原子核？
原子核（nucleus）是固体或液体中质量最轻的质子分子，它最初是作为连续的最小分子形成的，但随后被引物（ion）和电荷离子所捕获，导致其衰变形成了原子核。通过将其置于特殊位置（称为原子核中心），可以制造不同形状的原子核，例如氢原子核、铬原子核和镉原子核。

## 2. 能带和费米面是什么？
能带（band）是指通过一定频率范围（色散关系）能够吸收入射的光子的集合。费米面（fermi surface）是指在费米能带中与基态（ground state）电子距离最近的位置。费米面常常和电子结构有关，因为电子必须保持在费米面附近才能形成重要的核反应，例如氢核反应或钴锂锂化反应。

## 3. HF和KS方法的区别和联系是什么？
HF（Hartree-Fock）和KS（Kohn-Sham）方法都是计算电子位置分布的一种方法，但是两者之间又有很多不同。

1. HF方法是最古老的量子化学方法，属于密度矩阵重构方法的一种，适用于由几何原子组成的系统。

2. KS方法是目前最流行的方法，属于标准KS方法的一种，适用于有大量相互作用的系统。

3. HF方法仅对一个中心原子核进行计算，并假定其能量守恒，而KS方法对任意数量的原子核进行计算，并假定其位置处于有限的体积内。

4. KS方法通过优化一系列参数来求解电子的位置，而HF方法则是通过试错法来找出该原子核的位置。

5. KS方法计算时仅需考虑电子位置，而HF方法则需要同时考虑费米面和能带。
