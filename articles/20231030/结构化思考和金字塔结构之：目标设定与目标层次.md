# 1.背景介绍


在科技快速发展的今天，管理者们每天都面临着巨大的决策压力、创新能力的提升和资源的分配。如何做到充分把控业务需求、组织内部团队协作和资源管理是一个非常重要的问题。如何制定一个科学、可行的目标框架，并使得各级管理者都能认同这个目标框架，达成共识，确保计划落实落地，则是管理者们需要考虑的问题。  

“结构化思考”的概念，源于美国哈佛大学教授约翰·迈尔斯（John Miles）博士的著作《软件工程实践》（Software Engineering Practices），该书将工程实践分为“分析、设计、构建、测试和部署”五个阶段，而每个阶段又可以进一步细分为“计划、执行、监控、评审、改善”等子阶段。通过这种“分层”、“分级”的观念，作者提倡以“目标”为导向，将全过程分解为多个目标，每个目标分解为多个子目标，最终实现目标的落实。  

以前的管理方法也围绕着目标进行，如瀑布模型、迭代模型、功能点开发、项目管理等。但随着信息时代的发展，很多公司越来越注重目标驱动型管理，甚至不惜放弃项目管理制度，只要产品、服务、客户满意即可。这种新的管理方法应运而生，它能够帮助公司更加精准的把握业务发展方向，找到最有价值、最有利可贵的资源，为公司创造更好的发展机会。  

对于管理者来说，了解目标驱动型管理的基本原理，掌握目标层次结构的应用，能够帮助自己对业务有更加清晰的把握，为自己制定合理、可行的目标框架打下坚实的基础。  

# 2.核心概念与联系
## 2.1 目标
目标指明了企业要达成什么样的结果。目标通常以客观、可衡量的方式呈现，能够反映企业的长期目标。目标的作用主要有以下几方面：  
- 目标强调了企业的重视程度，是企业衡量其成功的关键指标；
- 目标有助于为人们提供参考信息，使得工作变得透明可观察；
- 目标可以帮助企业确定短期内应该完成的任务、研究课题、商业模式以及战略方向；
- 目标也是一种激励机制，促使员工努力完成预定的工作。
  
## 2.2 目标层次结构
目标层次结构（TLC），是一种关于目标划分和实施的有关原则。它是一种基于目标驱动型管理法则的一种管理体系，即使用TLC进行管理的人员认为，目标是分解和定义的，管理人员必须首先将目标分解成适当的层次结构。每个目标层次都由若干子目标组成，不同层次之间的关系是平行或递进的，不同的层次代表着不同的管理领域。  
  
目标层次结构的目的是为了梳理和完善管理中存在的各项目标，并有效地促进目标之间的沟通、协调和实现。TLC的基本要素包括：  
- TLC的顶层目标，也就是战略目标。战略目标直接影响企业的长期发展方向，是TLC的根本目的。这类目标往往是全局性的、既定的。
- 战略目标以支撑目标为导向，将其细分为多个支撑目标。支撑目标具有重要性，他们是企业追求的核心目的，是其他目标的基础。它们在TLC中的地位和作用和其他目标一样，也是多方面互动的结果。
- 支撑目标又进一步细分为多个实现目标。实现目标是企业真正去实现的目标。只有实现目标才能产生真正的效果。
- 在实现目标层次上，TLC还可以继续细分为不同管理层次上的目标。不同层次上的目标所侧重的范围不同，可以从产品、服务、渠道、流程等角度出发，同时满足不同层面的要求。TLC是建立在“层次与层次之间及不同目标与各目标之间的互相关联”的基础上的。  

## 2.3 优先级排序法
优先级排序法（PSP，Prioritization System Process），用于比较和确定多个待选事项的优先级，它起到了按优先级对事项进行分类、排序和筛选的作用。PSP依据一定的标准，把各种事项排列顺序分成不同的优先级，然后根据优先级的大小、难易程度等决定事项的优先级。  

优先级排序法将各个目标分解为单独的任务、事务，并为每个任务赋予一个优先级。一旦有多个任务同时出现的时候，就会采用优先级排序法进行处理。优先级排序法就是在TLC的基础上进行的一套工作流程。优先级排序法的几个特点如下：  
- 第一，按照优先级先后进行排列，优先级高的排在前面，优先级低的排在后面。
- 第二，优先级由低到高，优先级相同的进行再排序，靠近中心的优先级优先。
- 第三，优先级分为五档，分别对应：紧急、重要、高、中、低。

## 2.4 关键路径法
关键路径法（CPM，Critical Path Method），是一种利用矩阵图表的方法，将有向图或者无环图从源点到汇点的最短路径作为最佳方案来规划。关键路径法在金字塔型管理结构中，称为关键路径。它是基于熵的管理理论基础之上的，即每个节点的熵等于进入该节点的总入度减去离开该节点的总出度。那么，经过某条最短的边，该节点的总出度就一定比进入该节点的总入度小，也就是说，如果所有节点的熵一样的话，那么最短路径就是关键路径。关键路径法是一种动态的管理方法，每时刻都会重新计算最短路径。  
  
关键路径法是在TLC的基础上发展起来的一种方法。在TLC中，TLC顶层目标是关键路径，TLC的各个层次目标都是关键路径的支撑目标，这些目标的组合就是企业的长期战略。通过使用关键路径法，可以找到企业的关键路径。关键路径法的几个特点如下：  
- 第一，关键路径法没有考虑计划外活动，因此在实际运作中可能无法全部纳入考虑。
- 第二，关键路径法无法避免资源浪费，因而缺乏预测性。
- 第三，关键路径法要求目标层次结构必须精确。
- 第四，关键路径法不能适应变化的市场环境。
