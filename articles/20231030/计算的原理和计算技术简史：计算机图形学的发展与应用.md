# 1.背景介绍


计算机图形学（Computer Graphics）是利用计算机模拟现实世界，将三维或二维图像呈现给人眼所需的一门学科。它是视觉效果与计算机硬件、软件及数据之间的交互关系，是计算机视觉、虚拟现实、增强现实以及其他交互艺术的基础。目前，计算机图形学已成为计算机科学研究领域的一个重要分支。
计算机图形学起源于图形用户接口（Graphical User Interface, GUI），它是早期的用户界面的主要形式之一。随着图形界面设备的广泛使用，计算机图形学开始在几十年间持续产生影响，成为当今人类生活中的不可或缺的一项科技。在过去的50~100年间，计算机图形学的发展经历了3个阶段：第一阶段是基于像素点的方法，它用颜色编码等简单手段生成图像；第二阶段是基于矢量方法，它采用数学方程来描述对象的形状和光照属性，从而得到精确的图像；第三阶段是基于混合方法，它结合了前两种方法的优点，能够准确地还原物体的形状和细节，并兼顾流畅度与高保真度。但是，由于计算机性能的提升，第三阶段计算机图形学又发生了重大变化，进入到了以渲染管线为基础的高级技术。
图形学的发展给计算机带来的更广泛影响，其对社会文化产业、经济活动和管理模式都产生了深远的影响。计算机图形学对人们生活的影响也越来越大，如影像制作、动画制作、虚拟现实等。20世纪末到本世纪初，计算机图形学发展的速度很快，已经进入了一个全新的时代。
# 2.核心概念与联系
## 2.1 模型、视图、相机
计算机图形学由三个基本要素组成——模型（Model）、视图（View）、相机（Camera）。模型是一个三维物体的几何图形，它代表了一个事物的空间属性。视图是一个摄像头视角，它决定了观察者看到的物体的位置、方向和大小。相机则是拍摄照片的设备，它捕捉并记录在视野范围内的所有信息。
在绘制3D图像时，通常需要一个三维场景的模型，再通过视图找到观察者的位置和方向，最后通过相机进行拍摄，将场景投射到摄像头上。这些组件之间存在着一定的联系，它们的作用如下：
- 模型：决定了物体的形状、大小和数量，它是观察者看到的物体的外貌、结构、运动规律等。
- 视图：决定了物体在摄像头前的位置、大小、角度等，是观察者所看到的物体的局部。
- 相机：决定了摄像头的位置、姿态、视野范围等，是物体被摄取时的真实情况。
## 2.2 投影、正交投影和透视投影
### 2.2.1 投影
投影是指将三维物体映射到二维平面上的过程。投影有三种类型——正交投影、透视投影和超出图像边缘的投影。
正交投影就是平面与物体的垂直距离不变的投影，因此摆放物体时只能出现一个角度。这种投影方式在二维平面上也称为平行投影或平面投影。
透视投影是用来显示真实三维物体的一种投影方式，它将三维物体映射到二维平面上的过程类似于正交投影，不同的是透视投影是在物体离镜头距离不断增加的时候，依然保持屏幕面积的比例，使得物体看起来像是被“拉近”了。这种投影方式在二维平面上也称为经验投影或perspective projection。
超出图像边缘的投影即将物体投影到图像的边缘。这种类型的投影可用于显示遮挡物体的内部或外部轮廓，但不能显示部分遮挡物体。
### 2.2.2 正交投影与透视投影的区别
正交投影和透视投影都是把三维物体映射到二维平面上的过程，但是两者的差异还是比较明显的。正交投影是以坐标轴作为视线，而透视投影可以将物体投射到更大的空间中去。比如，一辆汽车的正反面都能看到后视镜，但如果透视投影一样的话，只有那里的左侧部分才能看到后视镜。而正交投影就能让所有地方都能看到后视镜。正交投影具有直观性，适用于静态图像，便于理解和分析，但无法正确显示部分遮挡物体。而透视投影更加逼真，适用于动态、运动图像，能够显示部分遮挡物体的内部和外部轮廓。透视投影的另一个优点是，通过改变视野，能够更好地显示物体的特点。所以，对于一些对真实感要求较高的场景，采用透视投影可能会获得更好的视觉效果。
