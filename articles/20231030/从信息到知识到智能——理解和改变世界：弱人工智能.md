# 1.背景介绍


## 一、什么是弱人工智能？
弱人工智能(WIA)，它可以模仿人的一些行为、技能或者说模式，达到某些目的，也可以自动执行许多重复性的任务。它通常是由人类专家开发出来的一套技术系统。但是，由于缺乏实际的人工运用经验，这些系统往往无法很好的学习人类的各种技能、习惯和行为，并且它们还受制于人的思维方式和工作方式。因此，弱人工智能具有独特的“自适应”能力，能够快速的适应环境的变化并利用其中的信息进行自我改进。

在当前的时代背景下，即使是较为简单的一些任务，也需要由专门的人员做出决策。比如，要识别一张图片里是否存在违法行为，就需要警察局出面调查。而要实现一个智能的道路导航系统，则需要专门的机器人驾驶员。弱人工智能的出现，则意味着不需要专门的领域知识或特殊技能，就可以自动化地完成复杂的任务。因此，弱人工智能也被称作智能助手、虚拟服务机器人、危机处理者等。虽然，弱人工智能的发展取得了令人满意的成果，但它的普及和应用仍处于一个十分重要的阶段。

## 二、如何定义弱人工智能？
弱人工智能应该具有以下几个特征：

1. 模拟人类：弱人工智能应该模仿人类的一些行为、技能或者说模式，达到某些目的，而且也能够快速的适应环境的变化。
2. 自主学习能力：弱人工智能应该具备自主学习能力，能够快速掌握新的知识、技能和模式。
3. 生物特征相似：弱人工智能应该具备某种与生俱来的特性，如视觉、听觉、嗅觉、触觉、味觉、精神等，因此，它们可以理解和处理人类更加丰富的感官信息。
4. 自适应能力：弱人工智能应该具有自适应能力，能够快速的适应环境的变化并利用其中的信息进行自我改进。
5. 智能自治：弱人工智能应该具有智能自治能力，能够独立解决复杂的问题而不依赖于外部干涉。
6. 有限资源：弱人工智能应该将有限的资源用于学习和运用，以保证有效的实施。

## 三、弱人工智能的应用场景
### （1）智能助手
作为生活中不可替代的助手，从最基本的日常生活需求到繁琐的工作流程，弱人工智能可以为人们提供很多便利。如：结账助手、保险咨询、闲聊小机器人、智能摄像头、智能语音助手等。

### （2）虚拟服务机器人
在企业服务上下游的上下游合作过程中，弱人工智能可以提升协同效率，减少重复劳动。如：工厂订单跟踪、客户反馈处理、采购预测分析、风险管理等。

### （3）危机处理者
弱人工智能可以帮助灾难应对人员和相关部门更快的发现和响应灾难性事件。如：地震报警、火灾监控、水库淹没、海啸预警等。

### （4）安全设备
安全设备通常都需要专门的维护人员来保障。而弱人工智能可以通过感知周边环境来识别异常行为，并及时上报，自动关闭电源，避免损失。如：消防监控、安防视频监控、卡车安全预警等。

### （5）自动驾驶
拥有无人驾驶能力的人民幸福指数已经上升到全球前列。但是，目前仍然存在着巨大的技术障碍。这其中，就包括如何让一辆汽车自动驾驶、安全驾驶、避障驾驶、无人驾驶。而弱人工智能在自动驾驶方面的研究正在蓬勃发展，主要靠计算机视觉、机器人技术、传感器等新技术。
