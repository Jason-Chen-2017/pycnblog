# 1.背景介绍


多晶硅和单晶硅由于材料结构不同而导致表面电导率(Surface dielectric constant)的差异性很大。多晶硅材料可以提供更高的表面电导率、更高的热导率、更好的电压耐受性、更好的附着力和耐磨能力。然而多晶硅的硬化过程较慢、硬化温度较低，导致重组晶体阶段的膨胀率很高。因此，多晶硅作为一种新型材料正在成为工艺领域中的重要突破口。

对于芯片制造技术来说，多晶硅材料的应用不仅仅局限于底层设计，同时也将会对芯片的制造方式产生影响。因为多晶硅结构特性的特点决定了其在制造电子芯片时的成像性能和性能指标。比如，双极射线二极管(DRT)的导通性取决于晶格的结构，如果晶格结构相邻的两极之间存在间隙，则导通性会下降；反之，如果晶格结构两极之间不存在间隙，则导通性可能会提升到一定程度。这种晶格特性对微电子设备的制造能力和可靠性具有非常大的影响。

另一个方面，对于重组晶体结构来说，多晶硅的纳米级表征能力也是吸引人的亮点。因为除了功能性增强外，多晶硅还可以有效地提升纳米尺寸、密度和厚度等。这些都是为了使得晶体制造工艺的整体效率达到最高水平。

总的来说，多晶硅作为一种新型材料正在成为芯片制造的基础工具，而且正在向更高精度、更复杂的功能方向迈进。
# 2.核心概念与联系
## 2.1 多晶硅材料
### 2.1.1 多晶硅材料简介
多晶硅由多种不同的结构物质构成，这些物质包括半导体材料（如硅、锗、铝）、金属杂质、镁、钛、锑等。通过多种不同的结构，多晶硅可以在不同级别的电流、功率和频率上工作。在制备电子器件中，多晶硅的结构可以形成常规晶体或半晶体，分别用于表面受控微电子制备和中间体积的微电子加工。

目前，主要两种多晶硅材料应用比较广泛：纳米型多晶硅材料和硅基薄膜型多晶硅材料。

### 2.1.2 纳米型多晶硅材料
纳米型多晶硅材料是指由多个不同结构分散的小型低温离心晶体所组成的多晶材料，通常称为纳米级晶体。通常，纳米级晶体由数百个亚稳态离子大小的正粒子构成，经过长时间的自旋操作，即可转换为具有不同结构和形状的多种晶体。不同结构之间的相互作用由核反应堆把相邻的晶体与其周围杂质结合形成新的纳米级晶体。这样就可以形成具有不同晶格结构、分辨率、厚度、电导率、可塑性等的多种多晶纳米型材料。

应用场景举例：
- 晶圆光刻胶（Nanocircular cement fiber, NCF）：这是一种由纳米多晶硫胞垢粉团（Nanoactive Silica, NAS）及纳米异铁薄膜（Indium Oxide Nanorod）组成的三维纳米级晶体。用作微型固体光栅的功能光刻胶材料，具有较高的分辨率、厚度和无缝性。
- 纳米孔单层超导材料（Nanopore superlattice TMDs）：这是一种由纳米多晶硫胞垢粉团（Nanoactive Silica, NAS）及纳米铝黄点（Copper Hydrogen Oxide, CO2N）薄膜（Carbon Nanotrilayer）组成的纳米级晶体。用于开发高能量、高速率、高容量的超导材料，具有良好的兼容性。

### 2.1.3 硅基薄膜型多晶硅材料
硅基薄膜型多晶硅材料是指利用特定结构的硅作为基材，通过多种不同的形状和厚度，堆叠而成的纳米级晶体材料。通常，此类材料由天然或化学ALLY拓扑结构所控制，具有更高的纳米级别的精度。例如，为高能量芯片制造提供良好的聚合物配置。

应用场景举例：
- 高能量元件制造：这类材料应用于像集成电路一样的集成电路元件制造，能够构建出具有极高能量耗尽能力的芯片，具有广泛的应用前景。
- 飞行器、气象站、天文台、医疗设备等环境监测：由于其低熔点和高导电性，硅基薄膜型多晶硅材料适合建造环境监测平台，实现智能化、智慧化的检测。
