# 1.背景介绍


随着云计算、大数据和人工智能技术的不断革命性的发展，计算技术也在不断向新的领域迈进。从古至今，各种不同的计算方法、模型一直都伴随着人类的活动，并给人的生活和社会带来了巨大的便利。如微积分、几何学、代数、分析学、物理学等等。然而，随着计算机的普及，越来越多的人开始把目光投向计算这个庞大的领域，由此产生了一股计算热潮。

云计算、大数据、人工智能以及物联网技术正在改变计算技术的本质，以及如何使得计算能够更好的服务于人类。其中边缘计算（Edge Computing）便是这样的一个技术方向，它将计算技术部署到网络的边缘节点，处理离用户距离最近的数据，减少数据中心的压力。无论是在桌面端还是在移动端，任何需要连接到互联网的设备，都可以运行边缘计算。

2017年5月2日，阿里巴巴集团宣布完成第一轮B轮融资，紧接着由英特尔、高通、苹果、百度、小米、腾讯、亚马逊等公司纷纷布局边缘计算市场。

2019年11月2日，英国皇家工业学会发布2020年度主要目标，其中的一个重要目标就是“打造世界上第一个可穿戴电子产品”。这是因为“一个整体”，“联网”，“高效率”以及“低功耗”四项特征将会成为人们共同关注的词汇。然而，这种希望被云计算的快速发展、数据和人工智能技术、物联网技术所冲击。

虽然边缘计算作为一个颠覆性的技术方向已经逐渐火爆起来，但现实情况却是云计算、大数据、人工智能以及物联网技术不断侵蚀着计算技术的边界。事实上，现有的计算技术不能真正满足需求，需要融合各个技术来提升计算能力。

2.核心概念与联系
## 1) 边缘计算
**定义:**

边缘计算是一种利用网络边缘服务器进行计算的技术手段，是一种计算形式，旨在将计算任务卸载到物理边缘位置，对其上的数据进行处理。

**分类:**

1. 根据功能作用分:

    - 数据中心内部计算
    - 边缘计算（离线处理）
    - 大数据计算
    - 混合计算（超算）

2. 根据计算方法分:

    - Fog computing
    - Edge computing

3. 根据计算资源分布范围分:

    - 私有云
    - 公有云
    - 边缘云

4. 根据系统分层结构分:

    - 数据层(存储、处理、传输)
    - 应用层(业务逻辑处理)
    - 设备层(物理资源)
    - 服务层(服务接口)
    
## 2) 云计算
**定义:**

云计算是一种通过网络访问远程计算机硬件资源的方式。云计算采用虚拟化技术，利用云服务器、云存储、云网络等资源提供商的优势，实现基础设施的按需付费、灵活扩容。通过网络将用户的需求调动到云端，可以大幅度降低成本，提高资源利用率，节省空间、运营成本。

**分类：**

云计算可以分为公有云、私有云、混合云三个层次。公有云服务由云服务提供商直接提供给所有人使用，用户无需购买自己的服务器。私有云服务则是指企业购买自己的数据中心，并在上面建立自己的计算、存储、网络等资源。混合云则是两者相结合，既包含私有云资源也包含公有云资源，可以实现更多的灵活性。

## 3) 服务器
**定义:**

服务器（Server）是一台或多台计算机软硬件、通信设备和网络设备组成的计算机系统。服务器是集群中计算机系统的总称，包含一组物理设备和网络资源，负责提供基础IT服务。

**分类:**

按照硬件类型划分，通常有：

1. 中央处理器(CPU): 中央处理器（Central Processing Unit, CPU），又称之为主机控制器或者中央处理器芯片，是整个计算机的“引擎”，负责指令的执行、数据处理和控制程序的运行。
2. 内存：内存（Memory）是用来存储数据的，所有的信息都要先加载到内存里才能进行处理。RAM（Random Access Memory）就是随机存取存储器，它的速度快，但是比ROM（Read Only Memory）的价格昂贵很多。
3. 固态硬盘：固态硬盘（Solid State Disk，SSD），它的最大优点是闪存化，可以保存永久数据且比传统硬盘便宜得多。硬盘上的所有数据都是以字节流的形式存在，只能顺序读写，不方便查找。
4. 机械硬盘：机械硬盘（Hard disk drive，HDD），其数据保存在磁道上，机械运行，价格昂贵。

按照性能与处理能力的划分：

1. 小型机: 一般是小于1U的玻璃箱或金属机架，主要用于个人电脑和嵌入式设备等。
2. 中型机: 是指具有中等规模的服务器，一般在1U至10U之间，大多数公司的服务器是此种型号。
3. 大型机: 是指具有较大规模的服务器，通常为超过10U的规格，是目前最常用的服务器规格。

## 4) 数据库
**定义:**

数据库（Database）是长期存储、管理和共享数据的一套系统。数据库分为关系型数据库和非关系型数据库。关系型数据库中数据以表的形式组织，每张表有若干字段（field）和记录行（record）。非关系型数据库则没有固定模式，允许不同数据格式的文档存在于同一个集合中。

**分类:**

按照存放数据位置的划分：

1. 客户端/服务器型数据库：基于客户端-服务器的数据库，服务器运行于网络环境内，客户端通过网络连接到服务器获取数据，数据库应用于分布式系统，数据存储在远程服务器。客户端/服务器数据库包括Oracle、DB2、Sybase、MySQL等。
2. 本地数据库：采用服务器安装在用户端，适用于局域网中。
3. 分布式数据库：采用多台服务器相互连接，使得数据库变得可扩展，适用于高负荷的网站。

按照数据结构的划分：

1. 关系数据库：由表格、表单和记录组成。
2. 键值存储数据库：根据键来查找对应的值。
3. 列存储数据库：把数据按列存储，非常适合分析型数据库。
4. 文档存储数据库：以文档的形式存储数据，不适用SQL语言。

## 5) AI
**定义:**

人工智能（Artificial Intelligence，AI）是研究、开发出来的让机器像人一样做某些特定工作的自然、高科技的技术。

**分类:**

按照目的划分：

1. 人工智能与认知：目的是让机器具备人类的理解和认知能力。
2. 人工智能与机器人：目的是让机器具备与人类似的动作和思维方式。
3. 人工智能与决策：目的是让机器能够主动做出判断、选择、协作等，用以解决复杂的业务问题。
