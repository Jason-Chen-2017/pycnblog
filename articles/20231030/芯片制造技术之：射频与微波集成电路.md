# 1.背景介绍


随着人类对信息的需求日益增加、产业链的不断整合，一些高科技领域新型的设备、系统已经产生出来，如量子通信、机器学习、大数据处理等等。然而，由于产品规模庞大，制造过程复杂，而且需要各个部门密切配合，因此，如何在保证高效率和准确性的同时降低成本成为一个难题。传统的工艺方法或机械流水线方法已无法满足需求的增长，因而出现了以半导体、集成电路(IC)及其软硬件平台为代表的新工艺方法。
通过研究各种新工艺方法的优点、缺点、应用范围，可以发现，其中最为重要且迫切需要解决的问题是如何将多种射频（RF）、微波（Microwave）和光波（Lightwave）信号集成到一起，并转换为数字信号用于信息传输、控制、测控等方面。这种新工艺方法就是本文要讨论的射频与微波集成电路。
射频集成电路(RFIC)是指通过电压信号（或激励信号）实现多种频率、波形和持续时间的无源/有源集成电路，可进行高速数据通信、监控、导航、远程操作、人机交互、环境监测、医疗诊断、机器人控制、环境保护、自动化运输等高性能应用。其中，射频集成电路中包含的RF组件主要包括：

1、功放：功放是一个将输入信号转换为不同频率的组件，它可将电信号的高频分辨率转变为低频分辨率；

2、天线：天线是一种传播无线电波的装置，能够将电磁波转换为无线电波；

3、微电调：微电调是将基带信号转化为加速度输出，可用于传感器和驱动器的输出控制；

4、衰减器：衰减器是一个电气元件，作用是降低射频信号的衰减，使其更容易被接收器识别；

5、功率分配器：功率分配器能够将混合信号中的不同频率分离开来，并为它们提供所需的处理能力；

6、解调器：解调器是根据各种标准或者协议对接收到的射频信号进行译码、解调、滤除、存储、播放等操作，从而将这些信号转换为能读懂的信息形式；

7、载波发生器：载波发生器负责产生导电性的时序信号，作为射频信道的基带信号；

通过使用上述电子元器件的组合，射频集成电路可实现对不同波段的射频信号的处理和传输。
而微波集成电路（Microwave Integrated Circuits，MIMICs），也称为微电子设备（Microelectronic Devices，MEDs）、微电路板（Microcircuit Boards，MCBs），简称MICB，它是利用微波信道进行通信、信息传输、环境感知等的集成电路。MIMIC通常由以下五大组成部分构成：

1、微磁场放大器：将基带微波信号转换成电磁波信号，采用以太网、蓝牙、WIFI、移动通信等常用网络传输技术；

2、微波前端解调器：解调器负责将微波信道收到的信号解码成电磁信号，并提取出基带信号，然后通过放大器输出至下一个模块；

3、短时干扰抑制器：用于消除微波信道中的短时噪声，防止信道干扰导致信息丢失；

4、射频放大器：将基带信号转换为高频信号，作为输出信号进入射频信道；

5、射频解调器：解调器负责将射频信道收到的信号解码，并提取出基带信号，输出至下一个模块。

通过MIMIC可以实现传感、控制、测控、遥感、环境感应、毒性探测、环境保护、自动化运输等功能，从而达到各项应用的目标。
# 2.核心概念与联系
## （一）电容器与非线性元件
电容器是把物理材料包裹在绝缘金属外壳内，用以给电极提供外部环境保护，常用的电容器有绝缘金属包覆层和绝缘纤维包覆层。非线性元件是指具有复杂结构的元件，其电导率、电感率和阻抗随电流变化而变化。典型的非线性元件有感耦合互联体、耦合晶体管、硅基二极管等。

## （二）基带信号
基带信号是指物理层和数字层之间传输数据的基本单元。对于基带信号的定义和分类，目前一般将其分为三类：原始基带信号、数字基带信号和IQ信号。原始基带信号是指模拟信号经过数字采样得到的一系列二进制序列，信号的采样率越高，对应的二进制串越长；数字基带信号是指经过数字编码后得到的一系列数字信号，信号的编码方式和参数决定了数字基带信号的信号格式；IQ信号是指模拟信号和数字信号的复数表示，可以用来描述全双工通信。

## （三）时变分压器、时延互换器
时变分压器是指可以变换信号幅度，延迟和相位的一种元件。时延互换器又称“寻址互换器”、“时延锁存器”，它能够改变信号在信道中的传播方向、相位和距离，改变信号在时域和空域上的分布，因而可以起到选择信道的方式、信号传输方向、信号传输速度的作用。

## （四）相空间与频谱
相空间（Frequency Domain）与频谱（Time Domain）都是信号分析的两个主要概念。相空间与频谱是相互独立的概念。相空间表示信号在空间坐标系中的变化，是物理意义上的信号，由两个坐标轴表示，分别为时间和振幅。频谱则反映信号在振幅频率之间的关系，是抽象意义上的信号，由频率和幅度的平面表示。信号在频谱中占据主导地位的区域，就是所谓的“信号峰值”。
