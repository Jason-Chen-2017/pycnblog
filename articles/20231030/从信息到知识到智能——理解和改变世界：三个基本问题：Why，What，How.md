# 1.背景介绍


传统意义上“智能”这一词语指的是机器具有的一些特定的智能能力、可以独立完成某项工作、解决某种特殊问题等。但实际上，“智能”是一个非常宽泛的概念，它既包含物质层面的智能，也包含精神层面的智能。本文试图用更加宏观的视角，分析和研究人类如何进化出具备“智能”的生物体，并且我们也可以对此做出更加积极的反馈。所以，在文章开头，我希望通过论述人的本性以及智慧的源泉，从哲学的高度，真正站在人类的角度出发，梳理并阐述人类智能的演变史及其驱动因素。

为了达成这个目标，需要明确以下几个关键词：

1.信息：指我们在生活中接收到的各种各样的信息，比如文字、图像、声音、视频等；

2.知识：是指我们学习过的各种信息的总结、归纳和概括，可以分成经验型的（记忆）和思想型的（思维方式），或者更广义的把握人类认知的深度和广度；

3.智能：是指一个个体能够按照自己的意愿，独立地完成各种任务、解决各种问题。其中包括自主意识、自我意识、创新能力等等，也可以将智能定义为智力、理性、情感以及理性判断力等多方面综合能力。

由此，我们可以看到，如果我们想要构建一个具有智能的社会，首先就要搞清楚如何获取、处理、组织和存储这些信息、知识和智能，以及如何让我们的智能发挥作用。只有这样，才能让人类不断进步、实现高度的平等、协同、自主和责任。

# 2.核心概念与联系
## 2.1 信息的分类与特性
所谓信息，就是当事人接收到的各种输入信号。包括有文字、图片、声音、视频等等。不同类型的信息都有其独特的特征，如文字是口头语言，涵盖的内容很丰富；而图像则是用于传递较少且含有空间信息的静态信息，像照片或手绘风格的作品；声音则是人耳听觉器官接受的语音信号，可以播放文字和歌曲；而视频则是由摄像头拍摄出的连续动态影像，属于交互式的媒介形式。

信息是带有随机性、私密性和不可靠性的，因此其处理和存储过程都需要有一套科学的方法论。信息的分类和特征可大致分为三种：

1. 时间性：即信息的生成、储存、传输的时间；
2. 空间性：即信息位于哪些位置，是否是在空间上流动、旋转；
3. 载体性：即信息被编码、加密、压缩的方式，例如数字编码或压缩技术；
4. 能量性：即信息消耗了多少能量，会引起电能的损失或环境污染等。

## 2.2 知识的种类与特点
所谓知识，就是对各种信息的整理、归纳和概括，并建立起对客观世界的认知模型。有经验型的（记忆）和思想型的（思维方式）。经验型知识主要基于一些外部的实践经验，通常以非语言形式存在；而思想型知识则是以符号形式，通常是基于一些古老的哲学理论或西方智库的经典著作。

根据知识的可利用性、传播范围及时效性等特点，知识又可以分为以下几类：

1. 经验型知识：保存下来的经验和真知灼见，以“智商”衡量，一般不会长期保存。如国学、历史、文学、艺术、哲学等；
2. 直觉型知识：不需要严格证实，靠感觉、直觉或直观形成的知识，如政治、经济、法律等；
3. 深思型知识：需要仔细分析、推敲和论证的知识，如工程技术、医学理论、金融理论、心理学理论等；
4. 智库型知识：由专业人士或大型机构进行系统化整理的知识，如专利、技术报告、商业计划书、企业分析报告等；
5. 宗教型知识：信仰或崇拜某个宗教的人士所接受的宗教知识，如佛教、道德经、伊斯兰教等；
6. 概率型知识：基于随机事件发生的概率分布和规律，描述客观现象的模型，如传染病、气候变化、股市趋势等；
7. 计算型知识：通过计算机模型和算法所得出的结果，如股票交易系统、航天飞机导航系统等。

## 2.3 智能的形成与演变
智能是人类发展出具备自主意识、自我意识、创造能力、解决问题能力等一系列能力的能力。也就是说，智能是人的非物质属性，与生俱来、天赋异禀。但随着技术的进步，人类已经有能力开发出复杂的电子芯片、机器人等物理机器，从而使之具备了强大的生物制造能力。

19世纪末，英国著名科学家皮亚杰·卡尼曼提出了“模仿猴子的进化史”，认为“狗也是一种猫的小孩”，从而使人类获得了与其他动物类似的进化优势。

之后的两百年间，智能的产生途径经历了两个阶段，即原始智能与现代智能。在原始智能时期，人类拥有的是直觉、感知、观察力等简单而有效的能力；到了现代智能时期，由于硬件性能的增强，人类的生物制造能力得到进一步提高，同时，对周边环境的理解能力也越来越好。通过演化，人类的智能在不同的方面都取得了突破性的进展。

20世纪初，美国的一位科学家埃里克·波普尔（Erick S. Powell）将智能分为七个阶段：

1. 语言理解能力（Linguistic Comprehension）：能够理解人类使用的语言，例如阅读、写作、口头表达和书面表达；
2. 运用智能行为（Artificial Behavioral Intuition）：能够适应自然、社会和环境，并制定与环境相适应的行动方案；
3. 潜意识（Cognitive Shadow）：与智能行为相辅相成，具备健全的观察、思考、决策能力；
4. 本能（Instinctive Intelligence）：来自自身生理本能，与情绪及习惯相联相通；
5. 直觉性（Spirit of Thought）：对一切事物保持开放、包容态度，喜欢尝试新的事物；
6. 创造能力（Creativity）：表现为对世界的洞察、理解、沉浸感、理性想象力和创造力；
7. 超常发展（Unnatural Expansion）：突破性的进步，可能与先天条件有关。
