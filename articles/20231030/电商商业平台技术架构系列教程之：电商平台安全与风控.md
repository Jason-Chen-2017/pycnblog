# 1.背景介绍


电商（e-commerce）平台是指以互联网为基础的信息化服务平台，是指通过互联网技术、计算机技术、网络技术和通信技术，借助互联网及网络技术为消费者提供全新的购物方式的电子商务服务平台。其核心特点是在线上销售，直接向用户提供商品或服务；多元化产品，适应不同类型的客户需求，满足不同的人群需求；高度个性化，根据顾客个人需求，提供个性化的定制化产品或服务。由于电商平台涉及到的信息量非常庞大，需要高安全性、高可用性、高可靠性，保证平台不因突发事件而崩溃。所以，对电商平台的安全和风控至关重要。

2021年，随着电商平台业务量激增，安全漏洞日益突出，众多大型电商平台遭遇网络攻击、黑客入侵、数据库泄露等安全威胁，严重危害公司利益和声誉，甚至导致商业损失。为了保障电商平台的安全，就需要对电商平台进行相应的安全与风控工作。

本文将从“企业电商”的角度，深入剖析如何构建电商平台的安全与风控体系，从而帮助企业解决电商平台的核心问题，提升企业电商平台的能力。
# 2.核心概念与联系
## 2.1 基本概念
安全和风险管理是保护计算机网络和信息系统免受恶意攻击和恶意行为，降低风险的一种手段。安全管理是指保障组织机构、业务单位、政府部门以及个人在信息技术行业中不受诸如网络攻击、数据泄露、个人信息泄露、系统恶意操作等各种安全威胁，促进组织机构的信息技术工作的开展的过程。风险管理是一种识别、分析、防范和减轻经济损失和社会影响的措施。一般来说，作为一种监督机制，安全管理可以帮助企业更好地抵御网络攻击、保障数据安全，维护网络正常运行；而风险管理则是以事故发生、事件发生、技术存在、人员管理、法律法规为依据，基于业务范围和风险来评估潜在风险，并采取有效的预案，提前做好预防措施的活动。

## 2.2 电商平台的特性
电商平台是一个网站，它具有以下的特征：

1. 用户多样性：电商平台上的用户包括匿名用户、普通用户、vip用户、忠实用户。

2. 交易模式多样性：电商平台支持各种交易模式，包括商品购买、订单支付、订阅服务、商品展示、在线咨询等。

3. 交易量大：电商平台每天都产生海量的交易记录，产生的数据量和用户数量都十分庞大。

4. 交易自动化：电商平台上的交易都需要由机器完成，需要各项操作均自动化处理。

5. 数字货币：电商平台支持各种数字货币，包括支付宝、微信、银联卡、钱包等。

## 2.3 风控与安全管理的区别
安全管理（Security Management）是一门与信息安全相关的学科，其职责是确保计算机系统和网络等信息资源不被非法访问、泄露、篡改、盗用或破坏。此外，安全管理还涉及到保密性、完整性、可用性和可控性，即如何保护机密信息不被未经授权的访问、使用和泄露。

风控（Risk Management）是指评估一个项目或者特定产品或服务的可能风险，并据此制定相应的管理措施以避免、减少或控制风险。风控也是指运用管理知识和技术，结合自身的管理理念和经验，对企业内部管理的决策、流程、资源及管理者的职能作出有力的指导，以减少或者消除企业面临的风险，保障企业的长期利益。

简单说，安全管理是保障网络安全的过程，包含保密性、完整性、可用性、可控性四个方面。而风控则是保障商业实体利益的过程，包含风险管理、成本管理、信用管理、持续改进五个方面。两者有着千丝万缕的联系和相辅相成的关系。

因此，电商平台的安全与风控主要面临两个方面的挑战：

1. 从外部入侵：电商平台属于大规模分布式的网络环境，外部的恶意攻击可能会影响到整个平台的正常运行。

2. 从内部入侵：电商平台的系统架构复杂，内部的攻击也容易造成损失。

如何构建电商平台的安全与风控体系，就是本系列文章要讨论的内容。
