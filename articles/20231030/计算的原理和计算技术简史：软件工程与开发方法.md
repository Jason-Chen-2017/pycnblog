# 1.背景介绍


## 1.1 计算机的历史回顾
计算设备在近代史上从远古的古董机开始走向发达，到二十世纪末成为家喻户晓的名词。历史上有两类主要的计算设备：19世纪晚期到二十世纪初，有联机计算机、个人计算机(PC)、便携式计算机等；二十世纪末到现在，则是集成电路(IC)、集成电子器件、网络设备、嵌入式系统、移动设备等各种各样的高速计算设备。
## 1.2 软件工程与计算技术
软件工程是一个非常重要的分支，它是计算机科学的一门基础学科，也是现代工程技术的一个重要组成部分。随着技术的发展，软件工程的范围也越来越广。它涉及到整个生命周期中需求分析、设计、编码、测试、调试、维护等各个环节，并通过计算机实现功能的自动化。目前，软件工程已经成为集应用、系统开发、需求管理等多方面知识的科技体系，具有重要的社会、经济价值。
计算技术领域的软件工程要解决的问题主要有三方面：一是效率低下、资源浪费严重的问题；二是技术进步速度过快、缺乏可靠性、安全性等质量问题；三是日益增长的数据规模、计算环境复杂化带来的软件复杂度增加问题。而计算技术又是指用硬件或软件的方式来处理信息的技术。
2010年，IBM公司推出了由Intel、AMD、NVIDIA和微软四家公司共同研发的通用计算平台——Sandy Bridge。该平台既能运行传统的应用程序，也可以运行新兴的云计算、大数据、物联网等新型应用。其中的计算技术如多核CPU、图形处理单元(GPU)、FPGA、深度学习加速芯片等正在不断革新。
计算技术的快速发展引起了软件工程师们对如何更好的应对这些挑战感到迫切。在这个过程中，学术界、工业界、产业界以及政策部门都在密切关注计算技术的影响力和未来发展方向。作为产业界代表人物，我国著名的IT科学院院士钟爱民先生曾经说过：“计算机科学就是解决如何利用信息提升生产力的问题。”
