# 2.核心概念与联系
## 2.1 数据可视化的目的和意义
数据可视化（Data Visualization）是指以图形的方式展示数据，通过直观的图形表现形式，帮助人们更好地理解和分析数据。数据可视化的目的就是为了让数据呈现形式变得更加直观、易于理解、清晰、有效，使人类更容易识别、接受、比较和理解复杂的数据。


数据可视化的过程可以分为三个阶段：数据收集、数据预处理、数据可视化。如下图所示：





* 数据收集：包括收集数据的原始信息，包括文本、数字、图片、音频、视频等各种形式的数据。
* 数据预处理：在收集到的数据中可能存在缺失值、异常值、同质性较高等问题，因此需要对其进行处理。这一步主要依赖数据处理算法完成。
* 数据可视化：经过预处理之后的数据就可以用图表或者其他形式的可视化手段呈现出来，便于人们快速了解数据的特性及关联关系。

## 2.2 可视化的数据类型
可视化的数据类型主要有以下几种：

1. 结构化数据：例如Excel文件中的表格数据，包含多维数据，如网页访问记录、销售订单、供应链条中的物料流动等。结构化数据的可视化通常采用柱状图、条形图、饼图、热力图等图表表示。

2. 非结构化数据：图像、视频、音频、文本等非结构化数据没有固定的格式或结构，因此无法直接用柱状图、条形图等简单方式表示。但是，仍可以通过某种聚类方法对这些数据进行降维并显示成网络图、树状图等。

3. 时序数据：时序数据由时间、地点和其他属性组成。因此，时序数据的可视化一般采用折线图、柱状图、散点图等方式呈现。

4. 空间数据：空间数据是指地理位置、景观、物理环境等特征的数据。一般情况下，空间数据的可视化采用的可视化形式包括散点图、气泡图、热度地图等。

5. 多元数据：多元数据是指具有多个维度的不同属性的数据，如股票市场数据中既有价格、涨跌幅等单个维度的数据，也有公司行业、市场营运情况等多个维度的数据。多元数据的可视化一般使用组合图、轮廓图等方式。
