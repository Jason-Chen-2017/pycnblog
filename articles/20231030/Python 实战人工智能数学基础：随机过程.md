# 1.背景介绍


## 一、什么是随机过程？
随机过程(Random Process)是指一系列随机变量的集合，这些随机变量不受外界影响而独立地生成，但都遵循一定规律。这些随机变量可以是时间上的连续变量或离散的时间点序列，也可以是空间上的离散变量或连续区域。通常认为随机过程是相互独立的，即任何两个随机变量之间不存在直接联系。换句话说，随机过程是一个描述客观现象发展过程的数学模型，是一种动态系统。简而言之，随机过程就是由随机变量构成的系统。

随机过程可以用于模拟和预测很多种实际世界中存在的动态现象。比如，流行病学、经济数据、股市价格走势、社会网络的生成等。通过分析随机过程中的数据，可以对其演化进行理论阐述，从而制定相应的管理策略。另外，在一些复杂的系统中，往往存在着多个随机过程，它们之间可以通过互动产生相互作用的影响。因此，掌握对随机过程的基本概念和各种应用方法，对于构建具有自主学习能力的自动控制系统以及解决复杂的问题至关重要。

## 二、为什么需要研究随机过程？
很多现实生活中的事件都是由随机过程驱动的，例如：股票价格的波动、生产效率的提升、道路交通拥堵情况、教育质量的下降等。随着技术的发展，越来越多的设备能够采集到海量的数据，如何理解和分析这些数据，以及用数据驱动经济政策、金融策略和政策的调整，就显得尤为重要。

在人工智能领域，由于大数据的涌入，机器学习算法的迅速发展，使得人们越来越重视数据的处理与分析。为了更好地理解和分析随机过程，需要进行一定的概率论和数理统计知识。

# 2.核心概念与联系
## 一、随机变量（Random Variable）
### 1.定义
一个随机变量是指由随机过程生成的一个数值，它可以是时空上的连续变量或离散的时间点序列。

### 2.特征
#### （1）真随机性(Stationary Randomness)
假设一个随机变量X，其分布在某一区间内以概率p和q不断变化。那么当这个区间内的样本足够多时，我们可以近似地认为X的分布将保持稳定，即
$$P\{X\in A\}=p$$
$$P\{X\in B\}=q$$
其中A和B分别是两个子集，且$A\subseteq X$,$B\subseteq X$.若$X$的真实分布是常数p或q，则称此随机变量是平稳的。

#### （2）连续性
随机变量X的取值可以是连续的，也可以是离散的。如果它是连续的，那么它的概率密度函数可以用积分表示；如果它是离散的，那么它的概率质量函数可以用计数表示。

#### （3）自然发生的原因
随机变量X的分布由潜在的确定因素或者随机过程决定。这类随机变量称为“非 deterministic” 或 “stochastic”。反之，若某个随机变量X只与一组固定的确定因素相关，则称该随机变量为 “deterministic” 或 “fixed”。

## 二、随机过程（Random Process）
### 1.定义
一个随机过程是由一组随机变量所组成的系统。每个随机变量的取值都是随机的，而且都与其它随机变量的取值无关。随机过程可以看做是这些随机变量形成的函数关系，而不是数值本身。

### 2.特性
#### （1）独立性
若$X_i,X_j$为两个随机变量，则称它们是相互独立的，即$P\{X_i=x_i,X_j=x_j\}=P\{X_i=x_i\}P\{X_j=x_j\}$，$X_1,...,X_n$是n个相互独立的随机变量序列，称为多元正态分布，它满足如下条件：
$$E[X]=\sum_{i=1}^n E[X_i]\quad Var[X]=\sum_{i=1}^n Var[X_i]+\sum_{i<j}^n Cov[X_i,X_j]$$
其中，Var[X]是随机变量X方差，Cov[X,Y]是X和Y协方差。

#### （2）平稳性
若$X_t$是关于时间的随机变量，且在任一时刻t处其分布恒定不变，则称其为平稳过程或平稳随机变量。平稳过程具有如下属性：
- 在任一时刻t，$X_t$的分布是确定的；
- 在任意时刻t及任意区间$[a,b]$，$P\{X_t\in [a,b]\}=P\{X_s\in [a,b]\}\forall s\leq t$；
- 如果$\exists c>0$，使得在区间$(-\infty,c)$上$P\{|X_t|\geq c\}>0$,则称过程$\{X_t\}_{t\geq 0}$为有界过程。

#### （3）常数漂移过程
如果存在一个常数$\mu$，使得对于所有$t$,有$E[e^{it\cdot \mu}] = 1$,则称这个随机过程为常数漂移过程。

## 三、马尔科夫链与状态空间
### 1.马尔科夫链
在一个时间尺度上，马尔科夫链(Markov Chain)是一个严格的随机过程，它具备如下三个性质：
- 每一步转移概率只依赖于当前时刻状态，与过去无关；
- 当前时刻状态只依赖于前一时刻状态，与未来的无关；
- 当前时刻状态只依赖于当前时刻前一状态，与不同时间阶段的其他状态无关。

例如，一个股票市场的收益率序列可以看做一个马尔科夫链，因为每一次投资的结果只依赖于当时的投资情况，而不考虑过去或者未来。在另一个例子里，假设一个二维马尔科夫链，在每个时刻，状态只能与周围的固定数量邻居的状态相关。这样的链被称为小型随机游走模型(Small-Scale Random Walk Model)。

### 2.状态空间
马尔科夫链可由一个状态空间Ω=(S，ρ)来定义，其中：
- S为状态空间，是不可空集，它可以是有限的或者无限的；
- ρ(s，s')为转移矩阵，它表示状态s转移到状态s'的概率；
- Σ为状态空间的取值集合。

例如，一个股票市场的状态空间可能是所有可能的股价值，而转移矩阵则表示股价的变动幅度。

## 四、定距过程（IID Discrete Time Random Process）
### 1.定义
定距过程(IID Discrete Time Random Process)是指时间间隔恒定而各时刻随机变量之间的独立关系，即$X_{k+1},X_{k+2},...X_n$相互独立，并且相互独立同分布。

### 2.性质
定距过程可以由下列参数描述：
- $X_0$：初始状态；
- $\{X_t\}_t$：第t时刻的状态；
- $\{d_t\}_t$：第t时刻的步长。

定距过程可以对任意两个相邻时刻的状态变量做如下关系：
$$X_{k+h}=\sum_{i=1}^{N_h} d_t X_t$$

## 五、Poisson过程（Poisson Random Process）
### 1.定义
如果$T_n$服从泊松分布，即$T_n$的期望为$\lambda$，那么$T_n$的样本空间就是整数集合$(0,\cdots, n)$，于是，$T_n$也服从一阶离散时间马尔科夫链，记作$X_t:=T_n(t)$。

$X_t$的分布也是泊松分布，记作$X_t\sim Pois(\lambda)$。

### 2.性质
- $E[X_t]=\lambda t$；
- $Var[X_t]=\lambda t$；
- $X_0$是定值分布；
- $X_{n+1}$依赖于$X_n$；
