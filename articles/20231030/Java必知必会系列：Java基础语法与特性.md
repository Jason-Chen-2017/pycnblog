# 2.核心概念与联系：
# 2.1基本概念
## 2.1.1 Java简介
Java 是一门面向对象的编程语言，是一种静态编译型的语言。它被称为“Write Once, Run Anywhere”，即只需编写一次，即可在各种平台运行。它的设计目标是为了提高软件开发效率，提供与平台无关性。同时，Java 支持多种编程范式，包括命令式、面向对象、函数式、事件驱动等。另外，Java 还提供了许多 API 库用于开发各种应用程序，包括图形用户界面、数据库访问、网络通信、并发处理等。Java 本身采用自然语言的形式，使得它具有很好的可读性和易用性。
## 2.1.2 类与对象
“类”是用来描述对象的模板，它定义了对象所具有的属性和行为。每个类都有一个唯一的名称，并定义了该类的变量、方法、构造函数等。而“对象”是根据类的模板创建出来的一个个具体实例，它是一个实体，由类中的属性和方法组成。对象也有自己的状态（也就是对象属性）和行为（也就是对象的方法），通过这些行为可以实现对内部数据的修改或者获取。
## 2.1.3 内存分配与生命周期管理
Java 中的所有变量都是在堆内存中进行分配的，堆内存是程序运行期间被动态申请和释放的内存块。JVM 的垃圾回收器负责自动释放不再使用的对象占用的内存空间，但是也需要手动释放一些不需要的对象。因此，需要对对象进行正确的生命周期管理，避免造成内存泄漏。
## 2.1.4 异常处理机制
Java 提供了一套完善的异常处理机制，用于处理运行过程中出现的各种错误或异常。它定义了一套继承体系结构，允许用户建立新的类型和自定义异常类。用户可以通过抛出异常的方式，通知调用者某个意料之外的事件发生了，并可以选择进行相应的处理。此外，Java 还提供了栈轨迹、printStackTrace() 方法打印栈信息等便于调试的工具。
## 2.1.5 并发与同步机制
多线程是提升程序执行效率的有效手段。Java 支持多线程程序的开发，但是它并不是完全免费的。开发人员必须考虑线程安全、死锁、竞争条件等问题，并确保线程间的同步互斥。Java 为并发提供了良好的支持，包括线程池、阻塞队列、信号量等机制。
## 2.1.6 反射机制
反射机制提供了运行时的类、方法、字段的动态加载、解析和调用。通过反射，可以灵活地控制程序运行时对象行为，从而实现高度的灵活性。例如，可以使用反射加载外部配置文件，根据配置文件配置程序运行参数，甚至可以在运行时创建对象。
# 2.2 数据类型与变量
## 2.2.1 数据类型
Java 有以下八种基本数据类型：
- byte：字节型数据，占1个字节，表示范围[-128，127]；
- short：短整型数据，占2个字节，表示范围[-32768，32767]；
- int：整型数据，占4个字节，表示范围[-2147483648，2147483647]；
- long：长整型数据，占8个字节，表示范围[-9223372036854775808，9223372036854775807]；
- float：单精度浮点型数据，占4个字节；
- double：双精度浮点型数据，占8个字节；
- char：字符型数据，占2个字节，表示范围[u0000，uffff]。
其中，byte、short、int 和 long 可以使用后缀 L 来标识。
## 2.2.2 变量声明规则
Java 的变量名必须符合标识符的命名规范，且严禁使用关键字。变量名首字符必须是字母，后续字符可以是字母、数字、下划线，但不能以数字开头。而且严格区分大小写，相同名字的变量虽然类型不同，但其作用域是一致的。
Java 的变量声明有两种形式，分别是类型声明和局部变量声明。下面是它们的语法规则：
```java
// 类型声明
Type name; // 声明变量的类型及变量名。

// 局部变量声明
name = value; // 声明变量并初始化值。

Type name = value; // 声明变量的类型、变量名，并初始化值。
```
上面例子中的 Type 表示变量的数据类型，比如 int、double、boolean 等。
局部变量只能在当前方法、构造函数或语句块内声明，并且作用域仅限于声明所在的区域，无法从其他地方访问。
## 2.2.3 常量与特殊符号
Java 中还有一些比较特殊的符号和关键字，它们被称为常量或保留字。如 final、static、this、super 等。final 关键字用于修饰局部变量、成员变量、方法，表示常量。static 表示静态的，成员变量和方法都可以加 static 关键字。this 表示当前对象的引用。super 表示父类的引用。
另外，Java 使用特殊符号代表空值，比如 null 代表空指针，void 表示没有任何返回值的函数。
## 2.2.4 数组
Java 中可以声明和使用一维、二维、三维等多维数组，数组的元素可以是任意数据类型。声明数组的语法如下：
```java
DataType[] arrayName = new DataType[arrayLength];
```
其中，DataType 表示数组中存储的元素的数据类型；arrayName 表示数组的名称；arrayLength 表示数组的长度。数组的索引从0开始，最大值为arrayLength - 1。下面是声明和使用一维数组的示例代码：
```java
public class ArrayDemo {
    public static void main(String[] args) {
        // 声明一维数组
        int[] arr = new int[5];

        // 初始化数组元素
        for (int i = 0; i < arr.length; i++) {
            arr[i] = i + 1;
        }

        // 输出数组元素
        System.out.print("数组元素: ");
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```
输出结果为：`数组元素: 1 2 3 4 5`。
# 3.关键字与运算符
## 3.1 关键字
Java 共有35个关键字，包括 8 个基本类型关键字（如 int、float、char 等）、5 个流程控制语句（如 if、for、while 等）、5 个包装类（如 Integer、Double、Boolean 等）、21 个修饰符（如 private、protected、static、final、synchronized、abstract、transient、volatile、native、strictfp）、1 个协变返回类型注解（@covariant）、1 个不安全的注解（@unsafe）、1 个矩阵乘法运算符（@）、6 个异常处理关键字（try、catch、throw、throws、finally、assert）。

## 3.2 运算符
Java 有以下11种运算符：
- 赋值运算符：=、+=、-=、*=、/=、%=、<<=、>>=、>>>=、&=、^=、|=
- 算术运算符：+、-、*、/、%
- 位运算符：~、&、^、|、>>、<<、>>>
- 比较运算符：<、<=、>、>=、==、!=
- 逻辑运算符：&&、||、!
- 条件运算符：? :
- 其他运算符：instanceof、cast、new、::、.

除了赋值运算符，其他运算符均有优先级、结合性、操作数要求等特性。下面是运算符的使用示例：
```java
int a = 5;   // 赋值运算
a += b * c; // 复合赋值运算
if (x > y && x < z || a == b ^ c!= d &&!e) {} // 逻辑运算
System.out.println((y >= x? x : y)); // 条件运算
System.out.println(Math.pow(2, 3)); // Math类方法调用
```
