## 4.1 Java代码实例

以下是一个简单的Java代码实例，用于实现一个简单的计算器应用：

```java
public class Calculator {
    public static void main(String[] args) {
        System.out.println("请输入两个数字：");
        int num1 = Integer.parseInt(args[0]);
        int num2 = Integer.parseInt(args[1]);
        int result = num1 + num2;
        System.out.println("两个数字之和为：" + result);
    }
}
```

在这个代码实例中，我们首先提示用户输入两个数字。然后，我们使用`Integer.parseInt()`方法将用户输入的字符串转换为整数。接着，我们使用`+`操作符将两个数字相加，并将结果存储在`result`变量中。最后，我们使用`System.out.println()`方法输出结果。

## 4.2 Swift代码实例

以下是一个简单的Swift代码实例，用于实现一个简单的计算器应用：

```swift
import Foundation

func calculate(num1: Int, num2: Int) -> Int {
    return num1 + num2
}

let num1 = 5
let num2 = 3
let result = calculate(num1: num1, num2: num2)

print("两个数字之和为：\(result)")
```

在这个代码实例中，我们首先导入Foundation库。然后，我们定义一个名为`calculate`的函数，用于将两个整数相加。接着，我们使用`let`关键字声明两个整数变量`num1`和`num2`，并将它们赋值为5和3。然后，我们调用`calculate`函数，并将结果存储在`result`变量中。最后，我们使用`print`函数输出结果。

## 4.3 Objective-C代码实例

以下是一个简单的Objective-C代码实例，用于实现一个简单的计算器应用：

```objective-c
#import <Foundation/Foundation.h>

int calculate(int num1, int num2) {
    return num1 + num2;
}

int main(int argc, const char * argv[]) {
    @autoreleasepool {
        int num1 = 5;
        int num2 = 3;
        int result = calculate(num1, num2);

        NSLog(@"两个数字之和为：%d", result);
    }
    return 0;
}
```

在这个代码实例中，我们首先导入Foundation库。然后，我们定义一个名为`calculate`的函数，用于将两个整数相加。接着，我们使用`int`关键字声明两个整数变量`num1`和`num2`，并将它们赋值为5和3。然后，我们调用`calculate`函数，并将结果存储在`result`变量中。最后，我们使用`NSLog`函数输出结果。
