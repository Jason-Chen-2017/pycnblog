以下是快速排序的Python代码实现：

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)

arr = [3, 5, 1, 6, 2, 4]
print(quick_sort(arr))
```

在代码的最后，我们创建了一个数组`arr`，并调用`quick_sort`函数对其进行排序。最后，我们打印出排序后的结果。
