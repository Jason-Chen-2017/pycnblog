```java
import org.junit.Test;
import org.junit.runner.RunWith;
import org.mockito.Mock;
import org.mockito.runners.MockitoJUnitRunner;

public class CalculatorTest {
    @Mock
    private Calculator calculator;

    @Test
    public void testAdd() {
        // 模拟calculator对象的add方法
        when(calculator.add(1, 2)).thenReturn(3);

        // 调用calculator对象的add方法
        int result = calculator.add(1, 2);

        // 验证calculator对象的add方法是否被调用
        verify(calculator).add(1, 2);
    }
}
```

在上述代码中，我们首先导入了Mockito的相关包。然后，我们定义了一个名为CalculatorTest的测试类，该类包含一个名为testAdd的测试方法。在测试方法中，我们使用@Mock注解来模拟Calculator对象，并使用when方法来模拟calculator对象的add方法。然后，我们调用calculator对象的add方法，并使用verify方法来验证calculator对象的add方法是否被调用。
