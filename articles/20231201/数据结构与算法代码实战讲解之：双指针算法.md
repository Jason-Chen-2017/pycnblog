# 2.核心概念与联系
双指针算法的核心概念是通过使用两个指针来遍历数据结构，以找到满足某种条件的元素。这两个指针分别称为快指针和慢指针。快指针通常从数组的头部或尾部开始，慢指针从数组的头部或尾部开始。通过移动这两个指针，我们可以找到满足某种条件的元素。

```python
def find_sum_to_target(nums, target):
    slow_pointer = 0
    fast_pointer = 0
    while fast_pointer < len(nums):
        if nums[fast_pointer] + nums[slow_pointer] == target:
            return (slow_pointer, fast_pointer)
        elif nums[fast_pointer] + nums[slow_pointer] < target:
            fast_pointer += 1
        else:
            slow_pointer += 1
    return None
```
