# 4.具体代码实例和详细解释说明

## 4.1泛型代码实例

以下是一个泛型列表的代码实例：

```python
from typing import List

def generic_list(items: List[object]):
    for item in items:
        print(item)

# 使用泛型列表
generic_list([1, 2, 3])
generic_list(["a", "b", "c"])
```

在这个代码实例中，我们定义了一个泛型列表的函数 `generic_list`，它接受一个列表作为输入，并打印出列表中的每个元素。我们可以使用这个函数来处理不同类型的输入数据，例如整数列表和字符串列表。

## 4.2模板元编程代码实例

以下是一个模板元编程的代码实例：

```cpp
#include <iostream>

template <typename T>
struct AddOne {
    static constexpr T value = T(1) + T();
};

int main() {
    std::cout << AddOne<int>::value << std::endl;
    std::cout << AddOne<double>::value << std::endl;
    return 0;
}
```

在这个代码实例中，我们定义了一个模板元编程结构 `AddOne`，它接受一个类型参数 `T`。我们使用 `static constexpr` 关键字来定义一个常量 `value`，它的值是 `T(1) + T()`。我们可以使用这个结构来计算不同类型的值，例如整数和双精度浮点数。
