# 4.具体代码实例和详细解释说明
在这里，我们将提供一个具体的代码实例，以及对其的详细解释。

```python
def sort(arr):
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] > arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
sorted_arr = sort(arr)
print(sorted_arr)
```

在这个例子中，我们实现了一个简单的冒泡排序算法。我们首先遍历数组，然后比较相邻的元素，如果发现两个元素顺序错误，我们就交换它们。我们重复这个过程，直到整个数组有序。
