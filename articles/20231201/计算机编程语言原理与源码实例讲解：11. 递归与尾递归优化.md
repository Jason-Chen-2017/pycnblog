# 4.具体代码实例和详细解释说明

递归的代码实例可以帮助我们更好地理解递归的原理和应用。以下是一个递归的代码实例：

```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)
```

在这个代码实例中，我们定义了一个递归函数 `factorial`，它用于计算一个数的阶乘。递归函数的定义包含一个调用自身的语句，即 `return n * factorial(n-1)`。递归函数的终止条件是 `if n == 0`，它用于避免无限递归。

```python
def factorial(n, accumulator=1):
    if n == 0:
        return accumulator
```

在这个代码实例中，我们定义了一个尾递归函数 `factorial`，它用于计算一个数的阶乘。尾递归函数的定义包含一个调用自身的语句，即 `return factorial(n-1, n * accumulator)`。尾递归函数的返回值是递归调用的参数，即 `n * accumulator`。尾递归函数的终止条件是 `if n == 0`，它用于避免无限递归。
