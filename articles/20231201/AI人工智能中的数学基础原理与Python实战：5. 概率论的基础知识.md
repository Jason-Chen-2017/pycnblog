# 2.核心概念与联系

## 2.1概率的基本概念

概率是一种用于描述事件发生的可能性的数学方法。它可以用来计算事件发生的可能性，也可以用来计算事件不发生的可能性。概率的取值范围是[0,1]，表示事件发生的可能性。

## 2.2概率的基本定义

概率的基本定义是通过事件的空集、事件的肯定集和事件的补集来定义的。事件的空集表示不发生的事件，其概率为0。事件的肯定集表示发生的事件，其概率为1。事件的补集表示不发生的事件，其概率为1-P(A)。

## 2.3概率的基本定理

概率的基本定理是用来计算多个事件发生的概率的。它可以用来计算两个事件发生的概率、三个事件发生的概率等。概率的基本定理是通过和、积、差等运算符来计算的。
