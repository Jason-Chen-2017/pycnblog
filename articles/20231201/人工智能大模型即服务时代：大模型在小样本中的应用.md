## 4.1 代码实例

以下是一个使用BERT模型在小样本中进行文本分类的代码实例：

```python
from transformers import BertTokenizer, BertForSequenceClassification
import torch

# 加载预训练模型和标记器
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
model = BertForSequenceClassification.from_pretrained('bert-base-uncased', num_labels=2)

# 加载小样本
data = [("I love this movie.", "positive"), ("I hate this movie.", "negative")]

# 将文本转换为输入格式
inputs = tokenizer(data, padding=True, truncation=True, return_tensors='pt')

# 进行预测
outputs = model(**inputs)
preds = torch.softmax(outputs.logits, dim=-1)

# 输出预测结果
for text, label, pred in zip(data, preds.argmax(dim=-1).tolist(), preds.data.tolist()):
    print(f"Text: {text}, Label: {label}, Pred: {pred}")
```

## 4.2 详细解释说明

上述代码实例主要包括以下几个步骤：

1. 加载预训练模型和标记器：使用Hugging Face的Transformers库加载BERT模型和标记器。
2. 加载小样本：将小样本加载到内存中，其中每个样本包括一个文本和一个标签。
3. 将文本转换为输入格式：使用标记器将文本转换为BERT模型所需的输入格式，包括token化、padding和截断等。
4. 进行预测：使用加载好的模型进行预测，并获取预测结果。
5. 输出预测结果：将预测结果输出到控制台，包括原始文本、标签和预测结果。
