# 4.具体代码实例和详细解释说明

以下是一个简单的Python代码实例，用于计算某个事件的概率：

```python
import random

def calculate_probability(n, m):
    total = 0
    for i in range(n):
        if random.random() < 0.5:
            total += 1
    return total / n

print(calculate_probability(1000, 500))
```

在这个代码中，我们首先导入了random模块，然后定义了一个calculate_probability函数，用于计算某个事件的概率。这个函数接受两个参数：n和m。n表示总的样本数，m表示满足某个条件的样本数。然后，我们使用一个for循环来生成n个随机数，并计算满足某个条件的样本数。最后，我们返回满足某个条件的样本数除以总的样本数，得到某个事件的概率。
