# 4.具体代码实例和详细解释说明

在这里，我们将提供一个具体的代码实例，以及它的详细解释说明。

```python
def search(query, index):
    results = []
    for term in query.split():
        for doc in index.get(term, []):
            if doc not in results:
                results.append(doc)
    return results
```

这段代码实现了一个简单的搜索算法。它接受一个查询和一个索引，然后返回与查询相关的文档列表。
