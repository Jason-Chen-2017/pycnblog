## 3.1 随机变量的概念和类型
随机变量是一个随机过程中的一个随机元素，它可以取任意值。随机变量的分布函数是描述随机变量取值概率的函数。随机变量的类型有两种：离散型和连续型。离散型随机变量的取值是离散的，而连续型随机变量的取值是连续的。

## 3.2 概率论基础知识
概率论是一门数学学科，它研究随机事件发生的可能性。概率是一个数值，表示某个事件发生的可能性。概率的范围是0到1，表示事件发生的可能性。概率的计算方法有多种，如频率法、定义法、比例法等。

## 3.3 随机变量的分布函数
随机变量的分布函数是描述随机变量取值概率的函数。分布函数的定义是：F(x) = P(X ≤ x)，其中F(x)是分布函数，x是随机变量的取值，P(X ≤ x)是随机变量X的取值小于或等于x的概率。

## 3.4 常见的随机变量分布
常见的随机变量分布有：均匀分布、指数分布、正态分布等。这些分布都有自己的特点和应用场景。

## 3.5 随机变量的期望和方差
期望是随机变量取值的平均值，用于描述随机变量的中心趋势。方差是随机变量取值离中心趋势的程度，用于描述随机变量的不确定性。期望和方差是随机变量的重要特征，用于描述随机变量的性质。

## 4.1 使用numpy库计算随机变量的期望和方差
numpy是Python中的一个数学库，可以用来计算随机变量的期望和方差。以下是一个使用numpy计算随机变量期望和方差的Python代码实例：

```python
import numpy as np

# 创建一个随机数列
random_numbers = np.random.rand(100)

# 计算随机数列的期望

print("随机数列的期望:", mean)

# 计算随机数列的方差
variance = np.var(random_numbers)
print("随机数列的方差:", variance)
```
