## 4.具体代码实例和详细解释说明
### 4.1 代码实例
以下是一个简单的自动化脚本实例，用于复制文件夹中的所有文件到另一个文件夹：

```python
import shutil
import os

# 设置源文件夹和目标文件夹
src_folder = "source"
dst_folder = "destination"

# 遍历源文件夹中的所有文件
for file in os.listdir(src_folder):
    # 获取文件的绝对路径
    src_file = os.path.join(src_folder, file)
    # 获取目标文件的绝对路径
    dst_file = os.path.join(dst_folder, file)
    # 复制文件
    shutil.copy(src_file, dst_file)
```

### 4.2 详细解释说明
上述代码实例的解释说明如下：

4.2.1 导入模块：在代码中，我们需要使用`shutil`模块和`os`模块。`shutil`模块提供了用于复制文件和目录的函数，`os`模块提供了用于操作文件和目录的函数。
4.2.2 设置源文件夹和目标文件夹：我们需要设置源文件夹和目标文件夹的路径，以便于复制文件。
4.2.3 遍历源文件夹中的所有文件：我们使用`os.listdir()`函数遍历源文件夹中的所有文件，并获取文件的名称。
4.2.4 获取文件的绝对路径：我们使用`os.path.join()`函数获取文件的绝对路径。
4.2.5 复制文件：我们使用`shutil.copy()`函数复制文件，将源文件复制到目标文件夹中。
