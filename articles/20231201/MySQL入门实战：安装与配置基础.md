# 2.核心概念与联系

在了解MySQL的安装与配置之前，我们需要了解一些核心概念和联系。

## 2.1 MySQL的组成

MySQL的组成包括：MySQL服务器、MySQL客户端和MySQL数据库管理工具。

- MySQL服务器：MySQL服务器是MySQL的核心组成部分，它负责存储和管理数据库数据，同时提供数据库的查询、修改、删除等功能。MySQL服务器是由C语言编写的，并采用多线程和多进程的设计，以提高性能和稳定性。
- MySQL客户端：MySQL客户端是MySQL的另一个重要组成部分，它提供了与MySQL服务器进行通信的接口，用户可以通过MySQL客户端向MySQL服务器发送SQL查询语句，并获取查询结果。MySQL客户端是由C语言编写的，并支持多种编程语言的接口，如C、C++、Java、Python等。
- MySQL数据库管理工具：MySQL数据库管理工具是MySQL的辅助工具，它提供了一种图形化的界面，用户可以通过数据库管理工具进行数据库的创建、删除、修改、查询等操作。MySQL数据库管理工具包括MySQL Workbench、phpMyAdmin等。

## 2.2 MySQL的数据库和表

MySQL的数据库是一组相关的表的集合，表是数据库中的基本组成单位，用于存储和管理数据。MySQL支持多种数据类型，如整数、浮点数、字符串、日期时间等，同时也支持多种数据库引擎，如InnoDB、MyISAM等。

- MySQL数据库：MySQL数据库是一组相关的表的集合，用于存储和管理数据。MySQL数据库可以创建、删除、修改、查询等操作。MySQL数据库的名称是唯一的，同一个MySQL实例中不能存在相同名称的数据库。
- MySQL表：MySQL表是数据库中的基本组成单位，用于存储和管理数据。MySQL表由一组列组成，列是表中的基本组成单位，用于存储和管理数据。MySQL表可以创建、删除、修改、查询等操作。MySQL表的名称是唯一的，同一个MySQL数据库中不能存在相同名称的表。

## 2.3 MySQL的数据库引擎

MySQL支持多种数据库引擎，如InnoDB、MyISAM等。数据库引擎是MySQL的核心组成部分，它负责存储和管理数据库数据，同时提供数据库的查询、修改、删除等功能。

- InnoDB：InnoDB是MySQL的默认数据库引擎，它支持事务、行级锁定和外键等功能。InnoDB是一个高性能、稳定、安全的数据库引擎，它的性能优越，使其成为许多网站和应用程序的首选数据库引擎。
- MyISAM：MyISAM是MySQL的另一个数据库引擎，它支持表级锁定和全文本搜索等功能。MyISAM是一个高性能、稳定、安全的数据库引擎，它的性能优越，使其成为许多网站和应用程序的首选数据库引擎。
