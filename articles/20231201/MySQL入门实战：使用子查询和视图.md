### 1.1.1 子查询

子查询是一种在SQL语句中使用的查询，它可以将一个查询的结果作为另一个查询的一部分。子查询可以用于查询、更新、删除等操作。子查询可以嵌套使用，可以实现复杂的查询逻辑。

### 1.1.2 视图

视图是一种虚拟表，它是一个查询的快捷方式。视图可以将复杂的查询语句简化为一个简单的查询语句，从而提高查询效率。视图可以用于查询、更新、删除等操作。

## 4.具体代码实例和详细解释说明

### 4.1 子查询的代码实例

```sql
SELECT * FROM table1 WHERE id IN (SELECT id FROM table2 WHERE name = 'John');
```

在这个例子中，我们使用了一个子查询来获取table1中id为table2中name为'John'的记录。

### 4.2 子查询的详细解释说明

在这个例子中，我们使用了一个子查询来获取table1中id为table2中name为'John'的记录。子查询的结果是一个包含id的列表，我们使用IN关键字来匹配table1中的id。

### 4.3 视图的代码实例

```sql
CREATE VIEW view1 AS SELECT * FROM table1 WHERE id IN (SELECT id FROM table2 WHERE name = 'John');
```

在这个例子中，我们创建了一个视图view1，用于获取table1中id为table2中name为'John'的记录。

### 4.4 视图的详细解释说明

在这个例子中，我们创建了一个视图view1，用于获取table1中id为table2中name为'John'的记录。视图的查询语句与子查询的查询语句相同，但是我们将查询语句封装到视图中，从而简化了查询语句。
