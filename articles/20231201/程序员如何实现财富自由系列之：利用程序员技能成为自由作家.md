## 4.1 自动发布作品

以下是一个使用Python编写的自动发布作品的代码实例：

```python
import requests
import json

# 设置电子书平台的API地址和API密钥
ebook_platform_api_url = "https://api.example.com"
ebook_platform_api_key = "your_api_key"

# 设置作品的基本信息
book_title = "你的作品标题"
book_author = "你的作者名称"
book_price = 9.99

# 设置电子书平台的API请求头
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {ebook_platform_api_key}"
}

# 设置电子书平台的API请求体
book_data = {
    "title": book_title,
    "author": book_author,
    "price": book_price
}

# 发布作品
response = requests.post(f"{ebook_platform_api_url}/books", headers=headers, json=book_data)

# 判断发布是否成功
if response.status_code == 200:
    print("作品发布成功")
else:
    print("作品发布失败")
```
