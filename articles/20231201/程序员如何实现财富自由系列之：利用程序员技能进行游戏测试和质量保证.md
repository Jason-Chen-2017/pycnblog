```python
import time

def slow_function(n):
    start_time = time.time()
    for i in range(n):
        pass
    end_time = time.time()
    return end_time - start_time

def optimized_function(n):
    start_time = time.time()
    for i in range(n):
        if i % 1000 == 0:
            pass
        else:
            pass
    end_time = time.time()
    return end_time - start_time

n = 1000000
slow_time = slow_function(n)
print("Slow function took {} seconds".format(slow_time))

optimized_time = optimized_function(n)
print("Optimized function took {} seconds".format(optimized_time))
```

在这个例子中，我们定义了一个slow_function函数，该函数执行了n次循环，每次循环都执行了一次空操作。我们也定义了一个optimized_function函数，该函数在每次循环中添加了一个条件判断，以减少空操作的次数。通过这种方式，我们可以看到优化后的函数执行时间比原始函数要短。
