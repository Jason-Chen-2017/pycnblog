# 2.核心概念与联系
# 2.1概率论基本概念
概率论是一门数学分支，研究事件发生的可能性。概率论的基本概念包括事件、样本空间、事件的概率、独立事件、条件概率等。

事件：是一个可能发生或不发生的结果。

样本空间：是所有可能发生的事件集合。

事件的概率：是事件发生的可能性，范围在0到1之间。

独立事件：是两个事件发生或不发生的结果不会影响彼此的事件。

条件概率：是一个事件发生的概率，给定另一个事件已发生。

# 2.2统计学基本概念
统计学是一门数学分支，研究数据的收集、处理和分析。统计学的基本概念包括数据、数据分布、统计量、统计假设、统计检验等。

数据：是事物的集合，可以用数字表示。

数据分布：是数据集合中各值出现的概率分布。

统计量：是数据集合的一个数值表示。

统计假设：是一个事件或现象的假设。

统计检验：是用来判断一个统计假设是否成立的方法。
