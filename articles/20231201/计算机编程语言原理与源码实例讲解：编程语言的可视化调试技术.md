以下是一些具体的代码实例：

1. 设置断点：

```python
def main():
    x = 10
    if x > 5:
        print("x > 5")
    else:
        print("x <= 5")

main()
```

在这个代码实例中，我们可以设置断点在`if`语句之前，以便在程序运行时暂停执行，以便查看程序的执行过程。

2. 查看变量：

```python
def main():
    x = 10
    y = 20
    z = x + y
    print("x =", x)
    print("y =", y)
    print("z =", z)

main()
```

在这个代码实例中，我们可以查看程序中的变量，以便了解变量的值和变化。
