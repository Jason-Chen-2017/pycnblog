# 2.核心概念与联系
上下文无关文法（Context-Free Grammar, CFG）是一种描述语言结构的形式，它可以用来描述编程语言的语法规则。CFG由一组产生式组成，每个产生式包含一个非终结符和一个或多个终结符。CFG的核心概念包括：

- 非终结符：表示语法符号，可以被替换或扩展的符号。
- 终结符：表示具体的字符或符号，不能被替换或扩展的符号。
- 产生式：描述如何将非终结符替换为终结符的规则。
- 文法规则：描述语法结构的规则，包括语法树、语法分析、语法生成等。
