## 2.核心概念与联系
Python的数据类型可以分为两类：基本数据类型和复合数据类型。基本数据类型包括整数、浮点数、字符串、布尔值和None。复合数据类型包括列表、元组、字典和集合。

### 2.1 基本数据类型
#### 2.1.1 整数
整数是Python中的一种数字类型，可以表示正整数、负整数和零。整数可以是长整数，长整数可以表示更大的数字。

#### 2.1.2 浮点数
浮点数是Python中的一种数字类型，可以表示小数。浮点数由整数部分和小数部分组成，整数部分可以是正数或负数，小数部分可以是正数或负数。

#### 2.1.3 字符串
字符串是Python中的一种数据类型，可以表示文本。字符串可以是单引号、双引号或三引号包围的文本。

#### 2.1.4 布尔值
布尔值是Python中的一种数据类型，可以表示真（True）或假（False）。布尔值用于表示条件判断和逻辑运算。

#### 2.1.5 None
None是Python中的一种特殊数据类型，表示一个空值。None可以用于表示缺失的值或无效的值。

### 2.2 复合数据类型
#### 2.2.1 列表
列表是Python中的一种数据类型，可以表示有序的、可变的集合。列表可以包含任意类型的数据，可以通过下标访问和修改列表中的元素。

#### 2.2.2 元组
元组是Python中的一种数据类型，可以表示有序的、不可变的集合。元组可以包含任意类型的数据，可以通过下标访问元组中的元素。

#### 2.2.3 字典
字典是Python中的一种数据类型，可以表示无序的、键值对的集合。字典可以包含任意类型的键和值，可以通过键访问字典中的值。

#### 2.2.4 集合
集合是Python中的一种数据类型，可以表示无序的、不可重复的集合。集合可以包含任意类型的数据，可以通过集合操作符进行集合运算。
