# 2.核心概念与联系

在进行AI和ML的研究之前，我们需要了解一些基本的数学概念。这些概念包括：

1. 概率：概率是一种数学概念，用于描述事件发生的可能性。在机器学习中，我们经常需要处理不确定性，因此概率是一个重要的概念。

2. 线性代数：线性代数是一种数学分支，它涉及向量、矩阵和线性方程组的研究。在机器学习中，我们经常需要处理大量的数据，因此线性代数是一个重要的工具。

3. 微积分：微积分是一种数学分支，它涉及连续函数的导数和积分的研究。在机器学习中，我们经常需要处理连续的数据，因此微积分是一个重要的工具。

4. 优化：优化是一种数学方法，用于找到一个函数的最大值或最小值。在机器学习中，我们经常需要优化模型的参数，以便提高其性能。

5. 信息论：信息论是一种数学分支，它涉及信息的量化和传输的研究。在机器学习中，我们经常需要处理信息，因此信息论是一个重要的概念。
