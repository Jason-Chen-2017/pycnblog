## 1.4 具体代码实例和详细解释说明

### 1.4.1 异步编程的具体代码实例

```python
import asyncio

async def async_func():
    print("Starting async function")
    await asyncio.sleep(1)
    print("Finished async function")

async def main():
    print("Starting main function")
    await async_func()
    print("Finished main function")

asyncio.run(main())
```

异步编程的具体代码实例如上所示。在这个例子中，我们定义了一个异步函数 async_func，它会在 1 秒钟后打印一条消息。然后，我们定义了一个主函数 main，它会在 async_func 完成后打印一条消息。最后，我们使用 asyncio.run 函数来运行主函数。

### 1.4.2 并发的具体代码实例

```python
import threading

def thread_func():
    print("Starting thread function")
    threading.sleep(1)
    print("Finished thread function")

def main():
    print("Starting main function")
    t = threading.Thread(target=thread_func)
    t.start()
    t.join()
    print("Finished main function")

main()
```

并发的具体代码实例如上所示。在这个例子中，我们定义了一个线程函数 thread_func，它会在 1 秒钟后打印一条消息。然后，我们定义了一个主函数 main，它会启动一个线程 t，并执行 thread_func。最后，我们使用 t.join 函数来等待线程 t 的完成。
