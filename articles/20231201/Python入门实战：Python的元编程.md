## 4.1 元编程的代码实例
以下是一个简单的元编程代码实例：

```python
class MyClass(object):
    def __init__(self, x):
        self.x = x

    def add(self, y):
        return self.x + y

# 创建元类
MetaClass = type('MetaClass', (MyClass,), {'add': lambda self, y: self.x * y})

# 创建元对象
obj = MetaClass(10)

# 调用元方法
print(obj.add(2))  # 输出: 20
```
