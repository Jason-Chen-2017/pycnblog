## 1.1 Java的历史与发展
Java是一种广泛使用的编程语言，由Sun Microsystems公司于1995年发布。它的设计目标是让程序员能够在任何平台上编写可移植的代码。Java的发展历程可以分为以下几个阶段：

1.1.1 Java 1.0（1995年）：Java的第一个版本，主要用于创建跨平台的应用程序。

1.1.2 Java 2（1996年）：Java的第二个版本，引入了新的类库和API，提高了开发效率。

1.1.3 Java 3（1998年）：Java的第三个版本，引入了新的类库和API，提高了程序的性能和可移植性。
