# 2.核心概念与联系

计算机存储技术的核心概念包括：存储单元、存储器层次结构、存储器类型、存储器性能指标等。

## 2.1 存储单元

存储单元是计算机存储技术的基本组成部分，它可以存储二进制数据。存储单元可以分为两类：存储器（Memory）和存储设备（Storage）。

- 存储器：内存，用于存储计算机当前正在使用的数据和程序。内存可以进一步分为随机存取存储器（RAM）和只读存取存储器（ROM）。
- 存储设备：硬盘、USB闪存等，用于长期存储数据。

## 2.2 存储器层次结构

存储器层次结构是计算机存储技术的一个重要概念，它描述了不同类型的存储器之间的层次关系。存储器层次结构可以分为五个层次：缓存、主内存、虚拟内存、磁盘和云存储。

- 缓存：高速缓存存储最常用的数据和程序，以便快速访问。缓存可以进一步分为数据缓存、指令缓存和预取缓存。
- 主内存：主内存存储当前正在使用的数据和程序。主内存可以进一步分为随机存取存储器（RAM）和只读存取存储器（ROM）。
- 虚拟内存：虚拟内存是主内存的扩展，它使用交换文件和交换区域来模拟更大的主内存。虚拟内存可以进一步分为页面和段。
- 磁盘：磁盘是长期存储数据的设备，它使用磁头来读写数据。磁盘可以进一步分为硬盘、固态硬盘和外部硬盘。
- 云存储：云存储是通过互联网来存储数据的服务，它可以提供更大的存储空间和更高的可用性。

## 2.3 存储器类型

存储器类型是计算机存储技术的一个重要分类标准，它可以分为两类：随机存取存储器（RAM）和只读存取存储器（ROM）。

- 随机存取存储器（RAM）：RAM是计算机当前正在使用的数据和程序的存储器。RAM可以进一步分为静态随机存取存储器（SRAM）和动态随机存取存储器（DRAM）。
- 只读存取存储器（ROM）：ROM是存储固定数据的存储器，它的数据不能被修改。ROM可以进一步分为只读内存（ROM）、只读存储器（EEPROM）和闪存（Flash）。

## 2.4 存储器性能指标

存储器性能指标是用于评估存储器性能的标准，它们包括：存储容量、读写速度、延迟、功耗等。

- 存储容量：存储容量是存储器能存储数据的大小，它通常以字节（Byte）或比特（Bit）为单位。
- 读写速度：读写速度是存储器读取和写入数据的速度，它通常以字节/秒（Byte/s）或比特/秒（Bit/s）为单位。
- 延迟：延迟是存储器从读取命令到实际读取数据的时间，它通常以微秒（μs）或纳秒（ns）为单位。
- 功耗：功耗是存储器在运行过程中消耗的能量，它通常以瓦特（Watt）或毫瓦特（mW）为单位。
