计算机图形学是一门研究计算机图像处理、生成和显示的学科。它的起源可以追溯到1960年代，当时的计算机图形学研究主要集中在计算几何、图像处理和计算机图形学的基本算法和数据结构上。随着计算机技术的不断发展，计算机图形学逐渐发展成为一门独立的学科，涉及到计算机图形学的各个方面，如计算机图形学的基本概念、计算机图形学的算法、计算机图形学的应用等。

计算机图形学的发展也与计算机技术的发展密切相关。从早期的矢量图形到现代的光栅图形，从简单的几何形状到复杂的3D模型，从单色显示到高清彩色显示，计算机图形学的技术不断发展，为现代计算机图形学提供了更丰富的可视化效果。

虚拟现实是一种使用计算机生成的人工环境，让用户感觉自己处于一个虚拟的世界中的技术。虚拟现实的起源可以追溯到1960年代，当时的虚拟现实研究主要集中在虚拟现实的基本概念、虚拟现实的技术和虚拟现实的应用上。随着计算机技术的不断发展，虚拟现实逐渐发展成为一门独立的学科，涉及到虚拟现实的各个方面，如虚拟现实的基本概念、虚拟现实的算法、虚拟现实的应用等。

虚拟现实的发展也与计算机技术的发展密切相关。从简单的显示技术到现代的虚拟现实设备，从单模态到多模态，从单人到多人，虚拟现实的技术不断发展，为现代虚拟现实提供了更丰富的交互和体验。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义三角形的三个顶点
vertices = np.array([[0, 0], [1, 0], [0, 1]])

# 绘制三角形

plt.axis('equal')
plt.show()
```

3. 绘制三角形：我们使用matplotlib库来绘制这个三角形，使用'bo-'参数来表示蓝色的圆点连接线。
