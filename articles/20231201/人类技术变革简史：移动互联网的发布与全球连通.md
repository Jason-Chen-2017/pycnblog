# 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来详细解释移动互联网的工作原理。

代码实例：

```python
import requests
from requests.exceptions import RequestException

def get_html(url):
    try:
        with requests.get(url, timeout=30) as response:
            if response.status_code == 200:
                return response.text
            else:
                print('Error:', response.status_code)
                return None
    except RequestException as e:
        print('Error:', e)
        return None

url = 'http://www.example.com'
html = get_html(url)
if html:
    print(html)
```

这段代码的主要功能是通过HTTP请求获取指定URL的HTML内容。它首先导入了`requests`库，然后定义了一个`get_html`函数，该函数用于发送HTTP请求并获取响应内容。如果请求成功，函数将返回响应内容，否则将返回`None`。最后，它通过调用`get_html`函数获取指定URL的HTML内容，并将其打印出来。

这段代码的核心逻辑是通过`requests`库发送HTTP请求，并获取响应内容。`requests`库是一个Python的HTTP库，它提供了一系列的HTTP请求方法，如`get`、`post`等。通过这些方法，我们可以方便地发送HTTP请求并获取响应内容。

在这个代码实例中，我们使用了`requests.get`方法发送HTTP GET请求，并设置了请求超时时间为30秒。如果请求成功，我们将返回响应内容，否则将返回`None`。最后，我们通过调用`get_html`函数获取指定URL的HTML内容，并将其打印出来。
