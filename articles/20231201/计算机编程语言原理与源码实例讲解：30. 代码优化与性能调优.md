## 4.1 代码实例
我们将使用一个简单的排序算法——冒泡排序来进行代码优化和性能调优。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

## 4.2 代码优化
我们可以通过以下方法对冒泡排序算法进行优化：

1. 减少内层循环的次数：我们可以在每次循环后，将最后一个元素放到正确的位置，从而减少内层循环的次数。
2. 使用二分法进行比较：我们可以使用二分法进行比较，而不是逐个比较。

优化后的代码如下：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        flag = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                flag = True
        if not flag:
            break
```
