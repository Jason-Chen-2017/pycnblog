# 1.简介
         
## 1.1 概述
随着互联网、移动互联网、物联网等新型数字化经济的发展，以及云计算、大数据、人工智能等新型产业的快速崛起，基于海量数据的分布式系统与大数据处理的需求已经越来越强烈。作为分布式系统和大数据处理领域的先锋者之一，中国科技大学信息工程学院副教授秦晖老师将带领大家认识并掌握分布式系统和大数据处理的基本知识、理论和方法，通过真实案例加强对分布式系统及其在数据分析、机器学习、人工智能等领域的应用。本课程旨在培养学生的思维逻辑能力、分析问题和解决问题的能力、团队合作精神、面向对象编程能力、系统工程的知识储备，这些能力将支撑学生自主性、求知欲和创新能力。本课程涵盖了分布式系统和大数据处理的方方面面，包括分布式存储系统、分布式计算系统、分布式文件系统、分布式消息系统、分布式数据库系统、分布式并行计算系统、高性能计算系统、大数据分析系统、云计算平台、大数据挖掘、图形处理等多个方向。
## 1.2 目标受众
本课程主要面向具备相关背景的本科生、研究生以及具有一定工作经验的IT从业人员，包括但不限于运维工程师、系统架构师、高级软件工程师、算法工程师、项目经理、产品经理、市场推广专员、互联网从业者、金融分析师、数据分析师、机器学习专家等。
## 1.3 上课方式
本课程以主题分享的方式进行，每周开两次课，第一、二节课每周一上午9:30-11:15，第二、三节课每周三上午9:30-11:15。
## 1.4 难度等级
本门课程的难度系数较高，一般参加本门课程的同学都具备扎实的计算机基础、系统工程基础知识、基本的编程能力。同时，本门课程适合需要全面系统掌握分布式系统和大数据处理技术的技术类专业选修。
## 1.5 参考书目
《大数据时代》、《高性能计算导论》、《数据挖掘导论》、《程序员必读之道》、《C++ Primer Plus》、《UNIX环境高级编程》、《数据结构与算法分析》、《数学之美》、《微服务设计》等。
## 1.6 交通路线
本门课程在北京西直门地铁站坐车到昌平路站，步行约5分钟左右，或乘坐367路、406路或动车到达北四环（另有昌平客运中心）。
## 1.7 时段安排
本门课程在上课时间上属于早上9点到下午5点30分之间，在此期间，同学们可以进入宿舍完成作业，做好充足的休息。
## 1.8 授课计划
1. 分布式存储系统
- 分布式文件系统
  - Hadoop的HDFS与MapReduce
  - Google File System（GFS）与Bigtable
- 分布式数据库系统
  - HBase和Cassandra
  - Apache Accumulo和Apache Cassandra

2. 分布式计算系统
- 分布式计算框架
  - Apache Hadoop MapReduce、Spark、Storm
  - Apache Tez、Apache Flink、Apache Spark Streaming
- 大规模并行计算
  - OpenCL、CUDA、OpenMP、MPI、Hadoop的MapReduce和Spark

3. 分布式消息系统
- Apache Kafka
- Apache Pulsar

4. 云计算平台
- Amazon Web Services
- Microsoft Azure

5. 大数据分析系统
- 数据采集、清洗、存储、转移
- 分析技术（统计模型、推荐系统、网络分析、文本挖掘）
- 可视化工具（Hadoop HDFS、Hue、ElasticSearch）

6. 大数据挖掘
- 大数据源挖掘（日志数据、用户行为数据、网站数据、金融数据）
- 机器学习（分类、回归、聚类、关联规则）
- 图论（社区发现、标签传播、链接预测）
