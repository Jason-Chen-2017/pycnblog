# 1.简介
         
数字化人才管理（Digital Talent Management）是指通过技术手段，利用大数据、云计算等新兴技术帮助企业将人才从就业岗位中“雇佣”出来。数字化人才管理解决了传统人才管理存在的管理难题，提高了公司内部人员管理水平，提升了工作效率和幸福指数。数字化人才管理的推出，为社会主义现代化建设提供了新的机遇。

## 1.数字化人才管理在中国的落地及应用
截止到2021年，我国已有超过9亿人口，其中仅有3.3%的人口具有数字技能。也就是说，全国约90万至100万人口正在接受数字化教育。据国家统计局2021年发布的数据显示，2019年全国城镇新增劳动力由133.2万人增长到了151.9万人。其中，97.2%的新增劳动力为农民工，占总体新增劳动力的93.7%，占比逐年提高。其中，计算机相关专业毕业生也占到新增劳动力的近7成以上。另一方面，自2015年起，科技部、财政部联合启动“互联网+大数据助推实体经济发展”政策，大力支持创业创新、科技产业和基础设施建设，有力推动着数字化人才管理这一重点行业的发展。

2020年初，我国数字化人才管理领域正蓬勃发展，企业纷纷将数字化人才用作职业发展的重点和利益保障。目前，百度、腾讯、华为、阿里巴巴、京东等一线企业均对员工进行“无门槛”的“技能考核”，评选出最具竞争力的员工并聘用；智联招聘、猎聘网、拉钩网等平台也都推出了与数字化人才管理相匹配的招聘信息。

在医疗卫生领域，数字化人才管理被用来管理临床研究资源。在这个行业，希望能够更好地利用大数据、机器学习等技术，实现精准医疗诊断、预防、治疗。据国外媒体报道，中国内地已有超过一千家医院建立了基于疾病检测数据的“数字细胞计数器”。相信随着医疗领域的进一步数字化，数字化人才管理也会逐渐成为医疗界的一个热点方向。

此外，在互联网金融领域，数字化人才管理也得到越来越多的关注。据观察，2020年全国互联网贷款规模达到13.5万亿元，与2019年同期同比增长近三成。在金融行业，数字化人才管理的应用主要聚焦于风险控制、知识产权保护、风险识别、综合分析等领域。

## 2.数字化人才管理要素
数字化人才管理可以分为五个要素：人才发现、人才培养、人才引进、人才激活和人才流通。其中的“人才发现”包括人才需求调研、人才评价、人才筛选、人才招聘等工作。“人才培养”指的是通过人才发掘、人才引进、人才安置、人才激活等方式，将符合条件的人才引入企业内部来培养。“人才引进”即是指利用外部资源如人才、技术、资源等把优秀的人才引入企业。“人才激活”是指激活和培养企业现有的优秀人才，包括降低新人培训门槛、提升员工能力等工作。最后，“人才流通”则是指人才的传播渠道，包括企业内部的招聘、培训、晋升等机制。

## 3.数字化人才管理流程
数字化人才管理的流程通常可分为以下四步：

1. **人才需求调研：**包括企业收集、整理职位描述、公司招聘网站的关键词搜索、人力资源系统的人才库查询等工作。通过分析职位描述、内部招聘渠道、以及候选人的个人履历和履历模板，可以帮助企业确定人才需求。
2. **人才评价：**利用人力资源工具和云计算服务商提供的分析工具，企业可以获得候选人的数据质量、熟练程度、沟通表达能力、学习能力、工作经验、薪酬待遇等信息，从而做出筛选评估，确定最适合的人才。
3. **人才招聘：**向第三方招聘网站提交简历、准备面试、进行笔试等，通过网络面试，找出最优秀的人选。同时，还可以通过面向终端用户的社交媒体或移动APP招聘人才。
4. **人才激活：**企业可以根据其内部人才库，使用人事管理系统、云计算服务商提供的自动化人才分配机制，将合格的人才部署到各个岗位上，促使人才更加有效流动起来。此外，还可以邀请业界知名专家做顾问，分享企业的数字化人才管理经验。

## 4.数字化人才管理优势
数字化人才管理给企业带来的主要优势有：

1. **海量人才需求：**数字化人才管理通过人才需求筛选过程，可筛选出海量候选人，可以覆盖全国各地、各行业、各岗位，可以满足企业需要。对于企业来说，这一优势非常重要。
2. **高效、快捷的人才招聘：**数字化人才管理通过第三方招聘网站等网络平台直接招聘，可以快速准确地找到优秀的人才。其招聘过程涉及简历筛选、面试等环节，可以极大减少企业招聘时间。对于企业来说，这一优势也是很吸引人的。
3. **培养出更多领导者：**数字化人才管理有利于培养出能够领导企业的领导者。他们掌握先进技术、具有丰富经验、擅长领导才能、懂得创新，这些能力将为企业提供更加优质的决策支撑。
4. **为经济发展注入强劲动力：**数字化人才管理不仅可以将企业的效率和生产力提升至全新的高度，而且还为经济发展注入强劲动力。由于人才被迅速导入、运用，产业升级和产业结构的调整都将会成为经济发展的重要趋势。

## 5.数字化人才管理未来发展方向
数字化人才管理未来的发展方向有：

1. **协同式人才管理：**目前，协同式人才管理主要依赖于分散式组织，而数字化人才管理可以让人才管理系统变得更加集中、更加规范、更加智能。通过云计算和机器学习等技术，协同式人才管理可以降低人力成本，提高人才管理效率。
2. **产品赋能：**数字化人才管理还处在早期阶段，但是也已经进入了一个伴随其后的发展阶段。例如，智联招聘网就推出了“智慧招聘“产品，可以实现短视频直聘、面试评估、HR审核等功能。这种产品的出现意味着人才管理的整体效率和效益都会得到提升。
3. **工具优化：**数字化人才管理目前还处于起步阶段，但是已经具备了相当的实用性和能力。但是仍然有许多地方可以进行改进，例如，完善的测试工具、数据库优化、流水线自动化、职业孵化中心等等。

## 6.数字化人才管理常见问题解答
1. 为什么要采用“无门槛”的人才评价方式？
   - 采用“无门槛”的人才评价方式，是为了降低评价过程的复杂度，缩短评审周期，缩小了评估范围。因为候选人提交的简历是清晰明了的，整个过程不需要太过繁琐的评判标准，只需对候选人的技能、语言、态度、综合素质等情况进行简单、客观的评价即可。

2. “无门槛”的人才评价方式有哪些弊端？
   - 有的一条弊端是“无门槛”评价方式容易导致人才倾向性偏见，无法区别真正的优秀者和普通人。另外，针对不同的岗位和部门，可能会有不同的技术要求，导致某些技能无法进行标准化衡量，无法排除中间层的影响。

3. 在数字化人才管理领域，如何激活优秀人才？
   - 激活优秀人才的方法有很多种，包括但不限于内部推荐、招聘网站上的电子荣誉认证、中介、主管介绍等。其中，内部推荐的方法属于“邀请制”的方法，有利于激活熟练、对岗位有经验的人才；而招聘网站上的电子荣誉认证、中介的方式则更灵活、可控。
