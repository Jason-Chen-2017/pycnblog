# 1.简介
         
“Tableau”是一款商业智能工具，它提供了丰富的数据可视化功能、仪表板、工作流等，其界面简洁、操作简单易懂、功能强大，非常适合处理海量数据的分析和展示。其口号“直观的发现insights，真正做到像看见一样”更是令人钦佩。无论是公司内部的信息分享，还是上市公司产品运营报告、政府机关公共服务、社会热点事件监测、新闻舆情分析等各行各业都需要用到Tableau。本文将重点从数据分析（包括数据预处理、探索性数据分析、文本挖掘和机器学习）以及数据可视化三个方面，分别对Tableau在数据可视化领域提供的功能进行剖析。
# 2.基本概念与术语
## 数据集
数据集：由相关数据组成的集合，通常包括结构化数据（如数据库表格）或非结构化数据（如文档、电子邮件、微博、图像、音频、视频等）。
## 数据类型
数据类型：指数据的种类和属性特征，如定量型、定性型、标称型、序数型、率型、间隔型、序距型、标本型、结构化型、非结构化型等。
## 属性
属性：指数据中能够唯一标识某一个实体的某个事物的一个特征，如人名、地区、年龄、职业、销售额等。
## 变量
变量：指具有数值性质的数据项，也称为自变量或因变量，包括连续变量和离散变量。
### 连续变量
连续变量：指变量取值的范围为连续或具有连续数值特征的数据项，例如，身高、体重、价格、利润、交易量等。
### 离散变量
离散变量：指变量取值的范围仅限于一定个数或者离散集合的值的数据项，例如，性别、种族、教育程度、职业、投资行为等。
## 数据模型
数据模型：指数据的逻辑结构和数据之间关系的形式，即数据结构、数据之间的联系和模式。常见的数据模型如层次结构模型、网络模型、星形模型、维尔莫依模型、网状模型、混合模型等。
## 数据元素
数据元素：指构成数据集的数据单元，例如，数据集中每一行数据就是一个数据元组或记录，每一列数据就是一个数据元素。
## 数据挖掘
数据挖掘（英语：data mining）是指从大量数据中发现有价值信息并对其加以整理、分类、关联、分析和评估的一门学科。数据挖掘通常采用经验知识或计算机算法来识别、分类、关联、预测和描述数据之间的关系和联系。数据挖掘方法有数据挖掘算法、数据模型、数据挖掘技术、统计学、模式识别、图像分析等多个分支领域。数据挖掘主要应用于金融、保险、航空航天、电信、制造、医疗等领域。
## 数据可视化
数据可视化（Data visualization 或 DataViz）是一个过程，通过图表、网络图、地图、柱状图、折线图等方式，将复杂的数据转换为易于理解的视图，以支持用户决策。数据可视化可用于评估数据质量、发现模式及趋势、促进发现更多潜在的商业价值。数据可视化的目的有四个方面：
1. 提供直观了解：数据可视化的重要任务之一是帮助人们更快速、准确地理解数据并找出关键点。
2. 为分析结果提供支持：数据可视化可以帮助分析人员从中获取见解并生成有说服力的证据。
3. 支持业务决策：数据可视化可以帮助业务决策者决定下一步的策略、业务方向及相关执行策略。
4. 改善用户体验：数据可视化可以提升用户满意度和参与度，并且可以通过优化数据呈现方式来实现业务目标。
