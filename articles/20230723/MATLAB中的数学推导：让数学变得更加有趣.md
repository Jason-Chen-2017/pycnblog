# 1.简介
         
MATLAB（矩阵编程语言）是一种基于图形用户界面的数值计算环境和科学仿真工具箱。它为数据分析、科学计算和系统控制提供了强大的数学计算能力，在工程学、经济学、金融学、生物学等领域都有着广泛应用。MATLAB自身提供的数据结构以及高效的计算性能，使其在各个领域都得到了广泛的应用。
但MATLAB自带的图形化界面功能，对于初级使用者来说并不是那么容易理解。因此，作者将介绍一些基础知识和编程技巧，帮助初级MATLAB用户快速上手。文章中所提到的所有知识点均来源于个人亲自实践，并结合书籍和视频教程进行讲解，确保阅读此文章后，读者能够用实际例子快速学习并理解相关知识。本文适合对MATLAB有一定了解但想进一步提升自己的编程水平的读者阅读。
# 2.基础知识
## 2.1.符号运算符
MATLAB支持以下符号运算符:
+ （加）
- （减）
* （乘）
/ （除）
^ （幂）
% （取模）
&& （逻辑与）
|| （逻辑或）
! （逻辑非）
> （大于）
< （小于）
>= （大于等于）
<= （小于等于）
== （等于）
~= （不等于）
这里要注意的是，当出现运算符优先级的情况时，应遵循从左往右的原则。例如：x - y / z * a + b中的乘法运算符*优先级最高，因此先执行。但是有些时候如果遇到括号的使用，可能导致运算符优先级的改变。比如说，在表达式(a + b) * c / d中，/运算符优先级比*运算符优先级高，因此先执行。所以我们需要格外注意，不要过分依赖运算符优先级，否则会影响运算结果。
## 2.2.索引
MATLAB允许多维数组的索引访问，即可以通过指定数组元素的坐标或者子阵列的方式获取特定的数据。例如，`matrix(i,j)`可以用来获取第i行第j列的元素的值，`matrix(m:n,k)`可以用来获取第m到第n行第k列的所有元素构成的子阵列。
## 2.3.分隔符
MATLAB中的分隔符包括逗号、冒号和分号。逗号和冒号可以视为同义词，都表示一个值的分割符，而分号表示语句结束。例如，`A = [1,2;3,4]`表示创建了一个2行2列的矩阵，其元素由逗号分隔，每一行之间通过分号分隔。
## 2.4.输入输出命令
MATLAB的输入输出命令包括`fprintf()`、`fscanf()`、`disp()`、`print()`、`clear()`和`save()`等。其中，`fprintf()`用于向文件输出数据；`fscanf()`用于从文件输入数据；`disp()`用于显示变量的值；`print()`用于打印变量的值；`clear()`用于清除工作区中的变量；`save()`用于保存当前的工作空间。这些命令都是非常重要的，熟练掌握它们可以有效地使用MATLAB。
## 2.5.条件判断语句
MATLAB的条件判断语句包括`if`、`else if`、`else`和`switch-case`语句。`if`语句用于条件判断，如`if x > 0`，`else if x < 0`和`else`分别对应不同情况下的处理；`switch-case`语句用于多重选择，类似于其他语言中的`case`语句。
```matlab
switch num
    case 1
        disp("num is equal to 1")
    case 2
        disp("num is equal to 2")
    otherwise
        disp("num is not equal to 1 or 2")
end
```
## 2.6.循环语句
MATLAB的循环语句包括`for`、`while`和`break-continue`语句。`for`语句用于迭代执行固定次数的循环体，语法为`for var = start:step:end`，其中var为循环变量，start为起始值，step为步长，end为终止值；`while`语句用于条件循环，直到指定的条件满足才退出循环，语法为`while condition`。`break`和`continue`语句用于跳出或继续执行循环。
