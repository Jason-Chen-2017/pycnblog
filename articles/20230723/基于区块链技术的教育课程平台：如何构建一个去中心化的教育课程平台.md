# 1.简介
         
近年来，随着数字经济的迅速崛起，越来越多的人开始选择用数字的方式学习、工作、交流，并希望在这个过程中享受到更好的效率。通过学习、工作、交流等方式获得知识或技能，获取收益之后，也希望能够转化成投资或成为员工。在这样的背景下，区块链技术逐渐被越来越多的人们所关注。区块链是一个分布式数据库，它存储着价值，并由全体节点进行共识，确保数据安全、真实性、完整性。基于区块链的教育课程平台可以提供全新的教育方式，通过将学校、老师、学生的数据分散到不同的节点上，实现数据的隐私保护和不可篡改。同时，区块链上的教育课程还可以进行真正意义上的交易，支付费用给老师或者学生，使得课堂活动变得更加透明和可追溯。基于区BLOCKCHAIN的教育课程平台有很多优点，例如：

1. 数据隐私保护：将用户个人信息储存在区块链上可以确保用户的信息不被泄露。

2. 保证合法权威：将用户身份和数据都存放在区块链上，系统只会授权合法的权力者进行写入。

3. 更加透明和可追溯：所有数据都记录在区块链上，具有高度的可追溯性，学校、老师、学生都可以在区块链上看到自己的操作记录，也方便了老师和学生的日常管理。

4. 降低成本：区块链上的教育课程价格可以低至几毫甚至几百元一节课，这比传统的线下授课方式更加便宜，可以让更多的人参与到线下的教育活动中。

5. 提升门槛低：区块链上的教育课程不需要像传统的线下授课一样付出许多门槛，任何一个具备基本计算机知识的普通人都可以上手创建自己的区块链教育课程。

基于这些优点，作者认为区块链技术可以构建一个去中心化的教育课程平台。文章将从以下几个方面详细阐述如何构建区块链教育课程平台。
# 2.基本概念术语说明
## 2.1 什么是区块链？
区块链（Blockchain）是一种比特币的替代方案，其核心是由一系列的记录在一个分布式数据库（区块链）中的交易构成。区块链共同维护了一个公开的账本，记录了所有的交易历史。每一个节点都会保存整个区块链的数据，并且会对每个节点上的数据进行验证，确认没有任何篡改。一般来说，区块链有三种功能：
- 分布式记账：通过加密算法将交易记录编码为一串不可更改的数字串，记录在分布式数据库中。
- 点对点网络通信：不同节点之间建立长久的连接，完成数据传输，形成了一个点对点的网络。
- 智能合约：智能合约是一种协议，用于定义智能合约规则，控制链上数据执行的逻辑。通过智能合约可以自动执行复杂的操作，如代币的发行、数字货币的兑换等。

## 2.2 什么是Dapp(去中心化应用程序)？
DApp，即Decentralized Application的缩写，指利用区块链技术开发的应用。DAPP是通过区块链底层技术，搭建的分布式应用，用户可以通过区块链网络进行应用之间的互动。DAPP最初于2014年发布，之后随着区块链技术的飞速发展，其生态也在蓬勃发展。相对于传统的Web应用，DAPP具有更高的弹性和灵活性。DAPP的设计理念是无需下载安装就可以直接运行，而不需要信任第三方的服务器。DAPP的基础设施采用了区块链的原生协议，如IPFS、Ethereum等，极大的减少了成本，提供了更好的安全性。DAPP通过智能合约与区块链进行交互，实现价值的创造、流通和交换。目前已经出现了许多基于区块链的DAPP产品和服务，如Polkadot钱包、Uniswap交易所、Aave借贷协议等。

## 2.3 什么是ERC-721？
ERC-721是一个用来标准化非同质化代币的接口标准，它描述了如何创建一个非同质化代币的基本属性及其交易行为。ERC-721定义了一套NFT相关的标准，其中包括NFT的定义、创建、销毁、分配、认证等流程，以及相应的事件通知。这些标准定义了各类NFT标准的兼容性和互操作性，促进了NFT的发展和应用。

## 2.4 什么是ERC-20?
ERC-20是一个用来标准化代币的接口标准，它描述了如何创建一个代币的基本属性及其交易行为。ERC-20定义了一套代币相关的标准，包括代币名称、符号、精度、总量、账户余额及交易等流程，以及相应的事件通知。这些标准定义了各类代币标准的兼容性和互操作性，促进了代币的发展和应用。

## 2.5 什么是Solidity语言？
Solidity是一种高级编程语言，它是以EVM为虚拟机的区块链智能合约的开发语言。Solidity语言是以太坊的客户端编程语言之一，可以用于编写智能合约。

## 2.6 什么是IPFS？
IPFS，即InterPlanetary File System，是一个分散式文件存储系统。IPFS使用点对点对等技术，存储用户上传的文件，并把它们链接起来，组成一张庞大的网状文件系统，赋予用户超高的可用性。该系统支持目录结构，灵活的副本策略，允许多个设备同时访问同一个文件，并具有一定程度的数据冗余度，能够有效解决单点故障问题。除此之外，IPFS还具备一定的去中心化特性，即任何人都可以将数据存储到网络上，也可以自由地复制、修改、删除数据。

## 2.7 什么是Ontology智能合约平台？
Ontology是由国家超导区块链研究中心主导的一个开源社区项目，它是一个面向分布式应用的高性能分布式区块链技术框架。Ontology生态包括链层、侧链、应用层、SDK四个层次。链层提供底层区块链功能，侧链实现跨链互操作，应用层主要面向应用开发者，提供区块链服务，SDK提供了一系列开发工具，帮助开发者快速开发智能合约。目前，Ontology已推出POW、POS、DPoS三个共识机制的版本，并支持Ontology公链、侧链、智能合约开发工具箱。
