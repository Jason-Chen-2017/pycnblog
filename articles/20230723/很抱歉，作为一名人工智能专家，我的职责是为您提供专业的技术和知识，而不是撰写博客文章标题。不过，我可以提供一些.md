# 2.为什么需要神经网络？
随着互联网的普及和信息技术的飞速发展，在这个快速变化的世界里，计算机已经成为无处不在的工具。在过去的十几年里，互联网的每一次新潮流都伴随着对人工智能的重视。但是，对于很多普通人来说，构建并训练这样复杂的模型实在是不太容易。因此，许多科研机构都转向基于机器学习（Machine Learning）的解决方案。其中一种机器学习算法就是神经网络（Neural Networks）。

神经网络是由一个或多个输入、输出层组成，中间穿插着一个或多个隐藏层。输入层代表网络接收到的信息，隐藏层负责处理信息，输出层则根据处理结果生成相应的输出。隐藏层中的节点被称作神经元（Neurons），每个节点接受输入信号并产生输出信号，根据激活函数的不同，节点可能处于不同的状态。输出层中的节点则对应于网络最终的输出，一般采用恒等映射或softmax函数进行处理。

神经网络在图像识别、文字识别、对象识别、翻译、自动驾驶、强化学习等各个领域均取得了成功。它具有高度抽象、非线性、层次结构、权值共享、梯度下降算法、正则化方法等特征，能够有效地处理复杂、非结构化数据。目前，越来越多的人开始关注神经网络的研究。

# 3.有哪些入门级教程可供学习？
虽然说入门级教程的数量还算少，但也有很多适合初学者学习。以下列举几个初学者友好的入门级教程供参考：

1. [Backpropagation in Neural Networks](https://mattmazur.com/2015/03/17/a-step-by-step-backpropagation-example/)——本教程详细讲述了神经网络中的反向传播算法的工作原理；
2. [Deep Learning with Python](https://www.manning.com/books/deep-learning-with-python)——本书介绍了如何用Python进行深度学习；
3. [Learn Tensorflow for Deep Learning](https://github.com/lazyprogrammer/machine_learning_examples/blob/master/tensorflow_tutorial/README.md)——本教程介绍了如何使用TensorFlow进行深度学习；
4. [PyTorch Tutorials](http://pytorch.org/tutorials/)——本站提供了对PyTorch库的相关教程。
