# 2.R语言概览
## 2.1 R语言简介
R是一门开源的、自由软件的、交互式的、基于GNU通用公共许可证（GPL）授权条款发布的统计计算和统计图形显示的语言和环境。它是一个用于统计分析、绘图、数据操纵、模式识别、分类与回归、时间序列分析、因子分析、聚类分析、关联分析、生态分析等领域的集成环境。R语言具备强大的数学和统计函数库，能够帮助用户完成数据分析、绘图、数据操纵、模拟、统计检验等任务。R语言被广泛地应用于数据分析、金融、生物信息、地理信息等众多领域。其语法类似于S语言，是一种可选的免费软件，主要面向数据科学家、数值计算人员和统计学爱好者。
R语言支持多种类型的数据对象，包括矢量、数组、列表、矩阵、数据框、日期时间、因子、时间序列等。这些对象存储着数据集合和相关的信息，并提供丰富的运算和操作功能。除了提供自带的命令行界面外，R语言还提供IDE集成开发环境，如RStudio、R Commander等。
## 2.2 安装配置
### Windows系统安装
对于Windows系统，可以从CRAN官网下载最新的R安装包，点击下载链接后将压缩包文件解压至指定目录，进入解压后的目录，双击打开R-x.y.z.exe运行安装程序即可。默认安装路径为C:\Program Files\R\R-x.y.z，双击打开R.exe启动R编辑器。此外，也可安装Rtools进行R包的编译安装。
### Linux系统安装
对于Linux系统，一般可以通过系统软件管理工具进行安装，如apt、yum等。例如Ubuntu系统下可通过sudo apt install r-base或者sudo apt install r-base-dev命令安装。安装成功后可在终端输入R命令进入R编辑器。
### Mac OS X系统安装
对于Mac OS X系统，可以从CRAN官网下载最新版本的Rdmg文件，双击打开安装程序安装。安装成功后，可在终端或应用程序中搜索R命令，打开R编辑器。
### 配置国内源
如果下载速度过慢或者网络不稳定时，可以在R编辑器中设置国内镜像源。在R编辑器中输入以下命令：
```
install.packages("BiocManager") # 在R编辑器中安装BioConductor管理包
BiocManager::install(c("devtools","httr")) # 在R编辑器中安装devtools和httr两个包
options(repos=structure(c(CRAN="https://mirrors.tuna.tsinghua.edu.cn/CRAN/",
                             BioC="http://mirrors.ibiblio.org/bioc/",
                             CRANextra="https://mirrors.tuna.tsinghua.edu.cn/CRANextra/"))) # 设置镜像源
```
之后，在R编辑器中重新安装所需要的包时，会自动从国内镜像源下载安装包。如需设置成系统级环境变量，可以修改~/.Rprofile配置文件，添加如下两行命令：
```
options(repos=structure(c(CRAN="https://mirrors.tuna.tsinghua.edu.cn/CRAN/",
                             BioC="http://mirrors.ibiblio.org/bioc/",
                             CRANextra="https://mirrors.tuna.tsinghua.edu.cn/CRANextra/"))) # 设置镜像源
save.image() # 保存配置
```
