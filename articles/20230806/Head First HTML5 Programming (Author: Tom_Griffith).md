# 1.简介
         
2010年时HTML5已经被很多人所熟知了，然而对于那些刚接触HTML的人来说并不太容易理解。由于HTML5的新特性众多，这本书将会带领读者了解最新版HTML5的知识。作者<NAME>经验丰富的前端工程师为大家带来了一堂关于HTML5编程入门的课程。
           本书适合初级到中级前端开发人员阅读，其主要面向对象是具有一定web开发基础的技术爱好者或爱学习者。对Web开发人员来说，这是一个难得的快速学习HTML5技能的好资源。如果您有兴趣探索HTML5、CSS3、JavaScript等最新的Web开发技术，或者想建立自己的个人博客网站，或想与全栈工程师共同开发项目，那么《Head First HTML5 Programming》是您的不二之选。
          书籍结构
            在第1章“Introduction”中，作者Tom将从零开始带领读者熟悉HTML5的一些基本概念。在介绍完HTML5之后，他还会通过一个实例来帮助读者理解Web应用的基本构成及运行机制。
            在第2章“Basic Concepts and Terminology”，作者Tom将深入浅出地介绍HTML5中的一些核心概念，例如标签、属性、文档类型、命名空间、实体引用等。同时，本章也会介绍HTML5的历史演变，以及如何做好准备迎接HTML5的发展潮流。
            在第3章“Core Algorithms and Operations”中，作者Tom将详细介绍HTML5的核心算法及其实现方法，包括DOM API、Event API、Canvas API、Web Storage API、File System API等。另外，本章还将为读者提供一些学习资源，例如MDN Web Docs、W3Schools、Mozilla Developer Network(MDN)等网站。
            在第4章“Code Examples and Explanations”中，作者Tom将用具体的代码实例和图示来帮助读者更加深刻地理解HTML5的相关概念。本章的内容可以帮助读者理解如何通过HTML5来实现各种功能，比如表单验证、AJAX请求、本地存储、音视频播放、图形绘制、WebGL渲染、文件上传下载、Web Worker处理等等。
            在第5章“Future Development Trends and Challenges”中，作者Tom将介绍HTML5的未来发展趋势和局限性，以及它对前端开发人员的意义。最后，还会介绍HTML5在移动设备上面的应用。
            在第6章“Appendix – Common Questions & Answers”中，作者Tom将回答读者可能会遇到的一些常见问题，包括浏览器兼容性、文档编码方式、字符集设置、SEO优化等方面的问题。最后，还会给出一些参考资料，供读者进一步学习。
            本书并非一本完整的教程，仅仅是一本学习手册。通过本书的学习，读者能够掌握HTML5的基本语法、核心概念、核心算法、代码实例、运行机制、未来发展趋势、 SEO优化、移动应用等核心技术，从而提升自己对HTML5的认识、理解、运用能力，达到职业生涯的发展目标。
