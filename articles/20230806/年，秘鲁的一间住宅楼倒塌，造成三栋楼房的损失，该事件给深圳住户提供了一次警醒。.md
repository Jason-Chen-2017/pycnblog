# 1.简介
         
2008年9月3日晚上8点左右，一件未经检验的、普通居民无法摆脱的惊喜发生了。一间秘鲁家庭楼房坍塌，导致二、三层楼房同时倒塌，造成三栋楼房的损失。
         
         2008年9月3日凌晨2点左右，由秘鲁特勤局的人员接到警报，上午7时许，已有四名消防员赶到现场，抢救出受损房屋的一些瓷砖和玻璃，但由于情绪激动，部分消防员不愿进去救援。
         
         “我也不能随便进去，我怕中伤，更担心被发现。”李德强告诉记者。李德强表示，在楼下停放着两架特别钢制的战机，直升机正在飞临楼顶，李德强想拦截投弹的危险，但是不知为什么没有成功。他看了一眼身旁的另一位消防员，他的表情异常惊慌。“看起来他很害怕。”李德强说道。
         
         没过多久，现场火灾便停止了。随后，消防队员撤离现场，专注于其他事项。直到第二天清晨，消防队员才陆续返岗，在现场派出所进行全面调查，并迅速将所有相关人员控制。
         
         在调查过程中，李德强向警方提供了两张示威照片，其中一张照片显示出几十个受损房子的建筑物外观，摩擦力很大，他不确定是否全部都是因天气原因导致的塌陷。“这是一种特殊情况，并不是一般塌陷。”李德强说。
         
         现场还有两座建筑物被破坏，这两座楼房分别位于市中心和富士康厂区内。据了解，这两座楼房的损失是一起自然灾害造成的。
         
         当天中午，在深圳宝安区火车站，一辆载满的列车突然鸣笛停运，引起广泛关注。当时，市长潘石屹开车送医院，但在途中，另一辆车突然停运，他将大家转移到了附近的小区。“今天的防空预警很重要，要做好应对措施。”潘石屹说。
         
         中国共产党第八次代表大会刚结束不到一个月时间，曾主导制定了党纲要求农村集体化和公私合营，这些曾经出现过的体制矛盾在近期又重新激化，有民众质疑政策性地引导变革方向，政府部门却往往视而不见。
         
         在国家安全形势复杂化的背景下，中国政府需要进一步推动经济社会发展，提高科技创新水平，增强综合国力，保持经济持续健康运行。秘鲁事件虽然只是一次非常罕见的意外，但它提供了一个警示，在应对当代复杂变化中的艰难时刻，需要善用信息科技手段，尤其是在疫情防控等方面。
         
         # 2.背景介绍
         9月3日，苏里南大学一群学生携带一些物资和行装从新墨西哥逃往秘鲁，准备参加秘鲁各界的集体活动。他们是一群在校学习、工作的新生。计划中的活动包括听讲演讲，学习科普知识，并邀请中国留学生共同参与。
          
         9月6日，苏里南大学秘密举行了秘鲁首都——苏哈托举办的“秘鲁峰会”，汇聚了海外华人秘密团体和各地人士，希望借此吸引华侨到秘鲁参观、旅游。为了配合演讲，政府为大会安排了一个交通亭，承接旅客。
         
         9itat8日上午9时半，一辆蓝色轿车驶进苏哈托游泳馆，并停靠在游泳池边。此时，密集的人流冲击着游泳池周围的树木和草坪，其中不少人的头发、头盔、帽子已经烧焦。
          
         9月7日下午1时许，密集的人流在苏哈托游泳馆前的观礼台前分批走出，并进入苏哈托博物馆参观。这里是世界上最大的、完整保存的秘鲁博物馆。这天，来自加拿大的两名华裔女子把秘鲁博物馆介绍给了观众。
          
         9itat8日下午2时至3时，来自加拿大的两名华裔女子跟随观众在游泳池边等候。等到人群散去，她们将沿着游泳池远眺。她们带来的问询让人们感到惊讶，因为在秘鲁几乎找不到什么可以令人惊奇的景点，全国人民都还年轻，对古老的神话题材兴趣不大。
          
         9itat8日晚上，中央电视台综合频道播出了深圳实习医生叶修复主治李德强在发言中介绍秘鲁一级棒球队司令员。李德强表示，在这种特殊的时刻，不要再忽略了事态发展变化的严重性。“首先应该高度评估这个事件对中国和世界的影响，之后如何加强科技基础设施建设、推动改善秘鲁及世界的对话合作，将成为我们必须继续努力的课题。”
