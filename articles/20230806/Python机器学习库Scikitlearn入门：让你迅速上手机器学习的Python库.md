# 1.简介
         
Scikit-learn是一个著名的开源机器学习(ML)库，由Python开发者打造。它不仅提供了许多高级的机器学习模型，而且提供了大量的可复用代码组件。Scikit-learn适用于数据挖掘、统计分析、文本处理等领域。
作为一个入门级的教程，本文将带你快速地学习Scikit-learn的一些基础知识和技能。如此，你就可以应用到实际的项目中。下面，我将详细介绍Scikit-learn。
# 2.什么是机器学习
机器学习（英语：Machine Learning）是指计算机利用算法、统计模型及模式从数据中提取知识，并运用这些知识对数据进行预测或决策的一种能力。机器学习可以分为三种类型：监督学习、非监督学习和强化学习。其中，监督学习就是训练数据有一个已知的目标值，通过算法得出模型，再用该模型对新的数据进行预测；而非监督学习则不需要标签，只需要输入特征和描述数据之间的关系即可；而强化学习则是希望机器在环境中学习长期的奖赏与惩罚信号，通过对环境的建模和实时调整行为策略，实现自主学习。
# 3.什么是Scikit-learn
Scikit-learn，是一个开源的机器学习工具包，提供了大量的机器学习模型和算法，包括分类、回归、聚类、降维等。其设计理念是简单性和实用性。它的API很简单易懂，使用起来也很方便。
# 4.主要功能
Scikit-learn的主要功能如下：
- 数据集模块：加载数据集、划分数据集、归一化数据集等。
- 数据预处理模块：清洗、标准化、缺失值处理等。
- 模型选择与评估模块：模型的性能评价、超参数优化等。
- 分类、回归、聚类、降维等算法模块。

# 6.准备数据集
一般来说，机器学习的数据集应该具备如下几个特点：
- 有标签的数据：即数据集中每个样本都有一个对应的标签，用来表示它所属的类别、族群、类型等。
- 每个属性具有相同的数量级：也就是说，所有属性都采用同样的刻度或单位。
- 数据能够被较好的分成两部分：训练集和测试集。

首先，绘制变量之间的散点图：
