# 1.简介
         
 “职场智商税”（Intellectual Hazard）是指在工作中出现失误、疏忽或无法适应情况时，会给雇主造成财务损失或个人伤害的现象。相对于一般的税收来说，这个税收在工资收入中所占比例较低，但对雇主和员工都是一个沉重负担。事实上，这个“职场智商税”也被称为“知识产权违规罚款”，这种罚款往往随着罪名的深入而升级，有时甚至可数百万元。
          2019年新冠疫情期间，这一现象得到了国际媒体和网络的广泛关注。虽然中国各地的公共政策在鼓励企业加强自律意识、管理好“职场智商税”，但更多企业并没有认真履行相关法律义务。因此，就此次事件，许多政府部门纷纷出台相关政策支持企业减少“职场智商税”和薪酬优惠，使更多的企业有动力履行法律义务。
          在此背景下，本文将探讨“职场智商税”的定义、种类、原因、计算方法、罚金标准以及扣除方式等。同时，我们还会研究如何为雇员提供避免“职场智商税”带来的伤害。
         # 2. 相关概念术语
         ## 1. 什么是“职场智商税”？
        简单来说，“职场智商税”是指在工作中由于知识不熟练、能力有限、态度恶劣、工作态度不端，造成严重后果而向雇主承担的一种经济惩罚。也就是说，这种税收的主要受益者是雇主，而实际承担的经济惩罚则由雇员承担。
        
         “职场智商税”是针对企业员工因工作不力造成的人身和财产损失，从制度层面上来说是一项社会保险机制，通过严格的管控措施保障了雇员的合法权益。它是为了规范雇主行为，控制员工的收入水平，防止他们因言辞激烈引起争议，进一步提高雇员的职业道德水平。
         ## 2. “职场智商税”的种类？
        “职场智商税”通常分为知识产权违规罚款（IPR），电脑盗用罚款（DUI），勾结外国公司违规参与经营罚款（OSFI），合同侵权罚款（WII），滥用职权罚款（SPF），薄饼罚款（SB），扰乱生产秩序罚款（TPP），经营性贿赂罚款（OPR），违反出差旅费规定的罚款（WCC），超时加班罚款（OT），怠工罚款（BNI），加班时间不足罚款（LTN）等几类。
         ## 3. “职场智商税”的原因？
        “职场智商税”的产生主要有以下五个原因:
         * **过于刻板的工作态度**
            - 恐吓、威胁、粗话、脏话等脏话、骂人、殴打、打压等方式导致员工心理压力过大，可能影响工作效率。
         * **知识不足或技能匮乏**
            - 缺乏必要的知识或技能导致员工无法完成工作，或者会因此导致产品质量、人员素质和客户满意度下降，甚至造成经济损失。
         * **不懂得如何协调团队**
            - 员工之间存在不良关系，难以有效地完成工作，会拖累团队的整体工作效率。
         * **缺乏专业精神**
            - 员工不能接受或无视“职场礼仪”，不能帮助员工树立正确的价值观、团队合作精神、解决工作困难的方法论。
         * **态度恶劣**
            - 员工表现出疏远他人的刻薄、傲慢、无礼、自私等品质，引起团队成员之间的矛盾，造成团队效率低下。
         ## 4. “职场智商税”的计算方法？
        “职场智商税”的计算标准根据不同类型的“职场智商税”而定。比如，知识产权违规罚款（IPR）根据受损商品的金额、专利数量等计算；电脑盗用罚款（DUI）按照盗用电脑设备一天盗用的次数计算；勾结外国公司违规参与经营罚款（OSFI）按照勾结外国公司以创造额外利润作为目的经营、销售国家机密技术文件等行为计算。“职场智商税”的计算主要依赖法律法规的规定。
         ## 5. “职场智商税”的罚款标准？
        “职场智商税”的罚款标准因各项规定和不同类型的“职场智商税”而异。比如，IPR的罚款标准一般为5%-20%；DUI罚款标准为1000元以上/每件盗用电脑设备；OSFI罚款标准可高达数十亿元。
         ## 6. “职场智商税”的扣除方式？
        雇员缴纳了“职场智商税”，一般情况下需要每月将其累积的收入按一定比例扣除，但是不同类型的“职场智商税”的扣除比例及基数也不一样。比如，IPR的扣除比例最高为70%，基数为10万元以下（主要指超支的部分）；DUI罚款的扣除比例可以达到100%，基数为每日盗用电脑设备数；OSFI罚款的扣除比例可以高达50%，基数为勾结外国公司出售国家机密文件的总金额。
         # 3. 为何“职场智商税”如此贵，你应该怎么办？——“职场智商税”和“不服从力”
        “职场智商税”是一个非常沉重的税负，也是非常棘手的问题。在不服从领导、不执行命令、未按时交社保、未按时缴纳税金、非法盗用、违反安全条例等方面都会受到惩罚。要想有效减轻或杜绝“职场智商税”的发生，企业的管理者首先必须明确自己的责任，在法律法规的框架内明确自己的权利义务。只有做到充分的引导、鼓励、培训、监督、责任感和自我约束，才能切实防范“职场智商税”的蔓延，保障雇员的合法权益，营造一个尊重员工、尊重职业操守、服务于国家公共利益的良好氛围。
        根据美国《2021年中国联邦政府“职场智商税”指南》（以下简称《指南》）中所述，要想降低“职场智商税”，企业的管理者可以采取如下措施：
        （一）定期进行训练，增强员工的职业道德意识。企业需要建立职业技能档案，对员工进行职业鉴定，并定期组织相关培训课程，让员工了解并熟悉相关法律法规、业务规则、协同配合的流程。
        （二）充分听取员工意见，及时处理问题。雇员因过于刻板的工作态度、不懂得协调团队、缺乏专业精神等原因可能对自己的待遇产生负面的影响。企业需要及时对员工进行教育，开放沟通、保护员工隐私，及时收集反馈信息，对问题进行深入分析，妥善处理。
        （三）坚持透明原则，落实职场监督责任制度。企业需要建立系统完整的职场监督制度，包括评估情况、对抗不服从、举报违规行为，以及处理决议结果，对抗不诚信行为。
        （四）搭建平台，优化社区资源。企业需要搭建职场辅导中心、业务交流平台，提升员工的自我发展能力，减轻或杜绝“职场智商税”。社区资源尤其重要，企业可以借助互联网、社交媒体等工具，构建平台化的社区资源，支持员工解决职场困难，减轻或杜绝“职场智商税”的发生。
        （五）落实科学管理，尊重客观规律。企业需要通过科学管理方法，全面加强职场管理，尊重现代管理的基本原理、法治理念、人才培养、信息管理等方面规律。同时，必须遵守公众利益和社会公德，推行管理行为符合公众共鸣，避免官僚主义和垄断行为，保证工作效率。这样既能够有效防范“职场智商税”，又能提高企业的整体竞争力和凝聚力，确保企业利润的长久增长。
        当然，还有其他一些方法来降低“职场智商税”。例如，增加对非工作性伤害的惩罚，减少亲密接触，加强同级管理，杜绝不正当竞争。企业的领导层和管理层也应该认真落实相关法律法规，树立正确的职业形象，坚守职业道德底线，遵守公民基本权利，维护员工的合法权益，促进劳动者的社会责任感和个人发展。
