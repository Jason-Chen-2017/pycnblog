# 1.简介
         
1990年代，标准C++已经成为众多高级编程语言中的主流。但是，这个语言的一些功能并没有得到充分利用，并且仍然存在很多需要改进的地方。例如，指针、内存管理、异常处理等机制都还处于初级阶段。因此，为了更好地利用现代计算机技术，以及有效地开发健壮、可维护、安全且具有高度性能的程序，Brian Kernighan提出了" Effective Modern C++ "，这本书就是一份系统的、全面、权威的C++编程指南。下面，我们一起阅读一遍这本书，以期发现其中所蕴含的内容，并逐步构造成一个系统化、连贯、易读的学习材料。
         
         我们不妨将其主要内容分为以下几个部分进行阐述。
         ## 第一章 引言
         1. C++的历史与演变；
         2. C++的设计哲学
         3. 为什么要学习C++？
         
         ## 第二章 C++的基础知识
         1. 数据类型及变量声明；
         2. 基本输入/输出操作；
         3. 条件语句、循环语句和数组；
         4. 函数、类、对象、动态内存分配、指针及相关运算符；
         
         ## 第三章 模板编程
         1. 什么是模板编程；
         2. 模板参数推导、模板函数和类；
         3. 模板的特化和偏特化；
         
         ## 第四章 面向对象编程
         1. 类的定义、继承和派生；
         2. 访问控制成员（public、private、protected）；
         3. 抽象基类和接口；
         4. 对象模型和内存管理；
         
         ## 第五章 STL
         1. STL中重要容器的用法；
         2. STL的迭代器和算法；
         3. STL的排序算法及其适应场景；
         4. STL中的其他重要组件；
         
         ## 第六章 异常处理
         1. 什么是异常处理；
         2. try-catch块的语法；
         3. 在堆上分配内存；
         4. 用abort终止程序；
         5. 可移植性考虑；
         
         ## 第七章 并行编程
         1. 并行编程的概念；
         2. 多线程编程；
         3. OpenMP编程；
         4. CUDA编程；
         5. MPI编程；
         
         ## 第八章 输入/输出处理
         1. 文件输入/输出基础知识；
         2. 使用fstream、ifstream和ofstream操作文件；
         3. 操作二进制文件；
         4. 使用stringstream读取数据；
         
         ## 第九章 字符串、编码与国际化
         1. string类的介绍；
         2. string类的基本操作；
         3. 字符编码的概念；
         4. Unicode与编码转换；
         5. 不同国家文字的编码表；
         
         ## 第十章 正则表达式
         1. 正则表达式的概念；
         2. 使用正则表达式处理文本；
         3. POSIX正则表达式语法；
         4. GNU库支持的扩展正则表达式语法；
         
         ## 第十一章 单元测试
         1. 单元测试的概念及优点；
         2. Catch2框架介绍；
         3. 测试驱动开发（TDD）介绍；
         
         ## 第十二章 内存管理
         1. new和delete的基本机制；
         2. 手动管理内存；
         3. 智能指针的介绍；
         
         ## 第十三章 编译链接与库
         1. 编译过程与目标文件；
         2. 连接过程与可执行文件；
         3. 静态库、共享库的构建和使用；
         
         ## 第十四章 调试与 profilng
         1. Debugging的介绍；
         2. 使用gdb进行调试；
         3. Profiling的概念；
         4. gprof介绍；
         
         ## 第十五章 软件工程实践
         1. 代码风格、命名规范、注释编写；
         2. 项目管理方法、版本管理工具；
         3. 持续集成、自动测试、文档生成；
         4. 遇到问题如何快速解决；
