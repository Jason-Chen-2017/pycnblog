# 1.背景介绍


## 投资加密货币之前的历史回顾
随着人们对比特币、莱特币等虚拟数字货币的炒作热度逐渐减弱，并引起了越来越多的投资者的担忧。同时，也有越来越多的朋友、亲戚、同事或公司开始尝试跟风买入或者持有虚拟货币。在这样一个信息时代背景下，如何高效地获取加密货币资产成为了很多人的选择。投资加密货币这个领域并不是一件容易的事情，它涉及到了各种投资经验、交易平台、选股策略、交易策略、金融工具、思维习惯等方面的综合运用，是一项需要长期累积经验才能掌握的技能。下面简要介绍一下投资加密货币之前的历史进程。
### 互联网金融爆发前的20年
互联网金融这个词汇在中国语境里更多的被用来形容网络支付这种新型的金融模式。互联网支付和虚拟货币之间没有必然的联系，更不用说像支付宝、微信支付这些支付工具都是由商家提供给用户的账户中转而已。互联网金融在很短的时间内就成为非常火爆的行业，其背后的原因很多，但归根到底还是因为互联网技术的发展和普及。2007年初，互联网刚刚崭露头角的时候，很多国家都纷纷发动贸易战。当时全球范围内的经济衰退使得国际贸易顺差急剧减少。为了抵消这一损失，各国政府开始采取各种措施进行货币贬值。到了2009年的亚洲金融危机，也是这一阶段的结束标志。随后，越来越多的人开始认识到，世界上其实存在着一种特殊的货币——美元，即便是贬值的过程也只是缓慢而且不可预测的，但它却占据着重要的地位。这时候，巴西、印度尼西亚、菲律宾等其他国家的人民还在盲目跟风持有美元，这些人认为持有美元可以获得更高的收益率和方便，所以才会迅速抛售现有的美元资产。2010年爆发的金融危机又一次将全球经济拉向了新的低谷。然而，由于国内贸易顺差的恢复，互联网支付的兴起让国内市场又重新焕发生机。

### 技术驱动的金融革命
互联网金融的兴起带来了巨大的变革，但是它也带来了一些问题。首先，商家提供的资金转移手段过于简单粗暴，现实世界中的交易往往需要更复杂的手续。另外，由于商家信用水平参差不齐，人们往往不敢轻易相信他们所提供的资金，这给互联网支付的效率带来了极大的不确定性。由于缺乏一套健壮、可靠、公正的机制保障资金安全，互联网支付系统也面临着各种问题。

2011年末，美国的安娜·卡普兰尼推出了第一款基于区块链技术的虚拟货币比特币。这一创举引起了巨大的关注。由于比特币采用独一无二的密码学技术，任何拥有一定量的电子计算机和存储空间的个人都可以创建自己的比特币账户，并且可以随时把它转账给别人。通过密码学方式保证其匿名性和防伪能力，比特币成为人们最喜爱的数字货币。但是，由于技术门槛较高，有些普通人并不会自己掌握比特币的秘钥，这给实际应用造成了很大困难。因此，需要借助于第三方钱包（即交易所）来管理比特币的账户，并做好备份和保险工作。

此时的互联网金融已经完全进入了一个新的阶段，是指利用互联网技术提升交易效率，降低交易费用和风险，提升个人和企业的绩效，从而实现金融产品或服务的全面覆盖。

### 20年间的金融危机和互联网金融的大发展
互联网金融的崛起给人们的生活带来了翻天覆地的变化，改变了物价的上涨、消费的便利程度、婚姻的交流和消费的热潮，以及人们对金融的依赖程度。在2011年至2013年，全球加密货币的总体规模超过1万亿美元，人们开始意识到，加密货币将成为更加灵活的支付方式、可以任意转换和交易的媒介、具有抗风险能力的数字资产。但是，加密货币也面临着一些新的挑战，例如过度追求央行的稳定性，导致货币供应量减少，这会导致通胀的上升；随着计算机算力的增加，交易确认时间的缩短，可能会导致交易成本的增加，进一步恶化整个支付系统的效率。同时，由于金融机构日益膨胀，管理成本和个人信息保护问题也日益突出，这也会对加密货币的未来发展产生影响。

2013年央行启动降息政策，2014年GDP连续第五个季度增速放缓，2015年欧洲金融危机爆发。在此过程中，许多国家陷入经济困难，加密货币的应用受到了严重影响。

过去的一年里，互联网金融的发展变得非常迅速，但也遭遇了很多挑战。首先，随着国内加密货币市场的萎缩，用户的预期增长速度放缓，加之加密货币的上涨与波动幅度太大，使得加密货币交易所的吸引力大为减弱。虽然有一些新兴的数字货币交易所开始出现，但它们的盈利模式仍然十分不合理，甚至还有人宣称它们存在严重的风险。其次，对于加密货币的监管问题，越来越多的国家提出了更多要求，以确保国家对其资产的控制权，比如GDPR的规定就是一项十分有效的保障。最后，在非洲等地区，由于对智能手机和电子支付的依赖程度增强，越来越多的人开始存入加密货币，这也导致加密货币价格飞涨。
