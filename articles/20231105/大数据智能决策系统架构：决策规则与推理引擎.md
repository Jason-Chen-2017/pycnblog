# 1.背景介绍


## 概述
随着互联网、移动互联网等新兴技术的发展，人们越来越依赖于计算机网络为他们提供各种各样的信息服务，从而促进了社会生产力的发展。近几年，基于大数据的决策支持技术得到快速发展，特别是在制造领域，据估计，每年的全球产量中约有80%-90%都是由大数据驱动的。例如，最近发布的数据显示，美国每年的制造业产值占GDP的比重将达到7%左右，其中46%的产出来自于电子制造业。在此背景下，如何通过大数据进行智能决策支持至关重要。

对于传统企业而言，其决策支持往往基于经验、直觉以及业务知识等软技能。但是随着信息技术的发展及其带来的便利性，特别是云计算、移动互联网的普及以及海量数据的生成，新的大数据技术也在提升人们对大数据的理解、分析能力及决策效率方面发挥着越来越重要的作用。因此，企业需要在大数据智能决策系统的设计上更加注重决策引擎这一关键模块，这项工作可以归结为两个部分：一是决策规则的建立，二是决策引擎的构建与部署。由于决策规则与决策引擎之间的复杂关系，目前市场上关于这两者构建的综合性技术仍然缺乏。本文试图对这一问题进行初步的探索，并总结目前国内外研究人员在这两个领域的最新成果，以期为企业提供参考。

## 现状与发展
### 数据获取与处理
19世纪90年代，欧洲出现了第一个大规模数据收集项目——费城山地自然历史博物馆（NEL）。从那时起，当时德国莱顿大学地理系的科学家们开始收集东方世界的珍贵遗址、地貌、气候、天体活动及自然资源数据。后来，这些数据被用来开发世界自然遗产数据库。同年，著名的统计学家哈罗德·布鲁姆·萨拉热斯曾经提出，“如果想要了解这个星球上的事情，就要采集大量的数据”。

20世纪初，随着互联网、移动互联网的蓬勃发展，更多的人能够在短时间内获得海量数据，如社交媒体上的海量评论、用户行为数据、地理位置信息、图像、音频、视频等。这些数据对于企业的决策支持具有不可替代的价值。

2012年，斯坦福大学的吴恩达教授在Coursera上开设了一个课程《大数据》（Introduction to Big Data），强调“任何复杂的应用都离不开大数据”“最重要的不是拥有多少数据，而是能够从数据中提取有用的信息”，同时提供了几个关键词：数据质量、数据存储、数据分析、数据挖掘等。

截止今日，随着人工智能、机器学习、大数据技术的不断进步，企业如何通过大数据进行决策支持已经成为一个巨大的课题。很多行业如制造、电信、金融等正在迅速采用智能化管理手段，也正逐渐走向智能管理的高峰。但是，如何把大数据技术应用到企业决策系统中，还存在很大的挑战。

### 规则定义与推理
规则定义即确定企业在面临某种特定场景下的决策逻辑和指导准则。企业根据不同的场景，设置不同的决策规则。规则定义属于静态的过程，主要基于专家的判定和启发，往往难以跟踪数据驱动的变化。而且，规则的有效性不能够反映真实情况，会导致决策结果偏离实际。

2010年，Google公司推出了一款名为PageRank的搜索引擎算法，利用链接结构建立起了一个超级链接图，并对每个网页赋予一个评分，表示该网页在搜索引擎中所占的重要程度。可见，算法是一种动态的、基于网络流量的规则推理方法。

对于规则定义方法的更进一步研究，也存在诸多研究方向。例如，面向推荐系统的规则定义方法，将用户与商品、上下文相关联，根据这些关联规则来推荐产品；人工智能技术也可以用于规则定义，通过反馈循环的方法训练模型自动识别新的规则并更新规则库。

推理引擎即基于规则库的决策引擎。它负责按照预先定义好的规则集，在实时的决策过程中，依据输入的数据，进行一系列的计算和推理，最终输出相应的决策结果。推理引擎的工作流程一般包括三个步骤：数据收集、数据预处理、规则推理。规则推理一般通过已有的规则或者模式匹配的方式来实现。推理引擎有两种形式，一种是专门针对某个领域或业务场景的推理引擎，另一种是通用推理引擎。

2009年，IBM的研究人员提出了一种基于规则的预测模型，叫做“IBM Common Decision Framework (CDF)”。CDF的基本思想是基于已有的规则和模式，将数据映射到相应的规则上，然后根据规则集合进行推理。2013年，Microsoft也推出了一套类似的预测模型，叫做“Expression Parser”。表达式解析器由一组内置函数和运算符构成，可以对文本中的语句进行解析和执行。表达式解析器可以生成目标语言代码，比如JavaScript、C#、Java等，这样就可以直接运行。

2011年，NASA的科学家团队提出了一种新的决策引擎——预测引擎（Predictive Engine）。预测引擎的基本思想是从历史记录中学习，分析数据之间的联系，并且建立预测模型，能够对未知事件做出决策。在2014年，谷歌推出了一个基于专家系统的决策引擎平台——谷歌的AppMaker，使得企业可以轻松的创建自己的智能决策系统。

## 发展路径
企业在决策支持中使用大数据智能决策系统，可以大幅度提升决策效率、降低决策错误率，减少人为因素影响。但同时，为了满足企业的个性化需求、提升决策的透明度和客观性，决策系统需要更加复杂的架构设计、智能决策算法的研发，以及数据的精确获取、高效查询等。

2015年，华为主任丁康旭受邀参加了美国太平洋地区、日本、台湾、中国等不同国家的大型会议，发表演讲。他说，华为正在研发一整套智能决策解决方案，其中包括如何将数据智能应用到企业的决策流程中、如何将决策引擎与第三方集成、如何保障决策系统的安全性、如何有效地运用数据分析工具和方法、以及如何实现整个系统的可靠运营等。他认为，未来智能决策系统将走向更加深远的方向，变得越来越智能，能够响应快速增长的数据和日益复杂的决策要求。

2017年底，特朗普政府发言人马克・安德森在推特上表示，企业将成为机器智能的第一批用户，因为大数据是连接所有人的信息，数据智能为企业带来了巨大的商机。特朗普称，“数据是一切的基础。”他呼吁各国政府与企业共同努力，加强数据治理、智能管理等工作，共同为全球的数据智能生态圈建设提供技术支撑。

总之，如何构建大数据智能决策系统，是一个综合性的课题。目前，业界涌现出的大数据智能决策系统的研究成果，既包含计算机技术的发展，又包含了工业、经济、法律、生物学、心理学等多个领域的科学家们的创新理念和实践。只有深入探讨这些理论和技术，才能找到一条合理、有效的路线，助力企业实现更加智能、更加高效的决策。
