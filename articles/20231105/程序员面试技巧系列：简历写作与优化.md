# 1.背景介绍



在工作中，你可能会被招聘为软件工程师或系统架构师。在面试的时候，简历就是你的佼佼者。不管是求职者还是雇主，面试官都会首先判断你的优点与缺点，然后再用一些评估手段考察你的技能是否达到要求。这时，你需要写一份符合标准的简历，展示自己的能力、经验、项目经验等，让对方对你的了解更充分、把握更准确。

　　简历的作用主要有以下几点：第一，它可以吸引人才；第二，通过面试筛选，你会得到优秀人才的青睐；第三，它也作为你个人信息的一个展示，帮助别人了解你，让面试官对你有个更好的了解。

但是，如果你只是按部就班地按照套路出牌，写一份简单的材料，往往容易被面试官看到你能力不足而怠惰跳槽，而且你也无法广泛展现你的经验。所以，你要思考一下如何写一份高质量、真实的简历，在找工作、面试、升职上更有利可图。下面，我将分享一些写作技巧，帮你创造一份专业、客观、精准的简历。

　　1.提前规划——以应聘者的需求为导向

如何才能写好一份简历呢？通常来说，一个优秀的简历应该具备以下特点：

1）突出自己的亮点：你的经历、学到的知识、技能、项目都要体现出来。
2）详细陈述自己的职业生涯：包括工作年限、从事职业、管理经验、团队协调经验等。
3）体现自己擅长的领域：对工作有独特的见解和理解。
4）具有吸引力：展示自己的性格、兴趣爱好、证书等。
5）深入浅出：突出自己的能力而不是堆砌。

但无论什么样的简历，都不能让人一目十行。面试官一般只会看前五页的内容。因此，最好提前三四天制定一个目标，包括的内容，还有任何需要注意的细节，并给简历编排进度表。

　　2.突出亮点——不仅关注自己的能力，还要突出自己的长处和短处

简历的核心内容是展示你的能力和经历。首先，你需要突出自己的个人价值。你可以分享一下你在某项事业上的成功与失败经验，与其他人的差距。你也可以描述自己突出的获奖记录、领导或客户授意的推荐信，这些都会增强你的形象。

其次，你需要展示自己的综合能力。除了你的专业知识外，你还要展现你的学习能力、沟通能力、协作能力、动手能力。你可以用项目实践来传达你的技能。比如，你可以举例出你最近完成的某个项目，或者说服别人如何改善效率，等等。

最后，你还要突出你的个人风格及品味。你要有独特的表达方式，且要与受邀面试者互动。你可以多方面的展示，包括时尚美容、旅游兴趣、文化气息、音乐喜好等。你可以用照片、视频等证明自己的承诺。

　　3.打造个人品牌——塑造简历中的形象

有些候选人喜欢用自己的名字来形容自己，因此，为了引起注意，你需要确保你的姓名能够代表你的个性、经历、理念。例如，你就可以采用昵称或头衔来代替自己的名字，让别人对你有更亲切的认识。

另外，还可以利用网络资源制作个人logo，并添加到你的简历中。这样做有助于增加你的知名度。

　　4.语言清晰——简洁、直接、生动

简历是对候选人的一种考察，也是他们了解你的机会。因此，你的简历一定要简洁、直接、生动。你需要精心设计每一页的结构，并尽量保持一致性。

你还需要注重语法和语气，仿照模板进行写作。这么做既可以减少错误，又可以帮助读者快速理解你的能力。

　　5.图文并茂——让你的简历更像自然语言

如果说“图文”比“文字”更能传达信息的话，那么简历就是“图文”。一般来说，使用图片来展示你的工作经历、项目经验、学科水平等，会增强你的直观感受，尤其适合那些笔者认为特别难于理解的事情。

此外，还可以使用超链接、列表、颜色、字体等元素来增加美感。

　　6.避免陷阱——注意你的细节和措辞

简历中不要出现任何错别字、缩写、数字和算术符号，否则可能会引起误会或遗漏。你也不应该使用奇怪的专业词汇，否则可能会令人疑惑。同时，你也要注意措辞习惯和语法，尤其是在提及专业术语时。
