# 1.背景介绍


光刻胶（Gold Phosphide）是一种高分子聚合剂，在应用于现代电子设备、显示器、摄像机等领域时，可提高其尺寸、厚度和电容性，并提供更好的耐磨性。由于光刻胶具有良好的形态稳定性，制作出来的产品质量较好，所以广泛用于电子设备、显示器、摄像机等领域。如今，光刻胶已经成为主流材料，各种芯片制造商都在使用光刻胶作为PCB板材、晶圆元件和集成电路的主要材料。在电子系统设计中，光刻胶还能够提供更高的集成度和功耗效率。但光刻胶制作过程往往比较复杂，光刻胶机组制备及精益求精是所有工程师必须具备的技能之一。本文将从以下几个方面对光刻胶的制造流程进行介绍：

1) 光源制备：光刻胶的制造过程需要用到离子激发裂变法或扩散膜法制备光刻胶。离子激发裂变法可以制备非常薄层级的光刻胶，但费用昂贵；而扩散膜法则需要用液体稀释光刻胶后再在稀释后的液体上增加一个固态模具，从而制备出合格的光刻胶。
2) 激光刻胶机：光刻胶机是最基础的制造设备。它主要由光源、显像装置、光刻胶切割机组、显影剂池、光刻胶沉积机组等配套设备组成。光源一般采用聚焦剂或离子束激发剂，它们能向目标物体照射激光，将其转换为电子流，传导至显像装置。此外，聚焦剂还可以检测光源的准确度。显像装置接收来自光源的激光信号，并将其转化为电子信号，输出至光刻胶切割机组。光刻胶切割机组经过刮削、切割和光刻，将电子流切碎为单个小粒子，并转换为固态硬膜形态，送入光刻胶沉积机组。最后，光刻胶沉积机组将这些固态硬膜堆叠起来，形成完整的光刻胶样品。
3) 精确控制：精确控制是光刻胶制作中重要的一环。光刻胶的层级越深，光刻胶的厚度也就越大。为了保证光刻胶的清晰度，我们必须把每一次调整的参数精确调节好，包括照射强度、光的位置、光刻胶厚度等。同时，要注意避免光刻胶浸泡在非特异性介质或者吸收其他气体的影响。
4) 表面处理：光刻胶在经历了极大的分散后，还需要进行表面处理。表面处理的方法有不同，有的只需要擦去一些氧化剂，有的则需要进行热处理。而对于不同的材料，不同的处理方法也会产生不同效果。因此，光刻胶制造人员必须非常细心地观察光刻胶的表面处理过程，从而选择正确的处理方式。
