# 2.核心概念与联系

## 2.1 远程工作术语

- VNC (Virtual Network Computing)虚拟网络计算，一种远程桌面的技术。
- SSH (Secure Shell)安全外壳协议，一种加密的网络传输协议。
- TeamViewer 一个专业的远程协作软件，可以实现不同平台间的远程控制。
- WebEx 会议系统是一个远程会议的平台。
- Slack 远程协作软件，提供团队沟通、计划和协作。
- Zoom 智慧型会议解决方案，是WebEx、Teams和其他会议服务的基础。
- GoToMeeting 会议服务，是一款优秀的远程会议软件。
- ScreenShare 屏幕共享软件，可以让远程用户看到计算机屏幕的内容。
- RemoteDesktopControl 远程控制软件，可以让远程用户操控自己的电脑。
- AnyDesk 提供在线远程会议功能的远程协作软件。
- DropBox 云端存储平台，可以实现文件的远程同步。
- GitHub 在线托管平台，可以实现代码的版本管理和远程协作。

## 3.1 TeamViewer 远程会议工具

TeamViewer 是一款非常著名的远程协作软件，具有很强的远程控制能力，可用于实现多设备之间的远程控制。它提供了一系列功能，包括音频会议、视频会议、屏幕共享、文件传输、远程桌面、电子白板、摄像头和麦克风。

安装 TeamViewer 可以根据不同的操作系统和硬件平台进行下载安装，并根据提示输入登录凭证即可登陆远程会议界面。

### 3.1.1 操作方式

打开 TeamViewer 客户端软件，点击右上角的 “Create a new session” 来创建会议。


输入会议 ID 和密码，确认后，您就可以开始与他人的远程会议。


除了远程控制、屏幕共享等基本功能外，TeamViewer 还提供了一个叫做 Instant Meetings 的高级功能，该功能可以帮助您实现快速、精准的远程会议。Instant Meetings 是基于对讲机技术的实时视频和语音互动机制，可自动匹配和连接多个参会者，为远程会议节奏和效果提升提供无限可能。

要启用 Instant Meetings ，您只需打开您的设置，然后按下 Instant Meetings 中的 “Turn on instant meetings” 按钮即可。如果您不希望被别人看到您正在使用 TeamViewer 时显示的任何内容，请不要勾选 “Share content” 复选框。


### 3.1.2 设置权限

默认情况下，所有与您处于同一局域网的设备都可以看到您的会议，这意味着您的隐私可能会泄露。如果希望您的设备只能参加特定会议或仅允许特定人员参加会议，可以编辑配置文件，并加入 IP 黑名单或白名单，这样只有指定的 IP 或用户名才可以加入会议。


### 3.1.3 直播会议

TeamViewer 的另一个高级功能就是直播会议。直播会议是指参与者以实时的方式观看远程桌面、共享屏幕和共享声音，与他们现场聊天，随时交流，随时分享想法。你可以在会议过程中开启或者关闭直播模式，也可以选择只向特定用户播放声音。


### 3.1.4 快捷键

TeamViewer 有许多快捷键，可以让您方便地控制远程会议。例如，您可以使用 “Ctrl + D” 来切换麦克风开关；“Ctrl + R” 来切换摄像头开关；“Alt + I” 呼出屏幕共享窗口；“F11” 切换全屏模式等等。


## 3.2 GoToMeeting 远程会议工具

GoToMeeting 是一款用于远程会议的应用软件。它的功能和稳定性都十分出色。它具有会议控制功能、可视化桌面共享、语音和视频会议等。它可以通过会议 URL 或手机设备号码进行远程访问。GoToMeeting 可以与第三方应用程序集成，如 Google Hangout，Zoom，Skype 等。

### 3.2.1 安装与配置

在安装 GoToMeeting 之前，请确保您的操作系统和浏览器满足其要求。由于 GoToMeeting 支持许多浏览器，因此建议您安装支持的浏览器。

进入 https://secure.gotomeeting.com/ 注册一个免费的 GoToMeeting 用户账号。登录后，创建一个新的会议或加入现有的会议。


如果您第一次尝试使用 GoToMeeting，您会看到如下画面。


点击 "Sign up for GoToMeeting" 链接，进入注册页面。填写必要的信息，然后点击 "Create an account" 创建一个新的账户。


创建成功后，您可以登录 GoToMeeting 并创建新的会议。

### 3.2.2 使用远程桌面

您可以在 GoToMeeting 中启动远程桌面功能，以便与其他用户共享屏幕。要启动远程桌面，请在主界面上点击 "Show desktop sharing options"。


点击 “Start screen sharing” 以共享你的整个屏幕。


随后，您可以选择将共享屏幕发送给您个人使用的设备，或者使用连接到 GoToMeeting 的其他设备。

### 3.2.3 视频会议

GoToMeeting 提供了一个可视化的视频会议界面。要进入视频会议模式，请点击 “Video conference” 按钮。


在视频会议模式下，您可以选择调出摄像头或麦克风、共享屏幕或文档、静音自己，或者邀请其他参会者参加会议。


### 3.2.4 共享文档

您可以在 GoToMeeting 中共享文档。要共享文档，请点击 “Shared Documents” 按钮。


在 “Document sharing” 页面，您可以选择添加文件、打开、关闭权限、锁定文档等。


### 3.2.5 聊天功能

GoToMeeting 支持实时聊天功能，您可以在会议期间与其他参会者聊天。要打开聊天功能，请点击 “Chat with people” 按钮。


您可以向参会者发送信息、查看历史记录、禁言、拉黑参会者等。

### 3.2.6 移动应用

GoToMeeting 提供了一套完整的移动应用，包括 iOS、Android 和 Windows Phone 版本，可让您随时随地参加您的会议。


## 3.3 Microsoft Teams 远程会议工具

Microsoft Teams 是微软推出的一款基于聊天的通信工具。它为团队提供了高度灵活的通信解决方案，包括音频/视频会议、共享文件、呼叫转接、会议室管理等。

### 3.3.1 安装与配置

Microsoft Teams 可以在浏览器、Windows、Mac OS 和 Linux 上运行。首先，访问 https://teams.microsoft.com/downloads 下载安装包。然后，双击下载的文件运行安装程序。


安装程序会提示您选择 Microsoft Teams 需要访问的语言。您可以根据需要更改语言设置。安装完毕后，启动 Microsoft Teams 客户端软件。


登录 Microsoft Teams 后，您可以开始使用各种功能。其中，最重要的是参与微软 Teams 上的音频/视频会议。


### 3.3.2 创建会议

要创建一个新会议，请单击左侧导航栏中的 “Meetings”。点击 “New meeting” 按钮。


选择会议类型，如即时通讯、电话会议或网络研讨会。设置会议名称、日期和时间。选择参加会议的参与者。点击 “Create” 按钮保存会议。


### 3.3.3 参与会议

当会议开始时，您可以通过点击会议开始前的倒计时来预约参加会议。


点击 “Join from your browser” 按钮或扫描会议邀请，以加入会议。


加入会议后，您可以在共享屏幕、视频会议、文件共享、聊天、呼叫转接、会议室管理等方面与其他参与者进行交流。


### 3.3.4 共享屏幕

您可以在 Microsoft Teams 中共享您的屏幕。要共享屏幕，请单击屏幕顶部的 “Participants” 按钮，然后单击“Share your screen” 按钮。


选择屏幕分享的源（如整个屏幕、单独窗口或应用程序）。选择是否希望共享屏幕的内容。选择屏幕共享的方向（如垂直或水平）、分辨率和比例。点击 “Start sharing” 按钮开始屏幕分享。


### 3.3.5 视频会议

Microsoft Teams 内置了视频会议功能。要加入视频会议，请单击 “Calls” 按钮。


选择加入会议的类型（如音频、视频或屏幕共享）。选择加入会议的源（如会议邀请、链接或电话）。点击 “Join” 按钮加入会议。


### 3.3.6 文件共享

您可以在 Microsoft Teams 中共享文件。要共享文件，请单击屏幕顶部的 “Files” 按钮。


上传文件或选择本地文件。为文件设置描述信息。点击 “Upload” 按钮开始上传。


### 3.3.7 呼叫转接

您可以在 Microsoft Teams 中转接呼叫。要转接呼叫，请单击 “Calls” 按钮，然后单击待转接的呼叫。


在转接界面中，选择转接目标。点击 “Transfer” 按钮。


### 3.3.8 会议室管理

Microsoft Teams 为每个会议室都提供了会议室管理工具。您可以查看当前会议室状态、占用的座位数、邀请他人进入房间等。要访问会议室管理，请单击屏幕左侧导航栏中的 “Rooms”。
