# 1.背景介绍


# 最近几年随着高端芯片领域的爆炸式增长，高性能处理器的需求也越来越迫切，这就要求芯片设计者们不断提升自己的设计技艺、处理速度等方面，以满足消费电子、航空航天、科学仪器等领域日益增长的应用需求。特别是在较新的光刻胶技术出现后，低成本、快速、可靠的金属材料再加上发光工艺的革新，使得传统的塑料、布、铝等材料不可替代。基于此背景，光刻胶的应用范围越来越广，光刻胶制品也逐渐成为高端芯片制造中的必备环节。光刻胶的制造方法可以分为：连续型、离散型、纳米颗粒型等3种类型，而不同类型的光刻胶都需要不同的处理方法才能形成芯片表面光滑、干净的效果。化学机械抛光就是一种常用的光刻胶制品制作方式，它是将普通的低纤维材料经过转盘、抛掷、晾干等施加特殊的外界力作用后形成具有一定电导率的微孔结构。
# 2.核心概念与联系
## 2.1 固体电介质的控制和特性
在化学机械抛光过程中，对固体电介质的控制主要通过改变其介电性和电导率来实现。固体电介质通常指除绝缘介质(例如气液、流体、空气)之外的介质，其包括空气中的氧气、油气、液态氮气等，它们是机械强度的源泉，也是影响发生器件特性的重要因素。不同类型的固体电介质之间存在不同的蕴涵量和分泌物分布，如酸性固体电介质蕴涵量高，容易受到重击，不利于抗压，而易燃固体电介质蕴涵量低，对抗力较弱，故在化学机械抛光中常用冷却后的固体电介质。电介质的电导率表示介质中电流量与电场力大小的比值。不同电介质所对应的电导率一般是固定的。

## 2.2 反应堆及其工作原理
在化学机械抛光中，当固体电介质遇到复杂而变化多样的化学反应时，会产生一个由不同反应物组成的反应堆，这些反应物由各种不同的化学物质组成，其中有些物质即使没有被激活，也能在反应堆中转移和增殖，称为化学支援反应。由于不同种类的化学反应物在不同温度下都有不同的物理状态，所以反应堆内部有很多层厚度不同的沉积液，这些沉积液能够结合成一个气液相互作用下的晶状结构，并生成出晶体，所以反应堆又称为固体电介质激发的机械结构。

## 2.3 晶体、二极管和钛-晶体管
化学机械抛光的目的就是通过反复的电扰动生成晶体，并将晶体制作为固体电介质激发的机械结构。为了更好地理解化学机械抛光的过程，有必要了解一些相关的物理基础知识。

1、晶体结构
晶体由晶格单元或者称为原子团的点阵排列组合而成，晶体的特性可以通过晶体张数和晶体半径的关系来描述。晶体张数和晶体半径的关系：α = (N*v)/r²，其中N为晶体的原子数目，v为晶体的体积，r为晶体的圆柱状半径，α为晶体的拓扑角度。晶体的晶格尺寸又称为晶胞大小，通常是由两个不同的原子核和一对带隙原子的距离来确定。晶体张数的增加可以增加晶体的晶格尺寸，也可以减少晶体的缺陷，但同时增加了制造成本。

2、二极管
二极管(Bipolar Junction Transistor，BJT)是一种重要的集成电路器件。它由两个腔室、三极管和电阻构成，电路输出电压由一个控制腔室中的三极管控制，根据二极管特性调整三极管中的漏栅电流从而实现对电流的调控。由二极管调控电流的方法可以分为表针法和电容法。表针法是通过使三极管的缺陷电流上升或下降的电压来调整二极管的偏置电压，使得二极管的电流发生偏转。电容法则是通过在输出端接入一个电容，使得电路输出端的电压跟随控制腔室中的三极管的偏置电压变化。

3、钛-晶体管
钛-晶体管(Ternary Silicon Multijunction Field-Effect Transistor, TSi/MFT)是一种最早的晶体管。它由三个电极(PT、PO和NC)、三个射极(L、W和V)、一个集电极和一个漏极电路(SD)构成，结构上类似于一维CMOS晶体管。与二极管和三极管不同的是，TSi/MFT除了三种电极外，还有一个中间缺口，能够容纳一定的电容和磁场。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 操作步骤
1. 实行光刻胶手术，去除杂质、修整、打胶脂。
2. 对外界刮蹭、打扰以及环境照射进行均匀排布，避免发生光刻旋转。
3. 将预备好的带有电离层的化学介质吹至光刻胶板上。
4. 用丝线做出直线弯曲状光谱条带，通过光刻胶板将光刻胶注入刮蹭、打扰以及环境照射的排布。
5. 使用3D打印机、胶水和溶剂打磨光刻胶板上的反应区段，精确匹配电离层的大小，并使其更加平滑、柔顺。
6. 使用浓缩液、氯气稀释液和小量化学溶剂注射光刻胶，使其完全氧化，并将发生层的层数保持在6~8层。
7. 在介质吹到板上并充分烘干之后，继续等待反应区段合拢，然后在中间位置放置刮蹭去除杂质的开口，切割光刻胶至一定大小，并扎牢。
8. 在靠近相反面的开口另起一道切割线，将光刻胶切割成4~12个单位长度的块。
9. 将打磨好的光刻胶包裹在滚珠袋中，箱盖密封，放入化学缓冲液中浸泡24小时，待块充分熟透才移至生产设备中。

## 3.2 数学模型公式详解
化学机械抛光的主要特征是一系列的随机晶体生成，因此，光刻胶板的设计和数量、光刻胶的质量、光刻胶的排列方式、生长方向、光刻胶的切割、压制等参数都直接影响最终的晶体生成。
