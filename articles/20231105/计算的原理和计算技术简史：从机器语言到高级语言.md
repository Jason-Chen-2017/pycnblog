# 1.背景介绍



1945年，图灵在剑桥大学研究发现了著名的图灵机。他根据机器人可能存在的两种状态——受控或无控制（即类似于人的运动状态）以及信息处理能力，提出了一种可编程的机器。他利用这个机器制造了逻辑电路，可以实现复杂的运算功能。 

1947年，约翰·冯·诺依曼等科学家开发出机器指令集MIPS（又称为麻省理工结构指令集）。该指令集定义了计算机的基本指令，包括数据存储、算术运算、条件判断、输入/输出及其他系统操作。它是世界上第一个真正意义上的通用计算机指令集。

后来随着计算机的发展，出现了多种高级语言，如Fortran、BASIC、COBOL、Pascal、C、Java、C++、Python等，它们将低级语言编写的程序转换成机器码，从而运行在不同的硬件平台上。现代的计算机系统主要由编译器编译源程序代码产生机器码，并通过计算机主板或芯片直接执行。

那么，什么是计算？计算是指对数据进行加减乘除、比较大小、逻辑运算、统计分析等各种运算的过程。计算的两种基本形式——计算机(computer)和计算能力(computational ability)。 

# 2.核心概念与联系
## 2.1 数据类型
数据类型是计算机语言中用于区分变量存储的数据值的形式，数据类型决定了变量的作用域，范围及其能否变化等属性。数据类型包括整型、浮点型、字符串型、布尔型、数组、结构体、指针等。
### 2.1.1 整型数据类型
整数（integer），也称作有符号整型、正整数、整数值，是表示整数的一种数据类型。整数数据类型一般用来表示数字值，比如0、-1、25、-300等。根据所占空间不同，整数数据类型又分为有符号整型、短整型、长整型、无符号整型四种。例如：
- 有符号整型：有符号整型也可以理解为带符号的整数，它能够表示负数，最高位为符号位，0代表正数，1代表负数；占据4字节（32位）或8字节（64位）内存，范围由负无穷大到正无穷大。例如：int a = -20; int b = 30; // 变量a的值为负20，变量b的值为正30。
- 短整型：短整型是一个短的有符号整型，占据2个字节的内存，范围为-32768~32767。例如：short c = -1000; short d = 20000; // 变量c的值为负1000，变量d的值为正20000。
- 长整型：长整型是一种更大的整型，占据4个字节的内存，范围为-2^31 ~ 2^31-1。例如：long e = 2147483647; long f = -2147483647; // 变量e的值为正2147483647，变量f的值为负2147483647。
- 无符号整型：无符号整型是一种不带符号的整型，占据1个、2个或4个字节的内存，范围为0~2^n-1，其中n表示所占用的字节数。例如：unsigned g = 65535; unsigned h = 4294967295; // 变量g的值为65535，变量h的值为4294967295。
### 2.1.2 浮点型数据类型
浮点型（floating point），也称为实数，是表示小数的一种数据类型。浮点型数据类型包括单精度浮点型（float）和双精度浮点型（double）。其中，单精度浮点型精度只有4位有效数字，但运算速度快，适合精密计算；双精度浮点型精度有8位有效数字，运算速度慢，适合一般计算。例如：
```cpp
float x = 3.14f; // 声明x为单精度浮点型变量
double y = 1.2E+3; // 声明y为双精度浮点型变量
```
### 2.1.3 字符型数据类型
字符型（character），也称为字符、串、单词或字节，是表示文本的一种数据类型。字符型数据类型就是一个单一的 ASCII 或者 Unicode 编码值。ASCII 是 American Standard Code for Information Interchange 的缩写，中文通常使用 GB2312 或 GBK 编码。Unicode 是 Universal Multiple-Octet Coded Character Set 的缩写，支持全球所有语言和字符。例如：
```cpp
char ch1 = 'A'; // 声明ch1为字符型变量，存放'A'对应的ASCII码值
wchar_t ch2 = L'\u0041'; // 声明ch2为宽字符型变量，存放'A'对应的UTF-16编码值
```
### 2.1.4 布尔型数据类型
布尔型（boolean），也叫逻辑型，是表示真假的一种数据类型。布尔型数据类型只有两个取值：true 和 false。表示真的条件表达式一般用关键字 true 来表示，表示假的条件表达式一般用关键字 false 来表示。例如：
```cpp
bool flag = true; // 声明flag为布尔型变量
if (flag == true){
    cout << "flag is true." << endl; // 此行代码只会被执行
} else {
    cout << "flag is not true." << endl; // 此行代码不会被执行
}
```
### 2.1.5 数组型数据类型
数组型（array），也称为表、列表、集合、矩阵，是将同一类型的多个数据项按一定顺序排列，组成一个集合的数据类型。数组型数据类型中的每个元素都有一个唯一的索引值，通过索引值访问数组中的元素。例如：
```cpp
int arr[3] = {1, 2, 3}; // 声明arr为整型数组，长度为3，分别存放1、2、3三个整型值
for (int i = 0; i < sizeof(arr)/sizeof(arr[0]); ++i){
    cout << arr[i] << " "; // 输出数组的各个元素
}
// 输出结果：1 2 3
```
### 2.1.6 指针型数据类型
指针型（pointer），是一种特殊的数据类型，它指向内存地址中的某个位置。它保存的是某一特定数据类型的值的地址，可以对其进行操作。指针型数据类型有三种分类：左值指针、右值指针和函数指针。
#### 2.1.6.1 左值指针
左值指针（lvalue pointer），也称之为左引用指针、左值引用指针，是一种指针，它指向左值（即寄存器变量、栈变量、静态变量、全局变量等变量）。左值指针只能对左值进行操作。
#### 2.1.6.2 右值指针
右值指针（rvalue pointer），也称之为右引用指针、右值引用指针，是一种指针，它指向右值（即临时变量、临时对象、纯右值等表达式的值）。右值指针可以对右值进行操作。
#### 2.1.6.3 函数指针
函数指针（function pointer），是一种指针，它指向函数的入口地址。函数指针只能指向已知且确定的函数入口地址，不能用于创建可变的函数调用。例如：
```cpp
void (*ptr)(int); // 声明函数指针
int main(){
    ptr = func; // 将func函数地址赋值给ptr
    return 0;
}
void func(int num){
    cout << "num: " << num << endl;
}
```
### 2.1.7 结构体型数据类型
结构体型（structure），也叫记录、联合、复合数据类型，是将不同的数据类型组合成一个新的数据类型，可以包含多个成员变量。结构体可以作为独立的数据单元来使用。结构体中的成员变量可以共享相同的内存区域，互不干扰。结构体型数据类型包含多个成员变量，每一个成员变量都有自己的名称和数据类型。例如：
```cpp
struct Point{
    float x, y; // 声明结构体Point的两个成员变量x和y
};
Point p1 = {-1.0f, 2.5f}; // 初始化p1结构体的两个成员变量x和y
cout << "x=" << p1.x << ", y=" << p1.y << endl; // 输出p1的两个成员变量
```
### 2.1.8 枚举型数据类型
枚举型（enumerator），是一种具有固定数目的名称的整数类型。枚举型数据类型非常适合用于代替整数，提升程序的可读性和可维护性。例如：
```cpp
enum Weekday{MONDAY=1, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY};
Weekday today = MONDAY + 3; // 设置today为星期四，3代表往后推3天
switch(today){
    case MONDAY:
        cout << "Monday"; break;
   ...
    case SUNDAY:
        cout << "Sunday"; break;
    default:
        cout << ""; break;
}
// 输出结果：Thursday
```
