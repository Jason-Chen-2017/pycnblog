# 1.背景介绍


信息传递是人类社会工作中一个重要组成部分，但如何传播信息却是影响社会效率的决定性因素。一味追求量的增加、营造热闹繁华的场面，反而会影响到信息的接收者对信息的理解、记忆和应用能力。因此，结构化思考、沟通技巧等在信息传递中的作用尤为重要。信息论和网络理论在研究信息传输领域的基础上提出了信息的三个层次：感知层、理解层和表达层。在人类学和心理学中，已经提出了很多传播信息的有效方法，例如制作各种幽默语句、使用抽象语言、在线视频、留言板、微博、博客等。但这些信息传递的方式仍然存在一些缺陷，如不够有效、组织混乱、容易误导、占用时间、制造焦虑等。而结构化思考和沟通技巧就是为了解决这个问题而产生的。它通过系统地呈现信息的层级关系，把复杂的信息分解成多个容易理解和记忆的关键词或短语，然后再将关键词和短语组织成相互关联的主题，最后再形成完整的观点或意图。结构化思考和沟通技巧可以有效地减少信息的冗余、降低重复度、提升信息的表达能力、促进信息的集中传递。结构化思考和沟通技巧还能够帮助记忆关键词，提高信息的可靠性和准确性，并且在一定程度上避免了错误或歪曲的信息传播。
信息传播中的结构化思考与沟通技巧的优越性体现在以下几个方面：
1. 简化复杂信息：结构化思考的主要目的是简化复杂信息并将其拆分成易于记忆和理解的元素。结构化思考使得信息主体更加专注，关注最核心的主题或事件。它能够使受众更容易接受新信息，并从中获得需要的信息。同时，结构化思考有利于信息的整合、整理和再利用，从而增强了信息的有效性和可信度。

2. 组织信息：结构化思考的第二个目的就是组织信息。由于信息要素之间的联系紧密，所以结构化思考需要将相关信息放在一起进行处理。不同类型信息之间还应该有适当的平衡，不能盲目堆砌所有的信息。结构化思考通过将信息按照不同的维度分类、归纳和组织，可以有效地让信息的重点突出。

3. 提高表达能力：结构化思考的第三个目的是提高表达能力。结构化思考需要选取恰当的词汇、措辞和图像来传达信息。传达信息时需要注意信息结构的清晰和层次性，也要考虑到受众的阅读习惯、表达水平和认识水平。

4. 促进信息的集中传递：结构化思考的第四个目的是促进信息的集中传递。结构化思考会逐步深入到信息的内容本身，所以受众可以先了解一个大的观点或者计划，然后再去细化、实施细节。结构化思考通过将信息划分成不同的模块，逐渐向下传递，会促进信息的快速、集中和精准。

5. 避免错误信息：结构化思考的第五个目的是避免错误信息。结构化思考能有效地揭示信息真正的价值所在。但是，结构化思考也可能伴随着信息的重复和错误传播，所以要防止这种现象的发生。

信息传播中的结构化思考和沟通技巧的局限性也在于：
1. 慢性传播模式：结构化思考和沟通技巧能很好地推动信息的传播，但对于慢性传播模式下的信息来说，它们可能会误导、延缓甚至放大信息的效益。

2. 技术限制：目前为止，结构化思考和沟通技巧主要用于口头交流，但文字形式的信息传播还有待改进。另外，在传播过程中还存在其他种类的约束条件，如时间、空间和工具等，这就要求结构化思考和沟通技巧具有更广泛的适应性和实用性。

3. 群体决策过程中的碎片化信息：结构化思考和沟通技巧的出现虽然有助于提高信息的有效性和可信度，但结构化思考本身就意味着缺乏全面的信息。在群体决策过程中，群体成员往往不会完全共同参与决策过程，只有决策结果才能成为整个团队的一个共识，这时候结构化思考的作用就变得更加有限。

总结一下，结构化思考和沟通技巧都是为了信息的传递和组织，但结构化思考更侧重于信息的系统性呈现和多样性表示；而沟通技巧则更侧重于信息的快速传播、有效沟通以及话题的融合。两种方法的良性互补、相互促进，在信息传播中扮演着至关重要的角色。
