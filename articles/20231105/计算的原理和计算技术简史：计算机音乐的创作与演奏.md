# 1.背景介绍


计算机时代诞生了各种各样的应用软件，其中不乏音乐相关的软件。比如说：iTunes、WinAmp、Foobar2000等。这些软件最大的优点就是具有高度的娱乐性。许多音乐爱好者都会把自己的音乐文件导入到音乐播放器中，或者通过网页浏览器来听歌。因此，如何用编程语言编写出具有高度交互性的音乐播放器，将成为计算机音乐界的一大研究方向。

在20世纪70年代，计算机科学界划分为两个阵营——人工智能（Artificial Intelligence）与计算理论（Computing Theory）。在此之前，计算机科学领域主要关注的是硬件，即程序执行时的机器设备和结构，而计算理论侧重于计算过程及其方法。两者合二为一的产物就是“计算哲学”（Computing Mind）。

20世纪90年代后期，计算机发展迅速，逐渐走向成熟，成为当今整个计算机技术领域的基础设施。由于各方面技术都涉及到了计算理论的理论基础上，所以两者产生了强烈的联系。

在20世纪90年代末至21世纪初，计算机音乐的发展可以分为三个阶段：1980年代：第一次计算机音乐革命，利用传统电子乐器进行音乐创作；1990年代：实时音频处理技术的发明，使得创作者可以实时地制作音乐；2000年代：现代音乐流行的互联网环境下，音乐作曲家可以进行数字化的创作。

在文章开始之前，我们先简单回顾一下1980年代计算机音乐的一些创意与产品。

## 一、第一次计算机音乐革命——洛克希德·马丁与布鲁斯•帕特里克

1980年代是人工智能领域蓬勃发展的时期。杰弗里·佩奇（John McPhee）和凯文·范·博格（Kevin Fisher Borges）团队在贝尔实验室开发出了MUD(Multi-User Dungeon)游戏，通过游戏中的角色扮演体验并创造出属于自己的音乐风格。他们还发明了一款名为"Ludicorp"的计算机键盘，让人们可以通过键盘来演奏音乐。


1983年，洛克希德·马丁（Logan McCoy）和布鲁斯•帕特里克（Bruce Paul）接受了哈佛大学人工智能研究中心的邀请，加入该中心后，共同研发出一种新的音乐生成系统，该系统能够通过计算机自动合成音乐，可谓“冒险者级”。这是第一次真正意义上的计算机音乐革命。

1984年，马丁·帕特里克开始在其MUD游戏中尝试用计算机合成音乐，他发现系统能够生成华丽动人的旋律，且没有任何人类演奏者的参与。帕特里克决定将其创新应用到自己的个人作品上，完成了一首经典名曲——《卡萨布兰卡之夜》（Cassius Claypool's Evening）。该歌曲使人耳目一新的旋律效果，也引起了轰动，最终取得了盛赞。

第一次计算机音乐革命，从一个科技玩具变成了艺术品，真实地展示了计算机科技在艺术创作中的巨大作用。随着第二次计算机音乐革命的开端，新的音乐生成系统被提出来，可以实现更加复杂的音乐创作。

## 二、实时音频处理技术的发明——李宗盛

1990年代，李宗盛（Wang Zhongsheng）受聘于苹果公司，并担任IT部主管，负责Apple Music的研发。在此之前，苹果公司曾推出过多个音乐播放器，但都是静态的。为了能够实现实时的音乐播放功能，李宗盛与美国建筑设计师约翰·伍德克拉克（Jordan Woodcock）一起开发出了基于音频的信号处理技术，包括变声器、均衡器、混响器等。

当时，李宗盛认为，通过音频处理技术能够创造出更加富有表现力、更具吸引力的音乐。因此，他利用其工作经历和对音乐的了解，帮助苹果公司开发出了Apple Music。Apple Music是中国移动电话之后最火热的音乐应用软件之一，截止2021年3月份，全球音乐播放量超过6亿。

## 三、现代音乐流行的互联网环境下——阿兰·图灵

2000年代，互联网蓬勃发展，电子音乐开始进入互联网的视野。阿兰·图灵（Alan Turing）是一位热衷于计算机和数学的学者，他创造了著名的“图灵机”，能够模拟人类的语言行为。

2003年，阿兰·图灵邀请了英国计算机科学家威廉·莫尔斯（Richard Moor）做客微软研究院，讨论如何把计算机变成一种音乐的工具。莫尔斯认为，计算机需要能够处理复杂音频信号、快速响应，并能高效生成音乐。

2004年，经过四年的沉淀，计算机音乐的终极目标终于实现。莫尔斯带着他的理念，与创始人艾伦·库珀（Allen Karp）共同开发出了一款名为Beatnik的软件，它是一个用于制作音效的音频编辑器，能够直接导入音乐文件，并能够自动生成音效。

2006年，Beatnik登陆iTunes Store，并获得了极大的欢迎。在短短几年时间内，Beatnik已经成为十亿日活跃用户的必备应用程式，累积下载量突破了1000万。

如今，越来越多的音乐创作者选择通过网络平台进行数字化的创作。当代音乐的浪潮正在席卷全球。
