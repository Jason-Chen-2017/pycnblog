# 1.简介
  

在1978年，英国科学家尼尔·费米·弗里德曼和他的合作者们制定了著名的“卡耦稳定模型”。该模型认为太阳光线的波长越长，太阳表面离得越远，将会受到的辐射力也就越小。然而实际上由于太阳的运动特性（一年四季围绕地球转），以及太阳光的折射现象等原因，太阳光波长的减弱并不是说简单就能实现的。因此，有必要对相关机制进行更深入的分析和探索。本文基于我国太阳光谱观测数据、经典微分方程模型及实验数据对太阳光谱的变化进行建模分析，进一步研究太阳光谱下太阳辐射的变化规律。

# 2.基本概念术语说明
## 2.1太阳光谱
太阳光谱（Sun spectrum）是由太阳光在不同波长上的能量分布组成的特征。其基本单位是每平方千米（μm^-2）。它是研究太阳光的主要手段之一。

一般来说，太阳光谱可分为三个区域：红色区域（300～440nm）、蓝色区域（440～700nm）和紫外线区域（700～2500nm）。其中，红色区域的辐射较强，蓝色区域辐射较弱，紫外线区域辐射最小。

## 2.2卡耦稳定模型
卡耦稳定模型（Clayton-Enquist stability model）是20世纪70年代由费米·弗里德曼和威廉·格雷戈特提出的。该模型假设太阳光波长越长，其辐射变暗的速度就越快。但是实际上，太阳的辐射性质决定的太阳光波长，其途径各种物体影响，并且温度、高度、位置的不同会导致不同的光谱分布，无法通过简单的数值计算直接确定太阳光谱的变化曲线。所以，卡耦稳定模型只是对太阳光谱的一种假设，只能提供一个方向性的认识。

## 2.3 欧拉-朗道方程
欧拉-朗道方程（Euler-Lagrange equation）描述的是力作用下的静止点内电场的偏导。其全称为“欧拉-朗道方程”，也被称为“能量守恒方程”，即力学系统的运动方程。它用法向矢量$\vec{F}$表示作用在某一位置处的力，而位移矢量$\vec{V}$则表示受力系统在此位置处的位移。则欧拉-朗道方程可以写作：

$$\frac{\partial \vec{V}}{\partial t} = - \nabla \cdot (\vec{F}+\mu_0\epsilon_{\rm s}\vec{E}) + Q,$$

式中，$\epsilon_{\rm s}$为介电常数；$\mu_0$为物质真空中的磁导率；$Q$为磁感应强度。注意这里的正负号。如果要求$\vec{V}$和$\vec{E}$满足保守散度关系，则有：

$$\rho(\nabla\cdot \vec{V}) = \rho(\nabla\cdot \vec{E}),$$

式中，$\rho$为流体密度。
