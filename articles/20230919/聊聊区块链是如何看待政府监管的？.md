# 1.简介
  

随着比特币、以太坊等区块链技术的崛起和普及，越来越多的人加入到数字货币行业中来。同时，越来越多的金融机构和监管部门也开始对其进行监管。那么，区块链对于政府监管有何影响呢？

最近，美国国会通过了一项“《美联储法案》”，该法案要求美国银行业和金融机构加入“危机管理模式”(CM) ——即，将一种新型经济危机预警系统的构建、评估和应对工作委托给第三方专业机构。

这种所谓的“危机管理模式”包含了以下几个层面：

1. 建立起一套标准化的、科学的危机评估机制；
2. 提供行业监管部门、财政部和法律界同仁之间共同参与的“第三方评审团体”(TPRT)；
3. 使用计算机模型模拟危机发生前后的金融市场状况，分析并识别出系统性风险；
4. 基于系统性风险的识别结果制定相应的应对措施，如降低基准利率、退出恶意投资者的资产、暂缓或取消贸易限制等；
5. 通过跟踪经过评估和应对过程的实际效益，持续优化危机管理模型和流程，确保能够及时发现、处置和减少新的风险。

在本文中，我想从以下几个方面来阐述一下区块链对于政府监管的影响。

1. 互联网的特性带来的信息孤岛现象。
由于信息的不对称性、价值主张的分散性，使得区块链的应用场景也产生了巨大的变化。传统的金融行业，往往依赖于中心化的机构，而联盟链的出现则打破了这一格局，将公链的数据全面开放，让所有人都可以参与其中，这极大地拓展了金融的边界，让金融活动更加透明。但是，联盟链虽然降低了中心化的成本，但它自身也存在很多缺点，比如信任问题、隐私泄露的问题等。在这种情况下，又回到了最初的问题——信息孤岛。

2. 在这个信息孤岛中，是否可以引进专业的高级智能合约？
目前，区块链在金融领域的应用已经得到了很好的发展。随着时间的推移，区块链应用场景的延伸正在逐渐形成，例如物联网（IoT）、生态健康、支付结算等领域。但是，在这些领域中，区块链究竟如何在政府监管中发挥作用，仍然是一个重要的话题。

联邦经济委员会（Federal Reserve Board）近期就设立了一个项目，就是通过区块链技术解决支付结算中的第三方担保问题。联邦经济委员会希望借助区块链技术来创建一套可信赖的第三方支付系统。该系统将集结各种交易当事人的身份信息、联系方式、收款账户信息等，形成一条记录流水账，并提供原告、被告双方进行协商的服务。一旦双方达成协议，授权受益人（包括商家、物流公司等）即可按照协议向受益人发放利息、付款。

此外，区块链还可以在维护知识产权方面发挥作用。最近，美国证券交易委员会(SEC)曾经提出，利用区块链技术来建立一个专属知识产权登记数据库。只要版权拥有者发布了作品，登记在区块链上的作品的权威证据就足够可靠，再通过审核就可以确认其拥有者。因此，任何试图侵犯他人的知识产权的行为都可以通过区块链技术来追溯责任。

3. 发展方向和影响力。
区块链在政府监管方面的发展具有很大的潜力，因为它是去中心化的，没有一个中心权威来决定某个规则是否适用或者实施起来是否安全。与此同时，区块链的易用性也让它在监管方面的推广和落地变得十分容易。因此，在未来，区块链对于政府监管的影响力必将扩大。

4. 总结。
在本文中，我们从区块链对于政府监管的影响上探讨了一些热点问题。第一，区块链的发展促使了信息的整合，对于政府的监管造成了信息孤岛的形成。第二，联邦经济委员会通过区块链技术解决支付结算中的第三方担保问题，有效增强了公众参与的意愿，有利于促进监管的透明化。第三，区块链对于维护知识产权的推动，有可能成为防范腐败的有效手段。最后，在未来，区块链对于政府监管的影响力必将扩大，提升公共治理的效率。
