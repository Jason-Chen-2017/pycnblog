# 1.简介
  

消费行为是指购买某种商品或服务，并在一定时间内持续使用其中的行为。从社会、经济、文化等角度看，消费行为有助于构建并维系个人和社会之间的联系，促进经济发展，保护环境、美好的家庭、健康生活。随着人们对生活质量的追求和对物价水平的关注，消费行为正在发生着翻天覆地的变化。越来越多的人选择了便宜、高性价比的产品和服务，而不是盲目追求高额的收入。在这种情况下，消费行为将成为衡量一个社会是否富裕、成功的重要指标。但是，正如某些社会学家所说，消费行为是一种建立在“自愿”基础上的主观判断，需要通过对数据的分析才能得出客观结论。因此，本文仅从“消费者”的视角出发，对消费行为的变迁进行剖析。
# 2.定义与分类
消费行为是指用户在日常生活中与商品或服务互动形成的一系列行为，包括浏览、下单、评价、分享、订阅、使用、交流、体验等。它是对商品或服务的认知、消费决策及购买行为的一种总结、组织和呈现方式。一般认为，消费行为分为两个层次：“需求层次”和“行为层次”。
- “需求层次”：需求层次是指满足特定目标的需要，包括获取新信息、提升能力、节省开支、购物享受、减少痛苦、改善身心健康等。它由四个要素组成：信息需求、能力提升、经济效益和享受。例如，超市的“优惠券”功能就是为了满足购物者的“经济效益”和“享受”需求；早餐店的“促销会场”功能也是为了满足消费者的“享受”需求。
- “行为层次”：行为层次是指消费者的直接购买、接触或使用某样商品或服务，包括浏览、购买、评价、分享、订阅、使用、交流、体验等。行为层次涵盖多个方面，如选择、预订、支付、配送、分享等。行为层次的突出特征是“消费者行为”，即消费者在实际行动过程中产生的购买意愿、购买习惯、选择偏好、消费决策过程等。
在“需求层次”和“行为层次”之间存在一个较大的鸿沟，即需求层次主要侧重于满足个人的需求，而行为层次则侧重于人与环境、商品或服务之间的互动关系。因此，消费行为也有二元对立之分，即“需求型消费者”和“行为型消费者”。
# 3.消费者群体的变化
在过去的三十年里，中国经济发展势头惊人，然而，消费者群体却经历了巨大的变化。中国共产党执政以来，国民经济蓬勃发展，消费者群体的规模和购买力都得到快速增长。过去几年，中国移动互联网的普及，使得人们不用到处跑亲戚朋友找手机号码，可以根据需求随时随地拍照上传购物行为记录。人们更多地以打车、外卖为交通工具，选择线上购物平台，喜欢用微信、QQ等社交工具分享、讨论电商平台上的商品。近年来，随着金融危机的影响和全球经济复苏的推进，中国消费者的消费需求也不断增加，但消费者群体却出现了一些新的变化。
首先，近年来，随着中国经济社会的发展，人们越来越重视个性化推荐系统，喜欢用不同方式参加购物活动。人们购买力的提升也促进了线上购物平台的兴起。但是，由于新鲜感和魅力化的“新鲜感”，导致在线购物模式下的“大众化”购物和抽象的评价等消费决策方式在一定程度上降低了用户的满意度。
其次，网络购物的兴起加速了人们购物需求的扩张。随着消费者对品牌建设、个人隐私保护等方面的关注，消费者逐渐把更多的精力放在掌握个人的数据和行为习惯上。尽管网络购物平台给予消费者多种购物选择，但用户购买决策仍然依赖于基于物品特性的相似性和购物实践经验。基于此，人们期待网络购物平台能够提供个性化的推荐引擎和购买决策，帮助消费者根据自己的消费习惯做出更加合适的购买选择。
再次，电子商务的蓬勃发展和电商平台的横向整合，让消费者可以在线上网购得到更多的品类和服务，而在线购物平台也更好地满足了人们的需求。但同时，随着网络安全的担忧和平台对顾客信息的收集和利用问题的增多，越来越多的人开始担心“数据共享”带来的隐私风险。
最后，随着社会的进步和经济的发展，消费者越来越依赖第三方信任，希望通过网络获得更加真诚的互动和支持。但同时，也越来越多的消费者发现，这些第三方平台往往无法提供足够的保障。当消费者发现这些平台无法满足其需求，甚至出现欺诈、滥用的情况时，他们的基本要求便会转向其他平台或实体。
综上所述，消费者的消费需求和购买行为已经发生了一定的变化。其中，需求型消费者的数量占比已从历史上最低点的1%上升到了20%，但购买力方面的差距却远未缩小。这种新的消费模式还将继续向前发展，并逐渐影响到社会结构和经济增长。
