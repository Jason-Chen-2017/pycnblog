# 1.简介
  

## 1.1 研究背景
随着基因组结构研究和生物信息技术的快速发展，全基因组测序（whole genome sequencing, WGS）已成为高通量测序领域的热门话题。随着序列数据量的增加，生物信息学研究者需要高效、准确地处理这些数据。近年来，基于核苷酸转录本的测序技术也逐渐成为主流。

RNA-Seq 是一种全基因组测序方法，它可以检测出整个基因组上所有非编码RNA的含量。一般情况下，一个人的基因组一般分为两部分，外显链和内隐链，外显链负责控制人的各种生理活动，而内隐链则是存储DNA复制本的基因，一般我们称之为mRNA或messengerRNA。而mRNA细胞之间传递信息的管道就称为宏复合层（polysome）。所以通过对宏复合层进行RNA-Seq测序，就可以获取到人的整个基因组上所有的RNA。

## 1.2 数据处理流程
RNA-Seq测序包括四个主要步骤：

1. DNA/RNA提取及转运。通常用实验室加工后的纯化D/NTP水平转运过来的DNA/RNA样品被分离出来并配对，进行大小选择后过滤干扰，经过比对，再进行双链互补，最后转移至下一步。

2. PCR扩增。将DNA/RNA样品经过PCR扩增成适应性标记（通常是引物、上色标签），然后将扩增后的标记上色，产生适当的RNA链。

3. 核磁共振。将PCR扩增产出的RNA通过核磁共振（NMR）方法转换成电子信号，从而获得感兴趣区域的序列。

4. 晶体测序分析。利用获得的RNA序列数据，进行二级分析，揭示其所代表的功能信息，如转录调节密码、蛋白质组装、结构预测等。

## 1.3 应用领域
### 1.3.1 生命科学领域
RNA-Seq在生命科学领域中处于基础工作岗位。它可以用于疾病诊断、基因功能研究、疾病防治、药物开发等方面。例如，通过RNA-Seq分析患者的血液和肿瘤细胞的mRNA序列，可以发现潜在的癌症突变、转座子修饰、基因重组等候选突变；通过RNA-Seq分析不同人群之间的差异基因表达，可以发现罕见的遗传疾病与特殊样本之间的相似性；通过RNA-Seq分析肿瘤细胞基因组上的miRNA标记，可以发现罕见的癌症靶向药物；通过RNA-Seq分析胚胎的内环境微生物群落，可以发现潜在的免疫细胞缺失和抑制等候选突变。

### 1.3.2 医学领域
RNA-Seq的应用不仅仅局限于生命科学领域。在医学领域，通过对不同种族的人类孤儿组织、神经元发放系统细胞以及动植物的长染色体进行测序，都可以发现微小的突变，并发现隐藏在它们中的疾病，比如艾滋病毒和前列腺癌症等。通过全基因组分析，也可以发现新病原体的模式和致病基因突变情况，进而开发针对特定疾病的治疗方案。

### 1.3.3 健康管理领域
通过全基因组的测序，我们可以通过相应的基因型和表观遗传学特点，识别身体各器官和系统的疾病发病机制。通过对全基因组数据的分析，我们还能够评估个体的内分泌抑制能力，监测糖尿病和运动障碍等疾病的风险因素。同时，人们还可以在基因组上发现内环境微生物的作用机制，通过这些知识，可以帮助医护人员设计相应的健康管理措施，提升生活质量。
