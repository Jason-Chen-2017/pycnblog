# 1.简介
  

## Chatbot 是什么？
Chatbot(中文：聊天机器人)，又称对话系统、文本输入设备，是一种可以与用户进行即时通信的机器人程序。它通过聊天的方式进行交流并自动响应用户的查询、指令或请求。主要特点包括高度自动化、随时可用、只需输入信息便可完成任务、符合多样性需求。在生活中，例如智能家居、新闻阅读、购物咨询、导航等方面都有广泛应用。Chatbot已经成为越来越普遍的应用场景，在创业公司、大型组织、政府部门也被部署用于营销及管理。Chatbot的开发也成为一个比较热门的话题。目前国内外各种Chatbot开发平台如微软小冰、Google Dialogflow、IBM Watson等相继推出，开发者可以使用这些平台快速构建自己的Chatbot。

本文将介绍如何利用Dialogflow创建一个简单的Chatbot。这个Chatbot是一个基于姓名的问候语，根据用户输入的姓名返回相应的问候语。Dialogflow是一个Chatbot的云服务提供商，提供一套完整的流程，包括对话设计、训练模型、API集成、消息交互等，并且免费试用。本文将详细介绍怎样从零开始搭建一个简单的问候语Chatbot。

## 准备工作

## 实体
实体（Entity）是指由一组事实和特征所描述的事物，例如人的姓名、地址、电话号码、邮箱等。在Dialogflow中，我们可以通过定义实体来收集用户输入的信息，使得对话更加有效。比如，我们可以创建一个名为"Name"的实体，将用户输入的姓名解析出来。 

进入Dialogflow控制台，选择左侧菜单中的"实体"页面，然后点击右上角的"+"按钮创建新实体。


创建一个名称为"Name"的实体，然后添加一些示例值："Alice"、"Bob"、"Charlie"。


创建成功后，我们就可以识别用户输入的名字了。假设我们用户输入的是"小明"，那么Chatbot就会返回"你好 小明，有什么可以帮助您呢?"。

## 意图
意图（Intent）是对话系统的核心部分，它描述了机器人应该做什么，例如询问用户信息、寻找兴趣爱好、进行交易等。

进入Dialogflow控制台，选择左侧菜单中的"意图"页面，然后点击右上角的"+"按钮创建新意图。


创建一个名称为"greeting"的意图，然后添加一些训练语句，比如："你好"、"你叫什么名字"、"早上好"、"晚上好"、"生日快乐"。


创建成功后，我们就可以根据不同的用户输入转入不同的响应函数了。假设我们用户输入的是"小明"，Chatbot会去匹配相应的意图，比如"greeting"，然后返回相应的问候语："你好 小明，我是你的私人助理，有什么可以帮到您的呢?"。
