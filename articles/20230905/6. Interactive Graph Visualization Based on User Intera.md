# 1.简介
  

图可视化领域一直处于蓬勃发展阶段。但由于各种因素的影响，其在用户体验、可用性、效率和效果上存在诸多不足之处。为了更好的服务于数据分析工作者的需求，相关研究工作者提出了多种针对图可视化的新方法，例如基于交互的算法、基于空间位置编码的新方法、等等。本文将基于系统化评估的理论，梳理当前已经提出的交互式图可视化技术以及各自的优缺点，并给出未来的研究方向和挑战。
# 2.基本概念术语说明
#    * **Graph**: 由结点（Node）和边（Edge）组成的网络结构数据结构。它可以用来表示数据之间的关系、结构关系、社会关系、金融关系等复杂信息。如社交网络、物流网络、电影网络等都是图结构。
#    * **Visualization**: 将复杂的数据转化为一种图形图像，使得人类容易理解、快速识别、获取有价值的信息。许多可视化技术都致力于突破传统静态图表的限制，提供更富有动态感知的视觉信息。目前，最热门的可视化技术包括矩阵树布局、傅立叶级数扩散、树状图嵌套、邻接矩阵和波浪图等。
#    * **Interactive Graph Visualization:** 指的是利用人机交互的方式，让用户对图的节点和边进行操作，从而得到增强的可视化效果。这项技术通过可视化算法根据用户交互实时生成新的视觉效果，提升了数据分析工作者的体验。例如，缩放、拖动、放大等操作，都属于人机交互方式。
# 3.核心算法原理和具体操作步骤以及数学公式讲解
## 3.1. UMAP(Uniform Manifold Approximation and Projection)
UMAP的主要思想是对高维数据的低维表示进行简化，同时保留原始数据中的信息。UMAP的计算流程如下所示：

1. 初始化随机的质心；
2. 根据高斯分布近似每个数据点所在的低维空间分布；
3. 在低维空间中计算距离，然后找到两个数据点之间直线距离最近的两个点，找到连线上的所有点；
4. 对这个新的低维空间进行重新映射，使得原始数据点的关系尽可能完整地保留下来；
5. 通过迭代更新的方法，最终收敛到一个比较好的结果。

下面的公式给出了UMAP算法的详细过程：
### 公式1：$d_{i}$
在$n$-维欧氏空间$\mathbb{R}^n$中，两个点$p_i,p_j\in \mathbb{R}^n$的欧氏距离$d_{ij}=||p_i-p_j||=\sqrt{\sum_{k=1}^n{(p_i^k-p_j^k)^2}}$。其中$^k$表示第$k$个坐标。
### 公式2：$C_{\epsilon}(p)$
以点$p$为中心，半径为$\epsilon$的球内所有点的集合。记$N_c(p,\epsilon)$表示$C_{\epsilon}(p)$内包含的点个数。则$C_{\epsilon}(p)=\{q|d_{pq}\leq \epsilon\}$。
### 公式3：$C^{*}_r(\cdot)$
以点$p$为球心，半径为$r$的球内所有点的集合。定义$P_r(\cdot)$表示所有满足条件的点的集合，即$N_c(p, r)=N_c(q, r), q\neq p$。则$C^{*}_r(\cdot)=\cup_{p\in P_r(\cdot)} C_{\frac{1}{2}r}(\cdot)$。
### 公式4：$\tilde{f}_{i}$
投影函数，它把$f_i$的函数值映射到新的低维空间中。$\tilde{f}_{i}$定义如下：
$$\tilde{f}_{i}(x)\triangleq f_i(W^{-1}\cdot x+w)$$
其中，$f_i:\mathbb{R}^{m}\to \mathbb{R}, W\in \mathbb{R}^{n\times m}, w\in \mathbb{R}^{n}$，是原函数$f_i$在$m$维向量$x\in \mathbb{R}^{m}$上的投影。
### 公式5：$f(\cdot)$
原函数$f_i$。
### 公式6：$d^{\prime}_{\boldsymbol{X}}(\cdot,\cdot)$
$\boldsymbol{X}$的距离函数，它用来衡量$f$和$\tilde{f}$的相似程度。设$g_{i}(\cdot)$是用某种函数形式的新函数，则$\tilde{g}_{i}(\cdot)$定义如下：
$$\tilde{g}_{i}(\boldsymbol{x})\triangleq g_{i}(W^{-1}\cdot \boldsymbol{x} + w)$$
其中，$g_{i}: \mathbb{R}^{n}\to \mathbb{R}, W\in \mathbb{R}^{n\times n}, w\in \mathbb{R}^{n}$是$f_i$对应的新函数。
$$d^{\prime}_{\boldsymbol{X}}(f(\boldsymbol{x}),g_{i}(\boldsymbol{y})) = ||\tilde{f}_{i}(\boldsymbol{x})-\tilde{g}_{i}(\boldsymbol{y})||_2^2 = ||W^{-1}\cdot (\boldsymbol{x}-\boldsymbol{y})+\left(W\cdot w - \tilde{f}_{i}(w)\right)-\left(W\cdot w - \tilde{g}_{i}(w)\right)||_2^2 $$
