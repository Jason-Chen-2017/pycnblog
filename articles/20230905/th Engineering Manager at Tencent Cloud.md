# 1.简介
  

腾讯云是国内最大的云计算服务提供商之一，其产品线覆盖虚拟机、容器、serverless等多种计算形态，并通过多样化的服务能力满足用户不同的业务场景需求。公司业务包括视频直播、游戏、电商、物联网、智慧零售等多个领域。截至目前，腾讯云已经成为行业龙头企业之一，在全球范围内拥有超过17亿用户。作为一家创新型互联网公司，腾讯云始终坚持以“客户至上”的理念，充分理解客户的需求和痛点，并用科技赋能，持续创造价值。
2019年初，腾讯云开始启动“8th Engineering Manager”系列活动，旨在为一批具有经验、深厚技术底蕴且具备很强责任感的技术人才培养起到示范作用。这是腾讯内部发起的一项重要活动，旨在推广、培养和激励更多优秀的人才加入我们的公司，共同推进腾讯云技术向前发展。每一次这个系列的活动都会由一位领军工程师（CEO）或最具权威的公司高层带领，带领更多的技术人才走入我们的队伍，参与到公司的技术决策当中来。
# 2.岗位要求
目标岗位：8th Engineering Manager at Tencent Cloud (Tencent Cloud技术委员会二级副主席)
职位描述：腾讯云技术委员会是腾讯内部的技术管理团队，负责腾讯云的关键技术方向的重大变革、架构设计、技术方案的制定及落地、资源规划与管理等工作。候选人的主要工作内容是：负责腾讯云技术委员会的日常工作，包括组织和跟踪委员会的决策，参与腾讯云技术委员会的决策进程，进行技术咨询和沟通，促进技术发展；熟悉各项技术，并能为委员会的决策提供可靠的参考依据；参与腾讯云内部各项技术工作，做好相关技术事务的协调，推动腾讯云技术体系的建设和成长。
职位要求：
1. 本科以上学历，计算机、软件、通信、电气工程等相关专业，至少5年相关工作经验；
2. 有良好的沟通表达能力、时间管理能力和团队精神；
3. 对产品研发和架构设计、基础平台建设有深刻的认识；
4. 有全面的计算机视觉知识和对各类数据库的熟练掌握；
5. 了解前端、后端、系统工程、安全、测试、运维、数据分析等相关技术领域；
6. 具有优秀的项目管理和解决问题的能力；
7. 具有较强的英文水平，并对所在国家/地区文化和社会现状有一定的了解。
任职资格：申请者需同时满足以下条件：
1. 年薪：25-40万元/月，期望越高越好，一年能够带来预计收入的2%~3%；
2. 性别不限，但女性优先；
3. 身体健康，无犯罪记录；
4. 担任候选人将代表腾讯云技术委员会承担日常管理工作，并且积极参与到腾讯云技术委员会的决策制订中来；
5. 提交者需自荐；

1. 组织和跟踪委员会的决策，参与腾讯云技术委员会的决策进程，进行技术咨询和沟通，促进技术发展

   - 对候选人能力进行评估，给予适当的培训和指导，确保候选人能做出可行的决策；
   - 主动参与组织分享及研讨会，为委员会成员及候选人提供一个良好的交流平台；
   - 收集、整理、分析委员会提出的意见建议，并及时反馈给委员会，对需要做出调整的地方进行修正；
   - 在合作方的帮助下，主动探索新技术、工具、方法论等应用于腾讯云技术发展，提升团队整体技术能力。
2. 熟悉各项技术，并能为委员会的决策提供可靠的参考依据

   - 积极投身社区，了解和尝试新兴技术，拓宽视野。
3. 参与腾讯云内部各项技术工作，做好相关技术事务的协调，推动腾讯云技术体系的建设和成长
   - 从事腾讯云关键技术的开发、测试、运营、实施，对腾讯云的技术体系、基础设施、产品进行规划、设计、实施和维护；
   - 与相关团队建立密切联系，利用相关平台进行知识共享和技术交流，促进腾讯云技术生态的完善；
   - 通过技术赋能，推动腾讯云技术体系的健康成长。
4. 主导腾讯云技术委员会的工作
   - 为委员会的技术决策提供决策支撑，协调资源和信息，提升委员会的整体运转效率；
   - 负责候选人的个人发展，发掘其潜力，引导其成长；
   - 主动为候选人寻找合适的工作岗位，提升公司的竞争力；
   - 根据委员会的需求，为委员会提供优质的培训和工作环境。
5. 协助团队成长
   - 和其他成员一起为团队提供技术上的支持和建议；
   - 演示公司的技术创新成果，分享成功经验；
   - 分享有关技术类的会议纪要，促进团队间的合作。
