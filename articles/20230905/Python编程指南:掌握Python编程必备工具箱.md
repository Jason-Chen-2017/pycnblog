# 2.基本概念术语说明
## 2.1 Python语言概览
Python 是一种面向对象的高级编程语言。它的设计哲学是“可读性很重要”。Python 最初于1991年由Guido van Rossum发明，第一版发布于1994年底。Python 的版本号采用年份表示法，即以2开头的版本代表2000年代，以3开头的版本代表2003年代。目前最新版本的 Python 是3.x系列，而2.x系列则处于维护状态。
Python 最具特色的就是其独有的缩进语法，使得代码的层次结构清晰、整齐，更容易理解和维护。同时，由于其跨平台特性，Python 可以轻松编写与部署在不同平台上的应用程序。
Python 拥有丰富的数据类型，包括整数、浮点数、字符串、列表、元组、字典等；还提供了面向对象的编程模型，允许创建自定义的对象类型，并可以基于这些对象构建更加复杂的应用。此外，Python 提供了许多的内置函数和模块，可以让程序员方便地完成各种任务。

## 2.2 核心语言组件
### 2.2.1 解释器
每一个运行 Python 脚本的机器上都会安装一个 Python 解释器（又称为 Python 虚拟机）。这个解释器是一个用来执行 Python 代码的程序，它接收用户输入的代码并把它翻译成计算机可以识别和执行的指令。解释器能够读取、分析、解释和执行源代码，然后返回结果给用户。

不同的操作系统都有自己的默认的 Python 安装包。因此，如果安装了多个 Python 版本，就可以通过设置环境变量来指定要使用的解释器。

```
#!/usr/bin/env python3
```

这个 shebang 行告诉系统调用 "/usr/bin/env" 命令来查找名为 "python3" 的可执行文件的位置。这样做可以在不同操作系统上使用相同的命令来启动解释器。

### 2.2.2 注释
单行注释以井号 # 开头，直到这一行结束：

```python
# This is a single-line comment
```

多行注释可以用三个双引号 " 或三重单引号 ''' 分割，并跟随至少两个空格或制表符：

```python
"""This is a multi-line
   comment."""

   or

   '''This is also a
      multi-line
     comment.'''
```

### 2.2.3 标识符
标识符是用于区分变量、函数、类、模块等名称的字符序列。

标识符规则遵循下面的约定：

1. 标识符只能由字母数字、下划线 (_) 和美元符号 ($) 组成。
2. 第一个字符不能是数字。
3. 大写字母表示的是公共（public）的，小写字母表示的是私有的（private）的。
4. 下划线前后的一个字母组合通常会被视为一个下划线形式的单词，比如 snake_case 或者 MACRO_CASE。

### 2.2.4 数据类型
Python 有五种基本的数据类型：

1. Number（数字）：整数、浮点数、复数。
2. String（字符串）：字符串文字用单引号或双引号括起来，使用反斜杠 \ 对特殊字符转义。
3. List（列表）：列表用方括号 [] 括起来，元素间用逗号隔开。可以包含不同类型的数据。
4. Tuple（元组）：元组用圆括号 () 括起来，元素间用逗号隔开。元组的元素不可变。
5. Dictionary（字典）：字典用花括号 {} 括起来，键值对之间用冒号 : 分割，每个键值对之间用逗号隔开。字典的键必须是唯一的，值可以取任何数据类型。

可以通过 type() 函数判断变量所属的数据类型。

```python
a = 1          # int
b = 2.3        # float
c = 'hello'    # str
d = [1, 2, 3]  # list
e = (1, 2, 3)  # tuple
f = {'name': 'Alice', 'age': 25}  # dict
print(type(a), type(b), type(c), type(d), type(e), type(f))  
```

输出：

```
<class 'int'> <class 'float'> <class'str'> <class 'list'> <class 'tuple'> <class 'dict'>
```

### 2.2.5 变量
变量用于存储数据的值。变量是 Python 中最基本的元素之一。使用变量可以有效地减少代码量，提高代码的可读性和健壮性。

变量声明语法如下：

```python
variable_name = value
```

例如：

```python
number = 10              # integer variable
pi = 3.14                # floating point number variable
name = 'John Doe'        # string variable
fruits = ['apple', 'banana']     # list variable
coordinates = (3.14, 2.71)      # tuple variable
person = {
  'name': 'Alice', 
  'age': 25,
  'city': 'New York'
}       # dictionary variable
```

也可以一次性声明多个变量：

```python
a, b, c = 1, 2, 'three'
```

注意：

1. 不要将关键字和保留字用作变量名。
2. 变量名应该具有描述性和意义，不要太长。
3. 变量名应该遵循字母大小写的规范。
4. 在同一作用域中，不允许重复声明相同的变量名。

### 2.2.6 表达式和语句
表达式是由变量、操作符和函数组成的计算单元。表达式最终的结果是一个值。语句是指用来执行某些操作的指令集合。

表达式可以直接放在 Python 脚本的任意地方，Python 会自动求值并返回结果。比如：

```python
>>> print('Hello, world!')
Hello, world!
```

语句一般以换行符结束，它们会导致 Python 执行相应的操作。比如：

```python
x = y + z
if x > 0:
    print('positive')
else:
    print('negative')
```

### 2.2.8 控制流
控制流是指根据条件选择执行的代码块。Python 使用 if-elif-else 结构来实现条件判断和循环。

#### if-else 结构
if-else 结构是 Python 中最基本的条件判断结构。根据条件判断结果，执行对应的代码块。

```python
if condition1:
    # do something here...
elif condition2:
    # do something else here...
else:
    # execute this block when all conditions are false
```

#### for 循环
for 循环用来遍历序列（如字符串、列表或元组）中的元素，每次迭代时，自动将序列中的下一项赋值给循环变量。

```python
fruits = ['apple', 'banana', 'orange']
for fruit in fruits:
    print(fruit)
```

#### while 循环
while 循环用来在给定的条件保持满足的情况下重复执行一段代码。

```python
i = 1
while i <= 5:
    print(i)
    i += 1
```

#### break 和 continue 语句
break 和 continue 语句用来修改循环的流程。break 语句跳出当前循环，continue 语句跳过当前轮次。

```python
fruits = ['apple', 'banana', 'orange']
for fruit in fruits:
    if fruit == 'banana':
        continue
    print(fruit)
    if fruit == 'orange':
        break
```

#### pass 语句
pass 语句什么都不做，它主要用于占位符。

```python
def myfunction():
    pass
```

## 2.3 文件读写
在 Python 中，可以使用 open() 方法打开文件，并通过文件对象（file object）进行读写。以下示例演示如何读取文本文件的内容：

```python
with open("filename.txt", mode="r") as file:
    content = file.read()
    print(content)
```

其中，mode 参数用于指定文件的打开模式：

- r：以只读方式打开文件，文件指针指向开头。这是默认模式。
- w：以写入方式打开文件，文件指针指向开头。如果文件不存在，就创建一个新的文件。如果文件存在，就覆盖之前的文件内容。
- a：以追加模式打开文件，文件指针指向结尾。如果文件不存在，就创建一个新的文件。
- rb：以二进制读模式打开文件，文件指针指向开头。
- wb：以二进制写模式打开文件，文件指针指向开头。
- ab：以二进制追加模式打开文件，文件指针指向结尾。

除了 read() 方法外，文件对象还提供了其他方法来操作文件。比如 seek() 方法可以移动文件指针，tell() 方法可以查看文件指针当前位置。
