# 2.iOS应用开发语言Swift
## Swift概述
Swift 是 Apple 推出的一种开发语言，主要面向 iOS 和 macOS 两个平台，取代了 Objective-C，可以在编译时执行类型检查和内存管理，并提供方便的语法糖和高级特性。它的语法和特性是为了帮助开发人员更高效地创建应用程序而设计的，同时还引入了一些安全和生产力工具。Swift 的第一个版本于 2014 年发布，最新版为 Swift 5.3。

## Swift语法特点
### 简洁明了
Swift 提供了一系列语法糖（Syntactic Sugar）来简化代码编写，比如可以使用 `let` 来定义常量，`var` 来定义变量，`->` 来表示函数返回值类型，`for...in` 来遍历数组或者集合，`if let` 来避免 Optional 值的 unwrap。这些语法糖让代码看起来更加简单、直观。

### 强类型
Swift 是一种静态类型的编程语言，它会在编译阶段进行类型检查。这意味着，如果编译器无法根据表达式的值来判断其数据类型，则会报错。另外，Swift 有着严格的命名规则，类、结构体、方法都需要符合驼峰命名法（首字母大写，后续单词小写）。

### 运行时安全
Swift 在运行时提供了安全机制，包括内存管理、异常处理和访问控制。当程序发生错误时，它可以通过异常来通知调用者，从而使程序可以被调试或重试。Swift 使用引用计数来自动管理内存，保证内存安全。此外，Swift 可以在编译时防止常见的编码错误，包括不安全的直接内存访问、野指针和栈溢出等。

### 高性能
Swift 被设计成一个优化的语言，支持基于 LLVM 的即时编译器，可以提升运行速度。与 Objective-C 比较，Swift 在运行时环境中提供了更多的底层 API 和能力。例如，Swift 通过闭包和函数式编程允许在运行时实现动态编程。另外，Swift 在标准库中提供了高性能的数据结构和算法，例如 Array、Dictionary 和 Set。
