# 2. 基本概念术语说明

首先，我们需要对目标检测及其相关术语有一个整体的认识。

## 什么是目标检测？

目标检测（object detection）是一个计算机视觉任务，它用于识别、定位、分类目标对象，属于计算机视觉中的一类图像理解任务。目标检测的主要功能是在输入图像或视频帧中检测到目标物体，并给出其位置、大小、形状等信息，帮助自动驾驶系统、监控摄像头和图像分析等领域做决策。



## 目标检测相关术语

1. **输入图片** （input image or input frame）：指待检测的图像或视频序列的一帧图像。

2. **检测框** （bounding box）：检测到的目标的矩形区域，由四个参数确定：中心点坐标x，y；宽w和高h；相对于输入图片左上角的偏移dx和dy。


   

3. **置信度** （confidence score）：检测框中包含目标物体的概率。

4. **类别** （class）：指识别出的目标物体的种类，如“狗”，“猫”等。

5. **锚框** （anchor box）：一种较小的边界框，作为候选区域生成器，使得网络可以预测出更准确的边界框。


   

6. **正负样本** （positive and negative samples）：指训练过程中的输入图片中是否含有目标物体、背景、或者其他。

## 1. 基于卷积神经网络的目标检测算法

### 1.1 YOLO v1 (You Look Only Once)

YOLO v1是最早期的目标检测算法之一。它通过在全连接层之间加入空间特征提取模块（spatial feature extractor），利用多尺度、长宽比不同的候选区域，结合不同尺寸的预测窗口，完成了目标检测的同时还保留了网络的速度优势。
