李航（后改名陈丽云），清华大学计算机系博士生、ACM亚洲区奖牌获得者，主要研究方向是统计学习、机器学习和数据挖掘。她于1987年出版了《统计学习方法》一书，这是一本经典著作，也是中国最早的统计学习方法综述。其间，她曾担任国际知名期刊Journal of Machine Learning Research(JMLR)主编、IEEE Transactions on Neural Networks(TNN)编辑，并是美国统计协会的正式成员。近几年，她在人工智能、机器学习领域发表论文已达50余篇，其中SCI、SSCI、EI收录的论文高达90篇。其主要贡献是提出了许多统计学习、机器学习方面的有效算法，如EM算法、PMM算法、朴素贝叶斯法、支持向量机SVM、决策树ID3、随机森林RF等，应用十分广泛。

总而言之，李航是一个对人工智能、机器学习、统计学习有着深厚功底的学者。她从小就受到激情地学习各种新知识、新理论，有着独特的视角，对新技术的发展始终保持敬畏之心，并乐于分享她所学到的知识，传播智慧。因此，与其说她是一位统计学家或是机器学习专家，不如说她是一位系统工程师和管理学家，同时又具备了探索精神、创新精神和学习能力，对新事物充满热情。


# 2.基本概念术语说明
# （1）监督学习
监督学习（Supervised learning）是指给定输入数据及其对应的输出标签（目标值），训练一个模型，使模型能够学习到映射关系，使得输入数据能够对应正确的输出标签，即模型能够预测新的、没有见过的输入数据的输出结果。监督学习由输入空间X和输出空间Y组成，输入X可以包括特征、文本、图像等，输出Y则是分类任务的标签或者回归任务的连续值。监督学习通常采用标注数据进行训练，其目标就是找到一个映射函数f: X -> Y，将输入映射到输出。目前监督学习可以分为以下两类：
1. 回归任务（Regression task）：目标是根据给定的输入变量预测一个连续实值输出，比如房价预测、气温预测等。
2. 分类任务（Classification task）：目标是根据给定的输入变量预测离散的输出变量，比如手写数字识别、垃圾邮件分类、病患诊断等。

# （2）无监督学习
无监督学习（Unsupervised learning）是指在没有给定输入数据的情况下，通过自组织的方式对数据进行聚类、分类、降维等处理，目的是寻找数据中隐藏的模式和结构，这种学习方式不需要有明确的输入输出匹配，而是对数据自发地形成结构。无监督学习通常也采用非标注的数据进行训练，但其目标不是预测输出结果，而是发现数据的内在联系和规律。目前无监督学习可以分为以下三类：
1. 聚类（Clustering）：目标是在给定的输入变量集合中找到尽可能多的相似样本，并将这些相似样本划入某些类别。例如，利用聚类方法可以把用户行为数据划分为“活跃用户”和“非活跃用户”，并对某些产品进行推荐。
2. 降维（Dimensionality reduction）：目标是降低输入变量的维度，简化数据结构，以便更好地分析和理解数据。例如，利用降维方法可以把高维的图像数据转化为二维图像，以便进行更加直观的分析。
3. 关联规则（Association rule learning）：目标是发现两个以上事件之间存在一定关联性的频繁项集，并据此推导出更多有用信息。例如，利用关联规则方法可以分析顾客在线购买习惯，以便优化营销策略。

# （3）半监督学习
半监督学习（Semi-supervised learning）是指既拥有少量的标注数据，又含有大量的无标注数据。为了完成训练任务，通常需要结合少量标注数据和大量无标注数据共同训练模型，促使模型在保证较高准确率的前提下，也能够获取到部分无标注数据的辅助信息。

# （4）特征工程
特征工程（Feature engineering）是指对原始数据进行转换、组合、过滤等处理，得到一个或多个用于训练的特征，用来预测或解决实际问题。特征工程的关键在于选择、设计和实现，它包括以下三个方面：
1. 特征选择：特征选择是指选择那些对于预测模型有用的特征，去掉无关紧要的、冗余的或噪声的特征，这样才能有效地训练模型，提升模型的效果。
2. 特征变换：特征变换是指将原始数据按照某种变换方式进行重新编码，如将文字描述的图像转换为数字表示；或是对连续型变量进行离散化、标准化等操作，将它们转换为一种易于处理的形式。
3. 特征抽取：特征抽取是指通过计算、挖掘、合并等方式从原始数据中提取出重要的、能够代表数据的特征，如人的脸部特征、图像的边缘等。

# （5）模型评估
模型评估（Model evaluation）是指通过测试数据集来评估模型的性能，判断模型是否达到了预期的效果。模型的评估一般包括以下三个方面：
1. 准确率（Accuracy）：准确率是指测试集上正确分类的样本数量与总样本数量的比值。如果准确率很低，意味着模型的预测能力差。
2. 召回率（Recall）：召回率是指所有正样本的数量与实际检出的正样本数量的比值。如果召回率很低，意味着很多负样本没有被检测出来，模型无法满足需求。
3. F1 Score：F1 Score是指准确率和召回率的调和平均数，值越接近1，说明模型的预测能力越强。

# （6）深度学习
深度学习（Deep learning）是指多层次的神经网络模型，能够自动提取特征、解决复杂的问题。深度学习一般包括以下三个方面：
1. 卷积神经网络（Convolutional neural network，CNN）：CNN是深度学习中的一种特殊模型，它在图像、语音、视频等数据中提取局部相关特征，从而取得优秀的效果。
2. 循环神经网络（Recurrent neural network，RNN）：RNN是深度学习中的一种模型，它可以捕捉时间序列数据中的长时依赖关系，从而取得优秀的效果。
3. 递归神经网络（Recursive neural networks，RNN）：RNN是深度学习中的一种模型，它能够建模多层次的结构，从而取得优秀的效果。
