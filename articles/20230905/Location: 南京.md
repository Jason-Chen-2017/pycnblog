# 1.简介
  

## 概览
### 什么是Location?
Location是一个基于机器学习和地图技术的定位技术平台，它通过对用户手机GPS数据的分析及与公共交通网络数据的结合，实现精准、时效性的城市定位功能。Location通过对数据采集、预处理、特征提取、模型训练等环节进行自动化处理，将位置信息作为一种服务向用户提供。
### 为什么要做这个项目？
随着移动互联网的普及，越来越多的人通过手机APP或者微信小程序来获取实时的地理位置信息，但这些应用一般只提供了基本的城市名称信息而已。实际上，在现代生活中，每个人的行踪轨迹可以反映其个人生活态度、消费习惯、社会影响力等方面的特征，如同一辆车在不同时间停下来又再次开动一样。因此，如何用GPS定位技术精准且实时地获取用户所在城市的信息就成为一个重要课题。同时，由于各个城市之间的公共交通信息、交通事件、气象变化、地铁站等动态信息的不断增加，精准、可靠的定位信息对社会经济活动的影响也非常重要。因此，如果能够基于公共交通、气象和地铁等多种数据源的海量数据，开发出具有高精度、时效性、高可用性的Location服务，将为全国各个城市的用户提供更准确、便捷、实时的定位信息服务。
### Location主要功能
1. 精确定位——采用机器学习方法对用户设备上收集到的GPS数据进行分析和预处理，将原始数据转换成易于理解、易于使用的格式；然后，利用Location自研的经典算法，进行特征提取，找出用户所处的具体位置区域和相关的地点信息，包括路牌号、街道名、商圈名、建筑物等。
2. 时效性——Location持续跟踪并更新各种数据源（包括GPS、公共交通、气象、地铁）中的最新信息，每隔一段时间就会自动进行一次“激活”，识别出设备的当前状态和所在位置，确保用户的位置信息始终是最新的、正确的、及时的。
3. 覆盖面广——Location平台支持超过17万个城市的覆盖范围，涵盖了北京、上海、广州、深圳、成都、杭州、厦门、苏州、西安、长沙、南京、济南、青岛、无锡、武汉、郑州等大中型城市。
4. 用户隐私保护——Location平台不会主动获取用户的任何个人信息，只有当用户授权后才能收集和分析设备数据，并且会严格遵守用户的个人隐私权。
5. 价格优势——Location服务的定价模式，目前采取按使用流量计费的方式。根据用户使用的流量大小，按照一定的固定价格向用户收费。未来，Location服务可能会逐步开放VIP版本，以适应付费客户的需求。
6. 技术领先——Location的基础设施技术已经得到业内多家公司的认可，采用了AI、大数据、云计算、人工智能、物联网等诸多领域的前沿技术，并积极探索更加高效、智能的解决方案。
7. 更贴近真实——Location不仅能精确定位用户所在的城市、地区，还能识别用户具体到几栋楼、几号单元或几号楼房等详细位置信息，能够帮助用户更加清晰地掌握自己的生活轨迹和周边环境信息。
