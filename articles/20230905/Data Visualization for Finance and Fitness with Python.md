# 2.基本概念术语说明
## 1.数据可视化（Data visualization）
数据可视化（英语：data visualization），是指利用图表、图像、演示文稿等各种媒体手段来呈现数据的一种手段。数据可视化的目的是为了让人更清晰地获取信息，更好地识别模式，帮助发现新的模式、趋势或问题。数据可视化的特点是直观易懂、有效呈现数据的特性。数据可视化有很多优点，但是也存在其局限性。总的来说，数据可视化是一种方法论，是把数据转化为可视化的形式，通过图表、图像、动画、视频等多种形式展示出来。数据可视化可以用于描述客观事物的特性，也可以用于揭示隐藏的信息，还可以用来引导决策。
## 2.数据类型
数据可视化的输入通常都是一个或多个数据集合。数据类型分为以下几类：
- 结构型数据：例如表格数据、数据库数据等；
- 非结构型数据：例如文本、音频、视频等非文字、非数字格式的数据；
- 半结构型数据：例如XML文件、JSON格式的数据；
- 时间序列数据：时间轴上的数据，时间相关的变量，例如股票价格、经济指标等；
- 地理空间数据：地理位置及分布特征的数据，例如地图、航空航天、海洋等；
- 多元数据：综合了以上类型的数据。
## 3.图表类型
常用的图表类型包括：
- 折线图、条形图、面积图、散点图：主要用于表示一组数据的变化趋势。
- 柱状图、箱线图：主要用于表示统计量的分布。
- 饼图、雷达图：主要用于表示数据的百分比分布。
- 词云图：主要用于快速显示文本的关键词。
- 颜色映射：将数据编码为不同的颜色，突出重点。
- 矩阵图：展示两个或多个变量之间的关系。
- 树图：展示树状结构数据。
- 旭日图：主要用于展示数据的分布和变化。
- 平行坐标系：主要用于比较多组数据之间的差异。
- 热力图：展示二维数据中的热度分布。
- 象形柱图：主要用于展示一组数据的分类情况。
