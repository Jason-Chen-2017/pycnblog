# 2.基本概念术语说明

## 什么是 Go？

Go 是 Google 于2007年9月开源的一种静态强类型、编译型、并发的编程语言，由一组简单而又高效的编程概念和语法组成。设计 Go 的目的是为了实现一个简单、有效且高效的编程环境，以支撑 Google 海量级应用的开发，并逐步发展成为更加接近计算机科学真正的编程语言。截至2020年6月份，Go 已被超过三分之一的知名互联网公司采用，在云计算领域、容器编排领域、微服务架构、机器学习、爬虫数据处理等多个行业得到广泛应用。

## 为什么要学习 Golang？

1. 跨平台能力：Go 语言可以在多种平台上运行，如 Linux、macOS、Windows、FreeBSD 和 NetBSD，并且能够实现自动适配。此外，Go 还拥有丰富的标准库，可以轻松编写底层系统调用或访问硬件设备接口。同时，由于 Go 提供了垃圾回收机制和自动内存管理功能，使得它具有比其他静态语言更高的执行效率。

2. 安全性能：Go 语言受益于其安全性、并发性和静态强类型特性，是构建健壮、高性能、可靠应用程序的不二选择。相对于 C++ 或 Java，Go 更容易编写不确定性的代码，而无需担心内存泄漏或其它危险因素。同时，Go 支持 Goroutine 和 channel 机制，使得编写异步代码变得十分方便。

3. 工程化能力：Go 的包管理器 govendor 可以自动处理依赖项，使得项目构建和测试更加方便，另外，还有官方工具集 goimports、gofmt、golint 等可以提升代码质量，使得编码更加规范、统一。

4. 大规模协作：作为 Google 一员，Go 语言社区已经积累了一大批开源贡献者，有大量优秀的开源项目可以使用。国内也有一些知名企业，如七牛、京东、美团等在生产环境中使用 Go 开发各种应用。同时，Google 根据 Google 大脑处理搜索请求的需求，搭建了自己的 Golang 代码搜索引擎，以提供更准确地搜索结果。

5. 生态系统：Go 语言虽然较新，但它的生态系统正在蓬勃发展。目前 GitHub 上有很多开源项目都是用 Go 语言编写，如 Prometheus、Delve、Kubernetes、Docker 等，这些项目都极具活力，值得关注。

6. 更好的学习体验：与其他静态语言相比，Go 的语法结构和习惯用法都比较简单，而且易于学习。因此，学习起来会更加容易一些，并且能够快速掌握基础语法。
