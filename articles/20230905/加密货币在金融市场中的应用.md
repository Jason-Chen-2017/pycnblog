# 1.简介
  

## 1. 什么是加密货币？
加密货币（Crypto Currency）即加密数字货币，也称加密数字钱包或加密数字货币系统，它利用密码学方法对数字货币进行保密处理，并通过网络传输流通。最早期的加密货币系统是Bitcoin，其后逐渐发展出了类似的产品，例如Litecoin、Ethereum等。
## 2. 为何要用加密货币？
传统上，货币的所有权是不受到任何掌控的，一旦发生了信用危机，银行等金融机构就会出现严重的损失。另一方面，由于货币的储备量有限，价格波动较大，用户无法预测将来的价值。
随着互联网的普及，各种金融工具被开发出来，能够实现匿名化和可追踪。不过，这些工具存在隐私风险，数据无法真正地被保护，人们容易丧失信心，因此需要一种新的解决方案。而加密货币正是一种基于密码学方法保障交易数据的安全、不可篡改的数字货币。
## 3. 加密货币产生的背景
在2017年中本聪（Satoshi Nakamoto）发布了比特币白皮书，设想了一个可以点对点交易的数字货币体系。白皮书主要阐述了如下四个方面的内容：

1. 点对点的交易机制：在点对点交易的环境下，双方之间的数据交易完全不需要第三方担保。

2. 没有中心化运营：没有一个集中的服务器来存储所有数字货币的账户余额和交易信息，而是由每个参与者独立运行自己的节点，相互通信完成交易。

3. 不依赖于货币当局的审批：基于公开且开源的代码，节点可以在不依赖任何政府的许可或许可证的情况下运行。这样就降低了货币当局的监管风险，增加了系统的健壮性。

4. 分散记账：采用分片的方法来确保整个系统不会因单个节点的日益增多而成为网络性能的瓶颈。

基于以上理念，自然会产生一种叫做“去中心化”的新词汇。但是，加密货币并不是一个完全的去中心化系统，它仍然存在中心化的部分，只是它的运作方式与其他去中心化系统截然不同。
## 4. 加密货币的运作原理
加密货币的原理比较复杂，但总的来说，就是把加密算法和密码学知识引入到数字货币的交易领域。简单地说，它是一种基于密码学的支付系统，所有交易记录都经过加密处理，不可以被普通的人所获取。目前，主流的加密货币算法包括RSA、ECC(椭圆曲线)、ECDSA(椭圆曲线签名算法)。

加密货币的交易流程非常直观，买方需要先付款给卖方，而且过程是高度加密的，不存在任何中间商赚差价，而卖方只需要确认收款即可，无需提供货物，提高了交易效率。此外，因为所有的交易记录都是加密的，所以即使黑客攻击了某家支付平台，也很难从交易记录中获取有效的信息。

加密货币还有一个非常重要的特性——可以进行数字身份认证。也就是说，每一个加密货币地址都对应唯一的一串数字，只有拥有这个地址的持有人才能在网络上进行交易，而其他人无法知道该地址对应的真实身份。这一特性可谓是“一次比一次危险”，因为一旦某个公司控制了某个地址，那么整个网络上的交易都会受到影响。因此，个人还是建议不要轻易使用加密货币，更推荐大家使用稳定可靠的支付系统，比如支付宝、微信支付或者银行转账等。

当然，加密货币还有很多其它优势，诸如匿名性、抗审查、去中心化、匿名支付等，这里就不再一一列举了。
