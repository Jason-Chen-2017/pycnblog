# 1.简介
  

随着深度学习（Deep Learning）技术的迅速发展，越来越多的人开始利用深度学习技术进行日常应用。而作为人工智能领域的顶尖专家之一，李飞飞教授也因此受到不少关注。他在他的一系列文章中提出了三点关于人工智能领域的公共价值观，即数据、人性、变通四个公共信仰，并与他所推崇的“齐一意”深度学习模式做了一个对比，认为要实现这一目的需要选择合适的机器学习算法。
但是，当李飞飞教授开篇讲述他对两者的不同看法时，没有直接给出他选取哪一种算法的具体原因。这次的实验会试图回答这个问题。作者希望通过实验研究发现，不同的人对于采用什么样的算法更有利于实现他的公共价值观，进而帮助他设计出一个更好的人工智能系统。

# 2.数据、人性与变通
数据、人性与变通是李飞飞教授提出的公共价值观。通过研究，可以发现每个人都有自己的偏好，并且这些偏好并不能完全客观地反映现实世界中的事物。此外，人类天生具有情绪甚至动机，会产生偏好或习惯，也就是说，即使存在客观的真相，人类也无法避免受到它们的影响。最后，因为环境变化、资源消耗等诸多因素的影响，很多时候人们会尝试突破性的解决方案——变通。为了最大限度地发挥人的潜能、克服人性中的恶习、保护环境、有效利用资源，人类不断努力追求自我完善、超越自我的梦想。
因此，理解并尊重人的基本需求，充分尊重他人价值观、喜好和想法，认同其可能产生的个人主张和行为，并不断保持开放心态，这是人工智能领域应当努力遵守的基本原则。而另一方面，为了从根本上减轻人工智能系统的风险和负面效应，防止出现危害人类健康、环境污染等问题，应该采取措施加强对数据的保护、不断更新算法模型，以及部署可靠的监管机制和管理体系，这才是人工智能领域应该注重的核心任务。

# 3.选择哪种算法？
目前，大多数人工智能系统的开发都是基于监督学习或者半监督学习技术，也就是用已有的大量数据训练一个模型，根据新的输入样本预测输出结果。这也是李飞飞教授给出的建议。但是，李飞飞教shkar的一系列文章的中心议题其实已经从两个算法中区分出来。首先，如果考虑到算法的准确率和效率，那么应该优先选择无监督学习算法。其次，虽然监督学习算法在某些领域表现更优秀，但同时还存在着一些缺陷。如过拟合、欠拟合、泛化能力差等。因此，在实际应用场景中，需要结合实际情况进行权衡，选择最适合当前问题的算法。这也是李飞飞教授对监督学习和无监督学习算法的比较。
