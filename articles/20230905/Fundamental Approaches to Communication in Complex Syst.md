# 1.简介
  


随着复杂系统科学的蓬勃发展，越来越多的人开始关注如何构建复杂系统，如何设计和运行它们的交互性、自组织能力及其控制策略等。同时，越来越多的复杂系统开始引入反馈机制作为复杂系统中的重要组成部分，而人们也希望能够更好的理解反馈的作用及其在复杂系统中的角色。本文将讨论反馈在复杂系统中的基本原理、特性和作用，并进一步阐述通过研究反馈的结构及其相互关系，从而帮助设计人员更好地认识和利用反馈来提升复杂系统的交互性、自组织能力及其控制策略。

# 2.基本概念术语说明

## 2.1 概念

**Complex System** (复杂系统)是一个由多个相互关联的元素构成的系统，这些元素之间的相互作用及其导致的行为通常具有高度复杂性。其典型特征包括复杂结构、动态演化、复杂控制以及高度非线性的非确定性。复杂系统中不仅存在着自然界中形成的复杂系统如城市、经济网络以及机器网络等，也存在着人类社会中形成的复杂系统如公司、组织、国家、社会等。通过对复杂系统进行分析、设计、控制和优化，人们可以提高效率、降低成本，改善社会福利，防止危机等。

**Feedbacks** (反馈)是指系统状态的一系列输入或输出信号，引起系统内部其他部分的状态变化。反馈机制是复杂系统中非常重要的组成部分，它是人类学习、工作、活动、决策等各个方面的关键要素。反馈机制不但可以对系统进行稳定控制、协调调节，而且还可以产生规律性、经济性和生物性等多种优势。在一般意义上，反馈可以分为两种类型：

1. Positive Feedback：正向反馈是指系统内外状态之间的关系，当系统状态改变时，系统的另一个状态也会发生相应的改变。典型的正向反馈如温度升高导致空气湿度下降、人的注意力分配方式影响个人行为等；
2. Negative Feedback：负向反馈则是指当系统状态改变时，外部环境会产生反馈，使得系统状态继续改变。典型的负向反馈如汽车里程表读秒导致减速、热量浪费导致体温升高等。

## 2.2 相关术语

**Agent-based Modeling** （基于代理模型）: Agent-based modeling (ABM) 是一种模拟复杂系统的方法，其中每个个体（Agent）都是一个可独立思考和行动的实体。ABM 最初是为了描述行为学、计算生物学、心理学以及心理物理学等领域所需的复杂系统建模方法，如细胞、神经网络、生命形式、计算机网络、群落模型、社会系统等。由于 ABM 模型倾向于用人类的方式模拟，因此也被认为是一种客观模拟方法。在现代应用中，ABM 在很多领域都有很广泛的应用，如交通、金融、工程技术、医疗保健、物流管理、资源分配、社会经济、科技开发等。

**Adaptive Control** (自适应控制): Adaptive control 是指根据系统实际情况调整系统参数、软硬件配置、设备设计、人员培训等，以优化系统性能、稳定性和效率，确保系统在各种情况下始终处于最佳状态。 adaptive control 的目的是让系统在各种条件下都能有较好的表现。

**Autonomous Agent** (自治agent): Autonomous agent（自治agent）是指可以完全自主地做出行为而不需要外部干预的智能体。它可以独立地运作、选择自己的行为，不需要依赖其他人的指令。Autonomous agent 可以为复杂系统提供更加精准的、高效的决策、操控和信息传递。

**Behavioural Modeling** (行为模型): Behavioural modeling (BM) 是一种用来模拟和理解复杂系统行为的模型。BM 方法往往由一些简单规则组成，例如判断哪些因素影响系统的行为，以及系统在什么条件下采用怎样的行为。BM 方法主要用于从系统的角度来理解系统的运作、演变以及各种反馈的作用。

**Criticality Theory** (临界性理论): Criticality theory (CT) 是指用来刻画系统临界点的研究范式。系统临界点是指系统进入某个状态或状态区域时的临界值。临界性理论主要研究系统不同状态、模式或变量之间复杂的相互关系以及对系统整体稳定性的影响。CT 有助于识别、分类和预测复杂系统在不同情景下的关键性节点、转折点和边界。

**Design for Complexity** (复杂系统设计): Design for complexity (dcf) 是指为了应对复杂系统带来的挑战，制订复杂系统的设计目标、规范要求和评估方案等。其目标是建立对复杂系统的完整、科学、可控的描述，以促进系统的设计、开发、测试、运营和维护等。DCF 方法的实施可以有效地避免系统的过度设计、缺乏合理控制和有效性保证，并且通过客观、合理的评估结果以证明系统的设计目标、效果、效益和成本效益。

**Stochastic Dynamic** (随机动态): Stochastic dynamic 是指系统的状态变量随时间的演化符合概率分布的过程。随机动态可以更好地反映真实世界的系统行为，更好地适应系统的复杂性和不可预测性，为系统控制提供更可靠、鲁棒的工具。

**Emergent Properties**: Emergent properties 是指系统在不经意的情况下由多个个体的共同作用的结果。通常情况下，系统中的某些特性是由系统的一些基本属性所决定的，这些特性在初始阶段是无法察觉到的，只有系统在运行过程中逐步出现。在系统的不同方面都可能出现这种现象，包括规则、结构、联系、动态、制度等。Emergent properties 提供了系统设计者和管理者以前所未有的视角来看待系统。
