# 1.简介
  

深入理解计算机系统（原书名：Computer Systems: A Programmer's Perspective）是一本开放性、全面性的著作，作者是爱德华·海克斯（Edward A. Harris），该书共7章，从最基础的计算机组成原理到高级语言系统实现细节，全面系统地探讨了计算机软硬件体系结构、程序设计方法、编译和运行过程、存储管理、文件系统等方面的内容。书中对程序员日常工作和生活中的应用提供了广泛的教学，并在作者精心实践的过程中提出了许多有价值的观点和经验。
## 作者简介
爱德华·海克斯（Edward A. Harris），1973年出生于美国加利福尼亚州旧金山市，现任微软公司高级技术人员兼软件工程师。他主要研究生物信息学、计算机科学及其应用领域，先后就读于清华大学和斯坦福大学。他是IEEE主管、ACM会长、计算机行业协会的会员，曾获图灵奖，并出版过多部计算机相关的著作，包括《浅谈软件工程》、《程序设计的艺术》、《精通UNIX环境编程》、《Unix下的C程序设计》等。

# 2.内容概述
计算机系统由硬件和软件两个层次组成，是运行应用程序所必需的必要组件。软硬件之间如何相互作用、合作，又是整个系统运转的关键所在。本书通过分析软硬件的整体架构，深入研究计算机的软硬件体系结构，从最底层的CPU指令集，到虚拟内存系统、网络通信协议，再到操作系统内核，系统软硬件各个层次之间的交互关系，最终给出程序员应该具备的知识和能力，能够利用这些知识快速开发出更加可靠和稳定的软件系统。书中还涉及了多种形式的算法和数据结构，如排序算法、查找算法、动态规划、贪婪算法、博弈论、概率论、线性代数、数论等，对于学习和掌握这类知识是非常重要的。

本书具有以下特点：
- 深入浅出，从基本原理出发，全面剖析计算机系统的软硬件组成及其相互作用；
- 技术深度，系统地探究软硬件之间复杂而微妙的相互作用；
- 创新见解，作者对计算机发展及其应用领域提供独到的见解；
- 专业严谨，注重对各种算法和数据结构的分析、设计、实现；
- 丰富的实例，深入浅出地描述软硬件系统的运行机制；
- 涉及范围广，兼顾计算机科学和软件工程两方面内容；
- 适合作为大学计算机课堂教材或进阶教材。


# 3.目录
本书共7章。第一章从程序执行的角度介绍计算机的软硬件体系结构，包括计算机机构、指令集体系结构、程序计数器、处理器指令、ALU、寄存器、存储器、输入/输出设备。第二章详细介绍微处理器结构，包括微指令、微地址译码、微布尔运算、微数据路径、微控制单元、微状态机。第三章重点介绍CPU架构，包括流水线结构、乱序执行、指令调度、分支预测、物理内存管理、异常控制。第四章介绍存储器体系结构，包括SRAM和DRAM、主存的层次结构、虚拟内存系统。第五章介绍系统总线结构，包括总线事务、主动、被动消息模式、信号传输速率、时延等。第六章探讨多核处理器结构，包括多处理器间同步、缓存一致性、多线程、任务调度、超线程、监督器等。第七章讨论文件系统、数据库系统、操作系统及安全性等。
