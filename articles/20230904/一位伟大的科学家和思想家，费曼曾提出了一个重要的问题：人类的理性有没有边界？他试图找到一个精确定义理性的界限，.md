# 1.简介
  

费曼（英国心理学家）是一位有影响力的心理学家和哲学家。费曼的贡献主要集中在两个方面：第一，他提出了一个重要的问题，“理性有没有边界”，并在此基础上探索了“真理”、“知识”、“理性”以及“超越自我的”等概念的内涵及其之间的关系；第二，他强调科学的重要意义，在他看来，科学是最可靠、可信赖的东西，任何可以证实的观念都必须依赖于科学。在这本书中，费曼就其研究成果进行了全面的阐述。
# 2.历史回顾
19世纪末，英国哲学家皮克，加上苏格拉底的启蒙思想，形成了对理性的信仰。费尔巴哈之后，理性成为英国的官方语言。但是，费曼认为这种理性的信仰是不够的，理性还应该具备其他的特质，包括“诚实”、“谦逊”、“适度”等，这些特质才构成了真正的“理性”。因此，他建立了自己的学说——系统理性学说。
# 3.理性与理论的区别
费曼指出，“理性”和“理论”是两种不同的概念。理性是指心智具有主动思维能力、解决问题能力、独立判断能力和解决现实问题的能力，是一种反映客观世界本质的能力。理论则是指一套规范，它把事物或人类活动的各种可能情况都考虑清楚，而不管它们是否合理或正确。比如，某个法律理论，就用来规定如何应付违反该法律的人群。理论是无法经验检验的，只能通过分析来确定它的真实性。
费曼认为，理性和理论之间存在着根本性差异。理性是真正的能力，理论是纯粹的知识。只有了解真理的机制才能真正的认识宇宙、理解世界，这就是费曼所说的“宗教化”现象。所以，要成为真正的科学家，就需要把两者分开。
# 4.人的理性有没有边界
费曼从另一个角度阐述了人的理性的概念，即“人类理性的产生方式”。他认为，人的理性是可以被社会所塑造的，并且其起源可以追溯到人类进化的早期阶段。为了认识人的理性，费曼认为，首先要弄清楚它的构成要素。“理性”由四个要素组成：
- 主动性：人的理性具有主动性，即它能够采取行动来创造新知。这一点体现在许多文明古老的国家里，人们为了建设新技术和人口锐减，往往会创造出新的方法来解决已有问题。这也是为什么古代农业、手工艺和医疗都能够得以延续的原因。
- 灵活性：人的理性具有灵活性，能够将各种信息综合利用，以达到有效决策。这是因为人的直觉、感觉、嗅觉以及运用语言、符号等信息获取新知识的能力都是人类的优势。
- 抽象性：人的理性具有抽象性，能够通过比较和概括等方法来处理复杂的现实世界，从而获得对问题的整体观察。
- 批判性：人的理性具有批判性，能够做到客观公正地评价和分析事物。比如，对于某个政策建议，人们通常会提出多个角度、层次的观点，从而给出不同的评价结果。

如果考虑到人的行为模式、生物特性、环境条件等因素，就能够发现，人的理性其实是一个不断循环、不断学习、不断进步的过程。而一个社会的发展又会影响其人们的行为模式、理性水平。由于人的理性具有不确定性、不完整性，所以，也就不存在一个确切的边界。这也表明，理性和理论是不能割裂的。
# 5.系统理性学说
费曼的系统理性学说认为，理性是依赖于一些基本原则，这些原则直接或间接地影响着人的行为方式、认识能力、决策能力、情绪控制等。而系统理性学说指的是，存在某种统一的理性体系，其原则固定不变，不可改变。这一系统理性体系由若干认知系统、行为系统、决策系统以及情绪系统组成。其中，认知系统负责获取信息并加以整理、组织，然后通过分析、推理、归纳等过程，形成对问题的有机认识；行为系统负责运用这些认识形成行动方案；决策系统则负责进行最终的决策；情绪系统则负责管理人们的情绪状态。
基于这样的理性模型，费曼提出了如下几个假设：
- 第一，每个人都有一个偏好系统。偏好系统由一系列偏好和目标组成，这些偏好和目标是长期形成的，并受到个人经历、社会环境、团体精神等多种因素的影响。不同的人往往对同样的事务具有不同程度的偏好。
- 第二，人的情绪也是由各项感官所感知得到的，而不是凭空产生的。也就是说，情绪是可以影响人类行为的。
- 第三，人们有选择的自由。这是由于人的理性并非是一成不变的，而是依赖于时代、社会的发展而不断进化的。
- 第四，存在着许多类似的行为模式，这些行为模式是由人类的天赋、习惯、兴趣、个人需求、周围环境、训练以及其他因素共同决定的。
- 第五，所有的行为模式都可以用概率的方法进行分析和预测，并以此作为决策依据。
# 6.“超越自我的理性”
系统理性学说在一定意义上还可以延伸，即超越自我的理性。费曼认为，超越自我的理性包含三层意思：
- 第一，超越自我的理性指的是超越个人自我的理性，即超越每个个体独有的理性。费曼认为，每个人都有自己固定的认识和决策系统，这种系统是在演化过程中形成的，并不是随着时间流逝而发生变化的。相反，我们生活中的绝大多数现象都可以追溯到原始人类的历史。
- 第二，超越自我的理性意味着超越个人所做出的判断。费曼认为，我们总是倾向于低估个人的知识、技能、经验等，而忽视他人对我们的理解和帮助。如果我们能够超越自我的思维模式，那么我们就可以看到许多之前看似不可思议的现象，而这些现象是由人类过去一段时间积累的经验和知识所导致的。
- 第三，超越自我的理性也可以称作“永恒的自我”理性，或者叫做“终极的主观理性”。超越自我的理性的关键是要把握现状，不要局限于过去和当下的经验。如果我们能够充分尊重事物的普遍性，从而洞悉所有事物的本质，那么我们就能有更深刻的理解、更准确的判断，而不会陷入愚蠢和盲目之中。
