# 1.简介
  

## 为什么要写这个课程？
首先，写给非计算机专业的人群来说，计算机科学本身就是一个非常高深的学科。但对于新手来说，了解一些基础知识、概念，至少可以让他们对编程有个大概的认识，从而更容易理解各种编程语言、框架或工具所解决的问题。另外，很多优秀的开源项目都是基于python编程语言开发的，掌握python的相关知识会使得自己对python语言有更好的理解。因此，本文将以《写给iOS开发者的Python入门课》这一系列文章的形式，向读者进行python编程的初步介绍和学习。希望通过阅读本系列文章能够让大家对python有所了解、有所感悟，进一步提升自己的python编程能力。
## 涉及哪些内容？
由于篇幅原因，本文不可能做到面面俱到，因此我将本文分为以下几个部分：
### Python基本语法、运行环境配置
主要介绍了python的基本语法和运行环境配置。

- 安装python
- 交互式命令行窗口（interactive shell）
- python源码文件结构
- 模块、包、命名空间
- 数据类型、运算符
- 函数定义、调用
- 条件判断语句、循环控制语句
### 数据分析实践
主要介绍了python在数据分析方面的一些用法。

- 基本数据结构——列表、元组、字典
- numpy库的使用
- pandas库的使用
- matplotlib库的使用
- scikit-learn库的使用
### web开发实践
主要介绍了如何利用python实现web开发。

- Flask框架的使用
- Django框架的使用
### 机器学习实践
主要介绍了python在机器学习方面的一些用法。

- 分类模型——逻辑回归、决策树、支持向量机
- 聚类模型——K均值聚类、DBSCAN聚类
- 降维模型——PCA、LDA、tSNE
- 深度学习框架——TensorFlow、PyTorch等
### 结语
本系列文章是《Python学习指南》的第六章节。在这章节中，我将结合自身经验，详细介绍python在数据分析、web开发、机器学习领域中的应用。期望这系列文章能够帮助读者快速上手python，掌握python在各个领域的应用技巧。如果您还有其他想法、建议，欢迎随时联系我进行沟通。
