# 1.简介
  

## 2.1 概述
智能驾驶(Autonomous Driving)是新一代人工智能领域的重点方向之一。随着传感器、图像识别、计算机视觉、机器学习等一系列人工智能技术的不断革命，以及未来自动驾驶汽车、电动汽车的到来，智能驾驶将成为一个互联网企业必备的技能。近年来，智能驾驶领域涌现出许多创新性的产品和服务，如汽车、Uber等服务于用户习惯的自动驾驶汽车。阿里巴巴在智能驾驶领域一直是领跑者，目前是全球最大的搜索引擎公司，也是国内市场上最早进入智能驾驶领域的互联网企业之一。
而华为作为世界五百强IT企业中规模最大的公司，在智能驾驶方面也是一个引领者。华为从2011年建立智能制造和智能交通两个子品牌之后，积极参与智能驾驶领域的竞争，率先实现了“量产汽车”项目。华为致力于推动智能驾驶产业向前发展，早已成为全球领先的市场化企业。华为有能力扛起智能驾驶新时代带来的巨大挑战，并持续提供有效的解决方案。
基于此背景，华为与阿里巴巴共同推出了一个名叫“智能驾驶导航系统”（英文名Smart Navigation System）的产品，通过智能地图和计算技术，帮助驾驶员更高效、安全、省油、节省时间，乃至免费开车。
## 2.2 业务背景
目前，智能驾驶领域主要分为两大类产品：
1. 交通导航产品：主要包括电动汽车、自主型汽车、卡车、公交车等，是驾驶员的第一辆交通工具。导航产品需要根据周边环境实时获取信息并指导驾驶员如何正确行驶，同时还需要考虑到道路情况、交通情况、红绿灯、停车位置、拥堵状况等因素影响驾驶安全。
2. 城市交通产品：主要包括地下城市地铁、出租车、快车、出口车等，是驾驶员非第一辆交通工具。由于在小区、地铁站、商场等复杂的区域驾驶，城市交通产品的导航要保证驾驶员安全、畅通、高效。
其中，导航产品的定位是指导驾驶员减少乘车次数，提升驾驶效率，降低成本；而城市交通产品的定位是增强驾驶员的交通能力，提升出行速度，保障出行安全，促进经济发展。
## 2.3 功能需求
### （一）驾驶员的使用场景及目标
作为中国互联网时代的第六大产业，智能驾驶作为核心竞争力领域，正逐渐成为越来越多的行业需求。当前，国内外众多互联网公司纷纷推出智能驾驶相关的产品或服务，如滴滴打车、京东驾驶、顺风车等。截止2019年5月，全国智能驾驶的相关产品、服务超过三千款，比上年同期增加了5%左右。
但事实证明，绝大多数人的日常生活场景都无法在短时间内满足驾驶需求。例如：在山区、机场或公路上，驾驶员需要快速准确地定位自己的位置，进行全面的导航指示，准确定位自己的目标并规划出行方案；而在异地驾驶时，驾驶员需要快速、轻松地切换目的地。因此，基于多场景需求，华为和阿里巴巴合作打造了一个智能驾驶导航系统，通过智能地图和计算技术，帮助驾驶员更高效、安全、省油、节省时间，乃至免费开车。
### （二）智能驾驶导航系统业务需求
- **提供交通工具的初始定位**：首先，通过手机或者其他方式对车辆所在地进行定位，确保导航系统的准确性；
- **用户交互界面优化**：其次，优化用户交互界面，增强用户体验；
- **提供指定路径的规划**：为驾驶员提供车辆行驶路径的规划，帮助驾驶者规避拥堵、选择路线并规划出行路线，提高行车效率；
- **提供历史轨迹的查询**：允许驾驶者查询过去一段时间的车辆行驶记录，帮助驾驶者了解自己的行车习惯和驾驶技巧，提升行车效果；
- **智能推荐路线**：通过对驾驶者的行车习惯分析，提供智能化的路线推荐，让驾驶者用最少的时间、花最少的钱就可获得高质量的驾驶体验；
- **免费试乘功能**：提供试乘功能，用户可以免费预约路测试乘，对驾驶者进行短期训练，确保驾驶技能的培养；
- **提供交通出行注意事项提示**：为驾驶者提供交通出行注意事项提示，避免出现恶劣天气对行车的影响，帮助驾驶者顺利完成出行任务；
- **提供地铁导航提示功能**：为驾驶者提供地铁导航提示，帮助驾驶者高效地利用地铁抵达指定地点，提高出行效率。
