### 5.2 源代码详细实现

```python
import bluetooth

# 扫描附近的蓝牙设备
nearby_devices = bluetooth.discover_devices(lookup_names=True)

# 遍历扫描到的蓝牙设备
for addr, name in nearby_devices:
    print(f"找到设备：{name}，地址：{addr}")

# 设备配对
device_address = "00:11:22:33:44:55"
bluetooth_device = bluetooth.BluetoothSocket(bluetooth.RFCOMM)
bluetooth_device.connect((device_address, 1))

# 数据传输
sensor_data = "温度：25℃，湿度：60%"
bluetooth_device.send(sensor_data.encode())

# 关闭连接
bluetooth_device.close()
```

### 5.3 代码解读与分析

上述代码首先使用`discover_devices()`函数扫描附近的蓝牙设备，然后遍历扫描到的蓝牙设备，打印设备名称和地址。接下来，通过`connect()`函数与指定的蓝牙设备建立连接，通过`send()`函数发送传感器数据，最后关闭连接。
