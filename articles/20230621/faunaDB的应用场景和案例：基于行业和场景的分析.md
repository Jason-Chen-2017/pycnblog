2. 技术原理及概念

2.1. 基本概念解释

 FaunaDB 是一款基于文件系统的分布式文件数据库，采用了分布式数据库的架构模式，将数据分布在多个节点上，通过冗余备份和数据复制来保证数据的高可用性和可靠性。

 FaunaDB 采用 B-tree 索引来对文件进行搜索和查询，支持多种文件格式，如 DFS(分布式文件系统)、SDFS( scalable distributed file system)、HDFS(Hadoop 分布式文件系统)等，同时还支持多种数据库模式，如关系型数据库、非关系型数据库、键值对数据库等。

 FaunaDB 采用了 GPT(Generative Pretrained Transformer)模型作为主节点的 ORM(Object-Relational Mapping)，支持 SQL 查询和大部分非 SQL 查询。同时，还支持对文件进行高并发读写和异步操作。
