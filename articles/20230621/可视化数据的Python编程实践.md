4. 应用示例与代码实现讲解

这里我们使用Python实现了一个散点图示例。首先，我们导入Seaborn库，并使用其函数绘制一个散点图。

```python
import seaborn as sns
import matplotlib.pyplot as plt

# 加载数据
data = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

# 绘制散点图
sns.set_style('whitegrid')
sns.plot_饼图(data, title='Data Visualization')
plt.show()
```
