# 2.机器学习的基本概念、分类模型、概率图模型
## 2.1 概念理解
机器学习（ML）是指利用数据训练计算机模型，使模型具备学习、推理能力，从而解决特定任务的方法学科。机器学习由监督学习、非监督学习、半监督学习、强化学习四个子领域组成。
### （1）监督学习
监督学习是机器学习的一个重要任务类型，其目标是在给定输入的情况下预测输出值或类别。监督学习包括回归分析、分类问题和标注学习三种方式。回归问题是预测连续变量的值，常用于预测房价、销售额等数量型变量；分类问题是预测离散变量的值，常用于区分不同类的图像、文本或视频片段；标注学习是同时预测多个变量的值，即同时预测多个标签。

在监督学习过程中，我们通常会有以下几个步骤：
1. 数据集准备：收集数据，清洗数据，划分数据集、验证集和测试集。
2. 模型选择：选择一个适合任务的模型，如线性回归、逻辑回归等。
3. 模型训练：通过训练数据，学习模型参数，使得模型能够对未知数据进行有效预测。
4. 模型评估：使用验证集或测试集对模型效果进行评估，评估指标包括准确率、召回率等。
5. 模型调优：针对模型效果不佳或模型过于简单时，可以通过调整模型的参数、添加正则项、引入其他特征等方式优化模型性能。

### （2）非监督学习
非监督学习是机器学习的一类任务，它的目标是在没有任何明显的标记信息的情况下，对数据进行聚类、降维或者生成概率分布模型。一般来说，非监督学习可以认为是一种更高级的监督学习，因为它不直接依赖于结果标签进行训练，而是借助于数据的内在结构进行学习。例如，在无标签的数据集上，可以使用聚类、生成式模型等方法对数据进行建模。

在非监督学习过程中，我们通常会有以下几个步骤：
1. 数据集准备：收集数据，清洗数据，划分数据集、验证集和测试集。
2. 模型选择：选择一个适合任务的模型，如K-均值、谱聚类等。
3. 模型训练：通过训练数据，学习模型参数，使得模型能够对未知数据进行有效聚类、降维或生成概率分布。
4. 模型评估：使用验证集或测试集对模型效果进行评估，评估指标包括轮廓系数、互信息等。
5. 模型调优：针对模型效果不佳或模型过于简单时，可以通过调整模型的参数、引入其他特征等方式优化模型性能。

### （3）半监督学习
半监督学习是机器学习中的一种策略，其目标是在有部分已知标签的情况下，对数据进行训练。其基本思路是通过人工标注的少量样本学习到一些全局性的规律，再通过规则或模型的方式去学习剩余的未标注样本，提升预测准确率。比如，在新闻文本分类中，有的样本只有摘要，而没有完整的文章文本，这时候就可以使用半监督学习的方法，利用少量已有标签的新闻，学习文本的主题、事件关系等特征，再基于此建立模型预测缺失的文章所属类别。

在半监督学习过程中，我们通常会有以下几个步骤：
1. 数据集准备：收集数据，清洗数据，划分数据集、验证集和测试集。
2. 模型选择：选择一个适合任务的模型，如朴素贝叶斯、支持向量机等。
3. 训练模型：首先利用人工标注的少量样本训练模型，得到模型参数。接着，利用剩余的未标注样本，通过规则或模型的方式学习这些未知数据的标签，提升模型的预测能力。
4. 模型评估：使用验证集或测试集对模型效果进行评估，评估指标包括准确率、F1分数等。
5. 模型调优：针对模型效果不佳或模型过于简单时，可以通过调整模型的参数、引入其他特征等方式优化模型性能。

### （4）强化学习
强化学习（Reinforcement Learning，RL）是机器学习中的另一种子领域。该领域的目标是基于环境的奖赏机制，在每一步中，智能体（Agent）接收到环境状态的观察，并根据这一信息进行决策，在下一步中执行动作，获得一个奖赏，并反馈给环境。这个过程重复迭代，直至智能体学会最大化长期的奖赏。

在RL中，智能体与环境之间的交互称为"转移（Transition）"，即智能体在当前状态执行动作后进入下一个状态，这一过程被记录成一系列的"轨迹（Trajectory）"。根据经验的积累，智能体会根据"轨迹"中某些特定的片段来改善它的策略，使之在类似环境中表现得更好。

RL的三个核心问题是：
1. 回合制（Episodic）：指每个episode（由智能体与环境交互的一段时间）结束后重新开始，这就要求智能体必须在某个任务上达到一个终止条件才算完成任务，否则就会一直循环。
2. 时序差分（Temporal Difference）：指智能体在当前状态采取行为a，会受到之前状态s和动作a的影响，下一步状态s'和奖励r的影响。
3. 马尔可夫决策过程（Markov Decision Process，MDP）：指智能体只能在当前状态s下做出动作a，不能看见后面的状态s’。
