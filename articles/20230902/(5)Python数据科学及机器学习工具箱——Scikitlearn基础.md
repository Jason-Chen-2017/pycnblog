# 3.基本概念术语说明
## 3.1 Python编程语言
Python 是一种开源的高级编程语言，其设计具有简单性、易学性、可移植性、功能丰富性等特性。自诞生以来，它被广泛使用于数据科学、网络爬虫、Web开发、科学计算等领域。

## 3.2 NumPy、pandas和Matplotlib
NumPy（Numerical Python）是一个基于 Numerical libraries 的 Python 包，它提供了矩阵运算、数组处理等功能。pandas（panel data）是一个开源的，BSD许可的库，它提供了高效的数据结构和数据分析功能。Matplotlib（matplotlib.org）是一个 Python 绘图库，可用于创建包括折线图、散点图、气泡图等在内的各类图表。

## 3.3 支持向量机（Support Vector Machine，SVM）
支持向量机（SVM）是一类二类分类器，利用超平面将数据集分割成两个部分，使得两部分间隔最大化。对于某个数据点来说，若其与超平面的距离比正负两侧的距离都要小，则它被认为是在正类别；反之，则认为是在负类别。这个过程就是做出预测。

SVM 可以用来解决一些分类问题，如垃圾邮件识别、图像分类、文本分类等。

## 3.4 梯度提升决策树（Gradient Boosting Decision Tree, GBDT）
GBDT （Gradient Boosting Decision Tree）是一种机器学习算法，它是建立在决策树之上的。GBDT 是一种集成学习方法，它将多个弱分类器组合起来，形成一个强分类器。 GBDT 的训练方式是迭代的。首先，它先用初始数据训练出一个基分类器。然后，它利用该基分类器的错误率来计算当前基分类器的权重。接着，它用上一步计算出的权重再次拟合出下一个基分类器。这个过程不断重复，直到达到预设的停止条件。最后，这些弱分类器的加权综合就构成了一个强分类器。

GBDT 对异常值比较敏感，对异常值不敏感的决策树容易过拟合，因此，GBDT 在很多任务上都有较好的性能。

# 4.Scikit-learn库简介
Scikit-learn 是 Python 中用于数据挖掘和机器学习的最流行的库。它提供了许多实现机器学习算法的函数和模块。Scikit-learn 由奥克兰大学、斯坦福大学、UC Berkeley 、雷锋研究院等计算机科学家和统计学家共同开发完成。

Scikit-learn 提供了几个重要的数据结构：

1. **Datasets** ：包含来自 sklearn.datasets 模块的经典数据集。这些数据集都可以通过 load_data() 方法加载，也可以使用 fetch_*() 函数直接获取。

2. **Feature Extractors and Transforms** ：用于从原始数据中抽取特征，或者将特征转换成不同的表示形式。如计数特征、文本特征、图像特征等。这些模块可以统一地处理不同类型的数据，并提供了统一的接口。

3. **Pipelines** ：用于构建机器学习管道，将数据预处理、特征工程、模型训练等流程连接起来。它提供了方便快捷的接口，可以快速搭建机器学习系统。

4. **Model Selection** ：用于评估和选择模型的函数。

5. **Metrics** ：用于评价模型性能的函数。
