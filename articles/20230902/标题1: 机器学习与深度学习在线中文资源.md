# 2.机器学习与深度学习概述
## 2.1 概念
### 2.1.1 机器学习
机器学习（英语：Machine Learning）是一门研究计算机怎样模拟或实现人类的学习行为，以获取新的知觉、做出决策并改善自身性能的交互式系统的学科。它是一系列关于计算机如何自动从数据中提取有效模式，并利用这些模式进行预测或决策的算法。机器学习理论主要关注的是设计有效的学习算法，使计算机具有智能，并将这一理论付诸实践。
机器学习是一种编程模型，旨在让计算机从数据中找寻模式，并在新的数据输入下重新训练模型。机器学习算法通常分为监督学习、无监督学习、半监督学习和强化学习四种类型。
#### 2.1.1.1 监督学习
监督学习（Supervised learning）是指计算机程序在训练过程中通过与已知正确答案的“标签”（label）配对的方式来学习数据的特征和规律，从而对未知数据进行分类或回归。监督学习有三种主要任务：分类、回归和聚类。
分类任务：假设给定一个输入向量x，我们的目标是在已知的输入-输出样本集合中找到一个函数f，使得该函数能够对新输入进行正确的输出预测。如垃圾邮件识别、手写数字识别等。
回归任务：假设给定一个输入向量x，我们的目标是预测一个连续实值的输出。如房屋价格预测、销售额预测等。
聚类任务：假设我们要把n个样本点分成k个簇，使得同一簇中的样本点尽可能相似，不同簇之间的样本点尽可能不同。如图像分割、文本聚类、用户分组、网页分类等。
#### 2.1.1.2 无监督学习
无监督学习（Unsupervised learning）是指计算机程序在训练过程不需要任何先验知识即可从数据中发现隐藏的结构。无监督学习可以看作是正则化的监督学习。它的特点是对数据没有任何明确的标记或结果，只关心数据的相似性、模式、分布等。如聚类、维度降低、概率模型。
#### 2.1.1.3 半监督学习
半监督学习（Semi-supervised learning）是指计算机程序在训练过程中既有 labeled data (带有标签的数据集) 和 unlabeled data (未标注的数据集)，但是没有带标签的样本比例较少。一般情况下，如果训练数据是无噪声的，那么此时的无监督学习也可称为半监督学习。半监督学习的主要目的是建立一个有监督的模型，同时利用无监督的方法来发现数据中的结构。半监督学习可以在有限的 labeled data 上进行训练，并且可以使用不同的模型组合。
#### 2.1.1.4 强化学习
强化学习（Reinforcement learning）是指计算机程序通过与环境的互动来学习，并依据环境的反馈对行为进行调整，从而最大化长期的累积奖励。强化学习的目标是让机器在一个环境中不断探索和学习，并根据学习的成果改善自身的表现。强化学习可以看作是带有强化信号（即奖赏）的监督学习。强化学习算法通常包括动态规划、蒙特卡洛树搜索、Q-learning、策略梯度方法等。
### 2.1.2 深度学习
深度学习（Deep learning）是机器学习的子领域，它也是一类通过人工神经网络算法训练的模型，专注于处理和学习大量数据的高效且准确的算法。深度学习的关键思想是使用深层次的网络连接在一起，形成多层非线性变换，从而学习抽象的 representations 或 features，这是机器学习中最具前沿意义的分支。深度学习包括卷积神经网络（Convolutional Neural Network，CNN），循环神经网络（Recurrent Neural Network，RNN），递归神经网络（Recursive Neural Network，RNN），深度置信网络（Deep Belief Network，DBN），长短期记忆网络（Long Short Term Memory，LSTM），变体型神经网络等。
## 2.2 常用方法
### 2.2.1 分类方法
- k近邻法(KNN): KNN 方法是一种简单而有效的机器学习方法，其基本思想是以某个点为中心，找出它与其他所有点距离最近的 k 个点，然后将属于某一类的那些点的投票权重设置为1，其他位置设置为0。
- 朴素贝叶斯法(Naive Bayes): Naive Bayes 方法是基于贝叶斯定理的一种简单而有效的分类方法。其基本思想是求解在给定特征条件下各个类别的先验概率，再根据 Bayes 定理计算后验概率，最后选择后验概率最大的类别作为当前实例的预测分类。
- 决策树(Decision Tree): 决策树是一种常用的分类和回归方法，它采用树状结构，每一结点表示一个属性，根节点代表整体，每个分支代表“是”或者“否”，叶结点表示预测结果。它可以处理任意维度的数据，并且可以同时处理类别变量和连续变量。
- 支持向量机(SVM): SVM 是一种二类分类器，它能够基于特征空间中数据的内在结构或曲线拟合进行分类。SVM 的基本思想就是找到一个超平面，在这个超平面上，不同的输入实例被映射到不同的边界区域，使得它们位于两个不同的类别上。
- 随机森林(Random Forest): 随机森林是一个集成学习方法，它通过构建一系列的决策树，结合各棵树的结果，生成最终的预测结果。不同于单棵树依赖全部训练数据，随机森林采用 bootstrap 采样方法生成若干训练子集，并在这些子集上训练一颗子树。它能够克服过拟合的现象。
### 2.2.2 回归方法
- 线性回归(Linear Regression): 线性回归是一种简单而有效的回归方法，其基本思想是用一条直线（或多条直线）来拟合样本数据。
- lasso回归(Lasso Regression): Lasso回归是一种正则化的线性回归方法，其基本思想是通过引入罚项，使得参数估计值具有稀疏性。
- ridge回归(Ridge Regression): Ridge回归是一种正则化的线性回归方法，其基本思想是通过引入罚项，使得参数估计值具有平滑性。
- 决策树回归(Decision Tree Regressor): 决策树回归是一种回归方法，它与决策树一样，也采用树状结构。它的区别是回归树的叶结点表示预测的实数值，而不是类别。
- 随机森林回归(Random Forest Regressor): 随机森林回归与随机森林一样，也是一种集成学习方法。不同的是，随机森林回归还包括了回归树作为基分类器，因此能够估计连续变量的非负异常值。
