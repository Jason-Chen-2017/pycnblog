# 1.简介
  

## 什么是网站？
网站(web site)是一种由服务器上存储的文件所组成的、通过互联网浏览器访问的一种网络服务。简单的说，网站就是通过互联网访问的信息资源，这些信息经过网页浏览器呈现给用户看。
## 为什么要建立网站?
网站的建立具有很大的商业价值，不仅可以提高个人或企业的知名度和影响力，还能让互联网从“信息孤岛”走向“信息繁荣昌盛的世界”。网站建设的好处很多，比如：
- 网站能够吸引更多的人流量，增加品牌知名度和市场份额；
- 通过网站的推广与销售，可促进产品销售增长，扩大客户群体；
- 可以发布和传播媒体内容，提供有效的营销手段；
- 提供优质的内容，满足用户的需求；
- 提升个人能力和积累，降低竞争对手的优势。
## 网站的构架
网站一般包括以下几个部分：
1. 服务器：服务器是网站运行的平台，负责提供数据支持和网络传输功能。每台服务器都有独立IP地址，它可以提供各种服务如：域名解析、网站托管、邮件发送等。最便宜的服务器主机方案是云服务器。

2. 域名系统（DNS）：域名系统(Domain Name System, DNS)是Internet使用的命名系统，它将域名转换成IP地址。当用户输入一个域名时，DNS服务器首先检查其缓存中是否有对应的IP地址。如果没有，则向根域名服务器请求该域名对应的IP地址，然后向权威服务器查询该域名解析，直到找到相应的IP地址。

3. 浏览器：互联网浏览器是访问网站的工具，不同的浏览器显示效果不同，但它们都遵循统一的HTTP协议，实现了网页的显示。目前主流浏览器有Chrome、Firefox、Safari、Opera、IE、Edge等。

4. HTTP协议：超文本传输协议(HyperText Transfer Protocol, HTTP)是用来从Web服务器获取网页的规则。

5. HTML：超文本标记语言(Hypertext Markup Language, HTML)是用于创建网页的标记语言，定义了网页的结构和内容。HTML文档由标签及内容组成，比如<html>表示网页的开头，<title>设置网页标题，<body>表示网页的正文，<h1>、<p>表示标题和段落。

6. CSS：层叠样式表(Cascading Style Sheets, CSS)用于控制HTML文档的表现形式，使页面更加美观、动态化。CSS通过选择器来指定网页元素的样式属性，如颜色、大小、边框等，并应用到相应的HTML元素上。

7. JavaScript：JavaScript是一种解释性的编程语言，用来添加交互和动画功能到网页上。JavaScript可以操控HTML文档中的对象、修改DOM树、使用Ajax技术进行异步数据交换。

8. 数据存储：网站需要保存数据库和文件，如MySQL、MongoDB、SQLite等，将用户上传的文件保存在云端硬盘上或在服务器本地磁盘上。

# 2.基本概念术语说明
## 网页的基本组成元素
1. 超链接（Link）：网页上的所有连接都是超链接。超链接通常用一个带下划线的蓝色文本显示，鼠标停留在上面就会出现一个下划线，点击后就会跳转到被链接的网页。

2. 图像（Image）：网页可以嵌入图片、音频、视频等多种媒体类型。图片可以在网页中插入，点击后会弹出大图，网页中也可以设置链接，当用户点击图片时，就自动跳到指定的链接网页。

3. 表格（Table）：网页可以制作表格，按照行列排版，可以用于展示数据、报表等。表格可以设置单元格内文字的样式、字体、大小、对齐方式等。

4. 框架（Frame）：网页可以包含多个框架，每个框架中可以嵌入其他网页。

5. 表单（Form）：网页可以制作表单，用户填写信息后提交给服务器处理，比如搜索框、评论框、订阅表单等。

6. 文件（File）：网页可以把本地计算机的文件上传至服务器，可用于分享资料、下载文件、上传图片等。

## HTML和CSS的区别
HTML(HyperText Markup Language)：超文本标记语言，是用来描述网页内容的标记语言。

CSS(Cascading Style Sheets)：层叠样式表，用于网页的视觉呈现。CSS提供了网页的布局、配色、字体等控制功能。

HTML和CSS的关系类似于电脑屏幕和显示器的关系，HTML是网站的骨架，而CSS是装饰这个骨架的工具。

## CSS选择器
CSS选择器用于选取网页中特定元素进行样式设置。CSS选择器有四种基本语法格式：

1. 元素选择器：选取某个标签的所有元素，语法格式为`tag`。例如：`div{}`选择的是所有的div元素。

2. class选择器：选取某个class类所有元素，语法格式为`.classname`。例如：`.highlight{}`选择的是所有class="highlight"的元素。

3. id选择器：选取某个id标识符的所有元素，语法格式为`#idname`。例如：`#main{}`选择的是所有id="main"的元素。

4. 组合选择器：组合多个选择器，语法格式为`selector1 selector2`，即选择器之间的空格隔开。例如：`div.highlight{}`选择的是所有class="highlight"的div元素。
