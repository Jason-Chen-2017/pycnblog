# 1.简介

# 2.背景介绍
计算机视觉领域很大的一部分工作就是构建各种各样的模型来识别和分析图片，比如人的眼睛、鼻子、嘴巴等。这些模型都是根据不同领域的经验和知识提炼而成。这里先不做详细介绍，介绍一下我自己对计算机视觉的认识。

人类对图像的感知能力，一般来说要远高于其他物种。在生物学上，人的视网膜和视盘分泌液分泌系统之间存在着一定的耦合关系，当看到物体时，神经元之间的连接就会被激活，这些活动会刺激大脑生成对其的认知。可见人类的眼睛显著比其他动物的眼睛更加灵敏，能识别物体的轮廓、颜色、形状等。

深度学习技术的到来使得计算机视觉领域发生了巨变。深度学习模型可以非常好的学习到底层的图像特征，将它们转化为可解释的表示，提升图像识别、分析、理解的能力。如卷积神经网络(CNN)，它是一个深层次的神经网络结构，能够从图像的原始像素中学习到一些特征，比如线条、形状、曲线等，并且对于某些特殊场景还具有良好的泛化能力。AlexNet便是最早的CNN模型之一，具有高效、实时的特点。
