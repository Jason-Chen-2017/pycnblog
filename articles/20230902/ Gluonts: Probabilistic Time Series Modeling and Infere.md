# 1.简介
  

在机器学习和数据科学领域里，时间序列模型（Time series model）被广泛地应用于各种场景下，如金融市场、气象预报、电信流量监控等。然而，基于概率分布的时序模型往往能更好地捕捉到时间序列的长期行为特征，并更准确地预测未来的变化。Gluon-TS 是 Apache 软件基金会 (Apache Software Foundation) 下的一个开源项目，它实现了高效且灵活的概率时序模型，支持对时序数据进行建模、推断和预测，并提供丰富的工具包接口用于定制化开发。Gluon-TS 的主要特点包括：

1. **统一的 API**： Gluon-TS 提供统一的 API 来构建和训练概率时序模型。用户可以用类似 Scikit-Learn 或 Pytorch 中的方法来快速搭建和训练模型。

2. **高度模块化的设计**： Gluon-TS 的核心组件均采用高度模块化的设计。它分离出不同功能的模块，如数据处理、模型定义、后端优化器、评估指标等，使得各个模块可以自由组合，构建复杂的概率时序模型。

3. **多种优化器选择**： Gluon-TS 支持多种优化器，如随机梯度下降法、Adam、Adagrad、RMSProp 等，满足不同的需求。

4. **丰富的数据集支持**： Gluon-TS 内置了多种时序数据集，方便用户快速上手。包括用 Airline passenger data 和 temperature time series 数据集构造的模型示例。

5. **模型可视化与分析**： Gluon-TS 提供了丰富的模型可视化和分析工具，可以帮助用户理解模型的预测结果。

6. **灵活易用的推断接口**： Gluon-TS 提供了一套灵活的推断接口，让用户可以直接获取分布或样本，而不是只限于预测结果。

# 2.基本概念术语说明
## 时序数据
时序数据（Time series data），又称时间序列数据，是指具有时间维度的观测值序列，其特点是随着时间的推移而单调增加或减少。典型的时序数据包括股票价格、天气信息、季节性因素、传感器读数、网络流量等。

## 模型
一个给定的时序数据，通常存在多个可能的模型来描述它的趋势和周期性，这些模型都可以用来进行预测和分析。时序模型可以分为三类：

1. 平稳模型（stationary models）：这些模型对时间的依赖关系没有显著的变化，例如截面平滑模型（ARMA）、移动平均模型（MA）。

2. 漏涨/泄露回归模型（regression models with leakage or drifting）：这些模型对时间的依赖关系显著改变，有些依赖于过去的值，有些依赖于未来的值。例如随机游走模型（Random walk regression）、时间序列分解模型（Decomposition）。

3. 混合模型（mixed models）：这些模型既包括平稳模型的部分，也包括漏涨/泄露回归模型的部分。

## 分布
在统计学中，分布（distribution）是一个具有参数的函数，描述了随机变量取值的概率。最常用的分布就是均匀分布（uniform distribution）、正态分布（normal distribution）等。在概率时序模型中，分布往往是一个模型的输出，用于刻画不同事件发生的概率。

## 采样
采样（sampling）是从概率分布中抽取样本的方法。概率时序模型通常需要通过实践经验或模拟过程生成样本，然后通过采样技术估计模型参数。

## 深度学习
深度学习（deep learning）是机器学习中的一种方法，它利用神经网络结构来表示数据的内部表示形式，可以自动提取数据的特征，并且可以有效地学习复杂的非线性关系。深度学习已经成功地应用在许多领域，如图像识别、自然语言处理、语音识别、推荐系统等。
