# 2.相关概念与术语
首先我们需要了解一些相关的概念和术语。
## numpy
numpy(Numerical Python)是一种开源的python语言的数值计算扩展工具包，用于科学计算和数据处理。它的主要功能是对多维数组进行快速计算。numpy提供了很多函数用来处理线性代数、傅里叶变换、随机数生成等方面的任务。
## random number generator (RNG)
随机数生成器（random number generator）是指由算法生成的，随着时间推移而不断变化的数字串。随机数生成器产生随机序列的方法一般包括：确定性算法和非确定性算法。确定性算法具有固定的算法结构，可以基于一定规则生成随机数序列；而非确定性算法则更加复杂，其随机数序列受到各种影响，例如环境噪声、初始状态、物理因素、操作系统随机数发生器、种子等。一般来说，为了保证随机数的质量和安全，最好选择可靠的随机数生成器。目前主流的随机数生成器有以下几种：

1. Linear congruential generator (LCG): LCG 是最早被提出的随机数生成算法。它的基本原理是用线性同余法生成伪随机序列，即在上一次随机数发生器的输出值的基础上，根据一定的算术关系，生成下一个随机数。LCG 的缺点是周期性较长，容易被恶意攻击，故通常只用于比较简单的安全加密应用。

2. Mersenne Twister: Mersenne Twister 是 MT19937 系列算法的最新进化，它比 LCG 更安全，并且具有很好的性质，如具有短暂性、高低温下稳定性、周期长、没有线性相关性等。

3. AES-based RNGs: 由于计算机硬件和软件的性能不断提升，AES加密算法得到了越来越广泛的应用。由于其快速计算特性，AES加密算法被广泛用于RNG的设计与实现。

4. Hash functions: 通过对输入信息进行哈希运算，生成满足特定分布的伪随机数。hash函数的随机性在于其不可预测性，不同输入信息会得到不同的输出，因此可以用于生成安全的随机数。

根据以上定义，可以看到，随机数生成器可以分为两类：确定性RNG和非确定性RNG。前者具有固定的算法结构，可以使用固定的计算规则生成随机数，易于理解和实现；后者则更加复杂，算法结构因素繁多，算法执行效率低下，存在诸多安全隐患。对于随机数的应用场景，可以使用不同的RNG组合策略，实现出色的随机数生成效果。

## Seed and state
随机数生成器的种子（seed）和状态（state）是两个重要的概念。种子是在随机数生成过程中起始的条件，它决定了随机数序列的初始状态。如果种子相同，则产生的随机数序列也相同；如果种子不同，则产生的随机数序列也不同。当种子不固定时，则随机数生成器应具有强大的抗纵弱能力，即使给定相同的种子，也应该生成不同的随机数序列。状态是指随机数生成器的当前状态，包括内部变量和其他随机数生成所需的数据。通过保存并恢复状态，可以实现对随机数生成器的连续性和可控性。
