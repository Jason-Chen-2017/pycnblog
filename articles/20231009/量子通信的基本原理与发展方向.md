# 1.背景介绍


## 1.1 什么是量子通信？
量子通信（Quantum Communication）指利用量子技术解决信息传输的技术，主要通过两类传输信道实现。
- 经典信道——在经典通信中，信息只能从一个点到另一个点传播，且路径存在阻碍时会发生丢包、重发等问题。
- 量子信道——利用量子态的存在及其对比关系来进行信息传输。可以将量子信道比喻成无线电信道中的光通讯，传播过程中比类波更容易穿透障碍物、抵达目标地点。
量子通信能够实现信息传输的两个特点：
- 编码效率高——量子通信采用编码方式来对信息进行处理，编码方式能提升信道容量，提高信息传输的速率。同时，编码后的数据具有突破性的优点，不易被偶然破坏或窃听。
- 传播距离长——量子通信的传播距离远大于经典信道。因此，通过量子信道传输的信息比通过经典信道传输的信息安全性更高。
## 1.2 为什么要做量子通信？
- 信息存储容量巨大——量子通信带来的信息存储容量超过了传统通信技术。对传统通信来说，信道容量受制于固定的速率、码元大小等，而对于量子通信，信道容量则由一系列的量子态决定，而这些态的数目随着空间和时间的度量而指数增长。因此，量子通信可以实现信息存储的指数级扩大。
- 传播距离长——量子通信带来的是信息的传播速度，相比于经典通信而言，它的传播速度几何倍数增长。这样一来，信息传输的时间就缩短了，信息储存的时间也变得更长。
- 没有极限寿命限制——由于量子通信所依赖的量子纠缠特性，使得它的寿命期限无限长。这是因为，量子纠缠是一种非可逆的过程，这意味着只要两个量子态相互作用，就永远无法完全重置它们之间的纠缠状态。因此，量子通信不会因某种原因而停止，可以一直持续到没有明显影响。
## 1.3 发展方向
量子通信的发展前景还是很广阔的。除了量子通信自身之外，与量子通信相关的技术还有很多。其中包括量子计算、量子信息存储、量子路由、量子网络、量子通信终端设备、量子通信网络管理等等。

根据Wikipedia上面的定义，现有的量子通信技术可以分为以下几类：
- 基础设施建设——包括通信网的量子化、量子计算平台的开发、多级结构量子纠缠系统的部署和维护等。
- 技术研发——包括量子通信协议的研究、量子通信系统的设计和实验验证等。
- 应用层开发——包括量子通信系统中应用层协议的设计和开发、基于量子通信的计算任务的分布式计算等。
- 服务提供者——包括各类商业公司、政府部门、高校、研究机构以及行业协会等提供量子通信服务的创新企业。

量子通信正在成为继纳米技术之后又一颗引领未来的科技蓝海。但作为一项重大科技突破，它也面临着巨大的挑战。未来，如何有效的开展量子通信领域的科研、创新，并且获得良好回报，仍然是一个重要课题。
