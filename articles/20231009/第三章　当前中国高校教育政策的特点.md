# 1.背景介绍


## 一、现状
近几年，我国高等教育由以培养研究生、本科生和专升本为主的传统模式演变为以市场经济条件下的知识经济模式发展。我国高等教育事业经历了从计划经济向市场经济转型、从“一分为二”（985工程、211工程）到“一分为三”（普通高中、普通院校、高职院校）、从直隶式办学向分校办学转型等过程。随着我国社会主义市场经济建设的不断深入，高等教育也面临着重塑、升级、优化的课堂教学方式、教材编写与改革、学科建设、学术质量保障、招生管理和选拔机制等方面的问题。因此，我们国家高等教育正在进行一系列改革，提高工作效率、加强学生培养质量，探索建立健全高等教育制度的新体系，同时完善招生制度和教学评价机制，减轻教师负担，促进学校科研产出、创新能力、人才培养与就业机会的有效整合。但是，当前高等教育发展仍然面临着诸多挑战。其中，我国高校教育存在的突出问题主要集中在以下几个方面：

1. **资源配置不足**。高等教育领域具有高度专业化和技能性，但往往缺乏相应的综合能力；课程设置不精准，课程内容难以真正吸收学生的基本知识和能力；授课教师素质低下，师资队伍缺乏互补性，师资队伍需要不断扩张。这些因素使得我国高校不仅无法提供优质的教学环境和满足学生需求，而且还严重影响了学生的学习效果、教育质量和就业前景。

2. **课程结构混乱**。中小学课程设置一般为普通班或基础班，学生只能选择所谓的通识课程、职业课程或学科课程，而不能自由选修其它主题课，因此中小学教学存在难以调整和完善的局面；成人教育课程设置较简单，没有充分考虑学生的不同阶段需要不同的教育视角和掌握不同的知识技能，导致学生认知、理解和运用知识技能上的差距较大；学生容易被迫接受单一课程的训练，缺乏合作、交流、学习的机会。

3. **评估不公开**。我国高校普遍存在“两个凤冠”，即一部分高校自称“造学术”，另一部分则对学术研究评价不公开、评估有偏向。比如有的高校只公布学生的总成绩，而不公布各科成绩详情；有的高校将老师教学评判委员会成员对学生的评价归咎于其个人品行缺陷等，这实际上对学生的实践能力和创造力考核无助于提升教学质量和师德风范；有的高校故意雇佣不良教师，利用双重标准、扭曲数据，歪曲学术评价指标，并强化其声誉，损害教师的名誉形象。这些做法既不符合学术规律，又助长官僚主义，扰乱社会舆论，妨碍社会公平竞争。

4. **认知能力及语言文化障碍**。高等教育越来越依赖于计算机网络、数字媒体、网络游戏、移动互联网、大数据等信息技术工具，为学生提供了在线学习平台、个性化学习工具，但很多学生由于种种原因，还是不具备学习的耐心和独立思考能力，甚至出现学习困难、自闭症等心理疾病。此外，高等教育还需要与世界各地保持沟通合作，解决语言文化差异带来的语言、文字、阅读等方面的障碍。因此，我们必须努力为学生培养出坚韧的生命意志和逻辑思维能力，进一步开拓多元化的教育资源，实现共同富裕。

5. **学生薪酬水平低**。目前，我国高校的研究生、博士生薪酬都相对较低，且学费极高。据报道，一些高校博士后辞职时多次被行政部门处罚，数额巨大，但很多学校却给予录取学位证书。一些硕士研究生入学时常被录取，但他们往往因取得奖项或就业而获得不菲的待遇，而非优厚的学费。此外，许多高校为了鼓励学生“留级回乡”，给予退学金或补贴等奖惩，往往造成学生返乡后的生活负担更大。因此，我们必须更加关注学生的实际工资水平和满意度，根据经济实力和受教育程度进行薪酬匹配，建立符合实际需要的学校选拔机制，提高学生的接受能力和求职意愿，促进学校之间的竞争、融合，让每个学生都享受到公平、尊重、自豪的社会声誉。 

以上五个方面，构成了我国高等教育发展面临的突出问题。我们作为一个国家，应该继续推进教育现代化，把注意力放在培养创新型人才上，让优秀青年创造更大的价值。如此，我们才能打造成为真正意义上的“先进生产者”。
