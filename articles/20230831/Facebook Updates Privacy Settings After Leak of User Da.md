# 1.简介
  


近日，有不少用户因泄露个人隐私信息而被Facebook删除账号，这个事件引起了许多人的关注和愤怒。Facebook在公开声明中表示，当发现用户数据被盗后，会立即更新其相应的隐私设置，其中包括访问过的所有个人资料、照片和视频都将被永久删除。

对于此次Facebook数据泄露事件，笔者有以下几点看法：

1. Facebook在这一事件中所采取的一系列行动可能并没有做到位，比如在公告中没有详细解释为什么要进行隐私设置的修改；另外也缺乏透明性，用户并不知道Facebook会对自己的隐私信息进行哪些处理。所以说Facebook应当加强公关力度，提供足够的细节信息，让用户能够清楚地理解其隐私权利受损的影响。

2. Facebook在公布这些隐私设置变更时应该有相应的解释，比如隐私设置变更后的功能有什么不同，用户应该如何适应新的隐私设置等。用户需要知晓在他们的个人资料、照片、视频中可以找到自己的数据以及在Facebook上的活动情况。

3. Facebook应当积极寻找外部安全漏洞，比如哪些服务可能会被黑客攻击或利用恶意数据窃取私密信息，如果出现类似事件，Facebook应当及早通过警报机制向用户发出通知。

4. 在国内互联网环境下，Facebook的用户隐私保护体系并不健全。Facebook在线上平台上难以保障用户的隐私安全，这是其他类似网站如脸书、推特等存在的普遍问题，值得推动政府部门通过新一代网络安全技术对Facebook上网行为进行监测、分析和管理。

# 2. 基本概念术语说明
# 2.1 用户数据

用户数据指的是，用户创建或上传至Facebook或其他相关产品的数据，这些数据可能包括但不限于姓名、照片、视频、联系方式、位置信息、账户密码等。

# 2.2 数据泄露

数据泄露是指第三方获取用户信息，通过非法手段传输、转移、复制、披露或者删除用户信息，导致用户的合法权益遭到严重威胁，甚至危害社会和国家安全的行为。

# 2.3 隐私政策

隐私政策是一份由用户对自身在网站、APP和其他网络服务上的个人信息收集、使用、共享及保护的声明。一般包含对收集的信息类型、目的、处理方法、处理权限、第三方共享、传输、存储、删除、安全保护措施、处理时间等规定。

# 2.4 技术揭秘

技术揭秘是一种专业的科技类文章，通常从事计算机、软件、网络或系统的专家们通过技术手段分析探索计算机系统运作原理，揭示其运作机理的专业性科技类文章。

# 2.5 终端设备

终端设备是指连接至网络的个人计算机、手机、平板电脑等移动设备。

# 2.6 电子邮件地址、IP地址

电子邮件地址（Email Address）用于标识用户身份，通常由字母数字组成，且最后一个字符为“@”号。IP地址（Internet Protocol Address）是一个Internet协议中的地址，它唯一标识互联网上的主机，每个主机都有唯一的IP地址。

# 3.1 概述

根据Facebook公开的关于用户隐私设置的变更通知，当发现用户数据被盗后，Facebook立即更新其相应的隐私设置，其中包括访问过的所有个人资料、照片和视频都将被永久删除。具体操作流程如下：

Step 1: 用户发出投诉举报
首先，用户需要向Facebook提交关于个人隐私泄露的投诉举报，举报可以通过社交媒体、网站或客服的方式提交。用户需要提供具体证据材料，包括个人信息发生泄露前后的对比，以及受影响的账号、设备、操作记录、日志等。

Step 2: Facebook核实数据泄露
第二步，Facebook将根据用户提供的相关证据，核实是否存在个人隐私数据泄露的迹象。Facebook将会进行严格的数据检查，确认用户数据没有泄露给任何第三方。同时，Facebook还将对用户数据在所有服务器上进行备份，确保用户数据的完整性和可用性。

Step 3: Facebook更新用户隐私设置
第三步，Facebook将立即更新用户的隐私设置，包括访问过的所有个人资料、照片和视频都将被永久删除。Facebook表示，在用户的数据被完全删除之后，将不会再对该用户进行任何隐私信息访问查询。而且，Facebook还将立即停止向已删除用户发送任何形式的消息、电子邮件或提示。用户的账号也将无法登录，而任何与账号关联的邮箱账号也不会再收到任何来自Facebook的邮件。

Step 4: Facebook公开数据泄露事故经过
第四步，Facebook将在官方网站、Facebook页面、Facebook App上发布公告，公告中详细解释为什么进行隐私设置的修改，以及用户的隐私权利受损的影响。用户将能够在官方公告中查阅到Facebook对此次隐私设置变更所作出的解释。用户同样可以在Facebook App中查看到具体隐私权利受损的具体内容。

# 3.2 计算公式

根据公开的公告内容，Facebook会在用户的数据被完全删除之后，停止对该用户进行任何隐私信息访问查询。具体原因如下：

1. 删除用户数据

   在用户数据被完全删除之后，Facebook将不再保存用户数据，因此也就不需要再对其进行任何隐私信息访问查询。

2. 不再向用户发送消息、电子邮件或提示

   虽然用户的数据已经被删除，但Facebook依旧不会再向该用户发送任何形式的消息、电子邮件或提示。这意味着，如果用户试图向Facebook咨询任何相关的问题，也只能得到不可达的回复。此外，如果用户试图通过主动联系用户的方式获得帮助，则仍然需要通过官方渠道进行申请。

3. 禁止用户登录账号

   当用户的账号被完全删除之后，他们将无法登录Facebook，也无法再重新注册账号。这意味着，如果用户想要继续使用Facebook，需要重新注册账号或通过其他方式导入已有的账号。

4. 停止向用户邮箱投递邮件

   由于用户账号已被删除，因此Facebook也不再接收到任何来自该用户的邮件。此外，Facebook也不再保存用户的邮箱地址。这意味着，如果用户希望通过邮箱联系Facebook，则需要向支持团队提交申请。
