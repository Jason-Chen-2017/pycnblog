# 2.词汇表、特征提取及命名实体识别
## 2.1 词汇表、特征提取及命名实体识别概述
### 2.1.1 概念和术语
#### 词汇表
词汇表（Vocabulary），是一个由单词组成的集合，其中每个单词都有其唯一的标识符，通常是整数或字符串。它是从大量文本中抽取出的单词总结，这些单词按照它们的出现次数、重要程度和各种统计指标排序。词汇表可以通过两种方式创建：手动创建和自动生成。手动创建可以通过人工收集各种语料（如电子邮件、新闻文章等）中的所有单词，然后根据它们的频率进行排序和筛选；另一种自动生成的方法是利用现有的语言模型（language model）。语言模型是通过对大量的文本数据训练得到的统计模型，它能够计算某些语句出现的可能性，例如“今天天气怎么样？”、“明天下雨吗？”。语言模型通过统计各种语言现象、词汇使用、语法结构等特征来表示语句，并预测下一个可能出现的词或短语。因此，当给定一个新的语句时，可以计算该语句出现的概率，从而找出最可能的词汇表，或者找到句法上和语义上的错误。

#### 特征提取
特征提取（feature extraction）又称特征抽取，是计算机视觉中常用的一种方法。它从原始图像或文本中提取有意义的特征用于后续的机器学习任务。特征提取的基本目标是从输入数据中提取出一些可以用来训练或测试机器学习模型的有效特征。在NLP中，特征提取也是一个重要的任务，因为文本的长度、语法、拼写、语义等多种属性可以反映文本的相似性和差异性。

#### 命名实体识别
命名实体识别（Named Entity Recognition，NER），是指从文本中识别出特定类型（如组织机构名、人名、地名、时间词等）的实体，并赋予这些实体一个名称或别称。NER有利于大幅度降低复杂的文本信息的分析难度，实现智能问答、文本分类、机器翻译等高层次应用。传统的NER方法一般包括基于规则和基于统计的两类算法。基于规则的算法通常采用基于一套固定的规则，对文本中的每一个词进行判断，确定它是否是一个命名实体。基于统计的算法则使用数据驱动的方式，统计训练数据中的命名实体分布和出现频率，然后基于此对新的数据进行预测。目前，基于深度学习的NER方法也越来越多。

#### 词汇表生成
词汇表的生成过程包括三个阶段：

1. 数据收集：首先，要收集足够多的文本数据，包含所有的句子。然后对这些数据进行清洗、过滤，确保它们没有错误的单词。

2. 分词：将数据中的句子切分成单词，并删除一些无关紧要的词，比如标点符号、停用词。

3. 词频统计：统计每个单词出现的次数，并根据次数大小对单词进行排序，形成词汇表。

4. 可视化展示：可视化展示词汇表，便于观察。

#### 特征提取
特征提取即对文本进行特征选择，从而提取出有用特征用于之后的分类、聚类等任务。常用的特征包括词频、词干、TF-IDF、文本长度、句法结构等。

#### 命名实体识别
命名实体识别包括两个阶段：

1. 命名实体识别任务定义：将输入文本划分成一系列实体，包括人名、地名、组织机构名、时间词等。

2. 命名实体识别算法设计：通过统计或规则方法对输入文本进行命名实体识别。命名实体识别可以帮助我们更好的理解文本内容，提升系统的准确性和效果。

3. Named entity recognition tools: 有多种开源的工具可以实现命名实体识别功能。如Stanford CoreNLP、Spacy等。
