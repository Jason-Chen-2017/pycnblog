# 1.简介
  

- 作者：赵嘉杰、李明新、徐鹏、陈其龙、王金庆、李东升
- 出版社：中国铁路出版社
- 年份：2021年

本书主要内容包括Java基础、集合框架、多线程并发编程、JVM高级特性、Spring全家桶、微服务架构、云计算与大数据等相关知识点。

本书作者通过对Java编程语言的理解、详细讲解Java语法特性及各种应用场景、详实的Java面试题分析、实用的Java工具库、流畅的讲解，帮助读者理解Java开发的整体架构、学习Java开发技能，具备实际工作中所需的Java开发能力和技能。
