# 1.简介
  

## 1.1 什么是Python？

Python 是一种通用编程语言，最初由Guido van Rossum开发，于1991年发布。它是高级编程语言、脚本语言、面向对象的语言，拥有简洁、易读、高效、动态等特点。其特点包括：

- 可移植性：支持多种平台，能够运行在Linux、Windows、macOS、Android、iOS、微控制器、路由器及其他设备上；
- 丰富的库：内置大量的第三方库，可用于数据分析、图像处理、机器学习、Web开发、自动化运维、系统管理等领域；
- 解释型语言：不需要编译成二进制代码即可执行，适合开发测试及实验阶段的快速迭代；
- 面向对象：支持类、模块、多重继承等特性，灵活地实现面向对象编程；
- 自动内存管理：自动回收不再使用的内存，减少程序崩溃的可能性。

## 1.2 为什么要学习Python？

Python 的广泛应用促进了数据科学和 AI 的发展，同时也吸引着 IT 从业者投入到 Python 技术领域中来。Python 拥有成熟的生态系统，被许多公司、研究机构、金融机构、政府等使用，并形成了一系列优秀的开源项目。而且，由于其简单易学、交互式环境、自动内存管理等特性，使得 Python 在数据科学、机器学习、爬虫、自动化运维等领域都有广泛的应用。所以，学习 Python 有助于提升个人能力、增强自信心、锻炼业务技能、解决实际问题。

# 2.基本概念术语说明
## 2.1 标识符（Identifier）

在程序设计过程中，我们经常会创建变量名、函数名、类名等各种名称，这些名称称之为标识符（Identifier）。标识符可以是字母、数字或下划线字符的组合，但不能以数字开头。

在 Python 中，所有的标识符（包括变量名、函数名、类名等）都需要遵守命名规则。命名规则主要包括：

1. 首字符必须是字母或者下划线
2. 其他字符可以是字母、数字、下划线
3. 不允许关键字作为标识符（后续章节会详细说明）

以下是一个正确的标识符示例：

```python
a_name = "Tom"
student_age = 25
total_marks = 100.5
this_is_my_variable = True
```

## 2.2 注释（Comment）

程序中用以辅助说明的代码段就是注释，一般是用来解释某些特定代码段的作用。在 Python 中，单行注释以 `#` 开头，可以用来标记单行语句，如：

```python
# This is a single line comment
```

而多行注释则是使用三个双引号 `"""` 或三个单引号 `'''` 来标记。多行注释可以让代码更加易读，便于理解和维护。

```python
"""
This is a multi-line 
comment in triple quotes.
"""
```

## 2.3 数据类型

数据类型（Data Type）是指变量所存储的数据值的种类，Python 中共有八种基本的数据类型，分别为整数、浮点数、字符串、布尔值、列表、元组、字典和集合。

### 整数（int）

整数（int）类型用于表示整数值，它的取值范围是负无穷至正无穷大。我们可以使用关键字 `int()` 创建整数。

```python
num1 = int(10)      # 十进制
num2 = int("10")    # 字符串
num3 = int(0xAF)    # 十六进制
num4 = num3 + num1  # 运算
print(type(num1))   # <class 'int'>
```

### 浮点数（float）

浮点数（float）类型用于表示小数值，它的取值范围是近似无穷大的十进制数，即小数点后有一定的精度。我们可以使用关键字 `float()` 创建浮点数。

```python
pi = float(3.14)     # 直接赋值
height = 1.75        # 小数点后有两位精度
weight = 75.2        # 小数点后有一位精度
speed = height / 3   # 计算速度
print(round(speed, 2))   # 输出速度保留两位小数
```

### 字符串（str）

字符串（str）类型用于表示文字，其中的每个字符都由单引号 `' '` 或双引号 `" "` 表示。我们可以使用索引方式访问各个字符，也可以通过切片的方式获取子串。

```python
s1 = "Hello World!"
s2 = 'I love Python'
s3 = "The quick brown fox jumps over the lazy dog."
sub1 = s3[2:8]    # 获取子串
char1 = s1[-1]    # 通过索引获取最后一个字符
word1 = s3[:7]    # 以索引切割字符串
count = len(s1)   # 获取字符串长度
print(char1 * 3)   # 重复输出字符串
```

### 布尔值（bool）

布尔值（bool）类型用于表示真值和假值，只有两个值：True 和 False。我们可以对表达式进行逻辑判断，比如比较运算符 `<=`、`>=`、`==`。

```python
flag = (1 > 2) or (3 == 4) and not(False)
print(flag)   # Output: True
```

### 列表（list）

列表（list）类型用于存储一系列的值，这些值可以是任意数据类型。我们可以使用方括号 `[ ]` 创建列表，并使用索引和切片的方式访问元素。列表也是可以修改的，可以添加、删除、替换元素。

```python
lst = ["apple", 100, "banana"]         # 列表的创建
lst.append("orange")                  # 添加元素
lst.pop()                             # 删除末尾元素
lst[1] = lst[1] - 5                   # 修改元素
new_lst = lst[1:]                     # 切片获取子序列
```

### 元组（tuple）

元组（tuple）类型类似于列表，不同的是元组是不可变的，只能读取其内容不能修改。我们可以使用圆括号 `( )` 创建元组。

```python
tup1 = ("apple", 100, "banana")       # 元组的创建
first, second, third = tup1           # 多变量赋值
fourth = (1,)                         # 单元素元组
fifth = first, fourth                 # 合并元组
sixth = sixth + second                # 错误的尝试，无法修改
```

### 字典（dict）

字典（dict）类型用于存储键值对形式的数据，其中键是唯一且不可变的，值可以是任意数据类型。我们可以使用花括号 `{ }` 创建字典。

```python
d1 = {"name": "John", "age": 30}            # 字典的创建
d1["city"] = "New York"                    # 添加键值对
del d1["age"]                              # 删除键值对
val1 = d1.get("gender", "unknown")          # 获取值，不存在返回默认值
d1.keys()                                  # 获取所有键
d1.values()                                # 获取所有值
```

### 集合（set）

集合（set）类型用于存储一组无序和无重复的值，元素可以是任意不可变数据类型。我们可以使用大括号 `{ }` 创建集合。

```python
s1 = {1, 2, 3, 3, 2, 1}               # 集合的创建
s2 = set([('apple', 5), ('banana', 3)]) # 用列表创建集合
len(s1)                               # 获取集合大小
```
