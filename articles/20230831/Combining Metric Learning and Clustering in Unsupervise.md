# 1.简介
  

人类活动识别(Human Action Recognition HAR)是指利用计算机视觉技术从视频或图像中检测并识别人的动作。人类活动识别在多个领域都具有重要意义，例如自动驾驶、人机交互等领域，它的成功也促进了机器人、虚拟现实等技术的革命。近年来，随着研究者对人类活动识别任务越来越关注，许多研究人员提出了不同的方法论。其中一个主要的方向就是使用无监督学习的方法进行人类行为识别。无监督学习的代表性算法之一是Metric Learning方法。本文主要基于这种方法提出一种新颖的无监督学习算法——CBHAC(Combined Based on Heterogeneous Actions Clustering)，即结合异质的动作聚类算法。该算法可以有效地利用不同类型的特征向量，提升对人类动作的识别性能。
# 2.相关工作
1. Metric Learning 方法
Metric Learning方法是无监督学习的一个子集。它通过一个距离函数来度量输入数据的相似性，然后学习到一个合适的映射关系将这些数据点投影到一个低维的空间中。因此，Metric Learning方法能够为数据建模提供一种非线性的方式，能够帮助数据降维、可视化、分类、聚类和异常检测。常见的Metric Learning方法有EM算法、谱嵌入法、低秩分解法、凸优化法等。

2. 动作聚类算法
动作聚类算法是无监督学习的一个领域，其目标是在相似的情况下把相似的行为组合成一组，不属于同一个行为的行为属于不同的组。常见的动作聚类算法包括K-means、DBSCAN、BIRCH、层次聚类、贝叶斯向量聚类等。

# 3. CBHAC 方法概述
## 3.1 模型架构
CBHAC模型由两个模块组成：基于特征的动作聚类(FBC)模块和基于结构的动作聚类(SBC)模块。其中FBC模块负责对每个动作提取高级特征，如特征表示、局部特征。而SBC模块则通过对多个动作的相似度进行综合评估，选择最合适的动作集作为最终的输出。

## 3.2 FBC 模块
FBC模块首先对输入的RGB帧图像进行特征提取，提取出颜色、形状、运动、空间分布等特征。然后，经过特征聚类器(Feature Clusterer)对这些特征进行聚类，将同类特征归于一类。这里，作者提出的特征聚类器是一种多层次网络，能够兼顾全局特征的表征能力和局部特征的稀疏性。具体来说，特征聚类器由多个阶段组成：第一阶段是卷积神经网络(CNN)提取全局特征；第二阶段是空间金字塔池化(SPP)提取局部特征；第三阶段是序列特征融合(SEQF)将多种局部特征按时间顺序连接起来，得到完整的全局特征。最后，基于软最大化约束(Soft Maximal Constraints，SMC)的聚类器用于对特征进行聚类。

## 3.3 SBC 模块
SBC模块根据基于特征的动作聚类的结果对不同动作之间的相似度进行建模。首先，作者计算了不同动作的中心特征，通过KL散度距离度量不同动作之间的相似度。其次，作者根据不同动作之间的时间跨度进行相似度匹配，确保只有紧密相连的动作才被认为是相似的。作者采用卡方检验来衡量不同动作之间的相似度，并使用信息增益(Information Gain)来选择最佳的动作子集。

## 4.2 超参数设置
为了训练CBHAC模型，作者设计了一套参数搜索策略。首先，作者固定训练批大小为16，将学习率设定为初始值0.1，减小至0.001。然后，作者逐步增加特征聚类器的卷积核数量，确定是否引入BN层。接下来，作者固定特征聚类器的层数和卷积核数量，用EM算法寻找最大熵模型参数。然后，作者固定EM参数，将初始学习率增大至0.1，然后逐渐衰减至0.001。最后，作者固定所有参数，用L-BFGS算法训练整个模型。具体的参数设置如下图所示。

# 5. 未来发展方向
CBHAC模型是一个新的无监督学习算法，在目前的视频动作识别任务中取得了良好的效果。但是仍然存在一些待解决的问题。例如，目前的算法假设所有的动作具有相同的全局结构。但实际上，不同类型动作往往具有不同的局部结构和上下文信息。因此，作者希望在之后的工作中，将CBHAC模型扩展到更加丰富的场景中，提升对不同类型动作的识别能力。此外，在人类行为识别中，要考虑不同类型的动作的依赖关系。例如，一个足球运动员的动作往往由多个手段构成，如传递、接近、拨号等。因此，作者需要考虑如何同时利用不同类型的动作信息，提升动作识别的准确率。
