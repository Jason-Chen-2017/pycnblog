# 1.简介
  

在社会中，公众参与型政策的实施，不仅可以提升公共服务水平、促进公民素质建设、保障人们权益，更能有效改善当地经济社会运行。然而，由于公众参与型政策的种类繁多，涉及范围广泛、制度设计难度大、执法监督压力大等特点，其执行力如何评价也是一个值得关注的问题。本文将阐述公众参与型政策执行过程中的常见误区，给出一个公众参与型政olicy执行力评价方法。最后，对未来发展方向做一些探讨，展望公众参与型政策的前景。
# 2.概述

## 2.1 什么是公众参与型政策？

公众参与型政策是指依据特定条件，利用公众的行动支持或者反对政府政策的一种制度安排，包括政策宣传、问责制、社会动员、选举投票等。公众参与型政策旨在激励、引导或影响公众积极参与到政策制定的过程之中，并通过各种方式实现政策目标的达成。

## 2.2 公众参与型政策类型

目前，中国已经存在多种类型的公众参与型政策，包括减税降费、减少农村留守儿童、开展城镇居民精神健康宣教、调查取证、金融领域的创新政策、地产市场的品牌建设、环保领域的污染防治、教育领域的免费教材等。其中，政策种类繁多、影响面广、制度设计复杂、执法监督压力大的特点，是当前公众参与型政策的典型特征。

## 2.3 为什么要做公众参与型政策？

公众参与型政策的实施，能够带来诸多好处，如提高公共服务水平、促进公民素质建设、保障人们权益；但是，同时也存在着潜在风险和挑战。原因之一是，公众参与型政策的种类繁多，涉及范围广泛、制度设计难度大、执法监督压力大等特点，使得各地区、各部门都需要进行相应的研究、论证、实施工作，甚至还可能出现政策风险、执法困境等问题。另一方面，随着信息技术的发展、国际化进程加快，公众参与型政策的应用、推广也会越来越普遍、深入，这势必会加剧公众的不满情绪，从而增加政策制定者的维稳成本。因此，对于公众参与型政策的执行力来说，应从以下几个方面进行评估。

# 3. 执行力评价方法

## 3.1 误区

### 错误认识

1. 简单粗暴的执行

    在很多情况下，简单的执行效果往往比预期的要好，譬如对房价上涨过快的政策的执行，经常会造成房价的快速上涨；对大学生的补贴政策的执行，往往会导致大量资源浪费；对垄断性企业的补贴政策的执行，往往会削弱了企业的竞争力。这些例子告诉我们，简单的执行往往会产生非常不好的结果。

2. 忽视基础设施

    在公众参与型政策实施中，基础设施的建设、管理、维护也扮演着重要角色。但通常忽视基础设施建设对政策的影响是最常见的错误认识。比如，没有建立起有效的就业培训机制，就很难使得劳动力得到充分培养，从而导致人口老龄化，工资低下；没有提升市容环境和交通便利，则可能会导致公共交通拥堵，车流恶化；没有落实好公共卫生事业，则很容易导致卫生资源短缺，群众反感、抗议，社会效益可能受损。总结起来，建设基础设施既不是政策制订的目的，也不能单独作为考虑因素。

### 不可靠的数据

公众参与型政策的实施往往依赖于多个渠道的数据。但由于不同渠道的可靠程度不同，数据质量也有差异。比如，政策制定的决定往往要由决策层级部门、职能部门、工作群体、个人来做出最终判断，如果决策层级部门之间存在较大的分歧、数据来源失真、预测模型存在偏差等情况，就会导致政策执行的不准确、不完整。而数据不准确、不完整的影响，也是公众参与型政策执行失败的一个重要原因。


### 偏向弱势群体

公众参与型政策往往着重于提升弱势群体的生活品质，但忽略了弱势群体的大多数人也有自己生活的需求，并且，公众参与型政策往往只针对少数人，而不是所有人。所以，公众参与型政策的执行效果往往受到大多数人的影响较小，且执行效果在时间上、空间上有所滞后。


### 刻板印象

公众参与型政策通常具有令人信服、畅销的形象，并且在很多场合都会被大众认同，不过，对于政策制定的目的、理念、实施方式、依据等问题，很多人都有自己的刻板印象。譬如，许多人认为“新政策”一定会带来好结果，“公民义务”背后一定还有别的道德义务；“公众参与”意味着“不服从”，其实“一切为了公众”。这些刻板印象会严重束缚公众参与型政策的执行力，导致制度设计、实施细节上的差错，导致政策执行效果的下滑。

## 3.2 方法概述

为了评价公众参与型政策的执行力，一种比较科学的方法是采用问卷调查的方式。也就是询问公众对政策的认同程度、满意度、改进建议，根据这些信息统计得到政策执行力的分数。但是，这种方式只能描述公众参与型政策的执行力，无法客观描述政策制定的目的、理念、制度设计、执法监督等方面的知识。因此，作者基于对现有的公众参与型政策的了解，提出了如下评价方法：

- “鼓舞力”（Engagement）——政策对公众参与的影响力。包括政策的宣传强度、活动时长、参与人数、参与率、参与倾向等。

- “动机效益”（Motivation Efficiency）——政策的社会效益，即使对某些弱势群体的参与也可以带来经济、社会、文化、心理上的改变。

- “公共价值”（Public Value）——政策对社会整体的公益价值。公众参与型政策能否实现公共价值的实现与否，取决于政府的决策逻辑是否公正、符合公众利益，以及政策制定的实施方式是否合理、能够发挥作用。

- “参与要求”（Participation Requirement）——政策实施之前的要求。包括满足参与条件的门槛、实施时间、接受方式、选举限制、适用对象、审批流程、年龄限制、文化背景限制、职业限制等。

- “仲裁权”（Arbitration Power）——公众参与型政策的司法复核权力。当政策执行出现问题时，公众是否有仲裁权进行投诉、申辩、调解等，以及是否能获得公正裁判。

# 4. 相关案例分析

本文将以两例案例来说明相关方法：

## 4.1 教育收费补贴政策的执行力评价

教育收费补贴政策是近几年来全国重点关注的热点话题之一，主要用于帮助低收入家庭解决子女学费问题。《第一财经》杂志曾经刊登过一组数据显示，截止2020年底，我国已有超过6亿名孩子受助于收费补贴。但是，2019年末全国7月份的新冠疫情爆发之后，学校关闭、校外培训陷入停顿，教育收费补贴政策似乎成为了最艰巨的任务。

根据《教育部办公厅关于建立健全我省义务教育收费补贴制度的意见》，教育收费补贴制度是指对义务教育阶段学生从幼儿园到高中毕业的费用按照不同的标准予以补贴，以提高教育收入。2018年，我省财政局接到了省教委的通知，发布了教育收费补贴方案，将义务教育阶段费用补贴的比例由30%提高到35%左右，从而增加补贴基数，缓解广大群众的负担。这一政策方案后来被媒体报道后获得了很大的声誉，被誉为“百步移天换一茶”。

但是，实际执行过程中，政策执行力的评价尚未得到专业机构的认可。《教育部办公厅关于建立健全我省义务教育收费补贴制度的意见》规定，每年公布义务教育阶段学生每生一次的补贴金额，但并未具体说明补贴金额如何计算。而我省实施的教育收费补贴，如何确定“每生一次”？参与人数、平均补贴金额、补贴方式等因素都不清楚。这让人们怀疑，“每生一次”是一个固定的数量，还是根据实际情况有所变化？

如果我们仔细分析一下该政策的执行过程，就会发现，无论是减免学费还是补贴费用，都存在巨大的不确定性和歧视性。首先，很多人并不知道自己应缴纳多少钱才能享受到补贴，有的人为了少付几块钱就能够享受到补贴，有的人根本就没有缴纳一分钱！其次，补贴的对象群体也不尽相同，有的是劳动力短缺的边远山区、县城的贫困户，有的是在西北角的贫困村、乡镇，还有的是重度残疾人。再者，不同的人群享受到的补贴金额也不同，有的人贫困户每分钱都补贴不到，有的人只有几十块钱就获得补贴，有的人却可以每年一次性补贴好几百块钱！这就导致，整个补贴制度并没有真正做到公平、公正。

另外，教育收费补贴政策的实施还存在严重的政治风险。由于补贴额度的增加，敢于缴纳教育费用的群体将越来越少，而在缴纳教育费用的群体则越来越多。这样，很多人缴纳的款项就会成为腐败的棺材，使得政府政策得不到有效执行。

因此，虽然有关部门的不断宣传教育收费补贴政策，使得民众对这个政策的认识得到加深，但实际执行力的评价仍然存在缺陷。我们可以看到，除非政府制定出具科学的执行计划和预算，否则公众参与型政策的执行力仍然是一个大问题。

## 4.2 煤炭改革政策的执行力评价

中国的煤炭改革政策是20世纪末期最具代表性的政治事件之一。改革开放初期，国家启动了大规模的煤炭普查，试图通过专项拨款、普查制度、奖惩措施等途径彻底解决煤炭供给不足的问题。2008年《中国煤炭报》援引中央领导人邓小平的话说，必须坚持节约优先、保护先进，切实发展煤炭。2009年6月，国家主席江泽民在谈到该政策时表示，“目前，一些地方的煤价已经走低到每吨3元上下，全国2010年煤价平均每吨5元左右”。但是，由于政策不力，大量国企煤矿无力回天。

为了解决这一问题，2011年，中共中央、国务院作出了《关于深化煤炭价格改革的若干意见》，提出“煤炭改革”八条方针：

1. 节约优先：坚持以节约为主，坚持公平竞争，确保基本公共品供应充足。
2. 普查价格：全面普查煤炭价格，建立统一价格机制。
3. 发展煤炭：发展煤炭产业，使价格向着合理均衡发展。
4. 公平价格：针对不同行业、不同地区、不同煤炭属性设置公平交易规则。
5. 灌溉优先：紧盯国内煤炭建设项目灌溉工程。
6. 提高效益：改善政企合作，降低企业经营成本，提高经济效益。
7. 深化电气化：强化煤炭电气化改造，形成绿色煤电。
8. 大力整治：对煤矿企业违法乱纪行为和不良行为深度整治。

但是，如何客观评价该政策的执行力呢？在制定政策时，如何识别政策的“错误”所在，如何避免“一刀切”的效果？应该采取哪些举措提升煤炭效益，降低企业经营成本，确保国际公平竞争？政策实施之后，如何运用法律手段保障公众的知情权？最后，如何衡量政策的“公正性”、“效率性”和“社会承认度”，有哪些政策要关注？

本文认为，不管是煤炭改革政策还是教育收费补贴政策，关键不在于标榜政策的“科学性”，而是在于对政策实施的全面把握，以及建立“科学有效的机制”。只有当民众真正能够明白，政策的“错误”在哪里，并且能够提出切实可行的改进方案时，公众参与型政策的执行力才会真正提高。
