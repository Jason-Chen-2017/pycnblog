# 2.术语说明
首先，需要理解一些基础术语，比如向量（Vector）、矩阵（Matrix）、张量（Tensor）等。向量就是一组数字，通常表示为[x1, x2,..., xi]；矩阵是一个二维数组，通常表示为[[a, b], [c, d]]或(a,b),(c,d)之类的形式；而张量则是一个多维数组。在本文中，我们主要关注向量、矩阵和标量（Scalar）。

# 3.基本操作方法
## 3.1 加法
加法操作在线性代数中代表着两个向量元素对应位置相加，运算结果是一个新向量，每个元素等于两个被加数相应位置的元素相加。例如，对于向量[2,3,4]+[1,-1,2]，得到的新向量是[3,2,6].

```python
v1 = [2,3,4]
v2 = [1,-1,2]
v3 = [sum(i) for i in zip(v1, v2)]
print(v3) # output: [3,2,6]
```

上述代码使用zip函数将两个向量对应的元素一一对齐，然后用列表推导式计算两个元素的加和，最后构造新的向量v3。

## 3.2 减法
减法操作也是线性代数中的重要运算符，其计算方式同样依赖于元素级差值，但输出是一个向量，其中每一个元素等于第一个向量相应位置的元素减去第二个向量相应位置的元素。例如，对于向量[2,3,4]-[1,-1,2]，得到的新向量是[1,4,2]:

```python
v1 = [2,3,4]
v2 = [1,-1,2]
v3 = [i-j for i, j in zip(v1, v2)]
print(v3) # output: [1,4,2]
```

这里也是用列表推导式计算两个元素的差值，并构造新的向量v3。

## 3.3 标量乘法
标量乘法是线性代数中最简单的操作，其运算结果是所有元素都乘以一个常数，输出仍然是一个向量，但所有的元素都乘以这个常数。例如，对于向量[2,3,4]*3，得到的新向量是[6,9,12]:

```python
v1 = [2,3,4]
s = 3
v2 = [i*s for i in v1]
print(v2) # output: [6,9,12]
```

同样也是用列表推导式计算每个元素乘以常数s，并构造新的向量v2。

## 3.4 矩阵乘法
矩阵乘法是线性代数中的另一个重要运算符，用于两个矩阵相乘或者一个矩阵与向量相乘。如果第一个矩阵的列数等于第二个矩阵的行数，那么就可以计算出第三个矩阵。例如，对于下面的矩阵乘法，A=(1,2), B=(-3,4;5,-6)，C=AB=((1*-3+2*5),(1*4+2*-6))，即A和B矩阵相乘得到C矩阵。

```python
import numpy as np

A = [[1,2]]
B = [[-3,4],[5,-6]]
C = np.dot(A,B)
print(C) # output: [[27, -29],[-21, 23]]
```

这里，np.dot()函数实现了矩阵乘法。
