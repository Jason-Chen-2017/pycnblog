# 1.简介
  

# 在深度学习领域，随着硬件性能的提升，GPU 的普及，深度神经网络的快速发展，使得深度学习技术得到迅速发展。深度学习模型越来越复杂，参数量越来越多，而传统的服务器硬件资源并不能很好地满足这些需求。基于此，云计算或分布式系统架构开始成为主流架构。Kubernetes 是目前最流行的开源容器编排系统之一，它可以提供资源管理、调度、服务发现等功能。它可以在多台机器上部署应用容器，通过 Kubernetes API 可以轻松地管理集群上的容器化应用。
