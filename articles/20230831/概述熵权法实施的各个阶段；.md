# 1.简介
  
:
“熵”是统计学中用于衡量信息不确定性的度量单位，其在概率论、信息论、热力学等领域都有重要的应用。熵权法是一种公正分配机制，即通过赋予决策者不同的权利值，来实现对资源的合理分配。熵权法的首次提出是在1979年，但是到2019年，已成为我国最具影响力的经济制度。

在上世纪90年代初期，中国面临着严重的产业结构转型危机，产业结构的变化带来了新的就业机会和人才需求，因此，政府决定借鉴西方的制度设计，引入熵权法。熵权法共分四个阶段，分别是探索阶段（1992-2005），发明阶段（2006-2012），试点阶段（2013-2015），广泛推行阶段（2016-至今）。下面，将详细阐述熵权法的相关理论和实践，并分享一些关键信息。

2.概述
熵权法是一套分配机制，用来解决分配资源的问题。资源可以是工资、股票、房地产等。在当时，由于城乡二元经济结构的形成，使得个人的收入差距很大，这对于分配资源具有一定的困难。随着社会的进步，人们开始接受教育，也获得了更多的技能和知识，因此，资源的分配更加公平和有效。同时，作为市场化改革的先驱者，政府也逐渐认识到，给予决策者不同的权利价值，能够更好的管理资源。

1979年，熵权法由美国经济学家霍兰德·科斯提出，指出劳动力可以按照自己的能力以及相互之间的相互作用的结果，被排名、奖励、赋予不同权利值，从而达到公平地对待各种需要资源的人。

熵权法由下列三个方面组成：

①个人自由权利——包括拥有个人收入、居住条件、医疗保健福利等个人生活方面的自由权利；

②集体自由权利——包括追求集体利益，比如参与公共事务、制定集体标准、议事协调等；

③合作自由权利——包括维持共同财富，鼓励人们互助合作等。

熵权法的工作方式如图1所示。根据熵权法所定义的个人、集体、合作权利，每个单位和个人都可以决定自己的权利价值，并依据这些权利价值对资源进行分配。例如，单位可以向个人提供更多的晚餐时间或午饭时间，提高职工待遇；个人也可以决定是否放弃个人收入，向合作伙伴分享他的金钱或物品。

3.熵权法的运作方式
## 3.1 发明阶段（1992-2005）：
在此阶段，该法由瑞士经济学家拉斐尔·费洛姆提出。1992年，费洛姆开始尝试对效用函数进行建模，研究人员发现如果考虑每个人的能力水平、生活品质和历史贡献，可以准确预测他们未来的收入水平。但是，费洛姆的模型忽略了社会的其他因素，例如，收入差距、成本差距、相互依赖关系、组织结构、政治体系等。

为了引入这些社会因素，费洛姆与兰德公司合作，提出了一种基于收益分布的公正分配机制，称为“金字塔效应”。该机制认为，越高层级的人拥有越多的权利，同时，也越容易享受到资源的优惠。于是，他提出了“金字塔收入分配”的概念。该法包括三个层级：最底层的是社区居民，享有最低限度的权利，但却不能享受特别的待遇；第二层级是商界精英，享有较高的收入权利，而且还可以在政治上与其他层级发生联系；第三层级是官僚阶层，拥有最高收入权利。

此外，为了减少不公平现象的发生，费洛姆建议限制企业或组织的竞争。为了做到这一点，费洛姆提出了一个“财务创造权”，允许企业或组织创造足够的财富，来支付给更有资格的人。为了防止个人无意义的消费，他还设立了一个“信息支付权”，允许个人在其他层级进行交易，但不会直接获得回报。

## 3.2 试点阶段（2006-2012）：
在2006年，加拿大的艾伯森大学（Ecole d’Abitbol）与英国剑桥大学（Cambridge University）合作，试验了熵权法在中国的运行效果。

该法最早起源于美国，1979年，费洛姆在加拿大艾伯森大学发表了关于公正分配问题的论文，而在2006年，他又在剑桥大学发现了这个概念。

第一批试点单位包括苏州银行、太仓证券、中国银行、江苏银行，他们纷纷试验熵权法，并取得积极反馈。但随着试点进行，一些试点单位出现了不公平现象，并且，很多人的观感是“国家干涉个人权利，造成不公平和负面影响”。因此，试点模式被终止。

## 3.3 广泛推行阶段（2013-2015）：
经过几年的试点实验，熵权法进入了广泛推行阶段。

在2013年，中国政府采纳了这套机制。主要目标是引导国有经济转变成非营利性的社会企业，为社会创造更好的就业机会。为此，中央政府在2013年启动了“国企改革”，并下发了《国有企业国有资产监督管理暂行办法》，从2014年起，国有企业在招投标、行政管理等方面均实行熵权管理。

为了避免经济规模扩张对税收征管和金融服务的影响，2013年，中央财政部印发了《关于减轻地方政府财政压力的若干意见》，明确了“完善地方财政收支监管体系，增强地方政府财税自主权，坚决遏制对企业税收的过度补助，防止不必要的资源浪费”。

在2014年，联邦财政部批准了《联邦储备法》，并发布《储备资金管理办法》，明确了“联邦储备委员会依据宪法设置储备金，并建立储备金管理制度，对国内金融机构及非金融机构的储备资金进行分类管理，审批并核准政府债券投资的准备金”。

另一方面，2014年，海关总署颁布了《海关进口货物临时出口许可办法》，允许海关管理部门临时运输商品，不需要特殊手续。

在2015年，中央财政部、国务院印发了《关于进一步减轻地方政府财政压力的若干意见》，宣布启动地方政府财政自筹计划，并提出完善地方财政收支监管体系的要求。

4.最后的启示
1.熵权法适用范围广泛：从发明阶段到当前阶段，熵权法已经得到全社会的广泛关注和支持。该法利用经济学中的效用函数理论，分析个人的能力、生活品质和历史贡献，为公众和集体创造了公平且可预测的收入。

2.熵权法的实施策略有待提升：在各地推行熵权法时，存在很多不可控因素，诸如基础设施缺陷、法律风险等等。如何有效降低不良后果、保持灵活适应、及时纠错等，都是需要改进的地方。

3.熵权法是一种经济制度创新：熵权法的提出和实施，充满了创新的思维。它不仅创造了一个新的经济制度，而且也在发展新经济领域、促进公平竞争方面发挥了巨大的作用。
