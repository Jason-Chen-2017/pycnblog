# 3.基本概念术语说明
## Python的特点
- 简单易学：Python语言被设计得非常简单，而且有大量的学习资源，使初学者可以很快上手。
- 动态编译：Python是一门动态编译语言，源文件只需一次编译便可执行。
- 可移植：Python可以编译成字节码，在不同的平台之间运行，还可以将Python程序转换成二进制码，随时可运行。
- 可扩展：Python提供了丰富的类库和扩展库，可以满足不同领域的需求。
- 支持面向对象编程：Python支持面向对象编程，支持类、继承和多态特性。
- 广泛应用：Python已被广泛应用于web开发、数据分析、脚本开发、系统管理、自动化测试等领域。

## Python的版本及发展历程
### Python 2和Python 3
Python从20世纪90年代末起就已经存在了，目前，仍然有大量的代码依赖于Python 2，而现在，Python 2将会逐渐淘汰。2008年1月份，也就是Python 2.7的正式发布，Python 3.0版本正式推出，并在2008年底前后成为主流版本。

#### Python 2.x版本特征
- 使用print语句来输出结果；
- 文件读取使用`file = open('filename')`方法打开，然后`data = file.read()`方式读取文件内容；
- 数据类型不区分整数和浮点数，单个数字可以是任何类型，例如3 + 4.0 = 7.0；
- 没有模块（modules）的概念；
- 没有异常处理机制；
- 不支持Unicode字符串；
- 不能处理多线程；

#### Python 3.x版本特征
- print()函数改名为print()，不需要加括号；
- 在输出文本时，使用函数str()来将其他类型的数据转化为字符串；
- 可以导入不同模块，不必像Python 2.x版本一样全部导入；
- 默认编码方式为UTF-8；
- 支持语法上的优化，比如类定义中的`class Person:`改为`class Person(object):`，字典键值对中冒号后不用加空格；
- 支持异常处理机制；
- 支持多线程；
