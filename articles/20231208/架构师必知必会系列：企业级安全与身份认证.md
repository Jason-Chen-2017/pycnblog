## 4.1 对称加密实例

对称加密是一种使用相同密钥进行加密和解密的加密方法。在企业级安全与身份认证中，对称加密主要用于加密用户密码，以确保密码信息在传输和存储过程中的安全性。

### 4.1.1 代码实例

```python
from Crypto.Cipher import AES

def encrypt(plaintext, key):
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(plaintext)
    return cipher.nonce, ciphertext, tag

def decrypt(nonce, ciphertext, tag, key):
    cipher = AES.new(key, AES.MODE_EAX, nonce=nonce)
    plaintext = cipher.decrypt_and_verify(ciphertext, tag)
    return plaintext
```

### 4.1.2 解释说明

在上述代码中，我们使用了Python的Crypto库来实现对称加密。`encrypt`函数用于加密明文，`decrypt`函数用于解密密文。

`encrypt`函数首先创建一个AES加密对象，然后使用`encrypt_and_digest`方法进行加密。`encrypt_and_digest`方法返回密文、验证标签和非对称加密对象的非对称加密对象。

`decrypt`函数首先创建一个AES加密对象，然后使用`decrypt_and_verify`方法进行解密。`decrypt_and_verify`方法返回明文。
