期望是随机变量的一个性质，它表示随机变量的平均值。期望可以是数值的，也可以是函数的。方差是随机变量的一个性质，它表示随机变量的离散性。方差可以是数值的，也可以是函数的。

## 3.5 期望
期望是随机变量的一个性质，它表示随机变量的平均值。期望可以是数值的，也可以是函数的。期望的计算公式为：

$$
E[X] = \sum_{i=1}^{n} x_i \cdot P(x_i)
$$

## 3.6 方差
方差是随机变量的一个性质，它表示随机变量的离散性。方差可以是数值的，也可以是函数的。方差的计算公式为：

$$
Var[X] = E[X^2] - (E[X])^2
$$

# 4.具体代码实例和详细解释说明
在这里，我们将通过Python代码来实现随机变量的概念、性质、分类、概率分布、期望、方差等概念。

```python
import numpy as np

# 定义随机变量
X = np.random.normal(loc=0, scale=1, size=1000)

# 计算随机变量的概率分布
prob_dist = np.histogram(X, bins=20, density=True)

# 计算随机变量的期望
expectation = np.mean(X)

# 计算随机变量的方差
variance = np.var(X)

# 输出结果
print("随机变量的概率分布:", prob_dist)
print("随机变量的期望:", expectation)
print("随机变量的方差:", variance)
```
