                 

# 1.背景介绍

大数据是指由大量、高速、多样化的数据组成的数据集，这些数据的规模和复杂性超出了传统的数据处理技术的处理能力。大数据处理的核心技术是数据模型与数据建模。数据模型是大数据处理的基础，数据建模是大数据处理的核心。

数据模型是对数据结构和数据关系的抽象，是数据处理的基础。数据模型包括关系模型、图模型、网络模型、时间序列模型等。数据建模是将实际问题转化为数据模型的过程，是数据处理的核心。数据建模包括数据收集、数据清洗、数据转换、数据分析等。
