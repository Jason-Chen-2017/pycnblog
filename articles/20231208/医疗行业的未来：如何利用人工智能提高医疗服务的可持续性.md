## 4.1 医疗资源分配算法实例

以下是一个简单的医疗资源分配算法实例：

```python
def allocate_resources(resources, demands):
    allocated_resources = {}
    for region, demand in demands.items():
        available_resources = resources[region]
        allocated_resources[region] = min(available_resources, demand)
    return allocated_resources
```

在这个算法中，我们首先定义了一个`allocate_resources`函数，它接受两个参数：`resources`和`demands`。`resources`是一个字典，其中键是地区，值是该地区的医疗资源数量。`demands`是一个字典，其中键是地区，值是该地区的医疗需求数量。

我们然后创建了一个空字典`allocated_resources`，用于存储分配后的医疗资源数量。我们遍历`demands`字典，为每个地区分配合适的医疗资源数量。我们使用`min`函数找到最小的可分配资源数量，并将其分配给该地区。

最后，我们返回`allocated_resources`字典，其中包含了每个地区分配的医疗资源数量。

## 4.2 医疗服务质量评估算法实例

以下是一个简单的医疗服务质量评估算法实例：

```python
def evaluate_quality(quality_indicators, weights):
    total_score = 0
    for indicator, weight in zip(quality_indicators, weights):
        total_score += indicator * weight
    return total_score
```

在这个算法中，我们首先定义了一个`evaluate_quality`函数，它接受两个参数：`quality_indicators`和`weights`。`quality_indicators`是一个列表，其中包含了各个质量指标的得分。`weights`是一个列表，其中包含了各个质量指标的权重。

我们然后创建了一个变量`total_score`，用于存储评估结果。我们遍历`quality_indicators`和`weights`列表，将各个质量指标的得分与其对应的权重相乘，并将结果累加到`total_score`中。

最后，我们返回`total_score`，即医疗服务的总质量得分。
