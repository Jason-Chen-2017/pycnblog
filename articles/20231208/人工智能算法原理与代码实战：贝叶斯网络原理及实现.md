# 2.核心概念与联系

贝叶斯网络（Bayesian Network）是一种概率图模型，用于表示和推理随机变量之间的条件依赖关系。它的核心概念包括：

1. 随机变量：随机变量是一个可能取多个值的变量。在贝叶斯网络中，每个节点都表示一个随机变量。

2. 条件依赖关系：条件依赖关系是随机变量之间的关系，表示一个变量的状态对另一个变量的状态有影响。在贝叶斯网络中，每条边都表示一个条件依赖关系。

3. 条件概率：条件概率是一个随机变量在另一个随机变量取某个值时，自身取某个值的概率。在贝叶斯网络中，每个节点的概率表示是基于其父节点的状态计算的。

4. 贝叶斯定理：贝叶斯定理是贝叶斯网络的核心数学原理，用于计算条件概率。贝叶斯定理表示为：P(A|B) = P(B|A) * P(A) / P(B)。

贝叶斯网络与其他概率图模型（如马尔可夫网络、隐马尔可夫模型等）的联系是：

1. 概率图模型：贝叶斯网络、马尔可夫网络、隐马尔可夫模型等都是概率图模型，用于表示随机变量之间的关系。

2. 条件独立性：马尔可夫网络和隐马尔可夫模型是基于条件独立性的概率图模型，而贝叶斯网络是基于条件依赖性的概率图模型。

3. 贝叶斯定理：贝叶斯网络、马尔可夫网络和隐马尔可夫模型都使用贝叶斯定理来计算条件概率。

假设我们要构建一个医学诊断的贝叶斯网络，随机变量包括：

2. 病理检查结果（Pathology Result）
3. 诊断结果（Diagnosis）

我们可以按照以下步骤构建贝叶斯网络：

1. 确定随机变量：我们确定了三个随机变量：症状、病理检查结果和诊断结果。

2. 确定条件依赖关系：我们可以根据实际情况确定条件依赖关系。例如，症状可能会影响病理检查结果，病理检查结果可能会影响诊断结果。

3. 确定条件概率：我们可以根据实际情况为每个随机变量的每个状态分配一个条件概率。例如，症状可能出现的概率为0.5，病理检查结果可能出现的概率为0.6，诊断结果可能出现的概率为0.7。

4. 构建贝叶斯网络：我们可以使用Python的pomegranate库来构建贝叶斯网络。首先，我们需要导入pomegranate库：

```python
from pomegranate import *
```

然后，我们可以根据我们的随机变量和条件依赖关系来构建贝叶斯网络：

```python
# 构建症状节点
```
