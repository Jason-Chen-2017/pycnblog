# 2.核心概念与联系

在数据仓库安全与隐私保护中，有几个核心概念需要我们了解：

1.数据安全：数据安全是指数据仓库系统中的数据不被非法访问、篡改或泄露。数据安全包括数据完整性、可用性和机密性等方面。

2.数据隐私：数据隐私是指数据仓库系统中的数据不被非法访问或泄露，以保护用户的个人信息和隐私。数据隐私包括数据保密性和数据披露限制等方面。

3.数据安全性：数据安全性是指数据仓库系统中的数据不被非法访问、篡改或泄露，并且数据的完整性、可用性和机密性得到保障。

4.数据保护法：数据保护法是一种法律法规，规定了数据仓库系统中的数据应该如何保护和处理，以确保数据的安全和隐私。

5.隐私法：隐私法是一种法律法规，规定了数据仓库系统中的数据应该如何保护和处理，以确保用户的个人信息和隐私得到保护。
