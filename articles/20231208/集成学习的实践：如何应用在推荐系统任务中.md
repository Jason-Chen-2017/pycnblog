以下是一个具体的代码实例：

```python
import numpy as np
from sklearn.ensemble import RandomForestRegressor
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

# 训练数据
X = np.random.rand(1000, 10)
y = np.random.rand(1000)

# 基本推荐算法1：线性回归
lr = LinearRegression()
lr.fit(X, y)

# 基本推荐算法2：随机森林
rf = RandomForestRegressor()
rf.fit(X, y)

# 集成学习方法：加权平均法
w1 = 0.5
w2 = 0.5
y_pred = w1 * lr.predict(X) + w2 * rf.predict(X)

# 评估指标：均方误差
mse = mean_squared_error(y, y_pred)
print("Mean Squared Error:", mse)
```

在这个代码实例中，我们首先生成了一组训练数据，然后选择了两个基本推荐算法：线性回归和随机森林。接下来，我们使用加权平均法将这两个基本推荐算法的预测结果进行加权求和，得到最终的推荐结果。最后，我们使用均方误差作为评估指标来评估推荐系统的性能。
