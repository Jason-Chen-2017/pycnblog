# 4.具体代码实例和详细解释说明

以下是一个简单的Go命令行工具示例，用于查看当前目录下的文件列表：

```go
package main

import (
    "fmt"
    "os"
    "os/exec"
)

func main() {
    cmd := exec.Command("ls")
    cmd.Stdout = os.Stdout
    cmd.Stderr = os.Stderr
    err := cmd.Run()
    if err != nil {
        fmt.Println("Error:", err)
        os.Exit(1)
    }
}
```

这个示例中，我们使用了`os/exec`包来执行`ls`命令，并将其输出和错误输出分别设置为标准输出和标准错误输出。如果命令执行失败，我们将打印错误信息并退出程序。
