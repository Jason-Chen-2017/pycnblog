```python
class RuleEngine:
    def __init__(self, rules):
        self.rules = rules
        self.working_memory = {}

    def match(self, condition):
        for rule in self.rules:
            if condition in rule:
                return rule
        return None

    def execute(self, rule):
        if rule:
            action = rule['action']
            data = self.working_memory.get(rule['data'], None)
            if data:
                action(data)

    def run(self, conditions):
        for condition in conditions:
            rule = self.match(condition)
            if rule:
                self.execute(rule)
```
在这个代码实例中，我们定义了一个`RuleEngine`类，它包含了规则库、工作内存等组成部分。我们还实现了`match`和`execute`方法，用于匹配和执行规则。
