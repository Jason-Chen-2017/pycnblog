# 2.核心概念与联系

5.机器学习：机器学习是指利用计算机程序来自动学习从数据中抽取信息，以便进行预测和决策。

6.自然语言处理：自然语言处理是指计算机对自然语言进行理解和生成的技术，用于处理舆情数据中的文本信息。
