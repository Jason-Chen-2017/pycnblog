1. 资源分配算法实现：

```python
def resource_allocation(users, resources):
    allocation = []
    for user in users:
        available_resources = []
        for resource in resources:
            if resource.available and resource.match(user):
                available_resources.append(resource)
        allocation.append(available_resources)
    return allocation
```

2. 用户评价算法实现：

```python
def user_evaluation(users, evaluations):
    user_scores = {}
    for user in users:
        score = 0
        for evaluation in evaluations:
            if evaluation.user == user:
                score += evaluation.score
        user_scores[user] = score
    return user_scores
```

3. 价格竞争算法实现：

```python
def price_competition(market, demand):
    prices = []
    for resource in market:
        price = 0
        for demand_item in demand:
            if demand_item.resource == resource:
                price += demand_item.price
        prices.append(price)
    return prices
```
