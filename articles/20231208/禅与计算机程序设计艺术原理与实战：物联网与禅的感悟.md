首先，我们需要设计一个智能的计算机程序，这个程序可以根据物联网中的数据进行实时分析和处理。我们可以使用Python语言来编写这个程序，并使用Scikit-learn库来进行机器学习和深度学习的训练。

```python
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 加载数据
data = pd.read_csv('data.csv')

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(data.drop('label', axis=1), data['label'], test_size=0.2, random_state=42)

# 训练模型
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)

# 评估模型
y_pred = model.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)
print('Accuracy:', accuracy)
```

其次，我们需要设计一个分布式的计算机程序，这个程序可以在多个计算机上进行并行处理。我们可以使用Python语言来编写这个程序，并使用PySpark库来实现分布式计算。

```python
from pyspark import SparkContext
from pyspark.sql import SparkSession

# 创建SparkContext
sc = SparkContext('local', 'myApp')

# 创建SparkSession
spark = SparkSession(sc)

# 读取数据
data = spark.read.csv('data.csv', header=True, inferSchema=True)

# 划分训练集和测试集
(train, test) = data.randomSplit([0.8, 0.2], seed=42)

# 训练模型
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(train)

# 评估模型
accuracy = model.evaluate(test, {model.getParam('metric'): 'accuracy'})
print('Accuracy:', accuracy)
```

最后，我们需要设计一个实时的计算机程序，这个程序可以在接收到新数据时立即进行处理。我们可以使用Python语言来编写这个程序，并使用Kafka和Flink库来实现实时处理。

```python
from kafka import KafkaProducer
from kafka import KafkaConsumer
from flink.streaming import StreamExecutionEnvironment
from flink.table import StreamTableEnvironment

# 创建Kafka生产者
producer = KafkaProducer(bootstrap_servers='localhost:9092')

# 创建Kafka消费者
consumer = KafkaConsumer('my_topic', bootstrap_servers=['localhost:9092'], value_deserializer=lambda m: m.decode('utf-8'))

# 创建Flink流处理环境
env = StreamExecutionEnvironment.get_execution_environment()
t_env = StreamTableEnvironment.create(env)

# 从Kafka中读取数据
data = t_env.read_kafka('my_topic')

# 进行数据处理
data = data.select('field1', 'field2', 'field3')

# 写入Kafka
data.write_kafka('my_topic')

# 启动Flink流处理任务
env.execute('my_app')
```
