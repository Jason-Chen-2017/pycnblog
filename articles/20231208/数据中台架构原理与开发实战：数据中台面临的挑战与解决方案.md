# 2.核心概念与联系

数据中台的核心概念包括：数据源、数据仓库、数据服务、数据应用等。

1.数据源：数据源是数据中台的基础设施，包括数据库、数据仓库、数据湖等。数据源负责数据的收集、存储和管理。

2.数据仓库：数据仓库是数据中台的核心组件，负责数据的处理、分析和应用。数据仓库将数据源的数据进行清洗、转换、聚合等操作，提供给数据应用进行使用。

3.数据服务：数据服务是数据中台的服务层，负责提供数据的访问和操作接口。数据服务将数据仓库的数据提供给数据应用进行使用，实现数据的统一管理和共享。

4.数据应用：数据应用是数据中台的应用层，负责实现企业的业务需求。数据应用通过访问数据服务，获取数据仓库的数据，实现数据的应用和分析。

数据中台的核心概念之间的联系如下：

3.数据服务与数据应用的联系：数据服务是数据中台的服务层，负责提供数据的访问和操作接口。数据应用是数据中台的应用层，负责实现企业的业务需求。数据服务与数据应用之间通过API等技术进行数据的访问和操作。
