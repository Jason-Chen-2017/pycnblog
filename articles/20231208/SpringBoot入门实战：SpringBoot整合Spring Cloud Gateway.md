                 

# 1.背景介绍

Spring Boot是Spring官方推出的一款快速开发框架，它的目的是简化Spring应用程序的开发，使开发人员能够快速地创建独立的Spring应用程序，而无需关注配置。Spring Boot提供了许多有用的功能，例如自动配置、嵌入式服务器、缓存管理、安全性、元数据、监控和管理等。

Spring Cloud Gateway是Spring Cloud的一部分，它是一个基于Spring 5的WebFlux网关，它提供了对Spring Cloud服务的路由、熔断、监控等功能。它的核心功能是提供一个基于Spring Boot的网关，可以将请求路由到多个服务，并提供负载均衡、安全性、监控等功能。
