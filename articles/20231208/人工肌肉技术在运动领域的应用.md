```python
import time
import RPi.GPIO as GPIO

# 设置GPIO引脚
GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)

# 设置PWM对象
pwm = GPIO.PWM(17, 50)

# 设置PWM占空比
duty_cycle = 50

# 启动PWM
pwm.start(duty_cycle)

# 控制PWM占空比
while True:
    duty_cycle += 1
    if duty_cycle > 100:
        duty_cycle = 0
    pwm.ChangeDutyCycle(duty_cycle)
    time.sleep(0.1)

# 停止PWM
pwm.stop()

# 清除GPIO引脚
GPIO.cleanup()
```

在这个示例中，我们使用了Raspberry Pi的GPIO库来控制一个电机的转速。我们首先设置了GPIO引脚，然后设置了PWM对象，接着设置了PWM占空比，并启动了PWM。在循环中，我们不断增加PWM占空比，以实现电机的转速控制。最后，我们停止了PWM并清除了GPIO引脚。
