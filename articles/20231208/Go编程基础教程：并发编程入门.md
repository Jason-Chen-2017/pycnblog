# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释Go语言的并发编程。

```go
package main

import (
	"fmt"
	"sync"
)

func main() {
	var wg sync.WaitGroup
	wg.Add(1)
	go func() {
		fmt.Println("Hello, World!")
		wg.Done()
	}()
	wg.Wait()
}
```

在上述代码中，我们创建了一个Goroutine，它会打印出"Hello, World!"字符串，然后调用`wg.Done()`来表示Goroutine已经完成执行。最后，我们调用`wg.Wait()`来等待所有Goroutine完成执行。
