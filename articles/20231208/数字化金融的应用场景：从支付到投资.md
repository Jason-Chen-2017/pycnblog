# 2.核心概念与联系

在数字化金融中，有几个核心概念需要我们了解：

1.数字货币：数字货币是一种电子货币，它不需要物理形式的货币来进行交易。数字货币的主要特点是去中心化、可匿名化、高度可扩展性等。比特币是目前最知名的数字货币。

2.数字支付：数字支付是指通过数字设备（如手机、平板电脑等）进行的支付。数字支付的主要特点是方便、快速、安全等。常见的数字支付方式有支付宝、微信支付等。

3.数字贷款：数字贷款是指通过数字设备（如手机、平板电脑等）申请的贷款。数字贷款的主要特点是简单、快速、便捷等。常见的数字贷款平台有蚂蚁花呗、京东贷等。

4.数字投资：数字投资是指通过数字设备（如手机、平板电脑等）进行的投资。数字投资的主要特点是便捷、高效、透明等。常见的数字投资平台有阿里巴巴的天猫金融、腾讯的腾讯金融等。

这些概念之间存在着密切的联系。例如，数字货币可以用于数字支付、数字贷款和数字投资等场景。同时，数字支付、数字贷款和数字投资也可以通过数字设备进行。因此，这些概念相互联系，共同构成了数字化金融的应用场景。

```python
def digital_payment(user_id, amount, timestamp):
    # 生成一个随机的key
    key = os.urandom(16)
    # 生成一个HMAC对象
    h = hmac.new(key, msg=timestamp.encode('utf-8'), digestmod=hashlib.sha256)
    # 计算HMAC的值
    signature = base64.b64encode(h.digest())
    # 生成一个数字支付的请求
    request = {
        'user_id': user_id,
        'amount': amount,
        'timestamp': timestamp,
        'signature': signature
    }
    # 返回数字支付的请求
    return request
```
