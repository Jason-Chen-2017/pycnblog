## 1. 核心概念与联系

在Python中，模块是一个包含一组相关功能的代码集合，可以通过导入的方式将其加载到当前的程序环境中，从而实现代码的重用和模块化。模块的导入可以让我们在编写程序时，只需要引入所需的模块，而不需要重新编写相关的功能代码。这样可以大大提高程序的可读性、可维护性和可扩展性。

### 1.1 模块的导入与使用

Python模块的导入通常使用`import`关键字来完成。例如，如果我们想要导入`os`模块，可以使用以下代码：

```python
import os
```

导入后，我们可以通过使用`模块名.函数名`的方式来调用模块中的函数。例如，我们可以使用`os.path`模块来获取文件路径相关的功能。

```python
import os

file_path = os.path.join('/home/user', 'file.txt')
print(file_path)  # /home/user/file.txt
```
