# 3. 核心算法原理和具体操作步骤以及数学模型公式详细讲解
# 3.1 先来先服务（FCFS）算法原理
先来先服务（FCFS）算法是一种简单的排队算法，它的原理是：客户按照到达时间顺序排队，先到达的客户先被服务。

# 3.2 先来先服务（FCFS）算法具体操作步骤
1. 当客户到达时，将客户加入队列。
2. 当服务器空闲时，将队列中第一个客户分配给服务器。
3. 当客户被服务完成时，将客户从队列中移除。
4. 重复步骤2和3，直到队列中所有客户都被服务完成。

# 3.3 先来先服务（FCFS）算法数学模型公式
排队长度：L = (λ / (μ - λ))
平均排队时间：W = L / λ = (λ / (μ - λ)) / λ = 1 / (μ - λ)

# 3.4 最短作业优先（SJF）算法原理
最短作业优先（SJF）算法是一种优先级调度算法，它的原理是：客户按照预计服务时间顺序排队，预计服务时间较短的客户优先被服务。

# 3.5 最短作业优先（SJF）算法具体操作步骤
1. 当客户到达时，将客户加入队列。
2. 计算每个客户的预计服务时间。
3. 将预计服务时间较短的客户放在队列头部。
4. 当服务器空闲时，将队列中第一个客户分配给服务器。
5. 当客户被服务完成时，将客户从队列中移除。
6. 重复步骤3-5，直到队列中所有客户都被服务完成。

# 3.6 最短作业优先（SJF）算法数学模型公式
排队长度：L = (λ / (μ - λ))
平均排队时间：W = L / λ = (λ / (μ - λ)) / λ = 1 / (μ - λ)

# 4. 具体代码实例和详细解释说明
# 4.1 先来先服务（FCFS）算法代码实例
```python
import math

def fcfs(arrival_times, service_times):
    queue = []
    for t in arrival_times:
        queue.append((t, service_times))
    queue.sort()
    waiting_times = []
    total_waiting_time = 0
    for t, service_time in queue:
        waiting_time = max(0, t - total_waiting_time)
        total_waiting_time += waiting_time + service_time
        waiting_times.append(waiting_time)
    return waiting_times
```

```python
import math

def sjf(arrival_times, service_times):
    queue = []
    for t, service_time in sorted(zip(arrival_times, service_times), key=lambda x: x[1]):
        queue.append((t, service_time))
    waiting_times = []
    total_waiting_time = 0
    for t, service_time in queue:
        waiting_time = max(0, t - total_waiting_time)
        total_waiting_time += waiting_time + service_time
        waiting_times.append(waiting_time)
    return waiting_times
```
