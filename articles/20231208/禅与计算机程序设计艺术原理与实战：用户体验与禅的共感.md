```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

2. 然后，我们使用一个 `for` 循环来遍历列表中的每个元素。这个循环的目的是将每个元素插入到已排序的有序序列中。

3. 在每次循环中，我们首先将当前元素存储在一个名为 `key` 的变量中。

4. 然后，我们使用一个 `while` 循环来找到当前元素应该插入的位置。这个循环的条件是 `j >= 0` 和 `arr[j] > key`。

5. 在每次循环中，我们将有序序列中大于当前元素的元素向右移动一个位置，以腾出足够的空间来插入当前元素。

6. 最后，我们将当前元素插入到有序序列的正确位置，并继续下一次循环。
