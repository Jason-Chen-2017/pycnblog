3.基于知识的推荐算法：基于知识的推荐算法利用外部知识（如用户的兴趣、物品的特征等）来推断用户的兴趣。这种算法通常使用知识图谱、关系学习等技术来实现。

2.数据预处理：对数据进行清洗、转换和特征提取，以便于后续的算法计算。

3.算法训练：根据选定的推荐算法，对数据进行训练，以便于生成推荐列表。

4.推荐列表生成：根据训练后的算法，为用户生成推荐列表。


