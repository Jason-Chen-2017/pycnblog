import numpy as np

def calculate_position(A1, A2, B1, B2, C1, C2):
    x = (A1 - A2) / 2
    y = (B1 - B2) / 2
    z = (C1 - C2) / 2
    return x, y, z

# 示例数据
A1 = 1.0
A2 = 2.0
B1 = 3.0
B2 = 4.0
C1 = 5.0
C2 = 6.0

# 计算无人机位置
X, Y, Z = calculate_position(A1, A2, B1, B2, C1, C2)
print("无人机位置：X =", X, "Y =", Y, "Z =", Z)
```

在这个代码实例中，我们定义了一个名为`calculate_position`的函数，用于计算无人机的位置。该函数接受六个参数，分别是无人机与卫星或基地站之间的距离。我们使用Numpy库进行数值计算。
