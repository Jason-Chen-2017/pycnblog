### 4.1 代码实例

我们将通过一个简单的排序算法——选择排序来进行说明。选择排序是一种简单的排序算法，它的时间复杂度是O(n^2)，其中n是输入序列的长度。

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if arr[min_idx] > arr[j]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr
```

### 4.2 详细解释说明

选择排序的主要思想是在每次迭代中找到数组中最小的元素，并将其与当前位置进行交换。这样，每次迭代后，数组的最小元素都会被放置在正确的位置。

选择排序的时间复杂度是O(n^2)，因为在每次迭代中，内层循环需要遍历整个数组。选择排序的空间复杂度是O(1)，因为它不需要额外的空间。

选择排序的稳定性是不稳定的，因为在某些情况下，它可能会改变输入序列的顺序。
