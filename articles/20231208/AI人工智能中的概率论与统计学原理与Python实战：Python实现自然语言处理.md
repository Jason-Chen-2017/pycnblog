# 2.核心概念与联系

## 2.1概率论

概率论是数学的一个分支，用于描述事件发生的可能性。概率是一个数值，表示事件发生的可能性。概率的范围是0到1，表示事件发生的可能性。概率的计算方法有多种，如频率法、定义法、比例法等。

## 2.2统计学

统计学是一门数学学科，研究的是从数据中抽取信息，并用这些信息来描述和预测现实世界的事物。统计学可以分为描述性统计学和推断性统计学。描述性统计学用于描述数据的特征，如平均值、中位数、方差等。推断性统计学则用于根据样本数据来推断总体参数。

## 4.1概率论

### 4.1.1定义法

```python
from math import factorial

def probability_definition(event, total):
    if event == 0 or total == 0:
        return 0
    return factorial(total) / (factorial(event) * factorial(total - event))

event = 2
total = 6
print(probability_definition(event, total))  # 0.16666666666666666
```

### 4.1.2比例法

```python
def probability_ratio(event, total):
    if event == 0 or total == 0:
        return 0
    return event / total

event = 2
total = 6

```

## 4.2统计学

### 4.2.1描述性统计学

```python
import numpy as np

data = np.array([1, 2, 3, 4, 5, 6])

# 平均值
mean = np.mean(data)
print(mean)  # 3.5

# 中位数
median = np.median(data)

# 方差
variance = np.var(data)
print(variance)  # 2.6666666666666665

# 标准差
std_dev = np.std(data)
print(std_dev)  # 1.606060606060606
```

### 4.2.2推断性统计学

```python
import numpy as np

# 样本数据
sample_data = np.array([1, 2, 3, 4, 5, 6])

# 总体均值
population_mean = 3.5

# 样本均值
sample_mean = np.mean(sample_data)
print(sample_mean)  # 3.5

# 样本标准差
sample_std_dev = np.std(sample_data)
print(sample_std_dev)  # 1.606060606060606

# 样本大小
sample_size = len(sample_data)

# 样本的基本统计量
sample_stats = np.array([sample_mean, sample_std_dev, sample_size])
print(sample_stats)  # [3.5 1.606060606060606 6]

# 假设检验
# 单样本t检验
t_statistic = (sample_mean - population_mean) / (sample_std_dev / np.sqrt(sample_size))
print(t_statistic)  # -0.0002739725685327478

# 双边t检验的p值
p_value = 2 * (1 - scipy.stats.t.cdf(abs(t_statistic), df=sample_size - 1))
print(p_value)  # 0.9997256853274782

# 结论
if p_value > 0.05:
    print("不能拒绝零假设")
else:
    print("可以拒绝零假设")
```
