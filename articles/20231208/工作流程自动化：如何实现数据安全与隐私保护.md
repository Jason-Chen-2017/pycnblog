### 3.1 哈希算法

哈希算法是一种用于将数据转换成固定长度的字符串的算法，常用于数据加密和数据脱敏。哈希算法的核心原理是将输入数据通过一系列的运算得到输出结果，输出结果是不可逆的。常见的哈希算法有MD5、SHA1等。

### 3.2 加密算法

加密算法是一种用于将数据转换成不可读的形式的算法，常用于数据加密和数据安全。加密算法的核心原理是将输入数据通过一系列的运算得到输出结果，输出结果是可以通过解密算法解密的。常见的加密算法有AES、RSA等。

### 3.3 脱敏算法

脱敏算法是一种用于将个人信息从原始数据中删除或替换的算法，常用于数据脱敏和隐私保护。脱敏算法的核心原理是将输入数据通过一系列的运算得到输出结果，输出结果是不包含个人信息的。常见的脱敏算法有姓名脱敏、身份证脱敏等。

### 3.4 数据分类算法

数据分类算法是一种用于将数据分为不同的类别的算法，常用于数据分类和数据安全。数据分类算法的核心原理是将输入数据通过一系列的运算得到输出结果，输出结果是数据的类别。常见的数据分类算法有决策树、支持向量机等。

### 4.1 数据加密代码实例

```python
import base64
from Crypto.Cipher import AES

def encrypt(data, key):
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(data)
    return base64.b64encode(cipher.nonce + tag + ciphertext).decode('utf-8')

data = "hello world"
key = b'1234567890abcdef'
encrypted_data = encrypt(data, key)
print(encrypted_data)
```

### 4.3 数据分类代码实例

```python
from sklearn.tree import DecisionTreeClassifier

def train_classifier(X, y):
    classifier = DecisionTreeClassifier()
    classifier.fit(X, y)
    return classifier

X = [[1, 2], [3, 4], [5, 6]]
y = [0, 1, 0]
classifier = train_classifier(X, y)
predicted = classifier.predict([[7, 8]])
print(predicted)
```
