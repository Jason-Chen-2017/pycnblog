# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释Python的代码规范与最佳实践。

假设我们要编写一个简单的计算器程序，可以加、减、乘、除四种基本运算。我们可以按照以下步骤来编写这个程序：

1. 定义一个函数，接受两个数字和一个运算符作为参数，并返回计算结果。
2. 使用if-else语句来判断运算符，并执行相应的计算。
3. 使用print函数来输出计算结果。

以下是这个程序的具体代码实例：

```python
def calculator(num1, num2, operator):
    if operator == '+':
        return num1 + num2
    elif operator == '-':
        return num1 - num2
    elif operator == '*':
        return num1 * num2
    elif operator == '/':
        return num1 / num2
    else:
        return 'Invalid operator'

num1 = 5
num2 = 3
operator = '+'
result = calculator(num1, num2, operator)
print(result)
```

在这个代码实例中，我们遵循了Python的代码规范与最佳实践：

- 我们使用了合适的命名约定，例如将函数名称设为`calculator`，变量名称设为`num1`、`num2`和`operator`。
- 我们使用了4个空格的代码格式化，并将每个语句放在单独的行上。
- 我们使用了文档注释来描述函数的功能和参数。
- 我们使用了if-else语句来判断运算符，并执行相应的计算。
- 我们使用了print函数来输出计算结果。

通过这个代码实例，我们可以看到Python的代码规范与最佳实践是如何帮助我们编写更好的代码的。
