$$
\Delta x_t = x_t - x_{t-1}
$$

其中，$x_t$ 是时间序列的第 $t$ 个数据点，$x_{t-1}$ 是时间序列的第 $t-1$ 个数据点。

## 3.2 移动平均

移动平均是一种用于平滑时间序列中噪声的方法。它通过计算数据的平均值来实现这一目的。移动平均的公式为：

$$
MA_t = \frac{1}{w} \sum_{i=-(w-1)}^{w-1} x_{t-i}
$$

其中，$MA_t$ 是时间序列的第 $t$ 个数据点的移动平均值，$w$ 是移动平均窗口的大小，$x_{t-i}$ 是时间序列的第 $t-i$ 个数据点。

def difference(x):
    dt = np.diff(x)
    return dt

print(difference_result)
```

import numpy as np

def moving_average(x, window_size):
    window = np.ones(window_size)/window_size
    return np.convolve(x, window, 'valid')

window_size = 3
moving_average_result = moving_average(x, window_size)
print(moving_average_result)
```

在这个例子中，我们使用 NumPy 库来实现移动平均。我们首先定义了一个数组 `x`，然后使用 `np.ones()` 函数来创建一个窗口数组，窗口大小为 `window_size`。最后，我们使用 `np.convolve()` 函数来计算数组的移动平均值。
