# 2.核心概念与联系
数据立方体是一种多维数据模型，它可以有效地表示和处理大规模的数据。数据立方体的核心概念包括：

- 数据维度：数据立方体可以包含多个维度，每个维度表示数据的不同属性。例如，在销售数据中，可能有多个维度，如产品、地理位置和时间。
- 数据粒度：数据立方体可以表示不同粒度的数据。例如，在销售数据中，可以表示总销售额、月销售额和日销售额等不同粒度的数据。
- 数据聚合：数据立方体可以通过聚合操作将多个维度的数据聚合到不同的粒度。例如，可以将产品、地理位置和时间三个维度的数据聚合到总销售额、月销售额和日销售额等不同的粒度。

```python
# 查询数据立方体
query_result = np.sum(data_cube, axis=2)
print(query_result)

# 更新数据立方体

print(query_result)
```
