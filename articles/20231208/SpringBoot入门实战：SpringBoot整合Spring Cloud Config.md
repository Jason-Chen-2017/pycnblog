                 

# 1.背景介绍

Spring Boot是Spring框架的一个快速开发的子项目，它的目标是简化Spring应用程序的开发，同时提供生产级别的依赖关系和配置。Spring Boot 2.0.0正式发布，Spring Cloud Config客户端支持Spring Boot 2.0。

Spring Cloud Config是Spring Cloud的一个组件，它提供了集中化的配置管理服务，可以让开发者在一个中心化的位置管理应用程序的所有配置，而不是在每个应用程序中单独维护配置文件。这使得开发者可以更轻松地更新和管理应用程序的配置，而无需重新部署每个应用程序。

Spring Cloud Config客户端可以与Spring Cloud Config服务器集成，以便从服务器获取配置。这使得开发者可以在运行时更新配置，而无需重新部署应用程序。
