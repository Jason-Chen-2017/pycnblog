## 1.2 核心概念与联系

在本节中，我们将讨论以下核心概念：

- 神经元
- 神经网络
- 前馈神经网络
- 反馈神经网络
- 深度学习

### 1.2.1 神经元

神经元是人工神经网络的基本组成单元。它接收输入信号，对这些信号进行处理，并输出结果。神经元由输入层、隐藏层和输出层组成。

### 1.2.2 神经网络

神经网络是由多个相互连接的神经元组成的计算模型。它通过模拟人脑中神经元的工作方式来解决复杂的问题。神经网络可以用来进行预测、分类和决策等任务。

### 1.2.3 前馈神经网络

前馈神经网络（Feedforward Neural Network）是一种简单的神经网络，它的输入和输出之间没有循环连接。输入通过隐藏层传递到输出层，而输出层的结果就是神经网络的预测结果。

### 1.2.4 反馈神经网络

反馈神经网络（Recurrent Neural Network）是一种复杂的神经网络，它的输入和输出之间存在循环连接。这种类型的神经网络可以处理序列数据，如自然语言和时间序列数据。

### 1.2.5 深度学习

深度学习是一种机器学习方法，它使用多层神经网络来解决复杂的问题。深度学习可以用来进行图像识别、自然语言处理和语音识别等任务。

### 1.3.1 反向传播算法

反向传播算法（Backpropagation）是一种用于训练神经网络的算法。它通过计算神经网络的梯度来优化神经网络的权重和偏置。反向传播算法的核心步骤如下：

1. 对于每个输入样本，计算输出层的预测结果。
2. 计算输出层的损失函数值。
3. 使用链式法则计算每个神经元的梯度。
4. 更新神经网络的权重和偏置。

### 1.3.2 梯度下降算法

梯度下降算法（Gradient Descent）是一种用于优化函数的算法。它通过在函数的梯度方向上移动来逐步减小函数的值。梯度下降算法的核心步骤如下：

1. 初始化神经网络的权重和偏置。
2. 计算神经网络的损失函数值。
3. 计算神经网络的梯度。
4. 更新神经网络的权重和偏置。
5. 重复步骤2-4，直到损失函数值达到一个满足要求的值。

### 1.3.3 激活函数

激活函数（Activation Function）是神经网络中的一个关键组成部分。它用于将神经元的输入转换为输出。常用的激活函数有：

- 步函数（Step Function）
-  sigmoid 函数（Sigmoid Function）
-  hyperbolic tangent 函数（Hyperbolic Tangent Function）
-  ReLU 函数（Rectified Linear Unit Function）

### 1.3.4 损失函数

损失函数（Loss Function）是用于衡量神经网络预测结果与实际结果之间差异的函数。常用的损失函数有：

- 均方误差（Mean Squared Error）
- 交叉熵损失（Cross-Entropy Loss）
- 对数损失（Log Loss）

3. 对于每个神经元，将该神经元的输入与其偏置相加，并将结果传递到下一层。

5. 对于最后一层的神经元，将其输出作为神经网络的预测结果。

3. 对于最后一层的神经元，将其输出与实际结果进行比较，并计算损失函数值。
4. 对于每个神经元，将该神经元的输入与其偏置相加，并将结果传递到下一层。
