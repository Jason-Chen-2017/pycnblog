```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if arr[min_idx] > arr[j]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(selection_sort(arr))
```

1. 我们首先定义了一个名为`selection_sort`的函数，它接受一个数组作为参数。
2. 在函数内部，我们使用两个循环来实现选择排序算法的核心操作：选择最小元素并交换其位置。
3. 最后，我们调用`selection_sort`函数，并将一个数组作为参数传递给它。
