                 

# 1.背景介绍

虚拟货币（Cryptocurrency）是一种基于数字技术的货币，它不受任何国家或地区的管理和监督。虚拟货币的创新主要体现在其去中心化的特点，它不受任何中央银行或政府的控制，而是通过分布式账本技术（Distributed Ledger Technology，DLT）实现了去中心化的交易和支付系统。虚拟货币的创新者通过开发新的加密货币和区块链技术，为全球金融市场创造了新的价值和机会。
