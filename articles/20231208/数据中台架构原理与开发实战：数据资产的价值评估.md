在数据中台架构中，数据资产价值评估的具体代码实例如下：

```python
import numpy as np
import pandas as pd

# 数据资产的价值评估
def data_value_evaluation(data_resources, weight, value_indicator):
    data_value = []
    for i in range(len(data_resources)):
        data_value.append(weight[i] * value_indicator[i])
    return np.sum(data_value)

# 数据资产的价值评估结果
data_resources = ['数据库', '数据仓库', '数据湖']
weight = [0.3, 0.4, 0.3]
value_indicator = [1000, 2000, 3000]

data_value = data_value_evaluation(data_resources, weight, value_indicator)
print(data_value)
```

上述代码实例中，我们首先定义了一个数据资产价值评估的函数，该函数接受数据资产、价值权重和价值指标作为输入参数，并计算出数据资产的价值。然后，我们定义了一组数据资产、价值权重和价值指标，并调用数据资产价值评估函数进行计算。最后，我们打印出数据资产的价值。
