### 选择排序代码实例

选择排序是一种简单的排序算法，它的时间复杂度为O(n^2)。以下是选择排序的Python代码实例：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
print(selection_sort(arr))
```

### 插入排序代码实例

插入排序是一种简单的排序算法，它的时间复杂度为O(n^2)。以下是插入排序的Python代码实例：

```python
def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr

arr = [5, 2, 8, 1, 9]
print(insertion_sort(arr))
```

### 冒泡排序代码实例

冒泡排序是一种简单的排序算法，它的时间复杂度为O(n^2)。以下是冒泡排序的Python代码实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [5, 2, 8, 1, 9]
print(bubble_sort(arr))
```
