### 4.1 代码实例

我们来看一个简单的代码实例，实现一个基于规则的决策系统。

```python
# 定义规则
rules = [
    {
        "name": "rule1",
        "condition": "a > b",
        "action": "insert into table1 (a, b) values (x, y)"
    },
    {
        "name": "rule2",
        "condition": "c < d",
        "action": "insert into table2 (c, d) values (z, w)"
    }
]

# 监听事件
event = "data_updated"

# 触发规则
for rule in rules:
    if event == "data_updated":
        if eval(rule["condition"]):
            # 判断条件满足
            print("条件满足")
            # 执行结果
            exec(rule["action"])
            print("执行结果")
```

### 4.2 详细解释说明

在这个代码实例中，我们首先定义了一些规则，每个规则包括名称、条件和结果。然后，我们监听了一个事件，这个事件是数据更新。接着，我们遍历了所有的规则，如果事件满足规则的条件，则执行规则的结果。

这个代码实例展示了规则引擎的基本实现过程，包括规则的定义、事件的监听、规则的触发、条件的判断、结果的执行。
