## 1.4 具体代码实例和详细解释说明

在第一次世界大战前的技术竞争中，各国利用了不同的代码实例来提高自己的军事实力和民用发展。这些代码实例的详细解释说明如下：

1. 电子技术：

   代码实例：

   ```python
   import numpy as np
   from scipy.signal import butter, lfilter

   def butter_lowpass(cutoff, fs, order=5):
       nyq = 0.5 * fs
       normal_cutoff = cutoff / nyq
       b, a = butter(order, normal_cutoff, btype='low', analog=False)
       return b, a

   def lowpass_filter(data, cutoff, fs, order=5):
       b, a = butter_lowpass(cutoff, fs, order=order)
       y = lfilter(b, a, data)
       return y
   ```

   详细解释说明：

   - 导入numpy和scipy.signal库，以便使用数学函数和滤波器。
   - 定义一个名为butter_lowpass的函数，用于创建低通滤波器。这个函数接受cutoff、fs和order参数，cutoff表示截止频率，fs表示采样频率，order表示滤波器的阶数。
   - 定义一个名为lowpass_filter的函数，用于对数据进行低通滤波。这个函数接受data、cutoff、fs和order参数，data表示需要滤波的数据，cutoff、fs和order参数与butter_lowpass函数相同。
