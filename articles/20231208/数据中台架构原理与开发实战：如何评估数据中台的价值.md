                 

# 1.背景介绍

数据中台是一种新兴的数据技术架构，它主要解决企业数据资源的整合、管理、分享和应用等问题。数据中台的核心思想是将数据资源作为企业重要资产进行管理，通过数据资源的统一管理，实现数据的高效运用，提高企业的数据利用价值。
