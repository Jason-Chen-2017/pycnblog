假设我们有一个包含学生成绩的数据集，数据集中包含学生的学号、姓名、成绩等信息。我们需要对这个数据集进行清洗和质量保证，以提高数据质量。

首先，我们需要对数据集进行清洗，包括填充、去重、规范化和清洗等步骤。具体代码实例如下：

```python
import pandas as pd
import numpy as np

# 读取数据集
data = pd.read_csv('student_scores.csv')

# 填充缺失值
data['score'].fillna(data['score'].mean(), inplace=True)

# 去重
data.drop_duplicates(inplace=True)

# 规范化
data['score'] = (data['score'] - data['score'].min()) / (data['score'].max() - data['score'].min())

# 清洗
data['score'] = data['score'] - data['score'].std()
```

