# 4.具体代码实例和详细解释说明
以下是一个简单的网络安全代码实例，它是一个简单的MD5加密算法实现：

```python
import hashlib

def md5_encrypt(data):
    m = hashlib.md5()
    m.update(data.encode('utf-8'))
    return m.hexdigest()

if __name__ == '__main__':
    data = 'Hello, World!'
    encrypted_data = md5_encrypt(data)
    print(encrypted_data)
```

这个代码实例中，我们使用Python的hashlib库来实现一个简单的MD5加密算法。我们首先创建一个MD5对象，然后使用update方法来更新对象的内容，最后使用hexdigest方法来获取加密后的结果。
