tokens = word_tokenize(text)

在这个代码实例中，我们使用NLTK库来实现知识发现算法。我们首先导入stopwords、word_tokenize和PorterStemmer模块，然后使用set方法来创建一个停用词集合，使用word_tokenize方法来分词，使用for循环来去除停用词，最后使用PorterStemmer类来进行词干提取。
