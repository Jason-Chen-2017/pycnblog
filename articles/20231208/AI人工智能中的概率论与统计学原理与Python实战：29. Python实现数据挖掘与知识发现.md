## 3.1概率论的核心算法原理
### 3.1.1概率的定义
概率是一个事件发生的可能性，它通常表示为一个数值，范围在0到1之间。

### 3.1.2概率的计算
概率可以通过不同的方法来计算，例如：

1.直接计数法：通过计算事件发生的次数和总次数，得到概率。

2.定义域法：通过计算事件发生的概率空间，得到概率。

3.条件概率法：通过计算事件发生的条件概率，得到概率。

### 3.1.3概率的性质
概率具有以下性质：

1.非负性：概率必须大于等于0。

2.单位性：概率必须小于等于1。

3.完全性：概率必须等于1。

## 3.2统计学的核心算法原理
### 3.2.1均值
均值是数据集中所有数据点的平均值，用于表示数据集的中心趋势。

### 3.2.2方差
方差是数据集中所有数据点与均值之间的平均差值的平方，用于表示数据集的不确定性。

### 3.2.3标准差
标准差是方差的平方根，用于表示数据集的不确定性的度量。

### 3.2.4相关性
相关性是两个变量之间的关系，用于表示它们之间的联系。

## 3.3概率论与统计学的数学模型公式
### 3.3.1概率论的数学模型公式
1.概率的定义：P(A) = n(A) / n(S)

2.概率的计算：P(A) = n(A) / n(S)

3.条件概率的计算：P(A|B) = P(A∩B) / P(B)

### 3.3.2统计学的数学模型公式
1.均值的计算：μ = Σ(xi) / N

2.方差的计算：σ² = Σ((xi - μ)²) / N

3.标准差的计算：σ = √(σ²)

4.相关性的计算：r = Σ((xi - μ)(yi - μ)) / (Nσxσy)

## 4.1概率论的代码实例
### 4.1.1概率的定义
```python
from math import factorial

def probability(n, r):
    return factorial(n) / (factorial(r) * factorial(n - r))

print(probability(6, 3))  # 输出：0.16666666666666666
```

### 4.1.2概率的计算
```python
from random import randint

def probability_calculation(n, r):
    count = 0
    for _ in range(n):
        if randint(1, 6) == r:
            count += 1
    return count / n

print(probability_calculation(1000, 3))  # 输出：0.166
```

### 4.1.3概率的性质
```python
def probability_properties(n, r):
    if n < 0 or r < 0:
        return "非负性违反"
    elif n == 0 or r == 0:
        return "单位性违反"
    elif n == r:
        return "完全性违反"
    else:
        return "正常"

print(probability_properties(1, 0))  # 输出："非负性违反"
```

## 4.2统计学的代码实例
### 4.2.1均值
```python
def mean(data):
    return sum(data) / len(data)

data = [1, 2, 3, 4, 5]
print(mean(data))  # 输出：3.0
```

### 4.2.2方差
```python
def variance(data):
    mean_value = mean(data)
    return sum((x - mean_value) ** 2 for x in data) / len(data)

data = [1, 2, 3, 4, 5]
print(variance(data))  # 输出：2.0
```

### 4.2.3标准差
```python
def standard_deviation(data):
    return (variance(data)) ** 0.5

data = [1, 2, 3, 4, 5]
print(standard_deviation(data))  # 输出：1.0
```

### 4.2.4相关性
```python
def correlation(x, y):
    n = len(x)
    mean_x = sum(x) / n
    mean_y = sum(y) / n
    return sum((x[i] - mean_x) * (y[i] - mean_y) for i in range(n)) / (n * standard_deviation(x) * standard_deviation(y))

x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]
print(correlation(x, y))  # 输出：1.0
```
