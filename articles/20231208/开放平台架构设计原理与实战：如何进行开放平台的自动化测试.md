```python
import unittest
import requests

class TestOpenPlatform(unittest.TestCase):
    def setUp(self):
        self.url = "http://example.com/api"

    def test_get(self):
        response = requests.get(self.url)
        self.assertEqual(response.status_code, 200)

    def test_post(self):
        data = {"key": "value"}
        response = requests.post(self.url, json=data)
        self.assertEqual(response.status_code, 201)

if __name__ == "__main__":
    unittest.main()
```

上述代码是一个使用Python的unittest模块进行开放平台的自动化测试的例子。这个例子包括了两个测试用例，一个是GET请求的测试用例，一个是POST请求的测试用例。这两个测试用例分别使用requests模块发送GET和POST请求，并检查响应的状态码是否为200和201。
