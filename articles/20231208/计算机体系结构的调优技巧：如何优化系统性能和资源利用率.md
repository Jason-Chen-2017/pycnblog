### 4.1 动态调度策略

我们可以使用以下代码实现动态调度策略：

```python
import queue

class Task:
    def __init__(self, priority):
        self.priority = priority

def dynamic_scheduling(tasks):
    tasks.sort(key=lambda x: x.priority)
    result = []
    while tasks:
        task = tasks.pop()
        result.append(task)
    return result

tasks = [Task(3), Task(1), Task(2)]
print(dynamic_scheduling(tasks))  # [Task(priority=1), Task(priority=2), Task(priority=3)]
```

在这个代码中，我们首先定义了一个`Task`类，用于表示任务的优先级。然后，我们实现了一个`dynamic_scheduling`函数，用于根据任务的优先级来调度任务。最后，我们创建了一个任务列表，并使用`dynamic_scheduling`函数来调度任务。
