
```python
np.random.seed(0)
X = np.random.normal(loc=0, scale=1, size=1000)

sigma = np.std(X)
z = (X - mu) / (sigma * np.sqrt(len(X)))
```

接下来，我们可以使用中心极限定理的公式来计算概率：

```python
prob = norm.cdf(z)
```
