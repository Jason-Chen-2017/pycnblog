## 4.4 数据挖掘代码实例

在本节中，我们将通过具体的代码实例来详细解释数据挖掘的实现过程。

### 4.4.1 数据挖掘代码实例1：聚类算法

```python
import pandas as pd
from sklearn.cluster import KMeans

# 数据
data = {'name': ['John', 'Alice', 'Bob', 'John', 'Alice', 'Bob'], 'age': [25, 30, 35, 25, 30, 35], 'salary': [5000, 6000, 7000, 5000, 6000, 7000]}

# 数据预处理
df = pd.DataFrame(data)
df_num = df[['age', 'salary']]

# 聚类算法
kmeans = KMeans(n_clusters=2, random_state=0).fit(df_num)

# 显示结果
print(kmeans.labels_)
```

### 4.4.2 数据挖掘代码实例2：关联规则算法

```python
import pandas as pd
from mlxtend.frequent_patterns import apriori
from mlxtend.frequent_patterns import association_rules

# 数据
data = {'name': ['John', 'Alice', 'Bob', 'John', 'Alice', 'Bob'], 'age': [25, 30, 35, 25, 30, 35], 'salary': [5000, 6000, 7000, 5000, 6000, 7000]}

# 数据预处理
df = pd.DataFrame(data)
df_items = df[['age', 'salary']]

# 关联规则算法
frequent_itemsets = apriori(df_items, min_support=0.5, use_colnames=True)
association_rules = association_rules(frequent_itemsets, metric="confidence", min_threshold=0.7)

# 显示结果
print(association_rules)
```
