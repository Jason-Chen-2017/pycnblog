$$
Reliability = \lim_{t\to\infty} P(no\ failures\ in\ interval\ [0,t])

## 4.1 代码实例

我们将通过一个简单的例子来说明框架的错误处理与容错的实现方法。假设我们有一个简单的计算器框架，它可以进行加法、减法、乘法、除法等四种基本运算。我们需要在框架中实现错误处理与容错的功能。

```python
class Calculator:
    def __init__(self):
        self.error_code = 0
        self.error_message = ""

    def add(self, a, b):
        try:
            result = a + b
            return result
        except Exception as e:
            self.error_code = 1
            self.error_message = str(e)
            raise

    def subtract(self, a, b):
        try:
            result = a - b
            return result
        except Exception as e:
            self.error_code = 2
            self.error_message = str(e)
            raise

    def multiply(self, a, b):
        try:
            result = a * b
            return result
        except Exception as e:
            self.error_code = 3
            self.error_message = str(e)
            raise

    def divide(self, a, b):
        try:
            result = a / b
            return result
        except Exception as e:
            self.error_code = 4
            self.error_message = str(e)
            raise
```

在这个代码实例中，我们定义了一个`Calculator`类，它有四个基本运算方法：`add`、`subtract`、`multiply`和`divide`。在每个运算方法中，我们使用`try-except`语句来捕获异常。如果在运算过程中发生异常，我们将设置错误代码和错误信息，并将异常重新抛出。

## 4.2 详细解释说明

在这个代码实例中，我们使用了`try-except`语句来捕获异常。当在运算过程中发生异常时，我们将设置错误代码和错误信息，并将异常重新抛出。这样，我们可以在调用运算方法时捕获异常，并根据错误代码和错误信息进行相应的处理。
