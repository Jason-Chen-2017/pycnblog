2. 如果客户的订单金额小于1000，但大于500，则将订单标记为“中等订单”。
3. 如果客户的订单金额小于或等于500，则将订单标记为“低额订单”。

我们可以使用以下的代码来表示这些规则：

```

  if order_amount > 1000 then
    label = "高额订单"
  end if

rule 2:
  if order_amount < 1000 and order_amount > 500 then
    label = "中等订单"
  end if

rule 3:
  if order_amount <= 500 then
    label = "低额订单"
  end if
```

在执行规则引擎时，我们需要提供一个事件，包括客户的订单信息。我们可以使用以下的代码来表示这个事件：

```

  order_amount = 800
```

根据事件和规则，我们可以通过以下的代码来执行规则引擎：

```

```

执行规则引擎后，我们可以得到以下的结果：

```
result:
  label = "中等订单"
```
