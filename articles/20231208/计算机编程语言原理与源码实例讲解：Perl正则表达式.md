# 4.具体代码实例和详细解释说明

在本节中，我们将通过具体的代码实例来详细解释Perl正则表达式的使用方法。

例1：匹配字符串中的数字

```perl
my $string = "123456";
if ($string =~ /\d+/) {
    print "匹配到数字\n";
} else {
    print "没有匹配到数字\n";
}
```

例2：替换字符串中的数字

```perl
my $string = "123456";
$string =~ s/\d+/X/g;
print $string;
```

例3：分割字符串

```perl
my $string = "123456";
@array = split /\d+/, $string;
print "@array\n";
```
