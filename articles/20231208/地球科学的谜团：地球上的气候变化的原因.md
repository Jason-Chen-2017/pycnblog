在气候科学中，常用的数学模型公式有以下几种：

1. 辐射平衡方程：$$
    S_0(1-a) = \sigma T^4
    $$
    其中，$S_0$ 是太阳辐射的强度，$a$ 是地球的反射率，$\sigma$ 是辐射常数，$T$ 是地球的温度。

2. 气候模型的基本方程：$$
    \frac{dT}{dt} = \frac{1}{c} (Q - G)
    $$
    其中，$T$ 是地球的温度，$t$ 是时间，$c$ 是地球的热容，$Q$ 是太阳辐射的强度，$G$ 是地球的热流。

3. 气候模型的动态方程：$$
    \frac{dT}{dt} = \frac{1}{c} (Q - G) + \frac{1}{c} (F - L)
    $$
    其中，$F$ 是人类因素对温度的影响，$L$ 是自然因素对温度的影响。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义参数
S0 = 1361.0  # 太阳辐射强度
a = 0.305  # 地球的反射率
sigma = 5.67e-8  # 辐射常数
c = 1.67e8  # 地球的热容

# 定义初始条件
T0 = 288.0  # 地球的初始温度
t0 = 0.0  # 时间

# 定义时间步长
dt = 1.0  # 时间步长

# 定义时间范围
t_end = 100.0  # 预测结束时间

# 初始化温度数组
T = np.zeros(int(t_end / dt) + 1)
T[0] = T0

# 循环计算温度
for t in range(1, int(t_end / dt) + 1):
    t = t * dt
    T[t] = T[t - 1] + (S0 * (1 - a) - T[t - 1]**4 * sigma * c) * dt

# 绘制温度变化曲线
plt.plot(T)
plt.xlabel('Time')
plt.ylabel('Temperature')
plt.title('Temperature Change')
plt.show()
```

接下来，我们初始化温度数组，并将地球的初始温度赋值给第一个元素。然后我们进入一个循环，每次循环计算地球的温度。在计算温度时，我们使用了辐射平衡方程，将太阳辐射强度、地球的反射率、地球的温度、辐射常数和地球的热容相乘，并将结果与时间步长相乘，得到温度的变化。最后，我们绘制温度变化曲线，以可视化地球的气候变化。
