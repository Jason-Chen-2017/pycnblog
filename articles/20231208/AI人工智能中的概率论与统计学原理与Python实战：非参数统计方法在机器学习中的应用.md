# 4.具体代码实例和详细解释说明
在Python中，可以使用Scikit-learn库来实现非参数统计方法。以下是一个使用Scikit-learn实现K-均值聚类的代码示例：

```python
from sklearn.cluster import KMeans
import numpy as np

# 创建数据集
X = np.array([[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]])

# 创建KMeans对象
kmeans = KMeans(n_clusters=2, random_state=0)

# 执行聚类
kmeans.fit(X)

# 获取聚类结果
labels = kmeans.labels_
print(labels)
```

在这个代码示例中，我们首先创建了一个数据集，然后创建了一个KMeans对象，并指定了要执行的聚类数量（n_clusters）和随机种子（random_state）。接着，我们执行了聚类操作，并获取了聚类结果。
