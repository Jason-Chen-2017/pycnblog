# 2.核心概念与联系

在人工智能诊断中，我们需要了解以下几个核心概念：

- 数据：数据是人工智能诊断的基础。我们需要收集并处理大量的数据，以便计算机能够学习和自动化决策。
- 特征：特征是数据中的一些特定属性，它们可以帮助计算机识别和分类问题。
- 模型：模型是人工智能诊断的核心组成部分。我们需要训练模型，以便它能够从数据中学习并预测未来事件或情况的发生。
- 评估：我们需要评估模型的性能，以便我们可以确定它是否适合解决给定问题。
