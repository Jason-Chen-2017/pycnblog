# 4.具体代码实例和详细解释说明

以下是一个禅意编程的代码实例：

```python
def fizzbuzz(n):
    for i in range(1, n + 1):
        if i % 3 == 0 and i % 5 == 0:
            print("FizzBuzz")
        elif i % 3 == 0:
            print("Fizz")
        elif i % 5 == 0:
            print("Buzz")
        else:
            print(i)
```

这个代码实例是一个简单的FizzBuzz游戏，它会输出从1到n的数字，如果数字是3的倍数，则输出"Fizz"，如果数字是5的倍数，则输出"Buzz"，如果数字是3和5的倍数，则输出"FizzBuzz"。

这个代码实例的解释如下：

1. 首先，我们定义了一个名为fizzbuzz的函数，它接受一个整数n作为参数。
2. 然后，我们使用for循环遍历从1到n的整数。
3. 接下来，我们使用if-elif-else语句来判断当前整数是否是3和5的倍数，如果是，则输出"FizzBuzz"，如果只是3的倍数，则输出"Fizz"，如果只是5的倍数，则输出"Buzz"，如果不是任何倍数，则输出当前整数。
4. 最后，我们使用print函数来输出当前整数或字符串。

这个代码实例是禅意编程的一个简单示例，它展示了如何通过简单、直观、自然和整体的思维方式来编写高质量的代码。
