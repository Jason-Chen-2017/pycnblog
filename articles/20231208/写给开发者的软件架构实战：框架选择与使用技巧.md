# 2.核心概念与联系

在开始讨论框架选择和使用技巧之前，我们需要了解一些核心概念。这些概念包括：

- 软件架构：软件架构是构建软件系统的蓝图，定义了系统的组件、它们之间的关系以及它们如何协同工作。
