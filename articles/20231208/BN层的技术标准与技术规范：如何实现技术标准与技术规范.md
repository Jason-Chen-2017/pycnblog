- 是否会下雨（Rain）
- 天气状况（Weather）

我们的条件依赖关系是：如果是下雨天，天气状况就是下雨；如果不是下雨天，天气状况就是晴天。我们的随机变量状态是：下雨（Rain）和晴天（Sunny）。我们的条件概率是：下雨天的概率为0.5，晴天的概率为0.5。

```python

# 创建随机变量
rain = pomegranate.distributions.DiscreteDistribution([0.5, 0.5])
weather = pomegranate.distributions.DiscreteDistribution([0.5, 0.5])

# 创建条件依赖关系
rain_to_weather = pomegranate.conditional.ConditionalProbabilityTable(rain, weather)

# 创建BN层模型
model = pomegranate.BayesianNetwork(rain_to_weather)
```
