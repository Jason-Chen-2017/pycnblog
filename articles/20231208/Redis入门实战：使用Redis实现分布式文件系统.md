Redis是一个开源的高性能的键值对存储系统，它支持数据的持久化，可基于内存（Volatile）或磁盘（Persistent）。Redis 提供多种语言的 API。Redis 可以作为数据库，缓存和消息队列的替代方案。Redis 支持数据的排序（Redis 支持定制排序），并提供API 实现数据的集合、差集、交集等。

Redis 是一个使用 ANSI C 语言编写、遵循 BSD 协议的开源（MIT 许可）、跨平台 Redis 数据库，服务器提供多种语言的 API。Redis 可以在内存中存储数据，也可以将数据持久化到磁盘。Redis 提供多种语言的 API，包括：C、C++、C#、Java、Python、Ruby、PHP、Node.js、Perl、Go、Objective-C、Swift 和 Elixir。

Redis 支持数据的排序（Redis 支持定制排序），并提供API 实现数据的集合、差集、交集等。Redis 客户端可以通过网络访问服务器来存储、修改和查询数据库。Redis 支持Pub/Sub（发布/订阅）功能，可以实现消息队列。Redis 支持主从复制，即主服务器可以复制数据到从服务器。Redis 支持Lua脚本（Redis 脚本）。Redis 支持数据的有限时间缓存。Redis
