# 2.核心概念与联系

在区块链技术中，核心概念包括：

- 区块：区块是区块链中的基本单元，包含一组交易数据和一个时间戳。每个区块包含前一个区块的哈希值，形成一个链式结构。
- 交易：交易是区块链中的基本操作，包括创建、验证和广播等步骤。交易可以包含多种类型的数据，如转账、合约执行等。
- 共识算法：共识算法是区块链中的一种机制，用于确定哪些交易是有效的，并添加到区块链中。最常用的共识算法有PoW（工作量证明）和PoS（状态证明）等。
- 加密技术：区块链技术使用加密技术来保护数据的安全性，包括公钥加密、私钥签名和哈希函数等。

- 共识算法：共识算法是区块链中的一种机制，用于确定哪些交易是有效的，并添加到区块链中。最常用的共识算法有PoW（工作量证明）和PoS（状态证明）等。
