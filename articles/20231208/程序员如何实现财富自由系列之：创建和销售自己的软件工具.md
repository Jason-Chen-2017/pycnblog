在这个部分，我们将提供一个具体的代码实例，并解释其工作原理。例如，我们可以使用Python语言来实现一个简单的计算器：

```python
class Calculator:
    def add(self, a, b):
        return a + b

    def subtract(self, a, b):
        return a - b

    def multiply(self, a, b):
        return a * b

    def divide(self, a, b):
        return a / b

calculator = Calculator()
result = calculator.add(5, 3)
print(result)  # 8
```
