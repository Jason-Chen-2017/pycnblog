```python
def solve(n):
    # 初始化结果
    result = 0
    # 遍历所有数字
    for i in range(1, n+1):
        # 计算结果
        result += i * i
    return result
```

在这个代码实例中，我们定义了一个名为`solve`的函数，它接受一个整数参数`n`。该函数计算从1到`n`的所有数字的平方和，并将结果返回。我们可以通过调用这个函数来计算任意一个数字的平方和。
