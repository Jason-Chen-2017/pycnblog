## 2.1 区块链

区块链是一种分布式、去中心化的数据存储和交易方式，它的核心思想是通过将数据存储在一个不可变的、链式结构的数据结构中，从而实现数据的完整性、可追溯性和透明度。区块链由一系列的区块组成，每个区块包含一组交易数据和一个指向前一个区块的指针。这种结构使得区块链具有一定的时间顺序和完整性。

## 2.2 加密货币

加密货币是基于区块链技术的一种数字货币，它的核心特点是通过加密算法来保证交易的安全性和隐私性。最著名的加密货币是比特币，它是第一个应用区块链技术的加密货币。其他常见的加密货币包括以太坊、比特币现金等。

## 4.1 创建加密货币地址

在创建加密货币地址时，我们需要使用哈希函数来生成地址。以下是一个使用ECDSA加密算法创建加密货币地址的代码实例：

```python
import hashlib
import ecdsa

# 生成私钥
private_key = ecdsa.SigningKey.generate(curve=ecdsa.SECP256k1)

# 生成公钥
public_key = private_key.verifying_key

# 生成地址
address = public_key.to_string().hex()

print(address)
```

在这个代码实例中，我们首先使用`ecdsa`库生成一个私钥。然后，我们使用私钥生成一个公钥。最后，我们使用公钥生成一个地址。地址是公钥的哈希值，它是加密货币的唯一标识。

在验证加密货币交易时，我们需要使用加密算法来验证交易签名。以下是一个使用ECDSA加密算法验证加密货币交易的代码实例：

```python
import hashlib
import ecdsa

# 生成私钥
private_key = ecdsa.SigningKey.generate(curve=ecdsa.SECP256k1)

# 生成公钥
public_key = private_key.verifying_key

# 生成地址

# 生成交易
transaction = {
    'from': address,
    'to': 'recipient_address',
```
