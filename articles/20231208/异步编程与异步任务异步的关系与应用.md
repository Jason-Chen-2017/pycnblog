## 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的异步任务示例来说明异步编程与异步任务异步的关系：

```python
import asyncio

async def task1():
    print("任务1开始执行")
    await asyncio.sleep(1)
    print("任务1执行完成")

async def task2():
    print("任务2开始执行")
    await asyncio.sleep(2)
    print("任务2执行完成")

async def main():
    task1_future = asyncio.create_task(task1())
    task2_future = asyncio.create_task(task2())

    print("任务1和任务2都开始执行")
    await task1_future
    await task2_future
    print("所有任务执行完成")

asyncio.run(main())
```

在这个示例中，我们创建了两个异步任务：`task1` 和 `task2`。这两个任务都是异步执行的，它们可以在等待其他任务完成之前开始执行。我们通过 `asyncio.create_task` 函数来创建这两个任务，并在主任务中等待它们的完成。

当我们运行这个示例时，我们可以看到任务1和任务2都开始执行，并在它们执行完成之后，所有任务执行完成。这个示例说明了异步任务如何在等待其他任务完成之前开始执行，从而实现了异步编程的目的。
