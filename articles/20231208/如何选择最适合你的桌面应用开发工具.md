1.基于GUI的开发工具：例如，使用Qt库开发桌面应用程序，我们需要编写以下代码：

```cpp
#include <QApplication>
#include <QLabel>

int main(int argc, char *argv[])
{
    QApplication app(argc, argv);
    QLabel label("Hello, World!");
    label.show();
    return app.exec();
}
```

在这个例子中，我们创建了一个QApplication对象，然后创建了一个QLabel对象并显示了它。最后，我们调用app.exec()函数来运行应用程序。

2.基于代码的开发工具：例如，使用Python编程语言开发桌面应用程序，我们需要编写以下代码：

```python
import tkinter as tk

def hello():
    print("Hello, World!")

root = tk.Tk()
button = tk.Button(root, text="Hello", command=hello)
button.pack()
root.mainloop()
```

在这个例子中，我们使用了tkinter库来创建一个简单的GUI应用程序。我们创建了一个按钮，当用户点击它时，会打印出“Hello, World!”。

