                 

# 1.背景介绍

SpringBoot是Spring框架的一个子项目，是一个用于快速构建Spring应用程序的框架。SpringBoot的核心思想是将Spring应用程序的配置简化，使开发人员能够更快地构建可扩展的Spring应用程序。SpringBoot提供了许多内置的功能，例如数据源、缓存、会话管理、定时任务等，这些功能可以让开发人员更快地构建应用程序。

Netty是一个高性能的网络应用框架，它提供了许多用于构建网络应用程序的功能，例如TCP/UDP通信、网络流量分析、网络协议解析等。Netty可以用于构建高性能、可扩展的网络应用程序。

SpringBoot整合Netty的目的是为了将SpringBoot的功能与Netty的功能结合使用，以便开发人员可以更快地构建高性能、可扩展的网络应用程序。
