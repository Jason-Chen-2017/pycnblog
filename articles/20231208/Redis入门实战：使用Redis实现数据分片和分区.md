                 

# 1.背景介绍

Redis是一个开源的高性能key-value存储系统，它支持数据的持久化，可基于内存也可以将内存数据保存在磁盘中，并提供多种语言的API。Redis是一个使用ANSI C语言编写、遵循BSD协议的开源软件。Redis的根目录下的default.conf文件中包含了许多选项，可以用来配置服务器。

Redis支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。不同的是，Redis的持久化是在后台进程中进行的，并且不会影响服务器的通常运行。Redis提供了多种语言的API，包括：C、Ruby、Python、Java、PHP、Node.js、Perl、Go、C#、Objective-C等。

Redis的核心特性有：数据结构的简单性、原子性、可扩展性、高性能。Redis支持五种数据结构：字符串(string)、哈希(hash)、列表(list)、集合(sets)和有序集合(sorted sets)。
