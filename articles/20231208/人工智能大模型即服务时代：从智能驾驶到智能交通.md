### 1.4.3 路径规划代码实例

以下是一个简单的路径规划代码实例，用于计算最短路径：

```python
import heapq

# 定义图的邻接表
graph = {
    'A': ['B', 5],
    'B': ['A', 5],
    'C': ['B', 3, 'D', 6],
    'D': ['C', 6, 'E', 2],
    'E': ['D', 2]
}

# 定义起点和终点
start = 'A'
end = 'E'

# 创建开放列表
open_list = [(0, start)]

# 创建关闭列表
closed_list = []

# 创建最短路径字典
shortest_path = {}

# 遍历图
while open_list:
    # 获取当前节点和最短距离
    current_node, current_distance = heapq.heappop(open_list)

    # 如果当前节点是终点，则结束遍历
    if current_node == end:
        break

    # 遍历当前节点的邻居
    for neighbor, distance, _ in graph[current_node]:
        # 如果邻居在关闭列表中，则跳过
        if neighbor in closed_list:
            continue

        # 如果邻居在开放列表中，但距离更短，则更新距离
        if neighbor in open_list:
            if distance < shortest_path[neighbor]:
                shortest_path[neighbor] = distance
                continue

        # 如果邻居不在开放列表中，则加入开放列表
        shortest_path[neighbor] = distance
        heapq.heappush(open_list, (distance, neighbor))

    # 将当前节点加入关闭列表
    closed_list.append(current_node)

# 打印最短路径
print(shortest_path)
```

### 1.4.4 控制代码实例

以下是一个简单的控制代码实例，用于控制自动驾驶汽车的行驶：

```python
import time

# 定义自动驾驶汽车的状态
class CarState:
    def __init__(self, speed, steering_angle):
        self.speed = speed
        self.steering_angle = steering_angle

# 定义自动驾驶汽车的控制器
class CarController:
    def __init__(self, car_state):
        self.car_state = car_state

    def control(self, speed, steering_angle):
        self.car_state.speed = speed
        self.car_state.steering_angle = steering_angle

# 定义自动驾驶汽车的行驶
def drive(car_controller, speed, steering_angle):
    car_controller.control(speed, steering_angle)
    time.sleep(1)

# 创建自动驾驶汽车的状态
car_state = CarState(0, 0)

# 创建自动驾驶汽车的控制器
car_controller = CarController(car_state)

# 设置自动驾驶汽车的速度和转向角度
speed = 10
steering_angle = 45

# 开始驾驶
drive(car_controller, speed, steering_angle)
```
