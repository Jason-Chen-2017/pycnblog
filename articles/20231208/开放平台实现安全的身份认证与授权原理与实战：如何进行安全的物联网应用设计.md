### 1.4.1 身份认证代码实例

以下是一个身份认证代码实例：

```python
import hashlib

def authenticate(username, password):
    # 加密密码
    encrypted_password = hashlib.sha256(password.encode()).hexdigest()

    # 检查用户名和密码是否匹配
    if username == "admin" and encrypted_password == "123456":
        return True
    else:
        return False
```

在这个代码实例中，我们使用了`hashlib`库来加密密码。我们使用了`sha256`算法来加密密码，并将加密后的密码转换为十六进制字符串。然后，我们检查用户名和密码是否匹配。如果匹配，我们返回`True`，否则返回`False`。

import json
