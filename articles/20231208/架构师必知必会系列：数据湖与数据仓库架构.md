# 4.具体代码实例和详细解释说明

## 4.1 数据湖代码实例

### 4.1.1 Hive

```
CREATE TABLE people (name STRING, age INT) ROW FORMAT DELIMITED FIELDS TERMINATED BY ',';

LOAD DATA INPUT FROM 'input.txt' INTO TABLE people;

SELECT name, age FROM people WHERE age > 30;
```

### 4.1.2 Pig

```
people = LOAD 'input.txt' AS (name:chararray, age:int);

FILTER people BY age > 30;

DUMP people;
```

### 4.1.3 数据湖代码实例解释说明

Hive代码实例中，我们首先创建了一个名为people的表，它包含两个字段：name和age。我们使用ROW FORMAT DELIMITED FIELDS TERMINATED BY ','来指定字段的分隔符。我们使用LOAD DATA INPUT FROM 'input.txt' INTO TABLE people来加载数据到people表中。我们使用SELECT name, age FROM people WHERE age > 30来查询people表中年龄大于30的记录。

Pig代码实例中，我们首先使用LOAD命令加载input.txt文件，并将其映射到name和age字段。我们使用FILTER命令筛选年龄大于30的记录。我们使用DUMP命令输出筛选后的结果。

## 4.2 数据仓库代码实例

### 4.2.1 SQL

```
CREATE TABLE people (name VARCHAR(50), age INT);

SELECT name, age FROM people WHERE age > 30;
```

### 4.2.2 数据仓库代码实例解释说明
