具体操作步骤如下：

1. 确定问题：计算一个数组中的和。

2. 分解问题：将问题分解为两个层次：

   - 层次1：找出数组中的最大值。
   - 层次2：将数组中的其他元素与最大值相减，得到其他元素的和。

3. 设定目标：

   - 层次1：找出数组中的最大值。
   - 层次2：将数组中的其他元素与最大值相减，得到其他元素的和。

4. 制定任务：

   - 层次1：遍历数组，找出最大值。
   - 层次2：遍历数组，将每个元素与最大值相减，得到其他元素的和。

5. 执行任务：

   - 层次1：遍历数组，找出最大值。
   - 层次2：遍历数组，将每个元素与最大值相减，得到其他元素的和。

6. 评估结果：对实施的结果进行评估，以确定是否达到了预期的目标。如果没有达到目标，可以根据评估结果调整目标和任务，并重新执行。

以下是一个简单的Python代码实例，实现了上述操作步骤：

```python
def find_max(arr):
    max_value = arr[0]
    for i in range(1, len(arr)):
        if arr[i] > max_value:
            max_value = arr[i]
    return max_value

def calculate_sum(arr, max_value):
    sum_value = 0
    for i in range(len(arr)):
        sum_value += arr[i] - max_value
    return sum_value

arr = [1, 2, 3, 4, 5]
max_value = find_max(arr)
sum_value = calculate_sum(arr, max_value)
print("数组中的和为：", sum_value)
```
