首先，我们需要定义一个函数，该函数接收一个数组和一个目标值作为参数，并返回一个布尔值，表示是否存在两个数字之和等于目标值。

```python
def two_sum(nums, target):
    # 使用字典来存储数组中每个数字及其对应的下标
    num_dict = {}
    # 遍历数组，将每个数字及其下标存入字典中
    for i, num in enumerate(nums):
        # 如果当前数字的和小于目标值，则将当前数字及其下标存入字典中
        if num < target:
            num_dict[num] = i
        # 如果当前数字的和等于目标值，则返回当前数字及其下标
        elif num == target:
            return [i, num_dict[num]]
        # 如果当前数字的和大于目标值，则从字典中移除当前数字及其下标
        else:
            num_dict.pop(num, None)
    # 如果没有找到两个数字之和等于目标值的情况，返回空列表
    return []
```

在这个函数中，我们使用了一个字典来存储数组中每个数字及其对应的下标。然后，我们遍历数组，将每个数字及其下标存入字典中。如果当前数字的和小于目标值，我们将当前数字及其下标存入字典中。如果当前数字的和等于目标值，我们返回当前数字及其下标。如果当前数字的和大于目标值，我们从字典中移除当前数字及其下标。最后，如果没有找到两个数字之和等于目标值的情况，我们返回空列表。
