```python
def two_sum(nums, target):
    left, right = 0, len(nums) - 1

    while left < right:
        sum = nums[left] + nums[right]
        if sum == target:
            return [left, right]
        elif sum < target:
            left += 1
        else:
            right -= 1

    return []
```

- 如果这个和小于给定的目标值，我们移动左指针，使其向右移动。
- 如果这个和大于给定的目标值，我们移动右指针，使其向左移动。
