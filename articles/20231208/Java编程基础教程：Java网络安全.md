## 1.4 Java网络安全的具体代码实例和详细解释说明

### 1.4.1 AES加密解密示例

AES是一种对称加密算法，它的加密和解密过程相同。下面我们通过一个具体的代码实例来演示AES加密和解密的过程。

```java
import javax.crypto.Cipher;
import java.security.Key;
import javax.crypto.spec.SecretKeySpec;
import java.util.Base64;

public class AESExample {
    public static void main(String[] args) throws Exception {
        // 初始化AES加密算法
        Cipher cipher = Cipher.getInstance("AES");

        // 初始化密钥
        Key key = new SecretKeySpec("1234567890".getBytes(), "AES");

        // 加密数据
        String plainText = "Hello, World!";
        byte[] encryptedData = cipher.doFinal(plainText.getBytes());
        String encryptedText = Base64.getEncoder().encodeToString(encryptedData);
        System.out.println("加密后的数据：" + encryptedText);

        // 解密数据
        byte[] decryptedData = Base64.getDecoder().decode(encryptedText);
        String decryptedText = new String(cipher.doFinal(decryptedData));
        System.out.println("解密后的数据：" + decryptedText);
    }
}
```

在上面的代码中，我们首先初始化了AES加密算法，并初始化了密钥。然后我们使用加密算法对明文数据进行加密，生成密文数据。最后，我们使用解密算法对密文数据进行解密，恢复明文数据。
