# 2.核心概念与联系
在探讨 Python 的更多可能性之前，我们需要了解其核心概念。Python 是一种解释型编程语言，其核心概念包括：

- 对象：Python 中的所有事物都是对象，包括整数、字符串、列表等。每个对象都有其自己的类型和属性。
- 变量：变量是 Python 中用于存储数据的容器，可以用来存储不同类型的数据。
- 数据类型：Python 中的数据类型包括整数、浮点数、字符串、列表、字典等。每种数据类型都有其特定的属性和方法。
- 函数：函数是 Python 中的代码块，可以用来执行特定的任务。
- 类：类是 Python 中的模板，用于定义对象的属性和方法。
- 模块：模块是 Python 中的代码组件，可以用来组织和重用代码。

这些核心概念之间存在着密切的联系，它们共同构成了 Python 的编程基础。
