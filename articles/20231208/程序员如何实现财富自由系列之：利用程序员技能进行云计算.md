# 2.核心概念与联系
云计算是一种基于互联网的计算模式，它允许用户在不同的设备上共享资源和数据。云计算可以分为三个主要部分：基础设施即服务（IaaS）、平台即服务（PaaS）和软件即服务（SaaS）。

程序员可以利用云计算技术来提高工作效率，降低成本，并实现财富自由。通过学习和掌握云计算技术，程序员可以为企业提供更多的技术选择，并为个人提供更多的技术选择。
