# 2. 核心概念与联系
量子显微镜是一种基于光子的显微镜技术，它利用光子的量子特性来观察微小的物质结构。量子显微镜的核心概念包括：光子、量子态、穴道理论和超分辨率。

## 2.1 光子
光子是光的基本单位，它是光的量子化的基本单位。光子具有波动性和粒子性，这种双重性质使得光子可以同时具有波动的特性（如光波）和粒子的特性（如光子）。

## 2.2 量子态
量子态是量子系统的一种状态，它描述了系统中粒子的位置、动量、能量等量子数的状态。量子态可以用纯量子态或混合量子态来描述，纯量子态表示系统中只有一个粒子的状态，混合量子态表示系统中有多个粒子的状态。

## 2.3 穴道理论
穴道理论是量子显微镜的核心理论，它描述了光子如何通过穴道来与物质相互作用。穴道理论认为，光子在穴道中产生特殊的振动模式，这些振动模式使得光子能够与物质中的原子和分子进行强烈的相互作用，从而实现超分辨率的观察。
