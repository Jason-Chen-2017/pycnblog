### 2.3 事件驱动

Go语言的图形界面编程主要采用事件驱动的方式，即根据用户的操作事件（如鼠标点击、键盘输入等）来更新界面和处理逻辑。以下是一个简单的事件驱动示例：

```go
package main

import (
	"fmt"
	"image"
	"image/color"
	"os"

	"github.com/fogleman/gg"
)

func main() {
	// 创建画布
	dc := gg.NewContext(800, 600)

	// 设置颜色
	dc.SetColor(color.RGBA{255, 0, 0, 255})

	// 绘制线条
	dc.DrawLine(100, 100, 700, 500)

	// 绘制矩形
	dc.DrawRectangle(200, 200, 400, 300)

	// 绘制圆形
	dc.DrawCircle(400, 400, 200)

	// 绘制文本
	dc.SetColor(color.RGBA{0, 0, 0, 255})
	dc.DrawStringAnchored("Hello, World!", 400, 50, 0.5, 0.5)

	// 保存图像
	if err != nil {
		fmt.Println("Error:", err)
		return
	}

	fmt.Println("Drawing completed.")

	// 事件处理
	dc.SetOnMouse(handleMouse)

	// 事件循环
	err = dc.Run()
	if err != nil {
		fmt.Println("Error:", err)
		return
	}

	fmt.Println("Event loop completed.")
}

func handleMouse(x, y int) bool {
	fmt.Printf("Mouse position: (%d, %d)\n", x, y)
	return true
}
```

在上述示例中，我们使用了`github.com/fogleman/gg`库来实现事件驱动功能。首先，我们创建了一个画布，然后设置颜色，接着绘制了线条、矩形、圆形和文本。接下来，我们设置了鼠标事件处理函数`handleMouse`，并启动事件循环。当用户在画布上移动鼠标时，`handleMouse`函数会被调用，并输出鼠标的当前位置。
