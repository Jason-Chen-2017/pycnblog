```python
import logging

# 定义一个简单的规则
def rule1(event):
    if event == 'order_placed':
        logging.info('Order placed')
        return True
    return False

# 定义一个事件监听器
def event_listener():
    while True:
        event = input('Please enter an event: ')
        if event == 'exit':
            break
        if rule1(event):
            print('Rule executed')

# 启动事件监听器
event_listener()
```

在这个代码实例中，我们定义了一个简单的规则引擎，它可以根据用户输入的事件来执行规则。当用户输入 'order_placed' 时，规则引擎将执行相应的动作，并记录日志。
