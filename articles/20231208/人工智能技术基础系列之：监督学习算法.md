$$
L(\theta) = \frac{1}{2m}\sum_{i=1}^{m}(h_\theta(x_i) - y_i)^2
$$

其中，$L(\theta)$ 是损失函数，$\theta$ 是模型的参数，$h_\theta(x_i)$ 是模型对输入数据 $x_i$ 的预测结果，$y_i$ 是对应的输出数据。

$$
\theta_{t+1} = \theta_t - \alpha \nabla_\theta L(\theta_t)
$$

其中，$\theta_{t+1}$ 是更新后的参数，$\theta_t$ 是当前参数，$\alpha$ 是学习率，$\nabla_\theta L(\theta_t)$ 是损失函数的梯度。

- 正则化：监督学习算法需要一个正则化方法来避免过拟合。正则化是一种常用的方法，它可以通过添加一个正则项来限制模型的复杂度。正则化的公式为：

$$

其中，$\lambda$ 是正则化参数，$\sum_{j=1}^{p}\theta_j^2$ 是模型参数的平方和。

### 1.4.6 优化模型

最后，我们需要根据评估结果来优化模型。我们可以使用梯度下降来优化模型：

```python
import numpy as np

# 定义梯度下降函数
def gradient_descent(X, y, learning_rate, num_iterations):
    m, n = X.shape
    theta = np.zeros(n)

    for _ in range(num_iterations):
        y_pred = X @ theta
        gradient = (X.T @ (y_pred - y)) / m
        theta = theta - learning_rate * gradient

    return theta

# 优化模型
learning_rate = 0.01
num_iterations = 1000
theta = gradient_descent(X, y, learning_rate, num_iterations)

# 更新模型
model.coef_ = theta
```
