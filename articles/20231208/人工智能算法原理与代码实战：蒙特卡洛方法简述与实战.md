例如，我们想要估计一个区间内的平均值。我们可以使用蒙特卡洛方法来实现这个功能。具体的代码实现如下：

```python
import random

def monte_carlo(a, b, n):
    # 生成n个随机数
    samples = [random.uniform(a, b) for _ in range(n)]
    # 计算样本的平均值
    mean = sum(samples) / n
    return mean

a = 0
b = 10
n = 10000
result = monte_carlo(a, b, n)
print(result)
```

在这个例子中，我们首先定义了一个随机变量X，表示一个区间内的数值。然后，我们生成了n个随机数，这些随机数表示X的取值。接下来，我们对这些随机数进行求和，来估计X的平均值。最后，我们输出了估计值。
