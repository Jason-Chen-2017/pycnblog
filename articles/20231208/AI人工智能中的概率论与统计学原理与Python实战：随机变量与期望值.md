# 2.核心概念与联系
随机变量是一个随机事件的数值表现形式，它可以用来描述一个事件的不确定性。随机变量可以是离散的或连续的，离散的随机变量可以用概率质量函数（PMF）来描述，连续的随机变量可以用概率密度函数（PDF）来描述。期望值是随机变量的一个重要性能指标，用于描述随机变量的平均值。期望值可以用数学期望公式来计算。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 概率质量函数（PMF）
离散随机变量可以用概率质量函数（PMF）来描述。PMF是一个函数，它的输入是随机变量的取值，输出是这个取值发生的概率。PMF的定义公式如下：

$$
P(X=x_i) = p_i
$$

其中，$P(X=x_i)$表示随机变量$X$取值$x_i$的概率，$p_i$表示这个取值发生的概率。

## 3.2 概率密度函数（PDF）
连续随机变量可以用概率密度函数（PDF）来描述。PDF是一个函数，它的输入是随机变量的取值，输出是这个取值发生的概率密度。PDF的定义公式如下：

$$
f(x) = \frac{dP(x)}{dx}
$$

其中，$f(x)$表示随机变量$X$在点$x$处的概率密度，$P(x)$表示随机变量$X$在区间$[a,b]$内的概率。

## 3.3 期望值
期望值是随机变量的一个重要性能指标，用于描述随机变量的平均值。期望值可以用数学期望公式来计算。数学期望公式如下：

$$
E[X] = \sum_{i=1}^{n} x_i \cdot P(X=x_i)
$$

对于连续随机变量，期望值可以用积分来计算。积分公式如下：

$$
E[X] = \int_{-\infty}^{\infty} x \cdot f(x) dx
$$

# 4.具体代码实例和详细解释说明
在Python中，可以使用numpy和scipy库来计算随机变量的期望值。以下是一个具体的代码实例：

```python
import numpy as np
from scipy.stats import norm

# 定义随机变量的概率质量函数
pmf = [0.4, 0.3, 0.3]
x = np.array([1, 2, 3])

# 计算随机变量的期望值
expectation = np.sum(x * pmf)
print("随机变量的期望值为：", expectation)

# 定义连续随机变量的概率密度函数
pdf = norm.pdf
x = np.linspace(-10, 10, 100)

# 计算连续随机变量的期望值
expectation = np.trapz(x * pdf(x), x)
print("连续随机变量的期望值为：", expectation)
```
