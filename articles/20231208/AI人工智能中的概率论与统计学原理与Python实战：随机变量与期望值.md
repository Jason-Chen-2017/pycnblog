# 2.核心概念与联系
随机变量是一个可以取任意值的变量，其取值是不确定的，但可以用概率分布来描述。期望值是随机变量的一个重要性质，用于描述随机变量的平均值。

随机变量可以分为两类：离散型随机变量和连续型随机变量。离散型随机变量的取值是离散的，而连续型随机变量的取值是连续的。

期望值是随机变量的一个重要性质，用于描述随机变量的平均值。期望值可以用公式E(X) = Σ(xi * P(xi))来计算，其中xi是随机变量的每个可能取值，P(xi)是xi的概率。

## 3.1 随机变量的概念
在Python中，我们可以使用numpy库来创建随机变量。以下是一个创建一个随机变量的示例：

```python
import numpy as np

# 创建一个随机变量
random_variable = np.random.rand(10)
print(random_variable)
```

在这个例子中，我们使用numpy的`rand`函数创建了一个随机变量。`rand`函数会生成一个10维的随机向量，每个元素都是0到1之间的随机数。

## 3.2 期望值的计算
在Python中，我们可以使用numpy库来计算随机变量的期望值。以下是一个计算期望值的示例：

```python
import numpy as np

# 创建一个随机变量
random_variable = np.random.rand(10)

# 计算期望值
expectation = np.mean(random_variable)
print(expectation)
```

在这个例子中，我们使用numpy的`mean`函数来计算随机变量的期望值。`mean`函数会计算随机变量的平均值。
