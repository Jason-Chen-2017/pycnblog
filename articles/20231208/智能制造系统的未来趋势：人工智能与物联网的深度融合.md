                 

# 1.背景介绍

智能制造系统是一种利用人工智能、大数据分析、物联网、云计算等技术，以提高制造过程的智能化程度，实现高效、环保、可持续发展的制造业务的新兴制造业模式。智能制造系统可以根据制造过程中的数据信息，实时调整制造参数，提高制造效率，降低成本，提高产品质量，降低环境影响，实现制造业务的智能化、可视化、可控制、可扩展等特点。

智能制造系统的发展，受到了人工智能、物联网等技术的支持。人工智能技术可以帮助智能制造系统进行数据分析、预测、决策等，从而实现制造过程的智能化。物联网技术可以帮助智能制造系统实现数据的集中存储、统一管理、实时传输等，从而实现制造过程的可视化。

智能制造系统的发展，也受到了制造业的需求和挑战的支持。制造业需要提高制造效率，降低成本，提高产品质量，降低环境影响，实现制造业务的智能化、可视化、可控制、可扩展等特点。因此，智能制造系统的发展，是制造业的发展需要和技术支持的结合。
