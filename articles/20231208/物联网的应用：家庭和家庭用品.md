# 4.具体代码实例和详细解释说明
在物联网家庭用品中，可以使用各种编程语言和框架来实现设备的控制和数据交换。以下是一个使用Python和MQTT协议的代码实例：

```python
import paho.mqtt.client as mqtt

# 设备列表
devices = [
    {
        'name': '智能灯泡',
        'topic': 'smart/light'
    },
    {
        'name': '智能音响',
        'topic': 'smart/speaker'
    }
]

# 连接MQTT服务器
def on_connect(client, userdata, flags, rc):
    print("Connected with result code " + str(rc))
    for device in devices:
        client.publish(device['topic'] + "/state", "online")

# 数据处理函数
def on_message(client, userdata, msg):
    print(msg.topic + " " + str(msg.payload))

# 创建MQTT客户端
client = mqtt.Client()
client.on_connect = on_connect
client.on_message = on_message

# 连接MQTT服务器
client.connect("mqtt.example.com", 1883, 60)

# 启动客户端
client.loop_forever()
```

这个代码实例使用了Paho-MQTT库来实现与MQTT服务器的连接和数据交换。在这个例子中，我们定义了一个设备列表，包括智能灯泡和智能音响。当客户端连接成功时，会发布设备的在线状态。当收到消息时，会调用`on_message`函数来处理消息。
