# 2.核心概念与联系
在参与互联网初创公司的股权计划之前，我们需要了解一些核心概念和联系。

## 2.1股权计划的类型
股权计划可以分为以下几类：
1. 员工股票计划（ESOP）：公司为员工分配股票，以激励员工努力工作，提高公司的价值。
2. 股票选择计划（SAR）：公司为员工提供股票，员工可以在一定的时间内选择是否购买股票。
3. 股票抵押计划（PPS）：员工将一定比例的股票抵押给公司，公司将为员工提供薪资补偿。
4. 股票购买计划（SIP）：公司为员工提供股票购买机会，员工可以在一定的时间内购买股票。

## 2.2股权计划的核心原理
股权计划的核心原理是将公司的股权分配给员工，以激励员工努力工作，提高公司的价值。这种激励机制可以有效地提高员工的工作效率和抵押力，从而提高公司的盈利能力。

## 2.3股权计划与公司发展的联系
股权计划与公司发展的联系非常紧密。通过股权计划，公司可以吸引和保留高素质的员工，提高公司的技术水平和管理能力。同时，股权计划也可以激励员工努力创新，提高公司的创新能力和竞争力。
