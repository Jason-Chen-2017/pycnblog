## 4.具体代码实例和详细解释说明

在本节中，我们将提供一个简单的Python程序，用于计算可穿戴设备的耐用性和可靠性。这个程序将使用以下公式：

耐用性 = 材料质量 * 设计质量 * 质量检测
可靠性 = 耐用性 * 设备性能

```python
import math

# 材料质量
material_quality = 90

# 设计质量
design_quality = 85

# 质量检测
quality_control = 95

# 设备性能
device_performance = 92

# 计算耐用性
durability = material_quality * design_quality * quality_control

# 计算可靠性
reliability = durability * device_performance

# 输出结果
print("耐用性：", durability)
print("可靠性：", reliability)
```

这个程序首先定义了材料质量、设计质量、质量检测和设备性能等参数。然后，它使用公式计算耐用性和可靠性。最后，它输出了结果。
