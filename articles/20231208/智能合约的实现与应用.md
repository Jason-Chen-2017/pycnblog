# 4.具体代码实例和详细解释说明
在这里，我们将通过一个简单的智能合约实例来详细解释其代码和执行过程。

```solidity
pragma solidity ^0.5.16;

contract SimpleStorage {
    uint storedData;

    function set(uint x) public {
        storedData = x;
    }

    function get() public view returns (uint) {
        return storedData;
    }
}
```

这个智能合约的名称是 SimpleStorage，它有一个 uint 类型的变量 storedData。

set 函数是一个公开的函数，它接受一个 uint 类型的参数 x，并将其赋值给 storedData。

get 函数是一个公开的函数，它是一个 view 函数，这意味着它不会修改合约的状态。它返回 storedData 的值。

要部署这个智能合约，我们需要编译它，然后使用一个以太坊客户端（如 Ganache）来部署合约。

```bash
solc --bin --abi --optimize SimpleStorage.sol
```

这将生成 SimpleStorage.bin 和 SimpleStorage.abi 文件。我们需要将这两个文件发送到以太坊网络，以便部署合约。

```javascript
const Web3 = require('web3');
const web3 = new Web3('http://localhost:8545');

const abi = require('./SimpleStorage.abi');
const bytecode = require('./SimpleStorage.bin');

const deploy = async () => {
    const accounts = await web3.eth.getAccounts();
    const gas = await web3.eth.estimateGas({ data: bytecode, from: accounts[0] });
    const result = await web3.eth.sendTransaction({ from: accounts[0], gas, data: bytecode });
    console.log(result);
};

deploy();
```

这个 JavaScript 代码使用 Web3 库来与以太坊网络进行交互。它首先获取当前账户的私钥，然后估计合约部署所需的气费。最后，它发送交易来部署合约。

要与合约进行交互，我们需要使用 Web3 库来调用合约的函数。

```javascript
const set = new web3.eth.Contract(abi, '0x...');

set.methods.set(123).send({ from: accounts[0] });
```

这个 JavaScript 代码创建了一个新的合约实例，然后调用 set 函数，将参数 123 发送给合约。

要获取合约的数据，我们需要使用 Web3 库来调用合约的 get 函数。

```javascript
const get = new web3.eth.Contract(abi, '0x...');

const result = await get.methods.get().call();
console.log(result);
```

这个 JavaScript 代码创建了一个新的合约实例，然后调用 get 函数，并使用 call 方法来获取合约的数据。
