例如，我们可以学习冒泡排序的具体代码实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [5, 2, 8, 1, 9]
print(bubble_sort(arr))
```

在这个例子中，我们定义了一个名为 `bubble_sort` 的函数，它接受一个数组 `arr` 作为参数。函数内部使用了两个嵌套的循环，第一个循环用于确定需要进行排序的次数，第二个循环用于比较相邻的元素并进行交换。最后，我们调用 `bubble_sort` 函数并输出排序后的数组。
