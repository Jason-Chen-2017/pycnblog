### 4.1 代码实例

以下是一个具体的N-Gram模型实现代码示例：

```python
import numpy as np
from collections import Counter

def ngram_model(data, n):
    # 获取序列数据
    sequence_data = data
    # 选择长度为N的子序列
    n_grams = [sequence_data[i:i+n] for i in range(len(sequence_data) - n + 1)]
    # 计算每个长度为N的子序列的出现次数
    count_dict = Counter(n_grams)
    # 计算所有长度为N的子序列的出现总次数
    total_count = sum(count_dict.values())
    # 计算每个长度为N的子序列的出现概率
    prob_dict = {k: v / total_count for k, v in count_dict.items()}
    return prob_dict
```
