## 4.1 智能合约代码实例

以下是一个简单的智能合约代码实例，使用Solidity编写：

```solidity
pragma solidity ^0.5.16;

contract SimpleContract {
    uint256 public total;

    function add(uint256 x, uint256 y) public pure returns (uint256) {
        return x + y;
    }
}
```

在这个代码实例中，我们创建了一个名为“SimpleContract”的智能合约，它有一个公共的uint256类型的变量“total”。我们还创建了一个名为“add”的函数，它接受两个uint256类型的参数，并返回它们的和。

## 4.2 数据验证代码实例

以下是一个简单的数据验证代码实例，使用Python编写：

```python
import re

def validate_email(email):
    pattern = r'^[\w\.-]+@[\w\.-]+\.\w+$'
    if re.match(pattern, email):
        return True
    else:
        return False
```

在这个代码实例中，我们创建了一个名为“validate_email”的函数，它接受一个字符串类型的参数“email”，并使用正则表达式来验证它是否是有效的电子邮件地址。如果电子邮件地址是有效的，函数返回True，否则返回False。
