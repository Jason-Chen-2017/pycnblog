# 4.具体代码实例和详细解释说明

在本节中，我们将通过详细的代码实例来帮助读者更好地理解多线程编程的实现和原理。

```java
public class ThreadDemo {
    public static void main(String[] args) {
        // 创建线程
        Thread t1 = new Thread(new Runnable() {
            @Override
            public void run() {
                System.out.println("线程1正在执行");
            }
        });

        // 启动线程
        t1.start();

        // 等待线程结束
        try {
            t1.join();
        } catch (InterruptedException e) {
            e.printStackTrace();
        }

        System.out.println("线程1已经结束");
    }
}
```

在上述代码中，我们创建了一个线程t1，并启动了该线程。然后，我们等待线程t1的结束。最后，我们输出了线程t1的执行结果。
