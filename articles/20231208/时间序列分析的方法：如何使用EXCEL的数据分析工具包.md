# 2.核心概念与联系

在进行时间序列分析之前，我们需要了解一些核心概念。

## 2.1 时间序列

时间序列是一种数据类型，其中观测值在时间上是有序的。时间序列数据可以是连续的（如每分钟的温度数据）或离散的（如每年的人口数据）。

## 2.2 时间序列分析的目标

时间序列分析的目标是找出时间序列数据的规律和趋势，以便进行预测和决策。

## 2.3 时间序列的特点

时间序列数据具有以下特点：

- 时间顺序性：时间序列数据的观测值在时间上是有序的。
- 自相关性：时间序列数据的当前值可能与过去一段时间内的值有关。
- 季节性：时间序列数据可能具有季节性变化，例如每年的气温数据。
