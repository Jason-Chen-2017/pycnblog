数学模型公式详细讲解：

$$

$$

$$

```python

def calculate_weight(quality, links, behavior):
    weight = quality * 0.5 + links * 0.3 + behavior * 0.2
    return weight

def decay_weight(weight, decay_factor):
    weight = weight * decay_factor
    return weight

# 示例数据

links = 100
behavior = 0.8

# 计算权重

```
