超导材料是一类具有零电阻性的材料，它们在特定温度下可以导电，但在更低的温度下会失去导电性。超导材料的研究和应用在过去几十年中取得了重大进展，尤其是在1986年，科学家Karl Alex Müller和Jerome Frederick Friedman在美国斯坦福大学发现了第一种高温超导材料，这一发现为超导材料的研究和应用打开了新的门户。

超导材料的发现和研究对于电子产品的性能提高具有重要意义。在传统的电子产品中，电阻是一个主要的限制因素，它会导致电子产品的功耗增加，并影响其性能。超导材料可以大大降低电阻，从而提高电子产品的性能。

```python
import numpy as np
import matplotlib.pyplot as plt

def blron_function(T, T_c, alpha):
    return alpha * T_c * (1 - T / T_c)

def peng_lang_model(T, T_c, beta):
    return beta * T_c * (1 - T / T_c)

K = peng_lang_model(T, T_c, beta)

plt.plot(T, sigma, label='电导性')
plt.plot(T, K, label='热导性')
plt.xlabel('温度')
plt.ylabel('导性')
plt.legend()
plt.show()
```
