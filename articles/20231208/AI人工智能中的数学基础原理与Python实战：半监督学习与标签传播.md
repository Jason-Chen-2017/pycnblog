2. 利用有标签数据集来训练模型。
3. 利用无标签数据集来进行模型验证和优化。
4. 利用模型对无标签数据集进行预测。
5. 利用预测结果来更新模型。

2. 将数据点之间的结构关系表示为邻接矩阵。
3. 利用邻接矩阵来传播标签。
4. 利用传播结果来更新标签。

$$
\begin{aligned}
&f(x) = \arg\min_{y \in Y} \mathcal{L}(y, x) \\

\end{aligned}
$$

```python

X = data.iloc[:, :-1].values

```

接下来，我们需要将数据集划分为有标签数据集和无标签数据集：

```python
labeled_data = X[y != -1]
unlabeled_data = X[y == -1]
```

然后，我们需要初始化标签传播模型：

```python
model = LabelSpreading(kernel='knn', alpha=0.5)
```

接下来，我们需要训练模型：

```python
model.fit(labeled_data, y[labeled_data != -1])
```

最后，我们需要进行预测：

```python
predicted_labels = model.predict(unlabeled_data)
```
