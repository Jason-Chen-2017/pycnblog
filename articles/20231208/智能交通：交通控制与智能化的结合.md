                 

# 1.背景介绍

智能交通是一种利用信息技术、通信技术、电子技术、电力技术、机械技术、物理技术等多种技术为交通系统提供智能化、自主化、可靠化、高效化、环保化、安全化、舒适化等特征的交通技术。智能交通是一种新兴的交通技术，其核心是通过对交通系统的全面、深入的信息化、智能化、网络化和集成化进行改造，实现交通系统的高效化、安全化、环保化、人性化和可持续发展。

智能交通的发展是为了解决交通拥堵、交通安全、交通环境污染等问题。智能交通的主要目标是提高交通运输效率，减少交通拥堵，提高交通安全性，减少交通事故，减少交通环境污染，提高交通服务质量，提高交通用户满意度。

智能交通的主要技术包括交通信息化、交通智能化、交通网络化、交通集成化等。交通信息化是指利用信息技术为交通系统提供信息服务，包括交通信息收集、交通信息处理、交通信息传播等。交通智能化是指利用智能技术为交通系统提供智能服务，包括交通智能控制、交通智能决策、交通智能应用等。交通网络化是指利用网络技术为交通系统提供网络服务，包括交通网络建设、交通网络运营、交通网络应用等。交通集成化是指将交通信息化、交通智能化、交通网络化等多种技术相结合，为交通系统提供全面、高效、智能、安全、可靠的服务。

智能交通的主要应用包括交通信息服务、交通安全服务、交通环境服务、交通用户服务等。交通信息服务是指利用信息技术为交通用户提供交通信息，包括交通路况信息、交通预报信息、交通导航信息等。交通安全服务是指利用智能技术为交通系统提供安全保障，包括交通安全监控、交通安全预警、交通安全控制等。交通环境服务是指利用环保技术为交通系统提供环保保障，包括交通排放控制、交通节能保护、交通环保应用等。交通用户服务是指利用服务技术为交通用户提供便捷服务，包括交通用户认证、交通用户定位、交通用户应用等。

智能交通的主要优势包括高效、安全、环保、人性化、可持续发展等。高效是指通过利用信息技术、智能技术、网络技术等多种技术，提高交通系统的运输效率，减少交通拥堵。安全是指通过利用安全技术，提高交通系统的安全性，减少交通事故。环保是指通过利用环保技术，减少交通环境污染。人性化是指通过利用人机技术，提高交通系统的用户满意度。可持续发展是指通过利用可持续技术，实现交通系统的可持续发展。
