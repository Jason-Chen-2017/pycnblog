## 4.1 数据压缩代码实例

以 Huffman 编码 （哈夫曼编码） 为例，我们来看一个数据压缩的代码实例：

```python
from collections import Counter, namedtuple
from heapq import heappop, heappush

def huffman_encode(data):
    # 统计字符出现的次数
    char_freq = Counter(data)

    # 构建堆
    heap = []
    for char, freq in char_freq.items():
        heappush(heap, (freq, namedtuple('Node', 'char freq')(char, freq)))

    # 构建 Huffman 树
    while len(heap) > 1:
        lo = heappop(heap)
        hi = heappop(heap)
        for node in lo[1:]:
            node.left = node
            node.right = node
        for node in hi[1:]:
            node.left = node
            node.right = node
        heappush(heap, (lo[0] + hi[0], lo[1], hi[1], Node('', lo[0] + hi[0])))

    # 构建 Huffman 编码表
    huffman_table = {}
    node = heap[0][4]
    build_huffman_table(node, '', huffman_table)

    # 编码数据
    encoded_data = ''
    for char in data:
        encoded_data += huffman_table[char]

    return encoded_data

def build_huffman_table(node, prefix, table):
    if node.char:
        table[node.char] = prefix
    if node.left:
        build_huffman_table(node.left, prefix + '0', table)
    if node.right:
        build_huffman_table(node.right, prefix + '1', table)

# 测试数据
data = 'aaabbbccc'
encoded_data = huffman_encode(data)
print(encoded_data)
```

这个代码实现了 Huffman 编码的数据压缩功能。首先，我们统计了字符出现的次数，然后构建了一个堆，将字符和它们的频率放入堆中。接着，我们构建了 Huffman 树，并构建了 Huffman 编码表。最后，我们将数据进行编码，得到压缩后的数据。

## 4.2 数据加密代码实例

以 AES 加密 （Advanced Encryption Standard，高级加密标准） 为例，我们来看一个数据加密的代码实例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

def aes_encrypt(data, key):
    # 获取 AES 加密对象
    cipher = AES.new(key, AES.MODE_EAX)

    # 加密数据
    ciphertext, tag = cipher.encrypt_and_digest(data)

    # 获取加密后的非对称密钥和消息认证码
    nonce = cipher.nonce
    return nonce, ciphertext, tag

# 测试数据
data = b'Hello, World!'
key = get_random_bytes(16)
nonce, ciphertext, tag = aes_encrypt(data, key)
print(nonce, ciphertext, tag)
```

这个代码实现了 AES 加密的数据加密功能。首先，我们获取了 AES 加密对象，并使用了 AES.MODE_EAX 模式进行加密。然后，我们加密了数据，并获取了加密后的非对称密钥和消息认证码。最后，我们输出了加密后的结果。

## 4.3 数据分析代码实例

以线性回归 （Linear Regression） 为例，我们来看一个数据分析的代码实例：

```python
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

def linear_regression(X, y):
    # 创建线性回归模型
    model = LinearRegression()

    # 训练模型
    model.fit(X, y)

    # 预测结果
    y_pred = model.predict(X)

    # 计算均方误差
    mse = mean_squared_error(y, y_pred)

    return model, mse

# 测试数据
X = [[1], [2], [3], [4], [5]]
y = [1, 4, 9, 16, 25]

model, mse = linear_regression(X, y)
print(model.coef_, mse)
```

这个代码实现了线性回归的数据分析功能。首先，我们创建了线性回归模型，并训练了模型。然后，我们使用模型进行预测，并计算了均方误差。最后，我们输出了模型的系数和均方误差。
