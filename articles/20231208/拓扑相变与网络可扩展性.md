# 2.核心概念与联系
在讨论拓扑相变与网络可扩展性之前，我们需要了解一些核心概念。

## 2.1 网络拓扑
网络拓扑是指网络中各个节点之间的连接关系。常见的网络拓扑有树状拓扑、环状拓扑、完全连通拓扑等。

## 2.2 网络可扩展性
网络可扩展性是指网络在不影响性能的情况下，能够随着节点数量的增加而保持稳定和高效的性能。网络可扩展性是网络设计和优化的重要指标之一。

## 2.3 拓扑相变
拓扑相变是指网络拓扑在不改变网络结构的情况下，通过调整连接关系来改善网络性能或性能指标。拓扑相变是提高网络可扩展性的重要手段之一。
