# 2.核心概念与联系

## 2.1 太空探测器
太空探测器是人类探索太空的工具，它们可以通过各种传感器来收集太空中的各种信息，如光、磁场、热、微波等。太空探测器可以分为不同类型，如光学探测器、磁场探测器、热传感器等。

## 2.2 太空探测器的轨道
太空探测器的轨道是指探测器在太空中的运动轨迹。根据探测器的运动目的和需求，它们的轨道可以是稳定的、循环的或者是直线的。例如，月球探测器的轨道是稳定的，火星探测器的轨道是循环的，而太阳探测器的轨道是直线的。

## 2.3 太空探测器的控制系统
太空探测器的控制系统是指探测器在太空中运动和收集信息的控制系统。控制系统包括导航系统、传感器系统、电源系统等。控制系统的主要作用是控制探测器的运动和收集信息。
