
在上述代码中，我们创建了一个ArrayList对象，并添加了三个元素到列表中。最后，我们通过System.out.println()方法来输出列表的内容。

## 4.2 HashMap实例

HashMap是Java集合框架中的一个实现类，它实现了Map接口。以下是一个简单的HashMap实例：

