```python

```

然后，我们需要定义损失函数：

```python
def loss(y_pred, y):

```

接下来，我们需要定义梯度下降更新规则：

```python
def gradient_descent(X, y, theta, alpha, iterations):
    m = len(y)

        gradient = (1 / m) * np.dot(X.T, y_pred - y)
        theta = theta - alpha * gradient
    return theta
```
