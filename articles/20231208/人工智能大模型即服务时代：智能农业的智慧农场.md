# 2.核心概念与联系

在智慧农场中，AIaaS技术的核心概念包括：

1. 人工智能（AI）：人工智能是指通过计算机程序模拟人类智能的技术，包括机器学习、深度学习、自然语言处理等。
2. 大模型：大模型是指在人工智能领域中，由于其规模和复杂性，需要大量计算资源和数据来训练的模型。
3. 即服务（aaS）：即服务是一种软件交付模式，通过网络提供软件应用程序和数据，用户无需关心软件的具体实现和运行环境。
