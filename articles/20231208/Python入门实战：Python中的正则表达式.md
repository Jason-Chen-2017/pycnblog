# 2.核心概念与联系

## 2.1正则表达式的基本概念

正则表达式（Regular Expression）是一种用于描述、搜索和替换文本中的字符串模式的语言。它是一种强大的文本处理工具，可以用来匹配、替换和提取文本中的特定模式。正则表达式可以用来处理文本、HTML、XML、JSON等各种格式的数据。

## 2.2Python中的正则表达式模块

Python中的正则表达式模块是re模块，它提供了一系列函数和方法来处理正则表达式。re模块提供了以下主要功能：

- `re.compile()`：将正则表达式字符串编译成正则表达式对象
- `re.match()`：匹配字符串的开始部分
- `re.search()`：搜索字符串中指定的模式
- `re.findall()`：返回字符串中所有匹配的子串
- `re.sub()`：用一个字符串替换另一个字符串中的匹配子串

# 4.具体代码实例和详细解释说明

## 4.1正则表达式的基本用法

以下是一个简单的正则表达式的基本用法示例：

```python
import re

# 定义正则表达式字符串
pattern = r'^[a-zA-Z0-9]+$'

# 定义要匹配的字符串
string = 'abc123'

# 使用re.match()函数匹配字符串
match = re.match(pattern, string)

# 判断是否匹配成功
if match:
    print('字符串匹配成功')
else:
    print('字符串匹配失败')
```

在这个示例中，我们定义了一个正则表达式字符串`^[a-zA-Z0-9]+$`，它匹配所有由字母和数字组成的字符串。我们还定义了一个要匹配的字符串`'abc123'`。我们使用`re.match()`函数匹配字符串，并判断是否匹配成功。

## 4.2正则表达式的高级用法

以下是一个正则表达式的高级用法示例：

```python
import re

# 定义正则表达式字符串
pattern = r'<[^>]+>'

# 定义要匹配的字符串
string = '<p>这是一个<b>测试</b>字符串</p>'

# 使用re.findall()函数找到所有匹配的子串
matches = re.findall(pattern, string)

# 打印匹配的子串
for match in matches:
    print(match)
```

在这个示例中，我们定义了一个正则表达式字符串`<[^>]+>`，它匹配所有的HTML标签。我们还定义了一个要匹配的字符串`'<p>这是一个<b>测试</b>字符串</p>'`。我们使用`re.findall()`函数找到所有匹配的子串，并打印匹配的子串。
