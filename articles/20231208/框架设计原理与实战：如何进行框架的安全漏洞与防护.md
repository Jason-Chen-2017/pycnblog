```python
class Framework:
    def __init__(self):
        self.data = []

    def add_data(self, data):
        self.data.append(data)

    def remove_data(self, data):
        self.data.remove(data)
```

在这个代码实例中，我们定义了一个 `Framework` 类，它有一个 `data` 属性用于存储数据，还有两个方法 `add_data` 和 `remove_data` 用于添加和删除数据。
