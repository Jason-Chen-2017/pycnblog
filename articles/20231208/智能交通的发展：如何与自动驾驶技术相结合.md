                 

# 1.背景介绍

智能交通是一种利用信息与通信技术为交通运输系统提供智能化、可控制、可预测、可优化和可持续的交通运输服务的方法。自动驾驶技术是一种利用计算机视觉、机器学习、人工智能等技术为汽车驾驶行为自动化的方法。自动驾驶技术可以提高交通安全性、提高交通效率、减少交通拥堵、减少燃油消耗、减少排放污染物等。自动驾驶技术与智能交通的结合可以更好地满足人们的交通需求。
