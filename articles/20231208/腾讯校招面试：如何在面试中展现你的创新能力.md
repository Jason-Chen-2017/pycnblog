## 4. 具体代码实例和详细解释说明

在腾讯校招面试中，展现创新能力的关键是要能够编写出高质量的代码。这里我们给出一个具体的代码实例，并进行详细的解释说明：

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)

arr = [3, 5, 2, 1, 4]
print(quick_sort(arr))  # [1, 2, 3, 4, 5]
```

在这个代码实例中，我们实现了一个快速排序算法。快速排序算法的原理是：选择一个基准元素，将其他元素分为两部分，一部分小于基准元素，一部分大于基准元素，然后递归地对这两部分元素进行排序。
