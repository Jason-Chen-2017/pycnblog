# 2.核心概念与联系

超导体技术的核心概念包括：

1. 超导体：超导体是一种具有极高导电性的物质，它可以在低温下导电，具有极高的电导率。超导体的发现可以追溯到1911年，当时荷兰物理学家Heike Kamerlingh Onnes在实验中发现了氢化锂（LiH）在极低温度下的超导性。

2. 超导性：超导性是指某种物质在极低温度下能够导电的特性。超导体的超导性是由于其电子的量子效应，即电子能量的量子化，导致电子在某些特定温度下可以通过物质内部的氢体层，从而实现极高的导电性。

3. 低温：超导体的超导性只能在极低的温度下实现，通常需要达到几千分之一至几百分之一的绝对零度（kelvin）。因此，超导体技术的应用需要结合冷却系统，以保持超导体的低温状态。

4. 冷却系统：超导体技术的应用需要结合冷却系统，以保持超导体的低温状态。冷却系统可以采用各种方法，如冷却液浇注、冷却液循环、冷却液压缩等。

5. 超导体材料：超导体材料的发现和研究是超导体技术的核心内容之一。目前已知的超导体材料主要包括氢化锂（LiH）、氢化钠（NaH）、氢化铵（KH）、氢化锂酸铵（Li2S-KH）等。
