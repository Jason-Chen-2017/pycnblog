Redis 是一个非关系型数据库，它的数据存储结构是键值对（Key-Value Pair）。Redis 中的键是字符串，值可以是字符串、列表、哈希、集合和有序集合等数据结构。Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以恢复初始内存状态。
