# 4.具体代码实例和详细解释说明

## 4.1 数据清洗

### 4.1.1 数据纠正

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 纠正数据
data['age'] = data['age'].apply(lambda x: int(x.replace(',','')) if x != '' else None)

# 保存数据
data.to_csv('data_corrected.csv', index=False)
```

### 4.1.2 数据删除

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 删除数据
data = data.dropna(subset=['age'])

# 保存数据
data.to_csv('data_deleted.csv', index=False)
```

### 4.1.3 数据替换

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 替换数据
data['age'] = data['age'].replace({'': None})

# 保存数据
data.to_csv('data_replaced.csv', index=False)
```

## 4.2 数据质量评估

### 4.2.1 数据准确性评估

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 准确性评估
accuracy = data['age'].apply(lambda x: x == x).mean()

# 保存结果
print('准确性:', accuracy)
```

### 4.2.2 数据一致性评估

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 一致性评估
consistency = data['age'].nunique() == len(data)

# 保存结果
print('一致性:', consistency)
```

### 4.2.3 数据完整性评估

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 完整性评估
completeness = data['age'].isnull().sum() / len(data)

# 保存结果
print('完整性:', completeness)
```

### 4.2.4 数据可靠性评估

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 可靠性评估
reliability = data['age'].apply(lambda x: x == x).mean()

# 保存结果
print('可靠性:', reliability)
```
