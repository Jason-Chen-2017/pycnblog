## 2.1 事件

事件是概率论中最基本的概念之一，它是我们观察或关注的某个结果或结果集合。事件可以是确定性的，也可以是随机的。例如，掷骰子的结果是一个随机事件，而计算机程序的执行结果是一个确定性事件。

## 2.2 样本空间

样本空间是一个事件发生的所有可能结果的集合。在概率论中，样本空间是一个非空集合，它包含了所有可能的事件。例如，在掷骰子的例子中，样本空间是{1,2,3,4,5,6}。

## 2.3 事件

事件是一个子集的样本空间，它表示某个特定的结果或结果集合。例如，在掷骰子的例子中，事件可以是{2,3,4,5}，表示掷出偶数。

## 2.4 概率

概率是一个事件发生的可能性，它是一个数值，范围在0到1之间。概率可以用来表示事件的可能性，也可以用来表示事件的不可能性。概率的计算方法有多种，包括直接计数、统计方法、模型方法等。

概率的基本定义是：对于一个事件A，它的概率P(A)是一个数值，范围在0到1之间，表示事件A发生的可能性。

概率的计算方法有多种，包括直接计数、统计方法、模型方法等。在本文中，我们将主要介绍直接计数和统计方法。

### 3.2.2 统计方法

## 3.3 概率的基本定理

概率的基本定理是概率论中非常重要的一个定理，它可以帮助我们计算多个事件的概率。概率的基本定理的公式是：

$$
P(A \cup B) = P(A) + P(B) - P(A \cap B)
$$

其中，P(A \cup B)是事件A或事件B发生的概率，P(A)是事件A发生的概率，P(B)是事件B发生的概率，P(A \cap B)是事件A和事件B同时发生的概率。

## 4.1 直接计数

我们可以使用Python的random模块来生成随机数，然后通过直接计数来计算概率。以下是一个简单的例子：

```python
import random

# 生成1000个随机数
numbers = [random.randint(1, 6) for _ in range(1000)]

# 计算掷出偶数的概率
even_count = sum(1 for num in numbers if num % 2 == 0)
even_probability = even_count / len(numbers)

print("掷出偶数的概率:", even_probability)
```

在这个例子中，我们使用random.randint()函数生成了1000个随机数，然后通过直接计数来计算掷出偶数的概率。
