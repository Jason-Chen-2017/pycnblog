# 2.基本概念术语说明
为了更好地理解和讨论AI相关话题，首先需要对一些基础概念和术语进行了解。
## 概念和术语
**Artificial Intelligence（AI)** 是指由人工制造的计算机系统模拟智能行为的能力，可以实现智能学习、自我改进和解决问题的能力。它是指以人的某些特质为特征，在一定程度上模仿人类的智能行为的机器。相对于我们熟悉的人类来说，人工智能系统通常具有以下特征：拥有自动决策能力；具备自我学习和分析能力；能够解决复杂的任务。人工智能系统的范围非常广泛，从图像识别到语音识别，再到模式识别和决策支持，无所不包。

**Machine Learning （ML）** 是指利用训练数据让计算机“自己”学习，从而解决特定任务的能力。ML 是 AI 的一种子分支，它允许计算机从经验中学习、优化和改善它的行为，以达到某种预测精度或者解决某个问题。

**Deep Learning （DL）** 是指利用多层神经网络结构来训练复杂的数据模型，通过深度学习技术来学习数据特征，最终达到较高的准确性和性能。DL 是 ML 的一种子分支，它主要用于处理图像、文本、语音等高维数据的处理。

**Supervised Learning （SL）** 是指计算机从给定的训练数据中学到的知识，然后应用到新的输入数据上进行预测的过程。这种学习方式依赖于一个已知的正确答案，即监督学习。例如，给定一张图片和对应描述信息作为训练数据，那么计算机就可以学习到标签到描述之间的映射关系。当遇到新的图片时，计算机就根据已有的知识，推断出对应的描述信息。

**Unsupervised Learning （UL）** 是指计算机从数据中抽取没有任何标签信息的结构性特征。其目标是在没有明确结果的情况下，对数据集中的相似性、簇、分布等进行识别和聚类。典型的应用场景包括图像压缩、文档主题模型、异常检测和聚类分析。

**Reinforcement Learning （RL）** 是指计算机通过与环境互动的方式学习、优化策略，以最大化期望的奖励和惩罚的过程。RL 技术可应用于很多领域，如自动驾驶、游戏、物流管理、强化学习、优化问题等。

**Neural Network（NN）** 是指由多个节点（或称为神经元）组成的简单或复杂的计算系统。它由输入层、隐藏层和输出层构成，中间有若干个隐含层。每层中都含有一个或多个神经元，每个神经元接收上一层所有神经元的输入加权求和之后传递给下一层，最后输出一个预测值或概率分布。

**Backpropagation（BP）** 是指误差反向传播算法。它是指利用误差信号在神经网络各层之间传递的方法，使得神经网络可以学习、优化参数，以得到更好的分类效果和预测精度。

**Training Set（TS）** 是用来训练模型的数据集合。

**Test Set（TS）** 是用来评估模型性能的数据集合。

**Hyperparameter（HP）** 是指模型训练过程中的超参数。比如，在神经网络中，用户可以设置隐藏层数量、神经元数量、学习速率、正则化系数等超参数，这些参数影响模型的性能和收敛速度。

**Fine-tuning（FT）** 是指微调模型的过程，即调整模型参数，使其在新任务上取得更好的性能。一般情况下，我们可以先使用较少的训练数据和参数，进行初步训练，再使用更多数据和参数对模型进行微调，以获得更好的性能。

**Dropout（DO）** 是指在模型训练过程中，随机关闭一些神经元，减少模型的过拟合现象。

**Batch Normalization（BN）** 是指对网络中间层的输出做归一化处理，使其输入的分布逐渐变换为标准正态分布。

**Regularization（Reg）** 是指对模型参数添加正则化项，防止模型过拟合。

**Overfitting（Ovft）** 是指模型在训练和测试数据上的性能相当，但是无法很好地泛化到新数据上。

**Underfitting（Ufpt）** 是指模型在训练和测试数据上的性能差异较大，无法很好地拟合训练数据。

## 深度学习算法流程总结
1. 数据预处理
    - 数据清洗、数据缺失值处理、特征工程、归一化
    - 分离训练集和验证集、划分数据集
2. 模型选择
    - 从机器学习模型库或参考文献中选取适合任务的模型
3. 模型训练
    - 设置超参数、定义损失函数、定义优化器
    - 定义训练步骤、批大小、迭代次数、早停法则
    - 执行训练和验证
4. 模型评估
    - 使用不同的指标对模型性能进行评估，如准确率、召回率、F1-score、AUC等
    - 对模型进行显著性检验，判断是否发生过拟合
    - 使用测试集评估模型的泛化能力
5. 模型部署和应用
    - 将训练好的模型部署到生产环境，应用于实际业务
    - 根据业务需求定期更新模型的参数、模型架构，保证模型的持续更新
