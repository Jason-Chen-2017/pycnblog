# 2.基本概念术语说明

Firstly, we need to clarify some basic terms used in AI related to real estate. 

## **Data**
In Machine Learning and Deep Learning, data refers to information obtained from sensors, logs, images, text files, databases etc. It plays an essential role in training machine learning algorithms and deep neural networks.

## **Supervised Learning**
In supervised learning, there is a pre-defined dataset with inputs and corresponding outputs. The algorithm learns how to map inputs to outputs by analyzing patterns in the data and adjusting its output accordingly. Examples include Regression analysis, Classification problems such as image recognition or spam filtering, Clustering problems like customer segmentation, and Sequence prediction models for time series forecasting.

## **Unsupervised Learning**
In unsupervised learning, the algorithm doesn't have any predefined targets or outputs, instead it tries to identify patterns within the input data without labeled examples. Unsupervised learning methods often use clustering, density estimation, and dimensionality reduction techniques. Some popular applications include Market Segmentation, Anomaly Detection, and Visualization & Exploration of Big Data.

## **Reinforcement Learning**
In Reinforcement Learning (RL), the agent interacts with the environment through actions taken at each step and receives feedback in the form of rewards or penalties. At each point, the agent must decide which action to take based on the state of the world. RL is typically applied in robotics, gaming, and other decision-making scenarios where reward signal is sparse or not always available.

Now let's talk about AI algorithms for real estate industry. We will first discuss three main areas of AI real estate: Financial Analytics, Valuation, and Interpretation. Then we will dive deeper into two specific subareas - Traditional Statistics and Multivariate Analysis.

### Financial Analytics
Financial analytics involves predicting future stock prices and valuations. There are many approaches used in financial analytics, ranging from technical indicators to neural networks. Here are some key steps involved in financial analytics for real estate companies: 

1. Collecting and Preparing Data: In order to analyze market trends, financial analysts need access to reliable historical data. They might collect data from public sources or from internal systems such as CRM software. Once they receive the data, they need to clean it up, organize it, and prepare it for analysis.

2. Feature Engineering: Extracting features from raw data helps the model learn important characteristics of the business. These features could be price levels, trends, volatility, economic indicators, or any other relevant factors that affect the company’s stock prices.

3. Model Selection and Training: Depending on the type of problem being addressed, appropriate machine learning algorithms can be chosen. For example, linear regression models can be used when the relationship between independent variables and dependent variable is linear. However, if the relationship is non-linear, more complex models such as support vector machines may perform better. Once the model is selected, it needs to be trained on the prepared data. During training, the model adjusts itself to minimize loss function, which measures the difference between predicted values and actual outcomes.

4. Model Evaluation and Validation: After training, the performance of the model can be evaluated against different metrics such as mean squared error, R-squared value, and accuracy score. If the model performs well, it can then be deployed in production system. However, before deployment, it is recommended to validate the model again on a separate testing dataset to ensure that it generalizes well to new data.
