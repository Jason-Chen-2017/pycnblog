# 1.背景介绍


摩尔定律在近几年突飞猛进地推动着计算机、手机等高性能设备的生产革命。其对于信息处理和通信领域的广泛应用也促使硅谷各大厂商不断探索新型芯片设计方法以满足需求的不同。

随着摩尔定律的迅速失效，由于基带传输速度越来越低，像移动互联网这样的低时延计算需要用到更快的芯片，同时传感器节点也开始逐渐增多，导致芯片尺寸变得更大。

紫外线是一种可以用于电子显示、照明、激光打印、医疗诊断及产品安全认证等方面的无线传播有效能的激光物质。根据紫外线对生物体的折射特性不同，存在两种不同的光刻技术：一是深紫外光刻(Deep Infrared Thermal Soldering)，即通过深层次的紫外线灵敏度将金属粉末嵌入到晶圆结构中，成为真空孔板结构；另一种是极紫外光刻(Extreme Infrared Thermal Soldering)，即通过产生超高热量的紫外线纳米粒子(IRN)作为超导体的介质，将固态材料(如硅、GaAs、InP等)嵌入晶圆中，成为透明孔板结构。

近些年来，无论是在传统的工艺流程中采用什么样的光刻技术，芯片制造中的关键环节都离不开紫外光刻技术。目前，国内外很多著名的半导体公司如华为、微芯、台积电等都在布局紫外光刻技术的研发，尤其是深度紫外光刻技术(Deep-IR Thermal Soldering)。

为了更好地理解深紫外光刻与极紫外光刻的区别以及如何选择合适的光刻技术，本文将从以下几个方面进行阐述：

3.关键参数：介绍了实验室测试过程中需要注意的关键参数，包括热升华时间、晶圆厚度、浓度、湿度等。

4.设备类型：讨论了目前最主流的两种光刻设备——晶圆、宇宙学习显微镜(ULI)的差异性。

# 2.核心概念与联系
深紫外光刻技术(DITS)

深紫外光刻技术又称深层次紫外光刻或深紫外光治玻璃法，是利用紫外线纳米粒子作为介质，将金属粉末嵌入到晶圆底壁中，形成真空孔板结构，而非透明孔板结构。其主要过程如下：

1.将金属粉末（如硅、Ag、Ga）经过化学反应、形成高分子化合物（如InGaN、GaNiTe）,作为锂电池或镍电池的浮子。

2.将电池浮子经过紫外线高强度反射，形成高能级的紫外线纳米粒子，既可用于制备新型晶圆，又可被导体微粒和其他高能级元素所吸收，形成嵌入晶圆中得到真空孔板。

3.采用特定装置将晶圆吹制成稀松平滑，具有相同厚度的硅片做掩模，同时还会固定住不同长度的锂电池的浮子。

4.采用稀释温度为2500℃～3000℃的光氧仪器，对晶圆上的硅片和浮子进行光刻刻蚀。

5.最后将晶圆涂上晶体物质（如SiO2）及填充层（如K2SO4或CaF2）。

极紫外光刻技术(EITS)

极紫外光刻技术又称“超级紫外光刻”，是利用超高能级的紫外线纳米粒子(IRN)作为介质，将固态材料(如硅、GaAs、InP等)嵌入晶圆中，形成透明孔板结构，其主要过程如下：

1.采用超高能级的紫外线纳米粒子(IRN)作为介质，将固态材料(如硅、GaAs、InP等)连同电极共聚，形成聚乙烯、聚氨酯等材料。

2.通过特定的加热方式使得聚乙烯/聚氨酯形成三维晶体，并在其中嵌入电极形成电场。

3.通过雕刻设备把晶体涂上不同颜色的电极，在电场作用下形成多种类型的材料，如B型晶体或C型晶体。

4.然后将材料层与电极层相互固定，实现真空孔板或透明孔板结构。

两者的区别与联系

深紫外光刻技术(DITS)和极紫外光刻技术(EITS)虽然都是基于紫外线的纳米孔技术，但它们在晶圆结构，热应力刻蚀技术以及制造工艺上存在很大的不同。

1.晶圆结构：深紫外光刻技术主要应用于制造高性能光刻晶体管，其晶圆底壁由真空孔板构成，尤其适用于制造高端芯片或内存条等存储设备。而极紫外光刻技术则应用于制造一切材料型的光刻晶体管，包括薄膜、保护层、器件底盘等，其晶圆底壁一般为透明孔板，配套透光电容器能够更容易地导通晶体管，降低成本。

2.热应力刻蚀技术：深紫外光刻技术在制备晶圆时，采用低分子化合物激发高能级的紫外线纳米粒子，能释放出巨大的热量，将金属粉末嵌入晶圆中。其热应力刻蚀效应能够制备出高强度的晶体，还可以提供一定的抗脆弱性，因此该技术通常用于制造新型存储设备、计算模块和高性能的集成电路。而极紫外光刻技术则采用超高能级的紫外线纳米粒子作为介质，将固态材料嵌入晶圆中，其热应力刻蚀效应比深紫外光刻技术要小，不会产生太大的热量损耗，因此该技术通常用于制造一些薄膜材料、保护层等。

3.制造工艺：深紫外光刻技术通常采用机械式的加热切削过程，较少涂层和修复，属于手工操作，费时费力，但是精度高，具有良好的耐久性。而极紫外光刻技术则采用电气加热机器对晶圆进行光刻刻蚀，精度高且快速，耐用性好，缺点是制备成本较高。

综上所述，深紫外光刻技术和极紫外光刻技术虽然在应用上有很多差别，但其核心技术的原理是一致的，可以共同服务于各种芯片制造领域。
