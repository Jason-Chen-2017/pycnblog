# 1.背景介绍


《现代物理前沿之：弦理论与超引力》由诺贝尔物理学奖获得者、美国地球物理研究所、耶鲁大学物理系教授马修·科德尔（<NAME>）撰写，该文试图回顾和阐明现代物理中最重要的两个发现——弦理论与超引力。弦理论认为，每一个空间曲线都可以看作由许多弧段组成，这些弧段称为弦。弦是一个对称的闭合曲线，在曲线的两端，有些弦被扭成不同的形状，形成波浪状；而在中间某处，则是一个圆弧。因此，弦的长度、方向及弯曲程度均可调节出多种不同的运动形式，而且弦的出现也使得许多现象变得更加复杂和不可预测。
另一方面，超引力理论认为，宇宙中的任何物体都具有“引力”。若观察两点之间的距离远远超过他们的质量大小时，则会出现一种特殊的能量冲击现象。这种现象叫做超引力，其强度正比于所受引力的倒数，即引力对体积的倒数。当引力远大于质量时，就会产生超引力。因此，超引力理论认为，存在着一种新的物理定律，它既不能用传统的微观理论予以解释，又不能用宏观量子力学来描述，只能从量子力学的角度来理解。超引力理论将宇宙物理问题分为两步走，第一步是确认引力行为背后的基本假设，第二步则是揭示这一假设的不完美性。
本文的主要读者是高等院校物理系的研究生、工程师或博士后，以及对物理学、数学和计算机有浓厚兴趣的人群。文章拟从以下几个方面讨论这两个发现：
（1）弦理论的起源
弦理论是指一切空间曲线都可以分解为弦的假设。如果弦理论正确的话，就不需要太多的现象形式了，因为弦就是最简单的运动形式，不存在其他的可观测到的运动形式。弦理论的首要贡献是对空间曲线上弦的数量、长度、方向及弯曲程度等方面进行严格的数学分析，并得到广泛而有效的结论。当然，这一切都是基于相对运动速度的假设。随着人们对空间直线运动的认识的进一步深化，弦理论也逐渐成为一个越来越重要的理论。
（2）弦理论的局限性
弦理论在物理上的应用非常广泛，但同时也存在很多局限性。首先，对于物理系统来说，由于局部性原理，即只有很小的区域内才能看到其完整的运动过程，所以弦理论无法用来研究整个空间的物理系统，只能局部地进行研究。此外，弦理论仅适用于空间曲线，对于直线、曲面、以及二维旋转曲面等其它几何结构，弦理论无法直接运用。其次，弦理论在研究弦过程中，仅考虑了单条弦的运动特性，忽略了多条弦或多个运动模式的共同影响。最后，弦理论对物理系统的描述仅局限于少量的参数，对于整个系统的全局视图缺乏清晰的描述。
（3）超引力理论的形成
超引力理论也是由马修·科德尔提出的。他说：“超引力理论的提出与弦理论不同。弦理论虽然被证实可行，但是它所隐含的一些假设在实际中往往不合理，例如空间几何结构的局部性假设，以及弦的运动特性没有考虑到整体系统的所有参与者。”超引力理论的基本假设是引力具有“形变”或者“空间旋转”的性质。这一假设可以用切向场或者旋转势场的概念来解释，这种势场与广义相对论的束缚相互作用，使得粒子从一点到另一点存在着无穷多的可能路径。然而，对于某些情况，超引力理论的力学表达式可以近似地表示为经典的微观力学。因此，超引力理论也可以用标准的力学方法来分析。但是，真正理解超引力理论需要深刻地理解并克服其理论与现实之间的鸿沟。
（4）超引力理论的应用
超引力理论可以应用于很多领域，如星际物理、金融市场和军事等。另外，超引力理论还可以借助计算机模拟和数值计算来对其进行验证。因此，由于其丰富的数学模型和物理现象，超引力理论已经成为物理学界的热门话题。
