# 2.核心概念与联系
## 文件（File）
计算机中的文件指的是储存在硬盘或内存中信息的单位。它可以是文本文档、图像、视频、音频、程序等。文件是对信息的一种组织形式。它包含了各种类型的数据、结构以及相关属性。通过文件，可以存储、管理和传输数据。文件操作是文件系统的一项重要组成部分。文件系统将信息存放在磁盘、光盘或其他介质上。Linux和Windows系统都内置了一套文件系统，使得用户可以方便地创建、删除、复制、移动和访问文件。
## 异常（Exception）
程序运行过程中出现的不正常状态或者特殊事件称为异常。常见的异常包括程序崩溃、运行错误、语法错误、输入输出错误、资源不足等。程序遇到异常后，会引起一系列严重的问题，如崩溃、数据丢失、程序终止等。为了保证程序的健壮性，程序应当具备良好的容错性，能够正确处理异常。Python使用try-except语句来处理异常。

## 创建文件
创建一个文件，可以使用open()函数。以下是用Python创建文件的示例：
```python
f = open('test.txt', 'w') # 使用'w'模式打开一个名为'test.txt'的文件，如果文件不存在则创建；如果文件已存在，则清空文件内容
```
其中'm'表示读写模式，'w'表示写入模式，意味着如果文件已存在，则清空文件内容。另外还有'r+'表示读写模式，表示从头开始读写文件。
## 读取文件
使用read()函数可以从文件中读取内容。以下是读取文件内容的示例：
```python
content = f.read()
print(content)
```
该代码读取文件的内容，并打印到控制台。
## 追加内容
使用write()函数可以向文件中追加内容。以下是向文件中追加内容的示例：
```python
f.write("hello\n") # 向文件中写入字符串"hello\n"
```
该代码向文件末尾追加内容。
## 修改文件
使用seek()和truncate()函数可以修改文件内容。以下是修改文件的示例：
```python
f.seek(0, 0) # 将文件指针指向文件开头
f.write("world!") # 替换文件内容为字符串"world!"
f.truncate() # 删除后面的内容，保留文件开头的字符串
```
该代码首先将文件指针指向开头位置，然后用字符串"world!"替换原来的字符串"hello\n",最后删除掉后面的内容，保留文件开头的字符串"world!".
## 关闭文件
关闭文件时必须调用close()函数。以下是关闭文件并释放资源的示例：
```python
f.close() # 关闭文件并释放资源
```
该代码关闭文件并释放资源。
## 操作文件和异常处理
总结一下，Python的文件操作可以分为以下几个步骤：
1. 用open()函数打开文件，并设置读写模式；
2. 使用read()函数读取文件内容，或者使用write()函数追加或修改文件内容；
3. 如果修改了文件内容，调用seek()和truncate()函数同步更新文件内容；
4. 当完成所有操作后，调用close()函数关闭文件并释放资源。

异常处理也是文件的重要组成部分。Python使用try-except语句来处理异常。如果try块中的代码发生异常，那么except块就会被执行。除此之外，还可以定义多个except块来捕获不同的异常。以下是一个例子：
```python
try:
    age = int(input("请输入您的年龄："))
    if age < 0 or age > 120:
        raise ValueError("年龄值无效！")
    print("您的年龄是%d岁。" % age)
except ValueError as e:
    print(e)
```
这个程序询问用户输入年龄，然后判断是否有效。如果有效，则显示年龄。否则，抛出ValueError异常并显示相应的信息。
