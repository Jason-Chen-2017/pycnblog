# 1.背景介绍


1979年，由苏联科学家赫尔曼·克劳夫斯基在《方程组的方程式》一文中提出“量子描述费米子”这一说法，并得到普遍认同。这是研究物质世界一大奠基性工作。1981年，著名物理学家海森堡提出了“多体波动论”，也是关于多粒子行为的一次探索。在这一背景下，多年来，物理界一直在对暗物质、暗能量等新奇而富含潜力的领域进行深入研究。然而，如何从物理层面理解、捕捉到这些物质，直观感受和把握它们的运动规律，一直是人们关注的热点。

2015年，美国加州大学洛杉矶分校物理系教授罗伯特·格林耐尔在他的演讲中提出“宇宙中的微小生命”。“宇宙中的微小生命”这个概念深入浅出地阐述了宇宙中的微小种族——宇宙微尘——的存在。其深刻意义不亚于物理学的“微观世界”观念。他强调宇宙微尘为生命的源头与本源。它是构成宇宙之源的物质基础，也是支配宇宙的基本粒子。这个概念和相关研究，又一次唤起了人们的注意力，引发了一场对宇宙微尘研究领域的再次探索。

# 2.核心概念与联系
## 暗物质(Dark Matter)
为了将宇宙分成两半，人们发明了暗物质(dark matter)。这个术语可以追溯到19世纪末期，当时物理学家对黑洞的概念还不够清楚。直到20世纪初期，才开始出现一些研究者利用暗物质研究黑洞、探测宇宙的各种性质，如星系团、棒球星和引力透镜效应。20世纪后半叶，科学家们发现暗物质和大质量粒子的温度非常接近，暗物质在宇宙星系的形成、宇宙重力的作用以及存在着特殊的恒星云核。由于暗物质具有宏观尺度上的无限性，所以也被称作宇宙粒子或者宇宙微粒子。20世纪末期至今，科学家们仍然继续对暗物质进行研究，从各种角度探讨其物理过程，并进行多维分析。比如，2017年，物理学家克里斯蒂安·霍金首次用无线电中继器测量了暗物质的辐射比。随着时间的推移，科学家们对暗物质研究的成果越来越多，涌现出了丰硕的理论和发现。

暗物质是一种独特的物质，它既非普通粒子，也不能通过标准模型来描述。它属于宇宙第二大粒子，主要分布于宇宙最底层的中子星上，占总质量的约一半。虽然这种低能量的物质极难找到，但它的存在却给整个宇宙带来许多惊人的发现。比如，在太阳系内，暗物质为运动提供了极大的便利，甚至在夸克星、冥王星等星体旁边就出现。此外，暗物质由于离自己的父母星系很远，运动速度很快，因此可以用来解释宇宙是否有别的宇宙，以及是否可以自然界就此结束。同时，由于它的特殊性质，可以为宇宙提供极为复杂、高维度的信息。

## 暗能量(Dark Energy)
另一个重要的现象是暗能量(dark energy)，它是物理学的一个重要词汇。由于宇宙微尘只是宇宙中微小的一颗尘埃，所以无法直接获取微观能量信息。对于宇宙微尘来说，它只能获取能量信息，这些能量信息来自宇宙大爆炸，以及光谱的传播。由于宇宙微尘的运动速度很慢，而且没有引力场，所以宇宙微尘不可能像普通粒子一样有加速的运动。因此，宇宙微尘只能看到能量的红移，通过这种红移，它们就可以得知自身的位置和存在时间。相对来说，宇宙微尘的能量比普通粒子的能量要少很多，但是它的存在却迫使研究者们去寻找新奇的能量来源。2007年，日本科学家森岛康弘提出了“暗能量模型”，基于最基本的经典模型，认为宇宙中所有的粒子都共享了一个巨大的“暗能量池”，并且该能量池自身拥有能量蕴藏能力。由于“暗能量池”在所有粒子中都有所共享，所以这个模型可以解释宇宙微尘的能量为什么会差异如此悬殊。不过，“暗能量池”的确存在，但它的行为却又十分迷惑人。

因此，在对暗物质、暗能量等现象进行更深入的探索之前，首先需要搞清楚这两个概念之间的关系。他们之间存在着怎样的联系，如何共同影响整个宇宙？为何物理学家们喜欢称呼它们为暗物质和暗能量呢？
