# 1.背景介绍


半导体（英语：semiconductor）是指一种能够在高温、高压下存储能量并迅速释放热量的金属表面制备材料，其构成多种化合物，包括电极嵌入的高电压钢等物质，以及电极附着在离子层上的金属电子晶体。由于其特性，它被广泛用于各种领域，如显示器件、集成电路板、汽车电机、摄像头、手机电池等。而制造这样的材料需要高精度的工程技巧和先进的制程技术。根据半导体的分类，主要分为三种，即晶格硅、三维硅和二维硅。按照结构大小的不同，晶格硅结构尺寸一般为1 nm到10 nm之间的小型材料；三维硅结构则相对较大，通常在几微米至几毫米之间，由一个完整的晶体单元组成；二维硅结构又称点扩胞硅，它的结构类似于晶格硅，但仅由少数几个扩散晶粒排列而成，便于制造出所需的电路板。 

随着技术的飞跃，越来越多的人开始关注半导体工艺和制造。许多企业开始从事制造半导体的研发，目前已经成为重要的发展方向。在此，我将以此作为我的研究课题，深入介绍半导体物理与器件的原理，探讨芯片制造所需的一些关键技术和理论知识，并展示一些具体实例。希望通过这样的方式，帮助读者更加深刻地理解半导体、了解芯片制造所涉及的相关原理，并且能提升个人的科研能力。 

# 2.核心概念与联系
## 概念
### 超导
超导是指电介质的质性与电性之间的一种平衡，超导体表现出比金刚石或其他基本粒子更强的能量传递能力。简单来说，就是热的能量可以通过电场转移到整个晶体中的所有位置上。因此，超导体的制造往往涉及到对金属的破坏、粉碎、转化、堆积、熔化等过程。所以，超导的设备可以实现物理层面的控制和信息处理。

### 共导带
对于一个导带，当两种或两种以上电流均存在时，如果它们串联起来的电流满足存在偏差电压关系，那么会形成共导带，即导带中每一处的导体都在产生电流。因此，共导带的存在意味着单个的电流无法到达导带的每一处。 

### 半导体
半导体是在经典力学中发现的一类物质，其两极结构本身不能对称。该物质在某些条件下会导电，有较大的导通率，且导电率可变。半导体制备的材料一般具有以下三个特征：1）绝缘性：反铜材料的部分也是半导体。2）超导性：有极少量的电子和量子跃迁。3）结构性能：正负极结构可调节电压。

### 电介质
电介质是指以介质极化（E-field）而非电导极化（H-field）作为载体，在平衡态下自旋纤维的场分布相互作用形成自旋场，在交流作用下形成一个对称的电子云。常见的电介质如低电压高电流材料如镍氢铝钛等，以及高电压低电流材料如锂电池等。

### 导电元件
导电元件（英语：Diode）是指具有导电性的任何电子或电磁感应元件。一般来讲，导电元件均包括导体（Semiconductor）与电极（Electrolyte）。当电流经过导体时，导体可以将它导向相反的方向，使得电流不再继续流动。因为导体含有电子，因此也可以把它看作是一种电子。导电元件通常包括一个导体，一个空穴，以及一或多个结（接触点）。导电元件包括专用、通用、模拟和数字四种类型。专用导电元件主要用于生产电子产品，如集成电路；通用导电元件用于各项应用，如照明、照顾者监控、医疗、机械、电气等方面；模拟导电元件包括LED、照明灯、激光器等，数字导电元件则包括射频收发信机、调制解调器、路由器、数据库服务器等。 

### 膜电极
膜电极是指使用绝缘膜包裹的电极，其内部的金属原子形成一个颗粒状的带状结构，结构内导带的变化会影响整个晶体的导电性能。一般情况下，膜电极都是介质绝缘的，需要附加电容或有源电路连接才能使用。

### 方波干扰
在使用超导材料时，由于开关状态切换，会出现方波干扰。如使用光刻胶制造LED，在开关控制时，可能会受到远处电流的影响。这种干扰会导致LED在某一瞬间闪烁，造成损坏甚至损伤。解决方法也很简单，如使用稳态电流激励、直流电压或直流光照等。

### 玻璃体
玻璃体是指由无限长的粒子构成的单层材料，玻璃体表面呈现出亚线性，玻璃质最薄的原子包含一定的电子云团，这些云团与层内外的电场和电流密切相关。玻璃体的制备主要依赖实核材料，如晶格石墨、硬质粉末、棕榈油石墨等。

玻璃体的特点包括导电性、自发性、固化性。导电性是指在不同温度和压力下的导通性和导阻性。自发性指玻璃体表面电子云团的动态演化，对周围的环境影响微乎其微；固化性则体现了玻璃体微观层的对称性，使得高温的封闭系统表面容易烧焦。
