# 2.核心概念与联系
## 2.1 编程竞赛概述
编程竞赛（Programming Contest）又称为代码编程竞赛或者源代码竞赛（Source Code Competition），它是指由计算机科学、信息工程或其他相关专业的人员，用计算机语言进行程序设计、编写、调试、测试并提交作品，然后由评委选出优秀作品代表进入决赛的程序设计竞赛。20世纪80年代以前的程序设计竞赛，基本上都是以“集锦”为主，即竞赛过程中评委向参赛者进行题目分析、阅读参考书籍或网页，之后由参赛者自行组装程序进行设计、编写、调试、测试并提交作品。随着计算机硬件性能的提高、软件技术的迅速发展以及程序设计素养的增加，“正式”的编程竞赛逐渐取代了“集锦”作为程序设计的唯一方式。
## 2.2 编程比赛类型
### （1）团队参赛比赛：这是目前比较流行的一种比赛形式，由团队或者单个成员完成一个项目的开发任务。每个人都有任务分配，共同完成。通常情况下，根据题目难度和计划所需的时间，规定每人负责哪些模块，每个模块的完成时间要求，以及最后的测试情况，最后投票选出优胜者。
### （2）个人参赛比赛：这种比赛形式适用于对编程能力要求不是太苛刻、想直接获取奖金的个人。选手除了要完成某个指定功能模块的编码，还需要掌握相应的算法和数据结构、软件工程的基本知识、高效的编程方法论等。这些都是需要花费一定的学习时间和知识积累的。通常情况下，每周至少发布一次题目，并设有多个难度等级的任务，经过较长时间的竞赛，会获得非常优秀的参赛者。
## 2.3 编程比赛方式
### （1）线下比赛：编程比赛一般选择地点比较固定，比如软件园、图书馆、学校等等。常见的是软件园举办的编程比赛，比赛时长一般为一天到几天，比赛官员可以提供食物和水等补给。此外，比赛场地开放性好，方便队员相互交流、学习。缺点是场地限制，并且比赛质量参差不齐。另外，如果遇到某些突发情况，比赛可能会取消。
### （2）线上比赛：编程比赛也可以选择线上的平台，如www.acm.org。这种比赛具有很强的网络竞争性，参赛者可以通过网络分享资源、交流心得、结识志同道合的伙伴。在线上的比赛还有很多平台可以选择，比如codeforces、leetcode等等。

## 3.1 ACM International Collegiate Programming Contest（ICPC）
ACM国际大学生程序设计竞赛是美国著名的计算机教育机构——美国计算机协会（ACM）举办的一项全球性的计算机程序设计比赛。它的目的是推广计算机程序设计知识、训练学生的编程能力、发现学生的创新潜力、激发学生的编程热情、鼓舞青年的社会责任感。国际大学生程序设计竞赛由多个国家和地区的计算机科学和工程师组成的队伍，进行算法竞赛。根据比赛的规则，参赛者首先需要选择一门竞赛编程语言，之后编写程序解决特定问题。程序的运行速度、内存占用率、逻辑复杂度、可读性等性能指标作为评判标准，最终排名靠前的队伍能够获得奖金。例如国际大学生程序设计竞赛的奖项可以包括 ACM Programming Contest Winner Prize、ACM Programming Contest Distinction Prize 和 ACM Programming Contest Master Prize。
## 3.2 ICPC总体流程
1. 比赛前准备：学习比赛规则、学习编程语言，熟悉编程环境；
2. 模拟赛：在指定比赛平台注册账号，下载比赛题库，编写程序模拟出测试数据，提交程序结果，评审比赛结果；
3. 正式赛：同样是在指定比赛平台注册账号，下载比赛题库，编写程序完成题目要求，提交程序结果，评审比赛结果；
4. 领奖：确认正确程序，即可获得奖金。
## 3.3 ICPC编程语言
ACM国际大学生程序设计竞赛支持多种编程语言，包括C++、Java、Python、C#等。不同语言间可混用，但应保证程序效率高且符合比赛的规则。
## 3.4 ICPC的赛制
ACM国际大学生程序设计竞赛采用两轮赛制，第一轮为模拟赛，第二轮为正式赛。

模拟赛：模拟赛阶段主要目的是训练学生对程序设计问题的理解能力，重点是训练学生分析、综合能力、时间管理能力、团队合作能力、问题求解能力。模拟赛阶段题目数量平均为三个题，每个题目都会给出一份输入文件和一份输出文件，参赛队员可以在时间限制内利用这两个文件完成程序设计。模拟赛的结束后，参赛队员将提交程序的源代码，获胜者将获得相应奖牌。

正式赛：正式赛阶段为真正的比赛。题目数量根据比赛难度、题型复杂度、程序运行时间等因素而变化。参赛队员可以在两周时间内完成指定题目的编码工作。程序设计范围涵盖程序设计、数据库、图形图像处理、机器学习、网络安全等多个领域。在正式赛结束后，依据各题的难易程度以及程序的运行时间、内存占用率、逻辑复杂度、可读性等性能指标，计算出各队员的得分，最终产生排名。

## 3.5 参加ICPC比赛需注意事项
- 充分准备：务必在正式比赛前学习足够多的算法和数据结构，对编程语言有深入的理解。
- 解决实际问题：参加比赛时，尽可能解决实际问题而不是套题，而且要尽可能往工程方向发展。
- 积极参赛：在正式比赛开始前，积极参加每一场比赛，认真学习比赛规则、使用编程工具。
- 提高编程能力：与别人的讨论学习可以提高编程能力。

## 4.1 用Python统计数字出现频率
```python
import collections

numbers = [int(i) for i in input().split()] # 输入数字序列

count_dict = {}
for num in numbers:
    if count_dict.get(num):
        count_dict[num] += 1
    else:
        count_dict[num] = 1
        
print(collections.Counter(count_dict)) 
```
输入数字序列`numbers=[1, 2, 3, 4, 5, 5]`，得到字典`{1: 1, 2: 1, 3: 1, 4: 1, 5: 2}`，表示数字出现的次数。其中`collections.Counter()`函数可以把字典转换成列表，按照字典元素的值进行排序。输出结果为`Counter({1: 1, 2: 1, 3: 1, 4: 1, 5: 2})`，按照值从小到大的顺序显示数字和出现的次数。
