# 1.背景介绍


一般来说，参与创业公司的股权计划（简称C轮），主要就是投资人参加、跟进创业公司，购买其上市公司的股票。对于一个新创公司而言，很多创业者都是第一次接触这个行业，对公司的运作模式及管理制度等都比较不了解，因此很难确定自己的目标是否适合参与C轮。

我作为一名程序员出生在90年代，进入社会的时间也非常早，那时参加创业团队基本都是学生或研究生。所以我对创业者的认识更多的是从直观的角度入手，通过观察、体验等途径来了解创业公司的运作方式，并结合自身的经历、知识、技能进行分析和判断。这也是我对C轮股权计划感兴趣的原因。

我所在的团队创办了某外卖平台，业务主要是配送外卖。很多外卖平台都会提供一年或三年的C轮融资，虽然这些企业在面临困境的时候仍然会受到重创，但由于他们拥有比较强的技术能力和资源实力，往往能够快速摆脱困境。另外，大部分情况下，即使遇到创业失败的状况，也能通过股权回购的方式将损失部分赎回。

本文将围绕上述两个例子进行阐述。首先，我们将会介绍C轮股权计划的构成要素，了解不同角色的参与者，然后讨论每个角色的责任和义务。

# 2.核心概念与联系
## C轮股权计划概览
C轮股权计划全称Crowdfunding IPOs，是一种众筹模式，主要为了吸引大量投资者参与到创业公司或基金的IPO过程当中，通过其购买上市公司股票的方式获得一定比例的股份。它是在创业公司IPO之后，再次开展的一轮筹资活动，目的是更好地分配资金到相应的公司上，提升创业公司的资产价值，并最终造就一款产品。

C轮股权计划包括四个阶段：早期支持期（Pre-Seed Phase）、早期启动期（Early Stage Funding）、公募期（Seed/Angel Rounds）、后期运营期（Post-Funding Phase）。其中，早期支持期是指创始人的个人债务还清，并通过各种途径筹集资金，相信早期筹资能够帮助创业公司迅速起飞；早期启动期则是在早期筹资结束后，进行初步的筹资准备，目的是筹措基金，以保证初期投资成功；公募期是在早期启动期末，创业公司开始运作，筹集初始资金，并发放股票给公募基金；最后，后期运营期则是指创业公司不断运营，累计分配资金，确保创业公司盈利。

其中，早期筹资的关键是创始人的个人事业负担。如果创始人无法为生计负担，可能需要向其他人借钱或者从国内到海外投资。借款不仅会影响创业公司的运转，而且也会限制创业公司的发展前景。同时，不管是个人还是公司的贷款都需要额外付费，增大了筹资难度。因此，创始人要小心谨慎，避免出现长期的个人债务。

另一方面，早期筹资是为了获得足够的资金，能够启动公司的第一个轮次运作，随着时间的推移，还需进一步投入资金，完成公司的第二个轮次，并且在公募期结束后，公司的第三个轮次才算真正结束。可以看出，C轮股权计划需要创业者投入时间精力，耗费精力才能筹集到足够资金启动公司的第一轮。

## 领投人(Investor)
C轮股权计划的所有者、创始人以及之前的所有者都是领投人。也就是说，大家共同决定了一笔资金的用途，并承诺把这笔钱用于创业项目。领投人的职责有两个：

- 一是参与投资决策；包括调研、竞标、选定资金用途及规模、向管理层报告IPO结果、推动IPO顺利完成。
- 二是支付相关费用。C轮股权计划的本金一般都是由创始人的个人资产所形成的，这些资产又往往需要给予税费和管理费，因此，领投人需要按时缴纳这些费用。

## 管理层(Management Team)
管理层是指负责筹资、管理以及监督创业公司的人员。他们的主要任务包括：

- 一是筹集资金；包括收集资金、与创业公司签订合同、筹措储备金以及初始的股东持股。
- 二是管理公司；包括公司设立、募集资金、招股、安排董事会、协助董事会议案。
- 三是监督公司；包括审计公司的财务报表、监督董事会成员的工作，及时发现并解决问题。

## 激励机制(Incentive Mechanism)
激励机制是指通过一些奖励措施激励创业者投入更多的资金。比如，发放优惠券、奖金、股息等。这可以帮助创业者在刚开始的时候保持积极性，随着时间的推移，投入资金越多，激励机制也会越活跃。

## 投资者
投资者是指有意或者无意加入C轮股权计划的个人、机构以及其它创业者。他们与创业公司签订协议，承诺使用其资金来参与筹资活动。投资者既可以直接参与IPO，也可以将资金用于公司的改革、扩张等。

## 银行(Bank)
银行一般都是C轮股权计划的资金来源，资金分为创始人持有的投资资金以及留存的奖励等，均归银行保管。银行的作用主要有两个方面：

- 一是支付各种税费、管理费、风险资本化指标等。
- 二是保障投资者的资金安全，防止欠款被骗取。

# 3.核心算法原理与操作步骤
## 公募基金(Seed/Angel Rounds)
公募基金通常是在C轮股权计划的最后一个阶段，其中包括了一批初始的筹资者，他们的目标是建立一支强大的公募队伍，并帮助创业公司实现持续性的扩张。公募基金通常为公司创始人及核心员工提供战略支持、财政资金、行业资讯以及投资建议。

公募基金通常采用两种方式筹资：

- 一是线下发起的种子基金（Series A Funding）。通过与创业公司直接签署合同，发放一定的种子金额，预期公司能够获得一定比例的回报。
- 二是线上发起的天使轮基金（Angel Round Funding）。通过向竞争对手公司发起融资，获胜者将获得巨额的资金。

公募基金的筹资范围十分广泛，除了提供战略支持之外，还包括物流、保险、媒体等方面的支持。因此，公募基金需要投资者充分认识公募基金的运作模式和优势，根据自身条件选择合适的投资机构。

## 私募基金(Private Equity)
私募基金也是目前最热门的另一种类型的资本运作模式。私募基金也属于非公开募集资金，并不是所有的私募基金都是C轮股权计划的申请方。私募基金的特点是高效率、低成本、交易成本低、持续性收益高。目前，绝大多数的私募基金都存在对金融、经济、政治以及科技领域的偏爱，以致于投资机构的参与度不足，导致IPO的成交率较低。但是，私募基金可以在开始的时候取得比较好的效果，但随着时间的推移，最终却难以持续发展。

私募基金通过跟踪并分析大量的创业公司，挖掘其背后的资源和商业模式，从而寻求建立一个独特的私募基金品牌。私募基金的资金来源主要是股权，对于创业公司而言，需要上市公司的授权，同时还需要与投资者签订相应的协议。私募基金可以与公募基金混合运行，比如在公募基金筹集资金后，再提供一些私募基金的参与。

## 融资租赁(Financing Lease)
融资租赁是指投资人租借资金，并接受协议约定的期限，在期限内由指定的金融机构从公司筹集资金。融资租赁的优点是不需要提前知晓创业公司的资料、人员信息，且适合那些没有市场基础或资金规模过小的创业公司。缺点是有可能会产生融资风险、环境复杂、管理费高等问题。因此，对于大型公司而言，更推荐发行新的上市公司或联合创始人资本。

## 发行新股(Issue New Shares)
发行新股是指创业者为其项目创立的新上市公司所拥有的股份，其优势是不需要为公司推销融资资金，只需接受IPO的审批即可。但是，发行新股也有它的缺陷。主要的问题包括公司估值过高、融资困难、上市容易走弯路等。同时，发行新股只能容纳少数具有独特性的创业公司，对于大型公司而言，更推荐发行新的上市公司或联合创始人资本。
