# 1.背景介绍


决策分析(decision-making)是指在某个特定条件下所作出的一种行动或判断。它是人类活动中不可缺少的一环。比如，你做一件事情需要决定是否要买车、选宿舍、去哪里吃饭等等。在进行决策之前，我们往往会把各种因素考虑在内，然后找出最佳方案或最合适的选项。而作为一名技术人员，我们的职责就是要提炼数据、运用经验方法和理论知识来分析、理解和解决复杂的现实世界问题。因此，掌握如何进行有条理的决策分析就显得尤为重要。

决策分析通常由4个过程组成：
1.目标定位（goal-setting）——为了实现某种目的或期望，确定本次决策的对象或范围。目标定位包括明确定义问题的意义、确定决策的范围以及界定预期收益或风险。
2.信息收集（information-gathering）——收集相关的信息并对其进行分类、整理和归纳。主要有两种方法：
    - 方法一：从众多的数据中分析出有利于决策的关键信息。
    - 方法二：主观性调查法——通过面对面或者其他非正式的方式了解相关人士的看法和观点。
3.决策过程（decision-process）——基于相关信息进行分析、比较、综合以及决策的最终产出。有时候可以将这一过程分为多个阶段，比如：
    - 概念验证阶段：验证假设和价值观是否能够支撑决策。
    - 初始方案制定阶段：基于关键信息，制定可行的决策方案。
    - 对照研究阶段：与他人的决策结果进行对比，确认自己所做的决策是否符合历史经验。
    - 客观决策阶段：最后根据全部可供参考的信息进行实际的决策。
4.决策评价（decision-evaluation）——对决策的结果进行分析和总结，阐述原因和后果，帮助做出更加准确、高效、科学的决策。

决策分析的理论基础是经济学和行为经济学，两者的关系类似于数学与物理学的关系。经济学用于描述市场中的交易行为，如消费者的购买力、货币供应量、通胀率等；而行为经济学则通过分析人的决策过程以及决策背后的动机，来解释为什么人们会作出这样或那样的决策。因此，我们需要结合经济学和行为经济学，理解决策者的决策行为。

决策分析也是结构化思维的一种。结构化思考（structured thinking）是指通过组织、分析和管理大量信息，把复杂的问题分解成较小的模块，再通过连接各个模块来解决复杂问题。与一般的常识、直觉相比，结构化思考的优点是更全面、系统、科学地反映问题的复杂性、规模和内部逻辑。结构化思考可以帮助我们发现问题的根源、避免陷入细节泥淖，从而提升决策效率。

决策分析还可以视作一个金字塔结构。金字塔结构（pyramid structure）是指按照层级顺序排列的主题和问题。决策分析就是在决策过程中沿着金字塔结构不断前进、爬上去，逐步深入到问题的底层。金字塔结构由四个层次构成，分别为“目的”，“信息”，“过程”和“评价”。下面让我们一起回顾一下这些层次。

第一层——目的：决定当前应该完成什么工作，即“目标定位”。通常可以通过需求文档或目标清单来界定目标。

第二层——信息：收集有关目标、候选方案和相关信息。这里通常涉及调查问卷、访谈笔记、数据采集、编程语言或工具的文档等。

第三层——过程：基于信息形成的认知过程，即“决策过程”。流程通常包括概念验证、方案开发、对照研究、客观决策等。

第四层——评价：对决策的结果进行分析，并阐述原因和后果，即“决策评价”。这需要对结果进行综合评估、分析、排序、归纳和总结。
金字塔结构的这种特点使决策者的决策行为更具系统性、规范性和科学性。虽然每一步都可能需要耗费大量的时间和精力，但通过关注和分析各个层次的结果，我们就可以有效地减少浪费。同时，结构化思考也能帮助我们避免陷入泥潭，更快地找到更好的决策方向。
