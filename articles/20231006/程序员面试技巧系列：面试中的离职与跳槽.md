# 1.背景介绍


## 1.1 程序员到底是什么？
程序员（Programmer）这个词语从古至今一直被用来描述计算机、电子游戏等领域的一类人才。早在上个世纪70年代末期，就有一些游戏程序员就已经深入到计算机系统开发之中，后来这些编程工作者通过不断地锤炼智慧、创新能力，逐渐形成了一支队伍——程序员队伍作为新兴的计算机职业，逐渐凸显并占据了主流。直到最近几年，随着移动互联网的发展，“程序员”的定义也越来越宽泛，越来越多的人将其视作能够把握复杂问题、能够解决实际问题的“一流”人才。

## 1.2 为什么要离职呢？
程序员离职，不止会给自己造成经济上的损失，更重要的是面临着职业生涯的巨大挑战。举一个最简单的例子：你面试了三家公司，最后选择去一家公司工作，但是因为某种原因，你没有被录用，于是你决定去另一家公司工作。那么，第二次面试之前，你应该如何调整心态、应对工作压力？如果第二次面试不顺利，又该怎么办？第一次面试时你有没有想过，你真正想要的可能不是一份工作而是让自己得到更多的自由时间。第二次面试之后，你又应该如何安排自己的时间？对于这个问题，如果你没有答案，那么你很可能会走进另一个误区——把自己限制在工作时间内，而忽略了身边的时间与空间。当你的身体再次紧绷的时候，你还能坚持吗？除了经济问题外，程序员离职后还存在着很多职业生涯的挑战，包括提升薪酬、升迁、晋升、个人发展等方面。因此，了解面试中的职场离职策略，帮助你更好的调整心态，在第二次面试时正确处理工作压力，并利用好空余时间进行个人发展是一个非常关键的能力。

## 2.核心概念与联系
本文主要基于以下三个核心概念和概念之间联系进行剖析：
- 情商（intelligence quotient/IQ）：反映了一个人的逻辑思维能力、决策能力、解决问题能力、学习能力、创新能力及职业操守等综合素质。反映一个人的思维方式以及潜力。  
- 兴趣（interests）：代表了一个人对事物的态度及其喜好。  

情商与兴趣直接影响到一个人的出路。情商高的人往往拥有极强的分析能力、逻辑思维能力和解决问题能力；兴趣广泛的人往往具有丰富的创意能力和创新能力。情商和兴趣在程序员之间扮演着重要角色，在面试中往往被问道。所以，了解情商与兴趣的不同侧重点、对自己的定位以及个人成功的影响，是成为一名优秀的程序员不可或缺的一环。

- 跳槽（job switch）：指的是面试结束后，若发现自己的能力与优势发生变化，可以尝试换一家更适合自己的工作岗位，以便提升自己当前工作岗位的薪酬或个人价值。跳槽的方式有多种，可以是面试前几个月汇总收集资料后找老板谈，也可以找到理想的工作机会投递简历；也可以在面试通过后主动告知老板有意向但没有实施跳槽。无论采用何种方法，跳槽都需要用一些比较有诱惑力的事情来吸引老板的注意力。  
  在工作单位多年后，程序员往往都会选择跳槽，这是因为他们知道自己在所属企业里已经积累了丰富的经验，并且知道下一步在哪里发展。这样，一旦新工作的待遇与前一份工作相比出现变化，他们就会立刻意识到自己的薪资可能有所降低。因此，跳槽对于保障个人发展有着十分重要的作用。
  如果你想换一份新的工作，那么，你应该怎么做？除了回顾自己之前的工作经历，你还可以调查相关职位的要求，以便在新工作中培养自己所需的知识与能力。同时，你也应该注意到，新的工作环境是否满足你的个人需求。比如，你是否期望可以获得更多的加班费或晋升机会？你是否期望可以获得较好的带薪休假？你是否期望可以享受到公司提供的福利待遇？当然，你还有许多其他可以考虑的问题，这些问题都取决于你自身的情况。比如，你准备辞掉当前工作，但你还希望留在公司，这时候你可以选择一种特殊的辞职协议或计划。
