# 1.背景介绍


光是维持人类生存的最低限度就需要一种能量的储存能力，而对于极高的能量密度，尤其是核聚变产生的高能粒子所带来的热辐射效应、激发出质子弹等等，这种超级能量的存储是难题，并且需要极高的集成电路和算法才能解决这一问题。因此，超级计算机工程的发展极其迅猛，主要分成两种模式：

1.超导计算（Supercomputing）：利用真空管、传动结构、偏振器及对称场效应在超导材料表面建立微结构，如半导体。通过不同纳米结构建设，可以使得这些材料具备高度的复合度及非常强大的集成度，从而构成超大规模集成电路，能够处理海量的数据信息。

2.超级流体动力学（Supersonic Flows）：利用超声速流体，如超新星带、恒星云团等，产生的强大风速，以及高压态冷却水等多种物理环境的制约，超级流体动力学可作为目前还不能完全解决的问题之一，具有极高的应用价值。

超导与超流是物理学界两个重要研究方向，两者的融合才形成了目前世界上大型科研和工程的主要研究领域。同时，两者在长远的发展趋势也将会影响到自然科学、社会科学、经济学、医疗卫生、工程设计、教育科技等众多领域，有可能引起巨大的科技革命性的变化。因此，《现代物理前沿之：超导与超流》的主题无疑是“面向未来、开创新纪元”，并呼唤青年科学家、工程师共同努力，实现这两项关键领域的突破。

# 2.核心概念与联系
## 概述
### 超导
超导现象是指由多个粒子束成的一个加速扩散的过程，使得它们的电荷、场分布失衡，它们的能量不均匀，形成一个类似于波的状态。由于电荷的紧张，使得这些电子脱离原有的晶格结构，而形成一个凝胶状的环状带隙，称为超导带隙 (superconducting gap)。超导带隙存在于绝缘介质的晶格层中，其材料包括硅、锂、钽、铊、镉、铜等。

超导带隙有着广泛的应用。如超导纠缠线圈、超导电容、超导金属氢燃料电池、超导航天飞机的空间系统、超导磁体、核反应堆，以及热能转换装置、太阳能电池等都运用到了超导带隙的概念。

### 超流
超流是指电、磁、能等各种现象在空间和时间上的高度非线性运动，所形成的非均匀性，使得高速的电磁波或者高加速度粒子运动，这个现象称为超流现象。超流现象是由热力学第一定律和牛顿第三定律所决定的。超流现象的发生需要多种因素作用，如宏观场温、高压液态冷却、扰动、厚度、粒子的光电效应等。超流现象发生在极端条件下，如恒星云团的产生、太阳黑子的爆炸、重力波的干涉、高能量光子的冲击、强磁场的作用、微弱粒子的流失、地球本身的振荡、等离子体的浮游、超导电路的崩坏、核聚变反应堆的辐射等等。

## 关键词与分类
超导：电荷的紧张使得带隙变成了一个凝胶状环状结构；超导材料包括硅、锂、钽、铊、镉、铜等；
超导带隙是由多个电子束成的一个加速扩散的过程；
超流：宏观场温、高压液态冷却、扰动、厚度、粒子的光电效应等因素导致的高加速度粒子运动；
