# 1.背景介绍


人类活动所带来的影响是极其广泛、复杂的。包括经济、社会、健康、环境等方面都存在着巨大的环境效应。环境科学的研究领域从微观到宏观都在探索如何更好地利用资源、保护生态、改善和适应自然环境。环境科学是解释并预测自然环境的主要领域，也是解决各种环境问题的关键技术之一。 

但是，目前环境科学面临的最大挑战是——数据的缺乏、方法的匮乏、理论的不完善。数据缺乏，意味着环境科学工作者缺乏对自然现象的直接感受，只能靠经验、直觉和判断，无法对现实世界的真实情况做出客观有效的描述；方法的匮乏，意味着环境科学工作者缺乏经过多年的试错检验，没有开发出可以普遍使用的有效技术手段；理论的不完善，意味着环境科学的研究成果仍停留在理想状态，理论层面还没有形成完整体系。为了突破这一困境，环境科学界提出了可持续发展(Sustainable Development)这一概念。它将人类活动所带来的环境影响分为三个阶段：

1.渐进型效应阶段:从单一部门到多个部门、市场及所有部门共同作用导致环境问题的爆发。如导致气候变化的污染物浓度上升、水源枯竭、矿产资源短缺、森林砍伐、湖泊干涸、土壤侵蚀、工业废弃物排放等。
2.平衡型效应阶段:因个别部门或市场作用而引起的局部效应，长期影响可能并不明显，但很快就会蔓延。如煤炭排放、化石燃料、大气污染、固体废弃物处理、环保法规的滞后跟进、环境气候目标设置失误等。
3.积极转变型效应阶段:当各部门、市场相互依赖时，产生的积极影响，能够迅速扩散到整个生态系统。如城市居民对空气质量的不满、农业生产对土壤的破坏、节能减排项目的失败、化石能源的推广、工业废弃物的回收、大气污染的减少等。

环境科学家们围绕可持续发展进行研究，但目前却缺乏统一的理论体系。因此，我认为文章的核心内容应该包括以下几个部分：

1.环境科学家们如何定义可持续发展？
2.环境科学家们如何区分“渐进型效应”、“平衡型效应”和“积极转变型效应”？
3.环境科学家们的研究方向有哪些？
4.基于统计分析的环境效应评价标准有哪些？
5.什么样的项目可以提高环境可持续性？
6.如何激励环境科学家共同努力？
7.环境科学家需要遵循什么样的科研方式才能取得成果？
8.环境科学家该怎样参与公众关切？
9.其他相关的可持续发展概念有哪些？
