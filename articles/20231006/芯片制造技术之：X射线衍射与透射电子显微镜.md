# 1.背景介绍


近年来随着摩尔定律的失效、处理器性能提升及内存、存储等芯片领域的不断发展，X射线（Radiation）的应用越来越广泛，特别是在工业领域。X射线并非单一的一种光谱，而是多种光谱相互叠加形成不同的波长，如红外线、可见光、紫外线、激光等。在传统的显微镜操作下，只能看到红色光谱，无法观察紫外线、可见光等其他波长的物质反射。所以，人们又寻找新的显微镜来解决这一难题，特别是低功率、成本较低、功能强大的透射电子显微镜。
一般来说，透射电子显微镜都包括三大组件：控制器、收集单元、显示屏，它们之间通过机械、电气或光学交流的方式进行数据传输，收集和显示各类反射数据。控制器通常是一个嵌入式控制器，用于控制各种参数，如操作模式、工作温度、电压、速率等；收集单元则是收集并记录反射数据，包括二极管反射率、分散反射率、点扩散系数等；显示屏则是一个显示器件，负责将收集的数据显示出来，比如显示图像或曲线图。透射电子显微镜常用的工作模式有示性模式、轮廓模式、镊托模式、表面模式、焊接模式等。
在本文中，我们主要讨论的是透射电子显微镜的结构、原理及其应用。由于篇幅限制，本文并不对所有相关知识点都做过详细阐述。作者水平有限，欢迎读者批评指正！希望大家能通过本文了解透射电子显微镜的基本原理及其应用。
# 2.核心概念与联系

固定粗糙度的胶膜层中的非晶格杂质，称为玻璃体，其中的酸、碱和氧等物质会产生玻璃化，导致玻璃体的表面发生膨胀，从而降低抗反射能力。矽填充层就是为了减轻这种影响而添加的一层特殊材料。

## 2.2 自由摆动层（Free-grating Layer，FGL）
FGL在固定粗糙度的胶膜层之上，形成一个孔洞。自由摆动层（FG）由带状组织、自由粒子反射元件组成。其中，带状组织称为钢颈（Steel Barrel），通常由大量的铁棒构成，能提供稳定的立柱作用；自由粒子反射元件（FRP）的结构由多个自由晶体管组成，能够自由地聚集在钢颈中，并产生不同频率的反射，帮助细胞组织生成蛋白质。

FGL与FG之间的连接处于纤维层中，属于装饰性结构。FG内侧的自由晶体管会吸附在钢颈的铁棒之上，构成一个完整的器件。通过选择不同角度放置FGL及其间隙，可以实现多视角的观察。此外，FGL还有助于增强孔径探测器的分辨力、速度及精度。

## 2.3 可调焦层（Adjustable Focus Layer，AFL）
为了进一步提高显微镜的抗反射能力，除了引入FGL外，还可在其顶端增加一个可调焦层（AFL）。顾名思义，它就是用来调整镜头聚焦、防止挥离等的，可以通过改变晶体管之间的距离来实现。它的作用类似于微透镜的聚焦系统，但它的特性要比普通聚焦系统更好。

## 2.4 活塞（Actuator）
活塞的作用主要是通过手术般的方法将感光器件的位置调整至正确的位置。激光照相机采用了切向曝光（Convergent Laser Beam Exposure，CLB），这种方式在显微镜前部有几个小孔洞，使用大口径光束通过这些孔洞将目标物体（如微生物）引射到测量位置。但这种方法对微小型的孔洞和需要稳定的微生物有些困难，而活塞就是用来改善这种现象的。活塞是通过驱动专门设计的感光器件移动来实现镜头调整的。

## 2.5 信号管道（Signal Pipeline）
信号管道的作用是接收感光器件（如激光照相机）输出的信号，并将其转换为数字信号，再通过USB端口传输给计算机。目前使用的信号管道通常是双工模式，即同时接收感光器件的信号并发送给计算机。

## 2.6 数据采集器（Data Acquisition Unit）
数据采集器的作用是获取来自信号管道的数据，并将其转换为图像或者数据文件。目前常见的数据采集器有两种类型，即场扫描图像采集器（Scanning Microscopy Image Acquisition，SMIA）和分光采集器（Electron Multiplying Tubes and Secondary Photodiodes，EMT-SPD）。

## 2.7 操作系统
操作系统（Operation System）的作用是管理、控制计算机硬件设备，为应用程序提供接口和运行环境。常用的操作系统有Windows、Linux等。

## 2.8 运算器（Calculator）
运算器的作用是进行各种数值计算、逻辑判断等任务。它通常与信号管道、数据采集器、显示屏等设备配合使用。
