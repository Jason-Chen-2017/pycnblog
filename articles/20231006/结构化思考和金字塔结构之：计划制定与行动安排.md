# 1.背景介绍


## 1.1 什么是计划？为什么要制定计划？
“计划”这个词汇在现代社会中已经成为非常重要的工作方法、组织形式或活动方式。然而，对于新手而言，关于“计划”到底是什么、如何制定计划、制订的计划应该具有哪些要求等，往往存在很大的困惑。因此，为了帮助读者更好地理解“计划”，本章将从三个方面进行阐述：

1. “计划”的概念和定义；
2. 为什么要制定计划；
3. 制订计划应该具备的基本要求及注意事项。
### 1.1.1 计划的概念和定义
“计划”（Plan）是指对某件事情的一组规划、安排。它用来保证计划中的任务按时完成，并产生相应结果。在管理学和社会学领域，“计划”这一术语已经被广泛应用。作为工作或事务的一个环节，它可用于规划、编制任务、分配资源、控制进度、协调计划者、处理意外情况等，也常与目标、过程、标准、衡量工具一起使用。例如，计划书就是一种“计划”。

20世纪70年代，英国科技史学家托马斯·马库罗（Thomas Malthus）提出了“需求决定供应”的观点，认为物质生产力存在极大的不确定性。他认为，随着人们的消费愈加需要，需求会随着时间的推移越来越多，最终导致供应过剩。因此，马库罗建议将物质生产视作一场博弈，供应商和消费者都希望能够以一种较低的成本得到更多商品。为了实现这一目标，他建议制订一个“计划”，即根据消费者的实际需求、经济状况、技术水平和市场环境，合理地分配产品和服务供给，提高产品的价格或降低利润率，从而最大限度地满足消费者的需要。

从上面的描述可以看出，“计划”不仅仅是一个简单的工作方法，也是一门独立的学科。它既涉及到人类行为的科学研究，又融入社会科学、管理学等其他学科的知识。“计划”包括计划的目的、范围、结构、方法、成果、风险、挑战、承诺、监督、沟通等方面。

制定“计划”并非易事。首先，需要考虑该计划所涵盖的内容、时间和地点。其次，还需要确定计划的优先级、紧急程度和承诺。第三，还需确定各个部门、个人的职责、能力和待办事项。最后，还要确保各个参与者达成一致的协议，且不断跟踪反馈。因此，制订计划不能一蹴而就，需要持续不断的反复斟酌、修正。

1.2 为什么要制定计划？
制定计划的初衷有很多，比如完善公司制度、团队培训、重构当前的工作流程、加强经营竞争力、增加员工满意度等。计划的制定，是管理人员为了更好地执行计划所作出的努力。其中的原因主要有以下几种：

1. 预期效果：计划的制定，是在管理层设想计划之后，以“做得好”“做得差”为导向，以实际结果为依据，通过对比分析、权衡取舍、梳理步骤、比较方案、提前调整措施等各种方式制定计划。这样一来，就可以提前发现计划的不足或潜在风险，并采取正确的措施予以克服。

2. 激发创造力：计划的制定，主要是为了激发创造力，使管理者能够在解决实际问题时，能够快速、高效地寻找到最佳路径。

3. 沟通协调：计划的制定，不仅仅是团队成员的努力，更重要的是，管理层需要和相关人员密切配合，才能将大家共同的意志、期望和抱负真正落实到行动中。因此，计划的制定需要综合各方面因素的考虑。

总之，计划的制定，既是为了完成工作、改善运行、促进企业成功，也是管理者创造性、客观、透明的重要手段。

1.3 制订计划应该具备的基本要求及注意事项
制订计划时，需要对计划的目的、范围、条件、限制、执行步骤等进行具体的制定，使之能准确、清晰、完整。如下表所示：

|   |     |       |
|:-:|:-:|:-:|
|  名称    |  作用  |  是否必选   |
| 目标   | 设置符合用户需要的目标，以确定计划的对象及效益。      | 是    |
| 范围   | 描述计划的目的、涵盖的内容、活动以及项目等。          | 是    |
| 条件   | 描述计划的起始条件、约束条件、环境条件和日程安排等。        | 否     |
| 限制   | 描述计划可能遇到的困难、缺陷和限制等。                     | 否    |
| 执行步骤 | 描述计划的具体工作分解及活动顺序，明确每项任务的具体目标。         | 是    |
