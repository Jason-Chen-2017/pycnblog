# 1.背景介绍


## 1.1 信息时代的到来
信息时代的到来，是指信息技术的快速发展促进了经济、科技和社会的繁荣昌盛。

随着计算机网络、电子商务等新兴技术的出现，互联网成为信息时代最具革命性的特征之一，在这个时代，每个人都可以获取到海量的信息，并且快速分享信息，导致信息泛滥，极大地丰富了人的认知和思维。但同时也带来了诸多问题。例如，由于信息过载、搜索引擎的滥用、垃圾邮件、广告推送的泛滥，信息过剩、不确定性、不透明，甚至产生各种安全漏洞，使得社会产生危机，产生了各种社会问题。

为了解决这些问题，科学技术和法律法规应运而生，它们试图规范信息社会的运行，保障公民的权利和自由。如著名的“万言书”，美国联邦政府颁布的“国防授权法案”，以及政府部门之间的合作模式。但这些法律条文并不能完全解决信息泛滥的问题，而且也会产生新的问题。例如，在信息过剩和分层过滤之后，社会的个体可能会被迫面临信息孤岛，缺乏资源，甚至陷入精神疾病等问题。

## 1.2 中国的信息现状及问题
中国是一个具有丰富的传统文化和历史的国家，其信息化程度远远高于其他国家，它有较成熟的基础设施建设、完善的社会治理机制和制度保障，政治文明程度很高，公民素养也很高，因此，当下中国的一些问题又值得关注，如互联网治理难题、信息管理弊端、互联网的政治参与、网络营销等方面的问题。

### 1.2.1 互联网治理难题
互联网的蓬勃发展已经成为全球一个重要的政治话题。互联网的普及和应用极大地提升了人们生活质量，但是同时也带来了一系列新的社会问题，比如信息的垄断、网络空间的分割、身份的匮乏、信息消费的无形化等。这些问题虽然可以借助政府或社会力量进行一定程度上的改善，但是需要花费巨大的成本和金钱。随着这种情况的蔓延，出现了严重的舆论监管和打压互联网的倾向。

2016年，某省公安厅召开调查，对互联网平台上各类违反公共利益的信息进行严厉打击，其中包括色情、暴力、赌博等违禁信息。2017年9月，据财新报道，杭州市区内两家电商平台的员工利用社交网络上传播淫秽物品，购买社保个人贷款。

总体来说，互联网的治理还存在以下难点：

1、网络空间的分割。互联网是一个高度集中的平台，任何信息都可以通过互联网传播，这样就造成了一个结果就是网络空间的分割，相互隔绝的人群无法沟通，彼此之间只能依赖于信息媒介。解决这一问题主要靠的是建立健康的网络氛围，尤其是在学校和企业中建立好的规则和制度，让不同阶层的人群能够平等地接触互联网，创造良好的信息交流氛围。

2、信息的泛滥。由于互联网平台的普及，使得用户能够快速获取大量信息。信息泛滥的问题表现在两个方面。首先，互联网平台提供的信息种类过多，可能让用户感到无所适从。其次，用户也容易接收到过多的、不相关的信息。因此，对于这类信息的接收，平台应该设置一些过滤规则，对那些不合适的信息进行屏蔽，或者进行划分，归类处理。

3、网络消费的无形化。由于互联网上大量的消耗型内容，比如视频、音乐、图片等，导致很多人都习惯于在网上消费，而不是花时间思考，甚至长期沉迷于网络游戏，导致网络的异化，降低了人类的正常生活质量。因此，平台应通过技术手段限制网络用户的网络消费行为，减少娱乐和消遣的元素，保持正常的工作学习生活的状态。

4、身份匮乏。由于互联网的普及，越来越多的人通过网络获取信息，也越来越多的人在网络创造自己的声誉，形成自己的价值观。但是同时，也带来了互联网身份的匮乏问题。由于互联网的特性，很容易将个人形象塑造成一种商品的形态，打乱了个人价值观的塑造。因此，一些平台还需要进行管理，确保平台上用户的个人信息准确、完整、及时的共享。

### 1.2.2 信息管理弊端
互联网时代，信息收集日益广泛，特别是短视频、直播等新兴媒介带来的海量信息。随着信息量的增加，如何有效整合、加工、分析、归纳、存储和分享这些数据成为当前信息化领域一个难题。

信息管理是一个复杂且综合性的过程，涉及众多环节，如收集、组织、存储、加工、检索、分析、呈现、过滤、使用、留存、安全、隐私、保护等。其中，分析、呈现、过滤、使用、留存四个环节均对信息的质量、真实性产生巨大影响。如何设计有效的管理策略、工具和流程，保障信息的真实、准确、及时和完整，是对互联网信息化管理的关键。

当前信息管理方法论还有待进一步探索。传统的数据管理方法论往往认为，信息应以文档形式存在，通过文件夹、笔记本等设备分类、索引、检索。然而随着移动互联网、云计算、大数据等技术的发展，现代信息管理的新方法论呼唤出更多的协同协作、协商管理、知识共享等更优雅的方式。

### 1.2.3 互联网的政治参与
近年来，随着新冠肺炎疫情和世界经济的全球化进程，人们越来越依赖互联网进行资讯、信息、产品的获取，在这样的背景下，互联网的政治参与也逐渐成为热点议题。许多人担心，互联网的加入会削弱公民参与政治的积极性，但实际上，公民参与政治的热潮正在变成事实。

2020年3月，台湾总统陈水扁在接受德国之声采访时表示，他最担心的事件是什么？他说，“未来很可能是一场没有硝烟的政客掌握政权。”这是对现任总统的最好回应。陈水扁表示，当前互联网有可能成为促进政治参与的平台，而且政治参与也在加速发展。

另一位参与民主的人士陈若仪认为，公民参与政治不会像过去想象的那样缺乏阻力。在互联网、微信、微博、支付宝等平台上，公民可以通过自己的声音、思想、行动和财富来参与到政局的决策过程当中。如果人们愿意投身其中，那么最终将形成一个有创新能力的政治文明。

### 1.2.4 网络营销的困境
互联网营销的发展已经成为一个令人咋舌的领域。正如一位社会学者所说：“营销即广告的扩展。”营销活动旨在满足人们的需求，以满足品牌、客户、市场三要件。但是，营销活动一旦进入数字时代后，就面临着非常复杂、挑战性的挑战。

1、互联网产品营销的新局面。互联网产品的分裂、异化、频繁更新，势必使营销人员面临着持续跟踪最新产品的艰巨任务。这也是网络营销领域里的一个突出的难题，因为对于每个产品的全貌和定位并不十分清楚，营销人员需要不断反复、反复、再反复地反映产品的变化、口碑、效果、竞争力等。

2、大数据、云计算、人工智能、机器人技术的改变。这些技术正在改变营销的模式和方式，使得营销人员需要找到更简单、快速、准确的方法来实现营销目标。

3、网络营销的社会责任感。互联网营销已经成为一种社会服务的范畴，需要考虑到社会责任。如何通过互联网提升居民的幸福感、社会公平感、社会责任感、道德勇气和社会共识等，是网络营销发展的一大挑战。
