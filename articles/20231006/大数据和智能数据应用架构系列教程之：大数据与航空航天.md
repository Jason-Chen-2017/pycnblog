# 1.背景介绍


## 一、概述
### （1）航空航天行业背景
随着航空航天技术的飞速发展，航空航天行业也正在经历一场信息化、数字化、智能化、自动化的革命性变革。航空航天行业的蓬勃发展已经成为世界舞台上，人类社会向更高级的智能时代迈进的一大标志。航空航天领域众多公司和创新者纷纷涌现，产品、服务和研发能力全面提升。在互联网大爆炸之后，航空航天行业迅速崛起，已经成为国际航空航天市场的重要组成部分。中国是航空航天发达国家之一，占据了航空航天市场的70%-80%份额，占全球航空航天市场的80%以上。但是由于国内航空航天行业信息化水平不够，很难适应新时代需求和新的市场竞争，尤其是在高频、复杂的信息交互场景下，例如视频直播、超高分辨率图像处理、高动态功耗卫星轨道等。因此，如何利用大数据技术和计算机视觉等新兴技术来实现航空航天行业的信息化升级、提升业务效益、优化资源配置，是中国航空航天事业发展的一个突出课题。
### （2）智能数据应用背景
今天，大数据的概念无处不在。从互联网、社交媒体到健康管理、金融风控等行业，人们越来越关注大数据带来的巨大商业价值。如今，对于各行各业来说，掌握大数据意味着无限的商机，掌握大数据的优质资源和能力，实现海量数据的快速整合、分析和挖掘，并转化为明智的决策支持。而随着大数据技术的发展，智能数据应用模式逐渐成为主流，包括电子游戏、虚拟现实、机器人技术、语音识别、图像识别、传感器网络、流计算等。根据国际标准，如IEC/IEEE 485、IEC/IEEE 732、ISO/TS 29110等，智能数据应用的层次分为“软”层和“硬”层。而随着云计算、移动计算、物联网技术、传感网技术的发展，“软”层的智能数据应用更加依赖云计算平台和互联网连接。
因此，基于大数据技术和智能数据应用模式的航空航天行业应用系统架构需要考虑三个关键点：

1、数据采集与存储；

2、数据处理与分析；

3、数据可视化展示。
