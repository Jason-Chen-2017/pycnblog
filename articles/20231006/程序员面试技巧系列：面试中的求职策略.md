# 2.核心概念与联系

## 2.1 概念

### 2.1.1 算法
计算机算法（英语：Algorithm）是指用来解决特定计算问题的一组指令，是一种重复执行的指令集合，可由人类或计算机设计出来，算法代表着计算的机制。算法是计算机科学领域里的一个重要分支，研究其应用、定义和实现。通俗地说，算法是指计算机用来处理事情的方法。它是指一系列操作的规则、方法或者计算过程，使得输入的数据得到正确输出结果。算法也被称作数学逻辑、计算方法、计算过程等。

### 2.1.2 数据结构
数据结构（Data Structure）是计算机存储、组织数据的形式化方法。它定义了相互之间存在一种或多种联系的数据元素之间的关系、逻辑结构和操作等。数据结构就是一些存储数据的容器，用于存储、管理和访问数据的集合。一般来说，数据结构有如下几种类型：

1． 集合（Set）：是指一个无序且没有重复值的集合，其特点是任意两个元素的组合都可能不存在，即集合中不含重复值。例如，{1,2,3}就是一个集合。

2． 有序集合（Ordered Set）：是指一组按某种顺序排列的元素，同时还包含重复值。例如，{(1,2),(2,3)}就是一个有序集合。

3． 列表（List）：是一种线性数据结构，表示一个序列。它包含零个或多个元素，并且每个元素有一个唯一标识符。其中，元素按照固定次序排列。例如，[1,2,"hello"]就是一个列表。

4． 堆栈（Stack）：是一种容器，只允许在顶端进行插入和删除操作。堆栈中的元素按照先进后出的方式排序。例如，stack = [2, 5, 8]就是一个堆栈。

5． 队列（Queue）：是一种容器，只允许在两端进行插入和删除操作。队列中的元素按照先进先出的顺序排序。例如，queue = ["A", "B", "C"]就是一个队列。

6． 树（Tree）：是一种数据结构，它是由n（n>1）个节点组成，可以看做是一个连通图中的子图，它的每条边都有方向。树通常用来描述层次化结构的数据。例如，{1,2,{3,4},{5,6}}就是一个树。

7． 哈希表（Hash Table）：是一种以关键字-值(key-value)存储的数据结构。其中，键值都是独一无二的，可以直接定位。哈希表支持快速查找、删除操作。

8． 图（Graph）：是表示由节点和边连接起来的网络状结构。图也可用矩阵表示法来表示。例如，G=(V,E) G={V, E}, V is the set of vertices and E is the set of edges. 例如，G=(V,E), V={1,2,3}, E={{1,2}, {2,3}}.

9． 动态集（Dynamic Set）：是指动态变化的集合，如集合中的元素会随时间的推移而增删。一般情况下，动态集的元素不能重复。例如，A∪B∪C表示的是集合A、集合B、集合C的并集。
