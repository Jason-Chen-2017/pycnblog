# 1.背景介绍


在IT行业，项目经理、高层管理者往往需要对多个复杂任务进行优先级排序，以确保尽可能地提升产品质量或效益。而如何设计一个合理的优先级评估方法是一个难点。根据业务特点，可以分为：静态优先级（基于历史数据）、动态优先级（基于实时数据）、灵活度优先级（可调整）等三种类型。一般情况下，优先级排序可分为五个级别，分别是：紧急、重要、适当、不紧急、不重要。但在实际应用中，由于不同类型的产品和服务，其优先级排序方式各不相同，因此需要根据不同的情况选择最适合自己的评估方法。本文将以“电商”行业的核心业务——商品销售为例，通过比较多种评估方法及其优缺点，给出一种新的评估优先级的方法——结构化思维法。
# 2.核心概念与联系
## 2.1 结构化思维法
结构化思维法是一种基于结构分析和逻辑推理的评估优先级的方法。它能够有效地捕捉影响业务决策的重要因素，并按照权重确定每个因素的重要性。其基本思想是以客观事实为基础，依据事物之间的相互作用及其影响程度，分析其优先级。因此，结构化思维法有利于发现因果关系、揭示原因规律以及形成统一的业务策略。

结构化思维法的基本要素包括：
- 客观事实：描述事物间的互相关联、相互影响，以及影响目标决策的关键因素；
- 先导因素：决定事物发展方向、指导发展进程的主要因素；
- 后裔因素：成为先导因素的支配因素或者促进其发展的相关因素；
- 权重：反映事物的重要性的尺度，可以直接计入优先级列表，也可以从客观事实中得出计算权重的公式；
- 概念结构：描述业务决策领域的逻辑关系和组织结构，用来理解如何处理影响业务决策的各种因素；
- 行为模式：用来描述人们在面对某类事物时的行为特征，如习惯法则、社交圈子、生活习惯等。

结构化思维法的应用场景包括：
- 优先级评估：用于制定项目计划、企业管理、资源配置等事务中，对多个任务进行优先级排序。如在零售业，客户可能要购买各种商品，需要根据客户需求、市场竞争力、客户满意度等综合因素做出适当的优先级。
- 决策支持：通过理解业务影响因素，找出导致影响的关键因素，将其纳入评价体系中，协助业务决策人员分析原因，设计更加科学、合理的方案。如在电子商务领域，在有限的筛选条件下，消费者可能面临很多种因素的组合，如何将这些因素组织起来、归纳、分类，帮助商家决定哪些商品应该优先上架、商品价格应该涨，就显得尤为重要。
- 策略制定：采用结构化思维法，对所处领域内的业务决策有全面的把握，在所有因素的基础上形成完整的判断过程，最终输出业务策略和执行计划。如银行业，如何识别投资者的喜好和倾向，以及分析其中关键因素对实体经济发展的影响，建立起完整且有效的管理体系，才是最重要的。

## 2.2 为什么要有结构化思维？
首先，结构化思维法具有直观、简单易懂、反映出实际情况、重视效率、抓住主要矛盾等特点，是一种能够有效评估业务优先级的方法。其次，结构化思维法直接与业务决策相关，能够对决策提出全景图，从整体角度分析优先级问题，为执行者提供可靠信息，有效减少决策误差。再次，结构化思维法兼顾客观、主观两个方面，客观分析结论与实际情况，主观评价结果与个人感受，充分反映了不同专业人员的价值取向，同时也能避免“测评偏差”。最后，结构化思维法不需要过多的预设条件或假设，能够满足各种业务类型和决策要求。总而言之，结构化思维法是一项十分有效的评估业务优先级的方法。

## 2.3 怎么使用结构化思维法？
结构化思维法的使用步骤如下：
1. 列举影响业务的关键因素，并尝试分析它们之间是否存在因果关系。
2. 根据业务特点，制定权重评定标准，并收集数据证明确实存在影响因素。
3. 将分析出的影响因素划分为几个优先级级别，并综合考虑每一个优先级级别的权重，形成权重矩阵。
4. 根据权重矩阵，将影响因素排名，确定优先级顺序。
5. 将结果呈现给相关人员，确定优先级顺序并制订相应的执行策略。

当然，结构化思维法远不止以上几步，还包括将步骤4中的客观分析，建立属于自己团队的评价体系，提供进一步的建议。

## 2.4 有没有系统的标准和流程？
结构化思维法有一套完整的标准和流程，包括：业务领域的标准定义、评判标准和权重设计、数据采集和处理、结果报告制作和分析。除了以上内容，还有一些关于优先级评估的其他标准、方法、工具等，如：业界共同认可的评判准则、流程模板、指标体系、工具库等。结构化思维法的优势就是可以有效地提升工作效率、降低决策错误率，给企业带来更多的收益。
