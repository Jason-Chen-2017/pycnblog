# 1.简介
  

信息传输(Information Transfer)是指信息从源头到终点的传递过程。信息传输可以分为机械、电子、光学等各种媒体，但其基本要素都是电信号。信息的发送者可以通过各种传输媒体将信息编码转换成电信号并通过无线或有线的方式传送到接收方，最终接收方对信号进行解码还原出原始信息。信息传输的定义中，最重要的是要有两个端点：源头和终点。其中，源头又称为发送器、发送设备或发送站，而终点则是接收器、接收设备或接收站。信息传输的对象一般包括文本、图像、音频、视频等多种形式的信息。通常情况下，信息传输是需要某种协议来规范数据的交换，这样才能使得各个结点能够正常的交换数据，实现通信的双向性。

在实际应用中，信息传输方式可以分为几种：

1. 单播（Unicast）：即一条信息只能被一个节点接收；
2. 广播（Broadcast）：即信息会被多个节点同时接收；
3. 组播（Multicast）：消息只会被部分节点接收；
4. 轮询（Round-robin）：当有多个接收者时，按照循环方式发送消息给每个接收者一次；
5. 轨迹路由（Traceroute）：跟踪数据包到达目的地所经过的所有路由节点。

今天，我主要讨论以下两种信息传输的方式：

1. 短信（SMS）
2. 彩信（MMS）

这是两种不同类型的数据短信息的传输方式。通过短消息服务（Short Message Service, SMS）或彩信服务（Multimedia Messaging Services, MMS），用户可以发送少量文本或图片等简单格式的信息，也可以通过文件、网页、视频等复杂格式的信息进行高级发送。在信息传输的过程中，由于通信信道带宽资源的限制，不同类型的消息可能会受到不同程度的影响，短消息服务和彩信服务各有优劣。

首先，对于短消息服务，它是面向个人用户的一种通信方式。用户可通过手机短信、短信转发功能或手机拍摄分享功能，将信息快速、低成本的传播出去。它具有低延迟、保证可靠的特点。但是，它的容量有限，最长支持70个字符，发送时间为短则几秒钟，长则数小时。此外，它没有提供足够大的容错能力，如果网络出现故障、手机丢失等问题，短信便无法收发。因此，短消息服务适用于不要求高效率、低延迟、保证可靠、容错能力较弱的场景。另一方面，短消息的隐私性较强，容易泄露个人隐私。

第二种信息传输方式——彩信服务（MMS）。它是面向群体、组织、商业等类型的通信方式，可以有效解决短消息服务的容量有限、延迟、不足的缺陷。彩信服务可以发送更加丰富的形式信息，包括图片、视频、音频、文件等多种格式。它提供更好的隐私保护机制，也提供了更加高效的服务质量保证。但是，由于信息的大小、复杂度及网络环境等因素导致的延迟、丢包率更高。另外，由于设备及网络环境的限制，彩信服务还存在着兼容性、流量费用、存储空间等问题。
