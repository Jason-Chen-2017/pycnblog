# 2.核心概念术语
## 2.1.什么是云计算？
云计算（Cloud computing）是一种利用互联网动态、按需分配计算机计算、存储、网络资源的服务模型，它通常指通过互联网远程访问计算机系统资源的方式。云计算分为公有云和私有云两种形式。公有云又称为基于雲端的云计算，是一种将云计算服务提供给公众使用的服务，一般由第三方公司提供。私有云则是在内部部署的本地数据中心内运行的，由用户自行运营、管理。

## 2.2.什么是Azure？
Azure 是 Microsoft 在 2010 年推出的云计算平台，是世界上最大的公有云供应商之一。它的服务涵盖虚拟机（Virtual Machines，VMs）、容器（Containers）、存储（Storage），数据库（Databases）、分析（Analytics）、机器学习（Machine Learning）、IoT（Internet of Things，物联网）、Web 和移动开发以及 API 等。Azure 提供了一个完整的工具套件包括门户网站、命令行界面 (CLI)、应用程序编程接口 (API)，还包括多个 SDK 语言（例如 C++、Java、Python、Node.js、PHP、Ruby 和.NET）。

## 2.3.什么是订阅？
Azure 订阅是 Azure 服务的计费单位，每个 Azure 账户都有一个默认的订阅，可免费使用 Azure 的大部分服务，如计算、存储和数据库。如果需要使用更多的服务或资源，可以购买其他订阅。每个 Azure 订阅都有其唯一的 ID，可以用来跟踪 Azure 使用情况。

## 2.4.什么是资源组？
Azure 中的资源组是逻辑容器，用于组织和管理 Azure 解决方案的资源。资源组可以帮助您以组形式处理 Azure 服务，并有助于管理生命周期、权限和策略。资源组中的所有资源必须属于同一区域，这样才能享受到 Azure 提供的所有优势。

## 2.5.什么是Azure Active Directory？
Azure Active Directory (AAD) 是 Microsoft 为其云服务和 Office 365 等业务线产品而创建的一项功能强大的标识和访问管理服务，可帮助客户管理标识和访问，确保合规性和安全性。AAD 可让用户进行单点登录，并通过多重身份验证增强应用安全性，支持本地目录同步，并为应用提供 OAuth 和 OpenID Connect 标准协议支持。

## 2.6.什么是Azure Cosmos DB？
Azure Cosmos DB 是一种完全托管的 NoSQL 文档数据库服务，它可以快速且一致地在全球范围内缩放。它提供免费的吞吐量选项，以及固有的低延迟、高吞吐量和可预测的性能。无论你的工作负载有多么复杂，Cosmos DB 都能提供可靠且持久的服务，并满足各种规模和性能需求。与 SQL 或 NoSQL 数据库不同，Cosmos DB 不需要预定义的模式或索引，因此可以快速响应多变的查询。Cosmos DB 支持多种编程语言，包括.Net、Java、JavaScript、Python、Node.js、Go、Xamarin、C++、PHP 和 Ruby。

## 2.7.什么是Azure Blob Storage？
Azure Blob Storage 是 Azure 中提供的云原生对象存储服务，它提供一种低成本的解决方案来存储非结构化的数据。Blob 可以保存任何类型的文件，包括文本、文档、媒体文件、应用安装包等。Blob 存储提供三个不同的冗余级别，以便在发生数据中心故障时保持数据的持久性和可用性。 Blob 存储还提供了高度可用的对象存储服务，保证低延迟和高吞吐量。

## 2.8.什么是Azure Key Vault？
Azure Key Vault 是 Azure 中用于保护加密密钥和机密（如密码、连接字符串、证书等）的服务。可以将加密密钥和机密存储在 Key Vault 中，而不是将它们直接放在代码或配置中，从而可以对它们进行控制。此外，还可以对这些密钥和机密设置审核日志记录，以便可以监视对其的访问。Key Vault 有助于解决以下几个关键问题：

1. 机密管理：可以使用 Key Vault 来存储和控制加密密钥和机密，包括应用程序凭据、存储帐户密钥、数据加密密钥、数据库凭据等。
2. 密钥轮换：可以使用 Key Vault 来管理密钥和证书生命周期。启用 Key Vault 的软删除功能后，就可以安全地轮换存储在 Key Vault 中的密钥和证书。
3. 密钥共享：由于 Key Vault 具有最高级别的安全性，因此可以将密钥和证书授予多个应用程序和用户，使得它们可以共享密钥和证书。
4. 监控：可以使用 Key Vault 的日志记录功能来跟踪对密钥和机密的访问，并根据需要设置审核规则来检测异常活动。
