# 1.简介
  

高性能计算平台（High Performance Computing Platform）一般由以下三个部分组成：超算中心、高端服务器和存储设备。超算中心由多个节点组成，每个节点可以提供高计算能力。在这些节点中，有一部分节点安装了GPU，使得它们可以进行高性能的科学计算和图形处理任务。高端服务器包括具有巨大内存、CPU核数等多种性能特征的服务器。存储设备则用于存储运行数据、输入输出文件等。超算中心、高端服务器和存储设备之间通过网络互联互通，形成整体的高性能计算平台。本文将主要介绍基于超级计算机模式的华为云模型下，如何构建一个高性能计算平台。
超级计算机模式
基于超级计算机模式的高性能计算平台，通常由超算中心、高端服务器和存储设备三部分构成。超算中心（Supercomputer Center）由超级计算机集群和计算节点组成。计算节点有多达数百万个，每台计算节点拥有强大的计算能力，可以进行海量数据的并行计算。超算中心还配备有专用的加速卡，采用完全异构的方式对计算资源进行利用，有效提升整个计算平台的计算性能。
超算中心的计算节点分布在不同的地域，采取多种网络连接方式，最大限度满足用户所在区域的访问速度要求。当用户提交作业时，可通过网页界面或命令行界面向超算中心申请资源，然后根据不同类型的任务选择合适的计算节点进行执行。超算中心采用多级调度系统对任务进行管理，根据资源的使用率自动分配资源，保证整体的高效利用。同时，超算中心还引入了预约系统，对计算资源进行细粒度的预定，从而降低计算资源的实际利用率，减少资源浪费。
超算中心中的各台计算节点一般都配置了不同的硬件设施，如CPU、主存、硬盘等。CPU则用来进行计算，由于不同节点的CPU的性能千差万别，因此每台计算节点配置的CPU核数也不尽相同。此外，计算节点除了具备常规的计算能力外，还可以配置高速网络接口、磁盘阵列、通用处理器单元（Graphics Processing Unit，GPU）等专用计算设备。GPU是一种集成并行运算单元，它能够为计算节点带来的性能提升，尤其适合于图像处理、图形密集型计算、机器学习等高性能计算领域。
高端服务器作为超算中心的核心，也是主要的计算资源之一。根据国际通行标准，目前的高端服务器已经超过1000台。据统计，上世纪90年代末到2000年代初期，中国高端服务器数量占全球总数的五分之一左右，并且逐渐增长。随着信息技术的进步和应用需求的变化，越来越多的应用要依赖于高端计算资源。例如，银行业务、金融、保险、地质勘探、工程建设、核电、海洋、地震、气象等领域均需要大量高性能计算才能取得更好的效果。因此，高端服务器成为推动高性能计算平台发展的一个重要领域。
存储设备是高性能计算平台的关键环节，也是存放运行数据、输入输出文件的设备。一般来说，存储设备被分为计算存储（Compute Storage，CS）、固态存储（Solid State Drive，SSD）、机械存储（Magnetic Storage，MS）和光存储（Optical Storage，OS）。对于存储容量需求较高的任务，采用机械存储或者光存储设备既可满足性能又可减少成本。另外，云平台的出现也促进了高性能计算平台的存储架构的革命性变革，云平台将存储功能下沉至底层物理存储设备，实现了存储的按需分配、弹性扩展、自动故障转移等功能。
因此，高性能计算平台的组成非常复杂，是一个综合性的系统。本文将围绕超算中心和高端服务器两个部分展开讨论。
# 2.核心概念术语
## 2.1 概念理解
首先，需要对一些核心概念和术语进行简单的了解。
### 超算中心
超算中心，是指由超级计算机集群和计算节点组成的计算平台，提供海量数据并行计算的能力。它由多个节点组成，每个节点可以提供高计算能力。同时，还可以对计算资源进行细粒度的预定，降低计算资源的实际利用率。
### 计算节点
计算节点，是超算中心的基本计算资源。每个计算节点拥有强大的计算能力，可以进行海量数据的并行计算。它可以配置多个CPU核，使用户可以快速完成各种计算任务。
### CPU核
CPU核，是指一个完整的处理器芯片，里面含有多个集成电路。它负责处理指令，执行运算指令，实现程序的运行。同时，CPU核也可以缓存数据和指令，加快运算速度。
### 网卡
网卡，即网络接口控制器，是用来连接外部网络的部件。它负责接收网络数据包、发送网络数据包、检错、纠错、流控制等工作。每块网卡都会有一个MAC地址，它唯一标识网卡。
### 主存
主存，又称随机访问存储器（Random Access Memory，RAM），是一种临时的、易失的计算机存储设备。它的大小决定了其可以存储的数据容量。它可以随机读取和写入数据，且速度快、价格便宜。
### 磁盘阵列
磁盘阵列（Storage Area Network，SAN），是由一组通过网络连接的磁盘驱动器组成的存储设备。它可以提供高速随机访问I/O，并可扩展到数万台服务器。
### GPU
GPU，即图形处理器，是一种独立的计算设备，专门用来处理图形图像和视频。它由图形处理单元（Graphics Processing Unit，GPGPU）组成，拥有独特的并行计算特性。其性能优于CPU，但相对昂贵。
### 存储设备
存储设备，是把计算机的数据保存起来，供后续使用或其他程序访问的地方。它分为计算存储、固态存储、机械存储、光存储等类型。对于存储容量需求较高的任务，采用机械存储或者光存储设备既可满足性能又可减少成本。
