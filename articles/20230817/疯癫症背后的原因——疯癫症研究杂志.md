# 1.简介
  

疯癫症是一个迅速蔓延、难以治愈的神经病，它的名称起源于德文“Entschärfung”，即折磨。科学家们认为，它与人的认知能力及其关系密切相关。人类只有在面对任务时才会产生一种疯狂的心态，这种心态可能伴随着强烈的情绪表达，并诱发各种身体性和精神性活动，甚至导致死亡。疯癫症虽然不是一种新病，但在很长时间内都没有得到足够的医疗措施治疗。疯癫症早期以有创症或轻度症状为主，随着病程的加深，严重的疼痛，恶心呕吐，头晕耳鸣等症状，最终患者会出现自杀倾向。

20世纪90年代中期，英国的一项多中心临床研究发现：与正常人的认知功能差异巨大的少数群体能够相当成功地控制疯癫症症状。他们有的表现出了智力障碍，有的表现出双侧痴呆，有的表现出反应过激，有的则表现出无意识行为。他们的心理健康情况与其认知功能水平成正比。当这一结论被宣布，引发了一场轰轰烈烈的反疯癫主义运动。从此，疯癫症也越来越受到关注，相关专业人员纷纷涌现。在这种浪潮之下，疯癫症研究进入了一个全新的领域——脑外科疯癫症研究。而作为该领域的重要组成部分，就是专门为儿童和青少年制作教育视频的电视系列节目——《疯癫症研究杂志》。这是一个集儿童保护、精神科治疗、疾病预防、疾病控制、康复训练、欧美影视剧制作及其宣传工作为一体的综合性专业刊物。由于疯癫症在儿童群体中普遍存在，《疯癫症研究杂志》为儿童和青少年群体提供了免费的资料学习和治疗方法指导，帮助他们逃脱疯癫症带来的压力，让他们能够获得心理上的平静。

## 疯癫症产生
### 概述
疯癫症（Fear of Doing Foolish）通常包括多种形式，如自责、不安、恐惧、躁狂、疑惑、反应过度、担忧、情绪化、焦虑等。根据病因不同，可分为强迫性疯癫症、恐慌性疯癫症、感觉性疯癫症等。这些疯癫症都具有以下特点：

1. 症状表现多样：不同疯癫症类型，病因各异，症状也各不相同。例如，强迫性疯癫症患者常患自卑、自责、性格内向、强烈的自尊心理、强烈的自我价值观等症状；恐慌性疯癫症患者常患恐惧、不安、焦虑、恐怖、敏感、意外等症状；感觉性疯癫症患者则表现为强烈的情绪化、幻想、狂躁、想象、欲望等症状。

2. 症状易发：疯癫症患者往往经历高度复杂的精神和生理活动后才能发生，因此，疯癫症症状容易发生在各种情况下，并且症状持续的时间较长。

3. 影响范围广：疯癫症既可影响自然界的生物，如昆虫、飞禽、动物，也可直接影响人的心智和行为模式，影响日常生活、社会规范等方方面面。

### 类型分类
常见疯癫症的类型分为四大类：

1. **强迫性疯癫症（Impulsive Fear-of-Doing-Foolish）**：强迫性疯癫症是指自发性地行动、言行，且目标明确、变化不定，以牺牲个人的内心价值观、追求物质享乐为代价，造成的疯狂行为。

2. **恐慌性疯癫症（Fearfulness-of-Doing-Foolish）**：恐慌性疯癫症是指对某种行为恐惧、紧张，即使遇到不可控的情况，仍然坚持不懈的行动。

3. **感觉性疯癫症（Sense-of-Doing-Foolish）**：感觉性疯癫症是指由幻觉或催眠所导致的对人类精神和肉体的高度控制和支配，依靠催眠法或精神分析法来判断他人的感受、意图，并通过不断刺激来创造、强化自己的想象、欲望、情绪。

4. **混合型疯癫症（Mixed-Feelings-of-Doing-Foolish）**：混合型疯癫症是指既具有强迫性疯癫症的骄傲、喜好和奢侈的态度，又具有恐慌性疯癫症、感觉性疯癫症等现象的患者。

## 疯癫症预防
疯癫症是高度个性化、脆弱的性格，其病因通常难以被准确界定，原因包括心理结构缺陷、幻觉、神经系统刺激、认知缺陷等。目前已开发出的疯癫症预防方法主要包括以下几种：

1. **社会支持**：通过各种方式鼓励和培养学生树立积极的态度，减少不良的风险因素，改善身心健康，增强个人的能力。

2. **医学检查**：对患有疯癫症的患者进行实质性的心理检查，如病理检查、行为评估、精神病理检查等，可以诊断出患者的疯癫症发病机制。

3. **心理治疗**：除了通过医学检查外，还有基于心理学的疯癫症预防方法，如内省、冥想、自我暗示、反思、压力管理等。

4. **职业培训**：提高患者的自我保护意识，增强职业操守，以防止疯癫症的发展。

5. **教育改革**：对儿童教育和青少年教育进行重新设计，强调社会责任感和道德教育，促进孩子的品德修养。

## 疯癫症治疗
目前针对疯癫症的治疗手段还比较单一，主要包括以下几种：

1. **精神科治疗**：包括心理疗法、药物疗法和循证疗法，对疯癫症患者进行精神治疗。包括精神分析法、创造性记忆法、启发式治疗法等。

2. **专业精神治疗师**：有些疯癫症患者需要依赖精神科医生，因为精神科生产生的药物无法有效预防疯癫症。因此，很多疯癫症家族成员和爱好者会选择寻找专业精神治疗师开展治疗。

3. **毒品中药治疗**：在疯癫症患者中，有一些患者会用中药代替尼古丁、阿片类药物治疗，其中甘草、雄黄、枸杞、麦芽等药物能够有效缓解疯癫症的症状。

4. **化疗治疗**：用于预防性治疗，缓解疯癫症的症状。如生物合成溶解剂、腐蚀性食品等。

5. **肉体按摩**：适合于那些想缓解疯癫症症状的家庭，可以沟通疯癫症的本质，通过按摩的方式来减轻疯癫症的影响。

6. **自杀预防科普**：在某些疯癫症患者经历疯狂之后，会对自己的生命产生极大抵触，所以，有必要提供一些自杀预防知识。

7. **积极思维训练**：正是因为疯癫症患者经历了高度复杂的精神和身体活动后才能触发疯狂，所以，需要有教育的改变和积极思维的训练。
