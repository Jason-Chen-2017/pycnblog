# 1.简介
  

深度学习（Deep learning）是机器学习的一个分支领域，其发展历史可以追溯到上世纪90年代，由Hinton等人在上个世纪80年代提出，但是由于其复杂性、运算速度慢、易受到攻击、训练效率低等特点而没能成为主流。随着互联网的发展和深度学习技术的进步，近几年来深度学习在图像处理、自然语言处理、音频识别、自动驾驶等领域都取得了重大突破。本书基于作者多年的研究和实践经验，全面系统地介绍了深度学习的基础知识、核心算法、常用数据集及应用场景，并给读者提供了一个更好的学习工具。希望能够帮助读者更好地理解和掌握深度学习，提升自身的实力。

《Grokking Deep Learning》一书的编写格式参照了其他高质量的技术博客文章，先提供一个大纲，然后详细阐述每个章节的内容。不同于一般的技术书籍，本书力求将内容通俗易懂，提供一套完整的学习路径，引导读者一步步地学习、实践，最后达到“用起来”的阶段。
