# 2.基本概念术语说明
## 2.1 Python 简介
Python 是一种通用的、开放源码的、跨平台的计算机编程语言，由 Guido van Rossum 在1989年圣诞节期间，为了吸引程序员而设计开发。

Python 是一种多范式编程语言，支持面向对象的编程、命令式编程、函数式编程等多种风格，并且具有丰富的第三方库和框架支持，能够有效提升开发效率和项目质量。

Python 被认为是一个高级，结构简单的编程语言，具有简洁、直观的语法，能够极大地促进编码速度。Python 被广泛应用于科学计算、Web 开发、数据分析、游戏编程、脚本编写等领域。

## 2.2 Python 发展历史

1989年，Guido van Rossum 为了打造一种适用于程序员的简单易懂的语言而创建了 Python。

1991年，第一版 Python 0.9.0发布，标志着 Python 语言正式进入成熟阶段，开始接纳不同版本的 Python 社区贡献者的参与，并逐渐形成庞大的开发者社区。

2000年至今，Python 已成为最受欢迎的语言之一。

## 2.3 Python 特点

Python 有以下几个显著特征：

1. 易学习性：Python 拥有一套优雅的语法，简单易懂，学习起来相当容易。

2. 互联网语言：Python 是一门开放源代码的、跨平台的、可移植的、面向对象的脚本语言，运行于服务器端和客户端。

3. 可移植性：由于 Python 是一种开源的语言，因此可以在不同平台上运行，比如 Windows、Linux、Mac OS X等。

4. 丰富的第三方库：Python 拥有大量的第三方库和工具箱，可供程序员使用，可以大大减少开发难度。

5. 可扩展性：Python 支持动态语言，这意味着可以在运行时修改程序，并不需重新启动程序即可更新。

6. 免费开放源码协议：Python 使用 GPL(GNU General Public License) 授权协议，因此任何人都可以自由地使用和修改该语言。

## 2.4 Python 语法和缩进规则
Python 使用严格缩进规则，每一行代码块都必须缩进相同的数量空格或者tab符号。

标识符（identifiers）指的是变量名、函数名、类名等，它们必须遵循标识符命名规范，也就是只能包含字母数字下划线，且不能以数字开头。

除此之外，还有如下约定：

1. 每个.py文件应包含一个单独的模块（module）。

2. 模块名应全部小写，如果模块名包含多个单词，则每个单词的首字母均采用大写字母。

3. 对于内部模块名和函数名，应尽可能短小，但要见名知意。

4. 不要对 Python 保留字（关键字）和特殊函数名使用大小写混合的方式。

5. 类名通常采用驼峰命名法，即每个单词的首字母均采用大写字母。

6. 方法名和函数名采用小写字母加下划线的方式，即所有单词均采用小写字母，并将多个单词连接成单词，中间用下划线隔开。例如：func_name()。

7. 常量名一般都是全大写的形式。

## 2.5 Python 注释
Python 中有两种注释方式:

第一种注释方式为“双井号”注释。这种注释的语法格式为: #注释内容，双井号后面可以添加注释内容。

第二种注释方式为“单行注释”。这种注释的语法格式为: //注释内容，单行注释只能出现在一行的末尾位置。

两种注释方式可以用来提醒自己需要添加更多注释，但不要滥用。过多的注释会影响代码的可读性和可维护性。
