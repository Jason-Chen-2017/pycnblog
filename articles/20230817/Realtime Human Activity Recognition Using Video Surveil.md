# 1.简介
  

随着社会和经济的发展，对人类活动的监测、分析、预测、决策等已经成为当今世界需求的一部分。同时，随着视频监控摄像头技术的普及，越来越多的人使用这种新型的监控设备来记录、监控人类活动。如何用视频监控摄像头实时地识别人类活动并进行相应处理，是一个非常重要的问题。而最近十年来，随着深度学习技术的飞速发展，基于神经网络的视频监控人类活动识别方法也逐渐火热起来。这篇论文就是一篇基于深度学习技术的视频监控人类活动识别的文章。

传统的视频监控人类活动识别方法需要依赖于一些手工特征工程的方法或是一些机器学习方法。这些方法往往耗费巨大的计算资源，并且不够准确。为了解决这个问题，近些年来，很多人提出了基于神经网络的视频监控人类活动识别方法。例如Google的YOLO方法，Facebook的Detectron方法，以及Face++的MTCNN方法等。这些方法都采用了深度学习技术，利用卷积神经网络(Convolutional Neural Network)进行人类活动检测。

然而，现有的视频监控人类活动识别方法仍存在一些缺陷。首先，这些方法仍然处于初级阶段，很难达到实时的要求。其次，训练数据集的获取、标注、存储等环节都比较繁琐，而且还需要大量的人力物力。另外，现有的基于神经网络的方法存在着严重的模型大小问题。因为神经网络模型本身的复杂性，导致其占用的硬件资源较大。另外，由于需要处理高帧率的视频流，因此，很多时候，这些方法还需要通过某种机制来优化处理效率。

因此，为了实现更加精准和实时的视频监控人类活动识别，我国科学院自动化所王春强团队提出了一种新的视频监控人类活动识别方法。该方法主要分为四个方面：（1）使用移动端及静态背景的视频作为输入；（2）使用特定的捕获图像预处理方式；（3）使用人工设计的特征提取器提取感兴趣区域特征；（4）使用深度神经网络进行人类活动识别。

在本文中，我们将详细阐述本方法的各个方面。第2小节将介绍相关术语。第3小节将介绍该方法的核心算法。第4小节将给出该方法的具体操作步骤。第5小节将介绍未来的研究方向。第6小节将介绍一些常见问题的解答。

如上图所示，特征提取器的任务是从输入视频流中提取感兴趣区域的特征。在我们的方案中，我们将使用三个单独的卷积神经网络来完成这一功能。第一个卷积神经网络处理输入视频流，第二个卷积神经网络处理第一个卷积神经网络的输出结果，第三个卷积神经网络处理第二个卷积神经网络的输出结果。每个卷积神经网络具有不同的作用。第一个卷积神经网络提取整个输入视频流中的全局特征。第二个卷积神经网络提取两个相邻帧之间的差异特征，这样就可以捕捉到快速运动和停止运动之间的区别。第三个卷积神经网络是人类活动识别器。它的作用是提取活动发生的区域的特征。
