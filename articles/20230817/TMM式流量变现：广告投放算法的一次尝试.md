# 1.简介
  

随着移动互联网的飞速发展，网民对互联网产品的依赖越来越强，各种新闻、社交、购物等网站也都迅速发展起来。这给传统广告市场带来了巨大的流量变现潜力。目前市面上已经有不少成熟的广告平台，比如Google AdSense、Facebook Advertising、Twitter Ads、Baidu DSP等，这些平台都提供了一些广告优化功能，用户可以通过后台设置各种规则来进行广告投放。但当下仍然存在以下两个问题：

1）传统广告投放模型过于保守，无法反映真实世界中用户的需求和消费习惯；

2）广告主在决策广告投放策略时往往忽视了流量价值，对同一个人的不同页面之间的流量差异没有办法衡量。
为了解决以上两个问题，新型的广告投放模式——TMM模型应运而生。该模型全称Targeted Marketing Message Mix Model（目标受众市场信息混合模型），由一组相关的维度决定了广告的展示方式、类型及频次，这些维度包括但不限于用户所在地区、性别、年龄、兴趣爱好、互动行为、网络喜好、视听欲望等。通过这种方式可以更好地匹配到用户的需求和喜好，提高广告的转化率。同时，由于模型中加入了不同的参数，可以让每个人看到的内容都不一样，能够更精准地分发信息。相比于传统的定向广告方式，TMM模型能更好的满足用户的个性化需求。

针对当前的广告投放市场环境，本文从以下几个方面论述了如何将TMM模型应用于广告投放中：

首先，本文将阐述TMM模型的基本原理和操作流程；

其次，对于广告投放场景中可能遇到的实际问题和挑战，作者给出了相应的解决方案；

最后，本文还会结合实例代码进行详细讲解，并给出一些具体建议。希望能帮助读者更加深入地理解TMM模型和广告投放的工作机制。

# 2.基本概念
## 2.1 TMM模型
TMM模型全称Targeted Marketing Message Mix Model（目标受众市场信息混合模型）。顾名思义，它是一个以目标受众信息为基础的广告投放模式。与传统的定向广告模式不同的是，TMM模型更注重用户的个性化需求，通过一组相关的维度来确定的用户访问目标网站后的广告类型、数量及时效。TMM模型中的主要元素如下：

1）目标受众信息：包括用户的个人信息（包括年龄、性别、兴趣、偏好等）、行为特征（例如点击、搜索、停留时间等）、设备信息（例如手机型号、操作系统版本等）、位置信息（例如所在城市、区域、商圈等）、网络浏览习惯（例如热门关键词、浏览行为模式等）。

2）信息组合：根据用户的信息来源及影响力程度，将所收集到的多种信息源按照一定权重组合成为可用于广告投放的唯一信息标识符。通常情况下，信息的权重越高，就越能体现其重要性。例如，用户年龄越大或用户搜索关键词包含某个特定主题越热门，信息权重就会增加。

3）广告层级结构：通过制定目标受众群体划分不同层级的广告计划，使得广告推送可以更加精准、细致、个性化。例如，针对青年人群可能采取不同的营销策略，针对老年人群则采取更多关注女性的广告。

4）广告形式：广告的类型可以包括文本、图片、视频、链接或其它形式。其中，视频广告由于具有特殊的时效性，能产生更大的效益。

5）广告选择：通过定义特定维度上的优质资源或广告主的品牌优势，可以对广告进行筛选和排序。例如，如果某一天降水量较大，广告主可能会倾向于投放相关的视频广告。

6）广告投放时间：广告的投放时间也可以根据用户的浏览习惯进行调整。例如，用户经常逛医院，就可以选择在凌晨或夜晚进行推广。

总之，TMM模型通过设计一系列的规则来决定用户的信息组合、层级结构、广告形式、时机，从而精准地推送适合用户的广告。

## 2.2 信息收集方法
在广告投放过程中，信息收集是非常重要的一环。信息收集的方式有很多，包括手工填写、自动获取、爬虫抓取、Cookie分析等。这里只讨论最常用的两种收集信息的方法——手动收集和自动收集。

1）手动收集：这是最基本的方法。广告主需要拜访目标受众群体，要求他们填写个人信息，比如姓名、电话号码、地址、年龄、性别、兴趣、偏好等。然后，广告主要在这些信息基础上进一步收集用户的行为特征、浏览习惯等信息。通过这种方式，广告主可以了解到用户的目标受众群体信息，甚至可以据此设计相应的广告活动。但是，手动收集的方法容易造成人力资源和时间成本的增加，且数据的收集难度较大。

2）自动收集：利用机器学习、人工智能等技术，广告主可以使用大数据和统计的方法，收集用户的信息、行为特征等。自动收集的方法不需要拜访受众群体，直接收集数据。通过分析大量的数据，广告主可以找出用户的个性化兴趣、喜好、偏好等，并据此设计相应的广告活动。自动收集的另一个好处是可以节省大量的时间和人力，减轻了管理人员的负担。当然，自动收集的缺点也很明显，它可能会捕获到一些非目标受众的信息。因此，自动收集的方法要慎用。另外，还有些网站或APP通过科技手段进行“跨设备”、“无痕浏览”，也可以收集到用户的信息。但由于这种收集方式难以证伪和追溯，因此，它们的效果一般不如人工方式。

# 3.广告投放原理和操作步骤
## 3.1 广告展示时机
广告的展示时机其实取决于广告主的广告创意质量、广告预算、广告投放渠道等因素，但在一个TMM模型的框架下，应该遵循以下几条建议：

1）广告优先展示在用户最感兴趣的地方：广告优先展示在用户最感兴趣的地方可以有效吸引用户的注意力，促进用户的购买、阅读、分享等行为。

2）定期定量投放广告：由于广告主的广告创意质量及广告投放渠道有限，定期定量的广告投放可以保证广告创意的及时更新。但是，定期定量投放不能替代个性化推荐，因为用户对广告的需求并不总是一致的。

3）突发事件及临近活动时也可展示广告：突发事件或临近活动时也应该展示广告，以增强用户的参与度，提升广告效果。

4）选择好的广告形式：推荐的广告形式应该具有吸引力、传播力、响应力等特点，可以使广告获得用户的点击、分享、关注等行为。

## 3.2 广告内容及显示方法
广告的大小、颜色、形式和位置也是广告的重要组成部分。通常情况下，不同的人群、商品或服务会有不同的广告投放方式和布局。因此，广告主需要设计符合用户需求的广告内容及显示方法。

1）广告内容的设计：广告内容应该简洁易懂，语言通俗易懂，避免冗长模糊，切忌错导。

2）广告的布局设计：广告应该考虑显示位置、布局形式等，尤其是在横幅式广告、视频广告等更为复杂的广告形式时，广告的布局设计非常重要。

3）广告的打开方式设计：广告的打开方式可以有多样化的选择，比如链接打开、电话拨打、信息分享等。

4）相关广告推荐：除了正式的广告活动外，广告主还可以考虑推荐相关产品、服务等广告，增加用户对该商品或服务的了解，并鼓励用户下单购买或进行其他操作。

## 3.3 流量变现的实现
广告投放是一个持续的过程，只有广告主充分了解用户的需求、喜好并把握时机，才能有效地进行流量变现。因此，实现流量变现的具体措施需要具体分析。但最基础的实现方式就是通过广告主的付费宣传和互动来实现。

1）付费宣传：广告主可以开设免费试用或付费赞助，提供广告收入来支持自己。

2）互动互动：广告主可以邀请用户参加线下或线上互动活动，激发用户参与互动。例如，用户注册之后奖励积分、优惠券等，又或者在线购物成功后可以获得奖励。

3）广告主自营营销：广告主可以建立自有的品牌形象、产品售卖、宣传策略，采用自己的媒体和网站来进行营销。这样既可以弥补广告投放的短板，也可持续保持收入。

总之，TMM模型为广告投放设计了一套完整、严谨的规则体系，对广告投放的时机、内容、方式进行了高度控制，并以一定的方式促进了用户的流量变现。因此，基于TMM模型的广告投放模式，可以提高广告投放效率、扩大广告效果，进而实现广告主和用户双赢。
