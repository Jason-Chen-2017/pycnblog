# 1.简介
  

字符串是编程中的基础数据类型。在很多编程语言中都内置了用于处理字符串的函数库，其中最常用的就是字符串拆分函数split()。而对于初学者来说，如何用Python实现字符串的分割，并将其返回到一个列表里面，却不是那么容易理解和体会。本文通过简单易懂的示例，循序渐进地向读者展示如何利用split()函数进行字符串的分割、元素返回以及使用正则表达式进行分割。最后还将介绍一下split()函数的一些扩展功能，如字符串的左右对齐、去除空白符等。
# 2. 基本概念和术语说明
## 分割(Split)与切片(Slicing)的区别
字符串的分割和切片是两种截取字符串的方式。前者是根据某个分隔符把字符串按照某种规则进行划分，并返回结果；后者是直接从字符串中按照指定的索引位置进行选取。两者的区别主要如下：

1. 分割的结果是一个列表，而不是字符串。
2. 分割的规则可以是固定长度或者不固定的。
3. 不需要指定起始位置，只需要给定一个终止位置即可。
4. 对字符串切片来说，切片操作的语法是[start:end:step]，也可称之为半闭半开（half-open）的切片。

## 正则表达式(Regular Expression, RE)
RE是一种描述字符串模式的方法。它是一套基于文本的模式匹配语言，使用了一系列简洁的符号来描述、匹配和替换文本中的各种结构。这些符号可以让你准确而精确地控制搜索或替换的策略。本文涉及的字符串分割都可以通过RE来实现。
## Python字符串对象
Python的字符串对象提供了一个方法`split()`，用来按照指定的分隔符对字符串进行分割，然后返回一个由分割出的子串组成的列表。`split()` 方法的工作方式如下：

1. 使用 `split()` 方法将指定分隔符与字符串连接起来，形成一个模式。
2. 将模式作为参数传递给 `re.findall()` 函数，该函数通过模式匹配查找字符串的所有出现位置。
3. 返回所有匹配的子串构成的列表。

`split()` 方法接收两个参数：

* `sep`，默认值为None，表示任意空白字符（包括空格、制表符、换行符等）。也可以设置为指定的字符，表示按照该字符进行分割。
* `maxsplit`，默认为-1，表示分割所有的子串。如果设置一个非负整数n，表示仅分割最多n个子串。

以下是一个例子：
```python
import re
s = 'The quick brown fox jumps over the lazy dog.'
result = s.split(' ') # ['The', 'quick', 'brown', 'fox', 'jumps', 'over', 'the', 'lazy', 'dog.']
```
上面的例子中，通过 `split()` 方法按照空格分割字符串，得到了8个子串，然后打印出了这个列表。

如果要使用多个分隔符，可以使用元字符集和特殊字符 `\|` 来组合不同的分隔符：
```python
s = 'apple,orange;banana|grape'
result = s.split(',|;') # ['apple', 'orange', 'banana|grape']
```
上面例子中，通过 `,|;` 分隔符对字符串进行了分割。

## 参数maxsplit的作用
参数maxsplit表示最多分割多少次，当分割次数超过maxsplit时，停止分割。例如：
```python
s = 'a b c d e'
result = s.split(' ', maxsplit=2) # ['a', 'b', 'c d e']
```
这里由于`maxsplit=2`，因此仅分割了两次，结果仍然是`['a', 'b', 'c d e']`。

## 正则表达式语法
正则表达式是一个类似字符串的序列，用来匹配各种字符串的模式。它的语法非常复杂，但它的规则很简单：

1. `.` 匹配任意单个字符，除了换行符。
2. `[...]` 匹配字符集合中的任意单个字符。
3. `[^...]` 或 `[!...]` 表示不在字符集中的任何单个字符。
4. `*` 重复零次或更多次。
5. `+` 重复一次或更多次。
6. `{m}` 重复 m 次。
7. `{m, n}` 重复 m~n 次。
8. `(expr)` 或 `(?P<name>expr)` 保存匹配到的 expr 的值。
9. `|` 表示或。

下表是一些示例：

运算符 | 描述 | 示例
--- | --- | ---
`.` | 匹配任意单个字符，除了换行符 | `'Hello World'.split('.')` => `['Hello', '', 'World']`
`[abc]` | 匹配字符集合中的任意单个字符 | `'hello world'.split('[a-z]')` => `['h', 'e', 'l', 'l', 'o', 'w', 'o', 'r', 'l', 'd']`
`[^abc]` | 匹配不在字符集 [abc] 中的任意单个字符 | `'hello world'.split('[^aeiou]')` => `['h', 'll', 'wrld']`
`[A-Z]` | 大写字母范围 | `'Hello World'.split('[A-Z]')` => `['H', 'ello World']`
`\w` | 匹配任意字母数字字符，等价于 `[a-zA-Z0-9_]` | `'hello,world'.split('\W+')` => `['hello', 'world']`
`\d` | 匹配任意数字字符，等价于 `[0-9]` | `'\d{3}\-\d{4}'.split('-')` => `['\d{3}', '\d{4}']`
`\s` | 匹配任意空白字符，包括空格、制表符、换行符等 | `'hello    world'.split('\s+')` => `['hello', 'world']`
