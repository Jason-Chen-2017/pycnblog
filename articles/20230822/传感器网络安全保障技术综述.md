# 2.传感器网络安全概述
## 2.1 传感器网络简介
什么是传感器网络？传感器网络(Sensor Network)又称网络传感器，是由传感器节点(Sensor Node)组成的分布式计算环境，其结构复杂、功能多样，能够实时收集、传输和处理大量的原始数据。传感器网络的构成包括多个传感器节点，每个传感器节点可由一台或多台微小传感器组成，这些微小传感器具有高灵敏度、低功耗、高性能和广泛的应用范围。传感器节点之间可以采用通信网络进行连接，形成互联网形式的传感器网络。

传感器网络的特点之一是分布式、动态的。它不仅具有多种数据采集方式，还具备强大的计算能力、存储能力和处理能力，能够满足各种应用的要求。传感器网络的安全性也成为重点关注的方面，主要原因有两个：第一，传感器网络的数据采集、传输、处理等环节可能存在隐私泄露、恶意攻击等安全风险；第二，传感器网络的各种节点以及与它们之间的网络通讯存在一定频率、范围、稳定性等方面的限制，使得传感器网络的安全防护十分困难。

## 2.2 传感器网络的定义、特征和分类
### 2.2.1 传感器网络的定义
传感器网络，英文名 Sensor Network，指由各种传感器节点组成，分布式、动态的计算环境，能够实时收集、传输和处理海量的原始数据，以满足各种应用的需要，是连接、管理和控制大量传感器的网络。传感器网络包括服务器端和客户端两部分。传感器网络服务于人的身体、健康、环境、经济、金融、交通、住房等各个行业，利用各种传感器设备，包括各种传感器和传感器网络设备，如摄像头、雷达、雨水检测仪、门磁感应器、气压计、光电二极管激光探测器、三轴加速度计、磁悬浮传感器、空气湿度检测仪、水流量计、温度计、湿度计等，采集大量原始数据并对其进行处理、分析、存储、转移，用作人工智能、大数据、网络安全等领域的核心技术基础设施，其中包括检测、跟踪、预测、识别、安全防范、故障诊断、运维、控制、通信等功能。

### 2.2.2 传感器网络的特征
传感器网络具有如下几方面的特征：

1．高度分布式、动态的特性：传感器网络中各个传感器节点分布地域广、分布式部署，不同传感器节点之间的连接较为复杂，且网络呈现动态演变的状态，传感器节点一般分为云端节点和本地节点两种类型，云端节点部署在云端，负责收集、传输、处理传感器数据，本地节点部署在传感器网络内部，主要负责数据的采集和处理。

2．数据采集、传输、处理的特点：传感器网络采集、传输、处理各类传感器数据，包括模拟信号、数字信号、环境参数、事件触发信号等，主要由终端节点收集、传输、处理。传感器网络的数据采用传感器标签、设备ID、时间戳、采集位置等标识符进行关联，并基于实时、动态的更新策略进行信息聚合和分析，产生对外输出的业务数据。

3．应用领域广泛：传感器网络应用于各行各业，包括医疗、环境、物流、工业、供应链、交通、住宿、安全、监测、安防、人工智能、智慧城市、智能金融、智能化教育等领域。传感器网络能够收集到大量的原始数据，涉及社会、经济、军事、科学、环境、政治、法律等诸多领域，有助于促进国家治理、经济发展、社会进步、减少财富损失、保障人民生命财产安全。

### 2.2.3 传感器网络的分类
传感器网络按照传感器节点的数量和分布区域，可以分为如下四类：

1．局域网传感器网络(LAN-based Sensor Network): 在同一区域内采用路由协议连接的一组计算机、传感器、控制器等网络元素，其分布和资源受限于单一地区，但能够短时间内产生海量的新数据。

2．城域网传感器网络(MAN-based Sensor Network): 跨越多个城市或区域，采用无线技术连接的一组计算机、传感器、控制器等网络元素，网络规模较大，传输距离、带宽、延迟等限制较少，但资源有限。

3．室内网传感器网络(WLAN-based Sensor Network): 嵌入到家庭、办公室等周边环境中，采集、处理、传输大量的传感器数据。由于其自然环境复杂、空间狭小，与其他传感器环境隔离，难以获取外部世界的输入信息，但由于处于室内环境，环境条件不太适宜于大规模的数据采集和处理，有利于长期稳定的运行和集成。

4．地下网传感器网络(Underground Sensor Network): 在地下的沙漠或地底通过特殊的方式进行采集、传输、处理传感器数据，资源受限，广泛应用于监测、环境保护、智能投资、农林、矿山等领域。
