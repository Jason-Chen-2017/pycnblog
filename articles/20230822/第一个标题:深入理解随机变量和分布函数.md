# 1.简介
  

“深入理解随机变量和分布函数”(Introduction to Random Variables and Distribution Functions)是一门针对程序员、数据科学家、机器学习工程师、统计学家等各个领域的核心概率论课程。相对于传统的公式推导方式，本课程将通过系统地学习多种典型分布的性质及其对应的随机变量的定义、各种分布之间的转换关系、概率密度函数、累积分布函数、边缘分布、条件分布等知识。掌握这些知识有利于提升数据分析、建模的能力，也能帮助解决实际问题中的数据相关性、模型选择等问题。
此课程主要适用于以下群体：程序员、数据科学家、机器学习工程师、统计学家等，需要对计算机基础、线性代数、概率论等知识有基本的了解。
# 2.目标受众
# 2.1 学生（中级）
参加本课程的学生可以有以下几点收获：
- 更好的理解和掌握一系列典型分布及其对应的随机变量的定义。
- 掌握随机变量的数学期望、方差、分位数计算方法。
- 了解分布函数与连续随机变量之间的对应关系，并能够运用这一关系进行抽样和估计。
- 了解分布函数的曲线形态，以及在实际应用中如何选取合适的分布。
- 掌握高斯分布、指数分布、正态分布、伽玛分布、泊松分布、卡方分布、超几何分布、贝塔分布、负二项分布、超平滑分布等分布的概率密度函数、累积分布函数、边缘分布、条件分布等知识。
- 对机器学习、深度学习、统计建模、数据挖掘等领域有所理解和实践。
- 具有强烈的求知欲和学习能力，深入浅出，善于总结归纳，有独到的见解。
# 2.2 中级以上职场人士
此课程的学习对于参与职业教育工作者来说尤其重要。职场人士通常具备较高的自学能力，但往往由于专业背景、经验不足、缺乏学习资源而难以快速成长。通过此课程的学习，他们将可以快速融会贯通各类概率论知识，从而在日常生活中更好地运用概率论模型和技巧。
除此之外，也有助于培养中级以上职场人士的沟通能力、团队协作精神、领导力、分析问题的能力、改进能力等综合素质。
# 2.3 数据科学家/AI工程师
数据科学家和AI工程师是最具天赋、需求最强的研究人员类型。对于这两类人群，这门课或许可以成为他们学习概率论的“捷径”。通过此课程，他们可以在短时间内掌握大量常用的概率论知识，在未来的工作和职业生涯中获得极大的帮助。
# 2.4 高校老师
高校老师需要面向全体学生讲授概率论课程。他们通常身兼数职，既要掌握实践能力，又要教授理论知识。通过此课程，他们可以有效地传递概率论的基本概念和方法论，辅助学生在科研、项目管理、职场等多个领域提升自己。
