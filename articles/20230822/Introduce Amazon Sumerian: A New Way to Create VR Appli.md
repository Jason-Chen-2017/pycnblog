# 1.简介
  
 
虚拟现实(VR)应用程序是一种与真实世界融合的沉浸式体验，通过设备上的头戴显示屏或控制器、眼镜、传感器等传感器获得用户输入并呈现三维场景及动画。由于VR的特性，使得VR应用程序能够让用户获得比在现实世界更高的自由度和真实感。但是，不同于其他行业如游戏领域，企业不仅需要做好VR开发的准备工作（包括虚拟现实技术方面的知识、VR硬件兼容性问题、编程语言方面的能力），还需要考虑部署到不同平台的兼容性、性能优化、功能扩展等等诸多因素。Amazon Sumerian是一个完全托管的云服务，可以帮助企业轻松创建、管理和部署VR应用程序。本文将介绍Sumerian的概况、功能特性、支持平台、部署方式、客户案例等。 

# 2.基本概念术语说明 

## 2.1 什么是虚拟现实 (VR)?  

虚拟现实(VR)是一种将真实世界中的物体、空间、声音、人员转化为电脑图像、声音的视听效果，并提供类似于实体真实世界的互动体验的方式。它利用计算机、显示屏、摄像机、控制器、眼镜、音频设备等各种设备创造出一种全新的视觉、触觉、听觉体验。通过高度虚拟化的数字模型，让真实世界中的物体看起来和活生生一样；并实现与真实世界交互、穿梭、控制，让用户产生身临其境的感受。 

## 2.2 为何要建立一个VR平台？  

随着VR应用需求的日益增加，企业越来越需要一种统一的解决方案。如今越来越多的公司开始关注VR的发展，包括游戏行业、电影制作、运动健身等。这些行业都有独特的业务需求，比如，游戏制作者需要针对VR进行开发，来提升玩家的视野和沉浸感；还有一些影迷和观众也对VR有着独到的需求，因为它能让他们看到别处的场景、拥有不同的感官体验，从而影响他们的生活。因此，构建一个能够满足这些需求的VR平台就显得尤为重要。 


## 2.3 Sumerian 是什么?   

Amazon Sumerian是一个完全托管的云服务，它可以让企业轻松创建、管理和部署VR应用程序。Sumerian提供了以下几个主要功能：

1. VR项目集成工具：Sumerian提供一个可视化的界面，可以轻松创建、编辑、发布和管理VR项目。用户可以很容易地导入各类3D模型、图像、视频，配置各类参数，并基于这些设定快速生成不同种类的VR项目，无需编码。

2. 云端渲染引擎：Sumerian使用云端渲染引擎(Cloud Rendering Engine)，它可以在服务器上进行渲染，不需要用户安装特殊的软件。它通过图形处理单元GPU的加速，在保证项目质量的前提下，极大地提升了渲染速度。

3. 支持多平台：Sumerian可以运行在各种设备上，包括Android手机、平板电脑、电脑等。目前，Sumerian已经支持iOS、Android、Windows、Mac OS、Linux、SteamVR、Oculus Quest等多个平台。

4. 简单的应用程序接口：Sumerian提供简单易用的VR应用程序接口API，允许用户与Unity3D或Unreal Engine等第三方引擎集成。通过此接口，用户可以方便地调用各类功能组件，并自定义自己的脚本逻辑。 

5. 用户数据分析：Sumerian提供数据分析功能，可以帮助用户了解应用程序的用户情况、访问次数、播放时长、VR距离、VR停留时间等指标。同时，它还提供报告功能，让用户可以详细查看每个项目的数据分析结果。

6. 高级安全机制：Sumerian支持HTTPS加密传输协议，具备安全防护的强大能力。它还支持身份验证和授权功能，可以限制只有经过授权的用户才能访问私密数据。

7. 服务监控和故障排查：Sumerian为用户提供了服务监控和故障排查功能。通过日志记录和异常错误检测，用户可以及时发现系统的问题并采取适当措施解决。
