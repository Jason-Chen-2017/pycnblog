# 1.简介
  

Tabular data是一个非常重要的数据类型。它描述的是一种结构化的数据表格形式，每一行代表一个数据样本，每个列代表一个特征或属性。常见的有CSV文件、Excel文件等。Tabular data在机器学习任务中扮演着至关重要的角色。它能够帮助我们对数据进行分析和处理，并快速地找到一些规律性的模式。

Kaggle是一个基于Web的竞赛平台，提供众多机器学习相关的比赛。每一个比赛都有多个评分标准，包括训练集的准确率、测试集的准确率、时间、内存占用量、提交次数限制等。其中的“Tabular”类比赛旨在解决一般的机器学习任务（分类、回归）之外，还需要考虑标签数据及其相关的变量之间的关系。比赛的目标就是通过预测某个变量的值，使得该变量具有最大的相关性。

# 2.基础知识
## 2.1 Kaggle上关于“Tabular”类的比赛的基本规则
### 2.1.1 介绍
Kaggle是一个基于Web的竞赛平台，用于研究和开发机器学习技术。旨在促进科技人员之间进行协作，分享知识，发现模式，创造未来。自2010年起，Kaggle已经成为业界领先的机器学习比赛网站。其用户遍布全球，涵盖了从企业、政府到个人。截止目前，Kaggle共举办过超过40万个比赛，其中“Tabular”类共计超过9万个。

在Kaggle上，“Tabular”类比赛是一种较为特殊的比赛。因为一般来说，对于分类、回归任务来说，我们的输入数据只能是数字，而对于Tabular data来说，输入的数据则不能是数字，只能是表格型的数据。因此，这种类型的比赛特别适合于处理与表格数据有关的机器学习问题。

比如，像广告、销售预测、天气预报等应用场景，我们可能需要根据某些统计数据、历史数据，甚至是外部因素如市场趋势等，预测某些属性值。这些属性值通常都是表格型的数据。

### 2.1.2 比赛难度
为了激励参赛者更加努力地完成“Tabular”类的比赛，Kaggle将其设定为平均水平以上水平的选手才可进入。所以，在参加比赛之前，应该对该比赛的要求有一定的了解。

在“Tabular”类比赛中，除了要有较强的模型性能能力外，还有一些硬性条件。

首先，原始数据集大小要达到一定数量级。比如，对于表格数据，通常都会存在很多种形式，并且大小不断增长。所以，参赛者需要准备好大量的原始数据，以备后续使用。

其次，数据集中必不可少的有标签数据。即每一条记录都有一个“target”值，用来表示该条数据的预测值。如果没有标签数据，则无法进行有效的训练和预测。

再者，输入数据的维度要保持一致。也就是说，所有数据必须是同一维度的。例如，若输入数据为一个张量，则输出数据的维度也必须为该张量的最后一维。否则，可能会影响模型的训练和预测。

此外，还有一些规则需要遵守，它们是：
- 提交的时间限制；
- 模型的大小限制；
- 可使用GPU加速；
- 使用官方的数据加载器；
- 有详细的描述文档。

除此之外，还有一些注意事项。比如，如果出现过拟合现象，那么可以通过增加更多的训练数据、减小正则化参数等方式来缓解。同时，还需要注意各项指标的计算方法、权衡取舍。在训练过程中，还需要监控GPU显存和时间，避免资源超限。
