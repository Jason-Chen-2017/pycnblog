# 1.简介
  

Java是一个非常流行的面向对象编程语言，在过去几年里它的发展速度超越了其它一些高级语言。因此，了解Java语言的特性、优势、局限等信息对于一个不断变化的职场环境来说至关重要。本书即是为广大的程序员提供必要的参考资料，帮助他们掌握Java语言的基础知识、提升工作效率、规避陷阱并更好地理解编程范式。
《Effective Java》是一本经典著作，它对Java中众多有效技巧进行了全面的总结和归纳。每一条Java最佳实践都具有鲜明的语言特色和针对性，读者能够根据自己的实际情况灵活运用这些技巧，从而写出更健壮、更高质量的代码。
这本书并不是教程或入门书籍，它深入浅出的剖析Java的各种特性，并给出各项技巧的例子。作者将这种丰富的语言知识与实例应用相结合的方式，生动形象地呈现出来，让读者可以把学到的东西转化成实际工作中的解决方案。
# 2.内容简介
## 一、什么是Effective Java?
《Effective Java》是一本Java编程规范的著作，作者是美国电气工程师道格拉斯·克罗克福德（<NAME>）的个人成果。克罗克福德是Java阵营的杰出创始人之一，他于1997年发表了名为“The Java Programming Language”的著作。该书首次系统地介绍了Java语言及其相关技术。后续又陆续出版了几本关于Java的著作。其中的《Effective Java》作为Java编程规范一书的集大成者被广泛认可。
《Effective Java》共分为十章，主要内容包括：第1-3章探讨Java的特性；第4-6章涉及并发处理、虚拟机和内存管理；第7-9章着重介绍设计模式、异常处理和性能优化方面的技巧；第10章详细介绍了Java平台技术，如 JDBC 和 Swing 。
作者认为，阅读《Effective Java》，能够掌握Java开发人员应具备的技能以及熟练掌握Java的各种特性和用法，有助于使程序员编写出更好的代码。阅读完之后，读者就可以充分利用Java语言的各种特性，提升自己在软件开发中的能力水平。
## 二、为什么要读《Effective Java》？
无论是学习Java还是编程语言，《Effective Java》都是必读的经典。阅读本书能帮助读者增强对Java语言的理解，提升自己的编程能力。除此之外，还能学到其他编程领域中通用的编程技巧。本书的学习效果如何还需要读者自己评价。下面是一些读者反馈：
### 1. 提升编码能力
《Effective Java》是一本极具实用性的书。作者通过大量示例和直观的语言描述，循序渐进地展示了一些Java编程中比较重要的最佳实践。每一章的内容都是密不可分的，读完整个书后，读者就能将所学应用到日常工作中，极大地提升自己的编程能力。
### 2. 更快速地理解Java
《Effective Java》既有一般性的知识也有特定领域的深度。作者对Java的各种特性和用法进行了详尽的分类和讲解。每条规则或技巧都提供了清晰的解释和示例。因此，读者可以根据自己的兴趣和能力选择适合自己的章节。另外，《Effective Java》的网站www.effectivejava.com上有大量的学习资源，读者可以在其中找到自己感兴趣的主题。
### 3. 为工作更轻松些
《Effective Java》不是教科书，而是一份实用手册。它涉及的知识点很多，难免会让新手望而却步。但是只要细心研读，并遵循作者的指示，读者就可以马上上手。而且，由于每条建议都有直接的实践意义，读者可以边学边用，快速入门。
## 三、《Effective Java》的内容范围
《Effective Java》涵盖Java语言的所有方面，从基础语法到泛型编程、注解、枚举、集合框架、I/O 流、网络编程、并发编程等，还有Java平台上其他方面的知识。每章的内容都很丰富，内容包括理论知识、示例代码、实操场景和建议。
### 1. 第一部分：Java语言的基本知识
这一部分介绍了Java语言的基本知识，包括变量类型、控制结构、数组、字符串、运算符、类和接口、方法。其中，核心内容有：类型安全转换、参数检查和文档注释。
### 2. 第二部分：创建和使用类
这一部分介绍了类的创建、成员变量、构造器、访问控制权限、方法的声明和调用，以及对象的初始化过程。其中，核心内容有：封装、继承、多态、组合以及final关键字的使用。
### 3. 第三部分：高级泛型
这一部分介绍了泛型技术的实现原理、边界类型、通配符、类型擦除以及自建容器类。其中，核心内容有：类型安全容器、标准库中的泛型类和接口。
### 4. 第四部分：注解处理
这一部分介绍了注解的定义、元数据（metadata）、批注的种类、生成注解的工具以及注解处理器的实现。其中，核心内容有：自定义注解、注解处理器API、编译时和运行时的注解处理机制。
### 5. 第五部分：枚举和注解
这一部分介绍了Java enums 和 annotations 的概念，以及如何使用它们来表示状态和配置选项。其中，核心内容有：使用枚举代替switch语句、使用注解来替代复杂的配置。
### 6. 第六部分：集合框架
这一部分介绍了Java Collections Framework 中的各种容器类，包括列表、队列、集合、映射、算法以及同步器等。其中，核心内容有：泛型用法、迭代器模式、集合排序以及线程安全。
### 7. 第七部分：输入输出
这一部分介绍了Java I/O 流的使用，包括文件输入/输出、序列化和字符集转换。其中，核心内容有：正确使用输入/输出流、避免缓冲区溢出以及字节排序。
### 8. 第八部分：网络编程
这一部分介绍了Java NIO（New Input/Output）中的SocketChannel和ServerSocketChannel、Java URLConnection API以及Java 7 中的改进。其中，核心内容有：正确使用 SocketChannel 和 ServerSocketChannel、URLConnection 代理设置以及 Java 7 中引入的 HTTP Client 技术。
### 9. 第九部分：并发编程
这一部分介绍了Java 5.0 中引入的并发包以及 JSR 166 中的一些关键类，包括 Executor 框架、原子变量类、并发工具类以及并发集合类。其中，核心内容有：线程池的正确使用、正确管理线程资源、共享资源的同步和锁定、ConcurrentHashMap 和 BlockingQueue 使用。
### 10. 第十部分：Java 平台技术
这一部分介绍了一些Java平台上的一些高级技术，包括JDBC、JavaBeans、JNDI、RMI、安全性、国际化、Swing组件等。其中，核心内容有：理解 JDBC 数据源、使用 JavaBeans 定义用户界面、使用 RMI 进行分布式计算、防止安全漏洞以及正确使用国际化支持。
