# 3.基础知识
- HTML(HyperText Markup Language)、CSS(Cascading Style Sheets)、JavaScript是构建Web页面的三大基石。
- HTTP(HyperText Transfer Protocol)，它定义了浏览器和服务器之间通信的规则。
- RESTful API(Representational State Transfer Application Programming Interface)，它是一种用于开发面向资源的应用的WEB服务接口标准。
- AJAX(Asynchronous JavaScript And XML)，它是一种利用JavaScript实现异步Web交互的技术。
- jQuery是一个轻量级的库，可以简化Web开发。
- Vue.js是一个渐进式框架，易于上手，适合快速搭建复杂的单页应用。

# 5.工具与技术
- IDE(Integrated Development Environment)：集成开发环境，比如VS Code、Sublime Text。
- Git(Version Control System)：版本控制系统，用来管理代码仓库及版本历史。
- Webpack：模块打包器，用来将模块编译成浏览器可识别的代码。
- Node.js：JavaScript运行时环境，用来执行后端代码。
- Docker：容器技术，用来部署应用到生产环境。
- Kubernetes：容器编排平台，用来编排Docker容器集群。
- Jenkins：持续集成平台，用来自动化编译发布代码。
- MongoDB：NoSQL数据库，用来存储数据。
- MySQL/PostgreSQL：关系型数据库，用来存储数据。
- Nginx：Web服务器，用来提供静态文件服务。
- Apache Tomcat：Java Web服务器，用来运行Java应用。
