# 1.简介
  

在本篇中，我们将讨论为什么一个企业或个人需要具备“创造性思维”，以及该思维对其提升产品、服务、竞争力的作用。


什么是“创造性思维”？

“创造性思维”是指能够发现自己的需求、洞察和把握未知，并把这些想法付诸实践、形成独特产品和服务的能力。创造性思维是一个复杂而多变的技能，它涉及到不同的学科知识、领域经验和创新能力。

为什么要有创造性思维？

“创造性思维”能够帮助企业、组织和个人：

1. 为客户提供独特价值：创造性思维能够发现客户需求、找出解决方案的奥秘、与之匹配的产品或者服务，并实现其价值的传播。因此，通过提升客户满意度，“创造性思维”可以帮助企业提升品牌影响力和忠诚度。

2. 提升产品质量：当我们具备了创造性思维能力时，我们就可以应用自己的理解和专长，创造出满足用户需求、可靠可信、安全有效、功能完善等各种性能的产品。而这种能力则会影响产品的销售收入、市场占有率和声誉。

3. 梳理组织结构：在一个由多个团队组成的公司里，每个人的能力都各不相同。而每一位员工都会具有一些独有的创造性思维，他们可以通过自己的判断和洞察力来分配工作，避免产生分歧和冲突。

4. 培养管理才能：“创造性思维”使得员工们能够清晰地认识到自己职责的边界，明白如何统筹全局，协调各方资源，而不是依赖于权威和官方命令。此外，“创造性思维”还能促进员工之间相互合作，共同推动产品和服务的发展。

5. 发展竞争力：无论是在战略层面上还是在竞争环境下，任何行业都需要持续创新的能力。在这样的环境里，“创造性思维”可以帮助企业充分发挥自身的资源优势和创新精神，开发出更加具有竞争力的产品和服务。



什么是“专业化思维”？

除了要具备创造性思维之外，作为专业人员，我们的知识、技能也越来越复杂。当我们能够以更专业的方式思考和处理问题的时候，我们才会获得更多的能力。

专业化思维包括以下几个方面：

1. 技术专业化：在IT行业，“专业化思维”的主要体现就是采用技术上的专长来完成工作。比如，如果我们想要从事软件工程工作，我们就应该对编程语言有较为深刻的理解。对于移动应用开发来说，我们就需要掌握iOS、Android等多平台技术的知识。

2. 业务专业化：业务专业化往往要求我们在一定的领域内具有高度的知识储备和丰富的实战经验。例如，如果我们希望成为一名HR专家，那么我们就会掌握相关的管理知识，如薪酬福利制度、招聘流程等。

3. 知识产权专业化：知识产权专业化要求我们对知识产权有深刻的认识和理解。在这个过程中，我们也会对商业模式和盈利模式等概念有所了解。

4. 营销专业化：营销专业化的关键在于找到与目标客户最匹配的产品和服务。例如，当我们希望成为一名营销专家时，我们就会充分研究市场营销的理论和方法，以及各种营销渠道的运用。

综上，“创造性思维”和“专业化思维”是企业成功的不可缺少的支柱。只有拥有了良好的创造性思维和专业化思维，企业才能在市场竞争中生存下来，并最终获胜。
