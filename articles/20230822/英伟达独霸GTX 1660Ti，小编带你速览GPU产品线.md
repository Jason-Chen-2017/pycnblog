## 2.2 核心组件和特性
NVIDIA GTX 1660 Ti 采用 PCI-E Gen 3 x16 接口，单板主频 1565 MHz，显存容量 6GB，其中 GDDR6X 显存为 4GB。由于采用了 Turing 架构，因此有着较好的性能和新颖的技术。下面我们就来逐一了解 GTX 1660 Ti 的主要特性。
### 2.2.1 Tensor Cores
GTX 1660 Ti 拥有超过 1,000 个 Tensor Cores，能够提供比 AVX2 或 AVX512 更高的性能。这些 Tensor Cores 可以同时执行多个独立的浮点运算，大幅提升计算效率。Turing Tensor Core 提供的额外算力主要来自于深度学习模型中卷积神经网络的计算需求。GTX 1660 Ti 在图像识别、机器视觉领域的表现尤为突出。
### 2.2.2 Tensor Cores 和 FP16 混合精度计算
Tensor Cores 的性能优势不仅体现在单精度浮点数 (FP32) 上，而且可以利用 FP16 混合精度计算，即在运算过程中先将数据转换为半精度 (FP16)，然后再进行运算，最后再转回 FP32，以节省显存空间。这样做虽然会引入一定损失，但是对于某些模型来说，可以大幅度提升性能。例如，ResNet-50 模型的训练过程，就能利用 Tensor Cores 和混合精度计算，并取得了很大的加速效果。
### 2.2.3 多线程并行计算
CUDA 和 OpenCL 等编程框架支持多线程并行计算，从而提升性能。对于复杂的计算任务，可以使用 CUDA 或 OpenCL 对不同的数据块或区域进行并行计算，实现真正意义上的多线程并行。
### 2.2.4 动态并行技术
NVIDIA Turing 架构引入了动态并行技术，允许多个 Tensor Cores 执行相同的工作。通过这种方法，就可以进行复杂的并行计算，将更多的计算资源分配给计算密集型任务。
### 2.2.5 半精度浮点数 (FP16) 和 INT8 量化计算
GTX 1660 Ti 支持基于 FP16 的混合精度计算，以节省显存空间，并有助于加快计算速度。INT8 量化计算技术可以减少计算量和功耗，尤其是在移动端设备上。
### 2.2.6 深度学习优化器
NVIDIA 使用其定制化的深度学习优化器 (Deep Learning Optimizer, DLO) 来加速深度学习模型的训练和推断。DLO 可以自动生成高效的优化计划，并根据模型的结构和规模调整计划。它还可以使用一系列的自动技术来检测并避免常见的错误，从而保证训练过程的稳定性和可靠性。
### 2.2.7 Graph Processing Unit （GPGPU）
GTX 1660 Ti 还拥有一个 Graphics Processing Unit （GPU），称为 Graphics Processing Unit （GPGPU）。GPGPU 是一种专用的加速处理单元，专门用于图形处理和实时计算。由于图形处理和视频游戏引擎的计算需求，GTX 1660 Ti 的 GPGPU 有助于提升游戏和渲染应用的性能。
### 2.2.8 Fermi 性能平衡
为了达到最佳性能，NVIDIA 为 GTX 1660 Ti 设计了三种级别的性能模式。其中两种模式分别是低功耗模式和核心性能模式。在核心性能模式下，计算资源主要集中在 Tensor Cores 中，能提供高达 8 TFlops 的计算性能。此外，还提供了精心设计的 Fermi 性能平衡算法，它将核心性能模式下的性能和功耗进行协调。在低功耗模式下，系统采用极低的电压，只运行必要的 GPU 核心，使得节能成本降低。
### 2.2.9 游戏 Ready
据报道，NVIDIA GTX 1660 Ti 将于 2020 年春季发售。这块显卡有望成为 PC、笔记本和移动平台的首选之选。英伟达 CEO Satya Narayana Kadivar 表示，通过这一系列的升级，将使得 GTX 1660 Ti 成为世界上性能最高的独立显卡。不过，目前还处于 Beta 测试阶段，可能还存在一些 bugs，希望 AMD 能够在这个过程里提供帮助。
