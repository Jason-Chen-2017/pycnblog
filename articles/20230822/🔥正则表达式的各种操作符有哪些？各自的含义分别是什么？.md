# 1.简介
  

正则表达式（Regular Expression）又称规则表达式、匹配表达式或匹配模式，它是一个特殊文本字符串用来向字符串搜索、替换等操作，它定义了一种字符串匹配的模式，通过该模式可以方便快捷的检索到所需的内容。本文将会详细介绍正则表达式的几种最常用的操作符，包括普通字符、字符类、限定符、分组、反向引用和其他一些符号。
# 2.普通字符
普通字符是指在正则表达式中表示一个精确字符匹配，也就是说当字符出现时就进行匹配，否则不匹配。如：a/b/c/d/e。如下图所示:


# 3.字符类
字符类是用方括号括起来的任意单个字符或者字符范围，它能够匹配指定集合内的任何一个字符，或是匹配没有指定集合内的任何一个字符。如：[abc]、[A-Z]、[0-9]、[^\w\d_]。如下图所示:


# 4.限定符
限定符用来控制正则表达式中的匹配规则，用于指定正则表达式的一个子集，限制它能匹配的字符数量，或是在指定的位置或条件下才能匹配到所需的内容。限定符包括点、星号、加号、问号、花括号、尖括号及反斜杠。如下图所示:


# 5.分组
分组是正则表达式中一个非常重要的概念，它的作用就是用来指定一个子表达式，这个子表达式可以被重复使用多次。分组有两种形式：捕获组（Capturing Group）和非捕获组（Non-capturing Group）。捕获组在正则表达式中通常被用在需要提取子串的地方；而非捕获组一般都是用在逻辑上对整个表达式做一些控制。如下图所示:


# 6.反向引用
反向引用，也叫递归匹配，即允许后面接着前面的正则表达式。其主要目的是从已经匹配过的表达式中提取信息。例如，对于邮件地址的验证，如果我们想验证出邮件地址的域名，就可以使用反向引用。如下图所示:
