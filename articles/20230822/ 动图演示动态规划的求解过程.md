# 1.简介
  

## 1.1什么是动态规划？
动态规划（Dynamic Programming）是一种求解最优化问题的一种方法，它利用了二维数组来表示子问题之间的关系。动态规划常常用来解决一些复杂的问题，例如，在很多种情况下，当给定一个问题，要求解出其最优解时，通常可以通过分阶段地构造最优解来进行，而使用动态规划可以有效地减少计算量并提高时间效率。因此，动态规划是一种经典的算法设计技巧，被广泛用于许多领域。

## 1.2动图演示动态规划的求解过程
首先，我们介绍一下一个经典的问题——背包问题。给定一组物品和每件物品的重量和价值，用这些物品装入背包，最多可承受多少重量？也就是说，要选择一些物品，将它们集合起来，最大程度地满足背包所能容纳的最大重量。假设有n件物品和一个容量为W的背包。如下图所示，左边的矩阵是一个最优决策图，右边的线条则是DP方程：
其中，$v_i$是第i件物品的价值，$w_i$是第i件物品的重量；$l_j$是装入第j件物品之前已经装入的物品中能够获得的最大值；$a_ij$是第i件物品与第j件物品组合后得到的价值增益。DP方程用数组来表示，状态转移方程如下：
$$dp[i][j]=max\{v_ix_i+\max\{l_k(w-\sum_{i\leq k<j}a_ix_{ik})\mid 0\leq k < j\}\}\quad (0 \leq i < n,\ 0 \leq j < W)$$
显然，$dp[i][j]$表示把前i件物品装入容量为j的背包里，所能获得的最大值。同时，由于在考虑每件物品都可能选取或不选取的情况下，总共需要考虑$2^n$种情况，所以这种方法的时间复杂度太高。为了提高时间效率，我们可以对上述矩阵进行初步预处理，并通过递推的方式求解$dp[i][j]$的值，最终得到最优解。
