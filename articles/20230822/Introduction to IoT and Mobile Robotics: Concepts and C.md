# 2. 基本概念、术语

## 2.1 AI、机器人技术、物联网技术

“Artificial Intelligence (AI)”、“Robotics” 和 “Internet of Things (IoT)” 是三个重要的计算机科学领域。它们分别是：人工智能、机器人技术以及物联网技术。

### 人工智能（AI）

人工智能(Artificial Intelligence, AI)，英文缩写为 A.I. ，是一个关于如何让电脑具有智能的科学与工程学术语。一般来说，人工智能主要研究如何使机器像人一样与人类交流、决策和制定行为，并且能够学习、推理和解决问题。人工智可以分为两个大的方向，即符号主义与连接主义。

符号主义认为人类的思维过程可以使用符号系统进行模拟，利用这种模拟可以实现人工智能。连接主义则认为人类由多个部件组成，每个部件都可以做一些独立的任务，这样就形成了一个整体的结构。通过对个体部件之间的联系进行模仿，人工智能可以模拟人的行为，并从中提取规律，从而实现对人的理解、判断、学习、推理等能力的提升。

机器学习和深度学习是人工智能的两种主要方法。机器学习方法根据输入的数据来调整模型的参数，使得模型能够基于训练数据更好地预测未知的数据。深度学习方法是基于神经网络的机器学习方法，它使用多个层次的神经网络来模拟人类的学习、推理、决策等过程。

### 机器人技术

机器人技术(Robotics)，是指利用计算机技术来制造、控制机器人的各种技术领域。机器人技术的应用领域十分广泛，从工业、家庭助理机器人到军事、科技等应用领域均可找到相应的应用。机器人技术可以分为如下几个主要方向：

- 工业机器人：工业机器人技术研究如何将普通机器人的功能与自动化技术结合起来，以提高效率、节省资源、降低成本，提升产量和产品质量。工业机器人技术在工业领域得到了广泛应用，如3D打印机、激光雕刻机、汽车零部件和组件、自动化金属加工等。

- 自动驾驶汽车：自动驾驶汽车技术旨在用机器人技术改善人类驾驶汽车的安全性、经济性和舒适性。自动驾驶汽车的设计需要考虑驱动力、视觉、听觉、嗅觉、触觉等多种感官信息的融合，同时还要兼顾操控精确性和灵活性。

- 运动控制机器人：运动控制机器人技术研究如何将高级的动作控制器和嵌入式处理器结合，来实现更高的精确性、控制速度和稳定性。运动控制机器人可以用于医疗、教育、自动化、环境维护等领域。

- 社交机器人：社交机器人技术旨在打破传统工业机器人的束缚，自由地和人类进行交流、协作和互动。社交机器人可以用来帮助老人、残障人士、贫困群体等人群完成日常生活中有益的重复性工作，减轻劳动者的压力和负担。

- 儿童机器人：儿童机器人技术旨在使儿童具备自己的能力，进行简单的拼装活动，并学会自主控制。儿童机器人有利于促进儿童发展，并且可以在没有父母的情况下进行自我保护。

### 物联网技术

物联网技术(Internet of Things, IoT)，英文缩写为 I.o.T. ，是由微软、谷歌、Amazon、小米等公司共同研发的一系列技术和规范，基于互联网技术、通讯技术、计算技术等新兴技术，使各种物理设备（包括传感器、智能电器、消费电子产品、环境噪声、人工智能等）相互连接，实现信息共享、数据的收集和分析，从而实现数据采集、处理和传输的自动化，达到集约、精准、实时地获取有关世界的一切信息、知识和服务。

物联网技术通常包括如下五大类技术领域：

- 终端设备制造商：物联网终端设备制造商负责生产物联网终端设备，包括嵌入式系统、智能手机、平板电脑、笔记本电脑、路由器、基站、汇聚节点等；

- 通信技术：物联网通信技术包括无线传输、有线传输、红外技术等，能够实现设备间、设备与云端之间、设备与终端用户之间的通信；

- 数据采集技术：物联网数据采集技术主要基于不同类型的传感器、激光雷达、RFID、毫米波、Zigbee、LoRa、Wi-Fi等不同方式对终端设备进行数据采集；

- 数据处理技术：物联网数据处理技术主要基于云端服务器进行数据处理，包括数据采集、数据存储、数据分析和数据展示等环节；

- 应用服务商：物联网应用服务商主要提供各种物联网服务，包括安全防护、物流配送、远程监控、智慧城市、智能电表、物联网支付、智能路由器等。

## 2.2 分类

物联网的分类是为了方便大家区分，主要分为三种：

- 一级设备（First Generation Devices）：第一代物联网设备，如两脚泡泡套装、开关插座等，主要应用于简单、便捷的家用应用。

- 二级设备（Second Generation Devices）：第二代物联网设备，如穿戴式智能健康监控、网关、传感器盒子等，是指通过新型的物理方式或装置，构建起能够与智能终端设备交互的设备，增强了对实体环境的感知能力、精密控制能力和数据采集能力。

- 第三代设备（Third Generation Devices）：第三代物联网设备，如智能制冷机、智能空调、智能洗衣机等，是指以一种标准化的方式来实现家用环境中智能化的设备。

目前，中国有关部门已经开始布局智能城市建设，这意味着物联网正在向各行各业迅速蔓延。

## 2.3 特征

物联网具有如下几个特征：

1. 高度互连的系统：物联网系统由互联网、传感器、感应器、终端、服务器、应用软件和其他信息系统组成，彼此高度互连，具有极高的灵活性和弹性。

2. 大范围部署：物联网系统跨越各行各业，覆盖范围广、部署规模大，遍及全球各个角落。

3. 异构性：物联网系统由不同的硬件、软件、协议以及业务模型组成，而且这些模块存在异构性。例如，传感器采集的都是数字信号，而处理和分析这些信号的硬件、软件等也是异构的。

4. 可扩展性：物联网系统是可扩展的，可以增加新的模块来满足新的需求。例如，当某个区域出现问题时，可以增加新的传感器和连接线路来补充系统。

5. 时变性：物联网系统是动态的，随着信息的快速更新、实时反馈、恶意攻击等等发生变化。因此，物联网系统需要快速响应、容错和动态调整。

6. 数据共享和流通：物联网系统采集的数据可以通过不同协议、不同协议的数据格式进行流通。例如，有些数据可以采用 JSON 格式、有些数据可以使用 XML 格式进行传输，这些数据既可以被终端设备直接使用，也可以被云端服务器使用。
