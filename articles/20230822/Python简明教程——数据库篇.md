# 1.简介
  

## 一、数据库概述
数据库(DataBase，DB)是按照数据结构来组织、存储和管理数据的仓库。它是广泛应用于各种各样的系统的关键组件之一，用于保存、检索、更新和维护敏感信息，并支持多种用户对数据的共享访问。常见的数据库产品有关系型数据库、对象关系数据库（如Oracle、MySQL等）、文档型数据库（如MongoDB、Couchbase等）。
## 二、为什么要用数据库？
### 1. 数据安全性
作为存储、管理和提取大量数据的主要工具，数据库的重要作用之一就是数据安全性。任何时候都不希望出现数据被非法侵入、泄露、篡改的情况。通过数据加密、数据隔离等方法，保证数据的私密性和安全性是非常重要的。
### 2. 数据完整性
由于数据保存在不同的地方，导致不同源的数据可能存在不一致性。因此需要一个统一的平台进行管理，通过确保数据完整性，可以避免数据错误、异常的发生，保证数据的准确性和真实性。
### 3. 数据迁移、备份、恢复
随着业务的快速发展，数据规模也在不断扩大。当数据量越来越大时，如何有效地管理数据、实现数据备份和迁移、及时发现数据错误、降低数据损失是一个值得考虑的问题。数据库提供了一系列功能来解决这些问题，例如，数据迁移可以在多个数据库之间同步数据；备份可以帮助数据长时间保留；恢复可以把误删或损坏的数据还原到原先状态。
### 4. 分布式计算
对于海量数据集分析、实时查询等需求，需要把数据分布到不同的节点上进行处理。这种情况下，数据库在性能、可靠性、扩展性等方面都会成为影响因素。
### 5. 提升开发效率
数据库是一项基础设施，用来储存、索引、搜索和处理大量数据的工具。通过使用数据库，可以提高开发者的开发效率，减少重复工作，节省资源。
## 三、数据库类型
1. 关系型数据库
    - MySQL: 是最流行的关系型数据库，由Oracle公司开发，属于商用数据库，提供强大的容错能力和丰富的特性。
    - Oracle: 它是甲骨文公司推出的关系型数据库产品，具有良好的性能、可靠性和稳定性，能够适应企业级环境下的复杂查询场景。
    - SQL Server: Microsoft公司推出的一款关系型数据库产品，具备强大的查询能力和灵活的数据模型设计能力，深受行业和政府部门的青睐。
    - PostgreSQL: 以开源的BSD许可证方式发布的免费数据库管理系统，功能强大、速度快捷、兼容性好、支持丰富的数据类型，适合于web应用、云计算、金融服务等领域。

2. 非关系型数据库
    - MongoDB: 一种NoSQL数据库产品，基于分布式文件系统构建，具备水平扩展、易用性、高性能等优点。
    - Redis: 一种开源的高性能键-值数据库，支持多种数据结构，具备极高的读写性能。
    - Cassandra: Apache Software Foundation推出的分布式NoSQL数据库，同时兼顾了ACID事务和高可用性，可以更好地满足网站的动态实时写入要求。
## 四、SQL语言
Structured Query Language（SQL）是关系型数据库语言，是一种标准化的计算机语言，用于管理关系数据库系统。其结构化查询语言（英语：structured query language，缩写为SQL），是一种专门用于管理关系数据库的标准编程语言，也是关系数据库管理系统中使用的语言。它允许用户通过标准的SQL命令，直接对关系数据库进行操作。
SQL语句的一般形式如下：SELECT <字段列表> FROM <表名> WHERE <搜索条件>;
## 五、创建数据库
创建一个新的数据库可以用CREATE DATABASE命令，语法如下：CREATE DATABASE database_name;
如果指定了IF NOT EXISTS关键字，则创建失败的情况下不会报错。如果数据库已经存在，则什么也不做。示例如下：
```sql
-- 创建数据库testdb
CREATE DATABASE testdb;

-- 如果数据库不存在，则创建数据库testdb
CREATE DATABASE IF NOT EXISTS testdb;
```
## 六、选择数据库
切换当前的数据库连接可以使用USE command，语法如下：USE database_name;
如果指定的数据库不存在，则会产生一个错误。示例如下：
```sql
-- 使用数据库testdb
USE testdb;
```
## 七、删除数据库
删除数据库可以使用DROP DATABASE命令，语法如下：DROP DATABASE database_name;
如果指定了IF EXISTS关键字，并且指定的数据库不存在，则什么也不做。示例如下：
```sql
-- 删除数据库testdb
DROP DATABASE testdb;

-- 如果数据库存在，则删除数据库testdb
DROP DATABASE IF EXISTS testdb;
```
