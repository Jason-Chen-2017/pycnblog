## 3.3数学模型公式详细讲解

### 3.3.1Needleman-Wunsch算法

Needleman-Wunsch算法是一种用于序列对齐的动态规划算法。算法的核心思想是通过对序列之间的每个位置进行比较，找到最佳的对齐方案。Needleman-Wunsch算法的数学模型公式如下：

$$
S_{ij} = \begin{cases} 0, & \text{if } i = 0 \text{ or } j = 0 \\ S_{i-1,j-1} + \delta(a_i, b_j), & \text{otherwise} \end{cases}
$$

其中，$S_{ij}$ 是序列$a$和$b$之间在位置$(i, j)$的对齐得分，$\delta(a_i, b_j)$ 是在位置$(i, j)$处放置一个匹配得分的对齐得分，$i$和$j$分别是序列$a$和$b$的位置。

### 3.3.2最大公共子序列算法

最大公共子序列算法是一种用于基因组比较的动态规划算法。算法的核心思想是通过对两个序列之间的每个位置进行比较，找到最长的公共子序列。最大公共子序列算法的数学模型公式如下：

$$
S_{ij} = \begin{cases} 0, & \text{if } i = 0 \text{ or } j = 0 \\ S_{i-1,j-1} + 1, & \text{if } a_i = b_j \\ \max(S_{i-1,j}, S_{i,j-1}), & \text{otherwise} \end{cases}
$$

其中，$S_{ij}$ 是序列$a$和$b$之间在位置$(i, j)$的最长公共子序列的长度，$i$和$j$分别是序列$a$和$b$的位置。

### 3.3.3支持向量机算法

支持向量机算法是一种用于基因功能预测的机器学习算法。算法的核心思想是通过对训练数据集进行分类，找到一个最佳的分类超平面。支持向量机算法的数学模型公式如下：

$$
f(x) = \text{sgn}\left(\sum_{i=1}^n \alpha_i y_i K(x_i, x) + b\right)
$$

其中，$f(x)$ 是输入向量$x$的预测值，$\alpha_i$ 是支持向量的权重，$y_i$ 是训练数据集中的标签，$K(x_i, x)$ 是核函数，$b$ 是偏置项。

我们将使用Needleman-Wunsch算法来实现序列对齐。以下是一个简单的示例代码：

```python

from Bio.SubsMat import MatrixInfo as matrix

# 定义两个序列
seq1 = "ATGC"
seq2 = "ATCG"

# 使用Needleman-Wunsch算法进行序列对齐
alignment = pairwise2.align.globalds(seq1, seq2, matrix="blosum62")

# 打印对齐结果

    print(aln)
```

在这个示例中，我们使用了BioPython库的`pairwise2.align.globalds`函数来实现Needleman-Wunsch算法。这个函数接受两个序列和一个比较矩阵作为输入，并返回一个对齐结果。我们将对齐结果打印到控制台。
