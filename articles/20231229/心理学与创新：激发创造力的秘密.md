# 2. 核心概念与联系

## 2.1 创造力与创新

创造力是指个体在思考、解决问题、发明新东西等方面表现出的能力。创新是将创造力应用于实践中，实现新的产品、服务、技术等的过程。

创造力和创新是密切相关的，但它们之间存在一定的区别。创造力是个体心灵的能力，而创新则需要个体将这些能力应用于实践中。

## 2.2 心理学与创造力

心理学研究了如何激发个体的创造力。心理学家们发现，个体的创造力可以通过改变思维方式、提高自信心、增强积极心态等方式来提高。

心理学原理可以帮助我们理解创造力的形成和发展。例如，心理学家们发现，个体在解决问题时，会使用不同的思维方式。这些思维方式包括：

- 开放式思维：这是一种广阔的思维方式，个体会考虑多种可能性，不断发现新的想法和解决方案。
- 关键词思维：这是一种针对性的思维方式，个体会关注问题的关键点，从而更有针对性地寻找解决方案。

这些思维方式可以帮助个体激发其创造力，从而促进创新。

## 3.1 激发创造力的心理技巧

### 3.1.1 思维导图

思维导图是一种图形表示的方法，可以帮助个体系统化地组织和表达他们的想法。通过绘制思维导图，个体可以更好地理解问题，并发现新的解决方案。

具体操作步骤如下：

1. 在纸上或电子设备上绘制一个大方框，将问题或主题写在方框中。
2. 从主题中分出几个子主题，将它们连接到主题方框。
3. 对于每个子主题，再次分出更多的子主题，直到达到满意的层次结构。

### 3.1.2 脑巧克力

脑巧克力是一种创意思维技巧，可以帮助个体在解决问题时，更有效地发挥创造力。

具体操作步骤如下：

1. 将问题或主题写在纸上或电子设备上。
2. 思考5-6个与问题相关的关键词，将它们写在纸上或电子设备上。
3. 将这些关键词摆放在不同的位置，尝试不同的组合，以发现新的想法和解决方案。

## 3.2 提高自信心的方法

### 3.2.1 自我调整

自我调整是一种心理技巧，可以帮助个体提高自信心，并发挥创造力。

具体操作步骤如下：

1. 当遇到困难时，先停下来，深呼吸，放松身体。
2. 思考自己的优点，并将其列在纸上或电子设备上。
3. 思考自己在问题中的能力和资源，并将它们列在纸上或电子设备上。
4. 将自己的优点和资源结合起来，找到解决问题的方法。

### 3.2.2 自我激励

自我激励是一种动机增强技巧，可以帮助个体提高自信心，并发挥创造力。

具体操作步骤如下：

1. 设定明确的目标，并将其写在纸上或电子设备上。
2. 思考自己为达到目标所需的动力，并将它们列在纸上或电子设备上。
3. 每天回顾自己的进展，并给自己提供积极的反馈。
