# 2.核心概念与联系
代表性检验统计量是一种用于评估样本代表性的统计方法。在人口学研究中，我们通常需要从一定的人群中抽取出一个样本，以便对整个人群进行研究和分析。为了确保样本的代表性，我们需要对样本进行评估，以确定其是否能够代表整个人群。

代表性检验统计量主要包括以下几个方面：

1. 抽样误差：抽样误差是指样本与人群之间的差异。抽样误差可以通过增加样本规模来减小。

2. 抽样偏差：抽样偏差是指样本中的特定特征与人群中的特定特征之间的差异。抽样偏差可以通过确保样本来自于人群的各个部分来减小。

3. 可信区间：可信区间是一个包含真实参数的区间，通常用于表示样本的可信度。可信区间可以通过计算样本的置信度来得到。

4. 统计力度：统计力度是指样本数据与真实参数之间的关系。统计力度可以通过计算样本的统计力度来得到。
