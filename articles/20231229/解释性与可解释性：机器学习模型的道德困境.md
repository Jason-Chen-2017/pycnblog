```python
X = scaler.fit_transform(X)

# 训练模型
model = LinearRegression()
model.fit(X, y)

# 计算特征重要性
feature_importance = model.coef_
print("特征重要性:", feature_importance)

# 计算决策规则
decision_rule = model.predict(X)
print("决策规则:", decision_rule)

# 可视化表示
import matplotlib.pyplot as plt

# 绘制特征相关性矩阵
plt.matshow(np.corrcoef(X.T), fignum=1)
plt.colorbar()
plt.xlabel('Features')
plt.ylabel('Features')
plt.show()

# 绘制决策规则矩阵
plt.matshow(decision_rule.reshape(-1, 1), fignum=2)
plt.colorbar()
plt.xlabel('Samples')
plt.ylabel('Decision Rules')
plt.show()
```

在上面的代码中，我们首先生成了一组随机数据，并将其划分为特征（X）和目标变量（y）。然后，我们对特征进行标准化处理，并使用线性回归模型对其进行训练。接下来，我们计算了特征重要性以及决策规则。最后，我们使用可视化工具绘制了特征相关性矩阵和决策规则矩阵。
