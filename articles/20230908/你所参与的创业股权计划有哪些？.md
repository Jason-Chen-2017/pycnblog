# 1.简介
  

创业股权计划（CIO），又称投资基金，即商业银行或其他金融机构发起设立的一项股权投资计划，旨在为初创企业提供创收渠道，通过向受益企业提供股权，帮助企业摆脱财务困境、扩大规模、拓宽经营空间、实现成长。

2019年7月，中国政府颁布了《网络金融租赁管理办法》，可以将股份公司中的创业投资基金用于开展网络贷款业务。这标志着中国推出了第一批国际化的网络金融租赁平台。此前，仅有的投资基金主要集中在大型国企和百强企业中。近年来，越来越多的创业公司陆续入驻金融机构，也纷纷设立创业投资基金。然而，实际运行情况却并不乐观。各类创业基金都面临着如下痛点：
- 投资环境恶劣，投资经验和资源有限；
- 管理难度较高，基金运作耗时长、风险大；
- 法律风险和政策限制，容易遭到监管部门追究责任；

# 2.市场竞争格局及影响因素
2018年全球创业领域的经历激发了一场新的产业互联网革命，但同时也带来了全新产业体系——金融科技创新领域的崛起。过去几年，全球金融科技创新的发展态势迅速加速，创业者不断寻找新的机遇，打造独特的创新产品、服务或模式，如区块链、云计算、人工智能、虚拟现实等。其中，以区块链为代表的分布式数据库技术正在成为创业者们倾注热情关注的重点领域。从一定程度上来说，分布式数据库赋予创业者了一个更自由的创意空间，让他们能够利用其强大的计算能力、可靠性和安全保障，在数字货币、供应链金融、身份认证等诸多领域进行创新探索。

另一方面，由于行业及国家之间的金融结构、规则制定等诸多原因，仍有很多创业者将创业投资置于无形之中，更依赖金融机构、地产开发商等资源配置机构的支持。因此，如何发掘这些创业者潜藏的巨大价值以及有效利用其资源、促进其成长，成为当下急需解决的问题。


# 3.创业股权计划类型
## 3.1.知识产权激励型创业投资基金
知识产权激励型创业投资基金是指商业银行或其他金融机构针对创业者及其企业提供的一种有力激励机制。该基金由商业银行直接投资和指导企业创立商标、设计、开发等创新事宜，提供给创业者以激励、优惠或奖励等方式，目的在于创造价值。该基金的获利来源包括：
- 资本回报，包括企业募资或企业现金分红；
- 增值服务，包括由商业银行向创业者提供管理咨询、培训、工具包等提升创业者能力的服务；
- 补贴，包括由商业银行以补助的形式向创业者提供福利或费用；
- 分配股息，即商业银行根据创业者持股比例分派股息，也称孵化期权。


## 3.2.股份公司基金
股份公司基金是最普遍的创业投资基金。创业者通过向金融机构购买特定数量的股份，获得股东权益。每年，创业者可按照股权分红的方式获取收益，或将股份转让给其它创业者。该基金的优点是成本低廉，管理方便，风险控制灵活。


## 3.3.网络金融公司投资基金
网络金融公司投资基金的特点是利用互联网金融平台建立起全新的创业投资模式。该基金相对于传统的股份公司基金，具有更高的盈利空间。创业者购买网络资金，将获得其投资额的70%至90%的年收益率，而且也可以选择按季度分配收益。该基金还可提供专业人士的管理咨询、广告服务、专属顾问团队等辅助服务。


## 3.4.创业培训基金
创业培训基金是指商业银行、地产开发商或其它机构通过提供与创业相关的培训课程、讲座或活动，吸引创业者加入。该基金为创业者提供各种金融、商业、法律、IT等方面的培训，激发其创业能力，降低创业门槛，有助于创业者提升个人品牌及能力。


# 4.如何筹建创业投资基金
## 4.1.准备工作
1. 申报：如同一般的企业投资，创业投资基金也是需要提交申请的文件，包括基金信息表、协议、缴付协议等，须备齐材料。

2. 发放资本：创业投资基金的初始资本一般在5万元左右，当然也可以适当高一些。由于创业投资基金主要是为初创企业提供创收渠道，所以创业者要在申请该基金之前就已经具备产品或者服务的核心技术和成果，有自主的市场定位能力，并且在财务、人力、物流、技术、商务等方面也做足准备。这就要求创业者做好充足的准备工作。

3. 准备运作资金：目前最简单的途径是雇佣员工。但是如果创业者所在的区域缺乏金融机构的支持，就只能由其亲友来筹措资金，比如通过债券或者担保来筹资。另外，还有一些高校联合创始人基金、公益组织投资基金等。

4. 招募团队：企业基金通常都需要设计一个团队来管理基金运作，这个团队的组成人员往往需要有企业管理经验，有独立思维、善于把控、承担责任心。

5. 决策层：创业投资基金一般由基金管理人或者信托管理人管理，这是一个比较复杂的环节，涉及对基金的运作方式、财务状况、策略调整、股权结构等方面。建议基金管理人选取一名能胜任的经验丰富的资产管理人作为决策层成员，他们有独立的判断、决策权，可以制定出符合业绩预期的基金管理策略。


## 4.2.基金运作原则
1. 用爱分享：创业投资基金属于投资机构，需要尊重创业者的劳动成果和积极探索。应鼓励创业者将自己的项目、产品、理念等公开发表，希望通过宣传和传播营造良好的氛围，将产品带给更多的人。

2. 创意驱动：每个人的视野不同，创业投资基金会根据创业者的创意水平，设置不同的赚钱目标。如有需要，创业投资基卡可提供详细的策略和方案，帮助创业者制定正确的投资方向。

3. 没有规则：创业投资基金没有具体的规则，它可以根据各方面的需求，设置不同的操作权限、交易条款和风险管理措施。基金会根据创业者的需求不断优化和更新，以达到最佳的管理效果。

4. 留住用户：创业投资基金应确保投资项目的完整性和可持续性，不会因为缺少用户而被夭折。它可以提供详尽的退出条件和投资捐献安排，给予创业者最后一次购物的机会。

5. 反馈机制：创业投资基金都会为创业者提供成功的忠告，并积极收集和分析数据。这样可以帮助他们改善运作模式和策略，使得基金的投资收益最大化。


# 5.未来趋势
当前，各类创业投资基金已经成为一种时尚，创业者们正在利用这一趋势快速创业。随着区块链、虚拟现实等新兴技术的发展，及人的消费水平的提升，传统的商业模式已经无法满足快速创新的需求，创业者们更加青睐使用网络投资基金，寻找一种新的商业模式。

网络投资基金的模式是基于互联网模式，其特点是利用线上信息、数据、资源进行创新投资。通过创业者的创新产品、服务或模式，在网络平台上提供销售服务或线上交易服务。它可以在创业者之间快速地传递信息、产品和服务，并提供互通、支付、结算等服务。

除了新型的网络投资基金外，中国的著名商业银行也推出了许多创业投资基金，如银华世纪创业投资基金、万向钱龙创业投资基金、天弘创业投资基金等，它们都致力于打造新的网络创业生态。这些基金的产生，就是为了迎接创业者的时代到来，帮助创业者探索新的商业模式，开启全新的商业模式。
