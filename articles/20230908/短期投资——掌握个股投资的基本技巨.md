# 1.简介
  

短期投资，也称为短线投资或零散投资，是一种投资策略，它通常分为股票、债券和场内基金等市场上卖出的股份的短期资金配置方式。在交易中，投资者可以利用一种简单的方法进行交易，即用固定的金额投入一定数量的股份，并根据市场的变化情况适时止损或者加仓。这种策略无需专门的专业知识或技能，只需要个人的时间和耐心，就可以从事有效的投资活动。由于这种模式对个人投资能力要求不高，易于实践，所以成为广大投资新手的首选策略。本文将通过展示如何正确地选择投资标的、了解主要指标、掌握交易规则、构建投资模型，以及建立风险管理机制四方面，为大家提供更多的帮助。
# 2.投资要素
## 2.1 个股选择
做短期投资，首先就需要确定自己的目标，就是投资哪些股票。如果目标只是短期的获利而已，那么完全没有必要花大量的时间精力来研究和判断投资标的的价值长期收益水平。一般来说，大多数投资者都不会将所有的注意力都放在一个固定的股票或基金上，而是依据自己的特点、经验、兴趣等综合条件，选择一些看起来像样的股票，试着跟踪其走势和表现出色的公司。这样既可以快速赚取第一笔钱，也可以避免犯错。因此，投资者在确定投资标的时，应该结合自身的理财偏好、资产状况、对公司的认识、个人投资经验等综合因素，综合考虑这些因素，选择适合自己风险承受能力、短期效益要求、股东数量要求和投资回报率的优质公司。
