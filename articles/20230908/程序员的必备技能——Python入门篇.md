# 3.Python概述
## 3.1 Python发展历史
- 1991年 Guido van Rossum 为了打造一种新的脚本语言而创建了Python。
- 2000年版本发布，引入Unicode支持，成为Python 2.0。
- 2008年版本发布，改进了语法、增加了异步特性、增加了对Java类库的支持。
- 2010年正式宣布Python 3.0版本。
## 3.2 主要特征
### 3.2.1 简单性
- Python 是一种直观且易学的编程语言。你可以用很少的代码就能写出可运行的程序。它的设计宗旨就是让代码看起来像自然语言。Python 的语法结构清晰、一致，消除了歧义，并有较强的可读性。可以快速阅读和编写代码。
### 3.2.2 易于学习
- Python 拥有丰富的标准库和第三方库，可以通过 pip 命令安装第三方库。
- 有大量的文档资源，包括权威的 PEP（Python Enhancement Proposals）文件、教程和示例代码。
### 3.2.3 可移植性
- Python 可以编译成字节码，可以在许多不同的操作系统上运行，并且提供丰富的接口支持。
- 使用 Unicode 编码，支持多语言环境。
### 3.2.4 丰富的数据类型
- 支持动态类型，变量无需声明即可赋值。
- 支持列表、元组、字典、集合等数据结构。
- 提供对文件的直接操作，提供对数据库访问的支持。
### 3.2.5 函数式编程
- 支持匿名函数和装饰器。
- 支持迭代器模式，可以方便地实现生成器。
### 3.2.6 面向对象编程
- 支持基于类的面向对象编程。
- 通过描述符可以自定义类属性的行为。
## 3.3 编程范式
Python 支持多种编程范式，包括命令式编程、函数式编程、面向对象编程。
### 3.3.1 命令式编程
- 以过程化的方式思考问题，按照指令一步一步执行。
- 比如计算斐波那契数列的算法，是命令式编程的一个例子。
```python
def fib(n):
    if n == 0:
        return 0
    elif n == 1 or n == 2:
        return 1
    else:
        a = 1
        b = 1
        for i in range(3, n+1):
            c = a + b
            a = b
            b = c
        return b
```
### 3.3.2 函数式编程
- 以数学的形式抽象计算模型，并关注输入输出的映射关系。
- 比如利用 map 和 reduce 函数进行列表操作，是函数式编程的一个例子。
```python
from functools import reduce

my_list = [1, 2, 3, 4]
result = list(map(lambda x: x*x, my_list))
print(reduce(lambda x, y: x+y, result)) # Output: 30
```
### 3.3.3 面向对象编程
- 将现实世界中的实体（数据或行为）建模为对象，以类和继承的方式来组织代码。
- 比如定义一个 Person 类，包含姓名、年龄、住址等属性和行为，是面向对象编程的一个例子。
```python
class Person:
    def __init__(self, name, age, address):
        self.name = name
        self.age = age
        self.address = address
    
    def greet(self):
        print("Hello! My name is", self.name)
```
