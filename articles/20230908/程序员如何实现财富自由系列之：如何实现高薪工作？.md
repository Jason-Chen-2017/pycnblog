# 1.简介
  

高薪是现在每个程序员都追求的目标。但在实际生活中，面对企业给予的各种福利、激励和待遇，大家总是会担忧自己的薪资水平是否达标。但如果你能够真正实现高薪工作，并建立起自己独立的技能提升能力，那么这将是一段令人肯定的旅程。然而实现高薪工作，需要付出艰辛的学习和投入。不管是精读手册还是看书学习，掌握一定的编程基础知识都很重要，只有做到这一点，才能不断提升自己的能力。下面我就用实际例子，带领大家一起实现高薪工作。
# 2.职场生涯中的“996”天
996工作制是一种为抢时间，而进行高度工作的工作方式。通常认为，996工作制可以让员工长期保持生产力，但是实践证明，这种制度在现代社会已经难以实现。事实上，工作日9-19点一周工作，甚至连加班都不能开，导致效率严重下降。因此，很多人选择逃避、退缩，甚至选择停滞。究其原因，主要是因为996工作制会使得员工缺乏充足的时间学习和提升自己，从而导致沉迷于工作、不自觉地长期处于低水平状态。
如图所示，996工作制让员工无法专注于工作，只能被打扰，大量的时间花费在各种社交媒体上，工作效率低下。如何解决这个问题呢？这里有几种方案：

1、改变工作方式：996工作制虽然不能保证每天都工作到9点，但是可以在早上进行适当的调整，比如早起几分钟，这样可以避免早上效率低的问题。另外，也可以在午后回家休息一下，或者转移到晚上工作，这样就可以规避掉996工作的困扰。

2、提升个人能力：除了改变工作方式，我们还可以通过提升自己的能力，比如锻炼身体，不吃辣食品等，来提高自己的工作效率。提升个人能力的方法还有很多，比如阅读和写作，主动学习和刻意练习等。

3、找到合适的环境：有些人可能会因过分拘泥于工作导致精神疲惫，甚至失去理智。解决这个问题也很简单，只需找一个安静、宽松、有助于思维和精神健康的环境，就可以帮助他们获得专注力和成果。

4、适当减少行业竞争：在这个行业里，很多人都急功近利，追求着高薪和一切好处，这些行为无异于是在为竞争而斗争，而非自身职业生涯发展。建议多考虑员工的长远发展，以及员工的需求，找到一种平衡的方式。

# 3.怎么提升自己？
在实现高薪工作的道路上，我们要通过以下几个方面逐步走向成功：

1、编程能力强化：首先，我们需要拥有一定的编程能力，才能理解计算机科学的运行机制、解决问题的思路，最终产出高质量的代码。我们的第一步应该是系统性地学习计算机科学相关的课程和书籍，系统地巩固基础知识。我们可以尝试学习Python、Java、C++等编程语言，了解底层的原理，掌握数据结构、算法、操作系统等核心技术。同时，我们还要注意培养自己写代码的习惯，不仅要编写优雅的代码，更要有勇气、有担当地推敲别人的代码，尤其是那些你没有亲身经历的复杂代码。

2、职业方向的切换：提升编程能力还不够，我们还需要突破编程界限，以职业发展为导向，转型进入更有挑战性的行业。这个过程中，我们可能需要换个思路，重新定义自己的价值观，找到新的职业方向。比如，之前我一直在关注数据库领域，最近又接触到了区块链，这两个领域都是非常具有挑战性的，我认为最好的方式就是利用我的编程能力、业务理解和兴趣，深入浅出的探索新领域，提升个人能力，实现职业卓越。

3、培养技能提升能力：学习完基础知识之后，我们还需要提升自己的技能提升能力。这包括自学能力、演讲能力、团队协作能力、管理能力等。我推荐大家每年多花一些时间投入到自学或参加免费的线下活动，不仅可以学习到新的知识，还可以锻炼自己的创造力、分析能力和表达能力。同时，我们还需要结合工作，建立自己的技能库，不断学习新技能，提升自己的个人能力。

4、鼓励自己积极进取：既然实现了高薪工作的目标，就应该热血满满地投入工作，为此，我们还需要刻意培养个人的动机、主动性和毅力，坚持不懈地学习和思考，并乐于分享自己的成果。

综上，实现高薪工作的关键，不是抱怨，而是脚踏实地、自律自信，脚踏实地、不断输出，一步一个脚印地提升自己的能力。
