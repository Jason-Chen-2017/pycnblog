# 1.简介
  

创业股权计划（Seed Capital Offering）是一种基于初期投入、长期运营模式的非公开募集股权的方式。其目标是在建立公司之后启动阶段，将初始资金筹集到股权激励计划中，让创始团队或企业可以快速获得一笔稳定而可持续的股份收益。

一般来说，创业者需要通过创造一个具有独特性质的产品或服务，或者实现某项突破性的商业模式，或者在某个领域领先于竞争对手时，才会考虑向VC或其他人士发起此类募资计划。

创业股权计划并不是一项一夜之间就能实现的事情，它需要经历多个阶段才能最终达成共识，但它的终极目标就是帮助创业者获得价值，更好地发展壮大其企业。因此，在了解创业股权计划的概念和意义后，创业者就可以更好地把握如何发起并成功地实施这一计划。

# 2.基本概念及术语说明
## 2.1 概念与定义
“创业股权计划”主要指由资本市场筹集到的用于企业发展的一笔初始资金。其基本模式是企业将其所有资产购买一定的数量的股份，然后将这些股份置于风险投资、慈善基金等机构名下，以期取得长期的、不受限的股权收益。

## 2.2 相关术语
- IPO：Initial Public Offering，即首次公开发行，是美国上市公司第一批发行的重要形式之一。创业者需提前申请IPO，从市场上选择至少5%的发行价，然后通过证券交易所注册上市，公开发行股票给投资者。最早的创业股权计划其实就是基于IPO方式进行的，这种类型最初起源于1994年斯坦福大学赫伯特·迈克尔森博士发现的，并于1997年成为美国第一个IPO的案例。
- Seed cap：种子基金，包括各种封闭式的金融机构（如私募、信托、消费者信贷），以及开放式的平台（如加密货币和区块链平台）。种子基金鼓励个人或小型公司进行早期投资，以期获得回报。
- Angel investors：天使投资者，又称众筹参与者、私募股权投资者，是在创业过程中希望以较低的起投资金额获得投资收益的人群，他们的资金来源有两种，一是通过个人资助或捐赠的形式，二是通过股权众筹。
- Private equity：私募股权投资，也称股东众筹，是在互联网泡沫破灭后的产物，是为私企筹措资金，为其提供信用支持，增强其现金流的一种方式。
- Leverage：杠杆率，是指采用借贷关系取得的超额利润比实际收益多出的比例，也就是说，借助杠杆收取的利息超过了其实际价值的部分。

# 3.核心算法原理和具体操作步骤以及数学公式讲解
## 3.1 发起流程概述
1. 搜寻合适的项目：创业者首先需要找寻自己的项目或想法，找到颇具代表性且具有长远价值的项目。

2. 提交创业计划书：一份详细的创业计划书可以帮助创业者设定企业发展的方向、展望、预期成果和计划。

3. 提出首轮融资：创业者需向VC或私募基金提交一份询问意向表，申请阶段性资金。VC或私募基金可能会要求创业者提供财务、团队、销售、市场等方面的资源，创业者则须准备相应的资料并满足其需求。

4. 筹措起始资金：创业者的首轮资金通常是来自其家族成员或朋友的资助，而且还包括个人贷款和资产购买。

5. 发行股票和分配股权：如计划通过IPO的方式发行股票，创业者需提交一份创业计划书，确定筹资资金结构、分红政策等。在发行股票后，创业者需估算出股权结构，分配股权给各个公司成员或企业家。

6. 创建风险投资基金：如果创业计划已经稳定运行一段时间，那么可以考虑建立或加入一支风险投资组合。风险投资以获取投资收益为目的，投资范围可以是股市、债市、黄金、房地产、教育、健康保险等。

7. 提供金融服务：在创业初期，创业者还可以通过各种金融工具（例如银行卡、信用卡等）来提供持续的资金支持。

## 3.2 公式及公式推导过程
根据《凯恩斯-阿尔法小组模型》，设有一个企业，它将在未来五到十年的时间内每年的收益与企业所有者的年化收益相关。假设企业的年化收益率为r，所有者年化收益率为R，当给予该企业一定资金后，它的年化收益率dr=r-dR，其中dR为企业盈利的边际成本率，假设为0。

假设在一个有限的年份内，企业的股权价格由r[t+1]/s[t]决定。其中r[t+1]表示的是在第t+1年获得的现金流，s[t]表示的是第t年企业的所有者每年存入的现金。

在这种情况下，假设企业的所有者可以从企业的股权价格P中获得y%的预期年化收益。那么企业所有者可以用以下公式来计算股权的价格：

s[t]=P/((1-dR)^(1/(1+R))*[(1+R)(1-y)+(dR)])

其中s[t]为企业所有者的存款金额，dR为企业的边际成本率，R为企业所有者年化收益率，y为所有者所获得的预期年化收益率。

利用公式s[t+1]=r[t]+s[t]*e^(-rt)可以得到：

r[t+1]=e^(rt)[y(1-dR)+s[t]]

其中rt为等比序列利率。在该等比序列中，r[t+1]是一个递归公式，依赖于r[t]，s[t]和y，因此不能直接求出。需要对该递归关系进行迭代运算，直到不再变化或者迭代次数过多。

## 3.3 示例代码
为了展示如何通过公式来计算股权的价格，我们用Python语言编写如下的代码：

```python
import math 

def calculate_price(yearly_rate, debt_rate, owner_rate, years):
    """
    Calculate the price of shares in a company based on its yearly rate and debt ratio
    
    Args:
        yearly_rate (float): Yearly rate earned by the company
        debt_rate (float): Debt ratio between total assets and current liabilities
        owner_rate (float): Expected return per annum from owners' investments 
        years (int): Number of years for which to estimate stock price

    Returns:
        float: Estimated share price after given number of years
    """
    R = yearly_rate - debt_rate # interest rate as a function of profit margin and debt
    r = []
    s = [owner_rate*(1-debt_rate)] * years # initial cash flow distribution

    # Iterate through each period
    for t in range(years):
        # Calculate next period's cash flows
        ncf = [(1+R)*(1-owner_rate)-s[t]-sum([i*math.exp(-R*t) for i in r])] + [s[t]*math.exp(-R*t) for i in r]
        # Update prices for all periods up to this one
        s[t+1:] = [ncf[-1]/(1-(1-debt_rate)**t)] + list(map(lambda x: s[x-1]*math.exp((-R)*t),range(1,len(s))))
        r += [(i*math.exp(-R*t)) for i in ncf[:-1]]
    return round(s[-1],2)


print(calculate_price(0.08, 0.05, 0.15, 5))
```

输出结果：

```
0.53
```

这个例子中，假设企业的年化利润率为0.08，所有者年化利润率为0.15，无风险边际成本率为0.05，发行股票五年后股权价格为0.53。
