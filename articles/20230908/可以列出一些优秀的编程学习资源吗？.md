# 1.简介
  

编程是一个非常枯燥乏味且难以传授的技能。当你第一次接触编程时，就会发现自己头脑里似乎只有各种名词和命令，却不知道该如何下手。面对一堆命令、语句和表达式时，真正的进步和提高并不是那么容易。所以，在学习编程之前，一定要先了解以下三个基础知识点：

① 命令行终端（Command-Line Interface）：通过命令行终端我们可以直接操作计算机硬件设备，执行系统命令、运行应用程序等。熟练掌握命令行终端操作方法可以让我们更加高效地进行编程工作。

② 编程语言（Programming Languages）：编程语言是指编译型编程语言或解释性编程语言。编译型编程语言通常需要经过编译后才能运行，而解释型编程语言不需要编译直接就可以运行。了解编程语言的种类和特点可以帮助我们更好地理解编程，并选择合适的编程语言进行应用开发。

③ 文本编辑器（Text Editors）：使用文本编辑器编写程序代码是编程的一大关键环节。熟悉常用的文本编辑器如Sublime Text、Vim、Atom等，能够极大的提高我们的编码速度。

# 2.教程
如果你没有编程基础或者是刚入门，建议你从以下地方找起：

① Code Academy：Code Academy提供初级、中级和高级编程课程，包括Python、JavaScript、Ruby和Java等语言。这些课程易于上手，而且有丰富的互动问答形式，可以帮助您快速解决问题。

② Learn Python the Hard Way：这是一本详细的Python编程教程，适合没有任何编程经验的新手。这本书的结构清晰，每章都分成多个小节，可以很好的帮你理解并实践Python相关的概念。

③ Treehouse：Treehouse是一个免费的在线编程学习网站，其主要收录了计算机科学、Web开发、数据库、设计、经济学、数据分析和物联网方面的内容。

④ Tutorialspoint：Tutorialspoint是一个老牌的IT技术教程网站，它提供免费的编程入门、提升技术能力、交流心得和项目经验等服务。

⑤ Freecodecamp：Freecodecamp是一个提供前端、后端、移动开发等全栈开发的学习平台，它也提供了大量的免费视频教程和编程挑战题目。

⑥ Mozilla Developer Network (MDN)：Mozilla Developer Network (MDN)是一个专注于为开发者提供网络技术文档和资源的网站，其中包含了非常齐全的文档资源。它还有较好的在线代码编辑工具，可以帮助你快速理解语法和用法。

# 3.资源分享
除了官方的教材和网站外，这里还可以推荐一些开源项目、博客和资源。这些资源涵盖了很多编程领域的最新动态和优质信息，可以帮助你快速成长。

① GitHub：GitHub是一个著名的开源代码托管平台，拥有庞大而活跃的社区生态系统。GitHub上有大量的开源项目供你参考，其中大多数都是由各个领域的专家精心制作的。

② StackOverflow：StackOverflow是一个基于Q&A的论坛，这里有着庞大且活跃的用户群体，通过自我回答和帮助他人解答问题，可以帮助你快速解决遇到的问题。

③ Medium：Medium是一个由多位读者和作者组成的社区平台，你可以在上面发布你的创意故事、技术文章、职场感悟、个人生活感受等，为自己的职业发展提供更多的帮助和指导。

④ hackernoon：hackernoon是一个免费的原创技术教程网站，这里有着海量的优质技术文章，你可以在里面找到很多有价值的信息。

⑤ codingame：codingame是一个在线游戏开发者社区，这里有着丰富的游戏题目，你可以在上面锻炼你的编程能力。

⑥ Coursera：Coursera是一个专注于提供优质教育资源的网站，这里有众多的免费课程，你可以选择适合自己的方向，获得最新的知识技能。

# 4.其他建议
除了上面推荐的一些学习资源之外，这里还可以推荐一些其他的方法。
