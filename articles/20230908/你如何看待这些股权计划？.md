# 1.简介
  

股权激励是一个非常古老且重要的经济学概念，它源于人类寻求追随自己的野心并从中获得报酬的本能动力。股权激励体系包括投资、奖金制度、保险、债券、期货市场等多种机制，为个人提供更高收入和物质利益。在股权激励的过程中，股东会受到股权持有者的广泛认可，能够获取更大的份额分红，也会得到法律上的保障，即使因违反保密协议而丧失部分价值也不会影响他们的股权。
股权激励机制可以让个人获得社会支持，但同时也可能会带来很多问题。首先，股权激励制度下，市场对个人的敌视和排斥越来越强烈，比如贪婪的长期投资者，会利用股权激励机制将更多的钱投入到容易受到伤害的行业或项目上，导致更多的损失；其次，股权激励制度存在隐形的风险，因为参与者需要承担过多风险，比如被人盯梢、背叛或者欠薪、职业道德败坏等等。因此，如果没有足够的政策支持和管理，股权激励机制可能会遇到很多麻烦。

在现代经济社会，人们越来越倾向于选择保守的生活方式，而不是依赖股权激励来获得经济利益。因此，人们逐渐转向以非营利组织的方式来实现自己的财富积累和社会责任。然而，无论是在推动经济发展方面还是维护公共利益方面，非营利组织都面临着巨大的挑战和困难。非营利组织的创立者往往需要兼顾财政能力、物质资本和精神资源等多种资源，无法保证每个人的投入都能产生好的效果。而且，由于成员的差异性太大，非营利组织的运作往往比较不规范，甚至会出现腐败现象。因此，虽然非营利组织很有可能成为中国未来的金融基础设施之一，但它们目前还处于起步阶段，尚处于弱势地位。

因此，如何理解并有效的运用股权激励机制，成为决定性的发展策略，对于构建更加公平、包容和健康的经济社会至关重要。为此，《3.你如何看待这些股权计划？》试图通过对一些著名的股权激励机制进行阐述，以及它们所面临的种种挑战与机遇，提出一些具体的应对建议。

# 2.定义及相关概念
## 2.1 股权激励的定义
股权激励是指由股权持有者将股权卖给其他股东所取得的回报，它既包含企业资产收益的收益，也包含社会福利及投资回报。

## 2.2 股权激励机制的分类
股权激励机制一般分为公开募集和私募集两种形式，具体细节如下：
- 公开募集：主要依靠个人捐赠和信托来筹集资金，有些公司还设立“员工持股基金”。这种形式通常用于较小的企业，也可用于受到捐赠影响的企业。
- 私募集：主要依靠私募股权基金来筹集资金，其目的也是为了企业资产的增值，但通常比公开募集更严格。私募集发起者通常要求独立核算，并按季度出资，以避免经理人对基金的过度投入或投机行为。私募集体系一般采用独家模式，只有自己拥有的股权才能加入该体系，因此容易受到竞争者或管理层的控制。

## 2.3 股权激励机制的衡量标准
股权激励机制通常以分红率（Dividend Yield）作为衡量指标，表示每股股利占其股息总额的百分比。分红率低于某个特定水平（如7%），则有可能意味着投资者适得其反，不仅失去了收益，还可能会损失金钱。因此，衡量股权激励机制的指标应该以实际收益为主，而非某种平均效益。例如，私募集激励机制的分红率可能高于公开募集激励机制，但如果将私募集资金与公开募集资金相加，仍有可能超过公开募集的收益水平。

# 3.知识产权激励机制（知识产权激励法）
## 3.1 什么是知识产权激励？
知识产权激励是指人们为了创造和传播其创新产品、设计理念或服务而给予他人知识产权保护，并给予相应的激励机制。

知识产权激励最早出现在美国，其前身是麻省理工学院经济学教授莫斯科大学费尔法拉邦学派的团队，他们提出的“专利制度”便是一种基于激励的知识产权保护方式。知识产权保护是一种建立在非竞争性条约基础上的机制，目的是保护创作者的成果免遭侵犯。

知识产权激励法是中国颁布的第一部关于知识产权的法律。《中华人民共和国知识产权法》(下称《知识产权法》) 第十五条规定，“国家对学术研究、文化创意、商业秘密和专利权的保护应当充分发挥政府作用，实行科技进步驱动发展的作用。”“国家鼓励创新，引导创新创业，创造更多的财富。”“鼓励实验室开发、企业投资创新、出版社出版、个人创新创造力培养等，促进科技和文化的创新发展。”

## 3.2 如何使用知识产权激励？
知识产权激励机制的关键是创造者对他的创造物拥有专属的所有权。因此，只要创造者遵守知识产权法，并按照法律的规定向创造者支付专利使用费，那么任何第三方均可以自由使用其创造的成果。

例如，假设A公司想要创建一个新的装置或产品，他可以申请专利并取得专利授权。此后，B公司就可以使用A公司的专利，无需支付额外的费用。这时，B公司就拥有了A公司的产权。如果另一个创造者C想要复制A公司的专利，那么就要向A公司支付额外的专利使用费。

知识产权激励机制旨在通过激励创造者创新、改进自己的工作，帮助创造者摆脱信息垄断的局面，创造出更好的作品和服务，增加创作者的收入和经济声誉。激励机制可以从以下几个方面体现出来：

1. 补偿制度：知识产权使用费是激励创作者创新创造的直接结果。国家根据创作者创造的价值的大小，设置相应的补偿额度，作为激励机制的一部分。当第三方使用了创作者的成果，若双方发生纠纷，可向创作者索取补偿费用。这样，创作者将获益，反而促使创作者继续创造。

2. 授权制度：知识产权法规定，除法律明确规定的情形外，创作者不得以营利为目的扩张其知识产权。因此，授权制度对于创作者来说是一个不可或缺的激励手段。创作者在申请专利时可以授予授权证书，允许他人使用其专利。授权证书除了限制第三方使用范围外，还可以赋予第三方优先权、接受超出普通使用许可的特殊使用权。

3. 撤销制度：在知识产权法规定的许可期内，创作者可以撤销其专利。如果法律禁止撤销，那么创作者只能等待专利无效或宣告破产。但是，有些专利在使用过程中可能存在问题，比如侵权、错字等。为了弥补这一缺陷，有些创作者可以在撤销之前向知识产权局提交异议，请求恢复被侵权的内容。知识产权局需要通过调查之后，核实真相之后再做决定。

4. 激励创作者创新创造：激励创作者创新创造有两个层次。一是奖励创作者对于自己的专利的积极贡献。二是惩罚不尊重创作者成果的人。如，在上市公司上市之后，上市公司可向创建公司的创始人或董事们发放奖金。此外，对于那些滥用专利的行为，上市公司可采取补偿措施。惩罚措施还可以包括裁掉上市公司的董事、监管部门、质量管理部门等。

## 3.3 知识产权激励有哪些典型案例？
近年来，知识产权激励的研究多数集中在案例研究上。在中国，应用知识产权激励最突出的案例就是郭台铭的故事。郭台铭生前曾在世，其夫人张惠妹在申请专利时，先后两次受阻，还退避三舍。郭台铭最终获得专利，并不得不继续留在京城。

李树菲是苏州一家服装公司的副总裁。2013年，她发现自己家居的电器频繁报废，担心由于它们违反了国家的知识产权法规，致使资产损失，所以不敢乱买。在网上找到了一个旧货箱，她购买了一批类似的产品，希望能解决问题。但旧货箱里面有很多遗留物，她觉得不合适，把它们全都扔掉。但她又担心这些旧货箱里的货物会在别人拿走之后造成过多的不良影响，不想向别人借来，所以她又退守了。

这两个例子说明了知识产权激励的威力。前者说明，知识产权激励可以带来商业环境的改变。后者说明，知识产权激励具有调节创作者行为、塑造公众形象的作用。
