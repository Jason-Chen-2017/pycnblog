2.字符串概述
Python 中的字符串是一个非常灵活的数据类型，它可以存储各种字符序列。其特点是由零个或多个字符组成，且所有字符都属于 ASCII 或 Unicode 字符集中的一个字符。

Python 中的字符串是不可变序列（不可变指的是字符串内容不能被改变），这意味着每当创建一个字符串时，系统都会给它分配固定大小的内存空间用于存放字符序列。当对字符串进行操作时，系统不会重新分配内存空间，而只是在原有的字符串上执行操作，然后返回新的字符串作为结果。这种方式可以有效节省内存开销，尤其是在需要大量重复创建相同字符串的情况下。

虽然 Python 中的字符串类型相较于其他编程语言更为灵活，但实际上仍然遵循着某些共同的特征，包括：

1. 单引号（'）和双引号（"）都可以用来表示字符串；

2. 使用反斜杠（\）转义特殊字符，使得字符串中的单引号、双引号、反斜杠可以被正确显示；

3. Python 中的字符串是按照Unicode编码方案来存储的，这意味着一个字符可能占据多个字节的内存空间。

4. Python 中的字符串没有大小限制，可以包含任意数量的字符，直到计算机的可用内存为止。

除了以上几条规则外，字符串还有以下几个特点：

1. 支持数字索引，可以通过索引来访问字符串中每个字符；

2. 连接运算符（+）可以用来将两个字符串组合成一个新的字符串；

3. * 操作符也可以用来重复字符串多次。

下面，让我们看一下 Python 中的字符串类型。

3.字符串的创建与操作
Python 中的字符串创建并不复杂，可以使用单引号或双引号括起来的任何序列字符来创建字符串。例如："hello world!" 可以用两种方式来创建字符串变量:

```python
str1 = "hello world!"    # 用双引号创建字符串
str2 = 'this is a test string.'   # 用单引号创建字符串
```

注意：如果要在字符串中嵌入引号，那么就需要用反斜杠（\）来转义。

字符串的长度可以通过内置函数 len() 来获取：

```python
len(str1)     # 获取字符串的长度
```

此外，字符串还可以用 + 运算符连接或者复制。

```python
str3 = str1 + ", goodbye."      # 将 str1 后面追加内容 ", goodbye." 
str4 = str3*3       # 将 str3 重复 3 次得到 str4
```

字符串的索引从 0 开始，可以用方括号 [ ] 来取出指定位置的字符。下标从左往右计算，第一个字符的下标为 0，依次类推。

```python
print(str1[0])        # 打印字符串第 1 个字符 h
print(str1[-1])       # 打印字符串最后一个字符!
print(str1[0:5])      # 打印字符串前 5 个字符 hello
```

字符串也可以被修改、截取、删除等操作。但是，由于字符串是不可变的，因此这些操作都会返回一个新的字符串。

```python
new_str1 = str1[:5] + "Python"         # 从头截取 5 个字符，然后添加 "Python"
new_str2 = new_str1.replace('o', '*')   # 替换字符串中的 o 为 *
new_str3 = sorted(set(str1))            # 按字符顺序排列去重后的新字符串
```

最后，字符串的遍历可以使用 for 循环来完成，也可以通过内置函数 iter() 将字符串转换为可迭代对象，然后利用 next() 函数或 in 关键字判断是否存在某个字符。

```python
for char in str1:
    print(char, end='')
    
if 'w' in str1:
    print('\nFound the character w!')
```

4.字符串的格式化
除了普通的字符串操作，Python 中还提供了字符串格式化的机制，可以根据指定的格式生成带有特定含义的字符串。这里以 %s 表示字符串占位符，%d 表示整数占位符，%f 表示浮点数占位符。格式化的语法如下所示：

```python
string % (variables...)
```

其中 variables 可以是任意类型的变量，包括数字、字符串甚至是另一个字符串。举例如下：

```python
age = 29
name = "Alice"
message = "%s was %d years old when she wrote this book." % (name, age)
print(message)    # Output: Alice was 29 years old when she wrote this book.
```

此外，还可以通过 format() 方法来实现格式化。

```python
value = 3.1415926
message = "{:.2f} is pi".format(value)
print(message)    # Output: 3.14 is pi
```

这样可以实现精确到小数点后两位的浮点数输出。

5.字符串编码
对于字符串，由于 ASCII 编码只定义了 127 个字符，因此无法表达一些非英文字母字符。为了能够处理更多种类的字符，Python 提供了更高级别的编码方案，包括 UTF-8、UTF-16、GBK、GB2312、ASCII、Latin-1 等。通常情况下，在创建字符串时自动选择合适的编码形式即可。

例如，中文字符"中文"可以用 UTF-8 编码：

```python
chinese = "中文"
encoded_chinese = chinese.encode("utf-8")
decoded_chinese = encoded_chinese.decode("utf-8")
print(encoded_chinese)     # b'\xe4\xb8\xad\xe6\x96\x87'
print(decoded_chinese)     # 中文
```

上面的例子中，encode() 方法可以将字符串编码成指定编码的 bytes 对象，decode() 方法则可以将 bytes 对象解码成字符串。

同时，还可以在文件读取、写入过程中指定编码。
