# 1.简介
  

## 1.什么是Serverless计算？
Serverless（无服务器）计算，是一种按需付费的计算服务。它意味着开发者不需要管理或维护服务器，只需要提供应用程序的代码并配置其运行环境即可。云服务商如AWS、Azure等提供公共云平台上的Serverless计算服务。通过这种方式，开发者可以专注于业务逻辑的开发，而不需要关心底层的基础设施建设、网络连接及服务器运维。当应用程序的访问量或者请求量增加时，Serverless计算服务会自动分配更多的资源给应用，从而降低成本并提升性能。
## 2.为什么要用Serverless计算？
- **降低成本**：云服务商根据使用情况计费，无需担心服务器运行，节省成本；
- **弹性伸缩**：按需伸缩资源，根据应用流量自动扩容；
- **更高效率**：不再需要考虑服务器的硬件、软件、网络等资源，从而降低了时间、经验及成本；
- **更快响应**：无服务器计算使应用快速响应请求，保证服务的可用性；
- **更便捷开发**：开发人员不需要关注基础设施，只需要编写业务逻辑代码即可。
以上这些好处都体现出了Serverless计算带来的各种优势。所以，很多公司选择基于Serverless计算平台搭建应用，甚至成为新的竞争对手。
## 3.什么是AWS Lambda?
Lambda是AWS提供的一项无服务器计算服务，允许开发者创建服务并在事件发生时执行代码。它的特点包括：
- 按使用量付费：客户只需为使用的每千万次运行付费；
- 使用任何编程语言编写代码：支持多种编程语言，如Java、Node.js、Python等；
- 可选的服务级别协议SLA：保证99.9%的可用性；
- 支持任何规模的应用程序：从微型函数到大型服务器集群，可轻松应对各种复杂场景；
- 内置日志记录功能：在控制台查看执行日志。
## 4.什么是API Gateway?
API Gateway（即网关）是一个托管服务，它可以帮助开发者发布、监控、保护和加速应用的API。通过使用API Gateway，开发者可以轻松地集成各种类型的后端服务，包括移动应用、网站、实时数据源、机器学习模型等。API Gateway可以帮助将HTTP RESTful API转换为HTTPS、JSON或XML等多种格式，还可以利用缓存、授权、监控、自定义域等功能增强API的能力。
