# 4.背景介绍
## 什么是Django？
Django（读音/dʒæŋɡoʊ/，又名“ジャンゴ”）是使用Python编写的一个开放源代码的Web框架，由基于模型–视图–模板（MVT）设计模式而成，由谷歌于2005年7月创建。Django名称取自开发该Web框架期间在俄勒冈州的公牛排名第一的贝克汉姆（Berkeley Cooperative）。此外，它还意味着“杰克逊迪哥”，恰巧也是另一种烤鸡肉。因此，Django可以说是一个全球知名的中文烤鸡。由于基于Python的特性和语法，Django也称为“瑞士甜点”。

Django框架旨在消除Web开发中的复杂性。通过提供一系列功能完善的模块，Django可以帮助用户快速地开发出功能丰富的网站，使得Web开发变得简单、快速和高效。该框架具有强大的ORM（对象关系映射），允许开发者直接映射数据库到程序代码，有效地简化了数据访问；支持多种模板引擎，适用于不同的用户界面设计风格；提供了强大的URL路由机制，可以实现动态网址生成，支持AJAX，使得网站的交互性得到提升；还有其他很多功能特性，比如权限控制、缓存机制、本地化支持等，都能极大地加快开发速度。

## 为什么要使用Django？
### 1.易上手
Django框架易上手，这是Django最大的优点。因为它的核心概念十分简单、容易理解，甚至几乎不用写代码就可以完成基本的功能。这种特性对初学者来说尤其方便，让他们能够更快地上手Django框架。另外，Django提供了详细的文档，帮助初学者快速入门。

### 2.开发效率高
Django框架开发效率高。由于Django框架遵循MVC（模型-视图-控制器）设计模式，因此它可以帮助开发者以一种清晰直观的方式组织代码。它内置了一整套常用的工具集，可以极大地提高开发效率。而且，Django还提供了一个自动生成管理员后台的工具，可以方便地管理站点内容。

### 3.灵活可扩展性强
Django框架具有灵活性和可扩展性强。Django可以使用第三方插件扩展其功能。其中，Django CMS（内容管理系统）就是使用Django开发的一个开源CMS系统，有很多第三方插件，可以满足不同类型的网站需求。另外，Django框架的ORM（对象关系映射）功能可以非常容易地映射关系型数据库到程序代码，不需要编写SQL语句。

### 4.免费、开源
Django框架是免费的，并且提供源代码。开源社区贡献者无限期地帮助维护这个框架，对开发者来说也是一种鼓励和激励。
