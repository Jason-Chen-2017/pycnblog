# 2.基本概念术语说明
## 概念篇
### Apple ID
Apple ID 是苹果提供的个人身份标识符（PII）。注册 Apple ID 可以获得 iPhone、iPad、Apple Watch、MacBook、iTunes Music Store 的在线服务。每位购买 Apple 的客户都必须先注册一个 Apple ID。虽然 Apple 在隐私方面采取了保护措施，但个人信息仍可能在第三方处泄露。

### Apple Arcade
Apple Arcade 是苹果公司推出的基于网络的免费游戏，包含了一些经典的街机类游戏。用户可以通过在苹果商店中下载游戏、登录社交媒体账户、关注别人的 Twitter 或 Facebook 页面获取奖励或积分，也可以与朋友互动玩耍。

### Apple Business Manager
Apple Business Manager 是一套管理工具，可以帮助企业管理 Apple 服务，包括 iOS、iPadOS、macOS、Apple Watch、Apple TV、iMessage、Apple Pay、App Store、Apple Configurator 等等。Apple Business Manager 可帮助企业收集数据、管理人员和设备、制定营销策略、优化生产力、提升效率、增加收益。同时，Apple Business Manager 提供了一系列 API，开发者可利用这些 API 为自己的业务应用定制化功能。

### Apple Card
Apple Card 是一款为消费者提供商品券或抵用卷的美国个人理财产品，具有低交易成本、免费购物、透明价格、可兑换商品优惠等特色。使用 Apple Card 用户无需开通银行账户，只要有信用卡即可享受美元、欧元、日元等货币的购物券优惠。

### AppleCare+
AppleCare+ 是一项升级计划，为所有 Apple 产品提供 1 年的保修期。该计划最早于 2017 年 6 月份推出，目的是确保消费者拥有 Apple 产品的长久可用性，更好地保障他们的消费权利。在保修期间，苹果将继续为用户提供技术支持和售后服务，并保持产品的最新状态。

### App Store
App Store 是苹果公司提供给开发者上传应用程序的网站，应用程序可以帮助用户在手机、平板电脑、Apple Watch 上使用功能，比如浏览照片、阅读新闻、播放音乐等。开发者需要向苹果提交软件开发者证书申请审核通过才能上架，上架之后就可以在 App Store 中购买和安装。

### iCloud
iCloud 是苹果公司推出的基于云端的数据存储和同步服务，用户可以在 iCloud 设备之间共享文件、照片、联系人、Calendars、Reminders 等数据。iCloud 不仅能够自动备份数据，还能在不同设备之间同步数据，让用户访问数据更加方便。另外，iCloud 还提供了安全云硬盘和数字证书存储，能够帮助用户存储个人数据。

## 技术术语
### APS (Application Programming Interface)
APS 是应用程序编程接口（API），它定义了应用组件之间通信的方法，主要用于支持分布式计算和网络应用的开发。

### Bitcode
Bitcode 是一种中间代码形式，它是编译为二进制码的应用的中间语言，可直接运行在 Apple 的 iOS 和 tvOS 设备上。当修改代码时，不需要重新构建整个应用，只需重新编译打包 Bitcode 文件，就可以直接运行应用。

### Core ML
Core ML （Core Machine Learning）是 Apple 提供的一个机器学习框架，能够帮助应用更快、更准确地识别图像、文本、声音、视频等不同类型的数据。开发者可以使用 Core ML 来训练机器学习模型，并集成到应用中，实现实时的预测功能。

### Metal Performance Shaders
Metal Performance Shaders 是 Apple 提供的一个高性能图形渲染引擎，由 C++ 编写，可帮助开发者创建自定义的高性能图形渲染效果。Metal Performance Shaders 使用 Shader 语言来描述渲染逻辑，并通过硬件加速处理来达到更高的帧率和流畅的渲染效果。

### Natural Language Framework
Natural Language Framework 是苹果公司推出的基于自然语言处理（NLP）的框架，能够帮助开发者提升 NLP 应用的性能和准确性。NLP 应用包括内容理解、情绪分析、聊天机器人、自动翻译、语音助手等。

### Xcode Release Notes
Xcode Release Notes 是 Xcode 软件的一项功能，用于显示每个 Xcode 更新的相关信息。用户可以根据需要查看新增功能、改进功能、已知问题和 bug 修复等详细信息，帮助他们及时了解软件的最新动态。
