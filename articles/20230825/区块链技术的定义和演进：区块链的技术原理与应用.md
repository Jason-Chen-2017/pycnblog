# 1.简介
  

随着社会的发展，互联网技术的飞速发展以及计算机硬件性能的提升，已经形成了海量数据信息积累，而这些数据的价值却无法被充分利用。而区块链技术就是为了解决这一问题而诞生的，它通过对数据进行加密、存储、数字签名等方式，实现数据的安全、不可篡改、全球化共享的目标。由于其分布式的特性，区块链技术将会给当今世界带来前所未有的经济效益和广阔的商业机会。

2017年9月，由IBM主导的 Hyperledger Project 开始了区块链领域的技术革命，提供了一种去中心化的、可信任的数据共享机制。这个项目提供了一个分布式网络平台——Hyperledger Fabric，运行在云端，能够容纳各种类型的区块链应用程序（如身份认证、记账等）。

区块链技术是一种分布式的数据库技术，使用密码学的原理来确保数据传输过程中的完整性、私密性、不可否认性以及不可抵赖性，从而实现“不同参与者之间共享数据”的需求。

区块链作为一种新的技术，它对人类文明产生的影响甚至超越了历史上任何技术。将来还将有更多的人类参与到这场伟大的变革中来。但同时，也有一些限制性因素在起作用，如法律、监管、效率、可靠性等等。因此，只有创造出更加安全、透明、可验证的区块链技术，才能真正的改变我们的生活。



# 2.区块链技术的基本概念与术语
## 1.什么是区块链？

区块链（Blockchain）是分布式数据库，具有自我修复能力、存储、共享数据、防篡改、不可伪造等特点。目前最常见的用途是数字货币、权益证明和防伪溯源等。

区块链作为一种新型的分布式计算技术，其本质上是一个共享数据库，每一个节点都维护着整个数据库的副本，并负责维护链上数据的一致性。节点之间采用各自验证的方式来确认数据合法性，每个节点只能接受其他节点的有效背书，节点可以根据自己的需要加入或退出网络。

区块链具有下列的特征：

1. 分布式共享数据库：区块链共识协议采用Paxos算法，所有的节点在同一时间保持同样的链状态，并通过不断试错来达成共识。区块链数据库也是分布式的，所有节点都保存了一份完整的数据库，节点之间的通信、数据传输都依赖于去中心化网络。

2. 数据不可篡改、不可伪造：区块链中的数据都是经过加密、签名等方式生成的，而且无法修改，数据无法伪造。

3. 隐私保护：区块链可以通过密钥管理系统来保障用户数据的隐私。

4. 防止双花攻击：在区块链上，如果两个不同的交易申请同样的金额或资产，则不会出现同时被打包到区块中的情况，保证交易的唯一性。

5. 智能合约：通过智能合约，可以对区块链上的操作进行自动化。

6. 可追溯性：区块链记录了所有数据的变更记录，使得任何时候都可以查到某个历史版本的数据。

7. 便捷支付：区块链可以让用户之间快速的进行资金转移，而且不受中间商的控制。



## 2.区块链的基本术语

### 1.节点（Node）

区块链的节点即网络上的计算设备，在区块链网络中扮演着重要角色。区块链网络中的每个节点都保存着本地的区块链副本，可以执行各种区块链相关的操作，比如发布交易、查询交易信息等。

### 2.区块（Block）

区块链中的数据存储在称之为区块的结构中，每个区块存储了一定数量的数据，相邻的两个区块之间可以通过哈希指针串接起来。区块链中的数据是不可更改的，一旦写入区块链就无法删除。

### 3.交易（Transaction）

交易是指发生在区块链上的一系列操作。每个交易都是一个命令，用于更新区块链中的状态。交易可以是新建账户、转账等，也可以是执行智能合约。

### 4.区块链（Blockchain）

区块链是由一组节点通过共识协议构建而成的，所有节点之间保持着高度一致的链状结构。链状结构中，任意一个区块的状态都会被记录下来，并由后续的区块继承。区块链中的数据是公开透明的，任何人都可以访问到。
