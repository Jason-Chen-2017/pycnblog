# 1.简介
  

近几年随着中国国内生产总值连续增长，中国的GDP占全球经济体中份额的比例也越来越高。根据IMF发布的数据显示，截至2019年，中国领先美国、日本和欧洲地区，并占据全球GDP的67.7%，而相对于美国和欧元国际地位的下降了12%-25%。中国成为世界上最大的GDP国家之一，不断扩大的GDP是经济体制改革和开放带来的必然结果，但是另一方面也伴随着通货膨胀率上升，经济增速放缓，国家财政收入缺口加大，本身又处于弱势经济体系状态，所以需要政府主导的货币政策从多角度进行布局，以支持实体经济的健康发展。  

传统的货币政策是依赖国库券和利率调节的，但这样的政策在经济转轨期和政局动荡时段不一定能够有效应对，更不利于促进实体经济的复苏。同时，为了解决央行持续贬值的风险，一些国家或地区会采取其他货币政策，如宽松货币政策、弹性货币供给、扩大反债务约束力度等，但由于缺乏透明度和监管机构配合，其效率不高，往往导致货币紧缩、泡沫化甚至危害金融秩序。因此，如何打造人民币资本市场的双边机制，是建立透明、规范、可信任、有效运作的关键环节，也是维护市场平稳运行的前提。

本文试图通过系统性、科学性、长远视野的方式，为读者揭示人民币资本市场的发展模式及其机遇与挑战，以及当下的双边货币政策形势的发展趋势和措施方向。
# 2.人民币资本市场概述
## 2.1 什么是人民币资本市场？
首先，什么是“人民币资本市场”（CNY market）呢？简而言之，就是用人民币购买商品的市场，或者说是在人民币市场上交易黄金、外汇、房地产、原油、石油、天然气、铜矿石等各种资本类商品的市场。

所谓“人民币资本市场”，其实就是中国境内各种资本类商品的交易市场，包括黄金、外汇、房地产、原油、石油、天然气、铜矿石等各种资本类商品。换句话说，即使一个普通的居民，也可以通过银行购买到这类商品的金融衔接，这些商品的价格都由人民币确定，也就是说，居民们通过人民币购买中国境内各种资本类商品，这些商品的价格都是人民币支付的，而不是由各国或地区外汇、美元、英镑等支付。

实际上，人民币资本市场是一个整体，包括了中国境内的各种金融领域，包括证券、保险、基金、股票、期货、债券等市场，以及商业、金融服务业等产业领域。这里的各个市场之间彼此联系紧密，比如股票、期货市场的资金流向，既受到资本流入的影响，也要受到资本流出的影响。比如，某只股票如果停牌，那么那些持有该股票的人就只能抛售股票获取现金。同样，当需求增加的时候，交易者可以选择增加资金进场，也可以选择减少资金出场。

一般情况下，人民币资本市场的活动主要是价差交易，即投资者卖出一种产品，买入一种更贵的产品，这种交易相对于国内的交易来说，具有更高的效益。这意味着，中国境内的人民币资本市场里面涉及到的各种金融市场，虽然细分的市场类型繁多，但是相互之间的市场结构却非常相似。比如，股票市场、债券市场、期货市场都是用来做资金的输入输出，而且都大致遵循着日内、日外两个通道，日内通道通常涉及债券市场，日外通道则涉及国外的期货市场和股票市场。同样，房地产市场、黄金市场、原油市场、石油市场等，都是一些商品的流通市场。而人民币资本市场与整个市场中的各个市场之间存在着交叉关系，不同类型的资金的进出将相互影响，所以确实是一个整体的市场。

## 2.2 为什么需要人民币资本市场？
由于中国在货币体系上的特殊性，以及制度上的不完善，特别是近年来冷战后出现的全球化背景下，中国经济面临巨大的国际竞争。比如，中国每年对外投资超过6万亿美元，其中约8万亿美元来自西方阵营。另外，一些大型国企改制、新兴市场蓄积，给予中国的劳动力市场和基础设施建设带来巨大优势。这些因素共同驱动着中国经济不断扩张，并且使得中国进入了一个新的货币政策时代。

在这个时代背景下，人民币资本市场的发展将产生深远的影响。首先，它可以帮助中国完成一系列经济政策的调整，尤其是货币政策的调整。譬如，调整脱离“双头 money”的局面，允许更多资本流入实体经济；调整结构性存款贬值、货币印花税、财政赤字等问题，以支持实体经济的长期发展；同时还可以推动国家的财富集聚效应，以及与国际接轨，并对国际贸易与投资政策产生重大影响。其次，它可以使得国际金融界接受中国货币政策，为中国的金融市场提供更有竞争力的环境，并对其国内外政策发挥积极作用。第三，它可以使得中国企业得到更多的资本金融支持，通过资本金融带动产业的创新升级，提升生产效率和品质，实现经济发展目标。最后，它还可以为中国与国际社会间的关系和政治经济发展提供更好的平台，促进中国的发展。

综上所述，人民币资本市场的重要性和作用已经被大家认识清楚。当然，这个市场还面临着很多 challenges ，比如，如何平衡中国经济的需求，以及如何促进资本流动的均衡、健康和充分利用？如何兼顾实体经济、基础设施建设、劳动力市场等多元性，让整个国民经济保持高度协调？如何推动政策的透明度、参与度和执行力度，确保其能顺利、有效地执行？如何将人民币资本市场与金融市场、贸易和投资市场等密切相关联，推动它们共同发展？这些问题目前还没有很好的答案，需要进一步研究探索。

# 3.CNY市场运作模式
## 3.1 CNY指数的定义
人民币资本市场的主要目的，就是构建以人民币计价的资本市场。但由于历史、经济、政治原因等复杂原因，人民币不是国家唯一货币，而是人民共和国国内储备货币，即“一篮子货币”。因此，在不同的时期，人民币资本市场都会使用不同的名词。一般而言，可以把人民币资本市场里面的各种资本商品指标统称为CNY指数，即在人民币计价的资本市场中，衡量某个特定资本商品的价格水平。例如，可以把股票市场的指数称为“股指”，而黄金市场的指数称为“贵金属指数”，等等。

不同的资本商品的指数可以分为以下四种：

1. 股指：股指是指股票市场上所有的股票的平均价格。股指的创始日期一般为1990年，它的涨跌幅度比较大，但短期内不会超过5%。股指与市盈率息差无关，只是指普通股票价格的变化。

2. 贵金属指数：贵金属指数的起点一般为2000年左右，主要用来测量黄金的价格波动。由于贵金属本身的属性，导致其价格比普通货币贵得多。黄金价格波动会引发社会的广泛关注，有时候甚至可以直接决定一国是否会发生通胀。但由于人民币的币值属性，贵金属指数可能不是真正反映黄金市场的指标。

3. 原油指数：原油指数是以中南坑压原油的油价作为参照物，反映原油的价格。1997年1月，中国正式开始实行低碳政策，原油价格开始上涨，直到2008年中国原油终端客户价格普遍上涨。

4. 石油指数：石油指数是国际能源组织（IEA）规定的一种对全球石油市场的贸易状况的评估工具。全球产油国的石油价格的波动都会影响石油消费，因此石油价格影响了石油市场，其价格指数也会反映其价格变化。

## 3.2 CNY市场的运作方式
CNY市场的运作主要分为两步：一是决定人民币的购买力，二是根据CNY指数购买资本。

第一步，决定人民币的购买力。一般来说，中国的GDP在增速放缓、失业率上升、失业保险支出占GDP比重逐渐减小的过程中，资金缺口加剧，人民币成为资金净流入的主力军。

除此之外，还有其他资本类商品也会进入CNY市场，包括黄金、原油、石油等。由于人民币本身的固定价值属性，因此上述资本类商品的价格也必须由人民币计价，因而也会受到CNY指数的影响。

除了上面介绍的两种方式，还有其他方式可以进入CNY市场。譬如，公司可以通过发行CBN(公司债券)、MDO(融资券)等国家债券或企业债券，购买国际上需要用到人民币资本的项目或资产，这些项目或资产的价格由人民币计价。再比如，借助电子支付系统和移动支付系统，个人或机构可以在CNY市场上进行交易，购买商品和服务。此外，越来越多的金融企业开始涉足CNY市场，进行跨境投资、保理业务、财富管理等业务。

总结而言，CNY市场是一个覆盖各个金融领域的市场，而且有着自己的规则，比如决定人民币购买力、制订交易规则等。由于发展速度快、环节多、条件复杂，因此CNY市场的运作过程还是很复杂的。但是，随着人民币资本市场的不断发展，必将为中国的金融发展创造新的机遇。
