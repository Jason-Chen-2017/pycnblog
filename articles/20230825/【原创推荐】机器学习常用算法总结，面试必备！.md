# 2.算法目录
## （1）监督学习

- **KNN(k-近邻)算法** ： KNN算法是一个非参数化的方法，它通过计算样本之间的距离来决定每个样本的类别。它最主要的优点就是简单有效。

- **逻辑回归** : 逻辑回归模型也被称为最大熵模型或线性回归模型。它假设输入变量之间存在一定联系，输出变量取值为0或1。可以理解为一种二分类算法。

- **决策树** : 决策树是一种常用的分类与回归方法。它的基本思想是基于数据集构建一颗树形结构，当待预测的对象出现在某个节点时，就可以依据条件将其划分成若干个子结点，然后进入下一个子结点直到预测结果。

- **支持向量机（SVM）**: 支持向量机（Support Vector Machine，SVM）是一种二元分类器，其特点是在保证高准确率的同时尽可能保持低错误率。SVM主要用来解决分类问题，它通过设置一系列的间隔边界，把不同类别的数据区分开来。

- **随机森林** : 随机森林是集成学习中的一类典型的学习器。它由多棵树组成，每棵树都由Bootstrap采样得到。该算法采用随机过程提升准确率，能够处理复杂且多维的分类任务。

- **朴素贝叶斯** : 朴素贝叶斯算法是一种概率分类算法。它假定所有特征之间相互独立，即认为在分类问题中，各个特征对于结果没有影响。该算法可用于文本分类、垃圾邮件过滤等场景。

- **神经网络** : 神经网络是深度学习中的一种重要模型。它通过模拟人类的神经网络行为来学习数据特征，使得识别、分类、回归等任务变得更加容易、精准。

## （2）无监督学习

- **聚类分析** : 聚类分析是无监督学习的一种常用技术。它利用数据的内部结构（比如特征）来对样本进行划分，并且不同的划分方式会产生截然不同的聚类结果。

- **关联规则挖掘** : 关联规则挖掘是无监督学习的一个关键技术。它基于数据集中频繁出现的模式，找出相似的模式及其关联规则。

- **主成分分析** : 主成分分析（Principal Component Analysis，PCA）是一种维度压缩算法。它可以帮助降低数据维度，并保留主要的特征。

- **K-均值聚类** : K-均值聚类是一种非监督学习的聚类方法。它通过迭代地移动聚类中心来对样本进行分割，最终达到聚类效果。

- **DBSCAN** : DBSCAN是一种基于密度的无监督聚类方法。它首先确定一个核心对象集合，然后扫描整个空间，从核心对象开始发现周围的孤立点。

- **EM算法** : EM算法是统计物理学和数理统计中使用的一种算法。它用来估计概率分布参数，是一种迭代算法，可以用于无监督聚类和期望最大化算法。

## （3）强化学习

- **Q-learning** : Q-learning是强化学习的一种算法，属于基于表格的学习方式。它在一定数量的时间步内学习状态转移的价值函数，并且以此来选择动作。

- **SARSA** : SARSA是强化学习的一种算法，属于时序差分学习方式。它在一定数量的时间步内学习状态转移的价值函数，并且以此来选择动作。

- **Actor-Critic** : Actor-Critic是深度强化学习的一种算法，属于模型-策略学习。它通过Actor与Critic两部分组成，即学习的Actor是策略函数，而Critic则是一个评估函数，用来评估所选策略的好坏程度。

- **深度Q网络** : 深度Q网络（DQN）是深度强化学习的一种算法。它通过训练两个网络，一个是前向传播网络，另一个是后向传输网络。前向传播网络将环境输入映射到动作空间，后向传输网络则使用Actor-Critic算法优化目标函数，将环境反馈给Actor进行更新。

## （4）概率图模型

- **无向图模型** : 无向图模型是一种概率模型，它用于表示概率分布，包括贝叶斯网络、马尔科夫网络、吉布斯玫尔德链。

- **有向图模型** : 有向图模型又称为马尔科夫网路，它通过建模随机系统中各个状态之间的转移关系，用于表示概率分布。有向图模型通常用于概率转移的建模以及数据挖掘。

- **概率图匹配** : 概率图匹配是一种无向图模型，它的特点是能在不知道网络结构的情况下，找到匹配具有最高概率的候选结构。它广泛应用于知识发现，关系抽取，事件聚类等方面。

- **变量消除** : 变量消除是一种有向图模型，它的作用是消除冗余因子，在因果推断和信息传递中起到重要作用。

- **马尔科夫随机场** : 马尔科夫随机场（MRF）是一种概率图模型，它通过描述观察到当前状态的所有前向消息，来刻画未来可能的状态。MRF有很好的理论基础，能够捕捉到状态之间转移的依赖关系。

- **条件随机场** : 条件随机场（CRF）是一种概率图模型，它的特点是通过约束所有可能的状态转移路径，来限制状态空间的大小。CRF可用于序列标注、语义角色标注等任务。

## （5）深度学习

- **卷积神经网络（CNN）** : 卷积神经网络（Convolutional Neural Network，CNN）是深度学习的一个重要模型，它能够自动提取图像特征。CNN是一种局部连接网络，能够提取空间上的相关性，实现了局部像素间的特征共享。

- **循环神经网络（RNN）** : 循环神经网络（Recurrent Neural Networks，RNN）也是深度学习中的一个模型。它可以捕捉时间上的序列依赖性，能够处理长时间序列数据的建模。RNN的设计灵感来自于人类的神经元网络。

- **递归神经网络（RNN）** : 递归神经网络（Recursive Neural Network，RNN）是一种深度学习模型，它将之前的输出作为下一次输入，并逐级生成输出序列。

- **长短期记忆（LSTM）** : 长短期记忆（Long Short-Term Memory，LSTM）是一种特殊的RNN类型，它能够记住之前的信息，增强长期依赖。LSTM用于文本建模、图像识别等领域。

- **Generative Adversarial Nets（GANs）** : 生成式对抗网络（Generative Adversarial Netowrks，GANs）是一种深度学习模型，它通过生成器网络生成新的样本，并通过判别器网络判断样本是否真实存在。GANs可以应用在图像、文本、音频、视频等领域。
