# 1.简介
  

## 1.1 定义
股票市场，即美国股市和英国股市等国家及地区的商品交易所交易的、纸浆制造业的固定收益性金融工具，其起源于中世纪末欧洲资本主义大都会之间的贸易交换和商品流通，并由此产生的双边市场经济。每个股票都是一个有一定价值的普通固定资产，其价格取决于供求关系、买卖双方的力量对比和市场的发展程度。

## 1.2 分类
股票市场按照影响范围和结构分为三类：

1. 主板股票市场：包括NASDAQ、NYSE等，它们是全球最大、最活跃的股票交易市场，主要从事证券投资业务，并且集中度较高；
2. 中小盘股票市场：也称为创业板市场，包括中国A股、美股等，这些股票主要面向个人投资者，注重企业经营和投资环境；
3. 次新股市场：包括BAT等，主要面向上市公司，是市场风险和交易成本更低的市场。

## 1.3 发展历程
股票市场的发展历程可分为几个阶段：

1. 第一代股票市场（约1790年左右）：这是从早期伊甸园时期直到1871年才逐渐兴建的交易所，是欧洲商人直接进行商品交易的地方；
2. 第二代股票市场（1871-1913）：这一阶段是早期股票市场的衰落时期，特别是在美国市场上，由于政府政策的限制和美国财政状况的恶化，大量的公司与投资者离开了这个市场；
3. 第三代股票市场（1913-至今）：它是近几十年来全球股票市场的黄金时期，主要由美国、中国、法国、德国、英国和日本五个主要国家共同发展，是美国、中国、欧洲、日韩四国以及一些亚洲国家的主要交易平台。

# 2. 基本概念与术语
## 2.1 OHLC(Open/High/Low/Close)
在图形分析或趋势发现领域，OHLC线是指一段时间内的开盘价、最高价、最低价和收盘价，用表示形式分别为：(open，high，low，close)。OHLC线可以帮助我们很好地理解股票价格的变动趋势，如上升还是下降，以及起伏的程度。

## 2.2 Moving Average(移动平均线)
移动平均线（Moving Average Line，MA）也叫移动平均曲线或平滑移动平均线，是用一定周期内的某种数据计算得到的数值平均值，它是将多个数值的加权平均值，应用到一定的周期内，用来反映该周期内的资料走势，一般移动平均线可以简单理解为某一段时间内的平均价格水平。

## 2.3 Momentum(动量)
动量，又称动力指标，英文名为momentum，是研究市场走势综合性的一种技术指标。动量指标用于判断市场处于超买还是超卖状态，属于量价指标的一部分。

动量不仅会随着行情的发展而发生变化，还会受其他影响因素的影响。比如，当短期涨幅过快时，动量也会上升；当股价进入下跌期时，股价的波动率也会减缓，动量也会下降；当短期股价涨停时，上升的动量将被压制住，而回落的动量则可能进一步拉高。因此，动量是一种比开盘价更具有市场意义的技术指标。

## 2.4 MACD(Moving Average Convergence Divergence)
MACD，即移动平均收敛异同（Moving Average Convergence Divergence），是通过计算长短期移动平均线之间的差距，来判断股价趋势和研判趋势背离程度的技术分析工具。MACD线由DIF、DEA和OSC三条线组成，其中DIF（Differentiation）代表股价的运行趋势，即一段时间内股价波动幅度的变化值，由（前一日收盘价－前一日最低价）－（前一日最高价－前一日收盘价）得到。DEA（Moving Average of Differentiation）是上一次的DIF值的EMA，目的是为了消除波动的影响，使得DIF值平滑且可观测。OSC（Oscillator）线就是两条移动平均线之差，目的是显示不同移动平均线之间的距离和移动方向。

## 2.5 Bollinger Band(布林带)
布林带（Bollinger Band，BB）是一种指标技术，它是由一个标准宽度的标准差来确立的。标准差是一个样本与总体均值的偏差程度的度量，其大小就表明了样本与平均值的距离。布林带是根据历史的股价数据计算出来的统计学模型。布林带由三个不同的线组成：布林上下轨线（upper band）、布林中轨线（middle band）、布林上下影线（lower band）。

布林带的作用在于突出股价上涨和下跌的变化。当价格超过上下轨线时，则会被认为处于上升阶段；当价格低于上下轨线时，则会被认为处于下跌阶段；若价格落入中轨线附近时，则说明股价处于震荡区间，这时候就可以利用其他技术指标来辅助观察。

## 2.6 Relative Strength Index (RSI)
相对强弱指数（Relative Strength Index，RSI）是一种量化技术指标，也是一种趋势跟踪技术。RSI把每日股价波动与一段时间的平均波动比较，通过计算相对于设定阀值而言的股价变化速度，帮助投资者掌握股价趋势、判断股价未来走势、识别买点和卖点。

RSI指标计算方式：
1. RSI＝（收盘价—最低价）/（最高价—最低价）×100
2. N=14，一般N取14日或者28日，斜率近似为6日线，不过一般N选择14日效果最佳。
3. K线第N日的RSI指标：当日的RSI值与N日前的第N日的RSI值比较，计算出相对强弱指数RS。
4. 当日的RSI值=（（N日前的RS值×(N-1))＋当日K线的RSI值）÷N。
5. RS指标以0～100的百分比表示。其数值越接近100，代表股价的上升趋势越强烈，股价上涨速度快，数值越接近0，代表股价的下跌趋势越强烈，股价下跌速度快。

RSI指标运用范围：

RSI指标常用于判断市场整体趋势，也可用于个股的趋势分析。对于整体趋势的判断，有上涨、下跌、震荡三种情况，而对于个股的趋势判断，也分为上涨、下跌、震荡三种。如果个股的RSI指标出现剧烈上涨或者下跌现象，则表示该股可能发生转折点，需要进一步观察。
