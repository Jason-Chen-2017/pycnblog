# 1.背景介绍


血管瘤是常见的慢性病，临床表现多为胸腔积液，以及结节、结石等。目前国外已经有一些基于机器学习的诊断方法，但在中国尚未有类似的方法。因此我从微生物群体层面出发，采用了血管瘤核酸测序技术(SNP-Seq)，通过分析核苷酸序列数据发现潜在感染因子，并利用线上网络分析的结果，对不同的菌株进行分类并给予相应治疗建议。
我的研究成果具有重要的科技价值和临床意义。首先，通过这种全基因组测序方式，我们可以较为准确地鉴定感染菌株及其亚型，对于提高治疗效率起到了至关重要的作用；第二，我们采用网络分析的方式，将不同类型的感染因子区分开来，更好的掌握病人得病原因，从而更好地施治；第三，此类技术的开放性和易用性，使它能够很快被其他医院应用到临床实践当中。
在这项工作完成后，我将获得国家科技进步奖励，支持厦门大学张璐教授领导的国家重点基金“华南海鲜批發推广中心”。同时也得到了学术界和科研人员的高度关注。
# 2.核心概念与联系
## 2.1 感染因子
感染因子是一种微生物，对人类细胞产生影响，并导致生殖细胞活性下降、免疫力下降等疾病。目前常见的感染因子包括：抗病毒素、免疫调节素、抗坏死素等。其中，免疫调节素具有与抗病毒素相似的功能，但抗病毒素具有单克隆和多克隆两种，能增强免疫应答，抑制病毒感染。抗坏死素能够将免疫系统抑制起来，有效阻止细胞膜炎、白血病的发生。另外还有一些家族成员也可能参与感染，例如：宿主细菌和自身细菌、真菌、病毒。

## 2.2 SNP-Seq
随着基因组的扩增，基因编码序列中的碱基突变逐渐增加，由此带来的无穷无尽的可能的变异组合，使得对微生物群体的传播变得复杂而危险。在测序技术出现之前，研究者通常采用核酸转录本对感染菌株进行检测，但现在的时代背景下，由于基因组的快速扩增，对检测到的所有核苷酸进行精确判断成本太高，所以，便有了各种序列测序技术。目前，最常用的两种测序技术——DNA免疫合成技术（DNase）和Sanger测序技术（Sequencing by Tiling Pathogen Amplicons）。前者通过核酸的大小区分，可检测到整个基因组序列上的突变，但需付出昂贵的昂贵成本；后者通过分割遗传材料，在特定染色体区域加工成精心设计的PCR片段，可以检测到每个SNP或单个碱基的变化，但在建立SNP-Seq数据库方面的效率不如前者。

## 2.3 核酸检测方法
目前主要的核酸检测方法有以下几种：
### ART（高通量核酸引物T）法
即Alu repeats-and-terminators technology，是一种用于检测核酸的技术。它的特点是在基因组内加入可选的A，T，G序列作为引物，通过构造靶向覆盖范围大，比对精度高的重复序列，在每个靶向重复区域检测到多种核苷酸片段。ART法可以准确检测小核苷酸甲烷胺阳离子（Cation-Realted Transient，CRT），用于检测具有免疫缺陷的皮肤。
### RT-PCR法
即Reverse Transcriptase Polymerase Chain Reaction，它是一种实时反向转录链反应法。它的基本原理是通过反向录制，将接触到的DNA甲基化，然后再将该DNA翻译为RNA。通过PCR技术检测到感染菌株的核苷酸序列。由于RT-PCR需要反向切割基因组，因此它的运行速度比核酸斑竹速率要慢很多。目前，RT-PCR法已成为新一代的核酸检测技术。
### CGH-Seq法
即Cytosine Gradient Hybridization Sequencing，它是一种抗核苷酸偏移技术。它采用了一对来自亲和宿主的干扰核苷酸对，通过将这些核苷酸互补配对、纳入配体结构的环状蛋白、热轧制造新连接、核苷酸的形态、载体的特性、化学修饰等因素共同作用，创造了丰富的微生物对比，并通过测序获得了清晰的核苷酸序列信息。CGH-Seq技术适用于检测具有复杂遗传模式的微生物，且抗检测出错能力强。

## 2.4 网络分析
网络分析（network analysis）是一种利用关系和链接的数据，来探索复杂系统的组织结构和功能。根据边缘数量，网络分析可以分为度分布、密度分布、关联规则分析等。微生物群体中出现不同形式的核酸序列之间的关系，就可以通过网络分析揭示不同感染因子的作用机制。
