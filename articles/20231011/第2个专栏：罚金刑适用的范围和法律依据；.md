# 1.背景介绍


## 1.1　“罚金刑”和国际惯例认可的“原罪与判决认定”模式
“罚金刑”是古代社会的一种刑罚手段，在现代则被赋予新的生命力和意义。在古代，罚金刑用于刑死人，判决认定包括有罪和无罪两种结果，严重者可以酷刑甚至处死；在当代，罚金刑还用于执行死刑、追溯不法分子等司法程序。国际惯例的“原罪与判决认定”模式是指：原告认罪，被害人可以请求判决认定；如果原告不能证明自己有罪，或其行为已触犯法律法规或道德规范，则应另行裁决。中国目前也没有严格遵守国际惯例的“原罪与判决认定”模式，仍然实行类似古代的严厉刑罚，这样既损害了刑罚执行效率又难以保障人们的人身自由。因此，有必要探讨如何构建并完善国际标准下的刑事司法模式和法律制度。
## 1.2　“罚金刑”的适用范围及法律依据
### （一）罚金刑适用范围
“罚金刑”作为古代社会刑罚手段，它适用于以下情形：
- 首次、过错造成严重后果且经过审判程序不能得到赔偿的个人；
- 情节恶劣，或无法预见、排除、避免，而危及他人合法权益的个人；
- 有组织犯罪活动，造成严重后果的组织、团体；
- 具有较强暴力倾向、侮辱、殴打、纠缠等嫌疑，造成严重后果的个人。
### （二）罚金刑法律依据
“罚金刑”作为古代社会刑罚手段，存在着诸多缺陷。其中最主要的问题在于“受害人权利”的承担问题。“罚金刑”一般采用追究刑事责任的方式，只有刑满释放后才可获得赔偿。这种方式虽然有助于减轻受害人的经济压力，但却忽略了对受害人的真正权利。因而，“罚金刑”并不符合现代刑事诉讼中应当给予“免罪的原则”。更糟的是，“罚金刑”没有办法保障受害人人身自由，它导致残疾、精神病患者的人身安全和自由受到威胁，并可能导致社会财富流失。因此，除了要尊重受害人的基本权利之外，加强法治建设和社会公平正义也是应对“罚金刑”的必要前提。
为了有效保护受害人的权益，我国近几年通过立法、制度、政策等方面相继出台了一系列刑罚改革政策。其中，《刑事诉讼法》的修改颇具代表性。1997年修订版通过后，“罚金刑”就作废，取而代之的是五种折算刑期、量刑折罚的方法。这些方法虽然不完全禁止“罚金刑”，但它们确保了公民的合法权益，保障了被害人的权益。其中，“三年缓刑”制度被认为是刑罚政策的里程碑。《刑事诉讼法》修改后的“三年缓刑”制度建立了公平、公正、透明的制度环境。
### （三）罚金刑的优点与长远目标
“罚金刑”作为古代社会的刑罚手段，它的优点是简便易操作，能够快速、经济地解决某些司法敏感问题。但是，它也存在一些缺陷。首先，“罚金刑”的执行效率低下，因为受害人都须服从紧急的刑事程序，需要在刑期内忍受多种痛苦，因此执行效率偏低。此外，“罚金刑”并未制定严格的法律，因此容易滥用，且随时可能面临刑事法律风险。因此，如何构建和完善现代刑事司法模式和法律制度，更好地运用“罚金刑”，成为长远目标。
