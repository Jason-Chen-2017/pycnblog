# 1.背景介绍


## 概述
随着互联网的发展、人工智能的崛起和新型机器学习的浪潮，大数据、云计算、区块链等新兴技术在人类社会的各个领域都处于蓬勃发展的阶段。为了让人们更好地掌握这些技术的最新进展，需要有相关技术专业人员对其进行研发、设计和应用。而如何合理地使用这些技术还需要有充分的知识积累和技艺水平。如今，分享知识成为了一种重要的方式，越来越多的人希望通过交流和研究发现新的趋势、规律、方法。但是当我们涉及到原创、翻译或者复制他人的创意、观点和技术方案时，原文作者应该如何给予作者署名？如果是完全复制他人的文字并不符合共同许可证（Creative Commons），作者又该如何引用他们的原文？
本文基于共同许可证（Creative Commons）的原则，介绍了在中文环境下如何正确引用他人的作品。希望能够为中国的IT从业者提供一些参考和建议。同时，我们也欢迎大家与我们一起完善这篇文章，将它推荐给更多的中国IT从业者阅读和学习。
## 关于共同许可证（Creative Commons）
共同许可证（Creative Commons）是由国际非营利组织Creative Commons基金会推出的一种知识共享协议。该协议允许创作者根据指定的许可条款向公众开放、再利用和改编他人的作品。创作者可以选择许可证下的一系列条款，其中包括授权免费的使用、分享、修改、衍生演绎、制作衍生物以及许可第三方公开发行等条款。通过这种方式，创作者可以自由地利用其他作品进行二次创作，也可以分享自己的作品，创造新的价值。许多知名作家、电影明星、博主、企业、组织已经加入或申请加入共同许可证的行列，致力于推动科技发展和公民能力构建。
### 为什么要加入共同许可证
共同许可证是分享知识、创造价值的重要途径之一，它的出现使得知识的传播、创新和传播变得更加容易、普遍和有效。它的主要优点如下：

1. 创作者不需要承担法律风险。无论是直接复制、再创造还是基于自己作品进行再创作，创作者都不用担心侵犯他人的知识产权。
2. 有助于创作者参与到公共讨论中。许多公共领域的议题都是由多人合作完成的。通过共同许可证，创作者可以帮助媒体调查记者采访记录，扩大研究范围，甚至于让自己的作品获得国际声誉。
3. 可促进信息的自由流通。许多社区网站都会要求用户遵循许可证的规则，在这些网站上发表的内容没有版权限制，适合公众使用。对于初创公司来说，能够快速扩张产品市场、吸引投资并获得支持也是十分重要的。

### 共同许可证的版本及版权信息
共同许可证共分两大类：版本1.0和版本3.0。它们之间的不同点主要是授权的限制。版本1.0对创造性工作的限制较少，允许包括摘要、插图、照片、录音等在内的非商业性用途。版本3.0则更为严格，只允许用于非商业目的的创作，且不可用于任何商业目的。除此之外，两个版本还有很多细微差别，比如版本的名称和版权的说明等。因此，在具体使用过程中，最好选取适合的版本。

另外，每个共同许可证下都有关于版权信息的说明，这对创作者和使用者都是非常重要的信息。共同许可证的版权信息一般会在作品描述前标注，例如：

> This work is licensed under a Creative Commons Attribution-ShareAlike 4.0 International License. To view a copy of this license, visit http://creativecommons.org/licenses/by-sa/4.0/. 

这个例子中的“This work”指的是文章的标题或标题的一部分。“Creative Commons Attribution-ShareAlike 4.0 International License.”就是共同许可证的版权声明，该声明表示作者授权给他人使用其作品，但是不得修改、使用该作品形成的派生作品必须继续遵守相同的许可证；最后一句话“To view a copy of this license,”指向该许可证的全文，供阅读。

如果是一个机构的写作或出版物，那么该机构必须声明自己遵守共同许可证，而且该书籍或期刊的所有章节、插画、照片、视频等作品都要采用相同的许可证。如果一部电影或电视剧的某些场景的剪辑、拼接等属于原创，那么剪辑组就要确保采用的许可证与原作一致，这样才能提供给公众。

### 使用案例
1. 以创造性的方式重新创作：许多创作者选择把自己的想法或者研究成果转化成具有独创性的作品。比如，以编程语言作为母语的程序员编写的脚本，可以用来教授计算机知识。人工智能研究员开发的机器学习算法也可以用来做广告优化或零售推荐。可以说，共同许可证为创作者提供了大量的创作空间。

2. 在媒体上传播自己的声音：有些创作者或者企业会设立自己的社群媒体平台。他们可以通过发布自己的观点、新闻、评论、分析结果、理念等来影响和推动公共事务。加入共同许可证可以增强自身品牌形象，提升其知名度。可以看出，共同许可证不是只有网络上传播自己的声音才值得鼓励的。可以加入共同许可证的媒体包括新闻媒体、报纸、期刊、广播电台、电视台、博客、播客、音乐平台、短视频平台、直播平台等。

3. 分享游戏开发技术：虽然游戏的创作形式并非完全独立，但也存在着一些共同的技术元素。比如，角色设计、关卡设计、人物模型、渲染技术等。加入共同许可证可以帮助更多的游戏玩家得到这些技术上的帮助。

4. 将自己创作的图片、视频、音频等数字内容分享给公众：部分创作者或者企业会将自己的作品以数字文件的形式发布在互联网上。加入共同许可证可以使作品获得更广泛的传播。可以分享图片的平台包括微博、贴吧、贴片图床、imgur等，可以分享视频的平台包括youtube、bilibili、vimeo等，可以分享音频的平台包括网易云课堂、网易云音乐、千千音效等。

5. 进行教育资源共享：很多教育机构和学校都在创办教材、题库等课程资源。加入共同许可证可以让学生受益，免去重复劳动，也可以在一定程度上宣传教育理念。可以查看共同许可证的列表，找到适合自己国家、领域的许可证。

总结一下，共同许可证最大的优点是分享知识、创造价值，并增加了社会公信力。如果大家想要使用或创造内容，那么应该先熟悉共同许可证的原则，然后按照相应的规则选择合适的许可证。
