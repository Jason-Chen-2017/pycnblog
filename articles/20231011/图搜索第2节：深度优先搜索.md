# 1.背景介绍


## 一、什么是图搜索？
在计算机科学中，图搜索是指通过图论的方法来找寻问题的解或者解决问题。简单来说，就是寻找一种有效的方法将一组节点或者顶点连接起来，使得各个顶点间能够连通或遍历到达其他顶点，从而找出符合某些条件的路径。换句话说，就是找到一条“有效”的、“可行”的路线，可以满足一些限制条件。图搜索算法的目的是通过对图的结构和边的权重进行分析，找到最短、最快、最优等最优解，而不是只是找出一条无效的、错误的路线。对于复杂网络、股市、航班安排、人力资源配置等问题都可以使用图搜索算法。
## 二、图搜索算法的分类
图搜索算法按其关键元素及其运作过程分成以下几类：
### (1) 深度优先搜索（Depth-First Search, DFS）
深度优先搜索算法是一种经典的图搜索算法，它通过一个递归的方式来遍历树或者图。算法的基本思想是：首先访问某个顶点，然后依次访问该顶点的所有邻接顶点，如果邻接顶点没有被访问过，则继续访问；否则跳过该顶点。当所有邻接顶点都已被访问完毕时，回退一步，再访问另一个尚未被完全访问过的顶点。这个过程一直重复，直到访问完整个图。DFS算法适用于连接性较强的图，例如，在社交网络中查找潜在联系人，在地图上寻找路径。
### (2) 广度优先搜索（Breadth-First Search, BFS）
广度优先搜索算法是一种用来搜索树或者图的搜索算法。它的基本思想是：从根节点开始，按层级遍历图，先访问离根节点最近的一层的节点，然后再依次访问这些节点的相邻节点，直到最后的叶子节点。BFS算法适用于求最短路径的问题。例如，在开放寻路网格中寻找最短路径。
### (3) 对称差分算法（Symmetric Difference Algorithm, SDA）
对称差分算法又叫做求差集算法，也属于图搜索算法，主要应用于两个集合之间的运算。该算法的步骤如下：

1. 用第一个集合A中的每个元素作为第一个队列Q1。
2. 用第二个集合B中的每个元素作为第二个队列Q2。
3. 从队列Q1和队列Q2中取出不同的元素加入结果集合C。
4. 如果队列Q1为空，则依次将剩余元素从队列Q2中取出，并将之加入结果集合C。
5. 如果队列Q2为空，则依次将剩余元素从队列Q1中取出，并将之加入结果集合C。
6. 将结果集合C返回。
7. 当集合A和集合B相同时，输出为空集。

SDA算法可以求出两个集合的不同元素，并且给出了一种求差集的方法。例如，可以求出一个学生所在班级内不合格的学生，或者求出两个集合中不重复的元素。
### (4) 基于堆栈的最小生成树算法（Minimum Spanning Tree, MST）
MST算法又叫做生成树算法，是一种图搜索算法，也是解决最短路径问题的重要工具。一般情况下，MST算法基于Prim算法或者Kruskal算法实现，即选择一条边，添加到最小生成树中，然后使生成树稠密，最后得到一条连接所有顶点的最短路径。MST算法的运行时间依赖于图的规模，所以也会受到图的复杂程度的影响。目前MST算法已经成为在图的划分和优化问题上取得重要的基础性工作。
