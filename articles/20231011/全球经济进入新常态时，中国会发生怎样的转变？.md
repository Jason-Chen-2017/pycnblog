# 1.背景介绍


## 全球经济进入新常态
近年来，全球产业结构、资本运动、金融危机等都在发生变化。随着新经济体系崛起，制造业持续走向世界领先地位，加上消费者的减少，人们对未来的期望也逐渐增加。因此，在这个背景下，中国外交政策开始收紧，中国正积极推进改革开放，关注国际局势的变化，评估中国应如何应对全球经济的发展。
## 中国经济发展方式的转型
中国经济在市场经济转型过程中一直处于“去产能”“去库存”的阶段，这使得中国经济发展遇到了很多 challenges ，其中一个重要的因素就是结构性调整。过去几十年间中国经济实现了从计划经济到市场经济的过渡，但随着新的全球产业体系的出现，带来新的经济全球化趋势。中国在进行结构性调整方面需要具备以下条件：

1. 制定目标、目标导向
2. 兼顾国内与国际需求
3. 提升竞争力
4. 完善基础设施
5. 扩大产业链

而中国通过推进结构性调整有如下优点：

1. 结构性升级促进民营经济发展
2. 激活城乡结合部队，激发创新能力
3. 国际化战略强化国际治理能力
4. 支持更多中小企业创新创业
5. 在更高层次上促进全球经济的健康发展

## 中国经济的未来展望
### 中国经济的长期趋势
2020 年我国国内生产总值增速较 2019 年同期提升了 7% ，预计今后五年保持在 6% 的增速，受益于 5G、智能制造、大数据、人工智能、区块链等新兴产业的快速发展，制造业销售量占比超过 50% ，经济总体呈现出扩张性增长。

2020 年我国制造业增加值同比增速也达到 6% ，其中工业增加值增速超过 4% ，受科技进步、服务业蓬勃发展等因素影响。

2020 年我国景气指数排名全球第四，消费价格指数有所下降 ，2020 年中国 GDP 累计亮红，中国经济处于行业领先地位。

2021 年前三季度中国产业规模将继续扩大，2021 年第一季度产业规模已经超过美国，成为世界第二大制造业集团。

这些统计数字显示出，中国经济已具备巨大的结构性优势，已经具备处理国际经济衰退、加快中国崛起、促进世界经济复苏的能力。

### 中国经济的现实挑战
#### 宏观经济面临的新挑战
当前全球面临三大周期：制造业、服务业、贸易逆差（三角债）。在制造业领域，美国等巨头生产的产品比中国制造的更受欢迎，中国主要依赖进口替代增长，此外，由于制造业的战略性作用，中国还面临着一些其他问题。例如，中国高铁建设进展缓慢、半导体短缺等。另外，对于短板产业，中国仍然是领先的。

贸易逆差问题主要由两方面造成：一是制造业国际化进程落后，二是中国出口实际占比偏低。

制造业国际化进程落后导致制造业产出过剩，同时制造业的国际订单规模太小，导致缺乏供应商支持。解决这个问题的方法之一是引入跨境电子商务平台，帮助中国企业更好地满足国际客户需求。另一种方法是增加服务业开支，鼓励实体经济转型。

中国出口占比偏低是由两个原因造成的。一是服务业国际化程度较低，导致中国出口不足；二是服务业结构中，软件服务占比太高，这种占比在欧美国家中较低。为了解决这两个问题，应该采取扩充进口，提升软件服务质量，推动服务业国际化发展。

#### 地方经济面临的新挑战
地方经济面临着土地流失、地方债务、企业破产、房地产开发难题等问题。

1. 土地流失问题：当地土地资源及其价值被压缩，以及土地征用及出让过程存在问题，导致土地流失。解决办法是拓展农村土地利用权，鼓励农民持续耕种。

2. 地方债务问题：土地征用及出让所得税等负担过重，导致地方债务问题。解决办法是偿还财政部门减轻的债务，推动流动资金及住房置换资金等的资金配置，尽早还清欠款。

3. 企业破产问题：由于土地征用及出让导致居民收入减少、土地成本上升，企业经营困难，产生债务风险，致使企业破产或倒闭。解决办法是规范土地征用及出让行为，减少土地流失并建立更好的土地信用机制，增强地方政府的融资能力。

4. 房地产开发难题：地方政府普遍缺乏专业化房地产资源，产权保护意识薄弱，土地信用机制脆弱，导致房地产开发难题。解决办法是加强土地信用、资源保护与引导，建立地方政府的专业化管理，培育房地产投资人群，稳妥推进房地产发展。

#### 劳动力市场面临的新挑战
当前劳动力市场面临着劳动力供给过剩、缺乏劳动力保障、工作岗位招聘供需矛盾、劳动力成本上涨、职业疲劳、失业率上升等问题。

1. 劳动力供给过剩：由于经济增长带来的工业化和高技术产业的蓬勃发展，导致劳动力需求增加，劳动力供给却始终不足，造成劳动力供给过剩。解决方案是优化劳动力结构，适时释放消费潜力，激发就业创造力，增加劳动力供给。

2. 劳动力保障问题：目前我国劳动力保障水平仍较低，包括就业保障、医疗保障、养老保障等问题。解决方案是依托互联网+劳动力信息化、新能源、数字经济、智能制造等技术，提供便利、安全、有效的劳动力保障。

3. 工作岗位招聘供需矛盾：由于就业市场供求不平衡，导致工作岗位需求大幅减少，招聘供应过低，造成劳动力成本上升。解决方案是构建充满创造力的招聘平台，尊重劳动者个性特点，保障就业形象，广泛吸纳新人才。

4. 职业疲劳问题：近年来，中国的低技术工种薪酬水平越来越低，职业疲劳现象越来越严重。解决方案是加强职业教育、社会监督、薪酬体系建设、晋升机制设计，引导广大劳动者追求体面的工作岗位。

5. 失业率上升问题：劳动力市场的不确定性、行业技术升级带来的人才流失、失业压力、税费负担等问题导致失业率上升。解决方案是提高就业转移支付，支持有能力的青年和学生提升职业技能，不断扩大就业市场。

### 中国经济发展的目标
中国经济发展的目标是：

1. 国际收支平衡

2. 发展服务业

3. 改善制造业

4. 推进国际化进程

5. 拓展就业空间

6. 解决中西部地区矛盾

7. 加强海外扶植

8. 建设开放型经济体系
