# 1.背景介绍


随着近视眼、哮喘和癫痫等传染性疾病的爆发，越来越多的人在进行孕检、胎儿转移及残疾患者重建方面的临床支持。目前孕检中采用的是“盲人版”的模态检测方法，即只有医生通过观察实物才能判断模态。而病理学却提供了许多可以用于无监督模态识别的方法，包括基于特征的算法和机器学习的算法，这些算法利用不同类型和数据集上的生理、生化、免疫学和影像学特征，对目标模态进行分类。近年来，有关无监督孕检模态识别领域的研究取得了很大的进步，尤其是针对一些模态的差异性，如弓形、异侧、甚至面部轮廓等。然而，如何有效地应用于实际应用仍是一个难题。本文将从病理学、统计学、计算机视觉、图像处理等相关知识出发，论述孕检模态识别中的无监督分类器设计方法，并对不同分类器的性能进行评价，最后探讨对新模态的适应性。
# 2.核心概念与联系
## 2.1 模态定义
无监督孕检模态识别（Unsupervised Modality Recognition for Fetal MRI）的关键在于对目标模态做分类，分为五种基本模态：圆柱体、椭圆、半圆盘、平面三角形、帽状天盘。其中，椭圆模态是最具代表性的一种，通常用目标的红润区域表示，椭圆的边界由若干条相互接触的线段构成。圆柱体模态指的是圆管状的模态，通常为外甥或不受控制的宝宝，圆柱体模态一般呈现出较高的形态复杂度和不规则性。半圆盘模态是指两部分构成的半圆，两端延伸到半径中心，模拟生殖器及卵巢。平面三角形模态则可以代表未经剪切的生殖器。帽状天盘模态一般出现在产褥期间，主要是由于创伤引起的异常形态变化。

## 2.2 模态相关性
无监督模态识别需要考虑两个模态之间的相关性。首先，不同模态之间存在模态间的结构性关系，例如圆柱体模态和椭圆模态是有相关性的；另外，不同模态还存在空间的局部相关性，如圆柱体模态与其他模态之间存在相邻关系。第三个关系是模态间的关联性，如椭圆模态与平面三角形模态是相邻的，而且存在相似性。四个模态都能够在一定程度上表征生理、生化和免疫信息。因此，无监督模态识别的关键在于找到能够捕捉到所有模态信息并进行正确分类的分类器。

## 2.3 非同质性
非同质性（Nonuniformity）表示模态分布不均匀或者模态形状变形严重。如平滑翼曲面，双曲面，环形面以及瘢痕，破碎片等，均属于非同质性模态。非同质性模态往往难以被成功地分类，因为它们不能准确地区分目标模态和背景噪声。

## 2.4 场景不变性
场景不变性（Scene Invariance）指的是模态在不同场景中的位置或光照条件不发生变化。在不同人的脊柱间，脂肪瘤，肠道包块以及正常组织组织下，脏器的形态都是不变的。因此，模态识别在不同的场景中应该具有一致性。但是现有的算法也不能够考虑场景不变性，因而会产生错误结果。

## 2.5 数据量不足
数据量不足（Insufficient Data）是指数据的规模过小，导致算法无法学习到足够的信息。一个典型的例子就是模态信息缺失，也就是没有足够的数据来学习如何描述不同类型的模态。因而，无监督模态识别的重要挑战之一就在于如何充分利用提供的数据，提升模态分类的效果。

## 2.6 模态的多样性
模态的多样性（Multimodality）表示模态变化的范围和方式非常广泛。模态的不规则性、模态的裂纹、渗漏、压紧、渗透、变形、错位、碰撞等都属于模态的多样性。模态的多样性使得分类器更加困难。为了克服模态多样性带来的挑战，现有的算法需要结合多个模态进行学习，提升模型的泛化能力。

## 2.7 模态相关性、场景不变性、数据量不足和模态多样性的综合影响
无监督模态识别所面临的最大挑战之一就是如何解决模态相关性、场景不变性、数据量不足和模态多样性的综合影响。现有的算法并不完全能够满足这些需求，需要进一步的改进。
