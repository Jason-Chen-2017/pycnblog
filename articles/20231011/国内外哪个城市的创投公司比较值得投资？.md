# 1.背景介绍


随着互联网的崛起，国际投行与互联网巨头纷纷争相布局，探索未来经济发展模式。中国在这一热潮中也占据重要地位，也对创投公司产生了兴趣，特别是在传统金融行业的衰落之后，社会的转型越发艰难，对于创业者来说，新的机遇正在不断出现。近几年，成都、杭州等创投热门城市已经成为国内投资者抢手的目标，那么这些创投公司到底在提升企业竞争力方面有多大优势？需要投资吗？
# 2.核心概念与联系
## 什么是创投
创投（英语：VC，全称是venture capital），是指以资本金融的方式追求增长、扩张并将其用于创新创业的一种金融活动。一般认为，VC的发起人首先要具备管理能力，并且信奉“自己做事，天下我有”。他们通过提供高风险的项目给予投资人，并帮助其寻找独特的商业机会，从而带来长期价值的企业。创投的目的就是为了找到商业模式最佳的项目，并将其带入市场。
## 创投公司在投资领域的角色
创投公司通常分为三种类型：
- **初创公司**：初创公司创立之初，就有一个长远的规划，并由投资人的资助完成整个项目的开发、推广、运营；
- **技术平台公司**：创投公司以服务于消费者为目标，帮助企业打造自己的数字化生态系统，包括云计算、大数据、物联网等新技术的研发与部署；
- **传统行业分析公司**：传统行业分析公司将受过专业培训的投资人团队引入现有领域，帮助其分析市场状况及客户需求，实现更大的商业利益。

## 创投公司的投资策略
创投公司的投资方式主要包括两种：
- **“围猎”模式**，即在市场上寻找具有足够发展潜力的项目进行加注；
- **“长线持股”模式**。

前者将所有资金都放在那些看起来有盈利空间的项目上，而后者则将股权长期分散给一些看起来可靠的公司，并持续跟踪其变动。
## 总结一下
创投是指以资本金融的方式追求增长、扩张并将其用于创新创业的一种金融活动，其发起人首先要具备管理能力且信奉“自己做事，天下我有”，然后通过提供高风险的项目给予投资人，并帮助其寻找独特的商业机会，从而带来长期价值的企业。在国内，创投热门城市主要包括成都、杭州、武汉、上海等。不过，光靠城市的好坏无法准确判断投资价值。因此，市场的发展也在发生着巨大的变化，如新兴产业、互联网科技的应用、人的工作方式、财富分配方式的重大变化等。总之，投资创投公司的目的是为了找到商业模式最佳的项目，并将其带入市场，实现对创业者的支持和激励。
