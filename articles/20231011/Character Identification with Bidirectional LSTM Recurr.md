# 1.背景介绍


## 机器翻译、文本摘要、文字识别、手写字识别等场景下都需要对输入的图片或者文字进行字符级的识别。字符级的识别意味着对每个字符的分类预测。本文通过双向LSTM网络结构，在不失真的情况下对图像中的文字区域进行字符级的识别。
## 特点
- 可扩展性强，网络可以适应不同字体大小、颜色变化、光照变化等各种场景下的文字识别。
- 使用双向LSTM结构，保证了信息的完整性，即使出现丢失或错误的字符也可以通过上下文信息恢复出来。
- 通过边界框回归修正检测出的文字区域，避免错检。
## 数据集
本文采用ICDAR 2019 数据集，共包括500张训练图片和500张测试图片，每张图片中包含多行英文单词和中文汉字混合的文字，共计约70万个字符。训练集中字体类型分布不均匀，但平均长度为40~50个字符。测试集中没有提供字符级别的Ground Truth标签，需要通过自动分割的方式获得对应的Ground Truth标签。
## 模型结构
### 模型输入层
该层主要包括两个输入：图像输入层（Input Image）和字符参数输入层（Character Parameter Input）。图像输入层主要用于获取特征图，将整张图片作为输入到网络中。字符参数输入层主要用于接收处理后的字符参数，比如字符所在的位置、尺寸等信息。
### CNN网络层
该层使用VGG16网络作为基础，并增加了一些卷积层提取更具全局特征的特征图。
### BiLSTM网络层
使用BiLSTM网络进行序列学习，将字符嵌入进去，从而实现对字符之间的上下文信息的学习。Bidirectional LSTM网络结构对原始输入数据做了双向运算，因此可以有效地捕获局部和全局信息，同时解决了深度学习中常见的问题——梯度消失或爆炸。
### 输出层
对于字符级别的识别任务，一般会采用CTC（Connectionist Temporal Classification）方法进行输出层设计。CTC方法是一种特殊的神经网络层，它能够正确识别时序数据，在语音识别领域也非常常用。本文的输出层采用全连接层后接一个softmax激活函数，使用CTC损失函数作为误差函数，通过反向传播更新权值，以此达到字符级的识别目的。
### 总结
基于上述模型结构和特性，本文提出了一种双向LSTM网络结构，用于对图像中的文字区域进行字符级的识别，其可扩展性强、准确率高且鲁棒性好。本文的数据集采用ICDAR 2019 数据集，能很好地验证该网络的能力。但是，由于数据集中缺少字符级别的Ground Truth标签，因此目前还无法比较不同模型在相同的数据集上的性能，只能看到网络的泛化能力。未来研究者可以考虑对实验数据集进行标注，构建新的字符级的图像识别数据集，以评估现有的技术水平。最后，本文希望借鉴双向LSTM网络的结构，用这种方法来改善传统手工编写规则或统计方法的字符级识别性能。
# 2.核心概念与联系
## LSTM(Long Short Term Memory)
LSTM是一个长短期记忆神经网络，由Hochreiter和Schmidhuber于1997年提出，是一种特别有效的递归神经网络。它有三种门结构，即输入门、遗忘门、输出门，其中输入门决定了哪些信息需要进入到记忆单元，遗忘门则决定了记忆单元中什么信息需要被遗忘，输出门则决定了应该输出多少信息。
## CTC(Connectionist Temporal Classification)
CTC是Connectionist Temporal Classification的缩写，一种用来解决序列标注问题的网络结构。它在深度学习中起着至关重要的作用，特别是在语音识别领域。CTC网络的输入是一个时间序列，网络根据当前时刻的状态以及之前时刻的输出预测下一个时刻的输出。在CTC网络的输出中，每个位置有一个概率分布，这个分布表示当前时刻状态下，每个可能的标签产生的概率。训练时，CTC网络依据真实标签计算得出损失函数，反向传播计算梯度，以此迭代优化网络参数，最终得到最优的网络参数。
