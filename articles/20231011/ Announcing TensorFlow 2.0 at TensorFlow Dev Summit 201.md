# 3.TensorFlow 是什么？
TensorFlow是一个开源的深度学习框架，能够进行机器学习和深度学习的相关任务，包括但不限于图像识别、语音识别、推荐系统、自然语言处理、无人驾驶等应用。

TensorFlow 一词通常指代的是 TensorFlow 框架，但是当我们提到 TensorFlow 时，通常指代的是下面的三个组件：

1. TensorFlow: 即 TensorFlow 框架本身，一个用于构建和训练机器学习模型的开源软件库；
2. TensorFlow Extended (TFX): 一个基于 TensorFlow 的可扩展平台，它为复杂的 ML 工作流提供了模块化的组件，如数据验证、特征工程、模型分析和评估等；
3. TensorFlow.js: 一个可以运行在浏览器中的 JavaScript API，允许开发者使用 TensorFlow 模型来执行预测、训练和转换数据。

因此，一般情况下，我们只需要关注第一个组件——TensorFlow。由于 TF 1.x 使用的编程语言是 Python，并且有较多的依赖库和第三方扩展，导致 TF 无法实现快速的发展和跨平台的兼容性，所以 Google 在 2018 年推出了 TF 2.0 ，采用了如下重大改动：

1. 效率提升。TF 2.0 提供更快的性能和更简单的模型构建方式，使得我们能够利用新硬件的能力来训练模型；
2. 更紧凑的语法。TF 2.0 拥有更简洁易用的语法风格，降低了新手学习成本，并减少了调试错误的可能性；
3. 可移植性。TF 2.0 可以很容易地移植到不同的设备上，从而实现跨平台的部署；
4. 灵活的集成。TF 2.0 被设计成一种可插拔的体系结构，可以轻松地与现有的工具和库集成，无缝集成；
5. 更好的可靠性。TF 2.0 中加入了自动并行化、分布式训练和超参数优化，增加了模型的可靠性。

因此，更新到 TF 2.0 有很多好处，包括：

1. 更快的性能。TF 2.0 使用的编程语言变成了 C++ ，具有更快的速度和更高的运算效率；
2. 更稳定的训练结果。相对于 TF 1.x ，TF 2.0 在数值精度和其他方面都有所改善；
3. 更便捷的模型构建。TF 2.0 提供了更多的模型构建器，使得新手用户可以快速构建出功能完备的模型；
4. 更广阔的应用范围。TF 2.0 可以广泛应用于各种类型的数据分析和机器学习应用场景，包括但不限于图像分类、文本分析、语音识别、序列标注等。

总结来说，TensorFlow 是 Google 开源的深度学习框架，它的最新版本 TF 2.0 带来了许多改变，如更快的训练速度、更方便的模型构建、跨平台部署等，能够极大的促进深度学习的发展。
