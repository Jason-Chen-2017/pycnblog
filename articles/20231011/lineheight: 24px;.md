# 1.背景介绍


## 一、什么是行高？
行高（Line Height）也称段落间距，即两段文字或者句子在竖直方向上的距离。一般情况下，行高越小，阅读起来就越容易集中精神。如果段落之间只有少量的空白，那么适当调大行高可增加可读性；反之，则需要适当缩短行高。行高过大或过小都会影响阅读体验。当然，每一个人的视力不同，对字号的感知能力也是不同的，所以说“不要一味追求行高”，而应该在合适的位置、合适的时间用对的工具做对的事情。

行高通常是一个相对单位，比如像素、磅值、厘米等。但也有特殊情况，比如字号不统一的文档、中文文本。另外，有些情况下行距也会改变行高。比如有些时候可能会出现多行中文字符或者英文单词跨行的问题，这时行距会改变行高。因此，行高的定义不能单纯依赖于像素或者磅值。

## 二、为什么要设置行高？
1. 适应阅读环境。通常，一张纸最好能容纳7～8行文字，否则阅读困难。超出了这个范围，阅读效果就会受到影响。所以，行高设置可以根据阅读距离、字号大小、阅读器尺寸、显示设备分辨率等因素制定。

2. 提升阅读速度。通过调整行距，可以减少阅读时对每屏幕阅读内容的反应时间。

3. 提高文本阅读质量。降低行距能够改善段落间隔、增加段落间隙。行距大的段落，阅读效果更佳。

4. 优化排版效果。当某段文字较长、字号较大时，行距过大，阅读效果会变差。所以，需要根据实际情况选择行距。

## 三、怎么设置行高？
设置行高，一般有以下几种方法：

### 方法一：使用默认行高

很多编辑器、软件都提供了设置默认行高的功能。如Word中的Ctrl+Alt+D（Windows）/ Cmd+Option+D(Mac)，在显示设置面板中找到“行距”标签页。点击“始终保持纯文字样式”后，即可设置浏览器默认的行高。




### 方法二：使用网页上设置行高工具

网站设置行高功能的地方一般都比较专业。比如简书设置行高的功能，可以通过鼠标选中文字，点击顶部工具栏的加号，然后选择“插入行高设置”进行设置。




### 方法三：使用字号设置自动计算行高

有的字体的字号并不是固定的，比如微软雅黑，其字号会随着浏览器窗口的变化而变化。此时，可以使用JavaScript或CSS，通过字号大小计算行高。

HTML代码如下：

```html
<div class="box">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
  <p>Suspendisse potenti. Nullam vitae erat at justo tincidunt sagittis vel ut urna.</p>
</div>
```

CSS代码如下：

```css
.box {
  font-size: 16px; /* 设置字号 */
  line-height: calc(1em +.5vw); /* 计算行高 */
}
```

这段代码通过calc()函数计算得到当前字号对应的行高，其中1em表示父元素的字号，.5vw表示浏览器窗口宽度的5%。这样设置之后，即使浏览器窗口大小发生变化，每个页面的行高都不会发生变化。
