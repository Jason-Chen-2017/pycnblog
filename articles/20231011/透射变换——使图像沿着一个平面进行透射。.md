# 1.背景介绍


## 1.1 概述

在数字图像处理中，图像透射是指将二维的图像沿着某一条直线或曲线进行拉伸、压缩，并反映到另一条直线或曲线上的一种效果。透射变换可用于以下场景：
- 视觉信息的增强或缩小。通过透射变换可以更加清晰地显示物体的边界信息。
- 对空间中的点的分布进行建模。通过透射变换可以将二维或三维的空间数据映射到另一维或高维空间，进而分析图像中的信息分布规律。
- 图像数据的重构或增强。透射变换可以将图像的结构转移到其他平面上，实现结构重新建模或者图像的纹理增强等。

## 1.2 形变的性质

透射变换是一类用来改变二维或三维形状的数学变换方法。它由两个基本要素组成：变换前的形状（原始形状）和变换后的形状（目标形状）。由于透射变换的目的就是用新的图像去描述原始图像的形状，因此在分析透射变换的性质时主要关注的是目标形状。

### 1.2.1 曲率参数与圆周率

对于透射变换来说，曲率是描述图像轮廓的关键因素之一。一般情况下，一张图像的曲率描述了在某方向上形变的程度。在透射变换中，曲率参数又称作变换因子，取值范围通常在[0,1]之间，1表示完全不变，0表示逆变换。

在具体计算中，若原始图像由两个相连的区域组合而成，则存在两个对应的变换因子。第一个变换因子定义了图像沿着原始区域所在的直线方向进行变换；第二个变换因子定义了图像沿着原始区域的斜线方向进行变换。

为了便于计算，人们定义了一个比值π/4。设定曲率参数时，其对应的值可以通过对原始图像的曲率大小进行归一化，即曲率除以π/4得到。如图1所示。

在实际应用中，通过将二维或三维形状投影到不同的基底上，可以将圆锥曲面的曲率参数转换为椭球的曲率参数，进而完成透射变换。由于椭球的中心与坐标轴重合，其曲率的表达式与圆锥相同。因此，从圆锥到椭球的变换也属于透射变换。

透射变换还有其他相关的性质，例如圆锥曲线沿直线的切线与切面都与圆锥曲线曲率无关，而沿弧度变化的函数则与圆锥曲线的曲率成正比。这些性质的原因是圆锥曲线的切线和切面的变形只影响弧度的一部分，而弧度的变化只有曲率参数才能影响。透射变换的研究以及计算方法具有广泛的应用价值。

## 1.3 图像透射变换的类型

### 1.3.1 反射变换

反射变换是指沿着一个直线或曲线从一个平面反射出去的图像被放大的过程。当源图像和目标图像的尺寸、位置、角度及方向都一样时，反射变换可看作是一种特殊的透射变换。
