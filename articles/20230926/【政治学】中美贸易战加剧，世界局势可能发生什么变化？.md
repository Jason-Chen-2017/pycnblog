# 1.简介
  

近日，美国总统特朗普发表声明，宣布将在今年6月1日前取消所有与中国直接签署的协议，并立即启动全面国防军事化部署。由于美国希望借此让更多中国企业退出市场，在此背景下，中美贸易战热度再次上升。从9月30日美方发布对台军售禁令到10月29日港府禁止美台直接通婚，中美贸易冲突在科技界掀起了一波新的热潮。中美两国之间经历了怎样的博弈，中美两国经济之间的关系如何演变，未来世界局势又将如何走向，这些都是值得探讨的热点话题。本文试图通过对相关知识进行梳理，对中美两国近期贸易战冲击、全球经济变化以及中美两国之间政治博弈的历史及其对未来的影响等问题进行探讨。
# 2.中美贸易战的历史进程
## 2.1 美国总统特朗普发表声明
美国总统特朗普于9月30日宣布，已将6月1日之前与中国签订的所有合同撤销，即撤回两国外交部长级以上会谈、5月份在中国东北建交、4月份对朝鲜努尔苏丹发动进攻、以及3月底与俄罗斯举行会晤等协议。此前的四项协议也均已终止，包括中方主张恢复5月份在北京的外长级会谈、11月份达成的新三条框架协议、10月2日与10月26日在京举行的政府首脑会议、和特朗普在9月30日与中国国家主席习近平举行的高级别会晤。特朗普表示，此举将带来"巨大的经济利益"，"对我国的影响将远超过经济利益"，"甚至有可能破坏两国关系稳定和互信"。但同时表示，美国方面并没有就此声明出售任何产品或服务给中国。
## 2.2 中美贸易战背景
中美贸易战是指美国政府（特朗普政府）和中国共产党之间的全面谈判和对抗。2007年，美国国务院以"贸易战"为名，向中国政府提出"零关税、开放、竞争"三个方面的要求，以期使中美贸易走出困境。2008年11月15日，中美贸易摩擦持续了10多天。2009年初，中美双方开始签署冷战时期领导人的联合声明，即双方对台湾、南海和钓鱼岛周边国家实施制裁。2009年12月13日，中国国家主席习近平在一次外长级会议上发表重要讲话，批评美国对台政策。随后，习近平在2010年6月访问印度时表示，"台湾问题与美国贸易问题没有必然联系"；习近平还强调，"不干涉中国内政、维护邻邦利益至上，是当前解决国际政治的正确方针"。
中美贸易战，带来了很多政治、经济和社会问题。其中最重要的是美国与中国的经济竞争以及对台湾、钓鱼岛、南海周边国家的制裁。而国内舆论也一直呼吁要对美国遏制中国崛起进行制裁，对于中国崛起采取更加严厉的制裁措施。随着国内舆论的日益加剧，越来越多的人注意到了中美贸易战带来的真正问题——贸易逆差、贸易壁垒、金融危机、腐败等。
## 2.3 全球经济变化背景
2008年金融危机爆发之后，欧洲、日本、韩国、台湾、香港、澳门等国家纷纷陷入贸易战，资本流动受阻。2009年，中国开始崛起成为世界第二大经济体。2010年后，世界经济快速增长，而中国却在增速放缓。2018年中国GDP超过美国成为世界第二大经济体。
中美贸易战引发的全球性变化，进一步刺激了全球对华开放力度。2010年，美国出兵驻守中亚，限制中国扩大投资、疯狂开发资源、贬低中国形象、削弱中国在亚太地区的竞争力。2014年，美国推出所谓"蓝皮书"，阐述中美博弈的对策。特朗普政府于2015年8月公布了所谓《冷战略略》，宣称美国应设法确保中国在其领土范围内进行经济自由化。2016年1月，英国、德国、法国、意大利、西班牙等国向美国出口美国农产品。2016年8月，国际货币基金组织(IMF)发布预计2017年中国GDP将超过美国。2017年6月，中国积极参与全球金融危机治理。2018年12月，白宫决定暂停与新冠病毒相关的经济活动，同时再次向世界展示"蓝皮书"，对美国及其盟友提出明年继续开放中国市场、扩大贸易顺差等要求。
# 3.中美两国经济关系演变
中美两国的经济关系也是一个重要的话题。经济规模以美元为单位的GDP数量比中国高出几个数量级，而且贸易顺差比欧美国家都大，这就导致两国之间的经济摩擦愈演愈烈。中美两国经济关系的演变可以分为以下阶段：
## 3.1 中美经济的同步增长
自2005年以来，中美两国经济一直保持着同步增长，由中国的GDP增速较快的优势赶超美国。到2015年底，两国GDP总量分别为7.9万亿美元和6.7万亿美元。
## 3.2 中美经济分裂
2016年底，中国对外贸易结构出现分歧，部分国企改革失败，引起美国的不满。之后，美国试图通过贸易战的方式，捍卫中国共产党的政权，试图将部分工业基础设施转移到美国。中美关系因此出现分裂，美国要求中国停止与美元结算、停止对伊拉克、阿富汗和其他少数国家的援助、和朝鲜的贸易协定。中国因而将经济损失降到了最小。2017年11月，中国外交部发言人姜瑜在记者会上说："美国从来没有向中国政府提供过帮助，哪怕是借助贸易战也只能使他们付出沉重代价。"然而，随着北京方面对美贸易限制的严厉打压，中国外商投资被迫减少。
## 3.3 中美经济脱钩
2018年，美国的国防部长埃斯珀·奥斯特马丁访华，中美贸易紧张局势再次升级。在此背景下，特朗普政府出台了针对中国市场的大规模制裁措施。如在2018年3月、6月、9月，中美双方连续发表声明不承认以色列石油出口禁令等。到2018年12月，特朗普宣布对中国52个重点项目实施制裁。这意味着中国企业在美元资本市场上的投资将受到严格限制。2019年4月，中国央行发行100亿美元支持美元全面贬值的工作目标，此举将使美国更难以购买美元资产。
2019年11月12日，特朗普政府决定结束与中国领导人习近平的双边贸易谈判，其间多次发表联合声明，称中国的行为已构成严重威胁美国利益，必须首先结束这一事件，立即结束一切形式的政治对峙，共同开发一个具有强大国际影响力的中国模式。
# 4.中美贸易战的影响
## 4.1 美国外交政策
美国在中美贸易战中的态度给美国的外交政策带来了深刻的启示。中美贸易战使得美国深感必须做好自己应对贸易战的策略。在贸易战中，美国建立了一个庞大的反制机制，让中国逃避。因此，美国必须彻底改变外交政策，建立自己的经济基础，不断深化合作。首先，美国必须认识到中美贸易战的影响，明白两国间实力悬殊、尖锐矛盾的真相，明白中国对美国经济的巨大影响，而不要为了与之斗争而背弃自己的利益。其次，美国必须做好准备，抢占中国市场先机，拉拢更多的中国企业参与到美国市场，实现美国和中国两个超级大国的整合。第三，美国需要尊重中国的领导层，赞扬中国在经济、文化、军事、科技等领域的领先地位。最后，美国要保持高度警惕，防范中国与美国的全面对抗，不要轻率行动，以免给中国带来损害。
## 4.2 中美关系
在中美关系中，还有很多问题需要深刻分析。比如，在当前的时局下，如何保障两国在国际事务中的霸权地位？如何才能有效地应对日益增长的全球化，避免发生新的贸易战和战略博弈？在与中方磋商中，是否应该把国家安全放在第一位？中国在世界上的声望和实力有多大？
## 4.3 中国的政治责任
中国作为发展中国家，在面临各种挑战时，应当承担起多方面的责任。一方面，中国必须做好实力反制，同时坚持政治正义，尊重国际公约，坚持中国共产党领导。另一方面，中国也应当继承发达国家的成功经验，在经济领域保持领先优势，完善国际金融机制，打造全球化经济圈，搭建更具综合性的金融和贸易体系。除此之外，中国还应当履行自己的法律义务，维护自己的政治权利，落实社会责任，增强全民族凝聚力。
# 5.未来世界局势展望
随着中美贸易战的持续发展，世界经济秩序正在发生剧烈变化。经济全球化加剧，民粹主义盛行，新兴市场受到挤压，贸易战对世界经济的冲击日渐凸显。2018年全球金融危机之后，世界各国也迎来了新的发展阶段，中美贸易争端频发，经济疲软、贸易摩擦加剧、房地产泡沫化、人民币升值风险加大等问题依旧存在。近几年，世界经济出现新的机遇，中国经济正以全新的方式复苏。中国在世界经济中占有重要地位，中美关系也有了新的变化。因此，未来世界局势会发生怎样的变化，取决于人类文明的发展水平、各国的制度环境、以及人类的共同利益。未来几年，随着中美关系恢复正常，经济全球化的进程会加快。美国、中国、欧盟、日本、韩国等国家会加强合作，采取不同的制度和机制，创造更多的合作条件，改善全球关系。而随着贸易战的结束，中美两个超级大国之间的霸权地位也会被削弱。一旦经济全球化和政治民主化的进程加快，世界经济的影响力和权力格局将重新构建。
