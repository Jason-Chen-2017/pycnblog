## 2. 技术原理及概念

### 2.1 基本概念解释

虚拟现实(VR)是一种模拟真实世界的视觉和交互体验的技术。增强现实(AR)是一种利用计算机技术将虚拟信息增强到现实世界中的技术。元宇宙是一种结合虚拟现实和增强现实技术，构建出虚拟世界并将其与现实世界相结合的技术。

### 2.2 技术原理介绍

虚拟现实技术通过计算机模拟出一个虚拟世界，并将用户带入其中。用户可以在其中感受真实世界的感觉，并通过计算机技术实现视觉和交互体验。增强现实技术则是通过计算机技术将虚拟信息投射到现实世界中，并通过现实世界的用户界面实现交互体验。元宇宙技术则是将虚拟现实和增强现实技术融合在一起，构建出一个虚拟世界并将其与现实世界相结合。

### 2.3 相关技术比较

虚拟现实、增强现实和元宇宙技术都属于计算机视觉领域，它们都能够实现对现实世界的模拟和交互体验。虚拟现实技术通常使用头戴式显示器、手柄等设备实现，而增强现实技术通常使用显示器、智能手机等设备实现，而元宇宙技术则使用虚拟现实和增强现实技术融合在一起，构建出一个虚拟世界并将其与现实世界相结合。

    def __init__(self):
        self.window = cv2.VideoCapture(0)
        self.camera = cv2.VideoWriter(0, "mp4", 60, (256, 192))

    def write_image(self, image):
        # convert image to numpy array
        image = np.array(image, dtype=np.uint8)

        # write image to video
        self.camera.write(image)
        self.camera.release()
        self.camera = None

    def play_image(self, image):
        # convert image to numpy array
        image = np.array(image, dtype=np.uint8)

        # play image on screen
        cv2.imshow('image', image)

        # wait for key press
        cv2.waitKey(0)

        # release camera
        self.camera.release()

    def close(self):
        # close window
        self.window.release()

    def keyPressed(self):
        # check for key press
        if cv2.waitKey(1) & 0xFF == ord('w'):
            self.close()
        elif cv2.waitKey(1) & 0xFF == ord('s'):
            self.write_image(self.image)
        elif cv2.waitKey(1) & 0xFF == ord('q'):
            cv2.destroyAllWindows()

    if __name__ == '__main__':
         MedicalOffice()
