     ```python
     def findInfluencePath(graph, start, end):
         queue = deque([start])
         visited = set([start])

         while queue:
             node = queue.popleft()
             if node == end:
                 return True

                 if neighbor not in visited:
                     queue.append(neighbor)
                     visited.add(neighbor)

         return False
     ```
