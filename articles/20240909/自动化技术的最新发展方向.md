**2. 编写一个算法，实现智能家居系统的设备控制。**

**题目：** 编写一个算法，用于控制智能家居系统的设备，如灯光、窗帘、空调等。

**答案：**

```python
class SmartHome:
    def __init__(self):
        self.devices = {
            'light': False,
            'curtain': False,
            'ac': False
        }

    def control_device(self, device, status):
        # 控制设备状态
        # device 为设备名称，如 'light'
        # status 为设备状态，如 True 表示打开，False 表示关闭
        if device in self.devices:
            self.devices[device] = status
            print(f"{device} 已设置为 {status}")
        else:
            print(f"设备 {device} 不存在")

# 示例
smart_home = SmartHome()
smart_home.control_device('light', True)
smart_home.control_device('ac', False)
```

**解析：** 这道题目考查对智能家居系统设备控制的理解和应用。通过定义一个类，实现设备控制功能。
