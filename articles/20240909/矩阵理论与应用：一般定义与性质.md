### 一、矩阵理论的基础概念

#### 1. 矩阵的定义
矩阵是由数字构成的矩形阵列，通常用大写字母表示，例如 \( A = \begin{bmatrix} a_{11} & a_{12} & \ldots & a_{1n} \\ a_{21} & a_{22} & \ldots & a_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ a_{m1} & a_{m2} & \ldots & a_{mn} \end{bmatrix} \)。矩阵的行数称为矩阵的行数（或高度），列数称为矩阵的列数（或宽度）。

#### 2. 矩阵的维度
矩阵的维度由其行数和列数确定，通常表示为 \( m \times n \)，其中 \( m \) 是行数，\( n \) 是列数。

#### 3. 矩阵的元素
矩阵中的每个数字称为矩阵的元素。例如，在矩阵 \( A \) 中，元素 \( a_{ij} \) 表示第 \( i \) 行和第 \( j \) 列的元素。

#### 4. 特殊矩阵
- 对角矩阵：除了主对角线上的元素外，其余元素都为零的矩阵。
- 单位矩阵：主对角线上的元素都是 1，其余元素都是 0 的对角矩阵。
- 稀疏矩阵：大部分元素为零的矩阵。

### 二、矩阵的性质与运算

#### 1. 矩阵的加法和减法
只有当两个矩阵的维度相同时，才能进行矩阵的加法和减法。矩阵加法和减法是对应元素相加或相减。

#### 2. 矩阵的乘法
矩阵乘法的结果是一个新矩阵，其行数等于第一个矩阵的行数，列数等于第二个矩阵的列数。矩阵乘法遵循结合律和分配律。

#### 3. 矩阵的转置
矩阵的转置是将矩阵的行和列互换。例如，如果 \( A \) 是一个 \( m \times n \) 的矩阵，则其转置 \( A^T \) 是一个 \( n \times m \) 的矩阵。

#### 4. 矩阵的行列式
行列式是矩阵的一个重要属性，用于表示矩阵的特性和计算矩阵的逆。行列式的值可以通过拉普拉斯展开或高斯消元法计算。

#### 5. 矩阵的逆
如果一个矩阵是可逆的，则其逆矩阵可以通过高斯消元法或矩阵求逆公式计算。逆矩阵可以用于解线性方程组和计算矩阵乘法的逆。

#### 1. 矩阵乘法
```python
def matrix_multiply(A, B):
    rows_A, cols_A = len(A), len(A[0])
    rows_B, cols_B = len(B), len(B[0])
    if cols_A != rows_B:
        raise ValueError("矩阵维度不匹配")

    result = [[0 for _ in range(cols_B)] for _ in range(rows_A)]
    for i in range(rows_A):
        for j in range(cols_B):
            for k in range(cols_A):
                result[i][j] += A[i][k] * B[k][j]
    return result
```

#### 2. 矩阵行列式计算
```python
def determinant(A):
    n = len(A)
    if n == 1:
        return A[0][0]
    if n == 2:
        return A[0][0] * A[1][1] - A[0][1] * A[1][0]
    
    det = 0
    for j in range(n):
        det += ((-1) ** j) * A[0][j] * determinant([row[:j] + row[j+1:] for row in A[1:]])
    return det
```

```python
def create_adjacency_matrix(graph):
    n = len(graph)
    A = [[0 for _ in range(n)] for _ in range(n)]
    for i, edges in enumerate(graph):
        for j in edges:
            A[i][j] = 1
    return A
```
