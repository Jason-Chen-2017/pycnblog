##### 1. 最小生成树算法

**题目：** 使用 Prim 算法求最小生成树。

**代码：**

```python
from typing import List

def prim_algorithm(edges: List[List[int]]) -> int:
    # 边的数量
    n = len(edges)
    # 最小生成树的权重总和
    total_weight = 0
    # 已选中的顶点
    selected = [False] * n
    # 初始化一个优先队列，用于选择最小权重边
    heap = [(0, 0, 1)]  # (权重，起点，终点)

    while len(heap) > 0:
        # 取出权重最小的边
        weight, start, end = heapq.heappop(heap)
        # 如果终点已选中，则忽略这条边
        if selected[end]:
            continue
        # 将终点加入已选中的顶点
        selected[end] = True
        # 更新总权重
        total_weight += weight
        # 将与终点相连的边加入优先队列
        for next_start, next_end, next_weight in edges[end]:
            if not selected[next_end]:
                heapq.heappush(heap, (next_weight, end, next_end))

    return total_weight
```

**解析：**

Prim 算法是一种贪心算法，用于求无向加权图的最小生成树。算法的基本思想是从一个顶点开始，逐步添加权重最小的边，直到包含所有顶点。

##### 2. 搜索排序数组

**题目：** 二分查找在排序数组中的搜索

**代码：**

```python
def search_sorted_array(arr: List[int], target: int) -> int:
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1
```

**解析：**

该算法利用二分查找的基本原理，在排序数组中查找目标元素。每次比较中点元素与目标值的大小，逐步缩小查找范围，直到找到目标元素或确定其不存在。
