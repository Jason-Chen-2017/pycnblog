#### 1. 如何分析知识付费平台用户的活跃度？

**题目：** 给定一个知识付费平台的用户行为数据，如何分析用户的活跃度？请至少提出两个指标。

**答案：** 用户活跃度可以从以下两个方面进行衡量：

1. **登录频率：** 用户在一定时间范围内登录平台的次数。通常可以用“登录次数/天数”来衡量。
2. **内容消费：** 用户在平台上的内容消费情况，如观看视频时长、阅读文章字数等。可以使用“观看时长/天数”或“阅读字数/天数”等指标。

**示例代码：**

```go
// 假设有一个用户行为数据结构
type UserBehavior struct {
    UserID    int
    LoginDate  time.Time
    ViewDuration float64
    ReadWords   int
}

// 分析登录频率
func calculateLoginFrequency(behaviors []UserBehavior) map[int]float64 {
    result := make(map[int]float64)
    for _, behavior := range behaviors {
        days := behavior.LoginDate.Sub(startingDate).Hours() / 24
        result[behavior.UserID] = float64(len(behavior.Logins)) / days
    }
    return result
}

// 分析内容消费
func calculateContentConsumption(behaviors []UserBehavior) map[int]float64 {
    result := make(map[int]float64)
    for _, behavior := range behaviors {
        result[behavior.UserID] += behavior.ViewDuration + float64(behavior.ReadWords)
    }
    return result
}
```
