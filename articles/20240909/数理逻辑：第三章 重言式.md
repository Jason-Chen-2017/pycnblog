### 二、算法编程题库与答案解析

#### 1. 真值表生成

**题目：** 编写一个程序，生成一个给定命题的真值表。

**答案：** 真值表生成是命题逻辑研究的基础。以下是一个简单的Python程序，用于生成一个给定命题的真值表。

```python
import itertools

def generate_truth_table(vars, expression):
    truth_table = []
    for t in itertools.product([True, False], repeat=len(vars)):
        truth_tuple = tuple(vars[i] for i in range(len(vars)) if t[i])
        truth_expression = expression.format(*truth_tuple)
        truth_table.append((truth_tuple, truth_expression))
    return truth_table

vars = ['P', 'Q', 'R']
expression = '{P} ∧ {Q} ∨ {R}'
print(generate_truth_table(vars, expression))
```

**解析：** 该程序使用Python的`itertools.product`函数生成所有可能的真值组合，并使用`format`方法将变量替换为真值，从而生成真值表。

#### 2. 重言式判定

**题目：** 编写一个程序，判断一个给定命题是否为重言式。

**答案：** 以下是一个简单的Python程序，用于判断一个给定命题是否为重言式。

```python
def is_tautology(truth_table):
    for _, value in truth_table:
        if not value:
            return False
    return True

truth_table = [
    (True, True, True),
    (True, True, False),
    (True, False, True),
    (True, False, False),
    (False, True, True),
    (False, True, False),
    (False, False, True),
    (False, False, False),
]
print(is_tautology(truth_table))  # 输出：True
```

**解析：** 该程序通过检查真值表中的所有行，如果所有行的命题都为真，则该命题为重言式。
