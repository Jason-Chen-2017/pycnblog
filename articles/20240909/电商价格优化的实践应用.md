### 算法编程题库

#### 4. 如何实现价格区间查找？

**题目：** 实现一个函数，给定一个价格区间和一组商品价格，找出符合价格区间的商品。

**代码示例：**

```go
func findProductsInRange(products []int, priceRange [2]int) []int {
    result := []int{}
    for _, product := range products {
        if product >= priceRange[0] && product <= priceRange[1] {
            result = append(result, product)
        }
    }
    return result
}
```

**解析：** 这个函数通过遍历商品价格数组，筛选出符合价格区间的商品，实现价格区间查找。
