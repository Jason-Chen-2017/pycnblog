### 一、面试题

#### 1. 如何解决功能固定性？

**题目：** 在软件开发中，如何避免功能固定性，确保系统能够灵活适应新的需求和变化？

**答案：** 
- **模块化设计：** 采用模块化设计，将系统划分为独立的模块，每个模块负责特定的功能。这样，当需求发生变化时，可以单独修改或替换模块，而不会影响其他部分。
- **抽象和封装：** 提高代码的抽象和封装程度，减少对具体实现细节的依赖。通过接口和抽象类，实现功能的松耦合。
- **持续集成与重构：** 定期进行代码重构和集成测试，确保系统的稳定性和灵活性。

#### 2. 思维僵化的表现有哪些？

**题目：** 在日常工作中，思维僵化的表现有哪些？

**答案：**
- **重复行为：** 坚持固有的工作习惯，不愿意尝试新的方法和工具。
- **缺乏创新：** 对新的想法和观点持怀疑态度，缺乏创新意识。
- **忽视反馈：** 对他人的建议和反馈充耳不闻，固执己见。

### 二、算法编程题

#### 1. 寻找旋转排序数组中的最小值

**题目：** 给定一个可能包含重复元素的旋转排序数组，找出并返回数组中的最小元素。

**示例：** 输入：`[3,4,5,1,2]`，输出：`1`。

**答案：**

```python
class Solution:
    def findMin(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1
        while left < right:
            mid = (left + right) // 2
            if nums[mid] > nums[right]:
                left = mid + 1
            else:
                right = mid
        return nums[left]
```

#### 2. 设计一个满足某人特定需求的数据库

**题目：** 设计一个数据库，能够处理如下查询：
- 插入：向数据库中插入新的用户和他们的需求。
- 查询：根据用户的年龄和需求，查询满足条件的用户。
- 更新：更新特定用户的年龄或需求。

**答案：**

```python
class UserDatabase:
    def __init__(self):
        self.users = []

    def insert(self, user_id: int, age: int, need: str):
        self.users.append({'user_id': user_id, 'age': age, 'need': need})

    def query(self, age_min: int, age_max: int, need: str) -> List[int]:
        result = []
        for user in self.users:
            if age_min <= user['age'] <= age_max and user['need'] == need:
                result.append(user['user_id'])
        return result

    def update(self, user_id: int, age: int = None, need: str = None):
        for user in self.users:
            if user['user_id'] == user_id:
                if age:
                    user['age'] = age
                if need:
                    user['need'] = need
                break
```
