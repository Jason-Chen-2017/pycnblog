**题目：** 给定一个商品价格和销量的数据集，编写一个函数，计算价格与销量之间的相关性。

**答案：** 可以使用皮尔逊相关系数计算价格与销量的相关性。

```python
import numpy as np

def correlation(x, y):
    mean_x = np.mean(x)
    mean_y = np.mean(y)
    std_x = np.std(x)
    std_y = np.std(y)
    num = np.sum((x - mean_x) * (y - mean_y))
    den = std_x * std_y
    return num / den
```
