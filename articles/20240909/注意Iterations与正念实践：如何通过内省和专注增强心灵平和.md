**2. 深度优先搜索：实现一个递归函数，计算一个给定二叉树的所有路径和。**

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def pathSum(root, targetSum):
    def dfs(root, path, pathSum):
        if root is None:
            return
        path.append(root.val)
        pathSum += root.val
        if root.left is None and root.right is None and pathSum == targetSum:
            print("找到一条路径：", path)
        else:
            dfs(root.left, path, pathSum)
            dfs(root.right, path, pathSum)
        path.pop()

    paths = []
    dfs(root, [], 0)
    return paths
```
