#### 6. 编写一个算法，检测数据中心网络中的瓶颈。

**算法描述：** 输入网络流量数据，找出网络中的瓶颈。

```python
from collections import Counter

def find_bottleneck(network_data):
    flow_counts = Counter(network_data)
    max_flow = max(flow_counts.values())
    bottleneck = [k for k, v in flow_counts.items() if v == max_flow]
    return bottleneck
```

**答案：** `find_bottleneck` 函数通过计数方法找到流量最大的节点，即瓶颈。

**解析：** 该算法能够快速识别网络中的流量瓶颈，有助于优化网络性能。
