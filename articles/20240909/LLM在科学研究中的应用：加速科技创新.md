##### 1. 科学计算

**题目：** 编写一个Python函数，计算两个向量的点积。

**答案：** 

```python
def dot_product(v1, v2):
    return sum(x * y for x, y in zip(v1, v2))
```

**解析：** 这个函数使用 `zip` 函数将两个向量对应位置的元素配对，然后使用 `sum` 函数计算这些元素的乘积之和。

##### 3. 自然语言处理

**题目：** 编写一个Python函数，使用NLTK库对给定的文本进行词频统计。

**答案：**

```python
import nltk
from nltk.tokenize import word_tokenize
from collections import Counter

def word_frequency(text):
    tokens = word_tokenize(text)
    return Counter(tokens)
```

**解析：** 这个函数首先使用 `nltk` 的 `word_tokenize` 函数将文本拆分成单词，然后使用 `collections.Counter` 对单词进行计数。
