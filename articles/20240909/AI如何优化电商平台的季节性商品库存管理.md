#### 3. AI优化季节性商品库存管理的算法编程题

**题目：** 编写一个算法，根据历史销售数据和当前时间，预测某季节性商品在未来一段时间内的需求量。

**答案：**

```python
import numpy as np
from sklearn.linear_model import LinearRegression

def predict_demand(history_sales, current_time):
    """
    根据历史销售数据和当前时间，预测未来一段时间内的需求量。

    参数：
    history_sales：列表，历史销售数据，格式为[[时间，销量]，...]
    current_time：当前时间，格式为时间戳

    返回：
    predicted_demand：预测的需求量
    """
    # 将时间转换为独热编码
    X = np.array([[1 if t == current_time else 0 for t in range(len(history_sales))] for history_sales in history_sales])
    # 获取销量数据
    y = np.array([sales for _, sales in history_sales])

    # 使用线性回归模型进行预测
    model = LinearRegression()
    model.fit(X, y)
    predicted_demand = model.predict(np.array([[1]]))

    return predicted_demand[0]

# 示例数据
history_sales = [
    [1, 10],
    [2, 15],
    [3, 20],
    [4, 25],
    [5, 30]
]
current_time = 5

# 预测
predicted_demand = predict_demand(history_sales, current_time)
print("预测的未来需求量为：", predicted_demand)
```

**解析：** 该算法使用线性回归模型根据历史销售数据和时间预测未来需求量。在实际应用中，可以结合更多因素（如天气、促销活动等）进行更准确的预测。
