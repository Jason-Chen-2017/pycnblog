##### 1. 设计一个算法，判断一个单词是否为其他单词的交错词。

**题目：** 给定两个字符串 s1 和 s2，请设计一个算法，判断 s2 是否为 s1 的交错词。例如，s1 = "street", s2 = " devs "，则 s2 是 s1 的交错词。

**答案：** 可以使用动态规划算法来判断 s2 是否为 s1 的交错词。定义一个二维数组 dp，其中 dp[i][j] 表示前 i 个字符的 s1 与前 j 个字符的 s2 是否匹配。

```python
def isScramble(s1, s2):
    if len(s1) != len(s2):
        return False
    if sorted(s1) != sorted(s2):
        return False
    n = len(s1)
    dp = [[False] * (n + 1) for _ in range(n + 1)]
    dp[0][0] = True
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]
            else:
                for k in range(i):
                    dp[i][j] = dp[i][j] or dp[i - k - 1][j - k - 1] and dp[i - k][j - k]
    return dp[n][n]
```

**解析：** 该算法首先检查 s1 和 s2 是否长度相等和排序相同，然后使用动态规划来计算两个字符串是否为交错词。外层循环遍历 s1 的长度 i，内层循环遍历 s2 的长度 j，判断当前字符是否匹配，如果不匹配，则尝试将 s1 的子字符串与 s2 的子字符串进行交错匹配。

##### 2. 设计一个算法，计算一个字符串的最长回文子串。

**题目：** 给定一个字符串 s，请设计一个算法，计算 s 的最长回文子串。

**答案：** 可以使用动态规划算法来计算一个字符串的最长回文子串。定义一个二维数组 dp，其中 dp[i][j] 表示 s 的子字符串 s[i..j] 是否为回文。

```python
def longestPalindrome(s):
    n = len(s)
    dp = [[False] * n for _ in range(n)]
    start = 0
    maxLen = 1
    for i in range(n):
        dp[i][i] = True
        if i < n - 1 and s[i] == s[i + 1]:
            dp[i][i + 1] = True
            start = i
            maxLen = 2
    for i in range(n - 2, -1, -1):
        for j in range(i + 2, n):
            if s[i] == s[j]:
                dp[i][j] = dp[i + 1][j - 1]
                if dp[i][j] and j - i > maxLen:
                    start = i
                    maxLen = j - i + 1
    return s[start: start + maxLen]
```

**解析：** 该算法首先初始化一个二维数组 dp，其中 dp[i][j] 表示 s 的子字符串 s[i..j] 是否为回文。然后，使用两个嵌套循环遍历所有的子字符串，更新 dp 数组。外层循环遍历 s 的长度 i，内层循环遍历 s 的长度 j，判断当前字符是否匹配，并更新 dp[i][j] 的值。最后，返回最长回文子串的起始索引和长度。
