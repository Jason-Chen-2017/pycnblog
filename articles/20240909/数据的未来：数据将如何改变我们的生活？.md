**2. 如何使用K-均值算法进行聚类分析？**

**答案：** K-均值算法是一种基于距离度量的聚类算法，其目标是找到K个中心点，使得每个点与中心的距离之和最小。

```python
# Python示例代码
from random import choice

def kmeans(points, K):
    # 随机初始化K个中心点
    centroids = [choice(points) for _ in range(K)]

    while True:
        # 分配点
        clusters = {}
        for point in points:
            distances = [(centroid, euclidean(point, centroid)) for centroid in centroids]
            closest_centroid, _ = min(distances, key=lambda x: x[1])
            if closest_centroid not in clusters:
                clusters[closest_centroid] = []
            clusters[closest_centroid].append(point)

        # 更新中心点
        new_centroids = {}
        for centroid, points in clusters.items():
            new_centroid = sum(points) / len(points)
            new_centroids[centroid] = new_centroid

        # 判断是否收敛
        if abs(new_centroids - centroids).max() < 1e-4:
            break

        centroids = new_centroids

    return centroids, clusters
```

**3. 如何使用决策树进行分类问题？**

**答案：** 决策树是一种常用的分类算法，它通过一系列的规则来对数据点进行分类。

```python
# Python示例代码
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier

# 加载数据
iris = load_iris()
X, y = iris.data, iris.target

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 创建决策树分类器
clf = DecisionTreeClassifier()

# 训练模型
clf.fit(X_train, y_train)

# 预测
y_pred = clf.predict(X_test)

# 评估模型
accuracy = clf.score(X_test, y_test)
print("Accuracy:", accuracy)
```
