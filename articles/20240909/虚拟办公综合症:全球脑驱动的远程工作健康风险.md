### 算法编程题库及答案解析

#### 5. 编写一个Python函数，计算远程办公员工的平均工作效率。
**题目：** 编写一个Python函数，输入员工每日工作时间和完成的工作量，计算平均工作效率。

**答案：** 

```python
def calculate_efficiency(work_time, work_done):
    if work_time == 0:
        return 0
    return work_done / work_time

# 示例
efficiency = calculate_efficiency(8, 10)
print("Average Efficiency:", efficiency)
```

**解析：** 函数计算工作效率，需要考虑工作时间和完成的工作量，返回二者的比值。

#### 6. 编写一个Python函数，判断远程办公员工是否存在健康风险。
**题目：** 编写一个Python函数，根据员工的健康数据（如视力疲劳、焦虑抑郁等指标），判断员工是否存在健康风险。

**答案：**

```python
def has_health_risk(data):
    # 假设健康数据阈值
    thresholds = {'fatigue': 5, 'depression': 3}
    risk_level = 0
    
    for symptom, value in data.items():
        if value >= thresholds[symptom]:
            risk_level += 1
    
    return risk_level > 0

# 示例
data = {'fatigue': 4, 'depression': 2}
print("Has Health Risk:", has_health_risk(data))
```

**解析：** 函数通过判断员工的健康数据是否超过阈值，计算风险等级，返回是否存在健康风险。
