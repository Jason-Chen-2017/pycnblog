#### 2.2 算法编程题

**题目1：** 如何用Go语言实现一个有序链表？

**答案：** 使用Go语言实现有序链表，可以参考以下代码：

```go
package main

import "fmt"

type ListNode struct {
    Val  int
    Next *ListNode
}

func main() {
    head := &ListNode{Val: 1}
    head.Next = &ListNode{Val: 3}
    head.Next.Next = &ListNode{Val: 5}

    printList(head)
}

func printList(head *ListNode) {
    for head != nil {
        fmt.Printf("%d ", head.Val)
        head = head.Next
    }
    fmt.Println()
}
```

**解析：** 该代码定义了一个 `ListNode` 结构体，表示链表的节点。通过创建节点并设置节点的 `Next` 属性，可以构建一个有序链表。在 `main` 函数中，创建了一个有序链表，并使用 `printList` 函数打印链表中的元素。

**题目2：** 如何用Python实现快速排序算法？

**答案：** 使用Python实现快速排序算法，可以参考以下代码：

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

arr = [3, 6, 8, 10, 1, 2, 1]
sorted_arr = quicksort(arr)
print(sorted_arr)
```

**解析：** 该代码定义了一个 `quicksort` 函数，用于实现快速排序算法。函数首先判断数组长度是否小于等于1，如果是，直接返回数组。否则，选择一个中间值作为枢轴（pivot），将数组分为小于、等于和大于枢轴的三个子数组，然后对子数组递归调用 `quicksort` 函数，最后将排序好的子数组合并。
