def anonymize_id_card(id_card):

    对身份证号码进行匿名化处理。

**3.2 数据加密**

**问题**：如何使用AES算法对用户数据进行加密？

**解决方案**：

```python
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad

def encrypt_data(data, key):
    """
    使用AES算法对用户数据进行加密。
    :param data: 待加密的明文字符串。
    :param key: 加密密钥。
    :return: 加密后的数据。
    """
    cipher = AES.new(key, AES.MODE_CBC)
    ct_bytes = cipher.encrypt(pad(data.encode('utf-8'), AES.block_size))
    iv = cipher.iv
    return iv + ct_bytes
```

**解析**：创建AES加密对象，使用CBC模式对数据加密，并使用Padding算法填充数据。

from Crypto.Cipher import AES
from Crypto.Util.Padding import pad

def encrypt_password(plaintext):
    """
    对用户输入的明文密码进行加密。
    :param plaintext: 明文密码（字符串）。

#### 题目3：数据访问控制

**问题描述**：设计一个基于角色的数据访问控制函数，根据用户角色和操作类型判断是否有权限访问特定数据。

```python
def check_permission(role, operation):
    """
    检查用户角色是否有权限执行特定操作。
    :param role: 用户角色（字符串）。
    :param operation: 操作类型（字符串）。
    :return: 是否有权限（布尔值）。
    """
    role_permissions = {
        "admin": ["read", "write", "delete"],
        "editor": ["read", "write"],
        "viewer": ["read"]
    }
    return operation in role_permissions.get(role, [])

# 示例
role = "editor"
operation = "write"
permission = check_permission(role, operation)
print(f"{role}角色是否有权限{operation}操作：{permission}")
```
