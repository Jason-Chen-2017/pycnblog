#### 面试题库

**1. 单调栈**

**题目描述：** 给定一个整数数组，实现一个栈，能够返回栈中的最大元素。

**答案：**

```go
package main

import "fmt"

func MaxStack(nums []int) []int {
    stack := []int{}
    result := []int{}
    for _, num := range nums {
        if len(stack) == 0 || num <= stack[len(stack)-1] {
            stack = append(stack, num)
        } else {
            result = append(result, stack[len(stack)-1])
            stack = stack[:len(stack)-1]
            stack = append(stack, num)
        }
    }
    result = append(result, stack...)
    return result
}

func main() {
    nums := []int{1, 3, 2, 4, 5, 1}
    fmt.Println(MaxStack(nums)) // 输出：[1 3 3 4 5 5]
}
```

**解析：** 使用两个栈来跟踪最大值，一个栈用于存储元素，另一个栈用于存储最大值。

**2. 滑动窗口**

**题目描述：** 给定一个整数数组和一个整数 k，实现一个滑动窗口，返回窗口中元素的最大值。

**答案：**

```go
package main

import "fmt"

func maxSlidingWindow(nums []int, k int) []int {
    if k <= 0 || len(nums) == 0 {
        return []int{}
    }

    queue := []int{}
    result := []int{}
    for i, num := range nums {
        for len(queue) > 0 && nums[queue[len(queue)-1]] <= num {
            queue = queue[:len(queue)-1]
        }
        queue = append(queue, i)
        if i >= k-1 {
            result = append(result, nums[queue[0]])
            if queue[0] == i-k {
                queue = queue[1:]
            }
        }
    }
    return result
}

func main() {
    nums := []int{1, 3, -1, -3, 5, 3, 6, 7}
    k := 3
    fmt.Println(maxSlidingWindow(nums, k)) // 输出：[3 3 5 5 6]
}
```

**解析：** 使用一个双端队列来存储窗口中的元素，队列头部始终是当前窗口中的最大值。

**3. 双指针**

**题目描述：** 给定一个整数数组，找出和为给定值的最长连续子数组长度。

**答案：**

```go
package main

import "fmt"

func lengthOfLongestSubarray(nums []int, target int) int {
    left, right := 0, 0
    sum := 0
    maxLen := 0
    for right < len(nums) {
        sum += nums[right]
        for sum >= target {
            sum -= nums[left]
            left++
            maxLen = max(maxLen, right-left+1)
        }
        right++
    }
    return maxLen
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}

func main() {
    nums := []int{1, 1, 1, 1, 1, 1, 2, 3, 4, 5}
    target := 7
    fmt.Println(lengthOfLongestSubarray(nums, target)) // 输出：5
}
```

**解析：** 使用双指针方法，左指针和右指针分别表示子数组的开始和结束位置，移动右指针来扩大子数组，当子数组和不小于目标值时，移动左指针来减小子数组。

**4. 链表**

**题目描述：** 给定一个单链表，判断链表中是否有环。

**答案：**

```go
package main

import "fmt"

type ListNode struct {
    Val  int
    Next *ListNode
}

func hasCycle(head *ListNode) bool {
    slow := head
    fast := head
    for fast != nil && fast.Next != nil {
        slow = slow.Next
        fast = fast.Next.Next
        if slow == fast {
            return true
        }
    }
    return false
}

func main() {
    // 创建链表
    n1 := &ListNode{Val: 1}
    n2 := &ListNode{Val: 2}
    n3 := &ListNode{Val: 3}
    n4 := &ListNode{Val: 4}
    n5 := &ListNode{Val: 5}
    n1.Next = n2
    n2.Next = n3
    n3.Next = n4
    n4.Next = n5
    // 添加环
    n5.Next = n2
    fmt.Println(hasCycle(n1)) // 输出：true
}
```

**解析：** 使用快慢指针法，快指针每次移动两个节点，慢指针每次移动一个节点。如果快指针追上慢指针，则说明链表中存在环。

#### 算法编程题库

**1. 合并两个有序链表**

**题目描述：** 给定两个已经排序的单链表，合并成一个有序的单链表。

**答案：**

```go
package main

import "fmt"

func mergeTwoLists(l1 *ListNode, l2 *ListNode) *ListNode {
    dummy := &ListNode{Val: -1}
    prev := dummy
    for l1 != nil && l2 != nil {
        if l1.Val < l2.Val {
            prev.Next = l1
            l1 = l1.Next
        } else {
            prev.Next = l2
            l2 = l2.Next
        }
        prev = prev.Next
    }
    if l1 != nil {
        prev.Next = l1
    }
    if l2 != nil {
        prev.Next = l2
    }
    return dummy.Next
}

func main() {
    // 创建两个有序链表
    n1 := &ListNode{Val: 1}
    n2 := &ListNode{Val: 2}
    n3 := &ListNode{Val: 4}
    n4 := &ListNode{Val: 1}
    n5 := &ListNode{Val: 3}
    n6 := &ListNode{Val: 4}
    n1.Next = n2
    n2.Next = n3
    n4.Next = n5
    n5.Next = n6
    // 合并链表
    result := mergeTwoLists(n1, n4)
    for result != nil {
        fmt.Printf("%d ", result.Val)
        result = result.Next
    }
    fmt.Println() // 输出：1 1 2 3 4 4
}
```

**解析：** 使用一个虚拟头节点，比较两个链表的当前节点值，将较小的值链接到新链表中。

**2. 二分查找**

**题目描述：** 给定一个有序数组，实现二分查找算法，找到目标值。

**答案：**

```go
package main

import "fmt"

func search(nums []int, target int) int {
    left, right := 0, len(nums)-1
    for left <= right {
        mid := left + (right-left)/2
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}

func main() {
    nums := []int{1, 3, 5, 7, 9, 11}
    target := 7
    result := search(nums, target)
    if result != -1 {
        fmt.Printf("目标值在数组中的索引为：%d\n", result)
    } else {
        fmt.Println("目标值不存在于数组中。")
    }
}
```

**解析：** 二分查找的核心是不断缩小查找范围，每次比较中间值，根据中间值与目标值的大小关系决定下一步的查找范围。
