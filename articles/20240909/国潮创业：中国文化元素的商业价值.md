#### 3. 编写一个算法，找出字符串中的最长不含有重复字符的子串。

**题目：** 编写一个算法，找出字符串中的最长不含有重复字符的子串。

**答案：** 可以使用滑动窗口的方法求解。

```python
def length_of_longest_substring(s):
    if not s:
        return 0

    left, right = 0, 0
    max_len = 0
    visited = set()

    while right < len(s):
        if s[right] in visited:
            visited.remove(s[left])
            left += 1
        else:
            visited.add(s[right])
            max_len = max(max_len, right - left + 1)
            right += 1

    return max_len
```

**解析：** 该算法通过维护一个滑动窗口，记录当前不含有重复字符的子串长度。当窗口中包含重复字符时，将窗口左边界向右移动，直到窗口中不包含重复字符。

#### 4. 设计一个算法，计算数组中两个元素的最大距离。

**题目：** 设计一个算法，计算数组中两个元素的最大距离。

**答案：** 可以使用一次遍历的方法求解。

```python
def max_distance(nums):
    if len(nums) < 2:
        return 0

    max_diff = nums[1] - nums[0]
    min_val = nums[0]

    for i in range(1, len(nums)):
        max_diff = max(max_diff, nums[i] - min_val)
        min_val = min(min_val, nums[i])

    return max_diff
```

**解析：** 该算法通过一次遍历，记录当前最小值和最大距离。每次更新最大距离时，比较当前元素与最小值的差值，如果差值更大，则更新最大距离。
