### 2. 空中自行车的算法编程题

#### 2.1 编写一个函数，计算给定速度和时间的空中自行车行驶距离。

**题目：** 编写一个函数`calculateDistance(speed float64, time float64) float64`，计算给定速度和时间下，空中自行车行驶的距离。

**答案：** 

```python
def calculateDistance(speed, time):
    return speed * time

# 示例调用
distance = calculateDistance(50.0, 2.0)  # 假设速度为50公里/小时，时间为2小时
print(f"行驶距离为：{distance}公里")
```

**解析：** 这个函数根据速度和时间直接计算距离，利用基本的物理学公式：距离 = 速度 × 时间。

#### 2.2 编写一个函数，计算空中自行车在特定风速下的实际速度。

**题目：** 编写一个函数`calculateActualSpeed(theoreticalSpeed float64, windSpeed float64) float64`，计算在给定理论速度和风速下的实际速度。

**答案：** 

```python
def calculateActualSpeed(theoreticalSpeed, windSpeed):
    if windSpeed > theoreticalSpeed:
        return 0
    else:
        return theoreticalSpeed - windSpeed

# 示例调用
actualSpeed = calculateActualSpeed(60.0, 10.0)  # 假设理论速度为60公里/小时，风速为10公里/小时
print(f"实际速度为：{actualSpeed}公里/小时")
```

**解析：** 这个函数根据风速调整理论速度，如果风速大于理论速度，则实际速度为0，否则实际速度为理论速度减去风速。

#### 2.3 编写一个函数，预测空中自行车在特定路线上的行驶时间。

**题目：** 编写一个函数`calculateTimeToReachDestination(distance float64, speed float64) float64`，计算给定距离和速度下，空中自行车到达目的地所需的时间。

**答案：** 

```python
def calculateTimeToReachDestination(distance, speed):
    if speed <= 0:
        return -1
    else:
        return distance / speed

# 示例调用
time = calculateTimeToReachDestination(100.0, 50.0)  # 假设距离为100公里，速度为50公里/小时
print(f"到达目的地所需时间为：{time}小时")
```

**解析：** 这个函数利用距离和速度计算时间，如果速度小于或等于0，则返回-1，表示无法计算。

**题目：** 编写一个函数`compareTravelTimes(hyperloopTime float64, bicycleTime float64) string`，如果超级高铁的时间小于或等于自行车的时间，则返回“超级高铁更快”，否则返回“空中自行车更快”。

**答案：** 

```python
def compareTravelTimes(hyperloopTime, bicycleTime):
    if hyperloopTime <= bicycleTime:
        return "超级高铁更快"
    else:
        return "空中自行车更快"

# 示例调用
travelMethod = compareTravelTimes(0.5, 1.0)  # 假设超级高铁时间为0.5小时，自行车时间为1小时
print(f"更快的交通方式是：{travelMethod}")
```

**解析：** 这个函数通过比较两种交通方式的行驶时间，返回更快的方式。在实际应用中，需要根据具体的距离和速度数据调用这个函数。
