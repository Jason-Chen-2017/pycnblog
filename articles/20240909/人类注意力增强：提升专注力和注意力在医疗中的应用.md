## 算法编程题库

### 6. 编写一个算法，用于计算一个给定数字序列的平均注意力持续时间。

**答案：** 

```python
def average_attention_duration(durations):
    total_duration = 0
    for duration in durations:
        total_duration += duration
    return total_duration / len(durations)

durations = [5, 7, 10, 3, 6]
print(average_attention_duration(durations))
```

**解析：** 该算法通过遍历给定数字序列，计算所有数字的总和，然后除以序列的长度，得到平均注意力持续时间。

### 7. 编写一个算法，用于识别给定数字序列中的注意力峰值。

**答案：** 

```python
def find_attention_peaks(durations, threshold):
    peaks = []
    for i in range(1, len(durations) - 1):
        if durations[i] > durations[i - 1] and durations[i] > durations[i + 1] and durations[i] > threshold:
            peaks.append(i)
    return peaks

durations = [3, 5, 8, 6, 10, 7, 4]
threshold = 5
print(find_attention_peaks(durations, threshold))
```

**解析：** 该算法通过遍历数字序列，判断当前元素是否大于其前后元素，并满足阈值条件，若满足则将其索引添加到 peaks 列表中。

### 8. 编写一个算法，用于优化注意力分配策略，以提高医疗诊断的准确性。

**答案：** 

```python
def optimize_attention_allocation(attention_scores, max_attention):
    allocation = [0] * len(attention_scores)
    total_attention = 0

    for i, score in enumerate(attention_scores):
        if total_attention + score <= max_attention:
            allocation[i] = score
            total_attention += score
        else:
            break

    return allocation

attention_scores = [3, 5, 2, 7, 4]
max_attention = 12
print(optimize_attention_allocation(attention_scores, max_attention))
```

**解析：** 该算法通过遍历注意力分数，依次将分数分配给每个元素，直到总注意力超过最大限制，实现注意力分配的优化。
