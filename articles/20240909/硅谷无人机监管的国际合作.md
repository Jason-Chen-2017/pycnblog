**题目描述：** 针对硅谷无人机监管的国际合作，设计一个算法来判断无人机是否在合法的高度范围内飞行。已知以下条件：
- 硅谷的城市区域限制飞行高度为 300 英尺。
- 在城市周边，飞行高度限制为 400 英尺。
- 无人机必须在飞行前进行注册，并获取一个唯一的 ID。
- 需要检查无人机的高度是否符合规定的区域限制。

**输入：**
1. 无人机的高度（英尺）。
2. 无人机的 ID。
3. 无人机的位置（城市区域或周边区域）。

**输出：**
1. 如果高度在合法范围内，返回“合法”。
2. 如果高度超出限制，返回“非法：超出高度限制”。

**算法描述：**
```go
func checkFlightHeight(height int, id string, location string) string {
    if location == "城市区域" && height > 300 {
        return "非法：超出城市区域高度限制"
    } else if location == "周边区域" && height > 400 {
        return "非法：超出周边区域高度限制"
    }
    return "合法"
}
```

**解析：**
这个算法首先检查无人机的高度是否符合城市区域或周边区域的高度限制。如果高度超出限制，则返回非法状态，并指出具体的原因。否则，返回合法状态。
