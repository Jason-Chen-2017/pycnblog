#### 2. 如何实现一个基于词嵌入的聊天机器人？

**题目：** 使用词嵌入技术实现一个简单的聊天机器人，能够理解用户输入并给出相应的回复。

**答案：**
```python
import gensim

# 加载预训练的词嵌入模型
model = gensim.models.KeyedVectors.load_word2vec_format("GoogleNews-vectors-negative300.bin", binary=True)

# 假设用户输入为 "我喜欢吃苹果"
输入词语 = ["我", "喜欢", "吃", "苹果"]

# 计算输入词语的平均词向量
输入向量 = np.mean([model[word] for word in 输入词语 if word in model], axis=0)

# 搜索相似的句子
相似句子 = model.most_similar(输入向量, topn=5)

# 输出相似句子
print("相似的句子：",相似句子)
```

**解析：** 这道题目考察了使用词嵌入技术实现聊天机器人的基本方法。
