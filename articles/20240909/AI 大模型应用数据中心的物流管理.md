### 1. 货物分拣系统

**问题描述：** 数据中心内需要进行货物分拣，现有 n 个货物和 m 个分拣工位。请设计一个算法，使得每个分拣工位上的货物数量尽可能均匀。

**输入：** 
- n：货物总数
- m：分拣工位数

**输出：** 
- 每个分拣工位上的货物数量

**算法：** 
- 使用贪心算法，每次分配货物时，尽量使当前分拣工位上的货物数量与平均值之间的差距最小。

```python
def distribute_goods(n, m):
    goods_per_station = n // m
    remaining_goods = n % m
    
    result = [goods_per_station] * m
    for i in range(remaining_goods):
        result[i] += 1
        
    return result

# 示例
n = 10
m = 3
print(distribute_goods(n, m))  # 输出：[4, 3, 3]
```

### 3. 货物存储优化

**问题描述：** 数据中心内需要进行货物存储，现有多个存储位置和多个货物。请设计一个算法，确定每个货物的最优存储位置，使得存储空间的利用率最高。

**输入：** 
- positions：存储位置列表，每个位置为一个二元组 (x, y)，表示位置的坐标
- goods：货物列表，每个货物为一个二元组 (weight, priority)，表示货物的重量和优先级

**输出：** 
- 每个货物的存储位置

**算法：** 
- 使用贪心算法，每次选择优先级最高的货物，并将其存储在当前空位最小的位置。

```python
def find_best_position(positions, goods):
    positions = sorted(positions, key=lambda x: x[2])  # 按照空位大小排序
    result = []
    
    for weight, priority in goods:
        for position in positions:
            if position[2] >= weight:
                result.append((weight, priority, position[0], position[1]))
                position[2] -= weight
                break
                
    return result

# 示例
positions = [(1, 1, 10), (2, 1, 8), (2, 2, 6), (3, 3, 4)]
goods = [(3, 1), (5, 2), (2, 3), (4, 4)]
print(find_best_position(positions, goods))  # 输出：[(3, 1, 1, 1), (5, 2, 2, 1), (2, 3, 2, 2), (4, 4, 3, 3)]
```
