**实例：** 使用Python编写一个简单的爬虫，获取某电商网站上某个商品的当前价格：

```python

def get_price(url):
    headers = {'User-Agent': 'Mozilla/5.0'}

    price_element = soup.find('span', {'class': 'price'})
    price = price_element.text
    return price

```

#### 7. 如何计算商品价格的加权平均数？

**题目：** 请描述如何计算商品价格的加权平均数。

**答案：** 计算商品价格的加权平均数的公式为：

\[ \text{加权平均数} = \frac{\sum_{i=1}^{n} (w_i \times p_i)}{\sum_{i=1}^{n} w_i} \]

其中，\(w_i\) 为权重，\(p_i\) 为第 \(i\) 个商品的价格。

**实例：** 假设有3个商品，价格分别为100元、200元和300元，权重分别为1、2和3，计算加权平均数：

```python
weights = [1, 2, 3]
prices = [100, 200, 300]

weighted_average = sum(w * p for w, p in zip(weights, prices)) / sum(weights)
print(weighted_average)
```

#### 14. 如何评估商品价格预测模型的性能？

**题目：** 请描述如何评估商品价格预测模型的性能。

**答案：** 评估商品价格预测模型的性能通常可以从以下几个方面进行：

1. **准确率：** 准确率是指模型预测正确的比例，计算公式为：
   \[ \text{准确率} = \frac{\text{预测正确数量}}{\text{总预测数量}} \]
2. **均方误差（MSE）：** 均方误差是指预测值与实际值之间的平均误差的平方，计算公式为：
   \[ \text{MSE} = \frac{1}{n}\sum_{i=1}^{n} (\hat{y_i} - y_i)^2 \]
   其中，\( \hat{y_i} \) 是预测值，\( y_i \) 是实际值，\( n \) 是总样本数量。
3. **决定系数（R²）：** 决定系数是指模型解释能力的好坏，计算公式为：
   \[ \text{R}^2 = 1 - \frac{\sum_{i=1}^{n} (\hat{y_i} - y_i)^2}{\sum_{i=1}^{n} (y_i - \bar{y})^2} \]
   其中，\( \bar{y} \) 是实际值的平均值。
4. **召回率（Recall）：** 召回率是指模型能够召回实际正例的比例，计算公式为：
   \[ \text{召回率} = \frac{\text{预测正确且实际为正例的数量}}{\text{实际为正例的总数量}} \]
5. **精确率（Precision）：** 精确率是指模型预测正确的比例，计算公式为：
   \[ \text{精确率} = \frac{\text{预测正确且实际为正例的数量}}{\text{预测为正例的总数量}} \]
