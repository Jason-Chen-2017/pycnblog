#### 2.1. 最长公共子序列

**题目描述：** 给定两个字符串，找出它们的最长公共子序列。

**答案解析：**

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```

#### 2.2. 最长连续序列

**题目描述：** 给定一个整数数组，找出其中最长连续序列的长度。

**答案解析：**

```python
def longest_consecutive_sequence(nums):
    nums = set(nums)
    longest = 0

    for num in nums:
        if num - 1 not in nums:
            curr = num
            while curr in nums:
                curr += 1
            longest = max(longest, curr - num)

    return longest
```

```python
def min_path_sum(grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i - 1][j - 1]
```

**题目描述：** 给定两个单词（beginWord 和 endWord），以及一个字典 wordList，找出从 beginWord 到 endWord 的最小变换次数。

```python
from collections import deque

def ladder_length(beginWord, endWord, wordList):
    words = set(wordList)
    queue = deque([beginWord])
    steps = 1

    while queue:
        for _ in range(len(queue)):
            word = queue.popleft()
            if word == endWord:
                return steps

            for i in range(len(word)):
                tmp = word[:i] + '*' + word[i + 1:]
                for j in range(len(wordList)):
                    if wordList[j].find(tmp) == 0 and wordList[j] not in queue:
                        queue.append(wordList[j])

        steps += 1

    return 0
```
