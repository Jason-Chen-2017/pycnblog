#### 1. 题目：编写一个函数，去除字符串中的重复字符。

**答案：** 可以使用哈希表（如 Go 语言的 map）来记录已经出现的字符，遍历字符串，将未记录的字符加入结果字符串。

```go
func removeDuplicates(s string) string {
    charMap := make(map[rune]bool)
    var result strings.Builder
    for _, char := range s {
        if _, ok := charMap[char]; !ok {
            result.WriteRune(char)
            charMap[char] = true
        }
    }
    return result.String()
}
```

**解析：** 该函数通过哈希表记录已出现的字符，遍历字符串，将未记录的字符写入结果字符串，从而去除重复字符。

#### 2. 题目：给定一个整数数组，找到所有重复的数字。

**答案：** 可以使用哈希表记录每个数字出现的次数，然后遍历哈希表，找出出现次数大于 1 的数字。

```go
func findDuplicates(nums []int) []int {
    numMap := make(map[int]int)
    var duplicates []int
    for _, num := range nums {
        numMap[num]++
        if numMap[num] > 1 {
            duplicates = append(duplicates, num)
        }
    }
    return duplicates
}
```

**解析：** 该函数通过哈希表记录每个数字出现的次数，如果出现次数大于 1，则将该数字加入结果数组。

#### 3. 题目：给定一个整数数组，找出所有缺失的数字。

```go
func findMissingNumbers(nums []int) []int {
    numMap := make(map[int]bool)
    for _, num := range nums {
        numMap[num] = true
    }
    var missing []int
    for i := 1; i <= len(nums); i++ {

        }
    }

}
```

**解析：** 该函数通过哈希表记录数组中出现的数字，然后遍历哈希表，找出不在数组中的数字，即缺失的数字。
