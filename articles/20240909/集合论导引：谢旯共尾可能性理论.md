#### 一、集合论的典型问题/面试题库

**问题1：什么是集合？**
- **答案：** 集合是由确定的元素组成的整体，这些元素可以是任意的对象，如数字、字母、图形等。集合用大写字母表示，如A、B等。
- **解析：** 集合是数学中最基本的概念之一，理解集合的定义对于后续学习至关重要。

**问题2：如何表示集合？**
- **答案：** 集合可以用列举法或描述法表示。
    - 列举法：将集合的所有元素一一列举出来，如A={1, 2, 3}。
    - 描述法：用描述性语句来表示集合的元素特征，如B={x | x是自然数且x小于5}。
- **解析：** 列举法和描述法是集合表示的两种常见方法，根据实际情况选择合适的表示方式。

**问题3：集合的基本运算有哪些？**
- **答案：** 集合的基本运算包括并集、交集、差集和补集。
    - 并集（A∪B）：包含A和B中所有元素的集合。
    - 交集（A∩B）：包含A和B中共有元素的集合。
    - 差集（A-B）：包含A中有而B中没有的元素的集合。
    - 补集（A'）：包含全集U中不属于A的元素的集合。
- **解析：** 集合运算在解决实际问题时具有重要意义，理解并掌握集合的基本运算对于提高数学能力非常有帮助。

#### 二、算法编程题库

**问题4：编写程序，实现两个集合的并集、交集和差集。**
- **代码示例：**
```python
def union(A, B):
    return A | B

def intersection(A, B):
    return A & B

def difference(A, B):
    return A - B

# 测试代码
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}
print(union(A, B))    # 输出 {1, 2, 3, 4, 5, 6}
print(intersection(A, B))  # 输出 {3, 4}
print(difference(A, B))   # 输出 {1, 2}
```

**问题5：编写程序，找出两个集合的对称差集。**

```python
def symmetric_difference(A, B):
    return (A - B) | (B - A)

# 测试代码
```
