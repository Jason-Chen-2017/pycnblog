## 二、算法编程题库

### 1. 编写一个算法，实现对一个数组的逆序排列。
**答案：** 可以使用冒泡排序、快速排序等算法实现。以下是一个使用冒泡排序的示例代码：

```python
def reverse_array(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 2. 编写一个算法，找出数组中重复出现的元素。
**答案：** 可以使用哈希表、排序等算法实现。以下是一个使用哈希表的示例代码：

```python
def find_duplicates(arr):
    seen = set()
    duplicates = []
    for num in arr:
        if num in seen:
            duplicates.append(num)
        else:
            seen.add(num)
    return duplicates
```
