### 二、算法编程题库：智慧管理与算法应用

**1. 字节跳动：判断两个链表是否相交**

**题目描述：** 给定两个单链表，判断这两个链表是否相交。

**输入：** 

- 第一个链表的头节点：`ListNode l1`
- 第二个链表的头节点：`ListNode l2`

**输出：** 

- 如果相交，返回相交节点；否则，返回 `null`。

**示例：**

```python
# 输入：
# l1: 1 -> 2 -> 3 -> 7 -> 8 -> 10
# l2: 4 -> 5 -> 6 -> 7 -> 8 -> 10

# 输出：
# 相交节点为：7
```

**答案：**

```python
# Python 代码实现
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None

def getIntersectionNode(headA, headB):
    if not headA or not headB:
        return None

    pA, pB = headA, headB
    lenA, lenB = 0, 0

    while pA:
        lenA += 1
        pA = pA.next

    while pB:
        lenB += 1
        pB = pB.next

    pA, pB = headA, headB
    if lenA > lenB:
        for _ in range(lenA - lenB):
            pA = pA.next
    else:
        for _ in range(lenB - lenA):
            pB = pB.next

    while pA and pB:
        if pA == pB:
            return pA
        pA = pA.next
        pB = pB.next

    return None

# 测试
l1 = ListNode(1)
l1.next = ListNode(2)
l1.next.next = ListNode(3)
l1.next.next.next = ListNode(7)
l1.next.next.next.next = ListNode(8)
l1.next.next.next.next.next = ListNode(10)

l2 = ListNode(4)
l2.next = ListNode(5)
l2.next.next = ListNode(6)
l2.next.next.next = ListNode(7)
l2.next.next.next.next = ListNode(8)
l2.next.next.next.next.next = ListNode(10)

intersection_node = getIntersectionNode(l1, l2)
if intersection_node:
    print("相交节点值为：", intersection_node.val)
else:
    print("不相交")
```

**解析：** 该题主要考察链表相交问题的解决方法。通过先计算两个链表的长度，然后让较长的链表先走长度差值，再同时遍历两个链表，找到相交节点。

**2. 京东：二分查找**

**题目描述：** 在一个有序数组中，找出目标值的位置。

**输入：** 

- 数组 `nums`：`[1, 3, 5, 6]`
- 目标值 `target`：`5`

**输出：** 

- 目标值的位置：`2`

**示例：**

```python
# 输入：
# nums = [1, 3, 5, 6]
# target = 5

# 输出：
# 位置：2
```

**答案：**

```python
# Python 代码实现
def search(nums, target):
    left, right = 0, len(nums) - 1

    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1

# 测试
nums = [1, 3, 5, 6]
target = 5
result = search(nums, target)
print("位置：", result)
```

**解析：** 该题是经典的二分查找问题。通过不断缩小查找范围，找到目标值的位置。如果目标值在数组中，返回其位置；否则，返回 `-1`。

**3. 美团：排序算法**

**题目描述：** 对一个无序数组进行排序。

**输入：**

- 数组 `arr`：`[3, 1, 4, 1, 5, 9, 2, 6, 5]`

**输出：**

- 排序后的数组：`[1, 1, 2, 3, 4, 5, 5, 6, 9]`

**示例：**

```python
# 输入：
# arr = [3, 1, 4, 1, 5, 9, 2, 6, 5]

# 输出：
# 排序后的数组：[1, 1, 2, 3, 4, 5, 5, 6, 9]
```

**答案：**

```python
# Python 代码实现
def quick_sort(arr):
    if len(arr) <= 1:
        return arr

    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]

    return quick_sort(left) + middle + quick_sort(right)

# 测试
arr = [3, 1, 4, 1, 5, 9, 2, 6, 5]
sorted_arr = quick_sort(arr)
print("排序后的数组：", sorted_arr)
```

**解析：** 该题采用快速排序算法实现，通过递归地对数组进行划分和排序，最终得到有序数组。
