### 4. 算法编程题库及解析

#### 4.1 实体识别

**题目：** 编写一个实体识别算法，识别电商搜索中的商品实体。

**答案：** 实体识别是一种从文本中提取出具有特定意义的实体（如商品名称、品牌、分类等）的技术。以下是一个简单的实体识别算法示例：

```python
import spacy

nlp = spacy.load("en_core_web_sm")

def extract_entities(text):
    doc = nlp(text)
    entities = []
    for ent in doc.ents:
        entities.append({"text": ent.text, "label": ent.label_})
    return entities

text = "I want to buy a new iPhone 13 Pro Max."
print(extract_entities(text))
```

#### 4.2 商品关系抽取

**题目：** 编写一个商品关系抽取算法，从电商搜索中的商品描述中提取出商品之间的关系。

**答案：** 商品关系抽取是一种从文本中提取出商品之间关系（如品牌、分类、配件等）的技术。以下是一个简单的商品关系抽取算法示例：

```python
import spacy
from spacy.tokens import Token

nlp = spacy.load("en_core_web_sm")

def extract_relations(text):
    doc = nlp(text)
    relations = []
    for token in doc:
        if token.dep_ == "compound":
            relations.append({"head": token.head.text, "dep": token.dep_, "child": token.text})
    return relations

text = "The iPhone 13 Pro Max has a great camera."
print(extract_relations(text))
```
