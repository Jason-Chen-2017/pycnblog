#### **二、算法编程题库**

**1. 计算字符串中回文字符串的数量**

**题目：** 给定一个字符串，请设计一个算法来计算其中回文字符串的数量。一个字符串是回文字符串，当且仅当它的正向和反向都相同。

**输入：** `s = "abba"`

**输出：** `4`（"a"、"b"、"b"、"abba"` 都是回文字符串）

**答案：**
```python
def countSubstrings(s: str) -> int:
    n = len(s)
    count = 0
    for i in range(n):
        # 以i为中心，向左右扩展
        count += expandAroundCenter(s, i, i)  # 奇数长度的回文
        count += expandAroundCenter(s, i, i + 1)  # 偶数长度的回文
    return count

def expandAroundCenter(s: str, left: int, right: int) -> int:
    count = 0
    while left >= 0 and right < len(s) and s[left] == s[right]:
        count += 1
        left -= 1
        right += 1
    return count
```

**2. 判断字符串是否是合法括号序列**

**题目：** 给定一个字符串，请判断它是否是一个合法的括号序列。一个合法的括号序列满足以下条件：

- 括号必须闭合。
- 左括号必须比右括号先出现。

**输入：** `s = "()"[]{}"`

**输出：** `True`

**答案：**
```python
def isValid(s: str) -> bool:
    stack = []
    mapping = {")": "(", "]": "[", "}": "{"}
    for c in s:
        if c in mapping:
            top_element = stack.pop() if stack else "#"
            if mapping[c] != top_element:
                return False
        else:
            stack.append(c)
    return not stack
```

**3. 计算最长公共前缀**

**题目：** 给定一个字符串数组，请找出其中最长公共前缀。如果不存在公共前缀，则返回空字符串。

**输入：** `["flower", "flow", "flight"]`

**输出：** `"fl"`

**答案：**
```python
def longestCommonPrefix(strs) -> str:
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s) and prefix[i] == s[i]:
            i += 1
        prefix = prefix[:i]
        if not prefix:
            break
    return prefix
```

**4. 计算二进制数中1的个数**

**输出：** `3`（二进制表示为 1011，有三个1）

**答案：**
```python
def hammingWeight(n: int) -> int:
    count = 0
    while n:
        count += n & 1
        n >>= 1
    return count
```

**5. 最长连续序列**

**题目：** 给定一个未排序的整数数组，找出最长连续序列的长度。

**输入：** `[100, 4, 200, 1, 3, 2]`

**输出：** `4`（最长连续序列为 `[1, 2, 3, 4]`）

**答案：**
```python
def longestConsecutive(nums) -> int:
    if not nums:
        return 0
    nums_set = set(nums)
    max_len = 0
    for num in nums:
        if num - 1 not in nums_set:
            current_num = num
            current_len = 1
            while current_num + 1 in nums_set:
                current_num += 1
                current_len += 1
            max_len = max(max_len, current_len)
    return max_len
```

**6. 字符串转换整数 (atoi)**

**题目：** 实现一个函数，将字符串转换成一个整数。该函数需要处理如下情况：

- 如果字符串不是数字开头，则忽略非数字部分。
- 如果字符串表示的数字超出整数范围，则返回整数溢出的结果。
- 如果字符串为空或只有空格，则返回 0。

**输入：** `"42"`

**输出：** `42`

**答案：**
```python
def myAtoi(s: str) -> int:
    INT_MAX = 2**31 - 1
    INT_MIN = -2**31
    i = 0
    sign = 1
    result = 0
    while i < len(s) and s[i] == ' ':
        i += 1
    if i < len(s) and (s[i] == '+' or s[i] == '-'):
        sign = -1 if s[i] == '-' else 1
        i += 1
    while i < len(s) and s[i].isdigit():
        digit = ord(s[i]) - ord('0')
        if result > (INT_MAX - digit) // 10:
            return INT_MAX if sign == 1 else INT_MIN
        result = result * 10 + digit
        i += 1
    return result * sign
```

**7. 最短编辑距离**

**题目：** 给定两个字符串，请找出将其中一个字符串转换为另一个字符串所需的最少编辑操作次数。编辑操作包括：插入、删除和替换。

**输入：** `"horse"` 和 `"ros"` 

**输出：** `3`（编辑操作为 `"horse"` -> `"ros"`：替换`h` -> `r`，替换`o` -> `s`，删除`e`）

**答案：**
```python
def minDistance(word1: str, word2: str) -> int:
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(m + 1):
        for j in range(n + 1):
            if i == 0:
                dp[i][j] = j  # 删除操作
            elif j == 0:
                dp[i][j] = i  # 插入操作
            elif word1[i - 1] == word2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]  # 无操作
            else:
                dp[i][j] = 1 + min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1])

    return dp[m][n]
```

**9. 字符串中的最近重复子串**

**题目：** 给定一个字符串，请找出其中最近重复的子串。如果不存在重复子串，则返回空字符串。

**输入：** `"abcabcabc"`

**输出：** `"abc"`（最近的重复子串为`"abc"`，出现在字符串的前三个位置）

**答案：**
```python
def findRepeatSubstring(s: str) -> str:
    n = len(s)
    for i in range(1, n // 2 + 1):
        if n % i == 0:
            substring = s[:i]
            repeat_index = s.find(substring, i)
            if repeat_index != -1:
                return substring
    return ""
```

**10. 二进制搜索树中的查找**

**题目：** 给定一个二进制搜索树，请实现一个查找函数，用于查找树中是否存在特定值。

**输入：** `root = [4,2,6,1,3,5,7]`，`target = 5`

**输出：** `True`

**答案：**
```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def searchBST(root, val) -> TreeNode:
    if root is None or root.val == val:
        return root
    if root.val > val:
        return searchBST(root.left, val)
    return searchBST(root.right, val)
```
