#### 实现一个基于协同过滤的推荐系统。

**解析：**

协同过滤是一种基于用户相似度的推荐算法，其核心思想是根据用户的历史行为，为用户推荐其他相似用户喜欢的商品。以下是一个简单的基于用户协同过滤的推荐系统实现：

```python
import numpy as np

def collaborative_filtering(ratings, k=5):
    # 计算用户之间的相似度矩阵
    similarity_matrix = np.dot(ratings, ratings.T) / np.linalg.norm(ratings, axis=1)[:, np.newaxis]

    # 为每个用户推荐相似用户喜欢的商品
    recommendations = []
    for user in range(ratings.shape[0]):
        # 计算用户与其他用户的相似度
        similarity_scores = similarity_matrix[user]

        # 选择最相似的 k 个用户
        top_k = np.argsort(similarity_scores)[1:k+1]

        # 计算推荐商品的评分
        recommendation_scores = np.dot(ratings[top_k], ratings[user]) / similarity_scores[top_k]

        # 添加推荐商品到推荐列表
        recommendations.append(np.argsort(recommendation_scores)[::-1])

    return recommendations
```

通过以上实现，我们可以为每个用户推荐其他相似用户喜欢的商品，从而提高推荐系统的准确性。
