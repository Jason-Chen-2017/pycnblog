#### 3. 如何解决算法编程题中的效率问题？

**面试题：** 给定一个整数数组，请实现一个函数，找出数组中两个数的和等于目标值的组合。

**答案：**
```python
def find_pairs(nums, target):
    nums.sort()
    left, right = 0, len(nums) - 1
    result = []
    while left < right:
        current_sum = nums[left] + nums[right]
        if current_sum == target:
            result.append([nums[left], nums[right]])
            left += 1
            right -= 1
        elif current_sum < target:
            left += 1
        else:
            right -= 1
    return result
```

**解析：** 这道题目考察了算法的效率优化。通过排序和双指针的方法，将时间复杂度降低到 O(n log n)，相比传统的方法（如使用哈希表）更加高效。
