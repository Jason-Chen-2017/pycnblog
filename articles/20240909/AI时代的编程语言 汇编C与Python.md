**解析与实例：汇编语言中的“跳转”指令**

**题目：** 请使用汇编语言实现一个跳转指令。

**答案：**

```assembly
section .data
    msg db "Hello, World!", 0

section .text
    global _start

_start:
    ; 跳转到消息处理部分
    jmp print_message

handle_message:
    ; 处理消息
    mov edx, len msg
    mov ecx, msg
    mov ebx, 1
    mov eax, 4
    int 0x80
    jmp exit

print_message:
    ; 跳转到消息处理部分
    jmp handle_message

exit:
    ; 退出程序
    mov eax, 1
    xor ebx, ebx
    int 0x80
```

**解析：** 在这段代码中，我们使用了 `jmp` 指令来实现跳转。`jmp print_message` 将程序跳转到 `print_message` 标签处，执行消息处理逻辑。`jmp exit` 则跳转到 `exit` 标签，执行程序退出。
