#### 2. 最优染色顺序

**题目：** 编写一个算法，确定纺织品的染色顺序，以最小化能耗。

**算法思路：** 可以使用贪心算法，选择下一个染色任务时，优先选择能耗最低的。

```python
def optimal_dyeing_order(dyes, fabrics):
    order = []
    while fabrics:
        min_energy = float('inf')
        for dye, energy in dyes.items():
            if energy < min_energy and fabrics.intersection(dye['compatible_fabrics']):
                min_energy = energy
                next_dye = dye
        order.append(next_dye)
        dyes[next_dye]['energy'] = float('inf')
        fabrics = fabrics.difference(next_dye['compatible_fabrics'])
    return order
```

**解析：** 该算法通过贪心算法选择能耗最低的染色任务，以最小化总能耗。
