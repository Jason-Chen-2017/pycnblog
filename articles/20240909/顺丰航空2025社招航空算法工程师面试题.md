#### 1. 二分查找
**题目：** 在一个有序数组中，找到目标值的位置。
```python
def search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```
**解析：** 这是一个典型的二分查找问题，通过不断缩小区间，高效地找到目标值。

#### 2. 股票买卖
**题目：** 设计一个算法，计算出最大利润。
```python
def maxProfit(prices):
    max_profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i - 1]:
            max_profit += prices[i] - prices[i - 1]
    return max_profit
```
**解析：** 通过遍历数组，计算相邻元素之间的差值，累加得到最大利润。

#### 3. 删除排序数组中的重复项
**题目：** 删除重复元素，返回新的数组长度。
```python
def removeDuplicates(nums):
    if not nums:
        return 0
    slow, fast = 0, 1
    while fast < len(nums):
        if nums[slow] != nums[fast]:
            slow += 1
            nums[slow] = nums[fast]
        fast += 1
    return slow + 1
```
**解析：** 通过双指针技术，保持 `slow` 指针指向当前应该放置下一个不重复元素的下一个位置，`fast` 指针遍历整个数组，找到不重复的元素进行替换。

#### 4. 字符串转换整数 (atoi)
**题目：** 实现 `atoi` 函数，将字符串转为整数。
```python
def myAtoi(s: str) -> int:
    sign, num, i = 1, 0, 0
    while i < len(s) and s[i] == ' ':
        i += 1
    if i < len(s) and (s[i] == '+' or s[i] == '-'):
        sign = -1 if s[i] == '-' else 1
        i += 1
    for char in s[i:]:
        if char not in '0123456789':
            break
        num = num * 10 + int(char)
    return min(max(num * sign, -2**31), 2**31 - 1)
```
**解析：** 考虑到边界条件，处理空字符串、正负号、数字字符和非数字字符。

#### 5. 合并两个有序链表
**题目：** 合并两个有序链表。
```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def mergeTwoLists(l1, l2):
    dummy = ListNode(0)
    curr = dummy
    while l1 and l2:
        if l1.val < l2.val:
            curr.next = l1
            l1 = l1.next
        else:
            curr.next = l2
            l2 = l2.next
        curr = curr.next
    curr.next = l1 or l2
    return dummy.next
```
**解析：** 通过虚拟头节点，避免处理第一个节点为 `None` 的情况。
