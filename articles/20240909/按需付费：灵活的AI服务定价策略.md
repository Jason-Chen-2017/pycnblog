#### 1. 实现一个按需付费的计费系统

**题目描述：** 实现一个按需付费的计费系统，根据用户的使用量进行计费。假设每100次API调用需要支付1元，超出部分每100次额外支付1.5元。

**输入：** 

- 用户API调用次数

**输出：** 

- 总计费用

**代码实现：**

```python
def calculate_fee(total_calls):
    fee = 0
    if total_calls <= 100:
        fee = total_calls / 100
    else:
        fee = 1 + (total_calls - 100) / 100 * 1.5
    return round(fee, 2)

# 示例
print(calculate_fee(50))  # 输出 0.5
print(calculate_fee(150))  # 输出 2.25
```

def calculate_fee(text_content, label_count):
