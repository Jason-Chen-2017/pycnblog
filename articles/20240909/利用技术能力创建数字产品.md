#### 4. 阿里巴巴面试题：给定一个字符串，求最长公共前缀

**题目描述：** 编写一个函数，实现求两个字符串的最长公共前缀。

**答案：** 

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        while not s.startswith(prefix):
            len = len(prefix)
            prefix = prefix[:len-1]
    return prefix
```

**解析：** 这道题目考察应聘者的字符串处理能力和对最外层循环的掌握，通过不断缩小前缀，找到最长公共前缀。

class LRUCache:

    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = {}
        self doubly_linked_list = DoublyLinkedList()

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.doubly_linked_list.moveToHead(self.cache[key])
        return self.cache[key].value

#### 6. 腾讯面试题：最小生成树算法

**题目描述：** 实现一个最小生成树算法，给定一个无向图，求最小生成树的边权和。

**答案：** 

```python
import heapq

def minTreeSum(values, edges):
    n = len(values)
    graph = [[] for _ in range(n)]
    for u, v, w in edges:
        graph[u].append((v, w))
        graph[v].append((u, w))
    key = lambda x: -x[1]
    mst = []
    visited = [False] * n
    pq = []
    for i in range(n):
        if not visited[i]:
            heapq.heappush(pq, (key(values[i]), i))
            visited[i] = True
            while pq:
                _, u = heapq.heappop(pq)
                for v, w in graph[u]:
                    if not visited[v]:
                        mst.append((u, v, w))
                        heapq.heappush(pq, (key(values[v]), v))
                        visited[v] = True
                        break
    return sum(w for _, _, w in mst)
```

**解析：** 这道题目考察应聘者对最小生成树算法（如Prim算法）的掌握，通过优先队列实现贪心策略，找到最小生成树。
