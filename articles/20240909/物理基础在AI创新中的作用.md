#### 3. 编写一个算法，用于计算物体在斜面上滑动的时间。

**题目：** 编写一个函数，输入物体的初始速度、斜面角度和摩擦系数，计算物体在斜面上滑动的时间。

**答案：**

```python
import math

def slide_time(initial_speed, angle, friction_coefficient):
    g = 9.8  # 重力加速度
    angle_rad = math.radians(angle)
    acceleration = -g * math.sin(angle_rad) * friction_coefficient
    time = math.sqrt(2 * initial_speed / acceleration)
    return time
```

**解析：** 该算法基于物理学中的动力学原理，计算物体在斜面上的滑动时间。通过将初始速度、斜面角度和摩擦系数作为输入参数，可以计算出物体滑动的总时间。
