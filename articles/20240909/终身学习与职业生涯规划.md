#### 二、算法编程题实战

#### 4. 两数之和

**题目：** 给定一个整数数组 `nums` 和一个目标值 `target`，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**答案：** 

```python
def twoSum(nums, target):
    hashmap = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hashmap:
            return [hashmap[complement], i]
        hashmap[num] = i
    return []
```

**解析：** 本题使用哈希表实现，时间复杂度为 O(n)，空间复杂度为 O(n)。通过遍历数组，将每个元素与其对应的目标值差存储在哈希表中，以便快速查找。

#### 5. 有效的括号

**题目：** 给定一个字符串 `s` ，判断 `s` 是否是有效的括号字符串。

**答案：** 

```python
def isValid(s):
    stack = []
    for char in s:
        if char in "([{":
            stack.append(char)
        elif not stack or (char == ")" and stack.pop() != "(") or (char == "}" and stack.pop() != "{") or (char == "]" and stack.pop() != "["):
            return False
    return not stack
```

**解析：** 本题使用栈实现，时间复杂度为 O(n)，空间复杂度为 O(n)。遍历字符串，将左括号入栈，右括号与栈顶元素匹配，若匹配则弹出栈顶元素，不匹配则返回 False。最后判断栈是否为空，空则表示字符串有效。

#### 6. 逆波兰表达式求值

**题目：** 实现一个基本的计算器来计算逆波兰表达式。

**答案：**

```python
def evalRPN(tokens):
    stack = []
    for token in tokens:
        if token in "+-*":
            b = stack.pop()
            a = stack.pop()
            if token == "+":
                stack.append(a + b)
            elif token == "-":
                stack.append(a - b)
            elif token == "*":
                stack.append(a * b)
            else:
                stack.append(int(float(a) / float(b)))
        else:
            stack.append(int(token))
    return stack.pop()
```

**解析：** 本题使用栈实现，时间复杂度为 O(n)，空间复杂度为 O(n)。遍历逆波兰表达式，根据运算符进行计算，将结果入栈，最后返回栈顶元素作为结果。
