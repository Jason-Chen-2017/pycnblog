### 算法编程题库与解析

#### 1. 代码优化

**题目：** 对以下代码进行优化，使其运行时间更长。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

**答案：**

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
```

**解析：** 优化后的代码在每次外层循环结束后，判断是否有元素交换，如果没有交换，说明数组已经有序，可以直接结束循环，从而减少不必要的迭代。

#### 2. 求最大子序和

**题目：** 给定一个整数数组 `nums` ，找到一个具有最大和的连续子数组（子数组最少包含一个数），返回其最大和。

**答案：**

```python
def max_sub_array(nums):
    if not nums:
        return 0
    max_sum = nums[0]
    curr_sum = nums[0]
    for i in range(1, len(nums)):
        curr_sum = max(nums[i], curr_sum + nums[i])
        max_sum = max(max_sum, curr_sum)
    return max_sum
```

**解析：** 采用动态规划的思想，`curr_sum` 表示以当前元素为结尾的最大子序和，当 `curr_sum` 小于 `nums[i]` 时，说明前面的子序列已经不可取，需要重新开始计算。
