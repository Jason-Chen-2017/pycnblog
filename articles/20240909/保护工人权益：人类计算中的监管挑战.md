#### 1. 请设计一个算法，用于监控员工加班时长，确保不超过法律规定的上限。

**题目：** 设计一个算法，监控员工加班时长，确保不超过法律规定的上限（如每周不超过 36 小时）。

**答案：** 可以使用以下算法：

1. 初始化一个字典，用于存储员工姓名和加班时长。
2. 每次添加加班记录时，更新字典中的对应员工的加班时长。
3. 每天结束时，检查员工的加班时长是否超过规定的上限，如超过，则发送提醒通知。

```python
def monitor_over_time(employees, daily_hours):
    over_time_alerts = []
    for employee, hours in daily_hours.items():
        if hours > 36:
            over_time_alerts.append(employee)
    return over_time_alerts
```

#### 2. 请设计一个算法，用于分析员工满意度，并根据分析结果优化员工福利。

**题目：** 设计一个算法，分析员工满意度，并根据分析结果优化员工福利。

**答案：** 可以使用以下算法：

1. 收集员工满意度数据，例如问卷调查结果。
2. 对满意度数据进行分析，提取关键指标，如平均满意度、满意度分布等。
3. 根据分析结果，制定针对性的福利优化方案。

```python
def analyze_satisfaction(satisfaction_data):
    average_satisfaction = sum(satisfaction_data) / len(satisfaction_data)
    satisfaction_distribution = {satisfaction: satisfaction_data.count(satisfaction) for satisfaction in satisfaction_data}
    return average_satisfaction, satisfaction_distribution
```
