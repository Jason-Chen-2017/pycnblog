#### 算法编程题解析

**3. 请实现一个简单的注意力分配模型，计算给定任务集合中的注意力权重。**

**题目描述：** 有一个包含N个任务的集合，每个任务有一个对应的难度系数。请设计一个算法，根据任务难度系数计算出一个注意力权重分配方案，使得总注意力权重最大化。

**输入：** N个任务及其难度系数。

**输出：** 一个长度为N的权重数组，表示每个任务的注意力权重。

**示例：**

```
输入：[1, 2, 3]
输出：[0.33, 0.33, 0.33]
```

**算法：**

```python
def attention_allocation(tasks):
    n = len(tasks)
    total_difficulty = sum(tasks)
    weights = [float(task) / total_difficulty for task in tasks]
    return weights
```

**解析：** 该算法采用简单的平均分配方法，计算每个任务的注意力权重。在实际应用中，可以根据任务的重要性和紧急程度调整权重分配策略。
