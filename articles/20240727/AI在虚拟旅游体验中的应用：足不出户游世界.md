## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

以推荐系统为例，假设用户集合为 $U$，物品集合为 $I$，历史行为数据为 $R_{ui}$，其中 $u \in U, i \in I$。推荐系统的目标是最大化用户对推荐的满意度。

**目标函数**
$$
\max_{\theta} \sum_{(u,i) \in R} \log \sigma(\theta_u \cdot \theta_i^T) - \sum_{(u,i) \notin R} \log(1 - \sigma(\theta_u \cdot \theta_i^T))
$$

其中 $\sigma$ 为sigmoid函数，$\theta_u$ 和 $\theta_i$ 为用户和物品的嵌入向量。

### 4.2 公式推导过程

假设用户 $u$ 对物品 $i$ 的评分 $r_{ui}$ 为 $y_{ui} = \theta_u \cdot \theta_i^T$，目标函数转化为：
$$
\max_{\theta} \sum_{(u,i) \in R} r_{ui} - \sum_{(u,i) \notin R} (1 - r_{ui})
$$

利用优化算法，如随机梯度下降，更新模型参数 $\theta$，最小化上述目标函数。

### 4.3 案例分析与讲解

以KNN推荐系统为例，假设用户 $u$ 的评分向量为 $x_u$，物品 $i$ 的评分向量为 $x_i$。

**算法步骤**
1. 计算用户 $u$ 和物品 $i$ 的欧氏距离 $d(u,i) = ||x_u - x_i||$。
2. 选取最近的 $k$ 个物品，记为 $N_{ki} = \{j | d(u,j) < d(u,i)\}$。
3. 计算物品 $i$ 在用户 $u$ 的评分预测值 $\hat{y}_{ui} = \frac{1}{|N_{ki}|} \sum_{j \in N_{ki}} y_{uj}$。
4. 更新用户 $u$ 对物品 $i$ 的评分预测 $\hat{y}_{ui}$，生成推荐列表。

```python
import torch
import torch.nn as nn
import torch.optim as optim
from sklearn.neighbors import NearestNeighbors

# 定义模型
class KNNModel(nn.Module):
    def __init__(self, n_users, n_items, embedding_size):
        super(KNNModel, self).__init__()
        self.user_embedding = nn.Embedding(n_users, embedding_size)
        self.item_embedding = nn.Embedding(n_items, embedding_size)

    def forward(self, user, item):
        user_embed = self.user_embedding(user)
        item_embed = self.item_embedding(item)
        return torch.cosine_similarity(user_embed, item_embed)

# 定义数据集
class UserItemDataset(torch.utils.data.Dataset):
    def __init__(self, users, items, ratings):
        self.users = users
        self.items = items
        self.ratings = ratings
        self.n_users = len(set(self.users))
        self.n_items = len(set(self.items))

    def __len__(self):
        return len(self.ratings)

    def __getitem__(self, idx):
        user = self.users[idx]
        item = self.items[idx]
        rating = self.ratings[idx]
        return user, item, rating

# 训练和测试
def train(model, train_dataset, test_dataset, epochs=10, batch_size=32, learning_rate=0.001):
    optimizer = optim.Adam(model.parameters(), lr=learning_rate)
    criterion = nn.CrossEntropyLoss()

    for epoch in range(epochs):
        model.train()
        for user, item, rating in train_dataset:
            optimizer.zero_grad()
            output = model(user, item)
            loss = criterion(output, rating)
            loss.backward()
            optimizer.step()

        model.eval()
        correct = 0
        total = 0
        with torch.no_grad():
            for user, item, rating in test_dataset:
                output = model(user, item)
                pred = output.argmax(dim=1)
                total += 1
                correct += (pred == rating).sum().item()
        print(f"Epoch {epoch+1}, accuracy: {100 * correct / total:.2f}%")

# 示例数据
users = [1, 2, 3, 4, 5]
items = [1, 2, 3, 4, 5]
ratings = [[1, 1, 2], [2, 2, 3], [1, 3, 1], [2, 1, 3], [3, 2, 2]]

train_dataset = UserItemDataset(users, items, ratings)
test_dataset = UserItemDataset(users, items, ratings)

# 训练模型
model = KNNModel(len(users), len(items), 10)
train(model, train_dataset, test_dataset)

```
