#### 5.2 源代码详细实现

以下是一个简单的示例代码，用于实现基于K-means算法的聚类分析。

```python
import numpy as np
from sklearn.cluster import KMeans

# 数据集
data = np.array([[1, 2], [1, 4], [1, 0], [10, 2], [10, 4], [10, 0]])

# 初始化KMeans模型
kmeans = KMeans(n_clusters=2, random_state=0).fit(data)

# 输出聚类结果
print("聚类中心：", kmeans.cluster_centers_)
print("每个样本的聚类标签：", kmeans.labels_)

# 输出聚类结果的可视化
import matplotlib.pyplot as plt

plt.scatter(data[:, 0], data[:, 1], c=kmeans.labels_, s=100, cmap='viridis')
plt.scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1], s=300, c='red', marker='s', edgecolor='black', label='Centroids')
plt.title('K-means Clustering')
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.legend()
plt.show()
```


   我们初始化一个KMeans模型，其中n_clusters=2表示我们要将数据分为两个聚类。

4. 输出聚类结果：
   ```python
   print("聚类中心：", kmeans.cluster_centers_)
   print("每个样本的聚类标签：", kmeans.labels_)
   ```

   我们打印出聚类中心（每个聚类的均值）和每个样本的聚类标签。
