## 4. 具体代码实例和详细解释说明

### 4.1 代码实例1：简单的排序算法

```python
def simple_sort(arr):
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] > arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr
```

这个代码实例是一个简单的冒泡排序算法。它通过比较相邻的元素，并将较大的元素交换到后面，以达到排序的目的。

### 4.2 代码实例2：使用内置函数进行排序

```python
def use_builtin_sort(arr):
    return sorted(arr)
```

这个代码实例使用了Python内置的sorted函数来进行排序。这种方法更简洁、易于理解，同时也更高效。

### 4.3 代码实例3：使用递归进行计算

```python
def recursive_calculate(n):
    if n == 1:
        return 1
    else:
        return 2 * recursive_calculate(n-1)
```

这个代码实例使用了递归来计算2的n次方。递归是一种简化代码的有效方法，它可以使代码更简洁、易于理解。
