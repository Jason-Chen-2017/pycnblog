# 4.具体代码实例和详细解释说明

MySQL的具体代码实例包括：创建数据库、创建表、插入数据、查询数据、更新数据、删除数据等。这些代码实例是MySQL的基础，通过学习这些代码实例可以掌握MySQL的使用方法。

创建数据库的代码实例如下：

```sql
CREATE DATABASE mydb;
USE mydb;
```

创建表的代码实例如下：

```sql
CREATE TABLE mytable (
  id INT PRIMARY KEY,
  name VARCHAR(255),
  age INT
);
```

插入数据的代码实例如下：

```sql
INSERT INTO mytable (id, name, age) VALUES (1, 'John', 20);
```

查询数据的代码实例如下：

```sql
SELECT * FROM mytable;
```

更新数据的代码实例如下：

```sql
UPDATE mytable SET age = 21 WHERE id = 1;
```

删除数据的代码实例如下：

```sql
DELETE FROM mytable WHERE id = 1;
```
