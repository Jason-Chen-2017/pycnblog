def calculate(num1, num2, operator):
    if operator == '+':
        return num1 + num2
    elif operator == '-':
        return num1 - num2
    elif operator == '*':
        return num1 * num2
    elif operator == '/':
        return num1 / num2
    else:

2. 编写一个简单的排序程序：

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)

arr = [5, 2, 9, 1, 7]
sorted_arr = quick_sort(arr)
print(sorted_arr)
```

在这个代码实例中，我们编写了一个简单的快速排序程序，它可以对一个数组进行排序。我们使用了禅学的忍耐和创造力来编写这个程序，以确保其正确性和可读性。
