# 2.核心概念与联系

## 2.1 大数据
大数据是指由于互联网、移动互联网等技术的发展，产生的数据量巨大、数据类型多样、数据流动性高、数据更新频率快等特点的数据。大数据具有以下几个主要特征：

1. 数据量庞大：大数据的规模可以达到PB甚至EB级别（1EB=10^18字节）。
2. 数据类型多样：大数据包括结构化数据（如关系型数据库）、非结构化数据（如文本、图像、音频、视频等）和半结构化数据（如JSON、XML等）。
3. 数据流动性高：大数据可以在不同的设备和系统之间快速传输和分析。
4. 数据更新频率快：大数据的更新速度非常快，甚至实时。

## 2.2 计算机辅助决策（CAD）
计算机辅助决策（CAD）是一种利用计算机和数学方法来解决复杂决策问题的方法。CAD技术可以帮助决策者更快速、准确地做出决策，提高决策的质量。CAD技术的主要应用领域包括：

1. 预测分析：通过对历史数据进行分析，预测未来的发展趋势。
2. 优化决策：通过数学模型和算法，找到最优解。
3. 模拟实验：通过计算模拟实验，评估不同决策的效果。
4. 决策支持系统：通过构建决策支持系统，帮助决策者更好地理解问题和做出决策。
