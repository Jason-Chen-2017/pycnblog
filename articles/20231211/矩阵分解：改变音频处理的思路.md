## 4.1 奇异值分解

我们将使用Python的NumPy库来实现奇异值分解。以下是奇异值分解的代码实例：

```python
import numpy as np

# 创建一个随机矩阵
A = np.random.rand(100, 100)

# 执行奇异值分解
U, S, V = np.linalg.svd(A)

# 打印奇异值矩阵
print("奇异值矩阵：")
print(S)

# 打印左奇异向量矩阵
print("左奇异向量矩阵：")
print(U)

# 打印右奇异向量矩阵
print("右奇异向量矩阵：")
print(V)
```

在上述代码中，我们首先创建了一个随机矩阵。然后我们使用NumPy库的`np.linalg.svd()`函数来执行奇异值分解。最后，我们打印了奇异值矩阵、左奇异向量矩阵和右奇异向量矩阵。

## 4.3 非负矩阵分解

我们将使用Python的NumPy库来实现非负矩阵分解。以下是非负矩阵分解的代码实例：

```python
import numpy as np

# 创建一个随机非负矩阵
A = np.random.randint(0, 100, (100, 100))
A[A < 0] = 0

# 执行非负矩阵分解
H = np.linalg.lstsq(A, np.eye(A.shape[0]), rcond=None)[0]

```

在上述代码中，我们首先创建了一个随机非负矩阵。然后我们使用NumPy库的`np.linalg.lstsq()`函数来执行非负矩阵分解。最后，我们打印了非负矩阵。
