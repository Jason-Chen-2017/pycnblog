### 4.1 代码实例

我们选择一个简单的排序问题，即对一个整数数组进行排序。我们将使用冒泡排序算法来解决这个问题。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
result = bubble_sort(arr)
print(result)
```

### 4.2 详细解释说明

在上述代码中，我们首先定义了一个名为 `bubble_sort` 的函数，该函数接受一个整数数组 `arr` 作为参数。我们使用冒泡排序算法来对数组进行排序。

冒泡排序算法的基本思想是：通过多次对数组中的元素进行交换，使得较小的元素逐渐向左移动，较大的元素逐渐向右移动。在每一次迭代中，我们从数组的第一个元素开始，与后续的元素进行比较，如果当前元素大于后续元素，则进行交换。这样，在每一次迭代中，最小的元素会被推向数组的末尾。

在代码中，我们使用了两个嵌套的 `for` 循环来实现冒泡排序。外部循环用于控制迭代次数，内部循环用于比较当前元素和后续元素，并进行交换。在内部循环中，我们使用了一个 `if` 语句来判断当前元素是否大于后续元素，如果是，则进行交换。

最后，我们将排序后的数组 `result` 打印出来，以验证排序是否成功。
