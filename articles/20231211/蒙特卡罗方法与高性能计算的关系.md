## 4.1 代码实例：蒙特卡罗方法求解π的问题

我们来看一个求解π的问题，通过蒙特卡罗方法来估计π的值。

```python
import random

def random_point():
    x = random.uniform(-1, 1)
    y = random.uniform(-1, 1)
    return (x, y)

def is_in_circle(point):
    x, y = point
    return x*x + y*y <= 1

def monte_carlo_pi(n):
    count = 0
    for _ in range(n):
        point = random_point()
        if is_in_circle(point):
            count += 1
    pi = 4 * count / n
    return pi

n = 100000
pi = monte_carlo_pi(n)
print("估计的π值为：", pi)
```

在这个代码实例中，我们首先定义了一个随机点生成函数`random_point()`，用于生成随机点。然后我们定义了一个判断点是否在圆内的函数`is_in_circle()`。最后，我们定义了一个蒙特卡罗方法求解π的函数`monte_carlo_pi()`，通过生成大量的随机点，并判断每个点是否在圆内，从而计算出估计的π值。

## 4.2 代码解释

1. 首先，我们通过`random.uniform(-1, 1)`函数来生成一个区间为(-1, 1)的随机数，并将其作为点的x和y坐标。
2. 然后，我们通过`is_in_circle(point)`函数来判断每个点是否在圆内。如果点在圆内，则计数器`count`加1。
3. 最后，我们通过`pi = 4 * count / n`来计算估计的π值。
