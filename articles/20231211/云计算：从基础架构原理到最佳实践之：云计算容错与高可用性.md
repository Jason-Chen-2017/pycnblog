                 

# 1.背景介绍

云计算是一种基于互联网的计算资源分配和共享方式，它可以让用户在不需要购买硬件设备的情况下，通过网络访问计算资源。云计算的核心思想是将计算资源、存储资源和网络资源等基础设施通过互联网进行集中化的管理和分配。

云计算的发展是为了解决传统计算机资源分配和管理的局限性。传统计算机资源分配和管理方式是基于单机或集群的本地计算资源，这种方式的局限性有以下几点：

1. 资源利用率低：传统计算机资源分配和管理方式的资源利用率低，因为计算机资源在空闲时间里不被充分利用。
2. 扩展性有限：传统计算机资源分配和管理方式的扩展性有限，因为需要购买更多的硬件设备来扩展计算资源。
3. 管理复杂：传统计算机资源分配和管理方式的管理复杂，需要购买更多的硬件设备来扩展计算资源。

云计算的发展可以解决传统计算机资源分配和管理的局限性，提高资源利用率、扩展性和管理简单性。
