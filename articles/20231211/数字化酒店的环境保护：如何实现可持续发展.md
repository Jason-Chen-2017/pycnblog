
```python
data = pd.read_csv('energy_data.csv')
```

接下来，我们需要进行数据预处理：

```python
X = data.drop('energy', axis=1)
y = data['energy']
X = X.values.reshape(-1, 1)

```

然后，我们需要训练模型：

```python
model = LinearRegression()

```

最后，我们需要进行预测：

```python
x_predict = np.array([[x]])
```
