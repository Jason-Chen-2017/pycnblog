# 4.具体代码实例和详细解释说明

## 4.1 流式处理代码实例

在本节中，我们将通过一个简单的流式处理代码实例来详细解释流式处理的具体操作步骤。

```python
import time
from kafka import KafkaProducer, KafkaConsumer

# 创建Kafka生产者
producer = KafkaProducer(bootstrap_servers='localhost:9092')

# 创建Kafka消费者
consumer = KafkaConsumer('test_topic')

# 生产数据
for i in range(10):
    data = {'key': i, 'value': i * i}
    producer.send('test_topic', value=data)
    time.sleep(1)

# 消费数据
for message in consumer:
    print(message.value)
```

在这个代码实例中，我们使用Kafka库来实现流式处理。首先，我们创建了一个Kafka生产者，然后创建了一个Kafka消费者。接着，我们生产了10条数据，每条数据包含一个key和一个value。最后，我们消费了这10条数据，并打印了它们的值。
