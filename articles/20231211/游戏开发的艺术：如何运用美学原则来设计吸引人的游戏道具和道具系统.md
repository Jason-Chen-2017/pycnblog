# 4.具体代码实例和详细解释说明
以下是一个简单的游戏道具系统的代码实例：

```python
import random

class Prop:
    def __init__(self, name, probability, cost):
        self.name = name
        self.probability = probability
        self.cost = cost

    def get_prop(self):
        if random.random() < self.probability:
            return self
        else:
            return None

    def use_prop(self):
        if random.random() < self.cost:
            return True
        else:
            return False

props = [
    Prop("道具1", 0.1, 0.5),
    Prop("道具2", 0.2, 0.8),
    Prop("道具3", 0.3, 0.3),
]

while True:
    prop = random.choice(props).get_prop()
    if prop:
        print("获得道具：", prop.name)
        if prop.use_prop():
            print("使用道具成功")
        else:
            print("使用道具失败")
    else:
        print("没有获得道具")
```

在这个代码实例中，我们定义了一个 `Prop` 类，用于表示游戏道具。道具有名称、获得概率、使用成本等属性。我们还定义了一个道具列表，用于存储游戏中的所有道具。在游戏中，我们会随机选择一个道具，并根据其获得概率和使用成本来决定是否获得和使用道具。
