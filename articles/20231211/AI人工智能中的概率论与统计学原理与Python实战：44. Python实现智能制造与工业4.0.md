# 3.1概率论与统计学的基本概念
在概率论与统计学中，我们需要了解一些基本概念，如随机变量、概率、期望、方差等。

随机变量：一个随机变量是一个可以取多个值的变量，每个值都有一个概率。

概率：概率是一个数值，表示某个事件发生的可能性。概率范围在0到1之间，0表示事件不可能发生，1表示事件必然发生。

期望：期望是一个随机变量的数学期望，表示随机变量的平均值。

方差：方差是一个随机变量的数学方差，表示随机变量的分散程度。

# 4.1概率论与统计学的Python实例
在Python中，我们可以使用numpy库来实现概率论与统计学的基本操作。

```python
import numpy as np

# 定义一个随机变量
x = np.random.normal(loc=0, scale=1, size=1000)

# 计算随机变量的期望
mean = np.mean(x)
print("期望：", mean)

# 计算随机变量的方差
variance = np.var(x)
print("方差：", variance)
```

# 4.2人工智能的Python实例
在Python中，我们可以使用scikit-learn库来实现人工智能的基本操作。

```python
from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

# 加载一个数据集
data = datasets.load_iris()

# 划分训练集和测试集
x_train, x_test, y_train, y_test = train_test_split(data.data, data.target, test_size=0.2, random_state=42)

# 创建一个随机森林分类器
clf = RandomForestClassifier(n_estimators=100, random_state=42)

# 训练分类器
clf.fit(x_train, y_train)

# 预测
y_pred = clf.predict(x_test)

# 评估分类器
accuracy = clf.score(x_test, y_test)
print("准确率：", accuracy)
```

# 4.3智能制造与工业4.0的Python实例
在Python中，我们可以使用pandas库来实现智能制造与工业4.0的基本操作。

```python
import pandas as pd

# 加载一个数据集
data = pd.read_csv("data.csv")

# 数据预处理
data = data.dropna()
data = pd.get_dummies(data)

# 划分训练集和测试集
x_train, x_test, y_train, y_test = train_test_split(data.drop("target", axis=1), data["target"], test_size=0.2, random_state=42)

# 创建一个预测模型
from sklearn.ensemble import RandomForestClassifier
clf = RandomForestClassifier(n_estimators=100, random_state=42)

# 训练模型
clf.fit(x_train, y_train)

# 预测
y_pred = clf.predict(x_test)

# 评估模型
accuracy = clf.score(x_test, y_test)
print("准确率：", accuracy)
```
