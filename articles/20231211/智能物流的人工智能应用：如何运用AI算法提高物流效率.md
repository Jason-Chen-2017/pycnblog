### 4.2 优化决策代码实例

以动态规划模型为例，下面是一个优化物流路径的代码实例：

```python
import numpy as np

# 数据集
distances = np.array([[0, 5, 9, 12],
                      [5, 0, 6, 10],
                      [9, 6, 0, 8],
                      [12, 10, 8, 0]])

# 动态规划模型
def min_distance(distances):
    n = len(distances)
    dp = np.zeros((n, n))
    for i in range(n):
        for j in range(i, n):
            if i == j:
                dp[i, j] = distances[i, j]
            elif i < j:
                dp[i, j] = min(dp[i, k] + distances[k, j] for k in range(i, j))
    return dp[0, n - 1]

# 优化结果
min_distance_result = min_distance(distances)
print(min_distance_result)  # 输出: 12
```
