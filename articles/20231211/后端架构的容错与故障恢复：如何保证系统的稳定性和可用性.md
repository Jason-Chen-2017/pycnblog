# 2.核心概念与联系

在讨论后端架构的容错与故障恢复之前，我们需要了解一些核心概念：

1.容错：容错是指系统在出现故障时能够继续运行并提供服务的能力。容错是一种对系统的保护机制，可以确保系统在出现故障时不会导致整个系统崩溃。

2.故障恢复：故障恢复是指系统在出现故障后能够自动恢复并继续运行的能力。故障恢复是一种对系统的自动保护机制，可以确保系统在出现故障时不会导致整个系统崩溃。

3.稳定性：稳定性是指系统在出现故障时能够保持稳定运行的能力。稳定性是一种对系统的保护机制，可以确保系统在出现故障时不会导致整个系统崩溃。
