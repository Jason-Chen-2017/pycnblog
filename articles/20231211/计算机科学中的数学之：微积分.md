## 1.1 微积分的历史与发展

微积分的历史可以追溯到古希腊时期的数学家阿里斯特芬诺斯（Archimedes），他通过方程的推导得出积分的概念。然而，微积分的系统性研究并未开始，直到17世纪的法国数学家埃尔拉瓦尔·德勒（Isaac Newton）和格雷戈·利兹（Gottfried Wilhelm Leibniz）独立发明了微积分的基本概念和计算方法。

随着时间的推移，微积分的理论和应用得到了不断的拓展。在20世纪初，俄罗斯数学家弗莱基·拉夫斯基（Andrey Nikolaevich Kolmogorov）和美国数学家艾伦·托尔斯逊（Alan Turing）的工作为微积分的数值计算提供了新的方法。在20世纪中叶，美国数学家埃德蒙·伯努利（Edmond Nelson Bernoulli）和詹姆斯·霍普金斯（James Hopper）为微积分的应用领域拓展提供了新的理论基础。

现在，微积分在计算机科学中的应用已经非常广泛，例如机器学习中的梯度下降算法、计算机图形学中的光线追踪算法等。此外，微积分在物理、生物、金融等多个领域也得到了广泛应用。

def gradient_descent(f, x, lr, n_iter):
    x_history = [x]
    for _ in range(n_iter):
        grad = np.gradient(f, x)
        x = x - lr * grad
        x_history.append(x)
    return x, x_history

import numpy as np

def ray_intersection(origin, direction, sphere):
    O = np.array(origin)
    D = np.array(direction)
    C = np.array(sphere.center)
    R = sphere.radius

    # 计算光线与球的交点
    t = np.dot(C - O, D) / np.dot(D, D)
    if t < 0:
        return None

    P = O + t * D
    d = np.linalg.norm(P - C)

    if d > R:
        return None

    return P

origin = (0, 0, 0)
direction = (1, 0, 0)
sphere = {"center": (0, 0, -5), "radius": 5}

intersection_point = ray_intersection(origin, direction, sphere)
print("交点:", intersection_point)
