1. DNA测序代码实例：

代码实例：

```python
import dna_sequence

def dna_sequence_analysis(dna_sequence):
    dna_sequence_length = len(dna_sequence)
    dna_sequence_average = sum(dna_sequence) / dna_sequence_length
    return dna_sequence_length, dna_sequence_average

dna_sequence = [1, 2, 3, 4, 5]
dna_sequence_length, dna_sequence_average = dna_sequence_analysis(dna_sequence)
print(dna_sequence_length, dna_sequence_average)
```

详细解释说明：

这段代码实例是一个DNA测序的分析程序。它首先导入了dna_sequence模块，然后定义了一个名为dna_sequence_analysis的函数。这个函数接受一个DNA序列作为输入，并计算DNA序列的长度和平均值。最后，它将计算结果返回给调用者。
