# 2.核心概念与联系

1.权重初始化：这是指为神经元之间的连接分配初始权重。权重初始化可以使模型在训练过程中更稳定地收敛。

2.偏置初始化：这是指为神经元的输出偏置分配初始值。偏置初始化可以使模型在训练过程中更稳定地收敛。
