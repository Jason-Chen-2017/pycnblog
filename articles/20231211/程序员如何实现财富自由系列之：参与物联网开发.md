# 2.核心概念与联系

在物联网开发中，有几个核心概念需要我们了解：

1.物联网设备：物联网设备是物联网系统中的基本组成部分，包括传感器、控制器、网关等。这些设备通过网络互联互通，实现数据的收集、传输和处理。

2.物联网协议：物联网协议是物联网设备之间的通信规范，包括TCP/IP、MQTT、HTTP等。这些协议定义了设备之间的数据格式、传输方式和安全性等方面。

3.物联网平台：物联网平台是物联网系统的核心组成部分，负责收集、存储、分析和处理设备数据。物联网平台可以是云平台、本地平台等。

4.物联网应用：物联网应用是物联网系统实现具体功能的软件，包括数据收集、数据分析、数据可视化等。

通过了解这些核心概念，我们可以更好地理解物联网开发的过程，并掌握相关技能和技术。
