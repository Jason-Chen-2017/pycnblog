                 

# 1.背景介绍

无服务架构（Microservices Architecture）是一种软件架构风格，它将软件应用程序划分为多个小的、独立的服务，每个服务都可以独立部署和扩展。这种架构风格的出现是为了解决传统的单体应用程序（Monolithic Applications）在性能、可扩展性、可维护性等方面的局限性。

函数计算（Function Computing）是一种基于无服务架构的服务模型，它将计算作为服务提供给其他服务调用。函数计算允许开发者将代码打包成可独立运行的函数，并将其部署到云端，以实现更高的灵活性和可扩展性。
