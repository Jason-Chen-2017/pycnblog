## 2.1 概率论
概率论是一门数学学科，它研究随机事件发生的可能性。在深度学习中，我们使用概率论来描述模型的不确定性，以及数据之间的关系。

概率论的核心概念包括：
- 事件：一个可能发生或不发生的结果。
- 样本空间：所有可能结果的集合。
- 事件的概率：事件发生的可能性，通常表示为0到1之间的一个数。

## 2.2 统计学
统计学是一门数学学科，它研究从数据中抽取信息。在深度学习中，我们使用统计学来分析数据，以便更好地理解其特征和模式。

统计学的核心概念包括：
- 数据：实际观测到的结果。
- 参数：数据中的某些特征。
- 估计：根据数据来估计参数的值。
