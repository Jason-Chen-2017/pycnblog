```python
def fast_pow(x, n):
    if n == 0:
        return 1
    if n % 2 == 1:
        return x * fast_pow(x, n - 1)
    else:
        return fast_pow(x * x, n / 2)
```

```python
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```

```python
def min_spanning_tree(graph):
    n = len(graph)
    result = []
    visited = [False] * n
    edges = []
    for i in range(n):
        for j in range(i + 1, n):
            edges.append((i, j, graph[i][j]))
    edges.sort(key=lambda x: x[2])
    for edge in edges:
        i, j, weight = edge
        if not visited[i] and not visited[j]:
            result.append((i, j, weight))
            visited[i] = True
            visited[j] = True
    return result
```

快速幂的时间复杂度是O(logn)，空间复杂度是O(1)。快速幂的核心思想是通过将指数分解为两个指数的和，然后递归地计算指数的幂，最后得到指数的幂。快速幂的时间复杂度是O(logn)，因为在每个递归层次中，指数的幂计算的次数是指数的位数。快速幂的空间复杂度是O(1)，因为在每个递归层次中，只需要保存一个指数的幂。

## 7.2 归并排序的复杂度分析

归并排序的时间复杂度是O(nlogn)，空间复杂度是O(n)。归并排序的核心思想是通过将数组分为两个子数组，然后递归地对子数组进行排序，最后将子数组合并为原数组的排序。归并排序的时间复杂度是O(nlogn)，因为在每个递归层次中，数组的分割和合并的次数是数组的长度。归并排序的空间复杂度是O(n)，因为在每个递归层次中，需要保存一个数组的副本。

## 7.3 最小覆盖子集的复杂度分析

最小覆盖子集的时间复杂度是O(2^n)，空间复杂度是O(2^n)。最小覆盖子集的核心思想是通过在每个步骤中选择最优解，然后递归地解决问题，最后得到问题的解。最小覆盖子集的时间复杂度是O(2^n)，因为在每个递归层次中，需要检查所有子集的覆盖情况。最小覆盖子集的空间复杂度是O(2^n)，因为在每个递归层次中，需要保存所有子集的状态。

## 7.4 最小生成树的复杂度分析

最小生成树的时间复杂度是O(n^2)，空间复杂度是O(n)。最小生成树的核心思想是通过在每个步骤中选择最优解，然后递归地解决问题，最后得到问题的解。最小生成树的时间复杂度是O(n^2)，因为在每个递归层次中，需要检查所有边的权重。最小生成树的空间复杂度是O(n)，因为在每个递归层次中，需要保存所有边的状态。
