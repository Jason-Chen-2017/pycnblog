## 3.1抽样分布
抽样分布是一种概率分布，它描述了从大型总体中抽取一个小样本的分布。抽样分布可以用来估计总体的参数，如均值、方差等。

### 3.1.1抽样分布的类型
抽样分布可以分为以下几种类型：

- 单样本抽样分布：这种类型的抽样分布用于估计总体的一个参数，如均值、方差等。
- 双样本抽样分布：这种类型的抽样分布用于比较两个不同的样本来估计它们之间的差异。

### 3.1.2抽样分布的计算
抽样分布的计算主要包括以下几个步骤：

1. 确定总体大小和样本大小：首先需要确定总体大小（n）和样本大小（n'）。
2. 计算抽样分布的参数：根据总体大小和样本大小，可以计算抽样分布的参数，如均值、方差等。
3. 计算抽样分布的分布度：根据抽样分布的参数，可以计算抽样分布的分布度，如标准差等。

### 3.1.3抽样分布的应用
抽样分布可以用于估计总体的参数，如均值、方差等。例如，如果我们想要估计一个总体的均值，可以从总体中随机抽取一个样本，然后计算样本的均值。这个样本的均值可以作为总体的估计值。

## 3.2假设检验
假设检验是一种统计学方法，用于测试某个假设是否可以接受。假设检验主要包括以下几个步骤：

1. 设定假设：首先需要设定一个 Null 假设（H0）和一个替代假设（H1）。Null 假设是我们希望验证的假设，替代假设是我们希望否定的假设。
2. 计算检验统计量：根据 Null 假设和替代假设，可以计算一个检验统计量。检验统计量是一个随机变量，它的分布可以用来判断 Null 假设是否可以接受。
3. 设定检验水平：需要设定一个检验水平（α），它表示我们愿意接受 Null 假设被拒绝的概率。
4. 比较检验统计量与检验水平：根据检验统计量与检验水平的关系，可以判断 Null 假设是否可以接受。如果检验统计量超出了检验水平，则 Null 假设被拒绝；否则，Null 假设被接受。

### 3.2.1假设检验的类型
假设检验可以分为以下几种类型：

- 单样本假设检验：这种类型的假设检验用于测试一个样本是否来自于一个特定的总体。
- 双样本假设检验：这种类型的假设检验用于比较两个不同的样本来判断它们之间的差异。

### 3.2.2假设检验的应用
假设检验可以用于测试某个假设是否可以接受。例如，如果我们想要判断两个产品之间的差异是否有统计学意义，可以进行双样本假设检验。通过比较两个样本之间的差异，可以判断 Null 假设是否可以接受。

以下是一个实现假设检验的 Python 代码实例：

```python
import numpy as np
from scipy import stats

# 设定 Null 假设和替代假设
H0 = "两个样本来自于相同的总体"
H1 = "两个样本来自于不同的总体"

# 设定检验水平
alpha = 0.05

sample1_data = np.random.normal(loc=0, scale=1, size=n_prime)

else:
    print("Null 假设被接受")
```
