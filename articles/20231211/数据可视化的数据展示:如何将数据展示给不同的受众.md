```python
import matplotlib.pyplot as plt
import numpy as np

# 生成随机数据
x = np.random.rand(100)
y = np.random.rand(100)

# 创建散点图
plt.scatter(x, y)

# 添加标签和标题
plt.xlabel('x')
plt.ylabel('y')
plt.title('Scatter Plot')

# 显示图像
plt.show()
```

在这个代码实例中，我们使用Python的Matplotlib库创建了一个散点图。我们首先导入了Matplotlib和NumPy库，然后生成了100个随机的x和y值。然后，我们使用`plt.scatter()`函数创建了一个散点图，将x和y值作为参数传递给该函数。接下来，我们使用`plt.xlabel()`、`plt.ylabel()`和`plt.title()`函数添加了标签和标题。最后，我们使用`plt.show()`函数显示了图像。

这个代码实例展示了如何使用Python的Matplotlib库创建一个简单的散点图。你可以根据你的需求修改数据和可视化方法，以创建更复杂的图表和图像。
