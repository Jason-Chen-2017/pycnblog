# 2.核心概念与联系

在了解YARN与IoT框架集成之前，我们需要了解它们的核心概念和联系。

YARN是一个分布式应用程序资源调度框架，它可以为大规模数据处理任务提供高性能和高可靠性。YARN由ResourceManager和NodeManager组成，ResourceManager负责调度和分配资源，NodeManager负责执行任务。YARN支持多种计算框架，如Hadoop MapReduce、Spark等。

IoT框架则是用于管理物联网设备的框架，它可以实现设备的连接、数据收集、处理和分析。IoT框架通常包括设备管理、数据管理、应用管理等模块，以实现设备的连接、数据的收集、存储和处理，以及应用程序的开发和部署。

YARN与IoT框架的集成，可以将YARN的高性能资源调度能力与IoT框架的物联网设备管理能力结合起来，实现高性能的物联网数据处理。
