## 4.2 敏捷开发设计实例

我们来实现一个简单的计算器应用，包括加法、减法、乘法、除法等功能。

### 4.2.1 前端开发

我们使用Vue.js来实现前端。首先，我们需要创建一个Vue项目：

```
vue create my-calculator
```

然后，我们需要安装一些依赖：

```
npm install axios --save
```

接下来，我们需要创建一个`App.vue`文件，用于定义计算器的布局：

```html
<template>
  <div id="app">
    <h1>Calculator</h1>
    <input v-model="number1" type="number" />
    <select v-model="operator">
      <option value="+">+</option>
      <option value="-">-</option>
      <option value="*">*</option>
      <option value="/">/</option>
    </select>
    <input v-model="number2" type="number" />
    <button @click="calculate">=</button>
    <p>Result: {{ result }}</p>
  </div>
</template>

<script>
export default {
  name: 'App',
  data() {
    return {
      number1: 0,
      operator: '+',
      number2: 0,
      result: 0
    };
  },
  methods: {
    calculate() {
      switch (this.operator) {
        case '+':
          this.result = this.number1 + this.number2;
          break;
        case '-':
          this.result = this.number1 - this.number2;
          break;
        case '*':
          this.result = this.number1 * this.number2;
          break;
        case '/':
          this.result = this.number1 / this.number2;
          break;
      }
    }
  }
};
</script>
```

### 4.2.2 后端开发

我们使用Node.js来实现后端。首先，我们需要安装Node.js和NPM：

```
npm install express --save
```

接下来，我们需要创建一个`app.js`文件，用于定义后端的API：

```javascript
const express = require('express');
const app = express();

app.get('/calculate', (req, res) => {
  const { number1, operator, number2 } = req.query;
  let result;

  switch (operator) {
    case '+':
      result = Number(number1) + Number(number2);
      break;
    case '-':
      result = Number(number1) - Number(number2);
      break;
    case '*':
      result = Number(number1) * Number(number2);
      break;
    case '/':
      result = Number(number1) / Number(number2);
      break;
  }

  res.send({ result });
});

app.listen(3000, () => {
  console.log('Server is running on port 3000');
});
```
