# 2.核心概念与联系

## 2.1 自动编码器

自动编码器是一种神经网络模型，它由输入层、隐藏层和输出层组成。输入层接收输入数据，隐藏层通过学习来压缩输入数据，输出层通过学习来解压缩压缩后的数据，从而实现数据的降维和增维。

自动编码器的目标是最小化输入数据与输出数据之间的差异，即最小化重构误差。通过学习压缩和解压缩的过程，自动编码器可以学习数据的特征表示，从而实现数据的降维和增维。

## 2.2 图像超分辨率

        super(Autoencoder, self).__init__()
        self.encoder = nn.Sequential(
            nn.Linear(3, 128),
            nn.ReLU(),
            nn.Linear(128, 64),
            nn.ReLU()
        )
        self.decoder = nn.Sequential(
            nn.Linear(64, 128),
            nn.ReLU(),
            nn.Linear(128, 3),
            nn.Tanh()
        )

    def forward(self, x):
        encoded = self.encoder(x)
        decoded = self.decoder(encoded)
        return decoded

# 初始化自动编码器
autoencoder = Autoencoder()

# 定义损失函数和优化器

# 训练自动编码器

    # 随机生成低分辨率图像

    decoded = autoencoder(x)
    # 计算重构误差
    loss = criterion(decoded, x)
    # 更新自动编码器的参数
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()

# 使用自动编码器进行图像超分辨率恢复
x_hat = autoencoder(x)
