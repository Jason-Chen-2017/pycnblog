## 4.3 并行算法实例
### 4.3.1 多线程实例
```python
# 导入线程模块
import threading

# 定义线程函数
def thread_function(x, n, step):
    for i in range(n):
        x = x + step

# 定义初始值
x = 1.0

# 定义迭代次数
n = 1000

# 定义迭代步长
step = 0.01

# 创建多线程
threads = []
for i in range(100):
    t = threading.Thread(target=thread_function, args=(x, n, step))
    threads.append(t)
    t.start()

# 等待多线程完成
for t in threads:
    t.join()

# 输出迭代结果
print(x)
```
在上述代码中，我们通过多线程来求解数字模型。我们首先定义了线程函数、初始值、迭代次数和迭代步长，然后创建多个线程并启动它们，最后等待多线程完成并输出迭代结果。
