## 4.1 原子核的性质和行为

原子核的性质和行为可以通过量子力学来描述。以下是一个简单的代码实例，用于计算原子核的能量：

```python
import numpy as np
from scipy.constants import h, me, c

def atomic_energy(Z, A):
    """
    计算原子核的能量

    Parameters:
        Z (int): 原子核的质数
        A (int): 原子核的质量数

    Returns:
        float: 原子核的能量
    """
    # 计算原子核的质量
    mass = A * me

    # 计算原子核的电子质量
    e = Z * me

    # 计算原子核的能量
    energy = mass * c**2 - e * c**2

    return energy

# 计算铂原子核的能量
energy = atomic_energy(82, 208)
print(energy)
```

在上述代码中，我们首先导入了 numpy 和 scipy.constants 库，然后定义了一个 `atomic_energy` 函数，用于计算原子核的能量。`atomic_energy` 函数接受原子核的质数 `Z` 和质量数 `A` 作为参数，并返回原子核的能量。

我们使用铂原子为例，计算其原子核的能量。铂原子的质数为 82，质量数为 208。我们将这些参数传递给 `atomic_energy` 函数，并将计算结果打印出来。
