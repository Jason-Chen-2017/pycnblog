Redis（Remote Dictionary Server，远程字典服务器）是一个开源的高性能的key-value存储系统，它支持数据的持久化，可基于内存（in-memory）也可基于磁盘。Redis 提供多种语言的 API，包括：Ruby、Python、Java、C 和 C#，通过 Redis 客户端可与服务器进行通信，实现各种各样的数据结构：字符串(string)、哈希(hash)、列表(list)、集合(sets)和有序集合(sorted sets)等。

Redis 的核心特性有：数据的持久化、高性能、丰富的数据类型、集群支持、可扩展性、跨平台等。Redis 是一个开源的使用 ANSI C 语言编写、遵循 BSD 协议的高性能的 key-value 数据库，并提供多种语言的 API。Redis 可以在内存中存储数据，也可以将某些数据保存在磁盘上，以达到持久化的目的。
