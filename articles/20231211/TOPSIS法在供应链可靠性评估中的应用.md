# 2.核心概念与联系

在讨论TOPSIS法在供应链可靠性评估中的应用之前，我们需要了解一些核心概念。

## 2.1 供应链可靠性评估

供应链可靠性评估是一种用于评估供应链的可靠性的方法。它可以帮助企业更好地管理供应链风险，提高供应链的稳定性和可靠性。供应链可靠性评估包括以下几个方面：

1. 供应链风险评估：评估供应链中的各种风险，如供应商风险、物流风险、市场风险等。
2. 供应链可靠性指标：定义供应链可靠性的指标，如供应链渠道的多样性、供应链流动性等。
3. 供应链可靠性评估模型：根据供应链可靠性指标，建立供应链可靠性评估模型，并对各个供应链节点进行评估。

## 2.2 TOPSIS法

TOPSIS（Technique for Order Preference by Similarity to Ideal Solution）是一种多标准评估方法，它可以帮助我们在多个标准下选择最优解。TOPSIS法的核心思想是：在多个标准下，选择距离最近的理想解的选项。

TOPSIS法的主要步骤包括：

1. 建立评估指标系统：定义评估的目标和标准。
2. 构建评估矩阵：将各个选项的评估结果填充到评估矩阵中。
3. 计算各个选项的权重：根据各个标准的重要性，计算各个选项的权重。
4. 计算各个选项的相似度：计算各个选项与理想解之间的相似度。
5. 选择最优解：选择距离理想解最近的选项。
