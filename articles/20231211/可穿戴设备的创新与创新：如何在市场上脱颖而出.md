# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的例子来说明如何实现可穿戴设备的创新。我们将创建一个简单的智能手环，用于监测用户的心率。

```python
import time
import board
import busio
import adafruit_bme280

# 初始化BME280传感器
i2c_bus = busio.I2C(board.SCL, board.SDA)
sensor = adafruit_bme280.Adafruit_BME280_I2C(i2c_bus)

# 获取心率数据
def get_heart_rate():
    # 获取心率数据
    heart_rate = sensor.heart_rate
    return heart_rate

# 主程序
while True:
    # 获取心率数据
    heart_rate = get_heart_rate()
    print("Heart rate: {:.2f} bpm".format(heart_rate))
    time.sleep(1)
```

这个代码实例使用了Adafruit的BME280库来获取心率数据。首先，我们初始化了BME280传感器，然后定义了一个`get_heart_rate`函数来获取心率数据。在主程序中，我们不断地获取心率数据并打印出来。
