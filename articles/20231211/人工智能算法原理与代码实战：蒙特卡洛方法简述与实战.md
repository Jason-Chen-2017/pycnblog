## 2.1 蒙特卡洛方法的基本思想

蒙特卡洛方法的基本思想是利用大量随机数生成的方法来近似求解问题的解。这种方法的核心是通过大量的随机样本来估计问题的解，而不是通过严格的数学推导来得到确定的解。这种方法的名字来源于法国数学家蒙特卡洛。

## 2.2 蒙特卡洛方法与其他随机算法的区别

蒙特卡洛方法与其他随机算法的区别在于其核心思想。其他随机算法如随机搜索算法、随机优化算法等，都是通过随机性来找到问题的最优解。而蒙特卡洛方法则是通过大量的随机样本来估计问题的解，而不是通过找到最优解来解决问题。

```python
import random

def monte_carlo(n):
    # 生成大量的随机数
    random_numbers = [random.random() for _ in range(n)]

    # 计算样本结果
    sample_results = [x ** 2 for x in random_numbers]

    # 统计结果
    mean = sum(sample_results) / n

    # 判断准确性
    accuracy = mean / (n - 1)

    return mean, accuracy

# 使用蒙特卡洛方法计算π
n = 100000
mean, accuracy = monte_carlo(n)
print("估计的π值:", mean)
print("估计的准确性:", accuracy)
```
