## 4.1 均值方差

```python
import cv2
import numpy as np

def mean_variance(image_path):
    # 读取图像文件
    image = cv2.imread(image_path)
    # 计算图像像素值的平均值
    mean = np.mean(image)
    # 计算平均值的方差
    variance = np.var(image)
    # 输出结果
    print('Mean:', mean)
    print('Variance:', variance)

if __name__ == '__main__':
    mean_variance(image_path)
```

解释说明：

1. 首先，我们使用 `cv2.imread()` 函数读取图像文件。
2. 然后，我们使用 `np.mean()` 函数计算图像像素值的平均值。
3. 接着，我们使用 `np.var()` 函数计算平均值的方差。
4. 最后，我们使用 `print()` 函数输出结果。

## 4.2 对比度

```python
import cv2
import numpy as np

def contrast(image_path):
    # 读取图像文件
    image = cv2.imread(image_path)
    # 计算图像像素值的最大值和最小值
    max_value = np.max(image)
    min_value = np.min(image)
    # 计算最大值和最小值之间的差值
    contrast = max_value - min_value
    # 输出结果
    print('Contrast:', contrast)

if __name__ == '__main__':
    contrast(image_path)
```

解释说明：

1. 首先，我们使用 `cv2.imread()` 函数读取图像文件。
2. 然后，我们使用 `np.max()` 函数计算图像像素值的最大值，并使用 `np.min()` 函数计算图像像素值的最小值。
3. 接着，我们使用 `-` 运算符计算最大值和最小值之间的差值。
4. 最后，我们使用 `print()` 函数输出结果。

import cv2
import numpy as np

def structural_information(image_path):
    # 读取图像文件
    image = cv2.imread(image_path)
    # 计算图像像素值之间的相关性
    correlation = np.corrcoef(image.ravel())[0, 1]
