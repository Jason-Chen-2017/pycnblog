```python
class Student:
    def __init__(self, name, age, gender):
        self.name = name
        self.age = age
        self.gender = gender

    def get_info(self):
        return f"姓名：{self.name}, 年龄：{self.age}, 性别：{self.gender}"

    def set_age(self, age):
        self.age = age

    def set_gender(self, gender):
        self.gender = gender

    @staticmethod
    def check_age(age):
        if age < 0:
            return False
        else:
            return True


def main():
    student = Student("张三", 20, "男")
    student.set_age(21)
    student.set_gender("女")
    print(student.get_info())
    print(Student.check_age(-1))


if __name__ == "__main__":
    main()
```

在这个代码实例中，我们对上一个代码实例进行了内心平静的设计。我们使用`@staticmethod`装饰器来定义一个静态方法`check_age`，用于检查年龄是否为负数。这样，我们可以更内心平静地处理年龄的检查问题，以便更好地保证程序的质量。
