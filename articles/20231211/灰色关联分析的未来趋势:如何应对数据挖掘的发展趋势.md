# 2.核心概念与联系

## 2.1 关联规律
关联规律是数据挖掘中的一个重要概念，它指的是在大量数据中找到两个或多个变量之间存在的关联关系。这种关联关系可以帮助我们更好地理解数据，并为决策提供依据。

## 2.2 灰色关联分析
灰色关联分析是一种新兴的数据挖掘方法，它可以在大量数据中找到隐藏的关联关系。与传统的关联规律分析不同，灰色关联分析不需要事先知道变量之间的关系，而是通过对数据的分析来发现关联规律。

## 2.3 与传统关联规律分析的区别
传统关联规律分析通常需要事先知道变量之间的关系，并且需要对数据进行预处理。而灰色关联分析则可以在大量数据中找到隐藏的关联关系，而无需事先知道变量之间的关系。

皮尔逊相关系数的计算公式为：
$$
r = \frac{\sum_{i=1}^{n}(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i-\bar{x})^2}\sqrt{\sum_{i=1}^{n}(y_i-\bar{y})^2}}
