# 2.核心概念与联系

在计算机视觉中，我们主要关注以下几个核心概念：

- 图像：图像是计算机视觉的基本数据结构，是由像素组成的二维矩阵。每个像素包含红色、绿色和蓝色三个通道的颜色信息，通常用RGB格式表示。
- 图像处理：图像处理是对图像进行预处理、增强、压缩等操作，以提高图像质量或简化后续的计算机视觉任务。
- 图像特征：图像特征是图像中的某些特点，可以用来表示图像的结构和信息。例如，边缘、角、文字等。
- 图像分类：图像分类是将图像分为多个类别的任务，例如人脸识别、垃圾分类等。
- 目标检测：目标检测是在图像中找出特定目标的任务，例如人脸识别、自动驾驶汽车等。
- 目标跟踪：目标跟踪是在视频序列中跟踪目标的任务，例如人脸识别、自动驾驶汽车等。
