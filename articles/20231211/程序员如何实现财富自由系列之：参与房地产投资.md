```python
# 使用移动平均线对价格数据进行平滑处理

data['change'] = data['price'].pct_change()

# 使用线性回归模型进行预测
model = ARIMA(data['price'], order=(1, 1, 1))
model_fit = model.fit()

# 预测未来价格

```

## 4.2 房地产投资组合优化
以下是一个简单的Python代码实例，使用权重优化方法进行房地产投资组合优化：

```python
import numpy as np
from scipy.optimize import minimize

# 收集房地产市场的相关信息
market_info = pd.read_csv('market_info.csv')

# 根据这些信息，为每个房地产市场分配权重
def objective_function(weights):
    return np.sum(weights * market_info['price'])

# 使用优化算法，找到最佳的投资组合
initial_weights = np.array([1/len(market_info) for _ in market_info])
result = minimize(objective_function, initial_weights, method='SLSQP')

# 得到最佳的投资组合
optimal_weights = result.x
```
