                 

# 1.背景介绍

数字化证券的发展是近年来金融市场创新的重要驱动力之一。随着数字化证券的不断发展，金融市场也在不断创新，这种创新为金融市场带来了更多的机遇和挑战。

数字化证券是指通过数字化技术实现的证券交易，包括数字证券、数字货币、数字资产等。数字化证券的发展主要受到以下几个方面的影响：

1. 技术进步：随着区块链、人工智能、大数据等技术的不断发展，数字化证券的技术实现得以实现。

2. 政策支持：政府和监管机构对数字化证券的支持也是其发展的重要因素。

3. 市场需求：随着市场需求的增加，数字化证券的市场份额也在不断增加。

数字化证券的发展为金融市场创新带来了以下几个方面的影响：

1. 提高交易效率：数字化证券的交易可以实现更快速、更高效的交易，降低交易成本。

2. 增强市场透明度：数字化证券的交易可以实现更高的市场透明度，提高投资者的信心。

3. 推动金融市场的国际化：数字化证券的发展可以推动金融市场的国际化，提高国际金融市场的竞争力。
