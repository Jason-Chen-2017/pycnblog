# 2.核心概念与联系
在讨论软件架构之前，我们需要了解一些关于软件架构的基本概念。软件架构是指软件系统的组件和它们之间的关系。它是软件系统的蓝图，用于指导软件的设计和开发过程。

软件架构可以分为两种类型：基于组件的架构和基于服务的架构。基于组件的架构是一种基于组件的设计方法，它将软件系统划分为一组可组合的组件，这些组件可以独立地实现特定的功能。基于服务的架构是一种基于服务的设计方法，它将软件系统划分为一组可以独立运行的服务，这些服务可以通过网络进行交互。
