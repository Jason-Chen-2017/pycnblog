角色授权是一种基于访问控制的安全性机制，它将数据库用户分为不同的角色，并为每个角色分配一组权限。这样，用户可以根据他们的角色来访问数据库中的数据和操作。

具体的操作步骤如下：

1. 创建数据库用户：
```sql

```

2. 创建角色：
```sql

```

3. 分配权限：
```sql
GRANT SELECT, INSERT, UPDATE, DELETE ON mydb.* TO 'admin';
```

4. 授权用户角色：
```sql
GRANT 'admin' TO 'admin'@'localhost';
```
