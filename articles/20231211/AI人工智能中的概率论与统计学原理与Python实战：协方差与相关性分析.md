# 2.核心概念与联系

在概率论与统计学中，协方差和相关性是两个重要的概念。它们可以用来衡量两个随机变量之间的关系。协方差是一种度量两个随机变量之间变化方向和程度的量，而相关性是一种度量两个随机变量之间的线性关系的量。

协方差是一种度量两个随机变量之间变化方向和程度的量。它可以用来衡量两个随机变量之间的关系，包括它们之间的方向和程度。协方差的数学模型公式如下：

$$
\text{Cov}(X,Y) = E[(X - \mu_X)(Y - \mu_Y)]
$$

其中，$E$ 是期望操作符，$\mu_X$ 和 $\mu_Y$ 是随机变量 $X$ 和 $Y$ 的期望值。

相关性是一种度量两个随机变量之间的线性关系的量。它可以用来衡量两个随机变量之间的关系，包括它们之间的方向和强度。相关性的数学模型公式如下：

$$
\text{Corr}(X,Y) = \frac{\text{Cov}(X,Y)}{\sqrt{\text{Var}(X)\text{Var}(Y)}}
$$

其中，$\text{Var}(X)$ 和 $\text{Var}(Y)$ 是随机变量 $X$ 和 $Y$ 的方差。
