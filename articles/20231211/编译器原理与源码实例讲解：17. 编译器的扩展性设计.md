在具体代码实例上，我们可以看一个简单的词法分析器的实现：

```python
class Lexer:
    def __init__(self, source_code):
        self.source_code = source_code
        self.position = 0

    def next_token(self):
        char = self.source_code[self.position]
        if char == '+':
            self.position += 1
            return 'PLUS'
        elif char == '-':
            self.position += 1
            return 'MINUS'
        elif char == '*':
            self.position += 1
            return 'MUL'
        elif char == '/':
            self.position += 1
            return 'DIV'
        elif char.isdigit():
            value = ''
            while char.isdigit():
                value += char
                self.position += 1
                char = self.source_code[self.position]
            return 'NUMBER', int(value)
        else:
            return None
```

在这个实例中，我们可以看到词法分析器是如何将源代码划分为一系列的词法单元的。我们可以看到，词法分析器可以识别加法、减法、乘法和除法运算符，以及数字。这个实例展示了如何实现一个简单的词法分析器，并且可以扩展到支持更多的词法单元类型和定义。
