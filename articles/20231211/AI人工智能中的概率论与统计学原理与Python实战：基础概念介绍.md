# 2.核心概念与联系
# 2.1概率论基本概念
概率论是一门研究不确定性的数学学科，它可以帮助我们更好地理解和处理不确定性。概率论的基本概念包括事件、样本空间、概率空间、事件的独立性和条件独立性等。

## 2.1.1事件
事件是概率论中的一个基本概念，它表示一个可能发生的结果。事件可以是确定的（例如：抛硬币正面）或不确定的（例如：抛硬币正面或反面）。

## 2.1.2样本空间
样本空间是概率论中的一个基本概念，它表示所有可能发生的结果的集合。样本空间可以是有限的、有序的或无序的。

## 2.1.3概率空间
概率空间是概率论中的一个基本概念，它表示一个事件发生的可能性。概率空间可以是离散的或连续的。

## 2.1.4事件的独立性
事件的独立性是概率论中的一个基本概念，它表示两个事件之间是否相互独立。如果两个事件之间是相互独立的，那么它们的发生或不发生是不会影响彼此的。

## 2.1.5条件独立性
条件独立性是概率论中的一个基本概念，它表示一个事件发生的可能性是否依赖于另一个事件的发生或不发生。如果两个事件之间是条件独立的，那么它们的发生或不发生是会影响彼此的。

混合概率模型是一种描述一个事件发生的可能性的概率模型，它可以用来描述一个混合的事件集合。混合概率模型可以是离散的、连续的或混合的。

def conditionally_independent_events(event1, event2):
    return {(event1_name, event2_name): event1[event1_name] * event2[event2_name] for event1_name, event2_name in event1.items() if event2_name in event2.items()}

# 4.4条件概率和贝叶斯定理
## 4.4.1条件概率
```python
def conditional_probability(event1, event2):
    return {event1: event1_probability * event2_probability / event2_probability for event1, event2_probability in event1.items() if event2_probability in event2.items()}
```
```python
conditional_probability_1 = conditional_probability(event_1, event_2)
```
## 4.4.2贝叶斯定理
```python
def bayes_theorem(event1, event2):
    return {event1: event1_probability * event2_probability / event2_probability for event1, event2_probability in event1.items() if event2_probability in event2.items()}
```
```python
bayes_theorem_1 = bayes_theorem(event_1, event_2)
```
