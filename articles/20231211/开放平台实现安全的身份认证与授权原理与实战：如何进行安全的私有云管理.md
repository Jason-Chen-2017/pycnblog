# 2.核心概念与联系

身份认证与授权是计算机安全的基本概念，它们在私有云管理中起着关键作用。身份认证是确认用户身份的过程，而授权是确定用户在系统中可以执行哪些操作的过程。在私有云管理中，身份认证与授权的核心概念包括：

1.用户身份：用户在系统中的唯一标识，通常是用户名或者邮箱地址。

2.密码：用户在系统中的密码，用于验证用户身份。

3.授权：用户在系统中可以执行的操作，如创建、删除、修改等。

4.角色：用户在系统中的角色，用于定义用户的权限和限制。

5.权限：用户在系统中的权限，用于确定用户可以执行哪些操作。

6.访问控制列表（ACL）：用于定义用户在系统中的访问权限。
