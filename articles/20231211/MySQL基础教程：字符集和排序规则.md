# 2.核心概念与联系

在MySQL中，字符集和排序规则是密切相关的两个概念。字符集用于将字符映射到二进制，而排序规则则用于根据字符集对字符进行排序。

字符集可以分为两类：字符集和字符编码。字符集是一种编码方案，用于将字符映射到二进制。字符编码则是字符集的一种具体实现，用于将字符映射到特定的二进制形式。例如，ASCII字符集是一种编码方案，用于将英文字符映射到二进制，而UTF-8字符集是一种具体的字符编码，用于将多种语言的字符映射到二进制。

排序规则则是根据字符集对字符进行排序的方法。在MySQL中，我们可以使用不同的排序规则来实现不同的排序需求。例如，我们可以使用ASCII排序规则来对英文字符进行排序，或者使用UTF-8排序规则来对多种语言的字符进行排序。

在MySQL中，我们可以使用COLLATE关键字来指定排序规则。例如，我们可以使用COLLATE ASCII_GENERAL_CI来指定使用ASCII排序规则进行排序，或者使用COLLATE UTF8_GENERAL_CI来指定使用UTF-8排序规则进行排序。

