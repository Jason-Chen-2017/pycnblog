## 1.4 智能家居设备的具体代码实例

以下是一个简单的智能家居设备的代码实例：

```python
import time
import requests

# 定义一个类，用于表示智能家居设备
class SmartHomeAppliance:
    def __init__(self, device_name, device_ip):
        self.device_name = device_name
        self.device_ip = device_ip

    def turn_on(self):
        # 向设备发送请求，使其打开
        url = f"http://{self.device_ip}/on"
        response = requests.get(url)
        if response.status_code == 200:
            print(f"{self.device_name} is turned on.")
        else:
            print(f"Failed to turn on {self.device_name}.")

    def turn_off(self):
        # 向设备发送请求，使其关闭
        url = f"http://{self.device_ip}/off"
        response = requests.get(url)
        if response.status_code == 200:
            print(f"{self.device_name} is turned off.")
        else:
            print(f"Failed to turn off {self.device_name}.")

# 创建一个智能家居设备对象
smart_home_appliance = SmartHomeAppliance("Light", "192.168.1.100")

# 打开灯
smart_home_appliance.turn_on()

# 关闭灯
smart_home_appliance.turn_off()
```
