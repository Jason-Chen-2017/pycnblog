## 4.1 代码实例

我们将通过一个简单的例子来解释马尔可夫链的概念和算法。我们将创建一个简单的马尔可夫链，其中有三个状态：A、B和C。我们将计算每个状态的转移概率，并使用动态规划来求解状态转移问题。

```python
import numpy as np

# 初始化状态
state_prob = np.array([0.5, 0.3, 0.2])

# 计算状态转移概率
transition_prob = np.array([
    [0.7, 0.2, 0.1],
    [0.3, 0.5, 0.2],
    [0.2, 0.3, 0.5]
])

# 使用动态规划求解
dp = np.zeros((3, 3))
dp[0, 0] = state_prob[0]

for i in range(1, 3):
    for j in range(3):
        for k in range(3):
            dp[i, j] = max(dp[i - 1, k] * transition_prob[k, j], dp[i, j])

# 输出结果
print(dp)
```

在上述代码中，我们首先初始化了马尔可夫链的状态。我们使用了一个概率向量来表示每个状态的初始概率。

接下来，我们计算了状态转移概率。我们使用了贝叶斯定理来计算每个状态的转移概率。
