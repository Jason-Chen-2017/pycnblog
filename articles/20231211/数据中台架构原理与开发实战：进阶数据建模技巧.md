                 

# 1.背景介绍

数据中台架构是一种可扩展、可重用、可集成的数据处理平台，旨在提供数据处理能力，为企业内部和外部的各种应用提供数据支持。数据中台架构的核心是将数据处理过程抽象为可复用的服务，实现数据的一次性处理，一次性发布，一次性消费。数据中台架构包括数据收集、数据清洗、数据处理、数据分析、数据存储等多个模块，为企业内部和外部的各种应用提供数据支持。
