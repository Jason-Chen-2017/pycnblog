# 2.核心概念与联系

在讨论宇宙大爆炸之前，我们需要了解一些核心概念，如宇宙、宇宙膨胀、原子核、氢原子等。

## 2.1 宇宙

宇宙是指整个宇宙的物质和能量的总和，包括星系、星球、行星、恒星、行星系统、星云、黑洞、星球系统等。宇宙的诞生和演变是由宇宙大爆炸引起的。

## 2.2 宇宙膨胀

宇宙膨胀是指宇宙的尺寸在时间的推移过程中不断增大的过程。根据宇宙大爆炸理论，宇宙的膨胀是由于宇宙中的物质和能量的分离和扩散所引起的。

## 2.3 原子核

原子核是原子的核心部分，是原子中的所有稳定和不稳定的核子所位于的地方。原子核由一种或多种核子组成，核子是原子核中的最小稳定部分。

## 2.4 氢原子

氢原子是一种轻原子，由一个正电子和一个核子组成。氢原子是宇宙大爆炸中产生的第一个原子，它是宇宙中最重要的原子之一。

宇宙膨胀率是指宇宙在一定时间内的膨胀速度与光速的比值。它是用来描述宇宙膨胀的一个重要参数。

其中，$z$ 是红移，$\lambda_0$ 是光源的波长，$\lambda$ 是观测到的波长。

## 3.3 宇宙膨胀模型

宇宙膨胀模型是用来描述宇宙膨胀过程的一个理论框架。目前有三种主要的宇宙膨胀模型：

```python
import math

def calculate_expansion_rate(R, c):
    return R * c

def calculate_redshift(lambda_0, lambda_):
    return (lambda_0 - lambda_) / lambda_0

R = 10  # 宇宙尺寸，以光年为单位
c = 3e8  # 光速，以米/秒为单位

expansion_rate = calculate_expansion_rate(R, c)

print("宇宙膨胀率:", expansion_rate)
print("红移:", redshift)
```
