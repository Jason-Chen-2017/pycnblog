```python
def fibonacci(n):
    if n == 0 or n == 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```

在这个递归算法中，我们首先检查输入的数字n是否满足递归终止条件。如果n等于0或1，则直接返回n。否则，我们递归地调用fibonacci函数，直到满足递归终止条件。

```python
def longest_subsequence(arr):
    n = len(arr)
    dp = [1] * n

    for i in range(1, n):
        for j in range(i):
            if arr[i] > arr[j]:
                dp[i] = max(dp[i], dp[j] + 1)
```

```python
def shortest_path(graph, start, end):
    n = len(graph)
    dist = [float('inf')] * n
    dist[start] = 0

    for _ in range(n):
        for node, weight in graph[dist[node].index(min(dist[node]))]:
```
