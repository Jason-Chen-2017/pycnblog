                 

# 1.背景介绍

Redis（Remote Dictionary Server，远程字典服务器）是一个开源的高性能的key-value存储系统，由Salvatore Sanfilippo开发。Redis支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。Redis不仅仅支持简单的key-value类型的数据，同时还提供list、set、hash和sorted set等数据结构的存储。

Redis支持数据的备份、读写分离、集群等。它和应用服务器是单独跑的，通过网络，应用服务器可以与Redis进行通信。Redis支持TCP/IP和UnixSocket协议，可以在内网和外网进行通信。

Redis是一个非关系型数据库，是一个高性能的key-value数据库，是一个可扩展的数据结构服务器，是一个提供多种语言API的数据存储服务器。

Redis的核心特点：

1. 内存数据库：Redis是内存数据库，数据存储在内存中，所以读写速度非常快。

2. 数据结构：Redis支持字符串(string)、列表(list)、集合(set)、有序集合(sorted set)、哈希(hash)等五种数据结构类型。

3. 数据持久化：Redis支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。

4. 集群：Redis支持集群，可以实现数据的读写分离和故障转移。

5. 高性能：Redis的性能非常高，可以达到100000次/秒的读写速度。

6. 开源：Redis是一个开源的数据库，可以免费使用。
