# 2.核心概念与联系
在KNIME中，数据质量检查主要包括数据清洗、数据校验、数据补全和数据纠正等方面。这些方法可以帮助我们发现和修复数据中的错误和不一致性，从而提高数据的质量。

数据清洗是指对数据进行预处理和筛选，以移除噪声、缺失值、重复值等。数据校验是指对数据进行验证，以确保其符合预期的格式和规则。数据补全是指对缺失值进行补充，以填补数据中的空白。数据纠正是指对错误值进行修正，以改善数据的准确性。

这些方法之间存在密切的联系。例如，数据清洗和数据校验可以相互补充，共同提高数据质量。数据补全和数据纠正可以相互依赖，共同改进数据的准确性。
