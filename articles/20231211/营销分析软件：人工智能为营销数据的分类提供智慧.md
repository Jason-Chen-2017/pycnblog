```python
# 数据预处理
X = np.hstack((np.ones((X.shape[0], 1)), X))

# 模型训练
model.fit(X, y)

# 模型验证
X_test = np.array([[7], [8]])
y_pred = model.predict(X_test)

# 模型应用
print(y_pred)
```

接下来，我们选择了线性回归算法，并使用Scikit-learn库中的LinearRegression类实例化模型。然后，我们使用训练数据进行模型训练。

接下来，我们使用验证数据进行模型验证。我们创建了一组新的输入变量X_test，并使用模型进行预测，得到预测结果y_pred。
