# 2.核心概念与联系

在讨论量子光学在空间探测领域的应用之前，我们需要了解一些核心概念。

## 2.1 光子

光子是光的基本单位，它是一个微小的能量量子。光子具有波粒二象性，即它同时具有波动性和粒子性。光子的速度是3.00×10^8 m/s，它可以传播在空气、玻璃、水等多种媒介中。

## 2.2 量子光学

量子光学是一门研究光子的科学，它研究光子的性质、行为和应用。量子光学与传统的光学有很大的不同，主要体现在以下几个方面：

1. 量子光学关注光子的行为，而传统的光学关注光线的行为。
2. 量子光学研究光子的波粒二象性，而传统的光学研究光线的波动性和粒子性。
3. 量子光学研究光子的量子态，而传统的光学研究光线的状态。

## 2.3 空间探测

空间探测是研究宇宙的一种方法，它涉及到观测天体、探测行星、研究宇宙的组成等等。空间探测技术的发展对于我们对宇宙的了解非常重要。空间探测可以分为两个部分：

1. 地球轨道探测：地球轨道探测是指在地球轨道上进行的探测活动，例如卫星观测、卫星导航等。
2. 深空探测：深空探测是指在地球轨道之外进行的探测活动，例如探测行星、探测星球等。
