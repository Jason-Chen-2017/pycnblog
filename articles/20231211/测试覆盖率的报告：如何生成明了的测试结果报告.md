```python

```

我们有以下测试用例：

```python

    assert add(1, 2) == 3
    assert add(-1, -2) == -3

    assert subtract(1, 2) == -1
    assert subtract(-1, -2) == 1
```

我们可以使用以下方法来计算测试覆盖率：

1. 使用代码覆盖工具：我们可以使用Python的unittest模块来测试代码，并使用coverage模块来计算测试覆盖率。

首先，我们需要安装coverage模块：

```bash
pip install coverage
```

然后，我们可以使用以下命令来测试代码并计算覆盖率：

```bash
coverage run -m unittest tests.py
coverage report
```

这将生成一个覆盖率报告，如下所示：

```

----------------------------------------------
tests.py            10      0   100%
----------------------------------------------
TOTAL                10      0   100%
```

从报告中我们可以看到，我们的测试用例覆盖了所有的代码行数，覆盖率为100%。

2. 手动计算：我们可以手动计算测试用例覆盖的代码行数。在上面的例子中，我们的测试用例覆盖了所有的代码行数，因此覆盖率为100%。
