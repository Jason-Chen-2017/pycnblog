# 2.核心概念与联系

在深度学习中，我们通常使用神经网络来进行模型训练。神经网络由多个节点组成，每个节点表示一个特征或变量，节点之间通过权重连接起来，形成一个有向图。通过对神经网络的训练，我们可以学习出特征之间的关系，从而实现模型的预测和分类。

集成学习则是一种将多个模型结合起来进行学习的方法。通常，我们会使用多种不同的模型（如决策树、支持向量机、随机森林等）来进行学习，然后将这些模型的预测结果进行融合，从而实现更准确的预测和分类。

在深度学习与集成学习的结合中，我们可以将多个深度学习模型（如卷积神经网络、循环神经网络等）进行融合，从而实现更高效和准确的模型训练。
