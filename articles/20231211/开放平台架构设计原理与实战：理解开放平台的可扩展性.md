### 4.1 分布式架构实例

以下是一个使用Python实现的简单分布式架构的代码实例：

```python
import threading
import time

def worker(name):
    while True:
        # 模拟处理任务
        time.sleep(1)
        print(f'{name} 处理任务完成')

# 创建工作线程
workers = []
for i in range(5):
    worker_thread = threading.Thread(target=worker, args=(i,))
    workers.append(worker_thread)
    worker_thread.start()

# 等待所有工作线程完成
for worker in workers:
    worker.join()
```

在这个代码实例中，我们创建了5个工作线程，每个线程都负责处理任务。通过使用Python的`threading`模块，我们可以轻松地实现分布式架构。

### 4.2 微服务架构实例

以下是一个使用Python实现的简单微服务架构的代码实例：

```python
import grpc
from concurrent import futures
import time
import logging

# 定义服务接口
class Greeter(grpc.serve):
    def SayHello(self, request, context):
        return grpc.greetings.Greeting(message=f'Hello, {request.name}')

# 定义服务实现
class GreeterImpl(Greeter):
    def SayHello(self, request, context):
        time.sleep(1)
        return grpc.greetings.Greeting(message=f'Hello, {request.name}')

# 定义服务器
def serve():
    server = grpc.server(futures.ThreadPoolExecutor(max_workers=10))
    server.add_insecure_port('[::]:50051')
    greeter = GreeterImpl()
    server.add_service(greeter)
    server.start()
    print('Server started, listening on port 50051')
    server.wait_for_termination()

if __name__ == '__main__':
    logging.basicConfig.level=logging.INFO
    serve()
```

在这个代码实例中，我们使用了gRPC框架来实现微服务架构。我们定义了一个服务接口`Greeter`，并实现了一个服务实现`GreeterImpl`。通过使用gRPC框架，我们可以轻松地实现微服务架构。
