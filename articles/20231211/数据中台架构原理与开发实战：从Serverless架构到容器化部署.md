```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 数据清洗
data = data.dropna()  # 删除缺失值
data = data.astype({'column1': 'int', 'column2': 'float'})  # 转换数据类型
data = data.fillna(data.mean())  # 填充缺失值

# 保存数据
data.to_csv('cleaned_data.csv', index=False)
```

在这个代码实例中，我们使用了pandas库来读取和处理数据。我们首先读取了一个CSV文件，然后对数据进行了清洗操作，包括删除缺失值、转换数据类型和填充缺失值。最后，我们将清洗后的数据保存到一个新的CSV文件中。
