# 2.核心概念与联系

## 2.1协方差

协方差是用来衡量两个随机变量之间的线性关系的一个度量。给定两个随机变量X和Y，协方差定义为：

$$
\text{Cov}(X,Y) = E[(X-E[X])(Y-E[Y])]
$$

其中，E[X]和E[Y]分别表示X和Y的期望，E[(X-E[X])(Y-E[Y])]表示X和Y之间的协方差。协方差的正值表示X和Y之间存在正相关关系，负值表示X和Y之间存在负相关关系，零表示X和Y之间没有线性关系。

## 2.2相关系数

相关系数是协方差的标准化后的形式，它表示两个随机变量之间的线性关系的强度。给定两个随机变量X和Y，相关系数定义为：

$$
\text{Corr}(X,Y) = \frac{\text{Cov}(X,Y)}{\sqrt{\text{Var}(X)\text{Var}(Y)}}
$$

其中，Var(X)和Var(Y)分别表示X和Y的方差。相关系数的值范围在-1到1之间，其中-1表示X和Y之间存在完全负相关关系，1表示X和Y之间存在完全正相关关系，0表示X和Y之间没有线性关系。

协方差的计算主要包括以下几个步骤：

1. 计算X和Y的期望E[X]和E[Y]。
2. 计算(X-E[X])和(Y-E[Y])。

4. 计算E[(X-E[X])(Y-E[Y])]。

## 3.2相关系数的计算

相关系数的计算主要包括以下几个步骤：

1. 计算X和Y的协方差Cov(X, Y)。
2. 计算X和Y的方差Var(X)和Var(Y)。
3. 计算Cov(X, Y) / sqrt(Var(X) * Var(Y))。

具体实现可以使用以下Python代码：

```python
import numpy as np

def corr(X, Y):
    Cov_XY = np.cov(X, Y)[0, 1]
    Var_X = np.var(X)
    Var_Y = np.var(Y)
    return Cov_XY / np.sqrt(Var_X * Var_Y)
```
