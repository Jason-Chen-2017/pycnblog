## 4.1 排序算法实例

以下是一个简单的选择排序算法的实现：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if arr[min_idx] > arr[j]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr
```

这个算法的时间复杂度为O(n^2)，空间复杂度为O(1)。它的核心思想是在每次迭代中找到最小的元素，并将其与当前位置进行交换。

## 4.2 数学模型实例

以下是一个简单的线性方程组解决方案的实现：

```python
import numpy as np

def solve_linear_equation(A, b):
    x = np.linalg.solve(A, b)
    return x
```

这个算法的时间复杂度为O(n^3)，空间复杂度为O(n^2)。它使用了Numpy库的`linalg.solve`函数，该函数可以直接解决线性方程组。
