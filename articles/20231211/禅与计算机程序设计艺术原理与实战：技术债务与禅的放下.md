# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明上述方法的具体操作步骤。我们将实现一个简单的排序算法，即冒泡排序。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个代码实例中，我们首先定义了一个名为`bubble_sort`的函数，该函数接受一个整数列表作为参数。然后，我们使用两个嵌套的循环来实现冒泡排序的算法。在外部循环中，我们遍历整个列表，并在内部循环中比较相邻的元素。如果相邻的元素不满足排序的条件，我们就交换它们的位置。最后，我们返回排序后的列表。
