# 2.核心概念与联系

在环境科学中，概率论主要用于描述和分析环境变量的不确定性。环境变量可以是气候、大气、水、生态系统等等。环境变量的不确定性可以分为两种：一种是随机性，另一种是不确定性。随机性是指环境变量的变化遵循某种规律，但是具体的变化是不可预测的。不确定性是指环境变量的变化是由于某些未知因素的影响，这些因素不能通过现有的知识来预测。

概率论可以用来描述随机性和不确定性。概率是一个数值，表示某个事件发生的可能性。概率可以用来描述环境变量的随机性和不确定性。例如，我们可以用概率来描述气候变化的可能性，也可以用概率来描述水资源的可持续性。
