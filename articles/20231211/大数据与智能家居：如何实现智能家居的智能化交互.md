## 1.4 智能家居的具体代码实例

在这里，我们将通过一个简单的智能家居设备的代码实例来讲解智能家居的具体实现。

```python
import time
import requests

class SmartHomeDevice:
    def __init__(self, device_id, device_type, device_name):
        self.device_id = device_id
        self.device_type = device_type
        self.device_name = device_name

    def get_status(self):
        # 获取设备状态
        url = f'http://{self.device_id}.com/status'
        response = requests.get(url)
        if response.status_code == 200:
            return response.json()
        else:
            return None

    def control(self, command):
        # 控制设备
        url = f'http://{self.device_id}.com/control'
        payload = {'command': command}
        response = requests.post(url, json=payload)
        if response.status_code == 200:
            return response.json()
        else:
            return None

if __name__ == '__main__':
    device = SmartHomeDevice('device_id', 'device_type', 'device_name')
    status = device.get_status()
    if status:
        print(f'设备状态：{status}')
        device.control('command')
    else:
        print('获取设备状态失败')
```

在这个代码实例中，我们定义了一个`SmartHomeDevice`类，用于表示智能家居设备。通过这个类，我们可以获取设备的状态和控制设备的状态。
