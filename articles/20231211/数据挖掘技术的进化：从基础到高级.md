# 2.核心概念与联系

在数据挖掘中，有几个核心概念需要我们了解：

1. 数据：数据是数据挖掘的基础，是从各种数据源收集、存储和处理得到的信息。数据可以是结构化的（如表格、关系数据库等）或非结构化的（如文本、图像、音频、视频等）。

2. 特征：特征是数据中的一些特征值，用于描述数据实例。特征可以是数值型（如年龄、体重等）或分类型（如性别、职业等）。

3. 目标：目标是数据挖掘的目的，是要预测或分类的变量。目标可以是数值型（如预测销售额）或分类型（如分类用户行为）。

4. 算法：算法是数据挖掘的方法，是用于处理数据和发现模式的计算方法。算法可以是基于统计的（如聚类、主成分分析等）或基于机器学习的（如支持向量机、决策树等）。

5. 模型：模型是数据挖掘的结果，是用于描述数据中发现的模式和关系的数学模型。模型可以是线性模型（如多项式回归、逻辑回归等）或非线性模型（如神经网络、随机森林等）。

6. 评估：评估是数据挖掘的重要环节，是用于评估模型性能的方法。评估可以是基于准确率、召回率、F1分数等指标。

7. 可视化：可视化是数据挖掘的一个重要环节，是用于展示数据和模型的方法。可视化可以是基于图表、图像、地图等。

# 参数设置

# 聚类结果评估

plt.show()
