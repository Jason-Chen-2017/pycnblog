## 1.4 具体代码实例和详细解释说明
以下是一个简单的智能合约的实例代码：

```solidity
pragma solidity ^0.5.16;

contract SimpleContract {
    uint256 public totalSupply;

    constructor() public {
        totalSupply = 1000000;
    }

    function transfer(address payable recipient, uint256 amount) public payable {
        require(amount <= totalSupply);
        require(msg.value == amount);
        totalSupply -= amount;
        recipient.transfer(amount);
    }
}
```

上述代码实例中，我们定义了一个名为SimpleContract的智能合约。这个合约包括一个公共变量totalSupply，一个构造函数，一个transfer函数。

- 构造函数：构造函数是智能合约的初始化函数，它在合约的部署过程中被自动调用。在这个构造函数中，我们初始化totalSupply变量的值为1000000。

- transfer函数：transfer函数用于实现资产的转移操作。它接受两个参数：recipient（接收方地址）和amount（转移的资产数量）。在这个函数中，我们首先检查amount是否小于totalSupply，然后检查msg.value（调用者发送的资产数量）是否等于amount。如果满足这两个条件，我们则会将totalSupply减少amount的值，并将amount的资产转移到recipient的地址上。
