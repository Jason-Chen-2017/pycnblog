# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释规则引擎的使用方法。

## 4.1 代码实例

我们将通过一个简单的质量检测问题来演示规则引擎的使用方法。

```python
# 定义质量检测规则
rules = [
    {
        "condition": "产品的颜色不正确",
        "action": "标记产品为不合格"
    },
    {
        "condition": "产品的尺寸不正确",
        "action": "标记产品为不合格"
    },
    {
        "condition": "产品的质量不满足要求",
        "action": "标记产品为不合格"
    }
]

# 定义产品信息
product_info = {
    "颜色": "不正确",
    "尺寸": "不正确",
    "质量": "不满足要求"
}

# 执行质量检测规则
for rule in rules:
    if rule["condition"] in product_info.values():
        rule["action"]
```

## 4.2 详细解释说明

在上面的代码实例中，我们首先定义了一组质量检测规则，每个规则包括一个条件部分和一个操作部分。然后，我们定义了一个产品信息字典，包括产品的颜色、尺寸和质量等信息。

接下来，我们遍历所有的质量检测规则，并检查产品信息中是否满足每个规则的条件部分。如果满足条件部分，则执行对应的操作部分，将产品标记为不合格。
