### 4.2 垃圾回收实例

```go

func main() {
    var a = make([]int, 10)
    fmt.Println(a)
}
```

在上述代码中，我们使用make函数创建了一个int类型的切片a，并使用垃圾回收机制自动回收不再使用的内存，从而避免内存泄漏和内存溢出等安全问题。

### 4.3 并发简单实例

```go
package main

import "fmt"

func main() {
    var wg sync.WaitGroup
    wg.Add(1)
    go func() {
        defer wg.Done()
        fmt.Println("hello")
    }()
    wg.Wait()
}
```

在上述代码中，我们使用goroutine和channel实现了一个简单的并发操作，这种模型简单易用，有助于避免并发相关的安全问题。
