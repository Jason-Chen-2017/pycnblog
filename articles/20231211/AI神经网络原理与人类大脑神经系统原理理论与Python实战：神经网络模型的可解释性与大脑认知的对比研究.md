### 2.4.1 前向传播

前向传播是神经网络中的一种计算方法，用于计算神经网络的输出。具体步骤如下：

1. 对输入数据进行预处理，将其转换为标准化的形式。
2. 将预处理后的输入数据输入到神经网络的输入层。
3. 对输入层中的神经元进行激活，得到隐藏层的输入。

5. 对输出层的输出进行激活，得到最终的预测结果。

### 4.1 线性回归

线性回归是一种简单的神经网络模型，用于解决简单的分类问题。我们使用 Python 的 TensorFlow 库来实现线性回归模型。

```python
import numpy as np
import tensorflow as tf

# 生成数据
x = np.random.rand(100, 1)
y = 3 * x + np.random.rand(100, 1)

# 创建模型
model = tf.keras.Sequential([
    tf.keras.layers.Dense(1, input_shape=(1,))
])

# 编译模型
model.compile(optimizer='sgd', loss='mse')

# 训练模型
model.fit(x, y, epochs=1000)
```

在这个例子中，我们首先生成了一组随机数据，然后创建了一个线性回归模型。接着，我们使用随机梯度下降（SGD）优化器来编译模型，并使用随机数据进行训练。

### 4.2 多层感知器

多层感知器是一种复杂的神经网络模型，用于解决复杂的分类问题。我们使用 Python 的 TensorFlow 库来实现多层感知器模型。

```python
import numpy as np
import tensorflow as tf

# 生成数据
x = np.random.rand(100, 10)
y = np.dot(x, np.random.rand(10, 1)) + np.random.rand(100, 1)

# 创建模型
model = tf.keras.Sequential([
    tf.keras.layers.Dense(10, input_shape=(10,), activation='relu'),
    tf.keras.layers.Dense(1, activation='sigmoid')
])

# 编译模型
model.compile(optimizer='adam', loss='binary_crossentropy')

# 训练模型
model.fit(x, y, epochs=1000)
```

在这个例子中，我们首先生成了一组随机数据，然后创建了一个多层感知器模型。接着，我们使用 Adam 优化器来编译模型，并使用随机数据进行训练。
