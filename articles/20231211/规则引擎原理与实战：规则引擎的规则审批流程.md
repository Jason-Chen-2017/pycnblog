# 4.具体代码实例和详细解释说明

在这里，我们将提供一个具体的规则引擎实现示例，以及相应的代码解释。

```python
# 规则定义
rules = [
    {"condition": "age >= 18 and salary > 10000", "operation": "grant_loan"},
    {"condition": "age < 18", "operation": "reject_loan"},
]

# 上下文信息
context = {"age": 20, "salary": 12000}

# 规则引擎实现
def rule_engine(rules, context):
    for rule in rules:
        if eval(rule["condition"]):
            rule["operation"](context)
            return

# 执行规则
rule_engine(rules, context)
```

在这个示例中，我们定义了一个规则引擎，用于根据规则集合和上下文信息，自动执行相应的操作。规则定义包括一个条件部分和一个操作部分，条件部分用于判断是否满足规则的触发条件，操作部分用于执行规则的操作。我们使用Python的`eval`函数来判断条件部分是否满足，并执行相应的操作。
