# 4.具体代码实例和详细解释说明

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[0]
        less = [i for i in arr[1:] if i <= pivot]
        greater = [i for i in arr[1:] if i > pivot]
        return quick_sort(less) + [pivot] + quick_sort(greater)

arr = [3, 5, 2, 1, 4]
print(quick_sort(arr))
```

在这个代码实例中，我们实现了一个快速排序算法。快速排序算法的核心原理是：

1. 选择一个基准值（pivot）。
2. 将基准值左边的所有元素都比基准值小，右边的所有元素都比基准值大。
3. 对基准值左边和右边的子数组递归地进行快速排序。
