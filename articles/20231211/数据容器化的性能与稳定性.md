## 4.1 代码实例

我们将通过一个简单的数据处理任务来演示数据容器化的实现过程。假设我们需要对一个大数据集进行排序。我们可以使用以下代码实现数据容器化的过程：

```python
import docker

# 创建 Docker 客户端
client = docker.from_env()

# 创建数据容器
data_container = client.containers.create(
    image='ubuntu:latest',
    command='/bin/bash -c "yum install -y sort && sort input.txt > output.txt"'
)

# 启动数据容器
data_container.start()

# 等待数据容器完成排序任务
data_container.wait()

# 获取排序结果
output = data_container.get_archive(path='/output.txt')

# 保存排序结果
with open('output.txt', 'w') as f:
    f.write(output.decode('utf-8'))

# 删除数据容器
data_container.remove()
```

在上述代码中，我们首先创建了一个 Docker 客户端，然后创建了一个数据容器，并指定了排序任务的命令。接着，我们启动了数据容器，并等待其完成排序任务。最后，我们获取了排序结果，并将其保存到本地文件中。
