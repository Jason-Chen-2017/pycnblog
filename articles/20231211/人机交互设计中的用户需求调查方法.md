# 2.核心概念与联系

在人机交互设计中，用户需求调查是一个非常重要的环节。用户需求调查的目的是为了更好地了解用户的需求，从而为用户设计更好的产品。用户需求调查方法有很多种，包括问卷调查、面试、观察等。

问卷调查是一种常用的用户需求调查方法。问卷调查通常包括一系列问题，用户需要根据自己的需求和使用习惯来回答这些问题。问卷调查的优点是它可以快速收集大量的数据，但它的缺点是它可能会导致回答者的答案不够准确。

面试是另一种用户需求调查方法。面试通常是一对一的，设计师会与用户进行面对面的交流，了解用户的需求和使用习惯。面试的优点是它可以收集更多的详细信息，但它的缺点是它需要更多的时间和精力。

观察是一种更为直接的用户需求调查方法。观察通常是通过观察用户在使用产品时的行为和反应来了解用户的需求。观察的优点是它可以直接观察用户的需求，但它的缺点是它需要更多的时间和精力。
