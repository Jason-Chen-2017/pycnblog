## 1.4 具体代码实例和详细解释说明

### 1.4.1 规则编写

以下是一个简单的规则实例：

```python
if age < 18 and gender == 'male':
    action = 'grant_discount'
else:
    action = 'normal_price'
```

在这个规则中，我们首先判断年龄是否小于18岁，并且性别是否为男性。如果满足条件，则执行`grant_discount`动作，否则执行`normal_price`动作。

### 1.4.2 事实存储

我们可以使用字典或其他数据结构来存储事实。例如，我们可以使用以下代码来存储一个事实：

```python
customer = {
    'age': 20,
    'gender': 'male'
}
```

在这个事实中，我们存储了一个客户的年龄和性别。

### 1.4.3 规则执行

我们可以使用以下代码来执行规则：

```python
def execute_rule(customer):
    if customer['age'] < 18 and customer['gender'] == 'male':
        action = 'grant_discount'
    else:
        action = 'normal_price'
    return action

result = execute_rule(customer)
print(result)  # Output: normal_price
```

在这个代码中，我们首先定义了一个`execute_rule`函数，该函数接受一个客户字典作为参数。然后我们根据客户的年龄和性别执行规则，并返回决策结果。最后，我们调用`execute_rule`函数并打印决策结果。
