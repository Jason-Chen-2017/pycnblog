# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明如何设计易用的API。我们将使用Python和Flask框架来实现一个简单的RESTful API。

首先，我们需要安装Flask框架：

```
pip install flask
```

然后，我们可以创建一个名为`app.py`的文件，并在其中编写以下代码：

```python
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/api/users', methods=['GET'])
def get_users():
    users = [
        {'id': 1, 'name': 'John Doe'},
        {'id': 2, 'name': 'Jane Doe'},
        {'id': 3, 'name': 'Bob Smith'}
    ]
    return jsonify(users)

if __name__ == '__main__':
    app.run(debug=True)
```

在这个代码中，我们创建了一个Flask应用程序，并定义了一个名为`/api/users`的API端点，它使用GET方法来获取用户列表。我们还创建了一个名为`get_users`的函数，它返回一个JSON格式的用户列表。

我们可以通过以下命令来运行这个应用程序：

```
python app.py
```

然后，我们可以通过以下命令来测试这个API：

```
curl http://localhost:5000/api/users
```

这将返回一个JSON格式的用户列表。
