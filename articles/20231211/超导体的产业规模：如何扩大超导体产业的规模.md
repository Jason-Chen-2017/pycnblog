# 2. 核心概念与联系

在讨论超导体产业扩大之前，我们需要了解一些核心概念和联系。

## 超导体

超导体是指在低温下具有零电阻的金属。它们的电导率远高于普通金属，可以达到数十万至数百万次普通金属的水平。超导体的发现可以追溯到1911年，当时的荷兰物理学家Heike Kamerlingh Onnes在实验中发现了氢化钠的超导性。

## 超导体的类型

超导体可以分为两类：类型I超导体和类型II超导体。类型I超导体，如氢化钠，在较低的温度下才会出现超导性。而类型II超导体，如铬盐、钛盐等，则在较高的温度下也可以出现超导性。

## 超导体的应用

超导体在许多领域具有重要的应用价值。例如：

1. 量子计算：由于超导体的低电阻和高速电流，它可以用于量子计算的核心组件——量子比特（qubit）的制造。
2. 磁共振成像（MRI）：超导体可以用于生成强大的磁场，从而实现高分辨率的磁共振成像。
3. 超导磁共振：超导体可以用于生成高强度磁场，从而实现高效的磁共振检测。
