# 2.核心概念与联系

## 2.1网络攻击

网络攻击是指利用计算机网络进行的非法、恶意活动，以破坏、窃取或损害计算机系统和网络资源的行为。网络攻击可以分为多种类型，如：

- 恶意软件攻击：利用恶意软件（如病毒、恶意代码等）进行攻击。
- 拒绝服务攻击：通过发送大量请求或数据包，使目标服务器或网络资源不能正常工作。
- 数据窃取攻击：通过破坏网络安全措施，窃取敏感数据。
- 网络侵入攻击：利用网络安全漏洞进行入侵，获取目标系统的控制权。

## 2.2网络防御

网络防御是指采取措施保护计算机网络和网络资源的过程。网络防御可以分为多种类型，如：

- 防火墙：一种网络安全设备，用于对外网络流量进行过滤和控制。
- 安全软件：如防病毒软件、防火墙软件等，用于保护计算机系统和网络资源。
- 网络监控：通过监控网络流量和系统活动，发现和处理网络安全事件。
- 安全策略：制定和实施网络安全政策，确保网络资源的安全性和可靠性。
