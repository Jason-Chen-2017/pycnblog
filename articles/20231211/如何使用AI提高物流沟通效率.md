### 4.1 自动语义分析（ASA）

以下是一个自动语义分析的代码实例：

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.tag import pos_tag
from nltk.chunk import ne_chunk

def asa(text):
    # 文本预处理
    words = word_tokenize(text)
    # 词性标注
    tagged_words = pos_tag(words)
    # 命名实体识别
    named_entities = ne_chunk(tagged_words)
    # 语义角色标注
    semantic_roles = []
    for chunk in named_entities:
        if chunk.label() == 'NE':
            semantic_roles.append(chunk.text())
    return semantic_roles

text = "John Smith is the CEO of XYZ Corporation."
semantic_roles = asa(text)
print(semantic_roles)
```

这个代码实例使用 NLTK 库来实现自动语义分析。它首先将文本转换为计算机可以理解的格式，然后对文本进行词性标注和命名实体识别。最后，它对文本进行语义角色标注，并返回语义角色列表。

### 4.2 深度学习模型

以下是一个深度学习模型的代码实例：

```python
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Embedding, LSTM

def create_model(vocab_size, embedding_dim, rnn_units, max_length):
    model = Sequential()
    model.add(Embedding(vocab_size, embedding_dim, input_length=max_length))
    model.add(LSTM(rnn_units))
    model.add(Dense(1, activation='sigmoid'))
    model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])
    return model

vocab_size = 10000

max_length = 100

```

这个代码实例使用 TensorFlow 库来创建一个深度学习模型。它首先创建一个序列模型，然后添加嵌入层、LSTM层和密集层。最后，它编译模型并返回模型对象。

### 4.3 语义角色标注（SR）

以下是一个语义角色标注的代码实例：

```python
import spacy

def sr(text):
    nlp = spacy.load('en_core_web_sm')
    doc = nlp(text)
    semantic_roles = []
    for token in doc:
        if token.dep_ == 'nsubj':
            semantic_roles.append(token.text)
    return semantic_roles

text = "John Smith is the CEO of XYZ Corporation."
semantic_roles = sr(text)
print(semantic_roles)
```

这个代码实例使用 Spacy 库来实现语义角色标注。它首先加载英文语言模型，然后对文本进行分词和依存关系分析。最后，它遍历每个标记，如果标记的依存关系是 nsubj（主语），则将标记的文本添加到语义角色列表中。
