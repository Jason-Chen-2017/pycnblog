# 4.具体代码实例和详细解释说明
## 4.1流式计算的具体代码实例
```python
import sys
from pyflink.datastream import StreamExecutionEnvironment

# 创建流式计算环境
env = StreamExecutionEnvironment.get_execution_environment()

# 创建数据源
data_source = env.from_collection([(1, "Hello"), (2, "World"), (3, "Flink")])

# 创建数据流处理器
data_processor = data_source.map(lambda x: (x[1], len(x[0])))

# 创建数据接收器
data_sink = data_processor.add_sink(sys.stdout.buffer.write)

# 执行流式计算任务
env.execute("Streaming Word Count")
```
