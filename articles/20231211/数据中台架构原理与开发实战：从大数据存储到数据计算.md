                 

# 1.背景介绍

数据中台架构是一种新兴的数据处理架构，它将数据处理分为两个部分：数据存储和数据计算。数据中台架构的核心思想是将数据存储和数据计算分离，使得数据存储和数据计算可以独立进行，从而提高数据处理的效率和灵活性。

数据中台架构的出现是因为随着数据的增长和复杂性，传统的数据处理方法已经无法满足需求。传统的数据处理方法通常是将数据存储和数据计算集成在一起，这样的方法在数据量大和计算复杂的情况下，效率较低，并且难以扩展和维护。

数据中台架构解决了这个问题，它将数据存储和数据计算分离，使得数据存储和数据计算可以独立进行。这样的方法在数据量大和计算复杂的情况下，效率高，并且易于扩展和维护。
