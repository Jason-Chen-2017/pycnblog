## 4. 具体代码实例和详细解释说明

在这里，我们提供了一个简单的Python代码实例，用于实现推荐系统的协同过滤：

```python
import numpy as np

def compute_similarity(user_item_matrix, user_id, item_id):
    user_item_vector = user_item_matrix[user_id]
    item_item_vector = user_item_matrix[:, item_id]
    similarity = np.dot(user_item_vector, item_item_vector) / (np.linalg.norm(user_item_vector) * np.linalg.norm(item_item_vector))
    return similarity

def recommend(user_item_matrix, user_id, top_n):
    user_item_vector = user_item_matrix[user_id]
    similarities = []
    for item_id in range(user_item_matrix.shape[1]):
        similarity = compute_similarity(user_item_matrix, user_id, item_id)
        similarities.append((item_id, similarity))
    similarities.sort(key=lambda x: x[1], reverse=True)
    recommendations = [similarity[0] for similarity in similarities[:top_n]]
    return recommendations

user_item_matrix = np.array([
    [0, 1, 2, 3, 4],
    [1, 0, 4, 3, 2],
    [2, 4, 0, 1, 3],
    [3, 2, 1, 0, 4],
    [4, 3, 2, 4, 0]
])
user_id = 0
top_n = 3
recommendations = recommend(user_item_matrix, user_id, top_n)
print(recommendations)
```

这个代码实例首先计算用户和物品之间的相似度，然后根据相似度对物品进行排序，最后返回排名靠前的物品。
