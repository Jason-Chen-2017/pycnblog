# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个具体的监控模型的可解释性的代码实例来详细解释其工作原理。

```python
from sklearn.datasets import load_iris
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# 加载鸢尾花数据集
iris = load_iris()
X = iris.data
y = iris.target

# 创建随机森林模型
rf = RandomForestClassifier(n_estimators=100, random_state=42)

# 训练随机森林模型
rf.fit(X, y)

# 预测输出
y_pred = rf.predict(X)

# 计算准确率
acc = accuracy_score(y, y_pred)
print("准确率：", acc)
```

在这个代码实例中，我们首先加载了鸢尾花数据集，然后创建了一个随机森林模型，接着训练了模型，并预测了输出，最后计算了准确率。
