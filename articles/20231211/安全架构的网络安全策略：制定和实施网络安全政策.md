# 4.具体代码实例和详细解释说明

## 4.1 数据加密的具体代码实例

```python
from Crypto.Cipher import AES

def encrypt(plaintext, key):
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(plaintext)
    return cipher.nonce, ciphertext, tag

def decrypt(nonce, ciphertext, tag, key):
    cipher = AES.new(key, AES.MODE_EAX, nonce=nonce)
    plaintext = cipher.decrypt_and_verify(ciphertext, tag)
    return plaintext
```

## 4.2 身份验证的具体代码实例

```python
import hashlib

def hash_password(password):
    return hashlib.sha256(password.encode()).hexdigest()

def verify_password(password, hashed_password):
    return hashlib.sha256(password.encode()).hexdigest() == hashed_password
```

## 4.3 访问控制的具体代码实例

```python
def check_role_permission(user, role, permission):
    return user in role and permission in role.permissions

def check_attribute_permission(user, attribute, permission):
    return user.has_attribute(attribute) and permission in attribute.permissions
```
