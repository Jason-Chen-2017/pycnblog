class DistributedQueue:

### 4.2 分布式任务调度的代码实例

以下是一个分布式任务调度的代码实例：

```python
import threading
import queue

class DistributedTaskScheduler:
    def __init__(self):
        self.task_queue = queue.Queue()
        self.worker_threads = []

    def add_worker(self, worker):
        self.worker_threads.append(worker)

    def schedule_task(self, task):
        self.task_queue.put(task)

    def run_worker(self):
        while True:
            task = self.task_queue.get()
            worker = self.worker_threads[0]
            worker.work(task)
            self.task_queue.task_done()

    def wait(self):
        self.task_queue.join()
```

在上述代码中，我们定义了一个分布式任务调度的类，它使用了Python的`queue`模块来实现任务调度和分配。我们可以通过`add_worker`方法添加工作者线程，通过`schedule_task`方法将任务放入队列，通过`run_worker`方法启动工作者线程，通过`wait`方法等待所有任务完成。
