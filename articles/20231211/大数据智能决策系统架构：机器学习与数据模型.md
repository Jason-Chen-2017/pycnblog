# 6.附录常见问题与解答

常见问题：

- 如何选择合适的机器学习算法？
- 如何处理大数据中的缺失值？
- 如何处理大数据中的异常值？
- 如何处理大数据中的高维性？
- 如何处理大数据中的分类和回归问题？

解答：

- 选择合适的机器学习算法需要考虑问题的特点、数据的特点和算法的性能。可以通过对比不同算法的准确率、召回率、F1分数等指标来选择合适的算法。
- 处理缺失值可以使用填充、删除、插值等方法。填充是将缺失值填充为某个固定值，如均值、中位数等。删除是将包含缺失值的数据行或列删除。插值是根据相邻的数据行或列来估计缺失值。
- 处理异常值可以使用异常值检测、异常值删除、异常值修改等方法。异常值检测是通过统计方法来检测异常值。异常值删除是将包含异常值的数据行或列删除。异常值修改是将异常值修改为某个合理的值。
- 处理高维性可以使用降维、特征选择、特征提取等方法。降维是将高维数据映射到低维空间。特征选择是选择数据中最重要的特征。特征提取是将原始特征转换为新的特征。
- 处理分类和回归问题可以使用逻辑回归、支持向量机、决策树等算法。逻辑回归是对分类问题的一种线性模型。支持向量机是一种非线性模型。决策树是一种树形结构。
