## 4.1 数据的预处理和清洗
```python
import pandas as pd
import numpy as np

# 读取数据
data = pd.read_csv('data.csv')

# 数据的缺失值处理
data = data.fillna(data.mean())

# 数据的异常值处理
data = data[(np.abs(data - data.mean()) <= 3 * data.std())]

# 数据的标准化和规范化
data = (data - data.mean()) / data.std()
```
