# 2.核心概念与联系

## 2.1索引的概念和类型

索引是一种数据结构，它用于存储表中的一部分数据，以加速数据的查询和检索。在MySQL中，索引可以是主键索引、唯一索引、非唯一索引和全文索引等不同类型。主键索引是一种唯一的非空索引，用于标识表中的每一行数据。唯一索引是一种可选的唯一索引，用于确保表中的某一列具有唯一性。非唯一索引是一种可选的非唯一索引，用于加速表中的查询操作。全文索引是一种特殊类型的索引，用于加速对文本数据的查询和检索。

## 2.2索引优化器的概念和作用

索引优化器是MySQL的一个重要组成部分，它负责根据查询语句和索引信息，选择最佳的查询执行计划。索引优化器的作用是提高MySQL的查询性能，减少查询的执行时间。索引优化器通过选择最佳的查询执行计划，来确保查询的效率和准确性。
