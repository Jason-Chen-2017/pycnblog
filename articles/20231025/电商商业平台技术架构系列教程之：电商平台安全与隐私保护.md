# 1.背景介绍


随着互联网技术的不断发展和普及，越来越多的人选择在线购物，甚至把在线购物作为生活中的重要一部分。在线购物商业模式快速发展，给消费者提供了丰富的商品及服务。由于其在线交易行为十分复杂，安全、隐私保护等方面需要考虑的因素也越来越多。

电商商业平台是一个可以产生价值的载体，其整合了众多交易主体（用户、卖家、第三方支付、物流公司）的数据及信息，通过提供在线交易、物流配送、促销活动等服务，形成电商生态圈。电商平台安全与隐私保护如何保障消费者数据的安全和隐私权利是影响消费者信任和社会满意度的关键因素。

因此，了解电商商业平台的安全和隐私保护机制，有助于我们更好地保障我们的消费者数据安全、合法、有效、透明，保障商业利益，提升产品质量，让消费者享受到高品质的服务和体验。

# 2.核心概念与联系
## 2.1 基本概念
### 2.1.1 概念定义
安全性（Security）是指保障计算机或网络免受恶意攻击、泄露、篡改、伪造或破坏等危害，并确保数据被访问、修改、删除、抵赖、传输、存储时符合预期功能和目的，包括技术上的安全性、管理上的安全性和运行时的安全性等。

数据隐私（Privacy）是指个人生活中用于直接或间接识别特定自然人的数据。比如说，一张名片上没有真实姓名，但可以通过照片识别出身份；一段公开的文字消息可能不经过加密就暴露了个人隐私，即使对方知道该信息，也是无法追踪它的源头。

### 2.1.2 相关术语
#### 2.1.2.1 敏感信息
敏感信息是指机密、隐私权利要求很高的信息，如人员信息、信用卡信息、个人健康信息、财产信息等。它属于个人信息的一类，其内容或意义对他人可能会具有比较重大的影响。

#### 2.1.2.2 数据流动
数据流动是指个人信息从源头（比如用户输入的内容、系统自动生成的数据等）流向目标处，可以划分为两种方式，一种是单向流动（比如用户将个人信息上传至网站），另一种是双向流动（比如用户将个人信息共享给第三方，而第三方再将个人信息传回给原始的用户）。

#### 2.1.2.3 数据存储
数据存储是指个人信息在计算机中的实际保存形式。一般情况下，数据存储可以分为硬盘存储、数据库存储、云端存储三种。

#### 2.1.2.4 基础设施（Infrastructure）
基础设施是指计算机网络、服务器设备、存储设备、通信设备等提供支撑的技术基础设施。基础设施可以被视为电脑安全、数据安全的“鸿沟”，是所有电脑安全攻防的根本。

#### 2.1.2.5 策略与制定
策略是指保护个人信息的总体规范，由政府部门或组织制定的。制定策略既要考虑各个层面的安全要求，还要结合当今各种技术手段及技术发展方向，灵活应对新的攻击手段出现。

#### 2.1.2.6 法律法规与合规
法律法规与合规是保障个人信息安全的主要手段。法律法规通常包含有关保护个人信息的各项规定，例如《保守国家秘密法》、《公民个人信息保护法》、《个人信息保护条例》、《全国人力资源和社会保障厅关于个人信息安全的若干规定》等。合规是指对于电子政务、电子商务等各种运营业务，依据相关法律法规的规定及要求进行严格遵循。

#### 2.1.2.7 审计与评估
审计与评估是指检验、衡量和分析企业、政府、社会团体、金融机构等各种机构在保护个人信息方面的能力和水平的方法。审计与评估主要目的是为了确定法律法规、政策、制度以及企业内部控制措施之间的相互关系，找出存在漏洞、潜在风险、削弱或者超出标准、违反行业规则等情况，保障个人信息的安全。

#### 2.1.2.8 流程和工具
流程和工具是指保护个人信息的过程及相关的工具，可以包括设计、开发、测试、部署、运维等阶段。流程和工具的作用是确保保护个人信息的工作遵守相关的安全法律法规、制度及要求。流程和工具还可以帮助实施者及运营者在关键环节发现并消除个人信息泄露的隐患。
