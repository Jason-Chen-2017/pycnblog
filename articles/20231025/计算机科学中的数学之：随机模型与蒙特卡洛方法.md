# 1.背景介绍


在现代化的计算机应用领域中，不可避免地会涉及到大量的数值计算、模拟和建模问题。而作为一种数值计算工具，概率论和统计学是非常重要的知识体系，可以用来帮助分析和解决复杂的问题。本文首先简要介绍一下随机模型和蒙特卡洛方法。
## 1.1 随机模型
在概率论和数理统计学中，一个随机变量（random variable）通常是一个离散或连续的实测值，它取某一特定值的概率分布称为该随机变量的概率密度函数（probability density function）。例如，抛掷骰子一般是一个随机事件，抛出不同点数的可能性服从一定的分布，通常用一个骰子的六面体为例，每一面的出现的频率都是相等的。而抛掷两个骰子，一个的点数之和等于另一个的点数之差的概率分布就称为“两骰子的点数之差”。还有一些随机变量是非确定性的，即它们的值不是唯一确定的，比如每次抛掷一个硬币时正反面的概率都不一样，因此其概率分布也是随机的。

一个随机系统就是由若干个随机变量组成的一个整体。在一个随机系统中，任意一个独立的事件发生的可能性都可以用一组函数来表示，称为随机过程（stochastic process）。比如抛掷两枚均匀硬币，其结果的随机过程可以用一个连续函数f(x)来描述，其中x代表着抛掷的次数，f(x)等于1/2的概率对应着偶数次的成功，f(x)等于1/2的概率对应着奇数次的成功。当x=0时，f(x)=0；当x=infinity时，f(x)=1。

随机模型和随机过程给我们提供了一种新的思维方式——从实际世界中抽象出来的概念可以用随机模型进行刻画，并从中得出统计学上的结论。随机模型有着严谨、透彻的理论基础，因而能够对复杂的现象进行深入的分析和研究。

## 1.2 概率空间、样本空间与事件
概率论和数理统计学的一个基本概念是随机事件。设S为定义在一个集合X上的事件的集合，如果对于每个元素x∈X，有$P_S(x)$，那么称事件S具有概率质量。概率空间（Probability Space）由三个部分组成：S为事件的集合，X为定义在这个集合上随机事件的取值，P_S(x)为X的元素x所对应的事件发生的概率。

一个概率空间可以分解为三个部分，即样本空间、定义域、概率分布。一个随机过程（stochastic process）对应于一个概率空间，其中事件对应于样本空间中的元素，随机变量对应于定义域中的值。比如，抛两枚均匀硬币，样本空间为{HH, HT, TH, TT}，定义域为{0, 1}，概率分布为P({H}, {T})={1/2, 1/2}×{1/2, 1/2}。

定义域（Definitional Set）与样本空间相同，即事件的取值。可以证明一个概率空间只存在一个定义域与一个样本空间。定义域与样本空间的关系类似于函数的输入与输出之间的关系。定义域与样本空间之间的映射关系是由随机变量决定的。

事件（Event）是样本空间的子集，而且事件具有封闭性，即给定任意一个子集E，必然有一个包含E的最小事件G，使得EG∈S。封闭性保证了事件的稳定性，任何事件都会在某种意义下被归约为最小的那些事件。例如，假设事件A包括1至7的所有整数，那么B=A-3,C=A-4,D=A-2,E=A-6都属于事件BCE。

## 1.3 随机变量及其分布函数
随机变量（Random Variable）是指取值为某个范围内实数的随机变量的统称。对于离散型随机变量，其取值可以限定为一定范围内的某个整数或某几个整数。举例来说，一个事件的发生次数是一个离散型随机变量。对于连续型随机变量，则可以将其所有可能的实数值都纳入考虑。

随机变量的分布函数（distribution function），也叫概率密度函数（probability density function），定义为X的某个值x出现的概率。分布函数可以分为两种形式，即概率质量函数（pmf）和概率密度函数（pdf）。概率质量函数，也叫概率分布函数，定义为X的每个取值x都有相应的概率$p_X(x)$。概率密度函数，定义为概率质量函数除以区间的长度。

对于离散型随机变量，分布函数是可列可加的，可以用前缀和的形式表述。对于连续型随机变量，分布函数是单调递增的，且积分恒为1。然而由于连续型随机变量的概率分布难以用前缀和的形式进行求和，所以只能采用近似的采样方法求得概率密度函数。

## 1.4 期望与方差
期望（expectation），又称为均值，是随机变量的数学期望，表示随机变量取到某个值的概率与这个值本身相关性很强。它可以理解为求随机变量的平均值。

方差（variance），表示随机变量取到某个值的波动程度，即这个值与平均值的偏离程度。标准差（standard deviation）是方差的算术平方根。

期望与方差是很多统计学的方法的基础。

## 1.5 随机变量的函数依赖与独立性
若随机变量X和Y互为函数依赖，则称随机变量X依赖于随机变量Y。换句话说，事件X发生的条件下，随机变量Y的值是确定的。例如，两个随机变量X和Y，其中X依赖于Y，表示的是事件X发生的概率与随机变量Y的某个值有关。这种情况下，X和Y之间就不再是独立的。反之，若两个随机变量X和Y不互为函数依赖，则称它们是独立的。
