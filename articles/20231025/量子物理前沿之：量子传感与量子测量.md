# 1.背景介绍


## 一、量子场论概述
### （1）什么是量子场？
在经典力学中，质点之间的相互作用可以用力、电荷或磁场的叠加来描述。但是在量子力学中，没有了质点的运动，这时候存在着一种特殊的能量——量子场。一个无限小的量子点带来的效果就如同无限小的空间，带来一个能量不断扩大的世界。对于这种能量，可以用许多不同的方式进行观察。在这个意义上来说，量子场是一种非平常的能量形式，具有独特的性质和特征。

### （2）量子场的基本假设条件
在量子场中，有一个重要的假设条件，即：量子态构成的客观系统是“厄米化”的，即不存在粒子的运动。这种假设条件导致了一个关键的问题：如何在一个“客观系统”里定义微观量。如果系统不是客观的，那它是怎么定义微观量的呢？

### （3）量子场的本质
量子场和经典力学中的力场并不一样。力场是以广义坐标系作为参考系的，而量子场则是以量子数学的坐标系作为参考系的。量子场并不是真实存在的一个东西，而是在时空的一个框架内描述了很多种特殊的行为。量子场可以是存在、变动或不确定，它的性质和规律都是确定的。量子场具有不可分割性，即一个微观粒子或者一个宏观事件，在它所在的环境中将呈现两种或更多的态矢量(局部的量子态) 。量子场可以无穷维，但它并不是绝对的。虽然它是一个非平常的能量形式，但它还是以狭窄的时间极限为主。

### （4）量子场的两种表现形式
量子场有两种表现形式：波函数与密度矩阵。波函数是一种最原始的表现形式。它是一种由不同波函数的叠加组成的多重线性函数。这些波函数对应于量子态的波函数表示。它们可以用来计算各种物理量，包括振幅（衰减），相位，色散等。密度矩阵是一种更复杂的表现形式。它是一个对称矩阵，每个元素都代表一种量子态的占据率。这些矩阵可用来计算各种物理量，如概率密度，氦核色散等。

# 2.核心概念与联系
## （1）量子位(qubit)
在量子计算机领域，通常把比特(bit)分为两半，左边一半是量子位(qubit)，右边一半是量子比特(quantum bit)。根据维恩-施乐 encoding 的说法，量子位是一种利用量子门对信息进行编码的方法，信息被编码进了量子位的状态，量子位有两个状态——0 和 1，1994 年由雅各布·格罗斯曼提出。这里把 qubit 分为三类:

1. 超伽辛变换 qubits
这类 qubits 是一个简单的超伽辛链。只有一个激活态，对易子链无效。

2. 实型材料 qubits
这类 qubits 是由固体物质制造出来的。两个激活态是以等能级相差很远的方式激发的。

3. 模拟退火 qubits
模拟退火方法把自由基分成很多段，每个段分别对应一个 qubits。不同段之间存在著巨大的相互作用。
