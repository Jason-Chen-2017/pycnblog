# 1.背景介绍


## 1.1什么是离散数学
离散数学(discrete math)是研究数字、逻辑、计量等数值或符号集合的一些基本性质及其运用方法的一门学科。它由微积分和概率论发展而来，是工程应用和计算机科学领域的基础课程。它是一种数学理论，以陈述命题、证明定理、建立数学模型和分析数据为主要任务。离散数学发展于19世纪末20世纪初，主要用于电子工程、电气工程、控制系统、通信工程、自动化学科等领域。目前，离散数学已经成为工程技术、经济学、生物学、社会学、心理学等各个学科中的基础课程，是许多大学、科研机构和企业进行学术交流、合作开发的重要工具。因此，掌握了离散数学对高级工程师来说是一个必备技能。
## 1.2为什么要学习离散数学
离散数学的课程中，重点放在了两个方面：1）数学模型；2）算法原理。这两者可以帮助我们更好地理解复杂的问题和解决问题的方法，对职业发展都有着十分重要的意义。因此，掌握了这门课程对于应届生、技术工作者和高级工程师来说，都是非常有必要的。至于知识面是否广阔，主要取决于自己的兴趣爱好和专业能力。另外，我认为，能够从不同角度看待问题、运用各种数学模型和算法，并且把它们综合在一起应用到实际工作中，能够帮助我们在日常工作中更快、更准确地理解业务需求、解决问题，同时也能够对将来找工作有所帮助。
# 2.核心概念与联系
## 2.1概率论
概率论(probability theory)是一门研究随机事件发生的概率以及这些事件之间的相互关系的一门科学。概率论的基本假设是，一个试验过程（可能包含无限多个结果）是随机的，而其结果只能按照一定规律分布在实践空间的某些样本上。概率论以两个基本概念——事件与概率（event and probability）为中心。事件是客观存在的一些现象或者现象的发生，比如抛硬币正反面出现的事件。概率则是指事件发生的可能性大小，通常用0~1之间的实数表示。例如，抛一次硬币出现正面的概率为0.5，抛两次硬币连续出现正面的概率为0.5^2=0.25。
## 2.2集合与函数
集合(set)是指一组元素的集合，在集合论中，集合又称为类、族、群、集合体、代数系统等。函数(function)是指将一个集合映射到另一个集合的一个规则。函数由输入变量与输出值组成，并满足两个条件：
- 满足确定性：函数输出的值仅依赖于输入变量的值，不会受到其他条件影响；
- 一致性：同一个输入变量映射到相同的输出值，即x->y，则x'->y'，当且仅当x'->y时才成立。
## 2.3可列集与序列
可列集(enumerable set)是指一个有穷集合，其中每个元素都有着与其序号相对应的值。一个可列集上的函数f(n)，其中n是自然数，称为可列集f的第n项，记作f[n]。任何可列集都可以构造出一个序列，序列的每一项均是该可列集的一个元素。序列可以由递推公式定义或者根据初始条件迭代计算得出。
