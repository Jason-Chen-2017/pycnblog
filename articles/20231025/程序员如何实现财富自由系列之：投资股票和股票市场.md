# 1.背景介绍


## 历史回顾
从古至今，都在追求财富自由。在古代，商人为了谋取利益，宁愿贩卖印刷品；在近现代，工程师为了追求更好的生活品质，宁愿创造出好用的工具。在信息时代，科技巨头们希望借助互联网的力量，释放出大把的财富。股票市场最初也是这样，它就是一个大众市场，很多投资者在这里花费心血、体验情怀、结交朋友。当时的股市交易规则相对简单，主要是通过预先设定的固定赌注进行博弈，最后决定谁会赢得胜利，但如今股市的规则越来越复杂，面临着更多的风险和挑战。
## 股票市场简介
股票市场是目前全球最大的金融市场，其交易平台遍布全世界，有超过十亿只股票上榜。每天早上八点钟，全球股市的涨停板或跌停板就已敲响警报，大家都抓紧买入卖出的机会，以期获得高超额收益。股市的规则比一般市场要复杂得多，包括开盘集合竞价，连续两个交易日的涨跌幅限制，午后停止交易等等。所以，投资者不仅需要深刻理解股市的运作机制，还需善于应付各种风险，才能确保获得更优质的投资收益。
## 宏观经济分析及分析方法
由于股市具有极高的实时性，因此很难对其进行宏观经济指标的跟踪。但是，可以利用一些简单的指标（如PMI）作为股市走势的综合判断依据。其计算方法如下：
1. PMI指标的计算方式为股票价格对流通市值的比值乘以市盈率。其中，市盈率=最近四个月的营业收入/最近四个月的净利润。
2. 如果PMI指标大于800以上，则表示股市处于上升趋势；如果PMI指标小于200以下，则表示股市处于下降趋势。
3. 根据股票的热度和牛熊市的排名，有些股票可能会出现“快速反弹”，这种情况下，投资者应尽可能不要轻易买入，而应等待股票走势平缓。
股市投资的原则，不能以过高的价格涨幅买入，也不能低买入，否则容易亏钱。只能选择适中的价格，充分发挥自己的投资潜力。
## 投资股票的建议
1. 了解市场前景。了解公司的经营情况、产品形象、资产负债表、现金流水、财务报表、业绩报告、股东大会决议、董事会决议等。对股市发展趋势、公司业务状况有所把握是成功投资的关键。
2. 充分了解行情。重视市场公共利益，关注企业盈利状况、财务状况、经营效率、股东大会决策、董事会决策等变化。经常关注行情消息，找准当前热点事件及未来的新闻热点，定期研究相关报道。
3. 做好心理准备。认识到投资是一个长期过程，每天都有新鲜感、新的机会和挑战。面对挫折、失败和不确定性，保持冷静、理智和耐心，坚持理性投资。
4. 不断寻求反馈。在股市投资中，首先要培养自我反馈能力。投资需要以较低的交易成本取得最佳收益，但同时也要考虑到资金的消耗、时间的紧迫和风险的承受程度。要及时接受业内讯息，及时评估市场信息，并作出合理的投资判断。
5. 分散投资风险。避免高风险资产的过度集中，采取合理的仓位管理。通过成长型资产配置，平衡股票投资和债券投资。
6. 定期总结、复盘。不要忽略了市场的波动，对于重要的资产，每季度或每年回头查看一下资产的投资收益和市场的变化。与老手交流，分享自己的经验和感悉。
## 股票市场前瞻
随着中国经济的发展，2020年股市迎来了一次惊喜的开启。截止今天，A股市值突破1万亿元，成为中国股市第一大市场。同期美股市值同比增速放缓，但较上季度大幅减少，市值仍然维持在1.7万亿元左右。国际股市与中国股市比起来，表现逆势增长，美国加息周期将拉长，美国GDP增速将受到影响，国际股市有可能遭遇创纪录的暴跌。股市的底部被填补了吗？是否有符合个人意愿的股票可以购买呢？还是说还是要等到新的底部才会被填补？
