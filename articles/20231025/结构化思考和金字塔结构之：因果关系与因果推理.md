# 1.背景介绍


## 1.1 为什么要做这个研究？
自然界存在着复杂的相互作用机制，当我们面对复杂的问题时，总是很难一下子看清整体的本质。一个简单的例子就是生命，从诞生到死亡都离不开无数细胞协同工作。但是每个细胞又具有多个功能，比如合成氨基酸，维持形态，保持神经连接等。当某一功能失灵，整个细胞的运作就会受到影响，从而导致一系列连锁反应。所以在医学诊断、生物技术开发中，人们需要分析复杂的系统因果关系，提取有用的信息，寻找导致疾病或其它问题发生的根源。


那么如何通过计算机语言、数学模型和仿真模拟的方法来分析系统因果关系呢？我们可以用图形的方式直观地呈现系统的结构，然后借助工具求证那些影响因素及其作用机理。如此一来，就能找出哪些因素是关键的、应该重点关注的，并以此分析疾病、预测疾病发展趋势，甚至发现新的治疗方法。但如何有效地分析、理解和呈现系统因果关系是个重要问题。因为这样才能洞察出系统性的原因、发现底层规律、调整策略、优化过程等。正因如此，探索结构化思考与结构化分析在现代科技发展中的作用越发重要。


## 1.2 概念与联系
### 1.2.1 结构化思考
结构化思考(Structured Thinking)是指通过结构化的方式组织、整理、分析、解决问题，从而取得更高效、更准确的结果。它的基本思想包括：

1.分而治之——把复杂的问题拆分成简单易懂的小块，逐步解决小块中的问题；

2.问题空间——以一种全局视角，看到问题的所有可能的因果组合，从而全面考虑系统因果关系；

3.行动空间——掌握不同的分析手段，通过尝试各种方法、工具，找到最适合的分析方式，使得问题得到简化、解决方案得到明晰。


### 1.2.2 金字塔结构
金字塔（Pyramid）是英国心理学家约翰·梅纳德·凯恩斯于19世纪末提出的认知分析模型。他将人的大脑组织分为五层，每一层都有自己的特点。下层的主要任务是认知感觉、判断和记忆；上层的任务是抽象思考和计划；中间层的任务是情绪控制和调节行为；顶端的任务是集中注意力和解决实际问题。这种组织形式称之为“金字塔”结构。





从金字塔顶端开始往下分析问题，有如下三个步骤：

1.问题识别——首先需要识别出问题。通过观察、比较、分析等方式发现问题的共性，确认出现了什么问题；

2.假设检验——建立假设，测试假设，修改假设，重复以上步骤，最后确定假设的正确性；

3.数据收集——收集足够的数据，使用统计方法对数据进行分析，找出问题的关键因素。


## 1.3 研究领域
系统因果关系与因果推理(Causal Reasoning and Inference)，简称为CRI。作为结构化思考和分析的一项重要研究方向，CRI研究的是如何使用分析、计算、模拟、图表等工具来发现、分析和理解系统的因果关系，以及如何利用这些知识来预测系统的演变方向，从而对生活进行有益的改善。CRI研究涉及多个学科，包括复杂系统科学、数学、经济学、心理学、历史学、哲学、人工智能、工程学、管理学等领域。
