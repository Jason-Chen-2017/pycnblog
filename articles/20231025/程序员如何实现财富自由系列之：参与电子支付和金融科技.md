# 1.背景介绍



随着人们对金钱的依赖越来越强烈，越来越多的人选择成为“货币大老粗”，赚取更多的钱财。而对于许多普通人来说，仅靠现金流通或积蓄买不到真正想要的东西。于是，各种形式的数字货币开始崛起，它们可以让人们在网上或者APP上花费、购物、交易加密货币甚至进行借贷，从而逃离日益严重的物价不振现象。虽然这些加密货币有着诸多优点，但随着时间的推移，它的特点也逐渐暴露出来——它没有任何实体，只存在于网络上，用户需要记住一个密码或码才可拥有相应的数字货币。这样一来，普通人就完全失去了通过实体经济获得金钱的方式，或者说失去了“非物质文化遗产”——这类方式在法律上属于贪污罪。

这是一种非常危险的形势。要想摆脱这个困境，首先需要明白，加密货币并不是什么好东西。它只是解决了信息不对称问题、跨境支付的问题，却忽略了一个根本性的症结所在——其本质是信用体系的崩溃。当人的信用基础被破坏得如此之厉害时，整个经济就会出现动荡。当人们丧失了对市场的信心，甚至连基本的食品都无法购买时，社会将变得毫无生气。

为了解决这个问题，区块链技术已经得到了广泛的应用。这种技术的独特性在于，它建立了一套信任机制，保证每笔交易的信息都是公开透明的，并且所有参与者都可以验证交易的合法性。这样一来，无论发生何种情况，无论个人经历过多少风险，都不会影响交易的合法性，从而不再需要担心资金链断裂导致的经济危机。

本文将从以下几个方面讲述参与电子支付和金融科技的思路和方法。

1.1 信息不对称问题

随着全球金融危机的加剧，许多国家和地区的政府开始采取一些措施来保护私人投资者的权益。其中包括限制或禁止某些类型的资产交易活动，比如股票、期货等。当人们发现自己的资产受到了影响时，往往会急需另一种资产来进行还债。这就是为什么很多人宁愿购置黄金、白银、外汇，因为这些资产具有更高的吸引力。

然而，由于国内外资本市场不同，因此买卖不同类型资产的成本相差悬殊。很难衡量到底哪种类型资产会更容易带来收益，因此造成人们不得不寻找新的备选目标。这就促使大家开始关注数字货币。

数字货币最大的特点是，它的使用不需要第三方的介入，不受金融机构的控制，可以免于审查，且可以在全球任意地点使用。但是，由于数字货币的发行过程十分复杂，且无需第三方的资助，安全和隐私仍然是一个难题。

因此，很多人认为，使用数字货币来购置资产是可以解决信用问题的。然而，事实上，使用数字货币来购置资产的后果同样是信用问题。首先，数字货币的发行数量极其有限，只有少数的加密货币公司能发行足够的货币，且购买者也不能依赖单一的平台或服务商。其次，数字货币的价格波动十分频繁，购买者需要小心翼翼的观察价格走势，避免发生意外损失。最后，尽管数字货币可以让人们快速且廉价的进行交易，但同时也意味着用户的身份可能被泄露给第三方，从而导致损失。

在使用数字货币进行实际交易之前，我们应该充分考虑相关法律法规和监管政策，尤其是关于个人信息的保护，以及数字货币的使用是否符合个人的合法权益。

1.2 跨境支付问题

跨境支付是指不同国家、地区之间的支付行为。许多人认为，跨境支付是电子支付的前身，因为电子支付曾是解决不同国家之间货币及产品流通的问题。实际上，跨境支付是在电子支付发展的基础上发展起来的新领域。

目前，世界各国均采用或试图采用跨境支付。在支付方面，除了美国支付领先的状况外，亚洲、欧洲等其他国家也纷纷加入这一行列。中国是最早提出对外支付的国家之一，而香港、台湾则正在积极探索对外支付市场。

由于电子支付的普及率不高，人们对跨境支付的需求仍然十分旺盛。不过，像以太坊一样的数字货币在今年的央行周报中表示，计划今年全面启动一项“数字货币跨境支付”项目。具体进展将由央行与各个银行共同决定。

根据周报显示，该项目预计将支持11个主要央行在中国开展业务，包括中国银行、建设银行、工商银行、农业银行、交通银行、邮储银行、招商银行、浦发银行、民生银行、兴业银行、光大银行。该项目的具体金额仍未公布。

除了对数字货币跨境支付的发展，跨境支付还会涉及法律上的障碍。例如，当一个国家的法律禁止某个银行向另一个国家提供互联网支付服务时，该银行就可以声称自己提供的是“跨境支付”。然而，事实上，许多跨境支付项目并未违反任何法律，这就让其他国家或地区担忧是否存在侵犯国家利益的嫌疑。

1.3 “信用体系的崩溃”问题

信用体系是一个庞大的系统，由各种规则、制度、流程和工具组成。人们的信用体系有助于确保个人或组织的合法权益，并防止各种金融风险。然而，信用体系在过去几十年间经历了巨大的变化。现在，信用体系通常由多方面的因素组成，如法律、经济、政治、社会、道德标准、道德规范、习惯法等。

比如，在大萧条时期，由于美国的衰落，许多企业资产的价值受损，银行借款也减少，使得消费者对银行的信用度急剧下降。这又进一步刺激了信用卡业的发展，促使大量个人选择信用卡作为支付方式。但是，在接下来的岁月里，随着消费者对信用卡的认识加深、信用卡制度的改革，以及消费者对金融的不安全感逐渐增强，信用体系的运行出现了危机。

另一个问题就是随着互联网金融的发展，各种虚拟账户及其使用的兴起。随着技术的发展，这类账户成为一种非常便捷的支付方式，但同时也导致了账户的滥用。当用户在虚拟账户中存储钓鱼网站或恶意代码时，该账户很容易遭到窃取。

总之，要避免“信用体系的崩溃”，我们需要重新构建信用体系，提升人们对实体经济和金融机构的信心。只有建立起诚实、透明、可靠的信用体系，才能确保人的合法权益，并保障金融活动的顺畅、稳定、有效。
