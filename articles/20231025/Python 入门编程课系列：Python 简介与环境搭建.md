# 2.核心概念与联系
## 2.1 Python 是什么？
Python 是一种高级编程语言，由 Guido van Rossum 在 1989 年圣诞节期间为了打发无聊的时间创造出来的。它最初被称作 “胶水语言”，因为它可以把不同的编程语言很好地“粘”在一起，也就是可以让不同语言编写的模块导入到同一个程序中，而不需要改动程序的代码，只需简单地修改一下 import 语句就可以了。当然，也可以直接调用 C 语言、Fortran 语言或其他语言编写的动态链接库。

Python 的主要优点有以下几点：
- 易学：Python 具有简洁的语法，允许程序员用更少的代码就能完成复杂的任务。
- 可移植性：Python 程序可在各种平台上运行，包括 Windows、Unix、Mac OS X、Linux 等。
- 丰富的库和工具：Python 有大量的第三方库和工具，使得开发人员能够快速解决应用程序开发中的大量问题。
- 互联网语言特性：Python 支持交互式的命令行界面和脚本文件，还可以作为 web 服务器和 web 浏览器中的解释器执行。

## 2.2 Python 历史
Python 之父 Guido van Rossum 在 1991 年发布 Python 第一版代码时，标志着 Python 成为免费、自由、开源的开发语言。

1995 年 5 月，Python 推出了 2.0 版本，主要变化如下：
- 提供了更紧凑的语法，简化了代码编写流程；
- 为整数提供了真正的除法，而不是返回浮点数；
- 更严格的错误检查，确保代码在编译和运行时不会发生任何错误；
- 添加了线程支持；
- 新增了许多标准库模块，包括数组、日期时间、数学运算、网络通信、文本处理等；
- 引入了一个新的包管理工具 pip，方便安装第三方模块。

2000 年 10 月 17 日，Guido van Rossum 在一次 Python 用户大会上宣布退出 Python 项目，随即，Python 2.x 版本进入维护模式。

2008 年底，Python 社区出现了较大的变化。由于发布了一个支持 Web 编程的框架 Django，因此 Python 社区也对其中的一些模块产生了兴趣。为此，Guido van Roshan 加入了一个新的开发团队，并于 2009 年 2 月成立了 Python Software Foundation（PSF）。

2010 年，Django 框架获得了业界的关注，它是一个非常优秀的 Web 框架，已经成为 Python Web 编程领域的标杆。

2018 年，Python 迎来了 3.x 版本，加入了许多新的特性，比如支持 Unicode，增加了 asyncio 模块等。

## 2.3 Python 2 和 Python 3 有什么区别？
目前，最新稳定版本的 Python 是 Python 3，也就是说，Python 2 的生命周期已结束，所有的最新功能都会首先在 Python 3 上实现。

然而，如果想在两者之间做选择，还是要了解一下他们之间的差异。

Python 2 与 Python 3 在很多方面都保持兼容。比如：
- 可以互相调用；
- 字符编码不同，但默认都是 UTF-8；
- 函数定义时的 def 关键字相同；
- print() 函数相同；
- range() 函数相同；
- input() 函数相同；
- open() 函数相同；
- raw_input() 函数已弃用；
- str() 函数相同。

Python 3 对 Python 2 中的某些特性进行了改进：
- 增强型语法；
- 使用 unicode 编码字符串；
- 删除了 import site 命令，改为自动搜索路径；
- divmod() 函数返回一个 tuple；
- xrange() 函数替换了 range() 函数；
- 整数除法默认返回一个整数，不再返回一个 float；
- 文件读写默认打开为二进制文件。

因此，如果你有 Python 2 代码，建议升级到 Python 3 来享受新特性带来的便利！
