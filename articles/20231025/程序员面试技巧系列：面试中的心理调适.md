# 1.背景介绍


在面试中，人们往往倾向于用自己的聪明才智来评判对方的才能、能力。这种傲慢自大的心态会影响到面试者的职场生涯，甚至导致其在求职路上选择投降。所以，如何在面试过程中控制自己的情绪，不被工作压力和个人困扰打败，是非常重要的一环。此外，根据对面试者的性格特点分析，提出一些针对性的面试技巧，能够帮助候选人更好地调整心态、开拓视野、收获精神满足感，从而加速面试顺利通过。

本文将通过详细阐述和实践“面试中的心理调适”这一领域的理论和实践方法。首先，我将从技术层面对“心理调适”这一主题进行综合分析，包括什么是“心理状态”，“焦虑症”，“创造性思维”，“死亡模式”等。然后，我将介绍一些面试技巧，如不断反问问题、面对面交流、品味经历、体谅面试官等，这些技巧能够帮助面试者适应面试环境并更好地掌握面试技巧，最后，我将给出面试中的心理暗示，以此作为补充材料，帮助读者解决面试时遇到的挫折、困惑或问题。
# 2.核心概念与联系
## 2.1 什么是心理状态？
在“心理调适”领域，“心理状态”（mental state）指的是人的正常心智活动所处的阶段或者状况。例如，在学习、休息、恢复等非精神活动期间，人们通常处于一种比较低级的“安静”或“冷静”的“心理状态”。但是，当有人面临复杂的工作任务、突发事件、或者生命中的关键关头，他就会进入一种高度警觉和敏锐的“感知状态”，即兴奋状态、紧张状态、激动状态、愤怒状态等。在这些状态下，人们会不由自主地参与到各种活动和事件的过程当中，逐渐地对环境和外部世界产生更强烈的关注，使得他们的行为方式也发生了变化。当人们处理完当前的任务或突发事件之后，也会进入一种“平静”或“稳定”的“心理状态”，这时的表现可能是坐着沙发，看着电视、做一些简单轻松的事情，甚至会出现一些幻想、梦境之类的念头。
## 2.2 为什么要控制自己？
在“心理调适”领域，为什么要控制自己的心理状态呢？一般来说，人的潜意识会引起我们注意。人们常说，自己只是活在世间，没有精神生活。但如果我们连自己都不知道自己在干嘛，那又有什么用呢？

实际上，人在不同的心理状态下所表现出的不同的行为方式会影响到我们的心理健康，比如在不断学习和思考的过程中，我们可能会放弃一些本可以用来娱乐的时间，因此在这种情况下，我们需要调整自己的心理状态。比如，我曾经有一个朋友，有一次跟我吵架，对方竟然气愤地说“你以为我能从你嘴里吞下去吗？”，显然，这让我十分震惊。因为我并不是在针对他。另一个例子，就是我最近忙着申请教育背景，却在很长时间内没有看到她回复我的消息。也许我真的错过了很多机会吧。

不过，我们还是可以通过一些具体的策略来控制自己的心理状态。在下面的部分中，我将主要从三个方面介绍在面试中应该如何控制自己的心理状态：

1. 注意力管理：在面试中，需要注意保持专注力，同时可以适当伸缩自己的注意力。比如，有些人可能会因兴奋过度而忽略周围的情况，这就需要通过适当的运用注意力管理技巧来避免这种情况的发生。
2. 适时跳槽：由于我们都是人类，无论身处何种角色，都存在着切换角色的可能。在面试过程中，我们需要适时跳槽，以避免在新环境中留下不必要的负担。
3. 情绪管理：面试过程中，我们需要及时调整自己的情绪，减少在应试环境下的情绪波动。通过适当的沟通技巧、正确的表达、自我调节、积极的态度，以及适当的安慰，可以有效地控制自己的情绪。

## 2.3 焦虑症——工作压力带来的焦虑感
为了更准确地描述“心理状态”，本章将继续介绍一下焦虑症这个话题。在工作中，我们经常会遇到各种各样的压力，而这些压力都会使我们产生焦虑症。焦虑症是指工作压力、创业压力、社会压力等诸多原因导致的精神疲劳、抑郁、发作等各种状况，而这正是由于缺乏心理准备而引发的。因此，了解和了解自己在职场上的压力是第一步，它可以帮助我们更好地控制自己的情绪。

焦虑症有时候还可以分为“躁狂症”、“躁狂噩梦”和“焦虑晕眩”三种类型。躁狂症是指身体发出刺耳的呻吟、疼痛和胀闷，往往伴随着一阵狂风般的心绞痛，并伴有高烧、头昏眼花、失眠、乏力等症状。躁狂噩梦是指突然获得某种奇怪的、难以捉摸的、令人毛骨悚然的异象，痛苦万分，情绪失控，情绪崩溃，即使睡眠质量尚可，仍难以入睡。焦虑晕眩是指突然感觉不到任何声音、呼吸、任何刺激，精神清醒后，头脑中出现大段的空白，举止十分混乱，全身酸痛，紧绷、僵硬，甚至全身发颤，无法集中注意力。

## 2.4 创造性思维——面对新的挑战时的关键因素
创造性思维是指在面对新的挑战、解决难题时，我们所依赖的思维方式，即我们所使用的大脑活动模式。我们日复一日地重复老套、枯燥的解决办法，以致于忘记了自己的智慧和天赋，也让我们陷入思维误区。因此，通过创造性思维，我们可以独立思考、发现问题的真相，找到适合问题的解决方案。

在面试中，我们应该善于运用创造性思维，提升面试技巧，提高解决问题的能力。比如，我们可以模仿面试官的语言、表情和姿势，通过简单的语言技巧和口头禅，来克服坚持刻板印象的习惯，把自己的想法说出来，提高面试效果。另外，我们也可以尝试在听取面试官讲解问题的同时，自己用自己的语言思考，或借助工具软件来辅助思考，来增强自己的思维能力。

## 2.5 死亡模式——如何应对失败、迷茫、挫折？
既然我们已经深入了解了“心理状态”，那么接下来就需要更全面的认识“死亢模式”了。死亡模式其实并不仅仅是一个概念，而是对于面试过程中不同情况下的反应、行为和心态。

在面试中，面试官经常会出题要求候选人对某件事或某项工作负责。在这种情况下，面试官会希望候选人提前作出一些准备工作，并能够在面试中展现出他们的专业知识水平。然而，面试时长只有30分钟，而且候选人可能会在规定时限内不会完成所有的准备工作。这时，候选人就会陷入严重的自责和懊恼之中，失去对自己未来的信心，甚至以失败告终。

因此，在面试过程中，面试官需要适时和候选人交流，为候选人提供更多的信息，同时鼓励候选人不要轻易放弃，勇于挑战面试官设定的标准。在应试时代，我们需要认识到自己的困境，多多和面试官沟通，争取最终的胜利。
