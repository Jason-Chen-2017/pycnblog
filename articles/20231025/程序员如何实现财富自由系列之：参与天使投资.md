# 1.背景介绍


## 什么是天使投资？
天使投资是指那些获得少量资金、且具备独特才能的人物或组织寻求专业人才或金融产品作为个人或企业的资本来源。天使投资通常是风险投资的一个分支，并将其资金用于对特定项目的研究、产品的开发或服务的提供。天使投资已经成为美国最重要的风险投资项目，占到整个风险投资市场的近50%。
## 为什么要参加天使轮？
1.投资天使是一个成长性的过程，在投资之前要对自己充满信心，因为除了成功，没有任何事情能够一蹴而就。做好准备、拥有强大的自我驱动力，才能让自己取得成功。

2.对于资金方面的需求也很大，天使投资团队需要对自己非常了解，而且需要足够多的经验和技能才能快速地找到适合自己的项目。因此，投资天使是一个最佳的选择。

3.天使轮是获取高收益的绝佳方式。其主要目的是帮助创始人获得精英资源，通过风险投资的方式来带动初创公司的发展。在美国，天使投资已成为一个引领创业热潮的新兴风险投资模式。

4.天使轮具有很高的吸引力，每年吸引几百万元以上资金，而这些资金来自各行各业的优秀人才。此外，天使轮可以给予创始人的机遇和支持，也可为创始人找到精神上的平静和自由。

5.还有许多其他原因，但这些都是最重要的。由于天使投资创造了全新的商业模式和价值观，而且在美国风险投资界占据重要位置，因此它将成为中国乃至全球创业者不可或缺的一环。

# 2.核心概念与联系
## 股权激励机制
股权激励机制（Equity Incentive Schemes）是一种由政府或企业或其他机构建立的非营利性组织，旨在鼓励创业者成立自己的企业或参与到某个组织的管理下，以取得暴利收入。它可以鼓励企业获得股份或债权，并将创业者的个人收入和利润分配给创业者持有的股份或债权。股权激励机制的目标是在组织内创建有影响力的企业，并让创业者受到资本家重视。

## 投资顾问
投资顾问是指从事非营利组织工作人员，并负责与客户建立业务关系的个人。他们专门负责筛选公司或项目并为创业者提供建议和咨询。他们也会主动参与一些会议或活动，以鼓励创业者提升个人能力和技能。投资顾问可帮助创业者寻找投资机会、搭建团队、促进合作。同时，投资顾问还可以提供培训和资源支持，提升创业者的影响力。

## 白领经济学
白领经济学认为，白领阶层的劳动激励和生活水平较高，因此，他们往往更青睐接受责任感，追求职业发展。为了获得社会的认同、尊重和赞扬，白领们往往更容易投身于组织中，并与同龄人一样，享有资本和地位。因此，白领经济学将职业发展与个人目的结合起来，以期改善生活质量。

## 创业维艰现象
创业维艰现象是指资本和人才短缺导致创业者无法成功启动新的企业，因此，出现了所谓的“创业难”。这一现象曾经长期困扰着创业者，但随着时间的推移越发突出。由于创业的资金来源依赖于国际上各种名校的大量投资，以及大众的创业热潮，许多创业者因没有足够的资本支撑，而陷入困境。

## 前景
从目前来看，世界正在经历由创业维艰到创业盛极的转变。投资天使将成为创业成功的最佳策略。如果有意向参加投资天使，首先应该考虑清楚自己是否具备良好的商业思维和业务能力。只有具有丰富的经验、积累的能力、高度的责任感和敏锐的洞察力，才可能找到正确的方向。
