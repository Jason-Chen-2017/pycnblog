# 1.背景介绍


## 1.1 粒子物理学
粒子（particles） 是现实世界中存在的基本构件，是微观世界的最小可观察单位。它的物理性质不仅要考虑其本身的性质如质量、电荷、相互作用等，而且还要考虑周围环境的影响因素，如空间位置、运动规律、受力情况等。粒子物理学的研究就是对粒子的基本性质进行研究和分析，从而能够描述粒子在空间和时间中的分布、运动、行为以及各种物理现象的本质。
### 1.1.1 简史回顾
古代的粒子物理学是一门独立的学科，它是通过实验的方式对粒子的性质进行研究和阐述。如南京大屠杀中各国记者发现死亡的人中多半出现了高能粒子并非被清除殆尽就是其中一个原因。距今三百年的时间里，人们一直在探索着粒子物理学的奥秘。随着二十世纪末二十一世纪初以降的科技飞速发展，以及加快了粒子物理学研究的步伐，近几十年的研究成果有了极大的发展。
### 1.1.2 发展方向
粒子物理学的发展至今仍然处于一个相当的发展阶段，其主要研究方向包括：
#### 1. 波粒二象性
这是粒子物理学最基础的研究方向，也是目前最热门的方向。这个方向认为粒子具有相对论意义上的“小波”结构，它可以分解为运动在不同频率下的分量，这些分量可以被解释为对时空弯曲程度的修正。据此，可以推测出粒子的内部结构、场的形式和宇宙的物理机制。
#### 2. 流体物理
流体物理方面，由于粒子在流体中的运动特性，已有很好的理论支持。特别是Bose-Einstein统计力学模型及其相应的系统物理理论为研究流体性质提供了重要的工具。另外，通过对流体中的微观粒子的研究，也能够为理解存在于流体中的诸多奇异现象提供新的解释。
#### 3. 介质物理
介质物理的研究目标是在宏观上研究由多个粒子组成的微观介质的物理过程。例如，在海洋中存在着复杂的流态物理过程，它既受到海水潮汐的驱动，又受到底层地貌的影响。因此，了解这些复杂的流态过程的物理机理，对于探寻海洋、湖泊和平流层等自然界中深藏玄机的秘密都非常有价值。
#### 4. 气体物理
气体物理是研究微观气体的物理性质的一门学术领域。尤其关注的是多种气体的相互作用以及它们所形成的大气层的结构和运动。同时，还有望开发一种准确、精细的气体相关的计算机仿真模型，以用于模拟探测、预报和预警。
#### 5. 核物理
核物理研究宇宙星际背景下的质子、中子和引力子等微小粒子的行为。它着重分析了不同宏观尺度下微观粒子的行为模式，以及它们与宇宙的基本规律之间的关联。
#### 6. 超弦论与弱有效粒子
超弦论是研究微观粒子行为的新理论，它建立在弦论和分支ings模型基础之上，试图解决弦论模型不能完全描述微观粒子行为的问题。另外，弱有效粒子（weakly interacting particle）是指数量级小于零度能级的微观粒子，它只能通过相互作用才能生存，但却又不能被完全抵消掉。这些微观粒子往往带有非线性特征，难以用传统理论进行研究。
## 1.2 高能物理学
高能物理学是指探究能量比通常认识到的更强大或者更强大的实体、宇宙或者粒子。比如说，高能物理学研究超弦粒子、弱粒子、重子、夸克以及其他种类的强大粒子等。高能物理学的研究要涉及到对粒子产生强大的吸引力的各种机制的研究。目前，高能物理学领域依靠一些比较先进的方法，如数值计算、量子场论、相对论等对粒子的能量更高的部分进行探讨。当然，我们也要看到，目前的高能物理学研究还处于起步阶段，很多理论尚未完全成熟，需要更多的实践检验。
# 2.核心概念与联系
## 2.1 动量、能量与质量
粒子的动量与能量是粒子的性质。质量则是动量守恒的重要条件。假设有一个粒子，它具有质量m，那么它在三维空间中的运动学量，就称为它的动量。动量是一个矢量，它由三个分量组成：
$$p = (p_x, p_y, p_z)$$
其中$p_x$,$p_y$,$p_z$分别代表了动量的$x$,$y$,$z$分量。

同样，为了给动量加以定义，我们还需要知道能量。能量的数值表示了粒子或系统的能量大小。当一个粒子具有一定质量时，它所释放出的能量与该粒子的质量成正比。

$$E=\frac{p^2}{2m}+\varepsilon_{\text{int}}+\varepsilon_{\text{ext}}$$ 

式中$\varepsilon_{\text{int}}$为内能，$\varepsilon_{\text{ext}}$为外能。
## 2.2 流体、介质与气体
在现代物理学中，流体、介质和气体是三种重要的物质。按照它们的性质不同，又可以分为固体（solid），液体（fluid），气体（gas）。

**固体**是绝对不会流动的材料。包括石墨、砂土、石英等。其无色，透明，摩擦系数为零。

**液体**是可以流动的、带有溶解质的物质。常见的液体有水、冰、水蒸气等。

**气体**是可以流动、没有形状且体积极小的液体。气体由气团、氩核、氢核等成分组成，气体动力学中的很多方法也适用于气体。

虽然固体、液体和气体是三种不同的物质类型，但是它们之间还有一些共同的性质。比如，它们都是由微观粒子组成的。
## 2.3 动力学
动力学研究的是物体的物理性质，包括静止、运动、阻力与能量的关系。动力学的研究对象主要是刚体、流体、液体、固体和气体。

**刚体**是具有固定质心和任意方向的物体，如钟表、立方体、建筑物等。

**流体**是具有广义坐标系的，具有一定的流速和一定的表面积的物体。流体分为均匀流体、平滑流体和不均匀流体。

**液体**一般是无限长的流体。

**固体**具有较大的表面积和较坚硬的外表。

**气体**具有一定的流速，热容，温度低，易于储存，密度很高，非均匀流体。
