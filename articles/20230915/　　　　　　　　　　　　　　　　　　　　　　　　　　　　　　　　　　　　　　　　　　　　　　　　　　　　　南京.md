# 2.概念定义及术语说明
## 2.1 什么是机器学习？
机器学习（英语：Machine learning）是一门研究如何使计算机系统通过学习、改进和实时调整来提高性能的方法学科。机器学习以数据为驱动，通过数据挖掘、概率论、信息论等多种手段获取知识和理解能力，并运用数据模型进行预测、决策和控制。机器学习利用训练数据，通过计算来确定输出结果的过程。由此，机器能够自动地学习、制定策略和解决问题。它的目标是让计算机像人的大脑一样，根据数据做出自主决定，这样就可以很好地完成各种各样的工作。目前，机器学习已成为重要且广泛使用的科学研究领域之一，其应用遍及从人工智能到金融、医疗、生物等多个领域。


## 2.2 机器学习的一般流程

机器学习的一般流程如下图所示：


1. 数据集收集：首先需要收集大量的训练数据，用于训练机器学习模型。
2. 数据预处理：对原始数据进行数据清洗、归一化、拆分训练集和测试集。
3. 模型训练：通过对数据集进行训练，得到一个模型。
4. 模型评估：使用测试集进行模型评估，判断模型的好坏程度。
5. 模型预测：将模型应用于新的输入数据，得到预测结果。

## 2.3 机器学习的类型
机器学习可分为监督学习、非监督学习、半监督学习、强化学习五大类。下面依次介绍每一种类型的特点：

1. 监督学习：监督学习是在给定输入X和期望输出Y的情况下，通过学习获得一个最优的预测模型，即找到一组由输入到输出的映射关系，使得模型对于未知输入的数据预测效果最佳。监督学习的典型任务如分类、回归和预测等。
2. 非监督学习：非监督学习是机器学习的另一种形式，不需要标签信息。它会尝试发现数据的内在结构，同时也不会关注数据的输出结果。非监督学习可以用于数据聚类、关联规则、基于密度的聚类、数据降维等。
3. 半监督学习：在监督学习的训练过程中，只有少量的标记数据可用，如何利用更多的未标记数据进行训练呢？这是半监督学习要解决的问题。
4. 强化学习：强化学习是机器学习的一个前沿领域，它试图建立一个与环境互动的系统，通过反馈获取信息，利用这个信息来进行决策。强化学习有助于解决许多与智能决策相关的任务，如机器人控制、AlphaGo围棋等。
5. 其他类型：还有很多机器学习的类型，例如深度学习、集成学习、迁移学习、增量学习等。
