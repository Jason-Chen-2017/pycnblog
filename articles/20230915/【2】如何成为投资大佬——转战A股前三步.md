# 1.简介
  

A股是中国股市中的重要股票之一。作为最具代表性、交易量最大的股票，它发展历史从1990年建业高峰期到2018年美国次贷危机爆发后的经济衰退期间，多次创造了高质量发展的奇迹。在A股交易所，每天都有成千上万个交易者通过电脑完成各种交易行为。无论何时，都可以找到交易者的身影。因此，A股能够吸引大量外国投资者投资。

然而，作为业内极少数具备一定实力的个人投资者，想要从事A股投资并非一件容易的事情。因为缺乏专业知识、经验以及勤奋努力。只有走出门槛，才能真正成为一名成功的A股投资者。所以，在准备将自己打造成为一名合格的A股投资者之前，我们应该首先了解一些简单的投资知识。下面就让我们一起学习一下投资的基本知识吧！

# 2. 投资知识概述
## 2.1 A股市场概况
### （1）市场规模
截至2020年7月，A股市场总体规模达到了1192亿元，较2015年同期增长38%，截至2019年底，A股市场总市值已超越所有国家中央银行市值。据证监会数据显示，截至2019年底，中国证券投资基金业协会估计全球规模以上证券公司共发行注册普通股共计25.8万只，其中A股占比3.1%左右。此外，A股还存在着6.36万家私募基金投资管理公司，每年平均发行新股数约为7.4万枚，同样数目约占全球私募基金市场的近三分之一。由于国内对A股市场形态的不了解及许多交易者对私募投资的限制，导致目前很多投资者对A股市场的认识还有待加强。但是，随着时间的推移，A股的资本结构和行业分布也逐渐显现出来。如今，在中国A股市场，我们可以看到医药制造、食品饮料、家用电器等领域的龙头企业占据着重要地位。随着私募基金积极参与到A股市场中，更多的私募基金开始以一线企业或优秀项目的形式加入到A股市场中，发掘投资潜力。

### （2）市场结构
截止目前，A股市场主要由三大类（申万行业指数、科创板、主板）组成，其中科创板是A股市场的重点保护区。A股的主板是以沪深两市上市公司为基础的主营业务板块，包括制造业、电子、日用百货、消费品、纺织服装等各个产业领域。主板市场又细分为不同的子市场，如生活百货、服饰鞋包等子市场。申万行业指数也称为“行业分类指数”，按申万信息发布的行业分类标准，把A股上市公司按行业划分成若干个细分行业，并分别建立相应的行业指数。

## 2.2 投资品种类型
A股是目前国内较为活跃的股票市场。在这个市场里，你可以选择不同的投资品种。如：股票、债券、现金、基金、期货等。下面简单介绍一下不同类型的投资品种。

### （1）股票投资
股票投资就是买卖股份。股票的数量决定了你的财富，价格则影响着你的收益。购买股票需要考虑股票价值、风险、投资回报率、股息率等因素。不同的投资者面临不同的风险，一般有股票市场风险、经济周期风险、政策风险、交易撒钱风险、短期政策风险等。股票投资的风险控制能力取决于你能否发现并有效管理这些风险。

### （2）债券投资
债券投资是一种长期投资方式，债券有利于消除风险、降低波动率、提升收益率。比较常用的两种债券是国债和企业债。国债是一种长期的政府发放的货币担保券，具有很高的息费率，主要用于防范通胀风险。企业债是一种长期向企业支付的金钱担保券，具有很高的利息率，使得投资者有更多的利润空间。

### （3）现金投资
现金投资是指以现金作为资本来进行投资。这种投资方式有诸多特点，包括投资效率高、风险低、收益可观。但同时也存在着各种风险，如熊市时期的交易大幅滑坡、货币贬值导致资本缩水、偿债能力差等。

### （4）基金投资
基金投资是指以集资证券的形式，进行资本配置的一种投资策略。基金投资给予投资者一种广泛的经营机会，投资者可以获得比较好的收益，在风险与收益之间进行权衡，适合那些具有较强实力、高风险偏好、追求稳定收益的人群。

### （5）期货投资
期货投资是指以期货作为标的，进行交易的一种投资策略。期货是一种看涨看跌的商品，随着时间的推移，其价格变化会呈现出预测性。交易者通过结合市场情况，制定目标，选择合适的期货产品，进行交易，以实现期货投资的目的。

## 2.3 投资策略
### （1）价值投资策略
价值投资策略是指将对特定目标或主题的关注与股票或其他投资工具相匹配。以个人的价值目标为核心，寻找有潜力可挖掘的价值，投资有助于实现这些价值的实现。价值投资者往往具有有远见的思维、开阔的视野和透彻的判断力。通过了解自我的价值需求，并建立信心、储备资金，在市场中寻找符合自己的价值目标的资产。

### （2）配置资产投资策略
配置资产投资策略是指根据投资组合目标，按照权重对一组不同资产进行配置，以期实现最大化的投资收益。配置资产投资策略侧重于选择较高回报的资产配置方案，因而可以提供更好的收益。该策略要求投资者对每个资产进行评级，然后确定它们的比例，在满足投资目标的同时，实现更高的总收益。配置资产投资策略与价值投资策略互补，即优先选择具有较高回报的资产，再考虑适当的资产配比。

### （3）混合投资策略
混合投资策略是指将股票、债券、基金等不同资产的优势进行综合利用。这种方法的优势在于，既能实现对特定目标的投资，又具有较高的流动性和易于管理的特点。混合投资策略属于集中风险的投资方式，适用于财务状况不佳、经营风险较大的投资环境。

## 2.4 股票分析

对于A股来说，市盈率(PE)是一个非常重要的指标，它的计算方法是通过上市公司所拥有的股东分红收入与净利润的比值得到的。这个比值越高，则说明公司的盈利能力越强。而市净率(PB)则是通过上市公司市值与最近一次的年度报告披露的净利润的比值来衡量的。这个比值越高，则说明公司的长期经营能力越强。还有市销率(PS)、股息率(DY)等更具体的指标，但它们的计算过程比较复杂，建议大家可以参考相关书籍进一步了解。

# 3. 选股策略
## 3.1 概念
选股策略是在确定一组股票中，哪些投资者最有价值，以及投资组合的构成应如何？投资者可以在不同的层次上制定选股策略，例如行业、概念、风格、市场等。我们今天讨论的是选股的第一步——分层选股。

分层选股的基本思想是基于市场变化、投资人的诉求、投资人的信心、资金的承受能力，以及投资机会的剩余空间，将资金分配给价值最高的投资者。分层选股不需要任何复杂的计算模型，它只是对不同投资者的需求做出合理分配。

分层选股的结果可以帮助投资者更好的定位自己的投资方向，并筛选出最有可能提供长期回报的投资项目。在这个过程中，投资者可能会根据自身的专业能力、交易手段和预期收益来调整资金分配比例。分层选股可以帮助我们选出最有价值的股票，进而得到更丰厚的回报。

## 3.2 原则
分层选股的方法有两种原则。第一种是成长性原则，即认为在某些情况下，收益最高的投资者更有能力、更喜欢冒险的尝试，另一些情况下，资金最多、交易频繁的投资者更适合中长期投资；第二种是稳定性原则，即认为有些投资者的兴趣在某一阶段暂时受阻，另一些投资者则一直保持着持续的兴趣。这两个原则是基于对投资者的需求的考虑，也可以看作是长期的、宏观的视角。

## 3.3 方法
以下是分层选股的具体方法。

### （1）行业分类法
行业分类法是指对不同行业内的所有股票进行逐一分析，将它们归入到几个主要的行业中，比如制造业、消费品、金融业、医疗保健、电子、计算机、运输设备、商业服务等。这样可以使投资者根据自己的需求，更快、更直观地了解各行业的最新信息。行业分类法的优点是易于理解，并且可以给投资者提供一个快速了解各个领域的途径。但是，这种方法不能完整反映各个行业之间的联系。

### （2）概念分类法
概念分类法是指将股票按照它们的共同特点进行分类，比如消费电子、医疗保健、通信、能源、材料、信息技术等。这种方法可以在分类时比较全面，并且可以反映不同行业之间的内部联系。但是，概念分类法需要花费大量的时间和精力，并且难以持续跟踪新出现的股票。

### （3）风格分类法
风格分类法是指对具有相同历史、发展轨迹和企业模式的企业股票进行分组，并命名它们，比如互联网金融、科技创新、传统金融、服务业等。这种方法可以帮助投资者更准确地判断公司的实力、运营策略以及社会责任。但是，风格分类法的应用范围局限于大众消费领域，无法提供涵盖整个行业的分析视角。

### （4）市场分析法
市场分析法是指根据交易者对个别股票的看法，对整个市场进行整体分析。通常采用短期交易和分析的方式，对同一市场上的同类产品进行比较，从而找出当前价格波动大的那些股票。市场分析法的优点是直观，而且可以快速取得比较准确的分析结果。但是，市场分析法需要投资者对整个市场的变化情况有清晰的认识，并充分考虑个体的感受。

# 4. 仓位管理
仓位管理，顾名思义，是指投资者控制自己的仓位，以获取最佳的收益。仓位管理的首要原则是保证资金的安全和最大化收益。仓位管理的方法有四种。

### （1）资金配比法
资金配比法指的是通过设置不同比例的资金，来控制各个投资者的投资目标。比如，可以给每个投资者设定固定的金额，或者通过计算各投资者的权益比例，将资金分配到不同的投资账户。这种方法可以让投资者合理地管理资金，并且可以帮助投资者减小仓位上的波动。

### （2）固定比例仓位法
固定比例仓位法指的是在一定的时间内，投资者所持有的股票仓位始终保持不变。这种方法使得投资者更加谨慎，避免对超额风险过度关注。固定比例仓位法的优点是简单易行，适合于中短期交易。

### （3）动态比例仓位法
动态比例仓位法指的是根据市场的行情，动态调整各个投资者的仓位比例。这种方法使得投资者根据自己的投资目标和风险偏好，调整自己的仓位比例，从而获得最佳的收益。动态比例仓位法可以让投资者在取得短期利润的同时，保持高收益。

### （4）风险平价法
风险平价法指的是通过对风险和收益进行逐步平衡，来获取最佳的回报。风险平价法的原理是根据投资者对不同投资品种风险的承受能力，调整各个投资者的资金比例，以实现最大化的收益。这种方法适合于长期持有或降低波动率。

# 5. 投资建议
如果你要开始投资A股，那么以下几条建议可以帮助你更好地投资。
- 明确你的目标。不要试图一次性搞懂一个股票的价值和技术含量。设立目标后，你就可以设定比较明确的投资计划。
- 考虑资金限制。每个投资者都有自己对资金使用及其限制的要求。定下目标后，应该根据自己的实际情况做出相应的调整，以维持现有的投资能力。
- 不要错失良机。投资市场每天都有大大小小的交易事件发生，但很多投资者没有意识到这些信息。要做好耐心等待和准确分析市场信息的准备。
