# 1.简介
  

## 大数据技术背景
随着互联网、移动互联网、物联网、云计算、大数据应用的火热，人们越来越关注如何从海量数据中找到有价值的信息，如何对大数据进行有效的分析处理，如何保障数据的安全和隐私，如何在线生成报表和精准推送相关消息，如何通过大数据分析实现商业决策优化等诸多需求。而目前，业界正在探索如何更好地实现大数据技术的落地。在这一过程中，Apache Hadoop、Spark、Storm、Flink等大数据框架得到了广泛应用，并逐渐成熟完善。在这些框架之上，基于Hadoop生态圈，如Hive、Pig、Sqoop、Flume等组件，更加容易构建大规模数据仓库，并支持离线和实时计算。同时，云计算平台也逐渐成为大数据领域的一个重要分支，包括Amazon AWS、Microsoft Azure、阿里云等。基于分布式存储、集群调度、资源管理等机制，云计算可以帮助企业降低成本、提高效率。此外，机器学习、图像处理、自然语言处理等新兴技术也扮演着越来越重要的角色。综合来看，大数据技术带来的改变不仅局限于技术本身，还伴随着人文社会的重大变革，其中最突出的变化莫过于个人数据的快速积累和丰富。
## DolphinInsight项目简介
DolphinInsight是基于Hadoop生态圈和云计算平台，面向金融行业的大数据可视化分析工具，提供了多维分析和结构化查询能力，支持海量数据和复杂查询场景下的高性能查询。其架构设计的宗旨就是为了满足金融行业各类需要快速准确地获取、处理、分析和反应金融数据的问题。DolphinInsight具备以下功能模块：
* SQL监控：采用SQL语句作为用户交互接口，提供一个统一的数据源层，将各种异构数据源抽象为同一视图，并提供自动提示、语法检查和执行计划提示功能；
* 元数据管理：提供元数据管理功能，包括数据模型定义、数据源配置、数据权限管理等；
* 数据质量：通过数据质量检测，能够发现数据异常和错误，及时掌握数据质量状况；
* 任务管理：支持运行SQL语句、同步数据到另一个数据源、数据导入导出、图数据库操作、模型训练等任务；
* 数据开发平台：提供一个基于web的交互式数据开发环境，支持多种数据开发语言(如Python、R、Scala)、图形化组件、数据集市等扩展能力；
* 智能监控：通过日志、指标和规则引擎，可以实时监控数据集市、应用系统、数据质量、作业运行等状态信息；
* 图数据库：DolphinInsight支持两种类型的图数据库，一种是基于内存的嵌入式Neo4j图数据库，另一种是分布式的基于图BLAS的图数据库DrillGraph。支持多种图算法，如推荐系统、图聚类、图搜索等；
* 数据导入导出：支持数据源之间的数据导入和导出，包括CSV文件、Excel文件、HDFS、MySQL数据库等；
* 数据共享：支持将数据集发布到数据集市或其他用户，也可以将数据集分享给他人；
* 企业级认证：DolphinInsight项目由金融云计算平台提供支持，拥有高度集成的大数据生态体系，并结合自身金融行业的实际应用经验，对生产环境中的系统进行高度保障。
