# 2.Python数据分析相关概念与术语
## 2.1 数据类型与数据结构
数据类型指的是在计算机中存储、组织、处理的数据的类型和形式。一般来说，数据类型分为两类：标量（Scalar）数据类型和容器（Container）数据类型。

### 标量数据类型
- 整数型int：用于存储整数值，如 1、2、3、4、5
- 浮点型float：用于存储小数或类似于小数的值，如 3.14、2.5、0.1、7.8e-5
- 布尔型bool：用于存储True或者False值
- 字符型str：用于存储文本信息，如 'Hello'、"World!"

### 容器数据类型
- 列表list：用于存储同种类型的数据，如 [1, 2, 3, 4, 5]、['a', 'b', 'c']
- 元组tuple：与列表类似，但元素不可修改，如 (1, 2, 3, 4, 5)、('a', 'b', 'c')
- 字典dict：用于存储键值对形式的数据，如 {'name': 'John', 'age': 30}
- 集合set：用于存储无序且唯一的元素，如 {1, 2, 3, 4, 5}、{'a', 'b', 'c'}

## 2.2 数据采集
数据采集是指从原始数据源（如数据库、文件、API接口等）提取需要的信息，包括获取、存储和转换数据等过程。在Python中，可以使用不同的模块完成数据采集任务，如pandas、SQLAlchemy等库。

## 2.3 数据清洗
数据清洗即指对已收集到的数据进行质量和完整性检查，去除噪声数据、缺失值、异常值，保证数据的正确性。数据清洗是数据预处理的重要环节，能够有效地过滤掉不完整、不符合规范的数据，使得后续分析更加准确。在Python中，可以使用 pandas 提供的DataFrames、Series等数据结构来实现数据清洗。

## 2.4 数据探索性分析
数据探索性分析是指通过各种统计方法对数据进行初步分析，以了解其特征、规律和结构，从而找出其中的模式、趋势、关联关系，从而更好地理解数据背后的含义。Python提供了一些可视化工具，如Matplotlib、Seaborn、Plotly等库，可快速绘制各种图表进行分析。

## 2.5 数据可视化
数据可视化是指将数据通过图像、图表的方式呈现出来，帮助人们更直观地发现、理解和分析数据。Python提供了一些可视化工具，如 Matplotlib、Seaborn、Plotly等库，可快速绘制各种图表进行可视化。

# 3.Python数据分析工具包
## Pandas
Pandas是一个开源的Python库，用于数据分析、数据处理和数据可视化。Pandas基于Numpy、Scipy、Matplotlib构建，提供了高效、快捷的dataframe数据结构，并提供了数据读取和存储、数据清洗、数据合并、数据选择、数据计算、数据透视表、数据聚合等功能，能轻松解决日常数据处理需求。

安装：
```
pip install pandas
```

## Scikit-learn
Scikit-learn是一个开源的Python库，用于机器学习、数据挖掘、数据预处理等任务。它提供了非常丰富的机器学习模型，能帮助我们快速搭建、训练模型，并且提供高效的预测性能。Scikit-learn基于Numpy构建，提供了丰富的数据处理和特征抽取的方法，并内置了常用评估指标。

安装：
```
pip install scikit-learn
```

## Statsmodels
Statsmodels是一个开源的Python库，用于统计模型构建、数据分析、假设检验等任务。它提供了很多统计模型，包括线性回归、logistic回归、时间序列分析、广义线性模型、混合分布模型等，并内置了常用的假设检验方法。

安装：
```
pip install statsmodels
```

## Matplotlib
Matplotlib是一个著名的Python可视化库，用于创建静态、交互式图形。Matplotlib基于NumPy构建，提供了常用的绘图函数，如折线图、散点图、直方图、三维曲面等，可方便地自定义样式和格式。

安装：
```
pip install matplotlib
```
