## 2.基本概念、术语说明
### 1.编程语言
计算机编程语言（Programming Language）是一种用来告诉电脑如何做某件事情的指令集合，不同编程语言之间存在着巨大的差异性。比如，汇编语言（Assembly language）就是把机器指令直接翻译成二进制代码，C语言（C Language）就是一种由Bjarne Stroustrup发明的通用编程语言。在计算机领域里，目前最流行的语言是基于C语言、Java、Python、JavaScript等的各种变体，还有一些其他的编程语言如Prolog、Fortran、PHP等。 

编程语言主要分为以下三类：

1. 底层语言：比如汇编语言、机器码语言
2. 中间语言：比如C语言、Java语言、Python语言
3. 高阶语言：比如SQL语言、HTML语言

### 2.变量类型
变量是存储在计算机内存中临时存放数据的地方，变量类型是指变量存储的数据类型。在编程过程中，你需要定义并赋予变量一个值。每个变量都有一个数据类型，它决定了变量可以存储什么样的数据，以及可以对这些数据执行哪些操作。常见的变量类型包括整型、浮点型、字符型、布尔型、数组、指针等。 

#### 整数型(integer)
整数型变量用于存储整数值，例如1、2、3、4、5等。整数型变量的范围比其它变量类型要大得多，可表示的数字也更多。例如，int数据类型可以表示的数值范围在-2147483648到+2147483647。 

#### 浮点型(float)
浮点型变量用于存储小数值，例如3.14、2.718、6.02E23等。浮点型变量的范围比整数型小得多，但它的精确度比较高。 

#### 字符型(char)
字符型变量用于存储单个字符，例如'a'、'b'、'c'等。字符型变量只能存储一个字符，且不能运算。 

#### 布尔型(boolean)
布尔型变量用于存储true/false两种值，表示逻辑判断结果。 

#### 数组
数组变量可以存储多个相同数据类型的元素，数组变量的声明形式如下： 

```
dataType arrayName[arraySize];
```

其中，`dataType` 是数组元素的类型，`arrayName` 是数组的名字，`arraySize` 是数组的大小。数组的索引从0开始，最大可以到数组大小减1。 

例如，定义一个整数型的数组`myArray`，数组大小为5，则可以使用如下语句进行初始化：

```
int myArray[5] = {1, 2, 3, 4, 5};
```

这样，数组`myArray`就被初始化为{1, 2, 3, 4, 5}。 

#### 指针
指针变量是一个变量，该变量存储的是另一个变量的地址。你可以通过指针访问它指向的变量的值和地址。指针变量的声明形式如下：

```
dataType *pointerName;
```

例如，定义一个整型指针变量`p`, 可以如下写法：

```
int a = 10; // 定义变量a
int* p = &a; // 将变量a的地址赋值给指针p
```

`&` 操作符是取址运算符，它返回一个变量的地址。通过指针`p`，我们可以修改变量`a`的值：

```
*p = 20; 
```

这样，变量`a`的值变为20。 

### 3.常量与表达式
常量（constant）是在程序运行期间保持不变的值，如π、圆周率、日期、年龄等。常量在程序中通常用全部大写的字母表示，如PI=3.1415926...。常量可以直接放在表达式中使用，也可以由变量计算得到。 

表达式（expression）是由运算符和操作数构成的一系列值的运算过程。表达式的值一般依赖于操作数的值，并且根据运算符的不同，可以产生不同的结果。常见的表达式示例如下： 

1. 数学表达式：3+4/2 
2. 关系表达式：x>y 或 x<=z 
3. 逻辑表达式：a && b ||!c
