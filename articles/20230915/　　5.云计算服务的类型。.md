# 1.简介
  

云计算是一种利用网络提供的端到端的服务。云计算可以让用户轻松地获得、扩充及共享计算机硬件资源、存储空间、应用软件等IT资产。云计算平台提供的服务包括计算、存储、数据库、网络等基础设施能力，还有丰富的基于这些能力构建的业务场景所需的各种软件服务和解决方案。
云计算服务的种类多种多样，比如计算、存储、网络、数据分析等等。本文将从如下方面对云计算服务进行分类，帮助读者更加直观地了解云计算服务的种类：

① IaaS：基础设施即服务（Infrastructure as a Service，IaaS）服务商提供虚拟化的计算、网络和存储基础设施。这些服务是按需付费的，用户可以通过接口调用的方式灵活部署和管理这些资源。例如亚马逊AWS EC2或微软Azure Virtual Machines都是IaaS的代表。

② PaaS：平台即服务（Platform as a Service，PaaS）服务商提供软件开发框架和运行环境。用户只需要部署自己的应用代码即可，不需要关心底层的计算、网络和存储资源的配置和管理。PaaS的代表包括Heroku、Cloud Foundry、Microsoft Azure Web Apps等。

③ SaaS：软件即服务（Software as a Service，SaaS）服务商提供可直接使用的应用程序。用户无需安装、升级或者维护应用，只要订阅服务并登录到相应的客户端软件，就可以立刻使用应用功能。SaaS的代表有SalesForce、Zoho、Adobe、Dropbox等。

除此之外，云计算还支持混合云架构，即把部分计算资源、网络和存储资源放在本地，而部分计算资源、网络和存储资源则被托管在云上。这种架构可以有效降低本地设备投入成本、提升资源利用率和节省运营成本。目前，国内外很多大型互联网公司都采用了混合云架构。云计算已经成为行业标配。

# 2.核心概念
## 2.1 云计算概念
云计算概念是云计算服务的基础，它定义了云计算的核心属性、特征及其实现方式。云计算是一种通过网络传输的数据处理服务，它主要由两个关键词组成——“网络”和“计算”。通过云计算服务，用户可以灵活地弹性伸缩计算资源，不受硬件性能限制。由于云计算服务随时提供及时可靠的资源，因此适用于各种规模的企业，尤其是那些想自行扩张的人工智能、大数据、移动互联网、物联网、云游戏等领域。

### 2.1.1 网络
云计算的一个重要特点就是网络带宽的无限扩充。网络连接了所有节点，每台计算机都有专门的网络连接，它们之间可以快速、高速地传递数据。云计算服务的网络由一系列连接在一起的网络设备组成，这些设备通过广域网互连，以尽可能低廉的价格提供网络带宽。例如，AWS提供了全球分布式的数据中心网络，每年可连接数百万台服务器，平均每个服务器链接数量约20个。这些网络设备大多具有安全防护、访问控制、故障诊断、流量调控、QoS等功能，能够使得网络稳定可靠、负载均衡、容错、高可用性。

### 2.1.2 计算
云计算计算资源通过虚拟化技术实现。这意味着用户实际上拥有的计算资源是虚拟机，而不是真实的服务器。实际上，云计算的计算节点大多通过虚拟机技术模拟出来的，称为“虚拟机”。虚拟机提供统一的操作系统，并在其中运行用户指定的应用程序。这样一来，用户就无需操心底层的服务器配置、维护、扩展等问题，只需要花费精力在应用开发上。云计算的计算节点通常配置较好的CPU、内存、存储等硬件资源，并配备相应的操作系统。用户可以在应用启动、运行过程中随时调整计算资源，甚至将其完全卸载。

### 2.1.3 存储
云计算的另一个重要特点就是存储的无限扩充。云计算存储资源的实现方式与网络类似，采用网络存储协议（NFS、CIFS、iSCSI等），用户可以快速、高速地读写数据。云计算中提供的存储资源既可以是永久存储，也可以是临时存储。存储资源主要分为两类：块存储和文件存储。块存储可以划分为高效的磁盘阵列SAN (Storage Area Network)或高速SSD (Solid-State Drive)，并通过网络传输；文件存储可以是对象存储、对象镜像等，也存在于网络中。由于云计算提供的存储资源不受服务器性能的限制，因此用户可以随时增减存储容量。

## 2.2 云计算服务类型
云计算服务的类型可以分为以下三种：

1.基础设施即服务(IaaS)：基础设施即服务（Infrastructure as a Service，IaaS）服务是云计算最基础的服务形式，它允许用户购买虚拟机、存储、网络等资源，并通过虚拟化技术来使用。云厂商为用户提供基础设施的虚拟化环境，如：虚拟机，硬盘，网络等，同时提供相关的服务，如：存储，DNS，安全性等。IaaS服务模型侧重于提供基础设施的租用，用户可以根据需要创建、删除、开启、关闭服务器，配置IP地址、路由器等。用户可以灵活选择不同的OS、程序语言以及服务软件，随时修改配置。目前，AWS、Azure、Google Cloud Platform、OpenStack、VMware vSphere等都属于IaaS的提供商。

2.平台即服务(PaaS)：平台即服务（Platform as a Service，PaaS）服务是云计算的一种新兴服务形式。它为用户提供了完整的软件开发环境，用户可以使用编程语言、工具集、中间件以及现成的数据库等技术，快速搭建应用。PaaS服务模型侧重于软件开发环境的建立，用户可以快速、方便地发布软件，而不需要关心底层基础设施的配置。PaaS的几个典型服务商有Heroku、Cloud Foundry、Microsoft Azure Web Apps等。

3.软件即服务(SaaS)：软件即服务（Software as a Service，SaaS）服务是云计算的一种新的服务形式。它向用户提供完整的业务解决方案，如项目管理、文档协作、邮件发送等，用户无须再下载安装软件，只需访问服务网站或客户端软件，就可以使用服务。SaaS服务模型侧重于服务的封装和销售，用户无需自己管理服务器、存储、网络等，只需订阅服务，即可使用软件。目前，越来越多的公司开始使用SaaS模式，如亚马逊的AWS S3、Dropbox、Zoho Office Suite等。

除了以上三种服务，还有一种混合云架构：也就是把部分计算资源、网络和存储资源放在本地，而部分计算资源、网络和存储资源则被托管在云上。这种架构可以有效降低本地设备投入成本、提升资源利用率和节省运营成本。国内外很多大型互联网公司都采用了混合云架构，如阿里巴巴、腾讯云、百度云等。
