# 1.简介
  

## 一、市场分析模式概述
市场分析模式（又称投资分析模式），是指根据市场状况，对其中的股票进行预测和评价。这种方式不是简单地通过观察股价涨跌的快慢来判断股票走势，而是依据一定的规则或方法，对过去的表现、当下的事件、未来的趋势做出预测和评价。市场分析模式有两种主要类型：技术指标型分析模式和量化分析模式。本文将重点介绍量化分析模式中最常用的技术分析方法——技术指标分析。
## 二、技术分析方法简介
技术分析（Trading Analyses）是以分析股市、外汇市、期货市等各种市场中炒作行为的交易者所创造的一个综合工具，它利用经济、金融、统计学和计算机科学的原理及相关理论，通过计算和分析数据，来预测市场的变化、预测交易者的决策行为，从而帮助交易者更好地理解市场，做出更有效的交易决策。技术分析在市场分析模式中扮演着重要角色，帮助投资者捕捉到投机性强的股票，发现风险的股票，识别高波动率的个股，制定出符合个人投资目标的交易策略。
### 1.技术指标分析简介
技术指标（Technical Indicator）是用于研究股市、外汇市、商品市场以及其他金融市场价格的变动趋势、运动规律、成交量、市盈率、波动率、财务数据等数字化信息的客观、可靠的指标。技术指标分析的目的是为了发现和研判市场的行情趋势、判断买卖时机、掌握主导市场走势，并改善投资者的投资决策。
#### 1.1 技术指标作用
技术指标的作用如下：
- 提供关于市场趋势的信息：技术指标能够提供市场变化的实时的、准确的、连续的信息。
- 辅助选股：技术指标可以协助投资者识别股票走势，找出“不跌反升”的股票；也可以筛选出有潜在风险的股票，给予警惕。
- 测试交易系统：技术指标可以帮助交易者测试自己的交易系统是否合理，以及是否会遭受系统性的错误。
- 检查行情的真实性：技术指标可以帮助投资者检查最新行情信息是否有异动，防止被炒作的股票。
#### 1.2 技术指标分类
技术指标分为以下几类：
- 趋势型技术指标：描述了股价或其他价格变动的趋向、周期性特征、方向性、盘整性，如移动平均线、指数移动平均线、布林线、三角线指标、MACD、KDJ、DMA、TRIX、VRSI、CCI等；
- 动量型技术指标：描述了股价波动的加速或减缓幅度，如相对强弱指标RSI、动量指标、能量指标、BOLL、瀑布线等；
- 超买超卖型技术指标：描述了股价的过度涨停或过度跌停现象，如倒锤头指标、肩顶指标、佳庆指标等；
- 量价指标：描述了一种股票的价值，如市盈率、市净率、每股收益、溢价率等。
#### 1.3 技术指标应用场景
技术指标的应用场景包括以下几个方面：
- 个股分析：技术指标可以帮助投资者分析某个特定品种的市场情况，判断其适合进入哪些行业、市场区域、发展阶段等。比如，沪深A股采用“技术线”及相关技术指标分析；期货市场的交易者采用“笔中枢”、“死叉”、“震荡”、“支撑位”等技术指标进行分析；黄金市场则采用多个衡量标准来判断黄金价格的走势。
- 大盘分析：大盘市场由于规模、流通量较大，需要通过技术指标来分析市场整体趋势，判断投资趋势及适合进入的股票。比如，NASDAQ采用“道琼斯指数”、“纳斯达克综指数”及多种技术指标来追踪全球股市的走势；上证指数也采用多种技术指标作为市场分析工具。
- 区间套利：技术指标的研究结果可以作为基准对冲套利品种的盘口及时调整，提高套利效率。例如，短线交易者利用技术指标进行多空对比，选择突破性高低点的品种进行交易，例如日线突破、月线突破等；长线交易者则以不同的时间跨度来研判价格的变动，找出相应的盈利机会，进行长期交易。
- 风险控制：投资者可以基于技术指标进行仓位管理，确定风险和效益的平衡点。例如，A股采用“成长线”、“估值指标”、“成长性指标”来识别公司的基本面趋势，并根据该信息进行仓位管理；债券市场采用“夏普比率”、“杠杆率”等指标来评估债券的风险水平，并结合对应交易平台的风控规则进行交易管理。
- 搜索股票：技术指标可以帮助用户快速搜索到感兴趣的股票，并帮助用户对其进行分析，判断其适合进入哪些行业、市场区域、发展阶段等。比如，搜狗输入法的股票搜索功能可以采用“技术指标”来展示查询结果的排名及相关信息。
- 交易系统测试：技术指标还可以用于测试交易系统的正确性、效率，帮助用户避免错误操作带来的损失。比如，交易者在决定采用什么样的交易策略时，可以参考一段时间内的“曲线救国”结果，判断系统是否具有应有的优势。
- 选股：技术指标可以在个股分析、大盘分析、区间套利、风险控制等过程中被应用，来识别出股票的走势、未来走势、风险水平、投资价值等。投资者可以通过技术指标的研判，根据自身特质、偏好、目的，来选择投资对象。
- 数据辅助：技术指标也是研究市场行情、分析股价的一种重要方式。比如，同花顺新浪网上的股票分析软件就集成了众多技术指标，供用户查看及分析。
