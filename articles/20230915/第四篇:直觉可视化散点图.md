# 1.简介
  

散点图（Scatter plot）是一种简单的图表形式，它用来显示数据的分布特征。在散点图中，各个数据点都用一个点来表示，坐标轴上有一个变量(通常是横轴)，另外两个变量的值在该变量对应的坐标系中表示。散点图有助于观察到数据的相关性、趋势性和拟合度，因此能够给出前所未有的洞见，同时也具有很好的解释力。本文将对散点图的一些基本概念进行说明并介绍其可视化技术。
# 2.基本概念及术语
## 2.1 散点图概念
散点图由一组数据点组成，其中每一个数据点都是一个二维或者三维向量，通过指定横纵坐标来确定每个数据点的位置。散点图可用来呈现一些数据的分布特征，包括线性关系、非线性关系、离群点、聚集性、异常值等。
## 2.2 散点图术语
* **样本**（Sample）：指原始数据中的一项数据，可以是一行记录，也可以是多行记录构成的数据集合。如：一条销售数据记录，可能包含客户ID，商品名称，购买时间，购买数量，付款金额等信息；或者是一个公司的员工信息记录，可能包含员工ID，姓名，部门，职称，薪资，工作年限等信息。
* **特征**（Feature）：指样本中的一个属性，即描述样本的某个方面或特质。如：一条销售数据记录中，可能包含客户ID、商品名称、购买时间、购买数量、付款金额等特征。
* **类别变量**（Categorical Variable）：类别变量又称分类变量、分类型变量，指分类数据，如客户性别、职称等。
* **连续变量**（Continuous Variable）：连续变量又称数值变量、量值变量、实数变量、变数变量，指实数、负无穷小到正无穷大的数值数据。
* **坐标轴**（Coordinate Axis）：坐标轴由横轴和纵轴两条线组成，分别对应变量的不同取值范围。
* **散点**（Scatter point）：散点是坐标系中表示一个样本的点。散点的横纵坐标表示该样本在两个特征上的取值，其颜色表示该样本的类别标签或其他属性。
* **样本中心**（Center of Sample）：样本中心是指样本数据分布所在的中心点。
* **回归直线**（Regression Line）：回归直线是两种变量间关系的一种直线，用于估计未知数据的分布规律。
## 2.3 数据准备
散点图可用于展示两种或以上变量之间的关系，所以这里假设有一个带有三个特征的数据集。为了便于演示，我们生成一个随机数据集。


```python
import numpy as np
from sklearn import datasets
import matplotlib.pyplot as plt
%matplotlib inline 

np.random.seed(1) # 设置随机种子，使结果可以重复产生
X_org, y = datasets.make_classification(n_samples=1000, n_features=3,
                                        n_redundant=0, class_sep=2, random_state=1)
plt.scatter(X_org[:, 0], X_org[:, 1])   # 可视化X1和X2的分布
plt.xlabel('X1')
plt.ylabel('X2')
plt.title("Visualization of X1 and X2")
plt.show()
```
