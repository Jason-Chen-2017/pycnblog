# 2.基本概念和术语
## 什么是机器学习？

机器学习（英语：Machine Learning）是一门人工智能的子领域，它利用计算机数据及统计学方法来研究和预测关于经验或从数据中提取的规律性，并应用到新的数据中以概率更高的方式做出决策。机器学习是一类人工智能技术，可以让机器自动分析和解决问题，而不需要一个明确的指令序列。它与统计学、模式识别、优化理论等多个学科密切相关。

## 为什么要学习机器学习？

学习机器学习的目的，首先是要弄清楚它到底是干什么的。它通过计算机处理大量数据，来提高效率、自动化、准确地执行重复性任务。学习它的目的是为了让计算机具备自我改善的能力，提高产品质量、解决复杂问题。那么，怎么才能学得更快、更好呢？当然，主要靠自己投入的时间和努力。

学好机器学习需要三个步骤：

1. 数据准备和清洗——掌握数据的特征、质量、分布等信息，掌握数据处理的方法和工具。

2. 算法选择——决定采用哪种模型或算法来拟合数据，了解算法的原理和特点。

3. 模型训练和评估——使用训练集和测试集分别训练和评估模型，根据模型性能指标调整参数，直到达到预期效果。

## 什么是 Deep learning?

深度学习（Deep Learning）是机器学习的一个子领域，它是建立在多层的神经网络上的。神经网络的每一层都由一组神经元组成，每个神经元接收前一层的所有输入，根据激活函数的值计算输出，并向下传递误差信号。这个过程反复迭代，最终达到对原始数据的逼近或预测。深度学习主要应用于图像识别、文本分类、语音识别、视频分析等领域。

## 为什么要学习 Deep learning？

学习 Deep learning 的目的是为了让机器具备强大的学习能力，理解复杂的问题，并且解决问题的能力更强。这也是人工智能技术目前发展的一个方向。那么，要学得更快、更好吗？当然，靠自己投入时间和努力。

学好 Deep learning 需要四个步骤：

1. 准备数据——收集足够的数据，尽量让数据有代表性。

2. 构建模型——搭建模型结构，并定义损失函数和优化器。

3. 训练模型——训练模型，使其能够快速收敛，产生较好的结果。

4. 测试模型——使用测试集测试模型的准确性。

## 概念术语

- 数据集（Dataset）：用来训练和验证模型的数据集合。通常有标签（Label）表示样本对应的类别，也可以没有标签，仅用于训练模型。
- 特征（Feature）：指的是用于描述样本的变量或者属性。特征可以是连续的，也可以是离散的。
- 标签（Label）：指的是样本对应的类别，是一个目标变量。
- 标记（Marker）：用于区分训练集和测试集的标识符。
- 特征工程（Feature Engineering）：是指从原始数据中提取特征，去除噪声、降维、归一化等操作，提高模型的鲁棒性和泛化能力。
- 划分训练集和测试集：把数据集划分为两个独立的集合，一部分用来训练模型，一部分用来测试模型的准确性。
- 模型（Model）：是基于特征向量空间中的点与超平面的分类函数或回归函数，用来对输入数据进行分类或回归。
- 超平面（Hyperplane）：由 n+1 个单位正交基向量决定的平面，可以用来进行二类分类或回归。
- 正则化（Regularization）：是一种通过添加一个惩罚项来减少模型过度拟合的技术。
- 交叉熵（Cross Entropy）：交叉熵是一个用来衡量两个概率分布p 和 q 之间差异的指标，是信息熵的另一种度量方式。
- 均方误差（Mean Squared Error）：又称为平方损失函数、残差平方和或残差平方和均值。是最小二乘法的代价函数。
