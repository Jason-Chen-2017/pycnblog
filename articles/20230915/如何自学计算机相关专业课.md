# 1.简介
  

在当今信息化时代，计算机科学成为各个领域的重要基础技能。作为本专业学生或者进修生，如何自学计算机相关专业课是一个巨大的挑战。本文旨在分享一些简单的方法或建议帮助大家快速入门学习计算机科学相关专业课，并且可以在此过程中找到个人发展方向。  
# 2.目的
首先，了解自己的兴趣点、目标以及学习方法，可以更好地了解这个专业需要什么样的知识、技能以及能力。其次，如果能够自主完成课程设计，能够独立解决实际编程问题，对学习的热情也会得到强化，并因此获得更多成就感和收获。最后，提升自身沟通交流、分析问题的能力，还能让自己融入到计算机科学的开源社区中。  
# 3.准备工作
为了能系统性地掌握计算机科学相关专业课的内容，需要在开头花些时间做好准备。最好的方法是在确定目标之后，按部就班地阅读课程教材。这是最有效的方式，因为你可以系统性地理解这些内容，然后再根据自己的兴趣选择学习哪些内容。以下是几条建议：  

1. 了解相关专业课的教授方式，而不是只是学完课就可以毕业。很多计算机科学相关专业课的内容都比较复杂，所以即使你只是考试通过了，也不代表你的水平就一定比其他同学高。一个合格的计算机科学学者应该具备足够的能力和知识来面向需求的环境和问题，而不是单纯地被一堆知识所驱动。

2. 在决定学习专业课之前，充分调研一下相关专业的培养方案。每年都会发布新的教材和新版本的教材，而且有时候还会临时调整某些内容。要想把握住专业课的真正重点，就需要仔细阅读相关的教材，甚至找老师咨询一下。

3. 订阅相关专业的视频网站，看看老师们的讲座笔记，以及其它一些免费的学习资源。通过观看视频课可以了解到该课所涉及到的内容、专业术语、算法、代码实例，还有演示如何用代码实现某个功能的过程。这样，你可以逐渐了解到相关的基础知识和技能，并有机会获得老师们的建议。

4. 学习之前，务必确保自己具备相关的专业知识和技能。有的专业课如机器学习或大数据处理等涉及到了高级算法、数学理论等方面的内容，要求学生具有较强的数学基础和统计学功底。这些课程需要有扎实的理论功底，才能充分理解其中的理论。而另一些课程则不需要太多的数学基础，只需要具备一些编程能力即可。

5. 如果你的英语能力较差，建议尽量早期进入相关专业的学习阶段。因为有些专业课需要翻译成英文授课。而且，阅读、听力、口语、写作等能力都是需要长期训练的，才能取得优异的效果。

# 4.关键技术和应用场景
通过前面的准备工作，我们已经知道了学习相关专业课需要具备哪些基本素质。但是很多人可能仍然不是很清楚自己究竟想要从事哪种类型的计算机应用开发工作。以下是一些关键技术以及对应的应用场景，供大家参考：  

- 数据结构与算法：有序集合、搜索树、图、排序、哈希表、动态规划、线性规划、贪婪法、回溯法、分治法、最短路径、最小生成树、网络流、字符串匹配、模式识别。这类课程需要掌握数据结构和算法相关的知识，包括数组、链表、栈、队列、散列表、二叉树、堆、哈夫曼编码、斐波那契数列等。可以尝试编写一些简单的算法题目来检验自己的编程能力。例如，你也可以写一个求最大公约数的函数。

- 操作系统：由于计算机硬件资源有限，操作系统就是一种非常重要的软硬件协同机制。掌握操作系统原理、进程管理、同步互斥、内存管理、虚拟存储器、文件系统等知识，可以帮助你理解系统的运行机制，以及如何利用操作系统实现相应的应用。可以参加操作系统原理与实践的课程，掌握Linux操作系统的相关知识，并在实验室搭建一个小型的操作系统实验环境。

- 计算机网络：计算机网络是实现不同计算机间通信的一项基础技术。掌握计算机网络协议、物理层、数据链路层、网络层、传输层、应用层等知识，可以帮助你理解计算机网络的工作原理，以及如何基于该技术开发各种网络应用程序。可以参加计算机网络原理、协议、实践等系列课程，掌握互联网、局域网、VPN、SDN等网络技术，并在实验室搭建一个小型的网络实验环境。

- 数据库：数据库是一种持久化的数据存储技术。掌握关系数据库的相关知识，如SQL语言、事务控制、查询优化、索引组织、并发控制等，可以帮助你理解数据库的工作原理，以及如何开发出高效、可扩展的数据库系统。可以参加数据库原理、理论与实践等系列课程，掌握MySQL、PostgreSQL、MongoDB等数据库产品，并在实验室搭建一个小型的数据库实验环境。

- 智能算法：智能算法就是计算机用来解决具体问题的计算模型或算法。它不仅依赖于数学推理和逻辑推理，还需要考虑计算机的运算性能、能耗特性、数据规模、计算精度、表达能力等多方面的因素。掌握关键技术、算法理论、应用场景、常见框架、案例研究等知识，可以帮助你更好地理解和运用智能算法。可以参加计算智能导论、模式分类与决策、机器学习、深度学习、推荐系统、文本挖掘等系列课程，并结合实际应用场景进行练习。

- 大数据处理：大数据处理的目的之一就是解决海量数据的存储、分析、挖掘问题。掌握大数据技术的最新进展、关键技术、应用场景、典型案例研究等知识，可以帮助你理解如何利用云计算、NoSQL、搜索引擎、数据采集、数据处理、数据分析等技术处理海量数据。可以参加大数据综合、实践、监督、项目等系列课程，掌握Hadoop、Spark、Storm、Flink等大数据处理平台，并在实验室搭建一个小型的大数据处理集群。

# 5.获取编程经验
如果你想在计算机科学相关专业课学习编程，可以先去找一份编程兼职工作，熟悉相关编程语言的语法规则、标准库、第三方库，以及如何调试代码。编程经验可以帮助你更好地理解和解决问题，并提升自己的动手能力。当然，你也可以直接申请一些技术培训班、微型公司招聘编程人员，或是找一本计算机相关专业的书籍学习。

# 6.自学策略
与其盲目自学，不如通过一些系统性的方法来管理学习任务，最终达到有效学习的目的。这里给出一些自学策略建议：

1. 制定详细的学习计划。将课余的时间安排好，比如每天学习一小时、每周学习一个主题等。不要浪费时间在无意义的事情上。

2. 建立良好的学习习惯。坚持使用合适的学习工具，比如闪卡、思维导图、笔记、文档、备忘录等。记录自己的学习过程，并定期总结复盘。

3. 关注课程反馈。反馈是学习最有效的方式。老师和助教会提供反馈，可以看到学生的学习情况，并及时帮助改善学习效果。定期跟踪学习进度，并与同学分享自己的收获。

4. 提升自己的沟通能力。最有效的沟通方式是直接与老师和助教沟通。但往往这种方式比较困难，需要克服一些心理障碍。比如，如果你觉得学习这门课很难，可能会极端地拒绝老师和助教的邀请，但其实自己学习的挫败感也会超过他们的鼓励。可以通过参与同学的讨论、主动提问、发起辅导计划等方式提升自己的沟通能力。

5. 寻找志同道合的伙伴。有些专业课往往有特定的交流群，在上面可以找到志同道合的同学交流。你也可以加入这些群里，与同学们一起分享学习心得、提升技能，并一起进步。

最后，不要忽视自己的兴趣和梦想。只要努力地付出，终将学到自己真正想要的东西。
