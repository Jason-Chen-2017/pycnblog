# 1.简介
  

　　什么叫做“制定商业计划”，它对商业活动的目标、领域、预期收益有什么样的要求？在企业运营中，应该如何进行商业计划的制定呢？一份好的商业计划需要包括哪些内容？下面，我将从宏观经济学、投资学、管理学等多个视角探讨这个问题。
　　首先，我们可以从两个方面回答开头的问题——背景介绍和基本概念术语说明。我们将会主要关注“商业计划”的定义及其涵盖的内容、分析各种商业计划制定的方法论，并进一步阐述制定商业计划的流程和注意事项。为了更好地理解商业计划，还需了解商业计划与商业模式之间的区别。
# 2.背景介绍

　　什么叫做“商业计划”，即企业在某个阶段所达成的目标，以及要实现这些目标所需付出的努力和资源。企业计划是指企业或组织的决策者制订的长远计划，目的是为了实现其目标而制定的一系列行动，例如扩张生产规模、增加盈利能力、改善产品质量、提高竞争力等。它是企业发展的蓝图，使得企业能够按计划前进，逐步实现其理想目标。

　　根据《中国统计年鉴》，2018年全国新增就业57万人左右，其中创业就业19万余人，占比达到74%；全社会新增就业8800万人，创业就业率达到56.6%。以创业就业为例，近年来创业就业已经成为百万富翁最重要的收入来源之一。另一方面，近年来各类创新型企业如科技公司、农林牧渔企业、房地产企业也频繁产生新项目，这些项目往往带来新型需求、竞争优势，传统企业则不断变换战略方向、布局新市场。因此，企业在制定商业计划时，必须面向长远，充分考虑到公司未来的发展方向和现实条件，寻求长远的增长潜力。

　  　　一个好的商业计划，应该既包括公司业务方向的战略性计划，又包括公司资源配置的整体规划。因此，商业计划制定的一般过程如下：

　　1.背景介绍：了解相关业务，明确目标，确定范围，制定衡量标准。

　　2.项目设置：制定周期、难易程度、产出标准、风险控制措施、奖惩机制。

　　3.资源配置：调配人力、物力、财力、技术支持、人员流动与管理，完成目标任务。

　　4.后续跟踪：完善经营方式，持续优化，提升竞争力。

　　由于商业计划涉及到多方利益关系、利益分配、人才培养、市场环境、税收政策等复杂因素，所以它是企业综合实力、统筹全局、有效协同作用的结果，也是社会发展的必然产物。因此，制定商业计划务必慎重，力争精准，否则将导致意外收获、悲剧结局。另外，企业还应当密切关注外部环境的变化，动态调整商业计划，以适应新的发展形势。

# 3.基本概念术语说明
　　为了准确、全面地讲清楚商业计划，我们首先需要对一些基本概念、术语做简单说明。以下是本章节涉及到的主要术语：


　　- **商业模式：**一种新的企业发展模式、新产品或服务的创造和传播过程。在商业计划制定过程中，它的目的就是为了使企业满足消费者的不同需求，实现企业价值最大化，最终赢得市场竞争。商业模式需要包括商业理念、核心价值和客户群体定位、用户体验、核心功能、市场细分、市场形态、分销模式、运营策略等方面的信息。

　　- **商业目标：**企业所追求的长远目标，是指企业发展到一定阶段时所企望达到的长远收益，通常包括核心业务收入、营业利润、员工满意度、研发投入、品牌声誉等指标。商业目标应该符合客观实际，不能过于宽泛或者高估短期效益。

　　- **商业预算：**企业在一段时间内用于经营管理的金钱资产，是指企业在给予经费时所设定的预算总额，对企业某项特定计划的支出、建设资金、雇佣人数、招聘人员、设备、供应商等均占一定的比例。商业预算根据企业计划的性质和规模而异，通常由企业财务部门制定，既可以独立于经营计划之外，也可以作为其经营计划的组成部分。

　　- **商业机会：**企业在市场上可获得的优惠券、活动、条件或其他资源。商业机会使企业能够获取一定的竞争优势，通过其产品或服务进入竞争者的领地。商业机会是企业寻找新增长点、改善经营方式、扩大边界的有利工具。

　　- **产品生命周期管理（PLM）系统：**一套完整的管理系统，用来创建、开发、交付和售卖商品或服务，包括产品设计、营销管理、采购、库存管理、供应链管理、生产制造、贸易管理等环节，以达到整体产品的生命周期管理。

　　- **商业计划书：**企业针对特定商业领域或产品或服务的详细、经营、营销、发展规划，包括营销计划、产品计划、人力计划、财务计划、战略规划、投资计划、合作计划、组织结构、信息系统、服务方案等。商业计划书可以帮助企业了解市场情况、客户需求、产品差异、竞争对手情况、预测市场情况、确定战略投资计划、谋求盈利空间和客户忠诚度等。

　　- **商业计划评审：**商业计划的评审包括对商业计划的专业性和实用性的评估，同时还要审核计划的内容、计划执行的实际情况和结果，确保计划能够真正落实到企业的日常经营中。

　　- **商业计划会议：**为企业制定、改进和完善商业计划的一次集体讨论，包括商业计划草案审查、问题陈述、讨论、决策与执行。商业计划会议一般是由企业管理层主持，但也可邀请专业咨询公司参加。
