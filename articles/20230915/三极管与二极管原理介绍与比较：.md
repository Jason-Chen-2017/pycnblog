# 1.简介
  

三极管（Triac）及其变体、二极管（Diode）及其变体是电子电路中重要的逻辑元件之一。它们都属于印制在半导体上的集成电路IC上，用于控制、放大或屏蔽电流信号，是电力系统中不可缺少的一环。在现代生活中，三极管已成为最常用的一种电源装置，不仅可以作为制冷设备，而且还可以作为空调、电视机顶盒等设备的电源。除此之外，三极管还可用来驱动电机、照明设备和其他电子设备。

本文将对三极管、二极管及其之间的区别与联系进行介绍并提供一些经典应用的案例，以帮助读者更好地理解三极管与二极管，更好地使用三极管及二极管。
# 2.三极管原理介绍
三极管是一个集中器件，由三个电阻组成，形成一个三角形三点火线。在某一时刻，只有其中两根电阻导通，其他一根导通处于隔离状态，称为开关；当另两根导通时，三极管通电，称为闭合状态。根据三种工作模式可分为：

- 滴电模式（Lamp Mode）：激励电流通过接通电路，导通另两端电阻，而第三条导线直连一个或多个受控器件，使受控器件上的感光元件发出可见光信号，并在一定时间内保持亮度，再次重复过程，即循环激励。
- 交流矩edMode（Power Mode）：负载电流通过接通电路，导通一条导线，直通第三条导线，将被控电路信号转换成相位相同的功率输出。该模式用于发射、接收无线电信号。
- 流量edMode（Flow Mode）：负载电流通过接通电路，导通两条导线，直通第三条导线，输出频率差为零。该模式用于加热、冲洗或空气过滤。

三极管的发展历程从单片三极管(MOSFET)到高压回路，再到混联、高效、低损耗等系列，逐渐形成了三类不同功耗的三极管。

1. 单片三极管(MOS FET)：最初的单片三极管(MOSFET)，一般只用一个二极管就可实现低门槛高精度的负荷分流。但由于它只有两个负极、一个阳极，所以只能产生垂直于导线的有功磁场。由于MOSFET门槛低、器件重量小，所以应用范围很广。

2. 高压回路三极管(Bulk-Filled Triac):高压回路三极管(Bulk-Filled Triac)采用了多个三极管的组合形式，实现了非常大的电压效率。它内部集成了多个三极管片，每个三极管片之间通过导体连接，形成一个多层三极管结构。这样既克服了单片三极管太小，无法承载大电流负荷的问题，又避免了噪声累积带来的影响。同时，它也比传统的三极管有着更好的性能。

3. 混联三极管:混联三极管采用了电容材料作为晶体硅基层。每种三极管都是在单个器件中孔填充整个电容，增强其抗辐射性。混联三极管因其硬化的材料结构，具有较高的电压效率。

4. 高效三极管:高效三极管采用偏导体材料，如锗电池材料、天然稳态氧化镍材料等。其厚度可以达到几十微米，因此可以实现非常高的电压效率，而且可以长期存储超高电压。

5. 低损耗三极管:低损耗三极管是指由微型IC组成的小型三极管。它在尺寸、绝缘性和噪声方面都有所降低，且其损耗很低，因此可以用于各种电子电路领域，如数字电路、微电子、IC设计等。

根据应用场景，三种类型三极管有不同的特性。

- 滴电模式三极管：适用于需要频繁改变电流或开关频率的环境。可以产生可见光的长波（可超过百万级的频率），通过反射的方式吸收光线，使接收对象产生相应反应。常用于制冷、照明等领域。
- 交流矩edMode三极管：适用于发射、接收无线电信号的应用，能够以高效的方式将信号转化成电能。常用于通信领域。
- 流量edMode三极管：适用于加热、冲洗或空气过滤等需要控制或稳定电流输出的应用。可以产生500MHz甚至更高的信号频率。常用于工业控制领域。

三极管常与电阻一起使用，形成三端电路。一般包括一个正极、一个负极、一个漏极以及一个时钟信号。正极负责导通二极管或三极管，负极负责隔离其他电路，漏极供电，时钟信号输入至漏极，控制时序。另外，也可以在时钟信号前或后加入一个隔离电容，防止反复触发。

三极管可以用于以下许多方面：

1. 常见电源电路
三极管通常用于各种电源电路，如照明、燃气、液化石油气等。三极管可以方便地引入多种电压，保证电源整体效率，减轻系统负荷。如户外温度补偿，车灯控制等。

2. 无源记忆电路
三极管可以作为二进制编码器，输出任意长度的比特串。如果把每个电压值看做“开”或“关”两种状态，那么每个电路就对应了一个比特。这种功能对于信息传输、数据处理、纠错等非常有效。

3. 放大器、缓冲电路
电流不能直接传递到电压，因此要用三极管放大信号的电压。还可以使用三极管构建一些简单缓冲电路，如电容式恒流稳定电路，电阻式单边谐波发生器等。

4. 时钟同步
时钟同步指的是由一个主时钟产生的计数脉冲同步，所有信号的脉冲都会以相同的速度发生，使得各个设备的时间保持一致。通常，这个信号会经过一串三极管，而这些三极管的时钟输入端与主时钟一致。

5. 负荷分流
在复杂负荷情况下，可以将同样的电流分成多个阶段分流，节省电源的功率，提高整体效率。这可以通过三极管的多段堆叠完成，如多路三极管、菊花链三极管等。

# 3.二极管原理介绍
二极管（Diode）是一种电阻元件，被誉为“黑匣子”。其两端分别连接两相电流，当其中任一相引起电流流动时，就会对另一相产生电流。两相导通，则导电性材料就会随着电流流动而产生损伤，当流动停止时，损伤就会消失。二极管是电容器中的基本元件，可用于制造功率电阻、变压器和电解膜。

根据工作模式可以分为如下四种：

1. SCR模式：SCR模式中的二极管是短截止二极管，也就是说，它的两个正极和一个负极之间只有一根导线相连，其余导线均为切断状态。在SCR模式中，损坏导线不会影响二极管的工作。一般用于放大电压或信号的损失或干扰。
2. DCM模式：DCM模式中的二极管由二极管栅极和二极管电极组成。二极管栅极是金刚石芯片，用以支撑二极管的两头连接导体，使得二极管可以绝缘耦合。二极管电极由一个导体、一个基底带和一组铁层组成，导体负责导通与不导通之间的切换，基底带负责流过二极管的电流。常用于降压、减小阻抗、保护电路等方面。
3. AFE模式：AFE模式即全功能二极管，其是一个三极管，除了导通两端电阻，还有第三条导线可以控制或选择导通哪一侧。该模式可以产生高低温度下的可变频率输出。
4. MOSFET模式：MOSFET模式由三极管变体、二极管栅极和二极管电极组成，主要用于高压应用，如汽车充电桩、步进电机驱动、功率分配、锂离子电池充电管理等。

# 4.三极管与二极管区别
三极管和二极管都是电容器中的重要元素，但是存在着一些重要的区别：

1. 发散参数
首先，三极管是轴向导电结构，导电性材料与导体固定在轴线上，导线数目越多，导通距离变长。相反，二极管是平行导电结构，导电性材料与导体都沿平行方向直线固定，导线数目也相对少一些，导通距离较短。

2. 导通方式
由于导电性材料的不同，三极管的导通方式与导线数目有关。在单线架模式下，导通方式为“感应泄露”：当导线导通时，正负极皆导通，导电性材料进入导电性区，导体部位发生短路。在双线架模式下，导通方式为“压电泄露”：当导线导通时，正极导通，导电性材料进入导电性区，导体部位发生短路。当导线失去导通时，导电性材料流过导体，导体部位恢复正常。

3. 上电条件
三极管的上电条件要比二极管复杂，必须满足两个方面的要求。首先，必须考虑导电性材料自然回流导致的熔丝烧坏等安全隐患。其次，为了防止二极管内部的漏电，必须考虑使用防雷排等措施。

4. 模式
三极管和二极管虽然都是轴向导电结构，但它们的工作模式却大不相同。三极管可以实现各种工作模式，如“滴电模式”，“交流矩edMode”，“流量edMode”，应用广泛；而二极管一般用于制作功率电阻、变压器、电解膜、电容、电阻，主要用于电源电路。

5. 使用场合
三极管应用广泛，但功耗高，电流能力有限；二极管应用较少，但功耗低，电流能力佳。因此，对于特定应用，优先考虑三极管，如照明、燃气、液化石油气；对于通用电路，优先考虑二极管。

6. 制造技术
三极管和二极管的制造技术也存在着巨大差异。三极管的制造技术依赖于代工方法，需专业工艺人员参与制造。二极管则采用电容材料加工工艺，属于印刷技术，简单易做。

综上所述，三极管和二极管都是电容器中的重要元素，但是存在着一些重要的区别。读者应该根据自己的需求、应用和需求来选取适合自己应用的三极管或二极管。
