# 3.核心概念及术语
## 3.1 数据结构
数据结构是计算机编程中的重要概念。它定义了如何组织和存储数据的方式，以及如何处理数据的方法。在Python中，数据结构主要有以下几种：

1. 列表(list)
2. 元组(tuple)
3. 字典(dict)
4. 集合(set)

### 3.1.1 列表 List

列表(List)，又称数组（Array），是一组按顺序排列的元素。一个列表可以包含多个数据项，这些数据项可以是同种或异种类型，可以动态添加或删除元素，并且列表长度不固定。列表可以用方括号 [] 来表示，列表元素之间用逗号隔开。

```python
# 创建列表
my_list = [1, 'hello', True]
print(my_list)

# 访问列表中的元素
print(my_list[0]) # 1
print(my_list[1]) # hello
print(my_list[-1]) # True (最后一个元素)
```

列表也可以嵌套，即列表中可以包含其他列表。

```python
# 创建嵌套列表
my_list = ['apple', 'banana', 'orange', ['grape', 'peach']]
print(my_list[3][0]) # grape
```

### 3.1.2 元组 Tuple

元组（Tuple）与列表类似，不同之处在于元组的元素不能修改。元组用圆括号 () 表示，内部元素用逗号隔开。

```python
# 创建元组
my_tuple = ('apple', 'banana', 'orange')
print(my_tuple)

# 访问元组中的元素
print(my_tuple[0]) # apple
```

### 3.1.3 字典 Dictionary

字典（Dictionary）是另一种数据结构，它是一个无序的键值对集合。字典的每个键都对应一个值。字典用花括号 {} 表示，内部元素用逗号隔开。

```python
# 创建字典
my_dict = {'name': 'Alice', 'age': 25}
print(my_dict['name']) # Alice

# 更新字典的值
my_dict['age'] = 26
print(my_dict['age']) # 26

# 添加新的键值对
my_dict['city'] = 'Beijing'
print(my_dict) # {'name': 'Alice', 'age': 26, 'city': 'Beijing'}
```

### 3.1.4 集合 Set

集合（Set）是一个无序的元素集合，不允许重复的元素。集合用花括号 {} 或 set() 函数创建。

```python
# 创建集合
my_set = {1, 2, 3, 2, 1}
print(my_set) # {1, 2, 3}

# 操作集合
my_set.add('hello')
print(my_set) # {1, 2, 3, 'hello'}
my_set.remove(2)
print(my_set) # {1, 3, 'hello'}
```

| 类型 | 描述 |

## 3.3 数据运算

Python 中有丰富的数值运算符，可用于进行基本的数据运算。

| 操作符 | 描述 | 示例 |
| ---- | ---- | ---- |
| + | 加法 | a+b |
| - | 减法 | a-b |
| * | 乘法 | a*b |
| / | 除法 | b/a (结果是浮点数) |
| // | 取整除 | b//a (结果是整数) |
| % | 模ulo | b%a (余数)|
| ** | 幂运算 | a**b (a^b)|

例如：

```python
x = 2
y = 3

print(x + y)       # 5
print(x - y)       # -1
print(x * y)       # 6
print(x / y)       # 0.6666666666666666
print(x // y)      # 0
print(x % y)       # 2
print(x ** y)      # 8
```

## 3.4 文件处理

Python 中的文件处理包括读写文件、文件查找、文件复制等操作。

### 3.4.1 读取文件

使用 read() 方法可以一次性读取整个文件的所有内容，返回的是一个字符串。如果需要每次读取一行，可以使用循环语句。

```python
# 以只读模式打开文件
with open('filename.txt', mode='r') as file:
    content = file.read()
    
print(content)
```

### 3.4.2 写入文件

使用 write() 方法可以向文件中写入内容。

```python
# 以追加模式打开文件
with open('filename.txt', mode='a') as file:
    file.write('\nThis is the new line.')
```

### 3.4.3 查找文件

os模块提供了用于文件和目录操作的函数接口。

- `getcwd()` 获取当前工作目录路径
- `listdir(path)` 返回指定目录下的所有文件名称的列表
- `isfile(path)` 如果路径指向的文件存在且为普通文件，则返回True；否则返回False
- `isdir(path)` 如果路径指向的是一个目录，则返回True；否则返回False

```python
import os

# 当前目录路径
curdir = os.getcwd()
print(curdir)

# 指定目录下的所有文件名
filenames = os.listdir('.')
for filename in filenames:
    print(filename)
    
# 检查文件是否存在
if os.path.exists('./file1'):
    print('file exists')
    
# 拼接文件路径
filepath = os.path.join('.', 'file2')
print(filepath)
```

### 3.4.4 复制文件

使用 shutil 模块可以实现文件的复制功能。

```python
import shutil

src = '/path/to/source/file'
dst = '/path/to/destination/file'

shutil.copyfile(src, dst)
```
