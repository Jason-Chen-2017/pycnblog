# 2. 概念与术语
## 2.1 定义与分类
“沟通”是指信息的传递、交流、沟通和互动，是人的正常活动。

沟通可以分为四个层次：

1. 直接沟通：当两个人存在直觉或直观联系时，他们之间就可以直接进行沟通。例如，在进行商务谈判、企业竞争时，双方就能够彼此相互了解，并明白彼此的需求。

2. 非直接沟通：当两个人无直接联系时，通过其他的方式才能实现沟通。例如，借助电视、报纸、网络等媒体进行的多次交流；参与群体活动，例如共同举办户外活动、举行社区活动等；进行线上或线下的见面会议等。

3. 影响沟通：影响沟通，又称心理沟通或身体沟通，是指用语言或行为来影响对方。它可能是不真实的，可能伤害了另一方，甚至可能会导致其发生性关系。

4. 技术沟通：技术沟通是指利用计算机、手机或智能设备等工具，采用语言进行信息的传递、交流和沟通。技术沟通可以使两个人之间无法直接接触，有利于打破现状、增加互动性。

除了沟通的不同层次，沟通的方式也是影响沟通的重要因素。主要有两种方式：

1. 语言沟通：语言沟通是指直接通过语言进行的交流，包括文字、电话、视频等。语言沟通较为直接，效率高，易于传播，但是语言限制太多。

2. 非语言沟通：非语言沟通是指通过其他的形式进行的交流，如表演、影像、绘画、摄影、交换意见等。非语言沟通需要建立亲密关系，而且需要大量的美化劳动。

除了沟通的不同层次与方式，还有许多沟通相关的概念与术语，如：

1. 对话框：一种交流模式，即把沟通活动分成几个阶段，每个阶段分别由不同的角色参与，有利于激发共鸣、减少焦虑和冒失，同时也容易引起误会。

2. 主题：沟通过程中所关注的问题领域或事件，如人生、工作、个人生活等。

3. 观点：是人们持有的关于某个问题、某件事物的看法、态度或评价。

4. 陈述：是有组织的语言陈述，通常是完整且正确的信息。陈述语句具有可操作性，使得读者清楚地知道应该做什么、为什么这样做。

5. 请求：请求是一种语言行为，旨在要求对方做出回应。请求语句一般是直接陈述事实、请求提供更多信息、或批评错误。

6. 确认：确认是一种语言行为，是指确认对方已经知道的事实。确认语句用于回答问题、澄清意见、补充细节等。

7. 约定：约定是指双方一再重复的陈述，用于保证双方之间的沟通顺畅、有序。约定的目的主要是让对方相信对方的内容一定是正确的。

8. 归属：归属是一种自然语言习惯，即将自己的言行和行为固定下来，认为自己一直拥有某种身份。归属往往带有感情色彩，有助于降低沟通的复杂性，促进感情上的互动。

9. 恭维：恭维是一种语言习惯，表示肯定或赞扬别人的言行。恭维既可以自然发生，也可以主动发起。如果恭维别人，可以表示尊敬和祝福，也能激发感情，产生积极的反馈。

10. 激发力：激发力是指以简单、有效的方式激发另一方的兴趣、参与感、情绪。激发力强的人往往倾向于更积极、直接、直言。

## 2.2 核心沟通技巧

在面对各种沟通问题时，我们可以使用一些核心沟通技巧来更有效地沟通。以下是一些常用的沟通技巧：

1. 提问：提问是一个关键的沟通方式，可以帮助我们准确地表达自己的意愿，并且可以得到及时的反馈。

提问是最基础也最重要的一步，要善于提问，才会有效地回答问题。提问的方法可以有很多种，比如开门见山地问问题，阐述自己的疑惑，提出建议，提供信息，或指出问题所在。在回答问题时，记住三条建议：

1) 识别关键词。首先，找出问题的关键词。这一步是十分重要的，因为回答关键词对应的问题，可以帮助我们快速理解问题。

2) 寻求支持。然后，要寻求足够的支持。一味的找借口、诉苦就显得不够有力。可以直接提供信息，或提出你的看法。

3) 直奔主题。最后，要直奔主题。这样才能更加清晰准确地回答问题。在回答问题时，要突出重点，把答案写成完整、易懂的语言。

2. 回应：回应是指接受别人的反馈并予以回应。回应能够帮助我们形成共识，发现新的问题或机会。回应的两种方式：自然回应和协商回应。

自然回应是指回应别人的直接语言。这种回应最直接，不会引起较多的争执。但是，不管从哪种角度看，都会有些不自然或不妥当的地方。

协商回应是指跟别人讨论问题，达成共识后再回应。协商回应能够更加主动，更加有说服力。但是，需要付出更多的时间和精力。

3. 倾听：倾听可以帮助我们更全面地理解对方的意思。倾听的目的是去理解对方想要什么，而不是仅靠听的热情。

4. 保持冷静：沟通的任何时候都不能做出过分的猜测、过于消极的态度或暴跳如雷。保持冷静是做好沟通的第一步。

5. 尽早结束：在对方讲完话或做完事后，要尽快结束对话，不要留下余地。

6. 积极应对：在处理具体事务时，要善于应对困难和问题，有耐心和勇气解决。在出现困难时，要坚持求同存异的心态，增强沟通能力。

7. 制定计划：制定沟通计划，并定期更新，能够有助于提升沟通能力。制定沟通计划时，要深入考虑各方的需求，选择合适的话题、时机，及时反馈反馈信息。
