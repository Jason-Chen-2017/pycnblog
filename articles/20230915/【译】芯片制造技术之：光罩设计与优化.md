# 1.简介
  

光罩（LCP）是一种非常重要的硬件组件，其作用在于降低物体的表面反射率、减少光线的漫延。它们具有各种形式，可以用来辅助各种不同用途的设备，比如显示器屏幕等，还可以为复杂结构提供保护层。
光罩的制作过程，包括光路设计、光栅布局、光栅加工、电镀、涂料应用、粉末处理等。光罩的性能会直接影响其应用领域的光学、功能、耗电效率、防护能力等性能指标。
为了更好的服务消费者，各大厂商都会花费大量精力在优化光罩性能上。但由于光罩设计技艺与制造工艺之间存在着巨大的差异，对于光罩的优化往往需要进行非常多的实践才能获得最优效果。因此，本文将从以下几个方面来阐述优化光罩的过程：
* 光路设计
* 光栅布局
* 光栅加工
* 电镀技术
* 涂料选择与配比
* 粉末处理方式及优化
* 其他参数优化
* 结论与建议
# 2.基础知识
## 2.1光路设计
光路设计即决定了光线经过光罩时的路径、方向和强度，也就是说，光路设计是制作光罩的关键一步。光路设计可以让光线达到目的地，并保证产品的整体美观、保真、吸引力和耐磨性。下面给出一些常用的光路设计原则：

1. 足够长：光路越长，它的长度也就越长，效率和分散度越高。通常情况下，光路的长度应该达到10米以上。

2. 平行光：不仅要考虑光照条件，而且要考虑同向或者不同向光线。不同向光线可能会被阻碍或影响，所以必须使光线平行。

3. 漏电保存：光路过短或者功率过低，都可能导致漏电。一般来说，光罩的电流应该足够大，能够承受较大的电压，但在适当的时候切断供电即可。

4. 无污染：光路应尽量避免含有有毒物质，比如腐蚀剂、防潮剂、化学品等。

5. 可信任：光罩的辐射来源必须可信任。如果光路由未知的供应商生产，那么很可能存在安全隐患。

6. 可靠：光路的纹路、镜像等因素不能影响光路的准确性和稳定性。

除了光路设计之外，光路还有其他一些重要的原则和要点。比如，宽通道是指光通道的宽度应够宽，能够接收到足够多的光，这样才能够保证光线的畅通和准确；而窄通道意味着光罩只能聚焦到一个小范围，对全视野的光线无法收敛，造成景深模糊。通过调整宽窄通道的大小和形状，可以调整光路的纵横比，提升光路的透明度。

## 2.2光栅布局
光栅布局即光栅的分布方式，也是影响光栅折射率的关键因素。光栅布局通常采用两种方式：角落布局和网格布局。角落布局又称为极地布局，是在光栅中心建立一个环形区域，然后排列多个子区域。这种方法虽然简单易行，但是它缺乏空间利用率，容易出现负荷均衡问题。网格布局即在一个圆形区域内布置若干个同心的小格子，每个格子之间的距离、角度也应相同。这种布局方式使得光栅的折射率分布密集、均匀，便于后续的处理。

## 2.3光栅加工
光栅加工包括表面处理、清洗、切割、打孔、油漆、晒伤处理、硫化处理、微熔胶填充、水洗等。光栅处理的方法很多，这里只讨论其中几种。

1. 表面处理：表面处理是对光栅的表面进行任何必要的修复、研磨、冲洗、烘烤等处理。表面处理应保证光栅的外观看起来很新鲜、光滑、无瑕疵，同时还应保持整体光效率、折射率、透明度。

2. 清洗：清洗的主要目的是去除杂质、防止污渍、增强光效。清洗时应密切注意保证清洗后的光栅没有油污、痕迹。

3. 切割：切割是将光栅分割成固定大小的小片段，方便批量处理。切割时应注意保证切割带来的光效率损失不要太大，否则会造成部分光栅的损坏。

4. 打孔：打孔是将光栅上的孔洞打补丁。打孔时应注意避免打孔扰乱光效、破坏光栅结构。

5. 油漆：油漆可以增加光栅的抗氧化能力，并对光路的阻碍程度进行改善。

6. 晒伤处理：晒伤处理是用晒在水中的螨虫蛋白消毒、稀释并涂抹在光栅上，促进光栅的光效率提升。

7. 硫化处理：硫化处理是将硫化物溶化在光栅表面，改善光栅的折射率。

8. 微熔胶填充：微熔胶填充是一种特殊的方法用于填充光栅表面，使光栅具有良好的折射性和透明度。微熔胶是一种不含有机械结构的软性金属，可以嵌入到光栅表面，使得光栅自身的表面没有固化形变。微熔胶填充可以有效地缓解光栅的缺陷，保障光栅的功能和耐磨性能。

## 2.4电镀技术
电镀是把薄膜电镀在光栅表面上，这样就可以改变光栅的电导性。电镀过程是以光的频率为导向，使晶体上的电子运动脱离晶体，因此，电镀的结果就是一个绝缘材料。电镀可以降低光的折射率，从而提升产品的抗光照能力。电镀技术有两种：直插和悬浮。

1. 直插：直插是在电子烧结晶表面下方电镀，相邻的两层微晶体的电性会发生相互感应，使之具有不同的平行线偏转。

2. 悬浮：悬浮是在光栅表面悬浮一层高层电镀层，用来增强光栅的抗紫外线性能。

电镀的作用范围远远超出了光栅本身，其扩散速度、厚度、稳定性、抗拉强度、抗冻、重力等参数都是其关键。另外，除了光栅本身的结构之外，电镀还会影响微晶体与晶体间的接触模式、晶体的温度变化、晶体的结构变化、晶体的微缺陷产生、微墩效应、环境影响等。因此，光栅制作中，电镀一定要慎重考虑。

## 2.5涂料选择与配比
涂料选取和配比是决定光栅的质量和光效的重要因素。通常来说，不同层级的光栅所需的涂料规格和配比各不相同。涂料的好坏直接影响光栅的整体性能。常用的涂料有绝缘涂料、无机溶剂、无钢化学肥料、海棉等。涂料选取的原则是保护和保护力强，无毒无腐蚀，防止光栅发生缺陷。其次，能够降低光学衍射率和折射率，并提高光栅的反射率。最后，应选择能够适应光路的光色和强度，并且能够维持产品整体的光效。

## 2.6粉末处理方式及优化
光栅粉末处理方式有多种，下面介绍几种常用的：

1. 粉末膨胀：膨胀是一个先期处理手段，其目的在于提高光栅的透明度、抗紫外线性能、防护性能。在光栅粉末膨胀时，应采用增强型碳酸钙烃或者钠化合物，或者镁化合物作为粉末，施加至光栅表面，帮助光栅表面组织成一层薄膜状。此类粉末的好处在于能够保持光栅结构完整性、不产生过度的反射现象，并降低光栅的阻碍效应。

2. 粉末铺盖：铺盖是一种对光栅表面进行纹理印刷的方法。这种方法可以在光栅表面添加多种纹理，从而使光栅表面看起来更绚丽、更美观。这种方法的优点是可以有效减轻光栅的表面积，而且可以充分利用光栅的光效，起到保护作用。

3. 粉末喷涂：喷涂是一种在光栅表面进行显著级别的塑性改良的方法。喷涂涂抹在光栅表面上，能够表现出多种质感特征，如变形、光泽、反射、彩绘等。这种方法的优点在于可以提高光栅的折射率、光效率和透明度，并保持整体的光泽。

4. 粉末切除：切除是一种极端灾难性的手段。切除手段直接摧毁了光栅的结构，严重威胁到了光栅的性能。应谨慎使用切除措施。切除时应该考虑到自己的职业安全。

光栅粉末的优化除了根据不同的光路来选择不同的粉末外，还应在空气中稳定下来，不产生震动，同时避免光的散射。其次，光栅粉末应穿透光栅，力求实现光栅表面呈现一致的颜色、质感、光泽和反射特性。最后，光栅的光效应在电镀之前、之后、以及光栅周围的空气中均要进行测试。如果光效失常，应及时进行修复。

## 2.7其他参数优化
光栅制作中还有许多其他的参数优化，下面列举几个常用的：

1. 选择性放电：选择性放电是一种技术，其目的在于减少光栅的辐射。选择性放电是基于发光二极管效应进行设计的，其过程是在发光二极管中加强电容，使之满足特定需求。由于发光二极管只有部分与光线直线相交，因此选择性放电可以过滤掉与光线垂直的辐射，从而减少光栅的辐射。

2. 液晶层堆叠：液晶层堆叠是一种结构，其目的在于提升光栅的表面抗锈蚀、抗蚀的性能。其方法是在光栅表面堆叠多个不导电的膜层，用以防止蚀掉阳极材料。

3. 液晶板布置：液晶板布置可以提高光栅的散热效率和耐磨性。其方法是将多个液晶板叠加在一起，并在它们之间铺设粘结层，以防止板间的摩擦。

4. 完美镀锡：完美镀锡是一种工艺，其目的在于在保证光栅光效的前提下，最大限度地压缩发光二极管的电流，从而提高光栅的整体性能。完美镀锡的方案首先使用镀锡硬化光栅表面的非粘锌亚丁，然后在整个过程的每一个步骤都进行温度控制。

5. 发光二极管等级：发光二极管的性能在特定工艺条件下会发生变化，因此，在光栅制作中，应选择符合自己的要求的发光二极管。

# 3.结论与建议
光栅制作是一个复杂而又艰难的过程。光路设计、光栅布局、光栅加工、电镀、涂料选择和配比、粉末处理、其他参数优化等，这些基础知识的掌握是制作光栅的关键。但光栅制作仍然是一个不断学习和研究的过程。光罩设计与优化这篇文章只是分享一些我认为可以提高效率的建议，大家有什么想法欢迎评论。
