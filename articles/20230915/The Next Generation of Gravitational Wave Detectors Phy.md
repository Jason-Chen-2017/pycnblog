# 1.简介
  

近年来，利用物理学中广义相对论效应所引起的量子对称性将引力波信号转化为一种理想模型，即所谓的“钟摆效应”。物质存在于相互作用下，量子效应使它们以超光速运动。这一理论可以解释宇宙中的许多现象，如飞行、旋涡、星系爆炸等。在这种观念的指导下，研究人员开发了用于检测宇宙传播粒子流体的探测器。由于这种新型探测方法的出现，各种速度的粒子都可以在低分辨率下被观测到。

目前，已有三种主要的 gravitational wave detectors（GW detector）：探测器、红外球探测器、中微子探测器。其中探测器的应用范围最为广泛。探测器由无源天线和电路板组成，它通过微透镜或双向偏振光栅照射高能粒子，并反射其信号到计算机。由于具有较低的灵敏度和噪声，探测器只能观测低能粒子的变化。

红外球探测器能够直接探测比电磁粒子更强大的能量粒子，如光子和弱粒子。这类能量粒子的形态往往与星系或其它宇宙实体发生关联。红外球探测器通常用可见光筛选低能光线，通过受光面上的反射或吸收，从而获得有关其状态的信息。由于能够探测更强大的能量粒子，红外球探测器可以观测到不同速度的粒子。但是，其灵敏度低且需要专业化设备。

中微子探测器不受磁场的干扰，只需在某个频率范围内检测到中微子即可。但中微子的数量太少，只能在对其进行探测时才会发出信号。由于只能在极小的时间段内捕捉到信号，因此无法检测到低能粒子的动态过程。由于中微子能量太高，在没有探测装置的情况下，人们只能通过宇宙学实验来估计中微子的速度。

# 2. 基本概念术语说明
# 粒子
粒子的相对论性质包括质量、能量、动量、能动量、四矢量和速度。质子以自己的速度（676km/s）向四周移动，被称为质子束，构成质子云。

粒子在相对论中具有等概率的可能性。相对于固体和空气，宇宙中的粒子拥有特别大的能量和大小。这些粒子可以从很远的地方源源不断地向周围空间输送能量。

大量微观粒子从宇宙中诞生，随着宇宙的扩大，宇宙中的粒子和它们之间的相互作用将以新的方式影响整个宇宙。

# 波
物理波是粒子及其波函数组成的量纲。在相对论定律的描述下，粒子拥有均匀的能量分布，并且每一种粒子的速度都是恒定的。因此，粒子产生的波也具有均匀的大小。波又分为两种类型：频率波和时空波。

频率波是能量的连续变化。它的大小随时间和位置而变化。时空波则是由空间和时间这两个维度上波函数之间的相互关系决定的。时空波随时间的演进而减弱，其大小也随之减弱。

# 比特
位元技术，英文为 bit。它是二进制数字系统中的一个元素，也就是说，它只有两个值——0 或 1，分别表示 “短” 和 “长”，或者“开” 和 “关” 。信息处理的基础就是利用位元来组织和储存数据。

在 Gravitational Waves 中，利用了 Gravitational Field 产生了 Gravitational Waves ，并通过离散存储芯片中的脉冲点阵输出信号。脉冲宽度代表了信号强度，即 Gravitational Waves 的强度。

# 量子
量子（quantum）一词的古希腊词源于希腊语 Ἀκτινομαρτία （Ὄκτις το μάρτυς）。量子的提出者是苏格拉底，他说：“在我看来，所谓的量子就是现有的事物的最小单元，是永远不能被拆开的，不过它的样子却像微小的颗粒一样。”

物理学家用量子来研究奇异性和无法观察到的现象。现代物理学里用到的大部分理论，比如量子力学、波动力学、相对论，以及多种材料的结构和电子 structure ，都建立在某些基本假设之上。量子理论和物理实际之间存在着巨大的鸿沟，这就需要跨越时空距离才能真正理解和测量。
