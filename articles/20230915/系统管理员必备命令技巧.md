# 1.简介
  

系统管理员是企业运行运维中必不可少的一环，掌握好系统管理常用命令的技巧可以让你高效、精准地管理服务器、数据库、网络设备等资源，提升工作效率，降低IT投入成本，并保障业务连续性。所以，了解系统管理员必备命令技巧是一件非常重要的事情！下面我将分享一些最常用的系统管理命令。

# 2.概览
## 2.1 Linux/Unix 命令
### ls命令
ls 命令用于显示目录下的内容。
```shell
ls [选项] 文件或目录名...
```
选项：
-a或--all：列出所有文件，包括隐藏文件。
-l或--long：以长数据串形式输出。
-d或--directory：仅列出目录名称。
-h或--human-readable：以易读方式容量显示。

### cd命令
cd 命令用于切换当前工作目录至指定目录。
```shell
cd [选项] directory
```
选项：
-p或--physical：表示更改工作路径到用户实际所在目录。

### pwd命令
pwd 命令用于显示当前工作目录。

### mkdir命令
mkdir 命令用于创建新目录。
```shell
mkdir [-p] 文件夹名称...
```
选项：
-m<权限>：设置目录权限。
-p或--parents：递归创建目录。

### touch命令
touch 命令用于修改文件的访问及内容的时间戳。
```shell
touch [-acdmtu] filename...
```
选项：
-a：只更改访问时间。
-c：不要创建不存在的文件。
-d<日期时间>或--date=<日期时间>：直接指定日期时间。
-m：只更正修改时间。
-t<日期时间>或--time=<日期时间>：同时指定存取和更改时间。
-r：<参考文件或者目录>：把指定文件或目录的日期时间设成和参考文件或目录的日期时间相同。
-u：只更正拥有者和群组。

### rm命令
rm 命令用于删除一个目录中的一个或多个文件或目录。
```shell
rm [-rf] 文件名...
```
选项：
-i或--interactive：在删除前询问确认。
-f或--force：忽略不存在的文件，从不给出提示。
-r或--recursive：递归删除目录。
-R或--recursive：递归删除目录。

### mv命令
mv 命令用于移动或重命名文件或目录。
```shell
mv [-fiu] source dest
```
选项：
-f或--force：覆盖同名文件或目录，不对非空目录进行删除操作。
-i或--interactive：若目标文件 (dest) 存在时，询问是否覆盖。
-n或--no-clobber：不覆盖已存在的目标文件。
-u或--update：对源文件进行更新，若 destination 比较旧则更新之。

### cp命令
cp 命令用于复制文件或目录。
```shell
cp [-adfilprsu] source dest
```
选项：
-a或--archive：复制文件或目录及其内含的文件。
-d：复制成为符号链接。
-f：强行复制文件或目录，不覆盖已经存在的目标文件。
-i或--interactive：若目标文件 (dest) 存在时，询问是否覆盖。
-l：复制成为硬连接，软连接可通过 ln 命令实现。
-p：保留源文件属性。
-r或--recursive：递归复制目录。
-s或--symbolic-link：复制成为符号链接。
-u或--update：复制仅当源文件比目标文件新或者目标文件不存在时。

### cat命令
cat 命令用于查看文件的内容。
```shell
cat [选项] 文件名...
```
选项：
-b或--number-nonblank：对非空输出行编号，由 1 开始对齐。
-n或--number：对所有输出行编号，由 1 开始对齐。

### head命令
head 命令用于显示档案的开头至标准输出里的内容。
```shell
head [选项] 文件名...
```
选项：
-c<缓冲区大小>或--bytes=<缓冲区大小>：指定读取的字节数。
-n<行数>或--lines=<行数>：指定显示行数。

### tail命令
tail 命令用于显示档案最后一部分的内容。
```shell
tail [选项] 文件名...
```
选项：
-c<缓冲区大小>或--bytes=<缓冲区大小>：指定读取的字节数。
-n<行数>或--lines=<行数>：指定显示行数。
-f或--follow：持续侦测所选文件，直到追加数据。

### grep命令
grep 命令用于查找匹配文本模式的字符串。
```shell
grep [选项] pattern file
```
选项：
-A<显示行数>或--after-context=<显示行数>：显示匹配行之后的内容。
-B<显示行数>或--before-context=<显示行数>：显示匹配行之前的内容。
-C<显示行数>或--context=<显示行数>：显示匹配行之前后各自的内容。
-E或--extended-regexp：支持扩展正则表达式。
-F或--fixed-strings：模式按照一般意义上对待（除了反斜杠）。
-G或--basic-regexp：支持基本正则表达式。
-I或--ignore-case：忽略字符大小写的不同。
-w或--word-regexp：只显示全词匹配的行。
-x或--line-regexp：只显示整个线索匹配的行。
-o：显示搜索到的匹配部分。

### find命令
find 命令用于在指定范围内查找文件。
```shell
find [搜索路径参数] [查询条件] 
```
选项：
-name：根据文件名查找文件。
-iname：根据文件名查找文件，不区分大小写。
-type：根据类型查找文件。
-size：根据文件大小查找文件。
-user：根据属主查找文件。
-group：根据属组查找文件。
-perm：根据权限查找文件。
-mtime：根据最近修改时间查找文件。
-atime：根据最近访问时间查找文件。
-ctime：根据文件状态变更时间查找文件。

## 2.2 Windows 命令
### dir命令
dir 命令用于显示目录中的文件及子目录。
```shell
dir [文件名] [选项]
```
选项：
-a：显示所有文件。
-b：显示批处理文件列表。
-d：显示只有驱动器号的文件信息。
-h：以易于理解的格式显示文件大小。
-t：按文件上次修改时间排序。

### type命令
type 命令用于显示指定命令的定义。
```shell
type cmd
```

### copy命令
copy 命令用于复制文件或将指定内容粘贴到剪切板中。
```shell
copy [选项] 源文件(夹) 目标文件(夹)
```
选项：
-y：不询问任何问题。

### del命令
del 命令用于删除文件。
```shell
del [选项] 文件名...
```
选项：
-q：安静模式，不提示删除确认信息。
-s：删除文件夹时，若文件夹为空，则删除文件夹。

### ren命令
ren 命令用于重命名文件或目录。
```shell
ren [选项] 源文件 更改后的文件名
```
选项：
-n：仅显示信息而不真正更改。

### attrib命令
attrib 命令用于设置或显示文件属性。
```shell
attrib [选项] 文件名...
```
选项：
-a：清除文件上所有属性。
-s：锁定文件。
-h：隐藏文件。

### start命令
start 命令用于启动指定的应用程序或文件。
```shell
start /[参数] 文件名 [参数]
```
参数：
-w：最大化窗口。
-h：最小化窗口。
