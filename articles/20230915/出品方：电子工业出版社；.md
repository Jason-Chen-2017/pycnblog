# 1.简介
  

这个项目是一个基于物联网的疾病预测系统。通过收集不同设备的数据并进行处理，可以准确预测患者是否患上了某种疾病。首先，我们需要理解一下什么叫做物联网(IoT)。顾名思义，它是指具有网络连接的物体，可以自动接收、传输和处理信息。也就是说，我们的这个疾病预测系统就是一种物联网应用。

物联网的架构由一个个节点组成，这些节点之间相互通信、共享数据。每台设备可以采集到各种传感器的信息，如温度、湿度、压力、光照强度等，这些信息被汇聚到中心服务器中，由算法分析处理后输出结果。最终，根据得到的结果及时向用户反馈，提醒用户自己需要注意的事项。比如，若患者高烧或胸闷，则系统会发出警报通知；若患者咳嗽不清，则会向医生提醒疼痛。因此，这个系统可以帮助我们在最短的时间内了解自己的健康状况。

# 2.基本概念术语说明
## 2.1 概念
物联网（Internet of Things，IoT）是一系列利用现代通信技术将物理世界各个设备互联互通的研究领域。它是一种技术集合，包括硬件、软件、服务、应用平台和标准化组织。其目的是促进物理世界和数字世界的融合，提升整个生活环境的智能化水平。物联网主要涉及以下几个方面：
- **数据采集**：物联网能够采集到来自各类设备的数据，如温度、压力、红外线强度、GPS坐标、呼吸信号、声音强度等，这些数据经过处理后，可以得出用户的个人情况或者社会动态等价于人的潜在价值。
- **数据存储与计算**：物联网中的数据既可以在本地数据库中保存，也可以实时的发送到云端，然后进行分布式计算分析，生成相应的决策结果。同时，物联网还可以对数据进行实时监控，从而发现异常行为，提前预警。
- **物联网终端控制**：物联网终端设备可以通过无线方式进行控制，可以实现各种功能，如智能开关、智能灯泡、智能插座、智能马桶盖、智能洗衣机等。可以使终端设备达到远程监控、安防、智慧城市、智能农业等多种用途。
- **场景联动**：物联网还可以将物理世界的变化同步到数字空间中，实现物联网设备之间的协同工作，创造新的智能应用场景，促进社会经济发展。比如，物联网可以帮助保险公司管理客户的数据，建立保单、客户关系管理系统；可以让监狱管理人员实时掌握刑侦犯罪嫌疑人在现场的动态；可以连接城市公共设施、智能家电，让居民享受更便利的生活。

## 2.2 术语表
下面列举一些与物联网相关的重要术语。
- **设备(Device)：**通常指的是用有线或无线方式连接在计算机网络上的硬件、软件和传感器的集合。如智能手机、电子计量箱、环境监测装置、空调扇、冰箱、厨房电器等。
- **网关(Gateway)：**网关是一个物理实体或网络设备，用来连接其他设备并提供安全可靠的连接和通信功能。网关通常包括连接设备和网管系统之间通信、控制、监视和转换数据的功能。
- **传感器(Sensor)：**传感器是一种检测物质流动、变换和传递的仪器或装置。物理世界的物体、人、活动、气候条件都可以通过传感器检测并记录下来。如温度传感器、压力传感器、加速度传感器、光电转矩传感器等。
- **协议栈(Protocol Stack)：**协议栈是指网络协议的层次结构，即网络通信的基础架构。分为网络层、传输层、应用层。其中应用层定义网络应用程序使用的协议，例如HTTP、FTP、DNS、SMTP等；传输层提供端到端的通信服务，例如TCP/IP协议；网络层负责底层数据包交换，主要解决网络路由、寻址和寻找路径的问题。
- **云计算(Cloud Computing)：**云计算是一种利用网络将服务器、存储、应用和服务的能力以按需付费的方式提供给用户的一种计算模式。云计算可以帮助企业降低内部支出，提升效率，实现业务快速增长和规模的扩展。
- **MQTT(Message Queuing Telemetry Transport)：**MQTT是物联网通信协议。它是轻量级的发布订阅消息协议，适用于物联网设备之间低带宽、低功耗的通信。MQTT协议支持设备之间、网关与云端的双向通信。
- **RESTful API(Representational State Transfer Application Programming Interface)**：RESTful API是一种基于HTTP协议的应用编程接口规范。它定义了客户端如何与服务端进行通信，以及服务器返回的资源表示形式。RESTful API通过URI标识资源，通过HTTP方法实现资源的创建、读取、更新和删除操作。
