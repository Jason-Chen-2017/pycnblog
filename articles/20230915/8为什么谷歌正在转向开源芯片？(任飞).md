# 1.简介
  

Google（一家美国科技公司）自称拥有“开创性的、创新性的、先进的、先进制造技术”，其硬件业务主要集中在手机领域。从去年开始，Google 正式将其芯片部门“Waymo”的开发工作收归开源，并计划在今年夏天上线。
作为一家非常有影响力的科技巨头，Google 自成立以来，其基础设施、应用软件、服务等方面都处于世界领先地位。但随着科技的不断发展，无论是信息技术还是硬件领域，都会出现一些问题。这些问题会成为科技界的“难题”，进而引起一些社会的思考和争议。2019 年，英国皇家学会就发布了一份调查报告《Google Opening Up its Ecosystem》，谈到 Google 在 2017 年对内部工具的收购后，“Google 不再专注于硬件生态系统的设计和研发”，转而致力于开源生态系统的建设。当然，这还只是简单的说法。如何看待开源和硬件结合的形势，是本文需要探讨的核心问题之一。
作为一家拥有全球影响力的科技企业，Google 有能力做出改变，使开源生态系统蓬勃发展。相对于闭源的商用硬件产品来说，开源的优势显现得更加突出。其主要原因如下：
- 更加灵活的组合方式：开源软件可以按照用户需求，自由选择其所需组件组合，实现更多样化的功能；
- 更可靠的生命周期管理：开源软件能够提供一个长期支持的版本发布方式，有效防止软件漏洞的扩散；
- 更广泛的参与者群体：开源社区成员遍及全球各地，能够吸纳众多开发者参与项目的开发，促进全球开发者之间的交流和协作。
此外，还有以下几点值得关注：
- 大量依赖开源软件的硬件设备正在快速增长：据彭博（Bloomberg）统计，截至 2020 年，超过 80% 的主板、服务器和网卡都基于开源软件构建；
- Google 即将推出首个开源芯片：去年，Google Waymo 开源开发平台获得美国计算机软件协会颁发的第五级认证，这也是其开源生态系统崛起的重要一步；
- Google 与 Linux Foundation 签署合作协议：去年，Google 和 Linux Foundation 共同发起了一项名为“Open Compute Project”（OCP）的合作计划，旨在将开源领域的关键技术沉淀成统一规范。该计划将包括 OpenCL、OPAE、Rockchip、Xilinx 等关键开源组件的标准化、编制成套件和开放源码。
以上四点，可以看出 Google 正在将开源生态系统用于硬件领域的开发，它认为，“开源让创意无限接近”、“最佳实践解决方案”。虽然如此，Google 也要看到开源生态系统可能产生的负面影响，比如软硬件兼容性问题、开源软件不可信任等等。因此，希望通过本文，能够为读者带来启发和思考，帮助读者更好地理解开源和硬件的结合。
# 2.基本概念术语说明
## 2.1 什么是开源
开源是一种公共许可协议，允许被他人利用、修改、分发某些软件源代码。它鼓励个人、团体、机构以及其他组织以共享知识的方式，为项目开发提供更多的透明度和控制。一般情况下，开源软件由许可证保护，允许任何人阅读、修改或共享源代码，但要求保留版权信息，以避免侵犯专利权。开源的目标是实现共识，而不是盗版。
## 2.2 什么是硬件开源
硬件开源是指把硬件设计、制造、验证、测试、生产环节中涉及到的所有环节，都以开源的方式进行。这样，从而方便第三方设计人员、制造商、开发者、测试人员、人员、国家等各方参与，共同推进硬件的发展。目前，全球有超过 200 个国家的多个开源生态系统组织正在支持硬件开源，有些组织甚至已经建立起自己的生态系统，例如 Linux Foundation、Freedom Ecosystem、Open Compute Project 等。其中，Linux Foundation 是最具代表性的开源硬件组织，它在支持软件、服务、教育等领域，也尊重、捐赠、共享硬件设计工具。
## 2.3 什么是华为开源芯片
华为开源芯片，是指华为在其关于芯片开源的政策文件中提出的开源行动。华为的开源行动，除了将其硬件作为开源资源供人们使用之外，更重要的是其开源技术栈——鸿蒙 OS（HarmonyOS），这是一种基于分布式系统软件架构的开源操作系统，可为智慧物联网终端设备提供运行环境。华为基于鸿蒙 OS，在开源社区推出了鸿蒙 HarmonyOS 操作系统，其中包含了 OpenHarmony 内核、OpenGPU 图形库、OpenUI 框架等多个开源组件。2021 年，华为推出了 HarmonyOS Mate 40 Pro 评测版，是首款采用基于鸿蒙 OS 的一体机。
