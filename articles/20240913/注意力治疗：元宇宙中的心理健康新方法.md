### 算法编程题与解析

#### 5. 编写一个算法，计算用户在注意力训练游戏中完成任务的平均时间。

**算法：**
- 记录用户完成每个任务的时间。
- 计算所有任务时间的平均值。

**代码示例（Python）：**
```python
def calculate_average_time(task_times):
    total_time = sum(task_times)
    average_time = total_time / len(task_times)
    return average_time

task_times = [5, 7, 3, 8, 2]
average_time = calculate_average_time(task_times)
print("平均完成任务的时间：", average_time)
```

#### 6. 编写一个算法，根据用户在注意力训练游戏中的表现，生成一个个性化的训练计划。

**算法：**
- 计算用户在游戏中的成功率和平均完成时间。
- 根据成功率和平均完成时间，为用户生成一个个性化的训练计划。

**代码示例（Python）：**
```python
def generate_training_plan(success_rate, average_time):
    if success_rate > 0.8 and average_time < 5:
        plan = "高级训练计划"
    elif success_rate > 0.5 and average_time < 10:
        plan = "中级训练计划"
    else:
        plan = "初级训练计划"
    return plan

success_rate = 0.7
average_time = 7
training_plan = generate_training_plan(success_rate, average_time)
print("个性化训练计划：", training_plan)
```
