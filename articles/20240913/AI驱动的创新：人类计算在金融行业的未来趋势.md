##### 1. 矩阵乘法优化

**题目：** 给定两个矩阵，编写一个高效的矩阵乘法算法。

**答案：**

```python
def matrix_multiply(A, B):
    rows_A, cols_A = len(A), len(A[0])
    rows_B, cols_B = len(B), len(B[0])
    if cols_A != rows_B:
        raise ValueError("矩阵维度不匹配")

    result = [[0 for _ in range(cols_B)] for _ in range(rows_A)]

    for i in range(rows_A):
        for j in range(cols_B):
            for k in range(cols_A):
                result[i][j] += A[i][k] * B[k][j]

    return result
```

**解析：**

- **检查矩阵维度匹配：** 确保两个矩阵可以相乘。
- **初始化结果矩阵：** 遍历每个元素，初始化为0。
- **三重循环：** 计算每个结果元素，通过矩阵相乘公式完成。

##### 2. 货币兑换问题

**题目：** 给定一组货币兑换汇率，编写一个程序计算从一种货币兑换到另一种货币的最优汇率。

**答案：**

```python
def optimal_exchange_rate(rates):
    start_currency = list(rates.keys())[0]
    current_currency = start_currency
    max_value = 1

    while current_currency != start_currency:
        next_currency = max(rates[current_currency], key=rates[current_currency].get)
        current_currency = next_currency
        max_value *= rates[start_currency][current_currency]

    return max_value
```

**解析：**

- **初始设置：** 设定起点货币和当前货币。
- **循环：** 依次寻找下一个最优货币进行兑换，直到回到起点货币。
- **计算最优汇率：** 持续计算兑换后的价值，直到回到起点货币。
