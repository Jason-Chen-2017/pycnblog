**题目：** 编写一个函数，对用户行为数据进行预处理，包括数据清洗、去重、排序等操作。

```python
def preprocess_user_data(data):
    # 数据清洗
    cleaned_data = [d for d in data if d['valid']]
    # 去重
    unique_data = list(set(cleaned_data))
    # 排序
    sorted_data = sorted(unique_data, key=lambda x: x['timestamp'])
    return sorted_data
```

##### 2. 用户反馈分类与统计

**题目：** 编写一个函数，对用户反馈进行分类和统计，输出每个类别的反馈数量。

**答案：** 以下是一个简单的用户反馈分类与统计函数：

```python
def classify_and_count_feedback(feedback_list):
    categories = {'功能问题': 0, '内容质量': 0, '服务问题': 0}
    for feedback in feedback_list:
        category = feedback['category']
        categories[category] += 1
    return categories
```
