### 4. 生活质量评估算法编程题

**题目：** 请编写一个算法，计算一个人的生活质量得分。假设生活质量得分由健康指标、教育指标、经济指标、社会支持指标和主观幸福感组成，分别占总分的30%、20%、20%、20%和10%。

**答案：** 

```python
def calculate_life_quality_score(health_score, education_score, economic_score, social_support_score, happiness_score):
    weight = [0.3, 0.2, 0.2, 0.2, 0.1]
    score = (health_score * weight[0] + education_score * weight[1] + economic_score * weight[2] + social_support_score * weight[3] + happiness_score * weight[4])
    return score

# 示例
health_score = 80
education_score = 70
economic_score = 90
social_support_score = 85
happiness_score = 75

life_quality_score = calculate_life_quality_score(health_score, education_score, economic_score, social_support_score, happiness_score)
print("生活质量得分：", life_quality_score)
```

**解析：** 该算法通过加权平均法计算生活质量得分，各指标的权重根据实际情况设定。实际应用中，可以结合更多数据和算法，提高评估的准确性。
