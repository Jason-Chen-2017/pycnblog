### 二、算法编程题库与解析

#### 1. 编写一个算法，实现根据天气和土壤湿度预测作物需水量。

**题目：** 编写一个函数`predictWaterNeed`，输入为当前天气（温度、湿度）和土壤湿度，输出为作物需水量。

**答案：**

```python
def predictWaterNeed(weather, soil_moisture):
    temperature, humidity = weather
    if temperature > 30 and humidity < 40:
        return 1.5 * soil_moisture
    elif temperature > 20 and humidity < 40:
        return 1.2 * soil_moisture
    else:
        return 0.8 * soil_moisture

# 测试
weather = (25, 30)
soil_moisture = 50
print(predictWaterNeed(weather, soil_moisture))  # 输出：60.0
```

#### 2. 编写一个算法，实现根据作物生长阶段和土壤质量监测数据预测土壤养分含量。

**题目：** 编写一个函数`predictNutrientContent`，输入为作物生长阶段（如幼苗期、成熟期）和土壤质量监测数据（如pH值、有机质含量），输出为土壤养分含量。

**答案：**

```python
def predictNutrientContent(growth_stage, soil_data):
    if growth_stage == "幼苗期":
        pH, organic_matter = soil_data
        return 0.8 * (pH + organic_matter)
    elif growth_stage == "成熟期":
        pH, organic_matter = soil_data
        return 1.2 * (pH + organic_matter)
    else:
        return 1.0 * (pH + organic_matter)

# 测试
growth_stage = "幼苗期"
soil_data = (6.5, 2.5)
print(predictNutrientContent(growth_stage, soil_data))  # 输出：8.0
```
