### 3. 算法编程题库

#### 3.1 判断一个字符串是否为回文

**题目：** 编写一个函数，判断一个字符串是否为回文。

**答案：**

```python
def is_palindrome(s):
    return s == s[::-1]

# 测试
print(is_palindrome("hello")) # 输出：False
print(is_palindrome("level")) # 输出：True
```

#### 3.2 求两个有序数组合并

**题目：** 编写一个函数，求两个有序数组合并。

**答案：**

```python
def merge_sorted_arrays(arr1, arr2):
    result = []
    i, j = 0, 0
    while i < len(arr1) and j < len(arr2):
        if arr1[i] < arr2[j]:
            result.append(arr1[i])
            i += 1
        else:
            result.append(arr2[j])
            j += 1
    result.extend(arr1[i:])
    result.extend(arr2[j:])
    return result

# 测试
print(merge_sorted_arrays([1, 3, 5], [2, 4, 6])) # 输出：[1, 2, 3, 4, 5, 6]
```

#### 4.2 求两个有序数组合并

**解析：**

这个函数使用两个指针分别遍历两个有序数组，比较当前指针指向的元素大小，将较小的元素添加到结果数组中。遍历结束后，将剩余的元素追加到结果数组中。

**源代码实例：**

```python
def merge_sorted_arrays(arr1, arr2):
    result = []
    i, j = 0, 0
    while i < len(arr1) and j < len(arr2):
        if arr1[i] < arr2[j]:
            result.append(arr1[i])
            i += 1
        else:
            result.append(arr2[j])
            j += 1
    result.extend(arr1[i:])
    result.extend(arr2[j:])
    return result

# 测试
print(merge_sorted_arrays([1, 3, 5], [2, 4, 6])) # 输出：[1, 2, 3, 4, 5, 6]
```
