### 3. 如何设计一个高并发的分布式锁？

**题目：** 使用 Go 语言实现一个高并发的分布式锁，并简要解释其工作原理。

**答案：**

```go
package main

import (
    "sync"
    "time"
)

type DistributedLock struct {
    sync.Mutex
    ch chan struct{}
}

func NewDistributedLock() *DistributedLock {
    return &DistributedLock{
        ch: make(chan struct{}, 1),
    }
}

func (l *DistributedLock) Lock() {
    l.ch <- struct{}{}
    l.Mutex.Lock()
}

func (l *DistributedLock) Unlock() {
    l.Mutex.Unlock()
    <-l.ch
}

func main() {
    var wg sync.WaitGroup
    lock := NewDistributedLock()
    for i := 0; i < 10; i++ {
        wg.Add(1)
        go func() {
            defer wg.Done()
            lock.Lock()
            time.Sleep(time.Millisecond * 100)
            lock.Unlock()
        }()
    }
    wg.Wait()
}
```

**解析：** 介绍分布式锁的基本原理，如互斥锁、饥饿避免等，并详细解释实现过程中的关键代码，如 `Lock` 和 `Unlock` 方法。
