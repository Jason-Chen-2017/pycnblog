**答案：**

```go
func reverseString(s string) string {
    runes := []rune(s)
    n := len(runes)
    for i := 0; i < n/2; i++ {
        runes[i], runes[n-1-i] = runes[n-1-i], runes[i]
    }
    return string(runes)
}
```

**解析：** 本题考察了字符串反转的基本算法。通过将字符串转换为 rune 数组，然后交换数组中前后的元素，实现字符串的反转。该算法的时间复杂度为 O(n/2)，空间复杂度为 O(n)。

```go
type UserActivity struct {
    UserID     int
    ActivityID int
    Time       time.Time
}

type UserActivitySystem struct {
    userActivities []UserActivity
}

func (uas *UserActivitySystem) AddActivity(userID, activityID int, time time.Time) {
    uas.userActivities = append(uas.userActivities, UserActivity{UserID: userID, ActivityID: activityID, Time: time})
    sort.Slice(uas.userActivities, func(i, j int) bool {
        return uas.userActivities[i].Time.After(uas.userActivities[j].Time)
    })
}

func (uas *UserActivitySystem) GetRecentActiveUsers(limit int) []int {
    recentActiveUsers := make([]int, 0, limit)
    for i := 0; i < len(uas.userActivities) && len(recentActiveUsers) < limit; i++ {
        recentActiveUsers = append(recentActiveUsers, uas.userActivities[i].UserID)
    }
    return recentActiveUsers
}
```

**解析：** 本题考察了系统设计能力和优化思路。通过维护一个有序的切片来存储用户活动，实现快速查询最近活跃用户的功能。该系统的时间复杂度为 O(n)，空间复杂度为 O(n)。
