##### 2. 注意力分配算法（Attention Allocation Algorithm）

注意力分配算法是注意力机制的核心部分，负责计算注意力权重，确定不同输入信息的重要性。以下是一道关于注意力分配算法的编程题：

**题目：** 实现一个基于加权的注意力分配算法，给定输入序列和权重，输出注意力分配结果。

**答案：** 可以使用矩阵乘法实现注意力分配算法。首先计算输入序列和权重的内积，然后对内积进行softmax变换，得到注意力分配权重。

```python
import numpy as np

def attentionallocation(inputs, weights):
    scores = np.dot(inputs, weights)
    probabilities = np.exp(scores) / np.sum(np.exp(scores))
    return probabilities
```
