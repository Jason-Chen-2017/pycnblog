#### 1. 蛋白质序列的比对算法

**题目：** 请解释生物信息学中常用的蛋白质序列比对算法，并给出一个示例。

**答案：** 常用的蛋白质序列比对算法包括局部比对（如Smith-Waterman算法）和全局比对（如Needleman-Wunsch算法）。这些算法用于计算两个蛋白质序列之间的相似度。

def smith_waterman(seq1, seq2):
    # 创建一个动态规划矩阵
    matrix = [[0 for _ in range(len(seq2) + 1)] for _ in range(len(seq1) + 1)]

    # 初始化矩阵
    for i in range(len(seq1) + 1):
        matrix[i][0] = 0
    for j in range(len(seq2) + 1):
        matrix[0][j] = 0

    # 填充矩阵
    for i in range(1, len(seq1) + 1):
        for j in range(1, len(seq2) + 1):
            match = matrix[i - 1][j - 1] + 1 if seq1[i - 1] == seq2[j - 1] else 0
            delete = matrix[i - 1][j] - 1
            insert = matrix[i][j - 1] - 1
            matrix[i][j] = max(0, match, delete, insert)

    # 返回最大得分及其位置
    return matrix[-1][-1], traceback(matrix, seq1, seq2)
