#### 算法编程题库与答案解析

##### 1. LeetCode 上的「最长公共子序列」

**题目：** 给定两个字符串 `text1` 和 `text2`，找出它们的最长公共子序列。

**答案：** 可以使用动态规划方法求解。

```python
def longest_common_subsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```

**解析：** 通过构建一个动态规划表格 `dp`，记录两个字符串的公共子序列长度，最终得到最长公共子序列的长度。

##### 2. LeetCode 上的「打家劫舍」

**题目：** 你是一个盗贼，打算从一家很长的小区里偷窃物品。每隔两间房子会有一间装有监控摄像头。计算你最多可以偷窃的物品的价值。

**答案：** 可以使用动态规划方法求解。

```python
def rob(nums):
    if not nums:
        return 0
    if len(nums) <= 2:
        return max(nums)

    dp = [0] * len(nums)
    dp[0] = nums[0]
    dp[1] = max(nums[0], nums[1])

    for i in range(2, len(nums)):
        dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])

    return dp[-1]
```

**解析：** 通过构建一个动态规划表格 `dp`，记录每个位置可以偷窃的物品价值，最终得到最大偷窃价值。
