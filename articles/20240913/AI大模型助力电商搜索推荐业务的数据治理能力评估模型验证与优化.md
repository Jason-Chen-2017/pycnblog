#### 4. 编写一个函数，用于计算电商搜索推荐业务中用户点击率。

**答案：** 

```python
def calculate_click_rate(clicks, total_shows):
    if total_shows == 0:
        return 0
    return clicks / total_shows
```

**解析：** 该函数计算用户点击率，点击率定义为点击次数除以展示次数。

#### 6. 编写一个函数，用于评估电商搜索推荐业务的商品推荐效果。

**答案：**

```python
def calculate_recommendation_effectiveness(ratings, recommended_products):
    total_ratings = sum(ratings.values())
    recommended_ratings = sum(ratings[product] for product in recommended_products)
    if total_ratings == 0:
        return 0
    return recommended_ratings / total_ratings
```

**解析：** 该函数计算商品推荐效果，效果定义为推荐商品的评分总和除以所有商品的评分总和。
