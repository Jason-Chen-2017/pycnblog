**1. 编写一个函数，计算给定日期在一年中的天数。**

**答案：**

```python
def day_of_year(year, month, day):
    days_in_month = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
    if year % 4 == 0 and (year % 100 != 0 or year % 400 == 0):
        days_in_month[1] = 29
    total_days = sum(days_in_month[:month-1]) + day
    return total_days
```

**2. 编写一个函数，判断一个年份是否是闰年。**

**答案：**

```python
def is_leap_year(year):
    return year % 4 == 0 and (year % 100 != 0 or year % 400 == 0)
```

**3. 编写一个函数，计算两个日期之间的天数差。**

**答案：**

```python
from datetime import datetime

def days_between_dates(start_date, end_date):
    start = datetime.strptime(start_date, "%Y-%m-%d")
    end = datetime.strptime(end_date, "%Y-%m-%d")
    delta = end - start
    return delta.days
```

**4. 编写一个函数，计算一个给定区间的所有闰年。**

**答案：**

```python
def leap_years_in_range(start_year, end_year):
    leap_years = []
    for year in range(start_year, end_year+1):
        if is_leap_year(year):
            leap_years.append(year)
    return leap_years
```

**6. 编写一个函数，计算一个给定日期的下一个星期几。**

**答案：**

```python
def next_weekday(date):
    weekdays = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"]
    day_of_week = weekdays[date.weekday()]
    days_to_next_weekday = weekdays.index(day_of_week) + 1
    next_date = date + datetime.timedelta(days=days_to_next_weekday)
    return next_date.strftime("%Y-%m-%d")
```

**7. 编写一个函数，计算一个给定日期的下一个工作日。**

**答案：**

```python
def next_business_day(date):
    days_to_next_business_day = 1
    while True:
        next_date = date + datetime.timedelta(days=days_to_next_business_day)
        if next_date.weekday() < 5:  # 0-4 对应周一到周五
            break
        days_to_next_business_day += 1
    return next_date.strftime("%Y-%m-%d")
```

**8. 编写一个函数，计算一个给定日期的过去第 N 个星期几。**

**答案：**

```python
def past_weekday(date, n):
    weekdays = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"]
    day_of_week = weekdays[date.weekday()]
    days_to_past_weekday = weekdays.index(day_of_week) - n
    past_date = date - datetime.timedelta(days=days_to_past_weekday)
    return past_date.strftime("%Y-%m-%d")
```

def days_since_today(date):
