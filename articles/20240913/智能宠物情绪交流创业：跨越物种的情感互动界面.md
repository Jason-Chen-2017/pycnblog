#### 4. 编写一个函数，用于根据宠物的年龄、体重和品种，预测宠物的平均寿命。

**题目描述：** 宠物的平均寿命受到年龄、体重和品种等多种因素的影响。编写一个函数，接收宠物的年龄、体重和品种作为输入，输出预测的宠物平均寿命。

**答案：**

```python
def predict_pet_life(age, weight, breed):
    # 根据年龄、体重和品种预测宠物寿命
    # 这里使用简单的线性回归模型作为示例
    life_expectancy = {
        'dog': 12.5,
        'cat': 15.0,
        'fish': 5.0
    }
    
    # 根据品种确定基础寿命
    base_life = life_expectancy.get(breed, 10)
    
    # 根据体重进行调整
    weight_adjustment = weight * 0.1
    
    # 根据年龄进行调整
    age_adjustment = age * 0.05
    
    # 计算最终寿命
    life_expectancy = base_life + weight_adjustment - age_adjustment
    
    return max(life_expectancy, 0)  # 寿命不能为负数

# 示例
print(predict_pet_life(5, 20, 'dog'))  # 输出：12.5
```

def recommend_training_plan(mood):
    # 根据宠物情绪推荐训练计划
    plans = {
        'happy': '继续当前训练计划，适当增加玩耍时间',

    }
    
    return plans.get(mood, '当前状态良好，维持现有训练计划')

# 示例
