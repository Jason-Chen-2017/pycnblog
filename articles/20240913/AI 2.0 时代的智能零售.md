### 2. 基于协同过滤的推荐算法

**题目：** 请实现一个基于用户协同过滤的推荐算法。

**答案：** 下面是一个简单的基于用户协同过滤的推荐算法示例：

```python
import numpy as np

def collaborative_filtering(train_data, user_id, k=5):
    # 计算用户与其他用户的相似度
    user_similarity = []
    for other_user in train_data:
        if other_user != user_id:
            sim = 1 - np.linalg.norm(train_data[user_id] - train_data[other_user])
            user_similarity.append((other_user, sim))
    
    # 排序相似度最高的用户
    top_k = sorted(user_similarity, key=lambda x: x[1], reverse=True)[:k]
    
    # 为当前用户推荐相似用户的商品
    recommended_items = set()
    for other_user, _ in top_k:
        recommended_items.update(train_data[other_user])
    
    return recommended_items

# 示例数据
train_data = {
    0: [1, 0, 1, 1],
    1: [1, 1, 0, 0],
    2: [0, 1, 1, 1],
    3: [1, 1, 1, 0]
}

# 为用户0推荐商品
recommended_items = collaborative_filtering(train_data, 0)
print("Recommended items:", recommended_items)
```

**解析：** 该算法计算当前用户与其他用户的相似度，选取相似度最高的 k 个用户，推荐他们的商品。这里使用了欧氏距离作为相似度度量。

### 3. 购物车商品组合优化

**题目：** 请设计一个算法，用于优化购物车中的商品组合，提高订单总价值。

**答案：** 下面是一个简单的购物车商品组合优化算法示例：

```python
def optimize_cart(cart, max_items=5):
    # 计算每个商品的价值
    item_values = {item: cart[item] * prices[item] for item in cart}
    
    # 对商品按照价值降序排序
    sorted_items = sorted(item_values.items(), key=lambda x: x[1], reverse=True)
    
    # 选择前 max_items 个商品组成优化后的购物车
    optimized_cart = {item: amount for item, amount in sorted_items[:max_items]}
    
    return optimized_cart

# 示例购物车和商品价格
cart = {'apple': 2, 'orange': 3, 'banana': 1, 'watermelon': 5}
prices = {'apple': 1.5, 'orange': 2.0, 'banana': 1.0, 'watermelon': 3.0}

# 优化购物车
optimized_cart = optimize_cart(cart)
print("Optimized cart:", optimized_cart)
```

**解析：** 该算法首先计算每个商品的价值（数量 * 价格），然后对商品按照价值进行降序排序，选择前 max_items 个商品组成优化后的购物车。
