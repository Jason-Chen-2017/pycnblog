### 常见最短路径算法

在解决最短路径问题时，常用的算法有迪杰斯特拉算法（Dijkstra算法）、贝尔曼-福特算法（Bellman-Ford算法）和弗洛伊德算法（Floyd-Warshall算法）。以下是对这些算法的简要介绍：

#### 1. 迪杰斯特拉算法（Dijkstra算法）

迪杰斯特拉算法是一种基于贪心策略的算法，适用于单源最短路径问题。该算法的时间复杂度为\(O(E \log V)\)，其中\(E\)是边的数量，\(V\)是顶点的数量。

#### 2. 贝尔曼-福特算法（Bellman-Ford算法）

贝尔曼-福特算法是一种基于松弛技术的算法，可以处理包含负权边的图。它的时间复杂度为\(O(VE)\)，其中\(V\)是顶点的数量，\(E\)是边的数量。

#### 3. 弗洛伊德算法（Floyd-Warshall算法）

弗洛伊德算法是一种基于动态规划思想的算法，可以计算所有顶点对之间的最短路径。它的时间复杂度为\(O(V^3)\)，其中\(V\)是顶点的数量。

### 源代码实例

以下是一个使用迪杰斯特拉算法求解单源最短路径问题的示例代码：

```python
import heapq

def dijkstra(graph, start):
    dist = {vertex: float('infinity') for vertex in graph}
    dist[start] = 0
    priority_queue = [(0, start)]

    while priority_queue:
        current_dist, current_vertex = heapq.heappop(priority_queue)

        if current_dist > dist[current_vertex]:
            continue

        for neighbor, weight in graph[current_vertex].items():
            distance = current_dist + weight

            if distance < dist[neighbor]:
                dist[neighbor] = distance
                heapq.heappush(priority_queue, (distance, neighbor))

    return dist

graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}

print(dijkstra(graph, 'A'))  # 输出：{'A': 0, 'B': 1, 'C': 4, 'D': 5}
```

在本例中，我们首先创建了一个包含四个顶点和边的图。然后，我们使用迪杰斯特拉算法来计算从顶点'A'出发到其他所有顶点的最短路径。算法的核心思想是维护一个优先队列，用于选择下一个访问的顶点。在每次迭代中，我们选择优先队列中的最小距离顶点，然后更新其相邻顶点的距离。最后，我们返回一个字典，其中包含了从起点到每个顶点的最短距离。
