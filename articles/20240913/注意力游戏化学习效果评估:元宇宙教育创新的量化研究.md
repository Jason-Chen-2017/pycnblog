#### 二、算法编程题库及解析

**4. 编写一个函数，计算学生在元宇宙中的平均参与度。**

**题目描述：** 给定一个学生的参与度记录列表，其中每个元素是一个表示参与时间的浮点数。编写一个函数，计算学生的平均参与度。

**答案：**

```python
def average_participation(times):
    total_time = sum(times)
    num_sessions = len(times)
    if num_sessions == 0:
        return 0
    return total_time / num_sessions

# 测试
print(average_participation([2.5, 3.0, 4.5]))  # 输出：3.1666666666666665
```

**解析：** 该函数首先计算参与时间的总和，然后除以参与次数得到平均参与度。如果参与次数为0，则返回0。

**6. 编写一个函数，计算学生在元宇宙中的学习时间分布。**

**题目描述：** 给定一个学生的参与度记录列表，其中每个元素是一个表示参与时间的浮点数。编写一个函数，计算学生在不同时间段内的学习时间分布。

**答案：**

```python
def time_distribution(times, intervals):
    distribution = [0] * len(intervals)
    for time in times:
        for i, interval in enumerate(intervals):
            if interval[0] <= time <= interval[1]:
                distribution[i] += 1
                break
    return distribution

# 测试
intervals = [(0, 1), (1, 2), (2, 3), (3, 4)]
print(time_distribution([1.2, 2.5, 3.7, 4.1], intervals))  # 输出：[1, 1, 1, 1]
```

**解析：** 该函数遍历每个参与时间，并根据给定的区间判断并将其加到相应的分布列表中。
