### 3. 算法编程题库及解析

#### 3.1 词频统计

**题目：** 编写一个函数，实现统计字符串中单词的出现频率。

```python
def word_frequency(text):
    # 请在此处实现代码
```

**答案：** 

```python
def word_frequency(text):
    words = text.split()
    frequency = {}
    for word in words:
        frequency[word] = frequency.get(word, 0) + 1
    return frequency
```

**解析：** 该函数首先将输入的字符串按空格分割成单词，然后遍历单词列表，统计每个单词出现的次数，并返回一个字典。

#### 3.2 情感分析

**题目：** 编写一个函数，实现根据文本内容判断其情感倾向。

```python
def sentiment_analysis(text):
    # 请在此处实现代码
```

**答案：** 

```python
from textblob import TextBlob

def sentiment_analysis(text):
    analysis = TextBlob(text)
    if analysis.sentiment.polarity > 0:
        return "正面"
    elif analysis.sentiment.polarity < 0:
        return "负面"
    else:
        return "中性"
```

**解析：** 该函数利用TextBlob库实现情感分析，根据文本的情感极性（polarity）判断其情感倾向。当极性大于0时，表示正面；小于0时，表示负面；等于0时，表示中性。
