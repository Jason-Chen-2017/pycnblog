def heuristic(node, goal):
    # 使用欧几里得距离作为启发式函数
    return ((node.x - goal.x) ** 2 + (node.y - goal.y) ** 2) ** 0.5

##### 2. 如何计算真空管道运输的能源消耗？

**答案：**
- 基于流体力学原理，计算空气流动的能量消耗。
- 考虑管道摩擦、温度变化等因素。

```python
def calculate_energy_consumption(volume, velocity, density, temperature):
    # 计算空气流动的能量消耗
    energy = 0.5 * density * volume * velocity ** 2
    # 考虑管道摩擦和温度变化
    energy += friction_energy_loss(volume, velocity, temperature)
    return energy

def friction_energy_loss(volume, velocity, temperature):
    # 计算管道摩擦的能量消耗
    friction_coefficient = 0.01  # 摩擦系数
    return friction_coefficient * volume * velocity ** 2 * temperature
```
