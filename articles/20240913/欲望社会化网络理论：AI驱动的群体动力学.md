### 算法编程题库

#### 7. 编写一个程序，模拟社交网络中信息的传播过程。

**答案：** 社交网络信息传播模拟程序可以使用图论算法来模拟信息在网络中的传播。以下是一个简单的示例：

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建图
G = nx.Graph()

# 添加节点和边
G.add_nodes_from([1, 2, 3, 4, 5])
G.add_edges_from([(1, 2), (1, 3), (2, 4), (3, 4), (4, 5)])

# 绘制图
nx.draw(G, with_labels=True)
plt.show()

# 传播信息
def propagate_info(G, start_node, info):
    visited = set()
    to_visit = [start_node]
    while to_visit:
        current_node = to_visit.pop()
        if current_node not in visited:
            print(f"节点 {current_node} 接收到信息：{info}")
            visited.add(current_node)
            to_visit.extend([neighbor for neighbor in G.neighbors(current_node) if neighbor not in visited])

# 模拟信息传播
propagate_info(G, 1, "大家好，我是一条新信息！")
```

**解析：** 这个程序使用NetworkX库创建了一个简单的图，表示社交网络中的节点和边。然后定义了一个函数`propagate_info`来模拟信息在网络中的传播过程。程序从起始节点开始，递归地遍历网络，将信息传播给未访问过的邻居节点。
