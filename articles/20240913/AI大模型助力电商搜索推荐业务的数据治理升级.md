### 三、算法编程题库及解析
#### 1. 单词搜索（LeetCode 212）

**题目描述：** 给定一个二维网格和一个单词，判断该单词是否可以通过网格中的递归路径找到。

**答案：** 使用深度优先搜索（DFS）算法，遍历网格，从起点开始递归搜索路径。

```python
def exist(board, word):
    def dfs(i, j, k):
        if k == len(word):
            return True
        if (
            i < 0
            or i >= len(board)
            or j < 0
            or j >= len(board[0])
            or board[i][j] != word[k]
        ):
            return False
        board[i][j], c = "#", board[i][j]
        res = (
            dfs(i + 1, j, k + 1)
            or dfs(i - 1, j, k + 1)
            or dfs(i, j + 1, k + 1)
            or dfs(i, j - 1, k + 1)
        )
        board[i][j] = c
        return res

    return any(dfs(i, j, 0) for i in range(len(board)) for j in range(len(board[0])))
```

#### 2. 最长公共前缀（LeetCode 14）

**题目描述：** 找出字符串数组中的最长公共前缀。

**答案：** 使用横向扫描法，从字符串数组中的第一个字符串开始，逐个比较前缀。

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""

    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s) and prefix[i] == s[i]:
            i += 1
        prefix = prefix[:i]
    return prefix
```
