#### 二、算法编程题库

**1. 编写一个程序，实现一个简单的情感分析算法，判断一段文字的情感倾向是积极、中性还是消极。**

**答案：** 
```python
import nltk
from nltk.sentiment import SentimentIntensityAnalyzer

def sentiment_analysis(text):
    sid = SentimentIntensityAnalyzer()
    score = sid.polarity_scores(text)
    if score['compound'] >= 0.05:
        return '积极'
    elif score['compound'] <= -0.05:
        return '消极'
    else:
        return '中性'

text = "我对这个产品非常喜欢，使用起来非常方便。"
print(sentiment_analysis(text))
```

**解析：** 这个简单的情感分析算法使用了自然语言处理库 `nltk` 的 `SentimentIntensityAnalyzer` 工具。通过计算文本的复合情感得分（`compound`），判断文本的情感倾向。

**2. 编写一个程序，使用决策树算法预测客户的购买意愿。**

**答案：**
```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
import matplotlib.pyplot as plt

# 加载数据集
iris = load_iris()
X, y = iris.data, iris.target

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 创建决策树分类器
clf = DecisionTreeClassifier()

# 训练模型
clf.fit(X_train, y_train)

# 预测
y_pred = clf.predict(X_test)

# 绘制决策树
plt.figure(figsize=(12, 12))
_ = clf.plot_tree(class_names=iris.target_names, filled=True)
plt.show()
```

**解析：** 这个程序首先加载数据集，然后使用 `train_test_split` 划分训练集和测试集。接着，使用 `DecisionTreeClassifier` 创建决策树分类器，进行训练和预测。最后，使用 `plot_tree` 函数绘制决策树的结构。
