#### 算法编程题库

##### 4. 背包问题

**题目：** 给定一个物品列表和总容量，求解最多可以装入物品的总价值。

**示例：**

物品列表：[2, 3, 4, 5]
总容量：5

输出：7（装入物品3和4，总价值为3+4=7）

**答案：**

```python
def knapsack(W, wt, val, n):
    if n == 0 or W == 0:
        return 0

    if wt[n-1] > W:
        return knapsack(W, wt, val, n-1)

    return max(val[n-1] + knapsack(W-wt[n-1], wt, val, n-1), knapsack(W, wt, val, n-1))

# 示例
val = [1, 6, 10, 16]
wt = [1, 2, 3, 5]
W = 7
n = len(val)
print(knapsack(W, wt, val, n))
```

##### 5. 最长公共子序列

**题目：** 给定两个字符串，求解它们的最长公共子序列。

**示例：**

字符串1：`ABCDGH`
字符串2：`AEDFHR`

输出：`ADH`

**答案：**

```python
def longest_common_subsequence(X, Y):
    m = len(X)
    n = len(Y)

    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if X[i-1] == Y[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    result = []
    i, j = m, n
    while i > 0 and j > 0:
        if X[i-1] == Y[j-1]:
            result.append(X[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] > dp[i][j-1]:
            i -= 1
        else:
            j -= 1

    return ''.join(result[::-1])

# 示例
X = "ABCDGH"
Y = "AEDFHR"
print(longest_common_subsequence(X, Y))
```

##### 6. 单源最短路径（Dijkstra算法）

**题目：** 给定一个无权图中所有边长均为正数的图，求解图中任意两点之间的最短路径。

**示例：**

```
0 --- 1 --- 3
| \ |  |
2 --- 4 --- 5
```

输出：

- `0` 到 `5` 的最短路径：`0 -> 1 -> 3 -> 4 -> 5`
- `2` 到 `5` 的最短路径：`2 -> 0 -> 1 -> 3 -> 4 -> 5`

**答案：**

```python
import heapq

def dijkstra(graph, start):
    n = len(graph)
    dist = [float('inf')] * n
    dist[start] = 0
    pq = [(0, start)]

    while pq:
        curr_dist, curr_node = heapq.heappop(pq)

        if curr_dist > dist[curr_node]:
            continue

        for neighbor, weight in graph[curr_node].items():
            distance = curr_dist + weight

            if distance < dist[neighbor]:
                dist[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))

    return dist

# 示例
graph = {
    0: {1: 1, 2: 4},
    1: {0: 1, 3: 2},
    2: {0: 4, 4: 1},
    3: {1: 2, 4: 1},
    4: {2: 1, 3: 1, 5: 1},
    5: {4: 1}
}
start = 0
distances = dijkstra(graph, start)
print(distances)
```

#### 算法编程题库

以下是算法编程题库中的三个问题及其解析：

##### 4. 背包问题

背包问题是经典的最优化问题，解决方法之一是使用动态规划。在上述答案中，`knapsack` 函数采用递归的方式实现动态规划算法。

- **解析：** 函数使用递归遍历所有可能的物品组合，并选择其中总价值最大的组合。如果当前物品的重量大于剩余容量，则忽略该物品；否则，计算包含和不包含该物品时的最大总价值，取两者中的最大值。
- **复杂度：** 背包问题的复杂度为 O(n*W)，其中 n 是物品数量，W 是总容量。这个算法的时间复杂度较高，适用于容量较小的背包问题。

##### 5. 最长公共子序列

最长公共子序列（LCS）问题是计算两个序列中最长公共子序列的长度。在上述答案中，使用动态规划算法实现。

- **解析：** 算法使用一个二维数组 `dp` 来存储子序列的长度，其中 `dp[i][j]` 表示 `X[0..i]` 和 `Y[0..j]` 的最长公共子序列的长度。如果当前字符匹配，则 `dp[i][j] = dp[i-1][j-1] + 1`；否则，取相邻位置中的最大值。
- **复杂度：** 最长公共子序列问题的复杂度为 O(m*n)，其中 m 和 n 分别为两个序列的长度。

##### 6. 单源最短路径（Dijkstra算法）

Dijkstra 算法是求解单源最短路径问题的经典算法。

- **解析：** 算法使用一个优先队列（最小堆）来选择下一个要处理的顶点，每次选择距离源点最近的顶点。对于每个顶点，更新其相邻顶点的最短路径长度。
- **复杂度：** Dijkstra 算法的平均复杂度为 O((V+E)logV)，其中 V 是顶点数量，E 是边数量。在实际应用中，当图中边数量较大时，Dijkstra 算法的时间复杂度可能会较高。
