#### 1. 典型面试题与简洁解答

**面试题 1：用栈实现队列**

**题目描述：** 实现一个用栈实现的队列。队列应支持 push、pop、peek 操作。

**简洁答案：** 利用两个栈，一个用于入队操作，一个用于出队操作。

```python
class MyQueue:
    def __init__(self):
        self.stack_in = []
        self.stack_out = []

    def push(self, x):
        self.stack_in.append(x)

    def pop(self):
        if not self.stack_out:
            while self.stack_in:
                self.stack_out.append(self.stack_in.pop())
        return self.stack_out.pop()

    def peek(self):
        if not self.stack_out:
            while self.stack_in:
                self.stack_out.append(self.stack_in.pop())
        return self.stack_out[-1]
```

**解析：** 通过双栈机制，实现了队列的入队和出队操作。在出队时，若出栈为空，则将入栈元素逐个移至出栈，从而实现出队操作。

#### 2. 算法编程题与高效算法

**编程题 2：寻找旋转排序数组中的最小值**

**题目描述：** 给定一个旋转排序的数组，找出并返回数组中的最小元素。

**简洁算法：** 使用二分查找法，时间复杂度为 O(log n)。

```python
def findMin(nums):
    left, right = 0, len(nums) - 1
    while left < right:
        mid = (left + right) // 2
        if nums[mid] > nums[right]:
            left = mid + 1
        else:
            right = mid
    return nums[left]
```

**解析：** 利用二分查找，将问题分为旋转点和非旋转点，逐步缩小搜索范围，找到最小值。
