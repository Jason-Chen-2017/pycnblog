##### 1. 如何评估一家互联网公司的数据能力？

**答案：**

* **数据存储：** 评估数据存储方案的容量、性能、安全性等。例如，企业是否采用分布式存储、云存储等方案。

#### 三、算法编程题库及解析

##### 1. 快手面试题：找出数组中第二大的数

**题目：** 给定一个整数数组 `nums`，找出数组中第二大的数。如果数组中没有第二大的数，返回 `-1`。

**答案：**

```python
def findSecondLargest(nums):
    if len(nums) < 2:
        return -1
    
    largest = second_largest = float('-inf')
    
    for num in nums:
        if num > largest:
            second_largest = largest
            largest = num
        elif num > second_largest and num != largest:
            second_largest = num
    
    return second_largest if second_largest > float('-inf') else -1

# 示例
nums = [2, 1, 5, 2, 3, 2]
print(findSecondLargest(nums))  # 输出：3
```

**解析：** 该题考察了数组遍历和逻辑判断能力。通过遍历数组，维护两个变量 `largest` 和 `second_largest`，分别表示最大值和第二大的值，最终返回第二大的值。

##### 2. 腾讯面试题：最长公共子序列

**题目：** 给定两个字符串 `s1` 和 `s2`，找出它们的最长公共子序列。

**答案：**

```python
def longestCommonSubsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    return dp[m][n]

# 示例
s1 = "ABCD"
s2 = "ACE"
print(longestCommonSubsequence(s1, s2))  # 输出："AC"
```

**解析：** 该题考察了动态规划算法。通过建立二维数组 `dp`，利用状态转移方程求解最长公共子序列的长度。最后，可以通过回溯找到最长公共子序列。
