#### 二、算法编程题

##### 4. 编写一个程序，实现基于 K-均值算法的空气质量预测。

**答案：** K-均值算法是一种聚类算法，用于将数据划分为 K 个簇。在空气质量预测中，可以将每个样本（如时间点的污染物浓度）视为一个簇，然后根据簇的中心来预测未来的空气质量。

```python
import numpy as np

def k_means(data, k, max_iterations):
    centroids = data[np.random.choice(data.shape[0], k, replace=False)]
    for _ in range(max_iterations):
        # 计算每个样本与聚类中心的距离
        distances = np.linalg.norm(data - centroids, axis=1)
        # 分配到最近的聚类中心
        labels = np.argmin(distances, axis=1)
        # 更新聚类中心
        new_centroids = np.array([data[labels == i].mean(axis=0) for i in range(k)])
        if np.all(centroids == new_centroids):
            break
        centroids = new_centroids
    return centroids, labels

# 示例数据
data = np.random.rand(100, 2)
k = 3
max_iterations = 100

# 运行 K-均值算法
centroids, labels = k_means(data, k, max_iterations)
print("聚类中心：", centroids)
print("样本标签：", labels)
```

**解析：** 以上代码实现了 K-均值算法，将随机生成的 100 个二维数据点划分为 3 个簇。算法首先随机初始化聚类中心，然后迭代更新聚类中心，直到聚类中心不再变化。

##### 5. 编写一个程序，实现基于回归分析的空气质量预测。

**答案：** 回归分析是一种预测模型，通过分析自变量和因变量之间的关系来预测因变量的值。在空气质量预测中，可以将污染物浓度（因变量）与气象条件（自变量）之间的关系建模，然后使用回归模型进行预测。

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 示例数据
X = np.random.rand(100, 5)  # 每个样本有 5 个气象条件
y = 2 * X[:, 0] + 3 * X[:, 1] + 5 * X[:, 2] + 7 * X[:, 3] + 11 * X[:, 4] + np.random.normal(0, 1, 100)  # 污染物浓度

# 创建线性回归模型
model = LinearRegression()
# 训练模型
model.fit(X, y)
# 预测
y_pred = model.predict(X)

print("回归系数：", model.coef_)
print("截距：", model.intercept_)
print("R^2：", model.score(X, y))
print("预测值：", y_pred)
```

**解析：** 以上代码使用线性回归模型来预测每个样本的污染物浓度。首先随机生成 100 个样本，每个样本有 5 个气象条件，污染物浓度是这些气象条件的线性组合。然后使用线性回归模型进行训练和预测，并输出回归系数、截距和 R^2 值。
