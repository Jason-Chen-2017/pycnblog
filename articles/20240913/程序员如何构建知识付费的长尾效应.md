### 5. 算法编程题库

**问题：** 设计一个算法，计算字符串中的最长公共前缀。

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s):
            if prefix[i] != s[i]:
                break
            i += 1
        prefix = prefix[:i]
    return prefix

# 示例
strs = ["flower","flow","flight"]
print(longestCommonPrefix(strs))  # 输出 "fl"
```

**解析：** 该算法通过逐个比较字符串中的字符，找出最长公共前缀。时间复杂度为O(N*M)，其中N为字符串数量，M为最长字符串的长度。
