## 算法编程题

### 1. 编写一个Python函数，实现数字孪生数据的预处理，包括数据清洗、去重和规范化。

**答案：** 

```python
def preprocess_data(data):
    # 数据清洗
    cleaned_data = [d for d in data if d is not None and not isinstance(d, str)]
    # 数据去重
    unique_data = list(set(cleaned_data))
    # 数据规范化
    normalized_data = [d.lower() for d in unique_data]
    return normalized_data
```

### 2. 编写一个JavaScript函数，实现元宇宙中虚拟角色的运动轨迹规划，要求轨迹平滑且符合物理规律。

**答案：**

```javascript
function plan_trajectory(start, end, duration) {
    const dx = end.x - start.x;
    const dy = end.y - start.y;
    const distance = Math.sqrt(dx * dx + dy * dy);
    const time_per_unit = duration / distance;

    const trajectory = [];
    for (let t = 0; t <= duration; t += time_per_unit) {
        const x = start.x + dx * t / duration;
        const y = start.y + dy * t / duration;
        trajectory.push({ x, y });
    }

    return trajectory;
}
```
