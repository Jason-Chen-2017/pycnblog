#### 1. 如何为宠物店设计库存管理系统？

**答案：**

- **数据结构：** 使用数组或列表存储宠物商品信息，如名称、价格、库存量等。
- **功能实现：** 实现添加、删除、查询和更新商品库存量的功能。

```python
class InventorySystem:
    def __init__(self):
        self.products = []

    def add_product(self, product):
        self.products.append(product)

    def remove_product(self, product_name):
        for i, p in enumerate(self.products):
            if p['name'] == product_name:
                del self.products[i]
                break

    def update_inventory(self, product_name, quantity):
        for p in self.products:
            if p['name'] == product_name:
                p['quantity'] = quantity
                break

    def get_inventory(self, product_name):
        for p in self.products:
            if p['name'] == product_name:
                return p['quantity']
        return 0
```

- **预约算法：** 根据客户需求和宠物店服务能力，合理分配预约时间。

```python
def schedule_appointments(appointments):
    appointments.sort(key=lambda x: x['time'])
    scheduled = []
    for appointment in appointments:
        if can_schedule(appointment, scheduled):
            scheduled.append(appointment)
    return scheduled

def can_schedule(appointment, scheduled):
    for s in scheduled:
        if appointment['pet'] == s['pet'] and abs(appointment['time'] - s['time']) < appointment['duration']:
            return False
    return True
```
