#### 一、极限与连续性面试题

##### 1. 极限的定义是什么？

**答案：** 极限是函数在自变量趋近某一点时函数值的趋势。形式上，我们说函数 \( f(x) \) 在 \( x \) 趋近于 \( a \) 时的极限是 \( L \)，如果当 \( x \) 趋近于 \( a \) 时，\( f(x) \) 的值能够无限接近于 \( L \)，但不必等于 \( L \)。

##### 2. 如何求解极限？

**答案：** 求解极限的方法包括直接计算、夹逼定理、单调有界原理等。常见的求解步骤如下：

1. 观察极限形式，确定使用哪种方法；
2. 对极限表达式进行化简；
3. 应用相关定理或公式求解。

##### 3. 什么是无穷小和无穷大？

**答案：** 无穷小是指在自变量趋近某一点时绝对值越来越小的量，无穷大是指在自变量趋近某一点时绝对值无限增大的量。无穷小与无穷大是相对的概念，只有在自变量趋近某一点的情境下才有意义。

##### 4. 如何判断函数的连续性？

**答案：** 函数 \( f(x) \) 在点 \( x = a \) 连续，如果满足以下三个条件：

1. \( \lim_{{x \to a}} f(x) \) 存在；
2. \( f(a) \) 存在；
3. \( \lim_{{x \to a}} f(x) = f(a) \)。

##### 5. 什么是连续函数的极限？

**答案：** 连续函数的极限就是函数值本身。即，如果函数 \( f(x) \) 在某一点连续，那么该点的极限就是该点的函数值。

#### 二、极限与连续性算法编程题

##### 6. 实现计算极限的函数

**题目：** 编写一个函数，用于计算函数 \( f(x) = \frac{\sin(x)}{x} \) 在 \( x \) 趋近于 0 时的极限。

**答案：** 

```python
import math

def limit_sin_x(x):
    return math.sin(x) / x

# 计算极限
limit = limit_sin_x(0)
print("极限值为：", limit)
```

##### 7. 判断函数的连续性

**题目：** 编写一个函数，用于判断给定的函数在指定点的连续性。

**答案：** 

```python
import math

def is_continuous(f, x):
    return math.isclose(f(x), lim(f, x))

def lim(f, x):
    try:
        return float(f(x))
    except Exception as e:
        return e

# 测试函数
def f(x):
    return x**2

# 判断连续性
print(is_continuous(f, 0))
```

##### 8. 求解连续函数的极限

**题目：** 编写一个函数，用于求解连续函数在指定点的极限。

**答案：** 

```python
import math

def limit_continuous(f, x):
    return f(x)

# 求解极限
limit = limit_continuous(f, 0)
print("极限值为：", limit)
```
