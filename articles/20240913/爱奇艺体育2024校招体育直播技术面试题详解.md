#### 算法编程题及解析

##### 6. 请实现一个算法，计算体育直播观众人数的实时统计。

**题目：** 请实现一个算法，计算体育直播观众人数的实时统计。

**答案：**

```python
def count_viewers(viewer_ids):
    viewers = {}
    for id in viewer_ids:
        viewers[id] = viewers.get(id, 0) + 1
    return len(viewers)

viewer_ids = ["A", "B", "C", "A", "B", "A", "D", "B"]
print(count_viewers(viewer_ids))  # 输出 4
```

**解析：** 该算法通过字典统计每个观众ID出现的次数，最终返回观众人数。

##### 7. 请实现一个算法，根据观众行为数据推荐合适的直播内容。

**题目：** 请实现一个算法，根据观众行为数据推荐合适的直播内容。

**答案：**

```python
def recommend_content(viewer_actions):
    content_ratings = {}
    for action, content in viewer_actions:
        content_ratings[action] = content_ratings.get(action, 0) + 1
    sorted_content = sorted(content_ratings.items(), key=lambda x: x[1], reverse=True)
    return sorted_content[0][0]

viewer_actions = [("watch", "篮球"), ("skip", "足球"), ("watch", "篮球"), ("watch", "篮球"), ("skip", "足球")]
print(recommend_content(viewer_actions))  # 输出 "篮球"
```

**解析：** 该算法根据观众对内容的观看次数进行排序，推荐观看次数最多的内容。
