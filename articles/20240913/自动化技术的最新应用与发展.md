**问题 1: 什么是自动化技术？**

**答案：** 自动化技术是指通过使用机器、计算机程序和其他技术手段，使机器或系统能够自动执行任务，而不需要人工干预。它涉及到机械工程、电子工程、计算机科学等多个领域。

**源代码实例：**

```python
# 一个简单的自动化脚本，用于发送电子邮件
import smtplib
from email.mime.text import MIMEText
from email.header import Header

def send_email(sender, receiver, subject, content):
    # SMTP服务器设置
    smtp_server = "smtp.example.com"
    smtp_port = 587
    smtp_username = "your_username"
    smtp_password = "your_password"

    # 创建 SMTP 对象
    server = smtplib.SMTP(smtp_server, smtp_port)
    server.starttls()

    # 登录 SMTP 服务器
    server.login(smtp_username, smtp_password)

    # 创建邮件对象
    message = MIMEText(content, 'plain', 'utf-8')
    message['From'] = Header(sender, 'UTF-8')
    message['To'] = Header(receiver, 'UTF-8')
    message['Subject'] = Header(subject, 'UTF-8')

    # 发送邮件
    server.sendmail(sender, receiver, message.as_string())

    # 关闭 SMTP 连接
    server.quit()

# 调用发送邮件函数
send_email('sender@example.com', 'receiver@example.com', 'Test Email', 'This is a test email.')
```

**问题 2: 自动化技术在制造业中的应用有哪些？**

**答案：** 自动化技术在制造业中的应用非常广泛，包括但不限于以下几个方面：

1. **自动化生产线：** 通过使用机器人、自动化设备和控制系统，实现生产线的自动化运作，提高生产效率。
2. **自动化检测：** 利用机器视觉和传感器技术，对产品进行自动化检测，确保产品质量。
3. **自动化装配：** 使用机器人进行零部件的装配，减少人工操作，提高装配精度和效率。
4. **自动化物流：** 通过自动化仓库管理系统和自动化输送设备，实现原材料和成品的自动化存储和运输。
5. **自动化质量控制：** 利用自动化测试设备对产品进行自动化测试，确保产品质量满足要求。

**源代码实例：**

```python
# 一个简单的自动化检测脚本，用于检测产品质量
import cv2
import numpy as np

def detect_quality(image_path):
    # 读取图像
    image = cv2.imread(image_path)

    # 将图像转换为灰度图像
    gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

    # 使用阈值分割图像
    _, binary_image = cv2.threshold(gray_image, 128, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)

    # 计算图像中黑色区域的面积
    contours, _ = cv2.findContours(binary_image, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    area = sum([cv2.contourArea(contour) for contour in contours])

    # 判断产品质量
    if area > 1000:
        print("Quality: Good")
    else:
        print("Quality: Bad")

# 调用检测函数
detect_quality('image_path.jpg')
```
