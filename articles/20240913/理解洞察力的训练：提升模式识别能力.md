##### 3. 图像识别

**题目：** 使用Python实现一个基于卷积神经网络的图像识别模型，并使用MNIST数据集进行训练和测试。

**答案：** 参考以下代码实现：

```python
import tensorflow as tf
from tensorflow.keras import layers
import tensorflow_datasets as tfds

# 加载MNIST数据集
ds, ds_info = tfds.load('mnist', split='train', shuffle_files=True, as_supervised=True)
train_ds = ds.take(60000)

# 构建模型
model = tf.keras.Sequential([
    layers.Flatten(input_shape=(28, 28)),
    layers.Dense(128, activation='relu'),
    layers.Dense(10, activation='softmax')
])

# 编译模型
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# 训练模型
model.fit(train_ds, epochs=5)

# 测试模型
test_ds = ds.take(10000)
test_loss, test_acc = model.evaluate(test_ds)
print("Test accuracy:", test_acc)
```

**解析：** 本代码使用TensorFlow实现了一个简单的卷积神经网络模型，用于手写数字识别。通过加载MNIST数据集进行训练和测试，模型可以学会自动识别手写数字。
