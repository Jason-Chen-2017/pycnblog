```python
def preprocess_prompt(prompt):
    # 去除字符串中的特殊字符
    cleaned_prompt = ''.join(c for c in prompt if c.isalnum() or c.isspace())
    
    # 将字符串转换为小写
    lower_prompt = cleaned_prompt.lower()
    
    # 返回预处理后的字符串
    return lower_prompt
```

**解析：** 这个函数首先去除字符串中的特殊字符，然后将其转换为小写，以便更好地处理和提取关键信息。
