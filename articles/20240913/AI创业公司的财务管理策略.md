### 算法编程题与答案解析

#### 1. 股票买卖的最佳时机

**题目：** 给定一个整数数组 `prices`，其中 `prices[i]` 是第 `i` 天股票的价格。如果你最多只能完成一笔买卖交易，设计一个算法来计算你所能获取的最大利润。

**答案：** 解法一：暴力法
```python
def maxProfit(prices):
    max_profit = 0
    for i in range(len(prices)):
        for j in range(i+1, len(prices)):
            profit = prices[j] - prices[i]
            max_profit = max(max_profit, profit)
    return max_profit
```
解法二：单次遍历
```python
def maxProfit(prices):
    min_price = float('inf')
    max_profit = 0
    for price in prices:
        min_price = min(min_price, price)
        profit = price - min_price
        max_profit = max(max_profit, profit)
    return max_profit
```
**解析：** 第一种解法采用两层循环，时间复杂度为 O(n^2)，适用于小规模数据。第二种解法通过单次遍历，找到最低价格和最大利润，时间复杂度为 O(n)，适用于大规模数据。

#### 2. 最长连续递增序列

**题目：** 给定一个整数数组 `nums`，返回其中最长连续递增序列的长度。

**答案：**
```python
def lengthOfLCIS(nums):
    if not nums:
        return 0
    count = 1
    res = 1
    for i in range(1, len(nums)):
        if nums[i] > nums[i - 1]:
            count += 1
            res = max(res, count)
        else:
            count = 1
    return res
```
**解析：** 该题使用一次遍历，用一个计数器 `count` 记录当前连续递增序列的长度，遇到不递增的数字时重置计数器。最终返回最大的连续递增序列长度。
