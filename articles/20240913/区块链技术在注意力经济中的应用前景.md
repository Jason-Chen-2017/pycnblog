### 算法编程题解析

#### 3. 设计一个基于区块链的数字版权保护系统。

**题目：** 设计一个简单的数字版权保护系统，确保创作者的作品在发布后，无法被未经授权的用户非法复制和传播。

**答案：** 可以使用智能合约来设计一个简单的数字版权保护系统。智能合约将定义创作者、版权信息和授权机制，用户在获取授权后才能合法使用作品。

```solidity
pragma solidity ^0.8.0;

contract DigitalCopyright {
    struct Work {
        string title;
        address creator;
        bool isCopyrighted;
        mapping(address => bool) accessRights;
    }

    mapping(string => Work) works;

    function createWork(string memory title, address creator) public {
        works[title] = Work({
            title: title,
            creator: creator,
            isCopyrighted: true
        });
    }

    function authorizeUser(string memory title, address user) public {
        works[title].accessRights[user] = true;
    }

    function checkAccess(string memory title, address user) public view returns (bool) {
        return works[title].accessRights[user];
    }
}
```

**解析：** 该智能合约定义了「Work」结构体来存储版权信息，包括标题、创作者、是否版权保护以及用户的授权状态。`createWork` 函数用于创建新作品，`authorizeUser` 函数用于授权用户，`checkAccess` 函数用于检查用户是否有权限访问作品。

#### 4. 实现一个去中心化的数字广告投放系统。

**题目：** 设计一个去中心化的数字广告投放系统，用户可以根据自己的兴趣选择广告，广告主可以根据投放效果调整广告策略。

**答案：** 可以使用区块链技术来实现一个去中心化的广告投放系统。系统可以记录用户的兴趣偏好和广告主的广告信息，并允许用户根据兴趣选择广告。

```solidity
pragma solidity ^0.8.0;

contract DecentralizedAd {
    struct Ad {
        string url;
        uint256 budget;
        bool isActive;
        mapping(address => uint256) impressions;
    }

    mapping(string => Ad) ads;

    function createAd(string memory url, uint256 budget) public {
        ads[url] = Ad({
            url: url,
            budget: budget,
            isActive: true
        });
    }

    function setActive(string memory url, bool isActive) public {
        ads[url].isActive = isActive;
    }

    function viewAd(string memory url) public {
        require(ads[url].isActive == true, "Ad is not active");
        ads[url].impressions[msg.sender]++;
    }

    function getImpressions(string memory url) public view returns (uint256) {
        return ads[url].impressions[msg.sender];
    }
}
```

**解析：** 该智能合约定义了「Ad」结构体来存储广告信息，包括广告链接、预算、活动状态以及用户的曝光次数。`createAd` 函数用于创建新广告，`setActive` 函数用于启用或禁用广告，`viewAd` 函数用于用户观看广告，`getImpressions` 函数用于获取用户的曝光次数。
