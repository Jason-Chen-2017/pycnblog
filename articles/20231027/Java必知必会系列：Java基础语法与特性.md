# 2.核心概念与联系
## 2.1数据类型
### 整型数据类型
整型数据类型分为四种：byte(字节)、short(短整数)、int(整数)、long(长整数)。它们的存储大小如下所示：
```
byte:      -128 ~ +127    (2^8)
short:     -32768 ~ +32767   (2^16)
int:       -2147483648 ~ +2147483647   (2^32)
long:      -9223372036854775808 ~ +9223372036854775807   (2^64)
```
一般情况下，建议使用int作为主要的数据类型，除非需要使用大整数或者负数。其他两种数据类型适用于特殊需求，比如索引值或者计数器。另外，byte、short、int、long这几种类型的数据在赋值时会自动类型提升，即小范围的数据类型可以隐式转换成大范围的数据类型。
### 浮点型数据类型
浮点型数据类型分为三种：float、double、BigDecimal。它们的存储大小如下所示：
```
float:     (-3.402823466E+38 ~ +3.402823466E+38)(约6位有效数字)
double:    (-1.7976931348623157E+308 ~ +1.7976931348623157E+308)(约15位有效数字)
BigDecimal:能够表示任意精度的数字，但计算比较慢。
```
一般情况下，建议使用double作为主要的数据类型，除非要求高精确度或者范围更广。另外，float类型的数据不能表示精确的值，double类型的数据可以使用 BigDecimal 来进行高精确度的运算。
### 字符型数据类型
字符型数据类型就是普通的单个字符，在Java中，字符型数据类型只有一个，就是char。它的存储大小与int类型相同，即2字节。
### boolean类型
boolean类型只能取两个值，true和false。它是一种轻量级的数据类型，占用一个比byte、short、int还要小的空间。
## 2.2变量作用域
变量在程序执行过程中，有三个范围：局部作用域、方法作用域、全局作用域。
### 局部变量
局部变量指的是在函数内部声明的变量，它只对当前函数有效，离开了这个函数之后，就不能再访问这个变量了。局部变量又分为形式参数和局部变量。形式参数是在定义方法的时候定义的，一般用来接收调用者传来的参数。局部变量则是在方法内定义的，可用于临时保存数据或中间结果。
### 成员变量
成员变量指的是在类的内部声明的变量，它是所有对象共享的变量，可以在不同的函数之间共享。
### 静态变量
静态变量指的是在类中以static关键字修饰的变量，它在类的所有对象间共享，即使是多个对象的同一个属性也是如此。静态变量可以通过类名直接访问。
### 实例变量
实例变量指的是在类的实例化过程中的变量，它只能由类的对象引用访问。实例变量可以通过this关键字访问。实例变量不是静态变量的一种。
### 概念总结
局部变量、成员变量、静态变量和实例变量都是局部变量的子集。但是，成员变量、静态变量和实例变量的范围和生命周期略有不同。其中，成员变量的范围是整个类，而实例变量的范围则限定于某个实例，属于对象的一部分；而静态变量的范围是整个类，也就是说，所有的对象共享同一个静态变量，静态变量只能通过类名来访问。因此，静态变量不能被实例变量重新赋值。
