# 1.背景介绍


## 1.1 什么是扫描电子显微镜？
扫描电子显微镜（Scanning Electron Microscopy, SEM）是一个利用电子显微镜对晶体材料进行成像、取样和分析的一类传感器。它的组成包括电子显微摄影设备（如曝光装置和扫尾装置），具有多种扫描方式的原子层扫描器（Atomic Layer Scanning, ALS）；还有显微镜和探测器的组合构成一个完整的结构。通常用做制备和测试晶圆材料的制品。
## 1.2 为什么要用它？
这是一种高科技高效能的现代化的成像手段，其目的是用来获取晶体物的形状、结构及各种性质信息。SEM 可以获得很高的分辨率，通过切割面上的不同区域来获取各个部分的信息。所以，在国际上被广泛应用于荧光显微镜、矿物电化学实验、水泥制品质量控制等领域。
## 1.3 有哪些相关技术标准？
目前国际上主要有ICCD标准、CMOS标准、CCD标准、TAS标准、PMTF标准、MFTF标准等。ICCD标准已经应用非常普遍。CMOS标准是在电容元件上采集亮度，采用阴极射线管（Planar Electro-Optical Integration，PEI）而不再需要带状透镜。CCD标准已经成为主流图像传感器，但在可编程性方面仍存在一些缺陷。
# 2.核心概念与联系
## 2.1 回顾
### IEC 60601-2-3电子显微术语词典
### 简单介绍一下电子显微学的概念
电子显微学的基本观点认为，在显微镜前的一个凹槽中放入被摘出的金属粒子，所出现的衍射波可以反映出晶体内的各种结构特征。这时将其称为“射出示象”或“电子显影”，即“射出”一定位置的金属粒子。晶体中存在着一些具有特殊功能的金属粒子，这些金属粒子可以通过被激发而转变到“激活状态”，在晶体表面上显示出特定颜色的反射光。这种反射光称为“电子束”。电子显微学就是研究如何在晶体上找到并使用这种特殊功能的金属粒子。由于对电子束光谱的解析力和计算能力的增长，我们越来越多地用电子显微学来研究晶体结构和功能。

## 2.2 电子显微学的基本概念
### 2.2.1 实心电子（Dry Electron）
实心电子又称无杂质电子，指电子质量小于等于60个质量单位的电子。简言之，实心电子可以独立进行特异性运动。
### 2.2.2 激活态（Activated State）
电子从实心态（无杂质电子）中释放出来后，会遇到各种固定的电子核或者其他电子。在这些电子核中的某些粒子（尤其是带负电荷的粒子）突触到晶体表面的受射层，会发生激发过程，使得这些电子的能量发生变化，产生了非金属粒子，其质量大于等于128个质量单位的“活跃电子”。在这些“活跃电子”中可以发现许多具有特殊功能的金属粒子，例如、二氧化锰（Septet），聚合酚（Perfluorooctane），六价氧基苯胺（Hexaflouride），亚砷酸盐（Acrylate）。在晶体表面，这些金属粒子呈现出特殊的蓝色、红色或黄色的颜色，称为“电子束”。因此，激活态可以理解为由某种特殊功能的激活的非金属粒子组成的晶体结构状态。
### 2.2.3 单分子摩擦（Single Atom Ferromagnetism）
单分子摩擦是指相邻的实心电子之间因为相互作用而无法自由移动。它导致某些电子的排斥作用，从而形成一块凹陷状的区域。常用的单分子摩擦实验方法是将一颗重子铁离子和铁棱镜粒子的混合物混合晶化，然后用双边共振仪测定所得结果。这种实验能够测定两种电子排斥辅助相互作用的单分子摩擦能级，并确定混合物的晶体结构。单分子摩擦还可以用波轨相干法分析晶体的基态、平衡态和倾向性相空间。
### 2.2.4 反向摩擦（Reversed Ferromagnetism）
反向摩擦是指每两个不同层之间的金属粒子都排斥彼此。当两层之间的距离增大时，反向摩擦带来的电子束就变弱，使得晶体上出现堆积的孔洞。这种现象可以用反向摩擦仪实验来观察。
### 2.2.5 软结构（Soft Matter）
软结构又称作晶体壳，是指晶体内部含有的某些几乎不能被外界所摧毁的分子，包括粒子核、岛屿、以及挡板的形成等。它既有助于晶体的功能，也可能影响晶体的特性。但是由于它们的存在，可以用半导体设备加以处理以提升其性能。
### 2.2.6 电子束曝光（Electron Bragg Exposure）
电子束曝光是指通过扫描电子显微镜进行光子束传递的过程，通过激发器激发特定类型（如偏振或垂直振）的晶体结构并从其上收集成像数据。常用的电子束曝光方法包括短光子束温度场、高功率偏振光子束、反复投影方法等。电子束曝光可用于测定晶体的体积分布、晶体界面间的电子势差、电子序数、氢键和玻璃体等性质。同时，它还可以用于化学和生物工程、药物开发、以及航空航天等领域。
### 2.2.7 电子显微镜（Electron Microscope）
电子显微镜是利用电子显微学原理来观察和分析晶体结构、相互作用、以及物质性质的一类传感器。它由曝光装置、扫尾装置和原子层扫描器组成。主要用于制备和测试晶体材料的制品、分析晶体材料的结构及功能、制备晶体镜片、以及研究晶体材料的某些性质。目前电子显微镜已经成为非常重要的科研工具。
### 2.2.8 微电子显微（Microelectronic Microscopy）
微电子显微是指用微型的、微电子部件构建的小型显微设备。这种显微设备可以在较小尺寸下（如nm级别）获取晶体结构、微观影像和结构化信息。它可以用于研究组织细胞、神经纤维、原子核等各种物质的微观结构。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 什么是电子束
电子束（Electron Bragg）是一种通过电子散射产生的电子束状结构，其由电子激发生成，具有光强和张量大小两个特性，不同的电子能带所激发出来的电子束形态各不相同。不同能带所对应的电子束的数量和位置也是不同的。电子束还具有向四周扩散的特性，即不同位置的电子束之间也会互相激发形成新的电子束。由于电子束在晶体表面被激发而形成，所以它具备了能量流动的特点。电子束的形状依赖于电子的粒子性质、以及电子激发时所在的位置。
## 3.2 电子束理论
电子束理论是关于电子激发与电子束的物理基础理论。它是一种描述电子结构以及电子媒介相互作用的理论。电子束理论认为，电子在进入晶体之后，首先经过吸收与损耗，将质子运动转换为电子运动，产生电子云。随后的逐步反馈，形成电子团簇和电子束。同一时间，可以存在多个电子团簇和电子束，它们的分布呈一定的规律。电子团簇、电子束以及相关能带之间存在复杂的关系。
### 3.2.1 电子束的组成
一般来说，电子束由以下几个元素组成：
- 电子核：以正电荷粒子为中心，质量不大，具有极大的内在势，能够合成和释放电子
- 电子带：由很多电子核组成，单个核带有多个单电子排列，带状排列，称为电子带。电子带具有明显的方向性和区分度
- 电子环：环状结构，有助于电子束扩散。含有至少三个电子核的集合。单个电子环有七个电子排列
- 电子翼：电子翼又叫双层振荡、膜气、运动薄膜。两个电子翼相互关联，形成一个电子栅格。电子栅格是电子束的组成单元
### 3.2.2 电子束的理想模型
电子束理论是以宏观层面的电子云和电子束，以及微观层面的电子团簇和电子环的观念作为研究对象，提出了一个理想模型。该理想模型认为，电子团簇以电子胞为基本单位，电子胞上包含多个电子棒，由电子团簇束缚，形成电子束。电子团簇的平均速度不大，电子团簇在电子束中沿着周期性抖动的轨迹运动。电子团簇可以分为无核团簇和有核团簇两种。电子团簇的强度由电子团簇间的相互作用所决定。电子束由电子团簇束缚而成，并且有一定的扩散范围。电子束的平均抖动速度比电子团簇的平均速度小。电子束可延伸成电子栅格。
### 3.2.3 电子束相关的量子力学
电子束理论指出，在静止状态下，电子团簇具有可观测的量子态，这表明电子团簇可以表示为波函数，可以进行量子运动。电子团簇是相互作用的产物，它们之间的相互作用受到相互作用的电子的频率和相关性的限制。当频率低于阈值时，只有特定能带的电子团簇才可能发生相互作用。另一方面，如果频率超过阈值，所有的电子团簇都可以发生相互作用，甚至可以相互影响，这表明电子团簇的相互作用不是完全随机的。当我们把晶体的平均体积看做是观察者所在位置的势函数时，电子团簇就变成了晶体的本征态。本征态可用来描述晶体的各种性质。通过观察电子团簇间的相互作用，可以推断出电子团簇的强度以及它们之间的相互作用。
## 3.3 电子显微学的主要技术问题
### 3.3.1 反射光的反演强度
反射光（Reflection）是指电子线圈受到某种电磁波击中后，放出的光。电子显微学所用到的反射光具有不同强度。一般来说，反射光越强，说明晶体的空间群中处于更低的配置序位的原子都没有受到击穿。反射光强度与晶体的空间群相互对应。对于近期比较流行的六价氧基苯胺晶体而言，空间群分别为d3h、c2v、b1u、f1g、p21/n。其中，d3h型、c2v型晶体中，反射光强度最强；而f1g型、p21/n型晶体中，反射光强度最弱。因此，晶体的空间群是反射光强度的一个重要依据。反射光的强度还与电子束的张量大小有关。张量大小越大，反射光强度就越弱。为了获得较好的反射光强度，可以采用反射小功率TEM、EPL和四极色电子显微（AFM）等技术。
### 3.3.2 激发状态的色谱
电子显微的重要目标是获得晶体结构的形状、位置和各种性质信息。激发状态的色谱（Spectrum of the Activated State）是通过记录激发状态下产生的不同颜色的光谱的方式来获得晶体结构的形状、位置和各种性质信息的一种方法。激发状态的色谱的数据，往往具有较好的分辨率，允许我们观察晶体中的各个部分以及其自旋等。除了采用激发状态的色谱外，还可以使用EMI（Electron Multiplying Instrumentation）技术获得晶体内的电子密度分布信息。EMI技术是用多条二极管对同一个光子束照射，把相同的光子束反复探测多次，就可以获得电子密度分布的分布图。电子密度分布数据能反映晶体的局部和整体的结构分布。
### 3.3.3 单分子摩擦的实验方法
单分子摩擦是研究单个电子排斥辅助相互作用的单分子摩擦能级的实验方法。单分子摩擦的实验方法包括二种：一是将一颗重子铁离子和铁棱镜粒子的混合物混合晶化，得到杂有小分子的晶体；二是将铁氧化镊粒子和钙离子混合晶化，通过侧向共振成像的原理测定混合物的晶体结构。这些实验技术帮助我们了解单分子摩擦能级的大小、分布以及向各个方向的作用。
### 3.3.4 反向摩擦的实验方法
反向摩擦是指每两个不同层之间的金属粒子都排斥彼此。当两层之间的距离增大时，反向摩擦带来的电子束就变弱，使得晶体上出现堆积的孔洞。这种现象可以用反向摩擦仪实验来观察。反向摩擦仪实验的主要步骤包括：用偏振光子束照射晶体，并拍摄成像图；用适当的场力和机械快门调整视野范围，保证观察到的孔洞大小能够适应视野范围；通过定向扫描的方式探索晶体，找出孔洞的位置和形状。
### 3.3.5 软结构的分析方法
软结构是指晶体内部含有的某些几乎不能被外界所摧毁的分子，包括粒子核、岛屿、以及挡板的形成等。软结构有助于晶体的功能，也可能影响晶体的特性。但是由于它们的存在，可以用半导体设备加以处理以提升其性能。目前已有多种软结构的分析技术，包括X射线衍射分析（XRFA）、热像相分析（RHEA）、二维材料切割（DEWETRON）、电子能级检测（ESD）、镀锌喉分析（QUANG）等。这些技术可以在一定条件下将软结构分离出来。
### 3.3.6 电子束曝光的技术原理
电子显微学的核心技术是电子束曝光。电子束曝光是利用扫描电子显微镜（SEM）对晶体材料进行成像、取样和分析的一类传感器。它的组成包括电子显微摄影设备（如曝光装置和扫尾装置），具有多种扫描方式的原子层扫描器（ALS）；还有显微镜和探测器的组合构成一个完整的结构。通常用做制备和测试晶圆材料的制品。然而，电子显微学在生产过程中的应用却相对滞后。最近，随着新型的激光、显微镜、显色衬底等产品的出现，电子显微学的应用领域正在发生剧烈的变化。基于这种变化，我国的科研人员也应当加强对电子显微学的研发，完善相关的技术措施。
