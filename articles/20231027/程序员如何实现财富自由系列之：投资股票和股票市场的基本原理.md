# 1.背景介绍



## 股票市场简介

股票市场是指股份有限公司交易所上公开买卖股份、债券、基金、房地产等产品、资本等市场活动。它是现代金融市场的重要组成部分。每天都会有成千上万个上市公司在股票市场上进行交易。在中国，股票市场上共有70多家上市公司，平均每天会发行股票数百万股，交易额达到上亿元。因此，股票市场是中国最重要的金融工具之一。

## 投资股票的目的

很多人对投资股票是什么感到困惑。或许会觉得很简单，买入一只股票就可以获得一点收益。其实不然。投资股票是为了创造价值和实现财务自由。

投资股票并不是简单的买卖。作为个人投资者，首先要了解相关法律法规，然后通过认真阅读相关书籍、网站的帮助，加上自己的分析和经验，才能把握正确的方向，抓住时机，实现最大化的投资回报。

## 为什么说投资股票比黄金更有价值？

1. 股票价格具有可预测性。我们知道股票市场的波动总体呈现正态分布，所以股票价格具有可预测性，而黄金的价格却没有这种可预测性。我们可以利用预测股票价格的工具进行投资，比如利用预测工具预测经济数据、宏观经济形势等，从而减少风险。

2. 股票的市盈率（P/E）远高于黄金的市价，而且也存在着规模效应。当股票的规模增大时，其市价也会随之增加，这就使得股票的购买力变得更强，也能带来更大的回报。当股票的市盈率过低或者停牌时，则被迫考虑其它形式的投资。

3. 股票的流动性强，能够承受短期和长期的市场波动。当股票的市场变化比较剧烈时，它的价格变化也会较快。当股票的供求关系发生变化时，它的价格也会相应地变化。

4. 股票可以保本。虽然股票市场是一个高度不确定性的市场，但股票的保本机制可以确保投资收益。如果一个公司破产重组，其股东可以分红，而不是像债券一样还本付息，这也是股票投资的一个优点。

5. 股票收益风险分散。股票投资的收益相比于债券、货币基金、黄金等投资品种来说，更容易被波动影响。因为股票投资涉及到的项目越来越多，受到宏观经济因素的影响也越来越大，因此股票投资的风险也越来越大。但股票投资的收益率相比债券投资来说，更容易分散到各项资产中去。

6. 股票投资比较安全。股票投资是一个风险比较大的投资，风险主要来自于公司的估值、业绩和政策，但也可以通过观察股票的走势、跟踪股票指数等方式减少损失。同时，股票投资还有很好的社交属性，可以结识志同道合的朋友，学习新知识，分享经验。
