# 1.背景介绍


## 1.1 大数据简介
大数据时代已经来临。大数据的定义可以从三个方面展开：数据量、数据复杂性、数据价值。随着互联网、移动互联网、物联网等技术的发展，以及社会的需求变化，大数据正在成为一种越来越重要的数据集。
### 1.1.1 数据量大
单个数据点，如一个用户点击次数、一条微博，都在数量级上呈指数级增长。企业所积累的海量数据成为了沉重的存储负担，传统数据库无法快速检索、分析，使得各个部门之间协同合作变得十分困难。由于数据量巨大，海量数据分析往往需要新型的数据处理系统及相应的算法支撑。
### 1.1.2 数据复杂性高
海量数据呈现出高度的内聚性、关联性、多样性。数据的维度增加，导致数据结构混乱，使得数据的存储、查询、处理等任务变得十分复杂。因此，对于海量数据进行有效的管理，需要对各种数据存储形式、存储方法、编码方式、压缩率、数据分区、数据一致性等进行综合考虑。
### 1.1.3 数据价值密度极高
大数据所产生的数据价值和价值的大小直接相关，其大小取决于数据本身的内容、结构、规模。对于某些领域如金融、房地产、政务等，甚至出现了价值连城的“超级大单”，而这些数据本身的价值一般情况下要远大于传统行业公司的巨额盈利。如何合理的利用大数据，实现真正意义上的商业价值，将成为迫切的问题。
## 1.2 智能数据管理介绍
智能数据管理是构建基于大数据技术的业务系统的核心要素。随着大数据处理技术的不断革新，人工智能（AI）、机器学习（ML）、深度学习（DL）等技术也越来越火热。人们越来越关注的是如何用大数据技术解决实际问题，提升工作效率、降低维护成本。智能数据管理涵盖以下五个层次：数据采集、数据处理、数据分析、数据挖掘、数据服务。数据采集通常包括日志采集、监控数据采集、数据传输、事件捕获等。数据处理则包括数据清洗、数据转换、数据挖掘、数据过滤等。数据分析主要包括数据统计、数据建模、数据挖掘等。数据挖掘通常包括模式识别、关联规则、异常检测、知识发现等。数据服务则包括数据可视化、数据分析、数据交换、数据共享等。下图展示了一个典型的智能数据管理流程：
