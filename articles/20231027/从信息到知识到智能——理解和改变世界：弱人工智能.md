# 1.背景介绍


## 概述
### 人类历史上最伟大的突破口莫过于计算机的出现，它极大的扩展了我们的认识能力，促使人类的生产效率大幅提升。然而，计算机带来的同时也给许多工程师、科学家、以及商界精英们留下深刻的负面影响。正如《蓝宇》中所写：“在当今这个全球化的时代，每个人都生活在一个巨大的舞台上，在这个舞台上，无论你的身份是什么，无论你是谁，你都可能成为新时代的冠军。”弱人工智能（Weak AI）则是一个非常热门的话题。它的出现会对人类的未来产生哪些重大影响？
## 为何要关注弱人工智能
### 当前的局面
近年来，随着人类产业的飞速发展和经济体量的迅速扩张，人工智能技术已经成为经济发展的一个支柱。但是，人工智能技术虽然在某些领域可以成功应用，但却依旧存在很多的问题。比如说：智能产品的缺乏、交通事故造成的财富损失、以及社会公平分配的不足等。如何解决这些问题，是目前值得我们去思考的课题。
### 关于弱人工智能的定义
弱人工智能是指具有一些基本功能或性能，甚至没有显著优势的智能机器，并通过其操控、学习、复制、模仿或擦除现实世界的一类人工智能。简而言之，弱人工智能仅具备部分人类智能所拥有的部分能力，因此被称为弱人工智能。弱人工智能主要分为：
- 低智能、表现差、可编程能力差或者没有意义的人工智能：如繁殖机器人、骗保机器人；
- 有一定的智能能力、部分可编程性但是远不能胜任实际工作的人工智能：如聊天机器人、虚拟助手等；
- 有一定智能、可编程能力以及实际应用价值的智能机器人：如物流跟踪机器人、智能垃圾分类机器人等；
- 模仿人类智能的有限范围智能机器人：如围棋机器人、围栏保镖机器人等；
- 只在一小部分领域能够成功实现人工智能的机器人。如运动机器人、扫地机器人、焚烧炉机器人等。
### 弱人工智能的特点
- 能力：弱人工智能一般只有很少的甚至几乎没有处理文本、图片、语音数据的能力，因此它们只能执行比较简单、重复性的任务。但由于它学习能力强，所以在学习过程中可以获得知识和技能，并进一步改善自己的操作技能，在更复杂、困难的场景中可以取得更好的效果。
- 规模：弱人工智能往往是在较小的规模上运行，因此部署、运营、维护起来比较方便。同时，在缺乏实力情况下，也可以通过大量的自主部署来提高效率。
- 空间限制：弱人工智能必须在有限的空间内运行，不能依赖于其他的机器或实体进行联合控制。如果环境变化很大，那么需要对自己进行相应的适应和调整。
- 数据：弱人工智能在运行过程中不需要收集海量的数据，只需持续地接收、分析数据就可以完成自己的任务。这样就避免了对个人隐私信息的泄露。
- 隐蔽性：弱人工智能一般都是隐藏自己背后的一些机制、算法，因此即便受到外界干扰也不会引起太大的注意。
- 可移植性：弱人工智能可以在不同的平台上运行，因此对于不同类型或规模的工程环境来说，它们的部署和应用都是十分容易的。
- 时延性：弱人工智能通常运行在较短的时间内，所以对实时性要求不高。而且，它们可以通过对一些关键事件的检测、预测等方式来自动化响应，使得它的反馈和执行速度都更快。
- 透明度：弱人工智能的一些机制和逻辑可以被观察到，但一般都是保持密封状态，不能被外界知晓。同时，由于它们的学习能力比较强，也可以通过日志文件来记录自己学习到的一些知识。
- 动态性：弱人工智能一般都是根据数据的不断积累、训练、优化等过程，不断更新自己的规则和算法，因此它的性能总是处于不断提升的过程之中。
