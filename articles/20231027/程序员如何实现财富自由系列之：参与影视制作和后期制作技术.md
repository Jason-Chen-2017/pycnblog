# 1.背景介绍


作为一个技术专家、程序员、软件系统架构师或CTO，在做项目或者从事创新工作时，都离不开媒体方面的支撑，比如拍摄电影、制作电视剧、游戏CG、平面设计等。在这些领域里，我们需要有专业的技能才能更好地完成任务，而无论是从事电影制作、设计还是后期制作都是非常重要的。但这些技能并不是一蹴而就的，还需要长时间的积累和学习。
在本文中，我将通过分享自己的一些经验以及心得，教会大家掌握好参与影视制作和后期制作的相关技能。我会尽量从宏观的视角出发，帮助大家理解影视制作的整体流程，进而指导大家正确选择自己擅长的领域，专注于熟练掌握那些可以提升效率的方法。希望通过这个系列的文章，可以帮助大家成为一名更好的影视制作人，并且达成更高的收入。
首先，让我们回顾一下影视制作的一般流程:

1.导演/策划进行构思，策划人员确定整部影片的目标。

2.摄影师根据导演意图拍摄影片素材，剪辑、调色、修饰镜头，合成声音，后期处理。

3.剪辑师负责对制作对象进行分工，把控每段素材的长度、画质，确保拼接成一个完整的故事。

4.修复师处理场地问题、消除拖尾、加入水印、减少对比度等。

5.美术、摄像师协助剪辑师进行美术创作，确保视觉效果达到标准。

6.色彩师根据剪辑师的需求进行配色，确保配色符合审美趣味。

7.剪辑师、修复师、美术师、色彩师、摄像师一起合作，拆分不同的人员工作，共同完成整个影片。

第二章节：主要概念
影视制作中最基础的概念是什么？答案是角色。
角色（Role）是指导演、编剧、导演、演员、配乐师、剪辑师、摄像师、配音师、后期制作、舞台美术、配色师等角色的组成和职能，是影视制作过程中最基本的单位。一般来说，一个角色会扮演一个特定的角色，其具体职能通常由导演、剧本作者决定。在制作电影时，角色往往具有较强的指导性作用，它将多个细小的人物组合成一个完整的故事线，给观众呈现一种全新的感受。
另外还有各种类型的角色，包括：主角、配角、支撑、科幻设定、恐怖分子、敌人、重口部件、内在情绪、人物关系等，这些角色都会影响到电影制作的不同阶段。
第三章节：前期准备
为了更好地参与影视制作，前期应该做哪些准备呢？
1.电脑硬件的要求
电脑硬件的配置对于影视制作是一个比较大的要求。我的建议是：

①配置较好的显卡（NVIDIA GTX 1080 或 AMD RX VEGA）；

②内存至少4GB以上；

③固态硬盘，能够保存大量的视频文件（如720P,1080P,4K）；

④网络条件良好，能够高速下载素材。

2.投资
投资对于影视制作者来说也是不可或缺的一环。投资可以从以下几个方面入手：

①原始设备的购买

②硬件设备的购买（配备摄像机、屏幕、耳机、麦克风等）

③软件的购买（使用3D建模、摄影修图软件、视频编辑软件、图片处理软件等）

④制作工具的购买（例如曝光控制软件、合成器、混响补正等）

3.熟悉的操作系统
熟悉的操作系统对于影视制作者来说是必须的。例如：Windows、Mac OS X、Linux等。
4.了解影视制作的基本知识
了解影视制作的基本知识对影视制作者来说是至关重要的。例如：电影制作的分类、编剧的职业道德、色彩学、相机、声音、表演等等。

1.收集素材
收集素材可以分为两种类型：

①根据剧情写的脚本

5.阅读剧本

第五章节：制作方案制定
制作方案制定的过程主要有以下几步：

1.筛选素材
首先需要筛选自己擅长的素材，并确认合适的时间点。

2.设定拍摄场景
决定拍摄的场景，在设置的拍摄角度上要保证拍摄的清晰度。

3.拍摄工作流程
决定拍摄工作的流程，包括拍摄、后期处理、编辑、校稿、混音等。

4.选择配音
选择配音主要是为了配合剧情需求，引起观众的共鸣。

5.预热工作
在工作前期充实激活自己的气场。
第六章节：采集配音剪辑
采集配音剪辑是辅助美声工作的一个环节。

1.收集素材
收集素材的过程主要包含两个部分，一是用专门的设备采集音频素材，二是根据自己音箱的性能制作配乐。

2.剪辑制作
剪辑制作主要涉及到配乐师和配音员之间的沟通协调和配音文件的精雕细琢。

3.混音制作
混音制作主要是为配音添加温度、情绪和气氛效果，使其融入整部影片之中。
第七章节：拍摄工作流程
拍摄工作流程包括如下几个部分：

1.导播剪辑
导播剪辑主要是负责导演自身的表现，包括对镜头的创意、对角色的塑造、节奏的控制、视听效果的处理等。

2.拍摄剪辑
拍摄剪辑主要是由剪辑师负责剪辑，对镜头进行布局、调整，以及拍摄过程中对声音的处理等。

3.后期处理剪辑
后期处理剪辑主要是对拍摄后的影像进行各种处理，包括降噪、美化、消除黑边、修复色差、优化曝光等。

4.拼接拆分
拼接拆分主要是对拍摄好的素材进行整体拼接，以及对不同素材进行拆分。

5.录制
录制是最后一步，通过编辑软件制作完成的素材再加上配音、音效、特效等，就可以制作成影片了。
第八章节：拍摄技巧
拍摄技巧是影视制作者们不可或缺的一项能力，这里总结了一些常用的技巧。

1.夸张拍摄
夸张拍摄是指在拍摄过程中突出某个特定的人物，打乱正常节奏，尝试拍摄出超乎常人的特殊效果。

2.误差控制
误差控制是在拍摄中控制相机运动的速度，按照既定的时间拍摄，从而保证拍摄质量。

3.明艳照片
明艳照片是指拍摄照片中的个别明亮的区域，用这种特有的照片拍摄足够的张数，可以打造出十足的色彩感染力。

4.使用预设剪辑
预设剪辑就是设置一些关键剪辑，当出现特定事件的时候自动触发，提升整体效果。

5.自动对焦
自动对焦能够大幅提高拍摄的精准程度。
