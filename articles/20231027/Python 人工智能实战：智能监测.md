# 2.核心概念与联系
## 什么是人工智能？
“智能”这个词语一直是科学界的热点话题。它既可以指机器的智能性，比如能自己学习、完成任务；也可以指人的智能性，即具备某种机器学习能力的个人。从定义上看，智能就是“机器的学习能力”，或者“人的表现力”。

目前，人工智能（AI）已经成为科技热点。2017年，美国商务部就宣布建立AI国家标准，要求所有公司、机构、组织、个人都要遵守这一标准。AI领域由两种主要分支组成——机器学习（ML）和深度学习（DL）。它们的区别在于所处理的数据类型不同，ML侧重于图像识别、自然语言处理等非结构化数据，而DL则针对高维度、多模态、海量数据的处理，是当下最流行的一种机器学习方法。

## 什么是人脸识别？
人脸识别，又称身份识别、面部识别、生物特征识别，是指根据人脸的各种外观信息（如眼睛、鼻子、嘴巴、胡须等），通过机器算法对特定用户进行身份认证。目前，人脸识别技术已广泛应用于身份鉴别、门禁系统、人脸安全、精神卫生、工地岗亭管理等领域。

### 人脸识别的应用场景
* 智能门禁系统：利用人脸识别技术，向访客提供身份验证，通过实时检测和比较来保障进入门内的安全。
* 医疗护理系统：利用人脸识别技术，自动识别患者面部，配合诊断诊断报告生成，提高患者满意度，降低损失。
* 情感分析系统：利用人脸识别技术，对用户发帖的情绪进行分析，用于舆情分析、商品推荐等。
* 人才招聘系统：利用人脸识别技术，将候选人的简历扫描上传至系统，可以进行筛选，快速入职。

## 为什么需要人脸识别？
目前人脸识别技术的应用十分广泛，主要原因如下：

1. 身份鉴别：人脸识别技术可用于身份鉴别，对拥有较高图片质量的人员进行识别和校验。此外，还可通过比较图像中不同的区域和属性，如眼睛、嘴巴、眉毛等，确认身份。
2. 社会情景：人脸识别技术正在受到越来越多的关注，它可以用于许多人类活动的计算机辅助系统。如，监控摄像头可以对人脸进行跟踪，报警系统可以通过拍照确定进入者是否是经过授权的人员。
3. 商业应用：智能手机、平板电脑、服务器等新型计算平台，均支持人脸识别技术。人脸识别技术可以用于各类商业应用，如基于人脸数据库的精准营销、人脸支付等。
4. 机器迁移学习：深度学习技术在人脸识别领域得到广泛应用，其中移动端设备中的人脸识别功能。通过将训练好的人脸识别模型迁移到其他平台，就可以实现跨平台的识别效果。
5. 个性化服务：人脸识别技术可以帮助个人定制个性化服务，如视频监控中的警示系统、语音助手等。
