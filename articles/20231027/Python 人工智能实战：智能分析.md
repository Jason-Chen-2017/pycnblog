# 2.核心概念与联系
## 机器学习(Machine Learning)
机器学习是指让计算机具备学习能力，从数据中发现模式并且改进性能的一种技术。其特点主要体现在以下几方面:
* 通过训练模型来解决特定任务，不需要编程人员手动指定规则。
* 模型能够自动分析输入数据并调整参数以达到预期效果。
* 通过反馈和迭代的方式不断改善模型的性能。
* 可以处理大量数据，有效提升计算效率。

与传统的统计方法不同，机器学习方法的目标是在数据中找到规律性，而非依赖于人的假设或经验。机器学习通常分为三种类型，即监督学习、无监督学习、半监督学习。
### （1）监督学习（Supervised Learning）
监督学习（又称为有监督学习）是一种学习方法，在此方法中，训练样本包括输入和输出，也就是学习系统接收输入之后预测输出的结果。学习系统根据已知的输入-输出对，利用算法不断修正自己的预测结果。监督学习算法的分类依据学习目标是否由标签进行标识。有监督学习中的典型任务包括分类、回归等。例如：图像识别、语音识别、推荐系统、文字识别、生物信息学。

### （2）无监督学习（Unsupervised Learning）
无监督学习是指机器学习技术用于从数据中发现隐藏的结构，无需任何先验假设或监督信号。它不需要为模型提供明确的输入/输出映射，而是通过对数据的潜在结构进行分析。无监督学习算法的典型任务包括聚类、密度估计、降维等。例如：图像聚类、视频分析、文本挖掘。

### （3）半监督学习（Semi-Supervised Learning）
半监督学习则介于监督学习和无监督学习之间，通过训练模型同时获得有限数量的标记数据和大量未标记数据。模型可以使用未标记数据来学习数据分布及特征，并根据标签信息对待标记数据进行辅助预测，帮助模型提高准确性。半监督学习算法的典型任务包括文档分类、情感分析等。例如：垃圾邮件过滤、新闻分类。

## 深度学习(Deep Learning)
深度学习是一种机器学习技术，它利用多层神经网络模型对大型数据集进行训练，模型的输出可以用来表示整个数据集的内部结构。深度学习的关键在于模型参数的组合方式。深度学习具有以下优点:
* 更好的泛化能力:深度学习模型在非线性转换层的堆叠使得模型的表达能力更强，因此可以有效地捕捉到数据的局部关系；
* 大容量和深度:深度学习模型的参数数量和层次都远大于其他类型的模型；
* 可解释性:深度学习模型的可解释性很强，因为每一层都可以表示为一组权重矩阵，通过分析权重矩阵的绝对值大小和偏置值，我们可以直观地理解模型的内部工作机制。

## 强化学习(Reinforcement Learning)
强化学习是机器学习领域的一个子领域，它使用与环境互动的方式来学习任务。强化学习系统可以选择执行哪些行为，最大化其累积奖励。与监督学习相比，强化学习不仅需要有输入输出对，而且还需要考虑系统内部状态和动作之间的关联。强化学习算法的典型任务包括游戏AI、AlphaGo等。
