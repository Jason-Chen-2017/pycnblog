# 1.背景介绍


在IC设计、微电子、机器人等领域都广泛应用着各种类型和尺寸的集成电路(IC)作为传感器、处理单元、电机驱动器、通信终端等各种功能部件的基础。而IC制造过程中涉及到的一项重要环节就是晶圆晶面材料的制造。
其中，集成电路制造过程中的半导体(即硅)作为主要制造材料占据了很大的比例，其制备工艺极其复杂。其中需要牵扯到两种工艺，即表面摩擦工艺和谐振腔工艺。表面摩擦工艺要求采用厚金属粘结或金属纤维粘结；谐振腔工艺则利用不含铁质物质的气泡形态、曲率均匀的气体来制备。
虽然早已成为IC制造领域的佳品，但其制备工艺仍存在一定的缺陷，如薄膜率低、精度低、厚度高等问题。如何提升集成电路制造的全过程中的关键环节——半导体制造技术就成为了一个重要课题。而这个课题的关键就在于理解半导体物理机制及其在制备工艺中的作用。因此，本文拟通过对半导体物理与器件原理的科普及阐述，从中推导出更多优化IC制备工艺的方法。
# 2.核心概念与联系
## 概念阐述
半导体（英语：semiconductor）指除氢以外的化合物材料，其含有少量或者没有电子的物质。在电子学上，半导体的表现形式包括金属，锂离子层，磁性介子。按照半导体的基本结构分类，可分为三种：绝缘体、有机体和超导体。

绝缘体由无线的带状缺陷构成，形成金属钢键、键的平行排列。它们表现出强大的电导性，是电子的最佳载体。这些绝缘体有很多形状和大小，从微小的颗粒状物质到大型钢管。

有机体通常包括复合物的种类繁多，并且具有相同或近似相同的化学组分，但由于内部存在相互作用的化学反应而被称为共晶体。最主要的有机半导体有单模，单模复合物是无水的，内部没有悬浮体或边界层。另外还有导体半导体，也称钙化物半导体，是一种非常特殊的有机半导体，其表面表现为导电的表面，形成导体晶体。其他类型的半导体还有贫铜，柠檬酸，金刚石，氕等。

超导体是指由两个以上硅块组成，充满能量传输且具有导电性、疏导性、热辐射性的半导体。在超导电路中，两个或多个超导体的并联结构可以实现高能耗的单色或多色光源输出。它可以大幅度提高发光效率，且不受电流限制，是构建真正的智能超导终端的必需品。

## 相关术语
### 材料参数
- 周期T：半导体中含有的二极管数量，通常为2或3个。
- 宽度W：中心极之间的距离。
- 深度D：从极区开始，器件最远处与摩擦孔的距离。
- 厚度d：半导体表面的厚度。
- 孔隙K：指在同一面积内，两极之间的空间隔离程度。
- 电负荷p：在低温条件下存在于半导体表面的电场能量。
- 导通系数n：反映材料导通率的指标，其值越高，材料的导通能力越好。

### 材料结构
- 子晶和晶体：指通过数十个甚至上百个有限晶格的无机结构构成的晶体，称为子晶；而整个晶体，又称晶体。常用的晶体有碳纳米管，硅玻璃，镍钴锌晶体，锰硅晶体等。
- 元件和器件：半导体制造过程中，集成电路是最重要的元件，即在电路板上的一个个零件。一般由各种金属、非金属、半导体组件和电极组成。器件的分类依据主要是结构，如晶体、电容、电阻、印制电路板等。
- 槽位与电极：在集成电路中，半导体作为电压源、电流源、电动势生成装置，根据作用特性不同，分为槽位电极、漏极、电感、耦合电极等。
- 开关电路：半导体器件所具有的开关特征，依赖于某些电性物质的位置分布。一般来说，开关电路的组成由电场、法拉第电阻、地磁场产生的磁场、感应电容、电感产生的电荷以及电信号传导通道组成。

### 设备特性
- 工作频率：指半导体器件能够正常工作的最高频率，以赫兹为单位。
- 闪烁频率：指两种不同类型半导体设备之间的闪烁交替频率，以赫兹为单位。
- 半导体功耗：指对半导体器件的消耗功率，以瓦特为单位。
- 容错性能：指系统中能承受一定电压脉冲的能力。
- 稳定性：指半导体器件在各种环境条件下的持续运行能力。
- 耐受性：指某种半导体器件承受的击穿、撞击、冲击等损害的能力。
- 可靠性：指半导体器件损坏时，其可保持有效工作的时间长短。
- 灵敏度：指对某些特定电平的响应速度，其单位时间内测量到的最大变化值。
- 电流增益：指采用特定的制备工艺制造出的半导体器件所具有的电流增益，即放大效应。

## 材料制备过程
目前，制备半导体制造过程的标准工艺有表面摩擦和谐振腔。表面摩擦工艺主要采用金属粘结或金属纤维粘结的方法制备，所用气体会释放出许多微弱的气泡状物质，使得珠状、乳白色的硅层结构逐渐粘聚形成密度较高的结构。谐振腔工艺则利用具有更高振动频率、容量、分散性和制造难度的气体或液滴，将各极之间结构相互嵌套，形成复杂的晶体。

表面摩擦工艺的制备过程包含以下几个关键步骤：

1. 使用热轨或液化器给金属颗粒注入有机溶液，使金属表面表面摩擦破裂，产生渣液。
2. 将渣液冲洗干净，剥去非晶体材料。
3. 取出表面摩擦后的子晶，然后用高性能的表面振动马达加快加熟过程。
4. 将加熟后的子晶进行切割，清除剩余的溶液。
5. 在新切好的晶体中再次进行表面摩擦，用不同厚度的材料粘结。

谐振腔工艺的制备过程包括：

1. 准备足够的气体或液滴，加入适当的溶液，用气压和低温蒸馏。
2. 演化过程先在各极之间摆放气泡状物质，同时对相邻的气泡进行匹配，直至各极完全嵌套。
3. 在嵌套的物质中进行切割，从而制成完好的晶体。

## 制备工艺优化方法
- 用厚度高、周期高的金属粘结晶体代替厚度低、周期低的有机晶体：摩擦制备之后的晶体中会存在许多微小的珠结和小颗粒状物质，这些会降低集成电路的制备速度。因此，用金属粘结晶体代替有机晶体，可以明显提升制备速度。
- 通过对液体/气体的选择、湿润条件的控制，提升晶体加熟速度和效率。
- 对反复切割的材料进行反复加熟，增加晶体制备的可靠性。
- 通过调整反复加熟次数、塑性材料厚度和工艺参数，减少空心、粘连、过分膨胀等问题。
- 提高导通系数、性能参数、成本效益和时间效率。
