# 1.背景介绍


社会网络分析（Social Network Analysis）简称 SNA，是一种基于网络关系的复杂系统科学研究方法。它从社会结构中提取出各种复杂模式、因果关系、群体行为等，并对这些模式进行概括、分类、可视化，使得不同个体之间的相互联系及其内在的规律更加明显。由于各种领域都涉及到与人际关系密切相关的数据处理与分析，因此，SNA 在现代社会具有广泛的应用价值。传统的 SNA 方法主要关注于静态网络结构、节点特征以及网络流动规律等方面。随着互联网信息爆炸式增长，动态网络结构、消息传递以及用户交互方式的不断变化，产生了海量的网络数据，这就需要有新的 SNA 方法来有效地处理海量网络数据。本文将从三个方面介绍大数据与社会网络分析相关的知识。首先，我们会讨论网络数据的定义、分类以及结构；然后，我们会学习一些最基本的网络分析方法以及其应用；最后，我们会介绍一些计算资源、平台以及工具，这些工具可以帮助我们处理大规模网络数据。

2.核心概念与联系
## 2.1 网络数据定义
网络数据指的是由网络结构、节点属性和边权重组成的数据集合。网络结构描述了各个节点之间关系的图形表示形式，节点属性包括节点所属的社团、角色等，边权重则刻画节点间关系的强度大小。网络数据共分三种类型：静态网络、动态网络和多维网络。静态网络指网络中的节点及其连接关系固定不变的网络，通常采用矩阵结构表示。动态网络是指网络中节点的连接关系发生变化的网络，它的特性是随时间而改变的。多维网络是指网络中节点的多元属性同时影响节点间关系的网络。

## 2.2 网络分析方法
### 2.2.1 网络拓扑结构
网络拓扑结构是指网络中节点之间关系的空间分布。主要有两种拓扑结构：配置型网络拓扑结构和链接型网络拓扑结构。配置型网络拓扑结构表现为一张节点与周围节点的连接关系图。链接型网络拓扑结构表示网络中的边缘节点和中心节点，中心节点与邻居节点连接的密集程度高，边缘节点则是短期的热点节点，中心节点聚集在周围。如城市交通网络的配置型网络拓扑结构；微博上的重要人物之间通过动态连接，构成了网络的链接型网络拓扑结构。

### 2.2.2 节点中心性与介数中心性
节点中心性是指一个节点在整个网络中所处位置的中心性。根据该中心性不同，节点可以分为以下五类：接近中心节点（prominent node），核心节点（core node），中间节点（intermediate node）， peripheral 节点（periphery node），孤立节点（isolated node）。接近中心节点是指该节点与网络的中心节点紧密相连，与其他节点间距离较短。核心节点是指该节点与整个网络的结构高度耦合，与周围节点有较大的联系。中间节点则是既非接近中心节点又非核心节点。介数中心性则是在网络拓扑结构中，节点与其他节点的最大距离的中心性。介数中心性大的节点距离中心节点越远，介数中心性小的节点距离中心节点越近。

### 2.2.3 节点关联性分析
节点关联性分析是指识别网络中相似或相关节点，判断两节点间是否存在某种联系或关联关系的方法。节点关联性分析通常采用两个节点的边的数量或权值的比较作为衡量标准，如 Jaccard系数、皮尔逊相关系数和共同邻居分析。Jaccard系数用来度量两个节点的共同好友的数量占两个节点的总好友数量的比率；皮尔逊相关系数用一个线性回归方程式拟合两个变量之间关系，其值介于-1和1之间。共同邻居分析利用节点的连接关系，计算各个节点与其近邻节点的共同邻居个数，就可以度量两个节点之间的相似度。

### 2.2.4 小世界法
小世界法是一种用来发现网络中重要节点的方法。它假定网络中不存在大量的小团体，且网络中的所有节点都存在于一个完美的完全图上，即网络中任意两个节点之间都有路径相连。由于每条路径的长度都是相同的，所以可以通过选择特定的节点来形成子网络，进而分析子网络的性质。一般来说，小世界法用于生成网络的一种近似模型，其结果可以用来理解网络的实际结构。

## 2.3 大规模网络分析平台
### 2.3.1 MapReduce
MapReduce 是一种编程模型和分布式处理框架，它适用于海量数据处理任务。其特点是在海量数据中进行分布式运算时，需要把任务分割成多个子任务，并且对每个子任务的结果进行汇总。MapReduce 可以通过编程接口方便地实现分治、合并、排序、过滤等操作。MapReduce 的编程接口是只允许 Map 和 Reduce 操作的，通过封装数据的逻辑关系来减少编程难度。

### 2.3.2 Apache Hadoop
Apache Hadoop 是一个开源的分布式计算平台，提供了一整套的基础设施，用于存储和处理超大数据集。Hadoop 将存储和处理分布在不同计算机上的海量数据集成为一项复杂的任务。Hadoop 有几个重要的组件，分别是HDFS（Hadoop Distributed File System）、YARN（Yet Another Resource Negotiator）、MapReduce、Hive、Pig等。HDFS 是 Hadoop 中的分布式文件系统，能够提供高容错性的数据存储能力。YARN 是 Hadoop 中用于资源管理和调度的模块，负责资源分配、任务监控、任务跟踪和容错。MapReduce 是 Hadoop 中一个分布式运算模型，用于对大数据进行并行运算。Hive 是基于 HQL（Hive Query Language）的SQL 解析器，用于将结构化的数据映射到 Hadoop 文件系统上，并支持 SQL 查询。Pig 是基于 PIG Latin（Pig 语言）的脚本语言，用于处理大数据集。

### 2.3.3 GraphLab
GraphLab 是一种可扩展的开源机器学习框架，用于构建、评估、使用图形结构数据。GraphLab 提供了一系列的高级机器学习算法，能够处理稀疏矩阵、实时更新的数据、海量数据的并行处理。GraphLab 通过自动处理内存、加速运算速度等功能，大幅提升了机器学习的效率。GraphLab 支持包括标准的机器学习模型在内的图模型，如 PageRank、K-means、Collaborative Filtering等。
