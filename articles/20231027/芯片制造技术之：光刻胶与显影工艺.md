# 1.背景介绍


在当下数字化进程的推进下，芯片行业正在经历一次历史性的转变。传统的裸晶片已经逐渐被一些新型材料（比如硅）所取代，而电子器件的发展也催生了新的材料（半导体、IC）。随着摩尔定律的失效，每一颗物理频率上可实现的量级都在增加，因此，芯片的规模将越来越大，制程要求也越来越高。这个时候，光刻胶技术已经成为制造高性能集成电路最主要的手段。那么，光刻胶的用途又是什么呢？它的应用领域有哪些呢？本文将详细阐述。
# 2.核心概念与联系
## 2.1 光刻胶简介
光刻胶（InkJet）是一种按照电子杂志的方式进行材料加工的技术，它利用在纤维状的电子介质中凝聚出各种微粒子，并与紫外线等各种能量相互作用形成单色或彩色的印迹。通过这种方式，将大容积的纳米纺织品制成具有不同颜色的产品，是现代印刷技术不可替代的一部分。目前，光刻胶技术在数字电路板、显示器件、激光打印机、无人驾驶汽车、医疗设备等各个领域都有广泛应用。在20世纪90年代初期，由于我国国内产业的需求远远超过对外贸易，光刻胶技术引起了巨大的关注。但如今光刻胶已经成为主流的制造方式，并且被广泛应用于多种各样的领域。 

## 2.2 光刻胶工艺流程
光刻胶工艺流程可以分为三个阶段：预处理、涂层研制、铸件熔蚀、工艺制备、 制品开发及应用。其中，预处理阶段，即将粘合剂（顺水或倒水）和碱基配料混合在一起，加入适量的无机溶质（例如氧化锌），使得粘结和混合均稳定；涂层研制阶段，则采用掺杂聚乙烯（PHE）（通常用直径1-10μm的聚乙烯钢来做成）或聚氯乙烯（PBO）（通常用直径2-7μm的聚氯乙烯钢来做成）制作电感材料，制成电感管脚（电极），并将其贴到模具中；铸件熔蚀阶段，将电感材料铸成硬件件件件，然后用倒置研磨或烘焙的方法，使得铁腐蚀，再经过浸泡，确保固化后使用；工艺制备阶段，即在使用之前，将铸件预先焊接上。然后，就可以将其作为元件，用于电子计算机的各种设备上。最后，就是应用环节，将这些元件嵌入各种电路板或者其它电子产品中，就完成了整个过程。如下图所示：



## 2.3 光刻胶的应用领域
光刻胶的应用领域从实际应用的角度来看，主要包括以下五大类：
### （1）半导体器件的制造 
光刻胶技术可用于制造绝大多数的微晶制品，如集成电路IC、二极管、三极管、照明管等；也可以用于制造高品质、高功率、低噪声的数字系统，如计算器、声卡、模式识别芯片等。
### （2）印刷品制造
光刻胶技术可用于制造大面积的印刷产品，如电子版的书籍、报刊、期刊、书画、日历、照片等。
### （3）平板显示屏制造
现在的平板显示屏和智能手机的出现，给平板显示屏设计和制造带来了新的机遇。光刻胶技术已经成为制造平板显示屏最主要的技术之一。
### （4）激光打印机、扫描设备的制造
激光打印机和扫描设备都是由不同的材料组合而成，其中包括玻璃、塑料、陶瓷、非金属、金属等材料。光刻胶技术可用于制造激光打印机、扫描设备，并提供更高的精度、更短的运行时间、更大的放大倍数、更高的输出功率。
### （5）电子产品的外壳设计
光刻胶技术还可以用于设计电子产品的外壳，如激光刻蚀仪、自动冲洗机、激光扫描仪、照相机镜头等。
