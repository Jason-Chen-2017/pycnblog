# 1.背景介绍


物理气相沉积是常规的半导体制造工艺中存在的一个问题。当导通电压超过一定阈值时，材料中的杂质和气体便会发生物理气相沉积，从而影响其性能和稳定性。物理气相沉积对半导体制造过程产生了严重的破坏作用，如频繁的缺陷开路、管路崩溃等。因此，为了减轻物理气相沉积对半导体的危害，国际上已经推行了一系列的防护措施，例如在制造工艺中加入层次化分层结构和复杂的加工工艺流程，控制设备温度、采用过滤器降低介电常数等等。然而，如何将物理气相沉积的问题转变为一个高效且有效的解决方案仍然是一个难题。目前，制造商们都在寻找新的处理方法，提升制造过程的安全性和可靠性。近几年，物理气相沉积问题逐步成为制造商面临的首要问题，许多制造商纷纷着力解决这一难题，并取得了令人满意的成果。
本文将主要围绕物理气相沉积与分子束外延之间的关系进行讨论。什么是物理气相沉积？为什么会产生物理气相沉积？物理气相沉积对制造过程有何影响？如何通过预处理、处理过程和后期维护等方式来降低物理气相沉积对制造过程的影响？这些都是本文需要回答的问题。
# 2.核心概念与联系
## 2.1 物理气相沉积的基本概念
物理气相（又称物理气）指的是一种由固态表面组织的气体或其他微观颗粒所组成的气体团，它在材料表面形成并随着流动的交叉相互作用而聚集。物理气相可以很容易地从材料表面侵入到微观层次结构里，而且会伴随着微观颗粒的形成和扩散。通过物理气相沉积的原因主要有以下几点：
- 导通电压过高
- 表面受热
- 气体内蒸发的气体粒子或极小颗粒
- 被薄膜与其它分子引起的分子间碰撞作用
- 热源对半导体电路内部物性的影响
物理气相沉积通常包括两个方面的影响，即电路性能损失和材料质量损失。电路性能损失是指由于材料表面积失去导通电流而导致的微电子的损耗、失真、爆炸、反应和扬声器件噪音等，而材料质量损失则是指表面积上物理气相带来的损耗。对于机械半导体来说，电路性能损失通常占主导地位，而且电路损耗对材料的健康危害也非常大。因此，物理气相沉积对机械半导体的制造过程构成了致命的威胁。
## 2.2 分子束外延的基本概念
分子束外延（又称分子布朗运动），是指微观层层叠加分布在晶体管或者芯片表面上的气相，具有空间方向和时间方向上的不均匀分布特征，不同层次的分子束呈现出不同的运动状态，使得晶体管表面上出现各种不同类型的色彩和粘度。分子束外延会引起电路损耗，例如信号丢失、信号波形变化、信号消失、偏移、干扰，甚至是毛刺的形成。
在PCB制造领域，分子束外延可能是造价高昂的主要瓶颈。一般认为PCB制造过程中的分子束外延可能会发生如下几种类型：
- 金属间隔层：这是由于金属表面表层表现出的特殊衍射性，使得金属之间的空间距离不同，即分子束之间的距离差异。这种空间差异往往会产生导电常数差异和导体阻抗差异，进而影响晶圆的结构和性能。
- 晶圆层：在晶圆层中，分子束可能是按照衍射的方式堆叠起来的，称之为“高分子束”，反映出材料的衬底层、表面层和互联层等各个层级之间的错位分布。
- 晶圆边界层：这是由于光栅层的缺陷和设计缺陷，晶圆边界层中存在电极对偶极双层结构，因此会吸收更多的波长。分子束的数量和导电常数在这个层级中也不同于其他层级。
- 逻辑层：逻辑层中的分子束在不同的逻辑单元之间可以进行任意的排列组合。
- 概率层：概率层中的分子束由于传导特性，也会产生空间相关性，这种相关性不仅可以影响电路的性能，还可能导致逻辑错误、电路板拧痕等后果。
因此，对分子束外延的控制显得尤为重要。
# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 处理方法
### 3.1.1 预处理
在制造之前预处理的方法，可以分为以下两种：
- 使用有效的加工工艺，如浸蚀、镀锌、熔融、烫镀、表面精磨等，来提高分子束的密度和结构。
- 在制造过程中加入环保措施，如经常清洁干净整个制造过程中使用的工具、中间液体容器等。
### 3.1.2 处理过程
#### 3.1.2.1 浸蚀
分子束的浸蚀技术是在制造过程中加入浸蚀层的一种技术。由于导通电压越高，晶格结构就会越紧密，但同时分子束的空间分布也就更不均匀，因此在浸蚀的时候需要将分子束的分布调整到均匀、正常的状态。目前常用的浸蚀方法有加热浸蚀和湿化浸蚀。
##### 3.1.2.1.1 加热浸蚀
加热浸蚀是一种简单直接的浸蚀技术，其基本思想是把分子束在低温下浸泡，然后再用高温将浸泡后的分子束烘焙出来。具体操作步骤如下：
1. 用低温（约几十摄氏度）将材料表面上所有分子束打烂、切断，在烘烤前缓慢移动整个分子束束头。
2. 将材料放入加热箱，加热至900°C，待温升至3000°C，先将分子束置于高速旋转（约每秒3000rpm）的旋转器内，等待暖化。
3. 经过几分钟后，关掉高速旋转。
4. 停止旋转后，将分子束放入烘烤室，设定恒温烘烤温度（约400-700°C）,即保持分子束的热量流不动，用快速风扇或冷却塔提供热激素，加快分子束蒸馏的速度。
5. 等待烘烤至达到预期效果，然后立刻关火。
##### 3.1.2.1.2 湿化浸蚀
湿化浸蚀属于比较高级的浸蚀技术，其基本思想是将材料在高温下加热到湿状态，然后利用湿性作为物理气相，抑制分子束的自由扩散，从而实现良好的物理气相分布。湿化浸蚀的方法包括低温浸蚀法和高温浸蚀法。
###### 3.1.2.1.2.1 低温浸蚀法
低温浸蚀法是最简单的一种浸蚀技术，基本思想是将材料在低温下表层打烂、切断，然后在高温下将打烂的表面及其周围的空间进行涂抹，将分子束分布到密集的区域内。操作步骤如下：
1. 用低温（约几十摄氏度）将材料表面上所有分子束打烂、切断，在烘烤前缓慢移动整个分子束束头。
2. 收集材料中表面容易产生分子束的位置，并设置好温度控制板。
3. 对表面进行低温浸蚀，使得分子束自然涌出，并留出足够的空隙，使得分子束可以在高温下形成孤岛。
4. 等待分子束在高温下形成孤岛，并保持15-20分钟，然后关火。
5. 对剩余的孤岛进行填充，使其尽量覆盖整个表面。
###### 3.1.2.1.2.2 高温浸蚀法
高温浸蚀法是另一种较为复杂的浸蚀技术，其基本思想是采用加湿设备将材料表面和靠近表面的空间处于湿状态，然后在低温下将表面除去，将分子束从其周围空间中分泌出来，使得分子束分布得更加密集，并且满足要求的导电常数。操作步骤如下：
1. 选择高温的加湿设备，如聚氮酸、水等，将材料表面和靠近表面的空间加湿至300-400°C。
2. 用低温（约几十摄氏度）将材料表面上所有分子束打烂、切断，在烘烤前缓慢移动整个分子束束头。
3. 设置温度控制板，保证分子束周围空间的温度低于材料的基板。
4. 用低温将打烂的表面及其周围的空间除去，分子束在涌出，并形成孤岛。
5. 通过高温的加湿设备将孤岛中的分子束重新分泌回到原有的空间中，同时在高温条件下保持孤岛稳定，不会因外部温度变化而变化。
6. 当孤岛中的分子束全部散发后，关闭加湿设备。
7. 对孤岛中的分子束进行分散，重复步骤4~6，直到完成整体的浸蚀。
8. 将材料放回原状，关火。
9. 重复上述操作，直到分子束全部浸蚀完全。
##### 3.1.2.1.3 温升浸蚀法
温升浸蚀法是一种比较实用的浸蚀技术，其基本思想是将材料在低温下表层打烂、切断，然后在高温下快速向内进行温升，使得分子束保持较好的平衡分布，并且抑制分子束自由扩散，从而达到良好的物理气相分布。操作步骤如下：
1. 用低温（约几十摄氏度）将材料表面上所有分子束打烂、切断，在烘烤前缓慢移动整个分子束束头。
2. 将材料放入预热器，等待预热至900°C。
3. 在预热后，用高温把材料表面和靠近表面的空间进行温升，使得分子束保持较好的平衡分布。
4. 等待温升达到一定程度，然后关火。
5. 重复以上步骤，直到完成整体的浸蚀。
6. 将材料放回原状，关火。
7. 重复以上步骤，直到分子束全部浸蚀完全。
#### 3.1.2.2 抑制扩散
对于微米级的分子束而言，自由扩散速度接近光速，这种情况下直接用加厚或者打磨方法来抑制自由扩散并不能起到太大的作用。因此，需要考虑通过改变分子束的温度来抑制自由扩散。目前常用的抑制扩散方法有热膨胀和控制光通量。
##### 3.1.2.2.1 热膨胀
热膨胀是指将材料表面打磨、磨光，然后在高温下对材料表面进行膨胀，使得分子束分布更加密集，从而抑制自由扩散。具体操作步骤如下：
1. 用低温（约几十摄氏度）将材料表面上所有分子束打烂、切断，在烘烤前缓慢移动整个分子束束头。
2. 将材料放入加热箱，加热至1000°C，等待热膨胀温度（约2000°C）。
3. 在1000-2000°C的高温下，对材料表面进行膨胀，使得分子束扩散得更快。
4. 等待膨胀完毕，关火。
5. 将材料放回原状，关火。
6. 重复以上步骤，直到分子束全部浸蚀完全。
##### 3.1.2.2.2 控制光通量
控制光通量是通过改变分子束的温度来控制其自由扩散。控制光通量的方法主要有以下三种：
- 悬浮光通量法：通过浸蚀和慢慢移动分子束，使其悬浮在晶体管表面，并逐渐增加它的光通量，从而抑制自由扩散。
- 热膨胀光通量法：通过将晶圆表面沾满气体，并在高温下对材料表面进行膨胀，抑制分子束自由扩散。
- 中心直线光通量法：通过让分子束沿着中心直线扩散，并使其在限定的时间范围内保持固定导电常数，从而减少自由扩散。
#### 3.1.2.3 低温处理
低温处理是指在物理气相沉积的过程中，在材料表面涂抹一些低温下的气体，使其沉积到导电层，抑制物理气相分布。目前常用的低温处理方法有固化处理、电阻处理、溶解处理。
##### 3.1.2.3.1 固化处理
固化处理是将较低温下的气体固化到导电层里，从而抑制物理气相的扩散。具体操作步骤如下：
1. 在制造过程中加入低温固化剂，如氮化钾，在晶格结构和导通层电介质上均匀搭建低温固化层。
2. 在固化层上铺设适量的低温气体，并进行连续的电阻测量，调整电阻至合适的值。
3. 重复步骤二，直到低温固化层和基础材料完全结合。
##### 3.1.2.3.2 电阻处理
电阻处理是将低温电阻材料加入导电层，通过电阻材料抑制分子束的自由扩散，从而达到物理气相分布的目的。具体操作步骤如下：
1. 在制造过程中加入导电材料，如导电陶瓷玻璃片。
2. 制作电阻层，在晶圆表面打造高导通电阻层。
3. 将电阻层下方的导电材料与基本材料平滑连接，确保导电层的一致性。
4. 将电阻层放置于烘烤室，进行热处理。
5. 等待烘烤温度达到300-500°C，然后关火。
6. 将电阻层下方的导电材料取出，并对电阻层进行除腐处理，确保电阻层没有积留的固化物。
7. 重复以上步骤，直到整条导线都被电阻层所覆盖。
##### 3.1.2.3.3 溶解处理
溶解处理是通过在制造过程中加入氧化剂等使得离子结晶到达高温，从而促使物理气相的聚合，最终达到物理气相分布的目的。具体操作步骤如下：
1. 在制造过程中加入氧化剂，如异型氧化钾。
2. 在导通层电介质上均匀搭建充氧层。
3. 将材料与充氧层连接，烘烤至温度达到300-500°C。
4. 在烘烤时，注意使材料在充氧层中均匀分布，从而避免产生孤岛。
5. 当烘烤温度达到300-500°C时，关火，将充氧层取出。
6. 将材料与充氧层分开，对材料进行除腐处理，以免造成墒土质。
7. 重复以上步骤，直到物理气相分布。
#### 3.1.2.4 温控
温控是指在处理过程中控制分子束的温度，避免分子束因温度变化而扩散。目前常用的温控方法有膨胀温控、均匀温控、暂态温控。
##### 3.1.2.4.1 膨胀温控
膨胀温控是指在使用膨胀浸蚀工艺时，控制分子束温度的大小，使得分子束在固定温度下具有较高的导电常数，从而抑制自由扩散。具体操作步骤如下：
1. 将分子束置于低温浸蚀，使其稳定下来。
2. 在低温下对材料表面进行膨胀，并以固定温度（如250-300°C）进行温控，使得分子束的温度达到指定值。
3. 以高温继续膨胀，使分子束膨胀得更快，保持温控。
4. 等待膨胀完毕，关火。
5. 重复以上步骤，直到分子束全部浸蚀完全。
##### 3.1.2.4.2 均匀温控
均匀温控是指在使用热膨胀工艺时，控制分子束温度的大小，使得分子束具有平均的导电常数，从而达到良好的物理气相分布。具体操作步骤如下：
1. 将分子束置于热膨胀，使其稳定下来。
2. 在固定温度（如250-300°C）下进行温控，使得分子束的温度达到指定值。
3. 重复步骤二，直到分子束全部具有相同的温度。
4. 等待均匀温控结束，关火。
5. 重复以上步骤，直到分子束全部浸蚀完全。
##### 3.1.2.4.3 暂态温控
暂态温控是指在处理过程中，首先在一定温度下使分子束保持稳定，然后采用慢慢增加温度的方法，使分子束温度逐渐升高，最后达到制备的温度，达到物理气相分布。具体操作步骤如下：
1. 将分子束置于热膨胀，使其稳定下来。
2. 采用平均温度（如250-300°C）进行温控，使得分子束的温度达到制备温度。
3. 在制备温度（如2000°C）下以慢慢增加温度的方法，使分子束温度逐渐升高。
4. 等待升温完毕，关火。
5. 重复以上步骤，直到分子束全部浸蚀完全。
#### 3.1.2.5 后期维护
在制造后期，还需要进行维护工作。其中最重要的工作就是修复漏气、缝合故障、涂层漏失和吊装等。修复漏气可以通过某些高温涂层等方式来实现，缝合故障可以通过给导线施加一定电流使其摩擦来修复，涂层漏失可以通过将材料涂层的厚度修补到完整，吊装可以通过用机器吊住整个板子来实现。
## 3.2 算法模型
### 3.2.1 单芯片实时沉积检测
#### 3.2.1.1 模型描述
物理气相沉积的检测模型基于物理气相流体动力学和相空间无量纲的假设，将物理气相沉积过程抽象成流场形式，即从空间点i点到空间点j点的物理气相存在一个流函数$\psi_{ij}(t)$，表示流速$v_{ij}(t)$和流向$u_{ij}$，其中速度$v_{ij}(t)=|\psi_{ij}(t)|\times u_{ij}$。根据流体动力学的基本假设，流函数满足泊松方程：
$$\nabla^{2}\cdot(\mu \nabla\psi_{ij})+\nabla p=\rho f_{ij},\quad t>0$$
其中粘度$\mu$、压强$p$和密度$\rho$是已知的常数；微元面元法则表示由微元面元$m_k$组成的流场$\psi_{kj}(x)$满足光滑；面元密度$\rho_k(x,y,z,\delta)$由密度场$\rho_k=f(\hat{\rho}_k)$和粘度场$\mu_k=f(\hat{\mu}_k)$定义，$\delta$是一个衍射度的正比系数；$f_{ij}(\theta,\phi)\equiv-\nabla\cdot u_{ij}+J_{ij}(\theta,\phi)$，即总位移$\nabla\cdot u_{ij}$与光化当前$J_{ij}(\theta,\phi)$的关系。另外，局部湍流分量由沿$x$轴的速度为$w^x_k(x,y,z,\delta)$、沿$y$轴的速度为$w^y_k(x,y,z,\delta)$和沿$z$轴的速度为$w^z_k(x,y,z,\delta)$表示，其中$\delta$是一个衍射度的正比系数。沉积过程中，通过边界条件，流场$\psi_{ij}$与外界无因果关系，并且是闭合区域，两端流函数值同号。则沉积过程可以表示为流函数的变化过程：
$$\frac{d}{dt}\psi_{ij}(t)=-\Delta_{ij}\nabla^{2}\psi_{ij}(t)+\nabla p_{ij}-\left[F_{\text{adv}}+\sum_{kl}^{N}D_{kl}\psi_{kl}(x)w_{kl}^x(x,y,z,\delta)+\cdots\right],\quad x\in S_{ij}$$
其中边界流函数$\psi_{ij}=0$；电场$\vec E_{ij}(t)$可以由任意的传输方程获得，但应当在其边界上保持固定不变，即$\psi_{ij}=c\int_{S_{ij}}\vec E_{ij}(t)n_{ijk}\mathrm{d}s$，其中$S_{ij}$是流线$ij$的边界；磁场$\vec B_{ij}(t)$可以由任意的传播方程获得，但应当在其边界上保持固定不变，即$\psi_{ij}=c\int_{S_{ij}}\vec B_{ij}(t)m_{ijk}\mathrm{d}s$；温度场$T_{ij}(t)$可以由理想气体计算获得，此处假设温度场是固定的，即$\frac{dT_{ij}}{dt}=0$；压力场$P_{ij}(t)$也是已知的；$D_{kl}$、$F_{\text{adv}}$是已知的加权系数。此时的求解方法是时空积分。
#### 3.2.1.2 检测策略
物理气相沉积检测策略可以分为三个层次，即目标选择层、检测参数选取层和检测算法层。
##### 3.2.1.2.1 目标选择层
物理气相沉积检测的对象是导电层中的微米级分子束。选择这样的目标主要有两个原因：一是微米级的分子束具有稳定和均匀的导电常数，可以用于测试电容的性能；二是实验平台限制了检测对象的数量。一般的实验平台能够检测到的分子束不超过100万个。
##### 3.2.1.2.2 检测参数选取层
物理气相沉积检测参数的选取需要结合电路性能、检测性能和成本考虑等方面进行。
- 电路性能：物理气相沉积对电路的性能影响可以测量为电容串联长度、导通电流、溢流电压等指标。
- 检测性能：物理气相沉积检测需要仔细设计检测算法，比如检测窗口、信号处理等。
- 成本考虑：检测设备的制造成本、检测工具的购买和维护费用、检测结果的分析和检索成本等。
##### 3.2.1.2.3 检测算法层
物理气相沉积检测算法一般分为基于时域和基于事件的两种类型。基于时域的算法采用积分形式进行物理气相沉积的监测，它能获得较高的检测精度和持续时间，但是由于仿真计算复杂度高，无法在实时环境中运行。基于事件的算法采用边缘检测、特征提取和分类算法进行物理气相沉积的监测，它能在实时环境中运行，但由于运算代价高，只能获得较低的检测精度。
### 3.2.2 整芯片电路实时检测
#### 3.2.2.1 模型描述
由于芯片制造技术高度依赖于PCB设计，因此检测整个PCB电路中的物理气相沉积是一项重要研究课题。提高PCB物理气相沉积检测的准确性与广度同样是一件具有挑战性的任务。
