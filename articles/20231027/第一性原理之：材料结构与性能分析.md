# 1.背景介绍


“第一性原理”（First Principles）是一个现代工程学术语，其首字母的原意为“first”，意指“首先”。在工程领域中，第一性原理研究的是实体物质及其组成的基本规则。这一观念深刻地影响了工程学科各个分支的发展，包括理论物理、数学、材料科学、机械设计、电气工程、计算机科学等等。一般认为，物理学的真相就在于第一性原理，而材料科学的真相则在于利用第一性原理构建材料的构型、特性、结构和性能。

随着材料科学技术的进步，传统的手段无法应对日益复杂的材料生产过程。作为物理学的一个应用领域，材料科学在解决人们遇到的各种材料问题上取得了长足的进步。材料科学的发现和应用，使得生产制造环境和生活方式发生了极大的改变，也是物理学发展至今的一项重要领域。近年来，随着信息技术的飞速发展、人口老龄化加剧、资源不断向富裕阶层倾斜，环境污染、食品安全、卫生健康等方面的问题越来越突出。这呼唤着我们要以第一性原理作为新的思维模式，正视并改善材料科学的发展方向。

在材料科学中，最基础的研究对象就是材料，由此产生了材料结构性能等一系列问题。材料结构性能主要从两个角度进行研究：一是从原子或分子到整个材料的结构和性能；二是从材料本身的物理特性、功能特性及其与人体的相关性等方面进行研究。结构性能研究有利于理解材料的功能、性能特征以及人体与材料之间的关系。结构性能问题通常包括主导材料结构与性能的因素、影响这些因素的结构形式、性能的稳定性和变异性、以及不同阶段的材料性能的演变情况等。

虽然材料结构性能是第一性原理的一个重要组成部分，但是材料结构性能的讨论往往比较枯燥，很少有学生可以真正把握材料结构性能的全貌。如何用简单、易懂的语言将材料结构性能表述清楚，并运用多种数学模型描述该材料的预测和评价方法，是一个较难但也非常有必要的任务。

# 2.核心概念与联系

## 2.1 结构性能测量方法
结构性能测量方法是指利用第一性原理建立的模拟建模、实验仿真或其他模型计算方法来确定材料的结构和性能的一种方法。目前主要有两种结构性能测量方法：

1. 主动孔径法：利用高频率激光束记录下材料的结构和强度分布，通过解析理论模型求出材料的主导结构，并根据测定的主导结构将材料组织成不同的性能层次。

2. 无损纠缠态法：在晶格常数变化的情况下，通过调整工作频率、功率、电流、压力、温度等条件，调节材料的机械、热学和电学属性，对其进行纠缠和转移，直到实现良好的性能。

## 2.2 熔融行为及其特性
熔融行为是在金属、非金属或半导体中使某些离子从一种状态转变为另一种状态的物理现象。熔融行为主要涉及物理性质与功能的转换，如从溶质中提炼出来的氢和气、从金属中形成电子、氮的降解、钛的锶聚合等，都属于熔融行为。熔融行为在材料科学中的重要作用是影响材料结构性能。

熔融行为可以分为以下几类：

1. 扩散行为：由于各种各样的气体粒子不均匀地分布在空间中，导致不同离子的起始位置处于空间中的不同位置，称为扩散行为。

2. 分聚行为：对于单独的离子，其位置远离其他离子时，会因受到其他离子的影响而发生分散行为，称为分聚行为。

3. 集团行为：在熔融体内部，由于物质的不均衡分布导致一些离子的紊乱，使得原有的结构发生变化，称为集团行为。

4. 气泡行为：当密度增加、粘度减弱时，会形成气泡，从而引起扩散和分聚行为，称为气泡行为。

5. 增大机架热浴：如果粒子越来越密集，那么存在于粒子之间的相互作用就越来越弱，使得引起的结构紊乱越来越大，称为增大机架热浴。

6. 电子交换：当离子处于不同状态时，可以通过电子的动能交换而被转移，从而改变其自旋，形成与真空状态不同的局域性，称为电子交换。

7. 电气相干：在多层铁电结晶中，一层电路负责产生半导体的半导体半波，另一层负责导通连接，这样就形成了一层金属立方体结构，称为电气相干。

8. 金属金属配位：在金属材料的装配中，不同类型的金属之间需要配合才能实现相对稳定的结构，称为金属金属配位。

## 2.3 导带和介电层
导带（BandGap）是指电子双极共存时的磁场范围，导带宽度小于极间距，大于金属半径，介电层（Insulator Layer）则是指在导带外存在的介电介质。

在传统的热传导结构中，绝缘材料、二氧化硅、陶瓷等都是导带介电层。在金属材料中，锗等多种元素在低温下的表面存在着电击、热膨胀等现象，这种现象是因为这些元素的低温平面出现着绝缘层所致。在这类材料中，可以看到介电层的存在，是介电行为的结果。除此之外，还有一些非金属材料存在着负载辐射，比如铝箔、石墨烯等。这类材料没有介电层的存在，属于真空状态。
