# 1.背景介绍


物理学一直有个梦想就是建立统一、系统的牛顿力学体系。从希腊哲学家那里获得了“三大基本定律”，包括万有引力定律、克服摩擦力定律和电动势定律。这是相当重要的一个理论体系，它在近代物理学发展中扮演着举足轻重的作用。但是，随着时代的进步，很多新的物理现象已经出现并被提出，这些新的现象在一定程度上破坏了已有的牛顿力学框架。这些新的现象包括爱因斯坦力学方程、量子场论、微观粒子性质以及量子纠缠等。其中最重要的是弦理论与超引力。弦理论和超引力是两个新现象，它们将带来一个全新的宇宙观。本文主要讨论弦理论与超引力。

弦理论认为，任何一种非均匀振动，都可以看作是多个弦相加而形成的。而每个弦则可以看做是无限长的一条直线。因此，弦理论认为在一切运动中都存在着弦。这个理论认为运动是由弦在空间中自由滑过构成的。

弦理论的基本假设是弦的长度是不断减小的，也就是说，每当弦越来越短，就相当于把运动的长度拉长了。这也是弦理论称为弦理论的原因。

超引力是指引力远超过普通的物理常数引力。一般认为，引力是物体受到外力作用时发生的震荡现象。通常认为物体的引力大小取决于其距离，而超引力则表示这样的距离更为遥远，甚至可以超越光速。超引力使得普通的力学原理无法描述这种强大的力量。

超弦理论认为，任何一种非均匀振动，都可以看作是具有不同频率的弦相加而形成的。弦的数量越多，就意味着弦的频率越高，就越类似于超弦。因此，超弦理论认为在一切运动中都存在着超弦。

超弦理论的基本假设是弦的数量和频率是不断增加的，也就是说，每当弦越来越多，频率也会相应增大。超弦理论在某种程度上解释了电磁场中磁感应强度的变化规律。

除了弦理论与超弦理论外，还有其他一些物理学理论试图解释各种奇特现象。例如，费米-艾里克罗伊尔理论认为，极化的光子可以通过两极之间的偶极相互作用产生磁场。另一种理论认为，微观粒子的大小比正常物质大得多，这些粒子在高速运动中遇到其他粒子的情况，可导致其分裂为两个较小粒子，这种现象称为惰性粒子霍尔效应。再如，马克思说，宇宙是一个自然法则的世界，即任何东西都是以少量微小的粒子为基础组成的，这些粒子组合起来共同产生出整个宇宙，而宇宙中的所有东西则遵循着相同的规则。但随着时代的推进，人们发现越来越多的物理现象，比如强子对物质的干涉，质子对能量的吸收，弱核聚变反应以及涡居天体等等。而物理学正是由于它的严谨性、充满创造力和巧妙的方法才逐渐成为科学研究的热点。

# 2.核心概念与联系
## 2.1.弦理论
弦理论认为，任何一种非均匀振动，都可以看作是多个弦相加而形成的。而每个弦则可以看做是无限长的一条直线。因此，弦理论认为在一切运动中都存在着弦。这个理论认为运动是由弦在空间中自由滑过构成的。

### 2.1.1.弦
弦是物体的运动的基本单元。一条弦可以看做是一个微小的绳子。一个绳子要保持原长，如果缩短或拉长，就会出现弯曲现象。这时候的弦就是一条长度不断减小的绳子。一条弦就是运动的一个基本单元。弦是物体运动的一个基本元素，它是运动的一个基本单位。

弦的长度如何才能保持不断减小呢？答案是通过多个弦相加实现。一个运动过程可以看做是多个弦相加得到的。在一个运动过程中，每个弦的长度都会逐渐减小。总之，弦理论认为在一切运动中都存在着弦。

## 2.2.超弦理论
超弦理论认为，任何一种非均匀振动，都可以看作是具有不同频率的弦相加而形成的。弦的数量越多，就意味着弦的频率越高，就越类似于超弦。因此，超弦理论认为在一切运动中都存在着超弦。

### 2.2.1.超弦
超弦是由具有不同频率的弦相加所形成的曲线。当我们观察一个曲线，总是可以看到类似于一个蝴蝶的形状。蝴蝶的翅膀上的绿色丝毫没有弯曲，但是中间的弦却突然变长，逐渐变曲。这就说明，一个运动的曲线可以由多个弦的叠加形成。这个理论认为，弦的数量越多，就意味着弦的频率越高，就越类似于超弦。

超弦理论的基本假设是弦的数量和频率是不断增加的，也就是说，每当弦越来越多，频率也会相应增大。超弦理论在某种程度上解释了电磁场中磁感应强度的变化规律。
