# 2.核心概念与联系
## 2.1 价值投资
价值投资（Value Investing）是指以股票市场上常见的每股净值的计算方式来分配资产组合中的资产，其目标是在特定的时间段内最大化或最小化某种特定风险的预期收益。投资者通常根据自己的理性和眼光判断和衡量企业的价值，并寻找一种比普通股票更好的投资标的。同时，还要注意以正确的时间尺度来评估收益率、分配资金，避免在短期波动时过度注重投资组合的整体收益。

## 2.2 投资分析工具
投资分析工具（Investment Analysis Tools）是指专门用于分析和研究股票、债券、货币市场、基金及其他财务产品的专业软件、硬件或互联网平台。它的功能一般包括：（1）历史交易数据的收集、分析、处理和展示；（2）财务数据分析、评价、预测；（3）投资建议、策略研究、仓位管理；（4）投资者关系维护。

## 2.3 天使投资
天使投资（Seed Funding）是指向创业者、初创企业或者其他寻找财富增长机会的人提供资金。由于这些企业往往具有高速发展的潜力，需要资金来持续发展壮大，但是这些企业也容易面临风险。所以，通过给予这些企业少量资金或融资，来吸引投资者来加入到这个过程中来，这就是所谓的天使投资。最常见的天使投资的方式有两种：A轮和B轮。

### A轮（Pre Seed）
A轮投资的目标是在最早的阶段募集资金，主要通过VC、私募、银行等机构向创业者提供资金。A轮的资金注入后，在一段时间内持续向前推进，最后实现企业的发展。在A轮，投资者获得的回报主要来自于初始的资金，当企业经过一段时间后发现增长效果不好，则出现退场甚至破产的危险。A轮的资金比较有限，一旦企业破产，投资者就只能拿着企业的损失赔付了。

### B轮（Seed & Series）
B轮投资的目标是要在B轮之前的基础上，进一步扩大融资规模，通过发起新的联合投资、引入第三方投资机构等方式，来提升资金规模，实现更大的盈利。B轮的资金注入之后，投资者可以期待企业的快速成长、上市发行等。此外，投资者也可以参与到企业的管理团队、参与决策制定中来。通过B轮投资，投资者可以继续为企业带来持续稳定的收益，甚至还能获得B轮投资项目的折扣。

## 3.2 风险筛选
项目以AI（人工智能）领域为主要方向，因此，风险因素主要有以下三个方面：

1. 技术难度
项目的技术难度较大，需要研究人员持续投入研发。同时，还要考虑到投资人的技术储备、对相关技术的理解以及职业道德。

2. 法律风险
项目涉及许多复杂的法律问题，包括合同争议、知识产权保护等，投资人应对这些问题多加小心。另外，如果项目涉及到实体经济，则需要注意实体经济的法律风险，包括税收、国安等。

3. 竞争风险
由于是建立在现有生态之上的，并且目前已经形成了大批企业和服务供应商，项目的竞争风险也十分高。因此，投资人应充分利用信息的优势，提高信息获取的速度，为项目出谋划策。

基于以上三大风险因素，投资人需要设置较高的标准来筛选项目。其中，技术难度和法律风险是可以接受的，而竞争风险则需要投资人特别关注，因为即使投资人能够降低它的影响，也不能完全消除它。

## 3.3 投资目标确认
基于对风险的识别，投资人可以制定如下四个目标：

1. 解决技术难度
项目的技术难度较大，因此，投资人应该着重解决这个难题。投资人应该根据自己的专业技能，配合开发人员和数据分析师等职业人员，解决算法、机器学习、深度学习、人工智能等技术难题。同时，投资人也应该积极参与竞赛活动，尝试提高自己的技术水平。

2. 扩大应用范围
项目涉及多个领域，因此，投资人要充分利用其优势，扩大应用范围。投资人应该在人工智能领域建立起全面的布局，扩大其应用边界。例如，项目可以开发可穿戴设备、助听器等；还可以开拓医疗领域、人力资源领域等。

3. 提升盈利能力
项目得到的投资额有限，而且盈利模式还不确定，因此，投资人需要快速找到新的盈利模式。投资人可以通过调研、试验等方式，寻找新的盈利模式，提升盈利能力。例如，项目可以推出区块链服务，使得企业的数据保持私密性，并提供安全保证；还可以开发电子交易系统，让消费者能够在线下和线上完成交易。

4. 建设伙伴关系
项目涉及各个领域之间的合作，因此，投资人要构建起共同的伙伴关系。投资人应该合作更多的企业、投资机构和技术专家，共同打造伙伴关系，开展合作。例如，项目可以搭建商业数据共享平台，使得企业之间数据能够互通；还可以与其他领域的公司合作，提供更专业的咨询和服务。

综上所述，投资人通过构建完整的投资框架，并定义清晰的投资目标，可以有效地筛选出适合自己风格和能力的项目。

## 3.4 资金筹措
投资人应该根据自己的意愿进行资金筹措。可以采用以下方式筹措资金：

1. 投资型基金
投资型基金是指以市场价格或加权价格买卖证券、债券或其他固定资产的基金。投资型基金通过购买、贩售、经纪、托管等行为获得投资收益。投资型基金投资风格简单、易管理，且投资收益率稳定可靠。

2. 跟踪型基金
跟踪型基金是指投资基金的一种，按设定的规则跟踪市场，按规定分红。跟踪型基金的投资风格偏于长期价值投资，其净值为股票市场的市盈率乘以份额加上基金的管理费用和托管费用。跟踪型基金可以承受较大的基金经理人、基金经理、管理层和客户的投资风险。

3. 回报率型基金
回报率型基金是指投资基金的一种，以加权的方法，根据投资人的历史业绩定期分派定额资金，以获得一定回报。这种基金的投资理念偏向于长期价值投资，以达到业绩显著提升的目的。回报率型基金的资金来源一般来自于所投资的基金经理人的薪酬，但也有例外。

4. 策略型基金
策略型基金是指投资基金的一种，以投资人的判断和预判，将资金投向可能产生最大收益的方向。策略型基金以不同的策略来优化基金的配置，以实现预期的投资目标。策略型基金的投资风格略显复杂，需要一定的专业知识才能正确运用，也存在较高的风险。

综上所述，投资人可以选择合适的投资工具来筹措资金，包括投资型基金、跟踪型基金、回报率型基金、策略型基金等，通过对不同类型的基金及其优缺点的了解，投资人可以根据自身的要求，制定更适合自己的投资方案。

## 3.5 实际操作
基于前述的投资理念和方法，投资人可以通过以下方式来进行实际操作：

1. 选择早期投资对象
项目成功的关键是早期的参与者，因此，投资人需要选择能够取得突破性进展的企业作为第一个项目。可以从大学、科技企业、国有大中型企业、创业企业等角度出发，寻找创业者们的个人资料、简介等资料，了解创业公司的发展状况和规划。

2. 发起募款申请
对于优质的创业者来说，可以进行预收费的申请。通过预收费的方式来筹措资金，可以降低创始人的风险。另外，也可与当地的政府部门或商业组织合作，直接向有需求的企业提供支持。

3. 创建高校行动网络
与顶尖大学合作，设立高校行动网络。高校行动网络包括了各高校的学生、教授、企业家等群体，帮助大学生和创业者结识彼此，与创业者交流信息，为创业者提供启发、资讯和协助。

4. 拓展社交媒体
除了在学校开展行动之外，还可以在社交媒体平台上建立办公室，推广项目消息。推送的形式可以包括推文、视频、演示文稿等，吸引更多的投资人加入到项目中来。

5. 召开年度招募会
将项目的信息透露到创业者的朋友圈中，与他们进行交流。邀请他们参与到项目中来，并询问他们对项目的关注程度、热情、期望收益等。

投资人可以参考以上五条操作步骤，一步步地推动项目的实施。
