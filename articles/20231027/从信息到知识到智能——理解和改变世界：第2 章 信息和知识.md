# 1.背景介绍


信息和知识是一个非常宽泛的话题，它涵盖了从感官世界到认知科学、计算科学、人工智能、语言学等各个领域。在这个时候，如何更好的获取、整合和处理信息和知识，确实成为一个非常重要的问题。正如莎士比亚所说："欲知幸福之道，先须省悟。"

什么是信息?信息就是可被加工处理的东西。在我们日常生活中，很多事物都可以用信息的形式进行描述，比如文字、图片、视频、声音等等。我们每个人都在不断地产生和接收信息。但是由于有限的信息量，人们对信息的认识还是有很大的限制。所以，要想了解更多的信息，就需要对信息进行加工处理，转换成其他形式的知识或者意义。

什么是知识？知识就是经过加工处理后的信息，它能够帮助我们做出各种判断、决策、行动和形成观点。例如，某个词或句子的意思、事件的原因分析、人物的性格特点、系统运作规律等等，都是由多种信息组成的复杂的知识。

什么是智能?智能可以指机器具有的某些能力，可以让机器自己做一些决策、理解和分析，甚至解决复杂的任务。目前，智能的研究越来越火热，因为人类的进步让智能机器人越来越接近自然界中的物体。但是，仅靠智能还远远达不到我们的需求。因此，如何结合信息和知识的力量，让机器学习、提升自己的能力，也是一件值得探索的课题。

# 2.核心概念与联系
## 信息
### 定义
信息(Information)又称为信号、数据或消息，它是可被加工处理的东西。信息既可以来自于物质世界（物理世界），也可以来自于精神世界（心理世界）。信息的来源包括：物理世界的各种现象、材料、能量、化学物质、生物、生态系统；精神世界的情绪、感觉、灵感、思维活动、言谈举止、行为习惯等等。

### 分类
信息的种类繁多，按大小分为两种：可转移的信息(transmissible information)和不可转移的信息(intransmissible information)。

- 可转移的信息：电磁波、光电信号、红外线、超声波、无线电信号、互联网等。这些信息可以在空间和时间上任意移动、复制、传输。
- 不可转移的信息：信件、短信、电话、传真、便签、书籍、摄影、照片、文字、声音、视频等。这些信息一般存在一定规模，不能在空间和时间上移动、复制、传输。

通过互联网传输的信息属于可转移的信息，而通过现实世界传递的信息属于不可转移的信息。
