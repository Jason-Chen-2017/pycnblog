# 1.背景介绍


## 什么是封装材料？
封装材料(又称厚涂层)是制作硅基电路板和集成电路板上重要的环节。封装材料主要由无机介质(如酒精、氮化物等)或有机介质(如陶瓷、玻璃、水泥等)制成。不同颜色的封装材料,可以赋予不同的含义。主要分为以下几类:

1. 金属密封层(又称绝缘子、绝缘层): 这一层主要包括硫化铝、苯乙烯、绝缘聚乙烯等金属,用以填充电路板表面上的凹洞、开孔,保护元器件的电气特性。在硅基封装电路板中一般采用3-4层金属密封层。

2. 膜(又称环氧树脂膜、EVA沉积层): 在集成电路板上,膜层是非金属层,在母片与其周围的信号之间架起一条透明、无色、可塑性强的导线。膜层一般与膜膜板或其他补偿材料混合后,上挂在封装层。

3. 漏极二极管(LED/LDO)(又称微透镜): LED电源的核心是一个微透镜。在封装层上,通过层压式光栅注入微型透明层(例如苯甲醛、酒精胶膜),形成单色光条,然后电流通过微透镜从微型透明层反射出来。由于电流由微透镜直接到达目标点,所以它具有很高的电阻率,即使某些电流较弱也不会损坏LED。

4. 晶圆(玻璃晶圆): 晶圆层是玻璃晶圆材料,用以填充电路板表面的开孔。玻璃晶圆一般表面有一层包覆层或表面防护层,可以减轻封装层对电路板造成的扭伤。

5. 多层填充层(填充电孔): 通过多层填充层将表面缺陷填充完整,然后利用膜来设计电路连接。

## 为什么要用封装材料？
封装材料的主要作用有以下几个方面:
1. 提高电路板的表面抗弯曲能力,避免刮孔、磨损等安全隐患。

2. 抑制电路元件短路的能力,保证电路的稳定运行。

3. 提高封装层的物理性能和导电性能,降低封装层对电路板造成的扰动。

4. 增加封装层对外部环境的适应性和抗干燥性,防止过载灾难发生。

5. 增强集成电路板的连续性和抗腐蚀性,保证集成电路板长时间处于密封状态。

6. 实现集成电路板上芯片之间的互联,提升集成电路板的性能。

## 封装材料分类
根据封装材料的功能、材料成分、寿命周期等因素,将封装材料分为两大类——无隔热材料和有隔热材料。
### 无隔热材料
无隔热材料指不带有电离辐射的金属与金属氢的组合,如硫化铝、硫化钴、三氟化硫、四氟化碳、铁橡胶、丙烯腈、乙炔、银粉等。这些材料天然具备良好的耐焊性、耐候性、耐温性及其他一些电性特征,可以作为电路板的绝缘材料。

### 有隔热材料
有隔热材料指采用带有电离辐射的金属与金属氢的组合,如绝缘聚乙烯、绝缘棕榈、晶圆硅、ZIF/Tantalum/Sapphire等。这些材料的电导率比无隔热材料要高,但由于不耐热,不能用于工艺用途,只能用于工艺保护。除此之外,还有一些特殊类型材料也可以用于封装材料,如二氧化硅、复合材料等。

## 封装材料选择标准
在决定用哪种封装材料时,需要综合考虑一下多种因素。下面列出了选择封装材料时的参考标准:
1. 质量属性: 除了自身的质量属性,还要考虑用到的材料质量是否受限、是否容易熔化、是否容易变形等因素。

2. 热处理性能: 如果是供电类电路,则用有隔热材料;如果是非供电类电路,则要选取耐候性好的无隔热材料。

3. 复原性能: 复原性能是指用去除杂质的手段,而对于电路板来说,杂质通常就是指膜层或其他与封装层有关的材料,因此选择耐摩擦性、耐碎性的材料是更好的选择。

4. 导电性能: 导电性能代表着封装材料表面的导电能力。在某些情况下,导电性能可能成为集成电路板的决定性因素。例如,在设计半导体开关电路时,就需要考虑导电能力的好坏。

5. 使用寿命周期: 使用寿命周期越长,封装材料的寿命越长。在长时间连续工作下,熔烈的金属会损耗,故能使用的封装材料需根据具体项目的需要进行选择。

6. 灵活性和可替换性: 一旦某个层次的封装材料出现漏点,就需要立刻进行换代。
