                 

# 1.背景介绍


## 智能定位简介
智能定位（Smart Locating）又称为“地图街景”，指的是能够基于真实世界中的传感信息和自身位置信息，精准快速准确地向用户提供地点、方向等信息的一种应用。智能定位通常包括两方面内容：

1.智能识别：利用各种传感器如GPS、IMU等获取用户的位置信息，并结合自身位置信息对设备进行定位；

2.智能导航：通过结合自身位置信息以及传感器获取到的用户目标信息，利用路径规划、路线规划等技术生成符合用户当前状态的最佳路径或导航路径，并将导航信息反馈给用户。

## 传统智能定位技术
目前市场上主要的智能定位技术有以下几种：

1.激光雷达定位：基于激光雷达的定位方法通常依赖于周围建筑物的特征来精确定位用户，通过扫描和探测不同频率的电磁波从而获得用户的位置信息。由于激光雷达在用户的周围区域内发出多余的无用信号，而且安装和维护成本较高，因此市场上很多厂商都没有采用这种技术；

2.GPS定位：GPS全球定位系统是目前用于智能定位的主流技术。其基本原理是在卫星或其他卫星导航系统中接收到电波后计算得到用户的经纬度坐标值。但由于GPS采用卫星导航系统，因在受到干扰时无法立刻更新其定位信息，因此定位精度存在不确定性。另外，在室外环境下，由于GPS只能获得用户高度的信息，难以确定用户所在的建筑物，而不能提供具体的导航提示；

3.WiFi定位：WiFi定位也是目前市场上热门的智能定位技术。它通过收集网卡上接收到的无线信号来获取用户的位置信息。它的优点是可以实时更新位置信息，缺点则是位置精度相对GPS来说较差，且容易受到环境影响；

4.蓝牙定位：蓝牙定位通常使用移动终端设备上的蓝牙模块收集近邻设备的蓝牙信号，并进行位置估计。由于蓝牙通信技术有广播功能，因此可能会导致定位误差增加，并且对于室内环境还存在一些限制。同时，由于蓝牙定位只能获取用户高度信息，不能提供具体的导航提示。

综上所述，传统智能定位技术已经成为人们生活中不可或缺的一项服务。然而，随着移动互联网和物联网的发展，越来越多的人开始关注智能定位技术的新进展。如何通过技术手段精准快速准确地获取用户的位置信息，以及如何通过路径规划、路线规划等技术生成导航路径，正在成为激烈竞争的焦点之一。

# 2.核心概念与联系
## GPS(Global Positioning System)
GPS(Global Positioning System)全球定位系统，是我国目前唯一能够提供海量定位数据的授时导航系统。通过GPS系统，机动车、船舶及飞机等移动车辆均可根据GPS卫星钟发送的信息，精确计算其真实或客观的经纬度坐标、航向角以及海拔高度，并随时掌握其真正的位置。它具备了两个主要特点：第一是海量数据采集能力，每年提供了超过1亿颗GPS卫星的数据；第二是独立的时间基准源，GPS时间同步非常可靠，具有世界级别的一致性。除了一般的定位功能外，GPS系统还可用于提供航空导航、矢量图显示以及图像测绘等多种应用。

## WGS-84坐标系
WGS-84坐标系是GPS系统的基础坐标系，定义了地球的椭球面。它是一个由IAG-GRS80共同建立的坐标系统。该坐标系统的基本单位为米，其原点位于地球重力加速度中心。WGS-84坐标系由GPS卫星接收到的广播信号，通过地心微分传播模型计算出一个基准时间戳UTC(Coordinated Universal Time)，即世界协调时，并通过公式将该时间戳转换为GPS坐标系下的经度、纬度以及高度。

## 北斗定位
北斗定位是中国国内第一个基于GPS卫星导航系统的互联网项目，在中国各大运营商的手机网络中广泛使用，包括电信、移动、联通以及电信运营商的4G、3G等产品线。北斗定位的定位精度可以在几十米到几百米之间，适用于室内外环境。除此之外，北斗定位还有着一系列在线产品和服务，如北斗开放平台、北斗智慧围栏、北斗导航、北斗地图等，这些产品的功能更加齐全，涵盖了GPS定位、轨迹记录、轨迹回放、语音导航、地图显示等多个领域。

## 微信定位
微信定位是由腾讯公司开发推出的基于蓝牙定位技术的地图街景应用。利用微信第三方账号的地理位置信息，实现对目标位置的定位、地图标注、路线规划和语音导航等功能。在2017年底，微信同时推出了一个内部版本的智能空间管理解决方案，旨在帮助企业管理企业内部地理位置信息，提升效率。

## 3D地图
在现代城市中，人们通常采用三维立体图的方式进行导航。三维立体图通常采用一种名叫城市立体化的技术。城市立体化就是将城市建设过程中的立体模型制作成二维平面的图片，这样就可以利用这张二维平面的图片来直观的呈现一个完整的城市形状，同时还能够展示出地区的内部结构。因此，三维立体图是近些年来各类导航设备中重要的一环。

## GPS单次定位精度
由于GPS卫星和其它地球卫星在轨道上运行的方式不同，它们产生的信号经历的阻力大小也不同。首先，GPS卫星自转比较慢，因此在空间中旋转过程中不会产生大的震荡，这样就使得GPS卫星产生的信号，可以有效的穿透大气层，保证定位精度。其次，GPS卫星之间的距离也是比较短，即使处于陆地或者山丘之上，也是可以做到高精度的。但在高空环境中，由于GPS卫星的运动范围较小，即使使用高强度的干扰信号，也难以捕捉到GPS卫星的任何信号，所以定位精度仍有限。
