# 2.核心概念与联系
## Python简介
### 什么是Python?
Python 是一种解释型、高级编程语言，它具有动态类型系统，支持多种编程范式，可移植性强，语法简单易懂。
### 我为什么要学习Python?
Python是一种高度灵活的语言，学习它的原因主要有两个：其一，数据科学需要用到大量的统计分析工具和数据处理函数；其二，前沿的深度学习框架TensorFlow、PyTorch等也是用Python编写的。
同时，由于Python内置的多种特性（例如类型检查）使得其在处理复杂的数据时效率非常高。另外，Python社区庞大且活跃，各种库和工具层出不穷。因此，掌握Python编程语言对于职场涉及到数据分析、机器学习、深度学习等诸多领域的工作岗位而言至关重要。
## 编程环境设置
### 安装Python
安装Python是个简单的过程，只需到官网下载对应版本的安装包并按照提示一步步进行即可。如果你的电脑已经安装过Python，那么你可以跳过这一步。
### 创建虚拟环境
建议创建一个虚拟环境来管理你的项目依赖关系。虚拟环境可以把不同项目的依赖隔离开来，避免不同项目间因缺少某些库造成的冲突。虚拟环境的创建方法有很多种，这里推荐使用Anaconda或Miniconda这样的包管理器。Anaconda包括了Python本身和许多流行数据科学包，非常容易安装。
### IDE选择
建议您选择一款能够高亮显示Python代码的集成开发环境（IDE）。我个人喜欢用Visual Studio Code。你可以通过插件安装Python插件来提升编码体验。
## 基础语法
### 数据类型
Python 有五种基本的数据类型：
- int（整形）
- float（浮点型）
- bool（布尔型）
- str（字符串型）
- list（列表型）
其中，int、float、bool是数值型，str是一个字符序列，list是一个元素序列。
### 变量赋值
```python
x = 10   # 整数
y = 3.14 # 浮点数
z = True # 布尔值
s = "Hello world!" # 字符串
l = [1, 2, 3] # 列表
```
### 注释
```python
# This is a comment. It will be ignored by the interpreter.

"""
This is a multi-line 
comment.
"""
```
### 打印输出
```python
print("Hello world!")     # 输出 Hello world!
a = 10                     # 将10赋予变量a
b = 20                     # 将20赋予变量b
c = a + b                  # 对变量a和b做加法运算，得到结果20
print(c)                   # 输出20
```
### 操作符
#### 算术运算符
|运算符|描述|示例|
|-|-|-|
|+|加法|a + b 输出30|
|-|减法|a - b 输出7|
|\*|乘法|a * b 输出200|
|/|除法，得到浮点结果|b / a 输出2.5|
|//|地板除，得到整数结果|b // a 输出2|
|%|求余|b % a 输出0|
|**|求幂|a ** b 输出1000000|

#### 比较运算符
|运算符|描述|示例|
|-|-|-|
|==|等于，当两边的值相等时为True|a == b 返回False|
|!=|不等于，当两边的值不相等时为True|a!= b 返回True|
|>|<|返回左边是否大于右边|a > b 返回True|
|>=|<=|返回左边是否大于等于右边|a >= b 返回False|
|=|赋值|a = 10 将变量a的值设置为10|

#### 逻辑运算符
|运算符|描述|示例|
|-|-|-|
|and|与，只有所有条件都满足时才返回True|a < 10 and b > 15 返回False|
|or|或，只要任何一条条件满足就返回True|a < 10 or b > 15 返回True|
|not|非，取反|not a < 10 返回True|

#### 成员运算符
|运算符|描述|示例|
|-|-|-|
|(in)|在...中，判断左侧元素是否存在于右侧对象中，返回True或False|5 in l 返回True|
|(not in)|不在...中，判断左侧元素是否不存在于右侧对象中，返回True或False|6 not in s 返回True|

#### 身份运算符
|运算符|描述|示例|
|-|-|-|
|is|判断两个标识符是不是同一个对象，即引用同一个内存地址|a is b 返回False|
|is not|判断两个标识符是不是不同对象，即引用不同的内存地址|a is not b 返回True|

#### 优先级
运算符优先级从高到低依次为：
- **
- ~ + - (单目运算符)
- *, /, //, %
- +, -
- >> <<
- &
- ^ |
- <= < > >=
- <> ==!=
- is is not
- in not in
- not
- and
- or
