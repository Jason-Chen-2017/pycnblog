# 2.核心概念与联系
## RASA (Reinforced Artificial Assistant) 智能助手
RASA是一个开源的机器人框架，它可以用于构建聊天机器人、自动问答机器人、检索式问答机器人、内容推荐引擎、广告分类器、聊天助手等多种类型的机器人。作为一个开源的聊天机器人框架，RASA具有以下的特性：
- 支持多种NLP组件，包括SpaCy、MITIE、Stanford NLP等
- 支持RESTFul API，方便集成到现有的系统中
- 支持在线训练和离线训练两种模式
- 有丰富的训练数据集、工具和库支持，包括开源的Conversational AI训练数据集、第三方语料库等
- 支持许多后端存储，包括SQL数据库、MongoDB等

## Bot Framework 
Bot Framework是一个面向Microsoft Azure Bot Service的SDK包。它的目标是帮助开发者构建功能完备、高度可伸缩、可扩展的聊天机器人。它提供了可编程的聊天组件和预定义消息交互，包括文本、图像、视频、音频、卡片等，并可以通过简单配置和代码即可实现功能强大的聊天机器人。除了平台外，它还提供开源的聊天机器人框架Bot Builder SDK，供其他开发者参考或借鉴。
