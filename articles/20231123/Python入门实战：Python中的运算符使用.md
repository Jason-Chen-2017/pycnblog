                 

# 1.背景介绍


运算符是编程语言中用于执行各种运算或操作的符号。在很多高级编程语言中都有提供运算符的支持，如C、Java、Python等。本文将对Python中使用的主要运算符进行简单介绍，并通过一些案例来更加深入地理解这些运算符的用法。文章将以教科书般的逻辑顺序进行组织。希望能够帮助读者快速了解Python中的运算符以及它们的作用，为之后学习及使用Python打下坚实的基础。
# 2.核心概念与联系
运算符一般分为以下几类：
- 算术运算符（Arithmetic Operators）：包括加减乘除余幂方根等基本算术运算符。
- 比较运算符（Comparison Operators）：包括等于、不等于、大于、小于、大于等于、小于等于等比较运算符。
- 赋值运算符（Assignment Operators）：包括简单的赋值=、复合赋值+=、-=、*=、/=等等。
- 逻辑运算符（Logical Operators）：包括短路与、短路或、非、异或、同真、同假等逻辑运算符。
- 位运算符（Bitwise Operators）：包括按位与、按位或、按位非、左移、右移等位运算符。
- 成员运算符（Membership Operators）：包括是否属于、是否不属于、是否为空、是否存在于等成员运算符。
- 身份运算符（Identity Operators）：包括身份运算符is和身份运算符is not。
- 迭代运算符（Iteration Operators）：包括成员运算符in和迭代运算符for。

本文只介绍算术运算符、比较运算符、赋值运算符、逻辑运算符和成员运算符的内容。其他类型运算符可以留到后面再介绍。
# 3.算术运算符（Arithmetic Operators）
## 3.1 四则运算符
Python语言支持的四则运算符如下表所示：

符号|描述
--|--
+|加法运算符，返回运算对象之和
-|减法运算符，返回差值
*|乘法运算符，返回被相乘的数的积
/|除法运算符，返回商
%|模运算符，返回除法后的余数
**|指数运算符，返回幂次方的值

### 使用举例
```python
>>> a = 10 + 20 # 相加
>>> print(a)
30
>>> b = 20 - 10 # 相减
>>> print(b)
10
>>> c = 3 * 7 # 相乘
>>> print(c)
21
>>> d = 9 / 3 # 相除
>>> print(d)
3.0
>>> e = 10 % 3 # 求模运算
>>> print(e)
1
>>> f = 2 ** 3 # 计算2的3次方
>>> print(f)
8
```

# 4.比较运算符（Comparison Operators）
## 4.1 等于运算符==
相当于数学上的等于，用来判断两个变量是否相等。比较的结果是一个布尔值True或者False。它的一般形式是: 

变量 == 变量 

或者 

变量 is 变量 

它的作用是判断两个变量是否指向相同地址。

### 使用举例
```python
>>> x = 10 
>>> y = 10 
>>> z = [10]
>>> x == y   # True 
>>> x is y   # False 
>>> x == z[0]    # True 
```

## 4.2 不等于运算符!=
与等于运算符类似，但它的结果是True或者False，表示两个值不相等。它的一般形式是: 

变量!= 变量 

或者 

not (变量 == 变量) 

它用来判断两个变量是否不是指向相同的地址。

### 使用举例
```python
>>> num1 = 5 
>>> num2 = 7 
>>> num1!= num2   # True 
>>> not (num1 == num2)   # False 
```

## 4.3 大于运算符>
用来比较大小，如果左边的变量大于右边的变量，则返回True，否则返回False。它的一般形式是: 

变量 > 变量 

或者 

变量 < 变量 

除此之外，还可以通过and关键字来组合多个比较条件。

### 使用举例
```python
>>> age = 18 
>>> salary = 5000 
>>> if age >= 18 and salary >= 5000: 
        print("满足条件") 
```

## 4.4 小于运算符<
类似于大于运算符，只是把大于换成了小于，如果左边的变量小于右边的变量，则返回True，否则返回False。它的一般形式是: 

变量 < 变量 

或者 

变量 > 变量 

除此之外，还可以通过or关键字来组合多个比较条件。

### 使用举例
```python
age = 18 
salary = 5000 
if age < 18 or salary < 5000: 
    print("不满足条件") 
else: 
    print("满足条件")  
```

## 4.5 大于等于运算符>=
类似于大于运算符，只是增加了一个等于的限定，也就是说，只有两个变量的值相等，并且左边的变量大于等于右边的变量时才会返回True，否则返回False。它的一般形式是: 

变量 >= 变量 

### 使用举例
```python
score = 80 
if score >= 60: 
    print("及格") 
elif score >= 50:
    print("中等") 
else: 
    print("不及格")
```

## 4.6 小于等于运算符<=
类似于大于等于运算符，只是把大于等于换成了小于等于，如果左边的变量小于等于右边的变量，则返回True，否则返回False。它的一般形式是: 

变量 <= 变量 

### 使用举例
```python
year = 2022 
month = 3 
if month <= 12 and year % 4 == 0 and (year % 100!= 0 or year % 400 == 0): 
    print("输入月份正确") 
else: 
    print("输入月份错误") 
```

# 5.赋值运算符（Assignment Operators）
## 5.1 赋值运算符=
用来给一个变量赋值，它的一般形式是: 

变量名 = 表达式 

表达式的求值结果会赋给变量。可以认为是最常用的赋值运算符。

### 使用举例
```python
a = 10     # 将10赋值给变量a
b = "Hello"   # 将字符串"Hello"赋值给变量b
```

## 5.2 复合赋值运算符
通过对已有的变量进行运算，得到新的值，然后重新给该变量赋值，而无需使用临时变量。其语法格式与普通赋值运算符相似，只是在等号两侧添加运算符。

常见的复合赋值运算符有：

符号|描述
--|--
+=|加等于
-=|减等于
*=|乘等于
/=|除等于
%=|取模等于
&=|按位与等于
\|=|按位或等于
^=|按位异或等于
<<=|左移等于
>>=|右移等于

### 使用举例
```python
number = 10
number += 5      # number = number + 5
print(number)    # Output: 15

string = "hello world"
string *= 3      # string = string * 3
print(string)    # Output: hello worldhello worldhello world
```

# 6.逻辑运算符（Logical Operators）
## 6.1 短路与运算符 and
它表示两个条件同时成立，返回值为True。它的一般形式是: 

条件1 and 条件2 

如果第一个条件为False，则直接返回False，否则继续判断第二个条件。例如：

```python
num1 = 10
num2 = 20
if num1 < 5 and num2 < 15:
    print("两者均小于5")
```

输出结果：两者均小于5

## 6.2 短路或运算符 or
它表示两个条件任意一个成立，返回值为True。它的一般形式是: 

条件1 or 条件2 

如果第一个条件为True，则直接返回True，否则继续判断第二个条件。例如：

```python
num1 = 10
num2 = 20
if num1 < 5 or num2 < 15:
    print("至少有一个值小于5")
```

输出结果：至少有一个值小于5

## 6.3 非运算符 not
它表示取反，即返回值为True则返回False，反之亦然。它的一般形式是: 

not 条件 

它会先对条件进行求值，然后根据条件的真假来确定返回值。例如：

```python
num = 10
if not num > 5:
    print("num的值小于等于5")
```

输出结果：num的值小于等于5

## 6.4 位运算符
位运算符是一种特殊类型的运算符，他作用于数字的二进制位而不是数字本身。位运算符是把数字看做二进制来进行计算的。Python提供了以下位运算符:

符号|描述
--|--
~|按位取反，即为每一位取反
&|按位与，对应位都为1，结果才为1
\||按位或，对应位只要有一个1，结果就为1
^|按位异或，两个对应位不同时为1，结果才为1
<<|左移动，数字向左移动指定的位数
>>|右移动，数字向右移动指定的位数

### 使用举例
```python
a = 0b1100    # 60的二进制表示为'0b11111100'
b = ~a        # 把60按位取反为'-61',十进制表示为'-61'
c = 0b1010 & 0b1111    # 按位与运算，保留0b1010的低4位，0b1111的高4位，结果为0b1010，十进制为10
d = 0b1010 | 0b1111    # 按位或运算，保留0b1010的低4位，0b1111的高4位，结果为0b1111，十进制为15
e = 0b1010 ^ 0b1111    # 按位异或运算，相同位置上为0，不同位置上为1，结果为0b1101，十进制为13
f = 0b1010 << 2         # 左移动运算，把0b1010向左移位2位，结果为0b101000，十进制为40
g = 0b1010 >> 2         # 右移动运算，把0b1010向右移位2位，结果为0b10，十进制为2
h = bin(0b1010)       # 返回0b1010的二进制表示
i = oct(0b1010)       # 返回0b1010的八进制表示
j = hex(0b1010)       # 返回0b1010的十六进制表示
```
