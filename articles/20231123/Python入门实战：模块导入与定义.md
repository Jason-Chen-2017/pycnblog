                 

# 1.背景介绍


Python（简称py）是一个强大的开源跨平台编程语言，它拥有着广泛的应用领域。从事机器学习、数据分析、Web开发、运维自动化等相关工作都可以用Python来实现。由于其简单易学、可扩展性强、社区活跃、丰富的第三方库支持，越来越多的公司开始转向基于Python进行开发。因此，掌握Python对你个人职业生涯发展至关重要。
在实际工程项目中，为了避免重复造轮子，我们需要将不同模块的代码拆分成不同的文件，每个文件就是一个模块。而模块之间需要相互调用，所以要了解如何导入、定义、调用模块以及一些常用的模块，就显得尤为重要了。本文从模块导入、定义、调用三个方面进行介绍。
# 2.核心概念与联系
## 2.1 模块导入
模块导入即我们所说的import语句。在导入模块时，如果模块名不是以“.”开头，那么就会到当前目录下搜索该模块文件；如果以“.”开头，则会先到当前目录查找该模块文件夹下的__init__.py文件，然后再加载该模块文件。
### 语法及用法
```python
import module_name            # 基本语法，导入整个模块
from module_name import name   # 从某个模块导入某个变量或函数
import module_name as alias    # 为模块指定别名
from module_name import *      # 将模块的所有变量和函数导入当前命名空间
```
### 使用注意事项
- 如果模块名冲突，则以最后导入的模块为准，不会进行覆盖。
- 在同一行导入多个模块时，不需要使用括号。
- 建议使用绝对路径导入模块。
## 2.2 模块定义
模块定义即我们所说的创建模块文件并写入代码的过程。模块定义时，应该遵守良好的编程规范，包括缩进、空格等要求，并按照一定的逻辑组织模块中的变量、函数和类。
### 语法及用法
模块定义文件的后缀名为`.py`。
```python
# 文件名: mymodule.py

def sayhello():
    print("Hello world!")

message = "This is a message."
```
### 使用注意事项
- 每个模块文件中只能有一个模块定义。
- 模块文件名称不能跟已存在的内置类型或关键字相同。
- 模块文件中的代码块之间应该有间隔，便于阅读。
- 涉及到外部资源的处理（如打开文件、网络请求），建议放到try...except...结构中进行异常捕获。
## 2.3 模块调用
模块调用即我们所说的运行时调用模块文件中的函数或方法，并获得返回值或者修改全局变量的过程。在调用之前，首先需要通过导入方式将模块加载到当前命名空间，然后通过`dot notation`或者直接访问变量的方式调用模块中的函数或属性。
### 语法及用法
模块调用一般使用以下两种方式之一：
```python
import mymodule               # 通过导入方式调用模块
mymodule.sayhello()           # dot notation
print(mymodule.message)       # 直接访问变量
```

也可以将模块加载到当前命名空间中之后使用`as`关键字为模块指定别名，然后通过该别名调用模块中的函数或属性。
```python
import mymodule as mm         # 指定模块别名
mm.sayhello()                 # 通过别名调用模块中的函数
print(mm.message)             # 通过别名访问模块中的变量
```
### 使用注意事项
- 模块调用不仅可以用于本地模块文件，还可以使用第三方库。
- 模块中的变量或函数只能在模块内被访问，在其他位置不可访问。
