# 2.核心概念与联系
## GUI（Graphical User Interface）
GUI是指图形用户接口，它是指计算机界面中仅包括图形元素（如图标、按钮、菜单、窗口等）而没有文字信息的图形用户界面。图形用户界面一般包含两部分：1、控件，主要是由各种图形按钮、文本框、滚动条、选项卡、列表框、表格、标签、进度条等构成；2、布局管理器，用于控制各个控件的位置、大小、间距等属性，使得它们能在屏幕上显示正确的形式。
GUI与非图形用户界面（CLI，Command Line Interface）相比，最大的区别在于，GUI提供给用户更加直观、直接、直观的交互方式。传统命令行的方式很难直观地展示大量的文字信息，GUI可以提供更多的视觉化的表达能力，让人更容易理解和操作。当然，如果要构建一个真正的商业级别的GUI程序，还需要考虑一些安全性、性能、扩展性等方面的因素。
## Tkinter
Tkinter是Python的一个标准库，它提供了简单、易用的GUI编程接口。它不依赖于任何第三方的图形界面工具包，其内部包含了一系列类似于Windows风格的控件，并提供了相应的事件处理机制。Tkinter的界面支持多平台，可以在Windows、Linux、Mac OS X等多种操作系统上运行。
## 模块介绍
Python有多个模块可用于图形用户界面编程，其中最流行的还是Tkinter模块。下面就介绍一下该模块的内容。
### tkinter模块
- Button类：用来创建按钮控件，点击按钮可以触发对应的事件函数。
- Canvas类：用来绘制不同类型的图形（点线、圆形、矩形、文本等），可以使用Canvas对象的方法进行图形变换、移动、删除等操作。
- Checkbutton类：用来创建多选框控件，选中或取消选中后会触发对应的事件函数。
- Entry类：用来创建单行输入框控件，用户可以通过键盘输入文本内容。
- Frame类：用来创建一个容器框架，可以容纳其他控件。
- Label类：用来创建文本标签控件，可以显示文字、图片等。
- Listbox类：用来创建列表框控件，用户可以从中选择多个项目。
- Menu类：用来创建菜单栏控件，可以添加不同的菜单项，每个菜单项都可以响应事件。
- Menubutton类：用来创建弹出式菜单按钮控件，用户可以选择不同的菜单项。
- Message类：用来创建消息提示框控件，可以显示提示信息。
- Radiobutton类：用来创建单选钮控件，用户只能从中选择一个选项。
- Scale类：用来创建滑动条控件，用户可以设置一个范围内的值。
- Scrollbar类：用来创建滚动条控件，用户可以在一个区域中浏览长文本或图像。
- Text类：用来创建多行编辑框控件，用户可以通过鼠标或键盘输入文本。
- Toplevel类：用来创建一个独立的窗口，可以在应用程序中打开多个窗口。
### 常用方法介绍
#### Tk()
Tk()方法用来创建Tkinter的主窗口，用于承载其它组件。
```python
import tkinter as tk
 
root = tk.Tk()   # 创建主窗口
root.mainloop()  # 进入消息循环
```

#### title()
title()方法用来设置窗口的标题。
```python
import tkinter as tk
 
root = tk.Tk()      # 创建主窗口
root.title("Hello")  # 设置窗口标题
root.mainloop()     # 进入消息循环
```

#### geometry()
geometry()方法用来设置窗口的尺寸及位置。参数是一个字符串，形式为"宽x高+x坐标+y坐标"，例如："800x600+100+100"表示设置窗口的宽度为800像素，高度为600像素，并移动到屏幕上的坐标(100,100)。
```python
import tkinter as tk
 
root = tk.Tk()       # 创建主窗口
root.geometry("400x300+100+100")  # 设置窗口尺寸及位置
root.mainloop()      # 进入消息循环
```
