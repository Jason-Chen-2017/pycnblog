# 2.核心概念与联系
## 什么是图像？
计算机视觉里的图像，就是由像素点组成的二维矩阵。其中每个像素点通常都由三种颜色组成——红色（R）、绿色（G）、蓝色（B）。每种颜色的取值范围都是0~255，其中0表示黑色，255表示纯白色。不同的设备和不同的软件应用程序可能有所不同，但一般来说，图像的尺寸大小都是相同的。例如，一般来说，摄像头的图像尺寸为1920x1080，而数字化相机的图像尺寸则为2048x1556。

## 什么是特征？
图像特征是一种对于图像中可见信息的描述符。特征提取是计算机视觉的一个基础过程，它可以通过对图像进行特征检测、匹配和提取来获得图像信息。特征提取是图像识别、目标检测、机器视觉等很多计算机视觉任务的基础。它主要包括以下几个步骤：

1. **边缘检测** 是确定图像中的明显特征点的方法。这些特征点往往具有极强的几何形状和局部曲率。有多种类型的边缘检测算法，如Canny边缘检测、Hough变换等。

2. **轮廓检测** 是确定图像中的所有外形轮廓的方法。例如，可以用来检测物体的边界、铆钉、瓶盖等。有多种类型的轮廓检测算法，如霍夫变换、直线检测等。

3. **特征匹配** 是根据已知特征点模板匹配来搜索新的图像区域的方法。例如，可以使用SIFT、SURF、ORB、AKAZE等特征点提取方法，然后通过比对获取特征点，最后通过特征匹配找到对应的图像位置。

4. **特征描述子** 是图像中每个特征点的向量形式，用于快速识别和匹配。描述子一般由一系列参数来表征，有时还包括图像区域的统计信息。典型的特征描述子包括直方图、HOG（Histogram of Oriented Gradients）、LBP（Local Binary Patterns）、BRIEF（Binary Robust Independent Elementary Features）等。

5. **几何变换** 可以对图像进行旋转、缩放、裁剪、透视等操作，从而更好地处理图像信息。图像几何变换可以帮助识别、处理目标物体和场景。


## 什么是视觉系统？
视觉系统是指机器具有的视觉感受器、处理装置和软硬件组件。它包括图像采集设备、图像信号处理设备、图像处理模块、显示设备、传感器、决策支持系统和学习机制等。视觉系统的功能可以总结如下：

1. **图像采集** 包括摄像机、激光测距仪、数字相机、扫描仪等。它们采集各种各样的图像和视频信号。

2. **图像信号处理** 通过各种图像处理算法和计算机视觉模块对图像信号进行预处理，形成可以识别和理解的信息。图像处理算法包括滤波、锐化、形态学操作、自适应阈值、直方图均衡化等。

3. **图像识别** 根据处理过后的图像信息，采用不同算法和模式对图像进行识别和分类，判断图像中是否存在特定物体。

4. **图像处理模块** 包括视觉定位、视觉导航、视频监控等。它的主要任务是对视觉输入进行分析和理解，将其转换为易于使用的符号形式。

## 什么是机器学习？
机器学习是关于计算机系统如何学习从数据中抽象出知识、改善性能的领域。与传统的编程方法不同，机器学习利用大量的数据训练计算机模型，模拟人的学习过程，并借此优化性能。机器学习有助于解决复杂的问题，提高效率，减少错误率，改善产品质量。机器学习主要包含以下几个要素：

1. **数据集** 是一组用于训练模型的数据样本。

2. **模型** 是学习数据的结果。常用的模型有决策树、随机森林、支持向量机、神经网络等。

3. **策略** 是指如何选择模型和训练模型的方式。有迭代法、梯度下降法、遗传算法等。

4. **代价函数** 是衡量模型优劣的指标。

5. **反馈机制** 是指系统如何调整模型参数以响应反馈。有监督学习、无监督学习、强化学习等。

## 什么是卷积神经网络？
卷积神经网络是深度学习的一种类型，用于处理具有空间相关性的图像和视频数据。它由多个卷积层、池化层、归一化层、全连接层等构成。卷积神经网络具有以下优点：

1. **多层特征** 卷积神经网络通过丰富的卷积层和池化层学习到各种高级特征，包括边缘、线条、纹理等。

2. **位置信息** 卷积神经网络可以捕获局部与全局的特征，并利用位置信息实现目标定位和检测。

3. **准确率** 卷积神经网络可以训练足够多的模型，从而达到比较好的效果。

4. **计算高效** 卷积神经网络由于具有高级特征提取能力，因此可以在CPU上快速运行，并在GPU上高效加速。
