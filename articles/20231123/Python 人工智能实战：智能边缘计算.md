# 2.核心概念与联系
本节将简要介绍本教程涉及到的主要核心概念，帮助读者更好理解本课程的内容。

1.云计算：云计算（Cloud computing）是一种新兴的计算模式，它利用互联网提供的廉价服务器资源按需付费的方式，快速部署和扩展计算能力。云计算可有效降低基础设施投资成本、提升业务水平和效率。其核心功能包括计算、存储、网络、数据库等服务，为用户提供了按需付费的弹性计算能力。

2.边缘计算：边缘计算（Edge Computing），也称为移动边缘计算或终端边缘计算，是指利用近距离计算资源和网络连接，对靠近数据的源头进行数据处理的技术。其特点是边缘计算节点离物理位置很近，因此处理速度快、能耗低、计算资源受限。

3.边缘端设备：边缘端设备（Embedded System）是指安装在物理设备上的计算机系统，具有自主执行能力，能够独立完成特定的任务。由于其无须连接到互联网就能够完成特定任务，边缘端设备通常被设计用于处理密集型数据量，且处理时间要求较高。例如，智能门铃、汽车安全系统、智能机器人、遥测站等。

4.人工智能：人工智能（Artificial Intelligence，AI），是研究和开发让计算机具有智能的技术。包括机器学习、深度学习、强化学习、统计学习、遗传算法、遗传编程、行为科学、认知神经网络、认知科学等多个领域。其中，深度学习、强化学习、统计学习方法成为当今的主要研究热点。

5.机器学习：机器学习（Machine Learning）是一类人工智能的子领域，旨在让计算机像人一样学习、解决问题。它借助于数据及已有的经验，自己模拟、改进一个过程，最终达到自动地发现、改进和使用的目的。机器学习算法通过观察、试错、归纳、凭直觉等方式从数据中找出规律、提炼出特征，并自动调整参数，使得模型在没有明确指导的情况下，依据输入数据产生一系列的输出结果。机器学习方法分为监督学习、无监督学习、半监督学习、集成学习等。

6.深度学习：深度学习（Deep Learning）是机器学习的一种方法，由多层神经网络组成。通过非线性变换，深度学习可以提取数据的全局结构和局部特性。深度学习的优势之一是通过组合简单单元而形成复杂模式的能力，可以应对复杂的数据表示和语音信号。

7.CNN：卷积神经网络（Convolutional Neural Network，CNN）是深度学习的一个子分支，它是深度学习中的一种最常用的模型。它是一个有利于提取图像特征的卷积层和池化层的结合体。CNN在图像分类、目标检测、视频分析等领域有广泛的应用。

8.OpenCV：OpenCV (Open Source Computer Vision Library) 是一款开源的基于BSD许可证的计算机视觉库。OpenCV 支持多种硬件平台包括CPU、GPU、DVP等。通过 OpenCV 可以轻松实现摄像机捕获、视频播放、图片读取、影像处理等功能。

9.RESTful API：RESTful API（Representational State Transfer Application Programming Interface）即“表述性状态转移应用编程接口”，是一种用来开发Web应用程序的软件架构样式。它具备简洁、灵活、易用的特点，能够让Web服务更加容易的被其他程序调用。

10.MQTT：MQTT（Message Queuing Telemetry Transport，消息队列遥测传输）是IBM公司发布的一款用于物联网的开源通信协议。它采用发布/订阅（publish/subscribe）模式，使得物联网终端之间能够互相通信。

11.Django：Django 是一款开放源代码的web应用框架，是Python下最流行的web框架之一。它由吉多·范罗苏姆、马斯克尔·巴士纳、马克·沃森、罗伯特·诺培、弗朗西斯·福山等人开发。它使用了MTV模式，即模型-视图-模板，并内置了很多模块，比如ORM、表单处理等，因此使用起来比较方便。
