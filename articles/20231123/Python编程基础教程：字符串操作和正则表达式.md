# 2.核心概念与联系
## 字符串操作
字符串是一种不可改变的数据序列，它用于表示文本信息，包括数字、字母、标点符号等字符组成的集合。字符串可以是任意长度，但通常会分割成若干个子字符串组成列表或者元组形式。字符串处理是计算机科学领域中的基础工作，也是Python的一项重要功能。字符串操作一般分为以下几类：
### 截取字符串
字符串截取是最常用的字符串操作方式，通过切片的方法获取子字符串，从而实现各种功能。下面列举几个常用的字符串截取方法：
#### 方法一：`[start:end]`
这是最常用的字符串截取方法，其参数表示要截取的子串起始位置和结束位置，即[起始索引:终止索引+1)。参数值均可以省略，省略时默认截取整个字符串。例如：
```python
s = "Hello World!"
print(s[:])   # 输出完整字符串："Hello World!"
print(s[2:])  # 输出从第3个字符开始到结尾的字符串："llo World!"
print(s[:-2]) # 输出从头开始到倒数第二个字符结束的字符串："Hel"
```
#### 方法二：`str.split()` 或 `str.partition()`
该方法可以将字符串按照指定字符分割成多个子串，返回由子串组成的列表。其原型如下：
```python
string_object.split(separator=None, maxsplit=-1)
```
其中，`separator`表示指定分隔符，如果没有指定，则默认为所有空白字符（空格、制表符、换行符）。`maxsplit`表示最大分割次数，`-1`表示不限制。例如：
```python
s = "Hello,World!"
lst = s.split(',')    # ['Hello', 'World!']
p = s.partition(',')  # ('Hello', ',', 'World!')
```
#### 方法三：`str.replace()`
该方法用来替换字符串中的子串，第一个参数表示要被替换的子串，第二个参数表示替换后的字符串。例如：
```python
s = "The quick brown fox jumps over the lazy dog."
new_s = s.replace('the', 'a')  # "A quick brown fox jumps over a lazy dog."
```
#### 方法四：其他一些内置函数
还有一些其他的内置函数也可以用来操作字符串，比如：`str.strip()`、`str.lower()`、`str.upper()`、`str.find()`、`str.count()`。这些函数都比较简单易用，建议熟练掌握。
## 正则表达式
正则表达式（Regular Expression，RE）是一种模式匹配技术，旨在从文本中找出符合某种规则或描述的字符串。它广泛运用于各种领域，如文本处理、文件检索、网络爬虫、数据库查询等。

正则表达式由两个部分构成，分别是模式（Pattern）和目标字符串（Target String）。模式描述了在目标字符串中匹配的内容，语法复杂且精确。目标字符串是要搜索的原始文本，经过模式匹配后，可能会得到一系列符合模式要求的字符串。

Python提供了re模块来支持正则表达式，包含四个主要功能：

1. re.match()：从字符串开头匹配模式。

2. re.search()：在整个字符串查找匹配。

3. re.findall()：找到字符串中所有匹配模式的子串，并将它们放在一个列表里。

4. re.sub()：找到字符串中所有匹配模式的子串，并将其替换为另一个字符串。
