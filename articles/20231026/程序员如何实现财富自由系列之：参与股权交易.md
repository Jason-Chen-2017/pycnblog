# 1.背景介绍



自从互联网泡沫破灭之后，人们对金钱的追求便开始受到重视。2017年9月，一篇题为《Facebook、Google 和 Apple 在 2017 年离场 》的新闻报道曝光了这一切的真相——互联网巨头们宣布计划在今年秋天裁员3万名员工，以此来控制整体的经济发展，同时扼杀创新和竞争力。

直到最近，尽管很多年轻人对财富的渴望早已超过生活质量，但对股权投资一直存有戒备。2018年夏天，美国证券交易委员会（SEC）发布了《关于启动 ADR 机制的公告》，呼吁投资者谨慎介入 ADR 机制并及时退出，避免出现金融危机带来的损失。但是随着 ADR 的推出，越来越多的中国民众开始购买 ADR 股票，也引起了对股市暴跌、回撤的担忧。

本文将会通过详解股权交易的基本概念和规则，分析其中的一些风险，以及给出一些方法，让你可以参与到股权投资领域，实现财富自由。

# 2.核心概念与联系
## 股权投资的概念
股权投资其实就是企业或个人拥有一定比例股份的权利，可以按照其价值进行投资。例如，A股上市公司每股股票面值为1元，那么我可以购买它100股，每股1元，然后用这笔钱来投资某个行业、企业或个人。

## 股权投资的运作规则
### 认购
买卖股权最基本的方式就是认购。如果某个企业或个人想购买某只股票，需要在一定的价格上认购。所谓的价格，指的是该企业或个人愿意支付给股东的每股股票数量。比如说，某些企业或个人愿意购买的价格可能是每股100元，或者1000元等。当股东决定购买股票时，必须要遵循一定的规则。首先，股东必须持有该企业或个人近期有价值的股票。其次，必须缴纳一定的认购费用。最后，必须经过公司董事会的批准才可以认购股票。

### 分配
认购成功后，股权进入到股东手中，即所得税的收入将用于缴纳未来的分红。分红通常由董事会根据每个股东持有的股份份额按比例分配，分红总额等于每股股息的总和。通常情况下，分红期限一般为5-10年。除非企业或个人提前终止股权，否则分红是永久性的。

### 增发
既然是股权，为什么还会有增发呢？增发是指把之前认购的股票转让给其他股东。比如说，如果A股公司由于合并、分立等原因，不得不削减成本，不能按股东认购的价格认购新的股份。那么A股公司可以把之前认购的股票转让给另一个股东，增加新的股份。新股东只需要持有增发后的股份即可，不需要认购费用。增发的目的主要是为了补充由于董事会调整而造成的缺口。当然，增发也存在一定的风险，包括新股东参股股息率低、交易商赚不到钱等。

### 交易
交易就是买卖股权的过程。投资者以自己认为合适的价格，根据交易规则，下单买入或卖出股票。双方必须达成协议，必须接受交易规则的约束，包括手续费、佣金、保证金等。交易完成后，双方都必须承担相应的责任，包括返还认购费用、付清分红等。

### 滥伐
滥伐是指企业或个人过分依赖股权。过分依赖股权可能会导致亏损，甚至亏空。例如，假设某只股票的价值高于它的市场价值。这种情况下，企业或个人可能为了一时的利益而掠夺该股票。滥伐不仅严重影响股东的利益，而且也会对企业的信誉和声誉产生负面影响。

为了防止滥伐，企业或个人应尽可能保持独立的管理层，降低决策权、盈利能力和行为风险。另外，投资者应当更加注意交易的节奏，尤其是在涉及较大的利润或较长时间的项目时。
