# 1.背景介绍


## 什么是量子纠错？
量子纠错（Quantum Error Correction）即通过对量子数据进行检错、纠正、恢复等处理而实现数据通信的可靠性。其基本原理是在物理层面上消除错误。
## 什么是量子编码？
量子编码（Quantum Coding）是指利用量子纠错技术对原始数据进行编码，从而使得经过编码的数据具有较高的信息熵，这可以提升传输效率、降低成本、提高可靠性。

# 2.核心概念与联系
## 通信基础
通信的三要素——信道、信号、噪声。在没有任何干扰的情况下，信号可以在信道上传输。然而当存在噪声时，就需要引入纠错机制才能保证信息传输的可靠性。通信过程中会产生很多错误，包括：
- 单比特错误（Bit Flip）：一个比特发生变化，导致该比特的值发生了反转；
- 比特串错误（Bit Transition）：多个比特发生交叉，产生错误的比特串；
- 调制失真（Modulation Degradation）：将相邻比特的频谱混淆，导致信道容量受损；
- 时延抖动（Timing Jitter）：发送端和接收端的时间不一致，导致信息丢失或延迟。

## 传统通信网络
传统通信网络中，每台设备都连接到一条固定的导线，其中的电磁波直接穿透导线，无法传递任何噪声。而现代通信网络是由许多的干扰源和微波天线组成，可以承载高强度的电磁波，并且可以采用各种技术对其进行纠错，例如多路切分、码流重建、帧结构设计、调制加密等。但是这些纠错技术都是基于概率论和统计分析的方法，并不能完全消除所有纠错误码。因此，如何结合理论、算法、工程实践，开发出一种具有高性能和扩展性的纠错技术就成为一个重要研究课题。

## 量子纠错
量子纠错通过破译量子态来纠正错误。由于量子纠错是通过对量子数据进行纠错处理，所以它依赖于量子信息处理理论中的纠缠定律。纠错过程由两部分组成：生成态和检测态。生成态主要用来产生待纠错的量子数据，检测态则用于检测错误并计算出错误位置。通过不断重复生成态、检测态、纠错的方式，最终能够把错误部分消除掉。

目前已有的量子纠错技术主要有两种：基于贝叶斯方法的纠错和基于信道检验的纠错。

### 基于贝叶斯方法的纠错
基于贝叶斯方法的纠错算法是使用贝叶斯概率公式来估计出错的概率分布，然后根据概率分布选择最有可能产生错误的数据作为输出。这种方法的优点是计算简单、准确度高、适用范围广，但缺点是要求输入数据的纠错概率分布必须事先知道。

### 基于信道检验的纠错
基于信道检验的纠错算法不需要知道错误概率分布，只需要对不同的测量结果进行统计分析即可。这种方法的优点是无需事先估计出错概率分布，速度快、灵活性高，但缺点是对噪声敏感、计算开销大。

## 量子编码
量子编码是指利用量子纠错技术对原始数据进行编码，从而使得经过编码的数据具有较高的信息熵。编码过程就是为了使得数据比特的排列尽可能地接近于系统发射出的码元。编码方式一般分为三种：离散余弦变换 DCT、正交变换 QR 分解和汉明距离 LDPC 编码。

DCT 是最早提出的数字编码方式，它可以有效地对数据进行编码，因为其具有线性相关性。DCT 的优点是有利于数据压缩、降低存储开销、提高传输速率，缺点是容易出现维数效应。

QR 分解是正交矩阵分解法，它通过计算正交矩阵实现数据编码。QR 分解的优点是实现简单、高效、稳定，但缺点是要求输入数据是高维矩阵。

LDPC 编码是一种低密度 parity check code (LDPC) ，它被设计用于在连续信道中进行通信。LDPC 编码中有一个伪随机码 (PRC) 和一个系统定义的 parity check matrix (SCM)，其中 SCM 定义了待编码数据的奇偶校验信息。LDPC 编码的优点是对噪声具有很高鲁棒性，可以应付复杂的环境条件，但缺点是计算复杂、内存占用大。
