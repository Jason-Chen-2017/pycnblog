# 1.背景介绍


## 概述
人类的视力在不断进步，同时由于生物体中存在着复杂的非线性电信号处理机制，使得我们对环境和自然界的物理事件更加敏感、细致、快速地进行观察和理解。而传统的线性光学只能获取很少的信息，无法真正地将宇宙中的各种微粒和场关联起来，这就产生了我们对高分辨率照片或模拟成像等更高的视觉效果时依然束手无策的现象。因此，为了能够真正地挖掘到宇宙中所有的信息，我们需要探索光谱的新领域——量子光学。量子光学是一种倾向于采用量子论作为研究框架的新型光学学科，其理论基础是量子力学。量子光学基于一个基本假设，即我们对光的感知能力存在一些限制，特别是在非常微弱的能量范围内。此处我们就来聊聊量子光学与传统光学的区别及联系。

首先说一下什么是传统光学。传统光学认为，所有波长的光都是平面且直线传播的，它们的变化具有均匀分布，形成光谱图谱的形式。这样的观点给我们提供了高度的空间分辨率，但它也造成了以下缺陷：

1.由于所有波长的光线平行且相同方向传播，所以不能真正地理解不同波长之间的相互作用，只能看到整体的图像，并且无法区分不同波长的光。例如：不能探测到两束不同颜色的光。

2.传统光学只适用于基本的光源结构，而对于大多数应用来说，这些限制并不是那么容易克服。例如：在没有考虑反射和透射的情况下，传统光学无法正确记录太阳系内部的物理行为。

3.由于传统光学的缺陷，使得它的应用受到了一些限制，例如，对于天文学的应用以及对精密仪器的设计来说都不太方便。

因此，为了解决以上三个问题，出现了一些新的光学技术，如多光子叠加（MCCD）、数字电子显微镜（DEM）等。他们的共同特征是利用非线性元件和能量捕捉的概念来探索物质行为，弥补传统光学存在的局限性。


接下来我们继续讨论量子光学，为什么要使用量子光学？因为，尽管传统光学可以捕捉到宇宙微小的信息，但我们发现其效果是受限于基本的感光材料，无法探索到宇宙深层次的结构。因此，量子光学的出现意味着人们对我们使用的基本材料进行了解释，打破了这一限制，从而开启了对真实世界的探索之旅。

量子光学的关键观念是，宇宙是由不可分割的微粒组成，微粒间存在着巨大的相互作用，而且微粒的运动是随机的。也就是说，不论光线的频率如何变化，光都无法被准确描述。换句话说，宇宙中存在着许多不同的量子态，而不同波长的光只能在这些态中找到它们自己最佳的表示方式。因此，量子光学的光谱特点表明：我们看到的是我们无法通过观察到的实际光线看到的东西，只能看到某个比任何其他波长都要强烈的效果。这是由于宇宙中微粒的微小性质决定的，它让我们看到的只能是一个比通常所能看到的更远的景色。


总结一下，传统光学只是人眼所见的几种不同波长的光，而量子光学可以描述整个宇宙的状态，揭示出微小但又深刻的物理规律。这两个学科之间有很多联系，如光子是波，而粒子则是物质的基本粒子，它们在空间中运动并进行各种相互作用，因此，它们都属于量子力学。但传统光学仍旧有很多局限性，比如不能真正理解不同波长之间的相互作用。而量子光学可以在人类视力无法完全理解宇宙时，提供高超分辨率的影像。
