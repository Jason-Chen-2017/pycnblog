# 1.背景介绍


目标管理（Target Management）是一种管理思想、方法和工具，通过对战略方向、业务计划和可行性进行分析、总结、排序，以及团队各成员的绩效、能力、工作态度和意愿，来制定或调整组织整体发展战略、策略及个人发展目标。目标管理是管理中最重要的概念之一，其核心理念主要包括目标分级、多维度绩效评估、持续性的目标改进、自我激励、适时反馈机制等。目标管理理论认为，只有把目标管理与优化管理相结合，才能有效地实现组织的长远发展。
# 2.核心概念与联系
目标管理的核心概念有以下几个方面：

1.目标层次结构：目标管理基于将组织的战略目标分成多个子目标，并给予每个子目标一个明确的时间和资源分配，形成目标层次结构，使得完成每个目标更容易被实施。目标层次结构通常采用倒金字塔（即目标为主导目标，小目标为支撑目标），也可能采用正金字塔，但一般都要求实施目标要先满足支撑目标。目标层次结构还可以用于设定战术目标和产品开发计划，并让整个组织的资源向着战略目标靠拢。
2.目标策略：目标策略是指如何将战略目标转化为可执行的具体目标，其中包括战略纵向和战略横向两大类型。战略纵向目标是指直接影响组织战略整体成功率的战略目标；战略横向目标则侧重于公司整体目标，其目的是为了满足业务部门内部目标和跨部门的外部目标。目标策略需要有所偏差，这时可以通过适当放权或者约束组织决策者来缓解组织的不确定性。目标策略还涉及到对目标质量的测评，以确定目标是否符合要求，以及是否存在风险、瓶颈和阻碍。
3.目标交付：目标交付是一个过程，它由组织目标管理专家和目标客户共同参与，根据目标管理团队制定的目标策略，将战略目标转换成可管理的工作产品，并将该产品提供给目标客户。目标交付包括以下三个阶段：
a.需求确认：目标管理人员会与目标客户沟通，调查其需求并制订交付方案。
b.交付规划：目标管理人员与交付团队一起制定交付时间、交付物品和交付方式。
c.交付验收：交付完毕后，目标客户和交付团队一起对交付结果进行验收。
4.目标执行：目标执行又称实际管理，是指通过建立透明度、协作与一致性的管理关系，以促进目标达成，并确保组织在目标期限内取得预期效果。目标执行有三个阶段：
a.明确目标：在每年或每季度开展一次目标清单会议，并由目标管理人员召集全公司、相关部门、目标客户以及相关干系人的代表座谈，以便更好地明确组织的目标。
b.建立管控机制：目标管理人员在现有制度基础上，加强各个职能部门之间的配合，形成各方利益的共识，帮助目标执行能够按照既定目标、标准、流程和服务水平进行。
c.结果跟踪：组织的各项指标、目标、经营业绩以及其他关键数据都会随着目标执行的进行而不断变化。通过结果跟踪，目标管理人员可以实时掌握目标的最新进展，制定下一步的目标。
5.自我激励：目标管理人员应具备独立判断力、责任心、专注力以及创新精神。通过自我激励，目标管理人员能够识别和解决问题，并将自身的价值观融入到目标管理过程中。同时，目标管理团队也可以借助自我激励机制来提升团队士气，促进目标达成。
6.自我监督：目标管理团队还应具备自我监督的能力，他们可以在需要时随时审视自己在目标管理中的表现，并据此做出调整。自我监督机制可以防止出现目标执行过程中因个人贡献带来的问题。

## 3.1 目标分级
目标分级是一个非常基本、重要且简单的目标管理方法。其核心思想是根据组织目标的重要性以及目标能够提供的核心价值，将目标划分成不同的层次，按层次逐渐增强组织的战略动力，从而提高组织的整体效益。常用的分级结构有四种：功能层级、市场层级、垂直分级和路径型分级。
### 功能层级
功能层级是最常用的分级结构。功能层级划分出来的目标具有比较明显的特性，比如创建、销售、运营、服务等。不同层级目标之间存在依赖关系，即层次上一个目标的完成往往对下一个目标的推进有着不可替代的作用。因此，在完成上层目标之前，往往不能真正意义上地实现下层目标。功能层级的优点是简单易懂，容易实现。缺点是只适用于目标数量少，较低层级目标容易滞后导致战略空白期。
### 市场层级
市场层级也叫垂直分级。这种分级结构将公司的目标分成不同的区域，如汽车、医疗、酒店、零售等。由于这些领域之间往往存在互斥和竞争关系，因此市场层级的目标往往具有更大的灵活性和吸引力。市场层级的优点是很好的将目标拓展到各个领域，让组织能够快速响应新的机遇，并有助于优化资源配置和满足客户需求。缺点是要花费更多的资源投入和人力。
### 水平分级
水平分级也叫路径型分级。这种分级结构将目标按照项目路径进行分级。每一个项目从初始目标（如开办新商店）逐步升级到最终目标（如营业额达到百万）。不同项目的目标间存在依赖关系，比如第一个目标完成后才考虑第二个目标。水平分级的优点是结构清晰，突出了目标的优先级，并且层次之间没有明显的界限，也没有统一的评价准则。缺点是易造成目标重复，降低效率。
### 垂直分级
垂直分级也是一种分级结构。在这个分级结构中，组织的目标分为不同的战略层次，如销售、服务、工程、管理、运营、财务等。不同层次目标往往关注不同的领域，因此在完成某些特定目标之前，不会对其他目标产生影响。这种分级结构的优点是很容易识别和理解各个层次目标之间的依赖关系，也易于设置层级目标，适用于大型组织或复杂的市场环境。缺点是不直观，并且在层次之间难以识别交叉口，需要额外的技巧才能处理冲突。
## 3.2 目标评估
目标评估是目标管理的重要环节，其目的就是给出一组目标，然后确定哪些目标可以真正落实、达到、实现。目标评估可以分为多种方式，如数字目标评估、评审会议评估、目标回顾评估等。
### 数字目标评估
数字目标评估是评价目标有效性的一种简单方式。通过对目标的达成情况进行数据收集、整理、统计、分析等，即可得到目标的达成率、整体质量、完成进度、风险、资源利用效率等。通过数字目标评估，可以直观地了解目标管理的效果、措施是否有效、组织的资源和人力是否充足，以及对未来发展的规划和布局。数字目标评估一般不需要具体的目标细节，只需简单地对目标进行打分、评估就能得出最终的结果。
### 评审会议评估
评审会议评估是指目标管理专家和目标客户一起举行的会议，由组织领导者和目标客户共同出席。在会议上，领导者将组织当前的目标进行简要介绍，目标客户陈述其需求，双方讨论评估建议。在这个评估过程中，双方经过充分论证，可以制定目标管理措施，提前发现和解决问题，并迅速采取行动。评审会议评估的特点是具有相互讨论、对话、协调的特征，也具有实时性，能够及时反映目标管理的实践状况。但是，它的缺点是耗费时间、精力，而且容易受到目标客户的质疑。
### 目标回顾评估
目标回顾评估是指将历史目标与目前目标进行比较，分析不同目标之间的重叠和区别，寻找新的或更新的目标，并规划未来需要继续实施的目标。通过目标回顾评估，组织能够识别最有效、最满意的目标，为实现新的目标提供一个切入点。目标回顾评估的特点是客观、公正、及时，比较容易操作。但是，它的局限性也很明显，即无法评估目标进度、管理效果以及管理成本。
## 3.3 目标变革与驱动
目标变革是指目标管理的有效方式。目标变革指的是调整、优化现有的目标和新目标，以达到组织战略目标、资源节约、外部奖励、管理效率、客户满意度等目标。目标变革的流程一般分为计划、变更、跟踪、验证、执行五个阶段。
### 目标变更
目标变更是指更改现有目标的过程。目标变更不仅可以用来修正不切实际或失误的目标，而且也可以用来实现组织目标。目标变更可以从战略目标、产品目标、市场目标、产品开发计划等各个角度来实现。目标变更对目标的规划与实施起到了至关重要的作用。但要注意，目标变更需要与各个利益相关群体密切合作，才能取得成功。
### 目标计划
目标计划是一个关键环节。目标计划即根据组织的战略目标、经营计划、发展计划，以及目标分级结构制定目标计划，即设定目标的范围、条件、过程、成果和衡量标准。目标计划应该以计划分类的方法进行。分类的方式可以分为短期、中期、长期三个类别。在短期目标中，主要是对某项活动的目标，如开展新产品销售，等等。在中期目标中，是对一段时间内的总体目标，如一年内要保持营业额百万，等等。在长期目标中，是指将来某个时间点的目标，如二十年后实现企业盈利百亿，等等。目标计划应当以项目的形式呈现，每个项目都有明确的目标、执行顺序和预期结果。目标计划的好处是可以为所有员工的目标执行提供依据。
### 目标优化
目标优化是指将目标管理纳入企业发展规划的一环。目标优化是实现战略目标、资源节约、外部奖励、管理效率、客户满意度等目标的重要手段。目标优化需要将目标管理作为创新发展的重要一环，在组织内部和外部鼓励目标管理的实践。目标优化是“做正确的事”而不是“不做正确的事”，只有坚持目标管理的有效性，才有可能做到专注、聚焦、协同、综合。目标优化可以提高工作效率、减少浪费、改善企业竞争力。
### 目标执行
目标执行是一个反复迭代的过程，即目标计划、目标优化、目标改进、目标反馈、目标监测、再次调整等等。目标执行是执行目标的最佳时机，也是排除障碍、调整目标的关键时刻。在目标执行过程中，组织各个部门都要密切配合，始终以目标为中心，确保目标在最短时间内获得实现。
### 目标驱动
目标驱动也是一个重要的管理环节。目标驱动就是从自身的行为、言论和表达中，学习到目标，并将之应用到日常工作、管理工作中。目标驱动可以提高员工的工作效率、促进员工的目标贯彻、改善工作绩效。同时，目标驱动也能减轻组织的压力，保证工作顺利、按时、高效地开展。
## 3.4 组织规模、分层与职能角色
组织规模越大，组织目标管理的复杂程度越高。所以，对于规模特别大的组织，目标管理制度可能还需要分层和职能角色的设置，否则就会出现职能衔接不齐、目标割裂的现象。
### 分层制
分层制是在不同层次引入目标管理，以便支持各层的目标实现。分层制可以让目标层次清晰、明确、独立。与此同时，分层制还可以有效地管理不同层面的资源和人员，从而提高资源利用率和目标实现的效率。目标管理分层制可以分为五层：战略目标管理层、产品管理层、资源管理层、职能管理层和治理管理层。
### 职能角色
职能角色是指为目标管理设立的一个或多个职能。目标管理需要将各个部门分派到不同的职能去处理目标。职能角色的设置可以有效地控制目标执行的进程，从而保证目标管理的科学性、完整性和有效性。对于规模庞大的组织，目标管理职能角色的设置也应该高度关注。
