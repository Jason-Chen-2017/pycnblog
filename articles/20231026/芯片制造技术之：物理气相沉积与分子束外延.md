# 1.背景介绍


## 概述
物理气相沉积是一种由于在微结构中形成的粒子在无量纲化的情况下无序排布，导致在晶体中的形状发生错误或者失真的现象。随着量子计算机的出现，人们越来越关注这一现象对芯片设计和制造过程中的性能、成本、功耗等产生的影响。物理气相沉积是由粒子质量、温度、压强、流速和流方向以及其他宏观因素共同影响的，它与微结构、分子链、构型、层析、半导体材料、光刻胶特性、制备工艺和器件温度都密切相关。由于物理气相沉积会影响微结构的形状、特性甚至功能，因此其对芯片制造过程产生直接的经济和生态效益。所以，对物理气相沉积进行准确的处理是一项关键性工作。
## 物理气相沉积的分类及原因分析
物理气相沉积是指微结构中相邻两个粒子之间存在着电子运动的现象，这种现象主要表现在以下几个方面:

1. 大小不均匀：相邻粒子大小和位置不一致。如图1所示，红色圆点为粒子，蓝色箭头为电子运动方向。由于不同大小的粒子存在着电子运动的可能，导致粒子之间的距离差异较大，从而引起错位。
图1：不同大小的粒子之间的距离差异。蓝色箭头为粒子间距差异导致的错位现象。
2. 方向不均匀：相邻粒子运动方向相同，但是方向不一致。如图2所示，蓝色垂线为电子运动方向，橙色圆圈为粒子，箭头代表运动方向。由于粒子运动方向的不一致性，导致粒子之间的相互作用无法准确反映到实际的晶体结构上，从而产生物理气相沉积。
图2：不同方向的粒子之间的物理气相沉积。蓝色垂线为粒子运动方向导致的错位现象。
3. 局部极小值改变：相邻粒子存在着局部极小值的变化。如图3所示，粒子数目越多，局部极小值的变化越明显。然而，由于粒子之间的相互作用，其局部极小值并不会因为粒子数量增加而减小。所以，粒子的数量越多，粒子之间的相互作用就越复杂，从而导致局部极小值的变化。
图3：粒子数目的增加导致局部极小值的变化。
4. 耦合不良：相邻粒子之间的耦合不良。如图4所示，青色方块为微团，紫色圆圈为粒子，蓝色竖线为电子运动方向。由于电子运动的复杂性，导致相邻微团之间存在着耦合不良。导致该区域的能量分布偏离平均状态，引起物理气相沉积。
图4：耦合不良导致物理气相沉积。

### 物理气相沉积的影响因素
1. 流体性质：流体性质包括粘度、动态 viscosity 和涡度等。粘度是指两相邻粒子之间的摩擦力。微粒体在流体性质下融合成晶格，使得粘度、流速等指标随时间变化，而其大小和位置也会随之变化。粘度和流速关系可以由阻尼振荡模型计算得到。动态 viscosity 是指粒子运动速度变化率，由粒子大小、空间分辨率和流速决定。在流体性质变化过程中，微粒体会失去均匀性，导致错位和失真。
2. 分子链：分子链长度会影响粒子运动路径长度，引起物理气相沉积。当相邻粒子之间的距离过长时，粒子链会变短，此时粒子的电子运动就更加不易准确定位。另外，由于多粒子的相互作用，粒子链可能会被破坏或损毁，导致物理气相沉积。
3. 构型：构型不仅仅取决于晶体的晶胞性质、层析性质和非晶聚性质，还受到粒子数量的影响。相邻粒子之间存在着密度和大小的不均衡，从而使得微细颗粒的存在。由于不规则晶体、高分子构型带来的分子错位，导致物理气相沉积。
4. 金属结合层：金属结合层是低温浸润的成分，它能够迅速升华，导致微观颗粒膨胀、纤维化、缩小。当微观颗粒与金属结合层接触后，其路径变短，引起物理气相沉积。
5. 外延：外延是指构型膨胀后的区域，因为外延中的微观颗粒的存在，导致外延的形状发生变化。由于外延中存在微观颗粒的存在，导致外延中各个方向上的电子运动方向不一致，从而引起物理气相沉积。
6. 制备工艺：制备工艺包括晶体的合成方法、熔点和温度等。当熔点过低、温度过高或制备工艺不当时，容易出现物理气相沉积。
7. 温度：电子在微观世界中运动时，由于粒子的质量和温度的影响，电子运动路径就会发生变化。为了避免物理气相沉积，微观世界中的温度要控制在某个临界温度以上。
### 物理气相沉积的处理策略
为了处理物理气相沉积对芯片制造过程的影响，需要采用不同的处理策略，如电子管法、电光法、电声法和晶体凝胶法等。下面对这些策略进行详细讨论。
#### 电子管法（Electronic Insulation）
电子管法是利用各种材料配以电子电路的方式进行隔离，其中包括铜金属氧化锌、电子绝缘材料、电子枪栓、电子蓄电池、定制硅材料等。通过这种隔离技术，可以有效的防止电子泄漏、干扰和污染，从而保障芯片制造过程的安全。但由于电子管材料代价高昂、难以精确控制，因此应用电子管法的芯片制造企业往往会采用模拟电子管、反直流技术等增强隔离效果的方法。
#### 电光法（Photolithography）
电光法也是利用电子感应器对晶体中的微观颗粒施以电光照射，使其进入电子隔离材料。电光照射使得电子陷入内层空间，而外层空间则呈空旷状态。通过这种方式，可以有效的解决物理气相沉积的问题。但是，由于电子感应器的尺寸、精度、处理能力等限制，电光法的芯片制造技术目前还处于初始阶段。
#### 电声法（Electroacoustic Composites）
电声法则是利用电子采集放大设备将晶体中的微观颗粒吹走，进而将它们合并成音频信号，实现了声波的形式，并将声音转移到任何需要的人机接口处。通过这种方式，可以有效的解决物理气相沉积的问题。但是，由于电子采集放大设备的复杂程度、高温升压等技术要求，电声法的芯片制造技术目前仍然处于起步阶段。
#### 晶体凝胶法（Lattice Acoustic Devices）
晶体凝胶法是利用微元晶体的特定特性——刘慈欣提出的对称性质，将微观颗粒凝结成晶体层，形成单层晶体。之后再通过结构性扩散、温度升降、扩散系数调整等操作，可以在保证设计的空间分辨率和速度的同时，将物理气相沉积的问题消除。由于结构性扩散、扩散系数调整等操作的复杂度较高，晶体凝胶法的芯片制造技术目前仍然处于起步阶段。
