# 1.背景介绍


近年来，随着摩尔定律的失效和半导体制造工艺的革新，电子器件领域日益走向了自动化。在传统的硅基半导体产业之外，我们看到越来越多的半导体厂商采用其他材料，如锂离子聚合物、氢、汞等金属元素进行制造。近几年，无论是通过数字电路设计、集成电路制造还是单片机、微控制器等系统级芯片的研发，都涉及到纳米级的高精度三维掌握。为了在电子制造领域创造更多惊喜，科技巨头们已经不再满足于利用元件的表面受力来实现材料的嵌入，而是转向了以全新的制造工艺方法，如纳米抛光技术、自由能激发技术（FER）、电场耦合技术、场效应纳米晶体管（MEMS）、集成电路伏击技术、光刻胶材料、等离子引力冷凝技术、等等，来制造具有更强的性能和高品质的集成电路芯片。

根据我国国家标准GB/T37279-2018中的相关规定，纳米级芯片需要在原材料自然形态、材料结构、导带以及厚度等条件上做出一定的优化。其中，纳米抛光（Nanolithography）技术作为一种实用新型制造技术，能够在保持较小尺寸、低温度下对超大规模纳米材料进行二次聚焊、切割，并实现无缝、透明、密封的芯片纹样结构。该技术可产生浓稠的玻璃纤维或铜模压弯或扭曲后的直线状纹理，可以帮助制造人员解决纳米材料受力限制，提升产品的质量和性能。

现代纳米抛光技术所依赖的制造工艺流程包括：

① 物理形态生成：基于纳米管、传送带或其他抛光材料的理想形态，合成由人造晶体所组成的晶体层和各层之间的相互作用。

② 晶体层照射：将生成的晶体层按照固定排列的电路板相位运输到纳米芯片的纳米器件表面，并使其发生旋转、震动等现象。

③ 纳米层修补：修补纳米芯片表面的开口区域，并缩小造型空间以达到掩蔽其他元件或者保护晶体层之间的物理连接。

④ 镜像光学放大：由于纳米芯片的绝缘性，其表面必须经过镜像光学放大才能发出光波。镜像光学放大的过程主要依靠对晶体层上的杂波的干扰来消除其影响，从而达到最终的平滑、清晰的图像。

本文将结合纳米抛光技术的原理、技术指标、制造工艺流程、以及使用时可能遇到的一些问题进行阐述。

# 2.核心概念与联系
## 2.1 纳米材料
纳米（nanometer）是指约等于10^-9m的一米长度，是量程极限内的一个长度单位。纳米级别（nanomaterial level），也称为微纳米级（micromaterial level）、超纳米级（hypermaterial level）、亚纳米级（amino material level）等，是电子计算机、智能手机、导航终端等各类用途的必备基础。在纳米级别上研究的材料，通常是由晶体元件、半导体原件、分子等材料组成的各种复杂晶体材料。

目前，在纳米级别上研究的主要材料是“晶粒”材料——俗称的硅晶体、氧化锆、石墨烯、金刚石、四方体晶体等等。这些材料的直径一般在0.01至0.1nm之间，厚度一般为0.5至1.2nm。晶体元件的极限厚度还不到纳米级别，因此通过借助纳米级的纳米抛光技术，它们还远远不能完全被纳入制造链条。但是，由于它具有高度非晶化性，具有良好的热处理能力，可以提供有利的制造条件。

纳米级别上研究的其它重要材料还有无机非金属材料、锂离子聚合物材料、氢氧化钛材料、氮化镁材料、金属氧化物材料等等。

## 2.2 化学机械抛光技术
### 2.2.1 何为化学机械抛光？
化学机械抛光(Chemical mechanical polishing, CMP)是一种利用化学反应、工业机器人及光学仪器的手段，将含有纳米材料的晶粒材料，经特殊手法切割、磨损、冲洗、加工等技术，最终得到纹理清晰、颜色鲜艳且光泽均匀的物体。这种技术已广泛应用于制造漆蕴含纳米晶体的材料，例如聚乙烯、聚丙烯等。

纳米抛光技术中最主要的组成部分是化学物质，包括废杂质、分子以及液态化学反应剂。用于纳米抛光的废杂质主要包括碘、硫酸、硝酸、异铵等，分子则包括聚乙烯晶体中成纤维附着的黄色杂质，液态化学反应剂主要包括硫酸腐蚀剂、溶解塑料助剂、氧化剂、腐蚀剂等。

### 2.2.2 化学机械抛光的基本原理
1. 化学冲洗：在不增加粘度的情况下将杂质完全混合、溶解或烘焙的化学反应称作化学冲洗。当废杂质为溶解塑料助剂时，化学冲洗可达到较高的粘度，但其反应速度较慢；如果废杂质为固体，则可迅速完成溶解或烘焙，但粘度较差。

2. 化学切削：通过对物料的纯洁分子构象的破坏、分解、修改、甚至排除的方式，利用各种机械工具、冲洗设备及电源，就可以实现物料的纹理去皱、修饰、定型、切割。化学切削过程可分为初步切削、精细切削、维护切削三个阶段。初步切削阶段仅使用最简单的、快捷的切削工具，并仅切除杂质所在的表面；精细切削阶段则通过调整切削工具的尺寸、重心位置和动作，逐步提高切削精度；维护切削阶段则通过对精细切削的结果，如边界处理、光学增强、润湿、晾干，使切削后的物料外观、纹理整齐、光泽均匀，使它达到可供制造或加工的状态。

3. 化学加工：对纳米材料的加工需采取强化工艺、调整纳米层、保护纳米层、加工反应剂等技术。加强工艺包括采用高质量硫酸腐蚀剂、乙酸腐蚀剂、腐蚀增厚剂、溶解塑料助剂、维生素、强塑剂等，调整纳米层可通过改变纳米间距、厚度、形状等方式，保护纳米层的方法可选择自封或包覆式隔膜材料，加工反应剂则可选择辅助沉淀剂、电极膜填充剂等。

4. 化学组装：纳米材料切割之后，会产生空隙和裂纹，这些物质会阻碍光学增强、晾干、润湿等过程，所以需要进一步进行组装。可以选择采用氩气灌注技术、飞机推进技术、激光切削技术、电镀技术等。氩气灌注技术就是把无纺布的基材、残余空白层和纳米器件层接入气体传感装置，检测出氩气，根据气体流量进行喷涂。飞机推进技术就是用飞行器从下往上引出纳米器件层，再与底部基材相连，逐渐将纳米器件层连接起来。激光切削技术则是把光线探测器指向纳米材料，在切割面上投射激光束，在每次温度下保持一致的时间，即可将反射出的激光杀死纳米层中的杂质，实现纳米层的精确化裁剪。电镀技术就是在切割面上喷涂电极，将纳米器件层浸入电极膜，实现纳米层的隔离。

### 2.2.3 化学机械抛光技术的优点和局限性
1. 抗氧化能力强：纳米抛光材料所含有的废杂质有防氧化的作用，所以纳米抛光材料非常抗氧化，保证了产品的性能。同时，化学机械抛光技术不需要使用高温，这在制造成本上也比较划算。

2. 耗材少、耗能低：化学机械抛光技术只需要使用一种材料、一种工具、一种技术，所以耗材和耗能相对较少。而且纳米抛光技术可保证纳米材料在一定条件下，可避免硬化，适合制造器件、产品。

3. 操作简单：化学机械抛光技术要求操作者熟悉材料、工具和工艺流程，但它没有传统工序繁琐，操作简单、安全又便捷。

4. 反应时间短：纳米抛光过程本身较短，在每秒钟内可完成。这样做不会导致系统故障，制造速度快，成本低。

5. 可实现真空抛光：纳米抛光过程还可用于真空抛光。此方法不需要使用任何底部支持或填充层，直接将纳米器件层从空气中抛出，再与底部基材相连，就可完成纳米器件的精确化裁剪。

6. 不受制冷影响：纳米抛光过程虽然要用高温进行，但不受制冷影响，保证了产品的安全性。

7. 抗腐蚀性能好：纳米抛光材料所含有的腐蚀剂可以有效地刺激纳米器件，防止纳米器件受到腐蚀破坏。同时，纳米抛光材料的厚度较小，不容易发生持续的物理接触，所以可以起到抗腐蚀的作用。
