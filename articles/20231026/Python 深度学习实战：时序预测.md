# 2.核心概念与联系
## 时间序列（Time Series）
时间序列是指一组连续的时间点上的观察值或变量。时间序列数据的特点是按照时间顺序排列，而且通常具有周期性，如年、月、日、小时、分钟、秒等。例如，每天的销售额、股票价格、物价水平等数据都是时间序列数据。

## 时间序列预测
时间序列预测是利用过去一段时间内的数据，来预测未来某一时间点上的值。与传统的预测不同，时间序列预测更注重长期的趋势和模式，能够更好地捕捉到时间序列的整体特征。

时间序列预测可以用于：
- 金融市场，对未来市场走势的判断；
- 经济市场，对未来的经济情况的预测；
- 环境保护领域，对城市空气质量、水污染等变化进行预测；
- 物流运输领域，对航运车辆、船舶的运行状况进行预测；
- 生物医疗领域，对患者病情变化进行预测等。

## 深度学习
深度学习是一种人工智能技术，它借助于神经网络的非线性组合结构，建立复杂的模型，通过迭代、梯度下降等优化算法，可以对复杂的数据进行有效的学习和预测。由于其在图像、文本、语音等领域的成功，深度学习已成为机器学习领域的一个主流技术。深度学习时序预测技术也是利用了深度学习的强大功能，取得了一定的成果。

## 智能监控
智能监控包括两方面内容：预警监控和异常监控。在预警监控中，通过对用户操作和外部环境数据的观察，系统能够主动给予相应的提示信息，如电话告警、短信通知、邮件通知等；而在异常监控中，通过对用户操作行为、应用状态、资源占用率、负载等数据进行监控，系统能够根据检测到的异常信号，采取行动，比如暂停服务、提升容错能力、降低资源消耗等。

## 时序数据
时序数据可以从多个维度来看待。首先，时序数据有多个时间维度。例如，针对一条客户订单的生命周期，可以将其划分为“创建”、“付款”、“发货”、“收货”、“评价”等多个阶段。每个阶段又可以进一步细分为多个子阶段，如订单创建的子阶段有“填写表单”、“支付订单”等等。这样，就可以用一个三维矩阵来表示一条订单的生命周期，其中第一维代表时间，第二维代表阶段，第三维代表子阶段。

此外，时序数据通常具有持续性和暂态性。持续性指的是时间间隔较长，如一天、一周或一个月等；而暂态性指的是时间间隔较短，如几分钟、几秒或极少量的毫秒级。临界点往往比较特殊，例如股票价格的跳变，或网络连接中断等突发事件。

## 回归问题
回归问题（Regression Problem）是指对输入变量和输出变量之间的关系建模，目的是用线性函数（或其他形式）来描述输入变量和输出变量之间的映射关系，并预测输出变量的具体值。常见的回归问题包括：单变量线性回归、多元线性回归、非线性回归、时间序列回归、分类回归、多任务回归等。
