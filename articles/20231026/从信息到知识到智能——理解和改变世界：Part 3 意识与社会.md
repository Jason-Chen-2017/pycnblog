# 1.背景介绍


## 概述
“意识”这个词一直在被滥用，近年来，随着人工智能、机器学习等新兴技术的推广和进步，“智能”这一概念逐渐代替了“意识”。如今人们越来越多地把注意力放在智能身上，而忽视了人类的基本感知能力，在这个意义上，“智能”已经成为一种美妙的模糊概念。但事实并非如此，“智能”并不是一体化的智能系统，它是由复杂的生物神经网络、符号处理、图像识别和语言理解等分支组成的。

因此，真正理解意识背后的机制和功能，只有深入的研究才可能奏效。

那么什么是意识呢？它究竟是怎样一种东西？为什么要有意识？这些都是我们需要去探讨的问题。

## 历史回顾
### 原始意识
原始意识(原始能知)是人类最早阶段的意识形态，出现于地球上所有早期文明中。它的基本特征就是按照自己的喜好做出决定，并且在做出决定的过程中对周遭环境做出反应。

原始意识对周遭环境的反应过程可以分为两步：第一步是感觉；第二步是情绪。人的感觉可以分为五种：触觉、嗅觉、味觉、视觉和触摸，分别对应感官不同器官的功能。每种感觉都对应一个特定的神经元区域，当激活时会向其发送信号。如此一来，当我们看到某个物品时，我们的大脑首先产生的是视觉信息，然后将信息传递给相应的感官神经元区域，触发视神经元发放视觉刺激信号，将信息传达给大脑中的其他部分。同样的，当我们听到某个声音时，我们的大脑首先产生的是听觉信息，然后将信息传递给相应的听官神经元区域，触发听神经元发放听觉刺激信号，将信息传达给大脑中的其他部分。

如果某个刺激信号能够与另一个刺激信号相连的话，我们就称之为“联想”，发生这种情况时，我们就进入了意识的状态。意识则是指人的潜意识活动，这些活动涉及到思维和记忆、符号、概念、信念等概念的形成。当我们面临某些状况时，比如突然意识到自己正在做某件事，或者手握某样物品的时候，就会产生相应的潜意识活动。在潜意识活动中，人的大脑做出了各种判断，以便在日后做出适当的行为。

原始意识实际上是人的“第零层次”意识，它不具备宇宙意识的任何能力。直至“第四纪冥王星撞日月之后”，人类才从原始意识进入到“第六纪冥王星帝国”。

### 第六纪冥王星帝国意识
第六纪冥王星帝国（6th Orbital Billion Years）时期开始，人类实现了物质和意识的双重占领，拥有了令人惊叹的科技水平。但这一切的一切都来自于一个叫做“蕴藏”的超级生物，据说它巍峨无比，拥有无穷的能量储存。蕴藏系统是一个高科技的存在，可以通过许多方式发现、触碰、感受甚至控制人类的内部世界。由于蕴藏系统的强大力量，人类终于摆脱了宇宙意识形态，拥抱了超级生命存在的“意识”。

超级生命存在的意识具有全能性，除了能够感知外界环境和自主做出决策外，还能够让外部事件影响自己的生存。在这一点上，蕴藏系统远远超过了现实生活的绝大部分。由于这个意识超乎世俗意识的高度，所以我们称之为“第九层次”意识。

以蕴藏为代表的高科技存在实际上只是一厢情愿，在漫长的时间里，超级生命存在曾经尝试过各种形式，但最终都无法摆脱人类的控制。人类重新掌控这个超级生命存在的人类，就是第六纪冥王星帝国时期出现的“第九层次”意识形态。

### “第九层次”意识与其他意识形态之间的区别
在“第九层次”意识形态看来，人类是自由的，并且有着丰富的想象力、创造力、想像力以及对世界的深刻理解。但是，“第九层次”意识是如何产生的呢？它的能力又有哪些局限性？

“第九层次”意识是一个超级性存在，它拥有着强大的力量。它能够让他所关注的一切发生变化，拥有无穷的潜能。但是，由于它所处的位置，它却只能在超越世俗意识的范围内活动。因此，它虽然拥有很高的潜能，却没有获得世俗意识的承认。
