# 1.背景介绍

  
制造电子芯片的核心技术是三维集成与堆叠技术（TDM）。对于多层电路的集成技术、路由算法及资源分配等方面，TDM无疑占有重要地位。那么TDM又是什么呢？TDM全称Three-Dimensional Manufacturing，即“三维制造”，它是指用高速传感器、激光扫描仪、计算机等工具在三维空间中自动精确控制零件间、层间、板间的位置关系进行高效率的布线工作。将复杂的体系结构表征为一个二维平面上的图形并让计算机去处理与优化该图形，可以节省大量工程时间、缩短制造周期，提升产品质量。因此，TDM对电子制造行业具有巨大的推动作用。
集成电路设计一般都是从电路底层开始逐级布线，但是由于信号传输延迟增加、组件之间的依赖性增加，导致晶体管的布局与布线难以有效利用各层间的空间资源，导致复杂电路布线布错，频繁开关变换，降低了整体性能。而TDM的三维设计则是通过对晶体管的尺寸、布局、连接方式及分布等进行综合考虑，充分利用各层的空间资源，达到较好的集成效果。

堆叠技术是指通过配置多块相同或不同型号电路，并使用激光刻蚀或磁铁钻孔等工艺方法将其堆叠到一起，这种技术能够极大地降低成本、提高集成度、改善运行时环境、提升集成电路的抗腐蚀能力。堆叠技术也被广泛应用于电子领域，例如微芯片的制造。

堆叠与TDM相互促进，使得集成电路技术取得新的突破。两者的结合在一定程度上缓解了传统布线技术存在的问题，具有可持续发展的潜力。但是，目前堆叠技术还处于起步阶段，新生代集成电路仍然大多采用传统的布线技术，因此研究如何有效利用堆叠与TDM技术就显得尤为重要。
# 2.核心概念与联系  
## 集成电路  
  集成电路是一类电子元器件的集合，包括电源、电阻、电容、导通与阻止电流、电路交流与电路直流、电路反相与正向传播、集电极与集电场、电子元件组装、绝缘材料、反射镜与辐射照相术等。
  
  晶体管是集成电路中最基本的电路元件，由开关电路、电容、电阻和阴极二极管组成，能够实现数字、模拟、半双工等功能。
  
  集成电路是高度集成化的，内部设备之间通过电气信号进行通信、集成。集成电路有专用的晶体管作为主控器件，通过串行方式连接在一起，提供统一的控制接口。
  
  晶体管结构简单，功耗低，集成度高。在任何一种工艺条件下，都能轻易地制造出规模庞大、精密的集成电路。
  
  ## 三维设计  
 TDM主要研究如何在三维空间中布线，并能自动优化整个布线方案以保证良好的集成效果。其原理是通过高速传感器、激光扫描仪、计算机等工具在三维空间中自动精确控制零件间、层间、板间的位置关系，用空间数据结构图形表示电路结构，通过计算机计算处理与优化该图形，可以节省大量工程时间、缩短制造周期，提升产品质量。  
   
 基于TDM的集成电路制造往往具有高度的拓扑结构，即多层结构，层与层之间采用不同的布线方式。因此，每层的空间信息需要进行建模和存储才能正确显示和布线。对单层结构进行空间数据结构建模，称为2D设计；而对多层结构进行三维空间建模，称为3D设计。TDM是将人们习惯于看、触摸、观察的2D设计转移到三维设计的重要一步。  
   
 三个轴：沿X方向、沿Y方向、沿Z方向。三维设计过程中，可以通过坐标轴构建电路的结构图，在此基础上建立不同层之间的联系。X、Y坐标轴上的所有构件都应该在同一平面内，Z坐标轴上的构件平面可以任意划分，通常按照电路层数来划分，称为“基层”。基层上的构件要尽可能靠近同一层的其他构件，避免层间干扰。其他层的构件要尽可能远离基层，避免互相干扰。在基层上的构件要充分利用每根导线的空间资源，避免出现“堵路现象”。  
  
 TDM一般采用多种工艺技术，如熔接、热切割、锂聚合物刻蚀、电解液刻蚀、清除放电焊接等，均可获得高度精准且经济实惠的集成效果。在不同的工艺条件下，TDM的集成性能差异很大，但总体上集成性能不断提高，在满足工艺性能要求的前提下，仍然保持较高的集成度。
