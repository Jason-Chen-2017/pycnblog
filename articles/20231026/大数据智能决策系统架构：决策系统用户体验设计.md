# 1.背景介绍


## 概述
随着互联网的飞速发展，各种互联网业务不断涌现出来，其中有很多应用涉及到对海量数据的分析处理。如电商网站的商品推荐、微博热搜榜单生成、搜索引擎的关键词排名等。这些应用通常都需要大规模的数据分析处理才能得出最优的结果，而在数据量大、复杂性高的情况下，如何快速准确地进行分析处理已经成为非常重要的问题。为了能够处理海量数据并给出合理有效的结果，目前主流的大数据分析框架主要有Hadoop生态圈中的MapReduce、Spark等计算框架和基于机器学习的大数据分析方法。然而，对于业务人员来说，他们需要掌握各种工具的使用方法并进行比较，才能更好地理解数据的特征及规律，提升数据分析处理能力。因此，需要开发一套智能决策系统来实现基于大数据的精准决策。

本文将从决策系统的用户体验设计角度，阐述大数据智能决策系统的架构设计。通过对已有的大数据分析框架和机器学习算法的梳理，以及业务的实际需求，设计出一个通用的大数据智能决策系统架构。

## 定义
“决策”是指根据输入条件产生输出结果的过程，其目的是为了选择最优的或者适当的做法来解决某个问题或达成某种目标。“智能”指该决策系统具有学习、推理、自我修正等能力，能够对新的信息进行实时跟踪反馈，改善其决策效果。“决策系统”是指由计算机程序所组成的一系列操作和规则，用于对各种客观事物的输入、分析、处理和输出，以达到预定的目的。“大数据”是指拥有海量数据特征的集合，它是指从多个数据源收集、整理、汇总得到的信息。大数据智能决策系统，就是利用大数据的力量来优化决策过程，提高决策效率和精度。

## 特点
1. 复杂多变：当前，大数据市场面临着多元化、混杂、结构化、多样化的环境，用户行为的输入也越来越多样化、多变。大数据智能决策系统需要能够有效识别用户真正关心的内容、领域，同时从大数据中发现隐藏的模式，然后运用自身的知识、经验和技能进行判断、决策。

2. 超高并发：大数据智能决策系统应针对快速变化、超大数据量的特点进行设计。由于数据的动态性、快速增长带来的新数据的迅速孵化，超高并发、快速响应的特性要求系统要具备非常高的处理性能。在保证正确性和效率的前提下，系统需要能够兼顾资源的开销。

3. 高度可靠：当前，大数据分析框架依赖于海量数据，因此必须保证数据的完整性、一致性和正确性。数据安全、隐私保护和可用性保证是大数据智能决策系统的核心环节之一，其准确性直接影响系统的生命周期。

4. 个性化服务：大数据智能决策系统需要支持个性化服务，包括基于兴趣、偏好、行为习惯、上下文等用户特征的个性化推荐、广告投放、群发等功能。它可以帮助用户快速找到最合适的产品或服务，提升用户体验。
