# 1.背景介绍


概率论与统计是数学的一个分支领域。从古代起，人们就希望能够通过数理逻辑的观点来对世界进行研究、预测和决策。在这个过程中，数学帮助人们洞察现实世界的各种规律和机制，掌握新的认识和方法。比如著名的牛顿力学、爱因斯坦相对论等，都是基于数学方法来解决宇宙中微观现象的过程。而概率论与统计则在人工智能、数据挖掘、经济学、生物信息学等领域有着广泛应用。
概率论与统计主要涉及以下三个方面：
- 概率分布（Probability Distribution）：描述随机变量或事件发生可能性的定量分布。例如，在抛掷两个骰子时，骰子的点数服从均匀分布；随机变量X的取值服从某种分布Y，即P(X=x)表示X的取值为x时的概率。
- 概率质量函数（Probability Mass Function）：描述离散型随机变量或事件发生可能性的概率分布。例如，抛掷两个骰子后，分别出现正面、反面、双面的概率分别为1/6、1/6、2/3。
- 随机变量的期望（Expected Value）：描述随机变量或事件的平均值或中位数。
概率论与统计是计算科学、管理科学、心理学、经济学、金融学、生物学等各个领域的基础学科。因此，学习概率论与统计对于成功地开展相关工作非常重要。同时，越来越多的人开始运用概率论与统计的理论来分析和处理实际问题，这也促进了概率论与统计的发展。
# 2.核心概念与联系
## 2.1 随机变量
随机变量（Random Variable）是指一个可以取不同值的离散或连续变量。可以是简单随机变量、组合随机变量或者随机向量。比如：抛掷两枚硬币获得的结果为正面、反面、双面的硬币投掷的次数。随机变量通常由样本空间与定义在该空间上的概率分布唯一确定。

随机变量的取值可以用大写字母表示，如X、Y、Z……，其取值范围一般用小括号{}括起来，并记载其取值所对应的概率。例如：X的取值为"1点"，有0.5的概率，X的取值为"2点"，有0.25的概率。

## 2.2 概率分布
概率分布（Probability Distribution）是一个离散型随机变量的分布，它描述了随机变量或事件发生的可能性。概率分布总是把所有可能的取值映射到一个非负实数上。概率分布可以分成两种类型——概率密度函数（Probability Density Function）和概率质量函数（Probability Mass Function）。

### 2.2.1 概率密度函数（Probability Density Function，简称PDF）
概率密度函数（Probability Density Function），又叫做密度函数，是描述某个随机变量在某个点的不确定性的曲线图，由极坐标变换所得，是概率论中的基本概念。特别地，当随机变量是离散型的时候，概率密度函数就是概率质量函数。

给定一个随机变量X的取值，概率密度函数f(x)给出了在该点上的概率值。它直观地表示了随机变量X的分布形状。当随机变量X落在某个确定的区间[a, b]内时，积分区域下的面积为f(x)*dx,其中dx为区间宽度。概率密度函数具有两个显著的特征：

1. f(x)>=0，且总面积为1。这是因为概率密度函数描述的是随机变量的概率，因此只要落入某个确定的概率值区间，它的概率值就应该是非负的；而且概率值总和应该等于1。
2. 如果两个点之间存在连续的函数，那么它们在积分区域内的面积就必然是连续的。如果一个随机变量的概率密度函数是连续可导的，那么随机变量的任何连续的子集都将有一个概率密度，而不仅是单点。

### 2.2.2 概率质量函数（Probability Mass Function，简称PMF）
概率质量函数（Probability Mass Function)，又叫做质量函数，是描述离散型随机变量或事件发生可能性的概率分布的曲线图，是概率论中的基本概念。它给出了随机变量X的所有可能取值及其相应的概率。概率质量函数可以由概率分布表或分布列表示。

概率质量函数具有如下特性：

1. PMF中，每一个事件的概率的和一定等于1。这是因为随机变量X只能是有限个不同的值，每个值对应着不同的事件，而这些事件发生的概率的和应该为1。

2. 如果事件A、B互斥，即A与B不会同时发生，那么A、B的概率相乘等于1减去A、B发生的概率之和。这是因为互斥事件相互排斥，只有在某种特殊情况下，才会同时发生。例如抛掷两枚硬币，两个硬币同时出现正面朝上的概率很低。但是，若从头到尾一次抛掷两枚硬币，只有一种情况出现："两枚硬币都为正面朝上"。

3. PMF随着时间的推移不断更新，并不断修正错误的估计。例如，一个班级中高年级学生所占比例从历史最低的5%上升到了今天的7%，这么大的改变是完全不可预料的。但是，由于各种原因造成这一改变，人们仍然可以认为高年级学生所占比例依然接近5%。

## 2.3 随机变量的期望（Expected Value）
随机变量的期望（Expected Value，简称E）是指随机变量的期望值，它用来刻画随机变量或事件的均值。我们可以认为事件X发生的概率为p，事件Y发生的概率为q，则事件X和Y同时发生的概率为pq。设X的概率分布为f(x)，Y的概率分布为g(y)，则E(XY)=E(X)E(Y)。

## 2.4 连续型随机变量的概率密度函数
对于连续型随机变量X的概率密度函数，定义如下：

$$f_X(x)=\frac{d}{dx} F_X(x),$$

其中，$F_X(x)$为CDF（Cumulative Distribution Function），即分布函数。CDF描述了随机变量X的分布，即随机变量小于或等于x的概率，记作$F_X(x)$。当$F_X(x)$单调递增时，X的概率密度函数$f_X(x)$也是单调递增的。当随机变量X为正态分布时，有$F_X(x)=\Phi((x-\mu)/\sigma), \quad x \in (-\infty,\infty)$, $f_X(x)=(\frac{1}{\sqrt{2\pi}\sigma})e^{-\frac{(x-\mu)^2}{2\sigma^2}}$。此时，随机变量X的均值为$\mu$，标准差为$\sigma$。

## 2.5 联合概率、条件概率、独立性
联合概率（Joint Probability）：在给定X和Y的条件下，随机变量X和Y同时发生的概率。记作$P(X=x, Y=y)$。

条件概率（Conditional Probability）：在给定X的条件下，随机变量Y发生的概率。记作$P(Y=y|X=x)$。

独立性（Independence）：如果两个随机变量X和Y相互独立，即在给定其他变量时，两个随机变量X和Y的概率只与X、Y本身有关，即$P(X=x, Y=y)=P(X=x)P(Y=y)$，则称X和Y相互独立。

## 2.6 数字和概率之间的转换关系
数字和概率之间的转换关系主要包括如下几类：
- 对数转换：$ln (p/q) = ln p - ln q$ 。
- 投影转换：如果$R(t) = E[\exp(Xt)]$，则$R'(x) = P(X≤x)=\int_{-\infty}^{x} R(t)dt$ 。
- 分布转换：如果$X \sim N(\mu, \sigma^2)$，则$\tilde X = \dfrac{X-\mu}{\sigma}$是标准化后的随机变量。$\tilde X$服从标准正态分布。

## 2.7 求和规则
求和规则（Sum rule）：设有n个随机变量$X_1,X_2,...,X_n$, 各自独立分布，且满足独立同分布假设。则对于任意函数h(x), 有

$$E(h(X))=\sum_{i=1}^n E(h(X_i)\cdot P(X_i)), $$ 

其中，$\cdot$代表连乘运算。
