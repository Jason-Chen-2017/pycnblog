# 1.背景介绍


在现代科技革命的浪潮中，物理学、工程学、经济学等各领域都已经进入了全新时代。无论从自然界还是人类社会的角度看，所谓的科技革命都是由对一些基本假设进行质疑、抽象化、变形，用更简单、更便于理解的方式推广和扩展知识。这样的变化引起了人们对科技的怀疑与反思，从而催生出“技术危机”这一带来的种种问题，而量子力学正是这些反思的代表。近年来，由于各种物理实验技术的飞速发展，使得量子力学研究成为当下最热门的科学方向之一。
但是量子力学不是简单的宏观电磁场理论，它还有着更加复杂的内涵。一个重要的方面是量子信息——这一具有重大影响的领域，也是量子力学的热点之一。因此，了解量子信息的基本概念、发展历史和相关理论，对于理解量子力学的本质、技术突破以及未来发展至关重要。


量子信息最早是由弗里德里希·冯·诺伊曼（<NAME>）和约翰·冯·米勒（John Mills）于1982年提出的。他们把量子信息定义为“关于宇宙或世界的一组客观事实或者事件，这些事实或者事件能够被量子态所记录并在量子态的演化过程中被非物质粒子所感知、捕获、处理和传输”。换句话说，量子信息就是量子特性的一种表现形式，这种表现形式能够让人们“无意识地”处理信息，探索新世界、获取新知识和寻求解决方案。量子信息的产生依赖于量子计算机的研制和量子通信技术的进步。但由于冯氏的词不达意、缺乏严谨的思维逻辑、意识流产，其理论最终被认为没有多少学术价值。随着近几十年的技术革命，量子信息逐渐成为人们关注的热点问题。

# 2.核心概念与联系
## 2.1 基本概念
量子信息可以分为两个主要概念：信息量与熵。
### （1）信息量
信息量（Information）通常用来衡量信息的大小，也可以作为衡量信息收纳、传输以及处理的手段。这里所指的信息是客观存在的实体或现象，包括像声音、图像、文字、视频、行为、物体等。信息的物理性质决定了它能否被编码为一串电信号，即是否可以转换为某种形式的比特序列。我们一般认为信息的单位是比特（bit）。一个比特位可以表示两种状态（0或1），所以信息的真实含义需要依赖于我们使用的编码方式。目前常用的编码方式有ASCII码、二进制补码、海明码、香农码、最小距离编码、忠诚度编码、信道容量编码、译码树编码等。其中最著名的是香农码，它是最常用的非典型编码方式。每个信息用某个特定的符号集表示，如 ASCII码中的字符'A'表示数值65；二进制补码中的负数用1填充表示，例如补码-3 用 111...111 0011 转化为0b11111111110011。信息越长，信息量也就越大。
### （2）熵
熵（Entropy）是指系统不确定性的度量。它是一个物理系统内所有可能的分布或状态所拥有的最大可能的信息量。系统的混乱程度越高，熵越大。不同的系统具有不同的熵，不同的熵水平对应于不同的认知能力和智能。比如，在极限状态下，一个确定性系统的熵为零，表示完全不确定性；而在混乱状态下，一个系统的熵就越大。量子系统的熵要高于经典系统。
## 2.2 量子比特与量子信息
### （1）量子比特
量子比特（Quantum bit）是指量子力学的基本单位，是一个能够表示两个相互叠加态的量子态。一个量子比特可以表示两个量子态：|0〉和|1〉。在理想情况下，任何一个量子比特的初始状态应该处于均匀叠加态，即随机分布在 |0〉 和 |1〉之间。为了保证量子比特的可控性，量子计算必须满足以下条件：第一，不可被外部噪声影响；第二，输出结果的准确度必须高于理想情况；第三，处理效率不能低于理想情况。所以，对于量子计算的设计者来说，如何控制量子比特的可靠性和准确性，对于量子系统的稳定性至关重要。
### （2）量子信息
量子信息（Quantum information）是指利用量子通信技术传送和处理的信息。它可以理解为非 classical 的信息，因为其不受制于语言，而是存在于量子比特上的信息。量子信息可以分为两大类：一类是利用量子通信进行信息传输，另一类则是利用量子计算进行信息处理。
#### （2.1）量子通信
量子通信（Quantum communications）是利用量子通信技术实现信息传输的通信媒介。古典通信基于电磁波，而量子通信则是利用量子态的性质传播信息。量子通信的基本原理是将信息编码为量子态的表示，然后通过改变其量子态的态矢（kets）来实现信息传输。一般来说，量子通信系统由两台通信设备构成，一台作为发送端，负责生成信息对应的量子态，另一台作为接收端，负责检测并解码接收到的量子态，实现信息的接收。量子通信可以利用传送任意的量子态，只需将该量子态按照一定规律进行编码即可。同时，利用量子通信还可以进行隐蔽通信，即只有部分信息对另一方可见。这一技术具有突破性作用，可以突破传统信息传输系统的性能限制，促进量子计算和量子通信等多学科的交叉创新。
#### （2.2）量子计算
量子计算（Quantum computing）是利用量子计算技术来模拟和解决复杂的问题。量子计算利用量子力学中的固态量子电路和软件，在数量上可观测地解决计算问题。它的优势在于解决不可能的困难问题，且在计算过程中保持对输入的态射影不变，不会因噪声而导致结果错误。此外，量子计算还可以解决离散的、随机的计算问题。量子计算的关键技术在于量子计算硬件、算法和软件。其中，量子计算硬件是指使用量子计算机芯片的计算机，通过控制其中的量子比特完成计算任务；算法是指量子计算机运行的指令集合，用于描述计算过程；软件则是指用于对运行结果进行分析和解释的工具。
