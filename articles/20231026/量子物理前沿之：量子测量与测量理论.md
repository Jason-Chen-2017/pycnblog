# 1.背景介绍


量子物理（quantum physics）研究的是非平凡世界的普遍规律及其在宇宙中的微观结构和宏观行为。在科学的应用方面，利用量子现象可以精确地测量到某些宇宙现象的各种量，如电磁场强弱、粒子质量、光子的波长等。量子信息学，特别是在量子计算和量子通信领域，则主要基于量子纠缠效应的分布性假设，通过制备具有量子态的信息素子（quantum qudit）而实现信息传输。因此，了解量子测量原理对于更好地理解量子信息的运行机制和应用十分重要。
量子测量是量子物理的一个分支，主要研究如何用特定的方法将量子态从一种状态转变为另一种状态，这个过程需要用到测量者所掌握的量子信息或称为测量对象。量子测量通过非经典的手段探测到量子态的概率分布，从而使得量子系统中的物理参数发生实质上的变化，如电荷、电流、粒子的运动轨迹等。由于量子物理中存在大量的不确定性，因此量子测量也需要特别关注测量结果的可靠性和准确性，即是否会受到干扰或者噪声影响。量子信息是构建在量子测量基础上的一个新的通信和计算模型。
# 2.核心概念与联系
## 2.1 概念
### 2.1.1 量子态
在量子力学里，量子态是一个向量或矩阵形式，描述了量子系统的本征态(eigenstate)和对应的本征能量(eigenenergy)。随着时间的推移，这些本征态会演化成各种不同的态，而最终态取决于测量方式。比如，在考虑一个量子位相(spin)的两个不同取值的水分子系统，若选择以角动量的角度进行测量，则可以测得两个不同的量子态，分别对应着两个不同角度。
### 2.1.2 测量
测量是一个过程，它利用量子态的特性对其进行测量并获取相应的信息。通过测量，我们可以把一个量子态表示为某个矢量或矩阵，然后用测量的方式获得该矢量或矩阵的元素值，从而对系统的某个属性进行观察。例如，对一个量子态进行测量可以得到电子的位置和能量等信息，而对其他类型的量子态进行测量则可以得到多种信息，如核磁共振频率、辐射通量、光子的数量、自旋电子数等。
### 2.1.3 测量基底
测量基底(measurement basis)是指用来表示测量对象的基，也就是用来建立测量的坐标系。不同的测量基底定义了不同形式的测量方式，例如，以直积空间中的基矢作为测量基底可以实现测量电子的位置，而以密度矩阵作为测量基底则可以测量电子的能谱。测量基底也可能与测量器件相关联，如光子检测器就采用了不同的测量基底。
### 2.1.4 测量值
测量值(measurement outcome)是指测量过程中，在某个基底下的量子态所对应的元素的值。不同的测量方式会得到不同的测量值，如以角动量进行测量会得到关于电子的相互作用的态矢或张量；而以电子的位置进行测量则会得到不同频率的光子或中子，或甚至是核反应堆中的粒子。
### 2.1.5 测量误差
测量误差(measurement error)是指当实际测量的结果与理想情况不一致时，就会产生的偏差。由于测量过程中通常还存在噪声、量子工程技术的限制等原因，导致测量结果并不是完美无缺的，这一点尤为重要。测量误差是量子技术研究的关键难点之一，需要通过数理统计的方法，通过多次重复试验、统计分析等手段来估计测量误差的大小，从而控制其大小并减少其影响。
## 2.2 基本理论
### 2.2.1 量子态表示法
#### 2.2.1.1 Bloch球
Bloch球是量子信息与控制学的一个重要工具。它是一个三维的圆锥形图表，用来展示一个波函数(wave function)在任一点的构型。波函数由许多正弦波或余弦波组成，它们具有相同的频率，但不同相位的叠加。每条曲线都代表了不同量子态，这些态的特征与图表的颜色、高度、方向有关。
#### 2.2.1.2 密度矩阵表示法
密度矩阵表示法(density matrix representation)是描述量子态的另一种方法。它把一个量子态表示为一个厄米矩阵，其中每个元胞(elementary cell)的维度与系统中量子比特的个数相同。每个元胞的元素表示了系统中特定比特的占据概率。量子态的期望值可以用这个矩阵的特征向量乘积来表示。
### 2.2.2 量子门操作
#### 2.2.2.1 基本门
基本门是一些具有预定义功能的门，如Pauli门、CNOT门、SWAP门、Toffoli门等。这些门允许对量子系统施加各种单比特、双比特或多比特操作，并产生简单的量子电路。
#### 2.2.2.2 复合门
复合门是通过对基本门进行组合而产生的门，如C-U门、Toffoli-Fredkin门等。它们可以用更复杂的方式实现相同的功能，而且可以将多个比特同时作用在一起。
#### 2.2.2.3 可逆门
可逆门(reversible gate)是指它可以在某种程度上“反过来”操作，从而达到扩大量子信息处理能力的目的。它通常可以通过对自变量进行交换，或者通过对某个比特进行克制等方式来实现。例如，CNOT门就是一个典型的可逆门，它可以在量子态翻转之后仍然保持原来的效果。
### 2.2.3 测量操作
#### 2.2.3.1 温度蕴含关系
温度蕴含关系(thermal state approximation)描述了氢气与氧气等低温物质之间的对立。低温物质的性质与氢气非常接近，并且与液态或气体的性质不同。这样，利用温度蕴含关系，可以简单地模拟高温物质的量子态，从而可以大幅简化量子计算机的设计和实现。
#### 2.2.3.2 测量效应
测量效应(measurement effect)是指为了测量一个特定量子态，我们必须采取怎样的测量方式。测量效应的类型包括主观测量效应和客观测量效应。主观测量效应是指人们无法控制自己做出的选择，因此只能依靠自己的直觉判断，而客观测量效应则是量子系统内部根据其内部的能量分配给不同测量基底的量子态，这种效应与时间、温度、磁场等环境因素有关。
### 2.2.4 量子态重构
量子态重构(quantum state reconstruction)是指如何用已知的量子态和测量结果来重建整个系统的量子态。量子态重构的目的是要找到最佳的测量方式，以便获得足够的信息来重建整个系统的量子态。量子态重构可以分为两类，一类是直接重构(direct reconstruction)，另一类是间接重构(indirect reconstruction)。直接重构要求量子系统能够进行完整的测量，而间接重构则不需要完全测量，只需要利用部分测量结果来推断出整个系统的量子态。
### 2.2.5 测量误差与信道
测量误差(measurement error)是指当实际测量的结果与理想情况不一致时，就会产生的偏差。测量误差的大小与信道(channel)的失真度以及信道本身的信噪比(SNR)有关。如果信道失真度很小，那么SNR越大，测量误差就会越大；如果信道失真度很大，SNR越小，测量误差就会越小。为了降低测量误差，必须提升信道的失真度或引入其他信道来抵消信道的失真度。
### 2.2.6 量子力学测量
量子力学测量(quantum mechanical measurement)是指用量子力学原理对一个系统进行测量。量子力学测量通常借助于测量不变量来实现，这些不变量都是关于系统本身的量子性质，如核电子数、玻色子数、电荷量等。由于这些不变量都是量子系统的一部分，因此可以通过测量它们来得到测量值，进而得到整个系统的量子态。例如，在放置一束光束后，可以利用透镜中的投影仪或数字摄像机来记录某些量子态的特定的特征，从而推断整个系统的量子态。
