# 1.背景介绍


## 写作给予程序员巨大的价值
程序员创造了大量的财富。他们可以设计、编写程序、开发软件、管理团队等。但这些都是需要时间和精力的劳动。并且在这过程中也会产生很多无形的收入——比如说股票、债券、奖金等。而想要达到财务自由就要靠自己创造出更多的收入来。
我认为一个人如果能够用编程的方式写出高质量的作品或者文章，那么他在这个领域将获得巨大的价值。写作就是为了生计。写作既可以提升个人能力，也可以让自己脱颖而出。从前有个词叫做“坐吃山空，跟着跑舞”，即使坐吃山空也比跑不起舞。只有走出舒适区，才能真正体验跑马拉松带来的快感。

在IT行业，写作也是很重要的一项技能。因为IT行业对个人能力的要求非常高。如果你有编程方面的天赋，那么学习写作也是一个不错的选择。而目前市面上有很多优秀的写作平台，包括各种博客平台、笔记平台、知乎、简书、微信公众号等等。通过写作，你可以获取到认同感，获得自己声望的同时也能够为社会贡献自己的力量。

## 为什么写作对程序员如此重要？
写作可以给予程序员巨大的个人能力提升。通过写作，你可以学到新知识，解决工作中的困难问题，培养自己的职业道德，最终提升自己的能力。举个例子，很多程序员觉得自己编程水平太低，根本没法写好文章。但是如果你把你所学到的编程技术运用到你的文章中，让读者能够理解，他就会迫不及待地想要阅读你的文章。这种阅读欲望会促进你对编程的热情。除此之外，写作还能锻炼你的口头表达能力、组织语言能力、总结能力、反省能力等。所以，只要你拥有良好的写作习惯，你的编程技能将会得到巨大的提升。

## 哪些人适合写作？
写作是一个休闲活动，不是所有人都适合写作。首先，写作需要长期坚持。这意味着你要花费大量的时间去学习、研究写作。其次，写作需要专注于某个主题。如果你想写一本有关计算机的书，那么你需要着重于该领域的知识。再者，写作需要积极主动。你需要主动推销自己的观点，而不是被别人左右或被期待。最后，写作需要付出努力。即使你是那种不擅长写作的人，但是你也不要放弃尝试一下，多花点心思，总会有不一样的结果。

## 想成为一名自由作家吗？
如果你已经具备了编程的基础，并渴望着写出优秀的作品，那么这篇文章绝对适合你。如果你打算开始写作，那么你需要做以下准备：

1. 找到适合自己的写作平台。除了个人网站外，你还可以利用微博、QQ空间、微信公众号、知乎、简书等进行写作。选择适合自己的平台，你就可以享受到平台提供的各种服务。
2. 计划好每周的写作目标。将写作目标设定清晰，并不断反思是否满足写作需求。
3. 制订好的写作计划。制定详细的写作计划，分阶段逐步完成。
4. 充分阅读相关文献。了解相关领域的最新动态，对自己的文章进行充分挖掘。
5. 把握写作时机。如果想快速突破，可以考虑写一些跟编程毫无关系的文章。对于大多数人来说，最早的学习编程是在大学时学习C语言。

如果你还没有开始写作，但是也想开始，那么我推荐你先做好充足的准备。因为在刚开始的时候，写作可能比较吃力。希望你能够克服自身的弱点，从我分享的这几点建议开始，慢慢来吧！
