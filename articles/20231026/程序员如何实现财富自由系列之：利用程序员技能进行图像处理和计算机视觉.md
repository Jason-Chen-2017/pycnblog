# 1.背景介绍


近年来随着人工智能、大数据、云计算等新兴技术的发展，对图像处理和计算机视觉领域的研究越来越火热。图像处理是指从摄像机或相机采集到的图像中提取有用信息并加工成适合各种应用的图像，而计算机视觉则是通过机器学习技术识别出目标物体的种类、位置及属性。由于图像处理与计算机视觉属于图像识别领域的重要分支，因此，掌握了图像处理与计算机视觉相关技术的人群将能够享受到高技能、高薪的待遇。
程序员就是擅长解决实际问题，精通编程语言，具有快速学习能力和强烈求知欲的人才，可以帮助企业在各个行业领域实现财富自由。为了帮助程序员实现图像处理与计算机视觉的高级技能，作者对程序员应该具备哪些知识点进行了详尽的阐述，并且将这些知识点与程序员需要掌握的技术工具相结合，构建了一套完整的实践指南，希望能够有效地传授给程序员。
