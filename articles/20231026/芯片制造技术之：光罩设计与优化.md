# 1.背景介绍


光罩是一个金属组件，它被用在电子元器件、集成电路（IC）上。光罩能够保护电子元器件和IC免受灼伤和溅射等外界影响，提高它们的抗干扰能力和耐腐蚀性。对于一个电子产品的开发而言，光罩是非常重要的一环。然而，光罩的设计与优化一直是当前IC设计领域的一项难点课题。由于各种各样的原因，光罩设计有诸多不易，比如设计时的缺陷、光流失效、电阻增大等等。因此，光罩设计的优化对电子产品的性能、功耗、成本有着重大的意义。
光罩的设计主要分为两大类：固定光罩（Surface-mounted、SMT）和可移动光罩（Mobile）。固定光罩的特点是在制作好的硅片上安装，可以在各种不同的情况下应用；而可移动光罩则可以随身携带，适用于各种应用场景。对于可移动光罩，除了材料、尺寸、耐腐蚀性和功能等方面都需要进行优化，还有它的制造工艺，如生产工艺、装配工艺、组装工艺等等。
# 2.核心概念与联系
## 2.1. 关键参数
固定光罩的关键参数主要包括：

* 材料: 最常见的光罩材料是聚氨酯纳米复合材料，其结构性较好，硬度适中。另外还有其他材料如锡、铝、铜、玻璃等，但需要注意材料的选择对光罩的性能、表现、耐腐蚀性等方面都有一定的影响。
* 尺寸: 光罩的尺寸决定了光线传播到导体上的范围。越小的尺寸意味着更小的光域，更好地保护电子元器件；而越大的尺寸意味着更大的光域，能够提供更好的视野。一般来说，尺寸一般为10mm～30mm。
* 防护层：有些光罩为了防止静电过多导致热膨胀，在导体表面添加了一层聚焦的保护层。这一层的材料与导体相同或略有差异。
* 激光角度：激光角度决定了光罩的性能。一般来说，即使使用激光照射导体，光罩也不能完全遮住所有的光束，这时会产生黑边或衍射。因此，光罩的设计应尽可能保证激光角度足够宽。但并不是所有的光罩都有激光角度限制。
* 视野范围：视野范围指的是光罩能透过多少距离看到整个电路板。视野越大，对电路板的保护就越充分。通常来说，可移动光罩的视野范围通常要远远超过固定光罩。
* 使用环境：光罩的使用环境也是影响光罩性能的重要因素。光罩越容易在室内、室外和高温条件下工作，这样才能获得更好的性能。

可移动光罩的关键参数主要包括：

* 材料：可移动光罩一般采用聚乙烯聚氨酯钢化玻璃材质，具有耐腐蚀性强、硬度适中的特点。
* 尺寸：可移动光罩的尺寸相比固定光罩较小，这有利于提升空间利用率，但同时也增加了光罩之间的摩擦，可能会损坏光罩。
* 电源：可移动光罩的电源通常使用蓄电池，这样可以长时间保存光罩的状态。
* 工艺：可移动光罩的制造工艺包括制作工艺、装配工艺和组装工艺等不同环节。制作工艺涉及光罩材料的切割、烧制、研磨、制作工艺等环节，完成后接入电路板。装配工艺将光罩装配到导轨上，包括清洁、打包等过程。组装工艺包括把多个光罩整合在一起，形成一个整体。

## 2.2. 相关术语
### 2.2.1. 光管
光管是通过制成一根导体而得到的，导体能够承受一定数量的光。光管可以用来产生光，也可以用来接收光。通过调整光管的参数来改变光的波长、色谱和亮度。

### 2.2.2. 元件漏光
元件漏光是指电子元件因所占用的光通道过多而出现的亮度减少或者损害，从而引起整体产品的不可靠。元件漏光可能发生在绝缘导体、信号线路、无源膜电容、输出电路等多个方面。其解决方法就是选用合适的光罩。

### 2.2.3. 模拟响应
模拟响应是指光罩带来的输出电压与电压源、偏置电压、系统电压的变化之间的关系。模拟响应有三种类型：

* 对称响应：输出电压与输入电压的变化相同；
* 不对称响应：输出电压的上升沿比输入电压的上升沿延迟一点；
* 双稳态：输出电压随输入电压的变化有周期性跳动。

模拟响应的优劣主要依赖于光罩材料和导体材料的特性。越厚的导体材料，其模拟响应越好，因为导体表面富含很多微粒，可以有效地加强导体表面的抗干扰性。而光罩材料则越薄，其模拟响应越差，因为光罩材料仅具有很少的微粒。

### 2.2.4. 灵敏度
灵敏度是指输出电压与外部变化的反馈程度。由于制造光罩需要精心设计，所以灵敏度往往会低于信号线的要求。而信号线的灵敏度又受到不同的温度、湿度、光照强度的影响。因此，在设计时应该根据产品实际使用环境的具体情况进行调整。
