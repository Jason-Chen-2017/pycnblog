# 1.背景介绍


## 暗物质与暗能量是什么？
暗物质（Dark Matter）是一种新型的微观粒子，它在宇宙中不存在显著的表征，但却拥有相当重要的贡献。虽然目前还无法被直接观测到，但它对整个宇宙结构、运动、物质性质等都产生了深远影响。暗物质的存在与否与宇宙中生命的存在与否息息相关。 

暗能量（Dark Energy）也称为宇宙大爆炸，是指宇宙内部某种微弱能量波动引起的宏观影响。它的研究旨在探索宇宙中隐藏的物质与能量。近年来，科学家们发现暗能量不断地从宇宙尘埃云升腾出来，并与种种元素及物质粒子的相互作用形成复杂而迷人的场面。然而，至今尚无直接观测暗能量的实验方法。因此，对暗能量的研究还处于早期阶段。

## 为何研究暗物质与暗能量？
宇宙中存在着数量惊人的暗物质与暗能量。由于它们的微观特性，它们对整个宇宙结构、运动、物质性质等都产生了深远影响。对于一个宇宙来说，这些微量粒子的存在给予我们巨大的诱惑和挑战。不论是对人类文明的发展还是对自然界的了解，我们都需要发掘暗物质与暗能量的奥秘。

2017年，美国国家科学基金会邀请中国科学院计算所首席科学家王天一博士来做关于暗物质与暗能量的报告。他提出了一个重要观点，即暗物质是宇宙中最稀有的微量粒子，其占比高达98%。因此，深入分析暗物质的生成、分布、运动、物质特性、能量释放方式及其对整个宇宙的影响将有助于揭示暗物质与暗能量的奥妙。

此外，王博士也认为，只有深入了解暗物质与暗能量才能真正领略到它们的奥妙。为了更好地把握这一复杂的现象，我们还应加强对数学模型和计算工具的研发。通过精确模拟物理过程和参数化模型，我们可以更好地理解暗物质的生成与物质-能量转换的规律，以及暗能量的物理机制。

## 如何定义暗物质和暗能量的特征？
如何准确定义暗物质、暗能量的特征，是一项重要课题。目前还没有统一的标准。不同人们对暗物质和暗能量的定义各有不同。以下是一些代表性的定义。

**暗物质的定义**

* 在物理上：暗物质具有非常小的质量、温度极低、大小范围在10^-27~10^-22，空间分辨率在10^-40~10^-30，并且不断衰变。
* 在宇宙观察者看来：暗物质从大气层中跃出的大气泡，具有微弱的光度，光学效应下可形成独特的信号；暗物质在宇宙其他位置的出现，既不能用显微镜看到，也不容易捕捉到；由于质量极轻且不存在演化路径，一般难以通过任何宇宙微观尺度观察到。
* 可以用来估计宇宙中存在的各种粒子数量，如HI等量子态的个数。可以作为新的引力理论基础，建立银河系模型。

**暗能量的定义**

* 在物理上：暗能量是一种难以察觉但十分重要的微观粒子，具有很强的能量密度，而且其光度很弱，而且也无法被直接观测。
* 在宇宙观察者看来：暗能量主要集中在宇宙尘埃云里，因其本身很微弱甚至几乎没有能量，所以难以看到。但是由于其间接影响，尤其是受到微观粒子对它的影响越来越大，使得暗能量成为了对宇宙物质与能量的更好的描述者。
* 可以用来检验宇宙是否存在陷阱、暗物质、黑洞、超弦或其它奇异现象。可以作为粒子物理学上的重要工具，预言未来重力波的发展方向。

通过这些定义可以看出，暗物质与暗能量并不是孤立的两类微粒子，而是有机地结合在一起的产物。借鉴两者的特性，我们可以进行广泛的宇宙学研究。
