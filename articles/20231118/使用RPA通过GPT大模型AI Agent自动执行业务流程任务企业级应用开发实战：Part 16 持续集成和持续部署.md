# 2.核心概念与联系
## 2.1 GPT-3、NLP、机器学习、深度学习等名词的含义以及联系
**1) GPT-3:**

GPT-3 是一种无模型的、由自然语言生成的 AI 技术。它是用“强化学习”训练出来的，而这种训练方法类似于模仿大脑神经网络的行为模式。在训练过程中，GPT-3 在不断迭代调整模型参数，使得它逐渐成为一个可以理解、自然语言生成的模型。

总之，GPT-3 可以根据输入的文本，自动生成符合语法规则、语义意图、风格统一的语句或段落。

**2) NLP(Natural Language Processing):**

自然语言处理（NLP），是指计算机科学的一门学科，它研究如何处理及运用自然语言。其研究的重点是设计与研制计算机系统，使其能够理解、分析和产生人类语言；促进人类与计算机之间的通信和交流。

NLP 的核心工作是从原始文本中抽取特征（Feature Extraction），然后将这些特征用于分类、预测、翻译、评估和管理任务。而基于 NLP 技术的聊天机器人、自动问答系统、搜索引擎、数据挖掘分析、金融交易决策等各项应用都离不开 NLP 技术。

**3) 机器学习、深度学习：**

机器学习（Machine Learning）是一个领域，它研究如何让计算机基于数据（Data）来做某些预测或者决策，并改善它的性能。机器学习的目标就是让计算机系统通过试错的方式，逐步地自我完善，最终达到某个任务所需的最佳状态。

深度学习（Deep Learning）则是机器学习的一个子领域，它研究的是基于大数据、神经网络等的多层次学习。深度学习的关键技术是神经网络，也就是人们熟知的多层感知器（Neural Network）。

## 2.2 RPA与AI自动化技术的区别与联系
RPA(Robotic Process Automation)和AI自动化技术虽然都是为了实现业务流程自动化，但二者之间还是存在很大的差异。RPA侧重于工作流自动化、信息采集、处理、编排，流程可视化、可编程化；而AI自动化侧重于深度学习、NLP等机器学习技术，实现模式识别、图像识别、语音识别、自然语言处理、推荐系统等功能。

两者最大的不同在于：RPA旨在实现精细化的业务流程自动化，适应性强，具有较高的易用性和灵活性；而AI自动化侧重全局考虑，能够更好地理解复杂的业务流程，在一定范围内做出正确的决策。因此，二者不能完全颠覆对方，也不能替代另一方。
