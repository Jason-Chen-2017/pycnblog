                 

# 1.背景介绍


文件操作是计算机编程中经常用到的功能之一。在现代应用领域越来越复杂、繁多的情况下，文件的操作更是成为一个绕不开的话题。

本文将以简单的读写文件举例，通过分析Python语言提供的相应模块和方法，来学习文件的基本操作技能。文章适合初级到中级Python开发者阅读。

# 2.核心概念与联系
## 2.1 文件操作术语
首先，让我们熟悉一些基本的文件操作术语，如“文件”、“目录”、“打开方式”等。

1. 文件（File）：指的是在存储设备上的数据，通常以二进制形式保存。在Windows操作系统中，文件通常存在于磁盘上，可以被创建、修改或删除；而在Unix/Linux操作系统中，文件可保存在内存中或者外存上。
2. 目录（Directory）：是用来存储文件的逻辑分组，类似文件夹的概念。在Unix/Linux系统中，目录也称作文件夹。目录由其中的文件名、目录名、连接符以及属性组成。
3. 路径（Path）：是指从根目录（即磁盘上的目录）到指定文件的所需的一系列目录及文件名。路径以"/"（斜杠）或"\\"（反斜杠）作为分隔符。
4. 打开方式（Open Mode）：打开文件的模式，用于设置文件操作的权限和行为。主要包括以下几种：
   - r：只读模式，只能读取已存在的文件内容。
   - w：写入模式，可新建或覆盖已存在的文件内容。
   - a：追加模式，可向已存在的文件后面添加新内容。
   - b：二进制模式，用于处理非文本文件（比如图片、视频）。

## 2.2 文件操作函数
Python中提供了很多文件操作函数，这里我们先了解其中最常用的三个函数。

1. open(file, mode)

   使用open()函数打开一个文件，并返回一个文件对象。

   参数：

   file -- 文件路径或文件描述符（整数类型），如果为字符串，则表示要打开的文件的名称。
   mode -- 打开文件的模式，'r' 表示只读模式，'w' 表示写入模式，'a' 表示追加模式，'b' 表示二进制模式。

   返回值：

   如果成功，则返回一个文件对象。否则抛出 IOError 异常。

2. read([size])

   从文件中读取指定的字节数，或直到文件的结尾。

   参数：

   size -- 可选参数，读取的字节数。默认为-1，表示读取整个文件。

   返回值：

   返回读取的内容，字符串类型。

3. write(string)

   将字符串写入文件末尾。

   参数：

   string -- 需要写入的内容。

   返回值：

   返回实际写入的字符数量，整数类型。

4. close()

   关闭文件。

   参数：

   无。

   返回值：

   无。

## 2.3 文件对象属性
文件对象除了上面提到的一些方法外，还有一些属性可以使用。

1. name: 获得文件对象的名字。
2. closed: 检测文件对象是否已关闭。
3. mode: 获取打开文件的模式。
4. encoding: 获取文件编码格式。

## 2.4 其他相关模块

os模块提供了更多的文件操作函数，如获取当前工作目录、改变当前工作目录、创建目录、删除文件、获取环境变量等。

shutil模块提供了对文件的高级操作，如复制、移动、重命名等。

tempfile模块提供了生成临时文件的相关函数。
