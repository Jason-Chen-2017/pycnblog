# 2.核心概念与联系
## 2.1 模块导入
模块导入是指从其他源文件（如文件、目录、压缩包）动态导入一个或多个模块到当前脚本文件中，并可以使用该模块中的对象、函数和变量。比如，可以从sys模块中获取当前系统的平台信息，也可以从os模块中获取文件和目录的信息。模块导入的方式包括以下几种：
1. import语句
2. from-import语句
3. 内置函数__import__()函数
## 2.2 模块搜索路径
当使用import语句或者from-import语句导入模块时，Python解释器必须定位所需模块的位置。Python查找模块的过程如下：
1. 当前目录（若存在）
2. 搜索路径列表（环境变量PYTHONPATH的值）
3. 默认安装位置
4. 如果都找不到模块则报ImportError错误。
## 2.3 模块定义
模块是一个独立的文件，其中包含可执行的代码，以及可以在其他地方使用的函数、类和变量。在模块中定义的函数、类、变量统称为命名空间，它的内容可以通过点运算符访问。为了让其他模块导入这个模块，必须通过以下三步进行定义：
1. 创建模块文件
2. 在模块文件开头添加特殊注释
3. 使用__all__变量声明要导出的对象名称
