# 2.核心概念与联系
## 2.1 条件判断语句 if...elif...else
条件判断语句又称为选择结构或分支结构，它是一种重要的流程控制语句。它根据条件（True/False值），决定是否执行特定代码块。一般来说，条件判断语句可分为两种：

1. if...else语句

   ```
   if condition:
       # 执行的代码块
   else:
       # 当条件为 False 时执行的代码块
   ```
   
2. if...elif...else语句

   ```
   if condition_1:
       # 执行的代码块 1
   elif condition_2:
       # 执行的代码块 2
   elif condition_3:
       # 执行的代码块 3
  ...
   else:
       # 当所有条件都不满足时执行的代码块
   ```

条件判断语句用于根据不同的情况执行不同的代码块，是程序执行的关键因素之一。由于其结构清晰、方便阅读和书写，因此应用非常广泛。 

## 2.2 循环结构 for...in...while
循环结构也是一种重要的流程控制语句。循环可以重复执行一个代码块，直到指定的条件满足为止。通常情况下，循环结构包括两个部分：

1. 初始化：初始化循环次数或变量等；
2. 循环体：执行循环内的代码块；

对于for...in...while语句，其中for指的是for-each，即遍历集合元素，而in则表示迭代对象中的每个元素；while则表示当指定条件为真时才继续执行循环。 

```python
for item in collection:
    # 每次迭代collection中的元素时执行的代码块
while condition:
    # 当condition为True时执行的代码块
```

循环结构在程序运行过程中起着至关重要的作用，是构建复杂程序的基石。

## 2.3 关系运算符及逻辑运算符 
关系运算符用来比较两个值之间的大小关系。常见的关系运算符有==、!=、>、<、>=、<=。 

逻辑运算符是用来连接关系运算符以构造复杂的条件表达式的。常见的逻辑运算符有and、or、not。

## 3.1 if...else 语句

### （1）概念定义
`if`是一个条件判断语句，它具有三种形式：

1. `if`语句：该语句只有一个条件，当这个条件为真的时候，执行语句块里面的内容；否则跳过。
2. `if-else`语句：该语句有两个条件，当第一个条件为真的时候，执行第一个语句块里面的内容；当第一个条件为假并且第二个条件为真的时候，执行第二个语句块里面的内容；当两个条件都为假的时候，跳过。
3. `if-elif-else`语句：该语句有多个条件，第一个条件不成立时，执行下一个条件的内容，直到某个条件成立时，执行相应的语句块的内容。否则，执行最后一个语句块的内容。

### （2）条件表达式

条件表达式是用来确定程序执行路径的关键部分。在`if`语句中，表达式作为判定条件，只有值为`True`时才执行语句块。

例如：

```python
if x > y:
    print("x is greater than y")
```

在这个例子中，`x > y`是条件表达式，如果其值为`True`，则执行语句块的内容，输出`"x is greater than y"`。反之，如果其值为`False`，则跳过。

### （3）循环结构嵌套

循环结构也可以嵌套在其他循环结构中，形成多层的循环结构。

```python
for i in range(1, n+1):   # 外层循环
    result = []           # 存储每轮循环的结果
    for j in range(i):    # 内层循环
        num = randint(1, 9)      # 生成随机数字
        result.append(num)       # 将数字加入结果列表

    print(f"第{i}轮结果:{result}")
```

在这个例子中，外层循环生成`n`个随机数，内层循环获取对应个数的随机数，并将它们添加到列表`result`。这样，便可以在不同层循环中实现不同功能，灵活应对各种需求。

### （4）else 语句

`else`语句是在条件为`False`时的默认执行代码块，跟在`if`语句后面即可。如果条件为真，那么执行对应代码块；如果条件为假，那么执行`else`代码块。

例如：

```python
a = 10
b = 20

if a < b:
    print("a is less than b")
else:
    print("a is not less than b")
```

在这个例子中，`a < b`的表达式的值为`True`，所以执行第一个打印语句；然而，如果`a=20`的话，由于`a`大于等于`b`，所以执行了第二个打印语句。

### （5）多重条件

`if-elif-else`语句允许编写多个条件判断，当多个条件都为真的时候，只执行第一个条件对应的代码块，跳过后续条件。

例如：

```python
age = 27

if age >= 18 and age <= 60:
    print("you can work or study.")
elif age < 18:
    print("you are too young to work or study.")
else:
    print("you are too old to work or study.")
```

在这个例子中，条件表达式可以写为`age >= 18 and age <= 60`，如果该表达式为真，则执行第一句话；如果表达式不为真，那么会自动进入到`elif`语句，判断另一个条件表达式`age < 18`，如果其为真，则执行第二句话；否则，执行第三句话。

### （6）嵌套语句

可以使用括号来组合多个语句，如：

```python
temp = (min + max) / 2
print(temp)
```

## 3.2 while 语句

### （1）概念定义

`while`语句是一种循环结构，它的结构和之前介绍的`for`类似，但有些不同：

1. `while`语句不需要事先知道循环次数。每次执行循环体，都会检查条件表达式，如果为`True`，则执行循环体，然后继续检查条件表达式，直到表达式为`False`。
2. `while`语句允许嵌套循环，也就是说，一个循环体里面还可能有一个或多个循环结构。
3. 如果条件表达式始终为`True`，那么循环体永远不会结束，除非你手动退出循环。

### （2）循环条件

`while`语句的循环条件由布尔表达式构成，该表达式必须返回布尔值，`True`表示循环继续，`False`表示循环退出。一般格式如下：

```python
while condition:
    # 循环体
```

### （3）例子

```python
count = 0
total = 0

while count < 10:
    total += count     # 每次循环计算累加值
    count += 1         # 计数器自增
    
average = total / 10
print("平均值:", average)
```

在这个例子中，条件表达式为`count < 10`，循环一次增加`count`的值，并把`count`的值加入到变量`total`中，直到`count`的值大于等于10。最终，计算`total`的平均值并输出。

### （4）死循环

如果条件表达式始终为`True`，则循环将一直继续，程序可能进入无限循环状态。为了防止这种情况，你可以设置一个超时时间，超过指定时间之后，循环会自动退出。

```python
import time

start_time = time.time()

while True:
    if time.time() - start_time > 10:        # 设置超时时间为10秒
        break                               # 超时退出循环
    
    # 你的程序代码
```

### （5）循环嵌套

```python
count = 0

while count < 5:
    num = input("请输入数字:")
    try:
        number = int(num)                     # 转换输入值为整数类型
    except ValueError:                        # 如果转换失败，显示提示信息
        print("输入错误！请输入一个整数。")
        continue                              # 跳过当前循环
        
    if number % 2 == 0:                       # 判断奇偶性
        print(number,"是偶数。")
    else:
        print(number,"是奇数。")
    
    count += 1                                # 计数器自增
```

在这个例子中，外层循环共执行五次，每次输入一个数字，并对其进行偶数判断。若输入值为整数，且为偶数，则显示`该数字是偶数`，否则显示`该数字是奇数`。输入值为空字符，则显示`输入错误！请输入一个整数。`。

### （6）无限循环

使用条件表达式作为循环条件，不能忘记包含初始条件的情况，否则会导致无限循环。

```python
n = 0

while n < 5:
    print(n)
    n -= 1          # 修改条件表达式为永不为真

# 此处进入无限循环，程序只能手动停止运行
```
