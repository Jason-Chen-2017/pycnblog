# 2.核心概念与联系
## RPA简介
### （一）什么是RPA
简单来说，RPA就是指利用计算机软件来替代人类干预、自动完成某些重复性繁重的工作，目前最常用的工具是微软的Automate系列产品，包括PowerPoint Designer、Word 2019 VBA、Excel Macro、Outlook Outlook Workflow等等。

### （二）RPA产品与服务
Microsoft Power Automate是一款云服务，提供一系列的创作工具和流程引擎，用户可利用该服务快速创建、部署、运行或管理自动化解决方案。例如，可以通过Power Point Designer来设计用PPT模板编排出来的流程，然后上传到Power Automate，就可以实现自动化审批、数据处理、文档生成、项目跟踪等功能。

IBM Watson Assistant则是另一款云服务，也是一种RPA产品，它提供了基于文本、语音、图像、动作指令等多种输入形式的对话式交互。IBM Watson 还提供了强大的规则引擎支持，用户可以根据自己的业务需求定制交互逻辑。

还有很多其他的RPA产品，比如Cobot、Saboteur、Automation Anywhere等。

## AI Agent简介
### （一）什么是AI Agent
AI Agent即人工智能（Artificial Intelligence，AI）系统，它是一个具有智能功能的计算机程序或者智能体。AI Agent既可以独立完成某项任务，也可以作为主动参与者参与进某个环境中，并模仿人类的行为做出相应反应。

### （二）分类与特征
根据AI Agent的功能，主要分为三大类：
1. 数据采集和处理：采集数据并进行分析后存储至数据库；
2. 决策和执行：根据收集到的信息进行决策，执行相应操作；
3. 模拟人类行为：能够让AI按照人的行为方式行动。

常见的AI Agent产品有Microsoft Bot Framework、Wit.ai、Dialogflow、LUIS等。

## GPT-2语言模型简介
GPT-2是Google推出的一种语言模型，基于BERT（Bidirectional Encoder Representations from Transformers）结构，采用Transformer编码器结构，预训练目标为语言建模和翻译任务，目前已超过10亿参数量，并开源。


## 自动化应用开发简介
### （一）自动化应用
自动化应用是指通过计算机软件技术，为企业自动执行重复性繁重的工作，应用场景如支撑型企业内部信息流转、订单处理、客户服务中心支持等。常见的自动化应用类型有：
1. 业务流程管理
2. IT服务管理
3. 技术支持

### （二）企业级自动化应用开发
企业级自动化应用开发通常包括以下几个方面：
1. 业务流程的自动化。通过设计并部署基于RPA的自动化业务流程，提升组织工作效率。
2. 技术支持的自动化。利用技术支持的RPA Agent，把传统的电话、网络技术支持转换为人机协助模式，实现客户满意度的提高。
3. 日常工作任务的自动化。利用人工智能技术，实现工作报告自动化，减少人力消耗，提升工作效率。

企业级自动化应用开发，除了要考虑开发人员的能力、知识储备之外，还要考虑公司资源、规范、法规等一系列因素。
