# 2.核心概念与联系
## 2.1什么是GUI？
GUI（Graphical User Interface，图形用户接口）是指电脑桌面上用于用户操作的图形化界面，它是由窗口、菜单栏、工具栏、控件组成。GUI程序可以有效提升用户体验，简化操作流程，降低使用难度。它不仅让用户直观感受到程序的运行效果，而且还可以提升用户的沟通协作能力。


## 2.2Python GUI库
目前，主流的Python GUI库包括Tkinter、wxPython、PyQt、PySide、Kivy、Remi等。其中，Tkinter和WxPython是最古老的两款GUI库，其他都是最近几年才出现的。

- Tkinter: 是最早发布的GUI库，基于Tk toolkit实现，支持Tk版本的所有操作系统。其语法和命令集都较为丰富。提供了Tkinter中最基本的GUI元素——widget，如按钮、标签、输入框、弹出菜单、列表框、滚动条等。但它的缺点也很明显，比如性能差、外观小气。
- WxPython: 是由ActiveState公司开发，它是一个跨平台的GUI库，主要基于wxWidgets库。它提供了丰富的控件，包括菜单栏、按钮、复选框、单选框、文本框、滚动条、下拉列表、对话框等。但是，由于它的跨平台特性，使得它更适合用于编写多平台上的应用。同时，它还有着良好的文档和示例，可供参考。
- PyQt: 是在Python 2.x版本中引入的GUI库，随后扩展到了Python 3.x版本，支持Windows、Mac OS X、Linux平台。其控件集丰富、组件设计精巧，可以快速创建出复杂的GUI应用。它的控件绑定、事件驱动、面向对象等特点也成为其重要优点。
- PySide: 是另一个跨平台的GUI库，基于Qt组件库。它的控件和功能相比于Tkinter和WxPython都更加齐全，而且控件的渲染效果也更加美观。
- Kivy: 是为python编程语言编写的开源GUI库，它内置了许多功能强大的控件，例如滑块、进度条、表单、按钮等。使用Kivy可以轻松制作跨平台的应用程序，并可以在许多移动设备上运行。
- Remi: Remi是一个基于浏览器的Python GUI库，可以用于快速开发交互式网页。使用Remi可以快速地生成HTML5、CSS3、JavaScript、SVG、Canvas、WebSockets等应用。
