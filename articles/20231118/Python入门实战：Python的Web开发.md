                 

# 1.背景介绍


## 一、web开发简史
- 1991年，蒂姆·伯纳斯-李（Tim Berners-Lee）在CERN开发了第一代WEB浏览器——Mosaic；
- 1994年，NCSA推出了WorldWideWeb(WWW)浏览器，它允许用户在因特网上查看信息、进行交易和其它交互；
- 1995年，因特网信息中心（NIC）为了获取更多的信息流量，将HTTP协议升级到1.1版本，并新增了对CGI、CSS等功能的支持；
- 1997年，万维网联盟（W3C）发布了HTML 4.01版本，规范了HTML的结构和语义；
- 2000年，Mozilla基金会开发了Mozilla Firefox浏览器，支持CSS、JavaScript等现代网页特性；
- 2005年，微软推出Internet Explorer 8，这是最初的“企业级”浏览器，占据了市场主流地位；
- 2007年，谷歌推出Chrome浏览器，对网络安全、扩展性、性能做出了很大的改善。
## 二、Web开发简介
### （一）Web开发概述
Web开发是指利用计算机技术及相关软件工具，通过网页制作与服务器技术实现从无到有的网站建设过程。通过多种编程语言，服务器环境配置，数据库管理等技能，Web开发人员可以快速构建自己的个人或企业网站，向广大访问者展示其产品、服务或者品牌形象。Web开发涉及众多领域，包括网页设计、网站建设、网络营销、网页制作、后台开发、数据库管理、运维维护等。以下将对Web开发的各个阶段及关键知识点作简要介绍。
#### Web开发四个阶段
- 招聘找工作：应届生求职和公司招聘环节。掌握本行业的核心竞争力和技能，善于总结经验教训，准确谈薪酬福利条件，搭建稳定的工作关系。
- 搭建开发环境：熟悉系统环境设置、安装各种开发工具、调试技巧等。掌握至少一种后端语言，了解相关框架和库的使用方法。
- 前端开发：主要使用HTML、CSS、JavaScript完成静态页面的制作，也可使用jQuery等框架和插件进行动态效果的实现。掌握JavaScript面向对象编程、事件处理机制、ajax通信等技术。
- 后端开发：使用Java、PHP、Python、Ruby、Node.js、Golang等各种编程语言实现后台业务逻辑的开发。掌握至少一种数据库技术，如MySQL、MongoDB等。
#### Web开发关键知识点
- HTML：超文本标记语言，用于描述网页的内容，并配合CSS、JavaScript一起使用实现美观、动态的网页效果。
- CSS：层叠样式表，用于定义网页的外观、布局及行为，使网页具有更好的视觉效果和可用性。
- JavaScript：高级脚本语言，用于实现网页的动态效果。
- HTTP协议：超文本传输协议，是Web开发过程中必不可少的基础。
- MySQL数据库：关系型数据库，应用最广泛的数据库之一。
- MongoDB数据库：NoSQL数据库，提供非关系型数据存储，基于分布式文件存储模式。
- RESTful API：一种基于HTTP协议的API标准，通过接口方式访问服务器资源。
- AJAX：Asynchronous JavaScript and XML，异步请求数据的Web开发技术。
- ORM：Object Relation Mapping，对象-关系映射，是一种编程技术，用于从关系数据库表中自动生成对应的对象，简化开发流程。
- MVC模式：Model-View-Controller模式，是一个设计模式，用于分离应用程序的关注点，提高代码的复用率和可维护性。
- 云计算：云计算是一种通过网络提供即时、按需、高度可扩展的计算能力的服务。
- DevOps：DevOps是Development（开发）和Operations（运维）的组合词汇，它强调开发者和IT运维之间互相合作，共同打造更好的软件服务。
- 测试自动化：自动化测试，是指让计算机软件执行一系列重复任务，检测软件质量是否符合要求，是一种有效的管理手段。
- CI/CD：CI/CD，即持续集成与持续部署，是一种全新的软件开发方式，用于减少软件开发过程中的停滞，促进软件频繁交付更新。
