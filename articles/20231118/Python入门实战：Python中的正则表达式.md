# 2.核心概念与联系

## 2.1 什么是正则表达式？

正则表达式（Regular Expression，简称regex或regexp）是用于匹配字符串中字符组合的模式。它是由<NAME>在1986年提出，后被 IEEE Computer Society 的 POSIX 标准所采纳并成为 IEEE Std 1003.1-2017 定义的正则表达式语法。正则表达式可以让我们轻松地进行文本匹配、替换、搜索及数据提取等多种文本处理任务。除了使用正则表达式之外，还有其他方式也可以对文本进行处理，比如 awk 和 sed 命令等，但它们都没有像正则表达式那样灵活方便。

虽然正则表达式很强大，但它的语法却十分简单。它主要由以下三个部分组成：

1. 元字符：正则表达式的基本单位，表示某种特定含义的字符或者序列。这些字符包括：`^ $. [ ] { } \ | ( )`。
2. 限定符：可选的特殊字符，它们修饰着元字符，用来控制匹配行为，如 `*`、`+`、`?` 等。
3. 非打印字符：一些不可见的 ASCII 字符，例如制表符、换行符等。

在编写正则表达式时，通常用斜杠`/`来界定开头结尾，即`/`开头的是完整正则表达式，`/foo/`指代整个字符串"foo", `/bar/i`表示区分大小写。

下面是一个例子：

```python
import re

pattern = r'\b[A-Z][a-z]+\s[A-Z][a-z]+'
text = 'The quick brown fox jumps over the lazy dog.'
match = re.search(pattern, text)
if match:
    print('Match found:', match.group())
else:
    print('No match found.')
```

这个例子使用 `\b` 来匹配单词的边界， `[A-Z]` 是大写字母，`[a-z]+` 是一系列小写字母，`\s` 表示空白字符， `[A-Z]` 表示大写字母，`[a-z]+` 表示一系列小写字母。正则表达式 `\b[A-Z][a-z]+\s[A-Z][a-z]+` 可以匹配 "The" 或 "the" 等单词，"quick" 或 "Quick" 等单词，"brown" 或 "Brown" 等单词，"fox" 或 "Fox" 等单词，"jumps" 或 "Jumps" 等单词，"over" 或 "Over" 等单词，"lazy" 或 "Lazy" 等单词，"dog." 或 "Dog." 等单词。注意到 `\b` 匹配单词的边界，因此它确保了匹配只有完整的单词。如果把 `\b` 去掉，那么 `\w+\s+\w+\s+\w+\.\s*\w*` 模式就可以匹配到所有句子。
