1. 动态规划：动态规划是一种解决最优化问题的算法方法，它通过构建一个动态规划表来存储子问题的解决方案，从而避免重复计算。动态规划的核心思想是将一个复杂问题分解为多个子问题，然后通过构建动态规划表来解决这些子问题。

动态规划的具体操作步骤如下：

1. 初始化动态规划表，将其初始化为0。
2. 遍历问题的所有子问题，并计算每个子问题的解决方案。
3. 更新动态规划表，将每个子问题的解决方案存储在动态规划表中。
4. 从动态规划表中获取问题的最优解。

动态规划的数学模型公式为：

$$
dp[i] = \min_{j=0}^{i-1} dp[j] + cost(i, j)
$$

2. 贪心算法：贪心算法是一种解决最优化问题的算法方法，它通过在每个步骤中选择当前最佳解决方案来逐步构建最终解决方案。贪心算法的核心思想是在每个步骤中选择当前最佳解决方案，以便在整个算法过程中得到最佳解决方案。

贪心算法的具体操作步骤如下：

1. 初始化算法的当前解决方案。
2. 遍历问题的所有子问题，并选择当前最佳解决方案。
3. 更新算法的当前解决方案，将当前最佳解决方案添加到当前解决方案中。
4. 重复步骤2和步骤3，直到问题的最优解得到。

    n = len(arr)
    dp = [1] * n

    for i in range(1, n):
        for j in range(i):
            if arr[i] > arr[j] and dp[i] < dp[j] + 1:
