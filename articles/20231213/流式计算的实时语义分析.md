# 2.核心概念与联系

## 2.1 流式计算
流式计算是一种处理大规模数据流的计算模型，它可以实时处理数据，并提供实时的分析结果。流式计算的核心思想是将数据流看作是一个无限大小的数据集，通过定义一系列操作符，可以对数据流进行实时处理和分析。

## 2.2 实时语义分析
实时语义分析是一种实时数据分析方法，它可以在数据产生的同时对数据进行分析，提供实时的分析结果。实时语义分析的核心思想是将数据分析任务拆分为多个小任务，每个小任务对应一个操作符，通过流式计算的方式实现实时分析。

## 2.3 联系
流式计算和实时语义分析是密切相关的。流式计算提供了实时处理数据流的能力，实时语义分析则利用流式计算的能力，实现对数据流的实时分析。
