# 2.核心概念与联系
ScyllaDB是一个分布式数据库，它使用C++编写，具有高性能和高可用性。ScyllaDB使用一种称为CRDT（Conflict-free Replicated Data Types）的数据结构，这种数据结构可以在多个节点上进行并发更新，而无需锁定。这使得ScyllaDB具有高度并发性和高性能。
