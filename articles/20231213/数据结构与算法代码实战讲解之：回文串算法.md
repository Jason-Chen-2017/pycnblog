## 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

### 3.1暴力比较法

暴力比较法是一种简单直观的方法，它通过将字符串从头到尾进行比较，如果发现任何不相等的字符，则判断该字符串不是回文串。如果所有字符都相等，则判断该字符串是回文串。

暴力比较法的时间复杂度为O(n^2)，其中n是字符串的长度。空间复杂度为O(1)，因为只需要常数级别的额外空间。

### 3.2KMP算法

KMP算法是一种基于字符串匹配的算法，它可以用于判断非对称回文串。KMP算法的核心思想是使用next数组来记录每个字符的下一个匹配位置，通过这个next数组来减少字符串比较的次数。

KMP算法的时间复杂度为O(n+m)，其中n是字符串的长度，m是模式的长度。空间复杂度为O(m)，因为需要额外的next数组来存储匹配位置。

### 3.3Manacher算法

Manacher算法是一种基于动态规划的算法，它可以用于判断非对称回文串。Manacher算法的核心思想是将字符串扩展为偶数长度的字符串，然后通过动态规划来记录每个字符的最长回文子串长度。

Manacher算法的时间复杂度为O(n)，其中n是字符串的长度。空间复杂度为O(n)，因为需要额外的数组来存储最长回文子串长度。

```python
def kmp(s, p):
    n = len(s)
    m = len(p)
    next = [-1] * m
    j = -1
    for i in range(1, m):
        while j >= 0 and p[j + 1] != p[i]:
            j = next[j]
        if p[j + 1] == p[i]:
            j += 1
        next[i] = j
    j = -1
    for i in range(n):
        while j >= 0 and s[i] != p[j + 1]:
            j = next[j]
        if s[i] == p[j + 1]:
            j += 1
        if j == m - 1:
            return i - m + 1
    return -1

def is_palindrome(s):

    n = len(p)

        while j >= 0 and p[j + 1] != p[i]:

            j += 1
        next[i] = j
    return kmp(s, s) != -1
```

### 4.3Manacher算法

```python
def manacher(s):
    n = len(s)
    p = [0] * (2 * n + 1)
    C = R = 0
    for i in range(1, 2 * n):
        if R > i:
            p[i] = min(p[2 * C - i], R - i)
        while s[i - p[i] - 1] == s[i + p[i]]:
            p[i] += 1
        if i + p[i] > R:
            C, R = i, i + p[i]
    return p

def is_palindrome(s):
    p = manacher(s)
    n = len(s)
    for i in range(n):
        if p[2 * i + 1] == 2 * n - 1 - i:
            return True
    return False
```
