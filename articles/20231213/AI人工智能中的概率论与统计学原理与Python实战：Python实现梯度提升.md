```python
def gradient_boosting(X, y, n_estimators=100, learning_rate=0.1, max_depth=3, random_state=42):
    n_samples, n_features = X.shape
    n_iterations = n_estimators
    y_pred = np.zeros(y.shape)

    for i in range(n_iterations):
        # 随机选择一部分样本
        random_idx = np.random.randint(0, n_samples, size=int(n_samples * learning_rate))
        X_sample = X[random_idx]
        y_sample = y[random_idx]

        # 计算损失函数的梯度
        gradients = np.zeros(n_features)
        for j in range(n_features):
            X_j = X_sample[:, j]
            X_j_pos = X_j + np.sign(X_j) * 0.1
            X_j_neg = X_j - np.sign(X_j) * 0.1
            y_pos = y_sample + (X_j_pos - X_sample).dot(gradients)
            y_neg = y_sample + (X_j_neg - X_sample).dot(gradients)
            gradients = (y_pos - y_neg) / (2 * 0.1)

        # 更新模型
        y_pred += learning_rate * gradients

    return y_pred
```

### 4.5训练模型

然后，我们需要训练模型：

```python

```

### 4.6评估模型

最后，我们需要评估模型：

