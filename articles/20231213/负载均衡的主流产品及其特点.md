## 4.具体代码实例和详细解释说明

### 4.1 基于IP地址的负载均衡示例

```python
import random

def load_balance(requests, servers):
    server_ips = [server['ip'] for server in servers]
    while requests:
        server_ip = random.choice(server_ips)
        server = next(server for server in servers if server['ip'] == server_ip)
        request = requests.pop(0)
        server['requests'].append(request)
```

在上述代码中，我们首先定义了一个负载均衡函数`load_balance`，它接受两个参数：`requests`（请求列表）和`servers`（服务器列表）。

然后，我们从`servers`列表中提取出所有服务器的IP地址，并将其存储在`server_ips`列表中。

接下来，我们使用`while`循环来处理所有的请求。在每次循环中，我们从`server_ips`列表中随机选择一个服务器IP地址，并将其与服务器列表中的服务器进行匹配。如果匹配成功，我们将当前请求添加到该服务器的请求列表中，并将当前请求从`requests`列表中移除。

def load_balance(requests, servers):
    server_loads = [server['load'] for server in servers]
    while requests:
