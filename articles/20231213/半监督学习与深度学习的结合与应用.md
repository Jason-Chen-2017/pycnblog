## 1.4 半监督学习的具体代码实例与解释

以下是一个具体的半监督学习代码实例：

```python
import numpy as np
import tensorflow as tf
from tensorflow.contrib import learn

# 数据预处理
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
Y = np.array([[0], [1], [1], [0]])

# 算法选择
model = learn.DNNClassifier(hidden_units=[10, 10], feature_columns=X,
                            n_classes=2, optimizer=tf.train.AdamOptimizer(0.01))

# 模型训练
model.fit(X, Y, epochs=1000)

# 模型评估
accuracy = model.evaluate(X, Y)
print("Accuracy:", accuracy)
```
