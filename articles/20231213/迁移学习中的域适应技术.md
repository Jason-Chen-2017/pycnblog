# 2.核心概念与联系

在迁移学习中，我们需要将已有的模型迁移到新的领域或任务上，以便在新的领域或任务上更好地进行预测。域适应技术是一种迁移学习方法，它可以帮助我们在新的领域或任务上快速训练模型，从而提高模型的泛化能力。

域适应技术的核心概念包括：

- 源域：源域是我们已有的数据来自于的领域，我们已经训练好的模型来自于这个领域。
- 目标域：目标域是我们需要迁移模型到的新领域，我们需要在这个领域进行预测。
- 共享层：共享层是我们在源域和目标域上训练的相同的层，这些层可以帮助我们在目标域上快速训练模型。
- 特定层：特定层是我们在源域和目标域上训练的不同的层，这些层可以帮助我们在目标域上进行更好的预测。

域适应技术的核心思想是：通过在源域和目标域上训练相同的共享层，我们可以在目标域上快速训练模型，从而提高模型的泛化能力。同时，通过在源域和目标域上训练不同的特定层，我们可以在目标域上进行更好的预测。
