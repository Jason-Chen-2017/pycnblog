# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来解释禅意面向对象设计的具体应用：

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError("子类必须实现 speak 方法")


class Dog(Animal):
    def speak(self):
        return "汪汪汪"


class Cat(Animal):
    def speak(self):
        return "喵喵喵"


def main():
    animals = [Dog("旺财"), Cat("小花")]

    for animal in animals:
        print(animal.speak())


if __name__ == "__main__":
    main()
```

在这个例子中，我们定义了一个 `Animal` 类，它有一个名字和一个 `speak` 方法。我们还定义了两个子类 `Dog` 和 `Cat`，它们 respective 实现了 `speak` 方法。

在 `main` 函数中，我们创建了一个 `animals` 列表，包含了一个狗和一只猫。然后，我们遍历这个列表，并调用每个动物的 `speak` 方法。

这个例子展示了禅意面向对象设计的一个具体应用，我们通过继承和多态来实现程序的可扩展性和可维护性。
