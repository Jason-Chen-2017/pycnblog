                 

# 1.背景介绍

随着互联网和数字技术的发展，安全监控已经成为组织和企业的核心需求。安全监控系统的主要目标是检测和预防潜在的安全威胁，以保护组织的信息资源和物理资产。然而，传统的安全监控系统存在两个主要问题：假阳性率较高，准确率较低。假阳性是指安全监控系统误报潜在的安全威胁，而实际上并没有发生任何安全事件。这种误报可能导致安全专家忽视真正的安全威胁，从而导致组织的信息资源和物理资产受到损害。

为了解决这些问题，人工智能（AI）技术被应用于安全监控系统，以降低假阳性率并提高准确率。AI技术可以帮助安全监控系统更有效地识别和分类安全事件，从而提高其准确性。
