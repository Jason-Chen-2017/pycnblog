# 4.具体代码实例和详细解释说明

在这里，我们以一个简单的例子来演示数据预处理与特征工程的融合：

```python
import pandas as pd
import numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.feature_selection import SelectKBest
from sklearn.feature_selection import chi2

# 数据清洗
data = pd.read_csv('data.csv')
data = data.dropna()  # 删除缺失值
data = pd.get_dummies(data)  # 一元一变
data = StandardScaler().fit_transform(data)  # 标准化

# 特征选择
X = data.iloc[:, :-1]  # 特征矩阵
y = data.iloc[:, -1]  # 目标变量
selector = SelectKBest(score_func=chi2, k=5)  # 选择前5个最相关的特征
X_new = selector.fit_transform(X, y)

# 特征工程
X_new = np.hstack((X_new, np.mean(X, axis=1).reshape(-1, 1)))  # 平均值特征
X_new = np.hstack((X_new, np.sum(X, axis=1).reshape(-1, 1)))  # 和特征
X_new = np.hstack((X_new, np.prod(X, axis=1).reshape(-1, 1)))  # 积特征
```

在这个例子中，我们首先对数据进行清洗，包括删除缺失值、进行一元一变和标准化。然后，我们使用相关性评分函数chi2进行特征选择，选择前5个最相关的特征。最后，我们进行特征工程，创建平均值特征、和特征和积特征。
