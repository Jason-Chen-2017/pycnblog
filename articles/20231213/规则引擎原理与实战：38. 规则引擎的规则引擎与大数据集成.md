```python

fact = {"age": 20, "gender": "male"}

# 定义规则
rule = {"age": ">=", "value": 18, "then": "adult"}

# 评估规则
def evaluate_rule(fact, rule):
    for key, operator, value in rule.items():
        if operator == ">=":
            if fact[key] >= value:
                return True
    return False

# 执行动作
def execute_action(fact, rule):
    if evaluate_rule(fact, rule):
        print("This person is an adult.")

# 执行动作
execute_action(fact, rule)
```

在这个代码实例中，我们首先定义了一个事实（`fact`）和一个规则（`rule`）。然后，我们定义了一个`evaluate_rule`函数来评估规则是否满足事实。接着，我们定义了一个`execute_action`函数来执行规则满足时的动作。最后，我们调用`execute_action`函数来执行动作。
