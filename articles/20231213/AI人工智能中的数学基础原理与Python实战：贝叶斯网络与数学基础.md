# 2.核心概念与联系

## 2.1 贝叶斯定理

贝叶斯定理是一种概率推理方法，它可以帮助我们从已知事件的概率分布中推断未知事件的概率分布。贝叶斯定理的数学公式为：

$$
P(A|B) = \frac{P(B|A) \times P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示已知事件B发生的条件下事件A发生的概率，$P(B|A)$ 表示已知事件A发生的条件下事件B发生的概率，$P(A)$ 表示事件A发生的概率，$P(B)$ 表示事件B发生的概率。

贝叶斯定理可以帮助我们从已知事件的概率分布中推断未知事件的概率分布，从而实现对未知事件的预测和决策。

## 2.2 贝叶斯网络

贝叶斯网络是一种用于建模和预测随机事件之间关系的图形模型。它由一组随机变量和它们之间的条件依赖关系组成。每个随机变量可以是离散的（如布尔值）或连续的（如数值）。贝叶斯网络可以用来建模复杂的随机系统，并利用这些模型进行预测和决策。

贝叶斯网络的核心思想是利用贝叶斯定理来描述随机事件之间的关系。每个随机变量在贝叶斯网络中都有一个条件概率分布，这个分布描述了该变量在其父变量给定的条件下的概率。通过利用贝叶斯定理，我们可以计算出每个随机变量在整个网络中的条件概率分布。
