## 2.2 社会网络效应的类型

社会网络效应可以分为以下几类：

1. 直接影响：一个节点的行为直接影响其他节点的行为。例如，一个人在社交网络上发布的信息可以直接影响其他人的信息传播。

2. 间接影响：一个节点的行为通过其他节点影响其他节点的行为。例如，一个人在社交网络上发布的信息可以通过其他人传播给其他人。

3. 强效应：一个节点的行为对其他节点产生较大的影响。例如，一个人在社交网络上拥有很多关注者，他的发布内容对其他人的信息传播产生较大影响。

4. 弱效应：一个节点的行为对其他节点产生较小的影响。例如，一个人在社交网络上拥有很少关注者，他的发布内容对其他人的信息传播产生较小影响。

### 4.1.1 数据收集代码实例

```python
import networkx as nx
import pandas as pd

# 读取社交网络数据
data = pd.read_csv('social_network_data.csv')

# 创建网络
G = nx.from_pandas_edgelist(data, source='from', target='to', edge_attr='weight')
```

### 4.1.2 数据收集详细解释说明

在这个代码实例中，我们首先使用`pandas`库读取社交网络数据。然后，我们使用`networkx`库创建网络，其中`from`和`to`分别表示节点之间的关系，`weight`表示关系的强度。
