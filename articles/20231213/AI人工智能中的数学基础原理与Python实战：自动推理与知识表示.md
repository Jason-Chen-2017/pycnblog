## 4.1 自动推理

我们将通过一个简单的逻辑推理实例来详细解释自动推理的实现方法。

```python
# 事实
事实 = {
    "人类": "动物",
    "狗": "动物"
}

# 规则
规则 = {
    "动物": "可以走路"
}

# 自动推理
def 自动推理(事实, 规则):
    # 初始化推理结果
    推理结果 = {}

    # 遍历事实
    for 动物, 类别 in 事实.items():
        # 遍历规则
        for 类别, 行为 in 规则.items():
            # 判断类别是否相同
            if 类别 == 动物:
                # 更新推理结果
                推理结果[动物] = 行为

    # 返回推理结果
    return 推理结果

# 执行自动推理
推理结果 = 自动推理(事实, 规则)

# 打印推理结果
print(推理结果)
```

上述代码实例中，我们首先定义了事实和规则。事实包括人类和狗的类别，规则包括动物可以走路。然后，我们定义了自动推理函数，该函数通过遍历事实和规则来得出结论。最后，我们执行自动推理函数并打印推理结果。
