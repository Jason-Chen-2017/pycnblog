## 4. 具体代码实例和详细解释说明

### 4.1 代码注入攻击的例子

以下是一个简单的Python代码实例，展示了代码注入攻击的例子：

```python
import sqlite3

def query(sql):
    conn = sqlite3.connect('example.db')
    cursor = conn.cursor()
    cursor.execute(sql)
    return cursor.fetchall()

sql = input('请输入SQL语句：')
print(query(sql))
```

在这个例子中，我们使用Python的sqlite3库来连接数据库，并执行SQL语句。但是，我们没有对用户输入的SQL语句进行任何验证和过滤，这导致了代码注入攻击的风险。

### 4.2 防止代码注入攻击的方法

为了防止代码注入攻击，我们可以采取以下方法：

1. 对用户输入的数据进行验证和过滤，以确保其安全性。
2. 使用参数化查询，而不是直接拼接SQL语句。
3. 使用安全的编程语言和框架，如Go、Rust等。

以下是修改后的代码实例，展示了如何防止代码注入攻击：

```python
import sqlite3

def query(sql, params):
    conn = sqlite3.connect('example.db')
    cursor = conn.cursor()
    cursor.execute(sql, params)
    return cursor.fetchall()

sql = 'SELECT * FROM users'
params = ('%',)
print(query(sql, params))
```

在这个例子中，我们使用参数化查询来执行SQL语句，从而防止代码注入攻击。
