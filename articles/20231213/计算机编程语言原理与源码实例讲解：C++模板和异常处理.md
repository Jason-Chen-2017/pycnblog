## 4.1 C++模板代码实例

在这个代码实例中，我们将创建一个泛型的栈类，使用C++模板实现。

```cpp
template <typename T>
class Stack {
public:
    void push(T value);
    T pop();
    T top();
    bool isEmpty();

private:
    std::vector<T> elements;
};

template <typename T>
void Stack<T>::push(T value) {
    elements.push_back(value);
}

template <typename T>
T Stack<T>::pop() {
    if (elements.empty()) {
        throw std::runtime_error("Stack is empty");
    }
    T value = elements.back();
    elements.pop_back();
    return value;
}

template <typename T>
T Stack<T>::top() {
    if (elements.empty()) {
        throw std::runtime_error("Stack is empty");
    }
    return elements.back();
}

template <typename T>
bool Stack<T>::isEmpty() {
    return elements.empty();
}
```

在这个代码实例中，我们定义了一个泛型的栈类，使用C++模板实现。我们定义了一个模板参数T，表示栈中的元素类型。我们实现了push、pop、top和isEmpty等方法，使用模板参数T进行操作。

## 4.2 C++异常处理代码实例

在这个代码实例中，我们将创建一个简单的计算器类，使用C++异常处理实现。

```cpp
class Calculator {
public:
    int add(int a, int b) {
        if (b == 0) {
            throw std::runtime_error("Division by zero");
        }
        return a + b;
    }
};

int main() {
    Calculator calculator;
    int a = 10;
    int b = 0;
    try {
        int result = calculator.add(a, b);
        std::cout << "Result: " << result << std::endl;
    } catch (const std::runtime_error& e) {
        std::cerr << "Error: " << e.what() << std::endl;
    }
    return 0;
}
```

在这个代码实例中，我们定义了一个计算器类，实现了一个add方法，用于计算两个整数的和。如果b为0，我们将抛出一个std::runtime_error异常。在main函数中，我们尝试调用add方法，如果发生异常，我们将捕获异常并处理。
