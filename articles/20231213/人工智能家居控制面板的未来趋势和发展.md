# 2.核心概念与联系

## 2.1 传感器
传感器是人工智能家居控制面板的基础设施，用于收集家居环境的各种数据，如温度、湿度、光线、空气质量等。传感器通常采用无线技术进行数据传输，如蓝牙、无线LAN等。

## 2.2 控制器
控制器是人工智能家居控制面板的核心组件，负责收集传感器数据，并根据用户设定和算法规则进行数据分析和处理，从而实现家居环境的智能控制。控制器通常采用微控制器或小型计算机硬件实现。

## 2.3 通信设备
通信设备是人工智能家居控制面板与外部设备的桥梁，实现数据的传输和交换。通信设备可以采用各种通信协议，如ZigBee、Wi-Fi、蓝牙等。
