在评估教育改革的成果时，我们需要使用一些算法来计算相关的指标。这些算法包括：

1.教育质量指数（EI）：教育质量指数是衡量教育质量的一个重要指标。EI的计算公式为：

$$
EI = \frac{\sum_{i=1}^{n} w_i \times Q_i}{\sum_{i=1}^{n} w_i}
$$

其中，$w_i$ 是各个质量指标的权重，$Q_i$ 是各个质量指标的得分。

2.教育成果指数（F）：教育成果指数是衡量教育成果的一个重要指标。F的计算公式为：

$$
F = \frac{\sum_{i=1}^{m} w_i \times R_i}{\sum_{i=1}^{m} w_i}
$$

其中，$w_i$ 是各个成果指标的权重，$R_i$ 是各个成果指标的得分。

3.教育改革成果评估指数（G）：教育改革成果评估指数是通过计算EI和F来得到的。G的计算公式为：

$$
G = \frac{EI + F}{2}
$$

通过计算G，我们可以得到教育改革的成果评估结果。

```python

def calculate_ei(weights, quality_scores):
    return np.sum(weights * quality_scores) / np.sum(weights)

def calculate_f(weights, result_scores):
    return np.sum(weights * result_scores) / np.sum(weights)

def calculate_g(ei, f):
    return (ei + f) / 2

# 示例数据
weights_e = [0.3, 0.4, 0.3]
quality_scores_e = [80, 85, 90]

weights_f = [0.3, 0.4, 0.3]
result_scores_f = [85, 90, 95]

ei = calculate_ei(weights_e, quality_scores_e)

g = calculate_g(ei, f)

print("教育质量指数：", ei)
print("教育成果指数：", f)
print("教育改革成果评估指数：", g)
```
