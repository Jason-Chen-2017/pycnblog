                 

# 1.背景介绍

无服务架构（Microservices Architecture）是一种新兴的软件架构风格，它将应用程序划分为多个小型服务，每个服务都独立部署和扩展。这种架构的出现，为应用程序的可扩展性、可维护性和可靠性提供了更好的支持。

函数计算（Function Computing）是一种基于无服务架构的计算模型，它将计算作为服务进行提供，用户可以通过API调用来使用这些计算服务。这种模型的出现，为开发者提供了更加简单、高效的计算资源。
