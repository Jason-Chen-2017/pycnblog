### 1.8.2 如何提交代码修改？

提交代码修改需要遵循项目的开发规范和流程，具体步骤如下：

1. 克隆项目：使用 Git 克隆项目到本地计算机。

2. 创建分支：创建一个新的分支，用于存放你的代码修改。

4. 提交代码：使用 Git 提交代码修改到分支中。

6. 等待审核：等待项目维护者进行代码审核，并根据审核结果进行修改。

7. 代码合并：如果代码审核通过，项目维护者将会将你的代码合并到项目中。
