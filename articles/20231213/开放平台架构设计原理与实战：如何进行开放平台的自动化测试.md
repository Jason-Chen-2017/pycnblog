```python
import requests
import unittest

class TestOpenPlatformAPI(unittest.TestCase):
    def setUp(self):
        self.url = "https://www.example.com/api"

    def test_get_user_info(self):
        response = requests.get(f"{self.url}/user/info")
        self.assertEqual(response.status_code, 200)
        self.assertIn("user_id", response.json())

    def test_post_user_info(self):
        data = {"user_id": 1, "name": "John Doe"}
        response = requests.post(f"{self.url}/user/info", json=data)
        self.assertEqual(response.status_code, 201)
        self.assertIn("user_id", response.json())

if __name__ == "__main__":
    unittest.main()
```

在这个代码实例中，我们使用Python的unittest模块来编写测试用例。我们首先导入了requests模块，用于发送HTTP请求。然后，我们定义了一个TestOpenPlatformAPI类，继承自unittest.TestCase类。在这个类中，我们定义了两个测试方法：test_get_user_info和test_post_user_info。

在test_get_user_info方法中，我们使用requests.get方法发送GET请求，并检查响应的状态码和响应体。在test_post_user_info方法中，我们使用requests.post方法发送POST请求，并检查响应的状态码和响应体。

最后，我们使用unittest.main()方法运行所有的测试用例。
