```python
import unittest
from selenium import webdriver
from selenium.webdriver.common.keys import Keys

class TestSecurity(unittest.TestCase):
    def setUp(self):
        self.driver = webdriver.Firefox()

    def test_login(self):
        self.driver.get("http://www.example.com/login")
        self.driver.find_element_by_name("username").send_keys("admin")
        self.driver.find_element_by_name("password").send_keys("password")
        self.driver.find_element_by_name("submit").click()

        # 检查是否登录成功
        self.assertEqual(self.driver.title, "Dashboard")

    def tearDown(self):
        self.driver.quit()

if __name__ == "__main__":
    unittest.main()
```

在这个代码实例中，我们使用了 Python 的 unittest 模块来进行端到端测试的安全性测试策略。我们首先创建了一个 TestSecurity 类，并继承了 unittest.TestCase 类。然后，我们使用 setUp 方法来初始化我们的测试环境，这里我们使用 Selenium 的 webdriver 模块来初始化我们的 Firefox 浏览器。

接下来，我们使用 test_login 方法来进行我们的登录测试。首先，我们使用 get 方法来访问我们的登录页面。然后，我们使用 find_element_by_name 方法来找到我们的用户名和密码输入框，并使用 send_keys 方法来输入我们的用户名和密码。接下来，我们使用 find_element_by_name 方法来找到我们的提交按钮，并使用 click 方法来点击提交按钮。

最后，我们使用 assertEqual 方法来检查我们是否登录成功。如果我们的标题是 "Dashboard"，则说明我们登录成功。如果我们的标题不是 "Dashboard"，则说明我们登录失败。
