# 2.核心概念与联系

## 2.1 地球科学与地球地质学的关系

地球科学是研究地球的科学，包括地球的形状、大小、组成、运动等方面。地球地质学则是地球科学的一个分支，研究地球的内部结构和地貌。地球地质学与地球科学的关系可以类比为地球地质学是地球科学的一个子集。

## 2.2 地球内部的地貌

地球内部的地貌包括地壳、氛围、地貌等。地壳是地球的表面，包括陆地和海洋。氛围是地球的大气层，包括大气、氩层和地磁场等。地貌是地球内部的地质结构和地质物质，包括岩石、矿物、水分等。

## 2.3 地球外部的影响

地球外部的影响包括地球的气候、海洋、生态等。气候是地球的大气层和海洋的温度、湿度、风速等因素的平均值。海洋是地球表面的水体，包括大洋、小海、湖泊等。生态是生物、地球和环境之间的相互作用，包括生物多样性、生态系统、生态平衡等。
