# 2.核心概念与联系

在图像分析和计算机视觉领域，有一些核心概念和联系需要我们了解。这些核心概念和联系包括：

1. 图像处理：图像处理是指对图像进行预处理、增强、压缩、恢复、分割等操作，以提高图像质量或减少图像数据量。图像处理技术是图像分析和计算机视觉的基础技术。

2. 图像特征提取：图像特征提取是指从图像中提取出有意义的特征，以便进行图像识别、分类、检测等任务。图像特征提取技术是图像分析和计算机视觉的核心技术。

3. 图像识别：图像识别是指从图像中识别出对象或场景，以便进行图像分类、检测等任务。图像识别技术是图像分析和计算机视觉的应用技术。

4. 图像分类：图像分类是指将图像分类到不同的类别中，以便进行图像识别、检测等任务。图像分类技术是图像分析和计算机视觉的应用技术。

5. 图像检测：图像检测是指从图像中检测出对象或场景，以便进行图像识别、分类等任务。图像检测技术是图像分析和计算机视觉的应用技术。

6. 深度学习：深度学习是一种基于神经网络的机器学习技术，它可以处理大量的图像数据，并自动学习出图像的特征。深度学习技术是图像分析和计算机视觉的核心技术。
