# 4.具体代码实例和详细解释说明
在这里，我们通过一个简单的排序问题来展示禅意编程和禅意算法设计的应用。

问题：给定一个整数数组，请编写一个程序，用于对数组进行排序。

我们可以使用以下的禅意算法来解决这个问题：

```python
def sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
print(sort(arr))
```

在这个代码实例中，我们使用了禅意编程的思维方式，将问题简化为一个排序问题，并使用了禅意算法设计的方法来解决问题。我们的代码是简洁的，易于理解和维护。
