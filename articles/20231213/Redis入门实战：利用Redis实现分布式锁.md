                 

# 1.背景介绍

Redis是一个开源的高性能的key-value数据库，它支持数据的持久化，可基于内存也可基于磁盘（即Redis支持将某些key-value存储在磁盘上，若查询频繁的放入内存中），并具有Master-Slave复制、列式存储、集群等高级功能。

Redis支持通过Pub/Sub机制实现消息通信，通过Lua脚本实现事务。Redis还支持有序的字符串(Redis Sorted Set)，可用于实现排行榜类型的应用。Redis还支持定期Snapshot保存，即定期将内存内的数据保存在磁盘上，采用ADVANCED格式的RDB文件，可以减少磁盘空间的占用。

Redis是一个使用ANSI C语言编写、遵循BSD协议的开源软件（源代码开放）。Redis的核心团队成员为 Salvatore Sanfilippo ，目前Redis的开发已经由Redis Ltd公司负责。

Redis的核心特性有：数据持久化、备份、重plication、集群、负载均衡、Lua脚本、高性能、安全、原子性、分布式锁等。

Redis的核心数据结构有：字符串(String)、哈希(Hash)、列表(List)、集合(Sets)、有序集合(Sorted Sets)、位图(Bitmaps)、hyperloglogs、Geospatial。
