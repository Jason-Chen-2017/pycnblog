# 2.核心概念与联系

## 2.1 什么是第三方库

第三方库是指那些不属于Python的内置库的库，它们提供了许多有用的功能和工具，可以帮助程序员更快地完成项目。这些库可以通过Python的包管理系统（如pip）进行安装和管理。

## 2.2 为什么需要第三方库

Python内置库提供了许多基本的功能，但在实际项目中，我们可能需要更高级的功能和工具来提高开发效率和项目的可扩展性。这时我们就需要使用第三方库。

## 2.3 如何选择第三方库

选择第三方库时，我们需要考虑以下几个因素：

- 功能：第三方库应该具有我们项目需要的功能。
- 性能：第三方库的性能应该满足我们的需求。
- 稳定性：第三方库应该是稳定的，不容易出现bug。
- 活跃度：第三方库应该有活跃的开发者社区，以便我们可以获得更好的支持和更新。
