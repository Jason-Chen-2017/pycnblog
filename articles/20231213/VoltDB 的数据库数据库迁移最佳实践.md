## 4.具体代码实例和详细解释说明

```java
// 创建 VoltDB 数据库
VoltDatabase db = new VoltDatabase("mydb");

// 创建表
VoltTable table = db.createTable("mytable", "id INT PRIMARY KEY, name VARCHAR(255)");

// 导入数据
List<Map<String, Object>> data = new ArrayList<>();
data.add(Collections.singletonMap("id", 1L)
      .put("name", "John"));
data.add(Collections.singletonMap("id", 2L)
      .put("name", "Jane"));
db.insert(table, data);

// 测试和优化性能
VoltQuery query = db.query(table);
List<Map<String, Object>> result = query.fetch();
System.out.println(result);
```
