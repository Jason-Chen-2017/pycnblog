# 2.核心概念与联系

在计算机图形学中，核心概念包括图像、图形、几何体、光照、纹理、动画等。这些概念之间存在着密切的联系，它们共同构成了计算机图形学的基本框架。

图像是计算机图形学的核心概念之一，它是由像素组成的二维矩阵。图像可以用来表示图形、几何体、光照、纹理等。图像的处理是计算机图形学的基础，包括图像输入、处理、存储、输出等。

图形是计算机图形学的核心概念之一，它是由几何体组成的三维场景。图形可以用来表示物体、场景、环境等。图形的处理是计算机图形学的核心，包括几何计算、光照计算、纹理计算等。

几何体是计算机图形学的核心概念之一，它是由点、线、面组成的几何对象。几何体可以用来表示物体、场景、环境等。几何计算是计算机图形学的基础，包括几何变换、几何运算、几何关系等。

光照是计算机图形学的核心概念之一，它是用来描述物体表面光线分布的属性。光照可以用来表示物体的颜色、光泽、阴影等。光照计算是计算机图形学的核心，包括光源模型、光照模型、阴影计算等。

纹理是计算机图形学的核心概念之一，它是用来描述物体表面纹理的图像。纹理可以用来表示物体的表面纹理、颜色、光泽等。纹理计算是计算机图形学的基础，包括纹理映射、纹理滤波、纹理重复等。

动画是计算机图形学的核心概念之一，它是用来描述物体在时间上的变化的过程。动画可以用来表示物体的运动、变形、碰撞等。动画计算是计算机图形学的核心，包括动画模型、动画运算、动画控制等。
