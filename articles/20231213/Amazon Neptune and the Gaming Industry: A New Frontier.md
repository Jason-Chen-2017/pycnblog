```python
import networkx as nx

# 创建一个图数据库
G = nx.Graph()

# 加载数据
G.add_node('Alice')
G.add_node('Bob')
G.add_node('Charlie')
G.add_edge('Alice', 'Bob', weight=1)
G.add_edge('Bob', 'Charlie', weight=2)

# 执行最短路径算法
shortest_path = nx.shortest_path(G, 'Alice', 'Charlie')

# 解析结果
print(shortest_path)  # ['Alice', 'Bob', 'Charlie']
```

在这个代码实例中，我们创建了一个图数据库，并加载了一些数据。然后，我们执行了最短路径算法，以查找从节点 'Alice' 到节点 'Charlie' 的最短路径。最后，我们解析了算法的结果，并输出了最短路径。
