```python
def group_data(data):
    age_groups = {
        '0-18': [],
        '18-35': [],
        '35-50': []
    }
    for age in data['age']:
        age_groups[str(age // 10) + '-' + str((age // 10 + 1) * 10)].append(data)
    return age_groups
```

最后，我们可以使用以下代码实现数据加密：

```python
def encrypt_data(data):
    for key, value in data.items():
        if key == 'password':
            data[key] = hashlib.sha256(getpass.getpass().encode()).hexdigest()
    return data
```
