# 2.核心概念与联系

## 2.1 规则引擎的基本概念

规则引擎是一种基于规则的系统，它可以根据一组预先定义的规则来处理数据和进行决策。规则引擎的核心组成部分包括规则库、工作流程和知识库。规则库存储了一组预先定义的规则，工作流程负责根据规则库中的规则进行处理，知识库存储了与规则相关的信息。

```python
    "rule1": {"condition": "age < 18", "action": "reject"},
    "rule2": {"condition": "age >= 18 and age < 60", "action": "approve"},
    "rule3": {"condition": "age >= 60", "action": "special_approve"}
}

# 定义输入数据
input_data = {"age": 20}

# 执行规则引擎
def rule_engine(rules, input_data):
    for rule_name, rule in rules.items():
        if eval(rule["condition"], input_data):
            return rule["action"]
    return "unknown"

# 执行结果
result = rule_engine(rules, input_data)
print(result)  # output: "approve"
```
