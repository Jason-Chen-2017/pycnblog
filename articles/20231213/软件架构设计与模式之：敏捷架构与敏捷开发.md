## 1.4 具体代码实例和详细解释说明

以下是一个具体的代码实例，以及详细的解释说明：

```python
class Component:
    def __init__(self):
        self.name = None

    def set_name(self, name):
        self.name = name

    def get_name(self):
        return self.name

class ConcreteComponent(Component):
    def __init__(self):
        self.name = "ConcreteComponent"

class Decorator:
    def __init__(self, component):
        self._component = component

    def set_name(self, name):
        self._component.set_name(name)

    def get_name(self):
        return self._component.get_name()

class ConcreteDecoratorA(Decorator):
    def __init__(self, component):
        super().__init__(component)

    def set_name(self, name):
        super().set_name(name + "A")

class ConcreteDecoratorB(Decorator):
    def __init__(self, component):
        super().__init__(component)

    def set_name(self, name):
        super().set_name(name + "B")

component = ConcreteComponent()
component.set_name("Component")
print(component.get_name())  # 输出：Component

decorator_a = ConcreteDecoratorA(component)
decorator_a.set_name("DecoratorA")
print(decorator_a.get_name())  # 输出：ComponentDecoratorA

decorator_b = ConcreteDecoratorB(decorator_a)
decorator_b.set_name("DecoratorB")
print(decorator_b.get_name())  # 输出：ComponentDecoratorADecoratorB
```

在这个代码实例中，我们实现了一个简单的组件和装饰器模式。通过组件和装饰器模式，我们可以实现系统的可变性、可扩展性、灵活性和易于维护。
