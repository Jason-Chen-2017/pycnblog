## 4.具体代码实例和详细解释说明

以下是一个具体的代码实例，用于说明如何设计一个高度互动的游戏玩法：

```python
import random

class Player:
    def __init__(self, name):
        self.name = name
        self.score = 0

class Game:
    def __init__(self, player1, player2):
        self.player1 = player1
        self.player2 = player2
        self.turn = player1

    def play_game(self):
        while True:
            if self.turn == self.player1:
                move = self.player1.make_move()
                self.check_move(move)
                self.turn = self.player2
            else:
                move = self.player2.make_move()
                self.check_move(move)
                self.turn = self.player1

    def check_move(self, move):
        if move == "rock":
            if self.turn.score > 0:
                self.turn.score += 1
            else:
                self.turn.score -= 1
        elif move == "paper":
            if self.turn.score < 0:
                self.turn.score += 1
            else:
                self.turn.score -= 1
        elif move == "scissors":
            if self.turn.score < 0:
                self.turn.score += 1
            else:
                self.turn.score -= 1

player1 = Player("Player 1")
player2 = Player("Player 2")

game = Game(player1, player2)
game.play_game()
```

在这个代码实例中，我们设计了一个简单的石头剪子布游戏。游戏中有两个玩家，每个玩家可以选择石头、剪子或布作为自己的行动。游戏的规则是，如果两个玩家选择的行动相同，那么玩家分数不变；如果两个玩家选择的行动不同，那么赢家的玩家分数加一，输家的玩家分数减一。
