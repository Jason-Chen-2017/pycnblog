```python
import math

def calculate_associative_force(mass1, mass2, distance):
    force_constant = 1.0
    return force_constant * (mass1 * mass2) / (distance ** 2)

mass1 = 10.0
mass2 = 20.0
distance = 5.0

force = calculate_associative_force(mass1, mass2, distance)
print("The associative force is:", force)
```

### 4.2 计算强关联网络

以下是一个计算强关联网络的Python代码实例：

```python
import networkx as nx

def create_associative_network(nodes, edges, weights):
    G = nx.Graph()
    G.add_nodes_from(nodes)
    G.add_edges_from(edges)
    G.edges[edges[0], edges[1]]['weight'] = weights[0]
    return G

nodes = ['A', 'B', 'C', 'D']
edges = [('A', 'B'), ('B', 'C'), ('C', 'D')]
weights = [1.0, 2.0, 3.0]

G = create_associative_network(nodes, edges, weights)
nx.draw(G, with_labels=True)
```

在这个代码实例中，我们首先导入了 `networkx` 库，用于创建和操作图。然后，我们定义了一个名为 `create_associative_network` 的函数，用于创建强关联网络。接着，我们输入了节点、边和权重，并调用该函数来创建强关联网络。最后，我们使用 `networkx` 库绘制了强关联网络。
