# 4. 具体代码实例和详细解释说明

在深入探讨流处理解决方案之前，我们需要了解一些具体的代码实例。

## 4.1 简单的流处理示例

```java
import java.util.stream.IntStream;

public class StreamExample {
    public static void main(String[] args) {
        // 创建一个流
        IntStream stream = IntStream.of(1, 2, 3, 4, 5);

        // 对流进行筛选
        IntStream filteredStream = stream.filter(x -> x % 2 == 0);

        // 对流进行映射
        IntStream mappedStream = filteredStream.map(x -> x * 2);

        // 对流进行排序
        IntStream sortedStream = mappedStream.sorted();

        // 对流进行终止操作
        int sum = sortedStream.sum();
        System.out.println(sum);
    }
}
```

在这个示例中，我们创建了一个整数流，然后对流进行筛选、映射、排序等操作，最后对流进行终止操作。

## 4.2 复杂的流处理示例

```java
import java.util.stream.Collectors;
import java.util.List;
import java.util.ArrayList;

public class StreamComplexExample {
    public static void main(String[] args) {
        // 创建一个列表
        List<Integer> list = new ArrayList<>();
        list.add(1);
        list.add(2);
        list.add(3);
        list.add(4);
        list.add(5);

        // 对列表进行流处理
        List<Integer> result = list.stream()
            .filter(x -> x % 2 == 0)
            .map(x -> x * 2)
            .sorted()
            .collect(Collectors.toList());

        // 打印结果
        System.out.println(result);
    }
}
```

在这个示例中，我们创建了一个整数列表，然后对列表进行流处理，包括筛选、映射、排序等操作，最后将结果收集到一个新的列表中。
