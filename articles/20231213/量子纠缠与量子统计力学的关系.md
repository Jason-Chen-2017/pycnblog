# 2. 核心概念与联系

## 2.1 量子纠缠

量子纠缠是指两个或多个量子系统之间的某种相互作用，使得它们的量子状态相互依赖。量子纠缠可以通过量子辐射、量子门操作、量子门控制等方式产生和操作。量子纠缠具有以下几个重要特征：

1. 非局部性：量子纠缠不仅仅是两个量子系统之间的局部相互作用，而是它们之间存在一种全局的相互依赖关系。
2. 超越位置：量子纠缠不受量子系统之间的物理距离限制，即使两个量子系统相隔很远，也可以实现纠缠。
3. 不可克隆性：量子纠缠是不可克隆的，即无法通过任何方式将纠缠态复制到另一个量子系统上。

## 2.2 量子统计力学

量子统计力学是一门研究量子系统统计规律和量子随机性的科学。它的主要内容包括：

1. 概率论：研究量子系统在不同观测条件下的概率分布。
2. 期望值：研究量子系统的期望值，即系统各量的平均值。
3. 不确定性：研究量子系统的不确定性，即系统各量之间的相互依赖关系。

量子统计力学与量子力学密切相关，量子统计力学的主要工具是概率论和数学统计方法，用于研究量子系统在不同观测条件下的统计规律。
