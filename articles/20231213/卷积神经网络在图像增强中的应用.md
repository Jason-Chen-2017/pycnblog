                 

# 1.背景介绍

图像增强是一种图像预处理技术，主要用于提高图像识别系统的识别率和准确性。图像增强的主要目的是通过对图像进行一定的变换，使得原图像中的信息得到提取，从而使图像识别系统更容易识别图像中的特征。图像增强方法有很多种，包括直方图均衡化、对比度扩展、图像裁剪、图像旋转、图像翻转、图像变形等。

卷积神经网络（Convolutional Neural Networks，CNN）是一种深度学习模型，主要应用于图像分类、目标检测、图像生成等任务。卷积神经网络的核心在于卷积层，卷积层可以自动学习图像中的特征，从而提高模型的识别能力。
