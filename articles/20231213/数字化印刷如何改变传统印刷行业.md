# 4.具体代码实例和详细解释说明

在这里，我们将提供一个简单的Python代码实例，用于实现文件格式转换的功能。

```python
import os
from PyPDF2 import PdfFileReader, PdfFileWriter

def convert_pdf(input_file, output_file):
    input_pdf = PdfFileReader(input_file)
    output_pdf = PdfFileWriter()

    for page_num in range(input_pdf.numPages):
        page = input_pdf.getPage(page_num)
        output_pdf.addPage(page)

    output_pdf.write(output_file)

if __name__ == '__main__':
    input_file = 'input.pdf'
    output_file = 'output.pdf'
    convert_pdf(input_file, output_file)
```

上述代码实例使用Python的PyPDF2库来实现PDF文件格式的转换。具体来说，代码首先导入了PyPDF2库，然后定义了一个名为`convert_pdf`的函数，该函数接受输入文件和输出文件两个参数。在函数内部，代码使用PdfFileReader类来读取输入文件，然后使用PdfFileWriter类来创建输出文件。接着，代码遍历输入文件中的每一页，并将其添加到输出文件中。最后，代码使用输出文件的write方法来写入文件。
