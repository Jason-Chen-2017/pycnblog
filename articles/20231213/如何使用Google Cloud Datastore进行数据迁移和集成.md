## 4.具体代码实例和详细解释说明

在Google Cloud Datastore中，数据迁移和集成的具体代码实例如下：

```python
from google.cloud import datastore
from google.cloud.datastore import query

# 创建数据库客户端
client = datastore.Client()

# 创建实体
key = client.key('Entity')
entity = datastore.Entity(key=key)
entity.update({
    'name': 'John Doe',
    'age': 30,
    'city': 'New York'
})
client.put(entity)

# 查询实体
query = client.query(kind='Entity')
results = list(query.fetch())
for result in results:
    print(result.keys())

# 排序实体
query = client.query(kind='Entity')
query.order = query.OrderBy('name')
results = list(query.fetch())
for result in results:
    print(result.keys())

# 限制实体
query = client.query(kind='Entity')
query.limit = 10
results = list(query.fetch())
for result in results:
    print(result.keys())
```

详细解释说明：

- 首先，需要创建数据库客户端。
- 然后，需要创建实体。
- 接下来，需要查询实体。
- 然后，需要排序实体。
- 最后，需要限制实体。
