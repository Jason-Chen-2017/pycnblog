# 2.核心概念与联系

在实时风控预警平台中，有几个核心概念需要了解：

1.数据收集：数据收集模块负责从不同来源收集数据，如数据库、API、文件等。这些数据可以是结构化的（如数据库表）或非结构化的（如文本、图像、音频等）。数据收集模块需要能够处理各种数据格式和协议。

2.数据处理：数据处理模块负责对收集到的数据进行清洗、转换和聚合。数据清洗包括删除重复数据、填充缺失值、去除噪声等。数据转换包括将不同格式的数据转换为统一的格式。数据聚合包括将多个数据源的数据聚合为一个数据集。

3.风险评估：风险评估模块使用各种算法和模型来评估风险。这些算法可以是统计算法、机器学习算法或深度学习算法。风险评估模块需要能够处理大量数据，并能够在实时环境下进行评估。

4.预警发送：预警发送模块负责将预警信息发送给相关人员，以便他们能够及时采取行动。这可以包括发送电子邮件、短信、推送通知等。预警发送模块需要能够处理不同类型的通知，并能够在实时环境下发送通知。
