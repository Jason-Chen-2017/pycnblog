# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的文本摘要生成示例来详细解释代码实现。

```python
import nltk
from nltk.tokenize import sent_tokenize
from nltk.corpus import stopwords
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

def extract_keywords(text):
    # 提取关键词
    keywords = []
    words = nltk.word_tokenize(text)
    stop_words = set(stopwords.words('english'))
    for word in words:
        if word not in stop_words:
            keywords.append(word)
    return keywords

def extract_sentences(text):
    # 提取句子
    sentences = sent_tokenize(text)
    return sentences

def generate_summary(text, keywords, sentences):
    # 生成摘要
    tfidf_vectorizer = TfidfVectorizer()
    tfidf_matrix = tfidf_vectorizer.fit_transform([text])
    cosine_similarities = cosine_similarity(tfidf_matrix, tfidf_matrix).flatten()
    sentence_scores = [cosine_similarity(tfidf_matrix[0], tfidf_vectorizer.transform([sentence]))[0][0] for sentence in sentences]
    summary_sentences = [sentence for sentence, score in zip(sentences, sentence_scores) if score >= 0.5]
    summary = ' '.join(summary_sentences)
    return summary

text = "The quick brown fox jumps over the lazy dog. The dog chases the cat."
keywords = extract_keywords(text)
sentences = extract_sentences(text)
summary = generate_summary(text, keywords, sentences)
print(summary)
```

上述代码实现了文本摘要生成的过程，包括关键词提取、句子选择和摘要生成等步骤。首先，我们使用NLTK库对文本进行分词，并去除停用词。然后，我们使用TF-IDF向量化器对文本进行向量化，并计算句子之间的余弦相似度。最后，我们选取余弦相似度大于0.5的句子，并将它们拼接成摘要。
