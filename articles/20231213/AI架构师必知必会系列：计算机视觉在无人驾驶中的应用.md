# 2.核心概念与联系

在无人驾驶技术中，计算机视觉主要包括以下几个方面：

1. **图像处理**：图像处理是计算机视觉的基础，主要包括图像的预处理、增强、压缩等。图像预处理是为了消除图像中的噪声和杂质，提高图像的质量。图像增强是为了提高图像中的对比度和明暗差异，以便更好地进行图像分析。图像压缩是为了减少图像文件的大小，方便存储和传输。

2. **图像分割**：图像分割是将图像划分为多个部分，以便进行特征提取和目标识别。图像分割可以通过边缘检测、分割算法等方法实现。

3. **特征提取**：特征提取是将图像中的有用信息提取出来，以便进行目标识别和定位。特征提取可以通过边缘检测、特征点检测、特征描述子等方法实现。

4. **目标识别**：目标识别是将提取出的特征与已知的目标进行比较，以便识别出目标的类别和属性。目标识别可以通过模板匹配、分类器等方法实现。

5. **目标定位**：目标定位是将识别出的目标在图像中的位置进行定位，以便进行跟踪和控制。目标定位可以通过边缘检测、特征点检测等方法实现。

6. **跟踪**：跟踪是将目标在连续的图像序列中进行跟踪，以便进行目标追踪和控制。跟踪可以通过卡尔曼滤波、深度学习等方法实现。

7. **控制**：控制是将目标的位置和速度信息传递给无人驾驶系统，以便进行路径规划和控制。控制可以通过PID控制、机器学习等方法实现。
