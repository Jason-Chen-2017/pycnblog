## 4.1 对称加密算法实例

```python
from Crypto.Cipher import AES

# 加密数据
def encrypt(data, key):
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(data)
    return cipher.nonce, ciphertext, tag

# 解密数据
def decrypt(nonce, ciphertext, tag, key):
    cipher = AES.new(key, AES.MODE_EAX, nonce=nonce)
    data = cipher.decrypt_and_verify(ciphertext, tag)
    return data
```

在这个例子中，我们使用了Python的Crypto库来实现AES对称加密算法。我们提供了一个加密函数`encrypt`和一个解密函数`decrypt`。

## 4.2 数据掩码算法实例

```python
import random

# 随机掩码
def random_mask(data):
    mask = ''.join(random.choice('0123456789abcdef') for _ in range(len(data)))
    return mask

# 固定掩码
def fixed_mask(data, mask):
    return mask * len(data)
```

在这个例子中，我们使用了Python的random库来实现随机掩码和固定掩码算法。我们提供了一个随机掩码函数`random_mask`和一个固定掩码函数`fixed_mask`。
