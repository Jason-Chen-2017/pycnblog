## 4.1 代码实例

我们来看一个简单的排序算法——选择排序。选择排序的核心思想是：在一个序列中，选择一个最小的元素，将其放在序列的起始位置。然后，在剩下的序列中，选择一个最小的元素，将其放在起始位置，直到整个序列排序完成。

```python
def select_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

## 4.2 详细解释说明

在这个代码实例中，我们可以看到选择排序的核心思想是：在一个序列中，选择一个最小的元素，将其放在序列的起始位置。然后，在剩下的序列中，选择一个最小的元素，将其放在起始位置，直到整个序列排序完成。
