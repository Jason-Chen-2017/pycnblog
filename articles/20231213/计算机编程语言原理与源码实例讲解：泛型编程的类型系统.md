## 4.1 泛型函数

我们可以定义一个泛型函数，它接受两个参数，并返回它们的和。这个函数可以处理不同类型的参数，例如整数、浮点数或字符串。

```python
def add(x, y):
    return x + y
```

在这个例子中，我们定义了一个名为`add`的泛型函数，它接受两个参数`x`和`y`。这个函数的返回值是`x + y`。我们可以调用这个函数，并传递任何类型的参数，例如整数、浮点数或字符串。

## 4.2 泛型类

我们可以定义一个泛型类，它可以存储任何类型的数据。这个类可以处理不同类型的数据，例如整数、浮点数或字符串。

```python
class GenericClass:
    def __init__(self, data):
        self.data = data
```

在这个例子中，我们定义了一个名为`GenericClass`的泛型类。这个类有一个构造函数，它接受一个参数`data`。这个类的实例变量`data`可以存储任何类型的数据。我们可以创建这个类的实例，并传递任何类型的参数，例如整数、浮点数或字符串。
