```python
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 创建量子电路
qc = QuantumCircuit(2)

# 初始化量子位
qc.h(0)

# 实现量子门
qc.cx(0, 1)

# 测量量子位
qc.measure([0, 1], [0, 1])

# 执行量子仿真
simulator = Aer.get_backend('qasm_simulator')
job = simulator.run(assemble(qc))

# 获取测量结果
result = job.result()
counts = result.get_counts(qc)

# 绘制测量结果
plot_histogram(counts)
```

在上述代码中，我们首先导入了`qiskit`库，并创建了一个量子电路`qc`。然后我们对第0个量子位进行初始化操作`qc.h(0)`，并实现了一个量子门`qc.cx(0, 1)`。接着我们对量子位进行测量操作`qc.measure([0, 1], [0, 1])`。最后，我们使用`qasm_simulator`后端执行量子仿真，并绘制测量结果。
