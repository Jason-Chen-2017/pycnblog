## 4.1 代码实例

我们来看一个简单的蒙特卡罗方法的代码实例：

```python
import random

def monte_carlo(n):
    sum = 0
    for i in range(n):
        x = random.random()
        y = random.random()
        if x**2 + y**2 <= 1:
            sum += 1
    return sum/n

print(monte_carlo(100000))
```
