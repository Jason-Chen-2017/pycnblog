## 2.1 错误类型

Go语言中的错误类型是一个接口类型，它只有一个方法`Error() string`。这意味着任何类型都可以作为错误类型，只要它实现了`Error() string`方法。

```go
type error interface {
    Error() string
}
```

## 2.2 错误处理的基本操作

Go语言中的错误处理的基本操作有以下几种：

1. 检查错误：在调用函数后，可以使用`if err != nil`来检查错误是否发生。
2. 返回错误：当函数调用失败时，可以返回一个错误值。
3. 处理错误：可以使用`if err != nil`来处理错误，并根据需要进行相应的操作。
