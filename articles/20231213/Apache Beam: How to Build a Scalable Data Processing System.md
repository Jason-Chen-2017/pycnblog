大数据处理系统的发展历程可以分为三个阶段：

1. 早期阶段，大数据处理系统主要是基于Hadoop MapReduce的。在这个阶段，数据处理的速度非常慢，并且需要大量的硬件资源。

2. 第二个阶段，是基于Spark的。Spark比Hadoop MapReduce更快，更高效，更易于扩展。但是，Spark仍然存在一些问题，例如：它不能很好地处理流式数据，并且它的代码可能不是很易于理解和维护。

3. 第三个阶段，是基于Apache Beam的。Apache Beam是一个开源的大数据处理框架，它可以让开发者更轻松地构建大数据处理系统。Apache Beam可以处理流式数据，并且它的代码更加易于理解和维护。
