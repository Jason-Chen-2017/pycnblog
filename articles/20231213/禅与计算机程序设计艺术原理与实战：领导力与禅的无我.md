## 3.1 排序算法

排序算法是计算机程序设计中的一个重要类型。排序算法用于对数据进行排序。禅的思想可以帮助我们更好地理解排序算法的原理和功能。

### 3.1.1 选择排序

选择排序是一种简单的排序算法。它的原理是从数组中选择最小的元素，并将其放在正确的位置。选择排序的时间复杂度是O(n^2)。

选择排序的具体操作步骤如下：

1. 从数组中选择最小的元素，并将其放在正确的位置。
2. 从剩余的元素中选择最小的元素，并将其放在正确的位置。
3. 重复步骤2，直到所有元素都被排序。

选择排序的数学模型公式如下：

$$
T(n) = \frac{n^2}{2} + \frac{n}{2}
$$

### 3.1.2 冒泡排序

冒泡排序是一种简单的排序算法。它的原理是从数组中选择最小的元素，并将其放在正确的位置。冒泡排序的时间复杂度是O(n^2)。

冒泡排序的具体操作步骤如下：

1. 从数组中选择最小的元素，并将其放在正确的位置。
2. 从剩余的元素中选择最小的元素，并将其放在正确的位置。
3. 重复步骤2，直到所有元素都被排序。

冒泡排序的数学模型公式如下：

$$
T(n) = \frac{n^2}{2} - \frac{n}{2}
$$

### 3.1.3 快速排序

快速排序是一种高效的排序算法。它的原理是从数组中选择一个基准元素，然后将其他元素分为两部分，一部分小于基准元素，一部分大于基准元素。快速排序的时间复杂度是O(nlogn)。

快速排序的具体操作步骤如下：

1. 从数组中选择一个基准元素。
2. 将其他元素分为两部分，一部分小于基准元素，一部分大于基准元素。
3. 对两部分元素进行快速排序。
4. 将基准元素放在正确的位置。

快速排序的数学模型公式如下：

$$
T(n) = 2T(n/2) + n
$$

## 4.1 排序算法实例

### 4.1.1 选择排序实例

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

arr = [5, 2, 9, 1, 3]
print(selection_sort(arr))
```

### 4.1.2 冒泡排序实例

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [5, 2, 9, 1, 3]
print(bubble_sort(arr))
```

### 4.1.3 快速排序实例

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [5, 2, 9, 1, 3]
print(quick_sort(arr))
```

## 4.2 搜索算法实例

### 4.2.1 二分搜索实例

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

arr = [1, 2, 3, 4, 5]
target = 3
print(binary_search(arr, target))
```

### 4.2.2 深度优先搜索实例

```python
def dfs(graph, start):
    visited = [False] * len(graph)
    stack = [start]
    while stack:
        vertex = stack.pop()
        if not visited[vertex]:
            visited[vertex] = True
            for neighbor in graph[vertex]:
                if not visited[neighbor]:
                    stack.append(neighbor)
    return visited

graph = {
    0: [1, 2],
    1: [2],
    2: []
}
start = 0
print(dfs(graph, start))
```
