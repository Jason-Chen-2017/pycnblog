# 2.核心概念与联系

多目标决策(MCDM)是一种计算机辅助决策(CDA)方法，它主要用于解决具有多个目标和约束条件的复杂决策问题。在这种决策问题中，决策者需要同时考虑多个目标，并在满足约束条件的同时，找到一个最优解。这种决策问题的特点是多目标、多因素、多方面和多层次。

MCDM方法的核心概念包括：

1.决策对象：决策对象是决策问题的主体，可以是个人、组织或系统等。

2.决策目标：决策目标是决策对象希望实现的目标，可以是经济、社会、环境等多个方面。

3.决策因素：决策因素是影响决策目标实现的因素，可以是资源、技术、政策等。

4.约束条件：约束条件是决策过程中需要满足的条件，可以是法律、规定、技术等。

MCDM方法与其他CDA方法的联系：

1.MCDM方法与单目标决策(STDM)方法的区别在于，MCDM方法需要同时考虑多个目标，而STDM方法只需要考虑一个目标。

2.MCDM方法与多约束决策(MCDM)方法的区别在于，MCDM方法需要同时考虑多个目标和约束条件，而MCDM方法只需要考虑约束条件。

3.MCDM方法与多目标优化(MTO)方法的区别在于，MCDM方法主要用于解决具有多个目标和约束条件的决策问题，而MTO方法主要用于解决具有多个目标和约束条件的数学模型。

from scipy.optimize import linprog

    {'type': 'ineq', 'fun': lambda x: x[1] - 5},
    {'type': 'ineq', 'fun': lambda x: x[2] - 5},
]

# 权重
weights = np.array([1, 1, 1])

# 目标函数

    return -np.sum(weights * x)

# 解决决策问题

    obj=objective,
    bounds=[(0, None)] * 3,
    constraints=constraints,
    method='highs'
)

# 输出结果
print(result)
