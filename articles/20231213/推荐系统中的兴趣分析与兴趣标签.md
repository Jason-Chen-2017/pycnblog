### 4.1 兴趣分析的代码实例

```python
import pandas as pd
from sklearn.cluster import KMeans

# 数据收集
data = pd.read_csv('user_behavior_data.csv')

# 数据预处理
data = data.fillna(0)

# 特征提取
features = data[['item_id', 'category_id', 'time']]

# 模型构建
kmeans = KMeans(n_clusters=5, random_state=0).fit(features)

# 模型评估
labels = kmeans.labels_
data['cluster'] = labels

# 兴趣分析结果
print(data)
```
