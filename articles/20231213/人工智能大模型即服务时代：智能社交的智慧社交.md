## 4.1 用户行为数据的预处理

```python
import pandas as pd
import numpy as np

# 读取用户行为数据
data = pd.read_csv('user_behavior_data.csv')

# 数据清洗
data = data.drop_duplicates()
data = data.dropna()

# 数据转换
data['time'] = pd.to_datetime(data['time'])
data['time'] = (data['time'] - data['time'].min()) / np.timedelta64(1, 'D')

# 数据归一化
data = (data - data.mean()) / data.std()
```
