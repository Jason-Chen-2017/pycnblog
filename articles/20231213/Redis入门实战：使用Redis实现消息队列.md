Redis是一个开源的高性能的key-value存储系统，它支持数据的持久化，可基于内存（in-memory）并提供多种语言的API。Redis是一个使用ANSI C语言编写、遵循BSD协议的开源软件。Redis的根目录下的默认配置文件名称是`redis.conf`。

Redis支持通过TCP/IP协议与其他网络客户端进行通信，也可以与其他Redis服务器进行通信。Redis支持运行在内存中(in-memory)的数据库，所有的数据都存储在内存中。Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。

Redis 支持数据的备份，即Master-Slave模式的数据备份。Redis 支持 Publish/Subscribe 模式，可以实现消息通信。Redis 支持Lua脚本（Redis Script），可以用来存储和执行Lua脚本代码。Redis 支持键空间通知（Key Event Notification），可以通知客户端键空间的变化。Redis 支持事务（Transaction），可以一次执行多个命令。Redis 支持有序集合（Sorted Set），可以进行有序的排序。Redis 支持Bitmaps（位图）和HyperLogLogs（超级逻辑日志）数据类型，可以进行比特位运算和统计独特值的数量。Redis 支持Geospatial（地理空间）数据类型，可以进行地理位置的存储和查询。Redis 支持测试内存泄漏（Memory Leak Test），可以检测内存泄漏。Redis 支持命令的监控（Command Monitoring），可以监控Redis命令的执行情况。Redis 支持客户端的鉴权（Client Side Authentication），可以对Redis客户端进行鉴权。Redis 支持TCP连接的keepalive（TCP Keepalive），可以保持TCP连接的活跃。Redis 支持虚拟内存（Virtual Memory），可以扩展Redis的内存空间。Redis 支持LRU驱除（LRU Eviction），可以根据最近最少使用的策略进行内存驱除。Redis 支持自定义Lua脚本（Custom Lua Script），可以自定义Lua脚本进行各种操作。Redis 支持密码保护（Password Protection），可以对Redis进行密码保护。Redis 支持TCP节流（TCP
