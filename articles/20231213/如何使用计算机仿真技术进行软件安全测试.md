假设我们有一个简单的网站，它使用HTTP协议进行通信。我们想要对这个网站进行安全性测试，以确保它不会受到XSS（跨站脚本）攻击。

首先，我们需要建立网站的模型。我们可以使用Python的`requests`库来模拟HTTP请求，并对网站进行访问。

```python
import requests

def send_request(url, data):
    response = requests.post(url, data=data)
    return response.text
```

接下来，我们需要建立XSS攻击的模型。我们可以使用Python的`beautifulsoup`库来解析HTML响应，并检查是否包含恶意代码。

```python
from bs4 import BeautifulSoup

def check_xss(html):
    soup = BeautifulSoup(html, 'html.parser')
    scripts = soup.find_all('script')
    if scripts:
        return True
    return False
```

最后，我们需要使用计算机仿真技术进行安全性测试。我们可以使用Python的`unittest`库来编写测试用例，并对网站进行XSS攻击测试。

```python
import unittest

class TestXSS(unittest.TestCase):
    def test_xss(self):
        url = 'http://example.com'
        data = {'input': '<script>alert("XSS")</script>'}
        response = send_request(url, data)
        self.assertTrue(check_xss(response))

if __name__ == '__main__':
    unittest.main()
```

通过运行这个测试用例，我们可以发现网站是否受到XSS攻击。如果网站受到攻击，我们可以采取相应的措施来修复安全漏洞。
