## 4. 项目实践：代码实例和详细解释说明

在本部分，我们将通过一个具体的项目实践来详细解释如何使用ChatGPT提问表单来构建问答系统。以下是一个简单的代码实例：

```python
import openai

openai.api_key = "your-api-key"

def ask_question(question):
    response = openai.Completion.create(
        engine="text-davinci-002",
        prompt=f"{question}",
        max_tokens=100,
        n=1,
        stop=None,
        temperature=0.5,
    )

    return response.choices[0].text.strip()

question = "What is the capital of France?"
answer = ask_question(question)
print(answer)
```

在这个代码实例中，我们首先导入openai库，并设置API密钥。然后定义了一个`ask_question`函数，该函数接收一个问题作为输入，并将问题传递给GPT-4模型。最后，我们通过调用`ask_question`函数来询问问题，并打印出回答。
