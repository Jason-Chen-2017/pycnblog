## 项目实践：代码实例和详细解释说明

在本文中，我们将提供一个简单的代码实例，展示如何使用 Python 和 scikit-learn 库来处理法律和规则问题。代码如下：

```python
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.feature_extraction.text import TfidfTransformer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import Pipeline

# 法律和规则文本
documents = [
    "法律规则一：未经授权不得进行传播活动",
    "法律规则二：必须遵守国家法律和法规",
    "事实信息一：公司进行传播活动",
    "事实信息二：公司遵守国家法律和法规"
]

# 分类标签
labels = ["rule", "fact"]

# 创建文本处理管道
text_clf = Pipeline([
    ("vect", CountVectorizer()),
    ("tfidf", TfidfTransformer()),
    ("clf", MultinomialNB())
])

# 训练模型
text_clf.fit(documents, labels)

# 预测
predicted = text_clf.predict(["公司进行传播活动"])
print(predicted)  # 输出: ['fact']
```

这个代码示例展示了如何使用 CountVectorizer 和 TfidfTransformer 进行文本处理，然后使用 MultinomialNB 进行分类。通过这个示例，我们可以看到如何使用 Python 和 scikit-learn 库来处理法律和规则问题。
