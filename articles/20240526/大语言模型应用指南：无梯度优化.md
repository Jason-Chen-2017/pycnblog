## 2. 核心概念与联系

无梯度优化（NGO）是一种新的优化方法，它不依赖于梯度信息。这种方法可以在大型模型训练中提高性能，减少计算成本，并降低内存需求。NGO方法与传统梯度下降方法的主要区别在于，它们不需要计算梯度。

## 3. 核算法原理具体操作步骤

NGO方法的核心思想是通过优化目标函数的值来优化模型。这种方法通常涉及到一个或多个“搜索策略”，这些策略用于探索目标函数的空间，并找到最优解。以下是NGO方法的主要操作步骤：

1. 初始化：首先，选择一个初始解作为起点。
2. 搜索策略：选择一个搜索策略，以便探索目标函数的空间。
3. 更新解：根据搜索策略更新解，直到满足停止条件。

import numpy as np

def f(x):
    return x**2 + 4*x + 4

def ng_optimize(f, x0, lr=0.01, max_iter=1000):
    x = x0
    for i in range(max_iter):
        # 搜索策略：随机选择一个邻居
        neighbor = x + np.random.uniform(-1, 1)
        if f(neighbor) < f(x):
            x = neighbor
    return x

x0 = np.array([0.0])
x_optimal = ng_optimize(f, x0)
print("最优解：", x_optimal)
