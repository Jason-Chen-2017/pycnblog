项目实践：基于规则的智能体

在这个项目中，我们将构建一个基于规则的智能体，用于控制一个简单的家用电器。以下是项目的代码实例：

```python
class SmartAppliance:
    def __init__(self, rules):
        self.rules = rules

    def perceive(self, state):
        # 传感器读取环境状态
        pass

    def decide(self, state):
        # 根据规则决策
        for rule in self.rules:
            if rule.match(state):
                return rule.action(state)
        return None

    def act(self, action):
        # 执行动作
        pass

class Rule:
    def __init__(self, condition, action):
        self.condition = condition
        self.action = action

    def match(self, state):
        # 判断规则是否满足
        return self.condition(state)

# 规则示例
turn_on_tv = Rule(lambda state: state["tv"] == "off", lambda state: state["tv"] = "on")
turn_off_tv = Rule(lambda state: state["tv"] == "on", lambda state: state["tv"] = "off")

rules = [turn_on_tv, turn_off_tv]

# 创建智能体
smart_appliance = SmartAppliance(rules)

# 执行控制
state = {"tv": "off"}
smart_appliance.perceive(state)
action = smart_appliance.decide(state)
smart_appliance.act(action)
```
