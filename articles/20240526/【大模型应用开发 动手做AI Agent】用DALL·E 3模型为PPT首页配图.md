## 5. 项目实践：代码实例和详细解释说明

以下是一个使用Python编写的DALL·E 3 API调用示例：

```python
import openai

openai.api_key = "your_api_key_here"

response = openai.Completion.create(
  engine="dall-e-3",
  prompt="A young programmer working at his desk",
  max_tokens=50,
  n=1,
  stop=None,
  temperature=0.5,
)

image_url = response.choices[0].text.strip()
print(image_url)
```

上述代码首先导入openai模块，然后设置API密钥。接下来，我们调用`openai.Completion.create()`方法，传入DALL·E 3引擎、图像描述（`prompt`参数）以及其他参数。最后，我们获取生成的图像的URL，并将其打印出来。
