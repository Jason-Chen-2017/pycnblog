2. 支持向量机：支持向量机的数学模型可以表示为：maxW(x) = b，其中W是超平面，b是偏置。支持向量机的损失函数可以表示为：L = (1/2N) * Σ(y - y')²，其中N是样本数，y是真实值，y'是预测值。

3. 决策树：决策树的数学模型可以表示为：f(x) = g(h1(x), h2(x), ..., hn(x)),其中f是目标函数，g是组合函数，h1, h2, ..., hn是子树。决策树的损失函数可以表示为：L = -Σ(y * log(p) + (1 - y) * log(1 - p)),其中y是真实值，p是预测概率。
