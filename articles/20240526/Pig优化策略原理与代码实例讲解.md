## 5. 项目实践：代码实例和详细解释说明

下面是一个 Pig 项目实践的代码实例：

```python
# 导入数据
data = LOAD 'data.txt' AS (a:chararray, b:chararray);

# 筛选数据
filtered_data = FILTER data BY a IS NOT NULL AND b IS NOT NULL;

# 分组并聚合数据
grouped_data = GROUP filtered_data BY a;

# 计算统计信息
result = FOREACH grouped_data GENERATE group, COUNT(b), AVG(b);

# 输出结果
STORE result INTO 'output.txt' USING PigStorage(',');
```

在这个代码示例中，我们首先导入数据，然后使用筛选操作筛选出不为空的数据。接着使用分组操作将数据按照某个属性进行分组，并使用聚合操作计算统计信息。最后，我们使用输出操作将结果存储到文件中。
