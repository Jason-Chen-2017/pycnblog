## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的资源调度代码实例：

```python
import os
import time
from multiprocessing import Pool

def task(x):
    print(f"Processing {x}")
    time.sleep(2)
    return x * x

if __name__ == "__main__":
    pool = Pool(4)
    results = pool.map(task, range(10))
    pool.close()
    pool.join()
```

在这个例子中，我们使用Python的multiprocessing库来实现资源调度。我们创建了一个Pool，包含4个进程，然后使用map函数来分配任务给各个进程。最后，我们关闭Pool并等待所有任务完成。
