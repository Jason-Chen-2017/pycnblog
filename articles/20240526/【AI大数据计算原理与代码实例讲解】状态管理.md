```python
class StateManager:
    def __init__(self):
        self.state = None

    def transition(self, x):
        if self.state is None:
            self.state = x
        else:
            self.state = (self.state + x) % 100

    def reset(self):
        self.state = None

    def get_state(self):
        return self.state

# 示例使用
sm = StateManager()
sm.transition(10)
sm.transition(20)
print(sm.get_state())  # 输出: 30
sm.reset()
print(sm.get_state())  # 输出: None
```

这个代码实例中，我们定义了一个`StateManager`类，它负责管理和处理数据。`StateManager`类有一个`state`属性，用于存储当前的状态。`transition`方法用于进行状态转移，而`reset`方法用于恢复到初始状态。`get_state`方法用于获取当前状态。
