## 2. 核心概念与联系

知识图谱是一种基于图形结构的数据存储方式，它可以表示实体、属性和关系之间的复杂联系。知识图谱的核心概念是节点（Node）和边（Edge）。节点表示实体，如人、地点或事物，边表示实体之间的关系。通过这种方式，知识图谱可以捕捉复杂的语义关系，帮助我们更好地理解数据。

AI代理工作流是一种用于自动执行特定任务的流程。AI代理工作流通常由一系列任务组成，这些任务可以自动执行或由人类用户触发。

知识图谱和AI代理工作流之间的联系在于，知识图谱可以用作AI代理工作流的数据源。通过将知识图谱集成到AI代理工作流中，我们可以利用知识图谱的强大功能来自动完成各种任务。

## 4. 项目实践：代码实例和详细解释说明

在本节中，我们将提供一个实际的项目实践案例，展示如何将知识图谱集成到AI代理工作流中。我们将使用Python和RDFlib库来实现知识图谱查询。

```python

from rdflib.namespace import RDF, FOAF

# 创建知识图谱
g = Graph()

# 添加实体和关系
g.add((URIRef('http://example.org/A'), RDF.type, FOAF.Person))
g.add((URIRef('http://example.org/B'), RDF.type, FOAF.Location))
g.add((URIRef('http://example.org/A'), FOAF.worksFor, URIRef('http://example.org/B')))

# 查询知识图谱
for s, p, o in g.triples((None, RDF.type, FOAF.Person)):
    print(f"{s} works for {o}")
```
