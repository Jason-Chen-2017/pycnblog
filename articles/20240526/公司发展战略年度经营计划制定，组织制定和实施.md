## 4.项目实践：代码实例和详细解释说明

为了更好地理解人工智能技术，我们需要通过实际项目来进行学习和实践。以下是一个简单的代码实例，用于实现一个简单的线性回归模型。

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据集
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([2, 3, 4, 5])

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 预测
y_pred = model.predict(X)

# 输出预测结果
print(y_pred)
```
