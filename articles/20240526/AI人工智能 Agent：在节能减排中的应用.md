## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的代码实例，展示了如何使用人工智能 Agent 来分析制冷系统的能源使用情况。

```python
import numpy as np
import pandas as pd

# 读取制冷系统数据
data = pd.read_csv('cooling_system_data.csv')

# 计算能源消耗
E = data['功率'] * data['时间']

# 分析能源消耗情况

E_std = np.std(E)

```
