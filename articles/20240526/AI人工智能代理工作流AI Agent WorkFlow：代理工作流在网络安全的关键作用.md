## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的AI代理工作流代码示例，用于检测网络流量中的异常行为：

```python
import numpy as np
from sklearn.metrics import mean_squared_error

def detect_anomaly(x, y, threshold=0.1):
    error = mean_squared_error(x, y)
    if error > threshold:
        return True
    return False

x = np.array([1, 2, 3, 4, 5])
y = np.array([1, 2, 2, 3, 4])

if detect_anomaly(x, y):
    print("Anomaly detected!")
else:
    print("No anomaly detected.")
```
