然后，我们可以使用以下代码来创建私人邮件助手：

```python
import openai

openai.api_key = "your-api-key"

def generate_reply(email_body):
    prompt = f"请为以下电子邮件生成回复：\n\n{email_body}\n\n回复："
    response = openai.Completion.create(
        engine="text-davinci-002",
        prompt=prompt,
        max_tokens=50,
        n=1,
        stop=None,
        temperature=0.7,
    )
    return response.choices[0].text.strip()

email_body = "我需要在明天早上9点召开会议。你能帮我安排吗？"
reply = generate_reply(email_body)
print(reply)
```
