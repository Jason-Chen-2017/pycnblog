## 4. 项目实践：代码实例和详细解释说明

在这个部分，我们将通过一个简单的示例来展示如何使用LangChain来构建一个基于大模型的智能系统。我们将构建一个简单的聊天机器人系统。

```python
import langchain as lc

# 加载模型
model = lc.load_model("gpt-3")

# 定义聊天逻辑
def chatlogic(prompt):
    return model.generate(prompt)

# 创建聊天系统
chatbot = lc.ChatSystem(chatlogic)

# 聊天
response = chatbot.chat("你好，欢迎使用聊天机器人")
print(response)
```
