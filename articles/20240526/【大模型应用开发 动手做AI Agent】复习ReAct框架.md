## 2. 核心概念与联系

ReAct框架的核心概念是反应策略。反应策略是一种基于感知和反应的方法，它允许AI Agent在不同情境下进行适应性地反应。反应策略是一种行为优化方法，它可以帮助AI Agent更好地适应不同的环境和任务。

反应策略的基本思想是：在一个给定的环境下，AI Agent通过观察其周围的环境来感知其周围的状态。然后，根据这些观察到的信息，AI Agent会做出相应的反应，以达到最佳的行为效果。

反应策略与传统的计划和规则方法不同。传统的计划方法通常需要预先定义所有可能的行为序列，而反应策略则只需要定义一个简单的规则来决定在给定的环境下如何做出反应。
