神经网络模型是人工智能算法的核心。下面是一个简单的神经网络模型的结构图：

```
输入 -> 层1 -> 层2 -> ... -> 层n -> 输出
```

每一层都是一个矩阵，输入层的矩阵大小是输入数据的维度，而输出层的矩阵大小是输出数据的维度。每一层之间的连接由权重参数表示。
