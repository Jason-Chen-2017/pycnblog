## 4. 项目实践：代码实例和详细解释说明

在本节中，我们将通过一个项目实践来展示如何使用监督学习进行预测。我们将使用Python语言和Scikit-Learn库来实现监督学习。

代码实例如下：

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

# 加载数据集
iris = load_iris()
X, y = iris.data, iris.target

# 切分数据集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 创建模型
model = LogisticRegression()

# 训练模型
model.fit(X_train, y_train)

# 预测
y_pred = model.predict(X_test)

# 计算准确率
accuracy = accuracy_score(y_test, y_pred)
print("准确率：", accuracy)
```

代码解释：

1. 首先，我们从scikit-learn库中导入需要使用的模块。
2. 然后，我们加载了一个示例数据集，iris数据集。
3. 接下来，我们将数据集分为训练集和测试集，测试集的大小为20%。
4. 之后，我们创建了一个LogisticRegression模型，并将其训练 fit 在训练集上。
5. 最后，我们使用训练好的模型对测试集进行预测，并计算预测的准确率。
