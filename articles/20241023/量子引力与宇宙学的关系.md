2.1 笛卡儿坐标系与洛伦兹变换

笛卡儿坐标系是一种用于描述空间点位置的坐标系，由三个相互垂直的直线（通常称为坐标轴）构成。在笛卡儿坐标系中，空间点的位置可以通过三个坐标值（x, y, z）来确定。笛卡儿坐标系在经典物理学中广泛应用，但在相对论中，由于光速不变原理的影响，我们需要使用洛伦兹变换来描述不同惯性系之间的时空关系。

洛伦兹变换是描述不同惯性系之间的时空变换关系的一个数学公式。它是一个四维变换，涉及时间坐标和空间坐标。洛伦兹变换的基本公式如下：

$$
x' = \gamma (x - vt)
$$
$$
y' = y
$$
$$
z' = z
$$
$$
t' = \gamma (t - \frac{vx}{c^2})
$$

其中，$x'$和$t'$是变换后的坐标和时间，$x$和$t$是原始坐标和时间，$v$是相对速度，$c$是光速，$\gamma$是洛伦兹因子，定义为：

$$
\gamma = \frac{1}{\sqrt{1 - \frac{v^2}{c^2}}}
$$

洛伦兹变换确保了在不同惯性系中光速不变，这是相对论的核心原理之一。通过洛伦兹变换，我们可以将一个惯性系中的物理事件描述转换到另一个惯性系中。

v_x \\

v_z \\
\frac{v}{c}
\end{array} \right)
$$

\end{array} \right)
$$

其中，$m$是物体的质量，$\gamma$是洛伦兹因子。

广义相对论是爱因斯坦在狭义相对论基础上提出的一个更一般的理论，它描述了在引力场中的物体运动和时空结构。广义相对论中的数学公式主要包括弯曲时空、引力场的数学描述和弗里德曼-勒梅特-罗伯逊-沃尔克度规。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义宇宙背景辐射的温度分布函数
def temperature_distribution(r, r_0, T_0):
    """
    计算不同距离处的宇宙背景辐射温度。

    参数：
    r：观测者与辐射源之间的距离（单位：Mpc）。
    r_0：辐射源的大小尺度（单位：Mpc）。
    T_0：辐射源的初始温度（单位：K）。

    返回值：
    T：计算得到的温度（单位：K）。
    """
    return T_0 * (1 / r) * (np.exp(-r / r_0) / (1 - np.exp(-r / r_0)))

# 模拟不同距离处的温度分布
r_values = np.linspace(1, 1000, 1000)  # 生成距离数组，范围从1到1000 Mpc，共1000个点
T_values = temperature_distribution(r_values, r_0=100, T_0=2.725)  # 计算温度值

# 绘制温度分布图
plt.plot(r_values, T_values)
plt.xlabel('Distance (Mpc)')
plt.ylabel('Temperature (K)')
plt.title('Cosmic Background Radiation Temperature Distribution')
plt.show()
```
