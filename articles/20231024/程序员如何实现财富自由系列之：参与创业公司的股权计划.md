# 1.背景介绍


## 1.1什么是股权计划？
股权计划（Equity Plans）是由企业或机构在一定期限内向个人或法人投资者发放股份的一种金融工具。该计划主要包括两种：

1. 价值投资计划：指的是企业或机构将其持有的企业价值转化为股东权益，并通过支付给相应的股东的股息、分红等方式筹集资本。企业所获得的价值越高，得到的股权就越多，投资者则可在公司的利润中获得一定的收益。例如，杭州某知名互联网公司创始人马云提出的价值投资计划，就是向所有员工发放股票，只要员工努力工作，公司价值的增加就会带动公司股权的增加。

2. 增值税优惠计划：指的是企业在不改变产品市场定位、客户满意度的前提下，通过优惠政策吸引更多的外资及合作伙伴进入，并将这些资金用于扩大业务或投资开发。增值税优惠计划既可以进行长期资本筹措，也可转化为短期获利。例如，阿里巴巴在淘宝网推出一项免税优惠活动，只要买家以支付宝购物、使用优惠码获得19.9元/年的消费券即可入驻阿里巴巴体系，从而帮助商家扩大流量及增加营销能力，同时也可以获益于当地增值税减免。

两者各有优劣点，且存在较大的政策风险和监管难度。但无论是价值投资计划还是增值税优惠计划，都是以股权激励的方式使企业运用资本快速积累、获取暴利。因此，如果能够按照合适的计划实施，就可以极大地提升个人的收益、企业的盈利，并获得财富自由。
## 1.2为什么要参加股权计划？
参与股权计划具有许多优势。首先，股权计划能有效激发员工的创造力和才智。企业中的人才是创造力的源泉，股权计划能促进员工之间的交流、沟通、协作和自我学习。其次，通过股权计划，员工将有机会成为自己喜欢的领域的一份子。创新企业往往都会举办一些内部竞赛和活动，让员工参与其中并锻炼技能。第三，股权计划能够为员工提供经济上的保障。作为投资者，股权计划可以帮助员工实现长期稳定收入。第四，股权计划有助于促进企业的长久发展。随着社会的发展和经济的增长，企业需要有更好的方式来分配利润。股权计划可以帮助企业更好地为员工发展福利和晋升职级，也能为公司实现长远的利益目标。
## 1.3企业如何选择参加股权计划？
选择参加股权计划的企业一般遵循以下几个基本原则：

1. 获得足够的授权：所谓“获得足够的授权”，是指企业必须获得投资人的同意才能加入股权计划。通常情况下，需要寻找具有充裕经验、熟悉市场的人才作为投资人，并向他们提供必要的信息。

2. 明确目的：企业需清楚自己的投资计划的目的，制定规划并对其进行公示。目的必须非常明确，且具有实际意义。否则，可能会被投资人的猜疑和抵触。

3. 雄厚实力：企业必须具备良好的组织资源、管理能力、执行能力和职业道德素养。投资者很容易看到，若企业缺乏雄厚实力，即便是合格的投资人也不会支持他们加入。

4. 提供条件：企业需提供一切必要的条件，包括资金、时间和能力，来促进其投资计划的顺利实施。如企业要求的回报率太低、费用过高等。

综上所述，参与股权计划是一个成长中的过程，企业需要长期坚持才能真正做到自己的付出得益。不管是价值投资计划还是增值税优惠计划，都有着各自的优缺点，并不是所有企业都适合自己选择。但在实践中，企业应当认真对待各种类型股权计划，选择最适合自己的方式来实现财富自由。
