# 1.背景介绍


“人生不如意事十有八九”，在现代社会，随着时间的推移，人们越来越容易陷入各种“烦恼”。无论是工作、个人、健康还是关系，我们都有时常面临各种困境或麻烦，但有时候却会看到一些新鲜事物的出现，或许可以给我们带来解脱，给我们新的机遇……然而，这样的新鲜事物往往伴随着更多的负面影响，比如可能会引起自己的抑郁、疲劳、沮丧等负面的情绪甚至死亡。所以，如何通过智慧的方法来控制这些负面影响，并且寻找一种方法让自己更加成熟，更有能力地应对各种艰难险阻，成为一个更好的人呢？这正是科学家们一直追求的目标——找到一条科技树上的路径，从根本上解决这一“人生不如意事”的问题。当时，科学家们已经在研究如何把人的思想、情感、记忆和精神活动的综合能力（Cognitive Capacity）转化成可操控的高级智能行为、脑力模拟工具、聘请外星人虚拟人物、使机器更像人类、控制科技武器等等，不胜枚举。由于自然界中复杂的现象实在太多了，所以科学家们也很早就认识到，人类的生活并不是单纯靠理性思维来完成的，而要结合动手、创造力、好奇心、和独特的个性才能更好地生活下去。所以，为了能够更有效地管理我们的负面影响，掌握知识、洞察真相、领悟道理、体验幸福、养成习惯、化解矛盾、创造力、执行力，以及持续学习的能力等，我们需要进行多种方式的训练。
而基于结构化思考模式的科学树，就是用于培养这种知识、能力、及技能的一套系统框架。这是一种建立在认知科学、工程科学、心理学、社会学等多个学科基础上的一套科学方法论，并以图示的方式呈现出来。它有助于引导科学家和学生发现、理解和掌握有关问题的关键所在，并帮助他们制定正确的行动方案和策略。下面我们就用结构化思考的形式，来了解一下什么是结构化思考法、金字塔结构以及如何利用这种方法来培养创造力。
# 2.核心概念与联系
## 2.1 什么是结构化思考法？
结构化思考法，是指将自身的思考过程或者问题按照一定结构化的方式组织起来，然后再根据结构化的思考方式进行处理和分析，从而达到提升自我能力、增强自我觉醒、完善自我的目的。结构化思考法包括“客观层次结构”、“主观层次结构”、“交叉点”三个层次。
### 2.1.1 “客观层次结构”
客观层次结构的作用是为了让整体的视角能完整的呈现出问题中的主要因素、差异、关联和相互依存关系，从而达到问题的全面、准确的描述。它也是结构化思考法的基石之一。
#### 2.1.1.1 何为主观层次结构？
主观层次结构是在客观层次结构的基础上构建起来的。它可以简单理解为主观意愿的集合，其中的元素应该具有具体的含义和形象，而不是指标、评价、量化标准。它在一定程度上能帮助我们客观看待世界，进而增强我们对事物的理解和把握。
### 2.1.2 “交叉点”
交叉点，又称“跳跃点”，是指不同结构之间存在的连接、联系点，使得不同结构中的信息流通与变化协调。结构化思考法中的交叉点则主要体现在两个层次结构之间的转换，帮助我们把握问题之间的相互联系、关联和依赖。
### 2.1.3 “跳跃点”
跳跃点，又称“循环点”，是指结构性思维方式中，不同结构之间存在的重复循环，造成信息流不顺畅。在结构化思考法中，跳跃点不仅会削弱结构的连贯性，还可能导致思考失误，并因此而对我们所处的局部区域产生影响，最终导致局部失效。因此，我们需要采取一些有效的措施来克服跳跃点。
## 2.2 什么是金字塔结构？
金字塔结构，是指结构化思考法最重要的组成部分，它是由多层次构成的多重架构，由顶层的主旨和主题支撑着整体的发展方向，底层则承载着具体的实现细节。我们可以通过建立起一个环环相扣的金字塔结构，使得复杂的各种问题在各个层次得到充分的展开、分析和处理，逐步转变为简单的、易于掌握的概念和技能。以下是金字塔结构的构成层次：
1. 抽象层（最上层）：该层对应的是结构性思维的基本单位，一般是对我们感兴趣的或有潜力的事物或事件的一个抽象概括。
2. 组块层（第二层）：组块层对应着我们思考和解决问题的过程，它把我们的问题划分为一些小块，并把每一小块视为一个局部的问题或事件。
3. 概念层（第三层）：概念层是在组块层的基础上，进一步将每一小块细化成一个概念。
4. 问题层（第四层）：问题层是概念层的上一层，主要是对概念层的概念进行进一步的分析和归纳，将概念化的事物分成若干个具体的问题。
5. 解决方案层（第五层）：解决方案层则是在问题层的基础上，展开问题，为问题提供具体的解决方案。
6. 具体行为层（最下层）：具体行为层又称为运作层，是在解决方案层的基础上，结合实际情况，将解决方案具体化，并应用到实际中。
通过这种方式，我们把复杂的问题转化为几个简单、紧凑的概念，然后根据概念制定一系列的、具体的、清晰的、可操作的操作问题，最后再具体化地落实到实际的操作中，这就是整个金字塔结构的运行逻辑。
## 2.3 如何利用金字塔结构来培养创造力？
金字塔结构作为结构化思考法中的一环，可以帮助我们建立起清晰的思考路径，以更好的发现、整合、理解和解释问题。此外，借助于不同的层次结构之间的互联互通，以及同层次结构内的各种思考机制，我们可以更好的做到知识的系统化、归纳、梳理、传播和检索，使得我们具备了一定的创造力。但是，在使用金字塔结构的过程中，我们应该注意以下几点：

1. 发散性思维：结构化思考法的关键在于能够发现并切割问题，其次才是解决问题。只有对问题进行切割和拆解，我们才有可能找到问题的答案；而对于那些既不能划分、又没有直接答案的复杂问题，结构化思考法只能是另辟蹊径，需要我们多花点功夫去摸索、观察、思考、感受，并试图找到共鸣。
2. 反馈循环：结构化思考法要求我们不断对思考结果进行验证和修正，直到得到最佳的答案为止。我们需要不断吸收、学习、总结、反省和修正我们的思路，才能不断走向更优秀的境界。
3. 分享自我经验：结构化思考法要求我们分享自己的经验、见解和知识，并且将这些知识传递给下一代，让后者从中获益。同时，这也促使我们对我们自己所做的事情有更加深刻的认识，从而更好的塑造和改进 ourselves。
