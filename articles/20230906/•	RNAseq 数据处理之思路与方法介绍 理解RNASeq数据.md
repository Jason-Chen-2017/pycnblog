# 1.简介
  

## 1.1 研究背景及意义
遗传学和生物信息学一直是医学界研究的热点方向，随着高通量测序技术的应用日益普及，人们越来越关注RNA病毒、宿主细胞的复杂起源和功能等问题。而关于新冠肺炎疫情的SARS-CoV-2病毒基因组的遗传改造还处于一个关键时期，因此大量的RNA-seq数据也随之产生。正如研究表明，许多RNA疾病的病原体是高度保守且不易复制的，比如黑色素病毒和艾滋病毒，这就要求对原始数据的处理具有极强的灵活性和敏捷性，否则很难从中发现蛛丝马迹。
本文将系统介绍RNA-seq数据的处理流程，并基于实际案例进行讲解。主要涉及到的知识点包括：构建参考基因组序列、质量控制、去除PCR多余序列、测序平台选取、目标检测实验设计、差异表达分析等方面。希望通过阅读本文，能够更好地理解、掌握RNA-seq数据的处理过程。

# 二、基础知识介绍
## 2.1 构建参考基因组序列
构建参考基因组的目的是为了更准确、精确的定位基因变异、筛选差异表达基因。我们通常需要构建一个全基因组序列或全蛋白序列，然后用它来代替某些基因的位置来做比对。构建参考基因组的方法有很多种，包括：单倍体、组装库、组装RNA合成片段，还有对转录本进行分割，构建组装的参考序列等等。最常用的方法就是组装RNA合成片段。组装RNA合成片段的流程如下图所示：
在这个流程里，首先选择一批的单克隆基因，然后对其进行组装，形成待组装的RNA序列。接下来，将这些序列与已经建好的参考基因组进行比对。首先，将待组装的RNA序列进行PCR扩增；然后将扩增后的片段与参考基因组中的所有单链序列进行比对；再把与参考基因组的相似度排名前几的片段选出，作为后续建库的模板。最后，将这些片段加上适当的修饰（例如标签），插入已有的DNA序列中，生成新的参考基因组。
## 2.2 测序平台选择
测序平台是一个至关重要的问题，不同的平台之间存在一些差异，影响了数据的质量和效率。主要有两类测序平台：
1）Illumina平台：由Illumina公司开发的一系列的高通量测序设备，包括微SCOPER、HiSeq、MiSeq、NovaSeq、NextSeq、HiScanSQ、MiniSeq等等。它们的主要特点是成本低廉、抗体配列灵活、测序效率高、适用于海量序列数据。
2）NGS平台：由各大主流生物信息领域的科研机构开发的测序平台，包括GenomeStudio、Nanopore、PacBio等等。它们的主要特点是成本较高、比Illumina设备快、适用于海量序列数据。
选择哪一种测序平台取决于不同领域的需求。比如，如果想要获取细胞核内、组织间、甚至全身的表达信息，那么Illumina测序平台更适合；而如果需要对肿瘤细胞进行定量解析，则需要使用那些支持长读长度的NGS平台。不过，目前尚不清楚哪种平台最适合用来处理SARS-CoV-2这一病毒所带来的新型冠状病毒。
