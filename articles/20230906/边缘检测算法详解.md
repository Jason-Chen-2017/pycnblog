# 1.简介
  

## 1.1 为什么需要边缘检测？
图像处理中边缘检测是最基础也是最重要的一项技术。通过边缘检测，可以从原始图像中提取出对问题研究更加重要的特征信息，例如：轮廓、尖锐、圆润等。边缘检测在图像分析、计算机视觉、模式识别、医学影像、视频监控等领域都有着广泛的应用。而这一功能就是图像检索、图像检索、目标跟踪、自动驾驶等关键技术的前置条件之一。

图像中通常存在很多不明显的边界或纹理信息，这些边缘信息对分析图像的有效信息提取是一个非常重要的任务。如图1所示，左侧为原始图像，右侧为利用边缘检测后的结果。通过边缘检测算法，我们能够从原始图像中提取出特征点、曲线、直线、区域等信息。对图像进行特征分割和分类后，还可以得到图像中各个物体的轮廓信息，方便我们进一步分析和理解。另外，边缘检测往往具有以下两个优点：

1.突出边缘信息，特别是噪声较少的图像；

2.可以用于图像增强，使得图像具有更好的视觉效果，增加了图像的质量。

因此，边缘检测在图像分析、图像检索、图像处理、图像增强等方面有着举足轻重的作用。

## 1.2 边缘检测算法
边缘检测算法有很多种，这里我们将从经典的Hough变换算法开始，逐步深入到目前主流的Canny、Watershed等算法。

## 1.3 Hough变换（直线检测）
Hough变换的原理是通过构建极坐标系中的直线去拟合输入图像中的所有直线。最简单的方式是将图像上的每一个坐标转换成极坐标，然后利用两条坐标表示一条直线，由此求解出该直线的参数（斜率k、偏移b），最后绘制出该直线。

Hough变换的过程如下：

1.预处理图像，灰度化、降低分辨率、二值化等。

2.根据图像的尺寸和形状生成极坐标系。

3.遍历图像的所有可能的线段，计算与相应直线的交点个数。

4.选择交点最多的直线作为输出。

Hough变换优点：

- 简单易懂
- 适用于几何形状的直线检测
- 可以实现快速处理时间短

Hough变换缺点：

- 只适用于平面图像
- 没有考虑到噪声
- 对光照强弱敏感

## 1.4 Canny 算法
Canny 算法是一个经典的基于边缘强度的方法，它的主要思想是先进行卷积运算获取图像梯度方向信息，再进行阈值筛选。通过高斯滤波、Sobel算子等掩模提取边缘，然后求取局部极大值点，通过边缘强度计算最终的边缘强度图。

Canny 算法优点：

- 速度快
- 不容易受到噪声影响
- 不受光照影响
- 边缘清晰
- 提供了一系列自适应的边缘检测方法

Canny 算法缺点：

- 有很多参数需要调节
- 需要进行边缘检测和环投影

## 1.5 Watershed 分水岭算法
Watershed 分水岭算法是一种用来分割图像的迭代推进算法。该算法通过先确定图像的拓扑结构，然后利用粗略的图像边界来细化，最后从细化的边界集合中分离物体。其基本思路是通过赋予每个像素一个灌溉高度，用该灌溉高度来形成一个图像顶点集的集合。随后，可以从每个顶点出发，沿着水流向下搜索，直到找到同类邻居中灌溉高度最小的顶点为止，这样就形成了一个区域。之后，可以重复这个过程，直到所有的顶点都属于某一个确定区域，图像被划分成不同区域。Watershed 算法可以用于分割多种类型的对象，而且适用于光照变化、复杂场景、噪声等情况。

Watershed 分水岭算法优点：

- 在图像中找出区域边界
- 适用于多种场景和环境
- 对光照变化和噪声敏感

Watershed 分水岭算法缺点：

- 计算量大
- 无法处理距离变换
- 边界不太准确
