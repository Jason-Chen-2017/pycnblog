# 3.基本概念术语说明

首先，了解一下阿里云视觉服务的基本概念、术语和框架。

## 3.1 什么是OCR？

通用印刷识别系统(Optical Character Recognition，OCR)是指将图像或者扫描件中文字、符号等内容进行识别、翻译、书写输出的一项计算机技术。它可以用于印刷文本识别、文档图像识别、身份证信息识别、票据识别、财务报表识别、手写文字识别等领域。OCR技术可以用于图像监控、办公自动化、电子商务、金融、医疗卫生、科技、工程、教育、住宿、公共安全等多个行业。

## 3.2 什么是云服务？

云服务是一种通过网络提供的可伸缩、可扩展、按需使用、高度可用且安全的IT服务。它是在广域网、私人网络或其他任何网络上的一项计算资源，可以通过互联网访问。云服务通常由云厂商提供，客户可以利用其所提供的各种功能、应用服务、数据等。

## 3.3 OCR技术组成

1. 字符识别：即把图像中的字符转化为字符序列。
2. 字符定位：识别得到的字符序列需要经过定位才能确定其在图片中的位置。
3. 字符连接：识别到的多个字符序列需要进行合并、整合，形成一个完整的字符串。
4. 模板匹配：模板匹配是指根据输入的字母、数字、颜色等特征，寻找对应的模板图案。
5. 语言模型：语言模型建立了不同词汇之间概率关系，并对文本进行概率计算。
6. 概率计算：概率计算通过词汇间概率关系对文本进行排序，使其具有可信度，从而达到文本识别的最终目的。

## 3.4 阿里云视觉服务介绍

阿里云视觉服务是阿里巴巴基于自然语言处理、计算机视觉、机器学习等技术提供的一站式图像识别服务。通过该服务，用户可快速完成图像识别、图片处理等工作。目前，阿里云视觉服务支持超过70种主流类型图像识别，并提供包括图像分类、物体检测、文字识别、人脸检测、车牌识别、logo定位、图像压缩、图片旋转、清晰度增强等功能。

## 3.5 什么是图像分类？

图像分类是对一张图像或一组图像进行分类、归类，将其划分到不同的类别之中，这些类别一般由人工创建和训练，或者是基于某些规则自动生成的。在图像分类中，输入一张或多张图片，输出图片所属的类别。图像分类是计算机视觉的重要任务，可以实现对图像的自动分类、检索、分析、检索。

## 3.6 什么是人脸检测？

人脸检测又称目标检测，是识别出图像当中的人脸区域，并对各个区域进行分析与标记的一项计算机技术。人脸检测的作用主要包括：判断照片中的人脸是否真实存在，判断摄像头拍摄的人脸是否有效，从人脸图像中获取人的属性信息，识别人脸关键点，从而实现人脸跟踪、人脸识别、年龄估计、性别判定、情绪分析等。

## 3.7 什么是文字识别？

文字识别就是把图像中的文字信息转换为机器可读的文字，属于计算机视觉方向下的一种技术。一般来说，文字识别的方法分为手动和自动两种。手动方法可以分为四个步骤：切割、定位、识别和校对；自动方法则通过程序实现文字识别，目前比较常用的方法有基于深度学习的技术。

## 3.8 OCR常见的优缺点

### OCR优点
- 可靠性高：准确率高，几乎100%正确识别率
- 速度快：秒级响应时间，适用于各种场景和图片
- 技术灵活：灵活易用，可针对不同场景进行个性化配置

### OCR缺点
- 时延较高：用户等待时间长
- 费用高：云服务收费，需要付费购买使用权限
- 只限于文本识别：OCR技术主要用来识别文本信息，对特定领域、图片模糊、噪声敏感、角度变化、大小缩放、背景干扰、字体不一致等场景不太适用
- 数据存储困难：OCR数据存储成本高，需要考虑数据的备份、迁移等问题
