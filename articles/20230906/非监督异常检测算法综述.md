# 1.简介
  

## 1.什么是异常检测
异常检测（Anomaly Detection）是在大数据量下对数据进行分析、探测和发现异常或异常数据点的过程，其目的是为了识别出从整体数据的模式中无法自拔的数据，这些异常数据可能代表了一些真实事件的出现或缺失。

## 2.为什么要做异常检测？
在很多情况下，我们都会遇到某种类型的“反常行为”，如信用卡欺诈、网络安全威胁、电力系统故障、物流系统崩溃等。但我们并不能总是立即发现这些“反常行为”，因为它们很难被直观地察觉。这时，需要通过各种手段来提高警惕性，包括预警、检测、应急处理、勘验、风险评估、反馈、警示等。异常检测就是一种预警手段之一。

## 3.有哪些比较好的异常检测算法？
目前，关于异常检测的算法有许多优秀的方法，下面列举几个比较著名的算法:

1. Isolation Forest
Isolation Forest是一种基于随机森林的异常检测方法。它不像其他异常检测方法那样依赖于训练数据中的正常样本。相反，它采用随机选择的样本集进行训练，并结合了决策树和随机森林等机器学习方法。这个方法的优点是能够捕获数据的全局结构，即便数据存在偏差也能较好地预测异常。

2. Local Outlier Factor (LOF)
Local Outlier Factor 是另一种异常检测算法，它采用密度的方法来计算数据之间的距离，并通过最小化平均的局部密度来寻找异常值。它的优点是可以自动发现不同大小的局部异常值，并且不需要任何训练数据。

3. One-Class SVM (OCSVM)
One-Class SVM 是一种支持向量机（SVM）的异常检测方法。它通过超平面将异常值与正常值分开，并找出属于异常值的样本。与其他异常检测方法相比，OCSVM 的优势在于对异常点和正常点都适用同一个模型，因此准确率更高。

4. DBSCAN
DBSCAN 是一种基于密度的异常检测方法，它扫描整个数据集并找到其中的核心样本。如果一个样本的密度在指定范围内低于某个值，那么它就成为一个孤立点，并作为异常点考虑。它的优点是能够识别任意形状的异常值。

5. Autoencoder Neural Network (AEON)
Autoencoder Neural Network 是一种深度学习方法，它采用编码器-解码器网络来捕捉数据的内部结构。它可以在数据中发现异常，并将异常值映射回原始空间。它能同时利用全局结构和局部特征来发现异常值。


除此之外，还有一些其它的异常检测算法，如 Histogram-based Outlier Detection (HBOS), Distance Basis Function (DBF), Multivariate Anomalous Trajectories (MATD), Robust LOF and kNN algorithm (ROCK). 这些方法都各具特色，各有千秋。在实际应用中，应该选取合适的算法，结合不同的场景、数据分布和业务需求，选择最合适的方案。

# 2. 基本概念术语说明
## 1. Anomaly Detection
异常检测（Anomaly Detection）是在大数据量下对数据进行分析、探测和发现异常或异常数据点的过程，其目的是为了识别出从整体数据的模式中无法自拔的数据，这些异常数据可能代表了一些真实事件的出现或缺失。

## 2. Abnormality/Novelty
异常或异常行为，一般定义为突出离群的，明显超过通常情况的，或者是数据集合中所不具有代表性的现象。例如，信用卡欺诈，网络安全威胁，电力系统故障，物流系统崩溃等。异常行为是对全体数据的可视化展示，如果有明显异常值，则可能代表着不正常的事情发生。

## 3. Positive vs Negative Examples
正负样本：通常，训练数据集由正例和负例组成，正例表示正常（典型）数据，负例表示异常（噪声）数据。由于我们需要区分正常的行为和异常的行为，所以该任务变得十分重要。

## 4. Contextual Anomaly Detection
上下文异常检测，即在上下文信息的帮助下进行异常检测，上下文信息包括外部环境条件、时间序列、位置信息、数据间的关联关系、数据生成机制等，能够提供额外的信息，提升检测效果。

## 5. Unsupervised Learning
无监督学习是指在无需标注数据集的情况下，通过数据自身的特性进行学习的机器学习方法。在异常检测问题中，无监督学习方法可以用于发现数据集中异常值的共性，其优点是不需要特定领域知识，可以有效地探索和理解复杂的数据分布。

## 6. Semi-Supervised Learning
半监督学习是指在训练数据中含有少量的标注数据，并在此基础上完成模型训练，以达到更好的性能。半监督学习可以协助模型建立更好的分类模型，同时保留了模型的鲁棒性。
