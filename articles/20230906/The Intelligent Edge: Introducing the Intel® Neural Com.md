# 2. Intel® Neural Compute Stick 2 产品介绍

## 2.1 产品概述
Intel® Neural Compute Stick 2 是一款由英特尔开发的可编程机器学习加速器，具有高性能的神经网络计算单元和低功耗模式。它搭载在英特尔 Atom® E390 处理器上，带有独一无二的 Intel® Movidius™ Myriad™ X VPU，集成了 Intel® Math Kernel Library 和 OpenVINO™ Toolkit。该产品是一种智能边缘计算设备，可以通过图形界面或命令行工具进行配置、编程和运行。因此，可以用来满足一些不需要执行复杂算法的简单任务。

## 2.2 核心组件
- Intel® Movidius™ Myriad™ X VPU
该芯片负责执行神经网络运算和运算加速，处理从图像识别到文字识别、对象检测和情感分析等多种任务。该芯片支持多种运算类型，如浮点运算、卷积运算、池化运算、归一化运算等。其性能足以支持实时预测和计算机视觉相关任务。

- Intel® Neural Compute Stick 2 模块板
该模块板配备有内存卡、电源接口、USB端口，支持系统级的安全隔离，使得数据安全不会泄露给第三方。模块板还提供两个接口，分别为Micro HDMI接口和VGA接口，方便用户连接显示屏幕。

- 清晰的指示灯
该产品搭载了不同颜色的指示灯，以供用户更直观地了解它的状态。其中绿色的“T”表示正在正常工作；黄色的“I”表示系统处于待机状态；红色的“F”则代表系统出现错误。

- 支持OpenVINO™ Toolkit
该Toolkit是一个开源框架，用于为神经网络模型进行优化、训练和部署，可以轻松地转换、移植、运行。它提供了C++、Python、Java和C#等多种编程语言的API。它也是为了提升智能边缘计算市场的应用普及率和效果，成为各个领域的标准。

- 提供代码生成器
用户可以使用内置的代码生成器来快速地创建训练模型。生成器会自动生成训练所需的原始代码，例如将数据集加载至内存中、定义卷积层、定义激活函数、定义损失函数等。只需要填写简单的配置文件即可完成模型的训练。

- 可编程的硬件接口
该产品的主控芯片采用了英特尔Atom E390处理器，它能够对外提供可编程的接口，即GPIO、I²C、UART、SPI、USB等，方便用户进行控制。通过这些接口，用户可以实现各种应用场景的需求，如手势识别、图像分类、视频监控等。

## 2.3 系统架构


Intel® Neural Compute Stick 2 的系统架构由主机和Myriad X VPU组成，两者之间通过PCIe接口进行通信。Myriad X VPU运行自己的运算引擎，接受从主机传来的指令并执行运算，同时向主机发送结果。主机负责向Myriad X VPU传输数据和接收运算结果，并通过HDMI、VGA等方式向用户输出结果。

## 2.4 工作原理
### 2.4.1 数据处理流程

当数据输入Intel® Neural Compute Stick 2 时，它首先经过模拟信号处理单元，把数字信号转变为模拟信号，然后再通过数字信号处理单元DSP进行加速处理。数字信号处理单元DSP依靠内存和加速器进行快速处理，可以同时处理多个并发任务，从而达到超低延迟的目的。数据经过处理后，才送入神经网络处理单元NPU进行神经网络运算。

### 2.4.2 运算过程

当数据到达神经网络处理单元NPU时，它就会执行神经网络的计算，得到结果后就送回主机，作为输出给用户。运算过程中，NPU会根据运算逻辑进行运算调度和加速处理，充分发挥算力资源，提高运算速度。
