# 2.数字经济基本概念
## 2.1 数字经济的定义
　　“数字经济”（Digital Economy）是指利用数字技术构建的经济体系，它可以看作是“信息技术经济”。2020年中国超2万亿美元的GDP数量，是国内数字经济快速发展的重要里程碑事件。美国政府近年来试图将数字经济纳入美国制裁范围。但由于其影响力太大，近些年来国际上呼声很高。

　　2019年1月，美国加州大学洛杉矶分校、密歇根大学、耶鲁大学、斯坦福大学、马普所等一起发布了“数据驱动创新峰会”（DDCI）。DDCI宣称数字经济的概念已存在于2017年，并对其定义、使命、挑战进行了阐述。数字经济是一个由计算、存储、通信和网络设备组成的经济系统，能够实现“计算开销可控、信息自由流通、生态环境友好、信任容易验证”，并且“能够让所有人都受益”。

　　那么数字经济究竟是什么呢？按照DDCI的定义，数字经济是在互联网、云计算、大数据、人工智能、区块链等新技术领域的应用形成的高度组织化的经济体系。数字经济由基础设施、生态系统、商业模式、服务、研究、开发人员五个方面构成，其中基础设施包括硬件、软件、网络、存储、通信等，其核心功能是提供处理数据的能力。通过网络连接、数据传输、安全防护等手段，把数据转化为有价值的信息，并将信息转化为产品或服务。生态系统则是指支撑数字经济运行的生态环节，如开放平台、服务平台、研究平台、开源软件。商业模式则是指基于数字经济实现的新型商业模式，如通过互联网商务、远程服务、虚拟现实、区块链等。服务则是指通过数字经济提供的各种服务，如人工智能、边缘计算、区块链等。研究则是指基于数字经济的新型研究领域，如机器学习、深度学习、区块链研究。开发人员则是指数字经济相关的技术人员和工程师。

　　综上，数字经济是指利用数字技术构建的高度组织化的经济体系。它包括计算、存储、通信、网络设备等基础设施，与云计算、大数据、人工智能、区块链等新技术相结合，支持全人类实现“共享经济”“协同效应”“资源共享”等理想，并成为实现产业升级的重要引擎。
## 2.2 数字经济的特征
　　数字经济具有以下几个特征：

1. 数据驱动：数字经济核心就是数据。据统计，截止至2020年3月底，全球数字经济规模已达2.9万亿美元。而这个数字也是恒定的，因为数字经济是依靠数据驱动的。

2. 高度组织化：数字经济所涉及到的技术层次众多，从硬件到软件再到互联网，甚至还有分布式系统、物联网、区块链等。因此，数字经济的设计和运营结构复杂、规则严苛，有条不紊地推进技术革命、企业创新、社会参与。

3. 服务经济：数字经济的目标是为消费者提供服务。这一点体现为数字经济的商业模式，包括基于数字经济实现的新型商业模式、基于数字经济提供的各种服务等。

4. 共享经济：数字经济的构架是围绕着数据共享构建的。数据共享不仅仅局限于数字经济内部，而且还扩展到整个社会。共享经济的出现将数字经济的深度、广度和影响力推向了一个全新的高度。

5. 个性化：数字经济的个性化特征表现在消费者对商品和服务的选择权上。这意味着每个消费者都有自己独特的偏好和需求，而这些需求往往会影响到他/她购买什么样的产品或服务。因此，数字经济不断追求个性化服务的实现。
