# 1.简介
  

随着互联网技术的飞速发展、物联网（IoT）技术的广泛应用、云计算、大数据分析的崛起等，越来越多的人将注意力转移到物联网安全这一重要领域。根据美国国家标准与技术研究所(NIST)定义的物联网安全标准，物联网设备面临的风险包括恶意攻击、物理攻击、电磁干扰、网络攻击、攻击者获取敏感信息、数据泄露、欺骗行为等。如何构建并保障物联网设备的安全，成为制约物联网发展和落地生态的关键问题。

2019年，美国政府颁布了物联网安全规定，旨在对抗物联网安全威胁，规范物联网安全体系建设和运营，建立健全物联网安全管理制度。从目前的情况看，由于互联网、物联网、云计算、大数据、人工智能、机器学习等新兴技术的快速发展，尤其是人工智能、自动化、大数据的快速增长，使得物联网安全已经成为越来越复杂、面临越来越大的安全威胁。

3.建设方向与目标
建立一个适合企业环境的“绿色”物联网安全体系，需要关注以下几个方面：
1. 安全管理：针对企业安全需求，制订安全管理规范、流程及工具，确保安全人员掌握必要的安全技能，具备应对网络攻击、物理攻击、恶意攻击、内部风险的能力。同时，还要制定内部政策、行业标准、行业惯例等安全管理制度和手册。
2. 安全工程：根据当前技术水平、安全管理制度和法律法规要求，确定安全工程建设的范围、方式、任务、时间节点、资源投入等。建立系统化的安全工作机制、组织体系和标准，包括安全工程师职业培训、内部训练、职业道德建设、项目管理方法论、人员任免等。为物联网安全解决方案提供支撑性的基础设施。
3. 智能检测：通过提升传感器采集数据质量、降低误报率、提高隐私保护等技术，提升智能检测能力。通过联动收集到的信息进行分析，进行可信设备发现、网络设备异常监测等。
4. 信息共享与交流：通过互联网的方式提供实时信息共享服务，包括平台、技术支持、咨询、培训等，让越来越多的人参与到“绿色”物联网安全建设中来。
5. 数据安全：物联网数据安全也是提升企业绿色物联网安全建设不可或缺的一环。数据安全既涉及数据的收集、存储、处理、传输过程中的保密、完整、可用性，也需考虑第三方对数据的访问控制、使用限制等安全性要求。

文章主要讨论“绿色”物联网安全体系的建设，以IoT作为主题。主要内容如下:

1. 安全管理

- 安全事件应急响应：建立符合现代国家标准的警察执勤制度、专门应急部门和流程，定期清理事件发生场所以及相关的设施。
- 安全人员能力建设：熟悉物联网安全知识、技能和工具，充分使用物联网安全管理软件，建立个人网络防御能力和应急反应能力。
- 安全设施建设：建立安全区域内的固定资产，根据安全环境要求对人员安全管理设施进行评估、改造、升级；建立建立、维护、监控、处置等日常安全管理工作流程。
- 设备认证：定期对进入物联网的设备进行认证，保证设备安全、性能满足生产规范。
- 技术支持：协助企业客户申请、购买、部署物联网安全相关技术产品和服务。

2. 安全工程

- 风险评估：根据组织机构、业务规模、系统架构、设备数量等条件，定期对物联网安全风险进行评估，识别出威胁源头、风险点、威胁类型和影响范围。
- 安全设计：采用层次分明的安全设计原则，制定详细的安全方案，按照安全要求评估产品和服务。
- 安全测试：根据评估结果和开发进度，结合测试用例执行静态、动态、渗透测试，进行安全漏洞分析、安全风险排查、审计检查等。
- 测试记录：记录各个阶段的测试过程、结果、问题、建议、回顾、整改过程，形成专业的安全测试记录。
- 安全事件响应：根据安全事件的严重程度、影响范围、预警、处置措施，制定灵活的应急响应机制，确保业务运行正常、资金安全和人员安全。

3. 智能检测

- 传感器测试：对传感器上电前后的初始测试进行检查，确保传感器能够读取有效信号。
- 漏洞扫描：定期对设备上装的软件、固件进行漏洞扫描，发现可能存在的安全漏洞。
- 欺诈行为检测：对设备上的数据流进行实时分析，检测出现异常的行为模式，从而发现可能存在的欺诈行为。
- 身份鉴别：对用户身份进行实时检测，发现存在异常的活动、登录、数据违规行为。
- 实时报警：基于设备数据的可视化报表及分析模型，及时发送安全警报，减少设备故障带来的损失。

4. 信息共享与交流

- 服务运维：建立物联网安全相关服务运维团队，承担产品和服务的售前、售后、技术支持工作，提供云端、移动端、大屏、人机界面等安全相关产品和服务。
- 技术攻关：利用云计算、大数据、人工智能、机器学习等新技术，推动物联网安全技术的持续创新和突破。
- 演练测试：举办安全演练活动，引导安全人员了解物联网安全的相关知识，进行技能提升、实践经验分享等。
- 用户参与：建立社区、论坛、邮件列表、线下聚会等，让越来越多的人参与到“绿色”物联网安全建设中来。

5. 数据安全

- 数据收集：设备产生的原始数据、日志、数据流都要收集，确保数据安全。
- 数据安全：采用加密技术对数据进行加密存储、传输、访问控制。
- 个人数据保护：确保个人数据安全，避免数据泄露、被篡改和窃取。
- 安全审计：每月对物联网设备的运行状况进行审计，发现存在的安全风险，做好应急准备。

总结：绿色物联网安全建设是一个复杂且艰难的过程。但只要有正确的方向、决心、措施，只要组织的人员具有相应的知识和能力，就一定可以成功实现“绿色”物联网安全建设。
