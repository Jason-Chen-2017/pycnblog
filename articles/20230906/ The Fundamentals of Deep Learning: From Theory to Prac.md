# 1.简介

# 2.历史概述
​		深度学习的起源可以追溯至1940年代，当时由阿兰·图灵提出的模糊机器人是深度学习的先驱。随着近代的人工智能的发展，机器学习也在不断进步，随着深度学习的问世，深度学习这个术语才正式确立。深度学习的起源可以追溯到上世纪50~60年代的快速发展期，由很多研究人员组成的小型团体一起研究神经网络的结构、学习能力、适应性、鲁棒性等方面，提出了一系列深度学习的模型，如自编码器、深层网络、自动编码器等。深度学习这个词的产生始于蒙特卡洛树搜索算法（Monte Carlo Tree Search, MCTS）与强化学习（Reinforcement learning, RL）之间的交流，MCTS被用于游戏行业，RL被用于控制领域。2012年ImageNet图像识别竞赛的冠军便是深度学习的一个里程碑事件。
​		深度学习这个名字得益于神经网络的构想，即深层次的多层神经元网络的集合。深度学习最早的名称叫作“深层网络”，而近年来也变成了“深度学习”。深度学习在不同的领域都有应用，但目前看来，它的优势主要集中在图像识别、自然语言处理、语音识别、自动驾驶等领域。深度学习的核心技术主要有两大类：
​		1. 模型选择和训练策略：深度学习算法通常采用梯度下降、随机梯度下降或其它基于梯度的方法，通过迭代更新模型参数来优化损失函数。其中，反向传播算法（Backpropagation algorithm, BP）以及其它一些优化方法，是目前深度学习的主流方法。
​		2. 基本模块：包括激活函数、正则化、优化器等，这些都是深度学习模型的基本要素。
​		因此，深度学习具有一定的通用性、普适性和高效率，能够解决各类复杂的问题，而且取得了非凡的成功。
# 3.分类与发展趋势
​		深度学习主要可分为三种类型：监督学习、无监督学习、半监督学习。按照任务的分类方式不同，又可以细分为以下几类：
​		1. 回归问题：预测一个连续值，如房价、销售额、薪资等。典型例子为线性回归、决策树回归、支持向量机回归等。
​		2. 分类问题：预测离散值，如图像分类、文本分类等。典型例子为逻辑回归、决策树、朴素贝叶斯、支持向量机、神经网络等。
​		3. 聚类问题：将样本分成若干个集群，如客户群、网页主题分类等。典型例子为K均值聚类、层次聚类、高斯混合模型聚类等。
​		4. 生成模型：生成新的数据样本，如图像生成、语音合成等。典型例子为循环生成模型、变分自动编码器、变分推断网络等。
​		5. 强化学习：模拟决策过程，基于奖励-惩罚机制进行训练，比如AlphaGo。
​		除了上面这些类型，还有其他类型的深度学习，如强化学习、迁移学习、增强学习等。
