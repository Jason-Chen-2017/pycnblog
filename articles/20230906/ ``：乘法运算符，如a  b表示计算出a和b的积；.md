# 1.简介
  

乘法运算符`*`用于计算两个数字的积或乘积，比如：5 * 3 = 15，7.5 * 2 = 15。实际上，当运算符两边的元素都是整数时，乘法运算的结果仍然是一个整数。但如果其中一个元素为浮点数（小数），则结果也会变成浮点数。例如，2.5 * 4 = 10。
# 2.乘法定义及性质
## 2.1 乘法定义
对于任意实数a和b，其乘积等于ab，即a × b = ab。在多元数学中，如果f(x)和g(x)是任意函数，那么它们的复合函数fg(x)满足如下关系式：
```
fg(x)= f(g(x))
```
这就是所谓的函数复合定律。同样地，对于两个向量X=[x1, x2,..., xn]和Y=[y1, y2,..., ym], 它们的点积（dot product）可以表示为：
```
X·Y=∑xi*yi
```
这个向量积形式就叫做向量积。对于标量和向量之间的乘法运算来说，乘积、点积等概念不再太重要了，因为乘法运算的结果既可能是标量还可能是向量，而其概念上仍保持一致。
## 2.2 乘法运算中的一些性质
### 2.2.1 可交换律
对于任意实数a和b，有 a * b = b * a 。这个可交换律可以用来消除括号的影响。因此，如果你先乘以b，然后乘以a，最后得到的结果和先乘以a，然后乘以b的结果一样，就说他们是可交换的。比如，3 * (4 + 5) = (3 * 4) + (3 * 5)，因此3*(4+5)和4+(3*5)是等价的。
### 2.2.2 次序交换律
对于任意实数a,b,c,有 a * (b * c) = (a * b) * c 。这个次序交换律表明乘法运算的优先级顺序是从左到右，而不是从右到左。因此，如果你先乘以a，再乘以b，最后乘以c，就会比先乘以a，再乘以c，最后乘以b的结果更加有效率。比如，3 * 4 * 5 = 3 * (4 * 5)。
### 2.2.3 结合律
对于任意实数a,b,c,有 a * (b + c) = a * b + a * c 。这个结合律表明，如果你先乘以a，然后分解成b+c两项，然后分别对b和c进行乘法运算，最后再相加，与先将a乘以b+c的结果再乘以a相同。因此，3 * (4 + 5) = (3 * 4) + (3 * 5)。
### 2.2.4 单位元
假设乘法运算有一个单位元，它与任何其他元素相乘都等于自身。这个单位元通常被称作"1"。在许多情况下，例如在计算机编程领域，"1"指的是整数1。但在文献中，经常把"1"理解为一个特殊的矩阵。不过，在线性代数的上下文中，乘法的单位元并不是“1”，而是不存在的元素。
