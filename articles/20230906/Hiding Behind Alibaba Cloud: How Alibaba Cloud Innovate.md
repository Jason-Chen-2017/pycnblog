# 2.基本概念术语说明：
## 2.1 云计算
云计算（Cloud Computing），也称为基于网络的计算服务，是指利用 Internet 网络为用户提供计算服务的一种服务模式。主要特点包括按需分配资源、高度自动化管理、自动弹性伸缩、服务质量保证和全球分布。目前，云计算服务得到了迅速发展，其覆盖范围已从个人电脑到互联网、移动通信、金融服务等各个领域，使得各种业务系统、应用部署变得更加便捷、高效、可靠。
## 2.2 大数据
大数据（Big Data）是指存储海量数据的集合，通常由非结构化或半结构化的数据组成，包括文本、图像、视频、音频、生物信息、社会网络、交易记录等。传统的数据仓库技术无法满足快速分析大数据的需求，因而产生了大数据挖掘、人工智能、机器学习等新兴技术。
## 2.3 深度学习
深度学习（Deep Learning）是机器学习的一种分支，它利用神经网络进行高层次特征学习，可以有效地识别、分类及预测数据。该方法经过多年的发展，取得了巨大的成功，现已被广泛应用于诸如图像识别、语音识别、搜索排序、推荐系统、金融风险评估、疾病检测、医疗保健等多个领域。
## 2.4 分布式数据库
分布式数据库（Distributed Database）是指分布式环境下数据库系统的设计及实现。简单来说，分布式数据库就是将数据库按照不同节点分布在不同的计算机上，并通过网络相互连接。目前，分布式数据库技术在很多重要领域都起到了很好的作用。
## 2.5 Hadoop MapReduce
Hadoop MapReduce 是 Apache 开源项目中的一个框架，用于编写和运行能够同时处理海量数据集的批处理任务。MapReduce 的主要特点是分而治之，即把复杂的任务拆分成多个简单的任务，并把每个任务分别分配给不同的节点去完成。它的编程模型类似于并行程序设计中的 Map 和 Reduce 操作。
## 2.6 Spark
Spark 是由前 Cloudera 公司开发的开源大数据集群计算框架。Spark 提供了快速的响应时间、易用的数据结构、丰富的工具支持，并且能够支持多种编程语言。Spark 通过 Lambda 函数提供了快速的交互式查询功能。
## 2.7 Tensorflow
TensorFlow 是 Google 开源的开源机器学习框架。它提供高效的计算能力和广泛的模型支持，可以用于构建大规模的深度学习系统。

## 3.1 波动率预测：
为了避免由于数据质量、时序关系等其他不可控因素导致的波动率预测结果偏差，阿里云采用了以下方案进行波动率预测：首先，分析证券、债券、股票三种品种之间的区别，确定要使用的模型类型；然后，使用阿里云自研的机器学习算法，对每天的数据进行处理，提取出股票的主力趋势、支撑阻力线、以及市场整体走势等特征，最终得出每日的波动率预测值。波动率预测可以帮助投资者做出正确的投资决策，比如选购特定股票、买入卖出期权等。
### （1）市场整体走势特征
阿里云基于证券、债券、股票三种不同品种，提炼了以下市场整体走势特征：
- 最大值-最小值差值特征：计算两个连续数据序列之间最大值和最小值之间的差值，作为趋势变化的大小指标。该特征反映价格变化的程度。
- 上升浪费比特征：计算近段时间最高价与最低价的差值的百分比，作为反映市场整体涨跌的大小指标。该特征衡量上升过程中的浪费情况。
- 上涨线-下跌线距离特征：计算上涨线与下跌线之间的距离，作为二条曲线组合的强弱程度。该特征衡量趋势的延续情况。
- 震荡宽度特征：统计近段时间内价格变化的幅度分布，作为反映短期波动的弱、中、强的信号。该特征反映价格运动的快慢程度。
### （2）支撑阻力线特征
阿里云基于五档的盘口数据，提炼了以下支撑阻力线特征：
- 支撑线特征：统计长期平均收益率在支撑线以上区域所占比例，作为反映当前价格是否处于“多头”状态的指标。该特征反映股价未来走势的方向。
- 阻力线特征：统计长期平均收益率在阻力线以上区域所占比例，作为反映股价的反弹力度。该特征揭示了股价下跌的压力。
- 支撑阻力线交叉特征：判断今日价格是否超出了支撑阻力线，作为股价走势的变化信号。当价格超出支撑线时，可能出现反弹；当价格低于阻力线时，可能出现收敛或反转。
### （3）主力趋势特征
阿里云基于高、低、开、收四个数据点，提炼了以下主力趋势特征：
- 收阳线特征：统计开盘价最高的一天与收盘价最低的一天之间的距离，作为反映主力趋势当前方向的指标。该特征反映主力趋势的进展情况。
- 超额上轨特征：计算收阳线的价位与收盘价的差值，作为反映主力趋势的超额发力水平的指标。该特征表明股价的发力强度。
- 持续时间特征：统计某个收阳线对应的时间长度，作为反映主力趋势持续的时间长度的指标。该特征表示主力趋势的延续力度。
### （4）模糊波动率预测
阿里云还采用了对未来三天、七天甚至一周波动率的预测，作为附加项，提升模型的准确性。这三个月内的平均波动率，也是用来作为波动率预测的依据。该方法可以更好地考虑到长期趋势的影响。
### （5）阿里云机器学习算法
阿里云基于 TensorFlow 框架，搭建了波动率预测模型，模型采用 LSTM 循环神经网络（Long Short Term Memory Network）算法。该模型对每种品种的数据输入都不一样，针对不同品种，分别训练模型参数，得到不同品种波动率的预测结果。模型的参数调整以在测试集上达到较好的效果为目标。
## 3.2 风险评估：
对于金融产品的投资，投资者往往对风险有着更为敏感的感觉。因此，阿里云希望通过技术手段，降低金融产品的风险。因此，阿里云开发了一款名为”银河：光谱风险评估“的风险评估工具，这款工具可提供客户“定向”风险评估。
### （1）风险控制标准
阿里云的风险控制标准包括：
- 发行量控制：设置合理的发行数量，并加强监管，防止发生黑天鹅事件。
- 投资策略控制：定期审视自己交易的仓位，确保投资策略的合理性。
- 仓位限制：限制客户的仓位，确保他们不会过度参与金融市场。
- 资产配置控制：根据资产类型的不同，设置不同的初始资产分配计划。
### （2）风险评估工具
阿里云的风险评估工具”银河：光谱风险评估“，能够对整个金融市场及客户进行实时的风险评估。该工具通过分析不同国家、市场、行业等金融产品的基础信息、行为习惯、所处的环境以及相关法律法规，建立起与客户交易的风险模型。风险模型结合客观指标、模型预测指标以及规则引擎，最终输出客户的“定向”风险评估结果。客户可以通过风险评估工具，对自己的投资行为、风险状况以及投资回报做出更加负责任的判断。
