# 2. 基本概念术语说明

## 2.1 概念：什么是Hello World?

"Hello world!" 是计算机程序员中非常著名的一个编程练习。它被广泛地用于教授初级编程概念，以及验证一个程序是否能正常运行。世界上第一个打印出来的 "Hello world!" 问候语源于 1970 年代末，由瑞典的学者 Edsger Dijkstra 提出，指出在打算开发新语言或系统时，应当首先编写输出 "Hello world!" 的程序，以确保编译器、解释器以及其他辅助工具正确运行。

## 2.2 算法和数据结构：什么是回文字符串？

回文字符串是一种最简单的序列数据类型，只允许以相同的方式进行读取和写入。通常情况下，回文串往往是自然界中的对象，如数字、单词、句子等。而对于一般的字符序列，判断其是否为回文串的方法有很多种，其中一种便是比较两个指针指向的元素是否相等，同时移动两个指针直到它们碰撞。

## 2.3 技术：什么是前端开发？

前端开发（Front-end Development）是一种综合性的开发过程，用来创建动态网站和应用程序用户界面，利用现代化的 Web 技术、JavaScript 等技术实现功能丰富、高效的用户体验。前端开发人员负责制作网页的静态结构、内容、以及用户交互，如网页布局、版式设计、字体排印、动画效果等，通过 HTML、CSS 和 JavaScript 语言实现。前端开发工程师需要会HTML、CSS、JavaScript、jQuery、AJAX、JSON、Bootstrap等技术。
