# 1.简介
  

本文主要分享了一些学习机器学习、深度学习等领域的入门材料。欢迎大家共同参与补充和完善。
# 2.课程
本视频由吴恩达老师开设的机器学习基石课程，是对机器学习的一个系统性认识。主要内容包括：线性代数基础、概率论基础、信息论基础、代价函数、几种优化方法、集成学习、半监督学习、强化学习等。总体收益适中，难度较低。
这是李沐老师的“动手学深度学习”课程，汇集了对深度学习的基础知识点，配合视频教程和python实现，让你可以快速上手实践。主要内容包括：神经网络基础、前向传播、反向传播、正则化、激活函数、损失函数、优化算法、卷积神经网络、循环神经网络、深度置信网络（DCN）、生成对抗网络（GAN）。付费课程，需要付费后才能观看。
本书是斯坦福大学出版社出的机器学习入门书籍。全书分三大部分：第1部分对统计机器学习的基本原理进行了详细阐述；第2部分重点介绍了机器学习中的典型算法，包括感知机、支持向量机、决策树、随机森林、K近邻、逻辑回归等；第3部分介绍了Python编程环境下机器学习模型的搭建及应用。深受广泛好评。
Coursera的机器学习课程由Prof. Andrew Ng老师担任课程主讲人，涉及的内容包括线性代数、概率论、信息论、机器学习算法、特征工程、深度学习、监督学习、无监督学习、强化学习、迁移学习等。该课程相当精彩，建议有一定机器学习基础的人士看一下，能够对自己机器学习的学习路径有所帮助。
MIT的深度学习课程从浅到深，从线性代数、优化算法到深层神经网络，并介绍如何通过TensorFlow实现这些算法。课程的难度不高，但是内容丰富，值得每一个对深度学习感兴趣的人学习。
斯坦福大学出版社的计算机视觉课程，全面介绍视觉识别的各个组成模块，如空间变换、几何变换、光流、特征提取、分类器等，还有经典的图像匹配、单词识别、行人检测、人脸识别等应用案例，很有深度且系统的视角。
斯坦福大学出版社的自然语言处理课程，主要介绍自然语言理解、文本表示、词性标注、句法分析、机器翻译、信息抽取、主题模型、情感分析、命名实体识别、语音合成、关系抽取、对话系统等内容。难度适中，但对于想了解自然语言处理技术的同学来说应该是不错的选择。
DeepMind公司推出了一款基于蒙特卡洛树搜索（Monte Carlo Tree Search，MCTS）算法的强力游戏AI——AlphaGo，无需任何自学习能力就能获胜世界围棋冠军。这是一个入门级教程，提供了对 AlphaGo 的简单介绍，并以围棋游戏为背景，以引导读者一步步熟悉强化学习领域的各种知识。
