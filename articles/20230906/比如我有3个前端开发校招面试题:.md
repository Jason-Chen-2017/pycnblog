# 2.基本概念术语说明
## 2.1 JavaScript 基操
javascript 是一种脚本语言，可以实现动态的网页更新，可以将 javascript 嵌入到 html 中，也可以单独作为一个 js 文件使用。它由 ECMAScript 和 DOM（文档对象模型）组成。

ECMAScript 是 javascript 的核心规范，定义了 javascript 语法和各种标准函数的行为。最新版的 ECMAScript 是 ECMA-262 规范。

DOM（文档对象模型），是一个接口，用于操作网页上的元素，比如增删改查元素，添加事件监听器等。

## 2.2 HTML/CSS 基操
HTML 是超文本标记语言（Hyper Text Markup Language）的缩写，用来创建网页结构。CSS（层叠样式表）用来设置网页的视觉风格。

HTML 使用标签来描述网页的内容结构，例如 head、title、body 等。每个标签都有开始标签和结束标签，中间可以放置相应的属性和内容。

CSS 通过选择器来控制页面元素的样式，例如 body 标签样式，h1~h6 标签样式等。CSS 可以对某些元素进行动画效果，并可以根据屏幕大小自动调整布局。

## 2.3 HTTP 基操
HTTP 是 Hypertext Transfer Protocol 的缩写，是一个应用层协议，用于从客户端向服务器发送请求和接收响应的数据。

HTTP 请求包括请求行、请求头部、空行和请求数据四个部分。请求行包括请求方法、目标 URL、HTTP 版本号。

HTTP 响应包括响应行、响应头部、空行和响应数据四个部分。响应行包括 HTTP 版本号、状态码、状态信息。

HTTP 状态码用于表示 HTTP 处理的结果。常用的状态码有 2xx 表示成功，3xx 表示重定向，4xx 表示客户端错误，5xx 表示服务器端错误。

HTTP 缓存主要是指浏览器缓存机制，浏览器会先检查是否存在该资源的缓存，如果有则直接返回缓存数据，否则才向服务器发起请求获取资源。

## 2.4 AJAX 基操
AJAX (Asynchronous JavaScript and XML) 是一种基于 XMLHttpRequest 对象实现的非阻塞异步通信技术。

AJAX 技术允许 Web 页面通过后台载入信息而无需刷新整个页面，有效地实现用户体验的提高。

AJAX 使用 XMLHttpRequest 对象来向服务器发送请求，从服务器获得数据并作出相应的更新。

## 2.5 JSON 基操
JSON 是 JavaScript Object Notation 的缩写。它是一种轻量级的数据交换格式，易于被人阅读和生成，并易于机器解析和使用。

JSON 数据在结构上类似于对象，但使用字符串而不是键值对的方式来存储。这样做的好处是可以在不同编程语言间传递。

# 3.核心算法原理和具体操作步骤以及数学公式讲解
## 3.1 数据类型转换
#### Number() 方法
Number() 方法用于将任意值转换为数字。如果参数是一个数字，返回该数字；如果参数是字符串，则尝试把它转换成浮点数；如果不能转换，则返回 NaN 。
```js
console.log(Number("12")); // 12
console.log(Number("-Infinity")); // -Infinity
console.log(Number("abc")); // NaN
```

#### parseInt() 方法和 parseFloat() 方法
parseInt() 方法用于将字符串转换成整数，parseFloat() 方法用于将字符串转换成小数。两个方法都接受一个字符串作为第一个参数，第二个参数可选，指定字符串的起始位置和结束位置。

parseInt() 方法与 Number() 方法的作用相同，只是 parseFloat() 方法更适用于浮点数。

```js
// parseInt() 方法
console.log(parseInt("12.34")); // 12
console.log(parseInt("-123")); // -123
console.log(parseInt("+123")); // 123
console.log(parseInt("abc")); // NaN
console.log(parseInt("12.34", 10)); // 12

// parseFloat() 方法
console.log(parseFloat("12.34")); // 12.34
console.log(parseFloat("-123")); // -123
console.log(parseFloat("+123")); // 123
console.log(parseFloat("abc")); // NaN
```

#### String() 方法
String() 方法用于将任意值转换为字符串。如果参数是一个字符串，则返回该字符串；如果参数是 null ，则返回 "null"；如果参数是 undefined ，则返回 "undefined"；如果参数是布尔值，则返回 "true" 或 "false"。
```js
console.log(String(123)); // "123"
console.log(String(null)); // "null"
console.log(String(undefined)); // "undefined"
console.log(String(true)); // "true"
console.log(String(false)); // "false"
```

#### Boolean() 方法
Boolean() 方法用于将任意值转换为布尔值。如果参数为真值，则返回 true；如果参数为假值，则返回 false。只有以下的值会被转换成 false：false、"" （空字符串）、0、NaN、null、undefined。
```js
console.log(Boolean("hello")); // true
console.log(Boolean({})); // true
console.log(Boolean([])); // true
console.log(Boolean(123)); // true
console.log(Boolean(-1)); // true
console.log(Boolean(0)); // false
console.log(Boolean(null)); // false
console.log(Boolean(undefined)); // false
```

#### typeof 操作符
typeof 操作符用来判断变量的数据类型。如果操作数是一个引用类型的值（数组或对象），则 typeof 操作符返回"object"；如果操作数是一个原始类型的值（字符串、数值、布尔值），则 typeof 操作符返回对应的类型名称。

```js
console.log(typeof "Hello"); // string
console.log(typeof 123); // number
console.log(typeof true); // boolean
console.log(typeof []); // object
console.log(typeof {}); // object
console.log(typeof null); // object
console.log(typeof undefined); // undefined
```

## 3.2 运算符优先级
运算符优先级（Operator Precedence）决定了表达式中各个运算符的执行顺序。

以下是最重要的几种运算符的优先级：

1. 括号 ()
2. 一元 +/- 、!
3. 乘除 % **
4. 加减 +/-
5. 位移 << >> >>>
6. 关系 < > <= >= instanceof
7. 相等 ==!= ===!==
8. 按位与 &
9. 按位异或 ^
10. 按位取反 ~
11. 逻辑与 &&
12. 逻辑或 ||

以下为运算符优先级的列表形式：

```
(),!, ~, ++ --, typeof, void, delete,., [ ], ->, new, sizeof, *, /, %, +, -, <<, >>, >>>, <, >, <=, >=, instanceof, ==,!=, ===,!==, &=, ^=, |=, &&, ||
```
