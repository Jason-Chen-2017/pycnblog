# 2.产品介绍
## Ansys Maxwell
Ansys Maxwell是一个基于分治方法的数值计算平台，用于求解偏微分方程组(PDE)和偏微分方程(ODE)。Maxwell模型可以处理各种问题，例如力学、热学、磁性、辐射和其他相关问题。它可以模拟时变流体的动力学行为、非均匀介质中的磁性特性、传播介质电流、两层超导结构中的感应电流、磁悬浮器中的磁力、建筑物结构的安全评估、水处理系统中的反吹滴现象、农业和环保领域的温室效应和灌溉剂设计等。

Ansys Maxwell产品由以下三种版本供用户选择：

1. Ansys Professional Edition: 是专业版，功能最丰富，但价格较贵；
2. Ansys Student Edition: 是学生版，适合初中及以下学历学习者；
3. Ansys Emagine: 是学习型应用版本，只针对少量电路进行数值分析。

本文将以Professional Edition版本为例进行介绍，介绍如何在Maxwell环境下进行金属材料的受压过程的研究。
## Ansys Maxwell的使用流程
### 准备工作
1. 安装ANSYS Maxwell Pro，首先要去官网下载安装程序并完成安装。如果需要激活，也要先进行激活；
2. 安装好后，打开Maxwell Pro并登录账号；
3. 在Models选项卡里，选择所需的分析类型；
   - Magneto-hydrodynamics (MHD): 测量流体动力学参数（如压强、温度、相场）；
   - Structural Analysis (SA): 求解结构或混凝土的重心、单元应变、节点强度、挤压、屈服力、轴向载荷等；
   - Mechanical APDL (MAPDL): 使用APDL语言求解物理与工程问题，如自由落体问题、内摩擦问题、流体静力学、应力输运、复合材料力学、结构材料力学、连续介质力学、地震、混沌、热流控制、固体绝缘体材料、塑性材料等；
   - Thermal Analysis (TA): 模拟热力学系统，包括壳体热传递、温度场积分、热槽罐模型、热力装置故障诊断、风机表面温度预测等；
4. 在Analysis选项卡中，选择所需的变量，设置相应的参数；
5. 通过导入相应的数据文件或直接输入数据的方式，添加数据；
6. 设置输出结果选项卡中的参数，确定结果文件的保存路径、格式、图形显示方式等；
7. 执行分析，得到结果文件。
### 研究案例——金属材料的受压过程
#### 数据收集阶段
使用浸润法测试金属材料的受压行为，即用钢珠先固定金属材料，然后将其放出钢珠孔中，观察钢珠的移动与金属材料的受压过程。
1. 用钢珠固定金属材料，并给予足够的劲度以使得金属材料完全嵌入钢珠中；
2. 将金属材料放在由钢珠孔决定的位置上，确保金属材料和钢珠的间隙不会过小，以便观察钢珠的活动范围；
3. 在金属材料前方放入探测器，探测器可以采用弧焊、加力焊或外加探测器等；
4. 对比检测和对准法确认钢珠活动范围；
5. 等待钢珠静止，然后利用平衡色散法测量钢珠与金属材料之间的偏移程度；
6. 根据平衡色散法的测量结果，确认钢珠活动范围。
#### 数据清洗阶段
1. 从测试记录中获取测量值，并删除掉不完整或无效的数据；
2. 检查数据完整性，确保没有缺失值或异常值；
3. 对照钢珠与金属材料的尺寸，校正测量值；
4. 删除误差大的测量点；
5. 对同一部件的不同测试进行合并，避免重复测量；
6. 生成修正后的测量值和残差；
7. 保存测量值与残差数据。
#### 数据分析阶段
1. 使用平衡色散法计算金属材料和钢珠之间的偏移程度；
2. 分别将金属材料受压区和钢珠区域分别标记，并绘制压力曲线；
3. 判断钢珠活动范围，并检验钢珠是否正常稳定；
4. 如果钢珠活动范围较窄，则考虑增加钢珠直径，减轻钢珠材料压迫；
5. 调整金属材料的厚度，调节金属材料分布，使得钢珠与金属材料之间的连接处出现明显的裂纹；
6. 如果钢珠活动范围较宽，则考虑增加钢珠直径，增大钢珠材料压迫；
7. 若钢珠在受压过程中发生钢珠与金属材料之间连接处裂纹，则考虑缩小金属材料的厚度；
8. 保存测量值与残差数据。
