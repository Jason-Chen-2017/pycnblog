# 1.简介
  

线性代数(Linear Algebra)是现代数学的一个分支，由向量空间、矩阵运算、线性变换等概念构成，是一种利用向量及其之间的加减乘除关系来进行高效运算、求解复杂系统模型、解决实际问题的方法论。本文将基于线性方程组的角度，带领读者了解线性代数的一些重要概念及其具体应用。

# 2.基本概念和术语
## 2.1 向量空间
向量空间(Vector Space)，也称线性空间或线型空间，是指一组向量所构成的集合，每一个向量都是该空间中的元素。这些向量一般存在加法和标量乘积两个操作，满足结合律、分配率、交换律以及对称性的四则运算规则，并有如下几个特点：
- 内积（Dot Product）：也叫做点积、内积或数量积，是两个向量对应位置上的乘积之和。
- 范数：又称模长或长度，是一个非负实数，定义为一组向量中所有分量的平方和的平方根，即sqrt[(x1^2)+(y1^2)+...+(xn^2)] 。
- 张成：在向量空间V上任取两组向量，它们的张成空间C(V)=Span(v1,...,vn)是一个新的向量空间，它包含V中所有的向量，而且它们的张成线性组合仍在C(V)上。

## 2.2 基和维数
设V是一个向量空间，V=Span(v1,...,vk)，其中vk∈Rk是V的基，则vk=(vk1,...,vkn), vk1,...,vkn属于Rn, vi的个数称为基的维数。显然，如果V中存在其他非零向量vj，使得vj∉Span(v1,...,vk)，那么基就不是唯一的。基的选取可以有不同的方式，如可以按线性无关的约束条件进行选择。

## 2.3 矩阵
对于任意两个相同维数的矩阵A,B，如果满足以下三个条件，称A,B为同型矩阵:
- 每一行都只有零或者是实数。即A[i][j]∈R, i=1,2,...,m; j=1,2,...,n。
- 有n个列向量形成了一个基，即B = C(A); B[i][j]=aj, j=1,2,...,k, C(A)[i][j]=ci, i=1,2,...,m, ci∈N, k=rank(A)。
- 对任意两个不同行号的行，其对应的列向量之间没有规定关系。即不存在下标i<j使得A[i]!=0且A[j]==0。

## 2.4 向量
若n维向量被称为矢量，记作：u=(u1, u2,..., un), v=(v1, v2,..., vn)，那么：
- 欧几里得距离(Euclidean Distance): 又叫L2范数或欧式距离，定义为sqrt[(u1-v1)^2 + (u2-v2)^2 +... + (un-vn)^2], 两点间的距离。
- 切比雪夫距离(Chebyshev Distance): 又叫Linf范数，定义为max(|u1-v1|, |u2-v2|,..., |un-vn|), 一组向量间的最远距离。
- 曼哈顿距离(Manhattan Distance): 又叫L1范数，定义为|u1-v1|+|u2-v2|+...+|un-vn|, 一组向量间的曼哈顿距离。

# 3.线性方程组的求解
## 3.1 直接法
当系数矩阵A和右端常数列b均满足如下条件时，方程Ax=b有一个唯一解，其中x就是所求的线性方程组的解。该方法的时间复杂度为O(n^3)。

## 3.2 置换矩阵
用单位矩阵I和置换矩阵P替换变量，则 Ax=b可转化为Px=Pb, 其中P是一个矩阵，使得P(Ab)=[PbA]b=b。若[PbA]b=e，则得到Ax=b的一个解。置换矩阵P的一个重要应用是求解线性方程组的特征值及相应的特征向量。其时间复杂度为O(n^3)。

## 3.3 归纳法
假设线性方程组有k个不等式约束，其形式为 Ax<=b, x>=c, x_j≤d, j=1,2,...,k；以及n个等式约束，其形式为 Ax=b。通过对第j个不等式约束进行一些处理，就可以把线性方程组化为k个n元一次方程组的线性组合。因此，可以先处理前k个不等式约束，再处理剩下的n个等式约束，最后用合适的方式得到方程组的解。这种处理方式的时间复杂度为O((k+n)*n*log n)。
