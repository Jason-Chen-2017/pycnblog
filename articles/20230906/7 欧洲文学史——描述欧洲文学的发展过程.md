# 1.简介
  

## 1.1 欧洲文学概述
欧洲文学（欧陆）指西方（主要是北美、法国等主要区域）以及亚洲（主要是日本、韩国、中国、朝鲜、俄罗斯、印度等地区）广泛传播的各种经典、近现代、新奇、有创造性的作品。世界各地的人民从事艺术创作，拓宽视野，开辟思想路线，丰富精神生活，体验民族风情，获得文化上的认同感和归属感。在世界范围内，欧洲文学已经成为一种自娱自乐的艺术形式，并逐渐成为人们追求精神生活方式的重要文化基础。欧洲文学还成为许多国家教育中的重要内容，可以培养学生的文学素养、阅读能力和英语水平。欧洲文学的研究在全球范围内得到了极大的发展。欧洲文学的产生、发展及其所体现的艺术价值和社会影响，都深刻影响着世界的文化发展方向、社会制度安排及人类精神文明发展方向。
## 1.2 欧洲文学特点
欧洲文学具有以下显著特征：
1. 散文创作：欧洲文学以散文为主，散文艺术的创作历史可追溯到公元前5世纪，但一般认为这时以前的欧洲文学已经进入一个新的时期，而今天的欧洲文学则处于较为成熟的阶段。
2. 代表作品：欧洲文学有很多优秀的代表作品，如海伦娜·索福克勒的《窗边的小豆花》，查尔斯·阿尔弗雷德·杜瓦尔德的《海上钢琴师》，葛兰西·赛德尔巴赫的《吉他手灵光阔》等。
3. 时代特色：欧洲文学也呈现出不同时期的特点。如欧洲中世纪的诗歌，古希腊的青铜器时期，两次世界大战之后的魔幻现实主义，以及近现代的浪漫主义。
4. 文本数量：欧洲文学的作品数量占全球文学作品总量的三分之二左右，其中包括文学、散文、诗歌、戏剧等类型。
5. 多样性：欧洲文学不仅具有不同时期的特点，而且它还具有丰富多样的文本类型，包括保守派的诗歌、悲剧、浪漫、叙述等，以及英国文学的现实主义、寓言、冒险等。
## 1.3 欧洲文学分类与编目
### 1.3.1 风格分类
欧洲文学的风格分为两大类：
- 古典文学：源远流长的希腊罗马、古埃及、古埃塞俄比亚和古斯拉夫文明的文学作品；
- 当代文学：近现代文学、启蒙时期的古典文学，以及20至50年代的浪漫主义、魔幻现实主义、现实主义等。

### 1.3.2 主题分类
欧洲文学的主题分为7个层次：
1. 宗教、伦理与道德
2. 政治、社会、经济
3. 语言、文字、音乐
4. 美学、绘画、工艺
5. 科学、技术、工程
6. 乡土气味
7. 爱情、悲剧、婚姻、生物学。

### 1.3.3 结构分类
欧洲文学的结构分为4种：
1. 诗歌
2. 散文
3. 小说
4. 戏剧。

### 1.3.4 语种分类
欧洲文学的语言包括七种语言：希腊语、亚美尼亚语、法语、意大利语、拉丁语、阿拉伯语和荷兰语。
### 1.3.5 作者分类
欧洲文学作者按特征可分为以下几类：
1. 传统派作家：这类作者对希腊罗马及古埃及时代的文学作品有比较深入的研究，他们主要研究哲学、神话、伦理、风土、城邦制度等。如伏尔泰、安东尼·海瑟薇、庞加莱、福楼拜等。
2. 创新派作家：这类作者在早期欧洲文学创作中表现出独创性，通常涉及艺术、哲学、历史、人物、情节等领域，形成了一套新的观念和创作理论。如萨拉马卡、玛丽亚·马歇尔、伊恩·艾尔默、汤米·康顿等。
3. 新兴派作家：这类作者不断融合当代文学和古典文学的创新理论，重视艺术表现力、对时尚、文学理想的追求。如海伦娜·索福克勒、查尔斯·阿尔弗雷德·杜瓦尔德、爱迪生等。
4. 批判派作家：这类作者提倡批判性阅读，重视文本的分析和探讨，致力于通过对文本的反省和思考，提升认识和理解。如雅克·沃尔夫、曼德拉·莱博特、卢卡斯·毕加索等。
