# 1.简介
  

2017年初，有一款基于VR平台的新型人机交互产品“虚拟现实眼镜”成功商用，成为首个真正意义上的虚拟现实应用产品。同时，由于市场反馈良好、虚拟现实行业蓬勃发展，使得该领域的市场估值也越来越高。那么，目前虚拟现实市场的估值情况究竟如何？本文将从宏观、行业和技术三个方面详细讨论虚拟现实市场的估值情况。
# 2.市场概况
## 2.1 VR行业概览
虚拟现实（Virtual Reality，VR）作为一个近几年刚爆发的新兴产业领域，其主要的应用场景是人类在虚拟环境中进行各种沉浸式的活动，如赛车、游泳、冒险等。然而，这个新兴行业的火热并不意味着就此结束，因为市场上还有许多其它领域的产业也正在快速崛起，它们的共同特点就是利用电脑图形学技术、生物医学模拟和机器人技术来实现虚拟现实。

例如，基于计算机视觉和人工智能的医疗器械研发领域已经成为VR产业链的重要环节，它能够让患者在虚拟环境中获得身体接触、学习技能以及诊断疾病，从而降低了其医疗成本。另外，全球消费者都喜欢VR设备来进行网购、看电影和游戏。

总的来说，虚拟现实行业覆盖了人类最擅长但又耗费资源、有着特殊科技含量的活动——沉浸式的虚拟世界。通过采用VR技术，用户可以获得更好的沉浸感、自主性和体验。因此，VR行业是一个激动人心且迅速扩大的产业。

## 2.2 VR行业估值情况
目前，VR行业的估值情况分三种类型：超高估值、高估值、低估值。

### 2.2.1 超高估值公司
高盛、贝恩、微软、康奈尔、Facebook、腾讯……这些典型的高收入超级亿万富翁们的头衔都为"超高估值"。这一类的企业估值达到几千亿美元甚至上亿美元，不少人认为VR行业都是这些人经营的。而且，这些大亨们似乎还在不断扩张业务规模，逼使其他企业难以进入市场。比如，微软曾经宣称进入VR领域将会占据整个科技界的霸主地位。但是，随着时间的推移，微软却慢慢失去了市场份额。

可以预期，过去十几年里，VR行业的估值呈现超高估值的态势可能会持续下去，原因如下：

1. 硬件的不断升级换代，VR设备的性能和兼容性已远超过之前的数字图像屏幕。
2. 大量的创新项目涌现出来，虚拟现实硬件产品以各种方式呈现出前所未有的独特性。
3. 对虚拟现实技术的广泛使用，导致VR技术行业的研究和开发不断深化。
4. 当前，虚拟现实技术仍处于早期阶段，市场准确性和完整性尚不明朗。
5. VR行业还处在发展初期，往往受到互联网、实体经济等经济因素影响，投资回报率极低。因此，即便有超高估值的声音，也很难让企业付诸实施。

### 2.2.2 高估值公司
谷歌、Nike、优步、百度、华为、OPPO、VIVO、乐视……这些公司的估值则相对较高，通常以五六千亿美元左右。他们已经完成了大量的VR项目，积累了丰厚的经验。不过，这些公司的估值表现在某些方面可能略低于普通消费者的认知水平，可能出现公司盈利能力偏差的问题。

其中，Nike、VIVO、乐视和OPPO的估值情况最为突出。Nike近年来一直处于估值疯狂的状态，其估值表现明显高于普通人士的认知范围。实际上，Nike将其估值提高到了超过1300亿美元。另外，VIVO和乐视均是中国区头部厂商，估值均占到整个行业的半壁江山。

可以看到，高估值的状况并非永久性的。随着VR行业的发展，估值水平也会逐渐走低，直至消失或转为高估值。这既是由市场的变化造成的，也是由行业的发展所带来的必然结果。

### 2.2.3 低估值公司
国内外一些企业，如腾讯、索尼、世嘉、三星、杭州奇志……虽然在市场领先地位，但估值都不及高估值公司。

腾讯是最著名的VR游戏公司之一。其估值已经超过了上千亿元人民币。不过，其估值并不意味着腾讯将会马上成为千亿富翁。相反，腾讯的估值主要反映的是其作为一家游戏公司在市场中的声誉、影响力和竞争力。

索尼、世嘉、三星、杭州奇志等公司的估值表现较低，仍处于较小的水平。不过，它们仍然是一批硕果仅存的公司，并没有让人望而卫舍。总的来说，VR行业的估值水平仍然处于上升期。

# 3. 虚拟现实技术概述
## 3.1 什么是虚拟现实
虚拟现实（Virtual Reality，VR） 是一种使用电脑图形学技术和计算机视觉技术将真实世界融入数字空间的技术。它主要用于为用户提供一种新的沉浸式的、虚拟的、增强现实的三维真实世界。

虚拟现实最主要的应用场景是人类在虚拟环境中进行各种沉浸式的活动，如赛车、冒险、梦境等。通过这样的方法，VR能够提供真实世界的感受，用户可以在虚拟世界中感受到身体、灵魂、情绪等各个方面的波动，产生独特的体验。

## 3.2 虚拟现实技术体系
虚拟现实技术的整体体系可以分为四个层次。

- 第一层次是系统层。这一层包括硬件和软件系统，负责提供真实世界的渲染、传感与处理。
- 第二层次是内容层。这一层包括各种虚拟现实的内容，如人脸识别、虚拟模拟器、虚拟世界等。
- 第三层次是服务层。这一层包括云计算、AR/VR、VR/AR通信、互动娱乐、体育、医疗等相关服务。
- 第四层次是生态层。这一层包括VR产业链的发展，包括硬件制造商、软件开发商、制造商、服务商等参与其中。

当前，虚拟现实技术的研发落后于人类社会的发展，但不可否认的是，它的发展方向无可替代。以下是虚拟现实技术的一些关键领域：

- 显示技术。当前，虚拟现实技术需要借助数字显示器来呈现虚拟画面。显示技术的革命性进步促使VR成为了未来三维图像技术的一部分。
- 混合现实技术。VR的混合现实技术采用多种形式融合实时图像、虚拟图像、实体图像，让用户在虚拟世界中获得更丰富的真实感受。
- 交互技术。VR的交互技术是指让用户与虚拟环境进行有效通信的手段。现有的VR交互技术包括触摸屏、鼠标、控制器、光线追踪、耳机等。
- 人工智能技术。VR的AI技术旨在让虚拟现实世界中的对象具有更高的智能性。人工智能技术的发展将为虚拟现实的应用和创新提供了巨大帮助。
- 移动设备技术。随着智能手机、平板电脑等移动终端的普及，VR将带来移动互联网的革命性变革。

## 3.3 VR市场现状
截止2019年1月，VR市场已呈现出巨大的成长势头。根据IDC数据显示，2018年全球VR销售额达到10.34万亿美元，与2015年的10.28万亿美元相比增长了6%。同时，2018年VR市场的VR头部公司包括Facebook、OCULUS、Valve、HTC Vive、Samsung Gear VR、PlayStation VR等。

IDC数据显示，VR行业的规模超过了2万亿美元，并且还有很大的发展空间。按照IDC预测，到2025年VR市场规模将达到1.3万亿美元。VR领域还有大量的前景值得探索。

# 4. 虚拟现实市场估值分析
## 4.1 虚拟现实市场规模
根据IDC的数据显示，截至2019年1月，全球VR市场规模已经超过2万亿美元。其中，美国市场规模达到1.47万亿美元，占比超过90%；中国市场规模达到2295亿元人民币，约占全球VR市场的8%。可以说，中国市场依然占据主导地位。

IDC预计，2025年，全球VR市场规模将达到1.3万亿美元，同比增长超过30%。从市场规模看，VR将成为继互联网之后，第二大人口最密集的行业。其次，VR为消费者提供了许多新鲜事物，如游戏、AR/VR、体育、医疗等，而这些领域的平均收入水平要高于互联网和教育。

# 5. 结论
本文从宏观、行业和技术三个方面分析了虚拟现实市场的估值情况。我们发现，虚拟现实市场的估值情况分三种类型：超高估值、高估值、低估值。超高估值公司占据了绝大多数，但估值水平仍不容乐观。高估值公司也不少，不过，其市场的平均估值也较高。而低估值的公司估值则较低，低于普通人的认知水平。

因此，对于虚拟现实市场的估值情况，我们只能从宏观、行业和技术三个方面，多角度地分析。
