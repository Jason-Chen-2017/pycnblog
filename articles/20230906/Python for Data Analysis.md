# 2.安装配置
首先，下载安装Anaconda或者Miniconda，这是基于Python的一个开源发行版，包括了Python本身和一些常用的第三方模块。Anaconda的最新版本通常比Miniconda的稳定性更好，因此推荐安装Anaconda。
Anaconda包含三个包管理器conda、pip和ipython，这些工具可以让用户方便地安装、更新、卸载各种Python模块。打开命令提示符或终端，输入以下命令下载并安装Anaconda：
```
> cd ~/Downloads   # 切换到下载目录
> wget https://repo.anaconda.com/archive/Anaconda3-2020.07-Windows-x86_64.exe    # 下载最新版本Anaconda
>./Anaconda3-2020.07-Windows-x86_64.exe    # 安装Anaconda
```
安装后，运行“Anaconda Prompt”（Windows系统）或“Terminal”（Mac OS X或Linux），输入下列命令进行测试：
```
(base) C:\Users\Admin> conda --version     # 查看conda版本
conda 4.9.2
(base) C:\Users\Admin> python --version      # 查看python版本
Python 3.8.5
```
如果没有出现任何错误信息，则证明安装成功。
# 3.核心概念及术语
## 3.1 数据结构及特点
数据结构是计算机中的抽象概念，用来存储、组织和共享数据的方式。不同的数据结构以不同的方式组织数据，使得不同的访问和操作方式成为可能。目前，常用的有以下几种：
- 数组（Array）：按一定顺序排列的一组相同类型的元素。
- 链表（Linked List）：节点之间存在指针关系的线性集合。
- 栈（Stack）：只能在表尾进行插入和删除操作的线性集合。
- 队列（Queue）：只能在表头进行插入和删除操作的线性集合。
- 散列表（Hash Table）：根据关键码值直接进行访问的数据结构。
- 树（Tree）：由结点和边组成的有限集合。
- 图（Graph）：由结点和边组成的任意集合，一般用邻接矩阵表示。
数据结构根据其特点，又分为静态数据结构和动态数据结构。静态数据结构在创建之后就固定不变，而动态数据结构可以随着数据的增加、删减而变化。常见的静态数据结构有数组、链表；动态数据结构有栈、队列、哈希表、树、图。
## 3.2 Python 中的数据类型
Python 中共提供了六个基本的数据类型，分别是整数 int，浮点数 float，布尔值 bool，字符串 str，元组 tuple，空值 None。
- 整数 int 可以采用十进制、八进制或十六进制表示。
- 浮点数 float 以科学计数法或十进制形式表示。
- 布尔值 bool 只能取 True 或 False。
- 字符串 str 用单引号'或双引号 " 括起来，可以包含数字、字母、特殊字符等。
- 元组 tuple 是另一个不可变序列数据类型，类似于列表 list。不同之处在于元组一旦定义，不能修改。
- 空值 None 表示缺少有效值，在某些特定情况下会被赋予其他含义。
Python 支持的数据结构与上述对应关系如下：
- 整数、浮点数、布尔值对应的 Python 数据类型是 int、float 和 bool，分别代表整数、浮点数和逻辑真假。
- 字符串对应的 Python 数据类型是 str。
- 元组对应的 Python 数据类型是 tuple。
- 列表对应的 Python 数据类型是 list。
- 字典对应的 Python 数据类型是 dict。
- NoneType 的 Python 数据类型是 None。
