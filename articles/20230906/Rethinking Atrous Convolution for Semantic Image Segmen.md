# 1.简介
  

## 1.1 概述
近年来随着计算机视觉领域的火热，多尺度的特征学习方法在语义分割任务中的作用越来越受到关注。经典的基于密集预测的FCN模型仅考虑了图像的全局信息，忽略了不同尺度上的细节差异，导致其准确率很难满足需求。然而，通过引入膨胀卷积(atrous convolution)的方式可以提取不同尺度上的细节特征。本文中作者提出了一个新的基于膨胀卷积的语义分割方法——DilatedFCN(DCFNet)。DilatedFCN不仅可以对输入图像进行全局信息的学习，还可以有效地提取不同尺度上感兴趣的细节信息，因此具有更高的精度。
## 1.2 模型设计
DCFNet的网络结构由编码器(encoder)、中间层池化(pooling layer)和分类器(classifier)三部分组成。编码器是一个标准的卷积神经网络，包括多个卷积层、下采样层以及最终的池化层。其中每一层都采用膨胀卷积(dilated convolutions)，即在卷积核周围添加跳跃距离。中间层的池化操作则可以将空间尺寸减半，保留不同尺度的全局特征，也能够提取不同尺度上的局部特征。最后，分类器的输出则得到了不同尺度的像素级别的语义标签。

### 1.2.1 膨胀卷积
膨胀卷积(dilated convolutions)是一种模拟空洞卷积(sparse convolution)的手段。卷积核在边缘处缺乏重叠，这样就可以避免卷积后邻居之间的相关性。而膨胀卷积就是在卷积核的每个单元之间设置不同的步长，从而使得卷积核在边缘处具有重叠。图2描绘了膨胀卷积的原理。
图2：膨胀卷积示意图。
DCFNet用到了5个膨胀卷积层，第一个卷积层的卷积核大小为3×3，步长为1；第二个卷积层的卷积核大小为3×3，步长为2，第三个卷积层的卷积核大小为3×3，步长为4，第四个卷积层的卷积核大小为3×3，步长为8，第五个卷积层的卷积核大小为1×1，步长为16。通过这种方式，DCFNet可以在不同尺度上抽取图像的全局和局部特征。
### 1.2.2 上采样与裁剪
由于空间尺寸缩小，模型输出的尺寸也随之减少，因此需要将这些低分辨率的预测结果进行上采样。这里的上采样指的是将低分辨率的预测结果进行插值或反卷积，恢复至与输入图像相同的大小。除此之外，作者还采用了裁剪策略，即只使用预测结果的上下半部分，因为所需的训练样本较少，所以不必把所有训练样本都输入网络。
### 1.2.3 FCN
FCN(Fully Convolutional Networks)是最早提出的语义分割网络，它通过全卷积网络来学习不同尺度上的全局特征，但其只能对固定尺度的图像进行处理。而DCFNet对不同尺度的特征进行学习，既可以获取全局特征，又可以获取不同尺度的局部特征，因此具有更高的精度。
## 1.3 数据集介绍
作者选择的语义分割数据集为ADE20k。ADE20k包含20,000张高质量图片，共有150种类别，属于二分类问题，即图片中是否存在目标。作者将数据集划分为训练集、验证集、测试集。训练集用于训练模型参数，验证集用于选择最优模型，测试集用于评估模型效果。

## 1.4 实验结果
作者在ADE20K数据集上进行了实验。实验结果表明，DCFNet在精度方面优于其他最新模型。在相同的数据集上，作者的方法比其它最新模型取得了更好的性能。但是，由于计算资源限制，作者无法实现全部实验结果，而且实验结果存在一些随机性，因此读者可以参考自己的实验结果。
