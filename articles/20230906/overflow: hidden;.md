# 1.简介
  

由于 CSS 的盒模型的设计，元素的宽度和高度等样式属性都是可继承的。然而在实际业务场景中，往往希望某些元素具有更特殊的特性，比如溢出隐藏或溢出滚动条显示等。比如，对于导航栏、侧边栏等元素，可能需要将其溢出的部分隐藏，让页面整体看起来更干净。因此，CSS 中的 overflow 属性就派上了用场。
该属性用来设置元素内容溢出元素框时如何处理。可以取三个值：visible（默认值）、hidden 和 scroll。visible 表示元素的内容不会被裁剪，可以正常呈现；hidden 表示内容溢出元素框范围之外的内容会被裁减掉；scroll 表示内容溢出元素框时会出现滚动条。当设置一个元素的 overflow 属性为 hidden 时，如果这个元素的内容超过它的尺寸，则超出部分会被隐藏，不渲染到视图中，但可以通过滚动条进行查看。通常用于导航栏、侧边栏等元素。
另一个属性叫做 overflow-x 和 overflow-y，分别用来控制元素的水平方向和垂直方向的溢出处理方式。
# 2.基本概念和术语
- Box Model：CSS中的盒模型，指的是 CSS 布局的基石。它定义了一个矩形区域，包含元素内容、内边距、边框、外边距四个部分。
- Content：就是我们通常认为的“内容”，包括文本、图像、视频、音频等元素。这些元素是块级元素还是行内元素由 display 和 inline-block 来决定。例如<p>、<div>、<h1>……等都是块级元素，而<span>、<img>、<input>……等都是行内元素。
- Padding：内边距，也称为内衬，是内容与边框之间的空间，用来隔开内容与容器边界。Padding 是透明的，即使没有背景色，内部元素仍然可以看到。
- Margin：外边距，是两个相邻元素之间的距离。当两个元素边缘碰到一起时，根据外边距的大小，它们不会发生重叠，而是相互留下间隙。Margin 也是透明的。
- Borders：边框，用来围绕元素的内容与padding。border 可以设定不同的颜色、厚度、样式等。
- Height and Width：元素的高度和宽度属性，可以指定元素的总宽度或总高度，或者只指定其中一个。CSS3 中引入 min-height/max-height/min-width/max-width 等属性来限制元素的最小高度和最大高度，最小宽度和最大宽度。
- Inline Block Elements：行内块元素(inline-block)，可以将元素设置为内联元素，同时保留其高度和宽度的特征，并且可以设置 padding 和 margin。常见的应用场景如图标、按钮、输入框等。
- Overflow：该属性用来设置元素内容溢出元素框时如何处理，可取值为 visible、hidden 或 scroll。
- Positioning：CSS中定位机制能够精确地控制元素的位置。包括relative、fixed、absolute、sticky四种定位模式。
- Z-index：元素堆叠顺序，层叠上下文。Z-index 属性确定元素的堆叠顺序，通过调整 Z-index 的值可以让前面覆盖后的元素显示在上面。
