# 2.概念术语说明
## 2.1 情感分析简介
情感分析（Sentiment Analysis）是指对一段文字或者文本的情绪分析，判断其所呈现出的情感状态（正面、负面、中性）。通常情况下，情感分析是利用自然语言处理技术，对一组带有观点色彩的文本（如新闻、产品评论、影评等），自动地给出一个客观且有建设性的评价，而不仅仅局限于主观的感情表达。情感分析具有广泛的应用领域，如垃圾邮件过滤、产品推荐、影评褒贬、职场人事管理、金融投资辨别、舆情监控等。

## 2.2 相关术语及定义
- **语料库**：由大量的文本数据组成的集合，用于训练模型。
- **文本数据**：有关某种主题或领域内的文字、图片、视频等媒体形式的数据，通常以文档或句子的形式出现。
- **分类模型**：根据训练好的词汇表、分类规则以及文本特征，对输入的文本进行自动分类。常用的分类模型有朴素贝叶斯、支持向量机、决策树等。
- **特征工程**：对文本数据的有效提炼、转换、选择等过程，使得分类模型能够更好地进行文本数据的分类。
- **分类结果**：对输入的文本进行情感分类的结果，一般用正面、负面、中性三种类型表示。

## 2.3 项目目标
根据输入的电影评论文本，对其进行情感分析，得到两种类型的标签，分别代表正面、负面和中性的情感。实现以下功能：

1. 对输入的电影评论文本进行情感分析，输出正面、负面、中性三种类型的标签。
2. 在线测试接口，通过用户输入的电影评论文本进行实时情感分析。
3. 将本系统集成进个人社交网络平台，方便使用者对自己喜欢的电影进行评论并获得相应的情感反馈。

## 2.4 项目假设
- 本项目的文本数据源为用户上传的电影评论，输入文本可能含有明显的观点色彩，故需要准确、快速地进行情感分析。
- 使用的分类模型为朴素贝叶斯分类器，其优缺点已有多篇科研论文论述。此外，朴素贝叶斯分类器对于小规模的数据集效果较差，故本项目选择其他更适合电影评论情感分析的分类模型。
- 输入文本的长度一般在100～2000个字符之间。
