# 1.简介
  

“类属性”和“实例属性”都是面向对象编程中重要的概念，但它们之间存在着一些细微差别。如果你对这两个概念不是很了解，这篇文章将会帮你理清楚这两者之间的关系并给出具体应用场景。
# 2.概念定义
## 类属性（Class Attribute）
类属性属于类本身的属性，所有实例共享该属性，即使不创建任何实例也依然存在。类属性通常可以通过类的名称来访问，而不能通过实例来访问。类属性可以通过关键字`@classmethod`或`@staticmethod`装饰器进行修饰，或者直接在类中声明即可。
```python
class Employee:
    num_of_employees = 0
    
    @classmethod
    def print_num(cls):
        return cls.num_of_employees
    
    def __init__(self, name):
        self.name = name
        Employee.num_of_employees += 1
        
emp1 = Employee("Alice")
print(Employee.num_of_employees) # Output: 1
emp2 = Employee("Bob")
print(Employee.num_of_employees) # Output: 2
```
上述示例代码中，`num_of_employees`是一个类属性，所有实例共享这个属性。每创建一个新的实例，都会增加`num_of_employees`的值。`print_num()`方法是一个类方法，可以从类本身而不是实例访问类属性。由于这个原因，它可以访问到类似“全局”变量的功能。此外，也可以在类外创建类属性：
```python
class MyClass:
    some_attr = "Hello World"
    
def my_func():
    print(MyClass.some_attr)

my_func()   # Output: Hello World
```
这里`MyClass`是类，`some_attr`是类属性。类属性可以在类内部、外部或类的方法中访问。
## 实例属性（Instance Attribute）
实例属性属于类的实例化对象，不同实例拥有不同的实例属性值。实例属性可以通过实例来访问，并且可以通过实例的`__dict__`字典来获取。实例属性可以通过实例的`self`参数来访问，也可以通过对象名加属性名的方式访问。实例属性可以被修改、添加或删除。
```python
class Employee:
    pass

emp1 = Employee()
emp1.name = "Alice"
emp2 = Employee()
emp2.name = "Bob"

print(emp1.name)    # Output: Alice
print(emp2.name)    # Output: Bob
```
这里，`name`是实例属性，不同实例拥有不同的`name`属性值。通过`__dict__`字典可以看到每个实例的属性和值的对应关系。通过对象名加属性名的方式也可以访问属性，如`emp1.name`。可以设置、修改、删除实例属性，如下所示：
```python
emp1.age = 25
del emp2.name

print(emp1.__dict__)      # Output: {'name': 'Alice', 'age': 25}
print(emp2.__dict__)      # Output: {}
```
这里，`age`是新添加的一个实例属性；`del emp2.name`语句用来删除实例`emp2`的`name`属性。
