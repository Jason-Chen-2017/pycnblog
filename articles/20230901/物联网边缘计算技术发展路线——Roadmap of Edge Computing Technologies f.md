# 1.简介
  

随着物联网技术的飞速发展、智能设备数量的激增、海量数据处理能力的提升、物联网终端设备的异构化、应用场景的多样性等因素的影响，物联网边缘计算（Edge computing）技术正在成为当今智能系统的重要组成部分。而边缘计算技术的发展也给予了人们越来越多的机遇和挑战。本文将梳理当前物联网边缘计算技术的主要研究方向、应用案例、架构模式、技术方案、前沿研究进展、未来的发展方向、关键技术突破点，并对这些方向展开更加详细地阐述。

# 2.基本概念、术语
## （1）什么是物联网边缘计算？
物联网边缘计算（英文：Edge computing）是一种基于云端的、在本地端进行计算的技术。其定义中“边”指的是靠近设备的数据或计算资源，从而降低网络流量、减少响应延迟、提高数据处理速度、节省带宽。可以简单理解为，物联网边缘计算是将核心计算任务从传统服务器向具有实时响应能力的移动设备上进行迁移，进一步实现对传感器、摄像头、机器人的高效率收集和分析。

## （2）物联网边缘计算的定义
- 物联网边缘计算是一个利用小型计算单元（如手机、平板电脑、嵌入式系统、消费类电子产品）进行实时的计算和数据的处理，并且部署在一个物理边界上，以最小化网络流量和处理时间，对业务的快速响应。
- 通过利用网状拓扑结构中的物理信息和边缘计算中心之间的通信，物联网边缘计算实现从源头到终点的数据传输，确保实时、精准的业务决策。
- 将应用程序部署在边缘设备上，能够以极低的延迟（毫秒级甚至微秒级）完成复杂运算和数据的处理，有效避免数据中心网络带来的延迟影响。
- 可以通过部署在边缘设备上的应用程序对物理世界的对象进行监控、检测、控制，及时响应本地事件，并提供即时反馈和预警机制。
- 物联网边缘计算体系架构采用分布式云计算、云平台、服务器less架构及其他相关技术，是物联网系统的重要组成部分。

## （3）物联网边缘计算的分类
根据物联网边缘计算的特点、目标、作用领域和技术手段，可以将物联网边缘计算分为以下几类：

1. 数据处理：包括数据的采集、存储、分析、传输、交换、过滤等功能。由于实时性要求高、带宽要求不高、CPU性能要求低，所以常用于边缘设备上。
2. 机器学习：涉及大数据分析、计算机视觉、自然语言处理等技术，对物联网边缘计算非常重要。
3. 实时协同决策：利用物联网设备的上下文信息进行智能决策，可进行连续的协调式管理，满足用户需求。例如，车辆的实时监控、远程维护、智能照明控制等。
4. 服务计算：服务计算是物联网边缘计算的一个分支领域，通过云计算资源进行离线批量处理，对远距离数据进行分析、建模，并实时产生结果反馈给用户。例如，流量预测、电力规划、风险评估等。
5. 智能运维：该领域解决的问题是在物联网设备之间如何安全、可靠、可信、快速地交付服务。它可以优化服务质量，通过自动化监控、预测、诊断，快速发现和诊断故障。
6. 可穿戴计算：该领域的目的是打造物联网设备背后可感知、可交互的个人计算设备。例如，眼镜中的AI助理。
7. 边缘计算平台：这类技术解决的是物联网边缘计算的基础设施建设，包括边缘节点的部署、资源分配、控制、安全、管理、部署等。
