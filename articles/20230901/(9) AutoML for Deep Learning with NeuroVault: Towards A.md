# 1.简介
  

传统的方法对于大型数据集往往效率低下且不易维护，AutoML是一种机器学习领域的最新研究方向。基于深度学习神经网络的自动化设计方法已经取得了显著进步，包括超参数优化、模型压缩等。然而，现有的AutoML方法仅适用于一些特定任务或特定的数据集。因此，如何为某些应用领域找到更加通用的CNN体系结构仍然是一个难题。NeuroVault是一种web-based工具，可提供可视化、注释和下载大量医疗影像数据的神经影像数据库。在本文中，我们将介绍NeuroVault如何结合AutoML方法来自动发现和选择多样化的CNN体系结构，用于计算机视觉和医疗图像领域的深度学习任务。我们将提出NeuroVault的AutoML方法并讨论其优缺点。
# 2.相关工作
目前，大多数的AutoML方法都基于规则或启发式的算法，即用户指定目标（如高精度或快速），然后系统从众多候选模型中搜索最佳模型。由于训练时间的限制，这些方法通常只适用于特定任务或数据集。例如，本文的目标是在不牺牲准确性的情况下，自动发现适用于手术切除或病理诊断的CNN体系结构。我们将考虑以下两个相似但又不同的AutoML方法：

1）Evolutionary Algorithms (EA): EA是遗传算法的一种变体，它利用基因变异、交叉、重组等操作来生成新的模型。它的目标函数可以定义为训练误差的期望值，但没有考虑到模型之间的复杂度。相应地，EA需要耗费大量的时间来搜索各个模型的配置。

2）Hyperparameter Optimization (HO): HO是在已知参数空间内搜索最佳超参数的过程。在训练阶段，它会计算所有可能的参数组合的代价，以选择具有最小代价的组合。HO也可以采用不同的策略来优化不同层的超参数。但是，它缺乏对模型间复杂度的考虑，无法有效应对深度神经网络中的各种交互作用。

综上所述，我们提出了两种方法，NeuroVault的AutoML方法及其特点。首先，NeuroVault是一个开源、易于使用的web服务，提供了可视化、注释和下载大量医疗影像数据的神经影像数据库。其次，NeuroVault的AutoML方法采用基于强化学习（RL）的强化学习算法。它利用强化学习机制探索各种模型架构的组合，同时兼顾预测性能和效率。第三，为了减少搜索空间并提高效率，我们提出了一个可插拔的模型集合，其中包含一系列有代表性的模型架构。第四，除了图像分类之外，我们还测试了AutoML方法在肿瘤分割和疾病检测上的性能，结果表明其能够发现适用于这两类任务的高效的模型架构。最后，我们通过评估AutoML方法的实际效率和预测性能来验证NeuroVault的AutoML方法。

# 4.概要
本文首先介绍了NeuroVault——一个web服务，提供可视化、注释和下载大量医疗影像数据的神经影像数据库。然后，我们详细阐述了NeuroVault的AutoML方法，主要有：

1. 模型搜索空间的可插拔化：我们设计了一套可插拔的模型集合，其中包含多种有代表性的模型架构。我们希望这个模型集合能够覆盖不同的深度学习任务、数据集、输入尺寸和其他特征。

2. 基于强化学习的模型设计：基于强化学习的模型设计算法，可以帮助自动发现具有最高预测性能和效率的模型架构。我们设计了一个RL框架，包括状态表示、动作空间、奖励函数和模仿学习。通过RL算法生成的模型都经过训练和评估，最后根据其性能排序。

3. 效果评估：为了确定模型架构的预测性能，我们进行了多项实验。我们的实验结果表明，NeuroVault的AutoML方法能够发现和选择各种CNN体系结构，用于医疗图像领域的深度学习任务，其效果均超过了传统方法。

最后，我们总结了NeuroVault的AutoML方法的优点和局限性，并展望了其未来的发展方向。
