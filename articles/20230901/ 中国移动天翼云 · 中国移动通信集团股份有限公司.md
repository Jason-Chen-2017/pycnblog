# 1.简介
  

中国移动天翼云（CMCC）是由中国移动通信集团（CMCC）所主办的网络基础设施服务商。其业务主要包括数据中心租用、综合业务平台（含会议、视频会议、云计算等产品及服务）、通信设备采购及售后服务、网络质量保障及通信内容创新等。

随着移动互联网、云计算、人工智能等领域的高速发展，电信运营商不断寻求创新的方式来提升用户体验、降低成本和节省运营成本。当前，中国移动正在构建一个数字化的、去中心化的、智能化的、协同化的系统，这种架构也将成为网络基础设施服务商的重要发展方向。

中移动希望借助互联网、云计算等科技革命带来的强劲势头，打造数字化、智能化、协同化的网络基础设施服务平台，为客户提供全面的、专业的网络解决方案。同时，借助互联网信息技术的传播力，帮助运营商实现信息共享、整合、交流、价值互通，从而在信息化、社会化的基础上，实现运营商与客户之间的连接和共赢。

中国移动天翼云是一个完整的互联网+通信系统，覆盖了以下功能模块：

- 数据中心租用：CMCC通过云服务平台和物理机房资源池，向合作伙伴提供可靠、安全、高性能的数据中心租用服务。
- 综合业务平台：CMCC的综合业务平台是基于“云 + 商”模式的网络基础设施服务，其优点包括降低运营成本、缩短网络建设周期、优化网络管理效率。CMCC的综合业务平台包括会议、视频会议、网页、域名、服务器租用、云计算、存储、负载均衡、防火墙、VPN、内容创新等产品及服务。
- 通信设备采购及售后服务：通过统一的采购渠道、专业设备提供商、数据分析、售后支持等机制，CMCC可以帮助客户快速、高效地采购通信设备，并提供专业的售后服务，确保通信设备的安全、稳定运行。
- 网络质量保障及通信内容创新：通过提升系统自身的能力，促进信息交换、沟通、学习的效率，CMCC可以对客户提供全面、细致的网络质量监控服务；通过开放性的数据接口，CMCC可以与第三方开发者分享通信数据，提升用户获取信息的效率；通过创新应用，CMCC可以在平台上实现数字内容的创新，提供具有商业价值的数字服务。

# 2.基本概念术语说明

2.1 CMCC
CMCC是指中国移动通信集团股份有限公司，是中国移动网络基础设施服务商之一，隶属于中国移动集团公司下属的子公司，总部位于深圳，主要为中国移动运营商提供移动网络通信以及相关业务的基础设施服务。

2.2 服务商
服务商（Subcontractor），是指由运营商委托运营商管理或运营的个人或企业，可直接使用运营商网络资源进行网络基础设施项目的实施或实施的合作伙伴。服务商可委托运营商提供设备工程安装、技术咨询、软件维护、设备维修、提供设备服务，提供专业培训、咨询、支持等服务，但不可对设备信息或隐私进行任何侵入。

2.3 小型站点
小型站点（Small Cell Site），又称为“蜂窝节点”，是中国移动提供给服务商租用的网络接入点，通常包含一个或多个基站、微波发射器和相应的电路板卡，能够承载固定数量的终端设备连接，能够满足用户弱交互要求的需求，并可通过自有的管理系统进行数据收集和处理。

2.4 物理站点
物理站点（Physical Site），是中国移动为合作伙伴租用的、覆盖整个城市或区域的覆盖网，由运营商统一规划，为服务商提供基础设施项目支持的区域，由运营商所有，可连接运营商所有用户终端设备，设备数量因运营商而异。

2.5 分支机构
分支机构（Branch），是指运营商的独立运营实体，既不是服务商的直连单位，也不是服务商的分支单位。分支机构可以通过运营商提供的网络基础设施项目获得一定的收益。

2.6 会议（Conferences）
中国移动的会议服务由世界顶级会议服务商杭州电子科技大学提供，其有超过90%的用户采用中国移动会议服务。

2.7 会员（Memberships）
会员（Memberships），即付费会员制度，是中国移动推出的一项会员制度，需要缴纳年费、月费或季费。

# 3.核心算法原理和具体操作步骤以及数学公式讲解
3.1 数据中心租用模型

CMCC的云服务平台提供了几种数据中心租用模型。首先是低配模型，适用于较小规模的合作伙伴，根据合作伙伴的需求，CMCC按每月的租金收取标准报价，然后使用CMCC的云服务平台快速申请到指定类型的低配服务器。其次是中配模型，适用于中型规模的合作伙伴，在低配模型的基础上，添加高性能CPU、更大的内存容量和更多的网络带宽，让服务商的工作负荷得到最佳发挥。再者，高配模型，适用于大型规模的合作伙伴，可以提供更好的服务质量，配置更加优秀的硬件，同时还可利用CMCC提供的商业级云服务，让服务商享受到一些独具匠心的优惠政策。最后，超高配模型，是针对那些非常苛刻的客户，可在高配模型的基础上，增加极端高性能的GPU计算资源、1T的本地存储、超高速网络、百万并发连接等。

CMCC的云服务平台允许合作伙伴预留实例，即申请虚拟机或者裸金属服务器等资源。当发生资源突发时，CMCC的云服务平台能够快速弹性扩充。另外，CMCC的云服务平台还提供价格预测工具，让合作伙伴能够快速预估出租期内的租金收取情况，并结合现有的账单信息进行账单精算。

3.2 设备采购流程

当合作伙伴需要购买设备时，可以选择线下采购、联谊采购或线上采购。前两者一般会涉及多次交易，中间可能要协调多个部门合作才能顺利完成，而线上采购则可以减少中间环节，把采购过程中的复杂的手续办理和差错风险都转移到CMCC的平台上。

CMCC的设备采购流程是先向服务商说明自己的需求，然后与运营商进行联系，确认设备的类型、数量、品牌和配置，然后合作伙伴可以选择指定经销商和渠道进行采购。CMCC会将采购信息提供给合作伙伴，同时合作伙伴也可以选择自己购买。

采购完成后，CMCC会及时通知服务商，并安排人员前往运营商确认采购商品是否符合服务商的要求，然后给予合作伙伴物料的接收，并与运营商签订正式采购协议。完成正式采购后，CMCC会将采购的设备信息发送给合作伙伴，并为合作伙伴开通网络服务授权，等待合作伙伴开通设备并租用服务。

3.3 会议系统搭建

中国移动的会议系统通过世界顶级会议服务商——杭州电子科技大学为合作伙伴提供。杭州电子科技大学是全球领先的国际会议中心，拥有一支核心的会议技术团队，以“终端互联网、协同云计算、移动计算、人工智能”为核心技术创新，可提供跨界融合、可持续发展的会议解决方案。

对于CMCC来说，杭州电子科技大学的会议系统为合作伙伴提供了可靠、安全、快速的会议解决方案。为了方便合作伙伴与会议参与者之间沟通、交流、分享，CMCC的会议系统采用知识库+智慧会议的方式，系统地记录与会者的需求、理解、喜好、疑问等，通过知识图谱进行内容匹配，智能分析会议主题，呈现最吸引人的会议内容。

通过杭州电子科技大学的会议系统，CMCC能够提升用户参与会议的体验、感受，为合作伙伴提供更高质量的会议服务。
