# 1.简介
  

“软实力”这个词最近几年逐渐成为程序员的一个热门话题。从网上引起的一系列争论不无道理。也有部分程序员认为这个词过于模糊、混乱甚至嘲讽一些。那么，什么样的能力或品质是我们需要具备的“软实力”呢？笔者认为，要构建一个真正具备“软实力”的程序员，首先需要达到以下五个层次：

1. 领域知识积累
作为一名程序员，首先需要对自己的专业领域有全面的认识和掌握。否则将无法开发出高效率的解决方案，更别说创造价值了。如同一座建筑工程，只有基础结构设计、施工规范制定、系统集成、施工监控等方面相当复杂的专业知识才能保证项目的顺利完工，并得到预期收益。

2. 技术知识广博
程序员除了需要领域知识外，还需要具有丰富的技术知识。因为程序运行在电脑上，因此我们需要了解各种编程语言的语法和实现原理。当然，也可以通过阅读开源代码学习一些高级技巧。

3. 沟通表达能力强
作为一个程序员，需要具有良好的沟通表达能力。软件开发涉及多个人员协作，所以我们必须善于和他人沟通交流，把自己的想法和观点表述清楚，才能让团队中的其他成员理解和接受。这一点尤其重要，因为有些时候程序员之间的交流往往比编程本身难得多。

4. 执行力强
程序员除了具备技艺之外，还需要有执行力。如果你的工作没有按时交付，或者自己解决某个问题的时间花费太久，都会招致大家的批评和指责。为了防止这种情况发生，我们需要建立长远的技术路线图，并坚持执行。一旦技术路线图设立清晰，并且自觉遵守，就可以在短时间内抓住大方向，快速取得进展。

5. 创新精神
最后，程序员还需要拥有独特的创新精神。创新，既是对已有技术的改进，也是对未知领域的探索。只有充分发挥个人的创意才能超越竞争对手，最终赢得市场的青睐。
综合起来，“软实力”就是要锻炼出一个擅长“领域问题、技术问题、沟通问题、执行问题、创新问题”的程序员。如果程序员能够全面地运用自己的软实力，他们可以实现自己的梦想，为客户创造价值。
