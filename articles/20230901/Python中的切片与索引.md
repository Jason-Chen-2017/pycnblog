# 1.简介
  

Python中，列表、元组和字符串都可以进行索引，而字典不可以。所以在索引的时候，要注意一下区别。对于字符串来说，我们可以使用索引来访问单个字符，而对于列表、元组、字典等序列数据类型，则可以通过索引或者切片的方式来访问某个元素或某一段子序列。那么这两种访问方式分别是什么呢？

# 2.基本概念
## 2.1 序列（Sequence）
在Python中，可以把一个容器类数据结构称为序列。比如：字符串、列表、元组、集合、字典都是序列的例子。一般来说，序列有以下几个特点：

1. 有序性：序列中的元素是按照一定顺序排列的。
2. 可变性：序列中的元素的值可以改变。
3. 成员性：可以用成员运算符“in”来判断某个元素是否属于该序列。

## 2.2 索引
索引指的是通过位置(位置值)获取元素的过程。Python中序列的索引从0开始。如果序列只有一个元素，其索引为0；如果序列有多个元素，索引就是每个元素的位置。比如，字符串"hello world"的索引为0到11，分别对应着'h', 'e', 'l', 'l', 'o','', 'w', 'o', 'r', 'l', 'd'.如下图所示：


在索引中，左侧表示序列的第一个元素的位置，右侧表示最后一个元素的位置加1。即len(sequence)-1。而当负数索引时，表示从右往左数的元素的位置。因此-1表示最后一个元素的位置，-2表示倒数第二个元素的位置，依此类推。

举例：假如有一个字符串s='hello world',那么s[0]就代表第一个字符'h', s[-1]代表最后一个字符'd', s[:5]代表前五个字符'hello'，s[6:]代表后七个字符'world'。

## 2.3 切片
切片是指从序列中取出一部分元素的过程。通过切片，可以将序列中的某一段元素提取出来，然后赋值给另一个变量。切片使用冒号(:)，并有三个参数:start，stop，step。其中，start和stop是切片的起止位置，step是切片的步长。如果省略掉start，默认是0；如果省略掉stop，默认是序列的长度；如果省略掉step，默认为1。当step为正数时，从左至右提取元素；当step为负数时，从右至左提取元素。

举例：假如有一个列表L=[1,2,3,4,5],那么L[::2]代表取所有偶数的子序列，得到结果[1, 3, 5]。L[::-1]代表逆序取整个列表，得到结果[5, 4, 3, 2, 1]。

## 2.4 深拷贝与浅拷贝
计算机内存中的数据是分块存储的。每块数据都有自己的地址，当程序对这个地址进行操作的时候，就会修改实际的数据。但当变量和它指向的对象被赋予新的值时，由于只是改变了变量本身的引用，而没有新建对象，所以会影响之前的数据。为了解决这一问题，需要不同的复制方法，分为浅拷贝和深拷贝。

### 浅拷贝
浅拷贝又称为浅复制，创建新对象并复制原始对象的引用，也就是说，对象的不同之处只表现在地址上。浅拷贝只会复制对象的第一层，如果里面还有对象，只会复制它的引用，不会递归地复制所有子对象的引用。浅拷贝在赋值时总是会创建新的对象，而不会修改现有的对象。

### 深拷贝
深拷贝又称为深复制，创建新对象并递归地复制所有的子对象。这种方式下，每一个新对象的修改都不会影响其他对象。这种方式在某些情况下效率比较高。但是，它也需要更多的内存空间，因为它会创建完整的副本。

# 3. 代码示例
```python
# 1.字符串的索引
string = "hello world"
print("string[0]:", string[0]) # h
print("string[-1]:", string[-1]) # d
print("string[:5]:", string[:5]) # hello
print("string[6:]:", string[6:]) # world

# 2.列表、元组的索引
list1 = [1, 2, 3, 4, 5]
tuple1 = (1, 2, 3, 4, 5)
print("list1[0]:", list1[0]) # 1
print("tuple1[0]:", tuple1[0]) # 1
print("list1[-1]:", list1[-1]) # 5
print("tuple1[-1]:", tuple1[-1]) # 5
print("list1[::2]:", list1[::2]) # [1, 3, 5]
print("tuple1[::2]:", tuple1[::2]) # (1, 3, 5)

# 3.字典的索引
dict1 = {'name': 'Alice', 'age': 20}
print("dict1['name']: ", dict1['name']) # Alice
print("dict1['age']: ", dict1['age']) # 20
```
