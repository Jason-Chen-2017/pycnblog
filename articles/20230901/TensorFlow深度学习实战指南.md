# 2. 机器学习基础
## 2.1 什么是机器学习？
机器学习(ML)是一类人工智能技术，是指让计算机学习从数据中提取知识、改善性能的方法。机器学习利用海量的数据来训练出能够预测新数据的模型，从而对现实世界的问题进行分析和决策。传统的统计方法和模式识别算法只能在有限的样本数据上进行，而机器学习方法可以从无限的、海量的数据中学习，因此能够从数据中发现隐藏的规律。

### 2.1.1 为什么需要机器学习？
在过去的几十年里，科技飞速发展，信息技术高度普及。人类在这个过程中创造了大量的数据，如音乐、视频、图像、文本、视频游戏等。如何从这些数据中挖掘信息、洞察潜藏，成为每个人都关心的话题。机器学习就是为了解决这一难题而诞生的。通过机器学习，我们可以训练出能够预测新数据的模型，从而对现实世界的问题进行分析和决策。

1. **解决“学习”过程中的困难**：传统的统计方法和模式识别算法只能在有限的样本数据上进行，无法从无限的、海量的数据中学习，因此难以满足需求；

2. **自动处理大量数据**：通过机器学习方法可以从无限的、海量的数据中学习，因此处理大量数据时，可以节省大量的时间和资源；

3. **准确预测新数据**：机器学习方法可以通过训练集数据来学习，从而在测试集数据上进行预测，准确率非常高。

总之，机器学习可以自动从数据中学习，提升我们的分析和决策能力。

## 2.2 机器学习的类型
机器学习分为三种类型：监督学习、非监督学习、强化学习。

1. **监督学习(Supervised Learning)**：又称为有监督学习。监督学习是一种在训练集上给出输入输出标签的学习方式，目的是找到一种映射函数f，将输入x映射到输出y。监督学习由输入数据X和输出数据Y组成，假设函数h和损失函数L确定了学习问题。学习目标是求解参数theta，使得在输入x处，输出y最可能的概率最大。典型的监督学习任务包括分类、回归、序列标注、推荐系统等。例如，垃圾邮件过滤器、图片分类、病情诊断、股票预测等。


2. **非监督学习(Unsupervised Learning)**：又称为无监督学习。无监督学习是一种没有给定正确输出标记的学习方式，目的是对数据进行聚类、关联、降维等可视化操作。典型的无监督学习任务包括聚类、降维、异常检测、推荐系统等。例如，图像分割、电影推荐系统、文档主题模型等。


3. **强化学习(Reinforcement Learning)**：它在训练过程中通过获得奖励和惩罚来探索新的行为策略，能够让智能体学会如何更好地采取行动。典型的强化学习任务包括游戏AI、自动驾驶汽车、虚拟现实、智能电网管理等。
