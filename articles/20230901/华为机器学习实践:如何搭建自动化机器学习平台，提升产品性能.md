# 1.简介
  

随着科技革命的深入，物联网、云计算等新型技术带来了巨大的商业变革和产业变迁。而这项变革对人类的生活也产生了深远影响。其中最重要的就是人工智能（Artificial Intelligence）的应用。人工智能技术包括计算机视觉、自然语言处理、语音识别、数据挖掘、无人驾驶等。人工智能在各行各业都有着广泛的应用，如自动驾驶、智能客服、智能交通、图像识别、垃圾分类、保险理赔等。但是应用到实际生产环境中，往往面临着以下三个难点：

1. 数据量不足：人工智能模型需要大量的数据进行训练，但是现实世界中的数据收集工作非常繁琐，缺乏有效的方法。尤其是在医疗保健领域，病人的记录通常采用问卷调查的方式，记录的内容非常零碎，而且没有统一的标准化系统。因此，如何从多个源头汇总得到海量数据的同时又确保数据的质量是一个极具挑战性的问题。

2. 模型准确率低下：人工智能模型通过大量的数据训练后，可以生成出精确的预测结果，但并不能保证模型的实际效果。因为现实世界中的情况千差万别，模型所做出的判断常常是有偏差的。比如，当检测一张图片上是否包含某种动物时，模型可能就将它判定为“非该种动物”。所以，如何根据数据及时调整模型的参数，提高模型的准确率成为当前机器学习研究者的共同关注事项。

3. 系统部署困难：人工智能模型的构建、训练、优化、验证等流程相当复杂，且涉及众多技术人员协作，部署成本较高。而传统的软件开发流程简单、易于理解。如何结合二者优势，实现一个全面的、高度自动化的机器学习平台，提供给用户快速、便捷地完成任务成为机器学习行业的研究热点。

基于上述三个难点，华为基于深度学习框架PaddlePaddle搭建了一个自动化机器学习平台，能够很好地解决以上三个难点。下面，我们详细介绍华为机器学习平台的搭建过程。

# 2.基本概念术语说明
## 2.1 PaddlePaddle
深度学习框架PaddlePaddle是华为开源的一个支持广泛的机器学习技术，包括图像分类、图像检索、文本分类、序列标注、强化学习等，并且提供统一的API接口，允许用户使用Python、C++或者Java进行编程。PaddlePaddle支持GPU和CPU两种运算模式，在海量数据上的分布式计算能力也十分优秀。当前，PaddlePaddle已经开源，并获得了社区的广泛关注。

## 2.2 概念
机器学习（英语：Machine learning）是指利用计算机来改善系统的性能或进行预测分析的一门技术。机器学习的主要特点是：计算机系统可以从经验E中学习到一般性规律，并依据此规律对未知数据进行预测。它是人工智能（AI）的一个分支，也是统计学、模式识别、数据挖掘、 computer science、数理统计和信息论等多个领域交叉的基础。机器学习算法的目的在于学习输入数据的模式，进而对未知数据进行预测和决策。

机器学习方法可大致划分为三类：监督学习、无监督学习、半监督学习。

1. 监督学习：监督学习是指由输入、输出对组成的数据集进行学习，即学习输入数据对应的正确输出。典型的监督学习包括分类、回归、聚类等，通过已知数据，利用机器学习算法对未知数据进行预测或分类。

2. 无监督学习：无监督学习是指对数据进行分析，而不给予任何标签信息，仅靠自身的特征进行数据聚类、分析、结构学习等。典型的无监督学习包括聚类、密度估计、关联分析等。

3. 半监督学习：半监督学习是指存在部分标记数据的场景，其中一部分数据被标记，另一部分数据未被标记，此时，可以使用部分标记数据的辅助来预测缺少标记数据的标记。典型的半监督学习方法包括堆叠式降维、最小知识点惩罚、软标签学习等。

## 2.3 概念
深度学习（Deep Learning）是指用多层感知机 neural network 的方式，对输入数据进行复杂的映射，并学习数据的内在规则。深度学习的发展促使神经网络从单层神经元，扩展到多层、甚至多层以上的神经网络，逐渐形成了一套完整的学习系统。深度学习是机器学习的一个分支，是取得突破性进展的关键因素之一。

深度学习技术是指通过多层次神经网络，在数据中发现隐藏的模式，以期达到对数据的建模和分析，并且进行预测分析。它的基本思想是，把原始输入信号转换为多层抽象特征，通过非线性函数的组合，得到最后的输出预测值。深度学习技术被广泛应用于图像识别、自然语言处理、生物信息学、医疗卫生、金融和经济领域。目前，深度学习在人工智能领域占据重要地位，取得了良好的成果。

## 2.4 概念
自动机器学习（AutoML） 是指通过自动化的算法搜索、超参数配置、神经网络架构设计、数据清洗、并行化等策略，对数据进行自动化的处理，并最终生成高性能的机器学习模型。自动机器学习旨在通过自动化的方法来寻找数据与模型之间的契合点，来优化数据分析的效率，并帮助企业在更短的时间内，取得更好的业务性能。
