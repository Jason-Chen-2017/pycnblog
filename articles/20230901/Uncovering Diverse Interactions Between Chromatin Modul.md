# 1.简介
  

ChIP-seq技术是对细胞免疫系统进行表观遗传修饰的最重要的手段之一。随着单核细胞自动机（single cell automation）的发展，测序数据量不断增加。如何将单细胞数据应用于ChIP-seq基因组功能分析变得越来越重要。

在本次研究中，我们搭建了一个新的数据集，它包含多个单细胞数据类型，包括ATAC-seq、DNase-seq、RNA-seq和CNV-seq等。通过使用深度学习方法，我们可以揭示其中的差异化调控作用。这些差异化调控包括染色质调节子（Chromatin Modulators），它们能够影响染色质结构和结合蛋白的表达。

针对这个数据集，我们提出了一种新的深度学习模型——Multi-Omics Attentive Network (MOAN)，它能够同时利用多种omic数据以提升预测精度。首先，我们利用Graph Convolutional Networks (GCN)建模染色质的连接网络，该网络能够捕获序列上下游关系信息。然后，我们将这些特征输入到Attention Networks (ANN)模块，该模块能够捕获染色质调节子在不同分子标记物上的作用。最后，我们将输出的表示传入一个全连接层中，该层能够计算出每个染色质调节子对某个基因的响应。

除此之外，我们还对模型进行了训练、验证和测试，并达到了与其他基线比较好的结果。在实验过程中，我们发现MOAN能够更好地识别基因之间的差异化调控作用，并挖掘潜在的治疗靶点。因此，我们认为本文的工作具有独特的理论和实践意义。


# 2.相关工作
在ChIP-seq技术的发展历史上，已经有许多研究基于染色质微结构、表达模式以及调控目标进行了相关的研究。由于染色质结构、转录本质以及生物体内DNA结合状态的复杂性，无法直接从单个细胞获得这些信息。目前，已经有很多相关研究致力于使用机器学习技术来识别染色质调节子对基因的影响。然而，大多数研究都仅限于用一个或少数几个omic数据作为输入，忽略了多个omic数据的互补影响。

目前，比较著名的一些深度学习模型有GRU、LSTM和CNN。虽然这些模型对于序列数据的处理能力较强，但是它们只能捕获单个omic数据对序列的局部影响，缺乏全局视角。另外，还有一些工作试图使用向量空间模型来描述染色质调节子之间的相互作用。但是这些方法存在可扩展性问题，并且对调控效应的敏感度很弱。

受到上述工作的启发，我们建立了MOAN模型来融合多个omic数据，以提高基因表达差异化调控的预测能力。首先，我们提出了一种新的注意力机制来捕获多个omic数据之间的协同作用，其中包括序列上下游关系和染色质调节子的不同作用。其次，我们采用GCN来建模染色质结构的连接网络，该网络能够捕获到底什么样的序列区域更能影响基因表达。最后，我们采用ANN模块来预测每个染色质调节子对基因的响应，并通过一个全连接层计算得到最终的输出。

此外，我们还证明了MOAN的有效性，通过对实际数据集的实验和评估，我们发现MOAN可以更好地识别基因之间的差异化调控作用，并挖掘潜在的治疗靶点。
