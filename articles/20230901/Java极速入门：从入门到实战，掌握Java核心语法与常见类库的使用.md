# 2.Java简介
## 2.1Java概览
### 2.1.1什么是Java？
Java（全称Java Platform Enterprise Edition，Java企业版平台）是由Sun Microsystems公司于2007年推出的跨平台开发语言。Java具有简单而易用的特点，而且支持多种编程范例，如命令行界面、Web服务端开发、移动应用程序开发、嵌入式系统开发等。
Java的设计目标是保证应用程序的安全性、健壮性和性能，并提供一种编译型的、面向对象的、解释型的动态编程环境，使开发者能集中精力开发应用程序的核心业务逻辑。Java的编译器可将源代码编译成字节码，再运行时加载执行。Sun公司担任Java标准化组织JCP(Java Community Process)的首席主席。它制定了Java的技术规范，并推动其发展。截至目前，Java已成为最流行的通用计算机编程语言，Java虚拟机（JVM）市场占有率超过90%。
Java虽然跨平台、支持多种编程范例，但是它的统一性和平台无关性导致了它应用范围的局限性。因此，Java程序在不同的操作系统平台上运行时，需要分别编译，而且还需安装对应版本的Java Runtime Environment (JRE)。同时，不同版本的Java之间也无法互相调用。另外，Java的内存管理机制比较复杂，容易出现内存泄漏或溢出错误。
### 2.1.2Java优势
Java拥有以下几个显著优势：
- 免费和开放：Java拥有完全的自由软件许可协议，任何人都可以在不受限制的情况下使用它，不必支付昂贵的许可费用；
- 可靠性：Sun公司将Java的可靠性和稳定性做到了极致，Java程序可以继续运行几个月甚至几年不间断地正常运行；
- 平台无关：Java可以运行在各种操作系统平台上，不需要额外安装或配置就可以运行，只要有相应的硬件设备就行；
- 安全性：Java通过虚拟机和JIT编译器来保障安全性，使得Java程序运行效率得到提升；
- 对象模型：Java的面向对象编程特性可以有效地处理复杂的数据和问题，而且通过接口和继承机制，可以方便地扩展和重用代码；
- 支持多种编程范例：Java支持面向对象的编程、命令行界面编程、分布式编程、动态网页编程等多种编程范例；
- 开源：Java是开源的，任何人都可以任意下载和使用Java代码，并基于Java开发者社区的努力不断改进它；
- 易学习：Java具有简单、易读、直观的语法结构，学习起来非常简单。
### 2.1.3Java开发环境搭建
安装JDK，有两种方式：
- 直接从Oracle网站下载安装包进行安装：下载地址https://www.oracle.com/technetwork/java/javase/downloads/index.html ，按照提示完成安装即可。
- 通过包管理工具安装：Mac用户可以使用HomeBrew进行安装，Linux/Unix用户可以使用apt-get、yum等命令进行安装。
设置JAVA_HOME变量，Windows系统添加JAVA_HOME到Path环境变量，Mac/Linux系统编辑~/.bash_profile文件增加如下两行：
```
export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_181.jdk/Contents/Home
export PATH=$PATH:$JAVA_HOME/bin
```
保存退出后，在控制台输入`source ~/.bash_profile`，使环境变量生效。
测试是否安装成功，打开终端输入`javac -version`查看Java版本号，若显示版本号则证明安装成功。
# 3.Java语法概览
## 3.1Java标识符
### 3.1.1标识符概述
在Java中，每个变量、函数、类、接口和包都有一个唯一的名字叫做标识符。标识符必须遵循以下规则：
- 字母（A-Za-z）、美元符($)、下划线(_)和数字组成；
- 不能以数字开头；
- 不允许关键字作为标识符；
- 区分大小写。
### 3.1.2有效标识符示例
```java
myName //合法
name123 //合法
_abc //合法
$money //合法
int myAge = 18; //非法，关键字不能作为标识符
```
## 3.2Java注释
### 3.2.1注释概述
Java注释用于描述程序代码，提高程序的可读性和易懂性。Java有单行注释和块注释。单行注释是//后的所有内容都会被忽略掉，可以用来标记当前行；块注释是/* */之间的所有内容都会被忽略掉，可以用来对大段的代码进行注释。
### 3.2.2单行注释
单行注释以//开始，注释的内容到行尾结束。例如：
```java
System.out.println("Hello World!"); // This is a single line comment.
```
### 3.2.3块注释
块注释以/*开始，以*/结束，注释的内容可以延续到下一行。例如：
```java
/**
 * This is a block comment.
 * It can span multiple lines.
 */
public class MyClass {
    public static void main(String[] args) {
        System.out.println("Hello world.");
    }
}
```
## 3.3Java数据类型
### 3.3.1数据类型概述
Java语言支持以下八种数据类型：
- 整型：byte、short、int、long
- 浮点型：float、double
- 字符型：char
- 布尔型：boolean
- 引用类型：class、interface、array
- 方法类型：() -> returnType
- 通配符类型：? extends T、? super T、?
- null类型：null
### 3.3.2整数类型
Java支持四种整数类型：byte（8位），short（16位），int（32位），long（64位）。整数类型用来表示带符号的整数值，包括正数和负数。例如：
```java
int num1 = 123;
byte num2 = 45;
long num3 = 123L;
short num4 = 23;
```
### 3.3.3浮点型
Java支持两种浮点类型：float（32位）、double（64位）。浮点型用来表示小数值，包括单精度和双精度。例如：
```java
float num1 = 3.14f;
double num2 = 2.71d;
```
### 3.3.4字符型
Java支持字符类型，其值为单个Unicode字符。例如：
```java
char letter = 'a';
```
### 3.3.5布尔型
Java支持布尔类型，其值只有两个：true 和 false。例如：
```java
boolean flag = true;
```
### 3.3.6字符串类型
Java支持字符串类型，即字符序列，其值由零个或多个字符组成。字符串类型的值用双引号"或单引号'括起来的任意文本。例如：
```java
String str1 = "hello";
String str2 = "world";
```
### 3.3.7数组类型
Java支持多维数组，其元素可以是相同或不同类型的对象。例如：
```java
int[][] arr = new int[3][4];
arr[0][0] = 1;
arr[2][3] = 5;
```
### 3.3.8其他数据类型
除了以上八种基本数据类型，还有以下几种数据类型：
- 方法类型：在lambda表达式中使用的函数类型；
- 通配符类型：代表某种未知类型，可以匹配任何数据类型；
- null类型：用null表示空指针。
## 3.4Java运算符
### 3.4.1算术运算符
Java支持以下算术运算符：
- 加法运算符：+
- 减法运算符：-
- 乘法运算符：*
- 除法运算符：/
- 模ulo运算符：%
- 自增自减运算符：++x、 --x
- 一元加法运算符：+x
- 一元减法运算符：-x
### 3.4.2关系运算符
Java支持以下关系运算符：
- 等于运算符：==
- 不等于运算符：!=
- 大于运算符：>
- 小于运算符：<
- 大于等于运算符：>=
- 小于等于运算符：<=
### 3.4.3逻辑运算符
Java支持以下逻辑运算符：
- 逻辑与运算符：&&
- 逻辑或运算符：||
- 逻辑非运算符：!
### 3.4.4条件运算符
Java支持条件运算符，即三元运算符，使用类似if-else语句。语法形式为：condition? expressionIfTrue : expressionIfFalse 。例如：
```java
int result = x > y? x + y : x * y;
```
### 3.4.5赋值运算符
Java支持以下赋值运算符：
- 赋值运算符：=
- 加法赋值运算符：+=
- 减法赋值运算符：-=
- 乘法赋值运算符：*=
- 除法赋值运算符：/=
- 模ulo赋值运算符：%=
- 位左移赋值运算符：<<=
- 位右移赋值运算符：>>=
- 位与赋值运算符：&=
- 位异或赋值运算符：^=
- 位或赋值运算符：|=
### 3.4.6其他运算符
Java支持以下其他运算符：
- instanceof运算符：检查对象是否属于特定类或接口，语法形式为表达式 instanceof type 。例如：
```java
if(obj instanceof String){
  System.out.println("It's a string");
} else if(obj instanceof Integer){
  System.out.println("It's an integer");
} else{
  System.out.println("Unknown object");
}
```
- 数组访问运算符：通过索引获取数组元素，语法形式为数组名[下标] 。例如：
```java
int[] nums = {1, 2, 3};
int n = nums[1]; //n的值为2
```
- 成员访问运算符：通过点号.获取对象的属性或方法，语法形式为对象名.属性或方法名 。例如：
```java
MyClass obj = new MyClass();
String name = obj.getName(); //name的值取决于MyClass对象的getName方法的返回值
```
- 条件运算符?: 可以替代if-else语句，语法形式为表达式1? 表达式2 : 表达式3 。例如：
```java
int max = x > y? x : y;
```
