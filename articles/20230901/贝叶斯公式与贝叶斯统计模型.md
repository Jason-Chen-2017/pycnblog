# 1.简介
  

　　贝叶斯统计是一种基于概率论的统计方法。它利用已知条件下随机变量的联合分布情况，根据样本数据计算参数的先验分布和后验分布，并利用贝叶斯公式求得后验概率分布。在实际应用中，贝叶斯统计模型用于解决复杂系统的参数估计、预测、决策等问题。
　　在贝叶斯统计中，概率论中的几个基本概念被用到：事件（Event）、样本空间（Sample Space）、样本点（Point）、概率（Probability）、似然函数（Likelihood Function）、条件概率（Conditional Probability）。这五个概念是贝叶斯统计的基础，理解这些概念对于掌握贝叶斯统计至关重要。
　　贝叶斯公式是贝叶斯统计中最重要的公式之一。贝叶斯公式描述了条件概率的关系，也称为"Bayes' theorem"。它可以用来表示一个给定条件下另一个条件的概率分布，或从观察到某些结果而获得信息的过程。贝叶斯公式为不同条件下的后验概率提供了一种有效的方法，使得人们可以对影响因素之间相互作用的过程进行建模。其形式为：P(A|B)=P(B|A)P(A)/P(B)，其中A和B分别代表两个不同的条件，P(A|B)为B条件下A发生的概率，等于P(B|A)P(A)的概率比值。当两个条件独立时，贝叶斯公式可简化成P(A|B)=P(A)*P(B|A)。为了计算复杂的概率模型，贝叶斯统计模型通常使用最大似然法或梯度上升法来估计参数，并通过模拟退火算法或采样方法来推断模型参数。
# 2.核心概念及术语
## 2.1 事件、样本空间、样本点、概率、似然函数、条件概率
### （1）事件
　　在概率论中，事件（Event）是一个描述客观现象或主观心理活动的对象，可以取两种状态：发生和不发生。如果事件发生，则称为“真”，否则称为“假”。例如，骰子投掷一次，出现的值为3的概率是1/6；投掷两次，第一次的值为3，第二次值为4，则第三次的值只能是3或4，且两次抛出的都是硬币正面，则第四次只能出现1/4的概率，依此类推。一般来说，事件可以是一切客观现象，如抛掷一次骰子、摔倒一次，也可以是某个主观事物，如一次正确答题、一次考试及格。
　　一个事件的样本空间（Sample Space）指该事件的所有可能情况。例如，投掷一次骰子，样本空间为{1,2,3,4,5,6}；投掷一次硬币，样本空间为{正面，反面}; 一次考试的结果，样本空间为所有可能的成绩，如90分以上、80分以上、70分以上、60分以下等。
　　样本点（Point）是指一个元素（数或物体），它构成了某个事件的样本空间。例如，投掷一次骰子，出现的数字就是样本点；投掷一次硬币，出现的正面和反面的样本点；一次考试的成绩，则是样本点。
### （2）概率
　　在概率论中，概率（Probability）是表示事件发生的可能性或概率的度量。它是一个客观量，其值在0~1之间。设A为一个事件，其概率为p(A)，意味着事件A发生的概率。具体地说，若样本空间S={A,B},其中A为真，B为假，则：
- P(A)表示事件A发生的概率；
- p(A)∈[0,1]；
- P(A)=0,iff A为假；
- P(A)=1,iff A为真；
- P(AB)表示同时发生事件A和事件B的概率。
### （3）似然函数
　　在概率论中，似然函数（Likelihood Function）是定义在样本空间上的一个函数，其定义域为样本空间，值域为实数。它描述了一个样本的可能性与它的条件概率之间的关系。
#### 概念阐述
　　在统计学中，似然函数是用来描述一个给定的模型产生数据的可能性。换句话说，似然函数是给定观测数据集X和模型参数θ，计算X出现的可能性。因为观测数据往往是存在误差的，所以需要考虑模型对观测值的不确定性，因此才引入了似然函数这个概念。具体来说，如果模型的预测是准确的话，那么似然函数就应该取较大的数值；如果模型的预测能力比较差的话，那么似然函数就会取较小的数值。当模型参数确定后，可以根据似然函数计算各个样本点出现的可能性。
#### 功能
　　似然函数具有很多作用，包括：
- 对模型进行训练：假设一个模型的概率密度函数为f(x;θ),然后选择使得似然函数最大的参数θ作为最佳参数。
- 模型的评价：计算似然函数，可以判断模型的准确性，比如在统计学习过程中，可以通过似然函数确定模型的分类效果好坏。
- 模型的预测：通过已知模型参数，对新样本的预测。
- 参数估计：用似然函数对参数进行估计。
