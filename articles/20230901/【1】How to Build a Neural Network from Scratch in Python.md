# 1.简介
  

近年来人工智能领域的火热也越来越显著。许多创业公司、科技巨头纷纷涉足人工智能领域，这些领域最突出的就是深度学习（Deep Learning）和机器学习（Machine Learning）。虽然很多创业者抱着“创新”、“做出突破性改变”的心态，但实际上，这一行业依然存在许多困难，特别是在解决一些复杂的问题时。因此，我认为对于刚入行的人来说，理解如何构建一个神经网络从零开始至少可以提供一个基本的思路。本系列教程将详细阐述如何用Python从零开始构建一个神经网络，主要包括如下内容：

1. 使用NumPy库来快速实现线性代数计算；
2. 从零开始实现神经网络中的激活函数（activation function）；
3. 用NumPy库构建多个层次的神经网络结构；
4. 梯度下降法求解参数更新值；
5. 实现误差反向传播算法及其推广——BP算法；
6. 模型保存和加载功能。
7. 可视化工具箱：TensorBoard。
本文将对第1节简单介绍一下前置知识和使用环境，然后开始进入正文。
