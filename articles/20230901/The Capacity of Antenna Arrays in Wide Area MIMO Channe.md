# 1.简介
  

## 1.1 问题的提出
宽带多径接方案(W-MIMO)已经成为多种高速宽带业务的主要选择之一。但是，由于W-MIMO信道中存在复杂的天线阵列结构，因此，如何设计有效的天线阵列结构对于稳定性、经济性和性能方面的影响至关重要。本文试图通过分析几个关键参数——天线数量、阵列空间损耗、系统容量、信号衰减等指标，来计算不同规模的天线阵列所能达到的最大数据传输率（MIMO）能力。在此基础上，还考虑到实际使用过程中还存在着许多其他不确定因素，如覆盖范围、环境条件、干扰源、用户位置分布、访问速度、传输率、用户行为等等，从而给出更全面准确的天线阵列设计建议。
## 1.2 定义与符号说明
- 大功率天线:一般用于Wi-Fi、蓝牙等通信业务场景中的主导信号，具有较大的辐射功率、衰减时间短、抗干扰性强等优点。
- 小功率天线:一般用于W-MIMO业务场景中的次要信号，能够有效降低主导信号对系统性能的影响。
- 几何天线:由理想几何体制成的天线结构，其体积小巧便携、集成度高且导线布置简单、具有良好的抗跌落能力。
- 大尺寸天线:通常包括方向盘、开口聚焦器、基座等组件构成的天线结构，应用于一些高速宽带业务中，但体积大、重量大、容易产生干扰。
- 小尺寸天线:通常只有开口聚焦器和直连天线等简单组成的天线结构，应用于特定的业务场景中。
- 天线阵列:由多个天线所组成的天线网络，其中各个天线可以是不同类型，比如几何天线、大功率天线、小功率天线、大尺寸天线或小尺寸天线等。
- 概览天线系统容量:由一个天线阵列所能提供的每秒传播的数据比特数称作概览天线系统容量，即每秒比特传输率（Mbit/s）。
- W-MIMO标准信道带宽:在MIMO技术下，信道宽度决定了信道可容纳的有效载荷的大小；为了方便计算，通常假设W-MIMO标准信道带宽约为1.4 GHz。
- 分辨率:指天线阵列中各个天线相互之间的距离。分辨率越小，则代表系统的间隔越大；分辨率越大，则代表系统的距离变得越远。分辨率的大小通常以瓦字数为单位表示。例如，1/32波长分辨率对应着波长/32，即1/32波长=31.25微米。
- 天线损耗:天线损耗反映了天线传递信号所需的能量，它会随着天线的增多而增加。
- 阵列空间损耗:阵列空间损耗是指在传输数据时，无线电信道天线阵列所占用的空间大小。
- 基站站点覆盖:指在需要部署W-MIMO天线阵列的基站覆盖范围。
- 业务利用率:指通过W-MIMO天线阵列实现业务的比例。
- 自适应多址技术(ADMA):也称为动态MIMO(DM)，是一种在无线通信网络上进行数据共享的技术，它能够利用信道资源的空闲情况，合理分配不同用户的频谱资源，提升整个网络的利用率。
- 入侵检测技术(IDT):是一种基于统计分析的方法，能够在用户之间的传输过程中识别并消除入侵的干扰信号。
- 频繁接入控制(FACL):是一种无线网络管理技术，用来防止对无线通信系统的恶意干预，保障网络正常运行。FACL可以对不同用户的信息流动进行优先级划定，将某个用户的优先级提高，保证信息安全。
- 用户停留时间(UST):指用户在某一时段内，保持无线通信状态的时间长度。UST越长，则代表通信质量越好。
- 信号衰减:信号衰减是指通过无线信道传输信号时，会发生的幅度变化。衰减越厉害，则代表信号的幅度变化越大。
- 相关参数说明
|   参数名    |              描述               |
|------------|-------------------------------|
| W-MIMO信道  | 指多路径多天线通信技术         |
| 天线类型    | 包括几何天线、大功率天线、小功率天线、大尺寸天线及小尺寸天线  |
| 天线数量    | 每个用户位置部署的天线数量     |
| 工作模式    | 可分为叠加、独立、混合三种工作方式，多路径天线一般采用混合型工作模式  |
| 分辨率      | 天线相互之间相距的距离，单位通常为瓦字数  |
| 天线损耗    | 天线的传递信号所需的能量，单位通常为瓦字数  |
| 阵列空间损耗  | 在传输数据时，无线电信道天线阵列所占用的空间大小，单位通常为瓦字数  |
| 数据容量    | 一秒钟最多能传输多少比特数据，单位通常为兆字节  |
| 时延        | 数据传输时间延迟，单位通常为微秒  |
| 费用        | 由网络设备产生的费用，如建设、维护、电力及其它成本  |
