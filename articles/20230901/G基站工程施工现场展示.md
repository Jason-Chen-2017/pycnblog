# 1.简介
  

## 1.1 任务目标
根据国家重点研发计划“华为5G基站及云计算中心施工现场展示”的要求，制作一系列具有代表性的“基站工程施工现场展示视频”，从基站配套的相关设施、设备入手，带领观众一步步了解华为5G基站施工的全过程。
## 1.2 背景介绍
近年来，随着5G网络的广泛普及和应用推广，5G基站建设也成为热门话题。2019年，华为公司完成了升级改造华为大型双频3D双模核融合小卫星通信天线（LANTEN）所需的所有前期准备工作，并在2020年正式启动了5G基站建设。本项目承接华为公司重点研发计划“华为5G基站及云计算中心施工现场展示”的任务，以“基站工程施工现场展示视频”作为主要创新输出形式，逐步构建起“华为5G基站及云计算中心展示”系列视频，将5G基站建设及运营的全貌呈现在公众面前，让更多感兴趣的企业能够了解到华为5G基站建设的实地细节，更加深入地参与到5G基站建设的过程当中，共同打造属于自己的5G基站。
## 1.3 研究内容
为了充分理解华为5G基站建设的全过程，制作“基站工程施工现场展示视频”对完成该项任务至关重要。由于基站工程规模庞大，需要精确把握每一个模块的布置和设备运行情况，因此，在制作过程中会有许多难免要借助照片或图表进行辅助说明，但同时还需注意力高度集中，以及能够清楚完整地叙述每个环节的工程操作流程，并做到视听效果的流畅性与准确性。此外，还有一些其他任务也应考虑到，例如：
* 播放时间控制：有的工程项目较复杂，一段时间内无法全部播放完毕，应该尽量压缩播放速度或者分成多段视频。
* 背景音乐和配音：为了突出演示效果，可以加入适当的背景音乐和配音人员进行配合。
* 时空同步：如果是多人参与的工程项目，可能需要考虑各自的工作进度与时长，同步播放视频。
* 环节间互动：如有多个环节之间存在交叉部分，可以增添一些互动机制，引导观众认识到相关信息。
* 原始视频：如果是硬件方面的研究，则需要原始工程视频资料，才能获取到准确的调试数据。
* 可修改的内容：视频内容可能随时间变化而发生改变，因此可以采取定期更新的方式，适应不同部门的需求。
* 浏览器兼容性：不同的浏览器和手机APP都可以方便地观看，提升观看体验。
* 后期制作风格：根据观看效果和投影仪类型，可以选择采用不同的后期制作风格，比如采用复古设计风格、卡通风格等。
* 风险评估：对于某些严肃的项目，如基站项目，需要提交质量保证报告，以避免出现不良反应。
## 2.基础知识介绍
基站工程一般包括五大环节：铺设、铁路与电力线路的安装、导电防护、室外安装通风与采光系统、固定测控与监测系统。其中，铺设环节占用工程的80%以上。由于工程大型化和结构复杂化，在铺设环节中，必然涉及到大量的设备、材料、工具的应用，这就给专业的工程师带来极大的挑战。
## 2.1 概念及术语
### 2.1.1 5G基站工程概况
目前，5G是一种新的移动通信技术，它基于宽带无线信道，提供比传统的4G信号强度更高、速度更快的数据传输能力。5G基站就是利用4G通信技术解决5G数据通信问题的重要组成部分，也是实现5G覆盖的关键设备之一。5G基站的目标是建立一座高速、高带宽、高可靠的大基站，为5G用户提供上网服务。5G基站建设的目标与任务是建立5G基站技术平台、构建5G基站产业链、培育5G基站建设者，以及促进5G基站建设与运营取得更大突破。
### 2.1.2 5G基站工程原理
5G基站的主要功能是通过通信技术提供上行、下行双向服务，为终端用户提供语音、视频、短息、直播等多种应用。其主要原理是将高频频段的4G信号经过采样、编码、调制、发送，最终形成符合5G标准的射频载波，在室外接收、处理后再转回4G信号用于终端用户的接入，实现用户与基站之间的通信。如下图所示：
### 2.1.3 铺设
5G基站的铺设是一个非常复杂、繁琐的工程项目。首先需要选址、勘察、确定施工路线、开挖土方、填海造田、架设支架、安装导线、铆接接头、固定设备。其次还需要进行施工地基的封顶、装饰土石方、铺设垂直支持、施工设备的安装和调试。最后还要对全基站进行软、硬件的测试，确保基站安装完好、稳固、运行正常。因此，5G基站的铺设工作十分复杂，往往需要专业的人才和资源，耗费巨大的金钱和时间。
### 2.1.4 导电防护
导电防护是指安装导线和机房设备等硬件设备的保护措施，它既不能损害基站的电源、信号传输，又能有效保障设备的安全。导电防护的作用主要有三个方面：一是预防火灾；二是抗雷击、火灾等安全威胁；三是降低设备故障率和维护成本。以下是导电防护的原理：
* 用压敷式挖掘法防火焰：通过预先对设备施加微小的压力，使设备具备抗风、抗腐蚀、抗火灾、抗油烟等能力，防止出现火灾。
* 使用陶瓷瓷砖拼装：通过预先填充陶瓷砖，使得导线不易受到灼伤，易于长久保存。
* 耐火材料与防火材料：通过加入耐火材料、防火材料，在安装前进行试验，确保导线的耐火性能。
### 2.1.5 室外安装通风与采光系统
室外安装通风与采光系统是基站在外部环境中进行的通风、采光、供暖、排水等环节。它与导电防护、设备安全、导线安装、管道安装等环节密切相关。室外安装通风与采光系统的原理如下：
* 温差控制：通过调节室外环境温度、湿度、蒸汽湿度、雷电、光照等条件，使设备处于正常工作状态。
* 静音控制：通过控制设备静音，防止设备噪声影响用户工作效率。
* 舒适化控制：通过设计舒适的安装接口，提高设备使用舒适度。
* 屏幕保护：通过控制屏幕颜色、亮度、布局位置，防止遮挡用户视线。
### 2.1.6 固定测控与监测系统
固定测控与监测系统是在基站建设后期用来检查设备是否按时、无故障，以及检测系统运行是否正常。它涉及到各种传感器的安装、校准、调试、配合，以确保基站系统工作正常。固定测控与监测系统的原理如下：
* 测压计：它是一种测量电压值的方法。通过对测压计的输出进行跟踪和记录，可以获得当前信号电压的大小。
* 电表：它是通过用电的方式来测量电压的。通过对电表的读数，可以知道电压的大小。
* 智能电网监控系统：智能电网监控系统是由智能电网的控制中心采集各个节点的实时数据，从而分析系统的运行状况，辅助对系统进行管理和维护。
