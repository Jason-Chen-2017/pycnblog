# 2.智能表计

## 2.1.定义

智能表计，即电能监测仪、电能记录仪和电能交换系统，是一种新的电能计量技术，它将电能数据采集、存储、传输和应用计算机技术实现电能数据的实时监控。目前，智能表计已广泛应用于电力、燃气、水等领域，并逐渐形成了自己的阵营，例如国标、IEC62196-21等。

## 2.2.分类

根据表计所使用的传输方式分为短距离传感器型表计和广域传感器型表计。短距离传感器型表计一般采用无线通信技术传输数据，如Zigbee、WiFi、蓝牙等，能够提供较高的数据采集率；广域传感器型表计使用卫星传输数据，能够提供更高的灵敏度和较长的使用寿命。除此之外，还有一些表计如光伏电池板表计、重力感应表计、激光雷达表计等。

按照用途分，智能表计主要分为收费用电表计和非收费用电表计。对于收费用电表计，主要用于公共用电、私人住宅和商业建筑等领域；对于非收费用电表计，主要用于居民生活、政府机关、医疗机构等领域。

## 2.3.功能

智能表计具有如下四大功能：

1. 数据采集：能够进行一定时间内的电能数据采集，精确到每秒采集一次；
2. 数据存储：智能表计通常会连接到电网管理系统中，将采集到的电能数据存储下来；
3. 数据处理：智能表计采集到的数据经过处理后，会转化成电能质量指标，如电压、电流、功率等；
4. 数据传输：智能表计将数据传输至服务平台上，便于用户查询和管理。

# 3.绿色发电

## 3.1.定义

绿色发电，即通过无源电力设备（如太阳能或风能）产生电能，不需要用电厂进行任何装置工程改造，即可获得电能。当前，由可再生能源资源（如地球能、火山烈岩、油气田、光伏电池板）产生的电能也可以称为绿色电力。

## 3.2.优点

1. 节省运营成本：绿色发电不需要建设任何设备，只需几公斤甚至几十公斤的无源电力设备即可；
2. 降低成本：通过绿色发电可以降低电力成本，特别适合受到资源约束的弱小国家或地区；
3. 更多的利润空间：绿色发电可以让企业获得更多的利润空间，尤其是受到环境污染严重的区域；
4. 提高效益：绿色发电可以促进社会经济发展，促进国际贸易和投资。

## 3.3.分类

根据产能类型分为可再生能源发电与不可再生能源发电两种。可再生能源发电包括太阳能、风能、水电等能源，它们可以产生热能，并且通过堆垛的方式可以批量生产。不可再生能源发电包括光伏、蓄电池板等不能产生热能的能源，但却可以利用特殊的技术将电能转换为电能。

# 4.可再生能源资源

## 4.1.定义

可再生能源资源，又称绿色能源，指拥有独特的能量储存特性和较强的自然对流特性，是指能够被利用作为热能或冷能，发电成果的能源资源。一般来说，可再生能源资源具有以下几个特点：

1. 有待开发的能力：指能源可以通过人为改变能量状态，使其具备适宜生物生活的条件，再由海洋生成带电粒子（电）或由气态原料生成含电粒子（雷电）来储存能量；
2. 有弹性：能源不仅能满足短期需求，而且还可以持续吸收外部输入，维持能量供应；
3. 可自然分解：可再生能源资源能够自然分解，不会因环境污染或自然变异而损坏；
4. 可再生性：能源的能量既可以存储下来供以后的使用，同时又可以在必要时利用自然对流进行再生。

## 4.2.分类

可再生能源资源分为地球能、火山烈岩能源、油气田能源和光伏能源。地球能是指火山活动产生的热量，包括天然气，固体气，氢气和氧气；火山烈岩能源是指在火山口产生的热量，包括火山灰，火山云，火山泥，火山煤，以及核辐射等；油气田能源是指油气资源释放出的热量，包括油田产生的蒸汽、热水和空气蒸汽；光伏能源是指光伏电池板产生的热量，可以形成热浴设备。

# 5.智能表计整合可再生能源资源的可行性评估

在完成了前面的综述性介绍后，下面介绍智能表计整合可再生能源资源的可行性评估的各个阶段及其关键技术。

## 5.1.需求分析阶段

这个阶段是指对需求进行初步的分析，确认需求的优先级和预算，制订方案。做好需求分析的同时，还要考虑到客户对政策的要求、现有产品技术的局限以及行业发展的压力等因素，避免出现一些意想不到的困难。

## 5.2.方案设计阶段

方案设计的主要任务是设计出符合客户需求的智能表计整合可再生能源资源解决方案，并向客户展示。方案设计的过程需要对整个流程进行梳理，包括需求分析、方案选型、系统设计、接口规划、测试计划、施工部署等。

为了充分利用可再生能源资源，智能表计需要考虑到其潜在的利用价值、成本效益以及环境问题。因此，在方案设计过程中，需要确定采用哪种形式的智能表计、可再生能源资源的形式、选取哪种可再生能源以及如何整合等。另外，还需要根据客户需求制定相应的施工计划，比如安装位置、安装材料、耗材以及安装人员的培训。

## 5.3.试运行阶段

试运行阶段就是使用实验室仿真软件模拟整个智能表计整合可再生能源资源解决方案，并对软件系统进行测试。测试的内容包括系统设计的准确性、数据采集与传输的连通性、能源与电力资源的控制、消费者端的操作界面、用户满意度评价等。如果测试发现有问题，则需要根据测试结果对解决方案进行修改，直到成功运行为止。

## 5.4.部署阶段

当试运行成功后，就可以正式部署智能表计整合可再生能源资源系统了。部署系统的过程包括设置网络连接、配置控制器等，并保证相关设备正常工作。一旦系统部署完毕，就应该密切注意系统运行状况，根据情况进行维护、升级和扩展。
