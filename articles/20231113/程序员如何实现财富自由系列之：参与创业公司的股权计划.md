# 2.核心概念与联系
## 什么是创业公司股权计划？
在创业公司中，成员股东均拥有普通股股份，称为“社保基金”（Community Insurance Fund），主要用于偿付成员退休、死亡或其他类似情况所带来的经济损失。由于众多创业公司的成立都是为了独特的利润目标，在与企业谈判完成并获得批准之前，成员股权并不是由政府提供的，而是在创始人即“创业者”的名下完成认购。也就是说，创业公司获得了有限的股权，需要由创业者自行筹集资金，并按照一定规则进行申购、拨款和管理。每年需要缴纳股息，这是最基础的收入来源。然而，由于规模的扩大、市场竞争的激烈，这种长期持股的方式可能遇到瓶颈。

为了克服这一局面，创业公司通常会设定所谓的“创业基金”，由创始人和领投人共同筹集资金，进行种子轮融资等方式进行资金的募集。同时，鼓励创业者向这些基金投资，进一步增加了公司的资本金的比例。这种方式可以降低融资成本，提高了融资效益。但是，由于创业公司本身具有高股票价格和极高的杠杆率，以致于投资者本身也很容易被骗，成为风险之所在。

为了解决以上症结，美国证券交易委员会（SEC）于2007年推出了“创业股权计划”，其基本思想是允许创业者发起公司的股权计划。股权计划是创业公司凭借其自身优势获得股份的一种计划，允许创业者私下设定资金、出资额度和分配方式，让股东拥有一定的股权。创业者根据设定要求、资金状况和公司发展情况，依照不同比例向社保基金或个人股票基金发放股份。目前，美国已经设立超过2000多个创业公司股权计划，相当数量的公司加入了私募股权计划。

## 如何参与创业公司股权计划？
创业公司股权计划可分为四个步骤：

1. 提交申请表：首先，需要提交一份申请表，包括以下信息：
- 公司名称；
- 创始人姓名、职位、身份证号码；
- 公司地址、电话、邮箱；
- 主要业务范围；
- 社保基金/个人股票基金账户信息。

2. 支付贷款：第二步，是向所需社保基金或个人股票基金支付适当的贷款，用于筹措资金。

3. 执行流程：第三步，是用社保基金或个人股票基金对股权进行管理。参与股权管理过程需要遵守以下流程：
- 公告：公开发布股权计划的具体内容和分配方案。
- 股东大会：召开股东大会，讨论提案，决定是否接受。
- 增发股权：对变动股权的增发办法进行审核和执行。
- 分配股权：对股权计划中的股权按比例分配给股东。
- 撤销股权：若发展失败或出现重大问题，股权计划可按申请撤销股权。

4. 管理费用：第四步，是由管理机构收取管理费用。

## 为何有必要参与创业公司股权计划？
参与创业公司股权计划可以带来诸多好处，例如：
- 拥有一定的权益：创业公司股权计划为创业者提供了与普通股一样的股权。创业者可以利用股权池建设商业模式，扩大产品或服务的市场份额，并通过股权获取利润。同时，股权也可以支持创业者的事业发展，促进创业者和公司之间的合作。
- 获得筹资能力：创业公司股权计划为创业者提供了筹集资金、运营公司的空间，这有助于创业者在创业初期迅速积累资金。由于公司具备一定规模，可以承受较大的运营成本，因此，创业者就可以直接从股权计划中获得部分现金流，降低投资风险。
- 减少风险：由于创业者的资本金来自个人投资，创业者本身具备较高的风险厌恶和投资预期。创业公司股权计划可以降低投资者参与投资的风险，并大大减小其损失。

因此，越来越多的创业公司选择加入私募股权计划，利用自身资源提升自身的价值，实现财务自由。参与股权计划不仅可以帮助创业者找到投资方向，提升个人品牌形象，还可以赢得社会各界的支持和关注。
