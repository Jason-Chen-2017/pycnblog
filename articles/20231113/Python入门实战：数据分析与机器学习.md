# 2.核心概念与联系
## 数据处理与分析
数据处理（Data Processing）、数据分析（Data Analysis）、数据挖掘（Data Mining）是三个重要的计算机科学词汇。其中数据处理就是指数据的清洗整理、转换、存储等操作，它需要对数据进行提炼、分类、整理、过滤等处理，以确保其数据的准确性、完整性、有效性。数据分析就是指利用数据进行研究、观察、分析、总结，通过对数据进行多维度的描述、分析、呈现，从而发现规律、解决问题、做出决策。数据挖掘是指从海量数据中挖掘隐藏的模式、信息、价值等。

## 数据结构
数据结构（Data Structure）是指计算机中存储、组织数据的方式。最基本的数据结构有两种：数组（Array）和链表（Linked List）。数组是一种线性结构，元素按顺序存储，可以随机访问。链表是一种非线性结构，每个结点只包含一个引用，指向下一个结点，不存储数据，只能顺序遍历。除此之外，还有队列、栈、散列表、树、图等数据结构。

## CSV 文件
CSV文件（Comma Separated Values，即以逗号分隔的值的文件）是一种常用的文本文件类型。它通常被用于保存表格数据，每一行代表一条记录，列与列之间采用逗号分隔。CSV文件通常包括文件头和数据两部分。文件头是对每一列数据的描述，通常在第一行出现，每个字段都有一个名称。数据则是真正的内容，在第二行之后依次按行出现，各项数据间用逗号分隔。

## Excel 文件
Excel文件是微软公司开发的一款桌面电子表格软件，可以在Windows、Mac和其他操作系统上运行。它支持各种办公文档、电子表格、财务报表、评论、说明书、目录、数据库和图表等多种格式的生成与编辑。

## 数据分析流程
数据分析过程中，一般会经历以下几个流程：

1. 数据获取：收集数据源（如：数据库、文件、网站），包括数据采集和数据清洗，以获取原始数据。
2. 数据预览：查看数据样例、检查数据质量、探索性数据分析，以便确定下一步分析的方向。
3. 数据可视化：数据可视化是数据分析过程中的一环，它可以帮助我们直观地看到数据之间的关系，辅助我们的分析过程。
4. 数据探索：进行单变量分析、联合分析、聚类分析等，以发现数据中的模式和规律。
5. 模型建立：根据数据样本建立模型，模型训练用来对未知数据进行预测。
6. 模型评估：对模型的效果进行评估，检查模型是否过拟合、欠拟合或者有效。
7. 模型推广：将模型推广到新的应用场景，通过数据更新和迭代更新模型，保持模型的最新状态。
