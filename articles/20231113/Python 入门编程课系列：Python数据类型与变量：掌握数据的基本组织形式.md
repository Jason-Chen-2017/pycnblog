在编程的世界里，数据是最基础也是最重要的一个环节。而对于数据的理解及操作方式往往决定了程序的质量、效率、复杂度等等方面的表现。了解数据的内部结构对编程者来说是非常关键的一步。在实际项目开发中，我们经常需要将不同的数据进行处理，比如字符串、整数、浮点数、布尔值、列表、元组、字典、集合等等。为了方便数据的存储、交换、检索、操作，Python提供了很多内置的数据类型和容器。那么今天，我们就来看一下Python的四种基本的数据类型——字符串、数字、布尔值、列表。本文主要着重于介绍如何利用Python的这几种数据类型来实现数据的结构化。
# 2.核心概念与联系
## 数据类型
### 字符串（String）
字符串是一种可以由零个或多个字符组成的序列，它用单引号或者双引号括起来。它可以用+运算符连接在一起，也可以被索引以访问其中特定位置的字符。

```python
string = 'hello' # 定义一个字符串
print(len(string)) # 获取字符串长度
print(string[0]) # 索引第一个字符
print(string[-1]) # 索引最后一个字符
print(string[0:3]) # 切片获取子串 hello
new_str = string + ', world!' # 连接两个字符串
```

在Python中，字符串属于不可变类型，所以当一个字符串被定义后，其值不能改变。如果要修改某个字符串，只能创建新的字符串。

### 数字（Number）
Python支持三种类型的数字：整型、浮点型和复数型。

- 整型（int）：它的大小取决于计算机系统中存放整数的位数，通常是一个带符号的32位二进制数。
- 浮点型（float）：它是以小数形式表示的实数。它的精确度远远不如整数。通常它占用64位内存空间。
- 复数型（complex）：由实部和虚部构成，即 a + bj (a 为实数部分，b 为虚数部分)。

```python
integer_num = 10   # 定义一个整数
floating_point_num = 3.14159    # 定义一个浮点数
complex_num = complex(1, 2)       # 定义一个复数
print(type(integer_num), type(floating_point_num), type(complex_num))     # 查看数字类型
```

在Python中，数字属于可变类型，可以对其直接进行操作。比如加减乘除，还可以使用floor()函数获得向下取整的值。

```python
x = 7 / 3          # 除法计算得到浮点结果
y = divmod(7, 3)   # 得到商和余数
z = pow(2, 3)      # 计算2的3次幂
w = round(3.14159, 2)   # 四舍五入保留两位小数
v = abs(-3.14159)       # 求绝对值
```

### 布尔值（Boolean）
布尔值只有True和False两种值。布尔值的作用是在判断条件时，把0、None、空字符串等值都视为False，其他任何值都视为True。

```python
flag = True       # 定义一个布尔值
result = flag and False or True        # 布尔运算示例
print(isinstance(flag, bool))           # 判断是否为布尔值
```

在Python中，布尔值属于可变类型。可以对布尔值进行逻辑运算（and、or、not），并返回布尔值。

### 列表（List）
列表是一个有序序列，可以容纳不同的数据类型，包括字符串、数字、布尔值。列表中的元素可以通过索引进行访问，并且可以进行切片操作。列表也支持迭代操作。

```python
list1 = [1, 'hello', True]         # 创建一个列表
print('Length of list:', len(list1))   # 打印列表长度
print(list1[0], list1[1], list1[2])   # 通过索引访问列表元素
list1[1] = 'world'                   # 修改列表元素
sub_list = list1[0:2]                # 通过切片操作获取子列表
for item in sub_list:                 # 遍历列表
    print(item)                       # 输出每个元素
```

列表的特点是可以动态增长，通过append方法可以添加元素到末尾，insert方法可以在指定位置插入元素。列表同样也可以删除元素。

```python
list1.append('goodbye')              # 添加元素到末尾
list1.insert(1, 'apple')             # 在第1个位置插入元素
del list1[0]                         # 删除第一个元素
```

在Python中，列表属于可变类型，对其进行的操作会影响到原始对象。因此，建议只读列表对象的方法和属性尽量不要修改它。
