                 

# 1.背景介绍


人类从远古时代起就一直在追求科技的进步。自然界就已经形成了种群分工合作共同繁衍的结构，而人类则是运用智慧和技术让这个结构变得更加复杂、高效。智慧来源于繁衍和发现新东西的能力，技术来源于生产制造和加工物质的手段。在经过漫长的演化后，人类的文明经历了数千年的创伤之后，才成为今天这样的模样。
近几十年来，随着经济的飞速发展和人口的扩张，科技和生产力的发展速度已经超过了人类历史上的任何时候。特别是互联网的崛起和移动终端的普及，使得人们不仅能够享受到科技带来的便利，还能够获取海量的数据、信息和服务。这不得不让人们思考如何利用这些数据、信息和服务来改善社会的条件，以及如何重新定义人类的生活方式。
人类在面对“技术变革”这一挑战时，首先要考虑的问题就是如何实现这个变革。在这个过程中，为了保障人类的生命安全，建筑行业也必须得到有效的保护和管理。建筑技术具有广泛的应用领域，包括教育、医疗、住宅、商业、金融等各个领域。建筑技术也要从基础设施、分布式计算、数字化、现代控制、大数据等多方面逐步地转型升级。
在这个过程中，建筑技术将会发生哪些变化？它将如何影响我们的生活？为了实现这个目标，需要如何应对挑战？如何处理这场战争？而这些都是技术专家们必须认真研究的问题。
因此，在本次分享中，我们将以《人类技术变革简史：城市化与建筑技术的发展》为主题，通过研究建筑技术的发展历史、变迁、功能和规律、以及其对人类生活产生的影响，来阐述技术变革对建筑行业的作用、变迁、局限性和挑战。
# 2.核心概念与联系
## 2.1 人的驯化方式——城市化
首先，我们需要了解一下人类在远古时期的驯化方式，即人的驯化目的在于延续、传承、分享一切。而人类驯化的方式是建立城市的。从前，我们看到的古老的动植物被称为野生动物，由于没有人类居住区，它们只能依靠自己掌握的工具觅食、繁殖，所以只能以捕鱼、捞鱼为生。古代人为了抢夺这些野生动物，只能用自己的小船、轮船、驼峰作为笼络之用。但随着人类驯化野蛮动物的努力越来越少，他们逐渐发现，虽然野生动物拥有强大的生存能力，但是却没有太多适合用来寻找食物的地方。于是，人类开始向南方迁移，北方出现了农田，在农田上耕作，获得土地肥力。人类向南方迁移的同时，又发现了一批非常聪明、聪慧且富饶的人类，并且开始利用这些人的知识和技能来驯化这些动物。例如：秦国就采用女娲补天的方法把那些尚未驯化成功的野猪驯化成昆虫。这种驯化方法的独特之处在于：女娲并不是恒定存在的神灵，她只是一种帮助修补生物细胞结构的仙草，而且天赋异禀的修补者甚至可以用她来治病、养猪、驾驭水牛、吹风。
经过漫长的演化，人类到了第五纪才完成了一次全面的大规模的跨越。所谓跨越，指的是超过生物本身的自我限制和束缚。人类第一次跨越完成后，开启了一个新的 era:“文明”。第一次大规模跨越后，人类又打开了一个新的机遇：“雨果主义”。这里的“雨果”指的是埃及著名作家阿瑟·雨果。他曾说过：“不论有多少人类，都无法阻止历史的进程，但是只有更强的力量才能推动进步。”这位才华横溢的老前辈教导了人们：“只有掌握了知识、工具和技能，才能征服整个星球。”他鼓励所有人都勇敢追赶历史的脚步，继续开拓和创造属于自己的时代。
这一切改变了人类的生存环境，而人类却没有注意到，它的驯化方式也发生了相应的改变。从第二纪开始，人类便有意识地利用自己的驯化资源进行合作共同繁衍。例如，人类刚开始驯化动物时，已经迈出了非常重要的一步：把牲畜留给更聪明、更富饶的后裔，而不是让祖先被动的流放。当时的驯化策略是以牲畜的性格、习性、品种、血统等人为特征进行分类，根据不同人群的生存需要来分配土地。这种策略曾经帮助许多部落、国家和地区都获得了丰厚的粮食收入。
进入第三纪后，人类学会了站在巨人的肩膀上，借助科技的发展，成功地开发出了基因工程、计算机编程、汽车、飞机等各种技术。而这一切的改变也催生了新的人类驯化需求，并且提出了新的人类驯化模式。其中最主要的就是“城市化”。
## 2.2 建筑技术的兴起——兴起的原因
建筑技术的兴起是人类跨越回归前三纪后发生的事件。建筑技术是在雨果主义之后，由欧洲人发现的，并且受到亚当·斯密等哲学家的影响，才被人类认识到。
在漫长的史实之后，历史学家费尔德·诺斯认为：建筑技术是从自然界迁移到人类脚下的过程。一段时间内，人类都处于动物阶段，在竞争激烈的地区，人类必须依赖工具、技术、魔术才能获得生存优势。这种依赖让人类的脚步远远地落后于野生动物的步伐，所以后裔们只能依靠自己小心翼翼的驯化行为获得足够的食物。一旦到达适合人的居住区域，人的生活就会受到极大的冲击。但经过文明史的考验，随着人类科技的发展，人类开始驯化动物，并利用科技的力量来建造新的居民区。人类驯化动物的技术水平越来越高，可容纳大量的动物越来越便宜。到第四纪末期，人类已开发出了极为先进的工具、技术和管理方法，可以用极低的价格建造出可供驯化的庞大复杂的动物。同时，随着城市的发展，人类又开始开发建筑技术来维持生存，并利用建筑的空间来分享信息、财富、制造力等资源。
## 2.3 建筑技术与人类生活的关系——建筑的功能与规律
建筑技术的核心功能之一是作为一种空间工具。建筑技术如同现实世界一样，可以提供人类使用，并影响人类的生活。建筑的功能主要包括三个方面：
- 一是支撑体系：建筑可以提供生活所需的基本材料，例如房屋、建材、家具、运输工具、通讯设备。
- 二是文化传承：建筑可以传递和承载历史、文化、艺术和语言，它可以塑造居民的精神世界和道德情操，促进人际关系的交流，塑造新的价值观念和规范。
- 三是医疗保健：建筑可以为人民提供医疗保健服务，它可以减轻或缓解疾病的影响。建筑技术的增多、应用范围的扩大，促进了人们对自然环境、生态环境的认识，并提升了人们的医疗保健水平。
人类目前的建筑技术主要包括基础设施、大楼、商业中心、居民区等。以下我们将具体介绍建筑技术的一些基本功能、规律。
### 2.3.1 基础设施建筑——土地利用效率的提高、空间布局的优化、交通网络的维护
基础设施建筑的目标是提高土地利用效率，这是通过设计合理的基础设施布局、控制噪音和空气污染来实现的。土地利用效率可以减少土地浪费、降低房价、提高居民收入。基础设施建筑的布局设计应当保证人们在建筑物里获得最大的舒适感，同时也应该兼顾城市的整体布局、交通网络、空气质量和节能性能等因素。基础设施建筑也应该受到周边环境的影响，应当注意保持建筑物内部的安静，以防止突发事故导致人员伤亡。基础设施建筑应当对每一个居民进行配套的服务，确保每个居民都能得到满意的服务。
### 2.3.2 大楼建筑——社区居住环境的建设、经济效益的增长
大楼建筑的主要功能是提供居住环境。大楼的制造、安装、维修等操作人员都由专门的人员负责。大楼的建筑、装潢通常采用统一的标准和规范，并受到一定专业训练，保证产品质量。大楼的布局设计应当围绕居民群体来设计，保证居民生活便利。对于外来的游客来说，大楼通常提供周边景点的门票，让他们可以免费参观，享受到更加舒适的居住环境。大楼还可以促进人们之间的交流和团队合作。
### 2.3.3 商业中心建筑——产业链条的整合、商业模式的创新、经济效益的增长
商业中心的主要功能是促进产业链条的整合，提升经济效益。商业中心的布局设计应当满足不同行业、不同客户群体的需求，例如物流、金融、零售业、餐饮、酒店等各个行业都应当在中心位置设立。商业中心的规划应当考虑空间布局、投资、配套设施等多方面因素。商业中心的管理者应当精通市场营销、市场活动策划、财务决策等专业技能。商业中心的建设投资通常会较大，但其投入产出比仍然很高，是防范竞争的关键环节。
### 2.3.4 居民区建筑——社会凝聚力的提升、人际关系的重构、文化的传播
居民区建筑的主要功能是提供居民的生活环境。居民区的布局设计应当绿化居民区，使居民区居住环境整洁、美观、居家舒适。居民区建筑应当促进人们的社交互动，创建亲密的关系网络。居民区建筑还可以改善居民的个人生活，提升社会凝聚力和集体记忆力。居民区建筑通常与邻居区、城镇的建筑结合起来，共同提供人们必要的服务和资讯。居民区建筑的规划应当考虑居民生活的便利性、采光、交通、安保等因素，并充分考虑周边的环境影响。
