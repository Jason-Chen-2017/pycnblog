### 4.2 公式详细讲解

#### 4.2.1 线性回归

假设我们有 $n$ 个样本点 $(x_1, y_1), (x_2, y_2), ..., (x_n, y_n)$，线性回归模型为：

$$y = \beta_0 + \beta_1x$$

其中，$\beta_0$ 和 $\beta_1$ 分别为模型的截距和斜率。求解过程如下：

1. **最小二乘法**：求解使得预测值与实际值差的平方和最小的 $\beta_0$ 和 $\beta_1$。
2. **正规方程**：将线性回归问题转化为求解线性方程组。

#### 4.2.2 逻辑回归

逻辑回归是一种广义线性模型，用于预测二分类结果。假设我们有 $n$ 个样本点 $(x_1, y_1), (x_2, y_2), ..., (x_n, y_n)$，逻辑回归模型为：

$$P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x)}}$$

其中，$P(y=1)$ 表示 $y=1$ 的概率。求解过程如下：

1. **最大似然估计**：求解使得样本数据联合概率最大的 $\beta_0$ 和 $\beta_1$。
2. **梯度下降法**：求解最优化问题。

### 4.3 举例说明

#### 4.3.1 线性回归

假设我们有以下数据集：

| $x$ | $y$ |
| --- | --- |
| 1   | 2   |
| 2   | 3   |
| 3   | 4   |
| 4   | 5   |

求解线性回归模型：

1. **计算样本均值**：

$$\bar{x} = \frac{1}{n}\sum_{i=1}^{n}x_i = \frac{1+2+3+4}{4} = 2.5$$

$$\bar{y} = \frac{1}{n}\sum_{i=1}^{n}y_i = \frac{2+3+4+5}{4} = 3.5$$

2. **计算样本协方差和方差**：

$$\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y}) = (1-2.5)(2-3.5) + (2-2.5)(3-3.5) + (3-2.5)(4-3.5) + (4-2.5)(5-3.5) = -1$$

$$\sum_{i=1}^{n}(x_i - \bar{x})^2 = (1-2.5)^2 + (2-2.5)^2 + (3-2.5)^2 + (4-2.5)^2 = 5$$

3. **求解正规方程**：

$$\beta_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2} = \frac{-1}{5} = -0.2$$

$$\beta_0 = \bar{y} - \beta_1\bar{x} = 3.5 - (-0.2)\times 2.5 = 4$$

4. **线性回归模型**：

$$y = 4 - 0.2x$$

#### 4.3.2 逻辑回归

假设我们有以下数据集：

| $x$ | $y$ |
| --- | --- |
| 1   | 1   |
| 2   | 0   |
| 3   | 1   |
| 4   | 1   |

求解逻辑回归模型：

1. **定义损失函数**：

$$L(\beta_0, \beta_1) = -\sum_{i=1}^{n}[y_i\log(P(y=1)) + (1 - y_i)\log(1 - P(y=1))]$$

2. **计算梯度**：

$$\nabla L(\beta_0, \beta_1) = \left(\frac{\partial L}{\partial \beta_0}, \frac{\partial L}{\partial \beta_1}\right) = \left(\frac{1}{n}\sum_{i=1}^{n}(y_i - P(y=1)), \frac{1}{n}\sum_{i=1}^{n}(x_i(y_i - P(y=1)))\right)$$

3. **迭代更新**：

$$\beta_0 = \beta_0 - \alpha\nabla L(\beta_0)$$

$$\beta_1 = \beta_1 - \alpha\nabla L(\beta_1)$$

其中，$\alpha$ 为学习率。

## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型

在字节跳动校招中，常见的数学模型包括：

- **线性回归**：用于预测连续值。
- **逻辑回归**：用于预测二分类结果。
- **支持向量机**：用于分类和回归问题。
- **神经网络**：用于图像识别、自然语言处理等。

### 4.2 公式详细讲解

#### 4.2.1 线性回归

线性回归模型的公式为：

$$y = \beta_0 + \beta_1x$$

其中，$\beta_0$ 和 $\beta_1$ 分别为模型的截距和斜率。

#### 4.2.2 逻辑回归

逻辑回归的公式为：

$$P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x)}}$$

其中，$P(y=1)$ 表示 $y=1$ 的概率。

### 4.3 举例说明

#### 4.3.1 线性回归

假设我们有以下数据集：

| --- | --- |
| 1   | 2   |
| 2   | 3   |
| 3   | 4   |
| 4   | 5   |

通过最小二乘法，我们可以求解出线性回归模型的参数。

#### 4.3.2 逻辑回归

假设我们有以下数据集：

| $x$ | $y$ |
| --- | --- |
| 1   | 1   |
| 2   | 0   |
| 3   | 1   |
| 4   | 1   |
