## 2. 核心概念与联系

### 2.1 G-函数

G-函数是一种特殊的矩阵函数，通常与矩阵的特征值和特征向量相关。给定一个方阵\( A \)，G-函数定义为：

\[ G(A) = PDP^{-1} \]

其中，\( P \)是矩阵\( A \)的特征向量矩阵，\( D \)是对角矩阵，其主对角线上的元素是矩阵\( A \)的特征值。G-函数的一个重要性质是，它可以将任意矩阵转化为对角矩阵，从而简化矩阵的计算和分析。

### 2.2 非奇异M-矩阵

非奇异M-矩阵是一类特殊的方阵，其特征值全为正。具体来说，如果一个方阵\( A \)的所有特征值都大于零，那么它就是一个非奇异M-矩阵。非奇异M-矩阵在优化问题和线性方程组的求解中具有重要意义，因为它保证了算法的稳定性。

## 3. 核心算法原理 & 具体操作步骤

### 3.1 矩阵特征分解

矩阵特征分解是G-函数的基础。给定一个方阵\( A \)，我们需要找到它的特征向量\( P \)和特征值\( D \)。具体步骤如下：

1. **求特征方程**：解方程\( |A - \lambda I| = 0 \)，得到特征值\( \lambda \)。
2. **求特征向量**：对于每个特征值\( \lambda \)，求解方程\( (A - \lambda I)X = 0 \)，得到对应的特征向量\( X \)。
3. **构建特征向量矩阵**：将所有特征向量按对应特征值排序，组成特征向量矩阵\( P \)。

### 3.2 G-函数的计算

利用特征分解的结果，我们可以计算G-函数。具体步骤如下：

1. **构建对角矩阵**：将特征值按顺序填入对角矩阵\( D \)。
2. **计算G-函数**：\( G(A) = PDP^{-1} \)。

### 3.3 非奇异M-矩阵的判断

判断一个方阵是否为非奇异M-矩阵，我们需要检查其特征值。具体步骤如下：

1. **计算特征值**：使用特征分解或数值计算方法得到方阵的特征值。
2. **判断特征值**：如果所有特征值都大于零，则方阵为非奇异M-矩阵。

### 4.2 公式解释

- **G-函数公式**：\( G(A) = PDP^{-1} \)，其中\( D \)是对角矩阵，主对角线上的元素是\( A \)的特征值。

### 4.3 举例说明

#### 4.3.1 G-函数举例

考虑以下方阵\( A \)：

\[ A = \begin{bmatrix} 2 & 1 \\ 0 & 2 \end{bmatrix} \]

我们首先求出特征值和特征向量：

- **特征方程**：\( |A - \lambda I| = \begin{vmatrix} 2 - \lambda & 1 \\ 0 & 2 - \lambda \end{vmatrix} = (\lambda - 2)^2 = 0 \)，得到特征值\( \lambda_1 = \lambda_2 = 2 \)。
- **特征向量**：对于特征值\( \lambda = 2 \)，解方程\( (A - 2I)X = 0 \)，得到特征向量\( X_1 = \begin{bmatrix} 1 \\ 0 \end{bmatrix} \)和\( X_2 = \begin{bmatrix} 0 \\ 1 \end{bmatrix} \)。
- **特征向量矩阵**：\( P = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix} \)。
- **对角矩阵**：\( D = \begin{bmatrix} 2 & 0 \\ 0 & 2 \end{bmatrix} \)。
- **G-函数**：\( G(A) = PDP^{-1} = A \)。

#### 4.3.2 非奇异M-矩阵举例

考虑以下方阵\( B \)：

\[ B = \begin{bmatrix} 3 & 2 \\ 1 & 3 \end{bmatrix} \]

我们首先求出特征值和特征向量：

- **特征方程**：\( |B - \lambda I| = \begin{vmatrix} 3 - \lambda & 2 \\ 1 & 3 - \lambda \end{vmatrix} = (\lambda - 3)^2 - 2 = \lambda^2 - 6\lambda + 5 = 0 \)，得到特征值\( \lambda_1 = 1 \)和\( \lambda_2 = 5 \)。
- **特征向量**：对于特征值\( \lambda = 1 \)，解方程\( (B - I)X = 0 \)，得到特征向量\( X_1 = \begin{bmatrix} -2 \\ 1 \end{bmatrix} \)；对于特征值\( \lambda = 5 \)，解方程\( (B - 5I)X = 0 \)，得到特征向量\( X_2 = \begin{bmatrix} 1 \\ 0 \end{bmatrix} \)。
- **特征向量矩阵**：\( P = \begin{bmatrix} -2 & 1 \\ 1 & 0 \end{bmatrix} \)。
- **对角矩阵**：\( D = \begin{bmatrix} 1 & 0 \\ 0 & 5 \end{bmatrix} \)。
- **非奇异M-矩阵判断**：因为特征值全为正，\( B \)是一个非奇异M-矩阵。

### 5.2 源代码详细实现和代码解读

#### 5.2.1 G-函数计算

以下是一个Python脚本，用于计算方阵的G-函数：

```python
import numpy as np

def g_function(A):
    eigenvalues, eigenvectors = np.linalg.eig(A)
    D = np.diag(eigenvalues)
    G = eigenvectors @ D @ eigenvectors.T
    return G

# 示例
A = np.array([[2, 1], [0, 2]])
G_A = g_function(A)
print("G(A):", G_A)
```

**代码解读**：
- `np.linalg.eig(A)`计算方阵\( A \)的特征值和特征向量。
- `D = np.diag(eigenvalues)`创建对角矩阵\( D \)，主对角线上的元素是特征值。
- `G = eigenvectors @ D @ eigenvectors.T`计算G-函数。

#### 5.2.2 非奇异M-矩阵判断

以下是一个Python脚本，用于判断方阵是否为非奇异M-矩阵：

```python
import numpy as np

def is_nonsingular_m_matrix(A):
    eigenvalues = np.linalg.eigvals(A)
    return np.all(eigenvalues > 0)

# 示例
B = np.array([[3, 2], [1, 3]])
print("Is B a non-singular M-matrix?", is_nonsingular_m_matrix(B))
```

**代码解读**：
- `np.linalg.eigvals(A)`计算方阵\( A \)的特征值。
- `np.all(eigenvalues > 0)`判断所有特征值是否大于零，返回True或False。
