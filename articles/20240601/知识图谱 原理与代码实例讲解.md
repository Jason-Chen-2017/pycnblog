知识图谱由节点和边组成，节点表示实体（例如，人物、地点、事件等），边表示实体之间的关系（例如，出生、住在、参与等）。为了使知识图谱具有实际用途，我们需要将知识图谱与现实世界的数据源（例如，数据库、API、文件等）集成。

知识图谱的核心概念包括：

1. 节点：表示实体。
2. 边：表示实体之间的关系。
3. 属性：表示实体的属性。

知识图谱的核心联系包括：

1. 实体之间的关系：知识图谱可以用来表示实体之间的各种关系，如“张三是李四的朋友”、“北京位于中国”等。

## 项目实践：代码实例和详细解释说明

要构建知识图谱，我们需要使用一些编程语言和库来实现。以下是一个使用Python和NetworkX库构建知识图谱的代码实例：

```python
import networkx as nx

# 创建一个图
G = nx.Graph()

# 添加节点
G.add_node("A")
G.add_node("B")

# 添加边
G.add_edge("A", "B")

# 查询节点间的关系
print(G.edges("A"))
```
