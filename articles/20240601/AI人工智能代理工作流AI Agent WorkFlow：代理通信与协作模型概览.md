## 1. 背景介绍

人工智能代理通信（AI Agent Communication）是研究人工智能代理如何通过各种形式的通信与协作来实现共识、协作与决策的领域。人工智能代理工作流（AI Agent WorkFlow）是人工智能代理通信的重要组成部分，它定义了人工智能代理如何协作以实现特定的任务和目标。

## 2. 核心概念与联系

人工智能代理工作流的核心概念是代理通信和协作模型。代理通信指的是人工智能代理之间通过各种形式（如语义、语法、信号等）进行信息交换。协作模型则定义了人工智能代理如何协同工作以实现共同目标。

人工智能代理工作流的核心概念与联系包括：

* 代理通信：人工智能代理之间的信息交换
* 协作模型：人工智能代理协同工作的方式
* 共识：人工智能代理之间的信息共享
* 决策：人工智能代理共同制定的决策
