# 2.面试技巧
1. 问问题之前先了解自己的情况和意图，回答者容易受到个人信息的干扰，造成沟通不畅；
2. 把握自己的优势，多提问可以增强被问者的乐趣，同时也可以锻炼自己；
3. 用“你”而不是“我”，用具体事实和数据说话；
4. 针对性的提问，选择最有可能回答的问题；
5. 仔细的回答问题，不仅可以增加赢得面试官掌声，也会让面试者有更深刻的理解；
6. 记得向面试官展示你的勤奋努力。

# 3.自我评估
1. 具备相关的工作经验和职称证书；
2. 有担任优秀的职务或职位的历史；
3. 精通自己所负责的行业的产品或服务的运营、市场推广等方面；
4. 有良好的道德品质和职业操守；
5. 工作认真负责，有较高的责任意识；
6. 拥有较高的敬业精神和较强的执行能力；
7. 在一定的时限内完成一定的任务。

# 4.学习方法
1. 每天阅读大量的有关该职位的报刊杂志和文章，关注所需的领域的信息；
2. 关注招聘会、期刊、网路讨论板块，参加各种交流活动，倾听听取别人的意见；
3. 花时间和精力投入到学习中，培养知识的兴趣和能力；
4. 为个人发展规划，制定并持续改善自身的职业发展计划。

# 5.谈薪资
1. 界定底薪和年终奖的比例；
2. 考虑工作压力，根据公司经济状况及岗位要求确定底薪和每月奖金；
3. 提供详细的薪酬构成，包括福利、房补、股票分红等；
4. 不要忽视员工的业绩评价，不宜炒作薪酬；
5. 对于小型创业公司来说，适当降低薪水也可缓解业务推进上的难题。

# 6.工作态度
1. 恭喜面试通过的人员要立即展开工作动作；
2. 提升自己的技能和个人魅力，树立自己的形象；
3. 沟通能力和组织能力都很重要，要做好周密的思考和决策；
4. 合理安排时间和精力，不要浪费精力在无意义的会议上；
5. 要适当休息，保持充足的睡眠。
