# 2.基本概念术语说明
## 2.1 Chatbot简介
### 什么是Chatbot？
Chatbot（中文名：聊天机器人），是一种通过与人类互动进行文字、语音或图像形式交互的“小型计算机应用程序”。这种机器人可以通过与人类进行即时的、不依赖于人类参与的交谈，达成目的并完成任务。Chatbot的出现将信息服务和智能设备连接在一起。它可以帮助用户快速获取所需的信息，提升工作效率，提高个人品牌形象。其核心能力是：与人类进行 conversation （会话）、回答用户的问题、根据上下文转换语言、做出决策。

2016年，Wired杂志上曾经报道过，世界上有超过5亿的个人电脑属于企业，大约90%的员工都配备了智能手机。据估计，全球每年产生超过200亿条消息，其中超过五分之一来自智能手机。传统的应用软件、网页和服务已无法满足当下信息爆炸的需求。

Chatbot作为新兴的IT技术之一，其能力远远超出了传统应用软件。为了适应智能手机对信息消费的需求，Facebook、微软、Google、Amazon等科技巨头纷纷推出基于Web和移动端的Chatbot平台，用户可以使用类似微信或WhatsApp的界面与机器人进行即时通信。例如，Facebook Messenger、Telegram Bot、Viber Bot和Microsoft Bot Framework都是非常受欢迎的Chatbot平台。

Chatbot可以辅助企业或组织改善内部流程、提升客户满意度，并降低人力成本。2016年，Verintelligence研究院发布了一份报告，称Chatbot将成为未来营销方式的重要组成部分。Chatbot可以为企业节省30%~50%的人力成本，同时还可以帮助企业打造忠诚度、留存率、促活跃度。因此，Chatbot的蓬勃发展势必为社会带来新的机遇。

## 2.2 AI简介
### 什么是人工智能（Artificial Intelligence，AI）？
人工智能（AI）是一个用人类智慧构造的计算系统，是指让机器拥有与人类一样的能力、运作方式。它是一种通用的、高级的计算机技术，包括概率统计理论、模式识别、自然语言处理、机器学习、计算机视觉、自动决策等，主要应用于解决各种复杂的、日益增长的计算任务。

2017年9月，艾伦·图灵宣布，他在麻省理工学院创建的“图灵测试”已经通过。这项测试就是为了证明计算机智能是否真正存在。如果一个人类的智能没有被计算机完全模拟，那么图灵测试就不会通过。因此，人工智能技术或将引领产业的变革。

人工智能的进步离不开数据的积累和应用。数据量越来越庞大，海量数据的处理才能构建出能够理解、学习和应用的数据模型。而对于AI来说，它的训练数据集也越来越庞大，如图像、文本、声音等多种形式的数据，这些数据所包含的知识越来越丰富。

随着机器学习、深度学习等技术的不断发展，人工智能已然成为影响力很大的领域。可以预见，未来AI的发展将对整个行业产生深远的影响。

### Chatbot与AI的区别与联系
Chatbot和AI的定义和功能相同。它们之间最大的不同在于，Chatbot是一个应用软件，由AI驱动，能够在与用户进行即时通信中提供有效的服务。Chatbot和AI之间的关系可以简单归纳为一句话——Chatbot = AI + Application Software 。

比如，早期的搜索引擎机器人就是用人工智能技术实现的，但由于其垂直性和局限性，逐渐转向了对话系统和聊天机器人。现在，聊天机器人已经发展成了一个综合性的IT技术，涵盖了许多领域，如金融、保险、美容等多个行业。除此之外，Chatbot还可以为企业提供业务咨询、调查研究、售前支持、售后服务等各种服务。

## 2.3 Facebook AI产品
### Facebook AI产品概览
Facebook AI产品分为三层：
1. Applications Layer：包括Messenger Platform、Workplace、Oculus Rift VR平台，用于构建智能客服、企业聊天机器人等应用。
2. Services Layer：包括Vision API、Speech API、Natural Language API等，可提供图像识别、语音识别、自然语言处理等技术能力。
3. Data Layer：包括TensorFlow、Torch等开源框架，可提供对语料库的分析、聚类、分类、推荐等技术能力。

Messenger Platform是Facebook旗下的智能对话平台，具有优秀的功能特性，包括安全、隐私、性能等方面的考虑。该平台由三个模块构成：API Gateway、Bot Manager、Conversation Manager，分别用于接口网关、机器人管理、对话管理。

Oculus Rift VR平台是Facebook发布的一款VR设备，支持使用户在虚拟现实中与智能机器人进行交流。该平台有助于企业建立品牌认知，扩大获客渠道，提高收入，促进商业变现。

### Messenger Platform
Facebook Messenger 是 Facebook 旗下提供给用户的智能对话平台。Messenger 的功能非常强大，能够与用户直接进行互动。除了具有网页版的交互体验，Facebook Messenger 还提供了 iOS 和 Android 应用，可以通过手机上的语音输入、自定义表情符号等方式进行交互。除了聊天功能，Messenger 提供了与 Facebook 生态系统相关的一些功能，如支付、分享等。

Messenger Platform 通过 Bot Manager 管理多个 Bot ，每个 Bot 可以为特定的用户群提供服务。其中，有些 Bot 支持有限的交互，例如只支持查询、问答类型的问题，其他 Bot 可以提供更为丰富的功能，例如提供预订机票、餐饮订购等。为了保证服务质量，Facebook 还提供企业认证机制，要求 Bot 具备法律规定的合法身份，并通过质量审核之后才能投放到生产环境中。

### Oculus Rift VR平台
Oculus Rift VR 是 Facebook 提供的第三方 VR 平台。该平台采用的是 HTC Vive 的硬件设备。Vive 相对于以往的 HTC 控制器要好很多，能够提供更加自由的空间导航。Oculus Rift VR 将为用户提供广泛的虚拟现实社交和娱乐场景，用户可以利用该平台进行交流、娱乐、学习、工作等。

Oculus Rift VR 平台提供了一个优雅的虚拟现实视角，可以看到整个现实世界。由于 Vive 使用 6 个摄像头，可以捕捉周围环境的深度影像，还可以为用户提供交互体验。Facebook 还提供了 AR 技术，允许用户创建和分享自己创作的虚拟物品。
