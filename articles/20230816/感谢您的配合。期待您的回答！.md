# 2.基本概念术语说明
首先，我们需要对机器学习、深度学习、神经网络等术语做一个简单的介绍。
## 2.1 机器学习
机器学习 (Machine Learning) 是指计算机系统通过训练与反馈来改进性能的科学方法。它所涉及到的算法和技术使计算机系统可以自动化地学习，从而发现数据的结构和模式，从而在未知环境下做出预测或决策。
机器学习包含以下五个主要步骤：

1. 数据收集：从互联网、数据库或其他来源获取数据；

2. 数据清洗：删除异常值、噪声数据等；

3. 数据准备：对数据进行特征工程、归一化处理、数据分割等；

4. 训练模型：选择模型算法，设置超参数，训练模型；

5. 测试模型：利用测试集评估模型效果。

其中，第一步、第二步以及第三步通常是数据预处理的过程，往往会涉及到多种工具、库和技巧。第四步和第五步则是构建机器学习模型的关键步骤。
## 2.2 深度学习
深度学习 (Deep Learning) 是一种基于神经网络的机器学习技术。深度学习通过多层非线性变换对输入进行映射，使得模型能够解决复杂的问题。它在多个任务上取得了非常好的效果，如图像分类、文本识别、语音识别、视频分析等。
深度学习的主要特点如下：

1. 使用深度学习框架构建模型；

2. 使用 GPU 或 TPU 的计算加速；

3. 数据增强、Dropout 等技术提升模型鲁棒性。

## 2.3 神经网络
神经网络 (Neural Network) 是由多个节点组成的有限元素的网络，是模拟人大脑神经元网络而形成的模拟理论模型。神经网络的原理是接收输入信号，根据输入信号与权重矩阵的乘积，产生输出信号。神经网络的每个节点都具有输入信号、输出信号和权重。输入信号经过加权得到输出信号。
每层神经网络中的节点有两种状态，分别是激活（Activation）和不激活（Inactivation）。当输入信号与权重的乘积超过阈值时，该节点就会被激活。如果某个节点的激活程度过低或者连接权重过小，那么该节点就会失去作用，成为不激活状态。不激活状态的节点不会影响后续节点的输入。
在神经网络的训练过程中，输入数据将经过多个隐藏层（Hidden Layer）的计算，最后输出预测结果。隐藏层中的节点是神经网络的核心部分，它们的激活状态决定着整个神经网络的预测准确率。
## 2.4 特征工程
特征工程 (Feature Engineering) 是指通过某些手段从原始数据中提取有效信息，并转换成适合机器学习使用的形式的过程。特征工程是机器学习中最重要的一环，它对输入数据进行抽象、转换、标准化等操作，最终生成训练集。
特征工程的目的是提取数据中对目标变量的相关性很强的特征，从而降低机器学习模型的难度和计算量，提高模型的泛化能力。特征工程往往会涉及到诸如数据探索、数据清洗、特征提取、特征选择等环节。
## 2.5 监督学习、无监督学习、半监督学习
监督学习 (Supervised Learning) 是指有标注的数据用于训练模型，根据已知的标签对数据进行学习，用于分类、回归、聚类、回归等任务。监督学习的典型问题有分类、回归、标记词袋模型、序列标注模型等。
无监督学习 (Unsupervised Learning) 是指没有标注的数据用于训练模型，算法对数据自身的结构进行分析，发现数据中的隐藏关系，用于聚类、Density Estimation、Dimensionality Reduction 等任务。无监督学习的典型问题有聚类、密度估计、降维等。
半监督学习 (Semi-supervised Learning) 是指有部分样本带有标签，另外部分样本没有标签，算法可以在有标签数据和无标签数据之间建立联系，利用部分样本的信息进行训练。
## 2.6 回归、分类、聚类、降维等问题
回归 (Regression) 是指对连续变量进行预测。回归模型在一定条件下可以近似表示为一条直线。
分类 (Classification) 是指对离散变量进行预测。分类模型在输入空间中划分成多个区域，并据此赋予样本的类别标签。
聚类 (Clustering) 是指将相似的数据集合到一起，类似于自动化的排序过程。聚类模型发现不同子空间中的模式，并将它们合并到同一个集群中。
降维 (Dimensionality Reduction) 是指从高维数据中找到线性组合的低维表示，以便于数据的可视化和分析。降维模型保留重要的特征信息，降低模型复杂度。
