# 1.简介
  

## Sharing economy
Sharing economy是一个颠覆性的经济模式，它使得人们可以在不购买商品的前提下，共享互联网资源，如时间、空间、网络带宽等，甚至可以免费获得服务。Sharing economy服务是由应用平台和提供者共同合作开发的，用户不需要进行安装，只需要连接到平台上就可以享受到这些服务。Sharing economy的一个重要特点是服务提供者通过提供高质量服务，消除用户之间的交易障碍，将更多价值添加到用户的生活中。
## Meituan Dianping’s Self-Driving Delivery Service (自动驾驶货运)
Meituan Dianping集团自主研发了自动驾驶货运技术，致力于提供在线用户零接触、无摩擦、高效率的全城通达及日间通达，并通过科技赋能改善交通运输体系，实现智慧地产的自动化、智能化、绿色化。基于共享经济模式，Meituan Dianping为其合作伙伴提供降低成本、提升效率、提升生活质量的自动驾驶系统。而Meituan Dianping作为共享经济的服务提供方，还会和相关合作方建立起长期稳定的合作关系。
### Self-Driving Delivery System (自动驾驶货运系统)
所谓自动驾驶货运系统就是一个基于机器学习和计算机视觉的自主驾驶平台，能够实时掌握用户的当前位置信息、目标地址以及车辆状态等信息，利用图像识别和深度神经网络技术，帮助用户准确规划出最短路径，然后再使用激光雷达或雷达感应器制造整体的路径规划。这个系统将通过扫描路段的红绿灯信号，判断该路段是否拥堵、堵塞程度以及停止行人的数量，根据不同的情况调节行驶方向，从而实现智能化、规划化、高效率的自动驾驶货运。
#### Key Points:
1. 用现有地图数据分析出道路状况，用图像识别和深度神经网络算法，预测路径。
2. 将路况、预测路线以及环境因素结合起来，生成最终的控制命令。
3. 设计路标引导用户行动。
4. 建立自动驾驶系统与相关生态系统之间的沟通机制，满足多种平台的需求。
5. 与第三方平台合作开展商务、市场推广活动。
6. 提供专业、安全、可靠的服务，为消费者提供最佳体验。
