# 2.KNN算法介绍
## 2.1 KNN算法的概念
KNN(k-Nearest Neighbors)算法是一种简单而有效的分类方法。KNN算法的基本思想是如果一个样本在特征空间中的k个最邻近的样本中的大多数属于某一类别，则该样本也属于这个类别。可以认为KNN算法简单而直观，就是用距离来衡量样本之间的相似性。

KNN算法的输入为n维特征向量X={x1, x2,..., xi}，输出为对应的类别y，k表示用于分类的最近邻个数。首先，计算待分类样本与各样本之间的距离；然后按照距离递增次序排序；再选取与目标样本距离最小的k个点；最后统计k个点所在类别数量并赋予给目标样本的类别。
## 2.3 KNN算法的优缺点
### 2.3.1 KNN算法的优点
1. 易于理解和实现：KNN算法的理论基础简单，易于理解和实现。
2. 无参数调整：KNN算法没有显式的超参数，不需要进行参数调优。
3. 处理高维数据：KNN算法可以处理高维数据的情况。
4. 结果可靠性高：KNN算法的结果受到训练集数据分布的影响较小。
### 2.3.2 KNN算法的缺点
1. 计算复杂度高：KNN算法的时间复杂度为O(nlogn)，当数据量很大时，KNN算法的训练时间和预测时间都比较长。
2. 次级问题：KNN算法对于离群点敏感，对异常值不敏感。
3. 数据稀疏时表现差：KNN算法对数据稀疏时表现不好。
4. 不适合多分类任务：KNN算法只适合二分类和回归任务。
5. 模型容易过拟合：KNN算法容易出现过拟合现象。
## 2.4 KNN算法的应用场景
KNN算法主要用于分类和回归问题。一般情况下，KNN算法用于分类问题是因为输入空间是由一组向量组成的，每个向量代表了一个样本的特征向量，不同类的样本存在差异化，因此可以利用KNN算法将样本划分为不同的类别。另一方面，KNN算法还可以用于回归问题，如根据手指尖位置预测手掌位置。
## 2.5 KNN算法的局限性
1. 对异常值敏感：KNN算法对于异常值的容忍度不够。
2. 只考虑了欧氏距离：KNN算法只考虑了欧氏距离作为距离度量方式，不具备更一般意义上的距离度量能力。
3. 对样本数量要求高：KNN算法对样本数量的依赖性比较强，即K值越大，对样本越敏感。
4. 不适合密集分类任务：KNN算法对密集分类任务效果不佳。
