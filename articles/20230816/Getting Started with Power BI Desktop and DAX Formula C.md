# 1.简介
  

Power BI Desktop是一个强大的商业智能工具，能够帮助用户将复杂的数据源导入到数据集中并进行分析处理，生成高质量的报表和可视化效果。本文将详细介绍如何利用DAX语言构建更加丰富、精准的计算模型，并展示其在数据分析中的实际应用。该文主要面向具有一定工作经验但又对业务分析或数据科学感兴趣的初级用户，包括但不限于程序员、数据科学家、商业分析人员等。
# 2.Power BI的基本概念和术语
## Power BI产品简介
Power BI（以下简称“Power BI”）是微软推出的一款基于云端的商业智能工具，提供商业用户进行快速洞察、建模和呈现数据的能力，帮助企业实现决策支持和可视化分析目的。其中，“Desktop”可以指代本地安装的软件版本，而“Online”则指的是基于Web的服务。目前，在线版本也正逐渐成为主流。 

Power BI Desktop提供了多个功能模块：
- 数据获取：通过连接各种数据源（如关系数据库、Excel文件、Oracle数据库、云存储等），可获取到原始数据；
- 数据预处理：对数据进行清洗、转换、规范化、合并等预处理工作，确保数据准确完整；
- 数据模型设计：定义数据模型，用于数据分析和报表生成；
- 数据可视化：采用多种图表类型（如柱状图、折线图、散点图等）及颜色编码，对数据进行直观的呈现；
- 报告生成：可通过类似Word或Excel的界面，根据数据模型生成专业的报表。

Power BI Online提供免费试用版，同时也有付费套餐供购买。用户可以通过创建自己的工作区和数据集来保存和分享自己的项目，协同合作。不过，个人版的免费试用版就够用了。
## DAX语言简介
DAX（Data Analysis Expressions）语言是一种声明式的表达式语言，用于在Power BI Desktop等商业智能工具中定义复杂的计算逻辑。DAX语法类似于SQL语言，但略有不同。

例如，下面的DAX表达式求得“销售额”列中所有非零值的总和：
```dax
SUM(FILTER(Sales, Sales[Amount] <> 0))
```

DAX语言涵盖了Power BI的所有功能模块，包括数据预处理、数据模型设计、数据可视化、报告生成等。它可以用变量表示数据（如关系表中的列），并通过函数、运算符、过滤器、公式等运算符执行复杂的计算。另外，DAX还内置了丰富的统计、数学、文本、日期和时间函数，以及用于数据整理、过滤和修改的内置函数库。

因此，只要掌握了DAX的基础知识和语法规则，就可以轻松地创建、编辑和维护复杂的商业智能模型。
## Power BI Desktop的配置与安装
### 安装与激活
Power BI Desktop可以在Windows、Mac OS X和Linux平台上安装运行。无需安装其它第三方软件，直接下载、解压后即可运行。由于需要联网才能正常使用，建议安装最新版本。


点击下载按钮后，选择适合自己电脑系统的安装包，双击打开安装程序，按照提示一步步完成安装过程。


安装完成后，启动Power BI Desktop应用程序。第一次启动时会出现许可协议界面。点击“接受”，然后系统会自动检测更新。如果没有找到新版本，则会提示“已经是最新版本”。


关闭许可协议窗口后，屏幕左侧的导航栏就会出现“文件”菜单，里面包含最常用的功能选项。


如果需要激活Power BI Desktop，则点击右上角的“？”，选择“激活”选项。输入你的Microsoft 365邮箱地址，点击“确定”后，系统会发送一封验证邮件到你的邮箱里。点击邮件中的链接，登录到Office 365管理员中心，找到“使用情况和报告”页面。勾选Power BI Desktop复选框，点击“应用”按钮。


激活成功后，Power BI Desktop的状态栏上会显示“已激活”。
### 设置默认语言
点击导航栏左边的“文件”选项卡，然后点击顶部菜单栏的“选项”按钮。进入设置页面后，在左侧菜单栏中点击“区域和语言”。点击“管理”按钮。


在弹出的设置窗口中，选择“系统语言”和“日期和时间格式”两项。


点击“应用”按钮，系统会提示是否重新启动Power BI Desktop，确认后重启。重新启动后，默认语言和日期时间格式都会被应用。
