# 1.简介
  

## 什么是生成式对抗网络？
生成式对抗网络(Generative Adversarial Network, GAN) 是由Radford等人于2014年提出的一种基于神经网络的无监督学习方法。它可以用于生成并驱动数据分布，或者在训练过程中使得网络产生真实的样本而不是只是噪声。其理论基础是对抗训练，即两个网络相互博弈，一个网络生成虚假的样本，另一个网络通过判别器判断该样本是否是真实的，并将其误分类的样本作为反馈提供给生成器让其继续训练，直到生成器逐渐完善数据的生成过程。

## 为什么要用GAN进行缺陷检测？
在缺陷检测领域，传统的方法主要依靠规则、统计模式等手段进行识别。这些方法通常需要较高的准确率以及人工标注的数据集才能取得比较好的效果。而生成式对抗网络（GAN）可以在训练过程中自动生成假样本，从而完全摆脱了繁琐的人工标注过程，能够达到更高的效率。而且，GAN的生成模型可以捕捉到特征之间的复杂关系，并且在缺陷检测中可以有效地发现模式。因此，基于GAN的缺陷检测模型具有以下优点：

1. 模型简单易懂：GAN模型是现代深度学习的一个分支，并且生成模型是简单的多层感知机，很容易理解；
2. 普适性强：GAN模型不仅适用于图像处理任务，还可以扩展到其他领域；
3. 可解释性好：生成模型可以提供对每个特征的解释，并且可以帮助我们理解底层特征的作用；
4. 鲁棒性强：GAN模型可以应对模型欠拟合和过拟合的问题。

# 2. 基本概念术语说明
## 数据集
数据集包括训练集、验证集、测试集，每一个数据集都是一个包含多个样本的集合。在缺陷检测中，我们一般只需要训练集和测试集，其中训练集用于训练模型参数，测试集用于评估模型的性能。训练集中包含正常样本、缺陷样本以及异常样本。

## 深度学习
深度学习是机器学习中的一个子分支，其利用计算机模拟人类的学习行为，以此来解决非线性方程或非凸优化问题。深度学习的相关研究取得了巨大的进步，其在图像、文本、语音、视频等领域均有着广泛的应用。

## 生成器与判别器
在深度学习的GAN框架下，生成器与判别器是两种不同的神经网络结构，分别用于生成与鉴别真实样本。生成器的目标是在判别器无法区分真实样本与生成样本的情况下，输出尽可能真实且可信的样本。判别器的目标则是识别输入样本是真实样本还是生成样本。


### 生成器
生成器是一个无参的神经网络，它的输入是随机向量，输出是由低维空间映射到的高维空间中的样本，这个过程叫做生成。生成器的目标就是尽可能模仿数据分布，从而生成高质量的样本。

### 判别器
判别器是一个带参的神经网络，它的输入是一组样本，输出是它们是来自数据分布的样本的概率。判别器的目标就是区分生成样本和真实样本。

## 对抗训练
在对抗训练过程中，生成器与判别器相互竞争，使得生成器通过迭代优化输出越来越接近真实样本，从而使判别器无法再分辨真实样本与生成样本。在这种情况下，生成器会被迫生成与真实样本距离较远的样本，这样就可以提升模型的鲁棒性和健壮性。
