# 1.简介
  

## 1.1 关于机器学习（ML）
什么是机器学习？它是一个广义上的概念，用来描述计算机如何学习、怎样学习、为什么要学习以及学到什么程度。机器学习作为一个领域，涵盖了许多不同的子领域，比如监督学习、无监督学习、强化学习、特征工程等等。

最常用的机器学习任务之一，就是分类任务。在这个任务中，输入的数据被分成多个类别，然后基于这些数据训练出一个模型，使得该模型能够准确预测新的输入数据的类别。例如，给定一张图片，识别出它是狗还是猫。

随着时间的推移，人们越来越重视机器学习的能力。近年来，传统的统计方法已经无法应对越来越复杂、高维、多源异构、多模态、海量数据的挑战。因此，越来越多的人转向机器学习。

## 1.2 关于深度学习（DL）
什么是深度学习？它是一种赋予计算机以学习深层次模式的机器学习方法。深度学习通常比传统的机器学习更好地理解和处理高维度数据。

深度学习的关键技术是深度神经网络（DNN）。DNN由一系列的简单但高度非线性的神经元组成，通过不断堆叠、连接并联合的方式学习数据表示形式。每一层都可以看做是一个转换函数，将前一层的输出映射到后一层的输入空间。从而使得模型具备学习深层次、抽象、非线性表示的能力。

除了DNN之外，还有一些其他的深度学习方法，比如卷积神经网络（CNN），循环神经网络（RNN），变分自动编码器（VAE）等。不过，目前最流行的是DNN。

# 2.深度学习的分类
深度学习的目标是解决某个具体的问题，可以分成两大类——监督学习和无监督学习。

1. 监督学习
    深度学习的监督学习一般包含两个阶段——预训练和微调。
    - 预训练（Pre-training）
        预训练是指在大规模未标注的数据上进行特征提取，取得预训练权重，再fine-tuning的过程，即先使用大量未标注数据训练深度神经网络，提取其中的特征，然后利用这些特征做特定任务的训练。如BERT、ALBERT等模型。

    - 微调（Fine-tuning）
        在有限的预训练数据上采用小量已标注数据来微调网络参数，达到更好的效果。一般来说，微调的目的是为了更好地适配当前任务所需的数据集和优化目标。如bert_base的微调。

2. 无监督学习
   - 概念：在无监督学习中，数据没有标签，只有输入数据，目的是寻找数据的结构和规律，用来发现数据的内部关系。常见的无监督学习模型有聚类分析K-means、深度学习的变分自编码器VAE、EM算法、GAN等。

   - 用途：无监督学习旨在发现数据的内在结构，并对数据的质量产生质疑，以此来发现潜在的结构模式和隐藏模式，用于评估真实世界的数据质量。
   - 示例：图像和文本数据可以作为无监督学习的输入，探索数据的内在分布模式，以此来发现图像或文本的潜在主题或风格。视频、声音、社会网络数据也可以被用于探索其内部分布结构。
   - 优点：不需要标签的输入数据能有效地进行数据分析和特征提取，从而获得关于数据的知识和洞察力，为机器学习提供有价值的参考信息。同时，使用无监督学习模型，可以节省大量的时间，而且有利于解决一些“大数据”问题。
   - 缺点：由于没有给定的标签，无监督学习模型很难评估它们是否准确，也不能很好地评估它们所捕获的信息。因此，无监督学习模型通常只适合于某些特定的任务。

总结一下，深度学习可以分为两大类——监督学习和无监督学习。监督学习是在已知输入数据与输出数据的情况下，训练模型学习数据的映射关系；无监督学习则是由输入数据得到输出数据，但是输入数据本身可能没有显式的标签。
