# 1.简介
  

## 数据可视化
数据可视化（Data Visualization）是指对数据进行图形化、图像化处理的过程。数据可视化技术可以帮助用户更好地理解和分析数据。它能够使人们快速识别出模式、发现异常值、理解业务变化，并在一定程度上帮助决策制定者做出明智的决策。数据可视化领域也是很多人才向往的领域之一，越来越多的人喜欢用自己的眼睛去看待世界，这也是数据可视ization的一大优点。

### 数据可视化领域的应用场景
数据可视化领域经过近十年的发展，得到了飞速的发展，各个领域都涌现出了许多炙手可热的数据可视化产品，包括银行业、零售业、医疗健康、交通、房地产等各行各业。不同行业、不同的场景、不同的需求，数据的形式也会有所不同。例如，金融数据主要以柱状图、折线图和饼图等形式呈现；电信运营商的数据则更多地采用柱状图或面积图；房地产商的数据则更倾向于采用词云、散点图、热力图等类型。因此，数据可视化不仅仅局限于传统的数据分析阶段，它还扮演着非常重要的角色，帮助企业实现数据价值的转化，提升公司竞争力。

目前，数据可视化工具市场的供需平衡趋势已经形成，企业与个人对数据可视化工具的需求已逐渐趋缓。据调研机构预测，到2025年，全球数据可视化工具的数量将达到4万亿美元，约占全球GDP的0.2%。由于数据可视化行业一直在蓬勃发展，所以，不可否认的是，数据可视化工具是一个复杂的产品及服务组合。

本文介绍的数据可视化工具——Tableau，其作为一种商业智能数据可视化工具，被广泛应用于各行各业。本文介绍的内容主要是基于Tableau 2020最新版本进行的。

## Tableau
Tableau 是一款商业智能软件工具，它能够通过直观的方式呈现数据、分析数据、制作报告。Tableau 的特点包括：

1. 可视化界面：Tableau 提供了丰富而灵活的可视化界面，从简单到复杂，无论您要分析的数据量有多大，都可以轻松应付；

2. 数据源支持：Tableau 可以直接从各种数据源导入数据，包括关系数据库、Excel、Spreadsheets、文本文件、JSON 文件等；

3. 报表制作：Tableau 提供丰富的可视化图表、矩阵、地图、动画、嵌入式计算组件等，能够满足您的多种创作要求；

4. 强大的分析功能：Tableau 在数据探索方面提供了多种分析函数，如聚类分析、关联分析、回归分析、分箱分析等，能帮您发现数据的隐藏模式；

5. 便捷协作能力：Tableau 提供团队协作功能，能够让多个团队成员同时编辑同一份报表，避免重复劳动，提高工作效率；

6. 更多惊喜等着您！

# 2.基本概念术语说明
## 2.1 Tableau 四个元素
1. Worksheet：报表页面，包括画布、视图区、工具栏、属性栏等，是用来设计报表的区域；

2. Dashboard：仪表板，包含若干个 Worksheet ，提供一种集中管理和查看工作负载的方法；

3. Data Source：数据源，是 Tableau 中最基础的概念，用于指定需要展示的数据；

4. Visualizations：可视化效果，是 Tableau 用于呈现数据的一种方式。常用的可视化效果有饼图、折线图、条形图等。


## 2.2 Tableau 使用流程
1. 安装 Tableau Desktop；

2. 创建一个新的工作簿或者打开一个已有的工作簿；

3. 添加数据源，选择要使用的数据源；

4. 通过拖拽、连接器、过滤器等方式添加可视化对象；

5. 设置数据集的格式和字段映射；

6. 将相关可视化对象放在合适的位置；

7. 调整布局，保存工作簿。

## 3.1 如何在 Tableau 中创建一条线图？

创建一条线图的方法比较简单，只需要在“可视化”菜单中点击“折线图”，然后将想要分析的字段拖到“轴”维度下，点击“下一步”即可。


如果想改变折线的颜色，可以在右侧属性栏中的“颜色”选项中选择，默认是浅灰色，也可以单击颜色块进行设置。
