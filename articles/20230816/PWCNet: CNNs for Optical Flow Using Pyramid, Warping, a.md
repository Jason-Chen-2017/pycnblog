# 1.简介
  

　　运动场（Optical Flow）的计算一直是计算机视觉领域的一个研究热点。近年来，Deep Learning 在这方面取得了巨大的成果，在各种视觉任务中都取得了很好的效果。但是，目前依然存在着两个主要障碍：

1. 准确率低下：对于高分辨率的运动场图像，目前基于深度学习的方法普遍存在着较大的准确率损失。

2. 计算复杂度高：即使采用更高效的模型，计算代价也是一个关键的问题。

　　8.PWC-Net 提出了一种新的CNN结构，利用先验知识建立图像金字塔（Image pyramids），在每一层进行不同尺寸的特征提取。通过将多尺度特征图组合成一个统一的视差场Cost Volume，PWC-Net 能够有效地捕捉全局的局部运动信息。它还可以有效地处理全局和局部信息之间的权衡。



# 2.相关工作
　　前文提到的三种方法：Supervised Flow Estimation、Unsupervised Flow Estimation with Neural Fields 和 最近的 PWC-Net 使用了多尺度特征图和 Cost Volume 来表示运动场，并充分利用了预训练网络的全局上下文信息。



## Supervised Flow Estimation

　　在传统的视频风格转换（Video Style Transfer）等任务中，通常使用的都是基于密集光流估计的方法。这些方法通常是先用卷积神经网络（CNN）提取图片的颜色空间特征，然后用光流场去拟合原始视频中的运动信息。由于光流模型对光照、遮挡、尺度变化等因素敏感，所以这些方法往往会产生较大的准确率损失。除此之外，这些方法只能输出空间上连续的运动，因此难以反映物体快速移动或物体运动形态的复杂程度。同时，因为需要依赖传统的优化算法，这些方法不具有实时性，计算时间较长。

　　　　




## Unsupervised Flow Estimation with Neural Fields

　　基于神经场（Neural Fields）的运动估计方法属于无监督的视觉几何方法。这些方法利用神经网络自编码器（AutoEncoder）来学习图像的高阶特征，并且将其输入到变分自动编码器（Variational Autoencoder）模型中，从而学习图像的全局特征和局部约束条件。这种方法能够捕捉到物体的全局特征，但缺乏对局部运动的建模能力。除此之外，这些方法基于手工设计的自适应光流函数，无法捕捉到真实世界物体的复杂运动模式。




## PWC-Net 的特点

　　8.PWC-Net 是一种有效的多尺度视差场模型，它能够利用先验知识来构建图像金字塔，在每个尺度上进行特征提取。这样就可以同时处理不同尺度上的信息。

　　　　1) 通过不同尺度的特征图，PWC-Net 可以有效捕获局部和全局信息，避免单一尺度上出现的效率损失。

　　　　2) 相比于传统的几何变换方法，PWC-Net 能够精确捕捉到运动的信息。

　　　　3) PWC-Net 使用了深度残差网络（Depth Residual Network）来建模全局上下文信息。这项创新可以帮助 PWC-Net 模型学习全局特征。

　　　　4) PWC-Net 使用了 Cost Volume 来表示全局上下文信息，它可以有效地处理全局和局部信息之间的权衡。
