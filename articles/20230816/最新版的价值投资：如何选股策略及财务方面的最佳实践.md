# 1.简介
  

“投资”这个词，在现代经济学中通常指的是一种资本积累的方式，即把时间和金钱投入到一些具有高预期收益的活动当中。一般来说，投资的目的是通过获得更多的利润来提高个人或者集体的收入和社会经济地位。而“价值投资”，又称为“经营性投资”，是指由企业或组织自身进行资本运作的方式。

市场从事价值投资已经成为不少投资者的选择。但“价值投资”所面临的挑战也越来越多，其中包括：选股的准确性、可持续性、盈利能力、公司治理等等。在进行有效率的价值投资时，投资者需要面对诸如信息不对称、内部矛盾、资本陷阱等诸多难题。对于一些曾经被认为在价值投资领域颇有建树的投资者来说，也越来越感到“非专业”之嫌。因此，“价值投资”引起了众多投资者的重视。

不过，真正的专业人士也并不是随随便便就能掌握“价值投资”的技巧的，需要大量的时间和心血。在研究“价值投资”的过程中，我们发现，其实可以分成三个层次——选股层、策略层、财务层。下面我们就针对这些层次，分享一些个人的投资经验和建议。
# 2.选股层
## 2.1 选股的目的
投资的第一步就是要找到合适的企业。对于一个想进入股市的初学者来说，很多朴素的疑问可能首先出现在这个环节上。比如，我该用什么样的方法选股？选股的标准是什么？选出的股票是否具有很好的业绩和盈利能力？以及，如果选错了股票怎么办？当然，这些问题都是要有备案的。接下来，我会逐一回答这些问题。
## 2.2 选股的方法
### 2.2.1 投资组合管理方法
在选股之前，了解一下投资组合管理方法，这样可以帮助我们更好地选择股票。投资组合管理方法是为了解决多样化的投资需求，制定多种类型的投资组合，且每种投资组合都有特定的风险偏好和目标收益。投资组合管理方法分为四个阶段：1）研究阶段（Strategic Analysis）；2）规划阶段（Portfolio Planning）；3）执行阶段（Execution）；4）结果分析阶段（Results Analysis）。

1.研究阶段：即对已知的信息和情报进行研究，根据研究结果制定出候选池中的股票池。如此可以将符合投资人的要求的股票进行分类。
2.规划阶段：设计出符合投资人的投资组合。通常情况下，投资组合的构成应当具备以下几点：权重分配，即投资组合中的每只股票占比多少；投资周期，即投资组合存续的时间长短；风险偏好，即投资组合能够承受的风险程度。
3.执行阶段：按照规划的投资计划进行股票交易。在执行期间，投资组合的变动往往十分迅速。因此，如何在保证安全的前提下保持收益，是一个极其重要的问题。另外，投资策略的调整、增减都需要考虑相应的风险，以防止投资损失。
4.结果分析阶段：根据执行情况、投资风格、市场情况、公司业绩表现等因素综合判断各个股票的优劣。如有必要，也可以根据历史业绩对投资组合进行重新优化。

### 2.2.2 价值投资选股的五条经验法则
#### 1. 相似的项目优先考虑
与同行业、同类型公司比较，相同或相近产品或服务的企业，能够给投资者提供更大的参考价值。
#### 2. 不完美的企业降低对该企业的关注度
对于那些不靠谱、不健全甚至虚假的企业，不妨放松警惕，避免过度关注。
#### 3. 没有足够信息的企业不要轻易入手
即使企业拥有极好的业绩，却仍然缺乏足够的信息，无法做出判断。
#### 4. 优质的资源优先考虑
比如优秀的研发团队、技术力量，能够帮助企业实现快速的发展。
#### 5. 股息率高的企业尽量避开
因为一旦股息率高，其价格必然贬值，降低了投资价值，造成损失。

以上五条经验法则在选股时具有指导意义，但具体操作还需具体分析。如，股票质量如何衡量？资本市场能否提供有价值的参考数据？证券监管部门有没有相关法律法规禁止所选股票？选股对象还有哪些限制？若不确定，则应先对各项条件有所了解，再决定是否投资。
## 2.3 “价值投资”的未来方向
“价值投资”作为一种新型投资方式，将在未来不断发展。在未来，“价值投资”可能会被其他类型投资纳入到股市之中。比如，传统的资本市场上，虽然企业的收益往往依赖于企业的规模、运营模式、供应链及经营效率等，但在“价值投资”里，很多时候投资者不需要关心这么多细节，只需要关注企业带来的价值。所以，未来的“价值投资”将是一个整合各种资源的过程，而且这种整合不是简单的堆叠，而是融合、结合、进化。因此，在“价值投资”发展的道路上，还存在着许多困难和挑战。
