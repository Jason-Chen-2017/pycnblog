# 1.背景介绍


## 1.1 什么是光刻胶？
光刻胶(英语: Photolithography)是将形状、文字、图案等经过化学冲击、金属表面或非金属表面上的不同强度的步骤加工成微孔结构的一种工艺，其目的是在特定条件下制作出具有一致性和重复性的产品形态。一般来说，光刻胶用于制作计算机显示器、打印机、激光打印机、电视机、数码相机、照相机、航空照明系统等应用领域。光刻胶可以产生微小的颗粒大小的粒子阵列，也可以产生一大块的整体像素阵列。它的产生过程如下图所示。
光刻胶主要分为垂直型光刻胶和流动型光刻胶。前者可以进行大规模集成电路设计；后者用于成像、印刷、电子、摄影、屏幕等方面的应用领域。
## 1.2 为什么要用光刻胶？
光刻胶的主要优点主要有：
* 材料：光刻胶在使用过程中对材料没有特殊要求，仅仅需要具有良好的透气性、耐腐蚀性、低热、高湿、密度适中的物质即可。而且，光刻胶还可以采用非金属材料，因此可以获得较大的粒度和高透氧性，便于控制微观尺寸的精确度。
* 反射率：光刻胶可以制成具有更高的反射率。这是由于光刻胶自身具有微小颗粒，并且通过改变相邻颗粒之间的相互作用，可以在无损情况下增强特定区域的反射率。对于传统工艺而言，即使利用光刻胶的工艺也只能是局部提升反射率，而不能实现全局的提升。因此，光刻胶在光刻胶产业的重要地位。
* 抗化学攻击：光刻胶在制备之前已经通过多个步骤对液晶材料进行了清洁、净化、去除异物、消毒，并在烘焙时加入一些防腐剂，从而降低化学攻击的风险。同时，光刻胶还可以减少噪声的影响。
* 可编程能力：光刻胶可以编程，可进行数字控制，实现精细的工艺控制。例如，它可以编程出不规则的纹路或透明结构，从而制作出具有极高的精度和可靠性的电子产品。
## 1.3 如何用光刻胶制作微型计算机芯片？
制作微型计算机芯片的过程包括以下几个步骤：
### （1）准备工作
第一步是准备好各种零件，如导电层、表面活性剂、抛光剂、溶解剂、瓷砖、锡纸、软膏、硅橡胶、玻璃纤维、压印胶、光刻胶料、镜头等。
### （2）切割基板
第二步是在微表面上进行多次斜切和横切，以得到足够大的切割厚度，在平面中心的位置，铺设导电层，用于制作微型计算机芯片。
### （3）放置镜头
第三步是在导电层上放置微小的镜头，用于刻画微型计算机的硬件及显示图像。
### （4）掩模刻画
第四步是进行掩模刻画，将计算机硬件、微型组件、接口设计出来，并嵌入导电层中。
### （5）回收与装配
最后一步是回收导电层、涂层，通过光刻胶注入电压，使微型计算机芯片完成。
## 1.4 如何用光刻胶制作显影胶卷？
显影胶卷（英语：Photographic embossing sticker），简称显影胶，是由透明胶膜或棉花糖等透明材料研磨、晾干，加工制成的可以用在个人相册、书刊、报刊上，作为饰品和插画品依据一定规范设计印制而成的传真遮盖图，又称遮盖胶。
显影胶卷的制作方法可以分为预先处理、涂层制作、显影成形三步。
### 预处理：
* 纹理纠正：先将照片先改色、调暗、调整透明度，再去除杂质，保证每一张照片都是清晰的、高质量的。
* 提炼轮廓：然后用压印胶把纹理稳定下来。压印胶是一种模具类化学原料，可以通过微波处理和转移能量的方式来提取纹理中的油墨，然后在显影胶卷的表面上涂抹起来，实现轮廓的封闭与生长。
* 滤色、修复：在药液的作用下，胶液会变红，不易于显影，所以需要用药液过滤掉杂质，并加热。修复时，胶渣或渣滓可能会在显影胶卷上留下缝隙，在修复时要尽量补齐这些缝隙。
### 涂层制作：
* 在显影胶卷纸张上涂上木糖皮，提起一点厚度，并在胶片表面涂上曳光膏，油漆膜或其他厚颜料，以消除潮湿气味。
* 用微量的曳光膏喷膠表面，避免留下的任何破洞或渣滓。
* 将显影胶卷纸张放到较暗的地方，避免太阳光照射到，以保护胶片表面不会受伤。
### 显影成形：
* 经过涂层蒸馏、打旋、倒置等手术后，将曳光膏整个涂上、抹到显影胶卷纸张上，就完成了显影胶卷的制作。
* 卷片保鲜期内拍摄足够多张照片后，用适量润滑剂和粉末，按指定的办法混合成厚实胶渣，然后涂抹到显影胶卷上，最后用刀扎进胶膜里。这样做可以防止胶渣老化，且胶卷结构容易保护，达到持久保存的效果。
# 2.核心概念与联系
## 2.1 显影胶与微型计算机芯片
显影胶卷通常与微型计算机芯片一起使用，用于制作个人相册、印刷品和报刊的封面，但它们不是同一个东西。显影胶卷是一种遮盖胶，用于遮盖传真照片上的隐私信息。微型计算机芯片是一种微型芯片，能够安装在个人消费电子设备中，以便方便地进行网络通信、视频播放、音频播放和计算工作。两者之间存在着紧密的联系。
## 2.2 何为光刻胶呢？
光刻胶是一种制造微型粒子阵列的工艺，它以无菌或微生物作为填充物，将输入材料经过转换、变化和结晶化等过程变成微小颗粒，并经过浸润处理，呈现出不同的强度、形状、颜色的特征，创造出图像的宏观效果。其中，原子级粒子阵列则可引起光电效应的作用，产生超高的信号量。故而，光刻胶可用来制作高分辨率电子元器件、激光元件、显示器件、照相机、打印机等电子产品。
光刻胶最初用于用光制造微型的电子元器件，包括集成电路、晶体管、芯片等。但是，光刻胶也是可以用来制作显影胶卷的材料。
## 2.3 二者有什么关系呢？
显影胶卷是一种遮盖胶，用于遮盖传真照片上的隐私信息。它与微型计算机芯片是两种不同功能的产品，虽然都是以电子元器件、显示器件为主，但两者却有紧密的联系。微型计算机芯片是一种微型芯片，能够安装在个人消费电子设备中，以便方便地进行网络通信、视频播放、音频播放和计算工作。显影胶卷虽然也与微型计算机芯片一样，都属于制作电子产品的一部分，但它的重要性却远远超过它所涵盖的功能。
