# 1.背景介绍


## 量子电动力学的概念与特点
在现代科技中,与过去相比,量子物理技术已经逐渐成为一个重要且引人注目的话题。不管是对待生命的研究、环境、材料等领域,还是对智能机器人的研究、通信网络的研究等技术领域,都在不断地探索着利用量子力学、量子信息、量子控制等基本物理原理来建造新的智能世界。量子电动力学（quantum electrodynamics,QED）正是一种利用量子力学、量子统计、量子场论、量子色粒子、量子纠缠等量子力学原理的基础科学。

量子电动力学的概念最早由图灵和莱昂哈德一起提出,用来描述在无穷小的时间内,两个微观粒子之间的状态转换过程,这对航空、农业、生物工程、太阳能源、核反应堆等领域都有着重大影响。而量子电动力学的实质则是将微观粒子看做离散的粒子,用量子化的数学方法来研究其行为、规律、运动。它是近些年量子技术爆炸式增长、高校和科研机构纷纷投入研究的热门方向。

量子电动力学主要研究微观粒子在空间和时间上的行为,尤其关注的是态空间纠缠(statistical-mechanical coupling),即不同位置、不同时刻处于不同状态的两个微观粒子之间的相互作用,可以用局部磁场模型来刻画其微观结构和场特性。通过这样的研究,可以揭示微观粒子的基本性质和物理规律。

# 2.核心概念与联系
## 波函数与波packets的概念
首先,让我们定义两个重要的概念——波函数（wavefunction）和波packet。

波函数（wavefunction），是指概率密度函数（probability density function）或动量密度函数（momentum density function）所描绘的一组波函数中的一个。波函数与观测到的波形有关。在相干媒介中,波函数通常具有无限多的可能取值；但在绝热媒介中,波函数只有两种可能取值,即等概率的正负号。因此,如果某个系统具有确定性,那么它的波函数只能是有限的;然而,当系统处于不确定状态时,它的波函数可以表示为关于时间或者空间的函数。一个理想的波函数应该具有如下的性质:

1. 任意时刻处于系统的任何位置,每个波函数都是相同的；
2. 如果把任意一个波函数的局部转移引入到另一个局域,那么结果仍然是一个波函数；
3. 给定任意一段时间内系统的状态,在该区间内,系统的所有可能波函数应该是平等的；
4. 当系统在给定的时间或者空间范围内发生哈密顿量的本征态（eigenstate）的叠加时,它应该有一种统计规律。

波packet（wave packet），是指把相关的粒子以某种具有空间-时间连续性的形式聚集在一起,并对它们进行研究的一种工具。它把单个粒子看作原子、氢原子、氘原子、电子云团等一系列粒子的集合,并把这些粒子按照相同的时间依赖关系联系在一起。它通过简化研究的方式来减少研究对象的复杂度,从而提高研究效率。

## 电子的哈密顿量和粒子的晶格场论的基本概念
为了理解微观粒子的行为,需要了解它们的基本特征、基本属性。

波函数的唯一确切定义依赖于许多假设条件,而对波函数来说,其意义恰好就是通过这些假设条件来表述。例如,在假设下,一束电磁波与空间中某点之间的距离为零,就可以认为这个点处于无穷小的能量差,使得它无法分辨清楚哪个原子在那里,因此需要通过引入更复杂的假设才能得到波函数的准确定义。

所以,对于一般情况,对于微观粒子的行为,可以从以下两个方面入手:

1. 通过研究它们的哈密顿量来找寻它们的基本性质；
2. 用晶格场论的方法来建立微观粒子的理论模型,然后运用这一模型来解释微观粒子的行为。

### 哈密顿量

量子力学研究微观粒子的基本规律,而微观粒子的行为是依赖于宏观量的,这些宏观量统称为哈密顿量。一个理想的微观粒子的哈密顿量应该满足以下几个条件:

1. 描述它的自由度（degrees of freedom），即它受到外界变量的控制；
2. 不随时间变化；
3. 系统的物理性质（如电荷量）只依赖于基底粒子的性质；
4. 由微观粒子的性质决定。

微观粒子的哈密顿量与所有其他微观粒子之间也存在一些微妙的联系。微观粒子的基态可以看成是它的最小受力态,这意味着它不能再分裂,其哈密顿量包含着如下的贡献:
$$\hat{H}_{min}=-\frac{\hbar^2}{2m}\nabla_x^2+V(\mathbf{r},t)$$
其中，$\hat{H}$为哈密顿量，$- \frac{\hbar^2}{2m}\nabla_x^2$为约化的动量守恒方程，$V(\mathbf{r},t)$为系统自身的势能。这是微观粒子的基态哈密顿量。

当一个微观粒子到达另外一个微观粒子的时候，会产生交换算符$\left[\hat{c}_i^\dagger,\hat{c}_j\right]$。交换算符描述了两个原子之间两种状态之间的转换。当两个原子到达同一个宏观坐标点时，两个原子都处于基态。如果两个原子到达不同的位置时，就会出现两者之间的交换。交换算符满足如下的形式：
$$[\hat{c}_i^\dagger,\hat{c}_j]=\delta_{ij}-\frac{\beta}{2}(\hat{c}_i+\hat{c}_j)^*$$
其中，$\hat{c}_i^\dagger$表示第$i$个原子的克崎希尔变换，$\hat{c}_j$表示第$j$个原子的克崎希尔变换。$\delta_{ij}$表示两个原子的自旋量不发生交换，$-\frac{\beta}{2}(\hat{c}_i+\hat{c}_j)^*$表示有相互作用处的势能。当一个原子的自旋量发生变化时，会发生跃迁。跃迁可以分为三类：激发跃迁、消失跃迁、跃迁转变。由于交换算符只能表述单个粒子之间的相互作用，因此其作用还需要结合微观粒子的态空间。此外，还有很多其他交叉反应。一个典型的交叉反应包含四种原子以及它们之间的各种交换或跃迁。
