# 1.背景介绍


情感分析（sentiment analysis）是一种自然语言处理技术，目的是识别出文本中的积极、消极或中性的态度。它可以应用于如自动回复、舆情监控、产品推荐等领域。情感分析通过对输入的文本进行分析、挖掘、归纳和抽取，从而判断其情绪，帮助企业更好地做出决策。它的优点包括：准确性、及时性、真实性、客观性、可迁移性。情感分析主要由两大类方法：
- 有监督方法：基于分类器或模型训练的算法，需要标签数据才能进行训练并得到模型参数，因此有监督方法所得结果具有很高的精度。例如，Sentiment Analysis with Naive Bayes（SAB），这是一种基于贝叶斯法的情感分析方法，它的训练集要求至少包含一半的积极和负面语料，并且特征空间较小；FastText 是另一个有监督方法，它可以自动提取文本特征并构建模型。
- 无监督方法：不需要任何先验知识即可进行训练，因此无监督方法所得结果具有更高的泛化能力。例如，Topic Modeling（TM），它利用潜在主题（topic）的发现对文本聚类并评估每个文档对于每一个主题的贡献度，提取关键词；Doc2Vec 可以将文档转化成向量表示形式，用于主题建模和文档相似度计算。
# 2.核心概念与联系
## 2.1 情感词典（Sentiment lexicons）
情感词典（sentiment lexicon）是一个预先定义好的一组正面、负面、中性的单词列表，通常用于计算情感值（sentiment value）。
有两种情感词典：
- 基于正例的情感词典：将一些常见的正面词汇和表达添加到情感词典中，并赋予它们一个正面的情感值，例如：positives = ['good', 'great', 'excellent']，相应的情感值为[1, 1, 1]。这种词典的一个缺点是可能会出现一些误判，例如，词典认为“好”是负面的，实际上却可能是褒义词，或者“好”这个词本身就是褒义词，但是由于词典的限制，情感分析仍会将它视作正面的。
- 基于反例的情感词典：将一些常见的负面词汇和表达添加到情感词典中，并赋予它们一个负面的情感值，例如：negatives = ['bad', 'terrible', 'awful']，相应的情感值为[-1, -1, -1]。这种词典的一个优点是可以避免一些误判，例如，如果出现“难受”，那么它应该被视作负面的，不会被认为是褒义词。然而，这种词典也存在一定的局限性，因为在某些情况下，情绪不容易用否定词描述清楚，例如，“我非常满意，但那个电影让人失望”。因此，基于反例的情感词典往往不能完美解决所有问题。
## 2.2 停用词表（Stop words list）
停用词（stop word）是在文本处理过程中，对句子进行切分或解析前必须剔除的词语。常用的中文停用词表共计1798个词。停用词表会影响文本处理的效果，例如，停用词可能会导致某个词或短语过多地影响情感值。
## 2.3 正则表达式（Regular Expression）
正则表达式（regular expression）是一种用来匹配字符串的强大的模式语言。它能让你方便、灵活地搜索、替换、校验文本内容。
## 2.4 NLTK（Natural Language Toolkit）
NLTK是python的一个自然语言处理工具包，其中包含了很多用于NLP的函数库和工具。NLTK提供了许多高级接口和功能，能够简化NLP任务的实现。NLTK还提供了一些用于数据集、语料库和模型的下载功能。
## 2.5 TensorFlow（TensorFlow）
TensorFlow是一个开源的机器学习框架，用于构建神经网络和深度学习模型。它支持许多主流的深度学习算法，包括卷积神经网络（CNN），循环神经网络（RNN）和递归神经网络（RNN）。TensorFlow还提供了一系列API和工具，用于构建分布式多机集群，提升运算效率和性能。
