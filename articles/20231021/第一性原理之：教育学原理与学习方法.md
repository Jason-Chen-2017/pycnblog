# 1.背景介绍


在过去的几年里，人们对教育产生了巨大的关注，特别是在当今社会，越来越多的人都希望自己的孩子能够受到良好的教育。而教育理论作为一个重要的学科，其理论基础就是人类的发展历史以及人的能力，因而也成为许多学者研究的热点。很多人将教育学分为三个层次，即元、宏观、微观。因此本文先从教育学的元层次——元教育理论入手，探讨如何理解元教育理论。


## 1.1 元教育理论概述
元教育理论由哈特·海德格尔于19世纪末提出，他认为人类社会发展的主要过程可以分为以下四个阶段：

1.早期阶段(Primordial Era)：生物进化阶段，整个社会包括人口很少、经济条件极其落后，人类开始追求生存，并制定适应环境的规则。

2.原始群体阶段(Original Peoples)：人口数量增加，社会制度逐渐完善，但仍处于生产力低下、经济不景气的状况。

3.奴隶阶段(Slaves Era)：现代工业革命发生后，人口开始贩卖奴隶劳动，开始出现种族隔离，导致人口增长放缓。

4.共和国阶段(Republic Era)：17世纪，人口开始增长，并带来社会的繁荣。在这个阶段，国家体系开始崛起，使得人们免除了奴役，社会开始向上流动。

元教育理论认为，这些阶段之间的相互作用影响着人类智慧、能力、道德等方面的发展。随着社会发展的推移，人类的知识、技能、想象力和意志力也日益增强，这些能力又会促进社会的进步。因此，理解元教育理论对于理解教育的发展以及个人发展具有重要的意义。
