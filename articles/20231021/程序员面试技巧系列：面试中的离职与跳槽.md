# 1.背景介绍



在工作中，程序员面试往往成为相当重要的环节。但同时，由于面试题目数量众多、候选人经验广泛、面试官水平参差不齐等特点，导致很多候选人会遇到“神奇的面试官”现象——他们常常询问一些比较难的问题，甚至迫于压力而选择放弃。本文将从以下三个方面阐述面试中的突发情况处理策略：

1.面试官来了

最常见的一个情况就是面试官来了，而这时通常都会发生什么？很简单，你可以直接拉开距离，别忘了记得“微笑”！面试官的态度非常重要，你要做的是善待他们，充分沟通，交流才能更有效率地准备你的面试。

2.候选人找不到合适的工作

有时候候选人根本就没有找到心仪的工作，这也许是因为他们的个人实力、工作积极性、兴趣爱好等各方面因素综合作用下形成的结果。那么，如何处理这种情况呢？首先，你需要明确自己的定位，再根据自己对候选人的了解以及竞争激烈程度，制定出“人尽其才”，“物尽其用”的策略；然后，给予候选人适当的帮助和支持；最后，鼓励和感恩，不要过度责怪候选人。

3.面试时间冲突

如果面试时间冲突不断，比如说两个候选人都觉得累，或者面试官提前打断了候选人，那么可以采取什么措施呢？首先，充分准备好备用方案，在随时准备的时候打开备用方案；其次，快速反应，果断拒绝。这是最基本的方法，还可以考虑利用团队协作，结对面试，通过“一言一语”进行沟通，确认候选人是否存在诚意。

基于以上分析，下面让我们一起讨论一下如何避免出现这些突发情况。

# 2.核心概念与联系

- 定时预约

定时预约是指面试官和候选人事先约定好一个面试时间，并由候选人主动申请调配资源到指定的时间面试。这个策略有助于确保面试官和候选人的时间不会被其他因素影响，也利于候选人准时提交简历。但是，定时预约也是有代价的，因为面试官和候选人可能在约定时间之前或之后，受限于各种原因无法赴约。此外，采用定时预约，也可以减少双方面试官的接触，防止有碍于候选人的工作。因此，该策略应尽量谨慎使用。

- 面试自愿放弃

面试自愿放弃是指候选人主动放弃面试。按照惯例，这样的行为一般都是出于一种自我保护的态度。然而，放弃面试又有着其独特的风险，可能会引起人身损害、财产损失、工作危机等严重后果。因此，在真正面试前，应主动征求候选人意见，如能取得理解，则可考虑保留面试；如不能取得共识或保持冷静，则可以提前结束面试，以免耽误及时续聘。另外，面试过程中，也应允许候选人多一点自主权，即使遇到不可抗力或恶劣天气，也可提前结束面试，并把面试经验总结记录下来。

- “神奇的面试官”现象

“神奇的面试官”的产生主要是由于候选人回避了不合适的问题，导致面试官难以理解候选人的需求，进而造成败笔。解决这个问题的方法包括：

1.收集面试官所问的问卷和提问

2.从多个角度观察候选人

3.了解候选人习惯和兴趣

4.充分探索面试官的想法

以上方法可以有效地帮助候选人判断面试官所问的问题是否合理，以便合理回答。
