# 1.背景介绍


## 为什么要学习微积分？
作为一个技术人，作为从事技术工作的人员，需要了解、掌握一定的数学基础知识。对于计算机工程师来说，掌握微积分是基本功。数学是科学的基石。计算机软件设计涉及到很多数学计算，特别是在图形图像处理领域。微积分也同样重要。因为它可以帮助我们更好地理解并解决一些实际的问题。微积分不是万能的，但是却可以提供给我们许多宝贵的信息。学习微积分可以让我们建立对世界的更好的理解，并有助于我们提高我们的技能水平，最终达到事半功倍的境界。
## 微积分的定义
微积分(Mathematics)是用符号语言研究函数的最初和基础的方法。微积分是一种用来研究连续介质内在结构的数学方法。微积分的研究从古希腊时期就开始了，从几何学开始，一直延伸到微积分学。微积分研究的是数据的曲线如何变化，或者如何靠近或远离某种目标值。微积分能够通过运用规则、公式、方程和抽象的观念来描述和预测数据变化。在不同应用领域，微积分都有不同的用途。比如工程学、经济学、统计学、物理学、数学物理学等。
## 微积分研究的主题
微积分研究的主题可以分为以下几类：
1. 概率论和随机过程。微积分可以用来描述随机事件发生的概率，以及随机变量的分布特征。
2. 函数的微分。微积分可以用来分析和求导方程的初始条件、局部最小值点、极值点等。还可以用来进行函数积分、求导、求根、插值等。
3. 矢量空间的微分。微积分可以用来分析二维和三维欧氏空间中点、直线、曲面、曲线、向量场的微分形式。还可以研究向量函数的方向导数、梯度、曲率、旋转、叠加、散度等。
4. 常微分方程。微积分可以用于研究非线性常微分方程。常微分方程是指微分方程组只有一个未知函数而没有常数项。常微分方程经常出现在力学、电磁学、控制论、信息论、电子工程、工程热力学、统计力学等学科。
5. 多元微分和代数变换。微积分可以用于研究多元函数和变换的微分形式。包括拉普拉斯算子、莱布尼兹变换、曲线积分、曲面积分、柱状图投影、坐标变换等。
# 2.核心概念与联系
## 一元函数和多元函数
### 一元函数
在微积分中，一元函数就是指一个变量只受一个自变量影响的函数。比如说 y = f(x)，其中 x 是自变量，y 是因变量，f() 表示一个一元函数，x 和 y 可以是一个实数，也可以是复数。
### 多元函数
在微积分中，多元函数就是指一个变量受两个或者更多自变量影响的函数。比如说 z = g(x,y),其中 x 和 y 是自变量，z 是因变量，g() 表示一个多元函数。
## 抽象变量
抽象变量（也就是虚拟变量）是一个非常重要的概念。微积分中常用的抽象变量有时间t，空间x，频率k。
## 基本运算
### 定积分和不定积分
定积分又称积分，是将曲线上的点的数量加权平均得到的一个数值。比如说，对于某函数f(x)在区间[a,b]上具有合适定义的曲线y=f(x)，则该曲线从x=a到x=b的定积分表示为:

I=∫_{a}^{b}f(x)dx

即先取左端点a和右端点b之间的某个矩形区域（可以用直线段、弦段），并沿着该区域在x轴方向做足够细小的切片，使得每个切片上的曲线部分（称为笛卡尔网格）的面积等于函数曲线的高度乘积dx。然后，根据曲线所在切片的宽度dx和切片的数量，将这些矩形区域的面积加总起来，所得结果即为曲线从a到b的定积分。

当曲线在区间上是光滑的时候，定积分和函数值相等；当曲线在某一点处存在凹陷时，定积分的值会比真实的值小，称为“下丁”积分；当曲线在某一点处存在凸起时，定积分的值会比真实的值大，称为“上丁”积分。

不定积分又称所积分之外的地方积分，不依赖于被积函数上确定的某一点的函数值，而仅仅取决于某一点和某一切片之间的曲率。其公式为：

∮_{c}^{d}f(u)\cdot du

其中，c和d表示积分的下界和上界，u是其他任意一点。不定积分可用来求解各种积分方程。比如：

（1）某函数f(x)关于x轴的曲率绝对值为R(x)，即H(x)=\frac{1}{R^2(x)}；求f(x)在区间[a,b]上的积分。由于曲线由一段曲线线段和无穷多个光滑曲线段组成，因此可以通过不定积分求解该积分。

(2)某一面积A和某一质心C确定后，取一带状区域D，C在边缘上。对D上的任一点P，取r以内的切线T，围绕T做不定积分，有：

∮_0^r[(x-cx)^2+(y-cy)^2]\cdot L dx dy=-ArL^2cos(\theta)

由双曲余弦定理，知道\theta=\frac{y-\sqrt{R^2-(x-cx)^2}}{x-cx}，可以把不定积分改写成与x、y和r无关的形式：

\int_0^{+\infty}\int_0^{2\pi}(r+Rr'cos\theta)\sqrt{R^2-r'^2}dr'dt=-\frac{A}{2}ln(2r+A)+K

其中，K=\frac{1}{2}ln(2A+|K|)，K=C，K是一个常数。

（3）已知一个抛物线f(θ)=A·sinh(bx/2)和曲率半径R，试证明该抛物线的最大值L和最小值H可以在不经过C点的情况下直接求出，即：

H=\frac{2\sqrt{\pi A}}{b}-A; L=-\frac{2\sqrt{\pi A}}{b}+A;\quad b>0,\quad R>0

由抛物线的曲率半径，可知曲率半径等于半径除以斜率。因此，抛物线曲率半径可化为半径除以tanθ，且θ等于2arctan(bx/2)。所以：

H=\frac{2A}{b}(tanh(-\frac{ab}{\sqrt{Ab^2-R^2}})-tanh(\frac{ab}{\sqrt{Ab^2-R^2}}))+A;\quad L=-H+\frac{2A}{b}

由不定积分的计算方法，可以对此积分进行计算。若令bx=2Rtanθ，则H=2A/(2Rtanh(-R))，L=-H+2A/(2Rtanh(-R))。

（4）某函数f(x)满足x^2f''(x)+xf'(x)+fx=0，试求f(x)在区间[-π/2,π/2]上的定积分。由于该函数存在极值点，可用不定积分进行计算。设tan(x)>0，则tan^3(x)<1/3tan(x)+1，故：

\int_-π/2^πf(x)dx=\int_-π/2^πx\cdot f(x)/tan^2(x)+f(x)dx\\=\frac{π}{3}\int_-π/2^πx\cdot tan(x)(x+tan^{-1}(x))dx\\=\frac{-π^3}{3}\int_0^{\pi/2}tan(x)tan(x')dx'+\left.\frac{(1+tan^2(x'))}{3}\right\rvert_{\pi/2}^0\\=-\frac{π^3}{3}[f(0)+\sum_{n=1}^{\infty}\frac{(-1)^n}{n!}]
