# 2.核心概念与联系
## 基本语法规则
Python是一种面向对象、动态语言，其基本语法规则如下：

1. 以缩进方式组织代码块；
2. 每条语句以分号结束；
3. 用制表符或空格缩进；
4. 不需使用括号，圆括号是可以省略的；
5. 使用英文单词表示变量名、函数名等；
6. 保留字（关键字）不能用于命名变量、函数名等。

## 数据类型
Python的内置数据类型包括四类：

1. 数字（Number）：整数（int），浮点数（float），复数（complex）。
2. 字符串（String）：字符序列（str），以单引号或双引号括起来的文本数据。
3. 容器（Container）：元组（tuple），列表（list），集合（set），字典（dict）。
4. 布尔值（Boolean）：True，False。

## 控制结构
Python支持的控制结构有：

1. if-else语句；
2. for循环；
3. while循环；
4. 函数定义与调用。

## 一、导入模块
在开始之前，需要先导入需要使用的模块。模块是Python中的一个重要概念。这里我推荐导入两个模块：numpy和matplotlib。

```python
import numpy as np #导入numpy模块
import matplotlib.pyplot as plt #导入matplotlib模块
```

## 二、绘图例程
首先，我们来创建一个简单的折线图来展示最基本的图形绘制方法。代码如下：

```python
x = [1, 2, 3, 4]
y = [2, 4, 1, 3]
plt.plot(x, y)
plt.show()
```

上述代码创建了一个 `x` 和 `y` 数组，然后使用 `matplotlib` 模块中的 `plot()` 方法绘制了一条折线。最后使用 `show()` 方法呈现图像。

如果要更换线形状或者添加额外的信息，只需要再调用不同的方法即可。比如，修改成蓝色虚线图：

```python
x = [1, 2, 3, 4]
y = [2, 4, 1, 3]
plt.plot(x, y, color='blue', linestyle=':')
plt.xlabel('X axis label')
plt.ylabel('Y axis label')
plt.title('Line Chart')
plt.legend(['Data'])
plt.grid(True)
plt.show()
```

上述代码增加了 `color`，`linestyle`，`xlabel`，`ylabel`，`title`，`legend`，`grid` 参数，使图形更加完善。
