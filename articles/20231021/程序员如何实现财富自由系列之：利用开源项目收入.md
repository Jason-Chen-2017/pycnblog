# 1.背景介绍


## 为什么要写这个系列文章？

在互联网、科技、金融等领域，程序员正在享受着巨大的成功，他们创造了许多伟大的产品和服务，如腾讯QQ、阿里巴巴、京东、美团等都属于世界级的公司。

但是有些时候，程序员也会产生一些心理上的压力，因为这些公司都拥有巨额的利润，而一些程序员却苦于没有收入来源。

本文旨在通过实操的方式教会大家如何从零开始用开源项目来获取收入，让大家感到财富自由的喜悦！


## 何谓财富自由？

财富自由是指个人或家庭获得足够的经济支配能力，能够独立自主地按照自己的意愿、兴趣、能力生活下去。简单来说就是尽可能多地实现自己的价值观，并不断扩充财富储备。

然而，一般人对财富自由的认识往往比较片面，或者是盲目的乐观主义。所以，需要建立一个理论的框架，基于经验、历史、理性分析、逻辑推理，并且需要把握到脚注中的提示，提醒自己做出正确的判断。



# 2.核心概念与联系


## 1.开源项目

开源项目（Open Source Project），简称OSS，是一种允许他人修改、分发其源代码的软件开发模式，通常托管在众多开放平台上供公众使用，是一种公益性的社会行为。OSS 的定义较为模糊，在很多情况下可以理解为源代码的共享，能够对外公布原始代码，并且接受社区贡献者的反馈。

## 2.知识产权保护

知识产权保护是指为了实现信息技术的商业化运用，保护知识产权成为重要的环节。任何人的创新能力都离不开知识产权保护。知识产权保护主要包括三个层次：版权保护、专利保护和商标保护。

## 1.Github Repository Analyzer

Github Repository Analyzer 可以帮助用户查看 GitHub 上公开仓库的信息和利用统计数据了解自己所关注的项目的活跃度，包括：

- 项目Star数量分布图：展示全球GitHub用户所关注的某个项目的Star数量分布。
- 最活跃的开发者列表：展示GitHub用户所关注的某个项目最活跃的开发者列表。
- 文件类型分布图：展示全球GitHub用户所关注的某个项目的文件类型分布。
- 源码语言分布图：展示全球GitHub用户所关注的某个项目的源码语言分布。
- 开源协议分布图：展示全球GitHub用户所关注的某个项目的开源协议分布。
- 每日新增提交量曲线图：展示全球GitHub用户所关注的某个项目每天新增的提交量数据。
- 依赖库的更新时间分布图：展示全球GitHub用户所关注的某个项目依赖库的更新时间分布。


### 使用方法

1. 访问 Github Repository Analyzer 的地址 https://github-repository-analyzer.com/ 
2. 在搜索框中输入自己关注的项目名称，例如：sympy。
3. 点击搜索按钮进行查询，即可看到项目相关的数据图表。
