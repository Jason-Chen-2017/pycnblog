# 1.背景介绍


## 概述
晶圆清洗（PCB）是PCB生产过程中经常使用的一项工程技术，其目的是利用低温低压等条件对浸润在母材上的电阻、电容、电感等物件进行清洗、去除杂质、化学溶解、提高耐候及纳米缩短寿命。在晶圆制造领域，PCB绝大多数都是采用由浆状的集成电路板来制造。
PCB清洗的主要原理是在晶格结构中加入一定量的苯乙烯粉末来破坏电性介质并将杂质从晶体中分离出来。通过不同的清洗方法，可以消除不同类型的杂质，如电子浪费、氮化物、过厚化物、焊接不良、阻燃性印刷等。晶圆清洗除了用来清洗浸在晶格结构上的各种电阻、电容、电感、硅基导体等微小元器件外，还可用于去除表层厚度较大的墨盒、色块和角钉等物料。去背面工艺可以极大地减少PCB的生产噪声，降低成本和降低寿命。同时，通过去背面工艺也可以降低PCB的摩擦和散热损耗。因此，对于制作含有垃圾的PCB而言，晶圆清洗与去背面工艺非常重要。
## 设计意义
晶圆制造是一个高度复杂的工程，因而相关技术知识需要有广泛的研究。PCB清洗与去背面工艺可以促进PCB的纯净、整洁、无噪声、易维护、易维修和降低成本等方面的特征，对于广大的工程师、科研人员、企业主等都具有重要的现实意义。
# 2.核心概念与联系
## 清洗晶圆
### 衬底层
在PCB上加入一层衬底层，可有效防止钢丝反流、破坏液化气体和发生腐蚀等。通常的做法是用油漆或电解质撒于该层，其厚度应保证在单个单元之间均匀分布，以防止杂质滞留。
### 刨光
晶圆对角线处的衬底层边缘有一定的刨光作用，刨光作用会使晶圆表面出现条纹状的斑点，且这些斑点随着晶圆的制作逐渐增多，最终形成一些尖锐的地方。所以，在晶圆清洗工序前先进行刨光处理可以避免掉电或其他情况导致的晶圆切割。
### 滤波
滤波是指对浸润在晶圆中的电阻、电容、电感等微小元件进行擦去、腐蚀，使其无法再接收外部环境的过程。常用的滤波方法有反射率检测（RRD）、电导率检测（ECD）、电场检测（EDT）和振动应变检测（OCD）。
## PCB进口导线路段的去背面工艺
PCB进口导线路段往往存在厚度较大的非铜/镍材料(例如金属)物料和皮革物料混杂在一起，这个时候就要考虑对导线路段进行去背面工艺。为了避免导线因承载的信号而起刺激作用和折叠破坏线路，一般情况下，导线上层要做出“烫痕”，即对其进行手工打磨使其具有光泽、韧性、平坦性。这种制作成本比较高，但对于环保或者电力安全来说也是非常必要的。
去背面工艺有助于降低PCB的处理成本，提升PCB的整洁程度，而且也能够避免PCB的断开等现象发生。
