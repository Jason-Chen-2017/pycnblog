# 1.背景介绍


近年来，中国在芯片领域积极布局紫外光刻技术的应用。主要是为了降低芯片的面板温度、提升面板质量、提高设备电性能、改善现场环境光照条件等方面的应用需求。其中，深紫外光刻与极紫外光刻都是目前得到应用最广泛的两种技术。

“深紫外光刻”（deep infrared (DIF) imaging）与“极紫外光刻”（extremely deep infrared (EDIF) imaging）是两种主要的紫外光刻技术。二者均属于后端设计或微观技术中，利用照射深度远超过普通的红外光刻技术所能达到的极限，通过特定的结构材料，将照射到其上的物体、组织反射出多种不同的光谱范围，从而对物体进行成像。目前国内外相关研究成果显示，对于某些复杂且复杂的器件，基于深紫外光刻技术能够达到非常好的性能指标，并且可以带来更加便捷、精准的产品开发。例如，目前用于光纤网络传输、卫星导航、超声探测等领域的芯片都采用了深紫外光刻技术。

本文将以一种技术导向的方式，全面介绍深紫外光刻与极紫外光刻的原理、特性及其应用场景。并通过具体例子和实例，帮助读者快速理解、掌握深紫外光刻技术。希望能给大家一个全新的认识，并对日后的创新、应用提供借鉴。

 # 2.核心概念与联系
## 2.1 概念简介
### 2.1.1 深紫外光刻技术
深紫外光刻技术，即利用照射深度远超过普通的红外光刻技术所能达到的极限，通过特定的结构材料，将照射到其上的物体、组织反射出多种不同的光谱范围，从而对物体进行成像。主要目的是解决摄像头在深处无法看到的物品、组织的成像难题。深紫外光刻的传感器一般配备有多条光通道，分别接收不同波长的入射光，如蓝光、红光、绿光、紫外线等，然后对这些光通道对应的光谱范围进行处理，从而得出图像。与普通的红外光刻相比，深紫外光刻有着巨大的优势，主要有以下几点：

1. 在最深处的物体上：由于深紫外光刻只能看到最深处的物体、组织，因此它可以用来对物体内部的结构进行成像；
2. 在频率不高的区域：由于深紫外光刻没有受限于物体相互之间的反射，因此可以在低频率区域（例如背景噪声和干扰物）上进行成像；
3. 对粒子的光学特性敏感：由于深紫外光刻可以直接观察粒子的光学特性（如吸收、扩散、衍射等），因此可用来观察各种复杂且动态的物体；
4. 不受摄像机本身影响：由于深紫外光刻不需要依赖摄像机本身的特性，因此不会受到摄像机本身性能的影响；
5. 可以长时间连续工作：由于深紫外光刻没有摄像机或其他设备的配合，因此可以长时间连续工作，可以延长成像时间；

### 2.1.2 极紫外光刻技术
在深紫外光刻技术中，对于某些复杂且复杂的器件，基于深紫外光刻技术能够达到非常好的性能指标，并且可以带来更加便捷、精准的产品开发。例如，当前用于光纤网络传输、卫星导航、超声探测等领域的芯片都采用了深紫外光刻技术。但是随着科技进步和人类生活水平的提高，越来越多的应用面临着高精度、高分辨率、高灵活性、高容错率的需求。为了满足这一需求，需要一种全新的极紫外光刻技术。这种极紫外光刻技术是指以不同于深紫外光刻的反射理论作为基础的新型紫外光刻技术，它主要解决两大难题，即深紫外光刻所不能解决的深度问题和分辨率问题。

极紫外光刻的传感器由多个光通道组成，它们的中心是反射镜单元，具有很强的自由扩散、自由吸收、自由折射能力，同时还有强大的聚焦和带阻功能，可以对不同光谱范围的物体进行成像。由于传感器的任意两个直线之间的距离都可以相差无穷大，因此它不受物体、样品、照明源或其他任何东西的尺寸限制。通过传统的反射机制、吹叶效应、材料响应及组合成像管等手段，极紫外光刻技术可以用成像管或显影结构自成一体，实现在无限的距离和角度范围内进行成像。而且由于它具有高灵活性，所以可以适应各种各样的条件，包括变化的光照条件、高频率的噪声、各种复杂的材料及各种类型的物体。

极紫外光刻的主要优点如下：

1. 观察复杂物体的内部结构：与深紫外光刻相比，极紫外光刻可以更好地观察复杂物体的内部结构。因为它对复杂的材料、物体内部结构的复杂度、多孔和复杂的结构具有较高的敏感性；
2. 提供更高的分辨率：由于传感器的每个通道的像素数量巨大，所以它的分辨率很高，可以达到几百万的分辨率级别；
3. 提供更高的精度：极紫外光刻传感器的精度要高于普通红外光刻传感器的精度，它可以对透射性物体、凹凸性物体和沉积物体进行成像；
4. 更加稳定、耐用、灵活：极紫外光刻传感器具有较高的可靠性、耐久性和灵活性，它可以长期运行，还可以处理高频的噪声、变化的光照条件、各种复杂的物体和材料；
5. 可扩展性：极紫外光刻传感器可以通过增设通道数进行扩展，可以配合各种光源进行使用。

### 2.1.3 深紫外光刻与极紫外光刻的比较
深紫外光刻技术、极紫外光刻技术虽然名称有些不同但其目的都是为了更高级、更精准的图像处理，也就是为什么有时候称之为深紫外光刻而有时候又称之为极紫外光刻呢？其实，二者的区别还是比较大的。

深紫外光刻技术旨在用于在一些特殊的情况下，如在密封的表面上发现内部的结构信息，或者进行医学、生物学、农业等方面深入研究时，需要制作非常精确的光谱图。它的传感器通常由光学元件组成，多条光通道分布在整个照射路径中，通过精心设计的结构，将输入的不同波长的光线分别传递到每条光通道中，从而对物体反射出的光谱范围进行处理，产生图像。由于传感器与结构本身的位置关系紧密，其成像效果十分逼真、清晰。

然而深紫外光刻技术存在着其局限性。首先，由于结构的缺乏，它的成像能力受限于所能接受的光的波长，其效果也会受到各种环境因素的影响。第二，由于摄像机本身的性能限制，深紫外光刻的传感器一般都没有随摄像头一起运动，导致其在停留的时间长短都会受到摄像机的影响。第三，由于传感器的结构固定在摄像机前面，因此在空间位置上受摄像机的限制，成像质量受到摄像机的限制。最后，由于传感器由不同波长的光线激发，会造成不同的光谱范围，所以对于某些特殊的物体，其成像效果可能会受到其自身的性质及其周围环境的影响。总结来说，深紫外光刻技术的成像效果受到多个因素的影响，其效果不一定适用于所有场景。

而极紫外光刻技术则完全不同于深紫外光刻技术。它认为深紫外光刻技术所面临的问题，实际上有很多可以克服的地方。极紫外光刻技术不仅可以解决深度问题和分辨率问题，而且可以完全克服摄像机本身性能的限制，可以长时间连续工作，可以提供高灵活性、高分辨率、高精度、高容错率的成像效果。而且它的结构设计可以使其成像管或显影结构自成一体，无需再依赖于传统的反射机制、吹叶效应、材料响应及组合成像管等手段，因此具有极大的自由度和开放性。它的特性使得其成像性能具有卓越的灵活性和稳定性，在光学、结构、物理、工程等诸多方面都有突破。

综上所述，深紫外光刻技术和极紫外光刻技术之间的区别主要体现在三方面：

1. 技术层次：深紫外光刻技术主要用于制作特定结构下的图像，如医学、生物学、农业等方面。而极紫外光刻技术是一种全新的基于反射理论的光刻技术，其结构可以自成一体，可以长时间连续工作，具有高灵活性、高分辨率、高精度、高容错率的成像效果。

2. 成像原理：深紫外光刻技术基于摄像机技术，其传感器多条光通道分布在整个照射路径中，通过精心设计的结构，将输入的不同波长的光线分别传递到每条光通道中，从而对物体反射出的光谱范围进行处理，产生图像。而极紫外光刻技术则是基于传感器与结构本身的相互作用关系，其结构由多个光通道组成，具有多光通道输入、自由扩散、自由吸收、自由折射的特点，可以观察各种复杂的物体，提供很高的分辨率、精度。

3. 发展方向：深紫外光刻技术有着严格的工程约束和仪器设计，由于传感器的结构固定在摄像机前面，其最大深度受限于摄像机本身的性能限制。而极紫外光刻技术是一种基于新的反射理论的光刻技术，其结构可以完全独立于摄像机，且可以用成像管或显影结构自成一体，具有非常大的自由度和开放性。极紫外光刻技术的发展已经取得了令人瞩目的成就，其功能和特性正在被广泛应用。
