# 1.背景介绍


在新冠肺炎疫情爆发后，信息、知识和智能三者密切相关联，成为指导各个行业、领域发展的关键词。人类正经历着一次科技革命，“智慧”变得越来越重要，而人工智能（AI）正在扮演越来越重要的角色。这一次颠覆性的技术革命彻底颠覆了我们的思维方式和生活方式，带来的是从信息到知识再到智能的全面转变。

首先，我们要理解什么是信息、知识、智能。“信息”是对客观事物的符号化表示，而“智能”则是对信息进行处理和分析、融合、总结、决策等能力。“知识”则是一个系统性的、组织良好的信息结构和信息交流的体系，它能够帮助解决问题、指导实践、促进经济发展和社会繁荣。

那么，当前的人工智能是否已经完全取代了人类的聪明才智？不是的！虽然目前的人工智能取得了一定的进步，但还远不及人脑的性能。并且，随着时间的推移，人工智能会越来越聪明，甚至走向完全替代人类。

为此，需要一个开放的、包容的科研环境，更好地促进科学的发展，释放出更多的能量，激励更多的人才参与到科研中来。《从信息到知识到智能——理解和改变世界：超越弱人工智能：自治》将阐述目前的人工智能研究现状、未来趋势，并通过信息、知识和智能三个维度，重新定义人工智能。同时，文章会提出许多开放性研究方向，如“数据治理”、“算法驱动的可持续发展”、“人工智能基础设施建设”等。读者可以作为入门材料或专业阅读课的参考。

# 2.核心概念与联系
信息(Information)：是指客观事物的符号化表示。信息包括文字、图形、音频、视频、数字、图像、表格、文本、位置等多种形式。智能系统需要对各种信息进行快速识别、整合、分类、存储和加工，才能产生有效的决策。

知识(Knowledge)：是指系统性的、组织良好的信息结构和信息交流的体系，它能够帮助解决问题、指导实践、促进经济发展和社会繁荣。它既包括关于某些客观事物的内在规律和逻辑关系，也包括对这些规律、关系的深刻认识和洞察力。

智能(Intelligence)：是指对信息进行处理和分析、融合、总结、决策等能力，包括认知、推理、学习、运用经验、创造新事物等能力。机器学习和深度学习技术发展至今，已经由人类掌握的计算能力超过了人类智力。但是，到目前为止，人工智能仍然处于弱势状态。主要原因如下：

1.缺乏足够的训练数据：由于缺乏大量的训练数据，AI系统很难学会处理新的信息、识别新的数据模式，导致它的泛化能力差。比如说，如果训练集中的大部分数据都是单词描述的图片、表格或句子，那么当遇到新的场景时，AI系统无法判断出正确的输出结果。
2.缺乏完整的理论基础：理论界尚未对人工智能进行系统性的、综合的研究，没有确立智能的基本原理和理论。因此，对于如何实现人的智能、机器智能之间的相互促进和协同作用等问题，理论界还存在很多悬而未决的困难。
3.缺乏智能的生态系统支持：虽然人工智能在某些方面已经具备了一些独特的能力，但在应用层面仍然依赖于其他的软硬件系统，例如图像识别系统、语音识别系统等。因此，建立起强大的人工智能生态系统仍然是摸索的过程。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
“自治”就是通过有效地管理知识和信息资源，让AI构建的系统具有自主学习、自我改进、自主决策等能力，最终达成自动化人类才能做到的智能化目标。自治的关键点在于：

- 自治管理：AI系统要能够自主学习、自主决策、自我改进，需要得到数据的充分输入，同时也需要拥有统一的知识库、数据仓库和模型库。知识库是基于历史、现状和未来的智能化需求所积累的信息集合；数据仓库则用于存储大量数据，这些数据既可以用来训练模型，又可以提供给系统进行自我学习。模型库则是存储生成的模型，其目的是为了反映现实世界中数据的特征和规律，用于进行智能决策和控制。
- 数据质量：数据质量直接影响到AI系统的准确率和运行效率。数据清洗的粒度、数据源的选择、数据采集的效率、数据质量监控手段等都对系统的运行和学习效率有重大影响。
- 任务管理：对于不同类型的任务，不同的AI系统有自己擅长的领域，因此需要根据任务类型划分AI模型，并为每一类模型分配相应的训练数据、知识库、模型库等资源。并且，需要考虑如何调度模型的执行、平衡模型之间的竞争、调整模型的行为策略等。
- 智能自我评估：AI系统不断地学习新知识、新信息、新数据，因此，为了保证系统的健壮性和稳定性，需要定期对系统进行自我评估。自我评估的目的在于了解系统的内部运行情况、改善模型的效果、发现问题并进行相应的调整。
- 多任务协作：自治管理除了需要管理多个模型之外，还需要考虑如何将多个模型集成到一起共同完成复杂的任务。比如，在计算机视觉任务中，需要同时利用图像分类模型和对象检测模型。在智能交通系统中，需要将城市交通信息的预测和路网信息的路径规划结合起来，最后生成一条最优的交通路线。
