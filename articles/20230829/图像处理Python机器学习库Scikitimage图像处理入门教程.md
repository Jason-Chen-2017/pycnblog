# 1.简介
  

## 1.1 什么是图像处理？
图像处理（Image processing）指的是从传感器、摄像机或扫描仪等设备捕获到的二维或三维图像信号，经过某些算法处理之后得到分析结果或者所需的输出。图像处理技术的广泛应用已经成为社会生活中的重要组成部分。

## 1.2 为什么要用图像处理？
由于数字信息的爆炸式增长和互联网、手机等新型通信技术的普及，使得图片成为我们获取和交流信息的主要途径之一。因此，掌握图像处理技术对于研究人员、工程师、科技人员、商业专家、媒体工作者、教育工作者等各行各业都十分重要。例如，图像处理技术用于航空航天、医疗Imaging、金融证券投资、科技创新领域，还有国防军事、科研生产领域等。

## 1.3 Python的应用领域
Scikit-image是基于Python的机器学习库，它提供许多功能强大的图像处理函数。在图像处理领域，Python是最常用的语言，其中包括计算机视觉、图形学、人工智能、机器学习、数据分析、网络安全、运维管理、云计算等领域。因此，通过对Scikit-image的深刻理解和实践，我们可以更好地解决图像处理相关的问题。

 ## 2.2 Scikit-image简介
 Scikit-image是一个基于Python的开源机器学习库，用于进行图像处理。其提供了许多用于数学变换、特征检测、图形学运算、滤波、分割等的函数，帮助用户快速进行图像处理、计算机视觉、机器学习等领域的研究。Scikit-image适合于以下领域的研究人员：
 ### 1) 图像处理与计算机视觉：图像处理包括滤波、锐化、边缘检测、形态学变换、轮廓提取、分类、分割等。而计算机视觉则涉及图像特征提取、目标识别、跟踪、配准等。
 ### 2) 生物医学与生态学：生物医学图像处理有解析、体积影像、超声影像、CT影像等。而生态学图像处理包括植被监测、林业遥感、水下图像识别、卫星图像处理等。
 ### 3) 数据科学与分析：数据科学与分析需要对海量的图像数据进行整理、分析和处理，Scikit-image提供便利的方法进行这一流程的实现。
 ### 4) 游戏开发：游戏开发中，图像处理是渲染、碰撞检测、角色控制等技术的重要组成部分。Scikit-image提供了一些帮助游戏制作人进行图像处理的工具。
 ### 5) 深度学习：深度学习中的图像处理方法则用于数据增强、图像合成等。Scikit-image提供了一些用于深度学习的图像处理函数。

## 3.2 显示图像
 有两种方式可以显示图像。第一种是直接使用matplotlib库的imshow()函数。
 ``` python
import matplotlib.pyplot as plt
plt.imshow(img)
plt.show()
``` 
这种方式会生成一个窗口来显示图像。第二种方式是在notebook中直接绘制图像。
 ``` python
%matplotlib inline
from IPython.display import Image
``` 
这种方式在notebook中会把图像嵌入到页面。
