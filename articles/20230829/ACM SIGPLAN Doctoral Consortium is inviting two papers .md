# 1.简介
  

ACM SIGPLAN（Association for Computing Machinery Special Interest Group on Programming Languages）是一个著名的计算机科学会议，其编辑部由美国计算机学会（ACM）牵头，每年举办一百多场国际会议，邀请论文征稿。而SIGPLAN Doctoral Consortium，又是一个独立的评审委员会，其任务是通过一系列讨论、审核、选题以及最终成果的发布，为高水平计算机编程领域的研究工作者提供一个平台。今年的Doctoral Consortium将于2021年4月在线上举行，有机会接受计算机视觉、自然语言处理等领域的新生论文。因此，本期文章向国内外计算机视觉、自然语言处理等领域的优秀研究人员征集最佳论文。欢迎参加此次活动！

# 6.报告文献参考资料

## 一、计算机视觉

### 1. OpenPose: Realtime Multi-Person Keypoint Detection and Pose Estimation Using Part Affinity Fields (ICCV 2019)

文章描述：OpenPose 是目前比较火的计算机视觉技术之一，它可以估计出人体关键点的位置，并给出人体姿态。它的网络结构基于“分离卷积网络”，使用三个卷积层，前两个用于提取特征，第三个用于估计关键点和姿态。但是，由于分离卷积网络存在参数冗余和对空间大小的敏感性，导致网络推理速度慢、内存占用过高，无法实时处理视频流。此外，OpenPose 还没有自适应地调整关键点数量和位置，而这些都是需要针对特定应用需求进行优化的。本文作者提出了一种新的关键点检测方案——Part Affinity Field (PAF)，该方案可以精确确定不同部位之间的相互作用，同时消除了空间上的限制，使得OpenPose可以处理任意数量的人体。最后，作者构建了一个深度学习框架——Lightweight OpenPose，可以运行在移动设备和嵌入式系统上，对视频流进行实时处理。

作者认为，本文的工作将计算机视觉技术带到了实时的状态，为关键点检测和姿态估计提供了新的思路。另外，使用 PAF 可以有效地解决因参数冗余或空间限制导致的问题，进一步推动了计算机视觉的发展。


### 2. EfficientDet: Scalable and Efficient Object Detection (ICML 2020)

文章描述：本文中，作者提出了一种通用的目标检测器架构——EfficientDet。EfficientDet 的主要特点是在不同尺度下都保持高效率，保证了检测速度快且精度高，同时兼顾了准确率与模型大小之间的权衡。

作者使用FPN作为基干网络，结合了多种锚点机制，根据目标的大小动态调整检测单元的数量，降低计算量和内存占用，减少模型大小。作者还提出了一套改进的损失函数，增强模型对背景、小目标、负样本的鲁棒性。最后，作者也提出了一种优化策略——标签平滑（Label Smoothing），即在训练时采用噪声标签平滑模型的预测结果，避免模型陷入局部最优。

作者认为，本文的工作总结了目前最先进的目标检测技术，成功地将性能、效率和准确率进行了充分的权衡。


## 二、自然语言处理

### 1. XLNet: Generalized Autoregressive Pretraining for Language Understanding (ACL 2019)

文章描述：XLNet 是 Google 团队在 2019 年 ACL 大会上提出的预训练语言模型，其效果超过 BERT 和 RoBERTa，取得了非常大的突破。XLNet 在保持最大深度的同时，在预训练阶段与微调阶段的处理方式完全一致，因此能够有效解决双塔设置带来的梯度消失的问题。本文详细介绍了 XLNet 模型的结构，以及如何利用其预训练特性来实现通用的语言理解任务。

作者发现，XLNet 的关键在于引入一种新的预训练目标——“递归交叉熵（Recurrent Cross Entropy）”。这种目标鼓励模型在生成长序列的过程中，关注序列的前面时刻的信息。作者证明了这种方法可以在预训练过程中学习到通用的上下文信息，而不是仅仅依靠单句的语言建模，而这是传统的语言模型所欠缺的。

作者还指出，XLNet 有着令人惊讶的性能，在许多 benchmarks 上超越了目前最好的模型，比如 GLUE、SQuAD、WikiText-103 等，甚至达到了 SOTA。
