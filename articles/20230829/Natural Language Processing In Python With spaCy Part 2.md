# 2. NLP入门
## 2.1 什么是NLP？
自然语言处理（Natural Language Processing，缩写作NLP）是指通过计算机从文本等数据中提取结构化信息，使之变得更加易于理解和分析。其目的是识别、理解和生成人类语言。这一过程涉及自然语言的语法、句法、语义等多方面知识。比如，词汇分析、句法分析、情感分析、语音合成、机器翻译、信息检索等都是NLP的应用领域。其中，信息检索是最常见的任务，是NLP中的“问答系统”或“文本分类”等子领域。

## 2.2 自然语言处理的任务
自然语言处理主要包括以下几种任务：
- 词汇分析：自动提取文本的关键词、短语或词组。
- 情感分析：识别文本的情感倾向。
- 命名实体识别：识别文本中人名、地名、机构名等实体。
- 句法分析：对文本进行分句、分词和词性标注。
- 语义分析：从文本中提取语义特征并做出推断。
- 文本分类：根据文本的主题、类型、作者等，对文本进行自动分类。
- 机器翻译：将一种语言的文本转换为另一种语言。
- 文本摘要：从长文本中生成摘要。

## 2.3 spaCy介绍
spaCy是一个用于处理和分析文本数据的开源机器学习库，能够快速轻松地完成各种NLP任务。基于Python开发，支持Windows、Linux、MacOS平台，并提供API接口。其主要功能如下：

1. 中文分词：中文分词工具可以将中文句子切分为词语。
2. 词性标注：spaCy对每一个词赋予正确的词性标记，如名词、动词、形容词等。
3. 命名实体识别：spaCy可以自动识别文本中的人名、地名、组织机构名、日期、时间等命名实体。
4. 依存句法分析：spaCy可以分析每个词语与其所依存的其他词语之间的依存关系。
5. 词汇资源扩展：spaCy支持用户扩充自定义词典、实体类型、规则等。
6. 模型训练：用户可根据自己的需求训练自定义模型。
7. 可视化工具：spaCy提供了丰富的可视化工具，如dependency parse tree、part-of-speech tags等。

## 2.5 Hello World
下面我们用spaCy编写一个简单的Hello World程序。该程序展示了如何导入spaCy并创建nlp对象。
```python
import spacy

# 创建nlp对象
nlp = spacy.load('en_core_web_sm') # 使用英文语言模型

text = "Hello world!"
doc = nlp(text)

print(doc.text)
for token in doc:
    print(token.text, token.pos_)
```
输出结果为：
```
Hello PROPN
world VERB
! PUNCT
```
