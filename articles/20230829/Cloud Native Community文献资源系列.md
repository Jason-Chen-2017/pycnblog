# 1.简介
  

## 云原生社区是什么？
云原生社区（Cloud Native Community）是一个由众多优秀云原生开源项目组成的社区，聚集了来自不同公司、组织和个人的志同道合的开发者、用户和云原生爱好者，通过分享云原生相关的经验、教程、工具及最佳实践，促进云原生技术的共建。社区目前已经有超过60个开源项目，其中有许多具有丰富的功能特性，例如Service Mesh、微服务、DevOps等等。随着云原生技术的不断发展，云原生社区也在不断壮大。

# 2.背景介绍
## 什么是云原生？
云原生(Cloud Native)是一种构建和运行可移植且可伸缩应用的技术方法论，旨在将云平台作为开发、测试和部署应用程序的环境。云原生应用是一个关注应用生命周期，以短板减治愈的方式设计和构建应用。云原生应用程序适用于容器、无服务器计算、微服务、自动扩展和动态伸缩的分布式系统架构。它的目标是使应用能够轻松地部署、扩展和管理，从而在云上或其他任何地方运行，而无需关心基础设施级别的问题。
云原生的定义是一种基于一组定义良好的标准、原则和模式的集合。这些标准、模式和最佳实践鼓励使用声明式配置而不是脚本来管理应用程序。云原生带来的新特性包括：快速迭代、弹性伸缩、服务发现、服务绑定、可观察性、负载均衡和监控等。
## 为什么要学习云原生？
随着云计算、容器技术、微服务架构和CI/CD流程的日益普及，越来越多的人们希望自己的软件系统能够更好地适应变化、弹性伸缩和服务于大量用户。而学习云原生技术正是为了能够利用云计算的高效率、可靠性和弹性资源，提升系统的性能、可用性和可维护性，适应业务快速增长的需要。了解云原生技术，能让你在实际工作中受益匪浅。
