# 2. 基本概念术语说明
# 1.1 什么是文学？
文学（英语：Literature），又称写作，指从认识到表达存在于有形文化中的各种信息、观念、价值观，以及关于自身社会生活和个性的人类活动的各种故事、文体和材料。一般来说，文学是由写作者与其他参与者通过书信、笔记、绘画或影像等媒介写成的一部长篇作品。[1] 

# 1.2 为什么要写作？
写作是一门学问。它与其他多种艺术相比，有着独特的优势。
- 一是写作可以将复杂的心智和直觉进行表达，从而使人们能够洞察世间万物之奥秘；
- 二是写作可以透过语言的符号、图像、声音、味道，来传达一段真正属于自己思想的东西；
- 三是写作能激发思维能力、沟通能力、感情表达能力、领悟能力，促使人们在日常生活中不断追求自我完善、自我提升；
- 四是写作可以促使我们审视社会，重新认识自我，反省个人，发现真相，找到与他人的共鸣。

总而言之，写作是一种以思辨的方式对外界世界进行深入、准确的描述，这是所有一切艺术都无法企及的能力。[2]

# 1.3 文学分类
文学可按主题来分，如科幻小说、推理小说、武侠小说、传记、爱情小说、诗歌等。[3]

科幻小说：主要描写虚拟现实、银河系、死亡、时间旅行、未来等科幻题材的长篇小说。[4]

推理小说：以推理小说为代表，以推理为主线，推动读者从各个角度出发，展现当代人类社会面临的种种疑难问题。[5]

武侠小说：以武功道德、兵器装备、剧情为中心，呈现江湖奇侠传奇。[6]

传记：以个人命运为主题，记录个人生平及其所处时代。[7]

爱情小说：小说以纯爱情为题材，刻画两性之间的情感纠葛。[8]

诗歌：也称浪漫主义文学，以对生命的热烈向往和追求，寄托于对生命的渴望和追求，是一种生活方式。[9]


# 1.4 怎样开始写作？
首先，了解自己想要写些什么吧！要写的内容不能太偏离自己的生活，但必须具备一定层次的重要性。例如，你最近在看一本书，想写一篇读后感来总结这本书，但又不得不先去做一些功课，比如复习相关章节。同样，如果你是从事新闻工作、公众关系工作，要写一篇影响力很大的评论文章，建议先考虑一下读者的期待、收益、满意度等因素。最后，确认自己的写作目的后，找个稿酬低廉的地方，购置一张好版式的纸质素材，开始创作吧！

# 1.5 适合写作的岗位
其实，无论是在社会上还是在写作专业，都有许多适合写手的岗位。这里列举一些比较典型的岗位：

新闻编辑、写手：负责编辑或整理新闻报道、社论等文稿，具有快速、敏捷的处理速度。

副刊编辑、编辑：负责杂志副刊的撰写、编辑工作，担任此职务的人通常具有较强的写作能力、组织能力、团队合作精神，适合于协助出版社进行编辑工作。

美术编辑、设计师：负责插画、照片、视频等作品的创作与编辑，具有丰富的创意设计技巧和精湛的图形构图能力。

文案策划、导演：负责广告、宣传片的策划、编排，具有良好的动手能力、统筹能力、创意思维。

教育培训、老师：负责写作教材、作文、试卷，具有深厚的文学修养、学科基础。

翻译、口译：负责译制文学、电影、广播剧等文本，担任此职务的人通常具有强烈的语言风格，具有良好的口头表达能力。
