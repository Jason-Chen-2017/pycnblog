# 2. 基本概念术语说明
## 2.1 概念和术语
### 2.1.1 分布式账本

分布式账本（Distributed Ledger Technology，DLT），是指用分布式网络技术来存储和记录所有交易信息的一个数据库，其核心特征包括：

1. 数据存储的分布性：任何人都可以加入到系统中，参与数据的共同体，所以系统不会像中心化系统一样只存在于少数机构或个人手里。分布式账本具备透明、非可靠、不受审查、永久、开放、容错等特性，因此能够抵御各种攻击，并能够很好的处理海量的数据。

2. 数据不可篡改：由于每笔交易都是记录在全局账本上的，因而任何人都无法篡改历史交易记录，使得整个系统具有真正意义上的不可篡改和不可伪造的特点。

3. 数据可追溯：系统记录每笔交易的具体信息，所以任何人都可以查询到历史信息，这也是其非中心化的特征之一。

### 2.1.2 比特币
比特币是第一个实现了分布式账本的平台，其独有的特征就是独一无二的算法，即POW(Proof-of-Work)算法。POW算法可以确保该平台上每一笔交易均需消耗一定量的算力，也就是计算资源、电力、内存等等，保证其去中心化和安全。目前比特币的总量已经超过21亿枚，占全球最大的比特币供应。

### 2.1.3 以太坊

以太坊是建立在区块链技术之上的平台，是一个开源、公开、点对点的虚拟世界。它的独特之处在于采用账户模型，所有的资产都由用户私钥控制，不同于传统银行账户体系，这种机制可以让用户的资产和交易受益匪浅。同时，以太坊还有很多其他特性值得关注，比如智能合约、去中心化自治组织以及扩展性方面等。
