# 1.简介
  

在过去几年里，通过语音识别（ASR）技术将口语语音转换成文字形式的过程越来越流行。由于音频数据量的爆炸式增长，因此ASR模型需要考虑各种计算资源和超参数设置。语言模型（LM）可以用来提升ASR模型的性能，并对声学、语言、上下文等进行建模，可以有效地处理噪声和停顿，从而提高识别精度。最近，谷歌团队团队发布了多种大小的ASR语言模型及预训练模型，例如“QuartzNet”、“Conformer”、“Jasper”。本文就介绍一下这些模型的详细信息。

语言模型与语言模型训练方式
语言模型是一种基于统计的方法，能够估计句子中各个词语的出现概率。它由一组单词组成的概率分布表现。即给定一个句子w，语言模型的目标就是计算出p(w)这个概率。语言模型通常包括N-gram模型、最大熵模型、混合语言模型、神经语言模型等。而语言模型的训练方式则有基于全局语料库的方式、基于传统机器学习方法的分层或联合训练的方式以及端到端（end-to-end）训练的方式。

# ASR模型概述
ASR模型的工作流程如下图所示。首先，输入的音频信号经过一个声学前端组件后得到MFCC特征，然后用CNN或者Transformer提取特征序列。之后用语言模型来预测下一个最可能的输出符号，或者计算语言模型对每一个可能输出的概率值，再选择概率最高的那个输出。


上图中，左侧的声学前端模块包括特征提取器、噪声增强器、窗函数等；右侧的特征序列送入一个卷积神经网络或transformer进行特征抽取。特征序列经过降维、变换和投影后作为语言模型的输入。语言模型接收到特征序列后，首先用N-gram模型或其他模型计算出概率分布p(w|h)，其中w表示当前要识别的文本，h表示历史文本序列。这里的历史文本序列指的是识别过程中已经识别出的一些字符序列。之后，语言模型会根据概率分布对下一个最可能的字符进行预测。最终，语言模型的输出作为ASR模型的输出，用于合成对应的语音信号。

# QuartzNet

QuartzNet中的卷积层由三层组成，第一层是时间卷积层，第二层是frequency卷积层，第三层是混合卷积层。时间卷积层中的卷积核具有时间性，能够捕获到时序信息；frequency卷积层中的卷积核具有空间性，能够捕获到空间信息；混合卷积层将两个卷积层的输出混合起来。
