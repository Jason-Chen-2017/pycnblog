# 1.简介
  

## 1.1 概念
图神经网络（Graph Neural Network, GNN）是在图结构数据中应用深度学习的方法。它能够捕捉到图结构数据中的复杂关系并提取出有用的信息。GNN的主要特点包括：
1. 可扩展性：GNN可以用于处理任意大小、形状和结构的图，而且可以在多种任务之间进行通用化。
2. 模块化：GNN模块化的设计允许将不同类型的子网络组合起来，提升性能和效率。
3. 自适应性：GNN可以通过学习节点或边上的特征自动发现图结构上有效的模式，因此无需预先定义特征。
4. 全局观察力：通过学习全局表示来捕获整个图的特性，有利于解决图形数据的关联性问题。

图神经网络的三个关键要素如下：
1. Message Passing Mechanism: 信息传递机制，用于对图中的节点传递消息。
2. Aggregation Operation: 聚合操作，用于融合来自邻居的信息。
3. Readout Function: 输出函数，用于整合不同邻居的信息。

图神经网络的演进过程：
1994年，R&S提出了图卷积神经网络GCN，这是第一个在图数据上的深度学习模型。
2017年，微软亚洲研究院团队提出了变分图卷积网络VGCN，采用自动学习的高阶邻域网络结构。
2018年，腾讯科技提出了图注意力网络GAT。
2019年，谷歌Brain团队提出了graph transformer。

目前，图神经网络已经成为深度学习领域中一种重要的技术，具有广泛的应用前景。由于其在图分析领域的独特优势和高度通用性，得到越来越多的关注。但同时也存在着一些短板和局限性，这些缺陷可能会影响到其真正落地的效果。因此，如何更好地理解、应用和改进图神经网络一直是一个重要的研究课题。

## 1.2 关键特征
图神经网络具有以下几个显著的特征：
1. 对图结构的建模能力：通过学习图上的全局表示和局部特征，能够建模出复杂的图数据结构。
2. 自适应的特征学习：无需事先定义特征，GNN 可以自动学习节点或边上的特征。
3. 模块化的设计：GNN 通过不同的模块实现对图结构的学习，如消息传递网络和读取函数。
4. 增量更新：GNN 在训练过程中能够快速迭代，可以利用历史数据进行学习和更新。
5. 拥有良好的全局观测能力：GNN 提供了足够的全局视图，能够捕获不同层次和范围下的图结构特征。
6. 端到端的优化：GNN 可以直接从输入数据中学习图结构及其表示，而不需要复杂的中间层，不需要手工设计特征。
7. 抗噪声和鲁棒性：GNN 在处理含有噪声的数据时有较强的抗噪声能力和鲁棒性。

图神经网络还有以下几个方面的突破：
1. 更多地关注局部信息：现有的 GNN 大都只关注全局信息，忽略了局部信息的学习。
2. 使用时空特征：GNN 使用时空特征进行信息编码。
3. 分布式图神经网络：分布式图神经网络提供了分布式计算框架，能够实现大规模图数据的训练和推理。
4. 深度学习的特异性：GNN 使用深度学习方法构建模型，能够提升图数据分析的精度和效率。
5. 自学习特征：图神经网络使用自学习特征学习节点或边上的特征。
