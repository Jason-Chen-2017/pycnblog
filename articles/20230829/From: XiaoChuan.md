# 2. 深度学习简介

## 1. 什么是深度学习？

深度学习（Deep Learning）是机器学习的一种方法，它是通过多层次的神经网络来解决某些特定任务的计算机模型，由输入数据经过处理后得到输出结果。深度学习系统通过堆叠层级的神经元互连，可以自动学习到数据的表示和模式，并从数据中发现隐藏的规律，达到比传统机器学习技术更好的性能。

## 2. 为什么要用深度学习？

1. 数据量大：深度学习可以处理大量的数据，这是其他机器学习方法所不能及的。

2. 模型复杂：深度学习模型的参数数量通常比其他机器学习模型参数数量少得多，因此可以很好地适应大规模数据集。而且，深度学习模型中的参数之间存在高度非线性关系，这种关系使得深度学习模型可以学习到非常复杂的特征表示，从而实现很高的预测精度。

3. 需要时间来训练：训练深度学习模型需要耗费大量的时间，但由于它的参数较少，所以训练速度也比较快。

4. 计算性能强：深度学习模型的训练、推断过程可以使用GPU进行加速，可以有效地节省时间和算力。

5. 泛化能力强：深度学习模型可以通过增加样本数量或训练更深层的模型来提升泛化能力。另外，深度学习可以用来解决某些特殊的问题，如图像分类、目标检测、语音识别等。

## 3. 深度学习的一些概念

- 神经网络：深度学习模型的基础单元叫做神经元（Neuron），组成一个神经网络。一个神经网络由多个神经元构成，每个神经元都接收来自上游的信号，然后进行加权运算得到输出。

- 层：每层代表了一个特定的任务，比如图像分类、文本分类、语音识别等。

- 损失函数：损失函数用于衡量模型的拟合程度。深度学习模型的训练就是在优化损失函数。

- 梯度下降法：梯度下降法用于更新模型的参数。

- 正则化：正则化是为了防止模型过拟合，减少模型的复杂度。

- 交叉熵损失函数：交叉熵损失函数是最常用的损失函数，用于衡量模型的预测准确率。
