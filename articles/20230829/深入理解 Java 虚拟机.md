# 1.简介
  

Java 是世界上最流行的编程语言之一，也是企业级应用开发的主要语言。由于其安全、稳定、跨平台等特性，Java 被广泛应用于各类关键任务领域，如后台系统开发、移动应用开发、分布式计算、Web 服务开发等。作为一门开源语言，它具有很强的扩展性和可移植性，并得到了广大的程序员的青睐。但是 Java 本身也存在一些不足，比如效率低下、运行速度慢、编译耗时长、内存占用大等。为了提高 Java 的运行性能，就需要对其底层的虚拟机做深入分析。本书通过系统地学习 Java 虚拟机（JVM）内部原理，重点讨论 Java 程序在运行过程中生成、执行字节码，以及虚拟机如何进行垃圾回收、性能调优等方面的知识。还会结合实际案例，深入剖析虚拟机优化原理、JVM 性能瓶颈及解决方案。
本书适用于喜欢阅读源码的人员，或者想要提升自己对 JVM 相关技术的了解的人员。该书可以作为 Java 程序员的必备读物，也可以帮助 Java 工程师更好地理解 Java 虚拟机，同时也能给想从事 Java 虚拟机研究工作的朋友提供一个系统全面、深入的学习教程。
