# 2.背景介绍

　随着互联网、移动互联网、物联网的蓬勃发展，人们越来越依赖智能手机、平板电脑、智能穿戴设备，获得海量的数据。越来越多的应用场景需要能够处理海量数据的智能系统，比如网页搜索、图像识别、自然语言处理、自动驾驶等。在这些应用中，人工智能（Artificial Intelligence, AI）就扮演着至关重要的角色。AI的研究始于上个世纪60年代末，那时候人们正在追逐智能机器人的野心，当时的计算机还是比较弱小的机器，只能运行一些简单任务。所以，机器学习（Machine Learning）被提出来作为一个新兴的研究领域。它的主要目标是利用已有的训练数据来建立预测模型，使得机器能够自己发现数据中的规律，从而对未知数据进行预测或者决策。

　机器学习的主要分为两类：监督学习（Supervised learning）和非监督学习（Unsupervised learning）。前者指的是机器人根据给定的训练数据进行训练，利用此数据建立预测模型，以便根据新的输入数据预测输出结果。后者则不需要任何训练数据，只需要原始数据，利用聚类、关联规则等原理进行数据分析，以发现数据的内在联系。

　　20世纪80年代，深度学习（Deep Learning）被提出来取代传统机器学习方法，成为主流的机器学习技术。它主要由多层神经网络组成，通过反向传播算法，通过梯度下降的方法不断调整网络权重，实现对数据的拟合，从而完成复杂任务的学习。深度学习成功的原因之一就是其具有高度抽象的特点，能够自动提取有效的特征，并且能够很好地处理非线性关系，从而获得更好的预测准确率。

　人工智能的发展可以归纳为以下几个阶段：

1. 符号主义阶段：诞生于1956年的图灵测试。在这一阶段，人们认为智能机器具备了理解和推理的能力。但是由于缺乏实用价值，因此没有产生太大的影响。

2. 连接主义阶段：诞生于1956年后期，主要是哲学家阿兰·图灵提出的一种思想，他认为人机交互应该是人与机器之间沟通的桥梁，并将这种沟通称为连接。当时还没有出现计算机，不过借鉴人类的思维方式，人们开始探索如何用计算机来模仿人类的行为。这场人机对话的盛宴也打破了神经网络研究的种种壁垒。但后来随着计算机性能的进步和图像识别、自然语言处理等领域的突飞猛进，人工智能迎来了更大的发展。

3. 认知主义阶段：主张要认识智能机器的本质，即它是基于感知和推理的能力，而非直接运算和判定。这一阶段的代表是小波分治算法，它利用信号处理的原理，对图像进行降噪和分割，并将得到的子图像作为新的输入，达到连续推理的目的。

4. 计算主义阶段：指的就是信息论、概率论、统计学、数理逻辑、编程、电路设计、计算机理论等计算机科学理论的创立者提出的概念，并且把它们应用到智能系统的开发中。主要的代表就是提出了深度学习的概念。这也是目前AI领域最热门的技术，也是本文主要介绍的内容。
