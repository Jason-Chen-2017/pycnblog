# 2.基本概念术语说明
## 2.1 TensorFlow
TensorFlow是由Google开发的开源机器学习框架，它是一种数据flow图编程环境。在这个图中，节点表示操作，边表示数据流动方向。可以利用该图定义任意的计算任务，比如进行张量计算、神经网络训练、强化学习等。它具备以下特征：

1. 数据流图

   TensorFlow中的数据流图是一种动态的计算图，它通过张量(tensor)之间的相互传输来处理输入数据和输出结果。这些张量代表着数据结构上的概念，比如矩阵或向量，这些张量的数据类型可以是整数、浮点数或字符串，可以被分割成多维的数组形式。

2. 自动微分

   TensorFlow采用自动微分技术，可以自动地求导并更新参数，从而达到更快、更准确的模型训练效果。

3. 并行计算

   TensorFlow支持多核CPU、GPU或分布式集群计算，可以同时运行多个操作，提升运算速度。

4. 可移植性

   TensorFlow可以在各种平台上运行，包括Windows、Linux、MacOS等。

5. 模块化设计

   TensorFlow提供丰富的模块化API，可以方便地构建复杂的神经网络模型。

总体来说，TensorFlow是一个很全面的、灵活的、功能强大的机器学习框架，它的图形计算能力、自动微分机制以及并行计算功能，都使得它能处理复杂的模型训练问题。

## 2.2 概念介绍
### 2.2.1 数据维度
TensorFlow的张量(tensor)是张量数组的统称，在机器学习领域，一般是用来表示矩阵、向量或者其他类型的数组。张量的维度决定了其存储数据的类型和大小。如下图所示，二维张量就是一个具有两个索引的矩阵，三维张量就是一个具有三个索引的矩阵，四维张量就是一个具有四个索引的矩阵。

### 2.2.2 占位符
在TensorFlow中，有些情况下会遇到不知道怎么填充的数据，这时就可以使用占位符，它表示输入数据的期望值，它的值在运行过程中才会确定。通常情况下，占位符是在创建计算图的时候定义，然后再传入实际的数据进行计算。例如：
```python
import tensorflow as tf
x = tf.placeholder("float", shape=[None, 2]) # 第一个参数表示数据类型，第二个参数表示维度
y = tf.placeholder("float", shape=[None, 1])
z = x + y   # 此时x、y都是占位符，不能直接用于运算
```

### 2.2.3 会话（Session）
在TensorFlow中，Session管理运行时环境，是运行图中的节点所需资源的上下文。每当想要执行一次计算时，就要创建一个新的会话对象，并将它传递给需要执行计算的函数。例如：
```python
with tf.Session() as sess:
    output = sess.run([z], feed_dict={x: [[1, 2], [3, 4]], y: [[5], [6]]})
    print(output[0])
```

## 2.3 Python编程语言
Python是一门高级的编程语言，被广泛应用于数据分析、科学研究、Web开发、机器学习等领域。Python支持动态类型系统和垃圾收集机制，允许程序员无需指定数据类型也可以轻松地编写程序。同时，Python还有许多优秀的第三方库，可以让开发者大大降低开发难度。

## 2.4 NumPy
NumPy（Numeric Python）是一个用于对多维数组进行数值运算的库。它提供了大量的数学函数，可以方便地进行数据处理，尤其是在线性代数、统计学、以及傅里叶变换等方面。NumPy非常适合作为不同项目之间的接口，因为它既可以用于本地数据处理，又可以方便地与其他工具整合在一起。

## 2.5 Pandas
Pandas是一个数据分析库，它可以高效地处理结构化数据，并且提供各种数据清理、分析、合并、重塑等功能。它内置的数据结构是Series（一维序列）和DataFrame（二维表格），可以轻松地将原始数据转化为结构化的数据集。

## 2.6 Matplotlib
Matplotlib是一个著名的绘图库，它可以用于创建复杂的图表、直方图、条形图等。它提供了简单的接口，可以很容易地创建各种二维图形。Matplotlib可以与NumPy、SciPy等工具结合起来，做出更多高级的图表。

## 2.7 Seaborn
Seaborn是一个Python数据可视化库，它主要是为了方便创建统计图形。它提供了更高级的接口，可以帮助用户创建更加美观的可视化。它可以与Matplotlib、Pandas等库一起使用，进行更加复杂的可视化。

## 2.8 Scikit-learn
Scikit-learn是一个机器学习库，它提供了许多优秀的分类、回归、聚类、降维等算法，可以极大地提升数据处理的效率。它集成了一些常用的机器学习算法，并提供了统一的接口，可以很容易地实现不同的模型。

## 2.9 Keras
Keras是一个高级的神经网络库，它提供了简单、可拓展的接口，可以快速地搭建深度学习模型。它可以与TensorFlow、Theano等深度学习框架或工具配合使用，实现模型训练和预测。

## 2.10 Caffe
Caffe是一个开源的深度学习框架，它最初由Berkeley Vision and Learning Center（基于林肯大学和麻省理工学院的计算机视觉和机器学习中心）开发。Caffe提供了一种高度模块化的设计方法，可以方便地进行模型组合和参数共享。

## 2.11 Torch
Torch是一个开源的深度学习库，它可以运行Lua脚本，具有简单而灵活的接口。它最初由斯坦福大学开发，目前由Torch基于LUA语言开发。Torch提供动态、显式的计算图，可以很好地实现跨平台部署。
