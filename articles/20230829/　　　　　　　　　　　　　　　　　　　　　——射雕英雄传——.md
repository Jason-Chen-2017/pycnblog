# 1.简介
  

“射雕英雄传”系列是百度公司出品的一套《射雕英雄传》、《神雕侠侣》、《笑傲江湖》、《倚天屠龙记》、《连城诀》等名著改编的网络电视剧。根据剧情推进时间顺序由古装到武侠再到现代科幻四个阶段。不同时期的角色及其互动情节构建了鲜明的文化风格。每个阶段都收到了不同程度的打磨，有的角色独具魅力，有的角色默默承受着重压，也有很多角色历经艰辛却依然意气风发。除了众多优秀的演员之外，百度还邀请了极具影响力的历史学家、作家、艺术评论家等人物参与剧集制作。为了更好地传播这一伟大的文化，百度选择了一个原创题材——射雕英雄传。该系列用了大量的资料、图片和影像素材拍摄制作，而视频制作则依赖于业内顶尖的技术人员和高质量的团队。
# 2.知识结构
射雕英雄传的知识结构可以概括为如下几个方面：

⑴ 历史：主要了解中国五代十国、两汉、魏晋南北朝、唐宋元明清及近现代史上众多重要事件的发展过程。

⑵ 政治：主要了解战争的起源、战争对国家和社会产生的影响以及军事、政治斗争的形式、组织机构和政策制定过程。

⑶ 文化：主要了解中国绘画、书法、音乐、舞蹈、戏曲、民俗、航海、旅游、兽医等众多方面的风貌。

⑷ 社会经济：主要了解社会生产方式的变迁、社会阶级分化、个人生活层次的提升、贫富差距的加大以及劳动力市场的不断扩张。

⑸ 技术技能：主要了解电脑、手机、微型计算机、汽车、飞机等产品及其制造方法、互联网的发展、通信设备及应用、信息安全保障、金融系统运作等领域的应用基础。

# 4.角色分析
射雕英雄传中主要有两种类型的人物：

⑴ 真人：《射雕英雄传》中的人物都是生活在当时社会各阶层的人物，他们的个性、肤色、智慧、品行与日常生活息息相关。其中，曹操、刘备、荀彧、袁绍、段延庆、张辽、李白、司马懿、欧阳锋、吴三桂、秦明、乔峰、欧阳予倩等角色是代表性角色，也是百度想传达的最重要的人物群体。

⑵ “雕”：《射雕英雄传》的角色都是以雕塑的形式出现，这些人物往往来自五代、六朝、七王、八胜等等不同的历史时期，为何同属一类？由于其年龄、阅历、禀赋以及明察秋毫的眼光，《射雕英雄传》中的“雕”让观众看到了一批革命领导人的英雄主义风范。他们所付出的牺牲与付出最终会得到回报，成为经验和教训的见证。这些人的形象与精神风貌深深影响了后续的历史人物。

# 5.故事背景
《射雕英雄传》改编自东周栎智社真人版的《神农本草经》、《鹿柴》，内容主要涉及五代十国的兴衰、百姓的艰难生活、和平变乱、民族的交往、文化的积淀等事件。该系列一共分为五个时期，每一个时期由不同类型的角色按照历史人物的生死时间进行呈现，故事的时间跨度很长。因此，观看者需要对整个系列的时空范围保持敏感，并结合自己的理解、知识背景进行真切的体验。

# 6.主角
主角介绍：
曹操：曹操（又称曹丕），字孟德，号青莲居士，江西永新人，昭武元年（公元210年）由太子丹建立中央政权。汉武帝即位后，陈家、呼延赞率军讨伐黄巾军，曹操于是率兵潜入黄巾军营中阻止，并率军奋勇向敌发起进攻，成功击破敌人，被推举为总兵首领。任用曹操为第三公九卿，以取代杨广。12岁时，他遭遇封建贵族骄横的袁绍，被立为中郎将，曹操怒斥骂其“贱种”。之后，他经过博弈，被迫弃官逃到海外，参与创办商鞅文献。幼时，因诈骗受尽折磨，父亲病亡，他便随母亲在远方寻找知音。在仙台师范学校学习，晚年遇到满清王朝的统治，成为职业小说家。

司马懿：司马懿（原名马超），字子建，号望远门客，后世人多称之为望远门旧臣。景泰初年加入了周朝，因表现得好而被封为吴三桂，徙居下邽，继续筹划中原大计。12岁时，他与谢灵运等人等人一起被送往楚国西安途中遇到逆旅顺口溜。司马懿被带往那儿时，先是见识周围的奇景，以为自己已经逢凶化吉，但看到其年轻美丽的容颜，又自嘲道：“真是越来越有眼光了！”。司马懿苦熬半辈子，终于等到蒙受周朝践踏的官爵家产。为了报答，他登基为吴王，称帝为“中国第一吴”，改元祥和。 1949年，因为反对苏联侵略而被迫离开吴国。 司马懿后来担任革命委员会主任，领导创建并实施革命计划，保护中华民国的独立、自由、平等。

李白：李白（名白居易），字孟轩，号青石先生，又号青石姑娘，别署司馆、仕伯、工伯，陕西剑门人。北宋末年进士，授太常少卿衔，官至光禄大夫。因擅长治水、恢复民间秩序等综合修养，被安排为国子监掌管文字、教育、考试事务。后历任礼部侍郎、钱塘副使、相国州县令、韩信州县令。一生坎坷，壮烈投笔。

欧阳予倩：欧阳予倩（生卒年不详），字建强，湖南湘乡人，洪武元年（公元211年）进士，授修撰，官至郎中，同年在官上改名欧阳修。曾任安阳府令、淮阴府推官、刘府大将军、中山县尹、枢密院侍读学士、工部侍郎。后到广东钦州、惠州等地任职，游学京津、粤海等地，专门从事甘蔗、茶叶的加工和销售。 其作品以描写农园春节饮食、收获季节、景色风光等元素为主，具有浓郁的民间气息。

乔峰：乔峰（生卒年不详），字子腾，号千峰，三国志云门人，上党降敌。二十三岁开始自学功夫，练习成长，担任了武林盟主。入封定军山，史称定军山乔峰。刚晚出师伐吴国，吴败，乔峰率众追击。被吴王指派，出援与群臣并肩作战，宣布与吴国决裂，追封定军山东郡郎中，受封为援军大将军。之后历任襄樊守将、沙城侍从、阎公之弱、刘府尹、直隶州县令、樊州刺史。活跃于淮南政局，号称“过河卒”。他与韩信、李延炼等人均在事业上有一定影响，创下了将军系列的辉煌。
