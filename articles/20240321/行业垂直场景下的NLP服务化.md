# 4. 具体最佳实践：代码实例和详细解释说明

以下是一个基于BERT的文本分类服务的代码示例:

```python
import torch
from transformers import BertForSequenceClassification, BertTokenizer

# 初始化BERT模型和分词器
model = BertForSequenceClassification.from_pretrained('bert-base-uncased')
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')

# 定义文本分类服务的API
def classify_text(text, label_list):
    # 对输入文本进行tokenize和转换为tensor
    input_ids = tokenizer.encode(text, return_tensors='pt')
    
    # 通过BERT模型进行推理
    output = model(input_ids)[0]
    
    # 获取分类结果
    probabilities = torch.softmax(output, dim=1)[0].tolist()
    predicted_label = label_list[output.argmax().item()]
    
    return predicted_label, probabilities

# 示例使用
label_list = ['positive', 'negative', 'neutral']
text = "This is a very good product, I highly recommend it!"
predicted_label, probabilities = classify_text(text, label_list)
print(f"Predicted label: {predicted_label}")
print(f"Probabilities: {probabilities}")
```

在这个示例中,我们使用预训练的BERT模型和分词器,定义了一个文本分类服务的API。输入文本经过tokenize和转换为tensor后,通过BERT模型进行推理,得到分类结果及其概率。这个服务API可以很方便地集成到各种行业应用中,满足不同场景下的文本分类需求。
