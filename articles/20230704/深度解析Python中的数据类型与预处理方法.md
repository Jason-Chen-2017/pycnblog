4. 应用示例与代码实现讲解
---------------------------------------

4.1. 应用场景介绍
假设我们要收集用户的输入年龄，并判断其是否超过 18 岁。

```python
def is_age_over(age):
    if age < 18:
        return True
    else:
        return False

user_age = int(input("请输入您的年龄："))
if is_age_over(user_age):
    print("您已超过 18 岁")
else:
    print("您的年龄不超过 18 岁")
```

4.2. 应用实例分析
在上面的示例中，我们首先使用 `input` 函数获取用户的输入年龄。然后，我们尝试使用整型、浮点型、布尔型等数据类型来存储用户的年龄。接下来，我们使用 `is_age_over` 函数来判断用户的年龄是否超过 18 岁。如果年龄超过 18 岁，我们将输出 "您已超过 18 岁"，否则输出 "您的年龄不超过 18 岁"。

4.3. 核心代码实现
```python
def is_age_over(age):
    if age < 18:
        return True
    else:
        return False

user_age = int(input("请输入您的年龄："))

if is_age_over(user_age):
    print("您已超过 18 岁")
else:
    print("您的年龄不超过 18 岁")
```

4.4. 代码讲解说明
在此部分，我们将对代码的每个部分进行详细的讲解，以便您更好地理解。

### 函数 `is_age_over`
- `age`：输入年龄
- `return`：返回一个布尔值，表示输入年龄是否超过 18 岁
- `if age < 18:`：输入年龄小于 18 时，返回 True
- `else:`：输入年龄不小于 18 时，返回 False

### 输入年龄
- 使用 `input` 函数获取用户的输入年龄
- 假设输入为数字类型，直接使用 `int` 函数转换为整型

### 判断年龄是否超过 18 岁
- 使用 `is_age_over` 函数，判断用户输入的年龄是否小于 18 岁
- 如果年龄小于 18 岁，返回 True
- 否则返回 False

### 输出结果
- 如果用户年龄超过 18 岁，输出 "您已超过 18 岁"
- 否则输出 "您的年龄不超过 18 岁"
