```python
def fibonacci(n):
    if n < 0:
        return "输入的数值不能为负数"
    elif n == 0 or n == 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

for i in range(10):
    print(fibonacci(i))
```
