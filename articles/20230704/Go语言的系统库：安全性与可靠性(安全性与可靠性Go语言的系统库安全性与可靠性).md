## 应用示例与代码实现讲解
-----------------------

Go语言的系统库提供了许多有用的函数和工具,可以用于各种操作系统编程和系统调用。以下是一个简单的示例,演示如何使用Go语言的文件系统接口来创建一个新文件:

```go
package main

import (
	"fmt"
	"io/ioutil"
	"os"
	"strconv"
)

func main() {
	var input string
	var num int
	var newFile string
	fmt.Println("Please enter the number of the new file you want to create: ")
	fmt.Println("Please enter the name of the file: ")
	fmt.Scanln(&input)
	fmt.Println("File number: ", strconv.Itoa(num))
	fmt.Println("File name: ", input)

	// 创建新文件
	num, err := strconv.Atoi(input)
	if err!= nil {
		fmt.Println("Error: ", err)
		os.Exit(1)
	}
	input = num
	newFile = input + ".txt"
	err = os.Create(newFile)
	if err!= nil {
		fmt.Println("Error: ", err)
		os.Exit(1)
	}

	// 向新文件写入内容
	contents := []byte("Hello, world!")
	err = ioutil.WriteFile(newFile, contents, 0644)
	if err!= nil {
		fmt.Println("Error: ", err)
		os.Exit(1)
	}

	fmt.Println("File created successfully!")
}
```

此示例代码首先使用用户输入创建一个新文件,然后使用`ioutil.WriteFile`函数将内容写入到新文件中。
