2.3. 相关技术比较
----------------

混合优化方法与其他优化方法的关系主要包括以下几种：

* 动态规划：动态规划是一种有效的组合优化方法，主要用于解决满足以下两个条件的问题：
	+ 初始化状态时，元素值为 0 或无穷大；
	+ 状态转移方程中的每个元素都只与前面的元素有关，与其他元素无关。
* 贪心算法：贪心算法是一种自贪的优化方法，通常用于满足以下两个条件的问题：
	+ 初始化状态时，元素值为 0 或无穷大；
	+ 在状态转移过程中，每次都选择当前状态下最优的元素。
* 分治算法：分治算法是一种将问题分解成子问题的方法，通常用于解决满足以下两个条件的问题：
	+ 问题的最优解可以被分解为子问题的最优解；
	+ 每个子问题都可以被解决。
