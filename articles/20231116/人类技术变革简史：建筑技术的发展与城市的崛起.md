                 

# 1.背景介绍


技术日新月异的社会，带来了经济飞速发展、社会生活极大丰富。然而随之而来的技术革命却引发了一场全新的变化。
物质的繁荣不再满足人们的需要，而交换是人的本能。随着互联网的普及和信息化的到来，人的思维也开始由单纯的物质消费转向了更加开放包容的数字世界。技术革命带来了新的生产力方式——机器、网络、数据库和软件等工具。
人类历史上曾经历过两次巨大的技术革命，一次是农业革命，另一个就是计算机革命。两者都标志着技术和生产力的发展历史性变革。
在这次技术革命中最重要的两件大事就是建筑技术的崛起和城市的兴起。建筑技术包括瓦砖、木结构、砌块、石材、混凝土、玻璃、钢铁等等，这些技术的发明使得建筑的质量和形式发生了翻天覆地的变化。今天的建筑技术已经远超古代，甚至成为我们生活的基本元素之一。
此外，建筑的材料越来越便宜、通用性越来越强，各种零部件的组合可以创造出无限种类的产品，让人们从此可以免费获得无穷的想象空间。建筑领域的科学研究和工程实践日益成熟，新技术的发现正在推动着这个领域的前进。
至于城市的崛起则是近年来的热点话题。当前，全球都处于经济危机和产业升级的关键时期，随着人口的迅速增加，人们越来越注重居住环境，对建筑的依赖越来越强烈。而城市的出现正好可以应对这种需求。
为了增强城市的生活品质和功能，建筑学科提出的很多方案和观念也正在改变着人们的生活习惯。如有轨电车、智能窗帘、智慧路灯、智能厨房等等。
除了以上两大技术革命和城市的兴起之外，还有一些其他的突破性技术的出现。如太空探索技术、无人驾驶汽车、虚拟现实、数据加密等。这些技术将会推动我们的社会和生活进入一个全新的阶段。
因此，总体来说，技术革命是一个伴生着市场经济规律而发生的变化。科技的发展影响着生产力的革新，制造业得到的利润也越来越多元化。随着技术革命的不断推动，建筑学科的发展日渐蓬勃。而人类社会也越来越依赖于城市。技术的进步引领着人类社会的向前迈进，驱动着我们创造出美好的生活，迎接着未来的变革。
# 2.核心概念与联系
下面我们来看一下技术革命最重要的两个部分：建筑技术的崛起和城市的兴起。
## 2.1 建筑技术的崛起
19世纪末，意大利建筑学家古斯塔夫·马蒂尔在他的《历史与文化的终结》一书中，把建筑技术分成四个阶段：
- 石器时代（Ancient Stone Age）：指从亚细亚到欧洲，各种矿石石材、泥土等采掘工艺的发展；
- 青铜时代（Classical Rome Architectural Age）：指罗马共和国的建筑风格，主要以青铜建筑为主；
- 中期石英时代（Medieval Stone-Casting Age）：指中世纪中期，工匠们开始利用水泥和黑色石头开始构筑；
- 耀眼的黄金时代（Lustrous Golden Age of Era）：指宗教改革和西方文明的兴起，特别是基督教的启蒙运动。
17世纪初，英国著名建筑师约翰·洛克菲勒设计了现代建筑的基石—摩门托河（Mason Street）。后来，摩门托河被用于十字街区等地的公共建筑中。18世纪，洛克菲勒的设计被亚当·斯密采用作为圆形的建筑骨架，被称为“黄金三角形”（Golden Triangle）。斯密曾说："世界上没有任何东西能够比黄金三角形更加伟大。"
20世纪，各国建筑业界都争相追逐摩门教的黄金三角形。英国、法国、德国等国都纷纷布局自己的建筑风格。其中的代表性建筑有皇后镇、巴黎圣母院、诺威克公园、圣母百花大教堂等。近年来，美国、日本、韩国等国相继出现了以斯塔夫·班扬为代表的主流建筑风格。
## 2.2 城市的兴起
20世纪初，伦敦的居民们开始慢慢认识到，自己离商业中心很遥远。因此，伦敦的商业发达带来了新的收入来源，导致人们对于城市的定义也发生了变化。1910年，洪堡乔治三世宣布成立首都广州，并确定了城市的定义："城市是居民生活的集散中心，而不是工厂或者企业的地方。"
20世纪20年代，随着经济的发展和移民潮的到来，中国开始了长达半个多世纪的城市化进程。在上个世纪末至今，中国城市化率超过了65%。在目前的人均GDP排名中，北京、上海和广州位列前三。
2016年，中国地图显示，北京是全国城市群面积最大的城市，占全国总面积的1/3左右，同时也是全国最繁华的城市。2018年，我国新增高铁路、城市轨道交通、综合立体交通等基础设施建设取得了巨大的成果。但还有许多区域和城市尚未实现小城镇化。
2019年，我国城镇化率已达81.4%。虽然有些城市的缩小已经到了历史低点，但是大部分城市依然存在人口密度偏低、基础设施欠发达、管理混乱的问题。如何通过政策引导、技术创新、人才培养等手段解决这些难题，仍然是未来需要解决的重大课题。
