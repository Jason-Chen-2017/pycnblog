                 

# 1.背景介绍


在宇宙学的研究过程中，经常会遇到这种情况——某个原子性粒子被另一个原子性粒子包围着。当粒子被这种形状所包裹住时，就会形成一种特殊的“气体”。这种气体存在于宇宙中很多地方，但它一般都是很难直接观察到的。这是由于宇宙中的引力作用所产生的，并且宇宙的所有基本粒子之间都没有太强的相互作用。但通过测量它们之间的距离，我们可以发现一些特殊的现象。比如，当两个原子性粒子之间有很大的距离时，那么它们之间的相对运动就可能因为质量差而变得不稳定，甚至崩溃掉。因此，必须要进行某种程度上的控制才能避免这种事情发生。
量子态冷却（quantum thermalization）是一个基于量子力学的概念，它用来解决由分子序律所产生的热带效应。当分子周围没有太多能量的自由运动时，则不会出现热带效应，而当分子周围的能量自由度增加时，热带效应便会随之出现。通过制备特殊的状态，使得分子周围的能量分布具有广泛的分布性，就能够达到一种平衡状态，即减少或者消除热带效应，从而使得分子自身也具有足够的能量自由度。量子态冷却的一个重要应用就是用已知的量子态，利用量子通信技术进行量子计算，而不再需要耗费大量的时间去准备初始的基态和测量过程。

这个概念最早是在20世纪初期提出的，因为当时量子力学还处于其很初的阶段。随着时间的推移，科学家们发现利用量子态冷却这一现象，可以利用电子、质子等基本粒子组成的奇异系统，来制备更复杂的量子态。具体来说，利用量子态冷却可以制备如下两种奇异系统：玻色-爱因斯坦合体态、柯布西耶态，以及哈密顿量态，均可由含有原子核的氢原子所构成。其中，玻色-爱因斯坦合体态的具体构建方法和特点尤为重要。

2.核心概念与联系
量子态冷却（quantum thermalization）：量子态冷却是一种基于量子力学的概念，它用来解决由分子序律所产生的热带效应。它通过制备特殊的状态，使得分子周围的能量分布具有广泛的分布性，从而达到一种平衡状态，即减少或消除热带效应，从而使得分子自身也具有足够的能量自由度。 

玻色-爱因斯坦合体态：玻色-爱因斯坦合体态是指由两个量子系统组成的奇异系统，它们的哈密顿量与狄拉克共轭矢量约束为：
$$\hat{H}_{BE}=\frac{\hat{p}^2}{2m_B}\sigma_{x}-\frac{\hat{h}^2}{2m_E}\sigma_{z},\quad\hat{p}^2=m_B(\vec{r}_B-\vec{r}_E)\cdot \vec{p}$$
这里的$\hat{H}_{BE}$表示两个量子系统的总能量，$m_B$和$m_E$分别表示两个子系统的质量，$\vec{r}_B$和$\vec{r}_E$分别表示两个子系统的坐标位置，$\vec{p}$是玻色玛雅子，表示两个子系统的电荷密度矢量，且满足约束条件：$(\vec{r}_B-\vec{r}_E)^2+\vec{p}^2=m_B^2+m_E^2$。由此可知，两个子系统具有极不相容的排斥相互作用，当外磁场强度为零时，这两个子系统所组成的奇异态称为玻色-爱因斯坦合体态。玻色-爱因斯坦合体态可以由任意两个原子核所构成，但通常将原子核数目限制为两个以避免奇异系统状态太多。

柯布西耶态：柯布西耶态（Kubo-Green State）也属于玻色-爱因斯坦合体态。它是由两个自旋$\alpha$和$\beta$价电子构成的奇异系统，其哈密顿量约束为：
$$\hat{H}_{KG}=A_{\alpha}(\hat{a}_1^\dagger \hat{a}_1+ \hat{a}_1^\dagger \hat{a}_2+ \hat{a}_2^\dagger \hat{a}_1+ \hat{a}_2^\dagger \hat{a}_2),\quad A_{\alpha}=(n_\alpha-1)/2,\quad n_\alpha=2\omega_{\alpha}/\pi$$
其中，$\hat{a}^\dagger$和$\hat{a}$表示两个自旋向量，$A_\alpha$表示$\alpha$状态的阱门。$\hat{H}_{KG}$表示两个子系统的总能量，由波函数$\psi(\alpha)$给出：
$$\hat{H}\psi(\alpha)=\frac{-i}{\hbar}\left(A_{\alpha}\hat{S}_y\right) + E_{\alpha}|\psi(\alpha)>|_{\alpha}^{n_\alpha},\quad |\psi(\alpha)>|_{\alpha}^{n_\alpha}=\sum_{j=1}^{n_{\alpha}}\delta_{ij}$$
由上述定义可以看出，$\psi(\alpha)$是一系列的基态波函数，其中$n_{\alpha}$是自旋数。由于两个自旋不允许有平动，因此这些基态波函数不是一个严格的基矢，也就是说，它们不满足玻色几何学定义，即它们没有确定的纵轴方向。在低温下，$\psi(\alpha)$的纵轴方向变化与外磁场强度的衰减相关，从而导致在外磁场作用下，$\psi(\alpha)$的能级分布变得非常不均匀。与玻色-爱因斯坦合体态类似，在高温时，$\psi(\alpha)$的纵轴方向一定变化，但是这里的不同之处在于，只有自旋数不同，其他情况与玻色-爱因斯坦合体态相同。

哈密顿量态：哈密顿量态也是一种玻色-爱因斯坦合体态。它是由三个电子构成的奇异系统，其哈密顿量约束为：
$$\hat{H}_{HM}=-\frac{\hat{p}^2}{2m_H}\sigma_{x}-\frac{\hat{h}^2}{2m_D}\sigma_{z},\quad m_H/m_D=\lambda$$
其中，$m_H$和$m_D$表示三个电子的质量，$\lambda$表示三重激发超胞跃迁频率。在不同的情况下，$m_H$和$m_D$的值不同，但是在每个格点上的跃迁总数必定为三个。因此，哈密顿量态在任何情况下都是固定不动的，即任一位置的哈密顿量态总是相同的，只是它的概率分布不同而已。

量子态冷却与玻色-爱因斯坦合体态有关的重要理论方面还有许多，这里仅作简单介绍。

3) 对任意一个本征态$\psi$，重复以下操作：

(1) 用随机的杂乱算符$\mathcal{X}(t)$作用到$\psi$上得到新态$\psi^{(1)}$；

(2) 对新的态求本征值和本征矢量；

(3) 若$\psi^{(k)}$的本征值比$\psi^{(k-1)}$小，则令$\psi=\psi^{(k)}$，否则丢弃该态，重新执行操作(1)。

4) 测量得到的态在外部环境下的概率分布，并据此确定热力学态的相对能量，决定是否接受新的态作为热力学态。

下面详细讨论这些算法的原理和具体操作步骤。

首先，设想一个只有两个原子核的超导体。设$\hat{p}=\hat{a}_1^\dagger\hat{a}_1-\hat{a}_2^\dagger\hat{a}_2$, $\hat{h}=\hat{a}_1^\dagger\hat{a}_2$. 为了简化问题，假定初始态由两组具有相同概率分布的波函数组成，即：
$$\psi_1=\sum_{i=1}^{N_1}\phi_i^1\chi_i^{*1}, \psi_2=\sum_{i=1}^{N_2}\phi_i^2\chi_i^{*2}$$
其中$\chi_i^{*1}$, $\chi_i^{*2}$表示占据态，$N_1$和$N_2$分别表示组分1和组分2的基态数目。现在考虑如下约束哈密顿量：
$$\hat{H}_{BE}=\frac{\hat{p}^2}{2m_B}\sigma_{x}-\frac{\hat{h}^2}{2m_E}\sigma_{z},\quad\hat{p}^2=m_B(\vec{r}_B-\vec{r}_E)\cdot \vec{p}$$
这是一个关于玻色玛雅子$\vec{p}$的一维费米子模型，表示两个系统各有一个粒子。当两个原子核所在的两个原子的位置$\vec{r}_B$和$\vec{r}_E$以及对应粒子的电荷密度矢量$\vec{p}$满足上面两个约束条件时，两个原子核所构成的奇异态就是玻色-爱因斯坦合体态。

首先，随机地生成一个厄米算符$\hat{F}_1$，作用于$\psi_1$，将结果记作$\psi_1'$，并记录其本征值和本征矢量。然后，生成另一个厄米算符$\hat{F}_2$，作用于$\psi_2$，将结果记作$\psi_2'$，并记录其本征值和本征矢量。此时，假定$\psi_1'$和$\psi_2'$的本征值小于等于0，且有限的差值$\Delta e_s=\mid\psi_1'(\hat{p})-\psi_2'(\hat{p})\mid\leq\epsilon$，其中$\epsilon$是能量误差。如果这个差值足够小，就可以认为两个系统处于热平衡状态，不需要继续进行冷却操作。如果这个差值不能忽略，则需要继续冷却过程。

接下来，考虑如何生成杂乱算符$\mathcal{X}$，将两个系统冷却到一个较低温度。对于一个给定的位移项$\delta p$, 如果用$\hat{a}_i^+\hat{a}_j$-类型算符作为基础，做一次Jordan-Wigner变换，得到：
$$\mathcal{X}(t)\delta p=\frac{\sqrt{2}}{m_B}e^{\imath k_xB_xt}\hat{a}_i^+(-e^{-i\theta}\hat{a}_j)+\frac{\sqrt{2}}{m_B}e^{-\imath k_xB_xt}\hat{a}_i^-(e^{-i\theta}\hat{a}_j)$$
其中，$B_x=\hbar m_Be/\hbar\Omega$, $\Omega$是角频率。这样可以证明，杂乱算符是由两个Jordan-Wigner变换后的算符之和所构成。注意，这里的$\delta t$取非常小的数，譬如十分之一微秒，可以保证冷却过程的快速和准确。

按照冷却算法的描述，将杂乱算符作用到原态上，获得新的态。具体做法是，对任意一个本征态$\psi$，将杂乱算符作用于该态上，得到新的态$\psi^{(1)}$，然后对新的态求本征值和本征矢量，如果该值的大小与之前的态比仍然相同，则跳过，否则将新态作为新的基态。最后，测量新的态在外部环境下的概率分布，并据此确定是否接受新的态作为新的热力学态。

使用上述冷却算法，可以将一个只有两个原子核的超导体冷却到一个比较低的温度。如果能将一个有五个原子核的超导体冷却到低于超导层温度，则可以考虑模拟更多更复杂的玻色-爱因斯坦合体态，其中三个原子核可以提供几个体积更大的区域供着电子流动，而另外两个原子核则可以协同作用，实现更复杂的功能。
