                 

# 1.背景介绍


最近很火的互联网企业初创阶段(ICO)已经成为风口的主要驱动力。新型冠状病毒疫情带来的全球经济危机，让一些国家尝试从零开始建立起庞大的科技和产业集群，利用互联网推出了创业企业，这些互联网初创企业将带动着新一轮的产业革命，改变整个经济的格局。如今，越来越多的人开始关注并参与互联网初创企业的创业，不仅能够对社会产生积极影响，而且还可以获得巨大的财富效应。那么，当参与到一个有潜力的创业项目时，如何管理好自己的股权？本文将从“股权”三个层面进行阐述：股权激励、债务偿还、股权转让。通过一步步地演示如何参与互联网初创企业的股权计划，帮助创业者更好的管理自己的股权。

# 2.核心概念与联系
## 概念
股权激励: 是指企业在IPO（Initial Public Offering）上市后，股东可以根据发行股份数量支付相应的股息或利润作为激励。对于一家企业而言，它发放股票的过程就是股权激励的过程。如果一家企业的获利能力远高于其发行股份的价值，它可能就倾向于购买更多的股票。相反，如果一家企业发行的股票太少，它的股东可能会感到沮丧，因此发行股票的数量也需要进行调节。

## 相关概念
- 发行价格：代表一家企业为了创造价值的决心，而向社会公开招股的第一天所设定的定价水平。
- 回报率：代表企业获得利润的能力。回报率通常与社会认可度和企业表现有关，以及企业的成长性。
- IPO筹款：是一种融资方式，是指企业首次向证券交易所或者基金会发售股票并取得一定数量初始投资额的行为。

## 关系
- 股权激励：是指企业发行股票时，向股东分配的奖金或红利。
- 回报率：是企业的表现对股东来说能带来多少收益。企业要想获得足够高的回报率，需要具有较强的盈利能力，能够通过产品或服务赚取足够多的利润。
- 发行价格：是为了满足企业的发展需求而设定的标价。比如，一家企业在IPO上市前发行价格为100元，之后由于竞争性激烈，导致的股票供需紧张，该企业便会提高发行价格，以维持营收增长及发行股票数量的稳定。
- IPO筹款：企业发行股票后，可以通过IPO筹款来完成公司的运营，获取大量的金钱支持。企业一旦发行股票后，企业股东都有很多种选择，他们可以在IPO前或者发行后把股票出售给其他人。虽然IPO筹款给企业提供了巨额的资金，但是IPO筹款的确也有很多缺点。
