                 

# 1.背景介绍


光子晶体、光子学器件是近几年热门的话题，其中光子晶体包含的含义深深影响着整个物理学界。它由带极的金属表面和无限多的原子构成，在电子结构方面的作用是一种高效、低能耗、可靠、易于制备和量化的材料。而光子学是研究物质中光子行为的一门学科。它是微观世界研究宇宙光子运动的重要分支。本文将从两个层次介绍光子晶体与光子学器件。

第一，光子晶体的定义和分类。首先，“晶体”一词指的是由原子或分子组成的表面金属的结合体。晶体具有两个属性：一个是其具有相同的形状和大小；另一个是其具有良好的拓扑结构。由于晶体表面金属的材料性质及其巨大的形状尺寸，晶体一直被广泛应用于制造各种各样的材料和工艺中。例如，金属锌、铝、铅锂等是最早发现的晶体材料。

而光子晶体则是晶体的一种特殊类型。它由一族光子构成，可以被光子所占据。一般来说，光子晶体是由精心设计的原子团或者反应堆所产生的。比如，核聚变反应堆制备的锈蚀石是一种典型的光子晶体，核反应堆制备的玻璃石雕刻品也是一种光子晶体。

根据晶体表面金属的不同，光子晶体又可以进一步细分。这时，我们需要考虑晶体内含有的原子数量和种类。通常，光子晶体包括以下五种不同的结构。

①有限电子晶体：它由金属表面上存在着少量电子的原子团构成，这种晶体称为有限电子晶体（FET）。
②自由电子晶体：它由金属表面上存在着任意数量的电子的原子团构成，这种晶体称为自由电子晶体（DFT）。
③半导体晶体：它由金属表面上存在着少量半导体原子构成的晶体，这是一种特殊类型的自由电子晶体。
④超导体晶体：它由金属表面上存在着超导体原子构成的晶体。
⑤氘核反应堆晶体：它由微小的氘离子颗粒经过反应堆制备而成的晶体。


第二，光子学器件的定义和分类。“光子学器件”一般指的是一种能够感受、传输或存储光子的装置。这里的“光子”泛指电磁波。光子学器件通常分为以下几类。

①照相器件：由镜头、闪烁LED、摄像头、激光器等设备组成的光学传感器，用于制备光学图像。
②真空管：利用电场产生或收集一定程度的热流，实现信息传输、加速器控制、环境感知、辐射通透等功能。
③衍射器件：通过改变光子的方向，调节辐射、集束、多普勒效应等参数，用来实施多项工程领域的应用。
④激光器件：使用激发材料引起电子脉冲，逆转流方向，被检测物感应，获得信号数据。
⑤传感器件：利用各种物理学原理测定某些物体的某些属性，并对信号进行处理，如声音、图像、力场、电压等。
