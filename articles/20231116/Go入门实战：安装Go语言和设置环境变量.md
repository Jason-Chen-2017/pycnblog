# 2.核心概念与联系

## Golang是什么?

Go(官方文档)是一门由Google开发，以一种静态强类型、编译型编程语言为目标的开源项目，具有简单易用、高效安全、并发机制、垃圾回收、结构化语法等特性，被广泛应用于云计算领域、容器编排领域、DevOps工具链、机器学习和 blockchain 等领域。它支持自动内存管理，能够实现快速和简洁的代码，适用于编写服务端程序和网络应用程序。

## 为何要学习Golang?

Go作为目前最火爆的云计算编程语言，在国内外都有不少的企业使用Go进行研发，它对性能的优化能力、灵活的并发模式、良好的静态类型检查、丰富的标准库、完备的工具链、友好的社区氛围、丰富的生态等诸多方面都吸引着许多喜爱它的程序员。

因此，如果你想成为一名优秀的Go工程师，或是需要提升自己的技术水平，掌握Go语言将是一个不错的选择。

## Go的特点

- 简单：Go语言有简单易学的特点，学习曲线相对其他语言来说较低。
- 静态类型：Go语言是静态类型的语言，不需要在运行时刻指定变量的数据类型，通过编译器的类型推导功能，可以发现很多bug。
- 并发编程：Go语言支持基于CSP（communicating sequential processes）模型的并发编程，其提供了轻量级的线程和同步机制，使得编写异步并发程序更加简单方便。
- 安全：Go语言提供内置的防御性编程特性，例如指针类型和内存安全保证，帮助开发者避免很多潜在的漏洞。

## Go的应用场景

Go语言主要应用场景如下：

- 系统编程：如Docker和Kubernetes等容器平台项目中的基础组件，以及一些Linux系统上运行的后台服务
- 网络编程：主要用于服务器端和移动设备应用的开发
- 系统工具开发：如用于开发各种命令行工具的Go语言包管理器 govendor 和构建工具 Glide
- 云计算：基于Go开发的微服务框架包含了包括Kubernetes，Envoy，Hugo和 Prometheus 在内的众多项目
- 机器学习：包括 TensorFlow，CNTK，Theano，PyTorch 等框架都是用 Go 开发的
- Blockchain：Hyperledger Fabric，Quorum，EthsoShard等平台项目中的共识算法组件使用 Go 语言开发
- 游戏编程：如Go语言编写的大型游戏引擎，其功能强大且高效地满足了游戏开发者的需求。

## Go环境搭建

### 操作系统要求

由于Go语言是开源项目，其源代码可以在任何类Unix系统上编译和运行，但建议使用Linux或Mac OS进行开发。

### 下载Go语言


### 安装Go语言

下载完成后，将安装包上传到服务器，然后执行以下命令进行安装：

```
sudo tar -C /usr/local -xzf go1.13.linux-amd64.tar.gz
```

该命令将解压后的Go语言文件拷贝到/usr/local目录下。

### 设置环境变量

编辑~/.bashrc文件，在文件末尾加入以下内容：

```
export PATH=$PATH:/usr/local/go/bin
```

保存后，输入source ~/.bashrc使之立即生效。

### 检验安装结果

打开终端，输入go version命令查看是否安装成功：

```
go version
```

如果输出类似以下信息，则表示安装成功：

```
go version go1.13 linux/amd64
```
