                 

# 1.背景介绍


### 没有背景知识的人如何快速地学习新知识？——信息传递与沟通技巧
在IT行业，作为技术人员我们经常需要从事各种各样的工作，如设计开发，测试等等。同时，我们也要协助业务部门进行产品分析、需求调研、设计评审等工作。随着业务的不断发展和市场竞争的激烈，我们的能力必将得到提升。为了能够更好地运用自己所掌握的知识，降低自己的工作难度，提高工作效率，技术人才往往需要花费较多的时间去学习相关知识。因此，提升个人职业发展是一个技术人的基本要求。而新知识的学习则是通过阅读、观看、听课、实践和思考来实现的。那么，如何有效地传授或分享自己的知识，使得他人可以快速地掌握并理解呢？怎样才能促进团队之间信息的交流，并互相帮助，最大限度地提高工作效率呢？这些都是信息传递与沟通的关键。下面，我将简要地谈一下IT技术人员的信息传递与沟通的一些技巧。
## 2.核心概念与联系
### 2.1.信息传递（Information Transfer）
信息传递是指两个或多个个体之间传播消息、信息或者其它各种能量的过程。信息通常包括文字、图像、声音、视频、指令等各种形式。信息的传递有两种方式：直接传递和间接传递。
  - 直接传递：直接传递是指两者之间直接存在某种联系或共同意识，因而可以通过语言直接传递。例如，男性朋友间相互打招呼；亲密关系中的父母和子女之间的对话；老板和同事之间直接聊天等。直接传递的信息通常采用书面语、口头语、电话、传真、电子邮件等方式传播。
  - 间接传递：间接传递则是指传递过程中有第三方介入，使消息或信号不容易被直接接收。例如，当一封信从一位女士手中传到另一位女士手里时，她可能需要帮忙拿走；一家公司的股东可能通过推销获得更多收益。间接传递的信息主要靠传播媒介和媒介的形式来实现，例如电视、报纸、广播电台、网络。
### 2.2.信息沟通技巧
信息沟通的技巧主要分为以下六类：
  1. 意识：首先，我们必须意识到信息沟通的重要性。要想实现信息沟通，我们应该清楚地知道自己的目的、要求和利益。沟通的对象应该是明确的，而且坦诚相待。每个人都应该有一个明确的目标，不论什么时候、任何情况下，都要把握主动权。只有充分沟通，才有可能产生良好的关系。 
  2. 提问：提问本身就是一种信息沟通的工具。提问可以让信息的发送者和接收者明白对方需要什么信息，并引导他们展开讨论。提问的对象应当清晰易懂，且容易被他人理解。提问的方式应当简单直接，并把话题引向实际议题。
  3. 举例：举例可以帮助信息的发送者更准确地阐述自己的观点。举例也是一种很好的自然而然的沟通方式。举例可以让对方以更直观的方式了解到信息的含义。
  4. 陈述：陈述可以传达信息的重点。陈述的过程可以起到澄清和解释的作用。陈述的对象的具体是指信息的内容应该丰富多样，没有固定的模板。当我们陈述的时候，最好遵循一定的结构，并且注意使用词语的选择。
  5. 对话：对话则是信息沟通的基本方式。对话可以让双方互相肯定自己的立场、观点，并解决彼此之间的冲突。对话需要确定谁先发言，谁做记录，以及时间分配。
  6. 模仿：模仿也可以提升信息沟通的效果。模仿可以让信息的接收者觉得自己是在与对方在沟通，而不是在接受别人的教条。模仿还可以将信息传递给他人，让自己变成一个社交角色。
### 2.3.金字塔型思维模型
“金字塔”一词源于希腊神话，表示心灵高贵的石头金字塔，其上建有八个阶层，依次为：感官、认知、意志、情感、动机、目标、愿望和情绪。一般来说，信息处理的过程可以概括为三步：观察-获取-加工，即由感官层获取信息后，再转移至认知层进行分析处理，然后转移至意志层进行决策生成新的信息。由于信息的复杂性，必须依据人的特点和情况制定相应的沟通策略，以便更有效地传播和利用信息。
  - “金字塔”思维模型将知识组织成五个层次：认识、表达、应用、归纳和应用，包括认识层、表达层、应用层、归纳层、应用层。每一层都可以有不同层次的子层，不同子层又可以划分成更小的组成层。这种层级的分层及子分层的方法使得知识信息的组织十分清晰。通过反复检索、分析、组织和总结，我们就可以理顺自己的信息、将知识转化为行为、洞悉世界、解决问题。
  - 在学习新知识时，我们会按照层次顺序进行学习。在第一层——认识层，我们可以专注于信息本质的学习。在第二层——表达层，我们可以对信息进行语言的组织和呈现，进行短平快的演讲，形成演说。在第三层——应用层，我们可以应用所学的知识进行创造性的思考，建立新的观点和方法论，发现内在的奥秘。在第四层——归纳层，我们可以从已有的知识汇总出新理念，进行批判性的思考，理顺学到的知识结构，确立自己的学术观。最后，在第五层——应用层，我们可以将学到的知识运用于生活。这样，我们就能形成有效的知识网络，并真正运用它们进行思考和行动。
  - 通过层级式的学习，我们既可以领略到知识的本质，又可以运用学到的知识解决实际的问题。这种学习模式，可以有效地激发我们的创ativity、分析thinking ability、解决problem solving skills，最终实现知识的积累和转化。
