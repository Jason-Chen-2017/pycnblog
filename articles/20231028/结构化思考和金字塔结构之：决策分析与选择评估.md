# 1.背景介绍


## 一、决策层面的问题背景
### （1）决策层面的重要性——“短平快”的原则
在信息时代，人们生活工作中的种种问题都需要决策者去处理。对复杂问题进行正确的决策往往是一种挑战，同时也是每一个人的必备技能。但是，过分强调决策层面的重要性，忽略了其他领域的职责、关系及个人能力等因素，就会导致决策层面陷入“单打独斗”或“重复造轮子”的境地。因此，需要结合其他领域的知识、能力、资源、经验等综合因素，综合考虑出一个合理的解决方案。而这种解决方案的实现，除了依赖于决策层面的能力外，还需要多方协同参与，包括技术人员、产品经理、市场营销人员、设计人员、商务人员等。所以，从全局上看，决策层面所扮演的角色是相当重要的。

### （2）决策层面的三种角色——信息主体、参与者、视角、评判标准
在决策层面，总会有各种不同的角色。一般情况下，决策层面的角色分为三个，分别为信息主体、参与者、视角。如图1所示，信息主体负责收集需求信息、汇总意见，提出建议；参与者负责组织信息、研究研究、准备材料；视角不同，可以是表面上看待问题的方式，也可以是在整体构架下观察世界的方式。最后，评判标准可以采用客观的、比如效益或投资回报率、主观的、比如认为哪个方案比较好、哪种方案最具吸引力等。一般情况下，视角越广，参与者就越多，信息主体就越能提供更加全面的信息，从而更准确地做出决定。

### （3）决策层面的任务——把握大的方向、掌控细节、沟通问题
目前，决策层面的任务主要集中在以下几个方面：

① 把握大的方向——决定如何把握未来的发展方向，包括制定政策、制定战略、选择新产品或服务、调整业务模式等。

② 掌控细节——为明确的目标或方案设定关键路径，明确执行顺序，并制定相应的行动计划。

③ 沟通问题——和各方面相关的人士或机构沟通，解决疑难杂症，以及提出适宜的建议。

由此可见，由于决策层面的职责繁多，需要相应的决策方法和技巧，才能有效地运用自己的智慧和能力来完成这些任务。这也正是为什么很多企业都会招聘专门的决策咨询师来帮助他们克服“决策的艺术”这一难题。

## 二、决策层面常见问题的分析
### （1）“有些事情我不知道该怎么做”的问题
“有些事情我不知道该怎么做”，这是每个决策层面都存在的问题。这可能源自以下几点原因：

1.	缺乏专业知识——不懂得根据业务需求和现状进行合理的决策。

2.	缺乏洞察力——只看到局部，忽略全局，缺少整体的视野。

3.	缺乏理解力——不能准确识别问题、原因和结果。

4.	缺乏资源——没有足够的执行资源，无法承担重任。

### （2）“一味依赖机器，无法形成长久稳定的团队精神”的问题
“一味依赖机器，无法形成长久稳定的团队精神”，这是另一个决策层面常见的问题。机器的作用是快速决策，但它并不能保证决策的可靠性、准确性和最终结果的一致性。事实上，不同场景下的决策可能会产生截然不同的结果。因此，为了能够持续不断地改进，我们需要培养员工的职业操守和品格，使其能够真正具备独立判断、学习、创新的能力，构建真正的长久稳定的团队精神。
