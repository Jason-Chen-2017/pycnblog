# 1.背景介绍


近年来，随着区块链技术的蓬勃发展，电商平台也逐渐探索并尝试利用区块链的应用。本文将从以下三个方面来对比介绍目前最热门的区块链应用——区块链支付和数字货币钱包，和传统支付方式之间的不同之处。
首先，区块链支付：其主要特点是用户无需进行真实身份认证，由支付平台代劳完成交易，即“支付”在整个流程中都被“托管”或“受托”。由支付平台发起，然后平台通过自己的私钥控制区块链账户实现资金的转移，确保用户的个人信息在交易过程中不泄露。区块链支付可以帮助交易双方解决跨境支付问题，并降低支付成本。目前市场上有众多的独立第三方支付公司正在研发基于区块链的支付服务，如微信支付、支付宝等，这也是当前区块链支付市场最大的赛道。
第二，数字货币钱包：顾名思义，数字货币钱包就是使用数字货币进行交易的钱包，比如BTC、ETH、XRP等。用户可以使用数字货币进行商品购买、提现、收款等一系列操作。数字货币钱包能够提供比传统支付更安全的交易环境，且支持多种数字货币的充值和提现。除此之外，数字货币钱包还具有提升用户交易效率、提高个人财富管理能力等优势。数字货币钱包的价值来源于其独有的去中心化特性，能够将所有用户的数字货币数据进行存储，并为用户提供便捷的跨境支付服务。
第三，两者的比较：由于区块链支付和数字货币钱包之间存在一些关键差异，这里我们将分别从以下两个方面进行比较分析。
区块链支付和数字货币钱包的用户体验：区块链支付属于弱监管模式，用户无需进行真实身份认证，但是用户可能会遇到信用卡欺诈、支付卡盗刷等风险。而数字货币钱包则有较强的安全性，用户可以在充值和提现时选择不同的数字货币进行交易，避免了一些意想不到的问题发生。同时，数字货币钱包支持多币种，使得用户的交易方式更加灵活，满足不同场景下的需求。
区块链支付和数字货币钱包的可扩展性：区块链支付能够将交易网络与生产、运输、物流等业务网络分离，让整个经济体系能够快速响应变化。数字货币钱包通常只负责支付环节，不参与生产、运输、物流等环节，对于电商平台而言，这可能带来一些影响。
区块链支付和数字货币钱包的成本开销：区块链支付的成本相对较高，由支付平台建立联盟链的成本、使用支付智能合约的成本、存储区块链账本的成本等，但这一切都是免费的。数字货币钱包的成本相对便宜，目前各家数字货币的价格都在下跌，用户可以在任意时间进行交易，基本不需要担心手续费。
综上所述，区块链支付和数字货币钱包都有其优缺点，没有一种支付方式能够完全取代另一种，这两种支付方式应结合起来才能获得更好的发展。
