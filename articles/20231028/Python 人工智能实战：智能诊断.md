# 2.核心概念与联系
## 1.1 概念
诊断模型：诊断模型是指根据已知的输入条件（如手术时间、病人的病情特征、实验室检查结果等），通过对数据的分析和模型建立的方法，推断出该输入数据属于哪个诊断类别或范围。通常情况下，诊断模型的目的是对数据的一种概括，从而减少真实数据的复杂性，帮助人们快速、高效地做出判断或判定。 

智能诊断：智能诊断，即通过计算机或者机器学习算法自动识别、诊断、分类、预测、预测、预测……各种各样的病症和疾病。这里面的关键就是“智”！换句话说，智能诊断模型不需要人类的参与，就可以自动检测出并诊断出某个疾病。它的主要目的是为了减轻疾病患者的痛苦，增加医院收入、降低患病率、节省成本、提高效率，以及提高治愈率。这样的目标往往要求更快的诊断速度、更高的准确率以及更好的诊断效果。同时，还有一些成本高昂的手术、药物和设备等治疗方法只能依靠“超级计算机”才能实现。

## 1.2 基本原理
SVM是一种支持向量机分类算法。它是一种二类分类器，通过寻找数据的间隔边界使得数据点被分到不同的类中。SVM算法的主要思想是找到一个超平面，使得数据点在这个超平面上的投影尽可能的接近，间隔最大化。通俗地讲，就是希望得到一个超平面，使得这个超平面能够将数据集中的正负两类点完全分开，而且这个超平面能够有效地将空间划分为较小的几块区域。

## 1.3 核心概念
- 支持向量：就是分类平面法线上距离远离分割面的那些点，这些点称为支持向量。支持向量在求解支持向量机参数时起着至关重要的作用。一般情况下，SVM算法对支持向量的选择依赖于“软间隔”思想，即如果一个点的两个坐标轴上的投影距离是相同的，那么这个点不能作为支持向量；如果它们距离是不同的，则可以选择它作为支持向量。

- 拉格朗日乘子：这是拉格朗日对偶问题的核心。拉格朗日对偶问题是指把凸二次规划问题或线性规划问题转化为拉格朗日函数的极值问题。假设有一组变量x，通过下列约束条件来优化：

max z=f(x)+g(y)

s.t.	a(i,j)*x+b(i,j)*y<=c(i,j), i=1,2,...,m; j=1,2,...,n
		d(i)*x+e(i)*y=f(i), i=1,2,...,p

	 x>=0, y>=0

其中，z是要最小化的目标函数；x，y是需要优化的变量；a，b，c是等式约束条件，d，e，f是不等式约束条件。

定义拉格朗日函数l(x,y,α,β):

l(x,y,α,β)=f(x)+g(y)-∑_{i=1}^mp_i[1-ε_i(α^Tx+β^Ty)]

其中，ε_i=margin(xi)，表示对第i个训练样本的支持向量，α和β是拉格朗日乘子。

将上面的约束条件转换为拉格朗日函数之后，SVM算法的求解就变成了求解两个拉格朗日乘子α和β的过程。

## 1.4 相关术语及其之间的联系
SVM：Support Vector Machine简称SVM，是一类二类分类模型，它利用核技巧将非线性问题变成线性可分问题，从而可以在复杂的非线性环境中进行有效的分类和划分。

核函数：核函数是一种非线性变换，能够将原始数据映射到高维空间中，从而可以用线性分类器直接进行学习。核函数的引入使得SVM具有很强的鲁棒性和适应能力。

核技巧：核技巧是指利用核函数将低纬空间映射到高纬空间中，从而可以用线性分类器直接进行分类和划分。核函数的构造往往由算法设计者自己确定。

线性可分问题：指数据点可以用一条直线进行分割。换句话说，数据点在空间上可以被画成一个超平面，并且数据的类别分布于超平面正负两个侧。

优化目标函数：优化目标函数是指给定超平面后，希望找到一个合适的α和β，使得分割的正确率达到最优。

分离超平面：若训练数据满足线性可分问题，则将数据集分为两类，分别对应于超平面两侧的集合。分离超平面就是定义的这种超平面。
