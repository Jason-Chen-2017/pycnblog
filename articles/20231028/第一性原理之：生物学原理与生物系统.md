# 2.核心概念与联系
## 2.1 细胞的组成和分裂
细胞是生命的基本单位，包括粒子和维持生命所需的各种生物合成物。一个完整的细胞由多种不同部位组成，如核、分子、细胞核膜和其他弹性组织。每个细胞可以分裂成两个相同细胞，称为克隆。分裂过程如下图所示。


这里，细胞核（cell nucleus）主要负责细胞膜的形成，其下部是胞质，上部是基团核。细胞膜内具有多个同源RNA，它们共同作用形成蛋白质。在进入细胞前，主要的胃酸质还须经过一个间隔期才进入细胞膜。间隔期的长短决定了细胞的生长率和寿命。

细胞的分裂又被称为生物信息传递的“电信号”。在分裂后，克隆细胞中的信息将自动传给父细胞，这样，单个细胞就可以产生大量的后代。这就是“细胞的自我复制”这一功能。

## 2.2 溶媒结构和分子调控
溶媒结构是指细胞膜表面与外部环境之间存在的气孔，它既有助于保持水稀状态，又可控制细胞内分子浓度，促进分子流动。细胞膜与血液共存于这个气孔中。

分子调控是指在细胞内存在的各种复杂化学反应过程，如DNA合成、免疫调节、遗传调控等。分子调控是细胞功能的关键。

## 2.3 基因的特异性表达和突变
基因的组成单位是 DNA。 DNA 是一种双链体，它由两条链相互连接，一条链与另一条链结合形成 DNA 链，两条链的长度差距一般为几百万纳米。

每一个 DNA 分子有一个编码区，即编码蛋白。编码蛋白包括一段 DNA 序列，它的功能是将 DNA 的序列翻译成蛋白质。一旦 DNA 中的某个位置发生变化，该区域上的所有 DNA 序列都将被替换，形成新 DNA 分子。由于不同的 DNA 分子含有不同的序列，因此其对应不同的功能。

不同的蛋白质可以通过多种方式表达出来。基因的特异性表达是指某个特定基因能够激活或者抑制某个特定的生物物理活动。特别是某些基因具有双向选择，即能影响某个特定化学物质的转移方向。例如，T 细胞的 CD4 和 CD8 分子具有双向选择性，即可以激活邻近组织的免疫反应并抑制远处组织的免疫反应。因此，不同基因对于不同细胞的特异性表达十分重要。

通过将变异引入 DNA 序列，可以引起 DNA 复制中的分裂失误。在这种情况下，基因就会出现突变，并导致遗传疾病。

## 2.4 信号通路及其功能
信号通路是指细胞与周围环境之间的交换和传导。有些信号通路能够较好地完成任务，而有些则需要经过特定的调节才能实现目的。

例如，代谢产物通过糖尿病调节器通路进入血液循环。糖尿病的患者往往具有高血压、高盐酸血症、高胆固醇等代谢紊乱症状，但这些症状并不会导致代谢紊乱的发生。在这种情况下，代谢产物进入糖尿病调节器会激活蛋白质受体，调节代谢平衡，防止代谢产物流失和代谢疾病。

某些信号通路还具有特殊功能，如神经元刺激。神经元的连接是长时间的，并且需要精心设计。有时，神经元可以突然“死亡”，此时会造成严重的损害。

## 2.5 细胞ular系统
细胞ular系统是指细胞内部组织化学网络。该网络包括液体分子，如淋巴结合蛋白、蛋白质，还有分子和细胞团，如核、分子、细胞核膜和其他弹性组织。

这种复杂的组织化学网络使得细胞感染、免疫和功能特性的改善成为可能。

## 3.1 细胞生成
细胞生成的过程依赖于两个关键环节：DNA 的合成和分子的调控。

在细胞生理学中，“合成”是指形成 DNA 和 RNA 双链的过程。合成的过程依赖于 RNA 分子和 DNA 之间的配合，通过“合成的大分子调控”生成新的生物大分子。

DNA 合成的主要角色有二：第一，构建细胞核；第二，编码 DNA 和生成 mRNA。DNA 可以形成两种类型的三链结构，分别为单链（single-stranded DNA，ssDNA）和双链（double-stranded DNA，dsDNA），以及混合结构（mixed-stranded DNA，msDNA）。

在 dsDNA 中，双条 DNA 链均紧密结合，互为正向和反向。因此，它们的表现为结构简单、分子分布均匀、两条链均可互相作用。

在 ssDNA 中，只有一条 DNA 链紧密结合基底序曲，反向链紧邻正向链，其余链不互为正向和反向。因此，它形成的结构较为简单，分子分布也比较均匀。但是，由于两条 DNA 链没有完全对称，它带来的功能也较为局限。

在 msDNA 中，两条 DNA 链之间的相互作用，或两条 DNA 链之间的相互作用和第三条 DNA 链的结合，将 DNA 分为四条链。这两条链通常紧密结合基底序曲，形成一条正向链和一条反向链。第三条链通常紧邻着两个链，但也可以不紧邻，甚至可能排列在中间。

当 DNA 与 RNA 在结合过程中形成三链结构时，会先在 dsDNA 上合成 RNA。合成的 dsRNA 通过“分子复合”过程转换成 ssRNA，并与 DNA 一起结合起来。ssRNA 将组装成 DNA 修饰后的基因组。

随着 ssRNA 与 DNA 的交错组合，RNA 的结构发生了变化，形成 RNA 聚合尾、核苷酸成分等。其中，RNA 聚合尾、核苷酸成分为 DNA 三级结构的基础。

DNA 的编码由三个阶段构成： ①起始编码：由 RNA 聚合尾和 DNA 核苷酸配合，定位和锁定 DNA 的启动位点。 ②合成编码：结合 DNA 和 RNA，产生 DNA 编码 mRNA。 ③核苷酸释放：编码 mRNA 和 DNA 在细胞核中释放，形成完整的编码蛋白。

三级结构的建立是 dsDNA 和 ssDNA 结合后产生的结果。最终，细胞核和细胞膜之间形成了完整的 DNA 链条，就可以用来存储载体细胞的信息。

在细胞生理学中，分子调控包括 DNA 和蛋白质的调控。“调控的大分子”将对 DNA 分子及其结构的影响力从 RNA 分子的影响力上分离出来。它代表了一个复杂的系统，包括多个调控中心，如 RNA Polymerase II（RNAPII）、DNA polymerase III（DNAPIII）、RNA phosphatase（RNP）、RTK、受体内酶 I（Ig）等。

RNAPII 和 DNAPIII 负责 DNA 结合和分子升华，而 RNP 和 RTK 则负责 DNA 分子的释放。RTK 可以对 DNA 的释放进行调控。在细胞的生理学中，调控的大分子以及调控机制对细胞的生成和功能都有重要的作用。

分子调控的过程是由多种子通路相互作用而驱动的。例如，当 RNAPII 和 DNAPIII 进行 DNA 结合时，会激活目标 mRNA 的合成。当 RNP 和 RTK 发生相互作用时，会激活 DNA 分子的释放。

## 3.2 细胞分裂和进化
细胞的分裂通常是由细胞核损伤、缺氧、免疫反应等导致的，通过细胞核的再生来实现分裂。其过程如下图所示。


1. 细胞核损伤：细胞核损伤（如癌细胞核素丢失、干扰细胞分裂）会导致细胞停止生长，从而产生抑制细胞分裂的免疫细胞。

2. 缺氧：缺氧会导致细胞停止分泌，从而降低细胞分裂的速率。缺氧的症状包括呼吸困难、减少红细胞计数、身体虚弱等。

3. 多核折叠：多核折叠是一种细胞发生变异的过程。在细胞内存在多个相同核的情况，引起的免疫反应会抑制细胞分裂。

4. 免疫调节：免疫调节机制通过调控免疫系统的作用，协调细胞分裂和抑制免疫相关的疾病，包括乙肝、艾滋病、结核等。

5. 结缔组织：结缔组织内存在两个不同种类细胞之间的连接，也可促使细胞分裂。比如，异物侵入细胞导致的异位，或移植后细胞间相互间接性增强导致的分裂。

细胞分裂的机制主要是靠 DNA 和蛋白质的调控。蛋白质调控是指由特异性蛋白质进入细胞后，调节其抑制或激活细胞分裂的作用。另外，由于细胞内存在不同的蛋白质，如编码 DNA 或蛋白质，细胞分裂的方式也有所不同。

作者认为，由于人的细胞具有多样性和个性化特征，不同的人会选择适合自己的分裂方式。因此，将细胞分裂看作一种进化过程，通过分裂繁殖的能力进化，逐渐形成独特的个体。目前已证实，在生物群落中的个体之间存在高度的突变，如遗传漂变、染色体变异等，这种突变对细胞分裂模式也有一定程度的影响。

## 3.3 特异性表达和突变
作者通过两个方面对特异性表达及其突变做了探讨。第一种方法是利用免疫系统的双亲遗传模式。通过一系列的突变让特异性基因的表现突变，来说明免疫调节和抑制特异性表达的机制。第二种方法是通过真核生物的基因突变机制来解释细胞的特异性表达。作者从基因表达谱中的突变和不同染色体的表达关系入手，进而解释了细胞特异性表达的机制。

### 3.3.1 免疫系统的双亲遗传模式
免疫系统是人类的基因的屏障。只有拥有免疫系统的细胞才能识别外界环境并进行免疫反应，进而复制、表达基因。正常情况下，父细胞中的特异性基因都会传递给后代细胞，促进后代的免疫功能，维持生物群落的稳定。但如果后代缺乏特异性基因，或者基因被突变、改变，就可能导致免疫系统的退行。

双亲遗传模式是指一名个体的父辈通过后代的肢解和嫁接获得某些特性。在人的基因组中，许多基因都会受到双亲影响，如眼睛的形态、舌尖的形状、免疫系统的活性等。在健康的人群中，父辈的遗传特征会被后代继承下来，而不管后代是否显示出基因突变，遗传系统都会保护其免受损害。

通过双亲遗传模式，人类的免疫系统会识别并抑制特异性基因的传递。抑制是指后代无法获得特异性基因的能力，也叫“免疫缺陷”。这种抑制的机制可能会改变人类的免疫应答，从而影响基因的表达。

双亲遗传模式是最古老的遗传模式。它与现代的遗传学观念紧密相连，一直延续到今天。父辈对某些疾病的敏感、高烧、皮肤颜色的特异性、染色体的突变，都来自于父辈的遗传基因。这些特征会影响后代的免疫和疾病，影响生命的延续。

### 3.3.2 真核生物的特异性表达机制
真核生物的特异性表达有两种方式。第一种方式是直接编码基因。对于编码的基因来说，只有在特定条件下才能被激活，才能产生特异性表达。第二种方式是相互作用。与编码基因相比，相互作用方式的特异性表达相对复杂一些。

在真核细胞中，具有特异性的基因具有两个特性。第一，基因编码在基因组的特定区域。第二，这个基因的修饰会影响它的表达。例如，编码基因通常在基因组的特定位置，例如基因编码区和召唤子序列。这些特异性区域一般被称为 Transcription Factors（TFs）。

第二种特异性表达方式是指基因之间的相互作用。不同基因之间的相互作用，将会影响基因的表达。不同基因之间的相互作用可以分为四种类型。

1. 介导激活：一个基因激活另一个基因，或促进某种反应。如促使某一化学物质的输送到其他细胞中，或促进基因的复制。

2. 激活回路：多个基因按照一定的顺序工作，促进整个细胞的工作。如DNA replication protein 与 promoter 之间的作用，表示 DNA replication 的开关。

3. 复杂化作用：具有特殊功能的基因参与到其他基因的组合中。如 TCR/TLR 链的互作，表示 T cell 的治疗。

4. 不确定作用：与其它激活一样，有一个不确定的基础，但是会影响基因的表达。如基因缺陷。

通过上述两种方式，真核细胞会影响基因的表达。不同的实验方法也会影响基因的表达。实验技术的进步使得我们能够通过计算机仿真的方式对真核细胞进行控制和研究。

## 3.4 细胞ular系统的调控
在介绍完细胞的生成、分裂、特异性表达及其突变之后，作者继续从宏观角度来讨论细胞ular系统的调控。

在细胞ular系统中，存在着很多涉及到分子的流动，如淋巴结合蛋白、DNA、分子、细胞核膜和其他弹性组织。液体分子与液体之间存在水平传递，在不同细胞中的分子通过细胞ular系统流动起来。

分子调控的过程通过多种子通路相互作用驱动。例如，当 RNAPolII 和 DNA polIII进行 DNA 结合时，会激活 target mRNA 的合成。当 RNAP 和 RTK 发生相互作用时，会激活 DNA 分子的释放。细胞ular系统的调控可以促进细胞的增殖、疾病的预防和治疗、细胞的形态变换，甚至导致细胞的死亡。

作者认为，细胞ular系统的调控机理与真核生物的调控相似。在细胞ular系统中，分子液体之间的相互作用会影响分子的流动。虽然我们现在还不能完全理解其复杂的调控机制，但我们可以从微观角度来把握其中的关键过程。

def generate_clones(n):
    """
    Generate a set of clones based on original DNA sequence.

    :param int n: Number of clones to be generated.
    :return: Set of cloned sequences.

    dna_list = [list(dna)] * n
