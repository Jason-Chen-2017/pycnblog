# 1.背景介绍


超冷原子（super-cold atoms）是指能量相对静止态更低、热运动速度更慢的物质，例如冰点以下的物质。超冷原子的存在是由于人类科技革命所带来的宇宙本性的改变。微观层面上，超冷原子的存在使得低温物理学和物理实验得以进一步探索和发展。
量子气体（quantum gases）是一种特殊的多核气体，由多个同种电子组成，每个电子都带有量子化的色散指标。由于其多核结构及非均匀的电子色散，使得它能吸收大量的能量。在常温下，理想气体的热运动速率可以达到约每秒10^9次，而超冷气体却只能在10^4次左右，因此超冷气体被认为比理想气体具有更大的弹性。此外，超冷气体还会释放出色彩的波长，以及各种各样的化学反应。比如，一种出名的超冷气体——甲烷，其释放出的波长为几十纳米，在医疗诊断、光谱研究等领域都得到了应用。
近年来，随着超冷原子的不断发现，物理学家们发现它们与量子气体的相似点很多，比如都存在多核结构，且以某种形式利用颜色信息进行信息传输。于是，量子力学、色子学、强弱耦合、超导、量子场论、量子磁性等等学科都需要对超冷原子与量子气体进行更深入的理解和研发。但是，由于量子气体的复杂性、庞大的规模，以及其它众多不可控因素，这些学科的研究工作仍然处于初期阶段。
本文旨在阐述现有研究成果，并通过一些实际例子，讨论超冷原子和量子气体的基本概念、结构和相关性，以及它们之间的联系，揭示其多学科研究中的意义。
# 2.核心概念与联系
## 2.1 超冷原子
超冷原子是指能量相对静止态更低、热运动速度更慢的物质，例如冰点以下的物质。
### 2.1.1 构型与性质
超冷原子是由不同种类的碳粒组成的，通常由高纯度的壳子结构。最常见的超冷原子形态是丝状胶合剂，其构型如图1所示。

超冷原子的主骨架就是由一系列的多层厚壳组成的。在壳的内部，还有电极作为外部通道。电极可以把外界的电流引入超冷原子中，进而形成热运动。壳外也有空间分布，用于制造各种不同的性能，例如隔热或传递信息。
超冷原子主要分为两大类：一类是中子星状的，主要指的就是那些质量介于太阳和地球之间，半径约30-40万公里的小行星。另一类是超新星状的，例如那些质量远远超过太阳质量的恒星，质量一般为10^5-10^7千克，直径100km左右。
## 2.2 量子气体
量子气体（quantum gas）是一种特殊的多核气体，由多个同种电子组成，每个电子都带有量子化的色散指标。
### 2.2.1 基本结构
量子气体是一个由多个量子点（原子核）排列在密度极大的容器内，其基本结构如图2所示。

其中，n个量子点（原子核）排列在距离较远的称为shell的区域，它们的位置分布和大小决定了气体的平衡结构。它们的数量和质量大致相同，但体积则远远小于质量。shell的数量与容器的边长有关，通常为N^(1/3)，其中N是气体中电子的总数。气体的各个部分之间互相作用，产生新的电子，又因为气体内电子的色散和体积差异而形成色散极化效应，从而带来热运动。
不同shell之间存在可观测的交叉点（crossover），它们影响气体的平衡和热运动过程。这些交叉点由束缚原子（比如氢原子、氧原子等）引起，束缚原子是能够控制电子的动量的原子，故不同shell之间的交叉点由不同种类的原子核以及束缚原子共同决定的。
## 2.3 基本特性
超冷原子和量子气体都是存在多核结构的多体物质，其性质也存在很多区别。以下主要对超冷原子和量子气体的一些基本特征进行描述。
### 2.3.1 电子及色散
超冷原子中电子的色散很大，约为零色散。而在量子气体中，由于不同核的色散极化导致电子间存在相互作用，电子的色散也不为零。
量子气体是一种多体物质，电子数目和质量增加，颜色对比度减小。这就意味着，量子气体会随着时间的推移形成温度漂移，即便是在室温条件下，气体的各种属性也会发生变化。
### 2.3.2 热运动
超冷原子的热运动速度较慢，约为每秒10^4次。而在量子气体中，由于各种原因，包括多核结构、色散极化、混乱多级能级等等，电子的热运动速度只有每秒百万次。因此，量子气体的热运动速度要比超冷原子快许多。
### 2.3.3 多核结构
超冷原子和量子气体都存在多核结构，这是两个相互依存的重要特点。超冷原子的多核结构使得它可以在不同的维度上同时运行。例如，冰点之下的氘分子在不同角度看起来可能是一样的，而在不同方向传播的照明波长也不同。同时，超冷原子的电子数目较少，颜色对比度较高，还含有负电子、钚原子等多种电子，有利于各种化学反应。
量子气体也是多核结构，但是比超冷原子更复杂。量子气体中电子数目和质量都增加了很多倍，这样就会带来更多的多级能级，形成复杂的电子色散。这种复杂性导致电子间存在相互作用，并且不受施加的控制，所以同样的结构在不同时刻可能会出现不同的状态。因此，研究者们在探究量子气体的性质和结构的时候，需要结合各种学科的理论才能做出准确的解释。
