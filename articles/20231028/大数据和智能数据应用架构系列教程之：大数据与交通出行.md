# 1.背景介绍


在当前的社会经济环境下，人们生活节奏加快，旅游消费能力增强，城市居民对出行方式更加重视，随着大数据的广泛应用、智能手机的普及、互联网的飞速发展，互联网经济模式正在逐渐转型为现代经济的必然趋势，在大数据时代，如何为出行服务提供有效的信息化支撑、大数据分析技术在交通场景中的应用、各种新兴互联网产品如智慧城市、智慧出行等如何提升交通信息服务水平，成为各路人才青睐的重要方向。本文将从交通出行场景出发，全面阐述大数据和智能数据应用在交通场景中的作用、架构模式以及相关技术栈。通过学习、实践和总结经验，使读者能够掌握大数据与交通出行的整体技能框架，并将其运用到实际工作中。
首先，交通出行领域涉及的人群广泛，主要包括路网规划、交通数据采集、交通态势感知、交通指数计算、路况管理、交通时刻表查询、轨道交通信息服务、道路标志识别、车辆营销、实时路况、路段疲劳预警、乘车安全驾驶、行程预测、智能交通诊断、智能交通导航、顺风车预约订座、动态路况展示等。其中，实时路况、路段疲劳预警、乘车安全驾驶等属于核心技术与应用，相对容易理解。而道路标志识别、轨道交通信息服务、车辆营销、智能交通诊断、智能交通导航、顺风车预约订座等属于新颖的技术或产品，需要进行较多研究。因此，我们选择“交通出行”这个主线作为本次教程的主题，围绕交通出行场景，探讨大数据与智能数据应用架构在其中的作用。
在中国目前的发展过程中，已经形成了一套完整的大数据管理体系，各类数据源广泛汇聚，形成了一个庞大的海量数据网络，其中包含许多关于交通的原始数据，这些数据以各种形式存在于网络中，由各个地区的人员、企业、组织、系统和设备产生、收集、存储、处理、分析、共享，不断积累、汇聚、完善，为政策制定、决策、监控、管理提供了有力的数据支持。
另外，随着智能手机的普及，智能移动终端的大量部署，使得人们生活节奏加快，不仅能够随时享受到便捷的交通工具，而且可以根据需求、场景进行交通信息的快速获取、查询和响应，使得人们可以直观地看到路况变化、个人位置、轨迹跟踪、事件预警等。
随着智能出行智能化的不断演进，各种新兴互联网产品如智慧城市、智慧出行、物联网、传感网、雷达网、GPS网等不断涌现，如何提升交通信息服务水平、做好轨道交通信息的共享、开放、有效利用成为越来越迫切的问题。
综上所述，可以发现，当前的人们生活节奏加快，交通信息服务需求增加，已成为事关人的生命健康的大事，通过有效地运用大数据、云计算、人工智能等技术手段，通过相关的行业应用、技术平台，为人们提供全面的、及时的、精准的交通信息服务，是构建一个智能化、大数据化的交通信息服务体系不可缺少的一环。
