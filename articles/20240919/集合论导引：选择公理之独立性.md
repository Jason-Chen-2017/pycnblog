1. 无条件性：对于任意的非空集合族，选择公理都存在一个满足条件的选择函数。
2. 独立性：在某些数学体系中，选择公理是可证的，而在另一些体系中，它可能是不可证的。

def selection_function(sets):
    """
    选择公理的实现
    :param sets: 集合族
    :return: 选择函数
    """
    return {i: next(iter(s)) for i, s in enumerate(sets) if s}
