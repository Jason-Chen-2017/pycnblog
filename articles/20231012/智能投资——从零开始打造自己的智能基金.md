# 1.背景介绍


## 概述
随着人工智能、机器学习等技术的不断革新，基于AI的智能投资已经逐渐走向成熟。国内外已有众多公司、团队开发出了智能投资系统或产品，而很多优秀的系统也已开始流行起来。那么，什么样的系统才算是“智能”呢？我认为，首先要达到“智能”的要求，系统需要具备以下几个方面：

1. 投资策略能力：能够根据市场环境及投资者的风险偏好、行为习惯、个性化需求等自动生成符合个人投资偏好的投资策略；
2. 数据分析能力：能够通过对市场数据的统计、分析、预测等手段，找出市场中潜在投资机会和趋势，并将这些信息反馈给投资者；
3. 模型构建能力：能够结合历史数据、投资者偏好、模型算法等因素，精心设计出能够预测出股票涨跌的回归模型；
4. 操作执行能力：能够有效地运用模型进行交易，最终实现“超级智能”的投资决策。
综上所述，可以说，具备以上四种能力的智能投资系统或产品就是“智能”的。由此可见，智能投资确实是一种具有前景的新兴领域，它正在引领着新的投资理念、新型模式的出现。但是，如何利用现有的技术资源开发出自己的智能投资产品或系统，依然是一个复杂且困难的任务。
## 智能投资产品分类
目前，智能投资产品按功能划分，主要包括两大类：量化择时类和系统策略类。
### （一）量化择时类
#### A、摩根标普道琼斯道琼斯指数智能投顾
该产品由摩根士丹利（JPMorgan Chase）推出，旨在帮助投资者识别并利用科技及创新的先进方法，规避风险，获得高收益。其核心算法为市场扫描（Market Scan），在摸底期间以高低价、成交量为基础，结合指数数据及传统选股规则，精确定位市场波动方向，制定出符合投资者预期的买入卖出的信号。如此一来，该产品可以自动处理、跟踪投资标的，在短时间内生成高效、精准的买卖点。另外，该产品还可以通过利用机器学习技术训练投资经验，改善日后的交易策略。
#### B、华盛顿奇点智能交易平台
该产品由华尔街知名投资者<NAME>牵头研发，基于机器学习算法和数据挖掘，通过大数据分析得出市场行情的走势，再结合投资人的判断力，做出投资决策。其核心算法为股价预测（Stock Price Prediction），通过分析历史数据、大盘指数和经济景气指数，得出未来股价走势，对行情变化做出及时的响应，为投资者提供实时的投资建议。同时，该产品还能根据投资者的持仓情况及建议，实时生成交易指令，引导投资者实现远见卓识。该产品是美国最早推出的智能交易平台之一，并被誉为美国第一例“智能投资产品”。
#### C、安邦集团智能投资交易平台
该产品由美国的安邦集团拥有，由资深机器学习专家李健（Lyle Liu）领衔研发，其核心算法为“大数据驱动分析”，能够对大数据进行分析，发现投资者偏好的买卖点，提供符合投资者意愿的交易策略。该产品目前正在海外推广应用，并在国内有一定知名度。
#### D、德勤智能投资管理公司智能投资服务
该产品由德勤（Deere）集团有限公司推出，其核心算法为“情绪分析”及“看懂电影”，基于投资者的分析及观影习惯，预测其投资偏好，并给予正确的投资建议。该产品目前已于2017年初完成在欧洲市场的推广试点，在当地区率先布局。
### （二）系统策略类
#### E、麦肯锡智能投资公司智能投资平台
该产品由麦肯锡集团（Mckesson Corp）推出，其核心算法为“透视技术”，通过对全球股市、经济景气及投资者的研究，挖掘投资者的投资习惯及需求，将他们画像转化为投资策略，让系统自动生成符合投资者喜好、条件的买卖点。该产品已于2017年9月进入中国市场推广试点。
#### F、赛富中国智能投资公司智能投资系统
该产品由赛富中国（Shanghai Junshi）集团推出，其核心算法为“指数期权交易”，通过利用机器学习及大数据分析技术，找出行业整体走势，预测行业指数的走势及趋势，再结合投资者的投资偏好，为他们提供相应的交易建议。该产品已于2018年9月份启动在香港的推广试点，开拓国内投资市场。
#### G、伊德利安云计算智能投资平台
该产品由伊德利安集团有限公司推出，其核心算法为“机器学习算法”，通过大数据分析及相关模型训练，识别投资者偏好的交易品种，并以此为基础，为投资者生成交易计划，确保投资收益。该产品目前已于2017年8月份启动在新加坡正式推广。
