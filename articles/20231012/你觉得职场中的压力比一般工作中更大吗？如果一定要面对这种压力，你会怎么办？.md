# 1.背景介绍


在许多企业里都存在着压力四处奔走的人才——工作压力、生活压力、关系压力、金钱压力。而各个领域的IT从业人员都面临着不同的职业压力，比如互联网开发工程师，测试工程师等。这些职业中，都有不少受过良好教育的资深技术人才，但对此类人员来说，职场压力并非易事，他们也常因各种因素而不得不承受压力。在面对职场压力时，很多技术人员或会出现以下情形：
- 追求完美：在职业生涯中，工程师们也都会面临着一段艰难的学习曲线，因此，追求完美可能成为他们选择职业道路的首选。然而，追求完美往往也是导致危害最大的因素之一。在自律、拒绝外界干扰的心态下，工程师们很容易放弃对自己要求的尊重，将心脏、身体和精神投入到工作上，甚至还会酗酒、吸毒。在追求完美的过程中，工程师们会无意识地忽略那些能够促进他们成长的东西，比如通过锻炼、交流、分享等方式建立个人联系，从而提高自我满足感。同时，在追求完美的过程中，工程师们也容易被周遭环境所影响，因为人际交往变得越来越重要。所以，即便是应聘面试的时候，技术人员还是会在自我价值观方面留下一些余地。
- 急功近利：对于个人能力方面的要求，工程师们在考虑自己的优势时，可能会盲目崇拜他人的能力，认为别人比自己强就行了，从而增加对自己能力的依赖。但实际情况却恰恰相反。对个人能力的关注和评判常常需要以数据为基础，对相关技能进行量化分析，并且在职业生涯中持续不断的提升。只有有针对性地进行教育、培训、实践、学习，才能让工程师真正拥有所需的知识和能力。
- 漠视同龄人：对于年轻人来说，没有机会接触到多元化的世界，无法理解不同文化的差异，缺乏同龄人的支持。这些年轻人往往对金钱、权力、地位有着过高期望，觉得“做正确的事”就是王道，然后一味盲目崇拜，拒绝改变。但是，职场中那些拖后腿的人却都活得更开心，他们把最基本的礼仪习惯培养出来，接受别人的评价，并且乐于分享自己的经验。
- 错失良机：许多工程师对自己所属团队缺乏足够信任，喜欢欺骗，甚至故意延误自己的职业发展。他们只关心自己做得好的地方，而忽视团队在整个项目中的作用，甚至连自己是否适合当这个角色都不知道。为了团队的发展，他们不顾现有的规定，随意改动项目计划，使自己在团队中处于不稳定的位置，从而错失良机。
总结一下，在职场中，除了在自律、自我管理等方面刻苦钻研之外，技术人员更需要的是保持独立思维、竞争意识、坚忍不拔的精神状态，勤奋努力的完成任务，并且得到全方位的支持。只有这样才能真正做到心中有数，脚踏实地，并最终胜出。
