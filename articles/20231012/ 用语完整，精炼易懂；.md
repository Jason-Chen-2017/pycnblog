# 1.背景介绍


## 一、名词解释
### （1）TCP/IP协议族：
- TCP(Transmission Control Protocol):传输控制协议，它是一种面向连接的、可靠的、基于字节流的传输层通信协议，由IETF的RFC793定义。
- IP(Internet Protocol)：互联网协议，它是网络层协议之一，主要用于数据包从源到目的地的路由选择及地址寻址。
- ICMP(Internet Control Message Protocol)：互联网控制消息协议，它是TCP/IP协议族中的一个子协议，用于处理因特网控制消息，如网际路由探测报文等。
### （2）网络模型：
- OSI模型（Open Systems Interconnection Model）：开放系统互连模型，它将计算机网络划分成七个不同层次，物理层、数据链路层、网络层、传输层、会话层、表示层和应用层。
- TCP/IP模型（Transmission Control Protocol/Internet Protocol Model）：传输控制协议/互联网协议模型，它将计算机网络划分为两层——网络层和传输层，并规定了通过该模型进行通信的基本规范和流程。
### （3）IPv4地址范围：
1. 网络ID：标识主机所属网络，0~127为特殊保留地址。
2. 主机ID：标识网络中各主机唯一的编号，0~255为特殊保留地址。
3. 网络号：一个整数，网络号取值范围1~32bit，在每个子网中唯一标识本网络，网络号一般由网络管理机构分配。
4. 子网掩码：用于确定某个主机的所在子网，也称作子网通配符。子网掩码是一个32位二进制数，由“.”分隔，左边起首八位全部置为1，后面的是二进制0。例如，192.168.100.255的子网掩码为255.255.255.0。
5. 默认网关：当一个主机想要访问外部网络时，需要经过默认网关才能到达该网络。默认网关是指直接连接到互联网上另外一个网络上的路由器或计算机。
### （4）物理接口类型：
- 以太网接口（Ethernet Interface）：最常用的以太网接口，由IEEE 802.3标准定义。
- 无线局域网接口（Wireless Local Area Network Interface）：支持IEEE 802.11协议标准的无线局域网接口。
- Token Ring接口（Token Ring Interface）：用以支持点对点连接的令牌环网络接口。
- FDDI接口（Fiber Distributed Data Interface）：为光纤分布式数据接口。
- ATM接口（Asynchronous Transfer Mode Interface）：异步传输模式接口，采用ISO 8802.3标准。
- PPP接口（Point-to-Point Protocol Interface）：点对点协议接口，一种可伸缩的数字串行接口。
### （5）LAN/WAN：
- LAN（Local Area Network）：本地局域网，通常由小型组织内部使用的计算机网络。
- WAN（Wide Area Network）：广域网，通常是指由大型组织之间共享的计算机网络。
### （6）集线器、交换机、路由器：
- 集线器（Hub）：集中所有计算机信号线路的设备，每台计算机都通过集线器连接到此。不安全、占用昂贵且耗电量大。
- 交换机（Switch）：根据数据包的目标MAC地址转发数据包至目的计算机的设备。连接同一个交换机的所有计算机都能收到发送给该交换机的数据包。安全、灵活但耗电量大。
- 路由器（Router）：用于连接两个或多个LAN之间的设备，通过静态路由或动态路由进行数据的传输。性能好、价格便宜、配置灵活。
