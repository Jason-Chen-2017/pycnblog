# 2.核心概念与联系
## 2.1 概念简介
量子计算机可以看作一个由量子比特(qubit)构成的计算系统。一般情况下，一个量子计算机由多个量子比特组成，每个量子比特都有自己的一套状态。而这些状态之间的关联关系形成了量子纠缠，也就是量子电路。而量子计算就依赖于此类量子电路的有效演变。

量子计算机的基本运作单位就是量子比特，每个量子比特都有两个原子性状态，分别为|0>和|1>，其状态转移规则如下图所示。


这张图中，|Ψ+>表示在态矢量上绘制出的两个向量之和，也就是 |0>|1> + |1>|0> = |0> (|0> 和 |1> 的任意组合)。此处的竖线|Ψ+>表示将两个量子比特合成为一个复合量子比特，即作为量子比特中的一部分。

其中，|+> 表示将两个量子比特构成的态矢量投影到另一个复平面（纯态矢量），这种态矢量称为贝叶斯态(Bell state)，一般形式如 Bell 基或 GHZ 态等。

|0>|1>|0> 或 |1>|0>|1> 是纠缠态。纠缠态中的两个量子比特之间存在某种限制条件，使得它们不能自由地进行相互作用。只有当这两个量子比特满足这种限制条件时，才能利用量子纠缠的效应生成计算结果。

### 2.2.2 多项式时间与不可分辨性
如果一道题目是多项式时间可求解的问题，意味着它可以在多项式的时间内求出答案。如果一道题目不是多项式时间可求解的，那么它的求解过程通常是“不可分辨”的，也就是说不存在一个有效的算法可以确定它是否正确地解决给定问题。

多项式时间就是指对于规模为 n 的输入，算法的运行时间至多为 O(n^k)，其中 k 是一个任意正整数。因此，多项式时间算法是一个最优的算法，在最坏的情况下，它只能达到固定的渐进时间界。

不可分辨性通常用来描述量子计算中的安全问题。在量子计算中，不可分辨性可以认为是指某个算法不能区分不同的输入在计算上是否产生相同的输出。如果一个算法是不可分辨的，那么就可以保证该算法是安全的，也就是说它不会产生错误的输出。在量子计算中，由于要捕捉和测量多于一个量子位，因此也带来了新的安全威胁——量子态的扭曲。尽管量子计算中的某些操作是可以精确确定的，但还有其他操作则需要使用不可分辨的算法来模拟。
