# 1.背景介绍


近年来，农业生产持续快速增长，但是农民的投资增速却显著放缓。随着人口向城市转移的进程加快，越来越多的人选择从事农业或工业等领域的创业项目，而农村地区的农业投资却很难满足需求。而世界各国都在推动农业的发展，同时也有利于我国的经济发展。如今，全球农业产出创历史新高，但农民仍然是一个饱受低收入、负债累累、贫困潦倒、疾病缠身的群体。根据ICIQ最新发布的数据显示，2019年，中国农业总产值超过了美国，成为世界第一大农产品供应国。这种状况令人震惊，尤其是在农民不得不面对高房价、高房租的情况下。因此，如何提升农业投资增速、扩大农民收入，并通过科技投入解决城乡居民住房问题是当前面临的课题。
# 2.核心概念与联系
## 2.1 什么是“农业”
农业（agriculture）是指利用自然资源（主要是天然气、水土流量、植物等）和人力实现耕作，主要包括种植庄稼、养殖畜牧业、农副产品及其他农产品销售、种植保护、旅游休闲、农业技术服务、动植物保健、林业和水利工程等。

## 2.2 为什么要“农业”
农业发展是人类文明发展不可缺少的一项基础工程，它能培育出丰富的生物、提供丰厚的收入、满足人们的日常生活需要。由于水土保持率高、气候适宜、地质结构复杂，加上人口过多、食物短缺，因而产生了近几百年来的严重粮食危机、水资源枯竭、人口老龄化、健康发展不平衡、资源浪费等问题，为此，农业也成为许多国家的难点之一。

## 2.3 农业的作用
农业作为人类社会赖以存活的基本需要，对环境、对人类的健康有着深远的影响。农业生产中普遍存在着“三农问题”，即“三农”问题，即“粮食问题、肉类问题、蛋品问题”。经过十几亿人口的农业革命，现代农业发展取得了巨大的成就，但农业对环境的污染以及人类健康的损害依然不能完全解决，造成了广泛的争论。由此可知，只有进一步加强农业的发展，才能促进中国的经济发展。

## 2.4 农业投资是如何增速放缓的
目前，中国农业总产值连续九年位列世界前列，产出的粮食占全球七分之一。农业投资也受到来自世界各国的压力。农业引领的大国关系已经形成，不仅意味着农业对中国经济的重要性日渐凸显，还将有助于扩大中国经济的规模。在世界范围内，尽管中国农业总产值创下新纪录并远超美国，但还有许多国家正在考虑减少或放缓农业投资。根据世行的数据显示，2017年至2019年，中国农业投资下降了13%-20%。截至2019年7月末，中国农业国际收支比排名第五，农业产出拉动国内生产总值的比例位列第二，农业对GDP贡献率排名第一，而且，GDP增长率方面的表现则非常好，农业占据了农业产出对GDP贡献的绝大部分。据此看，虽然中国农业总产值已超过美国，但其投资增速明显放缓，并且受到来自世界各国的压力。
