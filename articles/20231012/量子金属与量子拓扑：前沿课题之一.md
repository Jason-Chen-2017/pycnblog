# 1.背景介绍


近年来，随着宇宙学、原子物理学和信息科学等多个学科领域的深入发展，对量子纠缠、量子磁性、量子电荷的探索取得了重大进展。在各种热门领域中，例如量子通信、量子计算、量子传感器、量子光学、量子计算力学、量子生物工程等，都给人们带来了不少有意思的新发现。由于这些前沿研究领域的复杂性及难度，也需要专业人士的强烈兴趣和投入。随着量子技术在各个领域的迅速普及，相信越来越多的学者将从事此类课题研究。因此，本文试图通过《量子金属与量子拓扑》系列课程的形式，向广大的研究人员阐述量子金属和量子拓扑等量子科技的前沿研究方向。

# 2.核心概念与联系
## 2.1 什么是量子金属？
量子金属(quantum metal)指的是一种能产生并控制微观粒子行为的金属元素，其基本构成单位为原子，这些原子具有排斥量子效应，可以抵消相互作用中的单粒子效应。从宏观上看，量子金属表现出一种特殊的原子-电子混合体结构，形成两个半导体层级，一个为无填充的基态，另一个为带有费米子边界的副态。从微观上看，量子金属中具有超导性质的原子可以利用光学反射、透镜效应等现象实现自旋转移移的过程。


如上图所示，一个典型的量子金属由费米子和非费米子构成，两个原子分子的双极振动称为Fock空间，也称为束缚费米面(bound Fermi surface)。费米子具有负振动频率，处于费米面的上方；而非费米子具有正振动频率，处于费米面的下方。费米子带隙是实验上最方便的气体分子，而非费米子则大多数时候不是独立存在的。因此，量子金属还会出现奇异的配置，如带钛离子、全铝、锌基等。

量子金属还有一个重要特性是平衡反铜(polycrystalline)，即一个量子子晶中含有两种不同种类的费米子（如SnS和PbSe），但是两种费米子所占比例恰好达到相等的程度，这种类型的量子材料可以用于制备高功率设备，如量子通信网络。

## 2.2 什么是量子拓扑？
量子拓扑(quantum topological)是一个新兴研究领域，它研究如何构建量子系统以及如何把它们拼接起来，从而实现更高的量子纠缠程度或可控性。量子拓扑关注如何利用量子拓扑学构建更复杂的量子网络，更好的理解量子多体系统的性质和运动规律。

量子拓扑学的基本任务是研究群论中的拓扑群，即由一个基底群和一族胞元生成的群。通常来说，这个群具有非拓扑性，即具有指标数与自由度数不同的态空间。量子拓扑学的一个主要工作就是开发基于量子拓扑理论的方法来研究具有拓扑性质的群，从而解开它们的内在张力和保真度问题。量子拓扑学可以应用于许多领域，包括量子信息、量子计算、量子通信、量子纠缠、量子计算机、量子生命科学等。
