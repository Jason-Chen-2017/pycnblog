# 2.核心概念与联系
## 2.1 Introduction 

Artificial Intelligence (AI) refers to the design and development of intelligent computer programs that exhibit human-like intelligence. The term "intelligent" refers to the ability to reason, plan, learn, and adapt through trial and error. As an instance of machine intelligence, artificial intelligence can be further classified into three types: symbolic, connectionist, and hybrid. Symbolic AI involves using symbols (such as letters or concepts) to represent data, while connectionist AI uses neurons and connections between them to process complex patterns within data sets. Hybrid AI combines both symbolic and connectionist paradigms to achieve optimal performance. 

In this section, we will briefly introduce some key concepts associated with artificial intelligence in general and explain how they relate to Indian context.


### Machine Learning 

Machine learning is a subset of artificial intelligence that enables machines to automatically learn and improve their performance without explicitly being programmed. Machine learning focuses on training machines to recognize patterns and make predictions based on examples provided during training. There are two types of machine learning techniques commonly used in AI: supervised and unsupervised learning. Supervised learning involves labelling data points with known outcomes, while unsupervised learning relies solely on clustering, pattern detection, or dimensionality reduction techniques to extract insights from raw data. Other forms of machine learning include reinforcement learning, which explores environments and selects actions that maximize rewards over time. 


### Natural Language Processing 

Natural language processing (NLP) is a subfield of AI that involves teaching computers to understand and manipulate human language to enable conversational interactions, text analysis, information retrieval, and sentiment analysis. NLP involves analyzing large amounts of unstructured text data, extracting meaningful features, and representing them in a structured format suitable for downstream applications. One common technique employed in NLP is named entity recognition (NER), which identifies entities in text documents such as people, locations, organizations, etc., and assigning appropriate tags or categories. Another popular task in NLP is sentiment analysis, which involves identifying positive, negative, or neutral opinions expressed in text. Common NLP libraries like NLTK and spaCy provide pre-built tools for performing NLP tasks quickly and easily. 



### Knowledge Representation and Reasoning 

Knowledge representation and reasoning (KR) is another important area of AI that involves developing computational methods for managing, organizing, and reasoning about large knowledgebases and semantic structures. KR has been applied in various domains including natural language understanding, social media analysis, medical diagnosis, robotics, and financial markets. One example application of KR technology is DeepDive, a tool developed by Stanford University for large-scale data analysis.



## 2.2 Reinforcement Learning  

Reinforcement learning is a type of machine learning approach that aims to train machines to choose actions that maximize long-term reward rather than simply solving specific tasks. RL algorithms work by interacting with an environment and receiving feedback indicating whether the chosen action was successful or not. They then update their model of the environment based on the result and repeat the process until a desired level of accuracy or efficiency is achieved. Reinforcement learning has many practical applications in game playing, robotics, autonomous vehicles, and healthcare. Popular reinforcement learning frameworks include OpenAI Gym, TensorFlow/TensorForce, and Keras RL.



## 2.3 Neural Networks   

Neural networks are a class of machine learning models inspired by the structure and function of the human brain. Each layer of a neural network consists of interconnected nodes called neurons that receive input signals, pass them through weights, and generate output signals. The activation of each node is determined by applying a non-linear transfer function such as sigmoid or tanh. Neural networks are typically trained via backpropagation, which calculates the gradient of the loss function with respect to the parameters of the network and updates them accordingly. Popular neural network libraries include TensorFlow, PyTorch, and scikit-learn.
