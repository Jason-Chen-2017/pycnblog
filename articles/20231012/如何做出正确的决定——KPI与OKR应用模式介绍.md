# 1.背景介绍


## 什么是OKR？
OKR（Objectives and Key Results，即目标和关键结果）是一个由Google提出的目标管理工具，是一种全新的工作方式。它旨在建立一个基于目标的工作流，帮助公司实现更高效、高质量的业务。其基本原则是将目标分解成可衡量的“关键结果”（Key Results），然后制定每个目标的时间表和里程碑，每完成一个里程碑就对该目标进行评估。通过实时跟踪并持续改善关键结果的达成，可以帮助团队在最短时间内精准地实现业务目标。从而帮助企业确保业务的持续增长和价值实现。
## OKR的定义
> OKR (Objectives and Key Results)，即目标和关键结果，是由Google于2010年推出的项目管理方法论，是一种全新的工作方式。通过将目标分解成可衡量的“关键结果”，并制定每个目标的时间表和里程碑，团队成员可以用清晰的计划和目标驱动团队一系列的工作。它的核心原则是“激励性”，“科学性”，“透明性”。主要作用有三方面：

1.激励性：激励大家按时、高质量、及时地完成任务，这既有利于提升员工的工作绩效，也有助于促进公司整体的产品和服务水平的提升；
2.科学性：根据对目标的分析和预测，组织人员可以得出可执行的计划，使团队达到效益；
3.透明性：所有人都可以看到目标和关键结果，团队中的每个人都知道自己的贡献，每个人都有责任完成目标，让公司整体的工作进度、质量以及效率得到合理的管理。

## 为何要使用OKR
* 高效率的工作：OKR能快速且有效地对个人和团队的工作进行分配和指导。它不断迭代，快速适应变化，节省资源开支。
* 更好的沟通：OKR制定的目标会随着团队的进展不断更新，每周或每月都会聚焦一次，从而实现广泛而深入的沟通。
* 提升员工满意度：OKR能让员工看到自己和团队正在实现哪些目标，从而得到满意的反馈，进而影响整个团队的工作和生活方式。
* 优化工作流程：OKR通过分析员工的工作情况，调整工作方式，提升工作质量，改善管理，协同合作等方面，对工作流程进行优化。
* 提升团队士气：OKR能够激发团队的积极性，推动团队向更高的目标靠拢，提供多元化的想法，并释放员工的潜力。

# 2.核心概念与联系
## KPI（关键绩效指标）
### 什么是KPI？
KPI （Key Performance Indicator，关键绩效指标）是指企业用来衡量一个单位的绩效、性能、生产力、创新能力等最终输出指标，是企业绩效管理的重要手段。KPI的目的在于通过对衡量单位活动或者营销策略的真正效果的客观度量，来判断其质量、效率、成本及持续性的有效性、有效性、成本及持续性，以及是否符合预期。其往往采用客观、实际的数据，用于衡量各项指标的进步、稳定性和健康状态，有助于指导企业不断发展，确保目标的达成。

### KPI如何确定？
KPI确定需要注意以下几点：

1. 衡量指标的客观性：KPI的措施应该直接测量、量化并且能够直接反映企业目标的效益。
2. 可测性：KPI应是衡量指标的唯一标准，不能有其他外部因素干扰，如销售额等其他数据指标。
3. 跟踪机制：KPI的设定与实施均需有跟踪机制，一般由制定者主动上报实时信息，也可设定相应KPI审计、监控方案。
4. 优先级设置：为了保证高效的运作，KPI应按照优先级进行排列，并由制定者公布、监督。
5. 数据展示：KPI统计数据及图形展示，需要清晰、易读、直观。

## OKR和KPI有何不同？
### KPI和OKR都是衡量单位的工作绩效的方法。两者的区别如下：

1. 维度不同：KPI是衡量某个单一维度的绩效，如销售额；而OKR是衡量多个维度绩效的组合目标，如销售目标、研发目标、产品目标等。
2. 方法不同：KPI使用单一指标，如销售额，评判工作业绩的标准；而OKR使用不同目标，如销售目标、研发目标等，并制定相应的目标和关键结果，通过目标细化、里程碑式的逐步完善，从而促进各方面的效率提升。
3. 周期不同：KPI的统计周期较长，如季度报告、半年报告等；而OKR的制定周期通常较短，如几个月甚至几天，从而聚焦在当下工作的优先级决策。
