# 1.背景介绍


## 概述
近年来，随着电子商务的蓬勃发展，网上购物、社区购物、本地购物等新形态的消费模式已经成为主流。开源电子商务平台也是新生事物。作为一个开源项目，其不断完善的特性和广泛的社区参与者，让许多企业和个人能够将开源电商运用到实际生产中，并逐渐成为新的经济增长点。但是，由于国内尚未出现一套全面、统一的开源电商平台体系，各个公司自身又没有能力进行完整的开源电商实践，导致不同平台之间的竞争激烈，最终导致市场供需失衡。例如，只有京东才可以应用开源电商解决方案，淘宝、天猫等其他平台均无法应用。这种情况下，基于开源电商平台构建一个统一的平台体系乃至于产业链整合则是一个重要任务。
为了促进开源电商的发展和推广，中国政府期望通过举办相关的行业会议、培训、研讨会、交流会等方式，在业界、媒体、学术界、企业界之间形成一套完整的开源电商发展蓝图。而今年的“中国开源电子商务峰会”正是这一系列活动中的一环。本通知即是对此会议的开幕宣言，旨在向大家公布一些时间、地点、相关方面的信息，欢迎大家踊跃报名参加。

## 主题及范围
峰会主题：“数字化、云计算与区块链技术的全面结合”，邀请世界顶级技术专家，深入交流探讨区块链、人工智能、云计算、大数据、人机交互、物联网、量子计算、超算、网络安全、云计算平台、企业级分布式数据库、云存储等领域最新热门研究成果及产品，分享解决方案和经验；同时，邀请业界及政界领袖，展示区块链领域最前沿的发展动态和创新创业企业，分享他们的成功经验和企业的创新思路，展示区块链产业的前景和商机。
峰会范围：包括但不限于以下领域：
- 区块链领域前沿技术与创新，包括但不限于共识机制设计、智能合约、超级账本、隐私保护、联盟链、智能资产、边缘计算、AI、区块链智能应用
- 区块链应用案例解析，包括商业模式变革、金融应用、医疗健康、供应链管理、IoT、人防等领域的实践案例
- 区块链行业分析与投资评估，包括区块链领域的研究机构、投资机构、评估机构等
- 区块链技术交流和交流平台的建设，包括虚拟峰会、线下活动、线上论坛、微博、微信群、知乎、GitHub
