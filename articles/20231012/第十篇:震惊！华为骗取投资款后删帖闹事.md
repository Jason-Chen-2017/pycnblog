# 1.背景介绍



近日,网上有不少自媒体文章质疑华为通过华中联合研究所(WHU)骗取投资款。华为技术有限公司(武汉)副总裁刘勇表示,骗取华为投资金钱后影响到华为管理层的正常运作,导致很多部门的人身安全受到威胁。这些骚扰贴的背后，都是华为已经毫无察觉地向一帮外国政府官员提供了重要的敏感信息。


“华为何至于如此卑劣的骗取资源？华为真的不能够正常运转吗？”在这样的疑问下,一位匿名用户留言道。这件事情让很多人感到非常震惊。华为作为中国最高科技公司,对全球各国造成巨大损失,为这个企业的声誉和国家利益做出了巨大贡献。然而,这次的事件并没有触及华为的底线,它只是做了一个小小的让步。对于华为来说,它依旧是一家有着深厚历史渊源和领先优势的企业。


无论如何,无论华为对这次骗取资源的行为持什么态度,我们都不会忘记这样一个事实:华为的存在和发展离不开美国、英国等国家的资助。骗取这样的投资资源的行为背后,正是技术的进步和人才的力量。这也是为什么美国、英国等国家一直在支持华为发展的原因之一。


回顾过去几年间,华为在面临国际竞争的同时,也大力提升自己内部的制造能力,积累了巨大的商业价值。2017年的时候,华为以高端芯片和车用芯片领域的重组赛道第一的地位夺得了业界的尊重。今年五月份,华为与英特尔正式宣布合并,这标志着华为内部的重整旗鼓。为何华为会在这么短的时间内就改变方向,转移资源,却还要继续骗取投资资源?只有更深刻的理解才能彻底解决这一问题。


作为一家真正的顶级科技企业,华为必须坚守“不破不立”,并且充分利用市场的需求,持续加强自主研发能力,把握技术主动权,才能顺利走向成功。要想保护好自主创新能力,不被一些国际企业过度依赖,保障公司技术利益和国家利益的平衡,是华为不可推卸的责任。


# 2.核心概念与联系

美国国务院发言人斯蒂文斯表示,美国国家情报局正在调查一家名叫白宫旅行社(Baltimore Tourism Company)的公司盗取华为技术有限公司(Wuhan Huawei Technology Co.)的重要文件。


白宫旅行社作为美国最具权势的旅行社,拥有欧洲最高级别的出境许可。为了获取更多信息,白宫旅行社非法收集了一批华为数据。其中包括机密文件、业务计划、审计文档、关键项目资料、经销协议等。由于没有美国法律依据,可以断定白宫旅行社收集了这些信息用于各种目的。


另一方面,华为向白宫旅行社提供的信息与相关人员建立关系,或通过签订协议收取旅游费,这反过来又可能诱使白宫旅行社窃取相关信息。而这些途径也正在被用来掩盖华为的财务状况、各种策略目标,并借此牟取政治优势。


因此,美国国务院发言人斯蒂文斯表示,基于这些违规行为,美国政府正在考虑采取必要的法律手段进行调查。此外,白宫旅行社还可能受到其他类似情形的威胁,随时准备加剧骚扰行为。


但是,由于缺乏具体证据,以及华为在这件事情上的态度立场,很难判断这件事情是否构成了严重违反国际法的行为。我们必须以客观的态度看待这一事件。即便白宫旅行社采取的是非法行为,但如果是出于国家利益的需要,比如为了获取全球援助,那也是可以理解的。换句话说,我们还是应该以科学精神,站在法律的角度,严格遵循法律法规,尤其是在涉及敏感和复杂的交易纠纷时,应当充分尊重法律的约束。
