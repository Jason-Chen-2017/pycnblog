# 1.背景介绍


在Web开发中，我们经常需要对文本进行排版、布局、美化等处理，以便让其更加易读，友好地显示在用户面前。比如，我们可以使用CSS中的`text-align`，`display`，`float`，`margin`，`padding`等属性来控制文本的位置、大小、颜色等效果；或通过HTML标签的`div`、`p`等结构来控制文本的换行、缩进、分页等方式。这些都是基础的排版技巧，它们帮助我们快速高效地设计出具有视觉效果佳的页面。

然而，当我们使用浏览器浏览网页时，往往会发现浏览器默认给我们展示的字体不够好看。我们可能要花很多时间才能找到合适的字号、字体、字重、行距等参数，才能让文字看上去清晰舒服。而且，如果某些特殊字符的渲染效果不尽人意，或者浏览器本身的Bug导致页面不能正常显示，那么这些排版技巧就无能为力了。

所以，为了能够应对日益增长的web应用中复杂、多样化的排版需求，解决浏览器显示效果问题，W3C工作组推出了一个新的Web标准——CSS3 Text模块，它为文本排版提供了一个全新的视觉效果，可以实现更丰富的排版功能，包括文本换行、缩进、对齐、空白处理等，并兼容各主流浏览器。

本文基于这个新标准的一些特性、原理及特点，对文本排版的基本知识进行阐述，并结合实际案例分析该模块的一些使用方法及注意事项。

# 2.核心概念与联系

## 2.1 概念

### 2.1.1 text-indent
顾名思义，text-indent属性用来设置文本首行的缩进。值是一个长度值，可以是像素值（px）、百分比值（%）、em值等。

例如：

```css
h1 {
  text-indent: 2em; /* 设置文本首行的缩进 */
}
```

设置完成后，该段文本将向右缩进2em。

### 2.1.2 word-spacing
word-spacing属性用来设置字符之间的间距。值可以是像素值（px）、em值等。

例如：

```css
p {
  word-spacing: 1em; /* 设置字符之间的间距 */
}
```

设置完成后，字符之间的间距将增加1em。

### 2.1.3 letter-spacing
letter-spacing属性用来设置单词之间的间距。值可以是像素值（px）、em值等。

例如：

```css
p {
  letter-spacing: -0.1em; /* 设置单词之间的间距 */
}
```

设置完成后，单词之间的间距将减少0.1em。

### 2.1.4 line-height
line-height属性用来设置行高。值可以是像素值（px）、百分比值（%）、em值等。

例如：

```css
p {
  line-height: 1.5em; /* 设置行高 */
}
```

设置完成后，该段文本的行高将设置为1.5em。

### 2.1.5 white-space
white-space属性用来设置元素内的空白处理方式。取值如下：

| 值          | 描述                                                         |
|------------|--------------------------------------------------------------|
| normal     | 默认值，空白会被合并，文本不会换行                              |
| pre        | 空白会被保留，文本也不会换行                                  |
| nowrap     | 文本不会换行，但是文本会溢出容器                                |
| break-spaces| 会保留空格符                                                  |
| inherit    | 将继承父元素的white-space属性                                  |

例如：

```css
pre {
  white-space: pre; /* 保留空格符 */
}
```

设置完成后，<pre>元素中的空白符号将被保留。

## 2.2 属性总结

- `text-indent`: 设置文本首行的缩进。
- `word-spacing`: 设置字符之间的间距。
- `letter-spacing`: 设置单词之间的间距。
- `line-height`: 设置行高。
- `white-space`: 设置元素内的空白处理方式。

## 2.3 联系

除了上面提到的属性之外，CSS3 Text还提供了一些其他特性。其中，有一些特性可以用到上面的属性中。例如，text-shadow属性可以制作文本阴影，可以用到`text-indent`属性和`line-height`属性。

因此，应该结合实际情况，灵活运用不同的属性组合，来达到特定效果。
