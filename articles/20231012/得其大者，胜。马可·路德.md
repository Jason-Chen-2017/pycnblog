# 1.背景介绍


马可·路德（<NAME>），德国作曲家、钢琴家，被誉为“欧洲第一交响乐制作大师”。20世纪初，他将交响乐制作这一领域引向了极高潮。但随着时间的推移，许多人质疑路德是否真的如此出色？事实上，他把自己的创造力和才华奉献给了科技和艺术。在著名的德语电影《蒂凡尼的早餐》中，我们可以看到路德带领大家一起品尝他精致的音乐。

在1973年，路德在巴黎创立了他所说的“世界性的交响乐俱乐部”——指的是交响乐公司而不是个人单位。由于成功地推出了重要的作品，包括《布鲁克林Prelude》、《卡尔维诺Waltz》等，并赢得了非凡的收视率，使得路德的名字与声誉受到全球欢迎。

在20世纪70年代末期，路德逝世后，欧美各大文化机构纷纷要求为他题词，很多当时的明星纷纷创作身边小编，记述着自己的对联或诗歌赞美他，希望“赠予他新的生命”。然而，这些搭讪则没有产生太大的效果。最终，留下来的都是泛泛的感叹和肯定的评论，不少人觉得路德像个死了不活的人一样。



最近，“另一个马可·路德”出现在各大媒体报道中。这是一个深层次的传奇故事，令人惊异。这位名叫科特·杰弗森（Cote de Joe）的意大利作曲家虽然是路德的学生，却不是一位“重量级”的音乐家，相反，他只是个音乐批评家，认为路德的作品缺乏感染力。这位意大利歌手称他为“著名的路德派音乐批评家”，同时又反感路德本人。这两方面都让人不安。

这位意大利音乐批评家曾经向《时代》杂志介绍过路德的《海顿的广场》，被译成中文发表后，引起了轩然大波。2015年，他接受福斯特·史密斯采访时表示，自己对于路德的看法正在发生变化。他说：“路德很有才华，但他也很有风险。因为，如果你想真正有所作为，就应该知道自己究竟能做什么，而且不能做什么。如果你坚持不懈，只会招来厄运。”

2016年，意大利歌手科特·杰弗森再度翻唱路德的作品，这一次销售额达到了4.2亿美元，成为历史上销量最高的一张唱片。但这次翻唱却遭遇了质疑，质疑者认为这是歪曲、篡改了路德的作品。更离谱的是，还有另外一些媒体甚至站出来否认路德的存在。

# 2.核心概念与联系
简单来说，路德著作中的核心概念和关联的元素是什么呢？

核心概念：

1.交响乐：代表了现代音乐的发展趋势，它由4/4拍子组成，同时它也是所有音乐的基础。

2.旋律：即一段旋律，它首先要传达给听众，然后才是听众自己去创作。

3.音符：指乐器上的触动点，每个音符都有自己的音调、强弱和节奏。

4.歌词：即旋律中的一段文字。它可以用来传达情绪、描述人物心情、描绘场景。

5.声部：就是乐器，不同的乐器承载着不同的旋律，有的是管乐器，有的是架子鼓，有的还是竖琴。

6.节奏：指旋律在每个音符之间停留的时间长度。

7.节拍：指四分音符、八分音符等。

8.和弦：由两个以上同根音符组成的音阶。

9.主旋律：主要的旋律，一般情况下为四分音符和八分音符的连续音符，和弦声。

10.倾倒性音阶：属于和弦的一种。它的开头是全长音，而其余的音符全都会变短。

11.混音：是指将不同乐器或声音的声音混合在一起演奏。

12.曲目结构：由一系列的旋律组合而成的乐曲。

关联的元素：

1.旋律和节奏之间的联系：旋律之所以能够传达，是因为它可以产生节奏，节奏就可以影响旋律的方向、速度、音调以及音色。

2.歌词的目的：是为了传达旋律和情绪。

3.声部的作用：从乐器的角度上来说，它可以播放旋律。

4.节拍的作用：让乐器的运动速度慢慢减缓或者加速。

5.和弦的作用：把音符的关系组合起来，形成独特的旋律。

6.主旋律的作用：是整个乐曲的骨架，可以确定其他旋律的基准。

7.倾倒性音阶的作用：可以让音符的节奏变得不规则，反映出不同人的情绪变化。

8.混音的作用：可以把不同声音进行整合，创造出更多的色彩。

9.曲目结构的作用：控制整个乐曲的韵律，起到统一和调性的作用。
