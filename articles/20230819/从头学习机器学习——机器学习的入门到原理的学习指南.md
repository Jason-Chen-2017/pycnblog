# 2. 背景介绍
## 2.1 概述
### 2.1.1 发展历史与介绍
　　机器学习（英语：Machine Learning），是一个跨领域的交叉学科，涉及计算机科学、数学、统计学、理论物理、电气工程及其他多个学科的交叉研究。它是一套基于数据、算法、模型和系统的科研方法，目的是使计算机具有“学习”的能力，能够从数据中获取有效的知识，自动发现模式并予以预测或改善现有系统的性能。按照历史发展阶段划分，机器学习可分为监督学习、无监督学习、半监督学习、强化学习、迁移学习五大类。其中，监督学习一般由训练数据包括输入变量和输出变量构成，即有监督学习。监督学习通常需要大量的标记数据才能训练得到一个能够预测出输出的模型。

　　现在机器学习主要应用在自然语言处理、图像识别、音频识别、视频分析、推荐系统等领域。如今深度学习技术发展迅速，已经成为主流人工智能技术之一。深度学习算法不同于传统的机器学习算法，它采用多层次神经网络，输入数据经过层层计算之后形成新的特征，再将这些特征作为输出进行分类、回归。深度学习应用举例：图像分类、目标检测、图像描述、视频动作识别、文本情感分析等。

　　随着机器学习的应用范围越来越广泛，随着数据的增长速度和需求增加，监督学习算法也面临着严重的数据稀疏问题，如何利用数据有效地训练出稳定的模型是机器学习的一个重要课题。如何提升模型的精度，减少模型的误差，提升模型的鲁棒性，这些都是机器学习的重要研究问题。
