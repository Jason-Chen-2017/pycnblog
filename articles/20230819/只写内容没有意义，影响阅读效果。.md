# 2.相关背景介绍
## 2.1 什么是目标检测？
目标检测，即识别和定位图像或视频中感兴趣的对象（例如人脸、车辆、行人等）的位置、大小、形状、类别等信息。在图片或者视频中识别出目标后，可以进行跟踪、分类、行为分析、警戒、警报、决策等功能，提升生活质量和效率。目标检测一直是计算机视觉领域的一个重要研究方向，它的关键技术就是如何根据图像特征找到感兴趣区域并准确确定目标物体的位置。

## 2.2 深度学习
深度学习是人工智能的一个分支，它基于神经网络的生物学启发，模仿人脑神经元的分布结构，将多层神经网络堆叠得到一个具有多个隐层的非线性映射关系。深度学习系统的输入数据是低纬度（如灰度图像）或高维度（如激光雷达扫描数据），通过处理数据得到更高层次的特征向量，再通过非线性函数转换成输出结果。通过迭代学习，网络逐步自适应调整参数，最终达到模型准确预测目标的能力。

深度学习在目标检测领域的应用主要集中在两个方面：第一，通过卷积神经网络（CNN）模型提取图像的高级特征；第二，采用编码器-解码器结构构建多尺度的边界框回归网络（Region Proposal Network）。

## 2.3 SSD简介
SSD全称Single Shot MultiBox Detector，是深度学习的一种目标检测方法，主要解决单次输入图片产生大量候选区域的问题。相比于传统的基于区域Proposal的方法，SSD避免了产生大量候选区域所带来的时间和内存开销，从而达到更快的识别效果。SSD由分类子网络和回归子网络组成，前者负责在各个不同尺度生成类别置信度，后者则负责对每个候选区域进行精细回归，获取其在不同尺度下的坐标和长宽尺寸信息。SSD算法流程如下图所示：

## 2.4 单目标检测VS多目标检测
单目标检测（Object Detection）和多目标检测（Multi Object Detection）是两大类目标检测任务。单目标检测仅检测出一类目标，如人脸检测、车辆检测、行人检测等；而多目标检测同时检测出多个不同类别的目标，如同时检测多个人、车、行人等。通常情况下，多目标检测需要更强大的特征提取算法，如使用高级卷积神经网络。但是单目标检测由于不需要考虑多个目标间的干扰，所以速度更快，可满足实时场景的需求。
