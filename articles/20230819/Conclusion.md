# 2.相关概念及分类
## 2.1 机器学习
机器学习(ML)是指让计算机具备学习能力，从数据中发现模式，并根据模式做出决策或预测的一种技术。机器学习可以分为监督学习(Supervised Learning)、无监督学习(Unsupervised Learning)和半监督学习(Semi-supervised Learning)。
### 2.1.1 监督学习 Supervised Learning
监督学习就是给定输入的情况下，已知输出结果。比如，假设训练集中有一组输入数据X和对应的输出标签Y，希望学习一个函数f(x)来映射输入到输出。这个映射函数使得当输入数据X发生变化时，输出的变化也是显而易见的。例如，图像识别中的手写数字识别就是典型的监督学习任务。监督学习通常由以下几个步骤构成：

1. 数据收集：用手写字体或扫描件标记好输入和输出样本；或者自动生成输入-输出样本对；

2. 数据准备：清洗数据，去除噪声，特征工程等；

3. 模型设计：选择模型，定义损失函数，确定优化目标；

4. 模型训练：迭代更新参数，使模型拟合训练数据；

5. 模型评估：计算模型的性能指标，衡量模型的预测准确性和可信度；

6. 模型推广：将模型应用于新数据，获得预测结果。

在监督学习中，存在着两种类型的模型，即判别模型(Classification Model)和回归模型(Regression Model)。判别模型用于分类问题，如图像识别中的手写数字识别；回归模型用于回归问题，如电影票房预测、销售额预测等。两种模型都可以通过极大似然估计法(Maximum Likelihood Estimation, MLE)或正则化逻辑回归(Regularized Logistic Regression)进行训练。

### 2.1.2 无监督学习 Unsupervised Learning
无监督学习(Unsupervised Learning)没有标签的输入数据，目的是寻找数据的内在结构和规律。由于没有标签，因此无法评估学习效果，只能靠某种度量来评估学习过程是否收敛。一般来说，无监督学习可以分为聚类(Clustering)和降维(Dimensionality Reduction)两大类。聚类算法通过学习数据点之间的相似性，将相似的数据划分到同一个簇(Cluster)里，而降维算法则试图找到低维空间中的投影，将多维数据映射到二维甚至三维的子空间中。常见的无监督学习算法有K-Means、EM算法、隐变量前馈网络(HMM, Hidden Markov Models)和GMM(Gaussian Mixture Model)。

### 2.1.3 半监督学习 Semi-supervised Learning
半监督学习(Semi-supervised Learning)是指有部分数据既有输入也有输出，但只有一部分数据有标签。它的目标是在部分有标签的样本上训练模型，然后利用未标记数据增强模型的学习效果。其方法可以分为基于规则的方法和无监督模型辅助的方法。基于规则的方法往往依赖于专家的知识，通过规则判断输入的输出关系，如标记某些样本为有标签或无标签。无监督模型辅助的方法通过学习无监督模型来得到少量的标签信息，再利用有标签的样本辅助训练模型。常见的无监督模型包括EM算法、层次聚类的DBSCAN算法、GMM(Gaussian Mixture Model)等。

## 2.2 深度学习 Deep Learning
深度学习(Deep Learning)是机器学习的一种应用方法，它利用多层神经网络来学习复杂的数据表示形式。深度学习由多个隐藏层组成，每一层都是由许多神经元组成的，并通过不断传递信号来学习输入数据的表示形式。深度学习的一些关键技术有卷积神经网络(Convolutional Neural Network, CNN)、循环神经网络(Recurrent Neural Network, RNN)、递归神经网络(Recursive Neural Network, RNN)、注意力机制(Attention Mechanism)、长短期记忆网络(Long Short-Term Memory, LSTM)和变分自编码器(Variational Autoencoder, VAE)。深度学习的好处在于能够自动提取特征并表征全局，并有效处理高维度、非线性、缺乏标签的数据。

## 2.3 强化学习 Reinforcement Learning
强化学习(Reinforcement Learning, RL)是机器学习的一个子领域，它是通过与环境互动的方式来提升行为。RL最大的特点在于它是一个动态系统，系统会不断探索与环境的互动，通过反馈与环境的奖赏来塑造策略，最终达到最佳策略。RL的三个要素是代理(Agent)，环境(Environment)，环境状态(State)。代理是一种智能体，它可以在与环境互动过程中学习如何制定策略，环境是RL的外部世界，它提供给代理可感知到的信息，并向代理反馈奖励或惩罚，描述当前环境状态。RL的算法包括Q-Learning、SARSA、Actor-Critic、DDPG、TD3等。通过对代理的策略进行调整，RL可以解决复杂的连续控制问题，并应用于智能驾驶、自动交易等领域。
