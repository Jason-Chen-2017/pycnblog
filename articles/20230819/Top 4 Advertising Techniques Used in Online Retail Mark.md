# 1.简介
  

## 1.1 引言
在电子商务、互联网零售领域，广告一直都是重要的收入来源之一，其投放效果直接影响到产品销量和用户满意度。然而，如何投放有效且高效的广告仍然是一个值得探索的问题。在线零售市场（e-commerce marketplace）作为当今最主要的广告渠道之一，已经成为新的增长点。其中，市场营销策略有着巨大的影响力。本文将从多方面分析和总结目前最流行的四种市场营销策略：关键词广告、搜索广告、竞价排名广告和行为追踪广告。并且，根据Fashion eCommerce平台的特点，提出了对这四种广告策略的应用建议。
## 1.2 本文概要
本文主要内容包括以下几个方面：第一章是绪论部分，介绍相关背景知识及已有的研究成果；第二章是关键词广告部分，阐述关键词广告的定义、工作原理和优缺点；第三章是搜索广告部分，分析关键词广告在搜索广告中的作用以及优化方法；第四章是竞价排名广告部分，分析竞价排名广告的定义、工作原理、优缺点及应用场景；第五章是行为追踪广告部分，阐述行为追踪广告的定义、工作原理、优缺点及应用场景；最后一章给出了Fashion eCommerce平台上应用这几种广告策略的建议。
# 2 相关背景知识
## 2.1 搜索广告
搜索广告（Search advertising）是一种在网站首页、结果页面或搜索框等位置以各种方式呈现的广告。相对于直接向目标消费者展示商品详情页上的广告，搜索广告侧重于吸引用户查看店铺页面或进行搜索。一般情况下，搜索广告是通过跟踪消费者在搜索框输入关键字的时间、地点、设备和网络环境等信息，以收集用户偏好并向符合条件的用户展示广告而实现的。
### 2.1.1 Google Adwords
Google Adwords是谷歌推出的搜索广告服务，提供软性定向关键字广告服务，帮助用户发现相关的商业机会。它提供了丰富的广告设置选项，可以灵活控制广告的价格、时限、语言、位置、显示内容、链接地址等。Google Adwords的广告投放流程分为创意——创建——投放——监测——优化五个阶段。除了通过关键字广告外，还有基于兴趣标签、品牌建设和受众定位的广告类型。
图1：Google Adwords的广告投放流程
### 2.1.2 Bing Ads
Bing Ads是微软推出的搜索广告服务，全称为Microsoft Bing Ads。它提供了丰富的广告设置选项，可以灵活控制广告的价格、时限、语言、位置、显示内容、链接地址等。Bing Ads的广告投放流程分为创意——创建——投放——监测——优化五个阶段。除此之外，还支持跨设备、第三方监测工具的跟踪，可实时掌握各项数据。
### 2.1.3 Amazon Advertising
亚马逊的广告业务通过为消费者找到满意的购物体验，促进他们的回购率，提升平台的黏性，并帮助广告主赚钱，这是其最大的特色。它拥有庞大的在线广告库，可以为每位用户进行精准定位。同时，它还可以通过广告管理工具提供更多精准的控制功能。亚马逊也提供实时报告，可以让广告主了解自己的广告效果。
### 2.1.4 Facebook Advertising
Facebook的广告服务包括FeedAds、MessengerAds、Branded Content、In-Stream Video、Video Campaigns、Targeted Ads等多个产品，都可以帮助广告主生成并管理影响力较大的广告。它集成了直观的编辑器，可以快速制作具有强大效果的广告创意。同时，Facebook也提供很多广告分析工具，方便管理员监控各项指标。
## 2.2 竞价排名广告
竞价排名广告（Auction-based advertising）是指在线广告中，广告主以预先约定的竞价价格出价，所得出的一系列广告中，根据出价情况由系统自动按竞价顺序投放给广告客户。竞价排名广告常见于搜索引擎和社交媒体类市场，因其投放范围广泛、加之广告主预置的竞价价格，使其能够获取更加公平的广告效果。
图2：竞价排名广告流程示意图
### 2.2.1 CPC、CPM、CPE
CPC（Cost Per Click）是广告点击付费模型，即每次点击产生广告费用。CPM（Cost Per Mille）是基于时间的广告模型，即每次展示次数产生广告费用。CPE（Cost per Engagement）是基于用户参与度的广告模型，即每次有新用户访问产生广告费用。
图3：CPC、CPM和CPE的比较
### 2.2.2 Facebook Audience Network
Facebook Audience Network(简称FAN)，是Facebook推出的免费网络广告解决方案。FAN可以帮助广告主针对特定目标群体创建广告，如年龄、性别、兴趣、兴趣爱好、职业、消费习惯、地区等。通过FAN，广告主可以在不影响原生广告体系的前提下，快速扩大数字化业务规模，提高收益。
## 2.3 用户体验设计
用户体验设计（User Experience Design，简称UXD）是一门与人的交互、沟通及品味息息相关的专业学科。它涉及视觉、听觉、触觉、运动、味觉、身体、心理、认知等多种感官的协调配合，将需求、愿景、构想转换成专业的产品形象。其目的就是为了改善用户在使用产品时的体验，提升用户满意度。
UXD的关键是建立用户中心，倾听用户声音，善用工具，并且坚持反馈循环。以下是UXD相关的工作方法：
### 2.3.1 可视化设计
可视化设计（Visual Design）是一种基于图像的界面设计方法，旨在通过对元素的形状、颜色、纹理及布局的高度关注，把注意力放在产品中最重要的信息和操作，提升用户的认知和体验。
### 2.3.2 产品策略
产品策略（Product Strategy）是企业或团队围绕产品开发或推广开展的一系列活动，包括市场定位、品牌策略、产品规划、运营策略、组织结构、资源分配和财政计划等。其目的就是通过将产品推向市场，打造一支优秀的产品团队和核心团队。
### 2.3.3 人机交互
人机交互（Human–Computer Interaction，简称HCI）是一门研究人员和设计师使用的计算机应用、产品及服务之间的互动关系的科目。它注重计算机技术的演变、可用性、多样性、易用性、沟通性和智能性的影响。
### 2.3.4 全栈设计
全栈设计（Full Stack Design），即整个项目的用户体验设计，是一个综合性的过程，需要涉及到多种设计技能，包括视觉、互动、动机、结构、信息、动态和语言等。
# 3 关键词广告
## 3.1 定义
关键词广告（Keyword advertising，又称品牌内外链、关键词推送），是一种以关键词投放为基础的网络广告形式，通过在搜索引擎、社交媒体或移动应用上的搜索结果中插入相关的词条或链接，通过搜索消费者的关键词搜索到相关的商品或服务，并通过相应的展示形式吸引用户点击广告，实现商品和服务的推广。
## 3.2 工作原理
关键词广告的工作原理如下：
1. 投放方设置关键词
首先，投放方需要选择自己熟悉的关键词，比如“裤子”，然后在搜索引擎、社交媒体或移动应用上发布相关信息。
2. 用户搜索
用户搜索引擎、社交媒体或移动应用上的关键词后，就可看到广告信息。
3. 用户点击
当用户点击广告信息时，广告商会给予响应，比如购买商品或下载APP，并收取相应的佣金。
4. 流量变现
用户点击广告后，流量变现，广告商获得相应的收益。
### 3.2.1 适用场景
1. 在线零售市场
由于在线零售市场的快速发展，现在已经成为各个消费者的首选购物途径，因此，关键词广告经常被用于该领域的广告推广。
2. 美妆电影票房
以中国妇女健康频道为例，该频道推出“缝纫券”“护肤券”等关键词，旨在引导消费者去线上购买保健品和化妆品。
3. 投资基金
在传统金融投资过程中，投资顾问经常会选择一些有代表性的股票，并寻找有关关键词，例如“美元”、“黄金”等。
4. 汽车保险
汽车保险公司在社交媒体上也经常推出自己的关键词，例如“车损”、“质保”等。
### 3.2.2 效果评估
关键词广告的效果评估，可以基于以下几个维度：
1. 曝光率：广告被用户看到的次数。
2. 点击率：广告用户点击次数。
3. 转化率：实际完成交易的订单数量。
4. 重复点击率：同一个用户点击相同广告的次数。
5. 收益：广告变现金额。
6. 盈利能力：广告商从广告获得的收益是否超过广告投入的成本。
7. 时效性：广告的投放时间是否达到预期。
## 3.3 优化方法
关键词广告的优化方法有：
1. 内容优化：广告的内容应与相关的商品或服务保持一致，并且要符合用户的搜索需求。
2. 时效性优化：广告的投放时间应尽可能长，以吸引用户阅读完整内容。
3. 地域优化：广告投放的区域应与消费者所在地区相匹配，以提升广告效果。
4. 关键词编辑：搜索引擎、社交媒体或移动应用上的关键词应精准匹配相关商品或服务，并且不能太过宽泛，否则容易刷屏。
5. 竞争对手竞价：对竞争对手的关键词进行竞价，以吸引消费者购买相关商品或服务。
6. 受众定向：制定受众群体，限制广告投放的对象，从而提升广告效果。
7. 数据分析：收集并分析数据，发现用户最喜欢的关键词，提升相关商品或服务的推广效率。
8. 风格营销：通过品牌形象和样式营销，吸引消费者消费更多相关商品或服务。
