# 2.基本概念术语说明
首先，对于机器学习相关的基本概念、术语等的一些介绍：

- **机器学习** (Machine Learning) 是指计算机基于数据构建的模型，使计算机能够自动学习，并解决某些任务。它由 <NAME> 在20世纪60年代提出，并经过多年的发展。机器学习有三种类型：监督学习、无监督学习、半监督学习。监督学习，如分类、回归，训练样本带有标签信息；无监督学习，如聚类、降维，训练样本没有标签信息；半监督学习，既有标签信息又有无标签信息，其中一部分样本可能已经标记，另一部分样本则可以由模型自己判断标签信息。
   - **监督学习**：监督学习指的是给定输入变量 X 和输出变量 Y 的情况下，利用已知的训练数据对预测函数进行训练，从而得到一个模型，这个模型能够根据新的输入 X 来预测输出值 Y。监督学习分为回归分析和分类方法。分类方法中最常用的方法是决策树，它通过构造一系列的条件规则来分类，每个条件规则表示一个判别标准，用以区分不同的输出类别。回归分析的方法主要包括线性回归和多元回归。线性回归是一种简单的方法，它用来拟合一条直线或曲线，能够将一组特征的值映射到一个连续的实数值上。
   - **无监督学习**：无监督学习是指利用没有标签的数据集来发现数据结构。常用的无监督学习方法有聚类、异常检测、关联分析、概率论。聚类方法通过找寻相似性，将数据点划分到相似的类别中，但无法知道每个数据属于哪个类别。异常检测方法通过识别不符合一般数据模式的数据点，帮助发现异常数据。关联分析方法通过分析数据之间的关系，识别出变量之间的共同模式。概率论的方法主要包括朴素贝叶斯、隐马尔可夫模型、EM算法。
   - **半监督学习**：在某些情况下，我们既有输入数据 X，也有输出数据 Y，但大部分情况是只有输入数据 X。这时候就需要借助无监督学习和监督学习的结合来完成学习过程。半监督学习中的损失函数通常会有一个额外的正则化项，鼓励模型将已知的标签信息转换到未知的无标签数据上。
- **模型评估**：在实际应用中，模型是输入、输出和参数构成的黑箱子，对于模型的评估就是要衡量模型的泛化能力。机器学习提供了多种方法来评估模型性能。最常用的方法是留出验证集（holdout validation）和交叉验证（crossvalidation），即把数据集分为两个互斥的集合，分别作为训练集和测试集。留出验证集用于选择最优超参数，交叉验证用于评估模型在未见过的测试集上的性能。还有其他的评估方法，如重抽样（bootstrap resampling）、自助采样（bagging）、欠采样（undersampling）、过采样（oversampling）。
- **神经网络**：神经网络是一种多层次的模型，由多个节点组成，每个节点都是一个加权值的乘积和激活函数，目的是模拟大脑神经元的工作方式。神经网络的关键在于学习，通过迭代调整权值，神经网络就可以拟合任意复杂的非线性函数。典型的神经网络有卷积神经网络（CNN）、循环神经网络（RNN）和递归神经网络（RNN）。
- **图结构**：图结构模型是复杂网络的一种常用建模形式，它的特点是采用节点（vertex）、边（edge）、邻接矩阵表示网络，节点表示网络中的实体，边代表节点间的联系，邻接矩阵存储节点间的连接信息。图结构模型有很多变体，如有向图、无向图、加权图、带自环的图等。图结构模型可以非常方便地描述和处理现实世界的网络数据。
- **优化算法**：优化算法是机器学习领域的重中之重，用来找到全局最优解。目前，主流的优化算法有梯度下降法、随机搜索法、遗传算法、 Particle Swarm Optimization(PSO)。每种算法都有其特定的适应场景，在不同的任务中都有着明显的优势。
