# 1.简介
  
 # 1.背景介绍 # 背景

随着产业的不断升级、产业结构的变化、交通工具的不断更新迭代、新型城市出现等因素的影响，智能交通工具的运营管理也越发复杂。如何有效地管理智能交通工具并提升服务质量、降低成本，成为一个重要而艰巨的课题。

目前，智能交通工具的运营管理通常采取的是最简单直接的方式，即从零开始进行建设、维护、保养、维修等全过程。这种方式容易导致运营效率低下，且投入较高，造成不必要的成本开销。因此，提升智能交通工具的运营管理水平、优化运营成本及质量，具有举足轻重的意义。

在这样的背景下，相关专业技术人员需要进一步探索智能交通工具的运营管理优化方式，提出更加智慧化、精准化的运营策略。当前，中国乘用车协会推出了《智能城市高速路旅行车辆运行安全管理标准》(SCMA)，旨在规范智能交通工具及其相关基础设施的运营管理。但由于标准制定部门人手、工作量大、管理层成熟度低，仍存在一些问题。例如，行业内关于运营管理优化的共识缺乏，没有形成可行性方案；政策法规及技术创新机制不够健全、生态系统建设不充分；管理人员管理水平参差不齐、绩效评价指标不统一。

为了解决上述问题，中国乘用车协会主办了“智能交通工具运营优化策略研究”(SMRP)项目，围绕《SCMA》对智能交通工具的运营管理进行持续优化和改善。主要内容如下:

⒈ 推动智能交通工具运营管理标准化、制度化、科技化，提升交通工具运营管理的专业化程度；

⒉ 通过科研成果推动智能交通工具运营管理体系改革，进一步完善运营管理模型和方法论，创新运营管理模式；

⒊ 建立智能交通工具运营管理知识库，构建智能交通工具的运营管理共享平台，实现知识沉淀与分享；

⒌ 探索集成化运营管理体系和业务模式，将传统实体交通运输管理模式和智能交通工具运营管理相互融合，形成面向智能交通工具的整体运营管理模式；

⒍ 重视“企业—客户—社会”三个要素的关系，努力打造国际化、跨领域的智能交通工具运营管理行业联盟；

⒎ 为智能交通工具运营管理提供更多的创新驱动力和技术支撑。

# 3.核心主题 # 核心主题

## 3.1 现状分析 ## 
当前，智能交通工具的运营管理依然是一个“人为”的过程，其中包括整个交通管理链条中的各个环节，涉及多个部门和个人，制约着智能交通工具的运营质量。同时，由于法律、政策法规不断发展、标准不断更新、技术创新不断推进，智能交通工具的运营管理也越来越多地进入数字化时代。

随着智能交通工具的不断普及和应用，越来越多的用户依赖智能交通工具进行生活娱乐、出行接送等方面的需求。但是，由于长期依赖不断发展，现存的运营管理模式存在着很多问题。如：基础设施设备的供应和安装管理不到位、环保、安全、质量保证措施落后、运营管理人员薄弱等。

## 3.2 问题提出 ## 

当前，智能交通工具的运营管理还处于起步阶段，存在诸多困难。以下问题值得关注：

⒈ 过度依赖基础设施设备的供应和安装管理，导致设备质量不可靠、出现故障风险、隐患增加；

⒉ 没有建立完整的交通管理体系，管理部门的管理能力和资源配置无法满足智能交通工具运营需求；

⒊ 由于历史遗留原因，交通运输系统存在很多“人为因素”，管理者经验欠缺；

⒌ 交通运输系统各种数据信息不统一，存在数据孤岛，运营管理者无法有效处理；

⒍ 存在部分运营管理人员管理能力、认知水平不足、执行效率低下等问题；

⒎ 依据现有的运营管理管理制度和标准，很难构建符合要求的系统。

基于以上问题，《SMRP项目》提出了三大方向，即“驱动、改进、应用”，为解决智能交通工具的运营管理优化提供了参考路径。

# 4.核心观点 # 核心观点

## 4.1 规划及目标 ## 

2021年，《SMRP项目》将继续推进“驱动、改进、应用”的战略，围绕“智能交通工具运营管理优化”四项任务：

（1）推动智能交通工具运营管理标准化、制度化、科技化，提升交通工具运营管理的专业化程度；

（2）通过科研成果推动智能交通工具运营管理体系改革，进一步完善运营管理模型和方法论，创新运营管理模式；

（3）建立智能交通工具运营管理知识库，构建智能交通工具的运营管理共享平台，实现知识沉淀与分享；

（4）探索集成化运营管理体系和业务模式，将传统实体交通运输管理模式和智能交通工具运营管理相互融合，形成面向智能交通工具的整体运营管理模式；

四项任务的落实，将在未来的五至十年间，取得重要的成果。

## 4.2 策略建议 ## 

提升智能交通工具的运营管理水平、优化运营成本及质量，具有举足轻重的意义。通过“驱动、改进、应用”的策略建议，我们建议以下两方面做好相关工作：

（1）研究、定义智能交通工具的运营管理特征、优势，及其优化空间；

（2）制定智能交通工具运营管理优化策略，并贯穿始终。

我们期待着与社会各界的广泛反馈，让我们共同讨论和进步。
