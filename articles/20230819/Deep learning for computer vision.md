# 1.简介
  


## 什么是深度学习？
“深度学习”（deep learning）是指通过多层的神经网络来学习输入数据的特征表示或结构信息，并应用于监督学习、无监督学习、半监督学习、强化学习等领域。

## 为什么要用深度学习？

1. 特征提取：用深度学习可以直接从原始数据中学习到图像、音频、视频等各种复杂的特征表示，进而实现图像、声音、语言等的高级理解与处理。

2. 自动学习：训练好的深度学习模型可以对新数据进行快速准确的预测和分类，且不依赖于大量的人工标记。

3. 泛化能力强：深度学习模型具有很强的泛化能力，能够识别和理解未见过的数据。因此，它在自然语言处理、计算机视觉、生物信息学、医疗诊断、金融风控等多个领域都有着广泛的应用。

4. 大规模并行计算：传统机器学习算法需要处理海量数据时效率低下，深度学习通过大规模并行计算的方法解决了这个问题。

5. 模型简单可解释：深度学习模型的可解释性非常强，模型内部的参数权重可以直观地反映出数据的分布和信息流。此外，基于梯度的方法可以帮助我们更好地理解模型的行为及其偏差，从而获得更好的模型优化方向。

## 深度学习的历史

深度学习有多元化的发展史，不过，在20世纪90年代中期提出的多层感知机（MLP），作为一种简单有效的神经网络模型，已经成为深度学习的起点。随后，通过引入非线性激活函数、正则化方法、Dropout技术、深度置信网络等改进，深度学习逐渐走向成熟。近几年，随着大数据、GPU等计算硬件的普及，深度学习也越来越受到了青睐。

# 2.基本概念术语说明

## 数据集、样本、特征、标签
- **数据集**：所有用于训练或测试模型的数据集合称为数据集。
- **样本**：是指数据集中的一个个体或者实体，即一条数据记录。比如说，有图片数据集，每张图片就是一个样本，每个样本代表一张图片。
- **特征**：是指样本所包含的一些特性或信息，用来描述该样本。比如说，有一张图片样本，它的特征可能包括图片大小、颜色、亮度等。
- **标签**：是样本的类别，即样本所属的分类类别。比如说，在手写数字识别任务中，每张图片的标签就是数字。

## 图像分类、物体检测、目标跟踪、语义分割

- **图像分类**（Image Classification）：把图像归类为不同的类别。例如，给一张图片，机器学习模型能够自动判定它所属的类别是哪个。
- **物体检测**（Object Detection）：检测图像中的对象，并给出其位置、形状和类别。例如，给一张图片，机器学习模型能够自动定位出图片上所有出现的物体，并给出它们的位置、形状和类别。
- **目标跟踪**（Object Tracking）：在连续帧中跟踪目标，并给出其移动轨迹。例如，给一系列图像序列，机器学习模型能够自动追踪目标，并给出它们的移动轨迹。
- **语义分割**（Semantic Segmentation）：把图像中不同类别的对象分割出来，并给出它们的区域。例如，给一张图片，机器学习模型能够将图片中不同种类的建筑物、人像和背景等进行分割，并给出各个对象的区域。

## 卷积神经网络、全连接神经网络、循环神经网络

- **卷积神经网络**（Convolutional Neural Network，CNN）：是一种深度神经网络，主要由卷积层、池化层、全连接层组成。在CNN中，卷积核从输入图像中提取特定的特征，然后再与其他特征进行组合，最终输出识别结果。典型的CNN模型有AlexNet、VGG、GoogLeNet等。
- **全连接神经网络**（Fully Connected Neural Network，FCN）：是一种简单而灵活的神经网络模型。在FCN中，每一层都是与前一层的所有节点相连，并且所有节点共享相同的权值。典型的FCN模型有SegNet、U-Net等。
- **循环神经网络**（Recurrent Neural Network，RNN）：是一种深度神经网络，可以处理序列数据，如文本、时间序列数据等。在RNN中，存在隐含状态，并根据历史信息进行当前动作的预测。典型的RNN模型有LSTM、GRU等。

## 损失函数、优化器、验证集、评估指标

- **损失函数**（Loss Function）：是衡量模型预测结果与真实值的距离，并反映模型的性能的指标。常用的损失函数有均方误差、交叉熵等。
- **优化器**（Optimizer）：是更新模型参数的算法，用于最小化损失函数的值。常用的优化器有随机梯度下降法（SGD）、小批量随机梯度下降法（MBGD）、AdaGrad、Adam等。
- **验证集**（Validation Set）：用于估计模型在实际使用的情况下的泛化能力。通过用验证集上的预测结果与真实结果之间的差异来评估模型的泛化能力。
- **评估指标**（Evaluation Metrics）：是用来评价模型效果的标准指标。包括正确率、精确率、召回率等。正确率就是预测正确的图片数量占总图片数量的比例；精确率（Precision）是指预测为正的图像中有多少是真的正类，召回率（Recall）是指有多少真实的正类被成功预测出来。
