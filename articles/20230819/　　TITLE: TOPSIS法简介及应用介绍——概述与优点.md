# 1.简介
  

　　TOPSIS(Technique for Order of Preference by Similarity to Ideal Solution)（可相对性偏好顺序的技术）是一种评价指标设计方法。它以目标函数为中心，通过对各个方案进行排序，从而确定最优方案。TOPSIS法适用于多元决策问题中存在多个目标，或不同目标权重不均衡的问题，因而可以克服传统的简单加权平均值法易受离群值的影响。

　　TOPSIS法首先计算每个方案的“利益/风险”值，并根据它们之间的比较关系，由最高到最低的顺序排列方案。其基本思想是：对于一个决策问题，不同的方案可能具有不同的目标值，因此在做决策时需要考虑到它们之间的可比性，确保选择出最有吸引力、值得信赖的方案。在TOPSIS法中，各方案的“利益/风险”值被定义为距离目标期望值越近的程度，它等于两个方案之间的皮尔逊相关系数的正负号，再乘以两个方案之间欧氏距离的倒数。这种方法能够较好的解决相互依赖或不完全平衡的目标的优化问题，并且能够处理存在缺失值的数据。另外，TOPSIS法支持缺陷分布模型，可以更精确地反映目标函数间的相关关系。

　　除了在优化问题中提升效率之外，TOPSIS法还可以通过降低系统风险、增加业务成功率等方面对决策过程进行干预。在供应链管理领域，TOPSIS法可以帮助公司制定有针对性的采购计划；在销售领域，它可以帮助企业选择产品销售策略，达到更高的业绩目标；在环境污染防治领域，它可以帮助农民减少化肥、农药的损害，提高生产能力；在医疗领域，它可用于改进诊断准确率和患者满意度，减少医疗费用等。总之，TOPSIS法具有广泛的应用前景，能够有效推动决策科学、工程和经济的发展。
# 2.基本概念术语说明
　　## （1）“利益/风险”值（Ideal Risk-Reward Values）

　　TOPSIS法中的“利益/风险”值是一个衡量方案优劣的指标，它由方案的收益值和风险值组成。“收益值”是指目标值与参考值之间的差异，“风险值”则是指未知或随机变量发生变化所带来的损失或失去的机会。

　　利益/风险值越小，表明该方案越能满足用户的需求。换言之，方案的“收益/风险”值越接近于零，则代表着方案的选择性更强，具有更高的可能性被选择。

　　## （2）“目标期望值”（Target Expected Values）

　　“目标期望值”是指系统的期望收益。由于现实世界中系统往往存在多种情况，导致各个选项的收益不仅无法统一定义，而且还有很大的区别，因此，TOPSIS法必须引入目标期望值这一重要的概念。

　　设有n个方案，i = 1,..., n，且第i个方案的目标值Ri = (Yi - µi)^+ / σi^2，其中µi为系统的期望值，σi为标准差。则第i个方案的“目标期望值”Eri = (Yi - µi)^+ / σi^2，可表示为：

  Ei = max { Ri : i = 1,...,n} 

　　## （3）“皮尔逊相关系数”（Pearson correlation coefficient）

　　皮尔逊相关系数是统计学中用来衡量两个变量之间线性相关程度的统计指标。如果两个变量是正相关的，那么他们具有相同的方向和增长趋势，如果两个变量是负相关的，那么他们具有相反的方向和增长趋势，如果两个变量没有任何线性关系，那么它们就不能够帮助我们分析数据。皮尔逊相关系数的值介于-1与1之间，其中0表示不存在线性相关性，-1表示完全负线性相关，1表示完全正线性相关。

　　## （4）“欧氏距离”（Euclidean distance）

　　欧氏距离是两点之间的直线距离。在TOPSIS法中，欧氏距离是用来衡量各个方案之间的相似度的指标。它取两个方案之间的坐标差值的平方和开根号，即sqrt[(xi - yi)^2 +...+(xn - yn)^2]。

　　## （5）“TOPSIS置信度矩阵”（TOPSIS Confidence Matrix）

　　TOPSIS置信度矩阵显示了各个方案与其他方案之间的相关程度。置信度矩阵中的元素Cij表示第j个方案与第i个方案之间的相关程度。置信度矩阵通常是用0~1之间的数字来表示。置信度矩阵的每一行都以最大值1，每一列都以最小值0。

　　0表示两个方案毫无关联，1表示两个方案高度相关。当置信度矩阵中的元素越接近于1，表明两个方案的相关程度越高。当置信度矩阵中的元素越接近于0，表明两个方案的相关程度越低。置信度矩阵的对角线上的元素都是1，表示自己与自己的相关程度为1。因此，在实际应用中，置信度矩阵的其他元素通常不大于0.5，而是介于0和1之间。
