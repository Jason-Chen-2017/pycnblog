# 3.基本概念术语说明
## 3.1 图论概念
　　图论（Graph Theory）是数学的一个分支学科，用来研究对象之间的关系及其结构，是一种抽象的结构化的方法。在图论中，图是由若干节点和连接这些节点的边所构成的。图论中的节点称为顶点（Vertex），边称为弧（Arc）。图论的关键问题是研究对象的分布特征以及对象间的联系。例如，在社会网络分析（Social Network Analysis，SNA）中，图论可用来描述网络中的社交关系，以及研究用户之间的行为习惯等；在物流系统建模、网络安全、生态系统建模等方面，图论也扮演着重要角色。
## 3.2 约束满足问题（Constraint Satisfaction Problem，CSP）
　　约束满足问题（Constraint Satisfaction Problem，CSP）是一种二维离散型规划问题。给定一组变量集合V和一组目标函数F={f1, f2,..., fn}，以及一组约束条件C={c1, c2,..., cm}，其中每个约束条件c=({X}, {Y})，X和Y是两个或多个变量的集合，则约束满足问题是要找出一种变量赋值方案，满足所有约束条件且使目标函数的总和最大。约束满足问题属于NP完全问题，属于最优化问题。
## 3.3 基因编码
　　基因编码（Gene Coding）是指把生物信息转化为数字信号的过程。通常情况下，基因编码可以分为显性编码和隐性编码两种。显性编码指的是使用一组固定的符号来代表某种生物特征，例如，人类的DNA基因编码是由四种碱基组成，分别对应不同种类和状态的核苷酸。隐性编码指的是直接使用生物特征的某些属性来表示编码信息，例如，Protein Protein Interaction (PPI)就是一种隐性编码方式，可以把蛋白质之间的亲和力关系作为编码依据。
## 3.4 子图搜索
　　子图搜索（Subgraph Search）是一种图论的算法，用于从一个图中查找一个与另一个图同构的子图。通常来说，子图搜索有着广泛的应用，包括文献检索、图像识别、医疗保健、生态系统建模、网络划分、图数据库索引等。子图搜索算法一般分为两步：第一步，构造子图匹配（Matching）问题，找到具有相同结点集的两个图之间的匹配；第二步，判断是否真正存在这样的匹配，如果存在，就可以认为这一对图同构。
## 3.5 QUBO模型
　　QUBO（Quadratic Unconstrained Binary Optimization）模型是一种用来表示布尔型变量的二次规划问题。在QUBO模型中，变量X的取值为0或1，目标函数为∑xi*xj，xi*yj表示xi和yj两变量的权重，上述目标函数表示了两个变量之间的关系，其值越大，则该变量之间的关系越紧密。由于该模型的限制，所以只能求解整数规划问题。不过，它却可以转化成许多有用的问题，例如，它可以转化成线性规划问题，这在很多场景下很有用。QUBO模型可以和很多图论、约束满足、二进制优化、数值优化以及统计学习等领域有关。
