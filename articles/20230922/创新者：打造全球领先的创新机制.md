# 1.简介
  

创新是指对已有的某些东西进行改良、更新、提升或者创造出新的产品或服务的能力。近年来，随着互联网的飞速发展，科技已经成为每个人的必备品。无论是手机、平板电脑还是笔记本电脑，手机和电脑一样，都是要靠创新才能与时代共舞，创新就是生产力的源泉之一。

如何用创新机制推动社会变革是一个难题，特别是在面临快速变化的国际形势下。科技界人士普遍认为，创新机制不仅仅是个形式，更重要的是使得创新的过程能够在整个社会中获得广泛认同并传播开来，推动社会变革。

“创新者”是新兴创新企业的代理人，他们掌控着产品、策略和流程，并扮演着重要角色。早期的创新者有机构派遣人员，后来慢慢演变成个人，逐渐拥有自己的办公室、研究室、实验室、平台，并获得更多资源和支持。

围绕这一主题，今年刊登在MIT Technology Review杂志上的一篇文章——《Creating a world-class innovation mechanism》——试图通过阐述创新者的作用以及其背后的机制——创新市场的驱动、创新企业发展的机制、创新者资格的评估标准、创新者激励机制等，探讨创新机制如何帮助社会变革，以及如何建立一个世界一流的创新机制。

文章从创新者的产生背景出发，强调了两种创新机制的不同功能，即激励性创新机制（Incentivizing Innovation）和引导性创新机制（Guiding Innovation）。然后分析了现行的激励性创新机制，特别是在美国的威斯康辛大学，它主要包括奖金、奖励和诺贝尔奖、博士和硕士学位的认证、高额福利以及社交奖励等。还分析了引导性创新机制，如科技集聚中心（Silicon Valley Accelerator），由加州大学伯克利分校的哈佛商学院开发的一种创新孵化器，旨在促进创新企业的创新进程。最后，文章提出了未来的创新发展方向——把创新机制扩展到全球。
# 2.基本概念术语说明
## 2.1 创新市场
创新市场是指支持企业、组织和个人探索创新产品和服务的地方。现阶段，创新市场主要分为两种类型：
1. 大众创新市场（Massive Open Online Courses）：这种类型的创新市场主要是采用网络虚拟学习方式，以满足年轻人对新知识和新技术的需求。
2. 普适创新市场（Commons-Based Peer Production）：这种类型的创新市场鼓励个人、团体或公司之间共享产品和服务。

创新市场与实体经济、信息技术、消费经济、产业链等领域紧密相连，也与日益发达的科技创新力量和创新驱动力息息相关。
## 2.2 创新企业
创新企业是指采用创新机制加快产品和服务向前发展的企业。创新企业通常有三种形态：
1. 自主创新企业：此类企业将自身的创新产品作为核心业务来实现，并拥有独立的研发部门。
2. 外包创新企业：此类企业将其部分研发任务委托给第三方，而这部分研发工作的产物则成为核心业务。
3. 联合创新企业：此类企业将多个创新项目进行联合开发，共同解决一项或多项突出问题。

创新企业由创始人创立，其产品和服务向社会提供价值。
## 2.3 创新者
创新者是指在创新市场上具有一定实践经验并且能够推动创新企业在市场竞争中获得成功的人。创新者往往具有较高的职业素养、洞察力和创新意识，能够发现问题并提出创新建议，因此其成功概率比其他创新者高很多。
## 2.4 创新能力
创新能力是指对新产品或服务提出的看法及解决方案、产品设计、管理模式、运营方法、服务质量的能力。其特征有：
1. 突出创造性：创新能力能够超越常规的生产和技术方式，为产品或服务提供独具匠心的创新创意。
2. 深入分析能力：创新能力能够理解客户需求、市场环境和竞争对手的优缺点，为产品或服务制定有效的创新策略。
3. 业务敏锐度：创新能力能够洞察市场的最新动态和市场结构，根据实际情况调整产品或服务的发展计划。
## 2.5 创新市场的驱动因素
创新市场的驱动因素可以分为三个层次：
1. 政策经济因素：政府决定如何吸纳新思想和创新。
2. 科技创新推动力因素：新技术、新材料、新工艺催生了技术革命、产业革命，迫切需要新思维、新技术的提出和验证。
3. 人才创新潜力因素：创新者的培养、培训以及人才的积累，会产生新的创新潜力。
## 2.6 影响创新能力的关键因素
1. 产品创新：产品创新促进了产品升级换代的循环，而带来了创新能力的提升。
2. 服务创新：服务创新使得客户满意度得到提高，同时也提高了创新能力。
3. 管理创新：管理创新更关注企业内部的控制和协调，通过创新管理实现目标。
4. 研发创新：研发创新赋予企业高度的灵活性和创新能力，驱动其创新发展。
# 3.创新企业的发展机制
## 3.1 创新企业定义
创新企业是一个企业，其产品和服务正在向社会提供价值，但其创新采用的是创新机制。如果某个企业处于创新状态，那么该企业就属于创新企业。
## 3.2 创新企业的目标
创新企业应该致力于为客户创造价值的同时保持竞争力，让自己保持不断创新、优化进步的姿态。创新企业的目标可以概括为以下四个方面：
1. 提供独具创意和独特性的产品或服务：产品或服务的价值取决于其独特性，只有打破当前的瓶颈，才能达到更高水平的创新。
2. 在短时间内获得可观的利润：创新企业要在短期内取得可观的利润，否则很容易被竞争对手超越。
3. 占据主导地位的地位：为了保持竞争力，创新企业必须取得一定的声望和地位。
4. 有持续创新的勇气：创新企业需要充满韧性，不断突破，保持创新意识和创新能力。
## 3.3 创新企业的起步阶段
创新企业的起步阶段可以分为以下四个阶段：
1. 原始试错阶段：创始人在第一天就开始着手实验，尝试各种新产品、新服务。
2. 技术试验阶段：创始人能够运用自己的专长进行系统的试验。
3. 产品推出阶段：进入销售之前，创始人已经完成了一些功能测试、市场调查。
4. 流通领航阶段：流通领航阶段通常是创新企业的黄金期，在此期间企业能够积累足够的用户、订单等数据，以便打开新的市场。
## 3.4 创新企业的成长阶段
创新企业的成长阶段一般包括四个阶段：
1. 市场开发阶段：在市场开发阶段，创新企业要通过改善产品和服务，找到自己的定位，进行市场营销活动。
2. 企业自我完善阶段：企业自我完善阶段是创新企业最为重要的一个阶段，创始人或团队都会面临很多挑战。
3. 增长领航阶段：企业进入增长阶段，增长领航阶段是创新企业的核心。在此阶段，企业会通过扩大规模，提升竞争力，取得更大的成功。
4. 收购拆并阶段：最后，企业必须在收购拆并阶段考虑合并或分拆。通过合并或分拆，企业可以实现盈利的最大化，并获得巨大的声誉。
## 3.5 创新企业的营运模式
创新企业的营运模式主要包括两种：
1. 自主营运模式：这种模式意味着企业的所有权和运营权都由创始人或其创始团队所有。
2. 外部投资营运模式：这种模式意味着企业的所有权和运营权都委托给外部的投资人。
## 3.6 创新市场的角色与责任
创新市场的角色分为两个：
1. 主动创建创新环境：创新市场的主导者在这项活动中扮演着举足轻重的角色，要么倡议增加科技创新环境的营造，要么制定新的政策或法律，推动科技创新环境的建设。
2. 对创新环境负责：在创新市场中，每一个企业都应当对自己的行为负责，通过履行职责，为创新环境做出贡献。
# 4.创新者的作用和作用机制
## 4.1 创新者的产生背景
创新者的产生背景有两种类型：
1. 工作型创新者：工作型创新者主要是指拥有丰富的科技经历和工作经验的技术人才，是创新者的主流。他们通过研究和总结经验教训，创造出独特的技术产品。
2. 学习型创新者：学习型创new_entersr用户，首先要学会面对新事物和挑战。他们具备较强的逻辑思维能力、解决问题的能力，是创新的导师、分析家和睿智的助手。
## 4.2 创新者的发展阶段
创新者的发展阶段可以分为五个阶段：
1. 学生创新者：学生创新者刚刚接触到创新这个领域，可能会由于没有经验导致“惰性”。他们需要时间去接受和理解创新的理念，并培养自己的创新能力。
2. 成熟创新者：成熟创新者在接受过教育之后，已经掌握了创新所需的技能，能够提炼经验，形成独特的理念和创新能力。
3. 经验型创新者：经验型创新者是在创业、职业生涯的早期加入创新环境的人，他们已经掌握了很多技能，能够沉浸其中，看到创新的希望。
4. 中级创新者：中级创新者能够在一定的创新氛围中进行创新活动。他们的创新能力一般来说比较弱，但是对于少量的突破或改进，却十分敏锐。
5. 领军创新者：领军创新者深谙创新领域的规则，能够准确识别和利用创新机会。他们的创新能力极强，经常需要创新来突破既有问题。
## 4.3 创新者的目标
创新者的目标有两个：
1. 科技创新者的目标：科技创新者的目标是发现新产品或服务，提升创新能力，为公司或组织创造新的价值。
2. 员工创新者的目标：员工创新者的目标是为员工提供工作更加轻松的环境，提升员工的工作效率。
## 4.4 创新者的能力
创新者的能力主要有以下几种：
1. 潜能能力：创新者的潜能能力来源于个人的发明创造能力、分析能力、执行力、创新精神和组织领导能力等。
2. 聪明才智：聪明才智是创新者面对复杂环境时，对自我的掌握能力，不能缺少的必要条件。
3. 创意和表达能力：创新者的创意能力是重要的，能够产生独特的想法、设计、风格和艺术品。
4. 职业技能：创新者的技能是员工的职业素养，需要按照自己的知识和经验提升自我。
5. 团队合作能力：创新者需要通过团队合作提升自己，并且有能力提升创新能力。
## 4.5 创新者的激励机制
创新者的激励机制主要有两种：
1. 垂直激励机制：垂直激励机制是指基于个人能力的薪酬激励。如年终奖金、月度绩效奖励、年度奖金、年度股票发放等。
2. 水平激励机制：水平激励机制是指基于创新者整体能力的激励。如获得奖牌的数量、提名的次数、参赛的比赛、个人的书籍获奖、荣誉的授予等。
## 4.6 创新者资格的评估标准
创新者资格的评估标准有如下几个方面：
1. 创新能力：创新者必须具备一定的创新能力，可以说是最重要的一环。
2. 企业经验：创新者必须有丰富的企业经验，有能力理解客户需求、竞争对手、市场环境等。
3. 管理技能：创新者必须具有领导能力、战略思维能力和执行力，可以领导团队成员达成目标。
4. 文化素养：创新者必须具有良好的文化素养，其思想观念能够激发团队成员的创新精神。
## 4.7 创新者激励机制
创新者激励机制分为两大类：
1. 制度激励机制：制度激励机制是指企业为创新者设定相应的激励，以鼓励或引导其创新。如员工参加创业训练、给予创新礼遇、晋升晋薪、带动新人进来等。
2. 品质激励机制：品质激励机制是指创新者在项目、产品、过程和结果上表现出色，获得额外的奖励或奖牌。如展示自己的产品，或代表企业赢得金融奖、物联网奖等。
