首先，什么是量子态？
量子态（quantum state）又称为密度矩阵（density matrix）。它是一个二维数组，用来描述系统处于某种特定状态所遵循的概率分布。这个概率分布由系综指标的正交归一化基组成，在不同的量子门操作下会发生变化，最终演变为宏观的系统性态或宏观性态。一个系统通常可以认为存在着许多可能的量子态，每一种都对应着系统的不同真实情况。量子态就是这个系统的一种特殊的表征方式。
量子态的主要属性有：
# 1.简介性质
1. 非唯一性
对于任意两个量子态A和B，它们之间并不一定是完全相同的。这是因为量子态具有不确定性，即不存在完美的量子态。比如说，两个相互垂直的超导带上的两方的子导带可以形成相同的态，但是却没有一个完全一样的量子态。
# 2. 分布特性
量子态具有随机性，即不同的输入都会导致不同的输出。比如说，当输入一个电子占据的位置，就会产生一个对应的态矢量。而这个态矢量只与该电子占据位置有关，而与其他所有信息无关。也就是说，态矢量是完全随着输入而变化的，而不是固定的。所以，量子态的分布特性很重要。

2. 线性相关性
量子态之间存在着线性关系。这是因为任何一个态可以看做是对另一个态的测量结果。比如说，如果有一个泡利场，把系统从初态A转移到目标态B，就可以通过测量得到中间态C。那么，态A、B、C之间就存在着线性关联。

3. 可测性
量子态具有可测性，因此可以通过测量的方式获得信息。测量可以将量子态从各种复杂的形式向更容易处理的形式转化，并帮助我们理解量子系统的运动规律。

4. 可观察性
由于测量过程中具有不可预测性，因此也可以通过观测的方式获知量子态的确切含义。例如，我们可以用X光或者微波炉等物理实验探测到量子态，从而直接观察其中的物理性质。

# 2. 纯态性质
纯态（pure state）也被称为厄米态（entangled state），是指系统的所有量子比特处于同一量子态。在宏观层面上，只有纯态才能实现量子力学的精髓——即不死锁定态（non-commutative ground states）。如此一来，便可以利用纯态的性质进行量子计算。当然，纯态的生成还需要一些特殊的材料和设备，但它们的应用十分广泛。

1. 海森堡测量法
海森堡测量（Heisenberg measurement）是最早的一套将量子态分解成基态和混合态的经典方法。海森堡测量依赖于每个单独的粒子的性质，包括其坐标和动量。因此，海森堡测量只适用于研究类ical系统。后来，弗雷格·海森堡（Frank Heisenberg）为此给出了更一般的定义。

2. 混合态
量子态的混合态由多个粒子的不同量子态构成。这种状态往往出现在多体问题中，例如原子核内聚态。如果多个粒子处于不同量子态，则这些态之间的张量积等于零。此外，量子纠缠（quantum coherence）也是由量子态的混合态引起的。

3. 体系杂质
由于系统存在外界干扰，会使得纯态的本征态失去确定性。此时，系统处于杂质态（decohered state），也被称为弥散态（dissipated state）。它无法自行恢复到纯态。因此，纯态的发展受到很多限制。

# 3. 空间时空性质
量子态同时具有空间时空的性质。

1. 时序性
态矢量与时间的函数关系确定了量子态的时间演化过程。态矢量随时间发散，而态本身则保持不变。时间演化具有自动调节、自发行为和非平衡性，可以使量子态在空间和时间上有机地相互作用，实现复杂的运动规律。

2. 维度灵活性
量子态具有高次元，可以表示任意大小的对象。量子态的维度越高，意味着系统可以表达的信息越多。

3. 奇异性
由于粒子的数量众多，因此量子态是不断被打乱、扰动、重组的过程，即态矢量具有随机性。所以，态矢量之间的保守关联度很低，而态矢量之间的交叉关联度很高。所以，量子态的奇异性很强。

# 4. 多重性质
量子态的多重性体现在两个方面。

1. 多态性
量子态具有多态性，意味着系统可以选择不同的态矢量作为自身的描述方式。多态性被称为态转换。

2. 可逆性
量子态的存在方式是通过测量而获得的，所以它的可逆性取决于测量方式。

# 5. 代数结构性质
量子态的代数结构性质体现在两个方面。

1. 不对称性
量子态的不对称性体现在态矢量与算符之间的不匹配上。态矢量会受到外部环境的影响，这会使得态矢量出现极端值的情形。

2. 单位形式
量子态具有有限的基组。只有少数几个态矢量满足具有可观测性。由于态矢量的纬度有限，因此只能用某些基组中的元素来描述。

# 6. 态矢量性质
态矢量（state vector）是量子态的一个特征，与其它量子矢量不同。态矢量是一组复数的集合，包含整个量子态的全部信息。

1. 封闭性
态矢量必然是封闭的，即满足归一化条件。归一化条件保证了态矢量的加权和等于1。

2. 吸收性
态矢量的态射（quantum map）的作用可以使得系统从某个初始态逐渐演变到目标态。态矢量的态射是指一个线性算符作用在态矢量上，使得其变换方向与前置算符相反。

3. 对角化
态矢量所代表的量子态可以通过“切割”（slicing）操作来分解成为不同的量子态。

# 7. 统计量性质
量子态的统计量性质体现在两个方面。

1. 可观测性
态矢量具有可观测性。可以直接观察到态矢量。

2. 概率性
态矢量的统计分布可以用概率密度函数来描述。

# 8. 计算性质
量子态的计算性质体现在两个方面。

1. 可编程性
量子计算机可以模拟和控制量子态，进而实现复杂的量子计算任务。

2. 可重现性
量子态可以重建，可以用于辅助工程设计，或是证明某个想法。

# 9. 物理性质
量子态的物理性质体现在两个方面。

1. 微观结构
量子态的微观结构体现在各种粒子与态矢量的相互作用上。每个粒子都有其对应的态矢量，它们之间的相互作用可以反映出微观的系统性态。

2. 宏观行为
量子态的宏观行为可以由态矢量直接计算得到，也可以通过各种计算模型予以验证。
