# 1.简介
  

## 1.1 起源
Lua是一门小巧灵活的脚本语言。它由巴西里约热内卢天主教大学(PUC-Rio)里的一个研究小组开发出来，被称为“脚本化编程语言”。1993年，Lua诞生在一个叫作Moses Laird的人的实验室中。他本人也是一位伟大的科技活动家、对游戏编程感兴趣的人。为了鼓励编程者参与到科研工作中，Laird带领了一批人把LUA加入了自己的研究项目。后来，Laird把LUA开源并以GPL协议发布。由于其简单易用、容易学习的特性，Lua成为了一种非常流行的脚本语言。

## 1.2 发展历程
Lua从最初作为一种个人使用的脚本语言被开发出来，到现在已成为世界上最流行的脚本语言之一。到今天，Lua已经成长为一门通用的高级编程语言，既有功能丰富的标准库，又有简单但功能强大的第三方库支持。

### 1.2.1 Lua 5.1
Lua 5.1版本于2012年1月22日发布，是一个快速迭代的版本，将发布周期缩短到了每两周一次，并且实现了与历史上其他版本最大的变化——完全向下兼容。

### 1.2.2 Lua 5.2
Lua 5.2版本于2012年8月7日发布，主要改进内容如下：

1. 增加了错误处理机制，允许用户自己定义错误类型；
2. 支持UTF-8编码；
3. 添加了新的元方法__ipairs()，用于遍历迭代器；
4. 扩展了字符串的处理函数，例如查找子串、替换等；
5. 添加了table.move()函数，可以移动数组元素；
6. 提升性能；
7. 修复了一些bug。

### 1.2.3 Lua 5.3
Lua 5.3版本于2014年6月30日发布，主要改进内容如下：

1. 优化语法，新增了尾递归、局部变量声明方式、goto语句；
2. 增加协程，使得Lua具有异步IO的能力；
3. 引入LuaJIT，提升Lua性能；
4. 修复了一些bug。

## 1.3 为什么要创建Lua？
在Lua出现之前，有不少脚本语言出现过，比如Perl、Python、Ruby、JavaScript。其中Perl最早应用于web开发，而Python和Ruby则被认为是最接近计算机科学和工程领域的编程语言。然而，这些脚本语言都有自己的一些缺点，比如不能像面向对象编程一样提供面向数据结构的语法糖、运行效率低、没有自动内存管理等等。因此，Laird开发了Lua，希望能通过一种精简的脚本语言解决这些问题。

## 1.4 特色和优势
### 1.4.1 简单性
Lua采用简洁的语法，而且编译速度也非常快，这使得它可以在较短的时间内完成复杂任务。这与Java或者C++相比，让它更适合用于编写系统软件。

### 1.4.2 可移植性
Lua不需要特殊的编译工具即可运行，这使得它可以很方便地移植到各种平台上，包括Windows、Linux、Mac OS X等。

### 1.4.3 动态类型
Lua不要求类型定义，任何变量都可以当作一个值来使用，因此动态类型可以节省内存并提高运行效率。

### 1.4.4 函数式编程
Lua提供了丰富的函数式编程特性，包括高阶函数、闭包、匿名函数、协同程序等。

### 1.4.5 多线程支持
Lua自带的协程机制可以轻松实现多线程编程，而且几乎没有性能损失。

### 1.4.6 可嵌入
Lua提供一个简单的虚拟机接口，可以用于嵌入各种应用程序，如游戏引擎、数据库驱动程序等。

### 1.4.7 免费
Lua的源码完全公开，免费下载和使用，并且其源代码文件格式比较简单，对新手学习lua有帮助。
