# 1.简介
  

超声波是一种低成本、高速传播、广泛应用于航天领域的无线通信方式。它可以用于各种环境的监测，包括空气、水、土壤等。随着近几年航天器技术的飞速发展，超声技术也逐渐成为航天器的重要组成部分。目前，超声技术在航天科研中的应用已从原始的通过降落伞或者其它小型设备进行遥感数据的获取逐步演变为多种类型的应用。下面简要介绍一下不同类型超声技术的特点及其应用场景。
# （一）毫米波雷达（Microwave Radiometer，MWR）
毫米波雷达（Microwave Radar）是一种利用微波脉冲干扰的方式探测到的一种高分辨率、多普勒效应强的卫星电视、遥感卫星图像、海底环境探测的无线传感技术。它的工作原理就是产生较强的微电磁波，然后被大气层吸收后反射回到接收器上，被测量对象反射回来的微电磁波可以使接收器受到电磁波的干扰而转化成脉冲信号，通过接收这些脉冲信号并处理得到探测数据。毫米波雷达具有良好的抗干扰性、适应性强、拓扑广、部署便利等优点。MWR主要应用场景包括高精度的地下检测、潜在探测及高分辨率的海底地貌测量等。
MWR产品价格昂贵，但其带宽高、动态范围宽、灵敏度高，在航空航天领域已经成为一种标准技术。由于近年来航空航天信息技术快速发展，MWR已经成为航空航天领域的标配技术。同时，MWR还可用于高性能卫星和空间探测领域。
# （二）载波相控阵雷达（Doppler Sonar Radar，DSR）
载波相控阵雷达（Doppler sonar radar，DSR）是一种基于发射超声波探测和接收微波探测相互作用，对目标物体进行高速度移动、方向判断的无线通信技术。其原理是通过调制发射机和接收机之间的速度差，将激光信号转换为超声波信号并反射到目标物体上，经由目标物体上的导电材料反射回接收器上。通过接收超声波信号并进行测量获得目标物体运动状态的位置。DSR具有良好的抗干扰性、灵敏度高、可穿透薄膜介质、简单、经济、部署方便等特点。DSR应用场景主要包括航空、陆基、军事等领域。
DSR产品价格昂贵，但其部署安装容易、成本低、制造简单、电源设计合理，在航空航天领域已经成为一种成熟技术。同时，DSR也可以用于核爆、潜艇等特殊应用场景。
# （三）红外光电传感器（Infrared Light-Emitting Diode Sensor，ILD）
红外光电传感器（Infrared Light-Emitting Diode，ILD）是一种利用红外光束发射、接收、处理的无线传感技术。它的原理是在发射端（或辅助装置）通过LED发出红外光束，在接收端（或测量装置）接收该光束并转化成电信号。这种传感技术使用户能够实时获取、分析变化的物理环境，并得出精准的测量结果。ILD具有高灵敏度、成本低、易于部署、广泛应用的特点。ILD应用场景主要包括低功耗、高频、对地测量、环境规划、水文测量等。
ILD产品价格不菲，但其安装、使用容易、成本低、可靠性高，已经在航空航天领域广泛应用。
# （四）超声激光导航系统（Ultrasonic Laser Navigation System，ULS）
超声激光导航系统（Ultrasonic Laser Navigation Syste，ULS）是一种利用超声波探测和激光雷达定位能力，对自然界中的目标进行准确的无线定位的航空导航技术。其原理是通过装置内的探测系统对周围的环境进行扫描，将探测结果转换成红外激光信号，通过激光探测雷达将激光信号反射回发射机上，经由发射机上的激光束进行接收，将激光信号转换为超声波信号，在接收器上接收到超声波信号，在基础系统中进行计算，并用坐标系确定目标的位置。ULS具有良好的定位精度、精密度、高度灵活、简单、便携性好、部署易、成本低等特点。ULS应用场景主要包括固定导弹基地、军事侦察、民用无人机、环球飞行器等。
ULS产品价格昂贵，但其体积小巧、精密度高、功能丰富、耐受性强，已经成为航空航天领域的主流技术。虽然它的性能远不及其他技术，但仍然是各个领域的通用技术。
# （五）主动式超声雷达（Active Ultrasonic Radar，AUR）
主动式超声雷达（Active Ultrasonic Radar，AUR）是一种利用激光激光器发射、超声波反射、微电子探测相互作用的无线通信技术。其原理是将目标对象的声音转换为电磁波，通过无线电干扰，激光探测雷达通过激光线束探测到目标，将激光信号反射到发射端，激光探测雷达再通过超声波对发射机上接收到的超声波信号进行探测，最终通过微电子探测装置对发射到目标的超声波进行检测。AUR具有良好的隔离性、抗干扰性、高度灵活、精度高、价格低等特点。AUR应用场景主要包括对非线性雷击伤害、环境恶化建筑物、潜在物资保护等。
AUR产品价格不菲，但其安装、使用难度大、成本高、可靠性低，目前尚未得到广泛应用。
# （六）光学式激光测距仪（Laser Ranging Instrument，LRInst）
光学式激光测距仪（Laser Ranging Instrument，LRInst）是一种利用激光探测雷达、传感器、微电子控制元件等设备，对目标物体进行距离测量和三维定位的航空航天无线传感技术。其原理是先对目标物体进行激光散射，激光探测雷达对散射的激光信号进行接收，接收到信号的激光探测雷达通过计算方位角、偏移角等参数进行测距，在计算后的距离上绘制实时三维坐标图。LRInst具有成本低、适应性强、轻便、便于维护、可靠性高、精度高、价格便宜等特点。LRInst应用场景主要包括空中侦察、轰炸目标识别、地面工程测量、物体移动测距、环境监测、机器人导航、智能输送等。
LRInst产品价格便宜，但其成本高、可靠性低、耗能大、灵敏度差、实用性差，尚处于实验阶段。
综上所述，超声技术在航天科研中的应用各有特点。随着时间的推移，超声技术在航天科研中的应用会逐渐演进，发展出更加先进的应用技术。此外，随着自动驾驶汽车、火箭、卫星等航天器的发展，超声技术也会成为航天器的必备技术。因此，了解和掌握超声技术的基本知识、应用场景、基本原理，对于对接航天科研、提升航天器性能、扩展应用边界非常重要。
