# 1.简介
  

今天，社会上充斥着各种“一时之命、二用之年”、“命中注定、运气来”，然而现实世界却越来越不靠谱。实际上，每个人的生活都存在着很多不可预测的因素，而每当有事情发生，我们也很难通过直觉来抉择应该如何应对，只有“面对现实，勇敢选择，行动起来”才可能实现目标。

因此，作为人类，在面对种种困难的时候，我们需要认识到理性，保持冷静，做出正确的判断，并且适时的调整自己的行为，从而实现更好的结果。

但是，在面对复杂多变的时间管理，面对快速变化的需求，人们往往会陷入“我想过了吗？”、“要不要试一下？”等疑惑的境地。

在这种情况下，如果我们能够把时间管理当作一个能力来养成，那么就可以更好地统筹计划与实施，脚踏实地解决实际问题，同时摒弃一些看似合理、实用，但其实错误的内容。

因此，这次笔者将尝试以亲身经历和作者个人见解的方式，向大家展示如何理性面对时间紧迫而又没有方向感。

# 2.关于时间管理

首先，先简单介绍下什么是时间管理。

时间管理（Time Management）是指人们在日常工作、学习、生活中的时间分配，其目的就是为了达到最大化利用资源和完成工作任务的效率，并避免造成资源浪费，提高效益和幸福感。它既包括时间观念上的指导，也包含具体的时间安排方法及技巧。

简单来说，时间管理就是通过掌握有效的时间分配手段来完成任务并获得最佳效果的过程。

一般来说，时间管理可以分为以下几个方面：

1.管理时间：负责管理自己的时间，合理安排自己的时间分配；
2.节约时间：为工作创造条件，减少不必要的浪费，减少恐慌心理；
3.控制欲望：调节自己的欲望，从而增加工作效率和满足感；
4.重视重要事项：把时间花费在最重要的事情上，强化自信心和成就感。

# 3.时间管理的原则

1.保持专注力：专注于一件事情上，保证集中注意力，是时间管理的关键。
2.分清轻重缓急：优先考虑重要、紧急的事务；把握轻松、缓慢的事情；根据情况适时调整进度。
3.实践证明准确：习惯是建立起长久坚持的基础，实践、分析、检验才能得出结论。
4.树立远大目标：以终为始，突破局限，努力拓宽眼界。
5.知识和技能是本钱：培养自己核心能力，开拓新的知识技能，找到工作中真正值得投入的地方。

# 4.如何合理安排时间

那么如何合理安排时间呢？简单的说，就是按时分配精力、积极创新，并控制焦虑、放松。

- 把控好每天的时间：按照日程安排，把自己时间所剩无几的时间用来合理安排工作。
- 给予充裕的时间：充分利用空闲的时间，调整工作、学习节奏，不应总是在办公室时间冲刺，保证充足的休息时间。
- 分清轻重缓急：对重要、紧急的事务采取重点关注；积极主动承担轻松、缓慢的任务，给予适当的缓冲。
- 处理完当下的任务后，再处理下一个任务：充分利用时间，不要拖延，一次只做一件事，按顺序排队执行。
- 提升自己的能力：提升自己的能力，提高生产力水平，培养良好的沟通技巧，养成解决问题的能力。

# 5.如何控制焦虑

那么如何控制焦虑呢？很简单，就是“头脑风暴”。

- 通过系统整理你的时间线：找出你最近的事件，回顾一下你过去一周或一个月的工作计划。
- 对每件事都进行系统分析：评估每一件事的重要性和紧迫性，设定计划，详细罗列每件事的做法、原因、结果。
- 将你的压力转移到自控力上：承认你现在的状况，并尝试着改变，从而克服焦虑。
- 以自己时间为主线：将时间花费在你真正重要的事情上，只关注你要解决的问题和事情。

# 6.如何放松

最后，还有一种放松的方法，就是“冥想”。

- 在整个晚上或日落之前，来一场静坐练功；
- 深呼吸，自我放松，释放压力；
- 换一个角度看待问题，重新审视自己；
- 想象一下白天的生活，和普通人比比看。
