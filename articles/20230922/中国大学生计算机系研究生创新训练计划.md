# 1.简介
  

中国计算机学会于2020年推出了中国大学生计算机系研究生创新训练计划（以下简称“计划”）。该计划由中国计算机学会、南京大学、复旦大学联合主办，面向全国高校的本科计算机专业学生开设。计划共分六个模块，分别为“开放实验室探索性学习”、“算法设计与分析”、“数字媒体安全应用”、“数据库系统及应用”、“自然语言处理”和“人工智能算法”。截止目前，计划已经有近700余名高校学生报名参加，已有的100余所高校提供了实习机会，研究生数量逾万。学校开设该计划意在培养具有科技突破能力、具有国际竞争力的高级计算机专业研究生，促进科研和产业发展，服务社会进步。

# 2. 基本概念与术语
## 2.1 模块介绍
### “开放实验室探索性学习”
开放实验室探索性学习模块，主要围绕计算机视觉、机器学习、自然语言处理等领域开展。课程包括知识提升班、实践指导课、创新创业训练营、竞赛训练、项目交流与评价等内容。学生将学习到更多有关工程实践的课程。具体内容如下：

1. Knowledge Improvement Program: 本课程为学生提供学习计算机基础知识的机会，教授相关的编程语言、数据结构、算法、计算机网络、操作系统、系统结构、数据库管理等知识。通过本课程，学生可以掌握编程技术的实际应用，提升计算机能力。学生可选修课堂讲座或实验，为接下来的学习奠定基础。

2. Practical Training Program: 本课程为学生提供基于实际场景的计算机相关技术的培训，包括计算机视觉、机器学习、自然语言处理、信息检索、数据可视化、数据库应用等技术。通过本课程，学生可以快速上手这些技术并在真实场景中进行实操应用，增强对计算机科学的理解。学生可选修课堂讲座或实验，为接下来的学习奠定基础。

3. Innovation and Entrepreneurship Training Program: 本课程由南京大学的产学研一干部倾情打造，邀请科技企业、创业者、学者共同分享创新的创业经验、产品开发思路及方法论。通过本课程，学生可以了解创业环境、从零开始创业、参与到科技创业中的关键环节，提升创业能力。学生可选修课堂讲座或实验，为接下来的学习奠定基础。

4. Competition Training Program: 本课程的目标是培养学生运用各类竞赛的能力，以更好的应对国内外竞赛。课程包括编程比赛、数据结构比赛、算法竞赛、机器学习竞赛、信息安全竞赛等内容。学生通过完成这些比赛，可以在竞赛中锻炼自己的解决问题能力、团队协作能力及沟通表达能力。学生可选修课堂讲座或实验，为接下来的学习奠定基础。

5. Project Exchange Program: 本课程为学生提供海量的计算机相关项目，让学生能够更充分地展示自己的能力。在课程结束后，学生可以选择其中一个或几个进行实习或项目，并且与项目导师一起讨论研究。

### “算法设计与分析”
算法设计与分析模块，课程设置基于数据结构、算法、编程技术及其应用，以培养学生运用计算机科学的理论基础和工程实践能力，掌握算法设计与分析的能力。课程的特点包括：

1. Data Structure: 本课程首先介绍数据结构的概念，包括数组、链表、栈、队列、树、图等。学生将通过实践，熟悉数据结构的实现和应用。

2. Algorithm Analysis: 本课程介绍算法的运行时间分析、最优算法与平均时间复杂度的计算，还会涉及到贪心算法、动态规划、回溯法、分支界限法、递归法、线性规划、随机化算法等算法。学生将学习到多种算法的基本知识，并通过实践操作，使自己能够独立编写出有效的算法。

3. Programming Technology Application: 本课程是应用计算机的实践性训练课程。课程中包括C/C++、Java、Python、Web开发、数据库、机器学习、图像处理、多线程、网络通信、缓存等内容。通过课程的学习，学生可以更好地理解计算机科学的应用场景，并运用所学的技术开发相应的应用。

### “数字媒体安全应用”
数字媒体安全应用模块，侧重于介绍移动互联网和物联网技术的最新发展、数字媒体安全技术的最新进展和应用。课程的特点包括：

1. Mobile Computing: 本课程通过对最新发布的手机操作系统、芯片、安全机制、加密算法、电信运营商等方面的介绍，介绍移动互联网的发展现状和安全隐患。学生将了解到移动互联网的架构设计、网络安全防护、安全漏洞防范、病毒攻击防控、产品安全认证等技术要素。

2. Internet of Things: 本课程介绍物联网的概念和发展，阐述物联网技术的诸多特性及应用场景。学生将了解到物联网的概念、架构设计、设备安全、数据采集、数据传输、数据存储、数据处理等技术要素。

3. Digital Media Security: 本课程是介绍数字媒体安全技术的最新发展、相关标准及技术。包括加密技术、数字水印、视频监控、数字 rights management、可信执行环境、边缘计算等内容。学生将了解到数字媒体安全的相关发展方向、技术难题和方案，并掌握相应的技术知识和工具。

### “数据库系统及应用”
数据库系统及应用模块，课程设置基于数据库系统的概念、结构、模型、查询语言、完整性约束、索引、事务等概念，以帮助学生构建数据库系统、管理数据库、应用数据库。课程的特点包括：

1. Database Concepts: 本课程介绍数据库的概念、模型及其功能，包括实体-关系模型、层次模型、对象模型、函数模型、规范化模型等。学生将学习数据库的定义、逻辑结构、物理结构、存取方法、查询语言、完整性约束、事务、并发控制等相关知识。

2. Database System Design: 本课程介绍数据库系统设计的一般流程、优化策略、性能分析方法、安全性与权限控制、故障恢复技术等。学生将学习数据库设计的方法、原则、工具、过程及缺陷、调试方法、容错技术等知识。

3. Management Tools: 本课程介绍数据库管理工具的安装部署、使用方法、统计信息、故障诊断、日志记录、备份恢复等方面的知识。学生将学习MySQL、MongoDB、Redis等数据库管理工具的安装部署、使用方法、命令行操作、客户端工具及连接方式等知识。

### “自然语言处理”
自然语言处理模块，课程设置基于概率语言模型、词法分析、句法分析、语义分析、生成模型、信息检索、文本分类、文本摘要、对话系统等技术，以帮助学生掌握自然语言处理的基本理论、技术、应用。课程的特点包括：

1. Probabilistic Language Model: 本课程介绍概率语言模型的基本概念、形式语言、评估准则及意义。学生将学习上下文无关语言模型、马尔可夫链蒙特卡洛方法、条件随机场方法、HMM-Viterbi算法等概率语言模型的基本原理及应用。

2. Lexical Analysis: 本课程介绍词法分析的过程及规则，包括识别单词、标识符、特殊符号、空白字符、注释等。学生将学习正则表达式、最大匹配算法、词汇表、停用词列表等词法分析的相关知识。

3. Syntactic Analysis: 本课程介绍句法分析的过程及规则，包括确定构成句子的成分、标识符的作用范围、语法规范、文法等。学生将学习Shift-Reduce解析器、Top-Down解析器、Bottom-Up解析器、CYK算法、预测分析机等句法分析的相关知识。

### “人工智能算法”
人工智能算法模块，课程设置基于概率图模型、聚类、异常检测、决策树、支持向量机、神经网络、遗传算法、遗传规划、模拟退火算法、梯度下降算法等技术，以帮助学生掌握人工智能算法的原理、理论、实现、应用。课程的特点包括：

1. Graphical Models: 本课程介绍概率图模型的基本概念、形式语言、图表示、参数学习、概率推断等。学生将学习有向图模型、无向图模型、马尔可夫网络、最大熵模型、条件随机场模型、贝叶斯网络等概率图模型的基本概念及应用。

2. Clustering: 本课程介绍聚类的基本概念及技术。学生将学习多种聚类算法、距离计算方法、层次聚类、凝聚型聚类、密度聚类、软聚类等聚类算法的基本理论。

3. Anomaly Detection: 本课程介绍异常检测的基本概念及技术。学生将学习基于分布的异常检测算法、基于样本的异常检测算法、神经网络方法、混合模型方法等异常检测方法的基本原理及应用。
