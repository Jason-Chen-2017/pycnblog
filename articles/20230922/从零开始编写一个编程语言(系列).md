# 1.简介
  

随着人们对新技术的追逐、对技术创新的需求，越来越多的人开始关注如何开发出自己的编程语言。本系列文章将尝试通过简单易懂的教程，带领读者建立自己的编程语言，并且介绍一些其中的关键技术细节。
系列文章共分为七篇：

1. 从零开始编写一个编程语言（上）——如何定义编程语言？
2. 从零开始编写一个编程语言（下）——词法分析、语法分析、运行时环境、虚拟机与解释器等方面都有哪些要素？
3. 从零开始编写一个编程语言（中）——通过自己实现一个简单的编程语言，深入理解计算机系统和编译原理相关知识
4. 从零开始编写一个编程语言（下）——让你的编程语言更加强大，除了语法解析之外，还可以做什么？
5. 从零开始编写一个编程语言（上）——如何选择适合作为第一门语言的设计？
6. 从零开始编写一个编程语言（中）——如何编写错误处理模块，以便于让用户看到友好的报错信息？
7. 从零开始编写一个编程语言（下）——结合编译技术和运行时环境构建自己的编程语言运行时环境。

每个文章会从头到尾详细地介绍每一步要完成的工作，并在必要的时候给出实际可用的代码示例。欢迎大家积极参与，一起努力开发更棒的编程语言！
