# 1.简介
  

近年来，随着DNA芯片的集成电路功能的不断增强，传感器、微电子设备等新兴设备逐渐成为了DNA芯片中重要组成部分。同时，研究者们也发现，不同颜色的DNA纤维在高场效应下可以形成具有不同属性的浓液晶体结构，这些具有高阻抗性、长寿命等特点的蛋白质核酸可以通过DNA相互作用结合实现双向信息传输。然而，目前的设计方法都是采用专用或小批量的构建流程，使得制造过程耗时长、效率低下，而且容易产生孔隙、延伸开口等难题。因此，本文提出了一种新的多色DNA纤维晶体对接方法——宽场定向釉膜（Wide Field Directed Osmium Mordants，WFODM）。该方法通过稳定的化学反应和化学力学调控，将多种不同的DNA纤维层嵌入DNA底座中形成具有特殊功能的多色DNA晶体。通过这种方式，可以在同一个设计流程中生成多种不同颜色的DNA晶体，无需特殊工具，快速、可靠地完成DNA集成电路的制造。
# 2.多色DNA晶体

多色DNA晶体指的是由多个不同的颜色DNA纤维组成的一种结构，这种结构具有高度的相关性和相互作用特性，能够实现复杂功能。在核苷酸合成领域，多色DNA晶体是一个颇具代表性的例子。在世界各地，不同的国家都开发出了多色DNA晶体，例如，中国开发出了基于色碱基多态性的水稻玉米淀粉作物纤维系，日本开发出了基于瑞氏多态性的玫瑰胚细胞纤维系。


多色DNA晶体的结构是由多个不同的颜色DNA纤维组成的，它们之间可以自由相互作用以合成不同大小的多级结构。根据多色DNA晶体的结构特性，分为两类：非相互作用型多色DNA晶体和相互作用型多色DNA晶体。其中，相互作用型多色DNA晶体主要包括二氧化硅（BGA）和铁氧化锂（ITO）等晶体，具有更高的功能生物学表达效率。


# 3.宽场定向釉膜的设计原理

宽场定向釉膜（Wide Field Directed Osmium Mordants，WFODM）是一种定向釉膜（Mordant），它能够在宽场条件下形成具有特定功能的多色DNA晶体。所谓宽场，就是指能通过光照转换而不受限制的空间范围，其典型值为几百公里到数万公里。根据WFODM的设计原理，在高场效应下的DNA纤维属于软链末端，并且具有较大的釉性，而在低场效应下的DNA纤维属于硬链末端，具有较小的釉性。因此，我们可以通过控制二氧化硅和铁氧化锂之间的相互作用来形成具有特定功能的多色DNA晶体。

## 3.1 化学反应

WFODM的核心组成部分是三种釉质形成酵母，分别为二氧化硅、铁氧化锂、和颜色DNA纤维。我们需要在这些釉质之间建立一套固定的化学反应，才能保证WFODM中的三种釉质正常地交流。

1.化学反应：DNA-BGA和DNA-ITO间的联结、退火

在低场效应下，低压情况下，DNA-BGA和DNA-ITO之间存在单边结合，当体积增大时，可能会导致温度过高。因此，低场下DNA-BGA和DNA-ITO间的化学反应需要做一些退火处理，让它们能够良好的结合。

2.BGA-DNA间的相互作用、聚合

BGA和DNA的相互作用主要是BGA纳米键与DNA核苷酸的相互作用。在WFODM的设计过程中，需要考虑到BGA和DNA之间形成多重键相互作用，并在相互作用之后进行化学反应，使得它们的键能多聚合。

3.DNA-ITO间的相互作用

DNA和ITO之间的相互作用，主要是DNA螺旋段与铁氧化锂的相互作用，目的是形成具有特定功能的多色DNA晶体。

## 3.2 化学力学调控

除了对釉质之间的交流和键相互作用进行退火外，还需要通过某些化学力学调控手段来优化WFODM的性能。这些手段包括以下两个方面：

一、DNA功能的选择

由于WFODM能够极大地扩大有效的颜色纯度，因此，每一种颜色的DNA纤维都可以作为WFODM的功能元素。因此，我们需要在优化WFODM性能的同时，尽可能选择多种不同的功能元素。

二、釉质的分布和位置

釉质的分布和位置需要根据釉质的特性及其在DNA结构中的位置来进行设计。对于铁氧化锂，它存在于链尾部，并且会与多种类型的DNA核苷酸发生相互作用；而对于二氧化硅，它的分布情况取决于RNA链的组织形式。因此，我们需要合理地分配和布局釉质，才能获得最佳的WFODM性能。

# 5.未来发展方向

基于WFODM的多色DNA晶体对接技术已经成为DNA芯片制造中的一个重要技术。虽然WFODM在技术上取得了一定的突破，但还有很多改进的空间。在未来的发展方向，我们可以尝试以下的方法：

1.无缝合成: 现有的WFODM是通过分离化学反应获得的，其结构具有高分解性，在高场条件下容易出现气泡。如何通过半自动化的系统将WFODM在海拔高度下进行无缝合成，将为WFODM的发展提供更多的机遇。
2.物理结构的改进: 目前的WFODM主要以铁氧化锂和二氧化硅作为釉质形成酵母。但是，这样的纤维结构不利于在高场下进行密封，在低场下纤维的厚度会影响结构的性能。如何通过设计具有更加紧凑结构的釉质，并引入更多的弹性结构来提高WFODM的性能，将是我们在设计方面的另一个挑战。
3.DNA结构的改善: 在当前的设计中，多色DNA纤维具有自由头的特性，即可以单独或者配合其他不同的颜色DNA纤维嵌入。在实际应用中，这一特性往往会引起DNA互相干扰，影响DNA的功能。如何设计具有更优异的DNA结构，既能够减少DNA互相干扰，又保留了多种DNA纤维的功能，是我们在设计原理方面的一个挑战。
