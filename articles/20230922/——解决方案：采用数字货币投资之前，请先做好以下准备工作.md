# 3.基本概念术语说明
## 3.1 概念
### 3.1.1 什么是数字货币？
数字货币（Digital Currency），也称“区块链”或“数字经济”，是指利用数字技术和电子计算机，通过互联网将传统的虚拟货币制作成一套自然而然的规则，使货币产生、流通、结算等过程自动化、透明化、高度集中化的新型支付系统。它与传统的纸币、硬币等贵金属货币不同，不需要国家或者中心化的金融管理机关批准，可以自由流通、随时汇率兑换，具有独特的支付属性和特性。

### 3.1.2 为什么要用数字货币？
数字货币可以满足广大的需求，主要是由于：

1. 降低交易费用：数字货币采用比特币和其他密码货币相似的工作机制，它可以在任何地方进行自由转账和交易，交易费用极低，远远小于现金和银行的交易费用。

2. 不受物理世界的限制：数字货币可以任意发送，无需受地理和时间上的限制。而传统的支付方式要么需要提前到达指定地点，要么需要等待货物到达才能支付。

3. 防止洗钱和监管：因为数字货币没有实体形态，不存在黑客入侵或者电子烟交易，而且可以确切记录所有交易历史，不会存在资金被偷或者洗劫的风险。同时，因为其去中心化的特性，数字货币虽然有可能被政府和国安机构窃取，但仍然可以通过许可证、法律手段等保护用户隐私和数据安全。

4. 公开透明：任何人都可以查看和验证数字货币的交易记录，从而促进了货币的公开、透明和合规性。

5. 可编程的代币：数字货币可以使用各种编程语言和智能合约构建出不同的代币，可以用来代表各种价值，比如可以建立基于商品的代币来记录消费、消费能力、积分、游戏分数等；也可以建立公司、企业、政府部门的代币，比如股票、债券等。

6. 全球范围内流通：尽管目前只有少数国家拥有世界上规模最大的数字货币矿池，但只要有足够的算力，全球任何地方都可以快速生成新的数字货币，并通过国际互联网自由兑换。

### 3.1.3 数字货币的特点有哪些？
数字货币的一些重要特点如下：

1. 匿名性：数字货币不存储个人身份信息，交易双方完全对称，不存在交易方的名誉权。

2. 总量恒定：为了保持流通性，数字货币总量不会增减，相当于有限数量的黄金。

3. 没有中间商赚差价：交易双方直接进行交易，不存在中间商赚取差价的过程。

4. 最小单位：一个数字货币可以由最基本的“笔”组成，而不存在大小或数量上的差异。

5. 透明度高：数字货币的发行、流通、结算都是公开透明的。

6. 无记账权：数字货币不依赖于任何一个实体的信誉或法律，任何人都可以验证交易的真伪。

## 3.2 投资建议
### 3.2.1 投资门槛
目前，采用数字货币投资有一定的门槛要求，一般从以下几方面考虑：

1. 投资额度：数字货币交易所通常会设定最低起购金额和最高限额，超过这个额度的用户无法正常交易。

2. 资金成本：数字货币交易所的注册资金要求较高，相当于银行存款，也有成本维持运营的成本。

3. 投资周期：数字货币交易所的成立时间一般都比较长，常见的持币上线周期在一年以上，同时还存在较多的套利空间。

4. 买卖门槛：数字货币的购买和销售门槛较低，任何人都可以进行自由的数字货币交易。

### 3.2.2 风险提示
数字货币投资具有较高的风险，投资风险主要包括以下几个方面：

1. 炒币风险：数字货币的价格波动非常快，短时间内可能会出现价格回调。

2. 政策风险：数字货币具有去中心化的特性，国际局势变化频繁，政策也随之发生变化。

3. 法律风险：数字货币发行方并不遵守法律，可能存在反洗钱、反对贸易壁垒等法律风险。

4. 稳定性风险：数字货币属于波动性较强的加密数字资产，一直处于试探阶段，容易面临各种技术问题和突发事件导致的崩盘风险。

### 3.2.3 投资策略
对于初入数字货币投资的人，可以适当考虑以下策略：

1. 学习：掌握数字货币的知识和技能是成为专业的数字货币投资者不可或缺的一环。需要阅读数字货币的相关研究报告、分析文章、学术著作，搞清楚区块链、共识机制、POW/POS、侧链等相关概念和知识。

2. 掌握市场：数字货币的交易规模巨大，相关的平台遍布全球，需要选择适合自己投资目的和预期风险承受能力的平台，并了解相关的市场规则、活动以及各类交易对手的情况。

3. 观察分析：数字货币市场非常活跃，需要时刻关注市场动态，以及不同指标的变化趋势，以便做出适合自己的投资决策。

4. 交易胜率：数字货币具有相对高的交易成本，一旦错过了良好的时机可能会损失惨重。所以，对于熟练且对投资有自信的人士，适当降低交易金额和次数，实现更高的收益率。
