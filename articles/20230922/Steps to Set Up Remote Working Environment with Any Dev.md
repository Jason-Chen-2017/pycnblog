# 2.核心概念及术语
## 什么是远程工作？
远程工作（Remote working）是指工作从本地移动到远程执行的过程。企业通常会将人员派往某个特定地点，然后让他们到那里开办公室，或者将员工转移到附近的咖啡馆或别处。

## 什么是电脑？
电脑（Computer）是指能够按照指令进行自动化计算、存储和处理数据的机器。它包括主机（CPU）、主板、内存、硬盘、显示器、键盘、鼠标等零部件。

## 什么是互联网？
互联网（Internet）是由网络连接起来的全球性信息系统，所有人的个人计算机、手机、服务器、路由器都通过互联网相连。它是一个广泛使用的通信手段，也被称为万维网（WWW）。

## 什么是VPN？
虚拟专用网络（VPN）是指利用公共互联网之外的私密通道，建立安全可靠的远程连接。该连接允许用户通过加密的隧道发送和接收数据。

## 二、软件安装配置
### 安装 VirtualBox
VirtualBox 是一款开源虚拟机软件，可以通过创建虚拟机来实现远程办公环境。安装过程如下：

1. 前往 VirtualBox 的官方网站[https://www.virtualbox.org/]下载软件；
2. 将下载好的安装包上传至要安装 VirtualBox 的电脑上；
3. 使用记事本打开 VirtualBox_amd64.msi 文件；
4. 在弹出的窗口中点击“下一步”直到出现欢迎界面；
5. 点击“下一步”完成安装；

### 配置 VirtualBox 虚拟机
1. 创建新虚拟机：在 VirtualBox 的主页面左上角选择“新建”，或者按下快捷键“Ctrl+N”创建一个新的虚拟机；
2. 设置名称：输入虚拟机的名称，比如“远程办公桌面”；
3. 设置类型：选择“专用型”类型，以便于屏蔽掉除电脑外的所有外设；
4. 设置内存：根据自己的实际需求设置内存大小，建议最少分配8GB内存；
5. 磁盘映像：选择“创建新硬盘映像”，然后点击“下一步”。
6. 设置磁盘大小：选择“VDI（Virtual Disk Image）”文件格式，磁盘大小建议设置为50GB左右；
7. 设置创建位置：选择一个保存文件的路径，比如“D:\VBox VMs\”；
8. 镜像源选择：选择“使用现有的磁盘映像”并指定要使用的预先准备好的镜像文件；
9. 点击“创建”按钮完成创建。

### 配置 VPN
如果想要通过 VPN 来访问公司内部资源，可以参照以下操作：

1. 在 VPN 服务商官网注册账号；
2. 下载客户端软件，并按照其提示进行相关配置；
3. 使用 VPN 客户端登录 VPN 服务商；
4. 使用 VPN 后就可以访问公司内部资源了。

- Chrome Remote Desktop：一款开源软件，功能强大，能实现远程控制，还提供了多种配色方案；
- Windows Remote Desktop：微软推出的一款软件，只需安装软件后，即可实现远程控制；
- TeamView 5：微软推出的一款软件，功能类似于 TeamViewer，价格比 TeamViewer 低很多；
