# 1.简介
  

## 1.1 为什么要写这篇文章
最近很多朋友问到，“怎么才能更好的追踪自己的投资收益？”、“我每年的投入都很少，但总是不知道自己投了多少钱”，以及“应该怎样才可以做到持续性投资？”等问题。无论这些问题如何难以回答，如何来解决问题也是困扰每个人的共同需求。

本文即是对个人投资能力的测评和实践，通过原理说明及操作指导，希望能够帮助大家更好地了解和掌握自己的投资能力，使得自己在投资领域保持不断进步。
## 1.2 作者介绍
我是李建平，目前就职于腾讯公司。本文作者认为，每个人的投资方式都不同，有的偏爱长线投资，有的偏爱短期价值投资。但无论您选择哪种投资方式，都需要了解自己的投资逻辑和能力，并在合适的时候进行适当调整。

## 1.3 投资者之路漫漫
### 1978 年，在西雅图市中心的一处小区里，身穿浴衣、蓝色短袖连衣裙的二十八岁青年阿尔弗雷德·马丁·斯科特（Alphonse Machin）出生了。他天赋异禀，四肢灵活，热情洒脱，充满理想，生活颇富收入。然而，离开家乡之后的第一份工作并没有给他带来幸福快乐。

十多年前的某一天，阿尔弗雷德接到了邻居家的电话，邻居家正在举行葬礼，邀请邻居们来参加葬礼。阿尔弗雷德感慨万千，决定去参加。于是，他带着两个儿子和妻子，飞奔而至，抵达葬礼现场。那是一座古老的、采用新木材建造的建筑，破败不堪。邻居们都很忙碌，只好让他们等待。阿尔弗雷德看着眼前的景象，心想：如果不早点赶来，估计后果不堪设想。可是，又担心受伤，没有办法安慰他们。

两天后，阿尔弗雷德看到了邻居送来的鲜花，一个也没有捡起。他紧张万分，大声询问原因。邻居只好摆出一副沉默的表情，说：“没有捡到，因为之前都是在给其他家庭葬礼服务。”阿尔弗雷德听了，莞尔，这不是很奇怪吗？他觉得自己好像置身于商界，自视甚高。于是，他继续向邻居们索要道歉书，提醒他们把鲜花留给他们的孩子们。然而，邻居的态度却变得不耐烦了，频繁地打电话，要求阿尔弗雷德赶紧回家。

阿尔弗雷德沮丧地看着这个“麻烦”发生，又好像置身于陷阱之中，内心充满恐惧。他向妻子借了一把锤子，冲上前去，突然发现周围已堆满了碎纸、旧布，却没有别的东西可以用来砸碎这一切。他的手指肿胀，抓住锤子时，却一下子被碎片割伤。他愤怒地瘫倒在地，气喘吁吁。妻子把阿尔弗雷德的情况和他的母亲交代清楚，并赔偿了家用电器费用的损失。然而，阿尔弗雷德仔细想想，居然还没算到家庭的问题，就已经深陷这个窠臼。即便有儿女要来葬礼，他依然不知道应该怎么办。

阿尔弗雷德焦躁不安，只能眼睁睁地看着这个场景在自己面前重演。他考虑过早日移民国外，但后来意识到自己的能力仍然无法解决家庭问题。他曾经在朋友口中自嘲说：“我的家人全靠我，如果我死了，那么整个世界都将会毁灭。”然而，阿尔弗雷德觉得自己没有资格这样做。既然生命如此脆弱，何况还要面临这样的境遇呢？这种种矛盾一直萦绕在阿尔弗雷德的脑海里。

28 岁的阿尔弗雷德终于下定决心，开始努力学习投资知识。他从股票投资的基础知识开始，先是在证券交易所上市的公司获得个人投资资格，再利用自己的投资金流购买一些国际股票，慢慢积累起丰厚的投资收益。

1998 年，阿尔弗雷德大学毕业，加入了一家名为 SimpliVity 的创业公司。该公司的 CEO 是杰克·佩里·沃兹尼亚克（Jack Perry Wozniak），他喜欢研究创新理念，又擅长销售技巧。一天，佩里来到公司，聊起自己的创业理念，阿尔弗雷德满头大汗。

佩里听完阿尔弗雷德的创业理念之后，问他是否准备抛弃之前的行业，走上投资之路。阿尔弗雷德略显紧张，但还是坚持保留自己的原则。虽然他还不了解投资逻辑，但他觉得自己至少比之前的公司有些长进。

那一年，阿尔弗雷德刚刚发现市场并非阿尔弗雷德想象中的那般宁静，投资者纷纷抛售重仓股票，进入套利交易，造成巨大的财务风险。为了应对市场的不确定性，阿尔弗雷德的策略是量入为出，寻找符合自己的条件的投资标的。他找到了一个世界著名的计算机编程语言——Java，并开始学习其语法规则。为了实现这份兴趣，阿尔弗雷德开始跟公司签约开发 Java 相关软件。但是，由于市场风险太大，阿尔弗雷德放弃了这份工作。

如今，阿尔弗雷德已是创业公司的 CTO，负责管理开发团队，同时兼任投资顾问。据他自己的话说，自己虽然懂技术，但对投资逻辑却毫无所知。因此，他开始阅读关于投资的书籍、培训课程和专业咨询。但是，他也承认，自己并不能理解投资背后的逻辑。他希望自己能建立起自己的投资系统，成为真正的投资专家。

不过，阿尔弗雷德在自学过程中也体会到，投资还是一门学问，需要的是反复学习和训练。他还是需要静下心来学习各种理论和方法，并结合实际案例运用自己的能力，最终把自己塑造成一名有能力的投资专家。
