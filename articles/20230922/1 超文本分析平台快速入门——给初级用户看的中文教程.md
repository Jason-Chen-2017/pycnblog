# 2.基本概念术语说明
## 1.数据采集
数据采集通常指的是将外部数据源获取到计算机系统中并存储起来，其目的在于用于后续的分析处理或是加工，目前主要分为两种方式：
### 1) 文件传输协议（FTP）
文件传输协议（FTP），英文全称File Transfer Protocol，是Internet上用来在两个计算机之间进行文件传送的标准协议，它采用客户服务器模式，即用户需要首先建立自己的FTP服务器，然后再连接到指定的FTP服务器上进行数据上传下载。
### 2) Web scraping
Web scraping，也称网页抓取，是一种按照特定规则从互联网网站上自动收集数据的方法，其过程如下：
1. 获取需要爬取的网址；
2. 使用开发好的爬虫脚本或者框架，模拟浏览器请求网页；
3. 分析页面结构，定位要抓取的数据所在位置；
4. 从页面源码中解析出需要的数据；
5. 将数据存入数据库或者文件中。

## 2.文本解析
文本解析就是从数据中提取其中的文本信息，其中包括对HTML文档、XML文档、PDF文档、Word文档、Excel文档、图像等的解析。目前比较常用的解析技术有以下几种：
### 1) HTML/XML解析器
HTML/XML解析器可以用来提取HTML/XML文档中的文本、标签及属性信息。常用解析库包括BeautifulSoup、ElementTree等。
### 2) PDF解析器
PDF解析器可以用来提取PDF文档中的文本信息。目前常用的PDF解析器有PyPDF2、pdfminer等。
### 3) Office文档解析器
Office文档解析器可以用来提取Word文档、Excel文档等的文本信息。常用解析库包括python-docx、openpyxl等。
### 4) OCR识别技术
OCR（Optical Character Recognition，光学字符识别），是指通过扫描或摄像头等设备从图像、 scanned document 中识别文字和数字。当前最常用的OCR技术有Tesseract、Google Vision API、Microsoft Azure Cognitive Services等。

## 3.抽取关键词
关键词是文本内容中最重要的特征，也是搜索引擎的基础。关键字是人们最早认识某事物的主要依据，通过对文本内容的分析，从而对文档、网页或是其它信息源进行信息检索，最终达到目的。关键词抽取是文本分析中最基础也是最重要的一个环节，很多开源的工具都提供了关键词抽取功能。关键词抽取方法一般分为三种：
### 1) TF-IDF(Term Frequency–Inverse Document Frequency)
TF-IDF方法的基本思想是计算一个词语出现的次数与它的逆文档频率之间的比值，得到的结果是不重复的、唯一的单词集。TF-IDF权重反映了文档中每个词语的重要性，即一段文本中某个词语越重要，则该词语在文档中出现的次数就越多。
### 2) TextRank算法
TextRank算法是一种基于PageRank算法的文本相似性度量方法，它考虑词语间的关联性，认为两个句子间的相关性由它们共同拥有的关键词决定。
### 3) 单词贪婪法
单词贪婪法也称为词袋模型，是一种简单但有效的文本分类方法。它假设每一篇文档只是一个由单个词组成的集合，不同类别的文档可以通过不同的词表获得区分。单词贪婪法会选择那些最具代表性的词来描述整个文档的主题。

## 4.主题模型
主题模型是一种无监督的文本聚类方法，其基本思想是发现文档集合中的共同主题，发现主题间的关系，并对文档进行重新排序，使得文档更容易归类。目前比较流行的主题模型有LDA（Latent Dirichlet Allocation）、NMF（Nonnegative Matrix Factorization）、LSI（Latent Semantic Indexing）等。

## 5.信息检索
信息检索是基于检索模型来实现的，它是一套系统，用于按指定查询条件检索在大型数据库（如文档数据库或事务数据库）中存储的海量信息。它涉及信息检索的两个基本问题：索引问题和查询处理问题。

## 6.搜索引擎
搜索引擎是指提供与用户交互的网络应用程序。搜索引擎是互联网的支柱，主要服务对象是互联网上的信息。用户可以在搜索引擎上输入关键词、搜索相关信息，同时也可收藏喜欢的网页。

## 7.Python编程环境
Python是一种易于学习，开放源代码的通用编程语言，被广泛应用于各行各业。Python支持多种编程范式，包括命令式、函数式、面向对象、面向线程、面向事件驱动的异步编程。

## 8.数据可视化
数据可视化是将复杂的数据转化为图形、图像、符号、文字等媒介，方便人们观察、分析、决策。常见的数据可视化技术有柱状图、饼图、散点图、气泡图等。
