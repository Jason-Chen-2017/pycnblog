数据可视化（Data Visualization）是将数据用图表、图像等媒体形式呈现出来，其目的是通过对数据的直观显示，发现模式和规律，并有效地传递信息。作为一项技术性产品，数据可视化具有重要的社会意义。随着互联网的普及和商业的发展，数据可视化已成为许多行业的标志性产品，例如电子商务、金融、IT、房地产、旅游、生态环境等领域。

## Echarts特点
### 1.简单易懂
Echarts的API采用面向对象的思想，相比其他同类库更加简单易懂。通过提供统一的全局变量echarts和Chart类，可以轻松实现复杂的图表配置。同时还提供了对常用属性的简写方式，让初学者不必花费过多时间在配置细节上。
### 2.强大的交互功能
Echarts图表库提供的图形交互功能包括缩放、平移、鼠标悬浮、Tooltip提示框等，这些交互特性能够帮助用户更好地理解数据之间的关联关系和异常值。此外，Echarts还支持切换图表类型、动画效果和数据筛选、排序等高级操作，帮助用户提升分析效率。
### 3.丰富的图表类型
Echarts提供了丰富的图表类型，包括线形图、柱状图、饼图、散点图、雷达图、K线图、热力图等，能够满足一般数据可视化需求。除此之外，Echarts还提供对各种图表属性的定制能力，比如图例、视觉映射、标注文本、标签、边框、网格等，能够满足不同场景下的个性化需求。
### 4.良好的扩展性
Echarts的插件机制让其具有很强的扩展性，使得开发者可以根据自己的业务需求自由开发新的图表类型。开发者也可以通过组合不同的图表类型，创造出各种独有的、炫酷的可视化作品。
