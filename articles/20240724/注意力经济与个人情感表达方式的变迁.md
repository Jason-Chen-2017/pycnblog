同时，用户的情感表达方式也在发生重大变迁。传统的书信、日记等长文本形式逐渐式微，而短文本、图片、短视频等形式变得流行。用户通过点赞、评论、分享等形式表达情感，情感交互方式变得更加即时和多元化。

如何理解和应对这一变化，抓住用户的注意力，实现高效的情感表达和交互，成为研究的热点问题。本文将从注意力经济的角度出发，结合情感表达方式的变化，探讨其背后的原理和应用。

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class Attention(nn.Module):
    def __init__(self, in_dim, attn_dim):
        super(Attention, self).__init__()
        self.W_q = nn.Linear(in_dim, attn_dim)
        self.W_v = nn.Linear(in_dim, attn_dim)
        self.W_o = nn.Linear(in_dim, in_dim)
        self.softmax = nn.Softmax(dim=1)
    
    def forward(self, q, v, k):
        q = self.W_q(q)
        v = self.W_v(v)
        k = self.W_v(k)
        attn_weights = torch.matmul(q, k.transpose(1, 2)) / torch.sqrt(torch.tensor(k.size(2)))
        attn_weights = self.softmax(attn_weights)
        attn_output = torch.matmul(attn_weights, v)
        output = self.W_o(attn_output)
        return output, attn_weights
```
