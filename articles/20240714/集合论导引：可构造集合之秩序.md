## 2. 核心概念与联系

### 2.1 核心概念概述

为更好地理解集合论的基本概念和核心思想，本节将介绍几个关键概念：

- **集合(Set)**：由元素组成的无序整体，用大括号或描述性语言表示。例如，{1, 2, 3}表示一个包含1、2、3三个元素的集合。
- **元素(Element)**：集合中的单个元素，可以是数字、字母、符号等任意类型。例如，1、2、3都是集合{1, 2, 3}的元素。
- **元素个数(Cardinality)**：集合中元素的数目，用$\#(\mathcal{S})$表示。例如，$\#(\{1, 2, 3\})=3$。
- **空集(Empty Set)**：不含任何元素的集合，通常用$\emptyset$表示。例如，$\emptyset$表示空集。
- **集合运算**：包括集合的并集、交集、差集、对称差集等基本运算。例如，$A \cup B$表示集合$A$和$B$的并集，$A \cap B$表示$A$和$B$的交集，$A \setminus B$表示$A$对$B$的差集。

这些概念构成了集合论的基础框架，是后续深入学习的前提。

以下是一个简单的集合运算示例代码：

```python
from sympy import FiniteSet

# 定义集合A和B
A = FiniteSet(1, 2, 3)
B = FiniteSet(3, 4, 5)

# 计算并集、交集、差集和对称差集
union_set = A.union(B)
intersection_set = A.intersect(B)
difference_set = A - B
symmetric_difference_set = A.symmetric_difference(B)

# 打印结果
print("并集:", union_set)
print("交集:", intersection_set)
print("差集:", difference_set)
print("对称差集:", symmetric_difference_set)
```
