# 2.核心概念与联系
## 文件I/O
在计算机中，文件（File）是指储存在磁盘或其他介质上的数据集合。一般来说，每一个文件都有一个特定的名称标识，用于唯一地标识这个文件；另外，还有一些元信息描述文件的内容、大小、创建日期等属性。对于文件来说，它的输入输出操作(I/O)就是它和计算机外部设备之间交换数据的行为。文件I/O分为两个主要类型：
- 文本文件 I/O
- 数据文件 I/O
### 文本文件 I/O
文本文件是最常见的一种文件类型。它们通常存储纯文本形式的信息，例如文档、电子邮件、网页或者其他文字型的数据。文本文件I/O包括读取和写入两种操作：
- 读取（Read）：从文件中读取内容，并把内容放到内存中供后续处理。
- 写入（Write）：向文件中写入内容，或是修改已有的内容。
文本文件I/O操作的特点：
- 以字符流的方式进行操作。
- 支持多种编码方式。
- 可以根据需要调整缓冲区大小。
- 提供同步（synchronous）和异步（asynchronous）两种方式访问文件。
### 数据文件 I/O
数据文件是非文本文件。它们通常由几种格式组成，例如图像文件、音频文件、视频文件等。数据文件I/O相比文本文件I/O更加复杂，因为数据文件可能不只包含文字数据，还可能包含其他数据结构、数字化的信息。
数据文件I/O操作的特点：
- 以字节流的方式进行操作。
- 不支持多种编码方式。
- 不能随意调整缓冲区大小。
- 只提供同步（synchronous）模式。
## Python中的文件读写操作
在Python中，对文件的读写操作可以使用内置函数`open()`完成。`open()`函数返回一个文件对象，该对象包含了文件的各种属性和方法，可以用来读取或写入文件中的数据。
```python
file = open('filename','mode')
```
其中，参数`filename`指定要打开的文件名，参数`mode`指定了打开文件的模式。模式由三个字符组成，分别表示打开文件的模式、读写权限以及是否创建新文件。其取值如下表所示：
| 模式 | 描述                   |
|-----|------------------------|
| r   | 读取模式。默认模式。     |
| w   | 写入模式。如果文件不存在则创建新的文件。           |
| x   | 创建模式。如果文件不存在则创建新的文件。           |
| a   | 追加模式。在文件末尾添加数据。文件若不存在则自动创建。   |
| +   | 更新模式。可读又可写。                            |
