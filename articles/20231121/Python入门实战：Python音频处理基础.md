# 2.核心概念与联系
## 2.1 数字信号与数字信号处理
### 2.1.1 什么是数字信号？
数字信号是指用0和1表示信息的方式。由于两种状态的存在，即电压的低或高，这两种状态就产生了两种基本的逻辑值——0和1。在数理计算方面，数字信号也叫量化信号，它是二进制的。举个例子，一段声音信号的波形图，就显示出是由一串连续的0和1组成的。而一般来讲，非数字信号都称之为模拟信号。

### 2.1.2 数字信号处理与频谱图
数字信号处理是利用计算机对数字信号进行处理的过程。数字信号处理主要包括如下几个方面：

1. 采样（Sampling）：就是将连续的信号抽取出固定时间间隔的样本点。这个过程会消除噪声干扰，并保证信号的平滑性。
2. 量化（Quantization）：是指用特定的编码方式把模拟信号转换成数字信号。典型的编码方式是阈值分割法，它根据一个阈值把信号分为两类。
3. 编码（Encoding）：把数字信号变换成可以传输、存储和处理的形式。编码的方法有ASCII码、可变长编码、差分调制、脉冲编码调制等。
4. 信号传输（Signal transmission）：通过网络、移动通信或者其他传输介质传输数字信号。
5. 信号处理（Signal processing）：是指对数字信号进行各种滤波、变换、分析等运算，从而得到所需的信息。信号处理可以用来监测、识别、还原、合成和转码信号。
6. 信号分析（Signal analysis）：是指对数字信号进行时间-频率分析、频域分析、时频域分析、基带分析、频谱分析等，得到各种参数和特征。
7. 声卡（Sound card）：是一种能够将数字信号转换成模拟信号输出到声卡接口的硬件设备。
8. 显示器（Display）：可以用于显示声音的设备。

数字信号处理过程中，我们需要了解如何生成数字信号，然后把这些信号送至信号处理器上进行处理。为了方便观察，我们通常画出信号的频谱图。频谱图是一个波形分布图，横轴表示频率，纵轴表示幅度。我们可以从纵轴的位置来判断信号的类型。

## 2.2 音频文件格式
### 2.2.1 什么是音频文件格式？
音频文件格式是指音频文件存放在磁盘上后，计算机能够直接读取的文件格式。目前主流的音频文件格式有WAV、AIFF、MP3、FLAC等。其中，最常用的就是WAV和MP3。

### 2.2.2 WAV文件格式
WAV文件格式是一种比较简单的音频文件格式。它是RIFF文件的扩展名。RIFF文件结构包括Chunk列表、Format Chunk、Data Chunk三部分。其中，Chunk列表记录着每个Chunk的大小，Format Chunk记录的是音频数据的格式信息，比如通道数、采样率等；Data Chunk则记录了音频数据。

### 2.2.3 MP3文件格式
MP3（MPEG-1 Audio Layer III）文件格式是MPEG标准下用于无损压缩音频的一种格式。与AAC、OGG Vorbis等无损音频格式相比，MP3是MPEG-1 Audio Layer III的简称。它的优势在于占用空间小、编码速度快、解码速度快、播放效果良好、广泛支持。因此，MP3格式被广泛应用在各类音频领域。

### 2.2.4 FLAC文件格式
FLAC（Free Lossless Audio Codec）文件格式是可无损压缩音频的一种自由格式。它采用了哈夫曼树算法对音频信号进行编码，并在首部添加元数据信息，以便在解码时辨识出所使用的压缩模式、原始采样率等信息。FLAC格式比MP3更适合携带高品质音频。
