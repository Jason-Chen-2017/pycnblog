                 

# 1.背景介绍


## 什么是图像处理与计算机视觉？
图像处理（Image Processing）与计算机视觉（Computer Vision）是两个领域。
### 图像处理
图像处理（英语：Image processing），也叫数字图像处理，是指利用电脑或其他硬件设备从图象、视频等各种媒体中提取、处理、分析、显示及存储图像的一系列过程。图像处理分为“输入与输出”、“处理”和“显示”三个阶段。图像处理的目的是对原始信息进行加工，产生新颖而富含意义的信息。比如，拍摄的照片、扫描得到的文字、手写的字迹、摄像头捕捉到的物体都可以经过图像处理，转换成计算机能识别、处理、存储的形式。图像处理能够帮助我们捕获、理解和分析大量复杂的信息，使得人们在工作、学习、娱乐等各个方面获得新的体验。
图像处理主要包括以下几类技术：

① 形态学与分割
形态学（Mathematical Morphology）是图像处理的一个重要的分支。形态学的方法就是对图像的空间结构进行变换，将一些小的结构合并到一起，并去掉一些较大的结构。形态学的应用十分广泛，如图像形态学、水平低通滤波、锐化、腐蚀、顶帽消除、黑帽消除、骨架提取等。

② 特征检测与描述
特征检测（Feature Detection）是图像处理的一个重要分支。特征检测的作用是从一幅图像中找出其中的显著特征点或者区域，这些特征点或者区域往往代表着图像的某些特定的结构信息。典型的特征检测方法有：角点检测、边缘检测、Blob检测等。特征描述（Feature Description）则是通过特征点或者区域的一些属性，如位置、颜色、纹理、大小等，对其进行描述，从而更好地刻画其特征。

③ 几何变换与特征匹配
几何变换（Geometric Transformation）是图像处理的一个重要分支。几何变换的方法就是对原始图像的几何结构进行调整，使之符合人们对真实世界的认识和观察，如缩放、旋转、剪切、折叠、叠加、透视等。几何变换可用于图像增强、重构、分类、检索、跟踪等方面。

④ 噪声处理
噪声处理（Noise Reduction）是图像处理的一个重要分支。由于图像采集设备的噪声影响，导致图像中的噪声很大，因此需要通过一定的方法对噪声进行过滤、去除或减弱，从而让图像中的有用信息呈现出来。典型的噪声处理方法有：降噪、降采样、二值化、直方图均衡化等。

⑤ 压缩与恢复
压缩与恢复（Compression and Reconstruction）是图像处理的一个重要分支。图像数据是由许多像素组成的，对于传统的基于硬盘的图像数据来说，这些像素分布于磁盘上，占据了相当大的存储空间。因此，为了减少存储空间的使用，有必要对图像进行压缩。目前，压缩图像的方法有JPEG、PNG、BMP、TIFF、GIF等。所谓的恢复（Reconstruction）即是根据原始的图像数据和压缩后的图像数据，重新构建出原始图像。

⑥ 模板匹配与统计学习
模板匹配（Template Matching）与统计学习（Statistical Learning）是图像处理的两大分支。模板匹配的方法就是用已知的模板在另一个图像上搜索匹配的模式，搜索的范围可以是整个图像或者局部图像。典型的模板匹配方法有：蛮力匹配、基于概率的匹配、自适应阈值等。统计学习的方法就是将图像的一些特征向量映射到高维空间，通过机器学习的方法，对这些向量进行训练，从而建立特征与标签之间的映射关系，从而实现对新的图像的分类、识别、检索、描述等功能。
### 计算机视觉
计算机视觉（英语：Computer vision）是一个跨学科的研究领域，涵盖了多种图像、视频、雷达、生物医学图像、运动与环境感应等领域。它利用计算机和相关算法对实时图像、流视频进行采集、处理、分析、理解和表达。计算机视觉可以解决诸如目标检测、场景理解、物体识别、三维建模、姿态估计、空间映射、事件跟踪等众多领域的问题。计算机视觉已经成为行业和社会的热门话题，具有极高的学术价值和应用价值。
计算机视觉的核心任务就是从输入图像中检测、识别、分析和描述其中的信息，进而实现智能控制、机器人导航、虚拟现实、图像修复、新闻理解、广告定位、人脸识别等众多应用。现有的计算机视觉技术已经取得了长足的发展，并且不断涌现出新的进展。其中，大部分的研究都是在新兴的深度学习（deep learning）和脑机接口（brain-computer interface，BCI）等方面取得突破性的进展。
