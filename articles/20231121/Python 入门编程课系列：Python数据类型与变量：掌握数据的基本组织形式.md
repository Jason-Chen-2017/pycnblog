# 2.核心概念与联系
## 数据类型（Data Types）
数据类型是指存储在内存中的值得类型。数据类型决定了数据的读写方式，如何存储，能否参与运算，数据类型的作用就是限制数据的范围，为其提供一个整体的结构。在python中，主要的数据类型有：
- 数字类型(Number): int、float、complex
- 字符串(String)
- 布尔类型(Boolean)
- 列表(List)
- 元组(Tuple)
- 集合(Set)
- 字典(Dictionary)
### 数字类型
Python 支持四种不同的数字类型：整数 (int), 浮点数 (float), 复数 (complex), 长整型 (long)。整数类型可以使用下划线作为分隔符：
```python
number = 1_000_000 # 1,000,000
```
浮点数类型使用“.”作为小数点符号：
```python
pi = 3.14
```
复数类型由实数部分和虚数部分构成，虚数部分后加上“j”表示：
```python
c = 3+4j
print(type(c)) # <class 'complex'>
print(c.real)   # 3.0
print(c.imag)   # 4.0
```
长整型用于大整数，但通常被认为是不必要的。
### 字符串类型
字符串类型用来存储文本信息，其中的每个字符都用单引号或双引号括起来。字符串类型支持很多内置方法来操作字符串，比如索引、切片、拼接等。如下示例：
```python
string = "hello world"
print(string[0])     # h
print(string[-1])    # d
print(len(string))   # 11
```
### 布尔类型
布尔类型只有两个取值True和False，它用来表示真值和假值。在判断语句中经常需要用到布尔类型，比如：
```python
a = True      # a等于True
b = False     # b等于False
if a:
    print("a is true")
else:
    print("a is false")
```
输出结果为："a is true"。
### 列表类型
列表类型是 python 中最常用的一种数据类型，它类似于数组，可以在其中存储多个元素。列表中的元素可以通过索引访问或者迭代器进行遍历。列表类型也支持很多内置方法，比如 append(), extend() 等。如下示例：
```python
list = [1, 2, 3]
print(list[0])          # 1
print(list[::-1])        # [3, 2, 1]
list.append(4)           # 添加元素
list.extend([5, 6])      # 追加多个元素
```
### 元组类型
元组类型类似于列表，不同之处在于元组中的元素不能修改。元组类型一般用于不想被修改的场景，比如函数的参数。如下示例：
```python
tuple = (1, 2, 3)
print(tuple[:])         # (1, 2, 3)
```
### 集合类型
集合类型也是一种数据类型，不同之处在于集合只存储唯一的元素，元素无序且不重复。集合类型支持很多内置方法，比如 add(), remove(), pop() 等。如下示例：
```python
set = {1, 2, 3}
print(len(set))          # 3
print(2 in set)          # True
set.add(4)               # 添加元素
set.remove(3)            # 删除元素
```
### 字典类型
字典类型是一个键-值对形式的无序集合，可以通过键来获取对应的值。字典类型支持很多内置方法，比如 get(), update() 等。如下示例：
```python
dict = {"name": "Alice", "age": 20}
print(dict["name"])      # Alice
del dict["age"]          # 删除键值对
dict.update({"gender":"female"})    # 更新字典
```
