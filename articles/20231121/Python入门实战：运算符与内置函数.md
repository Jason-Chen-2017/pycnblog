                 

# 1.背景介绍


Python作为一种动态语言，允许运行时创建对象、执行动态类型检查等特性，其灵活的语法以及丰富的库和工具让它成为众多编程人员的最爱。对于一些初级开发者而言，学习并掌握Python的基本知识尤为重要。本文试图通过对Python中的运算符及内置函数的学习，帮助读者更好地理解Python语言以及相关应用场景。

# 2.核心概念与联系
Python提供了丰富的内置数据类型、控制结构语句、流程控制语句、文件I/O、异常处理机制、模块导入、面向对象编程等功能。其中运算符和内置函数是每一个Python程序员必备的基础。因此，在学习Python中重要的基础知识之一就是理解运算符和内置函数。

## 2.1 算术运算符

| 运算符 | 描述                                                         | 示例             |
| ------ | ------------------------------------------------------------ | ---------------- |
| +      | 加法运算符                                                   | a = b + c        |
| -      | 减法运算符                                                   | a = b - c        |
| *      | 乘法运算符                                                   | a = b * c        |
| /      | 除法运算符                                                   | a = b / c        |
| %      | 取余运算符（返回除法后的余数）                               | a = b % c        |
| **     | 幂运算符                                                     | a = b ** c       |
| //     | 地板除（整数除法）                                            | a = b // c       |
| &=     | 按位与赋值运算符                                             | a &= b           |
| |=     | 按位或赋值运算符                                             | a |= b           |
| ^=     | 按位异或赋值运算符                                           | a ^= b           |
| <<=    | 左移位赋值运算符                                             | a <<= b          |
| >>=    | 右移位赋值运算符                                             | a >>= b          |
| ==     | 等于判断运算符                                               | if a == b:...   |
|!=     | 不等于判断运算符                                             | if a!= b:...   |
| >      | 大于判断运算符                                               | if a > b:...    |
| <      | 小于判断运算符                                               | if a < b:...    |
| >=     | 大于等于判断运算符                                           | if a >= b:...   |
| <=     | 小于等于判断运算符                                           | if a <= b:...   |

## 2.2 比较运算符

比较运算符也称逻辑运算符，用于比较两个值的大小关系。

| 操作符 | 描述                             | 示例               |
| ------ | -------------------------------- | ------------------ |
| is     | 检查两个标识符是否引用同一个对象 | x is y             |
| is not | 检查两个标识符是否引用不同对象 | x is not y         |
| in     | 判断对象是否存在于序列中         | x in y (x是y的一部分) |
| not in | 判断对象是否不在序列中           | x not in y         |

## 2.3 赋值运算符

赋值运算符用来给变量赋值。

| 运算符 | 描述                                  | 示例                 |
| ------ | ------------------------------------- | -------------------- |
| =      | 将右边的值赋给左边的变量                | a = b (=号两边不能有空格) |
| +=     | 加上等于右边的值，再赋值给左边的变量    | a += b                |
| -=     | 从左边变量中减去等于右边的值，再赋值给它 | a -= b                |
| *=     | 将左边变量乘以等于右边的值，再赋值给它  | a *= b                |
| /=     | 将左边变量除以等于右边的值，再赋值给它  | a /= b                |
| %=     | 计算出两值之间的模（余数），赋值给左边变量 | a %= b                |
| **=    | 计算左边变量的指数次方，赋值给左边变量  | a **= b               |
| //=    | 将左边变量整除等于右边的值，再赋值给它  | a //= b               |

## 2.4 位运算符

位运算符主要用于对二进制数字进行操作。

| 操作符 | 描述                                   | 示例            |
| ------ | -------------------------------------- | --------------- |
| &      | 按位与运算符                            | a & b           |
| \|     | 按位或运算符                           | a \| b          |
| ^      | 按位异或运算符                         | a ^ b           |
| ~      | 按位取反运算符（反转所有位，即1变成0，0变成1） | a = ~b          |
| <<     | 左移动运算符（按位左移位，低位溢出后补零） | a = b << c      |
| >>     | 右移动运算符（按位右移位，高位溢出舍弃） | a = b >> c      |
| &=     | 按位与赋值运算符                       | a &= b          |
| \|=    | 按位或赋值运算符                       | a \|= b         |
| ^=     | 按位异或赋值运算符                     | a ^= b          |
| <<=    | 左移位赋值运算符                       | a <<= b         |
| >>=    | 右移位赋值运算符                       | a >>= b         |
