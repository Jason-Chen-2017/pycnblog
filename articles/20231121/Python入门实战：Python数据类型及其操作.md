# 2.核心概念与联系
数据类型，是指计算机内存中用来保存数据的不同结构形式，在计算机编程语言中，数据类型有以下几种：

1. 整型（int）：整数，包括正整数、负整数、零。例如：1、-7、0。

2. 浮点型（float）：小数，包括有限小数和无限大数。例如：3.14、1.0e-10。

3. 字符串型（str）：字符串，是由0个或多个字符组成的一串文字。例如："hello"、"world!”。

4. 布尔型（bool）：布尔值，只有两个取值True和False。例如：True、False。

5. 列表（list）：列表，是多个元素按照一定顺序排列的集合。

6. 元组（tuple）：元组，类似于列表，但不能修改元素的值。

7. 字典（dict）：字典，是一种映射表，可以存储键值对信息。

除了上述数据类型外，还有如下的复合数据类型：

1. 集合（set）：集合，是一个无序的、不重复的元素序列。

2. 数组（array）：数组，是一个固定大小的多维数组对象。

这些数据类型之间存在着一些相互关系，比如，整数、浮点数和布尔值都属于数字类型；字符串和列表都属于可变类型。

# 数据类型之间的转换
## int() 函数
### 将 float 转化为 int
通过 int() 函数可以将一个浮点数转换为整数：
```python
>>> x = 3.14
>>> y = int(x)
>>> print(y)
3
```
### 将 bool 转化为 int
True 和 False 在转换为 int 时会被转换为 1 和 0。
```python
>>> b1 = True
>>> i1 = int(b1)
>>> print(i1)
1
>>> b2 = False
>>> i2 = int(b2)
>>> print(i2)
0
```
### 将 str 转化为 int
将字符串转换为整数时，需要确保字符串中的所有字符都是数字才能成功转换。若字符串中包含除数字以外的其他字符，则转换失败并报错。
```python
>>> s1 = "123"
>>> i1 = int(s1)
>>> print(i1)
123
>>> s2 = "abc"
>>> i2 = int(s2) # ValueError: invalid literal for int() with base 10: 'abc'
```
## str() 函数
### 将 bool、int、float 转化为 str
分别将 bool、int、float 类型转换为字符串。
```python
>>> b1 = True
>>> s1 = str(b1)
>>> print(s1)
'true'
>>> b2 = False
>>> s2 = str(b2)
>>> print(s2)
'false'
>>> n1 = -123
>>> s3 = str(n1)
>>> print(s3)
'-123'
>>> f1 = 3.14
>>> s4 = str(f1)
>>> print(s4)
'3.14'
```
### 使用 format 方法输出 str
format() 是字符串对象的一个方法，可以格式化字符串。其中 {} 表示占位符，通过 {} 中的变量名引用后续参数。format() 的调用方式是字符串.format(*args, **kwargs)。

下面举例说明：

```python
name = 'John'
age = 30
salary = 5000
print('My name is {}, my age is {}, and I earn {}'.format(name, age, salary))
```
输出结果：
```
My name is John, my age is 30, and I earn 5000
```
