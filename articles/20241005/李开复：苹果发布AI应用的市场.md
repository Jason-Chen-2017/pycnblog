## 4. 数学模型和公式 & 详细讲解 & 举例说明

在人工智能应用中，数学模型和公式起着至关重要的作用。下面我们将介绍一些常用的数学模型和公式，并通过实际案例进行详细讲解。

### 4.1 激活函数

激活函数是神经网络中不可或缺的一部分，用于对神经元输出进行非线性变换。常见的激活函数包括：

1. **Sigmoid函数**：

   $$sigmoid(x) = \frac{1}{1 + e^{-x}}$$

   Sigmoid函数将输入值映射到（0，1）区间，常用于二分类问题。

2. **ReLU函数**：

   $$ReLU(x) = \max(0, x)$$

   ReLU函数将输入值映射到（0，+∞）区间，具有简单、计算效率高等优点。

### 4.2 损失函数

损失函数用于评估模型预测值与真实值之间的差距，常见的损失函数包括：

1. **均方误差（MSE）**：

   $$MSE(y, \hat{y}) = \frac{1}{m} \sum_{i=1}^{m} (y_i - \hat{y}_i)^2$$

   均方误差将预测值与真实值之间的差距平方后求和，常用于回归问题。

2. **交叉熵损失（Cross-Entropy Loss）**：

   $$Cross-Entropy Loss(y, \hat{y}) = -\sum_{i=1}^{m} y_i \cdot \log(\hat{y}_i)$$

   交叉熵损失衡量模型预测概率与真实概率之间的差距，常用于分类问题。

### 4.3 举例说明

假设我们使用神经网络进行二分类任务，输入特征为 \(x = [1, 2, 3]\)，真实标签为 \(y = 1\)。我们使用Sigmoid函数和交叉熵损失函数进行模型训练。

1. **初始化模型参数**：

   - 权重 \(W = [0.1, 0.2, 0.3]\)
   - 偏置 \(b = 0.1\)

2. **前向传播**：

   $$z = x \cdot W + b = [1, 2, 3] \cdot [0.1, 0.2, 0.3] + 0.1 = [0.4, 0.7, 1.0]$$
   $$a = sigmoid(z) = \frac{1}{1 + e^{-z}} = [0.6321, 0.9085, 0.9709]$$

3. **计算损失函数**：

   $$\hat{y} = a = [0.6321, 0.9085, 0.9709]$$
   $$y = [1, 0, 0]$$
   $$Cross-Entropy Loss = -\sum_{i=1}^{3} y_i \cdot \log(\hat{y}_i) = -[1 \cdot \log(0.6321) + 0 \cdot \log(0.9085) + 0 \cdot \log(0.9709)] \approx 0.4113$$

4. **反向传播**：

   - 计算梯度
   - 更新模型参数

   通过迭代优化模型参数，提高模型性能。
