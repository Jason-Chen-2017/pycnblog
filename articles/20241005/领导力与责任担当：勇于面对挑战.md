### 2.1 领导力的核心要素

领导力是引导和激励团队实现目标的关键能力。核心要素包括：

- 激励：领导者通过正面激励，激发团队成员的潜力，提高团队士气。
- 决策：领导者需在复杂情况下做出明智的决策，引导团队前进。
- 团队协作：领导者要协调团队成员，确保各成员充分发挥作用，共同实现目标。

### 2.2 责任担当的具体表现

责任担当是领导者面对挑战时的基本态度。具体表现包括：

- 自我驱动：领导者主动承担责任，不逃避困难。
- 决策：在面临挑战时，领导者要勇于做出决策，并承担决策后果。
- 团队协作：领导者要激励团队成员，共同面对挑战，实现目标。

```python
from django.db import models

class Course(models.Model):
    title = models.CharField(max_length=100)
    description = models.TextField()
    instructor = models.ForeignKey('Instructor', on_delete=models.CASCADE)

    def __str__(self):
        return self.title

class Instructor(models.Model):
    name = models.CharField(max_length=100)
    email = models.EmailField()

    def __str__(self):
        return self.name

class Enrollment(models.Model):
    course = models.ForeignKey(Course, on_delete=models.CASCADE)
    student = models.ForeignKey('Student', on_delete=models.CASCADE)
    date_enrolled = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return f"{self.student.name} enrolled in {self.course.title}"

class Student(models.Model):
    name = models.CharField(max_length=100)
    email = models.EmailField()

    def __str__(self):
        return self.name
```
