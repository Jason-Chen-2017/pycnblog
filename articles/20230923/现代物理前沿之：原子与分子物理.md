# 1.简介
  

## 1.1 物理学
物理学是研究如何在宇宙中制造、运动和变化的科学。物理学从微观到宏观都研究这一过程。
### 1.1.1 大本钟
一切物理现象都是由于受到大自然力量支配而产生的。大自然是一个既强又灵活的系统，它不断变化着，并随时准备接受挑战。每隔几十万年，其运作机制都将发生巨变，就像钟表一样，每隔一段时间走一下弧度，引起极其微妙的变化。在任何一个宇宙中，都存在着一个最主要的物理学主体——“大爆炸”或者“大帕金森”，他具有掌控整个宇宙的能力。这个主体的名称取自其对整个宇宙的吞噬作用。
### 1.1.2 中心思想
物理学的中心思想是普适性原理(universal principle)。这一思想认为，所有物质都遵循定律和规律，且这一规则不会因为任何人的意志而改变。事实上，这也是人们相信大爆炸的原因。人们无法测量到小尺度的微观粒子行为，因此只能根据大爆炸带来的影响范围和速度来估计整个宇宙的运行状况。
### 1.1.3 分类
物理学按照其发现及应用领域的不同，可以分成如下几类:
- 一般物理学
- 电磁学
- 核物理学
- 量子物理学
-...
## 1.2 生物学
生物学是研究生命活动、结构和变化的一门学科。生命活动包括细胞的生成、传播、利用资源、适应环境以及自我修复等。生命结构主要由细胞核、酶和修饰体三种基本单元构成。生命结构在一定的条件下形成，并经历由父母兄弟姊妹四代组成的“进化”过程。
## 1.3 分子生物学
分子生物学研究细胞内的分子。细胞中的分子排列、形态、功能等依赖于双层保护结构，即轴向保护层和平行保护层。在轴向保护层上存在一个中心核心，将细胞的基部与轴向细胞核区隔开；平行保护层则通过阻止细胞的外溢作用，维持细胞的独立性。分子生物学的研究使得我们能够对细胞进行精准的控制。
# 2.原子与分子
## 2.1 定义
原子是指最小的物质单位，也就是说，原子的结构与性质不能再分割。一个物质就是由若干个原子组成。
