# 1.简介
  

这个系列文章将教会大家怎么加入到一个创业公司的股权计划中，并且管理好自己的投资账户，积累独立的收入源，最终实现财务独立。在进入公司之后，你需要做哪些准备工作，阅读哪些资料才能更好的融入到这个行业，以及最后能得到什么样的回报。
## 创业公司的角色
创业公司的股权计划通常分成三种角色：创始人、投资者、管理层。创始人是企业的企业主，他们拥有所有权，根据企业规模可以把股份直接拥有或者分给他人持股；投资者则代表各界人士或机构向企业投资，并获得利润分成；管理层则负责公司的日常经营管理和运营等事务。
## 为什么要加入创业公司的股权计划
除了自身的创业激情外，加入创业公司股权计划还有很多理由：
1. 在创业公司你可以依靠稳定收入和安全保障，享受到较高的社会声誉和地位。
2. 创业公司往往设立多条产业链，你可以通过多元化的产品、服务或资本运用网络进行收益的再分配。
3. 创业公司的管理层往往都具有丰富的经验和技能，能够帮助你更好地管理你的投资账户，避免风险的侵蚀。
4. 如果你想脱离父母的经济压力，为自己提供更大的发展空间，加入创业公司股权计划是一个不错的选择。
5. 有很多创业公司都会筹集大量资金用于开发新的业务模式或产品，这些都可以通过加入创业公司的股权计划来实现。
6. 在创业公司里你可以享受到公司组织的各种福利，包括免费的咖啡、早餐、体育锻炼、住房贷款、旅游费、话费补贴等。
7. 您也可以在其他企业发展的同时兼顾个人事业，为自己留下一段完整的财富历程。
## 创建自己的创业公司
如果您还没有创业公司，那么可以通过以下方式创建一个：

1. 在网络上搜索“如何创建自己的创业公司”，寻找适合您的企业模式和流程。
2. 根据您的需求制定详细的设立方案，如注册、选址、布局、招募、培训、运营等。
3. 开展创业期间的所有商业活动，提升您的知名度、影响力，获得有价值的资源和收益。
4. 运用自己掌握的管理知识、经验、技能、资金、人才和管理经验，在资本市场的竞争中胜出。
## 加入创业公司股权计划的好处
1. 提升个人能力：当你加入到创业公司的股权计划中后，你就可以把更多的时间放在学习和工作上，为自己提供更丰厚的回报。你也能从中获取到公司独特的价值观和品牌形象，增强自己在竞争中更有竞争力。
2. 减少风险：创业公司的股权计划往往会建立起比较完善的保险机制，保障你不受到因股权转让等行为带来的损失。而且由于你是股东，你还可以委托别人来管理你的账户，增加自己的控制力。
3. 更好的了解企业：在创业公司里，你可以真正接触到不同领域的精英人物，分享你的想法和建议。你也可以与公司内外的同事交流，增进感情。
4. 跟随时代潮流：创业公司的股权计划已经成为一项快速发展的行业，相关的政策、法律、法规和规则也在不断更新迭代。通过加入创业公司的股权计划，你还可以跟踪最新的投资信息，保持对整个行业的最新认识。
5. 尽可能多的创造收益：创业公司的股权计划往往不断吸纳新成员，创造更多的价值。无论是在创业初期还是后期，加入创业公司的股权计划都可以让你获得丰厚的收益。
## 准备阶段
首先，你需要确认自己具备的条件是否满足创业公司的股权计划要求。一般来说，创业公司的要求如下所示：
1. 年龄：年满18周岁及以上。
2. 投资资格：必须持有一定数量的股份。
3. 投资额度：不能超过总投资额的5%。
4. 身份证件：身份证、护照、银行卡等。
5. 投资时间：需在创业公司注册满5年。
## 个人准备
在加入创业公司的股权计划之前，你应该准备好一些个人资料：

1. 个人陈述：建立一份个人简历，介绍一下自己擅长的方面、对公司的看法和评价、投资收益率以及预计作为独立创业者的理想。
2. 投资组合：明确自己的资金来源和投资方向。
3. 债券、基金：投资任何产品之前，首先考虑清楚风险性和收益。
4. 联系方式：输入你的电子邮箱地址，用来接收公司信息和询问投资问题。
## 浏览资料阶段
当你开始浏览创业公司的股权计划时，你应该先熟悉一下其中的主要资讯来源，包括：

1. 公告：收听创业公司的最新动态，有助于了解公司的发展情况和产品，以及你是否有兴趣加入。
2. 新闻：仔细阅读创业公司的新闻报道，了解公司的最新消息。
3. 会议通知：了解公司的重要会议安排，以便参加或观看。
4. 电视节目：观看创业公司的宣传片或资讯直播，了解公司的发展动态。
5. 官方网站：了解公司的信息、政策、章程，掌握公司的发展方向、人员结构、产品介绍等。
6. 社交媒体平台：关注公司的最新动态和资讯，和朋友们互动交流。
## 阅读手册阶段
在了解了创业公司的股权计划概况和资料后，你应该进一步阅读关于创业公司的股权计划的手册，包括：

1. 股权协议书：仔细阅读股权协议书的内容，确定个人在此协议下的权利和义务。
2. 风险告知书：仔细阅读风险告知书的内容，了解公司对个人投资的风险管理制度。
3. 风险监控工具：了解公司的财务审计工具、风险管理平台，掌握公司对投资者的风险状况。
4. 投资管理指南：阅读投资管理指南，了解如何管理个人的投资账户。
5. 职业发展指导：了解相关的职业发展指引，了解公司的求职招聘要求。
## 缴纳开立保证金阶段
当你了解到创业公司的股权计划的基本要求、政策、规则和要求后，就要开始缴纳保证金了。对于每笔投资，你都要缴纳50%的年化投资税，而保证金则用于购买公共资产并偿还未来可能发生的债务。
## 操作技巧阶段
当你缴纳保证金、购置公共资产、缴纳投资税后，你就应该开始操作了，包括：

1. 选择一家可以让你满意的创业公司。
2. 探索与该公司相关的业务和服务。
3. 寻找与目标公司相似的行业投资人。
4. 准备相关的证件和文件，包括身份证明、税务登记表、投资协议等。
5. 拨打运作中心热线或发送邮件，提交申请，开始申购产品和服务。
6. 填写相关申请表，提交申请，按时缴纳保证金，待审批。
7. 当项目获得批准后，确认投资金额，进行投资。
8. 每季度按照正常的业务经营操作，记录交易记录和财务状况。
9. 了解公司对您的投资情况和收益，并根据自己的判断进行调仓。
10. 对投资结果进行评估，了解自己的理财情况。
