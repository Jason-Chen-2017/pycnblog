# 2.基本概念术语说明
## 2.1 Mobile App定义
### 什么是Mobile App？
Mobile App是一种运行在智能手机、平板电脑或其他移动设备上的应用程序，它可以帮助用户完成特定的任务或者管理个人的信息。在现代社会里，越来越多的人把移动设备作为个人电脑、工作电脑或学习电脑的替代品，通过安装并打开移动应用实现各种各样的工作。而Mobile App却不同于桌面软件，因为它们可以直接访问设备硬件，因此具有独特的用户体验。
### 为什么要做Mobile App？
相比起传统的Web应用，移动应用拥有巨大的吸引力。首先，移动设备的尺寸小、屏幕大小广、带宽有限等特点使得App可以实现“轻量级”的特性，能够很好的满足用户的需求；其次，由于智能手机硬件的性能优势，移动应用能实现更快的响应速度，缩短用户等待时间，从而提高用户的满意度；第三，随着移动互联网的快速发展，移动应用逐渐成为新的核心价值流，因此越来越多的企业将重点放在这一领域，通过建立移动应用产品为客户提供服务。
### Mobile App的分类
- 游戏App：主要提供娱乐、消遣或社交功能，主要受众是玩家群体。
- 社交App：主要提供社交网络功能，如聊天、微博、QQ、微信，主要受众是社交圈子中的用户。
- 媒体App：主要提供媒体娱乐功能，如视频、音乐、图文分享，主要受众是喜欢看视频和听音乐的用户。
- 金融App：主要提供交易、支付、储存、消费功能，主要受信群体是金融工作者或个人。
- 工具App：主要提供工具性功能，如计算器、记事本、PDF阅读器等，主要受众是爱上使用各种工具的人。
- 旅游App：主要提供出行导航、热门景区推荐、酒店预订、租车服务等功能，主要受众是想出国旅游的人。
- 购物App：主要提供购物浏览、搜索、下单、评价、收藏、结账等功能，主要受众是需要购物的人。
- 健康App：主要提供身体检测、疾病统计、医生咨询等功能，主要受众是有健康危险意识的人。
- 生活App：主要提供新闻、日程安排、会议记录、兴趣推荐、地图导航等功能，主要受众是生活消费者。
- 教育App：主要提供学习资讯、作业提交、成绩查询、课程分享等功能，主要受众是希望提升职场竞争力的学生群体。
- 服务App：主要提供本地运动健身、保洁清洗、房屋维修、婚姻指导等功能，主要受众是需要解决实际问题的人。
- 新闻App：主要提供新闻、图片、视频资讯等，主要受众是喜欢阅读新闻的人。
- 信息App：主要提供常用网站的网页版，主要受众是爱上获取资讯的人。
## 2.2 Mobile App开发流程
### 移动应用开发流程

移动应用的开发流程分为五个阶段：需求分析、设计、编码、测试、部署和维护。
- 需求分析：移动应用的需求通常都比较复杂，需求分析过程非常重要。产品经理和设计师根据用户的需求制定移动应用的目标功能和用户体验。
- 设计：设计师负责移动应用的界面设计、风格设计和交互设计，确保移动应用的美观、可用性和高效率。同时，移动应用的业务逻辑设计也需要与产品经理一起讨论。
- 编码：开发者负责编写移动应用的源码，这个过程包含四个步骤：
  - 可视化编程工具：开发者需要选择一款适合自己团队的可视化编程工具来编写代码。
  - 编程语言：开发者需要熟练掌握一门适合移动端应用的编程语言，如Java、Objective-C、Swift、Python等。
  - 数据库设计：移动应用开发过程中，往往会用到各种类型的数据库，例如关系型数据库SQL Server、MySQL，NoSQL数据库MongoDB等。数据库的设计与选择也非常重要。
  - API接口调用：移动应用中可能会用到第三方API接口，如微信登录、微博分享等，API接口的选择也会影响到移动应用的运行效率。
- 测试：测试人员需要对移动应用进行测试，验证其功能、性能、兼容性、稳定性等。测试过程中，还需要找出移动应用的问题，并提出解决方案。
- 部署：部署人员需要将移动应用发布到各类移动应用市场，让更多用户下载和使用该应用。
- 维护：维护人员需要对应用进行更新、修复、优化，确保移动应用持续稳定运行。
### 技术栈介绍
移动应用的技术栈主要包括以下几种：
- 前端技术：HTML、CSS、JavaScript
- 后端技术：Java、Node.js、PHP、Python
- 数据库技术：SQLite、MySQL、Oracle
- 服务器技术：Apache、Nginx、IIS
- 消息推送技术：短信、邮件、微信、微博、QQ、通知栏
- 第三方SDK：微信SDK、支付宝SDK、友盟SDK、百度统计SDK
### IDE介绍
为了方便团队成员编写代码，开发者需要选择一款适合自己的集成开发环境(IDE)。目前最流行的有以下三种：
- Android Studio：由谷歌推出的基于IntelliJ IDEA的Android开发环境，基于Kotlin语言开发，支持Gradle构建系统，适合Android开发者使用。
- Xcode：苹果公司推出的一款基于Swift的iOS开发环境，由Swift语言开发，支持CocoaPods包管理器，适用于Objective-C开发者。
- Visual Studio Code：微软推出的一款开源跨平台编辑器，支持各种编程语言，被越来越多的开发者青睐。
