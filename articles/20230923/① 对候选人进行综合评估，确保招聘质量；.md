# 1.简介
  

## 概述
随着信息技术和互联网技术的快速发展，技术人员的需求也越来越多，企业纷纷寻求聘用AI、大数据、云计算、人工智能等领域的技术专家。而如何从海量技术职位中筛选出最合适的人才成为了企业面临的一个重要难题。作为一名技术专家，需要对自己的技能、兴趣、经验等多个方面充分掌握并投入到职业生涯中，才能实现自我价值。然而，如何快速准确地评估一个人的能力、意愿、潜力、天赋以及个人的学习能力，仍然是一个值得探索的问题。
传统的评估方法一般都是通过笔试或者面试的方式，这种方式效率低下且不公平。21世纪初期，人工智能研究者们提出了一种基于模型的方法，可以有效地评估候选人的能力、潜力和天赋。
## 方法
本文将介绍一种基于机器学习模型的方法，即Soft Skill Evaluator（SKE），该模型可以同时考虑人物的软性技能和硬实力，并根据不同场景下的情景来决定相应的推荐程度。该模型具有以下几个特点：

1. 考虑软性技能：该模型以问卷调查的方式收集候选人个人特色、工作经历、兴趣爱好、擅长的领域、能力短板及优势，从而分析候选人的软性技能。

2. 考虑硬实力：在现实生活中，候选人往往有自己独特的技能或能力，比如语言表达能力、团队协作能力、组织管理能力等。因此，该模型通过分析各个领域中候选人具有较高潜力、长处、与众不同的优秀案例，来衡量候选人的硬实力。

3. 灵活可变：该模型具备多种场景选择机制，能够应对不同职位要求、年龄范围等条件，为候选人提供最优推荐。

4. 可信度高：该模型采用了最新、最先进的机器学习算法，其精度和稳定性得到了验证。

## 应用场景
1. HRBP/人事行政人员：该模型可以帮助HRBP了解候选人能力特质，根据其特点确定是否进行笔试、面试或其他方式的推荐。

2. 技术顾问：该模型可以帮助技术顾问找到最合适的人选，为公司提供更优质的技术服务。

3. 培训机构：该模型可以在培训课程设计、专业教师选取和薪酬测算等环节给予候选人更加具体的建议。

4. 领域内外推荐平台：该模型还可以用于领域内外技术社区和推荐系统的推荐。

## 模型结构
如下图所示，SKE模型由两个子模型组成：Hard Skill Model（HSM）和 Soft Skill Model（SSM）。其中，HSM负责对候选人做硬实力建模，包括语言能力、编程能力、计算机知识、操作系统、数据库等；SSM则负责对候选人做软性技能建模，包括项目管理、业务分析、解决问题能力、沟通交流、团队合作等软性技能。
### HSM模型
HSM模型主要由三个模块构成，包括基础知识模型、项目模型和案例模型。
#### 基础知识模型
基础知识模型主要包括计算机语言、数据结构、操作系统、数据库等方面的知识。候选人通过回答一些简单问题，如“你知道什么是指针吗？”、“你知道什么是递归函数吗？”、“你知道什么是多线程编程吗?”，来评估候选人对这些基础知识的掌握程度。
#### 项目模型
项目模型主要基于项目管理知识，包括项目计划、需求分析、设计、开发、测试、发布、运维等流程。候选人通过参与某些项目并完成任务，来评估候选人对项目管理的熟练程度。
#### 案例模型
案例模型基于个人经验，采用多种项目代表性的案例来评估候选人的解决问题能力。例如，候选人可以列举最近接触过的难题，并向面试官描述解决该问题的过程。
### SSM模型
SSM模型主要由六个模块构成，包括项目管理、业务分析、解决问题、沟通交流、团队合作、英语口语等七个软性技能。
#### 项目管理
候选人在面试过程中表现出的项目管理能力。在回答项目管理相关问题时，候选人应该在此过程中有自己的理解和总结。项目管理包括项目策划、资源分配、风险控制、进度跟踪、文档编写等多个方面。
#### 业务分析
候选人在面试过程中表现出的业务分析能力。在回答业务分析相关问题时，候选人应该在此过程中清晰地阐释业务目标、竞争对手、商业模式、市场规模、客户群体、核心资源、商业计划等。
#### 撮合问题
候选人在面试过程中表现出的解决问题能力。在回答解决问题相关问题时，候选人应该能够识别关键信息，找出问题的根源，并有效地引导对话。
#### 沟通交流
候选人在面试过程中表现出的沟通交流能力。在回答沟通交流相关问题时，候选人应该能够积极主动，善于引导对方思路，对事物有自己的见解。
#### 团队合作
候选人在面试过程中表现出的团队合作能力。在回答团队合作相关问题时，候选人应该能够擅长组织团队，协助同事完成工作。
#### 英语口语
候选人在面试过程中表现出的英语口语能力。在回答英语口语相关问题时，候选人应该能够流利地运用英语，对英语听说读写等方面保持良好的素养。
#### 深度融合
候选人同时具备HSM和SSM模型的能力。HSM模型侧重于评估硬实力，可以从计算机基础知识、项目经验、案例研究等方面进行评估；SSM模型侧重于评估软性技能，可以从项目管理、业务分析、解决问题、沟通交流、团队合作、英语口语等方面进行评估。通过这两种评估结果的融合，SKE模型就能得出最终的候选人推荐。
