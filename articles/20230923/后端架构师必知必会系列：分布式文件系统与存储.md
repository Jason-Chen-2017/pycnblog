# 1.简介
  


## 1.1 概览

分布式文件系统与存储是一个很重要的方向，也是许多公司在持续开发自己的核心业务的过程中所依赖的基础设施。无论是云服务商还是私有部署，选择正确的分布式文件系统与存储都可以极大的提升公司的核心业务的可靠性、可用性和扩展性。本专栏旨在帮助读者快速了解并掌握分布式文件系统与存储的基本知识，以及不同厂商实现该功能的方法和技术细节。
