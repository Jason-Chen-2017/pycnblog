# 1.简介
  

X射线衍射（Radiation-Driven Deposition）或短波伴随光衍射是利用外来的高能辐射产生的可见光而 deposit 在物体表面上的一种现象。X射线进入组织后，由于其散射特性，会照射到周围组织内的组织细胞上，形成各种形式的紫外线病变，如囊状结节、炎症性疾病等。在医学、科研领域，X射线衍射是一种重要的医学手段和诊断工具。X射线衍射与透射电子显微镜（Electron Microscopy with Transmission Electron Microscope, EMTM），是两个常用的检测微生物标本的方法。EMTM 是利用电子束中的反射行为及透射率信息进行无创下的染色免疫学和多种生物特征分析的一种实用化的新型微观成像手段。

随着 X 射线对身体的危害越来越大，X射线的应用也越来越广泛，除了作为诊断手段外，还可以用来研究人体内部、组织、器官的结构、功能和生物活性。另外，由于 X 射线可以引起免疫反应，因此也被广泛用于医学研究。因此，X 射线的应用逐渐成为医学领域和科研领域的一个热点。

本文将从基础知识出发，对 X 射线衍射和透射电子显微镜的原理、概念、原理、特点及相应的检测方法进行阐述。在具体的原理与操作方法介绍之前，先回顾一下生物医学图像学的相关知识，包括模态学、色彩学、光学及空间位置关系等基本概念，以便更好的理解 X 射线衍射和透射电子显微镜的原理和作用。

# 2.生物医学图像学基本概念
## 2.1 模态学
模态（Modality）指的是由不同的频率成分组成的信号，由于 X 射线的波长较短，因此它只能发射一种频谱，即 X 射线单色光谱（Radioactivity Spectrum）。通常情况下，各种生物都具有两种或更多的色素，它们通过光合作用产生多种颜色的光谱，称为色谱。色谱可以看作是不同频率成分组成的信号，因而也是模态。

## 2.2 色彩学
色彩学是指通过感受、运动、合成和表达来传达信息的视觉系统的一门学问。主要研究对象的色泽和色调。在生物医学中，色彩学所涉及的对象主要是表皮和组织的各个部位，以及遗传信息中隐含的生物特征。生物染色体的编码顺序决定了染色体的颜色，即染色体直接影响到染色体基因的复制以及细胞的形态、生理活动及免疫力等生物学特性。

## 2.3 光学
光学是指描述光及其传播的方式的一门基础学科。在生物医学图像学中，光学主要研究视网膜与电脑屏幕等感光元件的结构和工作原理。在生物医学实验室里，常用到的光学设备有电子显微镜、照相机、激光笔等。

## 2.4 空间位置关系
空间位置关系是指一维、二维、三维、四维或高维的坐标系以及其之间的变化关系。空间位置关系在生物医学图像学中有重要的作用。常见的空间位置关系如三维空间的立方体、球面、四面体等；二维平面上的网格、图案以及轴线等；一维曲线和直线等。

# 3. X 射线衍射原理及特点
X 射线衍射的原理是利用外来的高能辐射导致的光在物体表面上的形成。光在物体表面的形成有两种方式，一是直接吸收进入体表的光，称为吸收衍射；另一种是反射扩散出去，称为反射衍射。当 X 射线穿过薄膜或其他感光材料时，由于其太阳光子的漂移，会被聚焦到靶物表面并发生强烈的衍射效应，即发生 X 射线衍射。

X 射线衍射是一种特殊的电磁辐射现象，它发生在材料表面，对其表面产生的辐射称为“衍射辐射”。这种辐射具有以下特点：

1. X 射线衍射在时间上是连续的，不间断地产生。在单位时间内，一个物质上只有一部分的衍射辐射能量转化成物质的能量。

2. X 射线衍射是直线传播的。在没有任何干扰的条件下，X 射线在大多数方向上的衍射效应是均匀的，这就是说，X 射线从某个点源射入物体，在接近入射点的方向都能产生相同的衍射效应。

3. X 射线衍射并非均匀分布的，它呈现出“随机”的行为。X 射线并不是按照一定规则在材料的各个部分上发生衍射。实际上，X 射线从某一点源射入之后，会在整个材料上按照一定的概率出现衍射。这一随机性使得衍射的效果具有不可预测性。

4. X 射线衍射不会主动撤退，它会一直存在于材料表面，直至完全消散。

5. X 射线衍射能够破坏真空隔离的结构。在传播过程中，某些材料表面可能会出现“耦合”，从而使得一些衍射带来的电磁辐射穿过两个结构的界限，这就可能损害隔离的结构。

一般来说，对于没有发生衍射的区域，有些生物学特征如染色体的结构、蛋白质分子的排列，是难以察觉的，但对另一些生物学特征则很容易察觉。例如，在血液中分子的排列、染色体的结构，都可能会受到 X 射线的影响而发生变化。而对于耳旁腺素的活性，如果它是通过 X 射线改变，就会显著改变呼吸系统的平衡。

由于 X 射线衍射会破坏真空隔离的结构，因此，临床上往往采用与 X 射线同种类的探测器（如超声波探测器）来代替普通的 X 射线探测器。这样可以避免由于 X 射线衍射带来的干扰而影响正常的病理检查。

# 4. 透射电子显微镜原理
透射电子显微镜（Transmission Electron Microscopy, TEM）是一种利用X射线传输阵列激发的自由电子共振信号进行晶体结构和矿物化学结构的结构和过程分析的一项高分子成像技术。TEM 可以在低剂量下快速成像，并且具有比传统成像方法更好的对真核生物表层的探索能力。

TEM 的基本原理是将X射线的离心辐射加速为真空信号，然后经由透射镜形成的衍射线路释放出来。透射镜具有小孔成像孔径，通过收集不同的离心电子波粒子，再通过反向过程得到原来的离心辐射。因此，透射镜既可以利用X射线阵列释放高分子分子团，又可以高精度的获得原有的离心辐射信号。

TEM 技术的关键在于如何获取有效的X射线阵列，可以选择的方法有高功率激光源、高功率光刻胶（HAL）、凝胶器件、微管成像器和传统晶体成像技术等。但由于要求X射线的辐射功率远大于激光源的辐射功率，同时也需要在保证结构准确性的前提下保证拍摄速度，因此TEM仍然是一个不成熟的成像技术。

# 5. EMTM 的原理和特点
电子显微镜（Electron Microscopy, EM）是利用电子束中的电场强度分布对单一原子、分子或整个晶体结构等进行结构和过程分析的分光光学实验。它可以对细胞、组织、体液和气体等样品的原子或分子的微观结构和动态过程进行全程跟踪观察。

透射电子显微镜（Transmission Electron Microscopy, TEM）是在EM技术的基础上发展起来的一种全新技术，其原理类似于衍射镜技术，可以把高能辐射通过电子束形成的离心辐射转变成无噪音、高质量的晶体镜像，并进行结构和过程分析。TEM可以利用各种不同的电子探测仪器和优化参数，实现在低剂量情况下高精度地解析结构、过程以及细胞或组织的微观结构。

TEM与传统的EM技术不同之处在于：第一，TEM通过调整电子束来生成特定类型的无偏离散电子束，从而可以高效、准确地模拟原子内部的电子结构。第二，TEM能够解析分子的相互作用过程，包括键相互作用、同位素键相互作用、胶体键相互作用、相互作用修饰以及螺旋相互作用等。第三，TEM不依赖金属表面，因此可以成像低分子到微米尺度下的细胞核、组织和体液。第四，TEM有助于揭示细胞、组织以及体液的蛋白质结构、酶配体化以及异突触等过程。

TEM的特点主要有：

1. 晶体成像性能高。TEM可以在低剂量下快速成像，而且结构清晰、分辨率高。

2. 非侵入性探测。TEM不依赖于体液、细胞、组织外壳以及成像介质的刺激，因此可以长期保持局部清洁和纯净状态。

3. 可定制化探测。TEM的各种参数可通过计算机控制，因此可以根据研究者的需求灵活调整。

4. 对真核生物表层的探索能力强。TEM探测真核生物表层的结构时，具有与传统晶体结构实验一样的成像精度。

5. 拥有广泛的应用。TEM技术广泛用于结构和过程分析、高通量分子筛选、基因组工程、细胞遗传学、生物技术、细胞计数等领域。
