# 1.简介
  

## 1.1 什么是云计算
云计算（Cloud Computing）是一种通过网络将数据、应用、服务等资源高度可扩展、可靠地提供给用户的一种计算模型。云计算服务能够让用户通过互联网随时访问所需的数据、软件和服务，并在需要时按需付费。云计算可以帮助客户在更短的时间内获得更多的能力、服务和收益，从而提升竞争力。目前，全球共有超过70个国家和地区实施了世界上最大的云计算中心，如亚马逊、微软、谷歌、IBM、HP、百度等。云计算的主要特征包括以下四点:

1. On-demand self-service: 用户可以根据需要快速配置和部署应用程序，灵活应对各种业务需求；
2. Broad network access: 用户可以通过任意设备、任何位置访问数据和应用程序；
3. Resource pooling: 云计算平台通过共享硬件资源池的方式降低成本；
4. Rapid elasticity: 可以快速弹性伸缩，适应不断变化的业务需求。

## 1.2 为什么要采用云计算
### （1）节约开支
云计算可以使企业节省大量的硬件投入、IT运维成本、服务器维护时间、采购物品成本、债务利息等。利用云计算服务，公司无需投入大量资金购买高性能的服务器，只需支付按使用的实际算力费用即可。并且，如果公司有需要，也可以按需调整硬件配置、数据规模和业务需求，同时确保服务质量。因此，采用云计算服务将有助于减少公司的财务压力，增加企业竞争力。
### （2）灵活应对各种业务需求
由于云计算服务具备高度的弹性和自动化特性，使得公司能够快速适应各种业务需求。比如，可以利用云计算平台快速搭建新产品或服务的测试环境，同时，也无需担心硬件配置过于复杂的情况下进行重构。云计算平台还能够提供广泛的存储服务，使公司可以根据需要迅速扩容。因此，采用云计算服务能够帮助企业应对各种业务变革，实现科技创新与市场转型。
### （3）实现跨区域或跨国分布式办公
由于云计算服务将基础设施作为服务的形式提供，用户可以轻松的构建起多层次的分布式应用系统，满足跨区域或跨国分布式办公的需求。公司不必再为传输、存储、计算、网络等环节的跨越而耗费大量的人力物力，而是可以依赖云计算服务的基础设施能力，简单快速地完成各种业务需求的实现。
### （4）快速响应业务发展
公司无需等待新硬件的到货、IT部门的扩张及运维人员的上岗，就可以利用云计算服务快速响应业务发展，提高竞争力。对于那些在成本和效率方面无法提升竞争优势的行业，采用云计算服务尤其重要。
# 2.云计算相关概念及术语
## 2.1 相关概念
### （1）虚拟机 Virtual Machine (VM)
虚拟机是一种通过软件模拟硬件功能的方法，它允许多个用户同时运行同一个操作系统，并且能够保证每个用户看到的环境都是一致的。虚拟机中通常包含操作系统、应用、文件和设置，可以像实际机器一样使用。虚拟机技术通过对底层硬件资源的抽象，为用户提供了便捷的使用计算机资源的方式。
### （2）云计算 Cloud Computing
云计算（Cloud Computing）是一种通过网络将数据、应用、服务等资源高度可扩展、可靠地提供给用户的一种计算模型。云计算服务能够让用户通过互联�网随时访问所需的数据、软件和服务，并在需要时按需付费。云计算可以帮助客户在更短的时间内获得更多的能力、服务和收益，从而提升竞争力。云计算最主要的特征包括：

* On-Demand self-service: 用户可以根据需要快速配置和部署应用程序，灵活应对各种业务需求；
* Broad network access: 用户可以通过任意设备、任何位置访问数据和应用程序；
* Resouce pooling: 云计算平台通过共享硬件资源池的方式降低成本；
* Rapid Elasticity: 可以快速弹性伸缩，适应不断变化的业务需求。

### （3）云服务 Cloud Services
云服务是一个按需提供的服务，可以实现包括数据存储、计算、网络等核心功能。云服务经过云供应商的托管和管理，用户只需要关注业务应用开发和运行，不需要考虑基础设施的管理。云服务又分为两种：公有云 和 私有云。公有云是指由公众可用的第三方提供的云服务，例如 Amazon Web Services (AWS)，Google Cloud Platform，Microsoft Azure 等。私有云则是指自己内部拥有的云服务，由自己的 IT 团队负责运营和管理。

### （4）公有云 Public Cloud
公有云是由第三方提供的云服务。公有云服务一般具有高可用性、安全性、可扩展性、灵活性和价格折扣优惠等优点。公有云服务可以按需使用，使用户享受到按需付费的便利。公有云服务可用于提供大量的基础设施，例如，服务器、存储、数据库、网络、IT 管理工具、基础软件等。因此，公有云可以降低云服务的成本，加快云服务的推广和普及。

### （5）私有云 Private Cloud
私有云是指自己内部拥有的云服务。私有云相比于公有云，具有更高的性能、可靠性和安全性。私有云主要由自己的数据中心进行托管和管理。私有云可以作为公司内部数据中心的替代，提供完整的自助服务，包括云计算、网络、存储、应用等一站式解决方案。

## 2.2 相关术语
### （1）IaaS Infrastructure as a Service
IaaS 是一种提供计算、网络、存储等基础设施即服务的商业模式。IaaS 服务提供者通过网络向最终用户提供计算资源、网络连接、数据存储等基础设施，最终用户只需要关心业务应用的部署和运行，不用再操心底层服务器、网络、存储这些基础设施的细节。IaaS 服务的典型代表就是 AWS EC2、Azure VMWare、Google Compute Engine。

### （2）PaaS Platform as a Service
PaaS 是一种提供平台即服务的商业模式。PaaS 服务提供者基于云端或本地环境提供软件编程接口（API），使得最终用户可以方便快捷的部署和运行各种应用程序。PaaS 服务的典型代表就是 Google App Engine、Heroku PaaS、Red Hat OpenShift。

### （3）SaaS Software as a Service
SaaS 是一种提供软件即服务的商业模式。SaaS 服务提供者基于云端环境提供完整的软件系统，包括应用程序、数据、基础设施和支持服务。最终用户只需要购买服务授权许可证，就可以获得服务。SaaS 服务的典型代表就是 SalesForce、Office 365、GitHub。
