## 3.3. 核心代码实现

```python
# 导入需要使用的库
import hcrrm
from hcrrm.api import Client

# 创建酒店客户关系管理系统实例
client = Client(
    url=hcrrm.settings.base_url,
    api_key=hcrrm.settings.api_key
)

# 导入客户数据
customers = client.groups.list_customers(['name', 'email', 'phone'], where={'name': 'guest'})

# 创建客户标签
labels = []
for customer in customers:
    label = {'name': customer['email'],'status': 'active'}
    labels.append(label)

# 设置处理规则
rules = []
for label in labels:
    rule = {'action':'mail', 'type': 'client', 'label': label}
    rules.append(rule)

# 发送客户邮件
client.rules.apply(rules, customers)
```
