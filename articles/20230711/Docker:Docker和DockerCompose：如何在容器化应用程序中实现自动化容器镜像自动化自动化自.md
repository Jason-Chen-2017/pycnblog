2. 技术原理及概念

2.1. 基本概念解释

2.1.1. 镜像（Image）

镜像是 Docker 中的一个概念，指的是容器映像。容器映像是一个只读的文件，用于描述应用程序的 Docker 镜像。Docker 镜像是由 Docker Hub 上的官方镜像和第三方镜像组成的。

2.1.2. 容器（Container）

容器是 Docker 中的一个概念，指的是 Docker 镜像的运行实例。容器是 Docker 的轻量级、可移植的虚拟化技术，允许应用程序在不同的环境中运行，而无需修改代码。

2.1.3. 仓库（Repository）

仓库是 Docker 中的一个概念，指的是 Docker 镜像的存储场所。Docker 仓库是一个中央存储库，用于保存和管理 Docker 镜像。

2.1.4. Dockerfile

Dockerfile 是定义 Docker 镜像构建规则的文件。Dockerfile 是一种文本格式，描述了如何构建 Docker 镜像。Dockerfile 支持 Dockerfile 语句，包括构建镜像的命令、构建镜像的步骤、镜像标签等。

2.1.5. Docker Compose

Docker Compose 是 Docker 官方提供的用于定义和运行多容器 Docker 应用程序的工具。Docker Compose 使用 YAML 文件编写，用于定义应用程序中的各个服务
