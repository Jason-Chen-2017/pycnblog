a * P + b * P + c * P + d * P + λ * (b * (P - a) + c * (P - b) + d * (P - c)) = q

其中，λ表示权值，权值为1表示每个虚拟服务器的重要性，权值为0表示每个虚拟服务器的重要性为0。

(2) 反余弦定理：

[a + b + c + d - λ * (b + c + d)] / 2 = P / q

其中，λ表示权值，权值为1表示每个虚拟服务器的重要性，权值为0表示每个虚拟服务器的重要性为0。
