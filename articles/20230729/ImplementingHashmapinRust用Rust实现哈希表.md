# 1.简介
         
 在计算机科学中，哈希表（Hash table）是一种根据键值对进行存储的数据结构。这种数据结构利用哈希函数将键映射到存储位置上，有效地避免了查找元素的时间复杂度过高的问题。哈希表的实现可以简单、快速、高效，因此在现代软件开发中有着广泛应用。

          本文将使用 Rust 语言实现一个简单的哈希表，并详细阐述其基本原理。我们还将探索一些 Rust 的特点及它们带来的好处，来看看这些特性是否能够帮助我们更好地理解哈希表的工作原理。最后，本文会给出一些扩展阅读材料，让读者了解到更多关于哈希表的知识。

          ```rust
          fn main() {
              let mut map = HashMap::new();

              // Insert a key-value pair into the hash map
              map.insert("A", "Apple");
              map.insert("B", "Banana");
              map.insert("C", "Cherry");
              map.insert(1, true);
              
              println!("{:?}", map);
              
              // Get the value of an existing key from the hash map
              assert_eq!(Some(&"Apple"), map.get("A"));
              
              // Accessing non-existent keys returns None
              assert_eq!(None, map.get("D"));

              // Iterate over all key-value pairs in the hash map
              for (key, value) in &map {
                  println!("{}: {}", key, value);
              }
          }
          ```
