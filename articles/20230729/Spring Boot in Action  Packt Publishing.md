# 1.简介
         
　　Spring Boot in Action（中文版）是Packt出版社出品的一本Java相关的技术书籍。它的主要作者是：克里斯托弗·里夫斯（<NAME>），是一位精通Java、Spring、Spring Boot等技术的资深软件工程师和项目经理。该书着重于对Spring Boot进行全面的系统性讲解，系统地讲解了Spring Boot各个方面、框架特性及最佳实践。《Spring Boot in Action》还兼顾了Spring MVC、Spring Security、Hibernate等其他技术细节，并结合实例向读者展示如何通过实际案例来实现相应功能。读者可以快速、轻松地理解Spring Boot所提供的各种便利特性。
         《Spring Boot in Action》由以下三章组成：
             一、Spring Boot基础知识
             二、Web开发
             三、企业应用
         本书适用于需要学习Spring Boot入门的Java开发人员和工程师。无需过多编程经验的读者亦可快速上手并运用其中的知识解决实际问题。

         ## 作者简介
             　　克里斯托弗·里夫斯（<NAME>）是一位精通Java、Spring、Spring Boot等技术的资深软件工程师和项目经理，是Spring技术团队成员之一，同时也是Spring官方认证讲师，是国际知名技术博客Spring Framework 3.0官博的主要撰稿人。他先后就职于几家公司，包括Oracle、Tivoli以及Accenture等领先的IT服务提供商。此外，他还是培训机构JaBa和Manning Publications的专家。
              
          　　为了帮助更多Java开发人员掌握Spring Boot，克里斯托弗·里夫斯精心设计了本书，它涵盖了Spring Boot的所有方面，从基础教程到最佳实践，让读者能够一站式掌握Spring Boot的基础知识、核心特性、高级特性及最佳实践。除此之外，书中还配有详细的代码示例，帮助读者更快、轻松地理解Spring Boot的工作机制。
            
            　　作为Spring系列技术图书的推广大使，克里斯托弗·里夫斯深受读者的青睐。2019年，他获选为Spring官方认证讲师，并授权出版图书，这使得本书迅速成为最受欢迎的Java技术图书。在美国和全球，《Spring Boot in Action》也已印刷多次，销售量超过40万册。
