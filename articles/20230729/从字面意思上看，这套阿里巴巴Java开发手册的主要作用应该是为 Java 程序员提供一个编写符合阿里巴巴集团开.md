# 1.简介
         
 编程语言是软件工程不可缺少的一部分，作为高级工程师，必定会涉及到语言基础、框架应用、数据结构等方面的知识。而其中之一就是 Java 语言。那么为什么需要有这样一份 Java 开发手册呢？原因是由于 Java 在各个领域都有很广泛的应用，包括企业级应用、互联网应用、移动应用、游戏、嵌入式应用等，因此编写一本适合所有 Java 开发人员阅读、遵守的开发规范可以有效地提升项目质量、降低成本，提升个人竞争力。此外，借助这份 Java 开发手册，可以帮助工程师更快地理解 Java 语法、踏入优秀开源项目中进行实践和总结，进而培养对其深刻的理解能力。
           据统计，目前全球有超过三分之一的 Java 开发人员，至少具备一年以上 Java 相关工作经验；而 Java 是面向对象的语言，具有高阶特性。为了帮助更多 Java 开发人员熟练掌握 Java 语言，编写出更高质量的代码，阿里巴巴集团推出了《阿里巴巴Java开发手册》（详见 http://www.alibaba.com/product/java-coding-guidelines.html）。
          此外，阿里巴巴集团也致力于建立一支拥有丰富项目实战经验的技术专家队伍，将 Java 开发手册与日常业务场景相结合，通过对常见问题和实际工作中遇到的疑难问题进行解答，帮助广大的 Java 开发人员加深对 Java 的理解和技能掌握。

          - 《阿里巴巴Java开发手册》的未来发展方向如何？
          有了《阿里巴巴Java开发手册》，就可以根据自己的实际情况制定一套属于自己的代码规范。比如，对于某些公司要求严格遵循某种代码规范，可以基于《阿里巴巴Java开发手册》进行改版和补充，重新审视自己的代码，减轻审核压力，提升工作效率。这样，既可以保持代码质量，又可以受益于《阿里巴巴Java开发手册》所提供的指南。
