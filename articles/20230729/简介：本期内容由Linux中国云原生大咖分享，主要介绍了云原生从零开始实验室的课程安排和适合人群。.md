# 1.简介
         
2020年是互联网时代的来临，无论是社交媒体、网络游戏还是电子商务都迎来了一个全新的互联网时代。随着容器技术、微服务架构的发展，以及DevOps、机器学习等新兴技术的出现，云计算技术也在蓬勃发展。云原生（Cloud Native）是一种构建和运行可弹性扩展的应用的新型态技术，它将应用程序作为服务部署到云环境中，通过自动化工具、管理框架及工具链来提升应用的性能、可靠性及安全性。它是基于云平台的一个软件开发方法论，旨在利用云平台提供的基础设施资源、软件定义网络以及自动化服务，来实现应用的高可用、可伸缩性和可靠性。在Kubernetes和Istio等开源技术的驱动下，越来越多企业和组织选择在公有云或私有云上采用云原生技术进行应用开发。本期的分享主要围绕云原生技术发展方向展开，介绍云原生从零开始实验室的内容，以及适合学生阅读和参加的人群。
         # 2.云原生定义
         “云原生(cloud native)”一词源自于希腊语“κόρος”，意指“混沌”，其含义表明软件系统应当对其运营环境的变化及其底层基础设施等因素做好应变准备。云原生这个概念最早由敏捷软件开发方法论的创始人们提出，并且有着多个实践者的共同理想。云原生关注的是云平台如何让应用开发人员不必关心底层平台的复杂性，而可以专注于业务逻辑的实现。云原生具备以下特征：
         - 以容器技术为基础
         - 消除语言绑定
         - 使用可观测性（Observability）
         - 服务导向（Service-oriented architecture，SOA）
         - 自动化（Automation）
         在IT技术领域，云原生代表了一种服务导向的设计思想，旨在构建能够良好运行于云端的分布式应用。云原生技术最初是由微软、Google、Red Hat、IBM等公司共同提出的。截至目前，很多公司都已经转型为云原生阵营。从这个角度看，云原生并非某个技术的替代品或取代品，而是一个融合了众多技术的整体运用方式。不同公司所采用的方法论各有不同，但云原生却提供了一种全面、可持续的软件开发方法论。
         # 3.课程安排
         本期内容将分两天进行，第一天将介绍云原生的发展历史以及相关名词的概念和解释。第二天则会结合实际案例，以编程的方式让读者亲手实践云原生相关技术。课程的结构如下：
         1. 云原生简史（李开复老师）：介绍云原生相关理论和名词的基本概念，并介绍云原生的发展历史。 
         2. Kubernetes入门和实践（刘晓宇老师）：通过实战案例介绍Kubernetes的基本概念、安装配置及最佳实践。
         3. Istio入门和实践（吴晟老师）：通过实战案例介绍Istio Service Mesh 的基本概念、安装配置及最佳实践。
         4. Prometheus监控实践（陈刚老师）：介绍Prometheus的基本概念，并通过实际案例展示如何通过Prometheus搭建集群监控、服务监控、自定义告警规则。
         5. Fluentd数据采集实践（周俊老师）：介绍Fluentd的基本概念，并通过实际案例展示如何搭建数据采集 pipeline 和自定义日志处理规则。
         6. Serverless实践（彭泽平老师）：通过实战案例介绍Serverless基本概念、AWS Lambda函数服务、Azure Functions服务、Google Cloud Function服务的使用及最佳实践。
         7. Knative实践（李文瀚老师）：通过实战案例介绍Knative基本概念、Serving和Eventing组件的使用及最佳实践。
         8. 函数式编程实践（刘佳慧老师）：介绍函数式编程的基本概念，并通过实际案例展示Java函数式编程及Python函数式编程的区别及使用场景。
         # 4.适合人群
         对于所有感兴趣、对云原生技术感兴趣，想要从零开始学习，想通过实践获取经验的学生，本次课程非常适合。由于每位讲师都有丰富的实践经验，因此能极大地提高大家的学习效率和积累，特别是那些希望在职场、工作中立足云原生技术的人。当然，课程内容覆盖面广，内容也将持续更新，敬请期待。
