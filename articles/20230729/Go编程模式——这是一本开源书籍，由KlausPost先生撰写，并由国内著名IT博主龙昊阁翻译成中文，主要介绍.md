# 1.简介
         
 随着云计算、微服务架构、容器技术的流行，Go语言在高性能、并发性和易学习性等方面都赢得了广泛关注。作为Go语言最具代表性的语言，它的简单易用、灵活的特性吸引了众多开发者投身其中，成为了构建云端系统的首选语言。因此，掌握Go语言中最常用的模式，可以帮助开发人员设计出更高效、更健壮的软件应用。

          在互联网时代，很多公司都选择将现有的业务迁移到云端平台，而Go语言自带的高并发和轻量级进程模型使其非常适合进行海量并发处理和实时的计算。而Go编程模式则可以帮助开发人员设计出具有可扩展性和弹性的软件系统。

          Go编程模式，是一本开源的书籍，由Klaus Post先生（以下简称Klaus）和他的同事们共同撰写。该书通过丰富的示例和详实的讲解，从多个角度对各种Go编程模式进行了分类整理，并给出了详细的代码实现。对于想要提升自己的Go编程水平、熟练掌握Go编程技巧和理论有极大的帮助。

          除了帮助读者快速掌握Go编程模式外，Klaus还将中文版的书籍翻译成了简体中文，力求为中文Go程序员提供便利。该书内容覆盖了Go语言的基础语法、并发模型、接口设计、数据库访问、Web编程等方面，值得读者参考和借鉴。

          本书原作者Klaus和他的团队都是Go语言爱好者、优秀的开源贡献者和布道者。他们曾在几乎所有大型互联网公司担任开发工程师、架构师或CTO角色，包括Google、Facebook、Netflix、Amazon、IBM、微软、腾讯、百度等。他们也出过许多关于Go语言的著述或演讲。Klaus Post先生非常欣赏他人的思想和见解，喜欢分享和探索，这也是他创作的另一重要原因。

          通过阅读和实践，读者能够逐渐形成Go编程模式的理解和能力，从而提升自己的能力和知识水平。通过阅读《Go编程模式》，你可以学会：

           - 使用Go语言创建多种类型的应用；
           - 更好地控制内存分配和垃圾回收机制；
           - 了解如何有效利用多核CPU资源；
           - 解决复杂的问题和构造抽象层；
           - 汇总不同模式的优点和局限性，帮助自己选择最适合的模式。

          如果你是一个经验丰富的Go语言开发者，但又对一些特定模式感兴趣，也许《Go编程模式》能够帮助你更好地理解和运用这些模式。如果你的目标是在现有业务上进行微服务化改造、面向云原生应用架构演进，或者只是为了让自己的编程水平更上一层楼，那么《Go编程模式》是一个值得一读的宝贵资料。
