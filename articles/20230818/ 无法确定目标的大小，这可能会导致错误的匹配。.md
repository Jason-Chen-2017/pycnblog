# 1.简介
  

今天，我将带领大家一起分享关于如何确定目标大小的问题，为什么该问题如此重要以及如何解决这一问题。在这个过程中，希望能够帮助你更好地理解目标定义、评价指标等。
首先，让我们来看一下目标是什么？目标一般分为两个层次。第一层次是业务层面的目标，比如销售量达到预期，订单完成率提升10%。第二层次是产品层面的目标，比如App下载量超过1亿，软件功能模块开发完成，用户满意度达到99.99%。
然而，如何衡量一个目标的大小，是一个难题。每个公司都需要制定目标，但往往难以界定清晰，尤其是在复杂情况下。通常情况下，目标的度量方法会根据实际情况制订出一套理论，然后依据规律进行判断。这套理论也非常复杂且多样化。比如，销售部门的目标可能用PPT展示方式进行，产品部门的目标则可能采用KPI（Key Performance Indicator）的方式。
但是，随着时间推移，一些影响目标定义的因素越来越多，这些因素包括市场竞争、客户反馈、外部压力等等。这些因素的变化都会影响目标的定义。那么，如何合理有效地处理这些影响呢？下面，我将结合实际案例，分享几种不同情况下，如何对目标进行确定，并阐述如何避免错误的匹配。
# 2.如何确定目标大小
如何确定目标大小，从本质上来说，就是找到一种衡量标准。既然要衡量目标，那么就需要有一个参考系，在这个参考系里，对目标的定义进行共识，比如说，目标达到多少才算成功。因此，目标的确定首先涉及到两个方面。第一个方面，如何定义目标。第二个方面，如何实现目标。
## 2.1 定义目标
定义目标可以简单粗暴，比如“促进两岸交流”，“建立先进的金融模式”。但不推荐这样做，因为这样做忽视了目标背后的具体指标。定义目标应该具体、充实、具有内在联系。比如“2020年前，我国城镇化率达到75%”、“2020年前，我国GDP总值增长率超过10%”。这样才能帮助你精准地量化目标，实现目标时更有动力。
## 2.2 度量目标
度量目标的方法很多，例如可以采用绩效考核、KPI考核、数据分析、竞品分析等。每种方法都各有优劣，需要结合自己的业务场景进行选择。
### 方法一：绩效考核
这种方法适用于快速迭代的业务系统，其中存在明确的指标，只需要测量当前的性能即可。但是，由于目标是不确定的，所以很容易被主观意愿所干扰。另外，由于没有客观标准，这种方式也不能体现全局的状况，只能局部地看到某个领域或个人的表现。因此，这种方式适用于不具有全局性影响的业务系统。
### 方法二：KPI考核
这种方法适用于具有稳定性要求的业务系统，其指标能够反映出业务的整体水平。这种方法的好处是客观可靠，且可以从整体上看到业务表现的全貌。同时，KPI考核还可以比较不同维度下的表现，并反映出业务的不同阶段成果。但是，KPI考核的方法也有很多局限性。
### 方法三：数据分析
这种方法通过对数据进行分析，找寻系统的瓶颈，找出改善点。通过分析数据可以得出理论上可行的方案，再根据情况进行调整。这种方法不仅能找到问题所在，还能给出行动方案，因此很适合复杂、动态的业务环境。
### 方法四：竞品分析
这种方法基于对目标的比较，找寻该行业或产品的竞争对手，进一步完善自己的目标。也可以发现同类产品的差距。这种方法虽然适用于比较大的领域，但是对于小型公司或新生事物可能无法奏效。
### 方法五：客户反馈
这种方法主要基于客户的意见。客户可以向你提供各种建议，包括问题和需求，以便为你提供指导。这种方法可以从多方面获取信息，可以获得更多的视角。但是，客户反馈不能直接对目标产生影响。
## 2.3 实现目标
实现目标的方法也是很多的，例如，可以采用积极的方法对产品进行改进、采用传统的管理方式进行管理等。不同的方法各有利弊。但最终还是需要落实到具体的执行上。比如，产品开发中，可以采用Scrum、Agile等敏捷开发方法；对市场营销策划，可以采用一些数据驱动的策略来创造更多的收益。
# 3.如何避免错误的匹配
目前，有很多企业已经采用了目标管理的方式。但由于目标管理的方式太过重视目标的定义、度量、实现等步骤，往往会让员工产生误解，甚至失去方向感。正确的做法是，应该从管理层面思考问题，不要纠结于目标的定义、度量、实现等步骤。目标的最终目的不是为员工设置目标，而是为了提高组织整体能力，加强沟通协作，共同打造更好的工作环境。所以，目标管理中，要做到规范管理，不断提升管理水平，减少决策误差。
