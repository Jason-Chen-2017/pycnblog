# 1.简介
  


在OCR（Optical Character Recognition）过程中，图像的背景信息和文本信息存在着高度重叠的情况。对于字符级别的精确识别，这种重叠信息是非常重要的。一般来说，一些技术如HOG（Histogram of Oriented Gradients），SIFT（Scale-Invariant Feature Transformations）等都是用来检测图像中的局部特征的，这些局部特征描述了图像中的纹理、形状和几何形态，从而对其进行分类。然而，局部特征与全局空间信息之间仍然存在着紧密的联系。因此，如何结合局部特征与全局空间信息是OCR领域的一个重要研究方向。

本文介绍了一个基于CNN（Convolutional Neural Network）的方法，该方法通过结合空间信息和局部特征来提升字符级的精确度。

# 2.相关工作

目前已经有许多关于融合空间信息和局部特征的OCR方法，如SSFAM，GEOCONV，BLOF，RANSAC，LSD等。但是这些方法都只能得到较高的准确率，不足以适应实际应用。本文认为，当前最优秀的OCR方法应该同时兼顾空间信息和局部特征。为了解决这个问题，本文设计了一种基于CNN的深度学习模型，该模型能够把视觉信息和空间信息结合起来，并且预测文本序列。

首先，CNN具有良好的处理图像数据的方式，能够提取图像中空间上的丰富特征，如颜色、结构、边缘等。而且，它能够对图像中不同区域之间的上下文关系做出反映，可以帮助定位文字位置。

其次，空间信息可以使用全局或局部图像上下文特征来表示，如全局线性变换、局部光流场、全局特征点匹配、全局描述子等。用全局空间信息对图像进行编码可以获得更强的全局特征，用局部空间信息可以获得更好的文字识别性能。

第三，局部特征可以使用局部图像统计信息和统计描述符（如直方图、SIFT、HOG、GLCM等）进行表示。局部特征有助于字符识别，因为它能够描述图像的细节信息，使得模型能够更好地区分单个字符。

最后，本文采用全卷积网络（Fully Convolutional Networks，FCN）作为主要网络结构，它能够学习到有效的特征组合策略。FCN利用底层的预训练神经网络对图像中的所有感兴趣区域（例如，像素块）进行分类，并产生相应的概率图，然后将其上采样到原始尺寸，最终输出文字区域的分类结果。本文通过最大池化层将特征图逐渐缩小，并将它们与输入图像一起送入下一层，从而增强特征组合的能力。

综上所述，本文提出的基于CNN的新型OCR方法旨在结合局部特征和全局空间信息，并通过学习有效的特征组合策略，提升文字识别的精度。

# 3.相关概念

## 3.1 CNN

CNN是一个深度学习模型，用于计算机视觉领域。CNN由多个卷积层和池化层组成，每一层是具有一定大小的滤波器和非线性激活函数的卷积层，并通过最大池化层减少网络的复杂度。网络输入是一张彩色或灰度图像，输出是一组预测值，即每个位置的标签或置信度。

## 3.2 FCN

FCN（Fully Convolutional Networks，全卷积网络）是深度学习中一个典型的卷积神经网络。它的特点是输入图片和输出特征图具有相同的大小，且输出的每个像素都对应于输入图像中的一个局部区域。这给FCN带来了很大的便利，因为它不需要手工设计特征选择策略，而是自动学习到特征组合的策略。

全连接层变成了全卷积层，意味着卷积层的权值共享被取消掉，全部用卷积核参数代替。卷积核的大小是输入图像的大小，步长等于1，这样就可以保证输出特征图和输入图像具有相同的尺寸。之后，再把输出的特征图上采样到原始输入图像的尺寸，就得到了最终的预测。
