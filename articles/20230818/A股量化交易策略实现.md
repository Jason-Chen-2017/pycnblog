# 1.简介
  

## 1.1 A股市场概述
A股（American Stock Exchange）是美国证券交易所推出的第一只股票市场，也是目前世界上规模最大的股票市场之一。截至2021年5月末，在中国开设的A股总市值已达到1322.7万亿元。随着互联网金融、数字货币等新技术的发展，A股已经成为国际主要的股票交易平台，并且越来越多的投资者和机构开始参与其中。
## 1.2 什么是量化交易策略？
量化交易策略是通过计算机技术分析市场行情并对其进行量化分析从而设计出符合自己预期的交易策略。简单的说，量化交易策略就是利用计算机技术去发现价值投资的真谛。
比如，阿尔法狗交易系统就是一种量化交易策略。它是一个机器学习算法，基于历史数据，结合个人风险偏好，选取适合自己的资产配置方案，帮助用户在短时间内实现盈利。
除此之外，还有很多其他的量化交易策略，如布林带交易、均线交叉交易、MA指标策略等。下面我们将介绍最常用的几种量化交易策略——EMAC策略、缠论策略、支持向量机策略、随机森林策略等。
# 2. EMAC策略
EMAC策略（Ease of Movement and Concentration）是由美国芝加哥大学Brian Ward教授提出的量化交易策略，是一种应用广泛的短期交易策略。该策略的特点是检测多空盘整状况，同时关注收盘价位的稳定性，以此判断是否有买卖空间。
## 2.1 EMAC策略简介
EMAC策略认为，市场中存在多个具有影响力的主动性商品，这些商品在短期内会形成强烈的价格震荡，这些影响力商品的平均价格水平可以反映整个市场的总体状态。因此，EMAC策略首先要获取主动性商品的信息，然后根据这些信息计算出平均价格水平、价格波动范围及趋势。
EMAC策略以三个参数作为基础，即能量指标（Elder’s Force Index）、多空比率（Mass Index）、移动平均线（Moving Average）。
### （1）能量指标Elder's Force Index
能量指标的计算方法是将一段时期内的收盘价之和除以平方根(N)乘以100，其中N表示这段时间的天数。这个值越大，则说明价格变化的幅度越大；如果这个值越小，则说明价格变化的速度越慢。一般来说，N=14或21天比较适宜。另外，还有一个可调节的参数K，它的大小决定了价格变化的敏感度。K的值越小，则能量指标的分量越小，反之亦然。
计算公式如下：
EF = [(CLOSE-LLV(LOW,N))/ (HHV(HIGH,N)-LLV(LOW,N))*SQRT(N)]*100
### （2）多空比率Mass Index
多空比率的计算方法是用收盘价的上涨日和下跌日的数量除以总天数，来衡量市场中的多空力量。这个值越高，则说明市场中的力量越强，市场的走势趋势可能会发生变化；如果这个值越低，则说明市METER存在弱势，市场的走势可能会暂时失衡，但可能就会回升或者转变方向。
计算公式如下：
MI = SUM((Close>Open?1:-1),N)/N
### （3）移动平均线Moving Average
移动平均线（MA）用来衡量一段时间内的平均价格水平，通常用简单移动平均线或权重移动平均线。它的计算公式为：
MA = (C+L+H)/3
其中，C为收盘价，L为最低价，H为最高价。

## 2.2 EMAC策略具体操作步骤
1. 根据历史数据，计算出能量指标和多空比率。
2. 若能量指标大于阈值（一般设置为50），且多空比率大于0.5，则进入买入区间；否则，进入等待区间。
3. 在买入区间，计算当前的移动平均线和前一日的移动平均线。
4. 如果当前的移动平均线下穿了前一日的移动平均线，则依据系统设置的止损价格，进行止损操作。
5. 如果移动平均线相当，则保持持仓不变；如果当前的移动平均线上穿了前一日的移动平均线，则清仓并重新计算新的移动平均线。
6. 当能量指标小于阈值（一般设置为30），且多空比率小于等于0.5，则退出买入区间。
## 2.3 EMAC策略的优缺点
### （1）优点
- 能够识别主动性商品；
- 可以快速响应市场行情，调整持仓；
- 有较好的抗风险能力，能够有效防范大跌，实现“无所不能”的长线布局。
### （2）缺点
- 操作复杂，需要考虑多空平衡，不易察觉；
- 对市场波动不敏感；
- 只适用于股票市场。
## 2.4 何时适合采用EMAC策略？
EMAC策略的应用场景主要有以下几种：
- 大盘走势形态无法被观测到的地方；
- 没有明显市场反弹的品种；
- 涉及多个品种的组合交易。
另外，EMAC策略有时也会面临一些潜在的问题，如：
- 判断反转可能存在误差；
- 由于对同一个市场的不同时期信息的处理方式不同，可能导致策略表现出现较大的差异。
综上，EMAC策略适用于那些有明确的趋势发展的品种，适合于炒作型品种；但对于那些没有明显趋势但是又有大量主动性的品种，例如黄金、白银等，采用EMAC策略可能面临风险。而且，EMAC策略不适用于股指、债券等市场，因为它们往往具有更复杂的规则。
