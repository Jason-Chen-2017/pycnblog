# 2.基本概念术语说明
## 概念
计算语言学（Computational Linguistics）是研究如何在计算机上处理语言的一门学科。它从人类语言和符号的生成、表达到人类的语言行为习惯、语法规则和词汇意义等方面进行探索。该领域涉及到的主要方法包括统计方法、规则方法、神经网络模型和深度学习模型等。而文本分析就属于这一领域的一个子领域。

## 定义
文本分析（Text Analysis）是指用计算机程序对大量的文本数据进行解析、分类、处理、检索、挖掘和可视化，从而发现其中的价值和潜在含义。通常情况下，文本分析所处理的数据包括非结构化的文本数据、图像数据、音频数据等。

## 技术
以下是计算语言学的一个重要分支——自然语言处理（Natural Language Processing，NLP）技术。

1. 词性标注（Part-of-speech tagging）
	- 将单词标记成其基本的词性，如名词、动词、形容词、副词等；
	- 通过词性标记，我们可以分析句法结构、识别命名实体、实现情感分析等；
	- 目前最流行的词性标注工具是Stanford POS Tagger、Pattern、NLTK等。

2. 命名实体识别（Named entity recognition，NER）
	- 根据上下文确定一个单词的“类别”、“种族”或“事物”，如人名、地点、机构名称、组织机构、时间和日期等；
	- 在金融、法律、政策领域有着广泛的应用；
	- 目前最流行的命名实体识别工具是Spacy、Stanza、MITIE等。

3. 情感分析（Sentiment analysis）
	- 对一段文字或微博进行正负面的情感分析，判断作者的态度倾向；
	- 可以用于营销策略、产品开发、舆情监控等领域；
	- 目前最流行的情感分析工具是VADER、TextBlob、Afinn、AFINN-165等。

4. 文本摘要（Text summarization）
	- 从大量文本中提取关键语句、主题，呈现出更简洁、紧凑的文字；
	- 可以用于新闻编辑、报道解读、产品介绍、评论调查等；
	- 目前最流行的文本摘要工具是GPT-2、Texygen、BERT-extractive-summarizer等。

5. 机器翻译（Machine translation）
	- 将一种语言的文本自动转化为另一种语言的文本；
	- 通常用于自动化生产过程，如从英语翻译成中文、从中文翻译成英语；
	- 目前最流行的机器翻译工具是Google Translate API、DeepL、Microsoft Translator、LibreTranslate等。

6. 拼写检查（Spell checking）
	- 检测和纠正拼写错误，帮助用户输入正确的文本；
	- 有助于改进输入法、辅助写作和降低拼写错误率；
	- 目前最流行的拼写检查工具是Hunspell、LanguageTool、Aspell等。

## 开源工具包
Python具有庞大的生态系统，几乎可以完成任何计算语言学相关任务。其中，比较著名的开源工具包如下：

1. NLTK（Natural Language Toolkit）
	- 一系列用于处理、操作和实现人工语言学习实践的工具包；
	- 支持多种自然语言处理技术，如词性标注、命名实体识别、语音和语言理解等。

2. Spacy
	- 基于开源的深度学习模型，适用于生产环境，并提供丰富的功能；
	- 提供多种功能，如词性标注、命名实体识别、句法分析、文本分类、情感分析、文本匹配等。

3. Stanford CoreNLP
	- Stanford CoreNLP是一个开放源代码的Java平台，实现了多项自然语言处理任务；
	- 包括命名实体识别、句法分析、语义分析、语音与语言理解等。

4. Gensim
	- 一个支持主题建模、文档建模、相似性计算、聚类、协同过滤等功能的Python库；
	- 可用于处理文本数据，包含了诸如Word2Vec、Latent Semantic Indexing等工具。

5. Keras
	- 一个基于Theano的深度学习库，具有易用性、性能和跨平台能力；
	- 提供了包括CNN、LSTM、RNN等不同类型的神经网络模型。
