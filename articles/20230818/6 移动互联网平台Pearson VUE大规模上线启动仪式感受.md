# 1.简介
  

从2017年5月开始到2019年11月，为了服务整个英国超过25亿消费者，移动互联网平台“Pearson VUE”在英国、美国、澳大利亚、新西兰、爱尔兰等多个国家和地区开启了大规模上线旅程。

如今，Pearson VUE仍在不断壮大，在新西兰、澳大利亚、加拿大、德国、法国、瑞士、比利时、波兰等几个国家也分别设立分支机构，帮助英国市场扩张。除了自身的研发外，Pearson VUE还依赖于其市场运营团队为客户提供解决方案。根据相关数据显示，截至目前，Pearson VUE在英国拥有100多万活跃用户。因此，这一过程涉及大量的人力和物力投入。
作为一个具有开拓性、大胆的企业，Pearson VUE认为自己还有很大的提升空间。在未来的发展过程中，他们期待能够获得更多的国际合作伙伴支持，并增强自己的知识产权。此外，公司还准备继续优化用户体验、增加收益、扩大市场份额、建立更好的营销机制。总之，Pearson VUE在移动互联网领域依然有极高的生命力和魅力，更值得期待！

# 2.背景介绍
## 2.1 Pearson VUE的起源
Pearson VUE（学业网），是一款基于云计算技术为教育培训机构和学校提供的一站式学习管理系统。2017年5月，Pearson VUE在英国启动，全称“Pearson VUE UK”。在仅短短两年的时间内，Pearson VUE已经吸引到了超过5万名用户，并且在多个国家市场均受到好评。截止到2019年11月，Pearson VUE在英国、美国、澳大利亚、新西兰、爱尔兰等多个国家和地区共计销售超过1500万套。Pearson VUE由一群年轻的创业者领导，拥有丰富的互联网开发经验，并致力于打造英国最优秀的移动应用商店。
Pearson VUE是一个大型企业，它的业务遍布全球。早在1998年，Pearson就开始向学校提供各种高质量的学业辅助服务，包括成绩单打印、期末复习、作业批改、专业测试、课后辅导、学业辅导等，这些都是全世界各地孩子们所需要的。但是，这个市场一直没有得到满足，于是在2004年才意识到要打造一家全新的移动学习管理系统。在很短的时间内，Pearson VUE便在英国市场占据了一席之地。在英国，该公司的营收已经远超其他一些IT咨询公司，而它在其他国家的布局则远非同行可比。Pearson VUE的布局策略也十分独特。首先，它不是独立的公司，而是Pearson UK的部分收益；其次，它不只是提供学业辅助，还参与到物联网、智能硬件、健康管理、零售等领域的研发；第三，它采用的是云端的服务模式，对学校内部系统进行整合，并通过Pearson UK提供的平台让学校的管理人员、老师、学生都能访问到Pearson VUE。
## 2.2 Pearson VUE的技术架构
Pearson VUE的技术架构主要分为四层。第一层为前端（Front-End）层，负责前端的设计、开发和性能优化工作。第二层为后台（Back-End）层，负责服务器的搭建、数据库的构建和配置，以及安全控制等工作。第三层为中间件（Middleware）层，负责消息的路由、缓存、搜索等功能。第四层为云平台（Cloud Platform）层，负责服务器资源的管理、部署、监控、备份等功能。
Pearson VUE使用基于React.js的单页面Web应用构建，客户端渲染的效果非常流畅，响应速度快。其使用的语言为TypeScript，可以有效提升代码的维护效率。同时，Pearson VUE还提供了浏览器插件和桌面客户端，方便学生和老师使用。

## 2.3 Pearson VUE如何做营销转化和销售？
Pearson VUE利用现代互联网技术推广产品，为消费者提供高质量的学习辅助服务。Pearson VUE拥有完整的销售渠道，其中包括线上销售渠道，例如学校网站、微型网站、微信公众号、微博账号等，还有电话销售渠道、网络营销渠道、实体展会、活动预告等。销售渠道覆盖了不同类型的用户，包括学生、老师、家长、媒体、销售人员等。
Pearson VUE目前有两种主要的销售方式。一种是B2B销售，即由销售人员直接跟学校或者学院的代表一起谈判，比如教授课程费用、赠送奖励等；另一种是B2C销售，即由Pearson VUE直接在个人消费者和企业主(学校或学院)之间谈判，比如购买课程、调取学习进度等。在所有的销售渠道中，Pearson VUE都选择直接上门的方式，因为它可以更快速地响应客户需求，而且不会拖延产品发布时间。Pearson VUE正在与第三方平台合作，为学习平台提供免费试用。
## 2.4 Pearson VUE如何通过AI提升用户体验？
Pearson VUE的AI系统可以提升用户体验。Pearson VUE使用机器学习和深度学习技术来构建用户模型，借助AI技术为用户提供更好的学习体验。AI系统包含了专家系统、自然语言处理、图像识别、推荐系统、个性化推荐系统等模块，目前已经应用在了Pearson VUE的产品中。在AI系统中，Expert System（专家系统）模块可以分析用户的问题、评论等内容，并回答出正确答案。Natural Language Processing（自然语言处理）模块则可以分析用户的话题、关键词，并生成相应的内容。Image Recognition（图像识别）模块可以分析用户上传的图片，判断图片中的题目类型，并自动生成相应的作业。Recommendation System（推荐系统）模块可以根据用户的历史行为、偏好等，为用户推荐相关的课程或资料。Personalization Recommendation System（个性化推荐系统）模块可以根据用户的个人信息，为用户推荐相似兴趣的课程或资料。
