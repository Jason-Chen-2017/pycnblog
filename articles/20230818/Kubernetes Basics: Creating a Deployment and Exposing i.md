# 1.简介
  

云计算已经成为当今 IT 技术的一大热词，越来越多的人开始关注这个新兴技术。容器技术的出现让云计算越来越接近于我们的日常生活，可以把应用部署在不同的环境中，解决效率问题。而 Kubernetes（下称k8s）则是一个开源的容器编排系统，它能够管理、调度和监控容器化应用的生命周期，是实现容器集群管理的标准工具之一。本文将对 k8s 的基础知识进行阐述，包括什么是 Pod、Deployment、Service 和 Ingress，并详细介绍它们的用法。本文不会涉及 k8s 中更高级的功能比如存储、网络、配置管理等内容，但会试图通过实际例子介绍 k8s 的基本用法。
