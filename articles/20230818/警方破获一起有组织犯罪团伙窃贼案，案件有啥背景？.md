# 1.简介
  

近日，云南省昆明市被一名男子捕杀身亡，事情发生后，警方发函进行调查。受害者杨某与他人的聊天记录、监控视频等资料在网上流传开，许多媒体、NGO组织、网站纷纷转载或报道了此事，引起舆论关注。警方破获一起有组织犯罪团伙窃贼案已经成为当前国际关注热点之一。

根据最新的信息，本次窃贼案由宏恩智能（北京）科技有限公司担任嫌疑犯，他们声称自己曾经和另一名男子建立联系，并收到一笔在银行账户中的款项，之后又在该账户中提现，导致对方长期盗取其银行卡信息和存款。事后，宏恩公司董事长吴泽鸿等人向法院提供了身份证和银行卡信息，并承诺将采取一切合法手段获取个人财产、设备及账户内资金。

虽然因涉嫌组织犯罪，警方目前仍在全力追缉犯罪分子，但已于今年5月底以“三无”形式宣布该案件清白。三无指的是不存在组织犯罪行为、个人不正当交易、与犯罪分子无直接联系。

本次事件给反洗钱、反腐败、弱势群体权益保护以及国家安全提供了巨大的线索。未来，警方的打击和查处工作将加强反恐怖的斗争，维护公平正义，促进社会公德。

# 2.基本概念术语说明
## 2.1 概念
组织犯罪（organized crime）即为通过组织及其成员的各种方法犯罪，包括黑帮、白领阶层、同行业、跨国公司、商业网络、金融集团、贸易组织、政党、学生团体等等。
## 2.2 相关词汇
- “组织犯罪”词汇出现频率较高且难以定义准确，主要用来形容犯罪活动，主要特点是非法占有、非法控制、非法交易、利用职务便利为个人谋取私利。因此，“组织犯罪”可以泛指各类不良活动，而非狭隘地指单一犯罪类型。
- 在中国，一般将“黑帮”和“白领”作为组织犯罪的分类标准，因为只要涉及的个人都有一定的经济或政治地位和社会资源，就可以把他们看作组织犯罪的对象。然而，在一些情况下，并不是所有的白领和黑帮成员都是犯罪分子，所以还需要结合一些其他的指标才可以真正判断出是否存在组织犯罪。
- 对于“跨境贸易”，“金融”也可视为组织犯罪。很多涉外企业和金融机构在国际交易时容易受到资金或信息的控制，从而成为非法持有资产的对象。
