3.1 股票可视化

股票可视化的实现流程：

- 使用 OpenCV 获取股票的图像
- 使用 Python 的 Matplotlib 库对图像进行可视化
- 使用 Python 的 Seaborn 库对图像进行加粗、加框等处理

代码实现：

```python
import cv2
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np

# 获取股票图像
img = cv2.imread('house_prices.jpg')

# 对图像进行可视化
plt.imshow(img, cmap='gray')
plt.axis('off')
plt.show()
```
