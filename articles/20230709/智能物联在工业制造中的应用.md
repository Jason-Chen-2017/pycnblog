### 3.3. 核心代码实现

智能物联技术在工业制造中的应用,需要编写核心代码,主要包括传感器数据采集、数据存储和数据分析等部分。下面是一个简单的示例代码,用于采集温度数据,并将其发送到云端进行数据分析:

```
#include <WiFi.h>
#include <Adafruit_Sensor.h>
#include <Adafruit_I2C.h>
#include <DHT.h>

const int sensorPin = 2; // 温度传感器连接的引脚
const int ledPin = 3; // 连接的 LED 引脚

Adafruit_I2C myI2C(4); // 用于读取温度数据的 I2C 对象
DHT myDHT(sensorPin, ledPin, myI2C); // 用于获取温度数据并点亮 LED 的 DHT 对象

void setup() {
  // 初始化 WiFi
  Serial.begin(9600);
  
  // 连接 WiFi
  connectTo WiFi();
  
  // 开启温度传感器
  pinMode(sensorPin, INPUT);
  
  // 开启 LED
  pinMode(ledPin, OUTPUT);
  
  // 设置 LED 为闪烁模式
  setInterval(ledOnOff, 50);
}

void loop() {
  // 读取温度数据
  int temperature;
  if (digitalRead(sensorPin) == HIGH) {
    // 将温度数据发送到云端
    String data = myDHT.readTemperature();
    writeToFile("temperature.txt", data);
  }
}
```

### 3.4. 代码讲解说明

上述代码中,我们使用 Adafruit_Sensor 和 Adafruit_I2C 库来读取温度数据。首先,我们定义了传感器连接的引脚编号 sensorPin,以及用于连接的 LED 引脚编号 ledPin。然后,我们创建了一个 Adafruit_I2C 对象 myI2C,用于读取温度数据。接着,我们创建了一个 DHT 对象 myDHT,用于获取温度数据并点亮 LED。

在 setup() 函数中,我们初始化了 WiFi 并连接到 WiFi,然后开启了温度传感器和 LED。最后,我们将 LED 设置为闪烁模式,每 50 秒切换一次状态。

在 loop() 函数中,我们使用 digitalRead() 函数读取传感器连接的引脚的值,如果引脚为高电平,则说明温度传感器正在工作。然后,我们调用 myDHT.readTemperature() 函数获取温度数据,并将数据发送到云端。最后,我们使用 writeToFile() 函数将温度数据写入文件 "temperature.txt" 中。
