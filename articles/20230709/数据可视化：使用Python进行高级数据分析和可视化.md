4.3. 核心代码实现

以下是一个简单的Python数据可视化示例代码，使用Matplotlib库实现基本的数据可视化功能：
```python
import matplotlib.pyplot as plt
import numpy as np

# 创建数据
x = np.arange(0, 100)
y = np.sin(x)

# 绘制数据
plt.plot(x, y)

# 添加标签和标题
plt.xlabel('X轴标签')
plt.ylabel('Y轴标签')
plt.title('sin函数绘制')

# 显示图形
plt.show()
```
该代码使用Matplotlib库绘制了一个sin函数的图形，并添加了标签和标题。
