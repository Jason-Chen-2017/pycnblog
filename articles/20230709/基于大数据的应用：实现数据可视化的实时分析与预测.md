2.1. 基本概念解释

数据可视化（Data Visualization）是一种将数据以图形、图表、地图等形式展现出来，使数据更加容易被理解和分析的技术手段。数据可视化的目的是通过视觉化的方式，将数据中隐藏的信息提取出来，以便于用户对数据进行实时监控、分析和预测。

Q: 如何保存图表为图片？
A: 可以使用 Matplotlib 库中的 savefig 函数将图表保存为图片。示例代码如下：
```python
import matplotlib.pyplot as plt

# 创建图表
my_chart = plt.plot(x, y)

# 保存为图片
my_chart.savefig('test.png')
```
Q: 如何使用 Matplotlib 库中的图表类型？
A: Matplotlib 库提供了许多常用的图表类型，如折线图、柱状图、饼图、散点图等。使用时，只需导入所需的库，然后根据需要调用对应图表类型的绘制函数即可。例如，使用柱状图：
```python
import pandas as pd
import matplotlib.pyplot as plt

# 创建数据
data = {'A': [1, 2, 3], 'B': [4, 5, 6]}

# 使用 pandas 将数据可视化
df = pd.DataFrame(data)
df.plot(kind='bar')
```
