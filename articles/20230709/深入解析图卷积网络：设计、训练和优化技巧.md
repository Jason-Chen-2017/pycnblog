# 2. 技术原理及概念

## 2.1. 基本概念解释

图卷积网络是一种用于图形数据的神经网络模型。它通过学习图形数据之间的邻接关系来完成图形分类、目标检测等任务。GCN 中的“图”指的是一个有向无环图（DAG），其中每个节点表示一个对象，每条边表示对象之间的关系。

## 2.2. 技术原理介绍: 算法原理，具体操作步骤，数学公式，代码实例和解释说明

GCN 的核心思想是通过聚合每个对象周围的邻居信息来更新对象的表示。具体操作步骤如下：

1. 原始数据预处理：将图形数据转换为 GCN 能够处理的格式，如 CSV 或张量形式。
2. 构建图：对原始数据进行处理，得到一个有向无环图。
3. 特征提取：从图中提取出表示每个对象的属性信息。
4. 邻居计算：计算每个对象周围的邻居信息。
5. 对象更新：根据邻居信息更新对象的表示。
6. 全局更新：对整个图进行更新，使得所有对象的表示都得到了更新。

## 2.3. 相关技术比较

与传统的卷积神经网络（CNN）相比，GCN 具有以下优势：

1. 处理序列数据：GCN 能够处理序列数据，如文本、图像等，而 CNN 主要用于处理离散数据。
2. 局部感知：GCN 能够捕捉局部特征，适用于复杂网络结构。
3. 可扩展性：GCN 更容易实现扩展，可以处理更大的图形数据集。
