### 2.2.1. 关键词提取

关键词提取是指从文本中自动提取出具有搜索引擎友好性的关键词或短语。这些关键词或短语可以用于网站的关键词密度分析，以及搜索引擎优化中的关键词替换和增加等操作。

```python
import numpy as np
import re

def keyword_extraction(text):
    # 去除停用词
    text = re.sub('[^a-zA-Z]', '', text).lower()
    # 特殊词替换
    text = re.sub('[^a-zA-Z0-9]','', text)
    # 词频统计
    word_freq = {}
    for word in text.split():
        if word not in word_freq:
            word_freq[word] = 0
        word_freq[word] += 1
    # 提取关键词
    keywords = [word for word, word_freq in word_freq.items() if word_freq > 1]
    return''.join(keywords)
```
