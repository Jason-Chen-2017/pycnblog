# 2. 技术原理及概念

## 2.1. 基本概念解释

2.1.1. 多线程模型

多线程模型是指在程序中同时执行多个线程，以达到并行执行的目的。在C++中，可以使用多线程模型来实现并行计算。

2.1.2. 线程并行执行

线程并行执行是指通过将多个线程合并为一个或多个并行执行单元（如多线程调度算法），使得线程能够在同一个时间段内并行执行，从而提高程序的处理效率。

2.1.3. 并行计算

并行计算是指在多个处理器上并行执行相同或不同的计算任务，以达到更高的计算效率。在C++中，可以使用多线程模型和OpenMP规范实现并行计算。

2.2.3. 并行计算库的数学公式

并行计算库中涉及到的数学公式主要包括：线程同步和数据并行。

线程同步是指在多线程环境中，线程之间同步执行的过程。常见的线程同步有：互斥量、信号量、条件变量等。

数据并行是指在多线程环境中，多个线程同时访问共享数据的过程。常见的数据并行有：数组下标并行、矩阵并行等。
