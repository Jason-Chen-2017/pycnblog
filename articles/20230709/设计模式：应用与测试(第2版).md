3.2. 核心模块实现

设计模式中的元素可以是类、接口、抽象类等，它们可以用来描述软件系统中的各种关系。在本文中，我们将实现一个简单的计数器设计模式。

```java
public class Counter {
    private int count = 0;

    public void increment() {
        count++;
    }

    public int getCount() {
        return count;
    }
}
```
