# 2. 技术原理及概念

## 2.1. 基本概念解释

AI在音乐创作中的应用主要包括以下几个方面：

1. 音频特征提取：AI可以从音频中提取关键特征，如旋律、和声、节奏等，以生成新的音乐作品。

2. 旋律生成：AI可以根据预设的旋律模板或基于训练数据生成新的旋律。

3. 和声生成：AI可以根据预设的和声规则或基于训练数据生成新的和声。

4. 节奏生成：AI可以根据预设的节奏规则或基于训练数据生成新的节奏。

2.2.1 音频特征提取

音频特征提取是AI生成新的音乐作品的第一步。首先，AI需要对音频信号进行预处理，如降噪、滤波等，以减少干扰因素。然后，AI可以从音频信号中提取出关键特征，如旋律、和声、节奏等。这些特征将用于生成新的音乐作品。

2.2.2 旋律生成

旋律生成是AI生成新的音乐作品的核心部分。AI可以根据预设的旋律模板或基于训练数据生成新的旋律。旋律生成的算法可以分为两种：

- 基于模板的旋律生成：AI根据预设的旋律模板生成新的旋律。这种方法的优点是能够快速生成高质量的旋律，缺点是生成的旋律可能过于简单，缺乏独特性。

- 基于训练数据的旋律生成：AI根据训练数据生成新的旋律。这种方法的优点是能够生成具有独特性的旋律，缺点是需要大量的训练数据，且训练过程可能较为耗时。

2.2.3 和声生成

和声生成是AI根据预设的和声规则或基于训练数据生成新的和声。和声生成的算法可以分为两种：

- 基于规则的和声生成：AI根据预设的和声规则生成新的和声。这种方法的优点是生成和声的规则性较强，缺点是生成的和声可能过于简单。

- 基于训练数据的和声生成：AI根据训练数据生成新的和声。这种方法的优点是能够生成具有独特性的和声，缺点是需要大量的训练数据，且训练过程可能较为耗时。
