# 2. 技术原理及概念

## 2.1. 基本概念解释

数据采集可以分为两个阶段：数据预处理和数据采集。数据预处理包括数据清洗、去重、格式化等操作，以便于后续的数据处理。数据采集包括从各种数据源中获取数据，并将其存储到本地或数据库中。
