# 2. 技术原理及概念

## 2.1. 基本概念解释

人工智能（AI）和机器学习（Machine Learning, ML）是两个不同的概念，但它们紧密相连。人工智能是指通过计算机模拟人类的智能思维、学习、推理和决策能力，使计算机具有类似于人类的智能水平。机器学习是实现这一目标的一种方法，它通过给计算机提供大量的数据和训练样本，让计算机从中学习规律和模式，从而实现自主学习和提高性能。
