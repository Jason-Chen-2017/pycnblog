# 2. 技术原理及概念

## 2.1. 基本概念解释

最大平均等待时间（MAWT）算法是一种基于期望等待时间（EWT）的优化算法。它的主要思想是在等待时间的决策中，让服务的平均等待时间尽量接近期望等待时间。MAWT算法主要考虑了等待时间加权因子、服务请求的响应时间和客户端的连接密度等因素。通过合理设置这些参数，可以提高服务的等待时间利用率，从而提高用户满意度。
