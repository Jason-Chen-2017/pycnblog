# 2. 技术原理及概念

## 2.1. 基本概念解释

元数据（Meta-data）是指描述数据的数据，是数据之上的数据，是数据管理的基础。元数据包括了数据的来源、格式、结构、质量、权限、安全等方面，是数据价值的体现者。

数据可视化（Data Visualization）是一种将数据以图形、图表、地图等形式，通过视觉的方式展示给用户的数据呈现方式。数据可视化有助于用户更直观地理解数据，提高数据分析的效率。
