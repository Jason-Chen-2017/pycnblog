---------

智能化工是指运用信息技术、自动化技术、物联网技术等多学科技术手段，对化工生产进行优化、升级和改造，实现高效、安全、环保的化工过程。智能化工的核心在于实现自动化、智能化管理，提高化工生产的效率和安全性。

4.3. 核心代码实现
-------------

```
import sqlite3
import json
import numpy as np
import matplotlib.pyplot as plt

# 建立数据库连接
conn = sqlite3.connect('data.db')

# 建立数据表
conn.execute('''CREATE TABLE IF NOT EXISTS data
             (id INTEGER PRIMARY KEY AUTOINCREMENT,
             sensor_name TEXT NOT NULL,
             sensor_value INTEGER NOT NULL,
             timestamp TIMESTAMP NOT NULL);''')

# 循环读取数据
while True:
    row = conn.execute('SELECT * FROM data')
    for row in row:
        sensor_name = row[0]
        sensor_value = row[1]
        timestamp = row[2]
        # 将数据存储到数据库中
        conn.execute('INSERT INTO data (sensor_name, sensor_value, timestamp) VALUES (?,?,?)',
                        (sensor_name, sensor_value, timestamp))
```
