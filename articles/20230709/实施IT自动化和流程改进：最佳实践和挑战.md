下面是一个使用Python语言实现的简单的自动化算法的实例。该算法可以搜索一个列表中的元素,并返回第一个出现该元素的索引。

```python
def search_element(arr):
    for i in range(len(arr)):
        if arr[i] == arr[i]:
            return i
    return -1
```
