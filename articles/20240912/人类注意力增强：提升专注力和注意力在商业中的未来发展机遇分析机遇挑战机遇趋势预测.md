##### 4. 编写一个算法，根据用户的注意力水平推荐学习资源。

**题目描述：** 给定一个用户注意力水平的数组 `attentionLevels`，其中每个元素表示用户在一定时间段内的注意力水平。设计一个算法，根据注意力水平推荐学习资源，要求学习资源的顺序与注意力水平的下降趋势一致。

**输入：**
```
attentionLevels = [5, 3, 7, 1, 9, 2, 8, 4, 6]
```

**输出：**
```
resources = ["资源1", "资源2", "资源3", "资源4", "资源5", "资源6", "资源7", "资源8", "资源9"]
```

**参考代码：**

```python
def recommend_resources(attention_levels, resources):
    sorted_resources = sorted(zip(attention_levels, resources), key=lambda x: x[0])
    return [resource for _, resource in sorted_resources]

attention_levels = [5, 3, 7, 1, 9, 2, 8, 4, 6]
resources = ["资源1", "资源2", "资源3", "资源4", "资源5", "资源6", "资源7", "资源8", "资源9"]
recommended_resources = recommend_resources(attention_levels, resources)
print(recommended_resources)
```

##### 5. 设计一个算法，计算用户在不同时间段内的平均注意力水平。

**题目描述：** 给定一个用户注意力水平的数组 `attention_levels` 和时间段数组 `time_periods`，其中每个元素表示用户在一定时间段内的注意力水平。设计一个算法，计算用户在不同时间段内的平均注意力水平。

**输入：**
```
attention_levels = [5, 3, 7, 1, 9, 2, 8, 4, 6]
time_periods = [[0, 2], [2, 4], [4, 6], [6, 8], [8, 10]]
```

**输出：**
```
average_attention_levels = [4.0, 5.5, 6.0, 7.0, 7.5]
```

**参考代码：**

```python
def average_attention_levels(attention_levels, time_periods):
    result = []
    for period in time_periods:
        start, end = period
        average_level = sum(attention_levels[start:end+1]) / len(attention_levels[start:end+1])
        result.append(average_level)
    return result

attention_levels = [5, 3, 7, 1, 9, 2, 8, 4, 6]
time_periods = [[0, 2], [2, 4], [4, 6], [6, 8], [8, 10]]
average_attention_levels = average_attention_levels(attention_levels, time_periods)
print(average_attention_levels)
```
