### 2. 设计一个算法，用于匹配求职者的技能与职位要求。

**代码示例：**
```python
def match_skills(resume_skills, job_req):
    # 将求职者和职位要求的技能转换为集合
    resume_skills_set = set(resume_skills.split(','))
    job_req_set = set(job_req.split(','))

    # 计算两个集合的交集
    common_skills = resume_skills_set.intersection(job_req_set)

    # 计算匹配度
    match_score = len(common_skills) / max(len(resume_skills_set), len(job_req_set))

    return match_score
```

**解析：**
该算法通过计算求职者技能与职位要求集合的交集，评估两者的匹配度。匹配度越高，说明求职者的技能与职位要求越接近。
