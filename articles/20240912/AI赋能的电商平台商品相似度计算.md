#### 2. 请解释如何使用余弦相似度计算商品之间的相似度。

**题目：** 请解释余弦相似度在商品相似度计算中的应用，并给出计算公式。

**答案：** 余弦相似度是一种衡量两个向量之间相似度的方法，适用于高维空间。在商品相似度计算中，可以将商品的属性表示为向量，然后计算两个向量之间的余弦相似度。

计算公式为：

\[ \text{相似度} = \frac{\text{向量A} \cdot \text{向量B}}{|\text{向量A}| \times |\text{向量B}|} \]

其中，向量A和向量B是商品A和商品B的属性向量，\(|\text{向量A}|\) 和 \(|\text{向量B}|\) 分别表示向量A和向量B的模长。

#### 3. 请解释如何使用KNN算法计算商品之间的相似度。

**题目：** 请解释如何使用KNN（K-近邻）算法计算商品之间的相似度，并给出实现步骤。

**答案：** KNN算法是一种基于实例的学习算法，通过在训练集上寻找与待分类样本最近的K个样本，然后根据这些样本的标签进行投票，确定待分类样本的标签。

在商品相似度计算中，KNN算法可以用于寻找与待比较商品最相似的K个商品。具体实现步骤如下：

1. **初始化K值**：确定一个合适的K值，用于计算相似度的商品数量。
2. **计算相似度**：计算待比较商品与其他商品的相似度，可以使用欧氏距离、余弦相似度等方法。
3. **排序**：根据相似度值对商品进行排序，找到与待比较商品最相似的K个商品。
4. **投票**：根据这K个商品的标签进行投票，确定待比较商品的标签。

### 二、算法编程题部分

#### 1. 使用余弦相似度计算两个商品向量的相似度。

**题目：** 给定两个商品的属性向量，使用余弦相似度计算它们的相似度。

**输入：** 
```python
vecA = [1, 2, 3, 4, 5]
vecB = [2, 4, 6, 8, 10]
```

**输出：** 
```python
0.7071067811865475
```

**解析：** 

```python
import math

def cosine_similarity(vecA, vecB):
    dot_product = sum(a * b for a, b in zip(vecA, vecB))
    magA = math.sqrt(sum(a ** 2 for a in vecA))
    magB = math.sqrt(sum(b ** 2 for b in vecB))
    return dot_product / (magA * magB)

vecA = [1, 2, 3, 4, 5]
vecB = [2, 4, 6, 8, 10]

similarity = cosine_similarity(vecA, vecB)
print(similarity)
```

#### 2. 使用KNN算法计算商品相似度。

**题目：** 给定一组商品属性向量和一个待比较的商品向量，使用KNN算法计算它们的相似度。

**输入：**
```python
train_data = [
    [1, 1, 1],
    [2, 2, 2],
    [3, 3, 3],
    [4, 4, 4],
    [5, 5, 5]
]

test_data = [2, 2, 2]
k = 2
```

**输出：**
```python
1.0
```

**解析：**

```python
from collections import Counter

def euclidean_distance(vecA, vecB):
    return math.sqrt(sum((a - b) ** 2 for a, b in zip(vecA, vecB)))

def knn(train_data, test_data, k):
    distances = [euclidean_distance(test_data, data) for data in train_data]
    nearest = sorted(range(len(distances)), key=lambda i: distances[i])[:k]
    labels = [train_data[i][0] for i in nearest]
    most_common = Counter(labels).most_common(1)
    return most_common[0][0]

train_data = [
    [1, 1, 1],
    [2, 2, 2],
    [3, 3, 3],
    [4, 4, 4],
    [5, 5, 5]
]

test_data = [2, 2, 2]
k = 2

similarity = knn(train_data, test_data, k)
print(similarity)
```
