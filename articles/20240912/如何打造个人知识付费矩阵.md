**题目 4：** 编写一个算法，用于分析课程评价数据，提取关键评价词。

**答案：**
```python
import nltk
from nltk.corpus import stopwords
from collections import Counter

def extract_key_words(reviews):
    stop_words = set(stopwords.words('english'))
    words = []
    for review in reviews:
        words.extend(nltk.word_tokenize(review.lower()))
    words = [word for word in words if word.isalnum() and word not in stop_words]
    return Counter(words).most_common(10)

# 示例数据
reviews = [
    "The course was fantastic and informative.",
    "I didn't find it useful at all.",
    "The instructor is knowledgeable and engaging."
]

print(extract_key_words(reviews))
```

**解析：** 这个算法使用自然语言处理库（如nltk）来提取文本中的关键评价词，并通过计数器（Counter）统计出现频率最高的词。

**题目 5：** 编写一个算法，根据用户的搜索历史，推荐相关的课程。

**答案：**
```python
def recommend_courses(search_history, courses):
    search_terms = Counter(search_history)
    scores = {}
    for course in courses:
        course_terms = Counter(course['keywords'])
        score = sum(min(search_terms[term], course_terms[term]) for term in search_terms)
        scores[course['title']] = score
    return sorted(scores.items(), key=lambda x: x[1], reverse=True)

# 示例数据
search_history = ["python", "data science", "machine learning"]
courses = [
    {"title": "Introduction to Python", "keywords": ["python", "programming"]},
    {"title": "Data Science Bootcamp", "keywords": ["data science", "analytics"]},
    {"title": "Deep Learning", "keywords": ["machine learning", "neural networks"]}
]

print(recommend_courses(search_history, courses))
```

**解析：** 这个算法通过计算搜索历史和课程关键词的相似度来推荐相关的课程，使用计数器来比较关键词的交集。
