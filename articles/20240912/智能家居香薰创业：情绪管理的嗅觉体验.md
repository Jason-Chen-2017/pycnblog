##### 3. 设计一个算法，计算香薰设备在不同浓度下的使用时间。

**题目描述：** 假设香薰设备有 A、B、C 三种浓度模式，浓度分别为低、中、高。每种浓度模式下，设备的使用时间分别为 2 小时、4 小时和 6 小时。设计一个算法，计算用户在使用香薰设备时，根据需求选择不同浓度模式，总共需要使用多长时间。

**答案：** 

```python
def calculate_usage_time(low_usage, mid_usage, high_usage):
    return low_usage * 2 + mid_usage * 4 + high_usage * 6

# 示例
print(calculate_usage_time(1, 2, 0))  # 输出 12
print(calculate_usage_time(0, 1, 1))  # 输出 12
print(calculate_usage_time(1, 1, 1))  # 输出 18
```

**解析：** 该算法通过简单的计算，实现了根据不同浓度模式和使用时间，计算总使用时间的功能。

##### 4. 设计一个算法，实现智能家居香薰系统的定时功能。

**题目描述：** 假设智能家居香薰系统支持每天定时开启和关闭香薰设备。设计一个算法，实现用户可以根据需求设置每天定时开启和关闭香薰设备的时间。

**答案：**

```python
class Timer:
    def __init__(self):
        self.on_time = None
        self.off_time = None

    def set_time(self, on_time, off_time):
        self.on_time = on_time
        self.off_time = off_time

    def check_timer(self, current_time):
        if current_time >= self.on_time and current_time < self.off_time:
            return True
        else:
            return False

# 示例
timer = Timer()
timer.set_time(8, 20)  # 设置每天早上 8 点开启，晚上 8 点关闭
print(timer.check_timer(9))  # 输出 True，表示当前时间 9 点香薰设备处于开启状态
print(timer.check_timer(22))  # 输出 False，表示当前时间 22 点香薰设备处于关闭状态
```

**解析：** 该算法通过设置定时开启和关闭时间，实现了智能家居香薰系统的定时功能。同时，通过检查当前时间，判断香薰设备是否处于开启状态。
