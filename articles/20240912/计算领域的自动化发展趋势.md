### 3. 自动化算法编程题

**题目：** 编写一个算法，实现将一个数组中的所有元素乘以 2，并返回新的数组。

**答案：**

```python
def multiply_by_two(arr):
    return [x * 2 for x in arr]

# 示例
input_arr = [1, 2, 3, 4, 5]
result = multiply_by_two(input_arr)
print(result)  # 输出 [2, 4, 6, 8, 10]
```

**解析：** 该算法使用列表推导式将数组中的每个元素乘以 2，并返回一个新的数组。这可以看作是计算领域中的一种简单自动化。

### 4. 自动化测试题

**题目：** 编写一个自动化测试用例，测试上述 `multiply_by_two` 函数的正确性。

**答案：**

```python
import unittest

class TestMultiplyByTwo(unittest.TestCase):
    def test_multiply_by_two(self):
        self.assertEqual(multiply_by_two([1, 2, 3]), [2, 4, 6])
        self.assertEqual(multiply_by_two([4, 5, 6]), [8, 10, 12])
        self.assertEqual(multiply_by_two([0, -1, -2]), [0, -2, -4])

if __name__ == '__main__':
    unittest.main()
```

**解析：** 该测试用例使用 Python 的 `unittest` 框架，验证了 `multiply_by_two` 函数在不同输入情况下的正确性。

### 6. 自动化运维题

**题目：** 使用 Python 的 paramiko 库，编写一个脚本，自动连接远程服务器并执行一个命令。

**答案：**

```python
import paramiko

hostname = 'remote.example.com'
username = 'admin'
password = 'password'

client = paramiko.SSHClient()
client.set_missing_host_key_policy(paramiko.AutoAddPolicy())
client.connect(hostname, username=username, password=password)

stdin, stdout, stderr = client.exec_command('ls -l')
print(stdout.read().decode())
client.close()
```

**解析：** 该脚本使用 paramiko 库连接到远程服务器，并执行了 `ls -l` 命令。
