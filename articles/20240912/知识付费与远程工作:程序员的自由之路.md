## 三、算法编程题库与解析

### 1. 如何实现二分查找？

**答案：** 二分查找算法的实现如下：

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    
    return -1
```

**解析：** 二分查找算法通过不断将搜索区间缩小一半，实现高效查找。时间复杂度为 O(logn)。

### 2. 如何实现快速排序？

**答案：** 快速排序算法的实现如下：

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    return quick_sort(left) + middle + quick_sort(right)
```

**解析：** 快速排序算法通过递归地将数组划分为左右两部分，实现高效排序。时间复杂度为 O(nlogn)。
