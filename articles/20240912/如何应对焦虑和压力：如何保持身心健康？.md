### 算法编程题库与解析

#### 6. 如何用算法解决“倒数第二个最大数”的问题？

**题目：** 给定一个整数数组，找出倒数第二个最大的数。

**答案：** 可以使用快速选择算法，时间复杂度为 \(O(n)\)。

```python
def find_second_largest(nums):
    if len(nums) < 2:
        return None
    
    largest, second_largest = float('-inf'), float('-inf')
    
    for num in nums:
        if num > largest:
            second_largest, largest = largest, num
        elif num > second_largest and num != largest:
            second_largest = num
    
    return second_largest if second_largest != float('-inf') else None
```

**解析：** 通过遍历数组，我们可以找到最大的两个数，从而解决倒数第二个最大数的问题。

#### 7. 如何使用深度优先搜索（DFS）解决“岛屿数量”的问题？

**题目：** 给定一个由 '1'（陆地）和 '0'（水）组成的的二维网格，计算岛屿的数量。

**答案：** 可以使用 DFS 算法，时间复杂度为 \(O(m \times n)\)。

```python
def numIslands(grid):
    def dfs(i, j):
        grid[i][j] = '0'
        for x, y in [[0, 1], [0, -1], [1, 0], [-1, 0]]:
            nx, ny = i + x, j + y
            if 0 <= nx < len(grid) and 0 <= ny < len(grid[0]) and grid[nx][ny] == '1':
                dfs(nx, ny)
    
    count = 0
    for i in range(len(grid)):
        for j in range(len(grid[0])):
            if grid[i][j] == '1':
                dfs(i, j)
                count += 1
    return count
```

**解析：** 通过 DFS，我们可以从每个陆地开始，将其及其周围的陆地都标记为访问过，然后计数。

#### 8. 如何使用广度优先搜索（BFS）解决“最短路径”的问题？

**题目：** 给定一个包含墙壁和地雷的二维网格，找出从起点到终点的最短路径。

**答案：** 可以使用 BFS 算法，时间复杂度为 \(O(m \times n)\)。

```python
from collections import deque

def shortestPath(grid, start, end):
    def neighbors(i, j):
        for x, y in [[0, 1], [0, -1], [1, 0], [-1, 0]]:
            nx, ny = i + x, j + y
            if 0 <= nx < len(grid) and 0 <= ny < len(grid[0]) and grid[nx][ny] != -1:
                yield nx, ny
    
    q = deque([start])
    visited = set([start])
    steps = {start: 0}
    
    while q:
        node = q.popleft()
        if node == end:
            return steps[end]
        
        for neighbor in neighbors(*node):
            if neighbor not in visited:
                visited.add(neighbor)
                q.append(neighbor)
                steps[neighbor] = steps[node] + 1
    
    return -1
```

**解析：** BFS 算法可以帮助我们找到从起点到终点的最短路径，通过不断地扩展未访问的节点，直到找到终点。
