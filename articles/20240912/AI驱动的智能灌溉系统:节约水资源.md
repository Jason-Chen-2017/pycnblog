```python
def calculate_irrigation_demand(soil_humidity):
    """
    计算土壤湿度与灌溉需求之间的关系。

    参数：
    soil_humidity（土壤湿度，0-100%）

    输出：
    irrigation_demand（灌溉需求，单位：升/平方米）
    """

    # 根据土壤湿度计算灌溉需求
    if soil_humidity >= 70:
        irrigation_demand = 0
    elif soil_humidity >= 40:
        irrigation_demand = 5
    else:
        irrigation_demand = 10

    return irrigation_demand
```

**解析：** 该函数根据土壤湿度计算灌溉需求。当土壤湿度高于 70% 时，不需要灌溉；当土壤湿度在 40% 到 70% 之间时，灌溉需求为 5 升/平方米；当土壤湿度低于 40% 时，灌溉需求为 10 升/平方米。

```python
def adjust_irrigation_strategy(current_soil_humidity, predicted_demand, historical_data):

    参数：
    current_soil_humidity（当前土壤湿度）
    predicted_demand（预测灌溉需求）
```
