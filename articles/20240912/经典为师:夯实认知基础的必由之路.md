#### 3. 二分查找

**题目：** 在一个有序数组中查找一个元素，并返回其索引。

```python
def binary_search(arr: List[int], target: int) -> int:
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

**答案：** 使用二分查找算法，通过迭代方式在有序数组中查找目标元素。

#### 4. 链表翻转

**题目：** 编写一个函数，翻转单链表。

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverse_linked_list(head: Optional[ListNode]) -> Optional[ListNode]:
    prev = None
    curr = head
    while curr:
        next_temp = curr.next
        curr.next = prev
        prev = curr
        curr = next_temp
    return prev
```

**答案：** 使用迭代方法，逐个节点改变其指向，实现链表的翻转。

### 三、系统设计与优化题

#### 5. 缓存设计

**题目：** 设计一个简单的LRU（最近最少使用）缓存。

```python
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = OrderedDict()

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.cache.move_to_end(key)
        return self.cache[key]

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.cache.pop(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)
```

**答案：** 使用OrderedDict实现LRU缓存，通过移动元素到末尾和删除超时的元素来保证缓存的有效性。

#### 6. 分布式锁

**题目：** 实现一个分布式锁。

```python
from redis import Redis

redis_client = Redis(host='localhost', port=6379, db=0)

def distributed_lock(lock_name: str, timeout: int = 30):
    end_time = time.time() + timeout
    while True:
        if redis_client.set(lock_name, "1", nx=True, ex=timeout):
            return True
        else:
            time.sleep(0.1)
            if time.time() > end_time:
                return False
```

**答案：** 使用Redis的SET命令实现分布式锁，通过`nx`（设置如果键不存在）和`ex`（设置过期时间）参数来保证锁的独占性和有效期。

#### 8. TCP/IP 协议

**题目：** 简述TCP/IP协议的工作原理。

**答案：** TCP/IP协议是一种网络通信协议，包括传输控制协议（TCP）和互联网协议（IP）。TCP提供可靠的、面向连接的、全双工的数据传输服务；IP负责将数据包从源地址传输到目的地址。TCP/IP协议的工作原理是：数据由应用程序发送到传输层，TCP将数据分割成段，并添加序列号和校验和；IP将TCP段封装成数据包，并添加源IP和目的IP地址；路由器根据目的IP地址转发数据包，TCP使用序列号和校验和进行错误检测和重传。

### 七、操作系统与并发题

#### 9. 进程与线程

**题目：** 简述进程与线程的区别。

**答案：** 进程是计算机中资源分配的基本单位，每个进程拥有独立的内存空间、系统资源等；线程是进程中的执行单元，共享进程的资源，如内存空间。进程与线程的主要区别在于：

- 进程是资源分配的基本单位，线程是调度和执行的基本单位。
- 进程间相互独立，线程可以共享进程的资源。
- 进程的创建、销毁开销较大，线程的开销较小。
- 多进程通信开销大，多线程通信开销小。
