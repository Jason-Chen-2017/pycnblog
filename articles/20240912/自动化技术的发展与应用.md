#### 算法编程题库

#### 4. 智能排序算法

**题目：** 编写一个排序算法，实现根据数字的奇偶性进行排序，偶数在前，奇数在后。

**答案：**

```python
def odd_even_sort(arr):
    evens = []
    odds = []
    for num in arr:
        if num % 2 == 0:
            evens.append(num)
        else:
            odds.append(num)
    return evens + odds

# 示例
arr = [3, 2, 1, 4, 5]
print(odd_even_sort(arr))  # 输出 [2, 4, 1, 5, 3]
```

**解析：** 该算法通过遍历数组，将偶数和奇数分别存储在两个列表中，最后将偶数列表与奇数列表连接，得到排序结果。

#### 5. 自动化路径规划

**题目：** 编写一个算法，实现根据起点和终点，规划出一条从起点到终点的最短路径。

**答案：**

```python
def find_shortest_path(graph, start, end):
    visited = set()
    queue = [(start, [start])]

    while queue:
        node, path = queue.pop(0)
        if node == end:
            return path
        visited.add(node)
        for neighbor, weight in graph[node].items():
            if neighbor not in visited:
                new_path = list(path)
                new_path.append(neighbor)
                queue.append((neighbor, new_path))

    return None

# 示例
graph = {
    'A': {'B': 1, 'C': 3},
    'B': {'A': 1, 'C': 1, 'D': 3},
    'C': {'A': 3, 'B': 1, 'D': 2},
    'D': {'B': 3, 'C': 2}
}
start = 'A'
end = 'D'
print(find_shortest_path(graph, start, end))  # 输出 ['A', 'B', 'D']
```

**解析：** 该算法使用广度优先搜索（BFS）算法，从起点开始遍历所有可能的路径，直到找到终点。
