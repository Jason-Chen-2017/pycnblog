#### 1. 计算观看直播的用户数（Python）

**题目：** 给定一组观看直播的用户ID，请计算不同时间段的观看用户数。

**代码：**

```python
def calculate_user_counts(user_ids, timestamps):
    user_counts = []
    last_time = None
    user_set = set()

    for user_id, timestamp in zip(user_ids, timestamps):
        if last_time is not None and timestamp != last_time:
            user_counts.append(len(user_set))
            user_set.clear()

        user_set.add(user_id)
        last_time = timestamp

    user_counts.append(len(user_set))
    return user_counts
```

**解析：** 该函数通过遍历用户ID和对应的时间戳，统计不同时间段内的观看用户数，实现用户数的实时计算。

#### 2. 商品推荐（Python）

**题目：** 使用协同过滤算法实现商品推荐系统。

**代码：**

```python
from sklearn.cluster import KMeans
import numpy as np

def collaborative_filter(ratings, k=5):
    # 构建用户-商品矩阵
    num_users, num_items = ratings.shape
    user_item_matrix = np.zeros((num_users, num_items))

    # 填充用户-商品矩阵
    for user, item, rating in ratings:
        user_item_matrix[user][item] = rating

    # 计算用户中心
    user_centers = KMeans(n_clusters=k).fit(user_item_matrix).cluster_centers_

    # 计算每个用户的推荐商品
    recommendations = []
    for user in range(num_users):
        distances = np.linalg.norm(user_item_matrix[user] - user_centers, axis=1)
        nearest_centers = np.argsort(distances)[:k]
        recommended_items = np.argmax(user_centers[nearest_centers], axis=1)
        recommendations.append(recommended_items)

    return recommendations
```

**解析：** 该函数使用K均值聚类算法计算用户中心，并根据用户中心进行商品推荐，实现协同过滤算法。
