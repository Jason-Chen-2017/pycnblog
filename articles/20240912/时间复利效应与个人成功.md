#### 1. 股票买卖最佳时机

**题目：** 给定一个整数数组 prices，其中 prices[i] 是第 i 天股票的价格。如果我们在第 i 天买入股票并持有到第 j 天，那么我们所能获得的最大利润为 (prices[j] - prices[i])。实现一个函数，计算并返回我们能够获得的最大利润。

**答案：** 

```python
def maxProfit(prices):
    max_profit = 0
    for i in range(len(prices) - 1):
        for j in range(i + 1, len(prices)):
            profit = prices[j] - prices[i]
            max_profit = max(max_profit, profit)
    return max_profit
```

**解析：** 该算法通过嵌套循环遍历所有可能的买卖时机，找出最大利润。时间复杂度为 O(n^2)，适用于小规模数据。

#### 2. 最长递增子序列

**题目：** 给定一个整数数组 nums，返回 nums 的最长递增子序列的长度。

**答案：**

```python
def lengthOfLIS(nums):
    dp = [1] * len(nums)
    for i in range(1, len(nums)):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```

**解析：** 该算法使用动态规划求解最长递增子序列。时间复杂度为 O(n^2)，适用于中大规模数据。

#### 3. 买卖股票的最佳时机 III

**题目：** 给定一个整数数组 prices，其中 prices[i] 是第 i 天股票的价格。如果我们在第 i 天买入股票并持有到第 j 天，那么我们所能获得的最大利润为 (prices[j] - prices[i])。实现一个函数，计算并返回我们在最多两笔交易中能够获得的最大利润。

**答案：**

```python
def maxProfit(prices):
    buy1, buy2, sell1, sell2 = float('-inf'), float('-inf'), 0, 0
    for p in prices:
        buy1 = max(buy1, -p)
        buy2 = max(buy2, buy1-p)
        sell1 = max(sell1, buy1+p)
        sell2 = max(sell2, sell1+p)
    return sell2
```

**解析：** 该算法使用动态规划求解在最多两笔交易中的最大利润。时间复杂度为 O(n)，适用于大规模数据。
