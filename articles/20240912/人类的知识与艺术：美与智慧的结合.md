### 面试题库

#### 1. 什么是闭包？

**题目：** 请解释什么是闭包，并给出一个闭包的例子。

**答案：** 

闭包是一种函数，它记得并访问了其定义作用域中的变量。即使在外部作用域中的变量发生变化，闭包仍然可以访问并使用这些变量。

**例子：**

```go
package main

import "fmt"

func main() {
    outer := "外部变量"
    func() {
        inner := "内部变量"
        fmt.Println(outer + " " + inner)
    }()
}
```

#### 2. Go语言中的协程是什么？

**题目：** 请简要介绍Go语言中的协程，并解释如何使用协程实现并发。

**答案：** 

协程是Go语言提供的一种轻量级并发机制。协程看起来像函数，但实际上是独立的执行单元。通过使用 `go` 关键字，可以启动一个新的协程。

**例子：**

```go
package main

import "fmt"

func main() {
    for i := 0; i < 10; i++ {
        go func() {
            fmt.Println(i)
        }()
    }
}
```

#### 3. 什么是接口？

**题目：** 请解释什么是接口，并说明如何实现接口。

**答案：** 

接口是一组方法声明，用于描述一个对象的行为。任何类型只要实现了接口中的所有方法，就可以被认为实现了该接口。

**例子：**

```go
package main

import "fmt"

type Animal interface {
    Speak() string
}

type Dog struct{}

func (d Dog) Speak() string {
    return "Woof!"
}

func main() {
    var animal Animal = Dog{}
    fmt.Println(animal.Speak())
}
```

### 算法编程题库

#### 1. 寻找旋转排序数组中的最小值

**题目：** 假设按照升序排序的数组在预先未知的某个点上进行了旋转。

请你找出并返回数组中的最小元素。

**答案：**

我们可以使用二分查找法来解决这道题。

**代码示例：**

```go
package main

import "fmt"

func findMin(nums []int) int {
    left, right := 0, len(nums)-1
    for left < right {
        mid := left + (right-left)/2
        if nums[mid] > nums[right] {
            left = mid + 1
        } else {
            right = mid
        }
    }
    return nums[left]
}

func main() {
    nums := []int{4, 5, 6, 7, 0, 1, 2}
    fmt.Println(findMin(nums))
}
```

#### 2. 最长公共前缀

**题目：** 编写一个函数来查找字符串数组中的最长公共前缀。

**答案：**

我们可以使用垂直扫描法来解决这个问题。

**代码示例：**

```go
package main

import "fmt"

func longestCommonPrefix(strs []string) string {
    if len(strs) == 0 {
        return ""
    }
    for i, v := 0, strs[0]; i < len(strs); i++ {
        for j := 0; j < len(v) && j < len(strs[i]); j++ {
            if v[j] != strs[i][j] {
                return v[:j]
            }
        }
        if i != 0 {
            v = v[:len(v)-1]
        }
    }
    return v
}

func main() {
    strs := []string{"flower", "flow", "flight"}
    fmt.Println(longestCommonPrefix(strs))
}
```
