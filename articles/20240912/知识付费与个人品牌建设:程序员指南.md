### 算法编程题与答案解析

#### 5. 实现一个函数，计算两个字符串的编辑距离

**题目：** 给定两个字符串 `word1` 和 `word2`，编写一个函数来计算将 `word1` 转换为 `word2` 所使用的最小操作次数。操作包括插入、删除和替换一个字符。

**答案：** 使用动态规划算法，实现代码如下：

```python
def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(m + 1):
        for j in range(n + 1):
            if i == 0:
                dp[i][j] = j
            elif j == 0:
                dp[i][j] = i
            elif word1[i - 1] == word2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]
            else:
                dp[i][j] = min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]) + 1

    return dp[m][n]
```

**解析：** 动态规划算法通过构建一个二维数组 `dp`，记录每个位置的最小编辑距离，最终得到 `dp[m][n]` 即为整个字符串的编辑距离。

#### 6. 设计一个LRU缓存

**题目：** 设计一个LRU（最近最少使用）缓存，支持 `get` 和 `put` 操作。

**答案：** 使用哈希表和双向链表实现LRU缓存，代码如下：

```python
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = OrderedDict()

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.cache.move_to_end(key)
        return self.cache[key]

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)
```

**解析：** 使用有序字典 `OrderedDict` 来实现LRU缓存，其中 `get` 和 `put` 操作都能在O(1)时间内完成。当缓存容量超出限制时，移除最久未使用的数据项。
