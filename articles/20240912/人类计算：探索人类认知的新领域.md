#### 1. 基于距离的分类算法
**题目：** 编写一个基于欧几里得距离的分类算法，实现对一组二维数据的分类。

**答案：**
```python
def euclidean_distance(x, y):
    return sum((a - b) ** 2 for a, b in zip(x, y)) ** 0.5

def classify(data, centroids, num_clusters):
    clusters = [[] for _ in range(num_clusters)]
    for point in data:
        closest_centroid = min(centroids, key=lambda c: euclidean_distance(point, c))
        clusters[centroids.index(closest_centroid)].append(point)
    return clusters

# 示例数据
data = [[2, 3], [5, 7], [8, 1], [6, 2]]
centroids = [[1, 1], [6, 5]]
num_clusters = 2

clusters = classify(data, centroids, num_clusters)
print("Clusters:", clusters)
```
