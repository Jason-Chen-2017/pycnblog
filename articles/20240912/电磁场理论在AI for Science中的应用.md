### 算法编程题解析

#### 4. 编写一个算法，计算一个二维电场的势能函数。

**答案：**

```python
def potential_energy_electric(charges):
    """
    计算二维电场中的势能函数。
    :param charges: 一个包含电荷值和位置的列表，每个元素是一个元组 (charge, x, y)。
    :return: 潮源电荷产生的总势能。
    """
    total_energy = 0
    for i, (q_i, x_i, y_i) in enumerate(charges):
        for j, (q_j, x_j, y_j) in enumerate(charges):
            if i != j:
                distance = ((x_i - x_j) ** 2 + (y_i - y_j) ** 2) ** 0.5
                total_energy += q_i * q_j / distance
    return total_energy

# 示例
charges = [(1, 0, 0), (-1, 1, 0), (2, 0, 1)]
print(potential_energy_electric(charges))
```

**解析：** 该算法通过计算每对电荷之间的相互作用势能，累加得到总势能。适用于计算二维电场的势能函数。

#### 5. 编写一个算法，求解一个二维磁场的势函数。

**答案：**

```python
import numpy as np

def potential_magnetic(B, r0, r):
    """
    计算二维磁场中的势函数。
    :param B: 磁场强度。
    :param r0: 磁场源的位置。
    :param r: 要计算的点的位置。
    :return: 磁场源产生的势函数值。
    """
    distance = np.linalg.norm(r - r0)
    return -B * np.dot(r - r0, r - r0) / (2 * distance)

# 示例
B = np.array([1, 0])
r0 = np.array([0, 0])
r = np.array([1, 1])
print(potential_magnetic(B, r0, r))
```

**解析：** 该算法根据比奥萨伐尔定律，计算磁场源在空间某点产生的势函数值。
