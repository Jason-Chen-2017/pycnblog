1. **最长公共子序列（LCS）**
   - **算法描述：** 动态规划，通过比较两个字符串的子序列，找到最长的公共子序列。
   - **答案解析：** 使用二维数组存储子问题的解，最终得到最长公共子序列的长度。
   - **源代码实例：**
     ```python
     def longest_common_subsequence(str1, str2):
         m, n = len(str1), len(str2)
         dp = [[0] * (n + 1) for _ in range(m + 1)]

         for i in range(1, m + 1):
             for j in range(1, n + 1):
                 if str1[i - 1] == str2[j - 1]:
                     dp[i][j] = dp[i - 1][j - 1] + 1
                 else:
                     dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

         return dp[m][n]
     ```

2. **二分查找**
   - **算法描述：** 在有序数组中查找目标元素的位置，时间复杂度为O(log n)。
   - **答案解析：** 通过不断缩小查找范围，逐步逼近目标元素。
   - **源代码实例：**
     ```python
     def binary_search(arr, target):
         left, right = 0, len(arr) - 1

         while left <= right:
             mid = (left + right) // 2
             if arr[mid] == target:
                 return mid
             elif arr[mid] < target:
                 left = mid + 1
             else:
                 right = mid - 1

         return -1
     ```
