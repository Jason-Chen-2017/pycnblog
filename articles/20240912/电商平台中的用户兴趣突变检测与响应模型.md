### 1. 利用 K-means 算法进行用户兴趣聚类

**题目：** 编写一个函数，利用 K-means 算法对用户行为数据进行聚类，识别出用户兴趣类别。

**答案：** 具体代码实现如下：

```python
import numpy as np

def kmeans(data, k, max_iter=100):
    # 随机初始化中心点
    centroids = data[np.random.choice(data.shape[0], k, replace=False)]
    
    for i in range(max_iter):
        # 计算每个数据点与中心点的距离
        distances = np.linalg.norm(data - centroids, axis=1)
        
        # 计算每个数据点所属的类别
        clusters = np.argmin(distances, axis=1)
        
        # 更新中心点
        new_centroids = np.array([data[clusters == j].mean(axis=0) for j in range(k)])
        
        # 判断是否收敛
        if np.linalg.norm(new_centroids - centroids) < 1e-5:
            break
        
        centroids = new_centroids
    
    return clusters, centroids
```

**解析：** 该函数首先随机初始化 k 个中心点，然后循环迭代，计算每个数据点与中心点的距离，并根据距离判断数据点所属的类别。最后，更新中心点，并判断是否收敛。

### 2. 利用 RNN 模型进行用户兴趣预测

**题目：** 编写一个函数，利用 RNN 模型对用户行为数据进行建模，预测用户兴趣变化。

**答案：** 具体代码实现如下：

```python
import tensorflow as tf

def build_rnn_model(input_shape, hidden_size, num_classes):
    model = tf.keras.Sequential([
        tf.keras.layers.SimpleRNN(hidden_size, input_shape=input_shape, return_sequences=True),
        tf.keras.layers.Dense(num_classes, activation='softmax')
    ])
    model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])
    return model
```

**解析：** 该函数定义了一个简单的 RNN 模型，包括一个 RNN 层和一个全连接层。输入层形状为 `input_shape`，隐藏层大小为 `hidden_size`，输出层为类别预测，使用 softmax 激活函数。
