#### 1. 听觉信号如何在神经元中传递？

**题目：** 简要解释听觉信号如何在神经元中传递，并给出一个示例。

**答案：** 听觉信号通过声波引起耳膜振动，进而激活耳蜗内的毛细胞。毛细胞将机械能转换为电信号，通过听觉神经传递到大脑皮层。以下是一个简单的电信号传递模型：

```python
class AuditoryNeuron:
    def __init__(self):
        self.charge = 0

    def receive_signal(self, voltage):
        self.charge += voltage
        if self.charge >= 50:
            self.fire_actionPotential()
            self.charge = 0

    def fire_actionPotential(self):
        print("Action potential fired!")

# 示例
auditory_neuron = AuditoryNeuron()
for i in range(20):
    auditory_neuron.receive_signal(1)
```

**解析：** 该模型简化了神经元如何响应电信号的过程。`receive_signal` 方法累积电荷，并在达到阈值时触发动作电位。

#### 2. 有机化合物如何在体内发挥功能？

**题目：** 请解释有机化合物如何在体内发挥功能，并举例说明。

**答案：** 有机化合物通过与其生物分子（如蛋白质、核酸等）结合，影响细胞的生理和生化过程。以下是一个有机化合物（药物）结合蛋白质的简单模型：

```python
class Protein:
    def __init__(self, name):
        self.name = name
        self.bind_site = None

    def bind(self, compound):
        if compound.is_compatible(self):
            self.bind_site = compound
            print(f"{compound.name} binds to {self.name}")

    def is_compatible(self, compound):
        return compound.name != "incompatible"

class OrganicCompound:
    def __init__(self, name):
        self.name = name

# 示例
protein = Protein("G Protein")
drug = OrganicCompound("LSD")
protein.bind(drug)
```

**解析：** 在此模型中，蛋白质与有机化合物（如药物）结合，从而改变其功能。

#### 3. 神经元之间的通信如何进行？

**题目：** 描述神经元之间通信的基本机制，并提供一个示例。

**答案：** 神经元之间的通信主要通过突触实现，突触前神经元释放神经递质，作用于突触后神经元的受体，产生电信号。以下是一个简化版的神经元通信模型：

```python
class Synapse:
    def __init__(self, neurotransmitter):
        self.neurotransmitter = neurotransmitter

    def fire(self):
        print("Synaptic transmission: Neurotransmitter released!")

class Neuron:
    def __init__(self, synapses):
        self.synapses = synapses

    def receive_signal(self, voltage):
        if voltage >= 50:
            for synapse in self.synapses:
                synapse.fire()

# 示例
synapse = Synapse("Dopamine")
neuron = Neuron([synapse])
neuron.receive_signal(60)
```

**解析：** 在这个模型中，神经元通过突触释放神经递质，传递信号给其他神经元。
