#### 一、典型面试题

**1. 什么是深度优先搜索（DFS）和广度优先搜索（BFS）？请分别给出应用场景。**

**答案：** 深度优先搜索（DFS）和广度优先搜索（BFS）是两种基本的图遍历算法。

- **深度优先搜索（DFS）：** 沿着一条路径深入到最远点，再回溯。适用于求解连通性、最短路径（在某些条件下）、图的遍历等问题。
- **广度优先搜索（BFS）：** 先遍历所有的相邻节点，再逐层遍历。适用于求解最短路径（无负权环）、拓扑排序等问题。

**2. 如何实现一个有序链表去重？**

**答案：** 可以遍历链表，判断当前节点是否等于下一节点，如果相等则删除下一节点。

**代码示例：**

```java
public void deleteDuplicates(ListNode head) {
    if (head == null || head.next == null) {
        return;
    }
    ListNode cur = head;
    while (cur != null && cur.next != null) {
        if (cur.val == cur.next.val) {
            cur.next = cur.next.next;
        } else {
            cur = cur.next;
        }
    }
}
```

**3. 请实现一个算法，找出数组中重复的数字。**

**答案：** 可以使用哈希表记录数组中已出现的数字，遍历数组，如果发现当前数字已经在哈希表中，则返回该数字。

**代码示例：**

```java
public int findRepeatNumber(int[] nums) {
    Set<Integer> set = new HashSet<>();
    for (int num : nums) {
        if (set.contains(num)) {
            return num;
        }
        set.add(num);
    }
    return -1;
}
```

#### 二、算法编程题

**1. 请实现一个函数，用于计算两个有序数组合并后的中位数。**

**答案：** 可以将两个有序数组合并为一个有序数组，然后求中位数。或者使用二分查找的方法，在不合并数组的情况下求中位数。

**代码示例：**

```java
public double findMedianSortedArrays(int[] nums1, int[] nums2) {
    int m = nums1.length;
    int n = nums2.length;
    int[] nums = new int[m + n];
    int i = 0, j = 0, k = 0;
    while (i < m && j < n) {
        if (nums1[i] <= nums2[j]) {
            nums[k++] = nums1[i++];
        } else {
            nums[k++] = nums2[j++];
        }
    }
    while (i < m) {
        nums[k++] = nums1[i++];
    }
    while (j < n) {
        nums[k++] = nums2[j++];
    }
    if ((m + n) % 2 == 0) {
        return (nums[(m + n) / 2 - 1] + nums[(m + n) / 2]) / 2.0;
    } else {
        return nums[(m + n) / 2];
    }
}
```

**2. 请实现一个快速排序算法。**

**答案：** 快速排序是一种基于分治思想的排序算法，其基本思想是通过一趟排序将待排序的记录分割成独立的两部分，其中一部分记录的关键字均比另一部分的关键字小，然后分别对这两部分记录继续进行排序，以达到整个序列有序。

**代码示例：**

```java
public void quickSort(int[] nums, int low, int high) {
    if (low < high) {
        int pivot = partition(nums, low, high);
        quickSort(nums, low, pivot - 1);
        quickSort(nums, pivot + 1, high);
    }
}

public int partition(int[] nums, int low, int high) {
    int pivot = nums[high];
    int i = low - 1;
    for (int j = low; j < high; j++) {
        if (nums[j] < pivot) {
            i++;
            int temp = nums[i];
            nums[i] = nums[j];
            nums[j] = temp;
        }
    }
    int temp = nums[i + 1];
    nums[i + 1] = nums[high];
    nums[high] = temp;
    return i + 1;
}
```
