### 2. 不确定性的接受与处理

不确定性是复杂性的伴生品。在决策过程中，我们需要学会接受不确定性，并找到合适的方法来处理它。以下是一道相关的算法编程题：

**算法编程题：** 实现一个函数，计算一组数据的方差，并处理可能的错误情况。

**源代码示例：**

```python
def compute_variance(data):
    try:
        n = len(data)
        mean = sum(data) / n
        variance = sum((x - mean) ** 2 for x in data) / n
        return variance
    except ZeroDivisionError:
        return "数据集为空，无法计算方差。"
    except TypeError:
        return "数据集格式错误，请输入数字。"
```
