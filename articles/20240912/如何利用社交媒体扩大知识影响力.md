##### 2. 社交媒体上的信息传播模型分析

**编程题：** 编写一个函数，用于分析社交媒体上的信息传播模型。

**满分答案解析：**
- **输入**：社交媒体上的信息传播数据，如发布时间、点赞数、评论数等。
- **输出**：信息传播模型的统计结果，如传播速度、传播范围等。
- **算法思路**：可以使用时间序列分析方法，如移动平均、指数平滑等。

```python
# 示例代码
def analyze_info_spread(info_data):
    time_diffs = [info_data[i+1]['timestamp'] - info_data[i]['timestamp'] for i in range(len(info_data)-1)]
    avg_time_diff = sum(time_diffs) / len(time_diffs)
    like_count = [info['like_count'] for info in info_data]
    avg_like_count = sum(like_count) / len(like_count)
    return {
        'avg_time_diff': avg_time_diff,
        'avg_like_count': avg_like_count,
    }
```
