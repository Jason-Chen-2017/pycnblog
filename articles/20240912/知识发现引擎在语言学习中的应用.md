#### 4. 算法编程题库

**文本分类**

- **题目**：编写一个文本分类程序，实现将输入文本分类到指定的类别中。
- **答案**：可以使用朴素贝叶斯分类器实现文本分类。以下是一个简单的实现示例：

```python
import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

# 示例数据
corpus = [
    "I love this book",
    "This is a great movie",
    "The food is delicious",
    "I hate this class"
]

# 标签
labels = ["positive", "positive", "positive", "negative"]

# 将文本转换为词频矩阵
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(corpus)

# 训练朴素贝叶斯分类器
clf = MultinomialNB()
clf.fit(X, labels)

# 预测新文本
new_text = "This class is not good"
X_new = vectorizer.transform([new_text])
prediction = clf.predict(X_new)
print(prediction)  # 输出 ['negative']
```

**情感分析**

- **题目**：编写一个情感分析程序，判断输入文本的情感极性（正面或负面）。
- **答案**：可以使用支持向量机（SVM）实现情感分析。以下是一个简单的实现示例：

```python
import numpy as np
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import LinearSVC

# 示例数据
corpus = [
    "I love this book",
    "This is a great movie",
    "The food is delicious",
    "I hate this class"
]

# 标签
labels = [1, 1, 1, -1]

# 将文本转换为词频矩阵
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(corpus)

# 训练线性SVM分类器
clf = LinearSVC()
clf.fit(X, labels)

# 预测新文本
new_text = "This class is not good"
X_new = vectorizer.transform([new_text])
prediction = clf.predict(X_new)
print(prediction)  # 输出 [-1]
```
