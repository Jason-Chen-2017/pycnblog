#### 2. 编写一个函数，用于生成个性化广告文案。

**题目：**

编写一个函数`generateAd`，该函数接收用户兴趣标签列表`userTags`和广告内容`adContent`，返回基于用户兴趣的个性化广告文案。

**答案解析：**

```python
import random

def generateAd(userTags, adContent):
    adCtemplate = "【{tag}】{adContent}"
    adTemplates = [adCtemplate.format(tag=tag, adContent=adContent) for tag in userTags]
    return random.choice(adTemplates)

# 假设adContent为通用广告文案，例如：“快来体验我们的美食之旅！”
```
