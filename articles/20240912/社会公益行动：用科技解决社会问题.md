#### 算法编程题库

### 1. 编写一个算法，用于计算公益项目的参与人数。

**题目描述：** 给定一个字符串，表示公益项目的参与人数，编写一个算法，计算参与人数。

**输入：** "20,30,40,50"

**输出：** 140

**答案：**

```python
def count_participants(numbers):
    return sum(int(n) for n in numbers.split(','))

# 示例
print(count_participants("20,30,40,50"))  # 输出 140
```

**解析：** 该算法使用字符串的 `split` 方法将输入字符串按逗号分割成列表，然后使用列表解析器将每个字符串转换为整数，并使用 `sum` 函数计算总和。

### 2. 编写一个算法，用于优化公益项目的推广策略。

**题目描述：** 给定一个用户行为数据集，包含用户点击、转发、捐赠等行为，编写一个算法，为每个用户推荐与其兴趣相关的公益项目。

**输入：** 用户行为数据集

**输出：** 用户兴趣相关的公益项目列表

**答案：**

```python
from collections import defaultdict

def recommend_projects(behaviors):
    project_interests = defaultdict(set)
    for behavior in behaviors:
        project_interests[behavior['project_id']].add(behavior['user_id'])
    recommended_projects = {}
    for project_id, users in project_interests.items():
        recommended_projects[project_id] = users
    return recommended_projects

# 示例
behaviors = [
    {'user_id': 1, 'project_id': 101, 'action': 'click'},
    {'user_id': 1, 'project_id': 102, 'action': 'forward'},
    {'user_id': 2, 'project_id': 201, 'action': 'donate'},
]
print(recommend_projects(behaviors))
```

**解析：** 该算法首先使用字典来存储每个公益项目的用户兴趣，然后为每个用户推荐与其兴趣相关的公益项目。这里使用了 `defaultdict` 来简化字典的初始化。
