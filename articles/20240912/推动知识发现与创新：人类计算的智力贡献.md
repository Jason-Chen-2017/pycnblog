**1. 寻找两个有序数组的中位数**

**题目描述：** 给定两个大小为 m 和 n 的有序数组 nums1 和 nums2，找出这两个数组的中位数。

**示例：**

```
nums1 = [1, 3]
nums2 = [2]
中位数是 2。
```

**答案解析：**

```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        nums = sorted(nums1 + nums2)
        n = len(nums)
        if n % 2 == 1:
            return nums[n // 2]
        else:
            return (nums[n // 2 - 1] + nums[n // 2]) / 2
```

**2. 最长公共子序列**

**题目描述：** 给定两个字符串 text1 和 text2，找出它们的最大公共子序列。

**示例：**

```
text1 = "abcde"
text2 = "ace"
最长公共子序列为 "ace"，其长度为 3。
```

**答案解析：**

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> str:
        dp = [[0] * (len(text2) + 1) for _ in range(len(text1) + 1)]

        for i in range(1, len(text1) + 1):
            for j in range(1, len(text2) + 1):
                if text1[i - 1] == text2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1] + 1
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

        res = []
        i, j = len(text1), len(text2)
        while i > 0 and j > 0:
            if text1[i - 1] == text2[j - 1]:
                res.append(text1[i - 1])
                i, j = i - 1, j - 1
            elif dp[i - 1][j] > dp[i][j - 1]:
                i -= 1
            else:
                j -= 1

        return ''.join(res[::-1])
```

**4. 最长回文子串**

**题目描述：** 给定一个字符串，找出其中最长的回文子串。

**示例：**

```
字符串 "babad" 应该返回 "bab"，因为它是字符串中最长的回文子串。
```

**答案解析：**

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        def f(l, r):
            while l >= 0 and r < len(s) and s[l] == s[r]:
                l, r = l - 1, r + 1
            return r - l - 1

        start, max_len = 0, 1
        for i in range(len(s)):
            a = f(i, i)
            b = f(i, i + 1)
            ans = max(a, b)
            if ans > max_len:
                start = i - (ans - 1) // 2
                max_len = ans

        return s[start : start + max_len]
```

**5. 子集划分问题**

**题目描述：** 给定一个无重复元素的整数数组 nums ，返回该数组所有可能的子集（幂集）。

**示例：**

```
nums = [1,2,3]
所有可能的子集为：
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```

**答案解析：**

```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        res = [[]]
        for v in nums:
            res += [t + [v] for t in res]
        return res
```

**6. 二进制表示中的质数个数**

**题目描述：** 给定一个整数 n，计算它在二进制表示中包含的质数个数。

**示例：**

```
n = 4，其二进制表示为 100，因此返回质数个数 2。
```

**答案解析：**

```python
class Solution:
    def countPrimes(self, n: int) -> int:
        if n < 2:
            return 0
        s = [True] * n
        s[0] = s[1] = False
        for i in range(2, int(n ** 0.5) + 1):
            if s[i]:
                for j in range(i * i, n, i):
                    s[j] = False
        return sum(s)
```

**7. 三数和**

**题目描述：** 给定一个整数数组 nums 和一个目标值 target，找出数组中三个元素的和为 target 的下标，并返回它们的和。

**示例：**

```
nums = [-1, 0, 1, 2, -1, -4]
target = 0
返回 [0, 1, 2]，因为 nums[0] + nums[1] + nums[2] = 0。
```

**答案解析：**

```python
class Solution:
    def threeSum(self, nums: List[int], target: int) -> List[List[int]]:
        nums.sort()
        ans = []
        for i in range(len(nums) - 2):
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            left, right = i + 1, len(nums) - 1
            while left < right:
                s = nums[i] + nums[left] + nums[right]
                if s < target:
                    left += 1
                elif s > target:
                    right -= 1
                else:
                    ans.append([nums[i], nums[left], nums[right]])
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
        return ans
```

**8. 搜索旋转排序数组**

**题目描述：** 给定一个旋转排序的数组，实现一个二分搜索算法，找出给定目标值的目标索引。如果目标值不存在，则返回它将在数组中插入的位置。

**示例：**

```
nums = [4, 5, 6, 7, 0, 1, 2]
target = 0
返回 4（数组中元素值为 0 的索引）。
```

**答案解析：**

```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        l, r = 0, len(nums) - 1
        while l <= r:
            mid = (l + r) // 2
            if nums[mid] == target:
                return mid
            if nums[l] <= nums[mid]:
                if nums[l] <= target < nums[mid]:
                    r = mid - 1
                else:
                    l = mid + 1
            else:
                if nums[mid] < target <= nums[r]:
                    l = mid + 1
                else:
                    r = mid - 1
        return l
```

**9. 打家劫舍**

```
nums = [1, 2, 3, 1]

```

**答案解析：**

```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        a, b = 0, 0
        for x in nums:
            a, b = b, max(a + x, b)
        return b
```

```
prices = [7, 1, 5, 3, 6, 4]
最大利润为 5，在第 2 天购买，在第 5 天出售（prices[1] = 1 和 prices[5] = 6）。
```

**答案解析：**

```python
class Solution:

        max_profit = 0
        for i in range(1, len(prices)):
            if prices[i] > prices[i - 1]:
                max_profit += prices[i] - prices[i - 1]
        return max_profit
```

**11. 合并两个有序链表**

**题目描述：** 将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

**示例：**

```
l1 = [1, 2, 4]
l2 = [1, 3, 4]
合并后的链表为 [1, 1, 2, 3, 4, 4]。
```

**答案解析：**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeTwoLists(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        if l1 is None:
            return l2
        if l2 is None:
            return l1
        if l1.val < l2.val:
            l1.next = self.mergeTwoLists(l1.next, l2)
            return l1
        else:
            l2.next = self.mergeTwoLists(l1, l2.next)
            return l2
```

**12. 汇总范围**

```
比如，数组 [1,2,3,4,5,6] 被分割为：[1,2,3], [4,5,6], []。
```

**然后返回数组形式的答案。**

```
nums = [0,0,0,2,0,0]
返回 [[0,0,0], [0,2,0], [0,0,0]]
```

**答案解析：**

```python
class Solution:
    def summaryRanges(self, nums: List[int]) -> List[List[int]]:
        if not nums:
            return []

        ans = []
        i = 0
        while i < len(nums):
            j = i
            while j < len(nums) and nums[j] == nums[i]:
```

**13. 删除链表的节点**

**题目描述：** 给定一个单向链表的头节点 `head` 和一个整数值 `val`，删除链表中所有值为 `val` 的节点。

**示例：**

```
head = [1, 2, 6, 3, 4, 5, 6], val = 6
删除值为 6 的节点后，链表变为 [1, 2, 3, 4, 5]。
```

**答案解析：**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def removeElements(self, head: Optional[ListNode], val: int) -> Optional[ListNode]:
        dummy = ListNode(0)
        dummy.next = head
        curr = dummy
        while curr.next:
            if curr.next.val == val:
                curr.next = curr.next.next
            else:
                curr = curr.next
        return dummy.next
```

**14. 合并区间**

**题目描述：** 给你一个区间列表，请你将其合并为越来越少数量的区间。

**示例：**

```
intervals = [[1,3],[2,6],[8,10],[15,18]]
合并后的区间列表为：
[
  [1,6],
  [8,10],
  [15,18]
]
```

**答案解析：**

```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        intervals.sort()
        ans = []
        for interval in intervals:
            if not ans or ans[-1][1] < interval[0]:
                ans.append(interval)
            else:
                ans[-1][1] = max(ans[-1][1], interval[1])
        return ans
```

**15. 合并有序数组**

**题目描述：** 给你两个有序整数数组 `nums1` 和 `nums2`，请你将 `nums2` 合并到 `nums1` 中，使 `nums1` 成为一个有序数组。

**示例：**

```
nums1 = [1,2,3,0,0,0], m = 3
nums2 = [2,5,6], n = 3
合并后的数组为 [1,2,2,3,5,6]。
```

**答案解析：**

```python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        """
        Do not return anything, modify nums1 in-place instead.
        """
        i, j, k = m - 1, n - 1, m + n - 1
        while i >= 0 and j >= 0:
            if nums1[i] > nums2[j]:
                nums1[k] = nums1[i]
                i -= 1
            else:
                nums1[k] = nums2[j]
                j -= 1
            k -= 1
        while j >= 0:
            nums1[k] = nums2[j]
            j, k = j - 1, k - 1
```

**16. 有效的括号字符串**

**题目描述：** 给你一个由括号字符串 `s` 组成的非空字符串，判断是否为一个有效字符串。有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

**示例：**

```
有效字符串为 ""，"()" 和 "()()"。
无效字符串包括 "(()","())()" 和 "("。
```

**答案解析：**

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stk = []
        for c in s:
            if c in '([{':
                stk.append(c)
            elif not stk or (c == ')' and stk[-1] != '(' or c == ']' and stk[-1] != '[' or c == '}' and stk[-1] != '{'):
                return False
            else:
                stk.pop()
        return not stk
```

**17. 二进制表示中的质数个数**

**题目描述：** 给定一个整数 n，计算它在二进制表示中包含的质数个数。

**示例：**

```
n = 4，其二进制表示为 100，因此返回质数个数 2。
```

**答案解析：**

```python
class Solution:
    def countPrimes(self, n: int) -> int:
        if n < 2:
            return 0
        s = [True] * n
        s[0] = s[1] = False
        for i in range(2, int(n ** 0.5) + 1):
            if s[i]:
                for j in range(i * i, n, i):
                    s[j] = False
        return sum(s)
```

**18. 环形链表**

```
给定链表 3->2->0->-4，判断链表中是否有环。
```

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        s = set()
        while head:
            if head in s:
                return True
            s.add(head)
            head = head.next
        return False
```

**20. 验证回文串**

**示例：**

```

```

**答案解析：**

```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        s = ''.join(c.lower() for c in s if c.isalnum())
        return s == s[::-1]
```
