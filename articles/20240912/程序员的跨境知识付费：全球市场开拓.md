### 二、算法编程题库及答案解析

#### 1. 货币兑换最小汇率

**题目：** 给定一组货币兑换汇率，编写一个算法计算最小汇率。

**答案：**

```python
def min_exchange(rates):
    min_rate = min(rates)
    for i in range(len(rates)):
        rates[i] /= min_rate
    return rates
```

**解析：** 该算法首先找到最小汇率，然后将所有汇率除以最小汇率，得到最小汇率的相对值。

#### 2. 跨境物流路线规划

**题目：** 给定一组跨境物流路线，编写一个算法计算最短路线。

**答案：**

```python
import heapq

def min_route(routes):
    min_heap = []
    for route in routes:
        heapq.heappush(min_heap, route[0])
    result = []
    while min_heap:
        result.append(heapq.heappop(min_heap))
    return result
```

**解析：** 该算法使用堆排序算法，找到最短路线。
