#### 2. 算法编程题库

##### 2.1. 腾讯 - 设计一个算法，找出数组中两个数的最小距离？

**问题描述：** 给定一个整数数组 `nums`，找出两个数的最小距离，使得它们在数组中的位置相邻。

**示例：**
```
输入：nums = [1, 3, 1]
输出：2
解释：数组中，最接近的两个元素是 `1` 和 `3`，它们在数组中的位置相邻，距离为 `2`。
```

**答案解析：**

这是一个典型的查找问题，可以通过排序和二分查找来解决。

1. **排序：** 首先将数组 `nums` 排序，时间复杂度为 O(nlogn)。
2. **二分查找：** 排序后，对于数组中的每个元素，使用二分查找找到与其相邻的元素，计算它们之间的距离，取最小值。

以下是 Python 代码实现：

```python
def minDistance(nums):
    nums.sort()
    min_dist = float('inf')
    for i in range(1, len(nums)):
        min_dist = min(min_dist, nums[i] - nums[i-1])
    return min_dist
```

时间复杂度为 O(nlogn)，空间复杂度为 O(1)。
