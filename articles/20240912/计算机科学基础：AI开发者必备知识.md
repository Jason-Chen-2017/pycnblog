### 1. 数据结构与算法

#### 1.1. 数据结构

**题目：** 请描述链表、数组、栈和队列的区别及用途。

**答案：** 

| 数据结构 | 描述 | 用途 |
| --- | --- | --- |
| 链表 | 通过指针连接的节点序列，无需连续内存空间 | 实现跳表、双向链表、循环链表等数据结构，适合动态增删节点 |
| 数组 | 连续的内存空间，支持随机访问 | 实现快速查找和排序，适用于固定大小的数据集 |
| 栈 | 后进先出（LIFO）的数据结构 | 适用于函数调用栈、表达式求值等场景 |
| 队列 | 先进先出（FIFO）的数据结构 | 适用于消息队列、等待队列等场景 |

#### 1.2. 算法

**题目：** 请简要描述以下排序算法及其时间复杂度：冒泡排序、选择排序、插入排序、快速排序。

**答案：** 

| 排序算法 | 描述 | 时间复杂度 |
| --- | --- | --- |
| 冒泡排序 | 比较相邻元素，若逆序则交换，重复过程直到排序完成 | O(n^2) |
| 选择排序 | 找到未排序部分的最小元素，将其与第一个元素交换，重复过程直到排序完成 | O(n^2) |
| 插入排序 | 将未排序部分的一个元素插入到已排序部分的正确位置，重复过程直到排序完成 | O(n^2) |
| 快速排序 | 选择一个基准元素，将小于基准的元素放在其左侧，大于基准的元素放在其右侧，递归地对左右子序列进行快速排序 | 平均 O(n log n)，最坏 O(n^2) |

### 3. 并发编程

#### 3.1. 并发模型

**题目：** 请简要描述以下并发模型：线程、进程、协程。

**答案：** 

| 并发模型 | 描述 |
| --- | --- |
| 线程 | 轻量级的执行单元，共享同一进程的资源，但独立执行 |
| 进程 | 独立的执行单元，拥有独立的内存空间和其他资源 |
| 协程 | 轻量级的线程，具有独立的栈和执行上下文，但共享进程的资源 |
