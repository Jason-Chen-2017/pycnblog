#### 5. 编写一个循环经济算法，实现资源的回收再利用。

**答案：** 

```python
def recycle_resources(input_resources):
    recycled_resources = []
    
    for resource in input_resources:
        if resource.is_recyclable():
            recycled_resources.append(resource)
            resource.recycle()
        else:
            resource.dispose()
    
    return recycled_resources

class Resource:
    def __init__(self, name, recyclable):
        self.name = name
        self.recyclable = recyclable
    
    def is_recyclable(self):
        return self.recyclable
    
    def recycle(self):
        print(f"{self.name} has been recycled.")
    
    def dispose(self):
        print(f"{self.name} has been disposed.")

input_resources = [Resource("Plastic", True), Resource("Glass", True), Resource("Metal", True), Resource("Waste", False)]
recycled_resources = recycle_resources(input_resources)
print("Recycled resources:", recycled_resources)
```

**解析：** 此题目考察对循环经济概念的理解和编程实现能力，涉及对象的创建和方法的调用。
