#### 算法编程题库

**题目：** 设计一个算法，根据双目标清单系统，从一组给定的任务中选择两个最重要的任务。

**输入：** 一组任务（例如：["任务A", "任务B", "任务C", ...]）和一个权重数组（例如：[3, 2, 1]），权重表示每个任务的重要程度。

**输出：** 返回两个最重要的任务及其权重。

**答案：** 一种简单的方法是使用贪心算法，从权重数组中选择两个最大的权重，然后从任务数组中选择对应的任务。以下是 Python 代码示例：

```python
def find_two_important_tasks(tasks, weights):
    # 将任务和权重配对，并按权重降序排序
    task_weights = list(zip(tasks, weights))
    task_weights.sort(key=lambda x: x[1], reverse=True)

    # 返回前两个任务的名称和权重
    return [(task, weight) for task, weight in task_weights[:2]]

# 示例
tasks = ["任务A", "任务B", "任务C"]
weights = [3, 2, 1]
print(find_two_important_tasks(tasks, weights))
```

**解析：** 这个算法的时间复杂度为 O(nlogn)，其中 n 是任务的数量。在实际应用中，可以根据具体情况调整排序策略，例如使用堆实现 O(n) 的排序。
