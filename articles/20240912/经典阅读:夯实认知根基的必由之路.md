#### 面试题库

**1. TCP 和 UDP 的区别**

**题目：** 请解释 TCP 和 UDP 的区别，并分别描述它们在互联网应用中的适用场景。

**答案：**

TCP（传输控制协议）和 UDP（用户数据报协议）是互联网中最常用的两种传输层协议。

- **TCP：** 连接导向，提供可靠的数据传输服务，确保数据的顺序正确到达。适用于对数据完整性要求较高的应用，如文件传输、邮件发送等。
- **UDP：** 非连接导向，提供不可靠的数据传输服务，数据可能会丢失或重复。适用于对实时性要求较高的应用，如在线游戏、视频直播等。

**2. 如何在 Golang 中实现一个无缓冲的通道？**

**题目：** 编写一个 Golang 程序，实现一个无缓冲的通道，并演示其阻塞特性。

**答案：**

```go
package main

import "fmt"

func main() {
    c := make(chan int)
    msg := 42

    go func() {
        c <- msg
    }()

    fmt.Println(<-c) // 等待接收数据并打印
}
```

**3. 快排的实现原理及代码**

**题目：** 请解释快速排序的实现原理，并给出一个快速排序的 Golang 实现代码。

**答案：**

快速排序是一种高效的排序算法，其基本思想是通过一趟排序将待排序的数据分割成独立的两部分，其中一部分的所有数据都比另一部分的所有数据要小，然后递归地对这两部分数据继续进行快速排序。

```go
package main

import "fmt"

func quickSort(arr []int, low, high int) {
    if low < high {
        pi := partition(arr, low, high)
        quickSort(arr, low, pi-1)
        quickSort(arr, pi+1, high)
    }
}

func partition(arr []int, low, high int) int {
    pivot := arr[high]
    i := low - 1
    for j := low; j < high; j++ {
        if arr[j] < pivot {
            i++
            arr[i], arr[j] = arr[j], arr[i]
        }
    }
    arr[i+1], arr[high] = arr[high], arr[i+1]
    return i + 1
}

func main() {
    arr := []int{10, 7, 8, 9, 1, 5}
    quickSort(arr, 0, len(arr)-1)
    fmt.Println(arr)
}
```

**4. 冒泡排序的实现原理及代码**

**题目：** 请解释冒泡排序的实现原理，并给出一个冒泡排序的 Golang 实现代码。

**答案：**

冒泡排序是一种简单的排序算法，其基本思想是通过反复交换相邻的逆序对，使得数组中最大或最小的元素逐步“冒泡”到数组的端部。

```go
package main

import "fmt"

func bubbleSort(arr []int) {
    n := len(arr)
    for i := 0; i < n; i++ {
        for j := 0; j < n-i-1; j++ {
            if arr[j] > arr[j+1] {
                arr[j], arr[j+1] = arr[j+1], arr[j]
            }
        }
    }
}

func main() {
    arr := []int{64, 34, 25, 12, 22, 11, 90}
    bubbleSort(arr)
    fmt.Println(arr)
}
```

#### 算法编程题库

**1. 最长公共前缀**

**题目：** 编写一个函数，用来找出字符串数组中的最长公共前缀。

**答案：**

```go
package main

import "fmt"

func longestCommonPrefix(strs []string) string {
    if len(strs) == 0 {
        return ""
    }
    for i := 0; i < len(strs[0]); i++ {
        for j := 1; j < len(strs); j++ {
            if i >= len(strs[j]) || strs[0][i] != strs[j][i] {
                return strs[0][:i]
            }
        }
    }
    return strs[0]
}

func main() {
    strs := []string{"flower", "flow", "flight"}
    fmt.Println(longestCommonPrefix(strs))
}
```

**2. 两数相加**

**题目：** 不使用 + 或 - 运算符，实现两个整数的加法。

**答案：**

```go
package main

import "fmt"

func add(a int, b int) int {
    for b != 0 {
        carry := a & b
        a = a ^ b
        b = carry << 1
    }
    return a
}

func main() {
    a := 1
    b := 2
    fmt.Println(add(a, b))
}
```

**3. 最长公共子序列**

**题目：** 给定两个字符串，找出它们的最长公共子序列。

**答案：**

```go
package main

import (
    "fmt"
    "strings"
)

func longestCommonSubsequence(text1, text2 string) string {
    dp := make([][]int, len(text1)+1)
    for i := range dp {
        dp[i] = make([]int, len(text2)+1)
    }
    for i := 1; i <= len(text1); i++ {
        for j := 1; j <= len(text2); j++ {
            if text1[i-1] == text2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    result := make([]rune, dp[len(text1)][len(text2)])
    i, j := len(text1), len(text2)
    for i > 0 && j > 0 {
        if text1[i-1] == text2[j-1] {
            result = append(result, text1[i-1])
            i--
            j--
        } else if dp[i-1][j] > dp[i][j-1] {
            i--
        } else {
            j--
        }
    }
    reverse(result)
    return string(result)
}

func reverse(s []rune) {
    for i, j := 0, len(s)-1; i < j; i, j = i+1, j-1 {
        s[i], s[j] = s[j], s[i]
    }
}

func main() {
    text1 := "abcde"
    text2 := "ace"
    fmt.Println(longestCommonSubsequence(text1, text2))
}
```
