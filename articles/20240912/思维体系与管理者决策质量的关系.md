#### 面试题 1：深度优先搜索与广度优先搜索

**题目：** 实现一个函数，使用深度优先搜索和广度优先搜索算法分别求解给定图的节点路径问题。

**答案解析：** 深度优先搜索（DFS）和广度优先搜索（BFS）是两种常见的图搜索算法。DFS 沿着一条路径深入到底，而 BFS 则逐层搜索。两种算法在求解节点路径问题时各有优缺点，具体实现如下：

```python
# 深度优先搜索实现
def dfs(graph, start, target):
    visited = set()
    stack = [(start, [start])]

    while stack:
        node, path = stack.pop()
        if node not in visited:
            visited.add(node)
            if node == target:
                return path
            for next_node in graph[node]:
                if next_node not in visited:
                    stack.append((next_node, path + [next_node]))

    return None

# 广度优先搜索实现
def bfs(graph, start, target):
    visited = set()
    queue = [(start, [start])]

    while queue:
        node, path = queue.pop(0)
        if node not in visited:
            visited.add(node)
            if node == target:
                return path
            for next_node in graph[node]:
                if next_node not in visited:
                    queue.append((next_node, path + [next_node]))

    return None
```

**解析：** 通过以上代码实现，DFS 和 BFS 分别求解给定图的节点路径问题。DFS 在寻找较短的路径时性能更佳，而 BFS 则适用于寻找最短路径。

#### 面试题 2：设计一个LRU缓存算法

**题目：** 设计一个 LRU（Least Recently Used）缓存算法，实现一个支持缓存的数据结构，并提供 `get` 和 `put` 操作。

**答案解析：** LRU 算法是一种常用的缓存替换策略，通过记录缓存项的访问时间来淘汰最久未使用的缓存项。以下是一个基于哈希表和双向链表实现的 LRU 缓存算法：

```python
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = OrderedDict()

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.cache.move_to_end(key)
        return self.cache[key]

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)
```

**解析：** 通过以上代码实现，LRU 缓存算法能够在 O(1) 时间复杂度下完成 `get` 和 `put` 操作。当缓存容量超过预设值时，自动淘汰最久未使用的缓存项。
