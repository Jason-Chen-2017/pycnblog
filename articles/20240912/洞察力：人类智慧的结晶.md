**答案**：
- **缓存一致性**：保证缓存和数据库中的数据一致性。
- **缓存命中率**：提高缓存命中率，减少对数据库的访问。
- **缓存淘汰策略**：选择合适的缓存淘汰策略，如LRU（最近最少使用）、LFU（最不常用）等。

### 二、算法编程题

#### 1. 最长公共子序列

**题目**：给定两个字符串，求它们的最长公共子序列。

**答案**：
```go
func longestCommonSubsequence(s1 string, s2 string) string {
    m, n := len(s1), len(s2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if s1[i-1] == s2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    idx := dp[m][n]
    ans := make([]byte, idx)
    i, j := m, n
    for idx > 0 {
        if s1[i-1] == s2[j-1] {
            ans[idx-1] = s1[i-1]
            i--
            j--
            idx--
        } else if dp[i-1][j] > dp[i][j-1] {
            i--
        } else {
            j--
        }
    }
    return string(ans)
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

**解析**：使用动态规划求解最长公共子序列问题，时间复杂度为O(mn)，空间复杂度为O(mn)。

#### 2. 单调栈

**题目**：实现一个单调栈，支持以下操作：push(x)，pop()，max()。

**答案**：
```go
type MonotonicStack struct {
    s []int
}

func NewMonotonicStack() *MonotonicStack {
    return &MonotonicStack{s: []int{}}
}

func (m *MonotonicStack) Push(x int) {
    for len(m.s) > 0 && x > m.s[len(m.s)-1] {
        m.s = m.s[:len(m.s)-1]
    }
    m.s = append(m.s, x)
}

func (m *MonotonicStack) Pop() int {
    return m.s[len(m.s)-1]
}

func (m *MonotonicStack) Max() int {
    return m.s[len(m.s)-1]
}
```

**解析**：使用单调栈实现最大元素查找，时间复杂度为O(1)。

#### 3. 签到请求数组

**题目**：给定一个整数数组 `signs` 表示 `n` 天内进行签到，每个位置上的整数代表签到情况，1 表示签到，0 表示未签到。请返回一个长度为 `m` 的数组，其中第 `i`（`0 <= i < m`）个元素是到第 `i` 天结束累计签到的有效天数。

**答案**：
```go
func checkRecord(signs []int) []int {
    ans := make([]int, len(signs))
    cnt := 0
    for i, sign := range signs {
        if sign == 0 {
            cnt++
            if cnt >= 3 {
                cnt = 0
                continue
            }
        }
        ans[i] = i - cnt
    }
    return ans
}
```

**解析**：遍历数组，遇到连续的0计数，当计数达到3时重置，否则累加。
