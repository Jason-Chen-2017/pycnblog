**2. 如何使用贝叶斯分类器进行文本分类？**

**答案：** 使用贝叶斯分类器进行文本分类，可以采用朴素贝叶斯（Naive Bayes）模型。以下是一个使用 scikit-learn 库实现文本分类的简单示例：

```python
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import make_pipeline

# 创建文本数据集
data = [
    "机器学习是一种数据分析方法，它使用算法从数据中学习，以预测未知数据的结果。",
    "深度学习是一种基于神经网络的机器学习技术，它在图像识别、语音识别和自然语言处理等领域取得了显著成果。",
    "贝叶斯分类器是一种基于贝叶斯定理的统计分类器，它在文本分类任务中表现出色。",
]

labels = ["机器学习", "深度学习", "贝叶斯分类器"]

# 创建文本分类器
model = make_pipeline(CountVectorizer(), MultinomialNB())

# 训练模型
model.fit(data, labels)

# 预测新文本
new_data = "深度学习算法在图像识别领域具有广泛的应用。"
prediction = model.predict([new_data])
print("预测结果：", prediction)
```
