## 例1：查找算法

我们来实现一个查找算法，用于查找一个整数数组中的一个整数。我们可以使用禅意思考的方法来观察问题，并尝试找到最简单和最直接的解决方案。

```python
def find(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1
```

在这个例子中，我们使用了一个简单的线性查找算法。我们观察了问题，并尝试找到最简单和最直接的解决方案。这个算法的时间复杂度是O(n)，空间复杂度是O(1)。

## 例2：排序算法

我们来实现一个排序算法，用于对一个整数数组进行排序。我们可以使用禅意思考的方法来观察问题，并尝试找到最简单和最直接的解决方案。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个例子中，我们使用了一个简单的冒泡排序算法。我们观察了问题，并尝试找到最简单和最直接的解决方案。这个算法的时间复杂度是O(n^2)，空间复杂度是O(1)。
