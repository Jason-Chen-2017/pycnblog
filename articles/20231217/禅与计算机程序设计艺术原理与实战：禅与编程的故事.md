## 3.1 深度优先搜索

深度优先搜索(Depth-First Search, DFS)是一种常用的图遍历算法，它通过不断地沿着一个路径向下探索，直到达到叶子节点为止。这种方法可以用来解决许多问题，包括寻找路径、查找循环和解决迷宫等。

在禅学中，深度优先搜索可以看作是一种直接的方法来探索问题的内部结构。通过深入地探索问题的各个方面，我们可以更好地理解问题，并找到更好的解决方案。

## 3.2 广度优先搜索

广度优先搜索(Breadth-First Search, BFS)是另一种图遍历算法，它通过从起始节点开始，沿着一条路径向外扩展，直到所有节点都被遍历为止。这种方法可以用来解决许多问题，包括寻找最短路径、查找连通分量和解决迷宫等。

在禅学中，广度优先搜索可以看作是一种更全面的方法来探索问题的内部结构。通过广泛地探索问题的各个方面，我们可以更好地理解问题，并找到更好的解决方案。

## 3.3 动态规划

动态规划(Dynamic Programming, DP)是一种常用的优化算法，它通过将问题分解为更小的子问题，并将这些子问题的解存储在一个表格中，以便于后续使用。这种方法可以用来解决许多问题，包括最长公共子序列、最长递增子序列和最短路径等。

在禅学中，动态规划可以看作是一种更有效的方法来解决问题。通过将问题分解为更小的子问题，我们可以更好地理解问题，并找到更好的解决方案。

## 4.1 寻找最短路径

假设我们有一个有向图，我们需要找到从起始节点到目标节点的最短路径。我们可以使用广度优先搜索算法来解决这个问题。

```python
from collections import deque

def bfs(graph, start, end):
    visited = set()
    queue = deque([(start, [start])])

    while queue:
        node, path = queue.popleft()
        if node == end:
            return path
        visited.add(node)
        for neighbor in graph[node]:
            if neighbor not in visited:
                queue.append((neighbor, path + [neighbor]))
```

在这个例子中，我们使用了广度优先搜索算法来找到最短路径。我们通过从起始节点开始，沿着一条路径向外扩展，直到找到目标节点为止。通过保持开放和灵活的心态，我们可以尝试不同的解决方案，并找到最短的路径。
