# 4.具体代码实例和详细解释说明

## 4.1 Spring Boot 具体代码实例

```java
@SpringBootApplication
public class SpringBootApplication {

    public static void main(String[] args) {
        SpringApplication.run(SpringBootApplication.class, args);
    }
}
```

上述代码是一个简单的 Spring Boot 项目的入口类，它使用 `@SpringBootApplication` 注解来自动配置 Spring 应用，并使用 `SpringApplication.run()` 方法来启动 Spring 应用。

## 4.2 Spring Cloud 具体代码实例

```java
@SpringBootApplication
@EnableDiscoveryClient
@EnableFeignClients
public class SpringCloudApplication {

    public static void main(String[] args) {
        SpringApplication.run(SpringCloudApplication.class, args);
    }
}
```

上述代码是一个简单的 Spring Cloud 项目的入口类，它使用 `@SpringBootApplication` 注解来自动配置 Spring 应用，并使用 `@EnableDiscoveryClient` 和 `@EnableFeignClients` 注解来启用服务发现和负载均衡。
