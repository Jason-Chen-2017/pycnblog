## 1.神经元（Neurons）

神经元是大脑中最基本的信息处理单元。它们接收来自其他神经元的信号，进行处理，并向其他神经元发送信号。神经元由以下部分组成：

- 输入端（Dendrites）：接收来自其他神经元的信号的部分。
- 主体（Soma）：包含了神经元的大部分生物学功能，如生长、维持等。
- 输出端（Axon）：传递信号给其他神经元的部分。

神经元的活动由它们的输入信号和权重决定。权重表示神经元对输入信号的影响程度。神经元的输出是输入信号通过激活函数后的结果。

## 2.激活函数（Activation Functions）

激活函数是神经网络中一个关键概念。它用于将神经元的输入信号转换为输出信号。激活函数的目的是引入非线性，使得神经网络能够学习更复杂的模式。

常见的激活函数有：

-  sigmoid 函数（S）：S(x) = 1 / (1 + exp(-x))
-  hyperbolic tangent 函数（tanh）：tanh(x) = (exp(x) - exp(-x)) / (exp(x) + exp(-x))
-  ReLU 函数（Rectified Linear Unit）：f(x) = max(0, x)

## 3.前馈神经网络（Feedforward Neural Networks）

前馈神经网络是一种简单的神经网络结构，其输入通过一系列层传递到输出层。每个层中的神经元只接收来自前一层的信号，不能接收来自同一层或后一层的信号。

前馈神经网络的学习过程通常涉及以下步骤：

1. 初始化神经网络权重。
2. 使用训练数据计算输出层的损失。
3. 使用反向传播算法更新权重，以最小化损失。
4. 重复步骤2和3，直到损失达到满足要求的水平，或者达到最大迭代次数。

## 4.反馈神经网络（Recurrent Neural Networks）

反馈神经网络（RNNs）是一种处理序列数据的神经网络结构。与前馈神经网络不同，RNNs中的神经元可以接收来自同一层或后一层的信号。这使得RNNs能够捕捉序列中的长期依赖关系。

RNNs的学习过程与前馈神经网络类似，但由于其反馈结构，需要特殊的处理方法来避免梯度消失/梯度爆炸问题。

## 5.深度学习（Deep Learning）

深度学习是一种通过多层神经网络实现的机器学习方法。深度学习模型可以自动学习表示，从而在处理复杂数据时表现出色。
