## 3.1 物流优化

### 3.1.1 运输路线规划

运输路线规划是物流中一个重要的问题，目标是找到一条最短的、最经济的运输路线。这个问题可以用图论来解决。

首先，我们需要构建一个有向图，其中每个节点表示一个城市，每条边表示一条路径，边的权重表示路径的长度。然后，我们可以使用迪杰斯特拉（Dijkstra）算法或者贝尔曼福特（Bellman-Ford）算法来求解最短路径问题。

迪杰斯特拉算法的基本思想是从起点开始，逐步扩展到其他节点，直到所有节点都被访问为止。具体步骤如下：

1. 将起点节点加入到优先级队列中，其优先级为0。
2. 从优先级队列中弹出一个节点，并将其优先级设为无穷大。
3. 遍历该节点的所有邻居节点，如果通过当前节点到该邻居节点的路径长度小于该邻居节点原有的最短路径长度，则更新该邻居节点的最短路径长度和对应的前驱节点。
4. 将该邻居节点加入到优先级队列中，其优先级为更新后的最短路径长度。
5. 重复步骤2-4，直到所有节点都被访问为止。

## 4.1 运输路线规划

### 4.1.1 迪杰斯特拉算法

```python
import heapq

def dijkstra(graph, start, end):
    distances = {node: float('inf') for node in graph}
    distances[start] = 0
    priority_queue = [(0, start)]

    while priority_queue:
        current_distance, current_node = heapq.heappop(priority_queue)

        if current_distance > distances[current_node]:
            continue

        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(priority_queue, (distance, neighbor))

    return distances[end]
```

### 4.1.2 贝尔曼福特算法

```python
def bellman_ford(graph, start, end):
    distances = {node: float('inf') for node in graph}
    distances[start] = 0

    for _ in range(len(graph) - 1):
        for node in graph:
            for neighbor, weight in graph[node].items():
                distance = distances[node] + weight

                if distance < distances[neighbor]:
                    distances[neighbor] = distance

    return distances[end]
```
