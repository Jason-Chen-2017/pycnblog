```python
def dfs(graph, start):
    visited = set()
    stack = [start]
    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)
    return visited
```

2. 广度优先搜索（BFS）：

```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    while queue:
        vertex = queue.popleft()
        if vertex not in visited:
            visited.add(vertex)
            queue.extend(graph[vertex] - visited)
    return visited
```

3. 动态规划（DP）：

```python
def dp(coins, amount):
    dp = [0] * (amount + 1)
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] = max(dp[i], dp[i - coin] + coin)
    return dp[amount]
```

4. 贪心算法（GA）：

```python
def greedy(coins, amount):
    coins.sort(reverse=True)
    total = 0
    for coin in coins:
        if coin <= amount:
            total += coin
            amount -= coin
    return total
```
