## 1.4 Python数据可视化的具体代码实例和详细解释说明

### 1.4.1 条形图实例

```python
import matplotlib.pyplot as plt

# 数据
categories = ['A', 'B', 'C', 'D', 'E']
values = [10, 20, 30, 40, 50]

# 创建条形图
plt.bar(categories, values)

# 显示图表
plt.show()
```

上述代码创建了一个简单的条形图，展示了五个类别的值。具体解释如下：

1. 导入Matplotlib库。
2. 定义数据，包括类别和值。
3. 使用`plt.bar()`函数创建条形图，将类别和值作为参数传入。
4. 使用`plt.show()`函数显示图表。

上述代码创建了一个简单的饼图，展示了四个类别的比例。具体解释如下：

4. 使用`plt.show()`函数显示图表。
