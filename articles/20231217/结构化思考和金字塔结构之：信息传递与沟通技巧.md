```python

import time

    def __init__(self):
        super(Publisher, self).__init__()
        self.topic = "news"

    def run(self):
        while True:
            print("Breaking news: The stock market is booming!")
            time.sleep(1)

class Subscriber(threading.Thread):
    def __init__(self, topic):
        super(Subscriber, self).__init__()
        self.topic = topic

    def run(self):
        while True:
            if self.topic == "news":
                print("I'm listening...")
            time.sleep(1)

publisher = Publisher()
subscriber = Subscriber("news")

publisher.start()
subscriber.start()
```

在这个例子中，我们将信息从最具体的层次建立到最抽象的层次，即发布-订阅的方式来实现信息传递和沟通。通过这种方式，我们可以确保信息的准确性和完整性，同时也可以提高信息传递的效率和效果。
