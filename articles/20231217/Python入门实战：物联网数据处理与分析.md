# 4.具体代码实例和详细解释说明
# 4.1 数据清洗

在本节中，我们将介绍一个简单的数据清洗示例。假设我们有一个包含温度、湿度和气压的数据集，我们需要对数据进行清洗。

```python
import pandas as pd
import numpy as np

# 创建一个包含温度、湿度和气压的数据集
data = {'temperature': [22, 23, 24, 25, 26, 27, 28, 29, 30],
        'humidity': [40, 45, 50, 55, 60, 65, 70, 75, 80],
        'pressure': [1000, 1010, 1020, 1030, 1040, 1050, 1060, 1070, 1080]}

# 创建一个DataFrame
df = pd.DataFrame(data)

# 检查数据是否缺失
print(df.isnull().sum())

# 删除缺失的数据
df = df.dropna()

# 检查数据是否重复
print(df.duplicated().sum())

# 删除重复的数据
df = df.drop_duplicates()

# 截断超出范围的数据
df['temperature'] = df['temperature'].clip(lower=0, upper=100)

# 将数据转换为数字数据
df['temperature'] = df['temperature'].astype(int)
df['humidity'] = df['humidity'].astype(int)
df['pressure'] = df['pressure'].astype(int)

# 将数据聚合为一个数据集
df = df.reset_index(drop=True)
```
