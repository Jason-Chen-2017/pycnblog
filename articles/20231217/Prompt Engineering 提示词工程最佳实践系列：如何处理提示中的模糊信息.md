# 2.核心概念与联系

在处理提示中的模糊信息时，我们需要了解一些关键的概念和联系。以下是一些重要的概念：

- **模糊信息**：模糊信息是指在提示中，用户提供的信息不清晰、不完整或不准确的信息。这可能导致AI模型无法理解用户的需求，从而产生不准确或不完整的回答。
- **清晰度**：清晰度是指提示中信息的清晰程度。更清晰的提示可以帮助AI模型更好地理解用户的需求，从而提供更准确的回答。
- **提示词工程**：提示词工程是一种技术方法，旨在设计和优化与AI模型互动的问题陈述。通过提示词工程，我们可以提高AI模型的回答质量，并减少模糊信息对回答准确性的影响。
