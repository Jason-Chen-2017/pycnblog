## 4.1错误处理的代码实例

在本例中，我们将使用PHP语言来编写一个简单的MySQL错误处理示例。

```php
<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "mydb";

// 创建连接
$conn = new mysqli($servername, $username, $password, $dbname);

// 检查连接
if ($conn->connect_error) {
  die("连接失败: " . $conn->connect_error);
}

$sql = "SELECT id, name FROM mytable WHERE age > 18";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
  while($row = $result->fetch_assoc()) {
    echo "id: " . $row["id"]. " - Name: " . $row["name"]. "<br>";
  }
} else {
  echo "0 结果";
}
$conn->close();
?>
```

在上述代码中，我们首先创建了一个MySQL连接，并检查连接是否成功。如果连接失败，我们将输出错误信息并终止程序执行。接着，我们执行一个查询语句，并检查查询结果是否有效。如果查询结果有效，我们将输出结果；否则，我们将输出“0 结果”。

## 4.2异常处理的代码实例

在本例中，我们将使用Python语言来编写一个简单的MySQL异常处理示例。

```python
import mysql.connector

try:
    cnx = mysql.connector.connect(user='root', password='', host='localhost', database='mydb')
    cursor = cnx.cursor()
    query = "SELECT id, name FROM mytable WHERE age > 18"
    cursor.execute(query)
    for (id, name) in cursor:
        print(f"id: {id} - Name: {name}")
except mysql.connector.Error as err:
    if err.errno == errorcode.ER_ACCESS_DENIED_ERROR:
        print("Something is wrong with your user name or password")
    elif err.errno == errorcode.ER_BAD_DB_ERROR:
        print("Database does not exist")
    else:
        print(err)
finally:
    if cnx:
        cnx.close()
```

在上述代码中，我们首先尝试创建一个MySQL连接，并使用try语句来捕获可能发生的异常。如果发生异常，我们将根据异常类型输出相应的错误信息。最后，我们使用finally语句来关闭连接。
