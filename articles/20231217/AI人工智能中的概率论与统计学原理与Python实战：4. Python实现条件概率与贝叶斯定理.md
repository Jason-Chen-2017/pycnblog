## 1.2 条件概率和贝叶斯定理的核心概念

条件概率和贝叶斯定理是概率论和统计学中的核心概念。它们为我们提供了一种处理条件和依赖关系的方法，从而帮助我们更好地理解和解决问题。

### 1.2.1 条件概率

条件概率是概率论中的一个核心概念，它描述了一个事件发生的概率，给定另一个事件已经发生的情况。形式上，条件概率可以表示为：

P(A|B) = P(A ∩ B) / P(B)

其中，P(A|B) 是条件概率，表示事件A发生的概率，给定事件B已经发生；P(A ∩ B) 是事件A和事件B同时发生的概率；P(B) 是事件B发生的概率。

### 1.2.2 贝叶斯定理

贝叶斯定理是概率论中的一个重要定理，它描述了如何更新已有的概率估计，给定新的信息。贝叶斯定理可以表示为：

P(A|B) = P(B|A) * P(A) / P(B)

其中，P(A|B) 是条件概率，表示事件A发生的概率，给定事件B已经发生；P(B|A) 是条件概率，表示事件B发生的概率，给定事件A已经发生；P(A) 是事件A发生的概率；P(B) 是事件B发生的概率。

贝叶斯定理可以帮助我们更新我们对某个事件的概率估计，给定新的信息。这在人工智能和机器学习中非常有用，因为我们经常需要根据新的数据来更新我们的模型和预测。
