# 4.具体代码实例和详细解释说明
在这里，我们以一个简单的计数器示例来展示 Uni-app 框架的具体代码实例和解释。

1. 创建一个名为 `counter.vue` 的页面文件，编写以下代码：

```javascript
<template>
  <view>
    <text>{{ count }}</text>
    <button bind:tap="increment">+</button>
    <button bind:tap="decrement">-</button>
  </view>
</template>

<script>
export default {
  data: {
    count: 0
  },
  methods: {
    increment() {
      this.count++;
    },
    decrement() {
      this.count--;
    }
  }
};
</script>
```

2. 在项目的 `pages.json` 文件中，添加以下代码：

```json
{
  "pages": [
    "counter/counter"
  ]
}
```

3. 运行项目，可以看到一个简单的计数器示例，每次点击 `+` 按钮，计数器值增加1；每次点击 `-` 按钮，计数器值减少1。
