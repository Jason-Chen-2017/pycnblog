## 4.1 规则存储示例

在这个示例中，我们将使用Python编程语言来实现规则存储。我们将使用SQLite数据库来存储规则。

```python
import sqlite3

# 创建数据库
conn = sqlite3.connect('rules.db')

# 创建表
c = conn.cursor()
c.execute('''CREATE TABLE rules
             (id INTEGER PRIMARY KEY,
              condition TEXT,
              action TEXT)''')

# 插入规则
c.execute("INSERT INTO rules (condition, action) VALUES (?, ?)",
          ('age > 18', 'can_vote'))

# 提交事务
conn.commit()

# 查询规则
c.execute("SELECT * FROM rules")
print(c.fetchall())

# 关闭数据库
conn.close()
```

在这个示例中，我们首先创建了一个SQLite数据库，然后创建了一个`rules`表来存储规则。接着，我们插入了一个规则，规则的条件是`age > 18`，动作是`can_vote`。最后，我们查询了规则，并将查询结果打印出来。

## 4.2 规则检索示例

在这个示例中，我们将使用Python编程语言来实现规则检索。我们将使用正则表达式来检索规则。

```python
import re

# 定义规则
rules = [
    ('age > 18', 'can_vote'),
    ('age < 18', 'cannot_vote')
]

# 检索规则
pattern = re.compile(r'age > 18')
matching_rules = [rule for rule in rules if pattern.match(rule[0])]

print(matching_rules)
```

在这个示例中，我们首先定义了一个规则列表，其中包含了两个规则。接着，我们使用正则表达式`age > 18`来检索规则。最后，我们使用列表推导来筛选出匹配的规则，并将匹配的规则打印出来。
