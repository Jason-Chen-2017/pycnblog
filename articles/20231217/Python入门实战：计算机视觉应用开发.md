## 2.核心概念与联系

在本节中，我们将介绍计算机视觉的核心概念和与其他相关领域的联系。

### 2.1 计算机视觉的核心概念

1. **图像**：图像是人类视觉系统所接收的信息的基本单位。图像可以分为两类：一是数字图像，即由像素组成的二维矩阵；二是模拟图像，即光信号的变化。

2. **视频**：视频是连续的图像序列，用于表示动态场景。视频的主要特征包括帧率、分辨率和码率等。

3. **图像处理**：图像处理是对数字图像进行各种操作的过程，包括滤波、边缘检测、形状识别等。图像处理的主要目标是提高图像质量、提取图像特征和减少计算量。

4. **图像分类**：图像分类是将图像划分为多个类别的过程，通常用于对象识别和图像搜索。图像分类的主要方法包括传统机器学习方法和深度学习方法。

5. **目标检测**：目标检测是在图像中识别和定位特定对象的过程，通常用于人脸识别、自动驾驶等应用。目标检测的主要方法包括传统方法和深度学习方法。

6. **对象识别**：对象识别是识别图像中特定对象并为其赋予标签的过程，通常用于图像搜索和图像标注等应用。对象识别的主要方法包括传统方法和深度学习方法。

### 2.2 计算机视觉与其他相关领域的联系

1. **人工智能**：计算机视觉是人工智能领域的一个重要分支，涉及到计算机对于图像和视频的理解和处理。其他人工智能技术包括自然语言处理、知识图谱、机器学习等。

2. **机器学习**：机器学习是计算机程序在未经过明确编程的情况下从数据中学习的技术。计算机视觉技术广泛应用于机器学习领域，包括图像分类、目标检测和对象识别等。

3. **深度学习**：深度学习是一种通过神经网络模拟人类大脑中的学习和思维过程的技术。深度学习已经成为计算机视觉领域的主流方法，能够实现图像和视频的高级理解和处理。

4. **图像处理**：图像处理是对数字图像进行各种操作的过程，包括滤波、边缘检测、形状识别等。图像处理技术在计算机视觉中起到关键作用，帮助提高图像质量和提取图像特征。

5. **人机交互**：人机交互是计算机与人类进行交互的技术。计算机视觉技术在人机交互领域有广泛的应用，例如手势识别、语音识别等。

6. **机器人**：机器人是具有自主行动能力的计算机系统。计算机视觉技术在机器人领域有广泛的应用，例如机器人视觉、机器人导航等。
