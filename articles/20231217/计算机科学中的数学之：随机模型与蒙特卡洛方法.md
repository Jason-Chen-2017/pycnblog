```python
import random

def monte_carlo(n):
    count_0 = 0
    count_1 = 0
    for _ in range(n):
        if random.randint(0, 1) == 0:
            count_0 += 1
        else:
            count_1 += 1
    return count_0 / n, count_1 / n

n = 100000
expectation_0, expectation_1 = monte_carlo(n)
print(f"硬币投掷结果的估计量：头面 {expectation_0:.5f}，尾面 {expectation_1:.5f}")
```

在这个代码实例中，我们首先导入了`random`模块，用于生成随机数。然后定义了一个`monte_carlo`函数，该函数接受一个参数`n`，表示生成的随机样本数量。在函数内部，我们使用了一个`for`循环来生成随机样本，并统计了头面和尾面的数量。最后，我们将头面和尾面的数量除以样本数量得到估计量，并打印出结果。

通过运行这个代码实例，我们可以得到硬币投掷结果的估计量：头面和尾面的概率分布接近0.5。
