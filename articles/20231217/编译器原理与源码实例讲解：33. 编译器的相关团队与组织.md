## 4.1 词法分析示例

在本节中，我们将通过一个简单的词法分析示例来详细解释词法分析的原理和步骤。

### 4.1.1 示例代码

```python
import re

def tokenize(code):
    tokens = []
    pattern = r'\s+|(["\']\w+["\']|[A-Za-z_][\w]*|[0-9]+|[\+\-\*\/\%\(\)\{\}\[\]\=\<\>\,\;\.\:\'\"]|[\'\"]?[a-zA-Z_][\w]*\'\'?|[0-9]+(\.[0-9]+)?)'
    for match in re.finditer(pattern, code):
        token = match.group(1)
        if token.startswith('"') or token.endswith('"'):
            token = token[1:-1]
        if token.startswith("'") or token.endswith("'"):
            token = token[1:-1]
        if token.startswith('0') or token.endswith('0'):
            token = token[1:-1]
        if token.startswith('0x') or token.endswith('0x'):
            token = token[1:-1]
        if token.startswith('0b') or token.endswith('0b'):
            token = token[1:-1]
        tokens.append(token)
    return tokens

code = 'print("Hello, world!")'
tokens = tokenize(code)
print(tokens)
```

### 4.1.2 解释说明

在上述示例中，我们使用了正则表达式来匹配程序代码中的词法单元。具体来说，我们使用了以下几个正则表达式：

1. `\s+`：匹配一个或多个空格字符。
2. `["'\']\w+["'\']`：匹配一个双引号字符串。
3. `[A-Za-z_][\w]*`：匹配一个标识符。
4. `[0-9]+`：匹配一个或多个数字字符。
5. `[\+\-\*\/\%\(\)\{\}\[\]\=\<\>\,\;\.\:\'\"]`：匹配一些特殊字符。
6. `[\'\"]?[a-zA-Z_][\w]*\'\'?`：匹配一个单引号字符串。
7. `[0-9]+(\.[0-9]+)?`：匹配一个数字。

通过调用`re.finditer()`函数，我们可以找到程序代码中的每个词法单元，并将它们添加到`tokens`列表中。最后，我们打印出`tokens`列表，以便查看词法分析的结果。

```python
import ply.lex as lex
import ply.yacc as yacc

tokens = ('NUM', 'PLUS', 'MINUS', 'TIMES', 'DIVIDE', 'LPAREN', 'RPAREN', 'ID')

# 词法分析器
def t_NUM(t):
    r'\d+'
    t.value = int(t.value)
    return t

def t_PLUS(t):
    r'\+'
    return t

def t_MINUS(t):
    r'-'
    return t

def t_TIMES(t):
    r'\*'
    return t

def t_DIVIDE(t):
    r'/'
    return t

def t_LPAREN(t):
    r'\('
    return t

def t_RPAREN(t):
    r'\)'
    return t

def t_ID(t):
    r'\w+'
    return t

def t_error(t):
    print(f"Illegal character '{t.value[0]}'")

lexer = lex.lex()

# 语法分析器
def p_expression(p):
    """
    expression : expression PLUS term
               | expression MINUS term
               | term
    """
    if p:
        if p[2] == '+':
            p[0] = p[1] + p[3]
        elif p[2] == '-':
            p[0] = p[1] - p[3]
        else:
            p[0] = p[1]

def p_term(p):
    """
    term : term TIMES factor
    | term DIVIDE factor
    | factor
    """
    if p:
        if p[2] == '*':
            p[0] = p[1] * p[3]
        elif p[2] == '/':
            p[0] = p[1] / p[3]
        else:
            p[0] = p[1]

def p_factor(p):
    """
    factor : NUM
    | LPAREN expression RPAREN
    | ID
    """
    if p:
        p[0] = p[1]

def p_error(p):
    print(f"Syntax error at '{p.value}'")

parser = yacc.yacc()

code = '3 + 4 * 2'
lexer.input(code)
while True:
    tok = lexer.token()
    if not tok:
        break
    print(tok)

expression = '3 + 4 * 2'
parser.parse(expression)
```

### 4.2.2 解释说明

在上述示例中，我们使用了`ply`库来实现词法分析和语法分析。具体来说，我们使用了以下几个步骤：

1. 定义`tokens`列表，用于表示程序代码中的词法单元。
2. 定义`t_NUM`、`t_PLUS`、`t_MINUS`、`t_TIMES`、`t_DIVIDE`、`t_LPAREN`、`t_RPAREN`和`t_ID`函数，用于匹配程序代码中的词法单元。
3. 定义`p_expression`、`p_term`和`p_factor`函数，用于匹配程序代码中的语法单元。
4. 使用`lex.lex()`函数创建词法分析器，并使用`parser.parse()`函数创建语法分析器。

通过调用`lexer.input()`函数，我们可以将程序代码输入词法分析器，并逐个获取词法单元。通过调用`parser.parse()`函数，我们可以将程序代码输入语法分析器，并检查其是否符合预期的语法结构。
