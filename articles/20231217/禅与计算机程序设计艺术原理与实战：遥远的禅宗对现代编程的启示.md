我们来看一个简单的排序算法的实例，即冒泡排序。冒泡排序是一种简单的排序算法，它通过多次遍历数组中的元素，将相邻的元素进行比较和交换，从而实现排序。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

def binary_search(arr, target):

    while left <= right:
        mid = (left + right) // 2
