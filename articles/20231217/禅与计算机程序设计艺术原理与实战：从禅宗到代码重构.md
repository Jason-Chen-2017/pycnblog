# 4.1 快速排序算法实例
在这个例子中，我们将介绍一个基于分区的快速排序算法。这个算法的核心思想是将一个大的排序问题分解为多个较小的排序问题，直到每个问题只包含一个元素，然后将这些元素按照给定的规则排序。

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[0]
        less = [x for x in arr[1:] if x <= pivot]
        greater = [x for x in arr[1:] if x > pivot]
        return quick_sort(less) + [pivot] + quick_sort(greater)
```

这个算法的时间复杂度为O(nlogn)，其中n是输入数组的长度。这个算法的空间复杂度为O(logn)，因为它使用了递归来实现排序。

# 4.2 二分搜索算法实例
在这个例子中，我们将介绍一个基于递归的二分搜索算法。这个算法的核心思想是将一个大的搜索问题分解为多个较小的搜索问题，直到找到满足条件的元素或者搜索区间为空。

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

这个算法的时间复杂度为O(logn)，其中n是输入数组的长度。这个算法的空间复杂度为O(1)，因为它使用了递归来实现搜索，但没有使用额外的数据结构来存储搜索区间。
