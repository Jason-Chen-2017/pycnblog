# 4.具体代码实例和详细解释说明

## 4.1 代码实例

以下是一个简单的 Go 语言命令行工具示例：

```go
package main

import (
	"bufio"
	"flag"
	"fmt"
	"os"
	"strings"
)

func main() {
	flag.Parse()
	reader := bufio.NewReader(os.Stdin)
	fmt.Print("Enter text: ")
	text, _ := reader.ReadString('\n')
	text = strings.TrimSpace(text)
	fmt.Println("You entered:", text)
}
```

这个示例程序通过解析命令行参数、读取输入、执行功能和写入输出来实现简单的文本处理功能。

## 4.2 详细解释说明

1. 解析命令行参数：`flag.Parse()` 函数用于解析命令行参数。
2. 读取输入：`bufio.NewReader(os.Stdin)` 函数用于创建一个新的 `bufio.Reader`，用于读取标准输入（stdin）。`reader.ReadString('\n')` 函数用于读取一行文本。
3. 执行功能：`strings.TrimSpace(text)` 函数用于去除文本两端的空格。
4. 写入输出：`fmt.Println("You entered:", text)` 函数用于写入输出。
