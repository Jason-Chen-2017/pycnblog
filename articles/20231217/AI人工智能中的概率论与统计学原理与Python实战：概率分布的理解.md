## 3.1 概率论基础

概率论是一种数学方法，用于描述和分析不确定性。概率论可以帮助我们理解和预测事件发生的可能性，从而为我们的决策提供基础。在概率论中，我们通常使用以下几个概念：

1. 事件：一个可能发生的结果，称为事件。
2. 样本空间：所有可能的结果集合，称为样本空间。
3. 事件的概率：事件发生的可能性，通常用P(E)表示，P(E)∈[0,1]。

### 3.1.1 基本定理

基本定理是概率论中最重要的定理之一，它规定了如何计算两个事件发生的概率。具体来说，如果事件A和事件B是独立的，那么它们的发生概率就是：

P(A∩B) = P(A) × P(B)

### 3.1.2 条件概率

条件概率是概率论中一个事件发生的概率，给定另一个事件已经发生的情况下。我们使用P(A|B)表示事件A发生的概率，给定事件B已经发生。条件概率的计算公式为：

P(A|B) = P(A∩B) / P(B)

### 3.1.3 贝叶斯定理

贝叶斯定理是概率论中一个非常重要的定理，它可以帮助我们计算一个事件发生的概率，给定另一个事件已经发生的情况下。贝叶斯定理的计算公式为：

P(A|B) = P(B|A) × P(A) / P(B)

## 3.2 统计学基础

统计学是一种利用数据和数学方法对事件进行分析的学科。统计学可以帮助我们理解数据的结构和模式，从而为我们的预测和决策提供支持。在统计学中，我们通常使用以下几个概念：

1. 样本：从一个大型群体中随机抽取的一组观测值，称为样本。
2. 参数：参数是一个群体的某些特征，通常用一个或多个数字来表示。
3. 估计量：一个参数的估计量是一个基于样本的量，用于估计参数的真实值。

### 3.2.1 均值和方差

均值是一个变量的所有观测值的总和除以观测值的个数。方差是一个变量的观测值从均值分布出来的离散程度的度量。方差的计算公式为：

σ² = Σ(x - μ)² / N

其中，x是观测值，μ是均值，N是观测值的个数。

### 3.2.2 最大似然估计

最大似然估计是一种用于估计参数的方法，它基于样本数据的概率最大化。具体来说，如果我们有一个参数θ，我们可以使用以下公式来计算最大似然估计：

L(θ) = Σlog(f(x_i|θ))

其中，f(x_i|θ)是参数θ给定时观测值x_i的概率密度函数。

### 3.2.3 贝叶斯估计

贝叶斯估计是一种用于估计参数的方法，它基于贝叶斯定理和先验分布。具体来说，如果我们有一个参数θ和一个先验分布P(θ)，我们可以使用以下公式来计算贝叶斯估计：

P(θ|X) = P(X|θ) × P(θ) / P(X)

其中，P(X|θ)是参数θ给定时观测值X的概率，P(θ|X)是参数θ给定观测值X的概率，P(X)是观测值X的概率。

## 4.1 计算均值和方差

```python
import numpy as np

# 生成一组随机数
x = np.random.randn(1000)

# 计算均值
mu = np.mean(x)

# 计算方差
sigma_squared = np.mean((x - mu) ** 2)

print("均值:", mu)
print("方差:", sigma_squared)
```
