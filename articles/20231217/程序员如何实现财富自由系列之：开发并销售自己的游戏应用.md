在开始学习如何开发游戏应用之前，我们需要了解一些核心概念。这些概念包括：

- 游戏开发：游戏开发是指从设计、编程、音效、图形设计等方面进行的过程，以创建游戏应用。
- 游戏引擎：游戏引擎是一个软件框架，用于简化游戏开发过程，提供游戏开发所需的基本功能和服务。
- 游戏平台：游戏平台是指用于发布和销售游戏应用的渠道，如苹果应用商店、谷歌应用商店等。
- 游戏应用：游戏应用是指一种软件应用程序，通过互联网或其他网络连接方式向用户提供游戏服务。

# 设置游戏背景颜色

snake_speed = [10, 0]

        if event.type == pygame.QUIT:
            game_loop = False

    # 更新蛇的位置
    snake_pos[0] = [snake_pos[0][0] + snake_speed[0], snake_pos[0][1] + snake_speed[1]]
