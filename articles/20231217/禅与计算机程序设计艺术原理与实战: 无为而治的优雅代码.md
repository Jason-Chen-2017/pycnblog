## 4.1 求最大公约数的代码实例

```python
def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b)
```

这段代码实现了求最大公约数的算法。它使用了递归的方法来计算最大公约数。当 $$ b == 0 $$ 时，返回 $$ a $$ ，否则返回 $$ gcd(b, a \% b) $$ 。

## 4.2 求最小公倍数的代码实例

```python
def lcm(a, b):

```

这段代码实现了求最小公倍数的算法。它使用了最大公约数的算法来计算最小公倍数。最小公倍数的公式是 $$ lcm(a, b) = \frac{a \times b}{gcd(a, b)} $$ 。
