# 2.核心概念与联系

禅宗是一种宗教和哲学思想，起源于中国。它强调直接体验和直接认识，而不是依赖于理论和概念。禅宗强调的核心概念包括：

1. 直指人心：禅宗强调直接体验和直接认识，而不是依赖于理论和概念。
2. 无言传心：禅宗认为，真正的智慧和见解无法通过言语来传达，只能通过直接体验来获得。
3. 一切皆空：禅宗认为，一切皆空，没有任何固定的形式和属性。
