# 2.核心概念与联系

在Rust编程中，条件语句和循环结构是编写复杂逻辑和控制程序流程的基本组件。以下是Rust中条件语句和循环结构的核心概念和联系：

## 2.1 条件语句

条件语句是一种用于根据某个条件是否满足来执行代码块的控制结构。在Rust中，条件语句使用`if`关键字来定义。

### 2.1.1 if语句

`if`语句用于根据一个布尔表达式的结果来执行代码块。如果条件为`true`，则执行代码块；如果条件为`false`，则跳过代码块。

```rust
if condition {
    // 如果condition为true，执行此代码块
}
```

### 2.1.2 if...else语句

`if...else`语句用于根据一个布尔表达式的结果来执行不同的代码块。如果条件为`true`，则执行第一个代码块；如果条件为`false`，则执行第二个代码块。

```rust
if condition {
    // 如果condition为true，执行此代码块
} else {
    // 如果condition为false，执行此代码块
}
```

### 2.1.3 if...else if...else语句

`if...else if...else`语句用于根据多个条件之一为真来执行不同的代码块。如果第一个条件为`true`，则执行第一个代码块；如果第一个条件为`false`，则检查第二个条件，依此类推。如果所有条件都为`false`，则执行最后的`else`代码块。

```rust
if condition1 {
    // 如果condition1为true，执行此代码块
} else if condition2 {
    // 如果condition1为false，condition2为true，执行此代码块
} else {
    // 如果condition1和condition2都为false，执行此代码块
}
```

## 2.2 循环结构

循环结构是一种用于重复执行代码块的控制结构。在Rust中，循环结构使用`loop`关键字来定义。

### 2.2.1 loop关键字

`loop`关键字用于定义一个无限循环，直到遇到一个断出（`break`）或返回（`return`）语句。

```rust
loop {
    // 执行此代码块，直到遇到break或return语句
}
```

### 2.2.2 while循环

`while`循环用于根据一个条件的结果重复执行代码块，条件为`true`时才执行。

```rust
while condition {
    // 如果condition为true，执行此代码块
}
```

### 2.2.3 for循环

`for`循环用于遍历一个可迭代的数据结构，如数组、切片、哈希表等。`for`循环的语法如下：

```rust
for item in iterable {
    // 遍历iterable中的每个元素，执行此代码块
}
```

# 4.具体代码实例和详细解释说明

在本节中，我们将通过具体的代码实例来详细解释条件语句和循环结构的使用方法和最佳实践。

## 4.1 条件语句代码实例

### 4.1.1 if语句实例

```rust
fn main() {
    let x = 10;
    if x > 5 {
        println!("x 大于 5");
    }
}
```

在上述代码中，我们定义了一个整数变量`x`，并使用`if`语句来检查`x`是否大于5。如果`x`大于5，则输出`x 大于 5`。

### 4.1.2 if...else语句实例

```rust
fn main() {
    let x = 10;
    if x > 5 {
        println!("x 大于 5");
    } else {
        println!("x 不大于 5");
    }
}
```

在上述代码中，我们使用`if...else`语句来检查`x`是否大于5。如果`x`大于5，则输出`x 大于 5`；否则，输出`x 不大于 5`。

### 4.1.3 if...else if...else语句实例

```rust
fn main() {
    let x = 10;
    if x > 5 {
        println!("x 大于 5");
    } else if x == 5 {
        println!("x 等于 5");
    } else {
        println!("x 小于 5");
    }
}
```

在上述代码中，我们使用`if...else if...else`语句来检查`x`的值。如果`x`大于5，则输出`x 大于 5`；如果`x`等于5，则输出`x 等于 5`；否则，输出`x 小于 5`。

## 4.2 循环结构代码实例

### 4.2.1 loop关键字实例

```rust
fn main() {
    let mut count = 0;
    loop {
        count += 1;
        if count > 10 {
            break;
        }
        println!("循环次数：{}", count);
    }
}
```

在上述代码中，我们使用`loop`关键字定义了一个无限循环。我们定义了一个整数变量`count`，用于记录循环次数。在循环中，我们增加`count`的值，直到`count`大于10，然后使用`break`语句退出循环。

### 4.2.2 while循环实例

```rust
fn main() {
    let mut count = 0;
    while count < 10 {
        count += 1;
        println!("循环次数：{}", count);
    }
}
```

在上述代码中，我们使用`while`循环来遍历10次。我们定义了一个整数变量`count`，用于记录循环次数。在循环中，我们增加`count`的值，直到`count`大于等于10，然后退出循环。

### 4.2.3 for循环实例

```rust
fn main() {
    let numbers = [1, 2, 3, 4, 5];
    for number in numbers.iter() {
        println!("数组中的元素：{}", number);
    }
}
```

在上述代码中，我们使用`for`循环来遍历一个整数数组`numbers`。我们使用`iter`方法来创建数组的迭代器，然后使用`for`循环遍历迭代器中的每个元素。

# 6.附录常见问题与解答

在本节中，我们将回答一些常见问题及其解答。

## 6.1 if语句中的多个条件

在Rust中，如果需要在`if`语句中检查多个条件，可以使用`if`语句与`else if`语句的组合。例如：

```rust
fn main() {
    let x = 10;
    if x > 5 {
        println!("x 大于 5");
    } else if x == 5 {
        println!("x 等于 5");
    } else {
        println!("x 小于 5");
    }
}
```

在上述代码中，我们使用`if...else if...else`语句来检查`x`的值。如果`x`大于5，则输出`x 大于 5`；如果`x`等于5，则输出`x 等于 5`；否则，输出`x 小于 5`。

## 6.2 循环中的break和continue语句

在Rust中，`break`和`continue`语句用于控制循环的执行。`break`语句用于终止循环，`continue`语句用于跳过当前迭代并继续下一次迭代。

例如，在以下代码中，我们使用`break`语句来终止循环：

```rust
fn main() {
    let mut count = 0;
    loop {
        count += 1;
        if count > 10 {
            break;
        }
        println!("循环次数：{}", count);
    }
}
```

在以下代码中，我们使用`continue`语句来跳过当前迭代：

```rust
fn main() {
    let numbers = [1, 2, 3, 4, 5];
    for number in numbers.iter() {
        if *number % 2 == 0 {
            continue;
        }
        println!("数组中的奇数：{}", number);
    }
}
```

在上述代码中，我们使用`for`循环遍历一个整数数组`numbers`。如果数组中的元素是偶数，则使用`continue`语句跳过当前迭代，不输出结果。如果数组中的元素是奇数，则输出结果。
