## 4.1 规则定义的实例

以下是一个基于事实的规则定义实例：

```python
rule "贷款审批" {
    when {
        $amount : double;
        $loan : Loan(amount == $amount);
        $credit : Credit(loan.customer == $loanCustomer);
        $rate : double;
    }
    then {
        $loan.approved = true;
        $loan.interestRate = $rate;
    }
}
```

在这个实例中，我们定义了一个名为“贷款审批”的规则，它根据贷款金额、贷款客户、信用评级和贷款利率来审批贷款。

## 4.2 规则执行的实例

以下是一个规则执行实例：

```python
Loan loan = new Loan();
loan.amount = 10000;
loan.customer = "John Doe";

Credit credit = new Credit();
credit.loan = loan;
credit.level = "A";

double rate = 0.05;

// 触发规则
rule.activate(loan, credit, rate);

// 执行规则
rule.fire(loan, credit, rate);
```

在这个实例中，我们创建了一个贷款对象和一个信用评级对象，然后通过触发和执行规则来审批贷款。
