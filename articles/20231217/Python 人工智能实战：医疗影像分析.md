## 1.2 核心概念与联系

### 1.2.1 医疗影像数据

医疗影像数据是指由医学影像设备生成的图像数据，如CT、MRI、X光等。这些数据通常是高维的、大规模的、不规则的，并且具有时空关系。医疗影像数据的主要特点是：

1. 高维性：医疗影像数据通常包含多个时空维度，如时间、空间、剂量等。
2. 大规模性：医疗影像数据的规模通常非常大，如GB、TB甚至PB级别。
3. 不规则性：医疗影像数据通常是不规则的，如图像、音频、视频等。
4. 时空关系：医疗影像数据通常具有时空关系，如CT扫描的不同时间点、MRI的不同时间点等。

### 1.2.2 计算机视觉与深度学习

计算机视觉是计算机科学领域的一个重要分支，它涉及到计算机对图像和视频数据的处理、分析和理解。计算机视觉的主要任务是从图像和视频数据中提取有意义的特征，并根据这些特征进行分类、检测、识别等任务。

深度学习是计算机科学领域的一个重要技术，它涉及到使用多层神经网络进行数据的处理、分析和学习。深度学习的主要优点是它可以自动学习特征，不需要人工手动提取特征，这使得深度学习在计算机视觉等领域具有很大的潜力。
