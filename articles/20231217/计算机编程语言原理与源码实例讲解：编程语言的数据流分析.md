数据流是数据流分析的基本概念之一，它表示一种数据的顺序和结构。数据流可以是一种线性结构，如链表或数组；也可以是一种非线性结构，如树或图。数据流可以存储在内存中，也可以存储在外部存储设备中，如硬盘或云存储。

数据结构是计算机科学的一个基本概念，它描述了数据如何组织和存储在计算机内存中。数据结构包括各种数据类型，如数组、链表、树、图等。数据结构的选择和设计对程序的执行效率和空间复杂度有很大影响。

    return filtered_data

    age_sum = 0

        age_count += 1
    age_avg = age_sum / age_count
