# 4.具体代码实例和详细解释说明

以下是一个简单的Python图形用户界面编程示例，使用Tkinter库：

```python
import tkinter as tk

def on_button_click():
    print("按钮被点击了")

root = tk.Tk()
root.title("简单示例")
root.geometry("300x200")

button = tk.Button(root, text="点我")
button.pack()

button.bind("<Button-1>", on_button_click)

root.mainloop()
```

这个示例中，我们创建了一个简单的窗口，添加了一个按钮，并设置了按钮的点击事件处理函数。当用户点击按钮时，会调用`on_button_click`函数，打印出"按钮被点击了"。
