## 4.1 线性规划示例

线性规划是一种常用的决策模型，用于解决具有线性目标函数和线性约束条件的决策问题。以下是一个线性规划示例：

```python
from scipy.optimize import linprog

# 目标函数
c = [1, 2]

# 约束条件
A = [[-1, 1], [2, -1]]
b = [2, 4]

# 决策变量
x0 = linprog(c, A_ub=A, b_ub=b, bounds=[(0, None), (0, None)])

print(x0)
```

这个示例中，我们使用了`scipy.optimize`库中的`linprog`函数来解决线性规划问题。目标函数为`c = [1, 2]`，约束条件为`A = [[-1, 1], [2, -1]]`和`b = [2, 4]`，决策变量为`x0`。

## 4.2 动态规划示例

动态规划是一种常用的决策模型，用于解决具有递归结构的决策问题。以下是一个动态规划示例：

```python
def fib(n):
    if n <= 1:
        return n
    else:
        return fib(n-1) + fib(n-2)

print(fib(10))
```

这个示例中，我们使用了动态规划来解决斐波那契数列问题。通过使用动态规划，我们可以避免递归调用带来的性能问题，并提高解决问题的效率。
