# 2.核心概念与联系

## 2.1 禅宗的基本思想

禅宗是一种哲学思想，它源于中国，后传播到日本、韩国等国家。禅宗的基本思想有：

- 直接体验：禅宗认为，人们可以通过直接体验来认识真实的事物。这种体验不需要经过思考、言语的中介。
- 直接认识：禅宗认为，人们可以通过直接认识来认识真实的事物。这种认识不需要经过概念、分析的中介。
- 直接见解：禅宗认为，人们可以通过直接见解来认识真实的事物。这种见解不需要经过判断、分析的中介。

## 2.2 流畅接口的基本思想

流畅接口是一种计算机程序设计思想，它融合了禅宗的无为思想和计算机程序设计的实践。流畅接口的基本思想有：

- 无为：流畅接口的设计思想是“无为”，即不要去改变事物的本质，而是让事物自然地发挥其本身的优势。
- 流畅：流畅接口的设计思想是“流畅”，即让事物自然地流动、自然地发展。
- 接口：流畅接口的设计思想是“接口”，即让事物自然地相互联系、自然地相互作用。
