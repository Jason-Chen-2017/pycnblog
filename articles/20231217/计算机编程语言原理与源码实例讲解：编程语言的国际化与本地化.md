### 1.4.1 UTF-8编码转换示例

UTF-8是一种常见的字符编码转换算法，它可以将源代码中的字符编码转换为目标编码。以下是一个简单的UTF-8编码转换示例：

```python
def utf8_encode(source, target):
    result = ""
    for char in source:
        if ord(char) < 128:
            result += char
        elif ord(char) < 2048:
            result += chr(ord(char) >> 6) + "="
        else:
            result += chr(ord(char) >> 12) + chr(ord(char) >> 6) + "="
    return result

source = "中文"
target = "UTF-8"
result = utf8_encode(source, target)
print(result)
```

在这个示例中，我们定义了一个名为`utf8_encode`的函数，该函数接受一个源字符串`source`和一个目标字符串`target`作为参数。函数通过遍历源字符串中的每个字符，判断字符的Unicode编码范围，并将其转换为UTF-8编码格式。最后，函数返回转换后的字符串。
