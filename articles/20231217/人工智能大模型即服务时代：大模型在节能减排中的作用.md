# 4.具体代码实例和详细解释说明
## 4.1 代码实例
以下是一个简单的深度神经网络代码实例，用于预测能源消耗：

```python
import tensorflow as tf

# 定义神经网络结构
model = tf.keras.Sequential([
    tf.keras.layers.Dense(64, activation='relu', input_shape=(10,)),
    tf.keras.layers.Dense(64, activation='relu'),
    tf.keras.layers.Dense(1)
])

# 编译模型
model.compile(optimizer='adam', loss='mse', metrics=['mae'])

# 训练模型
model.fit(x_train, y_train, epochs=10, batch_size=32)

# 评估模型
loss, mae = model.evaluate(x_test, y_test)
```

## 4.2 详细解释说明
上述代码实例主要包括以下几个部分：

1. 导入库：导入tensorflow库，用于构建深度神经网络。
2. 定义神经网络结构：定义一个简单的深度神经网络，包括两个隐藏层，每个隐藏层有64个神经元，使用ReLU激活函数。输出层有一个神经元，无激活函数。输入层输入维度为10。
3. 编译模型：使用Adam优化器，均方误差（MSE）作为损失函数，均方根（MAE）作为评估指标。
4. 训练模型：使用训练数据（x_train、y_train）训练模型，总共10个epoch，每个epoch的批量大小为32。
5. 评估模型：使用测试数据（x_test、y_test）评估模型性能，得到损失值（loss）和均方根（mae）。
