## 4.1 开放API的具体代码实例

开放API的具体代码实例可以使用Python编程语言来实现。以下是一个简单的开放API的具体代码实例：

```python
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/api/v1/users', methods=['GET'])
def get_users():
    users = [
        {'id': 1, 'name': 'John Doe', 'email': 'john@example.com'},
        {'id': 2, 'name': 'Jane Doe', 'email': 'jane@example.com'},
    ]
    return jsonify(users)

if __name__ == '__main__':
    app.run(debug=True)
```

在这个代码实例中，我们使用了Flask框架来实现一个简单的开放API。这个API提供了一个`/api/v1/users`端点，用户可以通过发送GET请求来获取用户列表。
