## 4.1 Python装饰器的具体代码实例

以下是一个简单的Python装饰器的具体代码实例：

```python
def log_decorator(func):
    def wrapper(*args, **kwargs):
        print(f"Calling {func.__name__}")
        result = func(*args, **kwargs)
        print(f"Called {func.__name__} with result {result}")
        return result
    return wrapper

@log_decorator
def add(a, b):
    return a + b

result = add(1, 2)
print(result)
```

在这个例子中，我们定义了一个名为`log_decorator`的装饰器函数，该函数接受一个函数作为参数，并在调用该函数之前和之后打印一些日志信息。然后，我们使用`@log_decorator`语法将`add`函数装饰为`log_decorator`装饰器的实例。当我们调用`add`函数时，会打印出日志信息。

## 4.2 Python生成器的具体代码实例

以下是一个简单的Python生成器的具体代码实例：

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

for value in fibonacci(10):
    print(value)
```

在这个例子中，我们定义了一个名为`fibonacci`的生成器函数，该函数接受一个整数`n`作为参数，并生成前`n`个斐波那契数字。然后，我们使用`for`循环遍历`fibonacci`生成器，并逐个打印出生成的数字。
