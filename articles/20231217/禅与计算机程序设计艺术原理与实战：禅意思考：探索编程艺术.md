## 4.1 最长公共子序列问题
最长公共子序列问题是计算机程序设计中非常常见的问题。给定两个字符串，我们需要找到它们的最长公共子序列。一个子序列是一个字符串中的一个子集，它可以通过删除某些字符得到。例如，字符串“abc”和“ahbgdc”的最长公共子序列是“ahbgd”。

## 4.2 动态规划算法
我们可以使用动态规划算法来解决最长公共子序列问题。动态规划算法的基本思想是将问题分解为子问题，然后将子问题的解组合成问题的解。

我们可以定义一个二维数组dp，其中dp[i][j]表示字符串s1的前i个字符和字符串s2的前j个字符的最长公共子序列的长度。我们可以使用以下公式来计算dp[i][j]：

$$
dp[i][j] =
\begin{cases}
1 & \text{if } s1[i-1] == s2[j-1] \\
0 & \text{otherwise}
\end{cases}
$$

我们可以使用以下代码来实现动态规划算法：

```python
def longest_common_substring(s1, s2):
    m = len(s1)
    n = len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    max_len = 0
    end_index = 0
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                if dp[i][j] > max_len:
                    max_len = dp[i][j]
                    end_index = i
            else:
                dp[i][j] = 0
    return s1[end_index - max_len: end_index]
```

这个代码实现了动态规划算法，并解决了最长公共子序列问题。我们可以使用禅意思考来理解这个算法，并找到更好的解决方案。
