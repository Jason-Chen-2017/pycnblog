1. 排序算法实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

上述代码实现了冒泡排序算法，时间复杂度为O(n^2)。

1. 搜索算法实例：

```python
def binary_search(arr, target):
    left, right = 0, len(arr)-1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

上述代码实现了二分搜索算法，时间复杂度为O(logn)。

1. 图算法实例：

```python
import networkx as nx

def dijkstra(graph, start):
    dist = {node: float('inf') for node in graph}
    dist[start] = 0
    visited = set()
    while len(visited) < len(graph):
        u = min(dist, key=dist.get)
        visited.add(u)
        for v, weight in graph[u].items():
            dist[v] = min(dist[v], dist[u] + weight)
    return dist
```

上述代码实现了Dijkstra算法，用于计算图中节点之间的最短路径，时间复杂度为O(|V|^2)。
