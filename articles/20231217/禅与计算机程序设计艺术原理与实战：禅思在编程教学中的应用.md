## 代码实例

我们将通过一个简单的排序算法来进行具体的代码实例和解释。以下是一个简单的冒泡排序算法的实现：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

## 代码解释

上述代码实现了一个简单的冒泡排序算法。冒泡排序算法的基本思想是通过多次遍历数组，每次遍历时将最大的元素移动到数组的末尾。遍历的过程中，如果发现当前元素大于下一个元素，则交换它们的位置。这个过程重复多次，直到数组排序完成。
