## 4.1 网络渗透测试代码实例

在本节中，我们将通过一个简单的网络渗透测试代码实例来详细解释网络渗透测试算法原理和操作步骤。

### 4.1.1 代码实例

```python
import requests
from requests.exceptions import RequestException

def scan(ip, port):
    try:
        url = f"http://{ip}:{port}/"
        response = requests.get(url, timeout=5)
        if response.status_code == 200:
            print(f"端口{port}开放")
        else:
            print(f"端口{port}关闭")
    except RequestException as e:
        print(f"请求失败: {e}")

if __name__ == "__main__":
    ip = "192.168.1.1"
    for port in range(1, 100):
        scan(ip, port)
```

### 4.1.2 代码解释

- 首先，我们导入了`requests`库，并定义了一个`scan`函数。`scan`函数接受一个IP地址和一个端口号作为参数，并通过发送HTTP请求来检查该端口是否开放。
- 在`scan`函数中，我们尝试通过`requests.get`发送一个HTTP请求到指定的IP地址和端口。如果请求成功，则说明该端口开放；如果请求失败，则说明该端口关闭。
- 最后，我们在主函数中定义了一个IP地址和一个范围，并通过调用`scan`函数对每个端口进行扫描。

```python
import os
import re

def check_file_permission(file_path):
    try:
        os.stat(file_path)
        with open(file_path, "r") as f:
            content = f.read()
            if re.search(r"root", content):
                print(f"文件{file_path}存在root权限")
            else:
                print(f"文件{file_path}没有root权限")
    except FileNotFoundError:
        print(f"文件{file_path}不存在")

if __name__ == "__main__":
    file_path = "/etc/passwd"
    check_file_permission(file_path)
```

### 4.2.2 代码解释

- 首先，我们导入了`os`和`re`库，并定义了一个`check_file_permission`函数。`check_file_permission`函数接受一个文件路径作为参数，并检查该文件是否存在并检查其权限。
- 在`check_file_permission`函数中，我们尝试通过`os.stat`获取文件的状态信息。如果文件存在，则通过打开文件并读取其内容来检查文件是否包含`root`权限。如果文件不存在，则说明文件不存在。
- 最后，我们在主函数中定义了一个文件路径并调用`check_file_permission`函数进行检查。
