from flask import Flask, request

app = Flask(__name__)

@app.route('/api/greet')

import os

def get_file(file_path):
    if os.path.exists(file_path):
        with open(file_path, 'r') as f:
            return f.read()
    else:
        return None

def save_file(file_path, content):
    with open(file_path, 'w') as f:
        f.write(content)
```

这个资源实例定义了两个函数 `get_file` 和 `save_file`，用于读取和保存文件。`get_file` 函数会检查文件是否存在，如果存在则读取文件内容，否则返回 None。`save_file` 函数会将文件内容写入文件。

@app.route('/api/greet')
def greet():
    name = request.args.get('name')
    if name:
        return f'Hello, {name}!'
    else:
        return 'Hello, stranger!'

@app.route('/api/greet_app')
def greet_app():
    name = request.args.get('name')
    if name:
        return f'Hello, {name} from greet_app!'
    else:
        return 'Hello, stranger from greet_app!'

if __name__ == '__main__':
    app.run()
```

这个协同实例定义了两个路由 `/api/greet` 和 `/api/greet_app`，当访问 `/api/greet` 时，会调用 `greet` 函数，当访问 `/api/greet_app` 时，会调用 `greet_app` 函数。这两个函数的实现相同，只是返回的字符串略有不同。

## 4.5 服务治理的实现

我们可以使用 Python 编程语言来实现服务治理的管理、监控和优化。以下是一个简单的服务治理实例：

```python
import time
from flask import Flask, request

app = Flask(__name__)

@app.route('/api/greet')
def greet():
    start_time = time.time()
    name = request.args.get('name')
    if name:
        response = f'Hello, {name}!'
    else:
        response = 'Hello, stranger!'
    end_time = time.time()
    print(f'Response time: {end_time - start_time} seconds')
    return response

if __name__ == '__main__':
    app.run()
```

这个服务治理实例定义了一个路由 `/api/greet`，当访问这个路由时，会记录开始时间和结束时间，计算响应时间，并将响应时间打印到控制台。
