## 4.1 推荐系统

推荐系统是一种根据用户的历史行为和喜好为其提供个性化推荐的技术。以下是一个基于协同过滤算法的推荐系统的具体代码实例：

```python
import numpy as np
from scipy.spatial.distance import cosine

def user_item_similarity(user_item_matrix):
    similarity_matrix = np.zeros((user_item_matrix.shape[0], user_item_matrix.shape[1]))
    for u in range(user_item_matrix.shape[0]):
        for i in range(user_item_matrix.shape[1]):
            if user_item_matrix[u, i] > 0:
                for v in range(user_item_matrix.shape[1]):
                    if user_item_matrix[u, v] > 0 and v != i:
                        similarity_matrix[u, i] += user_item_matrix[u, v] / np.sqrt(np.dot(user_item_matrix[u, :], user_item_matrix[u, :]))
                        similarity_matrix[u, v] += user_item_matrix[u, i] / np.sqrt(np.dot(user_item_matrix[u, :], user_item_matrix[u, :]))
    return similarity_matrix

def recommend(user_item_matrix, similarity_matrix, target_user, target_item, n_recommendations):
    user_item_similarity = similarity_matrix[target_user, :]
    similarity_scores = np.delete(user_item_similarity, np.where(user_item_similarity == 0))
    sorted_indices = np.argsort(-similarity_scores)
    recommendations = sorted_indices[:n_recommendations]
    return recommendations
```

在这个代码中，我们首先计算用户-项目之间的相似性矩阵，然后根据相似性得出推荐列表。

## 4.2 内容生成

```python
import openai

openai.api_key = "your-api-key"

def generate_text(prompt, temperature=0.7, max_length=100):
    response = openai.Completion.create(
        engine="text-davinci-002",
        prompt=prompt,
        temperature=temperature,
        max_tokens=max_length,
        top_p=1,
        frequency_penalty=0,
        presence_penalty=0
    )
    text = response.choices[0].text.strip()
    return text

prompt = "Once upon a time, there was a young prince who wanted to marry a beautiful princess."
generated_text = generate_text(prompt)
print(generated_text)
```

在这个代码中，我们使用GPT-2模型生成一段文本，其主题是一个小公主。
