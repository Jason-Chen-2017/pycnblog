# 4.具体代码实例和详细解释说明

## 4.1 代码实例

我们以一个简单的排序算法——冒泡排序为例，来展示代码实例。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

## 4.2 详细解释说明

冒泡排序是一种简单的排序算法，它的主要思想是通过多次对数组中的元素进行比较和交换，使得较小的元素逐渐向前移动，最终形成有序的数组。

在上述代码中，我们首先定义了一个名为`bubble_sort`的函数，并将待排序的数组作为函数的参数。在函数内部，我们使用两个for循环来实现冒泡排序的过程。

第一个for循环是外层循环，它用于遍历数组中的每个元素。第二个for循环是内层循环，它用于比较数组中相邻的元素。如果当前元素大于下一个元素，则交换它们的位置。这个过程会一直持续到数组中的所有元素都有序为止。
