# 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的排序算法的代码实例，并进行详细解释：

```
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

这个代码是一个简单的冒泡排序算法，它的原理是通过多次遍历数组，将较大的元素向后移动，以达到排序的目的。具体来说，它的工作流程如下：

1. 从第一个元素开始，与后面的每个元素进行比较。
2. 如果当前元素大于后面的元素，则交换它们的位置。
3. 重复上述过程，直到整个数组被排序。

这个算法的时间复杂度是O(n^2)，其中n是数组的长度。因此，当数组的大小增加时，算法的运行时间会增加相应的倍数。
