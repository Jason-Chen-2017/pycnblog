以下是一个使用ChatGPT API接口生成对话的示例：

```python
import openai

# 获取API密钥
openai.api_key = 'your-api-key'

# 调用API接口
response = openai.Completion.create(
  engine="text-davinci-002",
  prompt="你好，我是ChatGPT，很高兴见到你。",
  max_tokens=50
)

# 打印返回结果
print(response.choices[0].text)
```
