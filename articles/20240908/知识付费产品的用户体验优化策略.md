**代码实例**

以下是一个简单的Go语言示例，展示如何使用互斥锁（Mutex）来保护共享变量：

```go
package main

import (
    "fmt"
    "sync"
)

var (
    counter int
    mu      sync.Mutex
)

func increment() {
    mu.Lock()
    counter++
    mu.Unlock()
}

func main() {
    var wg sync.WaitGroup
    for i := 0; i < 1000; i++ {
        wg.Add(1)
        go func() {
            defer wg.Done()
            increment()
        }()
    }
    wg.Wait()
    fmt.Println("Counter:", counter)
}
```

在这个示例中，`increment` 函数使用互斥锁来保护对共享变量 `counter` 的访问，确保在多线程环境中不会出现数据竞争问题。
