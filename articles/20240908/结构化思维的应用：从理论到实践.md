#### 二、算法编程题与解析

##### 1. 最长公共子序列

**题目：** 给定两个字符串，找出它们的最长公共子序列。

**答案：** 
- **动态规划：** 使用二维数组记录子问题的解，然后根据状态转移方程求解。
- **代码实例：**

```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[m][n]
```

##### 2. 二分查找

**题目：** 在一个有序数组中，查找一个特定的元素。

**答案：** 
- **二分查找：** 通过不断将查找范围缩小一半，逐步逼近目标元素。
- **代码实例：**

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1
```

##### 3. 合并两个有序链表

**题目：** 合并两个有序链表。

**答案：**
- **递归：** 使用递归方法将两个链表逐层合并。
- **迭代：** 使用迭代方法，通过指针遍历链表，将元素逐一合并。

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_sorted_lists(l1, l2):
    if not l1:
        return l2
    if not l2:
        return l1

    if l1.val < l2.val:
        l1.next = merge_sorted_lists(l1.next, l2)
        return l1
    else:
        l2.next = merge_sorted_lists(l1, l2.next)
        return l2
```
