#### 2. 算法编程题

**题目：** 实现一个函数，计算字符串中重复出现的最长子串长度。

**答案：** 

```python
def longest_substring(s):
    n = len(s)
    max_len = 0
    start = 0
    visited = [False] * 256

    for i in range(n):
        if visited[ord(s[i])]:
            while s[start] != s[i]:
                visited[ord(s[start])] = False
                start += 1
        else:
            max_len = max(max_len, i - start + 1)
            visited[ord(s[i])] = True

    return max_len

# 测试
print(longest_substring("abcabcbb"))  # 输出 3
print(longest_substring("bbbbb"))  # 输出 4
print(longest_substring("pwwkew"))  # 输出 3
```

**解析：** 这个问题考察了字符串处理和哈希表的应用。通过哈希表记录字符的出现位置，可以快速判断子串的重复情况。

#### 4. 算法编程题

**题目：** 实现一个函数，找出数组中的第 K 个最大元素。

**答案：**

```python
def find_kth_largest(nums, k):
    nums.sort(reverse=True)
    return nums[k - 1]

# 测试
print(find_kth_largest([3, 2, 1, 5, 6, 4], 2))  # 输出 5
print(find_kth_largest([3, 2, 3, 1, 2, 4, 5, 5, 6], 4))  # 输出 4
```

**解析：** 这个问题考察了数组排序和查找算法。通过将数组排序，可以快速找到第 K 个最大元素。
