##### 题目1：计算知识付费服务的收益

**题目描述：** 给定一个知识付费服务平台的订单数据，计算平台的总收益。每笔订单包含商品价格和订单数量。

**输入：**
```
商品价格：100元
订单数量：3
```

**输出：**
```
总收益：300元
```

**答案：**

```python
def calculate_revenue(price, quantity):
    return price * quantity

price = 100
quantity = 3
revenue = calculate_revenue(price, quantity)
print("总收益：", revenue)
```

**解析：** 该题目考察基本的数学运算，通过乘法计算总收益。源代码中定义了一个函数 `calculate_revenue`，接收商品价格和订单数量，返回总收益。

##### 题目2：会员折扣计算

**题目描述：** 给定一个会员折扣率，计算会员购买商品后的价格。

**输入：**
```
原价：200元
折扣率：0.8
```

**输出：**
```
会员价格：160元
```

**答案：**

```python
def calculate_discounted_price(price, discount_rate):
    return price * discount_rate

price = 200
discount_rate = 0.8
discounted_price = calculate_discounted_price(price, discount_rate)
print("会员价格：", discounted_price)
```

**解析：** 该题目考察乘法运算，通过折扣率计算折扣后的价格。源代码中定义了一个函数 `calculate_discounted_price`，接收原价和折扣率，返回会员价格。

##### 题目3：用户评价系统

**题目描述：** 设计一个用户评价系统，用户可以对知识付费服务进行评价，系统统计平均分和好评率。

**输入：**
```
评价列表：[4, 5, 3, 4]
```

**输出：**
```
平均分：4.25
好评率：80%
```

**答案：**

```python
def calculate_average_rating(ratings):
    total_rating = sum(ratings)
    average_rating = total_rating / len(ratings)
    return average_rating

def calculate_positive_rate(ratings):
    positive_ratings = len([rating for rating in ratings if rating >= 4])
    positive_rate = (positive_ratings / len(ratings)) * 100
    return positive_rate

ratings = [4, 5, 3, 4]
average_rating = calculate_average_rating(ratings)
positive_rate = calculate_positive_rate(ratings)
print("平均分：", average_rating)
print("好评率：", positive_rate)
```

**解析：** 该题目考察列表处理和统计计算。源代码中定义了两个函数，`calculate_average_rating` 计算平均分，`calculate_positive_rate` 计算好评率。通过列表解析和统计计算，可以得到评价系统的平均分和好评率。
