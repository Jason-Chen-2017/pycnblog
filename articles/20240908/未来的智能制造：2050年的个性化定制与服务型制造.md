源代码实例：

```python
# Python 实现基于协同过滤的个性化推荐
import numpy as np

# 用户-物品评分矩阵
ratings = np.array([[5, 3, 0, 1], [2, 0, 3, 4], [0, 1, 3, 2], [3, 2, 0, 5]])

# 计算用户和物品的相似度
user_similarity = 1 - np.corrcoef(ratings.T).diagonal()

# 构建推荐列表
def recommend(user_id):
    user_ratings = ratings[user_id]
    similar_users = user_similarity[user_id]
    recommendations = []

    for other_user, similarity in enumerate(similar_users):
        if similarity > 0:
            other_ratings = ratings[other_user]
            common_ratings = (user_ratings * other_ratings).sum()
            recommendations.append((common_ratings / similarity, other_user))

    recommendations.sort(reverse=True)
    return [user for _, user in recommendations[:5]]

# 示例
user_id = 0
print(recommend(user_id))
```

**解析：** 该示例使用基于协同过滤的个性化推荐算法，计算用户和物品的相似度，并根据相似度推荐相关的物品。
