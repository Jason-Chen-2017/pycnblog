##### 2. 回文数

**题目：** 判断一个整数是否是回文数。

**答案：**

```python
def is_palindrome(x):
    if x < 0:
        return False
    reversed_x = 0
    temp = x
    while temp > 0:
        reversed_x = reversed_x * 10 + temp % 10
        temp //= 10
    return x == reversed_x
```

**解析：** 通过反转整数并比较原整数和反转后的整数来判断是否为回文数。

##### 3. 二维数组中的查找

**题目：** 在一个二维数组中，查找一个给定的元素是否存在于数组中。

**答案：**

```python
def searchMatrix(matrix, target):
    if not matrix:
        return False
    row, col = len(matrix), len(matrix[0])
    lo, hi = 0, row * col
    while lo < hi:
        mid = (lo + hi) // 2
        if matrix[mid // col][mid % col] < target:
            lo = mid + 1
        elif matrix[mid // col][mid % col] > target:
            hi = mid - 1
        else:
            return True
    return False
```

**解析：** 使用二分查找法在二维数组中查找目标元素。
