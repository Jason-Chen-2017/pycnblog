**1. 编写一个算法，计算两个字符串的相似度。**

**题目：** 编写一个函数，计算两个字符串的相似度，相似度越高，得分越高。

**答案：**

```python
def string_similarity(str1, str2):
    m = len(str1)
    n = len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1])

    return dp[m][n]

similarity_score = string_similarity("hello", "halloween")
print("Similarity score:", similarity_score)
```

**解析：** 这个算法使用了动态规划的方法，计算两个字符串的编辑距离，编辑距离越小，相似度越高。在这个例子中，`hello` 和 `halloween` 的相似度得分是 7。

**2. 编写一个算法，找出字符串中的最长公共子序列。**

**题目：** 编写一个函数，找出两个字符串的最长公共子序列。

```python
def longest_common_subsequence(str1, str2):
    m = len(str1)
    n = len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    result = []
    i, j = m, n
    while i > 0 and j > 0:
        if str1[i - 1] == str2[j - 1]:
            result.append(str1[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            i -= 1
        else:
            j -= 1

    return ''.join(result[::-1])

lcs = longest_common_subsequence("hello", "world")
print("Longest common subsequence:", lcs)
```

**解析：** 这个算法同样使用了动态规划的方法，计算两个字符串的最长公共子序列。在这个例子中，`hello` 和 `world` 的最长公共子序列是 `lo`。
