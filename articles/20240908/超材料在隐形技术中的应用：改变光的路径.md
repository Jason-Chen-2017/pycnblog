**题目描述：** 编写一个程序，模拟光线在超材料中的传播过程。要求输入超材料的折射率分布，输出光线传播路径和反射、折射情况。

**答案：**

```python
import numpy as np
import matplotlib.pyplot as plt

def simulate_light(n, x, y):
    # n: 折射率分布
    # x, y: 光线初始位置和方向

    # 初始化光线轨迹
    trajectory = np.array([x, y])

    # 模拟光线传播
    while True:
        # 计算光线与超材料的交点
        t = np.abs(n * (x - trajectory[0]))

        # 判断光线是否与超材料相交
        if t < 1e-6:
            break

        # 计算光线传播后的位置
        trajectory = trajectory + n * (x - trajectory[0])

    # 绘制光线轨迹
    plt.plot(trajectory[0], trajectory[1])
    plt.xlabel('X')
    plt.ylabel('Y')
    plt.show()

# 测试
simulate_light([1, 1], 0, 0)
```

1. **输入参数：** 折射率分布 `n` 和光线初始位置及方向 `(x, y)`。
2. **计算过程：** 根据折射率分布，计算光线与超材料的交点。如果光线与超材料相交，则计算光线传播后的位置，并绘制光线轨迹。
