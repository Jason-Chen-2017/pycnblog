**8. 设计一个算法，找出数组中重复的元素。**

**Python 示例：**

```python
def find_duplicates(nums):
    seen = set()
    duplicates = []
    for num in nums:
        if num in seen:
            duplicates.append(num)
        seen.add(num)
    return duplicates

nums = [1, 2, 3, 4, 5, 2]
print(find_duplicates(nums))  # 输出 [2]
```
