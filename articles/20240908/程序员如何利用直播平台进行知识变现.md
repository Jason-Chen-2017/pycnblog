```python
import pandas as pd

# 假设有一个用户观看日志数据，格式为：用户ID、直播ID、观看时长
user_log = pd.DataFrame({
    'user_id': [1, 1, 2, 2, 3, 3],
    'live_id': [101, 102, 101, 102, 201, 202],
    'watch_time': [10, 15, 5, 10, 20, 30]
})

# 计算用户总观看时长、平均观看时长和观看频次
user_stats = user_log.groupby('user_id').agg({'watch_time': ['sum', 'mean', 'count']})
user_stats.columns = ['_'.join(col).strip() for col in user_stats.columns.values]

# 数据可视化
import matplotlib.pyplot as plt

user_stats[['watch_time_sum', 'watch_time_mean', 'watch_time_count']].T.plot(kind='bar', figsize=(12, 6))
plt.title('User Behavior Analysis')
plt.xlabel('User ID')
plt.ylabel('Watch Time')
plt.show()
```

通过以上代码，可以实现对用户观看日志的数据分析和可视化，从而帮助平台优化内容推荐和互动体验。
