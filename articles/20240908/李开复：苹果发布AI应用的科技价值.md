```go
func isPalindrome(s string) bool {
    runes := []rune(s)
    for i, j := 0, len(runes)-1; i < j; i, j = i+1, j-1 {
        if runes[i] != runes[j] {
            return false
        }
    }
    return true
}
```

**解析：** 该函数通过遍历字符串的字符，从两端开始比较，如果所有字符都相同，则字符串是回文。
