## 算法编程题与解答

### 1. 最短路径算法

**题目：** 实现一个最短路径算法，用于计算农田之间最短的运输路径。

**答案：** 使用Dijkstra算法。

```python
import heapq

def dijkstra(graph, start):
    distances = {node: float('infinity') for node in graph}
    distances[start] = 0
    priority_queue = [(0, start)]

    while priority_queue:
        current_distance, current_node = heapq.heappop(priority_queue)

        if current_distance > distances[current_node]:
            continue

        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(priority_queue, (distance, neighbor))

    return distances
```

**解析：** Dijkstra算法通过优先队列来选择下一个最短路径，逐步构建最短路径树。

### 2. 供需匹配算法

**题目：** 设计一个算法，用于匹配农田产品与消费者的需求。

**答案：** 使用线性规划或供需匹配算法。

```python
import cvxpy as cp

def match_supply_demand(supply, demand):
    x = cp.Variable(len(supply))
    objective = cp.Minimize(cp.sum(cp.abs(x - demand)))
    constraints = [supply[i] * (1 - x[i]) <= 0 for i in range(len(supply))]
    prob = cp.Problem(objective, constraints)
    prob.solve()

    return [x[i].value for i in range(len(x))]
```

**解析：** 算法通过最小化供需差值，找到最佳的供应量分配。
