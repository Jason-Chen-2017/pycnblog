#### 编程题

**1. 实现一个知识付费平台的购物车功能，包括添加商品、删除商品、计算总价等功能。**

**答案：**

```go
package main

import (
    "fmt"
)

type Product struct {
    Name  string
    Price float64
}

type ShoppingCart struct {
    Products []Product
}

func (s *ShoppingCart) AddProduct(product Product) {
    s.Products = append(s.Products, product)
}

func (s *ShoppingCart) RemoveProduct(name string) {
    for i, product := range s.Products {
        if product.Name == name {
            s.Products = append(s.Products[:i], s.Products[i+1:]...)
            break
        }
    }
}

func (s *ShoppingCart) TotalPrice() float64 {
    total := 0.0
    for _, product := range s.Products {
        total += product.Price
    }
    return total
}

func main() {
    cart := ShoppingCart{}
    cart.AddProduct(Product{"编程之美", 88.00})
    cart.AddProduct(Product{"经济学原理", 98.00})
    fmt.Println("购物车总价：", cart.TotalPrice())
    cart.RemoveProduct("编程之美")
    fmt.Println("购物车总价：", cart.TotalPrice())
}
```

**2. 实现一个知识付费平台的用户管理系统，包括用户注册、登录、查询个人信息等功能。**

**答案：**

```go
package main

import (
    "fmt"
    "os"
)

type User struct {
    Username string
    Password string
}

var users = make(map[string]User)

func Register(username, password string) {
    users[username] = User{Username: username, Password: password}
    fmt.Println("注册成功！")
}

func Login(username, password string) {
    if user, ok := users[username]; ok && user.Password == password {
        fmt.Println("登录成功！")
    } else {
        fmt.Println("登录失败！")
    }
}

func QueryUserInfo(username string) {
    if user, ok := users[username]; ok {
        fmt.Printf("用户名：%s，密码：%s\n", user.Username, user.Password)
    } else {
        fmt.Println("用户不存在！")
    }
}

func main() {
    Register("zhangsan", "123456")
    Register("lisi", "123456")
    Login("zhangsan", "123456")
    Login("wangwu", "123456")
    QueryUserInfo("zhangsan")
    QueryUserInfo("wangwu")
}
```

**购物车功能：**

* 使用 `Product` 结构体表示商品信息。
* 使用 `ShoppingCart` 结构体表示购物车，包含一个 `Products` 字段，存储购物车中的商品列表。
* 实现 `AddProduct` 方法，向购物车中添加商品。
* 实现 `RemoveProduct` 方法，从购物车中删除指定名称的商品。
* 实现 `TotalPrice` 方法，计算购物车中所有商品的总价。

**用户管理系统：**

* 使用 `User` 结构体表示用户信息。
* 使用 `users` 字典存储用户信息，键为用户名，值为用户对象。
* 实现 `Register` 方法，完成用户注册功能。
* 实现 `Login` 方法，完成用户登录功能。
* 实现 `QueryUserInfo` 方法，查询指定用户的信息。

通过以上功能实现，可以构建一个简单的知识付费平台用户管理系统和购物车功能。在实际应用中，可以根据需求进一步扩展和优化。
