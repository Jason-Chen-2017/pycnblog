```python
import pandas as pd
from sklearn.ensemble import RandomForestRegressor

# 加载数据
data = pd.read_csv('knowledge_paid_data.csv')

# 数据清洗
data = data.drop_duplicates().dropna()

# 特征提取
X = data[['user_activity', 'content_quality', 'market_share']]
y = data['market_potential']

model = RandomForestRegressor(n_estimators=100)
model.fit(X, y)

```

**解析：** 通过使用机器学习算法，我们可以对知识付费产品的市场潜力进行预测。特征提取和模型训练是关键步骤，有助于提高预测准确性。
