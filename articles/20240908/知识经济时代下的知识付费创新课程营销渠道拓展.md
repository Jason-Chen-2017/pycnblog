##### 1. 推荐算法

**题目：** 设计一个简单的基于协同过滤的推荐算法。

**答案：**

```python
import numpy as np

def collaborative_filtering(ratings, k=5):
    # ratings 是用户-物品评分矩阵，k 是邻居数量
    user_item_similarity = np.dot(ratings, ratings.T) / (np.linalg.norm(ratings, axis=0) * np.linalg.norm(ratings, axis=1))
    top_k_indices = np.argsort(user_item_similarity)[::-1][:k]
    neighbors_ratings = ratings[top_k_indices]
    predicted_ratings = np.dot(neighbors_ratings, user_item_similarity[neighbors_ratings.nonzero()].reshape(-1, 1)) / np.sum(user_item_similarity[neighbors_ratings.nonzero()])
    return predicted_ratings

# 示例数据
ratings = np.array([[5, 4, 0, 0],
                    [4, 5, 0, 2],
                    [2, 0, 5, 0],
                    [0, 0, 1, 5]])

predicted_ratings = collaborative_filtering(ratings)
print(predicted_ratings)
```

**解析：** 该算法基于用户之间的相似度计算，为用户提供相似用户的评分预测，从而推荐物品。

##### 2. 广告投放优化

**题目：** 设计一个基于贪心的广告投放优化算法，以最大化收益。

**答案：**

```python
import numpy as np

def ad_optimization(bid_prices, click_probabilities, budget):
    # bid_prices 是广告位价格，click_probabilities 是点击概率，budget 是预算
    scores = bid_prices * click_probabilities
    sorted_indices = np.argsort(scores)[::-1]
    spent_budget = 0
    selected_ads = []

    for i in range(len(bid_prices)):
        if spent_budget + bid_prices[sorted_indices[i]] <= budget:
            selected_ads.append(sorted_indices[i])
            spent_budget += bid_prices[sorted_indices[i]]
        else:
            break

    return selected_ads

# 示例数据
bid_prices = np.array([1.2, 0.8, 1.5, 1.0])
click_probabilities = np.array([0.3, 0.4, 0.2, 0.5])
budget = 2.5

selected_ads = ad_optimization(bid_prices, click_probabilities, budget)
print(selected_ads)
```

**解析：** 该算法根据广告位价格和点击概率计算得分，并按照得分从高到低排序，选择能够最大化收益的广告位进行投放。
