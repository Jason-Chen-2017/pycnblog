**解析：**

```python

from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
courses = pd.read_csv('courses.csv')
user_history = pd.read_csv('user_history.csv')

# 构建课程向量
course_vectors = courses.set_index('course_id')['course_subject'].values

similarity_matrix = cosine_similarity(courses.set_index('course_id')['course_subject'].values)

user_recommendations = {}
for idx, row in user_history.iterrows():
```

#### 题目 5：社群互动分析

**问题描述：**
分析社群互动数据，识别核心用户和潜在核心用户，以及社群的活跃时段。

**输入：**
- 社群互动数据（CSV格式）
```csv
user_id,post_id,timestamp,likes,comments
1,1,2021-01-01 10:00:00,5,2
1,2,2021-01-02 10:00:00,10,5
2,1,2021-01-01 11:00:00,3,0
...
```

**输出：**
- 核心用户列表
- 潜在核心用户列表
- 社群活跃时段

**解析：**
社群互动分析可以使用社群活跃度、参与度等指标来识别核心用户。同时，可以通过分析互动时间来识别社群活跃时段。

**答案：**
```python
import pandas as pd
import numpy as np

# 加载数据
interactions = pd.read_csv('social_interactions.csv')

# 计算用户互动度
interactions['interaction_score'] = interactions['likes'] + interactions['comments']
user_interactions = interactions.groupby('user_id')['interaction_score'].sum()

# 确定核心用户和潜在核心用户
core_users = user_interactions[user_interactions > np.mean(user_interactions) + 2 * np.std(user_interactions)].index.tolist()
potential_core_users = user_interactions[user_interactions > np.mean(user_interactions) + np.std(user_interactions)].index.tolist()

# 排除核心用户，计算潜在核心用户
for user in core_users:
    potential_core_users.remove(user)

# 计算社群活跃时段
hourly_interactions = interactions.groupby(interactions['timestamp'].dt.hour).size()
active_hours = hourly_interactions[hourly_interactions > np.mean(hourly_interactions) + 2 * np.std(hourly_interactions)].index.tolist()

# 输出结果
print(f"Core Users: {core_users}")
print(f"Potential Core Users: {potential_core_users}")
print(f"Active Hours: {active_hours}")
```
