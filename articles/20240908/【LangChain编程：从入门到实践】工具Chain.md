### 3. 如何安装和使用LangChain？

**题目：** 请说明如何在本地安装并使用LangChain。

**答案：**

**安装：**

1. 安装Python环境。
2. 通过pip安装LangChain库：

   ```shell
   pip install langchain
   ```

**使用：**

1. 导入LangChain库：

   ```python
   from langchain import Chain
   ```

2. 创建Chain对象：

   ```python
   chain = Chain()
   ```

3. 添加链中操作：

   ```python
   chain.add_operation("print", func(x: str) -> str:
       print(x)
       return x
   )
   ```

4. 执行Chain：

   ```python
   result = chain.execute("Hello, World!")
   print(result)
   ```
