### 3. 请实现一个简单的代码生成器。
**面试题：** 使用Python实现一个简单的代码生成器，用于生成指定格式的Hello World程序。

**答案：** 

```python
# coding=utf-8

def generate_hello_world(filename):
    with open(filename, 'w', encoding='utf-8') as f:
        f.write('''\
        #include <iostream>
        int main() {
            std::cout << "Hello, World!" << std::endl;
            return 0;
        }''')

generate_hello_world('hello_world.cpp')
```

**解析：** 该代码生成器接受一个文件名作为参数，并将其写入一个C++文件中，内容为简单的Hello World程序。
