#### 二、算法编程题库

**1. 实现一个函数，找出字符串中的最长重复子串。**

**输入：** `s = "abcdabcd"`, 输出：`"abcd"`, 因为 "abcd" 在字符串中出现了两次。

**答案：**

```python
def longest_repeated_substring(s):
    n = len(s)
    for i in range(1, n // 2 + 1):
        sub = s[:i]
        if sub * (n // i) == s:
            return sub
    return ""

print(longest_repeated_substring("abcdabcd"))
```

**解析：** 该函数通过遍历字符串的长度，尝试不同的重复次数，找出最长的重复子串。

**2. 设计一个函数，计算字符串中每个字符的词频。**

**输入：** `s = "hello hello world"`, 输出：`{'h': 2, 'e': 1, 'l': 3, 'o': 3, ' ': 2, 'w': 1, 'r': 1, 'd': 1}`。

**答案：**

```python
def char_frequency(s):
    freq = {}
    for c in s:
        freq[c] = freq.get(c, 0) + 1
    return freq

print(char_frequency("hello hello world"))
```

**解析：** 该函数使用字典来存储每个字符的词频，通过遍历字符串，更新字典中的值。
