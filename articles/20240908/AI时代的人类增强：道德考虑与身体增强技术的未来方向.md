### 3. 算法编程题

**题目4：**请实现一个函数，找出两个整数数组中的最大公约数。

**答案4：**以下是使用Python实现的示例代码：

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

print(gcd(60, 48))  # 输出 12
```

**解析4：**该函数使用辗转相除法（也称为欧几里得算法）来计算两个整数的最大公约数。算法的基本思想是用较小的数字去除较大的数字，然后用余数替换较大的数字，重复这个过程直到余数为0，此时的较小数字即为最大公约数。
