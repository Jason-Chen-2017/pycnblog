#### 面试题库

##### 1. 什么是单例模式？请简要介绍其应用场景和实现方法。

**答案：** 单例模式是一种设计模式，确保一个类只有一个实例，并提供一个访问它的全局访问点。应用场景包括数据库连接、配置对象、线程池等。实现方法主要有以下几种：

- **懒汉式（懒加载）：** 在类加载时，不初始化实例，首次使用时初始化。
- **饿汉式（饿加载）：** 在类加载时，立即初始化实例。
- **静态内部类：** 通过静态内部类实现单例。
- **枚举：** 通过枚举实现单例，适用于需要多个实例的场景。

##### 2. 讲解一下Java中的内存泄漏问题，以及如何避免。

**答案：** 内存泄漏是指程序中某些内存资源无法被垃圾回收器回收，导致内存占用不断增加。Java中的内存泄漏通常由以下原因引起：

- 长生命周期对象持有短生命周期对象。
- 静态集合类（如HashMap）中遗留的键值对。
- 使用线程共享的对象，导致线程无法回收。
- 闭包或内部类引用外部类成员变量。

为了避免内存泄漏，可以采取以下措施：

- 及时释放不再使用的资源。
- 尽量使用局部变量，避免使用静态变量。
- 使用弱引用（WeakReference）。
- 使用软引用（SoftReference）和虚引用（PhantomReference）。

##### 3. 请解释一下TCP三次握手的过程，以及为什么需要它。

**答案：** TCP三次握手是建立TCP连接的过程，包括以下三个步骤：

1. 客户端发送SYN报文到服务器，并进入SYN_SENT状态。
2. 服务器收到SYN报文后，回应一个SYN+ACK报文，并进入SYN_RECEIVED状态。
3. 客户端收到服务器的SYN+ACK报文后，发送一个ACK报文给服务器，并进入ESTABLISHED状态。

三次握手的目的：

- 确认双方通信的可靠性。
- 建立双方初始序列号。
- 防止旧的连接请求建立错误的连接。

```java
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    if (l1 == null) {
        return l2;
    }
    if (l2 == null) {
        return l1;
    }
    if (l1.val < l2.val) {
        l1.next = mergeTwoLists(l1.next, l2);
        return l1;
    } else {
        l2.next = mergeTwoLists(l1, l2.next);
        return l2;
    }
}
```

```java
    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6};
        quickSort(arr, 0, arr.length - 1);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }

```

**答案：** 快速排序的基本思想是通过一趟排序将数组分为两部分，其中一部分的所有元素都比另一部分的所有元素要小，然后递归地对这两部分进行排序。

```java
public static void quickSort(int[] arr, int low, int high) {
    if (low < high) {
        int pivot = partition(arr, low, high);
        quickSort(arr, low, pivot - 1);
        quickSort(arr, pivot + 1, high);
    }
}

public static int partition(int[] arr, int low, int high) {
    int pivot = arr[high];
    int i = low - 1;
    for (int j = low; j <= high - 1; j++) {
        if (arr[j] < pivot) {
            i++;
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
        }
    }
    int temp = arr[i + 1];
    arr[i + 1] = arr[high];
    arr[high] = temp;
    return i + 1;
}
```
