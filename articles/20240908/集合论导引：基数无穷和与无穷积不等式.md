**1. 判断题：** 无穷集合的基数一定是无穷的吗？

**答案：** 无穷大。因为无穷集与有限集或无穷集的并集仍然是无穷集。

**1. 编程题：** 编写一个程序，计算两个无穷集合的笛卡尔积。

**答案：** 

```python
def cartesian_product(set1, set2):
    return [(x, y) for x in set1 for y in set2]

# 示例
A = [1, 2, 3]
B = ['a', 'b']
print(cartesian_product(A, B))
```

**2. 编程题：** 编写一个程序，判断给定集合是否为可数无穷集合。

**答案：** 

```python
def is_countably_infinite(s):
    try:
        iterator = iter(s)
        while True:
            next(iterator)
    except StopIteration:
        return False
    return True

# 示例
s = [1, 2, 3, ...]  # 无穷序列
print(is_countably_infinite(s))
```

**3. 编程题：** 编写一个程序，计算无穷级数的和，其中每一项为前一项的倒数。

**答案：** 

```python
def infinite_series_sum():
    a = 1
    sum = 0
    while True:
        sum += 1 / a
        a += 1
    return sum

# 示例
print(infinite_series_sum())
```
