### 6. 编写一个航班信息爬虫，获取航班起降时间、航空公司、航班号等信息。

**答案：** 使用Python的requests和BeautifulSoup库编写航班信息爬虫，代码如下：

```python
import requests
from bs4 import BeautifulSoup

def get_flight_info(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')
    flight_info = {}
    flight_info['departure_time'] = soup.find('div', {'class': 'departure-time'}).text
    flight_info['airline'] = soup.find('div', {'class': 'airline'}).text
    flight_info['flight_number'] = soup.find('div', {'class': 'flight-number'}).text
    return flight_info

url = 'https://www.example.com/flight/1234'
info = get_flight_info(url)
print(info)
```

**解析：** 该代码通过发送HTTP请求获取航班网页内容，使用BeautifulSoup解析HTML结构，提取航班起降时间、航空公司、航班号等信息。

5. **结果分析：** 分析聚类结果，对不同的延误原因进行分类。
