#### 二、算法编程题库

**1. 编写一个算法，计算家庭用电设备的最小功率组合，以实现家庭节能目标。**

**题目：** 请实现一个函数 `minPowerCombination`,输入家庭用电设备的功率列表 `powers`（单位：千瓦）和节能目标 `target`（单位：千瓦时），输出满足节能目标的最小功率组合。

**答案：**

```python
def minPowerCombination(powers, target):
    n = len(powers)
    dp = [[False] * (target + 1) for _ in range(n + 1)]
    dp[0][0] = True

    for i in range(1, n + 1):
        for j in range(target + 1):
            dp[i][j] = dp[i - 1][j]
            if j >= powers[i - 1]:
                dp[i][j] = dp[i][j] or dp[i - 1][j - powers[i - 1]]

    result = []
    i, j = n, target
    while i > 0 and j > 0:
        if dp[i][j] != dp[i - 1][j]:
            result.append(powers[i - 1])
            j -= powers[i - 1]
        i -= 1

    return result[::-1]
```

**解析：** 该算法使用动态规划方法求解最小功率组合问题，通过构造一个二维数组 `dp` 来记录在考虑前 `i` 个设备时，是否能够满足功率目标 `j`。最终根据 `dp` 数组回溯得到满足目标的最小功率组合。

**2. 设计一个智能家居系统中的事件触发机制，实现设备间的联动功能。**

**题目：** 请实现一个智能家居系统的事件触发机制，支持以下功能：

* 设备 A 指定条件满足时触发设备 B；
* 设备 C 指定条件满足时触发设备 D；
* 支持多个设备之间的复杂联动关系。

**答案：**

```python
class SmartHome:
    def __init__(self):
        self.devices = {}
        self.conditions = {}

    def register_device(self, device_name, device):
        self.devices[device_name] = device

    def set_condition(self, condition_name, condition_func):
        self.conditions[condition_name] = condition_func

    def trigger(self, device_name, condition_name):
        device = self.devices[device_name]
        condition_func = self.conditions[condition_name]
        if condition_func(device):
            device.action()

    def complex_trigger(self, trigger_rules):
        for rule in trigger_rules:
            device_name = rule['device']
            condition_name = rule['condition']
            self.trigger(device_name, condition_name)
```

**解析：** 该实现通过 `SmartHome` 类来管理智能家居系统中的设备、条件和联动关系。`register_device` 方法用于注册设备，`set_condition` 方法用于设置条件，`trigger` 方法用于触发设备，`complex_trigger` 方法用于处理复杂的联动关系。
