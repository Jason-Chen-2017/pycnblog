### 二、算法编程题实战

#### 1. 百度——字符串匹配算法

**题目：** 实现字符串匹配算法，找出字符串中的所有子串。

**答案：** 可以使用KMP（Knuth-Morris-Pratt）算法实现。

```python
def KMPpatternSearch(pattern, text):
    # 构建部分匹配表
    lps = [0] * len(pattern)
    j = 0
    i = 0
    while i < len(pattern):
        if pattern[i] == pattern[j]:
            lps[i] = j
            i += 1
            j += 1
        else:
            if j != 0:
                j = lps[j - 1]
            else:
                i += 1

    i = 0  # index for text[]
    j = 0  # index for pattern[]
    while i < len(text):
        if pattern[j] == text[i]:
            i += 1
            j += 1
        if j == len(pattern):
            print(f"Found pattern at index: {i - j}")
            j = lps[j - 1]
        elif i < len(text) and pattern[j] != text[i]:
            if j != 0:
                j = lps[j - 1]
            else:
                i += 1

# 测试
pattern = "ABABCABAB"
text = "ABABDBABABCABAB"
KMPpatternSearch(pattern, text)
```

**解析：** KMP算法是一种高效的字符串匹配算法，能够减少不必要的比较，提高匹配速度。

#### 2. 京东——排序算法

**题目：** 实现快速排序算法，对数组进行排序。

**答案：**

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

# 测试
arr = [3, 6, 8, 10, 1, 2, 1]
print("Original array:", arr)
print("Sorted array:", quick_sort(arr))
```

**解析：** 快速排序是一种高效的排序算法，其平均时间复杂度为O(nlogn)。
