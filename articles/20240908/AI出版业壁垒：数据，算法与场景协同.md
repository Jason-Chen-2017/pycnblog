### 算法编程题库

#### 1. 实现一个基于K-means算法的书籍分类器。

**题目：** 编写一个程序，使用K-means算法对一组书籍进行分类。

**答案：** K-means算法是一种聚类算法，可以通过以下步骤实现书籍分类：

- **初始化：** 随机选择K个书籍作为初始聚类中心。
- **分配：** 将每个书籍分配到最近的聚类中心。
- **更新：** 根据分配结果重新计算聚类中心。
- **迭代：** 重复分配和更新步骤，直到聚类中心不再变化。

**源代码示例：**

```python
import numpy as np

def kmeans(data, K, num_iterations):
    centroids = data[np.random.choice(data.shape[0], K, replace=False)]
    for _ in range(num_iterations):
        # 分配
        labels = np.argmin(np.linalg.norm(data[:, np.newaxis] - centroids, axis=2), axis=1)
        # 更新
        centroids = np.array([data[labels == k].mean(axis=0) for k in range(K)])
    return centroids, labels

data = np.random.rand(100, 2)
centroids, labels = kmeans(data, 3, 10)
print("Centroids:", centroids)
print("Labels:", labels)
```

#### 2. 实现一个基于TF-IDF的文本相似度计算器。

**题目：** 编写一个程序，计算两段文本的相似度，使用TF-IDF算法。

**答案：** TF-IDF（Term Frequency-Inverse Document Frequency）是一种用于文本相似度计算的指标，可以通过以下步骤实现：

- **计算词频（TF）：** 对于每个词，计算其在文档中出现的频率。
- **计算逆文档频率（IDF）：** 对于每个词，计算其在所有文档中出现的频率的倒数。
- **计算TF-IDF：** 将词频与逆文档频率相乘，得到TF-IDF值。

**源代码示例：**

```python
from sklearn.feature_extraction.text import TfidfVectorizer

def compute_similarity(text1, text2):
    vectorizer = TfidfVectorizer()
    tfidf_matrix = vectorizer.fit_transform([text1, text2])
    return tfidf_matrix[0] @ tfidf_matrix[1] / (np.linalg.norm(tfidf_matrix[0]) * np.linalg.norm(tfidf_matrix[1]))

text1 = "AI技术在出版业中的应用"
text2 = "人工智能在出版行业的应用前景"
similarity = compute_similarity(text1, text2)
print("Similarity:", similarity)
```
