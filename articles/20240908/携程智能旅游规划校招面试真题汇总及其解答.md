#### 面试题 1：如何评估一个旅游景点的热度？

**题目描述：** 给定一个旅游景点的ID，以及一组用户的活动日志，每个日志包含用户ID和活动时间戳，要求设计一个算法来评估每个景点的热度。

**答案：** 可以使用以下方法来评估一个旅游景点的热度：

1. **计算每个景点的访问频率：** 通过用户的活动日志，统计每个景点被访问的次数。
2. **计算每个景点的访问时长：** 通过用户的活动日志，计算每个景点的平均访问时长。
3. **结合访问频率和访问时长：** 将访问频率和访问时长进行加权，以综合评估景点的热度。

**代码示例：**

```python
def calculate_hotness(logs, spot_id):
    # 访问频率
    frequency = sum(1 for log in logs if log['spot_id'] == spot_id)
    # 访问时长
    total_time = sum(log['end_time'] - log['start_time'] for log in logs if log['spot_id'] == spot_id)
    # 计算热度
    hotness = frequency + (total_time / 60)  # 假设时间单位为分钟
    return hotness
```

**解析：** 通过访问频率和访问时长，可以较为准确地评估一个景点的热度。实际应用中，还可以考虑其他因素，如用户评价、天气情况等。

      .attr("y", function(d) { return y(d热度); })
      .attr("width", x.bandwidth())
      .attr("height", function(d) { return height - y(d热度); });

  g.append("g")
      .attr("transform", "translate(0," + height + ")")
      .call(d3.axisBottom(x));

  g.append("g")
      .call(d3.axisLeft(y));
