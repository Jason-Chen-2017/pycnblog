##### 1. 最长公共子序列（LCS）

**题目描述：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。

**示例代码：**

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

# 示例
str1 = "ABCD"
str2 = "ACDF"
print(longest_common_subsequence(str1, str2))  # 输出 3
```

**解析：** 使用动态规划求解，定义一个二维数组 `dp`，其中 `dp[i][j]` 表示 `str1[0..i-1]` 和 `str2[0..j-1]` 的最长公共子序列长度。遍历字符串，更新 `dp` 数组，最终得到最长公共子序列长度。

##### 2. 二分查找

**题目描述：** 在一个有序数组中查找一个目标值，返回其索引。

**示例代码：**

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1

# 示例
arr = [1, 3, 5, 7, 9]
target = 5
print(binary_search(arr, target))  # 输出 2
```

**解析：** 二分查找的基本思想是不断将查找范围缩小一半，直到找到目标值或确定不存在。通过更新 `left` 和 `right` 的值，逐步缩小查找范围。

##### 3. 背包问题

**题目描述：** 有一个背包，容量为 `V`，给定 `N` 个物品，每个物品的重量和价值已知，求解能够装入背包的最大价值。

**示例代码：**

```python
def knapsack(W, weights, values, N):
    dp = [[0] * (W + 1) for _ in range(N + 1)]

    for i in range(1, N + 1):
        for w in range(1, W + 1):
            if weights[i - 1] <= w:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - weights[i - 1]] + values[i - 1])
            else:
                dp[i][w] = dp[i - 1][w]

    return dp[N][W]

# 示例
W = 5
weights = [1, 2, 3, 4]
values = [1, 4, 5, 7]
N = 4
print(knapsack(W, weights, values, N))  # 输出 9
```

**解析：** 使用动态规划求解，定义一个二维数组 `dp`，其中 `dp[i][w]` 表示在前 `i` 个物品中，能够装入容量为 `w` 的背包的最大价值。通过更新 `dp` 数组，最终得到最大价值。
