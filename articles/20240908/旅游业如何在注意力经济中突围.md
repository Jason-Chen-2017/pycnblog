#### 算法编程题库

1. **旅游路线规划（最短路径算法）**

**题目：** 给定一个旅游景点的权重矩阵，设计一个算法计算从一个景点到另一个景点的最短路径。

**答案：** 可以采用 Dijkstra 算法来计算最短路径。首先，初始化一个距离数组，记录每个景点到起点的距离；然后，利用一个优先队列（如小根堆）选择未访问的景点，更新距离数组；最后，遍历距离数组，找到终点到起点的最短路径。

```python
import heapq

def dijkstra(graph, start):
    n = len(graph)
    dist = [float('inf')] * n
    dist[start] = 0
    pq = [(0, start)]
    while pq:
        cur_dist, cur_vertex = heapq.heappop(pq)
        if cur_dist > dist[cur_vertex]:
            continue
        for neighbor, weight in enumerate(graph[cur_vertex]):
            dist[neighbor] = min(dist[neighbor], cur_dist + weight)
            heapq.heappush(pq, (dist[neighbor], neighbor))
    return dist

graph = [
    [0, 2, 4, float('inf'), float('inf')],
    [2, 0, float('inf'), 1, 6],
    [4, float('inf'), 0, 3, float('inf')],
    [float('inf'), 1, 3, 0, 1],
    [float('inf'), 6, float('inf'), 1, 0]
]

print(dijkstra(graph, 0)) # 输出 [0, 2, 4, 5, 6]
```

**解析：** 这个问题考察了面试者对最短路径算法的实现能力，以及对时间复杂度和空间复杂度的理解。
