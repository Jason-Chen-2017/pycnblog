### 1. 股票投资策略

**题目：** 请简述价值投资和成长投资的特点，并谈谈您对这两种投资策略的看法。

**答案：**

- **价值投资：** 重视股票的内在价值，选择低估的股票进行投资。长期持有，等待市场认识到股票的价值。
- **成长投资：** 重视企业的成长潜力，选择具有高速增长潜力的股票进行投资。更注重短期内的股价波动。

**解析：** 价值投资和成长投资各有优劣，投资者应根据自身的风险承受能力和投资目标进行选择。

#### 算法编程题与解析

### 4. 股票买卖最佳时机

**题目：** 给定一个整数数组 `prices`，其中 `prices[i]` 是第 `i` 天的情绪。如果满足以下条件，则可以完成一趟买卖：

1. 选中一天进行买入，然后在接下来的几天内连续卖出。
2. 买入日不能是卖出日的当天和之前。
3. 卖出日不能是买入日的当天和之后。

请返回通过一趟买卖能获取的最大利润。

**代码示例：**

```python
def maxProfit(prices):
    max_profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i - 1]:
            max_profit += prices[i] - prices[i - 1]
    return max_profit
```

**解析：** 该算法通过遍历价格数组，记录每天与前一天价格的差值，将差值累加即为最大利润。

### 5. 预测股票价格

**题目：** 给定一个整数数组 `prices`，其中 `prices[i]` 表示第 `i` 天的股票价格。请设计一个算法，预测接下来 3 天的股票价格，并返回一个包含 3 个整数的数组，表示第 4 天的预测价格。

**代码示例：**

```python
from typing import List

def predict(prices: List[int]) -> List[int]:
    return [prices[-1], prices[-2], prices[-3]]
```

**解析：** 该算法直接返回数组中的最后三个元素，作为对未来三天的预测。
