2. **设计一个算法，找出一个字符串中的最长子串，其中包含的字符都不重复。**
   ```python
   def longest_substring(s):
       char_set = set()
       start = 0
       max_len = 0
       max_start = 0

       for i, c in enumerate(s):
           while c in char_set:
               char_set.remove(s[start])
               start += 1
           char_set.add(c)
           if i - start + 1 > max_len:
               max_len = i - start + 1
               max_start = start
       return s[max_start:max_start + max_len]

   # 测试
   print(longest_substring("abcabcbb"))  # 输出 "abc"
   ```
