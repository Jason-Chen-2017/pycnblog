#### 算法编程题库

### 3. 设计一个算法，找出一个未排序数组中的两个数，使得他们的和等于目标值。

**答案解析：** 这种问题常见于技术面试，它考察的是算法的设计和逻辑思维。可以通过使用哈希表来解决这个问题，将数组中的每个数作为键存储在哈希表中，然后遍历数组，对于每个数，用目标值减去当前数，查看哈希表中是否存在该差值。

**实例代码：**

```python
def findTwoNumbers(nums, target):
    hash_set = set()
    for num in nums:
        complement = target - num
        if complement in hash_set:
            return [complement, num]
        hash_set.add(num)
    return []
```

### 4. 给定一个字符串，找出不重复的最长子串。

**答案解析：** 这道题目考察了字符串处理能力和对哈希表的理解。可以通过使用哈希表来记录每个字符出现的最远位置，然后遍历字符串，更新哈希表，找出最长的不重复子串。

**实例代码：**

```python
def findLongestSubstring(s):
    last_seen = {}
    max_length = 0
    start = 0
    for i, c in enumerate(s):
        if c in last_seen:
            start = max(start, last_seen[c] + 1)
        last_seen[c] = i
        max_length = max(max_length, i - start + 1)
    return max_length
```
