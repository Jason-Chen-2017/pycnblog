### 2. 如何实现文本数据的情感分析？

**答案：** 实现文本数据的情感分析通常涉及以下步骤：

1. **数据预处理：** 清洗和标准化文本数据，包括去除停用词、标点符号、转换为小写等。
2. **特征提取：** 提取文本数据中的关键特征，如词袋模型、TF-IDF、Word2Vec等。
3. **模型训练：** 使用机器学习算法（如SVM、决策树、神经网络等）训练模型，将文本数据映射到情感标签。
4. **模型评估：** 使用交叉验证、混淆矩阵、准确率、召回率等指标评估模型性能。
5. **结果输出：** 对输入文本进行情感分析，输出相应的情感标签和置信度。

**示例代码：**

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
from sklearn.svm import LinearSVC
from sklearn.metrics import classification_report

# 数据准备
texts = ["我很开心", "我很生气", "我有点累"]
labels = ["积极", "消极", "中性"]

# 特征提取
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(texts)

# 模型训练
X_train, X_test, y_train, y_test = train_test_split(X, labels, test_size=0.2)
model = LinearSVC()
model.fit(X_train, y_train)

# 模型评估
predictions = model.predict(X_test)
print(classification_report(y_test, predictions))
```
