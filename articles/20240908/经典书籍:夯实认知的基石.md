#### 一、面试题库

**1. 算法面试中的常用数据结构有哪些？分别适用于什么场景？**

**答案：** 算法面试中的常用数据结构包括数组、链表、栈、队列、树、图等。这些数据结构适用于不同的场景：

- **数组**：适用于随机访问的场景，如查找、排序等。
- **链表**：适用于插入和删除操作频繁的场景，如实现 LRU 缓存等。
- **栈**：适用于后进先出（LIFO）的场景，如递归、表达式求值等。
- **队列**：适用于先进先出（FIFO）的场景，如任务调度、广度优先搜索等。
- **树**：适用于层次结构的场景，如二叉树、平衡树等。
- **图**：适用于网络、社交图等复杂结构的场景。

**2. 如何实现一个堆？堆在什么场景下使用？**

**答案：** 堆是一种优先队列数据结构，它可以高效地获取最大或最小元素。实现堆可以通过以下步骤：

- **初始化**：将数据元素存储在一个数组中，并根据元素之间的关系建立堆结构。
- **调整**：当插入或删除元素时，通过上滤（sift-up）或下滤（sift-down）操作来保持堆的性质。
- **获取最大/最小元素**：直接访问堆顶元素。

堆在以下场景下使用：

- **优先级队列**：在需要根据优先级顺序处理任务的场景，如任务调度。
- **最短路径算法**：如 Dijkstra 算法使用最小堆来获取最短路径。
- **拓扑排序**：利用堆来优化计算时间。

**3. 如何实现一个快速排序算法？**

**答案：** 快速排序是一种基于分治策略的排序算法，其基本步骤如下：

- **选择基准元素**：从数组中选一个元素作为基准。
- **划分**：将数组划分为两个子数组，一个包含小于基准的元素，另一个包含大于基准的元素。
- **递归排序**：对两个子数组分别递归执行快速排序。

**代码示例：**

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

arr = [3, 6, 8, 10, 1, 2, 1]
print(quicksort(arr))
```

**4. 什么是动态规划？请给出一个动态规划的问题示例。**

**答案：** 动态规划是一种优化算法，通过将问题分解为子问题并存储子问题的解来避免重复计算。

**示例问题：斐波那契数列**

**问题描述：** 计算斐波那契数列的第 n 项。

**动态规划解法：**

```python
def fibonacci(n):
    if n <= 1:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n]

print(fibonacci(10))
```

**5. 什么是贪心算法？请给出一个贪心算法的问题示例。**

**答案：** 贪心算法是一种在每一步选择中选择当前最优解的策略。

**示例问题：装载货物问题**

**问题描述：** 给定一组不同重量的货物和一辆载重为 W 的卡车，如何选择货物使得卡车的装载重量最接近但不超过 W？

**贪心算法解法：**

```python
def load_cargo(cargo_weights, W):
    cargo_weights.sort(reverse=True)
    total_weight = 0
    for weight in cargo_weights:
        if total_weight + weight <= W:
            total_weight += weight
        else:
            break
    return total_weight

print(load_cargo([1, 2, 3, 4, 5], 10))
```

#### 二、算法编程题库

**1. 寻找旋转排序数组中的最小值**

**问题描述：** 已知一个长度为 n 的数组，原本是升序排列的，但被旋转了 k 次（k < n）。例如，数组 `[0,1,2,4,5,6,7]` 可能被旋转成 `[4,5,6,7,0,1,2]`。请找出并返回数组中的最小元素。

**解题思路：** 使用二分查找的方法，每次比较中间元素和两端元素的大小关系，缩小查找范围。

**代码示例：**

```python
def find_min(nums):
    left, right = 0, len(nums) - 1
    while left < right:
        mid = (left + right) // 2
        if nums[mid] > nums[right]:
            left = mid + 1
        else:
            right = mid
    return nums[left]

nums = [3, 4, 5, 1, 2]
print(find_min(nums))
```

**2. 两数之和**

**问题描述：** 给定一个整数数组 `nums` 和一个目标值 `target`，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**解题思路：** 使用哈希表存储数组中的元素及其索引，遍历数组并查找 `target - nums[i]` 是否存在于哈希表中。

**代码示例：**

```python
def two_sum(nums, target):
    hashmap = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hashmap:
            return [hashmap[complement], i]
        hashmap[num] = i
    return []

nums = [2, 7, 11, 15]
target = 9
print(two_sum(nums, target))
```

**3. 无重复字符的最长子串**

**问题描述：** 给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。

**解题思路：** 使用双指针和哈希表记录字符出现的位置，移动右指针扩展子串，左指针收缩子串。

**代码示例：**

```python
def length_of_longest_substring(s):
    left, right = 0, 0
    max_len = 0
    char_index = {}
    while right < len(s):
        if s[right] in char_index:
            left = max(left, char_index[s[right]] + 1)
        char_index[s[right]] = right
        max_len = max(max_len, right - left + 1)
        right += 1
    return max_len

s = "abcabcbb"
print(length_of_longest_substring(s))
```

**4. 盗梦空间**

**问题描述：** 在一栋100层的楼上，有一个神秘的房间，每上一层都需要花费1分钟的时间，但是每层都会随机增加或减少需要的时间，你需要设计一个算法来找到进入该房间的最快时间。

**解题思路：** 使用动态规划记录到每一层所需的最短时间，从下往上计算。

**代码示例：**

```python
def fastest_time(rooms):
    n = len(rooms)
    dp = [[0] * n for _ in range(n)]
    for i in range(n):
        dp[i][i] = rooms[i]
    for i in range(n - 2, -1, -1):
        for j in range(i + 1, n):
            dp[i][j] = min(dp[i + 1][j], dp[i][j - 1]) + rooms[i]
    return dp[0][n - 1]

rooms = [3, 7, 1, 6, 4]
print(fastest_time(rooms))
```

**5. 合并区间**

**问题描述：** 给出一组区间，你需要合并所有重叠的区间，并返回合并后的区间列表。

**解题思路：** 首先将区间按起始值排序，然后遍历区间，合并重叠的区间。

**代码示例：**

```python
def merge(intervals):
    intervals.sort(key=lambda x: x[0])
    result = []
    for interval in intervals:
        if not result or result[-1][1] < interval[0]:
            result.append(interval)
        else:
            result[-1][1] = max(result[-1][1], interval[1])
    return result

intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]
print(merge(intervals))
```
