```go
// 假设用户A和用户B在元宇宙中的互动次数记录在一个切片中，编写函数计算他们的互动次数。

    // 实现逻辑...
}
```
**答案：**
```go
func calculateInteractions(interactions []int) (int, int) {
    countA, countB := 0, 0
    for _, interaction := range interactions {
        if interaction == 1 {
            countA++
        } else if interaction == 2 {
            countB++
        }
    }
    return countA, countB
}
```
**解析：** 该函数通过遍历互动记录切片，分别计算用户A和用户B的互动次数。
