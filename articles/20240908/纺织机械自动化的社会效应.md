```python
def minMachines(requiredProduction, machineCapabilities):
    # 对机器生产能力进行降序排序
    sorted_capabilities = sorted(machineCapabilities, reverse=True)

    machines = 0
    total_production = 0

    for capability in sorted_capabilities:
        if total_production + capability <= requiredProduction:
            total_production += capability
            machines += 1
        else:
            break

    return machines
```
