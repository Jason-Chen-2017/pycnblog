```python
    # 搜索包含关键词的课程
    matched_courses = []
    for course in course_data:
        if keywords in course['title'] or keywords in course['description']:
            matched_courses.append(course)
    # 返回搜索结果
    return matched_courses
```

**答案解析：** 遍历课程数据，检查课程标题和描述中是否包含关键词，返回匹配的结果。
