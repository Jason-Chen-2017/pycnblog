#### 2. 注意力经济下的算法编程题

**算法编程题：** 设计一个算法，计算在一个给定的社交媒体平台上，能够传播的最远距离。

**输入：** 一个社交媒体网络，表示为一个无向图，以及一个用户集合，表示为节点之间的连接。

**输出：** 最远距离。

**解析：** 这个问题可以通过广度优先搜索（BFS）来解决。首先，选择一个初始节点作为种子节点，然后使用 BFS 算法从种子节点开始进行搜索，记录每个节点的距离。最后，找到距离最远的节点，即为最远距离。

```python
from collections import defaultdict, deque

def longest_distance(graph, start):
    distances = {node: float('inf') for node in graph}
    distances[start] = 0
    queue = deque([start])
    while queue:
        node = queue.popleft()
        for neighbor in graph[node]:
            if distances[neighbor] > distances[node] + 1:
                distances[neighbor] = distances[node] + 1
                queue.append(neighbor)
    return max(distances.values())

graph = defaultdict(list)
graph['A'] = ['B', 'C']

start = 'A'
print(longest_distance(graph, start))
```

**解析：** 在这个例子中，我们使用 BFS 算法来计算从节点 'A' 开始的最远距离。首先，我们初始化一个距离字典，将所有节点的距离设置为无穷大。然后，我们从种子节点 'A' 开始，依次访问相邻节点，并更新它们的距离。最后，我们返回距离最远的节点的距离。
