**题目：** 请设计一个简单的基于人机协同的系统架构，并解释其核心组件和功能。

**答案：** 

系统架构如下：

```
人机协同系统架构
+----------------+       +----------------+
|    用户界面    |       |    后端服务    |
+----------------+       +----------------+
     |                  |
     | 交互请求         |                  |
     |----------------->|<-----------------|
     |                  |
     |    数据处理      |                  |
     |----------------->|<-----------------|
     |                  |
     |    辅助决策      |                  |
     |----------------->|<-----------------|
```
