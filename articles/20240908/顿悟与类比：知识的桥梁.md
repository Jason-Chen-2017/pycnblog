##### 题目1：快慢指针法求解链表中的环

**题目描述：** 给定一个链表，求链表中环的位置。

**答案解析：** 使用快慢指针法，快指针每次走两步，慢指针每次走一步。当快指针追上慢指针时，链表中存在环。

```go
func detectCycle(head *ListNode) *ListNode {
    slow, fast := head, head
    for fast != nil && fast.Next != nil {
        slow = slow.Next
        fast = fast.Next.Next
        if slow == fast {
            break
        }
    }
    if fast == nil || fast.Next == nil {
        return nil
    }
    slow = head
    for slow != fast {
        slow = slow.Next
        fast = fast.Next
    }
    return slow
}
```

##### 题目2：二分查找

**题目描述：** 给定一个有序数组，实现二分查找算法，查找目标值的位置。

**答案解析：** 二分查找算法通过不断缩小查找范围，实现高效的查找。

```go
func search(nums []int, target int) int {
    left, right := 0, len(nums)-1
    for left <= right {
        mid := (left + right) / 2
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}
```

##### 题目3：字符串匹配——KMP算法

**题目描述：** 给定一个文本字符串和一个模式串，实现 KMP 算法，找出模式串在文本中出现的所有位置。

**答案解析：** KMP 算法通过构建部分匹配表（next 数组），避免重复匹配，实现高效的字符串匹配。

```go
func KMP(text, pattern string) []int {
    next := make([]int, len(pattern))
    j := -1
    res := []int{}
    for i := 0; i < len(text); {
        if j == -1 || text[i] == pattern[j] {
            i++
            j++
            if j == len(pattern) {
                res = append(res, i-j)
                j = next[j-1]
            }
        } else {
            j = next[j-1]
        }
    }
    return res
}
```
