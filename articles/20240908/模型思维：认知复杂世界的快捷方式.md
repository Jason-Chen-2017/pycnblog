#### 算法编程题库

##### 题目 1：编写一个简单的线性回归模型。

**题目：** 编写一个Python函数，实现线性回归模型的基本功能，包括数据预处理、模型训练和模型预测。

**答案：** 下面是一个简单的线性回归模型实现的代码示例：

```python
import numpy as np

class LinearRegression:
    def __init__(self):
        self系数 = None

    def fit(self, X, y):
        X = np.insert(X, 0, 1, axis=1)
        self系数 = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)

    def predict(self, X):
        X = np.insert(X, 0, 1, axis=1)
        return X.dot(self系数)

# 示例数据
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 5, 4, 5])

model = LinearRegression()
model.fit(X, y)
print(model.predict(X))
```

##### 题目 2：实现一个基于k-近邻算法的推荐系统。

**题目：** 编写一个Python函数，实现k-近邻算法的基本功能，包括相似度计算、邻居选择和推荐。

**答案：** 下面是一个基于k-近邻算法的推荐系统的代码示例：

```python
from collections import Counter
from math import sqrt

def euclidean_distance(user1, user2):
    sum_of_squared_diff = sum((a - b) ** 2 for a, b in zip(user1, user2))
    return sqrt(sum_of_squared_diff)

def k_nearest_neighbors(train_data, test_user, k):
    distances = [euclidean_distance(test_user, user) for user in train_data]
    nearest_neighbors = sorted(zip(distances, train_data))[:k]
    neighbor_labels = [label for _, label in nearest_neighbors]
    most_common = Counter(neighbor_labels).most_common(1)
    return most_common[0][0]

# 示例数据
train_data = [
    [1, 5],
    [2, 4],
    [3, 6],
    [5, 3],
    [7, 1],
    [8, 2]
]

test_user = [4, 3]

k = 3
print(k_nearest_neighbors(train_data, test_user, k))
```
