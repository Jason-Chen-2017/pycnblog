```python

# 假设用户-物品评分矩阵为
user_item_matrix = np.array([[5, 3, 0, 1],
                              [4, 0, 0, 1],
                              [1, 1, 0, 5],
                              [1, 0, 0, 4],
                              [5, 4, 9, 2]])

# 实施SVD矩阵分解
U, sigma, Vt = np.linalg.svd(user_item_matrix, full_matrices=False)

# 重建评分矩阵
predicted_ratings = np.dot(U, np.dot(sigma, Vt))

# 假设我们要为第2个用户推荐商品

# 推荐评分最高的商品

```
