## 4.6数据安全

数据安全的具体代码实例如下：

```python
import pandas as pd
import hashlib

def encrypt_data(data, column):
    encrypted_data = data.copy()
    for index, row in encrypted_data.iterrows():
        value = row[column]
        encrypted_value = hashlib.sha256(value.encode()).hexdigest()
        encrypted_data.at[index, column] = encrypted_value
    return encrypted_data

data = pd.read_csv('data_result.csv')
encrypted_data = encrypt_data(data, 'result')
encrypted_data.to_csv('data_encrypted.csv', index=False)
```

解释说明：

1.使用pandas库的DataFrame的copy方法创建数据的副本。

2.使用hashlib库的sha256方法对数据进行加密。

3.使用DataFrame的to_csv方法将DataFrame对象转换为CSV文件，并指定文件路径。
