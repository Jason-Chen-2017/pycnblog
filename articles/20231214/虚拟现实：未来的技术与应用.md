                 

# 1.背景介绍

虚拟现实（Virtual Reality，简称VR）是一种将计算机生成的图像、音频和其他感官输入到用户的头盔、手臂、手套等设备中，使用户感觉自己身处于一个完全不同的虚拟世界中的技术。VR技术的发展历程可以分为三个阶段：早期的单目显示器、后来的立体视觉技术和最近的全身感应技术。

早期的单目显示器是由美国的Morton Heilig在1960年代开发的。他创造了一个名为“Sensorama”的设备，通过将视觉、听觉、触觉、嗅觉和纤维感应等感官输入到用户的头盔、手臂、手套等设备中，使用户感觉自己身处于一个完全不同的虚拟世界中。

后来的立体视觉技术是由美国的Ivan Sutherland在1960年代开发的。他创造了一个名为“Head-Mounted Display”的设备，通过将两个独立的图像分别输入到用户的左眼和右眼的眼睛中，使用户感觉自己身处于一个三维的虚拟世界中。

最近的全身感应技术是由美国的Palmer Luckey在2012年开发的。他创造了一个名为“Oculus Rift”的设备，通过将多个传感器和摄像头放置在用户的头盔、手臂、手套等设备中，使用户感觉自己身处于一个完全不同的虚拟世界中。
