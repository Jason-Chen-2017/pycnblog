# 4.具体代码实例和详细解释说明

在这里，我们以一个简单的机器学习任务为例，进行具体的代码实例和详细解释说明。

假设我们需要预测一个企业的未来销售额，我们可以使用线性回归（Linear Regression）算法进行预测。

首先，我们需要准备数据，包括历史销售额和相关的特征变量。然后，我们可以使用Scikit-learn库中的LinearRegression类进行训练和预测。

```python
from sklearn.linear_model import LinearRegression

# 准备数据
X = [[1], [2], [3], [4], [5]]  # 特征变量
y = [1, 2, 3, 4, 5]  # 历史销售额

# 创建模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 预测未来销售额
future_sales = model.predict([[6]])
print(future_sales)
```

在这个例子中，我们首先导入了Scikit-learn库中的LinearRegression类。然后，我们准备了数据，包括特征变量X和历史销售额y。接着，我们创建了一个LinearRegression模型，并使用fit()方法进行训练。最后，我们使用predict()方法进行预测，并输出未来销售额。
