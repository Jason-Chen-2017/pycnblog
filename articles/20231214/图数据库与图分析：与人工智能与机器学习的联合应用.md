## 3.1 图的表示

图数据库的核心概念是图，图由节点（ vertices ）和边（ edges ）组成。节点表示数据中的实体，边表示实体之间的关系。图可以用邻接矩阵或邻接表的形式来表示。

邻接矩阵是一个二维矩阵，其中每个元素表示两个节点之间的边的权重。邻接表是一个数组，其中每个元素是一个列表，列表中的每个元素表示一个节点的邻居节点。

## 3.2 图的遍历

图的遍历是图分析中的一个重要概念。图的遍历是指从图的一个节点出发，访问图中所有节点的过程。图的遍历可以用深度优先搜索（ DFS ）和广度优先搜索（ BFS ）等算法实现。

深度优先搜索是一种递归算法，它从图的一个节点出发，沿着一个节点的邻居节点的路径深入探索图中的其他节点。深度优先搜索可以用来发现图中的连通分量、发现图中的环等。

广度优先搜索是一种非递归算法，它从图的一个节点出发，以层次顺序访问图中的所有节点。广度优先搜索可以用来发现图中的最短路径、发现图中的最短路径等。

# 4.具体代码实例和详细解释说明

在这里，我们将提供一个具体的代码实例，以及对其解释的详细说明。

```python
import networkx as nx

# 创建一个图
G = nx.Graph()

# 添加节点
G.add_node(1)
G.add_node(2)
G.add_node(3)

# 添加边
G.add_edge(1, 2)
G.add_edge(2, 3)

# 遍历图
for node in G.nodes():
    print(node)

# 计算最短路径
shortest_path = nx.shortest_path(G, source=1, target=3)
print(shortest_path)
```

在这个代码实例中，我们使用Python的networkx库创建了一个图，并添加了三个节点和两个边。然后，我们遍历了图中的所有节点，并计算了从节点1到节点3的最短路径。
