## 2.1 goroutine
goroutine是Go语言中的轻量级线程，它是Go语言并发编程的基本单元。goroutine的创建和管理非常简单，只需使用go关键字即可。goroutine之间可以相互调用，并且可以在同一时刻运行多个goroutine，从而实现并发执行。

## 2.2 channel
channel是Go语言中用于实现并发通信的数据结构。channel可以用来实现goroutine之间的同步和通信，它是Go语言并发编程的核心概念之一。channel的创建和使用非常简单，只需使用make函数即可。channel可以用来实现各种并发模式，如生产者-消费者模式、读写锁等。

## 2.3 sync包
sync包是Go语言中用于实现并发同步的标准库包。sync包提供了一系列的同步原语，如Mutex、RWMutex、WaitGroup等，可以用来实现goroutine之间的同步和互斥。sync包提供了一种更高级的并发编程模型，可以用来实现更复杂的并发场景。

## 4.1 goroutine的使用
```go
package main

import "fmt"

func main() {
    go func() {
        fmt.Println("Hello, World!")
    }()

    fmt.Println("Hello, Go!")
}
```
在这个代码实例中，我们创建了一个匿名函数，并使用go关键字来创建一个goroutine。当主goroutine执行完成后，子goroutine会继续执行。

## 4.2 channel的使用
```go
package main

import "fmt"

func main() {
    ch := make(chan int)

    go func() {
        ch <- 1
    }()

    fmt.Println(<-ch)
}
```
在这个代码实例中，我们创建了一个channel，并使用go关键字创建了一个goroutine。goroutine通过channel发送一个整数值1，主goroutine通过接收channel的值来获取这个值。

## 4.3 sync包的使用
```go
package main

import "fmt"
import "sync"

func main() {
    var wg sync.WaitGroup
    wg.Add(1)

    go func() {
        defer wg.Done()
        fmt.Println("Hello, Go!")
    }()

    wg.Wait()
    fmt.Println("Hello, World!")
}
```
在这个代码实例中，我们使用sync包中的WaitGroup来实现goroutine的同步。我们创建了一个WaitGroup，并使用Add方法来添加一个等待的goroutine。当goroutine执行完成后，我们调用Done方法来表示这个goroutine已经完成。最后，我们调用Wait方法来等待所有的goroutine执行完成。
