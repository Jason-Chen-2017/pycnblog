# 2.核心概念与联系

在数据库系统中，复制和高可用性是两个重要的概念。复制是指将数据库中的数据复制到多个服务器上，以实现数据的备份和分布。高可用性是指数据库系统能够在故障发生时继续提供服务，以确保系统的可靠性。

复制和高可用性之间的联系是，复制可以帮助实现高可用性。通过复制数据，我们可以在多个服务器上存储数据，从而实现数据的备份和分布。当一个服务器发生故障时，其他服务器可以继续提供服务，从而确保系统的可用性。
