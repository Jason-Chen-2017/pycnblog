## 4.1 集中管理家居设备
为了实现集中管理家居设备，我们可以使用Python编程语言来编写代码。以下是一个简单的代码实例：

```python
import time

class Device:
    def __init__(self, name, state):
        self.name = name
        self.state = state

    def control(self, command):
        if command == 'on':
            self.state = True
        elif command == 'off':
            self.state = False
        else:
            raise ValueError('Invalid command')

    def get_state(self):
        return self.state

devices = []

def add_device(name, state):
    device = Device(name, state)
    devices.append(device)

def control_device(device_name, command):
    for device in devices:
        if device.name == device_name:
            device.control(command)
            break

# 添加设备
add_device('灯', True)
add_device('空调', False)

# 控制设备
control_device('灯', 'on')
control_device('空调', 'on')

# 获取设备状态
print(devices[0].get_state())  # True
print(devices[1].get_state())  # True
```

这个代码实例中，我们定义了一个`Device`类，用来表示家居设备。`Device`类有一个`control`方法，用来控制设备的状态，一个`get_state`方法，用来获取设备的状态。同时，我们还定义了一个`add_device`方法，用来添加设备，一个`control_device`方法，用来控制设备。

在代码中，我们添加了两个设备（灯和空调），并控制了它们的状态。同时，我们也获取了设备的状态。
