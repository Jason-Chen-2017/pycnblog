```python
def zen_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

在这个代码中，我们首先对数组进行遍历，然后找到最小的元素并将其与当前位置的元素进行交换。通过这种方式，我们可以使数组中的元素逐渐排序。
