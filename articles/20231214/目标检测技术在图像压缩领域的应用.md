# 2.核心概念与联系

## 2.1目标检测技术
目标检测技术是计算机视觉领域的一个重要分支，它的主要目的是识别图像中的目标物体。目标检测技术可以用于多种应用场景，如人脸识别、自动驾驶、视频分析等。目标检测技术的主要方法包括：

- 边界框检测（Bounding Box Detection）：将目标物体包围在一个矩形框内，然后对框进行分类和回归。
- 分割检测（Segmentation Detection）：将目标物体分割成多个区域，然后对区域进行分类和回归。
- 关键点检测（Keypoint Detection）：将目标物体表示为一组关键点，然后对关键点进行分类和回归。

## 2.2图像压缩
图像压缩是计算机图像处理的一个重要环节，它可以减少图像的大小，同时保持图像的质量和信息。图像压缩的主要方法包括：

- 有损压缩：有损压缩是指在压缩过程中丢失一定的图像信息，从而减少图像的大小。有损压缩的典型方法有JPEG、JPEG2000、WebP等。
- 无损压缩：无损压缩是指在压缩过程中不丢失图像信息，从而保持图像的质量和信息。无损压缩的典型方法有PNG、GIF、BMP等。
