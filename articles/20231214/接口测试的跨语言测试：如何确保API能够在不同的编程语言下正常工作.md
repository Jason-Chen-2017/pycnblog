# 2.核心概念与联系

在讨论跨语言测试之前，我们需要了解一些核心概念。

## 2.1 API

API（Application Programming Interface，应用程序接口）是一种规范，它规定了如何在不同的软件系统之间进行通信，以实现数据交换和功能集成。API 通常包括一组函数、类和方法，这些函数、类和方法可以被其他软件系统调用，以实现特定的功能。
