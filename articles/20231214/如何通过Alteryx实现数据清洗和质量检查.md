# 2.核心概念与联系
在进行数据清洗和质量检查之前，我们需要了解一些核心概念，包括数据质量、数据清洗、数据质量检查、数据准确性、数据完整性、数据一致性等。这些概念在数据分析过程中具有重要意义，可以帮助我们更好地理解数据清洗和质量检查的重要性。

数据质量是指数据的准确性、完整性、一致性等方面的程度。数据清洗是指对数据进行预处理和修正的过程，以消除错误、缺失值、重复值等问题。数据质量检查是指对数据进行验证和评估的过程，以确保数据的准确性、完整性和一致性。

在使用Alteryx进行数据清洗和质量检查时，我们需要关注以下几个方面：

- 数据清洗：包括去除重复数据、填充缺失值、修正错误数据等。
- 数据质量检查：包括检查数据的准确性、完整性和一致性等。
- 数据准确性：是指数据是否正确地反映了实际情况。
- 数据完整性：是指数据是否缺失或损坏。
- 数据一致性：是指数据是否在不同来源或不同时间点之间保持一致。
