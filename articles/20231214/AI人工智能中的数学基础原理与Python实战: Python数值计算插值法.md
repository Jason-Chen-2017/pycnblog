# 4.数学模型公式详细讲解

在本节中，我们将详细讲解插值法的数学模型公式。

## 4.1 线性插值公式

线性插值是一种简单的插值方法，它可以根据给定的两个数据点来构建一个线性函数模型。线性插值的公式如下：

$$
f(x) = y_0 + \frac{x - x_0}{x_1 - x_0} (y_1 - y_0)
$$

其中，$x_0$ 和 $y_0$ 是第一个数据点的坐标，$x_1$ 和 $y_1$ 是第二个数据点的坐标。

## 4.2 多项式插值公式

多项式插值是一种更复杂的插值方法，它可以根据给定的多个数据点来构建一个多项式函数模型。多项式插值的公式如下：

$$
f(x) = \sum_{i=0}^{n} a_i L_i(x)
$$

其中，$a_i$ 是多项式的系数，$L_i(x)$ 是基函数。
