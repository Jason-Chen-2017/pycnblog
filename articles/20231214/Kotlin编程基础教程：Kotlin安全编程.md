```kotlin
    println("$num1 + $num2 = $sum")

```

## 4.2 异常处理
```kotlin
fun main(args: Array<String>) {
    val num1: Int = 10
    val num2: Int = 0
    try {
        val sum: Int = num1 / num2
        println("$num1 / $num2 = $sum")
    } catch (e: ArithmeticException) {
        println("Cannot divide by zero")
    }
}
```
在这个代码实例中，我们定义了一个Int类型的变量num1，并将其初始化为10。然后，我们定义了一个Int类型的变量num2，并将其初始化为0。接下来，我们使用try-catch块来捕获ArithmeticException异常，并在catch块中进行处理。在这个代码中，当num2为0时，会捕获ArithmeticException异常，并在catch块中打印出"Cannot divide by zero"的提示信息。

## 4.3 资源管理
```kotlin
fun main(args: Array<String>) {
    val file = File("example.txt")
    try {
        val reader = file.bufferedReader()
        val content = reader.readText()
        println(content)
    } finally {
        reader.close()
    }
}
```
在这个代码实例中，我们定义了一个File类型的变量file，并将其初始化为"example.txt"。然后，我们使用try-finally块来管理文件读取资源。在try块中，我们使用bufferedReader()方法来创建一个BufferedReader对象，并使用readText()方法来读取文件的内容。在finally块中，我们使用close()方法来关闭BufferedReader对象，从而释放文件读取资源。
