```python
import requests
from bs4 import BeautifulSoup

def collect_data(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')
    return soup.find_all('div')

data = collect_data('http://example.com')
```

在这个代码实例中，我们使用了 Python 的 requests 库和 BeautifulSoup 库来收集网页数据。我们定义了一个名为 collect_data 的函数，它接受一个 URL 参数，并使用 requests 库发送 HTTP 请求，获取网页内容。然后，我们使用 BeautifulSoup 库解析网页内容，并找到所有的 div 元素。最后，我们返回找到的 div 元素的列表。

### 1.4.3 数据处理

```python
import pandas as pd

def process_data(df):
    df['text'] = df['text'].str.lower()
    df['words'] = df['text'].str.split()
    df['word_count'] = df['words'].apply(lambda x: len(x))
    return df

df = pd.read_csv('data.csv')
processed_df = process_data(df)
```

在这个代码实例中，我们使用了 Python 的 pandas 库来处理收集到的数据。我们定义了一个名为 process_data 的函数，它接受一个数据框参数，并对数据框进行清洗、转换和聚合。我们将文本内容转换为小写，然后将文本内容分割为单词，并计算单词的数量。最后，我们返回处理后的数据框。
