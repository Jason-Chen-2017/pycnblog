# 2.核心概念与联系

在行为金融学中，我们关注以下几个核心概念：

2. 情感因素：情感因素包括恐慌、欣喜、愤怒等，它们可能导致投资者的决策过程中出现波动。例如，投资者可能因为恐慌而卖掉一部分股票，或者因为欣喜而买入一部分股票。

3. 社会心理学因素：社会心理学因素包括群体行为、倾向性、信仰等，它们可能导致投资者的决策过程中出现集体行为。例如，投资者可能因为群体行为而同时买入或卖出一部分股票。
