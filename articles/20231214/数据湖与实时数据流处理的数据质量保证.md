### 4.1数据清洗

在数据清洗的过程中，我们需要对数据进行检查、修复和删除等操作。以下是一个具体的代码实例：

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 对数据进行检查
data['complete'] = data['column'].isnull().all(axis=1)
data['accurate'] = data['column'].str.contains('value').all()
data['consistent'] = data['column'].eq(data['column'].shift(1)).all()
data['reliable'] = data['column'].str.isnumeric().all()

# 对数据进行修复
data['column'] = data['column'].fillna('value')
data['column'] = data['column'].str.replace('value', 'value')

# 对数据进行删除
data = data[data['complete'] & data['accurate'] & data['consistent'] & data['reliable']]
```
