### 4.1 简单的随机数生成

### 4.2 计算概率

在这个例子中，我们将计算一组数据中的某个值出现的概率。我们将使用计数器变量来计算某个值出现的次数，然后将其除以总数据量来得到概率。

```python
import random

# 生成10个0-100之间的随机整数
random_numbers = [random.randint(0, 100) for _ in range(10)]

# 计算某个值出现的次数
value = 50
count = 0
for num in random_numbers:
    if num == value:
        count += 1

# 计算概率
probability = count / len(random_numbers)
print(probability)
```
