# 2.核心概念与联系
在虚拟现实与增强现实中，概率论与统计学是非常重要的。它们用于处理不确定性、预测行为、优化算法等方面。以下是一些核心概念：

1.随机变量：在虚拟现实与增强现实中，随机变量用于表示不确定性。它们可以是连续的（如位置、速度）或离散的（如颜色、形状）。

2.概率分布：概率分布描述了随机变量的取值概率。常见的概率分布有均匀分布、正态分布、指数分布等。

3.期望值：期望值是随机变量的一种度量，用于表示其取值的平均值。

4.方差：方差是随机变量的一种度量，用于表示其取值的离散程度。

5.协方差：协方差是两个随机变量之间的一种度量，用于表示它们之间的相关性。

6.最大似然估计：最大似然估计是一种用于估计参数的方法，它基于数据的概率密度函数。

7.贝叶斯定理：贝叶斯定理是一种用于更新概率的方法，它基于条件概率。
