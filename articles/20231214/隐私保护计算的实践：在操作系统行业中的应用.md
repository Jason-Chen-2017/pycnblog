## 4.1 加密技术的代码实例

以下是一个使用Python的加密技术的代码实例：

```python
import base64

def encrypt(message):
    key = b'secretkey'
    ciphertext = base64.b64encode(message.encode(key))
    return ciphertext

message = "Hello, World!"
ciphertext = encrypt(message)
print(ciphertext)
```

在这个代码实例中，我们使用了Python的base64模块来实现加密技术。我们首先定义了一个加密函数encrypt，然后使用这个函数来加密一个消息。最后，我们打印出加密后的消息。
