# 2.核心概念与联系

## 2.1 蒸馏技术的概念

蒸馏技术是一种将大型模型转换为较小模型的方法，以减少模型的计算复杂度和存储需求。通过蒸馏，我们可以生成一个较小的模型，这个模型的性能可以接近原始模型，但是计算复杂度和存储需求得到了显著的减少。

## 2.2 蒸馏技术与知识蒸馏的联系

知识蒸馏是一种特殊的蒸馏技术，它通过将大型模型转换为较小模型的方法，将模型中的知识转移到较小模型中。知识蒸馏的目标是保持模型性能的同时，减少模型的计算复杂度和存储需求。知识蒸馏可以通过多种方法实现，例如：

- 模型压缩：通过减少模型的参数数量，减少模型的计算复杂度和存储需求。
- 模型剪枝：通过删除模型中不重要的神经元和权重，减少模型的计算复杂度和存储需求。
- 模型量化：通过将模型的参数从浮点数转换为整数，减少模型的存储需求和计算复杂度。
