## 1.4 具体代码实例和详细解释说明

在本节中，我们将提供一个具体的代码实例，以便更好地理解智能合约的工作原理。

### 1.4.1 代码实例

以下是一个简单的智能合约代码实例：

```solidity
pragma solidity ^0.5.16;

contract SimpleContract {
    uint public totalBalance;

    constructor() public {
        totalBalance = 0;
    }

    function deposit() public payable {
        totalBalance += msg.value;
    }

    function withdraw() public {
        require(msg.sender.balance >= totalBalance);
        msg.sender.transfer(totalBalance);
        totalBalance = 0;
    }
}
```

### 1.4.2 代码解释

这个代码实例是一个简单的智能合约，它可以用于实现账户余额管理。这个合约包括以下几个方面：

1. 合约声明：合约声明包括合约名称和版本信息。这里我们使用Solidity语言，版本号为0.5.16。
2. 合约变量：合约变量包括公共变量和私有变量。这里我们定义了一个公共变量totalBalance，用于存储合约的总余额。
3. 合约构造函数：合约构造函数是合约的初始化方法。这里我们定义了一个构造函数，用于初始化totalBalance为0。
4. 合约函数：合约函数包括外部函数和内部函数。这里我们定义了两个函数，分别是deposit和withdraw。deposit函数用于接收ETH，并将其添加到totalBalance中。withdraw函数用于从账户中提取totalBalance，并将其转移给msg.sender。
5. 函数访问控制：函数访问控制用于限制函数的访问权限。这里我们使用require语句来限制withdraw函数的访问权限，以确保msg.sender的余额大于totalBalance。
