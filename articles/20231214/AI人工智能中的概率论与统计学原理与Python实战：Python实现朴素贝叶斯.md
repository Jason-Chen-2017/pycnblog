## 3.1 朴素贝叶斯算法的核心原理

朴素贝叶斯算法是一种基于贝叶斯定理的分类器，它假设特征之间相互独立。这种算法的核心原理是利用贝叶斯定理来计算类别概率，以便对新的数据进行分类。

贝叶斯定理是一种概率论的原理，它可以用来计算条件概率。贝叶斯定理的公式如下：

$$
P(A|B) = \frac{P(B|A) \times P(A)}{P(B)}
$$

在朴素贝叶斯算法中，我们需要计算条件概率$P(A|B)$，其中$A$是类别，$B$是特征。我们可以使用贝叶斯定理来计算这个条件概率。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明朴素贝叶斯算法的实现。

```python
from sklearn.naive_bayes import GaussianNB
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 收集数据
data = [[0, 0], [0, 1], [1, 0], [1, 1]]
labels = [0, 0, 1, 1]

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(data, labels, test_size=0.2, random_state=42)

# 训练算法
clf = GaussianNB()
clf.fit(X_train, y_train)

# 对新数据进行分类
y_pred = clf.predict(X_test)

# 计算准确率
accuracy = accuracy_score(y_test, y_pred)
print("准确率:", accuracy)
```

在这个代码实例中，我们首先收集了一组已知类别和特征的数据。然后，我们使用`train_test_split`函数来划分训练集和测试集。接下来，我们使用`GaussianNB`类来训练朴素贝叶斯算法。最后，我们使用`predict`函数来对新的数据进行分类，并计算准确率。
