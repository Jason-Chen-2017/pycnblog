# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释MySQL在云端的应用。

## 4.1 创建数据库

首先，我们需要创建一个数据库。以下是创建数据库的SQL语句：

```sql
CREATE DATABASE mydb;
```

## 4.2 使用数据库

接下来，我们需要使用刚刚创建的数据库。以下是使用数据库的SQL语句：

```sql
USE mydb;
```

## 4.3 创建表

然后，我们需要创建一个表。以下是创建表的SQL语句：

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL
);
```

## 4.4 插入数据

接下来，我们需要插入一些数据。以下是插入数据的SQL语句：

```sql
INSERT INTO users (name, email) VALUES
    ('John Doe', 'john@example.com'),
    ('Jane Doe', 'jane@example.com');
```

## 4.5 查询数据

最后，我们需要查询数据。以下是查询数据的SQL语句：

```sql
SELECT * FROM users;
```
