# 4.2 智能营销中的具体代码实例
在智能营销中，可以使用协同过滤算法来推荐相似的产品。以下是一个简单的协同过滤算法的代码实例：

```python
from scipy.spatial.distance import cosine

# 计算用户之间的相似度
def calculate_similarity(user_a, user_b):
    similarity = 1 - cosine(user_a, user_b)
    return similarity

# 推荐相似的产品
def recommend_similar_products(user_id, products):
    similar_products = []
    for product in products:
        similarity = calculate_similarity(user_id, product)
        if similarity > 0.8:
            similar_products.append(product)
    return similar_products
```
