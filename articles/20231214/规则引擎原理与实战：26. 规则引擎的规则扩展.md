```python

    {"condition": "age < 60 and profession == 'doctor'", "action": "buy insurance"},
    {"condition": "age < 60 and profession != 'doctor'", "action": "no need to buy insurance"}
]

# 将新规则集合合并到原始规则集合中

```

现在，我们的规则引擎可以根据用户的年龄和职业来判断是否需要购买保险。
