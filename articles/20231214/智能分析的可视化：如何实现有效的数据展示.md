# 4.具体代码实例和详细解释说明

在这里，我们以一个简单的数据可视化例子为例，介绍如何使用Python的matplotlib库进行数据可视化：

```python
import matplotlib.pyplot as plt
import numpy as np

# 创建数据
x = np.linspace(0, 10, 100)
y = np.sin(x)

# 创建图形
plt.figure(figsize=(8, 6))

# 添加图形元素
plt.plot(x, y, label='sin(x)')
plt.xlabel('x')
plt.ylabel('sin(x)')
plt.title('A Simple Sine Wave')
plt.legend()

# 显示图形
plt.show()
```

在这个例子中，我们首先导入了matplotlib库，然后创建了一个简单的数据集。接着，我们使用plt.figure()函数创建了一个图形对象，并使用plt.plot()函数添加了一个图形元素。最后，我们使用plt.show()函数显示了图形。
