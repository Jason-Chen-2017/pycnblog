## 1.背景介绍

实时数据处理是指在数据产生的同时对数据进行处理，以便实时获取有关数据的信息。随着数据的大量生成和存储，实时数据处理变得越来越重要。Oracle NoSQL Database是一种高性能、可扩展的NoSQL数据库，可以用于实时数据处理。

Oracle NoSQL Database是一款基于键值对的数据库，可以存储大量数据并提供快速访问。它支持多种数据类型，包括字符串、数字、布尔值和日期时间。Oracle NoSQL Database还支持分布式数据存储，可以在多个节点之间分布数据，从而提高性能和可用性。

## 2.核心概念与联系

在Oracle NoSQL Database中，数据以键值对的形式存储。键是数据的唯一标识，值是存储的数据。键值对可以存储在单个节点上，也可以存储在多个节点上，以实现分布式数据存储。

Oracle NoSQL Database还支持数据的版本控制。每次对数据进行修改时，数据库会自动生成一个新的版本号。这样，可以在多个节点之间同步数据，确保数据的一致性。

Oracle NoSQL Database还支持数据的索引。索引可以用于快速查找数据，从而提高查询性能。索引可以基于键、值或其他属性进行创建。
