## 4.3 聚类算法实例

我们可以使用Python的Scikit-learn库来实现基于内容的聚类算法。以下是一个简单的例子：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.cluster import KMeans

# 加载数据
data = pd.read_csv('data.csv')

# 将文本数据转换为向量
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(data['text'])

# 执行聚类
kmeans = KMeans(n_clusters=5)
labels = kmeans.fit_predict(X)

# 输出聚类结果
for label, group in groupby(enumerate(labels), lambda i_j: i_j[0]):
    print("Cluster %d:" % label)
    for index in group:
        print(data['text'][index])
```

在这个例子中，我们首先使用TfidfVectorizer类来将文本数据转换为向量。然后，我们使用KMeans类来执行聚类。最后，我们输出聚类结果。
