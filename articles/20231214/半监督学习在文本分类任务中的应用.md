# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明半监督学习在文本分类任务中的应用。我们将使用Python的scikit-learn库来实现半监督学习模型。

首先，我们需要导入相关库：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.semi_supervised import LabelSpreading
from sklearn.metrics import accuracy_score, f1_score
```

然后，我们需要加载数据：

```python
data = [
    ("这是一个正例", "正例"),
    ("这是一个负例", "负例"),
    ("这是一个正例", "正例"),
    ("这是一个负例", "负例"),
    ("这是一个正例", "正例"),
    ("这是一个负例", "负例"),
]
```

接下来，我们需要对数据进行预处理：

```python
texts = [row[0] for row in data]
labels = [row[1] for row in data]
```

然后，我们需要对文本数据进行特征提取：

```python
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(texts)
```

接下来，我们需要对模型进行训练：

```python
spreading = LabelSpreading(contagion_function="prefer_majority")
spreading.fit(X, labels)
```

最后，我们需要对模型进行评估：

```python
y_pred = spreading.predict(X)
accuracy = accuracy_score(labels, y_pred)
f1 = f1_score(labels, y_pred, average="weighted")
print("准确率：", accuracy)
print("F1分数：", f1)
```

通过这个具体的代码实例，我们可以看到半监督学习在文本分类任务中的应用。
