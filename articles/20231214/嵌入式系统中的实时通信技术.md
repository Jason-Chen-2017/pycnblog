# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的实时通信示例来详细解释代码实现过程。

示例：实现一个简单的MODBUS协议通信示例

```c
#include <stdio.h>
#include <stdint.h>
#include <stdbool.h>
#include "modbus.h"

// 初始化MODBUS通信
void modbus_init(modbus_t *ctx, const char *port, uint16_t baud) {
    // 初始化MODBUS库
    modbus_new_rtu(ctx, port, baud);
    // 设置从站地址
    modbus_set_slave(ctx, 1);
}

// 读取寄存器值
uint16_t modbus_read_register(modbus_t *ctx, uint16_t addr, uint16_t count) {
    // 发送读取寄存器请求
    modbus_read_register(ctx, addr, count);
    // 等待响应
    modbus_read_response(ctx);
    // 获取响应数据
    uint16_t value = modbus_get_register(ctx, addr);
    // 返回寄存器值
    return value;
}

// 主函数
int main(void) {
    // 初始化MODBUS库
    modbus_t *ctx = modbus_new_rtu("COM1", 9600);
    // 设置从站地址
    modbus_set_slave(ctx, 1);

    // 读取寄存器值
    uint16_t value = modbus_read_register(ctx, 0x0001, 1);
    // 打印寄存器值
    printf("寄存器值：0x%04X\n", value);

    // 释放资源
    modbus_free(ctx);
    return 0;
}
```

在上述代码中，我们实现了一个简单的MODBUS协议通信示例。首先，我们初始化了MODBUS库并设置了从站地址。然后，我们通过调用`modbus_read_register`函数发送读取寄存器请求，并等待响应。最后，我们获取响应数据并打印寄存器值。
