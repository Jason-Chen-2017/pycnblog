# 3.1 图的表示
图可以用邻接矩阵、邻接表、半边表等多种方式来表示。邻接矩阵是一种稀疏矩阵，每一行每一列的元素都表示一个节点，如果两个节点之间有边，则对应的元素值为1，否则为0。邻接表是一种稠密矩阵，每个节点对应一个链表，链表中存储了与该节点相连的所有节点。半边表是一种稀疏矩阵，只存储每个节点的一半边，另一半边需要通过反向图来获取。

# 3.2 图算法
图算法是用于处理图数据的算法，它们可以用于解决许多复杂问题，例如最短路径、最小生成树、连通分量等。图算法的核心思想是利用图的特性，例如图的顶点和边的数量、图的连通性等。

# 3.3 图模型

# 4.2 图的遍历
我们可以使用图的遍历算法来访问图中的所有节点和边。图的遍历算法包括深度优先搜索（DFS）、广度优先搜索（BFS）等。我们可以使用Python的NetworkX库来实现图的遍历。

```python
# 深度优先搜索
def dfs(G, root):
    visited = set()
    stack = [root]
    while stack:
        node = stack.pop()
        if node not in visited:
            visited.add(node)
            stack.extend(G.neighbors(node))
    return visited

# 广度优先搜索
def bfs(G, root):
    visited = set()
    queue = [root]
    while queue:
        node = queue.pop(0)
        if node not in visited:
            visited.add(node)
            queue.extend(G.neighbors(node))
    return visited
```
