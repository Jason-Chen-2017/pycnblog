```python
import numpy as np
import matplotlib.pyplot as plt

# 生成高维数据
np.random.seed(42)
n_samples = 1000
n_features = 100
X = np.random.randn(n_samples, n_features)

# 学习网络中的概率分布
mu = np.mean(X, axis=0)
K = np.cov(X.T)
Sigma = np.linalg.eig(K)[0]

# 将高维数据降维到低维空间
Y = np.dot(np.sqrt(np.diag(Sigma)), (X - mu))

# 可视化结果
plt.scatter(Y[:, 0], Y[:, 1])
plt.xlabel('Principal Component 1')
plt.ylabel('Principal Component 2')
plt.show()
```

在这个代码实例中，我们首先生成了一组高维数据。然后，我们计算了数据的均值向量和协方差矩阵。接下来，我们计算了协方差矩阵的特征值，并将其用于降维。最后，我们可视化了降维后的数据。
