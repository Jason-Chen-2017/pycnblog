# 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的代码实例，用于说明如何实现医疗资源的分配策略。

```python
import numpy as np

# 假设我们有一个医疗资源的需求矩阵需求矩阵demand，其中demand[i, j] 表示医院i需要医生j的数量
demand = np.array([[100, 200, 300],
                   [150, 250, 350],
                   [200, 300, 400]])

# 假设我们有一个医疗资源的供给矩阵supply，其中supply[i, j] 表示医院i可以提供医生j的数量
supply = np.array([[100, 150, 200],
                   [150, 200, 250],
                   [200, 250, 300]])

# 计算每个医院的需求和供应量
demand_sum = np.sum(demand, axis=1)
supply_sum = np.sum(supply, axis=1)

# 计算每个医生的需求和供应量
demand_mean = np.mean(demand, axis=0)
supply_mean = np.mean(supply, axis=0)

# 根据需求和供应量设置分配比例
allocation_ratio = np.minimum(demand_mean / supply_mean, 1)

# 计算每个医院的分配数量
allocation = np.dot(demand, allocation_ratio)

# 输出分配结果
print(allocation)
```
