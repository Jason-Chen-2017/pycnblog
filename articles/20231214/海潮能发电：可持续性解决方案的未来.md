# 2.核心概念与联系
海潮能发电是一种利用海洋潮汐变化来生成电力的可持续能源技术。它通过将海洋水流通过特殊设备来生成电力，从而实现能源的可持续性和环保。海潮能发电技术的核心概念包括潮汐能量、海潮变化、发电设备等。

潮汐能量是海潮能发电技术的基础，它是由海洋水流产生的力量。潮汐能量是由地球、太阳和月球的引力作用所产生的，它会导致海洋水的波动。海潮变化是潮汐能量的表现形式，它是海洋水的升降过程。

发电设备是海潮能发电技术的核心部分，它们通过将海洋水流通过特殊设备来生成电力。常见的发电设备有波能发电机、潮汐发电机等。
