## 4.1 代码实例
```python
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 创建量子电路
qc = QuantumCircuit(2)

# 初始化量子态
qc.h(0)
qc.h(1)

# 应用量子运算符
qc.cx(0, 1)

# 测量量子态
qc.measure([0, 1], [0, 1])

# 执行量子电路
backend = Aer.get_backend('qasm_simulator')
job = backend.run(qc)
result = job.result()

# 绘制测量结果的概率分布
plot_histogram(result.get_counts())
```

## 4.2 详细解释说明
1. 首先，我们导入了`qiskit`库，并从中导入了`QuantumCircuit`、`Aer`、`transpile`、`assemble`和`plot_histogram`等模块。
2. 然后，我们创建了一个量子电路，并指定其包含2个量子位。
3. 接下来，我们初始化量子态，即将量子位设置为某种特定的状态。在这个例子中，我们使用Hadamard门（h门）来初始化量子态。
4. 然后，我们应用量子运算符，即将Hadamard门应用于第0个量子位，并将CNOT门（cx门）应用于第0个量子位和第1个量子位。
5. 最后，我们测量量子态，并将测量结果绘制为概率分布。
