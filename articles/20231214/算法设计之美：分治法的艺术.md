## 4.1 代码实例：快速排序

快速排序是一种基于分治法的排序算法，它的核心思想是将一个数组划分为两个部分，一个部分的数值小于基准值，另一个部分的数值大于基准值。然后递归地对这两个部分进行排序，最后将排序后的两个部分组合成原数组的排序结果。

以下是快速排序的Python代码实现：

```python
def quick_sort(arr, left, right):
    if left < right:
        # 选择基准值
        pivot = arr[left]
        # 将小于基准值的数值放在左边，大于基准值的数值放在右边
        i = left
        j = right
        while i < j:
            while i < j and arr[i] <= pivot:
                i += 1
            while j > i and arr[j] > pivot:
                j -= 1
            if i < j:
                arr[i], arr[j] = arr[j], arr[i]
        # 对左右两个部分进行递归排序
        quick_sort(arr, left, j - 1)
        quick_sort(arr, j + 1, right)
    return arr
```

## 4.2 代码解释

1. 首先，我们定义了一个`quick_sort`函数，它接受一个数组`arr`、左边界`left`和右边界`right`作为参数。
2. 在函数内部，我们首先判断左边界是否小于右边界。如果是，则继续执行排序操作，否则返回排序后的数组。
3. 接下来，我们选择一个基准值，通常是数组的第一个元素。
4. 我们使用两个指针`i`和`j`来分别表示小于基准值的数值和大于基准值的数值的位置。
5. 我们遍历数组，将小于基准值的数值放在左边，大于基准值的数值放在右边。
6. 当`i`和`j`相遇时，我们将基准值放在相应的位置，并递归地对左右两个部分进行排序。
7. 最后，我们返回排序后的数组。

Q：分治法适用于哪些类型的问题？
