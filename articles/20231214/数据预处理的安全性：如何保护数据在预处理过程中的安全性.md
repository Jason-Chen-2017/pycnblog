# 2.核心概念与联系

数据预处理安全性主要包括数据的完整性、可用性和隐私性。数据的完整性是指数据在预处理过程中是否被篡改或损坏；数据的可用性是指数据在预处理过程中是否可以被正确地访问和使用；数据的隐私性是指数据在预处理过程中是否被泄露或滥用。
