## 4.具体代码实例和详细解释说明

以下是一个简单的游戏示例，演示了如何使用Pygame库进行游戏开发。

```python
import pygame
import sys

# 初始化游戏
pygame.init()

# 设置游戏窗口大小
screen = pygame.display.set_mode((800, 600))

# 设置游戏标题
pygame.display.set_caption("Pygame Game")

# 设置游戏对象
player = pygame.Rect(100, 100, 50, 50)

# 设置游戏循环
running = True
while running:
    # 处理事件
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    # 更新游戏对象
    keys = pygame.key.get_pressed()
    if keys[pygame.K_LEFT]:
        player.x -= 5
    if keys[pygame.K_RIGHT]:
        player.x += 5
    if keys[pygame.K_UP]:
        player.y -= 5
    if keys[pygame.K_DOWN]:
        player.y += 5

    # 绘制游戏对象
    screen.fill((0, 0, 0))
    pygame.draw.rect(screen, (255, 0, 0), player)
    pygame.display.flip()

# 结束游戏
pygame.quit()
sys.exit()
```

这个示例中，我们首先初始化Pygame库，然后设置游戏窗口的大小和标题。接着，我们创建一个游戏对象player，并设置其位置和大小。最后，我们进入游戏循环，处理用户输入事件，更新游戏对象的位置，并绘制游戏对象在屏幕上的位置。
