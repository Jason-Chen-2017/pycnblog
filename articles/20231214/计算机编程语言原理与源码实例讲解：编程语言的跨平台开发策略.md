## 1.2 核心概念与联系

在计算机编程语言的跨平台开发策略中，有几个核心概念需要我们关注：

1. 平台：平台是指计算机硬件和操作系统的组合。例如，Windows、Linux、Mac OS X 等都是不同的平台。
2. 编程语言：编程语言是用来编写程序的语言。例如，C、C++、Java、Python 等都是不同的编程语言。
3. 跨平台开发：跨平台开发是指在不同平台上编写和运行程序的过程。

这些概念之间存在着密切的联系。编程语言是用来编写程序的，而程序需要在某个平台上运行。因此，为了实现跨平台开发，需要在不同平台上运行的程序能够兼容不同的硬件和操作系统。

```python

import sys

    # 获取当前平台信息
    platform = sys.platform
    print("当前平台：", platform)

    # 获取当前操作系统信息
    os_name = os.name
    print("当前操作系统：", os_name)

    # 执行程序逻辑
    # ...

if __name__ == "__main__":
    main()
```

详细解释说明：

在这个代码实例中，我们使用 Python 语言编写了一个跨平台程序。首先，我们导入了 `os` 和 `sys` 模块，这两个模块提供了用于获取平台和操作系统信息的方法。然后，我们定义了一个 `main` 函数，在这个函数中我们使用 `sys.platform` 方法获取当前平台信息，使用 `os.name` 方法获取当前操作系统信息。最后，我们执行程序逻辑。
