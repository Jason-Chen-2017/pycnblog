## 4.1 异步编程的代码实例
```python
import asyncio

async def main():
    await asyncio.sleep(1)
    print("Hello, World!")

asyncio.run(main())
```
在这个代码实例中，我们使用`asyncio`库来实现异步编程。`asyncio`库提供了一系列的异步操作和事件循环，以便我们可以轻松地编写异步的程序。在这个例子中，我们定义了一个`async`函数`main`，它使用`await`关键字来等待1秒钟，然后打印“Hello, World!”。通过使用`asyncio.run`函数，我们可以运行这个异步函数。

## 4.2 并发的代码实例
```python
import threading

def print_numbers():
    for i in range(10):
        print(i)

def print_letters():
    for letter in "abcdefghij":
        print(letter)

thread1 = threading.Thread(target=print_numbers)
thread2 = threading.Thread(target=print_letters)

thread1.start()
thread2.start()

thread1.join()
thread2.join()
```
在这个代码实例中，我们使用`threading`库来实现并发。`threading`库提供了一系列的线程操作，以便我们可以轻松地编写多线程的程序。在这个例子中，我们定义了两个线程函数`print_numbers`和`print_letters`，它们分别打印数字和字母。然后我们创建了两个线程对象`thread1`和`thread2`，并分别将它们的目标函数设置为`print_numbers`和`print_letters`。最后，我们使用`start`方法启动线程，并使用`join`方法等待线程完成。
