```python
class Counter:
    def __init__(self, start):
        self.count = start

    def increment(self):
        self.count += 1

    def decrement(self):
        self.count -= 1

    def get_count(self):
        return self.count

counter = Counter(0)

for _ in range(10):
    counter.increment()

print(counter.get_count())  # Output: 10

for _ in range(5):
    counter.decrement()

print(counter.get_count())  # Output: 5
```

3. 我们创建了一个`Counter`对象，并初始化其`count`属性为0。

4. 我们使用一个`for`循环来增加计数器的值10次。

5. 我们使用另一个`for`循环来减少计数器的值5次。
