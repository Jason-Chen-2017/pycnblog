# 2. 核心概念与联系

在多目标决策中，我们需要同时考虑多个目标，并在这些目标之间进行权衡。这些目标可能是互相矛盾的，因此需要在满足所有目标的同时，找到一个最佳的权衡点。

## 2.1 决策目标

决策目标是决策过程中需要考虑的各种目标或需求。这些目标可以是数值型的，如最小化成本、最大化利润等；也可以是非数值型的，如满足某些质量要求、满足某些安全要求等。

## 2.2 决策选项

决策选项是决策者可以选择的各种可能性。这些选项可以是有限的，也可以是无限的。在多目标决策中，决策选项需要满足所有决策目标，并在这些目标之间进行权衡。

## 2.3 决策权衡

决策权衡是在多个目标之间进行权衡的过程。这个过程需要考虑所有决策目标，并在这些目标之间找到一个最佳的权衡点。这个权衡点需要满足所有决策目标，并在满足这些目标的同时，最大限度地满足决策者的需求。
