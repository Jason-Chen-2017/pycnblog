DBSCAN算法的具体代码实例如下：

```python
import numpy as np
from sklearn.cluster import DBSCAN

# 创建一个随机数据集
X = np.random.rand(100, 2)

# 创建一个DBSCAN对象
dbscan = DBSCAN(eps=0.3, min_samples=5)

# 使用DBSCAN对数据集进行聚类
clusters = dbscan.fit_predict(X)

# 打印聚类结果
print(clusters)
```
