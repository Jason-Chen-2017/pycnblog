数据中台是一种新兴的数据技术架构，它旨在构建一个统一的数据平台，以满足企业内部和外部的数据需求。数据中台的核心是提供一种标准化的数据接口，使得不同的系统和应用程序可以通过这些接口来访问和操作数据。

数据中台的发展背景主要包括以下几点：

1. 数据化经济的兴起：随着数据的产生和收集量不断增加，企业对于数据的利用和分析变得越来越重要。数据中台可以帮助企业更好地管理和利用数据资源。

2. 数据平台的普及：随着大数据技术的发展，越来越多的企业开始建立数据平台，以满足各种数据需求。数据中台可以作为数据平台的一部分，提供统一的数据接口和管理功能。

3. 数据服务化：随着微服务和服务化技术的普及，企业越来越关注数据服务化的概念。数据中台可以帮助企业实现数据服务化，提高数据的可用性和灵活性。

4. 数据安全和隐私：随着数据的产生和传输量不断增加，数据安全和隐私问题也变得越来越重要。数据中台可以提供一种标准化的数据接口，帮助企业更好地管理和保护数据。

```python
from flask import Flask, jsonify

@app.route('/users', methods=['GET'])
def get_users():
    users = [
        {'id': 1, 'name': 'John', 'age': 20},
        {'id': 2, 'name': 'Alice', 'age': 25},
        {'id': 3, 'name': 'Bob', 'age': 30}
    ]
    return jsonify(users)

if __name__ == '__main__':
    app.run()
```
