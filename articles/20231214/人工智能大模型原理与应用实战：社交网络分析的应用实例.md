# 4.具体代码实例和详细解释说明

## 4.1 代码实例
我们可以使用 Python 的 NetworkX 库来实现社交网络分析。以下是一个简单的代码实例：

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建一个有向图
G = nx.DiGraph()

# 添加节点
G.add_node("A")
G.add_node("B")
G.add_node("C")

# 添加边
G.add_edge("A", "B")
G.add_edge("B", "C")

# 绘制图
nx.draw(G, with_labels=True)
plt.show()
```

## 4.2 解释说明
这个代码实例创建了一个有向图，包括三个节点和两条边。我们可以使用 NetworkX 库的各种方法来进行图的遍历、连通性检查、最短路径等计算。
