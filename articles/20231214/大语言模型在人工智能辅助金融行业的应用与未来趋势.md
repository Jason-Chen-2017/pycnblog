## 4.1 代码实例1：文本清洗

```python
import re

def clean_text(text):
    # 删除非字母数字字符
    text = re.sub(r'[^\w\s]', '', text)
    # 转换大写
    text = text.upper()
    # 删除重复空格
    text = re.sub(r'\s+', ' ', text)
    return text
```

## 4.2 代码实例2：文本分词

```python
import jieba

def segment_text(text):
    # 分词
    words = jieba.cut(text)
    # 转换为列表
    words = list(words)
    return words
```

## 4.3 代码实例3：文本编码

```python
import numpy as np

def encode_text(words):
    # 词汇表
    vocab = set(words)
    # 词汇表编码
    vocab_to_int = {word: index for index, word in enumerate(sorted(vocab))}
    # 文本编码
    encoded_text = [vocab_to_int[word] for word in words]
    return encoded_text
```
