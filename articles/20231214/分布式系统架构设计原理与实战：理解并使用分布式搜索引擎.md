
- 2.将查询服务分布在多个计算节点上：

  ```python
  def distribute_query(query_service, nodes):
      for node in nodes:
          query_service.assign_to_node(node)
  ```

- 3.将排序服务分布在多个计算节点上：

  ```python
  def distribute_sort(sort_service, nodes):
      for node in nodes:
          sort_service.assign_to_node(node)
  ```
