## 4.1 创建数据表

首先，我们需要创建一个数据表，以便在进行数据可视化和报表生成时使用。以下是一个创建数据表的示例：

```sql
CREATE TABLE sales (
    date DATE,
    product VARCHAR(100),
    region VARCHAR(100),
    sales_amount DECIMAL(10,2)
);
```

在这个例子中，我们创建了一个名为sales的表，它有四个列：date、product、region和sales_amount。

## 4.2 插入数据

接下来，我们需要插入一些数据到表中。以下是一个插入数据的示例：

```sql
INSERT INTO sales VALUES
('2021-01-01', 'Product A', 'Region A', 100.00),
('2021-01-02', 'Product A', 'Region A', 150.00),
('2021-01-03', 'Product A', 'Region A', 200.00),
('2021-01-01', 'Product B', 'Region B', 200.00),
('2021-01-02', 'Product B', 'Region B', 250.00),
('2021-01-03', 'Product B', 'Region B', 300.00);
```

在这个例子中，我们插入了三天的销售数据，包括两个产品和两个地区。

## 4.3 查询数据

接下来，我们可以使用SQL查询语句来查询数据。以下是一个查询数据的示例：

```sql
SELECT date, product, region, SUM(sales_amount) AS total_sales
FROM sales
GROUP BY date, product, region
ORDER BY total_sales DESC;
```

在这个例子中，我们查询了销售数据的总额，并按照总额进行降序排序。
