# 2.核心概念与联系

## 2.1数据仓库的基本概念

数据仓库是一种用于存储和管理大量历史数据的系统，它通常用于支持企业的决策分析和业务智能应用。数据仓库的核心组成部分包括：

- **数据存储**：数据仓库使用关系型数据库或者非关系型数据库来存储数据。
- **数据索引**：数据仓库使用索引来加速数据查询。
- **数据查询**：数据仓库提供查询接口，用户可以通过这些接口来查询数据。
- **数据分析**：数据仓库提供分析工具，用户可以通过这些工具来分析数据。
