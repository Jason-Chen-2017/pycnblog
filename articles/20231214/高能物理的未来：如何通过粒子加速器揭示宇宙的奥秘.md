                 

# 1.背景介绍

高能物理是一门研究原子核和子atomic nucleus和子粒子的科学。高能物理研究人员通过碰撞高能的粒子来研究原子核和子粒子的性质和行为。这些碰撞通常发生在粒子加速器中，如大型碰撞器。

粒子加速器是高能物理实验的核心设备，它们通过加速粒子到高速来研究其性质和行为。粒子加速器可以分为两类：同步加速器和循环加速器。同步加速器通过同步电磁场来加速粒子，而循环加速器通过磁场循环来加速粒子。
