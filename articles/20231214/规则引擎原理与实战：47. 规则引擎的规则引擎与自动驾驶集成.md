# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的自动驾驶系统来展示规则引擎的具体代码实例和详细解释说明。

```python
# 定义规则
rules = [
    {"condition": "speed > 120", "action": "sound_alarm"},
    {"condition": "speed > 100", "action": "flash_lights"},
    {"condition": "speed > 80", "action": "display_warning"},
]

# 获取当前速度
current_speed = 110

# 匹配规则
matched_rules = []
for rule in rules:
    if eval(rule["condition"]):
        matched_rules.append(rule)

# 激活规则
activated_rule = max(matched_rules, key=lambda x: x["action"])

# 执行规则
if activated_rule["action"] == "sound_alarm":
    print("发出警报")
elif activated_rule["action"] == "flash_lights":
    print("闪烁灯光")
elif activated_rule["action"] == "display_warning":
    print("显示警告")
```

在上述代码中，我们首先定义了一组规则，每个规则包括条件部分（前提）和动作部分（结果）。然后，我们获取当前的速度并匹配规则，将满足条件的规则加入到匹配规则集合中。接着，我们根据匹配规则集合来选择最佳的规则，并执行最佳规则的动作部分（结果）来实现自动驾驶系统的决策。最后，我们输出自动驾驶系统的决策结果。
