# 2.核心概念与联系

## 2.1 VoltDB概述
VoltDB是一种高性能的分布式关系数据库管理系统，它基于新的ACID-compliant SQL语言。VoltDB可以处理大量的数据并提供低延迟的查询性能。它的核心特点包括：

- 分布式：VoltDB可以在多个节点上运行，从而实现数据的分布式存储和处理。
- 高性能：VoltDB采用了一种称为“分布式事务处理”的技术，可以提高数据处理的速度。
- ACID：VoltDB遵循ACID规范，确保数据的一致性、完整性和隔离性。
