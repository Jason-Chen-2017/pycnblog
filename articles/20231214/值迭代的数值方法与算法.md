## 4.1 代码实例

我们以一个简单的连续型优化问题为例，来演示值迭代算法的实现过程。

问题描述：

最小化连续函数$$f(x) = x^2 + 2x + 1$$，其中$$x \in [0, 1]$$。

解决方案：

1. 将连续型问题的状态空间离散化，将连续型状态转换为离散型状态。
2. 将连续型目标函数转换为离散型目标函数。
3. 对离散型问题进行迭代求解，直到满足某个终止条件。

具体的代码实现如下：

```python
import numpy as np

# 定义连续型目标函数
def f(x):
    return x**2 + 2*x + 1

# 离散化连续型目标函数
def f_discrete(x):
    return f(x)

# 初始化问题的状态值
V = np.zeros(100)

# 迭代求解
for i in range(1000):
    for j in range(100):
        V[j] = f_discrete(j)

# 输出结果
print(V)
```

3. 初始化问题的状态值，并对每个状态进行迭代求解。
