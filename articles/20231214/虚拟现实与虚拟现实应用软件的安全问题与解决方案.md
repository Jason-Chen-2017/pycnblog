```python
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad
from Crypto.Random import get_random_bytes

def encrypt(data, key):
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(pad(data, AES.block_size))
    return cipher.nonce, ciphertext, tag

def decrypt(nonce, ciphertext, tag, key):
    cipher = AES.new(key, AES.MODE_EAX, nonce=nonce)
    return unpad(cipher.decrypt(ciphertext), AES.block_size)
```

在这个代码实例中，我们使用了AES算法进行数据加密和解密操作。我们首先创建了一个AES加密器，并使用AES.MODE_EAX模式进行加密。然后，我们使用`encrypt`函数进行数据加密操作，并返回加密后的非对称密钥、密文和标签。最后，我们使用`decrypt`函数进行数据解密操作，并返回解密后的原始数据。
