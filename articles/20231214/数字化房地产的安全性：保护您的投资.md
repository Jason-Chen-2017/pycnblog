# 2.核心概念与联系

1.数字化房地产：数字化房地产是指通过数字化技术将房地产信息存储、传输、交易等各种操作进行数字化处理的房地产。数字化房地产的出现，为房地产市场提供了更加便捷、高效、安全、透明的交易环境，为房地产市场的发展创造了更多的发展空间。

4.信息安全性：信息安全性是指数字化房地产的信息传播需要保障信息的安全性，以防止信息被篡改、泄露等。
