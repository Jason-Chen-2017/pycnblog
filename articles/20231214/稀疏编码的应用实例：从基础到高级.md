# 2.核心概念与联系

## 2.1 稀疏数据

稀疏数据是指数据中大部分元素为零或者缺失的数据。例如，一个2000行2000列的矩阵，如果只有100个非零元素，那么这个矩阵就是稀疏的。稀疏数据的特点是数据稀疏性很高，非零元素之间存在较大的空洞。

## 2.2 稀疏矩阵

稀疏矩阵是一种用于存储稀疏数据的数据结构，它主要由三个部分组成：非零元素的值、非零元素的行索引和非零元素的列索引。稀疏矩阵可以有效地减少存储空间和计算复杂度，因此在处理稀疏数据时得到了广泛应用。

## 2.3 稀疏编码

稀疏编码是一种用于将稀疏数据转换为稀疏矩阵的编码方法。它主要包括以下几个步骤：

1. 遍历稀疏数据，找出非零元素。
2. 记录非零元素的值、行索引和列索引。
3. 将非零元素存储到稀疏矩阵中。
