### 4.2环境保护

```python
def emission(pollutant, time):
    emission = pollutant / time
    return emission

pollutant = 100
time = 10
emission = emission(pollutant, time)
print("污染物排放:", emission)

def ecosystem(biodiversity, disturbance):
    ecosystem = biodiversity / disturbance
    return ecosystem

biodiversity = 100
disturbance = 5
ecosystem = ecosystem(biodiversity, disturbance)
print("生态环境保护:", ecosystem)
```
