## 4.1 代码实例：强关联物理系统的数学模型建立
```python
import numpy as np
from scipy.optimize import minimize

def wave_function(x):
    return np.exp(-x**2)

def interaction_energy(x):
    return -wave_function(x)**2

def total_energy(x):
    return np.sum(interaction_energy(x))

x0 = np.array([0.0, 0.0, 0.0])
result = minimize(total_energy, x0)
x_opt = result.x
```
