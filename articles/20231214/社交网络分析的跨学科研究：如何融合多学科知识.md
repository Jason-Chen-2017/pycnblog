# 2.核心概念与联系
节点：社交网络中的基本单位，表示人、组织或其他实体。
边：节点之间的连接，表示关系或联系。
社会网络：由节点和边组成的网络结构，用于表示人与人之间的关系。
社会网络的性质：社会网络具有小世界性、集中性、环形性、模块性等特征。
社会网络的度量：包括节点度、边度、聚类系数、路径长度等指标，用于评估社会网络的性质。
社会网络的可视化：通过图形化的方式展示社会网络的结构和特征，以便更好地理解和分析。

流量中心性：节点的流量值越高，表示节点在社会网络中的重要性越高。

最短路径：节点之间的最短路径，用于评估节点之间的距离。
聚类：社会网络中的子网络，用于评估社会网络的结构。
