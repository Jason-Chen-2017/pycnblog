在本节中，我们将通过一个简单的规则引擎示例来详细解释规则引擎的具体代码实例和详细解释说明。

## 4.1 规则定义

在这个示例中，我们将定义一个简单的规则，用于判断用户是否满足购买条件。规则的定义如下：

```python
rule "判断用户是否满足购买条件"
when
    $age < 18
then
    $result = "用户不满足购买条件"
end
```

在这个规则中，我们使用了一个条件判断语句 `$age < 18` 来判断用户的年龄是否小于18岁。如果条件判断为真，则规则会执行并生成决策结果 `$result = "用户不满足购买条件"`。

## 4.2 事实定义

在这个示例中，我们将定义一个事实，用于表示用户的年龄。事实的定义如下：

```python
fact "用户年龄"
    age 20
end
```

在这个事实中，我们使用了一个事实属性 `age` 来表示用户的年龄，并将其设置为20岁。

## 4.3 规则引擎启动

在这个示例中，我们将启动规则引擎并初始化规则引擎的状态。规则引擎启动的代码如下：

```python
from jython_rules import RuleEngine

engine = RuleEngine()
engine.start()
```

在这个代码中，我们使用了 `jython_rules` 库来启动规则引擎。我们创建了一个 `RuleEngine` 对象，并调用其 `start` 方法来启动规则引擎。

## 4.4 事实输入

在这个示例中，我们将输入事实到规则引擎。事实输入的代码如下：

```python
from jython_rules import Fact

fact = Fact("用户年龄")
fact.set("age", 20)
engine.register(fact)
```

在这个代码中，我们使用了 `jython_rules` 库来输入事实。我们创建了一个 `Fact` 对象，并使用其 `set` 方法来设置事实属性 `age` 的值为20。最后，我们使用 `engine.register` 方法来注册事实到规则引擎。

## 4.5 规则执行

在这个示例中，我们将执行规则并生成决策结果。规则执行的代码如下：

```python
from jython_rules import RuleExecutionResult

result = engine.execute("判断用户是否满足购买条件")
print(result.get("result"))
```

在这个代码中，我们使用了 `jython_rules` 库来执行规则。我们使用 `engine.execute` 方法来执行规则 "判断用户是否满足购买条件"，并将其结果存储到 `RuleExecutionResult` 对象中。最后，我们使用 `result.get` 方法来获取决策结果的值，并打印到控制台。
