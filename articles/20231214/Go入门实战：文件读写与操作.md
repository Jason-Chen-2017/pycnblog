# 4.具体代码实例和详细解释说明
在Go语言中，文件读写操作的具体代码实例如下：

```go
package main

import (
    "fmt"
    "io"
    "os"
)

func main() {
    // 打开文件
    file, err := os.Open("example.txt")
    if err != nil {
        fmt.Println("Error opening file:", err)
        return
    }
    defer file.Close()

    // 读取文件内容
    reader := bufio.NewReader(file)
    content, _ := reader.ReadString('\n')
    fmt.Println(content)

    // 写入文件内容
    writer := bufio.NewWriter(file)
    _, err = writer.WriteString("Hello, World!\n")
    if err != nil {
        fmt.Println("Error writing to file:", err)
        return
    }
    writer.Flush()
}
```

在上述代码中，我们首先使用`os.Open`函数打开文件`example.txt`，并将其存储在`file`变量中。然后，我们使用`bufio`包来创建一个缓冲读取器和缓冲写入器，以便更方便地读取和写入文件内容。最后，我们使用`reader.ReadString`方法读取文件内容，并使用`writer.WriteString`方法写入文件内容。
