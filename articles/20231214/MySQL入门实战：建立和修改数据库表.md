CREATE TABLE 用户 (

    PRIMARY KEY (用户名),
    FOREIGN KEY (用户名) REFERENCES 部门(部门名称),
    INDEX 用户名索引 (用户名),
    ...

```

在这个例子中，我们建立了一个用户表，表中包含用户名、年龄等列。用户名是一个字符型列，年龄是一个整型列。用户名为主键约束，同时用户名为外键约束，表示用户名必须在部门表的部门名称列中存在。用户名列的数据会被存储在用户名索引中，以加速用户名的查询操作。
