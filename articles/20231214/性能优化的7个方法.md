## 2.1 性能度量

性能度量是衡量系统性能的指标。常见的性能度量包括：

- 时间复杂度：表示算法的执行时间与输入大小之间的关系。
- 空间复杂度：表示算法的内存占用与输入大小之间的关系。
- 吞吐量：表示单位时间内处理的任务数量。
- 延迟：表示从请求发送到响应返回的时间。

算法优化是通过改变算法的实现方式或选择更高效的算法来提高性能的方法。算法优化可以包括：

## 2.4 编译器优化

编译器优化是通过编译器对程序代码进行优化来提高性能的方法。编译器优化可以包括：

## 2.5 硬件优化

- 处理器优化：通过选择更高效的处理器来提高性能。

def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[0]
        less = [x for x in arr[1:] if x <= pivot]
        greater = [x for x in arr[1:] if x > pivot]
        return quick_sort(less) + [pivot] + quick_sort(greater)

arr = [5, 2, 8, 3, 1]
print(quick_sort(arr))
