## 4.2 社会动态模型的具体代码实例

社会动态模型的具体代码实例如下：

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义差分方程
def dydt(y, t):
    return y * t

# 初始条件
y0 = 1

# 时间步长
dt = 0.01

# 时间范围
t0 = 0
t1 = 10

# 解差分方程
t = np.arange(t0, t1, dt)
y = np.vectorize(dydt)(y0, t)

# 绘制图
plt.plot(t, y)
plt.xlabel('时间')
plt.ylabel('社会动态')
plt.show()
```

在这个代码实例中，我们使用了numpy库来实现社会动态模型。首先，我们定义了差分方程，然后设置了初始条件、时间步长和时间范围。接着，我们解了差分方程，并绘制了图。
