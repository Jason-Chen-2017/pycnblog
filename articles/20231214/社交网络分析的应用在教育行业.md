# 2.核心概念与联系

在本节中，我们将介绍社交网络分析中的一些核心概念，并讨论它们与教育行业相关的联系。

## 2.1 社交网络

社交网络是一种由人们之间的关系组成的网络，它可以用图论的方法来描述。在社交网络中，节点表示人，边表示人之间的关系。例如，在教育行业中，学生可以被视为节点，他们之间的交流、学习、合作等关系可以被视为边。

## 2.2 社交网络的度

度是指一个节点的邻接节点数量。在教育行业中，学生的度可以用来衡量他们与其他学生的关系密切程度。例如，一个学生与其他学生的交流频繁，则其度较高；而一个学生与其他学生的交流较少，则其度较低。

## 2.3 社交网络的中心性

中心性是指一个节点在网络中的重要性。在教育行业中，学生的中心性可以用来衡量他们在网络中的影响力和权重。例如，一个学生与其他学生的关系密切，则其中心性较高；而一个学生与其他学生的关系较少，则其中心性较低。
