### 4.1 正则表达式的匹配示例

以下是一个正则表达式的匹配示例：

```python
import re

# 定义一个正则表达式
pattern = r'\d{3}-\d{2}-\d{4}'

# 定义一个文本
text = '123-45-6789'

# 使用正则表达式匹配文本
match = re.match(pattern, text)

# 判断是否匹配成功
if match:
    print('匹配成功')
else:
    print('匹配失败')
```

在这个示例中，我们使用Python的re模块来实现正则表达式的匹配功能。我们定义了一个正则表达式'\d{3}-\d{2}-\d{4}'，用于匹配一个三位数字、中间有两个连字符、再后面是四位数字的字符串。然后我们定义了一个文本'123-45-6789'，并使用re.match()方法来匹配文本。最后，我们判断是否匹配成功，并输出相应的结果。

### 4.2 正则表达式的替换示例

