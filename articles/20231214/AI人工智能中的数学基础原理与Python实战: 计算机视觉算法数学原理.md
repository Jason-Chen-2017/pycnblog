# 2.核心概念与联系

在计算机视觉中，我们需要处理的数据主要是图像，图像是一个二维的数字信息，可以用数组或矩阵来表示。图像的数值表示是由像素组成的，每个像素代表了图像中的一个点，它的值是由RGB三个通道组成的。因此，计算机视觉算法的核心概念主要包括：

- 图像处理：图像处理是对图像数据进行预处理、增强、去噪等操作，以提高图像质量或提取有用信息。
- 特征提取：特征提取是从图像中提取出有关的信息，以便进行后续的识别或分类等操作。
- 模式识别：模式识别是根据特征信息，将图像分为不同的类别或分组。

这三个概念之间存在着密切的联系，它们共同构成了计算机视觉算法的核心流程。
