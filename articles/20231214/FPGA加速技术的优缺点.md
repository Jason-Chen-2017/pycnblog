### 4.1代码实例
我们将实现一个简单的加法器，用于加速计算任务。代码实例如下：

```verilog
module Add(input wire Clk, input wire A, input wire B, output reg Sum);
  always @(posedge Clk) begin
    Sum <= A + B;
  end
endmodule
```

### 4.2代码解释
1. 定义一个模块 Add，输入端口 Clk、A、B，输出端口 Sum。
2. 在 always 块中，使用 posedge 触发器，当 Clk 发生上升沿时，执行计算。
3. 在计算块中，将 A 和 B 相加，并将结果赋值给 Sum。
