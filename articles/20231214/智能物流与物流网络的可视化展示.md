## 4.1 代码实例

```python
import matplotlib.pyplot as plt
import networkx as nx

# 建立物流网络的图模型
G = nx.Graph()
G.add_nodes_from([('A', {'pos': (0, 0)}), ('B', {'pos': (1, 0)}), ('C', {'pos': (2, 0)})])
G.add_edges_from([('A', 'B'), ('B', 'C')])

# 可视化展示物流网络
nx.draw(G, with_labels=True, node_size=3000, node_color='blue', node_shape='o', font_size=10)
plt.show()
```

## 4.2 代码解释

1. 首先，我们需要导入所需的库，如matplotlib和networkx。
2. 然后，我们可以通过networkx库来建立物流网络的图模型。在这个例子中，我们建立了一个包含3个节点和2个边的物流网络。
3. 接下来，我们可以通过nx.draw函数来可视化展示物流网络。在这个例子中，我们设置了节点的大小、颜色、形状等属性，以便更直观地展示物流网络的信息。
4. 最后，我们通过plt.show函数来显示可视化的物流网络图形。
