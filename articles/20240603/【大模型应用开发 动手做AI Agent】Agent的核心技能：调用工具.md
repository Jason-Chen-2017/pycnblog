## 项目实践：代码实例和详细解释说明

以下是我们为AI Agent添加调用工具的代码实例：

```python
import requests
from bs4 import BeautifulSoup

class WebCrawler:
    def __init__(self, url):
        self.url = url

    def crawl(self):
        response = requests.get(self.url)
        soup = BeautifulSoup(response.text, 'html.parser')
        return soup

def main():
    url = 'https://example.com'
    crawler = WebCrawler(url)
    soup = crawler.crawl()
    print(soup)

if __name__ == '__main__':
    main()
```

在这个例子中，我们为AI Agent添加了一个网络爬虫调用工具，用于从网页中提取信息。这个工具使用了`requests`和`BeautifulSoup`库，AI Agent可以使用这个工具完成任务。
