# 1.背景介绍


## 什么是网络教育？
网络教育（英语：Online education）是一个指通过互联网、网络平台或其他信息技术媒介为大学生提供免费或者收费的网络学习服务的活动。它可以使学生的学习成绩达到一个较高水平，也可促进学生间的交流合作，提升全社会的素质。而现在，越来越多的国家都设立了网络教育的政策和制度。美国、欧洲、日本、韩国、中国等发达国家在推行网络教育方面均取得成功。据统计，至今已有超过1亿名学生接受网络教育，并形成了良好的学习环境。根据研究报告显示，90%以上学生已经完成了四年级，甚至超过六年级。他们正在努力实现自己的职业规划。网络教育的形式多样且有效，其优点很多。但也存在着一些不足。其中最突出的就是各个网站的教学内容难以统一，导致学生学不到完整的知识体系；同时，由于大家都在网络上进行交流，很容易造成信息 overload，无法及时得到老师和同学的帮助；此外，受到智能手机、VR 头盔等新技术的影响，学生可能会适得其反。

## 为什么要开发并销售自己的在线课程？
随着数字化时代的到来，网络教育的发展也变得迅速。但随之而来的也不仅仅是教育领域的变化。越来越多的人开始将个人品牌作为一种营销手段来打动顾客，比如说梦想成为全栈工程师，拥有独特的个人风格，能带来更多的收益。因此，教育公司希望能够开发自己的网络教育课程来吸引更多人参与。另外，除了吸引人才外，还可以赚取收入。由于课程的价格低廉，不少家长愿意购买，降低了购买门槛。如果开发者能够提供高质量的课程，则能够吸引大量的顾客消费。因此，“开发并销售自己的在线课程”是这类企业的热门方向。


# 2.核心概念与联系
## 在线课程、网络课堂、网络教育
- 在线课程(online course):通过网络发布的、以视频、音频、文字等方式传授知识、技能和能力的学习活动。在线课程往往具有短期、零散的性质，内容包括必修科目、选修科目、自主学习、项目实训等类型。
- 网络课堂(webinar):由第三方组织举办的一场虚拟的、专门针对某些特定目标群体的学习会议。网络课堂通常比传统的学习班更加随心所欲、自主选择，有利于培养学生的独立解决问题能力、坚持学习的决心以及知识的自主性。
- 网络教育(online education):指的是利用电子网络、移动通讯设备及各种网络工具进行的网络化教育。它强调通过网络环境的虚拟化来实现真正意义上的网络教育，是对传统教育方式和网络课程的混合。

## 技术教育和非技术教育
- 技术教育(technology education)：主要针对计算机、IT、软件、网络及相关专业人员开展。它以提升工作技能、解决实际问题为主线，倡导学生在生活中应用技术解决实际问题，提高职业技能。
- 非技术教育(non-technology education)：主要指除计算机、IT、软件、网络以外的其他学科，如商学、艺术、语言、人文等。非技术教育的目标是培养学生的理论知识、观察能力、创新精神、交流沟通能力等软 skills。

## 传统教育与网络教育的比较
|  | 传统教育 | 网络教育 |
|-|--------|-------|
| 内容 | 正规的学校教学内容 | 自由设置的在线课程内容 |
| 方法 | 普通的学校教室授课 | 通过网络授课 |
| 范围 | 针对具体的学生进行教学 | 全局范围内的职业培训，跨界地扩展知识面 |
| 费用 | 中学及以下阶段学费收费，中专、高职院校要收取一定比例的退费补贴 | 课程收费可按时段收费，不收取任何学费，价格透明，用户自负 |
| 效果 | 学习效果较好，但缺乏自主性、个人化 | 个人的网络身份使学生有更多的自主权、个性化选择 |
| 用户 | 有限的学生群体 | 大众化的网民群体，来自不同的学历、经验层次，具有互动性、社交性 |
