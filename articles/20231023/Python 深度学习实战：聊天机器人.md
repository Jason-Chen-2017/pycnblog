# 1.背景介绍


## 概述
在智能机器人的领域中，最知名的是苹果公司的Siri、微软公司的Cortana以及谷歌公司的Google Assistant等。这些AI助手通过语音识别、自然语言理解、机器学习等技术可以做出各种各样的功能。最近几年，越来越多的开发者、公司和研究机构都开始涉足这个领域。我国的优越产业环境也促使了人工智能的迅速发展。但是缺少一个完整的教程和工具链，让初入门的程序员望而生畏。所以，我希望通过《Python 深度学习实战：聊天机器人》这一系列教程，让大家可以轻松地上手AI编程，实现自己的聊天机器人。

## 目标读者
本教程面向具有一定编程基础的人群。如熟悉基本语法，对机器学习、NLP等概念比较了解，但不一定要精通。希望通过阅读本教程，能够快速入门并进行扩展，提升技能水平。
