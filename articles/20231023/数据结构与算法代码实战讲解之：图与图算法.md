# 1.背景介绍


## 一、什么是图
在计算机科学中，图(Graph)是由节点（Node）和边（Edge）组成的数据结构。在图论中，图是一个非常重要的概念。图可以用来表示复杂系统中各种对象之间的关系，同时也可以用来描述空间中的地图、网络结构、城市交通网络等。

## 二、图的特点
1. 有向图或无向图
2. 网状结构或树形结构
3. 有权值或带权值
4. 多个图之间可能存在相互联系或联系较少的关系。

## 三、图的应用场景
- 拓扑排序、最短路径算法
- 最大流最小割问题
- 分布式计算中路由选择
- 电子商务领域的推荐引擎、社交网络分析
- 网页搜索、画像设计、电影推荐系统、推荐系统

## 四、图的抽象数据类型定义
### 1. 顶点集合V 和边集合E
图由一个非空的顶点集合V和一个邻接表形式的边集E表示。其中，每个顶点用一个唯一的顶点名称或标识符表示；每个边是由两个顶点组成的有序对（称作弧），每条边都有一个特定的标签或权重，代表了该边的某种性质。

### 2. 每个顶点v∈V都有一个关联的度d(v)，即与v相连的边的个数。
对于无向图来说，两个顶点之间只需要维护一条边就可以表示两者间的联系。因此，无向图的度分布情况可以在O(|V|+|E|)时间内确定，其中|V|是顶点数，|E|是边数。

### 3. 每条边e=(u,v),其中u∈V和v∈V，都有一个关联的权值w(e)。
如果是无向图，则一条边的方向并不影响其权值。然而，在有向图中，一条边的方向决定了其方向上的流量的方向。例如，如果从顶点u到顶点v的边上限值为f(u,v)，那么通过该边发送的一条流量会流向v，但不会流向u。因此，有向图的权值分布情况也应当是在O(|V|+|E|)时间内完成的。

### 4. 对于无向图，所有顶点对间的路径数量为Ω(k^n),k为图的大小。
由于每条边均有唯一的起点和终点，所以每个顶点的出度和入度之和相同。因此，无向图的路径数量和各顶点的度相关。对于一般的有向图，则需考虑方向性。对于一条边(u,v)，其所有可达的顶点分别记作v'。如果从u到v的逆线路上不存在任何其他的边，则称该路径为从u到v的简单回路。同样地，对于有向图，顶点u的后继顶点v'及v的前驱顶点u'则构成了一个有向环。这样，有向图的简单路径的数量为Ω(|V|^2)。

### 5. 对称与反对称图的比较
具有相同顶点数和边数的不同图是同构的，换句话说，具有相同顶点和边的不同图也是同构的。在同构的图中，顶点与顶点之间的联系是双向的，顶点与边的对应关系也是双向的。另一方面，反对称图的顶点之间的联系是单向的，顶点与边的对应关系也是单向的。对于同构或反对称图，相应的算法和复杂度都会存在差别。

- 带权重的图

- 带权值的无向图

- 网状图或树形图
