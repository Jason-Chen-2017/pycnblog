# 2.核心概念与联系
## 数据类型
数据类型是计算机编程语言用来定义和管理数据的一个重要概念。Java有八种基本数据类型：整数类型、浮点数类型、字符类型、布尔类型、引用类型、数组类型、void类型。其中整数类型分为两类——整型和长整型，分别占据两个字节和四个字节的存储空间。

## 基本数据类型
- 整型类型：byte、short、int、long
- 浮点类型：float、double
- 字符类型：char
- 布尔类型：boolean
- void类型：Java没有单独的void类型，任何函数都没有返回值时，函数声明的返回类型为void。

## 变量
变量是内存中用于存放数据的位置标识符，它是一个命名实体，用于存放数据的值。在Java中，可以通过关键字var来定义一个变量。例如：int age = 20;

## 变量分类
按照变量在程序中的作用范围可以把变量分成三类：

1. 局部变量（Local Variable）：是在某个方法或者代码块内定义的变量，该变量只能在这个方法或代码块内访问，当这个方法或代码块执行完毕后，变量就会消失。
2. 成员变量（Member Variable）：是定义在类中、且不用static修饰的变量，称为类的成员变量，可以被所有实例化对象共享，一般作为类的属性存在。例如：public int count=10；
3. 全局变量（Global Variable）：是定义在方法外部、且不用static修饰的变量，表示整个项目中的一个全局变量，所有的对象都可以直接访问。

## 常量
常量就是固定数值的变量，通常以全大写的形式表示，其数值不能改变。在java中，常量用final修饰符来定义。例如：final double PI=3.1415926。

## 标识符
标识符是用来区分变量、常量、函数、类名、包名、接口名、注解名等各种名称的特殊字符序列。在Java中，标识符由字母、数字、下划线和美元符号组成，但不能以数字开头。

## 赋值方式
在Java中，变量的赋值可以使用等号(=)或者用运算符来完成。例如：

int x = y + z //y和z都是已知值，通过计算得到x的值。
int a = b % c //b和c都是已知值，a的值等于b除以c的余数。

## 数据类型转换
数据类型转换即将一种数据类型转换成另一种数据类型。Java语言提供了以下几种数据类型转换的方式：

1. 自动类型转换：Java程序在运行期间，编译器会自动检测表达式是否需要类型转换，必要时会进行类型转换；
2. 强制类型转换：通过强制类型转换关键字(如int)(value)来实现，强制将value转换成int类型；
3. 方法调用：某些数据类型之间的转换，可以转化为对象的相互调用，调用对象的特定方法，将结果转化为目标数据类型；
4. 构造器调用：对于用户自定义类之间的转换，也可以使用构造器来实现，通过调用特定构造器，将源对象转换为目标类型。

## 隐式类型转换
在编程过程中，数据类型之间可能会出现隐式转换现象。例如：

```java
int num = 10 / 3; // 此处num的类型是int
double result = num * 2.5; // 此处result的类型是double
System.out.println("The result is: " + result); // 打印结果：The result is: 7.5
```

由于`int/int`运算结果是整型，因此`num`被赋值给`result`的时候，`result`的类型变为了`double`。这是由于算术运算符优先级关系所导致的，乘法运算符的优先级比除法运算符的优先级更高。

## 显式类型转换
通过使用数据类型转换函数可以显式地将一种数据类型转换成另一种数据类型。Java支持以下两种类型转换函数：

1. `(类型名)value`：将value转换为指定的类型，适用于基本数据类型到其对应的包装类型、或者其自身类型之间的转换；
2. `Class.forName("类全限定名").newInstance()`：用指定的类创建一个新实例，适用于用户自定义类之间的转换。

## 类型转换错误
类型转换错误常见于以下几种情况：

1. 从小类型转换成大类型：例如将`int`类型转换成`long`，虽然最终的结果可能溢出，但是不会报错。
2. 从大类型转换成小类型：例如将`long`类型转换成`int`，虽然不会丢失信息，但是会损失精度。
3. 强制类型转换失败：例如将一个`String`类型转换成`int`，虽然字符串可以解析为整数，但是解析出错的话，转换也会失败。
