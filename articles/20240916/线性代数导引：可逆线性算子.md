以下是实现可逆线性算子的 Python 代码：

```python
import numpy as np

def inverse_linear_operator(A, v):
    A_inv = np.linalg.inv(A)
    return A_inv @ v

v = np.array([1, 2])

result = inverse_linear_operator(A, v)
print(result)
```
