其中，\( y \) 为预测结果，\( f \) 为激活函数，可以采用 ReLU、Sigmoid、Tanh 等函数。

### 4.2 公式推导过程

以房价预测为例，假设我们有以下数学模型：

\[ y = w_1 \cdot x_1 + w_2 \cdot x_2 + ... + w_n \cdot x_n + b \]

其中，\( w_1, w_2, ..., w_n \) 为权重，\( x_1, x_2, ..., x_n \) 为输入特征，\( b \) 为偏置。

为了优化模型，我们需要计算损失函数 \( J \)：

\[ J = \frac{1}{2} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \]

其中，\( y_i \) 为真实标签，\( \hat{y}_i \) 为预测标签。

为了最小化损失函数，我们可以使用梯度下降算法更新权重：

\[ w_1 = w_1 - \alpha \cdot \frac{\partial J}{\partial w_1} \]
\[ w_2 = w_2 - \alpha \cdot \frac{\partial J}{\partial w_2} \]
\[ ... \]
\[ w_n = w_n - \alpha \cdot \frac{\partial J}{\partial w_n} \]

其中，\( \alpha \) 为学习率。

### 4.3 案例分析与讲解

以一个简单的房价预测任务为例，假设我们收集了以下数据：

| 输入特征 | 真实标签 |
| -------- | -------- |
| 100      | 200      |
| 200      | 300      |
| 300      | 400      |
| 400      | 500      |

我们可以构建一个简单的线性回归模型，尝试预测房价。假设初始权重为 \( w_1 = 1, w_2 = 1, b = 0 \)，学习率为 \( \alpha = 0.1 \)。经过一轮梯度下降后，我们可以得到新的权重：

\[ w_1 = 1 - 0.1 \cdot \frac{\partial J}{\partial w_1} \]
\[ w_2 = 1 - 0.1 \cdot \frac{\partial J}{\partial w_2} \]

通过计算，我们可以得到新的权重：

\[ w_1 = 0.9 \]
\[ w_2 = 0.9 \]

此时，预测房价为：

\[ y = 0.9 \cdot 100 + 0.9 \cdot 200 + 0 = 290 \]

与真实标签 \( 200 \) 相差较大，说明当前模型存在较大误差。我们可以通过增加训练轮次、调整学习率等方法来优化模型。
