                 

# 1.背景介绍


人类历史上几乎没有任何一次技术革命可以比拟电气化时代。从远古到今天，发明了太多先进的科技产品，比如火车、轮船、电视机等等，无一不充分体现了人类文明发展的脚步。然而，由于地球资源的枯竭和工业革命的加速，使得人类的生产力受到了严重限制。

19世纪初，英国通过美国制造的蒸汽机发现了可以形成液态氢燃料。这是后来电的前身，引起了航空、交通、交通运输和核能领域的革命。

20世纪70年代中期，美国政府推出了电动车和电动飞机，第一次改变了整个社会的经济格局。这一举措在很大程度上彻底颠覆了工业时代，开启了现代信息时代。

但是随着电力的日益富裕和技术的进步，对环境的污染也越发严重，导致全球范围内的污染物排放量激增。在电力的应用发展方面，中国也是一个例子，通过建立中国最大的发电站来彰显自己的发展能力。

这次人类历史上的首次电气化革命，就是发生在电力革命之后。即电力革命带来的巨大的社会财富转移，以及对资源的巨大浪费。

20世纪80年代末，中国正处于改革开放的关键阶段。在国际上产生的巨大反响，给予了世界各国政府希望，也促使很多国家率先建立起核电站、风电场和太阳能光伏项目。这些项目的建设都将电力的利用效率提高了十倍以上。

但同时，由于核电站等高能耗设备的发展，又对环境造成了恶劣影响，并导致农药的滥用、重金属的污染以及对生态系统的破坏。因此，为了保护环境和国家安全，政府在2015年宣布停止建设新核电站、风电场和太阳能光伏项目，取缔旧有的核电站、风电场和太阳能光伏项目，实施限时停电。

虽然如此，在电气化革命的过程中，仍然有许多新的科技突破出现。2015年，李克强总理主持召开的科技政策新突破论坛上，清华大学的王晓东老师给大家展示了一套新的“5G+”产品方案，这是由中国共产党领导的联合国教科文组织（UNESCO）提出的一项关于如何利用超高频率、高功率的现代通信技术来实现智慧城市、智慧商务、智慧医疗、智慧农业、智慧社区等新一代科技产业的愿景。

其中就包括了通过“5G+”产品在户外携带大容积数据，连接到城市的各种传感器，实现大规模数据采集，帮助用户获得更精准、智能的信息服务。这一产品的研发和推广都是由中国工程院院士、原联想董事长徐小平所领导的。

这次的人类技术革命已经结束，但还有下一个人类技术革命的到来。当然，这次的电气革命以及其他的突破性科技革命，只是众多的突破点中的一个。它们之间还有很多耦合，我们还需要拭目以待。
