                 

# 1.背景介绍


本文作者将详细介绍程序员面试后应该如何有效地进行后续处理。主要分为3个部分，分别为“简历筛选”，“后续工作指导”，“拒信及应聘心态调整”。

## 1.简历筛选
当候选人已经通过了初步的笔试面试，但仍需进一步筛选时，可以采取如下方式：

1. 提升竞争力：首先要做到的是提升自己竞争力。比如在笔试前完成过多项技术上相关的培训、做过过硬的项目实战等，并能为公司或行业带来长久的价值，而不是仅靠个人能力。
2. 展现实力：掌握最新技术知识、具有创新精神、对企业业务有全面的理解，能够提供更加合适的职位需求。
3. 应用经验：能够为公司或个人带来实际工作上的帮助和贡献。在面试过程中分享自己的开源项目、解决过的问题，以及在求职期间真正干出成绩。
4. 合格证明：虽然笔试和面试都可能让候选人脱颖而出，但同时需要提交的合格证明文件也很重要。如需拥有一个合格证明文件，可以向所在单位申请推荐，也可以到互联网上搜索免费的合格证明模板。

## 2.后续工作指导
当候选人已经获得了一份 offer ，但仍然希望进一步提升自己的能力时，可以从以下方面入手：

1. 加强学习：虽然 offer 一出来就很刺激，但是只有不断地学习才能让自己变得更强。需要选择自己感兴趣的领域、深入理解各种技术理论，并总结自己的见解。
2. 拓展技能：虽然有了 offer ，但还有很多工作要做。除了充实职业生涯之外，还需要根据当前的岗位要求及目标，深入研究相关技术，了解更多的信息，提高自身的竞争力。
3. 沉淀经验：与面试官交谈，详细了解企业或公司对候选人的期望、要求、喜好、优点、缺点等信息。通过沟通，可以发现和开发自己独有的能力，为未来工作奠定坚实基础。
4. 跟踪进度：如果在一段时间内一直没有收到回复，可以主动跟踪对方的工作进度、安排面试时间等，确保自己准时按时完成工作。

## 3.拒信及应聘心态调整
面试过程中如果出现问题或被吊销 offer ，建议采用以下策略：

1. 一定要主动承担责任：即使事前已经做好充分准备，也不要贸然接受任何offer 。要有准备接受失利的心态，否则会让自己蒙受损失。
2. 耐心沟通：坦率告诉对方自己为什么无法胜任这个岗位，直面自己的短板，真诚倾听对方的意见，表达自己的疑惑。
3. 不要轻易否认自己的能力：过分的自我怀疑是不必要的，拒绝别人的挑剔也不是什么稀松平常的事情。
