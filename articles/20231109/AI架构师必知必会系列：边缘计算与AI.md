                 

# 1.背景介绍


　　什么叫做边缘计算(Edge Computing)？它从字面意思上理解就是将计算任务委托到边缘端设备，比如智能手机、穿戴设备、工业终端等设备进行计算，而不是像中心服务器那样，一次计算下来全都传回中心服务器，实现了分布式计算能力。这个词汇就像电脑时代的集中式计算一样，有很大的局限性。所以需要引入边缘计算的原因主要是为了提升计算性能和降低计算成本，这才有了物联网、区块链、云计算的崛起。

　　那么边缘计算又是怎么实现分布式计算能力的呢？简单来说，边缘计算由两种设备组成：终端设备（End Devices）和核心设备（Core Device）。终端设备通过网络连接到核心设备，并安装相应的软件应用。当终端设备的任务需要执行的时候，就会将任务发送给核心设备进行处理。这样，终端设备就可以把复杂的数据分析、高速运算等任务交给核心设备去完成。这种架构方式可以有效地分担核心设备的计算资源，让更多的终端设备享受到更好的服务质量。

　　随着人们生活环境的不断变化，越来越多的人在用手机、平板、笔记本、电视等触屏终端设备完成工作，并且这些设备一般都在各种各样的场景下都能接入互联网。因此，边缘计算引起了巨大的社会和经济变革潮流。微软亚洲研究院的陈军博士认为，“边缘计算将带来新的商业模式，其关键点之一就是能够将核心的计算任务转移到边缘端设备，从而减少运营成本、缩短响应时间、提高服务质量。”这也直接促使了“边缘计算”这一术语的兴起。另外，特斯拉这样的科技企业正是通过“边缘计算”的方式来实现自主驾驶和自动驾驶，并且通过“混合云”的方式部署模型并实施训练，使得算法不依赖于数据的存储和传输，实现端到端的服务。

　　2019年4月17日，在欧洲机器学习领域里，英伟达推出了一款新产品——Jetson AGX Xavier。这是一个用于工业界的类别级处理器，支持多个核心的GPU架构和强大的神经网络计算单元。可以预见，随着边缘计算越来越普及，未来边缘计算将成为一个重要的方向。

　　综上所述，边缘计算是一种新型的计算架构和计算范式，它借助于终端设备的分布式计算能力，利用云计算、超级计算机等云端资源帮助用户处理过剩的计算资源，同时通过更加智能的算法和模型帮助用户解决各种各样的问题。因此，边缘计算将对信息时代的社会和经济发展产生深远的影响。为此，需要掌握边缘计算相关技术，并善于将其应用到实际生产中。
# 2.核心概念与联系
　　这里我将边缘计算的一些核心概念和联系进行简要介绍。

  ### 边缘计算的定义
　　边缘计算就是将计算任务委托到边缘端设备进行处理，减少服务器负载和延迟，从而提升用户体验、节省成本、提升性能。最早，这项技术被称为Mobile Cloud，其主要关注移动设备上的计算。移动网络和移动应用正在蓬勃发展，云计算正在改变这个行业。如今，随着边缘计算技术的崛起，越来越多的企业、组织和个人发现了它的巨大潜力。

  ### 边缘计算的核心概念
　　边缘计算主要有以下几个核心概念：
   - 终端设备：就是最终用户的设备。终端设备可能是智能手机、手表、电话机、车载设备等；
   - 核心设备：就是提供计算能力的设备。核心设备可以是计算机集群、服务器、路由器、交换机等；
   - 智能网关：是运行在核心设备上的网关软件，可识别并管理终端设备，并根据资源需求调度任务到相应的终端设备上；
   - 边缘云：指的是核心设备上运行的私有云或公有云，用于提供数据处理、分析、计算等功能，可以帮助边缘设备进行分布式计算。

  ### 边缘计算与物联网、区块链、云计算的关系

  边缘计算与物联网(IoT)、区块链(Blockchain)、云计算(Cloud computing)有什么关系呢？下面这张图可以帮你理解它们之间的关系：
  

  - IoT：物联网，也称为智慧物流，是指用现代化的信息技术来实现各种物流自动化功能，以满足消费者日益增长的物流需求。其核心特征包括智能终端、海量数据、动态协作。通过智能物流，可以降低物流成本，提高效率和品质。
  - Blockchain：区块链，也称分片区块链或分散式区块链，是指去中心化的、无固定所有权的共享账本。其主要特征是：去中心化，不信任任何第三方；安全，基于公钥加密；透明，记录所有操作过程。区块链可以应用于金融、交易、医疗健康、供应链等领域，为解决复杂的共识问题提供一个可靠的基础设施。
  - Cloud computing：云计算，是利用互联网的云服务平台，通过网络提供计算机硬件、软件、数据存储等资源的一种计算方式。其核心特征包括按需付费、弹性扩展、异地容灾、高可用性等。通过云计算，用户可以快速获得资源，降低成本，提高可靠性，从而提高竞争力和市场份额。
