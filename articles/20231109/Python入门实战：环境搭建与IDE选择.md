# 3.环境配置
## 安装Python
1.确定下载版本和系统。官网上提供了多个Python版本供用户下载。其中，最常用的三个版本分别为：
   - Python 3.X (最新稳定版):推荐所有Windows用户下载此版本，它具有跨平台特性，安装简单、速度快、缺陷少、功能丰富，适用于大多数需求场景。
   - Anaconda (免费开源包):该包是基于Python的数据处理、分析和机器学习平台，由商业公司Anaconda公司提供。它预先安装了许多常用的第三方库，使得用户在使用时无需再单独安装。同时，Anaconda还提供了包管理器conda，通过它可以轻松地安装、更新、删除各种Python库。
   - Pyenv:Pyenv是一个Python版本管理工具，它允许用户在同一个系统上安装不同版本的Python。它可以通过pyenv命令来管理Python版本。
2.下载并安装。在线搜索下载链接，下载安装包，双击安装即可。安装过程中，根据提示设置环境变量PATH和PYTHONHOME。其中，PATH路径是指Python可执行文件的存放路径，PYTHONHOME是指Python的安装目录。
3.验证安装成功。在CMD中输入`python`，出现提示符`>>>`说明安装成功。
## 安装第三方库
Python的第三方库无处不在。这里推荐几个常用的第三方库，它们都非常有用：
- NumPy:NumPy是Python中的一种数值计算扩展，支持对多维数组和矩阵运算，提供大量的数学函数库。
- Pandas:Pandas是基于NumPy构建的一种开源数据结构，提供高级数据分析功能。
- Matplotlib:Matplotlib是Python中的一个绘图库，它提供了大量的绘图类型，可以满足复杂的图表效果。
- Seaborn:Seaborn是基于matplotlib的高级数据可视化库，它提供了更美观、更专业的可视化效果。
- scikit-learn:Scikit-learn是基于NumPy构建的一个机器学习库，它包含常用的数据预处理、特征工程、分类、回归、聚类、降维等算法。
除了上述常用库之外，还有很多其他的库可用，如果需要的话，也可以在线搜索并安装。安装第三方库可以使用pip命令：`pip install library_name`。
## 配置环境变量
Python的执行文件会把当前目录添加到环境变量`PATH`中，所以我们只需在`PATH`路径中添加Python的安装目录，就可以直接在任意位置打开终端，然后输入`python`命令启动Python交互式环境。这样做有一个弊端，就是可能会与已有的环境变量发生冲突。解决这个问题的方法是将Python的安装目录添加到环境变量`PATH`的末尾，而不是头部。这样就能保证不会与已有的环境变量发生冲突。具体操作如下：
1.找到Python的安装目录。右键单击桌面上的“计算机”，依次选择“属性”→“高级系统设置”→“环境变量”。
2.在系统变量中，找到名为`Path`的变量，双击编辑。
3.在编辑框中追加`:C:\Users\你的用户名\AppData\Local\Programs\Python\Python37`或者类似的路径。注意，这里的`你的用户名`应该替换为自己的用户名。
4.保存退出，重启计算机即可。
## IDE选择
目前，主流的Python编辑器有IDLE、Spyder、VSCode等。以下介绍一下这些编辑器的特点和优缺点：
### IDLE
IDLE(Integrated Development and Learning Environment) 是 Python 的默认编辑器。IDLE 支持几乎所有 Python 功能，包括语法高亮、自动缩进、语法检查、代码运行、调试等。它的界面简洁、功能强大，适合于初学者学习。但它只能在 Windows 操作系统下使用。
### Spyder
Spyder 是 Python 官方推荐的编辑器，它集成了许多方便使用的功能，比如：代码自动补全、代码智能提示、语法检查、代码运行、调试等。它也是支持 Windows 和 Linux 操作系统的。不过，Spyder 需要额外安装第三方插件才能获得完整的功能。
### VSCode
VSCode 是微软推出的 Python 编辑器，它提供了对 Python 文件的自动完成、语法检查等功能，并且可以在 MacOS、Linux 和 Windows 下运行。它使用起来比较舒服，功能也很强大。但是，它需要安装额外插件才有完整的功能。
最后，我们推荐大家使用Spyder作为Python的主力编辑器。
