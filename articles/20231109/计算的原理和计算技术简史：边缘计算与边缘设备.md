                 

# 1.背景介绍


## 1.1计算的定义及其历史形成
计算机（computer）一词最早由约翰·麦卡锡（John McCarthy）于1945年提出，指能够按照既定的程序执行特定计算任务、存储和检索数字资料的电子设备。随着计算机的应用领域的不断拓展，越来越多的人发现，计算机的处理能力、运算速度与价格等性能指标有待改善。例如，当时的美国商务部曾对硬件设计者要求他们提供每秒大约4亿个乘法运算的处理器，即使是如今摩尔定律又降到了每两年才升高一点。此时，科学家们发明了微处理机作为解决方案。可想而知，短时间内，人们对计算机的需求是如此之高，以至于有些商业公司甚至公然宣称自己的产品“就是一个电脑”。

计算机发明于上世纪50年代末到60年代初期，在当时还是一个非常狭小的领域，它仅限于解决人类计算和数据记录方面的需求。到了1960年代后期，计算机的能力已经超越了人类的计算能力，它可以进行更加复杂的运算。但是，它在商业和科研等领域仍处于边缘地位，主要原因还是缺乏技术支持。

另一方面，随着信息技术的发展，人们越来越关注数字化带来的信息革命。在这些年里，全球范围内出现了海量的数据，它们对于经济、金融、社会等各行各业都产生着巨大的影响。计算机科学家需要面临新的计算模式，并以此调整技术路线，通过云计算、大数据、人工智能、物联网等新型技术，实现数据的高度自动化、精准分析和智能决策，从而改变整个社会的运行方式。

为了达到这个目标，计算机科学家们开始关注计算设备的边缘部署，特别是那些适合嵌入大规模生产过程中的特殊设备——边缘计算设备。根据功能的不同，这类设备可以用于机器学习、图像处理、语音识别、环境监测、工业控制等诸多领域，而且各家厂商也在不断创新中尝试突破边界。

## 1.2什么是边缘计算
所谓边缘计算，就是把计算放在设备端，让设备通过自主计算获得所需结果。典型的边缘计算设备包括传感器、处理器、网络、屏幕、摄像头、终端设备等，所有这些设备共同组成了一个“计算集群”，并且工作在一个或多个位置，由边缘计算平台统一管理。这种架构可以让设备迅速收集和处理大量数据，并实时响应客户请求，同时减少中心服务器的负载。因此，边缘计算被认为是一种颠覆性的计算技术，它可以极大地降低中心服务器的资源消耗、节省带宽、提高用户体验。与传统计算相比，边缘计算的优势主要有以下几点：

1. 更快的响应时间：边缘计算设备在本地完成计算，响应速度要快于中心服务器；
2. 节省资源：边缘计算设备采用高效率的计算方法，可以节省大量的能源和空间资源，同时降低中心服务器的压力；
3. 降低延迟：由于数据在边缘计算设备上直接计算，所以无需传输到中心服务器，降低了中间环节的延迟；
4. 高度自动化：边缘计算设备可以根据上下文条件进行自主决策，让复杂的业务逻辑和规则自动化处理；
5. 智慧化应用：通过智能分析和决策，边缘计算设备可以快速理解并识别复杂事件，做出正确的应对措施。

## 1.3边缘计算技术领域
目前，边缘计算领域包含以下四种技术：
1. 边缘计算基础设施：主要指供应链协同、云计算基础设施、边缘计算框架、软件定义网络等，旨在打造一套完整的边缘计算基础设施；
2. 边缘计算平台：主要指边缘计算服务平台、边缘计算虚拟机、边缘计算应用程序、边缘数据中心等，旨在将云计算和边缘计算有机地结合起来；
3. 边缘计算应用：主要指边缘计算安全、边缘计算智能硬件、边缘计算智能终端、边缘计算连接器等，旨在将边缘计算技术应用到各个行业领域；
4. 边缘计算产业：主要指边缘计算基础设施商用化、边缘计算云服务商用化、边缘计算硬件商用化、边缘计算设备商用化等，旨在推动边缘计算进入更多的企业、组织和行业。

边缘计算是一个多研究、多学科、多领域的综合性研究领域，涉及到了物理学、电子学、计算机科学、数学、统计学等多个学科。因此，对边缘计算的研究应该注重理论联系实际，充分发挥边缘计算的各项能力，并借助现代计算机科学技术和相关标准，构建具有中国特色的边缘计算产业。
