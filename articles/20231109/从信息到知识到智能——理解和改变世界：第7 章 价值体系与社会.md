                 

# 1.背景介绍


什么是价值体系？为什么需要价值体系？价值体系是什么样子的？要怎样建设价值体系？如何确立自己的价值观？所谓的“我”、“自己”，还有“他者”，到底是什么意思？——这些都是许多人都关心的问题，但很少有人能够从头到尾完整地回答上述所有问题。本章试图对价值体系的概念做全面性阐述，帮助读者更好地了解这个概念及其在现代社会中的重要作用。
# 2.核心概念与联系
## 2.1 价值体系是什么
价值体系是一个指导行为的机制，它通过定义和界定个人或群体的人格特质和风格特点，并设定其目标及道德标准，帮助个体建立自我认同、形成自尊、达到共赢的价值理念和生活方式。价值体系包括三个层次：一是道德价值观层面，二是伦理价值观层面，三是文化价值观层面。
### 道德价值观层面
道德价值观层面的核心是信念、义务和责任。它们包括：自律、诚实、公正、诚信、尊重规则、尊重权威等。道德价值观是指个人在不同社会关系中遵守的基本准则和规范。
### 伦理价值观层面
伦理价值观层面的核心是义务、自由、平等、正义。它们包括：宽容、包容、尊重差异、公开透明、可预期、制度合理等。伦理价值观是指个人在不同社会环境中的追求和实践。
### 文化价值观层面
文化价值观层面的核心是情感和爱。它们包括：善良、美丽、自由、自尊、勤奋等。文化价值观是指个体对自然界、人类社会以及周边世界的看法和认识。
## 2.2 为什么需要价值体系？
价值体系的存在是为了塑造个体的性格，形成自我的认同。只有一个清晰的价值观，才能做到自我实现。同时，价值观是人的基本特征之一，也是构建有效的人际关系的基础。一旦形成了共同的价值观，那么各种感情、态度、表情和行为都会带来统一的标准，并且能够促进互相学习、促进团队合作、促进事业成功。因此，没有人际关系，就不会有真正的合作。
## 2.3 如何建设价值体系？
如何建设价值体系可以分为以下几个步骤：
1.探索性研究：理解自我，找到最重要的人生动机，寻找内心深处的美好愿望；
2.分析形成自己价值观的方法：阅读理解、调查研究、梳理归纳、写下笔记、观察分析、举证比较、亲身体验等；
3.坚持执行价值观：落实价值观并坚持执行，保持连续不断的输出，不断完善和优化自己的价值观；
4.传播价值观：将自己的价值观传递给别人，成为人际交往的桥梁，增强彼此间的信任。
## 2.4 如何确立自己的价值观？
每个人都必须独立思考和寻找自己的价值观。确定自己的价值观非常重要，需要考虑以下几方面：
- 历史情况：包括个人经历、家庭环境、社会角色等因素。
- 感受力：审视自我、与他人分享自己的感受，获取不同的视角。
- 需求力：了解当前时代的社会现实，判断需求和担忧。
- 使命驱动：基于自身的使命，确认和设置核心价值观。
- 价值偏好：拥有丰富的价值观念，如友善、理想主义、进取精神等。
- 社会影响：关注周边人的声音，反省自己的局限性，了解社会的喜怒哀乐。
- 时代气氛：判断现在是否是价值观较为稳定的时刻，拥抱变化，放弃过去的恶劣影响。
- 知识水平：阅读外国文献、掌握最新技术，学习人类文明发展史、心理学、社会科学等。
## 2.5 “我”、“自己”、“他者”分别是什么意思呢？
“我”、“自己”和“他者”都可以称呼个人或群体，都有其对应的词汇。“我”可以指代某个人或者集体；“自己”可以指代个人的能力、选择和态度；而“他者”可以指代自己不认可的一切、不合理的一切、无助于完成工作的任何事物。总之，以上三个词汇用来形容个人、自我和他人。但是，它们之间又存在着一些差异。例如，“我”、“自己”主要用于人格描述，意味着自己具有独特性，可以是积极主动的，也可以是消极被动的。“他者”一般都表示一系列别人的行为或影响。
