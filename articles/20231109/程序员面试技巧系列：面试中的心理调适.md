# 2.核心概念与联系
## 2.1 什么是心理调适？
“心理调适”（mental adaptation）指的是当面试官提出一些关于自身、家庭生活或职业道德方面的问题时，面试者应当如何回答。调适方式可以分为自我保护、信任感训练、团队协作等不同的维度。一般来说，在面试过程中，我们需要注意保持心态平稳，避免表现出内向、孤僻的不良习惯；倾听冷静地接受问题并回答，不要盲目跟风；尝试不同的视角看待问题；鼓励面试官重视候选人自我反馈。


## 2.2 为何要进行心理调适？
在做技术面试时，不管是面试官还是面试者，都会遇到许多心理上的问题。比如，我们通常会被问到一些性格缺陷，比如懒惰、骄傲、自负、偏执、贪婪等等。作为技术人员，我们的这些缺点都会影响到我们的工作能力，甚至可能会造成我们的职业生涯失败。而在面试中，面试者是否真正理解这些问题，是否能够承担起相应的责任，是决定是否拿到offer最重要的因素之一。因此，我们在面试前应该精心准备，确保面试过程中能够做到心理调适，以提升面试效果。

## 3.1 自我保护
首先，我们需要以一种积极的心态接纳面试官，不要觉得自己是一名弱者，没有作为，也不要自卑自责，把自己的不足展现出来。而后，我们可以采用以下方法进行自我保护：
- 提醒自己坚持锻炼身体，保持健康饮食 habits。
- 用心关注自己的情绪状态，不要被自己的情绪左右开来。
- 不要抱怨生活中的不幸事物，用理智克服困境。
- 想想为什么参加面试，是否有异性朋友或老乡的支持。
- 刻意练习，去外面散步、游览、拜访别人。
- 多与亲密的人在一起，共同进步。
- 在面试结束后，针对自己的一些不足，找回一些自信和勇气。

## 3.2 信任感训练
其次，要善于展示自己的专业知识，引导面试官产生信任感。这里主要有两个策略：
- 通过提问询问面试官是否喜欢分享自己的成长经历，以及自己的优缺点。
- 在面试结束后，给面试官一些反馈意见，提醒自己没有把握的地方，这样可以更好地培养信任感。

## 3.3 团队协作
第三，我们要做好团队建设，以便在团队中找到志同道合的伙伴。这里有两种方法：
- 如果你所在的公司存在相关部门，可以提前与他们打听相关信息，了解到公司业务方向和产品情况。
- 招聘的时候注意到你所属的领域或兴趣所在的企业，并邀约他们的成员参与面试。

## 3.4 聚焦面试问题
最后，我们要提升问题的关注度，集中注意力。也就是说，我们要集中精力，一次只专注于一个问题上。这种方式有助于我们集中注意力，防止分心、分神，同时也降低了我们注意力的疲劳。另外，可以通过掌握一些典型的问题，反复练习提升自己的记忆力和理解能力。
