## 2. 核心概念与联系

### 2.1 序数

序数是用来描述集合元素排列顺序的数学概念。它可以被定义为一个良序集，即一个完全有序的集合，其中每个非空子集都有一个最小元素。

* **有限序数:** 自然数 0, 1, 2, 3, ... 是有限序数。
* **无限序数:** 无限序数是大于所有有限序数的序数，例如 ω (第一个无限序数)。

### 2.2 序数乘积空间

序数乘积空间是一个由多个序数组成的集合。例如，两个序数 α 和 β 的乘积空间可以表示为 α × β，它包含所有形如 (α', β') 的有序对，其中 α' ≤ α 且 β' ≤ β。

## 3. 典型秩序的定义与性质

### 3.1 典型秩序的定义

典型秩序是序数乘积空间上的一个偏序关系，它可以用来比较不同序数的复杂度。

* **定义:** 设 (α', β') 和 (α'', β'') 是序数乘积空间 α × β 中的两个元素，则 (α', β') ≤ (α'', β'') 当且仅当 α' ≤ α'' 且 β' ≤ β''。

### 3.2 典型秩序的性质

典型秩序具有以下性质：

* **反身性:** 对于任何元素 (α, β) ∈ α × β，都有 (α, β) ≤ (α, β)。
* **反对称性:** 如果 (α', β') ≤ (α'', β'') 且 (α'', β'') ≤ (α', β')，则 (α', β') = (α'', β'')。
* **传递性:** 如果 (α', β') ≤ (α'', β'') 且 (α'', β'') ≤ (α''', β''')，则 (α', β') ≤ (α''', β''')。

### 3.3 典型秩序的例子
