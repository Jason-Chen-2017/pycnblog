Python Web框架Flask是一款轻量级、易用的Web框架，它的核心设计思想是“不要我们做事情，让我们专注于做事情”。Flask是基于Werkzeug WSGI库和Jinja2模板引擎构建的。它的设计哲学是“少是美”，它不包含对数据库的抽象层，不包含 ORM，也不包含模板引擎。这使得Flask 成为一个非常灵活的Web框架，可以根据需要扩展和定制。

Flask 的核心设计思想是“不要我们做事情，让我们专注于做事情”。这意味着Flask 不会对我们做一些事情，而是让我们专注于做事情。例如，Flask 不会自动生成模型，也不会自动生成视图。这使得Flask 成为一个非常灵活的Web框架，可以根据需要扩展和定制。
