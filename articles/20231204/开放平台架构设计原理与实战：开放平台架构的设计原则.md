### 4.1数据存储

我们可以使用Python的SQLite库来实现简单的数据存储。以下是一个简单的数据存储示例：

```python
import sqlite3

# 创建数据库
conn = sqlite3.connect('example.db')

# 创建表
cursor = conn.cursor()
cursor.execute('''CREATE TABLE example
                 (id INTEGER PRIMARY KEY, name TEXT)''')

# 插入数据
cursor.execute("INSERT INTO example (name) VALUES (?)", ('John',))

# 查询数据
cursor.execute("SELECT * FROM example")
rows = cursor.fetchall()
for row in rows:
    print(row)

# 关闭数据库
conn.close()
```

在这个示例中，我们使用SQLite库创建了一个简单的数据库，并创建了一个名为"example"的表。我们插入了一条数据，并查询了这条数据。最后，我们关闭了数据库连接。

### 4.2计算服务

我们可以使用Python的multiprocessing库来实现简单的计算服务。以下是一个简单的计算服务示例：

```python
import multiprocessing

def compute(x):
    return x * x

if __name__ == '__main__':
    pool = multiprocessing.Pool(processes=4)
    result = pool.map(compute, [1, 2, 3, 4, 5])
    print(result)
    pool.close()
    pool.join()
```

在这个示例中，我们使用multiprocessing库创建了一个计算池，并定义了一个名为"compute"的计算任务。我们使用map函数将计算任务应用于一个列表，并获取计算结果。最后，我们关闭计算池。

### 4.3网络通信

我们可以使用Python的socket库来实现简单的网络通信。以下是一个简单的网络通信示例：

```python
import socket

# 创建套接字
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# 连接服务器
server_address = ('localhost', 10000)
s.connect(server_address)

# 发送数据
message = 'Hello, World!'
s.sendall(message.encode())

# 接收数据
data = s.recv(1024)
print(data.decode())

# 关闭套接字
s.close()
```

在这个示例中，我们使用socket库创建了一个TCP套接字，并连接到本地服务器。我们发送了一条消息，并接收了服务器的回复。最后，我们关闭了套接字。
