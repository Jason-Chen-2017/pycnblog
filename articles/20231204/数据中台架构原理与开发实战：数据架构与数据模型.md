                 

# 1.背景介绍

数据中台架构是一种具有高度可扩展性、高性能、高可靠性和高可维护性的数据处理架构，它可以实现数据的集成、清洗、转换、存储和分发，为企业的各种业务应用提供数据支持。数据中台架构的核心是数据中台系统，它是企业数据资源的统一管理平台，负责整合、管理、分发企业内外部数据资源，为企业各业务系统提供数据支持。
