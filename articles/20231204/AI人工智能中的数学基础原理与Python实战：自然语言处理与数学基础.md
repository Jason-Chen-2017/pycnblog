# 2.核心概念与联系

在自然语言处理中，我们需要了解以下几个核心概念：

1. 词汇表（Vocabulary）：词汇表是一种数据结构，用于存储自然语言中的单词。词汇表可以帮助我们对文本进行预处理，如去除停用词、词干提取等。

2. 词嵌入（Word Embedding）：词嵌入是一种将单词映射到一个高维向量空间的方法，以捕捉单词之间的语义关系。常见的词嵌入方法有Word2Vec、GloVe等。

3. 句子向量（Sentence Vector）：句子向量是将句子映射到一个高维向量空间的方法，以捕捉句子的语义。常见的句子向量方法有Average Pooling、Max Pooling等。

4. 神经网络（Neural Network）：神经网络是一种模拟人脑神经元的计算模型，可以用于处理复杂的数据和任务。在自然语言处理中，我们常用卷积神经网络（CNN）、循环神经网络（RNN）、长短期记忆网络（LSTM）等。

5. 深度学习（Deep Learning）：深度学习是一种利用多层神经网络进行自动学习的方法。在自然语言处理中，我们常用卷积神经网络（CNN）、循环神经网络（RNN）、长短期记忆网络（LSTM）等。

6. 自然语言生成（Natural Language Generation）：自然语言生成是一种将计算机生成自然语言文本的技术。常见的自然语言生成方法有规则生成、统计生成、神经生成等。

7. 自然语言理解（Natural Language Understanding）：自然语言理解是一种将计算机理解自然语言文本的技术。常见的自然语言理解方法有规则理解、统计理解、神经理解等。

8. 语义角色标注（Semantic Role Labeling）：语义角色标注是一种将自然语言句子映射到语义角色图结构的方法，以捕捉句子的语义关系。

9. 命名实体识别（Named Entity Recognition）：命名实体识别是一种将自然语言文本中的实体（如人、地点、组织等）识别出来的方法。

10. 依存句法分析（Dependency Parsing）：依存句法分析是一种将自然语言句子映射到依存关系图结构的方法，以捕捉句子的语法关系。

11. 情感分析（Sentiment Analysis）：情感分析是一种将自然语言文本映射到情感标签的方法，以捕捉文本的情感倾向。

12. 机器翻译（Machine Translation）：机器翻译是一种将自然语言文本从一种语言翻译到另一种语言的技术。常见的机器翻译方法有统计机器翻译、规则机器翻译、神经机器翻译等。

13. 语音识别（Speech Recognition）：语音识别是一种将语音信号转换为文本的技术。常见的语音识别方法有隐马尔可夫模型（HMM）、深度神经网络（DNN）、循环神经网络（RNN）等。

14. 语音合成（Text-to-Speech）：语音合成是一种将文本转换为语音信号的技术。常见的语音合成方法有WaveNet、Tacotron等。
