## 1.6 异常处理的代码实例

异常处理的代码实例主要包括以下几个方面：

1. 定义异常类：我们可以使用java.lang.Exception类来定义异常类，并为其添加相应的属性和方法。例如：

```java
public class MyException extends Exception {
    public MyException(String message) {
        super(message);
    }
}
```

2. 抛出异常：在程序中，当发生错误时，我们需要抛出异常。我们可以使用throw关键字来抛出异常。例如：

```java
public void method() throws MyException {
    // 当发生错误时，抛出异常
    if (condition) {
        throw new MyException("错误发生");
    }
}
```

3. 捕获异常：在程序中，当发生错误时，我们需要捕获异常。我们可以使用try-catch语句来捕获异常。例如：

```java
public void method() {
    try {
        // 当发生错误时，抛出异常
        if (condition) {
            throw new MyException("错误发生");
        }
    } catch (MyException e) {
        // 捕获异常
        e.printStackTrace();
    }
}
```

4. 处理异常：在程序中，当发生错误时，我们需要处理异常。我们可以使用try-catch语句来处理异常。例如：

```java
public void method() {
    try {
        // 当发生错误时，抛出异常
        if (condition) {
            throw new MyException("错误发生");
        }
    } catch (MyException e) {
        // 处理异常
        e.printStackTrace();
    }
}
```

5. 恢复异常：在程序中，当发生错误时，我们需要恢复异常。我们可以使用回滚、恢复点等方法来恢复异常。例如：

```java
public void method() {
    try {
        // 当发生错误时，抛出异常
        if (condition) {
            throw new MyException("错误发生");
        }
    } catch (MyException e) {
        // 恢复异常
        rollback();
    }
}
```
