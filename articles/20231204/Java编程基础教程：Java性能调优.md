# 4.具体代码实例和详细解释说明

在Java性能调优中，我们需要看到一些具体的代码实例，以便更好地理解性能调优的具体操作。以下是一个简单的排序算法的实例：

```java
public class BubbleSort {
    public static void main(String[] args) {
        int[] arr = {64, 34, 25, 12, 22, 11, 90};
        bubbleSort(arr);
        System.out.println(Arrays.toString(arr));
    }

    public static void bubbleSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < n - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }
}
```

在上述代码中，我们实现了一个简单的冒泡排序算法，用于对整数数组进行排序。通过对比相邻元素的值，我们可以将较大的元素向右移动，从而实现排序。
