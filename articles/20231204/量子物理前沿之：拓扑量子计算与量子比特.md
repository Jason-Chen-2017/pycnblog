# 2.核心概念与联系

## 2.1 量子比特（Qubit）

量子比特（qubit）是量子计算中的基本单位，它可以存储和处理信息。与传统的比特（bit）不同，量子比特可以同时存储多个状态。量子比特的状态可以表示为：

$$
|\psi\rangle = \alpha|0\rangle + \beta|1\rangle
$$

其中，$\alpha$ 和 $\beta$ 是复数，满足 $|\alpha|^2 + |\beta|^2 = 1$。

## 2.2 量子纠缠

量子纠缠是量子计算中的一个重要概念，它描述了两个或多个量子比特之间的相互作用。量子纠缠可以让量子比特之间的状态相互依赖，从而实现更高效的计算。量子纠缠可以通过以下操作实现：

$$
|\psi\rangle \otimes |\phi\rangle \rightarrow |\psi\rangle \otimes |\phi\rangle
$$

## 2.3 量子门

量子门是量子计算中的基本操作，它可以对量子比特进行操作。量子门的一个常见例子是 Hadamard 门（H-gate），它可以将量子比特从基态 $|0\rangle$ 转换到超位态：

$$
H|0\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)
$$

```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 创建一个量子电路

# 将第一个量子比特的状态设置为 $|0\rangle$
qc.initialize([1, 0], [0, 0])

# 应用Hadamard门到第一个量子比特
qc.h(0)

# 将量子电路转换为二进制格式
qasm_code = qc.qasm()

# 使用Qiskit的Aer后端进行模拟
simulator = Aer.get_backend('qasm_simulator')

# 将量子电路转换为QASM格式
qasm_code = assemble(qc)

# 使用Qiskit的Aer后端进行模拟
result = simulator.run(qasm_code).result()

# 绘制结果
plot_histogram(result.get_counts())
```
