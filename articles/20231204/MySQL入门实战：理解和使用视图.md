## 4.具体代码实例和详细解释说明

### 4.1创建视图的代码实例

```sql
CREATE VIEW employee_view AS
SELECT employee_id, first_name, last_name, department_id
FROM employees;
```

### 4.2查询视图的代码实例

```sql
SELECT *
FROM employee_view
WHERE department_id = 10;
```

### 4.3更新视图的代码实例

WHERE employee_id = 1;

DELETE FROM employee_view
