# 2.核心概念与联系
在Python中，文件读写主要通过两个内置函数实现：`open()`和`close()`。`open()`函数用于打开文件，`close()`函数用于关闭文件。在使用这两个函数时，我们需要了解一些关键的概念，如文件模式、文件对象和文件操作。

## 2.1 文件模式
文件模式是指我们在打开文件时使用的参数，用于指定我们希望对文件进行的操作。常见的文件模式有以下几种：

- `r`：只读模式，默认模式。
- `w`：写入模式，如果文件已存在，则覆盖文件内容。
- `a`：追加模式，如果文件已存在，则在文件末尾添加内容。

## 2.2 文件对象
文件对象是一个类型为`file`的对象，用于表示一个打开的文件。我们可以通过`open()`函数创建文件对象，并使用各种方法对文件进行操作。例如，我们可以使用`read()`方法读取文件内容，使用`write()`方法写入文件内容，使用`close()`方法关闭文件。

## 2.3 文件操作
文件操作是指我们对文件进行的各种操作，如读取、写入、追加等。在Python中，我们可以通过文件对象的方法来实现文件操作。例如，我们可以使用`read()`方法读取文件内容，使用`write()`方法写入文件内容，使用`close()`方法关闭文件。

## 4.1 读取文件内容
以下是一个读取文件内容的代码实例：

```python
# 打开文件

# 读取文件内容

# 关闭文件
file.close()

# 打印文件内容
print(content)
```

解释：

1. 使用`open()`函数打开文件，创建文件对象。
2. 使用文件对象的`read()`方法读取文件内容。
3. 使用`close()`函数关闭文件，释放文件资源。
4. 使用`print()`函数打印文件内容。
