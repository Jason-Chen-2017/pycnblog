## 1.1 Java国际化和本地化的概念

Java国际化（Internationalization，简称I18N，I表示第一个字母“I”，18表示18个字母之间的间隔，N表示第十八个字母“N”）是指设计一个程序，使其能够在不同的语言环境中运行，而不需要对程序代码进行修改。Java本地化（Localization，简称L10N，L表示第一个字母“L”，10表示10个字母之间的间隔，N表示第十个字母“N”）是指为一个国际化的程序提供特定的语言环境，例如中文简体、中文繁体、英文等。

Java国际化和本地化的主要目的是为了让Java程序能够更好地适应不同的用户需求，从而更广泛地应用于不同的地区和市场。
