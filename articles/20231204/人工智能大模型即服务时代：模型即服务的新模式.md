# 2.核心概念与联系
在模型即服务的新模式中，我们需要了解以下几个核心概念：

1. 模型：模型是人工智能系统的核心组成部分，它可以用来进行预测、分类、聚类等任务。模型通常是通过训练数据集来学习的，并在新的数据上进行推理。

4. 分配：分配是将模型服务分配给不同的资源的过程。分配可以根据资源的可用性、模型的性能需求等因素进行调整。

5. 管理：管理是对模型服务的生命周期进行控制和监控的过程。管理包括模型的版本控制、服务的启动和停止、资源的分配和调整等操作。

这些概念之间的联系如下：模型服务是在资源上运行的，需要通过分配来分配给不同的资源。同时，模型服务的管理是为了确保其正常运行和高效使用资源。
