# 2.核心概念与联系

## 2.1 量子比特（Qubit）

量子比特（qubit）是量子计算中的基本单位，它可以同时存储多个状态。与传统计算机中的比特（bit）不同，量子比特可以存储0、1、0和1等多种状态。量子比特的状态可以用纯态或混合态来描述。纯态量子比特的状态可以用向量表示，如|0⟩和|1⟩。混合态量子比特的状态可以用密度矩阵表示。

## 2.2 量子门

量子门是量子计算中的基本操作单元，它可以对量子比特进行操作。量子门可以实现各种基本操作，如旋转、翻转、门控等。量子门的实现方式有多种，例如通过电磁场、光场、微波等。量子门的操作对于量子计算的实现至关重要。

## 2.3 离子阱

离子阱是一种量子计算硬件，它利用离子的电磁性质来存储和操作量子比特。离子阱通常由电场、磁场和微波等多种场来控制。离子阱的优点包括高度可扩展性、低噪声性和高稳定性。因此，离子阱量子计算在实验室和学术界中得到了广泛关注。
