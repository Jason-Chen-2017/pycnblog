1.2 2010年，Rust发布了第一个稳定版本，并开始积累用户群体。

1.3 2012年，Rust发布了第二个稳定版本，并开始积累社区支持。

1.4 2014年，Rust发布了第三个稳定版本，并开始积累更多的第三方库和工具。

1.5 2016年，Rust发布了第四个稳定版本，并开始积累更多的用户群体和社区支持。

1.6 2018年，Rust发布了第五个稳定版本，并开始积累更多的第三方库和工具。

1.7 2020年，Rust发布了第六个稳定版本，并开始积累更多的用户群体和社区支持。

Rust编程语言的发展历程表明，它是一种具有潜力的现代系统编程语言，具有内存安全、并发原语和类型系统等特点。

```rust
use serde_json::Value;

fn main() {
    let data = r#"[
        {"name": "John", "age": 20},
        {"name": "Alice", "age": 25},
        {"name": "Bob", "age": 30}
    ]"#;
    let json: Vec<Value> = serde_json::from_str(data).expect("解析失败");
    let mut cleaned_data = Vec::new();
    for item in json {
        let name = item["name"].as_str().expect("name不是字符串");
        let age = item["age"].as_i64().expect("age不是整数");
        if age > 0 {
            cleaned_data.push(item);
        }
    }
    println!("{:?}", cleaned_data);
}
```
