```python
prompt = "请生成关于人工智能的正确信息。"
```

然后，我们可以将这个安全提示输入到GPT-3模型中，生成模型的回答。例如：

```python
import openai

openai.api_key = "your_api_key"

response = openai.Completion.create(
  engine="text-davinci-002",
  prompt=prompt,
  max_tokens=100,
  n=1,
  stop=None,
  temperature=0.5,
)

print(response.choices[0].text)
```

通过这种方式，我们可以让模型生成关于人工智能的正确信息。
