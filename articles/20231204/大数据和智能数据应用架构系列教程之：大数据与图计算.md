# 4.具体代码实例和详细解释说明

## 4.1 代码实例

以下是一个使用Python语言实现的最短路径算法的代码实例：

```python
import heapq

def dijkstra(graph, start):
    distances = {node: float('inf') for node in graph}
    distances[start] = 0
    queue = [(0, start)]

    while queue:
        current_distance, current_node = heapq.heappop(queue)

        if current_distance > distances[current_node]:
            continue

        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(queue, (distance, neighbor))

    return distances
```

## 4.2 详细解释说明

上述代码实例实现了一个最短路径算法，它使用了堆队列（heapq）模块来实现优先级队列的功能。算法的主要步骤如下：

1. 初始化距离字典，将所有顶点的距离设置为无穷大，除了起始顶点的距离设置为0。
2. 将起始顶点加入优先级队列。
3. 从优先级队列中取出当前顶点和当前顶点的距离。
4. 如果当前顶点的距离大于距离字典中的距离，则跳过当前顶点。
5. 对当前顶点的邻居进行遍历，计算到当前邻居的距离。
6. 如果计算出的距离小于距离字典中的距离，则更新距离字典和优先级队列。
7. 重复步骤3-6，直到优先级队列为空。
8. 返回距离字典。
