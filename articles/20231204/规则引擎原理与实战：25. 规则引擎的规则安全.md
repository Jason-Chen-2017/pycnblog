# 2.核心概念与联系

在规则引擎中，规则是指一种用于描述系统行为的语句，它可以根据特定的条件来执行某些操作。规则引擎的安全性是指规则不能损害系统的安全性和数据的安全性。

规则安全性的核心概念包括：

- 规则的可信度：规则是否来自可信来源，是否经过审查和验证。
- 规则的完整性：规则是否被修改或损坏，以影响系统的安全性。
- 规则的访问控制：规则是否有权访问敏感数据，以保护数据的安全性。
- 规则的执行限制：规则是否有权执行可能影响系统安全性的操作。
