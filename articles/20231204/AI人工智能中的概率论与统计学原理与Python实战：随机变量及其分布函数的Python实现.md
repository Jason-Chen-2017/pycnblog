# 2.核心概念与联系

## 2.1随机变量

随机变量是一个数学函数，它将一个随机事件映射到一个数值域上。随机变量可以是离散的或连续的，离散随机变量的取值是有限的，而连续随机变量的取值是无限的。随机变量的概率分布是描述随机变量取值概率的函数，通常用概率密度函数（PDF）或累积分布函数（CDF）来表示。

## 2.2分布函数

分布函数是概率论与统计学中的一个重要概念，它描述了随机变量的概率分布。分布函数可以用来计算随机变量取值的概率，也可以用来计算随机变量的期望、方差等统计量。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1概率密度函数（PDF）

概率密度函数是一个连续函数，它描述了随机变量在某个值处的概率密度。概率密度函数的定义为：

$$
f(x) = \frac{d}{dx}F(x)
$$

其中，$F(x)$ 是累积分布函数。

## 3.2累积分布函数（CDF）

累积分布函数是一个累积函数，它描述了随机变量在某个值以下的概率。累积分布函数的定义为：

$$
F(x) = P(X \leq x)
$$

其中，$X$ 是随机变量，$x$ 是随机变量的一个值。

## 3.3期望

期望是一个数学期望，它描述了随机变量的平均值。期望的定义为：

$$
E[X] = \int_{-\infty}^{\infty} x f(x) dx
$$

其中，$f(x)$ 是随机变量的概率密度函数。

## 3.4方差

方差是一个数学概念，它描述了随机变量的离散程度。方差的定义为：

$$
Var[X] = E[(X - E[X])^2]
$$

其中，$E[X]$ 是随机变量的期望。

```python
import numpy as np

def pdf(x, mu, sigma):
    return 1 / (np.sqrt(2 * np.pi * sigma**2)) * np.exp(-(x - mu)**2 / (2 * sigma**2))

x = np.linspace(-10, 10, 1000)
mu = 0
sigma = 1

plt.plot(x, pdf(x, mu, sigma))
plt.xlabel('x')
plt.ylabel('f(x)')
plt.title('Probability Density Function')
plt.show()
```

## 4.2累积分布函数

```python
def cdf(x, mu, sigma):
    return 1 / 2 * (1 + np.erf((x - mu) / np.sqrt(2 * sigma**2)))

x = np.linspace(-10, 10, 1000)
mu = 0
sigma = 1

plt.plot(x, cdf(x, mu, sigma))
plt.xlabel('x')
plt.ylabel('F(x)')
plt.title('Cumulative Distribution Function')
plt.show()
```
