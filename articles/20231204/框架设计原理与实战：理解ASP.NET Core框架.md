                 

# 1.背景介绍

ASP.NET Core是一种开源的、高性能的、模块化的、可扩展的、跨平台的。NET框架，它是Microsoft公司推出的一款用于构建Web应用程序、Web API、RESTful API以及实时应用程序的框架。ASP.NET Core框架的核心组件是Kestrel服务器，它是一个高性能、轻量级的Web服务器，可以运行在Windows、macOS和Linux等操作系统上。

ASP.NET Core框架的设计理念是“简单且高性能”，它采用了模块化设计，使得开发人员可以根据需要选择性地包含所需的组件，从而实现更轻量级的应用程序。此外，ASP.NET Core框架还支持跨平台开发，使得开发人员可以使用Windows、macOS和Linux等操作系统上的开发工具来开发和部署应用程序。

ASP.NET Core框架的核心组件包括：

- Kestrel服务器：一个高性能、轻量级的Web服务器，可以运行在Windows、macOS和Linux等操作系统上。
- ASP.NET Core引擎：一个用于处理HTTP请求和响应的引擎，它提供了一系列的中间件组件，以便开发人员可以根据需要选择性地包含所需的组件。
- 依赖注入容器：一个用于管理应用程序中的依赖关系的容器，它可以帮助开发人员更容易地实现模块化的应用程序设计。
- 数据访问层：一个用于访问数据库的组件，它提供了一系列的数据访问技术，如Entity Framework Core、Dapper等。
- 身份验证和授权：一个用于实现身份验证和授权的组件，它提供了一系列的身份验证技术，如OAuth2.0、OpenID Connect等。
