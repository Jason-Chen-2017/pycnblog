# 4.具体代码实例和详细解释说明

## 4.1 字符编码转换

```python
import codecs

def convert_encoding(input_file, output_file, input_encoding, output_encoding):
    with open(input_file, 'r', encoding=input_encoding) as f_in:
        with open(output_file, 'w', encoding=output_encoding) as f_out:
            for line in f_in:
                f_out.write(line)

convert_encoding('input.txt', 'output.txt', 'utf-8', 'gbk')
```

上述代码实例是一个字符编码转换的示例，它使用Python的codecs模块实现了从UTF-8到GBK的转换。具体操作步骤包括打开输入文件、读取文件内容、打开输出文件、写入文件内容等方面。

## 4.3 文本处理

```python
import re

def text_processing(text):
    pattern = r'\d+'
    numbers = re.findall(pattern, text)
    return numbers

text = 'The numbers are 1, 2, 3, 4, 5.'
print(text_processing(text))
```

上述代码实例是一个文本处理的示例，它使用Python的正则表达式模块实现了从文本中提取数字。具体操作步骤包括正则表达式的定义、文本的匹配、匹配结果的提取等方面。
