# 2.核心概念与联系

在流式数据处理中，我们需要了解以下几个核心概念：数据流、流处理框架和流处理算法。

## 2.1 数据流

数据流是一种特殊类型的数据，它是一种持续的数据序列，数据以流动的方式通过系统进行处理。数据流可以是来自不同来源的，例如文件、网络、传感器等。数据流可以是有序的，也可以是无序的，它们可以包含各种类型的数据，如文本、图像、音频、视频等。

## 2.2 流处理框架

流处理框架是一种软件框架，它提供了一种抽象方法来处理数据流。流处理框架通常包含以下几个组件：

- 数据源：用于读取数据流的组件。
- 数据接收器：用于写入数据流的组件。
- 数据处理器：用于对数据流进行处理的组件。
- 控制器：用于管理数据流处理过程的组件。

流处理框架提供了一种统一的方法来处理数据流，使得开发人员可以更容易地实现流式数据处理任务。

## 2.3 流处理算法

流处理算法是一种用于处理数据流的算法。流处理算法可以包括各种类型的算法，如过滤算法、转换算法、聚合算法等。这些算法可以用于对数据流进行各种操作，如筛选、转换、聚合等。
