例子：编写一个程序，输出1到100之间的所有奇数。

首先，我们需要定义一个函数来判断一个数是否为奇数。

```python
def is_odd(n):
    return n % 2 != 0
```

然后，我们需要遍历1到100之间的所有数，并判断是否为奇数。

```python
for i in range(1, 101):
    if is_odd(i):
        print(i)
```

在这个例子中，我们首先定义了一个名为`is_odd`的函数，它接受一个整数参数`n`，并返回一个布尔值，表示`n`是否为奇数。然后，我们使用`for`循环遍历1到100之间的所有数，并调用`is_odd`函数来判断是否为奇数。如果是奇数，则输出该数。
