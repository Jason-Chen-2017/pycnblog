# 2.核心概念与联系
## 2.1人脸识别与生物识别的区别与联系
人脸识别和生物识别是两种不同的识别技术，它们的主要区别在于所采用的识别方法和特征。人脸识别技术主要基于人脸图像的分析，通过对人脸图像进行处理和特征提取，从而识别人脸。生物识别技术则主要基于生物特征，如指纹、生物特征等，通过对生物特征进行比较和分析，从而识别人。

人脸识别与生物识别的联系在于它们都是一种识别技术，用于识别人或生物。它们的共同点在于它们都需要对特定的特征进行分析和比较，以确定是否是同一人或生物。

## 2.2人脸识别的核心概念
人脸识别技术的核心概念包括：
- 人脸图像处理：人脸图像处理是指对人脸图像进行预处理、增强、分割等操作，以提取人脸特征。
- 人脸特征提取：人脸特征提取是指从人脸图像中提取出人脸的关键特征，如眼睛、鼻子、嘴巴等。
- 人脸特征比较：人脸特征比较是指将提取出的人脸特征与数据库中的人脸特征进行比较，以确定是否是同一人。

## 2.3生物识别的核心概念
生物识别技术的核心概念包括：
- 生物特征采集：生物特征采集是指从生物体上采集生物特征，如指纹、生物特征等。
- 生物特征提取：生物特征提取是指从生物特征中提取出关键特征，如指纹模板、生物特征描述等。
- 生物特征比较：生物特征比较是指将提取出的生物特征与数据库中的生物特征进行比较，以确定是否是同一生物。
