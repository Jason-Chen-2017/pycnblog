# 3.3 因果推断的数学模型公式详细讲解
因果推断的数学模型公式详细讲解如下：

1. 条件概率：条件概率是一种描述事件发生的概率的方法。条件概率可以用公式P(A|B)表示，其中P(A|B)表示事件A发生的概率，给定事件B发生。

2. 条件独立：条件独立是一种描述事件之间关系的方法。条件独立可以用公式P(A∩B|C)=P(A|C)×P(B|C)表示，其中P(A∩B|C)表示事件A和事件B同时发生的概率，给定事件C发生；P(A|C)表示事件A发生的概率，给定事件C发生；P(B|C)表示事件B发生的概率，给定事件C发生。

3. 因果关系：因果关系是一种描述事件之间关系的方法。因果关系可以用公式P(A|B)=P(A)表示，其中P(A|B)表示事件A发生的概率，给定事件B发生；P(A)表示事件A发生的概率。
