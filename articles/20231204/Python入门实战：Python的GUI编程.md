## 1.4 Python的GUI编程具体代码实例和详细解释说明
以下是一个简单的Python的GUI编程代码实例，它创建了一个简单的窗口和一个按钮：

```python
import tkinter as tk

# 创建窗口
window = tk.Tk()
window.title("Python GUI编程")
window.geometry("300x200")

# 创建按钮
button = tk.Button(window, text="点击我")
button.pack()

# 显示窗口
window.mainloop()
```

在这个代码实例中，我们首先导入了Tkinter库，然后创建了一个窗口对象，设置了窗口的标题和大小。接着，我们创建了一个按钮对象，设置了按钮的文本。最后，我们添加了按钮到窗口，并显示了窗口。
