Redis（Remote Dictionary Server，远程字典服务器）是一个开源的高性能的键值存储系统，它支持数据的持久化，可基于内存（in-memory）也可基于磁盘。Redis 提供多种语言的 API，包括：Ruby、Python、Java、C 和 C#，因此可以在任何语言中使用 Redis。

Redis 是一个使用 ANSI C 语言编写、遵循 BSD 协议的开源软件（ BSD 协议是一种进步的开源协议，允许软件可以被用于任何目的，包括商业用途，而不需要提供源代码或者对修改后的软件的任何限制）。Redis 的设计和使用 ANSI C 语言编写，使得 Redis 能够在许多平台上运行，包括 Windows、macOS、Linux、FreeBSD、OpenBSD、NetBSD、Solaris 和 AIX。
