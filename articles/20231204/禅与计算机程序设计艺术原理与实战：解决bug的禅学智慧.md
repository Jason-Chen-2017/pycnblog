## 4.1 代码实例

我们来看一个简单的代码实例，这个代码是一个计算两个数之和的程序：

```python
def add(a, b):
    return a + b

result = add(1, 2)
print(result)
```

在这个代码中，我们定义了一个名为`add`的函数，它接受两个参数`a`和`b`，并返回它们之和。然后我们调用这个函数，并将结果打印出来。

## 4.2 发现bug

当我们运行这个代码时，我们发现结果并不是我们预期的结果。我们发现，当我们将两个数相加时，结果并不是我们预期的结果。这是因为我们在计算两个数之和时，没有考虑到数值溢出的问题。

## 4.3 解决bug

为了解决这个bug问题，我们可以根据禅学的智慧，来修改代码。我们可以使用Python的`math.fsum`函数来计算两个数之和，这个函数可以避免数值溢出的问题。

修改后的代码如下：

```python
import math

def add(a, b):
    return math.fsum(a, b)

result = add(1, 2)
print(result)
```

在这个修改后的代码中，我们导入了`math`模块，并使用`math.fsum`函数来计算两个数之和。这样，我们就可以避免数值溢出的问题，从而解决bug问题。
