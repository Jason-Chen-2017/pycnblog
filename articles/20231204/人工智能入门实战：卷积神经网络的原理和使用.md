# 2.核心概念与联系
卷积神经网络（Convolutional Neural Networks，CNN）是一种深度学习模型，主要应用于图像分类、目标检测和语音识别等任务。CNN的核心概念包括卷积层、激活函数、池化层和全连接层。

卷积层（Convolutional Layer）：卷积层是CNN的核心组成部分，用于自动学习图像的特征。卷积层通过卷积核对输入图像进行卷积操作，以提取图像的特征。卷积核是一个小的矩阵，用于在图像中进行卷积操作。卷积层的输出通常会经过激活函数进行非线性变换，以增加模型的复杂性。

激活函数（Activation Function）：激活函数是神经网络中的一个重要组成部分，用于将卷积层的输出映射到一个更高维的空间。常用的激活函数包括sigmoid函数、ReLU函数和tanh函数等。激活函数的作用是将输入映射到一个更高维的空间，以增加模型的复杂性。

池化层（Pooling Layer）：池化层是CNN的另一个重要组成部分，用于对卷积层的输出进行下采样，以减少计算量和防止过拟合。池化层通常使用最大池化或平均池化来实现下采样。池化层的输出通常会经过全连接层进行分类。

全连接层（Fully Connected Layer）：全连接层是CNN的最后一个组成部分，用于将卷积层的输出转换为最终的输出结果。全连接层通过将卷积层的输出作为输入，并使用一组权重和偏置来进行线性变换，以生成最终的输出结果。

卷积神经网络的核心概念包括卷积层、激活函数、池化层和全连接层。卷积层用于自动学习图像的特征，激活函数用于将卷积层的输出映射到一个更高维的空间，池化层用于对卷积层的输出进行下采样，以减少计算量和防止过拟合，全连接层用于将卷积层的输出转换为最终的输出结果。
