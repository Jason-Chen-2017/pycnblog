# 2.核心概念与联系

1. 状态（State）：表示环境的当前状态，是深度强化学习中的一种观测。
2. 动作（Action）：表示环境中可以执行的操作，是深度强化学习中的一种行为。
3. 奖励（Reward）：表示环境给出的反馈，是深度强化学习中的一种评价。

6. Q值（Q-Value）：表示在给定状态和动作下取得期望奖励的函数，是深度强化学习中的一种评估。
