```go
package main

import "fmt"

func main() {
    // 错误处理示例
    err := divide(10, 0)
    if err != nil {
        fmt.Println("错误信息:", err)
    } else {
        fmt.Println("成功")
    }

    // 异常处理示例
    defer func() {
        if err := recover(); err != nil {
            fmt.Println("异常信息:", err)
        }
    }()
    divide(10, 0)
}

func divide(a, b int) error {
    if b == 0 {
        return fmt.Errorf("除数不能为0")
    }
    return nil
}
```

接下来，我们定义了一个异常处理的示例，函数divide()接收两个整数参数a和b，如果b为0，则触发一个异常信息"除数不能为0"，否则返回nil。在主函数中，我们使用defer关键字来注册一个匿名函数，该函数在divide()函数返回后被调用。如果在divide()函数中触发了异常信息，我们使用recover()函数捕获异常信息，并使用fmt.Println()函数打印异常信息。
