# 2.核心概念与联系

在实时计算平台的设计和实现过程中，我们需要了解以下几个核心概念：

1.数据流计算：数据流计算是一种基于数据流的计算模型，它允许我们以流式方式处理数据。数据流计算可以实现高性能的实时计算，因为它可以在数据到达时进行处理，而不需要等待所有数据都到达。

2.数据分区：数据分区是一种将数据划分为多个部分的方法，以便在多个计算节点上并行处理。数据分区可以提高实时计算平台的性能，因为它可以让多个计算节点同时处理数据。

3.流处理框架：流处理框架是一种用于实现数据流计算的软件框架。流处理框架可以提供一系列的API和工具，以便我们可以更容易地实现实时计算。
