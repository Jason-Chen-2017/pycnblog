- 冒泡排序：
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```
这个代码实现了冒泡排序算法，通过比较相邻的元素并交换它们来实现数据的排序。

- 二分搜索：
```python
def binary_search(arr, target):
    low, high = 0, len(arr)-1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```
这个代码实现了二分搜索算法，通过递归地将问题分解为子问题，然后回溯来找到满足条件的元素。

- 快速幂：
```python
def quick_pow(x, n):
    if n == 0:
        return 1
    elif n % 2 == 0:
        return quick_pow(x * x, n // 2)
    else:
        return x * quick_pow(x * x, (n - 1) // 2)
```
这个代码实现了快速幂算法，通过将问题分解为子问题，然后递归地解决子问题，最后将子问题的解合并为原问题的解。
