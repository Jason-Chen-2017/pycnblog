# 4.具体代码实例和详细解释说明

## 4.1量子门实现

```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile

# 创建量子门
qc = QuantumCircuit(2)

# 设置量子门类型
qc.h(0)  # 单位门
qc.rx(np.pi/2, 0)  # 旋转门
qc.cx(0, 1)  # 控制门

# 执行量子门操作
qc.draw()

# 读取量子粒子的状态
result = qc.get_statevector()
print(result)
```
