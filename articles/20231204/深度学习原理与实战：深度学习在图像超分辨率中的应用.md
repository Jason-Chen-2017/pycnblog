                 

# 1.背景介绍

图像超分辨率是一种通过对低分辨率图像进行处理，将其转换为高分辨率图像的技术。这种技术在各种应用场景中都有广泛的应用，例如视频播放、图像识别、图像增强等。深度学习在图像超分辨率技术中发挥着越来越重要的作用，主要是因为深度学习的强大表达能力和自动学习能力。

在本文中，我们将从以下几个方面来讨论深度学习在图像超分辨率中的应用：

1. 背景介绍
2. 核心概念与联系

6. 附录常见问题与解答

# 2.核心概念与联系

在深度学习中，图像超分辨率主要包括两个方面：

1. 生成模型：生成模型主要是通过学习低分辨率图像和高分辨率图像之间的关系，生成高分辨率图像。常见的生成模型有VGG, ResNet, DenseNet等。

2. 恢复模型：恢复模型主要是通过学习低分辨率图像的特征，然后将其恢复为高分辨率图像。常见的恢复模型有SRResNet, EDSR, RCAN等。
