```python
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```

在这个代码实例中，我们实现了一个递归的Fibonacci函数。函数接收一个整数参数n，并返回斐波那契数列的第n个数。

在这个函数中，我们使用了递归的方法来解决问题。当n小于等于0时，我们返回0；当n等于1时，我们返回1；否则，我们递归调用fibonacci函数，并将结果加在一起。
