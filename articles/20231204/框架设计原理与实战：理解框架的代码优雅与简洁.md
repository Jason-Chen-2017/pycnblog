假设我们正在设计一个简单的图书管理系统框架。我们需要定义一些基本的组件，如书籍、作者、出版社等。这些组件可以被组合和扩展以满足不同的需求。

以下是一个简单的代码实例：

```python
class Book:
    def __init__(self, title, author, publisher):
        self.title = title
        self.author = author
        self.publisher = publisher

class Author:
    def __init__(self, name):
        self.name = name

class Publisher:
    def __init__(self, name):
        self.name = name
```

在这个例子中，我们定义了三个基本的组件：Book、Author和Publisher。这些组件可以被组合和扩展以满足不同的需求。

例如，我们可以创建一个新的Book实例，并将其与Author和Publisher实例相关联：

```python
book = Book("The Great Gatsby", Author("F. Scott Fitzgerald"), Publisher("Scribner"))
```

```python
class Book:
    def __init__(self, title, author, publisher):
        self.title = title
        self.author = author
        self.publisher = publisher

    def get_info(self):
        return f"Title: {self.title}, Author: {self.author.name}, Publisher: {self.publisher.name}"
```
