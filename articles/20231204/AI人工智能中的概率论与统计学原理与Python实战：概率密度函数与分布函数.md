## 4.1概率密度函数的代码实例

在这个例子中，我们将使用Python的numpy库来计算一个正态分布的概率密度函数。

```python
import numpy as np

def normal_pdf(x, mean, std_dev):
    return 1 / (std_dev * np.sqrt(2 * np.pi)) * np.exp(-(x - mean)**2 / (2 * std_dev**2))

x = np.linspace(-10, 10, 100)
mean = 0
std_dev = 1

pdf = normal_pdf(x, mean, std_dev)

plt.plot(x, pdf)
plt.xlabel('x')
plt.ylabel('pdf(x)')
plt.title('Normal PDF')
plt.show()
```

在这个例子中，我们首先定义了一个正态分布的概率密度函数。然后，我们使用numpy的linspace函数来生成一个区间，然后使用我们定义的概率密度函数来计算该区间内的概率密度。最后，我们使用matplotlib来绘制概率密度函数的图像。

## 4.2分布函数的代码实例

在这个例子中，我们将使用Python的numpy库来计算一个正态分布的分布函数。

```python
import numpy as np

def normal_cdf(x, mean, std_dev):
    return 0.5 * (1 + np.erf((x - mean) / (std_dev * np.sqrt(2))))

x = np.linspace(-10, 10, 100)
mean = 0
std_dev = 1

cdf = normal_cdf(x, mean, std_dev)

plt.plot(x, cdf)
plt.xlabel('x')
plt.ylabel('cdf(x)')
plt.title('Normal CDF')
plt.show()
```

在这个例子中，我们首先定义了一个正态分布的分布函数。然后，我们使用numpy的linspace函数来生成一个区间，然后使用我们定义的分布函数来计算该区间内的概率。最后，我们使用matplotlib来绘制分布函数的图像。
