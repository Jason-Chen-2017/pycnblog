# 2.核心概念与联系

在本节中，我们将介绍线性回归和逻辑回归的核心概念，并讨论它们之间的联系。

## 2.1 线性回归

线性回归是一种简单的预测模型，它可以用来预测连续型变量。线性回归模型的基本形式如下：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n + \epsilon
$$

其中，$y$ 是预测值，$x_1, x_2, \cdots, x_n$ 是输入变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数，$\epsilon$ 是误差项。

线性回归的目标是找到最佳的参数值，使得预测值与实际值之间的差异最小。这个过程可以通过最小二乘法来实现。

## 2.2 逻辑回归

逻辑回归是一种分类模型，它可以用来预测离散型变量。逻辑回归模型的基本形式如下：

$$
P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + \cdots + \beta_nx_n)}}
$$

其中，$y$ 是预测值，$x_1, x_2, \cdots, x_n$ 是输入变量，$\beta_0, \beta_1, \beta_2, \cdots, \beta_n$ 是参数。

逻辑回归的目标是找到最佳的参数值，使得预测值与实际值之间的差异最小。这个过程可以通过梯度下降法来实现。

## 2.3 联系

线性回归和逻辑回归都是通过最小化损失函数来找到最佳参数值的。线性回归的损失函数是均方误差，而逻辑回归的损失函数是交叉熵。

$$
J(\beta_0, \beta_1, \cdots, \beta_n) = \sum_{i=1}^m (y_i - (\beta_0 + \beta_1x_{i1} + \beta_2x_{i2} + \cdots + \beta_nx_{in}))^2
$$

要找到这个最小值，我们可以使用梯度下降法。梯度下降法的基本思想是：从一个参数值开始，逐步更新参数值，使目标函数的梯度逐渐减小。

1. 初始化参数值：$\beta_0, \beta_1, \cdots, \beta_n$
2. 计算目标函数的梯度：$\frac{\partial J}{\partial \beta_0}, \frac{\partial J}{\partial \beta_1}, \cdots, \frac{\partial J}{\partial \beta_n}$

## 6.1 线性回归

### 6.1.1 问题：线性回归的优缺点是什么？

答案：线性回归的优点是简单易用，计算成本较低。线性回归的缺点是只能处理线性关系，对非线性关系不适用。

### 6.1.2 问题：线性回归的应用场景是什么？

答案：线性回归的应用场景包括预测房价、预测股票价格、预测销售额等。

## 6.2 逻辑回归

### 6.2.1 问题：逻辑回归的优缺点是什么？

答案：逻辑回归的优点是可以处理离散型变量，适用于二分类问题。逻辑回归的缺点是只能处理线性关系，对非线性关系不适用。

### 6.2.2 问题：逻辑回归的应用场景是什么？

答案：逻辑回归的应用场景包括邮件筛选、垃圾邮件分类、欺诈检测等。
