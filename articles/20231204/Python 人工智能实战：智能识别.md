1. 灰度化：将彩色图像转换为灰度图像，即将图像中的每个像素点的颜色信息转换为灰度值。
2. 二值化：将灰度图像转换为二值图像，即将图像中的每个像素点的灰度值转换为黑白两种。
3. 腐蚀和膨胀：通过对图像进行腐蚀和膨胀操作，消除图像中的噪声和边缘。
4. 平滑：通过对图像进行平滑操作，消除图像中的光照差异。
