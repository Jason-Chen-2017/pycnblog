# 4.具体代码实例和详细解释说明

## 4.1 Goroutine的创建和管理

```go
package main

import "fmt"

func main() {
    // 创建一个 Goroutine
    go func() {
        fmt.Println("Hello, World!")
    }()

    // 等待 Goroutine 完成
    fmt.Scanln()
}
```

在这个例子中，我们创建了一个 Goroutine，它会打印 "Hello, World!" 并等待用户输入。

## 4.2 通道的创建和管理

```go
package main

import "fmt"

func main() {
    // 创建一个通道
    ch := make(chan int)

    // 创建一个 Goroutine，向通道发送数据
    go func() {
        ch <- 42
    }()

    // 等待 Goroutine 完成
    fmt.Scanln()

    // 从通道读取数据
    fmt.Println(<-ch)
}
```

在这个例子中，我们创建了一个通道，并创建了一个 Goroutine，向通道发送数据。然后，我们从通道中读取数据。

## 4.3 Goroutine之间的通信和同步

```go
package main

import "fmt"

func main() {
    // 创建一个通道
    ch := make(chan int)

    // 创建两个 Goroutine，分别向通道发送数据
    go func() {
        ch <- 1
    }()

    go func() {
        ch <- 2
    }()

    // 等待 Goroutine 完成
    fmt.Scanln()

    // 从通道读取数据
    fmt.Println(<-ch)
}
```

在这个例子中，我们创建了一个通道，并创建了两个 Goroutine，分别向通道发送数据。然后，我们从通道中读取数据。
