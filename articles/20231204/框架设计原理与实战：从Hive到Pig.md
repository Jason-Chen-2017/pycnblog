## 4.1 Hive代码实例

Hive使用类SQL的查询语言，用户可以通过编写简单的SQL查询来处理大量数据。以下是一个Hive代码实例：

```sql
CREATE TABLE student (
    id INT,
    name STRING,
    age INT
);

INSERT INTO TABLE student VALUES (1, 'Alice', 20);
INSERT INTO TABLE student VALUES (2, 'Bob', 21);
INSERT INTO TABLE student VALUES (3, 'Charlie', 22);

SELECT * FROM student WHERE age > 20;
```

在这个例子中，我们首先创建了一个名为student的表，它有三个列：id、name和age。然后我们插入了三条记录到这个表中。最后，我们使用SELECT语句来查询年龄大于20的学生信息。

## 4.2 Pig代码实例

Pig是一个高级数据流处理语言，它提供了一种易于使用的数据流处理模型，用户可以通过编写简单的Pig语句来处理大量数据。以下是一个Pig代码实例：

```pig
student = LOAD 'student.txt' AS (id:int, name:chararray, age:int);
filtered_student = FILTER student BY age > 20;
DUMP filtered_student;
```

在这个例子中，我们首先使用LOAD命令加载了一个名为student.txt的文件，并将其加载到一个名为student的关系中。然后我们使用FILTER命令来筛选年龄大于20的学生信息。最后，我们使用DUMP命令来输出筛选结果。
