# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的Python脚本编程实例来详细解释其工作原理。

```python
# 创建一个名为script.py的Python脚本文件
# 编写代码，实现读取文件内容并输出到控制台的功能

# 打开文件
with open('example.txt', 'r') as file:
    # 读取文件内容
    content = file.read()
    # 输出文件内容
    print(content)
```

在上述代码中，我们首先创建了一个名为`script.py`的Python脚本文件。然后，我们编写了Python代码，实现了读取文件内容并输出到控制台的功能。

具体的操作步骤如下：

1. 使用`with`语句打开文件`example.txt`，并指定文件打开模式为读取模式（`'r'`）。
2. 使用`file.read()`方法读取文件内容，并将其存储到`content`变量中。
3. 使用`print()`函数输出文件内容到控制台。
