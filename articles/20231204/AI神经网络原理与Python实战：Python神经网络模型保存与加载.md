# 2.核心概念与联系

在本节中，我们将介绍一些核心概念，包括神经网络、神经元、权重、偏置、损失函数、梯度下降等。这些概念是构建神经网络的基础，理解它们对于理解神经网络模型的保存和加载至关重要。

## 2.1 神经网络

神经网络是一种由多个节点（神经元）组成的计算模型，这些节点之间有权重和偏置。神经网络的核心思想是通过对大量数据的训练，使网络能够学习出如何对新的输入数据进行预测或分类。

## 2.2 神经元

神经元是神经网络的基本组成单元，它接收输入，进行计算，并输出结果。神经元通常包括一个输入层、一个隐藏层和一个输出层。输入层接收输入数据，隐藏层进行计算，输出层输出预测结果。

## 2.3 权重

权重是神经网络中的一个重要参数，它用于控制神经元之间的连接强度。权重可以通过训练来调整，以便使网络能够更好地预测输入数据。

## 2.4 偏置

偏置是神经网络中的另一个重要参数，它用于调整神经元的输出。偏置可以通过训练来调整，以便使网络能够更好地预测输入数据。

## 2.5 损失函数

损失函数是用于衡量神经网络预测结果与实际结果之间的差异的函数。损失函数的值越小，表示网络的预测结果越接近实际结果。损失函数是训练神经网络的一个重要组成部分。

## 2.6 梯度下降

梯度下降是一种优化算法，用于优化神经网络的权重和偏置。梯度下降算法通过计算损失函数的梯度，以便找到权重和偏置的最佳值。
