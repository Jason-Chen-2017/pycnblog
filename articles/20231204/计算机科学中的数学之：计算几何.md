计算几何的核心算法原理包括点包、线段包、凸包等，这些算法的原理和具体操作步骤以及数学模型公式详细讲解如下：

1. 点包：点包问题是计算几何中的一个基本问题，它要求找出所有与给定凸包的交点。点包问题的核心算法是凸包算法，凸包算法的原理是利用凸包的性质，将所有点按照顺时针或逆时针顺序排列，然后从左到右依次判断是否属于凸包。具体操作步骤如下：

   1. 首先，将所有点按照顺时针或逆时针顺序排列。
   2. 然后，从左到右依次判断每个点是否属于凸包。
   3. 如果当前点与前两个点构成的三角形的内角小于180度，则当前点属于凸包。
   4. 如果当前点与前两个点构成的三角形的内角大于180度，则当前点不属于凸包。
   5. 重复上述步骤，直到所有点都判断完成。

2. 线段包：线段包问题是计算几何中的一个基本问题，它要求找出所有与给定凸包的交线段。线段包问题的核心算法是凸包算法，凸包算法的原理是利用凸包的性质，将所有线段按照顺时针或逆时针顺序排列，然后从左到右依次判断是否属于凸包。具体操作步骤如下：

   2. 然后，从左到右依次判断每个线段是否属于凸包。
   3. 如果当前线段与前两个线段构成的三角形的内角小于180度，则当前线段属于凸包。
   4. 如果当前线段与前两个线段构成的三角形的内角大于180度，则当前线段不属于凸包。

   在实际应用中，可以使用Python的NumPy库来实现计算几何的核心算法。以下是一个简单的例子：

   ```python
   import numpy as np

   def is_convex(points):
       n = len(points)
       for i in range(n):
           j = (i + 1) % n
           k = (i + 2) % n
           a = points[j] - points[i]
           b = points[k] - points[i]
           if np.dot(a, b) < 0:
               return False
       return True

   points = np.array([[1, 1], [2, 2], [3, 3], [4, 4]])
   print(is_convex(points))
   ```
