# 2.核心概念与联系

## 2.1 缓存的基本概念

缓存是计算机科学中一个非常基本的概念，它的核心思想是通过将经常访问的数据存储在内存中，以便快速访问，从而提高系统性能。缓存可以分为两种类型：内存缓存和磁盘缓存。内存缓存是指将数据存储在内存中，以便快速访问；磁盘缓存是指将数据存储在磁盘中，以便快速访问。

## 2.2 分布式缓存的基本概念

分布式缓存是指将缓存数据存储在多个不同的服务器上，以便在多个服务器之间共享数据。这样可以实现数据的高可用性、高性能和高可扩展性。分布式缓存可以分为两种类型：集中式分布式缓存和分布式分布式缓存。集中式分布式缓存是指将所有的缓存数据存储在一个服务器上，然后通过网络访问其他服务器；分布式分布式缓存是指将缓存数据存储在多个服务器上，并通过网络之间进行数据共享。

## 2.3 Memcached的基本概念

Memcached是一个开源的分布式缓存系统，它的设计目标是提供高性能、高可用性和高可扩展性的缓存服务。Memcached使用内存作为缓存存储，并通过网络与其他服务器进行数据共享。Memcached的核心组件包括：客户端、服务器和协议。客户端是用于与Memcached服务器进行通信的组件，服务器是用于存储和管理缓存数据的组件，协议是用于定义Memcached客户端和服务器之间通信的规范。
