## 3.1 选择排序算法

选择排序算法是一种简单的排序算法，它的工作原理是：

1. 首先从数组中找到最小的元素，并将其与数组的第一个元素交换。

2. 然后从剩下的元素中找到最小的元素，并将其与数组的第二个元素交换。

3. 重复上述步骤，直到整个数组被排序。

# 4.具体代码实例和详细解释说明

在这一部分，我们将通过一个具体的代码实例来详细解释选择排序算法的实现过程。

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

arr = [5, 2, 8, 1, 9]
print(selection_sort(arr))
```

在上述代码中，我们首先定义了一个 selection_sort 函数，该函数接收一个数组作为参数，并返回排序后的数组。

在函数内部，我们首先获取数组的长度，并使用两个 for 循环来实现选择排序算法的具体操作。

第一个 for 循环用于遍历数组的每一个元素，并找到当前元素之后的最小元素。

第二个 for 循环用于找到当前元素之后的最小元素，并将其与当前元素交换。

最后，我们调用 selection_sort 函数，并将排序后的数组打印出来。
