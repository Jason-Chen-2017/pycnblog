数据存储的代码实例如下：
```python
import sqlite3
import pandas as pd

# 创建数据库连接
conn = sqlite3.connect('data_storage.db')

# 创建数据表
conn.execute('''CREATE TABLE data_storage
                (id INTEGER PRIMARY KEY, data TEXT)''')

# 插入数据
data = {'id': [1, 2, 3], 'data': ['数据资源1', '数据资源2', '数据资源3']}
df = pd.DataFrame(data)
df.to_sql('data_storage', conn, if_exists='append', index=False)

# 查询数据
query = 'SELECT * FROM data_storage'
result = pd.read_sql_query(query, conn)
print(result)

# 关闭数据库连接
conn.close()
```

数据处理的代码实例如下：
```python
import pandas as pd

# 读取数据
data = pd.read_csv('data_storage.csv')

# 数据清洗
data = data.drop_duplicates()

# 数据转换
data['data'] = data['data'].str.upper()

# 数据整合
data_integrated = data.groupby('id').agg({'data': 'first'})

# 保存整合后的数据
data_integrated.to_csv('data_integrated.csv', index=False)
```
