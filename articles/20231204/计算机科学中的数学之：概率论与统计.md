# 2.核心概念与联系

在概率论与统计中，有几个核心概念需要我们理解：

1.随机变量：随机变量是一个随机事件的数值表现形式，它可以用来描述随机事件的不确定性。随机变量可以是离散的（如掷骰子的点数）或连续的（如体温）。

2.概率：概率是一个随机事件发生的可能性，它通常用0到1之间的一个数值表示。概率可以用来描述随机事件的发生概率，从而帮助我们做出更明智的决策。

3.期望：期望是一个随机变量的数学期望，它表示随机变量的平均值。期望可以用来描述随机变量的中心趋势，从而帮助我们更好地理解数据的分布。

4.方差：方差是一个随机变量的数学方差，它表示随机变量的离散程度。方差可以用来描述随机变量的分布程度，从而帮助我们更好地理解数据的不确定性。

5.条件概率：条件概率是一个随机事件发生的可能性，给定另一个随机事件已经发生的情况下。条件概率可以用来描述随机事件之间的关系，从而帮助我们更好地进行决策和分析。
