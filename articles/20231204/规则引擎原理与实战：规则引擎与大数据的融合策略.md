## 4.1 规则编写

```python
# 定义一个规则
rule1 = Rule(
    condition="age > 18",
    action="grant_access"
)

# 定义一个规则集
rule_set = RuleSet([rule1])
```

在这个例子中，我们定义了一个规则`rule1`，其中`condition`表示规则的条件，`action`表示规则的操作。然后我们将这个规则添加到了一个规则集`rule_set`中。

## 4.2 规则编译

```python
# 编译规则集
compiled_rule_set = rule_set.compile()
```

在这个例子中，我们通过`compile()`方法将规则集编译成可执行代码。

## 4.3 规则执行

```python
# 执行规则集
result = compiled_rule_set.execute(data)
```

在这个例子中，我们通过`execute()`方法将规则集执行，并将执行结果存储在`result`变量中。
