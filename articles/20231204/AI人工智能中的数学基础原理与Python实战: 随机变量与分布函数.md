## 3.1 均匀分布
均匀分布是一种常见的概率分布，用于描述随机变量在一个固定范围内的概率分布。均匀分布的概率密度函数为：

$$
f(x) = \frac{1}{b-a}
$$

其中，$a$ 和 $b$ 是均匀分布的范围，$x$ 是随机变量的取值。

## 3.2 指数分布
指数分布是一种常见的概率分布，用于描述随机变量的寿命。指数分布的概率密度函数为：

$$
f(x) = \frac{1}{\lambda} e^{-\frac{x}{\lambda}}
$$

其中，$\lambda$ 是指数分布的参数，$x$ 是随机变量的取值。

## 3.3 正态分布
正态分布是一种常见的概率分布，用于描述随机变量的取值。正态分布的概率密度函数为：

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$\mu$ 是正态分布的均值，$\sigma$ 是正态分布的标准差，$x$ 是随机变量的取值。

# 4.具体代码实例和详细解释说明
在Python中，可以使用numpy库来实现随机变量的概率分布。以下是一个Python代码实例，用于生成均匀分布、指数分布和正态分布的随机变量：

```python
import numpy as np

# 均匀分布
a = 0
b = 10
x = np.random.uniform(a, b, 1000)
print(x)

# 指数分布
lambda = 1
x = np.random.exponential(lambda, 1000)
print(x)

# 正态分布
mu = 0
sigma = 1
x = np.random.normal(mu, sigma, 1000)
print(x)
```

在上述代码中，我们使用numpy库的`uniform`、`exponential`和`normal`函数来生成均匀分布、指数分布和正态分布的随机变量。
