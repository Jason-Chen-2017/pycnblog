# 4.具体代码实例和详细解释说明

在这里，我们将提供一个简单的移动支付代码实例，以及对其详细解释。

```python
import hashlib
import rsa

# 生成密钥对
(public_key, private_key) = rsa.newkeys(1024)

# 生成支付信息
payment_info = "购买商品"

# 使用哈希函数生成支付信息的哈希值
hash_value = hashlib.sha256(payment_info.encode()).hexdigest()

# 使用公钥对支付信息的哈希值进行加密
encrypted_hash_value = rsa.encrypt(hash_value.encode(), public_key)

# 使用私钥对支付信息的哈希值进行解密
decrypted_hash_value = rsa.decrypt(encrypted_hash_value, private_key).decode()

# 验证支付信息的合法性
if decrypted_hash_value == hash_value:
    print("支付成功")
else:
    print("支付失败")
```

在这个代码实例中，我们使用了RSA算法生成了密钥对。然后，我们生成了一个支付信息，并使用哈希函数生成支付信息的哈希值。接着，我们使用公钥对支付信息的哈希值进行加密，并使用私钥对加密后的哈希值进行解密。最后，我们验证支付信息的合法性，并输出支付结果。
