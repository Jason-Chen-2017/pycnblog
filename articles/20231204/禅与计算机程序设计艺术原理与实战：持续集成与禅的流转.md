首先，我们创建一个Git仓库，并将代码上传到仓库中：

```
$ git init
$ git add .
$ git commit -m "初始提交"
```

然后，我们设置一个Jenkins的构建任务，并将其与Git仓库连接：

```
$ jenkins-cli.jar -s http://localhost:8080/ create-job Calculator -t 'git clone https://github.com/your-username/calculator.git'
$ jenkins-cli.jar -s http://localhost:8080/ build -j Calculator
```

接下来，我们编写一个简单的计算器程序：

```python
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    return x / y
```

然后，我们使用Git进行版本控制：

```
$ git add .
$ git commit -m "添加计算器功能"
```

最后，我们使用Jenkins进行自动化构建和测试：

