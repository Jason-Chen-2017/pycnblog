1.8.2 微积分：
```python
from scipy.integrate import odeint

# 定义微积分函数
def dydt(y, t):
    return [y[1], -y[0] - y[1]**2]

# 定义初始条件
y0 = [1, 0]

# 定义时间范围
t = np.linspace(0, 1, 100)

# 使用odeint函数来解决微积分问题
y = odeint(dydt, y0, t)

# 打印解决结果
print(y)
```

