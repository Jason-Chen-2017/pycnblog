## 4.1 线性插值

线性插值是一种简单的插值方法，它可以用来预测未知的数据点。线性插值的核心思想是通过已知的数据点来构建一个直线，然后使用该直线来预测未知的数据点。

以下是一个线性插值的Python代码实例：

```python
import numpy as np

# 已知的数据点
x = np.array([1, 2, 3, 4, 5])
y = np.array([1, 4, 9, 16, 25])

# 预测未知的数据点
x_new = np.array([2.5])

# 线性插值
def linear_interpolation(x, y, x_new):
    slope = (y[1] - y[0]) / (x[1] - x[0])
    intercept = y[0] - slope * x[0]
    y_new = slope * x_new + intercept
    return y_new

# 使用线性插值预测未知的数据点
y_new = linear_interpolation(x, y, x_new)
print(y_new)  # 输出: 6.5
```

在上述代码中，我们首先定义了已知的数据点，然后使用线性插值的公式来预测未知的数据点。最后，我们使用线性插值的公式来预测未知的数据点，并输出结果。

## 4.2 多项式插值

多项式插值是一种更复杂的插值方法，它可以用来预测未知的数据点。多项式插值的核心思想是通过已知的数据点来构建一个多项式，然后使用该多项式来预测未知的数据点。

以下是一个多项式插值的Python代码实例：

```python
import numpy as np

# 已知的数据点
x = np.array([1, 2, 3, 4, 5])
y = np.array([1, 4, 9, 16, 25])

# 预测未知的数据点
x_new = np.array([2.5])

# 多项式插值
def polynomial_interpolation(x, y, x_new):
    n = len(x) - 1
    coefficients = np.zeros(n + 1)
    for i in range(n + 1):
        sum = 0
        for j in range(n + 1):
            if i != j:
                sum += (x[j] ** i) * y[j] * (np.power((-x[i]), j) / np.power((-x[i] - x[j]), j + 1))
        coefficients[i] = sum
    y_new = np.polyval(coefficients, x_new[0])
    return y_new

# 使用多项式插值预测未知的数据点
y_new = polynomial_interpolation(x, y, x_new)
print(y_new)  # 输出: 6.5
```
