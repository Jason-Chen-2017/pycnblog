# 1.简介
  

近年来随着互联网、移动端、大数据等新兴技术的不断发展，各行各业都在用计算机解决各种问题，而计算机科学正成为各大高校必修课程之一。通过学习计算机科学，你可以掌握到以下技能：

1. 技术研究能力：理解并运用计算机科学理论、方法、工具及技术，从实际出发，提升个人能力；
2. 问题分析能力：熟练掌握计算机系统结构、运行机制、应用领域等知识，能够准确分析问题、找准根源，有效追踪、解决问题；
3. 创造力和团队精神：融合自己的理性和经验，创造性地提出解决方案，积极参与项目开发和管理，提升职场竞争力；
4. 沉浸式学习能力：深入浅出的学习方式，快速掌握各种知识和技能，享受学习过程中的快感。
当然，计算机科学还有其他很多优秀的用途，包括系统设计、安全防护、数据库管理、Web开发等。不同专业需要的计算机科学素养也不一样。对于一般的学生来说，掌握一些基础的计算机技术、算法和数据结构，足以应付日常生活中的各种需求，无需刻意钻研过多。

# 2.基本概念术语说明
# 1.数据结构与算法
数据结构（Data Structure）是指存储、组织和处理数据的形式、模式以及性能特点。主要分为抽象数据类型和非抽象数据类型两大类，抽象数据类型如栈、队列、堆、树等，非抽象数据类型如数组、链表、哈希表等。
算法（Algorithm）是指用来操作数据、计算的指令集、流程控制、计算规则或操作 procedure。它是数据处理的基石，也是数据结构的实现。它的功能是为了解决某个问题或效率高低。算法由若干个步骤组成，每一步都是对输入数据的一系列操作。

# 2.机器语言、汇编语言和高级语言
机器语言（Machine Language）是直接向处理器发出命令的低级编程语言，例如二进制码、机器指令等。汇编语言（Assembly Language）是人们可以阅读的、更易于编写和使用的高级编程语言，一般使用助记符表示机器指令。高级语言（High-Level Language），又称面向用户的编程语言，例如C、Java、Python等。高级语言编写的程序可被编译成汇编语言或者机器语言。机器语言、汇编语言与高级语言之间的关系可以简单的认为：汇编语言是机器语言的助记符，高级语言是汇编语言的扩展。

# 3.系统结构与运行原理
系统结构（System Structures）是指计算机硬件、软件、网络等构成的整体结构，包括系统总线、内存、处理机、外设接口、文件系统、操作系统等子系统。系统结构决定了系统能否正常工作。
系统运行原理（Operating Principles of Systems）描述了操作系统如何调度、分配资源、协同进程和线程、保障数据完整性、处理异常情况等。它涵盖了操作系统的功能、运行环境、管理、通信、同步、存储、文件系统、网络等方面的内容。

# 4.并发与并行
并发（Concurrency）是指两个或多个事件（程序、处理器、磁盘等）间发生交替执行，即两个或多个任务（程序）同时运行或同时进行。并行（Parallelism）则是在多核CPU上同时运行多个任务。

# 5.分布式与集群
分布式（Distributed）系统是指系统元素之间存在一种较强的耦合关系，因此通过增加更多的设备来扩展性能。分布式系统包括许多独立节点（Node）组成，每个节点都可以运行自己的操作系统和应用程序，它们彼此通过网络连接，共同完成一个全局的目标。
集群（Cluster）是指多台计算机组合在一起，共同提供服务，共存、共治，相当于一台巨大的服务器。集群通常采用廉价的服务器硬件，将计算能力集中到几百甚至几千台计算机上。

# 6.算法分类
算法分类有不同的标准，常见的有单词编码算法、排序算法、搜索算法、动态规划算法、图论算法等。以下按顺序介绍其中几个算法。

## 6.1 单词编码算法
单词编码算法（Word Coding Algorithm）是指将数据转换成固定长度的比特流，使其可以在不丢失信息的情况下传输。常用的单词编码算法有ASCII、UTF-8、GBK等。例如，"hello world"在ASCII下可能对应7个字节的序列，分别代表“h”、“e”、“l”、“l”、“o”、“ ”、“w”、“o”、“r”、“l”、“d”。UTF-8则可能对应三个字节的序列，前两个字节表示字符的UTF-8编码值，最后一个字节作为结束标记。

## 6.2 排序算法
排序算法（Sorting Algorithm）是指将一组数据按照指定顺序排列的方法。常见的排序算法有冒泡排序、选择排序、插入排序、归并排序、快速排序、计数排序、桶排序、基数排序等。

## 6.3 搜索算法
搜索算法（Search Algorithm）是指在一组数据中查找特定元素的方法。常见的搜索算法有线性搜索、二分搜索、斐波那契搜索、哈希搜索等。

## 6.4 动态规划算法
动态规划算法（Dynamic Programming Algorithm）是指在一个给定的问题中，把最优子结构的问题，递归地求解，并根据子问题的解得到原问题的解。动态规划算法解决的问题通常具有重叠子问题，即子问题之间存在重叠。

## 6.5 图论算法
图论算法（Graph Theory Algorithm）是指利用图形这种数据结构进行计算的算法。图论算法用于处理复杂网络、生态系统、交通流、信息流、布局、拓扑等问题。
