# 1.简介
  

JVM（Java Virtual Machine）是Sun公司于1995年推出的商用Java平台。它是一个运行在操作系统之上的抽象计算机环境，包括运行时数据区、类加载器、字节码解释器等组成。JVM将字节码编译成本机机器指令并执行，使得Java语言成为一门真正的通用语言。

随着JVM成为主流，越来越多的人开始关注JVM的内部实现。目前很多高级工程师、架构师都担任过CTO或技术总监的角色，而他们经常被问到这样一个问题："对于一个看上去非常复杂的技术，如果你仅仅是为了搞清楚它的原理，为什么还要花费那么大的精力？"

此外，由于互联网的蓬勃发展，许多同行对JVM技术产生了浓厚兴趣。比如，Netflix的工程师就对JVM底层的原理以及各种性能优化方法有比较深入的研究。另外，Google、Facebook等大厂也纷纷推出自己的运行时产品，如Android Dalvik、Chrome V8引擎等。因此，了解JVM的原理及其背后的技术体系，对于在技术路线中游走的技术人来说，也是十分必要的。

本书的目标就是为读者提供全面的JVM技术知识，并且按照一定的结构框架，细致深入地阐述JVM的各种实现机制及其运行机制。通过阅读本书，读者可以了解到JVM底层的工作原理、编译原理、虚拟机设计、内存管理、安全性等方方面面。同时，也可以加深对JVM的理解，培养对JVM技术的实践能力，为日后深入JVM技术的学习打下坚实的基础。
