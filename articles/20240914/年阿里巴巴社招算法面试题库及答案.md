##### 1. 数据结构相关

**题目：** 请实现一个栈的数据结构，支持 push、pop、peek 和 isEmpty 操作。

**答案：**

```java
class Stack {
    private Node top;
    private int size;

    private class Node {
        int val;
        Node next;
    }

    public void push(int val) {
        Node node = new Node();
        node.val = val;
        node.next = top;
        top = node;
        size++;
    }

    public int pop() {
        if (isEmpty()) {
            throw new NoSuchElementException();
        }
        int val = top.val;
        top = top.next;
        size--;
        return val;
    }

    public int peek() {
        if (isEmpty()) {
            throw new NoSuchElementException();
        }
        return top.val;
    }

    public boolean isEmpty() {
        return size == 0;
    }
}
```

**解析：** 该实现使用了链表作为底层数据结构，支持快速 push、pop 和 peek 操作。

##### 2. 算法相关

**题目：** 请实现一个快速排序算法。

**答案：**

```java
public class QuickSort {
    public static void quickSort(int[] arr, int low, int high) {
        if (low < high) {
            int pivot = partition(arr, low, high);
            quickSort(arr, low, pivot - 1);
            quickSort(arr, pivot + 1, high);
        }
    }

    private static int partition(int[] arr, int low, int high) {
        int pivot = arr[high];
        int i = low;
        for (int j = low; j < high; j++) {
            if (arr[j] < pivot) {
                swap(arr, i, j);
                i++;
            }
        }
        swap(arr, i, high);
        return i;
    }

    private static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```

**解析：** 该实现采用了经典的分治策略，将数组分为两部分，递归地排序两部分。

##### 3. 系统设计相关

**题目：** 请设计一个缓存系统，支持 add、get 和 delete 操作。

**答案：**

```java
import java.util.HashMap;
import java.util.Map;

class Cache {
    private Map<Integer, Integer> cache;
    private int capacity;

    public Cache(int capacity) {
        this.cache = new HashMap<>();
        this.capacity = capacity;
    }

    public void add(int key, int value) {
        if (cache.containsKey(key)) {
            cache.put(key, value);
        } else {
            if (cache.size() >= capacity) {
                int firstKey = cache.keySet().iterator().next();
                cache.remove(firstKey);
            }
            cache.put(key, value);
        }
    }

    public int get(int key) {
        if (cache.containsKey(key)) {
            return cache.get(key);
        }
        return -1;
    }

    public void delete(int key) {
        cache.remove(key);
    }
}
```

**解析：** 该实现使用了哈希表作为底层数据结构，支持快速 add、get 和 delete 操作。
