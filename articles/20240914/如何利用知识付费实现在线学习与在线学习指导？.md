#### 算法编程题 6：设计一个学习进度跟踪系统

**答案：**

```python
# Python 代码示例：学习进度跟踪系统

class LearningProgress:
    def __init__(self):
        self.user_progress = {}

    def update_progress(self, user_id, course_id, completed_lessons):
        self.user_progress[user_id] = {
            course_id: completed_lessons
        }

    def get_progress(self, user_id, course_id):
        if user_id in self.user_progress and course_id in self.user_progress[user_id]:
            return self.user_progress[user_id][course_id]
        return None

# 示例使用
progress_system = LearningProgress()
progress_system.update_progress("user123", "course101", [1, 2, 3])
print(progress_system.get_progress("user123", "course101"))  # 输出 [1, 2, 3]
```

**解析：** 该系统通过更新用户的学习进度，并提供获取学习进度的方法，实现了学习进度跟踪的基本功能。
