## 二、典型面试题与算法编程题

### 2.1 数据分析与预测

**题目：** 请实现一个函数，计算一组数的平均值。

**答案：** 

```python
def average(numbers):
    return sum(numbers) / len(numbers)
```

**解析：** 这个函数通过计算数组的和除以数组长度，得到平均值。

### 2.2 优化算法

**题目：** 请设计一个贪心算法，求解背包问题。

**答案：**

```python
def knapsack(values, weights, capacity):
    n = len(values)
    items = [[v, w] for v, w in zip(values, weights)]
    items.sort(key=lambda x: x[0] / x[1], reverse=True)
    total_value, total_weight = 0, 0
    for value, weight in items:
        if total_weight + weight <= capacity:
            total_value += value
            total_weight += weight
        else:
            break
    return total_value
```

**解析：** 这个贪心算法按照价值与重量的比例降序排列物品，依次选取物品，直到背包容量达到上限。
