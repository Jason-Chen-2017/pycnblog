## 1. 游戏规则概述

黑白棋，又称为五子棋、连珠棋，是一种两人对弈的纯策略型棋类游戏。游戏的目标是控制棋盘，通过放置自己的棋子（黑白相间），形成连续的五子直线（横、竖、斜线均可），从而取得胜利。

### 2.4. 棋盘初始化

**题目：** 实现一个棋盘初始化函数，初始化一个空棋盘。

**答案：** 使用二维数组初始化棋盘，每个元素设置为 0，表示空位。

```go
func initializeBoard() [][]byte {
    board := make([][]byte, 15)
    for i := range board {
        board[i] = make([]byte, 15)
        for j := range board[i] {
            board[i][j] = 0
        }
    }
    return board
}
```
