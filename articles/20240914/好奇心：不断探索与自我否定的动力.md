def find_duplicates(nums):
    seen = set()
    duplicates = []
    for num in nums:
        if num in seen:
            duplicates.append(num)
        else:
            seen.add(num)
    return duplicates

# 测试代码

面试官：请你设计一个算法，找出字符串中的最长公共子串。

应聘者：首先，我们可以使用动态规划算法解决这个问题。具体实现如下：

```python
def longest_common_substring(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    max_len = 0
    end = 0
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                if dp[i][j] > max_len:
                    max_len = dp[i][j]
                    end = i
            else:
                dp[i][j] = 0
    return s1[end - max_len: end]

# 测试代码
s1 = "abcdefg"
s2 = "azcedgfh"
print(longest_common_substring(s1, s2))  # 输出 "ced"
```

面试官：这个算法的时间复杂度和空间复杂度分别是多少？

应聘者：这个算法的时间复杂度是 O(m*n)，空间复杂度是 O(m*n)，因为我们需要使用二维数组来存储动态规划的状态。

- **模型训练：** 使用训练数据集训练模型，并调整模型参数。

**示例：**

```python
# 假设我们好奇的问题是：如何预测某个社交媒体平台的用户活跃度？

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# 数据收集
data = pd.read_csv('user_activity_data.csv')

# 特征工程
data['hour'] = data['timestamp'].apply(lambda x: x.hour)
data['weekday'] = data['timestamp'].apply(lambda x: x.weekday())

# 模型选择
model = RandomForestClassifier(n_estimators=100, random_state=0)

# 模型训练
X = data[['hour', 'weekday', 'activity']]
y = data['label']

model.fit(X_train, y_train)

# 模型评估

print("Accuracy:", accuracy_score(y_test, y_pred))

# 决策支持
new_data = pd.DataFrame({'hour': [12], 'weekday': [4], 'activity': [1]})

应聘者：在项目管理方面，我遵循以下步骤：

首先，我会制定详细的项目计划。我会与团队成员讨论项目目标、时间表和资源需求，确保每个人都清楚项目的进度和任务。我会制定项目里程碑，确保项目按时交付。

其次，我会建立良好的团队协作机制。我会组织定期会议，讨论项目进展和问题，确保团队成员之间的沟通畅通。我会鼓励团队成员积极参与项目决策，提高他们的责任感和参与度。

在风险控制方面，我会识别可能影响项目进展的风险，并制定相应的应对策略。我会定期评估风险，确保项目始终在可控范围内。

最后，我会评估团队成员的绩效，提供反馈和激励。我会与团队成员讨论他们的工作进展和困难，提供必要的支持和资源。我会根据团队成员的表现，给予奖励和晋升机会，激励他们持续改进。

通过这些方法，我能够有效地管理项目，确保项目按时、按质量交付，并提升团队的执行力。
