#### 3. 使用机器学习优化数据分析和决策

**题目：** 如何使用机器学习优化数据分析和决策？

**答案：**

* **分类和回归：** 使用分类和回归算法对数据进行预测和分析。
* **聚类和降维：** 使用聚类和降维算法发现数据中的模式和关系。
* **异常检测：** 使用异常检测算法检测数据中的异常值。

**举例：** 使用K-均值聚类算法分析客户购买行为。

```python
# 导入库
import numpy as np
from sklearn.cluster import KMeans

# 加载数据
data = np.array([[1, 2], [1, 4], [1, 0],
                 [10, 2], [10, 4], [10, 0]])

# 创建K-均值聚类模型
model = KMeans(n_clusters=2)

# 训练模型
model.fit(data)

# 输出聚类结果
print(model.labels_)
```
