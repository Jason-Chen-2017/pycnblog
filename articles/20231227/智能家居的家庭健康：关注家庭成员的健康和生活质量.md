## 4.1 血压监测代码实例

以下是一个血压监测设备的代码实例，通过电压感应技术实现血压数据的收集：

```python
import time
import Adafruit_BMP280

# 初始化血压监测设备
bmp280 = Adafruit_BMP280.BMP280()

# 收集血压数据
def collect_blood_pressure_data():
    while True:
        # 获取血压数据
        blood_pressure = bmp280.read_pressure()
        # 输出血压数据
        print("血压：", blood_pressure)
        # 等待一段时间
        time.sleep(1)

# 主程序
if __name__ == "__main__":
    # 开始收集血压数据
    collect_blood_pressure_data()
```

在这个代码实例中，我们首先通过`Adafruit_BMP280`库初始化血压监测设备。然后，我们定义了一个`collect_blood_pressure_data`函数，该函数通过调用`bmp280.read_pressure()`方法获取血压数据，并输出血压数据。最后，我们在主程序中调用`collect_blood_pressure_data`函数开始收集血压数据。
