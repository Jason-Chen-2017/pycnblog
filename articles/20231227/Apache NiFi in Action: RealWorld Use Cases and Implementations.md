                 

# 1.背景介绍

Apache NiFi 是一个高度可扩展的流处理系统，可以处理大规模数据流，并提供强大的数据流管理功能。它可以用于实时数据流处理、数据集成、数据流监控和管理等场景。NiFi 的核心概念是“流通”（Flow）和“处理器”（Processor）。流通是数据流的抽象表示，处理器是对数据流进行操作的实体。

NiFi 的设计目标是提供一个易于使用、可扩展、高性能和可靠的数据流处理平台。它支持多种数据源和目标，如 HDFS、HBase、Kafka、Elasticsearch 等。NiFi 还提供了丰富的数据处理功能，如数据转换、分割、聚合、分析等。
