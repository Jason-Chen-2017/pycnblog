## 4.2 代码实例

我们可以使用Python编程语言来实现这个问题的解决方案。以下是一个简单的代码实例：

```python
import random
import numpy as np

def simulate_traffic(green_time):
    total_flow = 0
    total_wait_time = 0
    num_cars = 100

    for _ in range(num_cars):
        if random.random() < green_time:
            total_flow += 1
        else:
            total_wait_time += 1

    return total_flow / total_wait_time

def monte_carlo_traffic(num_iterations, green_time_range):
    results = []

    for _ in range(num_iterations):
        green_time = random.uniform(*green_time_range)
        result = simulate_traffic(green_time)
        results.append(result)

    return np.mean(results)

if __name__ == "__main__":
    num_iterations = 10000
    green_time_range = (0.5, 1.0)
    best_green_time = monte_carlo_traffic(num_iterations, green_time_range)
    print("最佳红绿灯时间：", best_green_time)
```

在这个代码实例中，我们首先定义了一个`simulate_traffic`函数，用于模拟交通流量的通行效率。然后，我们定义了一个`monte_carlo_traffic`函数，用于使用蒙特卡洛方法来估计不同红绿灯时间的通行效率。最后，我们在主函数中调用`monte_carlo_traffic`函数，并输出最佳的红绿灯时间。
