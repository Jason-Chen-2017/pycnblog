# 2.核心概念与联系
## 2.1 数据质量的定义
数据质量是指数据的准确性、完整性、一致性、时效性和可靠性等方面的表现。数据质量是衡量数据是否能满足其目的的指标。

## 2.2 数据质量的影响因素
数据质量受到多种因素的影响，如数据收集、存储、处理和分析等过程中的错误、漏洞、噪声和偏见。这些因素可能导致数据的不准确、不完整、不一致、不时效和不可靠。

## 2.3 数据质量的衡量指标
数据质量可以通过多种衡量指标来评估，如准确性、完整性、一致性、时效性和可靠性等。这些指标可以帮助我们了解数据的质量状况，并采取相应的措施提高数据质量。

data1 = pd.read_csv('data1.csv')
data2 = pd.read_csv('data2.csv')

# 数据转换
data1['age'] = data1['birth_date'].apply(lambda x: x.year)
data2['age'] = data2['birth_date'].apply(lambda x: x.year)

# 数据清洗
