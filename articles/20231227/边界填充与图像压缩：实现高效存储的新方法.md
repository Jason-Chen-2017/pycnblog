# 2.核心概念与联系

在本节中，我们将介绍边界填充与图像压缩的核心概念，以及这两种技术之间的联系。

## 2.1 边界填充

边界填充是一种图像处理技术，它通过在图像的边界处添加额外的像素值来填充图像的空白区域。这种技术可以用于改善图像的质量，使其更加清晰和完整。边界填充可以通过以下几种方法实现：

1. 平均填充：在边界像素的周围，平均地添加新的像素值。
2. 反射填充：在边界像素的周围，反射已有的像素值。
3. 邻近填充：在边界像素的周围，选择最近的像素值作为新的像素值。

## 2.2 图像压缩

图像压缩是一种将图像文件大小减小的技术，它通过去除不必要的信息或减少图像的细节来实现。图像压缩可以分为两种主要类型：损坏性压缩（lossy compression）和无损压缩（lossless compression）。

1. 损坏性压缩：在这种类型的压缩中，图像的一些信息会被丢失，从而导致图像的质量下降。这种类型的压缩通常用于在存储和传输过程中减少图像文件的大小。
2. 无损压缩：在这种类型的压缩中，图像的所有信息都会被保留，因此不会导致图像的质量下降。这种类型的压缩通常用于在存储和传输过程中保持图像的原始质量。

## 2.3 边界填充与图像压缩的联系

边界填充与图像压缩之间存在密切的联系。边界填充可以用于改善图像的质量，从而使图像更加适合进行压缩。同时，边界填充也可以用于减少图像文件的大小，因为它可以填充图像的空白区域，从而减少需要存储的像素值数量。
