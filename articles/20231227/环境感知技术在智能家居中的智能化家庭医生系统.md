# 4.具体代码实例和详细解释说明

## 4.1 环境感知技术的具体代码实例

以下是一个简单的环境感知技术的具体代码实例：

```python
import time
import Adafruit_DHT

# 设置传感器类型和传感器连接引脚
SENSOR = Adafruit_DHT.DHT22
PIN = 4

# 读取温度和湿度值
def read_temperature_humidity():
    humidity, temperature = Adafruit_DHT.read_retry(SENSOR, PIN)
    if humidity is not None and temperature is not None:
        return {"humidity": humidity, "temperature": temperature}
    else:
        return None

# 主程序
if __name__ == "__main__":
    while True:
        data = read_temperature_humidity()
        if data is not None:
            print("温度: {:.2f}℃, 湿度: {:.2f}%".format(data["temperature"], data["humidity"]))
        time.sleep(1)
```

上述代码实例中，我们使用了Adafruit_DHT库来读取温度和湿度值。首先，我们设置了传感器类型（DHT22）和传感器连接引脚（4）。然后，我们定义了一个`read_temperature_humidity`函数，用于读取温度和湿度值。最后，我们在主程序中不断地读取温度和湿度值，并打印出来。
