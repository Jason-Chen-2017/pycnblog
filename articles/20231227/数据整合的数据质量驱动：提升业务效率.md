## 4.1 数据清洗

假设我们有一个包含客户信息的数据表，如下所示：

| 客户ID | 客户姓名 | 客户年龄 | 客户性别 |
| ---- | ---- | ---- | ---- |
| 1 | 张三 | 25 | 男 |
| 2 | 李四 | 30 | 女 |
| 3 | 王五 | 35 | 男 |
| 4 | 赵六 | 40 | 女 |

在这个表中，我们可以看到客户年龄的数据有误，需要进行数据清洗。我们可以使用以下Python代码来检查和修复这个问题：

```python
import pandas as pd

# 读取数据
data = pd.read_csv('customer.csv')

# 检查年龄数据是否正确
def check_age(age):
    if age < 0 or age > 150:
        return False
    return True

# 修复年龄数据
def fix_age(age):
    if check_age(age):
        return age
    else:
        return 0

# 应用修复函数
data['客户年龄'] = data['客户年龄'].apply(fix_age)

# 保存修复后的数据
data.to_csv('customer_fixed.csv', index=False)
```

在这个代码中，我们首先使用pandas库读取数据。然后，我们定义了一个检查年龄数据的函数`check_age`，以及一个修复年龄数据的函数`fix_age`。最后，我们使用`apply`函数将修复函数应用于数据中的年龄列，并将修复后的数据保存到一个新的CSV文件中。

## 4.3 数据集成

假设我们需要将上述客户信息数据与另一个包含订单信息的数据表进行整合。我们可以使用以下Python代码实现这个功能：

```python
import pandas as pd

# 读取客户信息数据
customer_data = pd.read_csv('customer.csv')

# 读取订单信息数据
order_data = pd.read_csv('order.csv')

# 将客户信息数据与订单信息数据进行整合
def integrate_data(customer_data, order_data):
    integrated_data = pd.merge(customer_data, order_data, on='客户ID')
    return integrated_data

# 保存整合后的数据
integrated_data = integrate_data(customer_data, order_data)
integrated_data.to_csv('integrated_data.csv', index=False)
```

在这个代码中，我们首先使用pandas库读取客户信息数据和订单信息数据。然后，我们定义了一个将客户信息数据与订单信息数据进行整合的函数`integrate_data`。最后，我们使用`merge`函数将客户信息数据与订单信息数据进行整合，并将整合后的数据保存到一个新的CSV文件中。
