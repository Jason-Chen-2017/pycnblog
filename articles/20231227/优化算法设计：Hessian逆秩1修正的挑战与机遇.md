# 2. 核心概念与联系

优化算法的核心概念包括：

1. 目标函数：优化算法的目标是寻找使目标函数取得最小值或最大值的点。
2. 梯度：梯度是目标函数在某一点的导数，用于衡量目标函数在该点的增加或减少速度。
3. 新罗尔条件：新罗尔条件是判断一个点是局部最小值、局部最大值还是梯度不存在的条件。
4. 二阶导数：二阶导数可以用来衡量目标函数在某一点的凸凹性。

Hessian逆秩1修正是一种优化算法的改进方法，它通过修正Hessian矩阵的逆来减少计算复杂度，从而提高优化算法的性能。
