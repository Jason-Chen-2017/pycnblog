# 2.核心概念与联系

数理统计在医学图像分析中的应用主要体现在以下几个方面：

1. 图像处理：数理统计可以帮助我们对医学图像进行滤波、平滑、增强、分割等操作，从而提高图像质量，减少噪声影响，并提取有关病变的特征信息。

2. 图像特征提取：数理统计可以帮助我们对医学图像进行特征提取，如灰度统计、形状特征、纹理特征等，从而对病变进行定性和定量分析。

3. 图像分类和聚类：数理统计可以帮助我们对医学图像进行分类和聚类，以便对病变进行分类和诊断。

4. 图像 реги册：数理统计可以帮助我们对医学图像进行注册，即将不同时间或不同模态的图像align，从而实现图像融合和比较。

5. 图像模板匹配：数理统计可以帮助我们对医学图像进行模板匹配，以便对病变进行诊断和治疗。
