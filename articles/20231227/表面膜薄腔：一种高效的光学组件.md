## 4.1 代码实例
以下是一个表面膜薄腔的具体代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt

# 光学材料的光学特性
n1 = 1.5
n2 = 1.0

# 膜层数
N = 3

# 光学效应
R = []
for i in range(N):
    R.append(np.abs(n1 - n2) / (n1 + n2))

# 折射率
n = []
for i in range(N):
    n.append(n1 / n2 * np.sin(np.radians(45)))

# 吸收率
I0 = 1000
I = 500
alpha = (I0 - I) / I0 * 100

# 绘制图像
plt.plot(R, label='Reflection')
plt.plot(n, label='Refraction')
plt.plot(alpha, label='Absorption')
plt.xlabel('Layer Index')
plt.ylabel('Optical Property')
plt.legend()
plt.show()
```

## 4.2 详细解释说明
在上述代码实例中，我们首先导入了`numpy`和`matplotlib.pyplot`库，用于数值计算和图像绘制。然后，我们定义了光学材料的光学特性（折射率`n1`和`n2`）和膜层数（`N`）。接着，我们通过循环计算了每层膜的反射率、折射率和吸收率，并将其存储在列表中。最后，我们使用`matplotlib.pyplot`库绘制了图像，展示了表面膜薄腔的光学特性。
