```python
import hashlib

class DistHashTable:
    def __init__(self):
        self.table = {}

    def put(self, key, value):
        hash_key = hashlib.sha256(key.encode()).hexdigest()
        p = len(self.table)
        self.table[hash_key % p] = value

    def get(self, key):
        hash_key = hashlib.sha256(key.encode()).hexdigest()
        p = len(self.table)
        return self.table.get(hash_key % p)
```

# 6.附录常见问题与解答

1. 问：并行计算与分布式计算有什么区别？
答：并行计算是指在多个处理器或计算节点上同时进行计算，而分布式计算是指将计算任务分布到多个计算节点上，这些节点可以在不同的时间进行计算。

2. 问：并行计算在矿业领域有哪些应用？
答：并行计算在矿业领域主要应用于矿产资源分析、矿业数据处理、矿业智能化管理等方面。

3. 问：并行计算需要哪些硬件资源？
答：并行计算需要多个处理器或计算节点，这些节点可以是单独的计算机，也可以是集成在同一台计算机上的多个处理器。此外，并行计算还需要高速网络来连接这些处理器或计算节点。
