#### 3.1.2 二阶导数

#### 3.1.3 极值定理的具体操作步骤

1. 计算函数的一阶导数和二阶导数。
2. 找到一阶导数为0的点。

4. 如果一阶导数在某一点为正，那么这个点是函数的最小值点，如果一阶导数在某一点为负，那么这个点是函数的最大值点。

import numpy as np

def first_derivative(x):
    return np.sin(x)

def second_derivative(x):
    return np.cos(x)

def extreme_value(x):
    return first_derivative(x)

def find_extreme_value(x):
    x = np.linspace(-np.pi, np.pi, 1000)

    first_derivative_x = first_derivative(x)
    second_derivative_x = second_derivative(x)
    return x, y, first_derivative_x, second_derivative_x

x, y, first_derivative_x, second_derivative_x = find_extreme_value(x)
