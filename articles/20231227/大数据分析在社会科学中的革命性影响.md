# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明大数据分析在社会科学中的应用。

## 4.1 代码实例

我们以一个社交网络数据的关联规则挖掘为例，来说明大数据分析在社会科学中的应用。

```python
import pandas as pd
from sklearn.associate import AssociationRule
from sklearn.metrics import accuracy_score

# 加载数据
data = pd.read_csv('social_network_data.csv')

# 数据预处理
data = data.dropna()
data = data[data['age'] > 18]

# 关联规则挖掘
rules = AssociationRule(data, metric='lift', min_threshold=1)
rules = rules.fit_transform()

# 评估规则
preds = rules.predict(data)
accuracy = accuracy_score(data['label'], preds)
print('Accuracy:', accuracy)
```

## 4.2 详细解释说明

在这个代码实例中，我们首先使用pandas库来加载社交网络数据。然后，我们对数据进行预处理，包括删除缺失值和筛选年龄大于18的用户。

接下来，我们使用sklearn库中的AssociationRule来进行关联规则挖掘。我们设置了一个阈值为1的阈值，以确保规则的有效性。

最后，我们使用accuracy_score函数来评估规则的准确性。
