### 4.1 电影推荐系统
我们可以使用关联规则挖掘算法来构建一个电影推荐系统。以下是一个简单的Python代码实例：

```python
import pandas as pd
from sklearn.associate import AssociationRule
from sklearn.metrics import mutual_info_classif

# 加载数据
data = pd.read_csv('movie_data.csv')

# 计算相关性
rules = AssociationRule(data, metric='confidence', min_threshold=0.5)

# 生成规则
rules = rules.generate_rules()

# 打印规则
print(rules)
```

### 4.2 音乐推荐系统
我们可以使用聚类分析算法来构建一个音乐推荐系统。以下是一个简单的Python代码实例：

```python
import pandas as pd
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

# 加载数据
data = pd.read_csv('music_data.csv')

# 数据预处理
data = StandardScaler().fit_transform(data)

# 聚类分析
kmeans = KMeans(n_clusters=5)
kmeans.fit(data)

# 打印聚类结果
print(kmeans.labels_)
```
