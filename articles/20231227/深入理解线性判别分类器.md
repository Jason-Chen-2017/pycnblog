                 

# 1.背景介绍

线性判别分类器（Linear Discriminant Analysis，LDA）是一种常用的统计学习方法，主要用于二分类问题。它的核心思想是找到一个线性分类器，将数据点分成两个类别。线性判别分类器是一种基于概率的方法，它假设每个类别的数据点在特征空间中具有一个高斯分布，并且这两个分布是独立的。

线性判别分类器的核心算法是最大化两个类别间距离，同时最小化内部类别的方差。这个目标可以通过优化一个对数似然函数来实现。具体来说，线性判别分类器会找到一个最佳的线性分类器，使得在该分类器上的错误率最小。

线性判别分类器的一个主要优点是它的算法简单易实现，另一个优点是它对于高维数据的表现很好。然而，线性判别分类器的一个主要缺点是它假设数据点在特征空间中具有高斯分布，这种假设在实际应用中很难满足。
