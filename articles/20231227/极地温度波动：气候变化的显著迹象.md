## 3.1 极地温度波动的数学模型

极地温度波动的数学模型可以通过以下公式表示：

$$
T_{t} = \alpha \times T_{t-1} + (1 - \alpha) \times \bar{T} + \epsilon_t
$$

其中，$T_{t}$ 表示时间 t 的极地温度，$T_{t-1}$ 表示时间 t-1 的极地温度，$\bar{T}$ 表示平均温度，$\alpha$ 表示温度衰减因子，$\epsilon_t$ 表示随机误差。

## 3.2 极地温度波动的计算步骤

1. 首先，需要收集极地温度数据，包括地面测量数据和卫星测量数据。
2. 然后，将收集到的极地温度数据进行预处理，包括去除缺失值、去除异常值等。
3. 接下来，使用数学模型公式计算极地温度波动。具体步骤如下：
   1. 计算温度衰减因子 $\alpha$。
   2. 计算平均温度 $\bar{T}$。
   3. 使用公式计算时间 t 的极地温度 $T_{t}$。
4. 最后，对计算结果进行分析，并绘制极地温度波动曲线。

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 加载极地温度数据
data = pd.read_csv('extreme_temperature.csv')

# 预处理极地温度数据
data = data.dropna()
data = data.drop_duplicates()

# 计算温度衰减因子

# 使用数学模型公式计算极地温度波动
data['T_t'] = alpha * data['T_t-1'] + (1 - alpha) * data['T_mean'] + np.random.normal(0, 0.1, len(data))

# 绘制极地温度波动曲线
plt.plot(data['T_t'])

plt.ylabel('Temperature')
