# 2.核心概念与联系

## 2.1 基函数

基函数是用于表示气候变化模式的基本函数。它们可以是线性函数、指数函数、三角函数等。基函数可以用来表示气候变化的趋势、季节性、周期性等特征。常用的基函数有：

1. 常数函数：f(t) = 1
2. 线性函数：f(t) = t
3. 指数函数：f(t) = e^t
4. 三角函数：f(t) = sin(t)、f(t) = cos(t)

## 2.2 函数内积

函数内积是用于计算两个函数之间的相关性的数学概念。函数内积定义为：

$$
\langle f, g \rangle = \int_{a}^{b} f(t) g(t) dt
$$

其中，$f(t)$ 和 $g(t)$ 是两个函数，$a$ 和 $b$ 是函数的定义域。

函数内积可以用来计算两个气候变化函数之间的相关性，从而帮助我们理解气候变化的规律和机制。
