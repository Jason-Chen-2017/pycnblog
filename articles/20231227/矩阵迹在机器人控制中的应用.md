# 2.核心概念与联系

## 2.1 矩阵迹基本概念

矩阵迹是一个矩阵的一个基本性质之一，它可以用来描述矩阵的一些性质和特征。矩阵迹的定义如下：

给定一个方阵A，其大小为n×n，迹tr(A)是A的对角线和的和，即：

$$
tr(A) = a_{11} + a_{22} + \cdots + a_{nn}
$$

矩阵迹具有以下性质：

1. 迹是线性的，即对于任意两个矩阵A和B，有tr(αA + βB) = αtr(A) + βtr(B)，其中α和β是实数。
2. 迹是不变的，即对于任意一个矩阵A的同态φ，有tr(φ(A)) = tr(A)。
