# 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个简单的Python代码实例来演示神经模糊系统在文本摘要和自动摘要生成中的应用。

```python
import numpy as np
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.cluster import KMeans
from sklearn.metrics import adjusted_rand_score

# 文本数据
texts = ["This is the first document.", "This document is the second document.", "And this is the third one.", "Is this the first document?"]

# 文本特征提取
tfidf_vectorizer = TfidfVectorizer()
tfidf_matrix = tfidf_vectorizer.fit_transform(texts)

# 文本表示学习
kmeans = KMeans(n_clusters=2)
kmeans.fit(tfidf_matrix)

# 关键信息提取
labels = kmeans.labels_

# 摘要生成
summary = " ".join([texts[i] for i in range(len(texts)) if labels[i] == 0])

# 摘要评估
ar_score = adjusted_rand_score(labels, kmeans.labels_)

print("Summary: ", summary)
print("AR Score: ", ar_score)
```

在这个代码实例中，我们首先使用TF-IDF向量化器对文本进行特征提取，然后使用KMeans聚类算法对文本进行聚类，从而提取关键信息。最后，我们将聚类标签为0的文本组合成摘要，并使用相似度指标（Adjusted Rand Score）来评估摘要质量。
