# 1.背景介绍


​    “边缘计算”是人工智能（AI）和计算机技术领域的一项热门研究方向，它主要关注于利用物联网、大数据和机器学习等新兴技术，将智能设备连接到云端，实现数据的收集、处理及实时推送，使得这些设备能够更加自主地进行计算、决策并做出相应的反馈。如今，越来越多的人开始关注和把目光投向边缘计算，希望通过边缘计算实现人机交互、智能应用、自动驾驶、运维监控等人类生活的方方面面。
​    在今天这个信息化、高度数字化的社会里，许多企业已经把重点放在把核心业务能力建设在云上，而对于边缘计算资源，却仍处于被忽视甚至被看作“边角料”。传统的数据中心的网络带宽已无法满足当今海量数据的传输需求；移动终端的处理性能也远不能满足人们对实时响应的需求；传感器设备的采集范围也已远超过过去几年的汇聚点。随着云计算的普及和边缘计算的火爆，越来越多的企业和个人渐渐意识到，对边缘计算资源的有效利用，可以降低成本、提高效率，并更好地服务于人类的生产生活。
​    在此背景下，“边缘计算与AI”便应运而生了。它是一种结合边缘计算、机器学习、图像识别等多个相关技术领域的新型研究范式。基于这种研究范式，可以帮助企业理解如何将最新的边缘计算技术运用到实际生产环节中，从而为用户提供更多实时的、个性化的信息和服务。
​    本系列文章共分为7篇，每篇文章侧重于一个核心主题。第一篇文章“边缘计算的定义、分类和发展”，阐述了边缘计算的概念、分类方法及其演进历程。第二篇文章“边缘计算平台的选型和架构设计”，介绍了边缘计算平台的选型建议、组件组合及方案实现。第三篇文章“边缘计算开发工具的选型和应用”，介绍了边缘计算平台的开发环境配置及使用的各项工具介绍。第四篇文章“边缘计算的安全体系及网络部署实施”，介绍了边缘计算的安全体系、网络部署规划、安全管理规范及实施方式。第五篇文章“边缘计算的运维保障和管理”，介绍了边缘计算平台的运维保障措施、管理工具及流程。第六篇文章“边缘计算的应用案例分享”，分享几个典型的边缘计算应用场景，包括智慧城市、智慧安防、智慧医疗等。最后一篇文章“边缘计算在未来行业中的机遇与挑战”，讨论边缘计算在未来行业中的发展前景及其面临的挑战。
​    通过这七篇文章，读者不仅可以了解边缘计算的概念、技术发展及现状，还能充分理解边缘计算所需解决的问题、方案及技术细节，更能搭建起边缘计算与人工智能的良好沟通平台，真正让自己的工作、学习、生活、行动都发生着变化。
