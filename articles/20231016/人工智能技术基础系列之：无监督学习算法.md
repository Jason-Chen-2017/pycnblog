# 1.背景介绍


## 概述
无监督学习（Unsupervised Learning）是指从数据中学习出一些特征而不给定任何标签，即没有任何显式的提示告诉机器学习算法该数据的真实目的。因此，在学习过程中不需要对数据进行标记，只需根据数据的分布情况和相似性推断出数据的含义就行了。由于没有指定目标，因此很难预测学习得到的结果到底好坏。但是无监督学习又提供了一种创新思路——可以利用聚类、分类等技术从原始数据中提取有意义的信息。无监督学习算法既可以用于聚类分析（Clustering），也可用于数据降维、数据集成、异常检测和推荐系统等领域。
## 关键术语
### 1. 聚类 Clustering
聚类分析的任务是在一组数据中发现隐藏的模式或结构。一般地，数据可以是多维度的，而且每个数据点都有一个唯一的标识符。聚类算法就是找到这种结构。

### 2. 降维 Dimensionality reduction
数据降维是指对高维数据进行转换，使其变得更容易处理和理解。降维的方法包括主成分分析PCA和线性判别分析LDA。其中，PCA通过计算样本协方差矩阵来寻找数据中最大方差的方向，并将数据投影到这个方向上；而LDA则基于数据之间的均值向量和协方差矩阵，利用最小重构误差准则寻找一个低维空间中的方向直线来表示样本。降维后的坐标系成为新的特征空间，通过它可以方便地了解数据的结构和关系。

### 3. 数据集成 Data integration
数据集成是指将不同来源、不同形式的数据集成到一起，实现共同的分析目的。数据集成方法包括合并、连接、关联、联合查询、异构融合。其中，合并主要是将多个数据集按照时间顺序或者数据粒度进行拼接，即按照某种规则把多个数据集合并起来；连接即把不同的实体信息（如商品、顾客、企业、人员）整合到一起，进行信息管理、客户数据库建设、数据共享；关联即在已有数据基础上发现潜在的关系和联系，增强数据内在的联系；联合查询即通过搜索引擎或其他方式找到相关的信息，然后再进行数据融合；异构融合即结合不同类型的信息，利用多个源头数据之间的联系进行数据整合。

### 4. 异常检测 Anomaly detection
异常检测是一种在大量数据中识别不正常或异常数据的过程。异常检测的方法包括基于密度的异常检测算法、基于聚类的异常检测算法、基于分类的异常检测算法、基于规则的异常检测算法。密度的方法包括基于距离度量的DBSCAN、基于密度曲面法的LOCAL DBSCAN;聚类方法包括k-Means、层次聚类、谱聚类等；分类方法包括分类树和决策树；规则方法包括正态分布、指数分布、门限值、波峰检测等。

### 5. 推荐系统 Recommendation system
推荐系统是指根据用户的兴趣和历史行为产生推荐产品的系统。推荐系统主要包括协同过滤、内容推送和召回算法。协同过滤是基于用户群体的兴趣偏好及其相似的物品特征进行推荐的技术；内容推送是基于推荐引擎的广告系统，通过对历史浏览记录进行分析来提供个性化的产品推荐；召回算法是推荐系统获取候选对象的过程，基于某些策略选择推荐的对象，如最热门的产品或最相关的产品。
