# 1.背景介绍


量子通信（quantum communication）和量子密码学（quantum cryptography）是近几年来物理学界对信息科技领域所产生的重大突破性研究。两者都源自于量子力学的动力学特性，而量子通信技术在利用量子纠缠（quantum entanglement）、量子调制（quantum modulation）等现象降低传播损耗、提升传输效率的同时，还引入了两类专用硬件——量子计算机和量子存储器，具有不可替代的战略地位。
# 2.核心概念与联系
## 概念
### 量子通信
- 量子通信（quantum communication）指利用量子技术解决传送比特串的过程中出现的问题，比如错误控制、干扰信道、传输时延等问题。
- 量子通信要达到信息传输的目的，通常需要传输的信息被编码成一种形式，使得接收方能够正确解析。编码方式中的一个重要方式就是量子态（qubit），这种量子态可以被认为是一个包含两种离散的量子态的复数，其中一个态就是比特态（bit）。因此，通过改变量子态的大小、方向或者叠加状态，就可以通过电磁波、光信号等载体进行信息的传输。
- 根据量子通信的分类标准，主要分为两类：
  - 一类是错误纠正型（error-correcting codes）：根据其编码方式，可以通过对传输的比特串进行错误检测和纠正，来实现信息传输的可靠性。这种方法能够克服传统通信中常见的噪声干扰、时延增加以及比特差错等问题。
  - 另一类是无纠错型（non-error-correcting codes）：这种编码方式对传输的信息不进行纠错，并通过隐蔽的方式将其传输。无纠错型的编码方式会受到噪声影响、时延抖动等问题的影响，但是由于无法对传输的数据进行错误纠正，所以传输的信息可能丢失或被错误读出。
### 量子密码学
- 量子密码学（quantum cryptography）是在密钥交换、加密及解密过程中的应用量子计算的科学研究领域。它基于量子力学的基本原理，以量子态的研究方法探讨各种加密方法的原理，将密钥生成、加解密等过程改造为基于量子化约的算法。
- 量子密码学可以把加密解密过程抽象为两个独立的任务：加密和解密。对于加密过程来说，输入的数据经过量子处理后生成新的数据，输出密文；对于解密过程来说，输入密文，经过相同的量子处理，得到解密结果。整个过程可以看作由复杂的量子算法加上高速的计算能力实现。
- 量子密码学主要由三类算法构成：
  - 流密码算法：主要用于实现对称密钥加密。它通过对数据流进行排列组合、混淆等方式，对密钥流进行混合，生成新的密钥流，然后使用这个新的密钥流加密数据流。流密码算法适用于大容量数据的加密解密。
  - 块密码算法：用于实现对称密钥加密。它通过对数据块进行加密，对每一位的数据进行加密，得到不同的密文。块密码算法适用于小容量数据块的加密解密。
  - 分组密码算法：用于实现非对称密钥加密。它采用公钥和私钥对，公钥用于加密，私钥用于解密。分组密码算法适用于长文本文件的加密解密。
## 联系
- 量子通信和量子密码学是密切相关的研究领域。
- 量子通信作为通信的基石，它极大的提高了信息的传输效率、可靠性以及安全性。但同时也带来了新的技术和挑战，如量子纠缠、量子调制等。
- 量子密码学试图从加密与解密的角度，来进一步探索量子计算技术的优点。目前，量子密码学已成为很多重要的安全技术的支柱，如银行卡支付、智能电网通信等领域都已经开始布局量子通信技术。
