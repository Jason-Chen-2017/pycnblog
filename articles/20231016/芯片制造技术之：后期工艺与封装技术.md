# 1.背景介绍


后期工艺包括设计布局、集成电路布线、集成测试、铭牌设计、底板焊接等环节。作为一个精细化、高端的芯片制造过程的一部分，这些工艺环节对整个芯片制造流程的效率和质量都有着极大的影响。我们无法从零开始，就要求设计者设计出无可挑剔的产品。因此，后期工艺中的每个环节都要依靠一定的专业知识才能正确进行，这也是本文主要关注的内容。

通过本文可以帮助读者了解后期工艺的相关知识，掌握设计者需要具备哪些技能与知识才能更好地完成各个环节。同时，也可以帮助读者提升自己的能力，总结经验并向其他技术人员、企业分享有益的经验教训。

 # 2.核心概念与联系
## 2.1 集成电路的时序逻辑
集成电路的时序逻辑（Timing）是指，通过反复的输入、输出信号，电路中的每个元件按其内部连接关系进行操作，按照规定时间产生正确的输出信号，这种控制方式可以使电路达到稳态。时序逻辑属于硬件系统的基础，没有它就无法实现现代计算机系统所要求的高速计算性能。所以，每一个设计者都需要了解时序逻辑的一些基本原理和规范。

### 时钟周期
在设计和建模集成电路时，时钟周期就是指时钟信号逝去的时间间隔，它通常以秒或毫秒计。时钟周期越短，则整个系统的处理速度越快；但如果时钟周期太长，则会导致晶体管过热、发热变形、机械损耗增加、时延增大、抖动增加等问题。时钟周期的确定直接影响系统的运行效率和功耗。一般来说，常用的时钟周期有以下几个值：

- 10ns: 普通应用领域中使用的最多的时钟周期，标准的操作系统、数据库、通信设备等都使用了这个时钟周期。
- 25ns: 非常适合于电子显示器、激光打印机、激光扫瞄仪、航空航天器等等行业领域。
- 1ps: 在金融、汽车、医疗、军事、核科学等领域也有广泛的应用。
- 100ps: 这个周期在传感器领域扮演了重要角色，如激光测距仪、红外遥控器等。

除了时钟周期，还有一个重要的参数是时钟频率。时钟频率是指每秒钟发生的脉冲个数，其单位是赫兹（Hz）。比如，时钟频率为10MHz，则意味着每秒钟发生100万次脉冲，即1us时间内发生了100万次脉冲信号。由于脉冲信号的周期性和随机性，使得集成电路能够工作正常，这也是为什么很多晶体管的驱动电压不能低于某个阈值，而只能在一定范围内波动的原因。

### 信号约束
信号约束（Signal Constraints）是指信号的上升沿和下降沿之间的延迟时间的范围，也就是说，信号约束其实就是系统的时延。在实际工程实践中，时延是影响系统性能的最关键因素之一，它直接决定了系统的运行速度、响应时间、容错率、稳定性以及耗能等性能指标。一般情况下，信号约束小于时钟周期，即系统的输入输出信号的上升沿和下降沿之间不超过时钟周期的一半，这样就保证了系统的时延不会超过时钟周期。但是，如果信号约束大于时钟周期一半，那么就会出现上升沿和下降沿的相互作用，产生抖动甚至混乱的情况。因此，设计者应该根据实际工程需要，选择合理的信号约束。

## 2.2 布局布线层次结构
布局布线层次结构是将电路的设计过程分为多个层次，其中最低层次为逻辑网表，其次是门电路设计、电气设计和电源分配等层次，最后才是单元封装、模块封装和电路封装等最终的输出结果。每一层次的功能不同，需要采用不同的方法、工具和流程才能顺利完成。

### 逻辑网表
逻辑网表（Logical Netlist）是用来描述集成电路所有逻辑电路以及它们之间的连接关系的文件。逻辑网表包含的元素有：门电路、器件连接、端口定义、阻抗匹配、时序约束、标注定义、电源分配等。逻辑网表提供了电路设计中最底层的设计信息，是之后的各种设计文件（例如电气设计、电源分配等）的输入。

### 门电路设计
门电路设计（Gate-Level Design）是把门电路映射到真正的电路的过程。门电路的电流输入、输出方向及组合方式与实际电路的对应部件相同，故而可以准确的捕获电路的行为特性。门电路设计的目标是在更高的逻辑层次上建立抽象模型，以方便后续的电气设计。

门电路的实现主要基于逻辑电路，包括常用电路、函数块、高级语言模型等，这些都是前人经验积累的产物。门电路的实现需要遵守电气设计规则，如避免设计冗余、降低噪声、符合传输特性等。门电路的布线方式通常采用面布线法、点布线法或直排布线法等。

### 电气设计
电气设计（Electrical Design）是根据逻辑网表生成电路组网图，然后分析、综合、布线，得到所需电路的集成电路。电气设计需要考虑电路的静态性能、动态性能、功耗等方面的指标，并且进行容量评估、阻抗匹配等。电气设计包括静态（DC）、动态（AC）、混合（EMI）和线性电源等类型。静态电路的实现可以使用LCC、LTCC或HSPICE等工具。动态电路的实现需要使用SPICE、PSpice或Proteus等工具。线性电源的实现则依赖于IC开发套件或是专业的定制IC设计团队。

### 电源分配
电源分配（Power Distribution）是为了解决不同电源之间的电压和电流需求之间的矛盾。电源分配器可以是电池（包括锂电池、磷酸锂电池、镍镉电池、锌电池等）、直流电机、交流发电机、太阳能源等。电源分配器一般配有大功率开关，能够很好的满足电源需求。

### 模块封装
模块封装（Module Encapsulation）是把电路组成的模块进行封装，以便于简化后期的组装、调试和维护。模块封装的方法有SOD（Standard OIA Device）、SOP（Standard OIA Process）、QFP（Quad Flat Package）、TSSOP（Through-Silicon Solder Paste）等。

### 电路封装
电路封装（Circuit Encapsulation）是把电路连接到晶体管电路板上的过程。主要由两大部分组成：电路连接器（Connector）和电路开关（Switching）。电路连接器用于连接电路模块之间的引脚、接口；电路开关用于连接模块之间的连接器。电路开关的功能主要是用于隔离、保护、调整电路的电源、信号、电压、电流等。电路开关又分为直连开关、双极开关、三极开关等。

## 2.3 封装技术
封装技术（Encapsulation Technology）是指利用数字信号的编码和解码将集成电路的组合、连接和布局参数存储到芯片内部或者外围的过程。主要包括软、硬件级别的封装技术。软封装技术是指利用PCB制作工具提供的规则来完成芯片的封装。硬件封装技术则是指利用芯片制造工艺中提供的功能来实现封装，如引脚阵列、触发器等。

### 引脚阵列
引脚阵列（Pin Array）是利用PCB制作工具提供的引脚排列模式构建的，能够使集成电路的外部接口具有标准化的形式，提高了外部接口的一致性。引脚阵列的设计和布局通常参照规则来进行，如通用异步接口（JTAG）、先进非阻塞接口（GBA）、串行接口（UART）、并行接口（PARALLEL）等。

### 触发器
触发器（Transistor）是一种集成电路结构，能够在电压触发事件后执行触发条件。触发器的种类繁多，如上拉/下拉触发器、触发开关、栅极触发器、触点触发器、发光二极管触发器等。触发器的类型主要取决于触发电压和触发条件，如“低于”“高于”“平衡”“跳变”“电平改变”等。
