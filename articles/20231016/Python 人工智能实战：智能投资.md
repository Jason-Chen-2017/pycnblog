# 1.背景介绍


## 1.1 什么是人工智能？
　　人工智能（Artificial Intelligence）是一个跨学科的研究领域，涉及计算机、心理学、数学、经济学等多个学科。它包括认知、推理、学习、语言、动作、知识表示、模拟、计划等多个子领域。通过让机器具有感知、理解和决策功能，人工智能可以做出比人类更聪明、更富创造力的决策。它的关键技术包括搜索、数据挖掘、模式识别、计算理论、运筹学、控制理论、信息论、机器学习、图灵机、神经网络、遗传算法、递归神经网络等等。
　　在20世纪70年代以前，人工智能通常被视为一个严肃的科学分支，受到严格的保密性要求，而在当时还没有成熟的算法和工具可用。直到20世纪90年代末期，随着计算机技术的迅速发展和多种算法和工具的出现，人工智能才真正成为一个热门话题。近几年，随着互联网和云计算技术的普及，人工智能的应用越来越广泛。现在，人工智能正在改变着我们的生活方式、工作方式，改变着全球产业链的方向，将带来极大的社会影响。
## 1.2 为什么要进行智能投资？
　　近几年，随着金融危机、全球经济下行压力加剧，股市价格疯狂下跌，投资者的参与率逐渐降低。传统的散户投资由于资金体量小、经验少、管理不善，导致风险过高。而通过算法交易，可以获得超额收益或减少损失。
　　在这样的背景下，智能投资机构应运而生。智能投资即建立基于人工智能的投资策略，利用人工智能技术分析股票，并采用机器自动化执行交易，帮助投资者实现资产价值最大化。最主要的优势是降低了交易风险，提高了投资回报。
## 1.3 智能投资的作用
　　智能投资的主要作用如下：
- 提升个人财富
- 节约时间
- 优化投资组合
- 规避风险
- 降低交易成本
- 提升公司盈利能力
- 增加公司竞争力
## 1.4 智能投资的分类
　　智能投资可分为两大类：
1. 量化智能投资(Quantitative Trading)：依靠统计学的方法，对指标和因素进行预测和分析，并结合制定的规则进行交易。通过对数据的分析，它能够识别市场中隐藏的信息，对未来的趋势进行预测并提前买入或卖出。
2. 高频智能投资(High Frequency Trading)：采用高频的数据源，如实时行情数据、股票交易数据等，通过对数据进行处理，提取信息，进行有效的交易。通过这种方式，它能够更加灵活地管理仓位，把握交易信号，从而提供比目前更好的表现。
## 2.核心概念与联系
## 2.1 股票市场
### 2.1.1 个股
　　个股指某个特定公司的股票，由简称股（S＆P、美股、港股等称法亦属）后跟数字编码组成，通常用小写字母表示。每年股票发行上市交易，会生成一个独特的代码，该代码在下一次交易中引用。
### 2.1.2 股市
　　股市，即股票交易所，是中国内地、香港和台湾三地建立起来的第一个股市，截止到2018年末，股市交易总量已占国民生产总值的近八分之一。股市是美国、英国、日本和韩国等国家设立的国家级组织，其职责就是根据预先确定的规则，通过经纪人与证券经营者之间的交易，在证券市场中自由流通。
### 2.1.3 股市概念
#### 2.1.3.1 开盘价
　　开盘价是指当天股价的最初发行价，等于前一日收盘价。开盘价一般较高，因为它只是上市交易中的一个初始价格。若要确定开盘价，需要考虑股票当前的供需情况，以及相关指标的状况。在交易结束之前，开盘价不会发生变化。
#### 2.1.3.2 最高价
　　最高价是指今天股价的最高点，同一天的最高价可能会出现多达十几个，具体取决于市场行情。
#### 2.1.3.3 最低价
　　最低价是指今天股价的最低点，同一天的最低价可能会出现多达十几个，具体取决于市场行情。
#### 2.1.3.4 收盘价
　　收盘价是指当天股价的最后一笔成交价，是该交易日的最终价格。收盘价是市场买卖双方的共识。一旦当天股价的最高价和最低价都确认下来，该天的收盘价就确定了。
#### 2.1.3.5 涨跌幅
　　涨跌幅是指当天股价相对于昨天的涨跌幅，百分比单位。涨跌幅反映了市场对某只股票的基本看法，主要用于判断股票的走势。对于一般投资者来说，涨跌幅比具体价格的数值更为重要。
#### 2.1.3.6 五分钟均线
　　五分钟均线是一种技术指标，它根据过去五分钟内的股价数据计算出来的中间值，是判断当前股价走势的关键技术。五分钟均线还可以作为趋势判断的辅助工具。
