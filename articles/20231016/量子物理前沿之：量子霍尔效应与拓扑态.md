# 1.背景介绍


## 1.1 什么是量子霍尔效应？
量子霍尔效应（quantum Hall effect）是指超导材料在平衡位置上，电流产生的偏振称为主效应（main effect）。但是随着温度升高，超导材料就会失去平衡，超导体会出现不同层之间的“垂直”偏振，这种现象叫做量子霍尔效应。在量子霍尔效应中，通过改变电子的空间分配，一个费米子中的两个激发能级可以相互作用形成两个不同方向的偏振，并且这些偏振发生相反的变化。因此，在一个良好的分层结构中，只有在存在电子介质的平衡时才可能出现全耦合的局域自旋，而在量子霍尔效应出现后，分层结构就会被破坏掉，从而导致全耦合局域自旋消失或者凸显出来。另一方面，量子霍尔效应也可以用来解释由莫里波紊乱引起的拓扑态，即热力学自由能的斜率随温度的变化而改变。

## 1.2 为何要研究量子霍尔效应？
研究量子霍尔效应主要是为了解决超导材料长期不平衡带来的问题。超导材料的平衡性受到超导电路的控制，通常来说，超导体经过一段时间的使用之后就会失去平衡，从而导致电子的空间分布发生漂移。然而，只靠改变外界环境并不能完全解决这个问题，需要更进一步的量子手段来控制其平衡性。量子霍尔效应就是一种量子效应，能够在一定条件下将多个低lying level电子连接起来，改变它们的相互作用的方式，实现其平衡。量子霍尔效应的实验基础仍处于早期阶段，目前有两种观点认为其存在可能性。一种观点认为，它是一种新型的自旋多重体缺陷，可以通过研究它来制备非厄米体、强基态超导体或弱相互作用的高等级晶格材料。另一种观点认为，它是由于局域自旋无法构建完整的自旋网络而导致的退相干现象。


# 2.核心概念与联系
## 2.1 定义
### 2.1.1 基本原理
量子霍尔效应是指当一个固体的高阶自旋演化到达瓶盖状态时，低阶自旋可能偏转。这一现象是因为具有共同的中心态的许多自旋子在量子态迁移中损失，这些态迁移一般遵循弱色散规则。当晶格绕晶体的晶体层数越多，这类损失就会越严重。随着温度升高，这一现象就变得更加明显，甚至会引起“热涨”现象，产生辐射效应，影响到光的传播。另外，在热力学自由能的斜率随温度的变化而改变，使得它逆转了一些，这就是为什么超导材料出现环状晶粒的原因。

### 2.1.2 分层材料的特点
由于热带的特性，超导材料一般包含两种晶粒，即正极材料和负极材料，分别包裹在不同层的超导体上。正极材料上含高电压，负极材料则接近零阻抗。正极材料具有类似的费米面，负极材料则在不同的频率位置上与正极材料有交叉对称性，这种奇特的物理性质使得可微瑞利曼轨道能够贮存超导体中的电子，并进行有效的磁性传递。此外，超导材料还有一个特殊的性质——亚稳态，即当超导体处于平衡态时，相邻的两个层上的离子的色散相等，而且层之间的色散随温度的变化呈线性关系。


## 2.2 拓扑态
### 2.2.1 热力学自由能
热力学自由能（Helmholtz free energy）是一个物理量，用以描述粒子和无序系统的均势。对于一个固定的物理过程P，其热力学自由能E(P)与该过程相关联，且总是与某种宏观尺度上的平均行为密切相关。例如，自由能与哈密顿场有关，但与具体的约束条件和系统构型无关。其物理意义是，如果考虑了所有可能的宏观尺度上可能的行为，那么总的自由能E(∞)就是系统内部自由能。

### 2.2.2 晶格结构的相互作用
晶格结构的相互作用与热力学自由能息息相关。实际上，晶格结构的相互作用既包括相邻的相互作用，也包括层间的相互作用。因此，当温度升高时，每个晶格尺寸都要随之缩小，这意味着各层之间的相互作用也随之减弱。与此同时，相邻的层往往具有类似的相互作用，因此它们之间的相互作用也减弱了。因此，冰淇淋上雪茄一样的圆圈越堆越多，越来越接近密度均匀的分布，这也表明了层间的相互作用越来越弱。
