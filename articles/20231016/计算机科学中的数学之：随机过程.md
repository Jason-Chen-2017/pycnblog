# 1.背景介绍


## 什么是随机过程？
随机过程（stochastic process）是指对一个时间或空间变量的取值随时间的变化而产生的序列。换句话说，就是一个具有随机性质的一系列现象或现象的发生、发展及演变过程。比如天气预报、股票价格、人口数量、运动轨迹等都是随机过程的例子。随机过程可以分为两个大的类别——连续型随机过程（Continuous stochastic processes，简称CSRPs）和离散型随机过程（Discrete stochastic processes，简称DSRPs）。下面分别对这两种类型的随机过程进行介绍。
### CSRPs（连续型随机过程）
CSRPs是在时间上可区分的随机过程。其典型代表就是随机波动（random walk），即一维空间中一段时间内，变量沿着固定的方向变化不断加速度，但总体趋势是保持稳定的随机过程。另外，常用的CSRPs还有均匀分布、指数分布、泊松分布等。由于随机性质导致，CSRPs的研究也具有极高的应用价值，比如金融市场的价格动量交易、制造业生产规划、工厂流水线控制等都涉及到CSRPs的研究。
### DSRPs（离散型随机过程）
DSRPs是一种在时间上不可区分的随机过程，该过程的时间间隔通常以“瞬时”（instantaneous）、“短期”（short-term）或“长期”（long-term）为单位。常用的DSRPs包括两种——马尔可夫链和扩散过程。
#### 马尔可夫链（Markov chain）
马尔可夫链是最基本的离散型随机过程，其特点是状态转移概率依赖于当前状态，而与过去无关。马尔可夫链由一系列状态组成，每一次状态转移只与前一状态相关，因而具有不可记忆性。常见的马尔可夫链有全转置矩阵、次态矩阵和时序矩阵三种形式。马尔可夫链还可以用于模拟投资市场，分析经济指标等。
#### 扩散过程（Diffusion process）
扩散过程又称为“方差反射定律”或“漫步方程”。它是一种常用的DSRPs，其特点是状态转移概率不依赖于当前状态，而只与之前状态及系统参数相关。常见的扩散过程有布朗运动、方波、分形曲线等。扩散过程可以用于模拟流行病、传染病的传播、混合气体的物理行为等。
# 2.核心概念与联系
## 随机变量
随机变量（Random variable）是描述随机现象的统计量。随机变量是一个函数，将任一输入值映射到实数或实数集。随机变量的定义域一般是一个实数集合，或者称为测度空间（measure space）。

假设X是一个随机变量，则其定义域为D = {x1, x2,..., xn}，其中xi ∈ R（i = 1,2,..., n），称为X的样本空间（sample space）。若Xi为X的一个单独的取值，即Ω = Xi，称Ω为X的事件（event），表示在X服从某一分布下，xi可能出现的结果；若Ω = {x : xi < xj} 或 Ω = {x : xi ≥ xj}，则称Ω为X的半可积（half-interval）事件。

随机变量的特性：

1．独立性：如果两个随机变量X和Y相互独立，则对于任意两个样本点xi和yj，P(X=xi, Y=yj) = P(X=xi) * P(Y=yj)，也就说X与Y没有关联关系。
2．同分布性：如果两个随机变量X和Y具有相同的概率密度函数f(x)，则它们也是同分布的。
3．一致性：如果两个随机变量X和Y存在联合分布P(X,Y)，那么它们是一致的，它们的分布给出了两个随机变量同时发生的概率。

## 概率分布
概率分布（Probability distribution）是随机变量的分布，用来描述随机变量的各个取值的概率。概率分布是一个离散的或连续的概率密度函数，用来估计或计算随机变量的概率。概率分布又分为两大类：一类是雅克比分布（正太分布、高斯分布、拉普拉斯分布等）；另一类是伯努利分布、二项分布、几何分布、狄利克雷分布、负二项分布等。

概率分布的属性：

1．联合概率分布：联合概率分布是一个表格，用来记录随机变量X和Y同时取特定值的概率。
2．边缘概率分布：边缘概率分布也是一个表格，用来记录随机变量X或Y单独取特定值的概率。

## 期望、方差、协方差、相关系数
期望（Expectation）：E[X]表示随机变量X的数学期望或平均值，表示随机变量的均值。期望的计算方法：

1．对于连续型随机变量，其期望可以使用如下公式计算：

    E[X] = int_{-∞}^{+∞} xf(x)dx
    = lim_{n->∞}(xn) / n
    
2．对于离散型随机变量，其期望可以使用如下公式计算：
    
    E[X] = sum_{i=1}^N p_ix_i
    
方差（Variance）：V[X]表示随机变量X的方差，描述随机变量的散乱程度。方差的计算方法：

1．对于连续型随机变量，其方差可以使用如下公式计算：
    
    V[X] = (int_{-∞}^{+∞}(x - E[X])^2 f(x)) / N
    = E[(X-E[X])^2]
    
协方差（Covariance）：Cov[X,Y]表示随机变量X和随机变量Y之间的协方差，描述随机变量X与Y变化的相互依存关系。协方差的计算方法：

1．对于两个连续型随机变量X和Y，其协方差使用如下公式计算：
    
    Cov[X,Y] = E[(X - E[X])*(Y - E[Y])]
    = E[XY - EX*EY]
    
2．对于两个离散型随机变量X和Y，其协方差使用如下公式计算：
    
    Cov[X,Y] = sum_{i=1}^Nx_iy_i - (sum_{i=1}^N x_i)(sum_{i=1}^Ny_i)/N
    
相关系数（Correlation coefficient）：r表示随机变量X和随机变量Y之间的相关系数，用来衡量X与Y之间线性关系的强弱。相关系数的计算方法：

1．对于两个连续型随机变量X和Y，其相关系数可以使用如下公式计算：
    
    r = Corr[X,Y] = cov[X,Y] / sqrt(var[X] * var[Y])
