# 1.背景介绍


射频（RF）与微波（Microwave）是两种不同类型的能量波，但又有着一些共同的特性。它们都是短波宽、高频率、无时程限制的电磁波，其传播速度约为每秒10^9米，即每毫秒3万英里。它们的主要用途包括通讯、遥控、感应、环境保护等。
从工程角度看，射频集成电路（RFIC）一般指的是利用射频电子技术、微生物技术或纳米技术构建的集成电路，具有强大的电力输出功率、高灵敏度、超低功耗、精确控制、适用于各种应用领域。微波集成电路（MIC）则是利用微波电子技术、纳米技术或高分辨率光刻技术构建的集成电路，具有广泛的分类和应用领域。

基于以上对比，我们可以发现，射频和微波两者之间的差异在于：
1. RFIC的晶体管有较大的尺寸，能够集成更多元件；而MIC的晶体管较小，只能容纳一个晶体管，所以MIC的功能更加简单、集中、便宜、高性能。
2. RFIC能够实现高速信号的传输，因而可以实现低延迟通信、数字音视频传输、航空航天、医疗远程诊断、视频监控等应用。而MIC由于功能更简单，无法达到上述功能要求，所以只能用于特殊场景或少数任务。
3. RFIC可以广泛地用于各个领域，可以满足各种需求，功能灵活且广泛。但是价格昂贵，通常需配备研发团队。而MIC的采购成本低廉、可靠性好，供应链齐全，因此被广泛应用于工业、军事、航空航天等领域。
综上所述，射频与微波的不同之处很大，影响了它们的应用和发展方向。如何把握平衡点，才能充分利用两者优势？让我们一起探讨下射频与微波集成电路的关键技术问题。
# 2.核心概念与联系
## 2.1.基本术语
RFIC(Radio Frequency Integrated Circuit)是指利用射频电子技术、微生物技术或纳米技术构建的集成电路。微波集成电路（MIC）是指利用微波电子技术、纳米技术或高分辨率光刻技术构建的集成电路。

在射频技术上，术语“基带”、“载波”、“基带信号”、“噪声”、“衰减脉冲”、“频谱”都有特定的意义。其中，“基带”代表基带信号，是指所有电信号的总称。“载波”代表了在信道中传播的信号。“基带信号”是指除去载波的其他频率信息的电信号。“噪声”代表的是信道中的散乱信号。“衰减脉冲”是指在一个时刻产生的长脉冲，随时间推移会逐渐衰减，最终演变为零。“频谱”是指信号变化的快慢程度。

在微波技术上，术语“基带”、“载波”、“基带信号”、“谐波”、“衰减脉冲”、“频谱”都有特定的意义。其中，“基带”代表基带信号，是指所有电信号的总称。“载波”代表了在信道中传播的信号。“基带信号”是指除去载波的其他频率信息的电信号。“谐波”是指通过频率转换发生的幅度变化。“衰减脉冲”是指在一个时刻产生的长脉冲，随时间推移会逐渐衰减，最终演变为零。“频谱”是指信号变化的快慢程度。

## 2.2.射频集成电路常用的分类方法
- 拉伸聚焊：这是利用绝缘材料通过交流耦合将微弱的导体连接起来的技术，它把微弱的激光信号转变成高频信号，并在空载状态下持续工作。
- 晶体管混杂：这是利用类似蚀蚁蛛网一样的晶体管互相串联的方式，同时将许多微型的电容放在晶体管上的技术。
- 想效设计：这种方法是指利用非对称晶体管结构的特性，结合等效电路的方法来解决电路板的问题。
- 电感技术：这种技术是指采用电感网络的电路原理，使信号在感器前面进行放大，提升高频信号的传输能力。

## 2.3.微波集成电路常用的分类方法
- 半导体制造：是指通过高分辨率光刻的方法来制作微波集成电路的一种方法。
- 硅片制造：是指利用微观结构的硅片将微波信道形成的集成电路。
- 湿疗干涉：是指利用湿疗技术将晶体管结构或者其他微型材料加入电路板中，提高微波集成电路的灵敏度。
- 偏振技术：是指通过偏振在空载状态下的微波的频率调制能力，来实现多个微波源的同时工作。

## 2.4.双频技术与发展趋势
双频是指把两个不同频段的信息加以组合，通过双方对称的特征来提升通信距离和可靠性，是高科技通信发展的一个重要趋势。在通信领域，双频技术已经成为过去几十年最具影响力的发展方向，它极大地增强了通信系统的容量、灵活性、可靠性和安全性。

2017年9月国际无线电联盟（IARU）将双频技术定义为“将同一信号的两个不同频率信号在通信信道中叠加，并以此作为单一信道信号的形式发送”。截至目前，双频技术的应用已经覆盖到了通信设备、技术、工程、经济等各个领域。

双频技术的发展趋势如下：
1. 发射双频：是指发射机直接在两个频段发送数据，不需要通过调制解调器再现处理，使得通信距离远、实质性数据容量大。
2. 中继双频：是指中继站直接接收两个频段的信号，在解调端再现处理，提升信号的带宽，增加系统的容量。
3. 天线双频：是指天线也可在两个频段工作，这样可以在同一个信道上同时收发两个不同频段的信息。
4. 接收双频：是指接收器可同时在两个频段接收信号，并根据信号强度自动切换到最佳的接收频率，提升系统的灵敏度和可靠性。
5. 多模式双频：是指在同一个信道中可以同时传送多个不同模式的信息，比如音频、视频等。
6. 数据双频：是指可以通过双向通信的方式传输数据，也可以是控制指令的双向传送。

## 2.5.微波网络拓扑
微波信道由微波接入点和微波终端构成。微波接入点由接入单元组成，负责接收来自外部环境的射频信号，并按照协议规则将信号编码、调制后发送出去。微波终端则是对外提供服务的实体。
