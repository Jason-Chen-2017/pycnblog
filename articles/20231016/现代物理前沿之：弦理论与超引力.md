# 1.背景介绍


弦理论和超引力是现代物理学的一个重要研究方向。其研究范围涉及到多种多样的物理现象，包括磁性、电荷性和电动机特性、粒子物理与波动、宇宙学方面的信息与暗能量、自然界的稳定结构、材料与固体物理等。弦理论与超引力的研究方法主要有微观和宏观两种。其中微观的方法大都采用数值模拟和实验，如弦论中的弦截面理论（英语：String Theory）、超导超弦物理（英语：Superconducting String Physics）。宏观的方法则利用基本粒子物理定律，如波函数相干变换、弦理论、超导超弦物理、量子光学以及固体物理中的狭义相对论和广义相对论。

20世纪60年代末70年代初，法国科学家莫纳斯·帕拉纳德·奥古斯特·海森堡（<NAME> Ogata）和他的学生亚当·戈尔巴格（Adrian Godzikh）等人提出了弦理论。他们假设弦是无限细长的微粒的聚合体，这种假设使得弦理论具有很高的普适性，而不仅仅局限于铜管、金属管或光纤这样的简单的管道结构。因此，弦理论也被称作“无限弦理论”。它是电气工程、电子学、材料科学、光学、天文学、生态学、心理学、航空航天、金融经济学、信息安全、控制理论、传感器技术、生物医学工程、工程力学等领域的一场革命性变革。

1958年，美国物理学家皮耶纳·马歇尔（Pierre Mariévêl）首次提出了超弦理论。他将弦理论扩展到一定长度上，即将弦拓展到原子核之间，并假设其力度远超过真空中的弦。马歇尔的超弦理论极大地丰富了弦理论的内容，引起了物理学界的重视。直到1987年才获颁诺贝尔奖。

2019年1月，国际著名物理学家马里兰·费舍尔（Marilyn Feeschle）博士在自己的个人网站上发布了一篇题为《没有弦的未来：弦理论为什么失败了？》的文章。她指出，弦理论的创始人们认为弦是没有必要的，因为铜、铝、铅等金属材料可以代替弦的功能。但是，她又发现，弦比金属更容易受到外力影响，而且随着金属含量的增加，弦的力度就下降了，所以弦理论的理想模型就破灭了。

在目前的时代背景下，如何正确认识弦理论的前景和未来是一个值得思考的问题。如何避免弦理论的“毁灭性毒素”（toxic poison）扩散蔓延、如何保障人类科技发展利益与长久健康发展，成为科学家们共同关注的话题。如何提升实用性、普遍性、先进性、可靠性、有效性，成为正确认识、行动指南和沟通桥梁，成为学术讨论的关键词。
