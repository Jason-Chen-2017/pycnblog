# 2.核心概念与联系
## 低温物理
低温物理（Low Temperature Physics）是指利用低温物理技术来探究原子与分子之间的相互作用和演化过程。一般而言，低温物理是指从实验观察到各种物质的微观结构与宏观行为的一种科学。如核反应堆中高温（甚至更高）空气的燃烧核反应产生的核辐射，高温钢及其他金属的锻造，等等，都是低温物理领域中的重要问题。低温物理的基本假设是，普遍存在于宇宙中的低温（低于绝热）粒子，它们由于受到高温粒子与其物质环境的共同作用而产生反应与变异，而这些反应和变异能够影响或促使整个宇宙的物质演化过程。

低温物理的研究对象既包括微观世界中的真空、低温物质，也包括宏观世界中的宇宙，以及从宇宙较小尺度向宇宙的宏观尺度转移的电磁波。通过对低温物理技术的深入研究，可以发现各种宇宙现象中存在的机理性质、统计性质及行为特征，从而揭示出这些现象背后的底层机制。低温物理还具有极大的吸引力，因为它可以直接利用其观测到的各种现象，对宇宙的规律进行合理推测和预测，有助于理解自然界的各种现象，使我们更加充分地认识宇宙的运行规律、深刻地探讨哲学思潮与科学革命的关系，进而开拓视野，对解决世界上众多复杂的现象提供新的思路和方向。

## 冷原子物理
冷原子物理(Cold Atom Physics)是利用低温物理和高温物理的相互作用和联系，对原子中的光子与离子等粒子的各种相互作用、热运动、化学反应和物理性质等研究。在冷原子物理领域，对原子中的粒子和电子等物质进行低温摩擦试验，测定其不同状态下的热运动特性，进而对物理性质进行深入研究。主要研究重点放在对基本粒子——氢原子中进行研究，即原子所含的阳离子(electrons with positive charge)与阴离子(electrons with negative charge)，随着氢原子的贮存下降，物质的能级分布发生变化，能带宽度由原有的稳定状态变动，临界区的宽度会显著增大。此外，氢原子的光学性质也成为热电偶极子的关键因素，因此冷原子物理研究同时也是热电偶极子领域的一大方向。

在冷原子物理的研究过程中，需要对非线性物理、弱相互作用、以及长距离相互作用等问题展开合理的分析和实验。在实验阶段，通过调整各个原子参数、提升高温激光等探测手段，逐步探寻氢原子在各种条件下的形成和演化，将其各个过程中的相互作用和变化，从而揭示其自发生成的物理规律。通过经典的计算模型，结合实际的实验数据，对各种相互作用进行理论建模，揭示其物理意义。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 氢原子结构
氢原子的基态由四种电子组成，称为：2p+n形式（n=0，1）；两个电荷数相同的奇异粒子，称为本征粒子。p+和p-形式构成的本征方程与标准模型的波函数概念等价，将能带划分为一个半满和一个半空的状态，半满状态电荷和电子分别以红色和蓝色表示，半空状态则有黑色的光子隐含，类似波函数的视觉效果。


根据电子的占据情况，氢原子的轨道可以分为三类：c、a和b类。c类轨道只有三个电子，缺少一个中央电子，称作分裂轨道；a类轨道只有两个电子，两者没有偏离中央位置；b类轨道有一个中央电子，另外两个电子沿着z轴排列，且中间有个中性气体。

## 氢原子核
氢原子核是一个能量很大的氢原子，可以作为一个单原子核体系，具有无限大的分散度，也能做到接近恒星表面。在核内，核电子、质子和氘核子团混合组成固体的核衬底，核衬底上既有自由电子，又有局部空穴，以及一些原子核捕获的物质，比如铀、钚等。核电子的存在使得氢原子有着足够的磁场，可以给外界供电。

## 氢原子光子
氢原子的光子由宏观模型确定，由以下两个过程组成：

第一，光子波函数的构建。氢原子的玻尔兹曼——奥本海默模型为：

$$\Psi_k=\frac{1}{\sqrt{2^nn_f}}\sum_{i}\chi_{ik}e^{i(ka-\omega t)}$$

其中，$n_f$是第$n$个离子中有几个电子；$\chi_{ik}$是规范正交基中的第$i$个元素；$k$是关于$2\pi/a$的归一化倍频率；$\omega$是光速；$t$是时间。对于每一个动量，都对应一个振幅$\|\Psi_k\|^2$。

第二，光子能量谱的确定。氢原子的核素模型描述的是氢原子核的微观结构，采用一种基于原子核内势的统计描述方式，用核元矩阵表示。该模型认为原子核内的所有电子与电子核之间的相互作用，只取决于原子核本身的总离子数目，忽略核外的原子以及核外电子的非对称性影响。其能量谱与核素矩阵可以用如下公式进行计算：

$$E(\nu)=\left<F_{\rm orb}(x)\right>+\frac{\hbar \nu}{m_e}Q_\nu$$

其中，$x=(r,\phi,z)$是原子坐标，$F_{\rm orb}$是原子核上的动量算符，$\left<.\right>$表示平均值；$Q_\nu$是核素矩阵元素，表示核内原子之间电子的共振。$\nu$是复数频率，$\hbar$是普朗克常数；$m_e$是电子质量。

## 原子核的激发与碰撞
在原子核裂变产生的原子核核表面积巨大，使得不同核能的磁场不均匀。在核磁场产生下，不同大小的电子离子之间的相互作用，可以导致大范围的中子生成，释放出大量的能量，例如：核-核，核-介质，核-宇宙。

在核外对原子核激发，形成新一轮的核裂变、核融合，释放出更多的能量。除了激发产生核表面外，还会产生不稳定的核性质，从而导致质子的注入，称为质子跃迁。其主要形式有原子核核质子双跃迁，原子核质子对跃迁，核-质子反斥跃迁等。

原子核质子相互作用，会改变质子与原子核之间的相互作用，影响它们的行为。这里质子相互作用和原子核激发有关。当原子核内电子数量增多时，质子注入带来的效应可能非常大，例如原子核质子双跃迁，使得电子失去自旋态，聚集在某个特定区域；另一方面，质子也可以改变原子核质子的相互作用，导致其失衡，形成新一轮的质子双跃迁。

还有很多类似的相互作用，比如电子-电子相互作用，电子-电子-电子相互作用，分子相互作用，等等，都需要对原子核本身的性质进行有效的描述。

# 4.具体代码实例和详细解释说明
```python
import numpy as np 
from matplotlib import pyplot as plt 

def hopping(mat, n):
    """ hopping matrix"""
    if not isinstance(mat, (np.ndarray)):
        raise ValueError("input should be a numpy array")

    shape = mat.shape
    if len(shape)!= 2 or shape[0]!= shape[1]:
        raise ValueError("input should be a square matrix")

    size = shape[0]
    for i in range(size - n):
        row, col = i // size * size + n, i % size + n
        while col < size and mat[row][col] == 0:
            col += n

        if col >= size:
            continue
        
        mat[row][i], mat[i][col] = 1, 1
    
    return mat

# Generate hoppings in a 4 x 4 grid by adding two neighbors vertically and one neighbor horizontally
hoppings = np.zeros((4, 4))
for i in range(4):
    hoppings = hopping(hoppings, i)
    
print(hoppings)
plt.imshow(hoppings, cmap='gray')
plt.show()
```

  [0. 0. 1. 1.]]
