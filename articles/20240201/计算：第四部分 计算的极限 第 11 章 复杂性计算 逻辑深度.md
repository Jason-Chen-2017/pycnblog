### 素数判定

* **素数** 是指只能被 1 和自身整除的自然数。
* **素数判定** 是指判断一个自然数是否为素数。

#### 素数判定算法

* 素数判定算法通常分为两种：试除法和筛选法。
* 试除法是一种简单直观的算法。
* 筛选法则是一种更加高效的算法。

##### 试除法

* 试除法是一种简单直观的算法。
* 它的基本思想是：对于给定的自然数 $n$，从小到大尝试除 $n$。
* 如果 $n$ 能够被某个数整除，则说明 $n$ 不是素数。
* 否则说明 $n$ 是素数。

###### 数学模型公式

* 试除法的数学模型公式如下：
$$
f(n) = \begin{cases}
0 & \text{if } \exists p \in \mathbb{N}, s.t. p < n, p | n \\
1 & \text{otherwise}
\end{cases}
$$

##### 筛选法

* 筛选法是一种更加高效的算法。
* 它的基本思想是：将所有小于 $n$ 的自然数按照某种顺序排列。
* 然后从小到大依次检查每个数，如果发现它是素数，则记录下来。
* 在检查完所有数之后，就可以得到所有小于 $n$ 的素数。

###### 数学模型公式

* 筛选法的数学模型公式如下：
$$
P_i = \begin{cases}
i & \text{if } i \text{ is prime} \\
P_j & \text{if } \exists j \in \mathbb{N}, s.t. P_j < i, P_j | i \\
0 & \text{otherwise}
\end{cases}
$$

### 旅行商问题

* **旅行商问题** 是指：给定一个图，求出一个使得路径最短的回路。

#### 旅行商问题算法

* 旅行商问题算法通常分为两种：贪心算法和动态规划算法。
* 贪心算法是一种简单直观的算法。
* 动态规划算法则是一种更加高效的算法。

##### 贪心算法

* 贪心算法是一种简单直观的算法。
* 它的基本思想是：从起点开始，每次选择离当前位置最近的点作为下一个点。
* 重复上述过程，直到回到起点为止。

###### 数学模型公式

* 贪心算法的数学模型公式如下：
$$
f(S, v_i) = \min_{v_j \notin S} d(v_i, v_j)
$$

##### 动态规划算法

* 动态规划算法是一种更加高效的算法。
* 它的基本思想是：将问题分解成多个子问题，求出子问题的解，然后合并子问题的解得到原问题的解。

###### 数学模型公式

* 动态规划算法的数学模型公式如下：
$$
f(S, i) = \min_{0 \leq j < i} f(S \cup \{v_j\}, j) + d(v_j, v_i)
$$

### 素数判定

* 下面是一个简单的试除法实现：
```python
def is_prime(n):
   if n <= 1:
       return False
   for i in range(2, int(n ** 0.5) + 1):
       if n % i == 0:
           return False
   return True
```

#### 算法分析

* 时间复杂度：$O(\sqrt{n})$。
* 空间复杂度：$O(1)$。

#### 算法优化

* 可以通过预先生成一个素数表来优化算法。
* 这样只需要查表即可。

### 旅行商问题

* 下面是一个简单的贪心算法实现：
```python
import heapq

def tsp(distances):
   """
   计算旅行商问题的最短路径
   :param distances: 邻接矩阵
   :return: 最短路径
   """
   n = len(distances)
   visited = [False] * n
   path = [0] * (n + 1)
   path[0], visited[0] = 0, True
   heap = []
   for i in range(1, n):
       heapq.heappush(heap, (distances[0][i], i))
   while heap:
       d, i = heapq.heappop(heap)
       if not visited[i]:
           path[len(path) - 1] = i
           visited[i] = True
           for j in range(n):
               if not visited[j]:
                  heapq.heappush(heap, (d + distances[i][j], j))
   return path
```

#### 算法分析

* 时间复杂度：$O(n^2)$。
* 空间复杂度：$O(n)$。

#### 算法优化

* 可以通过动态规划算法进一步优化算法。
* 动态规划算法的时间复杂度为 $O(n^2 2^n)$。
