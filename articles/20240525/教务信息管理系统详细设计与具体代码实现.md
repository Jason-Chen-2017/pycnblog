教务信息管理系统包括以下几个核心概念：

1. 学生信息：包括姓名、学号、年龄、性别、班级、学科成绩等。

1. 添加学生信息：

```python
import pandas as pd

students = pd.DataFrame(columns=["学号", "姓名", "年龄", "性别", "班级", "学科成绩"])

def add_student(学号, 姓名, 年龄, 性别, 班级, 学科成绩):
    students.loc[len(students)] = [学号, 姓名, 年龄, 性别, 班级, 学科成绩]

add_student(1, "张三", 20, "男", "1班", 90)
```

2. 查询学生信息：

```python
def query_student(学号):
    return students.loc[students["学号"] == 学号]
```

3. 删除学生信息：

```python
def delete_student(学号):
    students = students.drop(students.loc[students["学号"] == 学号].index)
```

4. 更新学生信息：

```python
def update_student(学号, 姓名, 年龄, 性别, 班级, 学科成绩):
```
