## 4. 数学模型和公式详细讲解举例说明

金融数据标注过程中，数学模型和公式的应用非常重要。以下是一些常见的金融数学模型和公式：

1. 波动率（Volatility）：波动率是衡量金融市场波动程度的指标，通常使用标准偏差计算。

公式：$$\sigma = \sqrt{\frac{1}{n-1}\sum_{i=1}^{n}(r_i - \bar{r})^2}$$

2. Sharpe 比率（Sharpe Ratio）：Sharpe 比率是衡量投资组合回报与风险之间平衡程度的指标。

公式：$$Sharpe Ratio = \frac{E[R_p] - R_f}{\sigma_p}$$

3. Black-Scholes 方程（Black-Scholes Equation）：Black-Scholes 方程是 Optionspricing 的基石，用于计算期权的理论价值。

公式：$$C(S, t) = SN(d_1) - Xe^{-rt}N(d_2)$$

        return 1
    else:
