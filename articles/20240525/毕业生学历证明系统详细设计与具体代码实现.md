## 4. 项目实践：代码实例和详细解释说明

以下是一个简单的毕业生学历证明系统的代码实例，使用Python和Flask框架进行开发：

```python
from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///graduation.db'
db = SQLAlchemy(app)

class Graduation(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80), unique=True, nullable=False)
    major = db.Column(db.String(120), nullable=False)
    level = db.Column(db.String(120), nullable=False)

@app.route('/search', methods=['GET'])
def search():
    name = request.args.get('name')
    result = Graduation.query.filter_by(name=name).first()
    if result:
        return jsonify({'name': result.name, 'major': result.major, 'level': result.level})
    else:
        return jsonify({'error': 'Not found'}), 404

if __name__ == '__main__':
    db.create_all()
    app.run(debug=True)
```
