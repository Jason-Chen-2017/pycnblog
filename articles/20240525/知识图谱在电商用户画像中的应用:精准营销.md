## 5. 项目实践：代码实例和详细解释说明

为了实现知识图谱在电商用户画像中的应用，我们可以使用Python编程语言和相关的数据处理库。以下是一个简单的代码实例：

```python
import networkx as nx
import pandas as pd

# 数据收集与整理
data = pd.read_csv('user_data.csv')

# 实体识别与关系抽取
G = nx.DiGraph()
for row in data.iterrows():
    user_id, actions = row
    for action in actions:
        item_id, relation = action
        G.add_edge(user_id, item_id, relation=relation)

# 用户画像构建
user_portraits = {}
for user_id in G.nodes():
    portrait = {}
    for item_id, edge in G.edges(user_id, data=True):
        portrait[edge['relation']] = item_id
    user_portraits[user_id] = portrait

# 个性化推荐
for user_id, portrait in user_portraits.items():
    print(f"User {user_id} recommends:")
    for relation, item_id in portrait.items():
        print(f"{relation}: {item_id}")
```
