## 5. 项目实践：代码实例和详细解释说明

在此，我们将以Python为例，展示如何实现教学在线网站的核心功能。我们将使用Scikit-learn库实现协同过滤推荐算法。

```python
import numpy as np
from sklearn.feature_extraction import TruncatedSVD
from sklearn.metrics.pairwise import cosine_similarity

# 假设我们已经获得了用户-课程矩阵A
U, V = np.random.rand(100, 20), np.random.rand(20, 100)

# 使用奇异值分解进行矩阵分解
svd = TruncatedSVD(n_components=10)
U_reduced = svd.fit_transform(U)

# 计算相似性矩阵
cosine_sim = cosine_similarity(U_reduced)

# 假设我们正在为用户1推荐课程
user_id = 1
similar_courses = np.argsort(-cosine_sim[user_id])[1:11]
print("Recommended courses for user 1:", similar_courses)
```
