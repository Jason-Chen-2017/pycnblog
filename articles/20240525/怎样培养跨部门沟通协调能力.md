## 5. 项目实践：代码实例和详细解释说明

在实际项目中，跨部门沟通协调能力的培养需要通过实践操作来验证。以下是一个简单的代码示例，展示了如何在项目中使用多个部门的数据：

```python
import pandas as pd

# 读取不同部门的数据
department1_data = pd.read_csv("department1_data.csv")
department2_data = pd.read_csv("department2_data.csv")

# 合并不同部门的数据
combined_data = pd.concat([department1_data, department2_data], axis=1)

# 进行数据分析
result = combined_data.groupby("department").mean()

# 输出结果
print(result)
```

在这个例子中，我们从两个部门（部门1和部门2）读取数据，然后将它们合并到一个数据框中。最后，我们对合并后的数据进行分组和平均计算，以便更好地理解不同部门之间的关系。
