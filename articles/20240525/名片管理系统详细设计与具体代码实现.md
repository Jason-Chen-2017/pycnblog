## 4. 项目实践：代码实例和详细解释说明

在此处，我们将使用 Python 语言实现名片管理系统。以下是一个简单的代码示例：

```python
class Card(object):
    def __init__(self, name, phone, email):
        self.name = name
        self.phone = phone
        self.email = email

class CardManager(object):
    def __init__(self):
        self.cards = []

    def add_card(self, card):
        self.cards.append(card)

    def search_card(self, name):
        for card in self.cards:
            if card.name == name:
                return card
        return None

    def edit_card(self, card):
        self.cards[self.cards.index(card)] = card

    def delete_card(self, card):
        self.cards.remove(card)
```
