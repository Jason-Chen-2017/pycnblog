## 2. 核心概念与联系

首先，我们需要理解大型语言模型（LLM）及其在多代理系统中的作用。LLM 是一种人工智能技术，通过学习大量文本数据来生成文本。这种技术已经被广泛应用于各种任务，如自然语言处理、机器翻译、问答系统等。我们可以将 LLM 的输出作为多代理系统的“语言”，以实现代理之间的通信和协作。

其次，我们需要理解多代理系统（MAS）是一个由多个代理组成的系统，其中每个代理可以独立地执行任务，并与其他代理进行通信和协作。多代理系统的目标通常是实现更高效的决策和自动化。我们将探讨如何利用 LLM 作为多代理系统的“语言”，以实现更高效的决策和自动化。
