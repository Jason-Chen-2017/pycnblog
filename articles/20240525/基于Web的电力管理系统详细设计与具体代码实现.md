## 4. 项目实践：代码实例和详细解释说明

下面是一个基于Web的电力管理系统的代码实例：

```python
import requests

def get_power_data():
    url = "http://example.com/api/power_data"
    response = requests.get(url)
    data = response.json()
    return data

def display_power_data(data):
    for item in data:
        print(f"{item['time']}: {item['power']}W")

if __name__ == "__main__":
    data = get_power_data()
    display_power_data(data)
```

上述代码首先导入了requests库，用于发送HTTP请求。然后定义了两个函数：`get_power_data`用于获取电力数据，`display_power_data`用于显示电力数据。最后，在主程序中调用了这两个函数，获取并显示电力数据。
