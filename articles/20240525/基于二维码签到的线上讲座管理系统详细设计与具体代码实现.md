### 3.1 生成唯一的二维码

我们使用了一个开源的二维码生成库 `qrcode`，通过调用该库的 `generate` 方法，可以生成一个唯一的二维码。

```python
import qrcode

def generate_qrcode(content):
    qr = qrcode.QRCode(
        version=1,
        error_correction=qrcode.constants.ERROR_CORRECT_L,
        box_size=10,
        border=4,
    )
    qr.add_data(content)
    qr.make(fit=True)
    img = qr.make_image(fill_color="black", back_color="white")
    return img
```

### 3.2 识别扫描的二维码

为了识别扫描的二维码，我们使用了一个开源的二维码识别库 `pyzbar`。通过调用该库的 `decode` 方法，可以将扫描的二维码解析成唯一的内容。

```python
from pyzbar import pyzbar

def scan_qrcode(image):
    decoded_obj = pyzbar.decode(image)
    return decoded_obj[0].data.decode("utf-8")
```
