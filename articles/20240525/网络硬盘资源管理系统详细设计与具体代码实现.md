## 4. 项目实践：代码实例和详细解释说明

在本节中，我们将通过一个简单的代码实例来讲解如何实现网络硬盘资源管理系统。以下是一个简化的代码示例：

```python
import numpy as np
from sklearn.cluster import KMeans

# 收集网络硬盘资源的数据
data = np.array([[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]])

# 使用K-means聚类算法对数据进行分类
kmeans = KMeans(n_clusters=2)
kmeans.fit(data)

# 获取聚类结果
labels = kmeans.labels_

# 根据聚类结果对网络硬盘资源进行分类
for i in range(len(labels)):
    print(f"网络硬盘资源{i}属于类别{labels[i]}")
```

在这个代码示例中，我们首先收集了网络硬盘资源的数据，然后使用K-means聚类算法对数据进行分类。最后，我们根据聚类结果对网络硬盘资源进行分类。
