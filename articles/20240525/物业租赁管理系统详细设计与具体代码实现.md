## 4. 项目实践：代码实例和详细解释说明

为了实现物业租赁管理系统，我们需要编写代码。这里我们提供一个简化的 Python 代码实例，展示如何使用 DFS 算法实现租赁合同的签订和物业维护的调度。

```python
import networkx as nx

# 创建图
G = nx.DiGraph()

# 添加节点
G.add_node('A')
G.add_node('B')
G.add_node('C')

# 添加边
G.add_edge('A', 'B')
G.add_edge('B', 'C')
G.add_edge('C', 'A')

# DFS 算法
def dfs(G, u, v):
    visited = set()
    stack = [u]
    while stack:
        u = stack.pop()
        if u not in visited:
            visited.add(u)
            for w in G[u]:
                if w != v:
                    stack.append(w)
                    if w == v:
                        return True
    return False

# 测试
print(dfs(G, 'A', 'C'))
```
