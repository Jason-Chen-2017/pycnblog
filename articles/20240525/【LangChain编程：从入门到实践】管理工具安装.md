```python
from langchain import Chatbot
from langchain import Datastore

# 创建一个数据存储实例
datastore = Datastore()
datastore.load_data("questions.json")

# 创建一个聊天机器人实例
chatbot = Chatbot()
chatbot.train(datastore)

# 向聊天机器人提问
response = chatbot.ask("什么是LangChain？")
print(response)
```

在上面的代码中，我们首先导入了langchain库中的Chatbot和Datastore两个类。然后我们创建了一个Datastore实例，加载了一个名为"questions.json"的数据文件。接着我们创建了一个Chatbot实例，并使用训练好的数据来训练Chatbot。最后我们向Chatbot提出了一个问题，并打印了其回答。
