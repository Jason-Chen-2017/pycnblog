## 4. 数学模型和公式详细讲解举例说明

为了计算计算机散件的价格，我们可以使用以下数学模型：

$$
Price = \sum_{i=1}^{n} P_i \times Q_i
$$

其中，$P_i$ 是第$i$个组件的单价，$Q_i$ 是第$i$个组件的数量，$n$ 是组件的总数。

## 5. 项目实践：代码实例和详细解释说明

为了实现这个系统，我们需要一个合理的代码实现。以下是一个简单的代码示例：

```python
class ComputerPart:
    def __init__(self, name, price, quantity):
        self.name = name
        self.price = price
        self.quantity = quantity

    def total_price(self):
        return self.price * self.quantity

parts = [
    ComputerPart("CPU", 500, 1),
    ComputerPart("Memory", 100, 2),
    ComputerPart("Hard Drive", 50, 1),
]

total_price = sum(part.total_price() for part in parts)
print("Total Price: $", total_price)
```
