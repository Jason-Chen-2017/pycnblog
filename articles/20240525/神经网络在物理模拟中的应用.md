### 4.1 物理现象的数学模型

假设我们要模拟一个自由落体的运动。根据物理学的知识，我们可以得到以下公式：

$$
y = \frac{1}{2}gt^2 + vt
$$

其中，y是物体落到地面的高度，g是重力加速度，t是时间，v是初始速度。

### 4.2 神经网络的数学模型

对于上述问题，我们可以使用一个简单的前向神经网络进行预测。以下是一个简单的前向神经网络的数学模型：

$$
y = W^T \cdot X + b
$$

其中，W是权重矩阵，X是输入数据，b是偏置。

## 5. 项目实践：代码实例和详细解释说明

下面是一个使用神经网络进行物理模拟的简单示例：

```python
import numpy as np
from sklearn.neural_network import MLPRegressor

# 数据生成
def generate_data():
    g = 9.8
    v = np.random.uniform(-10, 10)
    t = np.linspace(0, 2, 100)
    y = 0.5 * g * t**2 + v * t
    return t, y

# 训练数据
t_train, y_train = generate_data()
X_train = np.column_stack((t_train, np.ones(len(t_train))))
mlp = MLPRegressor(hidden_layer_sizes=(10,), max_iter=1000)
mlp.fit(X_train, y_train)

# 预测
t_test, y_test = generate_data()
X_test = np.column_stack((t_test, np.ones(len(t_test))))
y_pred = mlp.predict(X_test)

# 绘图
import matplotlib.pyplot as plt
plt.plot(t_test, y_test, label='真实值')
plt.plot(t_test, y_pred, label='预测值')
plt.legend()
plt.show()
```
