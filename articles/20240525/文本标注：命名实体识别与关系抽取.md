```python
import spacy
from spacy import displacy

nlp = spacy.load("en_core_web_sm")
text = "Barack Obama was born in Hawaii. He is the 44th president of the United States."

doc = nlp(text)
for ent in doc.ents:
    print(ent.text, ent.label_)
```

```

He PRP

United States GPE
```
