## 5. 项目实践：代码实例和详细解释说明

```python
import networkx as nx

# 创建一个有向图
G = nx.DiGraph()

# 添加节点和边
G.add_node("A")
G.add_node("B")
G.add_edge("A", "B", weight=10)

# 使用A*算法进行路径规划
path = nx.shortest_path(G, source="A", target="B")

print(path)
```
