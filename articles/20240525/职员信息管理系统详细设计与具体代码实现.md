
1. 查询算法：

我们需要一个查询算法来查询员工信息。我们可以使用Python的SQLAlchemy库来实现查询算法。

代码示例：

```python
from sqlalchemy import create_engine, Table, MetaData
from sqlalchemy.sql import select

engine = create_engine('sqlite:///employee.db')
metadata = MetaData()
employee = Table('employee', metadata, autoload_with=engine)

def query_employee(name):
    s = select([employee]).where(employee.c.name == name)
    result = engine.execute(s).fetchall()
    return result
```
