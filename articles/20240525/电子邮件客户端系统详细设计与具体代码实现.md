## 5. 项目实践：代码实例和详细解释说明

在本文中，我们将通过代码实例来详细解释如何实现电子邮件客户端系统。以下是一个简单的代码实例：

```python
import smtplib
from email.mime.multipart import MIMEMultipart
from email.mime.text import MIMEText

def send_email(sender, receiver, subject, body):
    msg = MIMEMultipart()
    msg['From'] = sender
    msg['To'] = receiver
    msg['Subject'] = subject
    msg.attach(MIMEText(body, 'plain'))
    try:
        server = smtplib.SMTP('smtp.gmail.com', 587)
        server.starttls()
        server.login(sender, 'your_password')
        text = msg.as_string()
        server.sendmail(sender, receiver, text)
        server.quit()
        print('Email sent successfully')
    except Exception as e:
        print(f"Error: {e}")

sender = 'your_email@gmail.com'
receiver = 'receiver_email@gmail.com'
subject = 'Test Email'
body = 'This is a test email from Python.'
send_email(sender, receiver, subject, body)
```

这个代码实例展示了如何使用Python发送一封电子邮件。

## 9. 附录：常见问题与解答

以下是一些关于电子邮件客户端系统的常见问题与解答：

1. **如何处理邮件过滤**？邮件过滤可以使用Python IMAP库中的search方法来实现，例如：

```python
import imaplib

mail = imaplib.IMAP4_SSL('imap.gmail.com')
mail.login('your_email@gmail.com', 'your_password')
mail.select('inbox')
result, data = mail.search(None, '(FROM "example@example.com")')
```

2. **如何处理邮件删除**？邮件删除可以使用Python IMAP库中的store方法来实现，例如：

```python
result, data = mail.search(None, '(FROM "example@example.com")')
for num in data[0].split():
    mail.store(num, '+FLAGS', '\\Deleted')
    mail.expunge()
```

3. **如何处理邮件标记为已读**？邮件标记为已读可以使用Python IMAP库中的store方法来实现，例如：

```python
result, data = mail.search(None, '(FROM "example@example.com")')
for num in data[0].split():
    mail.store(num, '+FLAGS', '\\Seen')
```
