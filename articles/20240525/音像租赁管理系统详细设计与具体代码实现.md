## 2. 核心概念与联系

音像租赁管理系统的核心概念包括客户（Customer）、租赁项目（Rental Project）、商品（Good）等。这些概念之间存在一定的联系：客户与租赁项目之间是一对多关系，即一个客户可以有多个租赁项目，而一个租赁项目又与多个商品相关联。

我们将使用SQLite作为数据库，设计以下三个表：

1. **客户表**（Customer）
2. **租赁项目表**（Rental Project）
3. **商品表**（Good）

### 5.2 代码实例

以下是音像租赁管理系统的代码实例：

```python
import sqlite3

# 连接数据库
conn = sqlite3.connect('vrms.db')

# 创建客户表
conn.execute('''
CREATE TABLE IF NOT EXISTS Customer (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    phone TEXT NOT NULL
)
''')

# 创建租赁项目表
conn.execute('''
CREATE TABLE IF NOT EXISTS RentalProject (
    id INTEGER PRIMARY KEY,
    customer_id INTEGER,
    good_id INTEGER,
    start_date TEXT,
    end_date TEXT,
    FOREIGN KEY (customer_id) REFERENCES Customer (id),
    FOREIGN KEY (good_id) REFERENCES Good (id)
)
''')

# 创建商品表
conn.execute('''
CREATE TABLE IF NOT EXISTS Good (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    price REAL NOT NULL
)
''')

# 插入数据
conn.execute('''
INSERT INTO Customer (name, phone) VALUES ('John Doe', '1234567890')
''')

conn.commit()
conn.close()
```
