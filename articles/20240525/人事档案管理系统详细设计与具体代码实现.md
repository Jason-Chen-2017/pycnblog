## 1. 背景介绍

人事档案管理系统（ Personnel Record Management System, PRMS ）是一种用于管理企业或机构员工个人档案的信息系统。它的主要功能是记录员工的基本信息、工作经历、教育背景、绩效评估等数据，并提供查询、统计、报表等功能，以便企业或机构进行人力资源管理。

## 2. 核心概念与联系

人事档案管理系统的核心概念包括以下几个方面：

1. 员工档案：员工的个人信息、工作经历、教育背景、绩效评估等数据。
2. 查询：根据员工的不同属性进行查询，例如姓名、职务、部门等。
3. 报表：生成各种报表，例如员工分布、薪资结构、绩效评估等。
4. 安全：确保员工档案的安全性和保密性。

这些概念之间的联系可以通过以下几个方面来体现：

1. 员工档案是系统的核心数据，其他功能都依赖于员工档案。
2. 查询功能是为了方便用户快速获取员工档案中的信息。
3. 报表功能是为了提供企业或机构的人力资源管理决策支持。
4. 安全功能是为了保护员工档案的隐私和安全。

### 4.1. 查询员工信息

假设我们需要查询所有年龄大于30岁的员工的姓名和职务。我们可以使用以下SQL查询语句：

```sql
SELECT name, position FROM employees WHERE age > 30;
```

### 4.2. 统计员工年龄分布

为了统计员工年龄分布，我们可以使用以下SQL查询语句：

```sql
SELECT age, COUNT(*) as num FROM employees GROUP BY age ORDER BY age;
```

### 4.2. 数据库连接

为了连接到MySQL数据库，我们需要使用`pymysql`库。在`config.py`文件中，我们可以定义数据库连接信息：

```python
import pymysql

class Config(object):
    DEBUG = True
    SQLALCHEMY_DATABASE_URI = 'mysql://username:password@localhost/prms'
    SQLALCHEMY_COMMIT_ON_TEARDOWN = True
    SQLALCHEMY_TRACK_MODIFICATIONS = False
    SQLALCHEMY_POOL_SIZE = 50
    SQLALCHEMY_POOL_RECYCLE = 300
```

### 4.3. 数据模型

在`models.py`文件中，我们可以定义员工档案模型：

```python
from flask_sqlalchemy import SQLAlchemy

db = SQLAlchemy()

class Employee(db.Model):
    __tablename__ = 'employees'
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(64), nullable=False)
    gender = db.Column(db.String(10), nullable=False)
    birthday = db.Column(db.Date, nullable=False)
    position = db.Column(db.String(64), nullable=False)
    department = db.Column(db.String(64), nullable=False)
    work_years = db.Column(db.Integer, nullable=False)
    education = db.Column(db.String(128), nullable=True)
    phone = db.Column(db.String(15), nullable=True)
    performance = db.Column(db.String(128), nullable=True)
```
