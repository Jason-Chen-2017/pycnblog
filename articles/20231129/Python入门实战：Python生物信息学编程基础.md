## 3.1 Needleman-Wunsch算法

Needleman-Wunsch算法是一种用于比较两个序列的局部最优比较方法，它可以找到两个序列之间的最长公共子序列（LCS）。算法的核心思想是动态规划。

算法的具体步骤如下：

1. 创建一个二维数组dp，其中dp[i][j]表示序列A的前i个字符和序列B的前j个字符的LCS的长度。
2. 初始化dp数组的第一行和第一列，将其设为0。
3. 遍历序列A和序列B的每个字符，对于每个字符，我们可以进行三种操作：
   - 插入一个新字符，这将导致dp数组的当前位置增加1。
   - 删除当前位置的字符，这将导致dp数组的当前位置减少1。
   - 替换当前位置的字符，这将导致dp数组的当前位置减少1，并且如果替换后的字符与序列B中的字符相同，则增加一个。
4. 遍历完所有字符后，dp数组的最后一个位置将表示序列A和序列B的LCS的长度。

## 3.2 Smith-Waterman算法

Smith-Waterman算法是一种用于比较两个序列的局部最优比较方法，它可以找到两个序列之间的最大局部公共子序列（LCS）。算法的核心思想是动态规划。

算法的具体步骤如下：

1. 创建一个二维数组dp，其中dp[i][j]表示序列A的前i个字符和序列B的前j个字符的LCS的长度。
2. 初始化dp数组的第一行和第一列，将其设为0。

   - 插入一个新字符，这将导致dp数组的当前位置增加1。

4. 遍历完所有字符后，dp数组的最大值将表示序列A和序列B的LCS的长度。

# 4.具体代码实例和详细解释说明

## 4.1 Needleman-Wunsch算法实现

```python
def needleman_wunsch(seq1, seq2):
    m, n = len(seq1), len(seq2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if seq1[i - 1] == seq2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j] - 1, dp[i][j - 1] - 1, dp[i - 1][j - 1] - 1)

    return dp[m][n]
```

```python
def smith_waterman(seq1, seq2):
    m, n = len(seq1), len(seq2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if seq1[i - 1] == seq2[j - 1]:
                dp[i][j] = max(dp[i - 1][j - 1] + 1, dp[i - 1][j], dp[i][j - 1])
            else:
                dp[i][j] = max(dp[i - 1][j] - 1, dp[i][j - 1] - 1, dp[i - 1][j - 1] - 1)

    return dp[m][n]
```
