# 遍历源文件夹中的所有文件
for file in os.listdir(src_folder):
    # 获取文件的完整路径
    src_file = os.path.join(src_folder, file)
    # 获取文件的扩展名
    ext = os.path.splitext(file)[1]
    # 获取目标文件夹的完整路径

## 4.2 GPT大模型AI Agent的具体代码实例

以下是一个使用Python语言编写的GPT大模型AI Agent代码实例，用于自动生成文章：

```python
import openai

# 设置API密钥
openai.api_key = "your_api_key"

# 设置要生成的文章主题
topic = "旅行"

# 生成文章
response = openai.Completion.create(
    engine="text-davinci-002",
    prompt=f"请根据以下主题生成一篇文章：{topic}",
    temperature=0.7,
    max_tokens=150,
    top_p=1,
    frequency_penalty=0,
    presence_penalty=0
)

# 获取生成的文章
generated_article = response.choices[0].text
print(generated_article)
```

在上述代码中，我们首先设置了API密钥。然后，我们设置了要生成的文章主题。最后，我们调用GPT大模型AI Agent的生成接口，并获取生成的文章。这个代码实例展示了如何使用GPT大模型AI Agent自动生成文章。
