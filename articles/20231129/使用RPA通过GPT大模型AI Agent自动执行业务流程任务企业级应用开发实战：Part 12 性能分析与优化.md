# 4.具体代码实例和详细解释说明

在本节中，我们将提供一个具体的代码实例，以及对其详细解释的说明。

```python
import openai

# 设置 API 密钥
openai.api_key = "your_api_key"

# 定义 GPT 模型
gpt_model = "gpt-3.5-turbo"

# 定义自动化任务
def automate_task(task):
    # 使用 GPT 模型处理任务
    response = openai.Completion.create(
        engine=gpt_model,
        prompt=task,
        max_tokens=100,
        n=1,
        stop=None,
        temperature=0.5,
    )

    # 返回 GPT 模型的响应
    return response.choices[0].text.strip()

# 执行自动化任务
task = "请帮我发送一封电子邮件，主题是 '项目进度报告'，正文是 '项目进度如下...'。"
response = automate_task(task)
print(response)
```

在这个代码实例中，我们首先导入了 `openai` 库，并设置了 API 密钥。然后，我们定义了一个 `automate_task` 函数，该函数接收一个任务作为参数，并使用 GPT 模型处理该任务。最后，我们执行了自动化任务，并打印了 GPT 模型的响应。
