## 4.1 自动化测试的代码实例
我们将使用Python的unittest模块来进行自动化测试。首先，我们需要创建一个测试用例类，并定义测试方法。然后，我们可以使用unittest模块的TestSuite类来组织测试用例。

```python
import unittest

class TestMyFunction(unittest.TestCase):
    def test_add(self):
        self.assertEqual(my_function(1, 2), 3)

    def test_sub(self):
        self.assertEqual(my_function(5, 3), 2)

if __name__ == '__main__':
    suite = unittest.TestSuite()
    suite.addTest(TestMyFunction('test_add'))
    suite.addTest(TestMyFunction('test_sub'))
    unittest.TextTestRunner().run(suite)
```

在上述代码中，我们首先导入了unittest模块。然后，我们创建了一个名为TestMyFunction的测试用例类，并定义了两个测试方法：test_add和test_sub。最后，我们使用unittest.TestSuite类来组织测试用例，并使用unittest.TextTestRunner类来执行测试。

## 4.2 性能优化的代码实例
我们将使用Python的timeit模块来进行性能优化。首先，我们需要定义一个需要优化的函数。然后，我们可以使用timeit模块来测量函数的执行时间。

```python
import timeit

def my_function(n):
    result = 0
    for i in range(n):
        result += i
    return result

def optimized_my_function(n):
    result = n * (n - 1) // 2
    return result

start_time = timeit.default_timer()
my_function(1000000)
end_time = timeit.default_timer()
print('my_function执行时间：', end_time - start_time)

start_time = timeit.default_timer()
optimized_my_function(1000000)
end_time = timeit.default_timer()
print('optimized_my_function执行时间：', end_time - start_time)
```

在上述代码中，我们首先导入了timeit模块。然后，我们定义了一个名为my_function的函数，该函数计算1到n的和。接着，我们定义了一个名为optimized_my_function的函数，该函数使用数学公式直接计算结果，从而提高了性能。最后，我们使用timeit模块来测量两个函数的执行时间。
