# 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来详细解释装饰器的使用方法。

```python
def decorator(func):
    def wrapper(*args, **kwargs):
        print("Before calling the function")
        result = func(*args, **kwargs)
        print("After calling the function")
        return result
    return wrapper

@decorator
def my_function():
    print("Inside the function")

my_function()
```

在这个代码实例中，我们定义了一个装饰器函数`decorator`，它接收一个函数作为参数。在`decorator`函数中，我们定义了一个新的函数`wrapper`，这个新的函数会在调用时，在原始函数`my_function`之前或之后执行一些额外的操作。

在原始函数`my_function`之前，我们调用了`wrapper`函数。最后，我们返回了`wrapper`函数。

当我们调用`my_function`时，会先执行`wrapper`函数中的代码，然后再执行原始函数`my_function`中的代码。
