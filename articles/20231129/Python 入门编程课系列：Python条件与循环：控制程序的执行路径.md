# 2.核心概念与联系

在编程中，条件与循环是两个非常重要的概念。它们的核心是控制程序的执行路径，使其能够根据不同的条件进行不同的操作。

## 2.1 条件

条件是一种控制结构，它允许程序根据某些条件来执行或跳过某些代码块。在 Python 中，我们使用 `if` 语句来实现条件判断。`if` 语句的基本格式如下：

```python
if 条件:
    # 执行的代码块
```

条件是一个布尔值，它可以是 `True` 或 `False`。如果条件为 `True`，则执行代码块；如果条件为 `False`，则跳过代码块。

## 2.2 循环

循环是另一种控制结构，它允许程序重复执行某些代码块，直到满足某个条件。在 Python 中，我们使用 `for` 和 `while` 语句来实现循环。

### 2.2.1 for 循环

`for` 循环用于遍历可迭代对象（如列表、字符串等）。其基本格式如下：

```python
for 变量 in 可迭代对象:
    # 执行的代码块
```

在每次迭代中，`变量` 将接收可迭代对象中的下一个元素。当所有元素都被遍历完毕时，循环将自动结束。

### 2.2.2 while 循环

`while` 循环是一种基于条件的循环。它会不断执行代码块，直到条件为 `False`。其基本格式如下：

```python
while 条件:
    # 执行的代码块
```

在每次迭代中，条件会被评估。如果条件为 `True`，则执行代码块；如果条件为 `False`，则跳过代码块。循环会一直持续，直到条件为 `False`。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过具体代码实例来详细解释条件与循环的概念。

## 4.1 条件判断的代码实例

```python
# 定义一个变量
age = 18

# 使用 if 语句进行条件判断
if age >= 18:
    print("你已经成年了！")
else:
    print("你还没有成年。")
```

在这个例子中，我们定义了一个变量 `age`，并使用 `if` 语句进行条件判断。如果 `age` 大于或等于 18，则执行 `print("你已经成年了！")`；否则，执行 `print("你还没有成年。")`。

## 4.2 for 循环的代码实例

```python
# 定义一个列表
numbers = [1, 2, 3, 4, 5]

# 使用 for 循环遍历列表
for number in numbers:
    print(number)
```

在这个例子中，我们定义了一个列表 `numbers`，并使用 `for` 循环遍历列表。在每次迭代中，`number` 将接收列表中的下一个元素，并执行 `print(number)`。

## 4.3 while 循环的代码实例

```python
# 定义一个变量
i = 0

# 使用 while 循环遍历 10 次
while i < 10:
    print(i)
    i += 1
```

在这个例子中，我们定义了一个变量 `i`，并使用 `while` 循环遍历 10 次。在每次迭代中，我们执行 `print(i)` 和 `i += 1`。当 `i` 大于或等于 10 时，循环将自动结束。

# 6.附录常见问题与解答

在本节中，我们将解答一些常见问题，以帮助您更好地理解条件与循环的概念。

## 6.1 条件判断的常见问题与解答

### 问题 1：如何使用多个条件进行判断？

答案：您可以使用 `and` 或 `or` 操作符来连接多个条件。例如：

```python
if 条件1 and 条件2:
    # 执行的代码块
```

或者：

```python
if 条件1 or 条件2:
    # 执行的代码块
```

### 问题 2：如何使用嵌套条件进行判断？

答案：您可以使用 `if-elif-else` 语句来实现嵌套条件判断。例如：

```python
if 条件1:
    # 执行的代码块1
elif 条件2:
    # 执行的代码块2
else:
    # 执行的代码块3
```

## 6.2 循环的常见问题与解答

### 问题 1：如何使用 break 和 continue 语句来控制循环？

答案：`break` 语句用于终止循环，`continue` 语句用于跳过当前迭代并继续下一次迭代。例如：

```python
for number in numbers:
    if number == 3:
        break
    print(number)
```

或者：

```python
for number in numbers:
    if number % 2 == 0:
        continue
    print(number)
```

### 问题 2：如何使用循环来遍历字符串？

答案：您可以使用 `for` 循环来遍历字符串。例如：

```python
for 字符 in 字符串:
    # 执行的代码块
```

在每次迭代中，`字符` 将接收字符串中的下一个字符。
