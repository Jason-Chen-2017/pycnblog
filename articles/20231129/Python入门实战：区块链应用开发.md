# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释区块链技术的实现过程。

```python
import hashlib
import time

class Block:
    def __init__(self, index, previous_hash, timestamp, data, hash):
        self.index = index
        self.previous_hash = previous_hash
        self.timestamp = timestamp
        self.data = data
        self.hash = hash

    def calculate_hash(self):
        sha = hashlib.sha256()
        sha.update(str(self.index).encode('utf-8'))
        sha.update(self.previous_hash.encode('utf-8'))
        sha.update(str(self.timestamp).encode('utf-8'))
        sha.update(self.data.encode('utf-8'))
        return sha.hexdigest()

class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(0, "0", time.time(), "Genesis Block", self.calculate_hash())

    def add_block(self, data):
        index = len(self.chain)
        previous_hash = self.chain[index - 1].hash
        timestamp = time.time()
        hash = self.calculate_hash()
        self.chain.append(Block(index, previous_hash, timestamp, data, hash))

    def calculate_hash(self):
        sha = hashlib.sha256()
        sha.update(str(self.chain[0].index).encode('utf-8'))
        for block in self.chain[1:]:
            sha.update(block.hash.encode('utf-8'))
        return sha.hexdigest()

# 创建一个新的区块链实例
blockchain = Blockchain()

# 添加一个新的区块
blockchain.add_block("新区块数据")

# 获取区块链的哈希值
print(blockchain.calculate_hash())
```

在上述代码中，我们首先定义了一个`Block`类，它用于表示区块链中的一个区块。每个区块包含一个索引、一个前一个哈希、一个时间戳、一组数据和一个哈希值。我们还定义了一个`Blockchain`类，它用于表示区块链。每个区块链包含一个链表，其中包含一个初始区块（称为“基因块”）和其他区块。我们还实现了一个`add_block`方法，用于添加新的区块到区块链中。
