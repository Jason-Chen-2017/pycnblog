   2. 然后，使用RNN对序列进行特征提取，RNN可以学习序列中的依赖关系。
   3. 接着，使用全连接层对特征进行分类，将特征映射到不同的类别上。
   4. 最后，使用回归层对物体的位置进行预测，将物体的位置映射到图像序列中的坐标。

   2. 然后，使用GNN对序列进行特征提取，GNN可以学习序列中的依赖关系。
   3. 接着，使用全连接层对特征进行分类，将特征映射到不同的类别上。
   4. 最后，使用回归层对物体关系进行预测，将物体关系映射到图像序列中的坐标。
