## 1.2 字典和集合的基本概念

字典（Dictionary）和集合（Set）是Python中的两种不同类型的数据结构。字典是一种键值对的数据结构，每个键值对由一个唯一的键和一个值组成。集合是一种无序的、不重复的元素集合，集合中的元素可以是任意类型的数据。

### 1.2.1 字典的基本概念

字典是一种键值对的数据结构，每个键值对由一个唯一的键和一个值组成。字典使用大括号 {} 来表示，键和值之间用冒号 : 分隔。字典的键是唯一的，可以是任意类型的数据，但值可以是任意类型的数据。字典的键值对可以通过键进行访问和修改。

例如，下面是一个简单的字典示例：

```python
# 定义一个字典
my_dict = {"name": "John", "age": 25, "city": "New York"}

# 通过键访问字典的值
print(my_dict["name"])  # 输出: John

# 修改字典的值
my_dict["age"] = 26
print(my_dict["age"])  # 输出: 26
```

### 1.2.2 集合的基本概念

集合是一种无序的、不重复的元素集合。集合中的元素可以是任意类型的数据。集合使用大括号 {} 来表示，元素之间用逗号 , 分隔。集合的元素是唯一的，即使在创建集合时，重复的元素只会被计算一次。

例如，下面是一个简单的集合示例：

```python
# 定义一个集合
my_set = {1, 2, 3, 4, 4, 5}

# 集合中的元素是唯一的，重复的元素只会被计算一次
print(my_set)  # 输出: {1, 2, 3, 4, 5}
```
