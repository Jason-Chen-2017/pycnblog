                 

# 1.背景介绍

随着人工智能技术的不断发展，自动化和智能化已经成为企业竞争力的重要组成部分。在这个背景下，RPA（Robotic Process Automation，机器人化处理自动化）技术的应用也逐渐普及。RPA 技术可以帮助企业自动化处理大量重复性任务，提高工作效率，降低成本。

在RPA技术的应用中，AI（人工智能）技术的发展也为RPA带来了新的机遇。GPT（Generative Pre-trained Transformer）大模型是一种基于Transformer架构的自然语言处理模型，它可以进行文本生成、文本分类、文本摘要等多种任务。在RPA的应用中，我们可以利用GPT大模型来构建AI Agent，让AI Agent自动执行业务流程任务。
