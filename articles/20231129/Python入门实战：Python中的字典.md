# 2.核心概念与联系

字典是Python中的一种数据结构，它是一种键值对的数据结构，可以用来存储和管理数据。字典的键是唯一的，可以是任何不可变的Python对象，如字符串、整数、浮点数等。字典的值可以是任何Python对象，包括其他字典、列表、元组等。字典使用大括号{}来定义，键值对之间用冒号：分隔，键值对之间用逗号，分号分隔。

例如，以下是一个简单的字典示例：

```python
my_dict = {
    "name": "John",
    "age": 25,
    "city": "New York"
}
```

在这个例子中，字典的键是"name"、"age"和"city"，它们对应的值分别是"John"、25和"New York"。

字典的核心概念包括：

- 键：字典中唯一的键，可以是任何不可变的Python对象。
- 值：键对应的值，可以是任何Python对象。
- 键值对：键和值的组合，用冒号：分隔。
- 字典：键值对的集合，用大括号{}定义，键值对之间用逗号分隔。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释Python字典的使用方法。

```python
# 创建一个空字典
my_dict = {}

# 添加键值对
my_dict["name"] = "John"
my_dict["age"] = 25
my_dict["city"] = "New York"

# 访问值
print(my_dict["name"])  # 输出: John

# 删除键值对
del my_dict["name"]

# 遍历字典
for key, value in my_dict.items():
    print(key, value)
```

在这个例子中，我们创建了一个空字典，然后添加了三个键值对。接着，我们访问了字典中的一个值，删除了一个键值对，并遍历了字典中的所有键值对。
