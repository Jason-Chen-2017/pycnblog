### 4.1 推理算法代码实例

以下是一个简单的推理算法代码实例：

```python
def infer(knowledge, question):
    for rule in knowledge:
        if rule.condition(question):
            return rule.conclusion(question)
    return None
```

在这个代码实例中，我们定义了一个 `infer` 函数，它接受两个参数：`knowledge` 和 `question`。`knowledge` 是一个列表，其中每个元素是一个规则，规则包含一个条件和一个结论。`question` 是一个问题。我们遍历 `knowledge` 中的每个规则，并检查其条件是否满足 `question`。如果条件满足，我们返回规则的结论。如果没有满足条件的规则，我们返回 `None`。

### 4.2 学习算法代码实例

以下是一个简单的学习算法代码实例：

```python
def train(X, y, theta, alpha, num_iters):
    m = len(X)
    for _ in range(num_iters):
        h = X.dot(theta)
        error = h - y
        theta = theta - alpha * X.T.dot(error) / m
    return theta
```

在这个代码实例中，我们定义了一个 `train` 函数，它接受五个参数：`X`、`y`、`theta`、`alpha` 和 `num_iters`。`X` 是一个矩阵，其中每一行是一个样本的特征向量。`y` 是一个向量，其中每个元素是一个样本的标签。`theta` 是一个向量，其中每个元素是一个线性回归模型的权重。`alpha` 是一个学习率，用于调整权重的更新速度。`num_iters` 是训练的迭代次数。我们遍历 `num_iters` 次，计算预测值 `h` 和真实值 `y` 之间的差异 `error`，并更新权重 `theta`。最后，我们返回更新后的权重 `theta`。
