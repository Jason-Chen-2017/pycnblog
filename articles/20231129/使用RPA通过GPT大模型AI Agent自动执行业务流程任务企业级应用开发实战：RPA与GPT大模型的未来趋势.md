## 4.2 GPT大模型的代码实例

以下是一个简单的GPT大模型代码实例，它可以用于生成自然语言文本：

```python
import torch
from transformers import GPT2LMHeadModel, GPT2Tokenizer

# 加载预训练模型和tokenizer
model = GPT2LMHeadModel.from_pretrained('gpt2')
tokenizer = GPT2Tokenizer.from_pretrained('gpt2')

# 生成文本
def generate_text(prompt, max_length):
    # 将提示文本转换为输入序列
    input_ids = tokenizer.encode(prompt, return_tensors='pt')

    # 生成文本
    output = model.generate(input_ids, max_length=max_length, num_return_sequences=1)

    # 解码文本
    text = tokenizer.decode(output[0], skip_special_tokens=True)

    return text

if __name__ == '__main__':
    prompt = '使用RPA通过GPT大模型AI Agent自动执行业务流程任务企业级应用开发实战：RPA与GPT大模型的未来趋势'
    max_length = 50
    text = generate_text(prompt, max_length)
    print(text)
```

在上述代码中，我们使用了Hugging Face的transformers库来实现GPT大模型的生成。transformers库提供了许多预训练模型和tokenizer，我们可以直接使用它们来实现自然语言处理任务。
