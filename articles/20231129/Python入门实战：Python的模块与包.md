# 2.核心概念与联系

## 2.1 模块

在Python中，模块是一个包含一组相关功能的文件。模块可以包含函数、类、变量等，可以被其他程序或模块导入使用。模块的文件名后缀为.py。

### 2.1.1 模块的作用

- 模块可以将代码分解成多个部分，使得代码更加可读性和可维护性强。
- 模块可以提高代码的可重用性，因为其他程序或模块可以导入使用。
- 模块可以提高代码的安全性，因为模块内的变量和函数不会泄露给外部。

### 2.1.2 如何创建模块

创建模块只需要创建一个.py文件，并将需要暴露给外部的变量、函数等定义在这个文件中。例如，我们可以创建一个名为mymodule.py的模块，内容如下：

```python
def hello():
    print("Hello, World!")
```

### 2.1.3 如何导入模块

要导入模块，可以使用import语句。例如，要导入上面创建的mymodule模块，可以使用以下代码：

```python
import mymodule
```

### 2.1.4 如何使用模块

要使用模块中的变量或函数，可以直接使用模块名。例如，要调用mymodule模块中的hello函数，可以使用以下代码：

```python
mymodule.hello()
```

## 2.2 包

在Python中，包是一个包含多个模块的目录。包可以将多个模块组织成一个整体，使得代码更加结构化和可维护性强。

### 2.2.1 包的作用

- 包可以将多个模块组织成一个整体，使得代码更加结构化。
- 包可以提高代码的可重用性，因为其他程序或模块可以导入使用。
- 包可以提高代码的安全性，因为包内的模块不会泄露给外部。

### 2.2.2 如何创建包

创建包只需要创建一个包含多个.py文件的目录，并在每个.py文件中定义需要暴露给外部的变量、函数等。例如，我们可以创建一个名为mypackage的包，内容如下：

```
mypackage/
    __init__.py
    mymodule.py
```

其中，__init__.py文件是包的初始化文件，它可以用来定义包中的变量、函数等。

### 2.2.3 如何导入包

要导入包，可以使用import语句。例如，要导入上面创建的mypackage包，可以使用以下代码：

```python
import mypackage
```

### 2.2.4 如何使用包

要使用包中的模块，可以直接使用包名。例如，要调用mypackage包中的mymodule模块中的hello函数，可以使用以下代码：

```python
mypackage.mymodule.hello()
```
