# 4.具体代码实例和详细解释说明

## 4.1 元类示例

```python
class Meta(type):
    def __init__(cls, name, bases, attrs):
        super().__init__(name, bases, attrs)
        cls.new_attr = "new_attr"

class MyClass(metaclass=Meta):
    pass

print(MyClass.new_attr)  # Output: new_attr
```

在上述代码中，我们定义了一个元类`Meta`，它在类的初始化过程中添加了一个新属性`new_attr`。然后，我们创建了一个类`MyClass`，并将`Meta`作为其元类。最后，我们打印了`MyClass`的`new_attr`属性，输出结果为`new_attr`。

## 4.2 装饰器示例

```python
def decorator(func):
    def wrapper(*args, **kwargs):
        print("Before calling the function")
        result = func(*args, **kwargs)
        print("After calling the function")
        return result
    return wrapper

@decorator
def my_function():
    print("Inside the function")

my_function()
```

在上述代码中，我们定义了一个装饰器`decorator`，它用来修改函数的行为。然后，我们使用`@decorator`装饰器语法将`my_function`函数进行装饰。最后，我们调用`my_function`函数，输出结果为：

```
Before calling the function
Inside the function
After calling the function
```

## 4.3 类型注解示例

```python
def my_function(x: int, y: float) -> float:
    return x + y

x = 10
y = 20.5
result = my_function(x, y)
print(result)  # Output: 30.5
```

在上述代码中，我们使用类型注解`x: int`和`y: float`来指定函数`my_function`的参数类型。然后，我们调用`my_function`函数，并将结果打印出来，输出结果为`30.5`。
