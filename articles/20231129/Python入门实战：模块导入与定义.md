# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个具体的代码实例来说明模块的导入与定义的过程。

## 4.1 定义模块
首先，我们创建一个名为`math_utils.py`的Python文件，并定义一个`add`函数：

```python
# math_utils.py
def add(a, b):
    return a + b
```

## 4.2 导入模块
然后，我们在另一个Python文件中，使用`import`关键字导入`math_utils`模块：

```python
# main.py
import math_utils
```

## 4.3 使用模块
最后，我们在`main.py`文件中，使用`from ... import ...`语句将`math_utils`模块中的`add`函数导入到当前作用域中，并调用该函数：

```python
# main.py
from math_utils import add

result = add(1, 2)
print(result)  # 输出: 3
```

- 使用模块的缓存机制：通过使用模块的缓存机制，可以提高模块的导入速度，提高程序的性能。

Q: 如何导入模块？
A: 使用`import`关键字，后面跟着模块的名称即可。例如：`import math_utils`。

Q: 如何使用模块中的内容？
A: 使用`from ... import ...`语句，将模块中的内容导入到当前作用域中。例如：`from math_utils import add`。
