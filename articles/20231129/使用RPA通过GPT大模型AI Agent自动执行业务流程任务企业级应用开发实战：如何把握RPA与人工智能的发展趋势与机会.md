## 4.1 RPA的代码实例

我们可以使用Python语言来编写RPA的代码实例。以下是一个简单的RPA代码实例，它可以从一个Excel文件中读取数据，并将其写入另一个Excel文件：

```python
import openpyxl

# 读取Excel文件
def read_excel(file_path):
    workbook = openpyxl.load_workbook(file_path)
    sheet = workbook.active
    data = []
    for row in sheet.iter_rows(min_row=2):
        data.append([cell.value for cell in row])
    return data

# 写入Excel文件
def write_excel(file_path, data):
    workbook = openpyxl.Workbook()
    sheet = workbook.active
    for row in data:
        for cell in row:
            sheet.cell(row=sheet.max_row + 1, column=cell.column_letter).value = cell
    workbook.save(file_path)

# 主函数
def main():
    input_file_path = 'input.xlsx'
    output_file_path = 'output.xlsx'
    data = read_excel(input_file_path)
    write_excel(output_file_path, data)

if __name__ == '__main__':
    main()
```

在这个代码实例中，我们首先使用`openpyxl`库来读取Excel文件，并将其中的数据存储到一个列表中。然后，我们使用`openpyxl`库来写入Excel文件，并将数据写入到新的Excel文件中。

## 4.2 GPT大模型AI Agent的代码实例

我们可以使用Python语言和Hugging Face库来编写GPT大模型AI Agent的代码实例。以下是一个简单的GPT大模型AI Agent代码实例，它可以生成一段文本：

```python
from transformers import GPT2LMHeadModel, GPT2Tokenizer

# 加载GPT2模型和词汇表
tokenizer = GPT2Tokenizer.from_pretrained('gpt2')
model = GPT2LMHeadModel.from_pretrained('gpt2')

# 生成文本
def generate_text(prompt, max_length=100):
    input_ids = tokenizer.encode(prompt, return_tensors='pt')
    output = model.generate(input_ids, max_length=max_length, num_return_sequences=1)
    text = tokenizer.decode(output[0], skip_special_tokens=True)
    return text

# 主函数
def main():
    prompt = 'Once upon a time'
    text = generate_text(prompt)
    print(text)

if __name__ == '__main__':
    main()
```

在这个代码实例中，我们首先使用`transformers`库来加载GPT2模型和词汇表。然后，我们使用`GPT2Tokenizer`类来编码输入文本，并使用`GPT2LMHeadModel`类来生成文本。
