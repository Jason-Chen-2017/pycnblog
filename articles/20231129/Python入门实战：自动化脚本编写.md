# 4.具体代码实例和详细解释说明
以下是一个简单的自动化脚本实例，用于读取一个CSV文件，并将其中的数据转换为JSON格式：

```python
import csv
import json

# 读取CSV文件
with open('data.csv', 'r') as f:
    reader = csv.reader(f)
    data = list(reader)

# 将CSV数据转换为JSON格式
json_data = json.dumps(data)

# 输出JSON数据
print(json_data)
```

这个脚本的解释说明如下：

1. 首先，我们导入了csv和json模块，这两个模块分别用于处理CSV文件和JSON数据。
2. 然后，我们使用with语句打开CSV文件，并使用csv.reader()函数读取文件中的数据。
3. 接下来，我们将读取到的数据存储在一个列表中。
4. 最后，我们使用json.dumps()函数将CSV数据转换为JSON格式，并输出结果。
