## 4.2 Python的数据类型

Python中的数据类型包括整数、浮点数、字符串、列表、元组、字典等。例如，我们可以创建一个整数变量，并将其值设置为10：

## 4.3 Python的函数

Python中的函数是一段可以重复使用的代码块，可以用来完成特定的任务。例如，我们可以创建一个函数，用来计算两个数的和：

```python
def add(x, y):
    return x + y
```

我们可以调用这个函数，并将其传递给两个数：

```python
result = add(10, 20)
print(result)  # 输出: 30
```

## 4.4 Python的类和对象

Python是面向对象的编程语言，它支持类和对象。类是一种模板，用来定义对象的属性和方法，对象是类的实例。例如，我们可以创建一个类，用来表示人：

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def say_hello(self):
        print("Hello, my name is " + self.name)
```

我们可以创建一个对象，并将其传递给这个类：

```python
person = Person("John", 30)
person.say_hello()  # 输出: Hello, my name is John
```

## 4.5 Python的模块

Python中的模块是一种包含多个函数和类的文件，可以用来组织代码。例如，我们可以创建一个模块，用来包含一些数学函数：

```python
# math.py
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y
```

我们可以在其他文件中导入这个模块，并使用其中的函数：

```python
# main.py
import math

result = math.add(10, 20)
print(result)  # 输出: 30

result = math.subtract(10, 20)
print(result)  # 输出: -10
```

## 4.6 Python的异常处理

Python支持异常处理，可以用来处理程序中可能出现的错误。例如，我们可以创建一个函数，用来计算两个数的和，并使用try-except语句来处理可能出现的错误：

```python
def add(x, y):
    return x + y

try:
    result = add(10, 20)
    print(result)  # 输出: 30
except Exception as e:
    print(e)  # 输出: 错误信息
```
