# 2.核心概念与联系

在Python中，函数是一种代码块，它可以接受输入（参数），执行某些操作，并返回输出（返回值）。函数的定义和调用是编程中的基本操作，它们使得代码更加模块化和可重用。

## 2.1 函数定义

Python函数的定义使用`def`关键字，后跟函数名和括号`()`，括号中可以包含参数。函数体使用冒号`:`开头，并包含一系列的代码行。以下是一个简单的函数定义示例：

```python
def greet(name):
    print(f"Hello, {name}!")
```

在这个例子中，我们定义了一个名为`greet`的函数，它接受一个名为`name`的参数。当我们调用这个函数并传递一个名字时，它会打印出一个问候语。

## 2.2 函数调用

函数调用是通过函数名和括号`()`来实现的。当我们调用一个函数时，我们可以传递参数给该函数，这些参数将作为函数体内部的局部变量。以下是一个简单的函数调用示例：

```python
greet("John")
```

在这个例子中，我们调用了`greet`函数，并传递了一个名字"John"作为参数。这将导致问候语"Hello, John!"被打印出来。


# 6.附录常见问题与解答

## 6.1 如何定义一个函数？

要定义一个函数，你需要使用`def`关键字，后跟函数名和括号`()`，括号中可以包含参数。然后，你需要编写函数体，使用冒号`:`开头，并包含一系列的代码行。

## 6.2 如何调用一个函数？

要调用一个函数，你需要使用函数名和括号`()`，可以传递参数给函数。

## 6.3 如何返回一个函数的结果？

要返回一个函数的结果，你需要使用`return`关键字，后跟你想要返回的值。

## 6.4 如何处理函数的参数？

函数的参数可以是任何Python数据类型，包括整数、浮点数、字符串、列表、字典等。你可以在函数定义中指定参数类型，也可以使用默认值和可变参数。

## 6.5 如何处理函数的返回值？

函数的返回值可以是任何Python数据类型。你可以在函数调用时直接使用返回值，也可以将其赋给变量，或者将其存储在数据结构中。
