# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1概率分布的基本概念

### 3.1.1概率密度函数

概率密度函数（PDF）是一个随机变量的概率分布的一个连续函数表示。PDF的定义为：

$$
f(x) = \frac{d}{dx}F(x)
$$

其中，F(x)是累积分布函数（CDF），表示随机变量X在取值小于等于x的概率。

### 3.1.2累积分布函数

累积分布函数（CDF）是一个随机变量的概率分布的一个连续函数表示。CDF的定义为：

$$
F(x) = P(X \leq x)
$$

其中，X是随机变量，P(X≤x)是X在取值小于等于x的概率。

### 3.1.3期望

期望是一个随机变量的数学期望，表示随机变量的平均值。期望的定义为：

$$
E[X] = \int_{-\infty}^{\infty} x f(x) dx
$$

其中，f(x)是随机变量X的概率密度函数。

### 3.1.4方差

方差是一个随机变量的数学方差，表示随机变量的离散程度。方差的定义为：

$$
Var[X] = E[(X - E[X])^2] = E[X^2] - (E[X])^2
$$

其中，E[X]是随机变量X的期望，E[X^2]是随机变量X的二次期望。

## 3.2常见的概率分布

### 3.2.1均匀分布

均匀分布是一种连续概率分布，其概率密度函数为：

$$
f(x) = \frac{1}{b-a}，a \leq x \leq b
$$

其中，a和b是均匀分布的参数，表示区间[a,b]内的随机变量X的取值范围。

### 3.2.2指数分布

指数分布是一种连续概率分布，其概率密度函数为：

$$
f(x) = \frac{1}{\beta} e^{-\frac{x-\mu}{\beta}}，x \geq \mu
$$

其中，μ和β是指数分布的参数，表示平均值和标准差。

### 3.2.3正态分布

正态分布是一种连续概率分布，其概率密度函数为：

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}，-\infty < x < \infty
$$

其中，μ和σ是正态分布的参数，表示平均值和标准差。

### 3.2.4泊松分布

泊松分布是一种离散概率分布，其概率质量函数为：

$$
P(X=k) = \frac{e^{-\lambda}(\lambda^k)}{k!}，k=0,1,2,...
$$

其中，λ是泊松分布的参数，表示平均值。

### 4.1.1计算概率值

```python
import numpy as np

def uniform_prob(a, b, x):
    return (x - a) / (b - a)

b = 10

print(uniform_prob(a, b, x))
```

```python
def uniform_cumulative_prob(a, b, x):

```

## 4.2指数分布

### 4.2.1计算概率值

```python
import scipy.stats as stats

def exponential_prob(mu, x):
    return stats.expon.sf(x, scale=mu)

mu = 1
x = 2
print(exponential_prob(mu, x))
```

### 4.2.2计算累积概率值

```python
def exponential_cumulative_prob(mu, x):
    return 1 - stats.expon.cdf(x, scale=mu)

mu = 1
x = 2
print(exponential_cumulative_prob(mu, x))
```

## 4.3正态分布

### 4.3.1计算概率值

```python
import scipy.stats as stats

def normal_prob(mu, sigma, x):
    return stats.norm.cdf((x - mu) / sigma)

mu = 0
sigma = 1
x = 0
print(normal_prob(mu, sigma, x))
```

### 4.3.2计算累积概率值

```python
def normal_cumulative_prob(mu, sigma, x):
    return stats.norm.cdf((x - mu) / sigma)

mu = 0
sigma = 1
x = 0
print(normal_cumulative_prob(mu, sigma, x))
```

## 4.4泊松分布

### 4.4.1计算概率值

```python
def poisson_prob(lambda_, k):
    return stats.poisson.pmf(k, lambda_)

k = 2

```

### 4.4.2计算累积概率值

```python
def poisson_cumulative_prob(lambda_, k):
```
