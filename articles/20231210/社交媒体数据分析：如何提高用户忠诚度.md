数学模型公式详细讲解：

- **度 centrality**：度 centrality 是用于衡量一个节点在社交网络中的重要性的指标，可以通过计算一个节点与其他节点的连接数来得到。公式为：
$$
Degree(v) = |E_v|
$$
其中，$Degree(v)$ 表示节点 v 的度，$|E_v|$ 表示节点 v 与其他节点的连接数。

- **Betweenness centrality**：Betweenness centrality 是用于衡量一个节点在社交网络中的中介作用的指标，可以通过计算一个节点在所有节点之间的路径中的数量来得到。公式为：
$$
Betweenness(v) = \sum_{s \neq v \neq t} \frac{\sigma_{st}(v)}{\sigma_{st}}
$$
其中，$Betweenness(v)$ 表示节点 v 的 Betweenness centrality，$\sigma_{st}$ 表示节点 s 和节点 t 之间的路径数量，$\sigma_{st}(v)$ 表示在节点 s 和节点 t 之间的路径中包含节点 v 的路径数量。

# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的例子来演示如何使用 Python 的 NetworkX 库来分析社交网络数据：

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建一个有向无权图
G = nx.DiGraph()

# 添加节点
G.add_nodes_from(['A', 'B', 'C', 'D', 'E', 'F'])

# 添加边
G.add_edges_from([('A', 'B'), ('B', 'C'), ('C', 'D'), ('D', 'E'), ('E', 'F')])

# 计算节点的度
degree_centrality = nx.degree(G)

# 绘制社交网络图
nx.draw(G, with_labels=True)
plt.show()
```

在这个例子中，我们首先创建了一个有向无权的图，然后添加了节点和边。接下来，我们使用 `nx.degree()` 函数计算了节点的度，并使用 `nx.draw()` 函数绘制了社交网络图。
