### 4.2 性能优化代码实例

以下是一个性能优化代码实例：

```python
import time
import requests

# 发送请求
start_time = time.time()
response = requests.get('http://example.com')
end_time = time.time()

# 计算响应时间
response_time = end_time - start_time
print('响应时间：', response_time)

# 优化代码
if response_time > 1:
    # 使用多线程发送请求
    import threading
    threads = []
    for _ in range(5):
        t = threading.Thread(target=requests.get, args=('http://example.com',))
        t.start()
        threads.append(t)
    for t in threads:
        t.join()

    # 计算响应时间
    response_time = end_time - start_time
    print('优化后响应时间：', response_time)
```

在这个代码实例中，我们使用 Python 的 `requests` 库发送一个 GET 请求。我们记录请求发送的开始时间和结束时间，然后计算响应时间。如果响应时间超过 1 秒，我们使用多线程发送请求，以提高性能。
