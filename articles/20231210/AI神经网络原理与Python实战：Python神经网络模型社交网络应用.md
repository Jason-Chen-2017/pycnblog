## 1.2 AI与神经网络的联系

AI和神经网络是相互联系的，神经网络是AI的一个重要组成部分。AI是一种通过计算机程序模拟人类智能的技术，而神经网络则是一种通过模拟人类大脑中神经元的结构和功能来解决问题的方法。

神经网络是AI的一个重要组成部分，它可以用来解决各种问题，如图像识别、语音识别、自然语言处理等。神经网络的核心是神经元（neuron）和连接它们的权重（weight）。神经元是计算机程序中的一个简单模型，它可以接收输入、进行计算并输出结果。连接权重是神经元之间的连接，它用于调整神经元之间的信息传递。

神经网络的训练过程是通过调整连接权重来使网络输出与实际结果最接近的过程。这个过程通常是通过优化某种损失函数来实现的，损失函数是衡量网络输出与实际结果之间差异的指标。通过反向传播（backpropagation）算法，我们可以计算出连接权重的梯度，并使用梯度下降（gradient descent）算法来更新连接权重。
