# 2.核心概念与联系

决策树是一种分类和回归的模型，它可以用来解决各种问题，如医疗诊断、信用评分、股票价格预测等。决策树的核心概念包括：节点、边、叶子节点、决策规则、信息增益、熵、基尼系数等。

决策树的核心思想是：通过对数据集的分割，将数据集划分为若干个子集，每个子集包含一定的特征，然后根据这些特征来决定哪个类别或者哪个值更适合这个子集。这个过程可以递归地进行，直到所有的数据都被分配到一个类别或者一个值上。

决策树的构建过程可以分为以下几个步骤：

1. 初始化：将整个数据集作为一个节点，并将该节点标记为叶子节点。
2. 选择最佳特征：从所有可能的特征中选择一个最佳的特征，作为当前节点的分裂基准。
3. 分裂节点：根据选定的特征将数据集划分为若干个子集，并为每个子集创建一个新的节点。
4. 递归：对于每个子集，重复以上步骤，直到所有数据都被分配到一个类别或者一个值上。
