# 2.核心概念与联系

## 2.1 双指针的概念

双指针算法的核心思想是使用两个指针来遍历数组或链表，这两个指针分别表示当前的起始位置和结束位置。通过不断地移动这两个指针，我们可以实现对数组或链表的遍历、查找、删除等操作。

## 2.2 双指针与其他算法的联系

```python
def removeDuplicates(nums):
    if not nums:
        return 0
    slow, fast = 0, 1
    while fast < len(nums):
        if nums[fast] != nums[slow]:
            slow += 1
            nums[slow] = nums[fast]
        fast += 1
    return slow + 1
```

在这个代码实例中，我们使用双指针算法来删除数组中的重复元素。我们使用两个指针slow和fast来遍历数组，slow指针表示当前的起始位置，fast指针表示当前的结束位置。我们的目标是将slow指针所指的元素复制到数组的开始位置，并将fast指针所指的元素移动到slow指针所指的元素之后。这样，我们就可以实现对数组的重复元素的删除。
