在这个代码实例中，我们将使用Python的TensorFlow库来实现一个简单的深度学习算法。

```python

model = tf.keras.Sequential([
    tf.keras.layers.Dense(64, activation='relu', input_shape=(100,)),
    tf.keras.layers.Dense(64, activation='relu'),
    tf.keras.layers.Dense(10, activation='softmax')
])

# 编译模型
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# 训练模型
model.fit(x_train, y_train, epochs=10)
```

在这个代码实例中，我们首先导入了TensorFlow库，然后定义了一个简单的深度学习模型。这个模型包括三个全连接层，其中第一个层有64个神经元，使用ReLU激活函数，输入形状为100。第二个层也有64个神经元，使用ReLU激活函数。最后一个层有10个神经元，使用softmax激活函数。

接下来，我们使用Adam优化器来编译模型，并使用sparse_categorical_crossentropy作为损失函数，并使用accuracy作为评估指标。

最后，我们使用训练数据来训练模型，并设置训练轮次为10。
