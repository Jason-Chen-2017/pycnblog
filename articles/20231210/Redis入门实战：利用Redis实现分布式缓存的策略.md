                 

# 1.背景介绍

Redis（Remote Dictionary Server，远程字典服务器）是一个开源的高性能key-value存储系统，它支持数据的持久化，可基于内存（in-memory）也可基于磁盘（Persistent）。Redis 提供多种语言的 API，包括：Ruby、Python、Java、C、C++、PHP、Node.js、Go、Lua、Objective-C 和 Swift。

Redis 的核心特性有：

- Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。
- Redis 将数据分为10个不同的数据类型：string、hash、list、set、sorted set 等，同时也提供了字符串（string）、列表（list）、集合（set）和有序集合（sorted set）等四种复杂类型的操作。
- Redis 支持数据的备份，即 master-slave 模式的数据备份。
- Redis 支持 Publish/Subscribe 模式。
- Redis 支持 Lua 脚本（Redis 脚本）。
- Redis 支持键（key）的自动删除。
- Redis 支持事务（transaction）。
- Redis 支持集群（Cluster）。
- Redis 支持虚拟内存（VM）机制。
- Redis 支持数据压缩。
