# 4.具体代码实例和详细解释说明
以下是一个Go图形编程的具体代码实例，用于绘制一个简单的矩形：

```go
package main

import (
	"fmt"
	"github.com/fogleman/gg"
)

func main() {
	dc := gg.NewContext(200, 200)
	dc.SetRGB(1, 0, 0)
	dc.DrawRectangle(20, 20, 160, 160)
	dc.Stroke()
	fmt.Println("矩形绘制成功")
}
```

代码解释：

- 导入Go图形库：`github.com/fogleman/gg`
- 创建一个绘图上下文：`dc := gg.NewContext(200, 200)`，指定绘图区域的宽度和高度。
- 设置颜色：`dc.SetRGB(1, 0, 0)`，设置矩形的颜色为红色。
- 绘制矩形：`dc.DrawRectangle(20, 20, 160, 160)`，指定矩形的位置和大小。
- 绘制矩形边框：`dc.Stroke()`，完成矩形的绘制。
- 输出提示信息：`fmt.Println("矩形绘制成功")`，表示矩形绘制成功。
