# 2.核心概念与联系

半监督学习技术的核心概念包括有监督学习、无监督学习、半监督学习以及相关联的术语。

## 2.1 有监督学习

有监督学习是一种机器学习方法，其中训练数据集中的每个样本都有一个标签。标签是指已知的输出值，可以是数值、分类或者其他形式。有监督学习的目标是找到一个模型，使其在未见过的数据上的预测性能最佳。常见的有监督学习方法包括线性回归、逻辑回归、支持向量机等。

## 2.2 无监督学习

无监督学习是一种机器学习方法，其中训练数据集中的每个样本都没有标签。无监督学习的目标是找到一个模型，使其在未见过的数据上的聚类、降维或其他性质最佳。常见的无监督学习方法包括聚类算法、主成分分析、奇异值分解等。

## 2.3 半监督学习

半监督学习是一种机器学习方法，其中训练数据集中的部分样本有标签，部分样本没有标签。半监督学习的目标是找到一个模型，使其在未见过的数据上的预测性能最佳，同时利用有标签的数据来指导模型的训练，并利用无标签的数据来补充训练数据集。半监督学习的核心思想是结合了有监督学习和无监督学习的优点，从而提高模型的准确性。

1. 首先，对训练数据集中的有标签的样本进行特征提取，得到特征向量。

3. 接着，对有标签的样本进行标签传播，将标签从有标签的样本传播到相似的无标签样本。
4. 最后，对无标签的样本进行预测，得到预测结果。
