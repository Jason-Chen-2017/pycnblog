# 2.核心概念与联系

在高能物理研究中，我们主要关注的是高能粒子，如电子、电子壳子、中子、中子壳子、氢子等。这些粒子在高能加速器中被加速到非常高的速度，然后通过碰撞来研究其性质和相互作用。

高能物理的核心概念包括：

- 粒子物理学：研究粒子的性质、行为和相互作用。
- 量子力学：研究微观世界的规律，包括粒子的波动性和概率性。
- 关系论：研究宇宙的起源、演化和结构。
- 高能加速器：用于加速粒子的设备，如欧洲核研究组织（CERN）的大型碱性氢子存储隧道（LHC）。
- 碰撞实验：通过碰撞高能粒子来研究其性质和相互作用的实验。
