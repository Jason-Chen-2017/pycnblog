## 4.1 加密算法实例

以下是一个使用AES算法进行对称加密的代码实例：

```python
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad

key = b'1234567890abcdef'
message = b'This is a secret message'

cipher = AES.new(key, AES.MODE_EAX)
ciphertext, tag = cipher.encrypt_and_digest(pad(message, AES.block_size))

print(ciphertext)
print(tag)
```

在这个代码实例中，我们首先导入了AES加密算法的相关模块。然后，我们定义了一个密钥和一个明文消息。接下来，我们使用AES.new()函数创建了一个AES加密对象，并使用encrypt_and_digest()函数进行加密。最后，我们打印了加密后的密文和标签。
