# 2.核心概念与联系

## 2.1条件概率

条件概率是一种概率的概念，它表示一个事件发生的概率，但是这个事件发生的条件是另一个事件已经发生了。例如，如果我们知道一个人是男性，那么这个人的概率是否会穿衬衫，就是条件概率。

条件概率可以表示为P(A|B)，其中P(A|B)表示事件A发生的概率，条件是事件B已经发生了。

## 2.2贝叶斯定理

贝叶斯定理是一种概率推理方法，它可以帮助我们计算条件概率。贝叶斯定理的公式为：

P(A|B) = P(B|A) * P(A) / P(B)

其中，P(A|B)是我们想要计算的条件概率，P(B|A)是事件B发生的概率，条件是事件A已经发生了，P(A)是事件A发生的概率，P(B)是事件B发生的概率。
