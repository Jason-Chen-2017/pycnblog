### 4.1 知识表示
首先，我们需要对闰年的定义进行知识表示。闰年的定义是：一个年份是闰年，当且仅当它是400年份，或者是4年份，但不是100年份。我们可以使用Python的字典数据结构来表示这个知识。

```python
knowledge = {
    "is_leap_year": {
        "is_400": True,
        "is_4": True,
        "is_100": False
    }
}
```

### 4.2 推理规则
接下来，我们需要定义一个推理规则来判断一个给定的年份是否为闰年。推理规则可以表示为一个条件和一个结果的组合。条件描述了一个逻辑公式，结果描述了在满足条件时所产生的效果。

```python
def is_leap_year(year):
    if knowledge["is_leap_year"]["is_400"] and year % 400 == 0:
        return True
    elif knowledge["is_leap_year"]["is_4"] and year % 4 == 0 and not knowledge["is_leap_year"]["is_100"] and year % 100 != 0:
        return True
    else:
        return False
```

### 4.3 推理过程
最后，我们需要根据给定的年份和知识基础设施来执行推理。我们可以调用上面定义的`is_leap_year`函数来判断一个给定的年份是否为闰年。

```python

print(result)  # True
```
