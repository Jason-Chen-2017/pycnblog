### 资源分配算法

我们将使用贪婪算法来实现资源分配。贪婪算法的核心思想是在每个步骤中，选择能够提高目标函数值的最佳选择。

```python
import numpy as np

def resource_allocation(resources, demands, capacities):
    allocation = np.zeros(len(resources))
    for i in range(len(resources)):
        max_demand = np.max(demands[i])
        max_capacity = np.max(capacities[i])
        allocation[i] = min(max_demand, max_capacity)
    return allocation
```

```python
import numpy as np

def information_propagation(graph, probabilities):
    propagated = np.zeros(len(graph))
    for i in range(len(graph)):
        for j in range(len(graph[i])):
            if np.random.rand() < probabilities[i][j]:
                propagated[i] = 1
    return propagated
```
