以下是一个简单的Python代码实例，用于实现回归算法：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据预处理
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([1, 2, 3, 4])

# 特征选择
X = X[:, 0]

# 模型选择
model = LinearRegression()

# 模型训练
model.fit(X, y)

# 模型验证
X_test = np.array([[5], [6]])
y_pred = model.predict(X_test)

# 模型预测
print(y_pred)
```
