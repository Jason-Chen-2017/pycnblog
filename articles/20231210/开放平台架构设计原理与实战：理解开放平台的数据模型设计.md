### 1.4.5 数据可扩展性和可维护性

我们可以使用动态数组来实现数据结构的可扩展性，如下所示：

```python
class DynamicArray:
    def __init__(self):
        self.data = []

    def add(self, item):
        self.data.append(item)

    def remove(self, item):
        self.data.remove(item)

    def get(self, index):
        return self.data[index]

array = DynamicArray()
array.add(1)
array.add(2)
array.add(3)
print(array.get(1))  # 输出: 2
```

在这个例子中，我们定义了一个DynamicArray类，它使用动态数组来存储数据。我们可以通过add、remove、get等方法来实现数据的增删查改操作。
