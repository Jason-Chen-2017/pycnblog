在本节中，我们将通过一个简单的例子来说明禅宗思想在算法设计和优化中的应用。

例子：求两个整数的最大公约数（GCD）

问题：给定两个整数a和b，求它们的最大公约数。

解决方案：

1. 问题分析：我们需要找到两个整数的最大公约数。
2. 算法选择：我们可以使用欧几里得算法来解决这个问题。欧几里得算法是一种常用的求最大公约数的算法，其时间复杂度为O(log(min(a,b)))。
3. 算法优化：在实现欧几里得算法时，我们可以使用递归来简化代码，提高代码的可读性。
4. 实现和测试：我们可以通过实现欧几里得算法并进行测试来验证算法的正确性和效率。

以下是欧几里得算法的Python实现：

```python
def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b)
```
