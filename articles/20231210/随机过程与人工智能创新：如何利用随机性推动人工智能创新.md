1. 随机搜索的代码实例：

```python
import random

def random_search(f, x_min, x_max, iterations):
    best_x = None
    best_f = float('inf')

    for _ in range(iterations):
        x = random.uniform(x_min, x_max)
        f_value = f(x)

        if f_value < best_f:
            best_x = x
            best_f = f_value

    return best_x, best_f

# 目标函数
def f(x):
    return x**2 + 2

# 随机搜索
x_min, x_max = 0, 1
iterations = 1000
best_x, best_f = random_search(f, x_min, x_max, iterations)
print("最佳解：", best_x, "最佳值：", best_f)
```

在上述代码中，我们实现了一个简单的随机搜索算法，通过随机选择候选解 x，从而找到最佳解。我们定义了一个目标函数 f(x) = x^2 + 2，并设定了搜索范围 x ∈ [0, 1]，以及搜索次数 iterations = 1000。通过调用 random_search 函数，我们可以得到最佳解 best_x 和最佳值 best_f。

```python
import numpy as np

def random_generate(x1, x2, mu, sigma, size):

```

3. 随机优化的代码实例：

```python
import numpy as np

def objective_function(theta):
    return np.sum((np.random.rand(10) - np.dot(theta, np.random.rand(10)))**2)

def gradient_descent(theta, learning_rate, iterations):
    for _ in range(iterations):
        gradient = 2 * np.dot(np.random.rand(10), np.random.rand(10) - np.dot(theta, np.random.rand(10)))
        theta = theta - learning_rate * gradient
    return theta

# 初始参数
theta = np.random.rand(10)
learning_rate = 0.01
iterations = 1000

# 随机优化
optimal_theta = gradient_descent(theta, learning_rate, iterations)
print("最佳参数：", optimal_theta)
```

在上述代码中，我们实现了一个简单的随机优化算法，通过随机调整参数 theta，从而找到最佳参数。我们定义了一个目标函数 objective_function(theta)，并设定了学习率 learning_rate = 0.01、迭代次数 iterations = 1000。通过调用 gradient_descent 函数，我们可以得到最佳参数 optimal_theta。
