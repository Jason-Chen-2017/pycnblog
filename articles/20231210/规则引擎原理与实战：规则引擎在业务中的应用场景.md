### 4.1 规则引擎的具体代码实例

以下是一个简单的规则引擎的具体代码实例：

```python
class RuleEngine:
    def __init__(self):
        self.rules = []

    def add_rule(self, condition, action):
        self.rules.append((condition, action))

    def execute(self, data):
        for condition, action in self.rules:
            if condition(data):
                action(data)
                break

# 定义一个规则
def check_age(data):
    return data['age'] > 18

def set_age(data):
    data['age'] = 18

# 创建一个规则引擎实例
engine = RuleEngine()

# 添加一个规则
engine.add_rule(check_age, set_age)

# 执行规则
data = {'age': 20}
engine.execute(data)
print(data)  # {'age': 18}
```
