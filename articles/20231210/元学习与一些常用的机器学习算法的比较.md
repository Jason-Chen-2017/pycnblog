# 2.核心概念与联系

在讨论元学习之前，我们需要了解一些基本概念。

## 2.1 监督学习、无监督学习和半监督学习

- 监督学习：在这种学习方法中，我们有一个标签的训练数据集，每个数据点都有一个已知的输出值。监督学习的目标是根据这个训练数据集来学习一个模型，然后使用这个模型来预测新的输入数据的输出值。
- 无监督学习：在这种学习方法中，我们没有标签的训练数据集。无监督学习的目标是根据这个训练数据集来学习一个模型，然后使用这个模型来发现数据中的结构和模式。
- 半监督学习：在这种学习方法中，我们有一个部分标签的训练数据集。半监督学习的目标是根据这个训练数据集来学习一个模型，然后使用这个模型来预测新的输入数据的输出值。
