# 2.核心概念与联系
在面试中，离职与跳槽的概念是相关的。离职是指你在当前公司离开的意思，而跳槽是指你在当前公司离开并加入另一个公司的意思。这两个概念在面试中是相关的，因为面试官会关注你是否有离职或跳槽的意思。

首先，我们需要了解一些数学模型公式。这些公式可以帮助我们更好地理解离职与跳槽的概念和联系。以下是一些重要的数学模型公式：

1.离职概率公式：P(离职) = 1 - P(留职)
2.跳槽概率公式：P(跳槽) = P(离职) * P(加入另一个公司)

这些公式可以帮助我们更好地理解离职与跳槽的概念和联系。
