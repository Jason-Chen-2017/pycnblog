## 4.1 音乐创作算法实例
我们将通过一个简单的马尔可夫链算法来生成音乐序列。

```python
import numpy as np

# 初始状态
state = np.array([0, 0])

# 状态转移概率
transition_prob = np.array([
    [0.5, 0.5],
    [0.5, 0.5]
])

# 生成音乐序列
music_sequence = []
while True:
    # 根据当前状态预测下一个状态
    next_state = np.random.choice(2, p=transition_prob[state])
    
    # 更新当前状态
    state = next_state
    
    # 添加当前状态到音乐序列
    music_sequence.append(state)
    
    # 判断是否结束
    if state == np.array([1, 1]):
        break

# 打印音乐序列
print(music_sequence)
```
