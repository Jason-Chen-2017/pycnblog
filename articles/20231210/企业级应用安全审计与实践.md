```python
import requests
from bs4 import BeautifulSoup

# 设置目标URL
url = 'https://www.example.com'

# 发送请求
response = requests.get(url)

# 解析HTML内容
soup = BeautifulSoup(response.text, 'html.parser')

# 找到所有的链接
links = soup.find_all('a')

# 遍历所有的链接
for link in links:
    # 获取链接的HREF属性
    href = link.get('href')
    # 打印链接
    print(href)
```

这个代码实例是一个简单的网页链接抓取程序，它使用了Python的requests库和BeautifulSoup库来发送HTTP请求并解析HTML内容。通过这个实例，您可以更好地理解如何使用Python来实现企业级应用安全审计的具体操作。
