### 4.1 直接体验数据

我们可以使用以下代码实现直接体验数据：

```python
import numpy as np

# 生成数据
X = np.random.rand(100, 10)
Y = np.random.rand(100, 1)

# 直接体验数据
for i in range(100):
    print(X[i], Y[i])
```

这段代码首先使用 numpy 库生成了 100 个随机的样本，每个样本包含 10 个特征。然后，我们使用 for 循环直接体验每个样本。

### 4.2 放下思考

我们可以使用以下代码实现放下思考：

```python
from sklearn.linear_model import LinearRegression

# 训练模型

# 预测
preds = model.predict(X)
```

这段代码首先导入了 sklearn 库中的 LinearRegression 类。然后，我们使用 LinearRegression 类的 fit 方法训练模型，并使用 predict 方法进行预测。
