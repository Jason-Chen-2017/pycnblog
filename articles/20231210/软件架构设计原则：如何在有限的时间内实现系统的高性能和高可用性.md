# 4.具体代码实例和详细解释说明

在软件架构设计中，我们需要编写一些具体的代码实例，以便于理解和验证算法原理。例如，我们可以编写一个简单的负载均衡器，使用轮询算法来分配请求。

```python
import random

def load_balancer(requests, servers):
    # 初始化服务器列表
    server_list = list(servers)

    # 遍历请求列表
    for request in requests:
        # 随机选择一个服务器处理请求
        server = random.choice(server_list)
        # 从服务器列表中移除已处理的服务器
        server_list.remove(server)
        # 处理请求
        server.handle_request(request)
```

这个代码实例中，我们使用了`random.choice`函数来随机选择一个服务器处理请求。这样可以实现负载均衡。
