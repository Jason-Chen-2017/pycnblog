# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来解释泛型编程的实际应用。我们将编写一个简单的泛型函数，用于计算两个数的和。

```cpp
template<typename T>
T add(T a, T b) {
    return a + b;
}
```

在上述代码中，我们定义了一个泛型函数`add`，它接受两个类型参数`T`，并返回两个数的和。我们可以使用这个函数来计算不同数据类型的和。例如，我们可以使用整数类型来计算整数的和，也可以使用浮点类型来计算浮点数的和。

```cpp
int result = add<int>(1, 2);
double result = add<double>(1.1, 2.2);
```

在上述代码中，我们使用了泛型函数`add`来计算整数和浮点数的和。我们使用了类型参数`int`和`double`来表示具体的数据类型。
