### 4.1. 数据库源系统的数据读取代码实例

```python
import pymysql

def read_source_db(host, user, password, database):
    conn = pymysql.connect(host=host, user=user, password=password, database=database)
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM table_name")
    rows = cursor.fetchall()
    cursor.close()
    conn.close()
    return rows
```
