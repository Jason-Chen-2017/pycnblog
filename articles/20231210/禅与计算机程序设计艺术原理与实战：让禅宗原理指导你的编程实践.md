### 4.1.2 解决方案

我们可以使用快速排序算法来解决这个问题。快速排序算法的核心思想是选择一个基准值，将数组分为两部分，一部分小于基准值，一部分大于基准值，然后递归地对这两部分进行排序。

### 4.1.3 代码实例

以下是一个使用Python实现快速排序的代码实例：

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)

arr = [3, 5, 2, 1, 4]
print(quick_sort(arr))
```

在这个代码实例中，我们首先定义了一个快速排序函数`quick_sort`。这个函数接受一个整数数组`arr`作为参数，并返回一个排序后的数组。

然后，我们创建了一个整数数组`arr`，并调用`quick_sort`函数来对其进行排序。最后，我们打印出排序后的数组。
