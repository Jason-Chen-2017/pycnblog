# 2.核心概念与联系

在关联规则挖掘中，我们需要了解以下几个核心概念：

1. 项目：项目是数据中的基本单位，可以是数值、分类或者其他类型的数据。

3. 支持度：支持度是一个项目或者项目集合在整个数据集中出现的次数，用于衡量项目或者项目集合的重要性。
4. 信息增益：信息增益是一个项目或者项目集合在整个数据集中出现的次数与其他项目或者项目集合的关联关系的度量标准，用于衡量项目或者项目集合的有用性。
