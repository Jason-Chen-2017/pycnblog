## 4.1 代码实例1：K-均值聚类与异常值识别
```python
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import silhouette_score
import numpy as np

# 数据预处理
data = np.random.rand(100, 3)
scaler = StandardScaler()
data = scaler.fit_transform(data)

# K-均值聚类
kmeans = KMeans(n_clusters=3, random_state=42)
kmeans.fit(data)

# 异常值识别
labels = kmeans.labels_
centroids = kmeans.cluster_centers_
distances = np.sqrt(np.sum(np.power(data - centroids, 2), axis=1))
silhouette_avg = silhouette_score(data, labels)

# 识别异常值
threshold = silhouette_avg - 0.5
exception_indices = np.where(distances > threshold)[0]
exception_values = data[exception_indices]
```
在上述代码中，我们首先使用K-均值聚类方法对数据进行分类。然后，我们使用异常值识别方法（如异常值的距离）来识别异常值。最后，我们将识别出的异常值进行进一步的分析和处理。

## 4.2 代码实例2：SVM异常检测
```python
from sklearn.svm import SVC
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import accuracy_score
import numpy as np

# 数据预处理
data = np.random.rand(100, 3)
labels = np.random.randint(2, size=100)
scaler = StandardScaler()
data = scaler.fit_transform(data)

# 数据分割
X_train, X_test, y_train, y_test = train_test_split(data, labels, test_size=0.2, random_state=42)

# SVM异常检测
svm = SVC(kernel='linear', random_state=42)
svm.fit(X_train, y_train)

# 异常检测
predictions = svm.predict(X_test)
accuracy = accuracy_score(y_test, predictions)

# 识别异常值
threshold = accuracy - 0.5
exception_indices = np.where(predictions != y_test)[0]
exception_values = X_test[exception_indices]
```
在上述代码中，我们首先使用SVM方法对数据进行分类。然后，我们使用异常值识别方法（如分类准确率）来识别异常值。最后，我们将识别出的异常值进行进一步的分析和处理。
