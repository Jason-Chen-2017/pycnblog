                 

# 1.背景介绍

在Python中，元组（tuple）是一种内置的数据结构，类似于列表，但是元组的元素不能修改。元组可以用来存储不同类型的数据，例如整数、字符串、浮点数等。元组的元素可以是任何类型的数据，可以包含其他元组、列表等。元组是一种有序的数据结构，可以通过索引访问其中的元素。

元组的语法非常简单，可以使用圆括号（）来定义元组。例如，我们可以创建一个元组，包含三个整数：

```python
t = (1, 2, 3)
```

在这个例子中，我们创建了一个元组t，包含三个整数1、2和3。我们可以通过索引访问元组的元素，例如，t[0]将返回1，t[1]将返回2，t[2]将返回3。

元组的主要优点是它们的元素不能修改，这意味着一旦元组被创建，它们的元素就不能被更改。这使得元组非常适合用于存储不需要修改的数据，例如用户ID、产品ID等。

在Python中，元组和列表之间的主要区别在于元组的元素不能修改，而列表的元素可以修改。例如，我们可以通过列表的索引来修改列表的元素，而不能通过元组的索引来修改元组的元素。

```python
# 创建一个列表
l = [1, 2, 3]

# 修改列表的元素
l[0] = 4
print(l)  # 输出：[4, 2, 3]

# 尝试修改元组的元素
t = (1, 2, 3)
t[0] = 4
print(t)  # 输出：(1, 2, 3)
```

在这个例子中，我们创建了一个列表l，包含三个整数1、2和3。我们可以通过列表的索引来修改列表的元素，例如l[0] = 4将修改列表的第一个元素为4。然而，我们尝试修改元组t的元素，但是这将导致错误，因为元组的元素不能修改。
