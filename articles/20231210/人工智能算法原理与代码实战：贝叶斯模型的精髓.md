# 2.核心概念与联系

在讨论贝叶斯模型之前，我们需要了解一些基本的概念和定义。

## 2.1 概率

概率是一种数学概念，用来描述事件发生的可能性。概率通常用0到1之间的一个数来表示，表示事件发生的可能性。例如，如果我们知道一个事件发生的概率为0.5，那么这个事件发生的可能性为50%。概率可以用不同的方法来计算，包括经验法、统计法等。

## 2.2 条件概率

条件概率是一种概率概念，用来描述事件发生的可能性，给定另一个事件已经发生。例如，如果我们知道一个人是男性，那么这个人是父亲的概率是多少？这个概率就是条件概率。条件概率可以用以下公式来表示：

P(A|B) = P(A和B发生)/P(B发生)

其中，P(A|B) 表示事件A发生的概率，给定事件B已经发生；P(A和B发生) 表示事件A和事件B同时发生的概率；P(B发生) 表示事件B发生的概率。

## 2.3 贝叶斯定理

贝叶斯定理是一种概率推理方法，可以用来更新和修改我们对事件发生概率的信念。贝叶斯定理可以用以下公式来表示：

P(A|B) = P(B|A) * P(A) / P(B)

其中，P(A|B) 表示事件A发生的概率，给定事件B已经发生；P(B|A) 表示事件B发生的概率，给定事件A已经发生；P(A) 表示事件A发生的概率；P(B) 表示事件B发生的概率。
