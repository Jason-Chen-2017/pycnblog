# 2.核心概念与联系

## 2.1 线性 spaces 的概念

线性 spaces 是数学中的一个概念，它是由一组线性无关的向量组成的集合。线性 spaces 的一个重要特点是它们可以用来表示线性方程组的解。线性 spaces 的一个重要应用是线性回归，线性回归是一种常用的预测模型。

## 2.2 多项式回归的概念

多项式回归是一种预测模型，它使用多项式函数来拟合数据。多项式回归的一个重要特点是它可以用来拟合非线性数据。多项式回归的一个重要应用是预测数据的趋势。

## 2.3 线性 spaces 与多项式回归的联系

线性 spaces 和多项式回归之间的联系是，多项式回归可以看作是线性 spaces 的一种特殊情况。具体来说，多项式回归可以用来拟合线性 spaces 中的数据，但是多项式回归不能用来拟合非线性 spaces 中的数据。

    model = LinearRegression()

    return model.coef_[0]

# 定义一个多项式函数的参数
