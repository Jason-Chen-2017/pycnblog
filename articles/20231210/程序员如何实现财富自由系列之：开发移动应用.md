## 3.1 数据库操作

数据库操作是移动应用开发中不可或缺的一部分，它可以帮助我们存储和管理应用的数据。在Android平台上，我们可以使用SQLite数据库进行操作，而在iOS平台上，我们可以使用Core Data等框架进行操作。

### 3.1.1 SQLite数据库的基本操作

SQLite数据库是一个轻量级的、嵌入式的、自由的公共域的关系型数据库引擎。它可以用于嵌入式应用程序中，如移动应用程序。SQLite数据库的基本操作包括：

- 创建数据库：`db = sqlite3.connect('my_database.db')`
- 创建表：`db.execute('CREATE TABLE my_table (id INTEGER PRIMARY KEY, name TEXT)')`
- 插入数据：`db.execute('INSERT INTO my_table (name) VALUES (?)', ('John Doe'))`
- 查询数据：`db.execute('SELECT * FROM my_table')`
- 更新数据：`db.execute('UPDATE my_table SET name = ? WHERE id = ?', ('Jane Doe', 1))`
- 删除数据：`db.execute('DELETE FROM my_table WHERE id = ?', (1,))`
- 关闭数据库：`db.close()`

### 3.1.2 Core Data的基本操作

Core Data是一个用于iOS平台的数据模型框架，它可以帮助我们更方便地管理应用程序的数据。Core Data的基本操作包括：

- 创建数据模型：`NSEntityDescription *entity = [NSEntityDescription entityWithName:@"Person" inManagedObjectContext:context];`
- 创建实例：`NSManagedObject *person = [NSEntityDescription insertNewObjectForEntityForName:@"Person" inManagedObjectContext:context];`
- 设置属性值：`[person setValue:name forKey:@"name"];`
- 保存数据：`[context save:&error];`
- 查询数据：`NSFetchRequest *request = [[NSFetchRequest alloc] initWithEntityName:@"Person"];`
- 更新数据：`[person setValue:newName forKey:@"name"];`
- 删除数据：`[context deleteObject:person];`

## 4.1 案例背景

我们需要开发一个简单的天气预报应用，该应用可以通过用户的位置信息获取当前天气情况，并展示在界面上。

## 4.2 案例实现

### 4.2.1 获取用户位置信息

我们可以使用Core Location框架来获取用户的位置信息。首先，我们需要在AppDelegate中注册位置服务：

```objc
- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
    [self setupLocationManager];
}

- (void)setupLocationManager {
    self.locationManager = [[CLLocationManager alloc] init];
    self.locationManager.delegate = self;
    self.locationManager.desiredAccuracy = kCLLocationAccuracyBest;
    [self.locationManager requestWhenInUseAuthorization];
    [self.locationManager startUpdatingLocation];
}
```

然后，我们需要实现CLLocationManagerDelegate协议中的方法来处理位置更新：

```objc
- (void)locationManager:(CLLocationManager *)manager didUpdateLocations:(NSArray<CLLocation *> *)locations {
    CLLocation *location = [locations lastObject];
    NSLog(@"latitude: %f, longitude: %f", location.coordinate.latitude, location.coordinate.longitude);
}
```

### 4.2.2 获取天气预报

我们可以使用URLSession来发起网络请求获取天气预报。首先，我们需要创建一个请求对象：

```objc
NSURLSession *session = [NSURLSession sharedSession];
NSURLRequest *request = [NSURLRequest requestWithURL:[NSURL URLWithString:@"http://api.openweathermap.org/data/2.5/weather?lat=37.7749&lon=-122.4194&appid=YOUR_API_KEY"]];
```

然后，我们需要创建一个任务对象来发起请求：

```objc
NSURLSessionDataTask *task = [session dataTaskWithRequest:request completionHandler:^(NSData * _Nullable data, NSURLResponse * _Nullable response, NSError * _Nullable error) {
    if (error) {
        NSLog(@"Error: %@", error.localizedDescription);
        return;
    }

    NSDictionary *responseDictionary = [NSJSONSerialization JSONObjectWithData:data options:NSJSONReadingAllowFragments error:nil];
    NSLog(@"temperature: %f", [responseDictionary objectForKey:@"main.temp"]);
}];

[task resume];
```

### 4.2.3 展示天气预报

我们可以使用UILabel来展示天气预报：

```objc
UILabel *temperatureLabel = [[UILabel alloc] initWithFrame:CGRectMake(0, 0, 320, 40)];
[temperatureLabel setTextAlignment:NSTextAlignmentCenter];
[temperatureLabel setFont:[UIFont systemFontOfSize:20]];
[self.view addSubview:temperatureLabel];
```
