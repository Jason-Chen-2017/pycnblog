# 4.具体代码实例和详细解释说明
以下是一个简单的Python代码实例，用于对声音信号进行傅里叶变换：

```python
import numpy as np
from scipy.signal import welch

# 读取声音信号
signal = np.load('signal.npy')

# 计算傅里叶频谱
f, pxx = welch(signal, fs=16000, nperseg=1024, nfft=2048, noverlap=1024)

# 绘制傅里叶频谱
plt.plot(f, pxx)
plt.xlabel('Frequency (Hz)')
plt.ylabel('Power Spectrum')
plt.title('Fourier Transform')
plt.show()
```

在这个代码中，我们首先使用`numpy`库来读取声音信号。然后，我们使用`scipy.signal.welch`函数来计算声音信号的傅里叶频谱。最后，我们使用`matplotlib`库来绘制傅里叶频谱图。
