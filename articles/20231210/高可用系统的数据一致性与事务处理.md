# 2.核心概念与联系

在高可用系统中，数据一致性是指系统中所有节点的数据必须保持一致性，即在任何时刻，系统中的任何两个节点之间的数据都应该保持一致。事务处理是指系统中的各个节点之间的数据操作必须保持一致性，即在任何时刻，系统中的任何两个节点之间的数据操作都应该保持一致。

为了实现高可用性和数据一致性，我们需要使用一种新的事务处理方法，即分布式事务处理。分布式事务处理是指在多个节点之间进行数据操作的过程，以保证系统中的所有节点的数据一致性。
