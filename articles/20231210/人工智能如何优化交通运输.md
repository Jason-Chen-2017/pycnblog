## 4.1 机器学习代码实例

我们可以使用Python的Scikit-learn库来实现机器学习算法。以线性回归为例，我们可以使用Scikit-learn库中的LinearRegression类来实现线性回归算法。

```python

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X_train, y_train)

# 预测
y_pred = model.predict(X_test)
```

在上述代码中，`X_train`和`y_train`是训练数据集，`X_test`是测试数据集。我们可以使用`fit`方法来训练模型，并使用`predict`方法来预测测试数据集的输出。

## 4.2 深度学习代码实例

我们可以使用Python的TensorFlow库来实现深度学习算法。以卷积神经网络为例，我们可以使用TensorFlow库中的Sequential类来实现卷积神经网络。

```python
import tensorflow as tf

# 创建卷积神经网络模型
model = tf.keras.Sequential([
    tf.keras.layers.Conv2D(32, (3, 3), activation='relu', input_shape=(64, 64, 3)),
    tf.keras.layers.MaxPooling2D((2, 2)),
    tf.keras.layers.Conv2D(64, (3, 3), activation='relu'),
    tf.keras.layers.MaxPooling2D((2, 2)),
    tf.keras.layers.Conv2D(64, (3, 3), activation='relu'),
    tf.keras.layers.Flatten(),
    tf.keras.layers.Dense(64, activation='relu'),
    tf.keras.layers.Dense(10, activation='softmax')
])

# 编译模型
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# 训练模型
model.fit(train_images, train_labels, epochs=10)

# 预测
predictions = model.predict(test_images)
```

在上述代码中，`train_images`和`train_labels`是训练数据集，`test_images`是测试数据集。我们可以使用`compile`方法来编译模型，并使用`fit`方法来训练模型，并使用`predict`方法来预测测试数据集的输出。
