```python

    return a + b

    a = 1
    b = 2
    result = add(a, b)
    print(result)
```

- 调用函数：在主程序中，我们调用了 `add` 函数，并将两个数 `1` 和 `2` 作为参数传递给它。然后，我们将函数的返回值赋给一个名为 `result` 的变量。
