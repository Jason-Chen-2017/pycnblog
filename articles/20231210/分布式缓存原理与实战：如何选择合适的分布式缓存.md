### 4.1代码实例1：缓存数据的存储

```python
class Cache:
    def __init__(self):
        self.data = {}

    def set(self, key, value):
        self.data[key] = value

    def get(self, key):
        return self.data.get(key)
```

在上述代码中，我们实现了一个简单的缓存类。该类提供了两个方法：`set`和`get`。`set`方法用于存储缓存数据，`get`方法用于读取缓存数据。

### 4.2代码实例2：缓存数据的读取

```python
cache = Cache()
cache.set("key", "value")
print(cache.get("key"))  # 输出：value
```

在上述代码中，我们创建了一个缓存对象，并使用`set`方法存储了一条缓存数据。然后，我们使用`get`方法读取了缓存数据，并输出了结果。

### 4.3代码实例3：缓存数据的更新

```python
cache.set("key", "new_value")
print(cache.get("key"))  # 输出：new_value
```

在上述代码中，我们使用`set`方法更新了缓存数据。然后，我们使用`get`方法读取了缓存数据，并输出了结果。

### 4.4代码实例4：缓存数据的删除

```python
cache.delete("key")
print(cache.get("key"))  # 输出：None
```

在上述代码中，我们使用`delete`方法删除了缓存数据。然后，我们使用`get`方法读取了缓存数据，并输出了结果。
