# 2.核心概念与联系
在Python中，函数是一种可重用的代码块，可以将多次执行的任务封装起来，以便在需要时轻松地重复使用。函数定义是将一段代码封装成函数的过程，而函数调用是在程序中使用已定义的函数。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 3.1 函数定义
在Python中，定义函数的基本语法如下：

```python
def 函数名(参数列表):
    函数体
```

其中，`函数名`是函数的名称，`参数列表`是函数接收的参数，`函数体`是函数的代码块。

例如，我们可以定义一个简单的函数，用于计算两个数的和：

```python
def add(a, b):
    return a + b
```

在这个例子中，`add`是函数名，`a`和`b`是参数列表，`return a + b`是函数体。

## 3.2 函数调用
在Python中，调用函数的基本语法如下：

```python
函数名(实参列表)
```

其中，`函数名`是要调用的函数名称，`实参列表`是实际传递给函数的参数。

例如，我们可以调用前面定义的`add`函数，并传递两个数字作为实参：

```python
result = add(3, 4)
print(result)  # 输出: 7
```

在这个例子中，`add(3, 4)`是函数调用，`3`和`4`是实参列表。
