## 4.1 直观性原理实例

直观性原理实例：设计一个简单的文件管理器界面。

```python
import tkinter as tk
from tkinter import ttk

class FileManager:
    def __init__(self, master=None):
        self.master = master
        self.master.title("File Manager")

        self.treeview = ttk.Treeview(self.master, columns=("size", "type", "date"), show="headings")
        self.treeview.heading("size", text="Size")
        self.treeview.heading("type", text="Type")
        self.treeview.heading("date", text="Date")
        self.treeview.column("size", width=80, anchor="e")
        self.treeview.column("type", width=80, anchor="e")
        self.treeview.column("date", width=80, anchor="e")
        self.treeview.pack(expand=True, fill=tk.BOTH)

if __name__ == "__main__":
    root = tk.Tk()
    app = FileManager(root)
    root.mainloop()
```

解释：这个代码实例是一个简单的文件管理器界面，它使用了`tkinter`库来创建一个树形视图，用于显示文件的大小、类型和日期。树形视图的列头已经设置为直观的标签，以便用户能够快速地理解和使用系统。
