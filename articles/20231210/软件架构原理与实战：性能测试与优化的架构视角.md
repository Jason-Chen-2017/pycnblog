以下是一个简单的负载测试代码实例：

```python
import time
import threading

def worker():
    for i in range(100000):
        time.sleep(0.001)

threads = []
for i in range(100):
    t = threading.Thread(target=worker)
    threads.append(t)
    t.start()

for t in threads:
    t.join()
```

在这个代码实例中，我们创建了100个线程，每个线程都执行了一个worker函数。worker函数中，我们模拟了100000次操作，每次操作耗时0.001秒。我们启动了所有线程，并等待它们都完成。

这个代码实例可以帮助我们评估系统在高负载下的性能表现。我们可以观察系统的CPU使用率、内存使用率等指标，以评估系统性能。
