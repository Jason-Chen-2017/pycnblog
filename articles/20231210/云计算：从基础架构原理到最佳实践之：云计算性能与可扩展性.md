                 

# 1.背景介绍

云计算是一种基于互联网的计算资源共享和分配模式，它允许用户在需要时从网络上获取计算资源，而无需购买和维护自己的硬件设施。云计算的核心思想是将计算资源、存储资源和网络资源等基础设施作为服务提供给用户，从而实现资源的共享和可扩展性。

云计算的发展背景主要包括以下几个方面：

1. 计算资源的快速发展：随着计算机硬件技术的不断发展，计算资源的性价比逐渐提高，使得更多的用户和企业开始考虑如何更好地利用这些资源。

2. 互联网的普及：互联网的普及使得计算资源的分配和共享变得更加便捷，同时也为云计算提供了基础设施。

3. 数据的庞大增长：随着互联网的发展，数据的生成和存储量逐年增长，这也为云计算提供了巨大的市场机会。

4. 企业对于IT资源的需求：企业在日常运营中需要大量的IT资源，如计算资源、存储资源和网络资源等，这也为云计算提供了市场机会。

import heapq

def scheduling(tasks, resources):
    task_count = len(tasks)
    resource_count = len(resources)
    assigned_tasks = []

    for i in range(task_count):
        task = tasks[i]
        resource = resources[heapq.nlargest(1, resources)[0]]

    return assigned_tasks
