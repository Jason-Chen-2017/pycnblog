### 4.1 数据清洗

假设我们有一个包含生产数据的CSV文件，其中包含生产量、成本、生产时间等信息。我们需要对这些数据进行清洗，以消除数据中的噪声、错误和缺失值。

```python
import pandas as pd

# 读取CSV文件
data = pd.read_csv('production_data.csv')

# 删除缺失值
data = data.dropna()

# 填充缺失值
data['cost'] = data['cost'].fillna(data['cost'].mean())

# 插值填充缺失值
data.interpolate(method='linear', inplace=True)

# 滤波去噪
data['production_quantity'] = data['production_quantity'].rolling(window=3).mean()
```
