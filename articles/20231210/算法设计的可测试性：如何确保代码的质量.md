## 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释算法设计的可测试性。我们将使用一个简单的排序算法——冒泡排序（Bubble Sort）来进行说明。

### 4.1 算法实现

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 4.2 算法复杂度分析

- 时间复杂度：冒泡排序的时间复杂度为 O(n^2)，其中 n 是输入数据的大小。这是因为在最坏情况下，每个元素都需要与其他所有元素进行比较。
- 空间复杂度：冒泡排序的空间复杂度为 O(1)，因为它不需要额外的空间来存储数据。

### 4.3 算法性能优化

我们可以对冒泡排序进行优化，以提高其性能。例如，我们可以使用一个标志位来记录是否进行了任何交换。如果在一次遍历中没有进行交换，那么已经有序，可以提前结束排序。这样可以减少冒泡排序的时间复杂度为 O(n)。

```python
def bubble_sort_optimized(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr
```
