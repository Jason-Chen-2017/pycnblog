## 4.2 C++异常处理的具体代码实例
```cpp
#include <iostream>

int main() {
    try {
        int a = 1;
        int b = 0;
        int c = a / b;
    } catch (const std::exception& e) {
        std::cerr << "Exception caught: " << e.what() << std::endl;
    }
    return 0;
}
```
在这个代码实例中，我们使用try-catch语句来捕获可能发生的异常情况。在try块中，我们定义了一个可能导致异常的代码块，即a/b。当b为0时，程序会抛出一个除数为零的异常。在catch块中，我们捕获了异常并输出了异常信息。
