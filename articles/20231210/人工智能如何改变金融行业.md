# 2.核心概念与联系

人工智能（Artificial Intelligence，AI）是一种计算机科学的分支，旨在让计算机模拟人类智能的行为和决策。人工智能可以分为两个主要类别：

1.机器学习（Machine Learning，ML）：机器学习是一种计算机科学的分支，旨在让计算机自动学习和预测数据。机器学习可以进一步分为监督学习、无监督学习和强化学习。

2.深度学习（Deep Learning，DL）：深度学习是一种机器学习的分支，旨在让计算机自动学习和预测复杂的数据。深度学习可以进一步分为卷积神经网络（Convolutional Neural Networks，CNN）、循环神经网络（Recurrent Neural Networks，RNN）和变分自编码器（Variational Autoencoders，VAE）等。

model.fit(X_train, y_train)

# 模型评估
