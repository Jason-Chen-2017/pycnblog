def longest_subsequence(sequence):
    dp = [1] * len(sequence)
    for i in range(1, len(sequence)):
        for j in range(i):
            if sequence[i] > sequence[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```

def quick_sort(sequence):
    if len(sequence) <= 1:
        return sequence
    pivot = sequence[0]
    left = [x for x in sequence[1:] if x < pivot]
    right = [x for x in sequence[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)
```

在这个实例中，我们首先选择一个基准元素pivot，然后将序列分为两个子序列，一个是小于pivot的元素，一个是大于或等于pivot的元素。然后我们递归地对这两个子序列进行排序，最后将排序后的子序列与基准元素组合起来得到排序后的序列。
