## 4.1 智能门锁的实现

智能门锁是一种常见的家居设备，它可以通过智能手机应用程序实现门锁的自动识别和开锁功能。以下是智能门锁的实现代码：

```python
import time
import random

class SmartLock:
    def __init__(self):
        self.is_locked = True

    def unlock(self):
        if self.is_locked:
            self.is_locked = False
            print("门锁已开锁")
        else:
            print("门锁已开锁")

    def lock(self):
        if not self.is_locked:
            self.is_locked = True
            print("门锁已锁定")
        else:
            print("门锁已锁定")

    def check_status(self):
        if self.is_locked:
            print("门锁已锁定")
        else:
            print("门锁已开锁")

if __name__ == "__main__":
    smart_lock = SmartLock()
    smart_lock.lock()
    smart_lock.check_status()
    smart_lock.unlock()
    smart_lock.check_status()
```

上述代码实现了一个简单的智能门锁类，它可以实现门锁的自动识别和开锁功能。通过使用`lock`方法，我们可以将门锁锁定，通过使用`unlock`方法，我们可以将门锁开锁。通过使用`check_status`方法，我们可以查看门锁的状态。

## 4.2 智能温控系统的实现

智能温控系统是一种常见的家居设备，它可以通过智能手机应用程序实现室内温度的自动调节。以下是智能温控系统的实现代码：

```python
import time
import random

class SmartTemperatureControl:
    def __init__(self, temperature_set_point):
        self.temperature_set_point = temperature_set_point
        self.current_temperature = 0

    def set_temperature(self, new_temperature):
        if new_temperature >= self.temperature_set_point:
            self.current_temperature = self.temperature_set_point
            print("温度已设置为：", self.current_temperature)
        else:
            self.current_temperature = new_temperature
            print("温度已设置为：", self.current_temperature)

    def get_temperature(self):
        print("当前温度：", self.current_temperature)

if __name__ == "__main__":
    smart_temperature_control = SmartTemperatureControl(25)
    smart_temperature_control.set_temperature(23)
    smart_temperature_control.get_temperature()
```

上述代码实现了一个简单的智能温控系统类，它可以实现室内温度的自动调节。通过使用`set_temperature`方法，我们可以将室内温度设置为指定的值。通过使用`get_temperature`方法，我们可以查看当前室内温度。
