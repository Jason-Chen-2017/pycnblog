以下是一个简单的代码实例：

```python
# 定义一个函数
def add(x, y):
    return x + y

# 调用函数
result = add(1, 2)
print(result)
```

在这个代码实例中，我们定义了一个名为`add`的函数，它接受两个参数`x`和`y`，并返回它们的和。我们然后调用这个函数，并将结果打印出来。
