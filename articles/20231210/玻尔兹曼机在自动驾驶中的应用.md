其中，$\mathbf{s}$ 是随机变量的状态向量，$\mathbf{W}$ 是权重矩阵，$\mathbf{b}$ 是偏置向量。

### 1.3.2 温度下降算法

温度下降算法是玻尔兹曼机的核心算法，它的目标是找到能量最低的状态。在每一次迭代中，玻尔兹曼机会随机选择一个变量并对其进行翻转，从而使能量发生变化。翻转的概率可以通过以下公式计算：

$$
p_{ij} = \frac{1}{1 + \exp(\beta(E_i - E_j))}
$$

其中，$p_{ij}$ 是变量 i 到变量 j 的翻转概率，$\beta$ 是温度参数，$E_i$ 和 $E_j$ 是变量 i 和变量 j 的能量。

```python
import numpy as np

class BoltzmannMachine:
    def __init__(self, n_visible, n_hidden):
        self.n_visible = n_visible
        self.n_hidden = n_hidden
        self.W = np.random.randn(n_visible, n_hidden)
        self.b = np.random.randn(n_visible)

    def energy(self, visible, hidden):
        return -0.5 * np.dot(visible, np.dot(self.W, hidden) + self.b)

    def sample_visible(self, hidden):
        z = self.energy(visible, hidden)
        p_visible = np.zeros((self.n_visible, 2))
        p_visible[..., 1] = 1 / (1 + np.exp(z))
        return np.random.choice(2, size=self.n_visible, p=p_visible)

    def sample_hidden(self, visible):
        z = self.energy(visible, hidden)
        p_hidden = np.zeros((self.n_hidden, 2))
        p_hidden[..., 1] = 1 / (1 + np.exp(-z))
        return np.random.choice(2, size=self.n_hidden, p=p_hidden)

    def gibbs_sampling(self, n_iter):
        visible = np.random.randint(2, size=self.n_visible)
        hidden = np.random.randint(2, size=self.n_hidden)

        for _ in range(n_iter):
            visible = self.sample_visible(hidden)
            hidden = self.sample_hidden(visible)

        return visible, hidden

# 使用示例
bm = BoltzmannMachine(10, 10)
visible, hidden = bm.gibbs_sampling(1000)
```
