### 2.1 知识图谱

知识图谱是一种用于表示实体、关系和事件的数据结构。它可以帮助计算机理解人类语言中的含义。知识图谱的主要组成部分包括实体、关系和事件。实体是知识图谱中的基本单位，它可以表示一个具体的实体，如人、地点、组织等。关系是实体之间的联系，它可以表示实体之间的属性、属性值、类别等。事件是一种特殊的关系，它可以表示一个具体的事件，如出生、死亡等。

知识图谱的构建主要包括以下几个步骤：

1. 实体识别：从文本中提取实体信息，并将其映射到知识图谱中。
2. 关系识别：从文本中提取关系信息，并将其映射到知识图谱中。
3. 事件识别：从文本中提取事件信息，并将其映射到知识图谱中。
4. 实体链接：将不同来源的实体信息链接到同一个实体中，以便于查询和分析。

### 4.1 知识图谱构建

我们可以使用以下代码来构建一个简单的知识图谱：

```python
from rdflib import Graph, Namespace, Literal

# 定义命名空间
ns = Namespace("http://example.org/")

# 创建图
g = Graph()

# 添加实体
g.add((ns.person, ns.name, "Alice"))
g.add((ns.person, ns.age, 30))

# 添加关系
g.add((ns.person, ns.knows, ns.person))

# 添加事件
g.add((ns.event, ns.name, "Birthday Party"))
g.add((ns.event, ns.date, "2022-01-01"))

# 保存图
g.serialize(format="turtle", destination="knowledge_graph.ttl")
```

在这个代码中，我们首先导入了 `rdflib` 库，它是一个用于创建和操作 RDF 图的库。然后，我们定义了一个命名空间 `ns`，它用于唯一标识实体、关系和事件。接着，我们创建了一个图 `g`，并添加了一些实体、关系和事件。最后，我们将图保存到一个 Turtle 文件中。

### 4.2 语义分析

我们可以使用以下代码来进行语义分析：

```python

from rdflib.query import Query

# 加载图
g = Graph().load("knowledge_graph.ttl", format="turtle")

# 查询实体
query = Query("""
    SELECT ?person WHERE {
        ?person ns:name "Alice" .
    }
""", initNdfs=True)
results = query.query(g)
print(results)

# 查询关系
query = Query("""
    SELECT ?person ?knows WHERE {
        ?person ns:name "Alice" .
        ?person ns:knows ?knows .
    }
""", initNdfs=True)
results = query.query(g)
print(results)

# 查询事件
query = Query("""
    SELECT ?event ?date WHERE {
        ?event ns:name "Birthday Party" .
        ?event ns:date ?date .
    }
""", initNdfs=True)
results = query.query(g)
print(results)
```

在这个代码中，我们首先导入了 `rdflib` 库，并加载了一个已经保存的图。然后，我们使用 SPARQL 查询语言来查询实体、关系和事件。最后，我们将查询结果打印出来。
