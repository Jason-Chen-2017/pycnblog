```python
import torch
import torch.nn as nn
import torch.optim as optim

# 首先，我们需要选择一个已经训练好的模型
model = nn.Sequential(
    nn.Linear(10, 20),
    nn.ReLU(),
    nn.Linear(20, 1)
)

# 然后，我们需要为新的任务创建一个新的数据集
X = torch.randn(100, 10)
y = torch.randn(100, 1)

# 接下来，我们需要对现有的模型进行一些调整
model[0].weight.data.normal_(mean=0.0, std=0.1)
model[1].weight.data.normal_(mean=0.0, std=0.1)

# 最后，我们需要对新的数据集进行训练
optimizer = optim.SGD(model.parameters(), lr=0.01)
for epoch in range(100):
    optimizer.zero_grad()
    output = model(X)
    loss = (output - y)**2
    loss.backward()
    optimizer.step()
```

在这个代码实例中，我们首先选择了一个已经训练好的模型，然后创建了一个新的数据集。接下来，我们对现有的模型进行了一些调整，并使用随机梯度下降算法对新的数据集进行训练。
