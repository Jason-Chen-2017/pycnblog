## 4.1 智能合约示例

以下是一个简单的智能合约示例，它用于实现一种基本的资金转移功能：

```solidity
pragma solidity ^0.5.16;

contract FundTransfer {
    address public owner;
    uint public balance;

    constructor() public {
        owner = msg.sender;
        balance = 0;
    }

    function deposit() public payable {
        balance += msg.value;
    }

    function withdraw() public {
        require(msg.sender == owner);
        payable(owner).transfer(address(this).balance);
        balance = 0;
    }
}
```

在这个合约中，我们有一个`owner`变量，用于存储合约的所有者地址。我们还有一个`balance`变量，用于存储合约的资金余额。

合约中有两个函数：`deposit`和`withdraw`。`deposit`函数用于接收资金，并将其添加到合约的资金余额中。`withdraw`函数用于提取资金，但是只有合约的所有者才能执行这个操作。
