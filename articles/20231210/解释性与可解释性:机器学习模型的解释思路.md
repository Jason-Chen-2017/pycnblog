```python
# 训练模型
model = MLPClassifier(hidden_layer_sizes=(10, 10), max_iter=1000, random_state=42)
model.fit(X, y)

# 模型简化
pca = PCA(n_components=2)
X_reduced = pca.fit_transform(X)
```

在这个代码实例中，我们首先加载了数据，然后使用多层感知器（MLP）模型进行训练。接着，我们使用主成分分析（PCA）将训练好的模型简化为2维。
