## 4.具体代码实例和详细解释说明

在MySQL项目管理与团队协作中，可以使用SQL语句来操作数据库。具体代码实例如下：

### 4.1创建项目表

```sql
CREATE TABLE projects (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    start_date DATE NOT NULL,
    end_date DATE NOT NULL,
    status ENUM('Not Started', 'In Progress', 'Completed') NOT NULL
);
```

### 4.2插入项目信息

```sql
INSERT INTO projects (name, start_date, end_date, status)
VALUES ('Project A', '2022-01-01', '2022-12-31', 'Not Started');
```

### 4.3查询项目信息

```sql
SELECT * FROM projects WHERE status = 'Not Started';
```

### 4.4更新项目信息

```sql
UPDATE projects SET status = 'In Progress' WHERE id = 1;
```

### 4.5删除项目信息

```sql
DELETE FROM projects WHERE id = 1;
```

### 4.6创建任务表

```sql
CREATE TABLE tasks (
    id INT AUTO_INCREMENT PRIMARY KEY,
    project_id INT NOT NULL,
    name VARCHAR(255) NOT NULL,
    description TEXT,
    start_date DATE NOT NULL,
    end_date DATE NOT NULL,
    status ENUM('Not Started', 'In Progress', 'Completed') NOT NULL,
    FOREIGN KEY (project_id) REFERENCES projects(id)
);
```

### 4.7插入任务信息

```sql
INSERT INTO tasks (project_id, name, start_date, end_date, status)
VALUES (1, 'Task A', '2022-01-01', '2022-01-10', 'Not Started');
```

### 4.8查询任务信息

```sql
SELECT * FROM tasks WHERE project_id = 1 AND status = 'Not Started';
```

### 4.9更新任务信息

```sql
UPDATE tasks SET status = 'In Progress' WHERE id = 1;
```

### 4.10删除任务信息

```sql
DELETE FROM tasks WHERE id = 1;
```

### 4.11创建团队成员表

```sql
CREATE TABLE team_members (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    role ENUM('Manager', 'Developer', 'Designer', 'Tester') NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL
);
```

### 4.12插入团队成员信息

```sql

       ('Bob', 'Developer', 'bob@example.com'),
```
