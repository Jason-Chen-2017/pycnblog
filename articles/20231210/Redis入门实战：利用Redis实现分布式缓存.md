Redis（Remote Dictionary Server）是一个开源的key-value存储系统，它支持数据的持久化，可以作为数据库、缓存和消息队列的替代方案。Redis 是一个开源的使用 ANSI C 语言编写、遵循 BSD 协议的高性能的 key-value 存储系统。Redis 可以用作数据库、缓存和消息队列的替代方案。

Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。不同于其他数据库，Redis 的数据是以键值对（Key-Value）的形式存储的。Redis 的数据结构包括字符串(String)、哈希(Hash)、列表(List)、集合(Sets)和有序集合(Sorted Sets)等。

Redis 是一个使用 ANSI C 语言编写的开源（BSD 许可）、跨平台（Windows，Linux，Mac OS X，UNIX 等）的后端数据存储服务。Redis 提供了多种语言的 API，包括：Ruby、Python、Java、PHP、Node.js、Perl、Go、C#、C++、R、Lua、Objective-C 和 Swift。

Redis 的核心特性有：数据持久化、备份、集群、数据分片、高可用性、定期保存、Lua 脚本、Pub/Sub、通知、事务、监视器、虚拟内存、BITFIELD 命令、SPARSE 命令、Geo 空间查询、有限压缩内存、模式匹配等。
