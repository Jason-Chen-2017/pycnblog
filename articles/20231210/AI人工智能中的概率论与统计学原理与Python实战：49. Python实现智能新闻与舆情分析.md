## 3.1 文本预处理
文本预处理是对文本数据进行清洗和处理的过程，主要包括以下几个步骤：

1. 去除标点符号：通过使用正则表达式，我们可以将文本中的标点符号去除。
2. 转换大小写：将文本中的所有字符转换为小写或大写，以便于后续的分析。
3. 分词：将文本中的单词进行分割，以便于后续的分析。
4. 去除停用词：停用词是那些在文本中出现频率较高，但对分析结果没有太大影响的词语。我们可以将这些停用词去除，以减少噪音。
