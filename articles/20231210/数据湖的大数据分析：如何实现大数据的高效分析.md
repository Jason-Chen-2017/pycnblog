## 4.1 Hadoop HDFS的代码实例

以下是一个简单的Hadoop HDFS的代码实例，它可以用来创建和上传文件：

```java
import org.apache.hadoop.fs.FileSystem;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IOUtils;
import org.apache.hadoop.conf.Configuration;

public class HDFSExample {
    public static void main(String[] args) throws Exception {
        // 创建一个Hadoop配置对象
        Configuration conf = new Configuration();

        // 创建一个FileSystem对象，用于访问HDFS
        FileSystem fs = FileSystem.get(conf);

        // 创建一个Path对象，用于表示文件的路径
        Path src = new Path("/user/hadoop/example.txt");
        Path dst = new Path("/user/hadoop/example.txt");

        // 使用IOUtils类的copyToLocalFile方法上传文件
        IOUtils.copyToLocalFile(fs.open(src), dst);

        // 关闭FileSystem对象
        fs.close();
    }
}
```
