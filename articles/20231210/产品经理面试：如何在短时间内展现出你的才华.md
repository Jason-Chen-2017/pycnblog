### 4.1 递归

递归是一种用于解决问题的方法，它通过对问题的子集进行递归调用，以便更好地解决问题。在这里，我们将提供一个递归的代码实例，并详细解释其中的每个步骤。

```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)
```

### 4.2 分治

分治是一种用于解决问题的方法，它通过将问题分解为多个子问题，然后对每个子问题进行解决，以便更好地解决问题。在这里，我们将提供一个分治的代码实例，并详细解释其中的每个步骤。

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[0]
        less = [x for x in arr[1:] if x <= pivot]
        greater = [x for x in arr[1:] if x > pivot]
        return quick_sort(less) + [pivot] + quick_sort(greater)
```

### 4.3 动态规划

动态规划是一种用于解决问题的方法，它通过将问题分解为多个子问题，然后对每个子问题进行解决，以便更好地解决问题。在这里，我们将提供一个动态规划的代码实例，并详细解释其中的每个步骤。

```python
def fibonacci(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)
```
