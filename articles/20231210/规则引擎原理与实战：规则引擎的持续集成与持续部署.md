# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来解释规则引擎的工作原理。

假设我们有一个简单的规则引擎，用于判断用户是否满足购买条件。规则如下：

1. 如果用户年龄大于20岁，并且购买金额大于1000元，则满足购买条件。
2. 如果用户年龄小于等于20岁，或者购买金额小于或等于1000元，则不满足购买条件。

我们可以使用以下代码来实现这个规则引擎：

```python
class RuleEngine:
    def __init__(self):
        self.rules = []

    def add_rule(self, condition, action):
        self.rules.append((condition, action))

    def execute(self, facts):
        for condition, action in self.rules:
            if condition(facts):
                action(facts)

# 示例用法
engine = RuleEngine()
engine.add_rule(lambda facts: facts['age'] > 20 and facts['amount'] > 1000, lambda facts: print('满足购买条件'))
engine.add_rule(lambda facts: facts['age'] <= 20 or facts['amount'] <= 1000, lambda facts: print('不满足购买条件'))

facts = {'age': 25, 'amount': 1200}
engine.execute(facts)
```

在这个代码中，我们首先定义了一个 `RuleEngine` 类，用于存储规则和执行规则。然后，我们使用 `add_rule` 方法来添加规则，每个规则包括一个条件函数和一个动作函数。最后，我们使用 `execute` 方法来执行规则，并传入一个事实字典。

在示例用法中，我们创建了一个 `RuleEngine` 实例，并添加了两个规则。然后，我们创建了一个事实字典，并使用 `execute` 方法来执行规则。最后，我们可以看到输出结果，表示用户是否满足购买条件。
