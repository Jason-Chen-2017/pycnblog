# 2.核心概念与联系

## 2.1 条件语句

条件语句是一种用于根据某个条件执行或跳过某个代码块的语句。在Python中，条件语句使用`if`关键字来表示。条件语句的基本格式如下：

```python
if 条件:
    执行的代码块
```

条件语句的执行流程如下：

1. 首先，程序会检查条件是否为`True`。如果条件为`True`，则执行`执行的代码块`。
2. 如果条件为`False`，则跳过`执行的代码块`，并继续执行下一个语句。

## 2.2 循环语句

循环语句是一种用于重复执行某个代码块的语句。在Python中，循环语句使用`for`和`while`关键字来表示。循环语句的基本格式如下：

### 2.2.1 for循环

```python
for 变量 in 序列:
    执行的代码块
```

### 2.2.2 while循环

```python
while 条件:
    执行的代码块
```

循环语句的执行流程如下：

1. 首先，程序会检查条件是否为`True`。如果条件为`True`，则执行`执行的代码块`。
2. 如果条件为`False`，则跳出循环，并继续执行下一个语句。

# 4.具体代码实例和详细解释说明

## 4.1 条件语句的实例

### 4.1.1 简单的条件语句

```python
age = 18
if age >= 18:
    print("您已经成年了！")
else:
    print("您还没有成年！")
```

在这个例子中，我们首先定义了一个变量`age`，然后使用条件语句来判断`age`是否大于或等于18。如果条件为`True`，则执行`print("您已经成年了！")`，否则执行`print("您还没有成年！")`。

### 4.1.2 嵌套的条件语句

```python
age = 18
if age >= 18:
    print("您已经成年了！")
    if age >= 20:
        print("您已经成年了！")
    else:
        print("您还没有成年！")
else:
    print("您还没有成年！")
```

在这个例子中，我们使用了嵌套的条件语句。首先，我们判断`age`是否大于或等于18。如果条件为`True`，则执行`print("您已经成年了！")`。然后，我们再判断`age`是否大于或等于20。如果条件为`True`，则执行`print("您已经成年了！")`，否则执行`print("您还没有成年！")`。如果`age`小于18，则直接执行`print("您还没有成年！")`。

## 4.2 循环语句的实例

### 4.2.1 for循环

```python
for i in range(1, 6):
    print(i)
```

在这个例子中，我们使用了for循环来遍历`range(1, 6)`。`range(1, 6)`生成了一个序列，包含1、2、3、4、5。然后，我们使用`for`循环来遍历这个序列，并在每次迭代时执行`print(i)`。

### 4.2.2 while循环

```python
i = 1
while i < 6:
    print(i)
    i += 1
```

在这个例子中，我们使用了while循环来遍历1到5之间的整数。首先，我们定义了一个变量`i`，并设置它的初始值为1。然后，我们使用`while`循环来判断`i`是否小于6。如果条件为`True`，则执行`print(i)`，并将`i`增加1。这个过程会重复执行，直到`i`大于或等于6，条件为`False`。

1. Q: 如何使用条件语句来判断一个数是否为偶数？
   A: 我们可以使用以下代码来判断一个数是否为偶数：

   ```python
   num = 5
   if num % 2 == 0:
       print("数是偶数")
   else:
       print("数是奇数")
   ```

2. Q: 如何使用循环语句来打印1到100之间的所有偶数？
   A: 我们可以使用以下代码来打印1到100之间的所有偶数：

   ```python
   for i in range(1, 101):
       if i % 2 == 0:
           print(i)
   ```

3. Q: 如何使用循环语句来求和一个序列中的所有元素？
   A: 我们可以使用以下代码来求和一个序列中的所有元素：

   ```python
   numbers = [1, 2, 3, 4, 5]
   total = 0
   for num in numbers:
       total += num
   print(total)
   ```

4. Q: 如何使用循环语句来找出一个序列中最大的元素？
   A: 我们可以使用以下代码来找出一个序列中最大的元素：

   ```python
   numbers = [1, 2, 3, 4, 5]
   max_num = numbers[0]
   for num in numbers:
       if num > max_num:
           max_num = num
   print(max_num)
   ```
