                 

# 1.背景介绍

食品科学与营养学是一门研究食品加工与成分调整的学科。食品科学与营养学研究的主要内容包括食品加工技术、食品成分的调整、食品营养价值的评价以及食品安全等方面。食品科学与营养学的研究对于提高食品的品质、安全性、营养价值以及节约食品生产成本具有重要意义。

食品加工是指将食品原材料经过一系列的加工处理后得到的食品。食品加工的目的是提高食品的品质、增加食品的营养价值、节约食品生产成本、提高食品的安全性等。食品加工的主要方法包括：烹饪、蒸馏、烘烤、蒸馏烘烤、冷搽、干燥、腌制、腌制加工、制作加工、加工加工等。

食品成分调整是指在食品加工过程中，根据食品的用途和消费者的需求，调整食品的成分比例，以实现食品的品质、安全性和营养价值的提高。食品成分调整的主要方法包括：加盐、加糖、加油、加调味料、加食用添加剂等。

食品营养价值的评价是指根据食品的成分比例、营养物质的含量和营养物质的质量，评价食品的营养价值的方法。食品营养价值的评价方法包括：营养价值评价标准、营养价值评价指标、营养价值评价方法等。

食品安全是指食品不会对人体造成危害的安全性。食品安全的主要要素包括：食品的生产、加工、储存、运输、销售和消费等。食品安全的主要方法包括：生产安全、加工安全、储存安全、运输安全、销售安全和消费安全等。

食品科学与营养学的研究对于提高食品的品质、安全性、营养价值以及节约食品生产成本具有重要意义。食品科学与营养学的研究方向包括：食品加工技术、食品成分调整、食品营养价值评价以及食品安全等方面。
