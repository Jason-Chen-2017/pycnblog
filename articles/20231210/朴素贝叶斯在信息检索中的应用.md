# 2.核心概念与联系

## 2.1 朴素贝叶斯

朴素贝叶斯是一种基于贝叶斯定理的概率模型，它假设各个特征之间相互独立。这种假设使得朴素贝叶斯算法在实际应用中具有较高的效率和准确率。朴素贝叶斯算法主要应用于文本分类、垃圾邮件过滤等领域。

## 2.2 贝叶斯定理

贝叶斯定理是概率论中的一个重要定理，它给出了如何计算条件概率。贝叶斯定理可以用以下公式表示：

$$
P(A|B) = \frac{P(B|A) \times P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示条件概率，即给定事件B发生的情况下，事件A的概率；$P(B|A)$ 表示概率条件，即事件A发生的情况下事件B的概率；$P(A)$ 表示事件A的概率；$P(B)$ 表示事件B的概率。
