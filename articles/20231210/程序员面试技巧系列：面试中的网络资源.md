## 3.1 TCP/IP协议
TCP/IP是一种网络通信协议，它定义了数据包如何在网络中传输。TCP/IP协议包括四层：应用层、传输层、网络层和数据链路层。每一层都有自己的功能和协议。

### 3.1.1 应用层
应用层是TCP/IP协议的最高层，它负责与用户进行交互。应用层协议包括HTTP、FTP、SMTP等。

### 3.1.2 传输层
传输层负责在网络中传输数据包。传输层协议包括TCP和UDP。TCP是一种可靠的传输协议，它保证数据包的顺序和完整性。UDP是一种不可靠的传输协议，它速度快但可能丢失数据包。

### 3.1.3 网络层
网络层负责将数据包从源地址转发到目的地址。网络层协议包括IP。IP是一种无连接的协议，它将数据包分割成小包并在网络中传输。

### 3.1.4 数据链路层
数据链路层负责在物理层上传输数据包。数据链路层协议包括以太网、PPP等。

## 3.2 HTTP请求和响应
HTTP是一种用于在网络上传输数据的协议。HTTP请求是从客户端发送到服务器的请求，HTTP响应是服务器发送回客户端的响应。HTTP请求和响应包括请求行、请求头、请求体和响应头、响应体等部分。

### 3.2.1 请求行
请求行包括请求方法、请求URL和HTTP版本。请求方法可以是GET、POST、PUT、DELETE等。请求URL是请求的资源地址。HTTP版本是请求所使用的HTTP版本。

### 3.2.2 请求头
请求头包括一些额外的信息，如请求的编码、请求的来源、请求的Cookie等。

### 3.2.3 请求体
请求体是请求中携带的数据。例如，在POST请求中，请求体可能包含请求的参数。

### 3.2.4 响应头
响应头包括一些额外的信息，如响应的编码、响应的来源、响应的Cookie等。

### 3.2.5 响应体
响应体是服务器发送回客户端的数据。例如，响应体可能包含一个HTML页面。

## 3.3 DNS解析
DNS是一种域名解析协议，它将域名转换为IP地址。DNS解析包括递归查询、迭代查询和缓存等。

### 3.3.1 递归查询
递归查询是客户端向DNS服务器发送请求的过程。客户端向DNS服务器请求域名的IP地址，DNS服务器会尝试解析域名并返回IP地址。

### 3.3.2 迭代查询
迭代查询是DNS服务器向根服务器、顶级域名服务器和授权域名服务器发送请求的过程。DNS服务器会逐级向上查询域名的IP地址，直到找到或者无法找到。

### 3.3.3 缓存
DNS服务器会将查询结果缓存，以便在未来查询相同的域名时可以快速返回IP地址。缓存可以提高查询速度，但也可能导致缓存失效。

## 4.1 使用Python的requests库发送HTTP请求
```python
import requests

url = 'http://www.example.com'
headers = {'User-Agent': 'Mozilla/5.0'}

response = requests.get(url, headers=headers)

if response.status_code == 200:
    print(response.text)
else:
    print('请求失败')
```
在这个例子中，我们使用Python的requests库发送一个HTTP GET请求。我们设置了一个User-Agent头，以模拟浏览器的请求。如果请求成功，我们将打印出响应体的内容。

## 4.2 使用Python的socket库发送TCP请求
```python
import socket

client_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
client_socket.connect(('www.example.com', 80))

request = 'GET / HTTP/1.1\r\nHost: www.example.com\r\n\r\n'
client_socket.send(request.encode())

response = client_socket.recv(1024).decode()

client_socket.close()

if response.startswith('HTTP'):
    print(response)
else:
    print('请求失败')
```
在这个例子中，我们使用Python的socket库发送一个TCP请求。我们创建了一个TCP套接字，并连接到服务器。然后我们发送一个HTTP GET请求，并接收服务器的响应。如果响应开始以'HTTP'为前缀，我们将打印出响应体的内容。

## 4.3 使用Python的dnspython库进行DNS查询
```python
import dns.resolver

query = dns.resolver.Question('www.example.com', dns.resolver.IN, 'A')

answer = dns.resolver.resolve(query)

for rdata in answer:
    print(rdata)
```
在这个例子中，我们使用Python的dnspython库进行DNS查询。我们创建了一个Question对象，指定了查询的域名、查询类型和查询类。然后我们使用resolve方法进行查询，并打印出查询结果。
