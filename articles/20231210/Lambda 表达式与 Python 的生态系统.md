## 4.具体代码实例和详细解释说明

在本节中，我们将通过具体代码实例来详细解释 Lambda 表达式的用法。

### 4.1 定义 Lambda 表达式

我们可以使用以下语法来定义一个 Lambda 表达式：

```python
lambda 参数1, 参数2, ..., 参数n: 表达式
```

例如，我们可以定义一个 Lambda 表达式来计算两个数的和：

```python
add = lambda x, y: x + y
```

### 4.2 调用 Lambda 表达式

我们可以在代码中调用 Lambda 表达式，并传入一个或多个参数。例如，我们可以调用上面定义的 add 函数：

```python
result = add(1, 2)
print(result)  # 输出: 3
```

### 4.3 使用 Lambda 表达式作为参数

我们可以使用 Lambda 表达式作为其他函数的参数。例如，我们可以使用 Lambda 表达式来定义一个排序函数：

```python
sorted_list = sorted([1, 3, 5, 2, 4], key=lambda x: x * 2)
print(sorted_list)  # 输出: [2, 1, 4, 3, 5]
```

在上面的例子中，我们使用 Lambda 表达式来定义一个排序函数，该函数将每个元素乘以 2，然后将结果作为排序的关键字。

### 4.4 使用 Lambda 表达式作为返回值

我们可以使用 Lambda 表达式作为其他函数的返回值。例如，我们可以定义一个函数来返回一个 Lambda 表达式：

```python
def create_square_function():
    return lambda x: x * x

square = create_square_function()
print(square(2))  # 输出: 4
```

在上面的例子中，我们定义了一个函数 create_square_function，该函数返回一个 Lambda 表达式，用于计算一个数的平方。
