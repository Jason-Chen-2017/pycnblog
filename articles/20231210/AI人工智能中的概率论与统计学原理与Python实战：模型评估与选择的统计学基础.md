# 2.核心概念与联系

在进行模型评估和选择之前，我们需要了解一些概率论和统计学的基本概念。这些概念包括随机变量、概率分布、期望、方差、协方差等。这些概念是模型评估和选择的基础，我们需要熟悉它们的定义和计算方法。

## 2.1 随机变量

随机变量是一个数值，其取值是随机的。随机变量可以分为离散型和连续型两种。离散型随机变量的取值是离散的，而连续型随机变量的取值是连续的。

## 2.2 概率分布

概率分布是一个随机变量的所有可能取值及其对应的概率的函数。概率分布可以用来描述一个随机变量的分布情况，可以帮助我们更好地理解随机变量的特点。

## 2.3 期望

期望是一个随机变量的数学期望，是随机变量的所有可能取值的平均值。期望可以用来描述随机变量的中心趋势，可以帮助我们更好地理解随机变量的分布情况。

## 2.4 方差

方差是一个随机变量的数学方差，是随机变量的所有可能取值相对于期望的平均差值的平方。方差可以用来描述随机变量的离散程度，可以帮助我们更好地理解随机变量的分布情况。

## 2.5 协方差

协方差是两个随机变量的数学协方差，是两个随机变量的所有可能取值相对于它们的期望的平均差值的平方。协方差可以用来描述两个随机变量之间的相关性，可以帮助我们更好地理解两个随机变量的关系。
