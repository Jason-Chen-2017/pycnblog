$$
T(n) = O(n^2)
$$

$$
S(n) = O(n)
$$

其中，$$ T(n) $$ 表示时间复杂度，$$ S(n) $$ 表示空间复杂度，$$ n $$ 表示问题规模。

## 1.4 具体代码实例和详细解释说明

### 1.4.1 代码实例

以下是一个简单的排序算法的代码实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 1.4.2 代码解释

上述代码实现了一个简单的冒泡排序算法。冒泡排序是一种比较简单的排序算法，时间复杂度为 $$ O(n^2) $$ ，空间复杂度为 $$ O(1) $$ 。

代码的主要步骤如下：

1. 遍历数组中的每个元素，从左到右。
2. 与当前元素相邻的元素进行比较。
3. 如果当前元素大于相邻元素，则交换它们的位置。
4. 重复步骤2-3，直到整个数组有序。
