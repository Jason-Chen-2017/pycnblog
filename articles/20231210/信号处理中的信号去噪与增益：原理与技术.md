# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明信号去噪与增益的核心算法原理和具体操作步骤。

```python
import numpy as np
import matplotlib.pyplot as plt

# 信号去噪
def low_pass_filter(x, h, N):
    y = np.zeros(len(x))
    for n in range(len(x)):
        for k in range(N):
            y[n] += h[k] * x[n-k]
    return y

# 信号增益
def amplify(x, a):
    y = np.zeros(len(x))
    for n in range(len(x)):
        y[n] = a * x[n]
    return y

# 信号生成
x = np.sin(2 * np.pi * 10 * np.arange(100))

# 信号去噪
h = np.hanning(101)
y = low_pass_filter(x, h, 51)

# 信号增益
a = 2
y = amplify(y, a)

# 信号绘图
plt.plot(x, label='原信号')
plt.plot(y, label='处理后信号')
plt.legend()
plt.show()
```

在上述代码中，我们首先生成了一个正弦信号，然后对其进行低通滤波处理，以消除低频噪声。接着，我们对处理后的信号进行放大处理，以提高信号的幅值范围和信息量。最后，我们绘制了原信号和处理后的信号，以观察信号的变化。
