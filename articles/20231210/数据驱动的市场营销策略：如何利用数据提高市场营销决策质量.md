# 3.3 数学模型公式详细讲解
数学模型公式详细讲解包括以下几个方面：
1. 线性回归：线性回归是一种用于预测因变量的统计方法，其公式为：$$ y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n $$
2. 逻辑回归：逻辑回归是一种用于分类的统计方法，其公式为：$$ P(y=1|x) = \frac{1}{1 + e^{-\beta_0 - \beta_1x_1 - \beta_2x_2 - ... - \beta_nx_n}} $$
3. 支持向量机：支持向量机是一种用于分类和回归的机器学习方法，其公式为：$$ f(x) = \text{sgn}(\beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n) $$
4. 决策树：决策树是一种用于分类和回归的机器学习方法，其公式为：$$ \text{if } x_1 \leq t_1 \text{ then } \text{if } x_2 \leq t_2 \text{ then } ... \text{ then } y = c_1 \text{ else } ... \text{ else } y = c_k $$
5. 随机森林：随机森林是一种用于分类和回归的机器学习方法，其公式为：$$ y = \frac{1}{K} \sum_{k=1}^K f_k(x) $$
