## 1.3 Python实战

Python是一种易于学习的编程语言，它的简洁性和强大的库支持使其成为人工智能领域的首选编程语言。在本文中，我们将使用Python来实现概率论与统计学在信号处理中的应用。

### 1.3.1 滤波

滤波是信号处理中的一个重要操作，它用于去除信号中的噪声。Python中的numpy库提供了许多用于滤波的函数，如medfilt2d、convolve等。

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成信号
t = np.linspace(0, 1, 100)
s = np.sin(2 * np.pi * 10 * t) + np.random.normal(0, 0.1, 100)

# 进行滤波
filtered_s = np.medfilt2d(s, (3, 3))

# 绘制信号和滤波后的信号
plt.figure()
plt.plot(t, s, label='Original Signal')
plt.plot(t, filtered_s, label='Filtered Signal')
plt.legend()
plt.show()
```

### 1.3.2 特征提取

特征提取是信号处理中的另一个重要操作，它用于提取信号中的有用信息。Python中的scipy库提供了许多用于特征提取的函数，如fft、fftpack等。

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.fftpack import fft, fftfreq

# 生成信号
t = np.linspace(0, 1, 100)
s = np.sin(2 * np.pi * 10 * t) + np.random.normal(0, 0.1, 100)

# 进行傅里叶变换
S = fft(s)
f = fftfreq(len(s), 1 / len(t))

# 绘制信号和傅里叶变换结果
plt.figure()
plt.plot(t, s, label='Original Signal')
plt.plot(f, 2. / len(s) * np.abs(S), label='Fourier Transform')
plt.legend()
plt.show()
```
