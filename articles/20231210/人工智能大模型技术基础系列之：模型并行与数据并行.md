## 4.1 模型并行的具体代码实例

```python
# 定义子模型
class SubModel:
    def __init__(self, weight):
        self.weight = weight

    def predict(self, x):
        return self.weight * x

# 创建子模型
sub_model_1 = SubModel(1.0)
sub_model_2 = SubModel(2.0)

# 计算子模型预测结果
pred_1 = sub_model_1.predict(x)
pred_2 = sub_model_2.predict(x)

# 汇总子模型预测结果
pred = pred_1 + pred_2
```
