# 2.核心概念与联系

在MySQL中，查询操作是通过SQL（结构化查询语言）来实现的。SQL是一种用于管理关系型数据库的语言，它提供了一种简洁、高效的方式来查询、插入、更新和删除数据。MySQL的查询操作主要包括SELECT、INSERT、UPDATE和DELETE等命令。

在MySQL查询操作中，核心概念包括：

- 表：MySQL中的表是数据的组织和存储的基本单位。表由一组列组成，列是表中的数据项。
- 列：列是表中的数据项，用于存储特定类型的数据。
- 行：行是表中的数据记录，每行对应一个完整的数据记录。
- 索引：索引是用于加速查询操作的数据结构，它可以帮助MySQL快速定位到特定的数据记录。
- 条件：条件是用于限制查询结果的一种约束，例如通过WHERE子句来指定查询结果的条件。

# 4.具体代码实例和详细解释说明

在本节中，我们将通过具体的代码实例来详细解释MySQL查询操作的核心概念和操作步骤。

例1：查询所有员工的姓名和薪资

```sql
SELECT name, salary
FROM employees;
```

解释：

- SELECT子句用于指定查询结果中需要包含的列，这里我们需要包含员工姓名和薪资。
- FROM子句用于指定查询数据的来源，这里我们从员工表中查询数据。

例2：查询年收入超过100000的员工的姓名和薪资

```sql
SELECT name, salary
FROM employees
WHERE salary > 100000;
```

解释：

- WHERE子句用于指定查询结果的条件，这里我们指定查询结果中的薪资必须大于100000。

例3：按照薪资从高到低排序员工姓名和薪资

```sql
SELECT name, salary
FROM employees
ORDER BY salary DESC;
```

解释：

- ORDER BY子句用于指定查询结果的排序方式，这里我们按照薪资从高到低进行排序。
