# 4.具体代码实例和详细解释说明

以下是一个具体的Python文件操作代码实例：

```python
import os

# 创建一个文件
file = open("example.txt", "w")
file.write("Hello, World!")
file.close()

# 打开一个文件
file = open("example.txt", "r")
content = file.read()
print(content)
file.close()

# 遍历一个文件
file = open("example.txt", "r")
for line in file:
    words = line.split()
    print(words)
file.close()

# 删除一个文件
os.remove("example.txt")

# 重命名一个文件
os.rename("example.txt", "example_new.txt")
```
