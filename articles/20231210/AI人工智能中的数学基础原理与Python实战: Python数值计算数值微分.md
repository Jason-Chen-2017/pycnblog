在Python数值计算数值微分中，我们主要使用的数值方法是Euler方法、Runge-Kutta方法等。这些方法的数学模型公式如下：

### 3.3.1Euler方法

Euler方法是一种简单的数值积分方法，它通过将微分方程转换为一个差分方程来解决问题。Euler方法的数学模型公式如下：

$$
y_{n+1} = y_n + hf(t_n, y_n)
$$

其中，$y_{n+1}$ 是下一步的解，$y_n$ 是当前步的解，$h$ 是步长，$f(t_n, y_n)$ 是微分方程的函数值。

### 3.3.2Runge-Kutta方法

Runge-Kutta方法是一种高效的数值积分方法，它通过将微分方程转换为多个差分方程来解决问题。Runge-Kutta方法的数学模型公式如下：

$$
\begin{aligned}
k_1 &= hf(t_n, y_n) \\
k_2 &= hf(t_n + \frac{h}{2}, y_n + \frac{k_1}{2}) \\
k_3 &= hf(t_n + \frac{h}{2}, y_n + \frac{k_2}{2}) \\
k_4 &= hf(t_n + h, y_n + k_3) \\
y_{n+1} &= y_n + \frac{1}{6}(k_1 + 2k_2 + 2k_3 + k_4)
\end{aligned}
$$

其中，$k_1, k_2, k_3, k_4$ 是中间变量，$h$ 是步长，$f(t_n, y_n)$ 是微分方程的函数值。

## 4.1代码实例

$$
\frac{dy}{dt} = y - t
$$

初始条件为：$y(0) = 1$，步长为：$h = 0.1$。

```python
import numpy as np

def dydt(t, y):
    return y - t

t0, tf, h = 0, 1, 0.1
t = np.arange(t0, tf, h)
y = np.zeros(len(t))
y[0] = 1

for i in range(len(t) - 1):
    y[i + 1] = y[i] + h * dydt(t[i], y[i])

print(y)
```
