1.运输路线规划：

我们可以使用Dijkstra算法来解决运输路线规划问题。Dijkstra算法是一种最短路径算法，它可以找到从起点到终点的最短路径。Dijkstra算法的具体操作步骤如下：

1.1. 初始化：将起点设为当前节点，将其距离设为0，其他所有节点的距离设为无穷大。

1.2. 选择：从当前节点选择距离最短的节点，将其距离设为当前节点的距离加上该节点与当前节点之间的距离。

1.3. 更新：将当前节点的距离更新为选择的节点的距离。

1.4. 终止：当所有节点的距离都被更新过后，算法终止。

Dijkstra算法的数学模型公式为：

$$
d(u,v) = d(u) + d(u,v)
$$

其中，$d(u,v)$ 表示从起点 $u$ 到终点 $v$ 的最短路径长度，$d(u)$ 表示起点 $u$ 到当前节点的最短路径长度。

我们可以使用贪心算法来解决运输资源分配问题。贪心算法的核心思想是在每个决策中选择能够提高目标函数值的最佳选择。贪心算法的具体操作步骤如下：

2.1. 初始化：将所有运输资源的状态设为可用。

2.2. 选择：从所有可用的运输资源中选择能够满足运输需求的最佳资源。

2.3. 更新：将选择的运输资源的状态设为已分配。

2.4. 终止：当所有运输需求都被满足后，算法终止。

贪心算法的数学模型公式为：

$$
\max_{x \in X} f(x)
$$

其中，$f(x)$ 表示目标函数的值，$X$ 表示可用的运输资源状态。

    distances[start] = 0

        current_distance, current_node = heapq.heappop(queue)

        if current_distance > distances[current_node]:
            continue

            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(queue, (distance, neighbor))
