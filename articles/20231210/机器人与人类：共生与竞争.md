2. 1960年代：机器人的应用开始扩散。在这个时期，美国的西蒙·艾伦·费曼（Simon A. Fahnestock）开发了第一个能够自主运动的机器人。

5. 1990年代：机器人的技术得到了更大的突破。在这个时期，美国的斯坦福大学开发了第一个能够学习和适应环境的机器人。

### 4.1 机器人的简单例子

我们可以通过一个简单的机器人例子来详细解释代码实例和解释说明。这个例子是一个简单的机器人，可以根据人类的指令来移动。

```python
import time

class SimpleRobot:
    def __init__(self):
        self.x = 0
        self.y = 0

    def move(self, direction, distance):
        if direction == 'left':
            self.x -= distance
        elif direction == 'right':
            self.x += distance
        elif direction == 'up':
            self.y += distance
        elif direction == 'down':
            self.y -= distance

    def listen(self):
        while True:
            command = input('请输入指令：')
            if command == 'stop':
                break
            elif command == 'left':
                self.move('left', 1)
            elif command == 'right':
                self.move('right', 1)
            elif command == 'up':
                self.move('up', 1)
            elif command == 'down':
                self.move('down', 1)
            time.sleep(0.1)

if __name__ == '__main__':
    robot = SimpleRobot()
    robot.listen()
```

### 4.2 代码实例的详细解释

在这个例子中，我们创建了一个简单的机器人类，可以根据人类的指令来移动。代码实例的详细解释如下：

1. 我们首先导入了 `time` 模块，用于控制机器人的移动速度。
2. 我们创建了一个 `SimpleRobot` 类，用于表示一个简单的机器人。
3. 在 `SimpleRobot` 类的 `__init__` 方法中，我们初始化了机器人的位置为 (0, 0)。
4. 在 `SimpleRobot` 类的 `move` 方法中，我们实现了机器人的移动功能。根据不同的指令，机器人会移动不同的方向和距离。
5. 在 `SimpleRobot` 类的 `listen` 方法中，我们实现了机器人的感知功能。机器人会不断地从人类接收指令，并根据指令来移动。
6. 在主程序中，我们创建了一个 `SimpleRobot` 对象，并调用其 `listen` 方法来启动机器人的感知功能。
