## 4.1 法律文书自动化

### 4.1.1 代码实例

```python
import jieba
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

def generate_legal_document(text):
    words = jieba.cut(text)
    vectorizer = TfidfVectorizer()
    vectorizer.fit_transform(words)
    similarity = cosine_similarity(vectorizer.transform([text]), vectorizer.transform([text]))
    return similarity
```

### 4.1.2 详细解释说明

1. 首先，我们使用 jieba 库对文本进行分词。
2. 然后，我们使用 sklearn 库的 TfidfVectorizer 对分词后的文本进行 tf-idf 转换。
3. 接着，我们使用 sklearn 库的 cosine_similarity 计算 tf-idf 转换后的文本之间的余弦相似度。
4. 最后，我们返回余弦相似度。
