# 2.核心概念与联系

在了解因果推断的核心概念之前，我们需要了解一些基本的概念：

- 因果关系：因果关系是指一个变量对另一个变量的影响。例如，饮食对健康的影响。
- 因变量：因变量是受影响的变量，也就是因果关系的结果。例如，健康。
- 原因变量：原因变量是对因变量产生影响的变量。例如，饮食。
- 弱因果关系：弱因果关系是指一个变量对另一个变量的影响，但不能确定是因果关系。例如，一个人的年龄与他的薪水之间的关系。

因果推断的核心概念包括：

- 因果模型：因果模型是用于描述因果关系的数学模型。它通过将因变量和原因变量之间的关系表示为一个函数来描述因果关系。
- 因果图：因果图是用于表示因果关系的图形表示。它通过节点和边来表示因变量和原因变量之间的关系。
- 干扰变量：干扰变量是因果关系中可能影响因变量的其他变量。它们可能会影响因变量的结果，因此需要考虑在进行因果推断时。
- 选择偏差：选择偏差是因果推断中可能导致结果不准确的因素之一。它发生在因果关系中选择了不符合要求的观察数据。
