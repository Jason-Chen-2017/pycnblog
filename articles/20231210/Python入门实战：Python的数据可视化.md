## 4.具体代码实例和详细解释说明

以下是一个具体的Python数据可视化代码实例：

```python
import matplotlib.pyplot as plt
import numpy as np

# 创建数据
x = np.linspace(0, 10, 100)
y = np.sin(x)

# 创建图表
plt.figure()
plt.plot(x, y)

# 添加数据
plt.xlabel('x')
plt.ylabel('sin(x)')

# 添加标签
plt.title('Sin(x)')

# 显示图表
plt.show()
```

在这个代码实例中，我们首先导入了matplotlib.pyplot和numpy库。然后，我们创建了x和y数据。接下来，我们使用plt.figure()函数创建了一个图表，并使用plt.plot()函数绘制了一个折线图。然后，我们使用plt.xlabel()、plt.ylabel()和plt.title()函数添加了数据标签和图表标题。最后，我们使用plt.show()函数显示了图表。
