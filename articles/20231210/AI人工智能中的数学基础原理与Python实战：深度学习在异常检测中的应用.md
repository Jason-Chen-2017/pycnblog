# 2.核心概念与联系
# 2.1 神经网络
神经网络是深度学习的基本结构，由多个节点组成，每个节点都有一个权重和偏置。节点之间通过连接层相互连接，形成一个复杂的网络结构。神经网络的核心思想是通过多层次的非线性映射来学习复杂的模式和关系。

# 2.2 损失函数
损失函数是用于衡量模型预测值与实际值之间差异的函数。在训练神经网络时，我们需要最小化损失函数，以便使模型的预测更加准确。常见的损失函数有均方误差（MSE）、交叉熵损失等。

# 2.3 梯度下降
梯度下降是一种优化算法，用于最小化损失函数。通过计算损失函数的梯度，我们可以找到最佳的权重和偏置，从而使模型的预测更加准确。
