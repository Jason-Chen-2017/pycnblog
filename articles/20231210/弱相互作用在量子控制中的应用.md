## 4.1 量子控制算法的Python代码实例
以下是一个简单的量子控制算法的Python代码实例：

```python
from qiskit import QuantumCircuit, Aer, transpile, assemble

# 初始化量子系统
qc = QuantumCircuit(2)

# 设定量子门操作
qc.h(0)
qc.cx(0, 1)

# 生成量子控制信号
pulse = assemble(qc)

# 执行量子门操作
simulator = Aer.get_backend('statevector_simulator')
job = simulator.run(pulse)
result = job.result()

# 检测量子状态转移
statevector = result.get_statevector(0)
print(statevector)

# 评估量子控制效果
fidelity = np.abs(np.inner(statevector, np.array([1, 0]))) ** 2
print(fidelity)
```
