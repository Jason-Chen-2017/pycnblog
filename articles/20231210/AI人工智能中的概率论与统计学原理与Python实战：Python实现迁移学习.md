# 4.具体代码实例和详细解释说明
在这里，我们将通过一个具体的Python代码实例来说明迁移学习的工作原理。我们将使用Python的scikit-learn库来实现迁移学习。

首先，我们需要导入所需的库：

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score
```

接下来，我们需要加载数据集：

```python
iris = load_iris()
X = iris.data
y = iris.target
```

然后，我们需要将数据集划分为训练集和测试集：

```python
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)
```

接下来，我们需要创建源任务的模型：

```python
knn_model = KNeighborsClassifier(n_neighbors=3)
knn_model.fit(X_train, y_train)
```

然后，我们需要创建目标任务的模型：

```python
knn_model_target = KNeighborsClassifier(n_neighbors=3)
```

接下来，我们需要将源任务的模型应用于目标任务上：

```python
knn_model_target.fit(X_train, y_train)
```

最后，我们需要评估模型在目标任务上的性能：

```python
y_pred = knn_model_target.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)
print("Accuracy:", accuracy)
```

通过这个代码实例，我们可以看到迁移学习的工作原理。我们首先创建了一个源任务的模型，然后将其应用于目标任务上，并根据目标任务的数据调整模型。最后，我们评估模型在目标任务上的性能。
