# 4.具体代码实例和详细解释说明

以下是一个具体的因果关系图可视化代码实例：

```python
import networkx as nx
import matplotlib.pyplot as plt

# 创建因果关系图
G = nx.DiGraph()

# 添加节点
G.add_node('雨')
G.add_node('天气')

# 添加边
G.add_edge('雨', '天气')

# 绘制因果关系图
nx.draw(G, with_labels=True)
plt.show()
```

在这个代码实例中，我们使用了`networkx`库来创建因果关系图。我们首先创建了一个有向图（`nx.DiGraph()`），然后添加了节点（`G.add_node()`）和边（`G.add_edge()`）。最后，我们使用`nx.draw()`函数绘制因果关系图，并使用`plt.show()`函数显示图像。
