## 4.1 代码实例
```python
import random

def generate_random_sample(n):
    samples = []
    for _ in range(n):
        sample = random.random()
        samples.append(sample)
    return samples

def calculate_mean(samples):
    mean = 0
    for sample in samples:
        mean += sample
    mean /= len(samples)
    return mean

samples = generate_random_sample(10000)
mean = calculate_mean(samples)
print(mean)
```
在这个代码实例中，我们首先定义了一个`generate_random_sample`函数，这个函数用于生成大量的随机样本。然后，我们定义了一个`calculate_mean`函数，这个函数用于计算这些随机样本的平均值。最后，我们调用了`generate_random_sample`函数来生成10000个随机样本，并调用`calculate_mean`函数来计算这些随机样本的平均值。
