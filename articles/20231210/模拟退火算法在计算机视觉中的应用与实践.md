## 4. 具体代码实例和详细解释说明

在本节中，我们将通过一个简单的图像分割问题来演示模拟退火算法的具体实现。

```python
import numpy as np
import random

# 目标函数
def f(x):
    return np.sum(x**2)

# 初始化
T = 1000
x = np.random.rand(2)
best_x = x
best_f = f(x)

# 温度下降策略
alpha = 0.99

# 迭代次数
iterations = 1000

# 迭代
for i in range(iterations):
    # 邻域搜索
    x_new = x + random.uniform(-1, 1, size=x.shape)
    # 选择
    if random.random() < np.exp(-(f(x_new) - f(x)) / T):
        x = x_new
    # 更新最佳解
    if f(x) < best_f:
        best_x = x
        best_f = f(x)
    # 温度下降
    T *= alpha

# 输出最佳解
print(best_x)
```

在上述代码中，我们首先定义了一个目标函数f(x)，其中x是一个2维向量。然后我们初始化温度T、初始解x、最佳解best_x和最佳目标函数值best_f。接着我们设定温度下降策略、迭代次数等参数。在迭代过程中，我们对每个迭代次数进行以下操作：

1. 邻域搜索：生成一个随机向量x_new，其中每个元素在[-1, 1]范围内。
2. 选择：生成一个随机数r，如果r < np.exp(-(f(x_new) - f(x)) / T)，则更新x为x_new。
3. 更新最佳解：如果f(x) < best_f，则更新最佳解为x，更新最佳目标函数值为f(x)。
4. 温度下降：将温度T乘以alpha。

最后，我们输出最佳解best_x。
