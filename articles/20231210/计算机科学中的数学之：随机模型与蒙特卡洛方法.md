## 4.1 代码实例：估计π的值

我们可以使用蒙特卡洛方法来估计一个圆的面积，从而得到π的估计值。具体步骤如下：

1. 生成大量的随机点，这些点将在一个正方形区域内随机分布。
2. 计算这些随机点中落在圆内的个数。
3. 根据随机点的数量，估计圆的面积，从而得到π的估计值。

以下是Python代码实例：

```python
import random

# 设置正方形区域的边长
edge_length = 1

# 设置随机点数量
n = 1000000

# 生成大量的随机点
random_points = []
for i in range(n):
    x = random.uniform(-edge_length/2, edge_length/2)
    y = random.uniform(-edge_length/2, edge_length/2)
    random_points.append((x, y))

# 计算随机点中落在圆内的个数
circle_points = 0
for point in random_points:
    if point[0]**2 + point[1]**2 <= edge_length**2:
        circle_points += 1

# 估计圆的面积，从而得到π的估计值
area = edge_length**2 * circle_points / n
pi_estimate = area / (edge_length**2 / 4)

print("π的估计值为：", pi_estimate)
```

在这个代码实例中，我们首先设置了正方形区域的边长和随机点数量。然后，我们生成了大量的随机点，这些点将在正方形区域内随机分布。接着，我们计算了这些随机点中落在圆内的个数。最后，我们根据随机点的数量，估计圆的面积，从而得到π的估计值。
