## 3.1 概率论基础
### 3.1.1 概率的基本定义
概率是用来描述事件发生的可能性的一个量。在概率论中，我们通常使用以下几种方法来计算概率：
1. 直接计算概率：通过直接计算事件发生的可能性来得到概率。
2. 概率空间：通过构建一个概率空间，我们可以计算事件发生的概率。
3. 条件概率：通过计算条件概率，我们可以得到事件发生的概率。

### 3.1.2 概率的基本定理
概率的基本定理是概率论中非常重要的一个定理，它可以帮助我们计算多个事件发生的概率。概率的基本定理可以表示为：

$$
P(A \cup B) = P(A) + P(B) - P(A \cap B)
$$

### 3.1.3 独立事件
独立事件是指两个事件发生的概率不受另一个事件发生的影响。在概率论中，我们可以使用以下公式来计算独立事件的概率：

$$
P(A \cap B) = P(A) \times P(B)
$$

## 3.2 统计学基础
### 3.2.1 参数估计
参数估计是统计学中非常重要的一个概念，它可以帮助我们根据数据来估计模型的参数。在参数估计中，我们通常使用以下几种方法：

2. 最小二乘估计：通过最小化残差平方和来估计参数。

### 3.2.2 假设检验
假设检验是统计学中非常重要的一个概念，它可以帮助我们验证一个假设是否成立。在假设检验中，我们通常使用以下几种方法：
1. 单样本t检验：用于比较一个样本的均值与一个已知的参数值。
2. 两样本t检验：用于比较两个样本的均值。

## 4.1 使用numpy计算置信区间
numpy是Python中非常重要的一个库，它可以帮助我们进行数值计算。在本节中，我们将通过一个简单的例子来演示如何使用numpy计算置信区间。

```python
import numpy as np

# 创建一个随机数组
data = np.random.rand(1000)

# 计算置信区间
confidence_interval = np.percentile(data, [2.5, 97.5])

print(confidence_interval)
```

在上面的代码中，我们首先导入了numpy库，然后创建了一个随机数组。接着，我们使用numpy的percentile函数来计算置信区间。最后，我们打印出置信区间的结果。

## 4.2 使用scipy计算置信区间
scipy是Python中非常重要的一个库，它可以帮助我们进行各种科学计算。在本节中，我们将通过一个简单的例子来演示如何使用scipy计算置信区间。

```python
from scipy import stats

# 创建一个随机数组
data = np.random.rand(1000)

# 计算置信区间
confidence_interval = stats.t.interval(0.95, df=len(data) - 1, loc=np.mean(data), scale=stats.t.scale(loc=np.mean(data), n=len(data)))

print(confidence_interval)
```

在上面的代码中，我们首先导入了scipy库，然后创建了一个随机数组。接着，我们使用scipy的t.interval函数来计算置信区间。最后，我们打印出置信区间的结果。
