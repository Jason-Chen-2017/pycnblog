## 递归算法

递归算法是一种解决问题的方法，它通过对问题的子问题进行递归调用来解决问题。例如，你可以使用递归算法来解决斐波那契数列问题。

```python
def fibonacci(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```

## 分治算法

分治算法是一种解决问题的方法，它通过将问题分解为子问题，然后递归地解决子问题来解决问题。例如，你可以使用分治算法来解决求最大子序列和问题。

```python
def max_subarray_sum(arr):
    if len(arr) == 0:
        return 0
    elif len(arr) == 1:
        return arr[0]
    else:
        return max(arr[0], max_subarray_sum(arr[1:]) + arr[0])
```

## 动态规划算法

动态规划算法是一种解决问题的方法，它通过将问题分解为子问题，然后递归地解决子问题来解决问题。例如，你可以使用动态规划算法来解决最长公共子序列问题。

```python
def longest_common_subsequence(arr1, arr2):
    dp = [[0] * (len(arr2) + 1) for _ in range(len(arr1) + 1)]
    for i in range(len(arr1)):
        for j in range(len(arr2)):
            if arr1[i] == arr2[j]:
                dp[i+1][j+1] = dp[i][j] + 1
            else:
                dp[i+1][j+1] = max(dp[i+1][j], dp[i][j+1])
    return dp[-1][-1]
```
