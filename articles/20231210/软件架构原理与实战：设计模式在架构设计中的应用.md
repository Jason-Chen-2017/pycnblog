假设我们要设计一个简单的文件系统，文件系统包括文件和目录两种类型的组件。我们可以使用组合模式来实现这个文件系统的设计。

```python
class FileSystem:
    def __init__(self):
        self.components = []

    def add_component(self, component):
        self.components.append(component)

    def remove_component(self, component):
        self.components.remove(component)

class Component:
    def __init__(self, name):
        self.name = name

class File(Component):
    def __init__(self, name, content):
        super().__init__(name)
        self.content = content

class Directory(Component):
    def __init__(self, name):
        super().__init__(name)
        self.components = []

    def add_component(self, component):
        self.components.append(component)

    def remove_component(self, component):
        self.components.remove(component)
```

在这个例子中，我们定义了一个`FileSystem`类，它包含一个`components`列表来存储文件系统的组件。我们还定义了`File`和`Directory`类，它们都继承自`Component`类。`File`类表示一个文件，`Directory`类表示一个目录。我们可以使用`add_component`和`remove_component`方法来添加和删除文件系统的组件。
