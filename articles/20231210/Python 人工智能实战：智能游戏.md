## 3.1 搜索算法

搜索算法是人工智能游戏中的一个重要组成部分，它用于搜索游戏状态空间，以找到最佳的行动。搜索算法可以是深度优先搜索（Depth-First Search，DFS）、广度优先搜索（Breadth-First Search，BFS）或其他更高级的算法。

### 3.1.1 深度优先搜索（Depth-First Search，DFS）

深度优先搜索（DFS）是一种搜索算法，它首先深入一个节点，然后再深入其他节点。DFS 可以用来解决一些具有有限状态的问题，如棋盘上的棋子。

DFS 的搜索过程如下：

1. 从起始状态开始，将当前状态放入搜索栈中。
2. 从搜索栈中取出当前状态。
3. 对当前状态进行评估，如果当前状态是目标状态，则返回当前状态。
4. 如果当前状态不是目标状态，则对当前状态的所有可能行动进行评估。
5. 对每个可能行动，将其对应的状态放入搜索栈中。
6. 重复步骤2-5，直到找到目标状态或搜索栈为空。

### 3.1.2 广度优先搜索（Breadth-First Search，BFS）

广度优先搜索（BFS）是一种搜索算法，它首先广度遍历所有可能的节点，然后再深入其他节点。BFS 可以用来解决一些具有有限状态的问题，如棋盘上的棋子。

BFS 的搜索过程如下：

1. 从起始状态开始，将当前状态放入搜索队列中。
2. 从搜索队列中取出当前状态。
3. 对当前状态进行评估，如果当前状态是目标状态，则返回当前状态。
4. 如果当前状态不是目标状态，则对当前状态的所有可能行动进行评估。
5. 对每个可能行动，将其对应的状态放入搜索队列中。
6. 重复步骤2-5，直到找到目标状态或搜索队列为空。
