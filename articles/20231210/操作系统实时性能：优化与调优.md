# 4.具体代码实例和详细解释说明

在这里，我们将提供一个具体的代码实例，以帮助您更好地理解操作系统实时性能优化和调优的具体操作步骤。

```python
import time
import threading

def task(name):
    print(f'任务{name}开始执行')
    start_time = time.time()
    # 模拟任务执行时间
    time.sleep(1)
    end_time = time.time()
    print(f'任务{name}执行完成，耗时{end_time - start_time}秒')

def main():
    # 创建两个任务线程
    task1 = threading.Thread(target=task, args=(1,))
    task2 = threading.Thread(target=task, args=(2,))

    # 设置任务优先级
    task1.set_priority(10)
    task2.set_priority(5)

    # 启动任务线程
    task1.start()
    task2.start()

    # 等待任务完成
    task1.join()
    task2.join()

    print('所有任务完成')

if __name__ == '__main__':
    main()
```

在这个代码实例中，我们创建了两个任务线程，并设置了它们的优先级。然后，我们启动这两个任务线程，并等待它们完成。通过这个例子，我们可以看到如何根据任务的优先级来调整任务的执行顺序，从而影响操作系统的实时性能。
