# 2.核心概念与联系

## 2.1 云计算与容器

云计算是一种基于互联网的计算资源共享和分布式计算模型，它可以实现资源的高效利用和灵活扩展。云计算主要包括三个基本服务：计算服务、存储服务和网络服务。

容器是一种轻量级的应用程序部署和运行方式，它可以将应用程序和其依赖的运行时环境打包在一个单一的文件中，从而实现应用程序的迁移和扩展。容器技术可以让开发人员更加轻松地部署和管理应用程序，同时也可以提高应用程序的性能和稳定性。

## 2.2 云原生技术

云原生技术是云计算和容器技术的结合，它将容器技术应用于云计算环境中，实现了应用程序的自动扩展和缩容。云原生技术可以根据实际的负载情况动态地调整应用程序的资源分配，从而实现更高的性能和可用性。

## 2.3 核心概念

1. **容器化**：容器化是云原生技术的基础，它可以将应用程序和其依赖的运行时环境打包在一个单一的文件中，从而实现应用程序的迁移和扩展。

2. **微服务**：微服务是一种应用程序架构风格，它将应用程序拆分成多个小的服务，每个服务都可以独立部署和扩展。微服务可以让开发人员更加轻松地管理和扩展应用程序，同时也可以提高应用程序的性能和稳定性。

3. **自动扩展**：自动扩展是云原生技术的核心功能，它可以根据实际的负载情况动态地调整应用程序的资源分配，从而实现更高的性能和可用性。

4. **自动缩容**：自动缩容是云原生技术的另一个核心功能，它可以根据实际的负载情况动态地调整应用程序的资源分配，从而实现更高的性能和可用性。
