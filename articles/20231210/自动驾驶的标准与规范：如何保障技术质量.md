# 2 核心概念与联系

自动驾驶技术的核心概念包括：

- 感知技术：自动驾驶技术需要对周围环境进行感知，这包括对车辆、人、道路、交通信号等的感知，这些感知需要通过摄像头、雷达、激光等设备来实现。
- 定位技术：自动驾驶技术需要对车辆的位置进行定位，这包括对地图、路径、目的地等的定位，这些定位需要通过GPS、IMU、LiDAR等设备来实现。
- 路径规划技术：自动驾驶技术需要对车辆的路径进行规划，这包括对速度、方向、路径等的规划，这些规划需要通过算法、模型等方法来实现。
- 控制技术：自动驾驶技术需要对车辆的运动进行控制，这包括对加速、刹车、转向等运动的控制，这些控制需要通过电子、机械等设备来实现。

这些核心概念之间的联系包括：

- 感知技术与定位技术：感知技术和定位技术是自动驾驶技术的基础，感知技术用于获取环境信息，定位技术用于处理环境信息，这两者之间需要紧密协同。
- 路径规划技术与控制技术：路径规划技术和控制技术是自动驾驶技术的核心，路径规划技术用于计算路径，控制技术用于执行路径，这两者之间需要紧密协同。

- 定位技术与控制技术：定位技术和控制技术是自动驾驶技术的基础，定位技术用于获取位置信息，控制技术用于处理位置信息，这两者之间需要紧密协同。
