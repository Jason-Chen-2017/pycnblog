在本节中，我们将提供一些具体的代码实例，以帮助你更好地理解触发器和存储过程的概念和操作。

## 4.1 创建触发器的代码实例

```sql
CREATE TRIGGER my_trigger
AFTER INSERT ON my_table
FOR EACH ROW
BEGIN
    -- 执行一些操作
END;
```

在上述代码中，我们创建了一个名为`my_trigger`的触发器，它在`my_table`表的插入操作后触发。`FOR EACH ROW`表示触发器会在每行数据插入后触发。

## 4.2 创建存储过程的代码实例

```sql
CREATE PROCEDURE my_procedure()
BEGIN
    -- 执行一些操作
END;
```

在上述代码中，我们创建了一个名为`my_procedure`的存储过程，它没有输入参数。我们可以在需要时调用这个存储过程，以执行一些操作。

## 4.3 调用触发器和存储过程的代码实例

```sql
CALL my_procedure();
```

在上述代码中，我们调用了名为`my_procedure`的存储过程。当我们需要执行存储过程中定义的操作时，可以使用`CALL`语句来调用它。
