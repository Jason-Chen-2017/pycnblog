# 2.核心概念与联系

## 2.1 漏洞扫描技术

漏洞扫描技术是一种自动化的网络安全技术，通过对网络系统进行扫描，发现潜在的安全漏洞。漏洞扫描器通常会对网络系统进行多种检查，如端口扫描、版本检查、网络服务检查等，以发现可能存在的安全漏洞。
