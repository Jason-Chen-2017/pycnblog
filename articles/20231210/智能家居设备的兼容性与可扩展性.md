# 4.具体代码实例和详细解释说明

在实际应用中，我们需要编写一些代码来实现智能家居设备的兼容性与可扩展性。以下是一个具体的代码实例和详细解释说明：

```python
import json
import requests

# 定义一个智能家居设备的接口类
class SmartHomeDevice:
    def __init__(self, device_id, device_type):
        self.device_id = device_id
        self.device_type = device_type

    def get_device_info(self):
        url = f"http://{self.device_id}.com/device_info"
        response = requests.get(url)
        if response.status_code == 200:
            device_info = json.loads(response.text)
            return device_info
        else:
            return None

    def control_device(self, command):
        url = f"http://{self.device_id}.com/control"
        payload = {"command": command}
        response = requests.post(url, data=json.dumps(payload))
        if response.status_code == 200:
            return True
        else:
            return False

# 创建一个智能家居设备对象
device = SmartHomeDevice("device1", "light")

# 获取设备信息
device_info = device.get_device_info()
print(device_info)

# 控制设备
is_controlled = device.control_device("on")
print(is_controlled)
```

在这个代码实例中，我们定义了一个`SmartHomeDevice`类，用于表示智能家居设备。这个类有一个`get_device_info`方法，用于获取设备信息，以及一个`control_device`方法，用于控制设备。我们创建了一个`SmartHomeDevice`对象，并使用这个对象来获取设备信息和控制设备。
