# 2. 核心概念与联系

在进行大数据分析和舆情监测之前，我们需要了解一些核心概念和联系。这些概念包括：

  1. 数据量庞大：数据量可以达到百万甚至千万级别，甚至更大。

  4. 数据分布性：数据可能分布在不同的地理位置和设备上。
  5. 数据复杂性：数据可能是结构化的、半结构化的或非结构化的。

- 数据清洗：数据清洗是一种对数据进行预处理和清洗的方法，以消除错误、缺失、噪声和异常值等问题，从而提高数据质量和可靠性。数据清洗是大数据分析的一个重要环节，可以帮助我们提取有用的信息。
