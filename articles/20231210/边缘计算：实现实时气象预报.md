## 4.2 气象数据预处理

在预处理气象数据之前，我们需要对数据进行一些清洗和处理。以下是一个简单的例子：

```python
import numpy as np

temperature = np.array(temperature)
humidity = np.array(humidity)
wind_speed = np.array(wind_speed)
pressure = np.array(pressure)

mean_temperature = np.mean(temperature)
std_temperature = np.std(temperature)
mean_humidity = np.mean(humidity)
std_humidity = np.std(humidity)
mean_wind_speed = np.mean(wind_speed)
std_wind_speed = np.std(wind_speed)
mean_pressure = np.mean(pressure)
std_pressure = np.std(pressure)

temperature_normalized = (temperature - mean_temperature) / std_temperature
humidity_normalized = (humidity - mean_humidity) / std_humidity
wind_speed_normalized = (wind_speed - mean_wind_speed) / std_wind_speed
pressure_normalized = (pressure - mean_pressure) / std_pressure
```

在这个例子中，我们使用了 NumPy 库来处理气象数据。我们将气象数据转换为 NumPy 数组，然后计算每个特征的均值和标准差。然后我们对气象数据进行归一化处理，以便在后续的分析中更容易处理。
