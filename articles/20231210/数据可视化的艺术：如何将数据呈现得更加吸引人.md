# 2. 核心概念与联系

## 2.1 数据可视化的目的

数据可视化的主要目的是将数据转换为图形、图表或其他可视形式，以便更好地理解和分析数据。这有助于揭示数据中的模式、趋势和关系，从而支持决策和分析。

## 2.2 数据可视化的类型

数据可视化可以分为两类：静态和动态。静态数据可视化是指不会随着时间的推移而发生变化的数据可视化，如条形图、饼图等。动态数据可视化则是指随着时间的推移而发生变化的数据可视化，如动画、交互式图表等。

# 4. 具体代码实例和详细解释说明

## 4.1 条形图实例

以下是一个使用Python的Matplotlib库创建条形图的代码实例：

```python
import matplotlib.pyplot as plt

# 创建数据
data = [5, 10, 15, 20, 25]

# 创建条形图
plt.bar(range(len(data)), data, width=0.5, color='blue')

# 添加标签和标题
plt.xlabel('Category')
plt.ylabel('Value')
plt.title('Bar Chart Example')

# 显示图表
plt.show()
```

在这个代码实例中，我们首先导入了Matplotlib库。然后，我们创建了一组数据，并使用`plt.bar()`函数创建了一个条形图。我们还添加了标签和标题，并使用`plt.show()`函数显示了图表。

## 4.2 饼图实例

以下是一个使用Python的Matplotlib库创建饼图的代码实例：

```python
import matplotlib.pyplot as plt

# 创建数据
labels = ['Category 1', 'Category 2', 'Category 3']
sizes = [15, 30, 55]

# 创建饼图
plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=90)

# 添加标签和标题
plt.axis('equal')
plt.title('Pie Chart Example')

# 显示图表
plt.show()
```

在这个代码实例中，我们首先导入了Matplotlib库。然后，我们创建了一组数据，并使用`plt.pie()`函数创建了一个饼图。我们还添加了标签和标题，并使用`plt.show()`函数显示了图表。
