# 2.核心概念与联系
## 2.1 Python简介
### 2.1.1 Python概述
- Python 是一种高级的、通用的、解释型、面向对象、动态数据类型的编程语言。
- Python 拥有相对较丰富的内置数据结构，例如列表、字典、集合、元组等；还提供列表推导式、生成器表达式、异常处理机制、垃圾回收机制，支持函数和匿名函数，支持多种编程模式，如面向对象的编程、命令式编程、函数式编程。
- Python 支持模块化编程，可以轻松实现面向对象的设计模式。
- Python 提供自动内存管理，支持交互式命令行环境，允许跨平台移植性。
- Python 的优点主要包括易学易懂、可读性强、运行速度快、适合解决各种工程问题、良好的文档和生态系统。
### 2.1.2 Python版本特性
- Python2 和 Python3 两个版本并存，并逐渐成为历史。
- Python2 的生命周期终结于2020年1月1日。
- Python3 发布了3.7版本，增加了许多新特性，比如异步 I/O，加强的 Unicode 支持等。除此之外，Python 3 引入了新的语法和一些功能上的改变，这使得 Python 在兼容性上比 Python 2 更好。
- 目前绝大多数第三方库、工具、框架都只支持 Python 3 或更高版本。
- 目前大部分的开源项目已经迁移到 Python 3 上。
## 2.2 Python语言特征
### 2.2.1 动态数据类型
- Python 是一种动态数据类型语言，这意味着变量不需要事先声明数据类型，而是在运行时根据赋的值决定数据类型。
- 例如，a=3后，变量 a 的值是一个整数。而b='hello'则是字符串。
- 使用动态数据类型能够很方便地创建可变的数据结构，避免了数据类型过分束缚的问题。
### 2.2.2 解释型语言
- Python 是一种解释型语言，这意味着程序在执行的时候不是一次编译，而是一条条语句从上往下依次执行。
- 解释型语言一般情况下较编译型语言运行速度要慢一些，但它不像编译型语言那样需要预先将代码编译成机器码才能运行。
- 当然也存在某些特定的情况，比如一些简单的脚本程序或者快速原型制作，可以使用解释型语言来提升效率。
### 2.2.3 命令式和函数式编程
- Python 中提供了两种编程方式：命令式编程和函数式编程。
- 命令式编程（Imperative programming）指的是基于语句的编程风格，通过命令的执行来改变程序的状态。
- 函数式编程（Functional programming）指的是基于函数运算的编程风格，函数作为第一公民，只接受输入参数并且返回一个输出结果。
- 在 Python 中，对于命令式编程，可以通过显式地对变量赋值来修改变量的值，也可以使用条件语句和循环语句实现流程控制；对于函数式编程，可以使用 lambda 来创建匿名函数，还可以使用 map() 和 reduce() 等函数来高效地操作数据。
### 2.2.4 面向对象编程
- Python 通过封装、继承、多态等机制实现了面向对象编程。
- 对象是一个具有状态和行为的封装体，包含数据和逻辑。对象之间的交互由消息传递机制实现。
- 通过类和对象可以构建出复杂的程序结构，简洁明了、易于维护。
## 2.3 Python语法规则
### 2.3.1 标识符
- 标识符就是用来命名变量、函数、模块等实体的名称。
- 标识符只能包含字母数字以及下划线(_)，且不能以数字开头。
- 可以采用驼峰命名法来定义标识符，即首字母小写，其他单词首字母大写。
```python
myName = "John" # CamelCase style
MyVariable = 10   # CamelCase style
file_path = "/home/user/"    # snake_case style
def myFunction():
    pass   # lowercase with underscores if needed
```
### 2.3.2 行和缩进
- Python 源代码文件以换行符\n结束，每行语句末尾必须有空白字符(包括Tab键)来表示语句块的层次关系。
- 每个代码块（如函数或if语句）应该缩进相同数量的空白字符。
- 推荐使用4个空格作为缩进单位，而不是1个或2个空格。
### 2.3.3 多行语句
- Python 的很多语法元素允许出现多行形式，这些语法元素被称为“代码块”。
- 如果代码块由多个连续的换行符构成，就算没有其他字符也是多行代码块。
- 在这种情况下，不需要显式地使用反斜杠(\)，而是直接换行即可。
- 下面的例子演示了一个多行注释。
```python
'''This is a multi-line comment. It can be used to add larger explanations or
descriptions for complex code segments.'''
```
### 2.3.4 特殊字符
- 有时为了表示特定含义，Python 允许在标识符、关键字、数字、字符串文字等中使用一些特殊字符。
- 以下是一些常见的特殊字符：
| 字符 | 描述 |
| ---- | ---- |
| \     | 表示转义符 |
| \'    | 表示单引号 |
| \"    | 表示双引号 |

| \t    | 表示制表符 |
| \n    | 表示换行符 |
| \r    | 表示回车符 |
| \f    | 表示换页符 |
