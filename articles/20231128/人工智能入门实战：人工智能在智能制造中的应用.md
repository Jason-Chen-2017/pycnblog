# 2.核心概念与联系
## 2.1 人工智能（Artificial Intelligence）
### 2.1.1 定义
人工智能（Artificial Intelligence，AI），通常指研究、开发、应用计算机技术来构造模拟人类智能的机器。20世纪70年代末，以弗雷德里克·卡尼曼为代表的科研人员提出了“机器学习”、“自然语言处理”和“知识推理”三个关键词，开启了人工智能的探索与开发进程。到上世纪90年代初期，人工智能成为研究热点，学术界和工业界纷纷投身其中，涉及领域有智能控制、图像识别、语音识别、机器人、专家系统、模式识别等。近几年，随着人工智能技术的飞速发展，人们越来越关注这些关键词背后的理论基础、算法理论、实践经验、应用效果、未来的发展方向，甚至还有一些关于对人工智能进行社会责任评估的新概念。

总体而言，人工智能是指研究、开发、应用计算机技术来构造模拟人类智能的机器。人工智能研究的对象是智能计算系统、智能生物系统、智能机械系统、智能系统。目前，人工智能技术主要应用于智能机器人的开发、自动驾驶汽车、虚拟助手、无人机、金融保险、文字识别、语音合成、知识图谱、商业决策分析、广告推荐等领域。

### 2.1.2 发展历程
人工智能的研究和发展历程可以概括为以下五个阶段。

1. 原始人工智能
1956年，艾伦·图灵在贝尔实验室提出的“模仿心理学”理论首次开启了人工智能的研究之路，奠定了人工智能研究的基石。

2. 符号主义人工智能
1970年代后期，人工智能研究进入了一个重要的转折时期，就是符号主义人工智�作为研究的主线逐渐走向成熟，尤其是在电子计算机的普及和技术革命的影响下。符号主义意味着计算机只能处理与符号形式的数据，由此产生了基于规则和表格的符号逻辑理论。

3. 连接主义人工智能
1980年代，连接主义人工智能研究重点转向于建立连接主义计算机系统的理论基础，目的是将感知、认知、理解以及动作的多个系统模块之间建立复杂的功能联系，最终完成对各种任务的智能响应。连接主义计算机系统具有高度的并行性，能够快速处理大规模的数据集，且易于部署、扩展。

4. 深层学习人工智能
深层学习的目标是借鉴生物学习的神经网络结构，构建复杂的非线性函数模拟人类的学习行为。与其他机器学习方法相比，深层学习人工智能利用大量训练样本、特征提取、正则化、无监督学习等技术来减少模型复杂度，提高学习性能。

5. 智能机器人
智能机器人的目标是实现实体动作的自动化，是人工智能技术的高级形态。在这样一个过程中，智能机器人不仅需要具备强大的计算能力，还需要能够自动获取丰富的感官信息、制定合理的动作序列、反馈结果、处理复杂的环境信息。

### 2.1.3 AI与智能制造
AI技术已经逐渐融入到智能制造领域，有三种主要的方式。

- （1）数字化转型——融合传统制造模式与AI技术。在过去的数十年间，制造业技术革命带来了巨大的发展，但由于缺乏自动化设备、无法满足顾客需求，许多企业仍然采用传统的工艺制程和手动操作，这就给制造业带来了巨大的困难。2014年，英特尔发布了第二代智能制造平台——雷锋智能制造平台（英文缩写为Littlefoot），在该平台上，用户可以通过简单地点击鼠标或手势操作，快速地完成生产过程。

- （2）智能设备嵌入制造——利用传感器、雷达、人工智能等技术将智能设备嵌入到工厂生产线中，为企业提供预测性的生产力和更可靠的生产方式。这套方案利用边缘计算、无人机等技术实现传感数据的收集、计算和传输，实现对工厂的整体管理。

- （3）云计算——构建基于云端的人工智能技术平台，可在全球范围内快速扩张，提高智能制造的能力水平。据观察，企业越来越依赖云计算平台，如亚马逊Web服务、微软Azure云服务等，他们的智能制造解决方案已从传统的工厂流程向工业4.0和物联网迁移。
