# 2.核心概念与联系
## 2.1 Python
Python是一种开源、免费、跨平台、可移植、解释性高的脚本语言。它的设计目的是用来进行简单且高效的数据处理、自动控制、网络通信等。目前，Python已逐渐成为云计算领域的事实标准编程语言。

## 2.2 流程自动化
流程自动化（Process Automation）又称为智能化办公，它指利用计算机技术将企业组织的各项工作流程转变为自动化的管理模式，通过计算机执行人工操作过程中重复性繁琐、易错、易出错的手动过程，缩短工作周期，提高工作质量，降低成本。流程自动化包括文档处理、数据导入导出、审批流处理、事务跟踪管理、资源管理、报表生成等功能模块。

## 2.3 Pywinauto
Pywinauto是一个开源Python库，用于Windows GUI测试，它提供了一个简单易用的API，允许你编写自动化脚本来处理各种Windows窗口应用程序的窗口控件，比如打开、关闭、移动、最大化、最小化、输入文本等。

## 2.4 Excel VBA编程
VBA(Visual Basic for Applications)，即Visual Basic宏，是微软开发的一种基于事件驱动的脚本语言，用以简化Microsoft Office应用程序的开发。VBA可以通过鼠标点击的方式运行脚本，也可以按照一定顺序进行运行。你可以通过修改脚本参数来实现各种各样的自动化操作。

## 2.5 OCR(Optical Character Recognition)
OCR(光学字符识别)，也称文本识别，是计算机对文字图像进行分析并输出其文字字符串的技术。通常OCR由字符识别引擎与字典词典组成。该技术被广泛应用于各类场景，如银行取款机、支付宝收款码、身份证扫描件、股票行情快照等。

## 2.6 Selenium
Selenium是一个开源的自动化测试工具，它提供了一套完整的WebUI自动化测试框架。它能够自动化浏览器驱动，让测试者无需关心底层技术就可以快速地进行测试。

## 2.7 Beautiful Soup
Beautiful Soup是一个可以从HTML或XML文件中提取数据的Python库。它提供了简单、快速、透明的API，能够轻松处理复杂的HTML页面。Beautiful Soup提供一个标签语法解析器来查找、导航、搜索文档树。

## 2.8 Scrapy
Scrapy是一个基于Python的快速、高级爬虫框架，主要用于web抓取。Scrapy具有强大的Spider API，可以灵活地定义爬取策略，同时内置了多种网页数据解析算法。由于Scrapy本身具有良好的扩展性，因此你可以很容易地为不同的网站编写新的爬虫。
