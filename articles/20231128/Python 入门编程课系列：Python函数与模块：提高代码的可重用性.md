# 2.核心概念与联系
## 函数（Function）
在计算机科学里，函数是用来实现特定功能的一段代码。它接受输入参数，执行指定的运算，然后返回输出结果。在不同的编程语言中，函数的定义方式略有不同。但无论如何，函数都是一个独立的、可执行的、可重复使用的程序片段。函数提供了一种封装代码的方式，使其结构更加整洁、易于阅读。另外，函数也提供了高效率的方法，避免了重复的代码编写，提高了代码的复用性。

在Python中，我们可以使用def关键字来定义一个函数。例如：

```python
def my_function():
    print("Hello World!")
```

my_function是一个函数名，后面跟着一对括号，里面可以放置函数的参数列表。然后在下一行使用冒号(:)分割函数头和函数体，函数体由一系列语句构成。当函数被调用时，函数体内的语句将被顺序执行，并且局部变量只在函数内部存在。例如：

```python
>>> my_function()
Hello World!
```

函数的参数可以有默认值，也可以不指定参数类型。例如：

```python
def greet(name="world"):
    return "Hello, {}!".format(name)

print(greet()) # Output: Hello, world!
print(greet("Alice")) # Output: Hello, Alice!
```

这里面的greet函数有一个参数name，默认值为"world"。函数体内部使用了字符串格式化方法，并通过return返回了一个字符串。我们可以通过调用该函数来传递参数，得到不同的问候语。

## 模块（Module）
模块（module）就是存放在文件中的Python代码集合，通常用于实现特定功能，可以被其他程序导入使用。它的特点包括：

1. 模块可以有自己独立的作用域，使得命名空间更加有效；
2. 模块提供接口（interface），使得其他程序更方便调用模块中的函数；
3. 模块支持多种形式的导入，使得代码更加灵活和模块化。

在Python中，模块的定义类似于函数的定义，但是模块没有参数，只有函数体。例如：

```python
# file: module.py
def add(x, y):
    return x + y
```

上面定义了一个叫做add的函数，保存到文件module.py中。那么这个文件如何使用呢？我们要先把模块文件添加到搜索路径（path），然后通过import命令导入模块，并调用其中定义的函数：

```python
# main program
import module

result = module.add(10, 20)
print(result) # Output: 30
```

在上面的例子中，我们通过import module从模块文件module.py中导入了一个叫做add的函数。然后通过模块名来调用该函数，并将两个数字相加。最后打印结果。如果我们想导入多个函数，可以在模块中定义多个函数，然后再调用即可。
