                 

# 1.背景介绍


智能边缘计算（英语：Smart Edge Computing），又称为嵌入式人工智能、边缘智能计算或感知计算，是指通过集成到用户终端设备的计算能力和处理能力，实现智能计算、智能监控、智能分析等功能的一种新型计算模式。它是由分布式的云计算平台、本地化的传感器网关、数据中心、应用编程接口（API）以及通信网络等硬件和软件组成。它能够在远距离、低带宽、弱电流环境下提供高效、可靠、精准的计算、处理和控制服务。2017年，华为宣布推出首款基于智能边缘计算的Mate X系列产品。
随着边缘计算领域的火爆，越来越多的公司、组织和个人开始关注和探索该领域，希望通过结合云计算、机器学习、IoT等技术，构建更智能、更便捷的生活方式。本书基于个人在边缘计算行业工作经验所编写，通过通俗易懂、具体详实的案例教程，旨在帮助读者快速掌握智能边缘计算的相关知识并具备可落地的实际应用能力。本书力求将智能边缘计算的核心理论和方法论讲透彻、细致入微，以期能够帮助读者搞清楚它的核心概念和发展方向。
# 2.核心概念与联系
## 2.1 智能边缘计算定义
智能边缘计算定义：智能边缘计算(Smart edge computing)，指将基于云的大规模分布式计算平台、联网的传感器网关、集成处理单元、操作系统、网络、存储设备以及其他网络组件等集成在用户终端设备上，实现智能计算、智能监控、智能分析等功能的一种计算模型。智能边缘计算一般包括两种类型的计算节点，分别为智能边缘端点(Smart Edge Endpoints)和智能边缘云(Smart Edge Cloud)。智能边缘端点(Smart Edge Endpoints)是部署在网络边缘位置的具有高算力和处理能力的嵌入式设备，通常配备了处理单元和运算能力强大的芯片组，可以对传感器信息进行实时采集、传输和处理；而智能边缘云(Smart Edge Cloud)是分布于全球各地的服务器集群，用于承载智能边缘端点，并为其提供计算、存储、网络和安全服务。通过在分布式、动态的环境中，实现大数据、图像、视频和实时事件处理等智能计算功能。

## 2.2 智能边缘计算关键要素
### （1）边缘节点
智能边缘计算的研究重点主要集中在边缘节点这一关键环节。边缘节点是一个完整的系统，由物理层、系统软件、传感器网关、网络层、应用程序运行环境等组成，可以独立完成某项特定的任务。如智能视频监控系统中的摄像头就是一个典型的边缘节点。

### （2）云计算平台
云计算平台是智能边缘计算的一个重要组成部分。云计算平台是在线服务提供商提供的计算、存储、网络和安全服务，可为智能边缘节点提供基础设施、应用开发环境、数据管理工具及服务。目前比较受欢迎的云计算平台有AWS、Azure、GCP等。

### （3）传感器网关
传感器网关是智能边缘计算的核心组件之一。传感器网关负责与边缘节点的连接、接收、处理和转发传感器信号。常用的传感器网关有串口网关、IP网关、NB-IOT网关等。

### （4）计算资源
计算资源是智能边缘计算的核心组件之一。计算资源可以是CPU、GPU、FPGA等加速芯片，也可以是内存、存储等计算资源。计算资源的选择需要综合考虑智能边缘计算的性能要求、边缘节点的功耗要求、边缘节点的网络带宽和延迟要求。

### （5）软件系统架构
软件系统架构是一个复杂的系统工程，需要涉及到数据库、通信网络、操作系统、编译器、调试工具、安全机制等多个方面。但总的来说，软件系统架构与普通的Web应用不同，其考虑因素更多，例如服务的分布性、弹性扩展性、可靠性保证、资源利用率、可用性等。

### （6）安全保障
安全保障是智能边缘计算的另一个重要组成部分。安全保障可以分为两层，即终端设备层面的安全和应用层面的安全。其中终端设备层面的安全涉及到安全、保密、体系结构等方面，应用层面的安全包括访问控制、数据加密、消息验证等。

### （7）服务开放接口
服务开放接口是智能边缘计算的另一个重要组成部分。服务开放接口可以向外部提供标准化的API，供外部应用调用。这些接口支持各种计算、存储、通信和安全等服务，既包括内置算法，也包括第三方服务。

### （8）联网协议
联网协议是智能边缘计算的另一个重要组成部分。联网协议包括无线局域网、无线广域网、卫星传输链路等，它们的设计和优化都有利于提升智能边缘计算的性能。

### （9）服务质量
服务质量是智能边缘计算的重要指标。服务质量直接影响用户满意度、经济价值和社会影响。因此，服务质量的好坏与智能边缘计算的应用场景密切相关。

## 2.3 智能边缘计算分类
目前，智能边缘计算有两个主要的分类：
1. 第一类：物联网+边缘计算（LoRaWAN + Smart Edge Computing）

   LoRa是一种新的低功耗的无线通信协议，该协议可以在短距离内传输长达几百kb的数据。LoRaWAN（Long Range Wide Area Network）采用LoRa协议制作的无线网络，可以实现智能化、智能监控、智能分析等功能。通过将LoRaWAN集成到智能边缘计算中，可以实现更快的响应时间、更低的功耗、更好的成本效益和可扩展性。

2. 第二类：边缘AI（Edge AI）

   边缘AI是一种与物联网相结合的AI技术，它利用AI算法将计算任务放在设备端，从而降低传感器与云端的通信延迟，提升设备的实时性和准确性。通过边缘AI，可以实现精准运动检测、智能交通预测、食品追溯等功能。
