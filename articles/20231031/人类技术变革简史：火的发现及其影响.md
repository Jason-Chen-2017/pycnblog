# 1.背景介绍


## 沙漠火灾历史回顾
从古至今，地球上曾经发生过很多次沙漠火灾。在发达国家，如美国，西欧等，人们通过抽烟、燃烧瓶等的方式对外界传染病毒，导致数千万人的死亡。然而沙漠地区却因为缺少维生素A，营养不良，导致出现沙漠火灾，可是也没有人预料到这么大的事情。直到20世纪70年代，英国的科学家惠特克·罗伊德和他的同事们携手研究出了沙漠火灾的原因。他们首次将沙漠火灾视为一种致命的生物疫情，并用炭疽病毒作为武器，对德国居民进行“空前的”烧伤，造成数百人丧命。

从古至今，由于沙漠火灾的频繁发生，产生了许多的问题。比如：

1.缺乏有效的应急预案和救治方案；
2.不可抗力因素的影响，比如气候变化等；
3.保护私有财产的人权，比如无辜百姓被火灾摧残后难以求生；
4.经济损失和社会动荡，比如房价上涨、员工下岗、个人信用评级下降等；
5.法律压制、监督机构的滥用，给社会造成巨大的负面影响。

## 火灾研究的起源
火灾研究的起源可以追溯到十分久远的历史。人们对火灾一直有着各种看法和认识，但是最早提出的火灾理论发表于19世纪。其中，马里兰大学教授、物理学家布朗·豪尔斯（Bronwen Dehers）认为，“火是由光子、电子或者其他微粒组成的中微子的聚合体”。豪尔斯认为，宇宙中的任何形式的能量都不能转化为光子、电子或者其他微粒，除非它碰撞形成中子。而这些宇宙中的物质或能量存在某种程度上的不确定性，当它们相互作用时，就可能形成火的产生。当这种情况发生在沙漠地带时，称之为“沙漠火灾”，是因为沙漠潮湿，沙尘暴使火焰喷吐出来。因此，19世纪末期，苏联科学家尼古拉·佩利（Nicholas Pillik）用中子学观点，首次系统阐述了沙漠火灾的本质。

随着时间的推移，学者们逐渐接受到这种中子学观点，并且探索火的产生机制。佩利发现，在核反应堆核燃料中释放的中子，可能会破坏原有的核芯，从而使火电厂燃烧的原子熔融和聚集成为核芯的一个巨大的变化，从而激活核反应堆中的其他原子，激发出由微型中子组成的大型中子。当微型中子和巨型中子相互作用的结果时，形成了一种叫做噬菌体的蛋白质团块状结构。噬菌体释放出的光子击中核燃料，形成火电厂燃烧的热焰。随着时间的推移，科学家们越来越多地使用中子学方法，对于火灾的研究更进一步，今天我们所看到的许多沙漠火灾，其实都是由中子泄露、核反应以及一些微观现象共同造成的。
