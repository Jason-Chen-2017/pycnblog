# 2.核心概念与联系
## 2.1 万有引力定律（Gravitation Law）
万有引力定律也叫爱因斯坦力学方程或普朗克定律。它是描述物体之间力作用在各个点之间的行为规律，在数学上表示为：

F=G/r^2(其中，F代表物体受到的力；G代表施加该力的观察者所在空间的普适常数；r代表两个相邻物体之间的距离)。

一般来说，引力可以分为两种：

1) 全重力引力：物体受到的重力加速度等于其自身质量除以平方的大小，即：
   
   a_g = G*M/r^2

   其中，a_g 为重力加速度，M 是物体的质量。
2) 惯性引力：物体受到地球引力或其他物体引力所引起的引力，称为惯性引力。惯性引力与质量无关，一般情况下，表现为球状运动或椭圆状运动。
   
   F = (μm_1*m_2)/r^2 (其中 μ 为系数，通常取值约为 0.1-0.3)。 

在宇宙中存在着无数种物理现象，如电场、磁场、光波等等，它们都受到万有引力定律的制约，包括导致运动的万有引力和引力变化带来的旋转、压缩和变形等多种物理现象。

## 2.2 电荷与电场
电荷是一个比例尺上的电量，单位电荷单位为 Coulomb（C）。不同物质的电荷量不一样，不同的电荷守恒。

假设有一个磁铁粘在一块金属表面，那么它会产生一个电场。把一个两极场放在表面附近，由于介电常数ε<<1，可近似认为无穷小。因此，当我们用交流电时，在磁铁周围形成的电场能够用两极场近似表示出来，这是因为交流电磁感应强度与场强相关联，而场强随着电荷数量增加而增加，这使得我们可以在各种条件下使用交流电，而不是像导线那样只能在某些特定场合才能使用。因此，只有真空环境下才可以产生静止导线、测电阻器等。

电场是指电荷分布在空间中的势，它可以用矢量形式表示，如位移电荷D·E/q（其中D是电荷的位移向量，E是电场的方向，q是电荷的数目）；也可以用标量形式表示，如电荷的总能量。

电场分辨率高，作用范围广，而且可以反映各种微观结构和宏观尺度的电荷分布。由于电场具有聚变、扩散和切变效应，所以它是研究运动物体及其载体电场的有力工具。

## 2.3 麦克斯韦方程组
麦克斯韦方程组是电磁学的一个基本方程集，由物理学家克里斯托弗·莫尔斯发现。麦克斯韦方程组是用来研究由电荷势给出的电场的。它可以表示为：

div(sigma grad v) = -i (grad phi)，其中，sigma 表示电导率，grad 表示梯度算符，v 表示速度，phi 表示电荷势。

第一个方程是电场方程式，第二个方程是薛定谔方程。
