# 1.背景介绍


​    深度学习（Deep Learning）是近几年非常火热的研究方向。它在图像、语音、视频等领域都取得了不错的效果。随着大规模数据集、计算能力的提升、GPU集群硬件的发展，深度学习技术已经逐渐成为各行各业应用的标配。如此高效的机器学习方法使得深度学习模型也越来越重要。
​    大型的深度学习模型往往由多个网络层组合而成。每个层的输入都是前一层的输出，每层又可以分为多个神经元组成。这样的一个复杂的结构使得深度学习模型很难用手动的方式进行设计。因此，模型架构搜索（Model Architecture Search）技术应运而生。该技术通过自动化的方法对复杂的模型架构进行探索并找到最优的解决方案。其中，基于深度学习的模型架构搜索技术成为最新研究热点。
​    在人工智能系统中，通常会使用许多不同的模型作为基准模型对数据进行预测。这些基准模型之间存在差异性，比如模型的复杂程度、所使用的特征或损失函数不同。这些差异性使得模型的性能无法统一评价。而自动模型搜索技术可以帮助我们找到一个合适的基准模型，既具有较好的性能，又能够快速有效地处理新的数据。通过这种方式，可以降低研究和部署模型的时间和成本。
​    本文将会从两个方面阐述自动模型搜索与架构优化。第一部分将介绍模型架构搜索的基本概念、分类及现有的研究。第二部分则会介绍如何利用深度学习的神经网络自动进行模型架构搜索。具体的内容包括：模型架构搜索的目标、模型架构搜索的算法原理、模型架构搜索的具体实现、算法参数设置、测试指标、模型架构搜索的评估及未来的方向。最后，本文还会给出一些相关技术的总结及典型案例，可供读者参考。

# 2.核心概念与联系
​        模型架构搜索是一个计算机科学领域里最重要的问题之一。它的目的是为了找到一个模型的最佳架构，即设计一个模型，这个模型的架构足够复杂、表达能力强、同时又保证其泛化能力较好。一般来说，模型架构搜索的任务可以分为两类：超参数调优和模型压缩。超参数调优就是调整模型中的超参数（比如学习率、权重衰减、正则项系数等）以提升模型的性能；模型压缩则是减少模型大小或者加速模型运行速度，减小模型的存储空间占用。
​        概括来说，模型架构搜索其实就是在优化搜索空间内寻找全局最优的模型架构。与传统的机器学习模型训练过程不同，模型架构搜索要考虑模型架构和超参数两个变量之间的交叉影响，而且可能涉及到大量的离散变量选择，也就是超参组合爆炸问题。所以，模型架构搜索的关键是建立一个高效的搜索算法，能够在大规模超参组合的情况下找到全局最优解。
​        由于模型架构搜索涉及到超参数搜索、模型架构搜索、模型压缩等多种因素，导致其优化问题形式更加复杂。模型架构搜索主要涉及超参数优化、模型架构搜索、模型剪枝等问题。因此，对于这类问题，模型架构搜索也需要一套完整的理论框架来进行分析。接下来，我们就来看一下模型架构搜索的几个常用的模型架构搜索方法。
