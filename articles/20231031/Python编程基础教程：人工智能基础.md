# 1.背景介绍

  
人工智能（Artificial Intelligence，AI），英文缩写为AI，是指让计算机具有智能的能力，能够自动地进行某些重复性的任务或者决策。它通常是指由人工神经网络连接起来的复杂系统，包括感知、推理、学习、交流等多方面的能力，通过大数据、机器学习、模式识别、自然语言处理等方式对外界输入信息进行分析、理解和加工，并作出相应的反应。  


人工智能目前正在成为一个引领行业的产业，无论从经济利益还是社会价值都处于世界前列。很多创业者都试图利用人工智能来打造新的产品和服务，如自动驾驶汽车、智能机器人、虚拟现实等。在传统行业中也逐渐出现了人工智能应用场景，如语音助手、图像识别、个性化推荐等。

随着计算机视觉、机器学习、自然语言处理、深度学习等领域的不断发展，人工智能技术也日新月异。相对于其他一些技术，比如金融科技、生物医疗科技、生物信息学、材料科学等，人工智能仍是一个比较小众的研究方向。2019年，中国互联网协会统计显示，全球只有6%-7%的科研人员投入到人工智能领域，而中国却占据整个领域的90%以上，其规模、影响力令世人瞩目。


本教程将从最基本的人工智能知识开始，包括定义、分类、相关技术及应用、未来趋势、经典案例、常见问题解决办法等。适合初级至高级工程师阅读。

# 2.核心概念与联系  
## 2.1 定义  
人工智能（Artificial Intelligence，AI），英文缩写为AI，是指让计算机具有智能的能力，能够自动地进行某些重复性的任务或者决策。它通常是指由人工神经网络连接起来的复杂系统，包括感知、推理、学习、交流等多方面的能力，通过大数据、机器学习、模式识别、自然语言处理等方式对外界输入信息进行分析、理解和加工，并作出相应的反应。  

## 2.2 分类  
根据AI所涉及的功能范围及分支结构不同，可以把人工智能分成四大类：机器学习、知识发现、模式匹配、基于规则的计算。下面，我们简要介绍一下这四大类。  

### （一）机器学习  
机器学习（Machine Learning）是人工智能的核心技术之一，旨在让计算机学习如何去做某件事情，而不需要被动地接收、解释或重复命令。它通过训练样本、归纳规则或优化参数，来对输入的数据进行预测、分类和回归。通过不断的学习、改进和优化，机器学习系统最终可以实现高度准确的预测和决策。机器学习可分为监督学习和非监督学习两种类型，监督学习要求输入数据既有标签又有描述信息，而非监督学习则不需要标签信息。

典型的机器学习应用有图像识别、文本分类、垃圾邮件过滤、病毒检测、推荐系统、人脸识别、语音识别、个性化搜索等。

### （二）知识发现  
知识发现（Knowledge Discovery）是人工智能的一类技术，主要关注对数据的结构和模式进行建模，从而找出隐藏于数据中的有用信息。其典型应用有市场营销、风险评估、遗传学、生物信息学、化学信息学、天文学等。

### （三）模式匹配  
模式匹配（Pattern Matching）是人工智能的一类技术，用于匹配输入数据中的模式，从而完成某种任务，如图像识别、事务处理、文本摘要、文档分类、信息检索等。其典型应用有文字识别、图像配准、图像增强、人脸识别、语音识别、手写体识别、对象跟踪、地图导航等。

### （四）基于规则的计算  
基于规则的计算（Rule Based Computer）是人工智能的一类技术，它采用一系列的规则来对输入数据进行处理和决策，而这些规则不能被学习或优化。典型应用有图像过滤、数据挖掘、文字识别、排序算法、游戏规则、股票交易规则等。
