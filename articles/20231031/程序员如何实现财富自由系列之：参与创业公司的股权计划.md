# 1.背景介绍


股权激励计划（Equity Incentive Planning）是指由个人或机构为某个创业公司或企业提供的资本方面的激励机制，其目的是通过向员工、董事会及决策者等提供股份的方式，吸引他们加入到创业公司或企业的董事局中，并提升他们的影响力，达到“激励创业”的目的。而对于创业公司来说，股权激励计划往往能够帮助其快速扩张，获取更多资金投入到公司内部运营，并且增强公司自身的竞争能力。由于股权激励计划的引入，创业公司逐渐成为风险投资人们的目标。

在过去的一段时间里，许多创业者都开始意识到自己需要考虑加入一家风险投资机构，寻求“股权激励计划”，获得所需的资金支持。然而，如何掌握正确的股权激励计划，如何精确地规划出一份让员工满意的股权激励计划，这些都是需要具备丰富经验的商业人士和投资人才能完成的任务。

为了能够更好地参与到创业公司的股权激励计划当中，以下是一些需要注意的问题：

1.筹措资金：创业公司需要根据投资规模、预期收益率、期限等因素进行资金筹措。在谈论股权激励计划的时候，可以明确清楚自己的目的、愿景、盈利模式，以及要达成的目标。并确定适合于公司发展的资金要求。如果资金资助计划已经存在，则可以询问创始人的意见，或者联系资助方取得资助情况。

2.如何提取价值：创业公司的发展需要长期投资，因此无法回避投资过程中的价值发现过程。通过将股权激励计划纳入公司管理体系中，创始人及员工都会受益。但另一方面，公司也需要承担相应的风险。因此，在股权激励计划的设计过程中，一定要慎重考虑其潜在损害。另外，建议评估股权激励计划的有效性，避免产生不必要的损失。

3.管理层结构：创业公司一般都有自己的管理层团队，成员可能包括CEO、COO、CFO等。为了确保各个部门之间的平衡，建议先选定一名带有激励或管理才能的人，建立专门的激励团队，或委派代表，做好交接工作。同时，制定激励策略时，最好搭配工具（如表格、调查问卷），根据实际情况调整激励计划。

4.宣传与培训：对于创业公司来说，宣传股权激励计划是一个很重要的环节。因为它能拉近员工与企业的距离，促进员工加入董事局，让他们有参与到创业公司管理层的机会。因此，需要经过充分的宣传和培训，使员工对该计划更加了解。

5.法律政策：参与股权激励计划涉及的法律法规众多，不同的国家甚至不同条款下的规定也不同。建议在开始之前，仔细阅读相关法律法规，熟悉激励计划可能面临的法律风险，并配合所在地区内的律师事务所进行辅导。

通过以上问题，创业者可以找到解决自己的财富自由问题的途径。
