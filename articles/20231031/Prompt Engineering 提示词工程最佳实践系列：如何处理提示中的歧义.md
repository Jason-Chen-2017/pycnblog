# 1.背景介绍


自动问答（Automatic Question Answering）是近几年非常热门的研究方向之一。机器学习、深度学习等技术在本领域取得了很大的成功，也促进了人们对计算机智能方面的一些理解。问答系统中，生成的回答经常会存在着歧义性。一般来说，由于语言的复杂性和变化多样性导致的语义模糊等，会使得系统的输出产生歧义。然而，对于用户来说，输入一段文字之后获取到清晰、准确的回复是十分重要的。因此，对于问答系统来说，如何准确、及时地处理用户输入、合理地进行自然语言理解并给出有效的回答就显得尤为重要。Prompt Engineering 就是为了解决这个问题而诞生的。
Prompt Engineering 是一项基于NLP技术的自动问答框架，通过匹配用户输入文本的关键信息，将其转换成标准化的模板，再根据用户问题生成对应的回答。该框架可以自动生成相关问题的提示，提升用户体验，减少用户错误输入，缩短答题时间。同时，也可以用于提供企业内部的FAQ系统。本文主要介绍如何使用Prompt Engineering 框架进行处理提示中的歧义。
# 2.核心概念与联系
## 2.1  Prompt
Prompt（引导句）是一个用来作为标准模板的输入的句子，它与用户的问题密切相关。当一个用户问题被识别出来后，系统便生成对应的回复。通过引入引导句，我们就可以提高系统的召回率和准确率。引导句的作用包括：

1. 将用户问题转化成可执行指令；
2. 减少无关的问题的出现；
3. 提供对话场景，增强对话效果；
4. 为未来的更细粒度的回复提供可能性；

## 2.2  NLU
NLU（Natural Language Understanding）即自然语言理解，它负责理解用户的输入信息，并将其转化成计算机可以处理的形式。其中包含了一系列的技术，如词法分析、语法分析、语音识别、实体抽取、意图识别、情感分析等。NLU 层的目标就是通过对用户问题的理解，生成合适的答案，并降低用户的自主程度。例如，我们可以利用 NLU 技术判断出用户问题的主旨，进而制定相应的答复策略，从而更好地满足用户的需求。

## 2.3  GPT-3
GPT-3 是由 OpenAI 团队于2020年推出的 AI 语言模型，能够实现拥有自己知识库、可以学习和创造的能力。其主流应用场景包括对话系统、文本生成、图像描述、摘要等。在今后的 AI 产品中，GPT-3 的应用也将越来越广泛。

## 2.4  Prompt Engineering
Prompt Engineering 是一套基于NLP技术的自动问答框架，通过匹配用户输入文本的关键信息，将其转换成标准化的模板，再根据用户问题生成对应的回答。该框架可以自动生成相关问题的提示，提升用户体验，减少用户错误输入，缩短答题时间。同时，也可以用于提供企业内部的FAQ系统。

如下图所示，Prompt Engineering由三大模块构成，包括 NLU 模块、生成器模块、引擎模块。其中，NLU 模块负责对用户输入的信息进行语义解析，生成词向量等候选答案列表；生成器模块基于候选答案列表生成符合当前语境的回答模板；引擎模块将候选答案列表、用户输入文本、模板映射到一起，并最终完成整个过程。通过 Prompt Engineering 可以将现有的FAQ数据库、已有问答系统的结构化数据转换为自动问答系统所需的数据。
