# 2.核心概念与联系
移动互联网主要由两类应用构成——手机应用（Mobile App）和网站应用（Web App）。移动应用与网站应用最大的不同就是用户端运行环境的不同。手机应用在用户端运行于智能手机或者平板电脑上，而网站应用则可以支持各个平台上的浏览器，包括PC和手机端。移动应用具有如下特性：

1. 沙盒机制：一个移动应用只能访问自己的数据，不能访问其他应用的数据。也就是说，每一个移动应用都是孤立的，互相之间没有权限、无法通信，彼此间不会产生影响。

2. 分发渠道限制：为了防止应用滥用，目前比较流行的限制就是通过第三方渠道分发应用。也就是说，应用只能通过指定的平台上架，并且只限定某些受众。

3. 用户付费意愿：由于移动互联网的信息获取能力和数据处理能力强，因此用户对应用的购买意愿一般都很高。比如，游戏类应用更是占据了用户的绝大部分注意力。

4. 免安装更新：移动设备的存储空间有限，因此应用一般都采用无需安装的轻量级方式，并且能够自动更新。

5. 数据隐私保护：移动应用具有数据本地化存储功能，因此能够避免互联网上数据的泄露风险。而且，移动设备的硬件性能和网络连接速度也逐渐提升，使得应用的性能和响应时间得到进一步提升。

网站应用则又称为云端服务，是一种基于Web的应用形式，它的特点是能够跨平台、跨终端、高度集成、具有高度的可伸缩性。例如，企业内部的门户网站就属于网站应用；在线教育网站也是一个典型的网站应用。与手机应用不同，网站应用的生命周期一般比较长，可以长久地提供服务。网站应用具有如下特征：

1. 快速响应：网站应用基于Web技术，具有较快的加载速度，用户可以及时获得最新信息。

2. 可扩展性：网站应用可以使用服务器集群、负载均衡器等组件，根据实际情况进行横向扩展，有效提高系统性能。

3. 安全性高：网站应用天生具备良好的安全性，任何攻击都会造成严重损失。

4. 免费托管：网站应用的部署和维护都不需要花费太多成本，它可以完全免费托管。

除了上面介绍的两种类型应用外，还有一种类型是混合应用（Hybrid App），这是指手机应用和网站应用的结合。这种应用既保留了手机端的用户体验，又融入了网站应用的功能。

综上所述，移动互联网应用主要分为两种类型——手机应用和网站应用。而后者又可以分为两种形式——云端服务和混合应用。接下来，作者将介绍这两种类型，分别对应两种类型的应用——移动应用和网站应用，以及最后介绍一款实用的移动应用——摩天钱包，这也是作者的主要关注点。

## 3.1移动应用
### 3.1.1HTML、CSS、JavaScript
HTML是最基础的标签语言，用于定义网页的内容结构，CSS用于定义网页的表现样式，JavaScript则用于实现动态效果和交互行为。作者将简单介绍一下HTML、CSS、JavaScript的基本知识，以及它们之间的关系。
#### HTML
HTML(HyperText Markup Language)即超文本标记语言，用于定义网页的结构。HTML的语法规则是“标签”+“内容”。其中标签用于描述内容的属性和结构，比如<head>、<body>等标签用于定义网页的头部和正文。HTML 5引入了许多新的标签，如语义化标签、新增表单元素等。除此之外，HTML 5还增加了很多新的特性，如Web应用程序、Web Sockets、绘图API等。
#### CSS
CSS(Cascading Style Sheets)即层叠样式表，用于定义网页的表现样式。它是一种用来表现HTML或XML文档样式的计算机语言。CSS 描述了HTML文档中元素的外观，包括大小、颜色、字体、边框、内边距、边界、轮廓等各种样式。CSS 3 引入了很多新的特性，如圆角、阴影、遮罩、动画、多列布局、弹性盒模型等。
#### JavaScript
JavaScript(JS) 是一种轻量级的脚本语言，用于实现网页的动态效果和交互行为。它是一种解释性语言，在运行时编译执行，具有动态类型和灵活的语法，支持面向对象、命令式、函数式等编程范式。它可以与 HTML 和 CSS 一起工作，也可以单独使用。JavaScript 提供了一系列 API，可以让 Web 开发人员创建丰富的交互式网页。
#### 三者关系
HTML是构建网页骨架的基本工具，CSS是给HTML添加视觉效果的工具，JavaScript则是赋予网页智能交互的工具。HTML + CSS + JavaScript = 移动应用！
### 3.1.2Apache Cordova
Apache Cordova是一个开源的移动应用开发框架。它允许开发者使用JavaScript、CSS、HTML及任意第三方库快速地开发出能够运行在iOS、Android、Windows Phone、BlackBerry 10、Firefox OS等平台上的移动应用。Apache Cordova提供了插件系统，开发者可以利用它来快速集成第三方插件，或者开发自定义插件，使得应用拥有丰富的功能和特性。
### 3.1.3Ionic Framework
Ionic Framework是一个开源的移动前端框架，它允许开发者使用Angular、TypeScript、Sass、CoffeeScript及其他前端技术构建移动应用。它围绕Apache Cordova打造，并增加了更高级的功能，如覆盖多个平台的统一设计语言、基于 Angular 的组件化开发、丰富的第三方库和插件支持等。Ionic Framework可以帮助开发者快速开发出高质量的移动应用。
## 3.2网站应用
### 3.2.1Node.js
Node.js是一个开源的服务器运行环境，它可以在服务器端运行JavaScript，因此也可以用于开发网站应用。Node.js采用事件驱动、非阻塞I/O模型，因此运行速度非常快。Node.js的包管理器NPM(Node Package Manager)，可以方便地下载、安装和管理第三方模块。通过npm，开发者可以快速搭建自己的网站应用，提高开发效率。
### 3.2.2Express.js
Express.js是一个基于 Node.js 的 web 应用框架，它是一个简洁而灵活的 JavaScript 框架，旨在提供一组强大而友好的功能。Express.js 可以方便地搭建 web 服务和 RESTful APIs，并提供一系列 HTTP 工具。通过 Express.js，开发者可以快速地构建网站应用，并与 Node.js 的生态圈深度集成。
## 3.3混合应用
混合应用是指手机应用和网站应用的结合。混合应用通常与两个框架一起使用——PhoneGap和Ionic，分别用于开发手机应用和网站应用。PhoneGap 可以帮助开发者构建快速、本地化、原生应用，而 Ionic 可以提供更加完整的、可复用的 UI 组件。
## 3.4摩天钱包
摩天钱包是一个移动应用，它提供类似支付宝、微信支付的支付功能，并且与很多金融机构合作，可以方便地收取存款、消费券等。因此，如果您有收入困难，可以考虑试用摩天钱包，并充值后开始尝试使用其支付功能。
