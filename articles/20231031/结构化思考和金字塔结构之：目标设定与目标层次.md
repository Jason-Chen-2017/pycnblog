# 1.背景介绍


随着互联网行业的蓬勃发展，其产品和服务的研发已经成为各大企业的必经之路，因此作为研发人员，我们不得不在此领域有所建树。随着IT技术日新月异、业务快速发展的时代背景，传统的瀑布开发模式逐渐瓦解，新的敏捷开发模式正在崛起。而敏捷开发的方法论也带来了新的思维方式、新型工作方式、新兴协作工具、新颖的流程管理模式等。其中目标管理是其中重要的一环。
目标管理即一个企业或组织的目标的制订、执行、跟踪、修正和完善过程，包括目标的设置、确认、设计、监控、评价和改进。目标管理是一个十分重要的过程，它对企业的产品、服务、管理、运营、人力资源等方面都产生着直接影响。比如，设定的目标可以是“持续盈利”或“年终奖金”，完成这个目标可以为企业创造出更多的经济效益；或者设定的目标可以是“客户满意度提升10%”，实现这个目标可以让客户获得更好的体验。只有正确地设定目标并落实到相关的工作、流程、管理制度中，才能确保目标顺利实现。
目标管理是企业的“IT第一生产力”。只有建立起科学、合理、可量化的目标管理机制，才能有效地优化管理、提升效率、改善质量。而且，由于目标管理机制严格遵循结构化思维和金字塔原理，能够有效地降低目标难度、增强执行效率、缩短周期、降低成本。因此，掌握目标管理知识、技能、能力，将成为每一位IT从业者不可或缺的基本素养。
那么目标管理又如何进行呢？目标管理分为目标确定、目标定义、计划实施三个阶段。
首先，目标确定，即确定企业需要达成的目标。比如，为客户提供优质产品和服务，扩大市场份额，拓展客户群体，这些都是需要达成的目标。目标确定后，目标定义则根据目标确定的内容细化程度不同，分为非常规目标、特别目标、关键目标、可变目标。当然也可以自行设计目标的层级结构。
然后，目标实施，通过目标的设计、调研、评审、分解、制定、协调和检查，最终使得目标达到预期效果。计划实施阶段主要由以下几个部分构成：
* 设计阶段：目标设计是指对目标的范围、内容、定义、目标方向、时间、难度、评估标准和预算进行综合考虑，以便使目标在规模上适应企业实际情况，并满足需求的要求。
* 选址阶段：选址是指目标选取的时间、地点、人员、资源、资金、设备等。计划的选址与企业的运营定位密切相关，不同的企业往往具有不同的选址方案。
* 执行阶段：执行计划就是以具体的方式实施计划，包括进度安排、人员配备、资源配给、设备投入等。计划实施至关重要，若没有有效的实施，就无法落实到相应的目标。
* 监测阶段：目标的实施过程还须得力于目标的监测手段。监测手段可以是内部的、外部的或是多种组合。通过目标的记录、跟踪、分析和总结，及时发现和解决问题，以保证目标的成功实现。
* 改进阶段：目标的实施过程也不能光靠自身，还需借助外部的支持，例如销售团队、财务部门、市场推广等，帮助企业根据实际情况调整计划，确保达成预期目标。
# 2.核心概念与联系
## 2.1 金字塔结构
目标管理是一个高度组织化的过程。为了方便目标管理和管理目标，引入了金字塔结构。它最初是由日本哲学家山本昌爾在20世纪50年代提出的，主要用于管理个人生活、工作、学习、人际关系、战争等事务。他把事务的层次分成四个象限——决定性层次、转化层次、组织层次和结果层次，并认为优先处理决定性层次，再向下转化处理转化层次，最后才是组织层次和结果层次。所以，按照目标管理的思想，企业的目标也应当按这种层次进行划分。比如，设置一系列的决策目标、任务目标、目标口号、条件目标、暂时的目标、长远目标等，并逐步缩小范围，直至达到组织目标或结果目标。目标越高，管理的复杂程度就越高。
## 2.2 “结构化”思维
目标管理是一个高度“结构化”的问题。采用金字塔结构，能让目标“层层剥开”，进而以更易理解的方式呈现出来。“结构化”思维是指应用问题分解法、迭代法、分类法等方法，将复杂问题分解成简单、易于管理的子问题，从而解决复杂问题。以目标管理为例，我们可以通过以下方式进行“结构化”思维：
* 设置目标分解计划：对整体目标进行分解，逐层明确目标的具体范围、措施、结果。
* 设立阶段目标：将目标细化到可量化的水平，每一阶段的目标应该具有清晰的指标、标准、时间表、依赖关系等。
* 设置决策制度：设立目标审核制度，由管理层批准每项目标的落实，避免出现意外行为或偏差。
* 梳理业务流程：业务活动的管理流程，如核准申请、审批程序、承诺结果、核算与报告等，应当进行合理设计，符合公司工作流程。
* 提高信息透明度：设立目标信息共享平台，分享、公示和反馈目标数据，有效提升协同效率，促进目标共识与一致性。
* 建立工作机制：设置目标驱动机制，组织团队合理分配目标工作，实现目标的顺利完成。
基于“结构化”思维，目标管理可以帮助企业降低管理复杂度、提升效率、节省资源，取得更好的结果。同时，目标管理还具有以下优点：
1. 目标管理能够有效地实现“层层剥开”和“更易理解”之间的平衡。
2. 采用“结构化”思维，能够简化目标管理过程，并将具体的实施计划和制度规范化，方便实施人员做好目标的落实。
3. 通过“结构化”思维，目标管理能够达成对目标的统一、准确、可控的管理，为企业提供更加清晰、科学、可控的管理目标。
## 2.3 五层目标管理理论
五层目标管理理论，是指将目标管理分为5个层次，分别是目标层次、策略层次、工程层次、规章层次、职能层次。该理论是美国麻省理工学院院士卡莱尔·戴维斯和英国教育专家丹尼尔·哈默于1991年发表的一项研究。它的目的在于帮助企业设计目标管理计划，通过合理规划，提升目标管理水平。其具体流程如下：
### 目标层次
目标层次：目标层次的目标是确立愿景、愿望或目标，目标的设置必须基于组织目标，并且是可衡量的。目标层次的作用是明确目标的愿景、愿望、关键要素，为后续的决策提供依据。
目标层次目标：设置目标时，需要关注以下几个方面：
* 是否有明确的目标描述
* 是否可衡量
* 有无可行性
* 目标与组织目标是否一致
* 目标落实是否可行
* 目标与其他目标的关系
* 是否存在替代方案
### 策略层次
策略层次：策略层次的目标是实现组织目标的具体措施，并促使组织有效执行策略。该层的目标须与组织目标保持一致。组织目标越明确，该层的目标就越明显。策略层次目标的功能有：
* 为组织目标提供可操作的方案或建议
* 沟通组织目标和执行策略之间的关系
* 在一定范围内衡量、沟通和修改策略
* 制定有效的组织实施机制
### 工程层次
工程层次：工程层次的目标是建立项目实施的各项制度，并确保项目成功的关键因素。工程层次目标应具有可复制性、可重复性和可验证性，并与组织目标相衔接。工程层次目标的功能有：
* 组织的各项制度是否完善、有效
* 项目执行中出现的问题是否得到妥善处理
* 成功的关键因素是否得到充分考虑
* 项目实施过程中是否出现过失误
### 规章层次
规章层次：规章层次的目标是确立可接受的管理制度，确保目标被正确落实。规章层次目标的功能有：
* 设定目标的指导思想、规则和制度
* 落实目标时是否符合公司规章制度
* 确保目标被准确执行
* 维护目标的连贯性和完整性
### 职能层次
职能层次：职能层次的目标是落实个人目标和个人责任，并促进个人和团队之间沟通、协调和合作。该层目标的功能有：
* 识别个人目标和个人责任
* 制定个人目标和个人责任的实施细则
* 明确个人目标和个人责任的优先次序
* 帮助个人完成个人目标和个人责任
