# 1.背景介绍



19世纪末，机械化工业蓬勃发展，智能机器和自动化设备极大的提高了生产效率、节约能源和消耗污染物等方面的问题，在电力、交通、通讯等应用领域均获得成功。然而，随着社会的进步，人们越来越依赖电力和通讯工具来完成日常生活的方方面面，电力行业的产值和利润都呈现下降趋势，而电力消费量的上升速度远不及通讯服务的增长速度。这种情况促使各国纷纷寻找新的能源来源，开发更加便宜廉价的电力产品。其中，美国占据先锋地位，在工程技术上取得突破性的进展，其所创造出的小型电动汽车已经成为当今最热门的新消费品之一。中国也曾经尝试过自主研发电动汽车，但是由于缺乏相应的基础研究成果，因此在后期的发展中受到了严重影响。所以，近年来，中国也在积极探索新的能源来源，并试图发展一种新型的可持续发电设备——火电或太阳能电池板。

20世纪70年代，国家发明了蒸汽机，当时还没有电动汽车。1976年，美国在科罗拉多州山谷的一座小镇建立了第一辆小型火电车（Model S），这是一种重量紧凑、高度精密的自用车辆，它主要用于产生和储存电能。在中国，广东、广西、江苏、浙江、河北、山西、安徽等省份都有相关研究和实践。到1980年代，日本、欧洲、韩国等国家也相继推出了自己的火电车产品。随着中国市场的不断扩张，相关研究人员纷纷进入这一领域，进行相关研究。

除了对火电车的研究外，中国也曾经参与了发展蓄水池、风力发电、核电、太阳能等新能源技术。其中，蓄水池发电法通过给水循环利用的蓄水池来发电，可获得较高的电价。1990年代初，郑州的湖泊被开发为蓄水池，这种发电方式就称为蓄水池法。风力发电，又称风电，可以利用风能来获得电力。1999年，中华民族伟大的领袖毛主席访问印度尼西亚时，雅诗瓦莱·卡尔瓦布·皮亚杰在那里建设了一座风力发电站。另外，太阳能发电也是一种绿色低碳发电的方式。1990年，中国发射了第一颗太阳能电池板，但由于缺乏足够的技术储备，至今仍处于测试阶段。但是，随着国家科技水平的发展，这种技术将会得到快速的发展。

21世纪初，随着人类的发展，传统的电能已无法满足需求。人们需要更多的能源来换取新能源的优势。2014年9月，美国白宫发布《2030年经济刺激计划》，计划在2030年前，用各种方式释放资本和技术，来促进发展新能源。其中包括引进世界上最大的500个颗星球，以及用火箭穿越太阳系运载人类。2015年6月，习近平总书记提出“走出去看看”，他说，我们正在抓住历史机遇，“向伟大的祖国——人类共同的回归表示永久庆幸”。基于以上原因，我国也在不同层面加大对电力、可再生能源领域的研发力度。

2019年，央视晚报发布了一个关于第三次工业革命的调查报告，从2001年到2018年，中国的5G、云计算、区块链等新技术的发展，已经对人类发展产生了深远影响。电动汽车也属于人类智能活动中的一个重要组成部分，在2025年，我们应该看到智能电动汽车与电动汽车并驾齐驱的未来。电动汽车发展的历程就是一个不断打磨的过程，要想真正解决一些零散的难题，需要我们花费更多的时间、精力、物质。因此，能不能以一篇文章的形式，深入浅出地讲解一下电动汽车的发展，让读者能够一窥电动汽车的发展脉络，从而更好地把握电动汽车这个急需的工具？
