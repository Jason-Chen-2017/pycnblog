# 1.背景介绍


概率论是应用统计学的一个分支学科，它研究的是随机事件发生的各种可能性及其相应的结果。在很多情况下，人们关心事件发生的可能性，而对已知结果的准确程度并不关心，因此概率论也被称作理性推断（rational inference）。概率论研究如何从样本数据中估计或预测出未知的数据值，它是计算机科学、工程学、经济学等其他各个领域的基础学科。但是，由于概率论的内容太广泛且深入，因而也容易令读者头晕目眩，所以需要一些简单易懂的描述性语言才能帮助读者快速理解。本文将介绍Python中实现概率论所需的基础知识。首先，我们需要知道什么是随机变量。
# 2.随机变量
## 2.1 概念
随机变量（random variable）是指一个函数，这个函数接收输入，输出是一个实数值或实向量。这种函数的输入集合通常被称作样本空间（sample space），而输出的实数值或实向量则称作样本点（sample point）。根据上下限的约束条件，随机变量可以分为离散型随机变量和连续型随机变量。假设随机变量X具有n个不同的取值，则它的样本空间由{x1, x2,..., xn}表示；如果随机变量X是连续型随机变量，那么它的样本空间就是R，其中R为实数集，如果X是离散型随机变量，那么它的样�空间就是Omega={x1, x2,..., xm}，其中m为自然数。
## 2.2 样本空间与分布函数
随机变量X的样本空间Omega定义了一个离散型的概率分布，分布函数（probability mass function, PMF）P(X=xi)为随机变量X可能取到各个值的概率。对于连续型随机变量，其概率密度函数（probability density function, PDF）p(x)代表了概率质量函数的积分形式。如果存在一个真正的随机变量X，那么X的样本空间和分布函数都是唯一的，它们都遵循两个重要的属性：
1. 单调性：若$a < b$，则$P(X \leq a) \leq P(X \leq b)$。
2. 规范性：$P(\Omega)=1$，$P(\varnothing)=0$。

## 2.3 随机变量的运算
### 2.3.1 组合律与指数化定律
对于随机变量X，Y和Z，如果$f_X: X\rightarrow Y$和$g_Y:Y\rightarrow Z$都是概率分布，那么$(g\circ f)_X=\frac{d}{dx}[g(f(x))]|_{x=X}$，$g(yf(x))=(y\cdot g(f(x)))$。$e^{f(X)}=\sum_{x\in\Omega} P(X=x)\cdot e^{f(x)}$。
### 2.3.2 折扣概率与期望
给定一个随机变量X，其折扣概率为$\frac{P(X=x)}{c_x}$,其中$c_x$为所有可能的值的总和。如果X的样本空间是连续型的，那么X的期望值为$\int_{\infty}^{-\infty} xf(x)dx$，如果X的样本空间是离散型的，那么X的期望值为$E[X]=\sum_{i=1}^{\infty}\omega_ix_i$。
