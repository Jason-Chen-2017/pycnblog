# 1.背景介绍


信息架构(IA)作为管理人员解决业务运营中信息收集、整理、存储、检索、应用、共享等过程的手段，而知识组织(KO)则是对人类知识进行分类、梳理、存储、应用的过程。由于互联网产品、服务的迅速发展、社会信息的日益丰富，使得传统的面向文档的目录结构不能很好地满足人们对快速获取信息的需求。信息架构与知识组织的理念是将所有信息从根本上分为四个层次，即业务层、主题层、内容层、关联层。在业务层，不同功能的相关数据、文档被分成不同的业务板块；在主题层，信息被划分为多个主题，如产品、客户、销售、市场营销等；在内容层，信息被精心制作，每个文档都经过了充分的编辑；在关联层，信息之间有着复杂的关系，如客户之间的关系、产品之间的关联等。
在信息架构与知识组织的理论指导下，互联网公司需要建立起信息的“金字塔”结构——业务层、主题层、内容层、关联层相互之间的交叉连接，信息得到有效整合、加工和呈现，提升人们的信息获取能力和信息质量。
# 2.核心概念与联系
## 2.1信息架构和知识组织
- **信息架构**（Information Architecture）：信息架构涉及到如何组织、编目、分类和管理企业的各种信息资源，以提高用户查询效率、降低维护成本、改善信息发现和利用效果。它是一种按照某种逻辑结构或模式组织、组织信息的方式。信息架构体系是为了解决信息的整理、获取、理解、沟通、传播、保存、分析和决策等问题，其目的是达到能够顺利、准确地获得所需信息的目的。
- **知识组织**（Knowledge Organization）：知识组织也称作知识建设，是指由人们接受教育、掌握技能、形成观点、创造知识并形成新事物的过程。知识组织是通过管理知识的结构与形式，加强个人能力培养、集体智慧开发、组织学习和社区交流等方式，来实现知识产出和社会进步。知识组织主要包括三个方面：结构（Structure），意义（Meaning），功能（Function）。结构是指知识组织中各个层次间存在怎样的逻辑联系，意义是指组织之后的结果是否可以产生价值，功能是指组织的方法、工具及手段，用于支持管理者的各种活动。

## 2.2金字塔结构的定义与特点
- **金字塔结构**（Pyramid Structure）：金字塔是一个具有三层结构的封闭型图形，第一层通常为最重要的信息和文档，第二层由较重要的信息和文档组成，第三层为较不重要的信息和文档。金字塔内的每一层都比上一层包含的信息更多。
- **特点**：
    - 层次性：按照顺序排列的信息都放在第i层，信息的数量随着层数的增加而减少，层与层之间的信息密度逐渐减小；
    - 继承性：越靠近顶层的节点，其含义越大，反映的范围也就越广泛；
    - 控制性：金字塔内的任何修改都会影响其他节点，金字塔的高度决定了信息的密度；
    - 透明性：所有的层次都是平行排列，直观易懂，每个层次中的内容、数量都可以清楚地看到。
