# 1.背景介绍


对于初创企业来说，往往处于起步阶段，需要自我激励、寻求外部资源支持才可能快速实现业务增长。虽然天时、地利、人和都对创业者有所帮助，但如何在法律、经济、管理等方面更有效地筹集资金、维持运营、获取外部资源仍然是一个难题。
互联网初创公司是一种特殊的创业模式，它既具有较强的社会影响力，又对市场有着独特的垄断性。因此，它们更容易受到国内外各种制约。如信息封锁、审查制度、严厉的反垄断法律、政策措施等。对于初创公司而言，如何能够合理运用这些制约因素并获得必要的资金来快速发展业务、扩大市占率，成为商界领袖，这些都是摆在他们面前的一课。

如何合理利用互联网初创公司的创新因素、积极参与互联网初创公司的股权计划就是一个重要的问题。股权计划是通过发行股份的方式进行的，初创公司可以通过发放股份的方式为自己创造出更多的价值。而且由于股权结构制度不同，发行股份也有相应的限制条件。例如，没有建立独立的董事会，或者担任股东的公司并不属于上市公司。在这一过程中，初创公司可以尽最大努力完成自己的目标，同时也要合理避开一些可能会影响其发展的弊端。因此，初创公司应当认真研究并选择适合自己的发行方式。

# 2.核心概念与联系
## 2.1 股权制度
股权制度是指股份权益所有权关系的法律规范，是企业管理中最基础的制度。一般情况下，企业都会设立董事会或类似的股东大会，由董事会成员共同决定企业的经营方向、管理层、财务状况等。股权制度规定了企业的各种权力和义务以及企业的控制关系。它主要包括：

1. 股东投票权:股东有投票的权力，即代表股东们通过决定企业发展方向的权利；
2. 实物转让:股权转让协议必须符合法律规定的条件，通常采用书面形式，必须保证双方的财产权利；
3. 优先股权:优先股权是指持有优先股份的企业除享有普通股份的所有权和相似的权利外，还拥有企业合并、收购等优先权；
4. 永续公司权利:永续公司是指企业不再发生分立破产的诉讼或其他债务危害企业生命健康时，终身持股的合法权利；
5. 股本固定:股本指的是企业持有的货币资产，股份比例是根据总股本计算得出的。

## 2.2 发行股份的方式
按照股权制度的分类，可分为三种类型：

1. 限售股份制(有限责任公司):在发行股份的时候就将全部股份都分发给公司股东，公司可以随时出售股份。这种制度保障了股东的投入及个人创造力，但可能带来高交易成本。目前仍然广泛应用于民营企业。
2. 预先募集股份制(股份有限公司):企业建立后，向股东募集一定数量的股份，但公司不得竞相买卖股份。这种制度的目的是为了防止有缺口时的投机行为，避免了股市过热。华尔街日报和雅虎中国都曾发生过这种现象。如今，很多大的科技企业都采用这种制度，如苹果、微软、谷歌等。
3. 社保基金股份制(非公开发行公司):这是一种非公开发行的制度。创始人按月缴纳固定金额的保险费，每年度分红一次。股份以社保基金为主体，除了股东个人分红外，还有很多与公司竞争的基金参与分红。由于股份无法直接转让，所以只能购买以实物形式出现的社保基金股份。华尔街日报称这类企业为“社保超级品牌”。

## 2.3 发行股份的限制条件
根据《证券法》第五十四条规定，发行股份的企业及其法人、其他组织不得做出下列行为：

1. 不得以直接或间接方式，利用股权差额的方式，或在发行股份的期限内以任何方式减少、增加已发行股份的数量；
2. 不得利用职务便利或其他不正当手段影响股权激励决策；
3. 不得向公司、基金或其他组织提供虚假的资料或提供错误的财务数据；
4. 不得违反证券法规定，故意避开或阻碍证券交易行为；
5. 不得用于组织或试图使企业屈服于政府要求的规定；
6. 不得提供不利于企业正常经营的资讯、招揽竞争者；
7. 不得提供广告或宣传信息，影响股权激励决策；
8. 不得含有蓄意毁坏或瘫痪财务记录、削弱经营信誉的文字、图片、图表或其它资料；
9. 不得提供损害企业名誉、声誉或利益的广告或宣传材料。

## 2.4 互联网初创公司的股权计划
互联网初创公司的发展离不开股权计划。根据《证券法》第三百八十四号文，互联网公司股权计划是指公司发行股份的方式，目的是为了满足股东的投资需求和分配股份，提高公司的盈利能力，促进企业的发展。股权计划的发行需遵循法律、行政法规和公司章程的规定，并遵守公司管理人员的职业道德和纪律。

互联网初创公司的股权计划包括A股、B股、H股、基金股份、社保基金股份等多种发行方式。其中，A股股权计划和B股股权计划都是公开交易的公司股权计划，其优点是获得股市大宗商品的价格优惠，但是需要遵守公开、透明、公平的原则。H股股权计划是美国著名的大型互联网公司股权计划，主要适用于大型互联网公司，比如阿里巴巴、腾讯、京东等。

社保基金股份制是非公开发行公司发行股份的一种制度。创始人按月缴纳固定金额的保险费，每年度分红一次。股份以社保基金为主体，除了股东个人分红外，还有很多与公司竞争的基金参与分红。由于股份无法直接转让，所以只能购买以实物形式出现的社保基金股份。社保基金股份制的优点是不需要像A股、B股股权计划那样在发行的时候就向股东募集股份，并且能够覆盖公司的运营成本，适合于成长性比较快的初创公司。
