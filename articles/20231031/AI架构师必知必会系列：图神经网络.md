# 1.背景介绍


　　图神经网络（Graph Neural Networks，GNN）近年来受到越来越多研究者的关注，在诸如社交网络、文本处理等领域具有极高的实用价值。早些时候，图神经网络曾被应用于生物学，现在则可以用来解决很多复杂的问题。

　　2020年以来，随着图数据流行的增加，基于图数据的相关任务也越来越多样化。传统的基于节点或子图的神经网络并不能很好地应对这种非同构性的输入，但借助GNN，我们可以通过学习图结构信息来进行特征提取、分类和预测。

　　GNN作为一个新的深度学习模型，其代表性和影响力还是需要时间去验证。本文将从以下几个方面对图神经网络进行全面的介绍：

　　1、绪论

　　　　　　　　第一章简要介绍了图神经网络的基本概念、方法、应用等。包括什么是图神经网络？为什么要用它？它能干什么？如何构建？

　　2、GCN

　　　　　　　　第二章介绍了第一个用于图神经网络的模型——图卷积网络(Graph Convolutional Network, GCN)。GCN是一个简单而有效的方法，能够在单个层次上捕获整个图的全局依赖关系。GCN通过利用邻居节点之间的相互作用来学习节点的表示，能够充分利用局部和全局的关系信息，取得更好的结果。

　　3、GAT

　　　　　　　　第三章介绍了另一个用于图神经网络的模型——注意力机制的图神经网络(Graph Attention Networks, GAT)。GAT的主要思想是在每个节点处引入注意力机制来选择重要的邻居节点，然后利用这些信息来更新当前节点的表示。通过这种方式，GAT可以在学习到全局依赖关系的同时，还能捕捉局部依赖关系。

　　4、GIN

　　　　　　　　第四章介绍了一种新的图神经网络模型——GIN——即图内自然语言理解。GIN的设计理念与GAT类似，也是利用邻居节点的信息来更新节点的表示。不同之处在于，GIN引入了图嵌入层来预训练节点的表示，使得模型更容易收敛。

　　5、GraphSAGE

　　　　　　　　第五章介绍了GraphSAGE的最新进展——一种连接不同粒度的聚合策略，通过构造不同的样本间的连接，来获取更多的全局依赖信息。GraphSAGE相比GCN、GAT等模型，有两个显著优点：

　　　　　　　　(1) 适用于异构图数据，比如，由异构网络生成的图数据；
　　　　　　　　(2) 无需预设邻居数量，能够发现不同规模的节点间的关系。

　　6、GraphUNet

　　　　　　　　第六章介绍了GraphUNet，这是一种新的图神经网络模型，与之前的模型相比，GraphUNet有一些独特的优点。首先，它使用多种连接策略，能够捕捉全局依赖信息和局部依赖信息，并且可以学习到不同类型节点之间的差异信息。其次，GraphUNet中的特征提取模块采用了变体残差网络(residual variant dense network)，可以实现更有效的特征学习，并保持准确率。

　　7、总结与展望

　　　　　　　　最后一章总结了本文所涵盖的内容，并给出了展望。除了这些主线，本文还希望引入更多其他的主题，比如，异步图神经网络、时间序列图神经网络、生成模型、GAN等等。作者也期待读者给予宝贵的意见建议。
