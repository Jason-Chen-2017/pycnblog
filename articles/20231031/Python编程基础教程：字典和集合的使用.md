# 1.背景介绍


## 什么是字典？
字典（Dictionary）是一个无序的键值对（Key-Value）映射数据类型，其中每个键都是唯一的，且值可以取任何形式的数据。它是一种高效的数据结构，可用于存储、查找和处理大量的数据。Python中的字典属于动态类型语言的一种内置数据类型。

## 为什么要用字典？
字典有以下几个优点：

1. 访问速度快：字典的索引方式是根据键快速定位值的过程，所以在字典中获取数据的时候，速度比列表或元组更快。
2. 查找方便：通过键可以很容易地找到对应的值。
3. 可变性：字典的值可以随时修改。
4. 有序性：字典中的元素会保持插入的顺序。

## 什么是集合？
集合（Set）是一个无序的元素序列，其中不允许重复元素。集合是一种高效的数据结构，可用于进行高级集合运算，比如交集、并集等。Python中的集合属于动态类型语言的一种内置数据类型。

## 为什么要用集合？
集合有以下几个特点：

1. 无序性：集合中的元素是无序的。
2. 不允许重复元素：集合中没有重复元素。
3. 支持集合运算：集合支持交集、并集、差集等多种集合运算。
4. 实用性：集合可用于高级的数据分析和数据处理。

# 2.核心概念与联系
## 字典的定义及其表示方法
字典由一系列键值对组成。每个键都对应一个值，键必须是不可变对象，例如数字、字符串、元组等不可变对象才能作为字典的键。值可以是任意类型的对象，也可以是另一个字典，甚至是自己定义的函数或者类。

字典的表示方法如下所示：

```python
{key1: value1, key2: value2,...}
```

注意：当某个键不存在时，对应的value值为None。

## 字典的方法
字典有一些常用的方法，包括：

1. dict()方法：创建字典，语法格式：`dict(zip(keys_list, values_list))`，例如：

   ```python
   d = {'name': 'Alice', 'age': 25, 'city': 'Beijing'}
   
   # 使用zip函数将两个列表合并到一起
   keys = ['name', 'age', 'city']
   values = ['Alice', 25, 'Beijing']
   
   # 创建字典
   new_d = dict(zip(keys, values))
   print(new_d) # {'name': 'Alice', 'age': 25, 'city': 'Beijing'}
   ```

2. clear()方法：删除字典中的所有键值对，语法格式：`my_dict.clear()`。

3. copy()方法：复制一个字典，语法格式：`my_dict.copy()`。

4. get()方法：获取指定键对应的值，如果该键不存在，返回默认值，语法格式：`my_dict.get(key)` 或 `my_dict.get(key, default=None)`。

5. items()方法：返回字典中的所有键值对元组，语法格式：`my_dict.items()`。

6. keys()方法：返回字典中的所有键，语法格式：`my_dict.keys()`。

7. pop()方法：删除字典中的指定的键值对，并返回对应的值，语法格式：`my_dict.pop(key[, default])`。

8. popitem()方法：随机删除字典中的一个键值对，并返回这个键值对的元组，语法格式：`my_dict.popitem()`。

9. setdefault()方法：如果字典中存在指定的键，则返回对应的值；如果不存在指定的键，则添加新的键值对，并返回默认值，语法格式：`my_dict.setdefault(key, default=None)`。

10. update()方法：将另外一个字典中的键值对更新到当前字典中，相同的键会被覆盖掉，语法格式：`my_dict.update(other_dict)`。

11. values()方法：返回字典中的所有值，语法格式：`my_dict.values()`。

## 集合的定义及其表示方法
集合是由一系列无序但唯一的元素组成的，并且元素不能够重复。集合可以使用花括号{}来表示，也可以使用set()来创建一个空集合。

集合的表示方法如下所示：

```python
{element1, element2,..., elementN}
```

## 集合的方法
集合也有一些常用的方法，包括：

1. add()方法：向集合中添加一个元素，语法格式：`my_set.add(element)`。

2. clear()方法：清空集合，语法格式：`my_set.clear()`。

3. copy()方法：复制一个集合，语法格式：`my_set.copy()`。

4. difference()方法：返回一个新集合，该集合包含了所有集合自身元素以及其他集合的元素，但是集合自身元素只出现一次，语法格式：`my_set.difference(*others)`。

5. intersection()方法：返回一个新集合，该集合包含了所有集合自身元素以及其他集合的元素，集合自身元素同时出现在两个集合中，语法格式：`my_set.intersection(*others)`。

6. isdisjoint()方法：判断两个集合是否不包含相同的元素，语法格式：`my_set.isdisjoint(other_set)`。

7. issubset()方法：判断一个集合是否是另外一个集合的子集，语法格式：`my_set.issubset(other_set)`。

8. issuperset()方法：判断一个集合是否是另外一个集合的超集，语法格式：`my_set.issuperset(other_set)`。

9. symmetric_difference()方法：返回一个新集合，该集合包含了所有集合自身元素，以及其他集合的元素，但是不同时出现在两者中，语法格式：`my_set.symmetric_difference(other_set)`。

10. union()方法：返回一个新集合，该集合包含了所有集合自身元素以及其他集合的元素，相同的元素只出现一次，语法格式：`my_set.union(*others)`。
