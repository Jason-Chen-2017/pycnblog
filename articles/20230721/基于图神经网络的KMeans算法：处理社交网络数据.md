# 2.基本概念术语说明
## 2.1 K-Means聚类算法
K-Means是一种无监督学习方法，用于对数据集中的样本点进行分组。其主要步骤如下：

1. 初始化k个中心点（质心）；
2. 计算每个样本点距离k个质心的距离，归属到最近的质心所对应的类别中；
3. 对每一类中的所有样本点重新计算新的质心；
4. 判断是否收敛，若收敛则停止，否则转至第二步；
5. 将每一个类别作为一个新的簇，每个簇代表了这个类别的样本点。

K-Means算法主要适合于凸的数据集。对于非凸的数据集，可能会陷入局部最小值，无法得到全局最优解。
## 2.2 图神经网络（Graph Neural Networks, GNNs)
图神经网络(Graph Neural Networks, GNNs)是一种基于图结构的数据表示学习模型。GNNs使用图结构的数据作为输入，通过学习图的全局信息来预测节点之间的关系。它可以有效地捕获节点间的空间关系、时间关系和邻居关系等全局信息。GNNs包含两大块组成部分：图卷积和图池化。
### 2.2.1 图卷积
图卷积是GNNs的核心模块之一。图卷积由两个部分构成：消息传递和特征更新。首先，图卷积利用邻居节点的信息，将各个节点的输入信号聚合到输出信号中。然后，将输出信号传播到整个图上，获得全局上下文信息。基于这一过程，图卷积能够提取出全局信息，并更新各个节点的表示。

图卷积在图像识别任务中被广泛应用。在图卷积网络(Graph Convolutional Network, GCN)中，节点嵌入向量的更新依赖于邻居节点的嵌入向量和归一化权重，而权重又依赖于节点间的相似性。这种更新方式鼓励节点保持稳定的特征空间分布，并在一定程度上抑制邻域内的噪声影响。
### 2.2.2 图池化
图池化也是GNNs的重要组成部分。它将图中不同区域的特征映射到同一向量空间中，使得不同区域间的信息可以通过连接向量进行传递。

图池化的目标是在节点的特征空间中进一步抽象化，以降低模型的计算开销。不同的池化层可根据具体需求选择，如顶点池化层只考虑单个节点的特征，边池化层考虑整个边缘，面池化层考虑整个面，而三维池化层考虑完整体。
## 2.3 图K-Means算法
图K-Means算法是将图数据转换为矩阵形式后，采用K-Means算法对矩阵进行聚类。具体步骤如下：

1. 把图转化为邻接矩阵；
2. 使用K-Means算法对邻接矩阵进行聚类；
3. 从聚类结果中得到节点分类及相应的聚类中心，即每个类的节点个数及其位置；
4. 根据聚类中心生成最终的图嵌入向量；

图K-Means算法的特点是：

1. 可以利用图结构进行聚类，对社交网络数据尤其有效；
2. 不需要任何训练数据，直接基于原始图数据进行聚类，实现快速、简单和易于理解；
3. 适用于具有不同结构和规模的图数据集，并能够较好地保留全局信息。
