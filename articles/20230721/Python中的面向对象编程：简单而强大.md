## Python简介
Python 是一种面向对象的高级语言，由 Guido van Rossum 于1991年在荷兰创建，并于2001年底成为开放源代码项目。它具有可移植性、丰富的标准库和第三方扩展库，可以应用在许多领域中，包括Web开发、系统脚本、网络编程、科学计算等。Python 的简单易用特性，能够轻松地将项目完成，同时又保证了其速度和效率。
## 什么是面向对象编程？
面向对象编程 (Object-Oriented Programming，OOP) 是一种计算机编程方法，它将代码中的数据和功能组织成一个个“对象”，每个对象都有自己的状态（数据）和行为（函数），可以通过消息传递的方式进行通信和协作。面向对象编程通过封装、继承、多态等特性实现对代码的抽象和模块化，提高代码的可维护性、复用性和扩展性。由于 OOP 把问题分解成各个对象之间的交互关系，使得程序的结构更加清晰、更加易读、更容易维护和扩展，所以越来越受到工程师们的青睐。
## 为何要学习面向对象编程？
Python 作为一种面向对象的高级语言，拥有众多优秀的特性，比如强大的动态类型机制，易用的语法和包管理工具 pip，快速易学的学习曲线，以及丰富的社区资源和开源库支持。相对于其他编程语言来说，Python 在处理大型项目时的可维护性和可拓展性更好，因此被越来越多的工程师们所采用。另一方面，由于 Python 具有极高的运行速度，并且具有简单、易用的语法，使得它被广泛用于各种领域的软件开发中，包括 Web 开发、机器学习、图像处理、游戏开发等。因此，学习面向对象编程，不仅可以掌握 Python 优秀的特性，还能利用其解决实际问题，提升个人能力，增加职业竞争力。
# 2.基本概念术语说明
## 对象、类、实例
### 对象
对象是面向对象编程中的重要概念。我们通常将某个客观事物（如房子、树、飞机、鸟类等）视为一个对象，例如，火车是一个可以移动的对象；一辆汽车就是一个对象，它既有位置信息，也有速度、方向等描述它的特征；一段字符串也可以看做是一个对象，因为字符串的功能包括大小写转换、连接、切割、查找等。总之，对象是对现实世界中事物的一个抽象、虚拟的形象，它具有一个或多个属性和行为，可以执行这些行为。
### 类
类是用来创建对象的蓝图或者模板，它定义了一个对象应该具有哪些属性和行为，包含的数据和函数。每当我们想要创建一个新的对象时，都需要根据这个类创建一个新实例，也就是该类的一个具体的实现。类也可以从已有的类继承，获得其父类的所有属性和行为，再添加或修改新的属性和行为。
### 实例
实例是由类创建出来的对象，它是内存中真正存在的对象。实例是类的具体实现，是在运行期间创建的。每个实例都有自己的属性值，可以调用其的方法来操纵其内部的数据。
## 属性、方法
### 属性
属性是指类的成员变量，它表示对象的状态，也就是在不同情况下会变化的值。属性是可以直接访问的，不需要通过任何方法获取。属性一般都是私有的，只能通过提供相应的接口方法来访问。
```python
class Car:
    def __init__(self):
        self.__color = "" # private attribute

    def set_color(self, color):
        self.__color = color
    
    def get_color(self):
        return self.__color

my_car = Car()
my_car.set_color("red")
print(my_car.get_color()) # output: red
```
上面例子中的 `__color` 是私有属性，外部无法直接访问或修改。但仍然可以使用 `set_color()` 和 `get_color()` 方法来设置和读取私有属性。注意：私有属性前面加上两个下划线代表私有属性。
### 方法
方法是指类可以响应的一组动作，它负责实现对象的逻辑功能。在 Python 中，方法是被绑定到实例上的函数，它们可以直接访问实例的属性。在方法内部，可以访问类的属性和方法，也可以调用其他的方法。
```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
        
    def study(self):
        print("%s is studying..." % self.name)
        
stu1 = Student("Tom", 20)
stu1.study() # output: Tom is studying...
```
上面例子中的 `study()` 方法就是一个实例方法。当调用 `stu1.study()` 时，就会执行这个实例方法，此时方法内部就可以访问 `name` 和 `age` 这两个属性。
