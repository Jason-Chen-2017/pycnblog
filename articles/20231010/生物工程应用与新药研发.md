# 1.背景介绍


“9.生物工程应用与新药研发”系列的文章将会分享一些关于新药开发及应用的科研工作。其中“9.1 新药生物工程实验”文章将介绍新药生物工程实验的相关知识、方法、操作等方面，重点介绍了在新药研发过程中如何进行实验设计和分析，提升产品的有效性和安全性。而“9.2 新药结构设计”文章将侧重于新药结构的设计方法，包括设计原理、流程和工具等，为新药设计提供一个更加可靠的依据。“9.3 新药分子科技”文章则着重于了解新药分子化学特性的影响，给予新药合适的分子体系和活性化学修饰，使其具有更好的药效。

新药研发是一个充满激情、充满挑战的任务。因此，需要一支高度组织化、创新型的团队才能有效地完成这一工作。本文仅就这一领域的一个例子，介绍一种新型的面向辅助药物（Chemo-Aid）的研发模式。

# 2.核心概念与联系
辅助药物即为用特定生物制剂、诱导诊断或疗法辅助疾病或疾病康复的方法治疗疾病的药物。辅助药物可用于对抗抗体免疫系统、防治化脓性皮肤病、缓解创伤后的休克恢复，以及降低风险并改善生活质量等方面。但是，许多人认为，只要掌握正确的生物工程原理和手段，就能开发出精妙的辅助药物。为了实现这一目标，建立起严格的制药规范，制造出高品质、安全性高、经济性好的辅助药物才是关键。

近年来，随着人工智能、生物芯片、新型药物材料、3D打印技术的出现，生物工程技术在新药研发中的作用越来越重要。现有的手段和方法已经无法满足需求，新的研究领域涌现出来，例如，生物信息学，生物医学和生物化学技术以及生物工程学科。根据功能的不同，辅助药物可分为基于体外机制的辅助药物、基于靶向微生物的辅助药物、以及基于核苷酸化合物的辅助药物三种类型。

采用基于体外机制的辅助药物作为研究对象，主要研究基于体外机制的辅助药物的新型开发、评价、检测及改进，并制定出符合要求的管理体系，使之能够起到预防、治疗及康复的作用。基于此，九月的“9.2 新药结构设计”文章，将介绍一些基于体外机制的辅助药物的制药原理及原理制备方法，并回顾目前有关基于体外机制的辅助药物的发展动态，以及国内外研究人员的最新进展。

采用基于靶向微生物的辅助药物作为研究对象，研究目标主要是开发新型靶向微生物药物，通过改良基因编码或抑制微生物群落形成路径依赖，改变微生物的生态平衡，激发其分子调控能力，并将其转化为化学物质，从而达到治疗或减轻疾病负面的目的。中国国家自然科学基金委的张维迎老师和王云鹏博士分别代表中国、美国团队参与这个领域的研究。

采用基于核苷酸化合物的辅助药物作为研究对象，由国家教委牵头筹建的“核苷酸药物技术开发中心”、“核苷酸药物利用中心”、“核苷酸药物识别平台”以及“核苷酸药物预测中心”等多中心开展研究。其中，“核苷酸药物技术开发中心”专门从事核苷酸药物研发、试验及商业化等方面，主要研究核苷酸药物的设计、化学组装、验证、发展、应用等领域。另外，联邦德国康希诺大学负责人卡尔·库茨斯教授指导的“核苷酸药物预测中心”则致力于制定出针对复杂多变、高度敏感、反应迅速的核苷酸药物的预测方法，并推广应用至核酸药物、DNA致癌药物等领域。

综上所述，通过以上三个方向的探索，对辅助药物的研发和应用，将会有一系列的突破，助力推动我国辅助药物的研发与应用能力不断提升。
