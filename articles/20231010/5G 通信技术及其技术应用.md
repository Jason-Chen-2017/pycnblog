# 1.背景介绍


# 5G（第五代高频无线）是一种新兴的通信技术，在近期将引领移动通信领域，但目前尚处于早期开发阶段。5G通信系统结构复杂、技术难度高、规模巨大，但实际部署上却发现难度甚至超过了2G。因此，关于5G技术的研发及其在通信领域的应用实践也成为热点话题。

2017年3月，美国国务院宣布推出“超级5G”计划，标志着5G技术突破4G时代，带来全新的宽带速度，提升系统性能，为社会带来更好的生活品质。为了能够更好地认识和掌握5G技术，做到关键技术了解、应用创新、政策制定等方面，将会促进5G技术的成熟和应用。

2019年6月，国际电联发布《2019中国网络安全报告》，指出2019年前十个月，5G技术部署规模已经从10亿台增长到了20亿台，每天新增的终端数量达到了3万多台。如果忽略掉中国方面的数字孤岛效应，仅以人口计，这一数据足够显示5G技术的全球影响力。

随着5G技术的迅猛发展，也带来了很多挑战，例如无线接入的速率提升、设备兼容性难题、芯片短板问题、安全问题等。

那么5G技术的核心问题、优势、局限及其未来的发展方向，以及如何应用5G技术帮助解决这些问题，都是值得关注的课题。

# 2.核心概念与联系
## 2.1 概念
### 2.1.1 基站、基带、干扰、信道共享
#### 基站
基站是5G通信中用于传输数据的站点，它由多个小型的卫星组成，在一定的范围内覆盖城市或省份，通过多条无线链路连接到用户终端。基站的作用主要是对接不同信号源之间的信号，并把信号传送到下一个目标点。

#### 基带
基带业务是指将数字信号转换成模拟信号，发送到UE中进行处理的过程。基带业务是在无线信道上完成的，可以看作是无线通信中的基本功能。

#### 干扰
干扰是指5G中存在的非用户数据，一般包括基站和终端本身产生的功耗、杂散能量、射频偏移、站内环境噪声等。干扰的存在会导致5G通信的不确定性，降低5G通信的可靠性和稳定性。

#### 信道共享
信道共享是指无线用户共享同一条无线信道。通过信道共享，可以实现多用户之间的通信和资源共享，增加通信的吞吐量和数据利用率。

### 2.1.2 LTE、NR、OFDM、TD-SCDMA
#### LTE
LTE(Long-Term Evolution)，中文名称长距 evolution-cell融合技术，是一种基于物理层的新一代的蜂窝移动通信技术标准。它使用4G频段，采用了高密度SMF集中器，使得4G频谱能有效利用。LTE的主要特点是采用了新的小区覆盖模式，可以支持任意小区的拓扑结构。LTE的高速率可以达到30Mbps，同时还兼顾了速度与价格。

#### NR
NR(New Radio)，即新型无线电，属于第三代移动通信技术标准，其特点是高灵敏度、高频率带宽、极低时延、无线功耗低等。华为自主研发的20类单核高通芯片型号，每秒可处理数据传输量可达4.3Tb/s。

#### OFDM
OFDM(Orthogonal Frequency Division Multiplexing)，正交频分复用技术，是一种可以有效提高带宽利用率的无线通信技术。它允许接收方同时接收多个子载波，提高系统整体带宽利用率。OFDM技术在实现用户间的数据共享时，需要采用分布式技术，并同时提供多种服务质量保证。

#### TDM-SCDMA
TDM-SCDMA(Time Division Multiple Access-Short Cell Digital Mobile Phone Communication)，中文名称时隙多址-短基站数字蜂窝移动通信，是一种集中调度、时隙划分多址方法的无线通信技术标准。这种方式可以更好地适应通信需求，并且具有较高的抗干扰能力。

## 2.2 研究对象与相关技术
5G的主要研究对象主要包括物理层、信息层、应用层。
### 2.2.1 物理层
物理层的研究主要关注5G基站、终端、传输介质、射频与雷达技术等物理层相关的问题。

#### 5G基站
5G基站的研究旨在制定基站的基础设施要求、布局设计、系统集成、运营管理、评价指标和优化方案等方面。5G基站的性能指标主要包括发射功率、覆盖范围、空中距离、上传速率、数据速率、用户数等。

#### 5G终端
5G终端的研究主要是关注终端的硬件、软件、驱动、系统架构、接口协议、应用系统、编程框架、QoS机制、安全机制、操作系统等。终端性能指标主要包括接收速率、基带速率、最大接入速率、终端类型、处理能力、存储容量、待机时间等。

#### 传输介质
传输介质的研究旨在探索新的传输媒介、通信技术，如光纤、微波、红外线、量子等等，并建立5G基站、终端与其他设备的互联互通机制。

#### 射频与雷达技术
射频与雷达技术是5G通信中涉及到的最复杂的技术。5G需要向现有的通信系统、基础设施、设备等投入巨大的投入，同时使用高科技的射频技术、雷达技术等，将传播路径进行精确的测量与建模。

### 2.2.2 信息层
信息层的研究主要关注5G通信所需的信息处理技术、编码与调制技术、通信网络管理技术、智能化系统等方面。

#### 信息处理技术
信息处理技术的研究旨在研制新的信息处理单元、编码方式、调制方式、解调方式等，来提升信息的处理性能和准确性。其中编码技术主要包括LDPC码、海赛尔编码等；调制技术主要包括QAM、OFDM、MU-MIMO等；解调技术主要包括OFDM、LLRs等。

#### 通信网络管理技术
通信网络管理技术的研究是希望能够建立统一的网络管理系统，对5G系统进行实时监控、预警、诊断、控制。同时，还要研究针对5G通信的特殊网络管理问题，如异构网络、动态变化的网络拓扑、高吞吐量的多流场景等。

#### 智能化系统
智能化系统的研究旨在开发新型的计算平台、机器学习算法、生物识别技术、无线传感网络等，来提升5G通信的系统性能、用户体验、经济效益。

### 2.2.3 应用层
应用层的研究主要关注5G通信中应用的各项技术，如视频会议、无人驾驶、增强现实、网络游戏、大数据分析等。

## 2.3 研究内容与意义
5G的研究内容非常丰富，涉及到通信系统的设计、架构、关键技术的研发、测试、验证、应用等方面，但研究的重心在基站、终端、无线信道、射频技术等方面。由于5G技术的复杂性，需要解决众多工程问题，因此需要高度的专业知识、经验积累和行业视野。

对于企业和个人来说，5G的研究主要的意义就是可以提升通信系统的传输速度、带宽、可靠性、数据利用率，降低通信成本，让人们的生活变得更加便捷舒适。但5G的应用也面临着许多挑战，需要进一步深入研究和实践。
