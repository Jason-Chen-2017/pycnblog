# 1.背景介绍


## 概述
1990年代初，随着科技飞速发展，人们生活水平急剧提高，信息化建设速度也极其迅速。移动互联网、社交网络、电子商务等新兴应用逐渐普及，给日常生活带来便捷。由于互联网的信息量极大，使得用户对各类产品功能和服务的需求量激增。而互联网上的用户界面设计与交互体验一直处于上升期。交互设计师作为从事UI设计领域的一线技术人员，需要具备良好的沟通能力和理解能力，能够理解并分析用户需求，提供符合用户预期的界面设计。同时还要有创造力和团队合作精神，不断迭代优化产品，将产品推向市场。

2017年，中国互联网行业经历了百花齐放的局面，互联网公司纷纷推出了丰富多彩的产品和服务，如今，“互联网+”成为主流词汇，消费者对于互联网产品的使用更加依赖于移动端。人们越来越注重品牌形象，希望用美感打动消费者，所以，品牌设计是最突出的设计环节。当下，无论是电商网站、零售平台还是手机App都存在非常多的品牌设计。例如阿里巴巴、京东、网易云音乐、腾讯视频等互联网公司都会进行品牌设计。

2018年底，我国线下校园迎来了第一次校园春季招聘，此次校园招聘主要面向全国各高校学生。线下招聘频道得到广泛关注，许多招聘机构纷纷筹划线下校园招聘活动，因此，招聘的方向也是广泛的。例如，某校招聘计算机相关专业的硕士研究生，或者招聘互联网企业的产品经理，本文将围绕“交互设计师”这个岗位开展分享。

在学生的视角来看，“交互设计师”的主要工作职责就是协助业务部门、产品经理或项目经理设计出符合用户心意且具有吸引力的产品界面、页面。在校招过程中，一些初级技术岗位可能会问到关于这一岗位的要求。比如，“请您描述一下您参与设计的产品，并阐明其功能、特点、目标用户、用户场景及主要界面设计方案？”，“希望您能说清楚您的设计理念？”“请您简要介绍您认为比较独特的创意？”。

# 2.核心概念与联系
## UI设计概述
UI（User Interface，用户界面）即用户接口，它是指计算机设备、软件和硬件之间的一种数据交互方式。它由图形用户界面、命令行界面、基于文本的界面三种形式组成。通过简单有效的界面设计和反馈机制，可以让人们直观、快速地获取、理解和操作信息，促进交流沟通，提高工作效率。

UI设计不是一门独立的课程，而是一个涵盖众多学科的跨界领域。它涉及多种艺术素养、视觉修辞、语言表达、交互手法、动效设计、美学设计、图形设计、Web设计等多个领域。根据UI设计师的专业程度和工作经验，分为初级、中级、高级三个级别。

初级UI设计师：入门级，一般只掌握基本的构想设计技巧，但对于复杂界面设计来说，仍然存在一定难度；
中级UI设计师：在初级基础上较为熟练，能够使用视觉元素进行设计，完成专业的产品设计；
高级UI设计师：能够运用多种设计手段进行优化，创建独具魅力的设计风格。

UI设计师除了负责制作产品的外观、布局、字体和色彩外，还需要对产品的可用性、可访问性、兼容性等方面进行测试，确保产品满足用户的需求。另外，UI设计师还需要和交互工程师、视觉设计师、用户研究者等一同参与项目的研发流程，配合后期的开发工作。

## UX设计概述
UX（User Experience，用户体验）是指产品或服务对用户的影响力、满意度以及满意程度。它的主要目的是为了改善产品或服务的使用体验，提供更加美好的使用环境。

UX设计师主要负责产品或服务的整体策划、设计、开发和营销。他通常负责将产品或服务的运营模式与用户需求结合起来，通过产品的设计、流程、内容、页面等，提升产品的可用性、可信度和用户体验。

UX设计师在产品设计时，应该考虑到用户的操作习惯、喜好、意识、认知、情感等方面的因素。这些因素会影响到用户的选择和使用行为，影响用户对产品或服务的满意度。在设计产品或服务时，必须遵循以下六个准则：
- 真实性原则：尽可能清晰准确地传达产品的真正含义和价值。
- 清晰性原则：设计出简单、容易理解的界面，避免过多的元素和复杂的设计。
- 效率性原则：优先考虑用户的实际操作时间，减少等待时间。
- 舒适性原则：降低用户的使用难度，使用户操作起来轻松自然。
- 可用性原则：确保所有人都可以使用产品或服务，符合用户的不同需求。
- 用户体验原则：充分利用各种设备及浏览器，提升用户的使用体验。

除了产品或服务的视觉效果外，还应注意界面上的文字，帮助用户理解产品或服务的含义。界面上的文字需要做到专业、简洁、清晰、易懂、一致。

## 交互设计与UI设计的关系
交互设计和UI设计之间的界限十分模糊，它们之间存在着相互依赖、相互补充的关系。

UI设计师首先需要明白用户需求，定义产品功能、体验目标和交互元素。他通过分析用户行为、浏览习惯和产品结构、需求等方面，制作出符合用户期望的界面。

交互设计师则需要通过逻辑思维、设计技巧、品牌识别、信息整理、导航规划、体验诊断等方法，来实现用户的直接需求。他首先需要在视觉和空间层面上实现产品的美感和可用性，然后转向信息层面、语音交互、触摸屏等技术手段，通过与用户进行真实体验的方式，实现更丰富、更深入、更富有层次的内容。

因此，交互设计师所扮演的角色，不仅只是UI设计师的补充角色，更是整个UI设计过程中的重要角色。它融合了人机交互、视觉艺术、文字表现、互动体验等多个学科，并且注重用数据、统计分析和用户研究等方式，进行专业的产品设计。
