# 2.核心概念与联系
## 2.1 Python循环机制
Python中有两种循环机制：

1. for循环
2. while循环

### 2.1.1 for循环
for循环用于遍历可迭代对象(iterable object)（如列表、元组等）中的每一个元素。其语法形式如下：

```python
for item in iterable:
    # do something with the item here
```

其中item表示每次遍历的元素值，iterable可以是一个序列类型或其他可迭代对象，如字符串，range函数生成器等。一般情况下，for循环会遍历iterable中的所有元素一次，直到全部元素都被遍历完毕，但也可以在for循环体内加入判断条件，使得循环能停止或跳过某些元素。 

### 2.1.2 while循环
while循环与for循环相反，它用来重复执行一个语句块，只要某个表达式（condition）成立就一直循环，直到条件不成立才结束。其语法形式如下：

```python
while condition:
    # do something repeatedly here until the condition is false
```

其中condition为循环判断条件，如果该表达式结果为True则循环继续运行，否则结束。

两者之间的区别主要有以下几点：

- for循环：适合遍历可迭代对象中的每个元素，完成特定任务；
- while循环：适合根据条件重复执行某段代码，直到满足条件才退出循环。

## 2.2 break和continue语句
### 2.2.1 break语句
break语句允许在for或while循环中提前结束循环，即跳出循环体。语法形式如下：

```python
break
```

示例代码如下：

```python
for i in range(5):
    if i == 2:
        break
    print(i)
```

输出结果为：

```python
0
1
```

由于执行了break语句，导致循环中断，因此后续的print()语句不会再执行。

### 2.2.2 continue语句
continue语句允许在for或while循环体中，跳过当前的这个迭代过程，直接进行下一次循环。语法形式如下：

```python
continue
```

示例代码如下：

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

输出结果为：

```python
0
1
3
4
```

由于执行了continue语句，导致当前循环体中的打印语句被忽略，而直接进入下一次循环。

## 2.3 循环控制语句break和continue的作用及特点
break和continue语句是非常重要的两个控制语句。这两个语句可以让程序员精细化地控制循环的流程，从而实现一些特殊功能。它们的作用包括：

1. 提前结束循环：break语句能够结束整个循环，break后的代码不会再执行；
2. 跳过某次迭代：continue语句能够跳过某一次迭代，从而实现某种特殊的控制。

### 2.3.1 break的作用
break语句的作用是提前结束循环，可以用在多层嵌套循环中，用于退出外部循环或者跳出循环体，避免后面的代码执行，节省时间和空间。

### 2.3.2 continue的作用
continue语句的作用是在for循环中，跳过当前的迭代过程，直接进行下一次循环。可以用在有条件地跳过某些项的情况，比如找出数组中的非零元素。

### 2.3.3 使用注意事项
break和continue语句在某些时候可能被滥用，但是用好了它们还是很有用的，正确使用它们能够提升程序的效率，增强程序的健壮性。因此，在编写程序时，应当注意避免滥用break和continue语句。下面是一些常见错误的用法，以及如何正确使用break和continue：

1. 没有写break或continue语句：忘记了写循环控制语句，导致循环没有正常结束。这种情况可以通过增加循环的终止条件来解决，或者使用异常机制来处理循环异常终止的问题。
2. 在if/else语句中使用break：不要在if/else语句中使用break语句，因为这样做只能跳出一层循环，而不是整个循环。正确的方法应该是在循环内部判断是否跳出循环。
3. 使用continue语句时不当：continue语句仅仅用于跳过循环中的某一次迭代，不能用于调节循环变量的步长。正确的方法应该是在循环中使用if/else语句实现循环变量的跳跃。
