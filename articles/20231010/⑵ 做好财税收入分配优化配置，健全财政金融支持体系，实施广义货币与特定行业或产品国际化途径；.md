# 1.背景介绍


从2018年开始，随着中国经济的快速发展，世界各国纷纷关注中国的财税状况和金融支持政策。对财税收入分配进行优化配置，确保企业税负可控、个人财富增值，是国内外政策制定者、主流经济学家等多方面因素综合考虑而做出的重大决策。财税收入分配优化配置还可以带动消费和社会经济结构优化，助力中国经济持续高速发展。
早在2016年1月，习近平总书记在中央纪委监察部举办的“四会一课”上就提出了关于加强国际金融合作的系列重点议题，其中就包括着重推进广义货币机制的发展和健全财政金融支持体系。随后，习近平在中央经济工作会议上的讲话中又进一步阐述了这一系列领域的重要性。
2019年年初，习近平总书记再次提出了构建完整的财政金融支持体系，必须兼顾长期解决国内经济结构性问题，加快转变过渡时期的就业形态，提升人民生活水平，解决人民币汇率升值问题等，他认为，“这项工作有重要意义”。
为了实现更加充分的财税收入分配优化配置，习近平总书闻命在中央纪委监察部召开的金融工作座谈会上阐述了具体目标。他要求，要增强战略定力，抓紧实施优化财税收入分配配置方案，切实保障企业税负可控、个人财富增值，努力创造更多就业机会，增加有效外资投资，提高广义货币和特定行业或产品的国际化水平，加强对贸易战、中国-美国关系、中东欧国家关系等问题的政治经济指导，推动金融机构抓好科技创新、对外开放，充分发挥人民币国际化功能。
此外，在所谓的“三会三场”学习开班式上，习近平总书记再次强调要深刻把握时代特征，抓住历史机遇，牢牢掌握改革主动权，构建中国特色的发展蓝图。一系列务实主张、措施配套、示范引领等，将由党的十九大报告、二〇二五年远景目标纲要等一系列文件共同落地、践行。
2.核心概念与联系
### 1）广义货币机制
广义货币机制是一种金融市场机制，它不是单纯地依赖某种特定货币作为主导货币，而是通过货币的多元化安排来增加资本市场的活力、降低交易成本、提高资本的流动性和使用效率。广义货币的建立有利于促进资本的自由流动，并消除对货币的主导地位。当前世界主要宗教信仰仍然处于严格限制状态，广义货币机制能有效缓解金融危机带来的经济社会影响。
广义货币机制的关键因素是多种货币之间的相互制衡作用，即价值的形成过程是由多种货币间的比价决定的。这种比价不同于一般的货币买卖，更倾向于由人类选择的内在逻辑而非直接市场价格决定。比如，黄金本身具有比其他商品更高的规律性，因此被视为独立的主权货币。白银、石油、天然气等金属性资产则在广义货币框架下更易流通和利用。人们可以选择自己的储蓄方式，也可以不受其他货币对其货币价值的影响。
### 2）财政金融支持体系
财政金融支持体系是一个综合性的政策保障机制，旨在满足国内和国际金融需求、促进经济发展。目前，我国财政金融支持体系既包括政府财政提供的基本支出，也包括国有部门和地方各级政府提供的专项补贴。国际上主要依靠的是联邦基金（联邦基金理事会）、国际货币基金组织（IMF）、美联储等金融机构提供资金。财政金融支持体系还有其他一些具体细节，例如，国债发行与管理，中央银行利率的设定和市场准入控制，企业金融资本市场的开发，实体经济金融机构的支持等。
国债发行与管理是指中央银行根据国债市场运行情况，制定适当的国债发行量和管理规则，帮助企业购买国债；利率的设定则是通过降低国债市场波动率来降低银行融资成本。市场准入控制是指设置一系列条件限制那些无效或不符合条件的金融机构和个人的进入。目前，我国居民生活金融（以商业保险为代表的商业贷款）、经营性贷款（如消费贷款、车贷款、房贷款、学生贷款等）、消费券都受到市场准入控制，这有效避免了诈骗、贩毒等违法活动。实体经济金融机构的支持则侧重于培育和完善中央银行与地方银行的业务模式，促进实体经济与金融机构的良性循环。
### 3）国际货币基金组织
国际货币基金组织（IMF）是国际货币基金的管理机构之一。IMF的任务是协调国际货币市场，促进跨国资金的流动性，并提供全球范围内货币政策支持。IMF的官方网站上列举了它的七个主要任务：
* 把握货币供应的统一管理
* 发展创新型跨境支付手段
* 提升金融机构的透明度
* 鼓励国际金融合作
* 促进国际货币市场的健康发展
* 涵盖发达经济体内部的货币发行
* 为国际组织提供灵活的金融服务
IMF的运作模式一般包括四个阶段：
第一阶段：基础设施建设
第二阶段：货币政策发展
第三阶段：组织发展和管理升级
第四阶段：全球金融合作平台
第一阶段的目标是建立起有效的金融服务平台，包括建立国际收支账户、信息公开共享、清算网络、技术支持等基础设施，为金融机构提供技术支撑、资金支持和服务。第二阶段的目标是加强对国际货币市场的研究和政策制定，扩大国际贸易和金融合作，完善新兴市场的货币政策，增加支持跨境支付手段的力度。第三阶段的目标是加强组织建设和管理升级，尤其是建立跨部门的交流合作机制，激发企业家精神和创新能力，支持区域和行业的金融发展。第四阶段的目标是形成一个全球金融合作平台，鼓励所有国际金融机构进行合作，同时增强全球金融市场的竞争力。
### 4）政府财政金融作用
政府财政金融作用是指中央政府在为国内经济发展提供基本社会保障的同时，对财政收入予以补贴。中央政府在计划生育、医疗卫生、教育、失业救济、税收优惠等方面均采取政府财政金融作用。这些政策的实施有助于扩大财政收入，并减少不必要的财政负担。另外，广义货币机制与政府财政金融作用的结合，让中央政府有机可乘，缩小国内财政赤字。
