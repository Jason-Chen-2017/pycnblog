# 1.背景介绍



“暴风雨”是新冠肺炎疫情期间互联网上热议的话题。在这个风雨飘摇、狂轰滥炸的时刻，我们应该如何把握住机遇，实现民族复兴？我们今天要讨论的就是这一个中心议题——“9.“暴风雨”之后的中国和平崛起”。

“9.“暴风雨”之后的中国和平崛起”指的是对中国政府所采取的一系列措施的回顾和总结，从中可以看出，无论是坚守共产党阵营的政治方针，还是转型国家主导战略，抑或依靠市场经济资源改善经济状况，都具有积极意义，而且并非空穴来风，还有待广泛深入地研究和探索。然而，我们也需要注意到，由于“新冠肺炎”疫情影响面广、持续时间长，各方利益方在此次危机中都受到巨大的冲击。因此，如何平衡不同方面的利益，促进合作共赢，是一个值得关注的问题。

随着疫情的结束，国际社会已经充分认识到了美国的角色在国际舞台上的重要作用。随着美国开启全球反恐战争，世界各国纷纷声称愿意承担起防御任务。在中国看来，也应当考虑到美国作为人类基石的重要性，也期待与美国的合作。

但是，我们也不能忽视政府作为公权力机构的责任和职能。中美关系发展至今，长久以来，双方经贸合作成果丰富，也互相依赖。虽然疫情使得一些国家出现经济衰退，但中国仍然保持了稳定的政治局面。这显示出，作为公共政策制定者，我们应当看到政府作为整体性的力量，依然存在对公众利益的维护和保障。

此外，在当前百年变局背景下，仍然有很多人不了解或不关心中国传统文化及其价值观。面对新世纪的到来，值得我们认真借鉴、学习的还有中国古代政治哲学、文化传统和精神气质。这也许能够为我们提供更有力的政策建议。

综上所述，“9.“暴风雨”之后的中国和平崛起”，对我们来说，是一份具有挑战性、思维宽阔的任务。如何基于科技、商业、政治等多种因素，合理有效地开展合作，推动中国社会的发展和进步，也是我们必须要思考和总结的。

 # 2.核心概念与联系

## 中国的崛起

“9.“暴风雨”之后的中国和平崛起”，其前提假设就是：“中国需要与世界接轨，更好地融入世界。”

首先，如何理解中国的崛起？我认为中国在过去的几千年里，经历了一系列不同的阶段。

- **夏朝初期**：鸦片战争、甲午战争后，被英国殖民者强占下来的封建遗产被迁移到北京，并创立了一座高楼林立的大帝王陵墓。隋唐时期是中国历史上的繁荣时期。
- **元朝末期**（1492-1564）：曹操出使西域，吞并周朝。宣统皇帝为了巩固政权，把大清帝国扩张到亚洲和太平洋。随着时间的推移，清政府的治理水平越来越高，成为世界上最强大的军事国家。
- **明朝初期**（1643-1727）：晚明政府在满洲建立的独立宗主国。中国成为重要的世界通用语言的国家。
- **清朝末期**（1868-1901）：辛亥革命后，建立了共和制，成为第一个直属于中国的大国。五四运动推翻了帝制，邓小平领导的改革开放政策提升了GDP的增速。
- **现代**（1949-至今）：建成社会主义伟大事业的康梁、戈尔巴乔夫等人都倡导全盘西化，引起国内异端分子激烈反弹，导致苏联解体、东欧剧变。近年来，以习近平同志为代表的党中央，秉持“一带一路”开放战略，让东北亚、南海诸岛、中东欧国家纷纷卷入中国经济版图。同时，美国正在加紧对华为、苹果、谷歌等公司技术垄断的打压，这些企业对中国的发展产生了不可替代的影响。

通过分析以上这几个阶段的发展进程，我们可以发现，中国在短短的几百年时间里，已经完成了一个完整的自我革命，从宋朝到明清时代，形成了一套自己的政治体系和社会制度。但是，面临“文化大革命”、“六四”、SARS等众多事件，国家快速进入了困境，国民的生活水平大幅下降，民间士人们发起“红二代”、“回归祖国”等活动，一系列变化让中国的命运发生了极大的变化。

## 暴风雨之后的抗疫

“9.“暴风雨”之后的中国和平崛起”将研究的内容定位于中国在“暴风雨”后的抗疫策略选择、社会动员、政治体制改革、国际合作等方面所做出的决策，旨在解读这些行动背后的深层原因、启示和历史教训。这包括以下几个部分：

### 一、如何判断新冠病毒是否人传人？

2019年12月初，武汉的武昌区有多名医护人员被确诊感染新冠肺炎。然而，当时没有任何相关信息，就凭感觉判断是潜伏期或不是，结果误判将他们送往ICU隔离。事后证实，该病毒为霍乱病毒的一种变种，可以传染人群。

这种病毒“潜伏期”较长，一般是几天至一周左右，而症状较重的发病者则会持续至少两三星期。新冠肺炎患者的生活常态是每天“走亲访友、聚餐娱乐”，在家办公或者在车站乘坐公共交通工具。很难有人持续观察在家中或车上传播的情况。所以，初期检测还比较粗糙。2020年1月，CDC组织了第一次全国新冠病毒样本检测，结果证明，已知的病毒株可以检测出新冠病毒，并对1月1日之后出现的大规模感染有一定预警能力。

那么，如何准确判断新冠病毒是否为人传人？目前，科研团队正在探索利用计算机网络流行病学数据进行可靠的反向工程验证。这项工作需要大量的经费投入，预计需要十年的时间才能完工。

### 二、如何应对新冠病毒的流行？

2020年1月23日，华南海鲜批发市场又遭遇猪瘟疫情。该市场被迫关闭，当天就有数十万猪饿死。这是一次严重的群体感染事件。

当时，有多家媒体报道，云南省普洱市和楚雄州李江县的市民，发现群众吃猪肉为难、呕吐腹泻，大量涌向县城，要求严惩凶手。据说，县医院的消毒水温度过低，导致空气污染，猪肉变质，导致无法消化。虽然，仍然有些地方暂停销售猪肉，但集中摆放的猪肉还是受害者吃不下，损失巨大。县公安局对猪场及外出人员集中收治，又拘捕了一批医务人员。

2020年3月16日，疫情在全国蔓延，国内外医疗资源供应出现短缺。河北省张家口市发生一起野生动物瘟疫，造成34人死亡，470余人感染。其中河北籍12人感染，其中20人在市区内居住，另有12人在郊外。

2020年5月，新冠肺炎疫情再次爆发，河南省禹州市突击检查，确定了主要感染源头。截止目前，该市有确诊病例336例，疑似病例830余例，累计治愈出院病例910例。

2020年7月31日，浙江宁波发生一起瘟疫，造成105人感染，102人死亡。浙江省中心医院在微信平台发布消息，称患者自驾游时被感染，已陆续转院到其他医院接受治疗。

面对日益严峻的形势，如何抗击新冠肺炎疫情，解决疫情后劲能否弥补？

2020年7月27日，习近平总书记在福建省厦门出席全国电视电话会议时表示，要坚持新发展理念、推动高质量发展，构建新发展格局。他强调，我们要按照“三个定位、两个率先”的要求，围绕人民生命健康安全，统筹疫情防控和经济社会发展，促进人的全面协同共赢，持之以恒开拓创新发展模式。

2020年8月27日，习近平总书记在全国抗击新冠肺炎疫情斗争暨全面推进新时代中国特色社会主义伟大事业大会上，提出“全面从严治疗、重点保障抗疫工作”的总体部署，并指出，“这是一次严峻的考验，必将激励全国各族人民奋勇开创新的历史新局”。

为了应对新冠肺炎疫情，如何给予医护人员更好的服务，以及如何抬高防护级别？

近年来，人们逐渐认识到，新冠肺炎疫情的抗疫和经济发展息息相关。据报道，今年上半年，日本成为全球第二个新冠肺炎确诊病例超千万例的国家，累计新增确诊病例超过25万。

2020年12月，武汉市卫健委主任钟灵对媒体表示，要坚持全方位、分层次、立体有效的防疫举措，落实“全民免费检测”“一劳永逸”“全覆盖”“快结果”“可追溯”，让每个人都获得基本的防护。

2020年9月，全国各省、自治区、直辖市启动应急响应，“非必要不离湖、坚决阻隔疫情”。“八月三连跳”、“永不放弃”的“疫后乡村”。

综上所述，如何抗击新冠肺炎疫情，如何应对流行病疫情，已经成为我们需要重视的课题。如何基于科技、商业、政治等多种因素，合理有效地开展合作，推动中国社会的发展和进步，仍然是一个关键问题。
