# 1.背景介绍


## 概述
华为从事信息化领域超过十年时间，已经积累了丰富的专业技能，包括IT规划、网络规划、IT管理、项目管理、营销策划、产品设计、硬件开发、软件开发、测试等多个方面专业知识。然而，像华为这样的国内顶尖科技公司，在人才资源上一般都不及美国、欧洲等国家，所以华为需要解决如何快速吸纳、培养具有自我驱动力、极致能力的优秀人才的问题。当下，科技企业逐渐转向以云计算、大数据、区块链为代表的新兴产业，为了更好地服务于客户群体，各个行业也在布局自己的高端人才市场。2019年3月，华为推出了“昇腾芯片”品牌，旨在打造一款通用处理器架构，同时兼容手机、平板电脑、嵌入式设备、IoT终端等不同应用场景。华为昇腾芯片拥有强大的AI算力、高达1000TFLOPS（每秒亿次）的处理性能，可以满足各种高速计算需求，包括视频渲染、机器学习、深度学习、语音识别等，正在成为当今最热门的计算平台之一。为了能够快速培养出具有华为高超人才素质的优秀人才，华为与高校合作推出了“华为研究生教育基金”。在华为研究生教育基金中，除了培养普通本科生，还设置“海外研究生”和“华为中国研究生”，让在华为工作的海外同学和华为员工享受到华为独有的训练环境。而华为的AI创新引擎、昇腾芯片高端人才都将由此产生价值。

HPCC(High Performance Computing and Communication Conference)全称“高性能计算与通信大会”，是华为推出的又一重大战略性科技会议。会议以“挖掘潜力、服务客户”为宗旨，聚焦于高性能计算与通信领域的最新进展、人才培养、技术沉淀、创新引领以及对行业应用的探索。通过主会场、分会场、展示厅、峰会展会、论坛展会、线下活动等形式，华为希望通过与国际顶尖学术机构保持密切合作，共同开展国际学术交流、科技前沿探讨以及产业对接，促进高性能计算和通信领域的交流合作，加强人才培养和交流机制建设，提升国际化水平。截止目前，HPCC已经举办了20余届，吸引了众多华为员工、华为爱好者、业界精英和行业专家，聚焦高性能计算和通信领域的学术创新、人才招聘、创客活动等一系列高端科技活动。

## 会议内容
HPCC是华为科技集团控股高层定期召开的高层级会议，旨在邀请专家学者以及华为员工以及业界顶将家一起研讨、探讨并分享关于高性能计算、通信以及其他相关领域的最新发展、前沿技术、创新思路、技术沉淀及对华为的应用。华为主管部门以及相关研究院所、单位等领导高度重视HPCC的意义与重要性，为其订立了具体目标并组织筹备方案。HPCC分为两个阶段，即前瞻型和高端水平型。前瞻型的目的是为了探索行业应用前景以及需求，收集研究产业、技术及应用领域的最新的发展信息；高端水平型的目的是为华为员工以及华为的主要顾客提供最前沿的研究信息。二者相辅相成，形成了华为高性能计算、通信、物联网等领域最具影响力的学术交流会议。
