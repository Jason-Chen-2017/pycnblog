# 1.背景介绍



现代建筑学是一门实践性很强、深入人心的科学研究领域。而建筑结构设计的核心任务之一就是如何利用材料的弹性和可塑性建造精美的结构体系。然而，由于现代建筑中的材料选择及制作技术存在着诸多不确定因素，例如材料成本高昂、材料组合不灵活等，导致了制作精细的、自适应的布置结构变得十分困难。因此，在建筑材料领域，基于传统结构设计方法并结合电脑辅助设计工具的“复合材料”正在成为新的关注点。随着复合材料的应用到建筑结构设计中，对于如何设计精准的、自适应的布置结构已经有了更加深入和广泛的研究探索。目前，较为主流的结构设计方法包括结构层次结构（SBS）、单元阵列构件（UCA）、嵌合材料（FIBRILLARY）和外表面受压复合材料（ECT）。但都存在一定的缺陷，例如设计难度高、缺乏信息共享、抗拉强度差等。基于上述原因，基于Flexible Fabric Sheets(FFS)的新型复合材料结构设计方法被提出，旨在克服上述缺陷，可以有效地满足复杂且多样化的建筑设计需求。FFS由多个有机物或微生物界面组成，具有高度的弹性和自我填充能力，能够通过拉伸或压缩来进行自适应调整。在FFS结构的设计中，可以使用传感器获取材料的变形信息，将其转换为电脑辅助设计工具提供给用户的视觉信息，从而实现结构的精细化和自适应性的扩展。此外，还可以通过模拟技术获得安全性能评价指标，帮助用户进一步规划和优化该结构的设计。

然而，为了验证FFS结构的设计方案的实际可行性，需要进行大量的实验测试，而且材料选取也会影响到实验结果的好坏。因此，本文试图运用大数据处理平台对不同材料组合下的FFS结构进行质量分析，并通过实验模拟对FFS结构的防护性能、舒适性、结构参数、优化空间进行预测，从而让读者了解FFS结构设计的方法、原理和实际应用的价值。在此基础上，作者希望能为读者提供一种关于FFS结构设计方法的直观认识，并对其在建筑设计领域的应用做出一些有益的建议。

# 2.核心概念与联系
## 2.1 FFS的定义
FFS（Flexible Fabric Sheets）是一种新型复合材料结构，它由多个有机物或微生物界面组成，具有高度的弹性和自我填充能力，能够通过拉伸或压缩来进行自适应调整。 FFSS的设计方法是基于传统结构设计方法并结合电脑辅助设计工具的，可以实现结构的精细化和自适应性的扩展。 FFS通过结构层级、单元阵列构件、嵌合材料和外表面受压复合材料四个方面进行分类。


## 2.2 SBS的特点
SBS（Structure-Based Structural Design，结构层次结构设计法），又称为框架结构设计法，是一种通过层级结构的线性布置来构造建筑的建筑结构设计法。框架结构的特点是使用一系列稳定的单元，按照固定顺序连接起来，从而构成一个完整的系统。结构层次结构可以将构件之间的空隙进行预留，使得外表面间隔率最小。一般来说，结构层次结构的设计使用最广泛的就是基桩设计法。

## 2.3 UCA的特点
UCA（Unit Cell Array，单元阵列构件法）是一种多单元的数组设计法，它采用几何均匀分布的单元，每个单元具有相同的尺寸和数量，且可以根据需要进行简化。阵列形式的单元可以有效地避免系统的孤立，为结构的连续性和完整性提供了保证。单位阵列构件法的特点是在保持单元的尺寸不变的同时，减小系统的尺寸，方便了布置时的材料选择和定位。

## 2.4 FIBRILLARY的特点
FIBRLARCY（Fibrillary Materials and Structures，纤维材料及结构）是一种由纤维状体或芳香团聚构成的复合材料，它可以作为层级结构的组成元素，也可以单独作为系统构件。纤维状材料一般采用全聚乙烯或聚乙烯酰胺聚合剂来实现。纤维状材料的结构与组织能力强，具有高度的耐力及自回敬能力。FIBRLARCY的结构特性可以实现超越单纯的层级结构。

## 2.5 ECT的特点
ECT（Elastic Composite Tiles，弹性复合板块）是一种外表面受压复合材料，它的结构类似于透明玻璃板块，具有高弹性及高附着力，可充分张开结构，对孔洞、陷阱等的防护功能极佳。复合板块在结构设计中占有重要的作用，在隔音、阻燥、防水、隔热等方面都有良好的应用。

## 2.6 FFS的特点
相比于传统的结构设计方法，FFS在材料组合上的优势主要有以下三个方面：

1. 复合材料的弹性。纤维状材料的耐力、弹性和自回敬能力都是普通材料无法替代的。纤维状材料还拥有能自我扩张的能力，这一特性对于实现精细化、自适应的结构设计至关重要。
2. 多材料的兼容性。不同材料的相互作用，比如粘附、粘聚、强化、带绕等，使得FFS的各项功能都得到保证。
3. 结构的自适应性。纤维状材料及其嵌合结构是可以自我填充的，这样就不仅能够实现不同的材料组合的布置，而且还可以自动调节嵌合结构的厚度，从而实现结构的精细化和自适应性的扩展。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
FFS结构设计主要依靠计算机辅助设计工具对其布置进行优化。FFS的原理是使用多个材料的协同作用，在一定程度上解决了传统结构设计方法存在的问题。下面的介绍主要基于钢圈框架结构进行说明，阐述FFS结构设计过程中各个环节的具体原理。

## 3.1 材料选择
首先，对材料进行选择。比如，FFS通常使用两种材料进行设计，即有机纤维或者微生物基材料和超胞膜材料。有机纤维材料有多个种类，如复合材料、有机纤维管道等；微生物基材料可以是线状结构的，如纤维状化石、分子纹状石墨、线状石英，也可以是网状结构的，如纽带状贝壳基岩、软骨砂壳基岩等；超胞膜材料是一种嵌合材料，具有极强的耐温、自回复和自合拍等特性。除了材料，还有一种重要的因素是起支架作用。

## 3.2 材料设计
材料设计需要考虑材料的大小和材料的稳定性。材料的尺寸大小要足够大，才能实现无限的自由度，并且材料要具备一定的弹性，所以材料的长度要足够长，而且应力应变应力最小，这样才能保证材料的绝对稳定性。除了材料的尺寸大小外，还需要考虑材料的形态。材料的形态应该尽可能平滑，这样可以在结构的设计中充分利用空间，但是不能过于丝毫，否则材料容易断裂、脆弱或浪费。

## 3.3 构件设计
构件设计用于将材料与固定零件、其它构件组装成符合要求的结构。构件的尺寸大小应尽可能小，这样就可以放入更多的材料，并且可以保证材料的组合稳定性。在构件设计时，需要考虑材料的位置。一般情况下，材料的位置应布置在空间的正中间，这样可以使得结构的弹性最大。如果材料的分布不好，则可以采用偏离中心位置布置材料。

## 3.4 接头设计
接头设计是指将材料与固体零件组装成符合要求的结构，接头设计往往需要进行复杂的设计，比如缝合，撤销，粘合等。在接头设计时，材料的纵向延伸距离应足够，这样才能实现材料的自适应性的扩展，并且材料的螺旋角度应合理，以便使材料能够充分螺旋。

## 3.5 设计参数选择
设计参数的选择往往决定了FFS结构的最终效果。一般来说，FFS结构的设计参数有以下几个方面：

1. 深度。深度决定了材料的密集程度，材料的厚度也会受到影响，厚度过大会导致材料的粘性减弱，而厚度过小又会导致材料磨损，所以深度要设置得适当。
2. 比例。纤维管道的比例决定了纤维管道宽度的大小，如果纤维管道宽度过大，则会导致混凝土的破碎，只能得到局部效果，如果纤维管道宽度过小，则会导致纤维管道无法承载重量，无法建立完整的结构。因此，比例应该适宜材料的尺寸大小和结构的空间要求。
3. 长度。纤维管道的长度决定了结构的尺寸大小，纤维管道越长，材料的厚度越薄，结构的厚度越厚，纤维管道也越长。长度应该适中，不能太长也不能太短，以免结构的重量不足。
4. 跳跃距离。纤维管道的跳跃距离决定了材料的间距，材料之间的间距越大，材料之间的耦合关系就越强，耦合关系也就越难产生，材料之间的弹性也就越小，结构的性能就会变弱。跳跃距离应该设置得适中，以确保材料之间的间距合适。
5. 摩擦系数。纤维管道的摩擦系数决定了材料之间的摩擦强度，摩擦系数越小，材料之间就会出现疏松的状态，这种状态下材料之间会发生弯曲，材料之间的滑动性也就减弱。摩擦系数应该设置得适当，以降低材料之间的摩擦强度，减少材料之间的弯曲。

## 3.6 结构优化
最后，结构优化是指对结构进行修正，使结构达到更好的性能。结构的优化方法有很多，这里只讨论一种最常用的结构优化方法——模拟优化。模拟优化是指用计算模拟的方式来求解设计参数，优化设计结果。模拟优化往往涉及到优化的目标函数、约束条件、搜索空间等，求解这些问题往往需要耗费大量的时间，所以模拟优化需要依赖于云计算来提升运算速度。云计算可以为工程师提供快速解决计算问题的能力。另外，还可以使用机器学习的方法来改善模拟优化的效率。
