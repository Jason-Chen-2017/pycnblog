# 1.背景介绍


在中国，海外华人群体占到了全球华人职场中的很大的比例。如果企业把精力放在培养海外华人，并且将海外华人纳入到公司文化建设中，那么将可以帮助该企业拓宽业务规模、提升产品品质，甚至还有可能实现盈利。

但由于海外华人的地理位置和语言环境不同，在日常生活和学习当中会面临一些特殊的困难。例如：

1. 了解海外的华人群体往往不如了解国内的华人群体简单直接，而海外华人群体往往具有更高的人口基数和文化素养；
2. 海外华人的语言环境相对较为落后，需要用多种方式来学习、交流、沟通，且无法被政府控制；
3. 海外华人与国内工薪阶层的劣势也是其不可忽视的一点。

因此，为了招揽更多海外华人加入公司团队，提升员工技能和工作效率，企业经常需要寻找海外优秀人才，开展海外项目，探索海外经营模式等。

在这种情况下，企业如何找到并留住海外华人就成了重点难题。

# 2.核心概念与联系
目前主要有两种海外人才招聘方式，分别是“外派”和“内推”。

## （1）外派
外派是指公司派遣海外优秀人才到公司进行工作。优秀人才包括能够胜任公司的关键岗位，同时也具有海外某一领域或行业的相关专长。

外派的方式主要分为两种，一种是直接派遣，即直接向海外指定的海外机构派遣海外优秀人才。另一种是由海外优秀人才自荐，这种方式通常是指一名海外优秀人才从事一项工作后向公司推荐应聘该职位，公司通过人力资源部门筛选确认后，授予该优秀人才相应的职位。

## （2）内推
内推是指企业邀请海外有声望的个人直接面试。优秀人才通常会受到来自海外知名投资机构或企业的资助或推荐，或者是来自海外异性恋社区。

外派和内推都属于招聘海外人才的有效方式，但是二者又存在着一些共同的特点。

首先，两者都会筛选海外优秀人才。而筛选的方法和标准则是不同的。例如，对于外派来说，公司通常会给出一份候选人报名表，要求候选人填写相关信息，然后由海外指定部门进行筛选。而对于内推来说，通常是由公司介绍海外公司、机构和个人。

其次，两者都会邀请海外优秀人才参加面试。然而，面试方式也不同。对于外派来说，海外优秀人才通常会接受各种面试形式，例如视频、电话、现场等。而对于内推来说，海外人才通常接受笔试面试。

最后，两者所邀请的海外人才各有不同背景，风格和能力，而这些都是影响其成功的重要因素。

综上，企业要想找到并留住海外优秀人才，首先应该明确自己的目标市场定位和需求，然后选择合适的方式进行招聘，并制定相应的筛选标准，让优秀人才真正符合自己的价值观和需求。
