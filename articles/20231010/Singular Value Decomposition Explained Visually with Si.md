# 1.背景介绍


Singular Value Decomposition (SVD) 是一种数值分析中常用的矩阵分解方法。它可以将任意一个实对称矩阵 A 分解成三个奇异矩阵 U、Σ 和 V^T 。其中，U 是 A 的左奇异矩阵（left singular vectors），V^T 是 A 的右奇异矩阵（right singular vectors）； Σ 是对角矩阵，其对角线上的值表示原始矩阵 A 的奇异值，这些值从大到小排列。SVD 可用于数据的降维、数据压缩等多种应用场景。比如，通过 SVD 可以发现数据的主成分，并利用它们来提取特征，进而进行数据可视化和分析。

本文重点介绍一种简单粗暴的对角化法求解 SVD ，其主要用途是在某些情况下可以获得快速、准确的结果。虽然该方法并不一定能得到最佳的结果，但对于某些特定的矩阵 A 来说，它的计算速度及精度都很好。在具体介绍 SVD 时，本文会先简要回顾一下对角化的方法，然后再进入如何使用简单对角化法求解 SVD 的过程。 

# 2.核心概念与联系
## 2.1 矩阵的奇异性和可逆性
设 A 为 n × m 的实对称矩阵，若存在非零向量 x 满足 AX = 0 或 AXA = AA，则称 A  为奇异矩阵。反之，若不存在这样的非零向量，则称 A 为可逆矩阵。如果 A 有复数元素或为非方阵，则不可逆。若 A 可逆，则可按以下方式分解 A=USVT：
- U 为 m × n 实正交矩阵，满足 UX = [x] ，其中 [x] 是长度为 m 的列向量，X 是 length of x 的单位向量。如果 A 是奇异矩阵，则 X 由下三角矩阵的第 k 个单位向量给出，此时 U 的第 i 行对应于 A 的第 i 个奇异值 Σ(i)。
- S 为奇异值对角矩阵，其对角线上的元素为从大到小排列的 Σ(i)，分别对应于矩阵 A 的奇异值。
- VT 为 n × n 实正交矩阵，满足 VTX = [x]^T ，其中 [x]^T 是长度为 n 的行向量，X 是 length of x 的单位向量。如果 A 是奇异矩阵，则 X 由上三角矩阵的第 k 个单位向量给出，此时 VT 的第 j 列对应于 A 的第 j 个奇异值 Σ(j)。

## 2.2 对角化矩阵
设 A 为任意实对称矩阵，存在一个 n × n 实对角矩阵 D 和一个 n × n 实单位矩阵 E，使得 AD=DE，则称 D 为矩阵 A 的本征值矩阵（eigenvalue matrix）。AD=λE，其中 λ 是 A 的 eigenvalues ，则 λ 为矩阵 A 的本征值。D 的第 i 行第 j 列的元素 Dij 为矩阵 A 在坐标 (i,j) 的本征值。

另外，有以下几个重要结论：
1. 如果 A 是奇异矩阵，那么 D 也是奇异矩阵，并且 A 的 eigenvalues 是递减的。
2. 如果矩阵 A 没有重复的 eigenvalue （即不存在多个相同的本征值），那么矩阵 A 是直交矩阵。
3. 如果矩阵 A 不含退化（defective） eigenvalue，那么 A 的秩 r 将等于矩阵 A 的最大本征值的个数。
4. 如果矩阵 A 是奇异矩阵，则 A 的对角化矩阵 D 将正定（positive definite）。

## 2.3 希尔伯特空间
设 H 为集合 C（ℝ^n） 上赋予内积的有限维向量空间，如果 H 中任意两个向量 u 和 v ，都有 ||u - v||≤ c(||u|| + ||v||) ，则称 H 为希尔伯特空间（Hilbert space）。希尔伯特空间的一个重要特性是，任意两条闭曲线在 H 中都存在唯一的极限点。因此，在 H 里研究函数（例如测度、范数、距离等）是很自然的。  

## 2.4 谱 decomposition
设 H 为希尔伯特空间，A 为 H 中的线性映射，则称 A 为矩阵 A 的谱 decomposition。当且仅当存在一个对称矩阵 Q ，使得 Q^TQ = I ，则称 A 是可对角化的。令 D 为矩阵 Q 的对角矩阵，则 D 表示矩阵 A 的本征值。D 的元素是从大到小排列的，其对应的本征向量顺序也由大到小。一般地，矩阵 A 的谱 decomposition 可以写为 A = WDW^T，W 为矩阵 D 的本征向量构成的矩阵。W 是 A 的 left-singular vectors，D 是 A 的 right-singular values。
