# 1.背景介绍


## OpenCV简介

OpenCV（Open Source Computer Vision Library）是一个开源的跨平台计算机视觉库，其由Intel、道格拉斯·福特·梅耶斯特、罗琦和其他志愿者共同开发维护，OpenCV作为开放源代码的计算机视觉库广泛应用于开发商业产品、移动设备、航空航天等领域。它包含了众多计算机视觉方面的基础算法和实用工具箱，能帮助开发人员快速开发出功能丰富、高效能的计算机视觉应用软件。

OpenCV主要由以下三个模块组成:

1. Video I/O 模块：负责输入输出视频流数据；
2. Image processing modules：提供高性能图像处理算法；
3. Machine Learning module：集成了机器学习算法和相关函数。

它的主要特性包括：

1. 灵活的接口：OpenCV支持多种编程语言，如C++、Python、Java、Matlab；
2. 高度优化：采用了多线程技术，能实现实时运算；
3. 跨平台：几乎可以在所有的操作系统上运行，适用于移动设备、嵌入式平台、服务器和云计算环境；
3. 可扩展性：可通过插件机制实现新的功能；
4. 文档完善：提供了丰富的教程、示例代码和参考手册。

## OpenCV的优点
### 跨平台
OpenCV可以运行在很多种平台上，包括Windows、Linux、Mac OS X、Android、iOS等主流的个人电脑、服务器、微控制器、消费级嵌入式系统等。你可以在任何需要使用的平台上使用OpenCV。

### 易用性
OpenCV提供了简单而友好的API，使得开发者可以很方便地进行图像处理工作。只需编写少量代码即可完成复杂的算法，比如边缘检测、模板匹配、形态学变换、特征提取、高精度数值运算等。

### 性能
OpenCV利用多线程技术，通过异步计算提升了图像处理的速度。同时，还有专门针对SIMD指令集的优化。这样就可以达到实时的效果。

### 兼容性
OpenCV采用C/C++语言编写，因此可以与各种语言接口集成，能够实现不同编程语言之间的交互。

### 扩展性
OpenCV采用插件机制，允许用户轻松实现自己的功能。比如，用户可以通过自定义的算法生成过滤器并加入到OpenCV中。这就增加了OpenCV的功能强度。
