# 2. 核心概念与联系

在深入学习神经网络之前，我们需要了解一些基本的概念和联系。以下是一些核心概念：

2.1 神经元：神经元是人工神经网络的基本单元，它接收输入信号，对信号进行处理，并输出结果。神经元由一个输入层、一个隐藏层和一个输出层组成。

2.2 权重：权重是神经元之间的连接，它用于调整输入信号的强度。权重可以通过训练来调整，以优化神经网络的性能。

2.3 激活函数：激活函数是用于将输入信号转换为输出信号的函数。常见的激活函数有Sigmoid、Tanh和ReLU等。

2.4 损失函数：损失函数是用于衡量神经网络预测结果与实际结果之间差异的函数。常见的损失函数有均方误差、交叉熵损失等。

2.5 反向传播：反向传播是一种训练神经网络的方法，它通过计算损失函数的梯度来调整权重。

2.6 深度学习：深度学习是一种基于神经网络的机器学习方法，它通过多层神经网络来解决复杂的问题。

2.7 卷积神经网络：卷积神经网络是一种特殊的深度学习模型，它主要应用于图像处理和自然语言处理等领域。

2.8 循环神经网络：循环神经网络是一种特殊的深度学习模型，它主要应用于时间序列数据处理等领域。

2.9 生成对抗网络：生成对抗网络是一种特殊的深度学习模型，它主要应用于图像生成和图像分类等领域。
