# 2.核心概念与联系

在开发智能家居产品之前，我们需要了解一些核心概念和技术。这些概念包括：

- 物联网（IoT）：物联网是指物理设备与计算设备之间的互联互通，这使得物理设备可以被控制和监控。智能家居产品就是基于物联网技术的产品。
- 云计算：云计算是一种计算模式，它允许用户在网络上使用计算资源，而不需要购买和维护自己的计算设备。智能家居产品通常需要与云计算服务进行交互，以实现各种功能。
- 人工智能：人工智能是一种计算机科学技术，它使计算机能够进行自主决策和学习。智能家居产品可以使用人工智能技术，例如机器学习和深度学习，来提供更智能的功能。
