## 4.具体代码实例和详细解释说明

以下是一个使用Python的Matplotlib库创建柱状图的具体代码实例：

```python
import matplotlib.pyplot as plt

# 创建数据
data = [5, 10, 15, 20, 25]

# 创建柱状图
plt.bar(range(len(data)), data)

# 设置图表标题和x轴标签
plt.title('Sample Bar Chart')
plt.xlabel('Category')
plt.ylabel('Value')

# 显示图表
plt.show()
```

在这个代码实例中，我们首先导入了Matplotlib库。然后，我们创建了一组数据，并使用`plt.bar()`函数创建了一个柱状图。接下来，我们使用`plt.title()`、`plt.xlabel()`和`plt.ylabel()`函数设置了图表的标题和x轴标签。最后，我们使用`plt.show()`函数显示了图表。
