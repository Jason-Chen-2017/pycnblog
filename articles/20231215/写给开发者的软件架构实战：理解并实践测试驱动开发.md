以下是一个简单的测试驱动开发的代码实例：

```python
def add(x, y):
    """
    添加两个数
    """
    return x + y

def test_add():
    assert add(1, 2) == 3
    assert add(-1, 1) == 0
    assert add(0, 0) == 0

if __name__ == '__main__':
    test_add()
```

在这个代码实例中，我们首先编写了一个名为 `add` 的函数，用于添加两个数。然后，我们编写了一个名为 `test_add` 的测试用例，用于验证 `add` 函数的正确性。最后，我们运行了这个测试用例，发现它通过了所有的测试。
