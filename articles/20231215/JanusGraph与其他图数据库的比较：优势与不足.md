                 

# 1.背景介绍

图数据库是一种非关系型数据库，专门用于存储和查询具有复杂关系的数据。图数据库的核心概念是节点、边和属性，它们可以用来表示实际世界中的实体、关系和属性。图数据库的优势在于它们可以更好地处理复杂的关系和网络数据，而不是传统的关系型数据库。

JanusGraph是一个开源的图数据库，它是一个基于Hadoop和Elasticsearch的分布式图数据库。JanusGraph的设计目标是提供高性能、高可扩展性和高可用性的图数据库解决方案。JanusGraph支持多种存储后端，包括HBase、Cassandra、Elasticsearch和其他关系型数据库。
