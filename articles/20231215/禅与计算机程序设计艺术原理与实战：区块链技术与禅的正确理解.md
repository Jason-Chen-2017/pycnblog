## 4.1 分布式共识算法实例
以下是一个简单的POW算法实例：

```python
import hashlib

def pow_algorithm(data):
    nonce = 0
    while True:
        hash = hashlib.sha256(data + str(nonce).encode()).hexdigest()
        if hash[:4] == '0000':
            break
        nonce += 1
    return nonce

data = 'Hello, World!'
nonce = pow_algorithm(data)
print(nonce)
```

在这个实例中，我们使用了POW算法来解决一定难度的数学问题，从而获得奖励。POW算法的核心思想是通过消耗计算资源来保证数据的安全性和完整性。

## 4.2 加密算法实例
以下是一个简单的RSA算法实例：

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

def rsa_algorithm(public_key, private_key, data):
    cipher_rsa = PKCS1_OAEP.new(public_key)
    encrypted_data = cipher_rsa.encrypt(data)

    cipher_rsa_private = PKCS1_OAEP.new(private_key)
    decrypted_data = cipher_rsa_private.decrypt(encrypted_data)

    return decrypted_data

public_key = RSA.import_key(open('public_key.pem', 'rb').read())
private_key = RSA.import_key(open('private_key.pem', 'rb').read())
data = 'Hello, World!'

encrypted_data = rsa_algorithm(public_key, private_key, data)
print(encrypted_data)
```

在这个实例中，我们使用了RSA算法来加密和解密数据。RSA算法的核心思想是通过使用两个不同的密钥来加密和解密数据，从而保证数据的安全性和隐私性。
