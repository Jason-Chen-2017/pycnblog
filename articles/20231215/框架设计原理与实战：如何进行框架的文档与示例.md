```python
from abc import ABC, abstractmethod

class TextProcessor(ABC):
    @abstractmethod
    def process_text(self, text):
        pass
```

接下来，我们需要创建一个模块来实现文本处理功能。这个模块将实现文本处理接口的方法。

```python
class PlainTextProcessor(TextProcessor):
    def process_text(self, text):
        return text.lower()
```

最后，我们需要创建一个主程序来使用文本处理模块。这个主程序将创建一个文本处理对象，并调用其方法来处理文本文件。

```python
def main():
    text_processor = PlainTextProcessor()
    text = "Hello, World!"
    result = text_processor.process_text(text)
    print(result)

if __name__ == '__main__':
    main()
```
