- 非对称加密：使用不同的密钥对数据进行加密和解密。例如，RSA加密。

### 4.1 数据加密

我们将使用AES加密算法来加密客户信息。AES加密算法是一种对称加密算法，使用同一个密钥对数据进行加密和解密。以下是AES加密和解密的代码实例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# 生成AES密钥
key = get_random_bytes(16)

# 加密客户信息
def encrypt(data, key):
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(data)
    return cipher.nonce + tag + ciphertext

# 解密客户信息
data = b'客户信息'
encrypted_data = encrypt(data, key)

# 解密客户信息
def decrypt(encrypted_data, key):
    cipher = AES.new(key, AES.MODE_EAX, nonce=encrypted_data[:16])
    data = cipher.decrypt_and_verify(encrypted_data[16:])
    return data

decrypted_data = decrypt(encrypted_data, key)
print(decrypted_data)
```
