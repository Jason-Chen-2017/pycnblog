## 4.1 自动驾驶汽车价格模型代码
```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression

# 读取数据
data = pd.read_csv('autonomous_car_price_data.csv')

# 数据预处理
X = data[['opening_price', 'development_cost', 'production_cost', 'sales_cost']]
y = data['price']

# 模型训练
model = LinearRegression()
model.fit(X, y)

# 模型预测
predictions = model.predict(X)

# 模型评估
from sklearn.metrics import mean_squared_error
mse = mean_squared_error(y, predictions)
print('Mean Squared Error:', mse)
```
