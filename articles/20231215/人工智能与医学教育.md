5. 模型评估：对训练好的模型进行评估，检查其性能是否满足需求。

6. 模型优化：根据评估结果，对模型进行优化，以提高性能。

7. 模型部署：将优化后的模型部署到实际应用环境中，实现自动化决策。

```python
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

def recommend(user_id, user_ratings, movies):
    # 计算用户与电影之间的相似度
    similarity = cosine_similarity(user_ratings, user_ratings)

    # 获取用户喜欢的电影
    liked_movies = movies[user_ratings > 0]

    # 计算用户喜欢的电影与其他电影之间的相似度
    similarity_liked_movies = cosine_similarity(liked_movies, liked_movies)

    # 获取用户没有看过的电影
    unseen_movies = movies[user_ratings == 0]

    # 计算用户没有看过的电影与用户喜欢的电影之间的相似度
    similarity_unseen_movies = cosine_similarity(unseen_movies, liked_movies)

    # 计算每部电影的推荐得分
    recommendation_scores = similarity_unseen_movies * similarity_liked_movies

    # 获取推荐的电影
    recommended_movies = np.argwhere(recommendation_scores > 0)

    return recommended_movies
```
