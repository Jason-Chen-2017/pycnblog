## 3.1 概率论基础
### 3.1.1 概率的基本概念
概率是一种数学概念，用于表示事件发生的可能性。概率通常表示为一个数值，范围在0到1之间。0表示事件不可能发生，1表示事件必然发生。

### 3.1.2 概率的基本定理
概率的基本定理是概率论中的一个重要定理，它可以用来计算多个事件发生的概率。概率的基本定理可以表示为：

P(A 或 B) = P(A) + P(B) - P(A 与 B)

### 3.1.3 条件概率
条件概率是一种概率概念，用于表示事件发生的可能性，给定另一个事件已经发生。条件概率可以表示为：

P(A|B) = P(A 与 B) / P(B)

### 3.1.4 独立性
独立性是一种概率概念，用于表示两个事件发生的可能性，不受另一个事件的影响。两个事件独立时，它们的条件概率相等：

P(A|B) = P(A)

## 3.2 统计学基础
### 3.2.1 样本和总体
统计学中的样本是从总体中随机抽取的一组数据。样本用于估计总体的参数。

### 3.2.2 描述性统计
描述性统计是一种统计学方法，用于对数据进行描述。描述性统计包括中心趋势、离散程度和形状等。中心趋势包括平均值、中位数和模数等。离散程度包括标准差、方差和分布宽度等。形状包括偏度和峰度等。

### 3.2.3 分类和聚类
分类是一种统计学方法，用于将数据分为多个类别。分类可以根据数据的特征进行分类，如性别、年龄、收入等。聚类是一种统计学方法，用于将数据分为多个组。聚类可以根据数据的相似性进行分组，如K均值聚类、层次聚类等。

### 3.2.4 推断
推断是一种统计学方法，用于从样本中估计总体的参数。推断包括估计和检验两种方法。估计是用于估计总体参数的方法，如平均值、中位数等。检验是用于验证假设的方法，如t检验、F检验等。

```python
import pandas as pd
from sklearn.cluster import KMeans

# 定义数据
data = pd.DataFrame({'age': [20, 30, 40, 50, 60], 'income': [10000, 20000, 30000, 40000, 50000]})

data['age_group'] = pd.cut(data['age'], bins=[20, 40, 60], labels=['young', 'middle', 'old'])

```

import numpy as np
