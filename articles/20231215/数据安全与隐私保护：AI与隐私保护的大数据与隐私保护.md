### 2.1 数据安全与隐私保护的核心概念

数据安全与隐私保护是两个相关但不同的概念。数据安全是指保护数据免受未经授权的访问、删除、修改等操作。隐私保护是指保护个人的隐私信息，确保这些信息不被泄露给未经授权的人。

### 2.2 数据安全与隐私保护的联系

数据安全与隐私保护之间存在着紧密的联系。数据安全是隐私保护的基础，因为只有数据安全才能确保隐私信息的安全。同时，隐私保护也是数据安全的一个重要应用场景，因为隐私信息需要保护免受未经授权的访问和泄露。

from Crypto.Cipher import AES

def encrypt(plaintext, key):
    cipher = AES.new(key, AES.MODE_EAX)
    ciphertext, tag = cipher.encrypt_and_digest(plaintext)
    return cipher.nonce, ciphertext, tag
