首先，我们需要对文本进行预处理。我们可以使用Python的NLTK库来实现这一步。以下是代码实例：

```python
import nltk
from nltk.corpus import stopwords
from nltk.stem import PorterStemmer

# 加载停用词列表
stop_words = set(stopwords.words('english'))

# 定义一个函数来去除停用词
def remove_stopwords(text):
    words = nltk.word_tokenize(text)
    filtered_words = [word for word in words if word.lower() not in stop_words]
    return filtered_words

# 定义一个函数来进行词干提取
def stem_words(words):
    stemmer = PorterStemmer()
    stemmed_words = [stemmer.stem(word) for word in words]
    return stemmed_words

# 对文本进行预处理
text = "I love you, and I love AI."
filtered_words = remove_stopwords(text)
stemmed_words = stem_words(filtered_words)
```

然后，我们需要计算文本之间的相似度。我们可以使用Python的scikit-learn库来实现这一步。以下是代码实例：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 定义一个函数来计算文本之间的相似度
def compute_similarity(texts):
    vectorizer = TfidfVectorizer()
    tfidf_matrix = vectorizer.fit_transform(texts)
    similarity_matrix = cosine_similarity(tfidf_matrix)
    return similarity_matrix

# 计算文本之间的相似度
texts = [text for text in texts]
similarity_matrix = compute_similarity(texts)
```

最后，我们需要使用信息检索算法来查找与给定查询“人工智能”相关的文章。我们可以使用Python的numpy库来实现这一步。以下是代码实例：

```python
import numpy as np

# 定义一个函数来查找与给定查询相关的文章
def find_related_articles(similarity_matrix, query):
    query_vector = vectorizer.transform([query])
    similarity_scores = np.dot(query_vector, similarity_matrix.T)
    related_articles = np.argsort(-similarity_scores)
    return related_articles
```
