## 4.具体代码实例和详细解释说明

在这里，我们给出一个简单的购物车模块的代码实例，并进行详细解释说明：

```python
class ShoppingCart:
    def __init__(self):
        self.items = []

    def add_item(self, item):
        self.items.append(item)

    def remove_item(self, item):
        self.items.remove(item)

    def get_total_price(self):
        total_price = 0
        for item in self.items:
            total_price += item.price * item.quantity
        return total_price
```

这个代码实例定义了一个购物车类，包括添加商品、移除商品、获取总价格等功能。

- `__init__` 方法是类的初始化方法，用于初始化购物车的商品列表。
- `add_item` 方法用于添加商品到购物车，将商品添加到商品列表的末尾。
- `remove_item` 方法用于移除商品从购物车，将商品从商品列表中移除。
- `get_total_price` 方法用于获取购物车的总价格，遍历购物车中的所有商品，计算总价格。
