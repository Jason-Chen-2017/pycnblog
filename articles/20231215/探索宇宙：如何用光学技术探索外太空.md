# 2.核心概念与联系

在探索宇宙的过程中，光学技术扮演着重要的角色。光学技术的核心概念包括光学镜头、光学望远镜、光学相机等。这些技术的联系可以通过以下几点来理解：

- 光学镜头：光学镜头是光学技术的基本设备，用于收集、传递和聚焦光线。它通过对光线进行折射和反射来实现光的聚焦，从而使得远离观察者的物体能够被观察到。
- 光学望远镜：光学望远镜是光学技术的重要设备，用于观察远离地球的天体。它通过将光线聚焦到一个点上来增强远离观察者的物体的图像，从而使得更远的天体能够被观察到。
- 光学相机：光学相机是光学技术的重要设备，用于捕捉光学镜头或望远镜所收集的光线。它通过将光线传递到感光元件上来捕捉图像，从而使得远离地球的天体能够被捕捉到。
