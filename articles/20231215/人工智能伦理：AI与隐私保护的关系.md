# 4.1 代码实例介绍
我们将通过一个简单的数据掩码算法来实现隐私保护。这个算法的核心思想是在原始数据上添加噪声，以保护个人隐私信息。

# 4.2 代码实现
我们将使用Python语言来实现这个数据掩码算法。首先，我们需要导入相关库：

```python
import numpy as np
import random
```

然后，我们需要定义一个函数来生成噪声：

```python
def generate_noise(data, noise_scale):
    noise = np.random.normal(0, noise_scale, data.shape)
    return data + noise
```

在这个函数中，我们使用了numpy库来生成噪声。我们将原始数据data和噪声noise相加，得到掩码后的数据。

然后，我们需要定义一个函数来解密掩码后的数据：

```python
def decrypt_data(data, noise):
    return data - noise
```

在这个函数中，我们将掩码后的数据data和噪声noise相减，得到原始的数据。

最后，我们需要定义一个函数来测试这个数据掩码算法：

```python
def test_differential_privacy():
    data = np.array([1, 2, 3, 4, 5])
    noise_scale = 1
    noisy_data = generate_noise(data, noise_scale)
    decrypted_data = decrypt_data(noisy_data, noise_scale)
    print("Original data:", data)
    print("Noisy data:", noisy_data)
    print("Decrypted data:", decrypted_data)
```

在这个函数中，我们首先定义了一个原始的数据data，然后定义了一个噪声比例noise_scale。接着，我们使用generate_noise函数生成掩码后的数据noisy_data。最后，我们使用decrypt_data函数解密掩码后的数据，并打印出原始数据、掩码后的数据和解密后的数据。
