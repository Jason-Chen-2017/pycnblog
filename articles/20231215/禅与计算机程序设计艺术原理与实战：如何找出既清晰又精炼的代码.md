例子：编写一个程序，输入一个整数n，输出斐波那契数列的第n项。

```python
def fibonacci(n):
    if n == 1 or n == 2:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

n = int(input("请输入一个整数："))
print("斐波那契数列的第%d项为：%d" % (n, fibonacci(n)))
```

在这个例子中，我们首先定义了一个递归函数`fibonacci`，用于计算斐波那契数列的第n项。然后，我们通过输入一个整数n，并调用`fibonacci`函数来计算斐波那契数列的第n项。最后，我们输出结果。
