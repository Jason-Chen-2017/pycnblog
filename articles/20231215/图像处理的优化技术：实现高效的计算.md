## 3.1 滤波算法

滤波算法是图像处理中的一个重要环节，它主要用于去除图像中的噪声。滤波算法的核心原理是通过对图像像素值进行加权求和来得到新的像素值。

### 3.1.1 均值滤波

均值滤波是一种简单的滤波算法，它的核心思想是将当前像素值与周围的8个像素值进行加权求和，得到新的像素值。均值滤波的数学模型公式为：

$$
f(x,y) = \frac{1}{8} \sum_{i=-1}^{1} \sum_{j=-1}^{1} f(x+i,y+j)
$$

### 3.1.2 中值滤波

中值滤波是一种更高级的滤波算法，它的核心思想是将当前像素值与周围的8个像素值进行排序，然后取中间值作为新的像素值。中值滤波的数学模型公式为：

$$
f(x,y) = \text{sort}\left(\sum_{i=-1}^{1} \sum_{j=-1}^{1} f(x+i,y+j)\right)
$$

边缘检测算法是图像处理中的另一个重要环节，它主要用于识别图像中的边缘。边缘检测算法的核心原理是通过对图像像素值的梯度进行计算来得到边缘强度图。

import numpy as np

def filter_image(image, kernel):
    filtered_image = np.zeros(image.shape)
    for i in range(image.shape[0]):
        for j in range(image.shape[1]):
            pixel_values = image[i-kernel.shape[0]//2:i+kernel.shape[0]//2+1,
