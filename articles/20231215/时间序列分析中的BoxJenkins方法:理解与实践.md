# 2.核心概念与联系

## 2.1 自回归模型

自回归模型（AR）是一种线性时间序列模型，它假设当前观测值是由过去的观测值生成的。自回归模型的基本形式如下：

$$
y_t = \phi_1 y_{t-1} + \phi_2 y_{t-2} + ... + \phi_p y_{t-p} + \epsilon_t
$$

其中，$y_t$ 是当前观测值，$y_{t-1}, y_{t-2}, ..., y_{t-p}$ 是过去的观测值，$p$ 是模型的阶数，$\phi_1, \phi_2, ..., \phi_p$ 是模型参数，$\epsilon_t$ 是白噪声。

自回归模型的主要优点是它简单易用，可以用于预测单变量时间序列。但是，自回归模型的主要缺点是它不能处理随机的时间序列数据，因此在实际应用中，自回归模型通常与其他模型结合使用。

## 2.2 移动平均模型

移动平均模型（MA）是一种线性时间序列模型，它假设当前观测值是由过去的观测值的平均值生成的。移动平均模型的基本形式如下：

$$
y_t = \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + ... + \theta_q \epsilon_{t-q} + \epsilon_t
$$

其中，$y_t$ 是当前观测值，$\epsilon_{t-1}, \epsilon_{t-2}, ..., \epsilon_{t-q}$ 是过去的白噪声，$q$ 是模型的阶数，$\theta_1, \theta_2, ..., \theta_q$ 是模型参数，$\epsilon_t$ 是当前白噪声。

移动平均模型的主要优点是它可以处理随机的时间序列数据。但是，移动平均模型的主要缺点是它不能处理非线性的时间序列数据，因此在实际应用中，移动平均模型通常与其他模型结合使用。

## 2.3 部分自回归模型

部分自回归模型（ARIMA）是一种混合模型，它结合了自回归模型和移动平均模型的优点。ARIMA模型的基本形式如下：

$$
y_t = \phi_1 y_{t-1} + \phi_2 y_{t-2} + ... + \phi_p y_{t-p} + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + ... + \theta_q \epsilon_{t-q} + \epsilon_t
$$

其中，$y_t$ 是当前观测值，$y_{t-1}, y_{t-2}, ..., y_{t-p}$ 是过去的观测值，$p$ 是自回归阶数，$\phi_1, \phi_2, ..., \phi_p$ 是自回归参数，$\epsilon_{t-1}, \epsilon_{t-2}, ..., \epsilon_{t-q}$ 是过去的白噪声，$q$ 是移动平均阶数，$\theta_1, \theta_2, ..., \theta_q$ 是移动平均参数，$\epsilon_t$ 是当前白噪声。

ARIMA模型的主要优点是它可以处理非线性和随机的时间序列数据。但是，ARIMA模型的主要缺点是它需要进行参数估计和模型选择，这可能会导致过拟合或欠拟合的问题。因此，在实际应用中，ARIMA模型通常需要与其他模型结合使用。

## 2.4 最小二乘法

最小二乘法是一种用于估计模型参数的方法，它的基本思想是将观测值与模型预测值之间的差的平方和最小化。最小二乘法的基本形式如下：

$$
\min_{\beta} \sum_{t=1}^n (y_t - \beta_0 - \beta_1 x_{1t} - ... - \beta_k x_{kt})^2
$$

其中，$y_t$ 是当前观测值，$x_{1t}, x_{2t}, ..., x_{kt}$ 是当前输入变量，$\beta_0, \beta_1, ..., \beta_k$ 是模型参数，$n$ 是观测值的数量，$\beta$ 是模型参数向量。

最小二乘法的主要优点是它可以用于估计线性模型的参数，并且可以处理高维输入变量。但是，最小二乘法的主要缺点是它需要观测值和输入变量之间存在线性关系，并且对于非线性模型，最小二乘法可能会导致过拟合或欠拟合的问题。因此，在实际应用中，最小二乘法通常需要与其他方法结合使用。
