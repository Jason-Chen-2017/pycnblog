```python
import hashlib
import hmac
import base64

def encrypt_data(data, key):
    # 使用HMAC算法对数据进行加密
    digest = hmac.new(key, data.encode(), hashlib.sha256).digest()
    # 使用Base64算法对加密后的数据进行编码
    encoded_data = base64.b64encode(digest)
    return encoded_data

data = "这是一条需要加密的交易数据"
key = "这是一个加密密钥"
encrypted_data = encrypt_data(data, key)
print(encrypted_data)
```

在上述代码中，我们首先使用HMAC算法对数据进行加密，然后使用Base64算法对加密后的数据进行编码。这样，我们就可以保证数据安全性。
