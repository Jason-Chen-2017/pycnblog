                 

# 1.背景介绍

决策支持系统（DSS，Decision Support System）是一种利用计算机科学技术为组织内部或外部的决策者提供有关决策过程的信息、数据、知识和模型，以帮助他们更好地做出决策的系统。DSS 主要应用于企业管理、政府管理、军事管理等领域，以提高决策效率和质量。

DSS 的核心概念包括决策支持系统的类型、决策支持系统的组成部分、决策支持系统的功能和决策支持系统的应用领域。

## 1.1 决策支持系统的类型

根据不同的应用场景和决策需求，DSS 可以分为以下几类：

1. 基于数据的决策支持系统（DDBMS）：这类系统主要利用数据挖掘、数据分析和数据可视化等技术，帮助决策者从大量数据中找出有价值的信息和知识。

2. 基于模型的决策支持系统（MDSS）：这类系统主要利用数学模型、统计模型和专家系统等技术，帮助决策者建立和解决决策问题。

3. 基于人类知识的决策支持系统（KDSS）：这类系统主要利用人类专家的知识和经验，帮助决策者进行决策。

4. 基于网络的决策支持系统（WDSS）：这类系统主要利用网络技术，帮助决策者从网络上获取信息和资源，进行决策。

## 1.2 决策支持系统的组成部分

DSS 的主要组成部分包括：

1. 数据库：DSS 需要一个数据库来存储和管理决策相关的数据和信息。

2. 数据仓库：DSS 需要一个数据仓库来存储和管理决策相关的历史数据和知识。

3. 数据挖掘和分析工具：DSS 需要一套数据挖掘和分析工具来帮助决策者从大量数据中找出有价值的信息和知识。

4. 模型库：DSS 需要一个模型库来存储和管理决策相关的数学模型和专家系统。

5. 用户界面：DSS 需要一个用户界面来帮助决策者与系统进行交互。

6. 决策支持引擎：DSS 需要一个决策支持引擎来实现决策支持系统的功能和功能。

## 1.3 决策支持系统的功能

DSS 的主要功能包括：

1. 数据收集和存储：DSS 可以从各种来源收集和存储决策相关的数据和信息。

2. 数据处理和分析：DSS 可以对决策相关的数据进行处理和分析，以找出有价值的信息和知识。

3. 模型建立和解决：DSS 可以建立和解决决策相关的数学模型和专家系统。

4. 决策支持和推荐：DSS 可以根据决策相关的数据和模型，为决策者提供决策支持和推荐。

5. 结果评估和反馈：DSS 可以对决策结果进行评估和反馈，以帮助决策者进行后续决策。
