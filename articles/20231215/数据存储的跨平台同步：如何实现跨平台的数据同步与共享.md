# 2.核心概念与联系

在讨论跨平台数据同步与共享之前，我们需要了解一些核心概念。这些概念包括：数据同步、数据共享、跨平台、数据存储等。

## 2.1 数据同步

数据同步是指在多个设备或平台之间，实现数据的实时更新和保持一致性。例如，当我们在手机上修改了某个文件后，同时在电脑上也能看到这个修改。这就是数据同步的意义。

## 2.2 数据共享

数据共享是指在多个设备或平台之间，实现数据的交换和访问。例如，当我们在一台设备上创建了一个文件，其他设备也可以访问和修改这个文件。这就是数据共享的意义。

## 2.3 跨平台

跨平台是指在不同操作系统、硬件平台或软件平台之间，实现数据的交换和访问。例如，当我们在Windows操作系统上创建了一个文件，然后在Mac操作系统上也可以访问这个文件。这就是跨平台的意义。

## 2.4 数据存储

数据存储是指在设备或平台上，将数据保存到持久化存储设备上，以便在需要时进行访问和操作。例如，当我们在云端存储上保存了一个文件，那么这个文件可以在多个设备上访问和修改。这就是数据存储的意义。

import os
import shutil
import socket

def sync_data(source_path, target_path):
    # 创建套接字
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    # 连接到服务器
    sock.connect(('localhost', 8080))
    # 读取文件
    with open(source_path, 'rb') as f:
        data = f.read()
    # 发送文件数据
    sock.sendall(data)
    # 接收文件数据
    with open(target_path, 'wb') as f:
        while True:
            data = sock.recv(1024)
            if not data:
                break
            f.write(data)
    # 关闭套接字
    sock.close()

def share_data(source_path, target_path):
    # 创建套接字
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    # 绑定到本地地址
    sock.bind(('localhost', 8080))
    # 监听连接
    sock.listen(1)
    # 接收连接
    client_sock, addr = sock.accept()
    # 读取文件
    with open(source_path, 'rb') as f:
        data = f.read()
    # 发送文件数据
    client_sock.sendall(data)
    # 接收文件数据
    with open(target_path, 'wb') as f:
        while True:
            data = client_sock.recv(1024)
            if not data:
                break
            f.write(data)
    # 关闭套接字
    client_sock.close()
    sock.close()

if __name__ == '__main__':

    sync_data(source_path, target_path)
