## 4.1 Map任务的代码实例

在Map任务中，每个Map任务接收一组输入数据，并将其划分为多个部分。每个部分都会被一个Map任务处理。Map任务的主要目的是将输入数据转换为一组（键，值）对，这些对可以被Reduce任务处理。

以下是一个Map任务的代码实例：

```python
import sys

def map(line):
    words = line.split()
    for word in words:
        yield (word, 1)

input_data = sys.stdin.readlines()
for line in input_data:
    for (word, value) in map(line):
        print(word, value)
```

在这个代码实例中，我们定义了一个map函数，该函数接收一行输入数据，并将其划分为多个部分。每个部分都会被一个Map任务处理。Map任务的主要目的是将输入数据转换为一组（键，值）对，这些对可以被Reduce任务处理。

在Reduce任务中，每个Reduce任务接收一组（键，值）对，并将它们聚合为一个更小的集合。Reduce任务的主要目的是将多个（键，值）对合并为一个结果。

def reduce(key, values):
