## 4.具体代码实例和详细解释说明

### 4.1 简单的图灵机实现

以下是一个简单的图灵机实现，用于计算输入的偶数和奇数：

```python
class TuringMachine:
    def __init__(self):
        self.tape = []
        self.state = 'q0'
        self.transition_table = {
            'q0': {'0': ('q0', 'R', '0'), '1': ('q1', 'R', '1')},
            'q1': {'0': ('q1', 'R', '0'), '1': ('q2', 'R', '1')},
            'q2': {'0': ('q2', 'R', '0'), '1': ('q3', 'R', '1')},
            'q3': {'0': ('q3', 'R', '0'), '1': ('q0', 'R', '1')}
        }

    def step(self):
        current_state, symbol, new_symbol = self.transition_table[self.state][self.tape[-1]]
        self.tape.pop()
        self.tape.append(new_symbol)
        self.state = current_state

    def run(self, input_string):
        self.tape = [symbol for symbol in input_string] + ['#']
        while self.state != 'q0':
            self.step()

    def get_output(self):
        return ''.join(self.tape[1:-1])

turing_machine = TuringMachine()
input_string = '10101'
turing_machine.run(input_string)
print(turing_machine.get_output())  # 输出：'110'
```

### 4.2 代码解释

上述代码实现了一个简单的图灵机，用于计算输入的偶数和奇数。代码的主要组成部分包括：

- `TuringMachine`类：用于表示图灵机的抽象类，包括状态、符号集、规则表和操作方法。
- `step`方法：用于执行图灵机的一个操作步骤，包括状态转换、符号更新和移动读写头。
- `run`方法：用于执行图灵机的计算任务，直到达到终止条件。
- `get_output`方法：用于获取图灵机的输出结果。

在代码中，我们创建了一个`TuringMachine`对象，并设置了输入字符串为'10101'。然后，我们调用`run`方法执行计算任务，并调用`get_output`方法获取输出结果。最后，我们打印输出结果为'110'。
