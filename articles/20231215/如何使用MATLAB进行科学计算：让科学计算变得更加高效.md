1.线性方程组解：

在MATLAB中，我们可以使用`A\B`的语法来解决线性方程组。例如，我们可以编写以下代码来解决线性方程组：

```matlab
A = [1, 2, 3; 4, 5, 6; 7, 8, 9];
B = [13; 17; 21];
X = A\B;
```

在这个例子中，我们定义了一个线性方程组Ax=B，并使用`A\B`的语法来解决它。最后，我们得到了解决方案X。

2.积分计算：

在MATLAB中，我们可以使用`integral`函数来计算积分。例如，我们可以编写以下代码来计算积分：

```matlab
f = @(x) x.^3;
lower = 0;
upper = 1;
result = integral(f, lower, upper);
```

在这个例子中，我们定义了一个积分函数f(x)=x^3，并使用`integral`函数来计算积分。最后，我们得到了积分结果result。

3.微分方程解：

在MATLAB中，我们可以使用`ode45`函数来解微分方程。例如，我们可以编写以下代码来解微分方程：

```matlab
function dxdt = myode(t, x)
    dxdt = x(2);
end

tspan = [0, 10];
x0 = [1, 0];

[t, x] = ode45(@myode, tspan, x0);
```

在这个例子中，我们定义了一个微分方程dxdt/dt=x(2)，并使用`ode45`函数来解微分方程。最后，我们得到了解决方案x。
