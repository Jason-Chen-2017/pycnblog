```python
import hashlib

def hash_data(data):
    # 使用SHA-256算法对数据进行加密
    hash_object = hashlib.sha256(data.encode())
    # 获取加密后的数据
    hex_dig = hash_object.hexdigest()
    return hex_dig

data = "Hello, World!"
hashed_data = hash_data(data)
print(hashed_data)
```

在这个代码实例中，我们使用了Python的hashlib库来实现数据加密。首先，我们导入了hashlib库。然后，我们定义了一个hash_data函数，该函数接收一个数据参数，并使用SHA-256算法对数据进行加密。最后，我们将加密后的数据打印出来。
