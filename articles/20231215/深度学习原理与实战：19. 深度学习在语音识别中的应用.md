```python

```

3. 编译神经网络模型：

```python
model.compile(loss='categorical_crossentropy', optimizer=Adam(), metrics=['accuracy'])
```

4. 训练神经网络模型：

```python
model.fit(x_train, y_train, batch_size=32, epochs=10, verbose=1, validation_data=(x_val, y_val))
```
