```python
class DataVirtualizationService:
    def __init__(self, data_sources):
        self.data_sources = data_sources

    def abstract_metadata(self, metadata):
        abstract_metadata = {}
        for data_source in self.data_sources:
            abstract_metadata[data_source] = data_source.abstract_metadata(metadata)
        return abstract_metadata

    def unify_management(self, abstract_metadata):
        unified_metadata = {}
        for data_source, abstract_data in abstract_metadata.items():
            unified_metadata[data_source] = self.unify_metadata(abstract_data)
        return unified_metadata

    def unify_access(self, unified_metadata):
        unified_access = {}
        for data_source, unified_data in unified_metadata.items():
            unified_access[data_source] = self.unify_access(unified_data)
        return unified_access
```

- `abstract_metadata`：实现数据源的元数据抽象，将数据源的元数据抽象成一个字典。
- `unify_management`：实现数据源的元数据统一管理，将抽象后的元数据进行统一管理。

Q：数据虚拟化技术与数据库虚拟化技术有什么区别？
