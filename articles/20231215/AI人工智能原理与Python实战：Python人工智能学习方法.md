人工智能（Artificial Intelligence，AI）是计算机科学的一个分支，研究如何让计算机模拟人类的智能行为。人工智能的目标是让计算机能够理解自然语言、学习从数据中提取信息、自主地决策以及与人类互动。人工智能的发展历程可以分为以下几个阶段：

1. 早期人工智能（1956-1974）：这个阶段的人工智能研究主要关注于模拟人类思维的过程，例如逻辑推理、知识表示和推理、自然语言处理等。这个阶段的人工智能研究主要是基于规则和知识的方法，也被称为知识工程。

2. 深度学习时代（2012-至今）：这个阶段的人工智能研究主要关注于神经网络和深度学习的方法，例如卷积神经网络（CNN）、递归神经网络（RNN）、变压器（Transformer）等。这个阶段的人工智能研究主要是基于数据和算法的方法，也被称为数据主义。

## 1.1 人工智能的发展历程

人工智能的发展历程可以分为以下几个阶段：

1. **早期人工智能（1956-1974）**：这个阶段的人工智能研究主要关注于模拟人类思维的过程，例如逻辑推理、知识表示和推理、自然语言处理等。这个阶段的人工智能研究主要是基于规则和知识的方法，也被称为知识工程。

2. **符号处理时代（1975-1980）**：这个阶段的人工智能研究主要关注于符号处理和知识表示的方法，例如规则引擎、框架系统、黑板模型等。这个阶段的人工智能研究主要是基于规则和知识的方法，也被称为知识工程。

3. **连接主义时代（1980-1990）**：这个阶段的人工智能研究主要关注于神经网络和并行计算的方法，例如前馈神经网络（Feed Forward Neural Network，FFNN）、随机梯度下降（Stochastic Gradient Descent，SGD）、反向传播（Back Propagation）等。这个阶段的人工智能研究主要是基于数据和算法的方法，也被称为数据主义。

4. **深度学习时代（2012-至今）**：这个阶段的人工智能研究主要关注于神经网络和深度学习的方法，例如卷积神经网络（CNN）、递归神经网络（RNN）、变压器（Transformer）等。这个阶段的人工智能研究主要是基于数据和算法的方法，也被称为数据主义。
