## 4.1 代码实例

我们选择一个简单的排序算法——冒泡排序，来作为代码实例。冒泡排序是一种简单的排序算法，它的基本思想是通过多次交换相邻的元素，使得较小的元素逐渐向前移动，较大的元素逐渐向后移动。

以下是冒泡排序的Python代码实现：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

arr = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(arr))
```

## 4.2 详细解释说明

上述代码实现了冒泡排序算法的基本步骤：

1. 首先，我们定义了一个名为`bubble_sort`的函数，该函数接受一个列表`arr`作为输入参数，并返回一个排序后的列表。

2. 然后，我们获取列表的长度`n`，并使用两个嵌套循环来遍历列表中的所有元素。

3. 在外部循环中，我们使用变量`i`来表示已经排序的元素数量。在每次迭代中，我们会将最后`n-i`个元素进行排序。

4. 在内部循环中，我们使用变量`j`来表示当前迭代中需要比较的元素。我们会比较相邻的两个元素，如果后一个元素小于前一个元素，则交换它们的位置。

5. 最后，我们返回排序后的列表。
