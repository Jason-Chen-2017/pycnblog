### 2.2 最小二乘法的数学模型

最小二乘法的数学模型可以表示为：

$$
\min_{w} \sum_{i=1}^{n} (y_i - w^T x_i)^2
$$

其中，$w$ 是模型参数，$x_i$ 是输入特征向量，$y_i$ 是输出标签。最小二乘法的目标是找到最佳的 $w$，使得平方和最小。

## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的例子来演示最小二乘法的具体实现：

```python
import numpy as np
from scipy.optimize import least_squares

# 数据生成
np.random.seed(0)
X = np.random.rand(100, 2)
y = np.dot(X, np.array([1, 2])) + np.random.rand(100)

# 模型定义
def model(X, w):
    return np.dot(X, w)

# 损失函数定义
def loss(w, X, y):
    return np.sum((y - model(X, w)) ** 2)

# 最小二乘法求解
w_opt = least_squares(loss, np.zeros(2), args=(X, y))

# 输出结果
print("最佳拟合曲线参数：", w_opt.x)
```

在这个例子中，我们首先生成了一组随机数据，然后定义了模型和损失函数。最后，我们使用 `scipy.optimize.least_squares` 函数进行最小二乘法求解，并输出结果。
