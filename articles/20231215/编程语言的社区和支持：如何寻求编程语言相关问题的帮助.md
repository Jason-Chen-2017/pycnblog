## 3.1算法原理
- 递归：递归是一种解决问题的方法，其中问题被分解为更小的子问题，直到可以解决。递归可以用于解决各种问题，如求阶乘、求斐波那契数等。
- 动态规划：动态规划是一种解决最优化问题的方法，其中通过分步求解子问题，得到最优解。动态规划可以用于解决各种问题，如最长公共子序列、背包问题等。
- 贪心算法：贪心算法是一种解决问题的方法，其中在每个步骤中选择最佳解，直到得到最终解。贪心算法可以用于解决各种问题，如旅行商问题、活动选择问题等。

```python
        return 1
    else:
        return n * factorial(n-1)
```

## 4.2详细解释说明
上述代码实现了一个求阶乘的递归函数。函数接收一个整数参数n，如果n等于0，则返回1，否则返回n乘以递归调用自身的结果。
