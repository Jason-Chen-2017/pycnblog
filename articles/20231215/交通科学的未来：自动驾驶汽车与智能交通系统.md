# 2. 核心概念与联系

## 2.1 自动驾驶汽车

自动驾驶汽车是一种通过采用计算机视觉、传感器、导航等技术来实现汽车自主驾驶的系统。自动驾驶汽车可以根据当前的道路状况、交通状况、天气状况等因素来实现汽车的自主驾驶。自动驾驶汽车的主要技术包括：

1. 计算机视觉：通过采用计算机视觉技术来实现汽车的视觉识别和定位。
2. 传感器：通过采用传感器技术来实现汽车的距离测量、速度测量等。
3. 导航：通过采用导航技术来实现汽车的路径规划和跟踪。

## 2.2 智能交通系统

智能交通系统是一种通过采用信息技术、通信技术、导航技术等多种技术来实现交通系统的智能化管理的系统。智能交通系统的主要技术包括：

1. 信息技术：通过采用信息技术来实现交通信息的收集、处理和传播。
2. 通信技术：通过采用通信技术来实现交通信息的实时传输和共享。
3. 导航技术：通过采用导航技术来实现交通路线的规划和导航。

## 2.3 自动驾驶汽车与智能交通系统的联系

自动驾驶汽车和智能交通系统之间存在着密切的联系。自动驾驶汽车可以通过与智能交通系统的交互来获取交通信息，从而实现更智能的驾驶。同时，智能交通系统也可以通过收集自动驾驶汽车的驾驶数据，从而实现更智能的交通管理。
