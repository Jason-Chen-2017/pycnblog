## 3.3 数学模型公式详细讲解

### 3.3.1 K-均值聚类

K-均值聚类是一种基于距离的聚类算法。其核心思想是将数据分为K个组，使得每个组内的对象距离相近，每个组之间的对象距离较远。K-均值聚类的数学模型公式如下：

$$
\min_{C}\sum_{i=1}^{k}\sum_{x\in C_i}d(x,\mu_i)
$$

其中，$C$ 是簇集合，$k$ 是簇的数量，$d$ 是距离度量，$\mu_i$ 是簇$i$ 的质心。

### 3.3.2 DBSCAN聚类

DBSCAN是一种基于密度的聚类算法。其核心思想是将数据分为紧密连接的区域，并将这些区域中的对象分为不同的组。DBSCAN的数学模型公式如下：

$$
\min_{C}\sum_{i=1}^{k}\sum_{x\in C_i}d(x,\mu_i)
$$

其中，$C$ 是簇集合，$k$ 是簇的数量，$d$ 是距离度量，$\mu_i$ 是簇$i$ 的质心。

## 4.1 代码实例

### 4.1.1 K-均值聚类

```python
from sklearn.cluster import KMeans
import numpy as np

# 数据
X = np.array([[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]])

# 创建KMeans对象
kmeans = KMeans(n_clusters=2, random_state=0)

# 训练模型
kmeans.fit(X)

# 获取簇中心
centers = kmeans.cluster_centers_

# 获取簇标签
labels = kmeans.labels_

# 获取距离簇中心的平方和
inertia = kmeans.inertia_
```

### 4.1.2 DBSCAN聚类

```python
from sklearn.cluster import DBSCAN
import numpy as np

# 数据
X = np.array([[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]])

# 创建DBSCAN对象
dbscan = DBSCAN(eps=1.5, min_samples=2, random_state=0)

# 训练模型
dbscan.fit(X)

# 获取簇标签
labels = dbscan.labels_

# 获取核心点集合
core_points = dbscan.core_sample_indices_

# 获取簇中心
centers = dbscan.cluster_centers_
```
