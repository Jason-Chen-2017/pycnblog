## 3.1 卡方检验
卡方检验是一种非参数的统计方法，用于检验两个或多个变量之间是否存在关联。它的基本思想是将数据分为几个类别，然后计算每个类别的观察值与预期值之间的差异。这个差异的平方将被累加，得到一个卡方统计量。如果这个卡方统计量的值较大，则说明两个或多个变量之间存在关联。

卡方检验的数学模型公式如下：

$$
X^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}
$$

其中，$X^2$ 是卡方统计量，$k$ 是类别数，$O_i$ 是实际观察值，$E_i$ 是预期值。

### 3.1.1 具体操作步骤
1. 将数据分为几个类别。
2. 计算每个类别的实际观察值和预期值。
3. 计算每个类别的差异平方。
4. 累加所有类别的差异平方，得到卡方统计量。
5. 比较卡方统计量与对应的分位数，判断是否存在关联。

### 3.1.2 Python代码实例
```python
import numpy as np
from scipy.stats import chi2_contingency

# 创建数据
data = np.array([[10, 20, 30], [15, 25, 35]])

# 计算卡方统计量
chi2, p, dof, expected = chi2_contingency(data)

# 判断是否存在关联
if p < 0.05:
    print("存在关联")
else:
    print("不存在关联")
```

## 3.2 独立性检验
独立性检验是一种统计方法，用于检验两个或多个变量是否相互独立。它的基本思想是计算每个变量之间的相关系数，如果相关系数的绝对值较小，则说明两个或多个变量相互独立。

独立性检验的数学模型公式如下：

$$
r = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2}\sqrt{\sum_{i=1}^{n}(y_i - \bar{y})^2}}
$$

其中，$r$ 是相关系数，$x_i$ 和 $y_i$ 是两个变量的观察值，$\bar{x}$ 和 $\bar{y}$ 是两个变量的平均值。

### 3.2.1 具体操作步骤
1. 计算每个变量的平均值。
2. 计算每个变量之间的相关系数。
3. 判断相关系数的绝对值是否较小，以及是否小于预设的阈值。

### 3.2.2 Python代码实例
```python
import numpy as np
from scipy.stats import pearsonr

# 创建数据
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])

# 计算相关系数
correlation, p_value = pearsonr(x, y)

# 判断是否存在关联
if p_value > 0.05:
    print("不存在关联")
else:
    print("存在关联")
```

Q1：什么是卡方检验？

Q5：什么是相关系数？
A5：相关系数是一个数值，用于衡量两个变量之间的关系。它的值范围在-1到1之间，绝对值越大，说明两个变量之间的关系越强。如果相关系数的绝对值接近1，则说明两个变量之间存在强正关系；如果相关系数的绝对值接近-1，则说明两个变量之间存在强负关系；如果相关系数接近0，则说明两个变量之间没有明显的关系。
