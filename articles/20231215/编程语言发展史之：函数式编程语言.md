# 4.具体代码实例和详细解释说明

在函数式编程语言中，具体代码实例可以用来说明递归的使用方法。以下是一个简单的阶乘计算的递归代码实例：

```
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)
```

在这个代码实例中，`factorial`函数是一个递归函数。它接受一个整数`n`作为输入，并返回`n`的阶乘。如果`n`等于0，则返回1，否则返回`n`乘以`factorial(n - 1)`的结果。

这个代码实例中的递归调用是`factorial(n - 1)`。递归调用的结果是一个值，即`n`的阶乘。
