# 2.核心概念与联系
在计算机网络面试中，你需要熟悉一些核心概念，包括：

- OSI七层模型：这是一种分层模型，用于描述计算机网络的组成和功能。每一层都有自己的职责，包括物理层、数据链路层、网络层、传输层、会话层、表示层和应用层。
- TCP/IP四层模型：这是另一种分层模型，用于描述互联网的组成和功能。它包括网络接口层、网络层、传输层和应用层。
- IP地址：IP地址是计算机网络中用于唯一标识设备的数字地址。它有两种形式：IPv4和IPv6。
- 端口：端口是计算机网络中用于标识进程或服务的数字端口号。它的范围是0-65535。
- 协议：协议是计算机网络中用于规定数据传输的规则和格式的一组规范。例如，HTTP、FTP、SMTP等。
- 网络协议：网络协议是计算机网络中用于实现数据传输的协议。例如，TCP、UDP、ICMP等。

在计算机网络面试中，你需要熟悉一些核心算法原理，包括：

- 哈希算法：哈希算法是一种用于将数据映射到固定长度的哈希值的算法。例如，MD5、SHA-1等。
- 加密算法：加密算法是一种用于保护数据安全的算法。例如，AES、RSA等。
- 分布式算法：分布式算法是一种用于解决多个计算机之间协同工作的问题的算法。例如，一致性哈希、分布式锁等。

- Q：什么是TCP四次挥手？
A：TCP四次挥手是一种用于断开TCP连接的方法，它包括客户端发送FIN包、服务器发送ACK包、客户端发送ACK包和服务器发送FIN包的四个步骤。
