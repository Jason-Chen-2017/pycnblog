RESTful API 是一种基于 REST（表述性状态转移）的应用程序接口设计风格。它使用 HTTP 协议进行通信，通过定义不同的 HTTP 方法（如 GET、POST、PUT、DELETE 等）来实现不同的操作。RESTful API 的主要优点是简单易用、灵活性强、可扩展性好等。

# 4.具体代码实例和详细解释说明
在这里，我们将通过一个具体的代码实例来说明如何使用 RESTful API 实现数据过滤。

假设我们有一个简单的数据集，包含以下信息：

```
{
  "name": "John",
  "age": 25,
  "city": "New York"
}
```

我们想要筛选出年龄为 25 岁的人。我们可以使用以下 RESTful API 接口来实现这个功能：

```
GET /api/people?age=25
```

在后端服务器上，我们可以使用以下代码来实现数据过滤：

```python
import json

def get_people(age):
    people = [
        {
            "name": "John",
            "age": 25,
            "city": "New York"
        },
        {
            "name": "Alice",
            "age": 30,
            "city": "Los Angeles"
        },
        {
            "name": "Bob",
            "age": 25,
            "city": "Chicago"
        }
    ]
    filtered_people = [person for person in people if person["age"] == age]
    return json.dumps(filtered_people)

if __name__ == "__main__":
    age = int(input("Enter age: "))
    response = get_people(age)
    print(response)
```

在这个代码中，我们首先定义了一个 `get_people` 函数，该函数接受一个 `age` 参数。然后，我们遍历数据集，并将满足条件的数据存储到 `filtered_people` 列表中。最后，我们将 `filtered_people` 列表转换为 JSON 格式并输出。
