# 2.核心概念与联系
在多云数据存储中，Bigtable是一种高性能、高可用性的分布式数据存储系统。它可以在多个地域之间进行数据复制，从而实现跨地域的数据存储和访问。Bigtable的核心概念包括：

- 分布式数据存储：Bigtable使用GFS（Google File System）进行数据存储，GFS是一种分布式文件系统，可以在多个数据中心之间进行数据存储和访问。
- 跨地域数据复制：Bigtable使用数据复制技术，可以在不同的地域之间进行数据复制，从而实现高可用性和低延迟。
- 数据分区：Bigtable使用数据分区技术，可以将数据划分为多个部分，每个部分可以在不同的地域进行存储和访问。
- 数据访问：Bigtable使用数据访问技术，可以在不同的地域之间进行数据访问，从而实现高性能和低延迟。
