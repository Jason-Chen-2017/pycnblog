# 2.核心概念与联系
在进入具体的算法原理之前，我们需要了解一些核心概念。

## 2.1 信息熵
信息熵是信息论中的一个重要概念，用于衡量信息的不确定性。信息熵的公式为：

$$
H(X) = -\sum_{i=1}^{n} p(x_i) \log_2 p(x_i)
$$

其中，$X$ 是一个随机变量，$x_i$ 是 $X$ 的可能取值，$p(x_i)$ 是 $x_i$ 的概率。信息熵的范围为 $0 \leq H(X) \leq \log_2 n$，当信息熵最大时，说明数据的不确定性最大，反之，当信息熵最小时，说明数据的不确定性最小。

## 2.2 条件信息熵
条件信息熵是信息熵的一种泛化，用于衡量已知某个条件下的信息不确定性。条件信息熵的公式为：

$$
H(X|Y) = -\sum_{j=1}^{m} p(y_j) \sum_{i=1}^{n} \frac{p(x_i|y_j)}{p(y_j)} \log_2 p(x_i|y_j)
$$

其中，$X$ 和 $Y$ 是两个随机变量，$x_i$ 和 $y_j$ 是 $X$ 和 $Y$ 的可能取值，$p(x_i|y_j)$ 是 $x_i$ 给定 $y_j$ 的概率。条件信息熵的范围为 $0 \leq H(X|Y) \leq \log_2 n$，当条件信息熵最大时，说明已知条件下的数据的不确定性最大，反之，当条件信息熵最小时，说明已知条件下的数据的不确定性最小。

## 2.3 相关系数
相关系数是用于衡量两个变量之间的相关性的一个度量。在灰色关联分析中，我们使用的是皮尔逊相关系数，其公式为：

$$
r(X,Y) = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n} (x_i - \bar{x})^2} \sqrt{\sum_{i=1}^{n} (y_i - \bar{y})^2}}
$$

其中，$X$ 和 $Y$ 是两个随机变量，$x_i$ 和 $y_i$ 是 $X$ 和 $Y$ 的可能取值，$\bar{x}$ 和 $\bar{y}$ 是 $X$ 和 $Y$ 的均值。相关系数的范围为 $-1 \leq r(X,Y) \leq 1$，当相关系数最大时，说明两个变量之间的相关性最强，反之，当相关系数最小时，说明两个变量之间的相关性最弱。
