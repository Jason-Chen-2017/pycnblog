# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的操作系统错误检测和恢复的代码实例来详细解释其工作原理。

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    int *p = (int *)malloc(sizeof(int));
    if (p == NULL) {
        // 错误检测：内存分配失败
        printf("内存分配失败\n");
        return 1;
    }
    *p = 10;
    printf("内存分配成功，值为 %d\n", *p);
    free(p);
    return 0;
}
```

在这个代码实例中，我们首先通过`malloc`函数分配了一个整型变量的内存空间。如果内存分配失败，我们将打印出"内存分配失败"的错误信息，并返回1。如果内存分配成功，我们将打印出"内存分配成功，值为 10"的信息，并将10赋值给`p`指针所指向的内存空间。最后，我们通过`free`函数释放了`p`指针所指向的内存空间。

这个代码实例中的错误检测和恢复过程如下：

1.错误检测：我们通过`malloc`函数的返回值来判断内存分配是否成功。如果返回值为`NULL`，则表示内存分配失败，我们将打印出错误信息。

2.错误恢复：我们通过`free`函数来释放内存空间，以防止内存泄漏。
