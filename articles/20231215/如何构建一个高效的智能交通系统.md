# 4.具体代码实例和详细解释说明

在这里，我们将通过一个简单的交通流量监控的代码实例来详细解释其实现过程。

```python
import cv2
import numpy as np

# 加载摄像头
cap = cv2.VideoCapture(0)

# 设置摄像头分辨率
cap.set(cv2.CAP_PROP_FRAME_WIDTH, 640)
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 480)

# 设置摄像头捕获速度
cap.set(cv2.CAP_PROP_FPS, 30)

# 循环捕获视频帧
while True:
    # 捕获一帧视频
    ret, frame = cap.read()

    # 如果捕获失败，则退出循环
    if not ret:
        break

    # 灰度化
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

    # 二值化
    _, binary = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

    # 显示原始图像
    cv2.imshow("Original", frame)

    # 显示二值化图像
    cv2.imshow("Binary", binary)

    # 按任意键退出
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# 释放摄像头资源
cap.release()

# 关闭所有窗口
cv2.destroyAllWindows()
```

在这个代码实例中，我们首先通过 `cv2.VideoCapture(0)` 函数加载摄像头。然后，我们通过 `cap.set()` 函数设置摄像头的分辨率和捕获速度。接下来，我们通过一个 `while` 循环捕获视频帧，并对其进行灰度化和二值化处理。最后，我们通过 `cv2.imshow()` 函数显示原始图像和二值化图像，并通过 `cv2.waitKey()` 函数等待用户按下任意键后退出程序。
