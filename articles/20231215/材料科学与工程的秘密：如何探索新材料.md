# 2.核心概念与联系

在材料科学与工程中，我们需要了解以下几个核心概念：

1. **材料性质**：材料性质是指材料具有的特征，如密度、硬度、挠性、热膨胀性等。材料性质对于材料的性能和应用具有重要影响。

2. **材料性能**：材料性能是指材料在特定条件下的表现，如强度、耐用性、热耐用性等。材料性能是材料的主要评价标准。

3. **材料应用**：材料应用是指材料在实际工程中的应用场景，如建筑、机械、化学等。材料应用对于材料的发展具有重要指导意义。

4. **材料模型**：材料模型是指用于描述材料性质和性能的数学模型。材料模型是材料科学与工程研究的基础。

5. **计算机科学与数学方法**：计算机科学与数学方法是指用于解决材料科学与工程问题的计算方法，如计算机模拟、数值解析、统计学等。计算机科学与数学方法是材料科学与工程研究的重要工具。
