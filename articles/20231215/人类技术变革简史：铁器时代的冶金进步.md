```python
import numpy as np
from scipy.optimize import minimize

def f(x):
    return np.sum(x**2)

def grad_f(x):
    return 2*x

x0 = np.array([1, 1])

result = minimize(f, x0, jac=grad_f)
print(result.x)
```

在这个代码实例中，我们使用了numpy和scipy库来实现一个简单的优化问题。我们定义了一个目标函数f(x)，它是一个二元函数，表示一个二维空间中的一个点。我们还定义了该函数的梯度grad_f(x)，它是一个一元函数，表示一个二维空间中的一个点的梯度。
