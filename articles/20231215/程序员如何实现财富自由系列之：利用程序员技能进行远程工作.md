## 4.1 代码实例

以下是一个简单的远程工作代码实例，用于计算两个数的和：

```python
def add(a, b):
    return a + b

if __name__ == '__main__':
    a = 10
    b = 20
    result = add(a, b)
    print(result)
```

## 4.2 代码解释

1. 定义一个名为`add`的函数，用于计算两个数的和。函数接受两个参数`a`和`b`，并返回它们的和。
2. 使用`if __name__ == '__main__':`语句来确保该代码只在主程序中执行。
3. 在主程序中，定义两个变量`a`和`b`，分别赋值为10和20。
4. 调用`add`函数，将`a`和`b`作为参数传递，并将结果存储在`result`变量中。
5. 使用`print`函数输出`result`的值。
