代码实例：

```python
import cv2
import numpy as np

# 读取图像

# 转换为灰度图像
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 高斯滤波
blur_image = cv2.GaussianBlur(gray_image, (5, 5), 0)

# 边缘检测
edges = cv2.Canny(blur_image, 50, 150)

# 显示图像
cv2.imshow('edges', edges)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

1. 首先，我们需要读取图像，使用`cv2.imread()`函数来实现。

3. 接下来，我们需要对灰度图像进行高斯滤波，使用`cv2.GaussianBlur()`函数来实现。

4. 然后，我们需要对高斯滤波后的图像进行边缘检测，使用`cv2.Canny()`函数来实现。
