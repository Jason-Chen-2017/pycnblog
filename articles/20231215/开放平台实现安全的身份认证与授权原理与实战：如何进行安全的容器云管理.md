# 2.核心概念与联系
在实现安全的容器云管理之前，我们需要了解一些核心概念，包括身份认证、授权、容器、云计算等。

## 2.1 身份认证
身份认证是确认某个用户是否是一个特定实体的过程。在容器云管理中，身份认证是确认用户是否具有访问某个容器或云资源的权限。

## 2.2 授权
授权是确定用户是否具有执行某个操作的权限的过程。在容器云管理中，授权是确定用户是否具有访问、修改或删除某个容器或云资源的权限。

## 2.3 容器
容器是一种轻量级的应用程序部署和运行的方式，它可以将应用程序和其依赖项打包到一个可移植的镜像中，并在运行时与宿主操作系统共享资源。

## 2.4 云计算
云计算是一种基于网络的计算模式，它允许用户在网络上获取计算资源，而无需购买和维护自己的硬件和软件。云计算可以提供各种服务，如计算服务、存储服务、数据库服务等。
