## 2.核心概念与联系

### 2.1 因果关系的定义

因果关系是指一个变量对另一个变量的影响。例如，一个人的体重可能会影响他的运动能力。在这个例子中，体重是因变量，运动能力是因果关系。

### 2.2 因果关系与其他概念的联系

因果关系与其他数据分析概念有密切的联系，例如：

- 相关性：相关性是指两个变量之间的关系，但它并不能证明一个变量是另一个变量的因素。相关性只能说明两个变量之间存在某种关系，但不能说明因果关系。
- 因果推理：因果推理是一种从观察到的数据中推断出的因果关系的方法。它通过利用已知的因果关系来推断未知的因果关系。
- 机器学习：机器学习是一种通过从数据中学习的方法来预测和决策的方法。它可以用来发现因果关系，但它也可以用来发现其他类型的关系，例如相关性。

import pandas as pd
from sklearn.linear_model import LinearRegression

# 加载数据
data = pd.read_csv('data.csv')

# 划分训练集和测试集
X_train = data.iloc[:int(len(data)*0.8), :-1]
Y_train = data.iloc[:int(len(data)*0.8), -1]
X_test = data.iloc[int(len(data)*0.8):, :-1]
Y_test = data.iloc[int(len(data)*0.8):, -1]

# 训练模型
model = LinearRegression()
model.fit(X_train, Y_train)

# 预测
Y_pred = model.predict(X_test)

# 评估模型
score = model.score(X_test, Y_test)
print('模型评分：', score)
