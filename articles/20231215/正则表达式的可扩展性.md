# 2.核心概念与联系

# 4.具体代码实例和详细解释说明
正则表达式的可扩展性可以通过代码实例来说明。以下是一个使用Python的re模块实现正则表达式匹配的示例：

```python
import re

# 定义一个正则表达式模式
pattern = r'\d{3}-\d{2}-\d{4}'

# 定义一个字符串
string = '123-45-6789'

# 使用正则表达式匹配字符串
match = re.match(pattern, string)

# 检查匹配结果
if match:
    print('匹配成功')
else:
    print('匹配失败')
```

在这个示例中，我们使用Python的re模块实现了一个简单的正则表达式匹配。我们定义了一个正则表达式模式`\d{3}-\d{2}-\d{4}`，它表示一个包含三个数字、一个连字符、两个数字和一个连字符的字符串。我们定义了一个字符串`123-45-6789`，然后使用`re.match()`函数进行匹配。最后，我们检查匹配结果，如果匹配成功，则打印`匹配成功`，否则打印`匹配失败`。
