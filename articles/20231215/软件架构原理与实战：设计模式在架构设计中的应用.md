### 4.1 代码实例

本节将给出一些具体的代码实例，以帮助读者更好地理解设计模式的实现方法。

#### 4.1.1 单例模式

单例模式是一种设计模式，它限制了一个类的实例数量，只允许存在一个实例。单例模式可以用来保证系统中只有一个实例，从而避免多个实例之间的冲突。

```python
class Singleton:
    _instance = None

    @classmethod
    def get_instance(cls):
        if cls._instance is None:
            cls._instance = cls()
        return cls._instance

    def do_something(self):
        pass

a = Singleton.get_instance()
b = Singleton.get_instance()
print(a == b)  # True
```

#### 4.1.2 工厂模式

工厂模式是一种设计模式，它定义了一个创建对象的接口，让子类决定哪个类实例化。工厂模式可以用来创建不同类型的对象，从而避免直接创建对象的代码重复。

```python
class Factory:
    def create(self, type):
        if type == 'a':
            return A()
        elif type == 'b':
            return B()
        else:
            return None

class A:
    pass

class B:
    pass

factory = Factory()
a = factory.create('a')
b = factory.create('b')
```
