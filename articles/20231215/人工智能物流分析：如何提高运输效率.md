## 4.3 优化算法

### 4.3.1 线性规划

在Python中，可以使用pulp库进行线性规划优化。以下是一个简单线性规划的代码实例：

```python
from pulp import LpProblem, LpMinimize, LpMaximize, LpVariable

problem = LpProblem("Transportation Problem", LpMinimize)

x1 = LpVariable("x1", lowBound=0)
x2 = LpVariable("x2", lowBound=0)

problem += x1 + 2 * x2

problem.solve()

print(x1.value(), x2.value())
```

### 4.3.2 非线性规划

在Python中，可以使用scipy库进行非线性规划优化。以下是一个非线性规划的代码实例：

```python
from scipy.optimize import minimize

def objective_function(x):
    return x[0]**2 + x[1]**2

def constraint_function(x):
    return x[0] + x[1] - 10

x0 = [1, 1]

result = minimize(objective_function, x0, constraints=[{'type': 'eq', 'fun': constraint_function}])

print(result.x)
```

### 4.3.3 遗传算法

在Python中，可以使用deap库进行遗传算法优化。以下是一个遗传算法的代码实例：

```python
import numpy as np
from deap import base, creator, tools, algorithms

creator.create("FitnessMax", base.Fitness, weights=(1.0,))
creator.create("Individual", list, fitness=creator.FitnessMax)

toolbox = base.Toolbox()
toolbox.register("attr_float", np.random.uniform, 0, 1)
toolbox.register("individual", tools.initRepeat, creator.Individual, toolbox.attr_float, 10)
toolbox.register("population", tools.initRepeat, list, toolbox.individual)

toolbox.register("evaluate", lambda ind: ind)
toolbox.register("mate", tools.cxTwoPoint)
toolbox.register("mutate", tools.mutGaussian, mu=0, sigma=1, indpb=0.5)

def optimize():
    pop = toolbox.population(n=50)
    hof = tools.HallOfFame(1)
    stats = tools.Statistics(lambda ind: ind.fitness.values)
    stats.register("avg", np.mean)

    pop, logbook = algorithms.eaSimple(pop, toolbox, 0.5, 0.2, 10, hof, stats=stats)

    return hof[0]

print(optimize())
```
