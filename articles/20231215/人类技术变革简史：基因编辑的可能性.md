# 2.核心概念与联系

在探讨基因编辑之前，我们需要了解一些基本的概念。首先，我们需要了解什么是基因，什么是基因编辑。其次，我们需要了解基因编辑的原理和技术。

## 2.1 基因和基因编辑

基因是生物体的遗传信息，它们存储在DNA（分子生物学中的一种双链螺旋结构）中。每个基因都包含一段特定的DNA序列，这个序列决定了生物体的特征和性质。基因编辑是一种技术，它可以修改生物体的基因序列，从而改变其特征和性质。

## 2.2 基因编辑的原理和技术

基因编辑的原理是通过对生物体的基因序列进行修改，从而改变其特征和性质。这种技术的主要方法有CRISPR/Cas9和TALEN等。CRISPR/Cas9是一种基因编辑技术，它可以通过将特定的RNA和Cas9蛋白结合在一起，来精确地修改生物体的基因序列。TALEN是一种基因编辑技术，它通过将特定的TAL效应器和DNA结合在一起，来修改生物体的基因序列。
