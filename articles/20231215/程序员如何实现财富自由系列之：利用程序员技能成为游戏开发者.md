## 4.1 游戏逻辑的实现

以下是一个简单的游戏逻辑的实现代码示例：

```python
import random

class Game:
    def __init__(self):
        self.player = Player()
        self.enemy = Enemy()

    def start(self):
        while True:
            action = self.player.choose_action()
            if action == "attack":
                self.player.attack(self.enemy)
            elif action == "defend":
                self.player.defend()
            elif action == "run":
                self.player.run()
            else:
                print("Invalid action")
                continue

            if self.enemy.is_dead():
                print("You win!")
                break
            else:
                if self.player.is_dead():
                    print("You lose!")
                    break

                self.enemy.attack(self.player)

class Player:
    def __init__(self):
        self.health = 100
        self.mana = 100

    def choose_action(self):
        action = input("What do you want to do? (attack, defend, run): ")
        return action

    def attack(self, enemy):
        damage = random.randint(10, 20)
        enemy.take_damage(damage)

    def defend(self):
        self.health += 5

    def run(self):
        self.health += 10

class Enemy:
    def __init__(self):
        self.health = 100

    def attack(self, player):
        damage = random.randint(10, 20)
        player.take_damage(damage)

    def is_dead(self):
        return self.health <= 0

    def take_damage(self, damage):
        self.health -= damage

if __name__ == "__main__":
    game = Game()
    game.start()
```

这个代码示例实现了一个简单的游戏逻辑。游戏中有一个玩家和一个敌人。玩家可以选择攻击、防御或逃跑。玩家的攻击和防御会随机产生不同的伤害值。玩家的血量和魔法值会随着时间的推移而变化。敌人会根据玩家的攻击来进行回击。当敌人的血量为0时，游戏结束。
