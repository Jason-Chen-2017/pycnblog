```prolog

```

代码实例2：计算一个数字的平方

```prolog
% 定义一个函数来计算一个数字的平方
square(X, Y) :- Y is X * X.

% 输入一个数字，然后通过函数来计算该数字的平方
?- square(4, X).
% 输出：X = 16
```

```prolog
% 定义一个谓词来判断一个数字是否是素数
prime(X) :- X > 1, X mod Y \= 0, Y <- 2, X is Y + 1.
```
