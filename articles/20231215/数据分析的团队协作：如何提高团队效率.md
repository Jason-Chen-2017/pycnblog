# 4.具体代码实例和详细解释说明

## 4.1 数据收集

```python
import requests
from bs4 import BeautifulSoup

url = 'https://www.example.com'
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

data = []
for item in soup.find_all('div', class_='item'):
    title = item.find('h2').text
    price = item.find('span', class_='price').text
    data.append({
        'title': title,
        'price': price
    })
```

在这个代码实例中，我们使用了`requests`库来发送HTTP请求，并使用了`BeautifulSoup`库来解析HTML内容。我们找到了所有的`div`元素，并提取了标题和价格。最后，我们将这些数据存储到了一个列表中。

## 4.2 数据清洗

```python
import pandas as pd

data = pd.DataFrame(data)
data.fillna(method='ffill', inplace=True)
data.replace('', np.nan, inplace=True)
data.dropna(inplace=True)
```

在这个代码实例中，我们使用了`pandas`库来创建一个DataFrame，并使用了`fillna`方法来填充缺失值，使用了`replace`方法来替换错误值，并使用了`dropna`方法来删除不一致的值。
