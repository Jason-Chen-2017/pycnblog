我们可以使用Python的MLxtend库来进行关联规则挖掘。以下是一个简单的关联规则挖掘代码实例：

```python
from mlxtend.frequent_patterns import apriori
from mlxtend.frequent_patterns import association_rules

# 加载客户购买数据
purchases = pd.read_csv('customer_purchases.csv')

# 对数据进行预处理
preprocessed_purchases = preprocess_purchases(purchases)

# 使用Apriori算法进行关联规则挖掘
frequent_itemsets = apriori(preprocessed_purchases, min_support=0.1, use_colnames=True)
rules = association_rules(frequent_itemsets, metric="confidence", min_threshold=0.7)

# 获取关联规则
association_rules = rules.drop(columns=['lift', 'confidence', 'support', 'count', 'original_rules'])
```

在上述代码中，我们首先加载客户购买数据，然后对数据进行预处理。接着，我们使用Apriori算法进行关联规则挖掘，并获取关联规则。
