## 1.4 具体代码实例和详细解释说明

### 1.4.1 代码实例
以下是一个简单的水能技术的代码实例：

```python
import math

# 水能技术的能量公式
def water_energy(mass, gravity, height):
    return mass * gravity * height

# 水能技术的功率公式
def water_power(energy, time):
    return energy / time

# 水能技术的效率公式
def water_efficiency(power, input_power):
    return power / input_power

# 主函数
def main():
    mass = 1000  # 水的质量，单位为克
    gravity = 9.8  # 重力加速度，单位为米/秒^2
    height = 10  # 水的高度，单位为米
    time = 1  # 时间，单位为秒
    input_power = 1000  # 水能技术的输入功率，单位为瓦特

    # 计算水能技术的能量
    energy = water_energy(mass, gravity, height)
    print("水能技术的能量：", energy, "瓦特·秒")

    # 计算水能技术的功率
    power = water_power(energy, time)
    print("水能技术的功率：", power, "瓦特")

    # 计算水能技术的效率
    efficiency = water_efficiency(power, input_power)
    print("水能技术的效率：", efficiency, "％")

if __name__ == "__main__":
    main()
```
