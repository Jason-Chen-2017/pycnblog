## 4.1 游戏例子：贪吃蛇
我们将通过一个简单的贪吃蛇游戏来详细解释游戏开发的具体代码实例。贪吃蛇游戏是一种经典的游戏，它的目标是让玩家控制一个蛇来吃食物，同时避免撞到障碍物。

### 4.1.1 游戏规则
贪吃蛇游戏的规则如下：

1. 玩家使用键盘控制蛇的移动方向。
2. 蛇会不断地移动，吃掉食物以增长自己的长度。
3. 如果蛇撞到障碍物或者自己的身体，游戏结束。

### 4.1.2 游戏场景
贪吃蛇游戏的场景包括：

1. 一个二维的游戏场景，用于显示蛇和食物。
2. 一个游戏界面，用于显示游戏的规则和进度。

### 4.1.3 游戏代码
我们将使用Python编程语言来实现贪吃蛇游戏。以下是游戏的主要代码实例：

```python
import pygame
import sys

# 初始化游戏
pygame.init()

# 设置游戏窗口大小
screen_width = 800
screen_height = 600
screen = pygame.display.set_mode((screen_width, screen_height))

# 设置游戏标题
pygame.display.set_caption("贪吃蛇")

# 设置游戏背景颜色
bg_color = (255, 255, 255)
screen.fill(bg_color)

# 设置蛇的初始位置和速度
snake_x = 100
snake_y = 100
snake_speed = 10

# 设置食物的初始位置
food_x = 400
food_y = 400

# 设置游戏循环
game_over = False
while not game_over:
    # 处理用户输入
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            game_over = True

        # 处理蛇的移动
        if event.type == pygame.KEYDOWN:
            if event.key == pygame.K_UP:
                snake_y -= snake_speed
            elif event.key == pygame.K_DOWN:
                snake_y += snake_speed
            elif event.key == pygame.K_LEFT:
                snake_x -= snake_speed
            elif event.key == pygame.K_RIGHT:
                snake_x += snake_speed

    # 更新蛇的位置
    snake_x += snake_speed * (pygame.key.get_pressed()[pygame.K_LEFT] - pygame.key.get_pressed()[pygame.K_RIGHT])
    snake_y += snake_speed * (pygame.key.get_pressed()[pygame.K_UP] - pygame.key.get_pressed()[pygame.K_DOWN])

    # 检查蛇是否撞到障碍物或者自己的身体
    if snake_x < 0 or snake_x >= screen_width or snake_y < 0 or snake_y >= screen_height:
        game_over = True

    # 绘制蛇和食物
    screen.fill(bg_color)
    pygame.draw.rect(screen, (0, 255, 0), pygame.Rect(snake_x, snake_y, 20, 20))
    pygame.draw.rect(screen, (255, 0, 0), pygame.Rect(food_x, food_y, 20, 20))

    # 更新游戏窗口
    pygame.display.flip()

# 结束游戏
pygame.quit()
sys.exit()
```

在这个代码中，我们首先初始化游戏并设置游戏窗口的大小。然后，我们设置游戏的标题和背景颜色。接着，我们设置蛇的初始位置和速度，以及食物的初始位置。

在游戏循环中，我们处理用户输入并更新蛇的位置。我们还检查蛇是否撞到障碍物或者自己的身体。最后，我们绘制蛇和食物，并更新游戏窗口。
