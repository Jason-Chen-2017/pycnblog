1. 线性回归模型：线性回归模型是一种用于预测水质指标变化的模型，它可以用来预测水质变化趋势，从而提高水质保护工作的效果。线性回归模型的公式为：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon
$$

其中，$y$ 是预测的水质指标，$x_1, x_2, ..., x_n$ 是输入变量，$\beta_0, \beta_1, ..., \beta_n$ 是模型参数，$\epsilon$ 是误差项。

2. 支持向量机模型：支持向量机模型是一种用于分类和回归的模型，它可以用来预测水质污染源，从而进行水质污染源控制。支持向量机模型的公式为：

$$
f(x) = \text{sgn} \left( \sum_{i=1}^n \alpha_i y_i K(x_i, x) + b \right)
$$

其中，$f(x)$ 是预测的水质指标，$x$ 是输入变量，$y_i$ 是标签，$K(x_i, x)$ 是核函数，$\alpha_i$ 是模型参数，$b$ 是偏置项。

3. 随机森林模型：随机森林模型是一种用于预测和分类的模型，它可以用来预测水质变化趋势，从而提高水质保护工作的效果。随机森林模型的公式为：

$$
\hat{y} = \frac{1}{m} \sum_{i=1}^m f_i(x)
$$

其中，$\hat{y}$ 是预测的水质指标，$x$ 是输入变量，$f_i(x)$ 是单个决策树的预测结果，$m$ 是决策树的数量。

```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression

# 数据预处理
data = pd.read_csv('water_quality_data.csv')
X = data.iloc[:, :-1].values
y = data.iloc[:, -1].values

# 数据分析

regressor.fit(X, y)

# 模型评估

r_squared = regressor.score(X, y)
```
