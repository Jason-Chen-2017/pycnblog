# 4.具体代码实例和详细解释说明

以下是一个简单的Python程序的实例，用于计算两个数的和：

```python
# 定义两个数
a = 10
b = 20

# 计算两个数的和
c = a + b

# 输出结果
print(c)
```

在这个实例中，我们首先定义了两个数a和b。然后，我们使用加法运算符+来计算两个数的和，并将结果存储在变量c中。最后，我们使用print函数输出结果。

这个实例展示了Python的基本语法和运算符的使用。通过这个实例，我们可以看到Python的简单性和易用性。
