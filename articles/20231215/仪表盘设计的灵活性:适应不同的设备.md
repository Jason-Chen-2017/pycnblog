## 4.3 交互设计

以下是一个使用Python的Tkinter库进行交互设计的示例代码：

```python
import tkinter as tk

# 创建窗口
root = tk.Tk()

# 创建拖动控件
drag_canvas = tk.Canvas(root, width=400, height=400)
drag_canvas.pack()

# 创建拖动事件处理器
def on_drag(event):
    x = event.x
    y = event.y
    # 更新图表位置
    drag_canvas.coords('drag_rectangle', x, y, x+100, y+100)

# 创建拖动控件
drag_rectangle = tk.Rectangle(drag_canvas, width=100, height=100, fill='blue')
drag_rectangle.bind('<B1-Motion>', on_drag)
drag_canvas.create_window(200, 200, window=drag_rectangle)

# 显示窗口
root.mainloop()
```
