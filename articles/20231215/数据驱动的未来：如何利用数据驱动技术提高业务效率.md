```python
# 数据清洗
data = data.dropna()

# 数据转换

# 数据整理

```

## 4.3 数据分析

```python
# 统计分析

# 机器学习
from sklearn.linear_model import LinearRegression

# 训练模型
model = LinearRegression()
model.fit(data[['income']], data['age'])

# 预测
pred = model.predict(data[['income']])
```
