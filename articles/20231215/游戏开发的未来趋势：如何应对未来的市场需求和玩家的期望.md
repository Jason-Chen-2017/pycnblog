```python
import numpy as np

class NPC:
    def __init__(self, state):
        self.state = state

    def decide(self, environment):
        if environment == "danger":
            self.state = "flee"
        else:
            self.state = "approach"

npc = NPC("neutral")
environment = "danger"
npc.decide(environment)
print(npc.state)
```

在这个代码实例中，我们创建了一个简单的NPC类，用于表示游戏中的一个非人角色。NPC类有一个状态属性，用于表示NPC的当前状态。我们还定义了一个decide方法，用于根据环境来更新NPC的状态。

在主程序中，我们创建了一个NPC实例，并将其状态设置为"neutral"。然后，我们设置了一个环境变量，并将其传递给NPC实例的decide方法。最后，我们打印出NPC的状态。
