1. Python代码：

```python
import numpy as np
from astropy.io import fits
from astropy.table import Table

# 读取光谱数据
data = fits.open('spectrum_data.fits')
spectrum_data = Table(data[1].data)

# 分析光谱数据
wavelength = spectrum_data['wavelength']
flux = spectrum_data['flux']

# 计算光谱分布
spectrum_distribution = flux / wavelength

# 保存分析结果
spectrum_distribution.write('spectrum_distribution.fits', overwrite=True)
```
