# 2.核心概念与联系
## 2.1 安全架构
安全架构是指在网络中部署各种安全设备和技术，以保护网络资源和数据的安全架构。安全架构包括防火墙、IDS/IPS、安全网关、安全操作系统等多种技术。

网络防护是指通过安全架构来保护网络资源和数据的过程。网络防护涉及到多种技术，如防火墙、IDS/IPS、安全网关等。
