# 2.核心概念与联系

词向量是一种用于表示词汇在语义上的数学模型，它将词汇表示为一个高维的数学空间，以便计算机可以对文本进行分析和处理。词向量可以帮助计算机理解语言的结构和语义，从而实现自然语言处理的目标。

词向量的核心思想是，相似的词汇在词向量空间中应该接近于彼此，而不相似的词汇则应该较远离。这意味着，我们可以通过计算词向量之间的距离来衡量它们之间的相似性。
