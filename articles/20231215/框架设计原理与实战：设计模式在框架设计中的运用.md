## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释框架设计的实现过程。

### 4.1代码实例

我们将通过一个简单的日志记录框架来详细解释框架设计的实现过程。

```python
# 核心部分
class Logger:
    def __init__(self):
        self.log_list = []

    def log(self, message):
        self.log_list.append(message)

# 插件系统
class PluginInterface:
    def log(self, message):
        pass

class PluginA(PluginInterface):
    def log(self, message):
        print(f"PluginA: {message}")

class PluginB(PluginInterface):
    def log(self, message):
        print(f"PluginB: {message}")

# 插件加载器
class PluginLoader:
    def __init__(self):
        self.plugins = []

    def load_plugin(self, plugin_class):
        self.plugins.append(plugin_class())

    def log(self, message):
        for plugin in self.plugins:
            plugin.log(message)

# 使用框架
logger = Logger()
logger.log("Hello, World!")

plugin_loader = PluginLoader()
plugin_loader.load_plugin(PluginA)
plugin_loader.load_plugin(PluginB)
plugin_loader.log("Hello, Plugins!")
```

在这个代码实例中，我们创建了一个简单的日志记录框架。框架的核心部分包含一个 `Logger` 类，它提供了日志记录的基本功能。插件系统包含一个 `PluginInterface` 接口，以及两个实现类 `PluginA` 和 `PluginB`。插件加载器负责加载和管理插件。
