```sql
    FOREIGN KEY (customer_id) REFERENCES customers(customer_id)
);
```

为了优化数据库性能，我们可以采取以下措施：

1. 使用B+树索引优化查询性能。我们可以为`order_id`、`customer_id`和`order_date`添加B+树索引，以提高查询性能。

```sql
CREATE INDEX idx_order_id ON orders(order_id);
CREATE INDEX idx_customer_id ON orders(customer_id);
CREATE INDEX idx_order_date ON orders(order_date);
```

2. 使用哈希索引优化更新性能。我们可以为`order_total`添加哈希索引，以提高更新性能。

```sql
CREATE INDEX idx_order_total ON orders(order_total) USING HASH;
```
