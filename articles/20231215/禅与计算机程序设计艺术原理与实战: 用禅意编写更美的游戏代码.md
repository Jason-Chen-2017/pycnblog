## 4.1 代码实例

以下是一个简单的游戏代码实例，用于实现一个简单的游戏：

```python
import pygame

# 初始化游戏
pygame.init()

# 设置游戏窗口大小
screen_width = 800
screen_height = 600
screen = pygame.display.set_mode((screen_width, screen_height))

# 设置游戏循环
running = True
while running:
    # 处理游戏事件
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    # 更新游戏屏幕
    screen.fill((0, 0, 0))
    pygame.display.flip()

# 结束游戏
pygame.quit()
```

## 4.2 代码解释

上述代码实例是一个简单的游戏代码，用于实现一个简单的游戏。代码的主要功能是初始化游戏、设置游戏窗口大小、设置游戏循环、处理游戏事件、更新游戏屏幕等。

1. 首先，导入pygame库，用于实现游戏功能。

2. 然后，初始化游戏，使用pygame.init()函数进行初始化。

3. 设置游戏窗口大小，使用screen_width和screen_height变量来设置游戏窗口的宽度和高度。

4. 设置游戏循环，使用running变量来控制游戏循环的执行。

5. 处理游戏事件，使用pygame.event.get()函数来获取游戏事件，然后根据事件类型进行处理。

6. 更新游戏屏幕，使用screen.fill((0, 0, 0))函数来填充游戏屏幕的颜色，然后使用pygame.display.flip()函数来更新游戏屏幕。

7. 结束游戏，使用pygame.quit()函数来结束游戏。
