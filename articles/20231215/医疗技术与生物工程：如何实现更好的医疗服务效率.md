```python
def assembly(reads):
    contigs = []
    for read in reads:
        if not contigs or contigs[-1][-1] != read[0]:
            contigs.append([read[0]])
        contigs[-1].append(read[1])
    return contigs
```
这个代码实例是一个基因组组装的函数。它接受一个读取序列列表作为输入，并将这些读取序列组装成完整的基因组序列。具体的步骤包括：

1. 遍历读取序列列表。
2. 如果 contigs 列表为空或者最后一个 contigs 元素的最后一个元素与当前读取序列的第一个元素不相等，则创建一个新的 contigs 元素，并将当前读取序列的第一个元素作为其第一个元素。
3. 将当前读取序列的第二个元素添加到最后一个 contigs 元素的末尾。
4. 返回 contigs 列表。
