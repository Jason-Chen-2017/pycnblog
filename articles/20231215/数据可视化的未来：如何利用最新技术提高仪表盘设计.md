# 2.核心概念与联系

在数据可视化中，仪表盘是一个重要的组成部分。仪表盘是一种用于可视化数据的图形界面，可以帮助用户更好地理解数据。仪表盘通常包括一个或多个图表，用于显示数据的趋势和变化。
