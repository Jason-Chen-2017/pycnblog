```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 创建一个量子电路
qc = QuantumCircuit(2)

# 添加一个H门到第一个量子比特
qc.h(0)

# 添加一个CNOT门到第一个量子比特和第二个量子比特
qc.cx(0, 1)

# 将量子电路编译为可执行的量子电路
qc = transpile(qc, basis_gates=['u', 'cx', 'h'])

# 使用Aer后端执行量子电路
simulator = Aer.get_backend('qasm_simulator')
job = simulator.run(qc)

# 获取量子电路的结果
result = job.result()
counts = result.get_counts()

# 绘制结果
plot_histogram(counts)
```

在这个代码实例中，我们首先创建了一个量子电路，并添加了一个H门和一个CNOT门。然后，我们将量子电路编译为可执行的量子电路，以便在后端上执行。最后，我们使用Aer后端执行量子电路，并绘制结果。
