# 2.核心概念与联系

在大数据智能决策系统架构中，核心概念包括：

- 大数据：大数据是指由于互联网、物联网等因素的数据量大、速度快、结构复杂等特点，使得传统数据处理技术难以应对的数据。大数据包括结构化数据（如关系型数据库）、半结构化数据（如XML、JSON）和非结构化数据（如文本、图像、视频等）。
- 智能决策：智能决策是指通过利用人工智能技术（如机器学习、深度学习、规则引擎等）来自动化决策过程的决策。智能决策可以提高决策的效率和准确性，降低人工干预的成本。
- 决策支持系统：决策支持系统是一种帮助企业人员进行决策的信息系统。决策支持系统可以提供决策相关的数据、知识和建议，以帮助企业人员更好地做出决策。

大数据智能决策系统架构的核心概念之间存在着紧密的联系。大数据是决策支持系统所需的数据来源，智能决策是决策支持系统的核心功能，大数据智能决策系统架构是将这两者相结合的一种系统架构。
