## 4.2 自然语言处理
以下是一个简单的自然语言处理代码实例：

```python
import numpy as np
import tensorflow as tf
from tensorflow.keras.preprocessing.text import Tokenizer
from tensorflow.keras.preprocessing.sequence import pad_sequences

# 文本数据
text = "这是一个简单的文本数据"

# 分词
tokenizer = Tokenizer()
tokenizer.fit_on_texts([text])
word_index = tokenizer.word_index
sequences = tokenizer.texts_to_sequences([text])

# 填充
max_length = 10
padded = pad_sequences(sequences, maxlen=max_length)

# 模型
model = tf.keras.Sequential([
    tf.keras.layers.Embedding(len(word_index) + 1, 16),
    tf.keras.layers.LSTM(32),
    tf.keras.layers.Dense(1, activation='sigmoid')
])

# 编译模型
model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

# 训练模型
model.fit(padded, np.array([1]), epochs=10, batch_size=32)
```

这个代码实例主要包括以下几个步骤：

1. 导入所需的库，例如numpy和tensorflow。
2. 定义一个文本数据。
3. 分词，使用Tokenizer将文本数据转换为序列。
4. 填充，使用pad_sequences将序列填充到固定长度。
5. 定义一个自然语言处理模型，例如一个Sequential模型，包含Embedding、LSTM和Dense层。
6. 编译模型，指定优化器、损失函数和评估指标。
7. 训练模型，使用填充后的序列进行迭代训练。

## 4.3 计算机视觉
以下是一个简单的计算机视觉代码实例：

```python
import numpy as np
import tensorflow as tf
from tensorflow.keras.preprocessing.image import ImageDataGenerator

# 图像数据

# 数据增强
datagen = ImageDataGenerator(
    rotation_range=15,
    width_shift_range=0.1,
    height_shift_range=0.1,
    horizontal_flip=True
)

# 生成图像流
generator = datagen.flow_from_directory(
    directory="path/to/directory",
    target_size=(224, 224),
    batch_size=32,
    class_mode='binary'
)

# 模型
model = tf.keras.Sequential([
    tf.keras.layers.Conv2D(32, (3, 3), activation='relu', input_shape=(224, 224, 3)),
    tf.keras.layers.MaxPooling2D((2, 2)),
    tf.keras.layers.Conv2D(64, (3, 3), activation='relu'),
    tf.keras.layers.MaxPooling2D((2, 2)),
    tf.keras.layers.Conv2D(128, (3, 3), activation='relu'),
    tf.keras.layers.MaxPooling2D((2, 2)),
    tf.keras.layers.Flatten(),
    tf.keras.layers.Dense(128, activation='relu'),
    tf.keras.layers.Dense(1, activation='sigmoid')
])

# 编译模型
model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

# 训练模型
model.fit_generator(
    generator,
    steps_per_epoch=100,
    epochs=10,
    verbose=1
)
```

这个代码实例主要包括以下几个步骤：

1. 导入所需的库，例如numpy和tensorflow。
2. 定义一个图像数据。
3. 数据增强，使用ImageDataGenerator对图像进行旋转、平移、缩放等操作。
4. 生成图像流，使用flow_from_directory方法从指定目录读取图像数据。
5. 定义一个计算机视觉模型，例如一个Sequential模型，包含Conv2D、MaxPooling2D、Flatten和Dense层。
6. 编译模型，指定优化器、损失函数和评估指标。
7. 训练模型，使用生成的图像流进行迭代训练。
