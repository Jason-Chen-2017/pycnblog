# 2.核心概念与联系

## 2.2 Socket通信的核心概念之Socket
Socket是Java网络编程中的核心概念之一，它是一个抽象的网络通信端点，用于实现客户端与服务器之间的数据传输。Socket通信是基于TCP/IP协议的，它可以实现双向通信。

## 2.3 Socket通信的核心概念之ServerSocket
ServerSocket是Java网络编程中的核心概念之一，它是一个用于监听客户端连接的Socket对象。ServerSocket可以监听指定的端口号，当客户端连接时，ServerSocket会创建一个新的Socket对象，并与客户端建立连接。

## 2.4 Socket通信的核心概念之InetAddress
InetAddress是Java网络编程中的核心概念之一，它是一个表示IP地址的对象。InetAddress可以用于获取IP地址、解析域名等操作。

## 2.5 Socket通信的核心概念之DatagramSocket
DatagramSocket是Java网络编程中的核心概念之一，它是一个用于实现UDP通信的Socket对象。DatagramSocket可以发送和接收数据包，但是它不保证数据包的顺序和完整性。

## 2.6 Socket通信的核心概念之TCP/IP协议
TCP/IP协议是Java网络编程中的核心概念之一，它是一种网络通信协议，用于实现可靠的数据传输。TCP/IP协议包括三层：应用层、传输层和网络层。

## 2.7 Socket通信的核心概念之UDP协议
UDP协议是Java网络编程中的核心概念之一，它是一种网络通信协议，用于实现无连接的数据传输。UDP协议不保证数据包的顺序和完整性，但是它具有较高的传输速度和低延迟。

## 2.8 Socket通信的核心概念之HTTP协议
HTTP协议是Java网络编程中的核心概念之一，它是一种用于实现网页浏览和网络通信的应用层协议。HTTP协议包括请求方法、请求头部、请求体、状态行、状态码、状态描述等组成部分。
