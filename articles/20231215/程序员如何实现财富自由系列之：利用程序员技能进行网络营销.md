## 4.1 SEO 优化代码实例
```python
import requests
from bs4 import BeautifulSoup

def get_html(url):
    response = requests.get(url)
    return response.text

def parse_html(html):
    soup = BeautifulSoup(html, 'html.parser')
    titles = soup.find_all('title')
    return titles

def optimize_title(title):
    words = title.split()
    words = [word.lower() for word in words]
    words = [word for word in words if word in ['programmer', 'financial', 'freedom']]
    return ' '.join(words)

def main():
    url = 'https://www.example.com'
    html = get_html(url)
    titles = parse_html(html)
    optimized_title = optimize_title(titles[0].text)
    print(optimized_title)

if __name__ == '__main__':
    main()
```
这段代码的实现原理是：首先，通过 `requests` 库获取网页的 HTML 内容，然后通过 `BeautifulSoup` 库解析 HTML 内容，找到所有的 `title` 标签。接着，通过 `optimize_title` 函数对标题进行关键词优化，将关键词 'programmer', 'financial', 'freedom' 添加到标题中。最后，打印出优化后的标题。

## 4.2 社交媒体营销代码实例
```python
import time
import random
from weibo import WeiboAPI

def get_hot_topics():
    hot_topics = ['编程', '财务自由', '技术']
    return random.choice(hot_topics)

def post_weibo(content):
    api = WeiboAPI(app_key, app_secret, access_token)
    api.comment(content)

def main():
    while True:
        hot_topic = get_hot_topics()
        post_weibo(hot_topic)
        time.sleep(random.randint(1, 60))

if __name__ == '__main__':
    app_key = 'your_app_key'
    app_secret = 'your_app_secret'
    access_token = 'your_access_token'
    main()
```
这段代码的实现原理是：首先，通过 `WeiboAPI` 库获取微博的 API 接口，然后通过 `get_hot_topics` 函数获取热门话题。接着，通过 `post_weibo` 函数发布微博，将热门话题作为微博内容。最后，通过 `time.sleep` 函数随机休眠一段时间，避免过快发布微博。

## 4.3 电子邮件营销代码实例
```python
import smtplib
from email.mime.text import MIMEText

def send_email(to_addr, subject, content):
    msg = MIMEText(content)
    msg['Subject'] = subject
    msg['From'] = 'your_email@example.com'
    msg['To'] = to_addr

    server = smtplib.SMTP('smtp.example.com', 25)
    server.starttls()
    server.login('your_email@example.com', 'your_password')
    server.sendmail('your_email@example.com', to_addr, msg.as_string())
    server.quit()

def main():
    to_addr = 'recipient@example.com'
    subject = 'Hello'
    content = 'Hello, this is a test email.'
    send_email(to_addr, subject, content)

if __name__ == '__main__':
    main()
```
这段代码的实现原理是：首先，通过 `smtplib` 库获取 SMTP 服务器，然后通过 `MIMEText` 类创建邮件内容。接着，通过 `send_email` 函数发送邮件，将邮件内容、主题、发件人和收件人作为参数传递。最后，通过 SMTP 服务器发送邮件。
