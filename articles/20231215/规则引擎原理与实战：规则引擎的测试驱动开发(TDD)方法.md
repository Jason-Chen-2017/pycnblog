# 4.具体代码实例和详细解释说明

在这里，我们将通过一个具体的代码实例来详细解释规则引擎的实现过程。

```python
# 定义规则库
rules = [
    {"if": "age > 18", "then": "can_vote"},
    {"if": "age < 18", "then": "cannot_vote"}
]

# 初始化工作内存
work_memory = {"age": 20}

# 评估规则
for rule in rules:
    if rule["if"] in work_memory and eval(rule["if"]):
        print(rule["then"])
```

在这个代码实例中，我们首先定义了规则库，其中包含了两个规则。然后，我们初始化了工作内存，并将年龄设为20。接下来，我们遍历规则库，并根据规则条件和工作内存中的数据来评估规则。最后，我们打印出满足条件的规则结果。
