### 4.1代码实例

```python
import time
import random

def task_execute(task_id):
    # 模拟任务执行过程
    time.sleep(random.randint(1, 10))
    result = random.choice([True, False])
    return result

def task_retry(task_id):
    # 模拟任务重试过程
    time.sleep(random.randint(1, 10))
    result = random.choice([True, False])
    return result

def task_execute_retry(task_id, max_retry_times):
    result = task_execute(task_id)
    if not result:
        retry_times = 0
        while retry_times < max_retry_times:
            result = task_retry(task_id)
            if result:
                break
            retry_times += 1
    return result

if __name__ == '__main__':
    task_id = 1
    max_retry_times = 3
    result = task_execute_retry(task_id, max_retry_times)
    print(f'Task {task_id} execute result: {result}')
```

### 4.2详细解释说明

在上述代码中，我们实现了一个简单的任务重试策略。具体来说，我们有一个`task_execute`函数，用于模拟任务执行过程，它会随机生成一个执行结果（True或False），并根据执行结果返回。

我们还有一个`task_retry`函数，用于模拟任务重试过程，它会随机生成一个重试结果（True或False），并根据重试结果返回。

最后，我们有一个`task_execute_retry`函数，用于实现任务重试策略。它首先调用`task_execute`函数执行任务，如果执行失败（结果为False），则进入重试循环，每次重试次数为`max_retry_times`，直到重试成功（结果为True）为止。

在主程序中，我们设置了一个任务ID（task_id）和最大重试次数（max_retry_times），然后调用`task_execute_retry`函数执行任务重试。最后，我们打印任务执行结果。
