# 4.具体代码实例和详细解释说明
在Python中，可以使用Matplotlib、Seaborn、Plotly等库来创建数据可视化图表。以下是一个使用Matplotlib创建条形图的代码实例：

```python
import matplotlib.pyplot as plt

# 创建一个空的子图
fig, ax = plt.subplots()

# 设置图表的标题和x轴和y轴的标签
ax.set_title('Sample Bar Chart')
ax.set_xlabel('X-axis')
ax.set_ylabel('Y-axis')

# 创建一个条形图
ax.bar([1, 2, 3, 4, 5], [10, 20, 15, 30, 25])

# 显示图表
plt.show()
```

在这个代码实例中，我们首先导入了matplotlib库，然后创建了一个空的子图。接着，我们设置了图表的标题和x轴和y轴的标签。最后，我们创建了一个条形图，并使用`plt.show()`显示图表。
