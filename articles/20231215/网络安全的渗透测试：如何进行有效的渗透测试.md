# 2.核心概念与联系

## 2.1 渗透测试的目的

渗透测试的目的是通过模拟黑客行为来探测网络系统的漏洞和弱点，以便组织能够及时发现和修复这些问题，从而提高网络安全防护水平。

## 2.2 渗透测试的类型

渗透测试可以分为内部渗透测试和外部渗透测试。内部渗透测试是指在组织内部网络系统进行渗透测试，主要关注组织内部的网络安全问题。外部渗透测试是指在组织外部网络系统进行渗透测试，主要关注组织外部的网络安全问题。

## 2.3 渗透测试的阶段

渗透测试可以分为以下几个阶段：

1. 收集信息阶段：通过对目标网络系统的信息收集，了解目标网络系统的拓扑结构、服务和设备等信息。
2. 扫描阶段：通过对目标网络系统进行扫描，了解目标网络系统的开放端口、服务版本等信息。
3. 漏洞探测阶段：通过对目标网络系统进行漏洞探测，了解目标网络系统的漏洞和弱点。
4. 漏洞利用阶段：通过对目标网络系统进行漏洞利用，实现对目标网络系统的控制。
5. 清除踪迹阶段：通过对目标网络系统进行清除踪迹操作，确保渗透测试过程中产生的踪迹被清除。
