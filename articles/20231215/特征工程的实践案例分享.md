# 2.核心概念与联系
在特征工程中，我们需要对原始数据进行预处理、转换和筛选，以生成有助于模型性能提升的特征。这些特征可以是原始数据的统计信息、数据的数学变换、数据的聚类结果等。通过对原始数据进行特征工程，我们可以让模型更好地理解数据，从而提高模型的预测性能。
