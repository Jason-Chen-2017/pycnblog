## 4.1文件读取

文件读取是一种通过计算机程序从文件中读取数据的技术。它可以帮助我们从文件中读取文本、图像和音频等数据。

以下是一个Python代码实例，用于读取文本文件：

```python
# 打开文件
file = open('example.txt', 'r')

# 读取文件内容
content = file.read()

# 关闭文件
file.close()

# 打印文件内容
print(content)
```

在这个代码实例中，我们首先使用`open`函数打开一个名为`example.txt`的文本文件，并以只读模式打开。然后，我们使用`read`方法读取文件的内容，并将其存储在`content`变量中。最后，我们使用`close`方法关闭文件，并使用`print`函数打印文件内容。

## 4.2文件写入

文件写入是一种通过计算机程序将数据写入文件的技术。它可以帮助我们将数据写入文件，以便在人工智能系统中进行处理和存储。

以下是一个Python代码实例，用于写入文本文件：

```python
# 打开文件
file = open('example.txt', 'w')

# 写入文件内容
file.write('Hello, world!')

# 关闭文件
file.close()
```

在这个代码实例中，我们首先使用`open`函数打开一个名为`example.txt`的文本文件，并以写入模式打开。然后，我们使用`write`方法写入文件的内容，并将其存储在`Hello, world!`字符串中。最后，我们使用`close`方法关闭文件。

## 4.3文件操作

文件操作是一种通过计算机程序对文件进行操作的技术。它可以帮助我们对文件进行重命名、删除和移动等操作。

以下是一个Python代码实例，用于重命名文件：

```python
# 重命名文件
import os

# 获取当前工作目录
current_dir = os.getcwd()

# 获取文件名
file_name = 'example.txt'

# 获取文件路径
file_path = os.path.join(current_dir, file_name)

# 重命名文件
os.rename(file_path, 'new_example.txt')
```

在这个代码实例中，我们首先导入`os`模块，然后使用`getcwd`方法获取当前工作目录。接下来，我们使用`os.path.join`方法将当前工作目录和文件名组合成文件路径。最后，我们使用`os.rename`方法重命名文件，将原始文件名`example.txt`更改为`new_example.txt`。
