# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 图像增强
### 3.1.1 直方图均衡化
直方图均衡化是一种常用的图像增强方法，它可以通过调整图像的直方图，提高图像的对比度和亮度。直方图均衡化的主要步骤如下：

1. 计算图像的直方图：通过统计图像中每个灰度级的出现次数，得到图像的直方图。
2. 计算直方图的累积分布函数（CDF）：通过对直方图进行累积计算，得到图像的累积分布函数。
3. 对直方图进行均匀分布：通过对累积分布函数进行均匀分布，得到均匀分布的直方图。
4. 映射原始图像：通过将原始图像的灰度级映射到均匀分布的直方图，得到增强后的图像。

### 3.1.2 高斯滤波
高斯滤波是一种常用的图像平滑处理方法，它可以通过对图像进行高斯模糊，去除图像中的噪声和噪点。高斯滤波的主要步骤如下：

1. 计算高斯核：通过对高斯函数进行积分，得到高斯核。
2. 对图像进行卷积：通过将高斯核与图像进行卷积，得到卷积后的图像。
3. 得到平滑后的图像：通过对卷积后的图像进行归一化处理，得到平滑后的图像。

### 3.1.3 锐化
锐化是一种常用的图像增强方法，它可以通过对图像进行锐化处理，提高图像的锐度和细节。锐化的主要步骤如下：

1. 计算图像的梯度：通过对图像进行差分处理，得到图像的梯度。
2. 计算梯度的绝对值：通过对梯度的绝对值进行累积计算，得到梯度的累积值。
3. 对图像进行锐化处理：通过将梯度的累积值映射到原始图像，得到锐化后的图像。

### 3.1.4 对比度扩展
对比度扩展是一种常用的图像增强方法，它可以通过对图像的对比度进行扩展，提高图像的对比度和亮度。对比度扩展的主要步骤如下：

1. 计算图像的对比度：通过对图像的直方图进行分析，得到图像的对比度。
2. 对比度扩展：通过对图像的对比度进行扩展，得到扩展后的图像。

### 3.1.5 自适应均衡化
自适应均衡化是一种常用的图像增强方法，它可以通过根据图像的亮度和对比度进行处理，提高图像的可见性和可读性。自适应均衡化的主要步骤如下：

2. 根据亮度和对比度进行处理：通过根据亮度和对比度进行处理，得到增强后的图像。

def histogram_equalization(image):
    # 计算图像的直方图
    hist, bins = np.histogram(image.ravel(), bins=256, density=True)
    # 计算直方图的累积分布函数
    cdf = hist.cumsum()
    # 计算均匀分布的直方图
    cdf_normalized = cdf * hist.max() / cdf.max()

import matplotlib.pyplot as plt

def gaussian_filtering(image, sigma):
    # 对图像进行高斯模糊
    filtered_image = gaussian_filter(image, sigma)
    # 对卷积后的图像进行归一化处理

from scipy.ndimage import convolve
import matplotlib.pyplot as plt

def sharpening(image):
    # 计算图像的梯度
    gradient_x = convolve(image, np.array([[-1, 0, 1], [-2, 0, 2], [-1, 0, 1]]), mode='constant', cval=0)
    gradient_y = convolve(image, np.array([[-1, -2, -1], [0, 0, 0], [1, 2, 1]]), mode='constant', cval=0)
    # 计算梯度的绝对值
    abs_gradient = np.sqrt(gradient_x**2 + gradient_y**2)
    # 对图像进行锐化处理
    sharpened_image = image + abs_gradient
    # 得到锐化后的图像
    plt.figure()
    plt.imshow(image, cmap='gray')
    plt.figure()
    plt.imshow(sharpened_image, cmap='gray')
    plt.show()
    return sharpened_image
