# 1.简介
         
　　当我们谈论跟踪目标时，一般都是提及其移动轨迹或运动状态。实际上，给定初始位置、速度、加速度以及时间等初始条件后，已知目标的运动轨迹，就可以通过微积分方法求出所有物体位置和速度变化关系。然而，矩形目标追踪任务本身非常简单且易于理解，尤其是在室内环境中，但是要实现真正精确、高效的跟踪目标，却是件复杂繁琐乏味的事情。我们无法像追踪圆锥曲线一样，将目标边缘弧线分成很多小段进行连续跟踪。而现实世界中的大部分目标都是由多个环绕在一起的矩形构成的，因此，单纯用传统的方法处理矩形目标会遇到很多问题。
        　　为了能够更好地处理矩形目标，我们需要采用一些特殊的处理方式。
        　　
        　　
        
        # 2.摘要
        在当前技术水平下，无法直接对矩形目标进行真正精确、高效的跟踪。原因主要有两个：一方面是由于矩形目标的性质，导致在保持封闭的前提下，对某个目标进行快速准确的检测和识别十分困难；另一方面，相机的视角变化导致了目标外观的变化，从而影响到目标位置信息的准确度。本文从对常见跟踪技术的分析入手，探讨如何利用机器学习方法开发新型矩形目标跟踪系统。
        　　首先，我们首先分析目前矩形目标跟踪方法存在的问题，并提出一种新的跟踪方案——多边形密集模式（Polygonal Density Pattern）。该方法基于图像特征点检测的能力，通过将目标多边形区域内的图像特征点聚集到中心位置，获得整个目标的中心坐标和形状信息。然后，根据中心坐标信息，对目标的形状进行估计，计算目标的边界框。通过这种方式，我们可以准确、快速、实时地跟踪到目标的任何位置。
        　　其次，我们还着重阐述一下矩形目标跟踪的关键问题。由于矩形目标具有闭合的特点，在保持封闭的前提下，对于某个目标的快速准确检测和识别十分困难。尤其是在室内环境下，目标总是处于某种不规则形状，使得目标的边界多达数个，而且边界的位置变化也十分剧烈。因此，在现实中，绝大多数的目标都呈现多边形状，即便只有一个轮廓。
        　　再者，相机视角的变化会导致目标外观的变化，从而影响到目标位置信息的准确度。同时，相机视角的变换会导致图像的畸变，这种畸变又会造成目标的形状和大小发生扭曲。因此，要开发有效的跟踪系统，还需考虑计算机视觉中的几何变换、光流、特征点检测等技术。
        　　最后，本文也着重论证了矩形目标的重要价值，认为在自动驾驶领域，尤其是在城市中高速公路、高架桥以及交叉口等复杂道路场景中，由于缺乏足够高质量的目标识别、跟踪技术，自动驾驲将很难取得突破性的进步。因此，研发高质量的目标跟踪技术至关重要。
