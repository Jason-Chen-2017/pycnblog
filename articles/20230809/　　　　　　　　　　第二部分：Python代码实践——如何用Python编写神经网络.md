        ## 2.基本概念术语说明
        ### 1.什么是神经网络？
        神经网络（英语：neural network），也称作“神经网络／网路”，是一个用来模拟生物神经系统的计算模型，由多个互相连接的简单单元组成，通过激活函数进行复杂数据的处理与学习。它是目前应用最普遍的深度学习方法之一。简单的说，就是一些输入数据通过一系列的加权运算得到输出结果，这个过程本身就是一个非线性函数，因此也被称作非线性多层次分类器。换句话说，神经网络是基于人脑神经元网络的研究与模仿，它将大量的神经元节点通过连接，形成一个具有多个层次的网络结构。网络的每一层都包括若干个节点（又称神经元），这些节点之间可以传输信息并完成特定的计算任务。在网络中，每个节点都根据之前的输出值与当前的输入值、各自的权重来更新自己的输出值，而整个网络则通过反向传播规则来更新每个节点的权重，以使得整个网络的输出结果逐步优化，使其更准确地模仿人类的神经元工作方式。
        由于神经网络在生物神经网络的结构和电信号传递的原理上都做了很多假设，所以实际的神经网络并不完全符合真实生物神NET元的工作原理。但它的结构与特点却已经成为一个非常重要且具有挑战性的研究课题。

        ### 2.神经元模型
        神经元模型（英语：neuron model）是神经网络中最基本的模型。它是基于生物神经元的研究成果，是人们研究神经网络的起点。它分为两大部分，第一部分是神经元内部的动态行为，即电信号的传递；第二部分是神经元之间的交流连接，即节点之间的信号传递。在神经元模型里，我们假设有一个带有输入、输出、阈值、激活函数的简单神经元，其结构如图所示：

        ### 3.多层神经网络
        多层神经网络（英语：multilayer neural networks）是指由至少三层的神经网络组成的神经网络结构。每一层都由多个神经元组成，每两个相邻层之间的连接通常采用全连接的方式。多层神经网络的典型结构如图所示：

        ### 4.误差反向传播算法
        误差反向传播算法（英语：backpropagation algorithm）是神经网络训练过程中采用的一种最常用的优化算法。它是一种基于误差的迭代算法，它利用神经网络前向传播结果与正确标签之间的差异来计算每个参数的导数，并通过梯度下降法沿着这个方向调整参数，以最小化网络的误差。具体算法描述如下：

        1. 初始化参数
        2. 对输入数据进行正向传播
        3. 根据正向传播结果和标签计算损失函数（如均方误差）
        4. 使用链式求导法则计算各个参数的梯度
        5. 更新各个参数的值，使得网络误差最小化

        误差反向传播算法通过反向传播计算各个参数的梯度，并通过梯度下降法来更新参数，使得网络误差最小化。
