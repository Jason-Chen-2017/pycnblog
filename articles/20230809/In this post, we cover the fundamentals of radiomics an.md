# 1.简介
         
Radiomics（流形调查）是一种基于医学图像、患者数据及计算机技术的生物特征分析方法。它利用量化的方式对生物组织的分型进行描述，特别适用于图像诊断及实时监测领域。通过在不同生物体表层或组织中提取有关生物特征的测定值，由此得出图像的病变征象，如肺癌、乳腺癌、结直肠癌等。目前，国内外已经广泛应用于肿瘤分类、癌症早期筛查、肝、胆管等器官病变的辅助诊断。随着近年来的重视，“Radiomics”成为越来越多领域的热门词汇。它的优势在于利用高精度图像和可靠的图像处理技术，通过对图像中肿瘤组织的特征进行多维度描述，帮助医生更好地鉴别各类型疾病。

# 2.Radiomic Measurements
Radiomics常用的生物特征包括体积、形状、密度、模糊度、灰度级别、纹理结构、线型、周围血管分布、血管形态、循环节宽度等。每一个特征都可以对应不同的生物属性和疾病特点。以下给出Radiomics研究的一个典型案例——肺癌。

1.体积
肺癌一般呈囊性质，具有直径约1.5~1.7米的大尺寸，相当于25~50平方厘米。因此，体积特征显得尤为重要，它能够反映肺癌的大小、形态以及病变位置。Radiomics中的体积特征通常采用切片（slice-based）或块（patch-based）表示法，从而保证了特征的连续性和全面性。

2.形状
肺癌以圆柱状核态发生，体积较小，核周围呈环状的渗漏，因此具有不规则的形状。在体素化切片和块表示中，这种不规则的形状可以通过计算曲率参数获得。

3.密度
肺癌组织具有膨胀、侵犯周围组织组织细胞的能力，因此具有高密度。在体素化切片和块表示中，密度也可以通过计算密度特征来获得。

4.模糊度
由于肺癌组织尺寸大且微笼相间，造成影像模糊。因此，模糊度特征也非常重要，它可以反映肺癌的模糊程度。Radiomics中模糊度特征的统计方法主要有：

1)直方图均衡化：是一种图像处理技术，其目的是使图像的每个像元的灰度级分布一致。直方图均衡化通过将输入图像中每个像元的灰度级映射到某一指定的灰度级范围内，并重新调整图像的灰度级分布，使得图像的整体亮度和对比度都保持一致。

2)局部差分：是一种滤波器，通过计算原始信号与少数几个邻居像元的差异来估计该像元的值。当卷积滤波器的大小固定或者空间步长固定时，就得到了局部差分。在这个过程中，靠近边缘的像元会产生较大的差异，远离边缘的像元则会产生较小的差异。

3)非线性滤波：是指在信号传递过程中加入一定的非线性变换，使信号看起来不再是以直线方式变化，从而突出特征。比如，傅里叶变换、基于小波的空间频谱分析等都是非线性滤波的例子。

5.灰度级别
肺癌组织呈现出一种黑色的灰色皮肤。其粗糙、肥厚、硬化、稀薄的外观，使得肺癌组织的等级很低，这也是为什么许多病理学家认为肺癌的发病率较高。除此之外，肺癌病灶的大部分组织都是淡灰色的，这对于灰色组织的分类也起着至关重要的作用。Radiomics中，灰度特征通常可以通过标准差来获得。

6.纹理结构
肺癌组织具有丰富的纹理结构，如颅内、体表、头盖骨等，这些结构对于癌症的发展都有着重要作用。纹理结构特征可以通过计算纹理指数来获得。

7.线型
肺癌组织的结构具有不规则的线型，如球状、球形等。因此，线型特征对于肺癌的分类非常重要。Radiomics中，线型特征可以通过计算直线距离、曲率、方向一致性等参数来获得。

8.周围血管分布
肺癌组织周围血管分散分布，有的组织表现出周围有充满血液的空气洞，这可能促进了肺癌细胞向周围组织中集聚。因此，血管分布特征对于肺癌的分类至关重要。Radiomics中，血管分布特征可以用血管密度、血管长度、血管宽度等参数来获得。

9.血管形态
血管形态可以理解为血管本身的复杂结构，如弯曲、狭窄、凹陷等。Radiomics中，血管形态特征可以用血管截面的百分比和数量等参数来获得。

10.循环节宽度
肺癌组织的循环节宽度与核的大小相关。因此，循环节宽度特征对肺癌的分类至关重要。Radiomics中，循环节宽度特征可以通过计算角蜂窝瘤的中心距血管中心的距离、环状分隔孔径和包覆层厚度等参数来获得。

总之，通过对不同生物特征的量化测定，Radiomics能够提供关于肺癌等不同疾病的精确信息。但是，单纯依靠Radiomics进行肺癌分类仍然存在着不足。首先，Radiomics在分类上的准确度依赖于多模态数据、对肺癌组织的分割及定位、结合相关特征。其次，Radiomics分析结果的可靠性也需要大量验证实验。最后，针对同一病人不同时间的CT图像，如何正确使用Radiomics分析其生物特征仍然是一个难题。为了解决以上三个问题，正在努力探索更多的Radiomics分析技术及应用领域。
