以下是对注意力训练和正念冥想的具体操作步骤：

#### 3.2.1 注意力训练

1. **任务选择**：选择具有一定挑战性但可控的任务，如阅读理解、数字记忆、图形识别等。

#### 3.2.2 正念冥想

2. **专注呼吸**：集中注意力在呼吸上，感受空气进出鼻腔的感觉。
3. **觉知体验**：将注意力逐步扩展到身体各部分，感受身体的放松和紧张状态。
4. **觉察思维**：在觉知过程中，观察思维的流动，不做评判或干涉。
5. **逐步深化**：随着练习的深入，逐渐延长练习时间，增加练习深度。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义训练参数
num_tasks = 100  # 数字记忆任务数量
num_trials = 10  # 每个任务执行次数
training_time = np.linspace(0, 1, num_trials)  # 训练时间

# 生成随机数字
np.random.seed(42)
numbers = np.random.randint(0, 10, size=(num_tasks, num_trials))

# 定义注意力变化函数
def attention_change(x, t):
    return 0.1 * t + 0.5 * np.exp(-0.5 * t) * np.sin(2 * np.pi * t)

# 计算每个任务在训练时间内的注意力变化
attention_values = []
for i in range(num_tasks):
    attention_values.append(attention_change(numbers[i], training_time))

# 计算总体注意力变化趋势
total_attention = np.sum(attention_values, axis=0)
plt.plot(training_time, total_attention, label='Total Attention')
plt.xlabel('Training Time')
plt.ylabel('Attention Change')
plt.legend()
plt.show()
```
