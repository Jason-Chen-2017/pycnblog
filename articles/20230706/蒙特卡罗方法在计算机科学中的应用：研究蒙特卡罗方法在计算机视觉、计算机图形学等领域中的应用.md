### 2.2.1. 生成随机数

在Python中，可以使用`random`库生成随机数。该库提供了生成服从正态分布（或其它概率分布）随机数的方法。

```python
import random

# 生成服从标准正态分布的随机数
random_num = random.normal(0, 1, 1000)

# 打印随机数
print(random_num)
```

```python
from PIL import Image

img = Image.generate_RGB(224, 224, (3, 3), (0, 0, 255), 50)

```

### 2.2.2.2. 建立随机化地图

在Python中，可以使用`random_walk`算法生成随机化地图。该算法基于随机漫步（Random Walk）原则，可以生成具有自然随机性的地图。

```python
import random

# 生成随机地图
map = random_walk(100, 100, 1)

# 显示地图
print(map)
```
