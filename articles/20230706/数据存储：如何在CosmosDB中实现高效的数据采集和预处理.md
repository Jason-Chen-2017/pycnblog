### 2.2.1. CosmosDB介绍

CosmosDB是一款高性能、可扩展、高可用性的分布式NoSQL数据库,具有高可靠性、高可用性和高扩展性。它支持多种数据模型,包括键值数据、文档数据、列族数据和图形数据等,可以满足各种应用场景的需求。

### 2.2.2. 数据采集

在CosmosDB中进行数据采集,可以使用多种方式。首先可以使用CosmosDB的API,通过HTTP请求获取数据。例如,可以使用以下代码获取CosmosDB中的所有文档:

```
GET https://<cosmosdb-url>/cosmosdb/<db-name>/<container-name>?query=SELECT+* FROM+<table-name>
```

其中,`<cosmosdb-url>`是CosmosDB的URL,`<db-name>`是数据库名称,`<container-name>`是容器名称,`<table-name>`是要查询的表名。

另外,也可以使用CosmosDB的SDK,通过编程语言获取数据。例如,使用Python获取CosmosDB中的数据:

```
from cosmosdb.client import CosmosClient

client = CosmosClient('<cosmosdb-url>')
result = client.get_table_states('<table-name>')
```
