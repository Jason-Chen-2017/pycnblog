### 1. 引言

React Native 是一个跨平台的移动应用程序开发框架，能够帮助开发者快速构建高性能、美观的移动应用。React Native 使用 JavaScript 语言编写，具有易于上手、高效能等优点。本文旨在介绍如何使用 React Native 工具链和自定义构建工具，构建高性能的移动应用。

### 1.1. 背景介绍

移动应用市场快速发展，开发者需要不断提高应用的性能以满足用户需求。传统的移动应用开发方式通常使用原生框架（如 Swift、Java、Kotlin 等）编写，这种方式开发效率较低，应用性能较差。而 React Native 作为一种跨平台开发方式，能够解决这个问题。
