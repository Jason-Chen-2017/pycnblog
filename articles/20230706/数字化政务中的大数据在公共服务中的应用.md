2.4. 代码实例和解释说明

以机器学习中的一个经典算法——K-Means聚类算法为例，进行代码实现和解释说明。

```python
# 导入相关库
import numpy as np
from sklearn.cluster import KMeans

# 数据预处理
data = np.array([
    [1, 2],
    [1, 4],
    [1, 0],
    [4, 2],
    [4, 4],
    [4, 0],
    [0, 2],
    [0, 4]
])

# 数据标准化
data_std = (data - np.mean(data)) / np.std(data)

# 聚类结果
kmeans_result = KMeans(n_clusters=3, random_state=0).fit(data_std)
```
