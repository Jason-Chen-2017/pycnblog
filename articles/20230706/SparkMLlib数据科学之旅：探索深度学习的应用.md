### 2.2.1. 神经网络

神经网络是一种模拟人脑神经元连接的计算模型，可以用于对复杂数据进行分类、预测和聚类等任务。在Spark MLlib中，我们使用Keras框架实现了一个简单的神经网络，用于对CIFAR-10数据集中的飞机图片进行分类。

```python
from keras.models import Sequential
from keras.layers import Dense

# 准备数据集
train_images =...
train_labels =...

# 创建神经网络模型
model = Sequential()
model.add(Dense(10, input_shape=(train_images.shape[1],)))
model.add(Activation('relu'))
model.add(Dense(10,))
model.add(Activation('softmax'))

# 编译模型并训练
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])
model.fit(train_images, train_labels, epochs=10)
```
