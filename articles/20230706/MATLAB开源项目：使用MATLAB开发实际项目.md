2. 技术原理及概念
-----------------

2.1. 基本概念解释

图像分割是指将图像中的像素分为不同的类别，通常为类别1和类别2。类别1表示目标物体，类别2表示背景。

2.2. 技术原理介绍：算法原理，具体操作步骤，数学公式，代码实例和解释说明

该项目的核心算法是基于MATLAB的图像处理工具箱中的“图像处理：形态学”函数实现图像分割。具体操作步骤如下：

```
% 读取图像
img = imread('image.jpg');

% 创建掩模
mask = (img > 120);

% 获取图像中类别的像素值
result = im2double(mask);

% 显示结果
figure;
imshow(img);
hold on;
plot(result{:});
legend('类别1', 'r', '类别2', 'g');
```
在上面的代码中，我们首先使用`imread`函数读取一张图片，并使用`im2double`函数将二进制图像转换为双精度图像。然后我们使用掩模`(img > 120)`创建一个掩模，表示为类1的像素。最后，我们使用`imshow`函数显示原图，并使用`hold on`命令保留坐标轴，使用`plot`函数将类1和类2的像素值绘制在图上，并使用`legend`函数给每个类别添加标签。
