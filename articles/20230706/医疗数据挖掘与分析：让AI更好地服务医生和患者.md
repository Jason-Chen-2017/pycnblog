### 4.2. 应用实例分析

在本文中，我们将实现一个简单的聚类算法来对数据集进行聚类。首先，需要对数据集进行清洗和预处理。然后，使用K-means算法对数据进行聚类。最后，将聚类后的数据进行可视化，以便更好地理解聚类的效果。

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
import matplotlib as plt

# 数据集
data = pd.read_csv("hospital_data.csv")

# 数据预处理
data = data[['weight', 'height', 'age', 'gender', 'address']]
data.dropna(inplace=True)

# 数据可视化
data.plot(kind='scatter')
plt.show()

# 聚类算法实现

kmeans.fit(data)

# 聚类结果可视化
kmeans.labels_
```

### 4.3. 核心代码实现

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
import matplotlib as plt

# 数据集
data = pd.read_csv("hospital_data.csv")

# 数据预处理
data = data[['weight', 'height', 'age', 'gender', 'address']]
data.dropna(inplace=True)

# 数据可视化
data.plot(kind='scatter')
plt.show()

# 聚类算法实现
kmeans = KMeans(n_clusters=3, n_neighbors=1000)
kmeans.fit(data)

# 聚类结果可视化
labels = kmeans.labels_
plt.scatter(data['weight'], data['height'], c=labels)
plt.show()
```
