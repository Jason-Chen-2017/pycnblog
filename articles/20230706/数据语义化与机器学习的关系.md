# 2. 技术原理及概念

## 2.1. 基本概念解释

数据语义化（Data Semanticization）是一种将原始数据转换为具有语义信息的数据的过程。语义化后的数据可以使机器学习模型更关注数据中的关键信息，从而提高模型性能。数据语义化主要有两个基本概念：词向量（Term Frequency-Inverse Document Frequency，TF-IDF）和概念图（ Concept Graph ）。

## 2.2. 技术原理介绍: 算法原理，具体操作步骤，数学公式，代码实例和解释说明

2.2.1. TF-IDF 算法原理

TF-IDF 是一种计算词向量相似度的常用方法。其原理是通过对词向量进行平方，再计算每个词向量与原词向量之间的余弦相似度。

```
import numpy as np

def tfidf(text, top_n=5):
    import nltk
    from nltk.corpus import stopwords
    from nltk.tokenize import word_tokenize
    from nltk.stem import WordNetLemmatizer
    from nltk.util import ngrams
    
    # 加载停用词
    nltk.download('punkt')
    nltk.download('wordnet')
    
    # 预处理词干
    def preprocess(text):
        for token in nltk.word_tokenize(text):
            if token not in stopwords.words('english'):
                return token
        return''.join([wordnet.lemmatize(token) for token in ngrams(text, n=1)])
    
    # 词频统计
    tokens = ngrams(text, n=1)
    freq_INDEX = nltk.freq(tokens)
    freq_ITEMS = list(freq_INDEX.items())
    freq_MAX = max(freq_ITEMS, key=lambda x: x[1])[0][1]
    
    # 计算词向量
    tokens = [preprocess(token) for token in ngrams(text, n=1) if token not in stopwords.words('english')]
    word_vec_scaled = [vector/tf_idf for vector in tokens]
    
    # 返回词向量
    return word_vec_scaled
```
