2.2.4. 代码实例和解释说明

以 Python 为例，给出一个实现定时器算法的代码实例：

import time

# 设定定时器，每过1小时执行一次
定时器 = time.PeriodicCall(lambda: (1 + 2) * 3 / 5, 1)

# 循环执行定时器，直到系统被关闭
while True:
   # 执行定时器
   time.sleep(定时器.total_seconds())
   # 打印当前库存数量
   print("当前库存数量:", (1 + 2) * 3 / 5)
```
