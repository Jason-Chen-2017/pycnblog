### 5.2 源代码详细实现

以下是一个简单的在线法律咨询服务的源代码实现：

```python
import spacy
import tensorflow as tf

# 加载NLP模型
nlp = spacy.load("en_core_web_sm")

# 加载深度学习模型
model = tf.keras.models.load_model("law_questions_model.h5")

# 用户提问
user_question = input("请提出您的问题：")

# 自然语言处理
processed_question = nlp(user_question)

# 智能问答
predicted_answer = model.predict(processed_question)

# 输出答案
print("法律答案：", predicted_answer)
```

上述代码实现了以下功能：

1. **加载NLP模型**：使用spaCy库加载英语NLP模型，用于处理用户输入的问题。
2. **加载深度学习模型**：使用TensorFlow库加载训练好的深度学习模型，用于生成法律答案。
3. **用户提问**：通过输入函数获取用户输入的问题。
4. **自然语言处理**：使用NLP模型处理用户输入的问题，将其转换为机器可理解的形式。
5. **智能问答**：使用深度学习模型对处理过的问题进行预测，生成法律答案。
6. **输出答案**：将生成的法律答案输出给用户。
