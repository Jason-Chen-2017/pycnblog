代码实例：

```sql
CREATE TABLE users (
    id INT PRIMARY KEY,
    username VARCHAR(255) NOT NULL,
    password VARCHAR(255) NOT NULL,
    email VARCHAR(255) NOT NULL
);

CREATE INDEX idx_username ON users (username);

SELECT * FROM users WHERE username='test';

EXPLAIN SELECT * FROM users WHERE username='test';
```

详细解释说明：

- 数据库模式设计：在这个例子中，我们创建了一个名为users的表，它包括id、username、password和email等字段。
- 数据库索引设计：在这个例子中，我们创建了一个名为idx_username的索引，它包括username字段。
- 数据库查询优化：在这个例子中，我们查询了users表中的所有记录，并使用EXPLAIN语句查看查询计划。
