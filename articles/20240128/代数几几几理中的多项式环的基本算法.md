## 1. 背景介绍

多项式环是一种特殊的代数结构，它由一组线性无关多项式组成。这些多项式可以用于表示和操作代数几几几理中的各种对象，如向量、矩阵、线性映射等。多项式环的基本算法是在这些对象之间进行运算的基础，它们在计算机图形学、数值分析、控制理论等领域具有广泛的应用。

## 2. 核心概念与联系

在多项式环中，每个多项式都是一个线性组合，其中系数是字段元素，项是变量的幂。多项式环的基本运算包括加法、减法、乘法和除法。这些运算的基础是多项式的基本属性，如常数多项式、单项式、零多项式等。

多项式环的核心概念与其他代数结构之间有密切的联系。例如，多项式环可以看作是环的特殊例子，其中元素是线性组合的多项式。此外，多项式环还与线性代数、数论等其他领域有密切的联系，这使得它在许多应用中发挥着重要作用。

## 3. 核心算法原理和具体操作步骤以及数学模型公式详细讲解

### 3.1 多项式加法和减法

多项式加法和减法是多项式环中最基本的运算。给定两个多项式 $f(x) = a_0 + a_1x + a_2x^2 + \cdots + a_nx^n$ 和 $g(x) = b_0 + b_1x + b_2x^2 + \cdots + b_nx^n$，它们的和和差分别定义为：

$$
f(x) \pm g(x) = (a_0 \pm b_0) + (a_1 \pm b_1)x + (a_2 \pm b_2)x^2 + \cdots + (a_n \pm b_n)x^n
$$

### 3.2 多项式乘法

多项式乘法是多项式环中的一种复合运算。给定两个多项式 $f(x) = a_0 + a_1x + a_2x^2 + \cdots + a_nx^n$ 和 $g(x) = b_0 + b_1x + b_2x^2 + \cdots + b_nx^n$，它们的乘积定义为：

$$
f(x)g(x) = (a_0b_0) + (a_0b_1 + a_1b_0)x + (a_0b_2 + a_1b_1 + a_2b_0)x^2 + \cdots + (a_0b_n + a_1b_{n-1} + \cdots + a_nb_0)x^n
$$

### 3.3 多项式除法

多项式除法是多项式环中的一种分解运算。给定两个多项式 $f(x) = a_0 + a_1x + a_2x^2 + \cdots + a_nx^n$ 和 $g(x) = b_0 + b_1x + b_2x^2 + \cdots + b_nx^n$，其中 $g(x)$ 是非零多项式，则 $f(x)$ 可以被 $g(x)$ 除以，得到一对多项式 $q(x)$ 和 $r(x)$，满足 $f(x) = g(x)q(x) + r(x)$，其中 $r(x)$ 是零多项式或者是 $g(x)$ 的常数倍。

### 3.4 多项式环的基本算法

多项式环的基本算法包括加法、减法、乘法和除法。这些算法的实现可以通过以下数学模型公式来描述：

- 加法：$f(x) + g(x) = (a_0 + b_0) + (a_1 + b_1)x + (a_2 + b_2)x^2 + \cdots + (a_n + b_n)x^n$
- 减法：$f(x) - g(x) = (a_0 - b_0) + (a_1 - b_1)x + (a_2 - b_2)x^2 + \cdots + (a_n - b_n)x^n$
- 乘法：$f(x)g(x) = (a_0b_0) + (a_0b_1 + a_1b_0)x + (a_0b_2 + a_1b_1 + a_2b_0)x^2 + \cdots + (a_0b_n + a_1b_{n-1} + \cdots + a_nb_0)x^n$
- 除法：$f(x) = g(x)q(x) + r(x)$，其中 $q(x)$ 和 $r(x)$ 是多项式，$r(x)$ 是零多项式或者是 $g(x)$ 的常数倍。

    def __sub__(self, other):

    def __truediv__(self, other):
        q = Polynomial([0] * len(self.coefficients))
        r = self
        for i in range(len(self.coefficients) - 1, -1, -1):
            r = r - (other.coefficients[i] // self.coefficients[0]) * Polynomial([self.coefficients[j] for j in range(i + 1)])
            q.coefficients[i] = other.coefficients[i] // self.coefficients[0]
        return q, r

    def __str__(self):
        return ' + '.join(f'{coeff}x^{i}' if coeff else f'{coeff}' for i, coeff in enumerate(self.coefficients[::-1]))
