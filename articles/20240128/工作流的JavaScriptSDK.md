- 活动（Activity）：工作流中的基本单元，表示一个具体的任务或操作。

### 4.1 代码实例

```javascript
const workflow = {
  activities: [

    { id: 'task2', name: '任务2' },
    { id: 'task3', name: '任务3' }
  ],
  connectors: [
    { id: 'connector1', source: 'task1', target: 'task2' },
    { id: 'connector2', source: 'task2', target: 'task3' }
  ]

// 使用JavaScriptSDK实现工作流功能
const { WorkflowSDK } = require('workflow-sdk');
const sdk = new WorkflowSDK();

sdk.defineWorkflow(workflow);
sdk.executeWorkflow();
```

### 4.2 详细解释说明

在上述代码实例中，我们首先定义了一个简单的工作流，包括一组活动和连接器。然后，我们使用JavaScriptSDK的API来实现工作流的功能。

首先，我们使用`defineWorkflow`方法来定义工作流。这个方法接受一个工作流对象作为参数，工作流对象包含一组活动和连接器。

然后，我们使用`executeWorkflow`方法来执行工作流。这个方法会根据工作流的定义，自动执行各个任务，并根据任务之间的依赖关系来确定任务的执行顺序。
