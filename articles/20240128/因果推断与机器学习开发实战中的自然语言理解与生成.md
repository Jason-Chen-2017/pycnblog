## 4. 具体最佳实践：代码实例和详细解释说明
以下是一个基于Python的基于规则的因果推断实例：

```python
def if_rain(weather):
    if weather == "rain":
        return "will_rain"
    else:
        return "will_not_rain"

weather = "rain"
print(if_rain(weather))
```

在这个实例中，我们定义了一个名为`if_rain`的函数，它接受一个参数`weather`，并根据`weather`的值返回一个字符串。如果`weather`的值为`"rain"`，则返回`"will_rain"`；否则返回`"will_not_rain"`。然后，我们将`weather`的值设置为`"rain"`，并调用`if_rain`函数，输出结果为`"will_rain"`。
