## 1. 背景介绍

Apache Flink 是一个流处理框架，它可以处理大量实时数据，并提供一种简洁的 SQL 语言来操作这些数据。FlinkSQL 是 Flink 的一个子项目，它将 Flink 的流处理功能与 SQL 语言相结合，使得处理大数据变得更加简单和高效。

FlinkSQL 的核心优势在于它的实时性、高性能和易用性。与传统的批处理系统不同，FlinkSQL 可以在数据到达时立即处理，而不需要等待所有数据到达。此外，FlinkSQL 可以处理大量数据，并提供高吞吐量和低延迟。最后，FlinkSQL 使用 SQL 语言，使得数据处理变得更加简洁和易于理解。
