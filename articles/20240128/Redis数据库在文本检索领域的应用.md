## 4. 具体最佳实践：代码实例和详细解释说明

在 Redis 数据库中，我们可以使用以下代码实例来实现文本检索功能：

```python
import redis

# 创建 Redis 连接
r = redis.StrictRedis(host='localhost', port=6379, db=0)

# 存储文档内容
r.hset('doc1', 'title', 'Redis 数据库在文本检索领域的应用')
r.hset('doc1', 'content', 'Redis 是一个高性能的键值存储系统，它具有快速的读写速度、高可扩展性和易于使用的接口。')

# 存储关键词
r.sadd('keyword1', 'Redis')
r.sadd('keyword1', '数据库')
r.sadd('keyword1', '文本检索')
r.sadd('keyword1', '高性能')
r.sadd('keyword1', '键值存储')

# 存储关键词和权重
r.zadd('keyword2', {'Redis': 10, '数据库': 5, '文本检索': 8, '高性能': 15, '键值存储': 7})

# 检索文档
doc1 = r.hgetall('doc1')
keyword1 = r.smembers('keyword1')
keyword2 = r.zrange('keyword2', 0, -1)

# 输出结果
print(doc1)
print(keyword1)
print(keyword2)
```

在这个代码实例中，我们使用 Redis 的 `HSET`、`SADD` 和 `ZADD` 命令 respectively 来存储文档内容、关键词和关键词与权重。然后，我们使用 Redis 的 `HGETALL`、`SMEMBERS` 和 `ZRANGE` 命令 respectively 来检索文本数据。
