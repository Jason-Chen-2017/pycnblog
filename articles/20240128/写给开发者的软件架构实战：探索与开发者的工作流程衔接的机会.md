代码实例：

以下是一个简单的代码实例，展示了如何使用Git进行版本控制和协作：

```bash
# 创建一个新的Git仓库
$ git init

# 添加文件到仓库
$ git add .

# 提交文件到仓库
$ git commit -m "初始提交"

# 创建一个新的分支
$ git branch new-feature

# 切换到新的分支
$ git checkout new-feature

# 在新分支上进行修改
$ vim index.html

# 提交修改到新分支
$ git commit -m "更新index.html"

# 切换回主分支
$ git checkout master

# 合并新分支的修改
$ git merge new-feature

# 删除新分支
$ git branch -d new-feature
```

详细解释说明：

在这个例子中，我们首先使用`git init`命令创建了一个新的Git仓库。然后，我们使用`git add .`命令将所有文件添加到仓库中，并使用`git commit -m "初始提交"`命令提交文件到仓库。接着，我们使用`git branch new-feature`命令创建了一个新的分支，并使用`git checkout new-feature`命令切换到新分支。在新分支上，我们使用`vim index.html`命令进行修改，并使用`git commit -m "更新index.html"`命令提交修改到新分支。然后，我们使用`git checkout master`命令切换回主分支，并使用`git merge new-feature`命令合并新分支的修改。最后，我们使用`git branch -d new-feature`命令删除了新分支。
