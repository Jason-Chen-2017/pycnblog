## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 语义角色标注（SRL）

以下是一个简单的Python代码实例，用于实现语义角色标注（SRL）：

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.tag import pos_tag
from nltk.chunk import conlltags2tree

def srl(sentence):
    # 分词
    words = word_tokenize(sentence)
    # 词性标注
    pos_tags = pos_tag(words)
    # 语义角色标注
    tree = conlltags2tree(pos_tags)
    return tree

sentence = "John bought a book for Mary."
tree = srl(sentence)
print(tree)
```

### 4.2 事件抽取（EE）

以下是一个简单的Python代码实例，用于实现事件抽取（EE）：

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.tag import pos_tag
from nltk.chunk import conlltags2tree

def event_extraction(sentence):
    # 分词
    words = word_tokenize(sentence)
    # 词性标注
    pos_tags = pos_tag(words)
    # 事件抽取
    tree = conlltags2tree(pos_tags)
    return tree

sentence = "John bought a book for Mary."
tree = event_extraction(sentence)
print(tree)
```
