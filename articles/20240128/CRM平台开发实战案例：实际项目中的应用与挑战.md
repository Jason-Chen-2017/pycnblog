代码实例：

```python
# 使用Python实现简单的CRM系统

import sqlite3

# 创建数据库
conn = sqlite3.connect('crm.db')
c = conn.cursor()
c.execute('''CREATE TABLE IF NOT EXISTS customers
             (id INTEGER PRIMARY KEY, name TEXT, phone TEXT, email TEXT)''')

# 插入客户信息
c.execute("INSERT INTO customers (name, phone, email) VALUES (?, ?, ?)",
          ('John Doe', '1234567890', 'john@example.com'))

# 查询客户信息
c.execute("SELECT * FROM customers")
print(c.fetchall())

# 关闭数据库连接
conn.close()
```

详细解释说明：

- 使用`sqlite3`库创建数据库，并创建`customers`表。
- 使用`execute`方法插入客户信息，包括客户名称、电话、邮箱等。
- 使用`execute`方法查询客户信息，并使用`fetchall`方法获取查询结果。
- 使用`close`方法关闭数据库连接。

## 8. 附录：常见问题与解答

Q：CRM系统与ERP系统有什么区别？

A：CRM系统主要关注客户管理，包括客户信息的收集、存储、管理和分析。ERP系统主要关注企业资源管理，包括供应链管理、生产管理、销售管理等。CRM系统和ERP系统可以相互集成，共同构成企业的整体管理体系。
