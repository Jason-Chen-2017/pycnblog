```python
    def __init__(self, order_id, order_details):
        self.order_id = order_id
        self.order_details = order_details
        self.status = "未支付"

    def pay(self):
        self.status = "已支付"

    def ship(self):
        if self.status == "已支付":
            self.status = "已发货"

    def complete(self):
        if self.status == "已发货":
            self.status = "已完成"

order = Order("001", "购买电子产品")
order.pay()
order.ship()
order.complete()
print(order.status)
```

在实际应用中，我们可以使用这个Order类来实现订单跟踪和状态管理。例如，当用户下单时，我们可以创建一个Order对象，并将其状态设置为"未支付"。当用户支付时，我们可以调用Order对象的pay方法，将其状态更新为"已支付"。当用户发货时，我们可以调用Order对象的ship方法，将其状态更新为"已发货"。最后，当用户完成订单时，我们可以调用Order对象的complete方法，将其状态更新为"已完成"。
