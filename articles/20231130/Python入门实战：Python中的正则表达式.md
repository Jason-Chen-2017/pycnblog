## 1.1 Python的正则表达式简介

Python的正则表达式模块是re，它提供了一系列的函数和方法来处理字符串。正则表达式可以用来匹配、替换、分割等字符串操作。它是一种强大的字符串处理工具，可以用来解决各种字符串处理问题。

## 1.2 Python正则表达式的核心概念

正则表达式的核心概念包括：

- 字符集：表示一个字符集合，可以匹配该集合中的任意一个字符。
- 量词：表示一个字符或字符集的出现次数，可以匹配0次或多次。
- 组：表示一个子表达式，可以用于捕获匹配的内容。
- 位置和组的捕获：可以用于捕获匹配的内容，并将其存储到特定的变量中。
- 反向引用：可以用于引用之前的捕获组的内容。
- 贪婪和非贪婪模式：可以用于控制匹配的方式，贪婪模式会尽可能匹配更多的内容，而非贪婪模式会尽可能匹配更少的内容。

## 1.4 Python正则表达式的具体代码实例和详细解释说明

以下是一个具体的Python正则表达式的代码实例：

```python
import re

# 定义正则表达式模式
pattern = re.compile(r'^[a-zA-Z0-9]+@[a-zA-Z0-9]+\.[a-zA-Z0-9]+$')

# 使用正则表达式对象的方法进行匹配
match = pattern.match('test@example.com')
if match:
    print('匹配成功')
else:
    print('匹配失败')

# 使用组和捕获
match = pattern.match('test@example.com')
if match:
    print(match.group(1))

# 使用反向引用
match = pattern.match('test@example.com')
if match:
    print(match.group(1))

# 使用贪婪和非贪婪模式
match = pattern.match('test@example.com')
if match:
    print(match.group(1))
```
