# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释Go语言的文档生成工具的使用方法。

```go
package main

import (
    "fmt"
    "golang.org/x/tools/go/packages"
    "golang.org/x/tools/go/types"
)

func main() {
    // 加载代码包
    pkg, err := packages.Load("./example", "", nil, nil, false)
    if err != nil {
        fmt.Println("加载代码包失败:", err)
        return
    }

    // 遍历代码包中的类型
    for _, p := range pkg.Types {
        // 遍历类型的方法
        for _, m := range p.Methods() {
            // 提取方法的注释信息
            comment := m.Doc.Text()
            fmt.Println("方法名:", m.Name(), "注释:", comment)
        }
    }
}
```

在上述代码中，我们首先使用`packages.Load`函数加载了一个名为`example`的代码包。然后，我们遍历了代码包中的类型，并遍历了每个类型的方法。对于每个方法，我们提取了方法的注释信息，并将其打印到控制台上。
