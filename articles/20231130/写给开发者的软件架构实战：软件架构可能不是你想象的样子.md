例如，我们可以提供一个简单的冒泡排序算法的实现，如下所示：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个实例中，我们定义了一个名为 `bubble_sort` 的函数，它接受一个数组 `arr` 作为输入，并返回一个排序后的数组。我们使用了两个嵌套的循环来实现冒泡排序的过程，即将数组中的元素逐个与相邻元素进行比较和交换，直到整个数组被排序。

我们还可以提供一个简单的深度优先搜索算法的实现，如下所示：

```python
def dfs(graph, start):
    visited = set()
    stack = [start]
    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)
    return visited
```

在这个实例中，我们定义了一个名为 `dfs` 的函数，它接受一个图 `graph` 和一个起始顶点 `start` 作为输入，并返回一个访问顺序的集合。我们使用了一个栈来实现深度优先搜索的过程，即从起始顶点开始，访问当前顶点的所有未访问的邻居，并将它们推入栈中，直到栈为空。
