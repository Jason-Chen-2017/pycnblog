# 4.具体代码实例和详细解释说明
在本节中，我们将通过具体代码实例来帮助你更好地理解网络请求和数据获取的概念和操作。

## 4.1网络请求的具体代码实例
```python
import requests

url = 'https://www.example.com'
headers = {'User-Agent': 'Mozilla/5.0'}
response = requests.get(url, headers=headers)

if response.status_code == 200:
    print(response.text)
else:
    print('请求失败')
```
在上述代码中，我们使用requests库发起了一个GET请求，并将请求头设置为User-Agent。我们还检查了请求的状态码，以确定请求是否成功。

## 4.2数据获取的具体代码实例
```python
from bs4 import BeautifulSoup

html_data = response.text
soup = BeautifulSoup(html_data, 'html.parser')

# 提取信息
info = soup.find('div', {'class': 'info'})
print(info.text)
```
在上述代码中，我们使用BeautifulSoup库解析了HTML数据，并使用find方法提取了我们需要的信息。
