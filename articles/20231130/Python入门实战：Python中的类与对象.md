## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释Python中的类与对象。

```python
class Student:
    # 类属性
    count = 0

    # 实例属性
    name = ""
    age = 0

    # 类方法
    @classmethod
    def add_student(cls, name, age):
        cls.count += 1
        student = cls(name, age)
        return student

    # 静态方法
    @staticmethod
    def get_description(student):
        return f"{student.name}是一个{student.age}岁的学生"

# 创建学生对象
student1 = Student.add_student("张三", 20)
student2 = Student.add_student("李四", 21)

# 获取学生描述
description1 = Student.get_description(student1)
description2 = Student.get_description(student2)

# 输出描述
print(description1)
print(description2)
```

在上述代码中，我们定义了一个`Student`类，该类包含以下几个部分：

1. 类属性：`count`是类属性，用于记录学生的数量。
2. 实例属性：`name`和`age`是实例属性，用于存储每个学生的名字和年龄。
3. 类方法：`add_student`是类方法，用于创建学生对象并返回。
4. 静态方法：`get_description`是静态方法，用于获取学生的描述。

我们创建了两个学生对象`student1`和`student2`，并使用`get_description`方法获取他们的描述。最后，我们输出了描述。
