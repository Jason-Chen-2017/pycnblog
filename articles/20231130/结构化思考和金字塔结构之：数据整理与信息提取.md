# 2.核心概念与联系
结构化思考是指将复杂问题分解为更小的、更简单的子问题，然后逐步解决这些子问题，最终得到整个问题的解决方案。这种思维方式可以帮助我们更好地理解问题的本质，并找到更有效的解决方案。

金字塔结构是一种数据整理和信息提取的方法，它将数据按照层次结构组织，从最基本的数据层次到最高层次的信息层次。金字塔结构可以帮助我们更好地理解数据之间的关系，并提取出有价值的信息。
