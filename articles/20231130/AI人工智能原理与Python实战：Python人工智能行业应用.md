## 3.1 线性回归

线性回归是一种简单的机器学习算法，用于预测连续型变量的值。线性回归的核心思想是通过拟合一条直线来预测变量的值。线性回归的数学模型公式如下：

y = θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ

其中，y是预测的变量值，x₁、x₂、...、xₙ是输入变量，θ₀、θ₁、θ₂、...、θₙ是模型参数。

线性回归的具体操作步骤如下：

1. 数据准备：将输入变量和预测变量存储到数组中。

2. 初始化模型参数：将模型参数初始化为随机值。

3. 梯度下降：使用梯度下降算法来调整模型参数，使得预测变量与实际变量之间的差距最小化。

4. 训练完成：当模型参数收敛时，训练过程结束。

5. 预测：使用训练好的模型来预测新的输入变量的预测变量值。
