# 4.具体代码实例和详细解释说明
在这部分，我们将通过具体的代码实例来详细解释C++模板和异常处理的使用方法。

## 4.1 C++模板的使用方法
```cpp
#include <iostream>

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    int a = 10;
    int b = 20;
    int sum = add(a, b);
    std::cout << "sum = " << sum << std::endl;
    return 0;
}
```
在上述代码中，我们定义了一个泛型函数`add`，它接受两个模板参数`T`，并返回两个参数的和。在`main`函数中，我们使用了`add`函数，并将其应用于两个整数`a`和`b`。由于`add`函数是泛型的，它可以处理任何类型的数据，因此我们可以将其应用于整数、浮点数、字符串等不同类型的数据。

## 4.2 C++异常处理的使用方法
```cpp
#include <iostream>
#include <stdexcept>

int main() {
    int a = 10;
    int b = 0;
    int c = a / b;

    try {
        std::cout << "c = " << c << std::endl;
    } catch (std::exception& e) {
        std::cerr << "Exception caught: " << e.what() << std::endl;
    }

    return 0;
}
```
在上述代码中，我们使用了`try`和`catch`关键字来捕获和处理异常情况。在`main`函数中，我们尝试将`a`除以`b`，但由于`b`为0，因此会引发一个除零错误。当异常发生时，程序会跳转到`catch`块，并调用`std::exception`类的`what`方法来获取异常信息，然后将其打印到控制台。
