# 2.核心概念与联系
概率图模型（Probabilistic Graphical Models，PGM）是一种用于表示概率分布的图形表示方法，它可以用来描述随机变量之间的关系和依赖性。在自然语言处理中，概率图模型可以用来建模文本数据，例如文本分类、文本摘要、文本生成等任务。

概率图模型的核心概念包括：

1. 随机变量：概率图模型中的基本元素是随机变量，它表示一个不确定的事件或属性。
2. 边：边表示随机变量之间的关系，边上的方向表示因变量和导变量之间的关系。
3. 节点：节点表示随机变量，节点之间通过边连接起来形成图。
4. 条件独立性：概率图模型中的条件独立性是指给定某些变量的值，其他变量之间的关系是独立的。
