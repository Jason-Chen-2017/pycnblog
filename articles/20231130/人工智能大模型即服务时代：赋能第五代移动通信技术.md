# 2.核心概念与联系

在人工智能大模型即服务时代，我们需要关注以下几个核心概念：

1. **人工智能（AI）**：人工智能是一种通过计算机程序模拟人类智能的技术，旨在解决复杂问题、自主学习和决策等。

2. **大模型**：大模型是指具有大规模参数数量和复杂结构的人工智能模型，通常用于处理大量数据和复杂任务。

3. **即服务（aaS）**：即服务是一种软件交付模式，将复杂的技术功能作为服务提供给用户，让用户只需关注业务逻辑即可。
