# 4.具体代码实例和详细解释说明

以下是一个简单的Python代码实例，用于实现情感词典的优化：

```python
import nltk
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import LinearSVC

# 数据收集与预处理
data = ["I love this movie!", "This is the worst movie I've ever seen."]
data = [nltk.word_tokenize(sentence.lower()) for sentence in data]

# 词汇扩展
expanded_words = set(nltk.corpus.wordnet.synsets('love').lemmas()[0].name())
data = [word for word in data if word in expanded_words]

# 短语提取
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(data)
y = [1, 0]
clf = LinearSVC()
clf.fit(X, y)

# 情感标签更新
def update_labels(sentence):
    words = nltk.word_tokenize(sentence.lower())
    X = vectorizer.transform(words)
    label = clf.predict(X)
    return label

# 评估与优化
def evaluate(sentence, label):
    predicted_label = update_labels(sentence)
    accuracy = 1 if predicted_label == label else 0
    return accuracy

# 测试
sentence = "I hate this movie!"
label = evaluate(sentence, 0)
print("Accuracy:", label)
```

在这个代码实例中，我们首先收集了一些情感倾向的文本数据，并对其进行了预处理。然后，我们通过词汇拓展和短语提取来优化情感词典。最后，我们使用支持向量机算法来更新词典中的情感标签，并通过准确率来评估词典的性能。
