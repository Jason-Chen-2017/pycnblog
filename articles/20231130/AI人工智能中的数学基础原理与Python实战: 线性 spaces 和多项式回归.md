# 2.核心概念与联系

在深入探讨线性空间和多项式回归之前，我们需要了解一些基本概念。

## 2.1 向量和矩阵

向量是一个具有相同数量的数字元素组成的有序列表。矩阵是由行和列组成的元素的二维数组。向量可以看作是矩阵的特殊形式，具有相同数量的行和列。

## 2.2 线性组合

线性组合是将向量或矩阵的元素相加的过程。例如，对于一个向量v=[a,b,c]和一个矩阵M，我们可以对M的每一行进行线性组合，得到一个新的向量。

## 2.3 线性无关和线性相关

向量或矩阵的元素之间的线性关系可以分为线性无关和线性相关两种。线性无关的向量或矩阵的元素之间没有线性关系，即不存在一个常数，使得某个向量或矩阵的所有元素都可以表示为另一个向量或矩阵的线性组合。线性相关的向量或矩阵的元素之间存在线性关系。

## 2.4 基础和维数

基础是一个线性无关的向量集合，可以用来表示一个向量空间。向量空间的维数是基础的数量。

## 2.5 线性空间

线性空间是一个向量空间，其中所有向量都可以表示为线性组合。线性空间的基础是一个线性无关的向量集合，可以用来表示整个向量空间。

## 2.6 多项式

多项式是一个数学表达式，由一系列项组成，每个项都是一个数字乘以一个变量的某次幂。例如，2x^2+3x+1是一个二次多项式。

## 2.7 回归

回归是一种预测方法，用于根据给定的输入特征值预测输出值。回归模型可以是线性的，也可以是非线性的。

现在我们已经了解了一些基本概念，接下来我们将深入探讨线性空间和多项式回归的原理和算法。
