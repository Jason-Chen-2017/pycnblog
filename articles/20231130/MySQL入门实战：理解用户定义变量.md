1. 首先，我们需要在配置文件中定义这个变量。在MySQL中，我们可以通过设置 `my.cnf` 文件来定义用户定义变量。我们可以在 `[mysqld]` 部分添加以下内容：

```
[mysqld]

```

2. 接下来，我们需要重启MySQL服务器以使更改生效。

3. 现在，我们可以通过以下查询来访问这个变量的值：

```sql
SELECT @@global.current_date_time;
```

这将返回当前的日期和时间。

4. 如果我们需要修改这个变量的值，我们可以通过以下查询来实现：

```sql
SET GLOBAL current_date_time = '2022-01-01 00:00:00';
```

5. 如果我们需要删除这个变量，我们可以通过以下查询来实现：

```sql
DROP GLOBAL VARIABLE current_date_time;
```
