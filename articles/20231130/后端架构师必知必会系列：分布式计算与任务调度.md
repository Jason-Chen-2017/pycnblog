# 2.核心概念与联系

在分布式计算和任务调度中，有几个核心概念需要我们了解：

1. 计算节点：计算节点是分布式系统中的基本组件，它们负责执行计算任务。计算节点可以是单个计算机、服务器或其他设备。

2. 任务：任务是需要执行的计算任务，可以是单个任务或多个任务组成的任务集。

3. 任务调度：任务调度是将任务分配给适当计算节点的过程，以实现最佳的计算资源利用率和任务执行效率。

4. 数据一致性：在分布式计算中，数据一致性是指在分布式系统中的多个节点上的数据必须保持一致。

5. 故障恢复：当计算节点或任务出现故障时，分布式系统需要具有故障恢复能力，以确保系统的可用性和稳定性。
