# 2.核心概念与联系

## 2.1 概率论

概率论是一门数学分支，它研究随机事件发生的可能性。概率论的基本概念包括事件、样本空间、概率等。

### 2.1.1 事件

事件是随机过程中可能发生的某种结果。例如，在一个六面骰子上掷出某一面的结果就是一个事件。

### 2.1.2 样本空间

样本空间是所有可能发生的事件集合。在上述骰子掷法例子中，样本空间为{1,2,3,4,5,6}。

### 2.1.3 概率

概率是一个事件发生的可能性，通常表示为一个数值，范围在0到1之间。概率的计算方法有多种，包括直接计数、定义域的大小等。

## 2.2 统计学

统计学是一门数学分支，它研究从数据中抽取信息。统计学的基本概念包括数据、统计量、分布等。

### 2.2.1 数据

数据是从实际情况中收集的信息。数据可以是连续的（如温度、体重等）或离散的（如年龄、性别等）。

### 2.2.2 统计量

统计量是用于描述数据的一个数值。例如，平均值、中位数、方差等。

### 2.2.3 分布

分布是描述数据的一个数学模型。常见的分布有正态分布、泊松分布等。

## 4.1 例子：蒙特卡洛方法估计π

我们知道，在一个正方形内随机选取一个点，该点与正方形边的交点的距离的平均值为π/4。因此，我们可以通过生成大量随机点，计算其与正方形边的交点的距离的平均值，从而估计π的值。

以下是一个Python代码实例：

```python
import random
import math

def monte_carlo_pi(n):
    count = 0
    for _ in range(n):
        x = random.uniform(-1, 1)
        y = random.uniform(-1, 1)
        if x*x + y*y <= 1:
            count += 1
    pi = 4 * count / n
    return pi

n = 100000
pi = monte_carlo_pi(n)
print("估计的π值为：", pi)
```

在上述代码中，我们首先定义了一个随机变量（x, y），并确定其概率分布（均匀分布）。然后，我们生成了大量随机样本（n个），并计算其与正方形边的交点的距离的平均值。最后，我们将随机样本的平均值作为估计π的值。
