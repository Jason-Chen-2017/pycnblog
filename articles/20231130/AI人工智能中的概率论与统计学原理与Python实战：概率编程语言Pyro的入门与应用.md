具体操作步骤如下：

1. 导入Pyro库：
```python
import pyro
import pyro.distributions as dist
import pyro.plate as plate
```
2. 定义概率图模型：
```python
with plate.Plate(name='plate', dims=2, size=10):
    x = plate.Variable(dist.Normal(0, 1))
    y = plate.Variable(dist.Normal(x, 1))
```

3. 进行概率计算：
```python
pyro.sample('plate', pyro.sample_plate(x, y))
```
