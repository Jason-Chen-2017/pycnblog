# 4.具体代码实例和详细解释说明
以下是一个简单的Python代码实例，用于对声音信号进行傅里叶变换和频域分析：

```python
import numpy as np
import matplotlib.pyplot as plt

# 读取声音信号
signal = np.load('signal.npy')

# 对声音信号进行傅里叶变换
fourier_transform = np.fft.fft(signal)

# 计算频域信号的幅度和相位
magnitude = np.abs(fourier_transform)
phase = np.angle(fourier_transform)

# 绘制频域信号的幅度和相位
plt.figure()
plt.subplot(2, 1, 1)
plt.plot(magnitude)
plt.title('Magnitude')
plt.subplot(2, 1, 2)
plt.plot(phase)
plt.title('Phase')
plt.show()
```

这个代码首先读取了声音信号，然后对其进行傅里叶变换，从而得到频域信号的幅度和相位。最后，使用matplotlib库绘制了频域信号的幅度和相位。
