# 4.具体代码实例和详细解释说明
在这里，我们将给出一个具体的代码实例，并进行详细解释：

```python
def longest_common_substring(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    longest, end = 0, 0
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                if dp[i][j] > longest:
                    longest = dp[i][j]
                    end = i
            else:
                dp[i][j] = 0
    return s1[end - longest:end]
```

这段代码实现了最长公共子序列的算法。我们首先创建了一个二维数组dp，用于存储子串长度。然后，我们遍历字符串s1和s2，如果当前字符相等，我们更新dp数组，并更新最长子串的长度和结束位置。最后，我们返回最长公共子序列。
