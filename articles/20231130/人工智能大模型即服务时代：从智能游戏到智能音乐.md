## 4.1 智能游戏的代码实例
以下是一个简单的智能游戏的代码实例：

```python
import random

class Game:
    def __init__(self):
        self.rules = Rules()
        self.environment = Environment()
        self.ai = AI()

    def play(self):
        while True:
            self.environment.update()
            action = self.ai.choose_action(self.environment.get_state())
            reward = self.rules.get_reward(action)
            self.environment.update(action)
            self.ai.learn(action, reward)

if __name__ == '__main__':
    game = Game()
    game.play()
```

在这个代码实例中，我们首先定义了一个 `Game` 类，这个类包括游戏的规则、游戏的环境和游戏的人工智能算法。然后，我们实例化了一个 `Game` 对象，并调用其 `play` 方法来开始游戏。
