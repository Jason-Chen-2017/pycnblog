# 2.核心概念与联系

## 2.1 随机变量与条件独立

随机变量是一个可能取多个值的变量，每个值的出现概率都是确定的。条件独立是指两个随机变量的值之间，给定某个或某些其他随机变量的值，是不相关的。

## 2.2 条件概率与贝叶斯定理

条件概率是一个事件发生的概率，给定另一个事件发生的情况。贝叶斯定理是一种概率推理方法，它可以用来计算条件概率。贝叶斯定理的公式为：

P(A|B) = P(B|A) * P(A) / P(B)

其中，P(A|B) 是条件概率，表示事件A发生的概率，给定事件B发生的情况；P(B|A) 是条件概率，表示事件B发生的概率，给定事件A发生的情况；P(A) 是事件A发生的概率；P(B) 是事件B发生的概率。

## 2.3 贝叶斯网络

贝叶斯网络是一种有向无环图（DAG），其节点表示随机变量，边表示变量之间的条件依赖关系。贝叶斯网络可以用来表示和推理概率关系。
