# 2.核心概念与联系

在了解容器安全与容器漏洞管理之前，我们需要了解一些核心概念：

- **容器：** 容器是一种轻量级的应用程序部署和管理方式，它将应用程序和其依赖项打包在一个可移植的镜像中，并在运行时与宿主操作系统共享资源。

- **容器漏洞：** 容器漏洞是指容器镜像或运行时环境中存在的安全问题，可能导致容器被攻击或受到损害。

- **容器安全性：** 容器安全性是指容器镜像和运行时环境的安全性，包括防止容器漏洞的发生和发现，以及确保容器运行时的安全性。
