# 2.核心概念与联系

在数据虚拟化技术中，核心概念包括数据源、元数据、数据模型、虚拟数据和虚拟表。

- 数据源：数据源是数据虚拟化技术的基础，它可以是关系数据库、NoSQL数据库、Hadoop等多种类型的数据源。
- 元数据：元数据是数据源的描述信息，包括数据源的结构、数据类型、约束等信息。
- 数据模型：数据模型是虚拟数据的抽象，它定义了虚拟数据的结构、数据类型、约束等信息。
- 虚拟数据：虚拟数据是通过数据虚拟化技术将多种数据源虚拟成一个统一的数据资源。
- 虚拟表：虚拟表是虚拟数据的具体表现形式，它是虚拟数据的一种访问接口。

数据虚拟化技术的核心联系是将数据源的元数据和数据进行抽象，通过一种统一的数据模型和访问接口来实现数据的统一管理和访问。
