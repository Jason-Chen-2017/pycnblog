# 4.具体代码实例和详细解释说明

以下是一个简单的代码实例，用于说明电商平台无服务器架构的具体实现：

```python
import boto3

def lambda_handler(event, context):
    # 获取请求参数
    product_id = event['product_id']
    # 调用数据库服务
    dynamodb = boto3.resource('dynamodb')
    table = dynamodb.Table('products')
    # 查询产品信息
    response = table.get_item(
        Key={
            'product_id': product_id
        }
    )
    # 获取产品信息
    product_info = response['Item']
    # 返回产品信息
    return {
        'statusCode': 200,
        'body': product_info
    }
```

在上述代码中，我们使用了无服务器架构的函数计算功能，将代码上传到云端，并配置了数据库服务的集成。当用户请求某个产品的信息时，无服务器架构会自动触发函数计算，并调用数据库服务来查询产品信息。
