# 2.核心概念与联系
在概率论中，我们主要关注的是随机事件的发生概率。概率是一个数值，表示某个事件发生的可能性。概率通常表示为一个0到1之间的数值，其中0表示事件不可能发生，1表示事件必然发生。

概率论与其他数学分支有很强的联系。例如，统计学是概率论的一个重要应用领域，通过对大量数据进行分析，我们可以得出关于某个事件发生概率的结论。同时，概率论也与信息论、线性代数等数学分支有密切的联系。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
在概率论中，我们主要关注的是几种基本概率模型，如泊松分布、二项分布、多项分布等。这些模型可以用来描述不同类型的随机事件。

## 3.1 泊松分布
泊松分布是一种描述连续随机事件发生的概率分布。泊松分布的参数是λ（lambda），表示事件发生的平均率。泊松分布的概率密度函数为：

P(X=k) = (e^(-λ) * λ^k) / k!

其中，k是事件发生的次数，e是基于自然对数的底数。

## 3.2 二项分布
二项分布是一种描述离散随机事件发生的概率分布。二项分布的参数是n（n trials）和p（p success），表示试验的次数和成功的概率。二项分布的概率密度函数为：

P(X=k) = C(n, k) * p^k * (1-p)^(n-k)

其中，C(n, k)是组合数，表示从n个事件中选择k个的方法数。

## 3.3 多项分布
多项分布是一种描述多个独立随机事件发生的概率分布。多项分布的参数是n（n trials）和p（p success），表示试验的次数和成功的概率。多项分布的概率密度函数为：

P(X=k) = C(n, k) * p^k * (1-p)^(n-k)

其中，C(n, k)是组合数，表示从n个事件中选择k个的方法数。

## 4.1 泊松分布
```python
import math

def poisson_probability(lambda_, k):
    e = math.exp(1)
    return (e**(-lambda_) * (lambda_**k)) / math.factorial(k)

# 例子
lambda_ = 2
k = 1
print(poisson_probability(lambda_, k))
```
在上述代码中，我们首先导入了math模块，用于计算指数和阶乘。然后我们定义了一个名为poisson_probability的函数，用于计算泊松分布的概率。最后，我们通过一个例子来展示如何使用这个函数。

## 4.2 二项分布
```python
def binomial_probability(n, p, k):
    return math.comb(n, k) * (p**k) * ((1-p)**(n-k))

# 例子
n = 5
p = 0.5
k = 2
print(binomial_probability(n, p, k))
```
在上述代码中，我们首先定义了一个名为binomial_probability的函数，用于计算二项分布的概率。然后我们通过一个例子来展示如何使用这个函数。

## 4.3 多项分布

def multinomial_probability(n, p, k):
    return math.comb(n, k) * (p**k) * ((1-p)**(n-k))
