以排序算法为例，我们可以看到以下代码实例：

```python
def sort_list(lst):
    # 排序算法的实现
    for i in range(len(lst)):
        for j in range(i+1, len(lst)):
            if lst[i] > lst[j]:
                lst[i], lst[j] = lst[j], lst[i]
    return lst

lst = [5, 2, 8, 1, 9]
sorted_lst = sort_list(lst)
print(sorted_lst)
```

在这个代码实例中，我们定义了一个名为`sort_list`的函数，用于实现排序算法。我们传入一个列表，然后通过函数内部的操作来实现列表的排序。
