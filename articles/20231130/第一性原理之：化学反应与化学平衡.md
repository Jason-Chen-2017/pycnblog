# 2.核心概念与联系

## 2.1化学反应

化学反应是指化学物质在化学过程中发生的变化，即化学物质之间的转化过程。化学反应可以分为两种：单反应和复反应。单反应是指只涉及两种化学物质的反应，如水化碳与氢的反应。复反应是指涉及多种化学物质的反应，如水化碳与氢和氧气的反应。

化学反应可以进一步分为以下几种类型：

- 成分反应：成分反应是指化学物质在化学过程中发生的变化，但不涉及外来物质的反应。例如，水化碳与氢的反应。
- 外来物质反应：外来物质反应是指化学物质在化学过程中发生的变化，并涉及外来物质的反应。例如，水化碳与氢和氧气的反应。
- 自发反应：自发反应是指化学物质在化学过程中发生的变化，并且不需要外来能量的反应。例如，水化碳与氢的反应。
- 外来能量反应：外来能量反应是指化学物质在化学过程中发生的变化，并且需要外来能量的反应。例如，水化碳与氢和氧气的反应。

## 2.2化学平衡

化学平衡是指化学反应在进行过程中，化学物质的分子数和质量相等，即反应方程式的两边的化学物质数量相等。化学平衡是化学反应的一个重要特征，可以用化学方程式来表示。

化学平衡可以分为以下几种类型：

- 完全平衡：完全平衡是指化学反应在进行过程中，反应方程式的两边的化学物质数量相等，即反应方程式的两边的分子数和质量相等。例如，水化碳与氢的反应。
- 部分平衡：部分平衡是指化学反应在进行过程中，反应方程式的两边的化学物质数量不完全相等，即反应方程式的两边的分子数和质量不完全相等。例如，水化碳与氢和氧气的反应。
