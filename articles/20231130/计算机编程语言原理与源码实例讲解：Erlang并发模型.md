                 

# 1.背景介绍

Erlang是一种功能性编程语言，主要应用于分布式系统和并发编程。它的并发模型是其最重要的特点之一，能够轻松地处理大量并发任务。Erlang的并发模型是基于消息传递的，它使用轻量级进程（lightweight processes，LWP）来实现并发，这些进程之间通过发送和接收消息来进行通信。

Erlang的并发模型具有以下特点：

1. 轻量级进程：Erlang中的进程非常轻量级，创建和销毁进程非常快速，并且进程之间相互独立，不会因为其他进程的错误而导致整个系统崩溃。

2. 消息传递：Erlang的并发模型是基于消息传递的，进程之间通过发送和接收消息来进行通信。这种模型具有高度并发性，可以轻松地处理大量并发任务。

3. 分布式系统支持：Erlang的并发模型支持分布式系统，可以轻松地实现跨机器的并发任务。

4. 错误处理：Erlang的并发模型具有强大的错误处理能力，可以在运行时检测并处理错误，确保系统的稳定性和可靠性。
