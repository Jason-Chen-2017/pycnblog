# 2.核心概念与联系

在了解MySQL的核心技术原理之前，我们需要了解一些关键的核心概念。这些概念包括：

- 数据库：数据库是一种用于存储、管理和查询数据的系统。它由一组相关的表组成，表由一组行和列组成。
- 表：表是数据库中的基本组件，用于存储数据。表由一组列组成，列表示数据的属性，行表示数据的实例。
- 索引：索引是一种数据结构，用于加速数据的查询。索引允许数据库引擎快速定位特定的数据行。
- 查询优化：查询优化是一种技术，用于提高数据库查询的性能。查询优化涉及到选择最佳的查询方法、使用最佳的索引和避免不必要的表连接等。
- 性能调优：性能调优是一种技术，用于提高数据库的整体性能。性能调优涉及到数据库配置、硬件优化、查询优化等方面。
