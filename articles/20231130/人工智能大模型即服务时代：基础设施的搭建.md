# 2.核心概念与联系

在构建大模型基础设施之前，我们需要了解一些核心概念。这些概念包括：

- 大模型：大模型是指规模较大的神经网络模型，通常包含大量的参数和层次。这些模型在训练和部署过程中需要大量的计算资源和存储空间。
- 分布式训练：由于大模型的规模，我们需要使用分布式训练技术来训练这些模型。分布式训练允许我们将模型训练任务分解为多个子任务，并在多个计算节点上同时进行训练。
- 模型部署：模型部署是指将训练好的模型部署到生产环境中，以便在实际应用中使用。模型部署需要考虑多种因素，例如性能、资源利用率和安全性等。
