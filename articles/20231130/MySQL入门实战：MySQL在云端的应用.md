# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来说明MySQL在云端的应用。

假设我们需要在云端创建一个简单的数据库，用于存储用户信息。以下是创建数据库和表的代码实例：

```sql
-- 创建数据库
CREATE DATABASE my_db;

-- 选择数据库
USE my_db;

-- 创建表
CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(255) NOT NULL,
  email VARCHAR(255) UNIQUE NOT NULL
);

-- 插入数据
INSERT INTO users (name, email)
VALUES ('John Doe', 'john@example.com');

-- 查询数据
SELECT * FROM users;
```

在这个代码实例中，我们首先创建了一个名为`my_db`的数据库，然后选择了这个数据库。接着，我们创建了一个名为`users`的表，该表包含`id`、`name`和`email`字段。最后，我们插入了一条用户记录，并查询了所有用户记录。
