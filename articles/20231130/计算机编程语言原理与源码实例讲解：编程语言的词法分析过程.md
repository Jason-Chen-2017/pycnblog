# 2.核心概念与联系

在计算机编程语言中，词法分析是编译器和解释器的一个重要组成部分。词法分析器负责将源代码划分为一系列的词法单元，这些词法单元将为后续的语法分析提供基础。

词法分析器的核心概念包括：

- 词法单元：词法分析器将源代码划分为一系列的词法单元，如标识符、关键字、数字、符号等。
- 正则表达式：词法分析器通常使用正则表达式或其他模式来识别词法单元。
- 扫描器：词法分析器也被称为扫描器，它负责识别源代码中的字符串并将其划分为词法单元。

词法分析器与语法分析器之间的联系是：词法分析器将源代码划分为词法单元，并将这些词法单元传递给语法分析器。语法分析器则负责将这些词法单元组合成有意义的语法结构，以便计算机可以理解和执行源代码。

# 4.具体代码实例和详细解释说明

以下是一个简单的词法分析器的Python代码实例：

```python
import re

def lexer(source_code):
    tokens = []
    pattern = r"[a-zA-Z_][a-zA-Z0-9_]+"
    for char in source_code:
        if re.match(pattern, char):
            tokens.append(char)
    return tokens

source_code = "int a = 10; float b = 3.14;"
tokens = lexer(source_code)
print(tokens)
```

在这个代码实例中，我们使用Python的re模块来实现词法分析器。我们定义了一个名为lexer的函数，该函数接受源代码文件的内容作为参数。在函数内部，我们使用正则表达式来识别源代码中的标识符。如果识别成功，则将该标识符添加到tokens列表中。

在这个例子中，源代码为"int a = 10; float b = 3.14;"。当我们运行程序时，输出结果为：["int", "a", "=", "10", ";", "float", "b", "=", "3.14", ";"]。这些词法单元将为后续的语法分析提供基础。
