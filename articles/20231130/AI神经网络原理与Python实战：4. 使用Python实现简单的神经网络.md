# 2.核心概念与联系

神经网络的核心概念包括神经元、权重、激活函数和损失函数。神经元是神经网络的基本组件，它接收输入，对其进行处理，并输出结果。权重是神经元之间的连接，它们决定了输入和输出之间的关系。激活函数是神经元的输出函数，它将输入映射到输出。损失函数是用于衡量神经网络的性能的函数，它将神经网络的预测与实际结果进行比较。

神经网络的核心联系包括前向传播、反向传播和梯度下降。前向传播是从输入层到输出层的数据传递过程，它用于计算神经网络的输出。反向传播是从输出层到输入层的数据传递过程，它用于计算神经网络的梯度。梯度下降是优化神经网络权重的方法，它通过不断更新权重来最小化损失函数。

## 3.1 前向传播

前向传播是神经网络的主要计算过程，它从输入层到输出层传递数据。在前向传播过程中，每个神经元的输出是其前一个神经元的输出和权重的线性组合，然后通过激活函数进行非线性变换。

具体操作步骤如下：

1. 对于每个输入样本，将输入样本的特征值与权重矩阵相乘，得到隐藏层神经元的输入。
2. 对于每个隐藏层神经元，将其输入与权重矩阵相乘，得到输出层神经元的输入。
3. 对于每个输出层神经元，将其输入与权重矩阵相乘，得到输出层神经元的输出。
4. 将输出层神经元的输出与真实标签进行比较，计算损失函数的值。
5. 对损失函数进行梯度计算，得到权重矩阵的梯度。
6. 使用梯度下降法更新权重矩阵。

数学模型公式如下：

输入层到隐藏层的输入：$h_i = \sum_{j=1}^{n} w_{ij}x_j + b_i$

隐藏层到输出层的输入：$y_i = \sum_{j=1}^{m} w_{ij}h_j + b_i$

激活函数：$f(x) = \frac{1}{1 + e^{-x}}$

损失函数：$L = \frac{1}{2n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$

梯度下降：$w_{ij} = w_{ij} - \alpha \frac{\partial L}{\partial w_{ij}}$

A：激活函数是神经元的输出函数，它将神经元的输入映射到输出。常见的激活函数有sigmoid、tanh和ReLU等。

Q：什么是损失函数？

A：损失函数是用于衡量神经网络的性能的函数，它将神经网络的预测与实际结果进行比较。常见的损失函数有均方误差、交叉熵损失等。

A：梯度下降是优化神经网络权重的主要方法，它通过不断更新权重来最小化损失函数。
