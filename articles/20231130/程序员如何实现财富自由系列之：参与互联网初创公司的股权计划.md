## 2.1 股权证书
股权证书是证明股东所拥有的股权的证明文件。股权证书包含了股东的姓名、股票数量、股票类型等信息。

## 2.2 股票
股票是公司的一种财产，代表公司的一部分所有权。股票可以在股票市场上买卖，股票价格随公司的业绩和市场环境而波动。

## 2.3 股权合同
股权合同是股东与公司之间的合同，规定了股东的权利和义务。股权合同包括了股权转让、股权抵押、股权转让等方面的规定。

        self.stock_type = stock_type

    def distribute_stock(self, stock_number):

        return stock_number

    def transfer_stock(self, stock_number):
        self.stock_number -= stock_number
        return stock_number

    def pledge_stock(self, stock_number):
        self.stock_number -= stock_number
        return stock_number
