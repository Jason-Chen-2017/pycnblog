1. 选择排序：选择排序是一种简单的排序算法，它的基本思想是在每次循环中找到数组中最小的元素，并将其与数组的第一个元素交换。以下是选择排序的Python代码实例：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

2. 二分搜索：二分搜索是一种高效的搜索算法，它的基本思想是将数据集分成两个部分，然后将中间的元素与目标元素进行比较，如果相等则返回其索引，否则将数据集分成两个部分，然后重复上述过程。以下是二分搜索的Python代码实例：

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)
