# 2. 背景介绍
## 2.1 概览
自驾车是指不需要驾驶人的参与、完全由机器驱动的车辆。截至目前，国内已有两款拥有自主驾驶能力的车型，分别是Tesla Model S和特斯拉Model Y，它们分别采用激光雷达、激光雷达相机、电子巡航控制、双LiDAR激光雷达等传感器进行自主导航。2017年，特斯拉宣布推出了第二代Model 3，将自动驾驶平台引入到旅客座舱车厢。

由于自驾车的高度敏感性、复杂性和多样性，使得自动驾驶技术一直处于蓬勃发展阶段。现有的自主驾驶系统还存在很多技术瓶颈、难点、缺陷，比如：
* 模型训练困难：模拟驾驶环境、设计行驶场景、收集驾驶数据、训练模型需要耗费大量的人力物力。
* 模型准确率差距大：针对每种环境和车型，模拟驾驶数据极少或者不存在，因此模型准确率必然存在偏差。
* 操作技术复杂：自主驾驶系统需要能够识别各种各样的动态变化，例如车辆位置、角度、速度、环境、场景等，并且具备相应的反馈策略才能保证安全和快速响应。
* 环境适应性差：不同环境下自主驾驶系统性能差异较大，不同的交通工具适用性也不同，同时需要考虑自主驾驶系统的鲁棒性。

因此，为了解决上述问题，中国国家重点实验室智能交通所（NCLT）于2018年在国际自动驾驶大赛中获得冠军，取得了很好的成绩。该团队首先提出了一套基于机器学习、强化学习的多目标决策优化方法，然后结合了专门针对自驾车的无人机、小轿车的激光雷达等传感器，构建了具有完整功能的自主驾驶系统。

从目前的情况看，中国的自动驾驶技术仍处于起步阶段，我们仍处于自驾车技术革命的起点。但是，随着近年来开源技术的发展，国内的自动驾驶技术也正在蓬勃发展，探索新的可能性。另外，随着工业互联网的发展，人们对自动驾驶的需求也越来越强烈。因此，相信未来，可以看到更多的自驾车产品出现在市场上。
