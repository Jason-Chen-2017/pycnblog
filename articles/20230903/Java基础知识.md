# 1.简介
  

Java 是由Sun Microsystems公司于1995年推出的一门面向对象的通用编程语言，是目前最流行的多种编程语言之一，并被广泛应用于网络开发、移动应用开发、分布式系统开发等领域。其强大的功能和高效率使得它成为一种热门技术。

# 2.Java概述
## 2.1 Java历史
### 1995年：Java诞生
1995年5月23日，为了打造一个开放源代码的可移植的开发环境，<NAME>和同事们决定开发一个名为“Oak”的基于C++开发的集成开发环境（Integrated Development Environment）。由于当时Sun还没有商业版的Java，因此需要自己找别人授权才可使用。不幸的是，此时的Oak只是一个很原始的版本，没有商业支持也无法进行销售。

1996年底，Duke大学实验室的一群工程师开始着手开发Java。他们开始着手编写一个名为Green的“安全”的编译器，可以消除一些Java代码可能带来的安全隐患。由于Green的设计初衷是作为编译器运行在虚拟机上，因此它的语法非常类似于Java。

1997年底，Java1.0正式发布。这是第一个商业版本，主要特性是自动内存管理、异常处理、反射机制、多线程编程、动态绑定。此外还有其他特性如Java Beans、RMI、JDBC等。

1998年初，JavaOne大会召开。这是一个Java用户和开发人员的盛会，吸引了许多著名的Java大牛参加。

2009年，Oracle收购BEA Systems公司之后，Oracle宣布Java将正式改名为Java SE。Java SE 7是Java 7 Update Release的简称，同时Java SE 8是Java 8 Update Release的简称。

### 2009年：OpenJDK诞生
Java社区发展迅速，Sun公司的垄断地位逐渐受到质疑。为了维护自己的霸主地位，OpenJDK应运而生。OpenJDK是一个免费并且完全开源的Java开发工具包（SDK），由Linux开源社区及其他厂商共同开发。OpenJDK的目标是捆绑Java开发所需的所有必要软件，包括Java Runtime Environment(JRE)和Java Development Kit (JDK)。

OpenJDK的官方网站：http://openjdk.java.net/

### 2011年：JDK9发布
2011年12月10日，JDK9正式发布。JDK 9引入了许多新特性，例如：增强型模式匹配、私有接口方法、JShell、HTTP客户端、增强的压缩类库、多CPU体系结构支持等。JDK 9采用基于模块化的开发方式，可以只下载所需的模块来节省空间和时间。

此外，JDK 9也修复了之前的Bug，包括JEP 223、JEP 224、JEP 225、JEP 226、JEP 227、JEP 228等。

## 2.2 Java优点
1.跨平台性：Java可以在任何操作系统平台上运行，而且有各个厂商提供的编译器或解释器，可以实现源代码到机器码的转换，即编译运行。

2.安全性：Java拥有高度安全性，在执行代码时不会产生危害环境的恶意行为。因为JVM的作用，Java代码执行的过程与系统操作系统隔离。另外Java通过编译器的检查，可以防止程序员犯错误。

3.易学习性：Java具有简单、容易学习的语法和结构，开发效率较高，且很多公司已经将Java作为首选开发语言。

4.内存管理：Java提供自动内存管理，对于动态分配的内存，Java会负责回收，从而避免内存泄漏。

5.支持多线程：Java支持多线程编程，允许多个任务同时执行，提升计算性能。

6.虚拟机支持：Java运行在虚拟机上，所以Java代码可以运行在不同操作系统平台上的虚拟机上，甚至可以运行在Android手机上面。

7.反射：Java提供反射机制，可以通过对象动态调用自身的属性和方法。

8.数据库连接：Java提供了JDBC（Java Database Connectivity）接口，可以轻松连接关系型数据库和非关系型数据库。

9.可靠性：由于Java是静态编译的，所以它有非常好的可靠性。这一优势使得Java成为云计算、移动应用、嵌入式设备、企业级开发的重要选择。
