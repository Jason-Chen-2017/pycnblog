## 2.基本概念术语说明
### 数据类型
Python支持多种数据类型，包括整数、浮点数、字符串、列表、元组、集合、字典等。其中数字类型有int（整数）、float（浮点数），字符型有str（字符串），其他的数据类型则需要根据具体情况使用不同的容器。
### 变量作用域
变量作用域指的是某个变量名称所存在的区域范围，或者说一个变量可以被定义的区域。在Python中，变量的作用域分为全局作用域和局部作用域两种。全局作用域指的是在函数外部，也就是在模块内声明的变量；局部作用域指的是在函数内部声明的变量，这些变量只能在函数内部访问。
### 可变对象和不可变对象
Python中，对象的可变性和不可变性是非常重要的一件事情。在Python中，数字、字符串、元组都是不可变对象，而列表、字典是可变对象。不可变对象意味着它们的值不能改变，如果尝试修改其值，会产生一个新的对象，旧的对象则不再存在了。可变对象则不同，它的元素可以改变，而且这些变化是直接反映在内存上，不需要重新创建对象。
### 流程控制语句
流程控制语句是在计算机执行程序时用来控制程序流程的命令。Python中有if-else、for循环、while循环、try-except语句等。
### 函数
函数是一种独立的代码片段，它可以接受参数作为输入，并返回输出结果。函数的定义语法如下：
```python
def function_name(parameter):
    # do something here
    return output
```
### 模块
模块是一系列相关代码的集合，一般存放在一个文件中，通过import语句引入到程序中使用。模块的导入语法如下：
```python
import module_name as alias
from module_name import func, var
```
## 3.核心算法原理和具体操作步骤以及数学公式讲解
### 四舍五入
Python中的四舍五入函数round()可以实现四舍五入的功能。该函数的调用语法为：
```python
result = round(number)
```
其中number表示需要进行四舍五入的数值。默认情况下，round()函数会返回一个最接近的整数。如果需要指定保留几位小数，可以使用第二个参数ndigits：
```python
result = round(number, ndigits)
```
其中ndigits表示需要保留的小数位数。比如round(3.1415926535, 2)就会返回3.14。
### 随机数生成
在Python中，用于生成随机数的模块random提供了两个方法。首先是seed()方法，它可以设置随机数生成器的种子，使得每次运行程序都生成相同的随机数序列。其次是randint()方法，它可以随机地生成一个指定范围内的整数。调用语法如下：
```python
import random
random.seed(a=None)   # 设置种子
num = random.randint(start, stop)    # 生成随机整数
```
其中start和stop分别表示要生成的整数的最小值和最大值。
