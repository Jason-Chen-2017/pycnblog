## 4. 数学模型和公式 & 详细讲解 & 举例说明

在开源项目中，数学模型和公式是算法设计的重要基础。以下是一个简单的数学模型和公式的讲解。

### 4.1 数学模型构建

1. **线性回归模型**：
   - **目标函数**：$$J(\theta) = \frac{1}{2m}\sum_{i=1}^{m}(h_\theta(x^{(i)}) - y^{(i)})^2$$
   - **梯度下降**：$$\theta_j := \theta_j - \alpha \frac{\partial J(\theta)}{\partial \theta_j}$$

2. **神经网络模型**：
   - **激活函数**：$$a_{ij} = \sigma(z_{ij})$$
   - **反向传播**：$$\delta_j = (a_j - y_j) \cdot \frac{da_j}{dz_j}$$

### 4.2 公式推导过程

1. **线性回归模型推导**：
   - **损失函数**：$$J(\theta) = \frac{1}{2m}\sum_{i=1}^{m}(h_\theta(x^{(i)}) - y^{(i)})^2$$
   - **偏导数**：$$\frac{\partial J(\theta)}{\partial \theta_j} = \frac{1}{m}\sum_{i=1}^{m}(h_\theta(x^{(i)}) - y^{(i)}) \cdot x_j^{(i)}$$
   - **梯度下降**：$$\theta_j := \theta_j - \alpha \frac{\partial J(\theta)}{\partial \theta_j}$$

2. **神经网络模型推导**：
   - **前向传播**：$$z_{ij} = \sum_{k=1}^{n}w_{ik}a_{ij}^{(k-1)} + b_j$$
   - **后向传播**：$$\delta_j = (a_j - y_j) \cdot \frac{da_j}{dz_j}$$
   - **权重更新**：$$w_{ij} := w_{ij} - \alpha \cdot \delta_j \cdot a_{ij}^{(k-1)}$$
   - **偏导数**：$$\frac{\partial J(\theta)}{\partial w_{ij}} = \delta_j \cdot a_{ij}^{(k-1)}$$

   - **目标**：预测样本的标签$y$。

### 5.2 源代码详细实现

以下是一个简单的线性回归模型的实现：

```python
import numpy as np
import matplotlib.pyplot as plt

# 数据集
X = np.array([[1, 2], [2, 3], [3, 4]])
y = np.array([2, 3, 4])

# 梯度下降
def gradient_descent(X, y, theta, alpha, num_iters):
    m = len(y)
    J_history = []

    for i in range(num_iters):
        h = np.dot(X, theta)
        error = h - y
        theta = theta - alpha * (1/m) * np.dot(X.T, error)
        J_history.append(np.linalg.norm(error)**2)

    return theta, J_history

# 模型训练
theta = np.array([0, 0])
alpha = 0.01
num_iters = 1000

theta, J_history = gradient_descent(X, y, theta, alpha, num_iters)

# 模型预测
h = np.dot(X, theta)

# 绘图
plt.plot(J_history)
plt.xlabel('Iterations')
plt.ylabel('J(θ)')
plt.title('Gradient Descent J(θ) History')
plt.show()
```
