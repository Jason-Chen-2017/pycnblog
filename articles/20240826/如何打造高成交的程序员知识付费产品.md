### 5.2 源代码详细实现

以下是一个简单的协同过滤推荐算法实现，用于为用户推荐课程。

```python
import numpy as np
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

def load_data(filename):
    data = pd.read_csv(filename)
    ratings = data.pivot(index='user_id', columns='movie_id', values='rating')
    return ratings

def collaborative_filtering(ratings, user_id):
    user_ratings = ratings[user_id]
    similarity = cosine_similarity(ratings, ratings[user_id])
    similarity[user_id] = 0
    recommendations = {}
    for other_user in range(similarity.shape[0]):
        if other_user == user_id:
            continue
        similarity_score = similarity[other_user][0]
        for movie_id, rating in ratings[other_user].items():
            if rating > 0 and movie_id not in user_ratings:
                recommendations[movie_id] = rating * similarity_score
    return recommendations

if __name__ == '__main__':
    ratings = load_data('data.csv')
    user_id = 1001
    recommendations = collaborative_filtering(ratings, user_id)
    print(recommendations)
```

3. 计算相似度：使用余弦相似度计算用户之间的相似度。
4. 推荐算法：根据相似度计算，为用户推荐未观看的电影。
