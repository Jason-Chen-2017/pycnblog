## 2. 核心概念与联系

### 2.1 认知心理学基础

认知心理学是研究人类思维过程的科学。它关注大脑如何处理信息，如何形成知识，以及如何运用这些知识进行决策和解决问题。在认知心理学中，以下几个核心概念对理解思维模式至关重要：

- **认知地图（Cognitive Map）**：个体在大脑中构建的内化世界模型，用于导航和决策。
- **心智模型（ Mental Model）**：个体对现实世界的理解和假设，影响其决策和行为。
- **认知框架（Cognitive Frame）**：个体在特定情境下使用的认知结构，影响其信息加工和解释。

### 2.2 思维模式与领导力

思维模式与领导力之间存在密切的联系。领导者的思维模式决定了他们的决策风格、沟通方式以及对问题的处理方法。以下是几种常见的思维模式及其对领导力的影响：

- **分析型思维（Analytic Thinking）**：强调逻辑推理和细节分析，有利于制定精确的决策和策略，但也可能导致过度分析，延误决策。
- **创新型思维（Creative Thinking）**：注重发散思维和创造性解决方案，有助于团队创新和突破，但可能缺乏实际可行性。
- **系统型思维（Systems Thinking）**：关注整体和系统的动态关系，有助于领导者从全局角度看待问题，但可能忽视局部细节。

import matplotlib.pyplot as plt

# 定义权重
weights = np.array([0.3, 0.2, 0.2, 0.2, 0.1])

# 初始思维模式
initial_m = np.array([0.5, 0.5, 0.5])

# 设定改进目标
goal_m = np.array([0.6, 0.6, 0.6])

# 模拟提升过程
def simulate_lift(m, goal_m, num_steps):

        delta_m = goal_m - m
        m = m + 0.1 * delta_m

    return m

plt.plot(range(1, 11), m_new)
plt.xlabel("Steps")
plt.ylabel("Mindset Score")

plt.show()

import numpy as np
import matplotlib.pyplot as plt

# 定义权重
weights = np.array([0.3, 0.2, 0.2, 0.2, 0.1])

# 初始思维模式
initial_m = np.array([0.5, 0.5, 0.5])

# 设定改进目标

def simulate_lift(m, goal_m, num_steps):
