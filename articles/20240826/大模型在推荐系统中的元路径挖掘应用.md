## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

#### 4.1.1 用户-物品图表示

设 \( G = (V, E) \) 为用户-物品图，其中 \( V \) 表示节点集合，包括用户节点和物品节点，\( E \) 表示边集合，表示用户和物品之间的交互关系。

#### 4.1.2 元路径表示

设 \( P \) 为用户和物品之间的元路径集合，每个元路径 \( p \) 可以表示为 \( p = (u_1, e_1, u_2, e_2, \ldots, u_n, e_n, v) \)，其中 \( u_1 \) 和 \( v \) 分别表示用户节点和物品节点，\( e_1, e_2, \ldots, e_n \) 表示用户节点和物品节点之间的边。

### 4.2 公式推导过程

#### 4.2.1 路径权重计算

设 \( w(p) \) 为元路径 \( p \) 的权重，可以表示为：

\[ w(p) = \sum_{i=1}^{n} w(e_i) \]

其中，\( w(e_i) \) 为边 \( e_i \) 的权重，可以表示为：

\[ w(e_i) = \frac{1}{|I(e_i)|} \]

其中，\( |I(e_i)| \) 表示边 \( e_i \) 的邻接节点的数量。

#### 4.2.2 元路径选择

设 \( \gamma \) 为元路径选择阈值，选择满足以下条件的元路径：

\[ w(p) \geq \gamma \]

#### 4.2.3 特征表示

设 \( \phi(u) \) 和 \( \phi(v) \) 分别表示用户和物品的特征向量，可以表示为：

\[ \phi(u) = \sum_{p \in P} \phi_p(u) \]
\[ \phi(v) = \sum_{p \in P} \phi_p(v) \]

其中，\( \phi_p(u) \) 和 \( \phi_p(v) \) 分别表示元路径 \( p \) 对用户和物品的贡献。

# 提取所有路径
paths = nx.all_simple_paths(G, source="u1", target="v3")

# 计算路径权重
path_weights = {path: sum(G.edge[path[i][0], path[i][1]]["weight"] for i in range(len(path) - 1)} for path in paths}

# 选择满足阈值的元路径
gamma = 0.5
selected_paths = {path: weight for path, weight in path_weights.items() if weight >= gamma}

# 编码元路径
path_embeddings = {}
for path, weight in selected_paths.items():
    path_embedding = np.mean([G.node[node]["feature"] for node in path], axis=0)
    path_embeddings[path] = path_embedding

# 构建用户-物品特征向量
user_embeddings = {}
for user in G.nodes:
    user_embedding = np.mean([path_embeddings[path] for path in selected_paths if user in path], axis=0)
    user_embeddings[user] = user_embedding

# 物品特征向量
item_embeddings = {}
for item in G.nodes:
    item_embedding = np.mean([path_embeddings[path] for path in selected_paths if item in path], axis=0)
    item_embeddings[item] = item_embedding

# 利用神经网络进行编码和表示
user_item_similarity = cosine_similarity(np.array(list(user_embeddings.values())), np.array(list(item_embeddings.values())))

# 生成推荐结果
recommendations = {}
for user, user_embedding in user_embeddings.items():
    user_embedding = np.reshape(user_embedding, (1, -1))
    similarity_scores = user_item_similarity.dot(user_embedding.T)
    recommended_items = np.argsort(-similarity_scores)[0][:10]
    recommendations[user] = [item_embeddings[item] for item in recommended_items]

# 输出推荐结果
for user, items in recommendations.items():
