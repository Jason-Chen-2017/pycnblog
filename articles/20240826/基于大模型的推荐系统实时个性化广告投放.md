### 4.1 数学模型构建

用户行为特征向量表示为 $X \in \mathbb{R}^{n \times m}$，其中 $n$ 为用户数量，$m$ 为特征维度。深度学习模型输出用户兴趣向量 $Y \in \mathbb{R}^{n \times k}$，其中 $k$ 为兴趣维度。

### 4.2 公式推导过程

使用Transformer模型进行用户兴趣建模，其损失函数为：

$$
L = \frac{1}{n} \sum_{i=1}^{n} \log P(Y_i | X_i)
$$

其中，$P(Y_i | X_i)$ 为用户 $i$ 的兴趣向量 $Y_i$ 在特征向量 $X_i$ 下的概率分布。

import torch
import torch.nn as nn
from transformers import TransformerModel

# 定义模型
class InterestModel(nn.Module):
    def __init__(self, n_input, n_hidden, n_output):
        super(InterestModel, self).__init__()
        self.transformer = TransformerModel(n_input, n_hidden, n_output)

    def forward(self, x):
        return self.transformer(x)

# 加载模型
model = InterestModel(5, 10, 3)
model.load_state_dict(torch.load('interest_model.pth'))

# 输入特征向量
x = torch.tensor([[1, 0, 1, 0, 0], [0, 1, 0, 1, 0], ...])

# 预测用户兴趣向量
y_pred = model(x)
