### 5.2 源代码详细实现
```python
import requests
from bs4 import BeautifulSoup

def get_goods_list(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.content, 'html.parser')
    goods_list = soup.find_all('div', class_='item J_MouserOnverReq')
    return goods_list

def get_goods_info(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.content, 'html.parser')
    price = soup.find('div', class_='price g_price g_price-highlight').text
    return price

if __name__ == '__main__':
    url = 'https://s.taobao.com/search?q=冰箱'
    goods_list = get_goods_list(url)
    for goods in goods_list:
        detail_url = goods.find('a')['href']
        price = get_goods_info(detail_url)
        print(price)
```

### 5.3 代码解读与分析
代码分为两部分：获取商品列表和获取商品详情。首先，使用`requests`库发送HTTP请求，获取淘宝商品列表页的HTML内容；然后，使用`BeautifulSoup`库解析HTML内容，提取商品详情链接；最后，再次发送HTTP请求，获取商品详情页的HTML内容，提取商品价格信息。

### 5.4 运行结果展示
运行代码后，将输出淘宝家电商品的实时价格信息。
