# 1.简介
         
12年前，微软推出了.NET 框架，它是一套现代化的开发框架，致力于解决当时应用程序编程领域的一些问题。它开创了一个全新的开发方式，并帮助计算机科学界的许多领域快速发展起来。然而，微软的历史告诉我们，它的成功也带来了一些问题，如性能不足、文档不全面、API 不一致等。为了解决这些问题，很多技术巨头纷纷投入大量资源在 Go 语言上，Go 是一门由 Google 团队设计开发的静态强类型、编译型、可移植的编程语言。其语法简单易懂、运行速度快、内存安全等特点，深受开发者的喜爱。因此，本书将从 Go 的基本语法开始，深入剖析 Go 语言的特性、数据结构、并发机制、Web 编程、分布式系统等方面，让读者了解 Go 语言独有的优势和功能，更好地掌握 Go 语言的应用技巧。  

          2019 年底，阿里巴巴集团宣布开源 Go 语言。该项目源于阿里巴巴内部系统的基础服务平台架构需求，并基于 Google 开源的 Go 语言打造，目标是在云计算时代实现基础服务平台的高性能、弹性伸缩和成本低廉。本书是作者对 Go 语言深度剖析及实践经验的总结，也是 Go 在开源社区的风采之作。

          本书适合没有任何 Go 语言基础的人学习 Go 语言，也可以作为 Go 语言进阶学习的导引。本书共收录 36 个核心知识点，涵盖 Go 语言的基本语法、数据结构、并发模型、Web 编程、分布式系统等各个方面。每章节都配有练习题目，可以帮助读者掌握相应的知识技能。通过本书，读者可以加深对 Go 语言的理解，提升编程能力，最终掌握 Go 语言进行 Web 服务开发和分布式系统架构设计。
