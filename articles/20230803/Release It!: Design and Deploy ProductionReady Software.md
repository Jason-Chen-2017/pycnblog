
作者：禅与计算机程序设计艺术                    

# 1.简介
         
 随着软件开发越来越复杂、产品规模越来越大、用户需求越来越多，工程师开发出来的软件也越来越庞大、功能越来越丰富、性能越来越好。作为一个“专业”的软件开发人员，你如何在市场上赢得尊重和青睐呢？本书将教会你如何设计、构建、测试并部署生产就绪的软件。通过真实案例学习的方式，作者基于实际场景和建议，提供全方位、深入的软件设计和开发指导，帮助读者更好的理解软件设计过程及其价值所在，更加有效地运用所学到的知识解决日常工作中遇到的问题。本书除了提供全面的专业知识教育外，还将通过案例的讲解和实践相结合的方法，启迪读者对软件设计的兴趣、能力、方法论和工具的掌握，进而达成更高质量的软件开发能力。

# 2.为什么要写本书？
如果说之前的“你不应该被软件开发领域所左右”的话，那么在本书发布后，你会发现这里面有太多的关于软件设计、架构以及开发过程中的实际经验教训。很多企业也遇到了一些“管理层”和“CEO”的质疑，他们担心自己的产品设计和开发不会满足市场的需求。但是这很难通过简单的“有没有代码”，“能不能快速交付”等简单的问题来回答。只有真正的了解软件开发的本质，才能真正找到创造商业价值的可能。

所以，本书的目标就是向你展示软件开发过程中的实际经验、方法论和技巧，让你能够清晰、准确地看待软件设计和开发的各个阶段，真正把握开发的关键点，提升你的职业道德，减少软件开发中的“万恶之源”。最后，也希望通过这些经验教训，帮助更多的人成长为“更优秀的软件工程师”。
