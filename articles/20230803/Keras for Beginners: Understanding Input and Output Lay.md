# 1.简介
         
1.1 传统机器学习任务需要针对输入数据进行特征提取、选择模型结构、训练模型并做预测，而深度学习方法则不需要手动设计特征提取网络或者神经网络层次结构。
         1.2 深度学习可以直接从原始输入数据中提取丰富且有效的信息，因此可以对一些传统机器学习任务无需过多特征工程工作，甚至可以用深度学习解决一些很难处理的问题。
         1.3 同时，深度学习也可以用来解决序列问题（如文本、音频、视频等），可以提取到长期和连续的时间特征信息，这也是深度学习和传统机器学习的一个显著区别。
         1.4 本系列文章将会带领你快速上手TensorFlow/Keras，通过理解输入输出层在卷积神经网络、循环神经网络及组合网络中的作用，帮助你更加充分地利用深度学习技术。

         # 2.基本概念术语说明  
         2.1 TensorFlow/Keras是一款开源机器学习工具包，是一个高级的、灵活的用于构建、训练和部署深度学习模型的框架。它可以运行在不同类型的设备上，包括笔记本电脑、台式机、服务器或移动设备，具有灵活的计算图机制，可轻松实现分布式训练功能。
         2.2 机器学习（ML）是人工智能的一个分支，其目的是让计算机能够“学习”从数据中获取的知识，并基于这些知识做出更好的决策或预测。最早由罗纳德·费洛威（Ronald Fisher）提出的，后被称作统计学习、机器学习或模式识别。
         2.3 卷积神经网络（Convolutional Neural Network，CNN）是一种基于图像处理的深度学习模型，由多个卷积层和池化层构成。卷积层是提取特征的，池化层是减少参数量的。
         2.4 循环神经网络（Recurrent Neural Network，RNN）是一种深度学习模型，可以捕获时间序列的依赖关系。
         2.5 组合网络（Hybrid Network）是结合了CNN和RNN的深度学习模型，可以捕获序列和图片的依赖关系。
         2.6 在机器学习中，数据的形式通常是向量或者矩阵，我们把这些数字数据作为输入向量送入神经网络进行学习。在CNN和RNN中，输入的数据是多维数组，我们需要对每个像素或者时序上的值进行特征提取，因此要先转换成向量或者矩阵。
         2.7 在深度学习过程中，有两类层次，第一类是输入层，第二类是隐藏层。输入层接收原始数据，首先进行数据的预处理，然后进入隐藏层。隐藏层通过一系列计算得到最后结果。在深度学习过程中，还有一类特殊层——输出层，用于输出预测结果。
         2.8 在CNN中，卷积核的大小一般是奇数，因为原始数据中的每一个像素都要与周围的相邻像素做差值，所以卷积核的宽度和高度都应该为奇数。对于池化层，其作用是降低计算复杂度和提取局部特征。
         2.9 在RNN中，RNN的本质是对序列进行建模，其中包含状态变量，可以捕获序列中的时间和空间上的依赖关系。在LSTM和GRU中，为了解决梯度消失和梯度爆炸问题，引入了门控机制。
         2.10 在组合网络中，通过堆叠CNN和RNN，可以同时获得序列和图像等多种数据之间的特征表示。
         2.11 数据集、数据增强、正则化、优化器、损失函数和评估指标是构建深度学习模型的重要组件。
         2.12 模型保存与加载是深度学习模型的关键环节。
         2.13 GPU支持是深度学习训练过程的关键瓶颈之一。
         2.14 TensorBoard是深度学习训练过程中的可视化工具，方便查看训练过程中的指标变化。
         2.15 命令行接口是深度学习模型训练的标准方式，方便部署和调参。
         2.16 本系列文章将用Python语言实现一些示例代码，演示如何构建深度学习模型。  
         2.17 本系列文章的前四章主要介绍输入层、卷积层、池化层、全连接层在CNN中的作用，包括如何设置过滤器大小、步长、填充、激活函数等。第五章将介绍循环神经网络（RNN）的概念和基本原理，包括如何设计RNN单元、序列长度、隐藏状态、循环层的数量等。第六章将介绍组合网络的概念，并用TensorFlow实现了一个分类模型，展示了如何用Numpy库进行数据准备、构建模型、训练模型、评估模型、保存模型。
