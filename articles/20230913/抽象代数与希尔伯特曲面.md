# 1.简介
  

## 抽象代数
抽象代数(Abstract Algebra)是指研究通过代数结构来研究几何图形、函数等对象的集合论的方法。抽象代数以代数结构的形式提供了一种形式化的语言来研究几何图形、函数等对象之间的关系。抽象代数通过定义和研究代数系统的基础理论、构造模型、求解计算问题以及分析其性质和应用等方面对几何图形、函数等对象进行了全面的探索。由于抽象代数的内涵更加抽象，因此，它不仅能用于数值计算领域还可以直接用于物理科学、生物学、工程学、心理学、哲学等其他领域。抽象代数的理论工具既包括群论、环论、模形式论、体论等，也包括范畴论、微积分学及图论等相关学科。

## 希尔伯特曲面
希尔伯特曲面是一个从二维空间到三维空间的双射映射，这个映射用两个变量$x$和$y$来表示，它的表达式为
$$z=f(x,y),\quad (x,y)\in \mathbb{R}^2$$
其中$f: \mathbb{R}^2\to \mathbb{R}$是定义在$\mathbb{R}^2$上的实值函数，称为曲面$S$的函数。

例如，在平面上有一个抛物线$y = x^2$, 对应的希尔伯特曲面可以用$z = xy$来表示。但是这个曲面没有办法完美地逼近任意曲面，所以才需要研究更复杂的曲面，如椭圆曲面或超椭球面。

## 概念术语
1.点(Point): 在直角坐标系$(x,y)$中，点$(a,b)$是具有坐标$a+bi$的点。对于一般的曲面$S$，点$P=(x_p, y_p, z_p)$代表曲面$S$的一个位置，其参数点坐标为$x_p + iy_p + iz_p = k$。

2.边(Edge): 曲面$S$的一条边是由两点$P_0$和$P_1$确定的一条曲线，并且满足：
   - $\overline{OP_0}\cap S = P_1\cap S$
   - $OP_0\cap OP_1 = O$
   - 如果$C$是过$P_0$和$P_1$的圆，则$CP_0\cap C$和$CP_1\cap C$都等于空集。

   如果曲面$S$存在无穷多个边，则称这些边构成曲面的一个边界。

3.边界(Boundary): 当曲面$S$的每条边都是曲面的边界时，曲面$S$的边界称为曲面的边界。

4.邻接点(Neighboring point): 对点$P=(x,y,z)$而言，$N(P)$表示$P$的邻接点，即所有处于曲面$S$与$P$连通的点，记作$NN(P)$。

5.基(Frame of Reference): 基$(E_i,\varphi_i)$的形式为：$E_i\cdot \varphi_i = e_i$，其中$\varphi_i$是第$i$个基矢量，$e_i$是标准基矢量，$E_i$是第$i$个参考直角坐标系的坐标轴。

6.表面(Surface): 位于某个基$(E_i,\varphi_i)$下的曲面$S$的表面是指沿着某一方向$n$，经过每个边界$L$的交点得到的曲面$T$。

7.切线(Tangent line): 在曲面$S$上的一点$P$的切线$T$是沿着$r$投影切向量的延长线，且垂直于过$P$的直线。

8.曲率(Curvature): 在曲面$S$上的一点$P$的曲率是指切线$T$的斜率，用符号$\kappa(P)$表示。

9.弧段(Arc segment): 曲面$S$的一条弧段$A$是由曲线$c$和满足$c(t)=P$的区间$I$确定，即$A$由曲线$c$在$I$上所落的点构成。

10.扇面(Cone): 位于某个基$(E_i,\varphi_i)$下的曲面$S$的扇面$C$是由弦曲面$C'$的两侧所形成的曲面，其中$C'$是由曲线$l$和$r$所形成的曲面。

11.切面(Cut-plane): 位于某个基$(E_i,\varphi_i)$下的曲面$S$的切面$C$是指沿着某个基矢量$\varphi_j$所确定的平面$F$，使得曲面$S$在平面$F$的外部，且只有曲面$S$的边界$L$在平面$F$上。
