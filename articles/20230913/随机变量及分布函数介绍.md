# 1.简介
  

## 1.1 为什么要学习随机变量及分布函数？
在现代数据科学中，许多任务需要依赖于概率论、统计学、数理统计等学科。为了解决实际问题，需要对数据进行建模、分析、处理等，那么如何描述这些数据，并通过概率论进行抽象和推断呢？就需要了解随机变量及其对应的分布函数（probability distribution）。

## 1.2 随机变量及分布函数的定义
### 1.2.1 概念阐述
随机变量（random variable）是一个变量，它可以取不同的值，但由于其本身没有定量表示方法，所以只能用概率分布（probability distribution）来描述该变量的取值情况。概率分布表示的是事件发生的频率，也称为分布函数（distribution function），把所有可能的结果按照其发生的概率给出一个概率分布。例如，抛一次硬币有两种可能结果“正面”或“反面”，因此这个变量的分布就是“正面”和“反面”各占50%的概率。

### 1.2.2 分布函数
分布函数的定义是“对于所有可能的结果，分别对应一个实数值，表示该结果发生的概率”。在概率论中，由某个变量X产生另一个变量Y，当且仅当X的分布函数存在时才成立。如果分布函数存在，则称Y服从X的分布。如果一个变量只有一个取值，则其分布就唯一确定了；但是，一般地，变量通常具有多个取值，因此其分布往往会多种多样。常见的分布函数包括以下几类：
- 离散型分布函数
    - 指示型分布函数
        - Bernoulli分布函数（伯努利分布）
        - Binomial分布函数（二项分布）
        - Geometric分布函数（几何分布）
        - Negative binomial分布函数（负二项分布）
    - 连续型分布函数
        - Uniform分布函数（均匀分布）
        - Normal/Gaussian分布函数（正态分布）
        - Exponential/Laplace分布函数（指数分布）
        - Student's t分布函数（学生t分布）
        - Cauchy分布函数（柯西分布）
        - Beta分布函数（贝塔分布）
- 分布函数的性质
    - 求和性
        如果X和Y都是独立随机变量，即两个事件互不影响，则X+Y的分布函数等于X的分布函数和Y的分布函数之和。例如，抛两枚硬币正面朝上的次数服从Bernoulli分布，每个硬币正面的概率为p，则两枚硬币正面朝上的总次数的分布函数f(x) = f_1(x) + f_2(x)，其中f_1(x)=p^x * (1-p)^(1-x), f_2(x)=(1-p)^x * p^(1-x)。
    - 可加性
        X和Y的分布函数可以相加或相乘得到一个新分布函数，这取决于它们之间是否相关。如果两个变量间没有关联，则它们的分布可以相加，即f(x+y) = f(x) + f(y)。否则，则它们的分布可以相乘，即f(xy) = ∫ f_1(u)f_2(x-u)du。
    - 单调递增
        对任意一点a，如果Y>a，则X>a的概率至少是Y>a的概率，也就是说，分布函数是单调递增的。此外，积分从0到无穷大时，分布函数的值一定大于或等于0。
    - 归一化系数
        如果积分值恒为1，则分布函数被称为标准化的或者标准化常数的。对于连续型随机变量，标准化常数的值为1/√π，而对离散型随机变量，标准化常数的值就是全部概率的乘积。
    
### 1.2.3 模拟与近似计算
根据分布函数，可以生成一组符合指定分布的数据集。这样就可以应用各种数理统计学的知识来做一些更复杂的分析。另外，也可以利用计算机来进行近似计算，比如求得概率密度函数（Probability Density Function, PDF）和累积分布函数（Cumulative Distribution Function, CDF）。这些近似计算的方法帮助我们估计某些统计量的值，从而对分布函数的形状和位置做出更好的理解。

### 1.2.4 总结
随机变量及其对应的分布函数是概率论的核心概念，也是统计学的重要组成部分。研究随机变量及其分布函数有助于我们理解现实世界中的数据，做出正确的假设，并基于这些假设做出预测。
