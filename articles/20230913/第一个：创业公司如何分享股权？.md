# 1.简介
  

大家好！我是刘烨，是一名创业公司CTO。今天，我们来聊一下股权、增持、减持、卖出等股权交易相关的一些常识和实际操作。希望通过这个系列文章能够对创业公司的股权管理有所帮助。
什么是股权？
股权是一个公司或企业拥有的部分或者全部的控制权。作为普通投资者，你在股权上并不一定拥有绝对的所有权。股权的获取方式通常包括四种：
- 发行新股：公司创立时，所有者首先获得的是股份。由于公司的发展壮大，创始人会把新产生的股份分给员工、合作伙伴甚至外部投资者。这种方式称为“转让所有股权”。
- 分红：公司运行过程中，员工的薪水按年度分配，但是并不是每位员工都可以分到股份。公司为了吸引更多的资金进入，给予员工一些奖励。这样，员工对公司的支持力度就增加了。公司从员工手中分得的股份称为“分红股”。这种方式也叫做“分配股权”。
- 向上市场推广：有些公司已经盈利，可以申请将其股票上市。此时，公司的股东可能会获得一些“发行价值”，但实际上并没有完全拥有公司的所有权。上市后的股票上市，也称为“上市股”。
- 公司重组/收购/转让：有时候，公司可能遇到经营困难，需要进行重组、收购或者其他形式的转让。这时，公司股东获得的权益将受到限制。
那么，对于创业公司来说，股权分为两大类——主要股权（common stock）和附属权（preferred shares）。其中，主要股权是由公司直接拥有的，占据绝对份额。附属权则是公司直接拥有的，但是又不占据公司股本总数的权益，可以被用来补充主营业务的现金流。
另，如果企业拥有不同于公众认知的“非公开发行股份”，例如房地产、黄金等，这些股份仍然属于公司，只是外界对它们并无控制权。
接下来，我将结合我的经验，通过几个具体案例，阐述股权交易的一些基本概念、法律法规、操作技巧和注意事项。希望能够帮助创业公司实现更好的长期稳定性和投资回报。
# 2.概念术语
## 2.1 “公开”与“私募”
对于企业来说，可以通过两种方式获得股权，即“公开”方式和“私募”方式。公开方式就是向公众公开挖矿所得的硬币，企业获得的部分股权是公众拥有的；而私募方式则需要通过认证的方式购买股权。公开挖矿其实也是一种私募，只不过是在股权证书上注明自己是“公开”的。这里需要特别注意的是，两种方式并不冲突，私募仍然可以获得公众拥有的股权。所以，两种方式的股权应该一起计算。
## 2.2 增持、减持与举例
增持：公司的股东在交易之前拥有了公司的全部股权，之后还继续拥有全部的股权，并以其现金支付持有该股权的现金股利，称为“增持”。如果公司实际上无须增加任何股东权益，只是出售掉部分股东的股份，也可以视为增持。
减持：公司的股东在交易前只拥有部分的股份，之后又持有其他股份。公司会优先从其他股东那里购买股份，来弥补自己的损失。除此之外，减持还有一种情形，即有人要离职，这时可以把部分股份卖掉。
举例：
假设A、B两家公司要增持某个同行业的公司的股权。
- A和B各持有50%的股份。
- A、B成立联合公司，合计持有100%的股份。
- B将其持有的50%股份转让给C。
- C占据了A、B的双方的75%股份。
- B卖掉剩余的25%股份，买入D公司的50%股份。
## 2.3 预留股权及其限制
“预留股权”是指公司发行股票后，给予股东的一些权益，但这些股权不能发放给任何第三方，只能用于配合并发行新的股票。比如，IBM公司的创始人之一布兰登·杜克说过，他曾经希望向员工、顾客等发放预留股权，但却没有成功。“预留股权”一词最初源自于英国议会法院对IBM案件的判决。该案件认为，IBM在向公众募集资金过程中，没有给予员工或顾客应有的预留权利，致使其欠费亏损。因此，“预留股权”一词被取消。
根据美国证监会定义，发行者可以允许特定类型的股东获得预留股权，例如“优先股权”或“共同基金”等。
不过，由于“预留股权”具有“不可转让”的性质，其购买、持有和交换过程均受到限制。
## 2.4 “可转让公司股份”
“可转让公司股份”是指在新股发行时，公司把剩余的股份全部转让给股东。这类股权有很强的收益率优势，一般企业发行新股时都会选择这种方式。可转让股权的数量受限，其总数和新股总数成正比。
“可转让股份”一词最早出现在1975年美国联邦储备委员会关于IBM的监管规则中。
截止目前，全球普遍认可“可转让股份”这一概念，并且美国证监会、中国证监会和日本证监会都纷纷采用它。
## 2.5 欠发行股权
欠发行股权是指企业准备发行股票但没有足够的资金，无法完成初始筹资的状态。一般而言，一家企业因资金短缺，或无法满足管理层和员工要求，导致无法顺利完成股票发行。当这家企业的股权欠发达时，股东可能选择向其他机构或个人借款发行股票。
不过，“欠发行股权”一词也存在一些误区。一个原因是，许多债券公司也能在债券发行之后先发行股票，即便债务规模较小也不影响股东拥有的权益。另外，有的创业公司在成立初期，可能会因为资金短缺而停止发行股票。这种情况下，只能靠员工带动或以借贷方式筹集资金。
所以，关于“欠发行股权”一词的争论也比较多。不同的人提出了不同的观点。我们需要综合考虑各方利益和情况，再决定是否购买。
## 2.6 可转换债券
可转换债券(convertible bond)是一种债券，当其价格超过发行时的基准利率时，投资者可以将其抵押，或购买公司的股票或债券，以获得债券公司的融资。这种债券不需要担保，其价格的变化往往与股票市场、债券市场上的股票价格变化同步。
不过，可转换债券也存在一些问题。例如，虽然它可以在市场上自由交易，但会产生持有成本，而且有可能被转为次级债务或企业信用等。另外，大量的可转换债券同时发行，可能会削弱企业的银行杠杆效应，影响企业的偿债能力。
# 3.案例解析
## 3.1 早期阶段的增持
假如你所在的创业公司刚刚成立了一两个月，正在寻找投资人的眼光。
这时，你有很多选择：一是找一些战略合作伙伴，跟他们签署协议，合伙创业；二是选择跟风，随着时间的推移，逐渐积累起一定的财富和资源；三是自己主导方向，创建自己的产品、服务和团队。
如果你有钱的话，建议优先选择第一条路，它既可以避免风险，又能提供商业模式的灵活性。当然，最好的选择还是自己主导，让你的创业成为生意的一部分。
你为公司筹建了一支高级工程师队伍，负责项目的技术研发工作。你和他们一起讨论，决定召集外部的投资者来跟你合作。
这时，你想知道，合作的条件是什么？是不是要获得全额股权，还包括固定期限的股息？以及什么时候赎回？在谁的努力下才能实现这套机制？
所以，你开了一个小群，询问投资者们的意见。很多人很好奇，包括你的团队成员。有的说，要全额股权，另一些则担心股权可能转让；有的说，可能不会设固定期限的股息，其他人也担心担心股东们的怨气；还有的人觉得还有些担忧，担心资金链断裂。
不过，你的团队有自己的判断和判断能力，他们最终达成一致意见，决定增持你所持有的股权。这种增持的方法是，公司持有原有10%股权，按照一定的增持比例，买进60%新的股权。
你和股东们达成了协议，你可以获得60%的股份，并按固定期限分红。
由于公司的估值已经超过了一定的期望值，股东们接受了你的设想，也支持你在增持中取得成功。然而，你也应该看到，公司的股权结构有点混乱。虽然公司股权在增持时已经完全集中在少数股东手中，但实际上还有很多股权没有得到分配，这就造成了效率低下和投资者的不满。
## 3.2 中期阶段的增持
公司的盈利状况开始逐渐好转。由于市场需求的增长，股东们开始欢迎公司回购其资产。这时，你面临着回撤的风险。你知道，增持的增持比例可能要调整，或者是减持，这取决于你们的预期回报是否会超过你的实际回报。
不过，你已经找到了投资人，可以坚定信心了。他们也看好你们的股权回报，也建议你保持现有的股权结构，不增加新的股权，让股东们自主选择回购哪些资产。
## 3.3 终末阶段的增持
公司的估值达到了目标，已经可以实现盈利。但是，随着时间的推移，竞争对手开始入侵，这时你会发现自己的股权价值开始消失殆尽。这时，你可能需要进行变革了。
你想把股权结构重新整理一下。你决定取消现有的增持计划，让公司重新调整它的股权结构。这样一来，公司的所有者可以得到最大的回报。
你对公司进行了调查，了解到很多创业公司的情况。你找到了乔布斯的《苹果事件》，里面有一条“抓住这个机会就不要错过它”的口号。不过，这位教父真的没错。
乔布斯认为，苹果手机的市场远远超过他的预料，一夜之间他把所有人的注意力都放在这个手机上。因此，他马上把所有的钱都花完了，只保留一套运营费用和一辆车，而把股票让给了清华大学的学生。股票到期后，他又回归到了苹果的股权结构中。
但他的想法并不是为了个人的利益。他的父母是位商人，他很乐意利用自己的年轻身份，让孩子们享受到商业上的快感。结果，他儿子的名声很快爆红，一夜之间就成为最具影响力的科技领袖。
在乔布斯看来，股权结构的改革意味着公司重新定位，公司股权结构可以产生更好的投资回报。因此，他选择去抢班夺宝，帮助投资人争取一定的收益。
不过，也有投资人不同意他的计划。有的投资人担心，如果失去了他们投资的股权，公司会面临破产、没有财源或者无法正常运营等严重的问题。他们担心股权的价值和安全性。
另一些人觉得，由于他们只是个学生，也无法养活公司，也没有相应的担保，因此，他们更愿意保留老旧的股权结构。
这时，你和其他投资人发生冲突。你觉得自己的经历足以证明，改变股权结构比种植树蛋容易得多。不过，你害怕受到反驳。
于是，你选择了掌握火候。你连夜召集你身边的朋友们，一起制定出一套新的股权结构。这样一来，你的公司就可以跑赢这场竞赛。
# 4.操作建议
最后，给出一些具体的操作建议。
## 4.1 筹资途径
股权投资的途径主要有两种：公开筹资和私募筹资。公开筹资的好处是投资资金公开透明，容易获得大量的投资人参与。私募筹资则是通过大众筹集资金的方式，减少了风险和不确定性。
对于创业公司来说，首先要选择自己的生意所在。如果你有资源雄厚，可以尝试做生意，然后扩张。如果处于萌芽阶段，试着做一些小范围的试验。然后，逐渐发展到成熟，并盈利。只有在成熟的阶段才适合做长期的股权投资。
## 4.2 增持与减持的时间
股权的获取方式有两种，一种是全股权买入，另一种是增持、减持。越是年轻的创业公司，股权获取的概率越大，需要在早期准备充分的资金。如果资金充足，建议增持。减持则是指将持有的股权少部分转让给其他股东，来弥补自己的损失。
建议增持时，请遵循以下规则：
- 每次增持应缴纳增持税，增持价格为每股股价的百分之五到十，具体缴纳税率视国家法律规定。
- 增持应长期进行，增持到一定年限后，请考虑将股权转让给第三方。
- 请密切关注公司财务数据，适当调整增持比例。增持后，公司股权结构将发生变化，需时重新评估是否适宜继续增持。
减持则是一个比较复杂的事情。通常情况下，股东会自愿选择减持。所以，也需要注意减持的规则。减持时，请确保减持的权益仅用于弥补企业亏损。其余部分将继续给予股东。
- 降低回购价格，提高杠杆比。
- 使用适当的减持工具，比如电子交易系统、增发股票工具。
- 在现金交割日之前，将股份交付给持有股权的第三方。
## 4.3 托管公司的股权
托管公司的股权是指把公司的股权转让给第三方托管，后者承担公司的义务和风险。托管公司的股权有助于解决公司股权过度流动、高质量的投资机会难以获得、以及合理收益难以实现等问题。
建议根据公司的实际情况选择托管公司。托管公司可以通过稀释股权价值、增加股东对公司的认可度，来提高公司的股权价值。同时，托管公司还可以为公司的偿债承担一定的义务。
## 4.4 清算协议
创业公司需要履行有关公司股权的义务，包括收取费用、归还债务、分配股息、缴纳增持税等。为了做到这一点，公司建立了清算协议。
清算协议的重要性不亚于诚信，尤其是在创业公司中。清算协议是一种协议，其中列出了所有股东的权益，包括实收资本、股权激励、债务、库存股本、预付款项、发行新股款、特许权等。通过清算协议，公司可以追踪和维持公司的全体股东权益。清算协议还可以确保公司的所有者和管理层获得足够的信息，以便决定下一步的行动。
清算协议还有一个重要作用，它为投资人提供了一种框架，使他们能够更加客观地判断公司的价值。投资人可以在清算协议中找到很多信息，包括股东的权益、交易价格、有关利润和损失、未来的发展方向、管理层的信息、内部管理制度等。投资人可以据此做出最佳判断。
