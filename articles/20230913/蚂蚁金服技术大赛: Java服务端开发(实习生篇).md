# 1.简介
  

## 大赛介绍
蚂蚁金服技术大赛旨在鼓励具有创新能力、热爱技术、对开源充满兴趣、追求卓越的学生或同学，通过参加一年一度的编程竞赛和技术分享活动，共同提升技术水平、找寻职场人生的同学们，同时帮助企业在创新时代更好的开拓市场，释放内需动力。

1. Java基础训练——包括数据结构、算法、计算机网络、数据库、操作系统、编译原理等课程学习；

2. Spring Cloud微服务架构实战——结合实际案例和技能考核，掌握微服务架构、Spring Boot框架、Docker容器技术、Kubernetes云平台等知识；

3. 服务端性能优化实践——通过优化手段分析和解决高并发场景下的问题，提升应用整体的处理性能。

## 比赛主题
Java服务端开发（实习生篇）主要包括Spring Cloud微服务架构实战、服务端性能优化实践两部分。
### Spring Cloud微服务架构实战
实现基于Spring Cloud微服务架构框架，构建完整的金融服务平台，包括交易系统、资产管理系统、报表系统等。要求具备扎实的Java基础、扎实的Spring/SpringBoot基础、精通微服务架构设计、微服务治理、高可用部署、容器化等技能。

本部分最关注的是学生/应届生要学习Spring Cloud框架的整体知识和开发过程中的坑点，同时学生/应届生要具备独立搭建微服务架构的能力，能够对比自己设计的架构方案和现有的流行技术进行分析和选择。同时，也需要学生/应届生持续跟踪市场需求，多听取企业技术分享，对于行业最新技术发展趋势做好科研准备。

### 服务端性能优化实践
在实践环节中，要求学生/应届生根据业务功能模块，首先制定性能测试计划，然后采用压力测试工具对各个模块进行压力测试，识别出系统瓶颈点并进行性能优化。通过对各个模块的性能指标进行评估，进一步得出改善后的系统架构。期待学生/应届生通过自身的不懈努力，锻炼自己的分析、问题定位和解决能力，成就自己在性能领域的一帆风顺！

## 2.2基本概念术语说明
## Java编程语言
Java 是由Sun Microsystems公司于1995年推出的Java SE标准版，是一个面向对象编程语言，拥有简单易用、安全性能优秀、支持分布式开发、可移植性强等特点。Java虚拟机(JVM)是Java运行环境，它是在用户电脑上安装的Java指令集解释器，用来执行编译后的Java字节码。Java具有自动内存管理、强语言类型检查和支持多线程的特点。

## JVM
Java虚拟机（Java Virtual Machine，JVM）是运行Java字节码的虚拟机。JVM提供了一个抽象层，使得Java程序可以被不同操作系统上的解释器或编译器执行。每一个编译过的Java源文件都有一个对应的class文件。JVM把class文件加载到内存中，运行时才会转换为机器指令执行。因此，Java的运行速度相较其他编程语言来说要快很多。

## Spring Cloud框架
Spring Cloud是一个微服务架构开发框架，它是为了构建微服务架构而创建的。Spring Cloud包含众多子项目，如配置中心Eureka、服务容错保护Hystrix、负载均衡Ribbon、API网关Zuul、消息总线Spring Cloud Bus、链路追踪Sleuth、分布式事务Seata等。这些组件能够快速构建分布式系统，降低了各个服务之间的耦合度，提升了系统的稳定性。Spring Cloud还提供了统一的配置管理、服务监控、服务调用等功能，极大的方便了微服务架构的部署与运维工作。
