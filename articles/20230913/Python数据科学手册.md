# 2.基本概念术语
## 2.1 Python语言概述
Python是一个高级编程语言，被誉为“瑞士军刀”。它具有简单易懂、免费、跨平台特性，能够轻松完成各种数据处理任务。它是动态类型语言，支持面向对象、函数式编程、命令式编程等多种编程风格。Python支持丰富的数据结构，包括列表、字典、集合、元组等；支持文件、网络、数据库、文本、图像等输入输出操作；支持正则表达式、XML、JSON、Pickle序列化与反序列化等；还有一些第三方库可以支持数学计算、图形绘制、音视频处理、Web开发等。
## 2.2 数据处理相关术语
数据处理是指从原始数据到最终结果的整个过程。数据处理过程中通常涉及多个阶段，如数据获取、清洗、转换、存储、分析和可视化等。下面列举一些重要的术语：
- 数据：原始信息，用于训练或预测模型的数据。可以是网页上的搜索日志、服务器日志、用户画像数据、运动监测数据、传感器读数、海量的图像、文本等。
- 数据源：数据所在的位置，可以是本地文件、远程服务器、数据库、消息队列、MQTT等。
- 数据加工：对原始数据进行清理、提取、过滤、转换等操作。如过滤掉无效数据、删除重复数据、保留关键字段、规范化日期格式、转换字段类型等。
- 特征工程：通过特征选择、特征转换、特征降维等方式对数据进行变换，提取重要的特征。
- 模型训练：采用机器学习算法对数据进行建模，生成预测模型。
- 模型评估：对训练好的模型进行验证，测试其性能。
- 模型推断：对新数据进行预测，基于已有的模型生成结果。
- 可视化：对数据进行直观呈现，帮助更好地理解和分析数据。
## 2.3 Pandas概述
Pandas（Python Data Analysis Library）是一个开源的Python数据分析工具箱。它提供了高性能、数据结构化、与numpy类似的接口。Pandas支持高级数据处理功能，包括缺失值处理、排序、时间序列操作、合并、分组、聚合等。Pandas接口友好，可以与其他第三方库结合使用，比如matplotlib、seaborn、plotly等可视化工具库。
