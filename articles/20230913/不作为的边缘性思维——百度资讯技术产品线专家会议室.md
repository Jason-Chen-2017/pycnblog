# 1.简介
  

互联网信息的快速爆炸已经让人们生活的方方面面都成为信息消费时代的一部分。随着互联网技术的不断进步，人们越来越多地将注意力集中在这个快速增长的信息海洋之上，而忽视了其中的一些负面影响。一手掌握互联网平台、为用户提供服务、拥有品牌、广告主或市场精英等角色的互联网企业，却可能被看作是一个“信息风口”，快速发展壮大的互联网企业都需要解决与此相关的问题，如何在信息不对称的情况下继续获得成功，成为行业“新锐”就是其中重要的课题之一。近年来，由于互联网信息的快速生成和快速消化，越来越多的互联网公司发现自己正在被下游客户所裹挟，而下游客户也正逐渐成为互联网公司的下游。因此，一个有效的方法论可以帮助互联网公司识别出自己处于不作为的边缘性思维，并通过自我剖析和业务分析找寻突破口，逐渐走向成功。

百度资讯是中国领先的互联网资讯服务平台，目前国内外的网站数量达到8亿以上，日均活跃用户数超过1亿。根据百度资讯最新报告显示，截至2021年7月底，百度资讯日均独立访客数已达到19.9万人次；2021年度流量总计超500亿，累计阅读量达到7.75亿次；平均每天浏览量超过1亿次，访问量已成为全球第一。除此之外，百度资讯还积极参与国际领域的新闻跟踪报道，推动国内外媒体共同关注百度资讯，并形成全民参与互联网民主建设的良好氛围。

本专家会议室邀请了百度资讯技术专家王辉博士为我们做关于“不作为的边缘性思维”主题的分享。王老师的研究方向包括信息网络、网络舆情、新闻事件检测、公众舆论监控、人工智能技术及应用等。王老师的研究工作十分有前瞻性，她的研究具有系统性、指导意义，可提供宝贵的参考价值。

# 2.什么是不作为的边缘性思维？
不作为即缺乏决策能力、操控权力、冲动性、猜疑心和任性等特征。强烈的不作为特征往往伴随着组织和个人崇拜等心态。比如，“没有吃饭就上班”、“骂他没用”、“长期霸占经济优势”、“不给钱就是不做事”等等。这些不作为的思维方式会造成人们沉浸在不知所措当中，且难以做出正确判断。正如著名社会学家费尔巴哈所说：“人的不作为导致错误行为。”这种偏执的心态既使得个人无法主动参与决策，也影响到了组织和团队的决策过程。

那么，如何才能打破不作为的思维模式呢？一种方法是采用“逆反心理”，即相信自己无能为力，不去推诿责任，而是基于自己的“经验教训”和“历史经验”进行不断修正，直到明白真相、避免错误出现。另外，也可以通过“社会契约理论”等人类学原理，来改变自我对组织的依赖和依赖关系。社会契约理论认为，一个组织所建立的社会契约，即是一种制度结构，它由一系列的契约规范组成，它们共同构成了一个完整的社会秩序。一般来说，一个组织的社会契约主要包含四要素：规则、义务、权利和义务之间的平衡关系。可以通过自我剖析、深度思考、创新设计等方式，塑造适合自己的社会契约。但是，在企业中，如何摒弃不作为的边缘性思维仍然是一个迫切需要解决的问题。

# 3.百度资讯技术产品线的不作为的边缘性思维
百度资讯是一个网络资讯服务平台，拥有超过8亿的独立访客、日均独立访客数超过1亿、流量总计超500亿、累计阅读量达到7.75亿次、平均每天浏览量超过1亿次，这是百度资讯成功的一个关键因素。同时，百度资讯还积极参与国际领域的新闻跟踪报道，推动国内外媒体共同关注百度资讯，并形成全民参与互联网民主建设的良好氛围，这一点也是它的成功的一个原因。

然而，百度资讯技术产品线存在着不作为的边缘性思维。百度资讯的技术架构和系统架构大多采用开源框架，因此，公司内部员工有能力进行系统架构的改进和升级，但由于业务部门成员的认识水平低、能力有限，很少会有相应的计划、筹备和支持。这就使得百度资讯的技术架构一直停留在过去，随着信息资料的快速增长，新闻的反馈速度加快，产品功能的迭代变得艰难，响应不过来。因此，百度资讯技术产品线的不作为的边缘性思维越发突出。

此外，百度资讯技术产品线还存在着其它不足。百度资讯公司的自我修养和个人品质保障意识薄弱，这就导致管理层习惯于依赖技术人员的个人意志，对待技术人员的方式不是开放、包容和善于协商的，往往会拒绝采纳他们的建议，甚至直接找领导谈话要求他们退出团队。这也促使技术人员去充满激情地投入时间研究技术，然而，由于技术人员对自己的定位和贡献认识不清晰，缺乏培训和教育资源，最终导致大量的人才异化、技术退化、产能过剩等现象。

# 4.如何打破百度资讯技术产品线的不作为的边缘性思维
为了打破百度资讯技术产品线的不作为的边缘性思维，百度资讯技术副总裁兼CTO张志刚提出以下六个方面的方案。

1.规范产品开发流程：开放的产品开发流程能够激励更多技术人员参与产品的开发，同时确保产品质量和效率，减少不必要的失误。除了通过培训等方式提升技术人员能力外，还可以通过工具的选择、技术文档的编写、自动化测试的引入等方式来规范产品开发流程。

2.重视技术能力建设：通过合作伙伴、讲师等形式培训和提供技术能力建设机会，促使技术人员认识到自己的专业能力和职业技能是企业不可替代的资源。传统培训模式往往不够灵活，因为学校设置的课程往往是不利于技术人员发挥潜力的。因此，需要借鉴国内顶尖IT企业的教学模式，开展信息技术职业生涯规划、职业生涯培训和职业技能培训。

3.集聚人才：创业公司都需要大量的人才，如何利用人才库进行有效分配，以及如何选取人才来解决实际问题是一个重要的战略选择。通过“知识星球”等公益社区形式，把学到的知识与人才资源结合起来，更好地集聚人才。

4.主动承担风险：百度资讯是一个非常复杂的技术产品，技术部门需要承担很多复杂的任务。如何主动承担风险，使技术人员有条理地处理复杂的项目，为公司创造更大的价值是一个关键。例如，不断完善数据质量标准、建立安全风险预案，减少系统漏洞等，都是为了确保公司的数据和系统更加健壮、安全。

5.树立技术人才观：技术团队并非只承担技术工作，而应更加注重解决实际问题、提升服务水准。例如，建立各项培训誓师、举办技术活动、设立技术奖项等，可以激发更多人才热情参与到业务中。同时，也应树立技术人才观，从人才角度思考问题，培养人才公平竞争、职业道德等品质。

6.创造新的工作环境：由于技术人员的不了解，产品团队和业务部门之间往往存在信息不对称，容易造成业务流程的不顺畅。如何创造新的工作环境，鼓励业务部门主动参与技术实现，把更多的精力放在业务建设上，是一个关键。例如，通过远程工作、定期举行技术分享会等方式，可以增加员工之间的沟通交流，促进业务流程的顺畅运行。

综上所述，打破百度资讯技术产品线的不作为的边缘性思维，需要综合考虑一系列方案，包括规范产品开发流程、重视技术能力建设、集聚人才、主动承担风险、树立技术人才观、创造新的工作环境等。只有打破不作为的边缘性思维，才能取得更好的效果。
