# 1.简介
  

## 1.1 发行股票的定义及其背景
在上世纪90年代，美国曾经有过一段关于创业者发起个人股权投资计划的讨论。但当时人们还不清楚这种投资行为到底能够带来什么样的收益，只是在寻找一种可能性。随着信息技术的飞速发展，创业者越来越多地选择了这个领域作为自己的投资方向，甚至出现了向股权投资发展的新趋势。虽然股市随着互联网的普及也越来越受到青睐，但发行个人股票并非每个人都适合，并且有很多限制条件需要满足才能得到该种投资。其中包括但不限于企业估值、注册资本、年营业额、业绩表现等要求。
随着这些年来股市的火爆，越来越多的人加入了股市的阵营，他们通过不同的方式获得收益，比如参加众多基金管理公司的活动、持股期权或其他证券类产品等。而这其中又有相当一部分人为了参与股市交易而对其有一些限制和认识上的困难。例如持有股票的要求最低要达到一定级别，有时还会被禁止发行股票。同时，还有很多人有特定的个人财产或特定职业身份，而不能够申请发行股票。因此，对于那些想要申请发行个人股票的人来说，了解一些必要的资格条件很重要。
那么这些资格条件是什么呢？它们有哪些具体限制？这就需要对相关法律法规和业务规则有所了解。在这份文档中，我将会简要介绍一些发行个人股票的资格要求，并且详细阐述每条要求的具体限制条件。
## 1.2 发行个人股票的资格要求
### 一、企业资质
发行个人股票首先需要有一家企业的法定代表人（或者被授权代表人）在场。即使是个人参与发行个人股票的私募股权投资基金，也必须由法定代表人签署“基金注册”文件并缴纳相应的保证金。根据美国证券监管机构要求，私募股权投资基金必须具备以下几个要件：

1. 成立时间：基金的成立时间至少应该有五年以上；
2. 投资能力：基金的资金来源必须是可靠的，能够提供高质量的投资；
3. 公司治理结构：基金必须严格遵守公司法规，且有独立的董事会；
4. 个人持股比例：私募基金必须要求所有成员持股总额不超过50%。

此外，个人还需要持有企业的注册证明，需要其营业执照、工商登记册、税务登记证件等有关资料。如果所持有的股份属于已退市的企业，还需要提供其原始股东会决议书。如果企业存在质押债务，则需要提供质押出具的保证函或担保书。另外，基金管理人还应保持积极的言行举止，并且确保所持股份没有任何重大违规记录。

### 二、资产负债率
个人股票可以设定的最低资产负债率为15%——这意味着企业的股东净资产占其总资产的比率至少为15%，而且其贷款余额和股息总额占其负倡イル。当然，个人也需要遵守国家对于非公开发行股票的限制。

### 三、年度经营报告
个人股票的第一大限制就是其经营的质量。一般要求其每年提交年度经营报告，并经过研究后，将其审核的结果发布出来。年度经营报告的目的是检查企业的实际运营情况，如销售收入、营业利润、资本支出、现金流、资产负债率、企业声誉、股东权益等指标。这些信息都会反映企业的实际情况，并帮助基金管理人判断该企业是否适宜作为基金的投资对象。同时，年度经营报告还可以用于筹措资金，通过其分析发现有价值的项目并购买它。

### 四、年度公司会议
对于发行个人股票的人来说，另一个重要的资格条件就是每年召开一次公司董事会，向股东进行股权激励。前几年，每年公司董事会的开销相当可观，但现在很多基金已经开始减少董事会的开支。每年召开一次公司董事会，可以让管理层得知业绩的最新消息、面对客户提出的疑问及建议，并就下一步的发展方向做出决定。例如，下一届董事会可以决定是否增发新股，修改基金制度，或者改变经营策略。

### 五、个人持股比例
一般而言，个人股票的持股比例是20%—30%之间。除了满足最低的资产负债率要求之外，这一限制还有一个原因是防止股东因经营亏损而索贿。如果一个企业的总股本比较小，比如只有200万股，但其股东却持有70%以上的股份，就会造成股东暴力，削弱其投资决策的能力。

### 六、被监管部门审核
一般情况下，个人持股比例在上市之前不会被立案调查。但是，一旦股票被上市，则可能会被法院或其他监管部门调查。如果被判处罚款或没收个人股票，则需要支付相应的赔偿。所以，在申请发行个人股票之前，需要注意自己的状况是否被监管部门注意。

### 七、资本金
个人发行个人股票的时候，除了需要准备一定的资金外，还需要花费一部分资本金。从理论上讲，个人股票的价值主要取决于股东们对其所持股份的认同程度。持有更多的股份，股东的认同度就更高，就会对企业的发展越来越感兴趣。同时，每个股东对股票所承诺的回报也都会影响企业的盈利水平。然而，由于持股需要缴纳一定比例的管理费用，基金管理人往往会选择有良好信用的公司来发行股票，以保证自己能够获得足够的收益。

### 八、资本积累
个人股票的另一个限制条件是没有做空机制。也就是说，个人股票的价格永远不会低于其股票面值的价值。虽然这是一个重要的限制条件，但并不是所有的人都适合持有股票。举个例子，如果一个人持有超过其所能承受的最大股票数目的股票，他或她就会失去对其股票的信心。所以，尽管持有个人股票会产生一定风险，但基金管理人还是希望把个人股票投资的收益最大化。
