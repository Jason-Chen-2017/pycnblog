# 1.简介
  

云计算是一种新的计算模式，它利用廉价、灵活、可扩展的基础设施资源提供按需计算能力，通过网络和软件技术将数据、应用和服务快速部署到用户手中。云计算的流行带动了信息化、金融、制造和其它领域的信息化运营模式的改变，也促进了企业科技创新。因此，掌握云计算相关知识，能够帮助我们更好地了解技术发展方向，并在具体业务中选择合适的工具。

# 2.基本概念术语说明
## 2.1 云计算概念
云计算(Cloud Computing)是一种计算模型，基于共享的硬件资源、软件资源和服务支持的互联网分布式计算环境。可以说，“云”就是连接性、弹性、按需使用、高度可靠、由第三方供应商提供的计算服务。
云计算的目的是通过网络和软件技术为用户提供计算、存储、网络、数据库、安全和应用服务等IT资源，通过一系列的自动化手段实现资源的动态分配、调度、管理，从而使计算服务得到有效利用和降低成本。
云计算涵盖了多种形式，主要分为公有云、私有云、社区云、混合云四大类。

### 2.1.1 公有云
公有云，即提供商提供公共资源给公众使用，基础设施由第三方供应商提供（如互联网公司），用户无需购买自己的服务器、存储设备和网络设备即可使用，主要服务包括计算资源、存储服务、数据库服务、网络服务、安全服务、应用服务等。
目前主流的公有云服务提供者有Amazon Web Services、Microsoft Azure、Google Cloud Platform等。

### 2.1.2 私有云
私有云，即自建数据中心，运行商自行承担基础设施设备的采购、安装、配置、维护等任务，用户独享自己的数据中心，主要服务包括计算资源、存储服务、数据库服务、网络服务、安全服务、应用服务等。

### 2.1.3 社区云
社区云，即公共数据中心托管服务，将数据中心托管于云平台之上，允许个人或组织分享数据中心的服务器、存储设备、网络设备和系统资源，其他成员也可以通过网络与数据中心进行交互。主要服务包括计算资源、存储服务、数据库服务、网络服务、安全服务、应用服务等。

### 2.1.4 混合云
混合云，既结合了公有云的优势，也兼顾了私有云的安全性、灵活性及本地化的特点，属于公有云和私有云之间的一个平衡。

## 2.2 IaaS、PaaS、SaaS
IaaS(Infrastructure as a Service)，即基础设施即服务，指提供计算、存储、网络等底层云计算服务，包括硬件虚拟化、服务器虚拟化、网络虚拟化、存储虚拟化等功能。
PaaS(Platform as a Service)，即平台即服务，指提供软件开发环境、编程接口、中间件等云计算服务。
SaaS(Software as a Service)，即软件即服务，指软件服务，按使用方式的不同，可以分为以订阅为核心的SaaS、以租用为核心的CaaS、以即用即付为核心的BaaS等。

## 2.5 云计算的特点
云计算具有以下五个重要特征：
1. 按需访问：无论是在线还是离线，客户都可以随时使用云计算服务。不需要预先支付费用，只需要按照实际需求付费。
2. 按量付费：按秒计费或者按事件计费，根据消费情况付费。
3. 可拓展性：可以按需增加资源，按用量收费。
4. 高度可用：采用多备份机制保证高可用性。
5. 即时弹性：可以自动缩放，随时响应客户请求。
