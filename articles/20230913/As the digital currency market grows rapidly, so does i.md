# 1.简介
  

近几年随着数字货币市场的火爆，投资者对加密货币的关注也越发强烈。对于那些刚接触加密货币市场、正在进行测试或准备进行财富规划的人来说，选择成为密集交易者是一个不错的选择。作者认为，作为一名优秀的加密货币交易者，需要具备以下几个品质：

1. 抽象能力：了解市场状况、技术分析和投资策略，并运用自己的眼光看待市场动向，做到冷静判断，找到符合自己需求的买卖点；

2. 抗风险能力：加密货币是一个新生事物，所面临的风险也是多种多样的，包括信用卡被盗、犯罪分子滥用信用卡等等，都可能导致资产损失甚至损失极大的风险，因此我们要有耐心且果断地应对这些风险，提高我们的安全意识和自我保护意识；

3. 实操能力：掌握足够的编程技巧和相关工具，善于利用社交网络资源和信息共享平台，建立可靠的投资组合，在实际场景中完成套利、交易等操作；

4. 专业精神：加密货币是一个新兴领域，很难就此找到一个十全十美的解决方案，因而我们不能仅局限于单一的方向，还应该注重综合性、客观性和长远性的考虑；

5. 坚持信念：尽管加密货币市场处于快速发展阶段，但长期来看仍然会保持稳定的结构，为了能够长久保持这个优势，我们必须保持对市场的敏锐嗅觉、不断反思、积累经验，尤其是在管理我们的资金方面。

通过以上五个品质，可以看出，作为一名优秀的加密货币交易者，除了具备广泛的知识面外，还需具有丰富的工作经历、团队精神、开阔的视野、独立思维能力和严谨求实的态度，才能真正走上一条成功之路。 

# 2.基本概念术语说明
加密货币的概念最早由中本聪在2009年提出，它是一个利用互联网技术提供去中心化、匿名、不可伪造、透明化服务的电子支付系统。目前，加密货币已经成为行业内热门话题，并已经逐渐成为热门话题。作者在这里介绍一下加密货币的基本概念、术语及特点。
## 2.1 加密货币概念及特点
加密货币（cryptocurrency）是一种通过计算机运算而不是纸刮头发的方式进行转账和交易的数字媒体。它是利用分布式记账方式来记录所有用户的余额变更情况的一种价值存储媒介，具有独特的特征。在加密货币系统中，所有参与者都可以公开验证每笔交易的信息，而且所有信息均通过加密算法加密保存，无论哪个参与者看到了交易记录，都无法获悉真实的信息。另外，加密货币系统中的所有交易都是公开透明的，任何人都可以查询到交易记录。由于加密货币系统中没有真实的人物参与，所有参与者都可以进行匿名交易，从而防止虚拟货币发展成为实物货币的模式。

加密货币具有如下特点：

1. 匿名性：加密货币系统完全隐藏所有用户的身份，使得所有交易过程及交易双方的真实身份均无法查知。同时，由于所有交易发生在区块链系统中，因此不受信任的第三方也无法插手交易，确保整个交易过程的隐私安全。

2. 智能合约：加密货币系统采用智能合约机制来确保所有的交易行为均被自动执行，保证交易的透明、公正和可追溯性。

3. 去中心化：加密货币系统不存在单一的中心节点或所有权者，而是采用分布式记账机制，允许任何人参与并贡献自己的计算力资源。这种去中心化特性意味着任何个人或组织都可以创建、运行及维护加密货币系统，从而使系统的参与者及交易行为更加公平、透明、自由。

4. 高度可靠：加密货币系统能够抵御各种各样的攻击，如病毒入侵、系统瘫痪、恶意竞争者、勒索软件等。同时，通过不可逆的加密算法，加密货币系统可以在出现系统故障时快速修复，有效保障交易的顺畅及可靠性。

5. 没有中间商赚差价：由于加密货币系统中的所有交易都是公开透明的，任何参与者都可以通过查询历史交易记录来核实真实的资金流向，从而避免虚拟货币成为实物货币。

6. 效率高：因为加密货币系统采用分布式记账机制，使得每个参与者都能够参与交易及生成新的代币，从而促进经济活动的活跃。

7. 可编程：加密货币系统能够实现基于智能合约的定制化操作，从而满足不同用户的不同交易需求。
