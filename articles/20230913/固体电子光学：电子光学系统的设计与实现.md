# 1.简介
  

## 1.1 发展历程及基本概念
电子显示器的出现使得显示器从古老的矿石屏幕发展到可以显示数字、图形等各种媒体的模拟信号。随着近年来科技的飞速发展和信息时代的到来，电子显示器已经成为众多应用领域中的重要工具。

电子显示器的主要功能有三种：显像、显示和操控。显像指的是通过将模拟信号转换成电流脉冲的方式对真实世界的物理对象进行反映，其输出为明暗变化或颜色的图像。显示则是指电子信号的传输，将模拟信号呈现给人眼所看到的最终效果。操控则是指在显示过程中根据人类动作对电子信号进行控制，使之产生特定效果，如对电视节目进行调频、声音响度的调整、播放器中的音乐播放等。

为了能够显示电信号，电子显示器通常由一组薄膜层、微结构、光学元件、调制解调器（包括解码器、补偿滤波器、发光二极管）等构成。显示器分为白平衡、色彩匹配和动态范围三个方面进行优化，以提高显示效果。一般来说，白平衡是指将显示器的输入信号按照亮度或色温来保持一致；色彩匹配即指把输出的图像的颜色符合观者的感官需要；动态范围指的是电子信号的最大值与最小值的差距。

电子显示器的具体型号各异，但一般可以分为两种类型：固定模式显示器和可移动显示器。固定模式显示器的特点是在设备不经过任何操作下就可切换不同的显示模式。比如，在水平垂直同步的VGA格式中，一种模式可以显示成两张彩色图片。而可移动显示器则不同，它可以在用户需要时通过手动移动和旋转进行切换。

## 1.2 固体电子光学
固体电子光学(SEM)是利用半导体材料的结晶结构及相关的物理过程来研究微观电子及其他辐射源的过程和结构的学术研究领域。由于具备独特的、非常复杂的光学特性，因此SEM可用来测量低分辨率下的电子规律，并具有更高的精度。固体电子光学是无线电电子学的基础，是一种研究微电子物理的新型方法。它的应用前景十分广阔，尤其适用于纳米级的微电子、集成电路、激光、照相机等关键领域。

### 1.2.1 SEM与传统显像技术的比较
传统显像技术包括视网膜显示器(RDS, Retinal Display System)，X射线显像系统(CATScan, Computed Tomography angiography and Atherosclerotic Trauma)等，它们都可以直接在组织内部进行显像。然而，这些显像技术往往存在以下缺点：

1. 操作复杂、耗费时间长。
2. 反映出来的图像往往不够清晰。
3. 受光源的影响较小。
4. 需要使用专门的设备。

而固体电子光学显像技术解决了以上问题。固体电子光学显像技术直接利用组织中存在的粒子(如离心分子)来记录电场分布，并非要在组织外部搭建显像平台。在组织内部的电子活动区域，电子就会自发地扩散出去，逃逸出粒子状的结晶，在结晶表面形成信号条带，然后再送回显像平台进行显像。这样，就可以避免麻烦的操作和耗时的传统显像技术，而且在不损害组织光学特性的条件下获得很高的精确度。而且，由于结晶结构本身具有良好的视觉性能，SEM显像还能够提供较高的分辨率和清晰度。

除此之外，由于SEM显像是采用材料自发扩散的方式来记录电子，所以它可以提供比传统显像更高的精确度，而且不受光源影响，因此适用于真实世界的复杂场景。但是，由于传感器采集到的信号不是实际的电流脉冲信号，所以不能立刻被人类的视觉系统所识别，而需要一些后处理才能得到有效信息。另外，SEM显像只能进行静止图像的显示，对于动态图像的显示效果较差。

### 1.2.2 电子显示器的组成
电子显示器的组成主要分为几个部分：光源、显示面板、元件、连接器、驱动电路、控制器等。其中，光源通常包括数码光源、激光束、等离子激光，光照和环境反射作用下会产生电流脉冲。显示面板主要包括模拟信号与电子信号之间的转换。元件包括各种器件，如镜面反射元件、微结构分割元件、调制解调器、放大器、显示电路等。连接器则负责电路的连接，驱动电路负责驱动电子的运动。控制器负责对电子信号进行调制、控制和传输。
