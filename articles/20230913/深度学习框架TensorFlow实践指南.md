# 1.简介
  

## 概述
本书是基于TensorFlow框架进行深度学习的实践手册，主要目的是让读者快速了解和掌握TensorFlow的一些基本用法、常见模型结构及其实现，并能对深度学习及其相关应用有更高层次的理解。

本书共分为两大部分，第一部分介绍了TensorFlow的基本知识、安装配置及环境搭建；第二部分重点阐述了TensorFlow在图像分类、文本分类、序列标注、语音识别等常见深度学习任务中的应用方法，并通过丰富的实例代码展示了如何利用TensorFlow实现这些功能。每章节后都给出了作者们的建议，希望能够帮助大家更好地理解、掌握和使用TensorFlow框架。

本书适合没有任何机器学习或深度学习基础的人士阅读。对于有一定机器学习和深度学习基础的读者，也可以作为进阶教程，或者作为查漏补缺的工具。
