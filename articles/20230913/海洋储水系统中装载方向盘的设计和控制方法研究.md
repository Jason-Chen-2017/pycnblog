# 1.简介
  

海洋储水系统（英文名Seawater Tank）是指用来作为运输、仓储或生活等应用场所的污水的集合体。海水中含有的淡化溶解物质和微生物可以被浸没在海水的污水里，成为海水污染物，而海水中含有的碳酸钙和其他化学成分也会对其造成破坏。因此，海水污染物对海洋储水系统的功能及其运作产生了严重影响。
在中国，海洋储水系统主要分布于陆地和内陆，如黄河、长江、珠江、三江、台风山口的水库、湖泊等处，广泛用于燃料供应、油气加注、港口经营等方面。这些海洋储水系统中的船舶经常要进行艰苦的紧急救援任务。虽然各地都建立了专门的船舶装备队伍，但由于各种原因导致船只过早落入水底，导致船只的生命安全受到威胁。所以，如何有效地控制船舶方向盘，确保海洋储水系统中的船舶安全驶离，是解决此类问题的一项关键工作。
本文将阐述一种基于轨道分析法的设计和控制方法来解决上述问题。
# 2. 概念和术语说明
## 2.1 海洋储水系统
海洋储水系统（英文名Seawater Tank），一般来说，是一个大型、多层的建筑物内，掩蔽着溶解在海水中的污染物的封闭空间。它容纳着大量的冷凝水或者水处理后产生的液态、固体、塑性物质、氮氧化物，是一种海水环境中庞大的水体物质积。这里的空气是不可能排出去的，只能靠这种污染物的作用。海洋储水系统的设计目的是为了防止这种污染物进入到该区域内的居民家中，并保持长期供水供给。在中国，它分布于陆地和内陆，如黄河、长江、珠江、三江、台风山口的水库、湖泊等处，广泛用于燃料供应、油气加注、港口经营等方面。
## 2.2 轨道分析法
轨道分析法（又称动力学分析法，Kinematics Analysis Method）是由美国航空航天局开发出的，用于描述运动学（机械运动学和力学运动学）的一种方法。该方法把复杂的空间曲线运动模型表示为一个整体，即把多片空间曲线束缚在一起组成一体的统一空间曲线。随着时间的推移，这一体空间曲线形态发生变化，从而反映出运动过程中的空间曲线运动特性。轨道分析法是一种直接测量和分析运动学数据的理论方法。
## 2.3 方向盘
方向盘，即船舶最重要的一个部件之一，是一种能够使船舶旋转、前进、后退以及侧向移动的装置。它在大多数船舶的控制系统中扮演着重要角色，其控制就是通过方向盘来实现船舶的动力效率最大化。方向盘由舵轮、齿轮、轴承、制动器、支柱、驱动电机组成。方向盘可实现船舶在岸上平稳、缓慢移动，进行正常航行，也可将船舶导向周边干燥的环境。
## 2.4 轨道——方向盘的相互作用关系
由于海洋储水系统中的船舶遭遇各种因素而无法继续航行，所以对于它们的方向盘进行优化，减少它们对地表的影响，提高其安全性、生命周期，具有十分重要的意义。

## 3.2 空间曲线运动模型及相对于坐标轴的角度
### 3.2.1 曲线运动模型
空间曲线运动模型，包括矩形曲线运动、圆弧曲线运动、曲面曲线运动等。举个例子，一条曲线可以由若干控制点连成，形成一个空间曲线，其中任意两个控制点之间的空间曲线段为一段曲线，这条曲线就被称为曲线运动模型。其一般表达式如下：

y = f(x) + ε (ε 为任意常数) 

f(x)，y是空间曲线的参数坐标。

矩形曲线运动模型：在矩形平面内运动的曲线，其轨迹为一条直线。方程形式：

dy/dx = k 

k为矩形曲线的倾斜角度。

圆弧曲线运动模型：在圆弧内运动的曲线。方程形式：

r cosθ = x - a 

r为圆弧半径，θ为圆弧曲线的起始角，x为圆心距。

曲面曲线运动模型：曲线在曲面的切面上的曲线运动。方程形式：

dz/dt = f(x, y, z, t) 

z是曲面的高度参数，t是时间参数。

### 3.2.2 相对于坐标轴的角度
对于任意一段空间曲线，其相对于坐标轴的角度值可以通过点积的方式求取。假设曲线在直角坐标系中的表述形式为：

p = (x0, y0) + r cos θ 

q = (x1, y1) + s sin θ 

其中，p, q分别是曲线上的两点，x, y为曲线参数坐标，θ为箭头方向，r, s为曲线的长度，p, q为实际情况中的点坐标。则相对于坐标轴的角度值为：

theta = arctan [ (qy - py)(qx - px)/(qx - px + qy - py)]
