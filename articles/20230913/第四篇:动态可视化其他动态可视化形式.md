# 1.简介
  

## 概述
时间序列数据，时序数据，或者说序列数据是数据的一种类型，它包含了过去发生的一系列事件及其顺序，每一个事件都带有时间戳。对于时序数据的可视化分析能够帮助我们对数据进行快速、直观地理解和分析，从而发现隐藏的信息或模式，提升我们的洞察力和能力。本文将介绍另外一些常用的时序数据可视化形式，包括雷达图、分面雷达图、旭日图等。

## 主要内容
1. 雷达图

雷达图（Radar Chart）是一种用面积填充图表示多变量数据的图表，每条轴代表一个变量，每一个面片的大小和颜色反映变量值大小。它的主要优点在于易读性强、突出重点、表达力强。

首先，让我们举个例子来认识一下雷达图的概念。假设我们有一个五维时序数据集(t1, x1), (t2, x2), (t3, x3),..., (tn, xn)，其中t1<=t2<=...<=tn，x1,x2,...,xn是对应时间的五个变量的值。


雷达图就是把每个时间和变量作为一个轴，把所有的数据点连接起来，这样就产生了一个完整的三维图形，用来显示数据的变化过程。如下图所示。


2. 分面雷达图

分面雷达图又称为面板雷达图，是为了解决雷达图不好处理同时存在多个变量的问题。它把雷达图按时间或空间切割成不同区域，然后分别显示每个区域内的所有数据。不同区域可以用不同的颜色来区别，用来显示不同因素之间的交互作用。

3. 旭日图

旭日图（Wind rose chart）也叫极坐标图，是一种二维图表，通常用于显示不同方向的风速和风向分布。它由一组由不同颜色的圆盘组成的线状符号组成，不同的颜色表示不同的风向。每一组线状符号，按照顺时针或逆时针方向，分成若干等分，每个等分代表不同风速段。符号半径越大，代表风速越大；符号间隔角越小，代表风速越分散。

除了这些一般性的时序数据可视化形式之外，还有很多其它类型的数据可视化形式，如折线图、条形图、箱型图等。这些形式也都可以用来可视化时间序列数据。
