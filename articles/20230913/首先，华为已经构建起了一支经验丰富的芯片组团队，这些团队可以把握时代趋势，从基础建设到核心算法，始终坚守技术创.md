# 1.简介
  

芯片组（英语：Chiplet）指的是集成电路板上的多个集成电路单元(ICU)；它可以集成相关模块并且可以被用来制造产品或解决特定问题。芯片组被设计、制造、测试和部署在处理器上，可进行集成计算、数据传输、网络通信、图像显示、多媒体处理等功能。一个完整的芯片组通常由许多不同器件组件组合而成，如存储器、处理器、接口设备、连接器和电源管理单元等，从而实现各种功能。
华为从2012年开始引入了芯片组设计方法，并在华为集成电路部门（IHC）共同推广。目前，华为拥有全球最大的芯片组设计研发中心——华为芯片技术研究院（HCT）。此外，华为还建立了覆盖全球的芯片组培训中心、产业园区和工程服务部，助力公司构建一支庞大的芯片组团队。
基于这一优势，华为持续投入芯片组设计和开发领域。目前，华为已建成了一个芯片组设计服务平台，包括硬件设计、工艺设计、软件设计、生态系统设计、工具链支持、芯片封装验证、量产认证、国际合作、品牌宣传等多个领域的能力及资源。华为的芯片组设计项目由业界领先的专业团队负责，每一个阶段都注重对项目整体的规划、策略和流程优化，确保项目按计划、按质量、按时交付。在2022年，华为将继续加强芯片组设计服务，以提升华为芯片组的整体水平。
# 2.基本概念术语说明
## 2.1、术语和缩写词
- ICU (Integrated Circuit Unit)，集成电路单元：芯片组上的一个个芯片。
- DSP (Digital Signal Processing)，数字信号处理：指通过一定的运算和控制规则，对输入信号进行分析、滤波、采样、变换、编码、解码等过程后生成输出信号的一类计算机处理器。
- SoC (System on a Chip)，系统集成子系统：是一个包含了CPU、内存、I/O接口控制器、微处理器、传感器、显示屏、触摸屏、GPS芯片等一系列部件的完整电子系统。
- IP (Intellectual Property)，知识产权：任何创建于私人所有之上的有形或无形产物。
- IC (Integrated Circuit)，集成电路：由多个不同的元件组成的电路板。
- FPGA (Field Programmable Gate Array)，可编程门阵列：一种可编程逻辑电路，其内部可编程电路的数量可以根据需求增加或者减少，灵活性高且价格便宜。
- AIB (Analog Input/Output Bus)，模拟输入/输出总线：连接FPGA的模拟输入和输出之间的总线。
- AD (Analog-to-Digital Converter)，模-数转换器：将模拟信号转换成数字信号的硬件电路。
- CPLD (Complex Programmable Logic Device)，复杂可编程逻辑器件：是一种可以进行全面配置和编程的器件，是一种高级的集成电路。
- PCB (Printed Circuit Board)，印刷电路板：用来制造集成电路板的电路板材料，也称硬件原件或硬件设备。
- TSMC，台积电（Technology and Smart Manufacturing Company），美国半导体工业巨头。
- CP (Customization Process)，定制化过程：是指从零开始按照客户的要求生产产品或者提供服务的过程。
- VES （Value Engineering System），价值工程系统：是华为高科技的核心工程体系，围绕价值驱动的管理理念，以数据驱动价值赋能业务的转型升级。
- SRAM，静态随机存取存储器：一种存储器芯片，用于储存存储程序的数据和指令。
- ASIC (Application Specific Integrated Circuit)，应用专用集成电路：用作特定的应用领域的集成电路。
- GB/s，Gigabit per Second，意即“十亿比特每秒”的速率。
- FIFO，先进先出队列：一种数据结构，它将最近接收到的数据存放在队列前端，最早接收到的数据存放在队列末尾。FIFO中数据的插入和删除都是在队列前端和末尾进行。
- LUT (Look Up Table)，查找表：一种电路板上用来保存和快速检索电路参数的表格。
- BRAM，块随机存取存储器：一种高速的静态随机存取存储器，常与DSP和FPGA配套使用。
- HDL (Hardware Description Language)，硬件描述语言：一种用类似于人类的语言来描述硬件电路和其他电子元器件的软件。
- GPP (Global Positioning Protocol)，全球定位系统协议：是利用卫星电视接收机、卫星导航系统和卫星云监控系统，结合地球自转模型及人们的航迹，精确计算出用户所在的经纬度和高度。
- LTE (Long Term Evolution)，长期演进：是当前移动通信领域的一个新的技术架构，它基于新的频段技术、新标准协议、多种网络架构和多种运行环境等，将信息技术发展到了新的水平。
- SCADA (Supervisory Control And Data Acquisition)，强制控制数据采集：是用于监控生产过程和管理设备的仪表化技术。
- RFID (Radio Frequency Identification)，射频识别：通过标签或超声波等方式来识别信息的电子标识技术。
- M2M (Machine to Machine)，机器对机器：是一种技术，允许不同网络节点之间交换数据或服务。
- IoT (Internet of Things)，互联网物联网：指的是利用互联网技术和现实世界物理对象相互作用，实现物联网的分布式系统。
- WAN (Wide Area Network)，广域网：指能够连接因特网多个区域的大型计算机互连网络。
- Wi-Fi (Wireless Fidelity)，无线信噪比：指单位时间内的无线收发信号衰减量。
- NAND Flash，N二极管闪存：一种高速闪存芯片，主要由NMOS晶体管构成。
- FFT (Fast Fourier Transform)，快速傅里叶变换：一种高效且精确的计算离散傅里叶级数的方法。
- CNN (Convolutional Neural Network)，卷积神经网络：是一种对图像进行分类、检测、识别的深度学习技术。
- LSTM (Long Short-Term Memory)，长短时记忆神经网络：是一种适用于序列数据的递归神经网络。
- GPU (Graphics Processing Unit)，图形处理器：一种专门用于图形渲染和动画处理的特殊芯片。
- ROM，只读存储器：一种只能用于读取数据的存储器芯片。
- SSD，固态硬盘：一种非易失性磁盘，它具有更快的响应速度，平均寿命比以前的硬盘长达10倍以上。
- RAID，廉价磁盘冗余阵列：一种磁盘阵列，它将多个硬盘组合起来，共同提供容量和性能，即使其中某些硬盘损坏了也可以正常工作。
- PCIe，桥接卡扩展接口：一种高速通道接口，可通过PCI Express总线扩展计算机系统的功能。
- USB，通用串行总线：一种外插式设备接口，可以在计算机和外部设备之间传输数据。
- Ethernet，以太网：一种以太网通信技术，它在连接不同的网络设备时所使用的物理层协议。
- TCP/IP，传输控制协议/互联网协议：是Internet的传输层和网络层协议。
- UDP，用户数据报协议：一种无连接的传输层协议，它不需要建立连接就可直接发送数据包。
- HTTP，超文本传输协议：是Web服务器和浏览器之间通信的协议。
- HTTPS，安全套接层协议：是HTTP协议的安全版本，采用SSL加密数据包。
- SSL，Secure Socket Layer：一种加密传输协议。
- TLS，传输层安全协议：是SSL的升级版。
- VPN，虚拟专用网络：是通过公共网络建立专用的网络连接，它为个人提供了无缝的网络访问。
- CDN，内容分发网络：是互联网服务提供商及运营者用来加速网站内容访问的网络基础设施。
- Docker，容器技术：一种轻量级的虚拟化技术，能够将应用程序打包成为一个独立的容器，通过容器引擎共享其运行环境。
- K8S，Kubernetes：是Google开源的容器集群管理系统。
## 2.2、重要抽象概念
### 2.2.1、可编程逻辑器件、FPGA
可编程逻辑器件（Programmable Logic Devices，简称PLD）是指带有可编程逻辑元素的IC，可以根据客户需求进行组装和编程，构成可以解决各种规模问题的集成电路。FPGA（Field Programmable Gate Array）是一种可编程的集成电路，其内部由可编程逻辑块构成，可以通过配置逻辑的方式改变内部电路的功能和布局。它的出现促进了计算机系统的迅速发展，提高了设计效率和灵活性。FPGA可实现非常精密的功能，但是缺乏对成本的控制，而且大小较大、功耗高、集成电路密度高，很难跟上CPU的发展步伐。因此，华为选择FPGA作为其芯片组的核心，其重点放在系统级芯片上。
### 2.2.2、高性能计算架构HPC（High Performance Computing Architecture）
HPC（High Performance Computing）是指通过将多核CPU、GPU、FPGA等各种芯片集成到一起，提升性能、并行计算能力、数据处理吞吐量等方面的能力。HPC架构向着高端市场转型，将导致系统架构发生剧烈变化，将面临新的计算任务调度、资源管理、编译优化、系统层面的优化等复杂问题。华为在HPC架构上持续投资，已经有多个成功案例。
### 2.2.3、互联网+（Internet Plus）、物联网（IoT）
互联网+是将科技、资讯、生活三要素融合为新互联网的发展趋势，融合互联网的各种能力，以提升用户体验、节约能源、降低成本、扩大影响力。这种转变将给经济社会带来深远的变革。同时，随着物联网的发展，物流、智慧城市、自动驾驶、环保等应用场景将会进一步增长。华为也致力于物联网的应用，将物联网技术和智慧应用于生活、健康、教育、军事、金融等领域。
### 2.2.4、模-数转换器、ADCs（Analog-to-Digital Converter）
模-数转换器（Analog-to-Digital Converter，简称ADC）是一种用于将模拟信号转换为数字信号的硬件电路。最早的数字计算机使用模拟信号采集数据，因此，模-数转换器成为处理模拟信号的必要组件。ADC的输入是模拟信号，输出是数字信号。典型的ADC包括如下几类：
- 静态模-数转换器（Static Analog-to-Digital Converter，简称SADC）：使用振荡器测量电压或电流并转换成数字信号输出。
- 可变模-数转换器（Variable Analog-to-Digital Converter，简称VADC）：在实时变化的信号上实时转换成数字信号。
- 时序模-数转换器（Time Sequence Analog-to-Digital Converter，简称TADC）：用于同步系统时钟的数字信号。
- 比较模-数转换器（Comparator Analog-to-Digital Converter，简称CACDC）：根据比较信号输出相应的数字信号。
- 独立多通道模-数转换器（Independent Multichannel Analog-to-Digital Converter，简称IMADC）：可实现多通道的高速转换。
### 2.2.5、存储器、SRAM、DRAM
存储器（Memory，简称Mem）是指用于暂存程序和数据的数据存储器。SRAM是一种随机存储器，其可以像动态RAM那样提供快速的读写速度。虽然SRAM有很小的访问延迟，但它的动态刷新特性使得其比动态RAM快很多。DRAM则是一种主动性较弱、速度比SRAM慢、容量比SRAM大、一次性读写的存储器。华为集成电路部门已经开发了一种新的存储器命名为FRAM，即快页式RAM（Flash Random Access Memory），它比DRAM快一些，价格低。
### 2.2.6、块级RAM、BRAM、URAM
块级RAM（Block RAM，简称BRAM）是一种比普通RAM（Static RAM）具有更高数据传输速率的存储器。它们可以更大程度地发挥动态RAM的存储能力。BRAM又分为两大类，一类是单端口多端口BRAM，另一类是双端口BRAM。单端口多端口BRAM可同时支持读写，两端口BRAM则支持同时读写。两种类型BRAM一般用于存储器中高速缓存。URAM（Ultra-low Power RAm）是一种无穷小电容的存储器，它的功耗小于10uA。
### 2.2.7、数据库引擎、NoSQL、OLAP
数据库引擎（Database Engine）是指运行SQL命令、维护数据、查询数据等相关功能的软件。NoSQL是指非关系型数据库，其不依赖SQL语法、面向文档、键-值对等数据模型。OLAP（OnLine Analytical Processing，在线分析处理）是指在不断产生海量数据后，需要快速分析数据的技术。OLAP通过提升查询速度，提高数据分析的灵活性、准确性和可视化能力。
### 2.2.8、云计算、容器编排、微服务
云计算（Cloud Computing）是指通过网络将IT服务、数据和资源聚合到一站式的服务平台上。华为在云计算领域已经取得长足进步。华为的创新是在开放、透明、标准的基础上，充分发挥云计算潜力，逐步实现资源池、弹性计算、快速部署、按需付费等特性。容器编排（Container Orchestration Platform，简称COP）是指将多个应用容器化、打包，并按照一定规则进行部署的技术。微服务（Microservice）是一种轻量级的、面向服务的架构模式，它将复杂的应用程序拆分成多个小型的、独立的服务，每个服务之间松耦合。
### 2.2.9、深度学习、CNN、LSTM
深度学习（Deep Learning）是一种通过多层次神经网络学习数据的技术，主要用于图像识别、语音识别、自然语言处理、视频理解等领域。CNN（Convolutional Neural Networks，卷积神经网络）是一种多层神经网络，主要用于图像识别。LSTM（Long Short-Term Memory）是一种循环神经网络，主要用于文本、语音、时间序列等数据预测。
### 2.2.10、全球定位系统、LTE、5G
全球定位系统（Global Positioning System，GPS）是以卫星为基地，依靠卫星的运动状态和轨道偏移等信息，计算得到地球的真实位置的全球性导航系统。华为正在着手将全球定位系统（GPS）用于5G网络中，并将其应用于车联网、智慧城市、智慧交通等场景。LTE（Long Term Evolution，长期演进）是当前移动通信领域的一个新的技术架构，它基于新的频段技术、新标准协议、多种网络架构和多种运行环境等，将信息技术发展到了新的水平。
