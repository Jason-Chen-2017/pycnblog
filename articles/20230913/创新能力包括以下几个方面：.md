# 1.简介
  

AI这个词被英文缩写成Artificial Intelligence (AI)或者Intelligent Agent (IA)，两者可以互换使用，但一般用缩写形式表示。特别是在机器学习、深度学习领域，将其中的“智”层面定义为一种机械性的模拟，并在此基础上引入计算机的一些特性，如计算能力和存储能力，赋予它学习、推理、自主的能力。而根据机器学习技术发展的历史进程，越来越多的人认为，这个词还会扩展到包括认知系统、逻辑系统等更高级的范畴中。那么，什么样的个人才能够成为真正的AI科学家呢？为了回答这个问题，本章先介绍一下传统机器学习的发展历程及其对人才的要求，再以典型的人才需求——机器学习工程师为例，谈一下他/她应该具备的基本素质。
# 2.机器学习的发展历史及其对人才的要求
机器学习(ML)这个概念最早由周志华教授于1959年提出。那时，他是斯坦福大学教授，是许多研究人员的学生。他提出的机器学习的定义是一个“使计算机具有学习能力的算法”。这种学习能力指的是让计算机能够从数据中自动分析出模式和规律，并且利用这些模式和规律对未知的数据进行预测或决策。因此，机器学习可以用于各种各样的问题，例如图像识别、语音识别、语言理解、生物信息分析、分类和排序等等。后来，机器学习的应用范围也变得越来越广泛，今天，很多应用都离不开机器学习。

- 数据处理能力：掌握数据预处理、特征工程、模型调优等技能。
- 编程能力：掌握Python、Java、C++等编程语言，了解机器学习框架、库，熟悉Python机器学习工具包。
- 模型训练能力：掌握机器学习模型的训练、评估、超参数优化、数据集划分、模型选择等技能。

3.业务理解能力：

作为一名机器学习工程师，你需要能够理解业务场景，清晰地阐述问题意义和目标。同时，你还需要懂得如何将你的工作结果转换为实际价值，能够提供一定的解决方案建议。

# 3.机器学习工程师的基本素质
因此，一个具备以上基本素质的机器学习工程师，具备如下几种能力：

1.数学基础：

- 线性代数、概率统计、随机过程、算法设计：机器学习的关键都是数学理论，要有扎实的线性代数、概率统计、随机过程、算法设计基础。

2.工程实践：

- 项目管理：擅长团队合作，善于组织，懂得如何拆分项目，分配任务，把控进度；
- 编程能力：掌握机器学习框架、库，熟练掌握Python、Java等编程语言，有良好的编程风格和习惯；
- 科研能力：自我驱动，积极参加国际会议和期刊，研究前沿技术，关注学术动态，能快速产出学术论文；
- 技术应用：用自己的知识、技能和方法帮助公司解决实际问题。

3.业务洞察力：

- 沟通能力：能够做到言之有物、能清楚表达自己的观点和意图；
- 领导力：热爱工作，对团队充满激情，对业务敏锐，能够带领团队达成目标；
- 产品经理：关注产品、市场和用户，喜欢把自己的产品理念和思路告诉大家；
- 商务理解：学会和不同部门建立有效的联系，对业务有全局性的把握。
