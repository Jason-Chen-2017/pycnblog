# 1.简介
  

自由职业是一个高新兴的职业类型。“自由”意味着不受企业约束，可以自行决定自己的工作时间、地点及方式。而“职业”也指由专业人士运作，负责某项具体任务，由其完成指定目标。在不加限制的前提下，成为一个自由职业者的人们通常具有高度个性化、灵活多变、创造力强、独立精神等特点。因此，许多人都会选择这样一种职业生涯。

2017年，全球仅有不到三分之一的劳动者拥有足够的收入和资产，且还有更多的人没有经济基础购买个人所有制企业的产品或服务。随着政府对自由职业者的打压越来越严重，越来越多的人要面临选择转型到市场营销、商务代理甚至全职创业的艰难抉择。“自由职业者可否过上自由职业者的生活？”这个话题很值得探讨。

首先，自由职业者究竟是什么样的工作状态？他们一般都有自己的工作职位，主要负责工作中的具体任务。除了具体的业务活动，还可能需要定期进行面试，参与公司组织的活动，或者协助管理团队。自由职业者往往非常忙碌，但同时也要注意休息，保持健康的身体，并且适当锻炼身体。更为重要的是，不要以盈利为目的，更不能过分关注财富。事实上，无论企业雇佣什么样的员工，自由职业者的工资一般都不高。但是，自由职业者也不是纸薄，可以通过参与各种社区组织、举办聚会、参加相关领域的演讲等形式获得满足感。

对于“为什么很多自由职业者选择创业”，这个问题也值得深思。这是因为，无论是在个人还是在商业层面，创业都是获得满足感、实现梦想、获得成就的过程。不管是个人还是组织，通过创造价值的过程，才会让你对自己的能力产生自信和自豪。而创业者往往为社会做出了贡献。所以，如果你真的想要自由，那么就去创造价值吧！

然而，创业并非一朝一夕之功。在成功创业之前，我们首先需要克服诸如创业的概念不清晰、创始人不了解自己的情况、无助于融资、创业失败导致破产等各种困难。我们还需要做好以下几点准备：

1. 设定一个明确的目标，并按时达成该目标。比如，你可以每月发布一篇周刊或总结，记录你的业绩、学习到的知识、和完成的任务。不要草率行事，要有条理、有计划。

2. 抓住机遇。尽早加入行业协会或社群，积极参与本地的技术、产品、营销、交流活动。帮助他人，鼓励别人。不要盲目跟风，迎接挑战。

3. 撬开筹备之旅。寻找合适的企业或组织，磨合自己的技能、团队和资源，给自己充份的时间。不要为了一时的成功而牺牲长远的发展。

4. 细心耐心。坚持日复一日的努力，不断寻求反馈和进步，增强自我成长能力。不要沉迷于短期，积极主动承担起长期负责任的责任。

5. 面对困难，保持冷静，乐观。虽然失败了也不要气馁，毕竟经历过失败的事情后，我们仍然可以从中学习。保持冷静和乐观，相信即使无法成功，仍有很多机会摆脱困境。

最后，也是最重要的一点，如果坚持走下去，那就一定会成功！即使失败了也不要气馁，坚持前进，有时迷失方向是正常的，但终归是自己不小心错失良机。
