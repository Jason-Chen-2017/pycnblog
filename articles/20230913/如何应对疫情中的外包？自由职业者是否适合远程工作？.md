# 1.简介
  

2020年初新冠病毒肆虐全球，导致许多国家经济受到严重冲击。许多国家采取了各种策略，例如关闭商店、限制旅行等，但当下的疫情却又给世界经济带来巨大的灾难性影响。许多企业和个人都面临着巨大的压力，不得不进行大规模裁员，他们只能通过远程工作的方式来实现团队协作，以应对疫情带来的影响。然而，自由职业者们对于远程工作方式是否适合自身就存在疑惑，因为自由职业者一般在较短的时间内无法接受真正意义上的长期工作，而且在短期内获得收入也比较困难。同时，许多自由职业者可能会担心自己的技能水平或语言能力可能会成为团队的障碍。因此，如何解除“职业枷锁”，帮助企业和个人摆脱“外包”的局面，成为了很多人的关切。本文将从以下几个方面阐述此问题：

1. 如何为个人/小型企业设置远程工作方案？
2. 为何企业采用远程办公？
3. 自由职业者的远程工作模式是否适合？
4. 为何有些自由职业者选择跳槽？
5. 没有工作、无收入时的自由职业者怎样才能保持学习和创造力？
# 2. 相关名词解释
## 2.1 “远程办公”
“远程办公”（Remote Work）是指通过网络、移动设备等远程访问公司的计算机系统和平台的方式进行工作。远程工作的主要优点是可以大大提高工作效率、降低运营成本，并提供更多弹性休假时间。2020年底前半段，许多顶级科技企业已经陆续开启远程办公模式，包括阿里巴巴、腾讯、百度、网易、京东、美团、滴滴等。
## 2.2 “外包”
“外包”是指承接企业或其他第三方公司完成某项任务，并按协议支付费用的一种服务形式。许多企业均已或准备实施“外包”模式，如Uber、Grab、美团外卖、饿了么美食外送等。“外包”模式意味着企业不需要雇佣专门人员来完成工作，而是由第三方承接任务并按预定的报酬付款。通常情况下，承接者需要掌握相关知识和技术，熟练掌握该领域的操作方法及规范，并在合同中承诺保密信息。
## 2.3 “自由职业者”
“自由职业者”（Freelancer），指在无固定工作地点、薪资和待遇要求的情况下，利用自己的专业技能、能力或兴趣，或者与他人共事为生的个人。通常情况下，“自由职业者”是不需任何担保的自主从业者，他们可以根据自己的专业技能或兴趣选择自己擅长的领域。“自由职业者”是远程工作的主要目标群体之一，也是最易被滥用，且风险最大的群体。有些“自由职业者”在其所属行业还未普及，或知识水平低下时，便尝试着进入“外包”的行列。
# 3. 为个人/小型企业设置远程工作方案
## 3.1 为什么要选择远程工作？
首先，为什么要选择远程工作呢？这是一个很难回答的问题。不同的人会有不同的看法。有的人认为，远程工作能够使员工更加聚集、更方便的沟通；有的人则认为，远程工作可以让员工感觉到更安全、舒服；还有的人说，远程工作可以帮助员工解决“断供”的问题。总之，每种人都有自己的想法。选择远程工作还是依照个人喜好，不可强求所有人的都选择远程工作。因此，下面我将分析一个个人选择远程工作的原因：
## 3.2 远程工作环境的好处
由于远程工作可以随时切换地点，所以即使在封闭的环境中也可以得到较好的工作氛围，比如出差、上班、聚会等。在这样的环境中，员工可以完全专注于工作，可以获得和同事之间分享，并且不需要面对面沟通，也不会担心语言或表达的困扰。而且，远程工作还可以节省因人员流动造成的物料消耗及移交的成本，也不用依赖于实验室设备。因此，远程工作的好处主要有以下几点：

1. 更充分利用人力资源：企业可以根据需要派遣远程工作人员，甚至直接雇佣远程工作人员。这是因为人力资源往往是难以调配的资源，远程工作可以有效利用到其余的人力资源，降低企业的运营成本。
2. 提升员工工作能力：远程工作可以让员工独立工作，提升其工作技能，减少不必要的沟通与管理，从而提升员工的工作能力。
3. 降低交通成本：远程工作可以通过互联网连接，使员工可以在任何地方工作。对于一些远距离的员工来说，远程工作可以降低物流成本。
4. 有利于员工健康：远程工作可以让员工远离繁重的工作环境，避免工作压力对身体造成危害，增加员工的免疫系统抵抗力，从而促进员工身心健康。
5. 减少企业拥堵：企业开出的营业执照都是经过审核的，如果开通了远程工作，则可以将额外开支用于设备维修、人力资源培训、网络维护等。这样既保证了企业正常运营，也能减少企业的拥堵程度。
## 3.3 为什么企业会采用远程办公？
企业为什么会采用远程办公呢？我认为，企业采用远程办公主要有以下几点原因：

1. 缩短企业的在线授课时间：许多企业采用远程办公后，可以将在线授课时间缩短，削峰填谷，大幅度减少企业的机房建设费用。这有利于降低企业的运营成本，缩短企业的企业文化建设周期。
2. 降低员工的培训成本：企业可以只派遣少量的远程工作人员进行小额培训，并根据需要调整个人的教学时间。这有利于降低员工的培训成本，提高员工的专业素养，提升员工的职业竞争力。
3. 拓展员工的视野：远程工作允许员工自由探索新的领域，而且不用面对严格的管理制度。这有利于拓宽员工的视野，增强员工的创新能力，扩大企业的影响力。
4. 确保员工的隐私权：远程工作可以帮助企业保护员工的隐私权。在远程工作的模式下，员工可以进行自我监督、自我约束，提升员工的个人安全。
5. 扩展员工的职业生涯：远程工作可以帮助员工获得不同行业的经历，从而开拓员工的职业潜力。
6. 把控员工的生活：远程工作可以帮助员工享受到在户外工作的舒适、惬意，并在满足工作需求的同时维持自己的精神状态。
7. 促进企业内部的创新：远程工作可以为企业内部的创新活动提供一个空间，鼓励员工发挥才干、探索未知，激发员工的创造力，形成相互磋砖的火花。
## 3.4 远程工作的缺陷
但是，远程工作也并非完美无瑕。远程工作虽然有许多优点，但同时也存在一些问题。下面是远程工作的一些缺陷：

1. 跨国界效率差：由于远程工作是在异地进行，因此相比于传统的长途汽车租赁等方式，其效率可能相对较低。因此，跨国界团队协作时，需要对交流工具、文件传输等进行投入，否则可能出现文字处理、表格处理等效率较低的问题。
2. 缺乏协同精神：在远程工作过程中，没有亲密的联系，成员间的沟通容易产生阻碍。这既降低了团队的凝聚力，也会妨碍工作进展。
3. 封闭式环境对习惯养成的影响：由于远程工作是高度孤立、封闭的，因此员工在这些环境中缺乏个人定制，容易导致习惯养成的缺失。
4. 人际关系的变动影响：由于远程工作的特殊性，成员之间的关系变化速度很快，这可能对组织的稳定性和工作效率造成负面影响。
5. 对创新、发现的支持有限：由于员工无法在此频繁切换工作地点，不能做到持续创新、追求最新技术。
6. 需要高度管理才能保障效率：远程工作人员一般不具备专业的管理技能，因此需要高度的管理才能保证效率。
7. 低薪资水平限制员工的创新能力：远程工作有助于缓解员工的职业限制，有利于激活员工的创新能力。但是，由于公司对员工的薪资结构设计不合理，存在低薪资水平的员工无法享受到这种激活。
8. 会造成信息不对称：远程工作将人们之间的沟通隔绝在互联网上，因此容易造成信息不对称，导致团队协作效率下降。
9. 行政化的困境：由于远程工作涉及到员工的大量培训，需要通过招聘、教育、管理等手段进行人员管理，因此不少企业存在管理人员过多、管理复杂、业务流程混乱的情况。
# 4. 自由职业者的远程工作模式是否适合？
## 4.1 为何有些自由职业者选择跳槽？
“跳槽”（Job Switching）是指选择另一份新工作。由于自由职业者一般无法承担长期的日常工作，因此频繁跳槽更为合适。但是，选择跳槽是否合适，关键还需考虑个人的发展方向、个人兴趣爱好、薪酬福利、工作环境、工作内容等因素。如果新工作和旧工作有很大的不同，那么选择跳槽更为顺理成章。但是，自由职业者作为一种资格不受限制的个人，在选择工作时还需结合自身的特质和能力，尤其在比较不确定的时候，更需谨慎，防止错过良机。
## 4.2 没有工作、无收入时的自由职业者怎样才能保持学习和创造力？
没有工作、无收入时的自由职业者（无业游民）应该选择参加一些开源项目，参与社区活动，参加讲座等，以提升自己的综合素质。当然，无论是哪种方式，都需要注意与周围的人和事保持一定距离，以免被别人注意到。另外，无业游民在学习中也应当保持积极心态，不断寻找突破口。
