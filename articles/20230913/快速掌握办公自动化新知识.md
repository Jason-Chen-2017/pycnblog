# 2.计算机视觉与图像识别
## 2.1什么是计算机视觉？
在图像处理、模式识别、机器人、虚拟现实等领域，计算机视觉(Computer Vision)一直处于一个重要的地位。它是指利用计算机技术来研究、创建、应用能够理解、分析和改善图片、视频和声音的能力。

计算机视觉的主要任务包括：
1. 图像采集和传输
2. 图像预处理
3. 对象检测与跟踪
4. 特征描述与匹配
5. 姿态估计与三维重建
6. 场景理解与理解

通常情况下，计算机视觉系统都会由三个组件组成：

1. 感知器(Perceptron): 简单而有效的图像分类器，能够识别各个物体及其位置。

2. 卷积神经网络(Convolutional Neural Network, CNN): 在多个层次上提取图像特征，能够识别复杂且具有丰富多样的对象，如人脸、汽车、行人、狗、猫等。

3. 深度学习(Deep Learning): 在特征提取过程中融合高级特征，能够从原始像素中抽象出具有丰富的语义信息，用于模式识别和决策支持。

## 2.2人脸识别与物体识别
### 2.2.1人脸识别
人脸识别是计算机视觉的一项关键技术，能够实现身份确认、活体检测、安全监控、情感分析、视频监控等功能。

常用的人脸识别算法有如下四种：
1. 基于模板方法的人脸识别: 使用已知的特定面部图像建立一个模型，然后用该模型对比其它输入图像，判断是否属于该模型。
2. 基于SVM的人脸识别: 通过训练样本建立人脸模型，利用SVM算法对输入图像进行分类，判断是否属于某个已知的人脸类别。
3. 基于深度学习的人脸识别: 使用卷积神经网络训练特征，对输入图像进行分类。
4. 基于神经网络的人脸识别: 用深度神经网络训练特征，对输入图像进行分类。

常用的人脸检测器有两种：
1. Haar Cascades: 是一种快速而准确的面部检测器，在实际应用中被广泛使用。
2. Convolutional Neural Networks (CNNs): 是一种深度学习技术，被证明在人脸检测上效果优秀。

### 2.2.2物体识别
物体识别一般是指对现实世界中的各种物体进行识别和分类。

常用的物体识别技术有如下几种：
1. 边缘检测法: 是指检测图像边缘、轮廓等信息，并根据这些信息进行特征描述。
2. 描述子提取法: 是指利用一些图像特征描述子对目标物体进行分类和定位。
3. 区域生长算法: 是指在图像中查找区域连通性并标记物体。
4. 局部特征描述符: 是指描述局部图像区域的特征。
5. 全局特征描述符: 是指描述整幅图像的特征。
6. 密度场：是指密度函数在一定的图像区域上的投影。
7. 方向梯度：是指梯度向量与图像强度之间的关系。

常用的物体检测器有如下几种：
1. Hough变换法: 是一种基于直线的检测算法，它在纵坐标方向上做投票，并在横坐标方向选择平面方位最大的直线段进行连接，得到线段的集合。
2. 霍夫直线变换法: 是一种基于直线的检测算法，其基本思路是先对图像进行二值化处理，然后分割出轮廓，然后使用霍夫变换求得图像中所有直线的曲率、倾斜角度等参数，最后根据这些参数进行筛选和选择。
3. 深度学习方法: 可以通过卷积神经网络进行物体检测。
