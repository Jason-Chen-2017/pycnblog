# 1.背景介绍


近年来,随着数字电子技术的飞速发展、无线网络技术的广泛应用、微波/红外等非线性光学传感器的普及和芯片设计能力的提升,人们越来越关注于芯片的制造工艺研究。由于芯片制造工艺高度依赖于纳米技术和化学加工技术,因此掌握这些知识对工程师的能力要求较高。

在这样的背景下,本文将从“化学机械抛光”和“平坦化”两方面,介绍下文所要阐述的芯片制造相关技术。前者是指利用分子拼接技术将多种金属粒子拧成针状,用低温手法将它们固定在一起的过程。后者是指利用反物质/气体/液相互作用的方法去除冷却液中的杂质、凝固不良层、化学物质等,从而使芯片的体积缩小、导电性能增强、耗能效率提高等。由于这两种技术均具有关键性的影响因素,因此也有很多相关研究和应用。
# 2.核心概念与联系
## （1）化学机械抛光
> 化学机械抛光（Chemical Vapor Deposition, CVD）是利用分子拼接技术将多种金属粒子拧成针状,用低温手法将它们固定在一起的过程。它的基本思想是在制备电路时,把需要的组件用低压高温的流体离心推到待制品表面,这样就形成了一定的材料孔洞,使得可以加强组件之间的连接。通过这样的方式来提高制造电路的效率,降低费用。

## （2）平坦化
> 平坦化（Polishing）是利用反物质/气体/液相互作用的方法去除冷却液中的杂质、凝固不良层、化学物质等,从而使芯片的体积缩小、导电性能增强、耗能效率提高等。其基本原理是利用工业级的蒸馏过程,将硅胶聚乙烯复合物吸收到超薄膜中,再经过反复冲洗,即可实现平坦化,使膜层完全清洁。平坦化后的芯片在尺寸上缩小了许多,而导电性能也得到显著改善。

## （3）CVD与平坦化的关系
CVD与平坦化之间存在着密切的联系和联系。在制备电路时,通过CVD技术将各种组件融入一起,将这些组件固定在一起。这种方法可以提高组件之间的连接,同时也减少了组件之间的摩擦,大大降低了电路板的尺寸和成本。另外,通过平坦化的过程,还可进一步优化芯片的成像性能,降低电路板的耗能功率。因此,在制备电路时,采用平坦化和CVD两种技术既可以提高产品的性价比,又可以保持产品的结构完整性和电路功能的稳定性。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## （1）化学机械抛光过程及原理
### （1）化学机械抛光过程
化学机械抛光（Chemical Vapor Deposition, CVD）是利用分子拼接技术将多种金属粒子拧成针状,用低温手法将它们固定在一起的过程。CVD主要由以下四步组成:

1. 分子预处理：在涂层（颜料）上涂胶状垫层和底胶层，用稀释或加热溶液浸泡一段时间。
2. 物质加载：用洗干净的液体（称作放射剂）和稀释好的金属粉末（称作物质）浸泡在颜料上，从而形成液面。
3. 活性分子投影：加入一个或多个能激活金属原子核的活性原子，把装有物质的液面反复抛光，使原子核开始运动并释放能量。
4. 求解：按照物质的分布情况，将活性原子投射到各个位置，进行晶格化和成型。

### （2）化学机械抛光原理
化学机械抛光的原理与利用聚乙烯膜来控制化学材料的表面张力有关。聚乙烯膜是一种非常耐腐蚀、耐磨损的材料。它能够很好地防止油脂或者各种粘稠液体的侵蚀，并且能够很好地保持稳定。在CVD过程中，将物质和活性原子混合在一起,使得原子核的运动轨迹被限制住,从而形成一个固态分子云。这个固态分子云形成的孔隙可以提供结构支持,从而实现分子的拧紧。

### （3）化学机械抛光的一些具体操作步骤
① 分子预处理：首先在涂层上涂胶状垫层和底胶层。胶状垫层用于降低分子粘结，底胶层用于保证分子混合更充分。然后用稀释或加热溶液浸泡一段时间，使涂层完全溶解。

② 物质加载：在涂层的表面，加入稀释好的金属粉末和液体（称作放射剂）。用一定的速度和角度旋转，使物质被缓慢注入涂层，但不能使金属粉末完全溢出。在放射剂和金属粉末融合之前，先做初步分离，保留晶粒残留物。

③ 活性分子投射：将一个或多个能激活金属原子核的活性原子加到放射剂中，然后用刚性毛丝将活性原子引向放射剂。根据不同的原子，不同的能量可以产生不同的化学效果。例如，在3D打印中，可以使用氮原子的炽镀法来增强分子的反应力；在PCB工艺中，可以使用锂原子的激发法来制造极小宽度的电容；在集成电路设计中，可以使用铝原子的盐洗法来消除电阻负载；在微晶片加工中，可以使用镍原子的激发法来形成负反馈的晶体管。在投射过程中，注意不要使原子核发生反应。

④ 求解：在蒸馏过程中，将负反馈的晶体管加热至700摄氏度以上，经过一定时间后，将晶体管上面的分子释放出来。晶格化和成型之后，即可完成所有必要的操作。

⑤ 使用结果：当硬件工艺和生产管理流程都完成之后，将晶体管注入到电路板上，从而完成电路的制备工作。

## （2）平坦化过程及原理
### （1）平坦化过程
> 平坦化（Polishing）是利用反物质/气体/液相互作用的方法去除冷却液中的杂质、凝固不良层、化学物质等,从而使芯片的体积缩小、导电性能增强、耗能效率提高等。其基本原理是利用工业级的蒸馏过程,将硅胶聚乙烯复合物吸收到超薄膜中,再经过反复冲洗,即可实现平坦化,使膜层完全清洁。平坦化后的芯片在尺寸上缩小了许多,而导电性能也得到显著改善。

### （2）平坦化的原理
平坦化的原理就是利用蒸馏的方法将硅酸钙(Ag2S)和硅酸异锂苯酯(ZnSeC)的聚乙烯复合物蒸馏到硬膜层中。蒸馏过程在高温下进行,从而可以清洁掉膜层中的杂质。蒸馏到硬膜层中的物质会在温度不断升高的情况下氧化掉, 从而使膜层中的硅和软组织得以保留, 其导电性能也得到提高。

### （3）平坦化的具体操作步骤
平坦化的具体操作步骤如下:

1. 将冷却液取出后放置在室温空调下, 使用冷却液埋入底胶层。
2. 将卤水和稀释的酒精放入清洁的机器中, 用电磁铁(或带电喷头)将它们均匀吸入底胶层。
3. 把有氧气体加入到冷却液中, 加入30~40%的氧气。
4. 在电缆上加入Ag2S和ZnSeC的分子,用电磁铁快速将它们嵌入底胶层内,此时Ag2S和ZnSeC已经完全混合。
5. 通过缓慢移动液体来压缩熔点, 使该液体成为一个均匀分布的球状结构。
6. 等待蒸馏到硬膜层, 膜层由金属陶瓷覆盖, 产生大量的磨损和腐蚀, 需要进行反复冲洗以确保膜层的清洁。
7. 当达到完美平滑状态时, 即证明平坦化成功。

 ## （3）CVD与平坦化的关系
 CVD和平坦化之间的关联关系是，两者都是为了解决电子设计难题，但又是两个截然不同的领域。CVD技术利用了分子拼接技术将多种金属粒子拧成针状，用低温手法将它们固定在一起。这一技术在制备电路时可以有效地降低费用, 提高效率, 而且还不需要用到任何特殊的设备。平坦化则更侧重于利用特殊的工艺过程来将特别的金属合金在芯片内部进行拆解。这两种技术都需要长期的实践经验和高端设备才能取得突破。在CVD技术出现之前，人们使用的主要技术是超声波辐射焊接(SSA)和场扫描(EMI)。
