# 1.背景介绍


## 为什么要做软件咨询公司？
很多人认为做软件咨询公司是很傻逼的事情。为什么一定要做软件咨询公司呢？我们可以从以下几个方面来解释这个原因：

1、职业选择问题。如果你是一个已经在职场上被成功打造的资深程序员，那么为什么非得找个软件咨询公司出来谈工作？虽然我不否认你在职场上的成就，但是除了程序员相关的工作外，还有很多领域都需要知识储备和技术能力。如果你只是想提升自己，找到一个有同样志向的另类工作岗位会更有利于你。

2、收益率问题。软件咨询公司的收入通常都比个人稍高，而且这是长期持有的。也就是说，一旦投入了资金，就没有退路。而个人则相对较少获得财富自由的机会。

3、技能学习价值问题。软件开发是一种专业技能，想要赚钱只能靠实力。如果你想提升自己的编程技能，那么最好的办法就是去做软件咨询公司，而不是继续编程。

4、软技能修炼问题。做软件咨询，你其实是在练习你的软技能，包括沟通协作能力、团队精神、沟通表达能力等等。这些软技能是你以后在日常生活中或商业中的不可替代的品质。

5、自由职业问题。软件咨询公司除了帮你提升技术水平外，还能给予你自由的时间和空间。你可以在家里安心工作或者利用假期出去参加培训课程。

## 软件咨询公司该怎么运营？
作为一个软件咨询公司，如何运营才能让自己的服务得到有效的回报？下面是几点建议：

1、给客户提供优质的产品或服务。即使你是一个初创型公司，也应该重视产品的设计、市场调研以及用户体验测试。在刚刚起步的时候，不要把希望放在产品本身，而是应该关注用户体验和解决方案的质量。

2、提升服务质量标准。目前的软件咨询行业仍然处于起步阶段，没有统一的评判标准。但由于技术的快速迭代和变化，不宜完全依赖于专业技能和经验的判断。因此，多采用科学的方法来衡量服务的质量，比如通过A/B测试来评估用户的满意程度、通过反馈环节来改善服务，同时也要适当放宽服务的范畴。

3、兢兢业业地维护客户关系。只要服务的质量和效率不断提高，就需要付出更多的时间和金钱维护客户关系。切忌将客户看得过高，客户关系管理也需要保持良好态度。如此才能构建起信任，让客户满意。

4、注意服务费用。软件咨询往往是收费的业务，而客户并不是总是愿意花钱的。所以，请记住不要为了一分钱就拒绝服务。如果一个服务确实不错，但价格太贵，那还是算了吧。尽量降低服务费用的同时，也不要盲目追求高端定制化的产品。

# 2.核心概念与联系
## 软件咨询公司有哪些核心概念与联系？
1. 咨询领域：软件咨询公司的主要业务活动就是帮助客户进行技术咨询。咨询领域包括编码、数据库、业务流程、系统集成、业务分析、项目管理、ERP、智能助手等。

2. 服务对象：软件咨��公司的目标客户主要是中小型企业、初创型公司和互联网企业。一般来说，客户会希望能够快速获取到有关其所需服务的信息，以便根据自身情况做决策。

3. 技术渠道：软件咨询公司往往通过网络平台、电话、传真、邮件等方式向客户提供各种服务，其中电话咨询、网络咨询等方式占据着较大的份额。另外，还有其他渠道，例如售卖咨询书籍、提供免费咨询服务、举办活动等。

4. 技术能力：客户可以通过咨询人员的实际操作能力了解到所涉及技术的具体细节，并结合自己的实际情况制定合适的解决方案。另外，软件咨询人员也有专门的技术经验，有时候甚至可以帮助客户解决一些技术难题。

5. 技术专长：软件咨询公司的首要任务就是为客户解决技术问题。客户的需求和技术要求不断增加，这也是软件咨询公司能够在竞争激烈的市场中维持长期增长的重要原因。因此，每个公司都会聘请具有不同技术专长的咨询人员。

## 软件咨询公司的定位
目前，软件咨询业已成为行业的热门产业之一，国内外软件咨询公司不计其数。但是，咨询公司的定位一直存在一些问题。下面我将尝试从软件咨询行业的角度出发，探讨一下软件咨询公司的定位应该如何定义？

首先，我们先把软件咨询企业划分为两种类型——定制软件咨询公司和服务型软件咨询公司。

1. 定制软件咨询公司（Customization Consulting）：顾名思义，这种类型的软件咨询公司专注于帮助客户定制符合特定需求的软件。定制软件咨询公司的目标客户通常比较有限，对所提供的服务一般性强，适用于大中型企业。这类软件咨询公司的优点是服务时效性强、价格不高，且由资深工程师主导项目，保证项目质量。当然，它也有缺点，例如项目周期长、项目风险大等。

2. 服务型软件咨询公司（Service-Oriented Consulting）：这类软件咨询公司以“服务”为中心，帮助客户解决其实际问题，而不是仅仅“定制”。它的优点是能够快速响应客户的需求，并提供可靠可信的服务，因为服务对象可能面临着极为复杂的问题。这类软件咨询公司需要能够理解客户的业务背景、业务需求、技术特点、客户偏好等方面的信息。例如，如果客户想要获得某个数据查询的服务，他们可能会寻求相关的咨询。服务型软件咨询公司的目标客户面向所有企业、组织，并且业务高度专业化。

软件咨询公司的定位，就像人的定位一样，没有正确的答案，只有符合自己业务模式的定位才最为准确。如果软件咨询公司作为一个服务型公司，并且服务客户也比较专业化，那就可以按照服务型软件咨询公司的定位来开展业务；如果软件咨询公司作为一个定制软件咨询公司，那就可以按照定制软件咨询公司的定位来开展业务。
