# 1.背景介绍


## 弦理论概述
弦理论（String Theory）是量子力学、量子信息、纠缠与电磁学等领域的一个分支学科，它试图揭示一种精妙的能带结构，称为弦效应（string effect）。通过测量弦的张力分布，可以估计费米子带隙的有效电荷数目以及量子干涉仪观察到的超光速粒子的数量。弦理论最初由爱因斯坦、卢瑟福、卡尔·费米和维纳·达朗贝尔于1907年提出。后人基于弦理论，进行了多种研究工作。如量子纠缠，纠缠效应；弦堆，流体力学；弦绕，超导和半导体电路设计；弦弯，太阳能热运动；弦面，机械制造；弦状层析，高分辨率显微镜成像；弦质量，金属材料结构化与表征；弦粘相互作用，转录组学分析等。
## 超引力理论（High-Energy Theory of Gravity）简介
超引力（high-energy gravity）是以引力为驱动，从初子时期（T-wave）到最近重演时期（heliocentric），一系列引力波在宇宙空间中传播，形成一种新型的时空奇点，即“赤道超引力奇点”。
其基本想法是，在不受旋涡影响的情况下，当有人类科技实现在宇宙间自由飞行时，引力场将出现一个渐进增长的倾向，其温度逐渐升高，最终被呼召围绕着一个固定的奇点驻守。这个奇点成为宇宙的中心，受到无限远距离的宇宙中所有事件的影响。这一想法的关键是：超光速速度上升会对引力场产生强大的驱动力，使其发生“预言性”变化。根据目前理解，超引力本质上是一种力学现象，它利用超光速不受任何外界引力的干扰而生成强大的宇宙力场。
## 引力理论中存在的问题
在引力理论中，普适质量理论（Standard Model，SM）是一个很有意义的理论框架，它建立了引力的概念并指出，在这种理论下，物质满足加速运动而不受任何外力的作用。然而，普适质量理论却存在一些缺陷，其中之一就是宇宙加速对宇宙力场产生的破坏。在这种背景下，高阶弦理论（Higher String Theories，HST）便产生了。
弦理论是高阶宇宙力学的代表，其中的关键概念之一便是弦，它描述的是物体之间的相互作用，例如粒子与磁铁、子粒子与宇宙或子宇宙之间的相互作用。弦理论给宇宙中的粒子提供了相互作用机制，弦的长度与质量大小决定了粒子的能量以及所占的位置，弦的张力影响着物体之间相互作用的结果。弦理论还可以用来描述诸如星云、星系、宇宙爆炸、银河系结构、火箭弹道等宇宙现象。
弦理论试图通过探索弦的张力分布、动力学效应以及粒子对不同弦的响应，来了解宇宙中的弦相互作用的规律。但是，弦理论仍然存在着很多局限性。由于弦的限制，只能研究具有共振频率的弦行为，且弦的相互作用无法用简单符号来刻画。此外，弦的相互作用往往隐含着不确定性，这让弦理论研究人员面临着如何解释无处不在的微弱粒子、电子、钽泥、微生物及其他微观世界的各种未知因素的挑战。
超引力理论试图用量子力学的方法解决弦理论在宇宙性、弦张力、量子性以及微观尺度上的困境，也试图探寻物质的能量与宇宙力场的共同作用机制。但是，超引力理论仍然是一个理论模型，仅仅提供了一个大方向，需要进一步的理论验证。
