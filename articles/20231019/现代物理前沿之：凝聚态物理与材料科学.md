# 1.背景介绍


## 凝聚态物理及其应用领域
超导材料、量子技术、芯片集成电路、超高压技术、半导体新材料开发等新兴的技术领域对超导性材料的研究带来了极大的挑战。物理学家们发现，除了基本的物质属性如导电性、点电性、电导率等外，超导性材料还具有独特的“无尺寸”性质和非金属属性。以核磁共振的方式制备的超导材料可以实现光的快速传递，即使在高温下也可以制成固态的介质；另外，基于超导原理所构建的磁性半导体可以在很短的时间内发生变化，从而形成了一种能够满足要求的金属材料，其性能甚至高于普通的金属材料。通过合成方法，这些材料的结构会比金属稳定得多，更适用于一些重要的应用领域。因此，“超导材料”这一术语已经成为引起人们广泛关注的问题。
由于超导性材料具有高度的自旋性和强度，也容易受到外部环境的影响，因而提出了更为复杂的物理理论。为了对超导材料的各种特性进行正确理解和建模，物理学家们发明了许多理论模型和计算方法，包括密度 Functional Theory (FT)、哈密顿量方法、张量网络方法、耦合能带方法、色散色散关系、量子化学模拟等。其中密度 Functional Theory 在过去几年中逐渐成为超导性材料研究的主流方法，其优势在于可以精确地刻画每个离子的运动学特性和辅助材料的相互作用。然而，密度 Functional Theory 在计算效率上有诸多限制，只能处理一些简单的系统。最近，随着对云计算的推广和超导量子芯片的应用，将密度 Functional Theory 用于真实超导应用变得越来越普遍。
另一个重要的研究领域是新型材料的设计。传统的设计流程是先预测某种材料的理想状态，然后对该状态进行优化，找出最佳的施工参数，再用实际的数据验证设计结果。这种方式耗时费力，且可能出现不准确的预测，无法达到完美设计。与之相反，设计目标驱动的方法能够在初始设计阶段就考虑更多的可能性，缩短设计过程的周期。通过对系统性模型的分析和改进，设计人员可以把注意力集中在材料的结构、行为和能量等方面，而不是仅关注预测阶段的“合理性”。近年来，凝聚态物理的应用越来越多，作为量子材料的基本组成部分，它在传感器、存储设备、智能电路、太阳能电池等领域都有着举足轻重的作用。
