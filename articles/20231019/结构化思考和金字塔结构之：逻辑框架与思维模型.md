# 1.背景介绍


什么是结构化思考？结构化思考就是从复杂的事物中抽象出其本质属性、特征和规律性，并通过对这些属性、特征和规律性的分析和归纳推导出逻辑思路，从而得到新的解决方案或新发现，也就是所谓“以数据说话”。结构化思考在生活中的应用也十分广泛。比如我们要做一件事情，可以先将目标拆解成一个个小任务，然后按照优先级、难易程度和紧急度排列顺序去完成每一个小任务，这样就能较快地完成目标。再如，我们写作文时一般都需要根据主题、观点、关键词、引语等内容逻辑清晰地组织句子。结构化思考是一种抽象能力，能够帮助我们理顺复杂的现实世界、提升工作效率、增加创新能力。因此，掌握结构化思考方法、技巧和技巧是很有必要的。
金字塔结构又称为层次型结构，它是最简单的结构类型。一个金字塔由五层构成：底部的最底层为基础层，上面的一层为主干层，中间各层为 supporting layers 或 supporting strata 。其作用是将复杂的问题或任务分解成相对容易解决的几个层次，使人们只需要关注较高的一层就可以很好地解决问题。金字塔结构作为一种结构化思维方式，被用来指导决策制定和管理，尤其是在企业组织中，用于分析、组织和总结多种信息源。比如，华尔街日报的高管们会把所有信息源整合成一张大网图，再根据权重，按重要性及其关系分类，形成不同角度的洞察视角。金字塔还可以用来进行思维训练、问题解决和科学研究。
结构化思考和金字塔结构是两种截然不同的思维方式，彼此之间存在着天壤之别。但是，它们都能给我们提供有效的工具和方法。下面我将从结构化思考方法、技巧和技巧，以及金字塔结构，探讨结构化思考和金字塔结构之间的联系和区别。
# 2.核心概念与联系
## 2.1 结构化思考方法
结构化思考方法（Structured Thinking Method），英文缩写为 STHM ，是一种适用于多种行业的分析、设计和决策领域的理论方法，该方法主要基于数据驱动和反馈循环的原则，并且高度关注产品或服务的生命周期全过程。其基本思想是：
- 将复杂的现实世界问题转换成简单易懂的数据表述；
- 使用数据驱动的方法来获取信息并分析，而不是凭直觉来判断；
- 充分利用反馈循环机制来修正决策，确保最终结果满足需求。
结构化思考方法的核心理念是“以数据说话”，即分析时要用数据和统计模型来对现象进行评估，再通过研究经验、知识、理论、道德、法律、社会、经济等各种因素综合分析，最后得出明智且合理的结论。STHM 方法还包括一些其它的方法规范，如：
- 概念建模：通过将复杂的实体和现象转变成易于理解和记忆的模式，能够更好地理解其内涵和关系。
- 业务流程建模：将商业活动的各个环节分解成多个阶段，每个阶段都有明确的功能和职责，可以让团队成员更轻松地理解各个阶段间的信息流动。
- 模块化设计：将复杂系统细化为多个模块，每个模块都有明确的输入输出接口和处理流程，可以实现更精准的系统控制。
- 风险评估和不确定性管理：运用概率论、随机分析和贝叶斯公式，能够对系统可能出现的各种问题进行定量评估，并制定相应的应对策略。
## 2.2 金字塔结构
### 2.2.1 定义
金字塔结构（Pyramid scheme）或层次型结构（Level of Analysis Scheme）是一种非常重要的结构形式。金字塔由五层构成：底部的最底层为基础层，上面的一层为主干层，中间各层为 supporting layers 或 supporting strata 。其作用是将复杂的问题或任务分解成相对容易解决的几个层次，使人们只需要关注较高的一层就可以很好地解决问题。
### 2.2.2 基本特点
1. 从下往上看，金字塔的每一层都是比前一层更抽象的。
2. 每一层的名称均以“层”结尾，并分别表示的是层次的深度。
3. 金字塔结构是一种分而治之的思维方法，也就是由问题层面向不同层次逐步简化。
4. 在金字塔结构中，最抽象的层次通常是“问题”层。这个层次是最外围的一个层次，它一般由用户提出来的问题或者业务上的问题等构成。
5. 在金字塔结构中，最里面的层次通常是“实体”层。这一层主要是为了把复杂的问题分解成一个个实体，每个实体都可以单独处理。
6. 在金字塔结构中，最上面的层次通常是“整体”层。这一层由多个实体组成，是解决问题的关键所在。
7. 在金字塔结构中，层次之间的关系是“顺着走”的。即，人们沿着金字塔的层次顺序，由低到高的逐渐解决复杂的问题。
8. 在金字塔结构中，底层通常不具备独立意义。他只是基础设施和底线，并非要达到的目的。
9. 在金字塔结构中，层次之间存在依赖关系，即某一层只能处于前面的某几层，不能直接跳跃过去。
10. 在金字塔结构中，层次之间的关系是紧密联系的。
11. 在金字塔结构中，人们能够从顶层快速看到整体情况，以及最重要的解决方案。
12. 在金字塔结构中，人们可以使用比较来分析各个层次之间的联系，并有针对性地对相关层次进行调整。
