# 1.背景介绍


## 1.1 概念
等离子体（dilute gas，DG）是一种气体，主要由氢、氧气和少量其他气体组成，经过一定的混合物分离后，气体中主要成分是氢气或氧气，称为孤立成分。常用单位是毫升、升和帕。其特点是在不同温度下可以形成不同的物质结晶。在宇宙中含有数量极多的有机物质和微生物，包括星系、行星、恒星、冰层等，这是由大量的不同种类的碳纳米氢化合物组成。因此，DG被称为“星系之眼”，具有广泛的应用领域。
## 1.2 基本特性
1. 基本结构：等离子体由氢、氧气、硅、钛等无机元素组成，有机化学反应的产物，呈现浓缩、金属、晶体的三相结构。
2. 运动状态：等离子体具有三种基本运动状态——摩擦性、液态、气体态。
   * 摩擦性状态：等离子体存在着热力学平衡、随机运动以及非均匀分布的运动，表现为导热膨胀效应、可瞬变性、不稳定性等。在流体力学的研究中，可以用温度场表示等离子体的运动。
   * 液态：等离子体在体积很小时，往往处于液态，此时表现出一种非常松弛的状态。液态的等离子体具有很强的阻尼性，当压力作用时，会产生液滴状颗粒并迅速散去，故液态的等离子体不能承受高于零头的加速度。液态的等离子体具有极弱的可塑性。
   * 气体态：等离子体随温度升高而进入气体态，气体态等离子体具有固态、融性及液氮特征，是高温物态中一个重要组成部分。
3. 分子运动：等离子体中的电子、原子、分子都随温度的升高而发生运动。在低温物态中，电子的运动基本上由磁场控制，原子和分子的运动基本上由微观运动引起。但到了高温物态中，由于热运动的影响，电子的运动趋于有规则地对称分布，原子的运动趋向于无序、随流变动，分子的运动趋于等距分布。
4. 可逆原子爆炸：等离子体中存在着大量的可逆原子。在高温物态中，它们被热辐射逐渐释放出来，爆炸分子对可逆原子进行合成。这种合成过程可能使得可逆原子大量分解或变质。在低温物态中，可逆原子爆炸的作用较小，但是某些分子仍然能够凝聚。
5. 静电场：等离子体的某些温度下存在着局部的静电场，即静电浓度随距离变化的电荷场。静电场的存在会对电子的运动产生不利影响，因为电子运动方向受到静电场的支配，使得电子变得容易受到外部电磁场的干扰而偏移运动。
## 1.3 分类
根据等离子体物理特性的不同，其分为碱性（液态）、溶性（气体态）和超硬度等三类。

1. 碱性等离子体：所谓碱性等离子体，就是指具有独特的各种化学成分，如氯气、乙烯、水素、钠、钼、铊、砷等，这些成分都能使等离子体中的电子更好地运动。

2. 溶性等离子体：所谓溶性等离子体，就是指等离子体能够在一定条件下，表现出气体的性质，这种性质主要表现在两个方面：其一，等离子体可以按照一定的规则形成金属的晶体。例如，在温度和压强的限定范围内，氢气、氧气以及混合气均可通过表面活性剂膏生成氢铊的晶体，可作物理仪器的催化剂，具有广泛的应用价值。另一方面，溶性等离子体还可以受到溶剂的作用，这种作用使等离子体中的电子形成聚集体，形成微尘，促进物质的融合。

3. 超硬度等离子体：所谓超硬度等离子体，就是指等离子体内部含有一种特殊的物质，这种物质具有很强的共振能力，具有不可逆分解、易燃性高等性能，具有很大的热容，可以形成一定的衍射线。
