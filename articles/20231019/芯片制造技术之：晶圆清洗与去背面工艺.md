# 1.背景介绍



电子集成电路(EIC)是高性能集成电路的一种重要形式，其集成电路设备通常由不同的电极、导体等组成，是各种电子元器件的复杂组合。随着电子集成电路的发展，晶圆制造技术也逐渐成为一个独立且重要的研究方向。晶圆制造工艺主要包括圆孔晶体的制造技术、片状晶体的设计、制备、测试及应用。对于生产制造过程中的环节，如清洁、切割、焊接、温度控制等，还有很多精密而复杂的工艺措施需要遵循。

一般来说，制作晶圆时会采用如下的流程：

1. 滤波、干燥、洗涤
2. 清洗
3. 去背面
4. 晶圆切割与焊接
5. 测试和验证

但在实际制作过程中，仍然存在着一些瑕疵，比如尘埃、污染物、氧化物等污染效应等问题。因此，精细化的晶圆制作工艺对于高品质的晶圆的制作至关重要。

晶圆制造的核心在于晶圆的清洗与去背面处理。在一定程度上，这两种工艺可以提升晶圆质量、减少反应过热的可能性，使晶圆不出现静电或者其他故障。除此之外，通过采用精心设计的工艺措施还能降低制造成本，节省生产时间。

# 2.核心概念与联系

## 2.1 概念解析

- 晶圆
  - 晶圆制造是电子集成电路制造中最基本的工艺，是电子元件的集合，其制造主要依靠晶圆相互作用、光照、温度和压力对离子的运动进行控制。晶圆制造技术是指将半导体材料或其他低阻态材料聚合，使其具有特异性的共聚物晶体结构，这种结构能够承载并传递电流，实现电子电路的集成功能。
- 质子骨架
  - 是晶圆表面的核心区域，由质量可观的质子核以及无形的直线状电子带隙所构成，微观结构和相互作用关系密切，是晶圆制造的关键。
- 氮层
  - 在晶圆内部，由于基底的不稳定性和环境影响，产生了多个分支脉络，形成了含氮的氮层。氮层处于晶圆表面，会吸收来自外部介质的气压，使晶圆具有很强的耐磨能力。
  - 蒸馏液氮化合物：将氮化合物（如硝酸、甲醛）浓缩至室温，作为某些化合物的溶剂。
  - 超导电路板：含有两套接头的晶圆可以用来制造超导电路板，其中一套接头将负责电源的工作，另一套接头则负责输入输出信号。
- 前溴层
  - 是晶圆表面在光照作用下的局部分布，一般由晶圆表面的侧壁结构生成，前溴层主要是在光照作用下形成新的质点和新结构，而且前溴层能够吸收大量的红外光谱，所以晶圆表面有多种颜色。
- 后溴层
  - 是晶圆表面在电介质作用下形成的负电荷层，主要受到自旋电场和化学反应的驱动，不同晶圆之间也会发生反应，所以晶圆表面呈现出不同颜色。
- 元件表层
  - 是晶圆内嵌有多个微生物团簇，它们常在晶圆内部构筑起电子膜，使晶圆具备了电传ivity能力。
- 渣子堆
  - 是晶圆表面积累了许多渣子（不完整的晶粒）时，它就会形成一堆，就像堆垃圾一样，通过加工处理，这些渣子堆就可以变成完整的晶圆。
- 角质层
  - 一般是晶圆表面发生明显缺陷，通过铣削等手段，可以切断晶圆表面，形成一个小孔，从而让晶圆变得脆弱。
- 暗层
  - 是晶圆内部渗透出来的多孔结构，晶圆在制造时，会将多种材料聚合，形成一个多层的晶棱。暗层上的突起经常会产生电阻，因此晶圆制造时应该注意避免暗层扩散。
- 功能层
  - 是晶圆表面覆盖的一层金属，具有起辐射作用，由于其作用，晶圆内部的辐射信号可以被反射到外界，所以晶圆的功能层往往经历长期的修复和维护。
  
## 2.2 相关术语

- 材料：锌，铜，银，镍，金，玻璃，石英，金刚石，铝，铀，锡，石墨烯，曼陀罗冷却液等
- 抽丝刀（蒸馏机器）
- 氢氧化镓钢钛（电阻器，隔离器）
- 沙丁胺溶液（去除元件表层的油漆，防止发生渣滓）
- 晶片去除机（连续旋转，快速将晶格破碎成碎片）
- 晶片（晶圆的一个孔径层）
- 转盘（晶圆上的微小作用区域，有助于加速融合和晶圆的稳定）
- 切割片（分离晶格和核芯的工具，能够切断晶圆表面，从而把晶圆切割成两个独立的部分）
- 油漆（水平垂直均匀分布的薄膜，主要用于晶圆表面修饰）
- 沥青（晶圆表面颜料，用于修饰、保护晶圆表面）
- 轧制（将切割片在二维平面上固定住）
- 放电液（通过电极、导体等连接器的力学作用，引起电流通路，将晶圆内的氮化合物与其他各种物质交换出去）
- 漆盒（小型化学试剂，用于杀死某些物质，如氧化物）
- 底蕴材料（矿石、卤水等）
- 过滤器（用于去除感光介质中的杂质，尤其是在晶圆制作过程中，由于制品的厚度较高，需要多次进行过滤，否则可能会影响晶圆质量）
- 切除斜切面（油墨的斜面，通过斜切面剔除油墨，保持晶圆清洁）
- 净化工（用于将晶圆中的油墨等噪声完全清除，并做好预防措施）
- 红外光（通过红外管道捕捉晶圆表面的特征，以便确定晶圆的清洁状况）
- 涂布（晶圆表面的装饰材料）
- 擦拭（将物质擦拭，以促进晶圆的清洁和硬化）
- 喷涂（将涂布粘贴在晶圆表面，以增加晶圆的硬度）
- 冲洗（将晶圆洗干净，然后再用所需涂布覆盖，达到晶圆的最终目的）
- 滤波器（用于将杂质或污染物从晶圆中过滤掉）
- 干燥箱（用于将晶圆中的杂质排出，并使晶圆清洁）
- 清洗（将晶圆表面进行清扫、打蜡、轻轻擦拭，以消除小尘埃、灰尘、氧化物等微生物，提高晶圆的质量）
- 焊接（将电介质结合晶体金属的绝缘性，然后用推土机拉动、向晶圆内部移动，进行焊接）
- 质量比（晶圆与低阻态材料的比值，用于控制晶圆质量）
- 内切圆（在晶圆内部形成一个孔洞，使其更容易进行清洁）
- 通风层（建立在晶圆表面的一层薄膜，并在白天进行气流通入，以消除晶圆内部的雾霾）
- 上摆（通风层的上部，以固定通风层于晶圆表面）
- 下摆（通风层的下部，以固定通风层于晶圆表面）
- 表面坍塌（由于环境因素导致的晶圆表面倒塌）
- 核烟（用某种物质封装，导电的气体，用来释放出晶圆内的所有氮化合物，而不仅仅只是氮）
