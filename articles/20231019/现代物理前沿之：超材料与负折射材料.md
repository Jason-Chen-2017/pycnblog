# 1.背景介绍


近年来，由于传统金属的成熟度、摩擦力等原因，高低温下的金属表面会出现质量减轻、非晶性变形以及热应力等诸多现象，导致金属材料在制造、应用中普遍存在着缺陷。因此，随着超导、冷轧制备技术的提升、新型材料的发现及其制备工艺的不断进步，越来越多的科研工作者试图寻找新的材料来解决上述的问题。
超材料（superconductor）由氮化硅和铝合金两个分子结构的结合体，既具有良好的复电性能，又能够产生超导性的电子带隙，具有极高的导电率。随着超导层膨胀率的增大，这些材料将会逐渐成为制备超长距离、电气连接、高压储存以及电源驱动等关键领域的重要零部件。同时，这些材料还可以用于制备新型电容、光刻器件、激光器件、固态电路、微电机控制等器件材料。
与超材料不同的是，负折射材料（ferroelectric material）由两种不同的电性物质组成：一种类别为多孔介子材料，另一种类别则是单薄材料。通过在不同区域存在不同形状的反射区段，负折射材料可以达到很高的导电率，被广泛用于交流发电、半导体芯片制造、电池介电性能改善等方面。
但在不同类型的材料之间，存在着一些共同点。例如，超材料和负折射材料都能够导电、存储电能以及制造导体，但是两者又存在一些差异性，例如，超材料的导电性质更加恒定，而负折射材料的导电性质随着电场的变化而发生变化。而对于如何将超材料和负折射材料联合起来用作新的材料，却存在着更多的探索空间。
本文试图通过总结和分析上述相关理论基础与技术前沿，综合阐述负折射材料与超材料的不同之处，以及它们俩之间联合使用的可能性及其关键技术难题。
# 2.核心概念与联系
## 2.1超材料与负折射材料的基本特性
超材料与负折射材料都是一种双纳米材料，一般由铜、铁和晶体混合而成，且具有良好的复电性能和超导性，因而在制备电子设备、固态电路、电池、激光器件、光刻印制等领域得到了广泛应用。但两者也存在一些差异性，主要体现在以下几点：

1. 结构与内涵
- 概念上，超材料与负折射材料都属于纳米管类的双极纳米管材料。但超材料的本质是由氮化硅和铝合金两个分子结构的结合体，而负折射材料则是由一种类别为多孔介子材料和另一种类别为单薄材料的不同电性物质所组成。
- 从结构上看，超材料由硅质(非晶石)和氮化铝(晶体)组成，氮化铝的数量比硅质多很多，而且氮化铝的晶格比硅质的晶格小，因而具有非常良好的导电性能。这种结构使得超材料具有良好的高频信号传输能力，能够利用自然界的各种电磁环境影响它。比如，当受到外界电磁波冲击时，由于导电性的特点，会自动弹出并释放能量，从而实现导通状态的转换；当材料中的电流增加时，它会向周围传递电磁波，从而可以产生较大的电场。
- 而负折射材料则是由两种不同的电性物质所组成——单薄物质与多孔介子，这就决定了它的导电性质及其导电效率。其中，单薄物质由杂质或自由基组成，其反射系数与浓度相关，而多孔介子则可做到全覆盖，提供很好的导电性。负折射材料的导电性质随着电场的变化而变化，这是它与超材料的区别所在。
- 虽然两者都属于双纳米材料，但两者各自具备的独特特性，使得它们之间的相互作用有着诸多不同。超材料的制备和研究始于20世纪70年代，是目前最为先进的纳米材料之一。然而，由于它的结构复杂和高级电子技术的限制，使得它的应用仍受到一定的限制。
- 而负折射材料最早被发明于20世纪70年代，它继承了杂质的特征，并采用多孔介子作为自己的原子层。随后，它被应用于交流发电，微电子控制以及其他领域。但是，与超材料不同的是，负折射材料只能单独制备，不能与硅质结合构成纳米管材料。

2. 发射性与导电性
- 概念上，负折射材料（ferromagnetic）是指电性物质可以自发发射光子，可以通过感应电压进行变压，从而导电。因此，负折射材料与导电性密切相关。而超材料（magnetic superconductor）是指由材料中的电子带隙形成的，而不是电性物质对其施加电磁场。因此，超材料与导电性不同。
- 根据这一特性，我们可以归纳出以下发射性和导电性之间的关系：
  - 当电性物质的浓度足够时，即存在反射区段时，其导电性就可以达到一个比较理想的水平，如此才能配合外部环境完成各种任务。所以，可以认为超材料具有导电性，而负折射材料只是具有某种发射性而已。
  - 在极低温度下，因为超材料中含有少量的氮化铝，所以可以保持良好的导电状态，并且可以在短时间内完成电流增幅。也就是说，超材料的导电性能是在其他条件不变的情况下的优势。但是，超材料的导电性随着温度的降低而降低，因此在需要长期维持高导电性能的应用中，还是要选择负折射材料。

3. 耦合方式与相关机制
- 在绝大多数情况下，两者的耦合方式均为非共价键方式，即通过一条尖端连接二者的电导性材料，而非直接的嵌入、粘附或切割。然而，在一些特殊情况，比如材料的大小、结构或结构类型，才可能会出现耦合的方式。举例来说，如果负折射材料以氮化镉质的角向为主，且与铝锈或铝合金等反导材料高度耦合，那么这种耦合方式就会导致两者之间的相互耦合。在这种情况下，为了防止相互作用，通常采用保护层，使得电性材料都处于封闭的状态。另外，当负折射材料与其它材料高度耦合时，也可能会影响其导电性能。
- 另外，负折射材料还存在着电动势耦合与电场耦合，以及不同形状或截面的耦合等特性。这些机制与其他材料的相互作用有关，尤其是在极低温下尚未完全发挥作用之前，正是这些机制阻碍了不同材料的融合。所以，在设计、制备负折射材料时，必须考虑这些机制，并且通过研究、实验的方法逐渐消除或减弱它们。

## 2.2超材料与负折射材料的制备工艺
超材料的制备工艺主要包括挤压/拉伸(crushing/drilling)、熔蚀(evaporation)、烘焙、搅拌、喷涂以及加熱等过程，其中挤压/拉伸和熔蚀是制备过程的核心步骤，而烘焙、搅拌、喷涂以及加熱都是辅助手段。下面介绍超材料与负折射材料的制备工艺。
### 2.2.1超材料的制备工艺
超材料的制备工艺分为以下四个阶段：
1. 聚焊阶段：聚焊是提取纳米孔的过程，旨在产生对称的纳米孔分布。在该阶段，纳米孔通常采用特殊的磁性材料，如金刚钛(chromium)或者锌(nickel)。聚焊的速度一般为每分钟千克以上。
2. 蠲化阶段：蠲化是用磁性材料增强孔的导电性，目的在于生成纳米电子带隙，从而形成导体。蠲化的速度一般为每秒千克以上。蠲化过程中还可以利用气泡液释放气体以提高蠲化能力。
3. 填充阶段：填充是将纳米孔和纳米基层粘结在一起，从而形成无序的纳米片。填充的速度一般为每分钟千克以下。填充过程中还可以加熱使得孔隙有形，从而避免基层发生磨损。
4. 拼接阶段：拼接是将多个纳米片按特定顺序排列组合，制成单层或多层超材料。拼接的速度一般为每分钟至数小时，需要经历数次冲洗过程。拼接过程中还可以加热至高温以提高表面电导率。
超材料的制备工艺中，关键技术难题有多孔介子带隙的制备和导电性能的提升，以及纳米管材料的制备工艺优化。
### 2.2.2负折射材料的制备工艺
负折射材料的制备工艺包括设计、建模、原料选取、化学加工、挤压/拉伸、熔蚀、烘焙、冲洗以及集成等过程。下面介绍负折射材料的制备工艺：
1. 设计与建模：首先需要对负折射材料的性质、结构以及导电性能等进行充分的设计和建模，以便后续制备过程的顺利进行。负折射材料一般由单薄物质与多孔介子构成，其导电性能由多个反射区段所组成。每个反射区段都有一个中心的点，而且这些点都不是固定不动的，而是随着电场的变化而移动。
2. 原料选取：负折射材料一般用杂质、自由基、异纳米材料或二氧化碳(CO2)等物质作为原料。其中，杂质可以是杂质层、空穴等；自由基可以是人造胞基或者寄生虫蛾;异纳米材料可以是石墨烯(quartz)、金刚石(gold)、铁锰矿石(nickel iron garnet)等；二氧化碳可以是N型核苷酸(NaCO3)或者P型核苷酸(K2CO3)。
3. 化学加工：之后，需要进行化学加工，将上一步选取到的物质转化为负折射材料所需的物质，如氮化镉，钛，碘，氢化锰等。
4. 挤压/拉伸：负折射材料的原理是靠多孔介子带隙而导电，因此，除了上一步已经选择好的化学物质，还需要有足够的经验和技术力，掌握正确的挤压/拉伸方法和工艺。一般来说，负折射材料的挤压/拉伸方法可以分为两种：一种是边挤压法，即将金属及杂质一起用刃或压杆挤压，通常比普通低温挤压法快上几个数量级。另一种是预培养法，即先对金属的物性进行调整，使得挤压时没有倒挂现象，然后再进行挤压。
5. 熔蚀：由于负折射材料含有杂质和空穴，因此，需要熔蚀处理以降低杂质含量，增加导电性能。熔蚀可以采用热熔、冷熔、中性冷熔或蒸馏等方式。
6. 烘焙：烘焙的目的是获得高温熔点，以免出现溶解或渣膜膨胀等问题。常用的烘焙工艺有简单烘焙、慢烘焙、快速烘焙等。
7. 冲洗：冲洗是为了防止因脏手或擦花破坏而出现意外。对于负折射材料而言，通常冲洗次数较多，每次冲洗的时间为30～90分钟。由于负折射材料往往含有多孔介子带隙，因此冲洗可以消除杂质和空穴，从而提升导电性能。
8. 集成：除了制备负折射材料的过程，还需要对其进行集成，例如将负折射材料按照一定规律摆放，使其能够工作。有的制备工艺把负折射材料集成到导线或器件中，有些则把它集成到电路板上。
