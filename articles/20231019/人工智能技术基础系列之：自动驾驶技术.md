# 1.背景介绍


自动驾驶（Autonomous Driving，AD），简称AD，是指通过计算机、机器人或其他智能设备，自主控制汽车、飞机等行驶物体的过程。2016年5月1日，美国政府正式批准了百万辆汽车和配套设施的全部上市，国际上率先开放了自动驾驶相关的领域，包括法律、技术、政策、商业模式等方面。

自动驾驶技术的关键技术，如基于图像识别的巡航导航、基于地图构建的路径规划、激光雷达/高德地图/高德路网的结合等，将使得汽车、飞机甚至是船只在复杂环境中快速安全、高效的行驶。据估计，到2025年，全球将拥有超过50%的汽车由自动驾驶驱动，相信随着这一自动驾驶技术的普及，汽车将成为不用坐在驾驶座上、不需要操心车况的低成本、环保、便携的交通工具。

下面，我就以汽车行驶过程中一些重要的技术组件——特征检测、计算机视觉、路径规划、状态估计、控制等进行介绍。希望能够对读者有所帮助。

# 2.核心概念与联系
2.1 感知与决策
自动驾驶涉及到对环境信息的感知，并据此做出决策，这是整个过程的核心。而对于汽车来说，能够感知到的环境主要有以下几类：

① 卫星影像：目前，针对自动驾驶应用，主要采用的是各种卫星提供的实时影像数据。

② 前摄像头图像：用于获取前方道路、侧方道路的信息。

③ LIDAR雷达测距：目前，这一技术正在逐渐被各大厂商掌握，得到广泛应用。

④ IMU信息：这一传感器可以获取车辆四周环境信息，例如，角速度、加速度、陀螺仪数据等。

2.2 立体视觉与深度学习
通过立体视觉的方式，将三维信息投影到二维平面，得到深层次的空间理解。深度学习技术则是利用这一新型的空间信息，进行更加准确的决策。

2.3 目标检测与跟踪
车辆需要根据外部环境信息，确定车辆的位置、方向、速度、距离等信息。在此，可以使用目标检测与跟踪技术，检测出可能出现的目标，并准确的进行追踪。

2.4 规划与控制
当车辆从当前位置驶离、进入某个区域或者遇到障碍物时，如何进行安全停车、避障等规划？如何根据规划结果，完成对车辆的控制，最终实现自动驾驶的目标呢？

这些关键技术，都涉及到多个领域，不同学科的研究人员正在密切关注。例如，计算机视觉技术主要从工程角度探索，目前的最新进展主要集中在目标检测、分类、分割、跟踪等领域；人工智能算法更偏向于理论研究，其研究目的在于找到最佳的自动驾驶算法模型。
