# 1.背景介绍


在现代社会，科技革命带来了从计划经济向市场经济转型、信息化高速发展的迅速进程。许多企业都采用新兴的计算机网络技术来提升竞争力和绩效，制定新的服务标准。为了满足不同业务领域的需求，各大公司不断拓展自己的业务规模和经营范围。这促使传统的电路板、插座等电气设备日渐式微，而这些设备只是过去的物理世界的表象。
随着人类进入智能手机时代，人们期望用更小巧更便携的电子元器件来实现各种功能。因此，电子技术正在成为越来越重要的一环。在过去的几年里，由于移动互联网、物联网、无人驾驶汽车、机器人等技术的兴起，电子学、电机工程、电气工程、电子电路设计、模拟电路设计、嵌入式系统等专业学科也逐渐火热起来。
人工智能（AI）技术也将成为现代科技发展的重要组成部分。基于深度学习、强化学习、集成电路设计、脑机接口、传感器网络等领域的研究，人工智能已经成为行业内最紧迫的技术方向。在此背景下，一门具有完整的电子学知识体系的电子工程学专业可能会成为科技创新、解决实际问题的重要路径。本专业人员通过掌握电路设计、模拟电路设计、数字集成电路设计、信号处理、电子信息工程等多个学科知识，能够帮助企业提升核心竞争力，改善产品质量，降低成本，提升企业服务水平。
因此，本专业人员必须具备较强的电子学基础，对电子工程理论、技术以及应用有全面的理解。这可以作为电路设计、电子元件开发等课程的导向。在本专业学生中，可以以《第一性原理之：电路设计与电子元件》作为学习目标来培养其电路设计、电子元件开发能力，并开展相关项目实践，帮助学生完成电路设计与电子元件相关项目的顺利毕业。
# 2.核心概念与联系
电路是一个单独的模块或组件，通常由电路蜂窝组成，它主要用于传输、控制和存储信息。通常情况下，我们将电子元器件和电路封装在一个整体当中，称为电路板。但这只是电路的一个方面，另一方面还有其他的构成要素，如连接电缆、接地盒、电源、电阻、电容、晶体管、集成电路。简言之，电路是一种电磁设备，由电阻、电容、电源、电子元器件等构成。
## 2.1 基本概念
**电流：**电流是指单位时间内流动的电荷数量。电流的大小取决于电荷的量和所占空间的跨径。单位A表示安培。

**电压：**电压是指单位时间内，电流通过某点所需的平均力。单位V表示伏特。

**电阻率:**电阻率是指在单位时间内，通过电阻产生的功率。单位Ω表示欧姆。

**阻抗:**阻抗是指电阻、电容或电感在两端之间的隔离电势差。阻抗越小，则分压越厉害，反之亦然。单位Ω表示欧姆。

**电阻、电容、电感:**都是介质性材料。其中，电阻是绝缘子，它能够抵御一定程度的电磁干扰，阻止电流流通；电容是蓄电容器，能够保存电能，防止电极承载不了电荷；电感是可以变形的导体，能够感受到磁场，可以导电，也可以被感应电动势吸收。

**电导率:**电导率是指电阻的导通率。通常用S表示，以平方米每公里的度数表示。

**电阻值、电容值:**电阻值反映电阻大小，即将阻抗的数值换算成阻抗值。以Ω为单位。电容值反映电容大小，以μF、mF、F为单位。

**绝缘子、电容、电感:**绝缘子能够抵御一定程度的电磁干扰，阻止电流流通；电容能够储存电能，防止电极承载不了电荷；电感可以变形导电，能够感受磁场，导电，或者被感应电动势吸收。它们三者都可以用来制造电路。

## 2.2 电路组成结构
电路的组成结构分为输入端、输出端和中间部件三个部分。输入端负责接收外部输入电信号，输出端则负责输出电信号。中间部件则包括电阻、电容、电感和电容器等电气装置，负责对电信号进行转换、过滤、传输、控制等。每个电气装置都有自己的特性、功能，可以通过数字技术进行组合配置。一般来说，一个完整的电路可以由多个这样的电气装置构成。

**激励电源:** 是一种电源，用来给电路施加正向电压，产生有序的电流。常用的激励电源有交流电源、直流电源、自控电源、场耦合电源。交流电源可以使用风扇或电磁炮对感应到的杂乱无章的电流进行压降；直流电源只能提供阻力，不能提供正电压；自控电源既可控制电流，又可产生有序电流。场耦合电源利用激波作用生成电流。

**偏置电阻:** 是一种电阻，不改变电压、电流方向的同时，将其与所连接的电阻隔开。偏置电阻的存在可以避免多路信号线路混叠引起的干扰。

**电容:** 电容的作用是存储电能，防止电势冲击导致系统失灵。常用的电容有整流电容、二极管电容、变压器电容等。电容是绝缘子，能够抵御超出其自身额定工作电流的电磁干扰。

**电阻:** 电阻是电的阻力。电阻的阻力会根据电压、电流和其他因素发生变化。常用的电阻有直流电阻、变流电阻、双绕组电阻、真空电阻。

**电感:** 电感的作用是传递电磁能量，进行感应电动势吸收。它能够感知和导电，所以可以用来做电路中焊盘的作用。常用的电感有电偶极子、电容触点、无线电感应元件等。

**电抗:** 电抗就是阻止电流流动的一切东西。对于绝缘子而言，电抗等于电阻；对于电感而言，电抗等于电感值；对于电容而言，电抗等于电容值。

**电容触点:** 在电容中插入一个微弱电流，可以令电流流过该电容而感受到电容的保护作用。

**电路腔:** 电路腔是电路外壳的下层部分，主要用来包裹电源、晶体管、负极、电容器等。

**负极、电容器:** 负极是电流的源头，负极里面通常有印刷电路板、晶体管组、电压表、示波器、测量仪器等。负极常用的材料有铜、铝、玻璃、锌、金属电极、聚氯乙烯电极等。电容器有蓄电池、太阳能电池、酒精电池、锂电池等。
