# 1.背景介绍


## 1.1 定义
大数据智能决策系统，英文名称 Big Data Decision Making System (BDMS)，是基于海量数据的复杂业务环境、高维度信息的分析处理能力及其特有的复杂决策需求，进行科学精准的决策制定、资源分配及风险控制等的一体化综合解决方案。它的功能包括：优化资源配置、提升决策效率、降低决策成本、提升决策质量、降低不确定性、增强决策信心、改善组织运营效率、实时反应市场变化、减少服务滞后、降低管理成本。
## 1.2 发展阶段
大数据智能决策系统已经进入了多元化的发展阶段。目前，BDMS 在发展过程中主要经历了三种类型：
* **传统决策系统**：即没有加入大数据特性的传统决策制定系统。在这种类型的决策系统中，决策数据往往采集的是静态的历史数据或现状，并依赖人的经验和直觉进行决策。由于决策过程缺乏信息价值和可扩展性，因此往往存在人为因素、经验偏差、决策效率低下、处理复杂任务困难等一系列弊端。
* **技术驱动型决策系统**：在技术驱动型决策系统中，加入大数据技术可以使决策更加智能、精准、实时。它通过收集海量的数据、进行数据分析、建立模型预测，从而对数据做出更好的决策。然而，这种类型的决策系统需要有较强的算法、编程能力、建模能力、决策库建设能力等技能。同时，还需要掌握分布式系统架构、数据库设计、高性能计算、网络通信、数据安全等高级专业知识。
* **大数据智能决策系统**：它将传统决策制定系统与技术驱动型决策系统相结合，通过大数据收集、分析、预测等方式，对历史数据进行预处理，结合多种场景的数据、因素、模型，进行有效的决策指导。在实现智能决策的同时，它还要具备跨部门协作、跨平台运行、高度弹性、容错处理等优秀的系统架构设计能力。
## 1.3 BDMS作用
BDMS 的主要作用如下所示：
### （1）优化资源配置
在传统决策制定系统中，资源配置往往受到组织内部人员经验、组织流程、决策权威以及决策模式等因素的制约。因此，传统决策制定往往存在如下问题：
* 各单位或个人专长不同，可能导致决策效率低下；
* 决策结果存在相互矛盾，存在“过度猜测”现象；
* 决策结果存在不确定性，存在“一刀切”现象；
* 投入决策资源过少，导致资源利用率低下；
* 决策成本过高，无法满足决策效益。

BDMS 通过算法和模型，可以自动识别业务中的关键节点（如贸易、投资、财务），为每个节点找到最佳的配置方案，从而优化资源配置。此外，在不同业务环节之间引入协同机制，可以有效地整合各方资源，减少决策过程中的冲突和矛盾，提升决策效率。
### （2）提升决策效率
在传统决策制定系统中，决策过程往往非常漫长，且耗费大量的人力、物力、财力。此外，在决策过程中容易出现决策偏差、错误判断、忽略相关因素、重复决策等问题。

BDMS 可以快速、准确、高效地分析业务数据、组织结构和现状，输出决策建议，并实时反馈市场情况。此外，采用大数据分析技术，可以发现业务中的模式和规律，形成决策模型，提升决策效率。在合理使用人力资源，扩大投入范围的前提下，可以缩短决策时间，提升决策效率。
### （3）降低决策成本
在传统决策制定系统中，决策资源的投入往往占到了整个决策项目的很大比例，因此，每一次决策都需要占用大量的时间和精力。

BDMS 使用机器学习方法和模型，可以分析历史数据、重点领域数据和动态数据，对信息价值进行评估，制定精准的决策。并且，通过协同工作，可以把多方的信息、资源共享于一处，提升决策资源共享效率，降低决策成本。另外，可以充分利用信息技术的力量，实时收集数据，缩短决策周期，提升决策速度和质量。
### （4）提升决策质量
在传统决策制定系统中，决策结果往往存在偏差、不一致性、失灵等问题。这些问题的产生，主要原因是人们通常没有充分认识信息价值的重要性，以及信息系统、决策过程及工作效率的限制。

BDMS 有助于分析业务数据，捕捉信息价值的真正含义，发现信息流动中的模式和规律，制定科学的决策标准和规则。在与外部合作方、团队成员建立密切联系的基础上，可以有效地减轻决策者的压力，提升决策的透明度、责任感、尊重性。
### （5）降低不确定性
在传统决策制定系统中，决策者往往无法精确预测所有的事件，因此，预测结果存在不确定性。

BDMS 使用数据分析的方法，对历史数据进行分析和处理，找出潜在的不确定因素，制定策略和计划，提升决策的可靠性、准确性、鲁棒性。同时，利用计算机模拟实验技术，生成多个假设方案，使得决策者能够在不确定性下获得更多的参考意见。
### （6）增强决策信心
在传统决策制定系统中，决策往往被视为“看起来像随机”的过程，决策结果具有不确定性，无法得到充分验证。

BDMS 可提供关于企业决策真实性、可靠性的可信度评估，并提供可靠的决策依据。另外，通过数据分析及结果确认，可将决策者引入新的思路，增强他们的决策信心和决断力。
### （7）改善组织运营效率
在传统决策制定系统中，组织的运营效率往往受到许多因素的制约，比如决策的时间、资源投入、决策质量、过程文档等。

BDMS 提供一套完整的流程，包括决策制定、执行、评价、持续改进四个阶段，以促进组织的健康发展、产品/服务的推广与销售，并降低决策风险。
### （8）实时反应市场变化
在传统决策制定系统中，决策者通常缺乏实时的市场信息。

BDMS 根据实时市场数据，对决策进行调整，实时反映市场的发展趋势。例如，根据运输业、零售业和金融业的发展情况，动态调整零售商业策划、市场营销策略，提升零售业的竞争力。
### （9）减少服务滞后
在传统决策制定系统中，决策过程往往存在着长期性、持续性和单一性。因此，在决策发生变更时，往往需要长时间的调整和总体的评估，才能达到满意的效果。

BDMS 可以实时监控业务情况，根据预期目标、市场状况以及内部策略调整，提升决策的灵活性、敏捷性和适应性。
### （10）降低管理成本
在传统决策制定系统中，管理决策的成本往往占到了整个组织运营成本的很大比例。

BDMS 在决策制定、执行和管理三个层面，均进行优化，降低管理决策的成本。例如，通过一体化管理，把决策、执行、评价、持续改进等环节的管理人才集成到一体，提高决策的透明度、可追溯性和可操作性。另外，可以根据实际情况，设置“黑箱”决策和条件决策，使决策变得更科学、精细化、可控。
## 1.4 BDMS典型应用
1. 风险管理与优化
风险管理和管理优化的目的是为了降低风险，促进客户满意度，提高组织整体的利润率。在此背景下，BDMS 可用于分析业务数据、行业报告、调查研究等资料，提炼风险评估、风险评价、管理决策建议等方面的共性，为风险管理提供决策依据。

2. 工厂布局优化
生产管理中的工厂布局优化，是为了提升产品/服务质量和生产效率，优化组织资源的分配和管理方式，促进生产过程的协同和减少单一部门的控制力度。此类应用主要包括仓库、物流、生产设备等多种环节，BDMS 可帮助分析现状、未来趋势，并制定管理决策，提升组织的整体效率。

3. 汽车零部件供应链管理
汽车零部件供应链管理主要面临一个复杂的供应链网络和巨大的订单需求，为了保证产品质量和效率，需要大量的人力、物力、财力投入，因此，该领域的BDMS 可应用在供应链管理、供应商管理、订单管理、成本管控等多个环节，为公司发展提供决策依据。

4. 电信网络资源分配与优化
电信网络资源分配与优化是电信网络管理的一个重要组成部分，在现代化过程中，资源分配和管理需要考虑多方利益、多元化网络、多种业务等各种因素。BDMS 可用来分析当前网络状态、客户需求、用户体验，制定优化方案，并实施，提升运营效率。

5. 数据中心建设与维护
数据中心建设与维护旨在优化数据中心的硬件、软件、网络等资源，提高整个数据中心的利用率，减小成本。在这样的背景下，BDMS 可以用来分析数据中心的现状、技术要求、客户服务水平、投资回报，制定优化建议，并实施，降低运营成本。
