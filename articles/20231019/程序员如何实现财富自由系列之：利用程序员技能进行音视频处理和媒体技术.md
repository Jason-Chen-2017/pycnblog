## 2.1 图像识别
图像识别（Image Recognition）是指利用计算机算法从图像或视频中提取关键信息，并对其进行分类、识别，确定图像内容的一项技术。图像识别通常分为两个子任务：特征提取与匹配，以及分类器训练与预测。
### 2.1.1 图像特征提取
图像特征提取（Image Feature Extraction）是指从图像或者视频中提取其特征的过程，包括图像的颜色、空间、纹理等方面信息的提取。不同的图像特征提取方法会对图像的处理速度和识别性能有不同的影响。目前主流的方法有基于像素的特征（Pixel-based Features），基于深度学习的特征（Deep Learning Features），基于区域的特征（Region-based Features）。
#### 2.1.1.1 基于像素的特征
基于像素的特征方法是根据每个像素点的亮度、色彩、明暗、对比度等因素计算得到的特征值。这种方法的缺点是对于灰度图像来说，没有颜色信息，只能识别出灰色或者黑白图像，但是对于较为复杂的场景，它还是可以取得不错的识别效果。

OpenCV 提供了 cv2.cornerHarris() 函数用来检测图像中的角点。
```python
import cv2

gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
corners = cv2.cornerHarris(gray, 7, 5, 0.1)

# result is dilated for marking the corners, not important
corners = cv2.dilate(corners, None)

img[corners > 0.01*corners.max()] = [0, 0, 255] # draw red circles around corners
cv2.imshow("Corners", img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
#### 2.1.1.2 基于深度学习的特征
基于深度学习的特征（Deep Learning Features）是基于卷积神经网络（Convolutional Neural Networks，CNNs）等神经网络结构，通过训练模型识别图像特征。在早期的时候，基于 CNN 的图像特征提取模型取得了很好的成果，但是它的内存占用比较大，并且计算量也比较大。现在，有些公司开始转向轻量级的模型来减少计算量。

Google 提出的 MobileNet 模型，就是一种基于深度学习的特征提取方法。
