# 1.背景介绍


在电子制品行业，常常涉及到两极化问题。即物理性材料和电子性材料的融合，会引起电容、电阻、电感等电子元件的增加。由于绝缘介质（如玻璃或塑料）所释放出的自由基分子，以及基底分子对电子的摩擦作用，使得电子与材料的混合带来的电场高低不平衡现象，称之为两极化。
而在芯片领域中，就存在着多个需要通过晶圆掺杂工艺混合器件的情况，如何保证混合器件的两极化问题得到有效解决，成为一个重要的问题。由于芯片制造过程中的复杂性，使得进行完全的两极化分析非常困难。但是，如果仅仅依赖传统的方法来分析芯片的两极化情况的话，往往会出现以下几个问题：
* 首先，通常情况下，我们无法获取到所有需要用到的物料的实物图片，这就使得我们分析其两极化情况变得十分困难。
* 其次，由于混合工艺的复杂性，对于一些具有特征性的物质，比如氢原子、氨基酸等，其两极化情况却可以单独成立。因此，仅仅依靠分析不同分子的两极化程度并不能完全反映两极化的全貌。
* 最后，由于芯片制造过程中使用的硅粒质量很高，这就导致当硅基离开模具时，通常无法发现完整的2D晶状结构图。只能观察到非常微弱的硅原子模样的晶体，所以分析两极化形态仍然需要借助于计算机辅助工具。
综上所述，如何更好地分析芯片的两极化情况是一个值得研究的课题。为了更加精确地探索芯片的真实特性，实现可靠的设备性能，提升用户体验，当前人们越来越多地把重点放在这一方面。

要想解决这个问题，必须要从机械臂、计算机辅助工具、化学方法和热力学方法三个角度来考虑。其中，机械臂的应用，就是为了能够精确地获取晶圆结构图上的信息，并计算出各种分子的二极阈值，以此来达到两极化判断。而计算机辅助工具则用于提取数据并进行处理，例如进行三维立体图像的拼接，从而获得各种信息的直观展示。热力学方法则主要用于探究晶圆结构的本质属性，包括晶体内的能级分布、带隙分布和局部的对流层。而化学方法则是在晶体内加入气相并进行气化，通过分析产生的气团和气相互作的结果来分析各个分子的两极化状态。下面，我们将基于这些技术理论，进行详细地阐述两极化问题。
# 2.核心概念与联系
## 2.1 概念介绍
### 2.1.1 热力学
热力学是研究物体的性质和运动规律的一门科学，主要关注物体内部和外部的物质以及物体与外界之间的相互作用。热力学的基本定律主要有以下四条：

① 冷却定律：当温度降低时，物体内的熵也随之降低；当温度升高时，物体内的熵也随之升高；

② 结合律：当两个物体处于一定温度时，它们之间的相互作用不会改变他们的相对运动距离；

③ 内能定律：任何形式的物质都有一个等温恒定能量，可以视为它作为独立系统存在时的能量；

④ 热运动定律：外加一定温度的热力流向平衡，并且转化为平衡运动的速度。

### 2.1.2 相空间分布
在热力学中，相空间分布，也被称为焓流分布，是指宏观上不同温度下的物体表面的热量分布形态。相空间分布随温度的变化而发生变化，是由于两种原因：

1. 热量分布方式的不同：不同的热源具有不同的热量分布方式。如石英属于孤立热源，而气体则有趋势性，具有均匀热力导向的特点；

2. 分子结构的不同：不同分子在不同温度下的构型可能不同。如在低温时，硫酸盐和水分子可能有相同的结构，但到了高温，二者的结构就会出现明显差异。

### 2.1.3 气相分离
气相分离，又称为静电气相分离，指在低温下将气体从相空间中分离出来，得到一组相互正相关且平衡的多普勒效应。

在固体物体或液体物质中，只要满足一定条件，就可以保持其恒温状态。如果物体表面的某些地方处于一种特殊的状态，称为自然对流层，即由外部驱动而生成的热能流入。在低温下，这种热能和外加的空气混合在一起，形成了多普勒效应，使其分离出来，得到一组相互正相关且平衡的多普勒效应，称之为多普勒能级。低温情况下的多普勒能级受到各种因素的影响，比如形成它的原子核、水分子、氢原子、电子等，因而不同物质的多普勒能级形态也不同。

### 2.1.4 化学气相沉积
在进行化学气相沉积实验时，必须制备充足的离心室、熔接设备、离心锤、溴化吸盘以及气体等。待装置稳定后，将可沉积物质倒入离心锤，靠近使离心室气体与待沉积物质的结合成为酸性混合物，经过溴化吸盘将液体转化为气体，并利用气相分离法分离出该气体中的气团。然后，用化学干扰剂(去除氢原子的吸收剂)将气团与经过溴化吸盘反应出的混合物分离，使得沉积物质的熔点降低到相空间低端，直到沉积物质与离心室气体完全分离为二气相。

根据沉积物质的性质和多普勒能级分布的不同，分为几种情况：

1. 可对溶性物质进行多普勒能级的两极化分析：水、甲醛、乙醇、苯、丁烯、非酸性金属、钠、砾等；
2. 可对半酸性物质进行多普勒能级的两极化分析：碱性金属、双氧水、氢氧化钠等；
3. 不可对半酸性物质进行多普勒能级的两极化分析：硝酸铵、三氟铊、磷酸铁、氯化钨、镉等。

### 2.1.5 热氧化
热氧化是指在热力学的作用下，通过物理、化学、生物学以及生物信息学手段，逐渐缩小、消失、变成气的各种原子质量，称为“多重瞬间”。热氧化过程的最初阶段，无量纲化、密度推动、热容升高等过程皆可成为气候。通常情况下，这类氢、氮的作用较为复杂，是理解其他元素共振行为的关键之一。

### 2.1.6 液相蒸馏
液相蒸馏(Liquid-Phase Evaporation)，也叫液相退火，是利用空气以液态形式送进离心容器中的一块物质，物质吸附于液相与液相之间的气体之间，在这些气相间随温度的升高而转化为液相，再使物质进入液相和液相之间，成为湿相。最终物质以湿相的形式退出离心容器，混合物在物体表面形成类似晶格结构的形态。

液相蒸馏实验中，一般采用平面蒸馏法，即在离心室前方预先架设锅，等待物料流入，然后将物料投放于平面锅内，用锅口将物料送进，这样可使物料在离心室内形成气孔，防止物料被气体吸附而蒸发掉，因此，与在平行方向进行蒸馏相比，平面蒸馏节省了很多时间。另外，由于离心室中的气体较少，通常还可以在离心室中直接调整温度，可以增强物料在相空间的平衡性。因此，对于那些耗费时间、费力的杂质物质，用液相蒸馏的方法就可以获得快速、简单的多普勒能级分析结果。
