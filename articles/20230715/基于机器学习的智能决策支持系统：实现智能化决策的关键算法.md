## 智能决策支持系统（Artificial Intelligence Decision Support System）简称 AIDSS，是指通过计算机程序、机器学习等技术，用信息系统的方法和工具，帮助用户做出更有效、更精准的决策。AIDSS 是一种新型的决策支持技术，它可以对多种复杂的决策流程、决策过程进行实时分析、评价和预测，并提供决策建议和自动化决策制定的方案，帮助企业在重大决策时更加高效地做出决定，提升企业绩效。
## 为什么要开发智能决策支持系统？
目前，由于互联网、云计算、移动互联网、大数据、人工智能等新的技术革命带来的高度信息化和碎片化，传统的决策方式在面对各种复杂的决策场景中变得束手无策。例如，如果想知道某一天的股市行情，可能需要花上几天时间研究并分析全球顶尖财经机构发布的数据，而对于一些比较简单的决策，比如在超市选购物品，就根本没有足够的信息来做出正确的选择。因此，如何通过高效率的计算机程序和数据处理，更好地解决复杂且无法预料的决策问题，成为了当前热门的技术方向之一。
## AIDSS 的优势
### 1.灵活性强
AIDSS 可以根据业务特点，灵活应用到不同的决策领域，包括金融、保险、人事管理、生产制造、销售服务、广告营销、供应链管理等多个领域。同时，也可作为辅助工具，提升个人或团队决策效率。
### 2.高效率
AIDSS 使用机器学习方法，能对海量数据进行快速分析，找到数据模式及其关联关系，从而提升决策效率，降低决策成本。此外，AIDSS 通过大数据、云计算等技术平台，可以帮助企业集中存储和分析数据，降低数据采集和管理成本，节省了大量的人力、财力和时间。
### 3.精确性高
AIDSS 采用多种机器学习算法，结合大量数据，可以精确地识别复杂的决策因素，找出数据的驱动力，指导人工智能的决策行为。它还通过模型评估、参数调优等方法，保证决策结果的可靠性和有效性。
### 4.可扩展性强
AIDSS 可根据业务规模及其他条件调整，满足用户需求，使其更加符合国际标准、法律法规、社会规范和组织目标。同时，也可作为整体服务的组成部分，帮助企业降低成本、提升产品ivity、优化营运水平。
### 5.成本较低
AIDSS 可以将复杂且耗时的决策流程自动化，降低了人力、财力、时间成本，进一步提升了效率和能力。此外，企业也可以获得收益，因为 AIDSS 减少了信息获取、数据处理、决策的依赖、知识积累的时间，直接产生了价值。
# 2.基本概念术语说明
## 数据集 Data Set
数据集是指用于训练、测试或分析模型的数据集合，其形式通常包括数据元素和属性。每个数据集都有一个目标变量，即希望预测或分类的输出变量，以及描述各个变量之间的关系的一些属性。数据集通常由不同来源的样本组成，这些样本一般按照一定规则进行抽象化处理，如按照时间、空间或属性进行划分，然后转换成易于理解的形式，比如表格、图形或矩阵等。
## 特征 Feature
特征是指对一个或多个观察点或对象所具备的某些方面或特征。特征通常表示一个对象的某种特点或差异。例如，房屋的大小、所在位置、户型、建筑风格等都是房屋的特征。
## 属性 Attribute
属性是指用来描绘事物的特征、属性、特征、品质或状态的陈述性名词。例如，“北京大学”是一个属性，“美丽”则是一个特征。在数据库中，属性是指一张表中的一列或多列。
## 类 Class
类是指具有相同属性或特征的事物的集合。例如，一班同学或学生就是一个班级，某件事物的属性一样，那么它们就是相同类的事物。在分类树中，类是指某个节点下的子节点所表示的内容。
## 属性值 Attribute Value
属性值是指一个事物在特定属性或特征上的取值，属性值一般可以取多种值。例如，某个属性值可能是“男”，“女”，“老年”，“幼儿”，或者其他取值。
## 样本 Sample
样本是指数据集的一个实例，即一个或一组客观事实或对象，用于训练或测试模型。样本可能包括输入特征值和输出标签，用于训练模型。
## 样本点 Observation/Instance
样本点是指数据集中代表性的个体或事件，代表着整个数据集的某种共同点。样本点一般表示为坐标轴上的一点或标记点。
## 预测 Target Variable/Output/Label
预测是指模型对给定输入变量的结果。预测的结果一般与实际情况存在偏差，预测值的准确性受到数据的影响。例如，在预测股票价格时，我们通常会得到一个估计值，但这个估计值往往不能反映真实的股价波动。
## 模型 Model
模型是对给定数据建立的一系列假设，用于对未知或新数据进行推测。模型中的假设对数据进行了归纳总结，并且能够预测某些结果或输出变量。例如，线性回归模型假设每一个输入变量之间是线性相关的，可以利用输入变量的值来估计输出变量的值。
## 代价函数 Cost Function
代价函数是用来衡量模型预测值与实际值之间的差距，并用来确定模型权重和偏置值。它定义了一个损失函数，越小则表示模型越准确。
## 目标变量 Input Variables/Attributes
目标变量是指用于训练或测试模型的变量，也叫做输入变量。它们通常对应着样本中的某些属性，用于描述数据，并帮助模型进行预测。例如，对于房屋价格预测问题，目标变量可能是房屋面积、房屋类型、所在城市、房屋朝向等。
## 参数 Parameters
参数是指模型中的权重或阈值，可以通过调节参数来调整模型的性能。例如，在线性回归模型中，参数包括截距项（bias term）、回归系数或斜率。
