# 2.基本概念术语说明
## 2.1 什么是物联网？
物联网(IoT)是由网络连接的物体、设备及其周边的一系列信息和服务的集合。它是利用Internet进行通信、数据交换、计算处理、智能控制的系统。其主要特点是数据采集、传输和处理都直接或间接地与现实世界紧密联系，具有“高度智能化”、“自动化”、“动态”特征。
## 2.2 物联网相关技术
### 2.2.1 MQTT协议
MQTT是一个轻量级的发布/订阅消息传输协议，用于对物联网中各种设备之间的数据流动进行管理。它是为低带宽和不稳定的网络环境设计的，但仍然能够提供可靠的服务质量。MQTT协议与TCP/IP协议族内其他成员关系密切，经过长期演进，它成为物联网设备之间通讯的事实标准。
### 2.2.2 小型单片机作为边缘节点
由于物联网边缘计算资源受限，因此，需要用微控制器作为物联网边缘节点。它们可以运行在小型功耗设备上，且功能简单易用。ARM Cortex-M MCU通常被选用，它们的运算速度快、内存容量小，同时也具有很高的抗电磁干扰能力和安全性。比如：NXP LPC1769、Microchip PIC16F1787、ST STM32F103C8、TI TM4C123GH6PM等都是满足要求的MCU。
### 2.2.3 HTTP协议
HTTP协议是用于分布式、协作式和超文本文档的传输协议，它由请求方法、URL、状态码、头部字段、内容等几个部分组成。它是一种灵活的、无状态的、自描述的协议。它支持文件上传、下载、创建新资源、或Web页面更新等操作。
### 2.2.4 图数据库数据库GraphDB
图数据库是一种非关系型数据库，它将数据组织成一个有向图结构。它提供了针对复杂查询的高性能查询引擎，可以处理海量的关联数据。它是一种适合于分析和模型化复杂网络数据的数据库。例如Neo4j、Infinite Graph等都是符合要求的图数据库。
### 2.2.5 智能服务器软件ServerLess
ServerLess是一种不需要预先购买服务器的“服务托管”模式，云服务商在用户的基础设施（如计算、存储和网络）上仅提供基础设施的租用费，然后，开发者可以根据需要部署自己的应用程序代码，而无需关心基础设施的维护。ServerLess的优点是降低了云计算平台的运营成本和IT支出，而且开发者只需要关注应用逻辑开发，不必考虑底层基础设施的实现。AWS Lambda、Google Functions、Azure Functions等云平台都提供了ServerLess的支持。
## 2.3 物联网相关产品和解决方案
### 2.3.1 穿戴式传感器模块Ultrasonic Sensor Module
Ultrasonic Sensor Module是一种可以测距的传感器模块，可以使用超声波的方式测量距离，它的检测范围大约为0～150cm，误差在毫米级别。它可以安装在智能手表、机器人、汽车等任何需要测距的地方。Ultrasonic Sensor Module包含了主机主控MCU、传感器、接口电路等软硬件部件。
### 2.3.2 聚类机器人采集云
聚类机器人采集云（PCACloud）是采用聚类机器人技术进行数据的采集和分析的云服务平台。用户可以将PCACloud与聚类机器人一起安装到智能工厂、仓库、运输等场所。聚类机器人采集云可以提升企业效率、节省设备运维成本、降低设备投入成本。PCACloud目前提供两种类型的聚类机器人方案：场景智能分拣机器人、货物收集机器人。
### 2.3.3 基于Arm Cortex-M的嵌入式IoT开发板OpenRTK
OpenRTK是基于国产STM32F429IGTx-MD的嵌入式开发板，具备丰富的传感器、显示屏、板载Wi-Fi、蓝牙、GPS等接口。它搭载了开源的RTOS系统（RT-Thread），使得用户可以快速、方便地实现物联网终端产品的设计。OpenRTK支持Arm GCC编译器，拥有丰富的外设驱动库和示例工程，包括传感器、驱动、显示屏、传感算法、系统、网络、UI等，可广泛应用于物联网行业。
### 2.3.4 AIOT平台基于DuerOS的智能安防系统
DuerOS是一款基于华为百度边缘计算平台的智能安防系统。DuerOS采用面向对象编程语言Java开发，其功能特性包括：物联网平台支持全协议接入；具有AI语音助手、智能窗帘、呼叫中心、监控系统、门禁系统、安防设备等智能化设备；具有大数据分析、报警系统、远程运维、数据可视化等功能。DuerOS还具有完善的安全、可靠、稳定、可伸缩、可扩展、以及互联互通能力。
