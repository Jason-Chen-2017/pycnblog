机器学习（Machine Learning）是人工智能领域的一个重要方向，是指计算机利用数据编程的方式从数据中提取知识，并运用这些知识对新的数据进行预测和分析，最终实现一些自动化的决策和行为。

而深度学习（Deep Learning）就是一种机器学习的子领域，它是通过建立多层神经网络将输入信号转化为输出信号。深度学习模型由输入层、隐藏层和输出层组成，中间还有多个隐藏层。每一层都包括多个节点或神经元。隐藏层的存在使得深度学习模型能够处理复杂的数据和输入，输出结果也比较准确。深度学习的高性能带来了很多创新性的应用。

如今，深度学习已成为许多行业的重要研究热点。人们越来越关注移动互联网、物流、金融、医疗等各个领域的AI应用，以及它们如何运用深度学习技术突破巨大的发展空间。

# 2.基本概念术语说明

1. 特征工程（Feature Engineering）：特征工程是在数据预处理过程中，基于业务理解和相关经验，选择、组合、转换、删除或增加特征数据，生成新的、有效的信息用于建模训练和预测。特征工程是整个机器学习流程中的不可或缺的一环，其目的在于帮助机器学习算法更好地适应特定问题，提升模型的效果。

2. 样本（Sample）：样本是用来训练和测试模型的数据集合。一般来说，样本数量越多，训练出的模型就越精确。

3. 属性（Attribute）：属性可以简单理解为样本的特征或维度。例如，图像中的像素点可以作为属性，文本数据中的单词或字符可以作为属性。

4. 标记（Label）：标记是一个目标变量，通常表示样本的分类标签或回归值。例如，对于垃圾邮件识别任务，“垃圾”、“正常”等标签都是标记。

5. 数据集（Dataset）：数据集是用来训练和评估模型的数据集合。数据集一般包括训练样本和测试样本。

6. 模型（Model）：模型是一个能够拟合数据的函数或映射。它能够从数据中抽象出规律，并对新数据做出预测和判断。

7. 损失函数（Loss Function）：损失函数衡量模型在预测和实际标记之间的差距，用于评估模型的优劣。

8. 优化器（Optimizer）：优化器用于更新模型的参数以最小化损失函数。

9. 测试误差（Test Error）：测试误差是指模型在测试数据集上的误差率。

10. 过拟合（Overfitting）：过拟合是指模型在训练数据上表现良好，但泛化能力弱导致在测试数据上的表现较差的现象。

11. 欠拟合（Underfitting）：欠拟合是指模型在训练数据和开发数据上表现很差，无法正确拟合训练数据，只能在测试数据上的表现较差。

## 算法一：逻辑回归（Logistic Regression）

逻辑回归又叫Logit回归，是一种线性回归模型，但是它的输出不是线性的，而是一个S形曲线。S形曲线上的任一点都对应着某个标签的可能性大小，因此可以用于二分类问题。与线性回归不同的是，逻辑回归没有显著性检验，所以有时候会出现欠拟合的情况。

## 算法二：支持向量机（Support Vector Machine）

支持向量机（Support Vector Machine，简称SVM）是一种监督学习方法，它主要用于二类分类问题。SVM通过求解最大边距约束下的最优化问题，寻找一个超平面（Hyperplane）将数据分割成不同的区域。

SVM最大的特点就是核技巧，它可以在低维空间实现非线性分类，并且核技巧可以有效解决维度灵活性问题。核技巧包括径向基函数（Radial Basis Functions，RBF）、多项式核、字符串核等。

### 模型描述：

SVM定义了一个凸二次范数函数Φ(x)，然后通过选择不同的核函数K(x,x')，将Φ(x)映射到希尔伯特空间H上。再通过求解优化问题max Θ(Φ)，s.t., yi(Φ(xi)) >= 1 for i = 1 to m，找到一个线性超平面，其权重θ=(w,b)。

![image](https://pic3.zhimg.com/v2-8eaab0ba69feac8d7ff7f702012b4a06_b.png)

其中，Φ(x)为特征映射，K(x,x')为核函数，θ为超平面的权重，yi(Φ(xi))为样本xi到超平面的距离。

### 操作步骤：

1. 导入所需模块；

2. 创建数据集，并划分为训练集和测试集；

3. 用SVM算法初始化模型参数；

4. 通过迭代算法不断更新模型参数，直到模型收敛；

5. 在测试集上计算模型的准确率。

SVM模型的KKT条件是：

1. 如果 δ(x) > 0 ，则 G(x) ≥ 0 。
2. 如果 δ(x) < 0 ，则 H(x) ≤ 0 。
3. 如果 δ(x) = 0 ，则 G(x) = H(x) = 0 。

如果满足KKT条件，则得到最优解。如果不满足KKT条件，则需要添加更多限制条件，使得得到的解有更好的性质。
