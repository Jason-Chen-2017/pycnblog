# 2. 基本概念术语说明
## 2.1 NLP简介
自然语言处理（Natural Language Processing，NLP），亦称文本分析、语音识别与理解、机器翻译和多媒体信息处理，是研究、开发计算机科学技术的一门重要分支。它使得电脑像人类一样可以理解和生成文字、听觉、视觉、触觉等各种输入信息，从而使计算机具有与人们沟通的方式和技巧一致的能力。

自然语言处理的任务通常包括：
- 文本预处理：即对文本进行初步处理，如去除标点符号、词干化、过滤停用词、同义词替换等；
- 关键术语抽取：即从文本中识别出最重要的、有代表意义的词汇，如名词、动词、形容词、副词等；
- 情感分析：即判断一个语句的情感倾向，如积极或消极；
- 智能问答系统：即让机器像人类一样回答复杂的问题，如对话系统、聊天机器人等；
- 文本分类：即根据文本的结构、语法、意思等对文本进行自动分类，如主题分类、文档分类、评论分类等；
- 信息检索：即根据用户查询词检索相关文档，如搜索引擎、信息推荐系统等；
- 机器翻译：即将一种语言的文本转换成另一种语言的文本，如英语到中文的翻译；
- 文本摘要：即从长段文本中抽取主要信息，如总结、反映中心思想等。

## 2.2 数据集介绍
我们使用的金融文本数据集来自国际期货交易所OpenStreetMap数据。该数据集共包含7万条带标签的每日微博信息。原始微博包含了大量的新闻评论、投票、活动信息等，部分评论甚至会涉及到大额交易价格，需要注意的是，这些数据并没有提供足够的训练数据。因此，为了更好地训练模型，我们随机从原始微博中选取一定比例的数据作为训练集，剩余数据作为测试集。我们分别将训练集划分为70%的训练集、30%的验证集，将验证集划分为10%的验证集、20%的交叉验证集，并从整个训练集中采样出10%的无标签数据用于模型无监督学习。以下是数据集简介：

1. 数据数量：7,111;
2. 数据来源：OpenStreetMap数据;
3. 数据类型：微博;
4. 数据内容：文本信息，每条微博都带有相应的标签信息;
5. 注释种类：4种标签（购买、交易信息、公司动态、评论）。
