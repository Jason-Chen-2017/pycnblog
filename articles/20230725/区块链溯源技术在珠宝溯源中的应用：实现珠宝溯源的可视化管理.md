# 2.基本概念术语说明
## “区块链”
区块链（Blockchain）是由加密技术和计算机网络互相协同工作完成的一项全新技术。它使数据可以被有效地记录、共享、传递和验证。“区块链”最初于2009年诞生于比特币的社区中，但是目前已被广泛应用于各种行业。“区块链”是一种分布式数据库系统，其主要特征是去中心化、高容错性、公开透明、不可篡改、并能够提供匿名性。其主要运用场景包括：数字货币、供应链金融、智能合约、医疗健康信息、存证存管、欧盟反垃圾邮件、智能投顾等。

## 2.1 珠宝溯源
珠宝的分类、价值观念、制造方法等一直在不断变化。如何追踪历史上珠宝的起源、流通和传播，帮助购物者做出决策并认识到珠宝的价值观念、风格和历史沉淀，就成为了当前珠宝界的热点话题之一。

## 2.2 块链接节点
区块链的底层结构是一个由多个节点相互连接的数据块，每个节点负责存储、维护、验证和发送其他节点的数据。这些节点中的许多机器运行着“区块链”软件，他们通过在本地计算生成新的区块，同时向其它节点同步数据，从而保持整个网络的高度同步。每个节点保存着完整的账本，能够识别并验证所有交易，并且可以用来创建个人账户、存储数字资产、进行交易等。区块链技术的关键是分布式系统下的共识机制，即对信息的整体性和准确性达成共识。当一个矿工将新的区块添加到链条中时，其他节点才会认可该区块。

## 2.3 IPFS(InterPlanetary File System)
IPFS（星际文件系统），一种旨在构建完全分布式的协议，用于存储和传输可能被分割成片段的文件。它还支持对文件内容的加密、版本控制、文件的分层存储、内容寻址以及内容发现。因此，IPFS可以用于在网络中存储和分享各种类型的静态和动态文件。
