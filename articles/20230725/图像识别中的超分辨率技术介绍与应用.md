# 1.简介
         
图像超分辨率(Super Resolution,SR)技术解决的是低分辨率图像放大到高分辨率图像的问题。在移动端设备上的图像处理、导航系统、数字相机等领域都需要用到SR技术。目前国内外主流的SR算法有基于傅立叶变换的算法、卷积神经网络的算法等。本文将对当前SR算法进行综述并介绍一些常用的算法模型。


# 2.超分辨率相关术语及概念
- 分辨率(resolution): 指图像在每一个像素点上所包含的信息量大小。通常来说，分辨率越高则图像细节越丰富，反之则图像细节越少。例如，现代照相机能够拍摄的照片分辨率从很高的193万枚/s（0.35毫米），到如今的16.7MP（约400万枚/s）。
- 普通图像采集: 是指原始图像数据的采集。这种方式的缺陷是图像信息量较低，无法体现出真实场景中的各种细节信息。
- 合成图像: 指对图像的不同分辨率区域之间进行混合、重构而得出的新图。例如，用于显示设备上的显示效果。
- 模糊: 在图像的放大过程中会产生模糊，这是由于采用了某些插值方法导致的。而这些插值方法可能导致噪声严重。
- JPEG压缩: 是一种常用的图像编码方式，它采用不同的压缩比例来表示低质量和高质量的图像。通过JPEG压缩算法，可以有效地降低图像的质量损失。但是同时也增加了压缩后图像的大小。因此，在超分辨率任务中，往往不使用JPEG压缩。
- 超分辨率: 是指通过某种技术将低分辨率图像放大到高分辨率图像。当图像的分辨率太低时，就需要用到SR技术。其目的是提升图像质量、提高图像的清晰度。目前，比较流行的SR算法有基于傅里叶变换的算法、卷积神经网络的算法等。
- 回退增强: 是指采用低分辨率图像进行超分辨率的方法。通过添加低分辨率图像的几何形状、颜色等信息，来抵消上采样过程产生的模糊效应。
- 扩张(expansion): 是指采用某种插值方法或掩膜方法将图像放大到更高分辨率。一般而言，当图像的分辨率越高时，扩张越多。
- 下采样(downsampling): 是指缩小图像大小，尺寸减半或三分之一，即图像分辨率降低。在SR任务中，下采样通常采用双线性插值法或其他更加高级的算法。
- 上采样(upsampling): 是指通过某种插值方法或者直接采样的方式将图像从低分辨率图像上恢复到高分辨率图像。
- 超分辨率误差: 表示超分辨率算法的准确性。一般认为，无论是基于傅里叶变换的算法还是卷积神经网络的算法，其超分辨率误差都是不可避免的。但可以通过优化超参数来降低误差。
