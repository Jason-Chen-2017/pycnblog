# 1.简介
         
现实世界中，很多场景都需要用到向量空间的概念。例如，在二维平面上，点p=(x,y)和点q=(u,v)，都可以表示成一个向量a=(x-u,y-v)。这样，一条直线l(t)=ap+bt+c=k，也就可以表示成两个向量：la=(ap,bp),lb=(cq,dq)。这样，就构建了由矢量组成的向量空间R^n(n=2或n=3)。在高维空间中，也是一样，例如，对于三维空间R^m(m>3)，就可以有三维矢量空间。

实际上，如果只是想将向量形式简化，并不需要完全理解它的含义。相反，理解一些基本的向量运算规则和几何意义，就会非常有帮助。例如，向量加法，向量减法，向量乘法，向量除法，以及向量的范数（模）、内积、外积、张成等等。这就是本文要讲的内容。

# 2.基础概念和术语
## 2.1 定义

首先，我们定义一下向量空间（Vector Space）的概念。

设V是一个非空集合，如果对任意的元素a,b∈V，满足如下三个条件，则称V是一个向量空间：

1. (加法) V中存在元素之间的所有可能的加法组合: a+b ∈ V
2. (乘法) V中任意元素a,b,c∈V有且仅有一个零元元素0，使得V中任意的元素a*b ∈ V
3. (反射律) 对所有元素a∈V，满足a+(-a) = 0 

如果还满足以下四个条件，称V为赋范向量空间：

1. 有限维向量空间：如果向量空间V的维数为n，则称V为n维向量空间。
2. 基：V中某个集合B={v1,v2,...,vn}称作V的基，其中vi∈V是关于Vi的单位向量，并且满足：
   - |Vi| = 1 
   - Vi * Vj = 0 (i≠j) 
3. 内积：V上定义了一个从U(m维空间)到R上的双线性映射Φ(v,w)，叫做Vi·Vj，记作 vi·wj = Φ(vi,vj)，其中vi, vj∈V，u, w∈U(m维空间)。Φ(v,w)满足如下的性质：
   - Φ(v,w)+Φ(w,v) = Φ(v+w,v) = Φ(0,v) = Φ(v,0) = 0
   - Φ(v,v) > 0，如果u·v = 0的话；否则，Φ(u,v) < ||u||*||v||，u·v > 0，则Φ(u,v) > 0
4. 子空间：假设V是一个向量空间，W<V是一个其子集，且满足如下两个条件：
   - W是V的一个真子空间(即W属于V，且W包含V中所有的向量)
   - W中没有零向量（不包含任何零向量）。
   
## 2.2 相关概念

以下是与向量空间密切相关的概念：

### 2.2.1 基变换
给定一个基(Basis)B={e_1, e_2,..., e_n}, 定义一个从V到其子空间W的基变换Φ(B)从B到W。

$$\phi_{e_1}= \lambda_1 e_1,\quad \phi_{e_2}= \lambda_2 e_2,\quad...,\quad \phi_{e_n}= \lambda_n e_n $$

其中$\{\lambda_1, \lambda_2,..., \lambda_n\}$ 是基B的特征值。

特别地，如果V=W，且V是赋范向量空间，则基B是其正交基(Orthogonal Basis)。

### 2.2.2 基 decomposition 
将一个向量放在一个子空间W中，可以表示成某个基的线性组合：$v=\sum_{i=1}^{n}\alpha_ie_i$, 其中$\alpha_i\in R$ 是系数，ei是标准正交基的基向量。

将V分解为多个子空间构成的族Q={Q_1, Q_2,..., Q_r}，每个Q_i都是向量空间。通过基变换可以把V表示成这些子空间之和。

当V=Q_1时，Q只有Q_1; 当V=Q_1 + Q_2 时，Q有Q_1和Q_2。

### 2.2.3 矩阵的特征值与特征向量

对于方阵A，如果存在λ(特征值)和对应的列向量v(特征向量)，使得Av=λv，则称λ为A的特征值，v为A的特征向量。

在线性代数中，向量空间可以看成是实数域上的向量空间。因此，线性变换f：V→W，f(v)是一个向量，它的值是一个向量空间内的元素。

对于线性方程组Ax=b, 如果向量空间V和W上都有相同的行数，而且该方程组有唯一解，那么就说存在向量w∈W使得Aw=bw。

可以通过求解特征值与特征向量来得到此解。

### 2.2.4 列空间与行空间

对于线性方程组Ax=b, 求解Ax=b所生成的矩阵A，如果A的秩为n，即A的行数等于列数，则可知n个列向量线性无关。因为Ax=b中取值不定，所以n个列向量不是独立的，而是充满整个矩阵的某些线性相关的部分，它们就形成了列空间C(A)。

若A的秩小于n，则可知A的秩为m，则m个行向量线性相关。因而Ax=b中取值不定，所以m个行向量不是独立的，而是协同作用，它们就形成了行空间C(A’)。

通常，我们希望找出线性方程组Ax=b的一个最小二乘解，因此需要计算得到列空间或行空间。但是，由于求解Ax=b的问题是NP难题，目前并不存在有效的算法来快速计算列空间或行空间。

### 2.2.5 空间的维度
如果一个向量空间V有一个基{e_1, e_2,..., e_n}, 则V的维数(Dimension)为n。

如果一个向量空间V不是赋范的，则其维数也为n。

如果一个向量空间V不是真子空间，则其维数不大于原空间的维数。
