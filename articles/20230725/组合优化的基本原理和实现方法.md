# 1.简介
         
组合优化（combinatorial optimization）是指将给定目标函数和约束条件转化成一个多项式或二次方程问题，从而求解最优解或者对策指导。其在计算机领域、运筹学、经济学、控制论等多个领域有广泛应用。
基于历史、统计学和线性代数的基础知识，组合优化可以解决很多实际问题，包括运输问题、制造问题、工程调度问题等。
现实世界中存在各种复杂的组合优化问题，但我们只考虑其中一种类型的组合优化——最小生成树问题（Minimum Spanning Tree Problem）。

最小生成树问题（MST），又称最小权重生成树问题（MWGT)，是连接所有顶点的无向图G=(V,E)中的顶点的集合T所构成的一棵树，其中每个边的权值（代价）之和最小。
对于一个连通无向图G，通常希望找到一个边数目和顶点数目相同的子图，使得这个子图的所有顶点之间都有路径相连（即它是一个树结构）。该子图的权重（代价）总和被定义为最小生成树（MST）的权重（代价）。

通过引入一些假设条件，可以将最小生成树问题分解成许多子问题。一般情况下，最小生成树问题都可以通过归纳法得到一个递推公式。但为了保证公式正确性、易于理解和实施，需要作出一些数学上的限制和简化。

# 2. 基本概念及术语的介绍
## 2.1 定义
### （1）组合优化

组合优化（combinatorial optimization）是指将给定目标函数和约束条件转化成一个多项式或二次方器问题，从而求解最优解或者对策指导。其在计算机领域、运筹学、经济学、控制论等多个领域有广泛应用。

典型的组合优化问题包括：

1. 优化求解约束最少的变量个数和最大化目标函数值的组合问题；
2. 对某些目标函数的取值进行最优调整，使其达到全局最优值的问题；
3. 求解某个范围内所有可能的解的集合，并确定每种解的重要性和概率分布的问题；
4. 分配资源到不同的任务上以满足特定约束条件的问题。

例如：

1. 搭建一幢公寓；
2. 选址房屋；
3. 进行生产物料的分配；
4. 为公司进行人才选拔。

### （2）约束条件与目标函数

约束条件是指对待决系数的界限要求。如购买产品时的购买数量、工作人员的工时、汽车的油耗限制。

目标函数是指在满足约束条件下达到目标的指标。如利润最大化、完成项目所需时间最小化、达到某种性能水平的工人比例、风险最小化等。

## 2.2 生成树问题（Minimum Spanning Tree Problem, MSP）

“生成树”表示一个连通无向图的边集合，它连接了所有顶点，而且连接着图中任意两顶点之间的最短距离。生成树问题就是找出一棵包含所有的顶点的生成树，且它的权值和最小。

### （1）相关概念

#### （1）生成树

生成树是指连接了所有顶点的无向连通图的子图，它包含所有顶点，而且包含了图中任意两个顶点之间的最短距离。

生成树的定义：一个连通无向图G=(V,E)的生成树是指连接了所有顶点的边集合T=(T1,...,Tn), 其中V∈N是图G的顶点集，E∈E(G)是图G的边集。

生成树与边权重最小化问题密切相关，因为给定无向图G，可以在O(|E|log|E|)的时间内求解出它的一棵生成树。当且仅当图G中没有负权回路时，它才有一个唯一的最小生成树。

#### （2）树的概念

树是一种无向连通图，其顶点都是互不相交的，边都是无向边。一个非空无向连通图G=(V,E)叫做树，如果它至少含有一个顶点，并且G除了自身以外没有其他不相交的子集，那么它一定是一棵树。

## 2.3 相关术语的阐释

### （1）顶点

一个图中的顶点代表着图中的一个结点或节点。在最小生成树问题中，顶点是图中需要进行连接的点。

### （2）边

一条边代表着图中的一条连接两个节点的连接线。在最小生成树问题中，边是连接图中的各个顶点的线段。

### （3）路径

路径是一条由顶点序列组成的可行线段。在最小生成树问题中，路径是连接两个顶点的简单线段。

### （4）回路

回路是一条首尾相接的路径，它的第一个顶点等于最后一个顶点。在最小生成树问题中，回路是不能包含在生成树中的环形边。

### （5）代价

代价是指把边连接起来的代价。在最小生成树问题中，边的代价就是边连接两个顶点之间的距离。

### （6）连通性

连通性是指顶点间存在一条路径，即两顶点可以直接连接起来。在最小生成树问题中，图G必须是连通的。
