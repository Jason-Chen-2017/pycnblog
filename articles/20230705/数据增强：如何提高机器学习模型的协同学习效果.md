# 2. 技术原理及概念

## 2.1. 基本概念解释

数据增强是一种通过对原始数据进行变换，增加数据的多样性，从而提高模型性能的技术手段。数据增强可以分为两类：正则化和非正则化数据增强。

正则化数据增强：通过对原始数据进行惩罚，引入噪声，使得模型更加鲁棒。常见的正则化方法有 L1 正则化和 L2 正则化。

非正则化数据增强：通过对原始数据进行变换，增加数据的多样性，从而提高模型性能。常见的非正则化方法有 min-max 变换、center-scale 变换和 normalization 变换等。

2.2.1 正则化数据增强

正则化数据增强的主要思路是对原始数据进行惩罚，引入噪声。具体操作步骤如下：

1. 对原始数据进行预处理，如缺失值填充、异常值处理等。
2. 对预处理后的数据进行标签分类，根据具体问题设定不同类别。
3. 对每个数据点，根据预设的惩罚因子，引入噪声。常见的惩罚因子有 epsilon、theta 和 r。
4. 保存处理后的数据。
