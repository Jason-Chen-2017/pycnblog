# 2. 技术原理及概念

## 2.1. 基本概念解释

区块链（Blockchain）是一种去中心化的分布式数据库，具有不可篡改性、匿名性和可追溯性等特点。区块链技术最初是为比特币（Bitcoin）的设计，但现在已应用于许多领域。

区块链溯源技术是在区块链基础上，通过将产品或服务的生产、加工、存储、运输等环节的数据上链，实现对产品或服务的可追溯性。

### 2.2.1 区块链原理

区块链是一种去中心化的分布式数据库，具有不可篡改、匿名性和可追溯性等特点。区块链数据由区块构成，每个区块都包含了一定的数据和一些元数据（如时间戳、前一区块的哈希值等）。每个区块都连接到前一个区块，形成了一个不可篡改的链条。

import json

def get_blockchain_info(address):
    url = "https://api.blockchain.info/v2/address/" + address
    response = requests.get(url)
    data = json.loads(response.content)
    return data
