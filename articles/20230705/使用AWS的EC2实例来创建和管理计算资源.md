### 7. 附录：常见问题与解答

7.1. Q: VPC 是什么？

A: VPC 是一种虚拟化网络服务，允许用户创建一个私有网络环境。用户可以在 VPC 中创建子网、安全组、子网掩码等，从而实现网络隔离。

7.2. Q: AWS 网络协议栈是什么？

A: AWS 网络协议栈包括数据包路由、协议头解析、TCP/IP 协议栈等，它们共同构成了 AWS 网络通信的基础。

7.3. Q: EC2 实例类型有哪些？

A: EC2 实例类型包括 t2.micro、t2.small、t2.large、t2.xlarge 和 r1.micro，每种实例类型都有不同的计算能力。

7.4. Q: 如何创建一个 EC2 实例？

A: 创建 EC2 实例需要进行以下步骤：
1) 登录 AWS 控制台
2) 导航到 EC2 控制台
3) 选择实例类型
4) 配置实例参数
5) 创建实例

7.5. Q: 如何删除一个 EC2 实例？

A: 删除 EC2 实例需要进行以下步骤：
1) 登录 AWS 控制台
2) 导航到 EC2 控制台
3) 选择实例
4) 删除实例
5) 确认删除
