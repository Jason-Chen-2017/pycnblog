### 案例一：随机数生成

假设我们需要生成大量的随机整数，用于某些高性能计算任务。使用传统的方法生成随机整数需要很长的时间。而使用缓存算法，可以在生成随机整数时节省大量的时间。
```python
import random

def random_integer(max_num):
    result = 0
    while True:
        try:
            result = random.randint(0, max_num)
        except:
            result = 0
        return result

# 计算大量随机整数的总和
sum_random_integer = 0
for i in range(1000000):
    sum_random_integer += random_integer(1000)

print("计算大量随机整数的总和:", sum_random_integer)
```
