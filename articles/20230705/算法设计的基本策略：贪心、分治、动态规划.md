# 2. 技术原理及概念

## 2.1. 基本概念解释

贪心算法、分治算法和动态规划算法是三种常用的算法设计策略。它们各自具有独特的优势和适用场景，在实际应用中具有广泛的应用价值。

贪心算法：

贪心算法是一种在动态规划问题中具有简单、高效的解法。它利用动态规划的自底向上的特点，每次都选择当前状态下最优解。因此，贪心算法的时间复杂度为 O(n^2)，其中 n 为状态数。

分治算法：

分治算法是一种将大问题拆分成小问题求解的算法。它将问题分解为子问题，并递归地解决子问题，最终将子问题的解合并成原问题的解。因此，分治算法的时间复杂度为 O(nlogn)。

动态规划算法：

动态规划算法是一种在满足状态转移方程的情况下，利用系统状态转移概率，从初始状态计算到终止状态的算法。它的时间复杂度为O(nlogn)，其中 n为状态数。
