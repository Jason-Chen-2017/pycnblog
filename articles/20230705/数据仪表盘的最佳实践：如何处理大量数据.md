import pandas as pd

# 数据清洗
df = pd.read_csv('data.csv')
df = df.dropna()  # 去除缺失值

df['gpa'] = df['gpa'] * 1.2  # 计算GPA的1.2倍

# 数据展示
df.plot.bar()  # 条形图
df.plot.line()  # 折线图

### 4.3. 核心代码实现

```python
import pandas as pd
import matplotlib.pyplot as plt

# 数据清洗
df = pd.read_csv('data.csv')
df = df.dropna()  # 去除缺失值
df = df[df['age'] > 18]  # 去除重复值

# 数据处理
df = df[df.gpa > 90]  # 选择GPA大于90的行
df['gpa'] = df['gpa'] * 1.2  # 计算GPA的1.2倍

# 数据展示
df.plot.bar()  # 条形图
df.plot.line()  # 折线图
df.plot.饼图  # 饼图
df.plot.polar()  # 圆图

# 打印数据
print(df.head())
print(df.info())

# 打印图表
plt.show()
```

在实现数据仪表盘的过程中，需要使用 SQL、PYTHON 和 matplotlib 等技术。

首先，使用 SQL 语言从数据库中读取数据，并使用 pandas 库对数据进行处理，去除缺失值和重复值，选择 GPA 大于 90 的行，并计算 GPA 的 1.2 倍。

然后，使用 matplotlib 库中的图表类型来展示数据，包括条形图、折线图、饼图和圆图等。

最后，使用 print 和 show 函数来打印数据和图表，以便更好地了解数据和图表。
