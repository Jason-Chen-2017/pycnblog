在实现应用场景时，需要使用 Python 语言进行编程，使用机器学习库（如 scikit-learn）进行机器学习算法的实现。以下是一个简单的机器学习核心代码实现：
```python
# 导入所需库
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier

# 读取数据
df = pd.read_csv('production_data.csv')

# 将数据分为训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(df[['quality_score','status']], df['fault_rate'], test_size=0.2, random_state=0)

# 特征提取
X = df[['fault_rate','status']].values

# 模型训练
knn = KNeighborsClassifier(n_neighbors=5)
knn.fit(X_train, y_train)

# 预测
y_pred = knn.predict(X_test)

# 输出预测结果
print('预测结果：')
print(y_pred)
```
