### 2.2.1. 无监督聚类算法

无监督聚类算法是一种基于数据相似性度的聚类算法，通过找到数据集中相似的数据点来确定聚类的中心点。其中，K-Means算法是最常见的无监督聚类算法之一，它的核心思想是迭代计算每个数据点到所属聚类中心点的距离，并更新聚类中心点的位置。

### 2.2.2. 有监督聚类算法

有监督聚类算法是一种基于数据分类的聚类算法，通过已知数据点的类别信息，将数据点分配到不同的类别中。其中，支持向量机（SVM）和K近邻（KNN）是最常见的有监督聚类算法之一。

```python
import numpy as np
from keras.models import Sequential
from keras.layers import Dense
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

# 加载Iris数据集
iris = load_iris()

# 将数据集拆分为训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(iris.data, iris.target, test_size=0.2)

# 创建SVM聚类器
svm = Sequential()

# 将数据点划分到不同的类别中
svm.fit(X_train, y_train)

# 测试数据点所属的类别
test_data = np.array([[6.5], [2.5], [7.5], [1.5], [8.5], [10.5], [12.5], [14.5], [15.5]])
print(svm.predict(test_data))
```
