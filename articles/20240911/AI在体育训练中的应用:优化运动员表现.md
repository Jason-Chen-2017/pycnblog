#### 二、算法编程题库及答案解析

**1. 题目：** 编写一个Python程序，读取一个文本文件，提取出其中所有的运动员姓名，并存储在一个列表中。

**答案：**  
```python
def extract_athletes(file_path):
    athletes = []
    with open(file_path, 'r') as f:
        for line in f:
            if '运动员姓名' in line:
                athlete_name = line.split(' ')[-1]
                athletes.append(athlete_name)
    return athletes

file_path = 'athletes.txt'
athletes = extract_athletes(file_path)
print(athletes)
```

**解析：** 这个程序首先定义了一个函数 `extract_athletes`，用于读取文本文件并提取出运动员姓名。然后，通过调用这个函数，将运动员姓名存储在一个列表中。

**2. 题目：** 编写一个Python程序，计算给定运动员在不同训练项目中的平均得分。

**答案：**  
```python
def calculate_average_score(scores):
    return sum(scores) / len(scores)

athlete_scores = [85, 90, 88, 92]
average_score = calculate_average_score(athlete_scores)
print("平均得分：", average_score)
```

**解析：** 这个程序首先定义了一个函数 `calculate_average_score`，用于计算给定列表中数字的平均值。然后，通过调用这个函数，计算运动员的平均得分。
