#### 面试题库

**1. 如何在并发编程中安全地读写共享变量？**

**答案：** 在并发编程中，可以使用以下方法安全地读写共享变量：

- **互斥锁（Mutex）：** 通过加锁和解锁操作，保证同一时间只有一个 goroutine 可以访问共享变量。
- **读写锁（RWMutex）：** 允许多个 goroutine 同时读取共享变量，但只允许一个 goroutine 写入。
- **原子操作（atomic 包）：** 提供了原子级别的操作，例如 `AddInt32`、`CompareAndSwapInt32` 等，可以避免数据竞争。
- **通道（chan）：** 使用通道来传递数据，保证数据同步。

**示例代码：**

```go
var counter int
var mu sync.Mutex

func increment() {
    mu.Lock()
    counter++
    mu.Unlock()
}

func main() {
    var wg sync.WaitGroup
    for i := 0; i < 1000; i++ {
        wg.Add(1)
        go func() {
            defer wg.Done()
            increment()
        }()
    }
    wg.Wait()
    fmt.Println("Counter:", counter)
}
```

**2. 如何在函数中传递结构体指针？**

**答案：** 在函数中传递结构体指针时，需要将结构体变量的地址作为参数传递。这样，函数可以修改结构体变量的内容，但不会改变结构体变量本身。

**示例代码：**

```go
type Person struct {
    Name string
    Age  int
}

func printPerson(p *Person) {
    fmt.Println(p.Name, p.Age)
}

func main() {
    person := &Person{"Alice", 30}
    printPerson(person)
}
```

**3. 如何实现单例模式？**

**答案：** 单例模式可以通过以下方法实现：

- 使用懒汉式（懒加载）：在类加载时不会创建对象，只有当第一次使用时才会创建对象。
- 使用饿汉式（饿加载）：在类加载时就会创建对象。
- 使用静态内部类：静态内部类只会被加载一次，可以实现单例模式。

**示例代码：**

```java
public class Singleton {
    private static class SingletonHolder {
        private static final Singleton INSTANCE = new Singleton();
    }

    private Singleton() {}

    public static final Singleton getInstance() {
        return SingletonHolder.INSTANCE;
    }
}
```

#### 算法编程题库

**1. 两数之和**

**题目描述：** 给定一个整数数组 `nums` 和一个目标值 `target`，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**示例：** 给定 nums = `[2, 7, 11, 15]`，target = `9`，因为 `2 + 7 = 9`，所以返回 `[0, 1]`。

**解答：**

```python
def twoSum(nums, target):
    hash_map = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hash_map:
            return [hash_map[complement], i]
        hash_map[num] = i
    return []
```

**2. 寻找旋转排序数组中的最小值**

**题目描述：** 已知一个长度为 `n` 的数组，假设这个数组是在首先将元素排序，然后再把数组的一部分轮旋转移，例如 `[0,1,2,4,5,6,7]` 变为 `[4,5,6,7,0,1,2]`。请找出并返回数组中的最小元素。

**示例：** 给定 `[3,4,5,1,2]`，输出 `1`。

**解答：**

```java
public int findMin(int[] nums) {
    int left = 0;
    int right = nums.length - 1;
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] > nums[right]) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    return nums[left];
}
```

**3. 回文数**

**题目描述：** 判断一个整数是否是回文数。回文数是指正读和反读都一样的数。

**示例：** 输入 `121`，返回 `true`。

**解答：**

```python
def isPalindrome(x):
    if x < 0 or (x % 10 == 0 and x != 0):
        return False
    reverse = 0
    while x > reverse:
        reverse = reverse * 10 + x % 10
        x //= 10
    return x == reverse or x == reverse // 10
```
