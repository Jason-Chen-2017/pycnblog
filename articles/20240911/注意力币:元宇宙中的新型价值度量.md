### 一、典型面试题

#### 1. 什么是注意力币？

**答案：** 注意力币是一种基于区块链技术的数字货币，用于衡量用户在元宇宙中参与的活跃度和注意力水平。它反映了用户在元宇宙中的价值贡献，可以作为交易、奖励和支付的手段。

#### 2. 注意力币的价值是如何确定的？

**答案：** 注意力币的价值通常由以下几个因素决定：

- 用户参与度：用户在元宇宙中的活跃度越高，其注意力币的价值越高。
- 元宇宙生态的吸引力：元宇宙生态的繁荣程度越高，注意力币的稀缺性越强，价值也越高。
- 市场供需关系：注意力币的供应量与需求量之间的平衡也会影响其价值。

#### 3. 注意力币与法币的关系是什么？

**答案：** 注意力币与法币之间的关系可以看作是一种虚拟货币与法定货币之间的兑换关系。在元宇宙中，用户可以使用法币购买注意力币，并在元宇宙中进行交易、支付等活动。

### 二、算法编程题

#### 1. 如何计算用户在元宇宙中的注意力币余额？

**题目描述：** 给定一个用户在元宇宙中的活动记录，包括参与的各种活动类型和持续时间，编写一个函数计算用户当前的注意力币余额。

**答案：**
```python
def calculate_attention_balance(activities):
    attention_points = 0
    for activity in activities:
        if activity['type'] == 'game':
            attention_points += activity['duration'] * 2
        elif activity['type'] == 'social':
            attention_points += activity['duration'] * 1.5
        elif activity['type'] == 'education':
            attention_points += activity['duration'] * 1
        else:
            attention_points += activity['duration'] * 0.5
    return attention_points

# 示例数据
activities = [
    {'type': 'game', 'duration': 60},
    {'type': 'social', 'duration': 30},
    {'type': 'education', 'duration': 120},
    {'type': 'work', 'duration': 60}
]

print(calculate_attention_balance(activities))  # 输出 330
```

#### 2. 如何设计一个注意力币的交易系统？

**题目描述：** 设计一个注意力币的交易系统，支持用户之间的注意力币交易，包括购买、出售和查询余额等功能。

**答案：**
```python
class AttentionCoinExchange:
    def __init__(self):
        self.user_balances = {}

    def buy_attention_coins(self, user_id, amount):
        if amount <= 0:
            return "Invalid amount"
        if user_id not in self.user_balances:
            self.user_balances[user_id] = 0
        self.user_balances[user_id] += amount
        return f"{user_id} has successfully bought {amount} attention coins."

    def sell_attention_coins(self, user_id, amount):
        if amount <= 0 or user_id not in self.user_balances or self.user_balances[user_id] < amount:
            return "Invalid amount or insufficient balance"
        self.user_balances[user_id] -= amount
        return f"{user_id} has successfully sold {amount} attention coins."

    def check_balance(self, user_id):
        if user_id not in self.user_balances:
            return "User not found"
        return f"The balance of {user_id} is {self.user_balances[user_id]} attention coins."

# 示例使用
exchange = AttentionCoinExchange()
print(exchange.buy_attention_coins('user1', 100))  # Output: user1 has successfully bought 100 attention coins.
print(exchange.sell_attention_coins('user1', 50))   # Output: user1 has successfully sold 50 attention coins.
print(exchange.check_balance('user1'))               # Output: The balance of user1 is 50 attention coins.
```
