**2. 算法编程题库：**

**题目 1：** 设计一个算法，用于计算用户在元宇宙中的注意力币收益。

**算法思路：**
1. 输入用户在元宇宙中的活动记录，包括活动类型、时间、参与程度等。
2. 根据不同活动类型的权重，计算用户在这些活动中的总收益。
3. 返回用户的总注意力币收益。

```python
def calculate_attention_coinRewards(user_activities):
    activity_weights = {
        'task_completion': 1.0,
        'virtual_shopping': 0.5,
        'content_creation': 2.0,
        'virtual_interaction': 0.8
    }
    
    total_rewards = 0
    for activity, level in user_activities.items():
        total_rewards += level * activity_weights[activity]
    
    return total_rewards
```

**题目 2：** 设计一个区块链数据结构，用于记录注意力币的转账交易。

**数据结构设计：**
1. 交易记录包含：交易ID、发送者地址、接收者地址、交易金额、交易时间。
2. 使用区块链的数据结构，确保交易的不可篡改性和去中心化。

```python
class Transaction:
    def __init__(self, transaction_id, sender_address, receiver_address, amount, timestamp):
        self.transaction_id = transaction_id
        self.sender_address = sender_address
        self.receiver_address = receiver_address
        self.amount = amount
        self.timestamp = timestamp
        
class Blockchain:
    def __init__(self):
        self.transactions = []
        self.current_transactions = []
        
    def add_transaction(self, transaction):
        self.current_transactions.append(transaction)
        if len(self.current_transactions) >= 10:
            self._mine_block()
            
    def _mine_block(self):
        if len(self.current_transactions) > 0:
            last_block = self.transactions[-1]
            new_block = Block(last_block.hash, self.current_transactions, last_block.index + 1)
            self.transactions.append(new_block)
            self.current_transactions = []
        
class Block:
    def __init__(self, previous_hash, transactions, index):
        self.previous_hash = previous_hash
        self.transactions = transactions
        self.index = index
        self.hash = self._calculate_hash()
        
    def _calculate_hash(self):
        block_string = str(self.index) + self.previous_hash + str(len(self.transactions)) + str(self.transactions)
        return sha256(block_string.encode()).hexdigest()
```
