#### 二、算法编程题库

### 1. 如何使用 Python 实现经济增长率的计算？

**题目：** 编写一个 Python 函数，计算给定年份的中国国内生产总值（GDP）增长率。

**答案：**

```python
def calculate_growth_rate(gdp_last_year, gdp_current_year):
    return (gdp_current_year - gdp_last_year) / gdp_last_year

# 示例
gdp_last_year = 1000000
gdp_current_year = 950000
growth_rate = calculate_growth_rate(gdp_last_year, gdp_current_year)
print("经济增长率：", growth_rate)
```

### 2. 如何使用 SQL 查询经济增速放缓期间，哪些国家的 GDP 增长率低于全球平均水平？

**题目：** 编写一个 SQL 查询语句，查询经济增速放缓期间（例如 2020 年），哪些国家的 GDP 增长率低于全球平均水平。

**答案：**

```sql
SELECT country, gdp_growth_rate
FROM country_economic_data
WHERE gdp_growth_rate < (SELECT AVG(gdp_growth_rate) FROM country_economic_data WHERE year = 2020)
AND year = 2020;
```
