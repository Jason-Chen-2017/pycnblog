### 额外算法编程题

#### 11. 实现一个基于区块链的简单版权登记系统。

**问题描述：** 创建一个简单的区块链系统，允许用户将版权信息登记到区块链上。版权信息包括：创作者名称、作品名称、创作日期等。

**答案：** 下面是一个简单的基于区块链的版权登记系统的示例代码。

```go
package main

import (
    "crypto/sha256"
    "encoding/hex"
    "fmt"
)

// 区块结构
type Block struct {
    Index     int       // 区块索引
    Timestamp string     // 时间戳
    Data      string     // 数据（版权信息）
    Hash      string     // 哈希值
    PrevHash  string     // 前一个块的哈希值
}

// 创建一个新的区块
func NewBlock(index int, data string, prevHash string) *Block {
    return &Block{index, Time.Now().Format("2006-01-02 15:04:05"), data, CalculateHash(data), prevHash}
}

// 计算块的哈希值
func CalculateHash(data string) string {
    hash := sha256.Sum256([]byte(data))
    return hex.EncodeToString(hash[:])
}

// 创建区块链
var blockchain = []*Block{}
prevHash := ""

// 添加区块到区块链
func AddBlock(data string) {
    newBlock := NewBlock(len(blockchain), data, prevHash)
    blockchain = append(blockchain, newBlock)
    prevHash = newBlock.Hash
}

// 打印区块链
func PrintBlockchain() {
    for _, block := range blockchain {
        fmt.Println(BlockToJsonString(block))
    }
}

// 将块转换为JSON字符串
func BlockToJsonString(block *Block) string {
    jsonBlock, err := json.MarshalIndent(block, "", "    ")
    if err != nil {
        return ""
    }
    return string(jsonBlock)
}

func main() {
    // 添加一些初始区块
    AddBlock("版权信息1")
    AddBlock("版权信息2")
    AddBlock("版权信息3")

    // 打印区块链
    PrintBlockchain()
}
```

**解析：** 这个简单的区块链系统包括一个`Block`结构体，用于存储每个区块的数据，以及一个`CalculateHash`函数用于计算哈希值。`AddBlock`函数用于将新的区块添加到区块链中。最后，`PrintBlockchain`函数用于打印区块链中的所有区块。

通过这个示例，我们可以看到如何使用区块链技术来登记版权信息，并确保这些信息的不可篡改性和可追溯性。
