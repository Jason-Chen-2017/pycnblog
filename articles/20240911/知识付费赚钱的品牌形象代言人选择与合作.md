### 4. 如何确定代言费用？

**题目：** 请设计一个确定代言费用的算法模型。

**答案：** 可以根据以下因素来确定代言费用：

- **代言人知名度：** 越知名的代言人费用越高。
- **品牌预算：** 品牌方的预算越高，代言人费用也越高。
- **合作时长：** 合作时间越长，代言人费用越高。

**算法模型：**

```python
def calculate_endorsement_fee(endorser_influence, brand_budget, cooperation_time):
    base_fee = 10000  # 基础费用
    
    # 根据代言人知名度调整费用
    if endorser_influence > 90:
        fee_adjustment = 1.5
    elif endorser_influence > 70:
        fee_adjustment = 1.2
    else:
        fee_adjustment = 1
        
    # 根据品牌预算调整费用
    if brand_budget > 5000000:
        fee_adjustment *= 1.2
    elif brand_budget > 1000000:
        fee_adjustment *= 1.1
        
    # 根据合作时长调整费用
    fee_adjustment *= cooperation_time
    
    total_fee = base_fee * fee_adjustment
    
    return total_fee
```

**解析：** 通过计算基础费用并根据代言人知名度、品牌预算和合作时长进行调整，可以得到代言费用的最终结果。
