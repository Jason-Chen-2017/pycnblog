#### 2. 如何实现一个实时健身指导的对话系统？

**题目：** 请设计一个实时健身指导的对话系统，用户可以输入问题，系统根据问题提供相应的健身指导。

**答案：**

```python
def fitness_guide(question):
    if "增肌" in question:
        return "增肌建议：进行高强度训练，每周训练 3-4 次，每次训练 60-90 分钟。"
    elif "减脂" in question:
        return "减脂建议：进行有氧运动，每周训练 3-4 次，每次训练 30-60 分钟。"
    else:
        return "请提出具体的问题，我将为您提供详细的指导。"

# 示例
print(fitness_guide("如何增肌？"))
```

**解析：** 这个简单的实时健身指导对话系统根据用户输入的问题，提供相应的健身指导建议。在实际应用中，可以结合更多语言模型和技术，提高对话系统的智能化水平。
