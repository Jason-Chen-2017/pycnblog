## 算法编程题库

### 6. 编写一个Python函数，实现特征重要性评估。

**答案：** 

```python
import numpy as np

def feature_importance_evaluation(model, X, y):
    """
    评估特征的重要性。
    
    :param model: 深度学习模型
    :param X: 特征矩阵
    :param y: 标签向量
    :return: 特征重要性得分
    """
    # 计算模型对每个特征的偏导数
    gradients = np.array([np.mean((model.predict(X[:, i].reshape(-1, 1)) - y).abs()) for i in range(X.shape[1])])

    # 特征重要性得分
    feature_importance = gradients / gradients.sum()

    return feature_importance
```

### 7. 编写一个Python函数，实现个体贡献度评估。

**答案：**

```python
import numpy as np

def individual_contribution_evaluation(model, X, y, threshold=0.01):
    """
    评估个体的贡献度。
    
    :param model: 深度学习模型
    :param X: 特征矩阵
    :param y: 标签向量
    :param threshold: 贡献度阈值
    :return: 贡献度得分
    """
    # 计算每个个体对模型预测的影响
    contributions = np.array([np.mean((model.predict(X[i].reshape(1, -1)) - y[i]).abs()) for i in range(len(y))])

    # 筛选超过阈值的个体贡献度
    significant_contributions = contributions[contributions > threshold]

    return significant_contributions
```
