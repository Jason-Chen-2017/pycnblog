##### **3. 处理并发时间问题的代码实例**

**题目：** 编写一个并发程序，实现多用户同时访问共享资源，并确保数据的一致性。

**答案：** 下面是一个简单的 Python 示例，使用锁（Lock）处理并发时间问题：

```python
import threading

lock = threading.Lock()
counter = 0

def increment():
    global counter
    lock.acquire()
    try:
        counter += 1
    finally:
        lock.release()

# 示例
threads = []
for i in range(10):
    t = threading.Thread(target=increment)
    threads.append(t)
    t.start()

for t in threads:
    t.join()

print("Counter:", counter)  # 输出 10
```

**解析：** 这个程序通过使用锁来确保多线程同时访问共享变量 `counter` 时的一致性。每个线程在修改 `counter` 之前都会获取锁，并在修改完成后释放锁。
