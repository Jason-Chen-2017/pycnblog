##### 编程题 2：集合的最大子集和
**题目：** 给定一个整数数组，找出最大的子集和，该子集的和必须大于等于给定值。
**答案：**
```python
def max_subset_sum(nums, target):
    dp = [0] * (target + 1)
    for num in nums:
        for j in range(target, num - 1, -1):
            dp[j] = max(dp[j], dp[j - num] + num)
    return dp[target]

nums = [1, 2, 5]
target = 7
print(max_subset_sum(nums, target))
```

### 1. 集合的基数是什么？
**题目：** 什么是集合的基数？基数有哪些性质？

**答案：** 集合的基数是指集合中元素的数量。基数是集合的一个基本性质，通常用符号`|A|`表示，其中`A`是一个集合。基数具有以下性质：

1. **非负性：** 任何集合的基数都是非负整数，即`|A| ≥ 0`。
2. **确定性：** 对于任何集合`A`，其基数是唯一的。
3. **可加性：** 如果集合`A`和集合`B`没有公共元素，那么它们的基数之和等于两个集合的基数之和，即`|A ∪ B| = |A| + |B|`。
4. **不变性：** 如果对集合进行有限次的并集、交集、补集等运算，其基数不会改变。

**举例：**
设有集合`A = {1, 2, 3}`，`B = {4, 5}`，则`|A| = 3`，`|B| = 2`，`|A ∪ B| = |A| + |B| = 3 + 2 = 5`。
