### 算法编程题 1：构建社交网络图以分析集体意识传播

**题目：** 请使用图论算法构建一个社交网络图，模拟集体意识的传播过程。

**答案：** 解题思路如下：

1. **数据结构：** 使用邻接表或邻接矩阵表示社交网络图。
2. **初始化：** 根据社交网络数据初始化图。
3. **传播模拟：** 从某个初始节点开始，将其影响力传播给相邻节点，直至整个网络。
4. **结果分析：** 统计传播后网络的连接情况、影响力覆盖范围等。

```python
from collections import defaultdict

class SocialNetworkGraph:
    def __init__(self):
        self.graph = defaultdict(list)

    def add_edge(self, u, v):
        self.graph[u].append(v)
        self.graph[v].append(u)

    def propagate_awareness(self, start):
        visited = set()
        awareness_level = {start: 1}
        queue = deque([start])

        while queue:
            node = queue.popleft()
            for neighbor in self.graph[node]:
                if neighbor not in visited:
                    visited.add(neighbor)
                    awareness_level[neighbor] = awareness_level[node] + 1
                    queue.append(neighbor)

        return awareness_level

# 示例
graph = SocialNetworkGraph()
graph.add_edge(1, 2)
graph.add_edge(2, 3)
graph.add_edge(3, 4)
print(graph.propagate_awareness(1))
```

**解析：** 该算法使用广度优先搜索（BFS）模拟集体意识的传播过程，从初始节点开始，逐步传播给相邻节点，记录每个节点的感知层次。

### 算法编程题 2：分析群体情绪变化

**题目：** 请设计一个算法，分析群体情绪的变化趋势。

**答案：** 解题思路如下：

1. **数据预处理：** 收集群体情绪数据，如社交媒体评论、调查问卷等。
2. **情绪分类：** 使用自然语言处理技术对文本进行情感分析，分类为积极、消极或中性。
3. **时间序列分析：** 将情绪分类结果按时间顺序排列，使用统计学方法或机器学习模型分析情绪变化的趋势。

```python
import pandas as pd
from textblob import TextBlob

def classify_sentiment(texts):
    sentiments = []
    for text in texts:
        blob = TextBlob(text)
        if blob.sentiment.polarity > 0:
            sentiments.append('positive')
        elif blob.sentiment.polarity < 0:
            sentiments.append('negative')
        else:
            sentiments.append('neutral')
    return sentiments

# 示例
texts = ["I love this product!", "This is the worst experience ever!", "It's okay, nothing special."]
df = pd.DataFrame({'text': texts})
df['sentiment'] = classify_sentiment(df['text'])
print(df)
```

**解析：** 该算法使用TextBlob库对文本进行情感分析，分类为积极、消极或中性，进而分析群体情绪的变化趋势。
