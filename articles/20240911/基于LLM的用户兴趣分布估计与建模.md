#### 2. LLM模型编程题

**题目：** 使用PyTorch实现一个简单的基于GPT-2模型的用户兴趣分布预测。

**答案：** 实现步骤：

1. 数据预处理：加载用户行为数据，进行清洗和编码。
2. 模型构建：定义GPT-2模型结构。
3. 训练：将数据输入模型，进行训练。
4. 预测：使用训练好的模型进行用户兴趣分布预测。

**代码示例：**

```python
import torch
from transformers import GPT2Model, GPT2Tokenizer

# 数据预处理
tokenizer = GPT2Tokenizer.from_pretrained("gpt2")
inputs = tokenizer.encode("用户最近搜索的关键词：人工智能，热门新闻：元宇宙技术进展，", return_tensors="pt")

# 模型构建
model = GPT2Model.from_pretrained("gpt2")

# 训练
outputs = model(inputs)
logits = outputs.logits

# 预测
predicted_interests = torch.softmax(logits, dim=-1).detach().numpy()
print(predicted_interests)
```
