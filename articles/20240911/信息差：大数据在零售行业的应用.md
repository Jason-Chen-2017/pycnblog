#### 算法编程题库

##### 1. 某零售企业在一天内收集了如下商品销售数据，请编写一个函数，计算每个商品的平均销售量。

**输入：** 
```python
[
    {"商品ID": "A", "销售量": [100, 150, 200]},
    {"商品ID": "B", "销售量": [50, 60, 70]},
    {"商品ID": "C", "销售量": [30, 40, 50]},
]
```

**输出：**
```python
[
    {"商品ID": "A", "平均销售量": 133.33},
    {"商品ID": "B", "平均销售量": 60.0},
    {"商品ID": "C", "平均销售量": 40.0},
]
```

**答案：**
```python
def calculate_average_sales(data):
    result = []
    for item in data:
        total_sales = sum(item['销售量'])
        average_sales = total_sales / len(item['销售量'])
        result.append({"商品ID": item['商品ID'], "平均销售量": round(average_sales, 2)})
    return result

data = [
    {"商品ID": "A", "销售量": [100, 150, 200]},
    {"商品ID": "B", "销售量": [50, 60, 70]},
    {"商品ID": "C", "销售量": [30, 40, 50]},
]

print(calculate_average_sales(data))
```

##### 2. 某零售企业需要对商品进行分类，根据销售量将商品分为“畅销”、“热门”和“滞销”三个等级。请编写一个函数，根据销售量对商品进行分类。

**输入：** 
```python
[
    {"商品ID": "A", "销售量": 300},
    {"商品ID": "B", "销售量": 150},
    {"商品ID": "C", "销售量": 50},
]
```

**输出：**
```python
[
    {"商品ID": "A", "分类": "畅销"},
    {"商品ID": "B", "分类": "热门"},
    {"商品ID": "C", "分类": "滞销"},
]
```

**答案：**
```python
def classify_products(data):
    thresholds = {
        "畅销": 200,
        "热门": 100,
        "滞销": 0,
    }
    result = []
    for item in data:
        for category, threshold in thresholds.items():
            if item['销售量'] >= threshold:
                result.append({"商品ID": item['商品ID'], "分类": category})
                break
    return result

data = [
    {"商品ID": "A", "销售量": 300},
    {"商品ID": "B", "销售量": 150},
    {"商品ID": "C", "销售量": 50},
]

print(classify_products(data))
```
