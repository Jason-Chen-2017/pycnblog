##### 1. 知识产权许可的主要类型有哪些？

**面试题：** 请简述知识产权许可的主要类型，并分别解释其含义。

**答案：**

- **独占许可（Exclusive License）：** 许可方授予许可方在特定地区、产品或时间段内独家使用知识产权的权利，第三方不得使用。
- **排他许可（Exclusive License）：** 许可方授予许可方在特定地区、产品或时间段内使用知识产权的权利，许可方可以自行使用，但不得再授予第三方。
- **普通许可（Non-Exclusive License）：** 许可方授予许可方在特定地区、产品或时间段内使用知识产权的权利，许可方还可以将同一知识产权授予给其他第三方。
- **交叉许可（Cross-License）：** 双方互相授予对方使用各自知识产权的权利，通常是相互解决知识产权纠纷的一种方式。

- **保密协议：** 双方在许可期间是否需要签署保密协议，保密的内容、范围和期限等。
- **知识产权保护：** 许可产品的知识产权保护措施，包括版权、专利、商标等。

- **谈判定价：** 通过双方谈判，根据许可方的需求、许可产品的市场潜力等因素，协商确定许可费用。

#### 算法编程题库

##### 6. 请设计一个算法，用于计算两个 IP 地址之间的距离。

**面试题：** 编写一个函数，计算两个 IP 地址之间的距离。IP 地址由四个整数表示，如 "192.168.1.1"。

**答案：**

```python
def ip_distance(ip1, ip2):
    def to_int(ip):
        return sum(int(i) << (8 * j) for j, i in enumerate(ip.split('.')))

    def distance(a, b):
        return abs(a - b)

    return distance(to_int(ip1), to_int(ip2))
```

##### 7. 请设计一个算法，用于计算两个域名之间的相似度。

**面试题：** 编写一个函数，计算两个域名之间的相似度。相似度越高，表示两个域名越相似。

**答案：**

```python
def domain_similarity(domain1, domain2):
    def similarity(a, b):
        return sum(i == j for i, j in zip(a, b))

    return similarity(domain1.split('.')[:-1], domain2.split('.')[:-1])
```

##### 8. 请设计一个算法，用于计算一个文本中所有字符串的并集和交集。

**面试题：** 编写一个函数，计算一个文本中所有字符串的并集和交集。

**答案：**

```python
def string_set_operations(text):
    words = set(text.split())
    union = set()
    intersection = set()

    for word in words:
        union.add(word)
        if word in words:
            intersection.add(word)

    return union, intersection
```

##### 6. 计算两个 IP 地址之间的距离

##### 8. 计算一个文本中所有字符串的并集和交集

该算法首先将文本拆分为字符串列表，然后计算并集和交集。算法定义了一个函数 `string_set_operations`，该函数首先将文本拆分为字符串列表，然后使用 `set` 数据结构计算并集和交集，最后返回并集和交集。

```python
def string_set_operations(text):
    words = set(text.split())
    union = set()
    intersection = set()

    for word in words:

        if word in words:
            intersection.add(word)

    return union, intersection
```
