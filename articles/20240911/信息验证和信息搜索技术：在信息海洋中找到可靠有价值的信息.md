## 面试题

### 1. 如何判断一个字符串是否是回文？

**答案：** 可以使用双指针法，分别从字符串的头和尾开始遍历，比较对应位置的字符是否相等。以下是 Python 实现代码：

```python
def is_palindrome(s: str) -> bool:
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1
    return True
```

### 3. 如何实现一个简单的前缀树？

**答案：** 前缀树（Trie）是一种用于存储字符串的有效数据结构。以下是一个简单的 Python 实现：

```python
class Trie:
    def __init__(self):
        self.children = [None] * 26
        self.is_end_of_word = False

    def insert(self, word):
        node = self
        for char in word:
            index = ord(char) - ord('a')
            if node.children[index] is None:
                node.children[index] = Trie()
            node = node.children[index]
        node.is_end_of_word = True

    def search(self, word):
        node = self
        for char in word:
            index = ord(char) - ord('a')
            if node.children[index] is None:
                return False
            node = node.children[index]
        return node.is_end_of_word
```

## 算法编程题

### 1. 如何实现一个高效的搜索引擎？

**答案：** 高效的搜索引擎通常包括以下几个关键组件：

1. **倒排索引（Inverted Index）：** 将文档中的单词映射到包含该单词的文档列表，以便快速检索；
2. **搜索引擎优化（SEO）：** 优化网站内容和结构，提高在搜索引擎结果中的排名；
3. **排名算法：** 根据用户查询和文档相关性，对搜索结果进行排序。

以下是一个简单的倒排索引实现：

```python
class InvertedIndex:
    def __init__(self):
        self.index = {}

    def add_document(self, doc_id, words):
        for word in words:
            if word not in self.index:
                self.index[word] = []
            self.index[word].append(doc_id)

    def search(self, query):
        results = []
        for word in query.split():
            if word not in self.index:
                return results
            results = list(set(results) & set(self.index[word]))
        return results
```

### 2. 如何实现一个基于词频的文本相似度算法？

**答案：** 基于词频的文本相似度算法可以通过计算两个文本的交集和并集，以及它们的长度，来评估它们之间的相似度。以下是一个简单的实现：

```python
def text_similarity(text1, text2):
    intersection = len(set(text1) & set(text2))
    union = len(set(text1) | set(text2))
    similarity = intersection / union
    return similarity
```

### 3. 如何实现一个基于语义的文本相似度算法？

**答案：** 基于语义的文本相似度算法可以通过自然语言处理技术，如词向量和句子嵌入，来评估两个文本之间的相似度。以下是一个简单的实现：

```python
import torch
import torch.nn as nn

class SentenceSimilarity(nn.Module):
    def __init__(self):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, embedding_dim)
        self.lstm = nn.LSTM(embedding_dim, hidden_size, num_layers=1, batch_first=True)
        self.fc = nn.Linear(hidden_size, 1)

    def forward(self, sentence1, sentence2):
        embed1 = self.embedding(sentence1)
        embed2 = self.embedding(sentence2)
        output1, _ = self.lstm(embed1)
        output2, _ = self.lstm(embed2)
        similarity = self.fc(torch.cat((output1[-1], output2[-1]), dim=1))
        return similarity
```
