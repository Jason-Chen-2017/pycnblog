#### 题目 1：用户活跃度分析

**题目描述：** 给定一段时间内用户的登录记录，分析用户的活跃度。

**输入：**

```
[
  ["user1", "2023-03-01"],
  ["user1", "2023-03-02"],
  ["user2", "2023-03-01"],
  ["user2", "2023-03-03"],
  ["user3", "2023-03-02"],
  ["user3", "2023-03-03"]
]
```

**输出：**

```
[
  ["user1", 2],
  ["user2", 2],
  ["user3", 2]
]
```

**答案：**

```python
from collections import defaultdict

def analyze_user_activity(logs):
    user_activity = defaultdict(int)
    for user, date in logs:
        user_activity[user] += 1
    return [[user, count] for user, count in user_activity.items()]

logs = [
  ["user1", "2023-03-01"],
  ["user1", "2023-03-02"],
  ["user2", "2023-03-01"],
  ["user2", "2023-03-03"],
  ["user3", "2023-03-02"],
  ["user3", "2023-03-03"]
]

print(analyze_user_activity(logs))
```

#### 题目 2：用户互动分析

**题目描述：** 给定一段时间内用户的互动记录，分析用户的互动活跃度。

**输入：**

```
[
  ["user1", "user2"],
  ["user1", "user3"],
  ["user2", "user1"],
  ["user2", "user3"],
  ["user3", "user1"],
  ["user3", "user2"]
]
```

**输出：**

```
[
  ["user1", 2],
  ["user2", 3],
  ["user3", 2]
]
```

**答案：**

```python
from collections import defaultdict

def analyze_user_interactions(interactions):
    user_interactions = defaultdict(int)
    for user1, user2 in interactions:
        user_interactions[user1] += 1
        user_interactions[user2] += 1
    return [[user, count] for user, count in user_interactions.items()]

interactions = [
  ["user1", "user2"],
  ["user1", "user3"],
  ["user2", "user1"],
  ["user2", "user3"],
  ["user3", "user1"],
  ["user3", "user2"]
]

print(analyze_user_interactions(interactions))
```
