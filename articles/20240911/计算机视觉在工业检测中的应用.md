### 6. 编写一个基于 Canny 算子的边缘检测算法。

```python
import cv2
import numpy as np

def canny_detection(image_path, min_threshold, max_threshold):
    # 读取图像
    image = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)
    
    # 使用 Canny 算子进行边缘检测
    edges = cv2.Canny(image, min_threshold, max_threshold)
    
    # 显示结果
    cv2.imshow('Canny Detection', edges)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

# 调用函数
canny_detection('image.jpg', 50, 150)
```

**解析：** 该代码读取输入的灰度图像，并使用 Canny 算子进行边缘检测。Canny 算子需要设置两个阈值，分别用于初步边缘检测和最终边缘保留。

### 7. 编写一个基于 Otsu 阈值法的二值化算法。

```python
import cv2
import numpy as np

def otsu_thresholding(image_path):
    # 读取图像
    image = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)
    
    # 使用 Otsu 阈值法进行二值化
    _, threshold = cv2.threshold(image, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    
    # 显示结果
    cv2.imshow('Otsu Thresholding', threshold)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

# 调用函数
otsu_thresholding('image.jpg')
```

**解析：** 该代码读取输入的灰度图像，并使用 Otsu 阈值法进行二值化。Otsu 阈值法通过计算图像的直方图，自动选择最优的阈值。

### 8. 编写一个基于形态学处理的轮廓提取算法。

```python
import cv2
import numpy as np

def morphological_contour_extraction(image_path):
    # 读取图像
    image = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)
    
    # 使用 Otsu 阈值法进行二值化
    _, threshold = cv2.threshold(image, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    
    # 获取结构元素
    kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (3, 3))
    
    # 使用形态学操作进行轮廓提取
    morph_image = cv2.erode(threshold, kernel, iterations=1)
    contours, _ = cv2.findContours(morph_image, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    # 绘制轮廓
    image_with_contours = cv2.cvtColor(image, cv2.COLOR_GRAY2BGR)
    for contour in contours:
        cv2.drawContours(image_with_contours, [contour], -1, (0, 255, 0), 2)
    
    # 显示结果
    cv2.imshow('Morphological Contour Extraction', image_with_contours)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

# 调用函数
morphological_contour_extraction('image.jpg')
```

**解析：** 该代码首先使用 Otsu 阈值法进行二值化，然后使用腐蚀操作提取轮廓。腐蚀操作能够去除图像中的噪声，从而获得更清晰的轮廓。

### 9. 编写一个基于 SIFT 算法的特征提取算法。

```python
import cv2
import numpy as np

def sift_keypoint_detection(image_path):
    # 读取图像
    image = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)
    
    # 使用 SIFT 算法提取关键点
    sift = cv2.SIFT_create()
    keypoints, descriptors = sift.detectAndCompute(image, None)
    
    # 显示结果
    image_with_keypoints = cv2.drawKeypoints(image, keypoints, None, (0, 0, 255))
    cv2.imshow('SIFT KeyPoint Detection', image_with_keypoints)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

# 调用函数
sift_keypoint_detection('image.jpg')
```

**解析：** 该代码使用 SIFT 算法提取图像中的关键点，并将关键点绘制在图像上。SIFT 算法能够提取出具有区分性的特征点，常用于图像配对和目标跟踪。

### 10. 编写一个基于支持向量机的分类算法。

```python
import cv2
import numpy as np
from sklearn import svm

def svm_classification(image_path, labels_path):
    # 读取图像和标签
    image = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)
    labels = np.loadtxt(labels_path, dtype=np.int32)
    
    # 使用灰度图像进行特征提取
    gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    flat_image = gray_image.reshape(-1, gray_image.shape[0] * gray_image.shape[1])
    
    # 使用支持向量机进行分类
    clf = svm.SVC()
    clf.fit(flat_image, labels)
    
    # 测试分类结果
    predicted_labels = clf.predict(flat_image)
    print("Classification Accuracy:", np.mean(predicted_labels == labels))

# 调用函数
svm_classification('image.jpg', 'labels.txt')
```

**解析：** 该代码使用支持向量机（SVM）对灰度图像进行分类。首先，将图像转换为灰度图像，然后提取图像的特征，最后使用 SVM 模型进行分类。
