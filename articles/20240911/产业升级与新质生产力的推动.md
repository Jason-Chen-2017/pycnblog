##### 编程题 1：给定一个整数数组，找出数组中的重复元素。

```python
def find_duplicates(nums):
    seen = set()
    duplicates = []
    for num in nums:
        if num in seen:
            duplicates.append(num)
        else:
            seen.add(num)
    return duplicates
```

**答案解析：** 该算法通过遍历数组，利用集合判断元素是否已存在。如果已存在，则将其添加到重复元素列表中。最后返回重复元素列表。

##### 编程题 2：给定一个字符串，找出最长的无重复字符子串的长度。

```python
def length_of_longest_substring(s):
    start = 0
    max_len = 0
    used_chars = set()
    for i, char in enumerate(s):
        while char in used_chars:
            used_chars.remove(s[start])
            start += 1
        used_chars.add(char)
        max_len = max(max_len, i - start + 1)
    return max_len
```

**答案解析：** 该算法利用双指针技巧，维护一个滑动窗口。当遇到重复字符时，将窗口左边界向右移动，直到去除重复字符。更新最长无重复字符子串的长度，并返回最终结果。
