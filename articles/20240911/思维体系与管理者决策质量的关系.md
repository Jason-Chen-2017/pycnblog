### 5. 实现一个基于支持向量机的分类算法

**题目描述：** 编写一个函数，实现一个基于支持向量机的分类算法。

**答案解析：** 支持向量机（SVM）是一种常用的机器学习算法，适用于分类问题。以下是一个简单的SVM实现：

```python
import numpy as np

def svm_fit(X, y, C=1.0, kernel='linear'):
    # 特征缩放
    X_mean = np.mean(X, axis=0)
    X_std = np.std(X, axis=0)
    X = (X - X_mean) / X_std

    # 初始化参数
    w = np.zeros(X.shape[1])
    b = 0

    # 梯度下降求解
    for epoch in range(1000):
        for x, y in zip(X, y):
            if y * (np.dot(w, x) + b) < 1:
                w -= C * (2 * np.dot(w, x) * x + np.random.normal(size=x.shape))
                b -= C * y

    return w, b

def svm_predict(w, b, x, X_std, X_mean):
    x = (x - X_mean) / X_std
    return (np.dot(w, x) + b) > 0
```
