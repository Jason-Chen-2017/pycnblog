## 3. 算法编程题库与解答

### 3.1 编程题 1：实现一个简单的行人检测算法。

- **题目描述**：给定一组摄像头图像，实现一个简单的行人检测算法，输出行人出现的区域。

- **解题思路**：可以使用 OpenCV 库进行图像处理，结合 Haar cascades 进行行人检测。

- **答案**：以下是一个简单的行人检测算法的实现：

```python
import cv2

# 加载 Haar cascades 模型
haarcascades = cv2.CascadeClassifier('haarcascade_fullbody.xml')

# 读取图像
image = cv2.imread('image.jpg')

# 转换为灰度图像
gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 检测行人
detected = haarcascades.detectMultiScale(gray, 1.1, 4)

# 绘制矩形框
for (x, y, w, h) in detected:
    cv2.rectangle(image, (x, y), (x+w, y+h), (0, 255, 0), 2)

# 显示结果
cv2.imshow('Person Detection', image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
