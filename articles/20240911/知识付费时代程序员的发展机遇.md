#### 1. 算法与数据结构面试题

**题目 1：** 请解释什么是哈希表？它有什么优点和缺点？

**答案：** 哈希表（Hash Table）是一种基于键值对的数据结构，通过哈希函数将键映射到表中的位置。优点包括：快速的查找、插入和删除操作（平均时间复杂度为O(1)）。缺点包括：可能存在哈希冲突，需要额外的处理（如链地址法、开放地址法等）。

**解析：** 哈希表利用哈希函数将键映射到数组中的位置，从而实现快速访问。当发生哈希冲突时，可以通过链表将多个具有相同哈希值的键存储在一起。

**代码实例：**

```go
package main

import (
	"fmt"
)

type HashTable struct {
	buckets []*Node
}

type Node struct {
	key   int
	value int
	next  *Node
}

func (ht *HashTable) Insert(key, value int) {
	hash := hash(key)
	node := &Node{key: key, value: value}
	if ht.buckets[hash] == nil {
		ht.buckets[hash] = node
	} else {
		// Handle hash collision
	}
}

func (ht *HashTable) Find(key int) int {
	hash := hash(key)
	if ht.buckets[hash] != nil {
		return ht.buckets[hash].value
	}
	return -1
}

func hash(key int) int {
	return key % len(ht.buckets)
}

func main() {
	ht := &HashTable{buckets: make([]*Node, 10)}
	ht.Insert(1, 10)
	ht.Insert(2, 20)
	fmt.Println(ht.Find(1)) // 输出 10
	fmt.Println(ht.Find(2)) // 输出 20
}
```

#### 3. 编程算法题

**题目 3：** 请实现一个函数，判断一个字符串是否是回文。

**答案：** 可以使用双指针的方法判断一个字符串是否是回文。

**解析：** 双指针分别指向字符串的头部和尾部，逐个比较两个指针指向的字符，如果都相等，则继续移动指针，直到两个指针相遇。

**代码实例：**

```go
package main

import (
	"fmt"
)

func isPalindrome(s string) bool {
    left, right := 0, len(s)-1
    for left < right {
        if s[left] != s[right] {
            return false
        }
        left++
        right--
    }
    return true
}

func main() {
    s := "racecar"
    fmt.Println(isPalindrome(s)) // 输出 true
}
```
