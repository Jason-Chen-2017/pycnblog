#### 4. 设计一个算法，找出给定字符串中的最长公共前缀。

**题目描述：** 编写一个函数，实现以下功能：找出给定字符串数组中的最长公共前缀。

**输入：** ["flower","flow","flight"]

**输出：** "fl"

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s):
            if prefix[i] != s[i]:
                break
            i += 1
        prefix = prefix[:i]
    return prefix

# 测试
print(longestCommonPrefix(["flower","flow","flight"]))  # 输出 "fl"
```

#### 5. 设计一个算法，实现快速排序。

**题目描述：** 快速排序是一种高效的排序算法，请编写一个函数，实现快速排序。

**输入：** [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]

**输出：** [1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

# 测试
print(quicksort([3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]))  # 输出 [1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]
```
