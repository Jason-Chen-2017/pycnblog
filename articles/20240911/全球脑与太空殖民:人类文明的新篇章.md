### 三、算法编程题库及答案解析

#### 1. 编写一个程序，计算两个数的最大公约数。

**题目：** 使用辗转相除法编写一个程序，计算两个数的最大公约数。

**答案：** 

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

# 示例
a = 24
b = 36
print(gcd(a, b))  # 输出：12
```

**解析：** 该程序使用辗转相除法（又称欧几里得算法）计算两个数的最大公约数，实现简单高效。

#### 2. 编写一个程序，实现二分查找算法。

**题目：** 实现一个二分查找算法，在有序数组中查找目标元素。

**答案：**

```python
def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

# 示例
arr = [1, 3, 5, 7, 9, 11]
target = 7
print(binary_search(arr, target))  # 输出：3
```

**解析：** 该程序实现二分查找算法，时间复杂度为 O(logn)，适用于大规模数据的搜索。
