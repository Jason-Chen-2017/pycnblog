### 3. 领导力相关算法编程题

**题目：** 实现一个投票系统，保证选举的公平性。

**答案：** 可以使用以下算法实现一个投票系统：

```python
class VotingSystem:
    def __init__(self):
        self.candidates = {}
    
    def add_candidate(self, name):
        if name not in self.candidates:
            self.candidates[name] = 0
    
    def vote(self, name):
        if name in self.candidates:
            self.candidates[name] += 1
    
    def get_winner(self):
        max_votes = 0
        winner = None
        for name, votes in self.candidates.items():
            if votes > max_votes:
                max_votes = votes
                winner = name
        return winner
```

**解析：** 该投票系统首先创建一个候选人字典，用于存储候选人和其得票数。在投票时，系统会增加候选人的得票数。最后，通过遍历候选人字典，找出得票数最高的候选人作为赢家。
