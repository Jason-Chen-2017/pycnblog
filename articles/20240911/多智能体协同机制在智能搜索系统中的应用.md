**答案：** 一个经典的多智能体协同算法编程题是“旅行商问题”（TSP，Travelling Salesman Problem）。该问题是一个优化问题，目标是在一组城市中找到一条最短的闭合路径，访问每个城市恰好一次，并返回起点。

**代码示例：**

```python
import itertools
import math

def distance(city1, city2):
    # 计算两个城市之间的距离
    return math.sqrt((city1[0] - city2[0])**2 + (city1[1] - city2[1])**2)

def calculate_tsp_distance(perm):
    total_distance = 0
    for i in range(len(perm) - 1):
        total_distance += distance(perm[i], perm[i+1])
    total_distance += distance(perm[-1], perm[0])
    return total_distance

def solve_tsp(cities):
    # 使用回溯算法解决旅行商问题
    best_distance = float('inf')
    best_permutation = None

    def backtrack(perm):
        nonlocal best_distance, best_permutation

        if len(perm) == len(cities):
            distance = calculate_tsp_distance(perm)
            if distance < best_distance:
                best_distance = distance
                best_permutation = perm
            return

        for city in cities:
            if city not in perm:
                perm.append(city)
                backtrack(perm)
                perm.pop()

    backtrack([])
    return best_permutation, best_distance

# 测试数据
cities = [(0, 0), (1, 1), (2, 2), (3, 3), (4, 4)]

# 求解
best_permutation, best_distance = solve_tsp(cities)

# 输出结果
print("Best Permutation:", best_permutation)
print("Best Distance:", best_distance)
```

**解析：** 该代码示例使用了回溯算法解决旅行商问题。首先，定义了计算两个城市之间距离的 `distance` 函数。然后，定义了计算旅行商路径总距离的 `calculate_tsp_distance` 函数。最后，定义了求解旅行商问题的 `solve_tsp` 函数，它使用回溯算法在所有可能的路径中找到最短路径。
