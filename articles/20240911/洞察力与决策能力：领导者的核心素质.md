```python
def knapSack(W, wt, val, n):
    dp = [[0 for _ in range(W + 1)] for _ in range(n + 1)]

    # 初始化表格
    for i in range(n + 1):
        for w in range(W + 1):
            if i == 0 or w == 0:
                dp[i][w] = 0
            elif wt[i - 1] <= w:
                dp[i][w] = max(val[i - 1] + dp[i - 1][w - wt[i - 1]], dp[i - 1][w])
            else:
                dp[i][w] = dp[i - 1][w]

    return dp[n][W]

# 测试数据

W = 50

# 求解背包问题
print(knapSack(W, wt, val, n))
```

**解析：**

背包问题是经典的贪心算法问题，通过选择价值最大的物品，使其总重量不超过背包容量。此题旨在考察应聘者对贪心算法的理解和应用能力。
