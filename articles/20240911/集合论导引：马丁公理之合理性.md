### 三、算法编程题库及答案解析

#### 1. 集合的并集、交集和差集

**题目：** 给定两个集合 A 和 B，请编写一个函数，计算它们的并集、交集和差集。

**答案：** 可以使用集合的高效操作来计算并集、交集和差集。

```python
def set_operations(A, B):
    union = A | B  # 并集
    intersection = A & B  # 交集
    difference = A - B  # 差集
    return union, intersection, difference

# 示例
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}
union, intersection, difference = set_operations(A, B)
print("并集:", union)  # 输出 {1, 2, 3, 4, 5, 6}
print("交集:", intersection)  # 输出 {3, 4}
print("差集:", difference)  # 输出 {1, 2}
```

#### 3. 集合的子集

**题目：** 给定一个集合，请编写一个函数，生成它的所有子集。

**答案：** 可以使用位运算来生成集合的所有子集。

```python
def generate_subsets(S):
    subsets = []
    n = len(S)
    for i in range(1 << n):  # 2^n 个子集
        subset = set()
        for j in range(n):
            if i & (1 << j):  # 判断第 j 个位是否为 1
                subset.add(S[j])
        subsets.append(subset)
    return subsets

# 示例
S = {1, 2, 3}
subsets = generate_subsets(S)
print("所有子集:", subsets)  # 输出 [{}, {1}, {2}, {1, 2}, {3}, {1, 3}, {2, 3}, {1, 2, 3}]
```
