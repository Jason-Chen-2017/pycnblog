#### 题目1：二叉搜索树转排序数组

**题目描述：** 给定一个二叉搜索树，将其转换为有序的排序数组。假设树中的节点值都是唯一的。

**示例：**

```
输入：[4,2,7,1,3,6,8]
输出：[1,2,3,4,6,7,8]
```

**答案：** 首先中序遍历二叉搜索树，然后将遍历结果转换为数组即可。

**解析：** 

```go
func BSTtoSortedArray(root *TreeNode) []int {
    var result []int
    var dfs func(node *TreeNode)
    dfs = func(node *TreeNode) {
        if node == nil {
            return
        }
        dfs(node.Left)
        result = append(result, node.Val)
        dfs(node.Right)
    }
    dfs(root)
    return result
}

// TreeNode 定义
type TreeNode struct {
    Val   int
    Left  *TreeNode
    Right *TreeNode
}
```

#### 题目2：航空电子系统故障诊断

**题目描述：** 设计一个算法，对航空电子系统的多个组件进行故障诊断，并返回最可能的故障源。

**示例：**

```
输入：[{"component": "GPS", "status": "fail"}, {"component": "ADC", "status": "ok"}, {"component": "COM", "status": "fail"}]
输出：["GPS", "COM"]
```

**答案：** 

```go
func FaultDiagnosis(failures []map[string]string) []string {
    var faults []string
    var count map[string]int = make(map[string]int)

    for _, f := range failures {
        if f["status"] == "fail" {
            count[f["component"]]++
        }
    }

    for c, v := range count {
        if v > 1 {
            faults = append(faults, c)
        }
    }

    return faults
}
```

**解析：** 通过遍历故障列表，统计每个组件的失败次数，如果某个组件在多个故障记录中均出现，则可能存在更严重的故障，将其加入最可能故障源列表。

#### 题目3：最小飞行路径

**题目描述：** 给定一组城市及它们之间的飞行时间，设计算法找到从起点到终点的最小飞行路径。

**示例：**

```
输入：{"A": {"B": 2, "C": 6}, "B": {"A": 2, "C": 1}, "C": {"A": 6, "B": 1}}
起点：A
终点：C
输出：2
```

**答案：** 使用深度优先搜索（DFS）结合剪枝策略来寻找最短路径。

**解析：**

```go
func MinFlightTime(airport map[string]map[string]int, start string, end string) int {
    var dfs func(string) int
    dfs = func(city string) int {
        if city == end {
            return 0
        }
        minTime := math.MaxInt32
        for nextCity, time := range airport[city] {
            if time < minTime {
                minTime = time
                dfs(nextCity)
            }
        }
        return minTime
    }
    return dfs(start)
}
```

#### 题目4：系统性能优化建议

**题目描述：** 提供一组系统性能指标，设计算法给出优化建议。

**示例：**

```
输入：CPU使用率：80%
内存使用率：90%
I/O等待时间：30ms
输出：优化建议
```

**答案：** 

```go
func PerformanceOptimization(cpu, mem, ioWait time.Duration) string {
    var advice string
    if cpu > 90 {
        advice += "考虑增加CPU资源；"
    }
    if mem > 90 {
        advice += "考虑增加内存资源；"
    }
    if ioWait > 20 {
        advice += "优化I/O操作，如使用缓存或优化数据库查询。"
    }
    return advice
}
```

**解析：** 根据系统性能指标，提供针对性的优化建议。对于高CPU和内存使用率，可以考虑增加硬件资源；对于较长的I/O等待时间，可以考虑优化I/O操作。

**示例：**

```
输入：航班信息（如：航班号、出发地、目的地）
输出：同步后的航班信息
```

**答案：**

```go
func SyncFlightInfo(flights <-chan Flight) {
    for flight := range flights {
        // 同步航班信息
        // 例如更新航班状态、更新乘客名单等
        // 此处模拟为睡眠一段时间表示同步操作
```
