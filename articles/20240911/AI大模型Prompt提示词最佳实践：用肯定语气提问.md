**答案解析：** 优化Prompt的关键在于确保其包含足够的上下文信息，同时避免冗余和歧义。使用肯定语气可以减少模型的困惑，提高生成回答的准确性。例如：

### 4. 使用肯定语气编写一个简单的聊天机器人

**题目描述：** 编写一个聊天机器人，能够接收用户的输入并使用肯定语气回答。

**答案解析：** 在编写聊天机器人时，可以使用肯定语气的模板来回答用户的问题，确保回答积极、友好。

**源代码实例：**

```python
def chatbot(response):
    prompts = {
        "你好": "你好！有什么可以帮助你的吗？",
        "再见": "祝您愉快！再见！",
        "天气": "今天天气非常好，适合外出活动。"
    }
    return prompts.get(response, "我不知道你的问题，但我会尽力帮助你。")

# 示例使用
user_input = input("你好！有什么可以帮助你的吗？")
print(chatbot(user_input))
```
