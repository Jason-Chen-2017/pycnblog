#### 三、源代码实例

以下是一个使用 Python 实现的简单文本分类器的源代码实例：

```python
import numpy as np
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB

# 加载数据集
data = [
    ("政治", "中国领导人在会议上强调经济安全"),
    ("经济", "股市暴跌引发投资者担忧"),
    ("体育", "C罗打破进球纪录"),
]

# 分割数据集
X, y = zip(*data)

# 特征提取
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(X)

# 模型训练
model = MultinomialNB()
model.fit(X, y)

# 模型评估
y_pred = model.predict(X)
accuracy = np.mean(y_pred == y)
print("Accuracy:", accuracy)
```

这个实例使用 sklearn 库实现了一个简单的文本分类器，包括数据预处理、特征提取、模型训练和模型评估等步骤。这个例子仅供示例，实际应用中可能需要更复杂的特征提取和模型选择。
