**题目：** 提出解决硅谷高房价问题的方案。

#### 1. 如何计算硅谷地区的房价中位数？

**题目：** 编写一个函数，计算硅谷地区房价的中位数。

**答案：**

```python
def median_house_price(house_prices):
    sorted_prices = sorted(house_prices)
    n = len(sorted_prices)
    if n % 2 == 0:
        return (sorted_prices[n // 2 - 1] + sorted_prices[n // 2]) / 2
    else:
        return sorted_prices[n // 2]
```

**解析：** 该函数首先对房价列表进行排序，然后根据列表长度判断中位数的位置，返回中位数。

#### 2. 如何分析硅谷地区房价分布的稳定性？

**题目：** 编写一个函数，分析硅谷地区房价分布的稳定性，输出稳定性指标。

**答案：**

```python
import numpy as np

def stability_index(house_prices):
    mean_price = np.mean(house_prices)
    var_price = np.var(house_prices)
    stability = mean_price / np.sqrt(var_price)
    return stability
```

**解析：** 该函数计算房价的均值和方差，然后通过稳定性指标（均值与方差的比值）来评估房价分布的稳定性。
