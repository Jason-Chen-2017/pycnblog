### 算法编程题库

#### 1. 数据清洗

**题目：** 给定一个包含缺失值和噪声的文本数据集，请实现一个数据清洗函数，去除缺失值和噪声。

**答案：** 
```python
import pandas as pd

def data_cleaning(data):
    # 去除缺失值
    data = data.dropna()
    # 去除噪声
    data = data[data.apply(lambda x: not x.str.contains('噪声').any()])
    return data
```

#### 2. 特征工程

**题目：** 给定一个文本数据集，请实现一个特征工程函数，将文本数据转换为数值特征。

**答案：** 
```python
from sklearn.feature_extraction.text import TfidfVectorizer

def feature_engineering(data):
    vectorizer = TfidfVectorizer()
    features = vectorizer.fit_transform(data)
    return features
```
