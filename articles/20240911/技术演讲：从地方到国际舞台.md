### 5. 编写一个算法，用于分析演讲中的关键词密度。

**题目描述：**
给定一段文本和一组关键词，计算每个关键词在文本中的出现频率。

**解析：**
```python
from collections import Counter

def keyword_density(text, keywords):
    text = text.lower()
    keyword_counts = Counter()
    for keyword in keywords:
        keyword = keyword.lower()
        keyword_counts[keyword] = text.count(keyword)
    return keyword_counts

# 示例
text = "The quick brown fox jumps over the lazy dog"
keywords = ["the", "quick", "brown", "fox"]
density = keyword_density(text, keywords)
print("关键词密度：", density)
```

### 9. 如何使用数据分析来评估演讲的效果？

**题目描述：**
编写一个算法，分析演讲后的反馈数据，评估演讲的效果。

**解析：**
```python
from collections import Counter

def assess_speech效果(feedbacks):
    feedbacks = [feedback.lower() for feedback in feedbacks]
    positive_feedbacks = [feedback for feedback in feedbacks if "good" in feedback or "excellent" in feedback]
    negative_feedbacks = [feedback for feedback in feedbacks if "bad" in feedback or "poor" in feedback]

    positive_count = len(positive_feedbacks)
    negative_count = len(negative_feedbacks)
    total_count = positive_count + negative_count

    positive_rate = positive_count / total_count
    negative_rate = negative_count / total_count

    return positive_rate, negative_rate

# 示例
feedbacks = ["演讲很好", "内容有些无聊", "演讲很精彩", "不感兴趣"]
positive_rate, negative_rate = assess_speech效果(feedbacks)
print("正面反馈率：", positive_rate)
print("负面反馈率：", negative_rate)
```
