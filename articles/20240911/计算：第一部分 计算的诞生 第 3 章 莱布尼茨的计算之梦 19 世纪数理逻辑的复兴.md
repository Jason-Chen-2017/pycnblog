##### 1. 计算二进制和十进制之间的转换
**题目：** 编写一个函数，将十进制数转换为二进制数。

**答案：** 

```python
def decimal_to_binary(n):
    if n == 0:
        return "0"
    binary = ""
    while n > 0:
        binary = str(n % 2) + binary
        n = n // 2
    return binary

# 测试
print(decimal_to_binary(10))  # 输出 1010
```

**解析：** 该函数使用循环将十进制数转换为二进制数，每次循环将数字除以2，取余数，然后将余数加到二进制字符串的前面。

##### 2. 计算二项式系数
**题目：** 编写一个函数，计算二项式系数 C(n, k)。

**答案：**

```python
def binomial_coefficient(n, k):
    if k > n:
        return 0
    if k == 0 or k == n:
        return 1
    result = 1
    for i in range(1, k+1):
        result *= (n-i+1) // i
    return result

# 测试
print(binomial_coefficient(5, 2))  # 输出 10
```

**解析：** 该函数使用动态规划方法计算二项式系数，通过循环计算每个中间结果，并最终返回结果。
