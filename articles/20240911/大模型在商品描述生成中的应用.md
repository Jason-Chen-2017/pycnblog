#### 7. 商品描述文本质量分析

**题目：** 编写一个Python脚本，使用BERT模型分析商品描述的文本质量。

**答案：**

```python
from transformers import BertTokenizer, BertModel
import torch

tokenizer = BertTokenizer.from_pretrained("bert-base-uncased")
model = BertModel.from_pretrained("bert-base-uncased")

def analyze_quality(text):
    inputs = tokenizer(text, return_tensors="pt")
    with torch.no_grad():
        outputs = model(**inputs)
    last_hidden_state = outputs.last_hidden_state[:, 0, :]

    # 使用简单的BERT特征进行文本质量评估（实际应用中需要更复杂的模型）
    quality_score = torch.mean(last_hidden_state).item()
    return quality_score

description = "这是一款功能强大的智能手机，拥有高清屏幕、长续航电池和高清摄像头。"
quality = analyze_quality(description)
print(f"文本质量分数：{quality}")
```
