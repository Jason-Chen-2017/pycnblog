**源代码实例：**

```python
# Python 示例：检测平台经济数据中的异常值
import numpy as np

def detect_anomalies(data, threshold=3):
    mean = np.mean(data)
    std = np.std(data)
    anomalies = []

    for i, value in enumerate(data):
        z_score = (value - mean) / std
        if np.abs(z_score) > threshold:
            anomalies.append(i)

    return anomalies

# 示例数据
data = np.array([1, 2, 2, 3, 4, 100, 5, 6, 7, 8, 9])

# 检测异常值
anomalies = detect_anomalies(data)
print("异常值的位置：", anomalies)
```

**解析**：该示例中的 `detect_anomalies` 函数用于检测平台经济数据中的异常值，基于 Z-Score 方法。通过计算每个数据点的 Z-Score，如果 Z-Score 超过设定的阈值，则认为该数据点为异常值。阈值可以根据实际情况进行调整。
