#### 3. 编写一个函数，实现数据去重功能，并确保去重过程中不丢失原有数据。

**答案：**

```python
def data_de duplication(data):
    unique_data = []
    for item in data:
        if item not in unique_data:
            unique_data.append(item)
    return unique_data
```

**解析：** 该函数通过遍历输入数据，将不重复的元素添加到新列表`unique_data`中，从而实现数据去重。

#### 4. 编写一个函数，计算一组数据的平均数、中位数和众数。

**答案：**

```python
def statistical_analysis(data):
    avg = sum(data) / len(data)
    sorted_data = sorted(data)
    mid_index = len(sorted_data) // 2
    median = sorted_data[mid_index] if len(sorted_data) % 2 != 0 else (sorted_data[mid_index - 1] + sorted_data[mid_index]) / 2
    mode = max(set(data), key=data.count)
    return avg, median, mode
```

**解析：** 该函数首先计算输入数据的平均数，然后对数据进行排序，找出中位数和众数。
