#### 面试题 2：自然语言处理（NLP）
**题目：** 请解释词嵌入（word embeddings）在自然语言处理中的作用。

**答案：** 词嵌入是将自然语言中的单词映射到高维向量空间的技术。词嵌入能够捕捉词语的语义信息，使得相似词在向量空间中距离较近。常用方法包括 Word2Vec、GloVe 等。

**解析：** 词嵌入是 NLP 中的一项关键技术，它使得机器能够更好地理解自然语言的语义。了解词嵌入的基本原理和应用，有助于深入理解 NLP 的相关技术。

#### 编程题 1：神经网络实现
**题目：** 编写代码实现一个简单的神经网络，用于手写数字识别。

**答案：** 神经网络实现代码如下：

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def forward_pass(inputs, weights):
    return sigmoid(np.dot(inputs, weights))

# 示例：训练一个简单的神经网络进行手写数字识别
inputs = np.array([1, 0, 1])
weights = np.array([0.1, 0.2, 0.3])

output = forward_pass(inputs, weights)
print("Output:", output)
```

**解析：** 这个简单的神经网络通过前向传播计算输出，并使用 sigmoid 函数进行激活。手写数字识别是神经网络应用的一个经典例子，实现这个算法有助于理解神经网络的基本原理。
