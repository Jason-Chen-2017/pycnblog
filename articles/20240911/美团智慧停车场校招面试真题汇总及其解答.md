#### 一、编程题

**1. 车位分配问题**

**题目描述：** 假设有一个停车场，停车场有N个车位，同时有M辆车需要停入。每辆车都有优先级，优先级高的车辆有优先停入的权利。请实现一个函数，返回每辆车停入后的剩余车位。

**输入：**
- 车位总数N
- 车辆数M
- 每辆车的优先级（从高到低）

**输出：** 剩余车位

**示例：**
```
输入：N = 5, M = 3, 车辆优先级 = [1, 2, 3]
输出：[2, 1, 0]
```

**答案：**
```go
func parkCars(N int, M int, priorities []int) []int {
    result := make([]int, M)
    priorityQueue := &heap.Heap{items: make([]int, 0, M)}
    
    for i := 0; i < M; i++ {
        priorityQueue.push(priorities[i])
    }
    
    for i := 0; i < N; i++ {
        if priorityQueue.isEmpty() {
            break
        }
        
        priority := priorityQueue.pop()
        result[i] = N - priority
    }
    
    return result
}
```

**解析：** 使用优先队列实现。首先将所有车辆的优先级插入优先队列中，然后依次取出优先级最高的车辆，计算剩余车位。

**2. 车位数量统计**

**题目描述：** 假设停车场有多个楼层，每个楼层有多个车位。请实现一个函数，返回每个楼层剩余车位数。

**输入：**
- 楼层数L
- 楼层车位数数组

**输出：** 每层剩余车位数

**示例：**
```
输入：L = 3, 车位数数组 = [5, 3, 2]
输出：[2, 0, 1]
```

**答案：**
```go
func countEmptySpaces(L int, spaces []int) []int {
    result := make([]int, L)
    
    for i := 0; i < L; i++ {
        result[i] = spaces[i] - 1
    }
    
    return result
}
```

**解析：** 直接计算每个楼层的剩余车位数。

#### 二、算法题

**1. 最短路径问题**

**题目描述：** 给定一个包含 M 个顶点和 N 条边的无向图，求图中任意两个顶点之间的最短路径。

**输入：**
- 顶点数M
- 边数N
- 顶点之间的边（二维数组）

**输出：** 最短路径（二维数组）

**示例：**
```
输入：M = 4, N = 4, 边 = [[0, 1, 2], [0, 2, 3], [1, 2, 1], [1, 3, 4]]
输出：[[0, 0, 3, 4], [1, 1, 2, 4], [2, 2, 3, 4], [3, 3, 3, 4]]
```

**答案：**
```go
func shortestPath(M int, N int, edges [][]int) [][]int {
    distances := make([][]int, M)
    for i := range distances {
        distances[i] = make([]int, M)
        distances[i][i] = 0
    }
    
    for _, edge := range edges {
        u, v, w := edge[0], edge[1], edge[2]
        distances[u][v] = w
        distances[v][u] = w
    }
    
    for k := range distances {
        for i := range distances {
            for j := range distances {
                distances[i][j] = min(distances[i][j], distances[i][k]+distances[k][j])
            }
        }
    }
    
    return distances
}
```

**解析：** 使用Floyd-Warshall算法求最短路径。

**2. 背包问题**

**题目描述：** 给定一个物品列表和价值列表，以及一个最大容量，求背包能容纳的最大价值。

**输入：**
- 物品数量N
- 每个物品的重量和价值
- 最大容量

**输出：** 背包能容纳的最大价值

**示例：**
```
输入：N = 4, weights = [1, 2, 3, 4], values = [1, 6, 10, 16], capacity = 5
输出：22
```

**答案：**
```go
func knapsack(N int, weights []int, values []int, capacity int) int {
    dp := make([][]int, N+1)
    for i := range dp {
        dp[i] = make([]int, capacity+1)
    }
    
    for i := 1; i <= N; i++ {
        for w := 1; w <= capacity; w++ {
            if weights[i-1] <= w {
                dp[i][w] = max(dp[i-1][w], dp[i-1][w-weights[i-1]]+values[i-1])
            } else {
                dp[i][w] = dp[i-1][w]
            }
        }
    }
    
    return dp[N][capacity]
}
```

**解析：** 使用动态规划求解背包问题。定义一个二维数组dp，其中dp[i][w]表示前i个物品在容量为w时的最大价值。
