### 2. 动态规划：能耗预测
**题目：** 使用动态规划方法，预测数据中心未来一天的能耗。

**答案：** 可以将动态规划应用于时间序列预测，建立能耗与时间的关系模型。

```python
def energy_prediction(history):
    n = len(history)
    dp = [[0] * (n+1) for _ in range(n+1)]

    for i in range(1, n+1):
        for j in range(1, n+1):
            dp[i][j] = dp[i-1][j-1] + (history[i-1] - history[j-1])

    return dp[n][n]
```
