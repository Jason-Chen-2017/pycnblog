**题目：** 编写一个函数，使用K-means算法对商品属性进行聚类，并返回聚类结果。

```python
import numpy as np

def k_means(data, k):
    # 初始化聚类中心
    centroids = data[np.random.choice(data.shape[0], k, replace=False)]
    
    # 迭代计算聚类中心
    while True:
        # 计算每个样本的簇分配
        labels = np.argmin(np.linalg.norm(data[:, np.newaxis] - centroids, axis=2), axis=1)
        
        # 更新聚类中心
        new_centroids = np.array([data[labels == i].mean(axis=0) for i in range(k)])
        
        # 检查聚类中心是否收敛
        if np.linalg.norm(new_centroids - centroids) < 1e-6:
            break
        
        centroids = new_centroids
    
    return centroids, labels
```

**解析：** K-means算法是一种基于距离的聚类算法，通过迭代计算聚类中心，直到聚类中心收敛。这个函数实现了K-means算法的基本流程，包括初始化聚类中心、计算每个样本的簇分配、更新聚类中心和检查聚类中心是否收敛。通过这个函数，可以对商品属性进行聚类，从而辅助构建商品属性层次结构。

def decision_tree_classification(data, labels, test_size=0.2):

    X_train, X_test, y_train, y_test = train_test_split(data, labels, test_size=test_size)

    predictions = classifier.predict(X_test)
    
    # 计算准确率
    accuracy = (predictions == y_test).mean()
    
    return classifier, accuracy
