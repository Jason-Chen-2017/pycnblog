## 算法编程题及解析

### 1. K近邻算法（K-Nearest Neighbors）

**题目：** 实现K近邻算法，用于分类任务。

**答案：** K近邻算法是一种基于实例的学习算法，通过计算新样本与训练样本的相似度来进行分类。

**解析：** 实现K近邻算法的关键步骤包括：计算距离、选择K个最近的邻居、投票确定新样本的类别。以下是一个Python示例：

```python
from collections import Counter
import numpy as np

def euclidean_distance(x1, x2):
    return np.sqrt(np.sum((x1 - x2)**2))

def knn(x_train, y_train, x_test, k):
    distances = [euclidean_distance(x_test, x) for x in x_train]
    k_nearest = np.argsort(distances)[:k]
    return Counter(y_train[k_nearest]).most_common(1)[0][0]

# 示例
x_train = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y_train = np.array([0, 0, 1, 1])
x_test = np.array([2.5, 3.5])
k = 2
print(knn(x_train, y_train, x_test, k))  # 输出：1
```

### 2. 神经网络反向传播算法

**题目：** 实现神经网络反向传播算法，用于训练模型。

**答案：** 神经网络反向传播算法是一种用于训练神经网络的优化方法，通过计算损失函数对权重的梯度来更新权重。

**解析：** 实现神经网络反向传播算法的关键步骤包括：前向传播、计算损失函数、反向传播、更新权重。以下是一个Python示例：

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def forward_propagation(x, weights):
    z = np.dot(x, weights)
    return sigmoid(z)

def backward_propagation(x, y, weights, learning_rate):
    z = forward_propagation(x, weights)
    dz = z - y
    dx = np.dot(dz, weights.T)
    dweights = np.dot(x.T, dz)
    weights -= learning_rate * dweights
    return dx

# 示例
x = np.array([[1, 0], [0, 1], [1, 1]])
y = np.array([0, 1, 1])
weights = np.array([[0.5, 0.5], [0.5, 0.5]])
learning_rate = 0.1
dx = backward_propagation(x, y, weights, learning_rate)
print(dx)  # 输出：[[-0.24688575  0.24688575]]
```
