# 2.基本概念和术语
## 2.1 域名系统DNS
域名系统（Domain Name System，DNS）是因特网的一项服务，它用于将主机名转换成IP地址。当在浏览器输入一个域名时，计算机首先会查询本地hosts文件是否存在相应的记录，如果没有则向DNS服务器请求解析该域名对应的IP地址。DNS服务器根据收到的域名信息返回相应的IP地址。
## 2.2 万维网WWW
万维网（World Wide Web，WWW）是一系列服务器端软件、网络协议和标准的集合，通过互联网访问到互联网资源。WWW由许多网站组成，每个网站由众多互相链接的页面构成。任何人都可以免费发布自己的网页供他人浏览。
## 2.3 IP地址
IP地址（Internet Protocol Address）是一个用四个字节的数字表示的计算机在网络中的唯一身份标识符。每台设备都必须分配一个唯一的IP地址，无论是家用路由器、服务器还是手机，都需要配置IP地址。
## 2.4 HTTP协议
HTTP（HyperText Transfer Protocol，超文本传输协议）是一种用于分布式、协作式和超媒体信息系统的应用程序协议。它使得 World Wide Web 上的数据交换更加有效、迅速，并允许交互性增强的多媒体信息的共享。HTTP协议由请求方法、状态码、URI、首部字段、内容及其他元素组成。
