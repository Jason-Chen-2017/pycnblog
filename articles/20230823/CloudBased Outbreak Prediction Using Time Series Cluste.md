# 1.简介
  

随着新型冠状病毒肆虐全球，如何快速准确地预测并阻断传染病的蔓延，已经成为一项亟待解决的问题。传统的预测方法主要基于实时感染者数量的统计模型，但这些模型存在一些局限性，比如对于海量数据的处理速度慢、对异常值敏感等。另一种基于机器学习的方法是时间序列聚类（time series clustering）法，该方法可以将许多不同的时序数据集中在一起，发现隐藏的模式或隐藏的时间段，从而预测流行病的扩散路径。时间序列聚类技术能够识别出数据中的常规模式和异常模式，并使用机器学习算法对其进行分类。因此，通过这种方式，云计算环境下的传染病疫情预测可以提高准确性及时性，缩短预测周期，降低资源消耗，提升预测结果的精确度。

本文试图将传统的基于感染者数量的预测方法与时间序列聚类法相结合，形成新的无监督预测模型——即通过聚类时间序列数据得到各个区间的“热度”和“离群”程度，进而确定每个时间点是否发生“爆发”事件。此外，为了减少参数设置的复杂度，本文还采用了基于多元高斯分布的聚类方法，可以自动选择合适的模型参数。最后，本文设计了一系列实验，对比两种方法的预测性能。
# 2.基本概念
## 2.1 传染病
传染病是指由细菌或病毒感染引起的慢性病，它通常会影响健康的一部分或所有人。在实际应用中，通过观察人群中各种传染病的发病情况、人群因某种传染病产生的症状、研究不同传染病之间的联系，我们可以了解目前世界上的各种传染病的特征、发病率和传播路径。根据其传播方式的不同，传染病可分为以下几类：
- 接触传染病：这是最常见且最严重的传染病。如：鼠疫、蚊子传播病、流感、艾滋病等。这些传染病病原微生物通过人体接触进入人体，并经过粘膜，神经系统传导到呼吸道、咽喉、肺泡、泌尿道等组织，致使感染者出现发热、乏力、咳嗽、胸闷、心悸等症状。随着接触传染病的逐渐普及，危害越来越大，有关部门也越来越关注。
- 潜伏期传染病：潜伏期传染病病原微生物是通过人体免疫系统排除的。如：狂犬病、流行性腮腺炎、登革热、霍乱等。这些病毒既不直接感染身体，又无法感染患者，只能潜伏于人体的某些组织中。当感染者得不到充足治疗或体力不支时，潜伏期传染病病毒可能导致更严重的后果。
- 内源性传染病：这种病原微生物能够在体内诞生并在先天基因上隐匿，传染给接触者，并在免疫系统抵抗。如：大白斑病毒、金黄色葡萄球菌等。大白斑病毒与肝脏微生物相似，但潜伏期较短，传染性强，主要由家庭暴露的产妇所感染，也能传给儿童。
- 噤声传染病：传染病病原微生物可以在空气中穿透人体的呼吸道，使感染者麻木、昏厥或瘫痪。如：麻疹、流行性感冒等。一般来说，噤声传染病病原微生物具有传播性强、容易感染人群的特点。
