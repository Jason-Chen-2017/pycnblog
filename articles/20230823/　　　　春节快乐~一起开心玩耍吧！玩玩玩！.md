# 3.基础概念及术语
## 3.1 概念介绍
　　什么是机器学习？为什么要进行机器学习？机器学习是什么样的？机器学习的定义又是什么？机器学习可以解决什么问题？接下来，我们一起来看一下这些问题的答案。
### 3.1.1 机器学习概念
　　机器学习(Machine Learning)是关于计算机如何利用数据来获取新知识，并使其作出预测或决策的一种方法。机器学习系统通过反馈并不断改进自身的模式来实现学习。机器学习的目的就是为了让机器像人一样能够“学习”（也就是理解）数据、分析数据、预测结果和执行任务。换句话说，机器学习是指让计算机能够自己发现并解决问题。

　　由于数据量越来越大、计算能力越来越强、需求也越来越多，许多实际的问题都需要依赖于机器学习技术来得到有效的解决方案。近些年来，机器学习技术引起了广泛的关注。与此同时，人工智能、生物信息学、控制工程等领域也都产生了重大突破。因此，机器学习领域的研究与发展正处在一个重要的转折点上。

　　机器学习所涉及到的知识面非常广泛，涵盖了从图像识别、文本处理、语音识别、行为分析、推荐系统、个性化系统、金融风险管理、预测分析等多个领域。机器学习可以用来解决各种不同的问题，如医疗诊断、图像分析、文本分类、垃圾邮件过滤、网页内容审核、股票交易预测、广告推荐、零售购物推荐等。

　　机器学习存在以下几个关键属性：

　　1. 数据驱动：机器学习的关键是获得大量的数据并持续不断地试错，才能找到最佳的解决方案。

　　2. 自动化：机器学习的目标是开发自动化的计算机程序，能够根据新的数据集来做出调整，不断优化性能。

　　3. 模型解释：机器学习的模型需要具有良好的可解释性，能够说明它背后的机制。

　　4. 群体智能：机器学习能够处理复杂的多样性，能够将不同人群的特征结合起来，形成共同的模式，然后针对该模式进行决策。

　　5. 高容量计算：机器学习的计算资源通常相当昂贵，而且往往需要大量的训练数据来完成模型训练。所以，对于那些计算能力很强的超级计算机来说，机器学习的使用还是比较有限的。

　　总的来说，机器学习是一种与人类智能紧密相关的科学，它是人工智能（Artificial Intelligence）的一个分支。机器学习是指让计算机学习，并借此实现智能化的过程，从而利用经验提升判断力、解决问题和行动力。机器学习的关键是取得数据驱动、自动化、模型解释、群体智能和高容量计算等多种能力，构建有效的模型，以解决复杂的、多样性的问题。
