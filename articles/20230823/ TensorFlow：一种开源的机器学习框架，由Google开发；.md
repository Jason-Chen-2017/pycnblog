# 1.简介
  

TensorFlow是一个开源的、基于数据流图（data flow graphs）的机器学习系统，其最初由Google开发，是一个谷歌内部使用的开源项目。在过去几年里，它已经被多个公司采用，包括微软、雅虎、百度、腾讯等。它提供了很多强大的机器学习算法模块，如神经网络、卷积神经网络、递归神经网络、支持向量机等，并且可以轻易地部署到多种平台上。而且它的高性能计算能力也使得它可以在处理高维度、实时数据流时表现出色。从这点来说，TensorFlow无疑是一个非常优秀的机器学习框架。本文将详细介绍TensorFlow。
# 2.基本概念术语说明
## 2.1 数据流图（Data Flow Graphs）
TensorFlow中的数据流图主要用来描述机器学习模型如何进行计算的过程。通过图结构的表示，可以直观地看到各个算子之间的依赖关系，并可以方便地对图进行优化和改进。数据流图的每个节点代表了某个特定的运算，比如矩阵乘法、加法或其他任意操作。边缘则代表了它们之间的数据流动方向。这样，整个模型就可看做由这些运算单元组成的有向图。如下图所示：
在数据流图中，每个节点都代表了一个张量(tensor)，而张量是多维数组。张量可以是一个标量，也可以是一个具有若干维度的多维数组。例如，一个张量可以是表示图像的像素值的矩阵，也可以是表示文本序列的向量。为了便于理解，以下我们用符号$\bf{x}$,$\bf{y}$,$\bf{z}$来分别表示三个张量。每个节点的输出张量可能与输入张量不完全相同，因为每当节点执行运算时，其会生成新的张量作为输出。
## 2.2 会话（Session）
TensorFlow中的Session用于管理执行图中的运算。Session能够将图中定义的变量和运算结果的值存储在内存中，并负责运行图中的指令。创建Session之后，可以通过调用run()方法来启动图的执行。Session对象一般会保存在代码中，以便于在不同的阶段复用。
```python
with tf.Session() as sess:
  # 使用张量初始化op
  x = tf.constant([...])
  y = tf.constant([...])
  
  # 创建op
  z = tf.add(x, y)

  # 执行op
  result = sess.run(z)

print(result)
```
上面代码创建了一个常量Op，然后创建一个加法Op，将两个输入张量相加。最后，它利用Session对象的run()方法来运行图，并返回结果。运行结束后，将打印出运算结果。
## 2.3 占位符（Placeholders）
占位符用于表示在实际运行中要传入数据的地方。在创建图之前，我们需要给每个输入张量指定一个有效的占位符。占位符可以用来输入不同大小的数据，或者是不同类型的数据。例如：
```python
a = tf.placeholder(tf.float32, shape=[None, 784], name='input')
b = tf.Variable(tf.zeros([784, 10]))
output = tf.matmul(a, b)
```
这里，a就是一个占位符，用来表示输入数据的地方。shape参数告诉TensorFlow该占位符期望接受多少维的数据，此处期望接收2维的浮点型数据，第一维表示样本数量，第二维表示每个样本的784维特征值。name参数给予占位符一个名称。b就是一个常量张量，用来表示权重矩阵。为了完成矩阵乘法操作，还需要将输入张量a和权重矩阵b连接起来，得到输出张量output。在执行的时候，我们需要传入实际的数据才能获得正确的输出。
