# 2.历史及特点
## 1.1 发展史
Python 从它的创始人 Guido van Rossum 发明于1989年，被设计用于编写自动化脚本。由于其“简单”、“易于阅读”、“交互式编程”、“跨平台”等特性，受到许多程序员和计算机爱好者的青睐，成为了一种高效、灵活且富有表现力的编程语言。

Python 第一次面世是在1991年，版本为 0.9.0，此后在 1994 年发布了 1.0 版。截至目前，已经发布了多个版本，主要分为 2.x 版和 3.x 版。

2000年，首个基于 Python 的 Web 框架 Flask 在世界范围内流行开来，它利用简单而有效的语法，轻松地实现了 Web 服务的快速搭建。

2003年，Django 出现，它是一款免费、开源、功能丰富的 Web 框架，拥有完整的 MVC 架构，支持多种数据库，是一个优秀的 Web 开发框架。

2007年，Python 2.x 进入维护模式，不再新增新特性，只有 bug 修复和性能优化。

2010年，Guido 病逝，他在 Python 的社区引起了轩然大波，“Python 之父”的称号由此获得确认。此时，全世界诸多编程语言纷纷加入或退出 Python 的阵营。

2018年，Python 3.x 版本正式发布，并带来了很大的改动。比如 print() 函数的改变，整数除法等运算符的更新，以及 Unicode 的默认支持等等。

## 1.2 特点
- **简单：** Python 是一种小型、高层次的编程语言，鼓励表达式而不是声明语句。因此，许多初学者认为 Python 比较难上手。但实际上，掌握一些基本语法规则，就可以轻松上手。
- **易读：** Python 拥有比其他语言更容易阅读的代码风格，这一点也促进了程序员之间的交流。对于习惯使用其他编程语言的人来说，也可以快速理解 Python 代码。
- **互动式编程:** Python 支持交互式编程环境，可以直接输入命令，立即看到运行结果。这个特性给初学者提供了方便，他们不需要编译或者安装就可以体验到编程的乐趣。
- **跨平台：** Python 可以运行于各种不同的操作系统平台，这意味着你可以编写一次，到处运行。
- **丰富的库和模块:** 有数量庞大的第三方库和模块供您选择。涵盖了文件、网络、数据库、文本处理、图形界面、科学计算等众多领域。有经验的程序员可以用这些库快速完成工作。
- **可扩展性：** 通过类和工具箱，您可以高度定制 Python 的环境，创建自己需要的功能。
- **文档完善：** Python 提供详尽的文档，学习起来不会感到困难。而且，通过访问 Python 官网，可以找到很多资源帮助您提升技能。
