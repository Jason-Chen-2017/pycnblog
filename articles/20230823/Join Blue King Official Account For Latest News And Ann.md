# 1.简介
  

蓝鲸智云（BlueKing）是一家基于人工智能、大数据、云计算、区块链等技术的企业级解决方案供应商，在内部管理系统、审计系统、网盘系统、工作流系统、监控系统等多个领域均有覆盖，业务覆盖范围广泛且涵盖电子政务、合同管理、知识产权、人力资源管理、SCM、ECM、ERP等多个领域。近年来，蓝鲸智云全面推进了自主研发的“智慧运营”产品线，赋能各行各业企业实现数字化转型、创新驱动、协同管理，助力企业构建“一体化、流程化、绿色化”的管理运营能力。2019年1月，蓝鲸智云宣布完成IPO，并发行价值12.5亿元人民币。截止目前，蓝鲸智云已服务于数百家客户，运营规模约2700万套/年，市场份额占蓝皮云总市场份额的50%以上。

# 2.核心概念及术语
- **IAM(Identity and Access Management)** 身份与访问管理系统:用于管理用户访问权限、角色管理等；
- **XShell**: Linux系统上一款功能强大的SSH客户端，可以替代传统的SSH工具进行终端连接、文件传输、命令执行等；
- **XManager**: 蓝鲸智云可视化运维平台，主要面向云平台、容器平台、物理机平台等进行自动化运维的部署和管理，支持轻量化安装部署；
- **GSE Agent**: GSE是蓝鲸智云公开的开源作业平台，GSE Agent 是运行在每个主机上的守护进程，负责将计划任务下发到Agent所在主机；
- **ESB(Enterprise Service Bus)**: 企业服务总线，是蓝鲸智云开发的一套分布式微服务通信框架，采用MQ中间件实现服务之间的数据交换，支持跨域调⽤，包括RPC、HTTP、MQ、Redis等协议，提供了多种扩展机制，满足复杂场景下服务调用需求；
- **BK Repository**: 蓝鲸制品库，是一个开源的制品库管理系统，包括代码、配置管理、镜像等多个仓库，支持蓝盾扫描、风险管控、管理生命周期等能力，兼容Maven、Docker等多种语言；
- **作业平台(GSE)**: 蓝鲸智云公开的开源作业平台，支持通过Web界面或API的方式创建和执行各种类型的任务，具备完善的权限控制、调度策略和任务监控功能；
- **GitRepo**: GitRepo 是蓝鲸智云云管平台中的一项服务，提供代码托管服务，支持git和svn两种版本控制协议；
- **Gsekit**: Gsekit 是蓝鲸智云团队开源的一套基于 Python 的作业平台 SDK，提供了一系列 API ，方便用户快速集成到自己的程序中。同时，也提供了一些公共组件，例如作业执行器、脚本运行器、数据订阅等，以便于用户复用或扩展。
