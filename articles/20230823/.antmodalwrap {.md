# 1.简介
  

## Ant Design Pro 是 Ant Design 的在线可视化搭建平台，可以帮助你快速地构建中后台应用、网站等前端页面，基于 Ant Design 设计语言和技术栈。它内置了最新的技术前沿、设计模式、最佳实践等资源，助力企业打造设计一流、功能丰富的产品和服务。你可以通过 Pro 在线工具轻松创建中后台前端应用，并在项目开发、测试、部署上线之间一键完成，让你更加专注于业务逻辑的创新。同时，Pro 还提供强大的权限管理功能，支持多角色、多租户、自定义数据模型等复杂场景，让你可以灵活应对各种业务需求。

## 为什么需要一个开源中后台前端框架？
Ant Design 是一个 React 组件库，基于 Ant Design 设计语言设计规范，提供了高质量的 UI 设计元素及动效，帮助业务人员提升用户体验和降低研发成本。它的开源社区生态系统也逐步完善，有非常多优秀的开源项目能够参考借鉴。而 Ant Design Pro 则是 Ant Design 在开源领域里的一款重要代表产品，它由 Ant Design 和 Umi 框架组成，Umi 是阿里巴巴集团技术监督部经过长期的探索和实践得出的前端架构，基于 TypeScript/React/dvajs 等主流技术框架实现，极大地方便了开发者的工作流程。

那么为什么要开源 Ant Design Pro？如果大家都依赖 Ant Design 的解决方案的话，是否意味着 Ant Design 将越来越难维护？因此开源社区会选择那些经历时间检验的开源框架，去跟进最新的技术发展趋势，做到不断更新迭代，保持自己所提供的开源产品质量。

值得注意的是，除了 UI 框架之外，阿里集团内部还有很多其他业务线技术团队也使用 Ant Design 来进行界面交互设计，因此开源 Ant Design Pro 能够帮助更多技术团队共同建设中后台前端应用。

## 特点
### 基于最新技术栈打造
- 使用最新的技术栈 React、TypeScript、Less、Umi 搭建的，新版本持续跟踪 React 技术栈的最新进展；
- 提供 Ant Design Vue、Angular 和 AngularJS 的模板，通过官方插件和自定义配置满足不同技术栈的定制需求；
- 支持扩展能力，通过可插拔的插件机制，社区贡献者可以方便地加入自己的模板和配置项，满足不同业务场景的个性化需求。
### 插件机制
- 通过插件机制，将更多业务场景的需求统一管理，降低扩展难度，提升开发效率；
- 内置多个高级模板，从基础模板、中台模板、多端模板等不同维度，满足不同业务场景的需求；
- 可自由定制，提供完整的开发调试环境，配合 VSCode 插件，使得前端开发环境和发布流程更贴近真实开发场景。
### 权限管理功能
- 内置了完整的权限控制体系，包括用户、角色、菜单、权限、接口等，满足企业不同角色和权限需求；
- 支持自定义数据模型，支持租户隔离、不同数据源的接入，满足中大型公司复杂权限管理需求。
### 国际化支持
- 基于 Ant Design 设计语言，国际化语言资源完备，组件翻译风格统一，适配各类浏览器语言；
- 提供友好的国际化开发指引，助力中后台前端应用全球化。
