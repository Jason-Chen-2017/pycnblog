# 1.简介
  

2022年是中华人民共和国建党百年之际，在这个重要的历史关头，我要通过向大家宣读中央政治局委员、中央纪委监委书记任正非于2021年1月至今日在全国政法工作会议上的讲话，为建立新一代信息通信体系作出贡献。这是一份值得认真阅读的重大文件，值得研究学习并引领未来的发展方向。
在过去的一段时间里，面对“去基站、绿色低碳、疫情防控”等方面的各种各样的信息化需求，我国已经开始进入到一个全新的信息通信体系建设时期。这个体系涵盖了网络、大数据、物联网、人工智能、智慧城市、超级计算平台等多个战略性领域。围绕这个体系的研究和创新，将不断推动我国经济社会发展，造福广大人民群众。因此，任正非先生代表党中央及全国人大常委会，亲切地向国内外广大干部群众致以诚挚的问候！
在刚才的讲话中，任正非就新一代信息通信体系的建设，着重阐述了以下七个方面：
1. 数字经济转型升级
数字经济转型升级不是一蹴而就的，需要长时间的积累和沉淀。我国已经率先实现了关键核心技术的创新，如全球移动通信网络全覆盖、无线电、雷达等传感器的研制和应用、核聚变反应堆的设计开发和利用、量子计算、人工智能、大规模并行计算系统的部署等。这些关键技术的突破、整合、迭代、应用，是构建新一代信息通信体系的不可或缺组成部分。数字经济转型还包括资源和产业的转移，如联通、移动通信、互联网、OLED、高性能计算、5G、生物医药等重点领域纷纷布局，力促数字经济快速发展。
2. 大规模宽带传输技术的研发与布局
大规模宽带传输技术在构架、制造、运营、管理、安全和售后等方面均取得了一定的进步。我国已形成由电信、城域网、企业网等多层次承载和集成的多功能宽带通信网络体系。随着我国2G、3G、4G、5G等不同频段的信息通信标准的研制与落地，以及卫星通信、人工智能、机器人、无人机等新兴通信技术的应用，在更高速率的同时也提升了用户的体验。未来，我国将加快推进可穿戴设备、边缘计算、智能交通等新兴领域的智能化应用。
3. 开源开放型云计算平台的建设
当前，云计算的发展方向正在发生重大变化。云计算是一种基于网络技术和服务模型的商用计算服务，它具备灵活、便捷、弹性等优势。基于云计算的创新服务模式正在迅速壮大。以Kubernetes为代表的开源软件定义基础设施（OpenShift）正在成为云计算的主要技术框架。未来，我国将充分发挥云计算平台的优势，积极参与构建和推进开源开放型云计算平台的建设。
4. 智慧城市战略的落地实践
智慧城市作为继互联网和物联网之后，第三次工业革命的重大转折点。从上世纪90年代末开始，我国建成的首个超大型超级城市——长春明月山城，就是我国智慧城市建设和应用的典范。长春明月山城将5G、车联网、人工智能、互联网、区块链等新技术融入其中，使其成为我国智慧城市的标杆。未来，我国将继续推动智慧城市建设、应用和服务，以探索未来智慧城市的形态。
5. 超级计算平台建设的共性和特点
超级计算平台可以由具有海量存储能力的计算存储集群、具有高度算力的计算集群、具有超高速网络连接能力的网络系统、具有海量计算资源的存储系统组成。其核心技术主要包括集群硬件技术、分布式计算技术、超融合异构计算技术、网络架构技术、云计算技术、信息采集技术、数据分析技术、机器学习技术、超级AI技术、大数据处理技术等。未来，我国将围绕超级计算平台的建设和部署展开新的尝试，共同打造具有自主知识产权的科技驱动的新一代超级计算平台。
6. 面向社会的ICT产业的发展
虽然我们仍然处于信息化建设的初期阶段，但数字经济、云计算、大数据、人工智能、超级计算平台等新一代信息通信技术的出现，以及互联网、物联网、智能手机、AR/VR等新一代终端技术的推出，正在为我们的生活和生产方式带来深远的变革。它们共同构成了一个庞大的ICT产业链条。ICT产业链条将在未来十年成为我国经济发展的支柱产业。因此，我们应该以对人类未来和国家发展的伟大贡献为己任，努力建设和完善中国特色的ICT产业体系，为经济发展、社会进步和民族振兴做出更大的贡献。
7. ICT标准体系的建设与完善
2022年，将迎来中国IOT、IIoT、区块链、人工智能、大数据、物联网、智慧城市、超级计算平台等新一代技术体系的刷新。ICT标准体系的建设和完善，将成为推动这一重大技术革命发展的主动力。
总结起来，新一代信息通信体系建设是一场复杂而艰巨的科技革命，必将对我国经济、社会、文化、教育、军事等方面产生深远的影响。任正非先生的这些讲话，以具体的事例，详细地阐释了信息通信体系的建设依托基础研究、制定政策、创新驱动、开放合作共赢的机制，以及探索未来智慧城市、超级计算平台等新一代信息通信技术的发展方向，对于我们应对国际和国内挑战、加强国际合作、推动创新和发展都起到了积极作用。
