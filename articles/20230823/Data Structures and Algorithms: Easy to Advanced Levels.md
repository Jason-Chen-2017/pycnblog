## 2. 数据结构与算法基础知识
### 2.1 数据结构
数据结构（Data Structure）是计算机存储、组织数据的方式。它定义了在内存中如何存储数据、数据之间的关系、数据的运算等，是计算机科学中的基础概念和研究对象。数据结构往往以抽象数据类型（Abstract Data Type，ADT）的形式出现，用来描述各种元素（数据值或数据对象）及该集合的规则和操作。常见的数据结构有栈、队列、链表、树、图、堆、散列表等。不同的数据结构，对于相同的问题都需要不同的算法来实现，因此，熟练掌握各种数据结构和算法是一项十分重要的技能。
### 2.2 数组和链表
数组（Array）是一种最简单的数据结构。它用一段连续的存储空间来存储同一类型的元素，可以用索引（Index）直接访问指定的元素。数组是静态的分配内存，因此当增加或删除元素时，必须重新申请新的内存，效率低下。数组的另一个特点就是插入和删除元素的时间复杂度都是 O(n) 。如下图所示，数组支持随机访问、插入和删除元素，但是缺乏动态扩容功能。当数组大小固定时，可以通过指针改变数组元素的值，这种方式较麻烦。
而链表（Linked List）是另一种常用的动态数据结构。链表由一系列节点组成，每个节点包含数据和一个指向下一个节点的引用。首节点称为头结点（Head），尾节点称为尾节点（Tail）。如下图所示，链表支持动态扩容，能够方便地插入和删除元素。链表也没有预先分配好的数组大小，而是在运行过程中根据需求自动调整内存。但是随机访问元素时间复杂度为 O(n)，需要遍历整个链表。
### 2.3 栈和队列
