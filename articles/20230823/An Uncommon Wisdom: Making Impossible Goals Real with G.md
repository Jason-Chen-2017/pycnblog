# 1.简介
  

> “Goals are dreams with deadlines.” – <NAME>

什么叫做“目标驱动型管理”，是指把目标、计划、任务等的设定和执行当作“自我实现”的首要动力。目标驱动型管理的关键在于构建具有高激励性、有挑战性的目标，并围绕这些目标进行活动。每一个目标都是一段对个人生活和工作的承诺，是由自己主导的奋斗历程。目标驱动型管理不是一项孤立的、单纯的工作或学习方法，而是融入了很多领域的知识、技能、专长和综合素质的综合过程。

目标驱动型管理是一种非常有效的管理方式，它能够帮助组织完成许多任务，也能够促进个人能力的发展和业务成功率的提升。但如果缺乏有效的目标设定和执行，目标驱动型管理将无助于达成组织的目标。因此，如何设置目标，如何实现目标，如何持续改善目标，以及如何确保目标达成，都需要系统地考虑到。

这篇文章的目标就是通过阐述“目标驱动型管理”的精髓，教会读者如何设定目标、如何实现目标、如何持续改善目标，以及如何确保目标达成。本文将从以下六个方面进行阐述：

1. 目标的定义及其层次结构
2. 目标层次结构下的四种管理模式
3. 如何设定目标
4. 如何实现目标
5. 如何持续改善目标
6. 如何确保目标达成

# 2.目标的定义及其层次结构
## 2.1 什么是目标？
目标是指对个人或组织目标的描述，是指对组织或个人生产、管理、服务、社会生活和价值观念的明确方向和期望。目标可以是短期的（如年度目标）、中期的（如五年计划）或者长远的（如美好的未来）。目标必须是可衡量的，能够让组织或个人看到其实现的前景，以及使组织或个人能够衡量其实现进度和结果的标准。目标的达成也必须是可测量的，能够让组织或个人知道其效益、效率、成果或效果，评估其是否满足其意图。目标是组织、团队、个人最宝贵的财富，也是管理活动的关键。

## 2.2 目标的层次结构
目标往往根据其重要性、紧迫性、复杂性和规模，被分为不同的层次结构。比如，最低层次的目标是企业愿景，它通常是企业长远发展的蓝图；较高层次的目标则包括产品和服务的市场营销、品牌建设、研发投资；中级目标则侧重于职业生涯的发展、个人成长，以及健康的身心健康；较低层次的目标则侧重于战略资源的开发、营运、保障等；甚至还有最高层次的目标，它可以包括价值观念的传播、社会责任的承担以及历史变革的引领。不同层次的目标往往又会受到不同的激励约束。

## 2.3 目标管理体系
目标管理体系是管理者为实现目标所制订的一套完整的管理程序。目标管理体系包括目标的确定、分析、制定、实施、监控、评估和改善。其中，目标的确定与选择直接关系到后续目标的实施和改善，目标的分析、制定和实施则可成为目标实现的基础。目标的监控和评估是目标管理的重要环节，是验证目标是否实现的有效手段。目标的改善既可以源于目标管理制度的完善，也可以在组织内部的协调配合下，通过新的目标细化和调整，来促进目标实现。目标管理体系应具备以下几个方面的特点：

1. 可预测性：目标管理体系要能够准确、快速、及时地反映目标的实现情况，便于评估目标是否已完成、提出对策和支持措施。
2. 制度化：目标管理体系的制定、实施、监控和评估应该有统一的制度框架和流程，有利于减少管理的沟通成本，确保目标顺利实施。
3. 结果导向：目标管理体系的目标不仅要达到目的，更应关注如何达到目标，以最大限度地实现组织的价值。

# 3.目标层次结构下的四种管理模式
目标驱动型管理的核心在于构建具有高激励性和挑战性的目标，围绕这些目标进行活动，以此实现组织或个人的目标实现。根据目标层次结构，目标驱动型管理还可分为四种管理模式。

## 3.1 基于组织目标管理模式
这种管理模式主要用于管理大型组织，如公司、政府部门和大型非盈利组织。基于组织目标管理模式的管理层目标往往是有组织的、长远的，而且一般都具有稳定的指标和相关数据。在这种模式下，管理层首先必须清晰地理解组织的目标、业务范围、现状和发展方向，并将其整合为一个连贯、完整的目标。然后，他们制定一系列具体目标，分解细化，映射到各个部门和业务线上。最后，管理层采用一系列工作流程和方法，集中力量，集体努力，推动目标实现。

## 3.2 基于产品/项目目标管理模式
这种管理模式主要用于管理创新型产品或项目，如互联网产品和创新项目。产品/项目目标管理模式的核心是针对某个特定产品或项目，制定明确且可衡量的目标。该模式的管理层主要负责制定一系列目标，将其分解细化，映射到相关部门和业务线上，并制定相应的任务和工作安排。管理层需要建立起有效的反馈机制，及时收集反馈信息，以便及时调整和完善目标，确保产品/项目的顺利推进。

## 3.3 基于个人目标管理模式
这种管理模式主要用于管理小型组织或个人，如家庭、学校和个人。个人目标管理模式的管理层目标往往比较具体和朴素，不需要太大的准备工作。管理层直接或间接地面向个人，通过个人自身的努力、关注、关怀和热情，建立目标，以促进个人成长、提升个人能力、塑造个人形象。个人目标管理模式往往倾向于运用快速反馈、迭代式创新、渐进式治理等方法，增强个人目标实现的有效性、敏捷性和动力。

## 3.4 混合管理模式
这种管理模式是以上三种管理模式的一种混合体，即将上述两种管理模式结合起来应用，既保留了两种模式独有的优点，也兼顾了两种模式的长处和短处。这种管理模式适合于大型组织和创新型项目，尤其需要激烈的变化和突破，同时也适合于个人和家庭。
