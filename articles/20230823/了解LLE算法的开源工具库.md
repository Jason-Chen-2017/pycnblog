# 1.简介
  

在大数据时代，数据的分布并非均匀的分布，而是存在着一些局部的聚类中心或区域，这些局部结构成为孤立点或噪声点。当数据集中存在大量的噪声点时，仅凭肉眼观察大数据可能难以发现真正有用的信息。因此需要一种降维的方法来提取有效的信息，消除这些孤立点或噪声点，使数据集中只保留有意义的数据。经典的降维方法如PCA、SVD等只能将数据投影到一个低维空间中，但其不能完全消除噪声点，只是将它们排除在外。随着大数据的爆炸性增长，孤立点越来越多，降维方法也逐渐变得越来越重要。如今已有不少基于学习的降维方法，如主成分分析(PCA)、因子分析(FA)、单值分布(SVD)、核PCA等，不过这些方法往往忽视了数据之间的关系，而人工构造特征又会引入额外的噪声。
而 Locally Linear Embedding (LLE) 是一种基于局部线性嵌入的降维方法，该方法能够很好地捕获数据中局部的非线性关系。它认为每个样本都可以看作是由局部的一个高阶张量所生成，而这个高阶张量中的每一项都可以看作是样本在该点附近的样本函数值之差，即样本函数的局部斜率。通过优化这些局部张量，LLE 可以将原始的样本映射到一个新的低维空间中，此时的低维空间中包含着数据的主要特征。因此，LLE 可用来探索高维数据内复杂的局部结构，并且在保持局部结构同时降低维度。此外，LLE 有别于传统的非线性降维方法如PCA，因为它可以同时考虑样本之间的关系和样本自身的特征，还可以在保持全局的情况下保证局部的维数。LLE 的优点有：
- LLE 直接利用了数据集中局部非线性关系，消除了潜在噪声点；
- LLE 不用像 PCA 那样计算协方差矩阵，所以速度上要快很多；
- LLE 在保持局部结构的同时降低了维度，而且这种降低并不是线性的，所以可用于非线性数据集；
- LLE 不依赖于固定的距离度量方式，适合不同类型的距离测度；
- LLE 对于复杂数据集有着良好的鲁棒性，能够自动识别数据中的模式变化。

# 2.基本概念术语说明
## 2.1 LLE 算法流程图
LLE 算法的主要工作包括两步：
1. 对数据集进行划分成局部领域。如图所示，第一步就是对数据集进行分区，将它划分成不同的子区域，每个子区域表示一个局部领域，子区域间相互独立。

2. 在局部领域内寻找局部样本与全局样本之间的联系。第二步就是对于每个局部领域，寻找其中样本与其周围邻域样本之间的联系，也就是寻找局部空间中样本之间的相关关系。如果找到了这种相关关系，就能够找到代表整个数据集的低维嵌入向量。寻找局部空间中样本之间的相关关系可以通过高斯核密度估计或局部线性嵌入算法实现。

## 2.2 核函数与核密度估计
核函数（Kernel function）是一个将输入信号转换为另一个维度的函数。通常，核函数是通过某种统计学模型来拟合数据的。核函数提供了一种在计算上比全特征空间更有效的方式，可以有效处理高维数据，并减少计算时间。核函数可以分为线性核函数、多项式核函数、径向基函数、字符串核函数等等。一般来说，核密度估计可以理解为在给定数据集上求取核函数后计算核函数的积分，得到一个概率密度函数。具体做法是先选定核函数，然后在指定邻域范围内对每个点计算核函数的值。不同核函数之间的选择会影响最终的结果，但是对核函数的选择往往与目标任务、数据类型以及可获得信息量等因素有关。

## 2.3 局部线性嵌入算法的特点及优缺点
局部线性嵌入算法（Locally Linear Embedding, LLE）可以被认为是核密度估计的扩展。它假设数据集是由多个局部领域组成的，每个领域内部存在着较强的相关关系，并通过将各个领域联系起来实现全局结构的重构。局部线性嵌入算法可以被看作是非线性数据集的线性判别分析。优点如下：
- 局部线性嵌入算法不用计算协方差矩阵，计算速度更快；
- 局部线性嵌入算法不必预先知道各个领域之间的边界条件，因此不需要手动设置核函数的参数；
- 局部线性嵌入算法不改变数据的拓扑结构，因此对局部结构具有更强的抗噪能力；
- 局部线性嵌入算法可以自动识别复杂数据中的模式变化，使得它在一些情况下有着更好的泛化能力。
局部线性嵌入算法的缺点如下：
- 局部线性嵌入算法对目标变量具有很强的依赖性，因此在非线性数据集上表现不佳；
- 局部线性嵌入算法无法处理含有大量噪声点的复杂数据集。
