# 2.核心概念
## 2.1 语音识别
语音识别（英语：speech recognition）是一个自然语言处理任务，用来把输入的声音或者说话人的声音转换成文本形式。早期，这个领域主要是靠手工定义规则进行识别，但随着技术的发展，越来越多的语音数据和更好的计算机硬件，使得越来越多的方法被提出，包括基于规则、统计模型、神经网络等等。语音识别系统需要完成以下几个方面的工作：

1. 特征提取：将语音信号转化成有用的特征，如MFCC(mel frequency cepstral coefficients)。
2. 模型训练：根据特征向量和对应的文本标签进行模型训练，找到最佳匹配结果。
3. 测试：测试识别准确率、性能指标。
4. 用户交互：为用户提供文本输出以及各种反馈，比如语音提示、错误修正建议等。

## 2.2 语音转文字
语音转文字（英语：speech-to-text）又称语音识别、语音合成或语音识别服务，是通过计算机从人类的声音中识别出所说的话并翻译成文字的过程。早年，语音转文字是一个简单的功能，但到了今天，我们越来越多地使用它，例如聊天机器人、对话系统、数字助手、广播台词解码器等。语音转文字需要完成以下几个方面的工作：

1. 声学模型：用声学参数描述声音的特性，声学模型一般采用统计方法来建立声学模型。
2. 发音模型：建立发音模型，将声学模型和实际发音联系起来。发音模型的构建需要考虑发音时的参数、环境条件等因素。
3. 概率模型：建立一个概率模型，用声学模型和发音模型生成可能的序列，并通过概率计算得到每个序列出现的概率。
4. 解码算法：将概率最高的序列作为最终输出结果，其中包括拼写错误检测、语言模型、搜索优化等。

## 2.3 人工智能
人工智能（英语：Artificial Intelligence，缩写为AI），也称机器智能、符号智能、生物智能、直觉智能、认知智能等。它的研究对象是智能体、智能机械及其子系统。在20世纪80年代末至90年代初，这一领域的研究与发展较为缓慢，主要是基于工程实现、复杂的数学模型及规则的开发。到21世纪初，随着硬件性能的增强和大规模数据集的积累，人工智能取得了突飞猛进的发展。近年来，人工智能技术的应用已经从简单到复杂，从信息检索、图像识别、语音识别到自然语言理解、深度学习等，形成了一整套完整的产业链。

## 2.4 端到端自动语音识别（E2E ASR）
端到端自动语音识别（英语：End-to-end Automatic Speech Recognition，简称E2E ASR），是指利用机器学习技术直接将语音信号转化为文本形式，不需要中间的组件，即可识别出所说的话。E2E ASR是在深度学习、深层神经网络、循环神经网络、注意力机制等基础上提出的一种新的语音识别技术。这种技术可以达到实时识别的效果，而且没有必要对声学模型、发音模型等做太多假设，直接使用原始的音频信号作为输入，而输出文本。
