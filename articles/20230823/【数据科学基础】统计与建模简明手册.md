# 1.简介
  

数据科学（Data Science）是一个跨学科的领域，涉及计算机、统计学、数学、理论经济学等多个领域。而对于数据科学相关的知识点，也不是完全通俗易懂。
相比于其他知识点的学习，在实际工作中应用较多的是统计模型和分析方法。很多公司都需要统计和建模能力来进行决策、预测、分析数据的趋势，甚至是投资产品的策略。因此，了解相关的统计知识，尤其是机器学习和深度学习方面的统计知识对公司员工和整个社区都是非常重要的。
《【数据科学基础】统计与建模简明手册》的目标就是要提供给大家一个全面且系统的学习数据科学的统计知识的方法。希望通过阅读本书，能够让读者掌握数据科学过程中所需的相关统计知识，以及如何运用这些知识解决实际问题。同时，《【数据科学基础】统计与建模简明手册》也希望能够帮助更多的技术人员、数据科学家、研究人员以及工程师从零开始学习数据科学的统计知识，提高职场竞争力。
本书将全面讲述数据科学的统计知识，包括概率论、线性代数、随机过程、信息论、统计推断、假设检验、分组分析、贝叶斯统计、回归分析、聚类分析、关联分析、分类树、异常检测、数据可视化、深度学习等。在每一章节的最后，还会给出相应的参考文献，方便读者进行更进一步的学习。
# 2.概率论
概率论是指把客观事物可能性的陈述表示出来，并赋予它们各自不同的概率值或概率分布，是人们研究随机现象的基础。概率论可用于描述各种现象出现的频率、规律性、相互影响以及偶然性，并由此求取概率估计、事件的条件概率、联合概率等重要工具。概率论有许多重要的数学理论和应用，如随机变量、期望、方差、协方差、独立性、卡方分布、正态分布、Beta分布、Gamma分布、Dirichlet分布等。由于概率论的广泛性和实用性，概率论成为数据科学的统计学的基础。下面我们就概率论中的一些重要概念及概念之间的关系做一些阐述。
## 2.1 随机事件
在概率论里，一个事件可以看作是某个确定范围内符合一定特征的一个或多个样本点。比如抛硬币的过程，是个随机事件；在一个足球比赛中，击中篮板的结果就是一个随机事件；某个时间段内的销售额、客户流量、病人的就诊率，这些都是随机事件。在概率论中，随机事件通常用希腊字母表示，比如X、Y、Z、W等。
## 2.2 概率空间与分布函数
在概率论中，随机事件的结果可以用离散型随机变量来表示，也可以用连续型随机变量来表示。当随机变量是离散型时，称之为概率空间或样本空间，即所有可能的结果构成的集合；当随机变量是连续型时，则称之为概率密度函数或概率密度曲线。样本空间由所有可能的结果构成，分布函数(Probability Density Function)是样本空间到[0,1]的连续映射。若随机变量X服从某种分布，则概率空间中任取两个点x,y∈X,有P(x<=Y<=y)=F(y)-F(x),其中F(·)是分布函数。
## 2.3 概率axiom
在概率论里，有三个基本的公理axiom，分别是：

1. 非负性：对于所有的事件A，P(A)≥0
2. 规范性：对于任意事件A，P(A)≤1
3. 可列可加性：如果A_1,A_2,...是一系列事件，并且A_i无间隔地排列，那么他们的并集也是事件：P(A_1∪A_2∪...)=P(A_1)+P(A_2)+...。

上述公理axiom，还有一些其他的定理，如单调性、条件独立性、充分不相容原理等。概率论家经过长久的研究，总结出了一整套的概率论体系。
# 3.基本概念术语说明
## 3.1 数据集、样本、样本点、样本空间、数据项、数据元素、特征、维度
数据科学及统计学中经常使用的一些术语或概念。下面按照重要性顺序逐一说明。
### 3.1.1 数据集
数据集（dataset），是指用来进行统计分析的数据的一组描述性信息。数据集可以是包括数字、文本、图像、声音、视频等在内的各种类型，也可以是结构化、半结构化或者非结构化的数据。
### 3.1.2 样本
样本（sample）是指随机抽取的一组数据集中的数据元素。
### 3.1.3 样本点
样本点（sample point）是指样本中的一个具体的数据元素。它可以是一个单独的值，例如邮政编码“9000”；也可以是一个记录，例如一条地址信息；也可以是一个属性，例如一个学生的年龄。
### 3.1.4 样本空间
样本空间（sample space）是指样本的所有可能结果构成的集合，它代表着所有可能出现的样本点，可以用一个事件或一组事件来刻画。比如，邮政编码“9000”就是一个样本点，而所有的邮政编码构成了邮政编码空间。
### 3.1.5 数据项
数据项（datum）是指指样本中代表性的数据元素，可以是某个属性的具体值，也可以是一个代表性的样本点。例如，“67”、“Jones Street”、“Female”、“M”、“Mary Smith”都是数据项。
### 3.1.6 数据元素
数据元素（data element）是指数据集中的一个基本单位，也就是说，它代表的是一个确定的、不可再分割的数据单元。比如，一个电子邮件就是一个数据元素。
### 3.1.7 特征
特征（feature）是指样本的一种主要属性。它可以是每个样本点的描述性信息，例如邮政编码“9000”的第一个数字“9”；它也可以是样本的总体性质，例如该样本集中是否存在重复值。
### 3.1.8 维度
维度（dimensionality）是指样本所具有的特征数量，也即样本的维数。
