下面是一个简单的 Python 代码示例，用于实现图像识别功能：
```python
import cv2
import numpy as np

# 加载图像
img = cv2.imread('hotel_image.jpg')

# 特征提取
特征 = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 识别酒店类型
type = 'unknown'

# 匹配特征
match_score = 0
for t in range(1, 10):
    # 提取特征
    match_img = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
    match_img = match_img[0:150, t-1]

    # 匹配特征
    match_score = match_score + (match_img == t)

    # 判断类型
    if match_score > 0.8:
        type = t

# 输出酒店类型
print('酒店类型为：', type)
```
