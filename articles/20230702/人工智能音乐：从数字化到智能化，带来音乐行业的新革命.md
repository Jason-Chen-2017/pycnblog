4.3. 核心代码实现

```python
import numpy as np
import random
from sklearn.datasets import load_iris

class Genetic MusicComposer:
    def __init__(self, size, flowchart):
        self.size = size
        self.flowchart = flowchart
        self.fitness = 0

    def _random_draw(self):
        return random.sample(self.flowchart, self.size)

    def _calculate_fitness(self, new_chart):
         fitness = 0
         for node in new_chart:
             if node[0] == '(':
                fitness += 1
             else:
                fitness -= 1
         return fitness

    def create_composition(self):
        new_chart = self._random_draw()
        fitness = self._calculate_fitness(new_chart)
        self.fitness = fitness
        return new_chart

# example usage
composer = Genetic MusicComposer(100, '\\'.join(map(str, [1]*100))
new_chart = composer.create_composition()
print(new_chart)
```

4.4. 代码讲解说明

在上面的代码中，我们定义了一个`Genetic MusicComposer`类，该类包含歌曲创作的逻辑。

首先，我们定义了一个`__init__`方法，用于初始化歌曲创作参数，包括尺寸（即创作的歌曲数量）、流向图等。

接着，我们定义了一个`_random_draw`方法，用于从流向图中随机选择一个节点。

然后，我们定义了一个`_calculate_fitness`方法，用于计算新节点的适应度。

最后，我们定义了`create_composition`方法，用于创建一首新的歌曲。

在`create_composition`方法中，我们使用`_random_draw`方法生成新的流向图，然后使用`_calculate_fitness`方法计算新流向图的适应度，最后返回新流向图。
