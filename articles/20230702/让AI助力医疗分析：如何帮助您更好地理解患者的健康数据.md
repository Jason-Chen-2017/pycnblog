4.3. 核心代码实现

假设您想实现一个基于决策树的疾病诊断模块，您需要实现决策树的训练过程以及病历数据的预处理、特征提取等步骤。以下是一个简单的 Python 代码示例，展示了如何实现决策树训练的接口：

```python
import numpy as np
from sklearn.datasets import load_iris
from sklearn.tree import DecisionTreeClassifier

# 加载数据集
iris = load_iris()

# 定义特征和目标变量
features = iris.data
target = iris.target

# 创建决策树
tree = DecisionTreeClassifier()

# 训练决策树
tree.fit(features, target)

# 返回决策树对象
return tree
```
