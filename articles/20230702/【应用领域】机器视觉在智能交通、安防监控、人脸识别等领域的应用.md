4.3. 核心代码实现

在实现机器视觉技术的过程中，核心代码的实现是非常重要的。以下是一个简单的机器视觉技术实现步骤：

```python
import cv2
import numpy as np

# 加载图像
img = cv2.imread('example.jpg')

# 特征提取
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
thresh = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)
contours, hierarchy = cv2.findContours(thresh, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

# 循环遍历
for contour in contours:
    # 计算周长
    x, y, w, h = cv2.boundingRect(contour)
    # 计算面积
    area = cv2.contourArea(contour)
    # 计算重心坐标
    x = w / 2
    y = h / 2
    # 计算重心面积
    inner_area = (x - 0) * (y - 0) * np.contourArea(contour)
    # 计算重心坐标
    x = x - inner_area * w / (2 * h)
    y = y - inner_area * h / (2 * w)
    # 绘制轮廓
    cv2.drawContours(img, [contour], [x, y, w, h], (0, 255, 0), -1)
```

在上述代码中，我们首先使用 OpenCV 库中的 `cv2.imread()` 函数加载了一张图片，并使用 `cv2.cvtColor()` 函数将其转换为灰度图像。接着，使用 `cv2.threshold()` 函数对灰度图像进行阈值处理，得到二值图像。然后，使用 `cv2.findContours()` 函数对二值图像进行轮廓检测，得到各个轮廓。在循环遍历的过程中，我们计算了每个轮廓的周长、面积和重心坐标，并使用这些信息绘制了轮廓。
