### 1.2.3 大模型与传统模型的对比

AI 模型分为传统模型和大模型两种。它们在构建、训练、部署、运维等方面存在显著差异，本节将进行对比探讨。

#### 1.2.3.1 构建

传统模型通常面临数据量较小、质量不uniform、 feature 缺失等问题。因此，需要对数据进行预处理，例如缺失值填充、异常值检测和去除、归一化和standardization 等。同时，传统模型也需要对 feature 进行特征选择和特征工程，以减少数据集维度并提高模型性能。

相比之下，AI 大模型在构建阶段要求的数据量更多，但对数据质量和 feature 的要求较低，因为大模型会自动学习 feature representation。这使得 AI 大模型在构建过程中更简单且高效，省去繁琐的特征选择和特征工程步骤。

#### 1.2.3.2 训练

传统模型的训练速度较快，因为它们的 model complexity 较低，可以在短时间内完成训练。然而，由于传统模型在数据量较小、quality 不uniform、feature 缺失等方面的限制，其 performance 普遍较差，难以满足当今复杂应用场景的需求。

与传统模型形成对比的是，AI 大模型在训练速度上存在显著优势。由于大模型的 parallelism 能力强、model complexity 较高，可以利用 GPU 等硬件资源并行训练，缩短训练时间。此外，AI 大模型的 performance 也显著优于传统模型，适用于越来越多的复杂场景。

#### 1.2.3.3 部署与运维

传统模型的部署和运维成本较低，因为它们的 model size 较小，可以轻松在各种平台和设备上部署运行。但由于传统模型的 performance 普遍较差，难以满足实时性和高精度的要求。

相比之下，AI 大模型的 model size 较大，部署和运维成本也较高。但由于大模型的 performance 优秀，适用于越来越多的实时性和高精度的场景。因此，AI 大模型的部署和运维成本在一定程度上可以被它的优异表现所 compensate。

## 2. 核心概念与联系

### 2.1 AI 模型与人脑

人类认知能力建立在数亿个神经元之上，它们相互连接形成复杂网络，负责信息处理和决策。类似地，AI 模型也是由大量参数组成，负责数据的处理和决策。两者在基本原则上是类似的，都是通过网络结构和参数调整来实现功能的。

但是，人类认知能力在处理复杂信息方面具有显著优势，因为它具有自适应、自我完善、多模态、抽象化等能力。相比之下，AI 模型只能通过固定的算法和参数来实现功能，难以适应新情况或学习新知识。

AI 模型和传统模型在构建、训练、部署等方面存在显著差异，主要体现在数据量、model complexity、parallelism ability 和 adaptability 等方面。
