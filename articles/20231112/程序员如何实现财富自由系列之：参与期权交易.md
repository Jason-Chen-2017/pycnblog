                 

# 1.背景介绍


## 1.1 期权的概念
期权（Option）是一个关于买卖双方权利义务关系的合约，由两个角色——做空者和认购者组成。做空者以给出某种现金的价格，要求在某个特定时期以某个固定的价格买入该物品（如股票、债券或商品），并以同等条件对其进行认购，换取远低于市场价值的某种金融资产。认购者则要在特定的时间内以相同的价格卖出这类物品并得到钱回报。期权既可以定价，也可以用来执行交易。在美国市场，期权通常指欧洲期权、美式期权或A股期权等产品。


期权在投资界有着非常广泛的应用，尤其是在债券市场。对于一个普通人来说，参与期权交易并不是一件轻松的事情，但对个人来说，这是一项高风险、大收益的投资活动。很多投资者认为期权交易是一种在资本市场上的高风险低回报的活动，因此在股票市场也经常被视作一种补充资本的方式。

## 1.2 期权交易的市场环境
### 1.2.1 历史
由于期权市场从诞生之初就是高度竞争的市场，它的规则也随着时间的推移不断演变。就像在股票市场一样，期权市场也存在着对冲手段和套保行为。但是，与股票不同的是，期权市场的规则更加复杂，有可能影响到市场的稳定性。另外，由于期权的含义多样，不同人的观点会发生分歧，使得市场具有较大的不确定性。

### 1.2.2 今日中国期权市场情况
目前，中国股票市场的股票期权相比其他国家都比较受欢迎，主要原因如下：
- 中国的股票市场和其他国家都没有严格限制证券期货交易的范围，可以进行A股、B股、H股等所有股票期权的交易；
- 在股票市场上的期权往往没有在期货市场上设立的障碍，因此，在股票期权市场中，相关交易费用和保证金开销较少；
- 在股票期权市场中，更多的交易者是持有者而不是投机者，这有利于保障期权市场的稳定性；

但是，中国股票期权市场的交易量还不足百万，仅占全球期权市场的百分之十几。国内外期权交易所相互之间的竞争也影响了市场的活跃程度。另外，随着中国股市及各个行业的快速发展，期权市场的持续性增强了，并且正逐渐成为中国经济增长的一个重要领域。

除了一些有关信息不透明的因素，中国期权市场还有以下几个关键特征：
- 无需注册登记，随时可交易；
- 消费者大多数为散户，可选择资产支持；
- 交易方式复杂，涉及认购人寻找期权、募集资金、协商交易细节等过程；
- 活跃期货市场，价格波动大，政策风险较大。

## 1.3 期权交易的优缺点
### 1.3.1 优点
- 投资灵活性高：对于个人而言，期权交易提供了一种多元化的投资选择，可以选择不同类型的期权，获得不同风险厌恶度的收益；
- 交易费用低：期权市场的交易费用一般都非常低，主要包括保证金、手续费、佣金等。交易者只需要按照市场规律支付佣金和保证金即可。同时，也不存在托管、集合资金、维系关系等麻烦，相对而言更加方便安全；
- 可获知未来价值：期权市场提供了一种获取未来的价值的途径。由于期权往往具有较长的生命周期，而且在当下时刻也有很大的变化，因此，了解这些期权的未来价值也是投资者的一种能力。
- 平衡交易风险：期权交易能够提供各种风险收益的平衡，适用于个人投资者和机构投资者。机构投资者可以通过向期权提供资金，以获得多个投资者共赢的效果；
- 收益稳定性高：因为期权市场的规则较为复杂，所以期权的持仓期长，收益率也会不断提升，不会出现阶段性的大幅滑坡，所以对于那些长期持有期权的投资者而言，能够获得稳定的收益是非常重要的；

### 1.3.2 缺点
- 交易次数有限：相比于其他的投资工具，期权交易具有较高的门槛，有限的次数只能带来很小的收益甚至损失，适合熟练、经验丰富的交易者；
- 操作复杂度高：期权交易的规则相对比较复杂，操作人员需要花费较多的时间来熟悉市场规则、理解市场走势、制定交易计划、执行交易、记录交易结果等等。如果遇到困难的话，还需要承担一定的风险；
- 对未来价值的预测准确度差：预测未来价值准确度一直是期权交易的一大难题。期权的公允价值和当时的市场价格之间存在着巨大的误差，所以即使预测到将来某种期权的价格会有所下调，实际的结果仍然可能很大；
