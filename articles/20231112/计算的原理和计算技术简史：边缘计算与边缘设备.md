                 

# 1.背景介绍


2010年5月，英国电信运营商BT的工程师克里斯托弗·瓦纳克（<NAME>）宣布了一项新计划——向全球提供超过90亿部智能手机。这个庞大的计划包括智能手机、平板电脑、笔记本电脑和其他所有依赖互联网或无线网络连接的设备。这标志着移动互联网正成为人们生活不可或缺的一部分，如同电话一样。然而，与传统的云计算服务不同的是，这一巨大的计划并没有采用传统的云计算模型。相反，它基于一种名为“边缘计算”的新技术。

那么，什么是边缘计算？简单来说，就是把最关键的计算任务放在用户的距离最近的地方，比如手机、路由器、机器人等。这样就可以节省大量的计算资源，提高通信效率、降低成本、缩短响应时间，甚至解决一些应用难题。举个例子，在现代工厂中，只有极少数人员需要拿到真正的生产工具，其余人员都被安排在密集的现场。而这类密集的现场是无法支持云计算服务的。因此，边缘计算可以帮助企业提升效率，降低成本，达到更好的社会效果。

当然，边缘计算也不是银弹。首先，边缘计算需要本地的设备支持，不像云计算那样可以随时扩展，因此存在部署、维护等一系列困难；其次，它不能处理所有的计算任务，比如需要分析极大的数据量、分析实时数据等，这些任务需要本地的高性能硬件才能满足需求；最后，安全问题也是个大麻烦。但是，随着边缘计算的发展，越来越多的企业和个人开始接受这种新的技术，希望通过它的高效率、低成本的优势来实现组织目标。 

# 2.核心概念与联系
为了方便理解和阐述，我们需要引入一些相关的术语。以下是一些重要的概念：

1. **云计算**：云计算是一种按需计算方式，利用大量可重用的计算机资源，通过网络将应用程序、数据、服务及相关支撑设施进行动态分配和共享。云计算服务提供商可以根据用户的需要快速、自动地扩展和管理平台，通过降低基础设施投资和管理成本，提供高度可靠和可伸缩的计算服务。 

2. **边缘计算**：边缘计算是指利用分布式计算的概念，将计算任务放在用户距离数据源或服务中心较近的位置，通过减轻服务端的负载和对数据传输速度进行优化，提升数据的处理能力和响应速度。边缘计算的特征主要有以下几点：

   * 对数据中心网络带宽的要求小
   * 使用户能够获得更快的响应时间
   * 提供了用户距离数据源距离较远的计算能力
   * 需要考虑底层硬件的性能和功耗限制
   * 可保证机密性、完整性、可用性和安全性

3. **边缘网关**：边缘网关是一个位于服务器或者其它计算节点边缘的计算网关设备。它的作用是对边缘设备上的服务进行拆分和编排，将各个模块组装起来，形成一个统一的服务接口。典型的场景是智能扬声器，它有两个控制通道，分别用来播放音乐和收听广播。

   通过边缘网关，可以对多个设备的请求进行聚合，并进行分配和处理。网关能够对边缘设备上运行的应用程序进行分类和分配，同时也提供了一种抽象和标准化的操作界面，使得内部网络可以对外暴露统一的服务接口。

4. **边缘节点**：边缘节点是一个有独立存储、计算资源且可作为计算节点加入计算网关的设备。它可能是一个PC服务器、嵌入式设备或者是一种边缘计算平台。边缘节点通常安装有运行服务的应用程序，并且可以接受来自网关的请求。

5. **计算调度**：计算调度是指根据某种策略对计算任务进行分配和调度，以便达到最佳资源利用率。目前，主要有两种方法：静态调度和动态调度。

   * 静态调度：静态调度是指根据已知的计算资源分布情况，对任务进行固定分配，使整个计算资源得到充分利用。静态调度的方法有先进先出法（FIFO）、最短任务优先（SPF）、最短剩余时间优先（SRT）和轮转法等。
   * 动态调度：动态调度是指根据实际的资源利用率、任务等待时间、任务完成时间等因素，实时调整计算资源的分配，使系统的资源利用率最高。动态调度的方法有弹性调度和可感知调度等。

6. **边缘云**：边缘云是由边缘计算、云计算和分布式计算技术等新兴技术组成的混合型分布式计算环境。它能够提供多种计算模式，比如智能网关、平台虚拟化、云计算、离线计算、高性能计算、超算等。边缘云服务提供商可以根据用户的需要快速、自动地扩展和管理平台，通过降低基础设施投资和管理成本，提供高度可靠和可伸缩的计算服务。

7. **边缘设备**：边缘设备是指具备计算能力，可以在网络范围内运行的设备，如路由器、移动设备、机器人、消费电子产品等。这些设备通常部署在服务区，具有经济性、灵活性和实用性，能够在极端环境下运行，适用于需要长时间持续计算但资源较少的应用场景。
