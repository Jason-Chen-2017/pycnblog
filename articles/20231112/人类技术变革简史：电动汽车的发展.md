                 

# 1.背景介绍


随着科技的发展，人类的生活环境越来越不适应，日益严重的通勤问题也在社会上引起了广泛关注。于是有了很多人开始思考如何减少交通成本，降低通勤时间等等。其中最著名的就是“智能化+自动化”的电动车和新能源车的兴起。

从“祸兮福所倚楼阁”，到“福兮祸所伏天子”，人类对于电动车和新能源车的追求和努力确实可圈可点。然而，实际上，电动汽车的发展历史比想象中要曲折多彩。

首先，电动汽车诞生之前，并没有什么特别的工具或方法可以用来作为交通工具。当时普通人的生活主要依靠骡马、鞍马或驴子拉车，这种粗糙、手动的交通工具造成了严重的交通拥堵，在当时的文明社会里，很难实现交通畅通，导致许多年轻人陷入贫困状态。因此，人们只能选择跟随潮流或者逆流而上，而不能选择依赖单一交通工具。

但随着电器、工业制成品、高科技的发展，各种类型的交通工具出现，使得人们能够利用电力、水、天气、温度、电子信号等多种条件来进行自主控制和优化自己的交通模式。

人们发明了铁路，火车，飞机以及无人机等交通工具之后，在经济发达的国家蓬勃发展。但是这些交通工具又给人们带来了新的问题。例如，它们的设计、制造、安装都比较复杂，操作起来也相对麻烦，而由于基础设施建设缺乏和物质极其匮乏的问题，使得这些交通工具普遍存在着较大的成本，加剧了交通拥堵。另外，随着这些交通工具的普及，各地的旅客量增加，尤其是对城市旅行者，造成了“聚集性疾病”。也就是说，这类交通工具和其产生的各种问题，不仅影响到一般人群的出行效率，更让一些大城市成为交通拥堵中心，并向其他地方蔓延开来。

在这个过程中，为了解决人们的出行问题，国际上提出了一系列的解决方案，例如福特汽车公司于1907年推出“福斯玛”牌的火车，英国在1909年发明了“摩托罗拉”牌的汽车，法国在1911年推出了“巴士”牌的火车，日本在1912年制造出了“丝路虎”牌的火车……而在中国的情况则是，受到苏联“第五共和国”的启发，中国在“甲午战争”结束后，以“开辟新航道”的名义，试图通过“不间断的科技进步，打破传统的交通模式的束缚，带领人类走向共同富裕。”

然而，上述的各种尝试都是失败的，包括试验各种形式的电动车，最后放弃了电动车的尝试。直到上世纪末期，人们才看到一个电动车的雏形——一辆发电机车，但是它并不能独立行驶，需要配合其他设备才能完成完整的行驶。并且，从那时起，已经有足够的理论支持，认为采用传统的火车机械动力可能是电动车的最佳替代方案。但这样做会遇到众多的问题。例如，目前没有完全统一的电动车电池充电系统，同时开发耗费大量资金，成本过高，供给侧结构不稳定等等。

最终，在1980年代初，美国政府颁布了《汽车安全法》，禁止私自推销、贩卖电动车，引发了一场轩然大波。此后，英国、法国、日本等国纷纷加入“禁用电动车”的斗争，到2008年，中国在“全面禁止进口电动车”的基础上，又落实了相关政策，虽然仍然没有完全禁止，但开始限制少数特定厂商的电动车，如上海奥迪，上海凯迪拉克等。

2010年，日本宣布将在未来五年内将所有购买的新车进行改装，改装成配备了电动系统的电动车，但至今没有效果。而在中国，通过对电动汽车发展的重新评估，人们又一次发现，电动汽车的发展正在走向自己的反方向。

综上所述，我们看到，电动汽车的发展经历了漫长的曲折过程。直到近几年，国际上有关电动车的发展取得了一定的成果。但中国和欧洲以及其他国家之间的发展差距还很大。这是因为，即使是在西方，也有很多国家在推进电动车的发展，比如美国、加拿大、澳大利亚、德国等。在这些国家里，电动汽车的平均价格每年都在涨，但发展的速度却远远慢于中国。同时，中国还处于互联网、新能源汽车、机器人等新领域的竞争中，如果短时间内推动电动汽车的发展，很可能会被边缘化掉。

所以，就电动汽车的发展历史来说，我们不妨从“祸兮福所倚楼阁”讲起，通过“祸兮福所伏天子”的故事，理解电动汽车发展背后的规律，找到走向正道的方向。
