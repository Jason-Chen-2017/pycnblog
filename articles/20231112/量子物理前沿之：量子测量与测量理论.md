                 

# 1.背景介绍


## 什么是量子？
量子（英语：quantum）或称许多电子系统中的“原子”，是指量子态的简化模型。
量子（希腊文：Klepton）是一类不确定性原理所研究的微观粒子，一般来说可以分为两类：波函数（波函数本身就不是确定性的）和固体状态（其结果仅受限于某些法则）。
## 为什么要研究量子物理？
量子物理是人工智能、材料科学、通信、生物工程等领域的一项重要研究领域，具有广阔的应用前景。我们生活中使用的电器、电子设备、机器人、无线通讯、化学反应、金融、航空航天、航海、军事等领域都离不开它。同时，量子纠缠效应的存在给某些复杂系统带来诸如线圈跳变、干涉仪、相干耦合、频率选择、量子门切换等诡异现象，有利于理解这些系统的内部运作机制，也对其设计提出了新的要求。
## 量子物理到底是个啥？
量子物理是研究量子世界，即由量子力学构建起来的宇宙学系统的一门学术分支。量子世界的基础是相互作用而非类ical世界。它由数个量子点组成的世界空间。这些量子点包括质子、电子、超夸克、中微子、双极子、单核和弱核子。它们之间存在着各种相互作用，使得不同种类的量子态混合在一起，形成各种不同的状态。为了研究量子物理，我们需要了解量子场、粒子、费米面、弦图模型及其相互作用。通过对各种量子现象进行观察与分析，我们能够从更高的角度认识这一复杂的现实世界。
## 量子物理的基本要素是什么？
一个完整的量子系统由许多微观粒子构成，这些粒子组成了一个量子系统，并受到外界的影响而不确定地演化。量子系统可分为两种：

1. 有限系统：通常由少量的量子粒子构成，这些量子粒子的数量有限且固定。例如，经典的晶体管就是由离散的磁铁组合而成。

2. 无限系统：即由无限多个粒子构成的量子系统，例如经典自由度大于等于2的波函数系综。

每一种系统都包含若干个量子指标，包括位置坐标、动量（描述粒子运动方向）、波函数表示、湮灭算符等，可以通过对这些量子指标的测量来获取系统的信息。通过改变量子指标的值，就可以控制系统的运动。

量子指标也分为两种：

1. 数字指标：有时也被称为量子比特，包括复数平面上的量子位（也称为量子比特），也称为qubit。

2. 符号指标：描述两个以上量子态之间的关联，例如相互作用张量或玻尔兹曼方程。

量子量是指量子指标的统计平均值。通常情况下，只有当所有量子态都平稳混叠到相同的平均值处时才会认为系统达到了完备态。在量子信息处理中，经常需要将杂乱无章的信息压缩至有限的编码长度，这是因为量子系统中能量最小的态对应的信息密度最高。所以，如何快速有效地把量子态映射到信息处理上是量子物理的一个基本问题。

量子纠缠（quantum interference）是指量子系统中不同量子态之间发生的相互作用，使得量子态发生扭曲、折叠、混合、混淆。纠缠强度由纠缠面的积分决定。通过制造混乱的态，我们可以探索量子世界隐藏的奥秘。

测量是量子物理中非常重要的一环，也是量子物理研究的核心。由于无法完全观测量子系统，因此只能利用量子系统中已知的量子态或可观测的量子性质来推断量子系统的物理性质。由于量子系统是不确定性原理的产物，因此测量得到的结果往往会因随机扰动而出现偏差。在实际操作过程中，我们必须对测量结果做适当的处理才能得到正确的结果。
