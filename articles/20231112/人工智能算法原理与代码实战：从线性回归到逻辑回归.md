                 

# 1.背景介绍


机器学习（Artificial Intelligence, AI）是一种让计算机“学习”的方式。它可以使机器能够自主地解决复杂的问题，能够实现从数据中发现规律，自动优化模型参数等。虽然近几年出现了一些神经网络、深度学习、强化学习等热门的机器学习方向，但其背后的基本算法仍然是线性回归、逻辑回归等传统的数学方法。所以本文主要介绍机器学习中的两种基础算法——线性回归和逻辑回归，并通过Python语言实现。

## 2.2 逻辑回归(Logistic Regression)
逻辑回归是一种分类算法。它也假设输入的特征与输出之间的关系是用一条曲线来描述的。不同于线性回归，它是一个二元分类器，其输出只能取两个值，如0或1。

## 2.3 回归与分类的区别与联系
从上述两者的特点就可以看出，两者的区别在于：

1. 回归问题中，目标变量是连续的，模型的输出是一个实数值，输出范围受输入影响。例如，预测房屋价格。
2. 分类问题中，目标变量是离散的，模型的输出是一个离散值，输出范围固定。例如，判断一个邮件是否为垃圾邮件。

同时，两者的联系也十分密切。比如，逻辑回归模型是将线性回归模型的输出映射到0-1之间，使得其输出只能取两个值。而线性回归模型可以将输出映射到任意一个实数值。所以，两者的应用场景十分广泛。
