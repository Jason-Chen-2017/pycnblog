                 

# 1.背景介绍


在进入公司面试的时候，我曾经历过无数次的面试官对我的问卷的回答质量做出评价，比如满意、不满意、基本上还行、可以接受。无论如何，作为候选人，如果你有一定的竞争力的话，无论面试结果如何，一定要主动沟通，坦诚交流面试心得。面试经验越丰富，面试过程中的话术也会越吸引人，所以，一个好的面试风格、坦诚表达能力、积极配合的工作氛围，都能提高候选人的能力和幸福感。当然，还有更重要的一点，就是在准备面试过程中要注意以下几点：

1） 提前准备好简历和职业规划

2） 做好笔记记录面试经验

3） 提前做好充足的功课测试和学习准备

4） 做好面试人员的专业素养培训

面试完之后，除了“恭喜恭喜”之外，最重要的是交流自己的经验和教训。不仅如此，也可以通过分享经验，帮助他人，提升个人职场竞争力。所以，面试后收到优秀面试通知时，一定要主动做好后续处理。下面我们就以“面试后处理”这个话题，从不同角度探讨面试后的后续处理方法，以期达到事半功倍的效果。
# 2.核心概念与联系
## 什么是后续处理？
后续处理，简单地说就是给面试官提供进一步的面试信息或指导，提升面试者的工作绩效、职场竞争力等。根据《程序员面试经典（第四版）》，后续处理分为以下三种类型：

1）定向邀约：包括直接约见面试官、电话联系面试官等方式，可以询问面试官对求职者的了解、感兴趣的方向、技能要求、学历情况、实习经验等信息，帮助求职者加快面试进度，缩短面试周期。

2）校园招聘信息：对于需要招聘对象的院校或者公司，可以找到该机构提供的招聘信息，了解该公司的薪酬福利待遇、岗位类型、工作性质、工作环境、面试难易程度、面试地点等信息，帮助求职者决定是否接受、修改面试时间表。

3）其他技能提升：针对求职者具有其他相关职业技能的求职者，可以为其安排一些证书培训课程、简历编写、项目推荐等方式，增强求职者的职场竞争力。

## 为什么要进行后续处理？
为什么要进行后续处理呢？举个例子，在入职之后，你可能会被要求去参加一个面试活动，甚至可能被邀请去参加某个讲座。这些活动往往会给你的工作带来很大的帮助，但是在一些情况下，你可能得不到相应的支持。而后续处理可以为你提供更多的支持，它可以让你获得收获、锻炼自身的能力、增强职场竞争力，帮助你顺利完成入职后的工作。
## 后续处理的方法有哪些？
在开始分享后续处理方法之前，首先要明确一下，后续处理的方法可以分为以下几类：

- **提供信息：**后续处理的方法之一，就是提供你认为有用的信息。例如，你可能会收到面试通知，让你再补充一份证件材料，或是让你做一个调查问卷。只要你想通过后续处理得到更多的信息，那么就可以选择提供那些信息。
- **邀约面试：**这是后续处理的另一种方法，可以帮助你更好地了解你所需要的人才。你可以把你的个人信息和相关的职位信息分享给面试官，询问他们是否有意愿接收你，并且要给予一定的筛选。这样，当你能够成功地通过面试官的邀约，就可以进一步了解该职位的具体要求。
- **宣传推荐：**还可以采用这种方式进行后续处理。你可以邀请面试官参加线下活动、做相关的讲座，鼓励别人报名，甚至会有名企供应商推介。这有助于你展示自己拥有某项技能、经验，并增加自己的可信度和声誉。
- **开设门槛：**一般来说，许多企业都会设置比较高的入职门槛，因此，入职之后不一定能立即开始工作。这时，你可以尝试通过后续处理的方式让候选人先进一步地准备考核。你可以邀请他们参加一个考核面试，由专业的考核委员会进行审核，看看他们是否具备所需的技能及知识。如果你希望他们通过考核，你也可以进行一定的奖励措施，例如优惠政策或额外津贴。这样一来，你就可以为他们的能力提供更可靠的参考依据。
- **寻找帮手：**最后，还可以通过各种方式寻找帮手。你可以携带一份简历，并寄给在相关领域工作的同事，鼓励他们去面试你的候选人。这样，可以有效地扩展你的网络人脉，扩大你的影响力。也可以帮助你得到更多的职位邀约，提升你的职场竞争力。
以上只是后续处理的方法，实际情况远比这复杂得多。不过，总体来说，后续处理是一个良好的应对面试时的有效方式。它可以帮助你提升自己的能力、改善自己的职场生态，而且不会花费太多的时间成本。
