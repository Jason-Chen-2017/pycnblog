                 

# 1.背景介绍


20世纪末至21世纪初，计算机和信息技术的飞速发展彻底改变了人类的生活。早在上个世纪五六十年代，芝加哥大学的美国科学家卡尔·弗里德曼就已经提出“计算机时代”的概念，认为当代科技界将要进入一个全新的时代。这一理念被后续的许多学者、企业家和政客广泛接受，并得到普遍认同。

20世纪90年代初期，著名的蒙特利尔国际工商学院的教授格雷戈里（Gregor Easterly）等人，开始对人工智能领域进行研究。他们发现电力的发展速度远远落后于技术的发展速度，即使是核能也还不到摩尔定律的速度。为了能够继续提高人类社会的生产力水平，需要一种新的能源发动机。

1967年，法国人拉玛·苏姆林（Rama Sumner）、约翰·斯科特·希尔顿（John Stoytchev）和伊恩·费尔南多·凯利（Isaac Kent）三个人合作开发了一种利用液体火箭反复喷射燃料，从而实现大功率的循环发电机的新型发电机——内燃机。

20世纪90年代末期，随着美国制造业的转型升级，内燃机市场的需求增加。经过几十年的发展，美国的内燃机产量已经达到了每小时5万余台，占全美产量的超过七分之一。但同时，由于战争、贸易战、环境污染、物价上升等原因，美国内燃机市场仍然处于较低水平。

# 2.核心概念与联系
## 燃料与热
内燃机主要由燃料所驱动，包括煤油、石油、天然气等。这种燃料可分为粘性燃料和液化燃料。
### 液化燃料：液化燃料分为固体燃料和液体燃料。
#### 固体燃料
顾名思义，固体燃料就是指那些不通过气体或者液态的温度传导的燃料。最早的固体燃料是石棉燃料，用棉花和纺织品作为原料制造出来的。后来发明出来的炉子燃料就是一种很典型的固体燃料。还有瓦斯燃烧剂、碳酸钠、镁等，都是属于这种。

#### 液体燃料
液体燃料是在气体或者液体中含有氢气的燃料。最早的液体燃料有气体燃料，例如油漆、木炭、乙醇等；还有水、冰、水银、石墨烯等固体燃料，经过加工制成气体形状的气体燃料。

两种燃料类型的混合称为二合一燃料，这是一种利用气体燃料和液体燃料相互搭配的方式实现复杂循环的现象。

### 粘性燃料：粘性燃料具有无限缩短生命周期的特点，也能充当润滑剂。目前已知的最先进的粘性燃料有软骨架金属、纺织品棒、粘胶石等。

## 发电机工作原理
内燃机的运行原理可以总结如下：
- 第一步：将氢气压缩成推进剂，推进剂带动引擎转动。
- 第二步：引擎逐渐加速，推进剂带动金属材料膨胀成为推进齿轮。
- 第三步：推进齿轮绕过目标物体，燃烧其中的液体燃料产生热能。
- 第四步：推力极大，推进剂发生回旋跳跃，燃料装填车的推力推动齿轮绕过目标物体，并最终穿透目标物体。
- 第五步：当齿轮全部燃烧完毕，引擎停止转动，推进齿轮排出的金属箔体逐渐固化，产出带电能，供给电网。

## 控制原理
内燃机运行过程中，需要进行各种各样的控制，比如风扇控制、燃料充放、工作压力控制、电池充电管理、电池充电效率等。根据不同地区，还有不同的风险控制措施，如土壤污染防控、建筑破坏防控、安全行驶距离控制等。
