                 

# 1.背景介绍



## 一、为什么要写这个文章

程序员经常会遇到这样的问题——“编程题”“数据结构”“网络协议”等等，自我介绍了一下自己，但很多同事还是看不懂，很难做出决定是否入职。

作为技术人员，应该好好思考一下自己的定位，如何展示自己的才能，才能获得更好的待遇和帮助他人，面对未知，面对困难，能够快速解决问题才是最重要的。如果没有个人品牌，职场竞争就会越来越激烈。因此，提升自己的能力也是需要长期坚持的。

为了帮助程序员进步，我在网上看到了一篇文章《像程序员一样思考》，里面介绍了程序员的一些优秀品质，以及一些所需具备的素养。看完之后，我觉得这些观点也很值得关注。于是，我花了一个礼拜的时间，梳理出这些知识点，制作了这篇文章。希望能够帮到大家。

## 二、核心概念与联系

### 1. 说服力（Persuasion）

说服力是一个人的内在动机，他愿意向别人证明自己的价值并让别人相信自己。所以，说服力是人类的天赋属性。每个人都可以聘用心胸宽阔、富有想象力的说服者，只要他们愿意付出足够的努力，就能塑造出说服力强的人格。

如何做到有说服力呢？下面是一些方法：

1. 提供直接的解决方案或建议。不要刻板地宣传你的观念或理论，而是直截了当地告诉受众你的看法或建议。
2. 开诚布公。在讨论过程中，要及时提供反馈和鼓励，而不是一味推诿责任或对事情视而不见。
3. 尊重不同意见。不同的人有不同的观点，别人对某个问题的看法也可能不同。接受差异、尊重多元的声音，能够促使人们认识到自己和别人的不同之处，从而产生共鸣。
4. 坦率沟通。保持沉默和匿名。保持言谈保持技术水平。

### 2. 领导力（Leadership）

领导力是指一个人具有管理他人以及团队工作的能力。在日常工作中，领导力可以帮助我们超越自我，处理复杂的事务，团结一致的意志，擅长团队合作，提升组织效率。领导力代表着一个人的气质，也体现了这个人的品行。

下面是一些领导力的原则：

1. 自律。这代表着一个人对自己的要求非常苛刻，并且能够按照计划行事。只有自己能够做到自律，才能有成功的机会。
2. 沟通。如果一个人能够广泛地交流和学习，那么他的表达能力和沟通协调能力将得到加强。
3. 勤奋。勤奋可以促使人精益求精，不断尝试新的事物。即使失误也能通过反复试错来纠正错误。
4. 耐心。耐心和细致是成功的保证。每个人都有足够的耐心等待事件的解决过程。

### 3. 独立思考能力（Independence of thought）

独立思考能力代表着一个人的分析问题、总结经验、生成新 ideas 的能力。它可以帮助我们识别新的问题、发现更有效的方法、找到更好的解决办法。独立思考能力是影响工作效率、增加创新性的关键因素。

下面是一些独立思考能力的指标：

1. 逻辑思维能力。理解、分析、归纳事实、寻找原因和规律。
2. 决策能力。能够正确地判断和评估信息的重要性。
3. 创造能力。通过观察、分析、创造新的解决方法来创造新的价值。
4. 懂得权衡。理解不同选择的利弊，并据此作出取舍。

### 4. 研究能力（Research Capacity）

研究能力代表了一个人对新事物的敏锐洞察力和求知欲。他们可以分析复杂的现象、收集各种资料，甚至是演绎和总结出科学的规律。研究能力可以帮助我们进行突破性的发展，找到更为可靠、更有效的解决方法。

下面是一些研究能力的指标：

1. 博览群书。阅读专业书籍，了解相关领域的最新动态和理论。
2. 自学能力。熟练掌握计算机语言、工具、软件，能够快速解决实际问题。
3. 数据分析能力。掌握各种统计学、数学方法，能够处理、分析大量数据。
4. 团队精神。学会适应变化，能够随时跟踪和评估自己的工作成果。
