                 

# 1.背景介绍


## 一、数学背景介绍
随机算法和概率论在信息科学、数学、工程等各个领域都扮演着极其重要的角色。其中随机算法的应用遍及电子游戏、数据安全、网络流量调控、信号处理、概率统计、经济模拟等多个领域。概率论则主要用于信息、生物、社会、经济、金融等方面的研究。概率论的基础就是离散数学、代数学以及集合论。

## 二、基本术语定义
### 1.随机变量(Random Variable)
随机变量通常用字母X表示，它是一个取自一个有限或无限集合的实值函数。例如：抛硬币，一次抛掷有两种可能的结果“正面”或者“反面”，则硬币出现正面或反面这种结果就是随机变量X的取值。如果将这个事件的结果做极限分解，即把所有可能结果的全体作为样本空间，那么X就是一个随机变量。对于连续型随机变量来说，X的值可以用一个连续函数来描述。

### 2.样本空间(Sample Space)
样本空间是指所有可能的随机变量取值的全体集合。例如：一次抛掷硬币，样本空间为{H,T}。如果样本空间是无限集合，称之为全体或样本空间。

### 3.分布(Distribution)
分布是对随机变量可能取值的概率进行给定的描述，表示随机变量的特征。分布有很多种形式，最简单的分布形式就是表格形式。表格中每一行对应一个取值，每一列对应一个概率。例如：抛硬币分布如下所示：
| 事件 | 概率（或概率密度） |
| --- | ---- |
| H | 1/2 |
| T | 1/2 |

### 4.期望(Expectation)
期望是指随机变量X的数学期望，用E(X)表示。期望值是随机变量的平均值，通过计算随机变量的各种可能取值对其概率乘积之和得到。例如：抛一次硬币，其期望值为1/2。

### 5.方差(Variance)
方差是衡量随机变量X的离散程度的一个指标，用Var(X)表示。方差越小，说明随机变量X的取值相差越少；方差越大，说明随机变量X的取值相差越多。方差刻画了随机变量的变异性。例如：抛两次硬币，其均值（期望值）分别为1和3，方差分别为1和2。

### 6.协方差(Covariance)
协方差是衡量两个随机变量X和Y之间的线性相关关系的一个指标。如果X和Y的分布同时变化，则称它们具有共同的协方差。协方差用Cov(X, Y)表示。协方差等于X和Y同时变化时，X偏离其期望值的距离（离散程度）乘以Y偏离其期望值的距离。当协方差为正时，说明X和Y之间存在正向线性关系；当协方差为负时，说明X和Y之间存在反向线性关系；当协方�为零时，说明X和Y之间不存在线性关系。例如：X和Y的分布为均匀分布，即P(X=x) = P(Y=y)，协方差为0。
