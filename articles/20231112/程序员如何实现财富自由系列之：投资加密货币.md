# 2.核心概念与联系



## 2.1.加密货币

加密货币，简称加密，是一种基于区块链技术的新型支付结算系统。它允许在网络上进行即时的点对点交易，并通过分布式记账的方式记录所有账户的余额变更信息。

目前，世界上最大的加密货币交易所Binance，已经推出了其数字货币交易平台“币安”，有超过70%的用户通过该平台进行数字货币交易。可以说，加密货币市场已经成为一个庞大的生态圈，其交易规模和参与者的规模都处于全球前列。

值得注意的是，加密货币和以往的虚拟货币有着根本性的不同。虚拟货币通常是由实体法人或组织发行的数字资产，可以被视为真正的货币，拥有独特的货币属性。而加密货币是通过区块链技术建立起来的支付系统，所有的用户及交易行为都会被记录到公开的区块链上，任何人都可以查询。

## 2.2.ICO(Initial Coin Offering)

ICO（Initial Coin Offering）即初始货币发行，是指通过发行加密货币的方式向市场推广新币种。ICO在流程上和普通的公司股权众筹类似，但区别在于有明确的目标，创始团队需要有丰厚的资金支撑，且大量的资金需要由早期的参与者认购。

与一般的股权众筹不同的是，ICO一般具有较高的吸引力和回报率，参与者需要首先提供一些现金或其他资产作为认购资金。通常情况下，ICO的价格非常保守，只有在达到一定程度的资金激励后才会上升到正常的价格水平。

## 2.3.DEX（Decentralized Exchange）

DEX是去中心化交易所，它是使用区块链技术的去中心化交易平台。它不受地理位置限制，用户可自由访问交易所，无需注册，不收取交易手续费，没有中间商赚差价。

与传统的中心化交易所相比，DEX的优势在于成本低廉、效率高、安全可靠、隐私保护方面得到了保证。同时，由于采用区块链技术，用户的交易记录是完全公开透明的，不会受到任何篡改或伪造的影响。

目前，国内外几乎所有的交易所都已支持基于以太坊区块链的DEX服务，例如币安、OKEx等。


## 2.4.比特币现金（BCH）、莱特币（LTC）、以太坊（ETH）……

除了以上提到的加密货币项目外，还有许多其它加密货币项目正在逐渐发展壮大。例如，比特币现金（Bitcoin Cash）、莱特币（Litecoin）、以太坊（Ethereum）……这些都是数字货币的代币名称。它们都属于同一套加密货币体系，只是采用不同的共识机制、区块确认时间和挖矿难度。大家可以通过对比各种代币之间的相似之处，从而更好地了解加密货币的基本特性。

## 3.1.确定投资目标与期望收益率

首先，需要明确自己对加密货币的认知和兴趣，包括以下几个方面：

- 投资范围：加密货币主要分为硬件钱包、Web3智能合约、NFT、DAO等不同类型。
- 投资目的：是否有着高收益、高风险的投资理念。
- 投资期限：整个投资周期的时间长短。

其次，确定自己的期望收益率。通常情况下，投资者会选择某个币种，看看该币种是否能够实现其预期收益的目标。如果某项策略能产生显著的超额收益，则该策略可能会被认为是有效的。



## 3.2.研究投资领域内的理财产品和策略

研究领域内的理财产品和策略。理财产品有不同的分类方式，如偏股票型、偏债券型、偏货币型、偏混合型等。不同类型的理财产品又可以分为个人型和机构型。不同的投资者也会关注不同的理财产品和策略。

在选择理财产品时，也需要根据投资者的经济情况、信用状况、风险承受能力和偏好进行综合判断。当存在多个投资选项时，经验丰富的投资者可能更倾向于选择风险比较小、抗风险能力强、收益回报率高的产品。



## 3.3.研究加密货币投资的历史发展及规律

加密货币投资的历史发展和规律。加密货币投资历史上主要有三种模式：

- 主流币种投资模式：最初的加密货币投资主要集中在以BTC为代表的主流币种。随着大众对密码学货币的关注度逐步提升，近年来更多的主流币种开始被纳入加密货币投资考虑。
- 智能合约投资模式：近些年来，智能合约也被证明为一种比较好的投资方式，其能够帮助个人和企业降低风险，提升效率。
- 期货投资模式：随着期货市场的出现，加密货币投资也有机会进入到期货市场。



## 3.4.了解各类加密货币的特性

了解各类加密货币的特性。这里，我们只讨论最重要的加密货币：比特币。



### 比特币的特性

比特币（BTC）的创世纪区块高度为0，同时也是全网最先启动的区块链项目。比特币的创世区块是用加密技术生成的一段数据，里面包含了创建比特币网络的所有基本信息，其中包括：

- 区块头（Block Header），记录区块的基本信息；
- 上一个区块的哈希值（PrevHash），指向上一个区块的散列值；
- 生成区块的节点的工作量证明（PoW）。

比特币的发行数量是通过难度自我调节的算法控制的，创始人的数量也是固定的。为了避免被攻击和垄断，比特币采用Proof of Work（PoW）的共识机制。比特币的单位是“聪（Satoshi）”，每个BTC的实际价值是确定的。每一次交易的费用（交易手续费）也是确定的，但是按比例计算。



## 3.5.选定加密货币的买卖策略

选定加密货币的买卖策略。目前，投资加密货币有两种策略：

- 做空策略：相对于市场价格，持有加密货币的方式称为做空。这种方式能够更加安全、稳健。
- 做多策略：相对于市场价格，持有加密货币的方式称为做多。这种方式能够获得更高的收益。



## 3.6.制定买卖加密货币的交易计划

制定买卖加密货币的交易计划。选择加密货币的交易周期、金额、次数，以及手续费。可以参考网站提供的工具进行交易计划的制作。也可以根据自己的经验和知识，进行市场分析和结合自己的观察和感受，制定出适合自己风险承受能力和能力的交易计划。

from pycoingecko import CoinGeckoAPI
cg = CoinGeckoAPI()

## 4.2.Java代码示例

```java
import java.io.IOException;
import java.util.List;

import org.apache.http.HttpResponse;
import org.apache.http.client.ClientProtocolException;
import org.apache.http.client.HttpClient;
import org.apache.http.client.methods.HttpGet;
import org.apache.http.impl.client.HttpClientBuilder;
import org.json.JSONArray;
import org.json.JSONObject;

public class GetCoinData {

    public static void main(String[] args) throws ClientProtocolException, IOException {

        String url = "https://api.coingecko.com/api/v3/coins";
        HttpClient client = HttpClientBuilder.create().build();
        HttpGet request = new HttpGet(url);
        HttpResponse response = client.execute(request);

        if (response!= null && response.getStatusLine().getStatusCode() == 200) {
            JSONObject obj = new JSONObject(EntityUtils.toString(response.getEntity()));

            JSONArray arr = obj.getJSONArray("coins");
            for (int i=0;i<arr.length();i++) {
                JSONObject item = arr.getJSONObject(i);
                System.out.println(item.getString("name"));
                System.out.println("symbol:"+item.getString("symbol")+"\n");
            }
        }
    }
}
```

输出结果：

```
Bitcoin
symbol: btc

Etherium
symbol: eth
...
```

## Q：加密货币的分类有哪些？

加密货币分为硬件钱包、Web3智能合约、NFT、DAO等四种。

硬件钱包是指加密货币钱包的物理形态。它是一个独立的设备，用来存储、接收、发送、转账加密货币。硬件钱包的发展方向是以硬件形式存储数字货币，并具有极高的安全性和隐私性。

Web3智能合约是指基于区块链的分布式应用编程接口，让开发者可以方便的构建基于区块链的应用。目前，Web3智能合约的平台有以太坊、Cosmos等。

NFT（Non-Fungible Token）是指不可互换的数字资产。它是一个代币，能够赋予数字资产非同质性和唯一性。NFT有着独特的艺术、游戏和文化价值。

DAO（Delegated Autonomous Organization）是指自托管的有组织结构的社区。它是一个自治机构，由一组决策者组成，决定着某些事情的发生。DAO旨在通过治理的方式解决系统中的权力和控制问题。
