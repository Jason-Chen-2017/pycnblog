                 

# 1.背景介绍


## 数据类型
计算机程序语言分为机器语言、汇编语言、高级语言。而Python 是高级编程语言中的一种。Python 在计算机底层对内存进行了管理，因此可以用很多种数据类型存储数据。比如整数型（int）、浮点型（float）、字符串型（str）、布尔型（bool）、列表（list）、元组（tuple）、字典（dict）。它们之间的区别和联系是什么呢？为什么在Python中要区分这几种数据类型？本章节将会介绍这些知识。

## 变量
变量就是用来存储值的容器。它包括两部分：变量名、变量值。例如，x=10就是一个变量名为x的变量赋值为10。每个变量都有一个唯一标识符，即变量名。变量名可以由字母、数字、下划线组成，但不能以数字开头。变量的值可以是任意的数据类型，如整数、实数、字符、字符串等。

# 2.核心概念与联系
## 数据类型
数据类型是指用于定义变量存储值的类型。比如整数型、浮点型、字符串型、布尔型、列表、元组、字典。不同的数据类型决定了变量所能存储的值的类型及其大小，并规定了对变量进行运算的方式。在Python中，可以使用内置函数type() 来查看变量的类型。对于Python来说，有两种数据类型：一种是内建数据类型，一种是用户自定义数据类型。内建数据类型包括整数型、浮点型、字符串型、布尔型、NoneType；用户自定义数据类型可以自己创建，比如类、模块、函数等。除了内建数据类型外，还有一些特殊的数据类型，例如集合、迭代器、生成器等。我们把所有的内建数据类型统称为“原生数据类型”，把所有的用户自定义数据类型统称为“复合数据类型”。

## 数据结构
数据结构是指数据元素之间关系的描述。比如数组、栈、队列、链表、树、图。不同的数据结构选择不同的方式来组织数据元素，对同样的数据结构的操作方式也不同。数据结构是编程的基础，是影响程序运行效率的一个重要因素。在Python中，有以下几种数据结构：

1. 列表：列表是最常用的数据结构之一。它是一个有序的序列，可以存放各种类型的数据。列表的索引从0开始，可以通过方括号[]来访问列表中的元素，还可以通过索引来修改或者删除元素。

2. 元组：元组是不可变的列表。它是一种有序序列，可以使用圆括号()来创建，也可以通过索引来访问元素。但是，元组不能修改元素值，只能创建新的元组。

3. 字典：字典是另一种常见的数据结构。它是一个无序的键-值对集合，可以根据需要添加或删除键值对。字典的键必须是唯一的，而且可以用方括号[]来访问其对应的值。

4. 集合：集合是一种无序的序列，不重复的值的集合。它的功能类似于数学上的集合，可以快速查找某个元素是否存在。

5. 序列：序列（sequence）是Python中的一切数据结构的顶级父类。它包括列表、元组、字符串、字节串等。序列有共性，比如都可以被索引访问、切片、拼接等操作。

## 可变与不可变
Python 中所有的数据类型都是不可变数据类型，这是由于 Python 的设计者们认为这样可以提高程序性能。当创建一个对象的时候，内存空间里就会分配好相应的数据。如果数据发生变化时，内存地址就不一样了，导致之前的旧地址的数据就失去了意义。

对于可变数据类型，比如列表，当数据发生改变时，内存地址不会改变，只是改变列表内部的元素指向地址，所以不产生新的内存地址。
