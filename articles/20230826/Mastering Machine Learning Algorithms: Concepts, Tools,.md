# 1.简介
  

## 概述
《Mastering Machine Learning Algorithms: Concepts, Tools, and Techniques for Data Scientists and Developers》由两位知名机器学习领域的顶级专家共同撰写，汇集了数据科学家、机器学习工程师、算法工程师、软件工程师等不同角色的一系列技术文章。本书将教会读者机器学习中最常用的算法及其相关概念，包括决策树、线性回归、朴素贝叶斯、KNN、SVM、聚类分析、关联规则 mining、协同过滤、推荐系统、深度学习、强化学习等内容。同时，还将分享一些实用工具、资源以及有关机器学习研究的最新进展。读者不仅可以从这本书中获益，而且也可以帮助自己构建自己的知识体系并进行更深入地理解。此外，本书也是其他机器学习相关书籍的前传，全面覆盖了机器学习的各种方面，成为一门实用而系统的机器学习指南。
## 作者简介
作者一：吴恩达，美国亚利桑那大学商学院客座教授，在该校担任助教，主要研究方向为统计学习方法、强化学习、机器学习。他的博士论文题目“Adapting Boosting to Differentiable Loss Functions”就使用了强化学习中的梯度法来训练分类模型。其后又参加了 Google 的机器学习研讨会，并在 UC Berkeley 担任访问学者。

作者二：李宏毅，美国德州大学奥斯陆分校计算机科学系博士，主要研究方向为机器学习理论、计算理论和优化算法。其博士论文于2017年发表，题目为“Differentially Private Empirical Risk Minimization with Adaptive Noise Budget”，提出了一个差分隐私保护下的经验风险最小化算法。除此之外，李宏毅在加拿大多伦多大学、斯坦福大学等处也拥有长期工作和研究机会。
## 关于这本书
《Mastering Machine Learning Algorithms: Concepts, Tools, and Techniques for Data Scientists and Developers》这本书的创作背景和目的都十分简单明了。它是一个开源的图书项目，作者希望通过提供详实的技术文档来帮助各位数据科学家、机器学习工程师、算法工程师和软件工程师快速上手并掌握机器学习。这个目标已经十分清晰，因此，这本书的内容也十分丰富。同时，它也是一本系统的机器学习指南，涵盖了机器学习各个方面的内容，包括算法理论、技术细节、编程实现、评估与比较。

另外值得一提的是，本书通过精心制作的目录结构，把机器学习的众多主题进行了完整的梳理，并对每一个主题都给出了极具挑战性的实践案例，并力求让读者感到有所收获。这种编写形式既适合数据科学家阅读，也很容易被非技术人员理解。这本书既有教程的形式，更有实践的项目应用，这些知识点能够帮助读者快速掌握机器学习的基本方法、理论和技巧。而这本书中深入浅出的论述和对比，也为读者提供了理论联系实际的帮助。

最后，《Mastering Machine Learning Algorithms: Concepts, Tools, and Techniques for Data Scientists and Developers》一书的中文版《机器学习算法详解》也即将出版，欢迎有兴趣的读者前往购买。
