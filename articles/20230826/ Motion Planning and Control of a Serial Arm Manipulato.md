# 1.简介
  

在本研究工作中，我将重点讨论串联机械手臂运动规划与控制。串联机械手臂（serial manipulator）是一个具有多个关节的机器人，它的关节之间通过连杆相互连接而成。串联机械手臂运动规划与控制是指将其从初始状态逐渐逼近到目标状态的过程。传统上，串联机械手臂控制通常采用基于逆遗传算法或者其他优化算法的方法。然而，这些方法对于复杂、非线性的系统控制问题往往存在着不适应性和效率低下的问题。为了解决这些问题，本文提出了一种新的串联机械手臂运动规划与控制方法——基于混合控制理论的序列博弈策略。此外，本文还设计并实现了一套开源的软件包，用于模拟、仿真、测试和验证串联机械手臂运动规划与控制的理论模型与实践方案。因此，本文旨在提供一系列有效、准确、高效的串联机械手臂运动规划与控制方法，同时启发读者对机器人控制技术的发展方向进行更深入的探索。
# 2.术语说明
在本章中，首先定义以下所需使用的术语。
- 惯性坐标系（Inertial Coordinate System,ICS）: 坐标系中的任何一个点都可以用三个自由度的旋转矩阵描述其相对于该坐标系的姿态角和位置。惯性坐标系主要由重力加速度计和磁力计所提供。
- 末端固定坐标系（Fixed End Effector Coordinate System, FEECS）: 在末端固定坐标系中，运动的关节只受力的限制，力的方向代表关节的转向。它也被称作TCP（tool center point）。FEECS的选择要考虑末端固定件的自重、摩擦弹性以及物体的密度。
- Jacobian矩阵：Jacobian矩阵是通过求导得到，它是一个列向量组成的矩阵，其中每一列对应于刚体的质心坐标偏移。它描述了刚体坐标系（body coordinate system）中的任一点所产生的约束冲量。
- Cartesian空间（Cartesian Space）: 在笛卡尔空间中，点被表示为三维空间中的笛卡尔坐标（x,y,z），即位置坐标和指向坐标。
- Joint space(关节空间): 关节空间是在两个不同关节之间的相对坐标系，它们之间的关节直径为零，所以称之为“关节空间”。关节空间坐标轴的原点在两个关节的连线上，并且沿着两条轴平行。坐标轴的方向由两个关节的起始点确定。
- Task space (任务空间) : 任务空间描述的是目标点或工作区，机器人的动作需要围绕这个工作区进行。在任务空间中，各个坐标轴与地面坐标系一致，X轴指向右侧，Y轴指向前方，Z轴指向上方。
- Tool frame （工具框架）: 工具框架描述的是与工具挂钩的参考系，在工具框架内，X轴与工具轴垂直且指向工具基部的正方向；Y轴指向工具的左侧，Z轴垂直于水平工具轴。
- Control Law（控制律）: 在机器人运动规划中，控制律是根据当前状态、目标状态及后续动作等信息计算出的用来矢量化描述机器人轨迹的函数。通常情况下，控制律由多种控制算法组合而成。
- Trajectory （轨迹）: 机器人在运动规划和控制中所形成的轨迹是一个点集或曲线，它由一系列由时间（t）和空间（x，y，z）坐标系所定义的点组成。

## 3.1 运动规划问题
串联机械手臂运动规划问题就是在已知目标位置以及一些运动学特性（例如关节长度、结构参数、质量等）的情况下，确定机器人各个关节的位置和姿态角度，使得整个机械臂从初始状态（例如初始姿态或初始位置）逐渐移动至目标位置，使得其接触目标物体。如下图所示，机器人由若干个关节组成，每个关节可能是一段弹簧搭接，连接到一起形成完整的机械臂。串联机械手臂运动规划一般分为运动学规划、动力学规划和控制规划三个阶段。
