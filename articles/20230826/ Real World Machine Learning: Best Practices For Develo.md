## 常见问题与解答
1. 为什么要研究机器学习？
  机器学习（Machine Learning，ML）技术越来越火爆，其发展历史可追溯至1959年费罗纳德·李宏毅和戴明·格鲁奇的两篇论文。当时科学界对机器学习技术还处于刚刚兴起的阶段，所以很多人并没有充分理解它的内涵和作用，所以才会产生许多误解。随着技术的发展和应用越来越广泛，机器学习技术已经成为新一代人工智能技术的支柱之一。因此，无论是想成为机器学习专家还是普通工程师，掌握机器学习相关的基础知识都是不可或缺的一步。

2. 机器学习算法的分类有哪些？
  机器学习算法可以根据不同的任务类型、数据量、输入变量和输出形式，以及使用的硬件平台，被划分为不同的类别。以下是常见的机器学习算法的分类：

  1. 监督学习
     监督学习（Supervised learning）是一种基于样本数据集进行训练，并利用已知目标变量的情况下，对未知数据的预测和分类的方法。监督学习方法包括分类、回归和聚类。
     
  2. 无监督学习
     无监督学习（Unsupervised learning）是指对数据集中的对象进行聚类、概率分割或分类的方法。其中，聚类方法对数据集中的对象进行分组，使得具有相似特性的对象归属于一类，而分类方法则是确定每个对象属于哪一类。
  
  3. 半监督学习
     半监督学习（Semi-supervised learning）是在训练数据存在少量标记的情况下，对整个数据集进行训练的方法。
     
  4. 强化学习
     强化学习（Reinforcement learning）是指一个agent在环境中不断尝试最大化累计奖赏（reward），并期望以此来影响环境的行为。其特点是试错学习，也就是通过不断探索来寻找最佳的策略。
  
  5. 集成学习
     集成学习（Ensemble learning）是利用多个基学习器组合起来改善学习效果的方法。它是指综合多个学习器的预测结果，来获得比单一学习器更好的预测结果。
     
  6.  deep learning（深度学习）
     深度学习（Deep learning）是利用深层神经网络来学习数据的特征表示的方法。深度学习由三部分组成：神经网络、激活函数和损失函数。
    
3. 有哪些常用的机器学习库？
  有如下几种常用的机器学习库：
  
  1. Scikit-learn
     Scikit-learn是Python的一个机器学习开源库，提供了常见的机器学习算法，如支持向量机、决策树、k近邻、逻辑回归等。
  
  2. TensorFlow
     TensorFlow是谷歌开源的机器学习框架，可以用于构建各种复杂的机器学习模型，可以实现自动求导、分布式计算、可视化、可移植性等功能。
  
  3. PyTorch
     PyTorch是Facebook开源的机器学习框架，功能强大且易于上手。它基于动态图机制，可以快速地进行模型构建和训练，适用于大规模数据处理。
  
  4. Keras
     Keras是另一种流行的机器学习库，它支持TensorFlow、CNTK、Theano后端。Keras是一个高级的基于符号编程的机器学习API，可以快速轻松地搭建深层神经网络模型。
  
  5. Caffe
     Caffe是一个基于CNN的开源库，其底层采用C++语言，可以高度定制化，可以在不同硬件平台上运行。
