# 1.简介
  
：

从高考结束到大学毕业，在计算机专业中学习编程，我一直坚持用大块的时间去学习编程相关知识，而学习编程过程中的痛苦经历也常常让我烦恼。今天，不妨分享一下自己学习编程时的一些经验、教训、心路历程，帮助更多的人了解如何走进编程世界。

# 2.高考前半段的编程学习之痛

高考前期，由于中国的高考制度特殊性，普通高校一般都会设置编程类选拔。这意味着学生们要花费大量的时间在编程上，而且往往只占到高考成绩的一小部分。由于对编程能力的重视，学生们一定会用各种方法学习编程，包括看书、刷题、看视频、观课等。然而即使是简单的变量赋值语句，阅读源代码也需要大量时间和精力。因此，学习编程的时候，往往需要很多重复劳动。

这种学习方式很耗时，并且收效甚微。经过几年的学习，学生们最终还是会忘记或遗忘关键的编程知识点，导致后面的工作进展受阻。而且，由于不断涉及的新知识，习惯了编程的人也很难适应学习编程的节奏。所以，高考前期的编程学习往往伴随着巨大的焦虑感，甚至到最后也没有坚定下来的学习方向。

# 大学第一学期的编程学习之痛

如今，大学计算机专业的课程结构较为复杂，大量的实践项目和作业。因此，学生们往往对编程能力更加渴望。这也是很多学生选择以计算机专业为导向，进入大学学习编程的主要原因。但是，进入大学后的第一学期，编程学习往往会遇到很多困难。

首先，是基础知识匮乏。很多高校的计算机系课程都只授予一些C语言、数据结构和操作系统方面的课程，而没有学习Python、Java或者JavaScript这样的高级编程语言。其次，还有就是编程工具的使用经验不足。学生需要学习各种编程工具，比如集成开发环境（IDE）、编辑器、编译器、调试工具等。这些工具的使用过程都比较曲折，而且还有不少陌生的操作，让学生头疼不已。再者，编程思维还不是很成熟，很多学生容易陷入编码思维的盲区，不知如何解决实际的问题。

因此，大学第一学期的编程学习，仍然会伴随着前所未有的挫败感和厄运。

# 大学第二学期的编程学习之痛

相比于大学第一学期的艰辛，大学第二学期似乎已经变得轻松许多。因为许多高校的计算机系在本学期开始修建编程环境，而且也开设了一门实践班，专门培养学生的编程能力。这也让学生有机会接触到大量的实践项目，提升自己的编程能力。但同时，这也给学生带来新的压力，需要通过课外学习来提升自己的编程能力。

例如，许多学生可能会担心自己的编程水平太低，无法胜任复杂的工程项目。因此，学生可以参加一些社团活动，组织编程比赛，学习编程的软技能。另外，有些学生可能希望直接跳出编程的单纯性，转向机器学习领域。而进行机器学习之前，学生需要掌握编程基础，因此也需要付出更多的努力。

总的来说，在大学第二学期，学生们学习的重心是提高自己的编程能力，而不是仅仅为了学习编程。因此，编程学习过程并不会非常枯燥，反而会获得非常丰硕的回报。

# 3.如何解决学习编程的痛苦

虽然编程学习是一个充满挑战、费时费力且有挑战性的过程，但在今日的互联网行业里，无疑是最值得骄傲的一种技能。作为程序员，你的职责就是通过编写代码，让程序能够实现目标功能，创造价值。所以，如何解决学习编程的痛苦，是所有程序员的共同追求。下面就让我们一起探讨一下如何解决学习编程的痛苦。

# 1.编程语言的选择

首先，应该明确学习哪种编程语言。推荐学习的语言有Python、Java、JavaScript、C++，等等。每种语言都有不同的学习曲线，但它们都可以用来构建完整的应用程序。由于不同语言之间的语法差异和特性差异，选用哪个语言都取决于个人喜好和具体需求。如果学完所有这些语言，可能有些吃力，但不会有什么损失。

# 2.自学还是买书

对于初学者来说，建议从书籍入手。这通常是学习编程最有效的方法。书籍可以帮助我们快速理解编程的基本概念、语法和机制。另外，许多优秀的开源项目的源码都是由多名程序员编写的，这可以极大地激发我们的学习兴趣。

如果自学，需要认真准备，花费大量的时间投入学习。建议找一个讲师指导自己学习，他/她可以教你一些重要的编程理论知识。自学的过程需要耐心、 persistence 和 持续的反思。

# 3.计算机科学和编程的关系

计算机科学是研究如何将指令转换为可执行的代码，它涉及硬件系统、软件工具、算法和编程语言等多个方面。编程语言是人类用来与计算机沟通的语言。计算机科学是工程师用高级编程语言进行软件开发的重要组成部分。编程不仅仅是完成任务的能力，更是一种知识、能力和方法。

# 4.最好的学习方法是模仿

学习编程的最佳方式莫过于模仿别人的成果。如果你看到别人写的代码，觉得他们的代码写得很漂亮，你可以模仿这种风格。模仿别人的代码既可以加强自己的编程能力，又可以提高效率。

# 5.如何提高自己的编程能力

要想提高编程能力，除了刻苦钻研之外，还可以通过以下途径：

1. 拓宽编程思维
举个例子，假如你面临一个问题，该如何解决呢？如何才能写出正确、健壮、可维护的代码呢？不要急于表现出来，而是先问自己这些问题，逐步找到自己的答案。

2. 使用文档和教程
当你遇到问题时，找一份详尽的文档或教程来学习。这可以帮助你快速查阅资料，提高解决问题的效率。另外，你也可以向身边的朋友、同事倾诉你的问题。让大家共同帮助你，你就会发现这个问题并非完全难以解决。

3. 提供反馈
这是学习过程中的一个重要环节。当你学习新知识时，一定要提供反馈。比如说，你看到某段代码运行错误了，要及时向其他人证明自己的错。你也可以将自己的疑惑和问题发布到社交网络，鼓励他人帮助你。

# 6.对比编程与其他艺术

计算机科学和编程是两个完全不同的领域，二者之间有很大的不同。编程与绘画或音乐不同，它需要有高度的逻辑思维能力、技艺和编程技术。

因此，编程可能与其他艺术有很多相似之处，比如美学、抽象、写作技巧。然而，编程与其他艺术之间也存在显著的区别。编程关乎算法和编程技巧，而绘画和音乐则侧重于美学和感染力。

# 7.总结

尽管编程学习过程会有诸多困难，但它是一个有趣、具有挑战性的领域。如何平衡学习与工作、自我发展，才是成功的关键。
