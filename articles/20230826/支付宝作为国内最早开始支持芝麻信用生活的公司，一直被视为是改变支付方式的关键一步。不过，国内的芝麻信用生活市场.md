# 1.简介
  

芝麻信用(芝麻信用生活)是支付宝生态中新颖的一种支付方式。由于手机支付机具普及率低、转账成本高等因素，芝麻信用无疑将为支付宝带来便捷、省心的转账服务。据报道，截至2021年3月末，支付宝在全球26个国家开设了芝麻信用生活业务的账户，包括香港、澳门、台湾、韩国、日本等，累计用户数超过70万。
# 2.概念术语
## 2.1 概念
**芝麻信用生活**是支付宝生态中的一种新的支付方式。主要优点如下：

1. **省心：**在线支付更加方便，用户无需到实体店取卡或刷卡。

2. **安全可靠：**通过芝麻信用认证后，用户可以享受支付宝系统提供的专属保护机制，保障付款信息的安全性。同时，还可以获得额外的信用分，提升交易额度。

3. **用户满意度高：**芝麻信用生活功能集成了支付宝的众多好评率，并打通了自助服务领域，用户可以在平台上查询、管理、优化收支明细，获取丰富的信用生活资源。

4. **支付场景多样化：**用户除了可以在支付宝完成正常的消费支付之外，也可以在芝麻信用生活进行信用消费。芝麻信用生活可以满足消费者不同类型需求，如便利商店、超市、大型购物中心、出行等。

5. **海量用户选择：**从最初的5个国家的芝麻信用生活账户发展到目前的26个国家的芝麻信用生活账户，依托支付宝生态，为广大用户提供了便捷的转账支付服务。

## 2.2 术语
**芝麻信用认证**：芝麻信用认证是指支付宝用户在使用芝麻信用生活时，需要提交身份信息、银行卡信息以及其他相关信息，由支付宝提供的免费、安全认证过程。

**芝麻信用分**：芝麻信用分是指支付宝根据用户的信用历史数据计算得到的综合评分。其评分体系以商家的信用记录、用户实名制信息、手机号码和支付交易数据等多个维度分析，对每笔交易进行客观的分级，以达到保障用户权益和支付安全的目的。

**芝麻分提现规则**：芝麻分提现规则是指用户确认芝麻信用分足够（一般要求大于等于1000分）后，可以申请提现。提现金额以“可提现额度”为准，即账户当前有效信用总额的1%。

**芝麻分兑换规则**：芝麻分兑换规则是指用户确认芝麻信用分足够（一般要求大于等于1000分）后，可以申请兑换。兑换规则以平台规定为准。

## 3.2 芝麻信用积分系统介绍
### 3.2.1 芝麻信用积分
芝麻信用积分是支付宝新推出的一项体系，旨在建立一个公平透明、客观公正的芝麻信用评价体系。该体系将“信用分”作为用户实时了解自己的信用状况的一种重要指标，支持用户在支付宝进行信用消费、抵扣等操作。用户每成功一次芝麻信用行为，都会产生相应的芝麻信用积分奖励。

### 3.2.2 芝麻信用积分机制
芝麻信用积分的机制如下：

1. 用户进行芝麻信用支付时，支付宝会识别用户身份信息、银行卡信息、地址信息等，并进行风险控制；

2. 支付宝核对用户信息无误后，将用户信息数据打包发送至芝麻信用评分中心进行信用评估，完成芝麻信用认证和芝麻信用积分奖励。

3. 如果用户持卡人未经授权，拒绝提供个人信息，造成账户资产被冻结，造成不良后果的，按照芝麻信用支付条款取消其信用积分奖励，并通过法律手段追究其法律责任。

4. 每次用户完成支付或充值，如果芝麻信用认证成功，支付宝将给予相应的芝麻信用积分。

5. 在实际应用中，支付宝会对不同用户的芝麻信用积分进行统计，进行排行榜展示，同时提供芝麻信用积分兑换福利，引导用户进行信用交易。
