# 1.简介
  

随着信息化和数字化发展，人们生活中的方方面面都被数字化，不仅对个人生活变得更便捷，也使得个人生活中的各个领域之间产生了互动，形成了所谓的“社会系统”。而人的认知、行为、情绪、甚至生理活动也被数字化收集、分析处理。例如，通过GPS和社会网络数据，你可以很容易地知道你身边发生了什么，以及与他人的关系是怎样的；从你的照片、视频、聊天记录等媒体资料中，你就可以发现你到底是一个什么样的人。而这种能力、数据的获取、处理与分析在过去几年越来越受到重视，并得到越来越多应用。然而，如何设计、构建这样一个复杂而庞大的社会系统，成为一名社会系统的设计者，却依然是一项繁重的任务。如何将自己定义为社会系统的设计者，成为了一个重要的课题。

本文试图通过回顾当前的研究进展及其挑战，阐述我国社会系统设计领域的最新热点话题，以及社会系统设计过程中的关键角色、技能、方法论以及思维模式等，结合中国特色社会主义理论，提出社会系统设计专业道路上的规划建议。希望能够激起读者对社会系统设计专业发展方向、知识结构以及相关政策措施等有更深入、全面的理解。

# 2.基本概念术语说明
## 2.1 社会系统
社会系统（social system）是一个相互作用而共同发展的群体的总称或整体，是指由一系列互相依赖的、按照一定规则运行的个体组成的制度、文化、组织、经济、军事、社会秩序和等级制度，这些制度、文化、组织、经济、军事、社会秩序和等级制度互相作用，产生影响，共同产生各种社会现象、社会关系以及人类生活。社会系统是一个不可分割的整体，它由制度、法律、规范、组织、信息技术、服务方式、心理因素等构成，包括物质、精神、行为三个层次。其中，物质层次又包括经济、金融、房地产、交通运输等领域，精神层次则包括组织、管理、意识形态等领域，行为层次则包括企业、市场、政府部门、人民群众等。社会系统就是在这三层次上进行交流、沟通、联系、合作、互动，产生制度、法律、规范、组织、信息技术、服务方式、心理因素等多个层面协调性的社会运行的系统。社会系统的定义如此模糊和复杂，可以让人感到非常困惑。但是，笔者认为，“系统”二字作为一个泛指，已经不能完全涵盖社会系统的全部要素。因此，我国社会系统通常以组织系统、信息系统、教育系统、医疗卫生系统等具体的名称来描述。

## 2.2 中国特色社会主义
中国特色社会主义是指党的领导下，中国共产党高度重视马克思列宁主义、毛泽东思想、邓小平理论、“三个代表”重要思想、科学发展观、习近平新时代中国特色社会主义思想为指导，坚持走中国特色社会主义理论体系，贯彻落实科学发展观，创造性破解发展难题，推进国家治理体系和治理能力现代化，在各方面取得重大战略性成果的社会主义国家制度。2018年12月，习近平主席在参加完中央全面依法工作会议后首次提出，要充分发挥好立法权，有效履行职责，积极推进政务公开和法治建设，确保人民政权长久执政。从1997年十月革命胜利至今，中国共产党和国家机关、企事业单位、科研机构、监狱、社会团体等主要领域都紧紧围绕党的中心工作——社会主义现代化建设，坚持改革开放路线，坚持中国共产党领导，确保党始终在党中央坚强领导下领导一切。以习近平同志为核心的党中央带领全党全国各族人民，推进社会主义现代化事业，奋力谱写中华民族伟大复兴的历史篇章。

## 2.3 社会系统设计的定义及内涵
社会系统设计是一种跨越复杂的领域、面向多个层面、涉及多个领域的专门技术工作，旨在解决目前存在的各种社会问题，提升社会效益，促进社会发展。社会系统设计作为一种专业技术，涵盖以下几个方面：
1.	需求和挑战分析（Requirement Analysis）：通过对社会系统的需求和挑战进行分析，识别社会系统存在的问题、机遇和要求，并制定相应的应对策略，确定社会系统目标和方向。
2.	系统结构设计（System Design）：从宏观角度考虑社会系统的功能、业务流程、实体关系、信息流动以及资源分配，进行详细的系统结构设计，完成需求分析后的实施方案。
3.	系统实现工程（Implementation Project）：针对系统结构设计的实施方案，制定系统实现工程计划，按计划在指定的时间段内逐步实现系统的开发、测试、部署和运营。
4.	维护更新（Maintenance and Update）：对系统的运行效果不断评估、分析和反馈，根据反馈调整系统结构和运作条件，改善系统的效益。

## 2.4 社会系统设计角色、技能和方法论
社会系统设计是一项跨越复杂的专业技术工作，需要由以下角色、技能和方法论参与：
1.	需求分析师（RA）：需求分析师负责对社会系统的需求和挑战进行分析，找出系统存在的问题、机遇和要求，并提供解决该问题的可行方案，帮助系统设计师确定社会系统目标和方向。需求分析师需要具备较高的逻辑思维、分析能力、问题解决能力，同时具有很强的责任感和团队精神。
2.	系统设计师（SDS）：系统设计师负责对社会系统进行系统结构设计，进行需求分析过程中，识别系统的功能、业务流程、实体关系、信息流动以及资源分配，并制定相应的设计方案。系统设计师需要具备较好的设计能力、系统分析能力、信息理论基础和计算机相关专业知识。
3.	系统实现工程师（SIE）：系统实现工程师负责执行系统实现工程项目，按照系统实现工程项目的计划、步骤和标准，进行系统的开发、测试、部署和运营。系统实现工程师需要有项目管理经验，有系统开发、测试、部署和运营的相关专业能力。
4.	知识专家（KE）：知识专家负责对相关专业知识和理论有一定的掌握。如信息系统、数据仓库、数据库、网络安全、工控自动化、生物医药等。知识专家需要熟悉各种社会系统设计领域的知识和理论，并且对社会系统设计的领域内外发展趋势有较深刻的理解。

以上四种角色、技能、方法论一起组成了社会系统设计的体系。 

# 3.研究概况
## 3.1 国际研究状况
### 3.1.1 美国
美国的社会系统设计专业也是一大热门学科。美国国立卫生研究院社会系统工程系是美国最知名的社会系统设计专业，其成就主要归功于其国际学术界和企业界的共同努力。在1997年的一场国际会议上，科学家们联合发表了一篇论文——《Designing Social Systems: A Systematic Approach to Understanding, Defining, Implementing, Maintaining, and Improving Social Interactions in the Public Sphere》，该论文系统阐述了社会系统设计的理论基础和方法论。该学科在美国已有30余年的历史，科学家们从不同的角度探讨社会系统设计，探索建立社会系统的有效机制和机制设计，有助于公众了解公共生活和管理，促进社会和谐稳定。

### 3.1.2 英国
英国的社会系统设计专业是在20世纪80年代由三家大学、两家研究机构、两个学校、两个专业委员会、五个科研基金以及14个国家社团联合组成。2013年，英国社会系统设计学会成立，自成立以来，已吸引众多社会系统设计领域的顶尖学者投身，致力于培养、传播社会系统设计理论、方法、工具和能力。

英国的社会系统设计分为五大领域：
1.	社会组织设计（SOD），关注社会组织的理念、结构、功能以及其与公众之间的关系。
2.	社会信息设计（SID），关注社会信息结构、生成、流动以及权威性的决定过程。
3.	社会工程设计（SED），关注社会环境、政治制度、经济发展以及社会安排与管理。
4.	社会技术设计（STD），关注新型技术对社会系统的影响。
5.	社会情境设计（SDD），关注社会场景的制造和运行，探索未来社会生活的可能性。

社会系统设计学科诞生于19世纪末期，逐渐形成完整的学科体系。截至2019年底，英国社会系统设计领域的科研工作人员超过8万人，总引用量超过1.2亿篇。

## 3.2 国内研究状况
### 3.2.1 湖南农业大学
湖南农业大学成立于1953年，现为湖南省唯一一所以培养农林工程技术人员为主的公立非科班子女本科大学，其社会系统设计专业属于社科类专业，主要研究社会系统理论、方法、工程与应用。其学历结构简单、专业选取范围广泛，既有博士生也有硕士生进入本专业。湖南农业大学社会系统设计学院成立于1987年，专注于社会系统的设计理论研究，并开设社会系统设计课程。2014年，湖南农业大学社会系统设计学院改隶属于湖南农业大学经济管理学院。

### 3.2.2 浙江大学
浙江大学社会系统设计学院拥有近千名学生，属于国内规模最大的社会系统设计学院之一。学校的创建日期为1979年，是海外办学的第一所社会系统设计学校，也是世界上规模最大的社会系统设计学校之一。学校以系统工程学、信息系统工程学、计算机科学与技术、管理学、法学等领域开展专业教育。2012年10月，学校进入新时代，引入国家社科基金管理局进行改革升级，引入浙江省人文科学院、杭州电子科技大学、湖南农业大学、武汉大学、中山大学、南京大学、北京大学、南昌大学、上海交通大学、北京航空航天大学、清华大学等一批优秀院校，形成了一支以工程为基础、科技为引领、国际视野、学科交叉、国际合作为驱动的集科研、教育、人才为一体的创新型团队。

### 3.2.3 陕西科技大学
陕西科技大学社会系统设计学院，成立于1995年，其成立标志着陕西科技大学进入了社会系统设计学科的发展道路。学院以经济、法律、社会科学、人文科学、计算机科学、管理学等多个学科建设为中心，在社会系统设计的各个方面进行深入研究，获得了科研成果丰硕的声誉。学院的学术氛围活跃，师资力量雄厚，在学科建设、人才培养、技术开发方面取得突出的成果。学院现有全日制教师330余人，其中研究生45人。

# 4.论文发表
## 4.1 一篇通讯：《“我们将自己定义为社会系统的设计者。”》
作者张燕军，副教授，博士生导师，浙江大学社会系统设计学院研究员，主要研究方向为“社会系统设计理论、方法和技术”。

Abstract:社会系统设计是一个跨越复杂的领域，面向多个层面，涉及多个领域的专门技术工作。如何将自己定义为社会系统的设计者，是这项工作的关键。本文回顾了当前的研究进展及其挑战，阐述我国社会系统设计领域的最新热点话题，以及社会系统设计过程中的关键角色、技能、方法论以及思维模式等，结合中国特色社会主义理论，提出社会系统设计专业道路上的规划建议。社会系统设计是一项跨越复杂的专业技术工作，需要由不同角色、技能和方法论参与。本文提出的“我们将自己定义为社会系统的设计者”思维模式，旨在激发社会系统设计专业的理论、方法、工程、工具、能力、思维模式和管理的能力，从而构建起一支行业能力强、学术水平高、创新能力强、业务水准优良的社会系统设计团队。
