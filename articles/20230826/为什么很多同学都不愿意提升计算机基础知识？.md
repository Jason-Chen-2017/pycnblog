# 1.简介
  

首先欢迎各位读者朋友们光临这篇文章。我是李超然，2017年进入华为公司CTO职位。作为一名CTO，面对着海量复杂且高价值的业务需求，我必然要担负起企业IT信息化建设及管理的重任。而对于计算机基础知识的要求，我也从业多年所积累的经验来看，初入行时并不轻松。
计算机基础知识包括算法、数据结构、操作系统、数据库、网络、编程语言等诸多方面，这些都是IT技术人员需要具备的基础知识。为什么会有那么多人不愿意提升计算机基础知识？可以从以下几个方面探讨这个问题。

1. 缺乏动力
不懂得计算机基础知识的人往往缺乏动力，尤其是刚入行的时候。他们可能认为只有通过参加培训班才能获得必要的计算机基础知识。但事实上，计算机基础知识的掌握不是一个一蹴而就的过程，它是一种持续学习的方式。在一个需求快速变化、竞争激烈的社会里，保持高速的发展速度，对计算机科学的要求就显得十分苛刻了。因此，人们更倾向于了解一些计算机相关的高端技能，如云计算、分布式计算、大数据分析等，而非计算机本身的基础。所以，知识面拓宽，能力要求相对较高。

2. 自制力差
基础知识方面的自制力差是造成计算机基础知识普及困难的一个原因。因为知识点众多，掌握起来就需要很强的抽象思维能力。但缺乏这种能力的人，往往无法将所学的内容落地到实际工作中。这时候，就需要有较强的解决问题能力和创新能力。如果缺少这两个能力，只能说还需要努力。

3. 技术债务
有些计算机高手对计算机科学本身还有浓厚的兴趣爱好。他们对技术有着长久的渴望，但却没有足够的时间去掌握所有细枝末节的知识。这导致其技术债务一直存在，最终导致精益求精无从谈起。一般来说，现阶段技术债务主要体现在两个方面：基础知识的不扎实、理论模型与实际应用之间的差距过大。只有扭转这些债务，才有可能真正实现计算机的智能化、自动化、高效化。

4. 对产业发展的阻碍
计算机科学作为IT产业的一支基石，对于企业的创新能力与市场份额都具有重要作用。但由于缺乏计算机基础知识，使得许多小型企业和初创企业在技术选型、技术架构设计上遇到困难。企业的技术路线走偏，导致技术浪费，甚至成为企业破产的导火索。因此，为了避免这一情况的发生，企业都必须把计算机基础知识作为自己的核心竞争优势，通过自己的努力来提升自己在信息技术领域的影响力。

总结以上四点原因，造成计算机基础知识普及困难的主要原因如下：

1. 缺乏动力
没有足够的动力，根本就不会花时间去学习。人们的想法往往就是“挣钱”，而不是“提升自己”。所以，只有在工作中才会产生动力，而学习计算机基础知识则是在不断消耗自己的时间。这就要求人们必须投入更多的精力，才能有所收获。

2. 自制力差
知识面拓宽，导致自制力差。学校教育往往强调一些知识的重要性，但忽略了一些知识的学习曲线和相关的知识的关联性。学生可能只知道如何用英语解释一些词汇，却不知道应该如何把学到的东西付诸实践。这就导致学生面临学习的巨大压力，并且对未来进步产生障碍。

3. 技术债务
计算机技术的进步速度远远快于人的想象，所以许多高手对它的热情是如此之高，以致于技术债务越堆越大。有的高手放弃一些技术，纠缠在某些技术理论的门槛上，导致了技术债务一直存在。

4. 对产业发展的阻碍
产业变革与创新不可逆，当技术知识不够扎实、理论模型与实际应用之间的差距过大，企业技术方向就会被束缚住。这就给企业带来了巨大的挫败感，并进一步推动着IT产业的发展方向。因此，计算机基础知识不仅是企业成功的必要条件，也是个人发展的捷径。
