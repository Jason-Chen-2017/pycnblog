# 2.概念术语说明：

机器学习（Machine Learning）是一门关于计算机如何利用数据改善自身性能的科学研究。它涉及算法构建、模型训练、应用与管理等过程，目的是使计算机具有学习能力。一般来说，机器学习可分为监督学习、无监督学习和强化学习三种类型。下面分别介绍它们的一些关键概念和术语。

2.1 监督学习（Supervised Learning）

在监督学习中，输入样本（input data）被标记（labeled）或分类（classified），输出结果（output result）也由人工给定，训练集（training set）由一组输入-输出对组成。通过学习从训练集中提取特征（feature），得到一个预测模型（prediction model），再用该模型对新的输入数据进行预测。监督学习包括分类（Classification）和回归（Regression）。

监督学习的典型任务如下：

1. 分类（Classification）：监督学习用于预测某个输入数据的类别标签。如手写数字识别（MNIST）、图像识别（CIFAR-10、ImageNet）、垃圾邮件过滤（Spam/Ham）等。根据给定的输入样本，预测其所属的类别。比如，给出一张图片，识别出其中的物体是猫还是狗。

2. 回归（Regression）：监督学习用于预测某个变量（output variable）的值。如气象、金融市场等预测问题。假设我们希望根据历史数据来预测今后的数据值。比如，根据历史股价信息预测今天的股价。

2.1.1 数据集（Dataset）

在监督学习中，训练集通常由一系列的输入-输出样本组成。每个样本包含一个特征向量（feature vector）和一个目标值（target value）。可以认为，训练集是机器学习的“脸”或者“脑”，它反映了问题的实际情况。比如，对于手写数字识别任务，训练集就是一系列的训练样本，每个样本都是由一个像素矩阵表示的手写数字图像，以及相应的数字标签。

2.1.2 特征工程（Feature Engineering）

特征工程是一个重要且耗时的环节。主要作用是从原始数据中提取有效的特征，帮助训练集更好地表征数据特征。它涉及很多机器学习技巧，如数据清洗、处理、转换等。特征工程的目的之一是降低维度，从而减少计算复杂度和模型大小。另外，也可以加速模型训练，因为更多的信息可以提供给模型以更快地进行优化。

2.1.3 模型训练（Model Training）

模型训练是指利用训练集中的输入-输出样本来估计模型参数，使得模型能够对新输入数据做出准确的预测。常用的模型有线性回归（Linear Regression）、支持向量机（SVM）、决策树（Decision Tree）、神经网络（Neural Network）等。当模型训练完成后，就可以对新输入数据进行预测。

2.1.4 模型评估（Model Evaluation）

模型评估是指衡量模型效果的方法。模型评估的标准包括准确率（Accuracy）、召回率（Recall）、F1 score等。准确率描述了正确预测的数量占所有预测的比例；召回率描述了真正的正例（Ground Truth）被预测正确的比例；F1 score综合了准确率和召回率，是一个常用的评价指标。最后，还可以通过绘制ROC曲线、AUC等指标，评价模型的优劣。

2.2 无监督学习（Unsupervised Learning）

无监督学习是指机器学习方法不依赖于已知的正确答案，而是通过自发发现数据结构和规律的方式寻找隐藏模式和内在联系。无监督学习通常被应用于聚类、关联规则挖掘、数据降维、异常检测等任务。它的特点是没有明确的输出目标，不需要对数据进行标记。

无监督学习的典型任务如下：

1. 聚类（Clustering）：无监督学习的聚类任务是将相似的数据点（data point）划分到同一个簇（cluster）中。常见的算法有K-Means、DBSCAN、层次聚类等。例如，要分析用户访问网站的行为数据，可以使用聚类算法将访问相同网页的用户划分到一个群组。

2. 关联规则挖掘（Association Rule Mining）：无监督学习的关联规则挖掘任务是从大量数据中发现频繁出现的项（item）之间可能存在的关联关系。通常采用Apriori、FP-growth、Eclat等算法。例如，在零售商店商品销售数据中，发现消费者往往同时购买具有相似属性的商品。

2. 降维（Dimensionality Reduction）：无监督学习的降维任务是在保持数据信息量的前提下，将数据维度从高维空间压缩到低维空间。常见的算法有PCA、SVD、T-SNE等。例如，在一堆文本数据中，可以通过降维算法找到关键词之间的关系，进一步分析数据。

2.3 强化学习（Reinforcement Learning）

强化学习（Reinforcement learning，RL）是指智能体（Agent）在与环境（Environment）互动过程中，通过长期试错逐渐提升自身策略（Policy）的方法，解决学习效用最大化的问题。它的特点是需要模拟器、奖励系统以及即时反馈机制。RL最早起源于机器人领域，近年来在各个领域都有很好的应用。

RL的典型任务如下：

1. 机器人控制：RL可以用于机器人的控制系统。如通过RL训练，智能体可以学习如何与环境交互，从而达到最大化控制效益。例如，自动驾驶汽车的控制系统。

2. 智能问答：基于RL的智能问答系统可以提供快速、准确、及时的答复。例如，通过强化学习训练的聊天机器人可以直接面对用户的询问，并快速、准确地回答。

2. 物流调度：RL可以用于物流调度。如在供应链上使用RL，智能体可以学习如何对货物进行调度，从而达到最小化运输成本。
