# 1.简介
  

随机矩阵(Random Matrix)是指对一个矩阵进行加权、切分等处理之后得到另一个新的矩阵，该矩阵是一个矩阵变量或概率分布，该变量在元素级上服从某个随机分布，因此可以用统计方法对其进行研究。随机矩阵理论是对矩阵代数中基础的线性代数概念、优化理论、集合理论及其它学科的基础理论的综合运用。通过学习随机矩阵理论，能够帮助我们理解非确定性系统、系统的不确定性以及复杂系统行为的本质。
本文将给读者介绍一下什么是随机矩阵，它有哪些组成部分，以及如何利用随机矩阵进行数据分析。
# 2.基本概念及术语
## 2.1 随机矩阵的定义
对于一个$n\times m$的矩阵$A=(a_{ij})$, 如果它的每个元素都符合均匀分布，即$\forall i=1,\cdots, n; \forall j=1,\cdots,m,\ P\{ a_{ij}\}=1/nm$,则称$A$为均匀分布的随机矩阵。一般情况下，随机矩阵满足以下四个性质：

1.$det(A)=c$,$A^{-1}$存在且唯一，即$AA^{-1}=I_m$ ($I_m$是$m\times m$单位矩阵)。

证明：定理三中已经证明了随机矩阵的方阵特征值全为1，因此可得$det(A)=|λ|=1$。设$X=\frac{1}{n} \sum_{i=1}^n A_i$,其中$A_i$是第$i$列，则$A=X^TX$。由于$X$符合独立同分布条件，故可推出$XX^T=\frac{1}{n} \sum_{i=1}^n (X_i X_i^T)$,且$X_i$为正交向量，因此$XX^T$也是对角矩阵，其对角线上的元素均为$\frac{1}{\sqrt{n}}$。取$A^{-1}=\frac{1}{\sqrt{n}} \sum_{i=1}^n B_i$,其中$B_i$是$A^{-1}$的第$i$列。则$AB=BA=I_m$,且$(AB)^k=A^kB^k$.因此，对任意$k>0$,有$A^{k-1}(AB)A^{k-2}...A^k=I_m$。由此可知$AA^{-1}=I_m$,且$A^{-1}=(A^TA)^{-1}A^T$。

当$det(A)<0$时，随机矩阵也可能存在。如果$det(A)>0$，则称$A$为有限维随机矩阵（finite dimensional random matrix）。否则，称$A$为无界随机矩阵（infinite dimensional random matrix）。

## 2.2 随机矩阵的一些重要术语
### 2.2.1 概率分布
随机矩阵的元素都是按照某种分布出现的，这些分布称之为随机矩阵的概率分布。例如，设$A$是$n\times m$的随机矩阵，$X_i$表示$A$的第$i$列，那么我们可以通过如下公式计算$P\{X_j=x\}$:
$$P\{X_j=x\} = \left|\int_{-\infty}^{+\infty} e^{i(\vec x-\vec c)\cdot \vec r} d\mu(\vec r)\right|^2$$
其中$\mu(\vec r)$是局部$n$-维空间$\mathbb R^n$上的概率密度函数，$\vec c$是$n\times 1$列向量，表示离散型随机变量$X_j$的均值，$\vec r$是局部坐标系下的任一点。

### 2.2.2 协方差矩阵
如果随机矩阵的概率分布为高斯分布，则对应于一个变量的协方差矩阵的特征值仍然是高斯分布的。但是，对于非高斯分布的随机矩阵，其对应的协方差矩阵不一定是高斯分布的。而当两个随机变量之间的依赖关系较强时，它们的协方差矩阵也可能发生变化。

假设两个随机变量$X$和$Y$具有联合概率分布为：

$P\{(X,Y)\}=\rho_{XY}(\theta)$

则随机变量$X$和$Y$的协方�矩阵为：

$\begin{bmatrix}\sigma^2_{X+Y}&\rho_{XY}\\\rho_{YX}&\sigma^2_{Y+X}\end{bmatrix}$

这里的$\sigma^2_{\{XY\}}$为随机变量$X$和$Y$的自相关函数，即：

$\sigma^2_{XY}=\int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty}e^{\frac{-1}{2}[\delta_X^2-\delta_Y^2-(X-\mu_X)(Y-\mu_Y)]}\mathrm{d}X\mathrm{d}Y$

当且仅当$\rho_{XY}=0$时，随机变量$X$和$Y$是独立的；当$\rho_{XY}>0$时，随机变量$X$和$Y$是正相关的；当$\rho_{XY}<0$时，随机变量$X$和$Y$是负相关的。

## 2.3 随机矩阵的分析
### 2.3.1 随机矩阵的模态
对于随机矩阵，模态就是指在一组固定的参数下，该矩阵随时间变化的规律性。如果将时间作为自变量，即$(t, X, Y)$，那么随机矩阵的模态就可以用$X(t), Y(t)$表示，即在不同时间点上，随机矩阵变量$X$和$Y$的模式或者行为。

为了刻画随机矩阵的模态，需要计算其对偶矩阵的特征谱。对偶矩阵是由随机矩阵所有特征向量组成的矩阵，这些特征向量构成了随机矩阵的信号模式。由于随机矩阵各个特征向量的模长等于1，因此随机矩阵的对偶矩阵的特征值也要满足$\lambda_i=\pm 1/\sqrt{n}$, $\forall i=1,\cdots,n$. 因此，只需考虑对角化的对偶矩阵的特征值即可，而不必考虑全体特征值。

当随机矩阵的概率分布为高斯分布时，其对应的对偶矩阵的特征值也属于高斯分布。而且，对偶矩阵的特征值为特征值的平方根，所以对应于各个特征向量的模长也等于1. 当随机矩阵的概率分布不是高斯分布时，对偶矩阵的特征值也会发生变化，但是对应于各个特征向量的模长还可以近似为1.

### 2.3.2 逆滤波
逆滤波是指通过估计后验概率分布，构造出一个逐渐消除过程噪声的模型。

考虑线性系统的观测序列$y_k=(y_{k1}, y_{k2}, \cdots, y_{kn}), k=1,2,\cdots,n$，其中$y_{ki}$是第$k$个观测值，$y_{kj}=Ax_{k-1}-Bu_{k-1}+w_k$, $w_k$是高斯白噪声。这样，就把观测序列变换成了一个新序列$x_k$, 其中$x_{k}=\mathcal F^{-1}_{n\rightarrow m}(y_k)$. 在这里，$\mathcal F$是关于$\mathbb R^n$到$\mathbb R^m$的线性映射，$\mathcal F^{-1}$则是关于$\mathbb R^m$到$\mathbb R^n$的线性映射。$\mathcal F$和$\mathcal F^{-1}$由线性变换给出，通常是通过求逆获得。

假设后验概率分布是高斯分布，即$p(x_k|y_1,y_2,\cdots,y_n)\sim N(A^\star x_{k-1},Q)$. 我们希望构造一个逐渐消除过程噪声的模型，使得我们可以更方便地处理实验数据。具体地，希望构造出一个递归方程：

$x_{k}|y_1,y_2,\cdots,y_n\sim N(\mathcal G_0 x_{k-1}, R)$,

$x_{k}|\mathcal G_1 x_{k-1}\sim N(\mathcal G_1 x_{k-1}, R)$,

$\cdots$,

$x_{k}|x_1,\cdots,x_{k-1}\sim N(H x_{k-1}, S)$,

$\mathcal G_0, \mathcal G_1, H$为先验信息，$R,S$为观测噪声的方差。

这里，$\mathcal G_0, \mathcal G_1$和$H$都是高斯过程模型，可以用来描述系统在不同状态下的行为。假设当前状态为$x_k$，那么在第$k$步预测下的状态的分布可以由$p(x_k|\mathcal G_k x_{k-1})\sim N(\mathcal G_k x_{k-1}, Q_k)$来描述。也就是说，当前状态$x_k$的预测值，由之前预测过的状态$x_{k-1}$及先验信息$\mathcal G_k$共同决定。

由此构造出的递归方程可以用来估计后验概率分布，并对实验数据进行逐步滤波。

## 2.4 随机矩阵分析中的其他常用工具
### 2.4.1 小波分析
小波分析是一种在信号处理中用来分析周期性现象的方法。在随机矩阵分析中，我们也可以将小波分析应用到随机矩阵上。对于一个随机矩阵$A$，假设其周期为$2\pi$, 则根据局部小波变换的定义，可以将其写成小波基底的乘积：

$A(\omega) = \sum_{j=1}^J a_j \phi_{j}(\omega)$

这里，$J$为小波基底数量，$a_j$是对应小波基底的参数。$\{\phi_{j}(\omega)\}$是$2\pi$周期的小波基函数，且它们的总体积为1。

通过小波分析，我们可以了解到随机矩阵的小波形式。例如，若$A$是偶函数矩阵，则小波基底为：

$\phi_j(\omega) = e^{-jk\omega}$ for all $j$

则$\{e^{-jk\omega}\}$ 是第一小波基，而$\{e^{-(j+\frac{1}{2})\omega}\}$ 和$\{e^{-(j-\frac{1}{2})\omega}\}$ 则是第二小波基。由于小波基的周期为$2\pi$, 因此$J=N/2+1$, 其中$N$ 为矩阵的大小。

### 2.4.2 函数逼近与参数估计
随机矩阵的元素是由某个函数产生的，因此随机矩阵元素间的联系也比较直观。常用的函数逼近方法包括最小二乘法、核插值法等。通过对一个随机矩阵进行参数估计，我们可以得到随机矩阵的各种性质，如模态、相关性、自相关性、混沌性、特征分解、稳定性等。参数估计的过程可以采用EM算法来实现。
