# 2. Google AI平台概览
## 2.1 Google AI Platform
&emsp;&emsp;Google AI Platform是Google AI相关产品的集合，目前包括四个产品和服务：
- **Google Cloud AutoML**：帮助企业自动训练机器学习模型；
- **Google Cloud Dialogflow**：可用于构建完整的对话系统；
- **Google Cloud Natural Language**：能够处理文本、音频和视频数据；
- **Google Cloud Translation**：提供免费的在线文本翻译服务。
&emsp;&emsp;除了以上四个产品和服务之外，还有一些其他相关产品和服务，如Google Cloud Vision、Google Cloud Speech、Google Home等。其中，Google Cloud AutoML提供了企业级的自动化训练平台，用于优化模型的效率和准确性；Google Cloud Dialogflow是一个完整的对话系统平台，支持多种对话形式（包括文本、语音、视频），并提供自定义技能和实体的训练与部署；Google Cloud Natural Language则提供处理文本、音频、视频等数据的工具。而Google Cloud Translation则提供了免费的在线文本翻译服务。
