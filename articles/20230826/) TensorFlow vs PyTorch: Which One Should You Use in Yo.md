# 1.简介
  

在现代人工智能技术发展中，深度学习框架无处不在，包括 TensorFlow、PyTorch等。它们都是开源平台，可以实现高效计算和快速开发，各有千秋。但究竟哪一个更适合你的项目，那就取决于你对这两个框架的了解程度、历史沿革及应用场景。本文将对这两款框架进行比较分析，帮助读者决定如何选用适合自己的深度学习框架。
