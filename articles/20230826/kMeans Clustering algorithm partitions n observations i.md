# 1.简介
  


k-Means聚类算法是一种无监督学习算法，它可以将n个观察值划分成k个簇，并使得每个观察值都属于距离其均值的最近的一个簇。该算法的目的是为了对数据进行分类、聚集、降维等。下面简要描述一下k-Means算法的主要步骤：

1.随机选择k个均值作为初始的聚类中心；

2.计算每个样本到各个均值的欧氏距离，将每个样本分配到距离其最近的均值所在的簇；

3.根据簇内样本的平均值更新簇的位置，直至所有的样本分配结束（即不再发生变化）或者达到最大迭代次数。

通过这样的迭代过程，算法最终能够将所有样本划分到k个簇中，并且每个样本都属于距离其均值最近的簇。

k-Means算法具有良好的鲁棒性，它的运行时间复杂度是O(kn)，是一个比较简单的算法，可以用于很多领域，如图像处理、文本处理、生物信息分析等。

# 2.基本概念及术语

## 2.1 数据集

k-Means算法所使用的训练数据集称为样本集或样本空间。样本集由m个向量组成，每一个向量表示一个训练样本。在实际应用中，样本通常是高维数据，因此，一般来说，样本空间中每一个向量都是由n个特征值所构成。如图2-1所示为一个样本集的示例，其中每个样本的特征值为(x1, x2)坐标值。

## 2.2 聚类中心

对于给定的k个聚类中心，k-Means算法会按照如下方式进行聚类：

首先，初始化k个初始聚类中心，即样本集中的m个样本点；

然后，重复执行以下两个步骤：

1. 对于每一个样本点x，计算它与k个聚类中心之间的距离，将距离最小的那个中心作为其所属的中心；

2. 更新k个聚类中心的值，使得它们代表了当前簇中的所有样本点。

这个过程一直迭代下去，直到每一个样本点的所属中心都没有发生变化或者达到了某个用户设定的最大迭代次数。

最后，每个样本点都会被分配到距离其均值最近的那个聚类中心，从而完成了数据的聚类。

## 2.3 均值向量

对于一组样本点，其均值向量是指这些样本点在各个方向上的加权平均值，即：

$$\mu_j=\frac{1}{|S_j|}\sum_{i\in S_j}x_i$$

其中$S_j$表示第j个簇中的样本点集合，$\mu_j$表示第j个簇的均值向量，$|S_j|$表示簇j中样本点的个数。

当样本集很大时，通常无法直接计算出均值向量，而需要采用迭代的方法逐步地优化。
