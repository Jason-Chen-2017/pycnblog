# 1.简介
  

我们作为一个程序员或程序员的后代，无时不刻担心着自己的技能在短时间内会过时，从而对学习过程产生了厌恶感。然而，这种恐惧往往来自于一种错误的想法——怕麻烦。其实，任何知识都有它的价值，只不过我们没有能力去衡量它的价值的大小罢了。因此，如何克服对学习的恐惧，正确认识学习的重要性、适用范围，更好地利用学习资源，创造出满足个人需求的产品和服务才是最重要的。本文将给出一些建议，帮助读者克服对学习的恐惧。
# 2. 为什么学习很难？
首先，“学习”这个词是多义的。很多人把学习理解为受教育程度的提高，这是不正确的。当我们学习的时候，真正重要的是理解和掌握知识，而不是只是学习知识的数量和质量。学习并不是一蹴而就的。它是一个持续不断的过程，需要不断的训练、实践、反复、总结。只有把每天所学付诸行动，才能真正学到东西，从而成为真正的“知识工人”。
其次，“学习”这个过程有复杂的机制，它涉及到认知、情绪、思维、行为等方面。不同人的学习方式各有千秋，但是它们有一个共同的特点，即刻意练习、坚持、积累。学习本身就是不断重复的过程，如果没有刻意的努力，甚至连重复都无法实现。因此，“为什么学习很难”还要进一步细化为以下三个原因：
- **认知缺陷**：包括分辨事物、识别模式、抽象思维等能力弱点；
- **情绪因素**：包括对学习本身缺乏耐心、焦虑、恐慌、抗拒等负面情绪的影响；
- **环境因素**：包括课程设置、授课方式、学习氛围等外部因素的影响。
第三个原因，即**社会因素**。如今的互联网带来了海量信息，但同时也引起了人们的注意力分散，导致人们容易被各种新鲜刺激所欺骗。这就使得人们容易错过重要的信息、忘记自己已掌握的知识，甚至陷入虚假的优越感之中。通过网络学习固然很方便，但同时也可能误导我们，误以为网络上的各种信息都是可信的。所以，了解世界上其他领域的专家的经验、研究成果，进行多角度的比较，这些都是成功学的有效手段。
# 3. 如何应对学习的恐惧？
为了克服对学习的恐惧，我们可以采取以下方法：
## 3.1 学以致用
首先，不要因为害怕而放弃自我学习的机会。学习的目的就是为了提升自己，提升自己的能力，让自己走向更好的境界。坚持做正确的事情，才能够长久坚持下去。因此，不要怕麻烦，去搜索相关的教材、学习资料，把书本上的知识和信息系统atically整合到自己的脑子里。如果无法马上理解和应用，则多加思考，多观察，多尝试，直到理解透彻。这样既可以巩固和提高自己的知识结构，又能够充实自己对世界的理解。最后，分享自己的知识，扩大自己的影响力，让更多的人受益。
## 3.2 建立技能树
第二，建设自己的技能树，形成专业的知识体系。虽然知识是枝杈之末，但知识仅是工具，真正的力量来自于培养专业知识的能力。人们容易陷入“术”、“道”、“术”、“道”，但是却忽视了“术”的基础，而培养基础知识往往是知识工作者成长的关键。因此，学习者需要建立自己的技能树，形成全面的专业知识体系。这里的“全面”意味着不局限于某个领域的专业技能，而是涵盖整个知识体系中的所有知识点。
构建技能树的方法是，找寻专业的标准、知识结构以及工作角色。比如，某些项目中要求有数据库的设计能力，那么就应该在知识结构上划分出数据库设计相关的知识。相应的职称或职责可以是数据库管理员、开发工程师、数据分析师等。
第三，制定学习计划。计划可以分为几个阶段：
- **初期阶段**：确定自己感兴趣的领域，找到相关的基础知识；
- **再修阶段**：学习最基础的专业知识；
- **上攻阶段**：进行专业知识的系统学习、深入理解、巩固学习效果。
第四，注重循序渐进。尽早的掌握基础知识是很重要的，但不能仓促上手，否则容易迷失方向。循序渐进的学习方式有助于消除新知识与旧知识之间的鸿沟。除了基础的专业知识外，还有很多次阅读、思考、试错等学习活动，不妨一试。最后，每个阶段都要反复练习，确保自己掌握的知识与实际工作相匹配。
## 3.3 梳理知识结构
第三，梳理自己的知识结构。我们在学习过程中，需要对所学到的知识点进行分类，并形成自己的知识结构。知识结构的目的是为了建立知识之间的联系，因此，它不仅对我们今天的学习十分重要，也是对未来的工作十分重要。
知识结构的层次分为三级：
- 一级主题：对知识进行大的分类，比如计算机科学、经济学、哲学等；
- 二级主题：对一级主题下的具体知识进行小分类，比如概率论、计算几何、逻辑学等；
- 三级主题：对二级主题下的具体知识点进行细分，比如随机变量、概率分布、贝叶斯概率等。
知识结构的作用主要有两个方面：
- 一是便于我们检索和理解知识，因为我们知道该查哪个主题下的哪个知识点；
- 二是指导我们实施学习计划，根据我们的知识结构，选取相关的主题、材料进行学习，从而加速自己的学习进程。
## 3.4 选择合适的学习资源
第四，选择合适的学习资源。由于学习的目的与方法不同，不同的人可能会采用不同的学习资源。例如，有的人喜欢听书，有的人喜欢看视频，有的人喜欢问问题，有的人喜欢练习。不同的人学习速度也不同。因此，应该根据自己的情况选取合适的学习资源。比如，对于职场新人来说，建议购买最新的职业规划书籍和资讯。对有一定基础的学习者，可以选择国外的、英文的学习资源。对于完全没有接触过的领域，推荐一些免费的线上课程或图书。这对我们进步很有帮助。
第五，利用社交媒体进行学习。学习本身是一件耗时的事情，需要花费大量的时间，所以可以通过社交媒体进行学习。在社交媒体上，你可以与志同道合的人讨论、互动，也能获得启发和建议。通过参与线上线下的讨论，也可以吸收到他人对自己的评论和建议。社交媒体上也能找到自己感兴趣的资源和信息，这对我们学习十分有利。最后，如果别人觉得你的知识分享很不错，也会鼓励你学习。这就像亲密关系一样，帮助我们改善自己的学习习惯。
最后，还有一些其它的方法，比如建立模拟考试系统、建立知识图谱等，这些方法也同样有效。总之，要学会“技艺高超”，才能充分发挥自己的潜力，克服学习的恐惧，有所收获！
