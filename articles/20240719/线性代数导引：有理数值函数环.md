```python
# 定义有理数域上的多项式环

phi = sympy.div(x**2 + 1, x - 1)

# 构建有理数值函数环

# 输出结果

```

- `sympy.div(x**2 + 1, x - 1)`：定义多项式映射 $\phi(x) = x + 1$。
- `R.quotient(x**2 - 1)`：构建有理数值函数环 $F_{\mathbb{Q}}$。
- `f.extend(phi, f)`：计算多项式映射的复合 $f \circ \phi$。

### 5.3 代码解读与分析
