## 2.1 Python简介
### 什么是Python?
- Python 是一种开源、跨平台、通用性强、高层次的面向对象编程语言，由Guido van Rossum编写，第一版发布于1991年。
- Python 是一种多范式编程语言，支持命令式编程（过程式编程）、函数式编程、面向对象编程等多种编程方式。它具有简洁、一致的语法风格，允许程序员用更少的代码完成更多的事情。
- Python 可移植性强，可以在各种UNIX及Windows版本上运行，并提供对多种硬件体系结构的支持。
- Python 拥有一个庞大的生态系统，其中包含多个第三方模块，能够轻松实现许多复杂功能。

### 为什么要学习Python？
- Python 是一种简单而优雅的语言，掌握它会让你获得以下便利：
    - 简单快速的学习曲线：Python 的语法很简单，易学，而且学习起来也非常快捷。只需要很短的时间就可以入门，并且几乎没有学习曲线。
    - 更丰富的库支持：Python 提供了丰富的库支持，比如数值计算、数据处理、图形展示、机器学习、Web开发等众多领域的解决方案。
    - 广泛的使用范围：Python 已被广泛应用于各个领域，比如科学计算、数据分析、机器学习、Web开发等。
    - 免费、开源：Python 是免费和开源的，你可以选择任意时期获取它的源代码或使用它的授权协议。
    
- Python 还有以下优势：
    - 互动式编程：Python 支持交互式编程，你可以用 Python 命令行或者 IPython Notebook 进行编程，可以方便地尝试新方法和想法。
    - 强大的库支持：Python 提供了大量的第三方库，可以非常容易地实现诸如图像处理、Web服务端开发、文本挖掘、数据分析等功能。
    - 高性能计算：Python 可以有效地利用计算机的多核CPU来实现高性能计算。
    - 可扩展性：Python 的语法允许用户创建自己的模块，可以快速地集成到你的应用程序中。
    - 可移植性：Python 可运行于各种操作系统上，支持多种硬件体系结构。

### Python应用领域
- Web开发：Python 广泛用于 Web 开发，包括网站后台开发、服务器端网站开发、API 开发等。
- 数据分析、可视化：Python 通常被用于数据分析、机器学习、信息处理等领域。Python 的库支持大大简化了数据处理流程，使得数据分析师、科学家、工程师们能够更加专注于分析问题的核心。此外，Python 有着良好的可视化功能，可以使用 Matplotlib 或 Seaborn 等库进行绘图。
- 自动化脚本开发：Python 常用于编写自动化脚本，比如进行邮件和定时任务的发送、批量文件处理等。
- 科学计算：Python 被广泛用于科学计算，有很多科学计算的库可用。NumPy、SciPy、Matplotlib 等库都是 Python 中最重要的科学计算库。
- 游戏开发：Python 在游戏开发领域得到广泛应用，例如 Pygame 和 PyOpenGL 模块。
- 机器学习：Python 被广泛用于机器学习领域，有多个机器学习框架可用。scikit-learn、TensorFlow、Keras 等都是 Python 中最流行的机器学习库。

## 2.2 Python环境安装配置
### Windows环境下安装Python
- 安装下载地址：[https://www.python.org/downloads/](https://www.python.org/downloads/)
- 从网站下载最新版本的Python安装包（Windows x86-64 executable installer）后，双击安装即可。
- 默认情况下，安装过程不会修改系统 PATH 变量，因此如果想要从命令提示符直接运行 Python，需要手动将 Python 安装目录添加至 PATH 变量中。
- 安装结束后，打开命令提示符，输入`python`，出现类似如下输出则表示 Python 成功安装：

    ```
        Python 3.7.7 (tags/v3.7.7:d7c567b08f, Mar 10 2020, 10:41:24) [MSC v.1900 64 bit (AMD64)] on win32
        Type "help", "copyright", "credits" or "license" for more information.
        >>> 
    ```

### Linux环境下安装Python
- Linux 发行版默认可能都自带 Python，所以建议在安装前先检查一下。
- 如果没有，则可以通过包管理器安装，比如 Ubuntu 上：
    
    ```bash
    sudo apt install python3
    ```

### MacOS环境下安装Python
- MacOS 默认自带 Python ，可以直接运行 `python3`。
- 如果还没有，则可以通过 Homebrew 安装：
    
    ```bash
    brew install python@3
    ```
