# 2.基本概念术语说明
## 差分进化算法
差分进化算法（Differential Evolution Algorithm，简称DEA），是一种模拟自然界中多样性进化过程的算法。它通过进化的方式不断调整问题参数，找出一个最优解。一般来说，在数学上，采用非线性多项式来定义种群的适应度值。种群中每个个体都是一个向量，表示该种群中的个体的参数。不同个体之间的差异性由均匀交叉方法来产生。一旦个体之间形成了差别性，就可以利用自然选择的方法来进化种群，找到一组具有较高适应度值的个体参数。最后，选取适应度值较高的一组参数作为最终的最优解。因此，DEA属于典型的基于种群（群体）的进化算法。
## 深度学习
深度学习（Deep Learning）是人工智能领域里的一门新兴学科。它涉及多层次的神经网络，并以此提升模型的复杂度、准确性和学习能力。目前，深度学习在图像识别、文字识别、语音识别、视频分析、无人驾驶等领域取得了非常好的效果。深度学习在图像分类、文本摘要、情感分析等任务上的表现还是颇为突出。其背后蕴藏着大量的研究工作，如参数初始化、正则化、激活函数、优化算法、梯度下降法、数据增强、防止过拟合、……
## 强化学习
强化学习（Reinforcement Learning，RL）是人工智能领域里的一个重要方向。它通过给予系统反馈以指导其行为的机制，来促进系统学习、提高效率和解决棘手的问题。强化学习的关键在于构建奖励信号，即系统给予其每一次行动或反馈的回报。这些奖励信号能够激励系统更好地探索环境，以期望获得更多的奖励。系统在完成任务或达到某种性能水平之前，一直循着所设定的奖励信号不断更新，直至学习到如何有效地实现其目标。强化学习通过模型预测与实际情况的差距，来决定系统采取哪些行动，从而影响系统整体的行为。
## 遗传算法
遗传算法（Genetic Algorithms，GA）是基于种群（群体）的进化算法。它通过一系列的变异和交叉操作，生成新一代的个体。在每一代种群中，只有少量的个体被保留，其余的个体被淘汰或放弃。下一代的种群是前一代种群的子集，从而保证了种群的多样性。适应度值也与所生成的种群相对应。遗传算法可用于各种问题求解，如函数优化、图染色、机器人导航、大规模组合优化、分类问题、预测等。
## Q-learning
Q-learning（Quantile Regression Method，QRM）是一种基于强化学习的机器学习方法。它试图通过与环境进行互动，学习到环境的状态和动作以及各个状态动作的价值。它假定智能体可以得到一个描述状态价值的估计值，并根据这个估计值来选择最佳的动作。Q-learning的特点在于可以按照概率来选择动作，而不是像普通的强化学习那样选择最大化价值的动作。
## 风险管理
风险管理（Risk Management）是管理过程中对可能发生的风险进行评估、分析、评估、监督和控制的过程。风险管理的目的在于帮助公司、政府和银行制定出具备预见性、可控性和经济效益的决策。风险管理包括风险评估、风险分析、风险控制、风险评估工具、风险保障计划等方面。其中风险评估工具包括损失函数、贝叶斯风险度量、条件风险度量、VaR、CVaR等。
## 博弈论
博弈论（Game Theory）是一门研究两个或多个玩家如何合作解决一个完全信息的有限和定性的游戏的数学理论。博弈论主要关注这样两个问题：（1）在游戏中，当各个玩家都知道对手的策略时，他们会选择什么策略；（2）当对手知道自己是谁，他会选择怎样的策略？博弈论对资本市场、竞争性和博弈论等问题进行了深入的研究。
## 一些符号说明
- $d$ : 样本空间维度
- $\epsilon$ : 解空间的上界，$\epsilon=\max_{x\in X}f(x)$
- $    heta$ : 函数参数
- $\pi$ : 策略
- $r(    au):=\sum_t r_t$ : 策略$    au$的奖励
- $v_{\pi}(x)$ : 在策略$\pi$下，状态$x$的状态值
- $q_{\pi}(x, a)$ : 在策略$\pi$下，状态$x$下的动作$a$的动作值
- $\alpha$ : 学习速率，$\alpha\in [0,1]$
- $k$ : 迭代次数
- $\bar{\mu}$ : 对抗攻击者的平均奖励
