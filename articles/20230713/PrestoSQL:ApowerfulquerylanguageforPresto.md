Presto是一个开源分布式SQL查询引擎。它是一个快速、可靠且高度可用的数据仓库。其设计目标是为在线分析处理提供快速响应时间，并允许复杂查询的执行而不必担心容量和资源开销。Presto支持跨数据源联合查询，支持多种数据类型，包括关系型数据库(MySQL, PostgreSQL, etc.)、HDFS、云存储等。它具有高性能、简洁的语法、易于使用、安全性高的特点。基于Presto构建的数据分析工具有如Pandas、Spark SQL等Python库或R语言包。随着Presto被越来越多的公司所采用，其社区也一直在持续增长。截止到2021年7月，Presto已经成为Apache Software Foundation顶级项目。

# 2.基本概念术语说明
## 2.1 SQL
Structured Query Language (SQL) 是一种用于管理关系数据库系统的标准语言。它是用于存取、删除、更新和管理关系数据库中数据的计算机语言。SQL命令通过数据库服务器发送给数据库执行，以便对数据库进行各种操作，例如查询、插入、更新、删除等。它是关系型数据库管理系统（RDBMS）使用的最广泛的语言之一。SQL是国际标准化组织（ISO）发布的SQL-92规范的一部分。

## 2.2 数据仓库
数据仓库是一种中心化的大型、集成的仓库，用来存储企业的关键信息。它作为企业数字化转型过程中的一环，具有很强的战略意义和作用。数据仓库利用数据采集、清洗、转换和维度建模的方式对企业内部和外部的信息进行集成。数据仓库可以支持企业决策、生产和营销活动的全面分析。数据仓库能够支持商业智能、运营和经营效率的提升。

## 2.3 分布式计算框架
分布式计算框架是指在网络上以集群形式运行的多个计算节点，彼此之间通过消息传递通信。目前，最流行的分布式计算框架有 Hadoop 和 Spark 。Hadoop 提供了海量数据处理能力；Spark 则提供了大规模数据处理能力。

## 2.4 查询优化器
查询优化器是指根据用户输入的SQL语句或者其他类型的查询请求，将其转换为一个或多个计算任务，并确定每个任务应该由哪个节点执行。优化器的主要工作是确保整个查询计划（Query Plan）是高效的，并且能充分利用资源，减少延迟和最大限度地提升整体性能。

## 2.5 数据湖
数据湖（Data Lake）通常指的是一个集中式的存储平台，可以存储各种类型的数据，并且具备较强的分析处理能力。它的主要优点是存储数据高效、快速、便于访问、具有极高的价值发现能力。数据湖与传统的数据仓库的不同之处在于它更侧重数据收集、存储和整理，旨在通过数据湖进行快速数据探索和分析。数据湖的另外一个重要特征是可以对非结构化数据进行存储，如文本、图片、视频等。

## 2.6 Presto
Presto是一个开源的分布式SQL查询引擎，它可以运行于 Hadoop 或 Apache Spark 上。Presto的名称来自“prestige”（优雅、威严），来自于开源界最具代表性的项目之一Apache Calcite，Calcite是一种强大的关系代数表达式求值引擎。Presto支持诸如Hive，Impala，Drill等不同的关系型数据库。Presto的另一个优点是在较低成本下，它可以在同样的硬件环境下提供快速查询响应时间。
