# 2.基本概念术语说明
## 2.1 React Native简介
React Native是一个开源跨平台开发框架，用于开发在iOS、Android和Web之间具有一致体验的应用程序。它允许你使用JavaScript和React构建native UI组件，同时使用JavaScript调用平台原生API。Facebook于2015年开源React Native后，迅速成为最流行的前端开发框架之一。它的优点包括快速响应，以及较低的学习曲线。这使得React Native成为开发人员工具箱里必不可少的部分。
## 2.2 A-Frame简介
A-Frame是一个基于HTML的实体框架，可用于制作虚拟现实和增强现实体验。它利用Web Components标准，可以轻松地创建基于Web的3D和VR元素。由Mozilla基金会开发，A-Frame支持所有主流浏览器，包括Chrome，Firefox，Safari，Opera等。
## 2.3 虚拟现实（VR）
虚拟现实（Virtual Reality，VR），一种数字技术，让用户进入一个虚拟的沉浸式世界，这种世界既不是真实的，也没有实体世界那么具体。用户在这个虚拟的世界里可以用眼睛、耳朵、手臂、控制器等与真实世界一样的交互方式。当用户进入这样的虚拟世界时，他们身处三维空间中的一个小圈子中，此时看到的都是透明的、凹陷的图像。因为没有实体世界的参与，所以这种虚拟现实又被称为“混合现实”。
## 2.4 微信开放平台
微信开放平台（WeChat Open Platform），是微信公众号运营管理平台，提供了一整套解决方案，涵盖第三方平台、开放平台、消息推送、支付、群组、数据统计、多终端SDK等功能模块。开发者可以利用开放平台提供的接口、SDK、模板、Demo等资源，快速完成微信公众号、小程序等应用的开发。
## 2.5 WebGL简介
WebGL（Web Graphics Library），是用于在浏览器上绘制、处理二维和三维图形的开源图形库。它提供了丰富的API，可以用来创造各种类型的数据图形、动画效果和交互效果。这些图形、动画效果以及交互效果的绘制、渲染均在GPU上进行，效率非常高。
## 2.6 WebRTC简介
WebRTC（Web Real-Time Communication，即WEB实时通信），是一项开放源代码的项目，它为浏览器和移动设备之间的音视频通话、直播、协作编辑提供支持。WebRTC基于Google开源的GK Media Engine（即GMeetCam），是一个适用于Web浏览器的视频通话引擎。
## 2.7 Oculus Rift、HTC Vive、Google Daydream、Oculus Quest 2、Windows Mixed Reality
VR头戴设备产品的统称。其中，Oculus Rift头戴设备是由Oculus VR开发的，通过安装一个头盔配合电脑、手机、鼠标和键盘，就能够在虚拟世界中获得沉浸式的全像率显示，实现户像化视野。HTC Vive头戴设备则是由HTC出品的，以人体工程学（PBR）材质渲染为特色，能够提供高度真实的画面质量，并且在多个方向上都可以自由移动。Google Daydream头戴设备也是由Google推出的，可以提供低延迟、顺滑、沉浸感知的全景视图。Oculus Quest 2头戴设备则是由Oculus官方推出的一款VR头戴设备，采用OLED屏幕、6DoF动捕捉和交互设计，采用PCH连接电脑，预装了SteamVR游戏平台。Windows Mixed Reality头戴设备是微软推出的一款VR头戴设备，采用蓝牙控制和绿色皮肤塑料材质，并且支持HoloLens等平台的互联互通。
