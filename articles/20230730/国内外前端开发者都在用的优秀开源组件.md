# 1.简介
         

         在这里我们不做过多的介绍，主要分享一下我认为值得关注、并且大家都在使用的优秀开源组件。由于文章篇幅原因，每种组件都只会介绍其中一个主流框架，比如React, Vue, Angular, Svelte, Lit等，后续可能还会增加其它组件，如果大家感兴趣的话也可以参与进来贡献自己的心得或组件库推荐。
         
         ## React
         
         ### Ant Design
         
         这是一款由阿里巴巴集团提出的基于Ant Design设计语言和React实现的UI组件库，提供了丰富的高质量React组件，帮助开发者快速搭建企业级中后台产品原型。
         
         Ant Design是一个设计系统，它不是一个单独的组件库，而是一个完整的体系。通过它提供的高质量React组件及其他特性，你可以轻松地进行移动端、Web端和Electron桌面应用的开发，满足你的业务需求。
         
         通过组件库可以让你更快、更简单地完成项目开发。同时，社区也积极响应各种反馈，并及时解决了bugs。
         
         https://ant.design/docs/react/introduce-cn
 

### Vant UI

Vant 是一款基于 Vue.js 的开放源码移动端组件库。Vant 中的组件兼容不同平台，使得开发者可以使用统一的 API 调用各个平台的能力。

其丰富的组件能够满足移动端开发者日益增长的需求，包括导航栏、底部导航、表单输入框、弹出层、图片懒加载、轮播图、日历、时间线、卡片组等。

https://youzan.github.io/vant/#/zh-CN/home


### Next UI

Next UI是一套基于React和TypeScript的免费开源组件库，主要用于移动端应用程序的研发。它提供了高质量的基础组件、丰富的交互效果、多种形式的展示模式、可自定义的主题、可访问性友好的交互逻辑、持久化存储等功能。

Next UI组件足够简洁易用、易于扩展和自定义，让您轻松创建具有吸引力、功能丰富且美观的用户界面。

https://www.nextui.org/docs/getting-started/installation


## Vue

### Element UI

Element 是一套为开发者、设计师和产品经理准备的基于Vue 2.x的桌面端组件库。提供了完善的基础组件以及强大的扩展组件，帮助你搭建好健壮的web应用。

目前提供了Data Table、Tooltip、Popover、Tree、Dialog、Input等基础组件。

文档地址：http://element-cn.eleme.io/


### iView UI

iView 是一套基于 Vue 的开源 UI 组件库，主要服务于 PC 和移动端产品。提供了丰富、直观、友好的组件，帮助你快速开发、个性化定制，适合所有从事 Web 前端开发的同学。

它的组件已经覆盖了绝大部分常用的前端组件，还提供了一些常用的工具方法。

文档地址：https://iview.typecho.dev/


## Angular

### NG-ZORRO

NG-ZORRO 是一套基于 Angular 9+ 的 UI 组件库，提供了一整套企业级中后台产品所需的基础组件。你无需自己重复造轮子，即可快速构建企业级应用。

提供了表单控件、表格组件、通知组件、树形组件、对话框、分页器、消息提示、上传下载、步骤条等多个UI组件。

该文档地址：https://ng.ant.design/docs/introduce/zh


## 小结

最后总结一下国内外前端开发者都在用的优秀开源组件：

1. Ant Design：一款基于Ant Design设计语言和React实现的UI组件库
2. Vant UI：一款基于Vue.js 的开放源码移动端组件库
3. Element UI：一套为开发者、设计师和产品经理准备的基于Vue 2.x的桌面端组件库
4. NG ZORRO：一套基于 Angular 9+ 的 UI 组件库
5. iView UI：一套基于 Vue 的开源 UI 组件库
6. Next UI：一套基于React和TypeScript的免费开源组件库
