# 1.简介
         
2021年7月1日起，微信支付中将只支持收银台模式付款，即在小程序、H5等场景下，收款码只能用于扫码付款。那么，微信支付为何会把这种功能关闭？此外，这样的关闭还会给企业带来什么影响呢？一起来看一下吧！
         # 2.基本概念和术语
         1）账户类型:公众号、服务号、企业号等
         2）商户申请开通收银台模式付款的条件:需要事先向微信支付平台提交申请表并通过审核,并且使用微信支付提供的付款码生成工具生成收款码。
         3）代金券:是一种优惠券，可以帮助商户实现营销。微信支付没有代金券功能。
         4）普通模式(非收银台)付款方式:包括微信支付、支付宝、银联支付等。
         # 3.为什么微信支付要关闭收银台模式付款功能？
         当时微信支付决定禁止了收银台模式付款的原因主要有以下几点：
         ①微信支付平台认为扫码付款的方式不能满足快速、便捷、安全支付的需求；
         ②微信支付平台会优先确保安全性，对于个人客户及少量商户而言，限制其在线支付行为对他们来说似乎不合适，所以目前针对个人、个体户和少量商户提供的都是收银台模式的支付方式；
         ③为了规范商户使用微信支付产品，微信支付平台暂停收银台模式付款功能；
         更详细的原因分析可以阅读微信官方的公告:https://pay.weixin.qq.com/wiki/doc/api/micropay.php?chapter=9_10&index=2#%E7%A6%81%E6%AD%A2%E6%94%AF%E4%BB%98%E8%AE%BF%E9%97%AE
         # 4.微信支付代替普通模式付款的痛点和好处
         如果企业继续采用普通模式(非收银台)付款方式的话，可以获得以下好处：
         1）简单方便，无需下载或安装任何软件，扫码直接付款即可完成支付；
         2）无需担心支付中的风险，不用担心信用卡被盗刷、不受到当地银行风险管理机制约束等，都由微信支付代管；
         3）不用关心商户信息采集、存储、处理等相关法律风险；
         4）微信支付不会根据不同支付场景、不同用户的支付习惯、消费能力等因素对付费业务做出不同的调整，因此用户体验更好；
         5）无需担心用户的隐私和数据安全问题，因为所有付款记录都在微信支付平台上进行管理，交易数据始终保持匿名。
         不过，如果企业采用收银台模式付款方式的话，则可以避免如下一些风险：
         1）客户扫码付款，极易导致商家失去顾客，或者顾客遭遇诈骗等恶意欺诈行为；
         2）商家收款失败，造成顾客经济损失和声誉降低；
         3）电脑或手机耗电过多，严重影响设备寿命；
         4）支付时所用的设备容易遭受物理攻击或网络攻击，造成个人信息泄露、财产被盗等安全风险；
         5）支付结果可能因为缺乏支付意愿、错误操作、不仔细核对付款信息等原因而出现延误或异常，造成商户损失、顾客投诉等后果。
         # 5.微信支付如何兼容其他支付渠道？
         由于微信支付已经宣布禁止使用收银台模式付款功能，但是微信支付仍然允许商户使用其他支付渠道收款，例如支付宝、银联支付等。但需要注意的是，微信支付承诺提供的支付接口仅用于微信支付本身的业务，不可用于第三方支付平台或服务。因此，如需要在微信支付上结算第三方支付渠道产生的手续费，商户需要另行收取相关费用。
         # 6.微信支付面临的挑战和机遇
         在短期内，微信支付仍然保持着增长势头，市场份额也在逐步扩张。但是，随着中国互联网支付领域的发展壮大，新技术的开发、移动支付的普及、信息化的应用以及商业环境的转变等，微信支付的未来仍然充满着挑战和机遇。
         1）电子支付标准的推进：目前，各类支付机构都会制定一套标准，用于规范各种支付渠道之间的支付接口协议。但是，如今以微信支付为代表的手机支付平台已经成为国际标准组织IMI（国际移动支付行业协会）的成员单位之一。未来，随着手机支付的快速普及，未来微信支付在电子支付标准的体系中将占有重要的角色。
         2）支付服务场景的升级：随着电商的火爆和线上零售的迅速崛起，越来越多的商家选择在线支付。如今，移动支付正在成为绝佳的选择，但目前来看，微信支付仍然处于支付场景较弱的位置。在未来，随着各类电商网站的建设，微信支付将会成为电商支付的标配，商家将无论是在线还是线下，均可在微信支付上进行支付。
         3）技术革新和供应链布局的更新：新的支付系统的设计、应用、运维等将会带来很多的挑战和机遇。举例来说，微信支付近年来的支付优化实践将为商家带来更多的效益，尤其是针对支付门槛较高的商家，提升收益率；同时，微信支付的支付接口也可以融入到电商平台、移动app等，提供更好的支付服务。
         4）用户支付习惯的改变：随着移动支付的普及，用户的支付习惯也在发生变化。比如，用户可能喜欢在饭馆吃饭时选择收银台模式付款，而在家里或公司工作时则选择用微信支付APP进行支付。未来，随着用户习惯的改变，微信支付也将迎来重大改革。在这种情况下，微信支付会根据用户的支付习惯及支付场景进行相应调整。
         # 7.微信支付未来的发展前景
         1）将全面提升微信支付的支付体验：微信支付一直秉持着简洁易用的理念，将一直坚持这一理念，将从产品策略、技术研发、运营、服务等多个角度不断推进我们的能力，让您在微信支付上享受到最好的服务。
         2）微信支付产品进一步完善：针对微信支付的APP、小程序等产品进行功能上的拓展，提升用户体验，并打磨用户支付习惯，使微信支付更具竞争力。
         3）服务能力的提升：以智慧商务为代表的微信生态圈正在蓬勃发展，商家可以根据自身的情况选择合适的微信支付解决方案，将微信支付服务的能力和规模拓展到更广阔的市场。
         4）微信支付生态的加强：在微信支付和生态圈的结合下，商家可以轻松连接微信支付和微信生态圈，达到资源共享、共赢的效果，促进用户的贴近感、连接感、消费习惯的形成。
         5）价值驱动下的商业模式创新：微信支付的价值驱动一直是我们追求的目标。未来，我们将全面采用价值驱动的商业模式创新，努力构建以用户价值为导向的收款模式，进一步探索微信支付的商业模式。
