## 什么是量化交易？
量化交易（Quantitative Trading）也称高频交易或“Algorithmic Trading”，是指以计算机技术对市场中的交易进行自动化执行，并通过计算、分析、预测等方法获得有关经济数据的一种新型的交易方式。简单来说，它是一个使用计算机技术实现的从交易者的观点，基于历史数据及其他数据源进行智能化投资决策的过程。量化交易不依赖于人类专业经验，可以节约时间和金钱成本，更重要的是，由于通过机器学习和统计分析的方法对市场数据进行实时监控，因此可以较为精确地将股价波动转化为交易信号，从而在一定程度上避免了人为因素的干扰和风险。除此之外，量化交易还具有以下优点：
- 投资组合优化：通过研究市场中多种因素的作用，量化交易平台可以自动生成多样化的投资组合，其中既包括股票投资组合，也包括期货仓位等其他投资标的；
- 预测性交易：由于量化交易算法能够根据历史数据进行分析和预测，因此可以帮助客户准确定位行情趋势和买卖点，提升交易效果；
- 绿色低碳环保：与传统的金融产品不同，量化交易可以使得参与者在不受地球生态环境影响的前提下获取更多的收益；同时，大部分交易所都无法提供足够的可持续发展保障，因此量化交易需要另辟蹊径，如通过共享充电站来筹集资金；
- 数据共享化：量化交易平台在发行期间不仅会向交易者开放原始数据，而且会将部分数据共享给第三方算法服务商，如用于制定策略的机器学习模型；这样可以降低平台自身的风险，增加数据安全性。
综上所述，量化交易已经成为一种全新的交易模式，其生命周期从启动到终止都被高度自动化，且利用先进技术和数据共享的方式，在减少人工操作和风险，同时实现绿色低碳环保的同时，又保证了最大的投资回报率。
## 为什么要学习Python？
如果你正在寻找一份可以为你提供完整的量化交易知识体系的职位，那么你一定不会错过Python。作为一个开源编程语言，Python有着庞大的第三方库资源，数十年来一直是进行量化交易的首选语言。Python的简洁语法、丰富的数据处理功能、强大的社区支持以及快速发展的发展速度让它越来越受欢迎。无论是初级程序员还是资深开发人员，只要掌握了Python，就可以轻松应付绝大多数量化交易相关工作。
## 安装Python
### Windows
Windows用户安装Python最简单的方法就是下载安装包并双击运行，安装程序会一步步引导你完成安装过程。如果你之前没有用过Python，建议在安装过程中勾选安装路径不要有空格或者中文字符，否则后续可能会遇到各种问题。
https://www.python.org/downloads/windows/
### MacOS
MacOS用户同样可以通过Homebrew命令安装最新版的Python：
```bash
brew install python
```

如果您需要安装旧版本的Python，可以使用pyenv，安装命令如下：

```bash
brew update && brew install pyenv
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc # or ~/.bashrc or whatever your shell's config file is called
echo 'eval "$(pyenv init -)"' >> ~/.zshrc # or eval "$(pyenv init -)" for bash users
exec "$SHELL"
pyenv install --list | grep "3\." # find the latest version of Python 3 and install it with:
pyenv install <version> # e.g., `pyenv install 3.7.9`
```
