                 

# 1.背景介绍



2021年5月，国务院新闻办公室、国务院综合事务管理部门联合印发《关于做好政府信息化工作的若干意见》，明确提出“建立健全政府信息化体系”的任务目标。信息化建设重点关注基础平台建设、政务信息化建设、数字政府服务建设、智能制造示范区建设等方面。
为落实“信息化建设取得重大突破、产业优化升级、政务服务精准响应、社会效益实现”五大任务目标要求，我国政府信息化建设逐步成为国家发展的重要策略领域之一。政府信息化建设已经成为一项具有重要生命周期的工程。为了支撑“数字政府服务建设”、“智能制造示范区建设”，加快构建政府信息化创新生态，推动政府治理能力现代化，国务院与地方各部门共同制定了相关指导性文件，包括《政府信息化规划（2021-2027）》、《国务院政府信息化行动计划》等。

为进一步做好政府信息化工作，推动我国政府信息化科技成果转化应用，2021年10月，国务院国资委发布《关于建立健全政府信息化创新生态机制的意见》。其中提到：“要围绕战略方向、关键核心技术、政策倾斜、能力培养等领域开展持续创新，以满足政府信息化创新需求。”“充分发挥政府信息化创新平台优势，集聚众多创新资源，构建政府信息化创新生态机制，共同推进政府信息化发展。”“健全创新评审机制，不断完善创新认证标准，让创新得以更好地应用于政府信息化。”

从以上情况可以看出，我国政府信息化建设蓬勃发展，发展模式也相对稳定，进入新的阶段。为此，我国将采取创新驱动发展战略，坚持创新主攻、服务民生、促进交流的价值观，推动政府信息化科技进步，构建更美好的政府信息化环境。

基于上述背景，我们认为，构建政府信息化创新生态机制是解决政府信息化发展的关键，也是推动我国政府信息化科技成果转化应用的有效方式。实现这个目标，需要依托创新驱动发展战略、持续创新、完善创新评审机制、支持政策倾斜、赋能政府相关部门，形成“政府信息化创新生态”。构建政府信息化创新生态主要包括以下四个层面：

第一层面是产业创新。围绕着核心技术的研发，开展先进制造、产业培训、战略合作、技术引进等创新活动。扩大国内外产业合作空间，加强政策沟通协调，激发创新力量。

第二层面是政策创新。围绕着政府治理改革和信息化建设的政策导向，探索政策法规制定、监管跟踪、经营决策与效能评估等方面的创新路径。推进政策预算管理体制改革，让政府更好地服务民生。

第三层面是人才创新。围绕着关键领域技术人才的培养，打造国内顶尖院校信息化学科、政府信息化人才培养体系，引领我国政府信息化教育走向世界前列。积极探索创新型人才培养模式，提升青年人创业创新能力。

第四层面是政策工具创新。围绕着政策工具的创新和整合，推进政策工具的整体设计、建设、应用，构建适应我国政府信息化政策和政策诉求的工具箱。建设统一规范的政策工具库，助力我国政府信息化科技创新成果转化应用。

# 2.核心概念与联系

## GPT大模型
GPT，即Generative Pre-trained Transformer。它是一种生成式预训练语言模型，能够自动生成文本，并进行不同程度的自然语言理解。GPT由OpenAI团队在2019年8月份发表的一篇论文所定义，其结构特点如下：

- GPT使用transformer结构，transformer是Google团队提出的一种序列转换模型，可以学习词语之间的相互关系和长短期依赖关系；
- GPT采用无监督训练方法，同时也利用语言模型的监督信号来完成自然语言理解任务，因此能够生成独特且高质量的文本；
- GPT的训练数据是联合训练的，既包含大量的文本数据，也包含海量的外部资源数据；
- GPT可以根据输入文本自动生成符合语法和风格的输出，而且生成的结果并不是单纯按照规则出现，而是融入了多种类型的数据，形成丰富的语言含义。

GPT能够根据输入文本生成不同维度的文本，比如文本摘要、问答回答、阅读理解、翻译、自动文本推荐等。

## AI Agent
AI Agent就是指具有自主学习能力的机器人或者虚拟助手。他们可以感知周遭环境、获取信息、存储知识、判断意图、操控机器人、分配任务等功能。

## RPA(Robotic Process Automation)
RPA，即以流程为中心的软件技术，它能够通过一系列操作来自动化完成重复性劳动，提高工作效率。流程指的是某项任务或工作的顺序、步骤及人员交接情况，在企业中通常用电子化的方式呈现出来，并且有助于减少人工参与，提升工作效率。如今，由于人类越来越依赖计算机，越来越多的人开始使用机器替代人类的很多日常工作。很多公司也希望能够提升自己的效率，以提高竞争力。所以，公司内部需要引入RPA来自动化各种业务流程，降低人工操作难度，增强企业竞争力。

## 政府部门的应用案例
近年来，随着政府网站的迅速发展，网站功能越来越复杂、涉及信息多样化，使得用户在浏览网页时更加关注具体的政策措施，而不是简单的看到政策声明。为此，一些研究机构开始寻找政府网站功能上的瓶颈，以便进行优化。

举个例子，美国食品药品安全署（FDA）就曾经采用了一个“提示”功能，当用户进入某些特定页面时，会展示一些“可能存在风险因素”的信息，提醒用户注意安全。由于该功能在用户浏览网页的过程中自动执行，所以FDA能够更好地保障用户的正常访问。

同时，在2021年5月，许多政府部门正陆续启动数字化转型，开展政务网络建设，但这些都离不开人工智能技术的支持。比如，美国国土安全部就开展了一个项目——“运输安全可视化工具”，其主要目的是利用人工智能技术，让乘客更直观地了解运输过程中的安全隐患。类似的项目还有很多，这些都给政府部门带来巨大的帮助，推动了政务网络的进一步智能化。

因此，构建政府信息化创新生态的关键是要真正关注政府信息化建设中的最佳实践，创新驱动发展，注重可靠性与可重复性，确保成功转型。只有充分发挥GPT的生成式能力，结合AI Agent的自主学习能力，才能实现有效且有针对性的政府信息化服务。
