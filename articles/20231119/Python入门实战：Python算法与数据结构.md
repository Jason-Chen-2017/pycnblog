# 2.核心概念与联系
## 数据类型
- 字符串（string）
    - 单引号和双引号表示相同的东西，这取决于你的文本编辑器设置
    - 三个双引号或三重单引号可以用于跨多行输入
    - 使用索引访问字符串中的字符，下标从0开始，-1表示最后一个字符，1表示第二个字符，依此类推
    - `len(s)` 函数返回字符串 s 的长度
    - `str[start:stop]` 返回子串 start 到 stop-1 的拷贝
    - `.join()` 方法用来连接多个字符串
    - `.split()` 方法按照空白符分割字符串
- 整数（integer）
    - 十进制、八进制、十六进制表示法均可
    - 没有大小限制，可以使用无限大的整数
    - 有两种类型的整数——长整型（long integer）和布尔型（boolean）
        * 当数字超过了 long 范围时，Python 会将其自动转换成长整型
        * True 和 False 分别表示真值和假值
    - 使用二进制、八进制、十六进制分别表示以 2、8、16 为基数的整数
- 浮点数（float）
    - 用小数点表示的浮点数，如 3.14
    - 可以使用科学计数法表示更大的数字，如 6.02e23 表示一个物质的原子数量
    - 支持“NaN” (Not a Number) 和 “infinity” 值
- 列表（list）
    - 以方括号表示，元素之间用逗号隔开
    - 通过索引访问列表中的元素，下标从0开始
    - list 的元素可以是不同的数据类型，包括其他列表
    - len() 函数返回列表的长度
    - 可对列表进行切片，`lst[:i]` 返回列表从头到 i-1 的拷贝，`lst[i:]` 返回列表从 i 到末尾的拷贝，`lst[::step]` 返回步长为 step 的切片
- 元组（tuple）
    - 类似于列表，但是不可修改
    - 通过索引访问元组中的元素
    - 元组也可以嵌套
    - len() 函数返回元组的长度
- 集合（set）
    - set 是由一组无序不重复元素组成的集合，用花括号 {} 表示
    - 可以对集合进行交集、并集、差集等运算
    - 初始化方式：`set([iterable])`，接受可迭代对象（如列表、元组等），或者直接用 `{element}` 创建一个空集合
    - 添加新元素使用 `add()` 方法，删除元素使用 `remove()` 或 `discard()` 方法
    - 判断元素是否存在于集合使用 `in` 操作符
- 字典（dict）
    - 键值对存储的容器，用花括号 {} 表示
    - 通过键来访问值
    - 对字典做更新、删除操作前必须先检查 key 是否存在
    - 初始化方式：`{key1:value1,...}` ，如果 value 不指定，则默认为 None
    - 获取字典的键值对数量 `len(d)`
    - 拷贝整个字典 `copy.deepcopy(d)`
