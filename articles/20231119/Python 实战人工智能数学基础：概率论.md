                 

# 1.背景介绍


概率论是现代数学的一个重要分支，它涵盖了几乎所有科学研究领域。它的研究范围之广、抽象程度高，从哲学、语言学到物理学都用到了概率论。概率论研究随机事件发生的各种可能性及其分布规律。机器学习、深度学习等领域的算法或理论都是建立在统计学基础上的。因此，掌握概率论知识对于成为一名合格的数据分析者、机器学习工程师至关重要。
由于其“洞见”的深度、广度和历史积淀，概率论已渗透到计算机科学、数学、经济学、心理学、法学、工程学等多门学科中。概率论主要包括随机变量、事件空间、样本空间、概率测度、独立同分布、条件概率、Bayes公式、随机向量、随机矩阵、随机过程等概念。通过这些概念，我们可以更好地理解机器学习算法、数据分析结果，并利用概率论对世界进行建模和预测。
# 2.核心概念与联系
## （1）随机变量（Random Variable）
设X是一个定义在一个随机试验空间S上的随机变量，其取值于S中的每个元素x称为X的取值。X的概率分布函数定义为Pr(X=x)，表示随机变量X在取值为x时发生的概率。随机变量还具有随机分布，即随机变量的概率分布随着时间变化。当一个随机变量满足一定条件时，它才是一个可观测的随机变量。如果没有任何限制，一般将所有可能的取值记为样本空间S。
## （2）事件（Event）
事件是概率论中一个基本概念。设A、B为两个事件，则：
- A∩B：A和B同时发生的事件；
- A∪B：A或者B发生的事件；
- ¬A：A不发生的事件，简称反事件或事件否定；
- P(A):A发生的概率，记做P(A)或 Pr[A]。
## （3）样本空间与事件空间
- 概率空间：给定一个样本空间，定义其相应的概率空间，记做S∈ℝ^n。其中，n为样本空间的维度，每一个元素对应一个随机变量。
- 事件空间：设S为样本空间，则可以把样本空间划分成互不相交的事件集，称为事件空间。每个事件集对应于样本空间的子集。每个事件由一个样本点组成。例如，一个六面骰子投掷，样本空间为{1,2,3,4,5,6}，事件空间为{{1},{2},{3},{4},{5},{6}}。
## （4）概率与概率分布
- 概率：设A为事件，Pr[A]表示事件A发生的概率。0<=Pr[A]<=1，Pr[A]越接近1，则A发生的可能性就越大。Pr[A]的单位是“概率”，取值范围为0~1。
- 概率分布：概率分布是指随机变量X在不同取值的情况下发生的概率。概率分布有三种类型：
  - 离散型分布：有限个可能的取值，如硬币抛掷的正面朝上、负面朝上、或者发光二极管。
  - 连续型分布：无限多个可能的取值，如均匀分布、指数分布、正态分布等。
  - 组合型分布：在多个随机变量的联合分布。如贝叶斯公式的应用。
## （5）随机变量的期望、方差、协方差与相关系数
- 随机变量的期望（Expected Value/Mean）：设X为随机变量，E[X]表示随机变量X的期望，用μ表示。随机变量的期望描述了随机变量的中间值。当随机变量X服从某种分布的时候，它的期望存在且唯一。
- 随机变量的方差（Variance）：设X为随机变量，Var[X]表示随机变量X的方差，σ^2表示。方差描述了随机变量偏离期望值期望的程度。当随机变量X的取值较为聚集时，方差会减小；当随机变量的取值较为分散时，方差会增大。
- 随机变量的协方差（Covariance）：设X、Y为随机变量，Cov[X,Y]表示随机变量X和Y之间的协方差。协方差衡量的是随机变量X和Y偏离它们的期望线的程度。当X和Y不相关时，协方差等于零。协方差的大小可以衡量两变量之间线性关系的强弱。
- 随机变量的相关系数（Correlation Coefficient）：设X、Y为随机变量，ρ(X,Y)表示X和Y的相关系数，用ρ表示。相关系数介于[-1,1]之间。当ρ=1时，说明X和Y正相关；当ρ=-1时，说明X和Y负相关；当ρ=0时，说明X和Y不相关。相关系数与线性回归分析密切相关。
