## 3.1 排序算法
排序算法是将一组无序的数据转化为有序的一组数据的过程，常见的排序算法包括冒泡排序、快速排序、归并排序等。

冒泡排序是通过比较相邻的两个元素，然后交换它们的位置，使得每一轮结束时，最大（或最小）的元素都会移动到最后一个位置。具体操作步骤如下：
```scss
for (i = 0; i < n-1; i++) {
  for (j = 0; j < n-i-1; j++) {
    if (arr[j] > arr[j+1]) {
      // swap arr[j], arr[j+1]
      temp = arr[j];
      arr[j] = arr[j+1];
      arr[j+1] = temp;
    }
  }
}
```
数学模型公式：冒泡排序的最坏时间复杂度为O(n^2)，平均时间复杂度也为O(n^2)。

快速排序则是通过选择一个基准值，然后将数组分成两个子数组，其中一个子数组的元素都比基准值小，另一个子数组的元素都比基准值大。具体操作步骤如下：
```perl
function quickSort(arr, left, right) {
  if (left < right) {
    let pivotIndex = partition(arr, left, right);
    quickSort(arr, left, pivotIndex - 1);
    quickSort(arr, pivotIndex + 1, right);
  }
  return arr;
}

function partition(arr, left, right) {
  let pivot = arr[right];
  swap(arr, left, right);
  let storeIndex = left;
  for (let i = left; i < right; i++) {
    if (arr[i] < pivot) {
      swap(arr, i, storeIndex);
      storeIndex++;
    }
  }
  swap(arr, storeIndex, right);
  return storeIndex;
}
```
数学模型公式：快速排序的最坏时间复杂度为O(n^2)，平均时间复杂度为O(nlogn)。

归并排序则是将原数组分成两个子数组，然后对这两个子数组分别进行快速排序，最后将排序后的结果合并成一个有序数组。具体操作步骤如下：
```css
function mergeSort(arr, left, right) {
  if (left < right) {
    let middle = Math.floor((left + right) / 2);
    mergeSort(arr, left, middle);
    mergeSort(arr, middle + 1, right);
    merge(arr, left, middle, right);
  }
  return arr;
}

function merge(arr, left, middle, right) {
  let leftCopy = cloneArray(arr, left);
  let rightCopy = cloneArray(arr, middle + 1, right);
  let i = 0, j = 0, k = left;
  while (i < length(leftCopy) && j < length(rightCopy)) {
    if (leftCopy[i] <= rightCopy[j]) {
      arr[k++] = leftCopy[i++];
    } else {
      arr[k++] = rightCopy[j++];
    }
  }
  while (i < length(leftCopy)) {
    arr[k++] = leftCopy[i++];
  }
  while (j < length(rightCopy)) {
    arr[k++] = rightCopy[j++];
  }
}
```
数学模型公式：归并排序的最坏时间复杂度为O(nlogn)，平均时间复杂度为O(nlogn)。

## 3.2 其他常用算法
除了以上几种常见的排序算法，还有一些其他的常用算法，如插入排序、基数排序、堆排序等。这里以堆排序为例，简要介绍一下其原理和操作步骤：
```perl
function heapify(arr, index, size) {
  let maxChild = index * 2 + 1;
  if (maxChild < size && arr[maxChild] > arr[index]) {
    index = maxChild;
  }
  let swap = arr[index];
  if (index != 0 && arr[index - 1] > swap) {
    index--;
  }
  arr[index] = swap;
  heapify(arr, index, size);
}

function heapsort(arr, size) {
  for (let i = size - 1; i >= 1; i--) {
    heapify(arr, i, size);
  }
  for (let i = size - 1; i > 0; i--) {
    swap(arr, 0, i);
    heapify(arr, 0, i);
  }
  return arr;
}
```
数学模型公式：堆排序的最坏时间复杂度为O(nlogn)，平均时间复杂度为O(nlogn)。
