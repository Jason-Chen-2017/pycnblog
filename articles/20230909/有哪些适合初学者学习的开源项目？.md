# 1.简介
  

开源项目对于想要学习编程的人来说，是一个极佳的学习资源。在中国特别是互联网行业，作为个人开发者或者企业内部员工，都可以从众多优秀的开源项目中获得帮助和指导。很多初学者为了接触开源，甚至会选择找一个学习成本低、结构清晰、功能齐全、代码规范完整的项目作为入门，这种方式能够快速上手学习新的技术。但是，刚开始接触开源项目的时候，往往会面临很多不理解的地方，因此我将以初学者视角出发，整理了一些适合初学者学习的开源项目。


# 2. GFW与Github
GFW（Great Firewall of China）即中国最大的防火长城，在全国各地普遍存在。这个防火墙拦截所有境外的网络流量，包括互联网上大量的开源代码、文档等。正因如此，越来越多的开源项目放弃托管在GitHub等代码托管平台，转向其他网站进行托管，并经常会遇到各种莫名其妙的问题，比如无法克隆或下载项目，项目文件缺失等等。如果有条件，建议开启梯子、翻墙。


# 3. awesome-go
awesome-go是一个开源的Go语言框架收藏集，里面涵盖了Go语言的各种开源库、工具以及第三方服务。它提供了从基础库到高级Web框架再到实用工具等多个领域的开源项目，这些项目都是由业界精英们在积极投入时间和努力下丰富而完善的。每一个开源项目都会给读者带来作者独有的见解，让他们能够更好地理解开源的精髓。而且awesome-go项目非常活跃，相信随着越来越多的开源项目陆续加入到该集中，它也会变得越来越优秀。


# 4. go-kit
go-kit是Go语言中的一套微服务框架。它的设计目标是通过提供一个强大的toolkit，帮助用户编写健壮、可扩展且易于测试的服务。go-kit包装了Go标准库，并提供了大量工具类组件，帮助工程师实现复杂而微妙的分布式系统架构。作为一名初学者，如果需要学习微服务，那么go-kit无疑是一个不错的选择。


# 5. go-micro
go-micro是一个Go语言中的微服务框架。它由微软开源并拥有大量的开发者参与贡献，目前已经成为云原生时代的事实标准。与go-kit不同，go-micro更加关注实现核心特性上的抽象化。因此，它适用于云原生环境下的微服务开发，并且提供了诸如服务发现、熔断器、负载均衡、消息总线等关键组件，帮助工程师快速构建应用。如果您想要探索微服务架构及其在开源世界中的影响，go-micro是一个不错的选择。

# 7. Gitbooks
Gitbooks是一款基于Node.js的书籍编辑器，它可以将 Markdown 文件转换为电子书，并提供在线阅读和打印功能。由于没有购买服务器或安装繁琐的插件，并且支持生成PDF、Epub等多种格式，因此Gitbooks适合用来编写技术类的文档。值得注意的是，Gitbooks背后有一个很活跃的社区，里面的经验丰富的开发者们能够分享各自的知识、经验以及教程。如果你想编写自己的开源技术书籍，可以使用Gitbooks搭建自己的平台。
