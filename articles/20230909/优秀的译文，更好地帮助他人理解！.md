# 1.简介
  

作为IT从业者，我们经常需要阅读大量的英文技术文章。如何把复杂的英文知识快速、准确地传达给别人？笔者认为，优秀的译文是必要的。英文是一种非常灵活而广泛的语言，中文是互联网世界中最通用的语言，因此我们在阅读、学习和讨论技术文章时应该多采用中文。但由于众所周知的翻译水平限制，造成许多英文技术文章难以直观地呈现原意。为了帮助读者更好地理解原文，提升他们的阅读能力，编者已经积累了一系列的译文资源，帮助英文阅读者快速、轻松地掌握各项知识点。这些译文既可作为随时参考书籍使用，也可通过微信公众号等渠道供广大英文读者获取，并提升英文技术文章的传播能力。下面，让我们一起来看看一些优秀的译文吧。
# 2.博客文章
## 机器学习：A Probabilistic Perspective（概率视角下的机器学习）
这是一篇2019年出的论文，作者来自Google Brain团队。全文翻译精良，逻辑清晰，有助于读者理解概率视角下机器学习的主要概念和方法。本文涉及的内容涵盖了统计学习、优化理论、贝叶斯学习等领域，值得一读。

## TensorFlow Lite for Microcontrollers: A Comparison with other Frameworks and Approaches （微控制器TensorFlow Lite：与其他框架和方法的比较）
本文介绍了微控制器TensorFlow Lite相关的基础设施、框架、方法和工具。从概念上阐述了TensorFlow Lite工作原理，然后对比介绍了ARM CMSIS、C++ API、Python API、MicroPython API和Arduino IDE等其他框架和方法。文章从用户需求出发，提供了针对不同开发人员群体的建议，希望能够帮助读者决定在项目中采用哪种方案。

## Google ARCore: Building Virtual Reality Applications on Mobile Devices（谷歌ARCore：在移动设备上构建虚拟现实应用）
本文详细介绍了谷歌ARCore，它是谷歌自2017年推出以来首个支持增强现实(AR)的系统，并提供免费的SDK。该系统可用于创建沉浸式的、真实的虚拟现实应用程序。作者首先介绍了AR的相关概念，包括特征跟踪、图像处理、机器学习模型和渲染技术。接着，通过分享两款开源应用案例——Walmart Go（2D虚拟现实购物车）和Face Mash（虚拟现实拼图游戏）进行了细致的介绍。最后，还向大家展示了ARCore的适用性及其未来的发展方向。

## How to Write Better Python Code (更好地编写Python代码)
本文提供了一些编程习惯，可以改善现有的Python代码，提高其效率和可维护性。其中重点提到了Python命名规范、测试驱动开发、结构化设计、文档字符串、异常处理、装饰器等内容。希望能够对正在编写Python代码的读者有所启发。

# 3.科普文章
## The Most Popular Programming Language in the World? （“世界上最流行”的编程语言是什么？）
本文列举了各种编程语言在过去5年间的历史数据，并对编程语言流行趋势进行了分析。结合不同的数据来源和统计方法，作者判断了“世界上最流行”的编程语言究竟是哪一个。随后，作者对选取的语言以及原因进行了深入分析，解释了为什么这个语言在流行。这篇文章很好的通过科普的方式呈现了编程语言发展变化的规律。

## Top Data Science Tools and Techniques （数据科学工具和技术排行榜）
本文总结了2019年的数据科学工具和技术。文中包括了R、Python、Tableau、Excel、Power BI、SPSS、SAS等多个领域的工具和技术。虽然每个领域都有自己的独特优势，但大家共同努力提升数据科学技能，这也是数据科学的一个重要目标。这篇文章通过技术手段和工具的分类对当前热门的技术进行了解释，并且会给出对新手的建议。

## What's Happening With Algorithms? （算法在发生些什么？）
本文通过对近期热门的算法和数据结构的调研，论述了算法和数据结构的主要发展趋势、演变过程和应用场景。值得注意的是，作者对技术的定义非常宽泛，并不限定于计算机领域。因此，文章也能帮助感兴趣的读者了解算法的动态演进。

# 4.教程文章
## JavaScript Crash Course for Beginners （JavaScript初级教程）
这篇文章旨在帮助刚入门的前端工程师学习JavaScript。作者从基础语法开始，逐步深入到浏览器中的原理。涉及了HTML、CSS、JavaScript、jQuery、React、Vue等多方面的内容，还有一些练习题目，对想要学习JavaScript的读者来说是一个不错的入门教程。

## Build a Web App Using Firebase Authentication (使用Firebase认证建立Web应用)
这篇文章通过一步步地指导，介绍了如何利用Firebase平台实现用户注册、登录、身份验证、密码重置、社交登录功能。作者通过创建一个完整的Web应用来展示如何将Firebase和前端技术相结合，为用户提供安全、便捷的访问服务。
