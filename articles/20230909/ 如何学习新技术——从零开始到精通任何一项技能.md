# 2.基础知识

## 1.计算机是干什么的？

计算机是指能够存储数据并通过一定方式对其加工处理的电子设备。它由处理器、主存、输入输出设备组成。

## 2.计算机系统结构？

计算机系统结构是指计算机硬件和软件构成的整体。其中，硬件又包括处理器、主存、磁盘等，软件则主要包括操作系统、应用软件、网络服务等。

## 3.什么是编译型语言、解释型语言？

编译型语言是在源代码编译之后生成可执行文件，然后运行。它的优点是运行速度快、效率高，缺点是占用空间大、编译时间长。解释型语言是在运行时动态翻译，不需要预先编译，运行速度慢、效率低，但占用的空间小、启动时间短。

## 4.软件为什么要分为前端和后端？

前端负责与用户交互、处理用户界面；后端负责业务逻辑处理、数据库访问、服务器资源分配等工作。

## 5.什么是DBMS、NoSQL？

数据库管理系统(DBMS)是一种系统软件，用于存储、组织、检索和管理数据的软件。NoSQL（Not only SQL）是非关系型数据库，是一种基于键值对存储模型的数据库。

## 6.什么是消息队列？

消息队列（Message Queue）是一个应用程序编程接口，应用程序组件通过向消息队列发送消息，其他应用程序组件通过订阅消息队列并接收消息来进行通信。

## 7.分布式系统架构模式有哪些？

1. 分布式计算：这种模式允许多台机器上的计算任务同步执行，因此可以提升计算性能。
2. 分布式存储：这种模式允许多台机器共享同一个数据存储，因此可以有效降低数据存储成本。
3. 分布式调度：这种模式允许多台机器协同完成特定任务，提高任务处理的容错性、可靠性及可用性。
4. 分布式集群：这种模式允许多台机器共同组成集群，提供冗余备份、容灾恢复等功能。

# 3.编程语言

## 1.什么是Java、Python、JavaScript？

Java是世界上最流行的开源语言，可以创建面向对象、健壮、安全的应用程序。Python是一门具有简单语法和易于阅读的脚本语言，适合用来进行大规模数据处理、科学计算、web开发和机器学习等。JavaScript是一种轻量级的解释性语言，主要用于网页交互、移动App开发和后端服务。

## 2.Java基础语法有哪些？

1. 创建类：`class MyClass{...}`
2. 创建对象：`MyClass obj = new MyClass();`
3. 成员变量：`public static int x;`
4. 方法：`public void myMethod(){...}`
5. 控制语句：if-else、for循环、while循环、do-while循环。
6. 异常处理：try-catch-finally。

## 3.Python基础语法有哪些？

1. 定义变量：`x=1`，`name="John"`
2. 打印输出：`print("Hello World")`
3. 条件判断：`if condition:... elif condition:... else:`
4. 循环遍历：`for i in range(n):... for i in iter:... while condition:...`
5. 函数定义：`def my_func():... `

## 4.JavaScript基础语法有哪些？

1. 数据类型：string、number、boolean、null、undefined、object。
2. 变量声明：let varName; const varName; let/const [varName1, varName2]; let {prop1: value1, prop2: value2};
3. 条件语句：if (condition) {...} else if (condition){...} else {...}
4. 循环语句：for (let i = 0; i < n; i++) {...} do {...} while (condition); while (condition) {...}
5. 函数定义：function funcName() {...}
