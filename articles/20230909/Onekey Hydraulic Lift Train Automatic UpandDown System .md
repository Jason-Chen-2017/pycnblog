# 1.简介
  

## 一、产品概述
Hydraulic lift train (HLT) 是一种高度集成化、柔性运输系统。其运行效率高、全身覆盖、可自动调节路面形状、适合于车辆、船舶、重型设备等不同类型载体的运输。
HLT 的主要优点包括：

1. 高度集成化：HLT 使用集装箱、传送带及电机进行驱动，高度集成化降低了配件损耗，提升了整体经济效益；

2. 柔性运输：HLT 可以根据载体大小、重量、形态及路面条件自适应调整作动，因此可以实现运输物体在“自由落体”现象中快速移动；

3. 可调节路面形状：HLT 可以将“路面”的形状由无轨道到有轨道转换，根据运载对象不同而调整路面形状；

4. 全身覆盖：HLT 在整体的高压驱动下，可覆盖整个身体，保证安全、完整的运输；

5. 高速运行：HLT 的运行速度比单独使用电机更快，时速超过万米；

6. 更长的寿命：HLT 的耗能、绝缘材料和机械性能都具有较高的耐久性，能够承受多次重复使用。

目前市场上几种类型的 HLT 有：

1. 发动机驱动 HLT：采用发动机驱动的 HLT 把货物交付给消费者后，通过可拖放或倒车让货物沿着所需路线流向目的地。这种 HLT 一般安装在客运汽车或货运卡车内。

2. 轴承驱动 HLT：轴承驱动 HLT 将货物从发货车或供应商处拉取出，然后再推入目的地。这种 HLT 用集装箱作为运载单元，可使用不同形状和大小的货物，具有抗风和振动的能力。

3. 混凝土驱动 HLT：混凝土驱动 HLT 用可伸缩的混凝土支架固定货物并加力驱动，不用担心货物弄湿或者滑落，能够实现准确的高度控制。

4. 电机驱动 HLT：电机驱动 HLT 通过自动扭矩调节支架的位置，使货物在空气或水中自由移动。这种 HLT 具有较大的空间容积积累能力和灵活性。

## 二、产品功能模块
HLT 是一套由驱动、制动系统、传输装置组成的整体系统。其中驱动系统用于产生动力，制动系统用于减少反力，传输装置则用于使载物体在自由落体状态下前进。HLT 包含两个主要的模块，即控制模块（Control Module）和感知模块（Perception Module）。它们各自负责不同的工作，下面我们将对它们逐个进行介绍。
### （1）控制模块 Control Module
#### 1. 发动机驱动型 HLT
发动机驱动型 HLT 是一个高度集成化、可靠性强、易于维护的驱动系统。它由连接至主缆的驱动发动机、电机和其他部件构成，并且还含有一个座舱来确保货物的安全。HLT 的发动机驱动的主要特点有：

1. 发动机驱动型 HLT 可以实现高度集成化。由于发动机位于 HLT 的中心位置，所以它可以很容易地添加或删除齿轮、连接组件、改装车身、修复故障。

2. 发动机驱动型 HLT 可以实现可靠性强。使用电子元件、控制程序、专用传感器和结构设计，发动机驱动型 HLT 提供了强大的抗力传动能力、良好的位置操纵性能和弹性运动特性。

3. 发动机驱动型 HLT 可以实现易于维护。制造商通常会提供丰富的工具和服务，如起落架、电源管理单元、保险柜、检验系统和维修服务，帮助客户轻松地修补、重新安装和维修他们的 HLT。

4. 发动机驱动型 HLT 具备高度耐用的火花散热系统，可以有效防止烫伤、开裂、冻结、撞击和刮伤，并减轻海浪、风暴等因素的影响。

#### 2. 轴承驱动型 HLT
轴承驱动型 HLT 使用一个轴承（连杆系）来将货物从发货车或供应商处拉取出，然后推入目的地。轴承驱动型 HLT 的主要特点有：

1. 轴承驱动型 HLT 可以实现高度集成化。与发动机驱动型 HLT 一样，轴承驱动型 HLT 中的齿轮可以随意增加、删除、替换，也可以安装在客运汽车或货运卡车的内部。

2. 轴承驱动型 HLT 可以实现多样化运载。在轴承驱动型 HLT 中，货物可以选择以各种形状、大小和颜色运载，同时也可以使用各种物料、设备、胶囊和精密工具。

3. 轴承驱动型 HLT 可以实现自适应调度。轴承驱动型 HLT 可以根据车辆或船舶的移动方向和轨道来调整货物的位置。

4. 轴承驱动型 HLT 可以实现高效的控制。由于轴承的作用，轴承驱动型 HLT 比发动机驱动型 HLT 更加灵活、精确。它可以自动改变货物的速度和方向，并防止货物的坠落和爆炸。

#### 3. 混凝土驱动型 HLT
混凝土驱动型 HLT 是一种可伸缩、高度集成化的驱动系统，它使用混凝土支架固定货物并加力驱动，不用担心货物弄湿或者滑落。混凝土驱动型 HLT 的主要特点有：

1. 混凝土驱动型 HLT 可以实现高度集成化。与发动机驱动型 HLT 和轴承驱动型 HLT 一样，混凝土驱动型 HLT 中使用的混凝土支架、混凝土座、杆件和夹具都是高度集成化的。

2. 混凝土驱动型 HLT 可以实现自适应调度。混凝土驱动型 HLT 可以根据运载对象的速度、位置、轨道方向来调整货物的位置。

3. 混凝土驱动型 HLT 可以实现较高的重量。由于货物周围的空气阻力小，因此混凝土驱动型 HLT 的重量不会过大，更适合重型货物。

4. 混凝土驱动型 HLT 可以实现空间容积积累。因为混凝土座位于支架上方，所以它可以帮助货物填充满空间。

#### 4. 电机驱动型 HLT
电机驱动型 HLT 也称为直流电机驱动型 HLT。它使用一个电机和支架来驱动货物，可以随时改变电机的转速。电机驱动型 HLT 的主要特点有：

1. 电机驱动型 HLT 可以实现高度集成化。电机驱动型 HLT 中使用的电机、支架和控制电路是高度集成化的，可以方便地替换部件。

2. 电机驱动型 HLT 可以实现便携性。与混凝土驱动型 HLT 一样，电机驱动型 HLT 可以随手持、卡车、飞机甚至燃气或液体驱动。

3. 电机驱动型 HLT 可以实现较高的重量。电机驱动型 HLT 的重量要比混凝土驱动型 HLT 小得多。

4. 电机驱动型 HLT 可以实现灵活的控制。电机驱动型 HLT 可以使用定速模式、自适应巡航、限速模式、超声波探测、测距、激光雷达探测等多种方式来控制货物的运动。

### （2）感知模块 Perception Module
#### 1. 活体检测 Active Sensing
为了让 HLT 在复杂的环境中正确识别、跟踪和识别目标，需要一种高精度、高实时、多模态、联网、跨平台的活体检测系统。主要包括摄像头、IMU、激光雷达、雷达外设、GPS等传感器。活动检测是 HLT 对对象位置信息和环境变化信息进行实时的获取和分析，进行相应的决策。

#### 2. 图像处理 Image Processing
为了实现更精准的路面的自适应、动态路况的变化和地图信息的融合，HLT 需要具有图像处理能力，对载体的图像进行清晰、高效的分割、计算、跟踪、识别、规划和决策。图像处理包含目标识别、路线规划、路径规划、地图匹配、场景识别、运动模型、运动预测、异常检测、三维建模、数据融合等。

#### 3. 数据传输 Transmission
为了实现通信的高效、实时、准确，HLT 需要数据的传输系统。主要包括串口通讯协议、WiFi、蓝牙、Zigbee等传输方式。传输系统的实现需要考虑信号干扰、传输距离、传输频率等因素，并保证数据传输的准确、实时、可靠。

#### 4. 控制和反馈 Control and Feedback
为了获得更高的效率和精度，HLT 还需要具有鲁棒、稳定的控制系统。控制系统的实现包含三级调节器控制、PID控制器控制、速度补偿控制、位置保护控制、红外循迹检测控制、车道保持控制、急停控制等。此外还需要具有实时、可靠、精确的反馈系统，以便于监控和优化系统的性能。
