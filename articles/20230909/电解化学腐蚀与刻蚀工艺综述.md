# 1.简介
  

电解化学腐蚀（Electrolytic Irreversible Polymerization, EIP）又称化学交联反应或电解液相互作用。它是一个尤其重要且广泛存在于生活中的过程，也是制造各种电子产品，例如无线传输、半导体、照明电池等，所需材料中最重要的组成元素之一。但是，电解化学腐蚀在不同材料之间、相同或不同的温度下出现了不同的行为模式，以及随着工艺技术的发展，各种腐蚀现象也日益复杂而多样化。因此，对电解化学腐蚀工艺、材料特性及工艺流程的综合认识至关重要。本文通过对《电解化学腐蚀及其工艺》、《电解化学、物理化学与热力学研究进展》等文献的综述，梳理电解化学腐蚀与刻蚀的基本理论、工艺条件和工艺模式，以及相关材料的结构及性能特征，并简要介绍电解化学腐蚀、刻蚀机理、腐蚀性质和防护措施等知识。文章末尾附有《电解化学腐蚀与刻蚀工艺综述》系列论文的目录。

# 2.电解化学腐蚀与刻蚀
## 2.1 什么是电解化学腐蚀
电解化学腐蚀（EIP），指的是一种电离过程中，两个或多个分子在一个物质上发生融合形成复合物的一类化学反应。这一现象由于自由基分子互相干扰，形成气泡，导致某种物质吸收二氧化碳气体而排出碳氢化合物，从而起到对环境的污染、生态破坏、生化代谢的作用。因此，电解化学腐蚀是众多环境污染、生物疾病、微生物传播等的重要原因。

## 2.2 为什么需要电解化学腐蚀？
电解化学腐蚀既可以作为一种无机物离子离层表面性质改变的有益机制，也可以作为一种有害的过程，对环境的影响因素有着广泛的贡献。其直接影响因素包括：

1. 健康：电解化学腐蚀会使细胞内存在自由基分子聚集，抵消免疫系统的作用，引起真菌感染、肺炎等不良反应；并且，还会减少细胞器官活性，引起癌症发病。

2. 污染：电解化学腐蚀是环境污染的主要形式，如废弃物、焦油、放射性物质等。

3. 生物物理：电解化学腐蚀在生命科学领域还有很多应用前景。如生物工程中利用电解化学腐蚀与遗传信息进行DNA修饰；以及生物体内的电解液相互作用，可调节生物的能量和蛋白质消化吸收，并产生多种生物反应。

4. 消耗资源：电解化学腐蚀对环境产生的影响是不可忽视的，特别是在电子设备的制造、测试、运输和消费领域。

5. 社会经济：电解化学腐蚀是一种垄断性的产业链环节，消费者不得不服从工厂商、供应商提供的服务，同时还存在“电解茅台”等不正当竞争行为。

## 2.3 电解化学腐蚀主要类型
目前，电解化学腐蚀主要包括四种主要类型：

1. 电解液相互作用型：这是电解化学腐蚀最普遍和常用的形式。电解液相互作用型，由氢气对磷酸盐的氢氧化硅-氯化钠 (SO2C) 反应所致，故称为电解液相互作用型。

2. 物体自然接触型：这种腐蚀通常发生在容器表面，对水、空气、土壤等大气相当敏感，同时还受到工艺条件、腐蚀物质和温度等诸多因素的影响。

3. 电容器/建筑物腐蚀型：这种腐蚀通常发生在窨井或建筑物的边缘、内部或周围。由于燃烧熔岩释放的热能损失较高，故这种腐蚀行为被称为电容器腐蚀型。

4. 化学栓膜腐蚀型：这种腐蚀发生在建筑材料上的化学微粒的结合。由于缺乏有效的抗腐蚀措施，故此成为影响建筑材料质量、耐用性和安全性的重要原因。

## 2.4 电解化学腐蚀原理
### 2.4.1 电解液相互作用型电解化学腐蚀
电解液相互作用型电解化学腐蚀，是指由氢气对磷酸盐的氢氧化硅-氯化钠 (SO2C) 反应所致的电解化学腐蚀。它的一般过程包括以下几个步骤：

1. 激发化学反应: 电解液中氢气通过少量的氢键激发氢氧化硅-氯化钠 (SO2C) 的反应，形成电离聚团。

2. 活性生成: 电离聚团激发SO2C的化学反应后，将释放出一定的电离气体，形成可以氧化的活性分子，它们与氢气共同进入氯化钠的活性烃分子。

3. 分散聚合: 活性分子经过数个跳跃而分散聚合，形成两批气泡，其中一条在氢氧化物内部，另一条则包裹着部分电离气体，此时就发生电解化学腐蚀。

4. 逆电解作用: 在腐蚀发生期间，部分电离气体由氧化物降解，构成了一定的电解液。在电解液作用下，氢气便逆向运动，释放出氧气，形成气态物质，导致腐蚀效率提高。

### 2.4.2 物体自然接触型电解化学腐蚀
物体自然接触型电解化学腐蚀，通常发生在水、空气、土壤等大气相当敏感的地方。它的原因是高浓度氢气对大气中的电解离子产生响应。氢气不断涌入、流动和扩散，形成沉积的大气电解液，它在小范围内长时间停留，形成电解化学腐蚀，形成气态物质。腐蚀发生后，电解液可以在很短的时间内就会融化。

### 2.4.3 电容器腐蚀型电解化学腐蚀
电容器腐蚀型电解化学腐蚀，主要发生在窨井或建筑物的边缘、内部或周围，由于燃烧熔岩释放的热能损失较高，故这种腐蚀行为被称为电容器腐蚀型。它的发生原因是低氢气压在熔岩表面导致熔化，高温烘烁熔岩，产生低碳氢化合物，高溢氢气和氢氧化硅-氯化钠 反应形成电解化学腐蚀。

### 2.4.4 化学栓膜腐蚀型电解化学腐蚀
化学栓膜腐蚀型电解化学腐蚀，主要发生在建筑材料上的化学微粒的结合。由于缺乏有效的抗腐蚀措施，故此成为影响建筑材料质量、耐用性和安全性的重要原因。它是因为材料的表面或裂纹以及腐蚀物质附着在一起，形成电解化学腐蚀。如图3-1所示，通常可分为下面两种情况：
