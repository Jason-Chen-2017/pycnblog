                 

# 1.背景介绍


作为IT界最火的新兴技术之一，人工智能（AI）正在引领着智能化的进程。而基于AI实现的自动化、无人化处理方案也在蓬勃发展。如何用AI助力业务流程管理，就是本文将要讨论的问题。在这个过程中，需要考虑到两个方面，第一是AI能力的训练、第二是 Agent 的性能。下面我们先从业务需求出发，介绍一下我们想要解决的业务需求。

## 业务需求
### 公司需求
某公司在运营中发现了一系列客户服务流程不完善的问题。这些流程包括但不限于，排队叫号、上门办理业务手册、缴费等。作为一个企业，如果想在短时间内提高效率，就要尽可能减少手动重复性工作量，而自动化则可以快速响应客户请求，有效节省人力资源。因此，公司决定建立一套AI系统来优化客户服务流程。


### 项目需求
#### 问题分析
**需求背景**：客户经常问询一些关于排队叫号、预约咨询、预定体检等流程，每天都会遇到各种各样的业务，甚至有些环节会忘记一些细节，导致客户在等待时长、费用、信息不全面的情况下，造成顾客滞留、交叉流动。

**需求现状**：目前，公司主要的客户服务工作均由人工完成，导致整个工作流程耗时较久、效率低下，且容易出现错漏或者错误。

**客户诉求**：希望能够简化排队叫号、预约咨询等客户服务流程，减轻工作负担，并提升工作效率。

#### AI方案设计思路
通过使用 GPT-3 搭建一个实体识别系统。利用GPT-3的强大语言模型，能够识别客户的订单信息、地址信息、名称信息等，根据上下文推断用户意图，直接进行对应的任务，如自动开具发票、生成预约记录等。而无需人工参与进一步操作，可以极大的缩短客户服务时间。
