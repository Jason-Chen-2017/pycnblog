# 2.核心概念与联系
## 2.1 变量和数据类型
### 2.1.1 变量
&emsp;&emsp;变量（variable）就是存放数据的容器。每个变量都有一个唯一的名字（标识符），用来识别这个变量，后续可以通过这个标识符访问该变量的值。变量的命名规则遵循PEP 8命名规范。

### 2.1.2 数据类型
&emsp;&emsp;数据类型（data type）描述的是变量所存储或运算的值的特性，包括大小、取值范围、精度、单位等。不同的数据类型在内存中占有不同的空间，因此相同的数据类型在不同编程环境下可能占用的空间也是不同的。Python中共有八种基本的数据类型：整数int、浮点数float、布尔值bool、字符串str、列表list、元组tuple、集合set、字典dict。

## 2.2 条件判断和循环语句
### 2.2.1 if语句
&emsp;&emsp;if语句是条件判断语句，即根据某个表达式的值（True或者False），决定是否执行对应的代码块。语法形式如下:
```python
if expression:
    code block # 当expression为True时执行的代码块
else:
    other code block # 当expression为False时执行的代码块
```

&emsp;&emsp;示例：
```python
x = int(input("请输入数字："))
if x > 0:
    print("正数")
elif x < 0:
    print("负数")
else:
    print("零")
```

&emsp;&emsp;以上代码首先获取用户输入的数字，然后根据该数字的正负，分别输出“正数”、“负数”或“零”。

### 2.2.2 for循环
&emsp;&emsp;for循环是一种重复执行特定代码块的语句。语法形式如下：
```python
for variable in iterable_object:
    code block # 每次迭代，变量的值都会从iterable_object中取出，然后执行一次代码块
```

&emsp;&emsp;示例：
```python
for i in range(5):
    print("*" * (i + 1))
```

&emsp;&emsp;以上代码打印输出如下图所示的星形图案：
```
   *
  ***
 *****
*******
 *****
  ***
   *
``` 

### 2.2.3 while循环
&emsp;&emsp;while循环是一种重复执行特定代码块的语句，但与for循环不同的是，while循环在执行之前需要先确定一个条件，只有满足条件才会执行。语法形式如下：
```python
while condition:
    code block # 只要condition为True，就会一直执行代码块
```

&emsp;&emsp;示例：
```python
count = 0
num = input("请输入数字:")
while count < len(num):
    print(num[count])
    count += 1
print('结束')
```

&emsp;&emsp;以上代码获取用户输入的数字，然后依次输出数字中的每一个字符。
