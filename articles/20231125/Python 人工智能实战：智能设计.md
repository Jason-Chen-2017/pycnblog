# 2.核心概念与联系

## 概念概览

机器学习（Machine Learning）：机器学习是指让计算机具备学习能力，并根据所学到的经验改善性能的一种技术。它是人工智能（Artificial Intelligence）的一个分支领域，其研究范围甚至比其他的领域都要广泛得多。

深度学习（Deep Learning）：深度学习是指通过对数据进行多层次的神经网络运算，从而达到基于输入数据的预测和识别目的的机器学习技术。深度学习已成为机器学习领域里最热门的研究方向之一。

图像识别：图像识别是指利用计算机技术从图片、视频或拍摄的照片中自动检测和识别出目标物体。图像识别的典型应用场景包括：人脸识别、手势识别、场景识别、文档识别、生物特征识别、 logo 检测、身份证OCR等。

文字识别：文字识别（Optical Character Recognition，简称OCR）是指通过计算机识别出手写或 printed 出来的文字的过程，通常用于解决扫描文字识别、文档文字识别、车牌识别等高精度识别需求。文字识别具有很高的准确性，同时也降低了成本，方便快捷。

数据分析：数据分析是指对收集到的、存储在数据库、文件或网络上的各种数字、文字、图形等数据进行统计、排序、关联、决策等分析处理的过程。数据分析的目的在于找出规律，发现模式，并通过对数据的呈现方式、报告结果、制定政策等，从而提高数据的价值、洞察商业机会、改进产品和服务。

智能设计：智能设计是指通过机器学习和深度学习技术，结合客户需求、市场竞争力、历史经验等因素，为企业提供有针对性的新产品和服务，提升品牌知名度、品质保障、利润增长等。
