# 2.核心概念与联系
## 2.1 机器学习的定义及其发展历程
### 2.1.1 机器学习的定义

机器学习（英语：Machine Learning）是一门交叉学科，涉及计算机科学、统计学、工程学等多个领域。它以数据为输入，通过训练算法生成模型，对未知的数据进行预测或决策，即所谓的“学习”。换句话说，机器学习可以让计算机提升自动化水平，并利用数据发现模式和规律，从而做出更加准确的预测或决策。由于机器学习模型一般都比较简单，因此也称之为“朴素贝叶斯”、“感知机”、“支持向量机”等。

### 2.1.2 机器学习的发展历史

1959年，<NAME>教授提出了“学习理论”，主要内容是研究如何用最小化误差的方法从数据中学习到有效的规则或模型。1970年，卡内基梅隆大学教授李航创立了西安电子科技大学，开设关于机器学习的课程，并成立了机器学习研究小组。1980年代末，随着互联网和大数据等信息技术的普及，机器学习得到迅速发展。从那时开始，机器学习领域经历了多次快速发展。

#### （1）机器学习理论

- 贝叶斯估计：贝叶斯方法是一种基于概率论的估计方法。在贝叶斯估计中，已知某些事情的发生频率后，根据这些频率计算未知事物的可能性。
- 感知机：感知机是最简单的神经网络，它由两层神经元组成，其中输入层接收输入信号，输出层产生输出信号。输入信号通过权重连接到隐含层的神经元，隐含层的每个神经元都计算输入信号经过sigmoid函数之后的值。如果该值超过阈值，则激活该神经元，否则不激活。输出信号就是激活的神经元的索引。
- 提升法：提升法是一种迭代算法，它使用多项式函数逼近损失函数的极限，并且每次迭代都会加入一个新的、更好的基函数集。
- EM算法：EM算法（Expectation-Maximization Algorithm）是一种最大似然估计算法，它用于参数估计。

#### （2）统计学习方法

- EM算法：EM算法是一种用于无监督学习的迭代算法。它的基本想法是，首先假设模型参数的初始值，然后用当前的参数值估计模型的期望；然后，根据这个期望计算参数的新的值，并重复这个过程，直至收敛。
- SVM：SVM是一种支持向量机（Support Vector Machine），它是一种二类分类器，能够有效地解决线性不可分的问题。
- CART：CART是一种决策树（Classification And Regression Tree）算法。

#### （3）深度学习

- BP算法：BP算法（Backpropagation Algorithm）是一种用于训练多层神经网络的优化算法。
- RNN：RNN（Recurrent Neural Network）是一种递归神经网络，它能够处理序列数据，如文本、音频、视频等。
- CNN：CNN（Convolutional Neural Networks）是一种用于图像识别和分类的卷积神经网络。

#### （4）强化学习

- Q-learning：Q-learning是一种在非静态环境下学习控制策略的模型-学习算法。
- Sarsa：Sarsa是一种在非静态环境下学习控制策略的半梯度算法。
- DQN：DQN（Deep Q Network）是一种用于游戏 AI 的强化学习算法。

#### （5）迁移学习

- AdaMatch：AdaMatch是一种数据增强技术，用于半监督学习任务中的标签偏差。
- BYOL：BYOL是一种无监督学习技术，可实现无监督特征学习。

#### （6）其他

- AlphaGo：AlphaGo 是人工智能围棋大师李世乭开发的一款机器人。他通过强化学习、蒙特卡洛树搜索以及前馈神经网络赢得了国际象棋界最高奖项。
- AlphaZero：AlphaZero 是谷歌 Deepmind 团队基于 AlphaGo 的强人工智能系统。
- One-shot learning：One-shot learning 是指仅有一个样本就可以完成学习的机器学习任务。
- Federated learning：Federated learning 是一种机器学习方法，其中多个参与者各自参与不同的数据集上的模型训练，最终汇总它们的模型参数。
