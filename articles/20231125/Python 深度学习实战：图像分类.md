# 2.核心概念与联系
## 2.1 什么是图像分类？
图像分类是计算机视觉中一种基本任务，它的目标是根据给定的图像或视频帧中所显示的对象或场景，将其划分为不同的类别。图像分类的目的在于自动对图像进行分类，将其归类到各个类别当中。例如，一张图像可能属于“狗”，“飞机”，“树”等多个类别之一。图像分类也是许多应用领域的关键问题，如安全保障系统、交通监控、导航、情感分析等。
## 2.2 为什么需要图像分类？
图像分类是人工智能领域的一个重要研究方向。图像分类有很多优点，比如减少了人力资源消耗，节省了时间成本；可以提高产品性能、降低成本；更好地满足用户需求。因此，图像分类具有广泛的应用价值。相比而言，传统机器学习方法依赖于特定的特征工程和手动设计特征，这些方法往往无法捕捉到图像中的所有信息，图像分类方法可以更加直接地识别图像中的物体。另外，通过图像分类可以对图片进行自动审核、图像内容分析、图像搜索引擎、视觉识别、视频监控、安全防范等方面的应用。
## 2.3 图像分类的分类方式
图像分类可分为基于结构的方法和基于内容的方法。
### 2.3.1 基于结构的方法
基于结构的方法主要依赖于特征提取和机器学习算法，其基本原理是将图像的几何信息转化为特征向量，再利用机器学习算法对特征向量进行学习和分类。这种方法通过对图像的空间信息、尺度信息、纹理信息等进行提取，可以较好地捕捉图像的特征，但往往存在很高的维度灾难（curse of dimensionality）。典型的基于结构的方法包括基于边缘检测、卷积神经网络（CNN）和区域生长网络（R-CNN），具体如下图所示：
