                 

# 1.背景介绍


在计算机科学领域中，条件语句（英语：Control statement）用于根据判断的结果，来决定下一步执行的流程。条件语句分成了五种类型，分别是：

1、IF-THEN语句：指明一个条件并给出对应的动作，如果满足这个条件则进行对应的动作。例如：“如果今天天气晴朗，我就去打羽毛球。”；“如果能抵押一辆新车，我就买保险。”

2、IF-ELSE语句：指明一个条件，如果满足这个条件则执行对应的动作A，否则执行对应的动作B。例如：“如果今天天气晴朗，我就去打羽毛球；否则，我就去爬山玩耍。”

3、IF-ELIF-ELSE语句：有多个条件判断，满足第一个条件时执行对应的动作A，满足其他条件时执行对应的动作B。例如：“如果今天天气晴朗，我就去打羽毛球；如果今天下雨，我就开车去海边溜达；否则，我就去爬山玩耍。”

4、FOR语句：从头到尾遍历一系列值，依次对每个值执行某种操作。例如：“我想吃香蕉，所以每次洗完澡都会去买香蕉”。

5、WHILE语句：重复执行一段代码，直到某个条件满足。例如：“我想买菜，但是我还没钱，所以每次买菜前都要检查一下是否有钱”。

# 2.核心概念与联系
## 2.1 Python条件语句概述
条件语句（Control Statement），又称为选择结构或条件表达式（Conditional Expression）。Python支持if-else语句、for循环语句、while循环语句。

Python的条件语句的语法形式如下：
```python
if condition:
    # if条件为True时执行的代码块
elif condition:
    # 如果前面的条件都不满足，那么按照此条件进行判断，如果为True时执行的代码块
else:
    # 当所有条件都不满足时，执行此代码块
```
## 2.2 条件表达式
在Python中，条件表达式是由if、elif和else关键字组成的表达式。它的一般语法形式为：
```python
expression1 if condition else expression2
```
其中，`condition`是一个布尔表达式，表示满足条件执行`expression1`，不满足执行`expression2`。`expression1`和`expression2`都是Python表达式，可以是任意合法的Python语句，也可以是函数调用或者变量赋值语句。当条件表达式的值为真(True)时，将返回`expression1`的值；否则返回`expression2`的值。

## 2.3 条件运算符
在Python中，条件运算符用于创建条件表达式。它由两个条件表达式（也可包含第三个默认表达式）和逻辑运算符组成。运算符包括and、or和not三个。它们的优先级规则为：not > and > or。

## 2.4 IF-THEN语句
### 2.4.1 简单示例
```python
number = int(input("请输入一个数字："))   # 用户输入一个整数作为判断条件
if number % 2 == 0:    # 判断输入的数字是否为偶数
    print("%d 是偶数" % (number))    # 输出输入的数字
else:
    print("%d 是奇数" % (number))    # 输出输入的数字
```
例子中，用户通过输入一个整数作为判断条件，若该整数是偶数，则输出输入的数字；否则输出输入的数字。利用%d来打印整数。

运行后，程序会提示用户输入一个数字，输入`6`后，输出结果为：
```python
6 是偶数
```

### 2.4.2 带有else子句的示例
```python
x = float(input("请输入 x 值："))
y = float(input("请输入 y 值："))

if x >= 0:       # 判断 x 是否大于等于零
    if y >= 0:   # 判断 y 是否大于等于零
        print("(%f, %f)" % (x, y))     # 输出坐标点
    else:        # y 小于零
        print("(infinity, infinity)")   # 输出无穷远点
else:            # x 小于零
    if y >= 0:   # y 大于等于零
        print("(-infinity, -infinity)") # 输出负无穷远点
    else:        # y 小于零
        print("(%f, %f)" % (-x, -y))    # 输出坐标点
```
例子中，用户通过输入 x 和 y 值作为判断条件，若 x 和 y 的符号均大于等于零，则输出坐标点；若 x 为负，y 为正，则输出无穷远点；若 x 为负，y 为负，则输出坐标点；若 x 为正，y 为负，则输出负无穷远点。

运行后，程序会提示用户输入 x 和 y 值，输入 `-2` 和 `3` 时，输出结果为：
```python
(-2.000000, 3.000000)
```

### 2.4.3 多重条件下的if语句
```python
a = int(input("请输入 a 值："))
b = int(input("请输入 b 值："))
c = int(input("请输入 c 值："))

if a < b:          # 判断 a 是否小于 b
    if b < c:      # 判断 b 是否小于 c
        print("三者之间存在大小关系")         # 输出结果
    else:           # b 大于等于 c
        print("a 小于 c，且 b 大于等于 c")    # 输出结果
elif a < c:        # a 小于 c，且 b 大于等于 a 或 a 等于 c
    print("a 小于 c")                    # 输出结果
else:               # a 大于等于 c 或 a 等于 b
    print("b 小于等于 a 或 b 等于 c")      # 输出结果
```
例子中，用户通过输入三个整型值作为判断条件，程序判断这三个值的大小关系。先判断 a 和 b 的大小关系，然后再判断 b 和 c 的大小关系，最后输出结果。

运行后，程序会提示用户输入三个值，输入 `1`, `2`, `3` 时，输出结果为：
```python
a 小于 c
```
