# 2.核心概念与联系
## 数据类型
Python 中有以下几种基本的数据类型：
- Numbers(数字)
- String（字符串）
- List（列表）
- Tuple（元组）
- Set（集合）
- Dictionary（字典）

### Numbers
Python 中的数字可以分为整型、浮点型、复数型三种：

1. 整数（int）类型
   - Python 中整型没有大小限制，可以用来表示任意大小的整数值，也可以使用补码形式表示负数。例如：`a = 123`、`b = -987`。
   - 使用`type()`函数可以检查变量所属的数据类型。
    ```python
    >>> a = 123
    >>> b = -987
    >>> type(a)
    <class 'int'>
    >>> type(b)
    <class 'int'>
    ``` 
2. 浮点型（float）类型
   - Python 中浮点型用来表示小数。它的大小范围比整型要宽松一些，通常可以表示更大或更小的数值，但不精确。例如：`c = 3.14`、`d = -2.5`。
   - 使用`type()`函数可以检查变量所属的数据类型。
    ```python
    >>> c = 3.14
    >>> d = -2.5
    >>> type(c)
    <class 'float'>
    >>> type(d)
    <class 'float'>
    ```
3. 复数型（complex）类型
   - 复数型由实部和虚部构成，用 a+bj 表示，其中 a 和 b 分别为实部和虚部。可以使用 `complex()` 函数或者将实部和虚部用 `j` 或 `J` 标记，来表示一个复数。例如：`e = complex(2, 3)`、`f = 1 + 2j`。
   - 使用`type()`函数可以检查变量所属的数据类型。
    ```python
    >>> e = complex(2, 3)
    >>> f = 1 + 2j
    >>> type(e)
    <class 'complex'>
    >>> type(f)
    <class 'complex'>
    ```

### String（字符串）
String 是 Python 中最常用的数据类型之一。字符串可以存储任何文本信息，可以以单引号 `' '`、双引号 `" "`、三引号 `''' '''` 或 `""" """` 来创建。如下例：
```python
>>> greeting = "Hello, world!"
>>> print(greeting)
Hello, world!
```
可以通过索引的方式访问字符串中的各个字符，从左往右依次为 `0` 到 `len(string)-1`，也可以通过切片的方式获取子串。例如：
```python
>>> name = "Alice"
>>> # 通过索引访问
>>> print(name[0])   # H
>>> print(name[-1])  # e
>>> # 通过切片访问
>>> print(name[:3])  # Alic
>>> print(name[1:])  # ice
```

### List（列表）
List 是 Python 中另一个非常常用的内置数据类型。列表中可以存储任意数量的元素，且每个元素可以是不同的数据类型，可以以方括号 `[ ]` 来创建。如下例：
```python
>>> numbers = [1, 2, 3]
>>> fruits = ["apple", "banana", "orange"]
>>> mixed_list = [1, "apple", True]
```
可以通过索引的方式访问列表中的各个元素，从左往右依次为 `0` 到 `len(list)-1`，还可以指定步长进行切片。例如：
```python
>>> numbers = [1, 2, 3, 4, 5]
>>> print(numbers[::2])     # 从索引 0 到 4，间隔为 2 的切片
[1, 3, 5]
>>> print(numbers[::-1])    # 从索引 0 到 4，反序排序后的切片
[5, 4, 3, 2, 1]
```
列表还提供许多内置方法来对元素进行操作，如 `append()` 方法用于添加元素到末尾，`insert()` 方法用于插入元素到指定位置，`pop()` 方法用于删除指定位置的元素并返回该元素的值，`remove()` 方法用于删除指定值的第一个元素，`sort()` 方法用于排序列表，`reverse()` 方法用于翻转列表，等等。

### Tuple（元组）
Tuple 是另一种有序列表，与列表类似，但是不同之处在于元组不能修改，只能读取。可以以圆括号 `( )` 来创建。例如：
```python
>>> person = ("Alice", 25, "female")
>>> city = (12345, "Beijing", "China")
```

### Set（集合）
Set 是 Python 中又一个非常常用的内置数据类型，集合是一个无序不重复元素的集。集合中的元素必须是不可变的，而且可以自动保持元素之间不存在重复关系。可以用花括号 `{ }` 创建集合，如下例：
```python
>>> s = {1, 2, 3}
>>> s.add(4)      # 添加元素到集合
>>> s.discard(2)  # 删除元素
```
集合提供的方法主要有：
- add()：向集合中添加元素；
- clear()：清空集合；
- copy()：复制集合；
- difference()：返回两个集合的差集；
- difference_update()：更新当前集合，使得它等于另外一个集合的差集；
- discard()：从集合中删除指定的元素，如果元素不存在则忽略；
- intersection()：返回两个集合的交集；
- intersection_update()：更新当前集合，使得它等于另外一个集合的交集；
- isdisjoint()：判断两个集合是否存在相同的元素；
- issubset()：判断当前集合是否为另外一个集合的子集；
- issuperset()：判断当前集合是否为另外一个集合的超集；
- pop()：随机移除集合中的一个元素并返回该元素的值；
- remove()：删除集合中的指定的元素；
- symmetric_difference()：返回两个集合的对称差集；
- symmetric_difference_update()：更新当前集合，使得它等于两个集合的对称差集；
- union()：返回两个集合的并集；
- update()：向当前集合中添加多个元素。
