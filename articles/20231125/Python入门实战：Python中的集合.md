                 

# 1.背景介绍


## 概念定义及其重要性
**集合（Set）**：又称无序、不重复的元素组成的集体。在编程中，集合用于存储一些互相之间没有任何关系的数据项或对象。比如，程序需要处理一个商品订单的数量，就可以用集合来存储这些商品，这样可以快速地判断某种商品是否已经在这个订单里了。而集合的另一个重要作用则是在数据分析、机器学习、人工智能等领域，用它来存储训练样本或测试集，对数据进行去重、交叉验证、划分训练集、测试集等操作都非常方便。

那么为什么要使用集合呢？很简单，因为集合让开发者可以利用高效率的方式管理复杂的数据结构。在实际应用场景中，我们可能需要处理大量的数据，但大部分情况下，相同的数据只会出现一次。对于这种“重复”数据，集合能够通过快速查找、删除、判重操作实现数据的快速存储、处理。而且，集合还提供了一系列的方法来完成各种集合运算，使得开发者可以灵活地控制复杂的数据流转流程。

总之，使用集合能够让我们快速有效地解决复杂的数据问题，并提升编程效率。

## Set与列表、字典的区别
与列表、字典不同的是，集合没有索引和顺序，因此不能像列表一样通过下标访问某个元素。同时，集合也不能修改元素的值。 

另外，由于集合不能包含重复值，所以如果插入一个重复值，集合会自动忽略该值。

# 2.核心概念与联系
## 集合的创建方式
创建集合的方式有三种：

1. 使用set()函数创建空集合：
   ```python
    s = set()
   ```
2. 使用{}创建集合：
   ```python
    s = {x for x in range(5)} # 创建一个集合{0, 1, 2, 3, 4}
   ```
3. 将可迭代对象转换为集合：
   ```python
    my_list = [1, 2, 2, 3, 4]
    s = set(my_list) # 创建集合{1, 2, 3, 4}
    # 可以直接使用{...}创建集合也可以将可迭代对象转换为集合
   ```

   **注**：在第三种方法中，如果可迭代对象中存在同名元素，则后面的值将覆盖前面值。比如：
   ```python
   lst = [('apple','red'), ('banana', 'yellow'), ('apple', 'green')]
   dct = dict(lst) # {'apple': 'green', 'banana': 'yellow'}
   ```


## 集合的基本操作
### 添加元素
- add() 方法添加单个元素到集合中：
  ```python
  s = {1, 2, 3}
  s.add(4)  
  print(s)    #{1, 2, 3, 4}
  ```
- update() 方法添加多个元素到集合中：
  ```python
  s = {1, 2, 3}
  s.update([4, 5]) 
  print(s)      #{1, 2, 3, 4, 5}
  
  t = {4, 5, 6}
  s.update(t)
  print(s)        #{1, 2, 3, 4, 5, 6}
  ```
  
### 删除元素
- remove() 方法删除指定元素：
  ```python
  s = {1, 2, 3}
  s.remove(2)
  print(s)     #{1, 3}
  ```
- discard() 方法删除指定的元素，如果元素不存在，则不报错：
  ```python
  s = {1, 2, 3}
  s.discard(2)
  print(s)       #{1, 3}
  
  s.discard(4)  # 不报错，没有效果
  print(s)       #{1, 3}
  ```
- pop() 方法随机删除元素，不传参时默认弹出任意元素：
  ```python
  s = {1, 2, 3}
  e = s.pop()
  print(e, s)    # 输出任意元素以及剩余集合{1, 2, 3}
  ```
- clear() 方法清空集合：
  ```python
  s = {1, 2, 3}
  s.clear()
  print(s)          # {}
  ```

### 查询元素
- issubset() 方法判断一个集合是否是另一个集合的子集：
  ```python
  s1 = {1, 2, 3}
  s2 = {1, 2, 3, 4}
  print(s1.issubset(s2))         # True
  print(s2.issuperset(s1))       # False
  ```
- union() 方法求两个集合的并集：
  ```python
  s1 = {1, 2, 3}
  s2 = {3, 4, 5}
  print(s1.union(s2))            # {1, 2, 3, 4, 5}
  ```
- intersection() 方法求两个集合的交集：
  ```python
  s1 = {1, 2, 3}
  s2 = {3, 4, 5}
  print(s1.intersection(s2))      # {3}
  ```
- difference() 方法求两个集合的差集：
  ```python
  s1 = {1, 2, 3}
  s2 = {3, 4, 5}
  print(s1.difference(s2))        # {1, 2}
  ```
- symmetric_difference() 方法求两个集合的对称差集：
  ```python
  s1 = {1, 2, 3}
  s2 = {3, 4, 5}
  print(s1.symmetric_difference(s2))   # {1, 2, 4, 5}
  ```
