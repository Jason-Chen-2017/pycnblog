3.3. 核心代码实现

假设我们有一组气象数据，我们使用假设检验方法来验证气温和降水之间的相关性，代码如下：

```python
import numpy as np
from scipy importstats as sps

# 生成模拟数据
np.random.seed(0)
n = 1000
date = np.arange(1, n+1)
temperature = np.random.normal(0, 10, n)
precipitation = np.random.normal(0, 10, n)

# 假设检验
correlation = sps.regplot(temperature, precipitation, 'o')

# 绘制散点图
plt.scatter(date, temperature, c=correlation)
plt.xlabel('Date')
plt.ylabel('Temperature (°C)')
plt.show()
```
