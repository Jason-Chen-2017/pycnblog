### 2.1. 基本概念解释

2.1.1. HTML

HTML 是一种标记语言，用于描述 Web 页面的结构和内容。它是一种客户端-server 结构的语言，客户端指的是浏览器，服务器端指的是 Web 服务器。

2.1.2. CSS

CSS 是一种样式表语言，用于描述 Web 页面的布局和样式。它是一种服务器端-客户端结构的语言，客户端指的是浏览器，服务器端指的是 Web 服务器。

2.1.3. JavaScript

JavaScript 是一种脚本语言，用于实现 Web 页面的交互效果。它可以与 HTML 和 CSS 相互配合，使得 Web 页面更加丰富和动态。

2.1.4. 前端框架

前端框架是一种开发工具，用于快速搭建 Web 页面。它包括一系列组件和工具，可以简化开发流程，提高开发效率。比较流行的前端框架有 Vue.js、React 和 Angular 等。
