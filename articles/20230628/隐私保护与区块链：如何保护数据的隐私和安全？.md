4.2. 应用实例分析

实例1：匿名交易

通过区块链技术，用户可以通过向区块链发送匿名交易信息来保护个人隐私。具体的实现方式包括：

```
// 节点A
const privateKey = await generatePrivateKey();
const to = "address";
const value = "100";

// 节点B
const publicKey = await generatePublicKey();
const amount = Buffer.from(value.toString(), "hex").toString("ascii");

// 发送交易
const tx = {
  from: "SENDER_ADDRESS",
  to: to,
  amount: amount,
  gas: "2000000",
  gasPrice: 10e9
};

const serializedTx = tx.serialize();

// 发送交易并获取交易哈希
const hash = await bs51251.createHash(serializedTx);

// 将哈希转换为字符串
const hashHex = hash.toString("hex");

// 节点B收到交易并验证
const txHash = await verifyTx(hashHex);
console.log("TX HASH:", txHash);

// 节点A收到交易并更新状态
console.log("TX status:", tx.status);
```
