### 4.3.4 智能体实现

```python
import numpy as np
from tensorflow.keras.layers import Input, Dense
from tensorflow.keras.models import Model

input_state = Input(shape=(4,))
input_action = Input(shape=(2,))

hidden = Dense(2)

output = hidden * np.tanh(input_state)

model = Model(inputs=[input_state, input_action], outputs=output)

for layer in model.layers:
    layer.trainable = False

model.compile(loss='mean_squared_error', optimizer='adam')
```

### 4.3.5 训练与测试

```python
model.fit(x=[(1, 0), (2, 0)], y=[(1, 1), (2, 1)], epochs=50, batch_size=1)

test_loss = model.evaluate(x=[(1, 0), (2, 0)], y=[(1, 1), (2, 1)], batch_size=1)[0]
print('Test loss:', test_loss)
```
