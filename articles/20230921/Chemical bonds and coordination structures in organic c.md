# 1.简介
  

目前，科技已经成为许多领域最重要的驱动力。高速计算、大数据、人工智能等一系列新兴技术正在推动着科研和产业革命。其中，化学分子的物理化学特性也十分重要。在生物医药领域，化学分子结构的分析对于预防、诊断、治疗等一系列医学应用至关重要。本文就将探讨化学分子的几种基本性质——分子角度的描述、六维分子坐标的表示方法以及多中心配体的结构和化学性质。
# 2. 基本概念术语说明
## 2.1 分子角度的描述（Averaged bond angle）

化学分子中的分子角度描述了两分子之间相互作用所形成的相互作用角度。由于化学分子可能存在一定的摩擦力，导致不同分子之间的相互作用角度呈现出正弦曲线或正切曲线的分布。因此，我们需要对角度进行平均处理，从而使得角度更加平滑。其具体方式为求出每个三角形面积的均值，并取平均值作为描述分子角度的方法。即：

$<\theta> = \frac{1}{N}\sum_{i=1}^Nt_i(\theta)$ 

其中$t_i(\theta)$ 为第 $i$ 个三角形的夹角 $\theta$ 的值，$N$ 为三角形数量。

## 2.2 六维分子坐标的表示方法(Cartesian coordinates)

六维分子坐标指的是由三个坐标轴x,y,z和三个规范方向p,q,r构成的笛卡尔坐标系下，无旋转不动点，三个坐标轴分别在单位圆半径内围绕其规范方向延申运动的位点。其表示形式为：$(x, y, z)^p (1, r^2, r^2-1)$，其中p, q, r为单位向量，且满足$p^2+q^2+r^2=1$。一般地，坐标值$(x, y, z)$ 表示无旋转不动点到原点的距离，$(r, p, q)=cos^{-1}(x, y, z)$ 表示三个规范方向的角度。

## 2.3 多中心配体的结构和化学性质

1. 多中心配体
   多中心配体（MOE）是指一种由多个金属氢原子或离子组成的复合物。它的结构可以由不同的中心点组成，具有独特的自洽和多态性。在化学领域，MOE的结构及其特性有着广泛的研究价值，能够为多种化学反应提供重要的指导。
   
2. MOE的结构
   在MOE中，每个中心由不同的原子团组成，这种特征常被称为“自聚”。MOE的结构包括了多个质点构成的多层相互作用结构。当不同中心之间的原子团发生交换时，可以形成一个新的MOE，或者改变其他原子团的位置以获得不同的分子结构。如图所示：


   其中，红色的点代表了多个质点构成的多层相互作用结构，蓝色的点代表中心。MOE中原子团之间的相互作用分为四种类型：

   - 第一类相互作用（类型I）：不同中心之间的原子团相互吸引，形成一个自由晶体。
   
   - 第二类相互作用（类型II）：不同中心之间的原子团相互排斥，形成一个硬壳。
   
   - 第三类相互作用（类型III）：不同中心之间的原子团相互吸引和排斥，形成一个稍纵向偏离的单极晶体。
   
   - 第四类相互作用（类型IV）：不同中心之间的原子团相互吸引和排斥，形成一个偏离的双极晶体。

   各类型相互作用产生的晶体都有不同的结构。MOE中的主要结构形式有：

   - 普通MOE：由单个中心和自由晶体构成，如图左上角的MOE。
   
   - 紧束缚氢分子晶体：由自由氢和特异氢之间形成的氢键形成的密封结合，称为“紧束缚氢分子晶体”，如图右下角的HOMO-LUMO 2s-2p轨道叠加形状的MOE。
   
   - 轨道交叉MOE：由两个不相容的中心和不同的多层相互作用结构构成的结构，如图中间的OH组分。
   
   MOE还可以具备有机性、反应性和化学活性。

3. MOE的化学性质
   MOE的化学性质是指物理性质和化学性质的总和。其主要特性表现在以下方面：

   - 自洽：MOE中不同元件之间存在自然的、局部的，且微观的相互作用，因而具有无序性和弹性，有助于在稳定性和性能上提高其性能。
   
     根据不同原子团之间的相互作用类型，MOE又分为三种状态：
     
     - 稳定型态：无相互作用，称为“稳定型态”。
     
     - 自由型态：除少数特殊情况外，所有元件具有相互作用，称为“自由型态”。
     
     - 混杂型态：有相互作用，但不完全自由，称为“混杂型态”。
   
   - 多态性：MOE的每个元件都可以按照其原子核和氢原子核配置的不同而形成不同的化学物质，具有不同结构、性质和功能，使它具有一种灵活、高度个性化、可塑性强、开放性强、有机性强的特点。
   
     根据MOE中各种元件的存在以及相互作用类型，MOE又分为如下几种：
     
     - 单重中心配体：只有一个中心和一个自由晶体，如HOMO和HETERO。
     
     - 双重中心配体：有两个中心和两个自由晶体，如HOMO-LUMO 和 HOMO-PIMO。
     
     - 多重中心配体：含有多个中心，自由晶体数量未知，称为“多重中心配体”。
