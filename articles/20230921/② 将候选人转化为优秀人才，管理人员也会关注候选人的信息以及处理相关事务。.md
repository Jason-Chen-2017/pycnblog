# 1.简介
  


一位IT技术经理，曾任职于某公司，现任职于另一公司，在工作中负责对人才招聘进行筛选、筛查和评估，对候选人进行面试筛选并安排面试日程。他经常参加一些社会招聘会，听取相关讲座及培训班的分享，向不同部门的人士了解企业需求，收集反馈意见。

他最近发现自己似乎一直不太重视候选人的个人信息。“我已经知道了某某人很有潜力，但每次都是到企业找人询问之后再决定是否录用。”

有一次，他收到一个应届生提交的简历，询问他对该职位的了解程度，就看到他之前参加过的一些讲座。这个应届生连自己的工作经验都没有写上，简历上的其他信息也很少。

他突然想起来之前有一个讲座，讨论了如何通过互联网广告提高销售额，其中提到了许多与其有关的信息，如市场分析报告、竞争状况、公司产品功能等。

当时，他忙着准备面试，于是把这次询问记录下来。但事后想想，他应该专注于收录候选人的信息，而非仅仅着眼于面试机会。这样才能更准确地判断候选人是否适合加入公司。

总体来说，“将候选人转化为优秀人才”是一个复杂而庞大的过程，涉及众多领域，甚至包括心理、教育、职业规划等多个方面。在组织中，很多时候管理者需要按照制定的标准，精心地筛选、筛查、选择和运用候选人。因此，要做好候选人管理，首先要注意的是及时准确地收集信息。

# 2.基本概念术语说明

候选人管理是指企业从招聘到录用的全过程，包括筛选、筛查、筛分、笔试、面试等多个环节，每个环节均需要候选人提交相关信息、阅读材料、完成流程、表达意见，以帮助企业正确判断候选人能力、经验、知识、技能和性格。

- 招聘启蒙期（筛选期）
- 招聘进行期（筛查期）
- 招聘进行期（筛分期）
- 笔试/技术测试
- 面试

候选人信息包括个人信息、履历信息、工作经历、项目经历、推荐信、毕业院校等。

候选人管理流程一般分为以下几个阶段：
1. 简历收集阶段：收集初步的信息，如姓名、联系方式、邮箱地址、年龄、职业方向、工作经验、个人描述、兴趣爱好、兴趣方向等。
2. 求职信审核阶段：根据求职要求和自身情况筛选简历。
3. 简历编辑阶段：修改简历格式、内容、词汇、错别字、语言习惯、口音等。
4. 邀请面试阶段：邀请面试官面试候选人，会给出面试题目，有针对性地问一些相关的问题。
5. 投递简历阶段：准备好简历，投递到相应的招聘网站或招聘数据库。
6. 笔试/技术测试阶段：接受面试邀请后，准备好答辩材料，进行笔试/技术测试。
7. 面试阶段：面试官会根据面试时间、候选人实际情况、个人陈述进行面试。
8. 结果确认阶段：接受面试结果，准备人力资源部发放职位。

面试过程中，候选人可能会向面试官提出问题，面试官在回答过程中会记录候选人的所有沟通和反馈信息，包括面试题目的回答、个人信息、工作经历、项目经历、兴趣爱好、志愿行为、证件、推荐信等。这些信息对企业来说是非常重要的参考依据。

常见问题与解答

1. 为什么要做候选人管理？

   候选人管理可以帮助企业过滤掉无效的候选人、扩大有效的候选人样本，提升企业的竞争力。

2. 有哪些方法可以提升候选人管理的效率？

   - 使用自动筛选工具：可以利用机器学习算法来实现简历快速筛选、过滤和排序，节省筛选的时间和资源。
   - 提升候选人反馈的效率：可以通过设置面试前或面试后反馈期限、面试官每轮交流时间和候选人反馈速度等方式，提高候选人提供信息和反馈时的主动性和积极性。
   - 优化简历筛选条件：调整简历筛选条件，如添加职位要求、提升语言水平、增加专业背景等，可以缩短筛选周期，提高效率。
   - 定期举行招聘会、技术交流活动等：定期举办招聘会、分享技术信息，可以促进候选人获取信息的渠道和方式，降低筛选门槛。

3. 怎样保障候选人信息安全？

    - 认真填写求职信：求职信必须包括个人信息、履历信息、工作经历、项目经历、推荐信、毕业院校等。
    - 使用数字签名：使用数字签名保护个人隐私，避免任何泄露。
    - 不透露候选人隐私信息：不要在面试过程中透露候选人个人信息，尤其不能泄露推荐人的联系方式。
    - 在电话中保护个人信息：如果面试中需要录音，尽量采用保密措施，例如戴着耳机、肘膝露胸等。
    - 对上司和同事保持高度的沟通：面试结束后，双方需保持正常沟通，商讨接下来的交往计划。

4. 为何在招聘中，人们往往较喜欢问个性化的问题？

   - 个性化的问题能够增加面试者的深度和广度，帮助他们展现自己的知识和能力。
   - 面试官通过问题和回答之间的关联性、间隔性、重复性，可以更容易揭示候选人的优缺点，找出不同类型的人。
   - 通过提出不同的问题，可以让面试官更加关注候选人的个性特点，通过面试策略的组合，帮助候选人找出最适合的职位。

5. 怎样选择面试官？

   - 由HRBP或行政助理负责推荐面试官。
   - 根据面试对象特点、面试要求、预算和时间等因素综合选出合适的面试官。
   - 使用一线或二线面试官，可以使得面试效果更佳。
   - 如果没有合适的面试官，也可以考虑委托其他人进行面试。
