# 1.简介
  

在图论中，“边”是一个最重要的概念。它代表了图中的连接关系，其相关信息包括节点两点间的距离、权重、是否是关键路径等等。当需要对图中的数据进行分析或处理时，通常首先需要将边转换成结构化的数组形式。

本文主要介绍一种简单有效的方法，利用邻接矩阵创建出包含所有边的数组 edges，再对其进行排序。这种方法的效率很高，同时也可用于计算图的相关性指标。


# 2. 基本概念术语说明
## 2.1 邻接矩阵

在图论中，邻接矩阵（Adjacency Matrix）是一种用于描述图中顶点间相邻关系的表格型表示法。其中，每个元素 A[i][j] 表示从顶点 i 到顶点 j 的“有向”边的数量。若 A[i][j] > 0，则说明存在一条从顶点 i 到顶点 j 的有向边；若 A[i][j] = 0，则说明不存在这样的边。下面的图示给出了一个 4x4 邻接矩阵的例子。


以上这个邻接矩阵对应着一个无向图，图中共有 16 个边。顶点 A 可以到达顶点 B 和 C，顶点 B 可以到达顶点 D，依此类推。

## 2.2 有向图和无向图

目前，图论主要研究的是连通图和非连通图。所谓连通图，就是任意两个顶点之间都存在着一条路径，不管是直线的还是弯曲的；而非连通图往往是一些孤立点组成的图。在有向图中，每条边都有一个方向；而在无向图中，一条边的方向没有特别的意义，因此边的起点和终点可以互换。
