# 1.简介
  

Clojure 是一个兼具动态和静态类型系统的函数式编程语言，它采用了 Lisp 和 Haskell 的一些特性，可用来开发大规模应用，具有出色的可移植性和高效率，特别适合开发大型分布式系统。
# 2.为什么使用 Clojure?
Clojure 与其他函数式编程语言不同，它不是一个纯粹的函数式编程语言。相反，它是一种支持面向对象、实用主义和数据驱动设计的多范式语言。在这种语言中，函数是第一等公民，数据也被视为一种值。通过这些特性，Clojure 可以编写出更容易维护的代码和更好扩展的应用程序。
同时，由于 Clojure 的动态类型系统，你可以轻松地组合函数、修改变量的值并创建复杂的数据结构。因此，如果你喜欢函数式编程，并且对动态类型和组合能力感到满意，那么 Clojure 将是你的最佳选择。
# 3.特性
- 数据透明性：Clojure 提供了全面的类型系统，允许开发者声明变量的类型，并使用编译器帮助检测错误。此外，Clojure 有内置的自动内存管理机制，使得开发者不必担心内存泄漏的问题。
- 透明并行：Clojure 提供了高度的并发性，可以让开发者轻松实现并行处理。Clojure 使用多个线程来并行执行代码，并且可以在不需要锁或显式共享变量的情况下进行通信。
- 函数式编程：Clojure 支持基于 Lambda 演算的函数式编程，并提供了丰富的高阶函数，用于构造各种数据处理模式。
- REPL：Clojure 提供了一个交互式环境，可以让用户快速尝试代码片段，或者编写完整的程序。
- 元编程：Clojure 提供了一系列强大的宏系统，可以让用户自定义自己的语言扩展，并实现任意转换。
- 平台独立性：Clojure 可运行于 JVM、Android、iOS、服务器端、浏览器端等多种平台上，可以让开发者编写跨平台的应用程序。
- 测试：Clojure 提供了一个成熟的测试框架，可以方便地进行单元测试、集成测试以及性能测试。
- 拓展性：Clojure 提供了多种拓展方式，包括宏、Java interop、编译、库依赖和插件等。
# 4.基本语法
- let 表达式：let 表达式用来绑定一组变量，并赋予它们初始值。
```clojure
(let [x 1 y 2 z (+ x y)]
  (* x z)) ; => 6
```
- if 表达式：if 表达式根据条件返回不同的结果。
```clojure
(if (even? n)
  (/ n 2)
  (- n 1)) ; 返回偶数则除以2，奇数减1。
```
- do 表达式：do 表达式是一个表达式序列，其中的每个表达式都会被求值。
```clojure
(do 
  (println "Hello World") 
  :done) ; 会先打印“Hello World”再返回 done 
```
- fn 函数：fn 是 Clojure 中定义函数的关键字，它接受参数列表、一个可选的文档字符串（用于生成 API 文档）、一个表达式体和可选的注释，并返回一个新的函数。
```clojure
(defn add [a b]
  "Add two numbers together"
  (+ a b))

; Usage: (add 1 2) ; returns 3
```
- recur 递归函数：recur 是 Clojure 中递归的关键字，它接受一个或多个参数，在每次递归调用时，将它们作为局部变量绑定到参数上，并继续执行表达式。
```clojure
(defn factorial [n]
  (if (= n 0)
    1
    (* n (recur (dec n)))))
```
