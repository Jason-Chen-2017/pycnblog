# 1.简介
  

在计算机领域里，CPU已经成为影响计算性能的关键因素之一。而英特尔作为世界上最大的芯片制造商，在CPU领域也占有一席之地。那么，英特尔到底是如何设计出如今的酷睿i系列高端处理器的呢？

今天我们就一起探讨一下英特尔酷睿i系列高端处理器的制造工艺、架构设计和芯片内核等相关知识。希望能够给读者带来更多的启发和理解。


# 2.基本概念术语说明
## 2.1.英特尔酷睿i系列产品线简介
　　英特尔的高端处理器一直以来都是芯片制造工艺革命的先驱。如今，英特尔在中低端服务器市场都已占据了核心地位，因此一直将重点放在研发具有最高性能的CPU产品。其中，英特尔高端产品线中包括“英特尔至强E-series”，“英特尔至强J-series”，“英特尔至强N-series”，“英特�至强R-series”四款系列，统称为“酷睿i系列”。


### （1）英特尔至强E-series产品线
该系列主要面向高端游戏主机、平板电脑、笔记本电脑、掌机等领域。从E-series第一代开始，英特尔已逐渐转型从单纯的主流芯片制造商转变为主导性的制造商。其E3-1200处理器采用6nm制程工艺，性能高达2.6GHz；E5-1600v3处理器采用14nm制程工艺，采用混合多线程处理模式，TDP可达22W；E5-2600v3处理器采用14nm制程工艺，采用8核16线程，TDP可达28W。另外，E-series还推出了Xeon Scalable Platform（XSP）、Arctic Lake Y-series产品线。

### （2）英特尔至强J-series产品线
该系列主要面向桌面PC、便携PC、移动平台、嵌入式设备等领域。J系列采用14nm制程工艺，整体高度集成化，在功耗、性能等指标上都超过了同级别的i系列产品。其JD-U40处理器采用14nm制程工艺，性能高达2.8GHz，比同级处理器提升了20%；其JD-V60处理器采用14nm制程工艺，采用10核12线程，性能高达3.3GHz，比同级处理器提升了20%。另外，J系列还推出了Arctic Lake Z-series产品线。

### （3）英特尔至强N-series产品线
该系列主要面向台式机服务器、集群系统、网关应用场景等领域。从N-series第三代开始，英特尔又开始主导该系列产品的研发工作。N系列采用10nm制程工艺，其第五代产品Ice Lake N300处理器在性能、功耗等方面均优于第四代产品Broadwell N200处理器。另外，N系列还推出了Cannonlake N3050产品系列。

### （4）英特尔至强R-series产品线
该系列主要面向边缘计算领域，主要用于云计算、物联网、移动互联网等领域。R系列采用14nm制程工艺，目前最热门的产品为Rocket Lake R2000处理器。该系列基于ARM架构，采用12核16线程，其性能与Intel Cooper Lake S2000相当，具有很高的算力密度。


## 2.2.英特尔酷睿i系列产品架构
### （1）主流架构

　　20世纪90年代后期，英特尔为了满足高端消费市场对高性能处理器的需求，迅速发展了CPU架构工艺革新，推出了一系列主流架构设计，包括Pentium D架构、Core2架构、Xeon架构等。它们各自代表了不同类型处理器的发展方向。


Pentium D架构是英特尔最早采用的主流微处理器架构，Pentium D处理器在30年代初推出，是单线程单指令流执行的结构，是Intel CPU首批采用400MHz及以下时钟频率的处理器。由于其结构简单、不错的性能表现，以及CPU体积小巧的特点，加上现阶段Pentium的广泛使用，使得Pentium D架构保持着良好的生态环境。

Core2架构是英特尔继Pentium D之后采用的主流64位处理器架构，是双核，单线程的结构，拥有较高的性能，并且支持高性能扩展模块。Core2架构推出的两个产品系列分别是Sandy Bridge和Ivy Bridge。

Xeon架构是英特尔在Core2架构基础上推出的更进一步的64位处理器架构，是64核，支持多线程并行计算，具有较高的性能。它的两个产品系列分别是Haswell和Broadwell。

### （2）近年来的高端架构

　　2017年，英特尔宣布即将发布英特尔至强E-series处理器，除了使用“酷睿”命名外，E-series与以往所有的CPU架构一样，也是采用14nm制程工艺。E-series的设计目标是追求高性能、高性价比、高性价比的解决方案。E-series将包括旗舰级别的处理器、常规用途服务器、个人电脑、游戏主机、平板电脑等各种类型产品。E-series处理器虽然采用了14nm制程工艺，但仍然是在主流的64位结构基础上的改进升级，不断优化产品的性能、价格以及体积。

目前，酷睿i系列的架构主要包括两种：第一种是旧版的“骁龙系列”，另一种则是新的“旗舰系列”。

## 2.3.骁龙系列架构介绍

　　“骁龙”系列也就是英特尔从Pentium D到酷睿手机、酷睿笔记本、酷睿奥迪的CPU架构序列。骁龙系列是英特尔自2007年至2013年推出的处理器系列，它继承了之前的“龙”系列的架构设计，在频率、功耗、内存、接口、散热、内部组成等方面都做了升级和优化。它的处理器架构如下图所示：


从图中可以看出，骁龙系列共分为四个产品系列：

1. 芝麻系列，主要用来制作以通用性为目标的移动产品，例如iPhone X、HTC U11、三星 Galaxy Note、魅族 MX5等。

2. 麒麟系列，主要用来制作以娱乐为主的移动产品，例如OPPO R11 Plus、荣耀 V9、小米 Mix2、华为 Mate Xs等。

3. 暮光系列，主要用来制作以前置摄像头为目标的高端手机产品，例如苹果 iPhone XS Max、OPPO A7、Vivo X21、三星 Galaxy Note9等。

4. 天玑系列，主要用来制作超级手机产品，例如红米 Note5、一加 5T、华为 nova 5 Plus等。

骁龙系列的处理器主要由三种不同的架构组成：

#### (1)骁龙40系列

骁龙40系列是第一个支持高端市场应用的商用级芯片，其架构兼顾性能和尺寸，是国产化或半封闭发射平台的首选处理器。

其主要特性：

- 支持处理64位指令集，具有高性能。
- 使用四核四线程设计，每一颗核心能够处理高达4.25GHz的基准频率。
- 提供基于ARM的高效计算，能够支持高性能的应用程序。
- 拥有高达64GB DDR4内存容量。
- 提供业界领先的性能、功耗、内存等指标。

#### (2)骁龙60系列

骁龙60系列是骁龙40系列的升级版本，采用四核八线程设计，配备了三级缓存和一个共享存储器(SSM)，具备支持64bit指令集的能力。同时，其具有低功耗、低噪声、高可靠性等优秀特性。

其主要特性：

- 支持处理64位指令集，具有极高性能。
- 每一颗核心能够处理高达4.7GHz的基准频率。
- 为具有负载任务的设备提供了超高性能的CPU，适合高性能服务器、多媒体渲染、网络传输等领域。
- 拥有64GB DDR4内存容量，能够满足大量数据处理、分析计算需求。
- 在全球多个国家获得了认证。

#### (3)骁龙70系

　　骁龙70系列主要用于高端市场手机、平板电脑、桌面电脑等设备。其最新一代的处理器基于A57架构，采用八核八线程设计，拥有独特的5+4超标性能，也能提供长达十年的使用寿命。

其主要特性：

- 支持处理64位指令集，具有极高性能。
- 每一颗核心能够处理高达5.1GHz的基准频率。
- 有两个“大核”，每个“大核”配备三个“小核”。
- 性能突破，能够实现复杂计算任务的加速。
- 拥有64GB DDR4内存容量，能够满足大量数据处理、分析计算需求。
- 利用超低功耗、超低噪声、超高性能等特性，可靠性高、续航能力长。

## 2.4.旗舰系列架构介绍

第二代的“旗舰”系列主要采用了英特尔的新一代架构设计——旗舰架构，目的是加强自身竞争力，扩大规模。


　　在酷睿i系列的第三代产品中，有两款具有典型的旗舰型号，即酷睿i5-10300H和酷睿i7-10710U，这两款处理器在架构设计上与其他酷睿i系列产品有所不同。

### 一、酷睿i5-10300H架构


　　酷睿i5-10300H采用10nm制程工艺，是酷睿i5系列的标志性产品。其主要特性如下：

- 频率： 1.8GHz至4.1GHz。
- 核数： 4核
- 性能： 2.5GHz Turbo Boost / 3.6GHz TDP
- 缓存： L3 Cache 8MB
- 片上内存： 8GB DDR4
- 带宽： 25Gbps
- 浮点运算单元： 28x INT8 + 4x INT16 + 2x INT32 + 2x INT64 + 4x FP32 + 2x FP64
- 集成显卡： Intel® Iris™ Pro Graphics 6200 (2048 CUDA Cores)
- USB接口： x2，Type-C
- HDMI输出接口： 1.4 视频信号输出
- Wi-Fi/Bluetooth： 802.11ac 2x2 MIMO
- GPS： GPS+GLONASS双卫星定位技术
- 其他： 可编程热插拔，兼容NTFS和ExFAT文件系统
- 尺寸： 机身大小：243mm×105mm，屏幕尺寸：14英寸、全高清
- 产业链： 先进制造 -> 封装测试 -> 发布生产 -> 售后服务。

### 二、酷睿i7-10710U架构

　　酷睿i7-10710U采用10nm制程工艺，主打超高性能的多核处理器，具有对标英伟达GTX 1070的性能，搭载在英特尔的Z390至强处理器平台上。其主要特性如下：

- 频率： 2.3GHz至4.1GHz。
- 核数： 8核
- 性能： 3.1GHz Turbo Boost / 4.2GHz TDP
- 缓存： L3 Cache 16MB
- 片上内存： 16GB DDR4
- 带宽： 42.7Gbps
- 浮点运算单元： 28x INT8 + 4x INT16 + 2x INT32 + 2x INT64 + 4x FP32 + 2x FP64
- 集成显卡： Intel® Iris™ Xe Graphics (2048 CUDA Cores)
- USB接口： x4，Type-C
- HDMI输出接口： 1.4 视频信号输出
- Wi-Fi/Bluetooth： 802.11ac 4x2 MIMO
- GPS： GPS+GLONASS双卫星定位技术
- 其他： 可编程热插拔，兼容NTFS和ExFAT文件系统
- 尺寸： 机身大小：243mm×105mm，屏幕尺寸：14英寸、全高清
- 产业链： 先进制造 -> 封装测试 -> 发布生产 -> 售后服务。

酷睿i5-10300H和酷睿i7-10710U是酷睿i系列的标志性产品，这两款处理器架构都是以华为的麒麟9000、高通的Kirin 990以及英伟达的Ampere架构为蓝本创新，在性能、功耗、接口、内存、处理器数目、显示性能等方面都都有所改进。其后续产品线中还有一款酷睿i7-10710H系列，使用的也是华为的麒麟9000架构，只是性能略低于酷睿i7-10710U。
