# 2.	术语解释及联系
## 1.什么是“在线课程”？
在线课堂（Online Courses）或称为网络课堂，是指通过互联网（包括电脑、手机等各种网络终端设备）进行远程授课、学习的一种学习方式。在线课程主要分为：网络直播课堂、网络研讨课、网络视频课堂、网络微课堂等。

## 2.	网络直播课堂
网络直播课堂又称为网络直播授课，是指通过网络平台向学习者提供高清直播视屏、语音，老师可以直接在线跟踪、参与，老师可以随时给予学生提问、回答疑问。

## 3.	网络研讨课
网络研讨课是利用网络平台进行远程教学的方法之一。网络研讨课一般分为网络小组讨论、网络小结分享、网络试题练习等。

## 4.	网络视频课堂
网络视频课堂是指通过网络平台上制作视频课程，并分发到众多用户的网络视频网站上。这些网络视频课程需要具备较好的编辑水平和很强的教学效果才能吸引学习者的注意力。

## 5.	网络微课堂
网络微课堂是利用互联网进行知识的传播，适合于对网络有所了解、对某些技能感兴趣但不具备专业训练经历的群体。网络微课堂通常提供短期内，轻松掌握一些基础知识，之后再参加正式的网络教学计划。

# 3.	自制在线课程的核心功能及其设计流程

## 1.	为什么要做自制学习课程？
为了应对学习压力，当今社会提供了许多学习方式，包括书籍、视频、直播等。但这些学习方式毕竟不能完全满足需求。现代人对接受新鲜事物的渴望程度远远超过对一定的培训。因此，越来越多的人选择了自制学习。

## 2.	自制在线课程的核心功能
1.	自主学习：学生可以根据自己的兴趣、能力、条件自由学习。

2.	随时随地学习：学生可以随时随地进入学习环境学习，没有学习的习惯也可以进行自主学习。

3.	免费获取：学生可以在任何时间、任何地方免费获取学习资源。

4.	课程组织：学生可以按照自己的学习进度安排学习顺序，每个单元的学习时间也可以根据自己的需要设置。

5.	评估工具：学生可以根据自己的学习情况进行有效的反馈。

6.	学习进度追踪：学生可以查看自己的学习进度、成绩，以及教师对他/她的评价。

## 3.	设计自制学习课程的流程
1.	选定学习主题：首先确定需要教授的主题、范围、难易程度。

2.	选择适当的授课方式：选择最适合学生学习的授课方式，比如视频、直播、问答等。

3.	确定学习目标：在充分听过课程后，学生应该能够理解与掌握相关的内容，并完成相应的测试题目。

4.	制作素材：教师需要制作课程的素材，比如课件、习题、答案等。

5.	推广课程：除了鼓励学生进行自主学习外，还可以通过不同渠道、媒介对课程进行宣传、推广。

6.	收集反馈：收集学生的反馈信息，分析学生的学习情况，改进授课方式，提升教学效果。

# 4.	如何通过免费的形式进行销售并收集反馈

## 1.	什么是“免费的形式”？
在很多国家，许多初创公司或个人就通过免费的形式开始在线课程的尝试。免费的定义往往比较模糊，可以泛指价格低廉、使用条件灵活的付费方式。这里的“免费”通常指的是发布课程的过程免费。像Coursera、Edx这样的在线课程平台，很多课程都是免费获得的，也就是说，我们不需要支付任何费用就可以获得这些课程。

## 2.	创建课程的基本要求
要想创建成功的课程，我们需要遵循以下几个基本要求：

1.	易懂易学：即使是初学者也应该能轻松理解课程内容。

2.	有针对性：需要关注学生的个性化需求，根据学生的特点制作不同类型的课程。

3.	不重复造轮子：不要重复发明轮子，搜索周边同类的产品。

4.	体系完整：对学科知识进行系统整理、归纳，同时善于使用图表、图片、动画等辅助表达形式。

5.	授课方法灵活：课程应以一种专业的角度进行讲授，逐步深入细节，帮助学生理解知识。

6.	结构清晰：课程内容应有序、有条理、层次分明。

## 3.	自主学习的重要性
自主学习在学习过程中拥有更大的弹性、更大的适应性。学生可以在课堂上随时找到自己感兴趣的东西，可以做到“脱离控制”。这种学习模式下，学生不会被人牵着鼻子走，可以边学习边聆听讲师的讲解。通过这种方式，学生可以不受干扰地集中精神，专注学习。

## 4.	如何收集反馈？
收集反馈的过程一般分为四个步骤：

1.	邀请回答问题：在开场白、课前准备阶段，可以邀请学生提出问题，引导学生思考。

2.	提问：课间提问是在课程结束后进行，以获得学生的反馈。

3.	反馈收集：在每章课程结束时，可以收集学生的反馈，不仅有助于提高教学效果，也能发现课程的缺陷与不足。

4.	评估反馈：对收集到的反馈进行综合评估，找出学生的差距。

通过以上四个步骤，我们可以收集到学生对课程的反馈。对于收集到的反馈，我们可以分析其原因，调整授课方式，提升教学效果。
