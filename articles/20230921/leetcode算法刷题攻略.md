# 1.简介
  

## 1.1 什么是LeetCode？

LeetCode是一个面试、学习编程技巧的网站。用户可以在这里练习算法、提升编程能力。由于其具有平台化的题库、题目难度不断升级、强大的交互性、丰富的算法训练环境等特点，也被越来越多的人用于拓宽编程经验、锻炼编程水平、测试自己的编程能力。除此之外，LeetCode还是一个真正意义上的社区，在这里可以找到志同道合的伙伴们一起学习，分享自己的算法思路和经验。

## 1.2 为什么要做LeetCode刷题？

LeetCode是一个成长中的社区，现在已经成为全球最大的技术学习平台。作为一名优秀的程序员或工程师，掌握好算法、数据结构的基础知识非常重要。但是，在工作中或面试中，很少会出现如此频繁的算法面试。为什么？因为工作与生活经常割裂。工作中常用到的编程语言、框架、工具等都是可以直接拿来用的，而算法、数据结构却需要通过刷题的方式才能熟练掌握。这就是为什么很多公司都在为应届生量身定制算法面试问题，帮助他们更快地进入到软件开发的行列。所以，做好LeetCode刷题攻略是一件很有必要且值得投入的事情。

## 1.3 怎么刷LeetCode？

LeetCode刷题的方法主要有以下四种：

1.每日一刷（推荐）：每天早上定时打开LeetCode官方网站，先用力阅读所有题目的描述，然后选择自己感兴趣的题目进行模拟，争取做出一份满意的提交。这样做既能获取新题，又保证了自己每日都在进步。

2.专项训练（可选）：有些题型比较难，或者需要有一定应用场景才能理解，这种情况下就要结合专项训练。例如，如果要学习并实现二叉树相关算法，那么就需要对二叉树有较为深刻的理解。因此，建议采用专项训练的方式，按照课程视频教材一步一步实践即可。

3.剑指offer系列（可选）：这个系列的题目一般难度介于Easy和Medium之间，适合用来测试候选人对计算机科学、数据结构、算法等各方面的能力。通过这类题目，可以确定候选人是否真正掌握这些知识点。当然，也可以作为其它刷题方式中的补充。

4.高频词汇（可选）：在刷完每日一刷、专项训练和剑指offer后，还有一些高频词汇或者特定算法类型的题目可以再次尝试。这样的题目有助于加深候选人的知识体系，增强对算法设计和分析的理解。

除了以上四种刷题方法外，还有一些策略或技巧可以进一步提升刷题效率，例如：

1.利用搜索引擎：LeetCode提供了多个搜索功能，可以根据题号、题目标题、标签等关键字快速查找所需题目。同时，也可加入LeetCode Cookie插件，自动识别登录状态，自动保存用户名、密码信息，避免重复输入。

2.抓紧热门：LeetCode上的题目数量非常庞大，为了确保能够胜任面试，不妨抓紧热门的题目来刷。热门的定义可以视情况而定，比如按每日一刷刷题量来衡量，前几天的题目就会变得十分热门。

3.适时跳槽：从最初的简单题目逐渐升级到困难题目，既能巩固基础，又能增强学习能力。同时，也应该适时跳槽，更换工作岗位来提升能力。
