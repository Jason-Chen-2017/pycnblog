# 1.简介
  

> 在越来越多的企业和研究人员都开始关注机器学习的最新进展，同时，越来越多的人们也开始着迷于这项技术，并不断寻找能够解决复杂问题的方法。而作为一名程序员或机器学习从业者，我们又该如何快速入门？这本书就是为了帮助这些人提供帮助。在本书中，作者通过精心设计的独具特色的“从零”系列教程，循序渐进地带领读者完成从基础知识到实际项目实践的全面学习。希望能够帮助读者达成以下目标：
1. 理解机器学习的基本概念、术语及其相关定义；
2. 掌握机器学习的分类、回归、聚类、降维等算法的原理和应用方法；
3. 了解深度学习、强化学习、神经网络、遗传算法、遗传编程等前沿领域的最新技术；
4. 通过动手实践的方式巩固学习到的知识，获得实际应用经验；
5. 加深对机器学习的认识和兴趣，培养数据分析、产品设计、系统工程、算法开发等方面的能力。

此外，本书还包括了一些实用工具和资源，如jupyter notebook配套环境，数据集下载地址，工具推荐，与专业人士交流，书籍购买链接，论文参考等。

本书适合计算机专业人员（非机器学习专业）阅读，涉及机器学习的基础知识和相关算法，并且具有较好的图文并茂的写作风格。阅读本书需要阅读者具有良好的英语水平，有一定编程经验或熟悉python或matlab等编程语言。阅读后，读者应该能够掌握机器学习的基础知识，能够解决实际问题，拥有机器学习的实际经验。
