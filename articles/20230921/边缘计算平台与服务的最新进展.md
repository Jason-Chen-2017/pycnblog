# 1.简介
  

## 什么是边缘计算？
“边缘计算”（Edge Computing）是一个新的信息技术领域，它借助于“物联网”、“移动宽带”等新型网络技术，将服务器、网络、存储、应用部署到距离用户较近的位置上。与传统的数据中心架构相比，边缘计算的主要特点是将数据处理能力分布到距离用户较近的地方，并通过低延迟、高带宽的网络连接，实现数据的实时处理、分析等功能。因此，边缘计算是一种高效、低延迟、低成本的计算方式。目前国内外已经有多个边缘计算平台和服务提供商，它们正在不断完善自身产品和服务，以满足各种边缘计算场景下的需求。

## 为什么要做边缘计算？
边缘计算可以用于以下场景：
- 数据安全：边缘计算可以在边缘设备中实施数据加密、过滤和传输过程，保障数据在传输过程中不被窃取或篡改。同时，边缘计算还可以提供数据质量保证、数据隐私保护和数据可用性监测等服务。
- 智能视频会议：边缘计算可以提供可靠且高速的视频流传输服务，同时也可以对摄像头数据进行实时处理，提供更加精准的画面效果。这样就可以实现远程会议中的交互性和高清视频。
- 游戏云渲染：边缘计算可以在玩家周围实时生成环境、角色、道具模型，并渲染出动画效果，在游戏中表现出真实感。这样就可以将云端渲染服务与网游结合起来，让玩家获得更加真实、流畅的游戏体验。
- 工业自动化：边缘计算可以用于监控生产线上各个节点的运行状况，提供实时的预警信息和故障定位，从而提升整个生产流程的效率。
- 工业强制力：边缘计算可以在工厂中部署集装箱监控系统，实时监测货运车辆的行驶路径和速度，分析货品的流通情况，甚至可以协助生产者进行产品防溢出检测。
- 环境监测：边缘计算可以利用无人机等传感器捕捉环境中的污染源，并根据污染的程度，对污染源进行实时监测和排除。这样可以帮助公共资源管理部门及时发现恶劣环境，保障人民群众的健康安全。

除了上述这些应用场景之外，边缘计算还有很多其它应用场景，如视频剪辑、在线教育、金融支付、AR/VR虚拟现实、数字孪生、地图导航等。

## 为什么要选择边缘计算平台？
边缘计算平台和服务提供商通常会提供多个产品和服务选项，包括开源软件、商用硬件、基于容器技术的云服务、以及专门针对特定行业或市场的定制解决方案。这里，我们重点关注一下开源软件这一选项。

首先，开源软件非常便宜，而且开源社区活跃，涵盖了不同背景的人参与开发，无论是小公司还是大企业都能快速入手。其次，开源软件能够兼容不同的硬件平台，因此你可以灵活选择部署在哪种类型的设备上。最后，由于开源软件的自由开放，你可以随时升级到最新的版本，随时分享你的见解和经验。总之，如果能找到适合自己的开源软件，就能很容易地搭建起边缘计算平台。

## EdgeX Foundry 是什么？
EdgeX Foundry 是 Linux Foundation 下的一个开源项目，主要致力于构建边缘计算框架。它的主要组件有：
- Core Data：一个面向对象的数据库，它封装了各种系统需要用到的数据对象，比如设备、事件、读写属性、命令等；
- Device Service：一个设备控制层，它负责管理边缘设备和提供各种接口，供应用程序调用；
- App Functions SDK：一个开发套件，它提供了多种编程语言的函数库，方便应用程序开发人员编写应用逻辑；
- Integrations and Devices：一系列组件，包括设备接入层、协议转换器、智能过滤器、消息路由器等，这些组件的作用是连接设备、转换协议、过滤数据、路由消息等；
- Supporting Services：一系列支持服务，包括元数据存储、配置管理、日志记录、访问控制、安全、密钥管理、熔断机制、服务发现、REST API 等；
- Applications：一个面向对象的框架，它将以上组件组合成完整的边缘计算应用，包括设备服务注册、配置、生命周期管理、数据收发、应用逻辑等。

基于此，我们可以看出，EdgeX Foundry 提供了一个完整的边缘计算框架，包括核心服务、应用函数库、设备接入层等，这些组件都十分重要。例如，设备服务可以用来注册各种设备，设备接入层可以用来转换协议，App Functions SDK 可以用来编写应用逻辑。但是，由于该框架还处于早期阶段，很多功能还没有完全完成，因此作为边缘计算平台和服务提供商，仍然需要持续投入时间和资源，不断完善和优化相关组件。
