# 1.简介
  

“为什么聪明人都喜欢建筑，而大部分人却对电影情有独钟？”这个题目并不难理解。一般来说，如果一个人对某些事物有兴趣，就会选择热爱它；然而，很多时候，也存在着这样一种情况——他们因为自己的聪明才智和见识而成为“精英”。这种现象在我们身边并不少见，例如，一些科技领域里的大牛，一些具有创造力的人物，还有一些社会活动家、企业家等。所以，了解这些“精英”背后的故事，对于我们理解这个世界和自我塑造可能非常有帮助。
正如我们所说的，这个问题可以涉及许多方面。本文将专注于电影和建筑两件事物的关系。如果你不是建筑和电影的专业人士，那么你只需要知道两个词汇“建筑”和“电影”，就可以读懂本文了。如果您还不是很确定，建议您先去百度一下这两个词的定义。当然，如果你确实对这两个主题感兴趣的话，那么恭喜你，你可以继续阅读下去。

# 2.基本概念术语说明
## 2.1.电影
电影（Movie），通常指一部制作过程完整且具有观赏性的一组静止图像或视频片段。在中国，“电影”一词多指用声音、色彩和影像捕捉到的画面构成的影像工作者进行剪辑、拼接、编辑、播放的多幕段电视连续剧形式的记录片。电影是欣赏力最高、观看时间最长的综合艺术作品之一。

## 2.2.建筑
建筑（Architecture）：建筑学（Architectural Studies）是以各种材料和结构为基础，借助工程学、艺术理论、历史学、物理学等学科的综合研究，探索人类活动如何影响周围环境及其变化、人类生活的主要方式及其可能的结果，以及如何通过构建技术手段改善人类居住环境，促进人类文明进步的领域。建筑学知识包括设计、施工、管理、土木工程、结构工程、城市规划、景观设计、人文地理等多个专业领域。

1.人类的行为习惯。作为一种复杂动物，人类的行为习惯会影响我们的心理状态。因此，“电影”情节的发生，与人的行为习惯密切相关。电影情节大多呈现出一种与人类预期不同的面貌，令人着迷和神魂颠倒。

2.人的想象力。人类的想象力不仅是我们极其擅长的能力之一，也是它的弱点。由于人类的视觉、听觉等器官的缺陷和限制，我们只能看到摄像机镜头下的画面，或者听到声音里传来的声调。于是，电影往往只能在强烈的画面音效的刺激下才能被拍摄出来。

3.人类的视觉系统。人类视觉系统会根据周遭环境的照射条件以及人的肢体运动，调整图像的色彩饱和度、亮度等属性。也就是说，不同光线和角度下的灯光组合，都会导致不同图像的显示效果。因此，对观众来说，只有一种视角是无法拍摄出符合审美需求的电影的。

4.人类创造力。作为一名艺术家，一个优秀的创作者需要用自己的手段来创造作品。同时，创意也是人类天生的好奇心和创造力的源泉。电影的制作者需要高度的逻辑思维能力、艺术创新能力和电脑绘画技术。

5.建筑的功能性。在建筑工程中，建筑物的功能直接影响人们的生活质量。从功能上看，电影院、咖啡馆、餐厅等建筑都是为了满足人们特定需求而设立的。这些建筑往往具有复杂的设计和工程技术，能够使得居民生活更加便利和舒适。

6.电影制作过程中的设计与艺术。在电影制作过程中，艺术家不仅需要考虑剧情和意境，还要充分考虑电影院空间的布局，以及包括观众在内的听众的情绪接受程度、角色的情感连接等因素。

7.历史因素。人类活动的改变，引起了电影的产生。同时，电影对人类社会发展的贡献也不可忽视。历史的演变使得人类社会的发展趋向逐渐前行，给电影业带来巨大的发展潜力。

8.演员的表演技巧。相信没有哪个电影的制作者能够完全依赖一种固定模式的表演技巧。不管是星级主角、经典演员，还是当代经典，每一部电影都应该尽可能保持青春气息、内敛感性、多元化表现。

9.审美疲劳。即使是对电影感到愉悦，也不一定能够给所有人带来快乐。受到审美疲劳影响的人，往往会更关注细节、关注呈现内容，而忽略了整体观赏体验。因此，很多人的电影观感更偏重小品，而非真正的代表电影时刻的大戏。

# 5.未来发展趋势与挑战
1.科技创新。由于互联网、手机、电子设备的普及，各项技术的革命已经带来了无限的创新的可能。比如，VR、AR、机器学习、区块链技术等技术正在对电影制作领域产生深远影响。

2.社交媒体上的热度。随着社交媒体平台的飞速发展，越来越多的人开始把注意力放在视频直播上，这就再次刺激了人们的创作欲望。当代电影人总是热衷于在社交平台分享自己的作品。但目前，电影制作者在社交平台上的曝光率较低，制作费用却一直在上升。

3.职业化、市场化。随着电影产业的迅速发展，越来越多的从业人员将从事电影制作。电影产业需要更加高端、精英化、市场化的玩法。另外，电影业的投资也逐渐增加，制作公司也应当树立起坚实的财务基础。

4.消费升级。现在，电影已经成为每个家庭、每个族群都不可或缺的一款日常生活商品。电影消费不仅仅局限于影院购票消费，也广泛涵盖在家的电影消费。消费升级将拉近观众与电影院的距离，使电影业成为一个独立实体。
