# 2. 基本概念术语说明

## 2.1 堆栈

堆栈（Stack）又称为栈，是一种特殊的线性表结构，其中的元素按照先进后出(First In Last Out)的顺序存储。栈具有以下几个主要操作:

1. Push(入栈): 添加一个新元素到栈顶，即放在栈顶位置上。
2. Pop(出栈): 从栈顶删除一个元素，使栈底元素成为新的栈顶。
3. Peek(观察栈顶元素): 返回栈顶元素但不删除它。
4. IsEmpty(判断栈是否为空): 检测栈是否为空。
5. Size(返回栈的大小): 返回栈中元素个数。

如下图所示，堆栈通常用一个数组或者链表实现。

## 2.2 队列

队列（Queue）又称为队列，是FIFO（先进先出）的线性表结构。队列具有以下几个主要操作:

1. Enqueue(入队): 将一个新元素加入队列的末尾，等待被处理。
2. Dequeue(出队): 删除队列头部的一个元素，并将该元素返回给调用者。
3. Front(查看队首元素): 查看队列头部的一个元素，但不删除它。
4. Rear(查看队尾元素): 查看队列最后添加的元素，但不删除它。
5. IsEmpty(判断队列是否为空): 检查队列是否为空。
6. Size(获取队列大小): 获取当前队列的元素个数。
