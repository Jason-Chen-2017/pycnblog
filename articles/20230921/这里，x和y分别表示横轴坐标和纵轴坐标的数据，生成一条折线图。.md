# 1.简介
  

## 1.1 折线图的概念及特点
折线图（又称曲线图、线性图）是一个用折线连接多组数据点形成的统计图，它呈现出变量随着分类变量变化的趋势，因此也被称为散点图或气泡图。 折线图由一条或者多条曲线组成，通常用于显示两个或多个变量随时间的关系。 折线图的优点在于直观易懂，有利于数据的观察、分析和预测。它的缺点在于不适合显示太多的数据点。因此，当数据的数量较多时，应选择其他更适合的数据可视化形式。
## 1.2 基础知识
### 1.2.1 折线图的应用场景
1. 不同年龄段消费者在购买商品过程中价值观的变化。

2. 某商品某特征值的变动对销售额的影响。

3. 年龄、地区等因素对电影票房的影响。

4. 不同职业群体工作年限对薪水的影响。

### 1.2.2 数据类型
折线图可以用作一种动态的展示形式。所要呈现的数据在时间维度上是连续的，但每一个时间点所关注的问题往往不是瞬间发生的事情，而是在一段时间内某个特定事件的结果。比如，经济指标如产出的增长率，社会指标如人口的增加率等等都可以作为折线图的一种数据形式。另外，折线图也可以呈现一系列的离散的事件，如发布会的参加情况。

## 1.3 折线图绘制方法
### 1.3.1 折线图的三个层次
折线图主要分为三个层次：

1. 折线图顶部：坐标轴。坐标轴是将表现数据的空间与物理世界进行关联的重要工具。坐标轴的位置可以决定数据的横轴范围、纵轴范围、角度等，从而影响数据的形状和密度。

2. 折线图下方：数据标签。数据标签是在折线图上给出具体数据信息的地方。

3. 折线图中间：折线。折线是用来连接每一个数据点的线条，它反映的是数据的变化趋势。当数据点多于三到四个时，折线图可能出现折线交叉、凹陷、凸起等复杂的现象，这种现象往往意味着数据呈现出聚集性或分离性的特性。

### 1.3.2 如何制作折线图？
1. 数据准备：折线图的数据需要根据一定的数据格式组织好。一般情况下，折线图的数据具有以下几个特点：
   - 有确定的横坐标或时间顺序。
   - 有确定的纵坐标或数值。

2. 设计细节：设计折线图的大小、颜色、字体、透明度、边框、空间、网格等，以达到完美的效果。

3. 编码实现：用编程语言或工具将数据画出来，比如Python、Matplotlib等。
