# 2.相关知识点回顾

## 2.1 深度学习简介
深度学习（Deep Learning）是一种人工智能技术，是指多层次神经网络通过训练大量数据来解决复杂任务，是实现人工智能的一种方式之一。深度学习最早由多伦多大学教授Hinton和他的同事Russakovsky于2006年提出，其后随着计算能力的增强、数据量的增加以及算法的进步，深度学习也越来越火爆。目前，深度学习已经成为许多领域的标杆技术，如图像识别、自然语言处理、物体检测、医疗诊断、推荐系统等。

## 2.2 什么是卷积神经网络
卷积神经网络（Convolutional Neural Networks，CNN）是深度学习中重要的模型之一。它是一个基于生物神经科学的理论基础上的深度学习模型，是人工神经网络（Artificial Neural Networks，ANN）的扩展。与传统的ANN不同的是，CNN主要关注局部特征，而不是全局特征。

CNN模型由多个卷积层（Convolution Layers）和池化层（Pooling Layer）组成，前者用于提取局部特征，后者用于降低特征维度并减少过拟合风险。具体结构如下图所示：

1. 输入层：输入图像的像素点。

2. 卷积层：包括卷积操作和激活函数。卷积层对输入图像进行卷积操作，提取图像的局部特征。

3. 激活函数：激活函数通常是ReLU，它是一个非线性函数，能够有效地防止梯度消失或爆炸。

4. 池化层：池化层一般采用最大池化或者平均池化。最大池化的目的是为了减小输出大小，提取局部区域的最大响应值；平均池化的目的是为了减少参数数量，提取局置区域的平均响应值。

5. 全连接层：全连接层对网络的输出进行预测，输出层的数量依赖于任务类型。

6. 损失函数：损失函数用来衡量模型预测结果与真实标签之间的差距，用于模型的优化和反向传播。

## 2.3 什么是循环神经网络
循环神经网络（Recurrent Neural Network，RNN）是另一种深度学习模型。它可以记住之前的信息，并根据这个信息对当前情况做出预测。它首先通过输入层接收输入数据，然后通过隐藏层进行计算，最后通过输出层输出预测结果。具体结构如下图所示：

1. 输入层：接收输入序列的向量表示。

2. 循环层：循环层用于保存和更新状态信息。它会遍历输入序列的每一个元素，并为每个元素分配特定的状态值。

3. 激活函数：激活函数通常是tanh或relu，用来确保神经元输出的值在一定范围内，从而避免梯度消失或爆炸。

4. 输出层：输出层用于对序列进行预测。它与传统的多层感知机类似，只是它的权重和偏置是在循环层中的，并且没有激活函数。

5. 损失函数：损失函数用于衡量模型预测结果与真实标签之间的差距。
