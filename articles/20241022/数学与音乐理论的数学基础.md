**5.1 复音程的概念**

复音程是指跨越一个或多个八度的音程。它们与普通音程不同，普通音程只涉及同一八度内的两个音符之间的距离。复音程可以分为以下几类：

- **复增五度**：跨越一个八度的增五度。
- **复减四度**：跨越一个八度的减四度。

**5.1.1 复音程的几何解释**

复音程的几何解释可以通过将其分解为几个普通音程来实现。例如，一个复完全四度可以分解为两个完全三度，每个完全三度跨越半个八度。这种几何解释不仅帮助我们理解复音程的结构，还可以用于分析和解释复音程在音乐中的效果。

**A.4 代码示例与音乐编程实例**

以下是一些简单的代码示例，展示了如何使用Python和MATLAB进行音乐编程：

**Python示例**：
```python
import numpy as np
import matplotlib.pyplot as plt

# 定义采样率
sr = 44100
# 定义频率
f = 440
# 定义时长
duration = 5
# 生成正弦波信号
t = np.linspace(0, duration, sr * duration)
x = np.sin(2 * np.pi * f * t)
# 绘制波形
plt.plot(t, x)
plt.xlabel('Time (s)')
plt.ylabel('Amplitude')
plt.title('Sine Wave')
plt.show()
```
