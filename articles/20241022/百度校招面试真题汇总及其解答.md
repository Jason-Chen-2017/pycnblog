#### 第1章：算法与数据结构基础

##### 1.1 基础算法概述

算法是计算机科学的核心，是解决特定问题的系统方法。在面试中，基础算法的理解和应用往往是最重要的。本节主要介绍几种常见的算法复杂度以及基础数据结构。

**1.1.1 算法复杂度分析**

算法复杂度分为时间复杂度和空间复杂度。时间复杂度衡量算法执行时间与输入规模之间的关系，通常使用大O符号表示。例如：

- $O(n)$：线性时间复杂度，算法执行时间与输入规模成正比。
- $O(n\log n)$：对数时间复杂度，常见于排序和搜索算法。
- $O(n^2)$：平方时间复杂度，常见于双重循环算法。

**1.1.2 数据结构基础**

数据结构是算法的基础，决定了算法的性能。以下是一些常见的数据结构：

- **链表**：由节点组成，每个节点包含数据和指向下一个节点的指针。适用于插入和删除频繁的场景。
- **栈**：后进先出（LIFO）的数据结构，适用于解决递归问题。
- **队列**：先进先出（FIFO）的数据结构，适用于解决同步问题。
- **哈希表**：基于键值对的存储结构，通过哈希函数将键映射到值。适用于快速查找。

##### 1.2 算法实践题

**1.2.1 简单算法题解**

以下是一些简单的算法题：

- **排序算法**：冒泡排序、选择排序、插入排序等。
  ```c
  void bubbleSort(int arr[], int n) {
      for (int i = 0; i < n - 1; i++) {
          for (int j = 0; j < n - i - 1; j++) {
              if (arr[j] > arr[j + 1]) {
                  int temp = arr[j];
                  arr[j] = arr[j + 1];
                  arr[j + 1] = temp;
              }
          }
      }
  }
  ```

- **查找算法**：二分查找、线性查找等。
  ```c
  int binarySearch(int arr[], int l, int r, int x) {
      while (l <= r) {
          int m = l + (r - l) / 2;
          if (arr[m] == x)
              return m;
          if (arr[m] < x)
              l = m + 1;
          else
              r = m - 1;
      }
      return -1;
  }
  ```

- **双指针算法**：适用于解决数组中的问题，如寻找两个数的和。
  ```python
  def twoSum(nums, target):
      left, right = 0, len(nums) - 1
      while left < right:
          sum = nums[left] + nums[right]
          if sum == target:
              return [left, right]
          elif sum < target:
              left += 1
          else:
              right -= 1
  ```

**1.2.2 中级算法题解**

以下是一些中级的算法题：

- **二分查找**：在有序数组中查找特定元素。
- **图算法**：解决图论相关的问题，如最短路径、拓扑排序等。
- **动态规划**：解决具有最优子结构的问题，如背包问题、最长公共子序列等。

**1.2.3 复杂算法题解**

以下是一些复杂的算法题：

- **回溯算法**：解决组合问题，如全排列、组合等。
- **分支限界**：用于解决大规模问题，如旅行商问题。
- **贪心算法**：用于解决最优子结构问题，如活动选择问题。

  ```python
  def twoSum(nums, target):
      nums_dict = {}
      for i, num in enumerate(nums):
          complement = target - num
          if complement in nums_dict:
              return [nums_dict[complement], i]
          nums_dict[num] = i
  ```

- **最长公共子序列**
  ```python
  def longestCommonSubsequence(nums1, nums2):
      m, n = len(nums1), len(nums2)
      dp = [[0] * (n + 1) for _ in range(m + 1)]
      for i in range(1, m + 1):
          for j in range(1, n + 1):
              if nums1[i - 1] == nums2[j - 1]:
                  dp[i][j] = dp[i - 1][j - 1] + 1
              else:
                  dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
      return dp[m][n]
  ```

  ```python
  class DistributedStorageSystem:
      def __init__(self, num_shards):
          self.shards = [Shard() for _ in range(num_shards)]

      def store(self, key, value):
          shard_index = hash(key) % len(self.shards)
          self.shards[shard_index].store(key, value)

      def retrieve(self, key):
          shard_index = hash(key) % len(self.shards)
  ```

  ```python
          self.servers.append(server)

      def get_server(self):
          return self.servers[0] if self.servers else None
  ```
