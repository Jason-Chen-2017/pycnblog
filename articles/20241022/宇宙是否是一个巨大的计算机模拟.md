**案例背景**：
假设我们想要模拟一个简单的宇宙，其中包含两个天体：一个质量为 \( m_1 \) 的星系和一个质量为 \( m_2 \) 的行星。我们使用N体模拟算法来模拟这两个天体在引力作用下的运动。

**开发环境搭建**：
- 编程语言：Python
- 科学计算库：NumPy、SciPy
- 图形库：Matplotlib

**源代码实现**：

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置参数
G = 6.67430e-11  # 万有引力常数
m1 = 1.989e30  # 星系质量
m2 = 5.972e24  # 行星质量
r_init = np.array([1e11, 0, 0])  # 初始位置
v_init = np.array([0, 2e4, 0])  # 初始速度

# 模拟时间
t_start = 0
t_end = 1e9
dt = 1e4

# 初始化位置和速度
positions = [r_init.copy()]
velocities = [v_init.copy()]

# 模拟循环
t = t_start
while t < t_end:
    # 计算引力
    r = positions[-1] - positions[-2]
    F = G * m1 * m2 / np.linalg.norm(r)**2
    a = F / m2
    
    # 更新位置和速度
    r_new = positions[-1] + velocities[-1] * dt
    v_new = velocities[-1] + a * dt
    
    positions.append(r_new)
    velocities.append(v_new)
    
    t += dt

# 绘制结果
plt.plot([p[0] for p in positions], [p[1] for p in positions])
plt.xlabel('X position (m)')
plt.ylabel('Y position (m)')
plt.title('Orbit of a Planet around a Star')
plt.show()
```
