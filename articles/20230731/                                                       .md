# 2.AI基础知识
首先，我们要回顾一下AI的基本知识。这里列举一些AI的相关基本概念：
## 2.1 人工智能
人工智能（Artificial Intelligence，AI）是模仿人的智慧与能力的机器，可以完成人类无法完成的重复性任务。它包括五大类核心任务，即推理、观察、交互、决策、学习。
   - **推理：** AI通过算法和模式识别技术，用自然语言、图像、声音、行为习惯等方式，模拟人的想法、情感、判断、推理等行为，实现人类的决策与理解。
   - **观察：** AI通过巧妙设计的数据收集和分析方法，实现对环境的实时监测，发现异常、异常状态、分布规律等信息，并做出有效反应。
   - **交互：** AI通过语音、文字、视觉、触摸、机械臂等方式，与用户进行沟通互动。通过合理响应、快速反馈和智能调节，使交互过程变得更加高效，提升交互效果。
   - **决策：** AI根据历史数据、知识库、计算模型等，对外界环境、内部情况、目标规划等进行分析，制定相应的决策方案。例如，驾驶汽车、自动驾驶汽车、订单预订、风险控制等。
   - **学习：** AI通过自身的学习和改进机制，不断优化其模型和策略，提升性能。
## 2.2 机器学习
机器学习（Machine Learning）是利用计算机算法从数据中提取知识的领域。它通过训练大量样本数据，对输入的模式进行分类、预测、聚类等，从而实现对未知数据的预测、决策和控制。典型的机器学习算法包括KNN、SVM、神经网络等。
   - **分类器（Classifier）：** 是一种基于数据集的模型，用于对某种数据进行分类，例如可以根据身高、体重、血糖指标等属性，预测病人是否患有心脏病。分类器通常由训练数据、规则和算法组成，算法负责对数据进行判别和分类。分类器可以分为两大类型，一是监督学习分类器，二是非监督学习分类器。监督学习分类器通过已知的正确标签训练模型，再对未知数据进行分类；非监督学习分类器不需要正确标签，直接通过数据结构和相似性关系进行分类。
   - **聚类器（Clustering）：** 是一种无监督学习方法，用于对数据集进行分组。算法通过分析数据之间的距离和相似性，将相似的样本归属于同一个群组。聚类器可以有层次聚类、凝聚层聚类、球状聚类、密度聚类、最大疏散聚类等多种形式。
   - **回归器（Regression）：** 是一种基于数据集的模型，用于对连续变量进行预测。回归器通过分析数据间的线性关系，将输入变量映射到输出变量的期望范围内，输出变量的值越接近期望，就越精确。
## 2.3 深度学习
深度学习（Deep Learning）是机器学习的一种子领域，它对大数据进行特征提取，采用多层神经网络，通过迭代的方式优化参数，提升模型的准确性和性能。
   - **卷积神经网络（CNN）：** 是一种用来识别图像的神经网络，特点是在卷积层中进行特征提取，通过池化层消除冗余信息，然后在全连接层进行分类。它可以对图片、语音、文字等进行多模态建模。
   - **循环神经网络（RNN）：** 是一种用来处理序列数据的神经网络，特点是循环连接各个单元，实现长期依赖和记忆功能。
   - **门控递归网络（GRU）：** 是一种扩展版本的RNN，添加了一个门控机制，用于解决梯度消失、梯度爆炸的问题。
   - **生成对抗网络（GAN）：** 是一种无监督学习的方法，由生成网络和判别网络组成。生成网络通过随机噪声生成假样本，判别网络通过学习判断样本是真还是假。生成网络的损失函数使得生成样本与真实样本尽可能接近，而判别网络则需要尽可能区分生成样本和真实样本。
   
## 2.4 强化学习
强化学习（Reinforcement Learning）是机器学习的另一子领域，它通过给予奖励和惩罚信号，训练智能体（Agent）在环境中不断调整策略，以达到最优的奖励总和。
   - **动态规划：** 在强化学习中，动态规划算法用于求解最优的策略。它把复杂的决策过程分解为子问题，每一个子问题对应着一个当前状态，需要寻找下一个状态的最优策略，最后得到整体最优的策略。
   - **蒙特卡洛树搜索：** 在强化学习中，蒙特卡洛树搜索（Monte-Carlo Tree Search，MCTS）是一种搜索算法，用于求解在每个节点上进行采样的情况下，达到当前状态的最佳策略。
   - **AlphaGo** 是人工智能史上具有里程碑意义的一项大项目，它由美国科罗拉多大学的李世石、柯洁、李明博等人于2016年联合提出。它是第一个完全自主地围棋，并且胜率超过了世界顶尖围棋选手。
