        # 2.数据定义和特点
        ## 2.1 数据定义 
        数据(data)是对客观事物的某种记录、测定、测量或计算所得的数字、文字、图形、声音、图像、视频等信息。
        
        ## 2.2 数据特点
        1. 数据大小 
        
        2. 数据结构
        
        3. 时效性
        
        4. 数据共享范围
        
        5. 数据保护方式

        6. 数据流动方向

        数据通常具有以下特征：
        - 大：数据规模越来越大；
        - 异：数据源自不同来源；
        - 冗：数据量过于庞大、混杂；
        - 分：数据存在多个层次；
        - 宏：数据可进行分析处理；
        - 噪：数据不容易获取或采集。

        总结一下，数据具有以下几个特点：
        1. 大小：数据有不同的大小级别，从微型到万亿级别；
        2. 结构：数据有不同的结构形式，包括表格、图片、视频、声音、文本等；
        3. 时效性：数据随着时间推移而变化，数据的生命周期由数据产生到消失；
        4. 共享范围：数据可以被不同应用系统之间共享；
        5. 数据保护方式：数据的安全、隐私保护需要得到保障；
        6. 数据流动方向：数据会从一个地方流向另一个地方，如来自终端设备上传输到云端。

        ## 2.3 社会数据定义
        “社会数据”(social data)是指面向社会和公共事务、代表社会现实的各种数据，主要涵盖社会、经济、文化、政治等多个方面的信息。目前社会数据主要包括以下五类：
        1. 结构化数据：结构化数据是指按照一定格式化模板，采用计算机、数据库等方式存储、整理、检索、分析和处理数据的一种数据。如：地理位置数据、客户行为数据、人口统计数据、组织结构数据、社会经济数据等。
        2. 不结构化数据：不结构化数据不受固定格式限制，其内容或结构都可能随时发生变化。如：微博、微信群聊、互联网评论、邮件通信记录、社交媒体活动轨迹等。
        3. 半结构化数据：半结构化数据一般是指具有一定的结构和模式，但内容或结构有缺陷，可能导致无法精确呈现数据的真实含义。如：互联网搜索日志、电影评分数据等。
        4. 非结构数据：非结构数据无明确的结构，只有信息的主题和分类，而没有明确的上下级关系。如：社交圈子，用户发布的信息聚焦一个或几个关键词，内容具有时空上的相关性。
        5. 流动数据：流动数据是指随时间推移不断产生更新或更新的统计数据。如：各个行业领域的数据展示、股票市场价格走势、财务报告数据等。

        从以上定义可以看出，社会数据是用来描述、分析和预测社会经济、社会生活、社会现象的一种信息。
