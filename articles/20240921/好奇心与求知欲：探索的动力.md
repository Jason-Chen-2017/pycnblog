## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

深度学习算法的核心是神经网络，神经网络的核心是神经元。神经元的基本数学模型可以用以下公式表示：

$$
f(z) = \sigma(w \cdot z + b)
$$

其中，$\sigma$ 为激活函数，$w$ 为权重，$z$ 为输入，$b$ 为偏置。

### 4.2 公式推导过程

神经元的激活函数 $\sigma$ 通常为 Sigmoid 函数或 ReLU 函数。Sigmoid 函数的推导过程如下：

$$
\sigma(x) = \frac{1}{1 + e^{-x}}
$$

ReLU 函数的推导过程如下：

$$
\sigma(x) =
\begin{cases}
0, & \text{if } x < 0 \\
x, & \text{if } x \geq 0
\end{cases}
$$

### 5.2 源代码详细实现

以下是一个简单的神经网络模型，用于猫狗图片分类：

```python
# 导入 TensorFlow
import tensorflow as tf

# 设置随机种子
tf.random.set_seed(0)

# 创建 TensorFlow 张量
x = tf.random.normal([100, 784])  # 输入张量，100 个样本，每个样本 784 维
w = tf.random.normal([784, 10])    # 权重张量，784 维输入，10 维输出
b = tf.random.normal([10])          # 偏置张量，10 维输出

# 定义激活函数
sigmoid = lambda x: 1 / (1 + tf.exp(-x))

# 定义神经网络模型
model = lambda x: sigmoid(tf.matmul(x, w) + b)

# 训练数据
x_train = x
y_train = tf.random.normal([100, 10])  # 输出张量，100 个样本，每个样本 10 维

# 定义损失函数和优化器
loss_fn = tf.reduce_mean(tf.nn.softmax_cross_entropy_with_logits(logits=model(x_train), labels=y_train))
optimizer = tf.optimizers.Adam()

# 训练模型
for _ in range(1000):
    with tf.GradientTape() as tape:
        loss = loss_fn(model(x_train), y_train)
    grads = tape.gradient(loss, [w, b])
    optimizer.apply_gradients(zip(grads, [w, b]))

# 评估模型
y_pred = model(x_train)
accuracy = tf.reduce_mean(tf.cast(tf.equal(tf.argmax(y_pred, axis=1), tf.argmax(y_train, axis=1)), tf.float32))
print("Accuracy:", accuracy.numpy())

# 绘制损失函数曲线
plt.plot(np.arange(1000), loss_fn(model(x_train), y_train).numpy())
plt.xlabel("Epoch")
plt.ylabel("Loss")
plt.show()
```

### 5.3 代码解读与分析

- 导入 TensorFlow 库和必要的库。
- 设置随机种子，确保实验结果可重复。
- 创建 TensorFlow 张量，用于存储输入、权重和偏置。
- 定义激活函数，选择 ReLU 函数作为隐藏层激活函数。
- 定义神经网络模型，使用 TensorFlow 的 `tf.matmul` 函数实现矩阵乘法。
- 训练数据，生成输入张量和输出张量。
- 定义损失函数和优化器，选择交叉熵损失函数和 Adam 优化器。
- 训练模型，使用 TensorFlow 的 `tf.GradientTape` 类实现自动微分。
- 评估模型，计算分类准确率。
- 绘制损失函数曲线，观察模型训练过程。
