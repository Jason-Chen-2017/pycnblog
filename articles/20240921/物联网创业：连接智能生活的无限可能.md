### 5.2 源代码详细实现

以下是一个简单的物联网项目示例，实现了一个智能灯泡的自动控制：

```python
from network import Network
import time
import json

# 配置网络
sta_if = Network.STA_IF
sta_if.active(True)
sta_if.connect("your_wifi_ssid", "your_wifi_password")

# 配置服务器地址
server_ip = "192.168.1.1"

# 初始化网络
def init_network():
    while not sta_if.isconnected():
        print("连接中...")
        time.sleep(1)
    print("网络已连接")

# 发送数据到服务器
def send_data(temperature):
    data = {
        "device": "smart_light",
        "temperature": temperature
    }
    headers = {
        "Content-Type": "application/json"
    }
    response = Network.http_post(url=f"http://{server_ip}:8080/temperature", headers=headers, json=data)
    print(response)

# 主程序
def main():
    init_network()
    while True:
        temperature = 25  # 这里使用一个模拟的温度值
        send_data(temperature)
        time.sleep(60)  # 每隔60秒发送一次数据

if __name__ == "__main__":
    main()
```

- **数据发送**：通过`http_post`方法，将温度数据发送到服务器。
- **定时发送**：程序每隔60秒自动发送一次数据。
