         # 2.TensorFlow
         ## 2.1 TensorFlow简介
         TensorFlow (简称TF) 是Google推出的开源深度学习平台，由Google的研究团队设计开发并维护。它最初被设计用于训练神经网络模型，后来逐渐演变为支持其他形式的机器学习，如图像识别、文本分类、声音处理、图神经网络等。其核心数据结构张量（tensor）可以看作多维数组，使得计算和训练模型变得十分简单、快速和可靠。
         2015年，深度学习框架 TensorFlow 以其简洁灵活、强大的性能、广泛覆盖各类模型的能力以及出色的文档和社区影响而备受瞩目。它已经成为目前最热门的深度学习框架之一，越来越多的公司和组织纷纷将其用于自身的深度学习系统开发之中。其中包括谷歌、微软、亚马逊、腾讯等。
         ## 2.2 TensorFlow安装配置
         ### 2.2.1 安装
         #### CPU版
             pip install tensorflow==1.15.0  
         #### GPU版
            暂不支持GPU版本的安装，可以直接下载安装cuda环境后再安装tensorflow-gpu。
         ### 2.2.2 配置
         在安装好 TensorFlow 之后，我们还需要配置一下环境变量，让 Python 找到 TensorFlow 的路径。
         1. 将安装好的 TensorFlow 拷贝到某个目录下（假设是 D:\ProgramData\Anaconda3）。
         2. 在用户目录下创建一个 `.bashrc` 文件，如果没有的话可以先创建，然后编辑文件：`vi ~/.bashrc`。
         3. 向 `.bashrc` 文件末尾添加以下两行命令：
            ```bash
            export PATH="$PATH:/D/ProgramData/Anaconda3"    # 添加该行，设置 PATH 环境变量，指定 Anaconda 目录为 TF 安装位置。
            source activate        # 设置当前 shell 使用激活的 conda 环境（若没用过 anaconda 可以忽略这一步）
            ```
            上述两条命令的含义分别为：
            1. `export PATH="$PATH:/D/ProgramData/Anaconda3"` 告诉 Linux/Unix 操作系统从指定的目录查找可执行文件的路径。
            2. `source activate` 命令激活当前 shell 中的 conda 环境。
            3. 如果你安装了anaconda，并且通过conda新建了一个虚拟环境，那么在上述第二步的时候就可以不需要进行这一步了。

         4. 执行如下命令使环境变量生效：`$ source ~/.bashrc`。
         5. 此时，如果打开新的终端窗口，输入 `which python`，输出应该为 `/D/ProgramData/Anaconda3/bin/python`，表示 Python 已成功连接到 TensorFlow 。
         6. 检查 TensorFlow 是否安装成功：在 Python 命令行中运行 `import tensorflow as tf`，如果正确安装，不会报任何错误信息；否则，根据报错信息排除故障。
         7. 安装成功！

     3. Pytorch
      ## 3.1 Pytorch简介
    Pytorch是Facebook AI Research (FAIR)团队在深度学习领域开发的一个基于Python语言的开源深度学习框架。Pytorch具有以下特征：

    - 基于动态图(dynamic graph)，能够实现自动求导
    - 支持多种设备，CPU/CUDA
    - 提供高效的 GPU 加速计算能力
    - 可扩展性强，模块化设计
    - 支持多种网络层、优化器、损失函数等

    2016年8月，Facebook AI Research团队宣布开源PyTorch，并发布了第一版。随后，大量企业开始采用PyTorch进行深度学习模型开发和应用，包括微软、网易等知名互联网企业。目前，PyTorch已经成为深度学习领域最热门的开源框架。
    PyTorch近几年来被各大深度学习框架吸收并融入自己的框架体系，在计算机视觉、自然语言处理、推荐系统等多个领域均取得了显著成果。
    下面我们就介绍几个常用的PyTorch模块，它们分别是：

    - 数据加载模块torchvision: 提供常用的图像数据集加载接口，例如CIFAR-10、MNIST等。
    - 模型构建模块torch.nn: 提供了丰富的卷积、循环、反向传播网络组件。
    - 优化器模块torch.optim: 提供了常用的优化算法，例如Adam、SGD等。
    - 损失函数模块torch.nn.functional: 提供了各种网络层、损失函数的API接口。
    - GPU加速模块torch.cuda: 提供了GPU加速相关接口，能够充分利用GPU资源。

    PyTorch还有很多模块功能，可以通过官方文档了解更多细节，感兴趣的读者可以自行探索。
    Keras
    Keras是一个基于Theano或TensorFlow之上的高级神经网络 API，提供了易于使用的前端接口，能有效简化深度学习模型的构建和训练。它是在 TensorFlow 和 Theano 之上构建，兼容纯净的NumPy 数组。
    Keras提供了一些常用的网络层和优化器，可以极大地简化深度学习模型的搭建，通过回调机制支持训练过程的动态调整。同时，它也提供了便利的接口，能够将训练结果存储在HDF5格式的文件里，或导出为不同的格式。
    下面我们介绍几个常用的Keras模块，它们分别是：

    - 数据加载模块keras.preprocessing: 提供了常用的图像数据预处理接口，例如ImageDataGenerator。
    - 模型构建模块keras.models: 提供了丰富的基础网络层和模型组件，例如Dense、Conv2D、MaxPooling2D等。
    - 优化器模块keras.optimizer: 提供了常用的优化算法，例如RMSprop、Adagrad等。
    - 损失函数模块keras.backend: 提供了基于Theano或者TensorFlow的不同框架的后端，提供统一的接口，方便移植。
    - GPU加速模块keras.utils.training_utils: 提供了GPU加速相关接口，能够调用相应的后端接口，完成相应的加速。

    Keras的功能也远不止这些，可以通过官方文档了解更多细节，感兴趣的读者可以自行探索。
