# 1.简介
         
1956年，蒂姆·伯纳斯-李发布了著名的“计算机论文”，从此开启了一个重要的纪元，人们用科技解决问题的奇想似乎已经实现了。1962年底，当时的美国研究机构贝尔实验室的物理学家克劳德·艾耶格特-霍金在《科学》杂志上发表了一篇题为《生命、宇宙和死亡》的文章，触及了科技和人类自然界的边界，并激起了无穷的探索之旅。人类通过“互联网”这一新兴技术不断发现新的可能性，构建起复杂而丰富的世界。但是，也正如当今的科技日益加速发展一样，互联网同样也面临着巨大的变革。2016年，阿里巴巴创始人马云曝出了“巨头王座”被清华大学工程院院长钱强瓦重新命名为“阿里巴巴集团”。互联网行业的争议也随之产生。特别是在新经济下，互联网行业领导者越来越看重竞争力。今年1月份，阿里巴巴集团董事长刘淇在新浪微博上发出文章《加快推进中小互联网创业平台创新升级》，呼吁各个创业公司拓展更多的市场，开展更高质量的创业活动。
         本文将梳理一下《互联网发展简史》这本书的主要内容，介绍一些相关背景知识，阐述互联网的发展历史。
         # 2.基本概念术语
         2.1 概念介绍
         在互联网的历史上，我们一般把网络（network）、信息（information）、万维网（World Wide Web）等术语叫做第一层次。“互联网”这个词由“internet”和“web”两部分组成，前者指网络技术，后者指互相链接的超文本系统。
         “互联网”这一概念最早出现于1969年美国国防部的官方网站。当时，美国政府与一些国家商务、电讯、金融界人士建立了联系。这些人意识到信息交流迅速成为商业行为的一种必需，因此便组织起来制订出了一系列条约。其中包括了《国际数据通信保护条例》、《国际电报传输保护条例》、《网络安全法》等。经过若干年的沟通协调，各方都认可了互联网技术及其带来的巨大潜力。
         2.2 发展阶段
         互联网的发展历程可以分为四个阶段：
         - 1969年，“互联网”概念出现；
         - 1980年代，互联网的蓬勃发展，新闻网站、邮件服务、电子商务等诸多应用得到蓬勃发展；
         - 2000年代，互联网的爆炸性增长，网民数量激增，内容更新速度加快，用户参与度逐步提升；
         - 当今，互联网正在经历第五个十年的百花齐放的变局，突飞猛进的创新力促使互联网迎来了崭新的发展时期。
         2.3 浏览器和搜索引擎
         20世纪80年代末，美国的两个学生提出的概念是：互联网 = 浏览器 + 搜索引擎。即，浏览器负责显示网页的内容，搜索引擎负责检索互联网上的信息。“引擎”二字的意思是“引擎盖车”，可以理解为提供引擎支撑的整体平台。搜索引擎可分为三个阶段：
         - 早期阶段：采用查询字符串搜索，仅对少数网站开放。
         - 中期阶段：引入分类目录和专业化索引，出现一批具有垄断地位的搜索引擎，例如谷歌、百度、雅虎等。
         - 新阶段：以用户需求驱动，形成智能问答机器人、人工智能助手等应用。
         2.4 万维网和因特网
         20世纪90年代中期，美国国家标准与技术研究院的成员约翰·斯托曼提出了概念“万维网(WWW)”，用来描述由许多互相连接的服务器所组成的超级大型网络。2004年10月，因特网正式名称确定为“因特网”（Internet），简称为“网”。“万维网”和“因特网”都是同一个概念。但是，为了区别，笔者习惯将它们分别称作“万维网”和“互联网”。
         2.5 TCP/IP协议族
         20世纪90年代末，互联网的发展面临着难以估计的问题，TCP/IP协议族一直是技术骚乱的主角。TCP/IP协议族中的三大协议分别是：传输控制协议（Transmission Control Protocol，缩写为TCP）、网络互连协议（Internet Protocol，缩写为IP）和地址解析协议（Address Resolution Protocol，缩写为ARP）。TCP/IP协议族为互联网的开发奠定了坚实的基础。
          2.6 HTML和CSS
         1989年，蒂姆·伯纳斯-李在《科学》杂志上发表了论文“超文本标记语言”（Hypertext Markup Language，HTML），标志着HTML的诞生。它是一种用于创建网页的标准标记语言，可以让作者方便地添加文字、图片、链接、表单等各种内容，还可以用脚本进行编程。到了20世纪90年代末，随着Web2.0的兴起，又出现了CSS，这是一个能够对HTML页面设计进行样式设置的方案。CSS的诞生标志着网页设计技术发生了翻天覆地的变化，极大地丰富了网页的视觉效果，并带来了全新的页面布局方式。
