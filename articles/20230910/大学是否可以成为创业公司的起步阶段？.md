
作者：禅与计算机程序设计艺术                    

# 1.简介
  

“大学”在今天的社会里无处不在。许多中小型企业都想要利用“大学”这个平台进行创业。为什么很多创业者选择上“大学”，而很多创业公司却放弃了“大学”的尝试呢？

“大学”这个名词近年来随着互联网的发展而流行起来。大家越来越倾向于用互联网的方式接触到知识、信息、资讯。当下最火爆的创业公司都已经转型为基于互联网的公司形态，包括社交网络、云计算、物联网等新兴领域。这些新的商业模式也需要创业者们借助互联网的工具，对自己的知识技能进行整合和运用。

那么，什么样的“大学”才算是创业公司的起步阶段呢？在这个问题上，我将结合我的个人经验和看法做一个分享。
# 2.背景介绍
首先，我想从我的个人经历出发，讲一下我是如何走上创业道路的。

2017年，我刚刚完成了本科毕业，进入了一所普通的四年制大学。这个时候，我并没有准备要开始创业。我只是希望能享受高等教育带来的优质教育资源，并且以后还能通过读书获取新知、提升自身能力。所以，对于“大学”这个入门级的创业平台来说，我并不十分了解。

2019年初，在接到一个牵头成立创业的大佬邀请后，我主动地跟他谈了一下自己对互联网+金融行业的理解和看法。当时，他给的方向很宏观，包括移动互联网、人工智能、区块链、新经济……等多个方面。

那个时候，我对金融领域还比较陌生，甚至连“区块链”这个概念都不是很熟悉，所以我回答他说我更愿意跟随各个创业公司一起探索，去做能够与之匹配的项目。

2019年6月份，我正式加入了聚焦于人工智能、区块链相关领域的创业团队。创始人是李子璇，她本科是清华大学的研究生，她领导着团队，主要负责前期的策略规划和产品设计。

2019年底，团队的规模已经突破百人，我们成功拿下了一个支付宝的广告招投标项目。之后，我们又陆续收获了微软、阿里巴巴等知名创业公司的合作。

但是，有一个问题一直困扰着我，我觉得还是应该探索一下这个平台的合适性。


2020年3月份，我收到了创业咨询类的内部资料，里面有一条关于“大学”是否可以作为创业公司的起步阶段的建议。当时我心里还是蛮纠结的，因为作为“大学”的学生，我并没有参加过任何创业课程，更没有经历过很多创业公司的打磨流程。

不过，作为一个顶尖的科研工作者，我很清楚每一个项目背后的复杂性，所以我决定直接照搬这个建议，让团队成员们自己搜索相关知识、学习相关技能，而不是靠别人给的建议。

此外，在此之前，我也在思考过一些相关的问题，比如如何利用“大学”这种平台进行有效的创业？需要花费多少时间、精力、金钱？有哪些方法论可以参考？还有哪些环节值得我们去关注？

# 3.基本概念术语说明
# 定义
“大学”指的是由国内的一所普通的高等院校开办的一门专业课程，提供“交通、生活、职业”等各个方面的辅导、培训和服务。它既是一种“平台”，也是一种“机会”。

“创业公司”是指依托互联网或者其他网络技术，为某一特定目标市场提供独特价值的组织。创业公司在很长的一段时间内都是由个人或几个人组建起来的，通常会秘密设立公司的名称，然后在网络上发布自己的产品或者服务，吸引用户的注意力。

# 创业公司的目标定位
根据《中国商业法》第五十条的规定，创业公司的目标市场范围主要包括以下几个方面：

1. 需要解决的现实或战略性问题；
2. 有潜在客户群体的市场，且这些客户可能不会接受非专业人员的顾客服务；
3. 具有独特性和竞争优势的市场；
4. 对创造和传播价值的产品或服务有需求的消费者群体。

# “大学”可以成为创业公司的起步阶段吗？
目前，中国大学生数量激增，“大学”被视作创业公司的起步阶段。但为什么“大学”依然被视为起步阶段，而且在互联网+金融领域占据了主流地位呢？

为了回答这个问题，我们可以从三个角度分析。

## 1.品牌效应
互联网+金融的蓬勃发展已经引起了行业内的热议。在国内，不少创业公司已经创建了知名品牌，比如微信支付、美团、滴滴出行等，它们为用户提供了方便、快捷的服务，让更多的人参与到金融领域。

这种“大学”品牌效应表明，只要创业公司拥有一定的品牌号召力，就可以把“大学”打造成创业公司的入门级平台。同时，“大学”还可以展示其丰富的资源和课程，吸引更多的创业者加入，促进“大学”课程的升级。

## 2.创业公司门槛低
在互联网金融领域，创业公司的门槛较低。由于涉及到各种加密数字货币、区块链等技术，目前创业公司最基本的要求就是懂技术。如果能够有一支全面掌握互联网金融技术的团队，则可以更轻松地处理这些复杂的业务，快速启动创业公司。

因此，“大学”创业公司的起步阶段，既可以充满活力，又可以简单易懂，对创业者有一定的吸引力。

## 3.免费课程
除了上述的品牌效应和创业公司门槛，“大学”还可以提供免费的课程，帮助创业者解决实际问题，提升创业能力。例如，根据中国大学生创业联盟的调查数据显示，88% 的中国创业者认为他们所在的城市缺乏专业的创业培训机构，所以“大学”是一个很好的平台。另外，“大学”还可以邀请一批国内的优秀创业者来分享自己的创业经验，更有利于创业者之间的交流。

综上所述，“大学”可以成为创业公司的起步阶段，虽然在中国的互联网金融行业尚处于起步阶段，但“大学”的确正在改变着创业者的命运。
