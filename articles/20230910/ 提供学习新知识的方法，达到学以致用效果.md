2.基本概念和术语
## 2.1 什么是机器学习？
机器学习，英文名为Artificial Intelligence，直译过来就是人工智能。从字面上来看，我们可以把它理解成指由计算机完成某种任务而自动提升性能的能力。实际上，机器学习更加复杂，涉及多个领域，如模式识别、图像处理、语音处理、机器翻译、语言处理、决策支持、生物信息等。这里只讨论机器学习与人工智能相关的内容。

一般来说，机器学习分为三类：监督学习、无监督学习、半监督学习。下面我们详细阐述这些概念。
### （1）监督学习（Supervised Learning）
监督学习，也就是训练数据中既含有输入特征向量x也含有输出标签y，也就是输入数据与期望输出之间存在对应关系，即标签y是可知的，称为监督学习。例如：在训练数据中给出了猫和狗的照片，通过分类器将它们区别开来。

### （2）无监督学习（Unsupervised Learning）
无监督学习，是指训练数据仅含有输入特征向量x，但没有标签y。这种情况下，要寻找数据中的结构和规律。常用的无监督学习算法有聚类、降维等。例如：在一堆用户行为数据中，利用聚类算法发现不同的用户群体。

### （3）半监督学习（Semi-Supervised Learning）
半监督学习，是指训练数据既含有输入特征向量x还有标签y，但只有少量数据拥有标签y。此时可以将标签y视为助教，引导模型学习到更好的特征表示。例如：假设我们收集了一批病人的患者数据，其中有些已经标注有症状，有些则没有。我们可以用半监督学习算法来判断哪些患者的症状容易被检测出来，然后根据这个信息来推荐相应的治疗方案。

## 2.2 什么是知识、模型、算法？
由于机器学习涉及多个领域，因此术语层次很深，如 knowledge、model、algorithm、feature、label、training set、test set 等。下面逐一解释。
### （1）Knowledge
知识，是指对世界的认识或者理解。例如，知识可能是一本书中所包含的关于神话、科学、历史、哲学等知识。

### （2）Model
模型，是指用来刻画现实世界的符号和结构。例如，神经网络模型是一个用于拟合数据的计算框架。

### （3）Algorithm
算法，是指指导模型运行的指令或规则。例如，朴素贝叶斯算法是一种基于概率分布的分类算法。

### （4）Feature
特征，是指对待学习样本的输入变量。例如，图片的像素点就是图像特征。

### （5）Label
标签，是指对待学习样本的输出变量。例如，垃圾邮件是否为垃圾就是标签。

### （6）Training Set
训练集，是指用于训练模型的数据集。例如，一组图片可以作为训练集，模型就可以根据其特征来预测标签。

### （7）Test Set
测试集，是指用于评估模型性能的数据集。例如，一组新的图片可以作为测试集，模型就可以衡量它的表现。
