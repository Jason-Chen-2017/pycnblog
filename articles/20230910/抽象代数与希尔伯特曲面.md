# 1.简介
  

## 抽象代数
抽象代数（英语：abstract algebra）是指研究集合、线性映射、群论、环论、交换群等基本对象及其相关运算规则，并运用数学语言、方法和技术研究这些对象之间的联系及其规律的一门学科。它是数学的一个分支，也是许多其他科学领域的基础。在计算机科学中，抽象代数被用来研究数据的表示、分析、运算、安全和复杂性。抽象代数还可以应用到物理学、工程学、经济学等领域。本文主要涉及抽象代数中的希尔伯特曲面，这是一种高阶的几何对象。
## 概念术语说明
### 范畴（Category）
$C(x)$是一个范畴，它由一个集合$X$和两个操作$\circ : C(x)\times C(y)\rightarrow C(x\circ y)$和$id_x: X\rightarrow C(x)$构成。其中，$x,y∈X$且$a\in C(x),b\in C(y)$，$f\circ g=(f\circ g)(x)=f(g(x))=g(f(x))$称为幺半边函子（Semilattice functor）。因此，范畴具有结合律。比如，函数范畴中的笛卡尔积、加法范畴中的加法、乘法范畴中的乘法都是范畴的例子。这个概念的定义可以推广到多维空间中的不同直角坐标系，也可用于研究对象之间的关系。
### 代数结构（Algebraic structure）
一个代数结构$A$是一个范畴$C(A)$上赋予一个二元运算符$+:\mathcal{S}(A)\times \mathcal{S}(A)→\mathcal{S}(A)$，满足结合律，并且满足以下三条条件：

1. $+$满足分配律：$(a+b)+c=a+(b+c)$
2. $\exists e_{A}: A\rightarrow +e_{A}+\mathcal{I}_{A}=A$, 这里$\mathcal{I}_A=\bigoplus_{\alpha}\delta_\alpha^\mathrm{(A)}\quad(\delta_\alpha^\mathrm{(A)}=\left\{a_{1},\ldots,a_{\ell_\alpha }\right\})$是$A$上的加单位元。称$e_{A}$为$A$上的加零元，$\mathcal{I}_{A}$为$A$上的乘单位元。
3. 对任意元素$x,y,z∈ A$,$xyyz^{-1}=xzy$。

上面三条要求对于同一个代数结构来说是不变的，并且仅依赖于该代数结构所属的范畴，即$A$和$C(A)$是独立的。举例来说，矩阵的加法和乘法构成了R模块和范畴的运算，而群的加法和乘法则构成了半群和群的运算。
### 群（Group）
一个群$G$是一个代数结构，它由一个集合$X$和两个二元运算符$+,\cdot$组成，满足结合律、分配律、交换律、群乘法定律，即：

1. 封闭性：$a,b∈ G,\forall x (x+a)\neq a$
2. 结合律：$(ab)c = a(bc)$
3. 幺半边操作：$a^{n}=na^{\mathit n-1}$ （$\mathit n$为自然数）
4. 可逆元素：存在$a^{-1}$,使得$aa^{-1}=\mathrm{Id}_G$ 

另外，群还要满足AB=BA和AC=CB的关系，即$a\cdot b=b\cdot a$ and $a\cdot c=c\cdot a$ 。很多特定群的研究都集中在其不同的特征方面，如正规群、循环群、陪集群、对称群、阿贝尔群等。
### 环（Ring）
一个环$R$是一个代数结构，它由一个集合$X$和两个二元运算符$+,-\cdot$组成，满足结合律、分配律、交换律、分配律、乘法的结合律、乘法的幺半边操作、乘法的可逆元素、乘法的结合律、乘法的幺半边操作、乘法的可逆元素，即：

1. 环的结合律：$r(-s)=-(rs)$
2. 环的分配律：$(r+s)t=rt+st$
3. 零元：存在唯一的$0_R$,使得$r+0_R=r$ for all $r\in R$.
4. 单位元：存在唯一的$1_R$,使得$r+1_R=1_R+r$ for all $r\in R$.
5. 可逆元：存在$r^{-1}$,使得$r\cdot r^{-1}=1_R$ for all $r\not=0_R$.
6. 同一律：$\forall p,q\in R,pq=qp$.
7. 乘法同余律：$r\cdot s=t$当且仅当$rs=ts$.

环还要满足AD=-BD的关系，即$a\cdot d=-b\cdot d$，群则不能。环的研究也可以分为不同的群环类，如整环、有限环、实环、复环等。
### 交换环（Commutative Ring）
一个交换环$R$是一个环，它对所有$a,b\in R$都有$a+b=b+a$.
### 环的商空间（Quotient space of ring）
设$R$是一个环，$I\subseteq X$是一个集合，那么环$R/I$的商空间，记作$R|_I$，是由$I$生成的$R$的子空间，但不是$R$本身。$\forall a,b\in R/I$,有$ab\in I$或$ba\in I$，且$(ab)'=ba, (ba)'=ab$。
### 希尔伯特曲面（Hilbert Spaces）
希尔伯特曲面（Hilbert space，也叫Banach space），又叫无穷维向量空间。它的研究对象是由标量场给出的曲线、曲面和弧线，而非一般的向量空间。一般地，我们可以把希尔伯特曲面的元素看做高维空间中的线性变换。也就是说，给定的曲线$C$,相应的线性变换$\varphi : U\rightarrow V$ ，满足：

1. $\mathrm{Im}(\varphi)=U\backslash W$
2. $\mathrm{Ker}(\varphi)=W\backslash V$

其中，$U$和$V$分别是曲线$C$的定义域和值域，$W$和$N(C)$分别是曲线$C$上面的无穷远点和曲率无限大的区域。
