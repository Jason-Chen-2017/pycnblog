# 1.简介
  

去中心化金融（Decentralized Finance，或称DFI）是一种由比特币支持的新兴法定货币，旨在通过减少中间人（如银行）对数字货币的信任，提高数字货币的流动性和价值，从而降低整个经济体系的不确定性。本平台将提供基于DFI的各种服务，包括但不限于去中心化交易所、DEX、借贷、保险等服务。

## DEX
去中心化交易所（Decentralized Exchange，DEX），又称去中心化交易所系统（Decentralized exchange system，DESS），是利用区块链技术打造的数字资产交易平台，用户可自由地进行数字货币买卖、套利，并可以随时获得成交量报酬。无论是个人还是机构，均可开设自己的DEX平台。其主要功能如下：

1. 抵抗“交易所撮合”风险：在DFI上交易的买卖成本更低，而且无需通过中介机构的撮合成本，完全解决了“交易所撮合”风险。此外，DEX上可以直接进行高级交易，比如止盈止损、杠杆交易等。
2. 提升用户体验：DEX具有极快的响应速度，用户可以在几秒内完成下单、撤销订单，并实时查看成交状态。
3. 满足多种数字货币类型交易需求：目前支持BTC、ETH、BCH、LTC、DOGE和USDT等主流币种的交易，其中BCH、LTC、DOGE等与比特币挂钩，让用户能够享受到比特币带来的便利。未来还将陆续推出更多币种的交易，进一步扩大规模。
4. 大幅降低费用：DFI及相关代币交易不需要手续费，仅需支付转账手续费。并且，用户可通过DFI购买数据存储、算力资源、以及其它用途的高级服务。这些都是当前平台的独特优势。
5. 有助于促进数字货币的发展：DFI的通胀率很低，在年底的较低价格水平，仍然可吸引广大的投资者，成为促进数字货币发展的有力工具。

## AMM
自动做市商（Automated Market Maker，AMM），是指由智能合约执行自动化的买卖交易。这些智能合约运行在区块链网络上，确保各个参与者的利益最大化。AMM的出现意味着去中心化交易所可以根据流动性提供方和流动性承载方的需求，对数字货币进行定价，实现套利。常见的AMM产品包括Uniswap、Sushiswap、Balancer、Curve等。

1. 减少交易成本：由于DEX上可以直接进行高级交易，比如止盈止损、杠杆交易等，所以减少交易成本是AMM的关键。AMM通过算法优化套利机制，使得交易双方都有收益，同时也有效降低了交易成本。

2. 降低交易滑点：同样是采用套利机制，AMM可以通过价格波动缓解交易滑点。当双方的报价相差较小时，套利机会多一些；相反，当报价相差较大时，套利机会就少一些。

3. 减轻手续费压力：在DEX上进行交易，无需缴纳交易手续费。但是，AMM需要支付手续费才能建立池子，因此交易过程中可能会产生手续费。但是，AMM利用算法自动化结算，降低了手续费损失。

4. 更加灵活的套利模式：DEX上只能进行一种类型的套利，而AMM可以自由组合不同的币种。这样，用户就可以通过调整不同资产的数量和比例，实现更加灵活的套利模式。

5. 有助于促进更多的加密资产进入市场：AMM的出现，为更多数字货币项目的进入创造了空间。尤其是在增持方面，AMM可以更好地满足投资者的需求。未来，将会有越来越多的数字货币项目加入到AMM中，以促进数字货币的整体发展。

## 借贷
借贷服务是基于区块链技术的新的金融形式，能够让用户在线向其他用户借款，也可以允许用户将现有的资产出售给他人，以换取他们的贷款。借贷服务的发展已经取得巨大成功，已成为新的金融形态之一。本平台将提供基于DFI的借贷服务，包括但不限于借贷、投资咨询、保险等服务。

1. 减少贷款违约风险：借贷服务允许用户借款，降低了贷款违约风险。由于借款人不会去管后面如何使用借款，所以不存在过高的风险。当借款期限结束时，即时结清。

2. 降低融资成本：借贷服务无需担心抵押品估值过高导致的融资成本问题。只要抵押物价值达到一定程度，它就会得到贷款。另外，借款人的信用评级也会影响最终的金额。

3. 更容易接触到借款对象：借款平台可以帮助用户找到自己感兴趣的借款对象，无论是爱好者还是企业，都可以通过借贷平台与之沟通。

4. 支持多种货币类型：借贷平台支持众多币种的抵押品，比如BTC、ETH、BCH、LTC等。只要抵押品满足要求，用户即可快速借款。未来，借贷平台将会支持更多币种的抵押品。

5. 降低服务费用：对于借款人来说，最重要的是钱，而平台却无法抽成。不过，借贷平台可以通过贷款人的信用评级进行补偿。未来，借贷平台也会通过AI算法，自动分析用户的行为习惯，进行智能化的反馈和推荐。

## 保险
保险是利用区块链技术实现的新的金融业务形式。用户可以借助平台提供的保险产品，快速获得抵御风险、维护资金安全的能力。本平台将提供基于DFI的保险服务，包括但不限于传统保险、寿险、财险等服务。

1. 提供多种保险产品：平台提供多种保险产品，包括传统保险、寿险、财险等，以满足不同类型用户的需求。

2. 不用担心违约风险：平台采用区块链技术，可以保证用户的资金安全。通过严格的审核流程，保险公司可以确保客户得到满意的服务。另外，平台还可确保被保险人没有任何隐瞒或欺诈行为。

3. 通过奖励机制提高客户忠诚度：平台通过奖励机制，鼓励用户长期坚持使用平台。例如，对于优秀的客户，平台会给予赠送礼品，比如优惠券、奖金。

4. 支持多种数字货币类型：平台可以支持众多的数字货币类型，包括BTC、ETH、BCH、LTC、DOGE等，这些币种都是与比特币挂钩的币种。

5. 降低服务费用：保险产品本身不收取任何费用，客户只需支付抵扣税费。平台则只收取管理费用的费用。未来，平台还将引入分成机制，提高服务质量。
