# 1.简介
  

量子计算（quantum computing）可以分为两类——模拟计算与真实计算。模拟计算是用计算模型或者模拟的方式计算某个量子系统的状态，可以理解为用数学模型模拟人眼看到的“黑箱”，通过这个模拟结果来求解实际的问题。而真实计算是利用量子力学中描述的量子态，在真实的物理系统上完成计算。因此，量子计算是一种技术，涉及到物理、数学、工程等多学科领域。但是由于要达到更高的计算性能，产生更多的存储空间等，所以基于量子计算机的计算平台也越来越多。

目前，主要的量子计算平台包括IBM的Qiskit、Rigetti Computing的Quil、AWS的Braket、Google的Cirq、Xanadu的Quantum Cloud Service以及Microsoft Azure Quantum等。他们都提供了多种编程语言的API接口，用户可以方便地调用进行量子计算任务。

本文将从以下几个方面介绍目前最火爆的两个量子计算平台——Qiskit 和 Quil。

# 2.Qiskit
## 2.1 Qiskit简介
Qiskit是一个开源的Python框架，用于开发和运行量子信息和量子电路相关的应用。它提供可扩展且易于使用的工具集，使得量子程序员能够轻松地构建、模拟和调优量子电路和算法。它支持的硬件平台包括模拟器（如 qasm_simulator），实际硬件设备（如 IBMQ Provider），也可以运行在超级计算机上。

Qiskit能够支持从各种角度进行量子编程，包括开发者、研究人员、教育者、学生等不同群体的需要。其中，开发者可以通过Python和Qiskit API来构建复杂的量子应用程序。而研究人员、教育者和学生则可以利用其丰富的工具集来学习量子信息、量子计算和量子控制论。

Qiskit旨在促进量子计算的开源化、普及化，并且吸纳量子技术的最新进展。它的目标就是成为一个开放的社区项目，鼓励更多的开发者参与到该领域的开发和建设中来。

## 2.2 Qiskit特点
### 2.2.1 可扩展性
Qiskit具有高度可扩展性，允许用户添加新硬件或模拟器，同时仍然保持足够的灵活性和便利性。开发者可以非常容易地创建新的组件，并使用现有的库和函数。例如，IBMQ Provider可以使用已有的登录系统和仿真服务，只需简单配置一下即可让其提供真实的物理量子设备上的量子计算功能。

### 2.2.2 易于使用
Qiskit API的设计风格符合一般用户的直觉和习惯，并带来了一致的开发体验。用户可以轻松地组合构成量子电路的基本元素，并获得丰富的图形化输出。还内置了许多经典算法，用户可以直接调用这些算法，无需实现自己的版本。此外，还有很多示例和教程，使得初学者可以快速入门。

### 2.2.3 工具包丰富
Qiskit API提供了丰富的工具包。例如，可以轻松地生成随机的量子电路，或者对参数进行优化。Qiskit还提供了一个数据库，用于保存量子电路和算法的元数据，包括作者、发布时间、摘要、引用等。

### 2.2.4 多样化的硬件支持
Qiskit支持多个硬件平台，包括模拟器（如 qasm_simulator）、实际硬件设备（如 IBMQ Provider）、以及超级计算机。用户可以根据需求选择合适的硬件平台来运行量子电路。

# 3.Quil
## 3.1 Quil简介
Quil 是由 Xanadu 公司开发的一门量子语言，用于编写量子程序。它属于量子编程语言类别中的编程语言，用于构建量子软件。目前，Quil 可以编译成基于 QVM 的 API，在模拟器上测试量子电路。除了 QVM 以外，其还可以编译成许多其他硬件后端的 API，比如 Rigetti 的 Forest 概念。Quil 的源代码文件可以被编辑器自动语法高亮显示，使得它易于阅读和编写。Quil 还提供了命令行工具和 Jupyter Notebook 的接口。Quil 允许用户通过声明性语法描述量子程序，并提供了编译器验证其正确性。

## 3.2 Quil特点
### 3.2.1 可扩展性
Quil 有着很强的可扩展性。它的 IR（中间表示法）提供了足够的自由度，允许用户自定义程序的行为。例如，用户可以在 Quil 中定义新的指令集，或嵌套其他函数来构造复杂的程序。Quil 提供了许多类似函数式编程语言的特性，用户可以用来解决量子计算问题。

### 3.2.2 量子语言类别
Quil 属于量子编程语言类别。它与量子线路语言 Cirq 或 IBM Quantum Assembly Language 不一样，这些语言是专注于控制逻辑的指令集。相反，Quil 是面向程序的量子语言。

### 3.2.3 支持范围广
Quil 支持大量的硬件后端，包括原生的模拟器、Rigetti 的 Forest 概念和 IonQ 的量子处理单元。这使得 Quil 成为量子计算领域最流行的语言之一。
