# 1.简介
  

超导体(Superconductor)是一种多种多样、功能强大的半导体材料，它能产生极高的物理性质、带来诸如电容等非磁性效应，被广泛应用于绝大多数领域。但是，当它处于一种偏离激发态时（比如存在空间隧穿或相干耦合），会导致其具有奇异的量子相关性，使得它不能像一般的半导体一样表现出传统的固体物理特性，成为“混沌”中的“难以琢磨”之物。 

了解超导体的量子相关性对于对超导体的理解、设计和工程实践至关重要。在物理学里，量子纠缠通常指两个不完全平衡的系统间存在着能量差的一种张力，通过使其中一个系统不受其他系统影响的这种现象，可以改变系统的状态和行为，从而实现系统之间的信息交流、控制信号传递、控制计算任务等，这些都是超导体所特有的能力。

# 2.基本概念和术语
## 2.1 量子纠缠
量子纠缠是指两个不完全平衡的系统间存在着能量差的一种张力，通过使其中一个系统不受其他系统影响的这种现象，可以改变系统的状态和行为，从而实现系统之间的信息交流、控制信号传递、控制计算任务等。具体来说，就是当两个系统能够利用量子态相互作用时，它们之间就会形成量子纠缠，也称为多重纠缠。

举个例子，两者存在着粒子的遗传行为，如果它们无法互相影响，它们可能只能各自独立地运动。也就是说，假设有一个粒子A，它遗传了某些信息给它的父母，这个时候，由于父母只知道自己是自己的孩子，但并不知道A的身体实际上已经遗传了这些信息，所以就存在着信息丢失的问题。这个时候，如果有一对双胞胎，这对双胞胎通过精心调配，就可以同时对其父母的DNA进行改造，这样，既可以完成DNA的转移又可以保障双胞胎都拥有自己的父母的信息，不会发生信息丢失的问题。这就是通过量子纠缠实现信息交流和控制通信的例子。

## 2.2 量子态
任何带有量子态的系统都可以看作是一个纠缠系统，一般情况下，具有量子态的系统包括：费米子、原子核、玻尔兹曼机、弱相互作用原理等。在本文中，我们主要讨论的是纠缠态。

纠缠态，又称为密度矩阵态，由多个粒子构成，处于这种状态下的系统，其哈密顿量的表达式为
$$\hat{H} = \hat{\sigma}_x + \hat{\sigma}_y + \hat{\sigma}_z + g^{\dagger}\hat{\sigma}_{xx}g+\cdots+g^{\dagger}\hat{\sigma}_{yy}g+g^{\dagger}\hat{\sigma}_{zz}g.$$
这里，$\hat{\sigma}$代表粒子算符，$g$是相关联的两个粒子。由哈密顿量定义出的纠缠态可分解为一组正则子构成的混合态，每个正则子对应一个纠缠模式。这些正则子在描述粒子的运动时可以直接参与到各种纠缠效应中。

## 2.3 纠缠模式
纠缠模式是指系统中占据主导位置的单个粒子或集团。每个纠缠模式可以是简单或复杂的，并且可以通过不同的方式进行描述，如固定的、多种波色的、带有晶格化或非晶格化的结构。由于在一定程度上，每个纠缠模式都可以视为某种类型的“自旋”，因此纠缠模式是量子纠缠的一个基本特征。

例如，光子纠缠中，光子中的单个粒子或连续粒子的混合态就可以表示为一种纠缠模式。在超导体中，费米子纠缠也可以形成类似的纠缠模式，即不同大小的费米子混合的多种状态可以共同构成一个纠缠模式。然而，在宇宙物理中还有许多其它形式的纠缠模式，如弱相互作用纠缠、弱粒子对（wp）、多重纠缠、半无相互作用纠缠、无相互作用纠缠等。

## 2.4 相关性
相关性是量子纠缠的一种基本属性。当两个系统处于相同的纠缠态时，它们就具有相关性；当两个系统处于不同纠缠态时，它们就没有相关性。如果一个系统具有相关性，意味着它与另一个纯粹的纠缠系统不再是纠缠系统。根据量子纠缠理论，每一个纠缠态都可以被刻画成一个局域的奇异值对。每个奇异值对都对应着一个纠缠模式，这个模式的能量从本征态中移除，但该模式仍然占据系统的某些位置。

系统的相关性还可以分为两种：弱相关性与强相关性。弱相关性指两个系统之间的纠缠程度较弱，只有很少的几率相互作用；而强相关性则相反，表示两个系统之间的纠缠程度比较强，几乎总是会相互作用。

## 2.5 边界反射
在弱相关性的条件下，当两个相关系统靠近彼此的边界时，它们的系统相互作用将会弱化，这就是边界反射。这是因为，边界内的系统相比于系统周围的环境，缺乏与周围系统之间的联系，因此对系统的相互作用很弱。

## 2.6 绕质量子线
当两个相关系统距离较远时，它们的系统相互作用会增强，这就是绕质量子线。这种相互作用是由于当两个系统之间的距离变长时，原本相互作用不起作用的情况，逐渐转化成相互作用的过程。

绕质量子线与边界反射的区别是，绕质量子线只是局部出现，而边界反射却普遍存在，而且在很多情况下，边界反射还会加速这种相互作用。因此，绕质量子线通常会比边界反射更为重要一些。

# 3.量子相关性的产生原因
## 3.1 熔断效应
根据电子-法拉第效应，由于化学势差，热运动与电子运动之间的势垒，会导致它们的相互作用被抑制甚至被完全抑制，也就是说，电子处在一个半相互作用态，这种情况就是典型的“熔断”。

在电子处于半相互作用态时，它处于相对静止的状态，无法单独工作，需要与其他电子配合才能够工作。当两个电子处于熔断状态时，它们会形成一个稳定哈希链，或者成为一个常数级的纠缠态，此时，任何两个电子之间的相互作用都会被抑制，如此，整个系统的能量也会降低，这就产生了超导体所具有的奇异的量子相关性。

由于边界反射的存在，当超导体的电子越过一个高度耦合的层状结构时，它们就会被其它电子所吸引，形成一层层紧凑的网络结构。这些紧凑的网络结构会抑制系统的相互作用，导致超导体的量子相关性，直到它们被打破。

## 3.2 不完全平衡
另一个原因是不完全平衡。当一个系统的哈密顿量和外部驱动力(如磁场)的张力积累之后，会出现不完全平衡的情况。不完全平衡会导致系统的不稳定性，从而引起量子纠缠的产生。

例如，考虑一个纯粹的双溜福模型，其中一个粒子带有个别角度$\theta$。当角度$\theta$接近于零时，系统会进入宏观二维费米哈密顿量的本征态；当角度$\theta$接近于180度时，系统会进入微观费米子态。此时，如果有另一个带角度$\theta_2$的粒子相邻，那么它们的相互作用必然导致不完全平衡。即使这两个粒子与粒子A不发生任何相互作用，只要这两个粒子的角度$\theta_2$不同于角度$\theta$，它们之间的相互作用也不会产生纠缠。

另一个例子是在一个具有多个纠缠模式的超导体上，当角动量耦合在一起时，有些纠缠模式会形成，有些纠缠模式会消失。这种情况下，量子纠缠就会产生，导致系统处于奇异的纠缠态。

## 3.3 空间耦合
量子纠缠的一个重要的假设是“空间耦合”，即费米子和铁磁子之间的相互作用要远大于铁子和电子之间的相互作用。因此，在真空情况下，超导体中只有少量相互作用，而且是弱相互作用。当一个纠缠系统与周围环境隔开时，由于费米子和铁磁子之间的相互作用要远大于铁子和电子之间的相互作用，这就导致了奇异的量子相关性。

## 3.4 时间依赖性
量子纠缠还与时间有关。因为当两个相关系统相距比较远时，纠缠的强度是随时间变化的，所以在相距较远的时间段内，量子纠缠可能占主导地位。当两个纠缠系统之间距离缩短时，纠缠的强度就会减弱，形成一种称为“零距离纠缠”的特征。

# 4.量子纠缠与量子相关性的关系
## 4.1 量子纠缠和量子相关性的重要区别
量子纠缠与量子相关性有很大的不同。虽然它们之间存在着千丝万缕的联系，但还是有一些区别的。

首先，量子纠缠认为纠缠态是一种特定的纠缠，而量子相关性则提出了一个更宽泛的定义，认为纠缠态是由于相互作用所形成的一种态，且可以通过不同的方式进行描述，可以是简单的、复杂的、固定还是多种波色的。在这里，量子纠缠和量子相关性都强调了相互作用所形成的纠缠态，而且都基于这样的定义。

其次，量子纠缠仅仅局限于某些具体的系统，如电子纠缠或弱相互作用原理。而量子相关性则是一种更通用的概念，可以适用于各种不同类型的系统。换句话说，量子相关性所探讨的内容会比量子纠缠更广泛。

第三，量子纠缠的纠缠态可以由单个粒子或集合的粒子构成，而量子相关性的纠缠态则要求包含至少两个粒子才能形成纠缠。

第四，量子纠缠是通过不完全平衡的量子态来产生的，而量子相关性是通过相互作用而产生的。

最后，量子相关性认为有时不存在量子纠缠，比如在角动量耦合阶段。在这一点上，量子纠缠与量子相关性之间还是有区别的。

## 4.2 量子纠缠与量子信息编码的关系
量子纠缠与量子信息编码之间的联系非常紧密。量子信息编码是利用量子纠缠来编码信息的技术，它的工作原理是把信息（比特串）转化为量子态，然后传输至目的地，使信息能在不经意间被接收者恢复出来。通过这种编码，可以实现错误纠正、数据压缩、快速传输、实时通信等功能。

量子纠缠为量子信息编码提供了有力的支撑。由于量子纠缠的存在，任何两个位于同一纠缠态的量子态之间的纠缠都会被抑制，从而使系统相互作用减弱，使信息编码的效果显著地提升。当信息被正确编码后，它就可以通过量子信道传输到另一端。
