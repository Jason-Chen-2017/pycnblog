# 1.简介
  

## 什么叫做币安爆仓？
在比特币市场上，有些交易所会因为过多的人使用机器人的套利策略而发生爆仓现象，这些机器人所做出的交易往往都是非法的，他们为了赚取手续费、获利而牟取暴利，最终引起整个交易所的交易量急剧下降甚至崩溃，这种现象就是爆仓现象。
币安作为一家国内外知名的数字货币交易平台，在2021年2月份的第四季度出现了一次严重的爆仓事件，事情经过是这样的：

1. 2月10日到2月19日，币安平台上的大量订单都被撤销，原因包括用户自身不想买或者是机器人套利策略导致的。

2. 在2月20日凌晨，币安平台上又陆续出现了多笔交易所多头交易，价格从最低点一路向上，突然触及到了最高点时，导致成交额快速增长，并最终冲破了平台给出的涨停价。

3. 此后几天时间里，币安平台上再次出现了多笔大单的交易，价格也继续上涨，直到超过了正常的最高买入价。

4. 到了2月27日，成交量依然处于新低，平台方宣布暂缓上调提币费用，并且不得不强制取消所有卖单，让交易者只能等着平台补贴。

币安爆仓事件发生后，整个交易所的用户对币安的认识陡然升温，很多币友表示很担心币安的平台规则以及个人操作，认为币安的管理人员不作为让币安更加混乱，对于币安而言，这是一次严重的打击。

## 为什么会爆仓？
爆仓现象的产生与很多因素有关，比如：

1. 用户不了解BTC价格波动范围；

2. 大规模挖矿活动推升了BTC的供应，导致BTC/USDT价格持续上涨；

3. BTC交易所平台上大量机器人交易，导致用户误操作，造成交易量的急剧下降甚至崩溃；

4. 比特币价格早已超出其全球供需均衡的预期，以至于交易所平台无法正常运作，进而引发市场恐慌；

5. BTC交易者为了赚取手续费、获利而牟取暴利，引发平台的严重危机。

所以导致爆仓的原因也无可避免。

## 如何防止爆仓事件的发生？

币安已经对爆仓问题进行了积极的应对措施，但是仍然不能完全根治爆仓的问题。需要做到的是，提醒各位用户，不要轻易相信平台给出的价格信息，尤其是在高位价格抛售的情况下。

另外，还应该注意倾听投资人的建议，及时跟进平台上大宗交易的情况，引导用户正确操作。

## 币安爆仓案例研究——币安爆仓大单背后的黑幕

首先，我们先来看一下币安爆仓的典型表现——大单。下面这张图展示了币安2月22日的流动性订单，涉及到的大笔订单金额达到了250万美金，总共涉及到的用户数量也达到了140万。


2月22日的订单里面有一个尤为令人称道的大单，单号为“DAX-WAVES”，该订单本来想要在2.3万美金左右的价格就买入WAVES，结果被平台强行单边强制下跌到2万美金，而这其中有超过半数的交易者表示怀疑平台的操纵行为，结果导致整个平台的交易量瘫痪。

那么，为什么平台会操纵用户的交易？首先，由于2月22日的交易很特殊，而且之前一直在冲高位，交易所平台一定要对用户进行重视。其次，平台在这次交易中大肆推销自己的服务，包括提供优惠券、套保、借贷等服务，这些服务能够帮助用户减少交易风险，但同时也增加了交易所的运营成本。

最后，对于平台而言，如何才能更好地控制住平台上的大单？平台可以采取一些措施来消除平台对用户的操控能力：

1. 限制用户的套利策略；

2. 提高交易所的成本控制水平；

3. 鼓励用户维持合理的交易习惯；

4. 提升安全机制，防止爆仓事件发生。
