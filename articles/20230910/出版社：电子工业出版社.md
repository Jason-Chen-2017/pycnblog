# 1.简介
  

## 1.1 电子工业出版社简介
“电子工业出版社”（俗称“四元文化发源地”）是由湖南省高级知识产权律师事务所建立的网络出版社。公司总部设在长沙，主要经营范围包括IT互联网产品、图书、电子书等。截至目前，已经成为中国最知名的网络出版社之一，连续九年荣膺中国网络出版前三甲品牌。在2019年一季度入围了第十二届全国新闻媒体自由创作赛亚洲区赛事。
## 1.2 出版物系列
电子工业出版社推出的出版物系列主要有以下几种：
* 技术类：涵盖计算机、编程、软件开发、数据库、系统分析、设计等领域。
* 经济类：涉及金融、保险、贸易、投资、银行、证券等各个领域的书籍。
* 情感类：包括爱情、婚姻、职场、工作、生活、家庭、旅游等主题的书籍。
* 其他类型：还有少儿教育、艺术、小说等不同主题的书籍。
# 2.基本概念术语说明
## 2.1 “创新出版社”的概念
“创新出版社”一词被提出于1996年的首届网络出版创刊会上，是网络出版领域里的一种新的观念。创新出版社强调“用新技术、新方法、新工具、新手段、新思路、新体验，重新定义电子出版领域”；他们与传统出版社在出版物的形式、布局、编辑方式上存在着巨大的差异。创新出版社把目光放到读者身上，在新颖的市场需求下，引导读者进入一种全新的阅读体验，创造性地解决复杂的信息需求，从而推动网络出版领域的革命。
## 2.2 《数字人民币》项目的提出
“创新出版社”创办的第一个产品——《数字人民币》，是在1997年由当时国家财政部委托的专业机构——中国国家电子研究所主办的一次会议后，由陈云翔、马天华、张铁军、王乐泉等四位作者一起完成的。当时的会议通过了一个“全面数字化改革方案”，即全面对政府、企业、个人、媒体及政府部门进行信息交流和服务，实现全球电子支付体系的新型数字化，并开创了一个重要的思想基地。这个方案产生了一个可操作性很强的新理论，即“共同构建新的信息技术架构”。因此，为了能够让更多的人理解这一理论、接受它的指导意义，陈云翔、马天华、张铁军、王乐泉等人联合创始了《数字人民币》项目。该书的第一版由《光明日报》杂志社主编，将于2005年出版。之后陈云翔带领出版社出版了有关数字人民币的多种参考文献、专著、白皮书、电子百科全书、学术期刊等，并且为创办者提供了极其宝贵的营销资源。至今，《数字人民币》已经成为中国网络出版领域的一项重要探索。
## 2.3 一本书就是一个章节
在信息时代，一本书不再是一个实体的作品，而是由很多章节组成的。这些章节可能是相互关联的，如序言、前言、正文、附录、引用文献、注释、致谢等。电子工业出版社将网络文学与网络出版相结合，采用“一本书就是一个章节”的创作模式，通过把网络出版中常见的图片、视频、音频、动画、表单甚至算法都转换成网络文档，使得读者可以在线阅读，避免因缺乏互联网条件无法阅读的困难。这样做既方便用户阅读，也满足了网络出版业的审美需求。
