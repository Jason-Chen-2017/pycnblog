# 2.为什么选择在美国做公益？
在美国做公益的原因很多，比如经济上的便利、良好的医疗保健体系、丰富多彩的文化氛围以及充满魅力的乡村风情、牛仔热闹的夜市和小吃城市。

然而，我认为最主要的原因还是对自我的定位。在美国，个人的社会责任比国内更加重要。作为一名公众人物，每天我们都要为自己的行为负责，用自己的双手、声音和言辞影响着我们的国家、民族、家庭及周边的人。在这里，你有能力为改变世界贡献一份力量。

换句话说，你更值得去认真思考：你是不是应该更加关注自己的生活、为世界作出微小的贡献。更进一步，你是否拥有足够的知识和技能，能够影响到这个国家的发展方向？更为重要的是，你希望通过自己的努力，帮助别人吗？这才是真正令人着迷的地方。

# 3.如何在美国找到合适的机会？
美国是世界上最大的资本主义国家，拥有庞大的外汇储备。同时，美国也是许多创业公司、企业的摇篮。美国的科技和金融驱动着创新的潮流。因此，在美国找到合适的机会是一个比较复杂的问题。但是，仍然有一些指导性建议可以帮助你：

1. 投资回报率高：美国的股票、债券、房产、消费、基金等投资领域都是靠着长期投资回报率来保持盈利增长的。因此，你需要留意相关的财务数据，判断其是否能够为你的创业奠定坚实基础。另外，美国的创业板市场比中国的创业板市场要受到更多的关注，而且还处于发达国家行列的前列。

2. 重视个人奉献精神：美国是一个高度自由的人口国度，每个人都有自己的事业、旅游计划、住宿费用。因此，在美国找到合适的机会，通常情况下都可以获得足够的资源支持。不过，就算有资金支持，也需要注意自身的时间精力和行动力。只有自己真正感兴趣的领域，才能使你持续努力。

3. 沟通能力强：在美国，个人之间的沟通是个十分重要的技能。因此，了解与被影响人的需求、态度、价值观很重要。除了善于自我介绍外，还需要学会和他人进行有效的沟通，并且能够让对方相信你。

4. 不要急功近利：很多时候，我们只知道自己想要什么，却不知道自己需要付出多少。在美国，你可以根据具体情况和目标，制订一个实际可行的方案，而不是想着马上就可以实现目标。

5. 关注业界动态：美国是一个创业大国，随时都会出现新的创业机会。因此，一定要留意新闻媒体的最新报道，了解一些目前最流行的创业模式。并且结合自己的兴趣爱好，构建一个独特的产品或服务。

最后，做公益的道路还远非完美无缺。你还需要综合运用自己掌握的各项能力，凝聚起自己的力量，创造共同的未来。
