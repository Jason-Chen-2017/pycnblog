# 1.简介
  

随着web技术的快速发展，前端开发已经成为一种全新的技术类别，也将成为最受欢迎的职业。前端工程师需要扎实的编程功底、设计能力、交互体验和团队协作精神等技能，才能胜任此项工作。作为一名优秀的前端工程师，掌握一些常用的技术栈会极大的提升自我竞争力和职场竞争力。因此，本文通过系统整理了前端开发者应具备的知识基础和技术能力，并给出相关资源供读者学习。

# 2.基本概念术语说明
## 2.1 HTML/CSS
HTML（Hyper Text Markup Language）即超文本标记语言，它是构建网页结构的基础语言。包括标签及其属性、元素嵌套规则、文档类型、字符编码、注释等内容。

CSS（Cascading Style Sheets）即层叠样式表，它用于对网页的风格进行美化。包括选择器、盒模型、颜色、文字大小、边距、背景、文字效果、布局、多列布局、动画、渐变、轮廓等各种样式设置。

## 2.2 JavaScript
JavaScript是一个脚本语言，可以实现动态页面更新、用户交互事件响应、数据交换、动画制作、移动应用开发等功能。主要用于控制页面的显示，添加交互动效，为网站提供更多吸引人的功能。

JavaScript具有丰富的对象、运算符、函数、异步处理等特性。使用ES6语法编写的高级语言具有模块化、异步处理、面向对象的特性，能更好地适应Web开发的需求。

## 2.3 Node.js
Node.js是一个基于Chrome V8引擎的 JavaScript 运行环境，使得 JavaScript 成为服务器端编程的一大利器。可以使用 JavaScript 的npm管理工具安装第三方模块，并配合其他如Express框架进行服务端的开发。

## 2.4 React Native
React Native 是Facebook推出的跨平台开发框架，可以开发原生应用（Android 和 iOS）或可访问网页的hybrid应用。它的特点是一次编写，到处运行，而且性能表现不逊于原生应用。

React Native 使用 JSX 来描述组件，使得代码更加易懂。同时还提供了许多内置组件和API，让开发者可以方便地进行移动App的开发。

## 2.5 Vue.js
Vue.js 是一套构建用户界面的渐进式框架。它采用自然的模板语法和简单灵活的组件系统，极大地简化了创建和维护大型应用的过程。

Vue.js 的核心库只关注视图层，不仅易上手，而且还便于与第三方库或既有项目整合。在实际场景中，它可以帮助开发者快速搭建小型应用，也可以快速接入第三方接口实现复杂功能。

## 2.6 AngularJS
AngularJS 是一款由Google开发的用于构建 web 应用的客户端开发框架。它有着简单、直观的语法和独特的编程模型，提高了开发效率和项目的可维护性。

## 2.7 jQuery
jQuery 是一款轻量级的 JavaScript 框架，它能够简化 HTML 文档的加载、选择、事件处理、动画以及 Ajax 操作。

## 2.8 TypeScript
TypeScript 是微软推出的一门强类型的 JavaScript 语言，是JavaScript的超集。它允许开发者在编译时检测代码错误，支持继承、接口、泛型等高级特性，提高代码质量和可维护性。

## 2.9 Webpack
Webpack是一个前端资源包和模块打包工具。它能够将不同文件模块化并输出浏览器可识别的静态资源，例如JavaScript、CSS、图片等。

## 2.10 Babel
Babel 是一款 JavaScript 编译器，可以将 ES6+ 版本的代码转换成传统浏览器可以识别的 ES5 或之前版本的代码。这样就可以达到代码的兼容性和可移植性。

## 2.11 Electron
Electron 是一个用JavaScript、HTML 和 CSS 构建跨平台桌面应用程序的开源框架。它使用 Chromium 和 Node.js 来构建一个完整的本地应用，拥有完善的 API 支持。

## 2.12 Sass/Less
Sass 和 Less 是 CSS 预处理器，它们扩展了 CSS 的语法，增加了变量、嵌套、混入、运算、函数等特性。这些特性使得 Sass/Less 可以更方便地进行 CSS 的维护和开发。

## 2.13 Yarn
Yarn 是 Facebook 提出的JavaScript 包管理工具，它解决了 npm 在依赖版本上的一些问题。它的速度比 npm 更快，并且使用起来更加方便。

## 2.14 Parcel
Parcel 是一个打包工具，它能够将 Web 应用的各个资源文件（JavaScript、CSS、图像等）自动合并、压缩、并为其添加适当的 headers，从而创建一个单独的文件。
