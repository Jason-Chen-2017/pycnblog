# 1.简介
  

在越来越多的应用场景中，区块链技术正成为世界金融领域的热门话题。很多创业者都想把自己的区块链项目投入市场，但却无从下手。区块链的诸多特性赋予了它巨大的商业价值，然而如何在不同的国家法律法规及监管要求下，准确有效地缴纳相应的税费，仍然是一个难点。本文将深入浅出地探讨区块链税务。
# 2.基本概念术语说明
## 2.1 什么是区块链税务？
区块链税务就是基于区块链网络的交易所、矿池、钱包等各种服务收取的税费。对于普通用户来说，区块链税务主要包括基础代币转账费、侧链平台收费以及开发工具或服务费。税务局也会根据本地区区块链行业的情况，对特定类型区块链项目进行适当的税收补贴。

## 2.2 什是基准代币（Token）？
基准代币通常是一种数字资产，其本身即代表着某个特定实体或者组织。在区块链系统中，它可以作为支付、验证、授权以及数据存储的媒介。例如比特币就是一种基准代币，用户可以在其上进行转账、合约调用以及存储。其他的一些典型的基准代币包括以太坊的ether、火币上的HT、Filecoin上的FIL等等。

## 2.3 为什么要收取区块链税务？
区块链技术带来的巨大商业价值促使区块链平台、应用程序、钱包等各类服务机构纷纷推出相关产品。这些产品通过区块链技术提供可靠、透明、安全以及低成本的数据存储服务，并支持传统互联网应用的扩展。但是由于区块链技术的复杂性及全球化特性，国内的区块链税务局需要认真研判，制定出相对合理的税收政策来保障平台运营者的利益。否则，平台背后可能会产生巨额的风险。

## 2.5 区块链税务相关法律法规
各个地区的区块链税务法律法规不尽相同。国内主要涉及区块链交易所、钱包和矿池收取费用的相关法律法规有：

1. 中华人民共和国政府关于鼓励区块链应用发展的法律文件：《鼓励区块链应用发展条例》

2. 中国电子商务委员会、中国计算机信息网络国际联合会关于规范区块链应用的系列专利：《区块链应用专利管理规定》、《区块链金融研究专利管理办法》、《区块链智能合约专利管理办法》、《区块链数据库专利管理规定》

3. 中国税法总则：《征收电子商务管理费暂行办法》、《税收征收管理办法》

4. 沪深两市和香港特别行政区有关区块链税务规定

5. 国外相关国家关于区块链税务的最新规定

# 3.税务合规性
## 3.1 区块链税务的合规性要求
首先，根据国际通行的“区块链税务专项指引”，区块链税务必须遵循不同国家的特定法律法规及监管要求。例如，美国一般都适用美国财政部颁布的绿色区块链税种；欧盟依据欧洲联盟和国际货物与服务贸易组织的绿色区块链服务章程；日本、韩国、印度、菲律宾等国家均有相关的区块链税收规定。因此，区块链税务的合规性比较重要。

其次，区块链税务还与区块链网络建立的法律法规有密切关系。区块链的发明目的之一是解决跨境支付问题，但在很多国家和地区，这个问题并没有得到充分解决。例如，以太坊被用于犯罪活动，中国政府则试图通过限制互联网支付来打压区块链的发展。另外，一些区块链平台可能存在反洗钱、反恐怖主义的违规行为，需要由平台方承担相应的税务风险。因此，区块链税务也是一项非常重要的法律工作。

第三，区块链税务还必须考虑区块链服务的盈利模式。很多区块链平台为了赚取区块奖励或者矿池的奖励，可能会设计一些诡异的规则，比如设置虚假的奖励机制，给予少量区块的奖励，或者惩罚涉嫌虚假宣传的内容。如果区块链平台滥用其诡计，就会受到严厉的处罚。所以，区块链税务就显得尤为重要。

最后，区块链税务还需要考虑区块链的衍生品市场。根据2019年发布的《全球区块链衍生品市场展望》，整个衍生品市场规模估计将达到3万亿美元。因此，区块链税务应该把衍生品的买卖纳入其税务范围，并且严格监控。同时，也应考虑到一些平台为了获取更多的区块奖励、矿池的奖励，而把用户数据泄露到第三方平台或个人。这样的行为可能导致个人隐私泄露以及较高的损失。所以，区块链税务的工作也十分艰巨。

综上所述，区块链税务确实是一项复杂的工作。不过，通过研究，提前制定一份完整且科学的税务规范，既可提升区块链的商业价值，又可规范区块链税务的执行，防止出现不必要的误解和纠纷。

# 4.基本税率计算方法
## 4.1 按转账收取的税费
区块链网络中的基本交易量只有转账，所以区块链税务的基本税率一般设置为每笔转账收取一定比例的税费。例如，支付宝收取0.3%的转账税，微信支付收取0.2%的转账税。

此外，区块链底层设施和协议可能会收取相似的税费，比如超级节点费用、区块确认费用、签名费用等。这些费用应该加计入每个区块的税费里。

## 4.2 侧链和开发工具收取的税费
在日益增长的区块链生态环境中，一些小团队或者个人就在尝试利用区块链技术发展新的业务模型。例如，一些团队正在开发和推广侧链，利用侧链实现功能上的分层，降低运行成本，提升效率。另一些团队也开发了基于区块链技术的工具和服务。例如，一些团队正在推出基于区块链的支付服务，为个人和商户提供更便捷、安全的付款方式。

但是，这些新产品或服务的引入也可能产生额外的税务费用。例如，用户可能需要支付给服务提供者的流量费、服务器成本、域名租用费、许可证费用等。这些费用必须与用户实际支付的金额进行比例配比，才能体现到税务上。

# 5.税务监督
区块链税务的一大挑战就是随时对账户进行监测。因为，在一些操纵账户的骗局活动中，往往会伤害无辜的用户的利益。因此，区块链税务需要建立一个公开透明的监管系统，让用户能够第一时间了解到自己账户的情况。

具体做法是，每个账户都有一个唯一的地址，可以将其绑定到用户的身份信息上。如果发现账户被盗用或转移到危险境地，就可以立刻通过接口查询到该账户的所有交易记录，并进行清算。通过监管部门的协助，也可以查阅到所有账户的账户余额、交易历史、活动记录等信息。
