# 1.简介
  

黄金是一个庞大的避险资产，作为一种有价值的储藏储蓄工具，它的价格具有无可替代性。长期看，黄金投资者的理财产品应该更重视保本保收益和低风险等方面。所以，如何更加有效地利用黄金作为财富来源，这是投资黄金的关键。在《3.什么时候买入黄金，什么时候卖出黄金？——黄金投资策略》中，将从以下几个方面，介绍如何判断黄金价格的波动情况，以及如何进行投资决策：

1. 确定黄金价格指标（即黄金的“价值”）；
2. 定义高于或低于其价格指标的黄金投资机会；
3. 根据所做出的投资决策，对黄金持仓进行调整；
4. 对金融市场风险的分析及预测方法。

# 2.黄金价格指标及判断方法
## 2.1 黄金价格指标
由于黄金存在巨大的市场波动性、难以估计的规律性、以及独特的储藏属性，因此，对黄金的价格有很强的预测能力是非常困难的。为了能够准确的估计黄金的价格，通常采用以下几个方法：

1. 市场报价法（Market Price Method）。该方法通过直接与供应商、交易中心和黄金业协会等金融行业组织合作，获取黄金价格的实时报价，并进行分析，然后基于分析结果制定黄金的价格指标。
2. 均衡指数法（Equilibrium Index Method）。该方法通过分析黄金价格随时间、地点和供应量的变化规律，设计出一种价格指标系统，包括一个或多个价格特征指标，用于衡量黄金在各个市场条件下的价格表现。
3. 模型预测法（Model-Based Prediction Method）。该方法通过建立基于历史数据、物理模型、认知科学、艺术、神经网络等机器学习技术的模型，根据经济、政治、社会等多种因素，预测黄金价格的走势。
4. 投资者技术法（Investor Technology Method）。该方法通过综合运用交易信号、量化分析、风险管理、情绪控制、金融知识等多种技术手段，结合投资者个人品味、心态和经验，客观评估黄金价格走势，并根据预测结果制定黄金投资策略。

## 2.2 确定黄金价格指标的方法
目前国际黄金价格指标主要包括如下几类：

1. 标准价格指标（Standard Price Index，SPI）：该指标以美国芝加哥商品交易所的标准黄金价格计算，该价格以加权平均方式得到。其优点是简单易懂，且市场接受度较高。缺点则是只能反映单一市场的黄金价格，不具备全球性和时效性。
2. 欧洲央行决议价格指数（Eurodollar Price Level，EDPI）：该指标由欧元区央行决议每天发布一次，用来表示黄金兑美元之间的双向对冲水平。它取自每日报价的对称分布，因而可以反映全球性、趋势性和异常变化。
3. 日本央行决议价格指数（Japanese Yen Dollar Price Level，JYDPI）：该指标由日本央行决议每隔六个月（即每年一次）发布一次，表示日本日元与美元之间的双向对冲水平。虽然跟欧洲央行决议价格指数类似，但它更为复杂、实时性也更高。
4. 欧洲黄金基准利率指数（European Gold Base Rate Interest Rates，EGRI）：该指标以欧洲央行为主要依据，以年度或季度方式计算，描述了欧洲各主要央行及金十数据集银行间兑付定价水平的变化趋势。因而，它可以反映不同国家央行政策的影响。
5. 亚洲黄金基准利率指数（Asian Gold Base Rate Interest Rates，AGRI）：该指标以亚洲央行、中国央行和印度央行为主要依据，以年度或季度方式计算，描述了亚洲各主要央行及金十数据集银行间兑付定价水平的变化趋势。它同样提供了不同国家央行政策的影响。
6. 中国黄金报价指数（Chinese Yuan Market Price Index，CMPI）：该指标以中国国家黄金局发布，以全国2000家黄金生产企业黄金库存报价为基础，结合库存报价与贸易价差、产销价比等指标，来反映国内外各黄金品种报价的变动趋势。
7. 大宗商品报价指数（Commodity Spot Price Index，CSPPI）：该指标以外汇、黄金、原油、钢铁等热门大宗商品市场的外汇报价为基础，结合各项调控信号，如沙特石油购买力提升、中国煤炭价格上涨、英镑贬值、俄罗斯黄金库存减少等，来反映全球大宗商品价格的变动趋势。

实际上，还有更多的价格指标可以使用。例如，瑞士央行的NGTI、西班牙央行的TNAPI等等。这些价格指标既能反映真实的黄金价格走势，又具有全球性和时效性。它们都是具有战略意义的参考价值指标。

## 2.3 判断黄金价格是否上涨的方法
黄金的价格具有周期性和随机性，因此，用某一具体价格去判断其是否上涨还是下跌，并不能准确反映黄金价格走势的发展情况。下面介绍一些判断黄金价格是否上涨的方法：

1. 斜率判断法（Slope Test）。该方法适用于高频交易，即每小时、每分钟都要判断价格的变动方向，并根据趋势行情去采取相应的操作。但是，这种方法耗费精力，且对大盘动荡不敏感。
2. 震荡线法（Trend Line Test）。该方法也是高频交易中的一种技术指标，通过建立价格震荡线来判断价格走势。该震荡线可以是折线图、趋势线、布林线等。它的优点是比较直观、快速，适用于个人投资者。缺点是，无法反映全局的趋势走势。
3. 移动平均线法（Moving Averages Test）。该方法适用于长期投资者，它采用固定时间长度的移动平均线（如三天、五天、七天、一月等），来显示过去一段时间的黄金价格走势。它的优点是比较准确、稳定，适用于长期投资者。缺点是，需要长时间的数据积累。
4. 平滑异同平均线法（Smoothing Phasing Averaging Test）。该方法首先计算黄金价格的年线指数，再计算该指数与前后两个月的平均指数的比值。如果比值超过某个阈值，那么可以认为市场上涨趋势已形成。该方法的优点是简单易懂、实时性高，可以应用于任何时间节点。缺点是，对价格反弹缺乏掌控力。

# 3.什么时候买入黄金，什么时候卖出黄金？
## 3.1 衡量黄金价值的方法
一般来说，衡量黄金价值的重要方法有两种：

1. 以现金的形式支付。这是指购买黄金时的法定价值。例如，若购买一枚白银的价格为10元，则按照10元计算，该枚白银为10克，可作为资产投资或偿还债务使用。
2. 在某一段时间内的黄金溢价。这种方法属于价值投资的范畴，用以测算黄金的未来回报状况。

## 3.2 买入黄金的方法
1. 资产配置法。该方法适用于资金有限、资产相对分散且风险较高的投资者。它通过把资产分为不同的类别，比如固定收益类、风险偏好类、价值追求类等，分配到不同的投资账户中，并设置适当的风险控制机制，防止投资损失。
2. 投资组合优化法。该方法适用于资金充裕、有一定资产配置经验、投资策略比较灵活的投资者。它通过考虑交易成本、资产平均风险、收益率增长速度、流动性等因素，综合选取最佳的黄金投资组合，并给予适当的提示。
3. 跟踪指数法。该方法适用于对一个或多个黄金指数感兴趣、保持持仓周期长的投资者。它通过设定目标指数的买入和卖出价格，来持续跟踪黄金指数的最新状态，提高盈亏比。
4. 套利法。该方法适用于对特定需求感兴趣、希望通过交易获得利润的投资者。它通过与其他市场（如外币、贵金属等）的黄金交易，以获利的方式实现黄金投资。

## 3.3 卖出黄金的方法
1. 保本卖出法。该方法是指在决定卖出黄金时，需要按照黄金所含有的一定数量的黄金元素（如黄金白色素、金锭、杂质等）来确定账面价值，并支付对应价值的现金。在买入时，也需支付相同的法定价值。保本卖出法可以保证黄金价值不会过分偏离其法定价值，从而帮助保障黄金投资者的损益。
2. 平价卖出法。该方法是指在决定卖出黄金时，只需按照黄金的现价卖出，而不计入费用和佣金。这样的卖出模式可能会造成盈亏不平衡，但可以帮助消除黄金投资者的担忧。
3. 逆回购法。该方法是指将黄金直接兑换成其他市场的货币，并将其卖出。这有助于降低证券市场上的冲击成本，提高黄金投资者的获利空间。然而，逆回购法容易受到美联储利率变化的影响。
4. 选择股票持有法。该方法是指在获得黄金回报后，购买并持有一支可代表黄金市场的股票。这样就可以获取股息收入，并保持持仓周期长。

# 4.未来发展趋势与挑战
随着黄金价格的不断上涨，黄金的使用方式正在发生变化。黄金越来越成为储藏金和交易资产的首选，也越来越受到投机炒作的诱惑。比如，2009年10月至2010年4月，包括阿根廷的马拉维、巴拿马的雅培、葡萄牙的伊莱索、苏丹的默里迪尔等国的央行宣布，将降息与限制黄金进口逐步放开，这对于部分投资者来说可能是一个利好的消息。另外，由于黄金的超额定价权、黄金兑现成本低、流通性强、储藏能力广、非理性定价、购买成本低、波动性大等特点，使得黄金价格的上涨率正在攀升。因此，黄金投资的未来仍然充满着各种挑战。以下是未来的黄金投资趋势和可能遇到的挑战：

1. 价格传导机制的进一步完善。目前，部分央行已经明确宣布，将进一步完善价格传导机制，释放价格信号的运行机制，改变定价权结构。对于黄金市场来说，这一轮改革的目标就是取消公允价值法和黄金资产定价权，由市场自发发掘新的交易 opportunities。比如，目前国际黄金价格指数的发展已经取得了巨大成功，其数据准确性、实时性、时效性都在逐步提升。国际黄金市场的全球化趋势，以及全球金融机构的频繁参与，也将推动市场机制的迭代升级，进一步完善价格传导机制。
2. 政策引导规则的变动。由于全球金融危机的影响，政府政策也经历了一系列的变动。包括美国的长期宽松货币政策、欧洲央行加息、日本央行利率决议等。黄金市场在全球化和国际金融机构频繁参与的过程中，也将面临不同的政策引导规则的变化。未来，在政策引导规则发生变化的环境下，黄金市场将会重新塑造自己的结构、功能、运作方式。比如，对于部分投资者来说，可能看到增加黄金使用方式的趋势，比如高端金矿开采、金融衍生品的交易、黄金黄连的日常交易等。在新的使用方式出现之前，黄金投资者可以适当寻找新的投资方向，比如数字货币、先进制业、海外黄金、黄金基金等。
3. 价格影响因素的新发现。随着全球经济和金融市场的不断发展，黄金投资者面临的各种价格影响因素将不断扩大。比如，金融市场的繁荣及产业链变革、产业结构转型、人民币国际化等。黄金投资者也将不断寻找新的价格影响因素，比如投机资金的增加、个人能力的提升、投资市场的变化、国际金融市场的格局等。在新的价格影响因素发生变化的情况下，黄金投资者可以继续跟踪指数或指数相关的交易品种，或尝试套利，以获得利润。
