# 1.简介
  

在物理界，我们一直以来都面临着挑战——如何更好地观测到宇宙的复杂特性？一个重要方向就是探索如何用望远镜和引力透镜来观测宇宙。其中，引力透镜具有长距离上天文观测的强大能力，而通过精心设计探测站点，就能够将引力透镜的输出信号转化成高频探测信号。但如果长距离引力透镜的设计难度很高的话，如何在短时间内把信号变换成高频信号呢？

为了解决这个难题，美国国家科学基金会近几年推出了一项计划——“从重构到重用”，即开发一个“长期”网络覆盖、廉价可靠的超高频探测系统，并推广到整个宇宙中，实现宇宙学科领域的重大突破。其目标是让我们能够探索宇宙的无限多样性，而不仅仅局限于固定星系中的类ical黑洞。


在本文中，我们要讨论的是这一研究计划背后的理论基础和技术进步。我们希望这篇文章能够教育读者：第一，对物理界正在经历的科研突破有个整体的认识；第二，了解目前存在的问题和限制，并明白由什么导致它们产生，以及如何缓解；第三，评估新方案是否可能达到所需效果，及如何实施部署。最后，提出一些行动性建议，以促进这一领域的发展。
# 2.核心概念
长期网络覆盖（long-term network coverage）：通过把运营成千上万颗引力透镜的探测站点互联，构建起覆盖整个宇宙的超高频网络。

超高频信号：以每秒数十倍的频率观测引力变化，能够探测到微弱引力变化，尤其是在光年范围内。

通信网络：由计算机网络技术（如“互联网”或“移动电话”）提供的低延迟、高带宽、高可靠性的传输系统，用于长期网络覆盖。

激光器：一种可以将物体直接照射到激光波段的照相机组件。

# 3.引力透镜原理
引力透镜的原理是通过使用离子束激发离子流，使得电磁场中的导体产生磁场，从而制造出聚焦区域内的高频电磁波。引力透镜最初由爱因斯坦、海森堡、马克莱门特、梅尔卡瓦等人创立，通过一个受激元件驱动并改变指向，可以生成持续不断的电磁波，即引力透镜产生的信号。

引力透镜的工作原理大概分为如下几个阶段：

1. 离子器件收集：将瞬间变化的引力场转换成离子流量。
2. 测量引力场：利用离子器件测量出来的离子流量计算出引力场分布。
3. 物质作用：由于引力透镜本身被测量材料所包裹，它可以引起附加效应，比如温度变化、空间摩擦、运动等。
4. 过程控制：引力透镜的本质是测量电磁场。过程控制包括将受测体系控制到较小的功耗状态、采用高效的传感器架构、考虑光路阻抗等。
5. 信号处理：根据测量到的电磁场数据进行信号处理，比如去噪、平滑、拟合，再把处理结果转换成可发送的电磁信号。
6. 信号传输：将处理好的电磁信号发射至远处的接收站，可通过卫星链路、电台、卫星等方式进行传输。

# 4. 信号传输协议
当前有两种信号传输协议：高速率数据链路（如IS-95）和低速率网络（如ATM）。

## IS-95
IS-95是一个高速率数据链路标准，由国际电信联盟（ITU）制定。IS-95标准规定了每秒钟能够传输的高速率数据传输速率，相比于现有的ATM标准，它可以更快地传输大量的数据。但是，它的传输速率仍然是受限的，需要高速光纤才能支持。

## ATM
ATM（Asynchronous Transfer Mode），即异步传输模式，是指传输数据时不需要建立连接。ATM适合于短距离的数据传输，相比于IS-95来说，其传输速度更慢，但传输范围更广。

# 5. 中继系统
中继系统是指将多个不同数据中心之间的信号串联起来，然后再对其进行调制、编码，送入一个中心节点，最终形成单一的信号输出。这一流程有效地降低了延迟、减少了网络拥塞，提升了传输成功率。

## LTE-A
LTE-Advanced，即高级长途卫星通讯网络（Advanced Long-Term Evolution, LTE-A）是一种基于LTEN标准的双向移动通信技术。LTE-A是随着5G的出现而兴起的，主要应用在了漫游、视频会议、高性能服务器、流媒体传输等场景。

## GNSS
GPS全球定位系统（Global Navigation Satellite System，GNSS），是导航卫星系统的一组卫星导航系统，由若干颗小型卫星固定在一块大地轨道上，用于持续、准确、高精度的定位。GNSS技术在漫游、地图显示、出租车、气象数据、航空管制、交通管制等方面都有着广泛的应用。

# 6. 中长期频谱

## 概念
中长期频谱（medium-to-long wavelength spectrum）是指有别于短波的频率范围，通常是介于300-2000nm之间。在超高频段的900-1700GHz之间，发出的信号具有接近于激光频谱的特征。

## 发展历史
在20世纪80年代，日本曾经做过一项小试牛刀，通过改进信号传输方式，来提高中长期频谱的传输效率。通过10兆BASE-T光纤，短距离就可以传输100GB/s级别的数据，可谓是一举两得！不过随着2004年长篇论文公布，对传输速率的限制越来越难以满足需求。后来，迅速调整了标准，在800-1000GHz频段，达到了每秒数百兆的传输速率，基本满足通信、娱乐等长时间应用。

## 技术进步
近年来，中国国内外多方面试验的结果表明，中长期频谱的真正潜力还在后边。一些国家试图开发一个具有高灵敏度的新型载波收发机，以及与之配套的天线设计。另一些国家也在探索利用中长期频谱的其他特性，如在音频领域的应用。

# 7. 未来挑战
当前，超高频信号的取得主要依赖于通信网络。因为通信网络一般都是由覆盖整个宇宙的超高频信号传输系统构建而成。但是，这样的系统部署是一件非常困难的事情。首先，通信网络是由各种通信设备，包括终端设备、集线器、中继器、路由器等组成，它们之间的布线结构复杂，信号调制解调器和编解码器数量众多，部署安装非常困难。其次，通信网络中存在很多关键的基础设施设备，例如铺设各类通信线缆、维护各类传输设备的正常运行，这又需要大量的人工操作。再者，通信网络的设计也需要遵循很多公共服务设施的规范要求，比如电力供应、水利工程、环保管理等。因此，即便我们今天能够通过某种方法，把电磁信号传播到数百万公里的距离，构建起一个超高频通信网络，部署至全宇宙，也依然有巨大的技术和经济挑战。

另一方面，中长期频谱虽然已经成为物理界探索的一个热点，但与其说它代表着一个全新的分支，不如说它只是长久以来一直存在的一个小窗口。除了一些领先的国家，普通民众对于中长期频谱的应用还很少。如果想让这一切有益进一步推动，我们需要更多的科学家、工程师、政治家、法律家一起共同努力，共同探索这个新领域的前景。
