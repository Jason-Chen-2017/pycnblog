# 1.简介
  

Pandas 是 Python 中一个很流行的数据处理库，它提供高级数据结构 DataFrame，可以将多种类型的数据源（Excel、CSV 文件等）导入到同一个 DataFrame 中进行统一管理和操作，同时提供了丰富的数学函数以及数据分析工具，能帮助我们快速处理数据。
为了让读者更加熟悉 Pandas 的一些基础知识和操作技巧，我们将以 Python 数据处理与可视化系列之 Pandas 数据处理实战这一专题作为第一单元的内容，通过对 Pandas 操作和数据处理的讲解，希望能够让读者对 Pandas 有个整体上的了解，掌握一些数据分析常用技能。

文章主要基于 Jupyter Notebook 撰写，每一章节会独立成一篇文章，每篇文章都配有参考文献和推荐阅读材料。
