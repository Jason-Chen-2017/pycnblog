# 2.基本概念术语说明
## 2.1 什么是机器学习？
机器学习（英语：Machine learning），是一门新的基于人工智能的技术。它允许计算机从数据中提取知识，以此来改善自动系统的性能、减少错误，甚至创造出新的功能。机器学习的目标就是让计算机“自己”去发现数据中的模式、规律和关联，并据此做出正确的预测或者决策。机器学习模型可以分成两大类：

监督学习（Supervised learning）：又称为有监督学习（supervised learning）。该方法训练一个模型，使其能够从训练数据中学习到一个函数，这个函数能够根据给定的输入预测相应的输出。例如，某个模型可能接收到一条邮件的信息，然后通过分析信息里的内容判断是否属于垃圾邮件，或识别图片中的人脸特征，最终决定是否放进收件箱。监督学习的关键是提供足够数量的训练数据，其中包括已知的输入和对应的输出。如果没有足够的训练数据，那么训练出的模型可能就很差劲了。

无监督学习（Unsupervised learning）：无监督学习是指对输入数据进行分析时，不需要提供任何的标签或输出，仅仅是希望能够发现数据的结构或模式。例如，根据图像的原始像素，无监督学习模型可能会生成一系列不同的图像风格，使得同样的图片可以有不同的视觉效果。无监督学习模型也可以用来聚类，即将相似的对象归为一组，并找出各组之间的联系。

强化学习（Reinforcement learning）：该方法是机器学习的一个子领域，它试图最大化奖励。它把环境看作是一个状态空间和动作空间的序列，通过不断执行动作来收集数据，同时根据所得到的数据更新策略。这种方法常用于游戏和其他需要长时间决策的领域。

还有一些其他的机器学习模型，如集成学习（Ensemble learning）、半监督学习（Semi-supervised learning）等。

总之，机器学习是一种跨学科的交叉领域，涵盖统计学、算法、计算机科学、人工智能、心理学、工程学等多个领域。它的研究目标是为了使计算机具有自然学习能力，从而能够自动发现数据中的模式、规律，并通过有效的模型预测或辅助决策。

## 2.2 为什么需要机器学习？
机器学习所面临的挑战主要有以下几点：

1. 数据量过大：大量的数据是影响机器学习效率和效果的重要因素。由于数据量太大，传统上采用静态分析的方法无法处理如此庞大的海量数据。目前，许多公司和机构采用的是实时、流式的计算方式来处理这么多数据。实时计算需要大量的计算资源，而且还依赖于巨大的存储空间。所以，机器学习可以提高处理数据的效率，降低硬件成本，并且可以实时反馈结果。

2. 模型参数过多：对于某些模型来说，比如随机森林（Random Forest），每棵树都有很多参数需要调节。这导致调参过程非常耗时。传统上，需要人工逐个优化参数，找到最佳的参数组合，这是一个十分费力的过程。机器学习可以自动地对参数进行优化，这样就不需要人工参与这一过程。

3. 模型易受到噪声影响：许多数据集都存在噪声，它们既有真实的分布，也有一些随机扰动。传统上，统计学和机器学习模型会对噪声敏感，因为它们希望捕获真实的规律。但当数据噪声较大时，机器学习模型往往会过拟合，失去泛化能力。为了克服这一缺陷，一些机器学习模型会引入一些噪声机制，以增强模型鲁棒性。

## 2.3 机器学习模型的类型
机器学习模型大致可以分成三种类型：

1. 分类模型（Classification model）：分类模型根据输入变量的取值将其划分到不同的类别中，常用的分类模型有逻辑回归（Logistic Regression）、支持向量机（Support Vector Machine）、K近邻（K-Nearest Neighbors）、朴素贝叶斯（Naive Bayes）等。分类模型一般用于处理离散的输入变量，如文本分类、图像分类。

2. 回归模型（Regression model）：回归模型预测输入变量与输出变量之间关系的线性方程，常用的回归模型有线性回归（Linear Regression）、决策树回归（Decision Tree Regression）、梯度提升回归（Gradient Boosting Regression）等。回归模型一般用于处理连续的输入变量，如房价预测、销售额预测等。

3. 聚类模型（Clustering model）：聚类模型将相似的输入变量归为一组，常用的聚类模型有k-means聚类、层次聚类、谱聚类、GMM聚类等。聚类模型一般用于处理非结构化的输入变量，如文本聚类、图像聚类。

## 2.4 深度学习的概念
深度学习（Deep Learning）是一门建立在机器学习上的学习方法，深度学习通过深层次的神经网络来学习数据的特征表示形式。它的主要优点是学习特征抽象的能力，能够进行高级的图像和文本分析。

## 2.5 概念术语的扩展
除了上述概念和术语之外，本文还会涉及到其他一些术语。这里做一下简单介绍。

- 标注数据（Labeled Data）：包括训练数据、验证数据、测试数据。训练数据是模型用于学习的实际数据，验证数据用于评估模型在训练过程中的表现，测试数据用于评估模型在真实条件下的表现。

- 无偏估计（Unbiased Estimator）：无偏估计是说模型的预测结果与真实值之间的误差期望等于零，这意味着对于任意的样本，其估计值的平均误差不超过真实值。在统计学中，无偏估计被广泛地使用，特别是在假设检验、留一交叉验证、正则化等领域。

- 迷你批（Mini Batch）：迷你批是指一次性训练一小部分数据而不是一次性训练所有数据。迷你批的好处是减少内存消耗和提高训练速度，适合于在内存受限的情况下训练模型。

- 模型参数（Model Parameter）：模型参数是指学习算法在训练过程中通过迭代更新的模型内部参数。模型参数包括权重、偏置等。

- 学习率（Learning Rate）：学习率是指在每一步更新中，模型权重的变化幅度大小。学习率设置过大或过小都会导致模型的训练效果不佳。

- 激活函数（Activation Function）：激活函数是指神经元的输出值转换为其激活水平的非线性函数。常用的激活函数有sigmoid函数、tanh函数、ReLU函数等。

- 梯度下降法（Gradient Descent Method）：梯度下降法是指模型参数不断更新，使得损失函数最小的过程。最常用的梯度下降法是批量梯度下降法、小批量梯度下降法和随机梯度下降法。

- 权重衰减（Weight Decay）：权重衰减是指在每一步更新中，对模型权重进行惩罚，目的是防止过拟合。

- 批标准化（Batch Normalization）：批标准化是一种数据标准化方法，目的是使数据分布更加一致，并提升模型的训练速度和稳定性。

- 样本权重（Sample Weight）：样本权重是指每个样本的权重，目的是给不同样本赋予不同的重要程度。

- 学习速率调度器（Learning Rate Scheduler）：学习速率调度器是指在训练过程中随着训练的进行，调整模型的学习率，以达到最优的训练效果。

- 参数初始化（Parameter Initialization）：参数初始化是指模型参数在训练前的初始值设置，通常用于防止模型出现局部最小值或爆炸/饱和问题。

- 遗忘曲线（Forgetting Curve）：遗忘曲线是指在测试阶段，模型表现与训练数据规模之间的关系。如果模型的容量（capacity）足够大，其在测试数据上表现良好，而在较小的数据规模（如较少量的训练样本）下却表现较差，这说明模型在忘记较小的数据规模时遇到了困境，即出现了过拟合现象。

- 数据扩充（Data Augmentation）：数据扩充是指生成更多的训练样本，以提升模型的泛化能力。数据扩充的方法包括随机缩放、旋转、裁剪、光学变换、噪声添加等。
