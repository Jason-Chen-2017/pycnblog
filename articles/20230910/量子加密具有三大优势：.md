# 1.简介
  
性：量子计算在密码学中的重要作用就是对数据的安全性进行保证。它利用量子力学中所蕴含的物理特性来构造一个由量子门构成的无比复杂的电路网络，使得电路中的各种元件都可以量子化，从而使得整个电路运行在任意精度下都不会出现错误。其通过用量子态来代替纠缠在一起的二进制数据，实现信息的传输、存储和处理。因此，量子加密的基础也就不再是类ical的密码算法，而是量子计算的相关理论和实践。而量子计算还具备另一个重要特征——误差可控。当参与运算的量子系统之间存在信道时，由于各种噪声干扰等因素导致的量子漂移会带来不可避免的损失。因此，量子加密除了具有保密性，还具有可靠性。
# 2.容错性：量子计算的另一重要特征是容错能力。传统的密码算法只能保证在有限的周期内对数据的正确性进行校验，但是当发生错误时，将无法恢复出原始信息。而量子计算中的纠错编码器和纠错译码器可以有效地纠正传播过程中产生的噪声影响，确保数据在传输过程中准确无误地传递给接收方。另外，根据量子态的统计规律性质，对于重复的数据输入，量子算法也能够产生相同的输出结果。因此，量子加密算法可以在一定程度上提升信息的安全性。
# 3.隐蔽性：量子计算的一个更进一步的特征是隐蔽性。相比于传统的加密方法，它的通信过程更加隐蔽。信息发送者只需要将要发送的信息编码到量子态中即可，不需要担心信息的内容被窃听或者拦截。这是因为在实际的量子通信过程中，电磁波本身就已经隐蔽了原始信息。另外，基于量子的身份认证方法也是一种很好的隐蔽手段。通过对信道中的量子态的测量，可以判断对方是否真的是发来的信息。在现代量子通信中，使用最多的仍然还是BBS模型，这种方式要求双方事先建立起密钥，并将共享密钥进行配对。BBS协议同样具有良好的隐蔽性。总之，量子加密具有三大优势：
# 1.简洁性：量子加密算法简单易懂，而且由于采用了量子计算的相关理论和原理，所以可以简化或推广到更多应用领域。
# 2.高效率：量子加密算法的计算速度远超传统密码算法，尤其是在对大量数据进行加密时，性能优势非常明显。
# 3.抗攻击性：量子加密算法具有不可破解的安全性，即使攻击者拥有整个加密系统的所有元素（包括信道模型），也无法取得任何有效信息。
