# 1.简介
  

Michelson interferometer (MI) 是一种在高分辨率微电子显微镜下观察光波形的方法。它的原理是在微透镜（或透镜）两侧安装一块类似于太阳光斑点的导体，通过把微透镜前后的两束光线投射到不同的区域，就能够记录两个光线之间的相互影响情况，从而得到高分辨率的观测结果。由于它所依赖的微透镜具有独特的微电子结构，所以它可以让实验者在较小的尺寸上制成单个晶圆片，具有非常大的灵活性。在该文中，我们将讨论MICHELSON INTERFEROMETER (MI)，并尝试对其进行一个简单的介绍。

# 2.基本概念
## 2.1 什么是微透镜?
微透镜是由亚光子构成的微型透明镜。这种结构使得微透镜可以用来拍摄微小的光孔，这种光孔被称为“孔隙”。与普通的透镜不同的是，微透镜在聚焦时不像普通透镜那样会出现反射，所以在各个角度上都能够清楚地看到图像。通常情况下，在微透镜两侧安装有一个导体，以便在不同的方向上提供不同的入射光线。

## 2.2 什么是光波?
光是由两种物质组成——波和电场。我们所说的光波指的是电磁波的一种类型，它主要由两种振动模式组成：短波（λ）和长波（Ω）。这两种振动模式形成了一个称作光谱的三维空间，称作波色函数（Wavelength-Color Function），它反映了光的强度分布以及颜色分布。光波又分为四种主要属性——颜色、频率、波长、方向，这些都是我们通常所说的光波的基本概念。

## 2.3 Michelson 干涉仪器
MICHELSON 干涉仪器是由科隆尼茨的一个工程师 Leonard Michelson 于1910年设计开发出来。这个设备的主要目的是为了探索微透镜与微波（X-rays）之间的相互作用，特别是光栅上的微波的产生。他创造出了一个干涉仪器的构架，这个构架由两个微透镜（分别在左右两边）、一个调节机械臂、一个受损控制柱、一个滤波器、一个计数柱、一个微调整装置和一台主机组成。

当 X-ray 通过两个微透镜，进入到调节机械臂的驱动下，就可以记录两束光线之间不断变化的关系，并记录了两束光线的速度信息以及叠加后得到的光谱图。这样，Michelson干涉仪器就提供了一种低功耗、高分辨率的手段来研究宇宙中的微观结构。

# 3.相关概念术语
## 3.1 滤波器
滤波器是用电子元件去除杂散物和其他无用信号的过程。它是一种物理装置，能够使信号中存在的噪声最小化或者抑制。它的工作原理就是基于某些过滤条件来选择要保留还是要删除的信号。滤波器的功能有很多，比如信号的截断，信号的平滑处理，滤除干扰信号等。

## 3.2 计数柱
计数柱是一个装置，用于统计反射光在不同阶段分散到镜面上的次数，通常将计数值显示为柱状图。

## 3.3 光谱扫描
光谱扫描是通过不同光波长的微小变化来收集物体的光谱信息，其原理就是调整两条光线之间的距离，使得不同波长的光在不同的位置上被捕获。通过这种方式，光谱扫描器能够获得具有不同特征的物体的光谱。

## 3.4 显影胶
显影胶是一种特殊的激光设备，其特点是能够同时产生多条光线。它利用双向折射现象，把通过透镜的光线分别反射回来的光线再次照射到镜头上，形成了一系列条状的光线束。由于在透镜两端引入了导体，使得透镜系统可以接受不同方向的入射光线，因此称之为“MICHELSON INTEGRATION TECHNIQUE”（MICHELSON集成技巧）。

## 3.5 矢量晶体
矢量晶体是一种由离心破碎的氮原子构成的晶体。在矢量晶体中，所有原子都有空间的坐标形式，称之为晶格基矢量。矢量晶体的晶粒大小一般在几十纳米到几百纳米之间，具有良好的热容性。矢量晶体的分布均匀，不会出现孔隙的现象。因此，矢量晶体可以用作微透镜的材料，而且在镜面上就能够产生高清晰度的图像。

# 4.技术方案
本文将结合 MICHELSON 干涉仪器的原理，对其进行一个简单的介绍。首先，介绍 MICHELSON 干涉仪器的原理；然后，介绍它在光栅上的应用及其优缺点；最后，给出 MICHELSON 干涉仪器在多光源下的改进方案。

MICHELSON 干涉仪器的原理如下：
MICHELSON 干涉仪器是由两个微透镜和两个滤波器组成的设备。第一个微透镜在两侧安装了一块导体，当 X-ray 通过这两个微透镜的时候，光子就会在导体上跳跃，被反射或吸收，最终会进入到第二个微透镜。第二个微透镜则会将两束光线通过同一个方向传输，之后将两束光线的混合结果通过滤波器进行处理，并最终记录在计数柱上。通过不同光波长的微小变化，从而获得物体的光谱信息。

对于实施 MICHELSON 干涉仪器，主要需要以下几个步骤：

1. 准备晶体样品：先准备一个微透镜能够容纳的晶体样品，推荐使用微体或矢量晶体，晶体的厚度应在10nm左右。
2. 安装微透镜与导体：在晶体的两侧安装两个微透镜，微透镜的长度、厚度与样品相同。然后在微透镜的两侧安装一个类似于太阳光斑点的导体，导体的直径也应与样品相同。
3. 调整器件的安装：将导体固定在微透镜处，并将微透镜放置在距离不同距离的晶体层中。通过微调整，使两张微透镜之间的距离能够达到数百毫米的级别。
4. 启动 X-rays：调整好各个部件后，启动 X-rays。由于 X-rays 的传输速度很快，因此只需稍作等待即可得到结果。
5. 数据处理：数据处理的过程分为三个步骤：（1）读取数据；（2）计算数据；（3）分析数据。首先，读取数据是将记录的数字转换为可视化的图像。计算数据是对数据进行计算，计算得到的数据用于分析数据的目的。分析数据的过程包括数据可视化、数据分析、结论。
6. 结果展示：得到可视化的图像后，即可将数据输出到计算机上，或者打印出来，供其他实验人员观看。

MICHELSON 干涉仪器在光栅上的应用及其优缺点：

MICHELSON 干涉仪器在光栅上的应用主要表现在其独特的能力，能够记录准确的二极光谱图。它可以记录与光源直接接触的二极光谱，也可以记录两极反射的光谱。但是，由于这种能力的限制，MICHELSON 干涉仪器往往只能用于特定的实验室或工厂，不能普及开来。另外，MICHELSON 干涉仪器还存在着一定的杂波干扰，它会引入额外噪声。因此，为了保证结果的准确性，MICHELSON 干涉仪器的检测频率是有限的。

MICHELSON 干涉仪器在多光源下的改进方案：
为了解决 MICHELSON 干涉仪器存在的一定的局限性，我国科研部门和企业已经提出了许多改进方案，如电磁干涉仪器、堆积型干涉仪器、共模干涉仪器、小型互联型干涉仪器等，但它们都需要大量投入，且难以广泛应用。因此，我国政府决定推出新的干涉仪器模型——MICHELSON MULTI-SOURCE PROBE（MMP），这是一种高效、低成本、多光源的干涉仪器，可以在不同光源下任意组合不同波长的 X-rays。其原理如下：

1. 干涉仪器由两个微透镜和两个调整器件组成，其中，微透镜和导体一样，调整器件包括一个变速箱和一个微调整装置。
2. 在晶体的两侧安装两个微透镜，微透镜的厚度和直径与样品相同。并且安装相应的导体。
3. 每个光源作为输入，使用独有的微调整装置进行调整，使得光线的入射角度不同。
4. 使用getHost()方法，可以获取到X-rays，用于给两束不同波长的光线。
5. 当 X-rays 从多个光源输入到微透镜之后，光子就会进入到导体上跳跃，被反射或吸收，最终会进入到第二个微透镜。然后两束光线经过同一个方向传输，最终混合到一起。
6. 记录在计数柱上。
7. 对计数柱上的结果进行处理，计算出整个系统的光谱信息。
