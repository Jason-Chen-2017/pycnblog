### 5.1 开发环境搭建

本文所使用的开发环境为 Python，依赖库包括 Pandas、NumPy、Scikit-learn 等。

### 5.2 源代码详细实现

```python
import pandas as pd
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据集
user_data = pd.read_csv('user_data.csv')
course_data = pd.read_csv('course_data.csv')

# 构建用户画像
user_features = user_data[['age', 'gender', 'occupation']]
course_features = course_data[['difficulty', 'duration', 'rating']]

# 计算用户和课程特征向量的余弦相似度
similarity_matrix = cosine_similarity(user_features, course_features)

# 为用户推荐课程
def recommend_courses(user_index, similarity_matrix, course_data):
    scores = similarity_matrix[user_index]
    sorted_scores = np.argsort(scores)[::-1]
    recommended_courses = course_data.iloc[sorted_scores[1:11]]
    return recommended_courses

# 测试代码
user_index = 0
recommended_courses = recommend_courses(user_index, similarity_matrix, course_data)
print(recommended_courses)
```

### 5.3 代码解读与分析

该代码实现了基于用户画像的内容推荐功能。首先，加载数据集，构建用户和课程的特征向量。然后，利用余弦相似度计算用户和课程特征向量的相似度，得到相似度矩阵。最后，为指定用户推荐相似度最高的前10门课程。
