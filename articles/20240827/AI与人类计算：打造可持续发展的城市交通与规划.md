## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1. 数学模型构建

在城市交通规划中，常用的数学模型包括流量分配模型、路径优化模型和交通信号控制模型。以下分别介绍这些模型的构建过程。

#### 4.1.1. 流量分配模型

流量分配模型描述交通流量在城市道路网络中的分配过程。常用的流量分配模型包括：

1. **用户均衡模型**：假设每个用户在选择路径时都希望使得总时间最短，根据这一假设建立流量分配模型。
   $$ v_{ij} = \frac{p_{ij}}{\sum_{k=1}^{n} p_{ik}} $$
   其中，$v_{ij}$ 表示从路段 $i$ 到路段 $j$ 的流量比例，$p_{ij}$ 表示从路段 $i$ 到路段 $j$ 的路径费用。

2. **最小费用最大流模型**：在保证总流量一定的前提下，使总费用最小。模型公式为：
   $$ \min \sum_{i,j} c_{ij} v_{ij} $$
   $$ \text{s.t.} \sum_{j} v_{ij} = f_i, \sum_{i} v_{ij} = f_j $$
   其中，$c_{ij}$ 表示从路段 $i$ 到路段 $j$ 的费用，$f_i$ 和 $f_j$ 分别表示路段 $i$ 和路段 $j$ 的流量需求。

#### 4.1.2. 路径优化模型

路径优化模型用于确定从起点到终点的最优路径。常用的路径优化模型包括：

1. **最短路径模型**：Dijkstra 算法和 A* 算法是最常用的最短路径模型。Dijkstra 算法公式为：
   $$ d(j) = \min_{i} (d(i) + w(i,j)) $$
   其中，$d(j)$ 表示从起点到终点 $j$ 的距离，$w(i,j)$ 表示从路段 $i$ 到路段 $j$ 的权值。

2. **动态规划模型**：动态规划模型用于解决多阶段决策问题，其公式为：
   $$ V_{i,j} = \min_{k} (V_{i,k} + C_{k,j}) $$
   其中，$V_{i,j}$ 表示从起点到终点 $(i,j)$ 的最优费用，$C_{k,j}$ 表示从阶段 $k$ 到阶段 $j$ 的费用。

#### 4.1.3. 交通信号控制模型

交通信号控制模型用于设计自适应交通信号控制策略，提高道路通行效率。常用的交通信号控制模型包括：

1. **定时控制模型**：定时控制模型基于固定的时间间隔，调整交通信号灯的配时。模型公式为：
   $$ t_{i,j} = t_{i,j0} + \alpha \cdot n $$
   其中，$t_{i,j}$ 表示从路段 $i$ 到路段 $j$ 的交通信号灯配时，$t_{i,j0}$ 表示初始配时，$\alpha$ 表示调整系数，$n$ 表示当前时段。

2. **自适应控制模型**：自适应控制模型基于实时交通数据，动态调整交通信号灯的配时。模型公式为：
   $$ t_{i,j} = t_{i,j0} + \alpha \cdot f(t) $$
   其中，$f(t)$ 表示实时交通流量，$\alpha$ 表示调整系数，$t_{i,j0}$ 表示初始配时。

### 4.2. 公式推导过程

#### 4.2.1. 用户均衡模型推导

假设有 $n$ 条路段，每条路段的流量需求为 $f_i$，交通网络中的流量分配满足以下条件：

1. 流量守恒：对于每条路段 $i$，有 $\sum_{j} v_{ij} = f_i$。
2. 流量比例：对于每条路段 $i$ 和 $j$，有 $v_{ij} = \frac{p_{ij}}{\sum_{k=1}^{n} p_{ik}}$。

将流量比例代入流量守恒条件，得到：
$$ \sum_{j} \frac{p_{ij}}{\sum_{k=1}^{n} p_{ik}} = f_i $$

化简后得到用户均衡模型：
$$ v_{ij} = \frac{p_{ij}}{\sum_{k=1}^{n} p_{ik}} $$

#### 4.2.2. 最短路径模型推导

假设有 $n$ 个节点，每两个节点之间的距离为 $d_{ij}$。从起点 $s$ 到终点 $t$ 的最短路径可以表示为 $P^*$，满足以下条件：

1. 最短路径：对于每条路径 $P$，有 $d(P) = \sum_{i,j} d_{ij}$。
2. 无环路径：路径 $P$ 中不存在重复节点。
3. 子路径最短：对于路径 $P$ 中的每个子路径 $P_i$，有 $d(P_i) \leq d(P)$。

根据条件 1 和 3，可以得到以下递推公式：
$$ d(P^*) = \min \{ d(P) \mid P \text{ 是从 } s \text{ 到 } t \text{ 的无环路径} \} $$

初始状态 $d(s) = 0$，$d(t) = \infty$。利用递推公式，可以得到最短路径 $P^*$。

#### 4.2.3. 定时控制模型推导

假设有 $n$ 个路段，每个路段的交通信号灯配时为 $t_{i,j}$。初始配时为 $t_{i,j0}$，调整系数为 $\alpha$。

根据交通流量预测，可以得到每个路段的流量预测值 $f(t)$。调整后的配时为：
$$ t_{i,j} = t_{i,j0} + \alpha \cdot f(t) $$

其中，$f(t)$ 可以表示为实时交通流量，$\alpha$ 为调整系数。调整系数 $\alpha$ 需要根据实际情况进行设置，以实现自适应控制。

```python
import numpy as np
from scipy.sparse.csgraph import dijkstra

# 1. 建立交通网络模型
graph = np.array([[0, 3, 1, np.inf, np.inf],
                  [3, 0, 2, 1, 4],
                  [1, 2, 0, 5, 2],
                  [np.inf, 1, 5, 0, 3],
                  [np.inf, 4, 2, 3, 0]])

# 2. 初始化路径
start = 0
end = 4

# 3. 路径搜索
distances = dijkstra(graph, start)
path = [start]
current = start

while current != end:
    next_node = np.argmin(distances[current, :])
    path.append(next_node)
    distances[current, :] = np.inf
    current = next_node

path = np.array(path)
print(f'Optimal Path: {path}')
print(f'Path Length: {np.sum(graph[path[:-1], path[1:]])}')
```
