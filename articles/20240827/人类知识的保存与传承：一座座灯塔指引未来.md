### 2.1 知识图谱

知识图谱是一种结构化、语义化的数据模型，用于表示实体、属性和关系。通过知识图谱，人类可以以一种直观、可扩展的方式理解和处理复杂的知识体系。

```python
import matplotlib.pyplot as plt

# 创建图
G = nx.Graph()

# 添加节点和边
G.add_nodes_from(['实体1', '实体2', '实体3'])
G.add_edges_from([('实体1', '实体2'), ('实体2', '实体3')])

# 绘制图
nx.draw(G, with_labels=True)
plt.show()
```
