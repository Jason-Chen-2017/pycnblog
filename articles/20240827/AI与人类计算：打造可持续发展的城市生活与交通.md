## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

在城市生活与交通领域，常用的数学模型包括交通流量预测模型、路径规划模型和信号控制模型等。

#### 4.1.1 交通流量预测模型

1. **线性回归模型**：
   $$y = \beta_0 + \beta_1 x$$
   其中，$y$ 表示交通流量，$x$ 表示影响因素（如时间、天气等），$\beta_0$ 和 $\beta_1$ 为模型参数。

2. **神经网络模型**：
   $$y = \sigma(\beta_0 + \beta_1 x)$$
   其中，$\sigma$ 表示激活函数，如 sigmoid 函数或ReLU函数。

#### 4.1.2 路径规划模型

1. **Dijkstra算法**：
   $$d(s, v) = \min_{u \in predecessors(v)} (d(s, u) + w(u, v))$$
   其中，$d(s, v)$ 表示从起点 $s$ 到终点 $v$ 的最短路径长度，$w(u, v)$ 表示边 $(u, v)$ 的权重。

2. **A*算法**：
   $$f(v) = g(v) + h(v)$$
   其中，$f(v)$ 表示从起点 $s$ 到终点 $v$ 的估价函数，$g(v)$ 表示从起点 $s$ 到节点 $v$ 的实际路径长度，$h(v)$ 表示从节点 $v$ 到终点 $v$ 的估计路径长度。

#### 4.1.3 信号控制模型

1. **线性规划模型**：
   $$\min z = c^T x$$
   $$\text{subject to } Ax \leq b$$
   其中，$x$ 表示信号灯的切换时间，$c$ 和 $A$ 分别表示目标函数和约束条件。

2. **动态规划模型**：
   $$V_t(x) = \min_{a_t} \{c_t(x, a_t) + V_{t+1}(x')\}$$
   其中，$V_t(x)$ 表示从状态 $x$ 在时刻 $t$ 开始的最优价值函数，$c_t(x, a_t)$ 表示在时刻 $t$ 在状态 $x$ 下采取行动 $a_t$ 的成本，$x'$ 表示采取行动 $a_t$ 后的状态。

### 4.2 公式推导过程

以交通流量预测模型为例，我们以线性回归模型进行推导。

1. **模型假设**：
   假设交通流量 $y$ 是影响因素 $x$ 的线性函数，即 $y = \beta_0 + \beta_1 x$。

2. **最小二乘法**：
   使用最小二乘法求解模型参数 $\beta_0$ 和 $\beta_1$。
   $$\beta_1 = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n} (x_i - \bar{x})^2}$$
   $$\beta_0 = \bar{y} - \beta_1 \bar{x}$$
   其中，$n$ 表示数据点的个数，$\bar{x}$ 和 $\bar{y}$ 分别表示影响因素 $x$ 和交通流量 $y$ 的平均值。

3. **模型验证**：
   通过计算预测值与实际值之间的误差，评估模型性能。

#### 5.2.2 路径规划

```c++
#include <iostream>
#include <vector>
#include <queue>
#include <unordered_map>
#include <cmath>

using namespace std;

struct Node {
    int id;
    vector<pair<int, int>> neighbors; // (neighbor_id, weight)
    bool visited = false;
};

int dijkstra(const vector<Node>& graph, int start, int end) {
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
    unordered_map<int, int> distances;
    distances[start] = 0;
    pq.push({0, start});

    while (!pq.empty()) {
        auto current = pq.top();
        pq.pop();
        int current_id = current.second;
        if (current_id == end) {
            return distances[end];
        }
        if (graph[current_id].visited) {
            continue;
        }
        graph[current_id].visited = true;
        for (auto& neighbor : graph[current_id].neighbors) {
            int neighbor_id = neighbor.first;
            int weight = neighbor.second;
            int distance = distances[current_id] + weight;
            if (distance < distances[neighbor_id]) {
                distances[neighbor_id] = distance;
                pq.push({distance, neighbor_id});
            }
        }
    }
    return -1;
}

int main() {
    int n, m, start, end;
    cin >> n >> m >> start >> end;
    vector<Node> graph(n);
    for (int i = 0; i < m; ++i) {
        int u, v, w;
        cin >> u >> v >> w;
        graph[u].neighbors.push_back({v, w});
        graph[v].neighbors.push_back({u, w});
    }
    int result = dijkstra(graph, start, end);
    cout << "The shortest path length is: " << result << endl;
    return 0;
}
```
