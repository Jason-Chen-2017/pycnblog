## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

小语言模型的数学模型主要基于神经网络和深度学习技术。其核心数学模型包括：

1. **词向量表示**：词向量表示是自然语言处理的基础，常用的方法有Word2Vec、GloVe等。
2. **神经网络结构**：神经网络结构主要包括输入层、隐藏层和输出层。常用的神经网络结构有循环神经网络（RNN）、长短时记忆网络（LSTM）、门控循环单元（GRU）等。
3. **损失函数**：损失函数用于评估模型的预测结果，常用的损失函数有交叉熵损失函数等。

### 4.2 公式推导过程

以Word2Vec为例，其核心公式推导如下：

1. **词向量表示**：

   假设词汇表V中有N个词，对于词w，其对应的词向量表示为$$ \mathbf{v}_w \in \mathbb{R}^d $$，其中d为词向量维度。

   $$ \mathbf{v}_w = \text{Word2Vec}(w) $$

2. **词向量计算**：

   对于一个句子$$ s = w_1, w_2, ..., w_n $$，其对应的词向量表示为$$ \mathbf{v}_s = [ \mathbf{v}_{w_1}, \mathbf{v}_{w_2}, ..., \mathbf{v}_{w_n} ]^T $$。

   $$ \mathbf{v}_s = \text{Word2Vec}(s) $$

3. **损失函数**：

   假设预测词向量$$ \mathbf{v}_\hat{w} $$，目标词向量$$ \mathbf{v}_w $$，损失函数为交叉熵损失函数：

   $$ L = -\sum_{w \in s} \log P(w|\mathbf{v}_\hat{w}) $$

   其中，$$ P(w|\mathbf{v}_\hat{w}) $$为词w在给定预测词向量$$ \mathbf{v}_\hat{w} $$下的概率。

### 5.2 源代码详细实现

以下是一个简单的Word2Vec算法的Python实现：

```python
import numpy as np
import gensim

def word2vec(sents, size=100, window=5):
    model = gensim.models.Word2Vec(sents, size=size, window=window)
    return model

sents = ["我喜欢苹果", "苹果很甜", "我喜欢吃苹果"]
model = word2vec(sents)
print(model.wv["苹果"])
```

### 5.3 代码解读与分析

- **数据预处理**：使用gensim库的`word2vec`函数，将句子列表`sents`作为输入，进行词向量嵌入。
- **词向量计算**：调用`word2vec`函数，生成词向量模型`model`。使用`model.wv`访问词向量。
- **词向量输出**：打印词向量，如`print(model.wv["苹果"])`。
