## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

样本重要性估计工具的核心数学模型如下：

$$
I_i = \frac{\sum_{j=1}^{n} \left| \frac{\partial J}{\partial X_{ij}} \right|}{\sum_{i=1}^{m} \sum_{j=1}^{n} \left| \frac{\partial J}{\partial X_{ij}} \right|}
$$

其中，$I_i$ 表示样本 $i$ 的样本重要性，$J$ 表示模型的损失函数，$X_{ij}$ 表示样本 $i$ 中的特征 $j$。

### 4.2 公式推导过程

假设我们有 $m$ 个样本，每个样本包含 $n$ 个特征。在训练过程中，模型对每个样本的损失函数为 $J_i$，其梯度为 $\frac{\partial J_i}{\partial X_{ij}}$。

为了计算样本的重要性，我们需要对每个样本的梯度进行求和，并除以所有样本的梯度求和。这样可以得到每个样本的重要程度。

### 5.2 源代码详细实现

```python
import tensorflow as tf
import numpy as np

# 生成模拟数据集
X = np.random.rand(1000, 10)
y = np.random.rand(1000, 1)

# 定义模型
model = tf.keras.Sequential([
    tf.keras.layers.Dense(units=1, input_shape=(10,))
])

# 编译模型
model.compile(optimizer='adam', loss='mse')

# 训练模型
model.fit(X, y, epochs=10, batch_size=32)

# 计算样本重要性
with tf.GradientTape(persistent=True) as tape:
    predictions = model(X)
    loss = tf.reduce_mean(tf.square(predictions - y))

gradients = tape.gradient(loss, model.trainable_variables)
importances = np.mean(np.abs(gradients), axis=1)

# 输出样本重要性
print(importances)
```

3. 编译模型：使用 Adam 优化器和均方误差损失函数编译模型。
4. 训练模型：使用训练集训练模型。
5. 计算样本重要性：使用梯度信息计算每个样本的重要性。
