# 1.简介
  

Sublime Text是一个非常流行、功能强大的文本编辑器。Sublime Text具有跨平台兼容性，安装方便、快捷键简单易用等特点。熟练掌握Sublime Text可以极大提高工作效率和产出。本文将会教你如何安装和配置Sublime Text。同时也将会给出一些基本的功能操作方法。让读者对Sublime Text有一个初步的了解和认识。
# 2.软件准备
首先下载并安装最新版Sublime Text，官方网站地址为https://www.sublimetext.com/。
# 3.软件基本设置
1.菜单栏：默认情况下，Sublime Text的菜单栏分为File、Edit、View、Selection、Find、Command、Tools、Help六个区域。其中，最常用的有File（文件）、Edit（编辑）、View（视图）、Find（查找）。

2.状态栏：状态栏显示了当前文件的路径、光标位置、编码方式等信息。

3.主题设置：你可以通过菜单栏中的Preferences -> Settings -> Color Scheme选定自己喜欢的主题，也可以直接在菜单栏中输入View -> Syntax Theme选定。

4.安装插件：如果你需要更多更好的功能，可以通过菜单栏中的Preferences -> Package Control安装插件。

# 4.常用快捷键
下面是一些Sublime Text的常用快捷键，大家可以在日常使用的过程中熟悉并掌握：

1.打开、关闭文件：Ctrl+Shift+N或Ctrl+P

2.保存：Ctrl+S

3.另存为：Ctrl+Shift+S

4.复制：Ctrl+C

5.粘贴：Ctrl+V

6.剪切：Ctrl+X

7.删除行：Ctrl+Shift+K 或 Shift+Del

8.跳转至行首：Home

9.跳转至行尾：End

10.向上滚动：Ctrl+Up

11.向下滚动：Ctrl+Down

12.查找：Ctrl+F

13.替换：Ctrl+H

14.选择多行：Shift+方向键

15.扩展/缩小选择：Ctrl+L

16.注释/取消注释：Ctrl+/ 或 Ctrl+Shift+/ 

17.运行/停止：Ctrl+B

18.编译：Ctrl+Shift+B

19.自动完成：Tab 

# 5.基础操作技巧
下面介绍几个基础的操作技巧：

1.使用标签页：你可以创建多个标签页来同时查看不同的文件，每个标签页都可以独立地进行编辑、运行代码或者编译等操作。

2.代码折叠：你可以点击行号旁边的箭头来折叠代码块。

3.代码跳转：你可以使用Ctrl+P快速打开项目中的文件并跳转到指定行。

4.代码格式化：你可以使用Ctrl+Shift+P调出菜单，然后搜索“format”并执行Format Document命令来自动调整代码格式。

# 6.其他注意事项
下面是一些Sublime Text使用时的注意事项：

1.不要频繁切换标签页：为了防止标签页混乱、内存占用过高，建议你在长时间操作时不要频繁切换标签页。

2.避免混用不同编码方式的文件：如果你的Sublime Text里既包含中文字符又包含非ASCII字符，那么可能会出现乱码的问题。
