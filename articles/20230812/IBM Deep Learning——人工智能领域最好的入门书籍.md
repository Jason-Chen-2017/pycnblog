# 2. 概念与术语

## 2.1 深度学习

深度学习（Deep Learning）是一种基于神经网络的机器学习方法。它所涉及的概念和理论都比较复杂，本文只讨论其中的基本概念和术语，更详细的内容请参考深度学习的相关文献。

### 2.1.1 神经网络

人类大脑包含超过一百亿个神经元，但只有约7千万个感受神经元对外部世界刺激做出反应。对于大多数生物体来说，大部分时间都处于不活跃状态。因此，大脑中的大部分神经元都处于无意识状态。

而人工神经网络（Artificial Neural Network, ANN），则模仿这种生物神经元的工作方式，包含许多互相连接的节点，每个节点对应于生物神经元中的一个感受器。当输入信息进入网络时，网络中的节点会加权求和然后激活输出信号，模仿神经元对刺激的响应过程。这样，人工神经网络就可以学习从输入到输出的映射关系。


ANN有三层结构：输入层、隐藏层、输出层。输入层接受外部输入信号，输出层计算结果，隐藏层则负责转换输入信号并传递给下一层。每个节点都会计算自己的权重，然后将信息发送给相邻的节点。这种结构可以表示非常复杂的函数关系，并且能够处理复杂的非线性关系。

### 2.1.2 感知机

感知机（Perceptron）是最简单的形式的神经网络模型，它只有输入层、输出层和隐藏层。其结构如图2-1所示。输入层接收外部输入信号，输出层根据输入信号是否满足某一阈值判断是否传输信息，隐藏层则不参与计算。输入信号通过权重与激活函数处理后，再送至隐藏层或输出层。

感知机是神经网络的起源，但是它的缺点也十分明显。它只能解决线性可分的问题，即输入信号的特征向量可以被划分为两个区域，不能表示更加复杂的非线性关系。而且无法学习异或问题、半导体纠缠等非线性问题。所以，近年来随着神经网络的发展，很多其它模型出现，例如卷积神经网络（Convolutional Neural Networks, CNNs）。

## 2.2 其他重要术语

### 2.2.1 TensorFlow

TensorFlow是Google开源的深度学习框架，具有以下特点：
- 支持跨平台：支持Windows、Linux、macOS等操作系统。
- 灵活：可以定义和训练任意的神经网络模型。
- 可移植：兼容不同硬件设备，可运行在CPU、GPU、FPGA等不同芯片上。
- GPU加速：可以利用GPU进行分布式训练，有效降低训练时间。
- 文档完善：有丰富的文档教程和示例代码。

TensorFlow使用Python编写，支持广泛的深度学习模型，例如：
- 卷积神经网络（CNNs）；
- 循环神经网络（RNNs）；
- 递归神经网络（RNNs）；
- 变压器网络（VAEs）；
- GANs；
- 强化学习；
- 和更多其他的模型类型。

### 2.2.2 Keras

Keras是另一种基于Theano或TensorFlow之上的高级神经网络 API，它提供了快速简便的方法来构建、训练和部署模型。它具有以下特点：
- 友好易用：提供简单直接的API接口。
- 跨平台：可以在不同的操作系统和硬件平台上运行。
- 可移植：可部署在桌面端、移动端、服务器端或云端。
- 轻量级：没有过多的依赖项。

Keras适用于以下任务：
- 基于数组或列表的模型定义；
- 训练与测试模型；
- 保存和加载模型；
- 序列化模型；
- 在端到端系统中集成多个模型；
- 对深度学习模型的可视化与解释。
