# 1.简介
         

在自然语言处理中，命名实体识别（Named Entity Recognition）是指识别文本中的人名、地名、机构名、日期、时间等命名实体。其目的是从文本中提取出有意义的信息，方便后续分析或索引。本文将详细阐述如何使用NLTK工具包进行命名实体识别。

1.1什么是命名实体？
命名实体就是一些具有特定意义的词汇或者短语，例如人名、组织机构名、国家名、城市名等，这些词汇通常被标注为相应的命名实体类型，如PERSON、ORGANIZATION、LOCATION、DATE、TIME、PERCENT等。命名实体一般可以划分为如下四类：

1.人物类命名实体：包括人员姓名、职称、职务等。如“张三”、“李某”、“经理”、“教授”、“研究员”。

2.地点类命名实体：包括地址、门牌号码、邮政编码、天气、景点、商圈等。如“北京市海淀区丰台科技园”、“上海市浦东新区六里桥路96号”、“广州市黄埔区徐泾滩”、“成都市高新区天府大道北段575号”、“52度广场”等。

3.组织类命名实体：包括公司、政府、大学、银行、媒体、运动队等。如“中国人民银行”、“北京大学”、“华为技术有限公司”、“南京航空航天大学”、“中国国际法院”等。

4.事件类命名实体：包括事件、事故、日期、时间等。如“八一二疫情”、“六月份左右”、“星期五晚上九点”、“2020年9月1日”等。

总之，命名实体的发现可以帮助我们更好地理解文本信息的含义、分类和结构，进而完成更多有价值的任务。

1.2为什么需要命名实体识别？
如果我们想对文本进行深入分析，需要进行命名实体识别。命名实体识别有如下几种用途：

1.信息检索：命名实体识别技术能够把上下文关联起来的词汇组合起来，提升搜索引擎的检索效果。

2.知识抽取：对于某些领域特定的主题词、情感词等，基于命名实体识别技术可以从文本中抽取其对应的知识。

3.文本分类：通过命名实体识别技术，可以根据文本内容自动确定其所属类别。

4.文本自动摘要：由于命名实体可能包含较多冗余信息，所以对命名实体的摘要提取非常重要。

5.文本生成：利用命名实体识别技术，可以实现由文本的命名实体生成新的文本。

因此，命名实体识别具有很强的实用价值，是一种用于文本处理的有效工具。

# 2.基本概念术语说明
## 2.1 文件读写
首先，我们需要对文件进行读写操作。在Python中，文件的读取与写入可以使用open()函数。
```python
file = open("example.txt", "r") # 以只读方式打开example.txt文件
content = file.read() # 将文件内容读到变量content中
print(content) # 打印文件内容
file.close() # 关闭文件
```
这里，"example.txt"是待读取的文件路径，"r"表示以只读的方式打开文件。使用file.read()方法可以获取文件的内容，并赋值给变量content。最后，调用file.close()方法关闭文件。

## 2.2 字符串分割
为了方便处理字符串，我们还需要对字符串进行分割操作。在Python中，字符串的分割可以使用split()函数。
```python
string = "hello world"
words = string.split() # 以空格为分隔符，分割字符串
print(words) # 输出["hello", "world"]
```
这里，"hello world"是一个待分割的字符串。调用string.split()方法可以按照空格为分隔符，将字符串分割成多个单词，并保存在列表words中。然后，我们就可以遍历列表words，对单词进行处理。
