# 1.简介
  

​        中国高校近年来招收研究生的计划日益激烈，有越来越多的中、美、港澳台等重点高校在陆续开设本科生课程及研究生课程。其中，985工程院校和211工程院校的研究生名额却逐渐饱和，形成了很大的资源浪费。为了更好的解决这一问题，减少资源浪费，降低留学生的压力，中国教育部决定启动「985、211研究生名额大幅上调」政策。这项政策将于今年底实施，但也存在着诸多难题，包括如何平衡留学生入学率、优质高校建设和自主招生的诉求。据统计，截至2021年7月底，全国共有超过6万余所普通高校、900多所985研究生院校、3000余所211研究生院校，其中985/211双学位名额总量分别约为1.5万/5000，占到了国内高校研究生名额的六分之一左右。虽然有985、211双学位名额激增的趋势，但是目前仍有不少中小学校或院校担心名额过少，无法满足学生的需求。因此，我们认为，要解决「985、211研究生名额大幅上调」问题，关键还在于引导高校不断扩大对本科生和研究生教育的投入，逐步缩小本科生规模，提升研究生学历结构，实现大学生就业的均衡化。
# 2.相关概念和术语
​        985工程(国家聘用培训基地)和211工程(院校自主招生工程)是中国政府设立的两套名校培养体系。目前，大多数高校都已经具备985工程和211工程各自的条件。985工程工程师将接受985学历认证；而211工程工程师则是在本科学习期间就可以参加研究生学习的。当研究生完成本科学习后，可以申请转入211工程进行研究工作。
​        在实行985、211工程时，各个高校根据自己的实际情况在985工程或者211工程中设置相应的项目，并进行相应的培养。一般来说，进入985工程的高校将有更多的MBA、PHD、JD等国际顶尖学府硕士项目、有较强的国际竞争能力，并且能够选择较好的专业方向；而进入211工程的高校将有丰富的研究经历、开放的研究环境、可自由选择研究方向、研究生招生渠道更加多样。

# 3.核心算法原理与具体操作步骤
​        在实施「985、211研究生名额大幅上调」政策之前，首先需要确定985/211双学位的分配比例，即双学位每年招收研究生的名额占全日制本科生的名额的比例。中国教育部在2020年下达通知，「我国985/211高校研究生招生资格比例应按国内高校综合实力平均值每年配额（大约每年每个学校可派出一定数量的研究生）进行调整」。这是基于近几年国内高校研究生招生数量逐渐增加的背景下，通过整体考虑社会经济状况、市场需求以及高校资源配置等因素，对双学位的分配比例进行动态调整。目前，2021-2022学年期间，已将985/211双学位名额分配比例由原先每年约为1:2提升到1:2.5。

​        根据教育部《关于推进985、211高校研究生招生制度的意见》，将从2021-2022学年开始实施的一系列政策目标，主要包括：

1. 更大范围的招生，增加择优录取机制。在符合研究生标准、具有突出的创新精神和学科核心竞争力、长期承接国家重要项目的同时，还将鼓励不同性别、不同背景、不同年龄段的学生融入研究生的招生队伍，增加同类学生的比例。
2. 提升毕业生待遇。减少对985、211本科生提供助学金等福利待遇，吸纳国内外高水平人才，确保所有学科、专业的优秀学生都得到充分的机会。
3. 扩大教育融合能力，鼓励高校和科研机构的合作。鼓励高校与科研机构互联互通，共同推进教育的长远发展。
4. 完善培养方案。完善985、211研究生培养方案，适应学科发展、高校发展的需要，强化选修课、通识教育和奖学金等制度。


由于历史原因，目前高校的录取比例偏高，导致留学生的积极性不足，需要引导高校在扩大招生规模的同时，也注重提升研究生的学历水平。例如，2020年，清华大学学生入学最多的专业是计算机科学，在全国高校研究生中占有三分之二。同时，很多学校借鉴国际顶尖学府模式，多开设本科生课程，还将大部分研究生选修课程外包给国际院校。这些举措既有助于高校进一步扩大本科教育规模，又鼓励研究生在同类专业中找到突破口，相互促进。

当前的名额分配比例为每年分配两个5%左右的名额，也就是说，每年有近四分之一的学生的录取名额都被占满。这个比例在很大程度上依赖于社会经济发展速度，高校招生条件也会受到越来越多学校的影响。随着今后研究生课程数量的增加、专业教育规模的扩大，名额分配比例会逐渐增大。而在2021-2022学年期间，高校每年的研究生规模预计将达到两千人左右，再加上国内外顶尖学校的优秀学生，名额总量估计将超过3万。如果按照传统的平均分配比例，每年分配的5%左右的名额仍然太少。因此，提升名额分配比例，更高效地利用名额资源，是「985、211研究生名额大幅上调」政策的关键。

另外，中国高校正在进行一系列的扶持计划，以吸引优质人才。这些扶持计划包括定向资助、入学优惠、就业支持等。例如，《2020年高校毕业生回国留学人员管理办法》提出，今后到访的留学人员可以在完成留学手续的同时，获得海外优先、免疫检验的旅游住宿优惠、住房补贴、入学优惠等福利。此外，各高校也在积极探索创新招生机制，如提供线上线下交流、自由竞赛、竞赛奖励等方式，鼓励学生在学习中积累经验。

```python
import numpy as np

x = [1, 2, 3]
y = [2, 4, 6]

coefficients = np.polyfit(x, y, deg=1) # 拟合直线的函数参数，deg为拟合阶数

print("方程: ", coefficients[1], "x +", coefficients[0])

# 拟合曲线
xs = np.array([i for i in range(min(x), max(x))])
ys = coefficients[1]*xs+coefficients[0] 

plt.plot(xs, ys, label="拟合曲线") 
plt.scatter(x, y, color='red', marker='+', label='原始数据') # 用红色星型标记原始数据点
plt.title('数据拟合')
plt.xlabel('X轴')
plt.ylabel('Y轴')
plt.legend()
plt.show()
```
