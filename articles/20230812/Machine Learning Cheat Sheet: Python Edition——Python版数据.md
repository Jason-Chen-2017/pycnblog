# 1.简介

# 2.机器学习术语
## 2.1 数据集（Dataset）
数据集指的是用于训练或测试模型的数据集合。通常来说，数据集分为两种类型：监督数据集和非监督数据集。监督数据集由输入数据与输出标签构成，即样本(sample)和标记(label)。而非监督数据集则没有标签，仅由输入数据构成。在实际应用中，通常会将数据分割为训练集、验证集和测试集。

## 2.2 特征向量（Feature Vector）
特征向量是一个数值向量，描述了一个样本的某些特征。它可以是离散的或者连续的，由属性(attribute)组成。特征向量通常是原始数据的抽象表示，是模型进行分析、预测的基础。

## 2.3 目标变量（Target Variable）
目标变量也称类别变量，是要预测的输出变量。它可以是连续的也可以是离散的，通常是因变量。

## 2.4 模型（Model）
模型就是用来对数据进行分析、预测和训练的东西。常用的模型包括线性回归、逻辑回归、支持向量机、决策树、随机森林、神经网络等。

## 2.5 假设空间（Hypothesis Space）
假设空间是指所有可能的模型函数的集合，每个模型函数都对应着一种模型结构。它确定了模型可能遇到的各种情况。

## 2.6 超参数（Hyperparameter）
超参数是指模型训练过程中的不可学习的参数。例如，线性回归模型中的截距项(intercept)就是一种超参数，因为它的值不能通过观察数据直接获得。超参数的选择对模型的性能影响很大，通常需要使用交叉验证等方法进行调参。

## 2.7 代价函数（Cost Function）
代价函数衡量模型的好坏，可以是某个误差函数的总和，也可以是某种度量标准。

## 2.8 精度（Accuracy）
精度是指模型预测的准确率。它是预测正确的正负样本占比。

## 2.9 损失函数（Loss Function）
损失函数是指模型的输出结果与真实结果之间的差距。

## 2.10 迭代（Iteration）
迭代是指模型从初始状态开始，利用训练数据不断调整参数，直到模型达到最佳效果。迭代次数越多，模型的训练效率越高，但是过多的迭代可能会导致欠拟合或过拟合的问题。

## 2.11 贝叶斯估计（Bayesian Estimation）
贝叶斯估计是统计学的一个方法，用于计算未知参数的概率分布。该方法建立在贝叶斯定理之上，认为已知数据上的似然比值等于未知数据的似然比值乘以先验概率。

## 2.12 最大似然估计（Maximum Likelihood Estimation）
最大似然估计是统计学的一个方法，用于找到观察到的数据的最佳参数取值。在统计学中，观察到的数据往往服从某种统计分布，因此可以使用极大似然估计法来找出这些分布的最佳参数。

## 2.13 EM算法（Expectation Maximization Algorithm）
EM算法是一种迭代算法，用于求解含有隐变量的概率模型参数。该算法是一个二阶段过程，首先固定模型的其他参数，然后利用期望步骤计算隐变量的期望值，然后再根据期望值更新模型参数。

## 2.14 概率密度函数（Probability Density Function）
概率密度函数（PDF）描述了一个随机变量取不同值的可能性，通常是连续的或者离散的。在概率论中，一个随机变量的概率密度函数是指对于不同的取值 x ，该变量落在某个特定区间 [x_1, x_2] 中的概率。

## 2.15 似然函数（Likelihood Function）
似然函数是一个统计概念，它表示了模型对观测数据的拟合程度，是一种对数似然值。如果模型的参数θ的似然函数是L(θ)，那么θ的最大似然估计参数θ* = argmaxL(θ)，可以通过优化的方法得到。

## 2.16 凸函数（Convex Function）
定义在Rn上的函数f(x)为凸函数，当且仅当对任意的0<a1, a2,..., an<|x|, 有：f(ax+b) ≤ af(x)+b−af(a)/(|a|-1), f'(x)>0。也就是说，对于任何给定的常数倍数a，都有f(ax+b)的下降速度小于f'(x)*|a|，其中f'(x)是f在x处的一阶导数。
