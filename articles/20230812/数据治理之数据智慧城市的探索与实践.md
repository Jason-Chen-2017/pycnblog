# 1.简介
  

随着人们生活节奏的加快、物联网、智能设备的普及、社会的不断进步、信息化进程的推进等，传统的数据管理方式已经逐渐告别舒适区。传统数据管理的方式存在很多不足，比如效率低、收集成本高、价值低、信息质量差、处理速度慢、存储空间小、数据孤岛等问题。如何有效利用数据、洞察人心、提升竞争力、实现实体经济转型是一个重要课题。

互联网+数据智能的到来带来了许多新的机遇和挑战，其中最重要的是利用大数据的新模式探索新的商业模式。近年来，随着大数据、人工智能、云计算、大规模计算技术的迅速发展，人们对数据的分析和挖掘变得越来越感兴趣。数据智慧可以帮助企业更好的理解客户需求、做好产品开发、降低运营成本、提升产品品质、优化服务质量等，从而实现实体经济的转型升级。

在现代经济中，“数据智慧”已成为一种制度性的重要工具。企业需要建立起数据资产管理系统，确保数据的安全、准确性、完整性和可用性。要实现数据智慧，企业首先要对自己的数据进行分类、归纳和管理，根据数据价值的不同，进行价值评估和风险控制，并制定数据价值管理策略。同时，还要保证数据信息共享、数据流动畅通、业务决策合理、技术创新顺利进行。

那么，什么样的数据才算是数据智慧呢？这个问题的回答将通过三个层面的叙述来阐释。

# 2.数据类型
数据智慧是由四个部分组成的数据。即结构数据（Structured Data）、半结构数据（Semi-structured Data）、非结构数据（Unstructured Data）和混杂数据（Mixed Data）。四种数据类型的定义如下图所示。
结构数据：结构数据包括数字、文本、音频、视频、图像等可被计算机直接识别、存储和操作的数据，如Excel表格、CSV文件、数据库、电子表格等。

半结构数据：半结构数据主要指那些非结构数据的集合，或者是结构化数据经过编码后形成的数据，如XML、JSON、HTML文档、网页、Word文档等。

非结构数据：非结构数据包括诸如电子邮件、语音、文本、图像、音频、视频等媒体及其组合形式的数据，其特征是大小不固定、不规则、分布广泛、跨平台等。

混杂数据：混杂数据通常是各种不同的数据结合在一起形成的一块数据，如各种文档、电子邮件、照片、视频、音频、文本等。它既包括结构数据也包括非结构数据。例如：语义Web数据（Semantic Web data）。

# 3.数据领域划分
数据智慧可以分为三类：位置数据智慧（Location Intelligence），社交网络数据智慧（Social Network Intelligence），行为数据智慧（Behavioral Intelligence）。三类数据智慧都可以分为多元化、协同化、精细化三个层次。

## （1）位置数据智慧
位置数据智慧可以为企业提供有关用户在何时何地访问某应用或网站的信息。利用位置数据智慧可以精准定位活动热点，为公司提供基于用户行为的定向推送，提升用户体验；也可以为企业提供相关政策、产品推荐、促销活动等信息，改善营销效果。但目前，位置数据智慧仍处于初级阶段。它的优势在于实时性强、精度高、费用低，但缺陷在于隐私保护、数据中心建设等方面还有待解决。

## （2）社交网络数据智慧
社交网络数据智慧包括分析用户关系网络、挖掘用户喜好、推荐商品和服务。通过了解用户的喜好、行为习惯、认识热点、提升品牌知名度、推广产品等，可以让企业与用户保持沟通互动，提升品牌影响力；还可以通过分析社交网络上的事件动态、热点话题、用户关系链等，提升用户满意度和生命体验。但由于缺乏公开的平台和标准，以及不同行业的不同特点导致功能缺失、资源匮乏等问题。

## （3）行为数据智慧
行为数据智慧能够根据用户的搜索记录、浏览记录、行为轨迹等信息，精准匹配用户的需求，推荐相关产品和服务。它可以帮助企业分析用户的使用习惯、操控用户的情绪、了解用户的心理状态、帮助用户找到喜欢的健康食谱，提升用户的生活品质。但行为数据智慧目前也处于初级阶段。它的优势在于能够有效挖掘用户偏好、习惯、喜好，缺陷则在于应用场景单一、数据获取难度大。
