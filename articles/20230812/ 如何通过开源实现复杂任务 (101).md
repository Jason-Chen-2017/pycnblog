# 2.基本概念术语说明
## 2.1 计算机科学及相关词汇
计算机科学（Computer Science）:研究如何用计算设备解决实际问题的方法。它包括计算机系统的设计、开发、维护、应用、运算、通信和存储等方面。计算机科学的研究目标是开发具有高效率、鲁棒性、适应性、可靠性、安全性和用户友好性的计算机系统和程序。

CPU(Central Processing Unit): 中央处理器，通常就是指个人电脑中的CPU。它的功能是进行算术运算、逻辑运算、控制运算、输入输出操作等。

RAM(Random Access Memory):随机存取存储器，它是由一个个大小一致的动态RAM芯片组成的，用来临时保存CPU执行指令和数据的。

ROM(Read Only Memory): 只读存储器，主要存储着启动装载程序、BIOS程序、操作系统内核等静态代码。

GPU(Graphics Processing Unit):图形处理器，用来执行图像和视频渲染、动画制作、CAD绘画、物理模拟等高速计算。

Internet of Things(IoT): 物联网，是一种将互联网技术和传感器技术结合使用的新兴网络，可以收集、存储、传输和处理大量的感测数据，同时利用这些数据进行远程控制和自动化管理。

云计算(Cloud Computing):云计算是一种将数据中心和服务放在雲端，通过第三方服务提供商访问的方式，将用户数据、应用服务和计算资源共享给用户的方式。目前主要有Amazon Web Services(AWS), Microsoft Azure, Google Cloud Platform等云服务平台提供。

数据结构（Data Structure）: 数据结构是组织数据的方式，也是指相互之间关系和联系的集合。数据结构分为抽象数据类型和数据对象两个层次。

算法（Algorithm）: 算法是指用于完成特定任务的一系列指令或者算子，在计算机中用来操纵数据以达到预定目的。

## 2.2 Python编程语言及相关词汇
Python编程语言：是一个高级、跨平台的 interpreted, interactive 和 object-oriented 的编程语言。它的最大特点是简洁的语法，可以快速上手。它广泛运用于各种领域，包括 web 应用、机器学习、web 爬虫、网络数据处理、网络游戏、金融科技等。

PEP(Python Enhancement Proposals)：Python增强建议，是社区提出的关于Python改进建议的规范文档。PEP 8 是 Python编码风格规范，它是针对Python代码的约定俗称。PEP 20 是 Zen of Python 的一句话，用于表述Python编程的准则。

Anaconda：是一个基于Python的数据科学包管理系统，支持Windows、Mac和Linux，提供了超过1700种开源数据科学包，并可以创建环境和安装包。

IPython：是一个交互式命令行工具，用来更方便地使用Python语言。

Jupyter Notebook：是一个基于Web的交互式笔记本。它具备可视化组件、代码编辑、运行、交互、公式编辑、历史记录、分享、可伸缩性等特性。

IDE(Integrated Development Environment，集成开发环境): 集成开发环境，又叫集成开发环境（Integrated Development Environment，IDE），是指一套完整的开发工具集，包括编译器、调试器、版本管理工具、文本编辑器、图形界面开发工具等。Python的集成开发环境有IDLE、PyCharm、Spyder、Eclipse等。

GitHub：是一个面向开源及私有软件项目的代码托管平台，可 以让开源者向世界展示自己优秀的开源项目。通过GitHub，程序员和其他技术人员可以便捷地把代码上传、修改、协作开发。

Virtualenv：是一个独立于系统Python目录之外的Python虚拟环境，可以用来隔离不同程序之间的依赖关系。

Git：是一个开源的分布式版本控制系统，用来管理代码。

Numpy：是一个开源的、用于科学计算的库，提供了高效的矢量、矩阵、向量运算函数。

Matplotlib：是一个用Python编写的2D绘图库，支持高度可定制的视觉效果，并提供多个图表类型。

Pandas：是一个开源的、用于数据分析、统计计算和数据挖掘的库。

Scikit-learn：是一个基于Python的开源机器学习库，用于分类、回归、聚类、降维、模型选择、特征工程、模型评估、信息论等任务。

TensorFlow：是一个开源的、用于机器学习的高性能数值计算框架。

Keras：是一个高级的神经网络API，基于Theano或TensorFlow构建。

OpenCV：是一个开源的、用于图像处理和计算机视觉的跨平台的机器视觉库。

NLTK：是一个开源的、用于自然语言处理的库，提供了一系列接口，用来处理文本数据、进行处理等任务。
