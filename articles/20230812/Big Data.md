# 1.简介
  

大数据（英语：big data）指的是指产生海量的数据，数据量大到超出了可管理的范围，对数据的处理、分析和挖掘变得十分困难，同时需要快速有效地进行大规模计算。数据采集、存储、处理、分析、挖掘和传播都成为当今企业面临的主要挑战。随着大数据技术的不断革新，越来越多的人关注如何从大数据中提取商业价值，如何利用大数据促进社会效益。本文将从基础的大数据概念和技术层面阐述大数据应用的相关知识和实践。
# 2.基本概念
## 数据收集及存储
1.什么是数据？
数据是指通过一定方式获得的关于客观事物的客观记录，是人们为了获取知识、解决问题以及提高工作效率而进行的一系列活动所产生的各种形式的数字和非数字信息。

2.什么是数据仓库？
数据仓库是一个独立于源系统的基于中心库、集成数据库和星型模式的总体数据集，用于支持决策支持、分析、报告和BI工具。它支持复杂查询，支持安全访问，并提供直观的视图显示。它从源系统获取数据，进行清理、转换、集成等处理后，存入集中的数据库或数据文件，供其他应用系统使用。

3.什么是NoSQL？
NoSQL（Not Only SQL，泛指非关系型数据库），即“不仅仅是SQL”的意思。它与关系型数据库不同，NoSQL拥抱非结构化、动态的数据模型。传统的关系型数据库采用预定义的表结构，每行数据都是固定格式的；而NoSQL数据库则允许动态设计表结构，灵活调整字段类型，更加灵活。NoSQL数据库通常由键-值、列族、文档、图形数据库组成。

4.什么是云计算？
云计算是一种利用互联网计算资源的网络服务。其核心是把服务器作为一个资源池，客户可以按照需求购买这些资源，只付费按使用时长结算，无需承担昂贵的IT支出。云计算提供的服务包括虚拟机、存储、网络、数据库、应用服务等。它使得用户可以快速部署应用程序、扩展容量、弹性伸缩，满足用户个性化的业务需求。

5.什么是Hadoop？
Hadoop是一个开源的分布式计算平台。它利用可靠的存储和分布式计算功能，提供海量数据的并行运算能力。它最初被用来进行网络搜索，但现在已逐渐成为大数据领域的标配技术。

6.什么是MapReduce？
MapReduce是一种编程模型，它把大型数据集拆分成独立的块，然后并行处理每个块，最后再合并结果。MapReduce框架用简单的编程接口实现，并充分利用集群的硬件资源。

7.什么是数据湖？
数据湖（英语：data lake）是指在现代数据仓库建设中经过长时间的积累、汇聚、整理、清洗、加工等处理后的海量数据存储区域，具有高度的非关系、半结构化特征。

8.什么是ETL？
抽取、传输、加载（英语：Extract Transform Load，简称ETL）是一个数据集成过程，其目的是将来自不同源头的数据转化为标准化、清洗、转换后的格式，并最终导入到数据仓库或数据湖中。

9.什么是数据采集？
数据采集是指从各类数据源（如数据库、应用程序、文件、网络等）收集、整理、处理的数据，对数据进行有效的存储、组织、检索和应用。它主要涉及三个环节：数据采集端、数据传输端和数据存储端。

10.什么是ETL工具？
ETL工具是用于实现数据采集、数据传输、数据存储等功能的专用软件。它们能够自动完成数据处理、校验、转换、连接、过滤等工作。

11.什么是Lambda架构？
Lambda架构是一种事件驱动的数据架构模式，用于解决数据流处理与数据存储之间的延迟问题。它由两个相互独立的功能组件组成，分别为数据采集层（即实时数据源）和批处理层（即离线数据源）。

12.什么是Hive？
Hive是一个基于Hadoop的开源分布式数据仓库软件。它通过SQL语言提供交互式的查询功能，能够将结构化数据映射成一个带有列名和结构的表格。Hive能够自动执行 MapReduce 任务，并生成优化的运行计划。

13.什么是HBase？
HBase是一个基于 Hadoop 的 NoSQL 数据库，它是一个结构化的、非关系型、分布式的数据存储系统。它采用高可用性的主/副本机制，支持海量数据存储，并且提供了行级别的查询功能。
