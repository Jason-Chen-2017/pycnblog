# 1.简介
  

## 概述
机器学习(Machine learning)是计算机科学的一个分支，是一类通过训练来提升计算机性能的算法和技术的集合。它主要用于解决非结构化和半结构化数据、缺乏特定领域知识的任务以及高维数据的分析与分类问题。2014年伯克利大学机器学习教授斯坦福·格雷厄姆博士提出了“机器学习”一词，并指出“机器学习”是一个定义模糊且充满挑战性的领域，其研究方向极广泛，涉及方方面面。
ICML（国际机器学习会议）是美国机器学习领域最具权威的学术期刊。每年都会举办一系列具有重大影响力的顶级会议，包括NIPS、ICLR（即上海交通大学自然语言处理实验室发起的2019年AI顶会）、AAAI（即中国人工智能学会组织的2017年亚太区顶会）。本文所要讨论的会议就是ICML 2017，该会议于2017年在美国加利福尼亚州纽约市召开，由ACM主办。
ICML的出版物包括论文集、全文索引、会议通知书等。作为经典的机器学习会议，ICML还会举行Workshop，提供热烈的交流与分享机会。在机器学习的推动下，新的算法、方法、工具层出不穷。ICML会议也积极探索新兴的机器学习应用，如自动驾驶、视觉、文本、生物信息等领域，促进科研合作和行业交流。
## 主题
近几年来，ICML已经成为国内外顶级机器学习学术会议，其覆盖的主题从基础理论到最新技术都非常丰富。除了传统的会议主题之外，ICML还开设了“Machine Learning for Healthcare”, “Machine Learning in Finance,” “Machine Learning for Cancer,” “Privacy and Security in Machine Learning,” “Applications of Deep Learning to Medical Image Analysis,” “Big Data and Machine Learning,” “Deep Generative Models and Applications.”等主题的分论坛。
本文以国际顶级会议ICML 2017为例，结合机器学习的相关理论和技术，将ICML的主题划分为以下四个部分进行阐述。

1.深度学习综述：ICML 2017的第一项主题是深度学习综述，以深度学习的历史、发展及其关键应用为主线，深入探讨深度学习的理论、算法、应用以及未来可能出现的变化。

2.智能决策：ICML 2017的第二项主题是智能决策，将传统的决策论、强化学习、贝叶斯统计、机器学习方法等纳入综合考虑，以指导应用开发者、公司运营者和学术界对未来的智能决策方向进行思考。

3.人机交互：ICML 2017的第三项主题是人机交互，围绕用户界面设计、认知心理学、行为经济学等方面，讨论人类与机器沟通、协作、学习的方式、技术发展、误判风险、注意力缺陷等现象。

4.可解释性与倾向性：ICML 2017的第四项主题是可解释性与倾向性，系统阐述了如何理解深度学习模型的预测结果，如何做好预测的可解释性工作，以及倾向性分析的现状与前景，以促进学术界和产业界的共同关注。
