# 1.简介
  

## Seaborn是一个基于Python的开源数据可视化库，用于statistical data visualization的一站式工具箱，提供了简单易用的接口用于数据可视化。
相比于Matplotlib、Plotly等传统的可视化库，Seaborn具有更高级的接口设计和功能，提供了更加便捷的绘图方式、统计函数及主题自定义能力。Seaborn可以帮助我们快速地创建漂亮的、复杂的数据可视化图表。通过本文我们将带您进入Seaborn的世界，并掌握其强大的绘图技巧。
## 1.1 Seaborn项目简介
### 1.1.1 主要特点
#### 1.1.1.1 可读性强
Seaborn内部采用了简洁的API语法，让你能专注于数据的探索与呈现。在熟悉API后，你可以轻松绘制各种统计图表，无需关心繁琐的编程过程。
#### 1.1.1.2 数据类型支持广泛
Seaborn支持pandas DataFrame、numpy arrays、Categorical变量、时间序列数据以及scipy的sparse matrices，可以直观地呈现出不同的数据类型。
#### 1.1.1.3 样式丰富多样
Seaborn提供了大量的预设主题，满足不同的应用场景。另外，还内置了一些常用统计图表模板，让你的可视化作品脱颖而出。
#### 1.1.1.4 交互式可视化
Seaborn的内建支持能够使得数据可视化过程更流畅。你不仅可以轻松地创建可交互的图形，也可以用鼠标或触控板对它们进行缩放、旋转、移动，从而得到更多的分析信息。

# 2.基本概念术语说明
## 2.1 什么是数据可视化？
数据可视化（Data Visualization）就是将复杂的数据转换成图表形式，用于描述、分析、发现和 communication 的工具。它是一种非常重要的工作。有了图表，才可以更好地了解、分析和表达数据。数据可视化最重要的作用之一是：用图形呈现数据信息，并直观地展示出来，让人们更容易理解、发现数据背后的规律和模式，从而产生更好的决策。当然，数据可视化不是一蹴而就的，它需要结合专业的统计学知识、计算机科学知识以及商业分析理念，才能发挥最大的价值。
## 2.2 为何要使用Seaborn?
在数据可视化领域，许多工具都可以满足需求，但是当数据量过大时，往往需要借助机器学习、深度学习等算法模型对数据进行降维处理，才能得到可视化效果；而Seaborn提供便捷的可视化接口，将复杂的数据集可视化呈现出来。同时，它还具备其他可视化工具所不具备的扩展性，如对交互式图表的支持，并且拥有很丰富的主题选项，满足多种类型的可视化需求。所以，在数据量较小、简单的数据可视化场景下，使用matplotlib或其他的工具即可完成任务，但在数据量较大、复杂的数据分析场景下，使用Seaborn则更为合适。
## 2.3 Seaborn的特点
### 2.3.1 简单易用
Seaborn内部采用了简洁的API语法，让你能专注于数据的探索与呈现。在熟悉API后，你可以轻松绘制各种统计图表，无需关心繁琐的编程过程。
```python
import seaborn as sns

tips = sns.load_dataset("tips")

sns.boxplot(x="day", y="total_bill", hue="sex", data=tips)
plt.show()
```
