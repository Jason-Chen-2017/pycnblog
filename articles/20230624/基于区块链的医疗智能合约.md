智能合约是一种基于区块链的程序，可以自动执行合约条款。智能合约可以通过编程语言实现，例如Solidity、Vyper等。

去中心化是指区块链技术采用分布式存储，没有中央机构控制。在区块链中，每个节点都有相同的账本，因此可以实现去中心化的交易和身份验证。

分布式账本是指区块链是一个分布式的数据库，每个节点都有相同的账本。因此，可以实现去中心化的交易和身份验证。

### 4.2 应用实例分析

下面是一个简单的医疗智能合约的示例：

```
pragma solidity ^0.8.0;

contract MyContract {
    mapping (address => mapping (address => uint256)) public balances;
    uint256 public totalSupply;
    constructor() public {
        totalSupply = 10000000;
        balances[msg.sender] = totalSupply;
    }

    function buy() public payable {
        require(msg.value > 0, "Insufficient amount");
        require(address(this).balance >= msg.value, "Total balance is too low");
        require(balances[msg.sender] >= msg.value, "Insufficient balance");
        balances[msg.sender] -= msg.value;
        balances[address(this)] += msg.value;
    }

    function sell() public {
        require(msg.sender == address(this), "Only owner can sell");
        require(balances[msg.sender] >= 0, "Insufficient balance");
        balances[msg.sender] -= msg.value;
        balances[address(this)] += msg.value;
    }
}
```

这个智能合约实现了一个医疗支付系统。当一个患者需要购买药品时，可以通过智能合约中的buy()函数发送支付信号，使得智能合约可以自动计算药品的供应量，并将药品发送给患者。当患者需要出售药品时，可以通过sell()函数发送出售信号，智能合约也可以自动计算药品的供应量，并将药品发送给医疗机构。
