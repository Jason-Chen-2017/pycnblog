# 1.背景介绍


## 什么是化学气相沉积？
在电子工业的历史上，由于金属与铁质间存在着一种相互作用，导致铁层含有一种特殊的材料叫做 化学气。这种化学气会随时间的推移而不断地增殖、腐蚀和损害电子芯片的性能。因此，电子工业必须正确处理化学气对电子芯片的影响。虽然在设计阶段需要注意减少化学气的生成，但在制造过程中也不可忽视其对电子芯片性能的影响。
电子芯片制造过程中的化学气相沉积可分为两种情况：一种是产生于放电过程中，另一种是电池充满过程中。前者属于材料级生物反应，后者属于电路级生物反应。以下将分别讨论这两种情况的发生与反应。
### 1.1 放电过程中的化学气相沉积
在电子芯片放电过程中，会产生一种称为“放电气”的微量化学物质，它经由机械加工成长并与电流连在一起形成一个电极，并且会跟电流平行运动。当电流进入电极时，放电气就会和电流密切相关，使得电极失去它原本的直线形状。

如图1所示，在电极与电源之间包裹着一个保护层，当电流与放电气相互作用时，保护层里的气体也跟着发生变化。在某些情况下，放电气甚至会将保护层完全吸入电极中。

通过观察电流密封电极内的放电气，可以发现它有一些具有涨落性的特征。一旦放电气的温度升高或降低，它就可能会引起电极内部的温差变化，使其形状发生变化。在电极表面还存在着一些小孔，这些孔能够形成一种气泡。如果电流频繁的交换，这些气泡也会聚集在孔洞中，形成更多的气泡并增加对周围环境的污染。

除了“放电气”，还有一些其他化学气也可能参与到放电过程中。例如，随着电流的增大，放电层会迅速膨胀，形成一种称为电磁辐射的现象。在这些电磁辐射中，会释放出一种金属颗粒，该颗粒的大小介于微米和毫米之间，具有极强的透射性。这种颗粒会吸附在电极表面并吞噬住其他物质。除此之外，还有一些其他微量化学物质也会加入放电过程中，它们通常不会对电流产生直接影响，但是却可以加剧放电气的膨胀效应。

### 1.2 电池充满过程中的化学气相沉积
在电池充满过程中，也会产生化学气，它主要包括两种形式，即一般物质性气体和重金属类气体。一般物质性气体有氧气、酸雷罗素、三氟乙醛、氮气等；重金属类气体有金刚石、铝锂土、硫酸钠等。

一般物质性气体的成份包含氢核苷酸、碳氢化合物、氮气等等，这些化合物聚集在电池表面形成液体，会吸附在离子层与表面交界处形成一个气泡状区域，称为物质性气泡。由于电池充满过程中的电场力作用，物质性气泡也会跟随电流移动。

重金属类气体的成份主要是重金属，例如铝、铜、锌等，这些重金属自然会结合在一起形成很强的电荷相互作用。在充满过程中，这些电荷的聚合会形成一个被称为电子气团的微型结构。这个电子气团的尺寸很小，可以忽略不计。由于电场力的作用，电子气团也会随着电流的移动而移动，形成一个气团状区域，称为电子气团气泡。

不同种类的化学气相沉积在不同情况下发生，这里只对放电过程中产生的化学气进行了深入研究。由于不同的化学气具备不同的特性，在制造电子芯片时应该根据其特性选择适宜的方法。
