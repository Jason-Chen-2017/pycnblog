# 1.背景介绍


半导体（英语：Semiconductor）通常指含有电荷或其它化学物质的金属材料；由于其特定的性质（光学、电学、磁性等）使它成为工业界、科学界和学术界研究的热点。随着人们对半导体技术的了解越来越深入，在半导体的制备、工艺及应用领域也有了越来越多的探索和实践。通过阅读本文，读者将能够：
1.了解半导体的基本原理，包括物理、化学、微观和宏观方面；
2.理解相关知识和技术在真空器件、二极管、硅片、功能区块、集成电路（IC）、FPGA、三维元件、超导材料、等领域中的应用；
3.掌握重要的技术工具和设备，包括玻璃模具、研磨机、光刻胶加工设备、LTO、晶圆波纹溯相技术、掩膜模拟技术、电镀设备、TEM/EDX、LWD等；
4.为自己的职业生涯规划一个良好的学习路径，并充分利用自己的兴趣和能力，提升职场竞争力。

## 2.2. 光学特性
光学特性：描述半导体的能量分散特性，即反射、吸收和透射等现象的特征。

常见光学特性：
- 内层：截止于零，材料中不存在衍射或吸收。
- 中层：越靠近表面的材料，衍射或吸收能力越强，能量传递速度也越快。
- 边缘层：介于中层与外层之间的层。
- 外层：与表面接触的层，其截止光波为零。
- 分层：介于内层与外层之间的层。
- 拟制边缘层：介于两层之间。
- 可裂变：可裂变材料拥有不断向两个方向反转的分层形态，可让许多分子同时流动。

## 2.3. 电学特性
电学特性：描述半导体的电荷运动特性，即电子运动、核化学反应、辐射等现象的特征。

常见电学特性：
- 单层（单极）材料：电容性材料，指电荷相互排斥且带隙地分布，一般用于存储电力、传输信号等用途，其典型代表是锗和铝。
- 二层（双极）材料：金属-绝缘子材料，指绝缘子作为介质、带隙的金属片嵌入到一个导体中，其电压分布随导体中电荷的移动而变化。典型代表是集电芯和绝缘子材料。
- 多层材料：非金属二极化材料，如二氧化硅材料，其电导率随温度的变化而发生变化，具有较高的耐受性和耦合能力，常用于集成电路设计和嵌入式系统中。

## 2.4. 功耗特性
功耗特性：描述半导体在特定电源条件下的性能，例如功耗、复合度、尺寸比等。
