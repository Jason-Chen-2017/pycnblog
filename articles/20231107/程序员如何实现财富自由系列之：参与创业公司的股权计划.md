# 1.背景介绍



股权投资（equity investment）是指利用股东持有的股份、债券或其他权益资产，按照规定的规则和价格，在市场上进行价值投资的方式，通过公司股权的转让来获得收益。股权投资的优点是收益稳定性高、风险低，适合对企业经营有激励的创业初期，能够短时间获得大额的回报。

如今的互联网行业蓬勃发展，信息化服务带来了巨大的生产力提升，亟须寻找新的增长点，比如创业板成为一个崭新的起点。那么就需要寻找对创业公司有吸引力且有潜力的项目，积极参加相关的股权投资计划，通过定向增发股票，扩大公司的股权结构，增加其股东的参与感。

# 2.核心概念与联系
## 什么是股权计划？

股权计划是一种融资工具，是指企业或个人向银行申请将其拥有的特定股权出售给第三方投资者的计划。投资者可选择购买股权的方式来实现股权投资，包括直接购买股权、委托第三方机构购买股权或者设立担保品种，将股权置于有价值的股东手中。这些计划都要遵循国家法律、监管要求和相关政策，并具有一定的固定目标和展望。根据不同的国家法律，股权计划的具体形式会有所不同。例如，在美国，股权计划一般需要由经过背景调查和资格审查的投资人提交申请，通过公开招标方式获得资金支持。另外，中国国内的股权投资也较为复杂，涉及到的许多环节均需制定相应的制度。

## 创业公司的股权计划

目前，在美国和中国，很多创业公司都在创办过程中都会考虑到自身的发展方向，并且设立相应的股权计划，加入到股权结构中。例如，Facebook在2004年推出“股权计划”，建立了一个名为“Facebook Ventures”的基金，用于筹集Facebook未来的发展资金。后来，Facebook又加入了另一个名为“Commonwealth Fund”的基金，相当于创始人的股权，主要用于投资价值观领域的企业、文化创意和基础设施建设等领域的创新。另外，一些在华为、阿里巴巴、腾讯、百度等知名科技公司，也曾开启过股权计划，不过都是些小小的项目。

因此，具备基本知识储备，做好准备，并与相关部门沟通，可以尝试参加创业公司的股权计划，扩大企业的股权结构，激活创业公司的资本动能，提高企业的利润率。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解
## 概念阐述

为了更好的理解股权投资的机制，先从以下几点定义了解一下：

1.股权：股权就是创业公司的所有者对企业拥有的普通股的占有量，占有份额越大代表股权的价值越高。

2.普通股：公司派生的，用来支付股息的股份，普通股由股东按照基本薪资每年按一定比例分配，每年发放股息。

3.持股者权益：是指该股份所对应于股东的权益。

4.分红：为创业公司所有者分派现金的一种方式，是公司每年支付股东的一种稀释补偿。

5.申购：向上交换普通股的行为，即向公司提交关于申请购买普通股的请求，公司接受股东的申请并公示相关的条件之后，股东可以向上交换普通股。

6.赎回：向下交换普通股的行为，即向公司索要普通股，并且按照某些条件退还股款。

7.增发：对已经发行的普通股进行增发，增加其数量，给予更多的股份配套上的权益。

8.转让：出售股权，把某一部分股权转让给另一位股东的行为。

9.优先股：是一种特殊的股份，拥有优先于普通股的投资份额，可以用于优先受益于创业公司的资本运作。

## 操作步骤

- **申购**

创业公司在IPO时申购成千上万的股份，其中一部分作为优先股。优先股由于其独特的受益权利，可以得到一定比例的分红，而且优先股持有者在申购普通股之前，优先先质押优先股。

- **认购**

一般情况下，创业公司会设定一定的认购门槛，只有满足条件的投资者才可以购买公司股权。投资者必须缴纳保证金才能购买股权，并承诺按时返还保证金，否则视为默认放弃，自动退出股权池。

- **增发**

对于有能力的创业公司来说，每年都会增发股份，确保公司在资本运作中的决策权力不至于落入某个具体的人手里，避免出现垄断的局面。对于不想承担增发费用的创业公司来说，可以通过借贷资金的方式，把增发后的股份转让给投资人。

- **流通盘**

虽然每个投资者都可以购买股权，但是每股股价还是有限制的，最高不超过创始人的债务，因为不能滥用股权，所以创始人将控制股权流动的部分，称为流通盘。

流通盘是指创业公司当前拥有的全部股份总额，而股东只能取得少部分股权，不能将全部股权卖出，这一部分股权称为流通股。

## 分红政策

分红政策又称赔付规定，是指创业公司支付给股东的股息，分为现金分红和实物分红两种，现金分红即收取股东名下的现金收入作为分红，实物分红则是将股权销售凭证或发放到股东手中，作为奖励。

分红政策的确定会直接影响到创业公司的盈利情况，如果分红太少可能会导致投资者望而却步，分红太多可能导致股东少得到股息，造成融资难题。

一般来说，分红政策包括年终分红和半年度分红，年终分红每年支付年终奖励，半年度分红每半年支付股东相应的股息。在分红政策确定之前，创业公司并不会公布多少分红，只会给出每个月收入分红的信息。

## 股权转换

股权转让是指把创业公司的股权卖给另一家公司，创业公司的股东仍然拥有股权。通常，股权转让可以通过股东之间的协议完成，也可以在创业公司内部讨论决定，但转让之前，必须满足一定的条件。

## 债务溢働

债务溢働是指股权投资产生的损失超过了初始投资时所支付的初始债务，这种情况往往是由两个原因引起的：一是项目执行出现困难，二是竞争对手把股权收购了。

解决债务溢働的方法包括：

- 提前偿还债务：提前偿还债务可降低公司及股东的损失。

- 拓展业务范围：拓展业务范围可以消除对原有产品及服务的依赖，增加公司与股东的回报。

- 寻找专业人士参与投资：找到专业人士参与投资可以分享公司管理权力，防止自己被别人牵制。
