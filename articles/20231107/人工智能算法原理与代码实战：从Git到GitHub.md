# 2.核心概念与联系
## 2.1 什么是机器学习？
机器学习（Machine Learning）是一种自然界（包括人类）与计算技术相结合的方式，它使计算机能够学习并改善它的行为，以获得所需的预测能力。简单来说，机器学习就是让计算机从海量数据中找寻 patterns ，根据 patterns 的结果进行决策或者改善性能。传统的编程，人工智能（AI），统计学习，模式识别都属于机器学习的范畴。

## 2.2 为什么要用机器学习？
由于现代信息技术的飞速发展，越来越多的数据被产生出来，而这些数据在海量的数据之下，依旧没有任何的价值。因此，我们需要用机器学习的方法，去发现这些数据的 patterns ，从而提高我们的效率，更好的服务于客户。另外，随着互联网的普及，越来越多的人开始了数据驱动的创新。所以，机器学习也成为当今最热门的分支之一。

## 2.3 如何选择合适的机器学习算法？
为了选择合适的机器学习算法，我们首先应该明确我们的目的。例如，如果我们想要预测用户对某个产品的喜欢程度，那么我们就可以选择回归算法；如果我们想要预测股票的价格波动，那么就需要选择时间序列预测算法；如果我们想要区分图片中的猫和狗，那么就可以选择图像识别算法。之后，我们还可以结合业务逻辑，比如产品推荐场景，我们可能选择协同过滤算法；对于电商场景，我们选择个性化排序算法等等。
