# 1.背景介绍


随着技术的进步，人们对信息技术的需求也越来越高，包括物联网、虚拟现实、移动互联网等等，而在信息传输、数据处理方面，传统的信息通信技术仍然占据主导地位。传统的信息通信技术以电磁波、光电信号等物理信道为载体进行信息传输。虽然目前有一些成果提出了利用微波、红外线等非电磁信道进行通信的可能性，但这些方案都是超越了信息论和计算复杂度的限制，而且仍然存在着各种各样的问题。因此，为了解决传统的信息通信技术所带来的种种问题，研究人员开始探索新的无线通信技术，比如基于量子信道的通信技术。

量子通信（Quantum Communication）是一个跨界的学科领域，其主要涉及两方面的研究：量子纠缠论（quantum optics）和量子信息论（quantum information theory）。其中，量子纠缠论中的量子纠缠态（quantum entanglement state），利用量子力学中描述两个不易分辨的量子系统间存在不可避免的相互作用而产生的混合态。它可以用来构建双向通信链路，实现信息的高速、可靠的传递。量子信息论则是利用量子纠缠态构建的编码原理，将宏观上的信息转化为微观上所需的信息，例如数字信号或量子比特流。在量子通信领域，广义上的量子通信是指利用量子纠缠态来构建信道进行通信的理论基础，而狭义上的量子通信则指仅用一种量子信道进行信息传输的应用。

量子密码学（Quantum Cryptography）也是研究如何利用量子纠缠态构造消息加密与解密方法的学术领域。一般情况下，消息的加密与解密依赖于对称加密算法（symmetric-key algorithm），即双方共享相同的密钥。但是，当密钥对遭到破坏时，就不能解密了。而当密钥尚未被破坏时，也无法有效保护消息的完整性。在量子通信和量子密码学领域里，人们提出了使用量子纠缠态构建的密码系统作为新的安全通讯机制。

# 2.核心概念与联系
## 2.1 量子纠缠态
量子纠缠态是量子力学中描述两个不易分辨的量子系统间存在不可避免的相互作用而产生的混合态。它可以用来构建双向通信链路，实现信息的高速、可靠的传递。常用的量子纠缠态有Werner State（瑞恩态）、Cat States（叶子态）、Gell-Mann Matrix States（格列曼矩阵态）等等。

### 2.1.1 Werner State（瑞恩态）
瑞恩态（Werner State）由两个正交的量子数列构成，分别为|0>和|1>。即|ϕ+> = |0>|1> + |1>|0>。由于它具有以下性质：

1. 可观测性：处于瑞恩态时，通过观察量子态可以测得任意一个态矢对应的平面波（surface wave）。
2. 不具备传播性：处于瑞恩态时，任何一个量子波都不会传播开来，反而会受到阻碍。
3. 可调制性：处于瑞恩态时，可以通过施加关于两个状态的门来改变状态的演化。

### 2.1.2 Cat State（叶子态）
叶子态（cat state）也叫做GHZ State，由三个正交的量子数列构成，分别为|00>,|11> 和|10>。即|ϕ> = (|00>|01> + |11>|10>)/sqrt(2)。这种态在进行通信时非常有用，因为每一个量子比特的信息都可以用三个比特来表示。假设发送者首先准备了一个叶子态，然后依次通过三角形状的转移，使得最后两个比特都是|1>状态，中间一个比特为|0>状态，这样就可以使用三个比特进行信息的发送。接收者收到这个信号后，只需要再通过三角形状的转移，便能恢复出原来的信息。

### 2.1.3 Gell-Mann Matrix States（格列曼矩阵态）
格列曼矩阵态（Gell-Mann matrix states）是利用李群 representations 的形式描述的，它的本质就是哈密顿量（Hamiltonian）与自旋（Spin）不变不变质量规定下的波函数（Wavefunction）。一个 n 维格列曼矩阵态，它的哈密顿量 H 是一个对角阵，且其对角元素对应于矩阵的第 i 行和第 j 列。因此，格列曼矩阵态可以视作一系列等价的单量子比特系统状态，每一个状态的概率都是相同的。 

对于一个二维格列曼矩阵态，每个量子比特的最初态可以是|0>和|1>，从而组成一个格列曼矩阵态。该态的物理意义是，两个量子比特以不对称的方式纠缠，使得它们的资源竞争激烈，并且具有高度的时间反应性，通常用于量子通信领域。

# 2.2 量子纠缠结构（Entanglement Structures）
量子纠缠结构（entanglement structure）可以分为静态纠缠和动态纠缠两种。静态纠缠是指量子系统处于稳定的状态下，两个量子系统之间没有不必要的关联。而动态纠缠则是指量子系统的时间演化过程中，两个量子系统之间的纠缠关系发生变化。

### 2.2.1 静态纠缠
静态纠缠是指量子系统处于稳定的状态下，两个量子系统之间的纠缠关系是不变的。静态纠缠可以分为简单的纠缠和复合纠缠。简单纠缠（Simple Entanglement）是指量子系统在某些特定情况下可以把两个相干的量子态绑定起来，并利用这些绑定关系对其他量子系统产生影响。

例如，在图 1 中，两个量子系统 A 和 B 处于不同粒子的基态，由于它们之间的动量差异巨大，导致他们的本征态迅速收敛到相似的状态。因此，A 和 B 可以把它们相连，构建出一个纯态，该纯态可以作为一个整体的描述。另外，图 1 中的 Y 轴上的箭头，代表了纯态 A 的有效率（Coherence）。即，A 可以通过自己的动量守恒定律消除 B 引起的玻尔兹曼效应。

除了利用图 1 中 Y 轴上的箭头进行纠缠外，还有一些简单的纠缠结构，如混合态、阻抗效应、反演态等。

### 2.2.2 动态纠缠
动态纠缠是指量子系统的时间演化过程中，两个量子系统之间的纠缠关系发生变化。

#### 2.2.2.1 Classical Dynamical Entanglement（经典动态纠缠）
经典动态纠缠（Classical dynamical entanglement）是指量子系统的时间演化过程中，利用量子纠缠态所产生的纠缠关系来实现信息的传输。这里所谓的时间演化，指的是量子系统的态矢在空间上翻滚、平移或者膨胀、缩小的过程。

例如，图 2 是经典动态纠缠的示意图。黑色的圆圈表示初始状态，灰色的圆圈表示演化之后的状态，虚线箭头表示纠缠的方向。这是由于初始状态处于一个受控的态矢中，经过一段时间的演化，系统的纠缠性逐渐增强，最终演化到一种具有高度纠缠性的态矢，因此可以达到信息的高速传输。

#### 2.2.2.2 Quantum Dynamic Entanglement（量子动态纠缠）
量子动态纠缠（Quantum dynamic entanglement）是利用量子纠缠态所产生的纠缠关系，实现量子通信的目的。

量子动态纠缠的实现方式如下：首先，两个量子系统 A 和 B 经历一段固定的演化，把它们从受控的态矢演化到某个态矢。随后，A 和 B 会引入一种纠缠结构，这个纠缠结构能够利用量子纠缠态的特性来快速传递信息。

量子通信的一个关键要求是对称加密，这就要求两个参与通信的系统必须在同一个纠缠结构下进行，才能保证信息的正确性和保密性。因此，如果要建立一个量子通信系统，必须先定义好该系统的纠缠结构。值得注意的是，不同的纠缠结构会导致通信效率和耗时有所差别。
