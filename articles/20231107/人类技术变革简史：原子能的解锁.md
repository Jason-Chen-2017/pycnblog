# 1.背景介绍


近代物理学研究已经证明了宇宙中存在着一种物质——原子。从某种意义上来说，原子本身就代表着一种智慧、力量和能量的合体。这种对真实世界的理解引出了一个重要的问题——原子能的来源。按照牛顿三定律，自然界所有的物体都遵循能量守恒定律：即总的能量等于质量乘以位移的平方，而质量和位移都是由对应的动量决定的。因此，如果把质量换成原子个数n，则能量守恒公式可以表示为：

E=m*v^2=n*(e*v)^2=(ne) * v^2

其中，e为电荷的基本单位，它决定了电荷的大小。由于所有元素都是由原子组成，因此电荷一定比质量小，所以用e来表示电荷的大小。通过牛顿第三定律可以推导出：电荷与质量的比值，即电荷的数目n，和电荷与空间位置的关系v（速度）的平方成反比。

从这可以看出，若要解开关于原子能的谜题，仅仅靠观察并不能完全得到这个谜底的真相。进一步分析，可以发现，无论是在地球上还是在宇宙其他任何地方，都没有找到一种物质能够凭空创造出足够多的电荷。这样就产生了另一个难点——如何创造出一种物质，使得它既拥有足够多的电荷，又不影响天体运行。经过几百年的努力，科学家们终于找到了这样的物质——氢，它既是电荷，又是质量！

从这两者的比较就可以发现，如果说原子只是质量的一个微小部分，那么氢则是原子的全部。气体一般只含有少量氢原子，但对于了解氢的性质、功效、与宇宙的关系，却是十分重要的。如今，氢的应用已渐渐成为现代生活的一部分。

# 2.核心概念与联系
本文主要讨论原子化时期的宏观事件及其影响。以下是一些相关的概念和联系。
- 原子:是一个最小的可利用的有序构成单元。根据电子结构性质，原子的结构中包含了两个电子核、两个电子阴极量子，一个半整数半半整数轨道。通过化学反应，一个原子内的所有物质团块都得到分离形成两个新的原子，称作原子核与原子外的物质称作电子。
- 电子与电子之间的相互作用形成了由能级所组成的电磁场，形成导电材料等。核裂变也会带来高电压，引起原子之间的反电离现象。
- 铱量(质子量):一个原子通常包含7个质子，这里的质子就是电子。
- 分子:由多个原子组成的具有五官的实体。质子氢原子通过化学反应分解成为无数颗分子，他们的性质和行为都不同于原子核。
- 中子:原子核内部的质量粒子，具有唯一的质量而与其他质量分子不存在共振。
- 原子核:主要由电子组成，具有两个半整数半半整数的轨道，每个原子核外壳上有四个碳原子。核芯、核反应堆和电池都属于原子核范畴。核活动导致电子膨胀，从而形成中子。核能很容易释放出来，它可以充当中央调节装置，引导宇宙间的物质运动。
- 太阳光:太阳的一种波长。光子就是电子，但是它们拥有更大的能量。
- 海森堡效应:宇宙中的微粒子激发出强大的核核反应，导致整个宇宙的运动加速。
- 原子弹:一颗放入太阳系中射爆的原子核，杀伤数千万人的生命。
- 原子灰:无色无味的元素，其化学结构非常独特。在高温下磁化，产生中子，中子带走质子带走氢带走电子，形成正电子、负电子与空穴。
- 氢核燃料:天体运行需要氢原子，氢核燃料可以转换成氢的燃料，比如火箭和太阳帆布。
- 比特币:一种数字货币。
