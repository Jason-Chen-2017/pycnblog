# 1.背景介绍


## 简介
芯片制造是一种复杂的工程活动，需要大量的人力、物力和专门设备的投入。其工程量巨大且复杂，制造过程受到许多因素影响，其中一个重要的影响因素就是晶体材料——多晶硅材料。
多晶硅材料(Polymer Silicones)是一种高分子化合物，具有超强的韧性、强度、耐磨、抗氧化、自然反应等特性。它可以应用于半导体(BGA)、电容器件、集成电路、集成电源、显示器件、卫星电池、太阳能电池等领域。随着产业的发展，越来越多的厂商生产多晶硅材料。
不同晶格系数的硅单胞形态完全不同的多晶硅结构可以让电路板、集成电路板、电容器等各种器件表面具有多样的光亮效果。在制造多晶硅芯片时，就需要考虑到电镀、切割、加工、连接、双层包装、焊接等环节，确保精度、品质和可靠性。
而电镀是指将杂质从材料中分离出来，去除杂质和保护微尺寸电路的功能性。电镀方法可以包括抹灰、切除、冲洗、压制、烘干、加热、冷却等。
切割是指将超导电路板中的微粒裂变为能量，并通过大量的颗粒运动传输能量至电路板上。切割过程涉及电极形状、厚度、粘度等参数的调整，根据电路的特性选择相应的切割方式和工具。
加工是指将电镀后的材料进行晶圆上升或者下降、变形、均匀化、无缝焊接、密封等操作，使得电路的性能得到提高。
连接则是指将电路组件连接在一起，完成器件的完整设计。连接方法有拉线、绝缘焊接、异物埋入、排隙连接、接头连接等。
双层包装是指将多晶硅材料层层堆叠后嵌入半导体材料，减少尘埃对器件的影响，保障多晶硅材料的安全稳定。
焊接是指用熔融的方法将材料焊接在贴在金属表面的金属皮带或金属母片上，通过施加物理力，使电路材料与表面之间的固相作用，达到设计要求的目的。
## 多晶硅生长与薄膜退火
多晶硅材料的生长过程通常分为两种类型——微晶硅型和晶圆型。微晶硅型与晶圆型的区别主要在于微晶硅型每层的晶粒大小仅为几十纳米，而晶圆型每层晶粒大小一般可达几百甚至上千纳米。在生长过程中，微晶硅型的细小晶粒容易受到外界环境的影响，容易出现错乱、开裂、破碎等现象；而晶圆型则不易受外界影响，大大减轻了劣化风险。
如图所示，微晶硅型的晶体由丝状柱状组成，因此称为微晶硅，也被称为一晶。微晶硅型的晶体在积累之后，会产生出阵列的晶粒，构成一整个晶体，这种晶体的每层晶粒都非常小，即微晶。微晶硅型的晶体通常被粘在一起，称为“垢砾”。这种晶体聚焊到稳定的金属表面后，由于微晶粒的凝结作用，产生出稳定的晶体壳，结构称为“晶体”（或晶瓣）。微晶硅晶体经过多次自然腐蚀和气泡处理，使晶体保持良好的整体性和结构性，具有很好的韧性和耐磨能力，被广泛用于芯片制造。
但是，微晶硅型的晶体生产周期较短，生长率低，缺乏连续性，在制造的各个环节中都存在一定程度上的缺陷。微晶硅型的材料性能往往受到物理性或化学性因素的限制。另外，微晶硅型晶体的尺寸小，而且无法提供足够的电流，难以满足芯片制造的需求。
晶圆型则为两种不同晶格系数的硅单胞形态完全不同的多晶硅结构，拥有很强的抗热、防锈、抗腐蚀、防震等特性。晶圆型的晶体的每个层都是晶圆形，远大于微晶型，每个层的晶粒直径约为100 nm至5 μm。这种晶体不易受物理性或化学性因素的影响，可供制造各种器件提供丰富的光亮效果，比如显示屏、电池、电子元件、IC、SoC等。晶圆型的材料能够有效地克服微晶硅型的一些缺点，尤其是在高分子化合物方面，使得其优点发挥得淋漓尽致。
晶圆型晶体在制备工艺上使用的是电镀加熟法和熔融法，即先将晶圆型材料在低温下进行高频化学加熟处理，使得晶体结构稳定，然后再加入温度适当的高分子物质，使其由石墨烯构成的晶球，再经过烘烤、切割、打粘、脱水等工序，最后将晶体壳嵌入一块或多块先导材料上，完成长期的存储。晶体中的化学成份经过多次腐蚀、混合和压制，使得晶体内残留的微观废弃物被清除掉，消除了晶体内的杂质和污染物，最终形成晶体壳，成为了可用的晶圆型电路制备材料。
多晶硅的生长退火过程包含了多种技术，其关键是多晶硅的自然结晶以及自然腐蚀。自然结晶又称晶体层化，是指多晶硅材料在浓缩状态下，通过蒸馏和镀膜到一个平衡的状态，产生出一个高分子的晶体壳。自然腐蚀是指在温度保持稳定时，晶体材料自身内部的化学物质和微观杂质，经过自然温度的加热而腐蚀，完成自然的磨损和分解，使其变得更易于处理。同时，退火还可以保护晶体材料的表面免受化学反应的影响。
# 2.核心概念与联系
## 多晶硅的特点
- 多晶性：晶体中的不同晶粒不发生交叉，每层晶粒之间存在顺序关系，这样便具有一种多晶性。
- 顺硬性：不同晶粒之间没有边界，不存在碳-晶粒的紧密联系，所以晶体中不会产生刺激性的反应，使得晶体的硬度较高。
- 光滑性：晶体的表面呈现出丝状的结构，这是因为不同的晶粒的融合导致晶体表面呈现出不同颜色的网格线，并非混色的凝胶状。
- 欧氏变换：晶体的各层之间存在独立的平移，这使得晶体在晶核之间形成欧氏位移，并且晶体结构具有单向性。
- 治愈力：多晶硅具有高治愈力，即使晶体的表面出现任何杂质的破坏、腐蚀、变形、摩擦等，也可以通过自然腐蚀和气泡处理，使晶体的结构完整、整齐、健康、平衡，并保持良好的功能性。
- 传导性：多晶硅具有良好的传导性，在两极高温的条件下，晶体的不同晶粒均可逐渐从固态转变为浓缩的状态，从而实现完整的结构和完整的功能。
## 多晶硅材料的分类与特性
### 多晶型材料
多晶硅材料指由两种以上不同的晶粒组成，这种材料的高分子化学结构特征使其具有多种多样的光亮特性。典型的多晶硅材料有马拉硅、莫桑比克硅、法拉第硅、阿里巴巴硅、驼峰硅、玛利亚金棕榈石等。
多晶型材料的特性如下：
- 深度耐氧：多晶硅具有强烈的深度耐氧性，能够存活于许多环境。
- 高分子感光：多晶硅具有极高的分子感光率，能够使用高分子的谐振腔光照。
- 抗光受压：多晶硅具有强大的抗光受压性，能够抵御高温下的曝光。
- 抗攻击：多晶硅具有抗攻击性，能够抵抗各种化学、物理攻击。
- 自然光泽：多晶硅具有纯净无污染的自然光泽，能够自然泄露在空气和水中。
- 高阻性：多晶硅具有高阻性，能够阻断大多数的高电压，使其成为高电压、高效率的电子芯片材料。
- 高强度：多晶硅具有很高的强度，能够提供最高的互感和电容耐久性。
- 可见光亮度：多晶硅具有较高的可见光亮度，能够被日光照射，能够在紫外线、激光、红外线下工作。
- 隔热效果好：多晶硅具有不错的隔热性能，在高温环境下，晶体的不同层均可有效地沉积。
### 微晶型材料
微晶硅材料指由一种晶粒组成，这种材料的结构特征与多晶硅类似，但它的特点是每个层只有一个晶粒。典型的微晶硅材料有稀土硅、振华石墨烯、特氟兰石墨烯、锆石、苯酚石墨烯等。
微晶型材料的特性如下：
- 高容量：微晶硅的容量高达数十微米，具有良好的对孔性能。
- 吸湿性：微晶硅的结构具有吸湿性，可以用于高压输送、湿气调节和智能窗户。
- 高效率：微晶硅的电感场分布较稀疏，能够有效提高电路的阻抗匹配效率。
- 小尺寸：微晶硅的晶粒大小只有几十纳米，小于晶圆型材料。
- 体积小：微晶硅的体积小，比较适合制作小型电路板。
- 结构简单：微晶硅的结构较简单，结构定义良好，容易掌握。
- 制造过程简单：微晶硅的制造过程较为简单，采用标准化工艺，不需考虑特殊工艺流程。
- 制造成本低廉：微晶硅的制造费用较低，其单件价格在几百到一千美元之间。
- 易制造：微晶硅材料的制造工艺简单，容易采用批量生产，大大节省时间和成本。
