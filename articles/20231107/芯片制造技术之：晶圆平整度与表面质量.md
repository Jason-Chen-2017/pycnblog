# 1.背景介绍


晶圆制造技术（又称硅制造、微晶制造等）是国际上主要的半导体制造工艺之一。其最早起源于古埃及的玛雅岩石中发现，在早期之后，随着数百年来人类对这种材料的需求的提升，晶圆制造技术逐渐成熟并开始受到越来越多的重视。它可以用来制作低纤维、高容量、可靠性高、无机化学物质密集、耐高温、耐碱性、超厚度等各个方面的半导体。由此产生的低功耗、高性能的集成电路（IC）正在成为目前应用最广泛的一种芯片制造工艺。然而，晶圆制造技术还存在一些尚待解决的问题，如晶圆微结构的晶圆平整度、低温表面质量等。因此，如何设计有效、经济实惠且高度精准的晶圆制造方法，是影响晶圆制造技术发展的关键环节。
晶圆微结构的晶圆平整度是指一个晶圆微观结构中各颗粒之间的空间距离，是影响晶圆制品质量的一项重要因素。当前国内外很多晶圆制造企业都把这一问题作为首要考虑因素，但很少有相关的研究工作。笔者将通过本文探讨晶圆微结构的晶圆平整度的定义及其指标计算方法，给出不同晶圆平整度的分类标准及其对应的晶圆制造工艺策略。最后，我们还会根据现有的晶圆制造技术的进展情况，总结未来的研究方向和前景。
# 2.核心概念与联系
## 晶圆微结构的晶圆平整度定义
晶圆微结构的晶圆平整度，即每个晶圆微结构中各颗粒之间的空间距离。按照定义，一个晶圆微结构的晶圆平整度一定要小于其基本长度L的一半，才能够保证这种晶圆微结构具有良好的晶粒晶顶距离分布，并达到预先设定的要求。晶圆微结构的晶圆平整度，可以用L-p/2 (L为晶圆的长度，p为晶圆的平行度)表示。其中，L是晶圆的基本长度；p是晶圆的平行度。为了简化计算，通常认为晶圆微结构的晶圆平整度等于其基本长度的一般值。例如，在无偏离轴的二维晶圆微结构中，晶圆平整度等于其基本宽度的一半。
## 晶圆微结构的晶圆平整度分类
根据晶圆微结构的晶圆平整度值的大小，晶圆微结构的晶圆平整度可以分为以下几类：
### Type I型平整度：当晶圆微结构的晶圆平整度值小于等于L的一半时，晶圆微结构的晶圆平整度属于Type I型平整度。
### Type II型平整度：当晶圆微结构的晶圆平整度值大于L的一半时，晶圆微结构的晶圆平整度属于Type II型平整度。
## 晶圆微结构的晶圆平整度指标计算方法
晶圆微结构的晶圆平整度指标可以通过一些参数的组合来计算获得。常用的计算晶圆平整度指标的方法有基于基本长度的指标（如L-p/2）、基于晶圆缺陷程度的指标（如Murphy比）和基于有限元分析技术的指标。
### （1）基于基本长度的指标
L-p/2就是最简单直接的晶圆平整度指标。对于二维晶圆微结构，可以通过计算基本长度L和晶圆平行度p，然后代入等式L-p/2=m得到其值。其中，m是一个常数，表示两个相邻晶粒之间距离的平均值。利用这个指标，就可以将晶圆微结构的晶圆平整度与其基本长度的关系绘图出来。例如，当m=1，即所有晶粒距离均为L/p/2时，就得到了一维晶圆微结构，晶圆平整度就等于其基本长度L/p/2。
### （2）基于晶圆缺陷程度的指标
Murphy比（也叫做缺陷比）是基于缺陷密度分布、晶格分辨率和晶圆形状等特征测量的一种晶圆微结构指标。Murphy比越接近1，晶圆微结构的缺陷越少，晶圆平整度越好。通过拟合Murphy比数据，可以估计相应晶圆微结构的基本长度L，从而确定该微结构的晶圆平整度。
### （3）基于有限元分析技术的指标
由于晶圆微结构具有复杂的晶圈排列结构，因此无法通过解析的方式进行计算。常用的有限元分析技术，如线性求解、三角单元法、正交网格法等，都是基于模拟的方式求解晶圆微结构的晶圆平整度。这些技术通过数值模拟的方法，对晶圆微结构的晶圆平整度进行估计，同时也反映了晶圆微结构实际的晶圆形态。
