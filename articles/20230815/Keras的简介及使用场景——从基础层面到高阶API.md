# 1.简介
  

Keras是一个基于Theano或TensorFlow之上的神经网络API，它能够帮助开发者轻松构建、训练并部署深度学习模型，特别适合于研究人员、工程师以及科研工作者。相比于传统的机器学习框架（如Scikit-Learn）而言，Keras的易用性更强、可扩展性更强、模型结构的构建更灵活、训练速度更快、内存占用率更低等优点，使得其在实际应用中得到广泛的应用。其提供了不同的高阶API，可以帮助开发者快速搭建复杂的神经网络模型，而无需担心底层实现的复杂性。
## 1.1 Keras的特点
* 模型架构清晰、简单易懂：Keras提供的模型构建接口很容易理解，通过几行代码即可构建出复杂的神经网络模型。并且提供了丰富的层、激活函数、损失函数、优化器等功能，使得模型的架构设计更加方便。
* 支持多种后端：Keras支持多个后端，包括Theano、TensorFlow、CNTK和Caffe。通过配置backend参数，可以选择不同后端运行模型，从而实现跨平台移植。
* 数据集处理容易：Keras提供了数据集预处理、生成工具，使得数据的准备变得十分简单。可以直接加载各种格式的数据，并对其进行处理，自动完成特征工程和归一化。
* GPU加速：Keras可以利用GPU计算加速，进一步提升模型训练速度和效率。同时，Keras也提供了分布式计算接口，可以方便地将模型训练并行到多个GPU上。
* 多样的高阶API：Keras提供的高阶API包括Sequential、Functional API、Model子类化API等。可以通过这些接口灵活组合不同类型的模型层，构造出各种复杂的神经网络模型。
* 生态系统完善：Keras拥有庞大的生态系统，包括大量的教程、模型库和工具，可以满足各个领域的需求。
