# 2.计算机编程基础概念
## 2.1 Python程序运行环境
### 2.1.1 编译型语言
首先，我们先要搞清楚什么是“编译型语言”。所谓编译型语言就是指编译后生成一个二进制可执行文件（EXE/DLL），运行时直接加载这个二进制文件就可以运行。例如C、C++、Java等。

### 2.1.2 解释型语言
解释型语言是在运行前不需要进行编译的脚本语言。它的解释器可以直接运行源码。Python、JavaScript、Ruby等属于解释型语言。

### 2.1.3 IDE和文本编辑器
IDE和文本编辑器主要用来编写、调试和运行程序。常用的IDE有PyCharm、VS Code、Sublime Text等；常用的文本编辑器有Notepad++、Atom、Vim、Emacs等。

## 2.2 数据类型
### 2.2.1 整数int
整数表示数字。整数可以带符号，带正负号，也可以不带符号。可以通过下面的方式定义整数变量：

```python
a = 1 # 整数
b = -2 # 负整数
c = +3 # 正整数
d = 0 # 零值
e = d ** 0 # 幂运算
f = e ^ (-e) # 反转义
g = abs(-7) # 绝对值函数
h = int('3') # 从字符串转换成整数
i = hex(255) # 将整数转换成十六进制字符串
j = oct(8) # 将整数转换成八进制字符串
k = bin(10) # 将整数转换成二进制字符串
l = float(2.5) # 浮点数
```

### 2.2.2 小数float
小数表示实数，即由整数和小数部分组成的数值，用“.”连接。小数也可以用科学计数法表示。小数也可以用浮点数的形式表示：

```python
a =.5 # 0.5
b = 1. / 2 # 0.5
c = -.75 # -0.75
d = 3.14e-2 # 0.0314
e = round(2.567, 2) # 2.57
```

### 2.2.3 布尔值bool
布尔值只有True和False两个值。常用于条件语句、逻辑运算符。

```python
a = True
b = False
c = a or b
d = not c and a or not b
e = c!= b == True # True
```

### 2.2.4 字符串str
字符串是一个序列，其中每一项都是字符。字符串用单引号或双引号括起来。字符串支持字符串拼接、索引、切片等操作。

```python
a = 'hello world'
b = "I'm OK"
c = "abc"[1] # 'b'
d = "abc"[-1] # 'c'
e = "Hello " * 3 # 'Hello Hello Hello '
f = "Hello, {name}!".format(name="Alice") # 'Hello, Alice!'
g = "{} is {}".format("John", 25) # 'John is 25'
h = '{first} {last}'.format_map({'first': 'John', 'last': 'Doe'}) # '<NAME>'
```

### 2.2.5 列表list
列表是一个有序集合，其中的每个元素都有一个编号。列表可以存储任意的数据类型，包括列表、整数、小数、布尔值、字符串。列表可以使用方括号([])包裹。

```python
a = [] # 创建空列表
b = [1, 2, 3] # 创建整数列表
c = ['apple', 'banana'] # 创建字符串列表
d = [[1, 2], [3, 4]] # 创建多维列表
e = [1, 'apple', True] # 列表中混合不同数据类型元素
f = list((1, 2, 3)) # 通过元组创建整数列表
g = list("hello") # 通过字符串创建字符串列表
h = len(d[0]) # 获取第一行列表长度
i = min(d[0]) # 返回最小元素
j = max(d[0]) # 返回最大元素
k = sum([1, 2, 3, 4]) # 求和
l = d.append([5, 6]) # 在末尾添加元素
m = d.insert(0, [-1, -2]) # 插入元素到指定位置
n = d.pop() # 删除最后一个元素并返回
o = d.remove([-1, -2]) # 根据值删除元素
p = d.extend([[7, 8]]) # 添加子列表到末尾
q = sorted(d[0]) # 对列表排序并返回新列表
r = all(d[0]), any(d[0]) # 判断所有元素是否满足条件
s = enumerate(['apple', 'banana']) # 生成索引和对应值
t = zip([1, 2, 3], ['apple', 'banana']) # 将两个列表压缩成字典
u = set(d[0]) | set([5, 6]) # 合并两个集合
v = iter([1, 2, 3]) # 迭代器
w = next(iter([1, 2, 3])) # 获得迭代器的第一个元素
x = dict(zip(['apple', 'banana'], [1, 2])) # 创建字典
y = x['apple'] # 访问字典的值
z = x.get('peach') # 访问字典的键值，如果不存在则返回默认值None
```

### 2.2.6 元组tuple
元组也是序列，但是只能存放不可变对象，通常用于函数参数传递。元组使用圆括号包裹。

```python
a = () # 创建空元组
b = (1, ) # 创建单元素元组
c = ('apple', 2) # 创建含多个元素的元组
d = tuple("hello") # 通过字符串创建元组
e = reversed(('apple', 'banana')) # 翻转元组
f = list(reversed(('apple', 'banana'))) # 转换为列表再翻转
g = divmod(5, 3) # 取商和余数
```

### 2.2.7 字典dict
字典是一个无序的键值对集合，可以随意地存取数据。字典中的值可以是任意类型，包括其他字典、列表、整数、小数、布尔值、字符串。字典使用花括号({})包裹。

```python
a = {} # 创建空字典
b = {'name': 'Alice'} # 创建带有键值对的字典
c = {'a': 1, 'b': 2} # 创建字典
d = {'apple': 1, 'banana': 2}.keys() # 获取字典所有的键
e = {'apple': 1, 'banana': 2}.values() # 获取字典所有的值
f = {'apple': 1, 'banana': 2}.items() # 获取字典的所有键值对
g = c.update({'c': 3}) # 更新字典
h = c.popitem() # 删除字典中最后一对键值对并返回
i = c.clear() # 清空字典
j = c.copy() # 拷贝字典
k = 'banana' in c # 查找键是否存在
l = c.setdefault('banana', 0) # 设置键值，如果不存在则添加并设置默认值
m = cmp(1, 2) # 比较大小
n = sys.getsizeof(c) # 获取字典占用的内存空间
```

## 2.3 操作符及表达式
### 2.3.1 赋值运算符=
给变量赋值。可以在一条语句中同时对多个变量赋值。

```python
a = 1
b = 2
c = 3
d, e, f = g = h = i = j = k = l = m = n = o = p = q = r = s = t = u = v = w = x = y = z = 0
```

### 2.3.2 算术运算符
加减乘除、取模、幂运算。

```python
a = 1 + 2
b = 1 - 2
c = 2 * 3
d = 2 / 3
e = 2 % 3 # 模运算
f = 2 ** 3 # 幂运算
```

### 2.3.3 关系运算符
等于、不等于、大于、小于、大于等于、小于等于。

```python
a = 1 == 2 # False
b = 1!= 2 # True
c = 3 > 2 # True
d = 2 < 3 # True
e = 4 >= 4 # True
f = 5 <= 5 # True
```

### 2.3.4 逻辑运算符
与、或、非、异或。

```python
a = 1 and 2 # True
b = 1 or 0 # True
c = not 1 # False
d = 1 ^ 0 # 1
```

### 2.3.5 成员运算符
在序列、字符串中判断某个元素是否存在。

```python
1 in [1, 2, 3] # True
'test' in 'test string' # True
'something' not in 'test string' # True
```

### 2.3.6 身份运算符
判断两个标识符是否指向同一个对象。

```python
a = b = [1, 2, 3]
print(id(a), id(b)) # 打印对象的地址
print(a is b) # 判断两个变量是否指向同一个对象
print(a == b) # 判断两个对象的值是否相等
```

### 2.3.7 条件表达式
根据条件表达式的结果决定执行哪个分支的代码。

```python
result = "even number" if num % 2 == 0 else "odd number"
```

### 2.3.8 运算符优先级
运算符优先级决定了表达式求值的顺序，高优先级的运算符会先计算。

```python
# 使用()改变运算顺序
a = 2 ** 3 + 4 // 2 * 5 % 2
b = ((2 ** 3) + 4) // 2 * 5 % 2

# 函数调用和成员访问的优先级比算术运算符低
a = func() + obj.attr
b = obj.func() + obj.attr
```
