# 1.简介
  
及背景介绍
当下，人们都把互联网、云计算和物联网作为新兴产业，并将其称之为“新生事物”。随着5G和物联网领域的火爆，越来越多的人们开始思考如何更好的运用这些新兴技术服务于社会。相比于传统的企业来说，物联网创业公司应具备一些独特的特征。本文首先介绍一下物联网的相关概念及基本知识，然后谈论一下初创公司和成熟公司之间的区别，最后探讨一些典型的创业项目及其成功经验。

1.1 什么是物联网？
物联网（Internet of Things, IoT）是一个涵盖众多领域的交叉学科，由一系列技术和协议组成，包括智能设备、网络、应用系统等。它利用现代信息技术对现实世界进行采集、处理和传输，将数据转化为信息，实现信息的自动化、收集、分析、存储和传播，最终实现“智能生活”的理想。

1.2 物联网的基本原理
物联网的核心理念是“连接万物”，利用各种传感器、执行器、终端设备等，不断地收集和传输数据。根据不同的物联网类型，可以分为以下几种：
1. 节点型物联网：基于无线传感网的部署，能够实现互联互通、数据采集、数据共享、远程控制等功能。适用于弱电环境、小区或建筑物内，提供低成本、分布广泛的服务；
2. 中央型物联网：在城市范围内，通过中央控制器、网关、终端设备、数据中心等设备实现连接、数据采集、数据存储、数据共享、远程控制等功能；适用于城市中高密度的物联网环境，且具有较强的安全保障；
3. 分布式物联网：基于无线传输、边缘计算、物流、信息采集等分布式计算能力，能够实现复杂的数据处理、监控、预测、风险检测、社会影响评价等功能。适用于具备海量数据的应用场景，如智慧城市、医疗健康、环境监测、供电工程、农业监测等领域。
根据物联网的不同特性和应用场景，不同厂商在研发上也有所不同，例如某些厂商侧重于智能硬件设备，另一些厂商则将重点放在物联网应用层面。
1.3 为什么要创业？
物联网产业蓬勃发展，虽然依然处于起步阶段，但已经有许多成功的创业案例。但是，对于创业者来说，创业前需要考虑自己是否具备相应的条件。下面介绍一下何时、如何、怎样才算是合格的创业者。

1.4 创业条件
1. 有能力：即使是在刚开始创业的时候，也是最重要的一环。物联网从业人员除了要熟悉物联网的基本概念和理论，还要熟练掌握相关的编程语言、工具、框架等。同时，需要有独立思维、热爱学习的精神，充分准备自己的创业计划。
2. 信心：创业初期，一定会有很多的雾霾和恐惧，需要充分迎接挑战和坚持。但是，也不要失去信心，坚持努力，以坦然面对和奋斗！
3. 资源：创业者应该善于利用自己的资源，比如时间、金钱、家人支持等等，尽可能地提升自己的技能和能力，做到开拓进取、自我驱动。
4. 决心：不管是创业还是投资，只要有决心，总能取得好的结果。创业就是脚踏实地、一门心思投入，努力去为社会做贡猎，为后续的发展保驾护航！

1.5 选择创业方向——初创公司 VS 有限公司
创业公司和创业项目可以分为两种，一种是初创公司，另一种是成熟公司。初创公司是指刚开始的创业项目，一般没有很大规模，只专注于某个细分领域或产品，主要靠个人能力、个人雄厚的经验、个人的主观能动性完成产品的开发，通常没有明确的盈利模式。而成熟公司则代表着市场和领域的需求被满足并且正在积极布局，拥有丰富的经验、资源、资金，并且具有完整的商业模式和完整的组织架构，能够长期稳定地运营。初创公司的优势在于快速验证产品的市场潜力，快速获得反馈，开发出可用的第一个版本；而成熟公司则可以保证产品的持久化，降低产品的平均成本，提升竞争力。

在实际运作过程中，初创公司可以选择市场份额少、规模较小的垂直领域，也可以选择战略定位偏向国际化、国际品牌的，比如IoT通信产品。有限公司则可以选择国内外著名、知名、有竞争力的公司作为投资方，投资进一步扩大规模，加强协同合作，形成完整的产业链条。在决定创业方向时，创业者应该综合考虑该领域的发展前景、产品的市场占有率、竞争对手的强烈反馈等因素。
