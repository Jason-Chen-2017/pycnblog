# 1.简介
  

每个人都希望自己变得更聪明，所以无论什么时候，都应该有动力去学习新的知识、提升自我能力、增强个人竞争力等。
然而，学习和实践并不能保证每个人的成长都如期而至。在实际工作中，也会遇到一些问题，比如：
- 项目延期、推后、进度变慢；
- 上级领导不作为、态度坚决、违反纪律；
- 团队成员之间摩擦、工作效率低下、缺乏协作精神。
这些问题产生的根本原因可能是因为没有正确地认识到自己的优势所在。
因此，要想成功地突破瓶颈，需要首先明确自己的优势所在，对自己的长处和短处进行梳理，并且时刻记住：
> “知识改变命运，能力培养人格”！
# 2.为什么学习技能比思考能力更重要？
首先，学习技能可以帮助我们掌握一些基础的计算机编程技术、解决实际问题的方法，从而有效地解决日常生活中的实际问题。
其次，学习技能还能提升我们的管理和沟通能力，包括交流表达能力、听说读写能力、团队合作能力等。
再者，学习技能也可以锻炼我们的独立思考能力、分析问题能力和创新能力。
最后，学习技能可以培养我们勤奋努力的习惯、品质面貌和成就感。
综上所述，学习技能比思考能力更重要。只有通过学习才能够将知识应用到实践中，在实际工作中才能充分发挥自己的才能和特长。
# 3.怎么确定自己的优势？
确定自己的优势，并不是一件简单的事情。正确理解自己长处和短处对于我们判断自己优势有着至关重要的作用。
## 3.1 为什么要把自己长处和短处搞清楚？
首先，了解自己的优点和缺点，才能知道自己该做哪些事情、不该做哪些事情，才能做出正确的判断。例如，假设你是一个很有潜力的人，却被老板否定了。但是你又不知道自己为什么被否定，可能只是自己的缺点，但也可能是老板压制了你的潜力。因此，要准确地判断自己的优势和弱点，能够给予最恰当的指导，帮助我们更好地发展。
其次，了解自己的长处，能够帮助我们改善现状。譬如，如果你是个有钱人，那么就会有很多钱来源，你可以利用这些钱来投资理财或购买高性价比的商品。同时，如果你擅长组织活动、教育、销售，那么就能够帮助其他人完成类似的任务，也会得到别人的肯定和赞扬。
再次，了解自己的短处，能够帮助我们开拓新天地。譬如，如果你是个创业者，那么就容易陷入困境，因此需要从细节中洞察问题，发现新的机会。另一方面，如果你善于倾听，那么就可以快速接触到别人的意见，也可以借此展现自己的观点。
最后，了解社会上的一些新闻事件，可能会让我们了解到社会的趋势。譬如，假设有一位员工被曝出心脏病毒，许多人可能会认为这个消息就是伤害他人的谣言。但是，如果是在美国，那些有亲戚在场的人可能会更加相信这一消息。因此，要善于观察，不断吸取新信息，否则可能会误入歧途。
综上所述，了解自己的长处、短处和优势，有助于我们分析问题、做出正确的决策，并更好地发展。
## 3.2 怎么界定“长处”和“短处”？
界定“长处”和“短处”，有一定的技巧。
### 方法一：身体素质
身体素质是人们容易被衡量的一种能力，它表现在三个方面：生理、心理和文化层面。其中，生理能力包括：腰部挺直、四肢健壮、全身血液循环畅通、精神力旺盛；心理能力包括：具有洞察力、开朗感、适应能力；文化能力包括：艺术鉴赏力、言辞表达力、丰富的历史阅历。
从各个方面来说，身体素质是衡量一个人身体抵抗力、保持力、成熟力、耐力的主要依据。因此，我们可以通过以下方式来评估自己的身体素质：
- 通过测量自己的年龄、体重、血压、脂肪、骨骼和肌肉强度等参数，掌握自己的身体健康水平。
- 根据自己的特色、喜好、兴趣、能力和行为特征，探索自己的价值和特质。
- 将自己的优势与缺点与自己拥有的身体资源（如智商、体力、眼光、品味等）结合起来，形成独特的个性特征。
### 方法二：履历与经验
履历与经验是衡量一个人的工作能力、专业能力的重要工具。一个人成功与否，往往取决于其个人经验、过往记录以及个人的能力。
个人经验可以通过个人的职业生涯、公司的雇佣情况、合作关系等获取。个人的能力则可以由个人掌握的技能、业务水平、业务素养等获得。
通过分析自己的履历、工作经验和相关技能，我们能够清晰地认识到自己的长处和短处。不过，由于履历可以让别人了解到你的工作态度和成果，因此一定要慎用。
### 方法三：爱好与特长
爱好是衡量一个人的个人魅力、家庭情调以及独立思维能力的有效手段。一个人的爱好决定了其幽默感、社交能力、抽象思维能力以及心胸放松能力。
爱好可以是一件小事，比如观看电影、看绘画，也可以是一项长远计划，比如参加一项长期志愿服务项目、建立自己的专业网络。
通过分析自己的爱好和特长，我们能够提升自己的独立思维能力。独立思维的能力是指能够从一系列的事物中找到真正重要的东西，并且能够以不同的角度来看待问题，从而找到解决问题的有效方法。
### 方法四：梦想与目标
梦想与目标也是衡量一个人的个人能力的重要指标。目标往往成为衡量一个人的成功与失败的关键因素，目标驱动着人们的自主性、主动性、创造性以及结果导向性。
如果一个人没有目标，那么他通常都是一头扎进自己的错误轨道里，即使有才华也很难达成理想。因此，目标能够帮助我们设置更有意义的任务和计划，为实现目标努力奋斗。
目标可以是短期的、临时的还是长期的。短期目标一般是完成某项具体任务，例如购买一双跑鞋；长期目标则比较宏大，涵盖整个人生，例如攀登珠穆朗玛峰。
通过分析自己的梦想、目标和现实，我们能够更好地激发和引导自己的潜力。
# 4.怎么培养自己的能力？
只有培养自己的能力，我们才能不断成长、不断进步。因此，要帮助自己培养能力，首先要清楚自己目前的困难和优势。然后，我们要寻找自己的兴趣爱好，选择一项自己热爱的事业或从事的领域，深入研究该领域的核心知识和理论。最后，基于自己的专业知识和个人能力，结合业内的先进技术，攻克一项重要的难题。这样，我们就能在短时间内积累起自己所需的能力。
