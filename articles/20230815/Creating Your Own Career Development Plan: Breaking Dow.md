# 1.简介
  

“Career Development”或称“职业发展”，是指个人迈向成功所需的不断学习、经历、参加竞争性活动及个人发展计划等全方面的能力、条件和过程。“Career Development”对每个人而言都非常重要，但由于职业的多样性、高度个性化和市场竞争，真正实现“Career Development”并不容易。一些成熟公司、优秀人才会帮助他们拓宽视野，提供更有效的“Career Development”策略，如2019年泰康康视集团创始人李斌在去年的一次采访中提出的五条“Career Development”建议：1. 在全球各地寻找工作机会；2. 创建自己的“Career Development”计划；3. 利用个人专长塑造个人品牌；4. 展示个人才能并付诸实践；5. 掌握自我成长的技能。

本文从“Career Development”的角度出发，总结了构建个人“Career Development”计划的关键环节，希望能提供帮助。

# 2.核心概念术语
“Career Development”涉及到以下几个方面：

1. 职业规划（Professional Planning）：指一个人的职业方向、职业生涯规划，包括工作内容、目标，以及目标分解任务和可行的方法。
2. 发展心态（Development Attitude）：指个人对职业发展的认识，包括目标导向、对环境的适应性、机遇意识等。
3. 职业生涯规划（Career Roadmap）：职业发展计划，包括目标、路径、方法、资源、时间，以及管理工具等。
4. 技能提升（Skill Improvement）：指个人获得新技能、改善工作表现的方式。
5. 个人风格（Personal Style）：指个人性格特征及对工作的看法，包括职业目标、生活目标、时间观、价值观等。

# 3.核心算法原理和具体操作步骤
1. 搜索全球、招聘信息、评估招聘要求、分析求职者特征。
通过搜索全球、招聘网站、论坛、微博等途径寻找有关相关岗位的信息、进行招聘信息的筛选、评估求职者的薪酬福利、工作经验、专业知识、兴趣爱好等要求。

2. 定义个人发展目标。
确定个人职业规划目标，制定可行的目标分解任务和可行的方法，并用文字、图表、流程图等方式记录下来，并与自己相匹配。

3. 对个人发展计划进行充分的规划。
根据个人职业规划目标，制定发展路线图，设置期望收益，并安排相应的项目和活动。将重点放在个人技能提升、个人品牌形象、个人发展规划等方面，包括教育、成长、职业发展等。

4. 设置目标和指标。
设置个人发展的目标和衡量标准，绘制个人发展目标图，对个人目标和进步状况进行跟踪。

5. 执行及反馈结果。
实际执行个人发展计划，不断完善、调整和优化，并以结果为依据，对个人发展进行反馈。

6. 提升个人技能。
采用系统化的教育方式，培养个人专业技能，保持学习的热情，主动进行自我成长。

7. 显示个人风格。
关注个人需求，倾听他人意见，对个人和工作环境进行审视，制定符合个人口味的个性化发展计划。
