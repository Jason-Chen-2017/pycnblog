# 1.简介
  

在天文学中，我们用到各种不同的光谱仪器，并对不同波长的光线进行测量。从某种意义上来说，所有的光谱都可以看作是一个连续的分布函数，而且它们具有独特的性质，比如光谱的不对称性、自相关性等等。因此，探索、分析和利用天文学数据的一条重要路径就是对高光谱图像进行分类。本文将介绍如何进行高光谱图像分类，以及如何从机器学习角度实现这一分类过程。
# 2. 关键词：高光谱图像分类；机器学习；卷积神经网络（CNN）；深度学习；天文学数据
# 3. 1 背景介绍
高光谱是由大气中的物体反射多种不同波长的光而形成的光谱，一般用两个坐标轴表示波长与电压之间的关系，通常横轴表示波长，纵轴表示电压，而光谱图则是在这样的一个坐标系里画出来的二维图像。由于天文观察者往往只能看到部分的高光谱图像，因此通过自动化的方法对其进行分类是非常必要的。

传统的图像分类方法采用的是手工设计的特征工程方法，它需要先收集和分析大量的图像样本，然后基于这些样本提取出有效的特征用于分类。然而，这种方法耗费时间和资源，且难以适应变化多端的天文数据。因此，基于机器学习的图像分类方法取得了巨大的成功，包括深度学习方法和卷积神经网络（CNN）。

深度学习方法的基础是深度神经网络（DNN），它由多个层组成，每一层之间存在非线性的连接，能够自动学习数据的特征表示。因此，它可以对高光谱图像进行更加精确的分类。而卷积神经网络（CNN）则是一种深度学习方法，它能够自动地从高光谱图像中提取出有用的特征，如边缘、线段、形状等。

本文将结合机器学习方法和天文学数据，介绍如何进行高光谱图像分类，即如何利用CNN模型来识别未知天文图像。
# 3.2 基本概念术语说明
首先，让我们回顾一下计算机视觉中常用的一些术语。
## 1.图像：图像是由像素组成的矩阵，每个像素代表图像上的一个点，颜色、亮度等信息编码在每个像素值中。一般来说，图像分辨率越高，图像就越清晰。

## 2.卷积核：卷积核又称滤波器或内核，是一个大小与图像相同的二维数组。它可以用来进行卷积运算，从而提取图像中的有用信息。

## 3.步幅stride：步幅就是在卷积过程中每次移动的距离。如果步幅为1，则每次只移动一步。

## 4.填充padding：填充指的是在图像周围添加一些额外的像素，以保证卷积后图像的尺寸不变。

## 5.池化pooling：池化层主要用来降低卷积层对位置的敏感度，将局部区域的特征整合到一起。池化层通常用平均池化或最大池化的方式进行处理。

## 6.激活函数activation function：激活函数是一种非线性函数，作用是将输入信号转换到输出空间，用于控制神经元的输出。激活函数的作用有助于使得神经网络能够拟合任意复杂的函数。目前最流行的激活函数是ReLU。

CNN模型中还有其他一些基本概念，如平坦层、dropout层、正则化、超参数等等。但这里不再赘述。
