# 1.简介
  


数据标准化是指对数据的特征进行统一化处理，即使得不同类型的数据具备相同的特征，方便后续分析处理。在数据仓库、数据集成、机器学习等应用场景中，数据标准化是不可或缺的一环。数据标准化相关的专利申请也日益增多，其途径有很多，包括向国际标准组织申请标准规范专利、向省、自治区、直辖市所在国家/地区申请数据标准化专利、向企业申请数据标准化服务专利等。根据《专利法》规定，申请专利时需明确提出要设计或者制作什么样的数据标准化工具、过程、方法，及其目的。本文将以数据标准化为例，详细阐述数据标准化专利申请的方法、原则、流程、方式等相关内容。


# 2.标准化概述
数据标准化（Data Standardization）是指对原始数据按照一定的规则进行重新编码、整合、重组，得到标准化数据，从而可以用于建模、统计分析、决策支持等应用，提升数据质量。一般来说，数据标准化包括以下几类方法：
1. 字段值归一化：将不同类型的数据转换到同一种度量系统下，如将价格单位转换为元；
2. 数据变换：采用预定义函数或者算法，将原始数据经过计算或者变换的方式转换为标准数据，如转换为Z-score值；
3. 数据清洗：处理脏数据、异常数据、重复数据、无用数据等，使数据变得更加完整有效；
4. 数据合并：通过多个表或者文件将不同来源的数据进行整合，生成最终的数据集；
5. 属性分配：将字段的含义赋予特定含义，便于后续模型训练、理解等。 

在专利申请中，数据标准化的作用主要是在研发阶段对原始数据进行清理和转换，使其符合标准形式，并可用于建模、统计分析、决策支持等应用。



# 3.专利申请数据标准化的准备

数据标准化专利申请前期准备工作主要分为以下几个步骤：

1. 收集数据：需要收集到原始数据集合。由于数据标准化涉及数据的格式、内容、采集来源、处理手段等方面，因此需要完整且准确的原始数据。

2. 数据处理：对原始数据进行清洗、转换、汇总等操作，将其转化成标准形式。数据处理可以选择使用开源工具或商用工具完成，也可以自行编写代码。

3. 提交数据报告：提交数据报告，描述原始数据、数据处理的相关信息。

4. 描述数据标准化方法：需要详细描述数据标准化的方法、原则、过程、方式。

5. 对比其他技术：对数据标准化相关的其他技术，如数据流水线、数据分类、数据过滤等进行对比，找到共性点、差异点。

6. 阅读、理解专利法：需要阅读专利法第四条《创新实践》中的“数据标准化”，并了解相关专利要求。

7. 提交专利申请表：完成申请表的填写，说明需要申请的数据标准化方法、原则、过程、方式，并提交给专利局。

# 4.专利申请数据标准化的流程

数据标准化专利申请的一般流程如下：

1. 检查申请对象：在核实申请人的身份信息、研究方向、公司实力、技术水平、市场份额等方面，核实是否能够提供专利申请。

2. 准备材料：收集原始数据，准备数据报告，描述数据标准化方法。

3. 起草审查意见书：根据专利法第五十一条关于专利申请程序条款的规定，起草审查意见书，陈述申请理由、依据、申请主体、申请事项、代理权、侵权责任等。同时，需要说明申请所涉及的专利类别、专利申请主旨、使用限制、著作权等因素。

4. 审查专利权威机构：申请者提交专利申请后，专利局或专利权威机构会审核该专利申请，核实申请者及其代理人的身份、履行职务、资格、法律承诺、责任承担、申请是否妥当、实施效果、争议解决等情况。

5. 咨询专利评估委员会：如果对申请者的专利能力和资格没有疑问，专利局或专ik权威机构会推荐专利评估委员会对申请者的专利情况进行评估。评估委员会会就专利申请人的案件、影响因素、申请期限、技术措施、专利支付情况、可实现性、可维权性等方面的建议和指导，提供专利评估报告。

6. 评估审查结果：审查专利权威机构审核完毕之后，提交的专利申请就会进入评估审查阶段，专利评估委员会会进行专利评估，找出适合的专利申请人。

7. 提交专利申请：根据专利法第三十八条规定，专利申请人及其代理人应当在两周内提交专利申请。专利局会审核并批准专利申请，并填写入帐册。

8. 开具专利证书：专利申请批准签发证书，并副本发送至申请人。

9. 撤销申请：申请人申请撤销申请，可申请注销专利。

# 5.专利申请数据标准化的注意事项

1. 注意公开披露数据：必须对标准化后的数据进行公开披露，并提供原数据供原作者验证。

2. 考虑使用免费数据：可以考虑使用免费数据进行数据标准化，但应当注意免费数据不能完全满足数据标准化需求。

3. 使用正版数据：可以使用从官网购买的数据进行数据标准化，但应尽量避免使用非法网站、官方数据。

4. 在国外数据上进行数据标准化：可以在国外专利申请中使用国际专利评估委员会进行数据标准化，但仍然需要按照专利法的相关规定公开披露数据。

5. 合理预算：数据标准化一般需要比较大的投入，建议做好资源预算。
