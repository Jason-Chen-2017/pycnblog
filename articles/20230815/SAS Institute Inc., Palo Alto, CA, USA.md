## 2.SAS产品和服务
### 2.1SAS统计产品SAS/STAT
SAS/STAT是SAS公司推出的功能最强大的统计分析软件。它支持的数据结构有表格型、树状型、平面型、空间型、时间序列型等；数据源可以是文本文件、数据库或多个关系型数据库；分析方法有Descriptive Statistics、Inferential Statistics、Multivariate Analysis、Cluster Analysis、Time Series Analysis、Regression Analysis等。SAS/STAT具有高度灵活的界面和直观的图形展示，能够有效提升分析效率。
### 2.2SAS交互式分析环境SAS/ETS
SAS/ETS是SAS公司推出的一款交互式数据分析环境，用于处理复杂的数据分析任务。它提供了从数据导入到模型训练和结果呈现的完整分析过程，并对交互式分析和编程能力要求较低，适合数据分析工作者使用。
### 2.3SAS可视化分析SAS Visual Analytics
SAS Visual Analytics是SAS公司推出的一款基于Web技术的可视化分析平台，支持多种数据类型，如文本、表格、图像、时间序列、空间数据等。通过拖放组件、调整参数、数据连接、表达式编辑、结果呈现等方式，可以快速创建具有独特性质的可视化分析报告。
### 2.4SAS云平台SAS Cloud Platform (SCP)
SAS SCP是SAS公司推出的一款用于在云端部署、运行和管理各种SAS产品和服务的解决方案。通过利用SCP，SAS用户可以访问云端资源，并按照既定的计划、容量和使用量购买虚拟服务器，实现在云端的高可用性和可扩展性。
### 2.5SAS移动设备APP（SAS On-demand Mobile App）
SAS On-Demand Mobile App是由SAS公司推出的免费的移动应用程序，用于帮助企业跨设备访问SAS的商业智能分析工具。该APP能够将复杂的数据分析流程转换成移动终端上的简单操作，降低了数据分析的门槛，提升了工作效率。
### 2.6SAS开放模型管理器Open Model Manager
SAS Open Model Manager是一个模型管理工具，用于构建、版本控制和分享数据分析建模项目。它提供了一个基于Web的模型仓库，其中存储着所有模型的元数据信息。通过它，SAS用户可以轻松地搜索、比较、复制、共享和更新模型。
## 3.核心算法原理和具体操作步骤以及数学公式讲解
### 3.1决策树算法
#### （1）原理
决策树算法（decision tree algorithm），也称作决策树学习算法，属于分类学习方法。它的主要思想是在数据集上构建一棵树，根节点代表数据的汇总信息，叶子结点代表的是数据的类别。通过对数据进行分析和测试，对原始数据进行分类。
#### （2）具体操作步骤
1. 数据预处理：对缺失值进行处理，根据数据量大小，采样或重抽样的方式对数据进行处理。
2. 属性选择：根据划分属性的相关指标进行优先级排序，选择排名前几的属性进行划分。
3. 生成决策树：使用ID3、C4.5、CART等方法生成决策树。
4. 测试：使用测试数据集对生成的决策树进行评估。
#### （3）数学公式
```
P(C|X)=P(X|C)*P(C)/P(X)
信息增益=信息熵 - 平均信息熵
信息熵=-SUM[pi*ln(pi)]
平均信息熵=(entro(D)-SUM[pi*entro(Di)])/|Y|
CART回归树的形式：y=f0+f1*x+...+fn*xn
```
### 3.2K均值聚类算法
#### （1）原理
K均值聚类算法（K-means clustering algorithm），也称作均值迁移算法。该算法是一种无监督学习算法，用于将给定的数据集分割成k个簇，使得同一簇中的点尽可能相似，不同簇中的点尽可能不相似。
#### （2）具体操作步骤
1. 初始化簇中心：随机选择k个点作为初始的簇中心。
2. 分配每个样本到最近的簇中心：将每个样本分配到距离其最近的簇中心。
3. 更新簇中心：将每个簇的中心移动到簇中的所有样本的均值。
4. 检查停止条件：若某次迭代没有改变簇的分配，则停止算法。
#### （3）数学公式
```
E(x)=sum(xi*mi)/(sum(mi))   mi:第i个样本属于簇j的频数  xi:第i个样本
R^2 = (SSres/(SSreg + SSE)) * (N - 1 / N - K)    SSres:残差平方和     SSreg:模型的拟合平方和     SSE:误差平方和
判别式分析法的假设检验：检验各个假设是否成立，例如系数是否显著小于0、标准差是否显著大于0。一般用Fisher精确法求最大似然值，然后用chi-squared或F分布进行检验。
PCA:主成分分析，通过求解样本协方差矩阵、特征值和特征向量，将变量按重要程度排列。
```
