# 2. 概念和术语
## 2.1 什么是机器学习？
机器学习(Machine Learning, ML)，是让计算机学习的一种学科，它利用数据的历史经验和规则，对未知的数据进行预测或分类。其目的是为了找到能够精确地模仿或解释已有数据的模型，并利用这个模型对新的数据进行预测、分类甚至控制。机器学习可分为监督学习、非监督学习、半监督学习及强化学习五大类。

## 2.2 为什么要使用机器学习？
随着互联网的发展，海量的数据涌现出来，这些数据带来的新机遇和挑战也是促使研究者们探索的方向之一。众所周知，传统的计算机只能处理少量数据，而当数据数量变得巨大时，传统的计算机的处理能力就会成为瓶颈。因此，人们又想到了另一种解决办法：让计算机自己学习。通过大数据和其他信息，计算机不断地去训练自己的模型，不断地去学习新的知识，从而有效地解决一些具有挑战性的问题。机器学习就是这样一种可以自动获取数据的计算机程序。

目前，机器学习的应用非常广泛，如图像识别、文本分析、垃圾邮件过滤、广告推荐、搜索排序、财务风险管理、生物标记、病毒检测等。简单来说，机器学习就是让计算机“学习”的过程。

## 2.3 什么是深度学习？
深度学习(Deep Learning, DL)是机器学习的一个子集，是指机器学习方法的集合，研究如何使计算机系统可以基于数据学习，并找到能够有效解决某些任务的模型。深度学习涵盖了多层次的神经网络结构、特征学习、非线性映射、概率分布估计等技术。深度学习可以解决很多不同领域的问题，包括图像识别、语音识别、语言理解、机器翻译、生物信息、金融市场预测、股票分析等。

深度学习的特点包括：

1. 模型一般较复杂，包含多个隐藏层节点；
2. 数据量大，样本多，需要大量数据训练；
3. 需要大量计算资源，加快训练速度；
4. 对中间结果的可视化分析十分重要；
5. 解决复杂问题时的表现优势。

## 2.4 机器学习算法的分类
机器学习算法按照输入的形式和输出的形式可以分为四种类型：

1. 有监督学习（Supervised Learning）：对于给定的输入-输出对进行训练，利用输入数据来预测输出值。典型的有监督学习算法包括支持向量机（SVM），朴素贝叶斯（Naive Bayes）和逻辑回归（Logistic Regression）。
2. 无监督学习（Unsupervised Learning）：无须提供目标值，通常采用无监督的方式发现数据内的模式。典型的无监督学习算法包括聚类（Clustering）和谱聚类（Spectral Clustering）。
3. 半监督学习（Semi-Supervised Learning）：结合有监督学习和无监督学习的策略。通过对有标签数据的学习，利用无标签数据推断出标签，增强模型的鲁棒性和效果。典型的半监督学习算法包括Label Propagation、Self Training、Co-Training、Graph Co-Training等。
4. 强化学习（Reinforcement Learning）：是指让计算机在不断探索过程中学习最佳的行为方式。典型的强化学习算法包括Q-Learning、SARSA、Actor-Critic等。

## 2.5 深度学习算法的分类
深度学习算法按照其所构建的模型结构可以分为两大类：

1. 神经网络（Neural Network）：由多个层连接的感知器组成，构成一个深层次的神经网络。典型的深度学习算法包括卷积神经网络（CNN）、循环神经网络（RNN）、递归神经网络（RNN）等。
2. 集成学习（Ensemble Learning）：集成学习是在多个基学习器之间共享训练数据的同时，提升最终预测能力。典型的集成学习算法包括随机森林（Random Forest）、AdaBoost、GBDT、Xgboost等。

## 2.6 机器学习和深度学习的区别
与普通的机器学习算法相比，深度学习在以下几个方面有明显优势：

1. 更复杂的模型结构：深度学习的模型多采用多层次的神经网络结构，使得模型更具表征能力，更适用于复杂的函数拟合问题。
2. 更高的学习效率：由于梯度下降算法的迭代次数大大减少，使得深度学习模型可以较为快速地进行训练和学习。
3. 全局学习能力：深度学习模型具有全局视野，能够对整体数据进行建模，因此能够捕捉到数据的全局特征，从而对复杂的问题建模。
4. 大规模数据集的利用：深度学习模型通常对大规模的数据集有很好的适应性，尤其是对长序列、图像、声音等数据。
5. 可解释性：深度学习模型往往具有高度的可解释性，可以帮助人们理解模型为什么会做出特定预测。

总之，深度学习已经成为近几年发展起来的新兴领域，虽然它的理论和模型框架仍有待完善，但它的优点已经得到充分认识和应用。因此，掌握深度学习算法和工具的技能显得至关重要。
