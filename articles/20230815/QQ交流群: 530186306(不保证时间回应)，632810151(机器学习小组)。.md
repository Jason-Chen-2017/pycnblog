# 2.AI概览
## 1.1 AI的定义
人工智能（Artificial Intelligence，简称AI），通常指让电脑具有智能的机器人、人工智障、通用计算设备、助手等产品和服务。它一般被定义为由自然界产生的智能机械所构成的集合。例如，AlphaGo击败了围棋世界冠军李世石，提出了一种基于人类博弈的下棋策略，AlphaZero则在围棋中实现了完全的自我对弈，并且在五子棋、象棋、俄罗斯方块、黑白棋、等棋游戏上击败了人类的先进水平。这些都是人工智能的典型例子。
## 1.2 AI发展的历史
### 20世纪70年代末到90年代初期
计算机的运算能力远远不能满足信息处理的需要。由于计算机只能执行简单的任务，只能对少量的数据进行处理，因此人们开始着力解决计算机处理大数据时的效率低下的问题。
计算机专家们发现，可以把大型数据集划分成若干个较小的子集，每一个子集只做简单的统计计算，就可以得到结果。这种处理方式称为MapReduce。这项技术在很长一段时间内取得了成功。但是随着数据量的增大，MapReduce也遇到了性能瓶颈，因为单个任务的执行时间过长。
为了改善这个问题，一些人开始着手探索其他的方法。如阿兰·图灵提出的“计算理论”、马尔可夫链蒙特卡罗方法、贝叶斯推理方法等。这些方法既要考虑数据的复杂性，又要快速处理。计算机专家们开始开发更快的处理方法，如并行处理、分布式计算、符号主义理论等。这些方法帮助计算机迅速处理海量数据，并使其达到前所未有的水平。
但随着技术的发展，计算机专家们发现，大数据仍然无法解决所有问题。对于某些特定领域，比如图像处理、语音识别、语言处理等，计算机还是无法胜任。于是，人工智能开始崛起。
### 2010年以来
随着互联网的发展，传统行业向移动端转型。用户的需求越来越多元化，各种业务都涉及到计算机。此时，各种领域的计算机专家们相聚在一起，共同努力，相互合作，逐渐形成了一个完整的计算机技术体系——人工智能。
人工智能包含三个重要领域：
- 人工智能技术：研究如何使计算机具备智能、模仿、学习、决策等能力。
- 人工智能应用：将人工智能技术应用到实际的业务领域。
- 人工智能研究：探索新型人工智能理论、技术及应用方法。
自20世纪70年代末到2010年以来，人工智能经历了激烈的变革。技术突飞猛进，创新能力极强，各种领域的研究领军人物纷纷出山，各个领域的研究人员纷纷投身其中。
其中，最具代表性的就是深度学习技术。深度学习技术的核心理念是通过学习，使计算机“自己学习”。深度学习技术最早在2012年由Google研究者<NAME>，<NAME>, <NAME>等人提出。
深度学习技术的主要特点有：
- 模型可以学习高层次的抽象表示，而不是简单的符号表示；
- 在很多情况下，不需要人为设置大量规则；
- 可以处理非结构化数据，如图像、文本、声音等；
- 通过采用多层次的递归和反馈循环，可以有效利用大量数据进行训练。
近年来，深度学习技术取得了巨大的成功。如Google、Facebook、微软、苹果、清华大学、Stanford大学等公司纷纷基于深度学习技术开发了一些新产品。截至目前，深度学习已成为人工智能领域的主流技术。
## 1.3 AI的研究机构、研究方向及其相关的主要成果
### 人工智能研究机构
截至2017年底，国内外的AI研究机构共计12个，按AI领域的垂直方向可分为如下几类：
- 数据挖掘与人工智能（DMRIA，Data Mining and RIse Artifical Inteligence）：DMRIA由多学科的研究人员组成，致力于收集、整理、分析和运用大量数据资源，开展数据挖掘、模式识别、机器学习和人工神经网络方面的研究。包括微软、Facebook、谷歌、雅虎、百度等七家公司，且每个公司均有专门的研究团队。
- 认知智能（CINA，Computational and Neural Intelligence）：CINA的研究目标是利用人类计算模型、学习系统、优化算法、认知机制、信息论等理论和技术，通过构建计算模型，构造大规模并行计算系统，开发新的认知计算模型，从而实现人类认知智能。包括卡耐基梅隆大学、斯坦福大学、伊利诺伊大学香槟分校、加州大学圣巴巴拉分校、普林斯顿大学等六所大学。
- 自然语言处理（NLP，Natural Language Processing）：NLP研究的是人类语言的特征、语法和语义，以及如何利用这些特征、语法、语义对人类语言进行建模、解析、生成、理解、翻译等处理。自然语言处理系统通常采用统计模型和规则引擎，实现对输入语句的理解、推理、分析、表达、组织、总结等功能。包括斯坦福NLP中心、北大中文信息学院、清华大学等。
- 机器学习（ML，Machine Learning）：ML的研究目标是通过数据编程、特征工程、模型训练、参数调优等方法，建立计算模型，实现对未知数据的预测、分类和聚类，同时确保模型的泛化能力。包括Stanford大学、清华大学、CMU、斯坦福大学、牛津大学、斯坦福大学等八所大学。
- 计算机视觉（CV，Computer Vision）：CV的研究对象是关于图像、视频、声音等各种感官信息的处理和分析，其研究目标是使计算机具备视觉智能。其研究重点是研制计算机视觉系统，使得机器能够按照人的观察和感受，对各种场景、物体、事物进行认识、识别、分类和检测。包括清华大学、斯坦福大学、浙江大学等。
- 机器人学（MR，Robotics）：MR的研究对象是人工生命，主要关注机器人器件、控制、模拟、学习、感知、通信等方面的研究。其研究重点是研究、设计、开发、验证、测试、部署、评估和应用各种人机交互、机器人控制、机器人导航、机器人规划、机器人学习等技术。包括清华大学、南京大学、康奈尔大学、波士顿大学等。
- 智能交互（EI，Interactive Intelligence）：EI的研究对象是人与计算机之间交互过程中的信息交换、意图理解、任务学习、反馈控制等方面的理论和技术。其研究重点是设计、开发、部署、验证、评估、维护智能交互系统，包括搜索、问答、指令执行、虚拟代理、深度神经网络等。包括百度、腾讯、讯飞、搜狗等公司。
- 物联网（IoT，Internet of Things）：IoT的研究对象是物理世界中网络的终端设备（Sensor）、传输介质（Network）和信息处理系统之间的互联互通，以及如何使之协同工作、融合资源、提升效率的理论和技术。其研究重点是设计、开发、部署、验证、评估、优化物联网系统。包括英特尔、瑞士苏黎世联邦理工、联想等公司。
- 数据库系统与认知技术（DBSTAC，Database System And Cognition Technology）：DBSTAC的研究对象是人工智能系统如何在数据库环境下运行，以及如何利用数据库技术和认知技术来支持人工智能系统的运行。其研究重点是设计、开发、部署、验证、评估、优化数据库系统及其上人工智能系统所需的计算机硬件、软件、算法、工具和框架。包括微软、卡耐基梅隆大学等。
- 信息安全（IS，Information Security）：IS的研究对象是信息技术（IT）和安全技术之间关系的基础理论和技术，其研究重点是设计、开发、部署、验证、评估、优化信息安全系统。包括美国国防部、加拿大艾伯威尔国际、德国科学院等。
### 人工智能的研究方向
除了以上列举的各个研究机构之外，还存在着一些特殊的研究方向。它们包括：
- 多模态、多样化与交互的认知技术（CMNI，Cognitive Multimodality, Natural Interaction）：CMNI旨在探究人类日常生活中的多个输入信号的交互作用及其影响，并从多源、多种形式、不同维度、不同载体等方面，推动人工智能技术的跨越式发展。
- 深度学习、强化学习、可解释性和鲁棒性的理论与技术（DRLRB，Deep Reinforcement Learning, RL Robustness and Beyond）：DRLRB试图为强化学习、深度学习等最新技术的理论、方法和技术范式提供更充分、更全面的理论支撑。
- 行为经济学与管理（BEAM，Behavioral Economics And Management）：BEAM试图探讨智能体（Agent）在多方博弈、组织、竞争、合作等方面的行为经济学，并结合管理学、组织学等专业领域的知识，把人工智能技术和经济学紧密联系起来，研究人工智能在经济学中的应用。
- 可扩展、分布式、并行计算与系统的高性能计算技术（HPCC，High Performance Computing with Scalable Distributed Systems）：HPCC试图通过有效利用计算机集群的计算能力、并行处理能力、内存带宽等特性，实现分布式计算与系统的高性能。
- 多智能体、多样性与协同的综合协同系统（MIMCO，Multimodal Intelligent Coordination Systems）：MIMCO试图研究智能体、协同系统在智能体之间、智能体与环境之间、智能体与人的协同过程中的控制、指挥、协调、决策等方面的综合协同技术。
- 生物信息、遗传学与进化算法（BPA，Bioinformatics, Genetics and Evolutionary Algorithms）：BPA的研究目标是利用生物信息学、遗传学、进化计算等理论和技术，开发新的生物计算模型，来了解、建模、模拟人类的基因及其演化过程。
- 数据挖掘与机器学习的应用与解决方案（DMAAS，Data Mining and Machine Learning Applications Solution）：DMAAS试图利用数据挖掘、机器学习、模式识别等技术，解决复杂的问题，创造价值。
- 人工智能基础设施的管理与可靠性（AIMR，Artificial Intelligence Infrastructure Manageability and Reliability）：AIMR试图通过研究人工智能基础设施的管理、可靠性、成本、效率和效益等方面，提高人工智能的应用效率和质量，为广大决策者提供更加智能、高效的决策依据。
- 跨领域的人工智能技术及应用（XTAI，Cross-Disciplinary AI Technologies and Application）：XTAI试图为AI领域的其他研究领域提供基础的理论和技术支撑，以便更好地理解人工智能的发展趋势和应用潜力。
- 复杂系统中的人工智能算法（CSALA，Complex System Algorithm for AI）：CSALA试图研究复杂系统中人工智能算法的设计、分析、优化和应用。
