# 2.核心概念与联系
## 数据类型
Python中共有六个标准的数据类型：

1. 数字（Number）：int、float、complex（复数）。
2. 字符串（String）：str。
3. 列表（List）：list。
4. 元组（Tuple）：tuple。
5. 集合（Set）：set。
6. 字典（Dictionary）：dict。
其中，整数（int）、浮点数（float）和复数（complex）都是数字类型，字符串（str）用于表示文本信息，列表（list）、元组（tuple）和字典（dict）则分别用于存储序列和映射。

## 运算符
Python中的运算符包括以下几类：

1. 算术运算符：加号（+）、减号（-）、乘号（*）、除号（/）、整除（//）、取余（%）、幂运算符（**）。
2. 比较运算符：等于（==）、不等于（!=）、大于（>）、小于（<）、大于等于（>=）、小于等于（<=）。
3. 逻辑运算符：与（and）、或（or）、非（not）。
4. 赋值运算符：等于（=）、加等于（+=）、减等于（-=）、乘等于（*=）、除等于（/=）、整除等于（//=）、取余等于（%=）。

注意：Python中没有 ++ 或 -- 的自增或自减运算符，只能直接用 += 和 -= 来实现。

## 分支结构
Python中的分支结构有两种：

1. if语句：if... elif... else。
2. for循环：for... in... 。

## 循环结构
Python中的循环结构有两种：

1. while循环：while... :。
2. range()函数：类似于C语言中的for循环。

## 函数定义
Python中的函数定义一般遵循如下规则：

```python
def function_name(*args):
    # 函数体
    pass
```

其中，函数名可以使用下划线连接多个单词，如`my_function`。`*`后面的变量名`args`，即函数的参数，可以是一个或多个。在函数体中，你可以通过参数传递值给函数，或者在函数内部返回值。

函数调用时，函数名后面加上括号`()`，然后传入相应的参数即可。例如：

```python
result = my_function("hello", "world")
print(result)
```

## 模块导入
Python中的模块导入比较复杂，但也可以简单理解成引用一个外部的`.py`文件，并调用里面的函数。比如，假设有一个`math_ops.py`文件，里面有两个函数：

```python
import math

def add(x, y):
    return x + y

def subtract(x, y):
    return x - y
```

那么，要使用这两个函数，就可以像下面这样导入该模块并调用函数：

```python
from math_ops import add, subtract

result = add(1, 2)
print(result)   # output: 3
``` 

由于在模块导入时只需要指定模块名（这里就是`math_ops.py`），而不需要指定文件的路径，所以导入也比较方便。另外，如果模块名太长，也可以缩写，只需把模块名放在当前目录下的 `__init__.py` 文件中即可，这样就可以像下面这样导入：

```python
from math_ops import add, subtract
from mypackage.subpackage.module_name import func1, func2
```
