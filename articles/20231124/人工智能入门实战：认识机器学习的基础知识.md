# 2.核心概念与联系
## 2.1 概念
机器学习(Machine Learning) 是一种数据分析方法，它使用计算机的编程能力来模拟、自动化地执行一系列的统计任务，从而达到预测、决策、分类等效果。它的输入是一个数据集，输出则是根据输入的数据，对未知数据进行预测或判别的某种模型。这种模型基于一定的规则和算法，并通过不断试错、优化来获得数据的预期输出。

机器学习研究的问题主要分为两类，一类是监督学习，另一类是非监督学习。
### （1）监督学习（Supervised Learning）
监督学习是指训练样本已有正确的标记（标签）。比如分类问题，即希望计算机根据输入的数据，自动划分数据到不同的类别之下；回归问题，即希望计算机根据输入的数据，预测出一个连续值作为输出结果。监督学习通常分为两步：

1. 数据准备：准备包含特征变量和目标变量的数据集。
2. 模型选择：根据数据的特点，选择合适的模型进行训练。如决策树、支持向量机、神经网络等。
3. 模型训练：根据选择的模型，利用训练数据对模型参数进行估计。
4. 模型测试：在测试集上测试模型的性能，通过调整模型参数或其他方式对模型进行改进。

### （2）非监督学习（Unsupervised Learning）
非监督学习是指没有标注数据，仅靠自身的特征提取能力进行学习，对数据没有先验的假设。比如聚类、降维、关联分析等。非监督学习通常分为三步：

1. 数据准备：准备包含特征变量的数据集。
2. 特征选择：选择数据中具有代表性的特征变量。
3. 模型训练：根据选择的特征变量进行聚类或其他降维等无监督学习算法。

## 2.2 术语
- 训练样本（Training set）：由输入特征向量组成的数据集，用于训练机器学习模型。
- 测试样本（Test set）：机器学习模型在新数据上的性能评估数据集。
- 特征向量（Feature vector）：描述输入数据的一串数字或符号组成的向量，用来刻画输入数据的一切特征。
- 标记（Label）：样本所属的类别或类别编号。
- 特征（Feature）：样本的某个属性，如人的年龄、体重、病症等。
- 训练（Training）：用已有的数据集拟合或建立机器学习模型的过程。
- 学习（Learning）：利用数据，从数据中提取知识，并转换为有用的信息或指令的过程。
- 预测（Prediction）：对新数据进行分类、回归等预测或建模的过程。
- 分类（Classification）：将样本划分到各个类别中的过程。
- 回归（Regression）：预测连续变量的值的过程。
- 聚类（Clustering）：将相似的数据样本合并成组的过程。
- 降维（Dimensionality reduction）：通过删除冗余特征，简化数据结构的过程。
- 标记变量（Target variable）：指示输入数据的真实结果。
- 监督学习（Supervised learning）：训练样本含有标记（目标变量），用于训练模型对输入数据的预测或分类。
- 无监督学习（Unsupervised learning）：训练样本不含有标记，用于训练模型发现数据内部的隐藏模式或关系。
- 回归问题（Regression problem）：预测数值的连续变量。
- 分类问题（Classification problem）：预测离散变量，即将输入数据划分到不同的类别中。
- 聚类问题（Clustering problem）：发现数据内的相似性，将相似的数据样本合并成组。
- 降维问题（Dimensionality reduction problem）：通过删除冗余特征，简化数据结构。
- 参数（Parameter）：表示模型的设置或超参数。
- 正则化项（Regularization item）：用来控制模型复杂度的项。
- 损失函数（Loss function）：衡量模型训练误差的函数。
- 代价函数（Cost function）：衡量模型预测误差的函数。
- 最大似然估计（Maximum Likelihood Estimation）：根据训练数据计算模型参数的方法。
- EM算法（Expectation Maximization Algorithm）：用于高斯混合模型参数估计的算法。
