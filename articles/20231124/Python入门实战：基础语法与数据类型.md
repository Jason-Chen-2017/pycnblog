# 2.核心概念与联系
## 2.1 概念总览
Python中的主要术语汇总如下：

1.变量：即存储数据的容器，使用标识符表示。在程序执行期间，变量的值可以发生变化。

2.数据类型：指明变量能够存储什么样的数据，可以分为数字类型、序列类型、映射类型、集合类型。

3.表达式：用于求值运算结果的语句。

4.语句：完整的代码片段，用于完成某项任务，一般以句号结尾。

5.注释：用来说明代码的文字，不会被执行。

6.条件语句：用来控制程序执行流程的语句，根据指定的条件选择执行不同的代码块。

7.循环语句：用来重复执行代码块的语句。

8.函数：用来封装一组相关代码并赋予其名称，方便调用。

9.模块：一个独立的文件，包含一系列定义和实现代码，可被导入到主程序文件中使用。

10.包：一个目录结构，里面包含一系列模块，可通过包的导入方式引用。

11.标准库：Python安装时默认带有的库。

12.第三方库：在互联网上已经存在的库。

## 2.2 基本语法与运算符
### 2.2.1 基础语法
#### 2.2.1.1 缩进规则
Python中的缩进规则很简单，一般来说，只要一行或者多行语句不是以缩进开始，那么就属于错误的语法格式。除非是定义一个新的函数或类的定义。

#### 2.2.1.2 编码规范
为了使得Python代码的可读性更好，我们应该遵循一定的编码规范。最重要的是用4个空格进行缩进，不要使用Tab键。每个py文件开头都需要添加类似这样的描述信息：#!/usr/bin/python。

#### 2.2.1.3 单行注释
Python中的单行注释以井号开头：

```
# This is a single line comment.
```

#### 2.2.1.4 多行注释
Python中的多行注释可以用三个双引号或三重双引号来创建：

```
"""This is a multi-line 
comment."""
'''It can also be written using triple quotes.'''
```

#### 2.2.1.5 print() 函数
print() 是输出文本的函数。该函数可以打印多个参数，用逗号隔开，每个参数后自动添加换行符。如果希望不换行，可以在末尾加上一个逗号。

```
>>> print("Hello world!")
Hello world!
>>> print("Hello", "world")
Hello world
>>> print("Hello", "world", end="")
Hello world
```

#### 2.2.1.6 input() 函数
input() 函数获取用户输入，返回一个字符串：

```
name = input("Please enter your name: ")
age = int(input("Please enter your age: "))
print("Your name is ", name, "and you are ", age)
```

#### 2.2.1.7 数据类型转换
有时候，我们需要把数据从一种类型转换为另一种类型，比如整数转化为字符串：

```
string_num = str(123)    # 将整数 123 转换为字符串 '123'
int_num = int('456')     # 将字符串 '456' 转换为整数 456
float_num = float('789.0')   # 将字符串 '789.0' 转换为浮点型 789.0
```

#### 2.2.1.8 运算符
Python中共有以下几种运算符：

1. 算术运算符：+ - * / % // **

2. 比较运算符：< > <= >= ==!=

3. 赋值运算符：= += -= *= /= %= //= **= &= |= ^= <<= >>=

4. 逻辑运算符：and or not

5. 位运算符：& | ^ ~ << >>

6. 成员运算符：in 或 not in

7. 身份运算符：is 或 is not

### 2.2.2 数字类型
#### 2.2.2.1 整形（int）
整数类型，通常叫做int，是一种不可变数据类型，也叫作整型。Python中整数的大小没有限制，可以使用十进制、二进制、八进制、十六进制来表示。如果整数过大，超出了内存的限制，Python会报MemoryError异常。

示例：

```
a = 123      # 十进制表示
b = 0xABC    # 十六进制表示
c = 0o123    # 八进制表示
d = bin(10)   # 将 10 转换为二进制字符串 b'1010'
e = oct(10)   # 将 10 转换为八进制字符串 '0o12'
f = hex(10)   # 将 10 转换为十六进制字符串 '0xa'
g = int('10', base=2)   # 使用二进制字符串 '10' 创建整数 2
h = int('ff', base=16)  # 使用十六进制字符串 'ff' 创建整数 255
```

#### 2.2.2.2 浮点型（float）
浮点型数据类型，通常叫做float，是表示实数的类型，也就是带小数的数字。它的大小与计算机的内存容量有关系。

示例：

```
a = 3.14       # 精确的表示圆周率
b = 1.0e-3     # 表示 1/1000 的 float 数
c =.5         # 表示 0.5 的 float 数
d = 10.        # 整数部分默认为 10
e = 3E8        # 用 E 来表示 3*10^8 的 float 数
f = 3.14e-10   # 用 e 来表示 3.14*10^(-10) 的 float 数
```

#### 2.2.2.3 复数型（complex）
复数型数据类型，通常叫做complex，是表示虚数的类型，由实数部分和虚数部分构成。复数的形式是 (real + imagj)，其中 real 和 imagj 分别表示实部和虚部。虚数部分的 j 是固定的记号，用于区分实数部分和虚数部分。

示例：

```
a = 3 + 5j   # 表示 3 + 5i 的 complex 数
b = 5j       # 表示 5i 的 complex 数
c = -2.5 + 3J    # 表示 -2.5 + 3i 的 complex 数
```

#### 2.2.2.4 类型转换
Python提供了很多内置函数用于转换不同类型的值。比如，将整数转换为字符串：

```
str(123)           # 返回 '123'
repr(123)          # 返回 '123' （官方推荐的输出形式）
oct(123)           # 返回 '0o173'
hex(123)           # 返回 '0x7b'
chr(97)            # 返回 'a'
ord('a')           # 返回 97
bool(123)          # 返回 True
bool('')           # 返回 False
int('123')         # 返回 123
float('3.14')      # 返回 3.14
complex('2+3j')    # 返回 (2+3j)
list(('apple', 'banana'))   # 返回 ['apple', 'banana']
tuple(['apple', 'banana'])  # 返回 ('apple', 'banana')
set(['apple', 'banana'])    # 返回 {'banana', 'apple'}
dict({'name': 'Alice'})     # 返回 {'name': 'Alice'}
```

### 2.2.3 布尔类型
布尔型数据类型，通常叫做bool，只有True和False两种值。布尔类型的值可以用在条件判断中，或者作为函数的参数和返回值。

示例：

```
flag = True
if flag:
    print('True')
else:
    print('False')
    
def my_func(value):
    if value < 0:
        return True
    else:
        return False
        
print(my_func(1))      # Output: False
print(my_func(-1))     # Output: True
```

### 2.2.4 字符类型
字符型数据类型，通常叫做str，是由零个或多个Unicode字符组成的序列，表示为引号(' ')或双引号("")。字符串通常用于表示人类可读的文本信息。

示例：

```
s = 'hello world'
t = "I'm OK"
u = r'\n'         # 原始字符串，其中 \n 被视为反斜杠和 n 的组合
v = u'中文'      # Unicode 字符串
w = """Hello, world!"""   # 多行字符串，用三重引号括起来的字符串
x = '''Welcome to 
    Pythoneers!!!'''     # 多行字符串，用三重引号括起来的字符串
y = len(s)             # 获取字符串长度，返回 11
z = s[6]               # 获取第七个字符 'w'，索引从 0 开始
```

### 2.2.5 列表类型
列表类型，通常叫做list，是一种有序的、可变的、可以保存任何类型值的序列。列表中元素的位置编号从0开始。

示例：

```
fruits = ["apple", "banana", "orange"]
numbers = [1, 2, 3, 4, 5]
nested_list = [['Apple', 'Banana'], ['Orange']]
empty_list = []
zeroes = list(range(10))     # 生成 10 个元素的列表 [0, 1, 2,..., 9]
strings = ''.join(map(str, numbers))    # 将列表 [1, 2, 3, 4, 5] 转换为字符串 '12345'
```

### 2.2.6 元组类型
元组类型，通常叫做tuple，是一种有序的、不可变的、可以保存任何类型值的序列。元组中元素的位置编号也是从0开始。

示例：

```
coordinates = (3, 4)
colors = ('red', 'green', 'blue')
person = ('Alice', 25, 'Engineer')
single_element = (1,)    # 如果元组中只有一个元素，则需要在元素后面添加逗号
```

### 2.2.7 字典类型
字典类型，通常叫做dict，是一种无序的、可变的、以键值对的方式存储数据的集合。字典中的值可以是任意类型，且键必须是不可变类型，如字符串、数字或元组。

示例：

```
grades = {"Math": 90, "English": 85, "History": 95}
students = dict([("Alice", 25), ("Bob", 20)])
employee = {}              # 空字典
```

### 2.2.8 NoneType类型
NoneType是一个特殊的类型，它的值只有一个，即None。None是一个占位符，表示缺少有效的值。None通常用在空值、尚未初始化的值、函数调用失败等场景中。

示例：

```
result = some_function()
if result is None:
    print("Nothing found.")
```
