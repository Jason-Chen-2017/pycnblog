# 2.核心概念与联系
Python作为一种高级语言，其具有强大的生态系统。为了更好地掌握Python，首先需要了解一些核心概念以及它们之间的关系。下面列出了一些概念的简单定义：
- **对象**：一般来说，所有东西都可以称之为对象，比如数字、字符串、列表、函数等。
- **变量**：存储对象的容器，可以赋予不同的值。
- **类型系统**：在程序执行期间，每个变量都有一个特定的类型。它决定了该变量能够被执行何种操作，以及变量所存储的数据的结构形式。
- **命名规则**：变量名必须由字母、数字、下划线或点组成，且不能以数字开头。
- **作用域**：不同的变量名可能指向相同的内容，但是它们处于不同的作用域，即它们在程序中的生命周期不一致。
- **引用计数**：为解决作用域问题，Python使用引用计数法来自动管理内存资源。
- **垃圾回收机制**：Python使用引用计数法来管理内存资源。当一个对象的引用计数变为0时，说明没有任何变量再引用它，此时Python会自动回收它所占用的内存空间。
- **注释**：以“#”符号开头的一行文本称为注释，用于解释代码或标记某些地方。
- **语句**：代码块的最小单元，可以是一个表达式或者一个声明语句。
- **表达式**：表达式求值后得到的值称为表达式的值。
- **运算符**：运算符用来执行特定操作，比如算术运算符、赋值运算符等。
- **控制流程**：通过控制流程，可以控制程序执行的顺序。
- **模块**：模块就是一个包含了多个语句的文件。
- **包**：包就是包含了多个模块的文件夹。
- **依赖管理器**：依赖管理器负责管理程序运行所需的第三方库。
- **虚拟环境**：虚拟环境就是隔离Python运行环境的工具。
- **迭代器**：迭代器是访问集合元素的惰性方法。
- **生成器**：生成器是一个返回一系列值的函数，每次调用它都会返回前一次调用之后产生的值。
- **异常处理**：异常处理是指在程序执行过程中发生错误时，通过捕获并处理异常的方式来保障程序的正常运行。
- **输入输出**：通过输入输出，可以向终端或文件传递数据。
- **文档字符串**：文档字符串可以让函数的功能、用途和用法等信息被外部阅读。
- **装饰器**：装饰器是改变其他函数行为的函数。
- **上下文管理器**：上下文管理器用于资源的申请和释放。
