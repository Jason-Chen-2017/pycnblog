                 

# 1.背景介绍


量化交易（Quantitative Trading）是指以计算机自动化的方法交易股票、期货或其他市场产品，并以有效的管理和控制手段进行交易。简而言之，量化交易就是利用算法技术来建立预测性的投资组合，在市场中发现符合预测的买卖信号，执行交易以获利。因此，量化交易的首要目的就是通过计算机程序来实现对人类大脑的模拟。量化交易涉及非常多的技术领域，包括机器学习、数学建模、统计分析、金融经济学、计算机科学等。本文主要关注的是最基本的量化交易技能，即使用Python语言掌握数据的导入、数据处理、数据可视化、回测和实盘交易的能力。文章结合自己的实际经验，从头到尾细致地讲解了如何掌握量化交易的核心技能。
