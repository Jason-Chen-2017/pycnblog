在信息技术行业中，数据的处理、分析和决策往往离不开编程语言的帮助，其中最常用的编程语言是Python。其功能强大且开源免费，可应用于各种领域，如数据科学、web开发、游戏编程等。很多初级的Python开发者，包括学校里的计算机基础教材中都会介绍一些基本的语法和基本的数据结构。而对于Python的流程控制，即如何编写更复杂的程序结构，还是比较少的介绍，特别是在企业级应用场景中，流程控制至关重要。因此，本文将从两个方面来介绍Python的流程控制相关知识：条件判断语句和循环语句。

# 2.核心概念与联系
## 2.1 条件判断语句
条件判断语句（英语：Conditional Statement）是指根据条件是否成立来执行不同的代码块。其一般形式如下：

if 条件表达式:
    执行语句1
    
elif 条件表达式:
    执行语句2
    
else:
    执行语句3
    
在上述代码块中，关键字 if 和 else 是必需的，分别表示当满足某个条件时执行的代码块，当所有前面的条件都不满足时，才会执行 else 中的代码块。多个 elif 可以用来表示一个范围内的多个条件，只有当所有的条件都不满足时，才会进入 else 中指定的代码块。

条件表达式可以是一个表达式或变量，当它的值为 True 时，对应的代码块才会被执行。比如：

x = int(input("请输入一个数字: "))

if x > 0:
    print("正数")
elif x < 0:
    print("负数")
else:
    print("零")

以上代码实现了简单的条件判断功能，当用户输入的数字大于0时，输出“正数”；当输入的数字小于0时，输出“负数”；其他情况均输出“零”。

## 2.2 循环语句
循环语句（英语：Looping statement）用于重复执行某段代码，通常是一个或多个语句。Python 的循环语句主要包括 for 循环和 while 循环。

### 2.2.1 for 循环
for 循环（英语：For loop）是一种非常常用的循环结构。其一般形式如下：

for var in sequence:
    statements
    
var 表示迭代变量，sequence 表示要遍历的序列对象（比如列表、字符串）。statements 表示循环体，可以有多条语句，但只能有一个缩进层次，并且必须跟着循环体的冒号(:)。

for 循环每次从 sequence 中取出一个元素赋值给 var，然后执行 statements 语句。如果 sequence 中没有值，则循环退出。

例如，我们可以使用 for 来计算 1~100 之间的质数个数：

count = 0   # 计数器初始化为0

for num in range(2, 101):   # 在 2~100 之间进行循环
    is_prime = True    # 每次循环开始都认为这个数是质数

    for i in range(2, num):   # 判断这个数是否是质数
        if num % i == 0:
            is_prime = False     # 如果发现有一个因子，则不是质数
            break                # 跳出当前循环
        
    if is_prime:          # 如果这个数是质数
        count += 1        # 计数器加一

print("质数的个数为:", count)   # 打印结果

上面这段代码中，首先定义了一个计数器 count，然后用 for 循环遍历范围为 2 ~ 100 的整数。每一次循环，我们都判断当前数是否为质数，方法是创建一个布尔型变量 is_prime 为 true ，然后对当前数进行分解，从 2 到 num-1 之间，看看该数是否能整除 num 。如果能整除，则 is_prime 变为 false ，并终止该循环。如果经过此过程之后，仍然还保持 is_prime 为 true ，则说明 num 是质数，并计数器 count 加 1 。最后，打印结果。

### 2.2.2 while 循环
while 循环（英语：While Loop）是另一种常用的循环结构。其一般形式如下：

while condition:
    statements
    
condition 表示循环条件，若为真（非零），则执行 statements 语句，否则跳过该循环。

while 循环最适合于循环次数不确定或者需要动态变化的情况。

例如，假设我们要统计 1+2+3+...+n 这个数列的和，但是 n 不一定给出，所以我们不能像直接求和那样写死一个值，而应该用 while 循环来完成这项任务：

sum = 0   # 初始化和为0

i = 1      # 设置起始值为1

while (True):     # 使用一个无限循环
    sum += i       # 求和
    i += 1         # 将 i 增加 1
    
    input_str = input("输入 'q' 结束：")   # 获取用户输入
    
    if input_str == "q":                  # 如果输入‘q’结束循环
        break                              # 跳出循环
        
print("和为:", sum)                        # 打印结果

上面这段代码中，设置了一个循环条件为 True （永远为真），通过 user_input = input() 函数获取用户输入，如果输入的是 q ，则终止循环。如果一直没有输入‘q’，那么该循环就一直继续下去，直到程序崩溃或手动停止。

## 2.3 运算符优先级
当表达式出现复杂关系的时候，运算符的优先级（Order of Operations）将影响计算的顺序。Python 提供以下六种运算符，它们按照优先级高低排列：

1. **
2. *, /, //, %
3. +, -
4. >, >=, <, <=,!=, ==
5. and
6. or

这些运算符的使用规则是：

- ** 表示指数运算符，例如 a**b 表示 a^b 。
- *,/,//,% 分别表示乘法、除法、整数除法、模运算符。
- +,- 表示加法、减法运算符。
- >,<,>=,<=,!=,== 表示比较运算符。
- and 表示逻辑与运算符，用于连接两个逻辑表达式，只有两个表达式同时为 True 时结果才为 True ，否则为 False 。
- or 表示逻辑或运算符，只要两个表达式中的任何一个为 True ，结果就为 True ，否则为 False 。
