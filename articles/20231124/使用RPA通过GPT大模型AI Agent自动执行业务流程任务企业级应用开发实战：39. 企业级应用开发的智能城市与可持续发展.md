                 

# 1.背景介绍


## 1.1 GPT-3
GPT-3(Generative Pre-trained Transformer-3)是一个基于Transformer的神经网络语言模型，可以生成文本、图像、音频等。该模型训练时使用了大量的海量数据，并预先对语言模型进行了训练，因此能够生成高质量的文本。2020年11月发布的白皮书宣布，GPT-3将在未来十年内成为真正的通用型人工智能。GPT-3可用于生成语言、图片、视频、音频等多种形式的内容。

## 1.2 项目背景
如今，城市建设工程越来越复杂化，而面对庞大的工程规模和日益增加的建设难度，如何提升城市的绿色发展、可持续发展成为了一个重要课题。在这个过程中，如何通过计算机科学的方式帮助城市建设者自动完成建筑施工、设计、审计、检测等各个环节任务，成为举足轻重的关键。

## 1.3 智能城市技术方案
本项目主要针对城市发展中的三个关键环节——施工、设计、审计与检测等，基于GPT-3技术方案进行自动化的流程自动化，整合云端资源实现数字化管控，完成城市建设任务，提升城市绿色发展、可持续发展能力。

# 2.核心概念与联系
## 2.1 RPA(Robotic Process Automation)
“RPA”（即“机器人流程自动化”），是一种计算机辅助技术，它借助现有的软件工具、硬件设备及人力，实现让电脑代替人的某些重复性工作，从而简化、加速或改变组织和管理的过程。目前，许多公司都在使用RPA解决一些重复性、耗时的工作。

## 2.2 GPT-3
GPT-3是一种基于Transformer的神经网络语言模型，它可以自动生成文本、图像、音频、视频等，属于开放领域AI模型。GPT-3训练时使用了大量的海量数据，并预先对语言模型进行了训练，因此可以生成高质量的文本。

## 2.3 项目背景
本项目中，主要采用的是IBM Cloud提供的Red Hat OpenShift平台作为基础设施，通过GPT-3技术结合开源库，开发出企业级应用，包括智能城市管理后台、建筑施工指引系统、全景视图导航系统等，实现业务流程的自动化、优化、监测，帮助建设者完成建筑施工、设计、审计、检测等各个环节任务。

### 3.2.2 模块功能详解
#### （1）建筑施工指引系统
建筑施工指引系统的目的是通过分析建筑工程信息，生成施工指导图，供建筑工人查看、理解工程信息，有效降低建筑施工上的工作负担。

系统分为两部分：
- 一部分由云端生成图形用户界面界面，用户输入建筑工程信息，比如地区、楼宇名称、工程类型等信息，系统根据输入的这些信息，向后端提交相应的请求，获得相应的建筑工程数据；
- 另一部分由后端生成建筑施工指导图。后端首先会接收到前端提交的数据，然后利用Python库进行数据清洗、数据存储、数据可视化等操作，将得到的数据转化为图形化格式。接着，基于第三方库pydotplus，根据接收到的建筑工程数据，生成建筑施工指导图。生成完毕的图形化文件可以输出至浏览器，供用户进行查看。整个过程不需要手动输入任何信息，只需简单的点击按钮即可运行系统。

#### （2）全景视图导航系统
全景视图导航系统的目的是通过摄像头采集建筑工程的照片及二维地图，提供完整的建筑工程全局视角，实现地图导航、立体感的呈现，极大提升建筑师的工作效率。

系统分为两部分：
- 一部分由云端生成图形用户界面界面，用户上传相关建筑工程照片，系统通过计算机视觉算法对照片进行分析，获取建筑结构以及地图信息；
- 另一部分由后端生成全景视图导航地图。后端首先会接收到前端上传的建筑工程照片，然后利用深度学习框架TensorFlow对其进行预测，提取建筑结构及其相对于地图的位置关系。接着，基于第三方库Open Street Map，结合建筑工程数据，生成导航地图。生成完毕的导航地图可以输出至浏览器，供用户进行查看。整个过程不需要手动输入任何信息，只需简单的点击按钮即可运行系统。

#### （3）运维自动化监测系统
运维自动化监测系统的目的是通过拍照或上传数据，实时获取传感器数据、环境状态，分析及处理，产生统计报表、预警提示，帮助建设单位提升管理水平。

系统分为两部分：
- 一部分由云端生成图形用户界面界面，用户上传相关数据，系统通过特定算法对图像进行分析，获取传感器数据及环境状态信息；
- 另一部分由后端生成运维报告。后端首先会接收到前端上传的数据，然后利用Python库进行数据清洗、数据存储、数据可视化等操作，将得到的数据转化为图形化格式。接着，基于第三方库matplotlib，根据接收到的运维数据，生成运维报告。生成完毕的图形化文件可以输出至浏览器，供用户进行查看。整个过程不需要手动输入任何信息，只需简单的点击按钮即可运行系统。
