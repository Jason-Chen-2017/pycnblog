                 

# 1.背景介绍


元组（tuple）是一个不可变序列类型，类似于列表。不同之处在于，元组中元素的值不能修改，而且元组中的所有元素都是按照顺序排列。创建元组的方式如下：

```python
t = (1, 'a', True)
```

元组也可以由一个表达式生成，即将一些已有的对象打包成元组。例如：

```python
a = 1, 2, 3     # 生成元组
b = ('hello', [1, 2])   # 嵌套列表的元组
c = a + b       # 将两个元组连接
d = c[1:3]      # 切片取出子元组
e = d * 2       # 通过重复操作符进行复制操作
```

元组的应用场景主要有以下几种：

1. 作为函数的参数：由于元组是不可变的，因此可以作为函数参数传递；
2. 返回多个值：返回多个值的函数或者方法都可以使用元组返回多个值；
3. 数据记录：数据库表、CSV文件等多种数据记录格式都支持元组存储数据；
4. 集合运算：元组之间的交集、并集、差集、对称差等集合运算都可以直接用元组实现。

# 2.核心概念与联系
## （1）元组的定义及特点
元组（tuple）是一个不可变序列类型，类似于列表。不同之处在于，元组中元素的值不能修改，而且元组中的所有元素都是按照顺序排列。创建元组的方式如下：

```python
t = (1, 'a', True)
```

元组也可以由一个表达式生成，即将一些已有的对象打包成元组。例如：

```python
a = 1, 2, 3    # 生成元组
b = ('hello', [1, 2])   # 嵌套列表的元组
c = a + b      # 将两个元组连接
d = c[1:3]     # 切片取出子元组
e = d * 2      # 通过重复操作符进行复制操作
```

## （2）元组的操作
### （2.1）访问元组元素
元组是按顺序存储的，因此可以通过索引（index）来访问其中的元素。下标从0开始，访问第i个元素只需将i放在方括号中即可：

```python
>>> t = (1, 'a', True)
>>> t[0]
1
>>> t[-1]
True
>>> t[1:]
('a', True)
```

### （2.2）更新元组元素
元组是不可变的，因此无法对其中的元素进行修改或删除。如果确实需要修改或删除元组中的元素，则只能创建新的元组。

### （2.3）合并元组
若要合并多个元组，则可以使用加法（+）运算符：

```python
>>> a = (1, 2, 3)
>>> b = (4, 5, 6)
>>> c = a + b
>>> print(c)
(1, 2, 3, 4, 5, 6)
```

### （2.4）比较元组
元组之间可以进行大小比较、相等性判断等操作，具体方法如下：

- 相等性判断：判断两个元组是否含有相同数量的相同元素，且对应位置上的元素也相同；
- 大小比较：元组的元素是不定长的，因此无法根据元素个数进行比较；
- 排序：可以使用sorted()函数对元组排序。

### （2.5）转换元组
元组可以转换成列表、字符串等其他形式，具体方法如下：

- 列表转换：通过list()函数可以将元组转换成列表；
- 字符串转换：通过join()方法可以将元组转换成以指定字符分隔的字符串。
