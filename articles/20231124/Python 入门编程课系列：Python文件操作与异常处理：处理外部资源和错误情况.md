# 2.核心概念与联系
## 文件操作

文件操作可以说是所有编程语言都需要掌握的基本能力。在本课程中，我们将通过一些实例代码以及图示来加强对文件的操作技巧。

### 文件打开模式

在Python中，打开文件的时候可以指定文件的打开模式。常用的打开模式如下所示：

 - `r` 读取模式：如果要读取的文件不存在，就会报错；
 - `w` 写入模式：如果要写入的文件不存在，就会创建新的文件，否则会覆盖原来的文件；
 - `a` 追加模式：如果要追加的文件不存在，就会创建新的文件，否则会在原文件末尾添加新的内容；
 - `rb` 以二进制形式读取模式；
 - `wb` 以二进制形式写入模式；
 - `ab` 以二进制形式追加模式；

### 文件关闭方法

在使用完文件之后，务必要记得关闭文件，防止占用过多系统资源。一般情况下，我们可以使用文件的`close()`方法来关闭文件。

```python
with open('file.txt', 'w') as f:
    # do something with file here...
```

上面的代码使用上下文管理器`with`，可以自动帮我们调用文件的`close()`方法，不需要再写额外的代码来关闭文件。

另外，还可以通过文件的`closed`属性判断文件是否已经被关闭。

### 输入输出操作符

Python中的标准输入输出操作符（`input()/print()`）用于与用户进行交互，读写控制台。在命令行运行Python程序时，默认使用的就是标准输入输出操作符。但是在文件输入输出时，也可以用`open()`函数打开文件进行读写。

以下代码演示了如何使用文件输入输出操作符进行读写操作：

```python
# write to a file using print() function
f = open('output.txt', 'w')
for i in range(10):
    print(i, file=f)
f.close()

# read from the same file using input() function
f = open('output.txt', 'r')
lines = f.readlines()
for line in lines:
    num = int(line.strip())   # strip whitespace characters like '\n' at end of line
    print(num * 2)
f.close()
```

上面的代码首先打开了一个文件`output.txt`，然后利用`range()`函数生成数字序列，并把每个数字用`print()`函数写入到文件中，最后关闭文件。然后再次打开文件，并利用`readline()`函数逐行读取，并把每行字符串转换为整数，并乘以2，打印出来。
