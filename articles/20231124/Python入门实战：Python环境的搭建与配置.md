### 安装Python
- Windows环境下，官网下载Python安装包安装即可；
- Linux环境下，可以使用包管理器进行安装，比如yum install python3。
- macOS环境下，可以从官网下载dmg安装包进行安装。

### 配置Python
- 创建路径：因为安装多个版本的Python可能会导致环境变量设置不当，导致找不到可用的Python版本，所以建议用户设置一个统一的Python目录，然后把不同版本的Python软链接到该目录下。
- 设置环境变量：一般情况下，只需要设置两个环境变量：PYTHONPATH和PYTHONHOME。PYTHONPATH指向Python安装路径下的site-packages文件夹，也就是模块文件存放位置；PYTHONHOME指向Python安装路径。

### IDE的选择和使用
Python有很多的集成开发环境(IDE)可以供选择，如IDLE、PyCharm、Spyder、Atom等。其中IDLE是最基础的Python集成开发环境，功能简单，适合小型脚本项目的快速编码。其它的三个集成开发环境则提供了更多的功能，如代码自动补全、代码调试、语法检查等。推荐使用的IDE是PyCharm，它拥有强大的代码编辑功能、运行调试功能、版本控制系统、单元测试功能等。PyCharm同时支持Windows、Mac和Linux平台。

### Python脚本文件的编写和运行
新建Python脚本文件（以py结尾的文件），编辑代码，保存文件。在终端输入python脚本文件名称启动运行，也可以打开IDLE工具，进入交互模式，执行代码。

### virtualenv创建虚拟环境
virtualenv是一个轻量级的Python环境管理工具，它能够帮你创建一个独立的Python环境，以避免系统Python环境的污染。你可以用virtualenv创建独立的Python环境，隔离依赖于你的项目的全局Python环境。虚拟环境也是一个临时的、隔离的Python环境，其中的包不会影响到系统Python环境。安装virtualenv后，你可以通过virtualenv命令来创建虚拟环境，然后激活虚拟环境，进入环境后就可以像在普通环境一样安装、卸载、升级、管理你的包。virtualenv命令在不同的操作系统平台下都可以找到，具体使用方法参考官方文档。

### Python语法和一些内置函数
- print() 函数用于打印输出文字到屏幕，可以接受任意数量的参数，逗号分隔，每个参数末尾默认添加换行符。
- input() 函数用于从键盘获取用户输入的字符串，并返回一个字符串类型的数据。
- len() 函数用于计算容器中元素个数。
- type() 函数用于显示变量数据类型。
- dir() 函数用于显示对象的所有属性和方法。
- str(), int(), float(), bool() 函数用于转换数据类型。

### 异常处理和调试方法
- try...except...finally块用于异常处理，当try子句中的语句发生异常时，程序就会跳过except子句并执行finally子句中的语句。
- assert用于断言，如果条件为False，assert语句会抛出AssertionError异常，停止程序的执行。
- pdb.set_trace() 函数用于调试，让程序暂停并进入调试器，方便查看变量状态及调用堆栈信息。

### Python模块和包管理
- pip (pip install package) 是Python包管理工具，能完成对Python第三方包的安装。
- setuptools (python setup.py build/install) 是Python打包工具，用于将Python模块打包成包发布。
- wheel (wheel name.whl) 是Python分发标准，它规定Python第三方包的命名方式。

### 文件处理
- open() 函数用于打开文件，默认模式为读取文本模式'r'，写入文本模式'w'，追加模式'a'，二进制模式'b'。
- read() 方法用于读取文件全部内容。
- write() 方法用于向文件写入内容。

### Web编程和爬虫
- requests 模块用于发送HTTP请求，解析响应内容。
- BeautifulSoup 模块用于解析HTML、XML页面。
- Scrapy 框架是一个用于Web爬取的快速、高效的框架，主要用于数据挖掘、网络爬虫等应用场景。

### 数据分析和机器学习
- NumPy 和 SciPy 是Python科学计算模块，用于进行矩阵运算和统计分析。
- pandas 可以用来进行数据清洗、处理、分析，还有一个强大的图表展示功能。
- scikit-learn 是一个基于SciPy的机器学习模块，用于数据预处理、特征抽取、分类、回归等机器学习任务。

### Qt快速应用开发
- PyQt是一个跨平台的Python库，用于构建高性能的GUI应用程序。
- PySide是一个基于Qt的Python模块，支持Qt的所有功能，适合桌面应用程序。

### 其他常见的Python开发技术
- unittest 测试模块，用于编写和运行单元测试。
- Flask 框架是一个基于Werkzeug的WSGI web框架，用来快速开发RESTful API。
- SQLAlchemy 是Python SQL数据库接口，用来操作关系数据库。
- asyncio 模块，用于实现异步IO编程，可以利用多核CPU提高程序的运行速度。
