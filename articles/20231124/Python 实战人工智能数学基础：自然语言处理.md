                 

# 1.背景介绍


自然语言处理(NLP)是计算机科学领域的一个重要方向。它利用计算机对文本进行建模、理解、分析、生成等功能。NLP 有两种主要的方式：语音识别与机器翻译。本文将介绍 NLP 的两大类应用——信息抽取与句法分析。
# 2.核心概念与联系
## 2.1 信息提取
信息抽取(Information Extraction, IE)是一个计算机从文本中自动地抽取结构化信息的任务。按照信息抽取的方法论，分为特征抽取、链接发现和关系抽取。
### 2.1.1 特征抽取 Feature Extraction
特征抽取指的是从文本中抽取出有意义的信息。一般来说，特征包括实体（Entity）、描述性词语（Descriptors）、动词短语（Verb Phrase）、名词短语（Noun Phrase）。特征抽取可以用于信息检索、数据挖掘、问答系统等领域。如下图所示。
### 2.1.2 链接发现 Linking Discovery
链接发现是对多个实体间的关系进行发现的过程。根据链接的强弱，可以分为基于规则的链接发现、基于统计的链接发现、基于知识库的链接发现。如下图所示。
### 2.1.3 关系抽取 Relation Extraction
关系抽取是从文本中抽取出有意义的关系，其目标是找到两个或更多实体之间的联系。关系抽取的关键在于设计精确且有用的模式。关系抽取的流程通常分为三步：实体发现、角色标注和关系抽取。如下图所示。
## 2.2 槽位填充 Slot Filling
槽位填充(Slot Filling, SF)是一个依据一定的模板完成信息的填写任务。通俗点说，就是按顺序将文本中的词语插入到空白的模板位置上。槽位填充也可称为序列标注(Sequence Labeling)，是计算机视觉、自然语言处理和人工智能的一个重要研究方向。如下图所示。
