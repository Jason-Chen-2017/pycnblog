# 2.核心概念与联系
## 2.1 什么是变量？
变量（Variable）是一个用于存储数据的占位符，你可以把它想象成一个盒子，盒子里装着你想要存储的数据，而这个变量就是这个盒子的钥匙。你可以通过这个钥匙来获取你所存储的数据。那么，什么是变量呢？简单的说，变量就是分配了存储空间的名字。它是一个抽象的概念，它是计算机中一个重要的组成部分，用来保存各种值的容器。
## 2.2 变量的分类
变量分为四种类型：
- 局部变量
- 全局变量
- 参数传递
- 返回值
### 2.2.1 局部变量
局部变量（Local variable）是在函数体内声明的变量，它的作用域只能在当前函数内部，函数执行完毕后就释放了该变量所占用的内存空间，局部变量通常只在函数内部使用。例如：
```python
def greet(name):
    message = "Hello, " + name + "." # 局部变量message
    print(message) # 使用局部变量message
    
greet("Alice") # Hello, Alice.
print(message) # 报错: message未定义！因为变量message只在greet函数内有效。
```
上述例子中的`message`变量是局部变量，它可以在`greet()`函数内部使用，但在外部无法访问。如果试图在外部使用，会出现`NameError`。
### 2.2.2 全局变量
全局变量（Global variable）是指在整个程序生命周期内都存在的变量，它可以在多个函数之间共享数据。全局变量可以通过在函数前面加上关键字`global`来修改。例如：
```python
x = 10 # 全局变量x
y = 20 # 全局变量y

def add():
    global x # 修改全局变量x的值
    y += x * 2 # 对y进行运算
    return y 

z = add() # 函数add的返回值为40
print(z) # 输出结果：40
```
在上述例子中，`x`和`y`是全局变量，函数`add()`通过`global`关键字修改了全局变量`x`，然后对`y`进行了一些计算，最后返回了结果。此外，函数`add()`还没有修改`x`，但是它可以通过其他方式访问到`x`，因此他也不是完全无用。
### 2.2.3 参数传递
参数传递（Parameter passing）是指当你调用函数时，将实际的参数传入到函数中，这样就可以使得函数工作。参数传递可以是值传递还是引用传递。在Python中，默认情况下，所有形式参数都是值传递，也就是说函数内部对形参的修改不会影响到函数外部的实参。然而，某些时候，我们希望函数内部对参数进行修改，这样可以实现更复杂的功能。这时，我们就可以通过引用传递的方式来解决这个问题。以下代码展示了如何通过引用传递来修改形参的值：
```python
a = [1, 2] # a是一个列表
b = a # b和a指向同一块内存空间
c = list(a) # 通过list()函数复制列表a得到新的列表
a[0] = 'hello' # 通过a改变值
print(a) # ['hello', 2]
print(b) # ['hello', 2]
print(c) # [1, 2]
```
以上代码展示了值传递和引用传递之间的区别，其中函数`copy()`通过列表解析式复制了一个新的列表，使得`a`和`c`指向不同的内存地址，所以修改`a`的值不会影响到`c`。
### 2.2.4 返回值
返回值（Return value）是指函数执行结束后，向调用方提供的一种机制，用于接收函数执行后的信息。函数执行结束后，会有一个返回值，这个值也可以是一个对象或者一个集合。下面的例子展示了如何使用返回值：
```python
def square(n):
    """
    计算整数的平方值。
    :param n: int, 整数
    :return: int, 平方值
    """
    result = n ** 2
    return result

result = square(3) # 函数square的返回值为9
print(result) # 9
```
在上述例子中，函数`square()`接受一个整数作为输入，计算它的平方值并返回。调用函数`square()`并将结果赋给变量`result`之后，就可以在主程序中使用`result`变量接收函数的返回值。
