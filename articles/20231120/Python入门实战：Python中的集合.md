                 

# 1.背景介绍


## 概念
**集合（set）** 是一种无序的不重复元素集。它是一个抽象的数据类型，用来存储一个集合中的元素，这些元素可进行增、删、改查。在Python中，集合可以用 set() 函数或 {} 进行创建。
## 相关知识点
- **字典**：在计算机科学中，字典（dictionary）是一种哈希表（hash table）数据结构。它通过键值对的方式存储数据，键可以是任意不可变对象，而值则可以是任意对象。字典是一种内置的高阶映射类型，具有以下特性：
  - 查找速度快：字典中的每个值都可以通过其键被快速找到。
  - 使用方便：字典允许通过键来访问和修改值，而不需要知道值的索引位置。
  - 插入删除操作快：插入和删除操作可以在 O(1) 的时间复杂度内完成。
  - 支持动态大小调整：字典的大小可以根据需要自动调整。
- **集合**：集合（set）是一种无序的、不可重复的元素组成的集合。集合中的元素没有顺序，而且只能包含数字、字符串、元组等不可变对象。由于集合中的元素不可重复，因此集合中的所有元素之间都是独立无关的。集合可以表示元素间的关系，例如可以表示学生之间的同学关系、班级之间的学生关系等。另外，集合也经常用于求交集、并集、差集等运算。

# 2.核心概念与联系
## 集合
### 定义
集合是由一个或多个元素组成的无序的、不可重复的元素集。集合中的元素没有先后顺序，而且元素不能重复出现。
### 创建集合
创建集合的方法有两种：
- 方法1：使用 `set()` 函数创建一个空集合。示例如下：
```python
>>> my_set = set()
```
- 方法2：使用花括号 `{}` 来创建一个空集合或者填充一个已有的集合。示例如下：
```python
>>> empty_set = {} # 创建了一个空集合
>>> numbers = {1, 2, 3} # 创建了一个集合并添加了三个元素
```
### 添加元素
添加元素到集合的语法是将元素放置在左边的花括号中即可。示例如下：
```python
>>> fruits = {'apple', 'banana', 'orange'}
>>> fruits.add('grape') # 使用 add() 方法向 fruits 中添加元素 'grape'
>>> print(fruits) # 输出 fruits 中的所有元素
{'orange', 'banana', 'apple', 'grape'}
```
### 删除元素
删除元素可以从集合中直接调用 remove() 或 discard() 方法。示例如下：
```python
>>> fruit = {'apple', 'banana', 'orange'}
>>> fruit.remove('orange') # 从 fruit 中移除元素 'orange'
>>> fruit.discard('banana') # 等效于 remove() 方法，也是从 fruit 中移除元素 'banana'
>>> print(fruit) # 输出 fruit 中的所有元素
{'apple'}
```
### 修改元素
可以使用 update() 方法将另一个集合的所有元素合并到当前集合中。也可以直接修改指定元素的值。示例如下：
```python
>>> nums = {1, 2, 3, 4}
>>> nums.update({5}) # 将元素 5 添加到 nums 中
>>> nums |= {7} # 等价于 update() 方法，也是将元素 7 添加到 nums 中
>>> nums[1] = 9 # 替换nums集合中的第二个元素为9
>>> print(nums) # 输出 nums 中的所有元素
{1, 9, 2, 7, 3, 4, 5}
```
### 集合操作符
- `|` 表示“或”运算。示例如下：
```python
>>> a = {1, 2, 3}
>>> b = {3, 4, 5}
>>> c = a | b
>>> print(c) # {1, 2, 3, 4, 5}
```
- `&` 表示“与”运算。示例如下：
```python
>>> a = {1, 2, 3}
>>> b = {2, 3, 4}
>>> c = a & b
>>> print(c) # {2, 3}
```
- `-` 表示“减去”运算。示例如下：
```python
>>> a = {1, 2, 3}
>>> b = {2, 3, 4}
>>> c = a - b
>>> print(c) # {1}
```
- `<=` 表示“等于或小于等于”运算。示例如下：
```python
>>> a = {1, 2, 3}
>>> b = {2, 3, 4}
>>> print(a <= b) # False
```
