# 2.核心概念与联系
## Python中的条件判断语句
在Python中，条件判断语句通常有两种形式，即if-else语句和if-elif-else语句。它们都属于选择结构，也就是说，当满足某种条件时，执行对应的代码块。
### if语句
#### if-else语句
Python中的if语句的基本形式如下：

```python
if condition:
    # 如果条件成立，则执行该代码块
else:
    # 如果条件不成立，则执行该代码块
```
其中condition可以是任何表达式，如果该表达式的值为True，则执行第一组代码；否则，执行第二组代码。

举个例子，假设我们要根据年龄来决定是否给出免费小礼品，规则如下：

1. 年龄小于等于18岁，则给出一瓶可乐；
2. 年龄大于18岁，且小于等于40岁，则给出一包雪糕；
3. 年龄大于40岁，则不提供免费产品。

这时候可以使用if-else语句来表示：

```python
age = int(input("请输入您的年龄："))   # 获取用户输入的年龄

if age <= 18:    # 判断年龄是否满足要求
    print("恭喜您获得一瓶可乐！")
else:
    if age > 40:
        print("抱歉，暂时没有满足条件的产品。")
    else:
        print("恭喜您获得一包雪糕！")
```
以上代码首先获取用户输入的年龄并将其转换为整数类型。然后根据年龄进行判断，并分别输出相应的消息。

#### if-elif-else语句
如果条件判断语句存在多个分支，则可以使用if-elif-else语句来替代。

```python
if condition1:
    # 若condition1成立，则执行该代码块
elif condition2:
    # 若condition1不成立，但condition2成立，则执行该代码块
elif condition3:
    # 若前两个条件均不成立，但condition3成立，则执行该代码块
...
else:
    # 如果所有的条件都不成立，则执行该代码块
```

if-elif-else语句的每一个elif子句都会检查下一个条件是否成立。直到某个条件成立后，对应的代码块就会被执行。如果所有条件都不成立，则执行else子句的代码块。

#### 嵌套if语句
Python允许在if-elif-else语句内嵌套其他的if语句。这种情况下，应确保不要出现冲突的情况。

例如，以下代码实现了一个两层嵌套的if-else语句：

```python
score = input("请输入您的成绩：")
score = float(score)  # 将字符串转换为浮点型数字

if score >= 90:
    grade = "A"
else:
    if score >= 80:
        grade = "B"
    else:
        if score >= 70:
            grade = "C"
        else:
            if score >= 60:
                grade = "D"
            else:
                grade = "E"
                
print("您的等级是",grade,"!")
```

此外，还可以结合多重嵌套来实现更多的复杂的逻辑判断。

#### 小结
Python中的条件判断语句有三种形式，即if-else语句、if-elif-else语句和嵌套if语句。if-else语句最简单，if-elif-else语句可以实现多分支条件判断，而嵌套if语句可以实现更复杂的条件组合。
