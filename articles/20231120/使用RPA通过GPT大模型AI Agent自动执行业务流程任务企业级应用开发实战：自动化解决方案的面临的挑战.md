                 

# 1.背景介绍


## 概述
随着科技发展和经济的迅速发展，各种IT服务的体量越来越大，而且对于企业内部而言，采用手动的重复性工作耗费了大量的人力、物力和财力资源，导致了效率低下和生产力损失，信息系统的维护成本越来越高。因此，需要借助自动化的方法，提升企业的工作效率。然而，自动化存在许多复杂的问题，如自动化脚本编写难度较高，反复调试和调整的周期长等，使得它无法完全满足所有需求，也极大的增加了实现自动化的复杂度和风险。但另一方面，如果能够采用人工智能的方法来进行自动化，就能够克服自动化存在的问题，并大大降低成本。因此，企业需要寻找一种方法，能够用人工智能的方式自动完成某个具体的业务流程任务，并在不影响企业现有业务的情况下，提升其工作效率。
### 什么是RPA(Robotic Process Automation)？
RPA，即“机器人流程自动化”，是一种通过软件、硬件设备及网络技术将复杂的业务流程自动化的技术。一般来说，RPA由两大部分组成：规则引擎和自动化引擎。规则引擎用于定义业务流程，它主要包括基于正则表达式的文本匹配、规则匹配和逻辑推理等。自动化引擎则负责对符合规则的流程进行执行，包括执行流程、数据抽取和数据转换等。


### GPT-3——大模型AI语言模型
GPT-3是一个开源的，英文版的大型语言模型，它的训练数据集是超过十亿个网页和论文。模型结构类似GPT-2，但是有一些改进，例如更大容量、更深层次的transformer模型。GPT-3可以生成连续的文本，并且表现出很强的学习能力。
### 为何选择GPT-3作为我们的AI语言模型呢？
1. GPT-3已被证明可靠，准确度超过95%。

2. GPT-3在NLP领域占据着一个重要的地位，它经过了测试验证其准确性、可靠性和效率。GPT-3有着广泛的应用前景，从自动作诗到自动创新报告都处于前沿。

3. GPT-3的自然语言理解能力远超其他的单一模型，可以理解上下文语义关系、词汇意思变化、句法依存关系等。

4. GPT-3的设计目标之一是能够理解和处理任意长度、复杂度的输入，因此适合于自动完成业务流程任务。

5. GPT-3使用了一种新型的训练模式，可以生成连续的文本，而不需要像传统的语言模型一样将生成的单个词或短语连接起来，因此能够更好的满足用户的需求。
