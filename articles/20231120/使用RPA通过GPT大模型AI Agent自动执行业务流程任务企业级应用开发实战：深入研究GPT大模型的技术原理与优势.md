                 

# 1.背景介绍


企业级应用开发涉及到多个角色人员的协作和合作，业务流程也逐渐被智能化需求所驱动，基于业务需求，GPT模型（Generative Pre-trained Transformer）可以对流程进行预训练，并能够自动生成可执行的业务场景、任务的指令文本，因此运用GPT模型开发企业级应用成为一个重要的方向。基于RPA（Robotic Process Automation）的自动化操作，可以提高工作效率，降低人力资源开销，还可以避免重复劳动，提升工作质量。
在本次分享会中，我将深入研究GPT大模型（Language Models are Unsupervised Multitask Learners）的技术原理与优势，并结合实际案例，带领大家实践利用GPT模型构建企业级应用，解决业务流程的自动化问题。主要内容如下：
# GPT的历史、结构和特点
# GPT-2的介绍、架构和改进之处
# GPT-3的简介、历史和架构
# 案例研究：如何用GPT构建HR助手，提升员工培训效率？
# 案例研究：如何用GPT优化供应商管理系统？
# 用GPT解决业务流程自动化的关键：结构化数据、实体识别、任务抽取、语义理解
