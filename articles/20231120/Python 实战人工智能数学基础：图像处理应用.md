# 2.核心概念与联系
## 2.1 什么是图像处理？为什么要进行图像处理？
图像处理（Image Processing）是指对图像或者视频进行修改、处理、分析、识别的过程。它可用于分析、理解、创建、呈现和编辑各种各样的图像，包括摄影、科技、生物医学、军事、航空等领域。计算机视觉、模式识别、人工智能、机器学习、数字孪生技术、图形计算技术等领域均依赖于图像处理技术。它的主要目的就是提取、处理、记录或分析图像信息，使图像具备其所应有的功能属性，达到图像数据的高效利用、准确表达和方便交流的效果。

图像处理技术可以解决诸如：对象检测、图像增强、图像检索、图像修复、图像存储等问题。同时，图像处理技术也与其他计算机技术紧密相连，例如信号处理、信息论、随机过程等，能够促进这几种技术的研发。

## 2.2 OpenCV 的安装与使用
OpenCV (Open Source Computer Vision Library) 是基于 BSD 协议的开源计算机视觉库，它由 Intel、Apple、National Instruments 和 NVIDIA 提供支持。OpenCV 支持许多种编程语言，包括 C++、Python、C、Java、MATLAB等，OpenCV 为开发人员提供了图像处理、计算机视觉、机器学习、信号处理、生物特征识别等方面的工具包。


### 安装
你可以从官方网站上下载安装最新版的 OpenCV ，或者直接使用以下命令安装：
```
pip install opencv-python
```
安装后可以使用 python 中的 cv2 模块来调用 OpenCV 。

### 使用方法
你可以先尝试运行以下 Python 脚本，了解如何读取、显示图片：

```
import cv2

cv2.imshow('image', img) # 窗口名'image'，显示图片
cv2.waitKey(0) # 等待按键输入
cv2.destroyAllWindows() # 销毁所有窗口
```

此外，还有很多图像处理相关的方法可以尝试，比如色彩空间转换、图像二值化与阈值化、图像形态学操作、轮廓发现与轮廓绘制、直方图统计与直方图均衡化、模板匹配等。这些方法都可以在 OpenCV 中找到对应的 API 接口。
