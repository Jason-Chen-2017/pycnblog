# Rust概述
Rust 是一种 systems programming language ，它提供了内存安全的保证，编译时检查异常、并发性、线程安全等等，可以用来编写性能敏感型的应用。它拥有两个主要特点：安全(safe)和惯用(familiar)。在没有运行时系统的情况下，编译器可以验证代码的正确性，使得程序员不必担心内存管理、数据竞争等复杂的问题。它还支持函数式编程、基于特征的接口、代数数据类型、模式匹配等功能，这些特性可以帮助程序员更加方便地编码和组织代码。Rust 语言由 Mozilla 基金会(Mozilla Foundation) 开发，Rust 项目是开源的，并遵循 Apache2.0 协议。

以下是Rust语言的几个主要特点：

1.静态类型安全：Rust 有类型推导和强制转换，使得编译器可以检测出很多类型错误，比如 null 引用、数组越界、内存泄漏等等。Rust 使用生命周期（lifetime）来消除一些错误，避免资源泄露。

2.运行速度快：Rust 的编译器能够在编译期间进行大量的优化，比 C++ 或 Java 编译器的优化更加智能。而且 Rust 有自动并行化的功能，可以在 CPU 上并行处理任务，使得程序的运行时间比同类语言要快很多。

3.并发性：Rust 支持通过消息传递并发的方式来解决多线程问题，使得 Rust 程序具有较高的并发性。消息传递模式允许程序员创建独立的小线程或协程，并且它们之间可以发送消息，而不需要共享内存。Rust 的标准库提供了通用的并发机制，如同步原语、互斥锁、条件变量等，可以满足日常需要。

4.扩展性好：Rust 的 trait 机制可以实现面向对象编程中的抽象类和接口，它还可以使用闭包和迭代器来简化编程。Rust 的生态系统涵盖了非常丰富的库，可以满足日常需求。

5.零开销抽象：Rust 提供了最小化运行时开销的内存管理机制，它采用 borrow checker 和借贷原则来消除一些内存上的错误。它使用枚举、指针、trait、闭包、闭包转移、生命周期等方式，来进一步减少运行时的开销。

总之，Rust 语言提供了高效且安全的编程环境，是一个值得深入学习的语言。
