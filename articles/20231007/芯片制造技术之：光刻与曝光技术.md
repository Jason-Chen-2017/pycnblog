# 1.背景介绍


光刻与曝光（Light Focusing and Exposure Technique）是制作集成电路板（Integrated Circuit Boards）、数字照相机、激光扫描仪等高精密模拟或数字设备所需的重要组成部件。光刻与曝光技术是工艺制图技术、工业装备工艺技术、计算机及自动控制技术、纺织科技等领域均需掌握的重要知识。光刻与曝光技术影响了所有高精密制造领域的产品设计和生产。在这个领域中，光刻与曝光技术主要有以下三个方面：

1.光栅技术：光栅技术是用非常小的物料将晶体管进行简单划分，形成铜、金属或玻璃瓶状的晶圆。这样做可以有效地降低了晶体管的材料浪费率，减少了制造成本。同时，采用光栅技术可以使得设备的尺寸缩小，适应更大的尺寸产品的制造。

2.结构层布置技术：结构层布置技术包括表面处理和材料选择两种方法。表面处理技术通过改变晶圆的表面结构，如涂覆、打磨、钻孔等手段，对晶圆结构进行优化调整，从而提升晶圆的品质和性能。材料选择技术则通过选择符合特殊应用要求的材料进行晶圆的构建，比如氮化镀锌或其他化合物的合金，可以实现高强度的导电性和耐受性。

3.照明和光源技术：照明和光源技术用于在结构层上进行光的吸收、反射、透射等功能。其中，发光二极管（LEDs）或激光笔的制作需要光源的辅助才能达到很好的显示效果。因此，光源技术是制作高精密光学产品的关键技术之一。

# 2.核心概念与联系
## 2.1 光刻（Lithography）
光刻技术指利用微粒粘结的方法将各种类型的微晶状物层层堆叠、溶化，经过光照的作用后，使晶体由粗粉末变成较细的粒子状晶体。由于每种类型微晶体的特性不同，光刻技术可分为显影光刻与隐影光刻，前者利用显微镜直接刻蚀晶体，后者利用反向投影和反射等方式将光微晶体置于微元晶体之间，再将光微晶体摧毁，最后形成粒子状晶体。

## 2.2 曝光（Exposure）
曝光又称为辐射度量或光度计，是指当光线投射在物体上的过程中，所产生的光的总能量、大小分布、方向分布等特征。它既可以用来测定显影像素的亮度，也可以用来评价整个胶片图像的色彩饱和度、动态范围及质感。由于图像的复杂性和多样性，曝光技术是制造各类高精密光学产品的关键技术之一。

## 2.3 光刻与曝光技术的联系与区别
光刻与曝光技术有很多类似之处，但是也存在一些不同之处。首先，光刻与曝光技术都是依靠晶体管和光电子技术实现的。其次，两者的工作流程差不多，都需要进行堆积、溶化、光照、撞击等过程。不过，光刻与曝光技术的具体方法却有所不同。光刻技术主要基于显影生长技术，将微晶粒从层层堆叠、溶化的显影层中清除出去，还原出晶体管的真空状态。曝光技术则是依据光的反射特性来测量晶体的厚度、粒度和颜色。

## 2.4 光刻与曝光技术的分类
目前，关于光刻与曝光技术的分类标准主要有以下几种：

1.静态光刻与动态光刻：静态光刻即在无光照情况下刻蚀晶体，而动态光刻则是利用反射、透射、吸收等现象进行刻蚀。在设备制造领域，静态光刻一般采用金属刚性模具的挤压刻蚀，而动态光刻则可采用微粒粘结的形式，该技术可以在低温下实现精确刻蚀。

2.全切片刻蚀与部分切片刻蚀：全切片刻蚀是指在整个晶体上均匀地进行刻蚀；部分切片刻蚀是指仅在需要刻蚀的区域进行刻蚀。全切片刻蚀是最简单但代价昂贵的一种光刻技术，而部分切片刻蚀技术则可以节省成本。部分切片刻蚀技术又可以分为微步切片与毫米刻蚀技术。微步切片是指用小型的颗粒或像素（微步）的方式，将晶体切片，然后在晶片内部继续进行刻蚀，最终形成整块晶体。毫米刻蚀技术是指将细小的物料（如红外线探测器中的像素）粘附在晶体上，然后利用光照的作用抹平这些微晶粒，最后还原出晶体的真空状态。

3.传统光刻与超新星光刻：传统光刻技术是指早期的光刻技术，主要使用光电子管和静电容的方式，将晶体层层堆叠，再将其挤压出晶体管。而超新星光刻技术是指利用微型结构的光刻技术，以放大器的形式实现晶体的立体化，并直接刻蚀出晶体管。

4.显影光刻与隐影光刻：显影光刻技术是指晶体被激光束反射、束斑等原因刻蚀出来的晶体；而隐影光刻技术则是在晶体表面，利用反射、透射、吸收等现象进行刻蚀，从而还原出真空状态。显影光刻一般用于造数字场成像相关设备，如数字单反相机、图像打印机等；隐影光刻技术则广泛应用于工业制图、印刷包装、纺织品加工、机器人、电子武器等领域。
