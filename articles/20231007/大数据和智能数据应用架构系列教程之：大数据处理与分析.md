# 2.核心概念与联系
## 2.1 大数据定义及其发展历程

### （1）大数据概述
大数据（英语：big data）指的是存储海量数据的现代技术。在过去的几十年里，随着科技的进步、生产的增加、社会的变迁，各种各样的数据源不断涌入我们的生活，比如网站点击流、社交媒体数据、网页浏览记录、交易数据、地理位置信息等。这些数据所产生的规模已超出了传统的单机数据库可以处理的范围。因此，如今已成为数据经济和数据中心的一个重要组成部分。

2001年，谷歌提出Google File System（GFS），用于存储大数据文件。随后，Google又开发了MapReduce框架，用于并行分布式处理海量数据，其并行度高达数千个节点。之后，雅虎推出了Apache Hadoop项目，基于HDFS、MapReduce等技术，形成了大数据生态圈。

2010年前后，国内开始出现了大数据热潮，包括阿里巴巴、百度、腾讯等互联网公司都在大力探索大数据发展的新阶段。其中，阿里巴巴于2012年提出“阿里云计算平台”，首期支持离线数据分析、批处理和机器学习等场景；百度则在2013年发布了“百度搜索引擎”；腾讯也推出了“腾讯云计算平台”。这些平台均提供了大数据计算、存储、分析等服务，帮助企业实现业务的快速发展，降低成本，改善用户体验。

### （2）大数据特点
#### 数据量大
据统计，全球每天产生的数据超过100ZB，主要来自移动互联网、物联网、传感器网络、车联网等各类传感设备。数据量的爆炸性增长已促使各类研究机构加强对数据的收集、存储、管理等方面的综合应用，形成了一套完整的大数据生命周期。

#### 数据种类多样
目前，大数据面临的主要挑战是数据量过大、数据复杂度高、数据类型多样、数据异质性高、数据源广泛。而这也是大数据的优势所在，它可以提供海量的信息，并把复杂的数据聚合为有价值的洞察，从而引导企业的决策更准确、更有针对性、更有效果。

#### 数据时代
随着大数据的发展，不同部门之间的合作日益紧密，各类信息渠道以及源头越来越多，难以统一获取。这就要求进行数据的整合、汇总，才能做到数据的精准和可靠。另外，由于数据质量保障的需要，不同的业务之间还要存在互补性，否则就无法发挥作用。因此，如何结合需求、技术、管理等多方面因素，形成一个完整的大数据生态系统，让数据产生价值，才是大数据的关键。

#### 智能运维
为了应对日益复杂的大数据环境，许多公司都开始采用机器学习的方法，在海量数据中找到模式、发现规则、预测结果。例如，人脸识别系统就可以利用深度学习技术，对摄像头拍摄到的照片进行分析，判断是否是特定个人，甚至给出相应的反馈。由于其低廉的成本、大数据量、多种形式的数据输入，智能运维正在成为众多企业的新宠。

## 2.2 大数据技术发展历程
### （1）数据采集方式
早期，大数据采集主要由互联网企业主动提供原始数据，如Web日志、点击流、留言板等；后来互联网企业逐渐转向企业内部的业务系统、交易平台和第三方数据源，如Erp、CRM、POS等。在此过程中，会有一些数据质量问题需要处理，比如数据错误、缺失、偏差、不一致等。另外，随着数据源越来越多，如何有效地进行数据采集、处理、存储、分析等操作也成为当前和今后的重点难点。

### （2）数据存储方式
根据数据的大小、生命周期、访问频率、可用性、数据安全等特性，大数据存储分为结构化存储、半结构化存储、非关系型数据库、消息队列和对象存储等。其中，结构化存储（RDBMS）、半结构化存储（NoSQL）、非关系型数据库（NoSQL）、消息队列（MQ）、对象存储（OSS）具有不同的数据结构、读写性能、扩展性、易维护性等特点。选择合适的存储方式，对提高大数据的处理和分析能力至关重要。

### （3）数据计算方式
大数据计算通常采用分布式计算框架，包括Hadoop MapReduce、Spark、Flink等。这些框架提供高吞吐量和容错性，可支持TB级以上数据的快速处理，并允许用户自定义计算逻辑。除此之外，还有基于图算法的分布式计算框架，如Pregel、Giraph等。这些框架可以进行图、社交网络、文本数据等复杂数据处理。

### （4）数据分析方式
大数据分析的方式主要有批处理分析、流式分析、机器学习和人工智能等。批处理分析适合静态、简单的数据，如广告投放、销售预测等；流式分析适合动态、连续的数据，如股票交易、实时推荐等；机器学习和人工智能可自动发现数据中的模式和特征，对数据进行分类、预测和建议等。

## 2.3 大数据应用案例
2016年初，英伟达宣布推出Jetson Xavier，为在边缘设备上运行机器学习和深度神经网络的消费者开发者提供了便利。同时，英伟达还将Xavier纳入NVIDIA深度学习处理单元（DPU）的家族，打通了高端边缘计算市场，实现了异构计算平台之间的互联互通。对于企业来说，基于海量数据的深度学习技术有助于提高工作效率，节省时间成本。例如，基于人脸识别的数据挖掘可以帮助企业更快捷地发现人才，减少重复劳动，提高生产效率。另一方面，基于图像检测的行为监控也可以用于实时跟踪活动轨迹，提醒员工注意隐私、健康状况和安全警示。

华为在智慧城市、智慧商业、数字孪生四大领域均取得突破性进展，其智能计算平台CloudLink、CloudTOPS等已经成为世界级的大数据应用案例。智能计算平台融合了神经网络、模式识别、图像处理、语音识别、计算存储等众多AI功能，可应用在金融、制造、公共事业、环保、医疗、运输、物流、电子商务等领域。目前，华为已通过“揭秘第七代华为智能计算平台”等一系列大胆尝试，积极参与AI领域的科技革命。

阿里巴巴在商家数据采集、计算分析和推荐等领域，也取得了重大进展。阿里巴巴开放平台阿普速度搜集、关联、分析和展示各类商家数据，可大幅提升商家数据价值。此外，阿里巴巴围绕海量商品数据建立了“阿里妈妈”数据共享中心，通过大数据技术赋能商家销售和营销。这种以“妈妈”为代表的大数据社区，使得多个商家之间共享同一份数据，为商家间的竞争培养了牢固的基础。

搜狗微信今年推出了微信搜索“背后”功能，让用户通过检索微信公众号、小程序、文章、话题等进行深度分析，了解微信用户群体画像、兴趣点、流量来源、行为习惯等。类似于微信搜索背后功能的其它应用还有QQ空间、微博舆情分析等。这项功能对于腾讯、搜狗、阿里巴巴、百度、京东、网易等互联网公司来说都是重要的数据基础设施。
