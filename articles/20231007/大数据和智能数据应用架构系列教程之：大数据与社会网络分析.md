# 1.背景介绍


## 什么是大数据和智能数据？
大数据和智能数据是在互联网时代产生的新的数据类型。它通常由海量数据组成，既包括结构化、非结构化甚至图像数据的形式。由于数据量太大，传统的数据库处理不了，于是人们开始采用一些新的处理方法来进行数据分析。所谓大数据和智能数据，就是指以高计算能力、海量存储容量、超级宽带传输速度为特征的一类数据。

而“大”数据和“智能”数据，则是“数据”这一术语的两个方面。首先，“大”数据指的是单个数据对象可能超过一个亿字节，其次，“智能”数据意味着可以对数据做出更复杂的判断。以金融行业为例，如果我们能够通过大数据和智能数据对客户行为和交易模式进行分析，就可以对贷款客户提供个性化的产品，从而提升服务质量并降低风险。同时，企业也可以利用大数据和智能数据提升其决策效率，增强竞争力，并改善管理决策。因此，“大数据”和“智能数据”既是一种新的技术也是一个蓬勃发展的行业。

## 为什么要进行社会网络分析（SNA）？
随着互联网网站、社交媒体、手机应用程序等各种形式的数据信息越来越多，数据获取的难度也越来越大。如何从海量数据中发现有价值的知识、模式和关系，成为当今社会生活中的重要问题。而社会网络分析（Social Network Analysis，简称SNA），便是目前最流行的数据挖掘方法之一。

SNA利用网络结构的特性和数据之间的关联性，通过分析关系和联系，发现人群的兴趣爱好、习惯和态度，帮助企业设计营销策略、优化营销方案、改善产品质量、减少风险，从而实现商业目标。因此，SNA已成为各大公司进行各项业务决策的重要工具。

## SNA的定义、分类及特点
### 定义
社会网络分析（Social network analysis，SNA）是研究人际关系、组织行为、地理分布等多种人口学和社会学因素，探索人、群体、组织的影响和行为规律的领域。它主要用于理解人群内部的群体关系、社会关系、地理位置关系、人文价值观念的相互作用和协调变化过程。

### 分类
SNA可以分为以下三类：

1. 面向个人的网络分析：对人的身心活动、言行交流、社会行为等方面进行分析；
2. 面向群体的网络分析：对群体规模、性格特点、团队合作等方面进行分析；
3. 面向组织的网络分析：对组织的业务、资源、人才培养、结构等方面进行分析。

### SNA的特点

1. 数据量大：SNA需要处理庞大的网络数据才能得到有效的结果。因此，SNA涉及的数据量一般都很大，但又不能忽略重要性质和数量级上的差异。
2. 模型复杂：SNA涉及到的模型、算法多且繁多。运用各种模型和算法，才能得到科学、准确的结果。
3. 时空不变性：很多SNA模型的假设都是基于静态网络状态或时间点的，因此，分析结果的有效性可能会受到局部环境、时期影响的影响。
4. 多样性：不同领域的SNA模型存在差异性，因此，在同一个行业内，同一模型的效果可能不同。
