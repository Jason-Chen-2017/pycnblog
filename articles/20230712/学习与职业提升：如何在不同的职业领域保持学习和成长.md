### 4.3. 核心代码实现

```
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

for i in range(10):
    print(fibonacci(i))
```

### 4.4. 代码讲解说明

在实现斐波那契数列计算器的过程中，我们首先需要定义`fibonacci`函数，用于计算斐波那契数列的前n项。在函数体中，我们使用if语句判断n是否小于等于1，如果是，则直接返回n。否则，我们递归调用`fibonacci(n-1)`和`fibonacci(n-2)`来计算斐波那契数列的前两项，并将它们相加得到结果。最后，我们在外层循环中调用`fibonacci(n)`来计算斐波那契数列的前n项，并输出结果。
