2. 技术原理及概念

2.1. 基本概念解释

边缘计算是一种分布式计算架构，旨在将计算和数据存储任务更接近数据源，从而减少数据传输延迟和网络拥塞。在安防摄像头实时数据处理和传输中，边缘计算可以有效降低视频数据从摄像头到云端处理的时延。

```
视频预处理：
1. 对摄像头拍摄的实时视频进行增强处理，包括亮度调整、对比度增强、色彩平衡等；
2. 对摄像头拍摄的实时视频进行降噪处理；
3. 将预处理后的视频数据进行压缩，减少数据传输的带宽；
4. 将经过预处理和压缩的视频数据通过 HTTP 协议传输到云端服务器。
```
