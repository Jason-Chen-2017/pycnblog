# 2. 技术原理及概念

## 2.1. 基本概念解释

MongoDB 是一款文档数据库，它支持 BSON（Binary JSON）格式的数据存储。MongoDB 中的数据分为文档和字段两种类型。

* 文档（document）：MongoDB 中的文档是以 BSON 格式存储的键值对集合，每个文档都具有独立的元数据。
* 字段（field）：MongoDB 中的每个文档都有一个或多个字段，字段具有名称和类型，用于描述文档的各个属性。

## 2.2. 技术原理介绍: 算法原理，具体操作步骤，数学公式，代码实例和解释说明

MongoDB 的核心数据结构是 BSON 对象，它支持灵活的查询、排序、聚合操作。

### 2.2.1 BSON 对象的结构

BSON 对象由键值对组成，其中键和值都可以是对象或数组。键可以是字符串、数字、特殊字符或二进制数据，值可以是字符串、数字、特殊字符或二进制数据。
