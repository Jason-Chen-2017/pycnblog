# 2. 技术原理及概念

## 2.1. 基本概念解释

数据仪表盘是一种数据可视化工具，通过图表、图例等方式将数据呈现给用户。数据仪表盘的用户体验设计原则主要包括以下几点：

* 可用性：数据仪表盘应易于使用，用户可以快速找到自己需要的信息，方便地进行数据探索和分析。
* 可读性：数据仪表盘中的数据应该清晰易懂，使用户能够理解数据背后的故事。
* 可理解性：数据仪表盘中的数据应该以易于理解的方式呈现，不要过于复杂或难以理解。
* 交互性：数据仪表盘应该提供丰富的交互功能，以便用户能够自由地探索数据。
