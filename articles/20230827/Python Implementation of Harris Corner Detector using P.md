# 1.简介
  

## 1.1 项目背景
图像处理中，对图像特征点检测一直是一个重要且基础的研究领域。一种主要的用于图像边缘检测的方法叫做Harris角点检测法(Harris Corner Detection)。该方法通过计算图像每一个像素邻域内的像素强度和梯度的协方差矩阵来描述图像区域的灰度结构以及边缘方向。在极值点处的响应值较高，而边缘处则具有较低的响应值。因此，可以利用这些信息来检测出图像中的角点，从而实现图像的边缘检测。本文基于PSO算法的研究,实现了一个简单的、有效的、快速的、并行化的Harris角点检测器。

## 1.2 概念及术语说明
### 1.2.1 Harris角点检测
Harris角点检测法是一种基于边缘和强度的角点检测方法，它通过计算图像局部邻域内的强度梯度共讹夫矩(Hessian Matrix)的特征值和特征向量，来确定角点位置和尺度。其特点是通过比较特征值对应的特征向量的导数的二阶矩来判断角点的位置和尺度。具体来说，对于每一个像素点P，首先通过微分算子求取到该像素点周围的梯度D(x,y)，然后用D(x+dx,y+dy)-D(x-dx,y-dy)/4[+D(x,y+dy)+D(x,y-dy)-D(x+dx,y)+D(x-dx,y)]作为邻域内的强度梯度共讹夫矩。根据高斯假设，对于图像空间中任意一点，其局部邻域内的强度梯度共讹夫矩应当是对称矩阵。通过将这个对称矩阵变换为特征值与特征向量形式，可以得到角点的位置及尺度信息。其中，特征值最大的地方对应于角点的位置；特征值的大小反映了角点的尺度；特征向量方向指向其方向变化程度最明显的方向。通过比较不同角点的特征值大小，可以检测出不同尺度和方向的角点。

### 1.2.2 角点分类
一般地，角点分类可分为两类：主要角点（prominent corner）和次要角点（non-prominent corner）。主要角点指的是能够明确表征边界和轮廓的角点；而次要角点则不一定能够立刻确定边界或轮廓，但却可能对某些领域建模、分类任务等有着重要意义。主要角点又可细分为特征点和非特征点两种类型：特征点指的是能够确定图像的某种特定结构特性，如斑点、牙齿等，而非特征点则可能只是一些额外的噪声点。

### 1.2.3 角点匹配
角点匹配是指利用角点检测的结果进行图像配准、图像拼接、图像重建等应用。在角点匹配过程中，检测到的主要角点会被用来估计相机姿态、相似物体的形状和外观，并与参考模型或其他图像的角点进行匹配，从而获得更加精确的位姿关系、三维形状和外观信息。

### 1.2.4 角点描述符
角点描述符由一个或多个描述子组成，用来描述每个角点的局部特征。常用的角点描述符有基于特征的角点描述符和基于分割的角点描述符。基于特征的角点描述符包括SIFT、SURF、ORB、BRIEF等；基于分割的角点描述符包括Haralick算子、Gabor函数等。

### 1.2.5 运动边界提议器(Motion Bound Proposers, MBPs)
MBPs是一种用来估计运动场边界的估计方法，其根据运动目标的速度和其他图像特征，来生成有助于定位运动目标的图像区域。MBPs可分为全局和局部两个类别。全局MBPs通常认为目标是静止的，并且只需要考虑整幅图像的几何结构；而局部MBPs则将相机视觉帧序列看作一个流动的动态系统，其会考虑连续帧之间的相互影响。常用的全局MBPs包括Horn-Schunck方法、Baker方法和Wang-Suen方法等；常用的局部MBPs包括HMM、USAC等。

### 1.2.6 稀疏性
稀疏性是指计算机视觉和图像处理领域中，描述对象内部结构所需的最小的采样密度。直观地说，即使是在数字图像上，也存在着一定的采样密度。例如，在经典的边缘检测中，通常采用一种小邻域的预测方法来获得足够多的样本。在这样的条件下，若图像的缩放比例较大时，仍然存在较大的不一致性。为了缓解这一不一致性，可以在图像上选择适当的模板窗口或者阈值化值来减少图像的采样密度，从而增加稀疏性。
