# 1.简介
  

Seaborn是一个基于Python的数据可视化库，它主要用于解决数据可视化领域的一些常见任务，例如：绘制概率分布图、关系图、时间序列图等。它的目标是成为一个功能全面的高级统计可视化库，提供简单易用的API接口，并提升其易用性和交互性。相比Matplotlib，Seaborn在API设计上更加简洁统一，还带来了一些独有的便利特性，例如内置数据集。同时，它还提供了更多的可视化主题和样式选项，使得生成复杂的多种图形变得非常容易。

# 3.基本概念及术语说明
首先，我们要明确几个概念，如图所示：

- 数据集（Dataset）：一般指用于分析的数据。通常包括两类信息：（1）数据本身（2）数据的标签/变量/属性。如散点图、线图中需要的两个变量之间的关系。
- 柱状图（Bar Charts）：用于呈现一组分类变量中的数值分布情况。可以理解为一系列柱体的高度代表数据的大小。
- 饼图（Pie Charts）：也叫花环图，用来呈现不同分类变量或数据集的比例或占比情况。
- 分面绘图（Faceted Plotting）：一种在多个维度上绘制同一数据的方法。可以将不同条件下相同变量的分布情况绘制在同一张图表上。
- 小提琴图（Violin Plot）：类似于箱型图，显示各个群组的分散情况。不同之处在于小提琴图展现了数据分布的概率密度，而不是直接用矩形条状表示数值大小。
- 计数图（Count Plot）：把具有相同分类的观测数量绘制成条形图。可以显示分类变量的频率分布。
- 热力图（Heatmap）：又称温度图，可呈现矩阵或表格型数据的全局分布。
- 散点图（Scatterplot）：用于呈现变量间关系的图表，每个点表示一个观测值。
- 棒图（Bar Chart）：用于显示单变量的值的分布情况，与柱状图比较类似。
- 折线图（Line Chart）：用于显示时间序列数据，一般用于比较数据的变化趋势。
- KDE图（Kernel Density Estimation plot）：展示变量的概率密度函数，同时提供了估计未知数据的可能性。
- 箱型图（Box Plot）：又称盒须图或盒式图，通过五个数值(最低值，下四分位数，中位数，上四分位数，最高值)描述一组数据。
- 核密度估计（Kernel density estimation）：一种非参数方法，能够估计概率密度函数，是一种广泛使用的统计分析方法。
- 描述统计学（Descriptive Statistics）：对数据进行汇总、分析和描述，得到数据整体上的特征的过程。
- 数据预处理（Data Preprocessing）：对数据进行清洗、整理、转换、过滤等操作，使其符合分析需求，提高分析效率的过程。
- 数据探索（Exploratory Data Analysis）：对数据进行初步的分析，包括数据检验、数据可视化、数据预处理等操作，帮助用户了解数据结构、特征分布、缺失值、异常值等信息，从而对数据产生兴趣和洞察力。
- 可视化（Visualization）：通过图形符号、颜色、大小、位置等元素对数据进行图形化表示，以直观的方式呈现出来。
# 4.相关概念
## 4.1 数据可视化基础知识
1. 一维数据：一维数据就是只有一个变量的统计数据，如一年生的人数、每月销售额、飞机起飞次数等。
2. 二维数据：二维数据指的是具有两个变量的数据，如：购物者与顾客之间的关系（购买频率、购买金额）、两部电影之间的评价（评分、评价长度）。
3. 聚类分析：数据可以按照某种规律（如距离、中心点、方差）将数据划分为若干组，每一组含有相似的特征。
4. 数据的类型：
    * 标称型（Categorical）：就是无序的、离散的、称量单位的。如性别、职业、颜色。
    * 标量型（Scalar）：是一个实数值，表示一个定量的数值。如身高、体重、价格、销量等。
    * 序数型（Ordinal）：序数型数据是指顺序性很强的数据。如学历、教育水平、婚姻状态。
    * 连续型（Continuous）：连续型数据是可以按照比例大小排序的数据。如年龄、收入、房价。
5. 数据分布：数据分布是数据的特征，指数据的各种值的范围、上下限、均值和中位数。
    * 频率分布（Frequency distribution）：指各个分类或离散数据值的频次。
    * 频率密度分布（Probability density function, PDF）：概率密度函数描述了随机变量取值范围内某个特定值发生的概率。
    * 累积分布函数（Cumulative Distribution Function, CDF）：给定一个随机变量x，CDF(x)表示小于等于x的元素的个数占所有元素的个数的比例。
6. 标准化：标准化是指将数据映射到[0,1]区间或者[-1,1]区间。如将销售额的均值为0，标准差为1。
## 4.2 Matplotlib
Matplotlib是一个基于Python的2D图表绘制库，提供了大量的用于创建二维图表的函数。Matplotlib可用于绘制线形图、散点图、直方图、三元图等各种图表。Matplotlib的接口简洁，文档齐全，是数据可视化的重要工具。
## 4.3 Pandas
Pandas是一个开源的Python库，提供了数据读写、数据筛选、数据合并、数据透视表等数据分析功能。Pandas可以让用户快速便捷地进行数据处理，支持大量文件格式，常用于金融、科学、统计、工程领域。
