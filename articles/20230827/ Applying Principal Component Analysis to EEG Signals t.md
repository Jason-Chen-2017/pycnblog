# 1.简介
  

在进行脑电信号处理时，检测、分类和分析由无明显特征的短暂突发性事件或周期性结构（如长期记忆痕迹）组成的数据集是一个重要的任务。特别是由于心脏病、神经疾病以及各种其他类型的精神疾病造成的运动障碍导致的长时间意志丧失或者一些大脑功能受损导致的短时间意志缺陷，都会造成脑电数据的不稳定性，使得特征提取、分类和检测变得困难甚至无法进行。而基于主成分分析（PCA）的方法可以有效地对脑电数据进行特征化并发现潜藏于其中的模式和相关性。本文将探讨如何利用PCA方法识别脑电数据的眼部闭合和复苏过程，并对此展开深入分析和实践。

# 2.概念术语及定义
- EEG（Electroencephalogram，脑电图）：一个测量脑电活动强度的实时波形记录。它由两种不同频率成分组成——一种代表过去的神经活动，另一种代表当前正在进行的神经活动。通常情况下，它们被分成512-1024Hz范围内的8个子通道，用于检测各类脑电活动，包括：
   - 前后区分（前额皱眉、瞌睡、眼球活动、注意力集中等）
   - 侧向区分（左右前后上下移动、运动感知、注意力外溢等）
   - 欧姆法单极细胞刺激（运动节奏、运动方向、大脑暂留、注意力漂移等）
   - 情绪和情感（惊恐、恐慌、焦虑、恢复、平静等）
   - 声音（前唇、下巴、咽喉、歌唱、嗓音变化）
   - 身体（皮肤震颤、头部转动、手臂活动等）
   - 驯化助剂（醒来时发出噪音、食物味道、饮用水味道等）
- Blink detection：眼部闭合现象。简单来说，就是人类视网膜与视锥之间的通信出现延迟，导致双眼快速关闭和打开，这种现象就称为“眼部闭合”。眼部闭合常常伴随着一系列的后果，如意识丧失、抑郁、失眠、昏厥等。它的主要表现形式是：一段时间内，双眼没有移动，屏幕上的图片也没有刷新，甚至连视线也不能跟上实时的人物。它的主要症状如：疲劳、头晕、失眠、混乱、易怒、呕吐、感到胸闷、注意力不集中、情绪低落。
- Remission：又称“眼部复苏”，指的是患者恢复意识后的一种心理状态，即具有正常视力，注意力清晰，精神饱满，并且保持活动能力良好，同时也能适应环境条件变化。
- PCA：主成分分析是一种基于统计手段对多变量数据集进行降维、分析和解释的方法。该方法通过找出数据的主成分，来表示这些变量之间最大可能的共同变化，从而简化数据分析工作。PCA通过线性变换的方式实现这一目标，得到的新变量与原始变量之间的关系可以用方差贡献率（Variance Explained Ratio）衡量。
- ICA（Independent Component Analysis）：独立成分分析是一种机器学习方法，它可以用来发现混杂在一起的数据中的隐藏模式，并将其分离出来。它可以理解为一种无监督的降维方法，目的是为了找到能够解释尽可能多的观察数据的变量，而不是试图从每个观察数据中推断出变量。
- Fp1-F7：四个典型的脑电波段，分别对应前额前茎，前额上皮，前额中叶，前额下叶。它们所占的频率分别为90hz、110hz、130hz和150hz。
