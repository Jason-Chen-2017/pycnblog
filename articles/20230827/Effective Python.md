# 1.简介
  

## 1.1 引言
《Effective Python》是一本关于Python编程语言的经典著作，是Python编程高级开发者必备的参考书籍之一。该书不仅涵盖了Python最常用的语法和特性，还提供了大量实用案例，帮助读者了解、掌握、运用Python进行各种实际应用开发。对于需要学习、掌握Python编程技巧的初级到中级程序员来说，《Effective Python》是一个值得推荐的资源。
## 1.2 作者简介
《Effective Python》的作者是Python之父Guido van Rossum，他也是一位非常优秀的开源项目及创业公司的创始人。在计算机界享有盛誉的是Guido先生，其影响力在全球范围内也很显著，有着卓越的思想和能力。另外，《Effective Python》的译者林海峰（姚期智）更是Python界的一位传奇人物，她也是Python语言的知名中文译者，出版了多部Python相关的图书和文章，包括《Python Cookbook》、《Python编程从入门到实践》等。

Guido是Python编程语言的创造者，他早年在荷兰大学学习数学，后来毕业进入芬兰赫尔辛基大学攻读数学、信息和计算机科学学位，并在那里获得了电气工程学士学位。然而，他对计算机编程的热情远没有停止，在90年代末创建了Python编程语言，一段时间后他离开了这个行业去创建自己的金融服务公司，这是他成为程序员的契机。

在Python编程语言的开发过程中，Guido曾经参加过PyCon会议，被称为“Benevolent Dictator for Life”，一直负责Python编程语言的发展方向，并且带领团队实现了许多大胆的尝试，但由于各种原因，Guido最终难逃历史的风雨。

回到《Effective Python》一书，Guido最初只是想给读者提供一个学习Python编程的指导方针，希望通过编写易于阅读的代码、清晰的注释和适当的命名规范，让他们能够较快地上手。如今，《Effective Python》已经成为全球广受欢迎的Python编程参考书籍。

除了Guido和林海峰两位大佬外，还有其他很多优秀的译者投稿或担任审校工作。其中，马超（张煜泽）翻译版本的《Python Cookbook（第二版）》已然成为Python编程圈不可或缺的一本书，它由许多经验丰富的Python开发者合著，是最具代表性的Python编程相关的教程书籍之一。而李兴（刘强东）则是极少数同时拥有Python和机器学习背景的技术大牛，他编写的《机器学习实践》也是一本具有独创性的Python编程语言参考书籍。

《Effective Python》的最新版发布于2020年，是由美籍华裔文学编辑张小龙（高远）主编，陈俊猛（张茹凤）、蔡欣然（李春玲）、邱勇昭（吴帅冬）、马红梅（陈天翼）、罗源（廖秋云）等译者组成的专业技术人员编译的经典著作。如今，《Effective Python》已得到众多程序员的喜爱，被誉为“最好用的Python参考书”、“最佳Python学习指南”。
