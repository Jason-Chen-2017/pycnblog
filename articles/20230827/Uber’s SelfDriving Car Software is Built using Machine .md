## 1.2 摘要
Uber自动驾驶车辆软件是一种基于云计算技术的软件系统，通过识别和跟踪不同环境对象并自动执行决策，驱动汽车执行前进、转弯、停车等动作。这些自动驾驶系统可以提高交通效率、减少拥堵和经济损失，是提升社会公共服务水平的一项重要举措。目前，Uber的自动驾驶车辆软件主要依赖于传感器、定位系统、底盘控制系统、通信系统、决策算法等硬件和软件模块。然而，Uber开发者们不仅想使自动驾驲车辆软件更加智能化、精准化，而且也期待通过ML技术来改进算法，缩短开发周期、降低成本。本文中，作者将详细阐述Uber使用ML平台构建自动驾驶软件的基本思路，以及Uber独有的一些独特优势。文章的分析发现，即使在这一过程中，Uber仍然坚持自己的模式，坚守传统车辆控制体系，保障交通安全和效率，尤其是在存在诸多系统级风险、道路异样场景下，其软件系统仍然能够稳定运行。

# 2.什么是机器学习？
## 2.1 机器学习的定义
机器学习（英语：Machine Learning）是人工智能领域的一个子领域，它研究如何让计算机具备自我学习能力。一般来说，机器学习是指一类能学习、优化并改善性能的计算机科学方法。与其他计算机科学分支相比，机器学习通常被认为更接近于人类的认知过程，因为它涉及到使计算机像人一样学习、制定决策、解决问题、适应变化。

机器学习通过输入、输出和算法三个基本要素，可以自动地从数据中学习并提取知识。机器学习所处理的数据可以是各种类型、大小和质量的，例如图像、文本、音频、视频等。机器学习模型由学习算法、训练数据、统计模型组成。学习算法用于指导模型学习，如分类算法、回归算法、聚类算法等；训练数据是模型所使用的数据集，用来训练模型，一般由多种类型、数量和质量的数据组成；统计模型则是机器学习的核心组件，包括概率分布、决策树、随机森林、神经网络等。

机器学习具有强大的学习能力，但它并不是一键打开的自动机器，而是需要大量的工程实践才能真正落地。机器学习模型的效果还受许多因素影响，例如数据质量、算法参数设置、训练样本数量、特征选择、正则化参数等。所以，机器学习需要进行有效的实验设计、超参数调优、模型评估等一系列的工程工作。
