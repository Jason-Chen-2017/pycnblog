# 2.环境准备
## 2.1 安装 Python
首先安装 Python 运行环境，可以从官方网站下载安装包安装，也可以使用各大 Linux 发行版的软件管理器进行安装。安装成功后，打开命令提示符（Windows 下叫“终端”）输入以下命令验证是否安装成功：
```python
python --version
```
如果输出了 Python 的版本号，表示安装成功。
## 2.2 安装IDLE
IDLE 是 Python 的交互式命令行界面（Command Line Interface，CLI）工具，可以方便地编写、运行、调试 Python 代码。要安装 IDLE，可以直接从 Python 安装目录下的 Scripts 文件夹里找到 IDLE-script.pyw 脚本文件，双击运行该脚本文件即可。
## 2.3 安装 Anaconda
Anaconda 是 Python 数据分析包的一个开源发行版本，提供了超过 1700+ 个科学库包。它包含了最新的 NumPy、SciPy、pandas、Matplotlib 等数据分析包，以及 IPython/Jupyter Notebook 等开发环境和工具。由于 Anaconda 中内置了许多科学计算和机器学习常用的工具包，非常适合于学习和快速上手数据分析。为了更好地演示，本教程还将安装 Tensorflow 和 Keras，两者都是深度学习框架。因此，建议阅读本文时，如果您的电脑配置较差，可以使用我提供的镜像源进行安装。
### Windows 系统安装方式
然后，在命令提示符（Windows 下叫“终端”）下进入解压后的文件夹，执行如下命令（按回车键执行）：
```
conda create -n py3 python=3 anaconda
activate py3
jupyter notebook
```
等完成所有依赖项安装后，会自动打开浏览器并启动 Jupyter Notebook。在浏览器中打开笔记本 `test_notebook.ipynb` ，根据提示完成安装。安装完成后，点击左侧 “New” 创建新笔记本。在“单元”菜单栏中选择第一个单元，输入 `print("hello world")`，然后按 Shift + Enter 执行该单元。如果正常输出了 “hello world”，则表示安装成功。
### Linux 系统安装方式
在 Linux 系统上安装 Anaconda 可以参照如下命令（安装前请确认已安装 git 命令）：
```
wget https://repo.continuum.io/archive/Anaconda3-2019.10-Linux-x86_64.sh
bash Anaconda3-2019.10-Linux-x86_64.sh
source ~/.bashrc
```
上述命令会下载最新版 Anaconda 安装包，并自动安装至当前用户目录下的 Anaconda3 文件夹；然后激活 Anaconda 环境并启动 Jupyter Notebook。如果需要在后台运行 Jupyter Notebook，可以在命令末尾添加 `--no-browser`。

至此，Anaconda 安装完毕。

# 3.基本概念术语说明
## 3.1 变量
变量是存储数据的容器，在程序中用来存放值。变量的名字通常由字母、数字和下划线构成，且严格区分大小写。其命名规则为：只能以字母开头，只能包含字母数字和下划线，长度不限。例如：`name = "Alice"`、`age = 30`。
## 3.2 字符串
字符串（string）是一个字符序列，即一串文字，由 0 个或多个 Unicode 编码的字符组成。字符串用单引号 `' '` 或双引号 `" "` 表示。
字符串的拼接操作是将两个或更多字符串按照顺序连接起来形成一个新的字符串。比如：`"Hello," + "world!"` 会返回 `"Helloworld!"`。
字符串的索引操作是指根据位置获取字符串中的某个字符。字符串的第一个字符的索引值为 0，第二个字符的索引值为 1，依此类推。可以通过整数索引或者切片来访问字符串中的子字符串。比如："Hello"[0] 将返回 'H'，"World"[2:4] 将返回 'll'。
字符串的长度（即字符个数）可以使用 `len()` 函数获得。比如：`'abc'.__len__()` 返回 3。
## 3.3 列表
列表（list）是有序集合，可以容纳不同类型的元素。列表用方括号 `[]` 来表示，元素之间用逗号 `,` 分隔。列表的索引和字符串类似，可以用整数索引或者切片来访问列表中的元素。
列表的更新操作是指给列表中的元素重新赋值。比如：`lst[1] = 100` 会把列表 lst 中的第二个元素的值改为 100。列表的插入操作是指向列表的末尾添加元素。比如：`lst.append(10)` 会把 10 添加到 lst 的最后面。列表的删除操作是指从列表中移除元素。比如：`del lst[0]` 会移除 lst 中的第一个元素。
列表的长度可以使用 `len()` 函数获得。比如：`lst.__len__()` 返回 4。
## 3.4 元组
元组（tuple）与列表类似，也是有序集合，但它的元素不能修改。元组用圆括号 `()` 来表示。
元组的索引和列表相同，可以用整数索引或者切片来访问元组中的元素。
元组的长度和字符串的长度相同，可以使用 `len()` 函数获得。比如：`(1, 2, 3).__len__()` 返回 3。
## 3.5 字典
字典（dict）是无序的键值对集合。字典用花括号 `{}` 来表示，每个键值对之间用冒号 `:` 分隔。字典的索引是通过键来查找值的，而非整数索引。字典的更新操作是指给字典中已存在的键赋值，新增的键值对会自动加入。比如：`d['age'] = 30` 会把字典 d 中键为 ‘age’ 的值更新为 30。字典的删除操作是指从字典中移除指定的键值对。比如：`del d['name']` 会把字典 d 中键为 ‘name’ 的值移除。
字典的长度可以使用 `len()` 函数获得。比如：`d.__len__()` 返回 4。
