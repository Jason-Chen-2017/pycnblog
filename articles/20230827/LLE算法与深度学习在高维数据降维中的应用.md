# 1.简介
  

## 一、什么是LLE？
Locally Linear Embedding (LLE) 是一种非线性降维方法，其关键思想是在局部空间中找到一个低维嵌入，使得原始数据的相似性关系能够较好地被保留下来，同时保证降维后的数据具有良好的可视化特性。

## 二、为什么要进行LLE？
传统的降维方法，如主成分分析PCA、线性判别分析LDA等基于全局的方法无法处理高维数据的复杂结构，而LLE可以在一定程度上解决这一难题。另外，LLE通过保留相邻点之间的相似性关系来有效实现降维，因此可以提高数据分析的精度。

## 三、如何进行LLE？
### （1）基本概念
1. 高维数据：指具有多个特征变量（attribute），且数量级越高的数据集；
2. 局部：指数据集中的某个区域，这里的区域通常是一个固定大小的领域，比如局部二阶领域；
3. 低维嵌入：通过某种映射将高维数据压缩到低维空间，同时保持了数据的局部结构；
4. 核函数：用于衡量两个样本点之间的相似性，不同核函数所对应的相似性度量有所区别；
5. 拉普拉斯近似：将高维空间的数据转换到低维空间时，需要对每个样本点所在的空间区域内进行计算，为了提高运算效率，通常采用矩阵分解的方法，即通过将高维空间数据矩阵先分解为三个矩阵A、B、C，然后将数据点映射到低维空间时只需计算它们与A矩阵相乘即可；
6. 寻找局部线性嵌入：在局部的每个领域内寻找一个低维嵌入，从而达到降维的目的。

### （3）具体步骤
#### a. 生成核函数K(xi,xj)，核函数根据距离的远近程度计算两个数据点之间的相似性。常用核函数包括多项式核、径向基函数核等；

#### b. 通过核矩阵K，计算协方差矩阵C=ΣΣ^(-1/2)*K；

#### c. 求解矩阵L=C*W，其中W是个权重矩阵；

#### d. 将数据集X映射到低维空间Z=XL；

#### e. 可视化：将映射后的数据可视化，不同类别的数据被映射到低维空间中靠近的区域，因此可直观地判断数据的聚类结果。

### （4）LLE存在的问题
由于LLE是一种基于局部的降维方法，只能保留局部相似性信息，对于离群点、噪声数据等没有很好的降维效果，且降维后数据的可解释性较差。因此LLE算法作为一种优秀的降维方法得到了广泛的应用。但是LLE算法也存在着一些缺陷，比如计算量比较大、复杂度高、参数不容易调节、运算速度慢等问题。

# 2. 基于LLE算法的超市商品推荐系统
## 1. 问题描述
假设有一个超市商场，系统可以根据用户的购买行为记录，对其可能喜欢的商品进行推荐。该系统接收到的用户浏览历史记录数据可能会包含大量冗余信息，例如用户的年龄、性别、居住地、消费习惯等，这些信息往往对商品推荐并无帮助。因此，需要对用户数据进行有效的筛选，去除这些无关因素，提取重要的用户特征，从而更好地进行商品推荐。

假设用户的浏览历史记录中包含以下特征：

1. 用户ID
2. 用户浏览时间戳
3. 商品ID
4. 商品价格
5. 商品类别
6. 商品品牌
7. 浏览过程的停留时间

## 2. 数据清洗与准备工作
首先对原始数据进行初步清洗：

1. 删除重复记录
2. 根据商品品牌和类别进行统计分析，删除无意义的品牌或类别
3. 对商品价格进行标准化，统一单位为元
4. 对浏览过程的停留时间进行归一化处理，使停留时间之间呈正态分布

之后将用户浏览历史记录按用户ID排序，方便后续处理。

## 3. 特征工程
### （1）用户浏览历史记录特征
通过对用户浏览历史记录特征进行分析，可以得到以下发现：

1. 用户的年龄、性别、居住地等大多数都与商品无关，所以可以将这些信息舍弃。
2. 商品价格与浏览时间间隔密切相关，可以作为用户的首次点击偏好，或者作为用户的购买决策依据。
3. 浏览过程的停留时间虽然与商品有关，但停留时间较短的用户反映出的偏好可能更加重要，因此可以使用分桶的方式进行离散化。

### （2）商品特征
对商品信息进行特征工程，主要包括商品品牌、类别和价格两方面。

- 商品品牌与类别：因为商品品牌或类别在浏览历史记录中并不会直接体现出用户的偏好，所以可以考虑将这两个信息融入商品的特征向量中。
- 商品价格：同样的，商品价格对商品的喜爱程度也会产生影响，因此可以通过商品价格的均值来代表一个商品的平均价值，再加上商品的点击次数等其他特征，建立商品的特征向量。

## 4. 模型训练
基于训练集，使用LLE算法对用户特征进行降维，得到商品特征降维后的表示向量。

## 5. 模型预测
给定一个新用户的浏览历史记录，基于LLE算法降维后的特征向量，结合商品的特征向量，使用线性回归模型进行推荐。

# 3. 深度学习简介
## 1. 前言
深度学习作为机器学习的一种方式，取得了巨大的成功，取得了一系列的突破性进展。随着互联网的发展，越来越多的人开始关注到人工智能、计算机视觉、自然语言处理等方向的研究。深度学习的兴起是以更快的训练速度、更强大的模型性能而闻名的，近些年来，深度学习的发展取得了举足轻重的作用。

## 2. 简介
深度学习由浅层神经网络与深层神经网络组成，具有高度的抽象能力、非线性拟合能力、自动提取特征的能力以及较强的学习能力。深度学习的目标就是对任意一个任务的输入数据，能够生成一个合适的输出结果。

## 3. 发展历史
深度学习的发展始于上世纪末期，当时，日本国立大学教授林明觉得通过组合特征来进行分类会比单独处理每一个特征效果要好，于是提出了卷积神经网络CNN。经过几十年的发展，深度学习已经逐渐成为机器学习领域的一块重要研究热点，拥有庞大的应用范围。

目前，深度学习主要分为三大类：监督学习、无监督学习和强化学习。

1. 监督学习：监督学习是深度学习的一种方法，它的目的是训练一个模型，使得输入数据与输出结果之间的关系能够被学习到，根据已知的数据对未知的数据进行预测、分类和回归等任务都属于监督学习。
2. 无监督学习：无监督学习是一种与监督学习相对的机器学习方法，它可以让算法自动发现数据中的隐藏模式，并试图找到数据中潜藏的关系、规律、结构等。
3. 强化学习：强化学习是一种与环境的交互式学习方法，它结合了动作选择、奖励分配、状态评估等机制，通过与环境的持续互动来优化策略，获得最大的奖励。
