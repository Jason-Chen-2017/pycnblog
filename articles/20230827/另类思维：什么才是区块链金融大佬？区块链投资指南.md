# 1.简介
  


前言

随着比特币的崛起、以太坊的兴起和火爆的ICO浪潮，越来越多的人开始关注区块链技术。那么，真正的区块链金融大佬到底是谁呢？又有哪些行业需要了解区块链技术？区块链投资中应该如何选择项目？这些都是值得探讨的问题。本文将从中小企业、个人以及各个投资领域的角度出发，全面剖析区块链金融大佬。同时，也会结合大量案例，为读者提供更具体、实际的信息，帮助读者准确把握区块链投资的方向。

在开始阅读之前，首先要搞清楚一些基础知识。

什么是区块链？

区块链是一种分布式数据库，其中的数据并非孤立存在于某个节点上，而是通过密码学技术相互连接，从而实现价值的传递。它主要由两大部分组成——存储层（Ledger）和共识层（Consensus）。存储层用于记录交易信息，而共识层则负责整个系统的正常运行。此外，由于区块链的不可篡改特性，也能够防止各种各样的经济活动的进行，使得区块链具备了去中心化、透明、安全等特点，可谓“链”的奥秘所在。

何为区块链资产？

所谓的区块链资产，即是基于区块链技术构建的数字货币或者加密资产，它不像传统的银行卡、信用卡那样可以被实体机构直接掌控。相反地，它通过区块链网络自动运行，确保所有用户都能得到公平有效的货币供应。所以说，区块链资产具有独有的资本运作模式，且更加灵活、无风险、便捷，且为不同行业的参与者提供了参与的平台。

什么时候适合采用区块链？

区块链能够在以下方面产生巨大的应用变革：
- 支付场景：现在越来越多的应用场景正在依赖于区块链技术。例如，支付宝、微信支付、苹果支付等都已经支持了数字货币的收款，在日益增长的数字化商业世界里，区块链将成为数字货币的重要角色；
- 防伪溯源：区块链上的数字资产可以提供真实可靠的历史信息，可以有效避免造假、抢夺等风险；
- 金融支付：区块链对数字资产的流通进行管理，让更多的实体参与到这一领域。比如，金融企业可以使用区块链进行金融支付、结算等服务的开展；
- 物联网（IoT）安全：通过区块链技术，可以对物联网设备的数据进行全面管理，提升安全性和隐私保护；
- 商业与社会：区块链能够对商业和社会的关系产生极大的影响，例如，区块链可以帮助中小微企业解决信息孤岛问题、物流难题，以及建立合规、透明、公正的贸易环境；

那什么才是区块链金融大佬？

区块链作为一种新的金融科技，其理念还是很新颖的。但如果将区块链纳入到金融的视野内，就会发现其真正的力量正在逐渐显现出来。目前，真正做到了第一枪杆子的区块链金融大佬只有两个——Hyperledger Fabric 和 Chain.com。后者是一个基于区块链的数字资产国际站点，并且已经进入相当复杂的阶段。这就意味着，区块链技术对于普通百姓来说依然远远不能掌握，只有金融大佬和政客们才能有更好的发展空间。不过，更多的金融企业、创业公司、投资人以及政客们正在寻找加入这个行业，因为他们可以为公众提供更方便、更可靠的金融服务。

什么行业需要了解区块链技术？

虽然区块链技术已有非常广泛的应用，但仍有很多行业和领域还处于萌芽状态。其中最需要注意的是数字钱包的发展方向。如果各家都在推动数字钱包发展，那么将会出现越来越多的数字钱包竞争，而最终取胜的肯定就是区块链技术带来的新型金融服务。另外，其他行业，如证券市场、知识产权、物流、医疗健康、电商、养老金、房地产、网络游戏、新能源汽车等等都有着自己的区块链项目。

4.区块链金融大佬是谁？

   Hyperledger Fabric 和 Chain.com 是目前真正的区块链金融大佬。
   
   Hyperledger Fabric 是 Hyperledger 基金会开发的一个开源框架，能够用于创建信任的分布式应用程序。该框架允许开发人员创建分布式计算应用，并将这些应用部署到多个网络中，实现共识和数据共享。
    
    而 Chain.com 是 Hyperledger 基金会官方推出的数字资产国际站点，为创业者和投资人提供高质量的区块链数字资产资讯。该站点目前聚集了一批优秀的区块链从业者，通过网站提供完整的区块链资讯、评测、教程、白皮书、路演等资料。而 Chain.com 的愿景是让区块链和数字货币的知识和工具为公众所用。
    
    当然还有很多优秀的区块链金融大佬或创业者。例如，Horizen 旗下的 ZBG Token，鼎盛的 Swirlds Blockchain Technologies，由锂离子工业集团创始人马修·震担任 CTO，Swisscom Blockchain Team 中的 Hannes Asmus 等，还有很多成熟的区块链投资人和项目。
    
    下面详细介绍一下这两个区块链金融大佬。
    
5.区块链金融大佬 Hyperledger Fabric

    Hyperledger Fabric 是 Hyperledger 基金会开发的一个开源框架，用来构建去中心化的联盟区块链系统。它由两个关键部件组成：Fabric SDK（分布式账本SDK）和 Hyperledger Fabric（联盟链），而且 Hyperledger Fabric 有多种编程语言版本的实现。
    
    区块链之父 IBM 董事局主席兼首席执行官 <NAME> 表示，“Fabric 是目前最重要的区块链技术项目”。
    
    区块链行业专家奥斯卡·迈克尔·舒曼也表示，“如果你想了解 Hyperledger Fabric，你一定要看 Hyperledger Fabric。”
    
    Fabric 架构图
    
    
    上述架构图展示了 Hyperledger Fabric 的整体架构。其中，Client App 发出交易请求，通过 Fabric Gateway 向 Ordering Service 提交交易请求，Ordering Service 将交易请求顺序提交给 Peer 结点，Peer 结点根据共识协议生成区块，并将区块传播至网络中的其他结点。
    
    交易请求中包含的信息包括交易信息（Transaction Info）、交易签名（Signature）等，这些信息经过排序后形成区块，并最终提交给网络中。
    
    同时， Hyperledger Fabric 使用可插拔的插件机制来扩展其功能。比如，除了默认的排序和共识机制，它还可以通过不同的共识插件进行配置。这样就可以轻松地切换共识机制，以满足不同的业务场景。
    
    此外， Fabric SDK 可以用来开发区块链应用，包括联盟链和分片链，也可以将其集成到现有应用程序中。
    
6.区块链金融大佬 Chain.com

    Chain.com 是 Hyperledger 基金会官方推出的数字资产国际站点，为创业者和投资人提供高质量的区块链数字资产资讯。该站点聚集了一批优秀的区块链从业者，通过网站提供完整的区块链资讯、评测、教程、白皮书、路演等资料。而 Chain.com 的愿景是让区块链和数字货币的知识和工具为公众所用。
