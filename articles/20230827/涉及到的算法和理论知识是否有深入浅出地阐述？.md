# 2. 概览

## 2.1 概率论

首先，概率论是指研究随机事件发生的概率、规律，以及确定各种可能情况发生的先后顺序。概率论的一个重要内容就是“贝叶斯定理”，它提供了关于概率分布的最基本的定理。

具体而言，假设有两个事件A和B，A、B都是独立的，且两者互不相容，则：

P(A∩B)=P(A)P(B)

如果对某个事件A有多个可能结果，则可以将其结果分成互斥的子集，每个子集称作类，类中各个元素构成一个集合，称为样本空间，例如：

S={a1,a2,...,an}表示事件A的样本空间；
C={C1,C2,...Ck}表示A的k个可能的类；
πi=P(Ci), i=1,2,...,k；

则：

1）P(A)=-∑pilogpi，当A为无穷集时，P(A)=1；

2）P(A|B)≈P(A)P(B/A)，这个定理表明，在给定B的条件下，估计事件A的概率等于A与B同时发生的概率；

3）P(A)P(B/A)=P(B)P(A/B)，这条定理表明事件A和B互换时，概率也会交换。

这些定理都是建立在可列取样本空间上的分布的基础上的。

## 2.2 线性代数

第二个部分介绍的是线性代数。线性代数研究对向量进行运算并取得线性组合的能力。向量是一个很重要的数据结构，它可以用于描述特征，也可以用于表示数据点之间的距离、相关性等。

在线性代数里，向量通常用列向量或行向量表示。一组列向量可以构成矩阵，一组行向量可以构成矩阵。矩阵可以有不同的维度，比如，二维矩阵的维度分别为m*n，m为行数，n为列数。通常情况下，矩阵有如下性质：

- 矩阵的加法、减法和数乘运算满足结合律和分配律；
- 对于任意的非零向量x，都存在一阶范数，即||x||，它表示x在由单位向量组成的方向上的投影大小；
- 如果矩阵A的秩r，则必然有n=nr。

特别值得注意的是，通过矩阵运算，可以进行数据的降维、归一化、线性回归等。

## 2.3 凸优化

第三个部分介绍凸优化。凸优化是一种基于约束的优化方法，它旨在最大化或者最小化目标函数，同时满足一些限制条件。在监督学习问题中，限制条件往往是输入的某个维度的值。

如今的很多机器学习任务都可以转化为求解凸优化问题，如分类问题、回归问题、聚类问题等。凸优化问题具有以下几个显著特征：

- 目标函数是凸的（convex function），也就是说，其函数形状类似于抛物线；
- 有一些变量是不受限制的（unconstrained variable），即变量值没有限制，只要不违反某些限制条件即可；
- 对目标函数的一阶导数是严格凹的（strictly convex），也就是说，它的一阶导数不会出现严重的倒锤效应。

因此，凸优化可以被看做是在优化问题上的切片处理方式。很多机器学习算法都可以看做是凸优化算法的实现。

## 2.4 统计学习方法

第四个部分介绍统计学习方法。统计学习方法（Statistical Learning Method，SLM）是机器学习中的一个重要分支。它旨在从数据中提取有用的信息，构建有效的模型，预测结果和未知的数据。

具体来说，统计学习方法的核心是构建模型，即学习到数据生成的过程。模型可以分成三种类型：

- 判别模型（Discriminative Model）：学习到一个决策函数，能够根据输入样本预测输出标签。典型的判别模型有感知机、Logistic Regression、支持向量机等。
- 生成模型（Generative Model）：学习到联合概率分布p(X,Y)。典型的生成模型有隐马尔科夫模型、Naive Bayes、混合高斯模型等。
- 混合模型（Mixture of Models）：将多个模型组合起来，形成更复杂的模型。典型的混合模型有EM算法、GMM、朴素贝叶斯等。

除了构建模型之外，统计学习还需要解决过拟合的问题。过拟合是指模型把噪声、离群点、错误例子等误导性的特征放到了模型里面，导致泛化能力下降。解决过拟合的方法有两种：

1. 正则化：限制模型的复杂度，使之不要过分依赖训练数据中的噪声或离群点。
2. 模型选择：评估模型在测试数据上性能，选择性能最好的模型。

## 2.5 深度学习

第五部分介绍深度学习。深度学习是机器学习的一个分支。它利用多层神经网络来模拟复杂的非线性关系。传统机器学习算法只能看到局部信息，而深度学习算法能够学习到全局信息。

深度学习有三个关键点：

- 使用多层网络模拟复杂非线性关系；
- 数据分布存在冗余，利用深度网络可以对数据进行抽象；
- 通过反向传播算法训练网络参数。

## 2.6 强化学习

最后一个部分介绍强化学习。强化学习（Reinforcement Learning，RL）研究如何在与环境互动过程中，学习到让自己生存下去的策略。RL适用于许多任务，包括游戏、机器人控制、金融市场等。

特别地，RL可以分为动态编程（Dynamic Programming，DP）和蒙特卡洛树搜索（Monte Carlo Tree Search，MCTS）。动态编程直接对MDP进行建模，得到最优的策略；而蒙特卡洛树搜索利用树结构来近似计算策略的价值函数。

与其他机器学习算法不同，RL是以序列的方式运行的，所以也需要考虑序列的特性。
