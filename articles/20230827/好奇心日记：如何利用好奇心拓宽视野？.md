# 1.简介
  


好奇心（curiosity）这个词语是由尼尔·波利亚在1910年提出的，他对外号为“生活之眼”，意指“追求各种新奇事物的能力”。即使是在科幻小说和漫威电影里，科学家、工程师也会因着好奇心而激发。

人们都喜欢做一些能够让自己感到兴奋的事情，比如阅读畅销书、听音乐、旅行、看电影等。但是，每个人的好奇心都是不同的，有的人觉得科技的革命正在进行，有的人在纠结谁将领导全球，有的只是怀抱着一种执念，希望通过探索发现新的可能性。 

无论是好奇心日记还是普通的个人日志，都可以帮助我们记录自己的好奇心，甚至是别人的好奇心。今天我要分享的是一个好奇心日记的完整流程，帮助你把好奇心变成现实。

# 2.前期准备
## 2.1 获取知识

首先，你需要找到你的兴趣爱好。如果你是一个计算机专业人员或者相关工作者，那么你肯定对计算机科学或者编程感兴趣；如果你是一个经济学专业人员，那你一定对经济学感兴趣；如果你是一个医生、教育专业人员，你就很可能对社会和健康保障有着浓厚的兴趣。

当然，除了兴趣之外，你还需要考虑时间和精力。每天花几分钟的时间来了解一下当下的热点新闻或世界经济走向都不算什么，但每天花几个小时去阅读一本新书则可谓“费力不讨好”。

所以，在开始创作之前，先收集一些相关信息，包括你要读什么书籍、哪些网站可以获取到最新资讯。这样一方面可以加快自己的学习速度，另一方面也可以避免重复劳动。

## 2.2 选择平台

在开始写作之前，你需要选择一款适合自己的笔记应用或博客平台。一般来说，笔记应用如OneNote、Evernote都非常适合写长篇大论的文章，而博客平台则更为适合短小精悍的散文、日记类文字的创作。

笔记应用的优势是它支持多种格式的文本编辑，因此你可以用简单易懂的语言来呈现你的想法，同时图片、表格、公式也能完美地嵌入其中。它的缺点是缺少交互性，你无法直观地看到你的创作结果，只能依赖上下左右键、滚轮等方式切换窗口；博客平台则有丰富的模板，而且界面清爽、导航简洁，因此更适合用来写简单的心情日记或随笔。

为了帮助你更好的管理你的文章，建议尽量选择免费或开源的平台，这有利于你的创作自由，同时也不会有太大的成本。而且，写作过程中遇到的任何困难均可以通过社区提供的资源或帮助来解决。

## 2.3 构建主题

写作过程中，你需要持续不断地学习、总结、吸收新的知识和经验。所以，建议你建立一个关于自己的主题或话题来进行观察、筛选和组织。

你可以从书单开始，逐渐扩展成主题列表，然后用颜色标签进行分类。这有助于你集中注意力，并在阅读、记录和回忆的时候快速找到所需的内容。

例如，我的主题列表包括：

- 计算机科学：包括程序设计、软件开发、数据结构、编译原理、计算机网络、数据库系统、操作系统、系统软件等。
- 经济学：包括国际金融、产业政策、投资策略、消费行为等。
- 医疗保健：包括医疗卫生、养老保险、医疗器械等。
- 教育培训：包括课程设计、教师训练、学生评估等。

# 3.创作阶段
## 3.1 意识流驱动的创作

创作的方式可以有很多种，但最有效的一种就是意识流驱动的创作。这种方法的关键在于，你在创作的同时，应该留意自己的兴趣所在、所处环境、有哪些疑问或担忧。

这样的话，当你被自己的好奇心所驱动，而又没有别的事情耽误时，你的思绪就会自动进入一个全新的境界。

当我第一次接触到计算机科学时，我便有了好奇心，于是便花了许多时间去阅读相关的书籍和论文。由于我对编程和计算机底层机制有极大的兴趣，于是便买了一本名为“深入理解计算机系统”的书。该书着重介绍计算机硬件、操作系统、编译系统、数据库系统、网络协议、分布式计算等方面的知识。

我写下了第一页，说道：“自从我读过这本书之后，我对计算机的认识发生了质的飞跃。”我慢慢地打开笔记本开始记录，记录了计算机的各种术语、概念、原理、架构和软件实现过程。第二页，我写道：“其实，这些知识都可以在实际应用中运用，而我也正是在阅读这本书的过程中逐步掌握它们。”

这时的我已经开始产生一定的成果，于是便想着如果我能把我的研究成果写成一篇长篇大论的文章，发表出来，引起大家的关注，或许这对我的职业发展、事业规划都会有所帮助。

没想到，竟然引起了一场轩然大悲的热潮。几乎同时，我也陆续读到了一些其他主题的书籍，于是我就把这些书籍放在一个文件夹里，并重新整理我的主题列表。

第三页，我写道：“当我发现这些主题都离不开计算机系统时，我又回到那本名为‘深入理解计算机系统’的书上继续学习。”我重新阅读了这本书的第四章——虚拟机，又花了时间去学习虚拟机技术的基础知识。

当我第一次从经济学转变成计算机科学，我也着迷于研究程序运行的详细机制，因此我又买了一本名为“深入理解程序执行”的书，这次我主要研究进程、线程、协程、异步I/O等方面的技术。

此时，我已经满脑子里的是程序运行的各种细节，却忽略掉了一些更重要的东西——比如用户体验。于是，我花了一些时间再读了一遍经济学方面的书籍。

也就是说，我在不停地阅读和思考，不断地调整自己的主题列表，并试图找出自己的兴趣共鸣。这样一来，我的文章的创作才会充满生气色彩。

## 3.2 记录方式

既然你已经有了一个好的主题列表，那么创作的第一步就是确定你的记录方式。你可能有以下两种选择：

1. 深度阅读：先从最初的需求出发，逐渐深入到某个细枝末节的地方，读完后深呼吸一口气，重新审视你的理解，确认你的疑惑是否得到了正确的答案，并反复确认并修正错误。
2. 主题阅读：从主题开始，逐渐跳到相关的参考文献或网站，浏览前人的研究成果，分析其中的理论和实际。这样可以使你快速理解作者的思路，理解他的研究思路，并找寻其中的优秀之处。

第二种方式显然比第一种方式更能给予你更高的概括性和准确性。对于我来说，我常常采用第一种方式，因为我比较喜欢阅读那些比较晦涩难懂的技术文档。

无论采用哪种记录方式，你都应该注重自己的语言表达能力，尤其是文章的结构。每条记录应该有一个主干，并配以必要的插图、照片和图表，以帮助自己梳理自己的思路。

# 4.后期制作

## 4.1 总结归纳

文章的最后一部分是总结归纳，也是完成整个创作过程中的重要一步。这一步的作用在于，让你从繁杂的材料中汲取营养，清理逻辑上的混乱，并形成一个整体的认识。

在文章的最后，你需要自己总结自己所学到的知识，包括新学到的技能、掌握的理论、思维模式、价值观、感悟等，并加以概括和展望。

总之，你要坚持阅读、写作，并且充分利用自己的兴趣，不断进化、突破，最终发现属于你的美妙世界！
