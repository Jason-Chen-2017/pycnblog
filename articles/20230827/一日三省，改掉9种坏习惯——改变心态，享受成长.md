# 1.简介
  

每个人都有自己坏习惯，好的习惯可以帮助我们更加有效地工作和生活。每天，我们都要做好9件坏习惯的防范。以下9种坏习惯需要被快速摒弃。让我们一起改掉它们吧！
# 1、过分关注时间安排。“按时交付”、“及时回复”等时间敏感要求，会导致我们无法集中精力处理重要事务。我们应该充分利用工作的时间，把注意力放到那些不紧急的事情上。如果不能完全杜绝这些行为，可以适量减少或者调整任务的优先级。
# 2、不重视重要事件。不去关注重要事件（比如突发的疾病），可能导致我们对后续情况漠不关心。同时，由于缺乏重视，可能会忽略一些重大灾难性事件带来的影响。我们需要记住，如果发生了灾难性事件，我们需要马上采取行动，对这个事件负责任。
# 3、懒惰地接受新事物。总是在意识不到新事物的存在时就充当信息源，这可能会导致我们不知所措且缺乏热情。我们应该坚持采用积极主动的方式，尤其是在看起来危险或困难的事情上。不要妄自菲薄，因为只看到表面，我们可能陷入恶性循环。
# 4、对于自己的承诺过于苛刻。即使对方给出了很高的期望，我们也可能违背承诺。我们需要谨慎行事，即使给对方压力也是如此。我们应该坦率地说出自己的想法，尤其是如果存在潜在风险或危险。
# 5、浮躁。总是充满活力、充满希望，甚至对未来的美好憧憬，可能导致我们浮躁不安。我们需要明确界定目标，设定时间表，并时刻保持冷静。
# 6、忙碌导致忽略个人兴趣。任何时候，只要有人打扰我们，就容易忽略个人兴趣和爱好。我们需要花费更多的时间来关注真正重要的事情。
# 7、觉得自己能力不足。“我明白”，“别担心”，“我学得很快”这样的口头禅，可能导致我们低估自己的能力。我们需要清楚地认识到自己有多优秀，并努力向前走。
# 8、过分在乎小利益。由于小利益导致的抵触心理，我们可能忘记了更大需求。我们需要摒弃这种错觉，考虑全面而广泛的发展。我们需要把注意力放到提升组织整体效率、增加收入和降低成本上。
# 9、对于学习和知识的追求过于热衷。很多人认为学习是一个枯燥乏味的过程，但实际上却非常享受。学习能够促进我们的职业发展，培养我们成为一名成功者。但是，当我们过度投入学习而没有深度思考时，我们可能走入误区。因此，我们应该认识到学习的必要性，但不要太过执着。
通过改掉9种坏习惯，我们就可以改善我们的工作和生活。我们将获得更加有效的工作时间，精神状态更加平稳，更有利于我们的成长。
