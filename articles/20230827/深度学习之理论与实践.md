# 2. 基本概念术语说明
## 2.1 概念
深度学习（Deep Learning）是一种机器学习方法，它的基本理念是构建多层次非线性分类器，使得深层神经网络能够学习到数据的内部结构并提取有效特征，从而实现模式识别、图像识别、自然语言理解等任务。它是一个基于学习的优化方法，不同于传统的统计建模或者决策树算法等白盒模型，深度学习的模型由多个非线性层组成，可以高度抽象的理解输入的数据，可以学习到复杂的函数关系。深度学习是指通过多层次组合低阶元素构成高阶元素，直至拟合数据结构的多层神经网络学习系统，这种学习系统能够从原始数据中发现隐藏的模式和规律，并逐渐预测未知数据，可以应用于很多领域，如图像识别、自然语言处理、生物信息学、金融分析、语音识别、推荐系统等。

## 2.2 术语
### 模型(Model)
深度学习中的模型通常是由一系列层(Layer)组成的多层神经网络。每层包括一组神经元(Neuron)，这些神经元之间存在着连接，每个神经元接收上一层的所有输出并计算当前层的输出值。神经网络的最后一层通常输出结果，但是中间层的输出也可以作为下一层的输入。

### 数据集(Dataset)
训练数据集用于训练模型，测试数据集用于评估模型的性能。数据集是指有标签的数据集合，用于训练模型去学习数据的内在特性及规律。一般来说，训练数据集需要比测试数据集更加全面，才能充分评估模型的泛化能力。

### 损失函数(Loss Function)
模型学习过程中使用的一个指标，用于衡量模型输出和正确输出之间的距离。损失函数一般采用最小化的方法，即用最小的loss减少模型的误差。例如，对于分类问题，常用的损失函数是交叉熵（Cross-Entropy）。

### 优化算法(Optimization Algorithm)
用于求解损失函数的参数，使得模型获得最优效果。目前常用的优化算法有随机梯度下降法(Stochastic Gradient Descent，SGD)、动量法(Momentum)、Adam、Adagrad等。其中，Adam算法是自适应学习率调整方法的一种，在一定程度上可以避免陷入局部极小值。

## 2.3 模型类型
深度学习模型可以分为三类：
* 基于神经网络的模型（Neural Network Modeling）：深度学习的基本模型是多层神经网络。深度学习的前身——人工神经网络（Artificial Neural Network，ANN），也属于这种模型。在过去的几十年里，基于神经网络的模型的研究和发展引起了强烈的社会变革，现代深度学习的主要工作都围绕神经网络模型进行。

* 基于树的模型（Tree-based Models）：决策树、随机森林、GBDT(Gradient Boosted Decision Tree)都是基于树的模型。它们在训练时使用了一组自顶向下的规则，并通过迭代的方式拟合数据中的复杂结构。决策树在学习时通过不断划分节点来建立分支，而随机森林和GBDT则利用了随机森林的正反馈机制，通过建立一系列的弱分类器来降低分类误差。

* 无监督模型（Unsupervised Learning Model）：无监督模型旨在学习数据中潜在的结构，但是没有给定目标变量。例如，K均值聚类可以将数据集划分为k个簇，PCA可以将高维空间转换为低维空间。这些模型不需要标签，可以用于降维、数据降噪、数据分析等。
