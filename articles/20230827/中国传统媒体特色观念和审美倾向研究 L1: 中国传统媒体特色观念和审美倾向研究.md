
作者：禅与计算机程序设计艺术                    

# 1.简介
  

随着近代以来中国文化的蓬勃发展，形成了丰富而多样的民族传统文化，其中也有不少既有艺术价值又具有传统意义的经典作品。但是由于各地区不同文化特点的影响，以及长期坚持“一山一制”的政治思想，导致“文化落后、落伍、不立足”的问题依然困扰着中国。
如今，随着现代化、数字化、移动互联网等新兴技术的应用，中国各个角落都开始出现了新型的传播模式——“网络文化”，如微博、微信、Instagram等社交媒体、新闻报道等传播平台，越来越多的人群转向网络媒体接受新鲜、精彩的信息，这些网络媒体就产生了一系列新的特色性格特征。
对传统文化的这种多样性和特殊性的再现，将持续创造和塑造出新的价值观念和审美倾向，而且还会不断激发民众对于自己文化的热爱、追求和认同。因此，了解这些特色及其影响力，对推动中国传统文化的发展、促进社会公平正义有着重要的作用。
本研究计划对上述关键词进行专门研究，探究在中华文化特有的复杂社会语境下，网络文化从各种渠道发酵出来，形成的独特影响因素和特色功能。通过研究网络文化中的各种传播形态、个性化设计和历史沉淀，提升我们的洞察力和理解力，帮助我们更好地理解当前网络文化带来的变化。
# 2.研究方案
本研究工作由两部分组成。第一部分是基于数字化时代特点的“网络文化调查”：主要根据当前流行的一些网络文化热点网站和行业报告，分析网络文化发展情况、特征及其影响因素。通过收集信息和参与实地访谈，拨开迷雾，全面揭示网络文化形态、形式、功能、社会影响、文化资源、价值认同等多个方面的内容，进一步得出研究结论。第二部分则是“审美传统研究”：运用当代艺术学、计算机科学、心理学等新兴学科方法，对西方的艺术传统和古代人物形象、技艺等艺术元素进行深入剖析和研究。通过对中国传统文化的系统梳理，尤其是通过学习分析古代戏曲艺术、明清雕塑艺术、春秋战国绘画艺术等传统艺术，考察其对于当代文化的影响和贡献，进而验证或质疑其传承价值。
# 3.研究任务
本项目的研究范围涵盖以下方面：

1. 对中国传统文化的现状、发展、变迁及其影响因素；

2. 了解当前网络文化所呈现出的新闻文化特征、商业文化特征、娱乐文化特征、个人文化特征、社会文化特征；

3. 把握网络文化与传统文化之间的比较优势、相关性、互动关系，分析网络文化在传统文化的演变过程及其发展趋势；

4. 研判网络文化的发展方向、未来走向，掌握未来网络文化的发展趋势、发展机遇、制约因素和策略；

5. 提供可靠的证据、数据支撑，对未来的网络文化发展有明确、清晰的判断。

# 4.研究成果
研究成果包括：

1. 将上世纪五六十年代、六七十年代、八九十年代作为研究对象，研究中国传统文化从几何形态到观念内涵的演变；

2. 基于已有信息以及网络文化实践与传播实践的总结，提出中国传统文化形成的原因、条件和进程；

3. 结合中国传统文化经典故事与文化名人形象，评估古代中国人的文化特征，发现古代中国人具有的共性与特点；

4. 论证古代帝王、诗人、士子在传统文化创作上的作用；

5. 辅助“审美传统研究”中的艺术史研究方法，对古代文化与当代文化的艺术冲突、交融进行系统研究，标志着“审美传统研究”取得重大进展；

6. 在一定程度上支持“网络文化”这一定义，展望未来网络文化的发展趋势。
