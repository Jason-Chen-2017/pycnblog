# 1.简介
  

随着计算机技术的飞速发展，机器学习也在逐步成为热门话题。无论是图像识别、文本分类、垃圾邮件过滤等应用场景，还是自然语言处理、推荐系统、风险预测模型等领域都涉及到机器学习的算法开发与应用。

但是，由于机器学习相关技术知识与工具过于复杂，即使有一些基础的了解，面对新手入门者仍会望而却步。因此，很多朋友会问我：如果你对机器学习感兴趣，可以试试自学吗？对于我这样的初学者来说，自学一个机器学习框架或库这种事情本身就是一件困难且耗时的事情。但是，既然已经准备好了心里舒坦了，就想一探究竟吧！

接下来，我将从以下几个方面，向大家介绍一下如何在没有编程基础但对机器学习感兴趣的情况下，利用MOOC平台进行快速入门学习：
# （1）了解什么是MOOC
# MOOC（Massive Open Online Course，大型开放网络课程）是美国massiveOpenOnlineCourses.org（MOOCs提供商）推出的一个基于网络的教育学习模式。它是一个由全球性出版社联合组织发起，旨在为学生提供全新的、高度互动的网络教学资源。通过MOOC学习，学生能够获得真正意义上的“Massive”，即拥有“海量”的课程，能够随时随地学习任何有关IT技能、职业规划、个人发展的知识。MOOC课程一般会免费开放给注册用户，只要遵守相关规则即可获得学习权利。
# （2）选择适合你的MOOC
根据自己的兴趣、能力、需求，选择相应的MOOC课程。推荐几款我觉得值得尝试的MOOC：
- Data Science Specialization (Coursera) - 这个专项课程从最基础的Python语言、SQL数据库入手，带领读者完成数据分析的完整流程，包括数据获取、清洗、特征工程、可视化、建模等环节。如果你具备了一定的编程基础，建议可以先跟着这个课程做一遍。
- Machine Learning (Stanford) - 斯坦福大学的这门MOOC课程主要是从数学和统计的角度，向读者展示机器学习的基本知识和原理。如果你对这方面比较了解，建议可以尝试。
- Neural Networks and Deep Learning (deeplearning.ai) - 谷歌AI实验室的这门MOOC课程提供了深度学习的最新方法论。它从神经网络的基本概念、优化算法、激活函数、损失函数等等入手，并引导读者用TensorFlow实现实践。如果你对深度学习、神经网络比较熟悉，建议可以尝试。
# （3）注册并完成课程任务
注册成功后，就可以进入课程主页，开始完成课程任务。MOOC的任务通常分为三个阶段：作业、编程练习、考核。首先，你需要完成作业。每道作业都会有相关的学习材料，阅读材料后，回答相关的问题，然后把答案提交给课程的评判系统。评判系统会自动生成一份作业的成绩单，反映学生完成作业的效率和质量。

其次，你可以进行编程练习。编程练习一般是在讲授课程期间，课程老师会提供一个小项目，让学生自己完成。编程练习的目的是为了锻炼学生的编程能力，帮助学生理解所学到的知识和技能。

最后，你需要进行考核。考核一般在第四个阶段，即考核前三次作业和编程练习之后。考核的内容往往包括课堂实践题、编程实践题、最终考核项目。通过考核，可以确认学生是否掌握了相关的知识、技能和能力，并知道自己还有哪些地方需要进一步学习。
# （4）结尾
本文介绍了如何在没有编程基础但对机器学习感兴趣的情况下，利用MOOC平台进行快速入门学习。相信读完本文后，你会更加清楚，利用MOOC平台可以快速入门机器学习。
