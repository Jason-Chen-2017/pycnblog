# 1.简介
         
 Rust 是一种现代系统编程语言，其设计宗旨在安全、并发和性能之间寻找平衡点。它的优点主要有三方面：安全性、速度快、简洁高效、内存管理自动化。该语言具有跨平台能力，可以在 Windows、Unix 和 Linux 等不同操作系统上运行。本系列文章将从基础知识入手，带领读者了解 Rust 的编程模型、语法规则及常用特性，并逐步应用到实际项目中，提升开发者的编程技巧和解决问题的能力。
           本系列将采用模块化的方式，从最基础的编程模型开始，逐步深入到所有重要功能的实现，通过实践案例引导读者快速上手 Rust 语言。文章涉及的内容包括：Rust 开发环境搭建、Cargo 包管理器、变量声明、数据类型、表达式、流程控制、函数、注释、结构体、枚举、特征、模块化、面向对象编程、错误处理、并发编程、测试用例编写等核心技术。

         # 二、准备工作
        - 安装 Rust 工具链：官方安装方式：https://www.rust-lang.org/tools/install
        - 配置 IDE：选择集成开发环境（IDE）来进行 Rust 代码编写，推荐使用 VS Code 编辑器。
        - 创建 Rust 项目目录：新建一个文件夹作为 rust 项目的根目录，并进入项目目录，执行命令`cargo new project_name`，创建新的 cargo 项目，默认会生成 src 文件夹，里面有 Cargo.toml 文件。

        ```shell
        mkdir myproject && cd myproject
        cargo new hello_world --bin  
        ```
        
        执行以上命令会在当前目录下创建一个新项目`myproject`，其中包含了一个可编译为可执行文件的 crate `hello_world`。该项目目录结构如下：
        
        ```shell
       .
        ├── Cargo.lock
        ├── Cargo.toml
        └── src
            └── main.rs
        ```
