# 1.简介
         

数据安全（Data Security）是一个非常重要的且复杂的领域，它包括三个方面：

1. 信息安全
信息安全也就是对敏感信息的保护，如个人身份信息、交易信息、财务信息等；

2. 物理安全
物理安全又称为机密性、完整性和可用性，意味着确保被保护的数据不被非法使用、破坏、修改或窃取；

3. 生命安全
生命安全就是保障人类生命安全所需的安全措施。包括保障医疗卫生设备和设施的安全、应急救援体系的建设、人员的身体健康状况的监测以及职业病防治措施等。

随着社会的进步、经济的发展、新技术的出现、法律的制定和实践的推广，数据安全管理日益成为一个重要的研究课题。而数据安全管理的定义、概念、方法论和工具已经相当成熟，并且多种形式的工具和手段陆续出现。这些成果使得保护数据的安全和有效率成为可能，也促进了人们的关注。

# 2.背景介绍
数据安全是信息安全、物理安全、生命安全三者的综合体。当前，信息安全与物理安全的重叠越来越少，更多地集中在生命安全这个领域，但实际上物理安全也是非常重要的一环。为了实现数据安全管理，企业需要建立全面的信息化和网络化管理机制，制定严格的管理规章和规范，严肃认真地对待各种各样的数据，确保他们的安全、私密、完整和可用。数据安全管理可用于保护企业、组织、政府部门等不同级别的信息，包括公司内部的信息、客户信息、供应商信息等。数据安全管理可分为以下几个阶段：

1. 数据收集阶段：从各个渠道收集数据，包括人员信息、购物行为数据、企业资料、政府信息、政策法规等。

2. 数据管理阶段：对收集到的数据进行整理、处理和分析，获取有效的见解和结论，并形成信息化基础设施，确保数据的安全、有效、准确、完整、隐蔽。

3. 数据使用阶段：通过管理工具和流程访问、使用、存储和传播数据，确保数据安全使用和共享。

4. 数据存储阶段：将数据的存档备份到不同的位置，确保数据的安全和完整。

5. 数据流通阶段：根据业务需要，对数据进行加密、脱敏、传输、存储、使用和销毁，确保数据安全流动。

# 3.基本概念术语说明
## 3.1 数据安全管理概述
数据安全管理（DAM），即数据安全管理是在企业内部设置的管理机制，用于保障数据的安全和完整。其目标是保障对企业数据的长期保存、使用、共享和传递过程中发生的风险，包括恶意攻击、内部威胁、泄露、丢失、篡改等安全风险。数据安全管理是信息和网络安全的综合体，包含的信息安全、物理安全、生命安全以及数据安全管理三个方面。一般来说，数据安全管理可以分为四个层次：

1. 组织层面：组织数据安全管理是指企业对其数据进行分类和分类管理，确保信息的安全，并保障组织的数据资产的完整性、保密性、可用性和可控性。

2. 应用层面：应用数据安全管理是指依据应用场景、数据类型和用途，对应用系统的相关数据进行分类、标记、归档和保护，对数据的读、写、复制、变更、删除、审计等操作进行审计和控制，保障数据的安全、可用性和完整性。

3. 操作层面：操作数据安全管理是指在管理人员和工作岗位上加强数据安全意识，制定相应的数据安全策略、手册、检查清单和审核流程，并对数据的使用、管理、传输等活动进行持续跟踪和检测，确保数据安全运营的顺利进行。

4. 产品层面：产品数据安全管理是指基于产品开发过程中的需求和设计，对产品的功能、特性、性能和接口设计进行安全测试和验证，并确保产品的运行环境安全，提升产品的安全性能和稳定性。

## 3.2 数据分类
数据分类对于数据安全管理都是至关重要的。数据分类可以将各类数据划分成不同的安全级别，并将它们分别进行权限控制和保护。数据分类的方法有两种：按存储介质分类和按数据使用分类。按存储介质分类又称为静态数据分类，是指将各类数据按照它们被存储和处理的方式来区分，如磁盘、磁带、磁卡、软盘、光盘、USB Flash等；按数据使用分类又称为动态数据分类，是指根据数据使用的频率、存储时间、访问权限、隐私级别、数据价值和可能产生的影响等因素进行分类。

静态数据分类的方法有两种：标签式分类和元数据分类。标签式分类是指通过文件标签的方式来进行分类，例如采用文件名作为标签；元数据分类是指通过文件的元数据（比如文件创建日期、文件大小、文件创建者、访问次数等）的方式来进行分类。动态数据分类的方法有多种，如用户权限分类、日程表分类、敏感数据分类、秘密数据分类、数据挖掘分类、算法输出分类等。

## 3.3 信息安全管理
信息安全管理（ISM）是指管理信息系统的完整性、可用性、安全性、认证性、访问控制等方面，以确保信息系统满足相关的业务需求。信息安全管理的任务是防止信息泄露、恶意攻击、内部威胁、信息违规，保障信息系统的运行和数据的完整性，防止恶意用户对信息系统的攻击和滥用。其中，基础设施、网络、应用程序和数据都需要得到充足的保护。

信息安全管理包括信息安全基础设施、信息安全网络、信息安全协议、信息安全技术、信息安全审计、信息安全流程、信息安全事件响应、信息安全意识培训、信息安全协调、信息安全管理评估、信息安全管理工具和方法、信息安全绩效考核等。信息安全管理是一种专门针对信息安全的管理方式，它的核心目标是确保信息系统的可用性、安全性、以及业务需求的遵守。

## 3.4 物理安全管理
物理安全管理（PSM）是指保障数据在物理介质上的存储、传输、使用和密封等安全。在现代信息技术的背景下，物理安全管理已成为保障数据安全不可或缺的一环。为了确保数据的安全，企业要制定相应的物理安全管理制度，包括加密、数据脱敏、数据备份、网络隔离等。同时，还应当有相应的预案和手段来保障用户和数据的安全。

物理安全管理还包括计算机和互联网设备、电子器件、运输工具、通讯线路、照明设备、安全检查设备等。安全关键型设备、数据中心设备等具有较高的物理安全级别。

## 3.5 生命安全管理
生命安全管理（LCM）是指由国际公约和国家、地方、企业等的要求，针对人类生命安全的各项义务和职责，防范、检测、隔离、处置和消除人的健康和安全事故。LCM有助于企业具备生命安全意识，减少损害人类健康和安全的危险因素。目前，世界各国都在推行LCM制度，尤其是中国共产党领导下的一些国家将人口超过1亿的国家列入了LCM领域。

生命安全管理的范围非常广泛，涉及到的方面很多，包括人员的健康教育、病害防治、医疗救助、生产经营、交通运输、信息通信、金融投资、养老保障等方面。
