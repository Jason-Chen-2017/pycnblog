# 1.简介
         

while循环（英文：While Loop）是一种控制语句，在执行过程中会一直循环执行其中的代码块，直到指定的条件满足为止。一般用法是：

 while 表达式(条件):
     语句块
     …..

其中表达式是一个布尔值（True或False），也就是说当该表达式的值为True时，才会执行循环体内的代码块；而执行完整个代码块后，又会判断一次该表达式的值是否还为True，如果仍然为True，则继续执行循环体内的代码块；否则，就跳出循环。

由于while循环结构具有反复执行代码块的特点，因此被广泛应用于各种场景。例如，在程序中需要循环遍历某些数据集合的时候，就可以通过while循环来实现；在进行复杂的业务逻辑运算时，也可以采用while循环的方式进行迭代处理，这样既可以提高程序的运行效率，同时也更加灵活可控。

除了上面所介绍的用途之外，while循环还有其他一些特点，如以下几点：
- 可选择终止条件；
- 都可以用for循环代替；
- 可以嵌套多个while循环；
- 在循环体中不一定要写代码块，只需使用缩进表示当前行属于循环体即可；
- while循环可以作为选择结构，配合if/elif语句、switch语句等来进行条件分支。

因此，在实际应用中，应善于掌握while循环的各种特性，更好地运用它来解决各种问题。
# 2.基本概念和术语说明
## 2.1 表达式（Expression）
表达式是由运算符及其操作对象组成的表达式，用来计算并返回一个值。Python语言支持的表达式种类繁多，包括赋值表达式、逻辑表达式、算术表达式、比较表达式、位表达式、成员资格表达式等。

表达式的语法形式为：

 [标识符=]操作对象[运算符操作对象]*

## 2.2 赋值表达式（Assignment Expression）
赋值表达式用于给变量赋值，语法形式如下：

 var = value

其中var为变量名，value为表达式或常量。例如：

 a = b + c
 x = y * z / 7

## 2.3 逻辑表达式（Logical Expression）
逻辑表达式用于对关系表达式的值进行逻辑判断，得到一个布尔值结果。通常逻辑表达式都是由关系表达式组成的。

关系表达式用于比较两个对象的值，其语法形式如下：

 操作对象 比较运算符 操作对象

比较运算符有以下几种：

|符号 | 描述 |
|---|---|
|= | 判断是否相等 |
|!= | 判断是否不等 |
|< | 判断左边对象是否小于右边对象 |
|<= | 判断左边对象是否小于等于右边对象 |
|> | 判断左边对象是否大于右边对象 |
|>= | 判断左边对象是否大于等于右边对象 |

例如：

 a < b           # 判断a是否小于b
 a <= b          # 判断a是否小于等于b
 a > b           # 判断a是否大于b
 a >= b          # 判断a是否大于等于b
 name == "Alice" # 判断name是否等于"Alice"

## 2.4 算术表达式（Arithmetic Expressions）
算术表达式用于对操作对象进行数学运算，并得到一个数值结果。算术表达式一般用于数值的计算。

算术运算符有以下几种：

|符号 | 描述 |
|---|---|
|+ | 加法 |
|- | 减法 |
|\* | 乘法 |
|/ | 除法 |
|% | 求模 |
|** | 指数 |

例如：

 3 + 4   # 返回7
 3 - 4   # 返回-1
 3 * 4   # 返回12
 3 / 4   # 返回0.75
 3 % 4   # 返回3

## 2.5 比较表达式（Comparison Expression）
比较表达式用于比较两个对象的值，并得到一个布尔值结果。比较表达式一般用于条件判断。

比较表达式的语法形式如下：

 操作对象 比较运算符 操作对象

比较运算符有以下几种：

|符号 | 描述 |
|---|---|
|= | 判断是否相等 |
|!= | 判断是否不等 |
|< | 判断左边对象是否小于右边对象 |
|<= | 判断左边对象是否小于等于右边对象 |
|> | 判断左边对象是否大于右边对象 |
|>= | 判断左边对象是否大于等于右边对象 |

例如：

 age >= 18         # 判断age是否大于等于18
 score!= 90       # 判断score是否不等于90
 fruit in ["apple", "banana"]      # 判断fruit是否在["apple", "banana"]列表中

## 2.6 位表达式（Bitwise Expression）
位表达式用于对二进制数字进行运算，并得到一个整数结果。位表达式一般用于二进制数据处理。

位运算符有以下几种：

|符号 | 描述 |
|---|---|
|\^ | 按位异或 |
|~ | 按位取反 |
|& | 按位与 |
|\| | 按位或 |
|<< | 左移位运算 |
|>> | 右移位运算 |

例如：

 5 ^ 3            # 返回6 (5的二进制表示101，3的二进制表示011，按位异或结果是110)
 0b11 ^ 0b10      # 返回0b01 (0b11的二进制表示11，0b10的二进制表示10，按位异或结果是01)
  ~ 5              # 返回-6 (-5的二进制表示101，按位取反结果是1101)
  5 & 3            # 返回1 (5的二进制表示101，3的二进制表示011，按位与结果是011)
  5 \| 3           # 返回7 (5的二进制表示101，3的二进制表示011，按位或结果是111)
  5 << 2           # 返回20 (5的二进制表示101，左移两位，结果是10100，对应的十进制值为32)
  5 >> 2           # 返回1 (5的二进制表示101，右移两位，结果是001，对应的十进制值为1)

## 2.7 成员资格表达式（Membership Expression）
成员资格表达式用于判断某个元素是否存在于一个集合中，并得到一个布尔值结果。成员资格表达式一般用于集合运算。

成员资格表达式的语法形式如下：

 操作对象 是否在操作对象列表中

例如：

 'apple' in ['apple', 'banana']        # 判断"apple"是否存在于["apple", "banana"]列表中
 'orange' not in ['apple', 'banana']     # 判断"orange"是否不存在于["apple", "banana"]列表中

## 2.8 条件语句（Conditional Statement）
条件语句是根据特定条件的判断结果执行相应的代码块，决定程序的执行流程。

条件语句分为三种：
- if-else语句；
- switch语句；
- 三元表达式（ternary expression）。

### 2.8.1 if-else语句
if-else语句是最简单的条件语句，根据特定条件的判断结果执行不同的代码块。语法形式如下：

 if 表达式(条件):
     执行语句块1
 else:
     执行语句块2

其中表达式(条件)是布尔值表达式，只有表达式为真，才会执行语句块1，否则，就会执行语句块2。

例如：

 num = int(input("输入一个数字："))
 
 if num % 2 == 0:
     print(num, "是偶数")
 else:
     print(num, "是奇数")

### 2.8.2 switch语句
switch语句也叫做条件表达式，根据特定条件执行不同代码块。switch语句的语法形式如下：

 match expression:
     case value1:
         code block for value1
         break
     case value2:
         code block for value2
         break
    ...
     default:  # 默认分支，可以省略
         code block for other values
         break 

match expression 是表达式（变量或者表达式）；case value 是表达式的值，可以是多个值;break 用于结束当前匹配项。default 为默认分支，可选。

例如：

 grade = input("请输入你的成绩:")
 
 match grade:
     case 'A':
         print("奖励一个漂亮的成绩！")
     case 'B':
         print("奖励一个良好的成绩！")
     case 'C':
         print("你这次的成绩不错！")
     case 'D':
         print("不要太自满哦！")
     case _:
         print("你的成绩很差，再努力一下吧！")

### 2.8.3 三元表达式
三元表达式也称为条件表达式，它的语法形式如下：

 result_expression if condition else alternative_expression

三元表达式首先求出条件表达式condition的布尔值，然后根据这个布尔值决定是否执行result_expression，还是执行alternative_expression。

例如：

 x = 10
 str_x = "positive" if x > 0 else "negative"
 print(str_x)   # 输出："positive"
