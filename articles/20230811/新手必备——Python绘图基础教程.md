# 2.基本概念术语说明
1. Matplotlib：Matplotlib是一个基于NumPy数组的绘图库。它提供了创建各种图形的方法，包括折线图、散点图、条形图、饼图等。Matplotlib能够生成高质量的矢量图和矢量图表，并可以保存成不同的文件格式，比如PDF、EPS、SVG等。
2. Axes：Axes是matplotlib中用于绘制图表的坐标系对象，表示图形的上下左右四个边界，每个figure中可以创建多个Axes。
3. Figure：Figure 是 matplotlib 中用于存放 Axes 的容器。同一个 figure 可以包含多个 Axes，并且不同子图的 Axis 可共享 x 和 y 轴。
4. Line：Line 主要用于绘制折线图。
5. Bar：Bar 是一种特殊的折线图，主要用于表示各个分类间的差异。
6. Scatter：Scatter 用于绘制散点图，每个点由一个 x 和 y 值组成。
7. Pie：Pie 图主要用于显示不同分类的比例。
8. Histogram：Histogram 主要用于表示变量的分布。
9. Boxplot：Boxplot 是描述统计信息的一种图表。

## 1. 创建基础图形

下面，我们将展示如何使用matplotlib画出一个简单的散点图。 

```python
import matplotlib.pyplot as plt

# 数据准备
X = [1, 2, 3, 4, 5]
Y = [2, 3, 5, 7, 9]

# 创建一个散点图
plt.scatter(X, Y)
plt.show()
```

此处需要注意，当运行该代码块时，程序会打开一个窗口，并显示一条黑色的绿色实线。这条线就是刚才所画的散点图。其中，“plt”即指matplotlib.pyplot库，而“scatter”函数则是在该库中用于绘制散点图的函数。该函数接受两个参数：X和Y，分别对应于横轴和纵轴的值。
