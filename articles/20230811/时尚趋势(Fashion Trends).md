# 2. 概念、术语及定义
## 2.1 时尚定义
**时尚（Fashion）**，是指面向消费者的服装和包包等穿着的样式、用品、配饰、化妆品及其他物品的设计和创作。其主流概念或说法之一就是指产品或者产业的时尚化和风格化。
## 2.2 时尚品牌定义
**时尚品牌（Fashion Brand）**，即具有明显的时尚特征的商标、名称或品牌。其主要包括有时髦、流行、潮流、时尚、休闲、古典、文化、韩国、日韩等十几种类型。
## 2.3 时尚业内的关键词
- **时尚热:** 关注时尚热点，了解热门时尚品牌新动态；
- **时尚展：** 参加时尚展览活动、参与时尚焕然一新的尝试；
- **时尚节：** 逛好莱坞影院、参与潮牌的时尚游、和小伙伴们一起追逐时尚潮流；
- **时尚月：** 享受到时尚美食、买到独特的时尚礼盒；
- **时尚咖啡厅：** 品尝到全球知名品牌的最新时尚咖啡；
- **时尚三件套：** 时尚风、韩式流行、时尚包装；
- **时尚男士连衣裙：** 时尚服饰加上休闲裤和高领毛衣；
- **时尚女神甜心杯：** 用甜蜜提升彼此情感，分享个人情绪的时尚饮品。
## 2.4 时尚分类标准
时尚在各个方面都呈现出广泛而细致的变化。目前，时尚业界共分为五大类：
- **休闲:** 关注价格低廉的运动服、短款外套、轻薄上衣、牛仔裤、棉麻袜等身材瘦长、比例合适但精湛的运动衫、莫代尔、花花公子、爱马仕、耐克、阿迪达斯等服饰品牌；
- **运动:** 以跑鞋、运动鞋、拖鞋、篮球鞋、休闲球鞋、板鞋、速干衬衫、滑雪服装、泳衣、冲锋衣、滑雪裤、冲浪衣等为代表的服装；
- **男装:** 以T恤、衬衫、套装、卫衣、西装、皮草、夹克、短裤、羽绒服、吊带、风衣、皮鞋、帽子等为代表的男装品牌；
- **女装:** 以针织衫、毛衣、卫衣、裙子、短裤、连衣裙、蕾丝衫、美腿袜、珠宝配饰、包包等为代表的女装品牌；
- **内衣:** 以牛仔褲、牛仔外套、打底裤、棉袄、棉服、羽绒服、雨伞、帽子、手套、围巾、手表、钱包、腰带等为代表的服装。
## 2.5 时尚品味的特点
- **功能性:** 醒目、时尚化的线条和创意;
- **服饰质感:** 柔软、蓬勃、舒适、华丽、温暖、奢华、惹火、刺激、尊贵、传奇、耀眼的颜色;
- **年龄层次:** 年轻气盛、职场风、老年快感、时尚焕发、任性、热衷、高雅;
- **包装形式:** 有特色、单一、整体、优雅、魅力;
- **时尚内涵:** 抓住时尚本质、展示自我、享受生活、做自己;
- **情调:** 精致、俏皮、诙谐、大气、亲切、浪漫、智慧、隐喻、节制。
