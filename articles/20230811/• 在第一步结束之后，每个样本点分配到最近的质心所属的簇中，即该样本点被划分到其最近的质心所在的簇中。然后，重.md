# 1.简介
         

K-means算法是一个经典的聚类算法。它可以用来进行无监督学习。在这个算法里，数据点可以分成任意数量的类别（簇），每一个类别里面都有一个中心（质心）。这个过程会不断重复，直到所有的类别都收敛到全局最优解，或者达到指定的最大迭代次数。因此，这个算法也叫做无监督学习方法，因为它不需要知道数据的标签信息。但如果知道了标签信息，那么就可以应用带标签的聚类算法，比如LDA、QDA等。

算法的第一步是随机选择k个质心，并将每个样本点分配到离它最近的质心所在的簇中。第二步是根据簇中的样本点重新计算每个簇的质心。第三步重复以上两个步骤，直到所有类的质心都不再发生变化或达到指定的最大迭代次数，算法结束。

2.算法相关术语及概念
# 1.样本点(Sample Point):每个数据集里面的一个向量，通常用x表示。

# 2.簇(Cluster):由相同特征的样本组成的集合，通常用μ表示。

# 3.质心(Centroid):簇中心，一般用μ表示。

# 4.距离(Distance):样本之间的距离，通常用||x - μ||表示。

# 5.初始值(Initial Value):随机初始化k个质心的方法，如随机选取、轮盘赌法等。

# 6.最大迭代次数(Max Iteration):算法最多执行多少次，若每次迭代结果不变，则称算法收敛。

# 7.自身相似度矩阵(Self Similarity Matrix):由距离函数计算得到的样本之间的距离矩阵，由k个质心作为行列，表示样本距离其最近质心的距离。

# 8.簇类别标识矩阵(Cluster Labeling Matrix):由k个质心组成的矩阵，记录每个样本所属的簇编号。

# 9.精英采样(Elite Sampling):选择部分最好的质心参加下一次迭代更新，使算法更加鲁棒，防止陷入局部最小值。
