# 1.简介
         

支持向量机（Support Vector Machine，SVM）是一种二类分类、线性可分的模型，它在机器学习领域中经过了多年的发展。而随着时代的变迁，支持向量机也变得越来越流行。尽管 SVM 在许多领域都获得了成功，但在机器学习界却并非孤独存在。它的身影总会被各种机器学习方法所牵绊，影响着它们的发展方向，因此值得我们探讨一下支持向量机到底扮演着什么样的角色。本文将围绕 SVM 的基础知识、发展历史、应用场景、研究成果等方面对其进行详细阐述。

## 2.2 为什么要用支持向量机？
1. 解决线性不可分的问题
支持向量机在处理线性不可分问题时仍然有效。例如，当样本点分布在多个半空间时，就可能出现不可分的情况。但是，如果输入空间被划分为足够多的子空间，那么任意两个子空间之间就可以完美地分离，此时就可以考虑使用支持向量机。

2. 使用核技巧
核技巧是支持向量机的一个重要优势。它利用核函数将输入空间映射到高维空间，从而在低维空间实现线性可分。一般来说，核函数是指对某个函数做变换，使其成为另一个函数，这样就可以解决线性不可分的问题。SVM 通过核技巧实现了输入空间和输出空间的非线性转换，从而解决了线性不可分的问题。目前常用的核函数有多项式核、径向基函数核、字符串核等。

3. 有利于模型选择
支持向量机提供了一个简单而有效的模型选择方法——正则化项。通过调节正则化系数，可以得到不同的模型。在正则化项比较大的情况下，SVM 会倾向于选取能够将样本点分开的最简单的模型；在正则化项比较小的情况下，SVM 会倾向于拟合训练数据的最佳模型。

4. 对异常值不敏感
支持向量机对异常值的敏感程度很低。这主要是因为异常值往往处在决策边界上的位置，而支持向量机只关心决策边界周围的数据点，所以对于异常值不产生影响。

5. 速度快
支持向量机使用核技巧后，可以在 O(n^2) 的时间复杂度内求解，速度较快。而且，当数据集较大时，由于采用了核函数的计算方式，SVM 不仅可以处理非线性数据，还可以处理高维空间中的数据。

## 2.3 支持向量机的应用场景
SVM 作为一种二类分类器，通常用于如下三个领域：

1. 图像识别和模式识别：SVM 可用于图片分类、物体检测等领域，如手写数字识别、文字识别、眼疾病诊断等。

2. 文本分类：SVM 可用于文本分类，如垃圾邮件过滤、新闻分类等。

3. 生物信息学：SVM 可用于进行基因序列分析、蛋白质序列分析、肿瘤分类等。
