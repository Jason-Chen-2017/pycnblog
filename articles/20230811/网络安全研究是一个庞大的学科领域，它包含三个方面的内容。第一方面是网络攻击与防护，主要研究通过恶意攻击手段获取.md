# 1.简介
         

网络安全（Cyber Security）是指计算机系统及其周边环境存在的安全威胁或漏洞，由计算机病毒、黑客攻击、网路攻击、电子通信线路中的故障、电信工程中的事故、社会工程攻击、内部人员非法获取数据等多种因素引起的对计算机系统的侵入、破坏、篡改、泄露、伪造、秘密窃取活动等所带来的危害。
网络安全的研究也是越来越多的学者涉足这个领域，由于国内外网络安全学科的蓬勃发展，相关专业领域也在不断扩大。网络安全研究涵盖的内容涉及广泛，如网络安全规划、网络设备安全、网络安全运营管理、网络攻击防御、网络安全评估、网络安全法律法规与标准化、网络安全技术理论、网络安全工具开发与应用、网络安全管理与防范、网络安全实践和分析、信息安全技术新型发展方向、网络安全产业化等等。
随着网络安全研究的进步和普及，研究人员还在继续寻找新的技术突破点和解决方案。网络安全已经成为当今社会的一个重要、紧迫的热点话题，也是一项复杂而又持续的科研任务。正因为如此，越来越多的高校和研究机构开始着力于网络安全研究的深入，探索全新的网络安全技术，以期实现网络安全科技的自主可控。
# 2.基本概念术语说明
## 2.1 概念定义
网络安全（Cyber Security）：是指计算机系统及其周边环境存在的安全威胁或漏洞，由计算机病毒、黑客攻击、网路攻击、电子通信线路中的故障、电信工程中的事故、社会工程攻击、内部人员非法获取数据等多种因素引起的对计算机系统的侵入、破坏、篡改、泄露、伪造、秘密窃取活动等所带来的危害。
## 2.2 名词解释
### 2.2.1 计算机病毒（Malware）
计算机病毒(malware)是一种具有侵略性、破坏性的计算机程序，具有控制用户系统、散播流氓软件、盗取个人信息、收集和发送恶意邮件、进入互联网后门等功能。主要分为三类：僵尸网络蠕虫病毒（Trojan），垃圾邮件病毒（Virus），特洛伊木马（Worm）。
### 2.2.2 网络攻击（Network Attack）
网络攻击（network attack）又称网络犯罪，是指利用计算机技术、电脑病毒或其他恶意程序进行的、企图入侵某台计算机系统、组织、企业或者组织网络的犯罪行为。网络攻击行为通常有三种类型：分布式攻击、基于策略的攻击、基于规模的攻击。分布式攻击：对单个目标网络进行暴力攻击，例如DDOS、SYN Flood、BGP劫持等；基于策略的攻击：通过对网络资源和服务进行故意设计的攻击方式，例如DNS欺骗、FTP Bounce、Socket缓存溢出等；基于规模的攻击：对整个网络进行拒绝服务攻击、对关键业务资源进行僵尸网络攻击等。
### 2.2.3 数据包嗅探器（Packet Sniffer）
数据包嗅探器是一种特殊的软件应用程序，它能够抓取数据报文并存储到文件、屏幕或网络接口卡上。最初用于电话线路交换机的数据包嗅探工作，后来逐渐扩展到局域网中。数据包嗅探可以帮助网络管理员捕获网络流量，同时也可以被用来恶意地获取信息，并可能导致数据泄露、身份盗用或网络攻击等严重后果。
### 2.2.4 IPSec协议（Internet Protocol Security）
IPSec协议是用于创建虚拟专用网络（VPN）和加密通道的安全技术。它建立在Internet Protocol (IP)之上，支持各种认证机制、加密算法、身份验证、防火墙、NAT（网络地址转换）和QoS（Quality of Service）等功能，可以为用户提供可靠且安全的通信传输能力。
### 2.2.5 VPN技术（Virtual Private Network）
VPN（Virtual Private Network）即虚拟专用网络，是指在公共互联网基础上搭建的专用网络，目的是将用户从公网上切割出来，使得用户只能访问专用网络上的资源。VPN可帮助组织满足信息隐私、网络隔离、网络流量控制、公司网络访问控制、远程办公、移动办公、安全监控等需求。
### 2.2.6 DNS协议（Domain Name System）
DNS（Domain Name System，域名系统）是因特网的一项服务，它负责把主机名转换成相应的IP地址，因而能够让用户更方便地访问互联网资源。域名系统运行于TCP/IP协议之上，使用UDP协议端口53。域名系统按照一定的层次结构存储了网站域名与IP地址之间的映射关系。
### 2.2.7 HTTP协议（HyperText Transfer Protocol）
HTTP（HyperText Transfer Protocol，超文本传输协议）是一种用于分布式、协作式和超媒体信息系统的请求/响应协议，属于应用层协议，它使客户端（Web浏览器或其他智能终端）能够向服务器（如：WWW服务器）提出请求、接受并处理 responses，并标识出使用的Web文档格式（如：HTML、XML、JSON等）。
### 2.2.8 SSL/TLS协议（Secure Sockets Layer/Transport Layer Security）
SSL（Secure Sockets Layer，安全套接层）和TLS（Transport Layer Security，传输层安全）是用于创建加密通道的安全技术。SSL提供了对网络连接的加密，确保数据在传输过程中不会遭受窃听、篡改或差错。TLS是SSL的继承者，它建立在SSLv3.0版本之上，添加了更多的安全性增强功能。
### 2.2.9 DHCP协议（Dynamic Host Configuration Protocol）
DHCP（Dynamic Host Configuration Protocol，动态主机配置协议）是局域网的网络通信协议，使用UDP协议端口67和68。该协议允许自动配置本地计算机的IP地址、子网掩码、默认网关等参数，从而简化网络管理工作。
### 2.2.10 ARP协议（Address Resolution Protocol）
ARP（Address Resolution Protocol，地址解析协议）是一种局域网的网络通信协议，使用ARP协议可以根据IP地址查询对应的MAC地址。
### 2.2.11 IPS协议（Intrusion Prevention System）
IPS（Intrusion Prevention System，入侵预防系统）是一种网络入侵检测系统，能实时监控网络活动，发现异常的网络行为并向网络管理员报警。
### 2.2.12 IDS协议（Intrusion Detection System）
IDS（Intrusion Detection System，入侵检测系统）是一种网络入侵检测系统，能够识别各种攻击行为，如对网络资源的非授权访问、端口扫描、后门程序的入侵、DoS攻击、漏洞利用等。
### 2.2.13 NTP协议（Network Time Protocol）
NTP（Network Time Protocol，网络时间协议）是用于同步不同系统时间的分布式网络协议。
### 2.2.14 SMB协议（Server Message Block）
SMB（Server Message Block，服务器消息块）是Microsoft Windows系统的网络文件共享协议，它支持Windows网络硬盘、打印机、串口和外部硬盘等资源共享。
## 2.3 定义注解
# 英文全称：Computer security or cybersecurity is a set of principles and practices that protect computer systems from unauthorized access, abuse, interference, or disruption. It involves managing the risk associated with system vulnerabilities, preventing malicious activities by detecting attacks and responding to them, and maintaining safeguards against potential threats such as viruses and worms.
# 中文翻译：网络安全（英语：cybersecurity）是指保护计算机系统免受未经授权访问、滥用、干扰或中断所产生的风险的原则和做法。它涉及管理系统弱点带来的风险，通过识别攻击并采取相应反应阻止恶意活动，并维护针对潜在威胁（如病毒和蠕虫）的有效防御措施。
