# 3. CV模块原理及特点 
　　Computer vision (CV) 是人工智能的一个重要分支，负责从图像、视频或其他信息中识别目标、区域、场景、对象等。它包括视觉处理、模式识别、机器学习、图像处理和分析等多个子领域。以下是CV领域的一些基本术语和概念。  

　　• Image：一幅数字化的照片、照片组或者视频中的一帧图像。 

　　• Feature：图像上某个局部区域，通常是矩形区域或圆形区域。 

　　• Template matching：模板匹配，是一种用来找寻图像或视频中的特定图像特征的方法。 


　　CV的目标是从图像、视频、或其他信息中提取有用的信息，以辅助人工智能系统进行决策。例如，在医疗行业中，能够从X光图像中检测患者的肿瘤；在生物信息学领域，利用图像识别基因，判断动植物是否健康；在智能驾驶领域，通过摄像头获取车牌识别牌照；在图像分类领域，可以根据图像识别其类别。 


　　目前，CV主要分为三大领域：图像处理、视频分析与跟踪、多模态技术。 

　　图像处理：包括计算机视觉、图形处理、图像检索与分析、增强现实与虚拟现实等。 

　　视频分析与跟踪：包括运动跟踪、多目标跟踪、人脸检测与识别、视频生成与编辑等。 

　　多模态技术：包括跨模态、融合、无监督、弱监督学习与聚类等。 

　　CV的应用一般分为两大类，一类是图像识别，如目标检测、图像分类、人脸识别、车牌识别等；另一类是视频分析与理解，如人体跟踪、运动跟踪、事件驱动分析等。 

　　由于CV的复杂性和广泛性，它的研究也逐渐成为热门话题。深度学习、生成模型、强化学习、混合方法、脑机接口等都扮演着越来越重要的角色。 

　　本节只会讨论其中的一个分支——图像处理。 

# 3.1 图像处理 

　　图像处理是指从图像、视频或其他信息中提取有用信息的过程。常见的图像处理任务有目标检测、图像配准、图像去雾、图像增强、图像压缩、图像分割、图像恢复等。 



　　目标检测：检测图像中的目标，如人脸、行人、车、狗、果冻等。 

　　图像配准：将一组相机拍摄的图像自动对齐、重叠。 

　　图像去雾：移除图像中的雾霾、湖泊、云层等。 

　　图像增强：增加图像的质量，如旋转、缩放、锐化、饱和度调整等。 

　　图像压缩：减少图像大小以降低存储成本。 

　　图像分割：将图像划分成多个子区域，如背景、人、建筑、道路等。 

　　图像恢复：通过已知图像中缺失的信息，补全原图像。 

　　图像处理的关键是特征提取与描述，这涉及到计算机视觉、机器学习、信号处理等多学科。深度学习方法得到了广泛关注。
