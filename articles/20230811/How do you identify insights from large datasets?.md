# 2. 概念术语说明
## 2.1 数据预处理

数据预处理（Data Preprocessing）是指对原始数据进行初步清洗、转换，使之符合分析目的和整合后数据的质量。主要包括：

1. 删除异常值：删除数据集中的异常值或极端值，这些值可能影响模型的精确性和准确性。

2. 数据转换：将非数值型数据转换为数值型数据，例如将分类变量转换为标签编码。

3. 数据标准化：对数据进行尺度变换，以期望得到具有均值为零、方差为单位方差的分布，从而消除量纲影响。

4. 数据拆分：将原始数据分割成多个子集，用于训练、验证和测试模型。

5. 属性组合：构造新的特征，通过组合已有属性的方式来增加数据集的多样性。

## 2.2 特征工程

特征工程（Feature Engineering）是指基于原始数据构造特征，旨在增加模型的预测能力，从而取得更好的效果。主要包括：

1. 离散特征：将连续变量离散化为不同的离散程度的变量，如将年龄分为青少年、中年、老年等；

2. 交叉特征：通过组合多个变量的值来创建新的变量，如商品ID与颜色组合为“商品ID+颜色”；

3. 统计特征：计算变量的概率密度函数、协方差矩阵等，如计算年龄的均值、方差、偏度和峰度；

4. 文本特征：利用文本数据提取关键词、主题等，如将商品描述转换为关键词列表。

5. 时间特征：将时间间隔转换为连续变量，如将距离今天的时间差转换为秒、分钟、小时等；

6. 空间特征：将坐标转换为几何形状，如将地理位置转换为距离中心点的距离等。

## 2.3 数据降维

数据降维（Dimensionality Reduction）是指通过某种方式来降低特征的个数，从而减少模型的复杂度。主要包括：

1. PCA（Principal Component Analysis）：将高维数据转化为低维数据，找到主成分方向和方差最大的方向，剔除余弦接近于零的方向。

2. LDA（Linear Discriminant Analysis）：找到所有类别的最大线性无关方向，将数据投影到这个方向上。

3. t-SNE（t-Distributed Stochastic Neighbor Embedding）：利用高斯分布假设，通过优化目标函数，将高维数据映射到二维或者三维空间。

## 2.4 聚类分析

聚类分析（Clustering Analysis）是指根据数据之间的相似性将相似的对象归类到同一类，实现降维和分类的过程。主要包括：

1. k-Means 聚类：采用迭代的方法，首先随机选择 k 个质心，然后将各个数据点分配到最近的质心所在的簇中。反复迭代，直到簇不再变化。

2. DBSCAN 聚类：该方法通过确定核心对象、噪声点和边界对象三个概念，通过发现对象之间的联系来进行聚类。

3. Hierarchical clustering 层次聚类：首先按照一定的评估准则将数据集合划分为多个簇，然后合并相邻簇，直至所有的点都属于同一个簇。

## 2.5 模型选择

模型选择（Model Selection）是指选择最适合当前任务的模型，通常采用多种模型组合来比较各模型的性能。主要包括：

1. 交叉验证法：通过将数据集划分成多个子集，分别训练和测试模型，得到多个模型的训练误差和验证误差，选择其中误差最小的模型作为最终模型。

2. ROC 曲线法：将模型输出的概率值绘制成 ROC 曲线，计算 AUC 值，选择 AUC 值最高的模型作为最终模型。

3. Grid Search 网格搜索法：采用多种参数配置，遍历各种可能的模型参数组合，选取最佳参数组合。

## 2.6 可视化分析

可视化分析（Visualization）是指通过将数据呈现为图像、表格或者图表的形式，帮助用户更直观地看出数据内在的规律。主要包括：

1. 数据分布图：展示数据集的样本分布，以帮助识别异常值。

2. 相关性图：展示两组或多组变量之间的相关性，以帮助识别出线性相关的变量。

3. 聚类图：展示数据集的聚类结果，以帮助识别出共同的模式。

4. 主成分分析：将高维数据转换为低维数据，以便对数据的结构进行可视化。

5. 决策树可视化：展示决策树模型的生成流程，以帮助用户理解模型。

## 2.7 其他方法

除了以上六大领域，还有一些其他的方法可以用来分析大型数据集，如：

1. 时序分析：处理和分析动态系统中产生的数据，如股市价格、地震、气候变化等。

2. 关联规则：发现模式和频繁项，帮助用户发现数据中的关联规则。

3. 推荐系统：根据用户行为和兴趣对商品进行排序，帮助用户找到感兴趣的内容。

4. 因果推断：检测数据中的因果关系，通过构建模型判断各个变量之间的因果依赖。
