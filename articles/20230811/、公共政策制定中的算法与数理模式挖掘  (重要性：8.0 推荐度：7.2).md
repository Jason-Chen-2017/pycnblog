# 1.简介
         

## 1.1 研究背景
随着公共政策领域的蓬勃发展，各级政府部门不断面临着如何准确有效地制定政策、促进社会经济发展等重大决策问题，如何实现公共政策的目标，已经成为制定者们在进行公共政策研究时的首要考虑。
当前，公共政策制定过程中存在许多复杂的技术问题，如：“政策冰山一角”现象频发、因制定公共政策而带来的社会、经济发展波动等。在此背景下，如何通过对公共政策制定的实践经验、技术工具的应用及其创新方法等方面进行科研探索，从中提取有效、高效、可行的公共政策制定方案、解决问题，是公共政olicy制定领域的迫切需求。

基于上述研究背景，我国法律研究机构特别会议组成“公共政策数理模式挖掘专委会”，成立于2019年8月，专门开展相关的科研工作。本委会成立的初衷是为了加强本行业的研究水平，培育新的理论基础、新方法，推动公共政策的科学化、规范化进程，推进公共政策制定过程的透明化、公正化。同时也希望借助自身资源，结合专业的法律、经济、政治、管理知识体系，共同推进公共政策制定前沿理论与方法的研究，进一步完善现有的公共政策制定框架与机制，为中国公共政策制定领域提供更具影响力的理论支撑。 

因此，我国法律研究机构特别会议组成“公共政策数理模式挖掘专委会”。专委会围绕公共政策制定中的算法与数理模式挖掘这一主题，承担着以下工作：
- （一）研究新兴公共政策制定方式，提升公共政策制定能力；
- （二）针对已有公共政策制定模式、制度设计，提出面向算法与数理模式挖掘的新型模式探索方法；
- （三）利用计算机系统、自动学习、模糊计算、统计分析等新型技术手段，提取并挖掘公共政策制定中的模式和规律；
- （四）整合专业的法律、经济、政治、管理背景知识，探索具有全球性意义的数理模式；
- （五）向社会公布并推广最新的公共政策制定模式、模式预测模型，促进公共政策制定科学化、规范化。

## 1.2 主要研究方向
“公共政策数理模式挖掘专委会”致力于公共政策制定过程中的数理模式的挖掘与分析，即如何从数据中发现并有效利用所蕴含的结构特征、演化规律、优化机制、制度设计理念、行为经济学原理、制约因素等，通过机器学习、数据挖掘的方法，对公共政策制定中潜藏的模式进行理解和分析，以期使公共政策制定具有更好的效率、公平性、制度化、可持续性。

1.模式识别：这是模式挖掘的一个重要组成部分，通过对已有的数据进行分析、处理、提炼等操作，能够发现所采集数据的内在规律，形成可用于制定公共政策的模式，提升公共政POLICY制定质量。

2.模糊计算：在现实生活中，各种复杂的现象多种多样且难以预测，而模糊计算理论认为，当系统变量多维度联动时，可以根据其他变量的不确定性对结果进行概率性描述，从而发现并揭示这些变量之间的相互作用。通过对公共政策制定过程中产生的数据进行模糊计算，可以对政策制定实施有更精确的把握。

3.数据驱动政策制定：公共政策制定需要充分运用公民的意志、经验、知识、信息等不同渠道的信息资源，如何将公众的信息输入到公共政策制定过程，以及如何用公众的信息呈现公共政策制定的方案、制度安排，以及如何用制定好的公共政策来推动经济社会发展，就是“数据驱动政策制定”的一个重要课题。通过对制定公共政策过程中获取的数据进行分析、处理，并通过建立起新型的公共政策制定模型、决策算法等，提供可信、准确、高效的制定建议，成为我们在公共政策制定中的关键任务之一。

4.优化与奖惩机制：制定公共政策是一个复杂的过程，如何根据现实世界中人类社会运行过程和外部信息的变化对政策制定的方式、措施进行优化，也是公共政策制定不可或缺的一部分。如何在保证公共政策制定的公平性、合理性、效率的同时，满足公众的诉求和期望，是优化与奖惩机制的关键。本专委会将对现有公共政策制定的机制、流程、规范等进行改革，研究如何通过制度安排、资源配置等方式，推动公共政策的优化、奖惩机制，以达到制定公共政策的“以人为本”、“社会化公共服务”的目标。

5.经济社会学：公共政策制定有利于促进经济发展，但如何正确评价经济社会学原理、应用数理模式进行经济社会的决策与制度设计，还是一项复杂的任务。本专委会将开展经济社会学领域的研究，探索如何通过数理模式挖掘来优化经济社会发展，并建立公共政策制定的原则、机制、指标体系，推动经济社会发展与公共政策制定共赢。

# 2.基本概念术语说明
## 2.1 数字经济
数字经济是指利用信息技术及网络技术，进行经济活动的一种新的模式。它将实体经济与信息经济有机地联系在一起，利用信息技术改变了实体经济的发展方式、组织结构、价值链，进而影响了实体经济的格局与结构。数字经济是指通过网络、云计算、物联网等方式，将人类生产的实体产业通过计算机网络、移动终端、传感器、无线通讯等形式转换为信息产业，并通过大数据、云计算、人工智能、物联网等新技术，提升产品品质、降低成本、节省时间、提升效率、降低环境污染、优化资源利用率等综合性能。数字经济正在从实体经济转变为新的经济增长方式。
