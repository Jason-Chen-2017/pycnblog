# 2.搭建过程
## （一）准备工作
首先，确定自己的博客域名。什么样的域名最适合您的博客呢？您可根据以下几个方面考虑：

1. **长度**：域名的长度决定了网站URL的长度，也影响搜索引擎对您的网站的抓取频率；建议长度不超过20个字符。

2. **层级结构**：域名的层级结构决定了网站的访问速度，也会影响网站权威性；建议域名层级不要太复杂，层次不要太多，比如：yourname.com.cn 或 www.blog.yourcompany.cn 。

3. **独立性**：域名独立性是指该域名是否能独占。独立域名通常不会被其他域名收录，因此可以起到很好的SEO效果。如www.yourcompany.cn 就属于独立域名。

4. **价格优势**：域名的价格越便宜，可以获得更多的曝光量，也可能有更多的消费者关注。但一般情况下，个人博客域名不会贵到足以支撑大型网站。

5. **品牌知名度**：品牌知名度比较大的域名可能会获得更多的竞争，也有助于提升公司形象。比如：www.google.com 。

另外，建议自己做好网站的风格设计、制作海报等准备工作，可以在后期做维护的同时还能给自己带来美感。

## （二）购买空间
购买空间是一个比较复杂的事情，因为要选择服务器、域名解析、邮箱设置、FTP上传设置等。这里以腾讯云存储为例，教大家如何一步步地把自己的博客部署上去。
### （1）注册账号
首先，我们需要注册腾讯云的开发者账户（如果没有的话）。
注册完之后点击“产品与服务”，然后点击“对象存储OSS”。
### （2）新建存储桶
点击左侧导航栏中的“存储桶”，然后点击右上角的“创建存储桶”按钮。
填写表单信息，包括“区域”、“访问类型”、“存储类别”、“存储容量”等。例如，我选择了“华南地区”，“公共读私有写”，“静态站点”，“标准存储”，“低频访问”的存储配置。
### （3）绑定域名
如果你没有域名，那么你可以跳过这一步。但是如果你有域名，那么请务必记住你的域名！
进入到存储桶首页，点击“域名管理”，然后点击“添加域名”。
填写域名信息即可完成域名解析。
解析成功之后，打开域名的DNS设置页面，找到域名的NS服务器，然后修改成腾讯云的名称服务器NS记录。
### （4）开启Web服务
存储桶建立完成后，我们需要开启Web服务，这样才能将我们的网站内容发布到Internet上。
点击“静态网站托管”后的“开启”。
### （5）配置FTP
配置FTP用来实现在本地电脑上传文件到OSS中。
找到控制台中的“访问密钥”标签页，复制对应的密钥。
打开客户端工具，输入腾讯云的账号密码，连接服务器IP地址和端口号。然后将刚才的密钥粘贴到“新建链接”界面。
连接成功后，就可以进行文件的上传、下载、删除等操作了。
## （三）配置GitHub Pages
GitHub Pages是一个非常方便的静态博客生成工具。如果你想搭建自己的专属博客，那么推荐使用GitHub Pages。
### （1）创建仓库
登录GitHub并创建一个新的仓库。仓库的名字必须为：username.github.io ，其中username代表你的GitHub用户名。点击下面的“Create repository”按钮。
### （2）关联仓库
在新仓库的Settings中找到“Pages”选项卡，然后选择“Source”为“master branch / docs folder”，再点击“Save”按钮。
### （3）克隆仓库
克隆刚才创建的仓库到本地电脑。
```
git clone https://github.com/{username}/{username}.github.io.git
```
### （4）安装博客模板
把博客模板放到克隆的仓库的根目录，然后执行命令：
```
git add. && git commit -m "first commit" && git push origin master
```
此时，GitHub Pages就会自动生成你的博客。
### （5）配置本地环境
打开克隆出来的仓库文件夹，编辑器打开“config.yml”文件。
更改title、name、description和baseurl字段值，然后保存文件。
接下来，运行如下命令：
```
bundle install
jekyll serve
```
浏览器打开http://localhost:4000查看博客效果。
### （6）编写博客文章
编写博客文章可以使用Markdown语法。编辑器推荐Typora，当然，也可以使用Vim或VSCode。
Markdown的语法教程很多，这里只介绍一些简单的知识。比如：
- 标题：在行首用#表示，分别对应1~6阶标题。
- 段落：两行之间空两个格符或回车键。
- 引用：在每一行开头加>。
- 粗体与斜体：两个星号或下划线包围的文字为粗体，一个星号或下划线包围的文字为斜体。
- 分割线：三个星号或下划线。
- 插入图片：![alt-text](url)。
- 插入链接：[text](url)，或者<url>。
### （7）发布博客
当完成博客编写之后，只需提交到仓库即可。然后访问{username}.github.io，就可以看到你的博客了。
至此，我们已经搭建了一个属于自己的专属博客，并且已经可以通过互联网访问到了。
