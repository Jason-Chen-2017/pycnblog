# 1.简介
         


随着互联网、移动互联网、物联网、生物技术、虚拟现实等新兴技术的不断发展，人工智能的研究也在不断深入。随着这些技术的发展，深度学习与强化学习已经成为越来越重要的研究方向。而在这个快速发展的时代背景下，基于Python语言的机器学习框架已经逐渐成为热门话题。所以，如何进行深度学习、强化学习、特征工程等应用，以及如何在实际项目中运用这些技术，都是许多人关心的问题。

Python机器学习系列教程将从基础知识、算法原理、代码实现、案例分析三个方面详细阐述Python机器学习的相关知识。文章力求把Python作为一个优秀的机器学习开发语言，结合数据科学、统计学、数值计算等多个领域的知识，让读者掌握Python机器学习的理论与实践技巧。本文由陈小鸣（台湾大学机器学习研究院讲席教授）撰写，审校过于潜望期间欢迎提出宝贵意见，共同打造更好用的Python机器学习系列教程。
