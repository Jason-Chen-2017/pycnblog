# 4.具体代码实例和详细解释说明

在这一部分，我们将通过具体代码实例来详细解释MySQL数据库安全与保护的5种方法。

## 4.1 数据库用户权限的管理

### 4.1.1 创建数据库用户

```sql
CREATE USER 'username'@'host' IDENTIFIED BY 'password';
```

### 4.1.2 设置数据库用户密码

```sql
SET PASSWORD FOR 'username'@'host' = PASSWORD('password');
```

### 4.1.3 授予数据库用户权限

```sql
GRANT SELECT, INSERT, UPDATE, DELETE ON database.* TO 'username'@'host';
```

### 4.1.4 撤销数据库用户权限

```sql
REVOKE ALL PRIVILEGES ON database.* FROM 'username'@'host';
```

### 4.1.5 删除数据库用户

```sql
DROP USER 'username'@'host';
```

## 4.2 数据库备份和恢复

### 4.2.1 全量备份

```shell
mysqldump -u username -p database > backup.sql
```

### 4.2.2 增量备份

```shell
mysqldump -u username -p --single-transaction -c database > backup.sql
```

### 4.2.3 恢复数据库

```shell
mysql -u username -p database < backup.sql
```
