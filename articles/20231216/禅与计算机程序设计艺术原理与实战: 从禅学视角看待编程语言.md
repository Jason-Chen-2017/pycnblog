## 4.1 动态规划实例

我们来看一个动态规划的实例，即计算最长子序列问题。在这个问题中，我们需要找到一个序列中最长的子序列，使得子序列中的每个元素都是唯一的。

```python
def longest_unique_subsequence(s):
    n = len(s)
    dp = [1] * n
    for i in range(1, n):
        for j in range(i):
            if s[i] != s[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```

在这个实例中，我们使用了禅学的原则来提高动态规划的效率。我们使用了递归来表示问题的状态，并使用了 memoization 来避免重复计算。

```python
def fast_pow(x, n):
    if n == 0:
        return 1
    elif n % 2 == 0:
        return fast_pow(x * x, n // 2)
    else:
        return x * fast_pow(x * x, n // 2)
```
