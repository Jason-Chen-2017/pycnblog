```
#include <iostream>
#include <cstdlib>

int main(int argc, char *argv[])
{
    if (argc != 2) {
        std::cerr << "Usage: " << argv[0] << " <program>" << std::endl;
        return 1;
    }

    std::string program = argv[1];
    std::string command = "gdb " + program;
    int ret = system(command.c_str());
    if (ret != 0) {
        std::cerr << "GDB failed to start" << std::endl;
        return 1;
    }

    return 0;
}
```

在这个代码实例中，我们使用GDB库来实现一个简单的交互式调试环境。具体实现步骤如下：

1. 检查命令行参数是否正确。
2. 获取程序名称。
3. 使用GDB库启动GDB调试器。
4. 检查GDB调试器是否启动成功。
