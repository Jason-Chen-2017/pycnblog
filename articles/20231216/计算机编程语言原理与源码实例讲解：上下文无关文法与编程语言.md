# 2.核心概念与联系
上下文无关文法（Context-Free Grammar，CFG）是计算机编程语言的一个重要概念，它用于描述语言的语法结构。CFG由一个非终结符和一个或多个终结符组成，它们之间通过一组产生规则相互关联。CFG的核心概念包括：

- 非终结符：表示语法中的符号，可以被替换为其他符号的非终结符或终结符。
- 终结符：表示语法中的符号，不能被替换。
- 产生规则：描述了非终结符如何被替换为其他符号的规则。
- 文法：一个由非终结符、终结符和产生规则组成的集合。

CFG与编程语言的联系在于，它们用于描述编程语言的语法结构。通过使用CFG，我们可以确保编程语言的语法是正确的，从而确保程序的正确性。
