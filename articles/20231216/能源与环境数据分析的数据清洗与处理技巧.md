- 均值（mean）：对一组数据的和除以数据的个数。
- 方差（variance）：对一组数据的平方差除以数据的个数。
- 标准差（standard deviation）：方差的平方根。

- 回归分析（regression）：对两组数据的平均值进行线性拟合。

import numpy as np

# 读取数据
data = pd.read_csv('energy_data.csv')

# 数据清理
data = data.dropna()  # 删除缺失值
data = data.fillna(method='ffill')  # 填充缺失值

# 数据转换
data['energy'] = data['energy'] * 1000  # 将数据从千瓦转换为瓦

# 数据分析
mean_energy = data['energy'].mean()  # 计算平均值
std_energy = data['energy'].std()  # 计算标准差
corr_energy = data['energy'].corr(data['time'])  # 计算相关性

# 数据汇总
