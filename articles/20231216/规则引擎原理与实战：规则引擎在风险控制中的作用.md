## 4.1 规则引擎的代码实例

以下是一个简单的规则引擎的代码实例：

```python
class RuleEngine:
    def __init__(self):
        self.rules = []

    def add_rule(self, rule):
        self.rules.append(rule)

    def trigger_event(self, event):
        for rule in self.rules:
            if rule.condition(event):
                rule.execute(event)

    def get_result(self, event):
        result = []
        for rule in self.rules:
            if rule.condition(event):
                result.append(rule.execute(event))
        return result
```
