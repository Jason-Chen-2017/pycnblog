## 4.1 代码实例

我们以一个简单的排序算法——选择排序为例。选择排序的核心思想是在每次迭代中，从未排序的元素中选择最小（或最大）元素，并将其放在已排序的元素的末尾。

以下是选择排序的代码实例：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[min_index] > arr[j]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

## 4.2 详细解释说明

从禅学的角度来看，这段代码的核心思想是：

1. 保持对问题的耐心：选择排序需要遍历整个数组，因此需要保持对问题的耐心，不要急于解决。
2. 精心设计：在选择排序的实现中，需要精心设计循环的范围，确保所有元素都被遍历。
3. 学习与不断改进：选择排序只能在每次迭代中找到一个最小（或最大）元素，因此其时间复杂度较高。如果要提高排序的效率，可以学习其他更高效的排序算法，如快速排序或归并排序。
