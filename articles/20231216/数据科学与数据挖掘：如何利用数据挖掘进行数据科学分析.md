## 4.2 关联规则算法的代码实例

```python
from mlxtend.frequent_patterns import apriori
from mlxtend.frequent_patterns import association_rules

# 创建购物篮数据
baskets = [
    ['A', 'B', 'C'],
    ['B', 'C', 'D'],
    ['A', 'B', 'E'],
    ['A', 'B', 'C', 'D'],
    ['B', 'C', 'D', 'E'],
    ['A', 'B', 'C', 'D', 'E']
]

# 生成频繁项集
frequent_patterns = apriori(baskets, min_support=0.5, use_colnames=True)

# 生成关联规则
association_rules = association_rules(frequent_patterns, metric="confidence", min_threshold=0.7)

# 打印关联规则
print(association_rules)
```

在这个代码实例中，我们使用了 mlxtend 库中的 apriori 和 association_rules 函数来实现关联规则算法。我们创建了一个购物篮数据，并使用 apriori 函数来生成频繁项集。然后，我们使用 association_rules 函数来生成关联规则，并使用信息增益度量来筛选出满足条件的关联规则。
