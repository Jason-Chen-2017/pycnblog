# 2.核心概念与联系
# 2.1 禅的核心概念
禅是一种哲学思想，它强调人与人、人与自然、自然与自然之间的共感和和谐。禅的核心概念包括：

- 直指人心：禅强调直接触及人的内心，而不是通过言辞来传达思想。
- 无言传心：禅认为，真正的智慧和觉悟无法通过言辞来传达，只能通过实践和体验来获得。
- 一切皆空：禅认为，一切皆为空，没有固定的形式和属性。
- 无上之道：禅认为，道路没有终点，每个人都可以不断地进步和成长。

class ShoppingRecommendationSystem:
    def __init__(self, user_preferences, product_features):
        self.user_preferences = user_preferences
        self.product_features = product_features
        self.shared_points = self.find_shared_points()

    def find_shared_points(self):
        shared_points = []
        for user_preference in self.user_preferences:
            for product_feature in self.product_features:
                if self.is_shared_point(user_preference, product_feature):
                    shared_points.append((user_preference, product_feature))
        return shared_points

    def is_shared_point(self, user_preference, product_feature):
        return user_preference in product_feature

    def recommend_products(self):
        recommended_products = []
        for shared_point in self.shared_points:
            recommended_products.append(shared_point[1])
        return recommended_products
