## 3.1 二分查找算法原理

二分查找是一种常用的搜索算法，它的原理是：在有序数组中，将搜索区间分为两个子区间，然后根据搜索关键字与子区间中的关键字进行比较，直到找到所需的关键字或搜索区间为空。

二分查找算法的具体操作步骤如下：

1. 确定搜索关键字。
2. 将数组分为两个子区间，一个是左子区间，一个是右子区间。
3. 比较搜索关键字与左子区间中的关键字，如果相等，则找到所需的关键字，搜索结束；如果大于，则将搜索区间移动到右子区间；如果小于，则将搜索区间移动到左子区间。

深度优先搜索是一种搜索算法，它的原理是：从搜索树的根节点开始，沿着一个路径向下搜索，直到达到叶子节点或搜索树的末端，然后回溯到上一个节点，并沿着另一个路径继续搜索。

深度优先搜索算法的具体操作步骤如下：

1. 初始化搜索树的根节点。
2. 从根节点开始，沿着一个路径向下搜索，直到达到叶子节点或搜索树的末端。

# 4.具体代码实例和详细解释说明

## 4.1 二分查找算法实现

```python
def binary_search(A, key):
    low = 0
    high = len(A) - 1

    while low <= high:
        mid = (low + high) // 2

        if A[mid] == key:
            return mid
        elif A[mid] < key:
            low = mid + 1
        else:
            high = mid - 1

    return -1
```

该代码实现了二分查找算法，它接收一个有序数组`A`和搜索关键字`key`，然后返回数组中关键字的索引。如果没有找到关键字，则返回-1。

## 4.2 深度优先搜索算法实现

```python
class Graph:
    def __init__(self):
        self.adjacent = {}

    def add_node(self, node):
        self.adjacent[node] = []

    def add_edge(self, node1, node2):
        self.adjacent[node1].append(node2)
        self.adjacent[node2].append(node1)

    def depth_first_search(self, start_node):
        visited = set()
        stack = [start_node]

        while stack:
            current_node = stack.pop()

            if current_node not in visited:
                visited.add(current_node)
                for neighbor in self.adjacent[current_node]:
                    if neighbor not in visited:
                        stack.append(neighbor)

        return visited
```

该代码实现了一个简单的图，并实现了深度优先搜索算法。图由一个字典`adjacent`表示，每个节点都有一个列表，表示与其相连的邻居节点。`add_node`方法用于添加节点，`add_edge`方法用于添加边。`depth_first_search`方法用于实现深度优先搜索，它返回从`start_node`开始的所有访问的节点的集合。
