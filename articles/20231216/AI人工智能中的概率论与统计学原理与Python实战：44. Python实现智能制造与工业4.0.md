# 2.核心概念与联系
在智能制造和工业4.0中，概率论和统计学的核心概念包括随机变量、概率分布、期望、方差、协方差等。这些概念是人工智能和大数据分析中的基础，它们可以帮助我们理解和预测数据中的模式和规律。

随机变量是一个随机过程中的一个变量，它可以取多个不同的值。概率分布是一个随机变量的所有可能取值及其对应的概率的分布。期望是随机变量的一个数学期望，它表示随机变量的平均值。方差是一个随机变量的一个度量，它表示随机变量的离散程度。协方差是两个随机变量之间的一个度量，它表示这两个随机变量之间的相关性。

## 3.1 最大似然估计
最大似然估计是一种用于估计参数的方法，它的基本思想是找到使目标函数取得最大值的参数。目标函数是一个随机变量的似然函数，它表示随机变量的概率密度函数。

假设我们有一个随机样本，每个样本都是一个随机变量的实例。我们可以使用最大似然估计来估计这个随机变量的参数。具体操作步骤如下：

1. 计算样本的似然函数。
2. 找到似然函数的最大值。
3. 将最大值对应的参数值作为估计值。

## 3.2 贝叶斯定理
贝叶斯定理是一种用于计算条件概率的方法，它的基本思想是将条件概率表达为两个概率的乘积。贝叶斯定理可以用来解决各种条件概率问题。

假设我们有一个随机事件A和一个随机事件B，我们需要计算条件概率P(A|B)。具体操作步骤如下：

1. 计算P(A|B)的表达式。
2. 使用贝叶斯定理计算P(A|B)。

## 3.3 线性回归
线性回归是一种用于预测随机变量的方法，它的基本思想是找到一个线性模型，使得模型的预测值最接近实际值。线性回归可以用来解决各种预测问题。

```python
import numpy as np

# 定义随机变量的概率密度函数
def pdf(x, mu, sigma):
    return 1 / (np.sqrt(2 * np.pi * sigma**2)) * np.exp(-(x - mu)**2 / (2 * sigma**2))

# 定义似然函数
def likelihood(x, mu, sigma):
    return np.sum(pdf(x[i], mu, sigma) for i in range(len(x)))

# 定义目标函数
def objective(mu, x, sigma):
    return -likelihood(x, mu, sigma)

# 定义初始参数值
mu0 = 0
sigma0 = 1

# 定义随机样本
x = np.array([1, 2, 3, 4, 5])

# 使用最大似然估计计算参数值
mu_max = np.argmax(objective(mu, x, sigma0))
sigma_max = np.sqrt(objective(mu_max, x, sigma0))

print("最大似然估计的参数值：mu =", mu_max, "，sigma =", sigma_max)
```
