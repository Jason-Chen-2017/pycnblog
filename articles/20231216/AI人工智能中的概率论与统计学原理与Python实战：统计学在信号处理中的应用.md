# 2.核心概念与联系
在信号处理中，我们经常需要使用概率论和统计学方法来处理数据。概率论是一门数学分支，用于描述不确定性的现象。统计学是一门数学分支，用于从数据中抽取信息，以便进行预测和决策。

概率论与统计学在信号处理中的应用主要包括以下几个方面：

1. 信号模型的建立：我们需要建立信号模型，以便对信号进行处理。信号模型可以是随机信号模型，也可以是确定信号模型。随机信号模型描述信号的不确定性，确定信号模型描述信号的确定性。

2. 信号的分析：我们需要对信号进行分析，以便提取有用的信息。信号分析可以是时域分析，也可以是频域分析。时域分析描述信号在时间域的变化，频域分析描述信号在频率域的变化。

3. 信号的处理：我们需要对信号进行处理，以便提高信号质量或提取有用的信息。信号处理可以是滤波处理，也可以是增强处理。滤波处理用于去除信号中的噪声，增强处理用于提高信号的可读性。

4. 信号的传输：我们需要对信号进行传输，以便将信号从一个位置传输到另一个位置。信号传输可以是无线传输，也可以是有线传输。无线传输使用电磁波进行信号传输，有线传输使用物理媒介进行信号传输。

5. 信号的识别：我们需要对信号进行识别，以便识别出信号的特征。信号识别可以是图像识别，也可以是语音识别。图像识别用于识别图像中的特征，语音识别用于识别语音中的特征。
