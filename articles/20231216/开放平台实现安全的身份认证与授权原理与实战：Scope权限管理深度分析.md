# 2.核心概念与联系

在开放平台上实现安全的身份认证与授权，需要了解一些核心概念和联系。这些概念包括：身份认证、授权、Scope权限管理、OAuth2.0协议等。

身份认证是指验证用户是否是真实存在的人，而授权是指验证用户是否有权限访问某个资源。Scope权限管理是一种基于OAuth2.0协议的权限管理方式，它允许开发者在用户授权的基础上，为用户提供更精确的权限控制。

OAuth2.0协议是一种基于标准的授权协议，它允许第三方应用程序访问用户的资源，而不需要获取用户的密码。OAuth2.0协议提供了一种安全的方式，以便用户可以授权第三方应用程序访问他们的资源，而不需要将他们的密码传递给第三方应用程序。
