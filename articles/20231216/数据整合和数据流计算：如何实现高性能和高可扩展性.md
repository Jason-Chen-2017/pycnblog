### 4.1数据整合的具体代码实例

在本节中，我们将通过一个具体的数据整合任务来详细解释数据整合的具体操作步骤。

假设我们需要将来自不同数据源的数据进行整合，以便进行更高级别的分析和处理。具体来说，我们需要将来自不同数据源的数据进行清洗、转换、汇总等操作，以便在后续的分析和处理过程中更容易获取有关数据的洞察力。

以下是一个具体的数据整合任务的代码实例：

```python
import pandas as pd

# 读取来自不同数据源的数据
data1 = pd.read_csv('data1.csv')
data2 = pd.read_csv('data2.csv')

# 数据清洗
data1 = data1.drop_duplicates()
data2 = data2.drop_duplicates()

# 数据转换
data1['age'] = data1['age'].astype(int)
data2['age'] = data2['age'].astype(int)

# 数据集成
data = pd.concat([data1, data2], axis=0)

# 数据汇总
result = data.groupby('age').mean()

# 输出结果
print(result)
```

在这个代码实例中，我们首先读取来自不同数据源的数据，并对这些数据进行清洗、转换、汇总等操作。最后，我们输出了整合后的数据，以便在后续的分析和处理过程中更容易获取有关数据的洞察力。

data = pd.DataFrame({
