3. 第三代商业智能（2000年代中至2010年代初）：这一阶段的商业智能主要是通过数据分析和数据挖掘的方式来分析数据，以帮助企业做出决策。这一阶段的商业智能主要是通过数据分析和数据挖掘的方式来分析数据，以帮助企业做出决策。

- 使用Python的pandas库进行数据聚合：

```python
import pandas as pd

# 创建DataFrame
data = pd.DataFrame({'x': [1, 2, 3], 'y': [4, 5, 6]})

# 聚合数据
aggregated_data = data.groupby('x').agg({'y': ['mean', 'std']})

# 打印聚合数据
print(aggregated_data)
```
