# 2.核心概念与联系

在探讨软件架构之前，我们需要了解一些基本的概念。

## 2.1 软件架构的定义

软件架构是软件系统的组件、模块、子系统的组织、关系和协作方式的大规模结构。它决定了软件系统的可靠性、可扩展性、可维护性等方面。

## 2.2 软件架构的特点

1. 软件架构是软件系统的长期决策，它决定了系统的性能、可靠性、可扩展性等方面。
2. 软件架构是软件系统的基础设施，它为软件系统提供了基本的结构和框架。
3. 软件架构是软件系统的核心，它决定了系统的功能和性能。

## 2.3 软件架构的分类

1. 基于功能的分类：根据软件系统的功能需求，可以将软件架构分为数据库架构、网络架构、应用架构等。
2. 基于组件的分类：根据软件系统的组件结构，可以将软件架构分为面向对象架构、面向服务架构、面向组件架构等。
3. 基于风格的分类：根据软件系统的设计风格，可以将软件架构分为层次结构架构、客户/供应商架构、卵形架构等。

### 4.3.1 数据库架构

我们使用MySQL作为数据库，设计了用户表、博客表等。

```sql
CREATE TABLE user (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(20) NOT NULL,
    password VARCHAR(20) NOT NULL,
    email VARCHAR(50) NOT NULL
);

CREATE TABLE blog (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    title VARCHAR(100) NOT NULL,
    content TEXT NOT NULL,
    create_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES user(id)
);
```

### 4.3.2 网络架构

我们使用Flask作为Web框架，设计了用户注册、登录、发布博客、查看博客等API。

```python
from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'mysql://root:root@localhost/blog'
db = SQLAlchemy(app)

@app.route('/register', methods=['POST'])
def register():
    username = request.json.get('username')
    password = request.json.get('password')
    email = request.json.get('email')
    user = User(username=username, password=password, email=email)
    db.session.add(user)
    db.session.commit()
    return jsonify({'message': '注册成功'})

@app.route('/login', methods=['POST'])
def login():
    username = request.json.get('username')
    password = request.json.get('password')
    user = User.query.filter_by(username=username, password=password).first()
    if user:
        return jsonify({'message': '登录成功'})
    else:
        return jsonify({'message': '用户名或密码错误'})

@app.route('/blog', methods=['POST'])
def publish_blog():
    user_id = request.json.get('user_id')
    title = request.json.get('title')
    content = request.json.get('content')
    blog = Blog(user_id=user_id, title=title, content=content)
    db.session.add(blog)
    db.session.commit()
    return jsonify({'message': '发布博客成功'})

@app.route('/blog/<int:blog_id>', methods=['GET'])
def get_blog(blog_id):
    blog = Blog.query.get(blog_id)
    if blog:
        return jsonify({'title': blog.title, 'content': blog.content, 'create_time': blog.create_time})
    else:
        return jsonify({'message': '博客不存在'})
```
