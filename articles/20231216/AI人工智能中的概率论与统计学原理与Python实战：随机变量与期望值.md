```python
import numpy as np

# 定义一个离散型随机变量
X = np.array([1, 2, 3, 4, 5])
P = np.array([0.1, 0.2, 0.3, 0.1, 0.3])

# 定义一个连续型随机变量
X = np.linspace(0, 10, 100)
P = np.exp(-X**2 / 2) / np.sqrt(2 * np.pi)
```

## 4.2 概率分布的类型

```python
# 计算离散型随机变量的期望值

```
