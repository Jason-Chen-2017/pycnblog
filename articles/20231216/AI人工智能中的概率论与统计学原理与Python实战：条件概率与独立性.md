## 3.1 条件概率
条件概率是概率论中的一个重要概念，用于描述一个事件发生的条件下，另一个事件发生的概率。条件概率可以表示为：

$$
P(A|B) = \frac{P(A \cap B)}{P(B)}
$$

其中，$P(A|B)$ 表示事件A发生的条件事件B发生的概率，$P(A \cap B)$ 表示事件A和事件B同时发生的概率，$P(B)$ 表示事件B发生的概率。

### 3.1.1 贝叶斯定理
贝叶斯定理是条件概率的一个重要应用，用于计算条件概率的逆向。贝叶斯定理可以表示为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示事件A发生的条件事件B发生的概率，$P(B|A)$ 表示事件B发生的条件事件A发生的概率，$P(A)$ 表示事件A发生的概率，$P(B)$ 表示事件B发生的概率。

### 3.1.2 条件独立性
条件独立性是条件概率的一个重要概念，用于描述两个事件在某个条件下是否相互独立。两个事件A和B在条件B发生的情况下是独立的，可以表示为：

$$
A \perp B | B
$$

## 3.2 独立性
独立性是概率论中的一个重要概念，用于描述两个事件是否相互独立。两个事件A和B是独立的，可以表示为：

$$
A \perp B
$$

### 3.2.1 独立性的定义
两个事件A和B是独立的，当且仅当它们的交集的概率与每个事件的概率之积相等。可以表示为：

$$
P(A \cap B) = P(A)P(B)
$$

### 3.2.2 独立性的性质
独立性有以下几个性质：

1. 交换性：$A \perp B \Leftrightarrow B \perp A$
2. 自反性：$A \perp A$
3. 传递性：$A \perp B, B \perp C \Rightarrow A \perp C$
4. 单调性：$A \perp B \Rightarrow A \cup C \perp B \cup C$

# 4.具体代码实例和详细解释说明

## 4.1 条件概率的计算
我们可以使用Python的numpy库来计算条件概率。以下是一个示例代码：

```python
import numpy as np

# 定义事件A和事件B的概率
P_A = 0.6
P_B = 0.5
P_A_B = 0.3

# 计算条件概率
P_A_given_B = P_A_B / P_B
print("P(A|B) =", P_A_given_B)
```

## 4.2 独立性的计算
我们可以使用Python的numpy库来计算独立性。以下是一个示例代码：

```python
import numpy as np

# 定义事件A和事件B的概率
P_A = 0.6
P_B = 0.5
P_A_B = 0.3

# 计算独立性
independent = P_A_B == (P_A * P_B)
print("A and B are independent:", independent)
```
