# 2.核心概念与联系
数据质量报告是一种系统性的数据质量评估方法，旨在评估数据的准确性、完整性、一致性、及时性和可靠性等方面的质量。数据质量报告包括以下几个核心概念：

- 数据质量指标：数据质量指标是用于衡量数据质量的标准，例如准确性、完整性、一致性等。
- 数据质量评估方法：数据质量评估方法是用于评估数据质量指标的方法，例如数据清洗、数据验证、数据审计等。
- 数据质量报告：数据质量报告是一种系统性的数据质量评估方法，包括数据质量指标、数据质量评估方法等信息。

数据质量报告与数据质量管理相联系，数据质量管理是一种系统性的数据质量控制方法，包括数据质量政策、数据质量标准、数据质量评估方法等。数据质量报告是数据质量管理的一个重要组成部分，用于评估数据质量的实际情况。
