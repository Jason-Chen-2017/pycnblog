# 2.核心概念与联系
# 2.1社会网络分析的核心概念
社会网络分析的核心概念包括：

- 节点：节点是社会网络中的基本单位，它可以是人、组织、事件等。
- 边：边是节点之间的关系，它可以是人与人之间的关系，也可以是组织与组织之间的关系，或者是事件与事件之间的关系。
- 网络：网络是节点和边的集合，它可以表示一个社会系统的结构。
- 中心性：中心性是一个节点在网络中的重要性，它可以通过节点的度（即与其他节点的关系数）来衡量。
- 集中性：集中性是网络中一些节点或边的占比，它可以用来衡量网络的不均衡性。
- 桥梁：桥梁是网络中连接两个分量的边的集合，它可以用来衡量网络的连通性。
