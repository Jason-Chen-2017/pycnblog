## 3.1 数值计算

数值计算是一种在计算机上进行数学计算的方法，它主要关注于求解数值解。数值计算的主要算法包括：

1. 迭代法：迭代法是一种通过逐步更新变量值来求解方程的方法。例如，梯度下降法就是一种迭代法，它通过逐步更新权重值来最小化损失函数。

2. 分差法：分差法是一种通过计算函数在某个点的差分来求解函数的值的方法。例如，中差法和逐差法是分差法的两种常见实现方式。

3. 积分法：积分法是一种通过计算函数的面积来求解积分的方法。例如，梯形公式和Simpson公式是积分法的两种常见实现方式。

### 3.1.1 梯度下降法

梯度下降法是一种通过逐步更新变量值来最小化函数的方法。它的核心算法如下：

1. 初始化变量值。
2. 计算函数的梯度。
3. 更新变量值。
4. 重复步骤2和步骤3，直到收敛。

梯度下降法的数学模型公式如下：

$$
\theta_{t+1} = \theta_t - \alpha \nabla J(\theta_t)
$$

其中，$\theta$表示变量值，$t$表示时间步，$\alpha$表示学习率，$\nabla J(\theta_t)$表示函数$J$的梯度。

### 3.1.2 中差法

中差法是一种分差法的实现方式，它通过计算函数在某个点的差分来求解函数的值。其算法如下：

1. 选择一个步长$h$。
2. 计算函数在某个点的差分。
3. 通过递归计算函数的值。

中差法的数学模型公式如下：

$$
f(x+h) = f(x) + hf'(x) + \frac{h^2}{2}f''(x)
$$

其中，$f(x)$表示函数的值，$f'(x)$表示函数的一阶导数，$f''(x)$表示函数的二阶导数。

### 3.1.3 梯形公式

梯形公式是一种积分法的实现方式，它通过计算函数的面积来求解积分。其算法如下：

1. 选择一个步长$h$。
2. 计算函数在某个点的梯形公式。
3. 通过递归计算积分的值。

梯形公式的数学模型公式如下：

$$
\int_a^b f(x)dx = \frac{h}{2}[f(x_0) + 2f(x_1) + 2f(x_2) + \cdots + 2f(x_{n-1}) + f(x_n)]
$$

其中，$a$和$b$表示积分区间，$x_i$表示积分区间内的点，$h$表示步长。

### 3.2.1 前向差分

前向差分是一种求解函数导数的方法，它通过计算函数在某个点的差分来求解函数的导数。其算法如下：

1. 选择一个步长$h$。

前向差分的数学模型公式如下：

$$

## 4.1 梯度下降法

我们来看一个简单的梯度下降法的Python实例，这个实例是用于最小化一个简单的二次方程。

```python
import numpy as np

def f(x):
    return x**2

def gradient(x):
    return 2*x

def gradient_descent(x0, alpha, iterations):
    x = x0
    for i in range(iterations):
        grad = gradient(x)
        x = x - alpha * grad
        print(f"Iteration {i+1}: x = {x}, f(x) = {f(x)}")
    return x

x0 = 5
alpha = 0.1
iterations = 100

x_min = gradient_descent(x0, alpha, iterations)
print(f"Minimum x: {x_min}")
```

在这个实例中，我们首先定义了一个简单的二次方程$f(x) = x^2$，并计算了它的导数$f'(x) = 2x$。然后，我们使用梯度下降法来最小化这个函数。我们选择了一个初始值$x_0 = 5$，学习率$\alpha = 0.1$，以及迭代次数$100$。最后，我们打印了每一次迭代的结果，以及最小值。

## 4.2 中差法

我们来看一个简单的中差法的Python实例，这个实例是用于求解一个简单的三次方程。

```python
import numpy as np

def f(x):
    return x**3 - 6*x**2 + 11*x - 6

def central_difference(x0, h, iterations):
    x = x0
    for i in range(iterations):
        f_plus_h = f(x + h)
        f_minus_h = f(x - h)
        f_prime = (f_plus_h - f_minus_h) / (2 * h)
        x = x - h * f_prime
        print(f"Iteration {i+1}: x = {x}, f'(x) = {f_prime}")
    return x

x0 = 1
h = 0.01
iterations = 100

x_prime = central_difference(x0, h, iterations)
print(f"Derivative at x = {x0}: {x_prime}")
```

在这个实例中，我们首先定义了一个简单的三次方程$f(x) = x^3 - 6x^2 + 11x - 6$。然后，我们使用中差法来求解这个函数的导数。我们选择了一个初始值$x_0 = 1$，步长$h = 0.01$，以及迭代次数$100$。最后，我们打印了每一次迭代的结果，以及导数在$x_0$处的值。
