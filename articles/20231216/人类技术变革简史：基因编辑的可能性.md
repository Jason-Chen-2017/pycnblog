# 2.核心概念与联系

基因编辑是一种修改生物组织基因序列的技术，可以在DNA层面进行精确的修改。这项技术的出现为人类提供了一种修改生物组织基因序列的方法，为生物科学和医学研究提供了新的可能性。

基因编辑技术的核心概念包括CRISPR/Cas9、基因编辑器、基因剪接等。CRISPR/Cas9是一种基因编辑技术，它可以通过RNA引导序列与特定DNA序列进行对比，从而识别并修改特定基因。基因编辑器是一种软件工具，可以帮助研究人员设计和实现基因编辑任务。基因剪接是一种基因编辑技术，它可以通过双链DNA断裂并重组，从而实现基因的插入、删除或替换。
