## 4.1 知识图谱的具体代码实例
以下是一个简单的知识图谱的具体代码实例：

```python
import networkx as nx

# 创建一个空的知识图谱
g = nx.Graph()

# 添加实体
g.add_node("人")
g.add_node("地点")
g.add_node("组织")

# 添加关系
g.add_edge("人", "地点", relation="属于")
g.add_edge("人", "组织", relation="工作在")

# 添加属性
g.nodes["人"].update({"名字": "张三"})
g.nodes["地点"].update({"名字": "北京"})
g.nodes["组织"].update({"名字": "阿里巴巴"})
```

在这个代码实例中，我们使用NetworkX库创建了一个简单的知识图谱。我们添加了三种实体类型（人、地点、组织），以及它们之间的关系（属于、工作在）。我们还添加了实体的属性（名字）。
