## 4.1 数据分布式存储

我们可以使用Hadoop Distributed File System（HDFS）来实现数据分布式存储。HDFS是一个分布式文件系统，可以让文件更好地利用资源，降低成本，提高效率。

以下是一个使用HDFS的代码实例：

```java
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.FileSystem;
import org.apache.hadoop.fs.Path;

public class HDFSExample {
    public static void main(String[] args) throws Exception {
        // 获取HDFS配置
        Configuration conf = new Configuration();
        // 获取HDFS文件系统实例
        FileSystem fs = FileSystem.get(conf);
        // 创建文件
        Path src = new Path("/user/hadoop/input/");
        Path dst = new Path("/user/hadoop/output/");
        // 复制文件
        fs.copyFromLocalFile(false, src, dst);
        // 关闭HDFS文件系统实例
        fs.close();
    }
}
```

在这个代码实例中，我们首先获取了HDFS配置，然后获取了HDFS文件系统实例。接着，我们创建了一个源文件和一个目标文件。最后，我们复制了源文件到目标文件。

## 4.2 数据分布式计算

我们可以使用Hadoop MapReduce来实现数据分布式计算。Hadoop MapReduce是一个分布式计算框架，可以让计算任务更好地利用资源，降低成本，提高效率。

以下是一个使用Hadoop MapReduce的代码实例：

```java
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;
import org.apache.hadoop.util.StringUtils;

public class MapReduceExample {
    public static void main(String[] args) throws Exception {
        // 获取Hadoop配置
        Configuration conf = new Configuration();
        // 获取Hadoop任务实例
        Job job = Job.getInstance(conf, "word count");
        // 设置任务输入路径
        FileInputFormat.addInputPath(job, new Path(args[0]));
        // 设置任务输出路径
        FileOutputFormat.setOutputPath(job, new Path(args[1]));
        // 设置任务Mapper类
        job.setMapperClass(Map.class);
        // 设置任务Reducer类
        job.setReducerClass(Reduce.class);
        // 设置任务输出键类
        job.setMapOutputKeyClass(Text.class);
        // 设置任务输出值类
        job.setMapOutputValueClass(IntWritable.class);
        // 设置任务输出键类
        job.setOutputKeyClass(Text.class);
        // 设置任务输出值类
        job.setOutputValueClass(IntWritable.class);
        // 提交任务
        job.waitForCompletion(true);
    }
}
```

在这个代码实例中，我们首先获取了Hadoop配置，然后获取了Hadoop任务实例。接着，我们设置了任务输入路径和任务输出路径。然后，我们设置了任务Mapper类和任务Reducer类。最后，我们提交了任务。
