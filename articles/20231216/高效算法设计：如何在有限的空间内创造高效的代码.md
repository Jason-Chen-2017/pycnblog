```python
def find_max_subarray(nums):
    max_sum = float('-inf')
    current_sum = 0
    for num in nums:
        current_sum += num
        max_sum = max(max_sum, current_sum)
        if current_sum < 0:
            current_sum = 0
    return max_sum
```

这个代码实现了一个求最大子数组和的算法。它的时间复杂度为O(n)，空间复杂度为O(1)。在这个算法中，我们使用了空间换时间的方法，通过使用一个变量`current_sum`来记录当前子数组的和，并将最大子数组和存储在变量`max_sum`中。通过这种方法，我们避免了使用额外的数据结构，从而降低了空间复杂度。
