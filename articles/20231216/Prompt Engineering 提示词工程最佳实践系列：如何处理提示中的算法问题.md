# 4.具体代码实例和详细解释说明

在处理算法问题时，我们可以通过编写代码实现算法的具体实现。以下是一些具体的代码实例和详细解释说明：

1. **冒泡排序**：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

这个代码实现了冒泡排序算法，通过多次比较和交换元素，将数据集按照升序排列。

2. **二分搜索**：

```python
def binary_search(arr, target):
    left, right = 0, len(arr)-1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

这个代码实现了二分搜索算法，通过将数据集划分为两部分，并根据目标值的大小选择不同的部分进行搜索，将搜索范围逐渐缩小，直到找到目标值。

3. **拓扑排序**：

```python
def topological_sort(graph):
    n = len(graph)
    indegree = [0] * n
    for node in graph:
        for neighbor in graph[node]:
            indegree[neighbor] += 1
    queue = [node for node in range(n) if indegree[node] == 0]
    result = []
    while queue:
        node = queue.pop(0)
        result.append(node)
        for neighbor in graph[node]:
            indegree[neighbor] -= 1
            if indegree[neighbor] == 0:
                queue.append(neighbor)
    return result
```

这个代码实现了拓扑排序算法，通过计算每个节点的入度，将入度为0的节点放入队列，并逐一弹出并添加到结果列表中，直到队列为空。
