```python
class Calculator:
    def __init__(self):
        self.history = []

    def add(self, a, b):
        return a + b

    def subtract(self, a, b):
        return a - b

    def multiply(self, a, b):
        return a * b

    def divide(self, a, b):
        return a / b

    def save_history(self, a, b, operation):
        self.history.append((a, b, operation))

    def display_history(self):
        print(self.history)

calculator = Calculator()
calculator.add(2, 3)
calculator.subtract(5, 3)
calculator.multiply(2, 3)
calculator.divide(4, 2)
calculator.save_history(2, 3, 'add')
calculator.save_history(5, 3, 'subtract')
calculator.save_history(2, 3, 'multiply')
calculator.save_history(4, 2, 'divide')
calculator.display_history()
```

在上述代码中，我们定义了一个Calculator类，用于实现计算器软件产品的功能。我们可以通过调用Calculator类的add、subtract、multiply和divide方法来进行数学计算，并通过调用save_history和display_history方法来存储和显示计算器的历史记录。

在上述代码中，我们首先定义了一个Calculator类，用于实现计算器软件产品的功能。Calculator类的__init__方法用于初始化计算器软件产品的历史记录。add、subtract、multiply和divide方法用于进行数学计算，save_history方法用于存储计算器的历史记录，display_history方法用于显示计算器的历史记录。

在主程序中，我们创建了一个Calculator对象，并调用其add、subtract、multiply和divide方法来进行数学计算。同时，我们还调用save_history和display_history方法来存储和显示计算器的历史记录。
