### 4.1 代码实例

我们来看一个简单的排序算法的例子，即冒泡排序。冒泡排序是一种简单的排序算法，它的时间复杂度为O(n^2)，空间复杂度为O(1)。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

### 4.2 详细解释说明

从上面的代码实例中，我们可以看到冒泡排序的主要思想是通过多次遍历数组，将较大的元素逐步冒泡到数组的末尾，从而实现排序。在每次遍历中，我们将数组中的每个元素与其后面的元素进行比较，如果后面的元素较小，则将其与前面的元素交换位置。这个过程重复进行，直到数组中的元素已经排序为止。

在编写这个代码时，我们需要关注代码的结构和风格，并保持代码的可读性和可维护性。同时，我们需要关注代码的执行效率，并尽可能减少不必要的开销。在这个例子中，我们可以看到冒泡排序的执行效率是较低的，因为它的时间复杂度为O(n^2)。因此，在实际应用中，我们可能会选择其他更高效的排序算法，例如快速排序或归并排序。
