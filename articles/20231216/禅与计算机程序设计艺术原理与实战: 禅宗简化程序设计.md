# 4.1 代码实例
我们来看一个简单的代码实例，这个代码实例是一个计算两个数的和的程序：

```
#include <stdio.h>

int main() {
    int a, b;
    printf("请输入两个数的值：");
    scanf("%d %d", &a, &b);
    int sum = a + b;
    printf("两个数的和是：%d\n", sum);
    return 0;
}
```

# 4.2 详细解释说明
这个代码实例是一个简单的计算两个数的和的程序。首先，我们使用 `#include <stdio.h>` 包含标准输入输出库，然后使用 `int main()` 定义主函数。在主函数中，我们使用 `int a, b;` 定义两个整数变量 `a` 和 `b`。接着，我们使用 `printf("请输入两个数的值：");` 输出提示信息，然后使用 `scanf("%d %d", &a, &b);` 读取两个整数的值。接着，我们使用 `int sum = a + b;` 计算两个整数的和，并将结果存储在变量 `sum` 中。最后，我们使用 `printf("两个数的和是：%d\n", sum);` 输出两个整数的和。
