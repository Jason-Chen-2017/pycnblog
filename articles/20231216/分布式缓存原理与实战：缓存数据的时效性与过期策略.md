# 2.核心概念与联系

## 2.1 缓存数据的时效性

缓存数据的时效性是指缓存数据在有效时间内是否能够保持准确和最新。时效性是一个相对概念，它取决于业务场景和应用需求。例如，在搜索引擎中，搜索结果的时效性要求较高，因为用户需要得到最新的信息；而在社交媒体中，评论和点赞的时效性要求较低，因为用户关注的是历史记录和趋势。

## 2.2 缓存数据的过期策略

缓存数据的过期策略是指缓存数据在有效时间到期后如何处理和清除。过期策略有多种实现方式，例如绝对过期时间、相对过期时间、定期清除、惰性清除等。每种策略都有其优缺点，需要根据实际业务场景和应用需求进行选择。
