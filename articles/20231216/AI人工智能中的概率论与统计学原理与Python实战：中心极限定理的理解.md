## 2.1 概率论与统计学

概率论是一门研究不确定性和随机性的学科，它提供了一种数学模型来描述和预测随机事件发生的可能性。概率论主要通过概率度量和概率模型来描述随机事件。概率模型可以是离散型的（如掷骰子）或连续型的（如抛硬币）。

统计学是一门研究从数据中抽取信息并进行推断的学科。统计学主要通过观测和数据分析来估计和预测随机事件的参数。统计学可以分为参数统计学和非参数统计学，它们的主要区别在于参数估计方法。

## 2.2 中心极限定理

中心极限定理（Central Limit Theorem, CLT）是概率论和统计学中的一个关键定理，它描述了随机变量的分布在大样本量下逐渐趋向于正态分布。中心极限定理的核心观念是，无论随机变量的分布是什么，只要样本量足够大，它们的样本均值的分布将逐渐趋向于正态分布。

中心极限定理的主要结论是：

1. 如果随机变量X遵循某个分布，那么它的样本均值$\bar{X}$在大样本量下逐渐趋向于正态分布。
2. 样本均值的标准差$\sigma_{\bar{X}}$与随机变量的标准差$\sigma_X$成反比，即$\sigma_{\bar{X}} = \frac{\sigma_X}{\sqrt{n}}$，其中n是样本量。

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成随机数据
np.random.seed(0)
n = 1000
X = np.random.normal(loc=0, scale=1, size=n)

# 计算样本均值和标准差
mean_X = np.mean(X)
std_X = np.std(X)

# 计算正态分布的参数
mu = mean_X
sigma = std_X / np.sqrt(n)

# 计算正态分布的概率密度函数
def normal_pdf(x, mu, sigma):
    return (1 / (sigma * np.sqrt(2 * np.pi))) * np.exp(-(x - mu)**2 / (2 * sigma**2))

# 计算正态分布的累积分布函数
def normal_cdf(x, mu, sigma):
    return (1 + np.erf((x - mu) / (np.sqrt(2) * sigma))) / 2

# 绘制正态分布的概率密度函数和累积分布函数
x = np.linspace(mu - 5 * sigma, mu + 5 * sigma, 100)
plt.plot(x, normal_pdf(x, mu, sigma), label='PDF')
plt.plot(x, normal_cdf(x, mu, sigma), label='CDF')
plt.legend()
plt.show()
```

在这个代码实例中，我们首先生成了一组随机数据，然后计算了样本均值和标准差。接着，我们计算了正态分布的参数，并使用正态分布的概率密度函数和累积分布函数来绘制正态分布的概率密度函数和累积分布函数。从图中可以看出，随着样本量的增加，样本均值的分布逐渐趋向于正态分布。
