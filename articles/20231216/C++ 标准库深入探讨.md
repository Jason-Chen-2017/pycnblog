## 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

### 3.1.排序算法

C++ 标准库提供了许多排序算法，如 `sort`、`reverse`、`unique` 等。这些算法可以帮助我们更简单地对容器中的元素进行排序。

- 选择排序：选择排序是一种简单的排序算法，它的基本思想是在每次迭代中选择数组中最小的元素，并将其放在当前位置。选择排序的时间复杂度为 O(n^2)。

- 插入排序：插入排序是一种简单的排序算法，它的基本思想是将数组中的元素逐个插入到有序的子数组中。插入排序的时间复杂度为 O(n^2)。

- 冒泡排序：冒泡排序是一种简单的排序算法，它的基本思想是将数组中的元素逐个与相邻的元素进行比较，如果相邻的元素不满足排序规则，则交换它们的位置。冒泡排序的时间复杂度为 O(n^2)。

### 3.2.搜索算法

C++ 标准库提供了许多搜索算法，如 `find`、`count`、`lower_bound`、`upper_bound` 等。这些算法可以帮助我们更简单地在容器中查找元素。

- 线性搜索：线性搜索是一种简单的搜索算法，它的基本思想是逐个检查容器中的每个元素，直到找到目标元素。线性搜索的时间复杂度为 O(n)。

- 二分搜索：二分搜索是一种高效的搜索算法，它的基本思想是将容器分为两个部分，然后逐个检查中间的元素，直到找到目标元素或者确定目标元素不存在。二分搜索的时间复杂度为 O(log n)。

## 4.具体代码实例和详细解释说明

### 4.1.排序算法示例

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {5, 2, 9, 1, 3};

    // 使用选择排序
    std::sort(numbers.begin(), numbers.end());

    // 输出排序后的数组
    for (int number : numbers) {
        std::cout << number << " ";
    }

    return 0;
}
```

在上述代码中，我们使用了 `std::sort` 函数进行排序。`std::sort` 函数使用了快速排序算法，其时间复杂度为 O(n log n)。

### 4.2.搜索算法示例

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {5, 2, 9, 1, 3};

    // 使用二分搜索
    int target = 3;
    auto it = std::lower_bound(numbers.begin(), numbers.end(), target);

    if (it != numbers.end() && *it == target) {
        std::cout << "找到目标元素" << std::endl;
    } else {
        std::cout << "没有找到目标元素" << std::endl;
    }

    return 0;
}
```

在上述代码中，我们使用了 `std::lower_bound` 函数进行二分搜索。`std::lower_bound` 函数返回目标元素的迭代器，如果目标元素不存在，则返回大于目标元素的第一个元素的迭代器。
