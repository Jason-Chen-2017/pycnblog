```python
import numpy as np

def preprocess(data):
    data = np.array(data)
    data = data - np.mean(data)
    data = data / np.std(data)
    return data

data = [1, 2, 3, 4, 5]
data = preprocess(data)
```

3. 数据分析模块的代码实例：

```python
import numpy as np

def analyze(data):
    mean = np.mean(data)
    std = np.std(data)
    return mean, std

mean, std = analyze(data)
```
