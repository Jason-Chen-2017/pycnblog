# 2.核心概念与联系

## 2.1 函数的定义

在Python中，函数是一个可以执行特定任务的代码块。函数可以接收输入参数，执行某个任务，并返回结果。函数的定义使用关键字`def`来开始，后面跟着函数名和括号中的参数列表。函数内部包含了一个或多个语句，用于执行任务。

例如，下面是一个简单的函数，它接收两个参数并返回它们的和：

```python
def add(a, b):
    return a + b
```

## 2.2 函数的调用

函数调用是指在代码中使用函数名来执行函数。函数调用使用圆括号来表示，并可以传递实参给函数。当函数调用时，Python会执行函数内部的代码，并返回结果。

例如，下面是一个函数调用示例：

```python
result = add(3, 4)
print(result)  # 输出: 7
```

# 4.具体代码实例和详细解释说明

## 4.1 函数的定义

```python
# 定义一个函数，接收两个参数并返回它们的和
def add(a, b):
    return a + b

# 定义一个函数，接收两个参数并返回它们的差
def subtract(a, b):
    return a - b

# 定义一个函数，接收两个参数并返回它们的积
def multiply(a, b):
    return a * b

# 定义一个函数，接收两个参数并返回它们的商
def divide(a, b):
    return a / b
```

## 4.2 函数的调用

```python
# 调用add函数，传递实参3和4
result1 = add(3, 4)
print(result1)  # 输出: 7

# 调用subtract函数，传递实参5和2
result2 = subtract(5, 2)
print(result2)  # 输出: 3

# 调用multiply函数，传递实参6和3
result3 = multiply(6, 3)
print(result3)  # 输出: 18

# 调用divide函数，传递实参8和2
result4 = divide(8, 2)
print(result4)  # 输出: 4.0
```

# 6.附录常见问题与解答

Q: 如何定义一个无参数的函数？

A: 要定义一个无参数的函数，只需在参数列表中不包含任何参数即可。例如：

```python
def greet():
    print("Hello, world!")
```

Q: 如何定义一个接收任意数量参数的函数？

A: 要定义一个接收任意数量参数的函数，可以使用*args和**kwargs来接收未知数量的参数。例如：

```python
def sum_numbers(*args):
    return sum(args)

def keyword_args(**kwargs):
    return kwargs
```

Q: 如何定义一个递归函数？

A: 要定义一个递归函数，可以在函数内部调用自身。例如，下面是一个递归函数，用于计算阶乘：

```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)
```
