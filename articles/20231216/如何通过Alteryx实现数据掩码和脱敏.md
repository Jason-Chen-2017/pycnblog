# 2.核心概念与联系

## 2.1 数据掩码与脱敏的定义与区别

数据掩码（Data Masking）和数据脱敏（Data Anonymization）是两种用于保护数据敏感信息的方法。它们的主要区别在于：

- 数据掩码是一种在数据处理过程中将敏感信息替换为虚拟数据的方法，以保护数据的真实值。例如，将姓名替换为虚拟姓名，电话号码替换为虚拟电话号码等。
- 数据脱敏是一种在数据发布过程中将敏感信息替换为无法追溯到真实用户的信息的方法，以保护数据的隐私。例如，将身份证号码替换为前几位加星号等。

```python
import pandas as pd

# 导入数据
data = pd.read_csv('data.csv')

# 进行数据掩码和脱敏操作
data['name'] = data['name'].apply(lambda x: '虚拟姓名')
data['phone'] = data['phone'].apply(lambda x: '虚拟电话号码')
data['id_card'] = data['id_card'].apply(lambda x: x[:4] + '***' + x[-4:])

# 导出数据
data.to_csv('masked_data.csv', index=False)
```

在上述代码中，我们首先导入了数据，然后使用apply函数对需要进行掩码和脱敏的字段进行操作。例如，我们将姓名替换为虚拟姓名，电话号码替换为虚拟电话号码，身份证号码截取为前几位等。
