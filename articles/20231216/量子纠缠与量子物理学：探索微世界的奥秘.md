## 4.1 量子纠缠的Python代码实例

以下是一个使用Python的Qiskit库实现量子纠缠的代码示例：

```python
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 创建量子电路
qc = QuantumCircuit(2)

# 创建基态
qc.h(0)
qc.cx(0, 1)

# 绘制量子电路
plot_histogram(qc)

# 执行量子电路
simulator = Aer.get_backend('statevector_simulator')
job = simulator.run(assemble(qc))
result = job.result()

# 获取计算结果
counts = result.get_counts()
print(counts)
```

上述代码首先导入了Qiskit库，然后创建了一个量子电路，并对两个量子比特进行操作。最后，使用Qiskit的状态矢量模拟器执行量子电路，并获取计算结果。
