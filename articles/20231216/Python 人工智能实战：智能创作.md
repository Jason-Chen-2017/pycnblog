4. 梯度下降：

```python
import numpy as np

# 定义损失函数
def loss(w, X, y):
    return np.mean((y - (X @ w)) ** 2)

# 定义梯度
def grad(w, X, y):
    return (X.T @ (X @ w - y)) / len(y)

# 定义梯度下降函数
def gradient_descent(X, y, w, learning_rate, num_iterations):
    for _ in range(num_iterations):
        grad_w = grad(w, X, y)
        w = w - learning_rate * grad_w
        loss_w = loss(w, X, y)
    return w

# 训练线性回归模型
w = gradient_descent(X_train, y_train, np.zeros(X_train.shape[1]), 0.01, 1000)

# 预测结果
y_pred = X_test @ w
```
