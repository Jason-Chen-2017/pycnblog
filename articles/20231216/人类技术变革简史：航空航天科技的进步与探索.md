                 

# 1.背景介绍

航空航天科技是人类历史上最重要的科技变革之一，它使人类从地球上脱离了重力束缚，进入了太空，探索了宇宙。航空航天科技的发展历程可以分为以下几个阶段：

1.1 古代航空梦想
1.2 现代航空发展
1.3 航天发展

## 1.1 古代航空梦想

航空梦想始于人类对天空的好奇心，古希腊和古埃及文明中就有关于飞行的神话。后来，人们开始尝试制造飞行器，如翼龙、翱翔鸟兽等。15世纪，意大利的弗兰茨·赫拉利（Francesco de Vieri）首次尝试用机械手段飞行，但失败。17世纪，法国的埃德蒙·瓦尔迪格（Aédmond Gabrielli de Ferrare）试图用气球飞行，成功飞行了12公里。18世纪，英国的埃尔登·艾伯特（George Cayley）提出了飞行的基本原理，并设计了第一架可控制的飞机。

## 1.2 现代航空发展

19世纪末，美国的伯克利大学教授弗里德曼·艾伯特（Frederick W. Lanchester）提出了流动动力学的理论基础，为现代飞机的设计和制造提供了理论支持。1903年，伯克利大学的兄弟们伯劳姆·伯劳姆（Wright brothers）成功飞行了第一架人造飞机。1914年，德国的阿德尔弗雷德·艾迪斯顿（Adolph Eckert）开发了第一架电机飞机。1939年，德国的翰勒·墨菲（Hans von Ohain）和埃德华·艾迪（Erhardt Eckert）开发了第一架液体燃料发动机飞机。1947年，美国的侯爵·艾伯特·德勒（Charles “Chuck” Yeager）在B-29飞机上的飞行测试中超越了音速障碍。1957年，俄罗斯的苏联发射了第一颗人造卫星，标志着航空航天科技的进入太空时代。

## 1.3 航天发展

1957年，俄罗斯的苏联发射了第一颗人造卫星，标志着航空航天科技的进入太空时代。1961年，美国的阿斯顿·迪杰斯（Yuri Gagarin）成为了人类历史上的第一位太空人。1969年，美国的阿劳德·林登（Neil Armstrong）成功着陆在月球，标志着人类首次脱离地球。1981年，美国的阿斯特罗尼亚空间中心（Johnson Space Center）成功发射了第一颗商业卫星。1998年，中国成功发射了第一颗卫星，进入了太空时代。2003年，美国的纳斯阿拉空间中心（NASA）成功发射了第一颗火星探测器，探索了火星。2012年，中国的天眼卫星（China Fengyun-3C）成功观测到了太阳的光环，标志着人类首次观测到太阳的光环。2015年，中国的天舟五号（Shenzhou-5）成功发射，标志着中国进入了人造卫星时代。2019年，中国的天舟十三号（Chang'e-4）成功着陆在月球的反面，成功进行了月球反面的地面探测，标志着人类首次探索到月球的反面。
