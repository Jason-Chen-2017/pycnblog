# 2.核心概念与联系

## 2.1 航空器

航空器是航空航天科技的核心概念之一。航空器是一种能够在大气层中飞行的机械设备，通常包括飞行器、发动机、控制系统、导航系统等组成部分。航空器的发展历程可以分为以下几个阶段：

1. 古代飞行器：包括箭簇、伞飞机、蝴蝶飞机等。
2. 初期飞机：包括 Wright Flyer、德国的Junkers Ju 52等。
3. 现代飞机：包括战斗机、商务机、私人飞机等。
4. 太空飞船：包括俄罗斯的Soyuz、美国的Apollo等。

## 2.2 太空探索

太空探索是航空航天科技的另一个核心概念。太空探索是人类进入太空、研究太空环境、探索太空资源和研究宇宙的活动。太空探索的发展历程可以分为以下几个阶段：

1. 初期探索：包括Sputnik、Yuri Gagarin等。
2. 月球探索：包括Apollo 11、Apollo 13等。
3. 太空站：包括国际空间站等。
4. 行星探索：包括马尔沃斯探测器、奔驰探测器等。
5. 太阳探索：包括Parker Solar Probe等。

## 2.3 联系

航空器和太空探索之间的联系是航空航天科技的核心概念之一。航空器和太空探索是两个相互联系的领域，它们的技术和理论在很大程度上是相互支持的。例如，太空探索需要高性能的飞行器和太空飞船，而航空器的发展也受到太空探索的影响。
