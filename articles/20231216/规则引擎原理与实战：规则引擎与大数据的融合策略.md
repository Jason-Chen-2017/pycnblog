## 4.1 规则引擎算法的代码实例

以下是一个简单的规则引擎算法的代码实例：

```python
# 定义规则
rules = {
    "rule1": "IF age > 18 THEN can_vote = true",
    "rule2": "IF income > 10000 THEN can_buy_house = true"
}

# 定义事实
facts = {
    "age": 20,
    "income": 12000
}

# 执行规则引擎算法
def execute_rule_engine(rules, facts):
    for rule, condition in rules.items():
        for fact, value in facts.items():
            if condition.replace("{0}", str(value)).lower() == "true":
                print(f"Rule {rule} is true")

# 执行规则引擎算法
execute_rule_engine(rules, facts)
```
