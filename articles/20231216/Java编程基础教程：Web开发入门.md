Java编程基础教程：Web开发入门

Java是一种广泛使用的编程语言，它的核心特点是“平台无关性”，即Java程序可以在任何支持Java虚拟机（JVM）的平台上运行。Java的发展历程可以分为以下几个阶段：

1. 1995年，Sun Microsystems公司发布了第一版的Java语言和开发工具。
2. 1998年，Java开始发展为企业级应用，Java平台（Java Platform）被发布。
3. 2000年，Java开始发展为Web应用，Java Servlet和JavaServer Pages（JSP）技术被发布。
4. 2004年，Java开始发展为企业服务应用，Java EE（Java Platform, Enterprise Edition）技术被发布。
5. 2011年，Oracle公司收购了Sun Microsystems，并将Java技术放入公司的产品线中。
6. 2014年，Oracle发布了Java SE（Java Platform, Standard Edition）和Java EE技术的新版本。
7. 2017年，Oracle发布了Java SE和Java EE技术的新版本，并将Java EE技术重新命名为Java Platform, Microservices（JPM）。
8. 2018年，Oracle发布了Java SE和Java EE技术的新版本，并将Java EE技术重新命名为Java Platform, Enterprise Application（JPEA）。
