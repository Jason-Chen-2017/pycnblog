1. 推荐系统：推荐系统的一个简单实现是基于用户历史行为的协同过滤算法。以下是一个Python实现的基于用户历史行为的协同过滤算法：

```python
import numpy as np

# 用户历史行为数据
user_history = {
    'user1': [1, 2, 3],
    'user2': [2, 3, 4],
    'user3': [1, 3, 5],
}

# 计算用户之间的相似度
def similarity(user1, user2):
    intersection = set(user1).intersection(set(user2))
    union = set(user1).union(set(user2))
    return len(intersection) / len(union)

# 根据用户历史行为推荐商品
def recommend(user_history, target_user):
    recommendations = []
    for user, history in user_history.items():
        if user != target_user:
            similarity_score = similarity(history, user_history[target_user])
            recommendations.append((user, similarity_score))
    recommendations.sort(key=lambda x: x[1], reverse=True)
    return [user for user, _ in recommendations]

# 测试
target_user = 'user3'
print(recommend(user_history, target_user))
```

2. 自然语言处理：自然语言处理的一个简单实现是基于TF-IDF（Term Frequency-Inverse Document Frequency）的文本相似度计算。以下是一个Python实现的基于TF-IDF的文本相似度计算：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 文本数据
texts = [
    '人工智能大模型即服务时代',
    '人工智能技术发展迅速',
    '人工智能改变世界',
]

# 计算TF-IDF矩阵
tfidf_vectorizer = TfidfVectorizer()
tfidf_matrix = tfidf_vectorizer.fit_transform(texts)

# 计算文本之间的相似度
similarity_matrix = cosine_similarity(tfidf_matrix, tfidf_matrix)

# 打印结果
print(similarity_matrix)
```
