```python
import matplotlib.pyplot as plt

# 生成数据
x = np.linspace(-5, 5, 100)
y = 2 * x + 3 + np.random.randn(100)

# 训练模型
coef, intercept = np.polyfit(x, y, 1)

# 预测

y_pred = coef * x_new + intercept

# 可视化
plt.scatter(x, y)
plt.plot(x_new, y_pred, color='red')
plt.show()
```

## 4.2 模型可视化的代码实例

我们以一个简单的支持向量机模型为例，来演示模型可视化的代码实例。

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn import svm

# 生成数据
x = np.random.rand(100, 2)
y = np.logical_xor(x[:, 0] > 0.5, x[:, 1] > 0.5)

# 训练模型
clf = svm.SVC(kernel='linear')
clf.fit(x, y)

# 预测
x_new = np.linspace(-1, 1, 100)
y_pred = clf.predict(np.array([x_new] * 100).reshape(-1, 2))

# 可视化
plt.scatter(x[:, 0], x[:, 1], c=y, cmap='Reds')
plt.plot(x_new, x_new, color='black')
plt.plot(x_new, -x_new, color='black')
plt.plot(x_new, y_pred, color='red')
plt.show()
```

在这个代码实例中，我们首先生成了一组随机数据，然后使用支持向量机模型来拟合这组数据。接着，我们使用可视化工具来可视化模型的决策过程，从而更直观地理解模型的工作原理。
