- 分治法：将问题分解为多个子问题，然后递归地解决这些子问题，最后将解决的结果合并为原问题的解决方案。
- 动态规划：将问题分解为多个子问题，然后根据子问题的解决方案来得出原问题的解决方案。
- 贪心法：在每个决策中选择当前看起来最好的选择，以便最终得到一个满足问题需求的解决方案。

## 4.1 Python
```python
# 初始化
data = []

# 输入
while True:
    input_data = input("请输入数据: ")
    if input_data == "exit":
        break
    data.append(input_data)

# 处理
result = []
for item in data:
    if item.isdigit():
        result.append(int(item))

# 输出
print("结果:", result)

# 结束
```
在上述代码中，我们首先初始化一个空列表`data`，然后通过`input`函数获取用户输入的数据。如果用户输入的是"exit"，则程序结束。否则，我们遍历`data`列表，将每个元素转换为整数并添加到`result`列表中。最后，我们输出`result`列表的内容。

## 4.2 Java
```java
import java.util.ArrayList;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        // 初始化
        ArrayList<String> data = new ArrayList<>();

        // 输入
        Scanner scanner = new Scanner(System.in);
        while (true) {
            String input = scanner.nextLine();
            if (input.equals("exit")) {
                break;
            }
            data.add(input);
        }
        scanner.close();

        // 处理
        ArrayList<Integer> result = new ArrayList<>();
        for (String item : data) {
            if (item.matches("\\d+")) {
                result.add(Integer.parseInt(item));
            }
        }

        // 输出
        System.out.println("结果: " + result);
    }
}
```
在上述代码中，我们首先初始化一个空的`ArrayList`对象`data`，然后使用`Scanner`类获取用户输入的数据。如果用户输入的是"exit"，则程序结束。否则，我们遍历`data`列表，将每个元素转换为整数并添加到`result`列表中。最后，我们输出`result`列表的内容。
