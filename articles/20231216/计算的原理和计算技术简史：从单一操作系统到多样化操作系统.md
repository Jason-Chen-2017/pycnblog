# 2.核心概念与联系

单一操作系统和多样化操作系统是计算机科学的两个核心概念。单一操作系统是指计算机系统中只有一个操作系统，这个操作系统是固定的，不能被替换或扩展。多样化操作系统是指计算机系统中可以运行多个操作系统，这些操作系统可以被替换或扩展。

单一操作系统和多样化操作系统之间的联系是计算机科学的发展历程。从单一操作系统发展到多样化操作系统，是计算机科学的一个重要的转折点。这个转折点使得计算机从数学计算和科学计算的工具变成了各种应用软件的平台。
