# 2.核心概念与联系
在图像处理中，模式识别技术主要涉及以下几个核心概念：

- 图像特征：图像特征是图像中具有特定性质的部分，可以用来表示图像的某些特征信息。常见的图像特征包括边缘、角、纹理、颜色等。
- 图像分类：图像分类是将图像划分为不同类别的过程，主要通过对图像特征进行提取和比较来实现。
- 图像识别：图像识别是将图像与已知模式进行比较，以确定图像中的特定对象或模式的过程。
- 图像匹配：图像匹配是将两个或多个图像进行比较，以确定它们之间的相似性或不同的过程。
