```python
import re

# 定义一个正则表达式
pattern = r'\d+'

# 定义一个字符串
string = '1234567890'

# 使用正则表达式匹配字符串
match = re.search(pattern, string)

# 判断是否匹配成功
if match:
    print('匹配成功')
else:
    print('匹配失败')
```

在这个代码实例中，我们使用了正则表达式`\d+`来匹配一个字符串中的数字。`\d`表示一个数字，`+`表示一个或多个。我们定义了一个字符串`'1234567890'`，然后使用`re.search()`函数来匹配这个字符串。最后，我们判断是否匹配成功，如果匹配成功，则输出`'匹配成功'`，否则输出`'匹配失败'`。
