```python
import requests
from requests.exceptions import RequestException

class OpenPlatformAPI:
    def __init__(self, base_url):
        self.base_url = base_url

    def get_data(self, endpoint, params=None):
        try:
            response = requests.get(self.base_url + endpoint, params=params)
            response.raise_for_status()
            return response.json()
        except RequestException as e:
            print(e)

api = OpenPlatformAPI("https://api.example.com")
data = api.get_data("/data", {"key": "value"})
```

在这个代码实例中，我们定义了一个名为`OpenPlatformAPI`的类，用于访问开放平台的API。这个类有一个构造函数，用于初始化API的基本URL。我们还定义了一个名为`get_data`的方法，用于发送HTTP GET请求到API的某个端点，并返回响应的JSON数据。

我们创建了一个`OpenPlatformAPI`的实例，并使用`get_data`方法发送一个GET请求到API的`/data`端点，并传递一个参数`key=value`。我们将API的响应数据存储在`data`变量中。
