# 2.核心概念与联系

## 2.1 数据集成与整合的概念

数据集成是将来自不同数据源的数据进行整合、清洗、转换和汇总的过程，以形成一个统一的数据集。数据整合是将数据集成后的数据进行分析、挖掘和应用的过程。在医疗大数据分析中，数据集成与整合是将来自不同医疗数据源（如电子病历、病理报告、影像数据、基因组数据等）的数据进行整合、清洗、转换和汇总的过程，以形成一个统一的医疗数据集。

## 2.2 医疗大数据分析与医疗数据分析的联系

医疗大数据分析是医疗数据分析的一个子领域，它涉及到大规模、高速、多源、多格式、多类型的医疗数据的分析。医疗数据分析是对医疗数据进行分析、挖掘和应用的过程，它涉及到的数据规模相对较小。医疗大数据分析与医疗数据分析的联系在于，医疗大数据分析是对医疗数据分析的扩展和提升，它需要处理更加复杂、更加大规模的医疗数据。
