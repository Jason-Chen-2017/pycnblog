## 4. 具体代码实例和详细解释说明

具体代码实例如下：

```python
import cv2
import numpy as np

# 图像的灰度处理
def gray_processing(img):
    img_gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
    return img_gray

# 图像的二值化
def binary_processing(img_gray):
    _, img_binary = cv2.threshold(img_gray, 127, 255, cv2.THRESH_BINARY)
    return img_binary

# 图像的腐蚀和膨胀
def erode_dilate(img_binary, kernel):
    img_erode = cv2.erode(img_binary, kernel)
    img_dilate = cv2.dilate(img_erode, kernel)
    return img_erode, img_dilate

# 边缘检测
def edge_detection(img_gray):
    img_edge = cv2.Canny(img_gray, 50, 150)
    return img_edge

# 特征提取
def feature_extraction(img_edge):
    img_feature = cv2.Laplacian(img_edge, cv2.CV_64F).var()
    return img_feature

# 特征比较
def feature_comparison(img_feature_1, img_feature_2):
    M = cv2.matchTemplate(img_feature_1, img_feature_2, cv2.TM_CCOEFF_NORMED)
    return M
```

1. 图像的灰度处理：通过调用`cv2.cvtColor()`函数，将彩色图像转换为灰度图像。

3. 图像的腐蚀和膨胀：通过调用`cv2.erode()`和`cv2.dilate()`函数，对图像进行腐蚀和膨胀操作。

4. 边缘检测：通过调用`cv2.Canny()`函数，对图像进行边缘检测。

5. 特征提取：通过调用`cv2.Laplacian()`函数，对边缘信息进行特征提取。
