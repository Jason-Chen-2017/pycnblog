数学模型公式详细讲解如下：

1. 线性回归：y = b0 + b1x，其中y是因变量，x是自变量，b0和b1是回归系数。
2. 多项式回归：y = b0 + b1x + b2x^2 + ... + bnx^n，其中x是自变量，b0、b1、...、bn是回归系数。
3. 逻辑回归：P(y=1|x) = 1 / (1 + exp(-(b0 + b1x)))，其中y是因变量，x是自变量，b0和b1是回归系数。
4. 支持向量机：min(1/2 * w^T * w + C * sum(max(0, yi - w^T * xi)))，其中w是权重向量，xi是输入向量，yi是标签，C是惩罚参数。
5. 随机森林：y = median(f(x, z_k))，其中f是决策树，z_k是随机森林中的决策树，k是树的数量。
