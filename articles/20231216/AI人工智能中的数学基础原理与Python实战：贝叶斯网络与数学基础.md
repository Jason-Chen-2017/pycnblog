# 2.核心概念与联系

## 2.1 概率论与统计学

概率论是一门数学分支，它研究随机事件发生的可能性。概率是一个事件发生的可能性，范围在0到1之间。概率论提供了一种数学模型，用于描述和分析实际世界中的不确定性。

统计学是一门研究如何从数据中抽取信息的科学。统计学使用概率论来描述和分析数据，以便从数据中提取有用的信息。

## 2.2 贝叶斯定理

贝叶斯定理是概率论中的一个重要定理，它描述了如何更新现有知识以包括新的证据。贝叶斯定理可以用以下公式表示：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 是条件概率，表示当$B$发生时，$A$发生的概率；$P(B|A)$ 是条件概率，表示当$A$发生时，$B$发生的概率；$P(A)$ 是$A$发生的概率；$P(B)$ 是$B$发生的概率。

## 2.3 贝叶斯网络

贝叶斯网络是一种概率图模型，用于表示和推理关于随机变量之间的条件依赖关系。贝叶斯网络由一组随机变量和它们之间的条件独立关系组成。每个随机变量都有一个条件概率分布，这些分布可以通过贝叶斯定理来计算。

贝叶斯网络的一个重要特点是它可以用来表示和推理关于随机变量之间的条件依赖关系。这意味着，通过观察某些变量的值，我们可以推断其他变量的值。这是贝叶斯网络的强大之处，也是它在人工智能和机器学习领域的应用之一。

## 4.1 使用pymc3构建贝叶斯网络

```python
import pymc3 as pm
import numpy as np

# 定义随机变量
A = pm.Beta('A', alpha=1, beta=1)
B = pm.Normal('B', mu=A, tau=1)

# 创建模型
with pm.Model() as model:
    model.add_sample_code('B = A + np.random.normal(0, 1)')

# 推理

```

在这个例子中，我们定义了两个随机变量：`A` 和 `B`。`A` 是一个贝塔分布，`B` 是一个正态分布，其参数是`A`。我们使用pymc3的`sample`函数进行推理，得到了1000个样本。
