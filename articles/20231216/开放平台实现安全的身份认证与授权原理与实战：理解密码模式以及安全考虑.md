# 2.核心概念与联系

## 2.1 身份认证与授权
身份认证是用户在访问资源时向平台提供身份信息的过程，以确认用户的身份。身份认证的主要目的是确保用户的身份信息是真实的，以保护用户的隐私和平台的安全。

授权是指用户在访问资源时，平台根据用户的身份信息来确定用户是否有权访问该资源的过程。授权的主要目的是确保用户只能访问他们有权访问的资源，以保护平台的安全。

## 2.2 密码模式
密码模式是一种基于密码的身份验证方法，它通过用户输入密码来验证用户的身份。密码模式的核心思想是通过密码的复杂性和长度来保护用户的身份信息。

密码模式的主要组成部分包括：
- 用户名：用户在平台上的唯一标识。
- 密码：用户在平台上设置的密码，用于验证用户的身份。
- 密码验证器：用于验证用户输入的密码是否与用户设置的密码相匹配的组件。
