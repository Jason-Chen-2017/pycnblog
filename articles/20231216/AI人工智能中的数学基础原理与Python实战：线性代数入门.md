## 4.1 向量和矩阵的加法和减法

```python
import numpy as np

# 创建两个向量
vector1 = np.array([1, 2, 3])
vector2 = np.array([4, 5, 6])

# 向量加法
vector_sum = vector1 + vector2
print(vector_sum)  # [5, 7, 9]

# 向量减法
vector_diff = vector1 - vector2
print(vector_diff)  # [-3, -3, -3]
```

```python
import numpy as np

# 创建两个矩阵
matrix1 = np.array([[1, 2], [3, 4]])
matrix2 = np.array([[5, 6], [7, 8]])

# 矩阵加法
matrix_sum = matrix1 + matrix2
print(matrix_sum)  # [[ 6,  8], [10, 12]]

# 矩阵减法
matrix_diff = matrix1 - matrix2
print(matrix_diff)  # [[-4, -6], [-4, -6]]
```

## 4.2 向量和矩阵的数乘

```python
import numpy as np

vector = np.array([1, 2, 3])
```

```python
import numpy as np

matrix = np.array([[1, 2], [3, 4]])
```

## 4.3 矩阵的逆矩阵

```python
import numpy as np

# 创建一个矩阵
matrix = np.array([[1, 2], [3, 4]])

# 计算逆矩阵
inverse_matrix = np.linalg.inv(matrix)
print(inverse_matrix)  # [[-0.5, 0.5], [1.5, -1.5]]
```

## 4.4 矩阵的秩

```python
import numpy as np

# 创建一个矩阵
matrix = np.array([[1, 2], [3, 4]])

# 计算秩
rank = np.linalg.matrix_rank(matrix)
print(rank)  # 1
```

## 4.5 矩阵的行列式

```python
import numpy as np

# 创建一个矩阵
matrix = np.array([[1, 2], [3, 4]])

# 计算行列式
determinant = np.linalg.det(matrix)
print(determinant)  # -2
```
