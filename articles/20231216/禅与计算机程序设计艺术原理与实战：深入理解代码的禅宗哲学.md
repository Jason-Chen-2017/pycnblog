## 3.1 排序算法

排序算法是计算机程序设计中的基础知识，常用于对数据进行排序。以下是排序算法的核心原理和具体操作步骤：

### 3.1.1 冒泡排序

冒泡排序（Bubble Sort）是一种简单的排序算法，其核心原理是通过多次交换相邻的元素，将较大的元素逐渐冒泡到数组的末尾。具体操作步骤如下：

1. 从数组的第一个元素开始，与相邻的元素进行比较。
2. 如果当前元素较大，则交换当前元素和相邻元素的位置。
3. 重复上述操作，直到整个数组被排序。

### 3.1.2 选择排序

选择排序（Selection Sort）是一种简单的排序算法，其核心原理是通过多次选择数组中最小的元素，将其放到数组的开头。具体操作步骤如下：

1. 从数组的第一个元素开始，找到最小的元素。
2. 将最小的元素与当前元素交换位置。
3. 重复上述操作，直到整个数组被排序。

### 3.1.3 插入排序

插入排序（Insertion Sort）是一种简单的排序算法，其核心原理是通过多次将新元素插入到已排序的数组中，将其放到正确的位置。具体操作步骤如下：

1. 从数组的第二个元素开始，将其视为新元素。
2. 将新元素与已排序的元素进行比较，将其插入到正确的位置。
3. 重复上述操作，直到整个数组被排序。

## 3.2 搜索算法

搜索算法是计算机程序设计中的基础知识，常用于对数据进行搜索。以下是搜索算法的核心原理和具体操作步骤：

### 3.2.1 线性搜索

线性搜索（Linear Search）是一种简单的搜索算法，其核心原理是通过遍历数组中的每个元素，直到找到目标元素。具体操作步骤如下：

1. 从数组的第一个元素开始，逐个遍历每个元素。
2. 如果当前元素与目标元素相等，则返回当前元素的索引。
3. 如果遍历完整个数组仍未找到目标元素，则返回-1。

### 3.2.2 二分搜索

二分搜索（Binary Search）是一种高效的搜索算法，其核心原理是通过将数组分为两个部分，并根据目标元素与中间元素的关系，不断缩小搜索范围。具体操作步骤如下：

1. 将数组分为两个部分，左半部分和右半部分。
2. 找到中间元素，与目标元素进行比较。
3. 如果中间元素与目标元素相等，则返回中间元素的索引。
4. 如果中间元素较小，则将搜索范围设为右半部分，重复上述操作。
5. 如果中间元素较大，则将搜索范围设为左半部分，重复上述操作。
6. 如果搜索范围被缩小到零仍未找到目标元素，则返回-1。

## 4.1 冒泡排序实例

以下是冒泡排序的具体代码实例：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在上述代码中，我们首先获取数组的长度，然后进行多次遍历。在每次遍历中，我们将当前元素与相邻元素进行比较，如果当前元素较大，则交换其位置。通过多次遍历，我们可以将较大的元素逐渐冒泡到数组的末尾，从而实现排序。

## 4.2 选择排序实例

以下是选择排序的具体代码实例：

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```

在上述代码中，我们首先获取数组的长度，然后进行多次遍历。在每次遍历中，我们将当前元素视为最小元素，并将其与已排序的元素进行比较。如果当前元素较小，则交换其位置。通过多次遍历，我们可以将最小的元素逐渐放到数组的开头，从而实现排序。

## 4.3 插入排序实例

以下是插入排序的具体代码实例：

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i-1
        while j >= 0 and key < arr[j]:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr
```

在上述代码中，我们首先遍历数组中的每个元素，将其视为新元素。在每次遍历中，我们将新元素与已排序的元素进行比较，将其插入到正确的位置。通过多次遍历，我们可以将所有元素插入到已排序的数组中，从而实现排序。
