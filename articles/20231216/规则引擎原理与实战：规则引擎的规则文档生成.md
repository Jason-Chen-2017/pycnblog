# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个具体的代码实例来详细解释规则引擎的实现。我们将使用Python编程语言来实现一个简单的规则引擎。

```python
import re

# 定义规则
rules = [
    {
        'condition': r'^(.*?) (疑似病例).*$',
        'action': '报告疑似病例'
    },
    {
        'condition': r'^(.*?) (确诊病例).*$',
        'action': '报告确诊病例'
    }
]

# 定义事实
facts = [
    '患者张三在医院就诊，疑似患上了新冠病毒感染',
    '患者李四在医院就诊，已确诊为新冠病毒感染'
]

# 规则匹配
for fact in facts:
    for rule in rules:
        if re.match(rule['condition'], fact):
            # 规则执行
            print(rule['action'])
```

在上面的代码中，我们首先定义了一组规则，其中包含了条件和动作。然后，我们定义了一组事实，这些事实将作为规则的输入。接下来，我们遍历了事实和规则，根据规则的条件来匹配事实。如果事实匹配了规则的条件，则执行规则的动作。最后，我们将执行的动作打印出来。
