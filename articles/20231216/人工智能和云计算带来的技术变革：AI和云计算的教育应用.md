## 4.1机器学习的代码实例

以下是一个简单的线性回归的代码实例：

```python
import numpy as np

# 生成数据
x = np.random.rand(100, 1)
y = 3 * x + np.random.rand(100, 1)

# 训练模型
theta = np.random.rand(1, 1)
learning_rate = 0.01
num_iterations = 1000

for i in range(num_iterations):
    hypothesis = theta * x
    cost = (hypothesis - y)**2
    gradient = 2 * (hypothesis - y) * x
    theta = theta - learning_rate * gradient

# 预测
x_test = np.array([[0.5], [1.5]])
prediction = theta * x_test
print(prediction)
```

在这个代码实例中，我们首先生成了一组随机的数据。然后，我们初始化了一个随机的参数 $\theta$，并设置了一个学习率和迭代次数。接下来，我们使用梯度下降法来训练模型，并计算出最终的参数 $\theta$。最后，我们使用训练好的模型来预测新的数据。

## 4.2深度学习的代码实例

以下是一个简单的卷积神经网络的代码实例：

```python
import torch
import torch.nn as nn
import torch.optim as optim

# 生成数据
x = torch.randn(100, 1, 28, 28)
y = torch.randn(100, 10)

# 定义模型
class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.conv1 = nn.Conv2d(1, 10, 5)
        self.fc1 = nn.Linear(10 * 28 * 28, 10)

    def forward(self, x):
        x = self.conv1(x)
        x = nn.functional.relu(x)
        x = x.view(-1, 10 * 28 * 28)
        x = self.fc1(x)
        return x

# 训练模型
net = Net()
criterion = nn.MSELoss()
optimizer = optim.SGD(net.parameters(), lr=0.01)

for i in range(1000):
    optimizer.zero_grad()
    output = net(x)
    loss = criterion(output, y)
    loss.backward()
    optimizer.step()

# 预测
x_test = torch.randn(1, 1, 28, 28)
prediction = net(x_test)
print(prediction)
```

在这个代码实例中，我们首先生成了一组随机的数据。然后，我们定义了一个简单的卷积神经网络模型，并设置了一个损失函数和优化器。接下来，我们使用梯度下降法来训练模型，并计算出最终的参数。最后，我们使用训练好的模型来预测新的数据。
