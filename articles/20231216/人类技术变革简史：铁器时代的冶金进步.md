# 4.具体代码实例和详细解释说明

在这里，我们将给出一个简单的矿物磨砂算法的代码实例，以及其对应的解释说明。

```python
import numpy as np

def mill_stone(stone, mill_time):
    milled_stone = stone.copy()
    for _ in range(mill_time):
        milled_stone = np.multiply(milled_stone, 0.99)
    return milled_stone
```

在这个代码实例中，我们使用了NumPy库来实现矿物磨砂算法。函数`mill_stone`接受两个参数：矿物`stone`和磨砂时间`mill_time`。在函数内部，我们使用了`np.multiply`函数来实现矿物磨砂的过程，将矿物与0.99相乘，表示矿物的质量减少。通过循环执行这个过程，我们可以实现矿物的磨砂。
