# 2.核心概念与联系

在深度学习推荐系统中，我们需要关注以下几个核心概念：

1. 用户行为数据：包括用户的浏览、购买、点赞等行为数据，用于构建用户的兴趣模型。
2. 商品特征数据：包括商品的属性、价格、评价等信息，用于构建商品的特征模型。
3. 用户兴趣模型：通过分析用户的行为数据，构建用户的兴趣模型，以预测用户对某个商品的喜好。
4. 商品特征模型：通过分析商品的特征数据，构建商品的特征模型，以评估商品的推荐度。
5. 推荐算法：将用户兴趣模型和商品特征模型结合，预测用户对某个商品的喜好，从而生成推荐列表。
