                 

# 1.背景介绍

云计算是一种基于互联网的计算资源分配和共享方式，它可以让用户在不需要购买硬件和软件的前提下，通过网络来使用计算资源。云计算的核心思想是将计算资源、存储资源和网络资源等基础设施通过互联网进行集中管理和分配，从而实现资源的共享和灵活调配。
