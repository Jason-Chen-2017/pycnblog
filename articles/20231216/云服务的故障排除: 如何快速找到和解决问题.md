```python
import traceback

def handle_exception(exc_type, exc_value, exc_traceback):
    logging.error("Uncaught exception", exc_info=(exc_type, exc_value, exc_traceback))
    traceback.print_exception(exc_type, exc_value, exc_traceback)

try:
    # 这里是我们的业务代码
    # 如果出现异常，则会被捕获并处理
except Exception as e:
    handle_exception(type(e), e, traceback.format_exc())
```

在这个代码实例中，我们使用了Python的logging和traceback模块来捕获和处理异常。当业务代码中出现异常时，这个代码会将异常信息记录到日志中，并将异常堆栈信息打印到控制台。这样，我们可以更容易地找到问题的具体位置和原因，并采取相应的解决方案。
