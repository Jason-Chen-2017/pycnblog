```python
# 数据收集
data = pd.read_csv('data.csv')

# 数据预处理
data = data.dropna()

# 数据处理
data['new_column'] = data['column1'] + data['column2']

# 数据存储
data.to_csv('processed_data.csv', index=False)

# 数据查询
query_result = data[data['new_column'] > 100]
```

在这个代码实例中，我们使用了pandas库来处理数据。首先，我们使用`read_csv`函数从CSV文件中读取数据。然后，我们使用`dropna`函数删除缺失值。接下来，我们使用`new_column`函数计算新的列值。最后，我们使用`to_csv`函数将处理后的数据存储到CSV文件中。最后，我们使用`query`函数对处理后的数据进行查询。
