## 4.1 基因组序列比对

### 4.1.1 使用Needleman-Wunsch算法进行全序列比对

```python
def needleman_wunsch(seq1, seq2):
    len1, len2 = len(seq1), len(seq2)
    dp = [[0] * (len2 + 1) for _ in range(len1 + 1)]
    for i in range(1, len1 + 1):
        for j in range(1, len2 + 1):
            if seq1[i - 1] == seq2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j] - 1, dp[i][j - 1] - 1, dp[i - 1][j - 1])
    return dp[-1][-1]
```

### 4.1.2 使用Smith-Waterman算法进行全序列比对

```python
def smith_waterman(seq1, seq2):
    len1, len2 = len(seq1), len(seq2)
    dp = [[0] * (len2 + 1) for _ in range(len1 + 1)]
    for i in range(1, len1 + 1):
        for j in range(1, len2 + 1):
            match_score = 1 if seq1[i - 1] == seq2[j - 1] else -1
            dp[i][j] = max(dp[i - 1][j] + match_score, dp[i][j - 1] + match_score, dp[i - 1][j - 1] + match_score)
    return dp[-1][-1]
```
