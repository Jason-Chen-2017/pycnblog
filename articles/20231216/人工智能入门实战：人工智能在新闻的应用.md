# 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的新闻筛选与推荐的例子来详细解释代码实现。

假设我们有一组新闻文本，我们希望将其划分为两个类别：政治新闻和经济新闻。首先，我们需要将新闻文本转换为数字形式，即将文本转换为向量。我们可以使用TF-IDF（Term Frequency-Inverse Document Frequency）技术来实现这一目标。

```python
from sklearn.feature_extraction.text import TfidfVectorizer

news_data = ['政治新闻1', '政治新闻2', '经济新闻1', '经济新闻2']
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(news_data)
```

接下来，我们需要训练一个机器学习模型来对新闻文本进行分类。我们可以使用支持向量机（SVM）来实现这一目标。

```python
from sklearn.svm import SVC

y = [0, 0, 1, 1]  # 0表示政治新闻，1表示经济新闻
clf = SVC()
clf.fit(X, y)
```

最后，我们可以使用训练好的模型对新闻文本进行筛选与推荐。

```python
new_news = '新闻3'
new_news_vector = vectorizer.transform([new_news])
prediction = clf.predict(new_news_vector)
print(prediction)  # 输出：[0]，表示新闻为政治新闻
```
