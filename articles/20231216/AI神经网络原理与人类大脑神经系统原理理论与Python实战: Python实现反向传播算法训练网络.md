## 1.1 背景介绍

人类大脑是一个复杂的神经系统，由大约100亿个神经元组成，这些神经元通过复杂的连接和信息传递来完成各种认知和行为任务。神经网络模拟了大脑神经元的结构和功能，通过学习和调整权重来逐渐学习和完成任务。

1943年，美国心理学家伯纳德·马克勒曼（Bernard Widrow）和他的学生马丁·德勒（Martin Arbib）提出了一种名为“适应性自动机”（Adaptive Neural Network）的神经网络结构，这是第一种实际应用的神经网络。随后，其他研究人员对神经网络进行了进一步的研究和发展，包括马尔科夫神经网络、反馈神经网络等。

1986年，美国的马克·喃普特（Geoffrey Hinton）和他的团队在深度学习领域取得了重要的突破，他们提出了“反向传播”（Backpropagation）算法，这是训练神经网络的主要算法之一。随后，深度学习开始广泛应用于计算机视觉、自然语言处理、语音识别等领域，取得了显著的成果。
