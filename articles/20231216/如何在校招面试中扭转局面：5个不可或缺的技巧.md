## 4.1 代码实例1：算法原理
```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[0]
        less = [i for i in arr[1:] if i <= pivot]
        greater = [i for i in arr[1:] if i > pivot]
        return quick_sort(less) + [pivot] + quick_sort(greater)
```
在这个代码实例中，我们实现了一个快速排序算法。快速排序是一种常用的排序算法，它的时间复杂度为O(nlogn)。我们可以看到，这个算法的核心思想是将数组分为两个部分，一个部分包含小于基准值的元素，另一个部分包含大于基准值的元素。然后我们递归地对这两个部分进行排序。
