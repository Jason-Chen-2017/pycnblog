# 2.核心概念与联系

## 2.1 神经网络基本结构

神经网络由多个节点（神经元）和连接这些节点的权重组成。每个节点都接收来自其他节点的输入，并根据其内部权重和偏置计算输出。这些输出然后传递给下一个节点，直到整个网络被遍历。


## 2.2 激活函数

激活函数是神经网络中的一个关键组件，它用于将输入节点的输出映射到输出节点。常见的激活函数包括Sigmoid、Tanh和ReLU等。这些函数可以帮助神经网络学习非线性关系，从而提高其表现力。

## 2.3 损失函数

损失函数用于衡量模型预测值与真实值之间的差距。常见的损失函数包括均方误差（MSE）、交叉熵损失（Cross-Entropy Loss）等。通过优化损失函数，我们可以调整神经网络的参数以提高其预测能力。

## 2.4 反向传播

反向传播是训练神经网络的一个关键步骤。它涉及到从输出层向输入层传播的梯度信息，以便调整权重和偏置。通过反复进行这个过程，神经网络可以逐步学习并优化其参数。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 前向传播

前向传播是神经网络中的一个关键步骤，它用于将输入数据传递到输出层。具体步骤如下：

1. 对输入数据进行标准化处理，将其转换为相同的范围。
2. 将标准化后的输入数据传递给第一个隐藏层。
3. 对隐藏层的每个节点进行计算，公式为：$$ a_j = f(\sum_{i} w_{ij} * x_i + b_j) $$
4. 将隐藏层的输出传递给下一个隐藏层或输出层，重复步骤3。
5. 对输出层的节点进行计算，得到最终的预测值。

## 3.2 后向传播

后向传播是训练神经网络的另一个关键步骤，它用于计算梯度信息并调整权重和偏置。具体步骤如下：

1. 计算输出层的损失值。
2. 对输出层的每个节点进行计算，公式为：$$ \delta_j = \frac{\partial L}{\partial a_j} * f'(a_j) $$
3. 将梯度信息传递回前一个隐藏层。
4. 对每个隐藏层的节点进行计算，公式为：$$ \delta_i = \sum_{j} \delta_j * w_{ij} $$
5. 更新权重和偏置，公式为：$$ w_{ij} = w_{ij} - \eta * \delta_j * x_i $$ $$ b_j = b_j - \eta * \delta_j $$

## 3.3 优化算法

在训练神经网络时，我们需要选择一个优化算法来调整权重和偏置。常见的优化算法包括梯度下降（Gradient Descent）、随机梯度下降（Stochastic Gradient Descent）、动量（Momentum）和Adam等。这些算法可以帮助我们更快地找到最优的参数值。
