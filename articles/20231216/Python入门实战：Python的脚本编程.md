## 2.核心概念与联系

### 2.1 Python脚本编程的核心概念

Python脚本编程的核心概念包括：

1.脚本：Python脚本是一种简单的程序，用于自动化执行一些重复的任务。它通常由一系列Python代码组成，可以通过命令行或其他方式执行。

2.模块：Python模块是一种包含一组相关功能的Python文件。模块可以被导入到其他Python脚本中，以便重用代码和提高代码的可读性和可维护性。

3.函数：Python函数是一种可重用的代码块，用于执行特定的任务。函数可以接受输入参数，并返回输出结果。

4.变量：Python变量是一种用于存储数据的容器。变量可以用于存储各种类型的数据，如整数、浮点数、字符串、列表等。

## 4.具体代码实例和详细解释说明

### 4.1 代码实例

以下是一个简单的Python脚本实例，用于读取一个文本文件，并统计其中每个单词的出现次数：

```python
import sys

def count_words(file_path):
    word_count = {}
    with open(file_path, 'r') as file:
        for line in file:
            words = line.split()
            for word in words:
                if word in word_count:
                    word_count[word] += 1
                else:
                    word_count[word] = 1
    return word_count

if __name__ == '__main__':
    file_path = sys.argv[1]
    word_count = count_words(file_path)
    for word, count in word_count.items():
        print(f'{word}: {count}')
```

### 4.2 详细解释说明

1.导入`sys`模块：`sys`模块提供了一些用于访问和操作系统的接口，如命令行参数和文件操作。

2.定义`count_words`函数：`count_words`函数用于读取文件，并统计每个单词的出现次数。函数接受一个文件路径作为参数，并返回一个字典，其中键为单词，值为出现次数。

3.使用`with`语句打开文件：`with`语句可以确保文件在操作完成后自动关闭，避免资源泄漏。

4.遍历文件中的每行：通过`for`循环遍历文件中的每行，并将其拆分为单词。

5.统计单词出现次数：通过将单词作为键添加到`word_count`字典中，并更新其值。

6.输出结果：通过`for`循环遍历`word_count`字典，并将结果打印到命令行上。

### 4.3 运行脚本

要运行此脚本，请执行以下命令：

```
python script.py file.txt
```

其中`script.py`是脚本文件名，`file.txt`是要处理的文本文件。
