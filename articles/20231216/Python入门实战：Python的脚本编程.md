## 4.具体代码实例和详细解释说明

### 4.1 文件操作示例

在本节中，我们将通过一个简单的文件操作示例来演示Python脚本编程的具体实现。

```python
# 打开文件
with open('example.txt', 'r') as f:
    # 读取文件内容
    content = f.read()
    # 处理文件内容
    words = content.split()
    # 统计单词出现次数
    word_count = {}
    for word in words:
        word_count[word] = word_count.get(word, 0) + 1
    # 打印单词出现次数
    for word, count in word_count.items():
        print(word, count)
```

上述代码首先使用`with`语句打开一个名为`example.txt`的文件，并读取其内容。然后，将文件内容按空格分割为单词列表，并使用字典来统计每个单词出现的次数。最后，使用`for`循环遍历字典，并打印每个单词及其出现次数。

### 4.2 网络请求示例

在本节中，我们将通过一个简单的网络请求示例来演示Python脚本编程的具体实现。

```python
import requests

# 发送GET请求
response = requests.get('https://api.github.com/users/octocat')
# 判断请求是否成功
if response.status_code == 200:
    # 解析JSON数据
    data = response.json()
    # 打印用户信息
    print(data['login'])
    print(data['name'])
else:
    print('请求失败')
```

上述代码首先使用`import`语句导入`requests`库，并发送一个GET请求到GitHub API的用户信息接口。然后，判断请求是否成功，如果成功，使用`json()`方法解析JSON数据，并打印用户的登录名和姓名。如果请求失败，打印“请求失败”。
