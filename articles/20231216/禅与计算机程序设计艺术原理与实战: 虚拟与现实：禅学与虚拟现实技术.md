我们可以选择一个排序算法，例如冒泡排序。冒泡排序的具体实现如下：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

在这个实例中，我们定义了一个冒泡排序函数，它接收一个数组作为参数，并将数组进行排序。冒泡排序的核心原理是通过比较相邻的元素，将较大的元素向右移动。

### 4.2 搜索算法实例

我们可以选择一个搜索算法，例如二分搜索。二分搜索的具体实现如下：

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

在这个实例中，我们定义了一个二分搜索函数，它接收一个数组和一个目标值作为参数，并将数组进行查找。二分搜索的核心原理是通过比较中间元素与目标值，将搜索范围缩小。
