# 4.具体代码实例和详细解释说明
# 4.1 代码实例
在这里，我们将给出一个简单的代码实例，它实现了一个简单的排序算法：

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

# 4.2 详细解释说明
这个代码实例实现了一个简单的冒泡排序算法，它的原理是通过多次遍历数组，将较大的元素向后移动，直到数组有序为止。具体来说，它的步骤如下：

1. 定义一个名为`bubble_sort`的函数，它接受一个数组`arr`作为参数。
2. 获取数组的长度`n`。
3. 使用一个`for`循环遍历数组，每次遍历都从数组的第一个元素开始。
4. 使用一个`for`循环遍历数组，每次遍历都从数组的第一个元素开始，并且只遍历到数组的最后一个元素。
5. 比较当前元素和下一个元素的值，如果当前元素大于下一个元素，则交换它们的值。
6. 重复步骤4和5，直到数组有序为止。
7. 返回排序后的数组。
