## 4.1 创建游戏窗口

首先，我们需要创建一个游戏窗口。我们可以使用Pygame库的set_mode()方法来实现这个功能。以下是一个简单的代码实例：

```python
import pygame

pygame.init()

# 创建游戏窗口
screen = pygame.display.set_mode((800, 600))
```

在这个代码中，我们首先导入了Pygame库，然后调用了init()方法来初始化Pygame库。接下来，我们使用set_mode()方法来创建一个游戏窗口，其大小为800x600像素。

## 4.4 处理用户输入

在这一步，我们需要处理用户的输入，例如按键、鼠标点击等。我们可以使用Pygame库的event()方法来处理这些输入。以下是一个简单的代码实例：

```python

    if event.type == pygame.QUIT:
        pygame.quit()
        sys.exit()
```

在这个代码中，我们使用event()方法来获取用户的输入事件。我们可以通过检查event.type来判断用户的输入类型。如果用户点击了关闭游戏窗口的按钮，我们可以使用pygame.quit()方法来关闭游戏窗口，并使用sys.exit()方法来结束程序。
