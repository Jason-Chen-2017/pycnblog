数学模型公式详细讲解：

1. MD5算法：MD5是一种广泛使用的哈希算法，它的输出长度为128位。其主要公式为：

$$
H(M) = H(M_1 || M_2 || ... || M_n)
$$

其中，$H(M)$ 是哈希值，$M$ 是输入数据，$M_1, M_2, ..., M_n$ 是数据的块，$||$ 表示字符串的连接操作。

2. SHA-1算法：SHA-1是一种安全性较高的哈希算法，其输出长度为160位。其主要公式为：

$$
H(M) = H(M_1 || M_2 || ... || M_n)
$$

其中，$H(M)$ 是哈希值，$M$ 是输入数据，$M_1, M_2, ..., M_n$ 是数据的块，$||$ 表示字符串的连接操作。

3. AES算法：AES是一种广泛使用的加密算法，其主要公式为：

$$
C = E_k(P) = E_k(P_1 || P_2 || ... || P_n)
$$

其中，$C$ 是密文，$E_k(P)$ 是加密操作，$k$ 是密钥，$P$ 是明文，$P_1, P_2, ..., P_n$ 是明文的块，$||$ 表示字符串的连接操作。

4. RSA算法：RSA是一种广泛使用的密码学算法，其主要公式为：

$$
C = E_n(M^e \mod n)
$$

$$
M = D_n(C^d \mod n)
$$

其中，$C$ 是密文，$M$ 是明文，$e$ 和 $d$ 是公钥和私钥，$n$ 是公钥和私钥的公共因数。

# 4.具体代码实例和详细解释说明
在这里，我们将提供一个简单的Python代码实例，演示如何使用MD5算法进行密码的加密和解密操作。

```python
import hashlib

def md5_encrypt(password):
    md5_hash = hashlib.md5()
    md5_hash.update(password.encode('utf-8'))
    return md5_hash.hexdigest()

def md5_decrypt(password, encrypted_password):
    md5_hash = hashlib.md5()
    md5_hash.update(password.encode('utf-8'))
    return md5_hash.hexdigest() == encrypted_password

password = "123456"
encrypted_password = md5_encrypt(password)
print(encrypted_password)

decrypted = md5_decrypt(password, encrypted_password)
print(decrypted)
```

这个代码实例首先导入了hashlib模块，然后定义了两个函数：`md5_encrypt` 和 `md5_decrypt`。`md5_encrypt` 函数用于将密码进行MD5加密，`md5_decrypt` 函数用于将加密后的密码进行解密。最后，我们使用了一个示例密码进行加密和解密操作，并输出了结果。
