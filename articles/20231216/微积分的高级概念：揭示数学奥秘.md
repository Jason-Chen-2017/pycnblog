## 2.核心概念与联系

### 2.1 微积分的基本概念

微积分的基本概念包括：微分、积分、极限、连续性等。这些概念是微积分的核心，也是微积分的基本工具。

- 微分：微分是将一个函数从一个点到另一个点的变化量，用于描述函数的变化速度。微分的符号为d，通常表示为f'(x)，其中f'(x)是函数f(x)的导数。

- 积分：积分是将一个函数从一个点到另一个点的累积变化量，用于描述函数的累积变化。积分的符号为∫，通常表示为∫f(x)dx，其中f(x)是函数，dx是一个虚小的变量。

- 极限：极限是在一个变量接近某个特定值时，另一个变量的值趋于某个特定值的概念。极限的符号为lim，通常表示为lim x→a f(x)，其中a是一个特定的值，f(x)是一个函数。

- 连续性：连续性是一个函数在某个点上的值与其邻近点的值之间存在连续性的概念。连续性的符号为C，通常表示为f(x)Cx=a，其中a是一个特定的值，f(x)是一个连续的函数。

### 4.1 代码实例：求导数

```python
import numpy as np

def derivative(f, x0, h=1e-8):
    return (f(x0 + h) - f(x0)) / h

def f(x):
    return x**2

x0 = 2
h = 1e-8
result = derivative(f, x0, h)
print(result)
```

在这个代码实例中，我们使用了Python的NumPy库来计算一个给定函数f(x)=x^2的导数。我们使用了一个简单的差分方法来计算导数，即将函数在某个点的值与其邻近点的值之间的差除以邻近点的距离。在这个例子中，我们选择了一个小的邻近点距离h=1e-8，以减少误差。最后，我们得到了导数的值result，并输出了结果。
