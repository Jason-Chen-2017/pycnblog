## 2.3 规则引擎与知识图谱

知识图谱是一种用于表示实体和关系的数据结构，它可以用于规则引擎的知识表示和推理。规则引擎可以通过访问知识图谱来获取实体和关系的信息，从而实现更复杂的决策逻辑。

```python
from rule_engine import RuleEngine

# 创建规则引擎实例
engine = RuleEngine()

# 添加规则
engine.add_rule("rule1", "IF age >= 30 AND income >= 50000 THEN bonus = 10000")
engine.add_rule("rule2", "IF age >= 40 AND income >= 80000 THEN bonus = 15000")

# 设置用户信息
user_info = {"age": 35, "income": 60000}

# 执行规则
bonus = engine.execute("bonus", user_info)
print("用户奖金：", bonus)
```
