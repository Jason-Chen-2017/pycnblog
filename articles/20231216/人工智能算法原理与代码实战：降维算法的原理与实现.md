# 2.核心概念与联系

在本节中，我们将介绍降维算法的核心概念和联系。降维算法的主要目标是将高维数据降到低维空间，以实现数据压缩、可视化和清洗等目的。降维算法的核心概念包括：

1. 高维数据：高维数据是指具有多个特征的数据，这些特征可以是数值、文本、图像等。高维数据的一个典型例子是人脸识别，人脸识别需要考虑到人脸的多个特征，如眼睛、鼻子、嘴巴等。

2. 低维数据：低维数据是指具有少量特征的数据。低维数据的一个典型例子是人脸识别的简化版，只考虑到人脸的一些关键特征，如眼睛、鼻子等。

3. 数据压缩：数据压缩是指将高维数据压缩到低维空间，以减少数据的存储空间和传输开销。数据压缩的一个典型例子是图像压缩，将高质量的图像压缩到低质量的图像，以减少存储空间和传输开销。

4. 数据可视化：数据可视化是指将高维数据转换为低维空间，以便人类更容易理解和可视化。数据可视化的一个典型例子是地图可视化，将高维地理数据转换为二维地图，以便人类更容易理解和可视化。

5. 数据清洗：数据清洗是指将高维数据清洗为低维数据，以去除噪声和冗余信息。数据清洗的一个典型例子是文本清洗，将高维文本数据清洗为低维文本数据，以去除噪声和冗余信息。
