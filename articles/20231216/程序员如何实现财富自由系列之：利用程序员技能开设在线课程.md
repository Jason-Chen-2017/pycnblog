## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的例子，演示如何编写代码实例和解释说明。

假设我们要教授一个基本的排序算法——冒泡排序（Bubble Sort）。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

上述代码实现了冒泡排序算法。首先，我们定义了一个名为 `bubble_sort` 的函数，接收一个列表 `arr` 作为参数。然后，我们使用两层循环遍历列表中的每个元素。如果当前元素大于下一个元素，我们交换它们的位置。最后，我们返回排序后的列表。

解释说明：

冒泡排序是一种简单的比较排序算法，它的时间复杂度为 O(n^2)。在每一次遍历中，它会将最大的元素“冒泡”到列表的末尾。通过重复这个过程，直到列表被完全排序为止。
