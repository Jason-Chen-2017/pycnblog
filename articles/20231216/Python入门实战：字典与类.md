# 2.核心概念与联系

- 值（Value）：字典中存储的数据。
- 键值对（Key-Value Pair）：字典中的基本单位，由一个键和一个值组成。

字典和其他数据结构之间的联系如下：

- 列表（List）：字典类似于列表，但是字典使用键值对来存储数据，而列表使用索引来存储数据。

- 元组（Tuple）：字典和元组都可以存储数据，但是元组的元素是不可变的，而字典的元素是可变的。
