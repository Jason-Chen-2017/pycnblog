磁化强度（B）是磁场中磁场线密度的矢量量，单位是特别大（T）。磁化率（σ）是磁化强度与磁化场强度（H）之间的关系，单位是无量纲。磁化曲线是磁化强度与磁化场强度之间的关系曲线，用于描述磁性材料在不同磁场下的性能特征。

$$

2. 测量磁化矩度（M）。
3. 将测量到的磁化矩度（M）与磁化场强度（H）代入公式，计算磁化强度（B）。

## 3.2 磁化率计算
磁化率（σ）可以通过以下公式计算：

$$

3. 将测量到的磁化强度（B）与磁化场强度（H）代入公式，计算磁化率（σ）。

```python
import numpy as np
import matplotlib.pyplot as plt

# 测试数据
H = np.array([1000, 2000, 3000, 4000, 5000])
M = np.array([100, 200, 300, 400, 500])

# 计算磁化强度
B = np.multiply(np.multiply(4 * np.pi * 1e-7, H), M)

# 计算磁化率
sigma = B / H

# 绘制磁化曲线
plt.plot(H, B, label='磁化强度')
plt.plot(H, sigma, label='磁化率')
plt.xlabel('磁化场强度（H）')
plt.ylabel('磁化强度（B）/磁化率（σ）')
plt.legend()
plt.show()
```
