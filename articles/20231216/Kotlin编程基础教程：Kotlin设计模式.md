```kotlin
// 定义一个接口
interface Animal {
    fun speak()
}

// 定义一个类实现接口
class Dog : Animal {
    override fun speak() {
        println("汪汪汪")
    }
}

// 定义一个类实现接口
class Cat : Animal {
    override fun speak() {
        println("喵喵喵")
    }
}

// 定义一个类实现观察者模式
class Observer(private val animal: Animal) {
    fun listen() {
        animal.speak()
    }
}

// 定义一个类实现依赖注入
class Injector {
    private val animals = mutableListOf<Animal>()

    fun addAnimal(animal: Animal) {
        animals.add(animal)
    }

    fun getAnimal(index: Int): Animal {
        return animals[index]
    }
}

fun main() {
    // 创建一个注入器
    val injector = Injector()

    // 添加一个狗
    injector.addAnimal(Dog())

    // 添加一个猫
    injector.addAnimal(Cat())

    // 获取一个狗
    val dog = injector.getAnimal(0)

    // 获取一个猫
    val cat = injector.getAnimal(1)

    // 创建一个观察者
    val observer = Observer(dog)

    // 观察狗的声音
    observer.listen()

    // 观察猫的声音
    observer.listen()
}
```

在这个代码实例中，我们首先定义了一个接口`Animal`，它包含了一个`speak`方法。然后我们定义了两个类`Dog`和`Cat`，它们 respective实现了`Animal`接口的`speak`方法。接下来，我们定义了一个`Observer`类，它实现了观察者模式，并包含了一个`listen`方法。最后，我们定义了一个`Injector`类，它实现了依赖注入，并包含了`addAnimal`和`getAnimal`方法。

在`main`函数中，我们创建了一个`Injector`实例，并添加了一个狗和一个猫。然后我们获取了一个狗和一个猫的实例，并创建了一个`Observer`实例。最后，我们使用`Observer`实例观察狗和猫的声音。
