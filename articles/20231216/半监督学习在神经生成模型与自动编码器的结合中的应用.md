# 2.核心概念与联系

## 2.1 半监督学习
半监督学习是一种学习方法，它使用有限的标注数据和大量的无标注数据进行学习。半监督学习的目标是找到一个模型，使得在有标注数据上的损失函数最小，同时在无标注数据上的损失函数最小或者最小化。半监督学习可以在有监督学习和无监督学习之间取得平衡，从而更好地利用有限的标注数据和大量的无标注数据。

## 2.2 神经生成模型
神经生成模型是一种生成模型，它使用神经网络来模拟数据生成过程。神经生成模型可以生成新的数据，同时也可以用于数据增强、数据生成等任务。神经生成模型的优势在于它可以学习数据的复杂结构，同时也可以生成高质量的数据。

## 2.3 自动编码器
自动编码器是一种无监督学习方法，它的目标是找到一个编码器和一个解码器，使得编码器可以将输入数据压缩为低维的编码，解码器可以将编码恢复为原始数据。自动编码器可以用于降维、数据压缩等任务。自动编码器的优势在于它可以学习数据的低维结构，同时也可以用于数据压缩和降维等任务。
