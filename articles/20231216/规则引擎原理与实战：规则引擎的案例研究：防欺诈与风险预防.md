- 如果用户在过去3个月内有多笔高额交易，并且交易来源地址与用户注册地址不匹配，则认为存在欺诈风险。

- 如果用户在过去3个月内有多笔高额交易，并且交易来源地址与用户注册地址匹配，则认为不存在欺诈风险。

我们可以使用Python编写如下代码实现：

```python

def check_fraud(user_info, transaction_info):
    # 获取用户的注册地址
    user_address = user_info['register_address']

    # 获取用户在过去3个月内的交易记录
    transactions = user_info['transactions']

    # 判断是否存在欺诈风险
    risk_level = 0
    for transaction in transactions:
        if transaction['amount'] > 1000 and transaction['source_address'] != user_address:
            risk_level += 1
        elif transaction['amount'] > 500 and transaction['source_address'] == user_address:
            risk_level += 1

    # 判断风险等级
    if risk_level >= 3:
        return '高风险'
    elif risk_level >= 1:
        return '中风险'
    else:
        return '低风险'
```
