### 2.3 推导规则

以下是一个使用SymPy库进行形式推理的例子：

```python
from sympy import symbols, Eq, solve

# 定义符号
a, b = symbols('a b')

# 定义公式
expr1 = Eq(a**2, b**2 + 2*b + 1)
expr2 = Eq(a - b, 1)

# 推导过程
solution = solve([expr1, expr2], (a, b))
print("解集：", solution)
```

在这个例子中，我们首先定义了符号a和b，然后定义了两个公式expr1和expr2。接着，我们使用solve函数求解这两个公式的解，得到解集。
