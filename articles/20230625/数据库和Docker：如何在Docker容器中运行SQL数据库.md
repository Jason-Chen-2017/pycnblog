- 2.1. 基本概念解释

Docker是一个开源的容器化平台，它的核心思想是将应用程序和底层系统分离，从而方便应用程序的部署、扩展和管理。Docker 镜像是一组文件集合，包含了应用程序和底层系统的源代码、依赖项和配置信息，可以被多个Docker容器运行。

SQL数据库是一种关系型数据库，它的数据以表格的形式存储，支持SQL语言进行数据的查询、插入、更新和删除操作。在Docker容器中运行SQL数据库，需要将SQL数据库的源文件和依赖项安装到Docker容器中，然后在容器中运行SQL语句进行数据操作。
