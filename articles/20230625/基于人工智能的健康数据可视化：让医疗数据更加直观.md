- 示例1：展示心率和血压的数据
```
import pandas as pd
import matplotlib.pyplot as plt

# 读取心率和血压的数据
heart_rate = pd.read_csv('heart_rate.csv')
blood_pressure = pd.read_csv('blood_pressure.csv')

# 绘制心率和血压的图表
plt.plot(heart_rate.index, heart_rate.values)
plt.plot(blood_pressure.index, blood_pressure.values)
plt.title('Heart Rate and Blood Pressure')
plt.xlabel('Time')
plt.ylabel('Heart Rate')
plt.show()
```
