### 3.3数学模型公式详细讲解

在数据驱动的决策中，主要使用的数学模型有以下几种：

- 线性回归模型：用于对线性关系进行建模和预测。公式为：$$ y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_n x_n $$
- 多项式回归模型：用于对多项式关系进行建模和预测。公式为：$$ y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_n x_n^2 $$
- 逻辑回归模型：用于对二分类问题进行建模和预测。公式为：$$ P(y=1|x_1,x_2,\cdots,x_n) = \frac{1}{1+e^{-\beta_0-\beta_1 x_1-\beta_2 x_2-\cdots-\beta_n x_n}} $$

- 决策树模型：用于对决策问题进行建模和预测。公式为：$$ \text{if } x_1 \text{ then } y = a \text{ else if } x_2 \text{ then } y = b \text{ else } \cdots $$
