Zero Code 是一种基于 Web 的低代码平台，它使用 HTML、CSS、JavaScript 等技术来开发应用程序。Zero Code 平台提供了一系列预定义的组件，如按钮、文本框、列表等，用户可以通过拖拽和点击来组合这些组件，实现应用程序的开发。
