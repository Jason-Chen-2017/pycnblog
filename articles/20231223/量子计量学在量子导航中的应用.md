### 4.1 量子计量学的具体代码实例

```python
import numpy as np
import scipy.linalg

# 准备量子系统
alpha = np.sqrt(0.6)
beta = np.sqrt(0.4)
psi = alpha * np.array([1, 0]) + beta * np.array([0, 1])

# 实现量子测量
M = np.array([[0, 0], [0, 1]])
result = np.inner(psi, M.dot(psi.conj().T))

# 处理测量结果
print("测量结果: ", result)
```
