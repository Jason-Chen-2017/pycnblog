## 3.1 边界填充算法原理
边界填充算法的主要目标是消除图像边界效应，提高图像的质量。边界效应是指在图像边界处，由于像素值的限制，导致图像质量下降的现象。通过边界填充算法，我们可以在图像边界处填充合适的像素值，从而消除边界效应。

## 3.2 边界填充算法具体操作步骤
1. 读取输入图像。
2. 判断图像边界像素是否需要填充。
3. 如果需要填充，则根据算法规则填充合适的像素值。
4. 保存填充后的图像。

## 3.3 边界填充算法数学模型公式
边界填充算法的数学模型公式主要包括以下几种：

1. 平均填充：将图像边界像素的像素值设为图像内部像素值的平均值。
$$
P_{border}(x, y) = \frac{1}{N} \sum_{i=1}^{N} P(i, j)
$$

2. 均值滤波填充：将图像边界像素的像素值设为周围邻域像素值的均值。
$$
P_{border}(x, y) = \frac{1}{M} \sum_{i=-m}^{m} \sum_{j=-n}^{n} P(x+i, y+j)
$$

3. 最邻近填充：将图像边界像素的像素值设为最邻近的图像内部像素值。
$$
P_{border}(x, y) = \underset{i, j}{\text{argmin}} \| P(i, j) - P(x, y) \|
$$

```python
import cv2

def average_fill(image):
    rows, cols = image.shape[:2]
    for i in range(rows):
        for j in range(cols):
            if i == 0 or j == 0 or i == rows - 1 or j == cols - 1:
                avg_value = np.mean(image[1:rows-1, 1:cols-1])
                image[i, j] = avg_value
    return image

    filled_image = average_fill(image)

if __name__ == '__main__':
    main()
```

```python
import numpy as np
import cv2

def contrast_adjust(image, factor):
    rows, cols, channels = image.shape
    adjusted_image = np.zeros((rows, cols, channels), dtype=np.uint8)
    for i in range(rows):
        for j in range(cols):
            for k in range(channels):
                adjusted_image[i, j, k] = np.clip(image[i, j, k] * factor + 128, 0, 255)
    return adjusted_image

def main():
    adjusted_image = contrast_adjust(image, 2)

if __name__ == '__main__':
    main()
```
