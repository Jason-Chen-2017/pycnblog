# 2.核心概念与联系

在了解不同项目管理工具之前，我们需要了解一些核心概念。这些概念包括：

## 4.1 Trello

Trello使用简单的数据结构来存储和组织项目信息。以下是一个简单的Python代码实例，展示了如何使用字典来表示Trello项目板的结构：

```python
class TrelloBoard:
    def __init__(self):
        self.lists = {}

    def add_list(self, name):
        self.lists[name] = []

    def add_card(self, list_name, name):
        self.lists[list_name].append(name)

    def move_card(self, from_list, to_list):
        if from_list in self.lists and to_list in self.lists:
            for card in self.lists[from_list]:
                self.lists[to_list].append(card)
            del self.lists[from_list]
```

这个代码实例定义了一个`TrelloBoard`类，用于表示Trello项目板的结构。通过调用`add_list`方法，可以添加列表，通过调用`add_card`方法，可以添加卡片，通过调用`move_card`方法，可以将卡片从一个列表移动到另一个列表。

class AsanaProject:

    def add_task(self, name):
