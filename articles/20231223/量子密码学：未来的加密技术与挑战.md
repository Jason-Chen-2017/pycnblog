# 2. 核心概念与联系

## 2.1 量子密码学与传统密码学的区别

传统密码学主要关注的是基于数字的加密技术，通常使用对称密钥算法（如AES）和非对称密钥算法（如RSA）来实现加密和解密。而量子密码学则利用量子物理学的特性，如纠缠和超位，为加密技术提供了新的手段。

## 2.2 量子密码学的核心概念

1. 量子比特：与传统的二进制比特不同，量子比特可以表示为0、1或两者综合的状态，这种状态被称为“叠加状态”。
2. 量子密钥交换：量子密钥交换是量子密码学中的一种密钥交换协议，它利用量子纠缠来实现安全的密钥交换。
3. 量子加密：量子加密是一种基于量子密码学的加密技术，它可以利用量子物理学的特性来实现更安全的加密。

## 3.1 量子密钥交换：BB84算法

BB84算法是量子密码学中最早的量子密钥交换算法，它由Bennett和Brassard在1984年提出。算法的核心思想是利用量子纠缠和基础状态的不可知性来实现安全的密钥交换。

### 3.1.1 算法原理

1.  Alice将每个比特都表示为两个量子比特，并随机选择每个量子比特的基础状态。然后，她将这些量子比特发送给Bob。
2.  Bob将每个量子比特以其实际的基础状态进行测量。如果测量结果相同，则保留；如果测量结果不同，则丢弃。
3.  Alice和Bob通过公共渠道交换测量结果的一部分，以同步他们的基础状态选择。
4.  Alice和Bob使用同步的基础状态选择来构建共享的密钥。

### 3.1.2 数学模型公式

假设Alice选择了基础状态$\ket{0}$和$\ket{1}$，并将其表示为两个量子比特的纠缠状态：

$$\ket{\Psi} = \frac{1}{\sqrt{2}}(\ket{00} + \ket{11})$$

Bob接收到的量子比特是$\ket{00}$或$\ket{11}$，但是他不知道Alice的基础状态选择。当Bob进行测量时，他将得到以下结果：

1. 如果Bob测量到$\ket{00}$，那么这个比特组成的共享密钥为$K_0 = 0$。
2. 如果Bob测量到$\ket{11}$，那么这个比特组成的共享密钥为$K_1 = 1$。

2. Bob测量$\ket{\Psi}$，得到测量结果$r_B$。
3. Alice和Bob通过公共渠道交换测量结果。

## 3.2 量子加密：一种基于量子门的加密算法

量子加密是一种基于量子密码学的加密技术，它可以利用量子物理学的特性来实现更安全的加密。以下是一个简单的量子加密算法的例子：

### 3.2.1 算法原理

1.  Alice将明文加密为量子状态，并将其发送给Bob。
2.  Bob使用量子门进行解密，得到明文。

### 3.2.2 数学模型公式

假设Alice将明文$M$加密为量子状态$\ket{\Psi}$，则$\ket{\Psi}$可以表示为：

$$\ket{\Psi} = \sum_{x \in M} \alpha_x \ket{x}$$

其中$\alpha_x$是复数系数，表示明文$x$在量子状态中的贡献。

当Bob使用量子门进行解密时，他将得到以下结果：

$$\ket{\Psi'} = \sum_{x \in M} \alpha_x \ket{x'} = \sum_{x \in M} \alpha_x U^{\dagger}(x) \ket{x}$$

其中$U^{\dagger}(x)$是解密操作，表示从明文$x$恢复原始明文。

### 3.2.3 具体操作步骤

1. Alice将明文$M$加密为量子状态$\ket{\Psi}$，并将其发送给Bob。
2. Bob使用量子门$U$进行解密，得到明文$M'$。
