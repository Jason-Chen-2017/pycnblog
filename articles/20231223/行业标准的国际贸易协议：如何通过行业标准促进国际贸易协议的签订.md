# 2.核心概念与联系

在讨论行业标准如何促进国际贸易协议的签订之前，我们需要了解一些核心概念。

## 2.1 行业标准

行业标准是一种通用的技术规格、测量方法或工程实践，它们为特定行业提供了一致性和可互操作性。行业标准可以是国家或国际组织制定的，也可以是行业内部的专业组织或企业自行制定的。行业标准可以涵盖各种领域，如产品质量、安全性、环境保护、数据格式、通信协议等。

## 2.2 国际贸易协议

国际贸易协议是国家之间在贸易领域达成的协议，规定了各种贸易政策和规则。国际贸易协议可以涵盖多个领域，如关税政策、贸易保护、技术交流、投资保护、竞争政策等。国际贸易协议通常是双方或多方国家之间的协议，例如中美贸易协议、北美自由贸易协定等。

2. 减少贸易歧视：行业标准可以帮助国家减少贸易歧视，提高贸易的公平性，促进国际贸易协议的签订。
3. 促进技术交流：行业标准可以促进国际技术交流，提高产品和服务的兼容性，降低交易成本，从而进一步促进国际贸易协议的签订。

# 加载数据

# 根据行业标准筛选数据
def filter_by_industry_standard(data, standard):
    filtered_data = data[data['quality'] >= standard['quality']
                         & data['safety'] >= standard['safety']
                         & data['environment'] >= standard['environment']]
    return filtered_data

# 计算符合行业标准的产品数量
def count_compliant_products(data):
    count = data.shape[0]
    return count

# 计算符合行业标准的产品占比
def calculate_compliant_ratio(count, total):
    ratio = count / total
    return ratio

# 主程序
if __name__ == '__main__':
    standard = {'quality': 0.9, 'safety': 0.8, 'environment': 0.7}
    compliant_data = filter_by_industry_standard(data, standard)
    total = data.shape[0]
    compliant_count = compliant_data.shape[0]
    compliant_ratio = calculate_compliant_ratio(compliant_count, total)
    print(f'符合行业标准的产品数量：{compliant_count}')
    print(f'符合行业标准的产品占比：{compliant_ratio:.2f}')
