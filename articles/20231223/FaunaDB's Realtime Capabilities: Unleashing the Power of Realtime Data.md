```sql

```

2. JOIN 语句：JOIN 语句用于将多个表进行连接。例如，以下是一个简单的 JOIN 语句：

3. GROUP BY 语句：GROUP BY 语句用于将数据按照某个或多个属性进行分组。例如，以下是一个简单的 GROUP BY 语句：

文档模型查询主要包括以下几个方面：


2. 基于文档的查询：基于文档的查询用于根据文档的 ID 进行查询。例如，以下是一个简单的基于文档的查询：

3. 插入：插入是一种数据插入技术，它用于将数据插入到数据库中。例如，以下是一个简单的插入操作：

## 4.3 图模型查询

```graphql
query {
  nodes(where: { user_id: { _eq: "12345" } }) {
    id
    name
    friends {
      id
      name
    }
  }

```

2. 图匹配查询：图匹配查询用于匹配图中的子图。例如，以下是一个简单的图匹配查询：

```graphql

    name
  }
```
