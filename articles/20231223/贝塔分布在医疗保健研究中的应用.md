# 2.核心概念与联系
贝塔分布是一种二参数的连续概率分布，用于描述一系列独立事件的成功和失败的概率。在医疗保健研究中，贝塔分布可以用于描述疾病的发生概率、治疗效果、药物剂量等问题。贝塔分布的核心参数包括：

- α：成功事件的数量
- β：失败事件的数量

这两个参数可以通过实验数据或观测数据得到。贝塔分布的概率密度函数为：

$$
f(x; \alpha, \beta) = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha) \Gamma(\beta)} x^{\alpha - 1} (1 - x)^{\beta - 1}
$$

其中，x ∈ [0, 1]，α、β > 0。

# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个简单的代码实例来演示如何使用贝塔分布在医疗保健研究中。我们将使用Python的Scipy库来计算疾病发生概率。

```python
import numpy as np
from scipy.stats import beta

# 假设我们有一组病例数据，其中有100例确诊为疾病的病例，其他病例为未确诊
positive_cases = 100
negative_cases = 200

# 使用最大似然估计计算贝塔分布的参数
alpha = beta.fit(positive_cases, negative_cases)
beta_ = 1 - alpha

# 计算疾病发生概率
success_prob = beta.cdf(0.5, alpha, beta_)
print("Frequency of the disease: {:.2%}".format(success_prob))
```

在这个例子中，我们首先假设有一组病例数据，其中有100例确诊为疾病的病例，其他病例为未确诊。然后，我们使用最大似然估计计算贝塔分布的参数 α 和 β。最后，我们使用贝塔分布的累积分布函数（CDF）来计算疾病发生概率。
