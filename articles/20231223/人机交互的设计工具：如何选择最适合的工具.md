- 梯度下降：

```python
def gradient_descent(X, y, learning_rate, epochs):
    m, n = X.shape
    theta = np.zeros((m, n))
    X = np.c_[np.ones((m, 1)), X]
    y = y.reshape(-1, 1)
    for epoch in range(epochs):
        gradients = 2/m * X.T.dot(X.dot(theta) - y)
        theta -= learning_rate * gradients
    return theta
```
