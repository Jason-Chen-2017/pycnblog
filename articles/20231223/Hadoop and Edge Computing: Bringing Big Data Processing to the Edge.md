# 2.核心概念与联系

## 2.1 Hadoop 简介

Hadoop 是一个开源的分布式数据处理框架，由 Apache 开发和维护。Hadoop 的核心组件是 Hadoop Distributed File System (HDFS) 和 MapReduce。HDFS 是一个分布式文件系统，用于存储大规模的数据。MapReduce 是一个分布式数据处理模型，用于处理这些数据。

### 2.1.1 HDFS

HDFS 将数据划分为多个块（block），每个块大小通常为 64 MB 或 128 MB。这些块存储在集群中的不同节点上，形成一个分布式文件系统。HDFS 的主要优点是其高容错性、易于扩展和高吞吐量。

### 2.1.2 MapReduce

MapReduce 是一个分布式数据处理模型，它将数据处理任务划分为多个阶段：Map 和 Reduce。Map 阶段将数据划分为多个键值对，Reduce 阶段将这些键值对聚合为最终结果。MapReduce 的主要优点是其易于扩展和高吞吐量。

## 2.2 边缘计算简介

边缘计算是一种计算模型，将计算和存储功能推向边缘设备，如传感器、IoT 设备等。这种模型可以减少数据传输延迟、网络拥塞和能源消耗。边缘计算的主要优点是其低延迟、高可靠性和高效率。

## 2.3 Hadoop 和边缘计算的联系

将 Hadoop 与边缘计算结合，可以实现大数据处理的高效和低延迟。通过将计算和存储功能推向边缘设备，可以减少数据传输延迟、网络拥塞和能源消耗。同时，Hadoop 的分布式数据处理能力可以处理大规模的、高速增长的数据。

## 4.1 Hadoop 的 MapReduce 示例

### 4.1.1 Map 阶段

在 Map 阶段，我们将数据划分为多个键值对。例如，我们可以将一个文本文件中的每个单词作为一个键值对。

```python
from hadoop.mapreduce import Mapper

class WordCountMapper(Mapper):
    def map(self, key, value):
        words = value.split()
        for word in words:
            yield word, 1
```

### 4.1.2 Reduce 阶段

在 Reduce 阶段，我们将这些键值对聚合为最终结果。例如，我们可以计算每个单词的出现次数。

```python
from hadoop.mapreduce import Reducer

class WordCountReducer(Reducer):
    def reduce(self, key, values):
        count = sum(values)
        yield key, count
```

### 4.1.3 完整示例

```python
from hadoop.mapreduce import Job

if __name__ == '__main__':
    job = Job()
    job.set_mapper(WordCountMapper)
    job.set_reducer(WordCountReducer)
    job.run()
```
