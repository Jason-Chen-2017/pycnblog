# 2. 核心概念与联系

在基因组学中，核心概念包括基因组、基因、基因片段、基因表达、基因修饰等。这些概念之间存在着密切的联系，以下将逐一进行阐述。

## 2.1 基因组

基因组是一种包含所有组织细胞的生物种的基本遗传信息，通常包括 DNA 或 RNA 的一系列线性或循环的分子。基因组中的 DNA 分子包含了基因的序列，这些基因编码了生物种的特征。

## 2.2 基因

基因是 DNA 分子中的一段特定序列，它编码了生物种的特征。基因可以被转录为 RNA，然后被翻译为蛋白质。每个基因都有一个特定的基因符号，用于表示该基因的功能。

## 2.3 基因片段

基因片段是基因组中的一段特定序列，它可以包含一个或多个基因。基因片段可以通过 PCR（聚合酶链反应）技术得到，这是一种在特定条件下使用特定 primer（引导序列）引发的 DNA 复制过程。

## 2.4 基因表达

基因表达是指基因在生物细胞中被转录和翻译为 RNA 和蛋白质的过程。基因表达可以被激活或抑制，这取决于基因表达的调控元素，如转录因子和微RNA。

## 2.5 基因修饰

基因修饰是指基因的表达和功能被修改的过程。基因修饰可以通过各种方法实现，如基因编辑、基因敲除、基因替换等。
