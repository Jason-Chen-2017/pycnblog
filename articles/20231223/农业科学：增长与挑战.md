## 4.1 线性回归模型

线性回归模型是一种常用的数学模型，它可以用来预测一个变量的值，根据另一个变量的值。以下是一个简单的线性回归模型的Python代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# 生成一组数据
x = np.linspace(0, 10, 100)
y = 2 * x + 1 + np.random.randn(100) * 0.5

# 绘制数据点
plt.scatter(x, y)

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(x.reshape(-1, 1), y)

# 预测值
y_predict = model.predict(x.reshape(-1, 1))

# 绘制预测结果
plt.plot(x, y_predict, 'r')

plt.show()
```

在这个代码实例中，我们首先生成了一组数据，然后创建了一个线性回归模型，接着训练了模型，并使用模型进行预测。最后，我们绘制了数据点和预测结果。

## 4.2 动态规划算法

动态规划算法是一种常用的算法，它可以用来解决一些具有最优性的问题。以下是一个简单的动态规划算法的Python代码实例：

```python
def fib(n):
    if n <= 1:
        return n
    else:
        return fib(n-1) + fib(n-2)

# 计算第n个斐波那契数
print(fib(10))
```

在这个代码实例中，我们首先定义了一个递归函数fib，它用来计算第n个斐波那契数。然后我们调用了这个函数，并输出了结果。
