# 2.核心概念与联系

## 2.1 复合函数的定义

复合函数是指由多个函数组成的函数，它们之间存在某种关系或依赖关系。例如，在一个软件系统中，一个函数可能会调用另一个函数，从而形成一个复合函数。复合函数的测试策略旨在确保这些函数之间的交互和数据流动是正确的。

## 2.2 单元测试

单元测试是软件测试的基本方法，它涉及到测试单个函数或模块的正确性和效率。在单元测试中，测试用例通常包括输入、期望输出和实际输出，以确保函数的正确性。同时，单元测试还可以测试函数的效率，例如测试函数的执行时间和内存使用情况。

## 2.3 集成测试

集成测试是在单元测试之后进行的，它涉及到测试多个函数或模块之间的交互和数据流动。在集成测试中，测试用例通常包括多个函数的输入、期望输出和实际输出，以确保函数之间的交互是正确的。同时，集成测试还可以测试软件系统的性能，例如测试系统的响应时间和吞吐量。
