# 4.具体代码实例和详细解释说明
具体的代码实例如下：

```python
import numpy as np

def power_function(T, K, n):
    return K * (T ** n)

T = np.linspace(1, 100, 100)
K = 10
n = 2
P = power_function(T, K, n)

import matplotlib.pyplot as plt

plt.plot(T, P)
plt.xlabel('Temperature')
plt.ylabel('Power')
plt.title('Power Function of Temperature')
plt.show()
```

这个代码实例中，我们首先导入了 numpy 和 matplotlib.pyplot 这两个库，然后定义了一个 power_function 函数，该函数接受温度、常数和幂指数作为输入参数，并返回核反应的强度。接着，我们使用了 numpy 库中的 linspace 函数，生成了一个温度范围从 1 到 100 的数组。然后，我们使用了 power_function 函数，计算了核反应的强度。最后，我们使用了 matplotlib.pyplot 库，绘制了温度和核反应强度之间的关系。
