# 2.核心概念与联系

## 2.1 FEC技术概述

纠错输出码技术是一种在信道传输过程中通过在信息数据上加密的方式，使得信息数据在传输过程中能够在发生错误时自动进行纠正的技术。FEC技术通常包括两个主要的部分：编码器和解码器。编码器在发送端用于将原始数据加密为加密数据，解码器在接收端用于将接收到的加密数据解密并进行错误纠正。

FEC技术的主要优势在于，它可以在信道传输过程中抵御噪声、干扰和丢失等影响因素，从而保证数据传输的可靠性。同时，FEC技术也具有低延迟和高吞吐量的特点，这使得它在VR领域中具有广泛的应用前景。
