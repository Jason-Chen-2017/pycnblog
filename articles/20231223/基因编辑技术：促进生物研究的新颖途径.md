## 4.1 基因序列识别的代码实例
```python
import re

def identify_gene_sequence(dna_sequence):
    gene_sequence_pattern = r'ATG[ATGC]{3}TGA[ATGC]{3}TAA[ATGC]{3}TGA[ATGC]{3}TAA[ATGC]{3}'
    gene_sequences = re.findall(gene_sequence_pattern, dna_sequence)
    return gene_sequences
```
上述代码实例中，我们首先导入了`re`模块，然后定义了一个名为`identify_gene_sequence`的函数，该函数接受一个DNA序列作为输入参数，并通过使用正则表达式来识别基因序列。

上述代码实例中，我们定义了一个名为`modify_gene_sequence`的函数，该函数接受一个基因序列和一个修改字典作为输入参数，并通过使用字符串的`replace`方法来实现基因序列的修改。
