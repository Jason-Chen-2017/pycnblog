1. 定义一个随机过程，其中每个事件的概率均为 $1/2$。

3. 对估计结果进行统计分析，得到问题的解。

具体代码实例如下：

```python

def monte_carlo(n):
    result = 0
    for _ in range(n):
        x = random.random()
        result += x**2
    return result / n

n = 100000
print(monte_carlo(n))
```
