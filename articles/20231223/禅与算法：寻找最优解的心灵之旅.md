## 4.1 代码实例：最短路径问题

最短路径问题是一种常见的图论问题，它需要找到两个节点之间的最短路径。我们可以使用迪杰斯特拉算法来解决这个问题。

### 4.1.1 迪杰斯特拉算法

迪杰斯特拉算法是一种用于求解最短路径问题的算法。它的核心思想是通过不断更新节点的距离值，以便找到最短路径。

以下是迪杰斯特拉算法的具体实现：

```python
import heapq

def dijkstra(graph, start):
    dist = {node: float('inf') for node in graph}
    dist[start] = 0
    pq = [(0, start)]

    while pq:
        current_distance, current_node = heapq.heappop(pq)

        if current_distance > dist[current_node]:
            continue

        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight

            if distance < dist[neighbor]:
                dist[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))

    return dist
```

### 4.1.2 解释说明

在这个代码实例中，我们使用了迪杰斯特拉算法来解决最短路径问题。我们首先初始化了距离字典，将所有节点的距离设为无穷大，except start。然后我们将start节点的距离设为0，并将其加入优先级队列。

接下来，我们从优先级队列中取出当前距离最近的节点，并遍历其邻居节点。对于每个邻居节点，我们计算从start到邻居节点的距离，如果小于当前距离，则更新距离值并将其加入优先级队列。

通过这种方式，我们可以逐步更新所有节点的距离值，以便找到最短路径。

## 4.2 代码实例：背包问题

背包问题是一种常见的组合优化问题，它需要找到一种物品组合，使得总重量不超过背包容量，同时最大化价值。我们可以使用动态规划算法来解决这个问题。

### 4.2.1 动态规划算法

动态规划算法是一种用于求解优化问题的算法。它的核心思想是将问题拆分成多个子问题，并将子问题的解存储在一个表格中，以便后续使用。

以下是背包问题的动态规划算法实现：

```python
def knapsack(items, capacity):
    n = len(items)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            item = items[i - 1]
            if w >= item['weight']:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - item['weight']] + item['value'])
            else:
                dp[i][w] = dp[i - 1][w]

    return dp[n][capacity]
```

### 4.2.2 解释说明

在这个代码实例中，我们使用了动态规划算法来解决背包问题。我们首先初始化了一个dp表格，用于存储子问题的解。dp表格的行表示物品，列表示背包容量。

接下来，我们遍历所有物品，对于每个物品，我们检查其重量是否小于或等于背包容量。如果是，我们计算将该物品放入背包中的价值，并将其与不放入背包的价值进行比较，选择更大的价值。

通过这种方式，我们可以逐步计算所有物品的最大价值，以便找到最佳物品组合。
