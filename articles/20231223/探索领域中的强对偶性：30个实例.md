## 1.1 强对偶性的基本概念
强对偶性是一种优化问题的解决方法，它通过将原问题分解为两个相互对偶的子问题来解决。这种方法的核心在于将原问题的约束条件转化为子问题的目标函数，而子问题的目标函数则可以转化为原问题的约束条件。通过这种方法，我们可以在保持问题解的同时，将原问题转化为一个更容易解决的子问题。

import networkx as nx

def shortest_path(graph, start, end):
    tree = nx.minimum_spanning_tree(graph)
    flow = nx.max_flow(tree, source=start, target=end)
    path = nx.shortest_path(tree, start, end, weight='flow')
    return path

graph = nx.Graph()
graph.add_edge('A', 'B', weight=1)
