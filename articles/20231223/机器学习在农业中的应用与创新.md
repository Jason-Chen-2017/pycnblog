接下来，我们需要使用线性回归算法来训练我们的模型。我们可以使用以下代码来完成这个任务：

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# 准备数据
data = np.array([[2000, 100, 5000],
                 [2001, 105, 5100],
                 [2002, 110, 5200],
                 [2003, 115, 5300],
                 [2004, 120, 5400],
                 [2005, 125, 5500],
                 [2006, 130, 5600],
                 [2007, 135, 5700],
                 [2008, 140, 5800],
                 [2009, 145, 5900],
                 [2010, 150, 6000],
                 [2011, 155, 6100],
                 [2012, 160, 6200],
                 [2013, 165, 6300],
                 [2014, 170, 6400],
                 [2015, 175, 6500],
                 [2016, 180, 6600],
                 [2017, 185, 6700],
                 [2018, 190, 6800],
                 [2019, 195, 6900]])

# 分离特征和标签
X = data[:, 0:-1]  # 生产量
y = data[:, -1]    # 价格

# 使用线性回归算法训练模型
model = LinearRegression()
model.fit(X, y)

# 预测价格
predicted_price = model.predict(np.array([2020, 200]))

# 绘制图表
plt.scatter(X, y)
plt.plot(X, model.predict(X), color='red')
plt.xlabel('生产量')
plt.ylabel('价格')
plt.title('农业生产价格预测')
plt.show()
```

通过上述代码，我们可以看到我们的模型已经成功地预测了农业生产价格。我们可以使用这个模型来预测未来的农业生产价格。
