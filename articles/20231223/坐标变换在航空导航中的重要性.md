# 4.具体代码实例和详细解释说明

以下是一个简单的Python代码实例，演示了如何实现从地球坐标系转换到航向坐标系的坐标变换：

```python
import numpy as np

# 地球坐标系位置信息
earth_coordinates = np.array([120.123, 30.456, 0.789])

# 航向坐标系转换参数
latitude = np.radians(30)
longitude = np.radians(120)

# 地球坐标系到航向坐标系的转换矩阵
rotation_matrix = np.array([
    [np.cos(latitude) * np.cos(longitude), -np.sin(latitude) * np.cos(longitude), np.sin(longitude)],
    [np.cos(latitude) * np.sin(longitude), np.sin(latitude) * np.sin(longitude), -np.cos(longitude)],
    [-np.sin(latitude) * np.cos(longitude), np.sin(latitude) * np.sin(longitude), np.cos(latitude)]
])

# 实现坐标变换
navigational_coordinates = np.dot(earth_coordinates, rotation_matrix)

print("地球坐标系位置信息:", earth_coordinates)
print("航向坐标系位置信息:", navigational_coordinates)
```

在这个代码实例中，我们首先定义了地球坐标系位置信息，并设定了航向坐标系转换参数（纬度和经度）。然后，我们计算了地球坐标系到航向坐标系的转换矩阵，并使用矩阵乘法实现了坐标变换。最后，我们打印了转换前后的坐标信息，以验证坐标变换结果。
