## 3.1 数据清洗

数据清洗是数据标准化的一种方法，它旨在删除数据中的错误、不准确、重复和缺失值。数据清洗可以帮助我们更好地理解和分析数据，从而提高模型的性能。

数据清洗的主要方法包括：

1. 删除错误值：删除数据中的错误值，例如非数值类型的值。
2. 删除不准确值：删除数据中的不准确值，例如包含错误的值。
3. 删除重复值：删除数据中的重复值，例如相同的值。
4. 填充缺失值：填充数据中的缺失值，例如使用平均值、中位数或最大值等方法。

## 3.2 数据规范化

数据规范化是数据标准化的一种方法，它旨在将不同的数据集转换为相同的格式。数据规范化可以帮助我们更好地理解和分析数据，从而提高模型的性能。

数据规范化的主要方法包括：

1. 最小-最大规范化：将数据值转换为一个范围内的值，例如0到1。
2. 标准化：将数据值转换为标准化的值，例如均值为0、方差为1。
3. 归一化：将数据值转换为一个固定的范围内的值，例如0到1。

2. 一hot编码：将类别变量转换为多维二进制向量。
3. 数值化：将类别变量转换为数值变量。

## 4.1 数据清洗

### 4.1.1 删除错误值

```python
import pandas as pd
import numpy as np

data = pd.DataFrame({
    'age': [23, 34, 45, np.NaN, 56],
    'gender': ['male', 'female', 'male', 'female', 'male'],
    'income': [50000, 60000, 70000, 80000, 90000]
})

data['age'] = data['age'].replace(np.NaN, data['age'].mean())
```

### 4.1.2 删除不准确值

```python
data['income'] = data['income'].replace('male', 0)
data['income'] = data['income'].replace('female', 1)
```

### 4.1.3 删除重复值

```python
data = data.drop_duplicates()
```

### 4.1.4 填充缺失值

```python
data['age'] = data['age'].fillna(data['age'].mean())
```

## 4.2 数据规范化

### 4.2.1 最小-最大规范化

```python
data['age_min_max'] = (data['age'] - data['age'].min()) / (data['age'].max() - data['age'].min())
```

### 4.2.2 标准化

```python
data['age_std'] = (data['age'] - data['age'].mean()) / data['age'].std()
```

### 4.2.3 归一化

```python
data['age_min'] = (data['age'] - data['age'].min()) / 10
```

## 4.3 数据转换

### 4.3.1 类别编码

```python
data['gender_encoded'] = data['gender'].astype('category').cat.codes
```

### 4.3.2 一hot编码

```python
data = pd.get_dummies(data, columns=['gender'])
```

### 4.3.3 数值化

```python
data['gender_encoded'] = data['gender'].map({'male': 0, 'female': 1})
```
