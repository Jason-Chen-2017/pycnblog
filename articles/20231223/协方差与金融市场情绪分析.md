# 2.核心概念与联系
协方差是一种度量两个随机变量线性相关程度的统计量，它能够衡量两个变量的变动同步程度。协方差的计算公式为：

$$
Cov(X,Y) = E[(X - \mu_X)(Y - \mu_Y)]
$$

其中，$X$ 和 $Y$ 是两个随机变量，$E$ 表示期望，$\mu_X$ 和 $\mu_Y$ 是 $X$ 和 $Y$ 的期望值。协方差的正值表示两个变量同步变动，负值表示两个变量相互抵消变动，零表示两个变量之间无线性关系。

在金融市场情绪分析中，协方差可以用于度量不同金融市场或金融工具之间的情绪同步程度。例如，可以计算股指和货币市场利率之间的协方差，以评估两者的相关性，从而预测市场行情变化。

```python
import numpy as np
import pandas as pd

# 加载数据
stock_data = pd.read_csv('stock_data.csv')
```

Q：协方差和相关系数有什么区别？

A：协方差是一种度量两个随机变量线性相关程度的统计量，它能够衡量两个变量的变动同步程度。相关系数是一种度量两个随机变量线性相关程度的统计量，它能够衡量两个变量之间的相关性。协方差的正值表示两个变量同步变动，负值表示两个变量相互抵消变动，零表示两个变量之间无线性关系。相关系数的值范围在-1到1之间，其中-1表示完全相反的关系，1表示完全相同的关系，0表示无关系。

Q：如何计算两个随机变量之间的相关系数？

A：两个随机变量之间的相关系数可以使用Pearson相关系数公式计算，公式为：

$$
r = \frac{\sum_{i=1}^n (X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum_{i=1}^n (X_i - \bar{X})^2}\sqrt{\sum_{i=1}^n (Y_i - \bar{Y})^2}}
$$

其中，$X$ 和 $Y$ 是两个随机变量，$n$ 是数据样本数，$\bar{X}$ 和 $\bar{Y}$ 是 $X$ 和 $Y$ 的样本均值，$(X_i - \bar{X})(Y_i - \bar{Y})$ 是数据点$(X_i, Y_i)$ 与样本均值的乘积，$\sum_{i=1}^n (X_i - \bar{X})^2$ 是数据点$(X_i, Y_i)$ 与样本均值的乘积的和，$\sum_{i=1}^n (Y_i - \bar{Y})^2$ 是数据点$(X_i, Y_i)$ 与样本均值的乘积的和。
