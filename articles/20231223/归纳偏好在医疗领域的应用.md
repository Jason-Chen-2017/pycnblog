# 2.核心概念与联系
## 2.1 概率论与归纳偏好
概率论是一门数学分支，它描述了事件发生的可能性。归纳偏好（Bayesian Inference）是概率论中的一个重要方法，它可以根据现有的数据更新我们的知识。

### 2.1.1 概率论基础
概率论中，事件的概率表示为事件发生的可能性。概率值范围在0到1之间，0表示事件不可能发生，1表示事件必然发生。概率论中的几个基本定理：

- 总概率定理：对于任何事件A，P(A) = P(A|B) * P(B) + P(A|B') * P(B')，其中B和B'分别表示事件A发生和不发生。
- 贝叶斯定理：P(A|B) = P(B|A) * P(A) / P(B)，其中P(A|B)表示事件A发生条件下事件B的概率，P(B|A)表示事件B发生条件下事件A的概率。

### 2.1.2 归纳偏好基础
归纳偏好（Bayesian Inference）是一种根据现有数据更新知识的方法。归纳偏好可以通过对事件A和事件B的关系进行描述，得到事件A的概率分布。归纳偏好的基本公式为：

P(A|B) = P(B|A) * P(A) / P(B)

其中P(A|B)表示事件A发生条件下事件B的概率，P(B|A)表示事件B发生条件下事件A的概率，P(A)和P(B)分别是事件A和事件B的先验概率。

## 2.2 归纳偏好与医疗领域的联系
