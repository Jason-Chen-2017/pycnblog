## 3.1 文本处理

### 3.1.1 文本清洗

文本清洗的主要步骤包括：

1. 删除HTML标签和特殊符号。
2. 转换大小写。
3. 去除停用词。
4. 词干提取。

### 3.1.2 分词

分词是将文本划分为有意义的单词或词组的过程。常见的分词方法有：

1. 基于规则的分词（如中文的切词）。
2. 基于统计的分词（如英文的 NLTK 库）。
3. 基于深度学习的分词（如 BERT 分词器）。

### 3.1.3 标记

标记是将文本中的实体、关系、情感等进行标注的过程。常见的标记方法有：

1. 基于规则的标记。
2. 基于统计的标记。
3. 基于深度学习的标记。

## 4.1 文本处理

### 4.1.1 文本清洗

```python
import re

def clean_text(text):
    # 删除HTML标签
    text = re.sub(r'<[^>]+>', '', text)
    # 转换大小写
    text = text.lower()
    # 去除停用词
    stop_words = set(['a', 'an', 'the', 'and', 'or', 'but', 'is', 'are', 'was', 'were', 'of', 'to', 'for', 'with'])
    words = text.split()
    text = ' '.join([word for word in words if word not in stop_words])
    # 词干提取
    from nltk.stem import PorterStemmer
    ps = PorterStemmer()
    words = text.split()
    text = ' '.join([ps.stem(word) for word in words])
    return text
```
