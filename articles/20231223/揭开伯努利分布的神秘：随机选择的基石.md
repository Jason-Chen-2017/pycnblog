# 2.核心概念与联系

## 2.1 伯努利分布的定义

伯努利分布是一种离散概率分布，用于描述一个随机变量的取值情况。假设有一个随机变量X，它只能取值为0或1，且P(X=0) = p，P(X=1) = 1-p，其中0 < p < 1。这种分布就称为伯努利分布，其概率密度函数为：

$$
P(X=0) = p \\
P(X=1) = 1-p
$$

## 2.2 伯努利分布的性质

伯努利分布具有以下性质：

1. 它是一个离散的概率分布。
2. 它只有两个可能的取值：0 和 1。
3. 它的期望为：

$$
E(X) = p \times 0 + (1-p) \times 1 = 1-p
$$

4. 它的方差为：

$$
Var(X) = p \times (1-p)
$$

5. 它的模型可以用于描述二元选择问题，如硬币抛币、骰子掷子等。

def generate_bernoulli(p):

from scipy.stats import bernoulli

## 6.1 伯努利分布与二项分布的区别

伯努利分布和二项分布都是离散概率分布，但它们之间有一些重要的区别。二项分布描述了在固定数量的独立试验中成功的次数，而伯努利分布描述了一个随机变量只能取0或1的概率分布。在二项分布中，参数p表示每次试验的成功概率，而在伯努利分布中，参数p表示随机变量只能取0或1的概率。

## 6.2 如何选择合适的伯努利分布参数

在实际应用中，选择合适的伯努利分布参数是非常重要的。通常情况下，我们可以使用最大似然估计（MLE）或方差梯度下降法（VGD）来估计参数。在某些情况下，我们还可以使用跨验证或交叉验证来评估不同参数值的性能，从而选择最佳的参数值。
