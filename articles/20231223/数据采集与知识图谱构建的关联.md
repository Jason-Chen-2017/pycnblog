## 4.1 数据采集代码实例

以下是一个简单的Python代码实例，用于从一个JSON文件中获取数据：

```python
import json

def load_data(file_path):
    with open(file_path, 'r') as f:
        data = json.load(f)
    return data

file_path = 'data.json'
data = load_data(file_path)
print(data)
```

在这个代码实例中，我们首先导入了`json`模块，然后定义了一个`load_data`函数，该函数用于从一个JSON文件中获取数据。在函数中，我们使用`with`语句打开文件，并使用`json.load`函数将文件中的数据加载到一个Python字典中。最后，我们调用`load_data`函数，并将获取到的数据打印出来。

## 4.2 知识图谱构建代码实例

以下是一个简单的Python代码实例，用于将获取到的数据转换为知识图谱中的实体和关系：

```python
def extract_entities(data):
    entities = []
    for item in data:
        entities.append(item['entity'])
    return entities

def extract_relations(data):
    relations = []
    for item in data:
        relations.append((item['entity1'], item['entity2'], item['relation']))
    return relations

data = [
    {'entity': 'A', 'entity1': 'B', 'entity2': 'C', 'relation': 'friend'},
    {'entity': 'D', 'entity1': 'E', 'entity2': 'F', 'relation': 'enemy'},
]

entities = extract_entities(data)
relations = extract_relations(data)
print('Entities:', entities)
print('Relations:', relations)
```

在这个代码实例中，我们首先定义了两个函数，分别用于提取实体和关系。在`extract_entities`函数中，我们遍历了数据列表，并将每个数据项中的实体添加到一个列表中。在`extract_relations`函数中，我们遍历了数据列表，并将每个数据项中的实体1、实体2和关系添加到一个列表中。最后，我们调用了这两个函数，并将获取到的实体和关系打印出来。
