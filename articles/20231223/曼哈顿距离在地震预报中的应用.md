# 2.核心概念与联系

## 2.1 曼哈顿距离

曼哈顿距离（Manhattan Distance）是一种计算两点距离的方法，它只考虑纵横坐标之间的直接距离，不考虑地球的曲面效应。在地震预报中，曼哈顿距离主要用于计算地震发生的地理位置和深度之间的距离。

曼哈顿距离公式为：
$$
d = |x_1 - x_2| + |y_1 - y_2|
$$
其中，$x_1$、$x_2$ 是两个点的横坐标，$y_1$、$y_2$ 是两个点的纵坐标。

# 4.具体代码实例和详细解释说明

在这里，我们以一个简单的Python代码实例来展示如何使用曼哈顿距离公式计算两点距离：

```python
import math

def manhattan_distance(x1, y1, x2, y2):
    return abs(x1 - x2) + abs(y1 - y2)

# 测试数据
x1, y1 = 1, 2
x2, y2 = 3, 4

# 计算曼哈顿距离
distance = manhattan_distance(x1, y1, x2, y2)
print("曼哈顿距离：", distance)
```

在这个代码实例中，我们首先导入了`math`模块，用于计算绝对值。然后定义了一个名为`manhattan_distance`的函数，该函数接受四个参数，分别表示两个点的横纵坐标。在函数内部，我们使用曼哈顿距离公式计算两点距离，并返回结果。最后，我们测试了这个函数，并输出了计算结果。
