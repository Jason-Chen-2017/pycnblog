# 2.核心概念与联系

曼哈顿距离的名字来源于纽约市的一条街道，即曼哈顿街（Manhattan）。它是一种简单的距离度量方法，可以用来计算两个坐标的距离。给定两个坐标（x1, y1）和（x2, y2），曼哈顿距离可以通过以下公式计算：

$$
M(\textbf{p}, \textbf{q}) = |x_1 - x_2| + |y_1 - y_2|
$$

其中，M（p，q）表示曼哈顿距离，（x1，y1）和（x2，y2）分别表示两个坐标的横坐标和纵坐标。

在地理统计分析中，曼哈顿距离具有以下特点：

1. 简单易计算：曼哈顿距离仅需要计算两个坐标的横纵坐标的差值，无需计算曲线距离，因此它非常简单易计算。

```python

    return abs(p[0] - q[0]) + abs(p[1] - q[1])

# 计算曼哈顿距离
```
