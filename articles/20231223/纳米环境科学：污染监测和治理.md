## 4.1 随机漫步模型实现
我们可以使用 Python 来实现随机漫步模型。以下是一个简单的随机漫步模型实现：

```python
import random

def random_walk(steps, x, y):
    for _ in range(steps):
        dx = random.choice([-1, 1])
        dy = random.choice([-1, 1])
        x += dx
        y += dy
    return x, y
```

在这个实现中，我们使用了 Python 的 `random` 模块来生成随机数。我们定义了一个 `random_walk` 函数，它接受一定数量的步数 `steps` 以及开始位置 `x` 和 `y` 作为输入。函数返回随机漫步后的位置。
