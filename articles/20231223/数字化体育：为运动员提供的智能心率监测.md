# 4.具体代码实例和详细解释说明

## 4.1 心率计算代码实例

以下是一个简单的心率计算代码实例：

```python
def heart_rate(heart_wave_num, time_interval, heart_rate_count):
    return (heart_wave_num / time_interval) * heart_rate_count
```

在这个代码实例中，我们定义了一个名为`heart_rate`的函数，该函数用于计算心率。该函数接受三个参数：`heart_wave_num`（心电波数）、`time_interval`（时间间隔）和`heart_rate_count`（心率计数）。该函数返回运动员的心率。

## 4.2 运动状态计算代码实例

以下是一个简单的运动状态计算代码实例：

```python
def motion_state(heart_rate, motion_time, fatigue_degree):
    if heart_rate < 100:
        return "轻度运动"
    elif heart_rate < 130:
        return "中度运动"
    else:
        return "重度运动"
```

在这个代码实例中，我们定义了一个名为`motion_state`的函数，该函数用于计算运动员的运动状态。该函数接受三个参数：`heart_rate`（心率）、`motion_time`（运动时间）和`fatigue_degree`（疲劳程度）。该函数根据心率的值返回运动状态。

## 4.3 疲劳程度计算代码实例

以下是一个简单的疲劳程度计算代码实例：

```python
def fatigue_degree(heart_rate, motion_time, motion_intensity):
    if motion_intensity < 50:
        return "轻度疲劳"
    elif motion_intensity < 70:
        return "中度疲劳"
    else:
        return "重度疲劳"
```

在这个代码实例中，我们定义了一个名为`fatigue_degree`的函数，该函数用于计算运动员的疲劳程度。该函数接受三个参数：`heart_rate`（心率）、`motion_time`（运动时间）和`motion_intensity`（运动强度）。该函数根据运动强度的值返回疲劳程度。
