## 4.1 代码实例

我们将通过一个简单的社交网络来说明贝叶斯网络在社交网络分析中的应用。在这个社交网络中，我们有三个人，他们之间存在三个关系。我们将使用贝叶斯网络来描述这个社交网络，并计算某个人是否会参加某个活动的概率。

```python
import networkx as nx
import numpy as np

# 创建一个有向无环图
G = nx.DiGraph()

# 添加节点
G.add_node("Alice")
G.add_node("Bob")
G.add_node("Charlie")

# 添加边
G.add_edge("Alice", "Bob")
G.add_edge("Bob", "Charlie")
G.add_edge("Alice", "Charlie")

# 计算某个人是否会参加某个活动的概率
def calculate_probability(G, activity):
    # 创建一个字典，用于存储节点的条件概率分布
    node_probability = {}

    # 遍历所有节点
    for node in G.nodes():
        # 计算节点的条件概率分布
        node_probability[node] = nx.maximum_clique_probability(G, node, activity)

    # 返回节点的条件概率分布
    return node_probability

# 计算某个人是否会参加某个活动的概率
activity = "party"
node_probability = calculate_probability(G, activity)

# 打印某个人是否会参加某个活动的概率
for node, probability in node_probability.items():
    print(f"{node} will attend the {activity} with probability {probability}")
```

## 4.2 详细解释说明

在这个代码实例中，我们首先创建了一个有向无环图，用于表示社交网络。然后，我们添加了三个节点，表示三个人。接着，我们添加了三个边，表示三个人之间的关系。

接下来，我们定义了一个函数 `calculate_probability`，用于计算某个人是否会参加某个活动的概率。这个函数使用了 `nx.maximum_clique_probability` 函数，该函数用于计算给定节点的条件概率分布。

最后，我们调用了 `calculate_probability` 函数，并打印了某个人是否会参加某个活动的概率。
