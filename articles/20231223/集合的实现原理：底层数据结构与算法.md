# 2.核心概念与联系
集合的核心概念包括：

1.集合的定义：集合是一组元素的集合，元素之间没有顺序关系，也没有重复的元素。集合可以用大括号{}表示，元素之间用逗号分隔。例如，{1, 2, 3}是一个集合，包含三个元素1、2和3。

2.集合的属性：集合有五个基本属性，包括：

- 集合的元素是唯一的：集合中的元素不能重复。
- 集合的元素是无序的：集合中的元素没有顺序关系。
- 集合的元素是不可替代的：集合中的元素是唯一的，不能替换。

3.集合的运算：集合有几种基本的运算，包括：

- 交集：两个集合的交集是它们共同包含的元素的集合。例如，{1, 2, 3}和{2, 3, 4}的交集是{2, 3}。
- 并集：两个集合的并集是它们所有元素的集合。例如，{1, 2, 3}和{2, 3, 4}的并集是{1, 2, 3, 4}。
- 差集：两个集合的差集是它们不同元素的集合。例如，{1, 2, 3}和{2, 3, 4}的差集是{1}。
- 笛卡尔积：两个集合的笛卡尔积是它们所有可能组合的元素的集合。例如，{1, 2}和{a, b}的笛卡尔积是{{1, a}, {1, b}, {2, a}, {2, b}}。

4.集合的表示：集合可以用不同的方式表示，包括：

- 集合表示：使用大括号{}表示集合，元素之间用逗号分隔。例如，{1, 2, 3}。
