# 2. 核心概念与联系
跨媒体新闻是一种利用多种媒体平台相互联系和协同工作的新闻报道方式，它的核心概念包括：

1. 多媒体：跨媒体新闻涉及到的媒体类型有很多，包括传统媒体（如报纸、电视、广播等）和互联网媒体（如网站、微博、微信、短信等）。
2. 融合：跨媒体新闻要求不同媒体平台之间相互融合，实现内容的共享和传播，从而提高新闻报道的质量和影响力。
3. 平台：跨媒体新闻需要建立在一种可扩展、易用、高效的平台上，以支持不同媒体平台之间的互联互通和数据共享。
