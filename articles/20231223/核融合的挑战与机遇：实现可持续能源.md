# 2.核融合的核心概念与联系

核融合的核心概念包括：

1. 核融合：核融合是指轻元素（如氢）在高温和高压下的融合过程，通过这个过程可以产生大量的能量。核融合是太阳的主要能源，也是我们可持续能源的希望。

2. 融合反应：融合反应是指两个原子核之间的融合过程。常见的融合反应包括氢-氢（D-D）反应和氢-氧（D-T）反应。

3. 融合反应器：融合反应器是一个用于实现核融合反应的设备。目前，科学家正在研究不同的融合反应器设计，如磁性吸引反应器（Magnetic Confinement Fusion，MCF）和膨胀反应器（Tokamak）。

4. 融合能源：融合能源是指通过核融合产生的能量。融合能源是一种可持续、安全和环保的能源，有望成为未来人类主要的能源来源。
