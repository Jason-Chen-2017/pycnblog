# 2.核心概念与联系

在本节中，我们将介绍时间序列分析和预测的核心概念，并探讨它们之间的联系。

## 2.1 时间序列数据

时间序列数据是按照时间顺序收集的数据点的序列。这些数据点通常是连续的，但也可以是离散的。例如，股票价格、气温、人口数量、网络流量等都可以被视为时间序列数据。

时间序列数据通常具有以下特点：

- 时间顺序：数据点按照时间顺序排列。
- 自相关性：当前数据点的值可能与过去一段时间内的数据点值有关。
- 季节性：某些时间序列数据可能具有季节性，即数据点值可能随着时间的推移而重复。

## 2.2 时间序列分析

时间序列分析是一种用于处理和分析时间序列数据的方法。它的目标是揭示数据中的模式、趋势和关系，从而帮助我们做出更明智的决策。

时间序列分析的主要方法包括：

- 趋势分析：揭示数据中的长期趋势。
- 季节分析：揭示数据中的季节性。
- 差分分析：通过计算连续数据点之间的差值来去除时间序列中的季节性和噪声。
- 自相关分析：通过计算数据点之间的自相关性来揭示时间序列中的关系。

## 2.3 时间序列预测

时间序列预测是一种用于预测未来时间序列数据点值的方法。它的目标是利用过去数据点的值和关系来预测未来数据点的值。

时间序列预测的主要方法包括：

- 自回归（AR）模型：假设当前数据点的值仅依赖于过去一定数量的数据点值。
- 移动平均（MA）模型：假设当前数据点的值仅依赖于过去一定数量的数据点值的差值。
- 自回归积分移动平均（ARIMA）模型：结合自回归和移动平均模型，以更准确地预测时间序列数据。
- 卷积神经网络（CNN）模型：使用深度学习技术来预测时间序列数据。
