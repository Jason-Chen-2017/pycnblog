## 4.1 代码实例
在这里，我们给出一个简单的代码实例，用于演示凸集分离的具体操作。

```python
import numpy as np
from scipy.optimize import minimize

def objective_function(x):
    return np.sum(x**2)

def constraint_function(x):
    return np.sum(x**2) - 1

def constraint_inequality_function(x):
    return np.sum(x**2) - 1

x0 = np.array([0.5, 0.5])

result = minimize(objective_function, x0, constraints=[{'type': 'eq', 'fun': constraint_function},
                                                        {'type': 'ineq', 'fun': constraint_inequality_function}])

print(result.x)
```
