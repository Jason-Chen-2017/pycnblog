## 4.1 具体代码实例

在这里，我们以一个简单的人工智能诊断疾病的例子进行说明。我们将使用一个简单的决策树算法来诊断疾病。首先，我们需要准备一些数据，包括患者的症状、体征和检查结果等。然后，我们可以使用Scikit-learn库中的DecisionTreeClassifier类来构建决策树模型，并使用train_test_split函数将数据分为训练集和测试集。最后，我们可以使用fit方法训练模型，并使用predict方法对测试数据进行预测。以下是一个简单的代码实例：

```python
from sklearn.tree import DecisionTreeClassifier
from sklearn.model_selection import train_test_split
import pandas as pd

# 加载数据
data = pd.read_csv("medical_data.csv")

# 将数据转换为特征和标签
```
