## 4.1准确性评估

```python
# 加载数据
data = pd.read_csv('data.csv')

# 加载元数据
metadata = pd.read_csv('metadata.csv')

# 比较元数据与数据的关系
accuracy = metadata.compare_with_data(data)

print('准确性:', accuracy)
```
