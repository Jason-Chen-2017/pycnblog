### 2.4.1 try-except 语句的使用

try-except 语句是 Python 异常处理的核心组成部分。它可以帮助程序员更好地处理程序中可能出现的错误和异常情况。下面是一个简单的 try-except 语句的示例：

```python
try:
    # 尝试执行的代码块
    result = 10 / 0
except ZeroDivisionError:
    # 异常处理代码块
    print("出现了除零错误")
```

在这个示例中，我们尝试执行一个会导致除零错误的操作。当发生这个错误时，程序会立即跳转到 except 子句，执行异常处理代码块。

### 2.4.2 raise 语句的使用

raise 语句可以用来抛出自定义的异常。下面是一个简单的 raise 语句的示例：

```python
def divide(x, y):
    if y == 0:
        raise ValueError("除数不能为零")
    return x / y

try:
    result = divide(10, 0)
except ValueError as e:
    print(e)
```

在这个示例中，我们定义了一个 divide 函数，它会检查除数是否为零。如果除数为零，它会抛出一个 ValueError 异常。在 try 子句中，我们调用 divide 函数，并捕获可能发生的 ValueError 异常。

### 2.6.1 try-except 语句的使用规范

1. try-except 语句应该只用于处理可预见的异常情况。如果出现不可预见的异常情况，应该使用其他方法来处理。
2. try-except 语句应该尽量短小，以便于阅读和维护。
3. try-except 语句应该尽量避免使用捕获所有异常，因为这可能会导致程序出现意外的行为。

### 2.6.2 raise 语句的使用规范

2. raise 语句应该尽量简洁，以便于阅读和维护。
3. raise 语句应该尽量避免使用捕获所有异常，因为这可能会导致程序出现意外的行为。

### 2.6.3 异常处理的最佳实践

3. 如果需要抛出异常情况，应该使用 raise 语句来抛出。
