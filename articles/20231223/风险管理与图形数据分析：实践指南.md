# 4.1 基本图算法实例
在本节中，我们将通过一个简单的图的连通性分析实例来演示基本图算法的使用。

```python
import networkx as nx

# 创建一个有向图
G = nx.DiGraph()

# 添加节点
G.add_node("A")
G.add_node("B")
G.add_node("C")

# 添加边
G.add_edge("A", "B")
G.add_edge("B", "C")

# 检查图是否连通
is_connected = nx.is_connected(G)
print("图是否连通:", is_connected)
```

在这个实例中，我们创建了一个有向图，并添加了三个节点和两个边。然后，我们使用`nx.is_connected()`函数检查图是否连通。输出结果为`False`，表示图不是连通的。
