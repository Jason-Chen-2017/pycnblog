# 2. 核心概念与联系

## 2.1 Close Curve

Close Curve（闭曲线）是一种在平面上连续且封闭的曲线，它的起点与终点相连，形成一个封闭的区域。闭曲线可以是多边形、圆形或者更复杂的曲线。在计算机图形学中，闭曲线是一个基本的图形元素，用于构建更复杂的图形。

## 2.2 曲率

曲率（Curvature）是描述曲线在某一点的弧度变化率的量，它反映了曲线在该点处的弧度变化速度。曲率是一个向量量，其方向与曲线上的正向相同。曲率的单位是弧度/长度。

## 2.3 曲率与Close Curve之间的关联

在Close Curve中，曲率是一个重要的性质之一。曲率可以用来描述曲线在某一点的弧度变化率，从而反映出曲线的形状和特点。在计算机图形学中，曲率是用于计算曲线的相交、接触、碰撞等属性的关键因素。

## 3.1 曲率公式

曲率公式可以用来计算曲线在某一点处的曲率。给定一个二阶可导的函数y=f(x)，曲率公式如下：

$$
\kappa (t) = \frac{|y'(t)|}{[1 + (y'(t))^2]^{3/2}}
$$

其中，$\kappa (t)$ 是曲率在时间t处的值，$y'(t)$ 是函数y关于时间t的一阶导数，$y''(t)$ 是函数y关于时间t的二阶导数。
