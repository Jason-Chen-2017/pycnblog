## 4.具体代码实例和详细解释说明

### 4.1 代码实例

以下是一个简单的凸函数优化问题的代码实例：

```python
from scipy.optimize import minimize

def f(x):
    return x[0]**2 + x[1]**2

x0 = [1, 1]
res = minimize(f, x0)
print(res.x)
```

### 4.2 详细解释说明

上述代码实例中，我们定义了一个凸函数`f(x)`，它是一个二变量的平方和。然后，我们使用`scipy.optimize.minimize`函数进行优化，并将初始化的优化变量`x0`设为`[1, 1]`。最后，我们打印了优化结果。
