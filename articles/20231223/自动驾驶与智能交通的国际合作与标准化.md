# 2.核心概念与联系

自动驾驶技术是指使用计算机视觉、传感器、导航等技术，以及人工智能、大数据等方法，实现汽车在道路上的自主控制和自主决策，以达到安全、高效、环保等目标。智能交通系统是指利用信息与通信技术、人工智能等方法，对交通流程进行优化和智能化管理，以提高交通效率、减少交通拥堵、减少碰撞等目标。

自动驾驶与智能交通系统之间的联系主要体现在以下几个方面：

1. 技术联系：自动驾驶技术可以为智能交通系统提供实时的车辆状态、道路状态等信息，从而实现更精确的交通预测和调度。
2. 应用联系：自动驾驶技术可以为智能交通系统提供一种新的交通模式，例如无驾驶车辆的共享服务。
3. 政策联系：自动驾驶与智能交通系统的发展和应用需要相应的政策支持，例如交通法规的修改、道路设施的改造等。
