我们可以通过以下代码来计算建筑物内部空间的连通性：

```
import networkx as nx

def calculate_connectivity(building):
    G = nx.Graph()
    for space in building.spaces:
        G.add_node(space.id)
    for space in building.spaces:
        for adjacent_space in space.adjacent_spaces:
            G.add_edge(space.id, adjacent_space.id)
    connectivity = nx.average_shortest_path_length(G)
    return connectivity
```

在上述代码中，我们首先创建了一个无向图G，将建筑物内部的每个空间作为图的节点。然后，我们遍历了建筑物内部的每个空间，并将与其相连的空间作为图的边添加到G中。最后，我们通过计算G的平均最短路径长度来得到建筑物内部空间的连通性。
