### 3.1.1 规则

规则是一种将条件与结果相关联的知识表示方法。规则可以表示为如下形式：

$$
IF \ condition \ THEN \ action
$$

其中，$condition$ 是一个或多个条件，$action$ 是一个或多个结果。

### 3.1.2 框架

框架是一种将知识组织为一种结构的知识表示方法。框架可以表示为一个或多个实例，每个实例都包含一个或多个槽位。槽位是实例的属性，可以具有一个或多个值。

### 3.1.3 描述符

描述符是一种将知识表示为一组属性和值的知识表示方法。描述符可以表示为一个或多个属性-值对。

### 3.1.4 概率模型

概率模型是一种将知识表示为概率分布的知识表示方法。概率模型可以表示为一个或多个概率分布，每个概率分布描述了一个或多个随机变量的概率。

## 4.1 知识抽取

首先，我们需要从文本中提取领域知识。我们可以使用自然语言处理技术来实现这一点。以下是一个简单的Python代码实例，用于提取文本中的实体和关系：

```python
import nltk
from nltk import pos_tag, word_tokenize

# 加载自然语言处理模型
nltk.download('punkt')
nltk.download('averaged_perceptron_tagger')

# 文本
text = "John loves Mary. Mary loves John."

# 分词
tokens = word_tokenize(text)

# 词性标注
tagged = pos_tag(tokens)

# 提取实体和关系
entities = []
relations = []

for word, tag in tagged:
    if tag.startswith('NN'):
        entities.append(word)
    elif tag.startswith('VB'):
        relations.append((entities[-1], word))
        entities.clear()

print(entities)
print(relations)
```

```

[('John', 'loves'), ('Mary', 'loves')]
```
