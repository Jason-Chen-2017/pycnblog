## 3.1 递归算法

递归算法是最直观的实现方式，它通过递归的方式逐步求解斐波那契数列的每一项。

递归算法的实现步骤如下：

1. 定义递归函数，如：
   ```
   def fib(n):
       if n == 0:
           return 0
       elif n == 1:
           return 1
       else:
           return fib(n-1) + fib(n-2)
   ```
2. 调用递归函数，如：
   ```
   print(fib(10)) # 输出 55
   ```

递归算法的时间复杂度为 O(2^n)，空间复杂度为 O(n)。由于递归算法的大量重复计算，其时间效率较低。

## 3.2 迭代算法

迭代算法是递归算法的一种改进，它通过循环的方式逐步求解斐波那契数列的每一项。

迭代算法的实现步骤如下：

1. 定义变量，如：
   ```
   a, b = 0, 1
   ```
2. 使用循环求解斐波那契数列的每一项，如：
   ```
   for i in range(10):
       print(a)
       a, b = b, a + b
   ```

迭代算法的时间复杂度为 O(n)，空间复杂度为 O(1)。相较于递归算法，迭代算法的时间效率较高。
