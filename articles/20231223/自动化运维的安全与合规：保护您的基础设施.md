## 4.1 配置管理示例

配置管理是自动化运维中的一个重要组件，可以帮助运维人员更快速地响应基础设施变更。以下是一个简单的配置管理示例：

```python
import json

class ConfigManager:
    def __init__(self):
        self.configs = {}

    def load_config(self, file_path):
        with open(file_path, 'r') as f:
            configs = json.load(f)
            self.configs = configs

    def get_config(self, key):
        return self.configs.get(key)

    def set_config(self, key, value):
        self.configs[key] = value
        with open('config.json', 'w') as f:
            json.dump(self.configs, f)
```

在上述示例中，我们定义了一个`ConfigManager`类，用于管理和控制基础设施配置。通过`load_config`方法，可以加载配置文件并将其存储在内存中。通过`get_config`方法，可以获取配置信息。通过`set_config`方法，可以修改配置信息并将其保存到配置文件中。

## 4.2 监控与报警示例

监控与报警是自动化运维中的另一个重要组件，可以帮助运维人员及时发现和处理基础设施故障。以下是一个简单的监控与报警示例：

```python
import time
from threading import Timer

class Monitor:
    def __init__(self, interval, callback):
        self.interval = interval
        self.callback = callback
        self.timer = None

    def start(self):
        self.timer = Timer(self.interval, self.callback)
        self.timer.start()

    def stop(self):
        if self.timer:
            self.timer.cancel()

    def schedule(self, delay, callback):
        self.stop()
        self.timer = Timer(delay, callback)
        self.timer.start()
```

在上述示例中，我们定义了一个`Monitor`类，用于实现基础设施监控与报警。通过`start`方法，可以启动定时任务，以固定时间间隔执行指定的回调函数。通过`stop`方法，可以停止定时任务。通过`schedule`方法，可以设置定时任务的执行时间和回调函数。
