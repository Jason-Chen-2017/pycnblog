# 4.具体代码实例和详细解释说明
在这里，我们将给出一个简单的Python代码实例，用于计算生物多样性指数（Biodiversity Index，BI）。生物多样性是自然资源保护的核心概念之一，生物多样性指数可以用于评估生物多样性的状况。

```python
import numpy as np

def biodiversity_index(species_num, individual_num):
    # 计算生物多样性指数（Shannon-Wiener指数）
    biodiversity_index = np.log(species_num) * np.log(individual_num) / (species_num - 1)
    return biodiversity_index

# 示例数据
species_num = 10
individual_num = 100

# 计算生物多样性指数
bi = biodiversity_index(species_num, individual_num)
print("生物多样性指数：", bi)
```

在这个代码实例中，我们首先导入了numpy库，然后定义了一个函数`biodiversity_index`，用于计算生物多样性指数。生物多样性指数是Shannon-Wiener指数的一种，它可以用来评估生物群体的多样性。生物多样性是自然资源保护的核心概念之一，因此计算生物多样性指数对于自然资源保护的决策非常重要。

在示例数据部分，我们给出了生物群体数量（species_num）和生物个体数量（individual_num）的示例数据。然后我们调用`biodiversity_index`函数，计算生物多样性指数，并打印出结果。
