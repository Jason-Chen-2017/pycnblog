# 2.核心概念与联系

## 2.1 超导性

超导性是指某些材料在零温度下，即使电流密度非常高，电阻也为零的现象。这种现象在常规材料中是不存在的，因此超导材料具有巨大的科学价值和应用前景。

## 2.2 超导材料类型

根据不同的物理原理，超导材料可以分为两类：

1. 典型超导材料：如氢氧化钠（Na2-xHxO2）、氢氧化锂（Li2-xHxO）等。这些材料的超导性是由于电子的超流现象，即电子在材料中的运动不受抵抗。

2. 高温超导材料：如钛碳超导材料（MgB2）、钛硫超导材料（Ce1-xTxCu2-ySr2yRuxY）等。这些材料的超导性是由于玄米体现象，即电子在材料中的运动受到一定的抵抗，但在特定条件下可以达到零电阻。

## 2.3 超导材料的性能指标

超导材料的性能主要由以下几个指标来衡量：

1. 超导温度（Tc）：超导材料的最高温度，在这个温度下仍具有超导性。

2. 耐压性能：超导材料在高压力下的性能，是否会损害其超导性。

3. 耐温性能：超导材料在高温下的性能，是否会影响其超导性。

4. 电导性：超导材料在零温度下的电导性，是否会影响其超导性。
