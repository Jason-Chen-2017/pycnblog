import requests
from bs4 import BeautifulSoup

## 4.3 数据处理

### 4.3.1 Python数据处理实例

我们可以使用Python编写一个脚本来处理能源使用数据。这个脚本可以对电力消费数据、燃气消费数据、水电量数据等进行清洗、整理、归一化等处理。

```python
import pandas as pd

# 读取CSV文件
data = pd.read_csv('energy_data.csv')

# 数据清洗
data = data.dropna()

# 数据整理
data['power_consumption'] = data['power_consumption'].astype(float)
data['gas_consumption'] = data['gas_consumption'].astype(float)
data['water_consumption'] = data['water_consumption'].astype(float)

# 数据归一化
data['power_consumption'] = (data['power_consumption'] - data['power_consumption'].mean()) / data['power_consumption'].std()
data['gas_consumption'] = (data['gas_consumption'] - data['gas_consumption'].mean()) / data['gas_consumption'].std()
data['water_consumption'] = (data['water_consumption'] - data['water_consumption'].mean()) / data['water_consumption'].std()

print(data)
```

## 4.4 数据分析

### 4.4.1 Python数据分析实例

我们可以使用Python编写一个脚本来分析能源使用数据。这个脚本可以对电力消费数据、燃气消费数据、水电量数据等进行分析、预测、优化等操作。

```python
import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression

# 读取CSV文件
data = pd.read_csv('energy_data.csv')

# 时间序列分析
data['date'] = pd.to_datetime(data['date'])
data.set_index('date', inplace=True)

# 线性回归模型
X = data.index.values.reshape(-1, 1)
y = data['power_consumption'].values
model = LinearRegression()
model.fit(X, y)

# 预测
X_future = pd.date_range(start='2021-01-02', end='2021-12-31').values.reshape(-1, 1)
y_future = model.predict(X_future)

print(y_future)
```
