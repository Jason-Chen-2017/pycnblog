### 4.1 代码实例

```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram, plot_bloch_vector

# 初始化量子电路
qc = QuantumCircuit(2, 2)

# 创建量子态
qc.h(0)
qc.cx(0, 1)

# 绘制量子电路
plot_histogram(qc.draw())

# 执行量子电路
simulator = Aer.get_backend('qasm_simulator')
qobj = assemble(transpile(qc, simulator), shots=1024)
result = simulator.run(qobj).result()

# 绘制结果
plot_histogram(result.get_counts())
```

### 4.2 代码解释

1. 首先，我们导入了`numpy`和`qiskit`库。
2. 然后，我们创建了一个量子电路`qc`，包含两个量子比特。
3. 接着，我们在第一个量子比特上应用了一个`Hadamard`门，实现了量子叠加。
4. 然后，我们将第一个量子比特与第二个量子比特用`CNOT`门连接起来，实现了量子纠缠。
5. 接下来，我们使用`qasm_simulator`后端执行量子电路，并设置了1024次样本。
6. 最后，我们绘制了量子电路和结果，可以看到量子态传输的实现过程。
