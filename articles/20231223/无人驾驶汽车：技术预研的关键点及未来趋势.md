# 2.核心概念与联系

无人驾驶汽车是指一种无需人工干预的汽车，它可以根据预定目的地自主决策、规划路线、控制速度、避免障碍物等，实现从起点到终点的自主驾驶。无人驾驶汽车的核心概念包括：自主驾驶、智能化、安全性、可靠性、效率等。

无人驾驶汽车的技术要素主要包括：

1. 传感器技术：包括雷达、摄像头、激光雷达、超声波等传感器，用于获取车辆周围的环境信息。
2. 计算机视觉技术：用于识别和分析车辆周围的物体、人、道路标记等。
3. 位置定位技术：包括GPS、激光雷达定位等方法，用于确定车辆的位置和方向。
4. 路径规划技术：用于根据车辆的目的地和环境信息，计算出最佳的路线和速度。
5. 控制技术：包括电子控制系统、动力系统等，用于实现车辆的自主驾驶。
6. 安全保障技术：包括碰撞防护、火灾防护、电源保护等，用于保障车辆的安全性。
