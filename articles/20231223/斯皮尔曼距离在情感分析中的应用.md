# 2.核心概念与联系

## 2.1 斯皮尔曼距离（Jaccard Similarity）

斯皮尔曼距离（Jaccard Similarity）是一种度量两个集合之间相似性的指标。给定两个集合A和B，斯皮尔曼距离可以定义为：
$$
J(A, B) = \frac{|A \cap B|}{|A \cup B|}
$$
其中，|A ∩ B|表示A和B的交集的大小，|A ∪ B|表示A和B的并集的大小。斯皮尔曼距离的范围在0到1之间，其中0表示两个集合完全不相似，1表示两个集合完全相似。

在本节中，我们将通过一个具体的代码实例来演示如何使用斯皮尔曼距离在情感分析中。

```python
import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.metrics import jaccard_score

# 文本数据
texts = ["I love this product", "This is a terrible product", "I am happy with this purchase"]

# 文本预处理和特征提取
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

# 计算斯皮尔曼距离
jaccard_scores = jaccard_score(X, X, average='weighted')

print("Jaccard Scores:", jaccard_scores)
```
