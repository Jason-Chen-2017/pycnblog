# 2.核心概念与联系

## 2.1 量子密钥分发（QKD）

量子密钥分发（Quantum Key Distribution, QKD）是一种基于量子物理原理的密钥分发方法，它可以实现两个或多个远程用户之间的安全通信。QKD的核心概念是利用量子比特的特性，如无法复制、不可知和状态叠加等，来实现信息的完全安全。

## 2.2 量子比特和经典比特的区别

量子比特（Quantum Bit, Qubit）和经典比特（Classical Bit, Bit）是两种不同的信息存储和传输方式。经典比特只能取0或1的状态，而量子比特可以存储0和1的混合状态，并且可以通过量子运算实现更高效的信息处理。

## 2.3 量子通信与经典通信的联系

量子通信和经典通信在传输方式上有很大的不同，但它们在实际应用中可以相互补充。例如，量子通信可以用于实现安全的密钥分发，而经典通信可以用于实现数据的传输和处理。因此，在医疗保健行业中，量子通信和经典通信可以相互协同，实现更高效和安全的数据传输。

# 3.核心算法原理和具体操作步骤以及数学模型公式详细讲解

## 3.1 BB84协议

BB84协议是一种最早的量子密钥分发协议，它的核心算法原理如下：

1. 发送方（Alice）从一个量子比特串中选择一部分作为信息比特，并将其余部分作为检测比特。信息比特和检测比特的状态是随机的。
2. 发送方（Alice）将信息比特使用单 photon 发送给接收方（Bob），将检测比特使用不同的光波长发送给一个监控站。
3. 接收方（Bob）对收到的信息比特进行基础测量，并将测试结果保存。
4. 监控站对收到的检测比特进行基础测量，并将测试结果发送给接收方（Bob）。
5. 发送方（Alice）和接收方（Bob）通过公开渠道交换检测比特的测试结果，并对信息比特进行错误纠正。

BB84协议的数学模型公式如下：

$$
P_{Z_B^n}(z_b^n|x^n)=2^{-n}\sum_{z_b^n}(-1)^{z_b^n}\rho_{Z_B^n|X^n}(z_b^n|x^n)
$$

其中，$P_{Z_B^n}(z_b^n|x^n)$ 是接收方（Bob）对信息比特的概率分布，$x^n$ 是发送方（Alice）发送的信息比特，$z_b^n$ 是接收方（Bob）测得的结果，$\rho_{Z_B^n|X^n}(z_b^n|x^n)$ 是相关概率分布。

## 3.2 错误纠正和密钥舍弃

在量子密钥分发过程中，由于通信环境的干扰，可能会出现错误的比特传输。因此，需要进行错误纠正和密钥舍弃操作。

1. 错误纠正：发送方（Alice）和接收方（Bob）通过公开渠道交换检测比特的测试结果，并对信息比特进行错误纠正。
2. 密钥舍弃：如果错误率过高，发送方（Alice）和接收方（Bob）可以对密钥进行舍弃，并重新进行量子密钥分发。
