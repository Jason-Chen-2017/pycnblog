# 2. 核心概念与联系

在这一节中，我们将介绍人类基因组的核心概念和联系。

## 2.1 基因组

基因组是一个组织或个体的所有染色质的总称。染色质是一个细胞的核的内部结构，包含了所有的基因。基因组是一个非常大的DNA分子，它包含了所有的遗传信息。

## 2.2 基因

基因是基因组中的一小部分，它们编码了特定的功能。基因可以被认为是遗传信息的单位，它们决定了一个生物的特征和功能。

## 2.3 核酸

核酸是基因组的构建块。核酸包括脂碱类核苷酸和糖碱类核苷酸。核苷酸是DNA的主要构成单位，它们可以组合成双螺旋结构的DNA分子。

## 2.4 遗传代码

遗传代码是基因组中的一系列规则，它们决定了如何将基因组信息转换为蛋白质。遗传代码包括三个基本要素：启动子、终止子和氨基酸氨基酸编码表。

## 2.5 染色质

染色质是基因组的物理结构。染色质是一个细胞的核的内部结构，它包含了所有的基因。染色质可以被分为两类：线性染色质和环状染色质。

## 2.6 遗传病

遗传病是由基因组中的变异引起的疾病。遗传病可以是单基因遗传病，也可以是多基因遗传病。
