## 3.1 排序算法
排序算法是计算机科学的基础，在面试中常被测试。常见的排序算法有：冒泡排序、选择排序、插入排序、归并排序、快速排序等。这些算法的时间复杂度和空间复杂度各异，在面试中需要熟练掌握。

### 3.1.1 冒泡排序
冒泡排序是一种简单的排序算法，它通过多次交换相邻的元素来排序。冒泡排序的时间复杂度为O(n^2)，空间复杂度为O(1)。

具体操作步骤如下：

1. 从第一个元素开始，与后续的每个元素进行比较。
2. 如果当前元素大于后续元素，交换它们的位置。
3. 重复上述操作，直到整个数组排序完成。

### 3.1.2 选择排序
选择排序是一种简单的排序算法，它通过多次选择最小（或最大）元素来排序。选择排序的时间复杂度为O(n^2)，空间复杂度为O(1)。

具体操作步骤如下：

1. 从数组的第一个元素开始，找到最小的元素。
2. 与数组的第一个元素交换位置。
3. 重复上述操作，直到整个数组排序完成。

### 3.1.3 插入排序
插入排序是一种简单的排序算法，它通过将元素插入到已排好序的子数组中来排序。插入排序的时间复杂度为O(n^2)，空间复杂度为O(1)。

具体操作步骤如下：

1. 将第一个元素视为有序子数组。
2. 从第二个元素开始，将它与有序子数组中的元素进行比较。
3. 如果当前元素小于有序子数组中的元素，将它插入到有序子数组的适当位置。
4. 重复上述操作，直到整个数组排序完成。

### 3.1.4 归并排序
归并排序是一种高效的排序算法，它通过将数组分割成两个子数组，递归地排序子数组，然后合并子数组来排序。归并排序的时间复杂度为O(nlogn)，空间复杂度为O(n)。

具体操作步骤如下：

1. 将数组分割成两个子数组。
2. 递归地对子数组进行排序。
3. 合并子数组，得到排序的数组。

### 3.1.5 快速排序
快速排序是一种高效的排序算法，它通过选择一个基准元素，将数组分割成两个子数组，递归地排序子数组，然后合并子数组来排序。快速排序的时间复杂度为O(nlogn)，空间复杂度为O(logn)。

具体操作步骤如下：

1. 选择一个基准元素。
2. 将小于基准元素的元素放在基准元素的左侧，大于基准元素的元素放在基准元素的右侧。
3. 递归地对左侧和右侧的子数组进行排序。
4. 合并子数组，得到排序的数组。

## 3.2 搜索算法
搜索算法是计算机科学的基础，在面试中常被测试。常见的搜索算法有：深度优先搜索、广度优先搜索、二分搜索等。这些算法的时间复杂度和空间复杂度各异，在面试中需要熟练掌握。

### 3.2.1 深度优先搜索
深度优先搜索是一种搜索算法，它通过从当前节点出发，深入探索可能的路径来搜索目标。深度优先搜索的时间复杂度为O(b^d)，其中b是分支因子，d是深度。

具体操作步骤如下：

1. 从起始节点开始。
2. 访问当前节点。
3. 从当前节点出发，深入探索可能的路径。
4. 如果找到目标，返回路径；否则，回溯并尝试其他路径。

### 3.2.2 广度优先搜索
广度优先搜索是一种搜索算法，它通过从当前节点出发，广度先探索可能的路径来搜索目标。广度优先搜索的时间复杂度为O(b^d)，其中b是分支因子，d是深度。

具体操作步骤如下：

1. 从起始节点开始。
2. 访问当前节点。
3. 将当前节点的邻居加入队列。
4. 如果找到目标，返回路径；否则，将队列中的下一个节点作为当前节点，重复上述操作。

### 3.2.3 二分搜索
二分搜索是一种搜索算法，它通过将搜索区间分割成两个子区间来搜索目标。二分搜索的时间复杂度为O(logn)。

具体操作步骤如下：

1. 将搜索区间分割成两个子区间。
2. 找到中间元素。
3. 如果中间元素等于目标，返回其索引。
4. 如果中间元素小于目标，将搜索区间更新为右子区间。
5. 如果中间元素大于目标，将搜索区间更新为左子区间。
6. 重复上述操作，直到找到目标或搜索区间为空。

## 3.3 动态规划
动态规划是一种解决决策过程问题的方法，它通过将问题分解成子问题，递归地解决子问题来得到最优解。动态规划的应用范围广泛，在面试中也是常见的题目类型。

### 3.3.1 0-1 背包问题
0-1 背包问题是动态规划的一个典型问题，它要求在一个有限的物品集合中选择一些物品放入背包，使背包的重量不超过限制值，并最大化价值。

具体操作步骤如下：

1. 将物品按价值从大到小排序。
2. 创建一个二维数组dp，其中dp[i][j]表示使用前i个物品，背包容量为j时的最大价值。
3. 遍历物品，对于每个物品，遍历背包容量。
4. 如果当前物品的重量小于或等于背包容量，则更新dp[i][j]的值为max(dp[i-1][j], v[i] + dp[i-1][j-w[i]])，其中v[i]是当前物品的价值，w[i]是当前物品的重量。
5. 返回dp[n][m]，其中n是物品数量，m是背包容量。

### 3.3.2  longest common subsequence (LCS)
LCS问题是动态规划的一个典型问题，它要求找到一个序列中的子序列，在另一个序列中同时出现。LCS问题的应用范围广泛，如文本编辑、序列对齐等。

具体操作步骤如下：

1. 创建一个二维数组dp，其中dp[i][j]表示第一个序列的前i个元素和第二个序列的前j个元素的LCS长度。
2. 遍历第一个序列和第二个序列，对于每个元素，遍历对应的序列。
3. 如果当前元素相等，则更新dp[i][j]的值为dp[i-1][j-1] + 1。
4. 如果当前元素不相等，则更新dp[i][j]的值为max(dp[i-1][j], dp[i][j-1])。
5. 返回dp[n][m]，其中n是第一个序列的长度，m是第二个序列的长度。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```
### 4.1.2 选择排序
```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
```
### 4.1.3 插入排序
```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i-1
        while j >= 0 and key < arr[j]:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr
```
### 4.1.4 归并排序
```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]
    left = merge_sort(left)
    right = merge_sort(right)
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```
### 4.1.5 快速排序
```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

## 4.2 搜索算法实例
### 4.2.1 深度优先搜索
```python
def dfs(graph, start):
    visited = set()
    stack = [start]
    while stack:
        vertex = stack.pop()
        if vertex not in visited:
            visited.add(vertex)
            stack.extend(graph[vertex] - visited)
    return visited
```
### 4.2.2 广度优先搜索
```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    while queue:
        vertex = queue.popleft()
        if vertex not in visited:
            visited.add(vertex)
            queue.extend(graph[vertex] - visited)
    return visited
```

## 4.3 动态规划实例
### 4.3.1 0-1 背包问题
```python
def knapsack(w, v, n):
    dp = [[0 for _ in range(n+1)] for _ in range(n+1)]
    for i in range(1, n+1):
        for j in range(1, n+1):
            if w[i] <= j:
                dp[i][j] = max(dp[i-1][j], v[i] + dp[i-1][j-w[i]])
            else:
                dp[i][j] = dp[i-1][j]
    return dp[n][n]
```
### 4.3.2 LCS
```python
def lcs(X, Y):
    m = len(X)
    n = len(Y)
    dp = [[0] * (n+1) for _ in range(m+1)]
    for i in range(m+1):
        for j in range(n+1):
            if i == 0 or j == 0:
                dp[i][j] = 0
            elif X[i-1] == Y[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]
```
