# 2.核心概念与联系

在深度学习领域，微调（fine-tuning）是指在预训练模型的基础上，对模型的一部分或全部参数进行再训练，以适应新的任务。通常，预训练模型是在大规模的数据集上进行训练的，例如ImageNet等。预训练模型已经学习到了一定的特征表示能力，因此在新任务上，我们只需要对模型的一部分参数进行微调，以适应新的领域特点，从而提高模型的性能。

在视频分析中，微调的视频分析模型可以帮助模型在特定领域的视频理解能力得到提升。通过对预训练模型的微调，我们可以使模型更加专门化，从而在特定领域的视频理解能力得到提升。
