# 4.具体代码实例和详细解释说明

## 4.1 音乐生成代码实例
以Python语言为例，下面是一个简单的音乐生成代码实例：

```python
import random

def generate_music():
    notes = ['C', 'D', 'E', 'F', 'G', 'A', 'B']
    rhythm = [1, 2, 3, 4, 1, 2, 3]
    music = []
    for i in range(10):
        note = random.choice(notes)
        rhythm_value = random.choice(rhythm)
        music.append((note, rhythm_value))
    return music

music = generate_music()
print(music)
```

在上述代码中，我们首先定义了音符和节奏，然后通过随机选择音符和节奏来生成音乐。最后，我们打印出生成的音乐。

## 4.2 艺术生成代码实例
以Python语言为例，下面是一个简单的艺术生成代码实例：

```python
import random
import matplotlib.pyplot as plt

def generate_art():
    width = 800
    height = 600
    colors = ['red', 'green', 'blue', 'yellow', 'purple', 'orange']
    art = []
    for i in range(width):
        x = i
        y = random.randint(0, height)
        color = random.choice(colors)
        art.append((x, y, color))
    return art

art = generate_art()
plt.scatter(*zip(*art))
plt.show()
```

在上述代码中，我们首先定义了画布的大小和颜色，然后通过随机生成点的位置和颜色来生成艺术作品。最后，我们使用Matplotlib库绘制生成的艺术作品。
