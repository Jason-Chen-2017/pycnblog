# 2. 宏平均的核心概念与联系
宏平均是一种衡量数据集离散性的统计量，用于衡量数据点与其均值之间的差异。它的公式为：

$$
MAD = \frac{1}{n} \sum_{i=1}^{n} |x_i - \bar{x}|
$$

其中，$x_i$ 表示数据集中的每个数据点，$\bar{x}$ 表示数据集的均值，$n$ 表示数据集的大小。宏平均与均值差（Mean Difference, MD）和均值绝对差（Mean Absolute Difference, MAD）等概念相关，它们都用于衡量数据集的离散性。

```python
import numpy as np

def mean_absolute_deviation(data):
    n = len(data)
    mean = np.mean(data)
    mad = np.mean(np.abs(data - mean))
    return mad
```
