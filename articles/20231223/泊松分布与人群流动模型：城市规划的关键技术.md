# 2.核心概念与联系
## 2.1 泊松分布
泊松分布是一种概率分布，用于描述连续随机变量取值的概率密度函数。它主要应用于统计学和概率论中，用于描述事件发生的概率。泊松分布的核心公式为：

$$
P(X=k) = \frac{e^{-\lambda}\lambda^k}{k!}
$$

其中，$P(X=k)$ 表示取值为 $k$ 的概率，$\lambda$ 表示事件发生的平均次数，$k$ 表示事件发生的次数。

## 2.2 人群流动模型
人群流动模型是一种用于描述人群在城市空间中运动的模型。它主要应用于城市规划和交通工程中，用于预测人群在不同时间和空间下的运动规律。人群流动模型的核心包括人群的运动规律、人群的分布规律以及人群的交互规律。

# 4.具体代码实例和详细解释说明
在本节中，我们将通过一个具体的代码实例来展示如何使用泊松分布来描述人群在城市空间中的分布。

## 4.1 代码实例
```python
import numpy as np
import matplotlib.pyplot as plt

# 设置城市空间的大小
city_size = 100

# 设置人群的平均密度
avg_density = 10

# 生成随机的人群分布数据
random_data = np.random.poisson(avg_density, city_size)

# 绘制人群分布图
plt.hist(random_data, bins=city_size, density=True)
plt.xlabel('人群密度')
plt.ylabel('概率')
plt.show()
```

## 4.2 详细解释说明
在上述代码中，我们首先导入了 numpy 和 matplotlib.pyplot 两个库，用于生成随机的人群分布数据和绘制人群分布图。然后，我们设置了城市空间的大小和人群的平均密度。接着，我们使用了 numpy 库中的 `np.random.poisson` 函数生成了随机的人群分布数据，其中的平均密度为 `avg_density`。最后，我们使用了 matplotlib.pyplot 库中的 `plt.hist` 函数绘制了人群分布图，并设置了 x 轴和 y 轴的标签。
