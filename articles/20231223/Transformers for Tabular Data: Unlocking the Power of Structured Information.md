# 2. 核心概念与联系

在深入探讨Transformer模型在表格数据处理中的应用之前，我们需要先了解一下Transformer模型的基本概念。

Transformer模型是2017年由Vaswani等人提出的一种新型的神经网络架构，它主要应用于自然语言处理任务。其核心思想是将序列到序列的编码和解码过程分为两个独立的子任务：编码器和解码器。编码器负责将输入序列（如文本）编码为固定长度的向量，解码器则将这些向量解码为目标序列（如翻译）。

在表格数据处理中，数据通常以表格形式存储，每行表示一个实例，每列表示一个特征。因此，将Transformer模型应用于表格数据处理的关键在于如何将表格数据转换为可以被Transformer模型处理的格式。
