# 4.具体代码实例和详细解释说明

在这里，我们给出一个具体的代码实例，以及其详细解释说明。

```python
import numpy as np

# 定义一个矩阵A
A = np.array([[4, 3, 2],
              [3, 2, 1],
              [2, 1, 1]])

# 使用LU分解算法求解矩阵A的LU分解
L, U = np.linalg.lu(A)

# 打印L矩阵和U矩阵
print("L矩阵：", L)
print("U矩阵：", U)
```

在这个例子中，我们定义了一个3x3的矩阵A，然后使用numpy库的`linalg.lu`函数求解矩阵A的LU分解。最终，我们打印了L矩阵和U矩阵。
