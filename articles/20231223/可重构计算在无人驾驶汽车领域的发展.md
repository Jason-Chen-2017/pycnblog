### 4.1.2 具体代码实例

我们将通过一个简单的Python代码实例来介绍可重构计算的具体代码实例和详细解释说明。

```python
import cv2
import numpy as np

# 读取图像

# 将图像转换为HSV颜色空间
hsv = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)

# 设置车辆的颜色范围
lower_color = np.array([0, 100, 50])
upper_color = np.array([255, 255, 180])

# 使用颜色滤波器提取车辆
mask = cv2.inRange(hsv, lower_color, upper_color)

# 使用形状滤波器提取车辆
mask = cv2.erode(mask, np.ones((5, 5), np.uint8), iterations=1)
mask = cv2.dilate(mask, np.ones((5, 5), np.uint8), iterations=1)

# 找到车辆的位置
contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

# 绘制车辆的边界框
for contour in contours:
    x, y, w, h = cv2.boundingRect(contour)
    cv2.rectangle(image, (x, y), (x + w, y + h), (0, 255, 0), 2)

# 显示结果
cv2.imshow('Car Detection', image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

在这个代码实例中，我们首先读取一张包含车辆的图像，并将其转换为HSV颜色空间。然后，我们设置车辆的颜色范围，并使用颜色滤波器提取车辆。接下来，我们使用形状滤波器进一步筛选车辆。最后，我们找到车辆的位置，并绘制车辆的边界框。
