# 2.核心概念与联系

## 2.1 量子霍尔效应（Quantum Hall Effect, QHE）

量子霍尔效应是一种在低温下在二维电子涂层中发生的现象，它是由德国科学家Klaus von Klitzing在1980年发现的。QHE在电子涂层中的运动受到电场和磁场的影响，导致电子在磁场垂直于涂层的方向移动时，形成一系列稳定的能级。这些能级之间的差值被称为霍尔级（Hall plateaus），它们是整数倍的基本电子霍尔常数（elementary charge）。

QHE的发现为微波技术提供了一种精确的电阻标准，因为它的精度可以达到一百万分之一。此外，QHE还在量子计算、量子通信和量子感知技术中发挥着重要作用。
