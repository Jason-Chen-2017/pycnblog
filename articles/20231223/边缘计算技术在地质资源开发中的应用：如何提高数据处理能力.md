## 4.1 实时监测案例
在这个案例中，我们将使用Python编程语言，使用Flask框架搭建一个简单的边缘计算服务。具体代码实例如下：

```python
from flask import Flask, request, jsonify
import time

app = Flask(__name__)

@app.route('/monitor', methods=['GET'])
def monitor():
    data = request.args.get('data')
    result = {'status': 'success', 'data': data, 'time': time.time()}
    return jsonify(result)

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
```

在这个案例中，我们使用Flask框架搭建了一个简单的边缘计算服务，接收客户端传递的数据，并将数据及当前时间返回给客户端。通过这个简单的案例，我们可以看到边缘计算技术在地质资源开发中的应用。
