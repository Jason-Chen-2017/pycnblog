# 2.核心概念与联系

集体行为研究关注于团队、组织和社会系统中人类行为的过程和结果。它旨在理解人类行为在团队和组织中的复杂性，并提出有效的方法来改进团队和组织的效率和成功。集体行为研究的核心概念包括：

- 团队动态：团队动态是指团队成员之间的相互作用和相互依赖关系。团队动态可以影响团队的性能和创新能力。
- 团队文化：团队文化是指团队成员共享的价值观、信仰和行为规范。团队文化可以影响团队成员的决策和行为。
- 团队领导：团队领导是指团队领导者的领导风格和领导方法。团队领导可以影响团队的成员满意度和团队的性能。
- 团队成员：团队成员是指团队中的每个人。团队成员的能力、动机和行为可以影响团队的性能和创新能力。
