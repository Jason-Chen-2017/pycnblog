### Blast算法的Python实现

Blast算法的Python实现如下：

```python
from Bio import pairwise2
from Bio.SubsMat import MatrixInfo as matrix

def blast(seq1, seq2, matrix='blosum62', gap_penalty=-1):
    alignments = pairwise2.align.globalds(seq1, seq2, matrix, gap_penalty)
    score = 0
    for alignment in alignments:
        for alignment_row in alignment:
            score += matrix.score(alignment_row[0], alignment_row[1])
    return score
```

Blast算法的Python实现详细解释说明：

1. 导入Bio库中的pairwise2和MatrixInfo两个模块。
2. 定义一个名为blast的函数，接受两个序列seq1和seq2，以及一个矩阵matrix和一个Gap Penalty参数。
3. 使用pairwise2.align.globalds函数，对两个序列进行全局对齐。
4. 遍历对齐结果，计算对齐得分，并返回得分。
