```python
import numpy as np

def calculate_temperature(albedo, solar_constant, pressure, temperature):
    """
    计算气温
    """
    # 计算辐射率
    radiation = albedo * solar_constant * (1 - pressure)
    # 计算温度
    return temperature + radiation / (1 + pressure)

def main():
    # 输入参数
    albedo = 0.3
    solar_constant = 1361.0  # W/m^2
    pressure = 0.1
    temperature = 288.0

    # 计算气温
    new_temperature = calculate_temperature(albedo, solar_constant, pressure, temperature)
    print("New temperature:", new_temperature)

if __name__ == "__main__":
    main()
```

在`main`函数中，我们输入了参数，并调用`calculate_temperature`函数计算新的气温。最后，我们打印出新的气温。
