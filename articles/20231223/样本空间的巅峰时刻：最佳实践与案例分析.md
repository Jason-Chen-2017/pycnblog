## 4.3 生成采样的代码示例

以下是一个使用生成采样方法构建样本空间的代码示例：

```python
import numpy as np

# 生成一个二分类问题的样本空间
def generate_sample_space(n_samples, n_features):
    input_data = np.random.rand(n_samples, n_features)
    label_data = np.random.randint(0, 2, n_samples)
    sample_space = set()
    for i in range(n_samples):
        sample_space.add((input_data[i], label_data[i]))
    return sample_space

# 获取样本空间
sample_space = generate_sample_space(100, 2)

print(sample_space)
```

在这个示例中，我们首先定义了一个生成采样方法，然后通过这个方法获取了样本空间。最后，我们将样本空间打印出来。
