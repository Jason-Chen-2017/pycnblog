以下是一个简单的太阳能电池板的代码实例：

```python
class SolarCell:
    def __init__(self, input_power, output_power):
        self.input_power = input_power
        self.output_power = output_power

    def efficiency(self):
        return (self.output_power / self.input_power) * 100
```

这个代码实例定义了一个名为 `SolarCell` 的类，该类包含一个名为 `efficiency` 的方法，用于计算太阳能电池板的效率。

# 4.2 太阳能充电站的代码实例
以下是一个简单的太阳能充电站的代码实例：

```python
class SolarChargingStation:
    def __init__(self, solar_cells, battery):
        self.solar_cells = solar_cells
        self.battery = battery

    def charge(self):
        total_input_power = sum(cell.input_power for cell in self.solar_cells)
        total_output_power = sum(cell.output_power for cell in self.solar_cells)
        efficiency = sum(cell.efficiency() for cell in self.solar_cells) / total_input_power
        self.battery.charge(total_output_power * efficiency)
```

这个代码实例定义了一个名为 `SolarChargingStation` 的类，该类包含一个名为 `charge` 的方法，用于将太阳能充电到锂离子电池中。
