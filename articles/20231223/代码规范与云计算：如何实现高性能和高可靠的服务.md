## 4.具体代码实例和详细解释说明

在本节中，我们将通过一个简单的例子来说明如何编写代码规范的实现。

例子：实现一个简单的排序算法，即冒泡排序。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

这个例子中，我们实现了一个简单的冒泡排序算法。冒泡排序是一种简单的排序算法，它的时间复杂度为O(n^2)。在这个例子中，我们遵循了以下代码规范的要求：

3. 代码结构：我们将排序算法封装在一个函数中，使代码更具有可维护性。
