## 4.2 空间算法代码实例

### 4.2.1 列表程序

```python
def create_list(n):
    return [i for i in range(n)]

def create_list_with_values(n, value):
    return [value for _ in range(n)]
```

### 4.2.2 列表程序的空间复杂度分析

1. 创建一个长度为 n 的列表的空间复杂度为 O(n)。
2. 创建一个长度为 n 的列表，并将 value 赋值给每个元素的空间复杂度为 O(n)。
