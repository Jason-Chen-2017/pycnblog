## 4.1 线性回归
```python
import numpy as np

# 数据
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])

# 参数
beta = np.zeros(len(x))
alpha = 0.01

# 迭代次数
iterations = 1000

# 训练
for i in range(iterations):
    y_predict = np.dot(x, beta)
    error = y - y_predict
    gradient = 2 * np.dot(x.T, error)
    beta -= alpha * gradient

# 预测
x_test = np.array([6, 7, 8])
y_predict = np.dot(x_test, beta)
print(y_predict)
```
