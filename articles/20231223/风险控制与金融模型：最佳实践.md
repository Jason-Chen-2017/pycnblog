### 4.1 最小均方误差（LMSE）

```python
import numpy as np

def lmse(y_true, y_pred):
    error = y_true - y_pred
    return np.mean(error**2)

y_true = np.array([1, 2, 3, 4, 5])
y_pred = np.array([1.1, 1.9, 2.8, 3.7, 4.6])

print(lmse(y_true, y_pred))
```
