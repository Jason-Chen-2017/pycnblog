在这里，我们将提供一个简单的Python代码实例，展示如何使用马尔可夫链来生成随机图像。

```python
import numpy as np
import matplotlib.pyplot as plt

# 初始化图像
X = np.random.randint(0, 256, (100, 100, 3))

# 设置状态转移矩阵
P = np.array([[0.25, 0.25, 0.25, 0.25],
              [0.25, 0.25, 0.25, 0.25],
              [0.25, 0.25, 0.25, 0.25],
              [0.25, 0.25, 0.25, 0.25]])

# 生成随机图像
for t in range(10):
    X = X @ P

# 显示图像
plt.imshow(X)
plt.show()
```
