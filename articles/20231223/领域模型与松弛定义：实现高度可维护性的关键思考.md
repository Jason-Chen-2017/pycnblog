# 2. 核心概念与联系

## 2.1 领域模型

领域模型是一种用于表示问题领域的方法，它将问题领域的概念和关系映射到计算机程序中。领域模型可以帮助我们更好地理解问题领域，并使得系统更容易维护和扩展。

领域模型包括以下组件：

- 实体：问题领域中的具体对象，如用户、产品、订单等。
- 属性：实体的属性，如用户的名字、年龄、地址等。
- 关系：实体之间的关系，如用户购买了某个产品。
- 规则：问题领域中的约束和限制，如用户年龄必须在18岁以上。

## 2.2 松弛定义

松弛定义是一种允许程序在某些情况下接受不完美输入的方法，从而提高系统的可维护性和灵活性。松弛定义可以帮助我们处理实际应用中常见的不完美数据和不确定性问题。

松弛定义包括以下组件：

- 松弛规则：定义了程序可以接受的不完美输入。
- 松弛判断：根据松弛规则判断输入是否满足要求。
- 松弛处理：根据松弛判断结果处理不完美输入，如调整输入、补全缺失信息等。

```python
def is_valid_age(age):
```
