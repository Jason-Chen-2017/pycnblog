# 2.核心概念与联系

在气候变化与灾害应对中，数字化技术的核心概念主要包括：

1. 大数据技术：大数据技术是指利用计算机科学技术（如分布式系统、并行处理、存储系统等）来分析处理和挖掘大量、多样化、高速增长的数据，以挖掘隐藏的知识和模式。在气候变化与灾害应对中，大数据技术可以帮助我们收集、存储和分析气候数据、地震数据、洪水数据等，从而提供有效的预测和应对措施。
2. 人工智能技术：人工智能技术是指使用计算机程序模拟人类智能的技术，包括机器学习、深度学习、计算机视觉、自然语言处理等。在气候变化与灾害应对中，人工智能技术可以帮助我们建立预测模型、分析影响因素、优化灾害恢复策略等。
3. 互联网技术：互联网技术是指利用计算机网络连接和交换信息的技术，包括网络协议、网络安全、云计算等。在气候变化与灾害应对中，互联网技术可以帮助我们实现数据共享、应用软件开发、灾害救援协同等。
