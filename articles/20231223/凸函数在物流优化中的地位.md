# 2.核心概念与联系

## 2.1 凸函数的定义

凸函数是一种特殊的函数，它在其定义域内具有一定的凸性。形式上，对于一个实值函数f(x)，如果对于任意的x1、x2属于其定义域D，且0≤λ≤1时，有f(λx1+(1-λ)x2)≤λf(x1)+(1-λ)f(x2)，则称f(x)是一个凸函数。

凸函数的一个重要特点是它的梯度（或导数）一直指向函数的上凸。这意味着在任何点上，梯度方向都是函数值增加的方向。因此，在优化问题中，我们可以通过梯度下降算法来找到凸函数的最优解。

## 2.2 凸函数与非凸函数的区别

与凸函数相对应的是非凸函数，非凸函数在某些区间内的凸性被破坏。换句话说，非凸函数的梯度方向可能在不同点上发生变化，这使得在优化问题中找到最优解变得更加复杂。

凸函数和非凸函数的一个重要区别在于它们的极值。对于凸函数，极值只能出现在其边界或内部，而非凸函数的极值可能出现在内部、边界或者在函数定义域的任何其他点。这意味着在非凸函数问题中，我们需要考虑更多的可能性，从而增加了求解难度。

## 4.1 代码实例

我们考虑一个简单的线性凸函数f(x)=2x-3，其梯度为g(x)=2x-3。我们的目标是通过梯度下降算法找到这个函数的最优解。

以下是一个Python代码实例：

```python
import numpy as np

def f(x):
    return 2 * x - 3

def g(x):
    return 2 * x - 3

def gradient_descent(x0, alpha, iterations):
    x = x0
    for i in range(iterations):
        grad = g(x)
        x = x - alpha * grad
    return x

x0 = 1
alpha = 0.1
iterations = 1000

x_opt = gradient_descent(x0, alpha, iterations)
print("The optimal solution is:", x_opt)
```

在上述代码中，我们首先定义了函数f和g，然后定义了梯度下降算法的主要步骤。接着，我们设置了一个初始点x0=1，学习率alpha=0.1，以及迭代次数iterations=1000。最后，我们调用了梯度下降算法，并打印了最优解。
