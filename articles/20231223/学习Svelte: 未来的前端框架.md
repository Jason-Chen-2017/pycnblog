## 4. 具体代码实例和详细解释说明

### 4.1 一个简单的Svelte组件实例

以下是一个简单的Svelte组件实例：

```html
<script>
  let count = 0;

  const increment = () => {
    count += 1;
  };
</script>

<button on:click={increment}>
  Count is {count}
</button>
```

在这个实例中，我们定义了一个`count`状态变量和一个`increment`方法。当按钮被点击时，`increment`方法会被调用，并更新`count`状态变量。这会导致组件的UI自动更新，显示新的`count`值。

### 4.2 使用计算属性

Svelte还支持使用计算属性。计算属性允许开发人员基于组件的状态计算新的值。以下是一个使用计算属性的例子：

```html
<script>
  let count = 0;

  const doubleCount = $track(() => count * 2);
</script>

<p>Double count is {doubleCount}</p>
```

在这个实例中，我们定义了一个`doubleCount`计算属性，它基于`count`状态变量计算新的值。当`count`状态变量更改时，`doubleCount`计算属性会自动更新，并导致组件的UI更新。

### 4.3 使用组件

Svelte支持使用其他组件。这使得开发人员可以将代码重用，并构建更复杂的UI。以下是一个使用其他组件的例子：

```html
<script>
  import MyComponent from './MyComponent.svelte';
</script>

<MyComponent />
```

在这个实例中，我们导入了一个名为`MyComponent`的其他组件，并将其插入到组件中。这使得我们可以重用`MyComponent`的代码，并构建更复杂的UI。
