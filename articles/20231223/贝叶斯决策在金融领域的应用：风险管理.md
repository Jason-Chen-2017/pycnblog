# 2.核心概念与联系

## 2.1 贝叶斯决策理论
贝叶斯决策理论是一种概率模型，它基于贝叶斯定理来更新我们对未知参数的信念，从而进行最佳决策。贝叶斯定理表示为：

P(A|B) = P(B|A)P(A) / P(B)

其中，P(A|B) 是条件概率，表示在已知B时，A发生的概率；P(B|A) 是条件概率，表示在已知A时，B发生的概率；P(A) 和 P(B) 是不条件概率，表示A和B的发生概率。

贝叶斯决策理论的核心是利用已有的信息来更新我们对未知参数的信念，从而进行更有针对性的决策。
