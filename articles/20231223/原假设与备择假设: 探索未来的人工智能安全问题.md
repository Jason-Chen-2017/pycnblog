# 2.核心概念与联系
原假设（Original Assumption）与备择假设（Backup Assumption）是两种不同的假设方法，它们在人工智能安全问题上的应用主要体现在以下两方面：

1. 原假设：原假设是指在设计人工智能系统时，基于某些假设来构建系统的方法。这些假设可以是关于系统的行为、系统的知识、系统的能力等方面的假设。原假设的主要优点是它可以简化系统的设计过程，使得系统更加易于理解和维护。然而，原假设的主要缺点是它可能导致系统的安全性能不足，因为这些假设可能不能完全捕捉到实际情况的复杂性。

2. 备择假设：备择假设是指在设计人工智能系统时，基于某些备择方案来构建系统的方法。这些备择方案可以是关于系统的不同行为、系统的不同知识、系统的不同能力等方面的方案。备择假设的主要优点是它可以提高系统的安全性能，因为这些备择方案可以捕捉到实际情况的复杂性。然而，备择假设的主要缺点是它可能导致系统的设计过程更加复杂，使得系统更加难以理解和维护。
