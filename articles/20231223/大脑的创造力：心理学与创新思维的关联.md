# 4.具体代码实例和详细解释说明
在Python语言中，生成拓扑排序算法的一个具体实现如下：

```python
import random

def generate_graph():
    nodes = set()
    edges = set()
    graph = {}
    for i in range(10):
        node = random.randint(1, 100)
        nodes.add(node)
        for j in range(5):
            edge = (node, random.randint(1, 100))
            edges.add(edge)
        graph[node] = edges
    return graph

def topological_sort(graph):
    sorted_nodes = []
    visited = set()
    for node in graph:
        if node not in visited:
            dfs(node, graph, visited, sorted_nodes)
    return sorted_nodes

def dfs(node, graph, visited, sorted_nodes):
    visited.add(node)
    for edge in graph[node]:
        if edge[1] not in visited:
            dfs(edge[1], graph, visited, sorted_nodes)
    sorted_nodes.append(node)

graph = generate_graph()
sorted_nodes = topological_sort(graph)
print(sorted_nodes)
```

这个代码实例首先生成了一个随机的图形结构，然后对其进行拓扑排序。最后，输出了排序结果。通过分析排序结果，可以生成有关人类创造力的有意义的信息。
