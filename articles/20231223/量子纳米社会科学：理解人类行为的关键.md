# 4.具体代码实例和详细解释说明
在这里，我们将给出一个简单的量子随机数生成算法的Python代码实例：

```python
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 创建一个量子电路
qc = QuantumCircuit(1)

# 添加一个量子门
qc.h(0)

# 将量子电路编译为可执行的量子代码
qc = transpile(qc, Aer.get_backend('qasm_simulator'))

# 将量子电路编译为可执行的量子代码
qobj = assemble(qc)

# 使用量子模拟器执行量子电路
simulator = Aer.get_backend('qasm_simulator')
result = simulator.run(qobj).result()

# 绘制结果
plot_histogram(result.get_counts())
```

这个代码实例创建了一个简单的量子电路，包含一个 Hadamard 门。Hadamard 门将量子比特从$$|0\rangle$$变为$$|1\rangle$$，或者从$$|1\rangle$$变为$$|0\rangle$$。通过测量量子比特，我们可以得到一个0或1的随机数。
