# 2.核心概念与联系
## 2.1 希尔伯特空间简介
希尔伯特空间是一种抽象的数学空间，它可以用来表示和处理无限维向量的集合。希尔伯特空间的核心概念是基础系（basis），即一个线性无关的向量集合，可以用来唯一地表示空间中的任意向量。希尔伯特空间的一个重要特点是，它可以用有限个基向量来近似地表示无限维向量的集合，从而实现图像压缩和恢复等操作。

## 2.2 图像空间与希尔伯特空间的联系
图像空间是一个高维向量空间，其中向量表示图像的灰度或颜色信息。图像空间与希尔伯特空间之间的关系是，通过将图像空间映射到希尔伯特空间，可以实现图像的特征提取和处理。具体来说，可以将图像空间中的向量表示为希尔伯特空间中的基向量的线性组合，从而实现图像的表示、压缩、恢复、分类等操作。

## 3.4 数学模型公式
希尔伯特空间的数学模型可以用以下公式表示：
$$
H = \text{span}\{ \phi_1, \phi_2, \cdots, \phi_N \}
$$
其中，$H$ 表示希尔伯特空间，$\phi_i$ 表示基向量，$N$ 表示基向量的数量。

图像压缩和恢复的数学模型可以用以下公式表示：
$$
f(x, y) = \sum_{i=1}^{N} a_i \phi_i(x, y)
$$
$$
a_i = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x, y) \phi_i^*(x, y) dx dy
$$
其中，$f(x, y)$ 表示图像空间中的向量，$a_i$ 表示基向量与图像空间向量之间的内积，$\phi_i^*(x, y)$ 表示基向量的复共轭函数。

## 4.1 代码实例
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.fft import dct

# 读取图像

# 将图像转换为一维向量
img_1d = img.flatten()

# 计算二维DCT，得到希尔伯特空间表示
img_dct = dct(img_1d)

# 对希尔伯特空间表示进行压缩
coeff_idx = np.argsort(img_dct.real**2 + img_dct.imag**2)[:100]
img_dct_compressed = img_dct[coeff_idx]

# 对压缩后的希尔伯特空间表示进行逆变换，得到压缩后的图像
img_compressed = dct.idct(img_dct_compressed)

# 显示原图像和压缩后的图像
plt.subplot(1, 2, 1)
plt.imshow(img, cmap='gray')
plt.title('Original Image')

plt.subplot(1, 2, 2)
plt.imshow(img_compressed.reshape(img.shape), cmap='gray')
plt.title('Compressed Image')

plt.show()
```
## 4.2 解释说明
在上述代码实例中，我们首先读取了一张图像，并将其转换为一维向量。接着，我们计算了图像的二维DCT，得到了图像在希尔伯特空间的表示。然后，我们对希尔伯特空间表示进行了压缩，只保留了最大的100个系数。最后，我们对压缩后的希尔伯特空间表示进行了逆变换，得到了压缩后的图像。最后，我们显示了原图像和压缩后的图像，可以看到压缩后的图像与原图像相似，但是存储空间较小。
