### 4.1 简单的智能合约实例

我们将通过一个简单的智能合约实例来详细解释其实现过程。这个智能合约实例是一个简单的交易合约，它允许两个用户之间进行交易。

```solidity
pragma solidity ^0.5.0;

contract SimpleTrade {
    address public buyer;
    address public seller;
    uint public price;
    uint public quantity;

    event TradeCreated(address buyer, address seller, uint price, uint quantity);
    event TradeCompleted(address buyer, address seller, uint price, uint quantity);

    function createTrade(address _buyer, address _seller, uint _price, uint _quantity) public {
        require(_buyer != address(0) && _seller != address(0));
        require(_price > 0 && _quantity > 0);

        buyer = _buyer;
        seller = _seller;
        price = _price;
        quantity = _quantity;

        emit TradeCreated(buyer, seller, price, quantity);
    }

    function completeTrade() public {
        require(buyer != address(0) && seller != address(0));

        buyer.transfer(price * quantity);
        seller.transfer(price * quantity);

        emit TradeCompleted(buyer, seller, price, quantity);
    }
}
```

这个简单的智能合约实例包括以下几个部分：

- 合约定义：这个智能合约定义了一个名为`SimpleTrade`的合约，它包括一些状态变量（如`buyer`、`seller`、`price`和`quantity`）和一些事件（如`TradeCreated`和`TradeCompleted`）。
- 构造函数：这个智能合约包括一个名为`createTrade`的构造函数，它允许用户创建一个新的交易。这个构造函数包括一些要求条件，以确保输入的参数有效。
- 函数实现：这个智能合约包括一个名为`completeTrade`的函数，它允许用户完成一个交易。这个函数首先检查输入的参数有效，然后使用`transfer`函数将金额转移到买家和卖家的账户中。
