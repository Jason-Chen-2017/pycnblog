# 2.核心概念与联系
# 2.1 随机过程的基本概念
随机过程是一种描述随机变量序列变化规律的数学模型，它可以用来描述一系列随机变量的变化规律。随机过程可以被定义为一个随机变量序列 {X(t), t ∈ T}，其中 T 是一个索引集，X(t) 是一个随机变量，表示在时刻 t 的观测结果。

随机过程的主要特点包括：

1.随机过程是一种随机系统，其状态随时间的变化而变化。

2.随机过程的状态可以用随机变量序列来描述。

3.随机过程的状态随时间的变化遵循一定的规律。

4.随机过程的状态随时间的变化是独立同分布的。

随机过程的常见类型包括：

1.离散随机过程：时刻 t 的随机变量 X(t) 只能取有限或计数可能的值。

2.连续随机过程：时刻 t 的随机变量 X(t) 可以取任意的连续值。

3.离散时间随机过程：随机过程的时刻仅仅是离散的。

4.连续时间随机过程：随机过程的时刻是连续的。

例如，模糊图像可以通过将原始图像与一定模糊核进行卷积来生成，这个过程就是一个随机过程。噪声图像可以通过将原始图像与一定噪声源进行混合来生成，这个过程也是一个随机过程。随机图像可以通过将原始图像与一定随机变量进行乘法或加法来生成，这个过程也是一个随机过程。
