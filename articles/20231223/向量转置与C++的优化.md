# 4.具体代码实例和详细解释说明
在C++中，我们可以使用STL的transform函数来实现向量转置的操作。以下是一个具体的代码实例：

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> vec = {1, 2, 3, 4, 5};
    std::vector<int> transposed_vec(vec.size());

    std::transform(vec.begin(), vec.end(), transposed_vec.begin(), [](int x) {
        return x;
    });

    for (int i = 0; i < transposed_vec.size(); i++) {
        std::cout << transposed_vec[i] << " ";
    }

    return 0;
}
```

在这个代码实例中，我们首先创建了一个一维向量vec，然后创建了一个新的向量transposed_vec来存储转置后的数据。接下来，我们使用STL的transform函数将vec中的数据复制到transposed_vec中。最后，我们使用for循环来输出转置后的数据。
