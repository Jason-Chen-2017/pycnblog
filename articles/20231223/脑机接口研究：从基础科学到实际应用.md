## 4.具体代码实例和详细解释说明

### 4.1 代码实例

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成一组随机电活动信号
np.random.seed(0)
signal = np.random.rand(1000)

# 对电活动信号进行处理和分析
filtered_signal = np.abs(np.fft.fft(signal))

# 绘制电活动信号和过滤后的信号
plt.figure()
plt.subplot(2, 1, 1)
plt.plot(signal)
plt.title('Original Signal')

plt.subplot(2, 1, 2)
plt.plot(filtered_signal)
plt.title('Filtered Signal')

plt.show()
```

### 4.2 详细解释说明

这个代码实例首先导入了 numpy 和 matplotlib.pyplot 这两个库，然后生成了一组随机电活动信号。接着，对电活动信号进行了处理和分析，使用了 numpy 库中的 `np.fft.fft()` 函数对电活动信号进行了傅里叶变换，从而得到了过滤后的信号。最后，使用了 matplotlib.pyplot 库绘制了电活动信号和过滤后的信号，以便观察到处理后的信号是否有所改善。
