### 4.1 高温稳定性测试代码实例

以下是一个高温稳定性测试的代码实例：

```python
import numpy as np
import pandas as pd

# 读取高温稳定性测试数据
data = pd.read_csv('high_temperature_stability_data.csv')

# 对数据进行预处理
data['temperature'] = data['temperature'].astype(np.float64)
data['time'] = data['time'].astype(np.float64)
data['stability'] = data['stability'].astype(np.float64)

# 对数据进行分析
slope = np.polyfit(data['temperature'], data['stability'], 1)
r_squared = 1 - np.sum((data['stability'] - np.polyval(slope, data['temperature'])) ** 2) / np.sum((data['stability'] - np.mean(data['stability'])) ** 2)

print('高温稳定性测试结果：')
print('斜率：', slope)
print('R²：', r_squared)
```
