## 3.1 前向差分法
前向差分法是求导法则的一种数值求解方法，它通过对函数的前向差分来近似求导。前向差分法的公式如下：

$$
f'(x) \approx \frac{f(x + h) - f(x)}{h}
$$

其中，$f'(x)$ 是函数的导数，$h$ 是步长。

具体操作步骤如下：

1. 选择一个函数 $f(x)$ 和一个步长 $h$。
2. 计算 $f(x + h)$。
3. 使用公式计算 $f'(x)$。

## 3.2 后向差分法
后向差分法是求导法则的一种数值求解方法，它通过对函数的后向差分来近似求导。后向差分法的公式如下：

$$
f'(x) \approx \frac{f(x) - f(x - h)}{h}
$$

具体操作步骤如下：

1. 选择一个函数 $f(x)$ 和一个步长 $h$。
2. 计算 $f(x - h)$。
3. 使用公式计算 $f'(x)$。

## 3.3 中心差分法
中心差分法是求导法则的一种数值求解方法，它通过对函数的中心差分来近似求导。中心差分法的公式如下：

$$
f'(x) \approx \frac{f(x + h) - f(x - h)}{2h}
$$

具体操作步骤如下：

1. 选择一个函数 $f(x)$ 和一个步长 $h$。
2. 计算 $f(x + h)$ 和 $f(x - h)$。
3. 使用公式计算 $f'(x)$。

```python
import numpy as np

def forward_difference(f, x, h):
    return (f(x + h) - f(x)) / h

def backward_difference(f, x, h):
    return (f(x) - f(x - h)) / h

def central_difference(f, x, h):
    return (f(x + h) - f(x - h)) / (2 * h)

# 测试函数

    return np.exp(-x**2)

# 测试点

h = 0.1

print("前向差分法:", forward_difference(test_function, x, h))

```

1. 步长 $h$ 的选择对结果的准确性有影响，小的步长可以获得更准确的结果，但也会增加计算复杂度。
