2. 对于每个随机样本，计算其对应的价值。

以下是一个简单的Python代码实例：

```python
import numpy as np

# 设置参数
S = 100
r = 0.05
T = 1

# 生成随机样本

# 计算价值

print("蒙特卡洛估计的价值:", monte_carlo_value)
```
