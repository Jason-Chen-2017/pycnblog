# 2.核心概念与联系

## 2.1 图像处理

图像处理是指对图像进行各种操作，以改善其质量、提取有意义的信息和特征，或者为其他应用提供支持。图像处理可以分为两个主要部分：

1. 图像输入：将图像从物理世界转换为数字世界，以便进行数字处理。
2. 图像处理算法：应用于图像数据的数学和计算方法，以实现特定的处理目标。

图像处理的主要应用包括：

1. 图像增强：提高图像的质量和可读性，以便人类或计算机更好地理解。
2. 图像分割：将图像划分为多个部分，以便进一步的分析和处理。
3. 图像识别：识别图像中的特定对象和特征，以便进行自动化处理和决策。
4. 图像合成：创建新的图像，以满足特定的需求和要求。

## 2.2 计算机图形学

计算机图形学是一门研究如何创建、表示、存储、处理和显示图形和图像的科学。计算机图形学的主要应用包括：

1. 3D模型建模：创建三维模型，以便在计算机屏幕上进行显示和交互。
2. 渲染：根据3D模型和光照条件，计算每个像素的颜色和亮度，以便在屏幕上显示出真实的图像。
3. 动画：创建和播放动态图形和图像，以表达故事和概念。
4. 虚拟现实：通过实时的图形处理和交互，实现用户在计算机生成的虚拟世界中的沉浸式体验。
