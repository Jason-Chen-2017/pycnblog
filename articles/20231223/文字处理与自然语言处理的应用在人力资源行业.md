## 4.1 招聘信息处理

### 4.1.1 Python 代码实例

```python
import os
import zipfile
import shutil
from docx import Document

def unzip_file(zip_file_path, extract_path):
    with zipfile.ZipFile(zip_file_path, 'r') as zip_ref:
        zip_ref.extractall(extract_path)

def process_docx(docx_file_path):
    doc = Document(docx_file_path)
    for para in doc.paragraphs:
        print(para.text)

def main():
    zip_file_path = 'resume.zip'
    extract_path = 'resume'
    unzip_file(zip_file_path, extract_path)
    for file in os.listdir(extract_path):
        if file.endswith('.docx'):
            process_docx(os.path.join(extract_path, file))

if __name__ == '__main__':
    main()
```

### 4.1.2 详细解释说明

1. 导入相关库：`os`、`zipfile`、`shutil`、`docx`。
2. 定义一个函数 `unzip_file`，用于解压 zip 文件。
3. 定义一个函数 `process_docx`，用于处理 docx 文件。
4. 定义一个主函数 `main`，用于解压 zip 文件，并遍历文件夹中的 docx 文件，调用 `process_docx` 函数处理。
5. 调用主函数 `main`。

## 4.2 简历筛选

### 4.2.1 Python 代码实例

```python
import re
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

def preprocess_text(text):
    text = text.lower()
    text = re.sub(r'\d+', '', text)
    return text

def main():
    resumes = [
        '杰出的软件工程师，5年工作经验，熟悉 Python 和 Java。',
        '高级的软件工程师，6年工作经验，擅长 Python 和 C++。',
        '优秀的软件工程师，4年工作经验，熟悉 Java 和 C#。'
    ]
    preprocessed_resumes = [preprocess_text(resume) for resume in resumes]
    vectorizer = TfidfVectorizer()
    X = vectorizer.fit_transform(preprocessed_resumes)
    cosine_similarities = cosine_similarity(X)
    print(cosine_similarities)

if __name__ == '__main__':
    main()
```

### 4.2.2 详细解释说明

1. 导入相关库：`re`、`sklearn.feature_extraction.text`、`sklearn.metrics.pairwise`。
2. 定义一个函数 `preprocess_text`，用于对文本进行预处理。
3. 定义一个主函数 `main`，用于创建一个简历列表，对列表中的简历进行预处理，使用 TF-IDF 向量化，计算简历之间的相似度。
4. 调用主函数 `main`。
