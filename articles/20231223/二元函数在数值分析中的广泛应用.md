### 4.1 求函数的梯度和曲率

```python
import numpy as np

def f(x, y):
    return x**2 + y**2

def gradient(x, y):
    return np.array([2*x, 2*y])

def curvature(x, y):
    return 4

x, y = 1, 1
print("梯度:", gradient(x, y))
print("曲率:", curvature(x, y))
```

### 4.2 求函数的极值和零点

```python
from scipy.optimize import minimize

def f(x, y):
    return x**2 + y**2

x0 = np.array([1, 1])
res = minimize(f, x0)
print("极值:", res.fun)
print("极值点:", res.x)
```

### 4.3 求函数的积分

```python
import numpy as np

def f(x, y):
    return x*y

def integral(a, b):
    return (b**2 - a**2) / 2

a, b = 0, 1
print("积分:", integral(a, b))
```
