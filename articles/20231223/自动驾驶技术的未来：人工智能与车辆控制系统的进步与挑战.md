- 级4：完全自动驾驶，例如 Google的自动驾驶汽车

## 4.1 计算机视觉代码实例

在本节中，我们将提供一个计算机视觉代码实例，用于检测道路边缘。

```python
import cv2
import numpy as np

def sobel_edge_detection(image):
    # 获取图像的灰度图
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

    # 使用Sobel算子检测边缘
    sobelx = cv2.Sobel(gray, cv2.CV_64F, 1, 0, ksize=5)
    sobely = cv2.Sobel(gray, cv2.CV_64F, 0, 1, ksize=5)

    # 计算梯度的模
    magnitude = np.sqrt(sobelx ** 2 + sobely ** 2)

    # 使用阈值对梯度的模进行二值化
    ret, binary = cv2.threshold(magnitude, 100, 255, cv2.THRESH_BINARY)

    return binary
```

在上述代码中，我们首先使用OpenCV库将图像转换为灰度图。然后，我们使用Sobel算子检测图像中的边缘。最后，我们使用阈值对边缘的梯度进行二值化，以获取边缘图。

## 4.2 机器学习代码实例

在本节中，我们将提供一个机器学习代码实例，用于训练一个支持向量机（SVM）模型。

```python
import numpy as np
from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC

# 加载鸢尾花数据集
iris = datasets.load_iris()
X = iris.data
y = iris.target

# 数据预处理
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 训练测试数据集
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# 训练SVM模型
svm = SVC(kernel='linear')
svm.fit(X_train, y_train)

# 评估模型性能
accuracy = svm.score(X_test, y_test)
print(f'SVM模型的准确度：{accuracy:.2f}')
```

在上述代码中，我们首先加载鸢尾花数据集，并对数据进行预处理。然后，我们使用训练测试数据集将数据集划分为训练集和测试集。接着，我们使用支持向量机（SVM）算法训练一个线性核SVM模型。最后，我们使用测试数据集评估模型的性能。
