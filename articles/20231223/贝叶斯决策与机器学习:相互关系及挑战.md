## 3.1 贝叶斯决策
贝叶斯决策主要基于贝叶斯定理，它是一种基于概率的决策理论。贝叶斯定理可以表示为：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A|B)$ 表示条件概率，即给定事件 $B$ 发生的情况下，事件 $A$ 的概率；$P(B|A)$ 表示联合概率，即事件 $A$ 发生的情况下，事件 $B$ 的概率；$P(A)$ 和 $P(B)$ 分别表示事件 $A$ 和 $B$ 的单变量概率。

在贝叶斯决策中，我们通过计算条件概率和联合概率来做决策。具体的操作步骤如下：

1. 确定决策空间和结果空间：决策空间包括所有可能的决策选项，结果空间包括所有可能的结果。
2. 确定损失函数：损失函数用于衡量不同决策选项对应的不同结果带来的损失。
3. 计算贝叶斯决策规则：根据贝叶斯定理，计算每个决策选项对应的期望损失。
4. 选择最小期望损失的决策选项：根据贝叶斯决策规则，选择最小期望损失的决策选项作为最优决策。

## 4.1 贝叶斯决策实例
我们来看一个简单的贝叶斯决策实例，假设我们有一个医疗诊断问题，需要判断患者是否患上了癌症。我们有以下信息：

- 患者年龄大于50岁的概率为0.6，患上癌症的概率为0.05%。
- 患者年龄小于或等于50岁的概率为0.4，患上癌症的概率为0.1%。

我们需要计算，给定一个患者的年龄，他的癌症患病概率是多少。

```python
def bayes_decision(age):
    if age > 50:
        cancer_prob = 0.05 / 100
    else:
        cancer_prob = 0.1 / 100
    return cancer_prob

age = 45
print(f"患者年龄为{age}岁，癌症患病概率为{bayes_decision(age):.2%}")
```

## 4.2 机器学习实例
我们来看一个简单的机器学习实例，假设我们有一个简单的线性回归问题，需要预测房价。我们有以下数据：

- 房价：100000, 200000, 300000, 400000, 500000
- 面积：50, 80, 100, 120, 150

我们需要计算，给定一个房间的面积，它的预测房价是多少。

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据
X = np.array([50, 80, 100, 120, 150]).reshape(-1, 1)
y = np.array([100000, 200000, 300000, 400000, 500000])

# 训练模型
model = LinearRegression()
model.fit(X, y)

# 预测房价
room_size = 120
predicted_price = model.predict([room_size])

print(f"给定房间面积为{room_size}平方米，预测房价为{predicted_price[0]:.0f}万元")
```
