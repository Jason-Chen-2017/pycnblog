```python
import numpy as np

def f(x, y):
    return np.exp(-(x**2 + y**2))

def gradient(x, y):
    return np.array([-2*x*np.exp(-(x**2 + y**2)), -2*y*np.exp(-(x**2 + y**2))])

def update(x, y, alpha):
    return x - alpha * gradient(x, y)[0], y - alpha * gradient(x, y)[1]

x, y = 0, 0
alpha = 0.01
for i in range(1000):
    x, y = update(x, y, alpha)

print("x =", x, "y =", y)
```

在上述代码实例中，我们定义了一个凸性函数 f(x, y) = exp(-(x**2 + y**2))，并求解了其梯度。通过使用梯度下降法，我们可以得到 x 和 y 的最小值。在这个例子中，我们求解了一个简单的能量最小化问题。
