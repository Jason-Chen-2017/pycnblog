## 1.1 恒等变换的基本概念

恒等变换是指在某个集合上的一个映射，使得映射的输入与输出相等。在数学上，我们通常用$f(x) = x$来表示恒等变换，其中$x$是集合上的一个元素。恒等变换是一种特殊的映射，它保持集合的结构不变，即对于任何两个元素$x$和$y$，如果$x \sim y$，那么$f(x) \sim f(y)$也成立。

## 1.2 群论的基本概念

群论是一种数学结构，它由一个集合和一个二元运算组成。在群论中，集合元素具有以下四个基本性质：

1. 关联律：对于任意两个元素$a$和$b$，有$a \ast (b \ast c) = (a \ast b) \ast c$。
2. 单位元：存在一个元素$e$，使得对于任意元素$a$，有$a \ast e = e \ast a = a$。
3. 逆元：对于任意元素$a$，存在一个元素$b$，使得$a \ast b = b \ast a = e$。
4. 交换律：对于任意两个元素$a$和$b$，有$a \ast b = b \ast a$。

群论的一个重要应用是密码学中的对称密钥加密，例如AES算法。
