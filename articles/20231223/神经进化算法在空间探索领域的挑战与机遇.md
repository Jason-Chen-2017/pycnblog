# 4.具体代码实例和详细解释说明

在这里，我们给出一个简单的NEA代码实例，用于寻找二维空间中的最小值。代码实例如下：
```python
import numpy as np
import random

# 定义适应度函数
def fitness(x):
    return np.sqrt(x[0]**2 + x[1]**2)

# 初始化种群
population = [np.array([random.uniform(-10, 10), random.uniform(-10, 10)]) for _ in range(100)]

# 设置终止条件
max_generations = 1000

# 开始进化过程
for generation in range(max_generations):
    # 评估种群中每个个体的适应度
    fitness_values = [fitness(individual) for individual in population]

    # 选择繁殖
    selected_indices = np.argsort(fitness_values)
    selected_individuals = [population[i] for i in selected_indices[-25:]]

    # 变异
    mutation_rate = 0.1
    mutated_individuals = [individual + np.random.uniform(-1, 1, size=individual.shape) * mutation_rate for individual in selected_individuals]

    # 更新种群
    population = selected_individuals + mutated_individuals

# 输出最佳个体
best_individual = population[np.argmin(fitness_values)]
print("最佳个体: ", best_individual)
print("最小值: ", -best_individual)
```
这个代码实例中，我们首先定义了适应度函数，然后初始化种群，设置终止条件，开始进化过程。在进化过程中，我们首先评估种群中每个个体的适应度，然后选择繁殖，对选择出的个体进行变异，最后更新种群。在结束时，我们输出最佳个体和最小值。
