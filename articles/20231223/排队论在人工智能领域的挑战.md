## 4.具体代码实例和详细解释说明

以下是一个Python实现的先来先服务策略的排队系统示例代码：

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置参数
lambda_ = 5
mu = 3
t = np.arange(0, 100, 0.1)

# 初始化队列长度和平均排队时间
queue_length = 0
average_waiting_time = 0

# 模拟排队系统
for i in t:
    if np.random.rand() < lambda_ / len(t):
        queue_length += 1
        average_waiting_time += i
    if queue_length > 0 and np.random.rand() < mu / len(t):
        queue_length -= 1
        average_waiting_time -= i

# 绘制队列长度和平均排队时间曲线
plt.plot(t, queue_length, label='Queue Length')
plt.plot(t, average_waiting_time / len(t), label='Average Waiting Time')
plt.legend()
plt.show()
```

这个示例代码首先设置了到达率（λ）和服务率（μ），然后通过模拟排队系统，得到队列长度和平均排队时间的曲线图。可以看到，队列长度和平均排队时间都随着时间的增加而增加，但是在服务台空闲时，队列长度会随着时间的推移而减少。
