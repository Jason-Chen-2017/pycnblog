# 4.具体代码实例和详细解释说明

具体代码实例：

1. 数据提取：

```python
import re
import xml.etree.ElementTree as ET
import json

# 数据提取示例
def extract_data(source):
    if source == 'file':
        with open('data.txt', 'r') as f:
            data = f.read()
            return re.findall(r'\d+', data)
    elif source == 'xml':
        tree = ET.parse('data.xml')
        root = tree.getroot()
        data = [int(child.text) for child in root.findall('number')]
        return data
    elif source == 'json':
        with open('data.json', 'r') as f:
            data = json.load(f)
        return data['numbers']

```

2. 数据转换：

```python
# 数据转换示例
def transform_data(data, unit):
    if unit == 'm':
        return [d / 1000 for d in data]
    elif unit == 'km':
        return [d * 1000 for d in data]

```
