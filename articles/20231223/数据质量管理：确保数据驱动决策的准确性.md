### 4.1 数据清洗

假设我们有一个包含客户信息的数据集，其中有一列表示客户年龄的数据。我们发现这些数据中有一些年龄值是不合理的，例如，有些客户的年龄值为负数。我们需要对这些数据进行清洗，以确保数据的准确性。

```python
import pandas as pd

# 读取数据集
data = pd.read_csv('customer_info.csv')

# 对年龄列进行清洗
data['age'] = data['age'].replace(-99, None)
data['age'] = data['age'].fillna(data['age'].mean())
```

在这个例子中，我们首先使用pandas库读取数据集。然后，我们对年龄列进行清洗。首先，我们将负数年龄值替换为None。接着，我们使用fillna()函数填充缺失值，将缺失值替换为年龄列的平均值。
