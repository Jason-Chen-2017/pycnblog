# 2. 核心概念与联系

## 2.1 欧氏距离的定义

欧氏距离是一种度量空间中两点距离的方法，它可以用来衡量两个向量之间的距离。欧氏距离的公式定义为：

$$
d(x, y) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)^2}
$$

其中，$x$ 和 $y$ 是 $n$ 维向量，$x_i$ 和 $y_i$ 分别是向量 $x$ 和 $y$ 的第 $i$ 个元素。

## 2.2 图像识别与欧氏距离的联系

在图像识别中，我们通常将图像转换为数字形式，即数字图像。数字图像是由一个二维矩阵组成的，每个矩阵元素表示图像中的一个像素。为了将图像识别问题转化为数学模型，我们需要将图像中的特征抽取成向量。这些向量可以用来表示图像的特征，如颜色、纹理、形状等。

在图像识别中，我们通常使用欧氏距离来衡量两个特征向量之间的相似度。例如，在人脸识别中，我们可以将人脸的特征抽取成一个向量，然后使用欧氏距离来计算两个人脸之间的距离，从而实现人脸识别。
