# 2.核心概念与联系

## 2.1 决策论

决策论是一门研究如何在有限信息和时间内选择最佳行动的学科。决策论可以分为两个子领域：

1. 确定性决策论：假设环境是确定的，决策者可以通过收集信息来预测未来的结果。
2. 不确定性决策论：假设环境是随机的，决策者无法预测未来的结果。

决策论的核心概念包括：

- 决策者：一个能够选择行动的实体，可以是人、机器或者组织。
- 决策空间：所有可能行动的集合。
- 目标函数：衡量决策者行动成功程度的函数。
- 约束条件：决策者行动的限制条件。

## 2.2 博弈论

博弈论是一门研究多个智能体在竞争和合作的过程中如何取得最优结果的学科。博弈论可以分为两个子领域：

1. 完全信息博弈：所有玩家都知道所有其他玩家的信息。
2. 不完全信息博弈：玩家之间的信息是分散的，每个玩家只知道自己的信息。

博弈论的核心概念包括：

- 玩家：一个能够选择策略的实体，可以是人、机器或者组织。
- 策略：玩家在游戏中采取的行动方案。
- Payoff：玩家获得的利益。
-  Nash equilibrium：一个在该策略下，每个玩家的策略都是他的最佳响应，这种情况下不会有玩家改变策略。

## 2.3 决策论与博弈论的联系

决策论和博弈论在某种程度上是相互补充的。决策论主要关注单个决策者在有限信息和时间内如何选择最佳行动，而博弈论关注多个决策者在竞争和合作的过程中如何取得最优结果。在实际应用中，决策论和博弈论可以结合使用，以更好地理解和解决复杂问题。

# 3.核心算法原理和具体操作步骤

## 3.1 确定性决策论

### 3.1.1 贝叶斯决策论

贝叶斯决策论是确定性决策论的一个子集，它假设环境是确定的，决策者可以通过收集信息来预测未来的结果。贝叶斯决策论的核心思想是使用贝叶斯定理来更新概率分布，从而选择最佳的行动。

具体操作步骤如下：

1. 定义决策空间和目标函数。
2. 收集信息并构建概率模型。
3. 使用贝叶斯定理更新概率分布。
4. 根据目标函数选择最佳行动。

### 3.1.2 动态规划

动态规划是一种解决确定性决策问题的方法，它通过递归地求解子问题来求解原问题。动态规划的核心思想是将问题分解为更小的子问题，并将解决子问题的结果存储在一个表格中，以便后续使用。

具体操作步骤如下：

1. 定义决策空间和目标函数。
2. 构建递归关系。
3. 使用动态规划表格存储子问题的解。
4. 递归地求解子问题并得到原问题的解。

# 4.数学模型的详细讲解

## 4.1 确定性决策论

### 4.1.1 贝叶斯决策论

贝叶斯决策论的数学模型可以表示为：

$$
P(a|e) = \max_a \int P(e|a,x)P(x)dx
$$

其中，$P(a|e)$ 表示在观测到结果 $e$ 的情况下，对于行动 $a$ 的概率；$P(e|a,x)$ 表示在采取行动 $a$ 并观测到变量 $x$ 的情况下，观测到结果 $e$ 的概率；$P(x)$ 表示变量 $x$ 的概率分布。

### 4.1.2 动态规划

动态规划的数学模型可以表示为：

$$
V(s) = \max_a [R(s,a) + \sum_{s'} P(s'|s,a)V(s')]
$$

其中，$V(s)$ 表示在状态 $s$ 下的最佳值；$R(s,a)$ 表示在状态 $s$ 并采取行动 $a$ 的奖励；$P(s'|s,a)$ 表示在状态 $s$ 并采取行动 $a$ 的情况下，转到状态 $s'$ 的概率。

## 4.2 不确定性决策论

### 4.2.1 贝叶斯网络

贝叶斯网络的数学模型可以表示为：

$$
P(x) = \prod_{i=1}^n P(x_i|pa(x_i))
$$

其中，$P(x)$ 表示变量 $x$ 的概率分布；$x_i$ 表示变量 $x$ 的父节点；$pa(x_i)$ 表示变量 $x_i$ 的父节点集合。

### 4.2.2 蒙特卡罗方法

蒙特卡罗方法的数学模型可以表示为：

$$
\hat{y} = \frac{1}{N} \sum_{i=1}^N y_i
$$

其中，$\hat{y}$ 表示估计值；$N$ 表示随机采样的次数；$y_i$ 表示第 $i$ 次随机采样得到的结果。

## 4.3 博弈论

### 4.3.1 纳什均衡

纳什均衡的数学模型可以表示为：

$$
\forall i \in \{1,...,n\}, \nexists a_i \in A_i, s.t. \forall j \in \{1,...,n\}, a_j \in A_j, 
\pi_j(a_{-i},a_i) < \pi_j(a_{-i},a_i')
$$

其中，$n$ 表示玩家的数量；$A_i$ 表示玩家 $i$ 的策略集合；$\pi_j(a_{-i},a_i)$ 表示玩家 $j$ 的利益，在其他玩家采取策略 $a_{-i}$ 的情况下，自己采取策略 $a_i$ 的情况下。

### 4.3.2 支持向量机

支持向量机的数学模型可以表示为：

$$
\min_{w,b} \frac{1}{2}w^Tw + C\sum_{i=1}^n \xi_i
$$

$$
s.t. \begin{cases} y_i(w \cdot x_i + b) \geq 1 - \xi_i, i=1,...,n \\ \xi_i \geq 0, i=1,...,n \end{cases}
$$

其中，$w$ 表示支持向量的权重向量；$b$ 表示支持向量的偏置项；$C$ 表示惩罚参数；$\xi_i$ 表示损失的惩罚项；$y_i$ 表示标签；$x_i$ 表示特征向量。

import numpy as np

# 定义决策空间和目标函数
def utility(a, e):
    return -10 if a != e else 0

# 贝叶斯定理
def bayes_theorem(e, x):
    return utility(x, e) * P(x|e)

# 求最佳行动
def best_action(a, e):
    return max(a, key=lambda x: bayes_theorem(e, x))

# 假设环境是确定的，决策者可以通过收集信息来预测未来的结果
def decision_making(a, e):
    return best_action(a, e)
