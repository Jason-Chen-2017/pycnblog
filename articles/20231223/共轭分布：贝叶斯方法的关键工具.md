# 2.核心概念与联系
共轭分布是一种概率分布，它描述了两个或多个随机变量之间的关系。共轭分布可以用来描述随机变量之间的条件依赖关系，也可以用来建立生成模型。共轭分布的核心概念包括：

1. 条件独立性
2. 条件概率密度函数
3. 共轭关系
4. 共轭分布的例子（如伯努利分布、泊松分布、多变量正态分布等）
