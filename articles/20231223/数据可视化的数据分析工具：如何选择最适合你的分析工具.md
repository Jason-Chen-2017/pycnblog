## 4.1 使用Python的Matplotlib库进行数据可视化

Matplotlib是一个流行的Python数据可视化库，可以用于创建各种类型的图表。以下是一个简单的条形图示例：

```python
import matplotlib.pyplot as plt

# 数据
categories = ['A', 'B', 'C', 'D', 'E']
values = [10, 20, 30, 40, 50]

# 创建条形图
plt.bar(categories, values)

# 设置标签和标题
plt.xlabel('Categories')
plt.ylabel('Values')
plt.title('Bar Chart Example')

# 显示图表
plt.show()
```

这个代码首先导入了Matplotlib库，然后定义了数据，包括类别和值。接着，使用`plt.bar()`函数创建了一个条形图。最后，设置了标签和标题，并使用`plt.show()`函数显示了图表。
