# 2.核心概念与联系

Dataiku 是一种数据管道工具，它提供了一种简单、可扩展和可维护的方法来构建、维护和优化数据管道。Dataiku 的核心概念包括：

- **数据源**：数据源是数据管道中的基本组件，它们提供了数据的原始来源。这可以是数据库、文件、API 等。
- **数据流**：数据流是数据管道中的另一个基本组件，它定义了数据如何从一个数据源传输到另一个数据源。
- **数据处理**：数据处理是数据管道中的一个关键步骤，它涉及到数据的转换、清洗、聚合等操作。
- **数据目标**：数据目标是数据管道的最终目的地，它定义了数据需要被存储、分析或预测的地方。

Dataiku 与其他数据管道工具的主要区别在于它的灵活性和可扩展性。Dataiku 允许用户轻松地创建、修改和删除数据源、数据流和数据处理步骤。这使得 Dataiku 可以应对数据管道的变化和扩展，从而提高数据管道的效率和可靠性。
