# 2. 核心概念与联系

在本节中，我们将介绍微生物的核心概念，以及它们与人类生活的联系。

## 2.1 微生物的种类

微生物可以分为以下几种：

1. 细菌：细菌是最常见的微生物之一，它们可以在许多环境中生存，如水、土壤、空气等。细菌可以分为好细菌和坏细菌，好细菌可以帮助人类生产食物、制造药物等，而坏细菌则可能导致疾病、污染等问题。
2. 菌丝：菌丝是细菌的另一种形式，它们通常存在于水中，可以参与食物加工、环境保护等过程。
3. 菌菌体：菌菌体是一种单细胞微生物，它们可以通过生物合成技术产生有价值的化学物质。
4. 病毒：病毒是最小的微生物之一，它们可以感染细菌、动物、植物等，导致疾病。

## 2.2 微生物与人类生活的联系

微生物与人类生活有着密切的联系，它们参与了许多生活过程，如：

1. 食物加工：微生物在食物加工中扮演着重要角色，如酵母在面包制作中产生二氧化碳，酸菌在酸奶制作中产生酸。
2. 药物制造：许多药物的制造过程中都涉及到微生物，如抗生素的制造需要利用细菌。
3. 环境保护：微生物可以帮助我们清洁污水、处理废弃物等，从而保护环境。
4. 生物合成：通过引导微生物的生物合成，我们可以产生有价值的化学物质，如燃料、药物等。
