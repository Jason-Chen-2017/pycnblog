# 4. 具体代码实例和解释

在这里，我们以一个简单的人力资源数据分析案例为例，介绍人力资源数据分析的具体代码实例和解释。

假设我们有一个员工基本信息表，包括员工ID、年龄、工作年限、职位、薪资等信息。我们希望通过对这些数据进行分析，找出员工薪资与工作年限的关系。

首先，我们需要对员工基本信息表进行清洗、整理和预处理。在Python中，我们可以使用pandas库来实现这一步骤。

```python
import pandas as pd

# 读取员工基本信息表
df = pd.read_csv('employee_info.csv')

# 对员工基本信息表进行清洗、整理和预处理
df = df.dropna()  # 删除缺失值
df = df[df['position'] != '']  # 删除空值
df['age'] = df['age'].astype(int)  # 将年龄转换为整数类型
df['experience'] = df['experience'].astype(int)  # 将工作年限转换为整数类型
df['salary'] = df['salary'].astype(float)  # 将薪资转换为浮点类型
```

接下来，我们可以使用scikit-learn库中的线性回归模型来分析员工薪资与工作年限的关系。

```python
from sklearn.linear_model import LinearRegression

# 创建线性回归模型
model = LinearRegression()

# 训练线性回归模型
model.fit(df[['experience']], df['salary'])

# 预测员工薪资
predicted_salary = model.predict(df[['experience']])

# 计算线性回归模型的R^2值
r2_value = model.score(df[['experience']], df['salary'])
```

最后，我们可以将预测结果与原始数据进行比较，评估模型的效果。

```python
import matplotlib.pyplot as plt

# 绘制员工薪资与工作年限的关系图
plt.scatter(df['experience'], df['salary'], label='原始数据')
plt.plot(df['experience'], predicted_salary, color='red', label='预测结果')
plt.xlabel('工作年限')
plt.ylabel('薪资')
plt.legend()
plt.show()
```
