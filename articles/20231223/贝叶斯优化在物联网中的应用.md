# 2.核心概念与联系

## 2.1 贝叶斯优化的基本概念

贝叶斯优化是一种基于贝叶斯定理的方法，它可以在面对不可知的函数或模型时，有效地找到最优解。贝叶斯优化的核心思想是通过构建一个概率模型来描述不可知函数的不确定性，然后根据这个模型来选择最有可能找到最优解的参数值。

贝叶斯优化的主要步骤包括：

1. 构建概率模型：首先，我们需要构建一个概率模型来描述不可知函数的不确定性。这个模型可以是任意的，只要能够描述函数的形状和变化规律。

2. 选择参数值：根据概率模型，我们可以得到一个预测分布，这个分布描述了不可知函数在某个参数值处的预期值和不确定性。我们需要根据这个分布选择一个参数值来进行实验。

3. 获取实验结果：我们对选定的参数值进行实验，获取到实验结果后，可以更新概率模型。

4. 更新概率模型：根据实验结果，我们可以更新概率模型，以便在下一次选择参数值时得到更准确的预测。

5. 重复上述步骤：直到找到一个满足要求的参数值，或者实验次数达到预设的阈值。
