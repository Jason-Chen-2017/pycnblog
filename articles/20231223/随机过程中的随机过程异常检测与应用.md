```python
import numpy as np
import pandas as pd
from sklearn.ensemble import IsolationForest
from sklearn.preprocessing import StandardScaler

# 生成随机过程数据
np.random.seed(42)
n_samples = 1000
n_features = 5
X = np.random.randn(n_samples, n_features)

# 添加异常值
X[np.random.randint(0, n_samples, size=5)] = np.random.uniform(low=-3, high=3, size=(n_features,))

# 数据预处理
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 异常检测
clf = IsolationForest(n_estimators=100, contamination=0.05)
y_pred = clf.fit_predict(X_scaled)

# 结果评估
print("异常值数量：", np.sum(y_pred == -1))
```

在这个例子中，我们首先生成了随机过程数据，然后添加了异常值。接着，我们对数据进行了预处理，并使用IsolationForest算法进行异常检测。最后，我们对结果进行了评估。
