```python
import time
import requests

class IoTDevice:
    def __init__(self, url):
        self.url = url

    def send_data(self, data):
        response = requests.post(self.url, data=data)
        if response.status_code == 200:
            print("Data sent successfully")
        else:
            print("Data sent failed")

if __name__ == "__main__":
    iot_device = IoTDevice("http://example.com/api/data")
    data = {"temperature": 25, "humidity": 50}
    iot_device.send_data(data)
```

在上面的代码实例中，我们首先定义了一个`IoTDevice`类，该类用于管理物联网设备。然后我们定义了一个`send_data`方法，用于将数据发送到服务器。接着我们创建了一个`IoTDevice`对象，并将数据发送到服务器。
