# 4. 具体代码实例和详细解释说明

在这里，我们以一个简单的例子来说明多线程并发模型的具体实现：

```c
#include <pthread.h>
#include <stdio.h>

void *my_thread_func(void *arg) {
    printf("Hello, World! From thread %lu\n", (unsigned long)pthread_self());
    return NULL;
}

int main() {
    pthread_t tid;
    int rc;

    rc = pthread_create(&tid, NULL, my_thread_func, NULL);
    if (rc) {
        printf("ERROR; return code from pthread_create() is %d\n", rc);
        exit(-1);
    }

    pthread_join(tid, NULL);

    return 0;
}
```

这个例子中，我们创建了一个线程，并在线程中打印出“Hello, World! From thread %lu”的信息，其中%lu是线程的ID。在主线程中，我们使用pthread_join函数来等待子线程的结束。
