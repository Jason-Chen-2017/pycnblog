                 

# 1.背景介绍

欧盟数据保护基本条例（General Data Protection Regulation，简称GDPR）是一项欧盟通过的法规，其目的是为了保护个人数据的安全和隐私。它在2018年5月25日生效，对欧盟成员国及与其相关的非欧盟国家的企业产生了重大影响。GDPR对个人数据处理的要求更加严格，强调了数据保护的原则和实践，包括数据最小化、数据清洗、数据安全等。
