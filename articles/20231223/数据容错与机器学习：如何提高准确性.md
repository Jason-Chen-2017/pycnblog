```python
import numpy as np

def check_error(data):
    error = 0
    for i in range(len(data)):
        if data[i] != np.sum(data[i]) / len(data[i]):
            error += 1
    return error / len(data)
```

#### 4.1.2 错误纠正示例

```python
def correct_error(data):
    for i in range(len(data)):
        if np.sum(data[i]) % len(data[i]) != 0:
            data[i] = np.array([np.mean(data[i]) for _ in range(len(data[i]))])
    return data
```
