## 3.1 数据清洗

数据清洗是数据科学过程中的关键环节，可以帮助我们提高模型的性能。本书中介绍了以下数据清洗方法：

- 缺失值处理：可以使用删除、填充（如均值、中位数等）、预测等方法处理缺失值。
- 数据类型转换：可以使用 pandas 库的 astype 方法将数据类型转换为其他类型，例如将对象类型转换为整型。
- 数据归一化：可以使用 Sklearn 库的 StandardScaler 进行数据归一化，将数据缩放到 [-1, 1] 或 [0, 1] 范围内。

## 3.2 数据可视化

数据可视化是帮助用户更直观地理解数据的一种方法。本书介绍了以下数据可视化方法：

- Matplotlib：是一个功能强大的数据可视化库，可以绘制各种类型的图表，例如直方图、条形图、散点图等。
- Seaborn：是一个基于 Matplotlib 的数据可视化库，提供了更高级的可视化组件，例如 Heatmap、Pairplot 等。
