# 2.核心概念与联系

## 2.1 弱相互作用

弱相互作用是一种在量子力学中描述的相互作用，主要表现在粒子的电磁互动中。它通常与电磁相互作用相结合，形成弱电磁相互作用。弱相互作用在许多核子衰变过程中发挥着重要作用，例如氢核的衰变。

## 2.2 光子分裂

光子分裂是指光子在介质中的辐射和吸收过程。在这个过程中，光子可以被吸收，转化为其他形式的能量，如热能或化能；也可以被辐射出来，转化为光能。光子分裂在许多物理现象中发挥着重要作用，如黑体辐射、光学和电磁共振等。

## 2.3 黑体辐射

黑体辐射是指一种在黑体表面发生的辐射现象。黑体是一个理想的热源，它可以在任何温度下发射所有频率的光。黑体辐射是一种重要的热传输机制，在天文学、气候学和物理学等领域具有重要的应用价值。
