## 1.1 马尔可夫链的基本概念

马尔可夫链是一种随机过程，其中的状态转移仅依赖于当前状态，而不依赖于过去状态。这种特性使得马尔可夫链成为一种强大的数据生成模型，可以用于文本生成、图像生成等应用。

### 1.1.1 马尔可夫链的定义

给定一个有限状态集合S = {s1, s2, ..., sn}，我们可以定义一个概率分布P = {p(s1), p(s2), ..., p(sn)}。在一个马尔可夫链中，每个状态的概率仅依赖于前一个状态，可以表示为：

p(s_t | s_{t-1}) = p(s_t, s_{t-1}) / p(s_{t-1})

其中，s_t 和 s_{t-1} 分别表示当前状态和前一个状态。

### 1.1.2 马尔可夫链的转移矩阵

我们可以使用一个转移矩阵A来表示马尔可夫链的状态转移。转移矩阵A是一个n x n的矩阵，其中A[i][j]表示从状态i转移到状态j的概率。转移矩阵A可以用来描述马尔可夫链的状态转移特性。

### 1.1.3 马尔可夫链的长度和状态

在一个马尔可夫链中，我们可以定义两个概念：长度和状态。长度是指随机过程的时长，状态是指随机过程在某一时刻的状态。通过调整长度和状态，我们可以生成不同的随机序列。
