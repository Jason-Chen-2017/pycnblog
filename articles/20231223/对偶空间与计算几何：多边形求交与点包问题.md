## 3.1 多边形求交

### 3.1.1 基本概念

给定两个多边形 $P$ 和 $Q$，求它们的交集。我们可以将多边形 $P$ 和 $Q$ 分为两部分：一部分是包含原点的，一部分是不包含原点的。我们可以通过计算多边形的交点来求解这个问题。

### 3.1.2 算法原理

我们可以通过以下步骤来解决多边形求交问题：

1. 计算多边形 $P$ 和 $Q$ 的交点。
2. 判断这些交点是否满足多边形的定义。
3. 如果满足多边形的定义，则返回交点；否则，返回空集。

### 3.1.3 具体操作步骤

1. 对于每个多边形 $P$ 的顶点 $p_i$，我们可以计算它与多边形 $Q$ 的交点 $q_{ij}$。
2. 对于每个多边形 $Q$ 的顶点 $q_j$，我们可以计算它与多边形 $P$ 的交点 $p_{ji}$。
3. 我们可以将这些交点存储在一个列表中。
4. 我们可以通过检查这些交点是否满足多边形的定义来判断它们是否属于多边形的交集。

### 3.1.4 数学模型公式

我们可以使用以下公式来计算多边形的交点：

$$
\begin{cases}
x = \frac{a_1x_1 + a_2x_2 + \cdots + a_nx_n}{a_1 + a_2 + \cdots + a_n} \\
y = \frac{b_1y_1 + b_2y_2 + \cdots + b_ny_n}{b_1 + b_2 + \cdots + b_n}
\end{cases}
$$

其中 $a_i$ 和 $b_i$ 是多边形的边长，$x_i$ 和 $y_i$ 是多边形的顶点。

## 4.1 多边形求交

我们可以使用以下代码来实现多边形求交问题：

```python
def intersection(p1, p2):
    intersections = []
    for i in range(len(p1)):
        for j in range(len(p2)):
            x1, y1 = p1[i]
            x2, y2 = p2[j]
            a = (y2 - y1) / (x2 - x1)
            b = y1 - a * x1
            x = (b - y1) / a
            y = x * a + y1
            intersections.append((x, y))
    return intersections
```

这个函数接受两个多边形作为输入，并返回它们的交点列表。我们首先遍历每个多边形的顶点，然后计算它们的交点。最后，我们将交点存储在一个列表中并返回。

## 4.2 点包问题

我们可以使用以下代码来实现点包问题：

```python
def contains(p, points):
    for point in points:
        x, y = point
        inside = False
        for i in range(len(p)):
            x1, y1 = p[i]
            x2, y2 = p[(i + 1) % len(p)]
            if x1 > x2:
                x1, x2 = x2, x1
                y1, y2 = y2, y1
            if x >= x1 and x <= x2 and y1 <= (y - x * (y2 - y1) / (x2 - x1)) <= y2:
                inside = not inside
            if x1 == x2:
                if y1 <= y <= y2:
                    inside = not inside
        if not inside:
            return False
    return True
```

这个函数接受一个多边形和一组点作为输入，并返回它们是否可以被多边形包含。我们首先遍历每个点，然后计算它与多边形的关系。最后，我们将关系存储在一个变量中并返回。
