### 4.1 AES加密和解密的Python实现

AES是一种对称加密算法，其加密和解密过程如下：

1. 生成一个128位的密钥。
2. 将明文数据分组，每组16个字节。
3. 对每个分组进行加密或解密。
4. 将加密或解密后的分组拼接成原始数据。

以下是AES加密和解密的Python实现：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# 生成一个128位的密钥
key = get_random_bytes(16)

# 加密明文
def encrypt(plaintext, key):
    cipher = AES.new(key, AES.MODE_ECB)
    ciphertext = cipher.encrypt(plaintext)
    return ciphertext

# 解密密文
def decrypt(ciphertext, key):
    cipher = AES.new(key, AES.MODE_ECB)
    plaintext = cipher.decrypt(ciphertext)
    return plaintext

# 测试加密和解密
plaintext = b'Hello, World!'
ciphertext = encrypt(plaintext, key)
print('加密后的密文:', ciphertext)

plaintext_decrypted = decrypt(ciphertext, key)
print('解密后的明文:', plaintext_decrypted)
```

### 4.2 RSA加密和解密的Python实现

RSA是一种异对称加密算法，其加密和解密过程如下：

1. 生成两个大素数p和q。
2. 计算n=p*q。
3. 计算φ(n)=(p-1)*(q-1)。
4. 选择一个大于1的整数e，使得gcd(e, φ(n))=1。
5. 计算d=e^(-1) mod φ(n)。
6. 使用n, e进行加密。
7. 使用n, d进行解密。

以下是RSA加密和解密的Python实现：

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# 生成RSA密钥对
key = RSA.generate(2048)

# 获取公钥和私钥
public_key = key.publickey().export_key()
private_key = key.export_key()

# 加密明文
def encrypt(plaintext, public_key):
    cipher = PKCS1_OAEP.new(public_key)
    ciphertext = cipher.encrypt(plaintext)
    return ciphertext

# 解密密文
def decrypt(ciphertext, private_key):
    cipher = PKCS1_OAEP.new(private_key)
    plaintext = cipher.decrypt(ciphertext)
    return plaintext

# 测试加密和解密
plaintext = b'Hello, World!'
ciphertext = encrypt(plaintext, public_key)
print('加密后的密文:', ciphertext)

plaintext_decrypted = decrypt(ciphertext, private_key)
print('解密后的明文:', plaintext_decrypted)
```

### 4.3 SHA消息摘要的Python实现

SHA是一种消息摘要算法，其主要功能是将输入的消息转换为固定长度的摘要。SHA算法有多种版本，例如SHA-1、SHA-256等。以下是SHA-256消息摘要的Python实现：

```python
import hashlib

# 计算SHA-256消息摘要
def sha256(message):
    hash_object = hashlib.sha256(message.encode('utf-8'))
    hash_hex = hash_object.hexdigest()
    return hash_hex

# 测试SHA-256消息摘要
message = 'Hello, World!'
hash_hex = sha256(message)
print('SHA-256消息摘要:', hash_hex)
```
