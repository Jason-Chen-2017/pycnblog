# 4.具体代码实例和详细解释说明
# 4.1 创建地理空间数据表
```sql
CREATE TABLE location (
  id INT PRIMARY KEY,
  name VARCHAR(255),
  latitude DECIMAL(9,6),
  longitude DECIMAL(9,6),
  radius DECIMAL(5,2)
);
```
上述代码实例创建了一个名为location的地理空间数据表，表中包含id、name、latitude、longitude和radius等字段。其中，latitude和longitude字段用于存储地理坐标，radius字段用于存储地理对象的半径。

# 4.2 插入地理空间数据
```sql
INSERT INTO location (id, name, latitude, longitude, radius)
VALUES (1, '北京', 39.9042, 116.4074, 6.0);
```
上述代码实例插入了一个地理空间数据记录到location表中，记录包括id、name、latitude、longitude和radius等字段。其中，latitude和longitude字段用于存储地理坐标，radius字段用于存储地理对象的半径。

# 4.3 创建地理空间索引
```sql
CREATE SPATIAL INDEX idx_location_geom ON location (geom);
```
上述代码实例创建了一个名为idx_location_geom的地理空间索引，索引包括location表中的geom字段。geom字段用于存储地理对象的空间信息。

# 4.4 执行地理空间数据查询
```sql
SELECT * FROM location
WHERE ST_DISTANCE(geom, ST_GEOMFROMTEXT('POINT(116.4074 39.9042)')) < 6.0;
```
上述代码实例执行了一个地理空间数据查询，查询条件为距离当前记录的地理对象的半径小于6.0。查询结果包括id、name、latitude、longitude和radius等字段。

# 4.5 执行地理空间数据分析
```sql
SELECT id, name, ST_AREA(geom) AS area FROM location
WHERE ST_INTERSECTS(geom, ST_GEOMFROMTEXT('POLYGON((0 0, 100 0, 100 100, 0 100, 0 0))')) = 1;
```
上述代码实例执行了一个地理空间数据分析，分析条件为当前记录的地理对象与给定的多边形相交。分析结果包括id、name和地理对象的面积area等字段。
