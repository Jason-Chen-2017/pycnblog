### 4.1.1 AES加密和解密

AES是一种对称密码学算法，它使用相同的密钥进行加密和解密。以下是AES加密和解密的具体代码实例和详细解释说明：

```python
from Crypto.Cipher import AES

# 加密
key = b'This is a key1234567890abcdef'
cipher = AES.new(key, AES.MODE_ECB)
plaintext = b'This is a secret message'
ciphertext = cipher.encrypt(plaintext)
print('加密后的数据:', ciphertext)

# 解密
decipher = AES.new(key, AES.MODE_ECB)
deciphertext = decipher.decrypt(ciphertext)
print('解密后的数据:', deciphertext)
```

### 4.1.2 RSA加密和解密

RSA是一种非对称密码学算法，它使用不同的密钥进行加密和解密。以下是RSA加密和解密的具体代码实例和详细解释说明：

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# 生成RSA密钥对
key = RSA.generate(2048)
private_key = key.export_key()
public_key = key.publickey().export_key()

# 加密
cipher = PKCS1_OAEP.new(public_key)
plaintext = b'This is a secret message'
ciphertext = cipher.encrypt(plaintext)
print('加密后的数据:', ciphertext)

# 解密
decipher = PKCS1_OAEP.new(private_key)
deciphertext = decipher.decrypt(ciphertext)
print('解密后的数据:', deciphertext)
```

## 4.2 认证技术

### 4.2.1 ECDSA签名和验证

ECDSA是一种椭圆曲线密码学认证技术，它使用不同的密钥进行签名和验证。以下是ECDSA签名和验证的具体代码实例和详细解释说明：

```python
from Crypto.PublicKey import ECC
from Crypto.Signature import DSS
from Crypto.Hash import SHA256

# 生成ECDSA密钥对
key = ECC.generate(curve='P-256')
private_key = key.export_key()
public_key = key.publickey().export_key()

# 签名
hash_message = SHA256.new(b'This is a message to sign')
signer = DSS.new(private_key, 'fips-186')
signature = signer.sign(hash_message)
print('签名:', signature)

# 验证
verifier = DSS.new(private_key, 'fips-186')
is_valid = verifier.verify(hash_message, signature)
print('验证结果:', is_valid)
```
