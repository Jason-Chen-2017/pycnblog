在图像分割中，常见的相似性度量算法包括：

1.欧氏距离：欧氏距离是一种用于衡量两个向量之间距离的方法。在图像分割中，我们可以使用欧氏距离来计算两个区域之间的相似性。欧氏距离的公式为：

$$
d(x,y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2 + \cdots + (x_n - y_n)^2}
$$

2.均值绝对差（MAD）：均值绝对差是一种用于衡量两个序列之间差值的方法。在图像分割中，我们可以使用均值绝对差来计算两个区域之间的相似性。均值绝对差的公式为：

$$
MAD = \frac{1}{n} \sum_{i=1}^{n} |x_i - y_i|
$$

3.皮尔逊相关系数（PCC）：皮尔逊相关系数是一种用于衡量两个序列之间相关性的方法。在图像分割中，我们可以使用皮尔逊相关系数来计算两个区域之间的相似性。皮尔逊相关系数的公式为：

$$
r = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2}\sqrt{\sum_{i=1}^{n}(y_i - \bar{y})^2}}
$$

4.结构相似性（SSIM）：结构相似性是一种用于衡量两个图像之间结构相似性的方法。在图像分割中，我们可以使用结构相似性来计算两个区域之间的相似性。结构相似性的公式为：

$$
SSIM(x,y) = \frac{(2\mu_x\mu_y + C_1) (2\sigma_{xy} + C_2)}{(\mu_x^2 + \mu_y^2 + C_1) (\sigma_x^2 + \sigma_y^2 + C_2)}
$$

其中，$\mu_x$ 和 $\mu_y$ 是区域 $x$ 和 $y$ 的均值，$\sigma_x$ 和 $\sigma_y$ 是区域 $x$ 和 $y$ 的标准差，$\sigma_{xy}$ 是区域 $x$ 和 $y$ 的相关性。

def euclidean_distance(x, y):
    return np.sqrt(np.sum((x - y) ** 2))

x = np.array([1, 2, 3])
y = np.array([4, 5, 6])

distance = euclidean_distance(x, y)
print("欧氏距离:", distance)
