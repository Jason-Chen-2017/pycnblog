# 2.核心概念与联系

API网关的实时监控与报警主要包括以下几个核心概念：

1. 监控：通过收集和分析API网关的各种性能指标，以了解其运行状况。
2. 报警：根据预定义的阈值和规则，自动通知相关人员或执行预定义操作。
3. 数据收集：通过各种方式（如API调用、日志收集、性能指标采集等）收集API网关的运行数据。
4. 数据分析：通过数据分析工具（如Kibana、Grafana等）对收集到的数据进行可视化和分析。
5. 报警策略：根据企业的需求和业务要求，设定报警策略，以确保系统的稳定运行。
