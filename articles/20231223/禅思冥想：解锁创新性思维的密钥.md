### 4.1 代码实例

假设我们需要解决一个问题：如何提高网站的访问量。通过禅思冥想的方法，我们可以尝试从不同的角度来看待这个问题。以下是一个具体的代码实例：

```python
import random

def generate_idea():
    ideas = ['SEO优化', '推广营销', '内容创作', '用户体验优化', '社交媒体营销']
    return random.choice(ideas)

def main():
    problem = '提高网站访问量'
    for _ in range(10):
        idea = generate_idea()
        print(f'第 {_ + 1} 次禅思：{problem} 的一个可能解决方案是：{idea}')

if __name__ == '__main__':
    main()
```
