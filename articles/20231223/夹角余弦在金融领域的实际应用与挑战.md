### 2.1 夹角余弦的定义

夹角余弦是一个数值，用于度量两个向量之间的角度和。它的定义公式为：

$$
\cos(\theta) = \frac{a \cdot b}{\|a\| \|b\|}
$$

其中，$a$ 和 $b$ 是两个向量，$\cdot$ 表示向量内积，$\|a\|$ 和 $\|b\|$ 表示向量 $a$ 和 $b$ 的长度。

### 2.2 夹角余弦与其他相关概念的联系

- 夹角余弦与向量内积的关系：夹角余弦可以看作向量内积的一个特例，它表示了两个向量之间的角度和。

3. 计算夹角余弦：$\cos(\theta) = \frac{a \cdot b}{\|a\| \|b\|}$。

### 4.1 使用Python计算夹角余弦

在Python中，我们可以使用NumPy库来计算夹角余弦。以下是一个简单的代码示例：

```python
import numpy as np

# 定义两个向量
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

# 计算夹角余弦
cos_theta = np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

print("夹角余弦:", cos_theta)
```
