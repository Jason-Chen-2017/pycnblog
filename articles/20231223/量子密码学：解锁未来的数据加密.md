                 

# 1.背景介绍

量子密码学是一种基于量子计算学的密码学技术，它利用量子计算的特性来解决传统密码学中的问题。量子密码学的研究起源于1984年，当时的科学家们发现量子计算可以解决一些传统密码学算法无法解决的问题。随着量子计算技术的发展，量子密码学也逐渐成为一种可行的密码学技术。

量子密码学的核心概念是量子密钥交换，它允许两个距离很远的量子通信端点安全地交换密钥。量子密钥交换的安全性来自于量子物理定律，特别是量子纠缠和无法复制的性质。这使得量子密码学在安全性方面超越了传统密码学。
