数学模型公式详细讲解如下：

1. 度中心性（Degree Centrality）：度中心性是指一个节点在社交网络中与其他节点相连的程度。度中心性可以用以下公式计算：

$$
Degree\ Centrality = \frac{link_{node}}{N-1}
$$

其中，$link_{node}$ 是节点与其他节点相连的链接数，$N$ 是社交网络中节点的总数。

2. 度相连性（Closeness Centrality）：度相连性是指一个节点在社交网络中与其他节点之间的平均距离。度相连性可以用以下公式计算：

$$
Closeness\ Centrality = \frac{N-1}{\sum_{j=1}^{N}d_{ij}}
$$

其中，$d_{ij}$ 是节点之间的距离，$N$ 是社交网络中节点的总数。

3. 桥接性（Betweenness Centrality）：桥接性是指一个节点在社交网络中控制其他节点之间的通信的程度。桥接性可以用以下公式计算：

$$
Betweenness\ Centrality = \sum_{s\neq i\neq t}\frac{n_{st}(i)}{n_{st}}
$$

其中，$n_{st}(i)$ 是从节点$s$ 到节点$t$ 的路径中经过节点$i$ 的数量，$n_{st}$ 是从节点$s$ 到节点$t$ 的路径数量。


2. 预处理数据：

我们可以使用Python的pandas库来预处理数据，如下所示：

```python
import pandas as pd

data = pd.DataFrame(data)

```

3. 构建社交网络图：

我们可以使用Python的networkx库来构建社交网络图，如下所示：

```python
import networkx as nx

G = nx.Graph()
for index, row in data.iterrows():
    G.add_node(row['id'], attributes=row)
```

4. 计算社交网络指标：

我们可以使用Python的networkx库来计算社交网络指标，如下所示：

```python
degree_centrality = nx.degree_centrality(G)

```

我们可以使用Python的matplotlib库来可视化结果，如下所示：

