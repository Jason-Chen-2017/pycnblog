# 2.核心概念与联系
电子竞技的核心概念主要包括以下几个方面：

1. 游戏规则：电子竞技的游戏规则是由游戏开发商设定的，不同的游戏可能有不同的规则。游戏规则包括游戏的目标、游戏的操作方式、游戏的胜负标准等方面。

2. 竞技规则：电子竞技的竞技规则是由电子竞技组织设定的，用于确保竞技的公平性和公正性。竞技规则包括比赛的规程、比赛的评委、比赛的奖项等方面。

3. 竞技队伍：电子竞技的竞技队伍是由一组游戏玩家组成的，这些玩家通常具有高度的技能和协同度。竞技队伍可以是专业队伍，也可以是非专业队伍。

4. 观众：电子竞技的观众是由一群喜欢观看电子竞技的人组成的，这些观众通常会在比赛期间提供支持和评价。

5. 直播平台：电子竞技的直播平台是由一些网络公司提供的，这些平台通常会在比赛期间提供直播服务，让观众可以在线观看比赛。

6. 市场化：电子竞技的市场化是指电子竞技的发展逐渐进入市场化的道路，通过各种市场活动和策略来提高品牌知名度和市场份额。

def game_model(guess, secret_number, max_rounds):

    while rounds < max_rounds:

        elif guess > secret_number:
            return "too_high"
        rounds += 1
    return "lose"
