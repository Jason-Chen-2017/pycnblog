# 2.核心概念与联系
# 2.1 并行计算
并行计算是指在多个处理单元同时执行多个任务，以提高计算效率。并行计算可以分为数据并行和任务并行两种类型。数据并行是指在多个处理器上同时处理同一个数据集的不同部分，而任务并行是指在多个处理器上同时执行多个不相关的任务。在医学图像处理中，并行计算可以提高处理速度，从而提高医疗诊断和治疗的效率。
