## 3.1BB84协议

BB84协议是量子密钥交换中最早和最常用的协议之一，它的核心思想是利用量子物理学中的量子叠加和量子纠缠来实现安全的密钥交换。

### 3.1.1原理

BB84协议的核心思想是，发送方（Alice）将随机选择一组量子比特序列作为密钥，然后将这些量子比特通过量子通信通道发送给接收方（Bob）。接收方（Bob）将收到的量子比特进行测量，并将测量结果通过经典通信通道发送回发送方（Alice）。发送方（Alice）根据接收方（Bob）发送回的测量结果，选择出一组共同的密钥。

### 3.1.2具体操作步骤

1. 发送方（Alice）将随机选择一组量子比特序列作为密钥，然后将这些量子比特通过量子通信通道发送给接收方（Bob）。
2. 接收方（Bob）将收到的量子比特进行测量，并将测量结果通过经典通信通道发送回发送方（Alice）。
3. 发送方（Alice）根据接收方（Bob）发送回的测量结果，选择出一组共同的密钥。

### 3.1.3数学模型公式

BB84协议的数学模型公式如下：

- **量子比特的状态**：量子比特的状态可以表示为$$ |0\rangle $$和$$ |1\rangle $$两种基态。
- **量子叠加**：量子比特可以处于量子叠加状态，如$$ |\psi\rangle = \alpha|0\rangle + \beta|1\rangle $$，其中$$ \alpha,\beta \in \mathbb{C} $$，$$ |\alpha|^2 + |\beta|^2 = 1 $$。
- **量子纠缠**：两个量子比特的纠缠状态可以表示为$$ |\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle) $$，$$ |\Phi^-\rangle = \frac{1}{\sqrt{2}}(|00\rangle - |11\rangle) $$。

```python
import random
import numpy as np
import qiskit
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram, plot_bloch_vector

# 创建量子电路
qc = QuantumCircuit(2, 2)

# 随机选择一组量子比特序列作为密钥
keys = random.sample(range(2), 4)

# 将密钥编码为量子比特
for i in range(4):
    qc.cx(i, keys[i])

# 将量子电路传输到量子计算机
qasm_simulator = Aer.get_backend('qasm_simulator')
qobj = qc.to_qobj(shots=1024)
result = qasm_simulator.run(qobj).result()

# 获取测量结果
counts = result.get_counts()
print(counts)
```

在这个代码实例中，我们首先创建了一个量子电路，然后随机选择了一组量子比特序列作为密钥。接着，我们将密钥编码为量子比特，并将量子电路传输到量子计算机上进行测量。最后，我们获取了测量结果并打印了结果。
