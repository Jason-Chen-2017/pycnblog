# 2.核心概念与联系

## 2.1 综合医疗服务
综合医疗服务是指为患者提供全面的医疗服务，包括诊断、治疗、康复、预防等各种医疗服务。这种服务模式可以让患者在一个平台上获得全面的医疗服务，提高医疗服务的质量和效率。

## 2.2 专业化与职业化
专业化是指医疗服务行业中的各种医疗服务进行分工和专门化，每个医疗服务的专业人员都有明确的职责和职能。职业化是指医疗服务行业中的各种医疗服务进行规范化和标准化，提高医疗服务的质量和安全性。
