## 3.1 欧几里得范数（L2范数）

欧几里得范数的定义为：
$$
\|x\|_2 = \sqrt{\sum_{i=1}^n x_i^2}
$$
其中 $x$ 是一个 $n$ 维向量，$x_i$ 是向量的第 $i$ 个元素。

在生物信息学中，欧几里得范数通常用于计算相似性和距离。例如，在基因序列的编辑距离计算中，欧几里得范数可以用来计算两个序列之间的差异。具体操作步骤如下：

1. 将两个基因序列转换为相同长度的序列，以便进行比较。
2. 计算两个序列之间的差异，即编辑距离。
3. 使用欧几里得范数计算编辑距离。

## 3.2 曼哈顿范数（L1范数）

曼哈顿范数的定义为：
$$
\|x\|_1 = \sum_{i=1}^n |x_i|
$$
其中 $x$ 是一个 $n$ 维向量，$x_i$ 是向量的第 $i$ 个元素。

在生物信息学中，曼哈顿范数通常用于计算基因表达谱的差异、基因功能分类等。具体操作步骤如下：

1. 将基因表达谱转换为数值向量。
2. 使用曼哈顿范数计算两个向量之间的差异。

# 4.具体代码实例和详细解释说明

在这里，我们提供了两个Python代码实例，分别演示了如何使用欧几里得范数和曼哈顿范数计算相似性和距离。

## 4.1 欧几里得范数示例

```python
import numpy as np

def euclidean_distance(x, y):
    return np.sqrt(np.sum((x - y) ** 2))

x = np.array([1, 2, 3])
y = np.array([4, 5, 6])

distance = euclidean_distance(x, y)
print(distance)
```

## 4.2 曼哈顿范数示例

```python
import numpy as np

def manhattan_distance(x, y):
    return np.sum(np.abs(x - y))

x = np.array([1, 2, 3])
y = np.array([4, 5, 6])

distance = manhattan_distance(x, y)
print(distance)
```
