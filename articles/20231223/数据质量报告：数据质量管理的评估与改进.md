# 4.具体代码实例和详细解释说明

以下是一个简单的Python代码实例，用于计算数据准确性指标：

```python
import numpy as np

# 预测值和实际值
predicted_values = np.array([1, 2, 3, 4, 5])
actual_values = np.array([1, 2, 3, 4, 5])

# 计算准确性指标
accuracy = np.mean(predicted_values == actual_values)
print("准确性指标:", accuracy)
```

在这个例子中，我们首先导入了numpy库，然后定义了预测值和实际值的数组。接着，我们使用了numpy的mean函数来计算准确性指标，即预测值与实际值的比例。最后，我们打印了准确性指标的结果。
