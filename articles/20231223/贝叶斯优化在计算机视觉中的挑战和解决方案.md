# 4.具体代码实例和详细解释说明
在这里，我们将通过一个具体的例子来展示贝叶斯优化在计算机视觉中的应用。我们将使用Python的Scikit-Optimize库来实现贝叶斯优化，并使用一个简单的线性函数来模拟计算机视觉任务。

```python
import numpy as np
from skopt import gp_minimize

# 定义目标函数
def objective_function(x):
    return np.sin(x[0]) + 2 * x[1]**2

# 使用贝叶斯优化来优化目标函数
result = gp_minimize(objective_function, [0, 0], n_calls=100, random_state=0)

# 输出最佳参数值和目标函数的最小值
print("最佳参数值:", result.x)
print("目标函数的最小值:", result.fun)
```

在这个例子中，我们首先定义了一个目标函数`objective_function`，这个函数模拟了一个简单的计算机视觉任务。接下来，我们使用Scikit-Optimize库的`gp_minimize`函数来实现贝叶斯优化。我们设定了100次调用，并设置了一个随机种子为0。最后，我们输出了最佳参数值和目标函数的最小值。
