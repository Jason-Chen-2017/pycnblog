## 4.1 代码实例

我们选择了一个基于K-均值算法的社群发现代码实例。以下是代码实例：

```python
import numpy as np
import sklearn
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs
from sklearn.metrics import silhouette_score

# 生成随机数据
X, _ = make_blobs(n_samples=300, centers=4, cluster_std=0.60, random_state=0)

# 使用K-均值算法进行聚类
kmeans = KMeans(n_clusters=4, random_state=0)
kmeans.fit(X)

# 评估聚类效果
score = silhouette_score(X, kmeans.labels_)
print('Silhouette Score: %.3f' % score)
```

## 4.2 详细解释说明

1. 首先，我们导入了必要的库，如numpy、sklearn等。

2. 然后，我们使用sklearn的make_blobs函数生成了随机数据，其中n_samples表示样本数量，centers表示聚类中心数量，cluster_std表示聚类标准差，random_state表示随机种子。

3. 接着，我们使用KMeans类的构造函数创建了一个K-均值算法对象，其中n_clusters表示聚类数量，random_state表示随机种子。

4. 使用fit方法对随机数据进行聚类，并获取聚类结果。

5. 最后，我们使用silhouette_score函数评估聚类效果，其中X是样本数据，labels_是聚类结果。
