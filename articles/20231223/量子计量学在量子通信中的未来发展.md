## 3.具体代码实例和详细解释说明

### 3.1 量子测量代码实例

以下是一个使用 Python 和 Qiskit 库实现的量子测量代码示例：

```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile, assemble
from qiskit.visualization import plot_histogram

# 创建一个量子电路
qc = QuantumCircuit(2, 2)

# 准备一个二级量子位的纯态
qc.initialize([1, 0], 0)
qc.initialize([0, 1], 1)

# 量子测量
qc.measure([0, 1], [0, 1])

# 运行量子电路
backend = Aer.get_backend('qasm_simulator')
aer_qobj = transpile(qc, backend).run(backend)

# 得到测量结果
counts = aer_qobj.result().get_counts()

# 绘制测量结果分布
plot_histogram(counts)
```
