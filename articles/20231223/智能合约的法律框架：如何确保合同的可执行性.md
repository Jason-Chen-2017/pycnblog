以下是一个简单的智能合约的具体代码实例：

```solidity
pragma solidity ^0.5.0;

contract SimpleContract {
    uint public num;

    function setNum(uint _num) public {
        num = _num;
    }

    function getNum() public view returns (uint) {
        return num;
    }
}
```

## 4.2 智能合约的详细解释说明

上述智能合约的详细解释说明如下：

1. `pragma solidity ^0.5.0;`：指定智能合约编程语言的版本为0.5.0。

2. `contract SimpleContract {`：定义一个名为`SimpleContract`的智能合约。

3. `uint public num;`：定义一个名为`num`的公共变量，类型为`uint`。

4. `function setNum(uint _num) public {`：定义一个名为`setNum`的公共函数，参数为`uint`类型的`_num`，没有返回值。

5. `num = _num;`：将`_num`的值赋给`num`变量。

6. `function getNum() public view returns (uint) {`：定义一个名为`getNum`的公共函数，没有参数，返回类型为`uint`，没有返回值。

7. `return num;`：返回`num`变量的值。

8. `}`：结束智能合约的定义。
