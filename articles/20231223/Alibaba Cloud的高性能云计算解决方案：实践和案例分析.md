## 4.1 分布式计算示例

以下是一个简单的分布式计算示例，用Python编写：

```python
from multiprocessing import Pool

def square(x):
    return x * x

if __name__ == '__main__':
    nums = [i for i in range(100)]
    with Pool(4) as pool:
        results = pool.map(square, nums)
    print(results)
```

在这个示例中，我们使用Python的multiprocessing库实现了一个简单的分布式计算任务。通过创建一个Pool对象，我们可以将计算任务分解为多个子任务，并在多个计算节点上并行执行。最终，结果会被集成为一个列表，并输出。

## 4.2 数据并行示例

以下是一个简单的数据并行示例，用Python编写：

```python
import numpy as np
from multiprocessing import Pool

def average(data):
    return np.mean(data)

if __name__ == '__main__':
    nums = [i for i in range(10000)]
    data = np.array_split(nums, 4)
    with Pool(4) as pool:
        results = pool.map(average, data)
    print(results)
```

在这个示例中，我们使用Python的numpy库实现了一个简单的数据并行任务。通过将大数据集分解为多个子数据集，我们可以在多个计算节点上并行处理。最终，结果会被集成为一个列表，并输出。
