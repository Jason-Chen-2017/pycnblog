```python
import numpy as np

def soil_quality_score(organic_matter, particle_size, water_retention, adhesiveness, stability):
    score = 0
    score += organic_matter * 0.3
    score += particle_size * 0.2
    score += water_retention * 0.25
    score += adhesiveness * 0.15
    score += stability * 0.1
    return score

organic_matter = 5
particle_size = 7
water_retention = 8
adhesiveness = 6
stability = 9

soil_quality = soil_quality_score(organic_matter, particle_size, water_retention, adhesiveness, stability)
print("土壤质量评分：", soil_quality)
```

在这个代码实例中，我们定义了一个土壤质量评分的函数`soil_quality_score`，该函数接受五个参数：有机物质量、粒子大小、水分裂解度、粘性、稳定性。这五个参数分别对应于土壤质量的五个主要指标。在函数中，我们将这五个参数乘以各自的权重相加，得到土壤质量评分。

然后，我们设定了五个参数的值，分别表示土壤有机物质量为5分，粒子大小为7分，水分裂解度为8分，粘性为6分，稳定性为9分。最后，我们调用`soil_quality_score`函数计算土壤质量评分，并打印结果。
