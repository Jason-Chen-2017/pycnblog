### 4.1 密码学实例

以下是一个简单的AES加密和解密的代码实例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# 生成密钥
key = get_random_bytes(16)

# 生成加密对象
cipher = AES.new(key, AES.MODE_ECB)

# 加密数据
data = b"Hello, World!"
encrypted_data = cipher.encrypt(data)

# 解密数据
decrypted_data = cipher.decrypt(encrypted_data)

print(decrypted_data)
```

在这个代码实例中，我们使用了AES加密算法来加密和解密数据。AES是一种对称密钥加密算法，它使用相同的密钥进行加密和解密。我们首先生成了一个16字节的密钥，然后使用这个密钥来创建一个加密对象。最后，我们使用这个加密对象来加密和解密数据。

### 4.2 哈希函数实例

以下是一个简单的SHA-256哈希函数的代码实例：

```python
import hashlib

# 生成哈希值
data = b"Hello, World!"

```

在这个代码实例中，我们使用了SHA-256哈希函数来计算数据的哈希值。SHA-256是一种摘要算法，它可以将输入数据转换为一个固定长度的哈希值。我们首先生成了一个字符串，然后使用这个字符串来计算哈希值。

### 4.3 数字签名实例

以下是一个简单的RSA数字签名的代码实例：

```python
from Crypto.PublicKey import RSA
from Crypto.Signature import PKCS1_v1_5
from Crypto.Hash import SHA256

# 生成密钥对
key = RSA.generate(2048)
private_key = key

# 生成数据

# 生成数字签名
signer = PKCS1_v1_5.new(private_key)
signature = signer.sign(data)

# 验证数字签名

    print("验证失败")
```

在这个代码实例中，我们使用了RSA数字签名算法来生成和验证数字签名。RSA是一种非对称密钥加密算法，它使用不同的密钥进行加密和解密。我们首先生成了一个2048位的RSA密钥对，然后使用私钥来生成数字签名。最后，我们使用公钥来验证数字签名。
