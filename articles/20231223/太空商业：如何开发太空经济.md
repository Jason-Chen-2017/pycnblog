我们使用一个简单的数据集来表示太空旅游市场的需求和价格：

```python
data = {
    'Price': [200000, 150000, 100000, 50000],
    'Demand': [100, 150, 200, 250]
}

```

### 1.4.2 模型训练

我们使用线性回归模型来预测太空旅游市场需求：

```python

X = df['Price'].values.reshape(-1, 1)
y = df['Demand'].values.reshape(-1, 1)

model = LinearRegression()
model.fit(X, y)
```

### 1.4.3 模型预测
