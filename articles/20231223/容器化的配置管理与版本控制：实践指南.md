# 4.具体代码实例和详细解释说明
# 4.1 创建容器化配置管理系统
在创建容器化配置管理系统时，可以使用 Docker 和 Kubernetes 等容器化技术。以下是一个简单的 Docker 和 Kubernetes 代码实例：

```
# Dockerfile
FROM ubuntu:18.04
RUN apt-get update && apt-get install -y git
COPY config.yaml /etc/config.yaml
CMD ["/bin/bash"]

# config.yaml
version: '3'
services:
  config-manager:
    image: config-manager:latest
    ports:
      - "8080:8080"
```

```
# Kubernetes deployment.yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: config-manager
spec:
  replicas: 1
  selector:
    matchLabels:
      app: config-manager
  template:
    metadata:
      labels:
        app: config-manager
    spec:
      containers:
      - name: config-manager
        image: config-manager:latest
        ports:
        - containerPort: 8080
```

# 4.2 将应用程序及其依赖项打包在容器中
在将应用程序及其依赖项打包在容器中时，可以使用 Docker 的多阶段构建功能。以下是一个简单的 Docker 代码实例：

```
# Dockerfile
FROM ubuntu:18.04 as build
RUN apt-get update && apt-get install -y git
COPY . /app
WORKDIR /app
RUN npm install

FROM node:18.04 as final
COPY --from=build /app /app
CMD ["npm", "start"]
```
