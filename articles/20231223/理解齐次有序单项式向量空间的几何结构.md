## 1.1 向量空间的基本概念

向量空间是一个线性结构，它包含了一个基础向量集合和一个内积或点积。向量空间的基本操作包括向量的加法、减法和内积或点积。向量空间可以用来表示各种类型的数据，如图像、音频、文本等。

### 1.1.1 向量的加法和减法

向量空间中的向量可以通过加法和减法进行操作。加法是将两个向量相加，得到一个新的向量。减法是将一个向量从另一个向量中减去，得到一个新的向量。这些操作遵循以下规则：

1. 向量加法是对称的：a + b = b + a
2. 向量加法是结合的：a + (b + c) = (a + b) + c
3. 每个向量都有一个对应的负向量，满足a + (-a) = 0

### 1.1.2 内积或点积

内积或点积是向量空间中的一个二元操作，它用于计算两个向量之间的度量。内积或点积的结果是一个数字，表示两个向量之间的相似度。内积或点积遵循以下规则：

1. 对于任何两个向量a和b，内积或点积a·b = b·a
2. 对于任何向量a和b，如果a·b = 0，则a和b是垂直的
3. 对于任何向量a，a·a >= 0，且如果a·a = 0，则a为零向量
