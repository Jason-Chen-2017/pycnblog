# 2.核心概念与联系
概率分布是一种用于描述随机事件发生的可能性的数学模型。在图像处理中，概率分布可以用来描述图像中像素值的分布情况，从而帮助我们更好地理解图像中的信息。

概率分布可以分为两种主要类型：连续概率分布和离散概率分布。连续概率分布用于描述连续随机变量的分布情况，如图像中像素值的分布。离散概率分布用于描述离散随机变量的分布情况，如图像中的边缘检测结果。

在图像处理中，概率分布可以用于实现多种不同的功能，如：

- 图像去噪：通过分析图像中像素值的概率分布，可以更好地识别噪声和真实信号，从而实现更好的去噪效果。
- 图像增强：通过分析图像中像素值的概率分布，可以更好地调整图像的亮度、对比度和饱和度，从而实现更好的增强效果。
- 图像压缩：通过分析图像中像素值的概率分布，可以更好地压缩图像的信息，从而实现更好的压缩效果。
- 图像分割：通过分析图像中像素值的概率分布，可以更好地将图像分割为不同的区域，从而实现更好的分割效果。

## 4.1 图像去噪
```python
import numpy as np
import cv2
import matplotlib.pyplot as plt

# 读取图像

# 添加噪声
img_noisy = img + np.random.normal(0, 20, img.shape)

# 计算概率分布函数
hist, bins = np.histogram(img_noisy, bins=256, range=(0, 256))

# 计算概率密度函数
pdf = hist / np.sum(hist)

# 去噪
for i in range(256):
    if pdf[i] > 0.01:
        img_denoised[np.where((img_noisy == i) & (pdf[i] > 0.01))] = i

# 显示结果
plt.subplot(121)
plt.imshow(img_noisy, cmap='gray')
plt.title('Noisy Image')
plt.subplot(122)
plt.imshow(img_denoised, cmap='gray')
plt.title('Denoised Image')
plt.show()
```
在上述代码中，我们首先读取了一个图像，并添加了一些噪声。接着，我们计算了图像中像素值的概率分布函数，并计算了概率密度函数。最后，我们使用概率密度函数来识别噪声和真实信号，并实现了图像去噪。

# 6.附录常见问题与解答
在这一节中，我们将回答一些常见问题。

### 问题1：概率分布和概率密度函数有什么区别？
答案：概率分布函数和概率密度函数都用于描述随机变量的分布情况，但它们之间有一些区别。概率分布函数用于描述离散随机变量的分布情况，而概率密度函数用于描述连续随机变量的分布情况。

### 问题2：如何计算概率分布函数？
答案：概率分布函数可以通过计算随机变量取值的概率来得到。例如，如果一个随机变量可以取值为1、2、3、4和5，分别对应的概率为0.2、0.3、0.1、0.4和0.1，那么概率分布函数可以表示为：

$$
F(x) = \begin{cases}
0.2, & \text{if } x=1 \\
0.2+0.3, & \text{if } x=2 \\
0.2+0.3+0.1, & \text{if } x=3 \\
0.2+0.3+0.1+0.4, & \text{if } x=4 \\
0.2+0.3+0.1+0.4+0.1, & \text{if } x=5 \\
0, & \text{otherwise}
\end{cases}
$$

### 问题3：如何计算概率密度函数？
答案：概率密度函数可以通过计算随机变量在某个区间内的概率密度来得到。例如，如果一个随机变量在区间[0,1]内取值，并且其概率密度函数为f(x)，那么概率密度函数可以表示为：

$$
f(x) = \frac{dF(x)}{dx}
$$

其中，F(x)是概率分布函数。
