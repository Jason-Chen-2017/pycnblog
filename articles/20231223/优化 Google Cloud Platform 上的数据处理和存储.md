## 2.Huffman 算法数据压缩

Huffman 算法是一种常用的数据压缩算法，可以帮助我们减少数据的存储空间。我们可以使用 Huffman 算法对数据进行压缩，以降低存储成本。以下是一个 Huffman 算法数据压缩的代码实例：

```python
import heapq

# 创建一个频率字典
frequency = {'a': 5, 'b': 9, 'c': 12, 'd': 13, 'e': 16, 'f': 45}

# 创建一个优先级队列
priority_queue = [[weight, [symbol, frequency]] for symbol, frequency in frequency.items()]
heapq.heapify(priority_queue)

# 创建一个Huffman树
huffman_tree = {}

# 创建一个Huffman编码字典
huffman_code = {}

# 遍历优先级队列
while len(priority_queue) > 1:
    lo = heapq.heappop(priority_queue)
    hi = heapq.heappop(priority_queue)
    for pair in lo[1:]:
        pair[1] += hi[1]
        heapq.heappush(priority_queue, [lo[0] + hi[0], pair])
    huffman_tree[lo[1]] = ''
    huffman_code[lo[1]] = lo[1]
    for pair in hi[1:]:
        huffman_tree[pair[0]] = huffman_code[pair[0]] + '1'
        huffman_code[pair[0]] = huffman_code[pair[0]] + '0'

# 打印Huffman编码
print(huffman_code)
```
