```python
base_material = np.array([1, 2, 3])

# 在基材表面进行镀膜
coated_material = np.hstack((base_material, metal_particle))

# 实现金属粒子在基材表面的紧密接触
contact_area = np.sum(coated_material**2)

# 验证镀膜效果
strength_enhancement = np.mean(coated_material) - np.mean(base_material)
corrosion_resistance = 1 / (1 + np.mean(coated_material) * np.mean(base_material))

print("镀膜后的材料强度：", strength_enhancement)
print("防腐污染率：", corrosion_resistance)
```

2. 接着，我们在基材表面进行镀膜，并将镀膜后的材料存储在`coated_material`变量中。
3. 通过计算`contact_area`变量，我们实现了金属粒子在基材表面的紧密接触。
