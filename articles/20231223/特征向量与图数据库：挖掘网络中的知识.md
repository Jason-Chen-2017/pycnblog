# 2.核心概念与联系
## 2.1 特征向量
特征向量是一种用于表示数据的向量，其中每个元素都是数据的一个特征。特征向量可以用来表示数据的多种特征，如颜色、大小、形状等。特征向量可以用于数据的分类、聚类、分析等多种应用。

## 2.2 图数据库
图数据库是一种用于存储和管理网络数据的数据库。图数据库使用图结构来表示数据，其中节点表示数据实体，边表示数据实体之间的关系。图数据库可以用于存储和管理社交网络、知识图谱、地理空间数据等多种类型的网络数据。

## 2.3 特征向量与图数据库的联系
特征向量和图数据库在处理网络数据时具有很大的联系。特征向量可以用来表示图数据库中的节点和边的特征，从而可以用于图数据库的分类、聚类、分析等应用。同时，图数据库也可以用来存储和管理特征向量数据，从而可以用于特征向量的存储、管理和查询等应用。
