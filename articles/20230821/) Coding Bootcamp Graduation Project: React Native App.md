# 2.背景介绍
React Native 是 Facebook 在 2015 年发布的一款用于开发基于 iOS 和 Android 的移动应用的跨平台开发框架。它由 Facebook 和开源社区共同创造，其最大的特点就是能够通过 JavaScript/TypeScript 语言来开发移动应用，而不需要分别为每一种平台编写代码。

React Native 最大的优点就是使用 JavaScript 来开发移动应用，这让前端工程师和后端工程师可以同时参与到项目的开发当中，从而提升效率。Facebook 在 React Native 上也提供了非常丰富的 API 和组件供开发者使用，使得开发者可以快速地开发出具有用户体验的产品。在国内，相比于其他技术框架，React Native 有着良好的口碑，正在逐渐成为 IT 公司重点关注的方向之一。

本文将以一个电商APP为例，讲述如何使用React Native技术开发一个真实的APP，涉及到的技术栈有：
- Node.js：用于构建服务器端环境；
- Express.js：Node.js上流行的Web框架；
- MongoDB：非关系型数据库，用于存储产品、订单、用户信息；
- React Native：移动端跨平台开发框架；
- Expo SDK：React Native官方推出的开发工具包；
- Jest & Enzyme：用于单元测试；
- Typescript：超级强大的类型检查机制；
# 3.基本概念术语说明
## 3.1. React Native概览
React Native 是 Facebook 发布的跨平台移动开发框架，是一个使用 JavaScript 来开发移动应用的框架。它诞生于2015年，目前已经成为一个全球知名的框架。它的主要优势如下：
- 简单易用：React Native 提供了一整套简洁的 API 和组件，开发人员无需学习特定平台的开发语言，即可快速开发出具有原生应用质感的应用；
- 性能卓越：React Native 采用原生组件渲染界面，因此具备媲美原生应用的运行速度；
- 灵活适应：React Native 支持多种设备，包括iOS、Android、Windows Phone等；

在本教程中，我们将以iPhone XS Max 为例，阐述其基本特点。
