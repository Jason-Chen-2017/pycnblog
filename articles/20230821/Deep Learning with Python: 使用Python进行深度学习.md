# 2.基本概念术语说明
## 深度学习（deep learning）
深度学习，即多层神经网络（Multilayer Neural Network）学习。它是人工神经网络（Artificial Neural Networks，ANN）的一种扩展，是指由多层神经元组成的具有“深度”结构的学习系统。深度学习用于对大型或高维度的数据进行分类、回归、聚类、模式识别和异常检测等各种应用场景。

典型的深度学习系统由输入层、隐藏层（也可以称为神经层）、输出层组成。输入层接收外部输入的数据，一般包括图像、文本、音频、视频等。隐藏层则包括多个神经元，每个神经元之间存在激活函数和权重，用来计算输入数据的特征，并将其转换为输出结果。输出层则对隐藏层的输出进行处理，计算出最终的预测值或者类别。

深度学习通过组合简单的神经元单元，模拟人的大脑神经网络，能够学习非线性函数、高阶函数、连续空间中的数据分布。深度学习也需要大量的训练数据才能有效地进行训练，因此深度学习通常采用迭代方式进行训练。

## 神经元（neuron）
一个神经元是一个具有单个或多个受体的小电路，接受来自其它神经元、其他感官器官或输入设备的信息，然后将这些信息加总，通过某种计算过程得到输出信号，在一定条件下激活它，输出信息给其他神经元或传导给组织其他区域。

典型的神经元由三部分组成：接受神经递质的轴突、发送神经递质的皮质、大约有一个扇形突触来连接两者。轴突负责输入信号的处理和传输；皮质则负责处理输入信号；突触则负责将输出信号传递给其他神�元。

## 激活函数（activation function）
在计算神经网络时，每当神经元接收到输入信号后，都会经过一定的计算过程，得到输出信号。而激活函数就是用来控制输出信号的大小和饱和度的函数。不同的激活函数对神经网络的学习能力和泛化性能有着不同的影响。目前常用的激活函数包括Sigmoid、tanh、ReLU、Leaky ReLU等。

## 优化算法（optimization algorithm）
为了使得神经网络的训练过程更快、更准确、更稳定，需要采用一定的优化算法。目前常用的优化算法有随机梯度下降法（Stochastic Gradient Descent，SGD），改进的随机梯度下降法（Adam，Adaptive Moment Estimation），动量法（Momentum）等。

## 模型（model）
在深度学习中，模型是指神经网络结构和参数集合，用于对输入数据进行预测、分类、回归、聚类、降维等目的。深度学习模型通常由三部分组成：输入层、输出层、中间层（也叫隐藏层）。输入层代表网络的输入，包括图片、文本、音频等；输出层代表网络的输出，通常是一个标量值或一个向量值；中间层则根据输入数据进行计算，并将结果送往输出层进行处理。

## 数据（data）
深度学习模型所需的数据由两部分组成：样本（sample）和标签（label）。样本即模型所使用的输入数据，标签则对应于样本的真实值。深度学习模型训练的目的是找到合适的参数配置，使得模型对于输入数据产生的标签预测尽可能准确。训练数据集包括训练样本和对应的标签；验证数据集则用于评估模型在训练数据上的表现，以决定是否停止训练；测试数据集则用于最终评估模型在实际应用中的表现。

## 损失函数（loss function）
在深度学习中，损失函数定义了目标函数，即模型训练过程中衡量模型输出和实际值的差距。深度学习常用的损失函数有平方误差损失函数（Squared Error Loss Function）、交叉熵损失函数（Cross Entropy Loss Function）等。

## 正则化（regularization）
正则化是一种在模型训练过程中对模型参数进行调整的方式，目的是防止模型出现过拟合（overfitting）现象。常用的正则化方法有L1/L2正则化（Lasso/Ridge Regression），最大绝对值抑制（Maximal Absolute Value Suppression，MAVS）等。

## 过拟合（overfitting）
在深度学习中，过拟合是指模型在训练阶段出现过度依赖训练数据，导致模型在测试阶段表现较差的问题。过拟合问题主要有两个原因：
- 训练数据量不足：如果训练数据量不够丰富，模型就容易出现过拟合现象。
- 正则化不足：如果没有充分地使用正则化方法（比如L1/L2正则化、Dropout正则化），模型也容易出现过拟合现象。

解决过拟合的方法有减少模型复杂度（增加隐藏层数量、减少参数数量）、提升模型的拟合能力（加大训练数据量）、采用正则化方法（L1/L2正则化、Dropout正则化等）等。
