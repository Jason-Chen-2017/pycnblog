# 1.简介
  

## 一、游戏简介
超级马里奥(Super Mario)是任天堂(Nintendo)于1985年推出的经典平台类游戏系列，共分成两部。第一部是马里奥争霸3，第二部则是马里奥兄弟。两者均由任天堂担当设计、开发，在日本和美国同步发行。
## 二、游戏背景
超级马里奥(SM)是一款多人、动作冒险类的游戏，通过一个超级英雄试图征服整个马里奥娱乐城市、打败邪恶的怪兽、收集金币与各种道具以获得胜利。游戏中的角色通常都是2D的立绘，具有独特的造型及穿着方式。游戏的地图结构相较之前的游戏作了很大的改进，增加了更多障碍、陷阱、道路以及开放世界的元素。
在游戏中，玩家扮演的是红宝石马里奥——一名普通的男孩，他不仅要克服强大的敌人，还要找回失去的精神力量，成功掌控整个城市、召唤出魔兽并消灭它们。与此同时，也要避免掉入怪物与草丛，必须注意适时保护好自己的财产和心脏。最终，红宝石马里奥成功拯救了整个马里奥娱乐城市，并开启了漫长的奥林匹斯星辰之旅……
## 三、游戏规则
### 3.1 游戏准备
游戏开始前，玩家需要准备一些必要物品才能开始游戏，包括：
- **硬币**：游戏中使用的主要货币，可以用来购买道具、收集金币或支付其它费用。
- **贝壳**：用来充值，购买道具等。
- **金块**：可用于升级技能点、购买武器装备、提升属性等。
- **药水瓶**：用来调制药剂。
### 3.2 游戏流程
游戏流程分为6个阶段：

1. **主城区**：玩家可以进行普通攻击、存活、获取物品、结识新的朋友、接受任务或秘密通关。玩家可以在主城区收集金币、奖励、复活等。
2. **迷宫区域**：玩家可以进行冲刺、跑步、爬楼梯、翻越悬崖或楼梯、抓住水晶，通过危险地形关卡进入其它区域。
3. **城堡区域**：玩家可以进行走钢镚、破坏房屋、远离怪兽，找到宝藏或解决秘密关卡。
4. **海滩区域**：玩家可以进行划船、采集鱼类、寻宝，或者进入危险海域。
5. **森林区域**：玩家可以进行四处游历、伐木、钓鱼，去野外远足、做料理或烧烤。
6. **战斗区域**：玩家可以选择职业并战斗，或者进行团队竞技，击败强大的敌人并领取胜利。
### 3.3 游戏操作
#### 3.3.1 移动
- **左键/右键+上/下/左/右箭头**：上下左右移动。
- **WASD快捷键**：上下左右移动，分别按W、A、S、D键。
- **X/Y方向键**：后退/前进，分别按X键、Y键。
#### 3.3.2 操作对象切换
- **Tab键**：切换操作对象，包括玩家、怪物、物品、界面等。
#### 3.3.3 摇杆操作
- **摇杆左右滑动**：移动视角。
- **鼠标左键单击**：射击、使用、点击。
#### 3.3.4 其他功能
- **Shift键**：打开菜单栏。
- **Ctrl键**：打开状态栏。
- **E键**：查看状态。
- **F12键**：截屏。
- **P键**：暂停游戏。
