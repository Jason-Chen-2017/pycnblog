# 1.简介
  


“非我的存在”这个概念很古老了，出现在哲学家们的著作里。我记得最早提出这种观点的人莱昂哈德·图灵（1902-1990）说过：“有一种东西，它与我无关，却能影响我的行为。”

我是一名程序员，对于这种哲学上的思想感兴趣。所以，我觉得写一篇专业的技术博客文章可以帮助广大技术人员加深对“非我的存在”这个概念的理解。并且通过一些真实案例，让读者在阅读的时候能更好地理解其中的意涵。

在开始之前，让我先给大家简单介绍一下“非我的存在”的概念。


# “非我的存在”的概念

“非我的存在”指的是存在于外部世界而与自己的潜意识、认知活动等没有关系的东西。“非我的存在”可以是物体或事物（比如月亮），也可以是抽象的概念（比如时间）。换句话说，所谓“非我的存在”，就是无法被自己的思维活动、感受或意志所支配的存在。


# 为什么要提到“非我的存在”？

“非我的存在”这个概念和现代社会的很多现象息息相关，其中最突出的是人工智能。我们生活中充满着许多有关机器学习、神经网络、图像识别等领域的应用。这些技术的背后，都离不开“非我的存在”。

计算机从本质上来说是一个信息处理器，它需要获取输入的数据并输出结果。如何让计算机能够处理这样庞大的复杂的数据，就需要我们借助于“非我的存在”。比如，一个图片、视频、文字，这些都是由“非我的存在”（如我们的眼睛、耳朵、鼻子）来创造的。

人的大脑也是一个“非我的存在”，它在处理图像、语音、语言时扮演着至关重要的角色。但是，大脑不是唯一的、无限的“非我的存在”。就像孔子所说：“吾日三省吾身。”对人的思维活动、感受、意志等进行干预，可能会影响自身的生存。因此，研究“非我的存在”对科技界、人工智能界、社会各个领域都有重要的意义。


# 为什么“非我的存在”会带来巨大的影响？

“非我的存在”这个观念产生的原因有很多。下面我列举一些比较重要的原因。

首先，“非我的存在”与个人的信仰、价值观和意志无关。比如，人类一直在寻找外星人来改善自己的生存环境，但外星人终究还是会与人类发生冲突。实际上，“非我的存在”反映了一个大趋势，即人类越来越依赖其他生命形式，而不是依靠自己。

其次，“非我的存在”使得人类的活动变得复杂和高级。当遇到“非我的存在”时，人类的活动就会更加有针对性，比如人工智能、机器学习、量子计算、人机交互等。

再次，“非我的存在”推动了社会的进步。20世纪末，苏联的核试验曾经破坏了人类赖以生存的物质基础。“非我的存在”改变了这一局面，因为我们可以在外部看到希望，因此，就可以采取行动改变局面。

最后，“非我的存在”会让生活变得更美好。就拿最近的全球气候变化来说，我们每天都能看到海洋、森林、草原上的微小变化。这些微小的变化，都会影响“非我的存在”，进而引起社会的变化。

综上所述，“非我的存在”将成为我们未来的一大课题。随着技术的发展、人的发明，“非我的存在”也逐渐成为影响我们的大趋势之一。如果我们能够以科学的态度来认识它，正是基于对它的尊重才能做到全人类的共同利益。
