# 1.简介
  


此书是由美国基金经理人、量化交易专家马克·皮尔森和道格拉斯·弗里德曼联合撰写的一本著作。该书主要研究的是市场价值投资（MVO）的理论基础、方法和实践。作者还深入探讨了新经济、区域经济等市场环境对市值管理的影响。

与此同时，本书受到越来越多研究人员的关注。一些著名投资公司如摩根士丹利、J.P.摩根、埃隆巴斯克企业管理集团等都参考本书的理论研究成果。研究表明，市值管理不仅可以让投资者更好地利用市场资源，更能够更有效率地实现目标，而且还可避免风险带来的损失。

在这本书中，皮尔森详细阐述了市值管理的定义、特点和原则。他首先将市值的度量标准分为静态市值、动态市值、价值市值等三类，并进一步讨论不同市值之间的相互关系及其管理意义。之后，他列举了市值管理的八种关键环节，并详细描述了它们如何协同作用。这些环节包括预测、跟踪、衡量、优化、控制、风险管理和评估等。皮尔森还详细论述了市值管理过程中可能出现的问题，并提出了相应的解决方案。

在皮尔森看来，市值管理可广泛应用于股票、债券、商品期货等市场，可有效促进资产配置和风险管理。市值管理也可用于组合投资、价值投资、量化宽客等领域。因此，市值管理是一种理论的研究，需要结合实际场景、市场环境及时掌握市场变化、更新自己的管理技能和做法。

# 2.市值管理的定义

市值管理（Value Management）通常指管理和调控所有市场中流动资产的价值，包括股票、债券、商品期权等证券产品和各种金融资产。市值管理通过调整市场中资产价格、数量或比例，使得整个市场在收益和风险方面均达到平衡，从而最大限度地发挥市场对投资者的效益。一般来说，市值管理包含四个层次：

1. 信息收集阶段：建立有效的信息渠道，获取投资者对市场信息的真实认识；
2. 投资决策阶段：通过历史数据和分析模型进行预测和分析，综合考虑多方面的因素制定正确的投资策略；
3. 执行执行阶段：将预测结果和分析模型转化为实际操作命令，通过系统性的方法实施调整；
4. 持续监控阶段：持续不断地进行信息收集、数据分析和执行操作，确保市场的运行状态始终处于平衡状态。

市值管理可追溯至19世纪末，奠定了市场有效运营的基础。到今天，市值管理已成为国际上最重要的战略性制度。

# 3.市值管理的原则

市值管理具有以下几个原则：

1. 独立性：市值管理必须与资产价格无关，否则会造成泄露。如，如果管理基金追求绝对收益，就不会受到股市波动的影响。
2. 可复制性：市值管理不能依赖某个特定的投资策略，应该能适应市场状况的变化。比如，随着政策或市场环境的变化，需要根据变化情况调整仓位，而不是一味采用策略。
3. 最小成本：市值管理应该保持低成本，才能降低管理风险。如，管理基金不要设置高门槛，因为存在被骗的可能性。
4. 透明性：市值管理过程必须公开，给予所有参与者充分的了解。
5. 稳健性：市值管理必须做好充足的准备工作，做到事先措施、事中检查、事后监控，减少不可抗力造成的损失。
6. 对称性：市值管理应由多个团队共同承担，各自负责不同的子市场。
7. 个性化性：市值管理可以根据个人兴趣、偏好和知识水平来选择不同的管理工具。

# 4.市值管理的主要环节

市值管理的流程总体可以分为以下几个环节：

1. 预测阶段：提前得到市场中各项资产的相关信息，分析资产的长期趋势，通过预测其价值大小、波动幅度和相关参数。例如，预测某只股票的涨跌幅度、下轨位置、当前价格、期望回报率。预测阶段可帮助我们准确判断投资方向和规模，并制定出色的投资策略。
2. 跟踪阶段：跟踪投资者心中的信念，根据最新市场数据和预测数据来做出调整。通过跟踪新闻、报告、研究报告等获取信息，跟踪不同市场资产的走势，包括股票、债券、指数等，可以为投资者提供最新的数据支持。例如，跟踪政府债券的资金流向，通过预测资金的流向和增速，制定稳健的金融计划。
3. 测算阶段：对投资者持有的资产进行详细的分析和评价。例如，检查每只投资标的的历史收益率、净资产收益率、年化回报率等指标，并进行比较和分析。识别资产的风险点，制定相应的防范措施。
4. 优化阶段：通过优化预测结果和市场状况，对资产的配置进行调整。例如，对不同资产的比例进行调整，调整仓位，使资产的收益率尽可能接近预期。
5. 控制阶段：保持整个市场的运行在合理的范围之内。例如，禁止非法配售、限制炒作或过度抢购股票等行为，保持股市不贬值。
6. 风险管理阶段：识别和减小投资者的风险。例如，监控各种投资产品的市盈率、溢价率等指标，发现异常情况时采取警惕性管理措施。
7. 评估阶段：总结经验教训，提升自己作为投资人的能力和品牌。例如，在较短的时间内，测试并行的方法、工具、管理方式，提升自己的投资绩效。

市值管理的关键是要把以上几个环节全部落实到位，提高管理效率、降低管理风险、提高管理绩效。

# 5.市值管理的应用

市值管理的应用遍及于众多领域，包括房地产、保险、医疗、军工、电子商务、零售等领域。在这些领域，市值管理可实现以下几点作用：

1. 资产配置：市值管理可以为企业或个人在多元化资产配置中提供依据，给予最优的资产组合。
2. 市场管理：市值管理可对整个市场进行有效的管理，保持市场整体的稳定和平衡。
3. 资产溢价和传导性：市值管理可以在所有市场中发现和解决投资者的资产溢价问题。
4. 结构性投资：市值管理的目标是在各个市场中发现最佳的结构性投资，充分发挥资本的价值。
5. 资产分配：市值管理可以更加科学地分配资产，符合风险和效益要求。
6. 股权激励：市值管理可实现股权激励的制度化、规范化，将更多的资源投入到股东当中。

# 6.市值管理的价值

市值管理的价值不亚于其他管理理念。它有助于企业降低风险、促进创新和创业，以及促进创业、规模化发展和个人成功。市值管理具备如下四个特点：

1. 控制风险：市值管理可以减少投资风险，提高投资回报，更好地满足投资者的需求。
2. 提高获利：市值管理可以提高投资者的获利空间，通过优化投资策略和结构，增强投资者的信心。
3. 激发创新：市值管理可以提出创新的想法，重新定义企业的核心竞争力，激发企业的创新潜力。
4. 促进创业：市值管理可以促进企业的发展，让更多的人进入这个领域，创造出更多的财富和产品。

因此，市值管理是投资的战略性制度，是投资者应对资产价值、政策变化、市场变迁所做出的最坚决的选择。市值管理是一个综合的管理理论，能有效地提升投资者的运用能力，为企业的管理提供有效的工具。
