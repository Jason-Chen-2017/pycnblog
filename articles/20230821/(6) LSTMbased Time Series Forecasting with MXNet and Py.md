# 2.基本概念术语说明
首先，我们先回顾一下时间序列预测任务的基本概念，它可以看做是一个连续变量序列到另一个连续变量序列的映射。时间序列预测任务一般包括以下四个子任务：

1. 时序数据的收集。
2. 时序数据预处理，即对原始时序数据进行处理，得到标准化、平稳化后的训练数据。
3. 时序数据的建模，即建立模型，将输入序列映射到输出序列。
4. 模型的训练，即训练模型，使得模型的参数与训练数据拟合最佳。

接下来，我们将对以上概念和术语进行进一步的定义和解释。

### （1）时序数据

时序数据（Time series data）是指一段时间内随着时间变化而产生的一系列观察值，这些观察值通常称为时间步或者观测点。例如，股票市场中每天的收盘价就是一段时间内的时序数据；季节性时间序列数据如气候数据也是时序数据。时序数据又可按是否是连续的、间隔的时间长短、不同时期的观测值组成不同的类型。

### （2）时序模型

时序模型（Time series model）是对时间序列数据的建模方法，它定义了如何根据过去的观察值预测未来的观察值，主要有三种： autoregressive model（AR）、 moving average model（MA） 和 autoregressive moving average model（ARMA）。

#### AR Model

AR模型（autoregressive model）是一种简单且广泛使用的时序模型，它认为在第t期的值只依赖于之前固定数量的期望值，即：$X_t = c + \sum_{i=1}^{p}\phi_ix_{t-i}$。其中c为截距项，$\phi_i$ 为AR系数，p表示AR模型中的参数个数。

#### MA Model

MA模型（moving average model）是AR模型的特例，当存在单位根（白噪声）或自回归效应时，它才有意义。假设在第t期$Y_t$的过程中除了受到影响的过去$q$期之外，其他所有期望都可以通过它们的移动平均来计算，即：$Y_t=\mu+\epsilon_t+L\epsilon_{t-1}+\cdots+L^{q-1}\epsilon_{t-q+1}$，其中$\mu$ 是单位根，$L$ 为单位根的自相关系数，$\epsilon_t$ 是白噪声，$|L|>1$。

#### ARMA Model

ARMA模型是同时考虑AR和MA两种模型的结果，它认为在第t期的值由两部分决定，第一部分由AR模型决定，第二部分由MA模型决定，即：$X_t = c + \sum_{i=1}^{p}\phi_ix_{t-i}+\mu+\epsilon_t+L\epsilon_{t-1}+\cdots+L^{q-1}\epsilon_{t-q+1}$。

### （3）训练数据与测试数据

训练数据（training data）、验证数据（validation data）、测试数据（test data）分别对应着模型的训练阶段、模型的超参数选择阶段和模型的测试阶段。训练数据用于训练模型参数，选择模型结构，验证数据用于选择模型超参数，测试数据用于评估模型的最终性能。一般来说，训练集占总数据集的70%，验证集占总数据集的20%，测试集占总数据集的10%。
