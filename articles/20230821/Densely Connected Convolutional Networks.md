# 1.简介
  

## 1.1 为什么要做论文
近年来，由于在图像分类任务中的硬件性能的提升、计算资源的增加、数据集的丰富，神经网络已经成为计算机视觉领域的一个热门研究方向。然而传统的卷积神经网络(Convolution Neural Network, CNN)仍然是一个有效且高效的模型。因此，很多研究者都试图探索更好的模型结构来进一步提升CNN的性能，比如残差网络ResNet、注意力机制Attention Mechanism等。但是这些模型结构往往会引入新的复杂性，因此，如何从根本上解决网络容量（即参数数量）的问题是一个关键难题。在这个问题上，论文作者认为有必要给出一种新颖的模型结构——Densely Connected Convolutional Networks (DCNN)。DCNN通过在多个不同层之间引入密集连接，可以有效地降低参数数量，同时保持准确率的提升。
## 1.2 Densely Connected Convolutional Networks 的特点
DCNN 是一种非常有创意的模型结构，它能在相邻的卷积层之间引入密集连接，使得参数数量减少，加快了训练速度并提升了准确率。其主要特点如下:

1. 多层间的密集连接：DCNN 中的卷积层通常都是空间域相邻的，因此，通过将多个相邻的卷积层连接起来，可以帮助模型学习到丰富的特征。

2. 分辨率下降问题的解决：由于 DCNN 中存在许多的隐层节点，因此，当特征图的分辨率低于输入图像时，信息就会丢失。因此，可以通过将网络中所有隐层的输出连接成一个，这样就能有效地提升分辨率。

3. 更多的可学习参数：DCNN 带来的更多的可学习参数意味着它具有更强大的表达能力。

4. 无监督预训练技术：DCNN 提供了一个无监督预训练的技术，可以直接应用于新的数据集，使得模型具备很好的泛化能力。

总体来说，DCNN 模型具有以下几个优点:
- 降低了参数数量，加速了网络的训练和推理时间。
- 解决了分辨率下降问题，保证了信息的完整性。
- 提供了一种全新的网络架构，有利于提升模型的精度。

## 2.1 VGG-16

## 2.2 DenseNet
DenseNet是由稠密块(dense block)和transition layer组成的。稠密块由多个卷积层和BN层组成，而且每一层都是连接前后所有的层，从而能够产生更大的感受野。transition layer则对特征图进行压缩，防止过拟合。DenseNet网络由多个稠密块和最终的分类器组成。如图2所示。

# 3.DCNN 原理及实现
DCNN 通过将多个不同层之间的卷积层连接起来，可以有效地降低参数数量，同时保持准确率的提升。DCNN 利用多个独立的卷积层替代 VGG-16 的全连接层，其中每个卷积层使用步长为 $1$ 和 $3$ 的卷积核，并且都带有 BN 操作。相邻两个卷积层之间存在一个扩张通道维度的连接，称之为 dense connection 。每个卷积层的输出通道数相等，而且是上一层输出的两倍，这样就可以有效的降低参数数量。接着，为了解决分辨率下降问题，DCNN 在不同卷积层之间引入密集连接，通过连接所有隐层的输出，可以有效地提升特征图的分辨率。最后，使用全局平均池化和丢弃层来作为分类器。
