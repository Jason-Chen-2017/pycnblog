# 1.简介
  

关于写作，归根到底是一个如何有效组织语言，引导读者快速理解的技巧。那么如何才能将这些技巧应用于创建好看、易懂、正确、知性的文案？这一切都要取决于我们的文案策划师具备的相关知识和能力。

在刚入门时，不妨先了解一下什么是文案策划。文案策划（英语：copywriter）是指从事广告、企划等媒体宣传的写作者。他们根据目标受众、需求及品牌策略制定、编辑、拍摄或撰写各种类型的媒体材料，如电视报道、网络内容、广播节目、海报、广告刊物、出版物、互联网网站等。本职工作主要包括创意策划、文章撰写、信息设计、网站维护、文案翻译和出版社社长等领域。

简单来说，文案策划就是一个写作者，他/她的任务是为媒体产品、网络内容或营销活动提供符合用户需求的新闻和市场营销信息。作为媒体人，你需要熟悉新闻事件的情况、消费者心理、竞争对手的优劣势，并与这些因素结合起来推出符合目标受众的价值观。因此，无论你的产品或服务是否成功，一定要为你的文字编写一份实用的、风格统一的、与受众阅读习惯相符的商业文案，以及一系列基于数据驱动的营销策略。

写好了文章，下一步就该去投放市场了。这里你会需要跟进报纸、新闻网站、论坛、社交媒体以及各种营销渠道，进行推广、促销、销售等工作，让你的信息更加透明、迅速地传递给目标用户。

# 2.1 文案类型
文案可以分成以下几种类型：

1. 普通型：普通型文案适用于绝大多数类型的营销活动。通常包含个性化的内容、产品介绍或服务项目信息等。

2. 专题型：专题型文案适用于大型的促销活动，涵盖多个行业或主题。通过一系列热点话题、产品优势或服务信息来吸引受众，提升目标客户的认识度和兴趣。

3. 连锁型：连锁型文案是由多个不同方向的信息组合而成的一组具有亲密性的消息。例如，航空公司需要发出一则旗帜性的消息，所以一般都会使用连锁型文案。

4. 悬浮型：悬浮型文案是一种很新的形式，它是在移动设备上展示的文本信息。它鼓励用户点击链接或者按钮，完成注册或参与活动。悬浮型文案属于短平快，容易引起用户注意，所以通常很少用于营销活动。

除了上面这四种类型外，还有一些特殊的文案类型：

1. 品牌形象文案：品牌形象文案在营销活动中非常重要。品牌形象文案描述的是企业的形象、声音和使命，是营销活动中的第一句话。品牌形象文案需要高度掌控、语调清晰、用词精准，以至于达到令用户记住、喜爱的程度。

2. 温馨提示文案：温馨提示文案常常出现在网站首页、APP的“帮助”页、产品内的帮助中心等位置。当用户在浏览某些产品或功能时，会看到这样的温馨提示。温馨提示文案应简洁、直接、明了，并配有相关的图标或提示信息，帮助用户快速解决问题。

3. 提示性文案：提示性文案即向用户介绍产品的特性、优势，或帮助用户解决特定问题。提示性文案适用于某些场景，如教育、金融、公共事业等领域。

# 2.2 文案结构
所谓文案结构，就是指依据所采用的媒体形式、平台或媒介的特点，来定义每条文案的主次关系、时效要求、视觉效果和引导用户操作方式。在做文案策划的时候，需善于利用不同的文案结构来达到营销目的。

比如，对于一封电子邮件，其结构应该遵循从顶部到底部的顺序，包括：标题、副标题、图片（视觉效果）、正文、图片（视觉效果）、签名、地址。
