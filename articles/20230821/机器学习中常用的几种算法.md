# 2.概率图模型（Probabilistic Graphical Model）
概率图模型是一种用于表示由随机变量及其条件独立性所构成的图模型，它是一种灵活且通用的建模工具，可以用图结构表示由各种随机变量组成的数据集。该模型的主要特点是描述分布，将数据视为是联合分布的形式，将各变量之间的关系用无向边连接起来，每个节点代表一个随机变量及其取值集合。概率图模型具有以下五条基本假设：

1. Markov假设：随机变量X的观察序列只依赖于当前时刻前面的信息，即X_t仅仅受到X_{t-1}的影响。
2. Chain Rule假设：已知所有变量的当前状态，则任意两个变量间的关系都只依赖于它们最近的共同祖先。
3. Factors假设：同一个变量的不同取值之间相互独立。
4. Cardinality假设：每个变量有固定的取值个数。
5. Independence Assumption（交互独立假设）：任意两个随机变量间的条件独立性都是确定的，并且与其他变量和观测无关。

概率图模型的学习通常采用极大似然估计或贝叶斯估计的方法，极大似然估计就是试图找到使数据出现的概率最大的模型参数，而贝叶斯估计则是求得模型参数后利用这些参数来计算数据的后验概率分布。概率图模型可以应用于很多领域，包括信息检索、图像分割、文字识别、生物信息学等。

# 3.决策树
决策树(Decision Tree)是一种常用的机器学习分类算法，它是一种树形结构。决策树模型是基于特征选择进行决策的一种算法，它可以做出预测，或者根据给定的输入数据，输出其所属的类别或离散值。决策树模型由树结构和分支结点组成，树的每一条路径对应着一个测试，根据数据中的特征选择最优的一条路径作为判断的依据。其基本原理是：从根节点开始，递归地划分训练样本，在每个节点根据特征选择最优的一个特征，然后分裂为两个子节点，直到所有的训练样本都被分配到叶节点。如下图所示：
# 4.支持向量机
支持向量机(Support Vector Machine, SVM)是一种二类分类模型，它是一种二项损失函数的线性模型。SVM 通过找到最佳的平面将数据分为两类，使得 margins（边缘距离）最大化。当数据集中存在异常值时，SVM 会将异常值的 margin（边缘距离）增大，从而可以更好的分类。SVM 的基本算法流程为：1、首先，确定一个超平面作为分类的决策边界。2、然后，通过不断修正超平面直至找到使得数据集的误分类最小的超平面。3、最后，用此超平面对新的测试样本进行分类。如下图所示：
# 5.K近邻法
K近邻法(K-Nearest Neighbors, KNN)是一种非监督学习算法，它是一种基于距离度量的分类和回归方法。KNN 从训练数据集中找到与新输入实例最接近的 k 个实例，然后基于这 k 个实例的类别投票决定新实例的类别。KNN 可以用于分类、回归、关联规则学习、异常检测、推荐系统等任务。如下图所示：
