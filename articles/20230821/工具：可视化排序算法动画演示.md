# 2.背景介绍
在大学学习中，我们会接触到一些排序算法，如冒泡排序、选择排序、插入排序等。但其实，在实际工作中，我们更多地使用的是快速排序、归并排序、堆排序等高级排序算法。因此，了解一些更加基础的排序算法对我们理解高级排序算法非常有帮助。
排序算法的目的是按某种顺序排列元素，对于一个已知长度为n的数据序列，其排序过程通常分为三个步骤：
1. 比较：比较两个元素，如果满足要求则交换位置；
2. 分割：将序列分成两部分（称作左右子序列），其中一部分的所有元素均比另一部分的小，称为左边子序列或左半部分；
3. 折叠：重复执行比较和分割，直至所有元素都被排序完毕。

# 3.基本概念术语说明
## 数据结构
由于算法需要对特定类型的数据进行排序，所以我们需要先了解相应的数据结构。我们首先讨论最基本的数据结构——数组（Array）。数组是一种线性存储结构，它是用来存储相同类型的元素的集合。数组中的每一个元素都有一个唯一的索引值。
## 排序算法
### 插入排序(Insertion Sort)
插入排序（英语：Insertion Sort）是一种简单直观的排序算法。它的工作原理是通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。
### 希尔排序(Shell Sort)
希尔排序是一个基于比较的排序算法。该方法将已划分的序列按下标的一定增量分组，对每组使用直接插入排序法排序。然后缩减增量再进行类似的排序，直到整个序列有序。
### 选择排序(Selection Sort)
选择排序（Selection sort）是一种简单直观的排序算法。它的工作原理是首先在待排序的元素中找到最小（大）元素，存放到起始位置，然后，再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。经过几轮迭代后，整个序列将变成有序。
### 冒泡排序(Bubble Sort)
冒泡排序（英语：Bubble Sort）也是一种简单直观的排序算法。它重复地走访过要排序的数列，一次比较两个元素，如果他们的顺序错误就把它们交换过来。走访数列的工作是重复地进行直到没有再需要交换，也就是说该数列已经排序完成。这个算法的名字由来是因为越小的元素会经由交换慢慢“浮”到数列的顶端。
### 快速排序(QuickSort)
快速排序（Quicksort）是对冒泡排序的一种改进。快速排序的基本思想是通过一趟排序将要排序的数据分隔成独立的两部分，其中一部分的所有元素都比另外一部分的所有元素都小，然后再按此方法对这两部分数据分别排序，直到整个数据都有序。
### 堆排序(Heap Sort)
堆排序（英语：Heapsort）也是一种基于比较排序的算法。它是不稳定排序算法。所谓不稳定排序算法就是指该算法不会像所有其他算法那样按顺序排序。

## 1. 插入排序 Insertion Sort
**基本思路**：插入排序的基本思想是将一个数据插入到一个已经排好序的有序数据集合中，使之仍然有序。一般来说，插入排序都采用在数组/列表中循环遍历的方式实现。


**过程描述**:

1. 将第一个元素看做已经排序好的，把第二个元素依次和它比较，插入到前面已经排好序的有序数组中的适当位置。

2. 从第三个元素开始，与前面的元素依次比较，若小于当前元素，则与当前元素相邻。即第i个元素与第[i-1]~[0]个元素比较，插入到第[i-1]个元素后面。

3. 重复上述过程，直到所有元素均插入完毕。

**数学公式说明:**

1. 每一步都是从后往前插入，所以时间复杂度为O(n^2)。

2. 在最坏情况下，每个元素都只需逆序，即全部移动到数组开头，那么最坏情况下的时间复杂度为O(n^2)。


```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```
