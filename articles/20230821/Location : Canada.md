# 1.简介
  

Canada是一个全球性的国家，拥有超过79%的人口居住在城市中，同时也是一个多元文化、多民族的国家。Canada是一个高度发达的国家，拥有世界上最丰富的经济发展潜力。Canada也是加拿大的一部分。它的历史上曾经是一个殖民地，但近年来，随着经验主义者的兴起，越来越多的人正逐渐把它当作一个自由的国家来看待。人们生活水平在世界各国中名列前茅。Canada有非常先进的科技、医疗保健设施，在世界上的影响力是独一无二的。

Location : Canada（加拿大）简介
加拿大是一个由五大洲组成的北美大陆国家，位于美国西部，南临大西洋，北邻墨西哥海峡。加拿大是一个高度发达的农业国，拥有庞大的森林资源，自然资源充足。加拿大是一个多元化的社会，其不同的语言、宗教信仰及社会结构都使得加拿大成为一个多样化的国家。加拿大是一个高度发达的经济体，主要经济指标为GDP总值，占全球经济总量的2.2%。据统计，加拿大是全球第四大赢家，并且在全球贸易额排名前十。因此，加拿大是当今世界上真正意义上的自由经济体。

人类历史上曾经几乎没有过一个自由的国家。古代，埃及、印度、罗马尼亚、斯里兰卡等国家在被纳粹屠杀之后，陷入了暴政的蹂躏之中。而中国古代帝王即位后，为了夺取天下，不惜牺牲小国，企图将其驱逐出境。而清政府则试图通过制定法律赋予君主以超越个人利益的权力，让整个国家的百姓实现平等。于是，19世纪末，英国诞生。19世纪80年代，美国总统华盛顿宣布成立联邦政府，并承认了加拿大作为独立国家的合法身份。同年，加拿大与美国签订了第一个海外投资协定。

20世纪90年代，加拿大成为全球第八大市场，主要原因是其产业结构多元化、跨越多个领域、高科技含量较高，适合加强经济整合。目前，加拿大已成为欧洲、北美、亚太及非洲四个主要区域和地区的主要贸易伙伴。2011年，加拿大第三季度GDP总值突破9万亿美元，创历史新高。

Location : Canada 技术概况
- 天气服务(Meteo)：提供实时天气数据，包括每日气象、日出日落、七日预报等信息；
- 消费电子商务：加拿大最大的消费电子商务平台，拥有超过一千亿美元的营收和服务记录；
- 儿童用品：拥有世界最大的儿童用品市场，消费者在这里可以找到各种儿童用品、玩具、早教产品；
- 农业/林业：加拿大最大的农业和林业专业机构，拥有超过300多所研究中心和大学；
- 大学：加拿大最大的科研和教育机构之一，拥有超过10万名留学生；
- 媒体：加拿大最大的三大新闻通讯社之一，拥有超过150亿美元的营收；
- 公共卫生：加拿大拥有全国最好的公共卫生系统，覆盖全国近13万个街道；
- 铁路交通：拥有超过一千座重要铁路线，可连接全国各处；
- 商业银行：加拿大最大的商业银行之一，是全球第一家商业银行；
- 航空运输：国际航空领域的老牌公司，积累了丰富的历史资料；
- 石油和天然气：加拿大拥有世界上最大的石油和天然气开采基地，加拿大气象局宣称该地区是世界上核能和风能发电的最佳产地。
