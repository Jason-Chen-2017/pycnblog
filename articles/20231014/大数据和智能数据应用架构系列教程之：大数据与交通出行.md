# 1.背景介绍


## 大数据的概念与特点
### 数据量大
互联网、移动互联网等新时代带来的海量数据存储需求，使得在过去几年产生了大量的数据信息，数据种类也越来越多。如今，人们生活中所产生的数据已经远超原始数据，各种各样的数据源已经涌现出来并纷纷对个人或组织产生影响。从而形成了庞大的社会数据市场。
### 数据采集实时性要求高
随着人口、经济、地域等因素的不断变化，社会运动、政策法规的变化需要及时的了解信息，因此对于数据的获取、处理和分析都要在短时间内快速响应，否则可能造成损失甚至事故。同时，数据采集过程也需要具备高容错性，即对数据传输过程中出现的错误、异常等进行及时处理。
### 数据类型多样且复杂
由于各种各样的社会行为、活动产生的大量数据，各种类型的分布在不同业务领域和场景下，如：图像、文本、视频、音频、位置、社交网络、点击流、日志、订单、交易等。这些复杂的数据难免会带来新的挑战。
### 数据价值高度异构
由于数据采集、存储及处理需要对不同类型的场景和业务需求进行考虑和开发，因此，数据的价值也呈现不同的高度。如客户、产品、服务、交易、营销、媒体、知识产权、金融、电信、健康、食品安全、环境保护等，每一种数据的价值都不同。
### 数据使用效率低
无论是从数据分析还是数据应用的角度看，数据本身就是一个具有商业价值的载体。但是，如何通过有效地收集、整合、分析、展示、传播数据，将数据转化为有价值的商业价值，是每一个企业都面临的核心问题。由于信息过载，数据处理效率不足，数据处理流程长期处于停滞状态，很难产生改进效果。
### 大数据与云计算的结合
随着大数据的数量激增，云计算平台的发展，使得云端的数据中心成为实现大数据分析的重要工具。随着云计算的普及，更加注重大数据的价值发现、数据治理、数据可视化、数据驱动业务变革，带来了巨大的机遇。但同时，大数据的生命周期长、数据价值复杂，容易产生数据孤岛、数据隔离、数据隐私问题，需要与云计算相结合。

## 大数据与交通出行
随着互联网、移动互联网的蓬勃发展，交通方式发生了翻天覆地的变化。每年城市交通量的增加、乘车人数的增加、绿色出行的推广，让我们看到交通管理变得日益复杂、智能化，以前人们无法想象的速度快速发展。在这种背景下，如何利用大数据来提升交通管理？如何保障城市的交通秩序和交通安全？如何进行出行驾驶习惯的优化？今天就以大数据和交通出行作为主要话题，探讨如何利用大数据技术提升交通管理和出行效率。

# 2.核心概念与联系
## 大数据概念简介
大数据指的是一种存储海量数据的系统，一般由多台服务器组成的集群，通过网络链接，能够存储、处理和分析海量的数据，它可以用来做很多领域，包括电子商务、金融、科技、军事、经济、医疗、农业、保险等等。以下是大数据领域的一些基本概念：

1、数据量大（Volume）：指数据规模非常庞大，通常是指TB、PB级别的数据。

2、数据源多样（Variety）：数据既有结构化、半结构化、非结构化等各种形式，又包括来自各个领域的信息。

3、数据实时性要求高（Velocity）：指数据收集、传输、处理的时间间隔非常短。

4、数据类型多样且复杂（Veracity）：数据既有结构化的数据，也有非结构化的数据，还有包括图片、视频、音频、文本等各种多样化的数据类型。

5、数据价值高度异构（Value）：数据既有经济价值，也有社会价值、文化价值等。

6、数据使用效率低（Efficiency）：数据处理流程长期处于停滞状态，很难产生改进效果。

7、大数据与云计算的结合：由于数据量太大，云计算平台提供的资源能力无法满足数据处理的要求，因此需要结合云计算平台来实现大数据分析。

8、数据孤岛、数据隔离、数据隐私问题：当数据量和数据使用者越来越多时，数据孤岛、数据隔离、数据隐私问题会越来越突出。
