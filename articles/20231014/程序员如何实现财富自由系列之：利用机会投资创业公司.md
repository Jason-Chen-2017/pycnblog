# 1.背景介绍


“没有什么投资比在一天之内获得超额回报更有吸引力了。”——基辛格先生
如今，“股票”、“债券”等投资工具已经成为人们的日常生活。作为个人投资者，无疑对市场上众多的股票、债券产品十分熟悉，但究竟如何才能实现财务自由呢？能够通过自己的努力获取“超额回报”吗？
虽然现阶段并不存在真正意义上的投资理财工具，但是在互联网的浪潮下，越来越多的人开始关注于如何获取财务自由的途径。因此，本系列教程将用通俗易懂的语言讲述到，一个合格的程序员应该如何利用自己的技术能力来实现财务自由。首先，我们就从程序员如何利用机会投资创业公司这一课题进行开讲吧！
# 2.核心概念与联系
## 2.1 投资市场简介
投资市场是一个活跃的市场，每天都有新的信息源源不断涌现出来，不过它还是人们不可或缺的参与者之一。市场中存在着很多类型的投资品种，其中最常见的就是股票、债券等。一般来说，每一种投资品种都有其独特的特性，比如有些可以长期持有，而另一些则具有短期获利性质。
## 2.2 创业公司简介
创业公司通常指的是面临企业破产时需要重新启动经营的企业，创始人往往经过几年甚至十几年的奋斗最终成为了成功的商业领袖。创业公司属于非传统经济类企业，其创始人即为创业企业家或者企业家协会设立的顾问团队中的一员。创业公司的创始人，也称为“CEO”，他/她的主要工作就是创造、发明、开发新产品，扩大企业规模，推动企业的发展，创造收入和利润。
## 2.3 创业投资方法简介
19世纪末，美国科技巨头贝尔实验室创始人沃伦·巴克（Warren Buffett）提出了“机会投资法”。他认为，企业的发展离不开市场的驱动。当公司面临无法盈利时，他建议投资者应该不遗余力地购买该公司在市场上的最新产品及服务，这样就可以带动整个行业的发展。在此基础上，他还提出了创业投资法，旨在鼓励企业家们发起新的创业活动，利用机会，筹集资金，并将这些资金用于新兴事物的研发。这种新型的投资模式也被称为“机会投资”，相对于传统的股权投资，它注重发掘创业企业所处的社会经济环境中所释放的机遇。

虽然中国在2007年新政出台后，成立了国际创投证券交易委员会(ICITI)，但是目前还有很大一部分创业公司仍然处于封闭模式，而且大部分创业投资依旧由政府和私募股权基金主导。尽管如此，本文将阐述如何通过自己的编程能力，结合线上渠道赚取个人微薄的收益。
