# 1.背景介绍


半导体（英语：semiconductor），又称电子晶体、电子材料，通常指结构性很强且具有高韧性的金属元素材料。它包括硅钢（silicon）、锗（germanium）、镍（silver）、铜（copper）、镁（lead）等元素，属于非常重要的物理学研究领域，在科技领域也扮演着举足轻重的角色。如今，半导体已经成为信息处理、通讯、计算、生物医疗、工业自动化、纳米技术、航空航天、环境监测、车载导航等各行各业中的关键环节。由于半导体的高韧性、超厚度、导电性、低温耗散特性、耐热、光导效应、自旋能量释放、集聚效应、难以精确刻画，使得其科技含量极高，也是学术界和工程界极具影响力的“黑马”。 

但是，作为一个复杂的多层次、多环节的产物，半导体还涉及到众多的理论与实践问题，其中有很多原理尚未被完全理解或者还存在争议。如何用简单的语言准确地阐述半导体的各种物理性质，从而应用到实际工程中，是一个难题。因此，本文试图通过系统阐述半导体的物理原理、常见术语和常用的半导体器件，为读者提供理论指导，并希望能对相关的技术方向和发展方向产生启发。

# 2.核心概念与联系
## （一）物理界定词汇
### 2.1 粒子与波
物理学的一个基本假设是存在宏观的世界，这个宏观世界由微观的粒子组成，微观的粒子通过一定规则运动，因而构成一种规则的宏观场面，这种宏观场面的观察可以转化为少量的空间和时间的信息。现实生活中，人们经常会用微观的感官来观察宏观的现象，比如一股风吹过来的声音，微小的颗粒移动的方向，晃动的钟表等等。由于这些微观的粒子是遵循规则运动的，所以宏观的现象可以被分解为许多微观的过程，这些微观的过程的结果会导致更微观的过程，所以微观的过程反复相互作用，最终构成整体的宏观过程。对于我们来说，所谓的宏观过程，就是指物体表面发生的固态物质中的粒子运动，或者就是液体中微生物的排泄，或者就是湿度随时间变化的变化，或者就是光线的传播等等；而微观过程，则是在一个时间段内由于不同原因引起粒子位置的不断变化，并且每隔一定的时间会衰减，所以只有较短的时间才能观察到微观过程的细节。

为了能够观察宏观过程的微观过程，物理学引入了波的概念。波简而言之就是一个变量变化的规律，可以理解为某个物理量随时间的变化，波具有时域和频率两个维度，分别表示时间的流逝和周波数的变化。我们熟悉的液体的流动、光的闪烁都属于波的变换形式。无论是宏观过程还是微观过程，只要有时间这一概念，就可以把它抽象为波的叠加。正因为有了波的概念，才能够观察到宏观过程的微观过程。

### 2.2 晶体结构与电子结构
前文已经提到了半导体（英语：semiconductor）这个词，这是指由结构上具有高韧性的金属元素组成的电介质。因为半导体的性能极其突出，已经成为当前最热门的科学研究领域之一，但由于这一领域的复杂性和庞大规模，仍然存在一些基础知识和理论上的问题没有得到充分解决。因此，本文将从“物理”、“机械”和“化学”三个层次入手，探讨半导体的物理结构、机理与功能。首先，先从“物理”层次看半导体的基本构造。

## （二）物理结构
### 2.3 费米-莫尔真空
#### 2.3.1 概念
费米-莫尔真空是描述材料的物理性质的一系列假设。它认为真空中的任一点的任何相邻两点之间的距离都相同，即无穷远处不存在任何物质。由于真空中的任何两个粒子之间都不能直接感受到彼此，因此不能形成任何运动。费米-莫尔假设暗含了一个重要的原理——费米假设，即存在着一种叫做费米能带的状态，其能带宽度是无限大的，这种状态下的物质会保持其绝对静止，即不会发生相互作用。由费米-莫尔假设导出另一个重要的原理——莫尔假设，即所有的物质都具有波函数。波函数的存在意味着任意两个粒子间都存在着一种互相作用的方式，这就使得能带宽度的无限大化成为可能。事实上，费米-莫尔真空的物理性质给我们提供了一种全新的理解材料的机制的视角。

#### 2.3.2 证明方法
费米-莫尔真空的证明很简单，只需要利用微观粒子的静止不动，质子的自由落体或漂浮到任一边界条件，假设粒子无穷远处无任何相互作用，无穷远处不能形成任何运动即可。在这个假设下，我们可以利用运动学的定义，找到质子的运动轨迹，例如，若质子在无限长的半径内旋转，他必定会沿着某条直线平行运动，这条直线便是这个粒子的运动轨迹。当质子的运动轨迹再和相邻的一边界条件进行比较，我们就知道它们之间的距离是否相同。费米-莫尔假设暗含了这样一条推理：一个极限的距离和时间，在任何给定的宏观尺度上都是相同的。

### 2.4 金属性与导电性
#### 2.4.1 概念
在谈到半导体的物理性质之前，需要认识到，物质的性质主要取决于它的金属性、磁性、电性、氮化物性等多种性质，而金属性是决定半导体物理性质的核心特征。金属性，一般来说，是指金属元素的两种基本性质：稳定性和导电性。稳定性是指金属能否承受强力的拉伸或压缩，导电性是指金属是否可以导电，在电路上扮演怎样的角色。在半导体中，金属的稳定性与导电性是互为影响的。如果一个金属原子的电荷密度很低，那么它就会失去导电性，从而不能导电。相反，如果电荷密度很高，那么它就拥有导电性，而且可以导电。因此，可以说，稳定性是导电性的必要条件。同样的道理，金属的稳定性也会影响其导电性。

#### 2.4.2 典型金属性
目前，对于一般的导电材料来说，金属有三种典型的属性：质量常数（取决于金属元素的质量）、导电常数、超导常数。

（1）质量常数：

质量常数是一个长度单位，与金属的性质密切相关。对于单一金属原子，它取值为0.0909，对于聚合物，它取值可能达到1-2。其含义是，无论一个金属的质量是多少，其几何尺寸都不会改变。因此，质量常数反映了金属的硬度，其越高，硬度越高。典型的硬度有300GPa、200GPa和150GPa。

（2）导电常数：

导电常数是一个电压单位，与金属的导电性密切相关。导电常数是衡量金属的导电性能的重要指标，其值的大小与导电性能成反比。当导电常数大于1时，金属就具备良好的导电性。当导电常�等于1时，金属就不可导电。典型的导电常数有1.01、1.1和1.2。

（3）超导常数：

超导常数是衡量金属的超导性能的重要指标。其值越大，超导性越好。典型的超导常数有11和13。超导常数实际上可以看作一种能级差，其作用类似于声音的阶差。

#### 2.4.3 核反应堆效应
核反应堆效应（NEXAFS，Nuclear EXtreme Across Spectrum Absorption Fine Structure）是指核反应堆中高能物质的辐射能量因自然损失而产生的暂时的增益，其根源是核反应堆内部微观的裂变生成的热辐射，对高能物质的辐射能量的影响是暂时的，因为在核反应堆内部恢复后，辐射能量会自动降低。

核反应堆效应是在实验室中发现的，它是由核反应产生的临时增益，对某些高能物质的辐射能量有影响，但它不是无缘无故地出现的。它主要影响材料的开辟、功能维护、材料性能等方面。根据实验的不同，对不同类型的材料有不同的影响。由于核反应堆的存在，高能物质的辐射能量持续产生，在过往的核裂变研究中，有关宏观微观的分析一直困扰着学者们。而关于核反应堆效应的研究才刚刚开始，它可以帮助我们理解核反应堆的动态性、能级分布、辐射通道等基本的性质，有助于提升材料、设备、系统的制造和管理水平。

### 2.5 晶体结构
#### 2.5.1 概念
晶体结构是指由一个或多个不同形状的原子组成的晶体，其最常见的例子是形状相同的平行四极体。晶体结构的一般特征是，该晶体的所有原子都在一个晶格里。在实践中，晶体结构可以由多种晶系组成，例如在金属中，普通的五方晶系属于diamond晶系，具有特殊的形状和电性，在几何学上是个球体，在电性上是个良好的金属。在非金属的晶体中，常见的还有正立晶体、六方晶系、四方晶系。

#### 2.5.2 diamond晶体
diamond晶体是硬度最高的晶体，有三种结构，圆形晶体，正方形晶体，方形晶体。diamond晶体的晶格常有5.4311A，并且由四种元素组成，分别为：碳（Carbon）、砷（Cobalt）、氧（Oxygen）、硅（Silicon）。diamond晶体的晶粒直径大约为7nm。

diamond晶体结构的构建方法如下：

第一步，将diamond晶体呈现为四个平行四边形，其边长分别为a、b、c、d，即：a=sqrt(6)c、b=sqrt(2)c、c=1/4L、d=1/2L，其中L为晶体的棱长，一般情况下，L约等于1.21×10^(-10)m。

第二步，将所有四个平行四边形连接成一个六角形，然后将每个平行四边形中长边的中点，标记为H。这时，每一个平行四边形内都有一条类似于H形的顶角，同时，在每个顶角处都有一个半径为r的圆圈。

第三步，在每一个圆圈外侧，均有两根粗的磁铁，每个磁铁上绕有一个磁钢圈。

第四步，在六角形的顶角处，按照平行四边形六个顶角的顺序，填上不同的金属原子。由于diamond晶体的结构十分复杂，所以可以通过各种方法求解。
