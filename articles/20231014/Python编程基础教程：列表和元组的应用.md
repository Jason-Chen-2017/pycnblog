# 1.背景介绍


Python作为一种高级语言，在处理数据的同时支持多种数据结构、函数式编程等特性，是一个非常灵活的编程语言。Python对数据的存储和管理提供了很好的抽象机制，允许用户灵活地选择各种数据结构进行存储。本文将向大家介绍两种最基本的数据结构——列表和元组。
## 列表List
列表（list）是 Python 中最常用的基本数据类型，它可以存储多个元素，并且可以随时添加、删除或修改元素。列表中的元素可以是任意类型的对象，甚至可以包含其他列表或者字典。列表是用方括号 [] 表示的。
例如，以下示例代码创建了一个包含字符串、整数和浮点数的列表：

```python
my_list = ['hello', 123, 3.14]
print(my_list)
```

输出结果为：

```
['hello', 123, 3.14]
```

列表可以包含任意数量的元素，并可通过索引访问其中的元素：

```python
>>> my_list[0]   # 获取第一个元素
'hello'

>>> my_list[-1]   # 获取最后一个元素
3.14

>>> my_list[1:3]   # 获取第二个到第三个元素
[123, 3.14]
```

列表的长度可以使用 `len()` 函数获取：

```python
>>> len(my_list)
3
```

还可以通过 `append()` 方法向列表中添加元素，也可以通过 `+` 操作符连接两个列表：

```python
>>> my_list.append('world')   # 在列表末尾追加元素
>>> print(my_list)
['hello', 123, 3.14, 'world']

>>> new_list = [2, 3, 4] + [5, 6]   # 通过 '+' 运算符连接两个列表
>>> print(new_list)
[2, 3, 4, 5, 6]
```

## 元组Tuple
元组（tuple）是另一种不可变序列数据类型。它与列表类似，但是一旦初始化后就不能再修改。元组是用圆括号 () 表示的。
与列表一样，元组也是可以存储不同类型元素的序列。唯一的区别就是元组由圆括号 () 来表示，而列表则由方括号 [] 来表示。
例如，以下示例代码创建一个包含三个整数的元组：

```python
my_tuple = (1, 2, 3)
print(my_tuple)
```

输出结果为：

```
(1, 2, 3)
```

元组中的元素不能被修改，所以无法添加或删除元素。只能读取元素，而且可以按照索引访问。

元组的主要功能包括传递参数、返回值、检查数据完整性等。虽然元组是不可变的，但如果其中包含不可变对象的引用，依然可能引发一些问题。
