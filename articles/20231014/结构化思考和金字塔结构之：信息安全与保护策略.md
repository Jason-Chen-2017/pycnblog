# 1.背景介绍


## 一、定义
信息安全（InfoSec）或安全信息技术（Security Information Technology），是指保障数据和信息安全、防止信息泄露、发现和防范恶意攻击、保障网络和系统安全等技术手段和规范的运用。信息安全应由内部管理体制、外部保险机制和关键技术三者共同保障。在计算机网络环境下，信息安全可以防止数据泄露、身份识别、访问控制、数据完整性、通信隐私保护、通信安全、电子威胁检测、网络病毒感染检测、系统漏洞检测、入侵监测、反恐、数据恢复、违规处理、法律法规监管、行业标准的应用、社会工程学和价值观的引导等方面的问题。

# 2.核心概念与联系
## 一、互联网架构
互联网是由网络、服务器、浏览器、数据库、应用程序和其他用户等元素组成的一种基于网络技术的分布式系统，是一种开放的、全球性的平台，任何人都可利用该系统进行数据共享和协作。互联网架构包括两大部分：
- **网络层**主要包括Internet网络、因特网路由器、网络地址转换设备（Network Address Translation Devices）、互联网连接设备和骨干网。互联网本身就是一张庞大的网络，它把所有用户的数据、文件和应用通过边缘设备传递到互联网上。网络层负责将不同网络连接起来，让它们之间互相通讯、共享资源、传递数据。
- **应用层**是实现互联网应用的基础设施，是互联网最复杂的部分。它包括各种客户端、网页浏览器、电子邮件客户端、即时通讯工具、搜索引擎、游戏客户端、移动应用、微博客等。这些应用需要与网络资源交互，并获取必要的信息。应用层一般不直接接触底层网络，而是在中间加了一层应用层协议。比如，HTTP协议用于实现互联网上的超文本传输；FTP协议用于实现文件传输；SSH协议用于远程登录；SMTP协议用于发送邮件；POP3协议用于接收邮件。
## 二、攻击类型
### （1）垃圾邮件
垃圾邮件是一种常见的信息骚扰电子邮件，其中包含恶意链接、广告宣传、病毒或者其他恶意软件。收到垃圾邮件可能会受到多种影响，比如丢失重要的业务信息、打击商誉、欺诈骗取利益等。因此，企业要保护自己免受垃圾邮件的侵害，有必要设置有效的反垃圾系统来筛查、隔离并拦截垃圾邮件。
### （2）网络钓鱼
网络钓鱼是利用虚假网站诱骗用户输入个人信息，并在用户不知情的情况下转移其个人信息到他人伪造账户上的过程。网络钓鱼通常以欺诈的方式诱骗用户提交敏感信息，如银行卡号、密码等。攻击者会通过各种手段，如向用户发送勒索软件、垃圾短信、诱使用户分享个人信息、引诱用户下载恶意软件、引导用户点击链接等方式进行攻击。
### （3）拒绝服务攻击
拒绝服务攻击（DoS attack）是一种尝试耗尽网络资源或拖垮网络正常运行的攻击。攻击者通过在受害者的计算机上实施洪水般的网络流量，占满带宽、CPU 和内存，导致网络瘫痪甚至瘫痪整个目标服务器。
### （4）SQL注入
SQL注入（SQL injection）是指攻击者通过在web表单中插入恶意SQL指令，将自己的私密信息或者权限提升，从而获得对数据库的访问权限，篡改、删除或添加数据，最终达到攻击目的。常见的攻击方式如下：
- 恶意的字符（空格、单引号、双引号等）
- 延伸攻击
- 注释符
- SQL语句
- 条件语句
- 存储过程调用
- 混淆函数
