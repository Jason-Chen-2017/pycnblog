# 2.核心概念与联系
## 变量(Variable)
变量是存储数据的地方，它表示一个实体或值的标识符。在编程中，可以使用各种数据类型存储不同种类的数据，如整数、浮点数、字符、布尔值、字符串等。而变量就是这些数据的代表。
## 数据类型(Data type)
数据类型是指变量所存储数据的分类、规格和单位。不同的数据类型决定了变量能够保存的数据大小、范围、精度、进制等属性，且具有不同的操作性质。常用的数据类型有以下几种：

1. 整型（integer）：整数类型用于存储整数值，如短整型Short、整型Int、长整型Long。它的取值范围比单字节的char类型更大，所以能表示更大的数字。常用的整数类型有byte、short、int、long。
```java
// byte类型示例
byte num = 12; // -128 ~ +127
// short类型示例
short num2 = 32767; // -32768 ~ +32767
// int类型示例
int num3 = 2147483647; // -2147483648 ~ +2147483647
// long类型示例
long num4 = 9223372036854775807L; // -9223372036854775808 ~ +9223372036854775807
```
2. 浮点型（floating point）：浮点类型用于存储小数，如单精度float、双精度double。它的取值范围比单字节的float类型更大，所以能表示更高精度的小数。常用的浮点类型有float、double。
```java
// float类型示例
float fnum1 = 3.14F; // 3.14
// double类型示例
double fnum2 = 3.14D; // 3.14
```
3. 字符型（character）：字符类型用于存储单个字符或符号，如char。字符类型通常采用ASCII码的方式表示。
```java
char c = 'A'; // ASCII码值是65
```
4. 布尔型（boolean）：布尔类型只有两个值：true和false，它用来存储真假信息。
```java
boolean flag = true;
```
5. 字符串型（string）：字符串类型用于存储文本数据，如String。
```java
String str = "Hello World";
```
## 操作符与表达式
运算符是一种特殊的符号，用来执行某些操作。比如加减乘除、关系运算符、逻辑运算符等。运算符有优先级和结合性之分，确保运算顺序的准确性。运算符可出现在表达式左边或右边，也可以出现在任何位置。在Java语言中，除了赋值运算符(=)，其他操作符都可以作为表达式出现。
### 算术运算符
- `+` ：加法运算符。如 a=b+c 表示将c与b相加，并将结果赋给变量a。
- `-` : 减法运算符。如 a=b-c 表示将c从b中减去，并将结果赋给变量a。
- `*` : 乘法运算符。如 a=b*c 表示将b与c相乘，并将结果赋给变量a。
- `/` : 除法运算符。如 a=b/c 表示将b除以c，并将结果赋给变量a。
- `%` : 模ulo运算符。如 a=b%c 表示求b对c的模（余数），并将结果赋给变量a。
- `++` : 自增运算符。变量的值加1，返回新的值并赋给变量。如 a=b++ ，则b的值加1，再将新值赋给变量a。
- `--` : 自减运算符。变量的值减1，返回新的值并赋给变量。如 a=b-- ，则b的值减1，再将新值赋给变量a。

### 关系运算符
- `>` : 大于运算符。如 (a>b) 判断a是否大于b。
- `<` : 小于运算符。如 (a<b) 判断a是否小于b。
- `>=` : 大于等于运算符。如 (a>=b) 判断a是否大于等于b。
- `<=` : 小于等于运算符。如 (a<=b) 判断a是否小于等于b。
- `==` : 等于运算符。如 (a==b) 判断a是否等于b。
- `!=` : 不等于运算符。如 (a!=b) 判断a是否不等于b。

### 逻辑运算符
- `&` : 按位与运算符。如 (a&b) 将a和b进行按位与运算。
- `|` : 按位或运算符。如 (a|b) 将a和b进行按位或运算。
- `^` : 异或运算符。如 (a^b) 将a和b进行异或运算。
- `!` : 非运算符。如 (!a) 对a进行逻辑取反。
- `&&` : 短路与运算符。如 (a && b) 只要a为true，才判断b。
- `||` : 短路或运算符。如 (a || b) 只要a为false，才判断b。

### 条件运算符
条件运算符是三元运算符。语法如下：
```java
(condition)?expression1:expression2;
```
condition为真时，值为expression1，为假时，值为expression2。
