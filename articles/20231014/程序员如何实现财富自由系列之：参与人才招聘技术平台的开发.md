# 1.背景介绍


当下大多数企业都在招聘技术人员、产品经理等高端职位。然而，不少公司由于没有相关技能，而难以招到优秀的人才。因此，为了提升自身能力和提高公司竞争力，越来越多的企业开始将招聘过程中的人才需求跟进到技术团队中。于是，招聘人员的参与就成为公司对人才的最大需求。近年来，随着国际化、城市化的影响，招聘市场也在迅速扩大。
招聘技术人员作为一项快速发展的行业，每天都有大量新生代员工加入。而整个招聘过程都需要极高的效率才能吸引到有潜力的应聘者。但是，由于招聘过程复杂且费时，技术要求并非所有人都能够满足。同时，由于招聘市场遍布全球，公司为了在各个市场进行竞争，往往会选择不同的语言招聘，因此招聘平台要具备较强的国际化能力和灵活性。此外，很多公司都希望通过招聘平台精准匹配候选人才，从而更好的发掘其优势资源。基于这些需要，我国已建成众多具有先进招聘技术平台功能的公司，如：Boss直聘、拉钩网、智联招聘、前程无忧等。但由于各大招聘平台的功能繁杂，用户门槛高，技术门槛低，导致企业无法将技术资源投入到最佳使用，从而造成了较大的经济损失。
正因为如此，很多企业在寻找技术合作伙伴时，都放弃了传统的方式，转向社交网络、微信群、微博、论坛、贴吧、知乎等获取信息。由于这些渠道的信息来源广泛，价格便宜，速度快，容易形成信息茧房效应，最终导致企业面临的信息过载、筛选困难、信息质量差的问题。基于这些情况，有必要构建一个专门面向招聘技术人员提供的互动交流平台，让求职者们可以直接参与到招聘过程中，实现共享、共赢、共建，真正实现“拼爹”的美好愿景。这样一来，企业只需付出较小的个人成本即可获得大量人才的帮助，更有效地提升招聘效果，降低经济成本。
# 2.核心概念与联系
## 招聘平台定义
招聘平台是一个利用计算机技术与网络技术实现信息化管理，并通过一定方式将人才供需双方连接起来的综合性信息交流工具。它由一系列的服务模块组成，主要包括发布招聘信息、搜索招聘信息、接收来自求职者的简历、邀请笔试、面试、聘用确认等功能。通过实现以上功能，可以让求职者、职业者之间信息更加及时、准确、流畅地共享。平台通常分为网站、App、微信小程序、智能硬件等形式。
## 消歧义词词典
为了方便大家理解，以下给出了一个消歧义词词典，方便大家查阅。这里给出的词语列表仅供参考，后续文章可能还会涉及其他新的名词或术语，具体使用时，请根据上下文及具体需求具体确定。
招聘信息：指招聘活动中所有应聘者的基本信息，包含姓名、性别、年龄、工作年限、期望薪酬、学历、经验、技能、邮箱、手机号码等信息。
职位描述：即招聘信息下方的职位详情。
自荐信：表示求职者主动提交的推荐函。
内推机会：表示雇主针对特定候选人的面试邀请。
职位邀请函：类似于招聘信息，也需要填写招聘岗位和职责要求等详细信息。
简历：候选人上传至平台的个人履历，包含个人基本信息、工作经验、教育背景、项目经验、科研成果、获奖情况、推荐信等内容。
笔试题：面试之前，平台会随机发布一些编程、设计、业务等相关的笔试题目。
面试官：由平台推荐的一线面试官或负责审核评价的第三方中介机构。
面试结果：职位面试官通过面试时给出的答卷。
通知书：从平台发出邀请函的确认函。
面试问卷：面试官用于确认面试者个人信息和面试条件的问卷。
一轮面试：指一次完整的面试流程，包含笔试、编程题目、面试环节等。
二轮面试：指次级面试，通常采用视频、电话或者远程面试方式。
电话面试：电话面试方式是指面试官打电话到求职者的工作单位或住处，面试者根据自己的陈述回答问题，也可以向面试官提问。
视频面试：视频面试是在现场或者在线视频会议上完成的。
远程面试：远程面试是一种面试方式，面试官、被面试者均需使用网络进行通信，远程协助完成面试。
## 信息流向与合作机制
当求职者发布招聘信息时，平台首先会将信息分享给平台内的其他用户，如同他人分享信息一样，其他用户可以通过这些信息了解招聘信息的内容。之后，平台会整理这些信息并通过适当的手段进行排序，使得求职者更容易找到感兴趣的职位。求职者与职位相关的相关人员之间的联系也非常紧密。当求职者看到相关的人物时，就会积极报名参加这次的活动，之后参与这次活动的相关人员则可进行密切的沟通。如果成功与某个人建立关系，他们就可以通过这个关系得到职位的更多信息。当求职者有意向的情况下，会发送自己简历，平台会对其进行初步筛选。
招聘流程：求职者发布招聘信息后，通过适当的方式，会受到平台内的反馈。职位的申请者可能会收到一封招聘邀请函。简历通过平台后，会送到相应的职位相关人员进行面试。三方的面试官会对简历进行筛查，并在指定的时间内进行面试。职位的相关人员收到面试通知后，会给予面试官针对简历的建议，并与求职者沟通面试经验。通过对面试官和求职者的面试，双方可以形成满意的合作关系，为合作签署协议或委托代理。合作协议一般会规定双方的权利和义务，如分享个人信息、违约惩罚、保密协议等。
招聘模式：目前国内招聘技术人员使用的主要模式有：
- 站内招聘：就是在招聘网站上直接发布招聘信息，这种方式用户比较简单，可以根据招聘信息的相关度进行排名，但缺点也很明显，只能限制用户在平台上的时间，而且被筛掉的概率也比较高。
- 搜索推荐：就是通过关键词、标签、分类来进行搜索，然后给予热度排名，满足用户的需求。这种方式也是现在很火热的一种方式，虽然可以把控用户的搜索范围，但同时也存在严重的链接嫌疑。
- 社交招聘：通过平台上的群、圈子、论坛等，结识和相互了解，建立深度的社交关系，可以更好的帮助求职者解决相关问题。这种方式是现在国内比较成熟的招聘方式。
- 深度合作招聘：就是企业通过集成多个招聘平台，实现招聘信息的自动同步、统一展示，从而提高平台的发现率、群体黏性，最大程度的实现招聘信息的有效传递。通过这种方式，企业可以充分发挥招聘信息的优势，同时也避免平台的不良影响。
总结来说，招聘平台可以满足求职者的各种需求，帮助企业搭建自己的招聘信息发布和招聘交流平台。不同类型的平台存在不同的招聘模式，可以按照企业的实际情况，结合自身的招聘策略进行选择。
