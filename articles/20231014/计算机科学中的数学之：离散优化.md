# 1.背景介绍


在实际应用中，有很多需要求解的问题都可以抽象成一些数学模型，例如求最大流、最小费用流等问题。由于这些问题是NP完全问题（即不存在一个有效算法可以在多项式时间内解决），因此通常只能通过启发式方法来求解。最简单的启发式方法是贪婪算法或近似算法，它们的时间复杂度较高，但是在大部分情况下得到的解往往比较接近全局最优解。另外，还有更复杂的启发式方法，如模拟退火法、蚁群算法等，它们更加贴近真实的计算硬件，但耗时也更长。
在本文中，作者将重点介绍在计算机科学中用到的离散优化问题——运筹学中的约束整数规划问题，它是一种具有广泛应用意义的优化问题。约束整数规划问题是一个具有复杂性的难题，其核心思想是通过对变量进行约束来建立目标函数。其目的是找到使目标函数达到最优值的整数解，换言之，就是在给定约束条件下，找到最佳的选择或者方案。
约束整数规划问题一般来说分为线性规划、二次规划、非线性规划三类，本文主要讨论线性规划、整数规划。
# 2.核心概念与联系
## 2.1.约束整数规划问题的定义及性质
约束整数规划问题（Constraint Integer Programming Problem，CIPS）是指满足一定约束条件下的整数线性规划问题，即如下形式的最优化问题：

$$\begin{array}{ll} \mbox{minimize: }&\quad c^T x\\ \mbox{subject to:}& \\ Ax &\leq b \\ x_i &\in Z, i=1,\ldots,n \end{array}$$

其中$c=(c_1,\cdots,c_n)$ 是目标函数的系数向量，$x=(x_1,\cdots,x_n)^T$ 是决策变量向量，$A=(a_{ij})$ 是系数矩阵，$b=(b_1,\cdots,b_m)$ 是右端常数列，$Z=\{z_1,\cdots,z_k\}$ 表示决策变量的取值范围，即变量 $x_i$ 的取值必须属于 $\{z_1,\cdots,z_k\}$。此问题的求解可以通过转换得到其它形式的整数线性规划问题。

CIPS 满足以下性质：

1. 可行性：如果 CIPS 中存在可行解，则该解满足最优化问题的任意约束。

2. 强对偶性：设整数线性规划问题的最优解为 $x^\ast$ ，则 CIPS 中的最优解 $y^\ast$ 在约束上恰好满足：

   $$\forall (i,j) \in E, y_{ij}\cdot s_i + y_{ji}\cdot s_j = 1,\quad s_i,s_j \in \{0,1\},\quad i<j$$
   
   这里，$E$ 是图 $G=(V,E)$ 的边集合，$V$ 是顶点集合，$y_{ij}=1$ 表示变量 $x_i$ 和 $x_j$ 在图中相邻。
   
3. 容斥原理：CIPS 可以看做是由两类子问题组成：$K$ 子问题（称为可行子问题）和 $K$ 个互斥子集的组合问题（称为容斥子问题）。假设集合 $S_1,\cdots,S_k$ 都是不同的整数集合，且 $|S_i|=l_i$ ($1\le i\le k$) 。容斥子问题的目标是找出使得所有可行子问题不冲突的整数解集 $\cup_{i=1}^k S_i$ 。容斥原理证明了可行子问题和容斥子问题之间存在如下关系：

   $$D(I)\subseteq D(J), \text{ if and only if }\forall j\notin J:\exists i\in I:(D(I)-D(J))\cap\{s_i\}=0$$

   式中 $I,J,$ 和 $D(\cdot)$ 分别表示可行子问题、容斥子问题、依赖图的一组变量。
   
4. 路径依赖性：CIPS 有着路径依赖性，即对于任何整数解 $(x_1^{\ast},\cdots,x_n^{\ast})\neq (0,\cdots,0)$ ，存在至少一条割线从 $(x_1^{\ast},\cdots,x_n^{\ast})$ 到 $(-1,-1)$ 。换句话说，即使是非常优秀的整数解，也可能存在其他可行解使得目标函数变小。
