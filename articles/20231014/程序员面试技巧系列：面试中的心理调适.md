# 1.背景介绍


什么是心理调适？
心理调适是指在面试过程中，应聘者和面试官都需要以自我为中心，增强自信心、坚定决心、找到自己的兴趣点和方向，以更高效的方式参加面试。而面试中的心理调适又往往带来一定的压力，因为我们很容易被迫做出一些抉择，比如说放弃或延期某些准备工作等。因此，面试前后及时调整自己的心态和精神状态非常重要。
那么，面试中出现心理调适现象，是否代表着我们的能力低下呢？如果真的是这样的话，怎么才能提升自己呢？这些都是值得探讨的问题。
作为技术人员，我们一定会遇到很多“黑天鹅”事件——突如其来的信息 overload、重重困难险阻、无法预料的突发状况。作为一名优秀的技术人才，如何在面对这些障碍时，做好心理上的准备，成为一个更好的自己？下面就让我们一起来谈谈面试中的心理调适吧！
# 2.核心概念与联系
首先，我们先了解一下一些基本概念。

2.1 自我认知
自我认知就是判断自己到底是不是真正想要的东西，并且有能力去解决它。这需要的是开阔眼界、识别机会、客观分析、主动沟通、理解能力等方面的综合能力。

2.2 行为习惯
行为习惯是指人们在特定环境下自动形成的一种反复无常的行动方式。这包括个人爱好、性格特征、职业经历、生活方式等等。不同人的人生模式也不同，有的喜欢追求刺激、有的喜欢规划、有的偏好计划，总之，人的行为习惯千差万别。要学会克服它们，才能实现长久的成功。

2.3 刻意练习
刻意练习其实就是借助一定的训练方法，将我们擅长的技能、知识培养到最大限度。只有不断的刻意练习才能使我们进步，并掌握领域的最佳实践。

2.4 沟通技巧
沟通技巧可以分为以下几类：
  - 情感表达
  - 语言表达
  - 团队协作
  - 协商能力
学习沟通技巧可以帮助我们更有效地跟面试官进行沟通，掌握一套积极向上、包容性强、富有说服力的沟通技巧，能够更轻松地与他人交流沟通。

2.5 任务分配
任务分配是指根据工作需求、技能特质、时间紧迫性等因素，将不同的事情分派给不同的成员。任务分配的目的就是让每个人在尽可能短的时间内完成任务，从而确保项目按时交付，达成共识。

根据以上概念，我们一起看一下面试中常见的心理调适现象。

## 3.1 减少焦虑的八个技巧

1. 提前安排好面试
首先，提前将考试时间安排好，不要让面试变成一场抢时间的比赛。

2. 不要过多的询问猜测
不要因为疑惑或者面试官的提问而产生烦恼，应该主动推动面试官向你提供具体信息，避免陷入猜测和推敲当中。

3. 以身作则
不要冒充任何人，一定要回归自己的实际情况。如果你觉得自己的能力不足，不要慌张，主动承担责任。

4. 把握面试的节奏
在面试过程中保持冷静、镇定，不要想太多，试图用一句话就结束面试。

5. 不要担心不被录取
即使失去了这次的面试机会，也不要气馁，坚持继续努力。

6. 鼓起勇气接受挑战
遇到不熟悉的工作、业务、产品时，不要怕失败，因为这种挑战不仅激发了你的潜能，而且还可以锻炼你的沟通、协作和团队建设能力。

7. 制造团队氛围
一定要组织好团队，营造出像样的工作环境，吸引着志同道合的伙伴加入团队。

8. 学会运用直觉
试着去做一些小事，不要把注意力完全放在面试上面。

## 3.2 心理暗示的七种方法

1. 声音大
对面试官说话时，不要过于高声喧哗，尤其不要在上课时或者交集时喧哗。

2. 悬崖勒马
面试官可能会嘲笑你的才华，你需要学会用非暴力手段维护自己的权威。

3. 提问之前先阅读材料
为了得到更好的答案，建议在听完面试官对你所了解的情况之后，再花一定的时间阅读相关的材料，确保你能够全面客观地回答面试官提出的问题。

4. 记住隐喻
记忆某个词时，不要用文字来记忆，而是换一种方式让它隐约浮现出来。例如，当你看到面试官手里拿着一块红布时，你脑海中会出现类似游戏《捉迷藏》中的隐喻：英雄需要通过拨开红布进入迷宫，找到一条出口并逃离。

5. 用身体语言做出表情
面试官可以通过你的表情、动作来传递消息，让你看到你有说话的权利。

6. 当面试官问到什么时候开始准备的时候，你可以假装进入考试状态，并提前十分钟出来。

7. 每次面试结束，要记得在合适的位置放下电话，不要光背着背包坐在座位上，这样可能会影响自控力。

## 3.3 团队合作的六个心理因素

1. 相互尊重
面试过程中不能随意指手划脚，尤其不要嘲弄对方。一定要做到平等互利，交流与反馈能够有效地促进彼此的成长。

2. 坦诚相待
对方的言语越清晰越好，不要过分夸大自己的优势。当你的优势被揭露时，可以尝试指出对方的弱点，提升团队凝聚力。

3. 建立信任
始终保持良好的信任关系，和对方分享你对于公司、部门、职位、个人的价值观、理念和目标。尤其是在困难阶段，相互鼓励，能够让你更坚定信心，对接下来的进展产生积极作用。

4. 明辨上下级关系
面试官可能会关心你属于哪一层，请不要露出疏离的神色，要为团队成员之间建立清晰的界限。当你明白自己的位置时，就会做到平衡工作和生活。

5. 共赢共存
在面试过程中，不要只顾眼前，一定要站在整个团队的角度考虑问题，团结一致同心。不管遇到什么困难，都要保持乐观主义，持续不懈向前迈进。

6. 热忱配合
面试官的每个评价都很重要，但是不要过度评估他人的能力，要关注他们的工作态度、沟通表达能力和团队合作意愿，共同提升整体品牌形象。

## 3.4 问题解决时的四种心理机制

1. 创造性思维
面试中遇到困难，一定要运用创造性思维。每当你遇到一个问题时，需要考虑三个问题：为什么会这样？怎么办？有没有其他的方法？如果有其他的方法，你会采用哪一种？

2. 自我评估
面试官会针对你提出的问题进行评估，目的是为了让你知道自己还需要改善哪些方面，并且让你自己知道该以怎样的姿态回答。这也是评估自己的一个过程。

3. 计划与执行
面试前要有一套详细的计划，并且制定时间表，在必要的时候适时跳槽。不要急于求成，先找到一个满意的工作，然后再考虑扩大规模。

4. 保留自信
面试结束后不要低头，要相信自己是可靠的，不会出现错误的结论，你还有很大的发展空间。
