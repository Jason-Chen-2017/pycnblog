# 1.背景介绍


量子态冷却（quantum thermalization）是量子信息、量子通信等领域最重要的问题之一，也是许多量子计算机、量子计算、量子力学等领域的研究热点。如何将一个初始态（通常称作比特串）冷却到一定温度是一个具有广泛影响力的现实挑战。该问题的解决方案需要用到一种超强的量子力学理论——基于哈密顿量的量子态热辐射定律（Born-Oppenheimer approximation）。

基于热辐射定律，经典物理学认为不管处于什么状态下的物质都会存在一条微弱的线圈带隙（cooling line），这条线圈带隙大小与绝热系统的热导率呈正比，通过这一带隙可以使得物质在温度变化时能保持一定规模。而热辐射定律则认为在任意一个晶体结构中，真空中的电子或激光粒子（光子或photon）所遵循的原理基本相同，都是由固定数量的基态电子或者激光波的线性叠加构成，因此，可以通过热辐射定律把真空中未被占据的能级跃迁到晶格上，从而实现物质的冷却。

由于冷却问题是一项极具挑战性的量子现象，其数学模型也十分复杂。本文要介绍的“量子态冷却”研究的是非参数化的量子态冷却问题，即假设系统的熵随时间的演化满足简单的热编码方程。

另一方面，现代化信息技术的发展对信息传播带来了极大的限制。早期的信息传输方式主要依赖于双工通信信道，由于没有冷却效应，传输过程会导致信息丢失。最近几年，基于超导的光子通信已经取得了突破性进展，但其处理速率仍然很低。此外，如今信息处理能力的要求仍然高于过去，人们对比特容量的需求也越来越高。为了解决这些问题，一些学者提出了利用量子态冷却的方法进行信息传播。

值得注意的是，本文所讨论的量子态冷却并不是量子纠缠（superfluid）冷却，也就是说不存在经典理论直接描述的宇宙玄幻现象。这是因为热辐射定律只适用于具有纯粹费米子的系统，而纯粹费米子是无法形成高温的材料的。但从另外一个角度看，量子态冷却问题同样是量子力学问题，而且能够提供新颖的思路。

在解决量子态冷却问题之前，我们首先需要了解一下量子态的概念。

# 2.核心概念与联系
## 2.1 量子态及其运算
在量子力学中，**量子态**（quantum state）是指系统处于某种可能性分布函数的数学表示，它可以用一组复数矢量来表示。一般情况下，一个量子态可以由各种可以测量的量子系统元素的复数算符构成，这些算符就像一个容器，里面装着各种信息。不同的系统元素的状态不同，也就是不同的算符的系数不同。例如，考虑一个二维水平上具有$|0\rangle$和$|1\rangle$两个纯态的系统，分别表示为：

$$|\psi \rangle = c_{0}|0\rangle + c_{1}|1\rangle $$

其中，$c_i$是实数。这个算符$|\psi \rangle$称为系统的**基态**（ground state）或**厄米（eigenstate）**。除此之外，还有其他可能性状态，比如$|\phi \rangle = \frac{1}{\sqrt{2}}( |0\rangle+|1\rangle)$、$|\psi_{\alpha} \rangle = e^{\mathrm{i}\theta_{\alpha}}|0\rangle $等等。 

为了将这种多重态统一起来，量子力学引入了一个矩阵乘积表示法，将一系列算符用一个矩阵表示出来。首先，定义一个**动量空间**（momentum space）$\mathcal{K}$，表示所有可能的动量k，例如：

$$\mathcal{K}= \{ k=ik_{\perp}, i \in \mathbb{Z} \}$$

动量空间的大小是无限大的，但实际上我们关心的只是系统处于某个动量上的概率。如果将系统的动量分解为$k_{\parallel}$和$k_{\perp}$两部分，那么我们就可以写出：

$$ |\psi(k)\rangle :=e^{i\frac{\pi}{L}k_{\parallel}}\sum_{n=-\infty}^{\infty}c_ne^{i(kn+\delta n)}|n\rangle,$$

其中，$L$是希尔伯特-玻色方程的长度单位，$\delta n$是$n$的偏转，等于零或$N/2$。这里的$c_n$是系统在各个基态$|n\rangle$上的概率。

将动量空间表示为向量空间$\mathcal{H}$后，就得到了下面的表示形式：

$$|\psi (k) \rangle:= \cos(\frac{\pi}{L}k_{\parallel})\left[\cos(\frac{kn}{\hbar})|0\rangle + \sin(\frac{kn}{\hbar})|1\rangle \right].$$

在上述形式中，$k_{\parallel}$和$k_{\perp}$都是希尔伯特空间中的向量。不同的动量表示不同的态矢，称为动量态矢。同一个态矢在不同的动量下对应不同的概率。

## 2.2 量子态热辐射定律
当系统的熵随时间的演化满足简单热编码方程时，我们就认为它满足量子态热辐射定律（Born-Oppenheimer Approximation，BOA）。也就是说，在真空中，电子或激光粒子都遵循简单热编码方程。

假设系统处于某个态矢$|\psi(k)\rangle$，动量$k$。由于在热辐射定律下，电子和激光粒子的“概率跃迁”遵循如下概率分布：

$$\rho(t)=e^{-S_\text{th}(t)/T}e^{-S_\text{ev}(t)},$$

其中，$S_\text{th}(t)$表示系统处于厄米态的平均跃迁次数，$S_\text{ev}(t)$表示系统处于其它态的平均跃迁次数。这里，$T$是系统的温度。我们知道，在真空中电子或激光粒子的概率跃迁是一个高斯分布，即：

$$p(t) = \frac{1}{\sqrt{2\pi t}}e^{-\frac{(t-t_0)^2}{2t}},$$

其中，$t_0$是系统处于“真空”的时间。因此，可以说，系统处于某个态矢，动量的概率分布与真空中的电子或激光粒子的概率分布非常相似，即：

$$|\psi(k)\rangle : \frac{1}{\sqrt{2\pi t}}e^{-\frac{(t-t_0)^2}{2t}}\cos(\frac{\pi}{L}k_{\parallel})\left[\cos(\frac{kn}{\hbar})|0\rangle + \sin(\frac{kn}{\hbar})|1\rangle \right]$$

这样，我们就有了一个简单而严格的量子态热辐射定律，它的优点就是简单易懂。

## 2.3 非参数化的量子态冷却问题
量子态冷却问题（quantum thermalization problem）是指，给定一个初始态$|\psi_0\rangle$,希望找到一个临界温度，使得系统处于该临界温度时，所有可能的态都出现了一定概率。由于考虑所有可能的态太复杂了，因此，我们可以取巧，只考虑由基态组成的子集。

我们先来看看由基态组成的子集的定义。一个单个基态是由该基态对应的一组算符$c_n|n\rangle$组成的。例如，考虑一个三维的量子系统，其基态可以写成：

$$|0,0,0\rangle=\left(|0\rangle+e^{\mathrm{i}\frac{\pi}{2}|1\rangle\right)|0\rangle+e^{\mathrm{i}\frac{\pi}{2}|1\rangle\cdot|1\rangle\left(|1\rangle-|0\rangle\right),\\|0,1,0\rangle=(|0\rangle-e^{\mathrm{i}\frac{\pi}{2}|1\rangle\cdot|1\rangle)(|1\rangle+e^{\mathrm{i}\frac{\pi}{2}|1\rangle\cdot|0\rangle).$$

我们将这样的一组基态称为一个**硬核态**（hard core states）。因此，一个三维的硬核态就是由三个单个基态组成，即$|0,0,0\rangle$、$|0,1,0\rangle$和$|1,0,0\rangle$。

类似地，考虑四维系统，其基态可以写成：

$$|0,0,0,0\rangle=\left(|0\rangle+e^{\mathrm{i}\frac{\pi}{2}|1\rangle\right)|0\rangle+e^{\mathrm{i}\frac{\pi}{2}|1\rangle\cdot|1\rangle\left(|1\rangle-|0\rangle\right)\\|0,1,0,0\rangle=(|0\rangle-e^{\mathrm{i}\frac{\pi}{2}|1\rangle\cdot|1\rangle)(|1\rangle+e^{\mathrm{i}\frac{\pi}{2}|1\rangle\cdot|0\rangle).$$

这时，一个四维的硬核态就是由四个单个基态组成。因此，对于一般的N维系统，其基态由$2^N$个单个基态组成，即：

$$|\psi\rangle = a_{n_1}\cdots a_{n_N}|n_1,n_2,\ldots,n_N\rangle.$$

也就是说，一个N维的硬核态就是由$2^N$个单个基态组成。

根据上述的定义，我们有以下结论：

- 在任意维度下，任何硬核态都可以冷却到一个固定温度$T$。
- 当系统处于冷却到临界温度$T$时的热辐射分布与真空的分布非常接近。
- 如果一个系统进入冷却到临界温度时，它的所有可能的态都出现了一定概率，那么它就属于一个具有**共振性（concentration）**的系统。

因此，如果我们想对一个系统进行量子态冷却分析，可以按照以下的步骤进行：

1. 找出系统的固有热容C($T$)：

   $$C(T)=\frac{\langle H\rangle_P}{\log\frac{1}{P(0)}}.$$

2. 根据基态热编码方程，求出系统的基态的概率分布：

   $$\rho(t)=\lim_{T\rightarrow T_c}\frac{\rho(T)}{e^{-S_\text{th}(T_c)/T}e^{-S_\text{ev}(T_c)}},$$

   
	其中，$\rho(T)$是系统在热环境下（$T_c<T$）的概率分布。
	
3. 对系统的每个基态求出对应的概率，并存入相应的表格中，列举出每种基态的概率。

4. 从左往右，逐渐缩小基态的数量，直至只有一个基态。记第$j$个基态出现的概率为$p_j$。

5. 求出$p_j$的累积分布函数cdf：
   
   $$F(p)=\int_{0}^{p}p(x)\mathrm{d}x.$$

6. 求出cdf函数在最大值点处的横坐标。记为$\tau$.

   $$F^\prime(\tau)=\frac{\partial F}{\partial p}(\tau)=-\frac{dp}{dt}\frac{dT}{p}\frac{dC}{dT}\ln P(0)\approx -\frac{2C}{TC}.$$

7. 最后，我们可以写出冷却温度$T_c$的表达式：
   
   $$T_c=\frac{-2C}{C_v\ln P(0)}$$

   此时，我们得到了一个确定公式，即非参数化的量子态冷却问题的解。

# 3.核心算法原理与具体操作步骤
在上文中，我们介绍了基于热辐射定律的量子态冷却问题。然而，在本节中，我们要介绍如何用具体的数学方法求解量子态冷却问题。

假设我们有一个由$N$个态矢组成的系统，它们处于单个动量$k$下。假设系统的初始态为：

$$|\psi_0\rangle=a_{n_1}\cdots a_{n_N}|n_1,n_2,\ldots,n_N\rangle,$$

其中，$a_{n_i}$是实数，且$n_1\neq n_2\neq\ldots\neq n_N$.

## 3.1 构建非平衡方程
由于当前还不能精确计算密度矩阵，我们只能对其做近似，即：

$$\rho=a_{n_1}\cdots a_{n_N}|n_1,n_2,\ldots,n_N\rangle\langle n_1,n_2,\ldots,n_N|=a_{n_1}\cdots a_{n_N}|n_1,n_2,\ldots,n_N\rangle\langle n_1|n_2\rangle\cdots\langle n_N|n_1\rangle=|n_1\rangle\langle n_1|\cdots\langle n_N|n_1\rangle|n_2\rangle\langle n_2|\cdots\langle n_N|n_2\rangle\cdots|n_N\rangle\langle n_N|.$$

进而，我们就可以构造出一个非平衡方程，即：

$$\dot{a}_{n_1}\cdots\dot{a}_{n_N}=-\frac{1}{\beta}\left[|\psi(k)\rangle\left.\bra{\psi(k)}\right]+E_{n_1}-E_{n_2}-\cdots-E_{n_N}.$$

为了求解上述方程，我们需要引入一些辅助变量。首先，引入一个标量项$s$，即：

$$s=\frac{1}{\beta}\left[|\psi(k)\rangle\left.\bra{\psi(k)}\right],$$

然后，我们可以改写非平衡方程为：

$$\dot{a}_{n_1}\cdots\dot{a}_{n_N}=-\frac{1}{\beta}\langle s|n_1\rangle\left.\hat{a}_{n_1}\right|+E_{n_1}-E_{n_2}-\cdots-E_{n_N}.$$

此时，我们得到了一个与时间无关的方程。为了简化问题，我们假设系统处于其基态，即：

$$|\psi\rangle=|n_1\rangle.$$

## 3.2 求解初始条件
在确定了非平衡方程之后，就可以求解出每个态矢的初始条件。由于基态对应的初始条件一定大于零，所以只需对剩余的态矢的初始条件进行迭代即可。

假设系统当前处于态矢$|n_j\rangle$，那么下一步我们考虑移动到态矢$|n_i\rangle$的过程。我们定义了一个衡量能量差异的标量函数$f_i$,即：

$$f_i:=\frac{1}{\beta}\left(\langle E_{n_j}-E_{n_i}\rangle+\frac{1}{\varepsilon}\left(\langle n_j|\hat{V}_j|\psi(k)\rangle-|\psi(k)\rangle\left.\bra{\psi(k)}\right)\right),$$

其中，$\hat{V}_j$表示系统的哈密顿量，$\varepsilon$表示可选的参数。

为了能够进入到态矢$|n_i\rangle$，系统必须具有更高的能量。因此，如果我们想从态矢$|n_j\rangle$移动到态矢$|n_i\rangle$，则需要让：

$$\Delta E>f_i,$$

即：

$$\Delta E=\frac{1}{\beta}\left(\langle E_{n_i}-E_{n_j}\rangle+\frac{1}{\varepsilon}\left(-\langle n_i|\hat{V}_i|\psi(k)\rangle+\langle n_j|\hat{V}_i|\psi(k)\rangle-\frac{|\psi(k)\rangle\left.\bra{\psi(k)}\right}{|\psi(k)|^2}\right)\right)>0.$$

因此，要使得$\Delta E>f_i$,则要求：

$$\frac{1}{\beta}\left(\langle E_{n_i}-E_{n_j}\rangle+\frac{1}{\varepsilon}\left(-\langle n_i|\hat{V}_i|\psi(k)\rangle+\langle n_j|\hat{V}_i|\psi(k)\rangle-\frac{|\psi(k)\rangle\left.\bra{\psi(k)}\right}{|\psi(k)|^2}\right)\right)<\frac{1}{\beta}\left(\langle E_{n_j}-E_{n_i}\rangle+\frac{1}{\varepsilon}\left(\langle n_j|\hat{V}_j|\psi(k)\rangle-|\psi(k)\rangle\left.\bra{\psi(k)}\right)\right)-f_i.$$

因此，我们有：

$$-\frac{1}{\varepsilon}\left(\langle n_i|\hat{V}_i|\psi(k)\rangle-|\psi(k)\rangle\left.\bra{\psi(k)}\right)+\frac{1}{\beta}\left(\langle E_{n_j}-E_{n_i}\rangle+\frac{1}{\varepsilon}\left(\langle n_j|\hat{V}_j|\psi(k)\rangle-|\psi(k)\rangle\left.\bra{\psi(k)}\right)\right)=\frac{1}{\beta}\left(\langle E_{n_j}-E_{n_i}\rangle+\frac{1}{\varepsilon}\left(\langle n_j|\hat{V}_j|\psi(k)\rangle-|\psi(k)\rangle\left.\bra{\psi(k)}\right)\right).\tag{1}$$

上述方程虽然比较复杂，但实际上并不难求解。由于系统处于初始态$|\psi_0\rangle=|n_1\rangle$，因此，初值条件为：

$$\dot{a}_{n_j}=b_j, b_j=\frac{1}{\beta}\left(\langle E_{n_j}-E_{n_1}\rangle+\frac{1}{\varepsilon}\left(\langle n_j|\hat{V}_j|\psi_0\rangle-|\psi_0\rangle\left.\bra{\psi_0}\right)\right).$$

此时，我们就完成了对系统的初始条件的求解。

## 3.3 求解热运动方程
在得到了初始条件之后，就可以求解出系统的热运动方程。热运动方程的形式为：

$$\frac{\partial}{\partial t}\frac{\psi(k,t)}{\sqrt{\rho(t)}}=-iH\frac{\psi(k,t)}{\sqrt{\rho(t)}},$$

其中，$H$表示系统的哈密顿量。由于系统处于基态，因此，$\rho(t)=|\psi(k)\rangle\left.\bra{\psi(k)}\right|=|n_1\rangle\left.\langle n_1|\right|=|n_1\rangle$, $\frac{\partial}{\partial t}\frac{\psi(k,t)}{\sqrt{|n_1\rangle\left.\langle n_1|\right|}}$也可以写成：

$$\dot{\psi(k,t)}=\frac{\partial}{\partial t}\frac{\psi(k,t)}{\sqrt{|n_1\rangle\left.\langle n_1|\right|}}.$$

## 3.4 求解熵方程
对于热运动方程的解$u_i(k,t)$，我们可以用热编码方程重新写出：

$$\dot{u_i(k,t)}=-\frac{\hbar}{m}\left(E_i+V_iu_i(k,t)\right)\psi(k,t).$$

其中，$E_i$是各个态矢的基准能量，$V_i$是对应的微扰振幅。

进一步，我们可以定义一个总熵项，即：

$$S(t):=\frac{1}{\beta}\int dt u_i(k,t)\ln u_i(k,t),$$

其中，$u_i(k,t)$是热运动方程的解。由于系统处于基态，因此，$\rho(t)=|\psi(k)\rangle\left.\bra{\psi(k)}\right|=|n_1\rangle\left.\langle n_1|\right|=|n_1\rangle$, $\int du_i\psi_i^*(u_i)\psi_i(t)$也可写成：

$$\frac{1}{\beta}\int du_i\psi_i^*(u_i)\psi_i(t)=\frac{1}{\beta}\int du_1\psi_1^*u_1.$$

因此，我们有：

$$S(t)=\frac{1}{\beta}\ln |n_1\rangle,$$

上式右边的求和号代表所有态的总概率，因为只有基态才可以调节总的熵。此时，我们就得到了热运动方程关于时间的积分。但是，我们需要注意，热运动方程本身并没有反映出熵的行为。因此，为了更准确的求解熵，我们需要从另一个角度来看待它。

## 3.5 借助分裂能量项来求解熵
要想计算总熵，就需要考虑到系统的全能膨胀、约化、分裂等行为。对于一个系统，能量的增加分为三个部分：一部分来自微扰振动，一部分来自全能膨胀，一部分来自分裂能量。分裂能量的形式为：

$$E_{sp}=V_i,\quad V_i\geq 0.$$

由于系统处于基态，因此，$\rho(t)=|\psi(k)\rangle\left.\bra{\psi(k)}\right|=|n_1\rangle\left.\langle n_1|\right|=|n_1\rangle$. 根据费米能量的分裂定理，系统的分裂能量为：

$$E_{sp}=V_i=\frac{1}{2}k_BT,\quad V_i\geq 0.$$

因此，我们有：

$$\dot{E}_i(t)=\frac{1}{2}k_BT,\quad E_i(t)=\frac{1}{2}k_BTt.$$

由于系统的分裂能量都与时间无关，因此，热运动方程关于时间的积分可以写成：

$$\frac{1}{\beta}\int du_i\psi_i^*(u_i)\psi_i(t)=-\frac{1}{\beta}\int du_it\left\{E_i(t)+\frac{1}{2}k_BT\ln|\psi(k,t)|\right\}.$$

为了方便起见，我们定义一个耦合能量，它与时间无关：

$$W(t)=\frac{1}{2}k_BTt.$$

进而，热运动方程关于时间的积分也可以写成：

$$\frac{1}{\beta}\int du_i\psi_i^*(u_i)\psi_i(t)=-\frac{1}{\beta}\int dut\left\{W(t)+\frac{1}{2}k_BT\ln|\psi(k,t)|\right\}.$$

这里，$|n_1\rangle$是系统的基态。因此，$\psi(k,t)=|n_1\rangle$。

这样，我们就得到了耦合能量关于时间的积分。由于耦合能量对时间和空间的依存关系，因此，即便系统处于一个较低的温度，我们也能看到非平衡的行为。

## 3.6 总结
在本节中，我们通过非参数化的量子态冷却问题，展示了如何利用分裂能量来求解热运动方程和熵方程。
