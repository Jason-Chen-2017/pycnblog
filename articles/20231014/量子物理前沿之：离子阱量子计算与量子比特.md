# 1.背景介绍


量子物理(Quantum Physics)是一个很神秘的领域，它涉及到很多现代科技发展史上的重大发现。但近几年，随着量子计算机和量子传感器等新型机器设备的出现，人们逐渐意识到量子信息与量子技术的巨大潜力。而量子计算在当今科技中的重要作用也越来越凸显。

量子计算通过对物质的干扰、环境的影响以及对物质性质的测量等行为，生成并利用能量。目前，由于物理世界本身存在不可测量的元素——“弦论”的观念，因此，研究者无法利用基态(Ground State)和超导性(Superconductivity)来研究量子计算的各种特性。但是，量子计算可以利用量子干涉实验(NMR、电子显微学、X射线衍射等)或模拟经典计算机的计算过程。

量子比特(Quantum Bit),由华盛顿大学的 Steven Elias教授提出，是一个量子位。他认为，量子比特就是具有两个量子态——|0> 和 |1> 的量子系统。而量子比特的计算则依赖于量子门(Quantum Gate)。

然而，为了实现更加有效率的量子计算，Elias教授指出，要实现一个量子系统需要使用许多量子比特。如果使用了太少的量子比特，则其效率会变得低下。如果使用了太多的量子比ット，则可能会出现许多冗余计算资源。所以，如何选择合适数量的量子比特，就成为量子计算中最重要的问题。

同时，量子计算还面临着许多复杂的数学问题，如：精度要求、容错率、计算成本以及错误纠正等等。这些都将成为量子计算未来的挑战。

# 2.核心概念与联系
## （1）量子态(Quantum States)
量子态（Quantum state）通常表示为两个不约束的子空间的直积|ψ(n)>=|0>|psi(m)>=(|0>,|psi>)^T=(|0>,|ϕ_n>)^T，其中<|psi>为一矢量。这个量子态由两个离散矢量组成：|0>(通常记作ket zero)，对应于量子态的两个基态（量子叠加态中的两级基态），即ket null和ket positive；|ϕ_n>(通常记作ket positive)，对应于量子态的量子态，即所谓的superposition states。
量子态的叠加状态(superposition state)可以表示为<ψ>=Σ_n|χ_n>|<ϕ_n>|<ρ>|^2，其中<ϕ_n>代表纯粹的能级，<ρ>代表混合态。量子态也可以分解成纯态(pure state)和混合态(mixed state)。纯态通常是一个量子态，它只有两个或两个以上能级处于激活态，而混合态既包含纯态也包含非纯态。

## （2）量子逻辑门(Quantum Logic Gates)
量子逻辑门（Quantum logic gate）是指由基本的量子门构成的门电路，用来实现量子计算机的功能。目前，主要使用的量子门包括：Hadamard门(Hadamard gate)，Pauli门(Pauli gate)，CNOT门(Controlled-not gate)，Toffoli门(Toffoli gate)，Swap门(SWAP gate)，Fredkin门(Fredkin gate)。

常用的量子逻辑门包括：

### Hadamard门(Hadamard gate)

Hadamard门是一个单量子比特的门，它使得该比特的量子态从|0>变为(|+>+|->),也就是使得该比特处于相反的两个能级的交替排列。如下图所示:


对任意一个量子比特，Hadamard门由Pauli-X门与Pauli-Z门组成：

H = 1/√2 * (X⊗I + Z⊗X) = 1/√2 * [(1+i)/2] [(-1+i)/2]

其中，I指的是单位矩阵，X代表Pauli-X门，Z代表Pauli-Z门。

### CNOT门(Controlled-not gate)

控制 NOT (controlled not, CNOT) 门是最基础的两个比特控制的单量子门。它由一个控制比特和一个目标比特组成，当控制比特为1时，目标比特的量子态发生反转。如下图所示：


CNOT门的矩阵形式如下：

CNOT = |00><00| + |00><01| + |01><10| + |01><11|, |00><00| + |01><11| + |10><10| - |11><01|> 

### Toffoli门(Toffoli gate)

Toffoli门(Toffoli circuit or Fredkin's multi-controlled AND)是一个三个比特控制的单量子门。它结合了CNOT门和CNOT门之间的控制，当所有三个控制比特都为1时，两个目标比特的量子态发生反转。如下图所示：


Toffoli门的矩阵形式如下：

Toffoli = |000><000| + |001><001| + |010><010| + |011><011| + 
           |100><100| + |101><101| + |110><111| + |111><110|>


### Swap门(SWAP gate)

Swap门(SWAP circuit or Peres gate)是一个两比特控制的双量子门。它交换两个量子比特之间的数据。如下图所示：


Swap门的矩阵形式如下：

SWAP = |00><00| + |01><10| + |10><01| + |11><11|

### Bell门(Bell gates)

Bell门是由两个受控-NOT门(Controlled-not gate)演变而来的。Bell门的主要目的是让两个方块的量子态进行完美的重叠(entanglement)。比如说，在Bell-State|Phi^+><Phi^+|中，量子比特0和1处于基态态，这两个量子比特之间已形成了一定的纠缠。

- Controlled-Phase门(Controlled-phase gate)

  Controlled-phase gate 是一种三比特控制的单量子门，其作用是当两个目标比特同时受到同一控制比特控制时，改变它们之间的相位。Controlled-phase gate 可以用于实现量子通信。如下图所示：


  Controlled-phase gate 的矩阵形式如下：

  Controlled-phase gate = |000><000| + |001><001| + |010><010| + |011><011| +
                           |100><100| + |101><101| + |110><110| + |-11><-1-1|>

  如果两个目标比特分别为|00>和|11>，则有Controlled-phase gate = |-π/4><-π/4|，如果两个目标比特分别为|01>和|10>，则有Controlled-phase gate = |π/4><π/4|。

- Controlled-Z门(Controlled-z gate)

  Controlled-z gate 是一种四比特控制的单量子门，其作用是当三个目标比特全都受到同一控制比特控制时，改变它们之间的相位。如下图所示：


  Controlled-z gate 的矩阵形式如下：

  Controlled-z gate = |0000><0000| + |0001><0001| + |0010><0010| + |0011><0011| +
                      |0100><0100| + |0101><0101| + |0110><0110| + |0111><0111| +
                      |1000><1000| + |1001><1001| + |1010><1010| + |1011><1011| +
                      |1100><1100| + |1101><1101| + |1110><1110| + |-111><--1-|

  如果三个目标比特都为|000>或|111>，则 Controlled-z gate = |-1><-1|， 如果三个目标比特都为|001>或|110>，则 Controlled-z gate = |+1><+1|。
