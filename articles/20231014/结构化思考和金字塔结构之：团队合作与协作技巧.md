# 1.背景介绍


为了更好的掌握团队合作和协作能力，建设有效的团队，提升个人职业能力，成功塑造优秀工程师形象，企业需要关注并实施团队建设方面的管理。作为公司的管理者，如何能够有效地运用集体智慧、组织流程和管理工具来提高工作效率，推进团队共同发展，是每一个企业都需要关注的问题。而管理团队的人才，也是企业中重要的资源和竞争力。因此，建立健全的团队管理机制，可以极大地促进公司的发展，增强员工的创新能力和动力，实现各项工作的顺利完成。
人际关系的发展是一个动态不断演进的过程，因此对于团队合作和协作，存在着各种不同的方式和方法，如沟通技巧、配合默契、委派任务等。下面，我们就以面对面交流的方式，介绍一些在不同团队环境下，如何有效应对不同情况所采用的合作模式和协作手段。

        在信息时代，快速的发展和变化，使得企业需要具备一定的协作意识。面对复杂、多元的信息需求，企业必须充分利用信息技术的力量，提升协作效率和质量。在信息化程度较高的今天，企业已经可以采用远程办公模式来协助员工完成工作。远程办公模式能大幅度节省企业的开支，降低成本。但是，远程办公模式也带来了新的协作问题。如何让远程工作方式中的人员和群体互相融洽、共享情绪、解决突发事件，成为企业团队合作的关键呢？下面，我将阐述基于结构化思考的团队合作与协作技巧。

# 2.核心概念与联系
## 什么是结构化思维
结构化思维，又称层次分析法或分类思维，是指通过大脑的多层次分化结构来进行分析处理的一种思维方式。它强调的是知识的系统性、结构化、明晰，将认知活动从零散的、按部就班的个别认知活动中抽取出来，集中到一起，然后再重新整合，形成具有逻辑性和完整性的整体认知。结构化思维是一种理解和组织复杂问题的有效方法，适用于任何涉及多个因素的决策和日常事务。结构化思维的基本原则是：通过分类、归纳和概括，找出解决问题的关键要素，并且按照其优先级和依赖性重新排序。
结构化思维的一般步骤包括：

1. 构想或定义目标。首先，设定一个目标，或着论证问题的现状和原因。

2. 将目标转化为概念。然后，把目标转变成可操作的概念，即划分出相关的主题。

3. 识别关系并组织主题。接着，根据主题之间的相互关系，确定它们的结构顺序。

4. 分辨目标的层次。最后，把目标分解成不同层次上的任务或步骤，即按阶段逐步解决。

## 什么是金字塔结构
金字塔结构，是指以结构化思维思想作为指导的管理科学的一类体系。它由四层构成，即上层、中间层、下层、最基层。层次之间的关系是金字塔的架构，金字塔形象地代表了管理者的视野、理想和价值观。金字塔结构的各层各自承担不同的作用，具有完整的功能和责任。最高层级，也就是最基层，是信息的源头；中间层，是知识的载体；上层，则是执行的主体。信息越来越少，对人的注意力越来越集中，知识和技能越来越丰富，而人的精神才能越来越丰富。因此，构建一个有效的、富有活力的管理团队，关键就是在正确的位置倾听、理解、倾注力量，从最基层打通管理链条，传达并传递管理信息。

## 团队合作与协作的关系
团队合作与协作，指的是团队成员之间形成紧密的合作关系，进而共同促进工作的完成。但是，只有真正懂得“合”的精髓，团队合作与协作才能得心应手，真正发挥作用。下面，我们就以沟通技巧为例，分析一下在不同团队环境下的合作技巧。

## 沟通技巧
### 一言以蔽之——“不要迷信直觉！”
“直觉”这个词可能是绝大多数人的第一反应。但是，研究表明，绝大多数人对现实世界的感受往往不是直接的，甚至是错误的。而且，与直觉产生冲突的，通常不是直觉本身，而是对它构建的假设。比如，我们会认为，那只老虎上树一定是凶猛的，这便导致我们忽略了现实世界中很多事情的复杂性。所以，当遇到困难时，一定要努力找寻真相，并准确诠释真相背后的原因，而不是去抱着直觉做决定。 

### 六种沟通技巧
- “聆听对话”：这种沟通方式是指在面对面交谈中，观察对方的反应，了解对方的看法和想法。双方一起倾听对方的话语，分析反映出的真实想法，并与此沟通。
- “观察他人行动”：在小组讨论中，你可以观察到其他成员的表情、举止、行为，以及他们的思维模式。通过观察他人的行为和言谈，你可以得到进一步的启发，并尝试找出他们的共鸣点。
- “假装自己很亲切”：在面对面交谈时，你可能会被看作是个陌生人，所以你的肢体语言可能会显得冷淡，但却没有失去真诚。其实，你可以假装自己很亲切，这样你就可以融入对方的生活，也可以更好地表达自己的情感。
- “询问原因”：如果某件事发生了偏离预期的结果，你可以向对方提出原因。在提出原因之前，应该先了解对方的真实想法、理由以及可能出现的原因。试着找出解决问题的可能性。
- “质疑你的直觉”：你要认识到，你的直觉往往是片面的、不完全的，尤其是在面对复杂的事情时。因此，你要提醒自己，要保持警惕，要提出质疑。有时，需要反驳和批评才能找到更好的方案。
- “展示你的观点”：展示你的观点的方法有两种，一种是提前准备好的讲座形式，另一种是匿名的方式。如果你希望有人帮忙回应，你可以选择后者。

### 提供建议和意见的五大技巧
- “说给我听”：如果你想获得对方的建议，请尝试讲给他听。如果你和对方已经熟悉，那么你就可以像对待自己的问题一样，一边讲一遍听。这样，对方就会有机会反思，并提供更多的意见。
- “平静心态”：刚开始的时候，不要有太大的压力，保持冷静的心态，与对方一起面对困难。当你感觉到压力开始在影响你的计划时，你就可以切换到开放的状态，让对方知道你的兴趣和能力。
- “按要求细致入微”：如果你准备分享你的建议，请务必细致入微。不要让对方觉得你在嘲弄你，而是要告诉对方你的真实想法。把想法逐一陈述，并且给予详尽的解释，让对方容易理解。
- “鼓励听众”：面对你的建议，对方往往会有不同的看法，你要鼓励对方支持你的想法，并为此付出时间。鼓励大家合作，可以激发大家的积极性，促进工作的进展。
- “找到借口拒绝”：如果建议无法令对方满意，或者你与对方的感情色彩不同，请坦率地表达拒绝的原因。不要过于自负，这样会让对方觉得你不够关心。拒绝建议的同时，也请记得用友善的语言表达自己的态度。

### 从对话中学习——如何说服别人
如何说服别人，这是所有团队领导者都需要了解的技能。只有充分理解“为什么”，才能真正做到说服。这六个技巧，在不同的情况下，都能派上用场。但这里有一个重要的技巧，就是“矫正陷阱”。

矫正陷阱（Counterintuitive）是指使用捷径、奇特的方式，让别人误入歧途。这是说服别人的捷径，但有些时候，也可以用在帮助自己实现目标上。下面，我总结了几种矫正陷阱：

1. 假装比对方聪明：很多人以为自己比对方聪明，实际上，在群体讨论中，往往反而更加笨拙。在和朋友们谈论话题时，不要夸张自己的聪明，相反，要努力去说服对方。
2. 用平实的语言说服别人：虽然说话要小心谨慎，但用平实的话语提出诉求，会有助于人们听清楚你的真正目的。例如，“我认为你做的这个方案很好。”“你的问题是什么？”“我不明白你的意思。”等等。
3. 使用时髦的口号：虽然年轻人喜欢追风破浪，但实际上，年长者也不易被感动，因为他们习惯了流行的语言和新潮的词汇。试着别用过时的词语，因为他们往往不能察觉到这些改变。例如，“革命成功！”“企业蒸蒸日上！”“上帝给你一颗脑袋！”等等。
4. 强调自己的优势：不管对方是否相信你，还是觉得你老土，还是说自己只是借口，都没关系。有时候，即使只是在一两句话里押韵，也能激起对方的共鸣。例如，“你厉害啊！”“你怎么就这么厉害？”“你比他们都厉害！”等等。
5. 不要说服对方：不要指责对方不懂事，也不要试图强化自己的优势。在谈论事情时，记住，先给出自己的看法，再站在别人的角度思考。尤其是那些言行不合的情况，多多包容，也许你就会发现合作的最佳方式。
