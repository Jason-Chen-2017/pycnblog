# 1.背景介绍


晶圆制造(又称为薄膜制造)是硅基半导体制造领域中最重要的一环。它利用微透镜反射、压电、低温等多种传导方式将微型颗粒物浓缩到非常小尺寸。晶圆制造对高端市场来说占有重要地位，其中就包括手机、电脑主板、存储设备、笔记本电脑等各类产品的制造。但是，晶圆制造技术有诸多复杂性。它的制程复杂、工艺精细、制造方法繁杂、特殊要求多样，不少品牌均采用不同工艺流程，导致同一个晶圆可能有不同的形状和排列方式。此外，由于晶圆制造过程中需要长时间进行紧张的工作条件，加之电子元件功能不断演进、技术更新迭代，使得晶圆制造过程呈现出复杂、不断变化的特点。
# 2.核心概念与联系
为了更好的理解晶圆制造的相关知识，首先需要了解一下其中的一些基本术语和概念。
## 2.1 集成电路IC设计方法
集成电路(Integrated Circuit, IC)是由不同器件按照规定的布线顺序组装而成的功能完备的电路。IC在制造上主要分为两大类，即封装集成电路(封测IC)和堆叠集成电路(锂电池等离子电容等)。常用的封测IC有SMD(Samsung MicroDesign,美国三星MicroDesign公司)、QFP(Quad Flat Package,四极平行包装)、BGA(Bus Grid Array,总线网格阵列)等；而堆叠集成电路则主要是面向一般应用需求。目前，绝大多数工厂生产的IC都是封测IC。
## 2.2 晶圆制造工艺流程
晶圆制造工艺流程分为以下几个阶段：原始设计、预处理、成形、填充、注塑、摆正、铣削、验证测试、切割、装配。整个工艺流程所需的时间可达几十至上百个工作日。其中，预处理阶段属于拍照阶段，因为它涉及高温烘烤、电解质吹扫、气泡回旋等技术。
## 2.3 背掌工艺
背掌工艺是晶圆制造的一个重要组成部分，也是硅基半导体领域中最具创新性的技艺。背掌工艺的关键在于将晶圆下方的颗粒物从晶体表面切除掉，使之完全脱离原有的晶圆结构，并从掌面部附近提取材料进行处理。背掌工艺的主要过程有去背面、开掘机、纳米扩散、电镀、干燥等。其中，去背面和纳米扩散两个工序较为复杂，耗时较长。
## 2.4 位移扩散
位移扩散是利用激光刻蚀法、冲击冲击法、切割切割法、硬化钻孔法等方法将晶圆表面和背面间隙内的晶圆附着层分离，并向外扩散的一种晶圆制造工艺。位移扩散制造的晶圆具有透明度高、隔热能力强、集成度高、导通率高等优点。
## 2.5 中空制造
中空制造是指用硅膜厚度很大的无缝膜或漆膜覆盖晶圆周围的空白区域，从而实现将晶圆在结构上分布的垂直方向划分为上下两个层面的一种制造工艺。中空制造的产生主要原因是解决厚膜/漆膜上的二氧化硫、碳酸钙、磷酸盐等物质聚积问题，提高晶圆的表面积和导通率，增强IC的整体性能。
## 2.6 晶圆制造综述
综上，晶圆制造技术有很多复杂性，其技术流程复杂、工艺精细、制造方法繁杂、特殊要求多样，但只要认真学习和掌握相关的知识，就可以通过不同工艺流程将杂糅的晶圆制造工艺有条不紊地进行。
