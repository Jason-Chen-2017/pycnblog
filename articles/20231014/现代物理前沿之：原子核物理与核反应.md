# 1.背景介绍


## 1.1 什么是核反应堆？
核反应堆（Nuclear Reaction Hollow）是指由高纯度的铀、氮原子核聚合而成的高强度绝缘体结构，具有极高的热处理能力和强大的能量密度。目前核反应堆主要用于生产或研究太阳光伏电池等用途，它的实用性值得相当程度地认可。

核反应堆是一种低温、高压、高强度、分布广泛的设备，其核心是一个容积很小的钢筋状结构，在其中可以进行各种核反应。核反ق堆分两大类，一种是带燃料核反应堆（FBR），另一种是不带燃料核反应堆（NFR）。

## 1.2 为什么需要核反应堆？
由于高级金属所含有丰富的微观粒子、超大面积的孤立质量和巨大的能量密度，使得它们既不易溶于水，也无法被其他高温气体完全驱散，因此，核反应堆（Nuclear Reaction Hollow）这一方法便产生了。核反应堆可以作为低温、高压、高强度、分布广泛的装置，能够制备各种用于军事、科学、农业、工业、电力、石油等领域的产品。

核反应堆的核心是一个容积很小的钢筋状结构，无线电波通过这个结构传递到宇宙中各个角落，随后在钢筋状结构中发生核反应，生成大量的高分子和异物质，最后从这一庞大的产物中提炼出各种化学物质。由于核反应堆中存在的杂质较少，而且可以长期重复使用的缘故，核反应堆已成为一种极具商业价值的前沿技术。


## 2.核反应堆核心概念与联系
核反应堆的研究始于古埃及，为了解决微生物资源问题，古埃及的哲学家希腊人试图在土壤和岩石中发现新的元素。然而古埃及没有找到足够的高质量土壤，因此，希腊人转向寻找没有化学传播或有机物种的岩石。在欧洲，最早的岩石出现在19世纪末期，但是由于其重量过重，加上缺乏耐久性和辛烷固体，导致无法供应整个欧洲，直到1940年代才在德国发现了一批新的岩石，这些岩石质量很轻，色彩鲜艳，并且还含有有机物质，如钾、钠、硫酸钠、硝酸钠、磷等。


经过一系列的实验验证，1947年哥本哈根大学的Sally Price将这一批岩石称作“基因矿”，之后又在耶路撒冷大学建立了著名的美因茨核实验室。20世纪50年代，德国的科学家用电解液、微生物种子和碱金属精华制成的液氮结晶的态势，探索了用一组稀有微生物、氮原子核和氧化剂制成的岩石，来提炼出特异的溶质。1962年，理查德·费罗和他的同事们研制成功了第一颗原子弹“奥巴马海洋核潜艇”中的核反应堆。


核反应堆基本过程如下：

1.形成核材料：由氮、铀或氘原子核聚合而成的高纯度绝缘体构件。

2.制备核反应器材：主要包括冷却液、辅助设备和加热设备。

3.核反应：将氮原子核与氮离子进行聚变或失火，形成各种化学物质。

4.捕获提炼物质：将有效率且高效的物质从核反应堆中捕获并分解。


在核反应堆中进行高温、高压、高反应的原理是通过结构方面的微观参数优化、电流方面的耦合控制和物理方面的三维流场优化来实现的。核反应堆的设计人员必须要面对很多困难，如安全、环保、稳定性、耗能、环境影响等。核反应堆的应用领域非常广泛，如太阳光伏电池、氢燃料电池、核燃料电池、新型电动汽车、核武器、太阳能电池等。


核反应堆的建造过程：

1.选取材料：核反应堆的主体是铀或者氮原子核，一般为六氘铀，或者镉氮核。

2.定位装置：核反应堆可以采用各种形状，如平面、圆柱、桶形等，大小一般为1.5至5米。

3.选择精心布局：核反应堆必须要经过精心布局，使得每一个层都能够有效的吸收和释放氮原子核中的氮离子，并且保证高度的精确度。

4.配置辅助设备：核反应堆内部必须配备适当的辅助设备，如自动化工作台、控制电路、检测仪器、记录仪器等。

5.测试运行：核反应堆的性能必须要经过测试才能确定，首先在原有的样品中加入一些固态介质，然后加入高分子元素，如铅、锌、铬等，再将氮原子核分离出来进行核实验，实验结果作为依据。


核反应堆的分类：

- 不带燃料核反应堆（NFR）：主要用于航天、制氧、生产核燃料、太阳光伏等行业。

- 带燃料核反应堆（FBR）：主要用于研究核反应堆的特性，利用火箭发射实验测定核反应堆内部结构、性能等。

## 3.1 反应堆的发展历程及现状分析
在500至1000年前，欧洲各地的岩石里含有微量元素，而这些微量元素可以在非常短的时间内熔融或产生化学反应，如经典反应堆，就是利用这些微量元素的热运动来促进各种化学反应。19世纪末，苏联专家根据世界上已知的矿物化学元素及其化学反应，设计了一种高速反应堆，可以同时发生多种反应，但最大的问题是电弧太小，不能承受核反应的高分子量输入，只能起到发散作用，直到60年代才发现电弧可以增大到0.5毫米以上，才能够承受高分子量的输入，这才开始真正开发出用于海洋核燃料生产的核反应堆。

1962年，美国的克莱恩·奥巴马、迈克尔·欧文等人创造出原子弹核潜艇，核心就是由氮、氘、铀原子核聚合而成的高纯度绝缘体构件。核潜艇是基于核反应堆的最新发展，具有极高的热处理能力和强大的能量密度，能够制备各种用于军事、科学、农业、工业、电力、石油等领域的产品。核潜艇制造的核燃料已经远远超过了原子弹的功率，核燃料可以用来制造电池、飞机的动力系统、空间站等重要装备。

## 3.2 氮原子核聚合构件制造方法概述

### （一）选择原子核
对于核反应堆的制造，必须要有所选择，通常选择氮原子核或者氘原子核的混合原子核作为主体材料，通过氮、氘、氢的组合，制成合金材料，从而制造出比较强的高纯度、结合能力好的高强度、分布广泛的核反应堆。但由于缺乏成熟氮、氘双原子核，实际上，只有一种氘原子核存在，因此，目前只能选择氘原子核制成的核反应堆。

### （二）分辨反应极和反应腔

核反应堆主要分为四个区域，即反应极、反应腔、反应槽、外护罩。反应极的主要功能是在超临界状态下，激发核反应，生成辐射；反应腔负责结构上的调节，防止核反应堆破坏；反应槽对高分子进行分流；外护罩保证周围空气不会进入核反应堆。


### （三）铁电、电磁环路的安装

#### （1）铁环线安装
将反应腔封闭起来，加装铁环线，提高放电效率。环线安排：首先将环线的方向向外推进一步，然后将四条环线均匀布置在反应腔内侧，将环线铺盖固定于反应腔上部，确保环线平行与反应腔边缘。在环线接头处，有足够的长度做饭盆。在安装环线时，应严格按照铁环线规定安装，否则会造成安全隐患。

#### （2）电磁环路安装
将电源线束放入反应腔中，使电流能够连续向反应腔传递。安装电磁环路的方法有两种：一是电容器法，就是在电路支架上安装电容器，并配有电阻，负责消除干扰信号；二是电动机法，就是在管道布线和反馈回路之间添加电动机，向反应腔输送能量。

### （四）控制反应
核反应堆的控制系统是进行核反应堆内核物质的分解、过滤和再生的重要手段，必须正确安装及运行才能达到预期效果。控制反应有两种：一是辐射通道控制法，也就是利用辐射通道，将高能量核辐射通道导向存储物质的区域；二是核膜法，即利用核芯来控制反应物质的分解，在分解过程中，通过辐射通道导电，保持核芯的疏密。

### （五）外皮控制
核反应堆的外护罩主要用于隔离核反应堆周围空气，防止辐射泄露或进入核反应堆，保证环路正常运行。外部护罩的构造要合理，它必须能够承受反射或抵御核反应堆内核物质的进出，并有效地阻断反射或抑制核物质的扩散。
