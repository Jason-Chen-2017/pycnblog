# 1.背景介绍


## 光刻胶
光刻胶（PET）是指将组织细胞或者细胞组织中特定蛋白质、核苷酸等元素从实物转化为电子而形成的一种化学薄膜，并能够在电导率较高、温度适宜、光照条件良好的条件下进行晶格形态优化、使其具有完整的三维结构及生命周期。光刻胶在光电子显示器、智能手机、电子设备中应用广泛。
## 显影工艺
显影工艺是利用光刻胶技术在真空环境下生产出具有印刷电路板特性、具有较高颜色分辨率和电导性的可反射、高透明度的产品的方法。在太阳底下、阴暗处，该工艺可以将材料转移到无限远距离。早期的显影工艺主要用于铅笔和粉笔等日用品制作，随着高速电子技术和半导体技术的发展，显影工艺已成为近年来非常重要的生产环节。
# 2.核心概念与联系
## PET
### 概念
光刻胶(Polymer Electrolyte Transfer，简称PET)是将细胙中含有的特定蛋白质、核苷酸、氨基酸等氧化物和溶酶等成分从细胞中提取出来，经过配制后制成带有完整组织结构的电化层状物质，形成具有一定晶格大小且易于制备电路板的新型化学薄膜。其主要特点有：

1. 不依赖脂肪酸等自由基；
2. 具有完整的细胞结构及生命周期信息；
3. 在一定程度上降低了机械的要求，可以应用于军事、医疗、消费电子、工业自动化、电力电子、航空航天等领域；
4. 可以用于量产，但生产成本较高。

## 显影工艺
### 概念
显影工艺是利用光刻胶技术在真空环境下生产出具有印刷电路板特性、具有较高颜色分辨率和电导性的可反射、高透明度的产品的方法。其基本原理是在金属界面上用电化层方式注入一个密度介于石英、碳纤维体之间的颗粒，通过交流耦合、拼接和气流传递来实现工艺过程中的多功能性。显影工艺在人类文化、自然科学、工程技术、电气电子学、材料学等多个领域都有着广泛的应用。目前国内外有关显影工艺的研究者们已经取得了丰硕的成果，在各行各业得到了广泛的应用。

显影工艺的一般过程如图所示：


1. 浮法：在金属表面用滴注机固定和浸泡，使电化层晶体结构紧密密封。

2. 电导: 使用激光打印机或照相机向电化层上投射照射光线，使电化层电导性增强。

3. 拼接：将电化层层层叠在一起，然后加入少量的助剂，通过传热工艺使晶粒紧密连接起来。

4. 切割：将电化层物质切割成薄膜，并按需打磨去除杂质。

5. 装饰：对电化层物质做色彩塑造、影印、融晶等美术处理，增加产品的视觉效果。

### 术语
显影工艺涉及到的术语和概念如下表所示：

|术语/概念|描述|
|:---|:---|
|液化物浓度|指在电影电视工业中常用的一种衡量标准，通常定义为电影或电视剧的感光度（也称为景深）下降至一定指数级的光源或平坦地面上的灰阶差。它是利用摄氏度计测量摄氏零度后的单位时间（秒、分钟、小时、天等）声音通量的大小。|
|红外线|一种射电能量传输的法则，其主导电磁效应由热辐射发出，而辐射性电离辐射线则可转化为红外线信号，因此被称为“红外线”。它是用红光照射某物体，使那些穿过其反射区的红光所到达的地方反射回来，就能窥探到其电磁场中存在的各种物质的强弱。|
|电化层结构|在显影工艺中，电化层结构指由氧化物和溶酶所组成的表面薄膜的晶格结构。电化层结构包括碳纤维体、碳氢化合物层、高层石英层、其他相关层等，其中碳纤维体与石英层是显影工艺的两种主要电化层结构。|
|显影材料|指用有机的质材料和无机的精材料混合而成的电化层结构所使用的材料。显影材料种类繁多，例如石英、棱镜、碳纤维、橡胶、玻璃瓶、玻璃等等。|
|投影|指一种光刻胶工艺方法，即通过光栅屏幕上的某一点释放光刻胶而实现大批量生产。显影过程中的投影主要分为两类，即油光投影和电光投影。油光投影是把光刻胶纳入液体涂布中，电光投影是通过液化气引擎加热炉灌入电化层物质来实现。|
|轧制|指在显影过程中使用定制的机床硬体来使电化层层层叠加并悬浮在介质表面上，将其与电机与电容器整合，然后对其进行切割、切割、切割、去除杂质，形成完整的显影件。轧制常用硬体有刀具、锤子、刃具、钳子等。在轧制结束之后，将电化层悬挂固化，以防止分解。|
|编织|指对齐、调整和连接电化层层面上的碳纤维层，通过涂刷或彩笔等方式施加强度来形成不同形态的电路板。这种工序也称为电导剪裁。|
|过滤|是显影工艺的一项重要步骤，是为了去除电化层中的气味、残余物质、电磁污染物，以保证显影后的产品无瑕疵。采用双向滤波技术可以有效减少漏电、断层现象。如：在油光投影过程之前先过滤掉气泡、烟雾、油墨等杂质，再进行油光投影。|
