# 1.背景介绍


决策系统作为业务的关键部分，其功能在于通过历史数据、现实数据和预测模型为企业提供高效可靠的决策依据和建议。但如何将人力资源管理(HRM)和人事信息管理（PIM）融合到决策系统中，充分发挥HRM和PIM的作用，确保企业能够更好地执行工作计划并实现其目标？

随着互联网经济的发展和数字化进程的不断加速，越来越多的公司开始了“移动化”转型。通过提升员工工作效率、优化管理流程等手段，使得公司的人力资源管理人员日益成为企业绩效的重要支柱。而人事信息管理是一个独立于实体组织之外的独立系统，它承担了管理人才选拔、薪酬福利、培训、职位发放、离退休管理等职责。

如何将HRM和PIM融合到决策系统中，构建一个完整的企业决策系统呢？本文将探讨这种在线学习型的决策系统架构，它可以有效地整合不同部门的资源，提升决策精准性、效率和效果。另外，本文还将介绍基于机器学习的决策系统架构，它可以根据大数据分析，进行自动决策。

# 2.核心概念与联系
## 2.1 HR信息系统(Human Resource Information System, HRS)
HRS由人力资源信息、数据、信息技术三方面构成。HRS的信息源包括员工档案、工作计划、奖惩信息、绩效评估结果等。HRS的数据由人力资源专业人员进行收集整理，并通过数据仓库进行存储、清洗、转换、加工，并按相应要求对外提供数据服务。HRS的信息技术主要用于人员资格管理、薪酬制度管理、待遇管理、培训管理、公共关系管理、招聘管理等。

## 2.2 PIM信息系统(Personal Information Management System, PIM)
PIM由人事信息、数据、信息技术三方面构成。PIM的信息源包括个人基本信息、人员组织架构、工作信息、职务信息、婚姻信息、教育经历、工作职称等。PIM的数据由各个组织或部门的人员自行填写、提交和维护。PIM的信息技术主要用于管理和处理各种人事信息，如员工的工资、职位、薪酬、办公室地址、手机号码等。

## 2.3 概念、分类及联系
HRMS是指具有人力资源管理功能的集成管理系统，包括人力资源信息系统、绩效管理系统、薪酬管理系统、招聘管理系统、员工培训管理系统、绩效评价系统、劳动合同管理系统、离退休管理系统、招聘信息管理系统等。HRMS将人力资源管理和其他相关业务系统集成在一起，为企业提供最完善的人力资源支持和决策支持。

PIMS是指具有人事管理功能的集成管理系统，包括档案管理系统、组织结构管理系统、人事风险控制系统、人事知识管理系统、教育培训管理系统、培训信息管理系统、业务咨询管理系统、人才推荐系统、招聘信息采集系统等。PIMS将人事管理和其他相关业务系统集成在一起，为企业提供最全面的人事管理支持和决策支持。

结合以上两个系统，HR-PIM-RMS，即 Human Resources – Personal Information Management – Rule-based System，即人力资源–人事管理–规则引导型系统。其中，HR即HMS，是人力资源信息管理系统；PIM即PMS，是人事信息管理系统；RMS即RBS，是基于规则的决策系统。

基于HRMS和PIMS构建的HR-PIM-RMS决策系统可以满足以下需求：

1. 员工发展规划：HRS可以提供员工培训、人才推荐、员工激励、满意度调查等功能，帮助企业建立员工职业发展规划，引导企业优化员工工作内容和管理方式。
2. 薪酬决策：HRS可以在多个角度提供薪酬报告，为企业提供薪酬决策参考。同时，PIM还可以向员工提供相关薪酬福利政策，促进员工合理配置自己的薪酬。
3. 招聘管理：HRS可以与招聘网站进行数据交换，获取更多候选人的信息，并通过人力资源专业人员筛选，提升招聘质量和效率。PIM也可以向招聘人员提供个人信息，帮助招聘人员更快地匹配适合的人才。
4. 培训管理：HRS可以与内部培训机构或外部培训机构进行数据交换，汇总、整理企业的培训情况，并进行实时反馈，确保培训质量。PIM可以与合作的教育机构共享员工的教育背景，并收集并分析学生反馈信息，提升教学效果。
5. 绩效管理：HRS可以统计员工的绩效数据，并与管理层进行分享，进行绩效分析，提升员工绩效。PIM可以收集员工工作态度和工作成果，分析员工满意程度，提升企业整体的工作效率和产品质量。
6. 工作流程优化：HRS可以根据现有的工作流程，提供优化方案，增强企业的整体竞争力。PIM可以通过实时获取员工信息，调整工作流程，提升员工工作效率。
7. 数据驱动决策：HRMS可以利用数据进行决策，而不是依靠传统的规则和逻辑。PIM还可以提供人才库建设、人才分析、个人履历分析等功能，为企业提供更好的人才选取和管理能力。
