# 1.背景介绍


## 一、什么是摘要生成？
自动摘要（Automatic summarization）也称文本抽取，是指通过自动地从原始文本中选取一定长度的句子或段落，重新组合而成新的文本摘要，用于传递重要的信息，并提高文章的说服力。自动摘要的目的是为了简化、快速地将文章的内容向读者传递给听众。对于长篇报道或者新闻类的文章来说，一般都会有一个较大的篇幅，但其中很多内容都可以删减或者合并，在传播过程中会比较费时，而且可能会引起争议。因此，对其进行自动化地处理是非常必要的。

## 二、为什么要用自动摘要？
### （一）降低认知负担
从信息传递的角度看，大多数人的阅读时间都是有限的。如果长篇报道被压缩到一两个小时内阅读，阅读者就会感觉厌烦或者消耗大量的时间。此外，记忆、理解、产生情绪、思维活动等大脑资源也是有限的，当文章太长时，需要花费大量的时间去解读和消化内容，对认知负荷加重，甚至影响正常的工作学习。所以，通过自动摘要的方式可以有效地节省时间和精力，达到降低认知负担的效果。
### （二）提升说服力
对于技术类、商业类、观点类等文本，人们需要花费大量的精力去阅读和理解，但是往往只有极少的部分能够抓住作者的本质目的。对于这些文章，往往存在着作者的矛盾、偏好、倾向性、误导性等弱点，通过摘要提炼出的关键词、句子、论据等信息，可以帮助读者快速了解到作者的立场、观点、目的，从而对文章进行更准确、客观的评价，具有提升说服力的作用。
### （三）降低编辑工作量
编辑人员通常需要花费大量的时间审阅、校稿和编辑文章，而且由于编辑人员的个人能力和经验不同，对相同内容的摘要可能有不同的选择。如果使用自动摘要，编辑人员仅需根据自己的判断，即可快速制作出最适合的摘要。这不仅节省了编辑时间，还能提高整体工作效率，改善写作风格。
## 三、如何实现自动摘要？
### （一）短语摘要方法
最简单的方法是利用短语摘要方法，即按照文章中的语句来确定摘要。该方法的基本思想是：首先对文章中的每个句子计算单词出现频率，然后将最高频率的几个单词和它们前后的一些句子作为摘要的组成部分。这种方式在一定程度上提高了摘要的可读性和流畅性。
### （二）自动摘要模型
目前，比较常用的自动摘要方法有两种：Latent Semantic Analysis (LSA) 和 Recurrent Neural Network (RNN)。LSA 是一种基于矩阵分解的统计方法，将文章中每一个词转换为一个实数向量，再由这些向量聚集在一起，以寻找某些主题的共同特性；RNN 使用循环神经网络，它能够模仿人的语言动机，以生成具有自然风格的文本摘要。
### （三）关键句抽取算法
另一种比较常用的摘要生成算法叫做关键句抽取算法，该算法通过分析文章中的关系及结构，识别出文章的主要中心词和中心句，然后按照中心句向后扩展，直到把整个文章都包括进来。由于中心句前后并不一定包含足够的上下文信息，因此生成的摘要就不一定全面，但它的优点是不需要机器学习模型，速度快且易于实现。
