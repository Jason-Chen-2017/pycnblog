# 1.背景介绍


超材料(superconductor)在近几年蓬勃发展，并且逐渐成为主流材料，它利用结构上的一些特殊性质，诸如共价键、高隙结构、多层超导体、超导和弱相互作用等特点，实现了比传统金属更高的导电性能和导热性能。然而由于超材料的表面存在着极大的厚度和低韧性，使得它们在应用上有着很大的不便。随着超材料的深入发展，已经出现了一批新的“逆反子”，即负折射材料(insulator)。负折射材料的制备方法主要有钢-铁(Si-Mo)界面摩擦，将某些原本处于不同层次的正序和负序材料挤压成一个整体，导致材料之间的晶格融合，从而形成具有异常透过率的负折射材料。在此基础上，进一步利用结构分离和光刻等工艺方法，可以生产出越来越多种类的负折射材料。近些年来，人们对负折射材料的理解加深，认识到它与传统金属或锂聚合物等材料在某些方面有不同的特性。比如，负折射材料中存在着许多本身是无限维度的态密界面，可以通过空间结构的巨大张力改变介电的能量分布；另一方面，负折射材料还具有良好的高速传导特性，可用于制造传感器、激光元件、微纳米芯片、氟化工产品等领域。
超材料与负折射材料的产生及相关理论研究，将开启新时代的科技革命和产业变革。

# 2.核心概念与联系
## 概述
超材料是指具有非常高导电性、导热性和单一极化性等特征的结晶体系。一般来说，超材料由多种无序的金属原子链组成，在杂质、空穴、膜和内层填充材料等表面之间进行随机排列形成复杂的结构，整个结构具有透射率不规则、低密度、表面积大的特点。对于天文学、医疗、航天航空等领域应用至关重要。但超材料的制备方法繁琐复杂，成本高昂。另外，超材料制备过程中往往会出现膨胀变形、晶格断裂等不可预测的情况，影响其性能和使用寿命。因此，为了降低制备难度和保证性能，人们提出了许多技术手段来简化超材料制备流程，并减少不确定性。目前，人们在制备超材料的过程中，尤其是利用负折射材料来制作高性能、耐用且多功能的材料时，都采用了这一策略。

负折射材料(Insulator Materials)是在晶体管、导线、电缆、太阳能电池等器件等电子设备的绝缘材料中的一种，可以提供高达10%的阻抗，也即可以屏蔽物体的一半以上反射光。因此，它被认为是一种最佳的绝缘材料，已被广泛应用于高精度、高可靠性、高功率、高效能的各种电子设备中。对于工程实践而言，负折射材料既可以承受长时间、大范围的高温高压实验，又可以满足要求短时间、小范围、低温低压实验。此外，负折射材料还可以作为材料的接头件，使得多个器件组装成为一个具有不同性能和灵活性的器件网络，有效解决器件之间的性能冲突问题。

在某种程度上，超材料与负折射材料之间存在着类似的地方。它们都是通过非常复杂的晶体结构来实现超导和弱相互作用。但是，超材料是多晶体结构，而负折射材料则是一个晶体孔，仅有一个轴向反应的极。而且，超材料的单位体积较小，晶粒大小不均匀；而负折射材料的单元体积相当大，整个材料的体积往往几乎等于晶体单元数量的乘积。所以，它们可以说是两种截然不同的材料类型。


## 共价键、高隙结构和多层超导体
超材料的基本构造方法是采用共价键和高隙结构，这种方式能够有效地生成具有良好带隙性、强度高、低纳威、高磁场性和坚固耐磨性的超导性材料。在普通的超材料中，晶体的构型往往具有四面体、五边形、六面体或环状结构。在不同晶体结构中，共价键和高隙结构是控制晶体性能的关键因素。共价键通常呈现两个相邻轴向平行分布的键，它们通过导通的条件差或平衡势而相互关联。高隙结构指的是晶体的表面的一部分浮于晶体表面而不会对称配位，它们允许晶体中的原子按照单向的模式排列。通过这些配置，超材料能够容纳更多的原子，并且在自然界的各个方向上进行单向运动。超材料中的多层超导体则是指由不同层级的硅质堆叠形成的晶体结构。超材料的多层超导体的设计原则就是尽可能地使每个晶粒（“层”）在三个方向上带隙性增强。这样，每一层的晶粒就能够在其他层的层间反复纠缠，形成局部的长距离连接。


## 超导和弱相互作用
超导性是指超材料中的原子能经过相互作用的过程转化为磁性。超导性是两种性质的组合，一是多层级、多晶体性；二是自由自在的自由电子相互作用的过程。超材料通过引入高速的，而非凡的辐射效应，实现了一种结构层面的超导现象，其效率高达数百万亿分之一，远高于单纯依靠玻璃等玻尔兹曼机械导体的设计。超导性还能够兼顾表面和内部的感应性，这使得它成为一种富有吸引力的材料类别。另外，超材料在晶体上引入的弱相互作用也可以显著地改善其性能。超材料中的弱相互作用包括电子-电子相互作用和电子-空穴相互作用，前者对电子带隙、带电荷迅速反应，后者则令原子的振动频率发生变化，并在晶体内部制造干涉场，进一步促进了晶体的性能提升。此外，超材料还有其自己的称谓，即负粒子对称性。负粒子对称性意味着可以在晶体的任一层独立形成、对称的偶极子，这种性质又可以称作反负粒子对称性。通过引入负粒子对称性，超材料可以克服自由能带延伸带来的影响，提升其导电性能。


## 所谓的负折射材料
所谓的负折射材料，其实就是把多层超导体里面某些层级的晶粒先挤压，然后再加入的一种特殊的材料。它的制备方法是将某些层级的正序和负序材料按一定方式挤压或嵌入，从而获得不规则的结构，使之能形成一个具有低导电性、高导热性、单一极化性的孔隙材料。一般情况下，负折射材料的制备需要利用钢-铁(Si-Mo)界面摩擦，将某些原本处于不同层次的正序和负序材料挤压成一个整体，使得材料之间的晶格融合，形成具有异常透过率的负折射材料。对负折射材料的理解将促使我们对其更多的认识和理解。
