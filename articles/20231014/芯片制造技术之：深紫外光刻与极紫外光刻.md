# 1.背景介绍


人们一直以来都认为电子显示器和手机的历史都是在光刻技术蓬勃发展的年代。然而，到了最近几十年，由于光刻技术的不断革命性的发展，单纯靠光刻技术已经无法满足我们的需求。在这个背景下，我们逐渐开始关注光刻技术面临的新技术革命，特别是深紫外光刻（DIC）和极紫外光刻（EIT）技术。顾名思义，深紫外光刻即利用深层紫外线进行光刻技术；而极紫外光刻则是利用非常高的红外光线或者短波辐射进行光刻技术。

这两项技术都属于显著改变传统光刻技术的革命性技术，可以极大地提升像素精确度、工艺精度、硬件设计灵活性、产品性能等方面的能力。另外，由于新型光刻技术的迅速发展，其产业链也将会发生巨大的变化。

随着深紫外光刻和极紫外光刻技术的普及，光刻技术的突破性发展使得我们逐步从传统的封闭产业向开放经济转型。但同时，这种转变带来的新的挑战也越来越多，例如产品定价高、成本高、质量差、产品体积大、缺乏先进成套装备支持、技术限制等。因此，如何更好地运用新型光刻技术、拓宽光刻技术的市场策略、创新管理方法、整合国内外研究资源、保持科技创新氛围等，成为众多工程师和科研人员关心的问题。

基于这些背景，我们考虑到《芯片制造技术》一书将成为众多工程师和科研人员关注的重点。为了丰富阅读者的知识面和全面性，对《芯片制造技术》一书中涉及到的相关知识点进行详细介绍和阐述，并配以实例验证，力求提供更为准确和全面的综述。本文就是《芯片制造技术之：深紫外光刻与极紫外光刻》的开篇。

# 2.核心概念与联系
## 2.1.深紫外光刻（DIC）
> 深紫外光刻(Digital In-Circuit Ophthalmology，DIC)是利用深层紫外线进行光刻技术的一门新型技术。它主要用于图像增强(image enhancement)与矫正(correction)，通过对不同频率的红外线、长波红外线或X射线的模拟信号(模拟成像)进行扫描，将图像信号的锐化效果降低至可接受的范围。

光刻技术是指通过各种光源在电路板上设计晶圆或金属结构，使其具有特定形状和光泽色彩，以达到某种功能的过程。深紫外光刻是在光刻过程中使用深层紫外线作为激光源，将材料表面层进行光刻，达到一定程度的光电转换效应，实现了单层表面晶体材料的真空膜结构光刻。优点是具有一定范围的图像锐化效果较高、层次清晰、成本低廉，应用领域广泛。

由于深紫外光刻能够产生真空膜结构光刻，而且深层紫外线反射率很高，所以它可以使用超低频率的射线技术。但在使用时，还需要注意抗干扰与防护措施，避免其对感光、运动损伤、腐蚀、刺激物反映等造成的影响。

深紫外光刻技术的应用范围包括以下几个方面:

1. 晶圆光刻机: 该技术可在晶圆材料上进行光刻，可达到较高的硬度。

2. 微晶聚合物光刻机: 通过合成微晶聚合物(metal nanoparticles)的放电状态，可进行单层真空膜结构光刻。

3. 晶体光刻机: 在晶体表面上进行真空膜结构光刻，具有较好的成熟度，降低了成本。

4. 光刻喷头: 在硅基层面上进行真空膜结构光刻。

5. 高性能集成电路(HPIC): 集成电路的各个元器件之间通过晶体结构进行连接，通过真空膜结构光刻技术来优化电路板性能。

## 2.2.极紫外光刻（EIT）
> 极紫外光刻(Extremely-Infrared Thermal Ophthalmology, EIT)是利用高度红外或短波辐射进行光刻技术的一种新型技术。它的应用包括人眼镜肿瘤的切除术、运动损伤后的矫正、缺陷治疗、冻结死亡病人的存活时间延长等。由于它的高频率与极端红外线所释放出的辐射强度相比普通的光刻技术而言，其能更好地控制光刻元素与衬底之间的反射系数，使整个结构更加稳固、透明。

1977年，麻省理工学院的兰德尔·凯里（<NAME>ier）提出了这一技术，他发现高频率的红外光线能对细小的空间进行真空膜结构光刻。随后，他对该技术进行了改进，提出了极紫外光刻这一概念，即通过红外或短波辐射作为激光源，将材料表面层进行真空膜结构光刻。

1982年，微软公司开发出了专利“光刻自动化设备”以及相应的技术，用于医疗影像中的手术切除和运动损伤后的矫正。2001年，世界卫生组织(WHO)颁布了报告《强烈辐射照射在儿童耳朵引起的辐射疲劳》，宣称紫外线辐射对儿童耳朵造成的影响已由1990年代初期的1/3减少到目前的2/3，并且随着年龄的增长，此类事件正在变得越来越常见。

2003年，著名的发明家玛丽亚·范康（Mary Fack）等人完成了第一台以极紫外光刻作为激光源的真空膜结构光刻器。通过提高激光功率，这台光刻机可以实现分光束、双倍分辨率和非对称光刻。虽然这项技术具有不可替代的优势，但同时也存在着诸如快速响应时间慢、低温耗高等技术性风险。

## 2.3.光刻与显像技术关系
光刻与显像技术在近几年一直处于走向更高技术水平的关键时期。过去几十年间，光刻技术从孕育期开始进入了一个新纪元，带来了飞跃式发展。首先，它推动了数字技术的发展，如计算机图形学、移动互联网、生物识别技术、自动控制技术等。其次，它推动了材料科学的发展，比如微晶聚合物、晶体复合材料的高效光刻技术。再次，它让我们看到了人类视觉领域的全新发展。

总而言之，光刻技术已成为自然界发光体系建构的基石，它在显著改变人类的光学、电学、机械、材料等科学基础上，催生出许多新一代的显像技术。
