### 4.1 数据库连接
首先，我们需要建立与数据库的连接。在ASP中，我们可以使用ADODB对象来实现这一点。

```asp
<%
Dim conn
Set conn=Server.CreateObject("ADODB.Connection")
conn.Provider="Microsoft.Jet.OLEDB.4.0"
conn.Open Server.MapPath("database.mdb")
%>
```

### 4.2 数据查询
然后，我们可以使用SQL语言来查询数据库中的数据。

```asp
<%
Dim rs
Set rs=Server.CreateObject("ADODB.Recordset")
rs.Open "SELECT * FROM Job", conn
%>
```

### 4.3 数据显示
最后，我们可以在网页中显示查询到的数据。

```asp
<%
While Not rs.EOF
    Response.Write(rs("JobTitle") & "<br>")

Wend
rs.Close
Set rs=Nothing
conn.Close
Set conn=Nothing
%>
```
