## 5. 项目实践：代码实例和详细解释说明

```python
import nltk

def extract_symptoms(text):
  tokens = nltk.word_tokenize(text)
  pos_tags = nltk.pos_tag(tokens)
  symptoms = [word for word, pos in pos_tags if pos.startswith('NN')]
  return symptoms

text = "我头痛，发烧，咳嗽。"
symptoms = extract_symptoms(text)
print(symptoms)  # 输出: ['头痛', '发烧', '咳嗽']
```
