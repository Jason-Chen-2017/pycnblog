## 4. 数学模型和公式详细讲解举例说明

### 4.1 商品相似度计算

可以使用余弦相似度来计算商品之间的相似度：

$$
similarity(A, B) = \frac{A \cdot B}{||A|| \cdot ||B||}
$$

其中，$A$ 和 $B$ 分别表示两个商品的特征向量，$\cdot$ 表示向量内积，$||A||$ 和 $||B||$ 分别表示向量 $A$ 和 $B$ 的模。

```python
def search_products(keyword):
    # 对关键词进行分词
    words = tokenize(keyword)
    # 查询数据库，匹配商品信息
    products = Product.objects.filter(name__icontains=keyword)
    # 对结果进行相关性排序
    products = sorted(products, key=lambda p: relevance(p, words), reverse=True)
    return products
```
