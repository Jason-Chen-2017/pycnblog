# 集合论导引：莱维力迫扩张

## 1. 背景介绍
### 1.1  问题的由来
集合论作为现代数学的基础,在数学逻辑、拓扑学、代数学等众多数学分支中有着广泛而深刻的应用。然而,康托尔创立的朴素集合论却导致了数学中的悖论和矛盾。为了解决这些问题,数学家们提出了多种公理化集合论体系,其中最著名的当属策梅洛-弗兰克尔集合论(ZFC)。

在ZFC的框架下,数学家们研究了许多有趣而深刻的问题,其中一个重要的课题就是基数算术。基数算术研究无穷集合的大小比较和运算。然而,在ZFC中,一些基本的问题如连续统假设(CH)却无法被证明或证伪。这启发数学家去研究ZFC的扩张,看看在更强的公理体系下这些问题是否可解。

### 1.2  研究现状
在20世纪60年代,数学家们发现了一类重要的扩张方法,即力迫扩张。通过在ZFC的基础上添加力迫公理,可以得到ZFC的保守扩张。在力迫扩张的模型中,许多在ZFC下独立的命题如CH有了确定的真值。因此,力迫法成为集合论研究的一个有力工具。

以色列数学家阿齐里尔·莱维(Azriel Lévy)和美国数学家罗伯特·索洛维(Robert M. Solovay)在20世纪60年代提出了一种特殊的力迫扩张,称为莱维力迫扩张或莱维-索洛维力迫扩张。莱维力迫扩张在集合论和模型论的研究中有着重要的应用。

### 1.3  研究意义  
莱维力迫扩张对集合论和数学基础有着重要的意义:

1. 莱维力迫扩张提供了一种研究ZFC独立问题的有力工具,使我们对集合论中的困难问题有了新的认识。

2. 莱维力迫扩张揭示了ZFC的模型的多样性,加深了我们对集合论模型论的理解。 

3. 莱维力迫扩张与描述集论和递归论有着密切的联系,推动了这些领域的发展。

4. 莱维力迫扩张启发了其他力迫扩张方法的研究,极大地拓展了数学家的研究视野。

## 2. 核心概念与联系

在介绍莱维力迫扩张之前,我们先回顾一些集合论的基本概念。

- **ZFC公理**：策梅洛-弗兰克尔集合论(Zermelo-Fraenkel set theory with the axiom of choice)是现代集合论的标准公理系统,包括外延性公理、空集公理、配对公理、并集公理、幂集公理、无穷公理、分离公理、替换公理和选择公理。

- **序数和基数**：序数是良序集同构类构成的集合,用希腊字母$\alpha,\beta,\gamma$等表示。基数刻画了集合的大小,对任意集合$X$,存在唯一的基数$|X|$与$X$对等。

- **连续统假设(CH)**：实数集的基数$2^{\aleph_0}$等于第一个不可数基数$\aleph_1$。CH在ZFC中独立,无法被证明或证伪。

- **构造性公理**：在ZFC的基础上添加一些公理,可以得到ZFC的扩张。常见的构造性公理有巨基数公理、$\diamondsuit$原理、马丁公理等。

- **内模型和外模型**：如果$M$是$V$的一个传递子类,且$M$对ZFC的每条公理都是绝对的,那么称$M$是ZFC的一个内模型。相应地,$V$称为$M$的外模型。

- **超限递归**：在序数的类上定义的函数称为超限递归函数。超限递归理论是集合论的重要分支,与描述集论和递归论有着密切联系。

- **鲁棒性**：一个数学概念在力迫扩张下保持不变的性质称为鲁棒性。研究概念的鲁棒性是集合论的重要课题之一。

莱维力迫扩张利用超限递归函数对ZFC模型进行扩张,得到ZFC的保守扩张。同时,莱维力迫扩张保证了一些重要概念的鲁棒性。因此,莱维力迫扩张成为研究ZFC独立性问题和模型论的重要工具。

## 3. 核心算法原理 & 具体操作步骤
### 3.1  算法原理概述
莱维力迫扩张的基本思想是在给定的ZFC模型$M$上,利用超限递归函数对$M$中的集合进行编码,得到$M$的一个扩张模型$N$。具体来说,莱维力迫扩张通过以下步骤实现：

1. 固定一个超限递归函数$F:Ord^{<\omega}\to Ord$。
2. 对$M$中的每个集合$x$,定义它的基本码$\ulcorner x\urcorner=\{\ulcorner y\urcorner:y\in x\}$。
3. 对每个序数$\alpha\in M$,定义$\alpha$的次码为$F$在$\alpha$上的函数值$F(\alpha)$。
4. 定义扩张宇宙$N=\{x\in M:x\subseteq \bigcup_{\alpha\in Ord^M}V_\alpha^M\}$。

可以证明,对任意的超限递归函数$F$,由莱维力迫得到的扩张模型$N$都是ZFC的模型,且$N$与原模型$M$有相同的序数。因此,莱维力迫实现了ZFC的保守扩张。

### 3.2  算法步骤详解

下面我们详细介绍莱维力迫扩张的具体步骤。

**步骤1：选取超限递归函数$F$**

莱维力迫扩张的第一步是选取一个合适的超限递归函数$F:Ord^{<\omega}\to Ord$。通常我们要求$F$满足以下条件：

1. $F$在序数的有穷序列上定义,且值域为序数。
2. $F$是非减的,即对任意$\vec{\alpha},\vec{\beta}\in Ord^{<\omega}$,如果$\vec{\alpha}\subseteq\vec{\beta}$,则$F(\vec{\alpha})\leq F(\vec{\beta})$。
3. $F$是连续的,即对任意的$\delta\in Lim$,有$F(\delta)=\sup\{F(\alpha):\alpha<\delta\}$。

满足以上条件的超限递归函数称为良好的编码函数。

**步骤2：定义基本码**

对$M$中的每个集合$x$,我们定义它的基本码$\ulcorner x\urcorner$如下：

$$\ulcorner x\urcorner=\{\ulcorner y\urcorner:y\in x\}$$

直观地说,集合$x$的基本码就是$x$中所有元素的基本码的集合。注意到基本码的定义是递归的,因此需要用超限递归来定义。

**步骤3：定义次码**

利用给定的超限递归函数$F$,我们可以为每个序数$\alpha\in M$定义一个次码$F(\alpha)$。直观地说,次码反映了序数$\alpha$在函数$F$下的编码。

**步骤4：定义扩张宇宙**

有了基本码和次码的概念,我们就可以定义莱维力迫扩张得到的新模型$N$：

$$N=\{x\in M:x\subseteq \bigcup_{\alpha\in Ord^M}V_\alpha^M\}$$

其中$V_\alpha^M$表示$M$中阶高为$\alpha$的累积层次宇宙。直观地说,$N$是由$M$中所有的集合构成的,但是这些集合的元素必须来自某个$V_\alpha^M$。

**步骤5：验证$N$为ZFC模型**

最后一步是证明$N$构成ZFC的一个模型,这需要验证ZFC的各条公理在$N$中都成立。同时还要证明$N$是$M$的保守扩张,即$N$与$M$有相同的序数,且$M$中成立的$\Sigma_1$语句在$N$中也成立。

莱维力迫扩张的证明用到了大量的集合论和模型论技巧,是集合论中的一个重要结果。

### 3.3  算法优缺点

莱维力迫扩张作为一种重要的模型构造方法,有以下优点：

1. 莱维力迫提供了一种构造ZFC保守扩张的一般方法,使我们能够得到多种不同性质的模型。

2. 莱维力迫保证了扩张模型与原模型有相同的序数,这使得我们能够在扩张模型中研究序数的性质。

3. 莱维力迫得到的模型保持了许多重要概念的鲁棒性,如projective sets、Lebesgue measurability等。

4. 莱维力迫与描述集论和递归论有着密切的联系,推动了这些领域的发展。

当然,莱维力迫也有一些局限性：

1. 莱维力迫得到的模型依赖于所选取的超限递归函数,因此并非所有的莱维力迫扩张都有好的性质。

2. 在莱维力迫扩张中证明相关结果需要复杂的技巧,对初学者来说比较困难。

3. 莱维力迫并不能解决所有的独立性问题,有些问题在莱维力迫扩张下仍然独立于ZFC。

莱维力迫扩张在集合论和数学逻辑中有广泛的应用,主要有以下几个方面：

1. 研究ZFC的独立性问题,如CH、Suslin假设等。

2. 研究ZFC模型的结构理论,如inner model problem等。
