### 4.1 无穷小量的代码实例

以下是一个使用Python实现的无穷小量的代码实例。这个代码实例计算了当一个圆的半径趋近于无穷大时，圆的周长和直径之比趋近于圆周率$\pi$的过程。

```python
import numpy as np

def circle_circumference_ratio(radius):
    circumference = 2 * np.pi * radius
    diameter = 2 * radius
    return circumference / diameter

radii = np.linspace(1, 1e6, 100)
ratios = [circle_circumference_ratio(r) for r in radii]

print("圆的半径趋近于无穷大时，周长与直径之比趋近于：", np.mean(ratios))
```
