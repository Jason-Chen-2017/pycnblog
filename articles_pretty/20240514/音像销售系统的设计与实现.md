$$
cos(X,Y) = \frac{X * Y}{||X||_2 * ||Y||_2}

以 Java 为例，我们可以使用 Spring Boot 框架来快速开发后端服务。以下是一个简单的用户管理功能的代码示例：

```java
@RestController
@RequestMapping("/user")
public class UserController {
    @Autowired
    private UserService userService;

    @PostMapping("/register")
    public Result register(@RequestBody User user) {
        userService.register(user);
        return Result.success();
    }

```

这段代码中，`@RestController` 是一个 Spring 的注解，表示这是一个 RESTful 风格的控制器。`@RequestMapping("/user")` 表示这个控制器处理的请求的 URL 路径是 "/user"。`@Autowired` 是 Spring 的一个注解，用来自动装配依赖的 Bean。
