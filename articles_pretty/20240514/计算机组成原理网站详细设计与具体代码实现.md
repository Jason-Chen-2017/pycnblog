* **存储器：** 用于存储程序和数据的部件，包括内存和外存。
* **输入/输出设备：** 用于与外部世界交互的设备，例如键盘、鼠标、显示器、打印机等。

## 4. 数学模型和公式详细讲解举例说明

### 4.1  CPU 指令执行模型

CPU 指令执行模型可以用以下公式表示：

$$
\text{指令周期} = \text{取指周期} + \text{译码周期} + \text{执行周期} + \text{访存周期} + \text{写回周期}
$$

其中：

* **取指周期：** 从内存中读取指令。
* **译码周期：** 将指令解析成 CPU 可以理解的操作码和操作数。
* **执行周期：** 执行指令的操作。
* **访存周期：** 访问内存读取或写入数据。
* **写回周期：** 将执行结果写回内存或寄存器。

### 4.2  内存寻址模型

内存寻址模型可以使用以下公式表示：

$$
\text{物理地址} = \text{段基址} + \text{偏移地址}
$$

其中：

* **物理地址：** 内存单元的实际地址。
* **段基址：**  内存段的起始地址。
* **偏移地址：** 相对于段基址的地址偏移量。
