## 4.具体最佳实践：代码实例和详细解释说明

### 4.1 安装Docker

在Ubuntu系统上，我们可以使用以下命令来安装Docker：

```bash
sudo apt-get update
sudo apt-get install docker-ce
```

### 4.2 构建Docker镜像

首先，我们需要创建一个Dockerfile。Dockerfile是一个文本文件，它描述了如何构建Docker镜像。以下是一个简单的Dockerfile示例：

```Dockerfile
# 使用官方Python运行时作为父镜像
FROM python:3.7-slim

# 设置工作目录
WORKDIR /app

# 将当前目录内容复制到容器的/app目录中
ADD . /app

# 安装所需的包
RUN pip install --no-cache-dir -r requirements.txt

# 对外暴露端口
EXPOSE 5000

# 运行app.py
CMD ["python", "app.py"]
```

然后，我们可以使用以下命令来构建Docker镜像：

```bash
docker build -t text-processing-service .
```

### 4.3 运行Docker容器

最后，我们可以使用以下命令来运行Docker容器：

```bash
docker run -p 5000:5000 text-processing-service
```

这条命令会启动一个新的Docker容器，并将容器的5000端口映射到主机的5000端口。
