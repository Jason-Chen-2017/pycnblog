## 4. 数学模型和公式详细讲解举例说明

### 4.1 GAN的损失函数
#### 4.1.1 生成器损失函数
$$\min_G \max_D V(D,G) = \mathbb{E}_{x \sim p_{data}(x)}[\log D(x)] + \mathbb{E}_{z \sim p_z(z)}[\log (1 - D(G(z)))]$$
#### 4.1.2 判别器损失函数 
$$\max_D V(D,G) = \mathbb{E}_{x \sim p_{data}(x)}[\log D(x)] + \mathbb{E}_{z \sim p_z(z)}[\log (1 - D(G(z)))]$$
#### 4.1.3 GAN训练的minimax game

### 4.2 AdaIN的数学表示
#### 4.2.1 Instance Normalization
$$IN(x) = \gamma (\frac{x - \mu(x)}{\sigma(x)}) + \beta$$
#### 4.2.2 AdaIN的仿射变换参数
$$AdaIN(x, y) = \sigma(y) (\frac{x - \mu(x)}{\sigma(x)}) + \mu(y)$$
#### 4.2.3 AdaIN在风格迁移中的作用

### 4.3 ALAE的潜在空间操作
#### 4.3.1 潜在空间插值
$z = \alpha z_1 + (1 - \alpha) z_2, \alpha \in [0, 1]$
#### 4.3.2 潜在空间算术运算
$z_{new} = z_1 + z_2 - z_3$
#### 4.3.3 潜在空间方向操作
$z_{new} = z + \alpha d, \alpha \in \mathbb{R}$
