### 5.2 业务逻辑的代码示例

下面是一个简单的业务逻辑的代码示例：

```python
# 计算员工的实际工资
def calculate_salary(employee):
    # 计算基本工资
    base_salary = employee.hours * employee.rate

    # 计算加班费
    overtime = max(0, employee.hours - 40) * employee.rate * 0.5

    # 计算奖金
    bonus = employee.sales * 0.01

    # 计算津贴
    allowance = 500 if employee.position == 'manager' else 0

    # 计算扣款
    deduction = employee.absence * 100

    # 计算实际工资
    salary = base_salary + overtime + bonus + allowance - deduction

    return salary
```
