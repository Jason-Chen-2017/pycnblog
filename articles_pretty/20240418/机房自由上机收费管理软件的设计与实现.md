```python
class User:
    def __init__(self, name, password):
        self.name = name
        self.password = password
        self.login_time = None
        self.logout_time = None

    def login(self):
        self.login_time = time.time()

    def logout(self):
        self.logout_time = time.time()

class Fee:
    def __init__(self, rate):
        self.rate = rate

    def calculate(self, user):
        usage_time = user.logout_time - user.login_time
        fee = self.rate * usage_time
        return fee
```

在这个代码实例中，我们首先定义了一个用户类（User），这个类包含了用户的姓名、密码、登陆时间和退出时间等属性，同时还包含了用户登陆和退出的方法。然后，我们定义了一个费用类（Fee），这个类包含了计费率的属性，同时还包含了计费的方法。这个方法首先计算用户的使用时间，然后根据计费率进行计算，最后返回计费结果。
