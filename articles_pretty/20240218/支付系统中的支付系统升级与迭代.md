### 4.1 支付系统的代码实例

下面是一个简单的支付系统的代码实例，使用Python语言编写。这个代码实例实现了一个简单的支付请求处理流程。

```python
class PaymentRequest:
    def __init__(self, amount, method):
        self.amount = amount
        self.method = method

class PaymentProcessor:
    def process(self, request):
        if request.method == 'credit_card':
            self._process_credit_card(request)
        elif request.method == 'paypal':
            self._process_paypal(request)
        else:
            raise ValueError('Invalid payment method')

    def _process_credit_card(self, request):
        print(f'Processing credit card payment for {request.amount}')

    def _process_paypal(self, request):
        print(f'Processing PayPal payment for {request.amount}')

request = PaymentRequest(100, 'credit_card')
processor = PaymentProcessor()
processor.process(request)
```

### 4.2 代码实例的详细解释说明

这个代码实例中，`PaymentRequest`类代表一个支付请求，包含支付金额和支付方式两个属性；`PaymentProcessor`类代表一个支付处理器，包含处理支付请求的方法。在处理支付请求时，根据支付方式的不同，调用不同的处理方法。
