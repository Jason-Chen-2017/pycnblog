下面给出一个基于PyTorch的多任务学习模型的代码实现示例:

```python
import torch.nn as nn
import torch.optim as optim

class MultiTaskModel(nn.Module):
    def __init__(self, input_dim, task_num):
        super(MultiTaskModel, self).__init__()
        self.shared_layers = nn.Sequential(
            nn.Linear(input_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 128),
            nn.ReLU()
        )
        
        self.task_specific_layers = nn.ModuleList([
            nn.Sequential(
                nn.Linear(128, 64),
                nn.ReLU(),
                nn.Linear(64, 1)
            ) for _ in range(task_num)
        ])
        
    def forward(self, x):
        shared_features = self.shared_layers(x)
        task_outputs = [task_layer(shared_features) for task_layer in self.task_specific_layers]
        return task_outputs
```

在训练过程中,我们可以采用联合优化的策略,同时优化各个任务的损失函数,利用任务之间的相关性来提高整体性能。
