## 4. 具体最佳实践：代码实例和详细解释说明

以智能洗衣机产品为例,说明如何基于前述的建模和推理方法,构建其知识图谱并进行不确定性分析。

```python
import numpy as np
from scipy.spatial.distance import cdist

# 定义洗衣机实体及其属性
entities = ['LG_WashingMachine_Model1', 'Haier_WashingMachine_Model2', 'Siemens_WashingMachine_Model3']
attributes = ['Capacity', 'EnergyEfficiency', 'NoiseLevel', 'SpinSpeed']

# 构建属性矩阵
attr_matrix = np.array([[8.0, 'A', 48, 1400], 
                       [7.5, 'A+', 52, 1200],
                       [9.0, 'A++', 45, 1600]])

# 计算洗衣机之间的相似度
sim_matrix = 1 - cdist(attr_matrix, attr_matrix, metric='cosine')

# 基于相似度构建知识图谱
import networkx as nx
G = nx.Graph()
G.add_nodes_from(entities)
for i in range(len(entities)):
    for j in range(i+1, len(entities)):
        G.add_edge(entities[i], entities[j], weight=sim_matrix[i,j])

# 可视化知识图谱
import matplotlib.pyplot as plt
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True, node_color='lightblue', edge_color='gray', width=2)
plt.show()
```

通过上述代码,我们构建了一个简单的智能洗衣机知识图谱,并计算了不同型号之间的相似度。这为后续的不确定性建模和推理分析奠定了基础。
