### 3.1 微积分在计算美学中的应用

在计算美学中，微积分被用于描述形状的变化、计算曲线的长度、求解积分等问题。例如，在计算曲线长度时，可以使用微积分的积分公式：

$$
\int_{a}^{b} \sqrt{1 + (y'(x))^2} dx
$$

其中，$y'(x)$ 表示曲线的斜率，$a$ 和 $b$ 是曲线的起点和终点。

### 3.2 微积分在视觉感知中的应用

在视觉感知中，微积分被用于计算图像的边缘、分割、识别等问题。例如，在计算图像边缘时，可以使用微积分的梯度公式：

$$
G(x, y) = \sqrt{(G_x(x, y))^2 + (G_y(x, y))^2}
$$

其中，$G_x(x, y)$ 和 $G_y(x, y)$ 是图像梯度的 x 和 y 分量。

### 4.1 计算曲线长度的代码实例

```python
import numpy as np

def curve_length(a, b, y):
    x = np.linspace(a, b, 1000)
    y = y(x)
    dy = np.diff(y)
    return np.sqrt(np.sum(dy**2))

def example_function(x):
    return np.sin(x)

print(curve_length(0, 2*np.pi, example_function))
```

### 4.2 计算图像边缘的代码实例

```python
import cv2
import numpy as np

def edge_detection(image):
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    grad_x = cv2.Sobel(gray, cv2.CV_64F, 1, 0, ksize=5)
    grad_y = cv2.Sobel(gray, cv2.CV_64F, 0, 1, ksize=5)
    gradient_magnitude = np.sqrt(grad_x**2 + grad_y**2)
    return gradient_magnitude

edge_image = edge_detection(image)
cv2.imshow('Edge Image', edge_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
