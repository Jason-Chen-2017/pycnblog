### 4.1 数据分析

假设我们有一个CSV文件，包含一些商品的价格和销量数据。我们可以使用Pandas库来读取这个文件、计算平均价格和总销量等。

```python
import pandas as pd

# 读取CSV文件
df = pd.read_csv('goods.csv')

# 计算平均价格
average_price = df['price'].mean()

# 计算总销量
total_sales = df['sales'].sum()

print('平均价格：', average_price)
print('总销量：', total_sales)
```

### 4.2 网络爬虫

假设我们想要抓取一个新闻网站上的新闻标题和摘要。我们可以使用Scrapy库来开发一个简单的网络爬虫。

```python
import scrapy

class NewsSpider(scrapy.Spider):
    name = 'news'
    start_urls = ['https://news.baidu.com/']

    def parse(self, response):
        # 提取新闻标题和摘要
        for news in response.css('div.news-item'):
            title = news.css('h3.title a::text').get()
            abstract = news.css('p.abstract::text').get()
            yield {'title': title, 'abstract': abstract}
```
