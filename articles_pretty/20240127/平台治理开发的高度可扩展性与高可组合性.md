class ConsistentHash:
    def __init__(self, nodes):
        self.nodes = nodes
        self.hash = {}

    def add_node(self, node):
        self.nodes.add(node)

    def remove_node(self, node):
        self.nodes.remove(node)

    def register(self, key):
        node = self.hash.get(key)
        if node:
            return node
        else:
            for i in range(len(self.nodes)):
                node = self.nodes[(i + hash(key)) % len(self.nodes)]
                if node not in self.hash:
                    self.hash[key] = node
                    return node
            return self.nodes[0]

## 8. 附录：常见问题与解答

1. **Q：什么是一致性哈希？**

   **A：**一致性哈希是一种分布式系统算法，它可以帮助企业实现高效的数据分布和负载均衡。它的核心思想是将数据块映射到一个虚拟环，然后将虚拟环中的节点映射到实际的节点上，从而实现数据的分布和负载均衡。

2. **Q：什么是B+树？**

   **A：**B+树是一种多路搜索树，它的叶子节点存储了有序的关键字和指向子节点的指针。B+树的特点是它的所有叶子节点都具有相同的深度，这使得B+树具有较好的查询性能。

3. **Q：什么是Bloom过滤器？**

   **A：**Bloom过滤器是一种概率数据结构，它可以用来判断一个元素是否在一个集合中。Bloom过滤器的特点是它可以在空间和时间上达到很高的效率，但是它可能会产生一定的错误概率。
