- 使用知识图谱来提高地理信息系统的效率和准确性，例如使用知识图谱来优化查询、分析、可视化等。

代码实例：

```python
import geopy.distance

# 定义两个地点
point1 = geopy.Point(39.9042, 116.4074)
point2 = geopy.Point(22.5451, 114.0594)

# 计算两个地点之间的距离
distance = geopy.distance.distance(point1, point2).km
print(f"两个地点之间的距离是：{distance} 公里")
```

详细解释说明：

- 使用geopy库来计算两个地点之间的距离。
- 将两个地点转换为geopy.Point对象。
- 使用geopy.distance.distance函数来计算两个地点之间的距离，并将距离以公里为单位输出。
