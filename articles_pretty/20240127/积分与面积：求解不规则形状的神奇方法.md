import numpy as np

def triangle_area(base, height):
    return 0.5 * base * height

def trapezoid_area(base1, base2, height):
    return 0.5 * (base1 + base2) * height

def polygon_area(vertices):
    area = 0.0
    for i in range(len(vertices) - 1):
        area += 0.5 * (vertices[i][0] * vertices[i + 1][1] - vertices[i + 1][0] * vertices[i][1])
    area += 0.5 * vertices[-1][0] * vertices[0][1]
    return abs(area)

# 三角形

Q: 积分方法在计算不规则形状的面积时，有哪些优势和局限性？

A: 积分方法在计算不规则形状的面积时具有以下优势：

- 灵活性：可以应用于各种形状，包括简单的形状（如三角形、梯形）和复杂的形状。
- 准确性：可以得到较为准确的面积结果，尤其是在使用数值积分方法时。

然而，积分方法也有一些局限性：

- 复杂性：对于复杂的形状，积分方法可能需要较复杂的计算过程。
- 精度：在使用直接积分和累积积分时，可能会出现精度问题。使用数值积分方法可以提高精度，但可能会增加计算复杂度。

Q: 如何选择适合不规则形状面积计算的积分方法？

A: 在选择适合不规则形状面积计算的积分方法时，需要考虑以下因素：

- 形状复杂程度：对于简单的形状，直接积分和累积积分方法可能足够。对于复杂的形状，可能需要使用数值积分方法。
- 精度要求：根据精度要求选择合适的积分方法。直接积分和累积积分方法可能不够精确，需要使用数值积分方法。
- 计算复杂程度：考虑使用简单的积分方法，以减少计算复杂程度。如果计算复杂程度较高，可以考虑使用数值积分方法。
