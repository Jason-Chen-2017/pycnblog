## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 代码实例1：Python基本数据类型

```python
# 整数
a = 10
print(type(a))  # <class 'int'>

# 浮点数
b = 3.14
print(type(b))  # <class 'float'>

# 字符串
c = "Hello, World!"
print(type(c))  # <class 'str'>

# 列表
d = [1, 2, 3, 4, 5]
print(type(d))  # <class 'list'>

# 字典
e = {"name": "Alice", "age": 25}
print(type(e))  # <class 'dict'>

# 集合
f = {1, 2, 3, 4, 5}
print(type(f))  # <class 'set'>
```

### 4.2 代码实例2：Python函数

```python
# 定义函数
def add(a, b):
    return a + b

# 调用函数
result = add(10, 20)
print(result)  # 30
```

### 4.3 代码实例3：Python模块

```python
# 导入模块
import math

# 使用模块
result = math.sqrt(16)
print(result)  # 4.0
```

### 4.4 代码实例4：Python类

```python
# 定义类
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Hello, my name is {self.name} and I am {self.age} years old.")

# 创建对象
person = Person("Alice", 25)

# 调用方法
person.greet()
```

Q：Python中的整数和浮点数有什么区别？

A：Python中的整数和浮点数的区别在于，整数是一种不包含小数部分的数值类型，而浮点数是一种包含小数部分的数值类型。

Q：Python中的列表和字典有什么区别？

A：Python中的列表和字典的区别在于，列表是一种可变序列类型，可以用来存储多个值，而字典是一种键值对类型，可以用来存储多个键值对。

Q：Python中的类和对象有什么区别？

A：Python中的类和对象的区别在于，类是一种模板，用来定义对象的属性和方法，而对象是类的实例，用来存储属性和方法的具体值。

Q：Python中的函数和模块有什么区别？

A：Python中的函数和模块的区别在于，函数是一种代码块，可以用来完成特定的任务，而模块是一种.py文件，可以用来定义函数和变量。
