## 5. 具体最佳实践：代码实例和详细解释说明

以下是一个Go语言实现的简单网络扫描示例：

```go
package main

import (
	"fmt"
	"net"
	"time"
)

func main() {
	start := time.Now()
	fmt.Println("Start scanning at", start.Format("2006-01-02 15:04:05"))

	for i := 1; i <= 100; i++ {
		ip := fmt.Sprintf("192.168.1.%d", i)
		conn, err := net.DialTimeout("tcp", ip+":80", time.Second*5)
		if err != nil {
			fmt.Printf("Failed to connect to %s, error: %v\n", ip, err)
		} else {
			defer conn.Close()
			fmt.Printf("Successfully connected to %s\n", ip)
		}
	}

	end := time.Now()
	fmt.Println("End scanning at", end.Format("2006-01-02 15:04:05"))
	fmt.Printf("Total time: %v\n", end.Sub(start))
}
```

这个示例程序会扫描192.168.1.1到192.168.1.100的IP地址，尝试连接每个IP地址的80端口。如果连接成功，则输出成功的IP地址，如果失败，则输出错误信息。
