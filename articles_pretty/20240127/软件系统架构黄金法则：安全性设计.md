### 4.3 访问控制

在系统中，访问控制是一项重要的安全性措施。以下是一个使用基于角色的访问控制（RBAC）的代码实例：

```python
class Role:
    def __init__(self, name, permissions):
        self.name = name
        self.permissions = permissions

class User:
    def __init__(self, name, roles):
        self.name = name
        self.roles = roles

def has_permission(user, permission):
    for role in user.roles:
        if permission in role.permissions:
            return True
    return False

user = User('Alice', [Role('Admin', ['read', 'write', 'delete']), Role('User', ['read'])])
print(has_permission(user, 'read'))  # True
print(has_permission(user, 'write'))  # True
print(has_permission(user, 'delete'))  # True
print(has_permission(user, 'update'))  # False
```

在这个例子中，我们使用了基于角色的访问控制（RBAC）来控制用户对系统资源的访问。我们定义了Role和User类，并实现了一个has_permission函数来检查用户是否具有某个权限。
