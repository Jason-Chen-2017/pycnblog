## 2. 核心概念与联系
知识图谱是一种基于图的数据库，可以存储和管理大量的实体和关系。实体是知识图谱中的基本单位，例如人、组织、地点等。关系则是实体之间的联系，例如属于、成立在等。通过建立这些实体和关系的联系，我们可以构建一个代表政治与国际关系领域知识的图。

在政治与国际关系领域，知识图谱可以帮助我们解决以下问题：

- 政治体系的分类和比较
- 国际组织的成员和活动
- 国际贸易和投资的规则和协议
- 国际事件和冲突的预测

以下是一个简单的代码实例，用于计算两个实体之间的距离：

```python

# 创建一个图
G = nx.Graph()

# 添加实体和关系
G.add_node("中国")
G.add_node("美国")
G.add_edge("中国", "美国", weight=10)

# 计算距离
distance = shortest_path_length(G, "中国", "美国")

print(distance)
```

在这个例子中，我们首先创建了一个图，然后添加了两个实体（中国和美国）和一个关系（中国与美国之间的距离为10）。最后，我们使用了Shortest Path Algorithm来计算两个实体之间的距离。
