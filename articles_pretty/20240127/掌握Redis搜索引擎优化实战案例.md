## 4. 具体最佳实践：代码实例和详细解释说明

在实际应用中，我们可以使用以下代码实例来优化搜索引擎性能：

```python
import redis

# 创建Redis连接
r = redis.StrictRedis(host='localhost', port=6379, db=0)

# 存储文档元数据
r.hmset('doc1', 'title', 'Redis搜索引擎优化', 'summary', 'Redis是一个高性能的键值存储系统')

# 存储搜索关键词
r.zadd('keywords', {'redis': 1, 'search': 1, 'engine': 1, 'optimization': 1})

# 存储文档相关度信息
r.zadd('doc_scores', {'doc1': 100})

# 搜索关键词
keywords = r.zrange('keywords', 0, -1)

# 获取文档相关度信息
doc_scores = r.zrange('doc_scores', 0, -1)

# 计算相关度
relevance = {}
for keyword in keywords:
    score = doc_scores[0]
    if keyword in keyword:
        score += 1
    relevance[keyword] = score

# 排序相关度
sorted_relevance = sorted(relevance.items(), key=lambda x: x[1], reverse=True)

# 输出搜索结果
for keyword, score in sorted_relevance:
    print(f'{keyword}: {score}')
```

在这个代码实例中，我们使用了Redis的哈希、ZSet和排序功能来实现搜索引擎优化。我们首先存储了文档元数据、搜索关键词和文档相关度信息。然后，我们使用了ZSet的范围查询功能来获取搜索关键词和文档相关度信息。最后，我们使用了排序功能来计算和排序相关度。
