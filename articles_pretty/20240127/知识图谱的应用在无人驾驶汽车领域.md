## 4. 具体最佳实践：代码实例和详细解释说明
以下是一个简单的代码实例，用于构建道路网络模型：

```python
import networkx as nx

# 创建一个有向图
G = nx.DiGraph()

# 添加道路节点
G.add_node("A")
G.add_node("B")
G.add_node("C")

# 添加道路边缘
G.add_edge("A", "B", weight=10)
G.add_edge("B", "C", weight=15)

# 计算最短路径
shortest_path = nx.shortest_path(G, source="A", target="C")
print(shortest_path)
```

这个代码实例使用了Python的networkx库，创建了一个有向图，并添加了道路节点和边缘。最后，计算了从节点A到节点C的最短路径。
