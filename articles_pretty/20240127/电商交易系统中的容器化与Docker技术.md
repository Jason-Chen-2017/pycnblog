### 4.1 代码实例

以下是一个使用Docker技术在电商交易系统中运行应用程序的代码实例：

```bash
$ docker run -d -p 8080:80 --name myapp myapp
```

在这个代码实例中，我们使用`docker run`命令运行一个名为`myapp`的Docker容器，并将其映射到主机的8080端口上。

### 4.2 详细解释说明

在这个代码实例中，我们使用`docker run`命令运行一个名为`myapp`的Docker容器。具体来说，我们使用以下参数：

- `-d`：后台运行容器。
- `-p 8080:80`：将容器的80端口映射到主机的8080端口。
- `--name myapp`：为容器命名。

通过这个代码实例，我们可以看到如何使用Docker技术在电商交易系统中运行应用程序。
