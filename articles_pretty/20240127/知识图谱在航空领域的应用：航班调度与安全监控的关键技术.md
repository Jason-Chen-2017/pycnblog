## 4. 具体最佳实践：代码实例和详细解释说明
### 4.1 代码实例
以下是一个使用 Python 语言编写的知识图谱查询和推理示例：

```python
from scipy.sparse import csr_matrix
from sklearn.metrics.pairwise import cosine_similarity

# 创建图谱数据
data = {
    '航空公司': ['Air China', 'China Southern', 'China Eastern'],
    '航空机': ['B737', 'A320', 'A380'],
    '航空站': ['PEK', 'CAN', 'PVG'],
    '航空路线': ['PEK-CAN', 'PEK-PVG', 'CAN-PVG']
}

# 创建图谱矩阵
graph = csr_matrix((data['航空路线'].count(i) for i in data['航空路线']), shape=(len(data['航空公司']), len(data['航空站'])))

# 计算航空公司之间的相似度
similarity = cosine_similarity(graph)

# 打印结果
print(similarity)
```

### 4.2 详细解释说明
上述代码首先创建了图谱数据，包括航空公司、航空机、航空站和航空路线等实体。然后创建了图谱矩阵，矩阵的行表示航空公司，列表示航空站，矩阵的元素表示航空公司与航空站之间的关系。最后使用 `cosine_similarity` 函数计算航空公司之间的相似度，并打印结果。
