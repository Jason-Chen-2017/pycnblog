## 4. 具体最佳实践：代码实例和详细解释说明

在矿业领域，知识图谱技术可以用于勘探和开采规划的各个环节，如资源评估、开采计划、环境影响等。以下是一个具体的代码实例，展示了如何使用知识图谱技术进行资源评估：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 加载资源数据
resources = ["地质数据", "气候数据", "地形数据", "基础设施数据"]

# 构建词向量
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(resources)

# 计算资源之间的相似度
similarity = cosine_similarity(X)

# 打印资源之间的相似度
print(similarity)
```

在这个代码实例中，我们使用了TF-IDF向量化技术，将资源数据转换为向量，并计算了资源之间的相似度。这个相似度可以用于资源评估，帮助勘探和开采工程师更好地理解和利用资源数据。
