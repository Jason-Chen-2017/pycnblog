在ROS中，开发机器人对话系统的最佳实践可以参考以下代码实例：

```python
#!/usr/bin/env python

import rospy
from std_msgs.msg import String

def callback(data):
    rospy.loginfo(rospy.get_caller_id() + ' I heard %s', data.data)

def listener():
    rospy.init_node('listener', anonymous=True)
    rospy.Subscriber('chatter', String, callback)
    rospy.spin()

if __name__ == '__main__':
    listener()
```

在这个代码实例中，我们首先导入了ROS的相关库。接下来，我们定义了一个名为`callback`的函数，这个函数将接收来自`chatter`话题的消息。在这个函数中，我们使用了ROS的`loginfo`函数来记录接收到的消息。最后，我们定义了一个名为`listener`的函数，这个函数将初始化ROS节点，并订阅`chatter`话题。
