## 4. 具体最佳实践：代码实例和详细解释说明
在实际应用中，我们可以使用以下代码实例来构建建筑物信息的图结构：

```python
from scipy.sparse import csr_matrix
import networkx as nx

# 创建一个空的有向图
G = nx.DiGraph()

# 创建一个空的稀疏矩阵
A = csr_matrix((0, 0), shape=(10, 10))

# 添加建筑物信息
G.add_node("A", name="建筑物A", structure="玻璃瓶结构", material="玻璃")
G.add_node("B", name="建筑物B", structure="钢结构", material="钢")
G.add_node("C", name="建筑物C", structure="混凝土结构", material="混凝土")

# 添加关系
G.add_edge("A", "B", weight=0.8)
G.add_edge("B", "C", weight=0.9)
G.add_edge("C", "A", weight=0.7)

# 添加属性
A[0, 1] = 0.8
A[1, 2] = 0.9
A[2, 0] = 0.7

# 打印结果
print(G.nodes())
print(G.edges())
print(A)
```

在这个代码实例中，我们首先创建了一个空的有向图和一个空的稀疏矩阵。然后，我们添加了建筑物信息，并添加了关系和属性。最后，我们打印了结果。
