## 3. 核心算法原理和具体操作步骤以及数学模型公式详细讲解

三角积分方法的算法原理是基于三角函数的性质。具体操作步骤如下：

1. 将积分区间划分为多个等分区间。
2. 在每个区间内，使用三角函数进行积分。
3. 将每个区间内的积分结果相加，得到整个积分的结果。

数学模型公式如下：

$$
\int_{a}^{b} f(x) dx = \sum_{i=1}^{n} \Delta x_i f(x_i)
$$

其中，$n$ 是区间划分的次数，$\Delta x_i$ 是每个区间的长度，$x_i$ 是每个区间内的一个点。

```python
import numpy as np

def f(x):
    return np.exp(-x**2)

a = 0
b = 1
n = 1000
x = np.linspace(a, b, n)
dx = b - a

y = f(x)
area = dx * np.trapz(y, dx)

print("积分结果:", area)
```

在上述代码中，我们首先定义了一个函数$f(x)$，然后使用`numpy`库中的`trapz`函数计算积分。`trapz`函数接受积分区间、积分区间内的函数值和区间长度作为参数，并返回积分结果。
