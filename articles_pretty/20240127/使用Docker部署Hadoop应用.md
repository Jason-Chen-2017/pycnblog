Hadoop主要由以下两个核心组件构成：

- **Hadoop Distributed File System (HDFS)：**HDFS是Hadoop的分布式文件系统，它可以存储大量数据并提供高性能和可扩展性。HDFS将数据分成多个块，并在多个数据节点上存储，以便在多个节点之间分布式计算。

- **MapReduce：**MapReduce是Hadoop的分布式计算框架，它可以处理大量数据并提供高性能和可扩展性。MapReduce将数据分成多个部分，并在多个节点上进行并行处理。
