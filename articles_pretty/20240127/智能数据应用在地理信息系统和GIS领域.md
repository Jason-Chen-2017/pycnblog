数学模型公式详细讲解：

- **线性回归**：用于预测连续变量的模型。公式为：$y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon$。
- **逻辑回归**：用于预测类别变量的模型。公式为：$P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n)}}$。
- **支持向量机**：用于解决线性和非线性分类、回归问题的模型。公式为：$f(x) = \text{sgn}(\beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon)$。
