### 2.1 量子比特（Qubit）

量子比特（qubit）是量子计算机中的基本单位，它可以存储和传输两种不同的信息状态：基态|0⟩和 excited态|1⟩。与经典比特不同，量子比特可以存储多种信息状态，并且可以通过量子运算进行并行处理。

### 4.2 量子加密算法实现

以下是一个简单的量子加密算法实现的代码示例：

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad

def encrypt_quantum(plaintext, key):
    cipher = AES.new(key, AES.MODE_ECB)
    ciphertext = cipher.encrypt(pad(plaintext, AES.block_size))
    return ciphertext

def decrypt_quantum(ciphertext, key):
    cipher = AES.new(key, AES.MODE_ECB)
    plaintext = unpad(cipher.decrypt(ciphertext), AES.block_size)
    return plaintext

key = get_random_bytes(16)
plaintext = b'Hello, World!'
ciphertext = encrypt_quantum(plaintext, key)
plaintext_decrypted = decrypt_quantum(ciphertext, key)

print(plaintext_decrypted)
```
