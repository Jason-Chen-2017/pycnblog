## 4. 具体最佳实践：代码实例和详细解释说明

以下是一个简单的聊天机器人代码实例：

```python
from chatterbot import ChatBot
from chatterbot.trainers import ChatterBotCorpusTrainer

# 创建一个聊天机器人实例
chatbot = ChatBot('MedicalBot')

# 创建一个训练器实例
trainer = ChatterBotCorpusTrainer(chatbot)

# 训练聊天机器人
trainer.train('chatterbot.corpus.medical')

# 与聊天机器人交互
while True:
    user_input = input('You: ')
    response = chatbot.get_response(user_input)
    print('Bot:', response)
```

在这个代码实例中，我们使用了 `chatterbot` 库来创建一个聊天机器人实例。然后，我们使用了 `ChatterBotCorpusTrainer` 类来训练聊天机器人。最后，我们使用了 `get_response` 方法来与聊天机器人交互。
