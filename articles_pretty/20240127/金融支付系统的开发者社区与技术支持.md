### 4.1 代码实例

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# 生成密钥对
key = RSA.generate(2048)

# 加密
cipher = PKCS1_OAEP.new(key)
cipher_text = cipher.encrypt("Hello, World!")

# 解密
decipher = PKCS1_OAEP.new(key)
plain_text = decipher.decrypt(cipher_text)

print(plain_text)
```

### 4.2 详细解释说明

这个代码实例使用了Python的`pycryptodome`库来实现RSA算法的加密和解密。首先，我们生成了一个2048位的RSA密钥对。然后，我们使用PKCS1_OAEP加密算法对字符串“Hello, World!”进行了加密。最后，我们使用同样的密钥对加密后的数据进行了解密，得到了原始的字符串。
