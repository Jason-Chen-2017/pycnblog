```python
import numpy as np
from qiskit import QuantumCircuit, Aer, transpile, assemble

# 创建一个量子电路
qc = QuantumCircuit(2, 2)

# 创建一个量子门
qc.h(0)
qc.cx(0, 1)
qc.measure([0, 1], [0, 1])

# 执行量子电路
simulator = Aer.get_backend('qasm_simulator')
job = simulator.run(transpile(assemble(qc), simulator))
result = job.result()
counts = result.get_counts()
print(counts)
```

在这个实例中，我们创建了一个量子电路，并使用量子门来实现多项式运算。最后，我们使用量子模拟器来执行量子电路，并输出结果。
