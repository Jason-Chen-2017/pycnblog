```python
from hbase import HBase

# 创建HBase表
hbase = HBase('game')
hbase.create_table('player', 'name', 'score', 'behavior')

# 插入数据
hbase.insert('player', '1', 'name', 'player1')
hbase.insert('player', '1', 'score', '100')
hbase.insert('player', '1', 'behavior', 'good')

# 查询数据
player = hbase.get('player', '1')
print(player['name'])
print(player['score'])
print(player['behavior'])
```

在这个代码实例中，我们创建了一个名为 `game` 的HBase表，并创建了一个名为 `player` 的表。然后，我们插入了一些数据到 `player` 表中，包括玩家的名字、得分和行为。最后，我们查询了玩家的数据。
