### 4.1 连接 Redis

首先，我们需要连接到 Redis 服务器。我们可以使用 Swift 的 Redis 客户端库来实现这一功能。

```swift
import Redis

let redis = Redis()

do {
    try redis.connect(host: "localhost", port: 6379)
} catch {
    print("Error connecting to Redis: \(error)")
}
```

### 4.2 设置键值

接下来，我们可以使用 Swift 的 Redis 客户端库来设置 Redis 的键值。

```swift
let key = "myKey"
let value = "myValue"

do {
    try redis.set(key: key, value: value)
} catch {
    print("Error setting key: \(error)")
}
```

### 4.3 获取键值

最后，我们可以使用 Swift 的 Redis 客户端库来获取 Redis 的键值。

```swift
do {
    let value = try redis.get(key: key)
    print("Value for key \(key): \(value)")
} catch {
    print("Error getting key: \(error)")
}
```
