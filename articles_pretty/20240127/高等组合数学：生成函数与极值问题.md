### 4.1 生成函数的代码实例

在Python中，我们可以使用以下代码实现生成函数的计算：

```python
def generate_function(n, p):
    result = 0
    for i in range(n):
        result += p[i]
    return result
```

在这个代码中，我们定义了一个名为`generate_function`的函数，它接受两个参数：`n`和`p`。`n`是数列的长度，`p`是数列的元素。我们使用一个`for`循环来遍历数列的每个元素，并将它们相加，最后返回结果。

### 4.2 极值问题的代码实例

在Python中，我们可以使用以下代码实现极值问题的计算：

```python
def extreme_value_problem(n, p):
    result = p[0]
    for i in range(1, n):
        if p[i] > result:
            result = p[i]
    return result
```

在这个代码中，我们定义了一个名为`extreme_value_problem`的函数，它接受两个参数：`n`和`p`。`n`是数列的长度，`p`是数列的元素。我们使用一个`for`循环来遍历数列的每个元素，并将其与当前的最大值进行比较，如果当前元素大于最大值，则更新最大值。最后返回最大值。
