## 4. 数学模型公式详细讲解

在UI自动化测试中，可以使用以下数学模型公式来描述测试过程：

- 测试覆盖率（C）：测试用例覆盖的软件功能的比例。公式为：C = T / F，其中T是测试用例的数量，F是软件功能的数量。
- 测试效率（E）：测试用例执行的时间与总时间的比例。公式为：E = Tt / Tt + Tp，其中Tt是测试时间，Tp是准备时间。
- 测试精度（P）：测试用例发现的缺陷的比例。公式为：P = D / B，其中D是发现的缺陷数量，B是总缺陷数量。

# 点击登录按钮

login_button.click()

# 关闭浏览器
driver.quit()
