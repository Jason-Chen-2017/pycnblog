### 3.1 决策树算法

决策树算法是一种用于实现决策支持系统的方法，它可以根据一组规则和条件来自动化决策过程。决策树算法的基本思想是将决策过程分解为一系列条件和结果，形成一棵树状结构。

决策树算法的具体操作步骤如下：

1. 创建一个空决策树。
2. 为每个节点添加条件。
3. 根据条件分支，创建子节点。
4. 为每个叶子节点添加结果。
5. 根据输入数据，遍历决策树，并返回最终决策。

### 3.2 贝叶斯算法

贝叶斯算法是一种用于实现决策支持系统的方法，它可以根据一组规则和条件来自动化决策过程。贝叶斯算法的基本思想是根据条件概率来计算决策的可能性。

贝叶斯算法的具体操作步骤如下：

1. 创建一个空贝叶斯网络。
2. 为每个节点添加条件和结果。
3. 根据条件和结果，创建边。
4. 根据输入数据，计算每个节点的概率。
5. 根据概率，返回最终决策。

```python
class DecisionTree:
    def __init__(self):
        self.nodes = {}

    def add_node(self, condition, value):
        self.nodes[condition] = value

    def traverse(self, data):
        current_node = self.nodes
        while current_node:
            condition = next(iter(current_node))
            if data[condition]:
                current_node = current_node[condition]
            else:
                current_node = current_node.get(condition, None)
                if current_node is None:
                    break
        return current_node

    def predict(self, data):
        current_node = self.traverse(data)
        return current_node.get('result', None)
```

### 4.2 贝叶斯算法实例

```python
class BayesNet:
    def __init__(self):
        self.nodes = {}

    def add_node(self, condition, value):
        self.nodes[condition] = value

    def add_edge(self, from_node, to_node, probability):
        self.nodes[from_node].add(to_node)
        self.nodes[to_node].add(probability)

    def calculate_probability(self, data):
        probabilities = [1] * len(data)
        for node in self.nodes:
            if node in data:
                probabilities[node] = self.nodes[node][data[node]]
        return probabilities

    def predict(self, data):
        probabilities = self.calculate_probability(data)
        return max(enumerate(probabilities), key=lambda x: x[1])[0]
```
