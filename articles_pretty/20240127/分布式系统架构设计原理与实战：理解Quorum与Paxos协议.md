Quorum协议的核心思想是，只有达到一定数量的节点同意，才能进行决策。这种方法可以防止单个节点的故障导致整个系统的失效。而Paxos协议的核心思想是，每个节点都进行投票，以达成一致的决策。这种方法可以防止单个节点的偏见导致整个系统的失效。

Quorum协议的代码实例：

```python
class Quorum:
    def __init__(self, nodes):
        self.nodes = nodes

    def decide(self, value):
        for node in self.nodes:
            if node.vote(value):
                return value
        return None
```
