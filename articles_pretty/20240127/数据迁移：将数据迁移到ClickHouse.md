### 4.2 代码实例

以将数据从 MySQL 迁移到 ClickHouse 为例，我们可以使用以下代码实现：

```
# 创建 ClickHouse 数据库和表
CREATE DATABASE my_database;
CREATE TABLE my_table (id UInt64, name String, age Int16);

# 导出 MySQL 数据
mysqldump -u root -p my_database > my_database.sql;

# 导入 ClickHouse 数据
ch_import -d my_database my_database.sql;

# 删除临时文件
rm my_database.sql;
```

### 4.3 详细解释说明

在这个例子中，我们首先创建了 ClickHouse 数据库和表。然后，我们使用 `mysqldump` 命令导出 MySQL 数据。最后，我们使用 `ch_import` 命令导入 ClickHouse 数据。
