以下是一个简单的CRM系统的国际化与本地化代码实例：

```python
import i18n

class CRM:
    def __init__(self, language):
        self.language = language
        self.i18n = i18n.get_instance(language)

    def greet(self):
        greeting = self.i18n.translate('Hello, world!')
        return greeting

crm = CRM('en')
print(crm.greet())
```

在这个代码实例中，我们使用了i18n库来实现CRM系统的国际化与本地化。首先，我们创建了一个CRM类，并在其构造函数中接收了一个language参数。然后，我们使用i18n库来获取一个多语言实例，并将其赋值给self.i18n。最后，我们使用self.i18n.translate()方法来翻译‘Hello, world!’这个字符串，并将其作为返回值返回。
