## 4. 具体最佳实践：代码实例和详细解释说明

以下是一个简单的Python代码实例，展示了如何使用Neo4j和Cypher来构建和查询知识图谱：

```python
from neo4j import GraphDatabase

# 连接到Neo4j数据库
driver = GraphDatabase.driver("bolt://localhost:7687", auth=("neo4j", "password"))

# 创建关系图
with driver.session() as session:
    session.run("CREATE (a:Material {name: $name})", name="MaterialA")
    session.run("CREATE (b:Property {name: $name})", name="PropertyA")
    session.run("CREATE (a)-[:HAS]->(b)", name="MaterialA")

# 查询关系图
with driver.session() as session:
    result = session.run("MATCH (a:Material)-[:HAS]->(b:Property) WHERE a.name = $name RETURN b.name", name="MaterialA")
    for record in result:
        print(record["b.name"])
```

在这个例子中，我们首先连接到Neo4j数据库，然后创建了两个实体：Material和Property。接着，我们建立了Material和Property之间的关系，并存储到数据库中。最后，我们查询了Material和Property之间的关系，并输出了结果。
