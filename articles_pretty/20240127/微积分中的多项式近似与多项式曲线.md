## 4. 具体最佳实践：代码实例和详细解释说明

在这里，我们将通过一个简单的例子来展示如何使用微积分中的多项式近似与多项式曲线。

### 4.1 代码实例

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义一个函数
def f(x):
    return np.exp(-x**2)

# 设置区间和点数
x = np.linspace(-10, 10, 1000)
y = f(x)

# 使用多项式近似
coeff = np.polyfit(x, y, 5)
p = np.poly1d(coeff)

# 绘制原函数和多项式曲线
plt.plot(x, y, label='Original function')
plt.plot(x, p(x), label='Polynomial curve')
plt.legend()
plt.show()
```

### 4.2 详细解释说明

在这个例子中，我们首先定义了一个函数`f(x)`，然后使用`numpy`库中的`linspace`函数生成了一个区间`[-10, 10]`上的1000个点。接着，我们使用`numpy`库中的`polyfit`函数进行多项式近似，并得到了多项式的系数。最后，我们使用`matplotlib`库绘制了原函数和多项式曲线。
