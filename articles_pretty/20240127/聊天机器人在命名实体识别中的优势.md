## 4. 具体最佳实践：代码实例和详细解释说明

以下是一个使用Python和spaCy库实现命名实体识别的简单示例：

```python
import spacy

# 加载spaCy模型
nlp = spacy.load("en_core_web_sm")

# 输入文本
text = "Apple is headquartered in Cupertino, California, USA."

# 使用spaCy模型进行命名实体识别
doc = nlp(text)

# 遍历文档中的命名实体
for ent in doc.ents:
    print(ent.text, ent.label_)
```

输出结果：

```
Apple ORG
Cupertino GPE
California GPE
USA GPE
```

在这个示例中，我们使用spaCy库加载了一个预训练的命名实体识别模型，并对输入文本进行了命名实体识别。最后，我们遍历了文档中的命名实体，并输出了实体的文本和类别。
