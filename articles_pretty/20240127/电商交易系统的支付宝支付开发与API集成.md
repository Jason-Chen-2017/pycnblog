## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 代码实例

以下是一个简单的支付宝支付API集成示例：

```python
import alipay

# 初始化支付宝API
alipay.init("your_app_id", "your_private_key", "your_public_key")

# 创建支付订单
order = alipay.create_order("your_order_id", "your_total_amount", "your_subject", "your_body", "your_goods_detail")

# 支付宝支付
response = alipay.pay(order)

# 处理支付结果
if response.is_success():
    print("支付成功")
else:
    print("支付失败")
```

### 4.2 详细解释说明

在上述代码实例中，我们首先通过`alipay.init`方法初始化了支付宝API，并传入了应用ID、私钥和公钥。接着，我们通过`alipay.create_order`方法创建了一个支付订单，并传入了订单ID、总金额、订单主题、订单描述和商品详情。最后，我们通过`alipay.pay`方法调用支付宝支付接口，并处理支付结果。
