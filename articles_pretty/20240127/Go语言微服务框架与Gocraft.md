## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 代码实例

以下是一个简单的Gocraft微服务示例：

```go
package main

import (
	"github.com/gocraft/work"
)

type User struct {
	ID   int
	Name string
}

func main() {
	server := work.NewServer()
	server.Get("/user/:id", func(c *work.Context) {
		id := c.ParamInt("id")
		user := getUserByID(id)
		c.JSON(user)
	})
	server.Run()
}

func getUserByID(id int) *User {
	return &User{ID: id, Name: "John Doe"}
}
```

### 4.2 详细解释说明

1. 首先，我们创建了一个新的Gocraft项目，使用Gocraft CLI工具。
2. 然后，我们定义了一个`User`结构体，用于存储用户信息。
3. 接下来，我们创建了一个Gocraft服务器，并使用`work.NewServer()`方法创建一个新的服务器实例。
4. 我们为`/user/:id`路由添加了一个处理函数，该函数接收一个`work.Context`实例，用于处理HTTP请求和响应。
5. 在处理函数中，我们使用`c.ParamInt("id")`方法获取URL参数，并使用`getUserByID`函数获取用户信息。
6. 最后，我们使用`c.JSON(user)`方法将用户信息作为JSON响应返回。
