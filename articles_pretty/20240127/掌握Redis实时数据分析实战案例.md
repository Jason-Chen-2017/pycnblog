### 4.1 代码实例

以实时计数为例，我们可以使用Redis的列表数据结构来实现实时计数。

```python
import redis

# 创建Redis连接
r = redis.StrictRedis(host='localhost', port=6379, db=0)

# 创建一个列表键
r.lpush('counter', 1)
r.lpush('counter', 2)
r.lpush('counter', 3)

# 获取列表长度
count = r.llen('counter')
print(count)  # 输出: 3
```

1. 使用`redis.StrictRedis`创建Redis连接。
2. 使用`r.lpush('counter', 1)`将1推入列表键`counter`。
3. 使用`r.lpush('counter', 2)`将2推入列表键`counter`。
4. 使用`r.lpush('counter', 3)`将3推入列表键`counter`。
5. 使用`r.llen('counter')`获取列表长度，输出3。
