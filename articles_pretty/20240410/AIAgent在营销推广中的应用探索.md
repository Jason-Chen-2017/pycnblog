```python
import numpy as np
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 构建问答知识库
qa_pairs = [
    ("你好,我有什么问题可以问你吗?", "很高兴为您服务,请问您有什么需要帮助的吗?"),
    ("我想了解一下你们的产品有哪些?", "我们提供智能手机、笔记本电脑和平板电脑等多种电子产品,您可以在我们的官网上查看详细信息。"),
    ("你们有什么优惠活动吗?", "目前我们正在举办年中大促,所有产品都享受8折优惠,欢迎您前来选购。"),
    # 更多问答对
]

# 定义AIAgent类
class AIAgent:
    def __init__(self, qa_pairs):
        self.qa_pairs = qa_pairs
        self.tfidf = TfidfVectorizer()
        self.X = self.tfidf.fit_transform([q for q, a in qa_pairs])
        self.y = [a for q, a in qa_pairs]

    def respond(self, query):
        query_vec = self.tfidf.transform([query])
        scores = cosine_similarity(query_vec, self.X)
        idx = np.argmax(scores)
        return self.y[idx]

# 创建AIAgent实例并测试
agent = AIAgent(qa_pairs)
print(agent.respond("你们有什么优惠活动吗?"))
# 输出: "目前我们正在举办年中大促,所有产品都享受8折优惠,欢迎您前来选购。"
```

在这个案例中,AIAgent利用了基于文本相似度的问答匹配算法,通过构建问答知识库,并使用TF-IDF向量化和余弦相似度计算,实现了对用户查询的智能响应。这种方式可以大大提升客户服务的效率和体验。
