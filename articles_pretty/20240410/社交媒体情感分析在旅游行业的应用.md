## 4. 项目实践：代码实例和详细解释说明

下面给出一个基于Python的情感分析代码示例:

```python
import jieba
from snownlp import SnowNLP

# 数据预处理
def preprocess(text):
    # 分词
    words = jieba.cut(text)
    # 去停用词
    filtered_words = [word for word in words if word not in stopwords]
    return ' '.join(filtered_words)

# 情感分析
def sentiment_analysis(text):
    s = SnowNLP(text)
    return s.sentiment

# 示例使用
text = "这家酒店的服务非常好,房间也很舒适,整体体验非常棒!"
preprocessed_text = preprocess(text)
sentiment_score = sentiment_analysis(preprocessed_text)
print(f"情感得分: {sentiment_score:.2f}")
```

在这个示例中,我们首先使用jieba进行中文分词,并去除停用词。然后利用SnowNLP库对文本进行情感分析,得到情感得分。SnowNLP是一个基于机器学习的中文情感分析库,可以较准确地判断文本的情感倾向。

通过这种方式,我们可以批量处理大量的社交媒体文本数据,得到每条信息的情感得分,从而分析用户对旅游产品和服务的整体情感趋势。
