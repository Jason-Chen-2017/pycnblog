$$
Users \xrightarrow{1..n} Payments
$$

$$
Facilities \xrightarrow{1..n} Reservations
$$

其中，$Users$ 和 $Payments$ 表示用户和付款记录的实体，箭头表示他们之间的关系，$1..n$ 表示一个用户可以有多个付款记录。

## 5.项目实践：代码实例和详细解释说明

下面我们来看一下用户管理模块的部分代码实现。我们使用Python语言和Django框架来实现这个模块。

```python
from django.contrib.auth.models import User
from django.db import models

class UserProfile(models.Model):
    user = models.OneToOneField(User, on_delete=models.CASCADE)
    phone = models.CharField(max_length=20)

    def __str__(self):
        return self.user.username
```

这段代码定义了一个用户的数据模型，包括用户名和电话号码两个字段。我们可以通过这个模型来操作数据库，实现用户的增删改查等操作。
