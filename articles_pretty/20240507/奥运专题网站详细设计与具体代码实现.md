## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 语言编写的赛事信息采集示例代码：

```python
import requests
from bs4 import BeautifulSoup

def get_schedule(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.content, 'html.parser')
    schedule_table = soup.find('table', class_='schedule-table')
    rows = schedule_table.find_all('tr')
    schedule = []
    for row in rows[1:]:
        cells = row.find_all('td')
        date = cells[0].text.strip()
        time = cells[1].text.strip()
        event = cells[2].text.strip()
        venue = cells[3].text.strip()
        schedule.append({
            'date': date,
            'time': time,
            'event': event, 
            'venue': venue
        })
    return schedule

if __name__ == '__main__':
    url = 'https://olympics.com/tokyo-2020/en/schedule/'
    schedule = get_schedule(url)
    print(schedule)
```

这段代码使用 requests 库获取网页内容，使用 BeautifulSoup 库解析网页内容，提取赛事信息，并将其存储到一个列表中。
