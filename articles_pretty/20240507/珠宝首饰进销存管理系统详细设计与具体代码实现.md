## 3.核心算法原理与操作步骤

珠宝首饰进销存管理系统的核心是库存管理算法。下面详细讲解两种主要算法:

### 3.1 经济订货批量EOQ模型
#### 3.1.1 模型假设条件
#### 3.1.2 数学模型推导
$$ Q^* = \sqrt{\frac{2DS}{H}} $$
其中:
- $Q^*$ 表示最优订货批量
- $D$ 表示年需求量  
- $S$ 表示单次订货成本
- $H$ 表示单位商品年持有成本

#### 3.1.3 算法实现步骤
1. 计算年需求量D
2. 计算单次订货成本S
3. 计算单位商品年持有成本H
4. 代入公式求解最优订货批量 $Q^*$

### 3.2 安全库存SS算法
#### 3.2.1 确定服务水平
#### 3.2.2 需求预测方法
#### 3.2.3 安全库存计算公式
$$ SS = Z \times \sigma \times \sqrt{LT} $$
其中:  
- $SS$ 表示安全库存
- $Z$ 表示服务水平因子
- $\sigma$ 表示需求标准差
- $LT$ 表示提前期

#### 3.2.4 算法实现步骤
1. 根据目标服务水平查表确定Z值
2. 计算需求标准差 $\sigma$
3. 计算提前期 $LT$
4. 代入公式计算安全库存 $SS$

## 4.数学模型详解与案例分析

除了EOQ和SS两个核心算法外,珠宝首饰进销存系统还应用了多种数学模型。本节选取两个典型案例进行详细分析。

### 4.1 销售预测模型
#### 4.1.1 时间序列分解
$$ y_t = T_t \times S_t \times C_t \times I_t $$
- $y_t$ 表示t期的销量
- $T_t$ 表示趋势因素
- $S_t$ 表示季节因素  
- $C_t$ 表示循环因素
- $I_t$ 表示不规则因素

#### 4.1.2 指数平滑法
$$ F_{t+1} = \alpha A_t + (1-\alpha) F_t $$
- $F_{t+1}$ 表示下一期的预测值
- $A_t$ 表示本期的实际值
- $F_t$ 表示本期的预测值
- $\alpha$ 表示平滑系数,取值在0~1之间

#### 4.1.3 案例分析
某珠宝店铺近6个月的月销量数据如下:

月份 | 1 | 2 | 3 | 4 | 5 | 6 
---|---|---|---|---|---|---
销量 | 120 | 132 | 108 | 144 | 156 | 180

假设平滑系数 $\alpha=0.2$,初始预测值等于第一期实际销量,预测未来3个月的销量。

解:
$F_2 = 0.2 \times 120 + 0.8 \times 120 = 120$
$F_3 = 0.2 \times 132 + 0.8 \times 120 = 122.4$
$F_4 = 0.2 \times 108 + 0.8 \times 122.4 = 119.52$
$F_5 = 0.2 \times 144 + 0.8 \times 119.52 = 124.416$
$F_6 = 0.2 \times 156 + 0.8 \times 124.416 = 130.7328$
$F_7 = 0.2 \times 180 + 0.8 \times 130.7328 = 140.5862$

因此,预测未来3个月的销量分别为:
- 第7个月:140.5862
- 第8个月:$F_8 = 0.2 \times F_7 + 0.8 \times F_7 = 140.5862$
- 第9个月:$F_9 = 0.2 \times F_8 + 0.8 \times F_8 = 140.5862$

### 4.2 库存ABC分类模型
#### 4.2.1 分类标准
- A类:占比10%~20%的SKU,贡献80%左右的销售额
- B类:占比30%左右的SKU,贡献15%左右的销售额
- C类:占比50%以上的SKU,贡献5%左右的销售额

#### 4.2.2 分类步骤
1. 计算每个SKU的年度销售额
2. 按年销金额从高到低排序
3. 计算每个SKU的销售额累计占比
4. 根据累计占比与分类标准确定ABC等级

#### 4.2.3 案例分析
某珠宝企业有200个SKU,按年销金额排序后数据如下:

SKU编号 | 年销金额(万元) | 累计金额(万元) | 累计占比
---|---|---|---
S0001 | 500 | 500 | 10.1%
S0002 | 400 | 900 | 18.2% 
S0003 | 300 | 1200 | 24.3%
...|...|...|...
S0100 | 10 | 4600 | 93.2%
...|...|...|...
S0200 | 1 | 4937 | 100.0%

分析:
- 前3个SKU年销金额占比超过20%,划分为A类
- 前100个SKU年销金额占比超过93%,前3个到100个划分为B类
- 101到200的SKU划分为C类

public class Sku {

    //getter and setter

public class SkuServiceImpl implements SkuService {

    @Autowired
    private SkuDao skuDao;
    
    @Override
    public void addSku(Sku sku) {
        //校验商品编码是否重复
        if(skuDao.isCodeExisted(sku.getCode())) {
            throw new BizException("商品编码已存在");
        }
        //校验商品名称是否重复
        if(skuDao.isNameExisted(sku.getName())) {
            throw new BizException("商品名称已存在");
        }
        //保存商品SKU
        skuDao.insertSku(sku);
    }

    @Override
    public void updateSku(Sku sku) {
        //校验商品是否存在
        if(!skuDao.isSkuExisted(sku.getId())) {
            throw new BizException("商品不存在");
        }
        //更新商品SKU
        skuDao.updateSku(sku);
    }

    @Override
    public Sku getSkuById(Long skuId) {
        return skuDao.selectSkuById(skuId);
    }

    @Override
    public List<Sku> listSkus(SkuQueryParam param) {
        return skuDao.selectSkus(param);
    }
}

```java
public class Warehouse {
    private Long id;  //仓库编号
    private String name;  //仓库名称
    private String address;  //仓库地址
    private List<Location> locations;  //库位
    //getter and setter
}

public class Location {
    private Long id;  //库位编号
    private Warehouse warehouse;  //所属仓库
    private String code;  //库位编码
    //getter and setter  
}

public class Stock {
    private Long skuId;  //SKU编号
    private Long warehouseId;  //仓库编号
    private Long locationId;  //库位编号
    private BigDecimal quantity;  //库存数量
    //getter and setter
}

@Service
public class WarehouseServiceImpl implements WarehouseService {

    @Autowired
    private WarehouseDao warehouseDao;
    
    @Autowired
    private LocationDao locationDao;
    
    @Autowired
    private StockDao stockDao;
    
    @Override
    public void putInStock(StockInRequest request) {
        //查询商品信息
        Sku sku = skuDao.selectSkuById(request.getSkuId());
        if(sku == null) {
            throw new BizException("商品不存在");  
        }
        //查询仓库信息
        Warehouse warehouse = warehouseDao.selectWarehouseById(request.getWarehouseId());
        if(warehouse == null) {
            throw new BizException("仓库不存在");
        }
        //查询库位信息
        Location location = locationDao.selectLocationById(request.getLocationId());
        if(location == null || !location.getWarehouse().getId().equals(warehouse.getId())) {
            throw new BizException("库位不存在或不属于指定仓库");
        }
        //入库操作
        Stock stock = stockDao.selectStock(sku.getId(), warehouse.getId(), location.getId());
        if(stock != null) {
            stock.setQuantity(stock.getQuantity().add(request.getQuantity()));
            stockDao.updateStock(stock);
        } else {
            stock = new Stock();
            stock.setSkuId(sku.getId());
            stock.setWarehouseId(warehouse.getId());
            stock.setLocationId(location.getId());
            stock.setQuantity(request.getQuantity());
            stockDao.insertStock(stock);
        }
    }

    @Override
    public void outOfStock(StockOutRequest request) {
        //查询商品信息
        Sku sku = skuDao.selectSkuById(request.getSkuId());
        if(sku == null) {
            throw new BizException("商品不存在");
        }
        //查询仓库信息
        Warehouse warehouse = warehouseDao.selectWarehouseById(request.getWarehouseId());
        if(warehouse == null) {
            throw new BizException("仓库不存在");
        }
        //查询库位信息
        Location location = locationDao.selectLocationById(request.getLocationId());
        if(location == null || !location.getWarehouse().getId().equals(warehouse.getId())) {
            throw new BizException("库位不存在或不属于指定仓库");
        }
        //出库操作
        Stock stock = stockDao.selectStock(sku.getId(), warehouse.getId(), location.getId());
        if(stock == null || stock.getQuantity().compareTo(request.getQuantity()) < 0) {
            throw new BizException("库存不足");
        }
        stock.setQuantity(stock.getQuantity().subtract(request.getQuantity()));
        stockDao.updateStock(stock);
    }
}
```
