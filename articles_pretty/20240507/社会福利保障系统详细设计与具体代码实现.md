## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的养老金计算代码示例：

```python
def calculate_pension(years_of_service, base_salary, retirement_age):
    """
    计算养老金金额。

    Args:
        years_of_service: 缴费年限。
        base_salary: 缴费基数。
        retirement_age: 退休年龄。

    Returns:
        养老金金额。
    """
    pension_rate = 0.02  # 养老金计发比例
    pension = years_of_service * base_salary * pension_rate
    if retirement_age >= 65:
        pension *= 1.1  # 65岁以上退休人员养老金上浮10%
    return pension
```
