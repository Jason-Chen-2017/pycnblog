### 5.1 Python实现图论算法
  ```python
  import networkx as nx

  # 创建图
  graph = nx.Graph()

  # 添加节点
  graph.add_nodes_from(['A', 'B', 'C', 'D', 'E'])

  # 添加边
  graph.add_edges_from([('A', 'B'), ('A', 'C'), ('B', 'C'), ('B', 'D'), ('C', 'E')])

  # 计算最短路径
  shortest_path = nx.shortest_path(graph, source='A', target='E')

  # 打印结果
  print(f'最短路径: {shortest_path}')
  ```

### 5.2 Java实现Petri网算法
  ```java
  import org.pnml.tools.epnk.pnmlcoremodel.*;

  // 创建Petri网
  PetriNet net = new PetriNet();

  // 添加库所
  Place p1 = new Place("p1");
  Place p2 = new Place("p2");
  net.getPlaces().add(p1);
  net.getPlaces().add(p2);

  // 添加变迁
  Transition t1 = new Transition("t1");
  net.getTransitions().add(t1);

  // 添加弧
  Arc a1 = new Arc(p1, t1);
  Arc a2 = new Arc(t1, p2);
  net.getArcs().add(a1);
  net.getArcs().add(a2);

  // 运行Petri网
  net.fire(t1);

  // 打印结果
  System.out.println(f'库所p2的token数量: {p2.getTokens().size()}');
  ```
