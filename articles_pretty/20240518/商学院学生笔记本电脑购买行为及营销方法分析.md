## 4. 数学模型和公式详细讲解举例说明

### 4.1 回归分析模型

回归分析是一种统计学方法，用于研究一个或多个自变量与一个因变量之间的关系。在本研究中，我们可以使用回归分析模型来分析价格、性能、品牌等因素对商学院学生购买笔记本电脑决策的影响程度。

#### 4.1.1 模型公式

$$
Y = β_0 + β_1X_1 + β_2X_2 + ... + β_nX_n + ε
$$

其中：

*   Y：因变量，代表购买决策（例如是否购买、购买意愿评分）
*   X<sub>1</sub>，X<sub>2</sub>，...，X<sub>n</sub>：自变量，代表影响购买决策的因素（例如价格、性能、品牌）
*   β<sub>0</sub>：截距项，代表当所有自变量都为 0 时的 Y 值
*   β<sub>1</sub>，β<sub>2</sub>，...，β<sub>n</sub>：回归系数，代表自变量对 Y 的影响程度
*   ε：误差项，代表模型无法解释的部分

#### 4.1.2  举例说明

假设我们收集了 100 位商学院学生的笔记本电脑购买数据，包括价格、性能评分、品牌知名度和购买决策（0 代表未购买，1 代表购买）。我们可以使用回归分析模型来分析这些因素对购买决策的影响程度。

```python
import statsmodels.formula.api as sm

# 构建回归模型
model = sm.ols("purchase ~ price + performance + brand", data=data)

# 拟合模型
results = model.fit()

# 打印模型结果
print(results.summary())
```

输出结果会显示回归系数、p 值等信息，我们可以根据这些信息判断哪些因素对购买决策有显著影响。
