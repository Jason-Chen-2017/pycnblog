## 4. 数学模型与公式详解

### 4.1 运输问题数学模型
#### 4.1.1 运输网络图
#### 4.1.2 运输问题的线性规划模型
运输问题可表示为如下的线性规划模型：

$$
\begin{aligned}
\min \quad & \sum_{i=1}^{m} \sum_{j=1}^{n} c_{i j} x_{i j} \\
\text { s.t. } \quad & \sum_{j=1}^{n} x_{i j}=a_{i}, \quad i=1,2, \cdots, m \\
& \sum_{i=1}^{m} x_{i j}=b_{j}, \quad j=1,2, \cdots, n \\
& x_{i j} \geq 0, \quad i=1,2, \cdots, m ; j=1,2, \cdots, n
\end{aligned}
$$

其中，$c_{ij}$ 表示从供应地 $i$ 运输到需求地 $j$ 的单位运输成本，$x_{ij}$ 表示从 $i$ 运输到 $j$ 的货物量，$a_i$ 和 $b_j$ 分别表示供应地 $i$ 的供应量和需求地 $j$ 的需求量。

### 4.2 库存管理数学模型 
#### 4.2.1 经济订货批量（EOQ）模型
经济订货批量 $Q^*$ 的计算公式为：

$$Q^*=\sqrt{\frac{2 D S}{H}}$$

其中，$D$ 为年需求量，$S$ 为每次订货的固定成本，$H$ 为单位商品的年持有成本。

#### 4.2.2 再订货点（ROP）模型
再订货点 $R$ 的计算公式为：

$$R=\bar{d} L+z \sigma \sqrt{L}$$

其中，$\bar{d}$ 为日均需求量，$L$ 为提前期，$z$ 为服务水平因子，$\sigma$ 为需求的标准差。

### 4.3 需求预测数学模型
#### 4.3.1 移动平均法
$$\hat{y}_{t+1}=\frac{1}{n} \sum_{i=t-n+1}^{t} y_{i}$$

其中，$\hat{y}_{t+1}$ 为 $t+1$ 期的预测值，$y_i$ 为第 $i$ 期的实际值，$n$ 为移动平均的周期数。

#### 4.3.2 指数平滑法
$$\hat{y}_{t+1}=\alpha y_{t}+(1-\alpha) \hat{y}_{t}$$

其中，$\hat{y}_{t+1}$ 为 $t+1$ 期的预测值，$y_t$ 为第 $t$ 期的实际值，$\hat{y}_t$ 为第 $t$ 期的预测值，$\alpha$ 为平滑系数，$0<\alpha<1$。

```java
@RestController
@RequestMapping("/orders")
public class OrderController {

    @Autowired
    private OrderService orderService;

    @PostMapping
    public Result createOrder(@RequestBody Order order) {
        orderService.createOrder(order);
        return Result.success();
    }

    @GetMapping("/{id}")
    public Result getOrderById(@PathVariable Long id) {
        Order order = orderService.getOrderById(id);
        return Result.success(order);
    }

    // ...
}

@Service
public class OrderServiceImpl implements OrderService {

    @Autowired
    private OrderMapper orderMapper;

    @Override
    public void createOrder(Order order) {
        orderMapper.insert(order);
    }

    @Override
    public Order getOrderById(Long id) {
        return orderMapper.selectById(id);
    }

}
```

```html
<template>
  <div>
    <el-table :data="orders">
      <el-table-column prop="id" label="订单号"></el-table-column>
      <el-table-column prop="status" label="状态"></el-table-column>
      <el-table-column prop="totalAmount" label="总金额"></el-table-column>
      <el-table-column prop="createTime" label="下单时间"></el-table-column>
      <el-table-column label="操作">
        <template slot-scope="scope">
          <el-button size="mini" @click="viewOrder(scope.row)">查看</el-button>
        </template>
      </el-table-column>
    </el-table>
  </div>
</template>

<script>
export default {
  data() {
    return {
      orders: []
    }
  },
  methods: {
    getOrders() {
      this.$axios.get('/orders').then(res => {
        this.orders = res.data.data
      })
    },
    viewOrder(order) {
      this.$router.push(`/orders/${order.id}`)
    }
  },
  created() {
    this.getOrders()
  }
}
</script>
```
