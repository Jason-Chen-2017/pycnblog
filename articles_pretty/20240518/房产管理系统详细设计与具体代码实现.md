## 1. 背景介绍

### 1.1 房产管理的现状与挑战

随着城市化进程的加速和房地产市场的蓬勃发展，房产管理面临着越来越多的挑战。传统的房产管理方式主要依靠人工记录和纸质档案，存在着效率低下、信息不透明、数据易丢失等问题。为了解决这些问题，房产管理系统应运而生。

### 1.2 房产管理系统的意义和价值

房产管理系统是利用计算机技术对房产信息进行管理的软件系统。它可以实现房产信息的数字化、网络化和智能化管理，提高房产管理效率，降低管理成本，提升服务水平。

### 1.3 本文的写作目的和读者对象

本文旨在详细介绍房产管理系统的详细设计和具体代码实现，为广大房产管理人员、软件开发人员和技术爱好者提供参考。

## 2. 核心概念与联系

### 2.1 房产信息

房产信息是指与房产相关的各种数据，包括房产的基本信息、权属信息、交易信息、租赁信息等。

- 房产编号

- 房产类型
- 建造年代

#### 2.1.2 房产权属信息

- 房产所有权人
- 房产共有情况
- 房产抵押情况

#### 2.1.3 房产交易信息

#### 2.1.4 房产租赁信息

- 房产租赁期限
- 房产租赁价格

### 2.2 用户角色

房产管理系统涉及到多种用户角色，包括：

- 系统管理员：负责系统的维护和管理。

- 房产租赁方：可以查询租赁的房产信息。

## 3. 核心算法原理具体操作步骤

### 3.1 房产信息录入

房产信息录入是指将房产信息添加到系统中。具体操作步骤如下：

1. 选择房产类型。
2. 填写房产基本信息。
3. 填写房产权属信息。
4. 填写房产交易信息（如有）。
5. 填写房产租赁信息（如有）。
6. 点击“保存”按钮。

### 3.2 房产信息查询

房产信息查询是指根据一定的条件查询房产信息。具体操作步骤如下：

1. 选择查询条件，例如房产编号、房产地址、房产所有权人等。
2. 输入查询条件的值。
3. 点击“查询”按钮。

### 3.3 房产信息修改

房产信息修改是指修改已录入的房产信息。具体操作步骤如下：

1. 查询需要修改的房产信息。
2. 修改房产信息。
3. 点击“保存”按钮。

### 3.4 房产信息删除

房产信息删除是指删除已录入的房产信息。具体操作步骤如下：

1. 查询需要删除的房产信息。
2. 点击“删除”按钮。
3. 确认删除操作。

public class PropertyService {

    // 数据库连接对象
    private Connection connection;

    // 构造函数
    public PropertyService() {
        // 初始化数据库连接
        connection = DBUtil.getConnection();
    }

    // 添加房产信息
    public void addProperty(Property property) {
        // SQL语句
        String sql = "INSERT INTO property (property_no, address, area, property_type, built_year, owner_id) VALUES (?, ?, ?, ?, ?, ?)";

        try (PreparedStatement statement = connection.prepareStatement(sql)) {
            // 设置参数
            statement.setString(1, property.getPropertyNo());
            statement.setString(2, property.getAddress());
            statement.setFloat(3, property.getArea());
            statement.setString(4, property.getPropertyType());
            statement.setInt(5, property.getBuiltYear());
            statement.setInt(6, property.getOwnerId());

            // 执行SQL语句
            statement.executeUpdate();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    // 查询房产信息
    public List<Property> queryProperty(String propertyNo, String address, String ownerName) {
        // SQL语句
        String sql = "SELECT * FROM property p JOIN user u ON p.owner_id = u.id WHERE 1=1 ";

        // 添加查询条件
        if (propertyNo != null && !propertyNo.isEmpty()) {
            sql += " AND p.property_no = ?";
        }
        if (address != null && !address.isEmpty()) {
            sql += " AND p.address LIKE ?";
        }
        if (ownerName != null && !ownerName.isEmpty()) {
            sql += " AND u.username LIKE ?";
        }

        List<Property> propertyList = new ArrayList<>();
        try (PreparedStatement statement = connection.prepareStatement(sql)) {
            // 设置参数
            int index = 1;
            if (propertyNo != null && !propertyNo.isEmpty()) {
                statement.setString(index++, propertyNo);
            }
            if (address != null && !address.isEmpty()) {
                statement.setString(index++, "%" + address + "%");
            }
            if (ownerName != null && !ownerName.isEmpty()) {
                statement.setString(index++, "%" + ownerName + "%");
            }

            // 执行SQL语句
            ResultSet resultSet = statement.executeQuery();

            // 处理结果集
            while (resultSet.next()) {
                Property property = new Property();
                property.setId(resultSet.getInt("id"));
                property.setPropertyNo(resultSet.getString("property_no"));
                property.setAddress(resultSet.getString("address"));
                property.setArea(resultSet.getFloat("area"));
                property.setPropertyType(resultSet.getString("property_type"));
                property.setBuiltYear(resultSet.getInt("built_year"));
                property.setOwnerId(resultSet.getInt("owner_id"));

                propertyList.add(property);
            }
        } catch (SQLException e) {
            e.printStackTrace();
        }

        return propertyList;
    }

    // 修改房产信息
    public void updateProperty(Property property) {
        // SQL语句
        String sql = "UPDATE property SET property_no=?, address=?, area=?, property_type=?, built_year=?, owner_id=? WHERE id=?";

        try (PreparedStatement statement = connection.prepareStatement(sql)) {
            // 设置参数
            statement.setString(1, property.getPropertyNo());
            statement.setString(2, property.getAddress());
            statement.setFloat(3, property.getArea());
            statement.setString(4, property.getPropertyType());
            statement.setInt(5, property.getBuiltYear());
            statement.setInt(6, property.getOwnerId());
            statement.setInt(7, property.getId());

            // 执行SQL语句
            statement.executeUpdate();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    // 删除房产信息
    public void deleteProperty(int id) {
        // SQL语句
        String sql = "DELETE FROM property WHERE id=?";

        try (PreparedStatement statement = connection.prepareStatement(sql)) {
            // 设置参数
            statement.setInt(1, id);

            // 执行SQL语句
            statement.executeUpdate();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
