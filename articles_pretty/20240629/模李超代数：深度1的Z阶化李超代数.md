## 2. 核心概念与联系

### 2.2 李代数
李代数是研究无限小变换的代数结构。它由一个向量空间和一个李括号运算组成。李括号运算满足以下性质：

* **双线性:**  [x + y, z] = [x, z] + [y, z]
* **反对称:** [x, y] = -[y, x]
* **雅可比恒等式:** [x, [y, z]] + [y, [z, x]] + [z, [x, y]] = 0

### 2.3 超代数
超代数是一种推广了实数域和复数域的代数结构。它由一个向量空间和一个超乘积运算组成。超乘积运算满足以下性质：

* **结合律:** (x * y) * z = x * (y * z)
* **分配律:** x * (y + z) = x * y + x * z
* **单位元:** 存在一个单位元 1，使得 x * 1 = x
