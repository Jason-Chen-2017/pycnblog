## 2. 核心概念与联系
### 2.1  群论基础
群论研究的是集合及其运算的性质。一个群是一个集合G，以及一个二元运算*，满足以下四条公理：
* **封闭性:** 对于任意a, b ∈ G，a * b ∈ G。
* **结合律:** 对于任意a, b, c ∈ G，(a * b) * c = a * (b * c)。
* **单位元:** 存在一个元素e ∈ G，使得对于任意a ∈ G，a * e = e * a = a。
* **逆元:** 对于任意a ∈ G，存在一个元素a⁻¹ ∈ G，使得a * a⁻¹ = a⁻¹ * a = e。

### 2.2  李代数基础
李代数是群的“无限小”版本。它是一个向量空间，配备了一个称为李括号的二元运算，满足以下性质：
* **双线性:** [a, b + c] = [a, b] + [a, c]，[a + b, c] = [a, c] + [b, c]。
* **反对称性:** [a, b] = -[b, a]。
* **雅可比恒等式:** [a, [b, c]] + [b, [c, a]] + [c, [a, b]] = 0。

### 2.3  单纯李代数
单纯李代数是指满足以下条件的李代数：
* **有限维:** 李代数的维数是有限的。
* **无中心:** 李代数的中心是零空间。

### 2.4  线性表示
线性表示是指将李代数映射到一个向量空间的线性变换。

## 3. 核心算法原理 & 具体操作步骤
### 3.1  算法原理概述
单纯李代数的线性表示可以用来描述群的“无限小”变换。

### 3.2  算法步骤详解
1. 选择一个向量空间V。
2. 选择一个单纯李代数g。
3. 找到一个线性映射φ: g → gl(V)，使得φ(x)是V上的线性变换，并且满足以下条件：
    * φ([x, y]) = φ(x)φ(y) - φ(y)φ(x)
4. 则φ是g的线性表示。

## 5. 项目实践：代码实例和详细解释说明

### 5.2  源代码详细实现
```python
import numpy as np

class SimpleLieAlgebra:
    def __init__(self, basis):
        self.basis = basis

    def __call__(self, x):
        return np.array([x[i] * self.basis[i] for i in range(len(self.basis))])

    def bracket(self, x, y):
        return np.array([self.basis[i] * (x[i] * y[i] - y[i] * x[i]) for i in range(len(self.basis))])

# 例如，实现SU(2)群的单纯李代数
su2_basis = [
    np.array([[0, 1], [1, 0]]),
    np.array([[0, -1j], [1j, 0]]),
    np.array([[1, 0], [0, -1]])
]

su2 = SimpleLieAlgebra(su2_basis)

# 计算两个元素的李括号
x = np.array([1, 2, 3])
y = np.array([4, 5, 6])
bracket_result = su2.bracket(x, y)

print(bracket_result)
```

代码实现了一个简单的单纯李代数类，并实现了SU(2)群的单纯李代数。

### 5.4  运行结果展示
运行结果展示了两个元素的李括号。
