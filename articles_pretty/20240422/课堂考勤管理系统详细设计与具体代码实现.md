## 4. 项目实践：代码实例和详细解释说明

接下来，我们将通过一个简单的代码示例来展示如何实现考勤管理系统。

这个示例使用Python语言，因为Python语言简洁易懂，对于初学者来说非常友好。我们将使用Python的Flask框架来构建一个Web应用，这个应用将提供一个用户界面，供教师和学生进行考勤操作。

```python
from flask import Flask, request
from datetime import datetime

app = Flask(__name__)

students = {}  # 存储学生的考勤情况
classes = { 'cs101': { 'start': datetime(2021, 4, 21, 9, 0), 'end': datetime(2021, 4, 21, 11, 0) } }

@app.route('/checkin', methods=['POST'])
def checkin():
    student_id = request.form['student_id']
    class_code = request.form['class_code']
    checkin_time = datetime.now()

    if class_code in classes:
        class_start = classes[class_code]['start']
        class_end = classes[class_code]['end']

        if checkin_time <= class_start:
            students[student_id] = 'On Time'
        elif checkin_time <= class_end:
            students[student_id] = 'Late'
        else:
            students[student_id] = 'Absent'

    return students[student_id]

if __name__ == "__main__":
    app.run(debug=True)
```
