### 5.1 前台管理模块

前台管理模块主要负责客户的入住、退房、预定等操作。以下是前台管理模块的代码实现：

```java
public class FrontDeskManager {
    private RoomManager roomManager;
    private CustomerManager customerManager;
    private ReservationManager reservationManager;
    
    public FrontDeskManager() {
        roomManager = new RoomManager();
        customerManager = new CustomerManager();
        reservationManager = new ReservationManager();
    }
    
    public void checkIn(Customer customer, Room room) {
        roomManager.checkIn(room);
        customerManager.addCustomer(customer);
    }
    
    public void checkOut(Customer customer, Room room) {
        roomManager.checkOut(room);
        customerManager.removeCustomer(customer);
    }
    
    public void reserve(Customer customer, Room room, Date startDate, Date endDate) {
        reservationManager.reserve(customer, room, startDate, endDate);
    }
}
```

### 5.3 财务管理模块

财务管理模块主要负责宾馆的财务管理、报表生成等操作。以下是财务管理模块的代码实现：

```java
public class FinanceManager {
    private List<Reservation> reservations;
    
    public FinanceManager() {
        reservations = new ArrayList<>();
    }
    
    public void addReservation(Reservation reservation) {
        reservations.add(reservation);
    }
    
    public void removeReservation(Reservation reservation) {
        reservations.remove(reservation);
    }
    
    public double getTotalRevenue(Date startDate, Date endDate) {
        double totalRevenue = 0;
        for (Reservation reservation : reservations) {
            if (reservation.getStartDate().after(startDate) && reservation.getEndDate().before(endDate)) {
                totalRevenue += reservation.getRoom().getPrice() * reservation.getDays();
            }
        }
        return totalRevenue;
    }
}
```
