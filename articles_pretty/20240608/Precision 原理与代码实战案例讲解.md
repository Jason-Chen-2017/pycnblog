## 5. 项目实践：代码实例和详细解释说明
在实际项目中，我们经常需要处理高精度计算的问题。以下是一个使用 Python 进行高精度计算的项目实践示例：

```python
# 示例代码
def factorial(n):
    # 使用 decimal 模块进行高精度计算
    with decimal.localcontext() as ctx:
        ctx.prec = 100  # 设置精度为 100 位
        result = decimal.Decimal(1)
        for i in range(1, n + 1):
            result *= decimal.Decimal(i)
        return result

# 调用函数计算阶乘
n = 10
factorial_result = factorial(n)
print(factorial_result)
```

在上述示例中，我们定义了一个名为`factorial`的函数，用于计算一个数的阶乘。在函数内部，我们使用`decimal.localcontext`创建了一个上下文管理器，将精度设置为 100 位。然后，我们使用一个循环计算阶乘，并将结果存储在`result`变量中。最后，我们将计算结果打印出来。

通过使用高精度计算，我们可以确保在处理大数值时不会出现精度损失的问题。
