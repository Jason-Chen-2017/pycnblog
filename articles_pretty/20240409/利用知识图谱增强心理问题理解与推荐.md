## 4. 项目实践：代码实例和详细解释说明

下面我们来看一个基于知识图谱的心理问题理解和推荐的实际应用案例。

```python
import networkx as nx
import matplotlib.pyplot as plt
from py2neo import Graph, Node, Relationship

# 连接 Neo4j 数据库
graph = Graph("bolt://localhost:7687", auth=("neo4j", "your_password"))

# 创建节点和关系
depression = Node("PsychologicalProblem", name="Depression")
anxiety = Node("PsychologicalProblem", name="Anxiety")
insomnia = Node("PsychologicalProblem", name="Insomnia")
therapy = Node("Treatment", name="Therapy")
medication = Node("Treatment", name="Medication")

graph.create(depression)
graph.create(anxiety)
graph.create(insomnia)
graph.create(therapy)
graph.create(medication)

rel1 = Relationship(depression, "CAUSES", insomnia)
rel2 = Relationship(anxiety, "CAUSES", insomnia)
rel3 = Relationship(depression, "TREATED_BY", therapy)
rel4 = Relationship(depression, "TREATED_BY", medication)
rel5 = Relationship(anxiety, "TREATED_BY", therapy)

graph.create(rel1)
graph.create(rel2)
graph.create(rel3)
graph.create(rel4)
graph.create(rel5)

# 查询和可视化
query = "MATCH (n)-[r]-(m) RETURN n,r,m"
result = graph.run(query).data()

G = nx.Graph()
for row in result:
    G.add_node(row["n"]["name"], label=row["n"]["name"], node_type=row["n"].labels[0])
    G.add_node(row["m"]["name"], label=row["m"]["name"], node_type=row["m"].labels[0])
    G.add_edge(row["n"]["name"], row["m"]["name"], label=row["r"].type)

pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True, node_color=['lightblue' if n == 'PsychologicalProblem' else 'lightgreen' for n in G.nodes()])
edge_labels = dict([((u, v,), d['label']) for u, v, d in G.edges(data=True)])
nx.draw_networkx_edge_labels(G, pos, edge_labels=edge_labels)
plt.show()
```

这个代码示例演示了如何使用 Python 和 Neo4j 构建一个简单的心理健康知识图谱,并进行可视化展示。

首先,我们连接到 Neo4j 数据库,并创建了几个代表心理问题和治疗方法的节点。然后,我们建立了这些节点之间的关系,如"导致"、"治疗"等。

接下来,我们运行一个 Cypher 查询来获取所有节点和关系,并使用 NetworkX 库将其转换为图形结构。最后,我们使用 Matplotlib 库对知识图谱进行可视化展示,清楚地展示了各个心理问题之间的联系以及它们与治疗方法的关系。

通过这个简单的示例,我们可以看到知识图谱技术如何帮助我们更好地理解和分析心理健康相关的知识,为后续的诊断和治疗推荐提供支持。
