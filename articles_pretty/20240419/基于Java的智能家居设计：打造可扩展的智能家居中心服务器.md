## 5.项目实践：代码实例和详细解释说明

在这个部分，我们将会通过一个简单的例子来演示如何设计和实现我们的智能家居中心服务器。

### 5.1 设备接口

首先，我们需要设计一个通用的设备接口。这个接口应该包含一些基本的操作，比如打开设备、关闭设备、获取设备状态等。

```java
public interface Device {
    void turnOn();
    void turnOff();
    boolean isOn();
}
```

### 5.2 设备实现

接下来，我们需要为每种设备提供一个实现类。在这个例子中，我们只实现一个简单的灯泡类。

```java
public class LightBulb implements Device {
    private boolean isOn;

    @Override
    public void turnOn() {
        isOn = true;
    }

    @Override
    public void turnOff() {
        isOn = false;
    }

    @Override
    public boolean isOn() {
        return isOn;
    }
}
```

### 5.3 服务器设计

然后，我们需要设计我们的服务器。服务器需要有一个列表来存储所有的设备，以及一些方法来操作这些设备。

```java
public class HomeHub {
    private List<Device> devices;

    public HomeHub() {
        devices = new ArrayList<>();
    }

    public void addDevice(Device device) {
        devices.add(device);
    }

    public void turnOnAllDevices() {
        for (Device device : devices) {
            device.turnOn();
        }
    }

    public void turnOffAllDevices() {
        for (Device device : devices) {
            device.turnOff();
        }
    }
}
```

### 5.4 服务器实现

最后，我们需要实现我们的服务器。实现过程中，我们需要确保服务器可以正确地操作所有的设备，以及处理来自设备的请求。

```java
public class Main {
    public static void main(String[] args) {
        HomeHub homeHub = new HomeHub();
        LightBulb lightBulb = new LightBulb();

        homeHub.addDevice(lightBulb);
        homeHub.turnOnAllDevices();
        homeHub.turnOffAllDevices();
    }
}
```

在这个例子中，我们首先创建了一个HomeHub对象和一个LightBulb对象。然后，我们将LightBulb对象添加到HomeHub中，然后通过HomeHub来操作LightBulb。
