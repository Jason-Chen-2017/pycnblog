## 4. 具体最佳实践：代码实例和详细解释说明

### 4.1 知识图谱构建

知识图谱构建的代码实例如下：

```python
import rdflib

# 创建知识图谱
g = rdflib.Graph()

# 添加实体
entity1 = rdflib.URIRef('http://example.org/entity1')
entity2 = rdflib.URIRef('http://example.org/entity2')
g.add((entity1, rdflib.RDF.type, rdflib.OWL.Class))
g.add((entity2, rdflib.RDF.type, rdflib.OWL.Class))

# 添加关系
relation = rdflib.URIRef('http://example.org/relation')
g.add((entity1, relation, entity2))

# 保存知识图谱
g.serialize('example.rdf', format='xml')
```

### 4.2 知识图谱推理

知识图谱推理的代码实例如下：

```python
import rdflib
from rdflib.plugins.sparql import prepareQuery

# 加载知识图谱
g = rdflib.Graph()
g.parse('example.rdf')

# 定义查询语句
query = prepareQuery('''
    SELECT ?entity1 ?entity2
    WHERE {
        ?entity1 <http://example.org/relation> ?entity2 .
    }
''', initNs={'ex': rdflib.Namespace('http://example.org/')})

# 执行查询
for row in g.query(query):
    print(row)
```

### 4.3 知识图谱可视化

知识图谱可视化的代码实例如下：

```python
import networkx as nx
import matplotlib.pyplot as plt
import rdflib

# 加载知识图谱
g = rdflib.Graph()
g.parse('example.rdf')

# 创建图形对象
G = nx.Graph()

# 添加实体
for entity in g.subjects(rdflib.RDF.type, rdflib.OWL.Class):
    G.add_node(entity)

# 添加关系
for entity1, relation, entity2 in g:
    G.add_edge(entity1, entity2)

# 绘制图形
pos = nx.spring_layout(G)
nx.draw_networkx_nodes(G, pos)
nx.draw_networkx_edges(G, pos)
nx.draw_networkx_labels(G, pos)
plt.show()
```
