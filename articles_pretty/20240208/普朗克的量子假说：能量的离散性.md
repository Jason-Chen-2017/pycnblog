## 2. 核心概念与联系

普朗克的量子假说是基于黑体辐射的实验结果提出的。黑体是一种理想化的物体，它能够吸收所有入射的电磁波，并将其全部转化为热能。黑体辐射的谱线分布规律是指，黑体辐射的能量密度与波长的关系不是连续的，而是呈现出一系列峰值和谷底。这个规律无法用经典物理学的理论来解释，因此需要新的理论来解释。

以下是一个 Python 程序，用于模拟黑体辐射的谱线分布规律：

```python
import numpy as np
import matplotlib.pyplot as plt

h = 6.626e-34  # 普朗克常数
c = 3e8  # 光速
k = 1.38e-23  # 玻尔兹曼常数

def planck(wavelength, temperature):
    a = 2 * h * c ** 2
    b = h * c / (wavelength * k * temperature)
    intensity = a / (wavelength ** 5 * (np.exp(b) - 1))
    return intensity

wavelengths = np.linspace(1e-9, 3e-6, 1000)
temperatures = [3000, 4000, 5000, 6000]

for temperature in temperatures:
    intensity = planck(wavelengths, temperature)
    plt.plot(wavelengths * 1e9, intensity, label=str(temperature) + 'K')

plt.xlabel('Wavelength (nm)')
plt.ylabel('Intensity (W/m^2/nm/sr)')
plt.legend()
plt.show()
```
