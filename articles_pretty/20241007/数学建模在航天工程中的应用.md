## 2. 核心概念与联系
### 2.1 轨道力学
轨道力学是研究航天器在地球或其他天体引力场中的运动规律。其核心在于牛顿运动定律和万有引力定律的应用。轨道力学的基本方程如下：

$$
F = G \frac{m_1 m_2}{r^2}
$$

其中，$F$ 是引力，$G$ 是万有引力常数，$m_1$ 和 $m_2$ 是两个物体的质量，$r$ 是两物体之间的距离。

### 2.2 控制理论
控制理论是研究如何通过外部输入来控制系统的动态行为。在航天工程中，控制理论主要用于控制航天器的姿态和轨道。控制理论的核心在于设计控制律，使得航天器能够按照预定的轨迹运动。

### 2.3 数值模拟
数值模拟是利用计算机模拟物理系统的行为。在航天工程中，数值模拟主要用于预测航天器的运动轨迹和姿态变化。数值模拟的核心在于解微分方程。

### 2.4 核心概念联系
轨道力学、控制理论和数值模拟是相互关联的。轨道力学提供了航天器运动的基本方程，控制理论提供了控制航天器运动的方法，数值模拟则提供了预测和优化航天器运动轨迹的工具。

## 3. 核心算法原理 & 具体操作步骤

```python
def orbit_dynamics(t, y, mu):
    r = y[0:3]
    v = y[3:6]
    r_norm = np.linalg.norm(r)
    r_hat = r / r_norm
    a = -mu * r_hat / r_norm**2
```

```python
def control_law(y, r_ref, v_ref, k_p, k_i, k_d):
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 轨道力学模型
轨道力学模型的核心方程是牛顿运动定律和万有引力定律。具体公式如下：

$$
\frac{d^2 \mathbf{r}}{dt^2} = -\frac{\mu \mathbf{r}}{|\mathbf{r}|^3}
$$

其中，$\mathbf{r}$ 是航天器的位置向量，$\mu$ 是引力常数。

### 4.2 控制理论模型
控制理论模型的核心方程是控制律。具体公式如下：

$$
\mathbf{u} = \mathbf{K} (\mathbf{r}_{\text{ref}} - \mathbf{r}) + \mathbf{K}_i \int (\mathbf{r}_{\text{ref}} - \mathbf{r}) dt + \mathbf{K}_d (\mathbf{v}_{\text{ref}} - \mathbf{v})
$$

其中，$\mathbf{u}$ 是控制输入，$\mathbf{r}_{\text{ref}}$ 和 $\mathbf{v}_{\text{ref}}$ 是参考轨迹，$\mathbf{K}$、$\mathbf{K}_i$ 和 $\mathbf{K}_d$ 是控制增益矩阵。

### 4.3 数值模拟模型
数值模拟模型的核心方程是常微分方程。具体公式如下：

$$
\frac{d\mathbf{y}}{dt} = \mathbf{f}(\mathbf{y}, t)
$$

其中，$\mathbf{y}$ 是状态向量，$\mathbf{f}$ 是动力学方程。

```python
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

def orbit_dynamics(t, y, mu):
    r = y[0:3]
    v = y[3:6]
    r_norm = np.linalg.norm(r)
    r_hat = r / r_norm
    a = -mu * r_hat / r_norm**2
    return np.concatenate((v, a))

def control_law(y, r_ref, v_ref, k_p, k_i, k_d):
    e_r = r_ref - y[0:3]
    e_v = v_ref - y[3:6]
    u = k_p * e_r + k_i * np.cumsum(e_r) + k_d * e_v
    return u

def numerical_simulation(t_span, y0, mu, k_p, k_i, k_d):
    t = np.linspace(t_span[0], t_span[1], 1000)
    sol = solve_ivp(orbit_dynamics, t_span, y0, args=(mu,), method='RK45')
    y = sol.y
    u = control_law(y, r_ref, v_ref, k_p, k_i, k_d)
    return y, u

# 参数设置
mu = 3.986004418e5  # 地球引力常数
r_ref = np.array([7000, 0, 0])  # 参考位置
v_ref = np.array([0, 7.66, 0])  # 参考速度
k_p = np.array([1, 1, 1])
k_i = np.array([0.1, 0.1, 0.1])
k_d = np.array([0.01, 0.01, 0.01])
y0 = np.array([7000, 0, 0, 0, 7.66, 0])

# 模拟
y, u = numerical_simulation((0, 1000), y0, mu, k_p, k_i, k_d)

# 绘图
plt.figure(figsize=(10, 6))
plt.plot(y[0], y[1])
plt.xlabel('x (km)')
plt.ylabel('y (km)')
plt.title('Orbit Trajectory')
plt.grid(True)
plt.show()
```
