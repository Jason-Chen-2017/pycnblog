## 2. 核心概念与联系
### 2.1 图论基础
图论是研究图（网络）的数学分支。图由节点（顶点）和边（连线）组成。图论中的基本概念包括：
- **节点**：图中的基本单元。
- **边**：连接两个节点的连线。
- **度**：节点的连接数。
- **路径**：节点之间的连接序列。
- **环**：从一个节点出发回到自身的路径。
- **连通性**：图中任意两个节点之间是否存在路径。

### 2.2 随机图模型
随机图模型是图论中的一个重要概念，其中节点和边的连接遵循随机过程。最著名的随机图模型是Erdős-Rényi模型（ER模型）。
- **ER模型**：ER模型中，节点之间存在边的概率是固定的。设图有n个节点，每对节点之间存在边的概率为p。ER模型的数学表示为G(n, p)。

### 2.3 小世界网络模型
小世界网络模型是一种具有高聚集性和短路径长度的网络。Watts-Strogatz模型（WS模型）是小世界网络模型的一种实现。
- **WS模型**：WS模型首先构建一个完全图，然后以一定的概率重连边，形成小世界网络。数学表示为G(n, k, p)。

### 2.4 无标度网络模型
无标度网络模型是一种节点度分布遵循幂律分布的网络。Barabási-Albert模型（BA模型）是无标度网络模型的一种实现。
- **BA模型**：BA模型通过节点的优先连接机制生成网络。初始时有m个节点，每次添加一个新节点时，新节点与现有节点以一定的概率连接。数学表示为G(n, m)。

## 3. 核心算法原理 & 具体操作步骤
### 3.1 ER模型
ER模型的生成算法如下：
1. 初始化一个空图，包含n个节点。
2. 遍历每对节点，以概率p生成一条边。
3. 返回生成的图。

```python
def er_model(n, p):
    import random
    graph = {}
    for i in range(n):
        graph[i] = []
    for i in range(n):
        for j in range(i+1, n):
            if random.random() < p:
                graph[i].append(j)
                graph[j].append(i)
    return graph
```

### 3.2 WS模型
WS模型的生成算法如下：
1. 初始化一个完全图，包含n个节点。
2. 以概率p重连边，形成小世界网络。
3. 返回生成的图。

```python
def ws_model(n, k, p):
    import random
    graph = {}
    for i in range(n):
        graph[i] = []
    for i in range(n):
        for j in range(i+1, min(i+k, n)):
            graph[i].append(j)
            graph[j].append(i)
    for i in range(n):
        for j in range(i+1, min(i+k, n)):
            if random.random() < p:
                new_node = random.randint(0, n-1)
                while new_node in graph[i] or new_node in graph[j]:
                    new_node = random.randint(0, n-1)
                graph[i].remove(j)
                graph[j].remove(i)
                graph[i].append(new_node)
                graph[new_node].append(i)
                graph[j].append(new_node)
                graph[new_node].append(j)
    return graph
```

### 3.3 BA模型
BA模型的生成算法如下：
1. 初始化一个包含m个节点的完全图。
2. 每次添加一个新节点，以一定的概率连接到现有节点。
3. 返回生成的图。

```python
def ba_model(n, m):
    import random
    graph = {}
    for i in range(m):
        graph[i] = [i]
    for i in range(m, n):
        graph[i] = []
        for j in range(m):
            if random.random() < m / (i - 1):
                graph[i].append(j)
                graph[j].append(i)
    return graph
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 ER模型
ER模型的数学表示为G(n, p)。ER模型的期望度为np，期望聚集系数为p(1-p) / (1-p + p^2)。

### 4.2 WS模型
WS模型的数学表示为G(n, k, p)。WS模型的期望度为2k，期望聚集系数为(1-p) / (1-p + p^2)。

### 4.3 BA模型
BA模型的数学表示为G(n, m)。BA模型的期望度为2m，期望聚集系数为(1 - 1/m) / (1 - 1/m + 1/m^2)。

def degree_distribution(graph):
    degree = [len(graph[i]) for i in graph]
    degree.sort()
    return degree

def plot_degree_distribution(degree):
    plt.hist(degree, bins=50, density=True)
    plt.xlabel('Degree')
    plt.ylabel('Probability')
    plt.title('Degree Distribution')
    plt.show()

    ws_degree = degree_distribution(ws_graph)
    ba_degree = degree_distribution(ba_graph)
