## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 数学模型
数学模型是用数学语言描述现实世界的问题。例如，我们可以用数学模型来描述一个简单的数学问题：

- **问题描述**：给定一个整数n，判断n是否为素数。
- **数学模型**：定义一个函数is_prime(n)，返回值为True表示n是素数，返回值为False表示n不是素数。

### 4.2 公式
公式是数学模型的具体表达形式。例如，我们可以用公式来描述一个简单的数学问题：

- **公式**：is_prime(n) = True if n > 1 and (for all k in 2 to sqrt(n), n % k != 0) else False

### 4.3 详细讲解
详细讲解如下：

- **is_prime(n)函数**：判断一个整数n是否为素数。
  - **输入**：一个整数n。
  - **输出**：True表示n是素数，False表示n不是素数。
  - **实现**：遍历2到sqrt(n)的所有整数，检查n是否能被这些整数整除。如果不能被整除，则n是素数。

### 4.4 举例说明
举例说明如下：

- **例子1**：判断3是否为素数。
  - **输入**：n = 3。
  - **输出**：True。
  - **解释**：3不能被2整除，因此3是素数。

- **例子2**：判断4是否为素数。
  - **输入**：n = 4。
  - **输出**：False。
  - **解释**：4能被2整除，因此4不是素数。

### 5.2 源代码详细实现和代码解读
源代码详细实现如下：

```python
import math

def is_prime(n):
    if n <= 1:
        return False
    for k in range(2, int(math.sqrt(n)) + 1):
        if n % k == 0:
            return False
    return True

# 测试代码
print(is_prime(3))  # 输出：True
print(is_prime(4))  # 输出：False
```
