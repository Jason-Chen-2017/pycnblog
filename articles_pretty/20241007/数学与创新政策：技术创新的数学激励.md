#### 线性规划问题
给定目标函数和约束条件：
- 目标函数：`maximize z = c^T x`
- 约束条件：`A x <= b`，`x >= 0`

### 4.2 公式

### 4.3 详细讲解
线性规划问题可以通过单纯形法或内点法等算法来求解。这里我们使用单纯形法为例。

#### 单纯形法
1. **初始化**：找到一个初始可行基。
2. **迭代**：
   - **选择进基变量**：选择使目标函数增大的变量。
   - **选择出基变量**：选择使基变量减少的变量。
   - **更新基**：更新基变量。
3. **判断**：当所有非基变量的检验数都不大于零时，停止迭代。

### 5.2 源代码详细实现和代码解读
```python
import numpy as np
from scipy.optimize import linprog

# 目标函数系数
c = [-3, -2]

# 约束条件矩阵
A = [[1, 1], [2, 1]]

# 约束条件右侧值
b = [4, 6]

# 变量非负约束
x0_bounds = (0, None)
x1_bounds = (0, None)

# 求解线性规划问题
res = linprog(c, A_ub=A, b_ub=b, bounds=[x0_bounds, x1_bounds], method='simplex')

# 输出结果
print("最优解：", res.x)
print("最优值：", -res.fun)
```

### 5.3 代码解读与分析
- **目标函数系数**：`c = [-3, -2]`，表示目标函数 `z = -3x - 2y`。
- **约束条件矩阵**：`A = [[1, 1], [2, 1]]`，表示约束条件 `x + y <= 4` 和 `2x + y <= 6`。
- **约束条件右侧值**：`b = [4, 6]`，表示约束条件右侧的值。
- **变量非负约束**：`x0_bounds = (0, None)` 和 `x1_bounds = (0, None)`，表示变量 `x` 和 `y` 非负。
- **求解线性规划问题**：使用 `linprog` 函数求解线性规划问题。
- **输出结果**：输出最优解和最优值。
