### 3.2 图论算法
图论算法是模拟暗物质子结构的关键。我们将使用Dijkstra算法来计算最短路径，以及Prim算法来生成最小生成树。这些算法可以帮助我们更好地理解暗物质子结构的连接性和分布。

#### 3.2.1 Dijkstra算法
Dijkstra算法用于计算图中两个节点之间的最短路径。其伪代码如下：

```plaintext
function Dijkstra(graph, start):
    create vertex set Q
    for each vertex v in graph:
        dist[v] ← INFINITY
        prev[v] ← UNDEFINED
        add v to Q
    dist[start] ← 0
    while Q is not empty:
        u ← vertex in Q with min dist[u]
        remove u from Q
        for each neighbor v of u:
            alt ← dist[u] + length(u, v)
            if alt < dist[v]:
                dist[v] ← alt
                prev[v] ← u
    return dist, prev
```

#### 3.2.2 Prim算法
Prim算法用于生成图的最小生成树。其伪代码如下：

```plaintext
function Prim(graph):
    create vertex set Q
    for each vertex v in graph:
        key[v] ← INFINITY
        parent[v] ← UNDEFINED
        add v to Q
    key[start] ← 0
    while Q is not empty:
        u ← vertex in Q with min key[u]
        remove u from Q
        for each neighbor v of u:
            if v in Q and length(u, v) < key[v]:
                key[v] ← length(u, v)
                parent[v] ← u
    return parent
```

### 4.2 公式
引力模型的基本公式为：

$$
F = G \frac{m_1 m_2}{r^2}
$$

其中，$F$ 是引力，$G$ 是引力常数，$m_1$ 和 $m_2$ 是两个物体的质量，$r$ 是两个物体之间的距离。

def calculate_gravity(positions, mass):

#### 5.2.4 计算最短路径
```python
def calculate_shortest_path(positions, gravity):
    G = nx.Graph()
    for i in range(len(positions)):
        for j in range(i + 1, len(positions)):
            G.add_edge(i, j, weight=gravity[i, j])
    return nx.dijkstra_path(G, source=0, target=len(positions) - 1)
```

#### 5.2.5 生成最小生成树

def generate_minimum_spanning_tree(positions, gravity):
    G = nx.Graph()
    for i in range(len(positions)):
        for j in range(i + 1, len(positions)):
            G.add_edge(i, j, weight=gravity[i, j])
    return nx.minimum_spanning_tree(G)
