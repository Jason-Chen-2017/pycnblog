## 2. 核心概念与联系
### 2.1 小世界网络的定义
小世界网络是一种具有高聚类系数和短路径长度的网络结构。高聚类系数意味着网络中的节点倾向于形成紧密的社区，而短路径长度意味着任意两个节点之间的距离较短。

### 2.2 聚类系数
聚类系数衡量网络中节点之间的紧密程度。具体定义如下：
$$
C = \frac{2 \times \text{数量的三角形}}{\text{数量的三元组}}
$$
其中，三元组是指三个节点两两相连的组合，三角形是指三个节点完全相连的组合。

### 2.3 平均路径长度
平均路径长度衡量网络中任意两个节点之间的平均距离。具体定义如下：
$$
L = \frac{1}{N(N-1)} \sum_{i \neq j} d(i, j)
$$
其中，$N$ 是网络中的节点数，$d(i, j)$ 是节点 $i$ 和节点 $j$ 之间的最短路径长度。

### 2.4 小世界网络的形成机制
小世界网络的形成机制可以通过Watts-Strogatz模型和Barabási-Albert模型来描述。

#### 2.4.1 Watts-Strogatz模型
Watts-Strogatz模型通过以下步骤生成小世界网络：
1. **初始结构**：首先构建一个完全图，每个节点都与所有其他节点相连。
2. **随机重连**：以一定的概率随机重连边，形成随机网络。
3. **参数**：关键参数是重连概率 $p$，当 $p=0$ 时，网络为完全图；当 $p=1$ 时，网络为随机图。

#### 2.4.2 Barabási-Albert模型
Barabási-Albert模型通过以下步骤生成小世界网络：
1. **初始结构**：首先构建一个初始网络，包含 $m_0$ 个节点。
2. **增长过程**：每次添加一个新节点，该节点与现有网络中的 $m$ 个节点相连。
3. **优先连接**：新节点倾向于与高度节点相连，即度数较高的节点更容易被新节点连接。
4. **参数**：关键参数是初始节点数 $m_0$ 和每次添加的新节点数 $m$。

function generate_Watts_Strogatz(N, k, p):
    1. 初始化完全图
    2. 对于每个节点 i:
        3. 从节点 i 的邻居中随机选择 k-1 个节点

    5. 返回生成的网络
```

    1. 初始化初始网络，包含 m_0 个节点

        3. 从现有网络中随机选择 m 个节点
        4. 与这 m 个节点相连
    5. 返回生成的网络
```

### 4.3 举例说明
假设我们有一个包含5个节点的小世界网络，节点之间的连接如下：
```

```
- **聚类系数**：计算三角形和三元组的数量。
  - 三角形：(1, 2, 3) 和 (4, 5)
  - 三元组：(1, 2, 4), (1, 2, 5), (1, 3, 4), (1, 3, 5), (2, 3, 4), (2, 3, 5)
  - 聚类系数 $C = \frac{2 \times 2}{6} = \frac{2}{3}$
- **平均路径长度**：计算每个节点对之间的最短路径长度。
  - (1, 2) = 1
  - (1, 3) = 2
  - (1, 4) = 1
  - (1, 5) = 2
  - (2, 3) = 1
  - (2, 4) = 2
  - (2, 5) = 1
  - (3, 4) = 2
  - (3, 5) = 1
  - (4, 5) = 1
  - 平均路径长度 $L = \frac{1+2+1+2+1+2+1+2+1+1}{10} = 1.4$

### 5.2 源代码详细实现和代码解读
```python
import networkx as nx
import matplotlib.pyplot as plt

def generate_Watts_Strogatz(N, k, p):
    G = nx.watts_strogatz_graph(N, k, p)
    return G

def generate_Barabasi_Albert(N, m):
    G = nx.barabasi_albert_graph(N, m)
    return G

def calculate_clustering_coefficient(G):
    return nx.average_clustering(G)

def calculate_average_path_length(G):
    return nx.average_shortest_path_length(G)

def visualize_network(G):
    plt.figure(figsize=(8, 6))
    nx.draw(G, with_labels=True, node_size=500, node_color='skyblue', font_size=10, font_weight='bold')
    plt.title("Social Network Visualization")
    plt.show()

# 示例代码
N = 10
k = 4
p = 0.2
m = 2

# 生成Watts-Strogatz模型
G_WS = generate_Watts_Strogatz(N, k, p)
print("Watts-Strogatz Clustering Coefficient:", calculate_clustering_coefficient(G_WS))
print("Watts-Strogatz Average Path Length:", calculate_average_path_length(G_WS))

# 生成Barabási-Albert模型
G_BA = generate_Barabasi_Albert(N, m)
print("Barabasi-Albert Clustering Coefficient:", calculate_clustering_coefficient(G_BA))
print("Barabasi-Albert Average Path Length:", calculate_average_path_length(G_BA))

# 可视化网络
visualize_network(G_WS)
visualize_network(G_BA)
```
