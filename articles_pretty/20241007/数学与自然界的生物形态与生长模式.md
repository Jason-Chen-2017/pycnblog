## 2. 核心概念与联系
### 2.1 斐波那契数列
斐波那契数列是一个非常著名的数列，其定义如下：
$$
F(n) = \begin{cases} 
0 & \text{if } n = 0 \\
1 & \text{if } n = 1 \\
F(n-1) + F(n-2) & \text{if } n > 1 
\end{cases}
$$
斐波那契数列在自然界中有着广泛的应用，例如植物的叶子排列、花瓣的数量等。

### 2.4 混沌理论

function fibonacci(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1

#### 3.1.2 迭代方法
```pseudo
function fibonacci(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    a, b = 0, 1
    for i from 2 to n:
        c = a + b
        a = b
        b = c
    return b
```

import numpy as np
