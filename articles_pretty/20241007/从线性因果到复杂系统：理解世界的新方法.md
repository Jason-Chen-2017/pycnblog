## 2. 核心概念与联系

### 2.1 复杂系统的定义
复杂系统是由大量相互作用的组件组成的系统，表现出非线性动力学和自组织特性。复杂系统的核心特征包括：
- **非线性动力学**：系统状态随时间变化的非线性关系。
- **自组织**：系统在特定条件下自发形成有序结构的现象。
- **涌现现象**：复杂系统中局部规则的集合导致全局复杂行为的现象。

### 2.2 核心概念原理
#### 2.2.1 非线性动力学
非线性动力学描述了系统状态随时间变化的非线性关系。非线性动力学系统的特点是系统状态的变化不是简单的线性叠加，而是相互作用的结果。非线性动力学系统可以表现出混沌、周期性、分岔等行为。

#### 2.2.2 混沌理论
混沌理论研究非线性动力学系统中确定性与随机性之间的关系。混沌系统表现出高度敏感的初始条件依赖性，即使初始条件微小的差异也会导致系统行为的巨大差异。混沌理论的核心概念包括：
- **洛伦兹吸引子**：描述混沌系统中周期性行为的典型例子。
- **分岔理论**：研究系统参数变化导致系统行为变化的现象。
- **Lyapunov指数**：衡量系统对初始条件敏感性的指标。

#### 2.2.3 自组织临界性
自组织临界性是指系统在特定条件下自发形成有序结构的现象。自组织临界性系统表现出临界态行为，即系统在临界点附近表现出高度的敏感性和复杂性。自组织临界性系统的核心概念包括：
- **沙堆模型**：描述自组织临界性系统中局部规则的集合导致全局复杂行为的现象。
- **相变理论**：研究系统在临界点附近的行为变化。

### 2.3 核心概念联系
复杂系统的核心概念之间存在紧密的联系。非线性动力学是复杂系统的基础，混沌理论和自组织临界性是复杂系统中常见的现象。复杂系统中的涌现现象是系统局部规则的集合导致全局复杂行为的现象，是复杂系统的核心特征之一。

```python
import numpy as np
import matplotlib.pyplot as plt

def nonlinear_dynamics(u, t, sigma, rho, beta):
    du_dt = sigma * (u[1] - u[0])
    dv_dt = u[0] * (rho - u[2]) - u[1]
    dw_dt = u[0] * u[1] - beta * u[2]
    return [du_dt, dv_dt, dw_dt]

def solve_nonlinear_dynamics(u0, t_span, sigma, rho, beta):
    from scipy.integrate import solve_ivp
    sol = solve_ivp(nonlinear_dynamics, t_span, u0, args=(sigma, rho, beta), dense_output=True)
    return sol

u0 = [1.0, 1.0, 1.0]
t_span = (0, 100)
sigma = 10.0
rho = 28.0
beta = 8.0 / 3.0

sol = solve_nonlinear_dynamics(u0, t_span, sigma, rho, beta)
t = np.linspace(t_span[0], t_span[1], 1000)
u = sol.sol(t)

plt.plot(t, u[:, 0], label='x')
plt.plot(t, u[:, 1], label='y')
plt.plot(t, u[:, 2], label='z')
plt.legend()
```

```python
import numpy as np
import matplotlib.pyplot as plt

def lorenz_attractor(x, y, z, sigma, rho, beta):
    dx_dt = sigma * (y - x)
    dy_dt = x * (rho - z) - y
    dz_dt = x * y - beta * z
    return [dx_dt, dy_dt, dz_dt]

def solve_lorenz_attractor(x0, y0, z0, t_span, sigma, rho, beta):
    from scipy.integrate import solve_ivp
    sol = solve_ivp(lorenz_attractor, t_span, [x0, y0, z0], args=(sigma, rho, beta), dense_output=True)
    return sol

x0, y0, z0 = 1.0, 1.0, 1.0
t_span = (0, 50)
sigma = 10.0
rho = 28.0
beta = 8.0 / 3.0

sol = solve_lorenz_attractor(x0, y0, z0, t_span, sigma, rho, beta)
t = np.linspace(t_span[0], t_span[1], 1000)
x, y, z = sol.sol(t)

fig = plt.figure()
```

```python
import numpy as np
import matplotlib.pyplot as plt

def sandpile_model(h, threshold):
    h_new = np.copy(h)
    for i in range(h.shape[0]):
        for j in range(h.shape[1]):
            if h[i, j] >= threshold:
                h_new[i, j] -= 4
                if i > 0:
                    h_new[i-1, j] += 1
                if i < h.shape[0] - 1:
                    h_new[i+1, j] += 1
                if j > 0:
                    h_new[i, j-1] += 1
                if j < h.shape[1] - 1:
                    h_new[i, j+1] += 1
    return h_new

def simulate_sandpile(h0, threshold, steps):
    h = np.copy(h0)
    for _ in range(steps):
        h = sandpile_model(h, threshold)
    return h

h0 = np.zeros((10, 10))
threshold = 4
steps = 100

h = simulate_sandpile(h0, threshold, steps)
plt.imshow(h, cmap='hot', interpolation='nearest')
plt.colorbar()
plt.title('Sandpile Model')
plt.show()
```
