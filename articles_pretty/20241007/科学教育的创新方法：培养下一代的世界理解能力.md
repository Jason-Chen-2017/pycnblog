## 3. 核心算法原理 & 具体操作步骤
### 3.1 递归算法
递归是一种常见的算法设计方法，通过将问题分解为更小的子问题来解决问题。递归算法的基本思想是将问题分解为一个或多个相同类型的子问题，然后递归地解决这些子问题。递归算法的关键在于定义递归基和递归步骤。

#### 伪代码
```plaintext
function recursive_algorithm(n):
    if n == 0:
        return 0
    else:
        return n + recursive_algorithm(n-1)
```

### 3.2 二分查找算法
二分查找是一种高效的查找算法，适用于有序数组。通过将数组分成两部分，然后递归地查找目标值，可以快速找到目标值。二分查找的关键在于将数组分成两部分，并递归地查找目标值。

#### 伪代码
```plaintext
function binary_search(arr, target, low, high):
    if low > high:
        return -1
    else:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            return binary_search(arr, target, mid+1, high)
        else:
            return binary_search(arr, target, low, mid-1)
```
