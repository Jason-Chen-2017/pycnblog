## 4. 数学模型和公式详细讲解举例说明

### 4.1 线性回归模型

线性回归模型用于预测连续型变量，例如预测酒企的销售额或股价。模型的公式如下：

$$
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n + \epsilon
$$

其中，$y$ 是预测值，$x_1, x_2, ..., x_n$ 是自变量，$\beta_0, \beta_1, ..., \beta_n$ 是模型参数，$\epsilon$ 是误差项。

### 4.2 逻辑回归模型

逻辑回归模型用于预测二元分类变量，例如预测一只股票是否会上涨。模型的公式如下：

$$
P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n)}}
$$

其中，$P(y=1|x)$ 是预测概率，$x_1, x_2, ..., x_n$ 是自变量，$\beta_0, \beta_1, ..., \beta_n$ 是模型参数。


## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 和 scikit-learn 库构建线性回归模型的代码示例：

```python
import pandas as pd
from sklearn.linear_model import LinearRegression

# 加载数据
data = pd.read_csv('wine_data.csv')

# 选择自变量和因变量
X = data[['GDP', 'CPI', 'Income']]
y = data['Sales']

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 预测
y_pred = model.predict(X)

# 评估模型
print(model.score(X, y))
```

**代码解释：**

1.  首先，使用 pandas 库加载数据。
2.  然后，选择自变量和因变量。
3.  接着，创建线性回归模型。
4.  之后，使用 fit() 方法训练模型。
5.  最后，使用 predict() 方法进行预测，并使用 score() 方法评估模型的性能。
