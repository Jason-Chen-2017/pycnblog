### 2.1 群论基础

群论研究的是数学结构中的“群”，即一组元素和一个结合运算，满足封闭性、结合律、存在单位元以及每个元素有逆元的性质。群的概念适用于描述系统中的对称性变换，如旋转、反射、平移等。

### 2.2 李群与李代数

李群是连续群的一种特例，其元素构成一个拓扑空间，并且在每个点处都带有连续的乘法运算和逆运算。李群的导出李代数是一个赋有向量空间结构的李代数，它是李群在无穷小尺度下的近似描述。李代数通过李括号运算来刻画李群的局部性质，而李群则通过其上的流形结构来描述全局性质。

### 2.3 李群与李代数的关系

李群与其导出的李代数之间存在密切联系。李群的结构可以通过其对应的李代数来研究，反之亦然。李群的李代数决定了李群的局部结构，而李群则决定了李代数的全局结构。这一关系在理论物理中尤为重要，因为许多物理系统的对称性都可以通过李群和李代数来描述和分析。

以下是一个简单的例子，展示如何使用Python计算SU(2)群的李代数元素的李括号运算：

```python
import numpy as np

class su2_matrix:
    def __init__(self, mat):
        self.mat = np.array(mat)

    def __add__(self, other):
        return su2_matrix(self.mat + other.mat)

    def __neg__(self):
        return su2_matrix(-self.mat)

    def __mul__(self, scalar):
        return su2_matrix(self.mat * scalar)

    def __rmul__(self, scalar):
        return self * scalar

    def adjoint(self):
        return np.array([[0, self.mat[0][1], -self.mat[1][0]],
                         [-self.mat[0][1], 0, self.mat[0][0]],
                         [self.mat[1][0], -self.mat[0][0], 0]])

    def commutator(self, other):
        return self.adjoint() @ other.mat - other.adjoint() @ self.mat

su2_a = su2_matrix([[0, 1], [-1, 0]])
su2_b = su2_matrix([[0, -1j], [1j, 0]])

print(\"su2_a = \", su2_a.mat)
print(\"su2_b = \", su2_b.mat)

comm = su2_a.commutator(su2_b)
print(\"Commutator [su2_a, su2_b] = \", comm.mat)
```

这段代码定义了一个名为`su2_matrix`的类，用于表示SU(2)群的元素，并实现了加法、减法、标量乘法和李括号运算（即李代数的通勤运算）。通过这个简单的例子，可以直观地看到如何在程序中实现李群和李代数的基本操作。
