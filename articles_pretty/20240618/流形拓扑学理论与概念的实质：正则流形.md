## 2. 核心概念与联系

### 流形的定义与性质

- **流形**：流形是一类具有局部欧几里德结构的数学空间。在每个点周围，流形都可被映射到欧几里德空间的一个小区域，保持局部的一致性和连贯性。这种结构使得流形能够描述各种几何对象，从简单的曲线到复杂的高维空间。

### 正则流形的定义

- **正则流形**：正则流形是在流形的基础上引入了额外的光滑性假设。这意味着流形上的每一个点都存在一个局部坐标系，使得在该坐标系下的过渡变换满足一定的光滑性条件。这样的流形在数学分析、物理理论和数据科学中具有更广泛的适用性和更强的理论基础。

# 定义流形上的距离度量
def manifold_distance(point1, point2):
    return cdist([point1], [point2], metric='euclidean')[0][0]

# 数据映射至流形上的平均值计算
def manifold_mean(data, manifold_distance):
    n_samples = len(data)
    mean_point = data[0]
    for i in range(1, n_samples):
        mean_point = manifold_distance(mean_point, data[i])
    return mean_point

manifold_mean(data, manifold_distance)
