## 4. 数学模型和公式详细讲解举例说明

在本节中，我们将介绍一个简单的数学模型，即独立事件模型（Independent Events Model）。该模型假设每个事件发生时，不考虑其他事件的存在。以下是一个简单的公式：

P(A ∩ B) = P(A) \\* P(B)

其中，A 和 B 分别表示两个独立事件，P(A) 和 P(B) 表示事件 A 和事件 B 的概率。这个公式可以用于计算多个事件之间的关系，从而帮助我们理解信息传播行为。

```python
import numpy as np
from sklearn.ensemble import RandomForestClassifier

# 数据加载
data = np.load('social_network_data.npy')

# 特征提取
X = data[:, :-1]
y = data[:, -1]

# 模型训练

# 预测
```
