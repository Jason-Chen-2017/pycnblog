## 1. 背景介绍

### 1.1 随机现象与概率论

概率论，作为数学的一个重要分支，研究的是随机现象的规律性。随机现象，顾名思义，指的是在相同条件下，可能出现不同结果的现象。例如，抛掷一枚硬币，可能出现正面或反面；掷骰子，可能出现1到6中的任意一个点数。

### 1.2 随机变量的引入

为了更好地研究随机现象，我们引入了随机变量的概念。随机变量，简单来说，就是将随机现象的结果数值化。它可以是一个数字，也可以是一个向量，甚至可以是一个函数。

## 2. 核心概念与联系

### 2.1 随机变量的分类

根据取值的不同，随机变量可以分为两大类：

*   **离散型随机变量**：取值只能是有限个或可数无限个的变量。例如，抛掷一枚硬币的结果（正面或反面）、掷骰子的点数（1到6）。
*   **连续型随机变量**：取值可以在某个区间内连续变化的变量。例如，一个人的身高、一棵树的高度。

### 2.2 分布函数与概率密度函数

*   **分布函数**：描述随机变量在任意取值点处的累积概率。
*   **概率密度函数**：描述连续型随机变量在任意取值点处的概率密度。

### 2.3 期望与方差

*   **期望**：随机变量取值的平均值，反映了随机变量的集中趋势。
*   **方差**：随机变量取值的分散程度，反映了随机变量的波动大小。

## 3. 核心算法原理具体操作步骤

### 3.1 离散型随机变量

对于离散型随机变量，我们可以通过列举所有可能的取值及其对应的概率来描述其分布。例如，抛掷一枚硬币，正面和反面的概率都是1/2。

### 3.2 连续型随机变量

对于连续型随机变量，我们需要使用概率密度函数来描述其分布。例如，正态分布的概率密度函数为：

$$
f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$\mu$ 为期望，$\sigma$ 为标准差。

## 4. 数学模型和公式详细讲解举例说明

### 4.1 伯努利分布

伯努利分布是最简单的离散型随机变量分布之一，它描述的是只有两种可能结果（成功或失败）的事件的概率分布。例如，抛掷一枚硬币，正面为成功，反面为失败，其概率分布为：

$$
P(X = 1) = p, P(X = 0) = 1 - p
$$

其中，$p$ 为成功的概率。

### 4.2 正态分布

正态分布是最常见的连续型随机变量分布之一，它在自然界和社会科学中都有广泛的应用。例如，人的身高、体重、考试成绩等都近似服从正态分布。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 Python 代码模拟抛硬币

```python
import random

def toss_coin():
    """模拟抛硬币"""
    if random.random() < 0.5:
        return "正面"
    else:
        return "反面"

# 模拟抛硬币 10 次
for _ in range(10):
    print(toss_coin())
```

### 5.2 Python 代码生成正态分布随机数

```python
import numpy as np

# 生成 1000 个服从标准正态分布的随机数
random_numbers = np.random.randn(1000)

# 打印前 10 个随机数
print(random_numbers[:10])
```
