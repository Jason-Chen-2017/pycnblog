## 2. 核心概念与联系

AIAgentWorkFlow 主要包含以下核心概念：

*   **Agent：** Agent 是 AIAgentWorkFlow 中的基本执行单元，它可以执行一个或多个任务。
*   **Task：** Task 是 Agent 执行的具体操作，例如数据预处理、模型训练、模型推理等。
*   **Workflow：** Workflow 是由多个 Task 组成的流程，它定义了 AI 应用的执行逻辑。
*   **Pipeline：** Pipeline 是一组按顺序执行的 Task，它可以用于数据处理、模型训练等场景。
*   **Model：** Model 是训练好的机器学习模型，它可以用于预测、分类等任务。

这些核心概念之间存在着紧密的联系，例如：

*   Agent 可以执行多个 Task，Task 可以属于多个 Workflow。
*   Workflow 可以包含多个 Pipeline，Pipeline 可以包含多个 Task。
*   Model 可以用于 Task 中，例如模型训练 Task 和模型推理 Task。
