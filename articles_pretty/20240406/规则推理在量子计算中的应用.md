## 4. 项目实践：代码实例和详细解释说明
我们以经典命题逻辑中的"modus ponens"规则为例,展示量子规则推理的具体实现。

"modus ponens"规则可以表示为:
- 如果 $p$ 则 $q$
- $p$ 成立
- 因此 $q$ 成立

我们将其编码到量子态中:
- 量子比特$\ket{p}$和$\ket{q}$分别表示命题$p$和$q$
- 蕴含关系$p\rightarrow q$编码为受控非门$CNOT(\ket{p},\ket{q})$

量子电路如下所示:
```
   ┌───┐
q_0: ┤ X ├
   └───┘
   ┌───┐
q_1: ┤ H ├
   └───┘
```

其中,第一个量子比特表示命题$p$,第二个量子比特表示命题$q$。受控非门$CNOT$实现了蕴含关系的推理,而哈达马德门$H$则用于准备量子叠加态。

通过测量第二个量子比特,我们就可以得到命题$q$的结果。这个量子电路实现了并行高效的规则推理。
