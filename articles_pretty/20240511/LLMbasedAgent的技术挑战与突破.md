## 1. 背景介绍

### 1.1 人工智能发展浪潮

人工智能技术近年来发展迅猛，其中以深度学习为代表的机器学习算法取得了突破性进展，并在图像识别、自然语言处理、语音识别等领域取得了令人瞩目的成果。然而，传统的深度学习模型往往局限于特定任务，难以适应复杂多变的现实世界环境。为了解决这个问题，研究人员开始探索能够自主学习、适应环境、完成复杂任务的智能体（Agent）。

### 1.2  LLM-based Agent 的兴起

近年来，大型语言模型（Large Language Model，LLM）的出现为构建更加智能的 Agent 带来了新的机遇。LLM 具有强大的文本理解和生成能力，能够从海量数据中学习知识，并根据用户指令生成自然流畅的文本。将 LLM 与 Agent 结合，可以构建 LLM-based Agent，使其具备更强的推理、规划、决策能力，从而更好地解决现实世界中的复杂问题。
