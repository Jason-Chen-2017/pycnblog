## 4. 数学模型和公式详细讲解举例说明
### 4.1 工程量计算模型
#### 4.1.1 土方工程量计算公式
$$V = \frac{(A_1 + A_2)}{2} \times L$$
其中，$V$ 表示土方工程量，$A_1$ 和 $A_2$ 分别表示两个断面面积，$L$ 表示两个断面之间的距离。
#### 4.1.2 钢筋工程量计算公式
$$W = \rho \times V$$
其中，$W$ 表示钢筋重量，$\rho$ 表示钢筋的密度，$V$ 表示钢筋体积。
#### 4.1.3 混凝土工程量计算公式
$$V = L \times B \times H$$
其中，$V$ 表示混凝土体积，$L$、$B$、$H$ 分别表示长、宽、高。
### 4.2 预算编制模型
#### 4.2.1 人工费用计算公式
$$C_L = \sum_{i=1}^n Q_i \times P_i$$
其中，$C_L$ 表示人工费用，$Q_i$ 表示第 $i$ 种人工的工作量，$P_i$ 表示第 $i$ 种人工的单价，$n$ 表示人工种类数。
#### 4.2.2 材料费用计算公式  
$$C_M = \sum_{j=1}^m Q_j \times P_j$$
其中，$C_M$ 表示材料费用，$Q_j$ 表示第 $j$ 种材料的用量，$P_j$ 表示第 $j$ 种材料的单价，$m$ 表示材料种类数。
#### 4.2.3 机械费用计算公式
$$C_E = \sum_{k=1}^l Q_k \times P_k$$
其中，$C_E$ 表示机械费用，$Q_k$ 表示第 $k$ 种机械的工作量，$P_k$ 表示第 $k$ 种机械的单价，$l$ 表示机械种类数。

### 5.3 预算编制模块代码实现
#### 5.3.1 人工费用计算
```java
public class LaborCostCalculator {
    public static BigDecimal calculate(List<LaborItem> laborItems) {
        BigDecimal totalCost = BigDecimal.ZERO;
        for (LaborItem item : laborItems) {
            BigDecimal cost = item.getQuantity().multiply(item.getUnitPrice());
            totalCost = totalCost.add(cost);
        }
        return totalCost;
    }
}
```
LaborCostCalculator类提供了计算人工费用的静态方法，根据人工项列表中各项的工作量和单价计算总费用。
