## 1. 背景介绍

### 1.1 概率论的舞台

概率论，就像一个宏大的舞台，上演着随机现象的精彩戏剧。在这个舞台上，随机变量便是那一个个鲜活的角色，它们用自身的取值，演绎着各种可能的剧情。

### 1.2 随机变量的登场

随机变量，顾名思义，就是取值带有随机性的变量。它们不同于普通的变量，有着固定的值，而是像一个个调皮的精灵，在概率的指引下，跳跃于不同的数值之间。

### 1.3 离散与连续：两种不同的性格

随机变量的世界里，存在着两种截然不同的性格：离散型和连续型。离散型随机变量，就像一个个跳跃的音符，只能取有限个或可数个孤立的数值；而连续型随机变量，则像一条流畅的旋律，可以在某个区间内取任何值。


## 2. 核心概念与联系

### 2.1 概率分布：随机变量的剧本

每个随机变量都有自己的剧本，那就是它的概率分布。概率分布描述了随机变量取不同值的可能性大小，就像是为随机变量的演出设定了不同的出场概率。

### 2.2 离散型随机变量的剧本：概率质量函数

对于离散型随机变量，它们的剧本被称为概率质量函数 (PMF)。PMF 将每个可能的取值对应到一个概率值，就像是为每个音符标注了出现的频率。

### 2.3 连续型随机变量的剧本：概率密度函数

连续型随机变量的剧本则被称为概率密度函数 (PDF)。PDF 描述了随机变量在某个区间内取值的概率密度，就像是描绘了旋律在不同音高上的起伏变化。

### 2.4 期望与方差：随机变量的个性

除了剧本，每个随机变量还有自己的个性，那就是它的期望和方差。期望描述了随机变量取值的平均水平，就像是演员的平均身高；而方差则描述了随机变量取值的分散程度，就像是演员的身高差异。


## 3. 核心算法原理具体操作步骤

### 3.1 离散型随机变量的概率计算

计算离散型随机变量取某个特定值的概率，只需要查阅它的 PMF 即可。例如，掷一枚骰子，点数为 3 的概率就是 PMF 在 3 处的取值。

### 3.2 连续型随机变量的概率计算

计算连续型随机变量落在某个区间的概率，需要对 PDF 进行积分。例如，测量一个人的身高，落在 1.70 米到 1.80 米之间的概率就是 PDF 在该区间上的积分值。

### 3.3 期望与方差的计算

期望的计算方法，对于离散型随机变量，是将每个取值与其对应的概率相乘后求和；对于连续型随机变量，则是对取值与其对应的 PDF 乘积进行积分。方差的计算方法，则是对每个取值与其期望的差的平方与其对应概率的乘积求和或积分。


## 4. 数学模型和公式详细讲解举例说明

### 4.1 伯努利分布：抛硬币的模型

伯努利分布是最简单的离散型随机变量模型，它描述了只有两种可能结果的试验，例如抛硬币。其 PMF 为：

$$
P(X = k) = p^k (1-p)^{1-k}, \quad k \in \{0, 1\}
$$

其中，$p$ 表示成功的概率。

### 4.2 正态分布：自然界的常客

正态分布是最常见的连续型随机变量模型，它描述了许多自然现象，例如身高、体重等。其 PDF 为：

$$
f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$\mu$ 表示期望，$\sigma$ 表示标准差。


## 5. 项目实践：代码实例和详细解释说明

### 5.1 Python 代码模拟抛硬币

```python
import random

def toss_coin():
  """模拟抛硬币"""
  if random.random() < 0.5:
    return "正面"
  else:
    return "反面"

# 模拟抛硬币 10 次
for _ in range(10):
  print(toss_coin())
```

### 5.2 Python 代码生成正态分布随机数

```python
import numpy as np

# 生成 10 个期望为 0，标准差为 1 的正态分布随机数
random_numbers = np.random.randn(10)
print(random_numbers)
```
