## 5.项目实践：代码实例和详细解释说明

在项目实践中，我们以Python和SQLite为例，展示了一个简单的个人通信簿管理系统的实现。首先，我们需要创建一个SQLite数据库，并定义联系人表：

```python
import sqlite3

conn = sqlite3.connect('contact_book.db')

c = conn.cursor()

# 创建联系人表
c.execute('''
    CREATE TABLE contacts
    (id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    phone TEXT NOT NULL,
    city TEXT NOT NULL);
    ''')

conn.commit()
conn.close()
```

然后，我们可以实现添加、删除、修改、查询联系人的功能：

```python
def add_contact(name, phone, city):
    conn = sqlite3.connect('contact_book.db')
    c = conn.cursor()
    c.execute("INSERT INTO contacts (name, phone, city) VALUES (?, ?, ?)", (name, phone, city))
    conn.commit()
    conn.close()

def delete_contact(id):
    conn = sqlite3.connect('contact_book.db')
    c = conn.cursor()
    c.execute("DELETE FROM contacts WHERE id=?", (id,))
    conn.commit()
    conn.close()

def update_contact(id, name, phone, city):
    conn = sqlite3.connect('contact_book.db')
    c = conn.cursor()
    c.execute("UPDATE contacts SET name=?, phone=?, city=? WHERE id=?", (name, phone, city, id))
    conn.commit()
    conn.close()

def query_contact(city):
    conn = sqlite3.connect('contact_book.db')
    c = conn.cursor()
    c.execute("SELECT * FROM contacts WHERE city=?", (city,))
    contacts = c.fetchall()
    conn.close()
    return contacts
```
