## 4. 数学模型和公式详细讲解举例说明

### 4.1 关联规则分析

关联规则分析用于发现数据项之间的关联关系，例如购买手机的用户也倾向于购买手机壳。

*   支持度：数据项集在数据集中出现的频率。
*   置信度：规则前件出现时，规则后件也出现的概率。
*   提升度：规则前件出现时，规则后件出现概率的提升程度。

### 4.2 聚类分析

聚类分析用于将数据对象划分为不同的类别，例如将用户根据购买行为划分为不同的用户群体。

*   K-means 聚类：将数据对象划分为 K 个簇，使得簇内对象相似度最大化，簇间对象相似度最小化。
*   层次聚类：通过不断合并或分裂簇，形成树状结构的聚类结果。

## 5. 项目实践：代码实例和详细解释说明

```python
# 导入必要的库
import pandas as pd
import matplotlib.pyplot as plt

# 读取电子产品销售数据
data = pd.read_csv('electronic_products_sales.csv')

# 对不同品类产品的销售额进行统计
sales_by_category = data.groupby('Category')['Sales'].sum()

# 创建柱状图
plt.bar(sales_by_category.index, sales_by_category.values)
plt.xlabel('Category')
plt.ylabel('Sales')
plt.title('Sales by Category')
plt.show()
```
