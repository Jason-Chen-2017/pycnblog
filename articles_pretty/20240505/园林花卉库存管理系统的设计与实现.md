## 5.项目实践：代码实例和详细解释说明

下面以Python和MySQL为例，展示如何实现数据查询功能。首先，我们需要安装Python的MySQL驱动，然后连接数据库，并执行SQL语句。

```python
import mysql.connector

# 连接数据库
cnx = mysql.connector.connect(user='root', password='password', host='127.0.0.1', database='inventory')

# 创建cursor对象
cursor = cnx.cursor()

# 执行SQL语句
query = ("SELECT name, quantity FROM flowers WHERE quantity < %s")
cursor.execute(query, (20,))

# 获取查询结果
for (name, quantity) in cursor:
  print("Flower: {}, Quantity: {}".format(name, quantity))

# 关闭连接
cursor.close()
cnx.close()
```

在这段代码中，我们首先连接数据库，然后创建一个cursor对象，用于执行SQL语句。然后，我们执行一个SQL查询语句，查询库存量小于20的花卉。最后，我们遍历查询结果，并打印出花卉的名称和数量。
