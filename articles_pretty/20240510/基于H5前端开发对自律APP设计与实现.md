## 4. 数学模型和公式详细讲解举例说明

为了让APP具备更强大的功能,背后需要一些数学模型和公式作为支撑。本章节将详细讲解其中的几个关键模型。

### 4.1 习惯养成的幂律分布

$P(X=k) = Ck^{-\alpha}$ 

其中 $C$ 和 $\alpha$ 是常数。这个幂律分布可以用来预测坚持一个习惯的概率。

### 4.2 时间使用的帕累托法则

把时间按重要性分成 A、B、C 三类,其比例大致为:
$$
\begin{aligned}
A : B : C &= 15\% : 20\% : 65\% \\
\text{Contribution} &= 65\% : 20\% : 15\%
\end{aligned}
$$

通过聚焦 A 类最重要的任务,可以用20%的时间获得80%的成果。

### 4.3 激励系统的频率控制

如果奖励太频繁,就失去了激励作用;如果太少,又起不到正强化的效果。可以用如下的变换函数控制奖励频率:

$$
f(x) = \frac{1}{1+e^{-\alpha(x-\beta)}} 
$$

### 5.2 番茄钟倒计时

```js
let endTime, timer
function start() {
  endTime = +new Date() + 1000 * 60 * 25
  timer = setInterval(countdown, 1000)
}

function countdown() {
  const remain = endTime - new Date()
  if(remain < 0) {
    clearInterval(timer)
    return
  }
  const m = String(Math.floor(remain / 60000)).padStart(2, '0')
  const s = String(Math.floor(remain / 1000) % 60).padStart(2, '0')
  document.querySelector('.time').innerHTML = `${m}:${s}`
}
```

### 5.3 习惯打卡页

```html
<ul id="habits"></ul>
```

```js
const habits = [
  { name: '早起', days: 10 },
  { name: '跑步', days: 3 },
  { name: '冥想', days: 7 } 
]
const html = habits.map(h => `
  <li>
    <span>${h.name}</span>
    <span>${h.days}天</span>
    <button>打卡</button>
  </li>
`).join('')

document.querySelector('#habits').innerHTML = html
```
