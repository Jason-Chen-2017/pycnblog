### 4. 数学模型和公式详细讲解举例说明

#### 4.1 净现值 (NPV)

NPV 是指未来现金流的现值总和减去初始投资额。它用于评估投资项目的盈利能力。

$$
NPV = \sum_{t=1}^{n} \frac{CF_t}{(1+r)^t} - Initial Investment
$$

其中：

*   $CF_t$ 是第 t 年的现金流
*   $r$ 是折现率
*   $n$ 是项目的生命周期

#### 4.2 投资回报率 (ROI)

ROI 是指投资收益与投资成本的比率。它用于衡量投资项目的效率。

$$
ROI = \frac{Net Profit}{Investment Cost} \times 100\%
$$

### 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 计算 NPV 的代码示例：

```python
def npv(rate, cashflows):
    """
    计算净现值 (NPV)

    Args:
        rate: 折现率
        cashflows: 现金流列表

    Returns:
        净现值
    """
    total = 0.0
    for i, cashflow in enumerate(cashflows):
        total += cashflow / (1 + rate)**i
    return total
```
