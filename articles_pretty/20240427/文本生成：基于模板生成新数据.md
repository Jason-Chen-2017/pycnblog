模板是基于模板文本生成方法的核心要素，它定义了生成文本的结构和内容框架。模板可以是简单的字符串，也可以是包含占位符的复杂结构。占位符用于表示需要填充的具体内容，例如：

```
"今天天气{天气}，气温{气温}摄氏度。"
```

在上述模板中，`{天气}` 和 `{气温}` 就是占位符，分别表示天气状况和气温数值。

### 2.2 数据源

数据源是用于填充模板占位符的真实数据来源。数据源可以是结构化数据，例如数据库、电子表格等；也可以是非结构化数据，例如文本文件、网页等。

### 2.3 生成规则

```python
def generate_weather_report(weather, temperature):
    template = "今天天气{weather}，气温{temperature}摄氏度。"
    report = template.format(weather=weather, temperature=temperature)
    return report
```
