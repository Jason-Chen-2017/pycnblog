### 4.1 注意力机制的数学模型

注意力机制的核心思想是计算查询向量（query）与键向量（key）之间的相似度，并以此作为值向量（value）的权重。常用的注意力机制模型包括：

* **点积注意力（Dot-product attention）**： $$Attention(Q, K, V) = softmax(\frac{QK^T}{\sqrt{d_k}})V$$

* **缩放点积注意力（Scaled dot-product attention）**： $$Attention(Q, K, V) = softmax(\frac{QK^T}{\sqrt{d_k}})V$$

* **加性注意力（Additive attention）**： $$Attention(Q, K, V) = softmax(v^T tanh(W_1Q + W_2K))V$$

其中，Q、K、V分别表示查询向量、键向量和值向量，$d_k$表示键向量的维度，$W_1$、$W_2$、$v$表示可学习的参数。

2. 将服装数据库中的图像信息和文本信息分别编码为特征向量，作为键向量K和值向量V。
3. 使用注意力机制计算每个服装与用户之间的相似度，并以此作为权重，对服装的值向量进行加权求和。
4. 将加权求和后的结果作为最终的推荐列表。

import torch
import torch.nn as nn

class MultimodalFusionModel(nn.Module):
    def __init__(self, image_dim, text_dim, hidden_dim):
        super().__init__()
        self.image_linear = nn.Linear(image_dim, hidden_dim)
        self.text_linear = nn.Linear(text_dim, hidden_dim)
        self.attention = nn.MultiheadAttention(hidden_dim, num_heads=4)
        self.output_linear = nn.Linear(hidden_dim, 1)

    def forward(self, image, text):
        image_emb = self.image_linear(image)
        text_emb = self.text_linear(text)
        fused_emb, _ = self.attention(image_emb, text_emb, text_emb)
        output = self.output_linear(fused_emb)
        return output
