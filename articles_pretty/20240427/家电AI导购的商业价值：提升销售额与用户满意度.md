家电AI导购常用的推荐模型包括：

*   **协同过滤 (CF)**：根据相似用户的行为推荐产品。
*   **基于内容的推荐 (CB)**：根据产品特征推荐相似产品。
*   **混合推荐**：结合CF和CB，提高推荐效果。

### 4.1 协同过滤

协同过滤算法基于以下假设：相似用户会喜欢相似的产品。CF算法分为基于用户的CF和基于物品的CF。

*   **基于用户的CF**：找到与目标用户相似的用户，并将这些用户喜欢的产品推荐给目标用户。
*   **基于物品的CF**：找到与目标用户喜欢的产品相似的产品，并将这些产品推荐给目标用户。

CF算法的数学模型可以使用余弦相似度或皮尔逊相关系数来计算用户或物品之间的相似度。

### 4.2 基于内容的推荐

基于内容的推荐算法根据产品特征推荐相似产品。例如，如果一个用户喜欢购买大容量的洗衣机，那么可以向他推荐其他大容量的洗衣机。

CB算法的数学模型可以使用TF-IDF等文本相似度算法来计算产品之间的相似度。

```python
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv('appliances.csv')

# 文本特征提取
tfidf = TfidfVectorizer()
tfidf_matrix = tfidf.fit_transform(data['description'])

# 计算相似度
similarity = cosine_similarity(tfidf_matrix)

# 推荐产品
def recommend_products(product_id):
    # 获取相似产品
    similar_products = similarity[product_id].argsort()[::-1][1:]
    
    # 返回相似产品列表
    return data.iloc[similar_products]['product_name'].tolist()

# 示例
product_id = 10
recommended_products = recommend_products(product_id)
print(f"与产品 {data.iloc[product_id]['product_name']} 相似的产品：")
print(recommended_products)
```
