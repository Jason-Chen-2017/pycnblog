## 5. 项目实践：代码实例和详细解释说明

以下是一个使用BERT模型进行实体识别的代码示例：

```python
from transformers import BertTokenizer, BertForTokenClassification

# 加载预训练模型和词表
model_name = "bert-base-uncased"
tokenizer = BertTokenizer.from_pretrained(model_name)
model = BertForTokenClassification.from_pretrained(model_name)

# 输入文本
text = "Apple's new iPhone 14 is available in four colors."

# 将文本转换为模型输入
inputs = tokenizer(text, return_tensors="pt")

# 模型预测
outputs = model(**inputs)

# 获取预测结果
predictions = outputs.logits.argmax(-1)

# 将预测结果转换为实体标签
labels = [model.config.id2label[p] for p in predictions[0].tolist()]

# 输出实体标签
print(labels)
```

输出结果：

```
['B-ORG', 'O', 'B-PRO', 'I-PRO', 'O', 'O', 'O']
```

其中，'B-ORG'表示组织实体的开始，'B-PRO'表示产品实体的开始，'I-PRO'表示产品实体的延续，'O'表示其他。
