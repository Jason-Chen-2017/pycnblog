### 4.1 词袋模型

词袋模型是一种简单的文本表示方法，它将文本视为无序的词集合，忽略语法和词序。每个词都表示为一个特征，其值表示该词在文本中出现的次数。例如，以下两条评论：

* "这个玩具太棒了！我的孩子很喜欢它。"
* "这个玩具质量很差，很快就坏了。"

可以用以下词袋模型表示：

| 词语 | 玩具 | 太棒了 | 孩子 | 喜欢 | 质量 | 差 | 坏了 |
|---|---|---|---|---|---|---|---|
| 评论1 | 1 | 1 | 1 | 1 | 0 | 0 | 0 |
| 评论2 | 1 | 0 | 0 | 0 | 1 | 1 | 1 |

### 4.2 词嵌入

词嵌入是一种更复杂的文本表示方法，它将词语映射到高维向量空间，捕捉词语之间的语义关系。例如，"汽车" 和 "卡车" 的词向量在向量空间中会比较接近，而 "汽车" 和 "苹果" 的词向量则会比较远。

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 和 NLTK 库进行情感分析的示例代码：

```python
import nltk
from nltk.sentiment import SentimentIntensityAnalyzer

# 初始化情感分析器
sia = SentimentIntensityAnalyzer()

# 示例评论
text = "这个玩具太棒了！我的孩子很喜欢它。"

# 分析情感
sentiment = sia.polarity_scores(text)

# 打印情感分数
print(sentiment)
```

输出结果如下：

```
{'neg': 0.0, 'neu': 0.325, 'pos': 0.675, 'compound': 0.8074}
```

* `neg`：消极情感分数
* `neu`：中性情感分数
* `pos`：积极情感分数
* `compound`：综合情感分数，范围为 -1（非常消极）到 1（非常积极）
