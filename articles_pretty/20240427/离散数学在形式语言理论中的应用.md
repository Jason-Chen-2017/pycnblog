## 2. 核心概念与联系

### 2.1 形式语言

形式语言是符号的有穷序列，这些符号来自一个有限的字母表。例如，以下是一个形式语言的例子：

```
L = {a^n b^n | n ≥ 0}
```

这个语言由所有形如 a^n b^n 的字符串组成，其中 n 是一个非负整数。例如，字符串 "ab"、"aabb" 和 "aaabbb" 都属于这个语言，而字符串 "aab"、"aba" 和 "ba" 都不属于这个语言。

### 2.2 形式语法

形式语法是一组规则，用于描述一个形式语言。形式语法通常用产生式来表示。例如，以下是一个形式语法的例子：

```
S -> aSb | ε
```

这个语法有两个产生式：

* S -> aSb：这个产生式表示可以用字符串 "a"、非终结符 "S" 和字符串 "b" 来替换非终结符 "S"。
* S -> ε：这个产生式表示可以用空字符串来替换非终结符 "S"。

这个语法生成的语言与上面提到的语言 L 相同。

### 2.3 自动机

自动机是一个数学模型，用于识别形式语言。自动机可以分为两类：有限状态自动机和下推自动机。

* 有限状态自动机 (FSA) 是一种简单的自动机，它只有一个状态集和一个转移函数。FSA 可以识别正则语言。
* 下推自动机 (PDA) 是一种比 FSA 更强大的自动机，它有一个栈。PDA 可以识别上下文无关语言。

## 3. 核心算法原理具体操作步骤

### 3.1 正则语言的识别算法

正则语言可以用 FSA 来识别。FSA 的识别算法如下：

1. 从 FSA 的起始状态开始。
2. 读取输入字符串的第一个符号。
3. 根据当前状态和输入符号，找到下一个状态。
4. 重复步骤 2 和 3，直到读取完输入字符串的最后一个符号。
5. 如果最终状态是接受状态，则接受输入字符串；否则，拒绝输入字符串。

### 3.2 上下文无关语言的识别算法

上下文无关语言可以用 PDA 来识别。PDA 的识别算法如下：

1. 从 PDA 的起始状态和空栈开始。
2. 读取输入字符串的第一个符号。
3. 根据当前状态、输入符号和栈顶符号，找到下一个状态、执行的操作和压入栈的符号。
4. 重复步骤 2 和 3，直到读取完输入字符串的最后一个符号。
5. 如果最终状态是接受状态且栈为空，则接受输入字符串；否则，拒绝输入字符串。
