## 2. 核心概念与联系

### 2.1 极限与连续

极限是微积分的基础，描述了当自变量趋近于某个值时，函数值的变化趋势。连续性则要求函数在定义域内没有间断点。

### 2.2 导数

导数描述了函数在某一点处的瞬时变化率，即函数值的增量与自变量增量的比值在极限情况下的值。导数是优化算法的核心，用于寻找函数的极值点。

### 2.3 积分

积分描述了函数曲线下的面积，可以理解为将曲线分割成无数个小的矩形，然后将这些矩形的面积求和。积分在概率统计、物理学等领域都有广泛应用。

## 3. 核心算法原理

### 3.1 导数计算

*   **定义法:** 使用导数的定义式，通过求极限来计算导数。
*   **求导法则:** 常见的求导法则包括幂函数求导、指数函数求导、对数函数求导、三角函数求导等，可以简化导数的计算过程。

### 3.2 积分计算

*   **不定积分:** 求导数的逆运算，即找到一个函数，使其导数等于被积函数。
*   **定积分:** 计算函数曲线在一定区间内的面积，可以使用牛顿-莱布尼茨公式，将定积分转化为不定积分的计算。

## 4. 数学模型和公式

### 4.1 导数定义

$$
f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}
$$

### 4.2 常见求导公式

*   幂函数: $(x^n)' = nx^{n-1}$
*   指数函数: $(e^x)' = e^x$
*   对数函数: $(\ln x)' = \frac{1}{x}$
*   正弦函数: $(\sin x)' = \cos x$
*   余弦函数: $(\cos x)' = -\sin x$

### 4.3 牛顿-莱布尼茨公式

$$
\int_a^b f(x) dx = F(b) - F(a)
$$

其中，$F(x)$ 是 $f(x)$ 的一个原函数。

## 5. 项目实践

### 5.1 Python 代码示例

import numpy as np

# 定义函数
def f(x):
  return x**2

# 计算导数

# 计算定积分
