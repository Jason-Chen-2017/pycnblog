## 1. 背景介绍

### 1.1 AI与数据建模

人工智能 (AI) 的兴起离不开数据。数据是 AI 的燃料，而数据建模则是将原始数据转化为可用于 AI 算法的有效表示的关键步骤。在数据建模过程中，理解和应用概率分布至关重要，尤其是针对离散型数据的建模。

### 1.2 离散概率分布的意义

离散概率分布描述了离散型随机变量的概率规律。它们为我们提供了强大的工具来理解和预测事件发生的可能性，例如：

*   文本分析中，词语在文档中出现的频率
*   图像识别中，像素值的分布
*   推荐系统中，用户点击特定商品的概率

## 2. 核心概念与联系

### 2.1 随机变量与概率分布

随机变量表示随机现象的数值结果，而概率分布则描述了每个可能取值出现的概率。

*   **离散型随机变量:** 取值只能是有限个或可数无限个，例如抛硬币的结果（正面或反面）。
*   **连续型随机变量:** 取值可以在某个范围内连续变化，例如人的身高。

### 2.2 常见的离散概率分布

*   **伯努利分布:** 描述单次试验中事件发生或不发生的概率，例如抛硬币的结果。
*   **二项分布:** 描述 n 次独立重复试验中事件发生 k 次的概率，例如抛 10 次硬币，出现 3 次正面的概率。
*   **泊松分布:** 描述单位时间或空间内事件发生的平均次数，例如一小时内网站的访问量。
*   **几何分布:** 描述首次成功前所需试验次数的概率，例如投篮命中前需要投篮的次数。

## 3. 核心算法原理具体操作步骤

### 3.1 概率质量函数 (PMF)

PMF 描述了离散型随机变量每个取值的概率。例如，对于抛硬币的伯努利分布，PMF 为：

$$
P(X = k) = \begin{cases}
p, & \text{if } k = 1 \\
1-p, & \text{if } k = 0
\end{cases}
$$

其中，$p$ 表示正面朝上的概率，$k$ 表示结果（1 表示正面，0 表示反面）。

### 3.2 累积分布函数 (CDF)

CDF 描述了随机变量取值小于或等于某个特定值的概率。例如，对于上述伯努利分布，CDF 为：

$$
F(x) = \begin{cases}
0, & \text{if } x < 0 \\
1-p, & \text{if } 0 \leq x < 1 \\
1, & \text{if } x \geq 1
\end{cases}
$$

### 3.3 期望值和方差

期望值表示随机变量的平均值，方差表示随机变量的离散程度。

*   期望值：$E(X) = \sum_{k} k \cdot P(X = k)$
*   方差：$Var(X) = E[(X - E(X))^2]$

## 4. 数学模型和公式详细讲解举例说明

### 4.1 伯努利分布

伯努利分布是最简单的离散概率分布之一，它描述了单次试验中事件发生或不发生的概率。

*   参数：$p$ (事件发生的概率)
*   PMF：$P(X = k) = p^k (1-p)^{1-k}$，其中 $k \in \{0, 1\}$
*   期望值：$E(X) = p$
*   方差：$Var(X) = p(1-p)$

**示例：** 抛一枚硬币，正面朝上的概率为 0.5，则正面朝上的期望值为 0.5，方差为 0.25。

### 4.2 二项分布

二项分布描述了 n 次独立重复试验中事件发生 k 次的概率。

*   参数：$n$ (试验次数), $p$ (事件发生的概率)
*   PMF：$P(X = k) = {n \choose k} p^k (1-p)^{n-k}$，其中 $k \in \{0, 1, ..., n\}$
*   期望值：$E(X) = np$
*   方差：$Var(X) = np(1-p)$

**示例：** 抛 10 次硬币，正面朝上的概率为 0.5，则出现 3 次正面的概率为 ${10 \choose 3} (0.5)^3 (0.5)^7 \approx 0.117$。

## 5. 项目实践：代码实例和详细解释说明

**Python 代码示例 (使用 NumPy 和 SciPy 库):**

```python
import numpy as np
from scipy.stats import bernoulli, binom

# 伯努利分布
p = 0.7  # 事件发生的概率
rv = bernoulli(p)
x = rv.rvs(10)  # 生成 10 个随机样本
print(x)

# 二项分布
n = 10  # 试验次数
p = 0.5  # 事件发生的概率
rv = binom(n, p)
x = rv.pmf(k=3)  # 计算 k=3 时的概率
print(x)
```
