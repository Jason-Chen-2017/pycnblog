## 4. 数学模型和公式

### 4.1 协同过滤

协同过滤算法通常使用矩阵分解或近邻算法来计算用户和商品之间的相似度。例如，余弦相似度公式可以用来衡量两个用户或商品之间的相似程度：

$$
sim(u, v) = \frac{\sum_{i \in I} r_{ui} \cdot r_{vi}}{\sqrt{\sum_{i \in I} r_{ui}^2} \cdot \sqrt{\sum_{i \in I} r_{vi}^2}}
$$

其中，$u$ 和 $v$ 表示两个用户或商品，$I$ 表示所有商品集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分。

## 5. 项目实践：代码实例

以下是一个使用 Python 实现的简单协同过滤算法示例：

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(data.pivot_table(index='userId', columns='itemId', values='rating'))

# 为用户推荐商品
def recommend(user_id, num_recommendations):
    # 找到与目标用户最相似的用户
    similar_users = user_similarity[user_id].argsort()[::-1][1:num_recommendations+1]
    # 获取相似用户喜欢的商品
    recommendations = data[data['userId'].isin(similar_users)]['itemId'].unique()
    return recommendations
```
