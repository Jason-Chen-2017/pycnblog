### 2.1 规则

规则是规则匹配的核心，用于定义文本中需要识别和标注的模式或实体。规则通常由以下元素组成：

* **模式**: 指代文本中需要匹配的字符串或正则表达式。
* **标签**: 用于标识匹配到的文本的类别或属性。
* **条件**: 可选的附加条件，用于限制规则的适用范围。

例如，以下规则用于识别文本中的日期：

```
模式: \d{4}-\d{2}-\d{2}
标签: 日期
```

### 2.2 正则表达式

正则表达式是一种强大的文本匹配工具，可以用于定义复杂的模式。正则表达式使用特殊的符号和语法，可以匹配各种文本模式，例如：

* **字符类**: `[abc]` 匹配 a、b 或 c 中的任意一个字符。
* **数量词**: `*` 匹配前面的字符出现 0 次或多次。
* **分组**: `(abc)` 将 abc 作为一个整体进行匹配。

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 实现的简单规则匹配示例：

```python
import re

# 定义规则
rules = [
    (r"\d{4}-\d{2}-\d{2}", "日期"),
    (r"[A-Z][a-z]+", "人名"),
]

# 文本预处理
text = "今天是 2024-04-26，我遇到了 John。"
tokens = text.split()

# 规则匹配
for token in tokens:
    for pattern, label in rules:
        if re.match(pattern, token):
            print(f"{token}: {label}")
```

输出结果：

```
2024-04-26: 日期
John: 人名
```
