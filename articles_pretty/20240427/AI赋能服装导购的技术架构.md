## 4. 数学模型和公式详细讲解举例说明

### 4.1 用户画像构建

* K-means聚类算法：将用户划分为不同的群体，构建用户画像。
* $$ J(C) = \sum_{k=1}^{K}\sum_{x_i \in C_k} ||x_i - \mu_k||^2 $$
* 其中，$J(C)$表示聚类误差，$K$表示聚类数量，$x_i$表示第$i$个用户的数据，$\mu_k$表示第$k$个聚类的中心点。

### 4.2 服装推荐算法

* 余弦相似度：计算用户或服装之间的相似度。
* $$ similarity(x, y) = \frac{x \cdot y}{||x|| \cdot ||y||} $$
* 其中，$x$和$y$分别表示两个用户或服装的特征向量。

import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载用户数据
user_data = pd.read_csv('user_data.csv')

# 计算用户之间的相似度矩阵
similarity_matrix = cosine_similarity(user_data)

# 获取目标用户的相似用户列表
target_user_id = 1
similar_users = similarity_matrix[target_user_id].argsort()[::-1]

# 推荐相似用户购买过的服装
recommendations = []
for user_id in similar_users:
    recommendations.extend(user_data[user_data['user_id'] == user_id]['item_id'].tolist())

# 去重并输出推荐结果
recommendations = list(set(recommendations))
print(recommendations)
