$$

$$

$$

其中，$e_i$ 表示第 $i$ 个实体的向量表示，$r_k$ 表示第 $k$ 个关系的向量表示，$d$ 表示向量维度。

### 4.2 关系评分函数

$$

其中，$M_k$ 是关系 $r_k$ 的矩阵表示。

## 5. 项目实践：代码实例

以下是一个简单的 Python 代码示例，演示如何使用 Neo4j 图数据库构建知识图谱，并进行简单的查询：

```python
from py2neo import Graph

# 连接 Neo4j 数据库
graph = Graph("bolt://localhost:7687", auth=("neo4j", "password"))

# 创建节点
graph.run("CREATE (p:Person {name: 'John'})")
graph.run("CREATE (m:Movie {title: 'The Matrix'})")

# 创建关系
graph.run("MATCH (p:Person {name: 'John'}), (m:Movie {title: 'The Matrix'}) CREATE (p)-[:WATCHED]->(m)")

# 查询
result = graph.run("MATCH (p:Person)-[:WATCHED]->(m:Movie) RETURN p.name, m.title")

# 打印结果
for record in result:
    print(record["p.name"], "watched", record["m.title"])
```
