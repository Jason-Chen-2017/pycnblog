### 4.1 SIR传染病模型
#### 4.1.1 SIR模型的基本假设
#### 4.1.2 SIR模型的微分方程
$$
\begin{aligned}
\frac{dS}{dt} &= -\beta SI \\
\frac{dI}{dt} &= \beta SI - \gamma I \\
\frac{dR}{dt} &= \gamma I
\end{aligned}
$$
其中，$S$表示易感者，$I$表示感染者，$R$表示康复者，$\beta$表示传染率，$\gamma$表示康复率。

### 4.2 时间序列预测模型
#### 4.2.1 ARIMA模型
$$
\phi(B)(1-B)^dX_t = \theta(B)Z_t
$$
其中，$\phi(B)$是p阶自回归系数多项式，$\theta(B)$是q阶移动平均系数多项式，$d$是差分阶数，$Z_t$是白噪声序列。

## 5. 项目实践：代码实例和详细解释说明
### 5.1 数据获取与预处理
#### 5.1.1 使用Python从多个数据源获取疫情数据
```python
import pandas as pd

# 从腾讯新闻获取疫情数据
url = 'https://view.inews.qq.com/g2/getOnsInfo?name=disease_h5'
data = pd.read_json(url)
data = data['data'].apply(pd.Series)
```
#### 5.1.2 数据清洗与转换
```python
# 提取中国各省市数据
china_data = data[data['countryName']=='中国'] 
china_data = china_data[['provinceName','currentConfirmedCount','confirmedCount','suspectedCount','curedCount','deadCount']]
```
#### 5.1.3 数据入库
```python
from sqlalchemy import create_engine

# 连接MySQL数据库
engine = create_engine('mysql+pymysql://user:password@localhost:3306/covid19')
# 数据写入数据库
china_data.to_sql('province_data', engine, if_exists='replace', index=False)
```

import streamlit as st
import plotly.express as px

# 设置页面标题
st.set_page_config(page_title='新冠疫情实时监控面板', layout='wide')

# 侧边栏

# 各省市疫情数据
else:

                         locations='provinceName', 
                         locationmode='country names',
                         color='confirmedCount',

    province = st.selectbox('请选择省份', china_data['provinceName'].unique())
    
    province_data = china_data[china_data['provinceName']==province]
    st.table(province_data)
