在推荐算法中，我们可以使用 Python 语言和 scikit-learn 库来实现协同过滤算法。下面是一个协同过滤算法的 Python 代码示例：

```python
from sklearn.metrics.pairwise import cosine_similarity

def recommend(user, ratings):
    similarity = cosine_similarity(ratings)
    similar_users = similarity[user]
    recommended_items = ratings[similar_users > 0.5].mean(axis=0)
    return recommended_items
```

这个 Python 代码的意思是根据用户的评分数据，计算用户之间的相似度，然后根据相似用户的评分数据，推荐合适的旅游服务。
