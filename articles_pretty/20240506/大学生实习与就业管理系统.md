## 4.数学模型和公式详细讲解举例说明

### 4.1 协同过滤推荐
#### 4.1.1 用户-物品评分矩阵
$$
R=
\begin{bmatrix}
r_{11} & r_{12} & \cdots & r_{1n}\\
r_{21} & r_{22} & \cdots & r_{2n}\\
\vdots & \vdots & \ddots & \vdots\\
r_{m1} & r_{m2} & \cdots & r_{mn}\\
\end{bmatrix}
$$
其中$r_{ui}$表示用户$u$对物品$i$的评分。

#### 4.1.2 相似度计算
用户$u$和用户$v$的相似度可以用余弦相似度计算：

$sim(u,v) = \frac{\sum_{i\in I_{uv}}r_{ui}r_{vi}}{\sqrt{\sum_{i\in I_u}r_{ui}^2}\sqrt{\sum_{i\in I_v}r_{vi}^2}}$

其中$I_{uv}$是用户$u$和$v$共同评分的物品集合，$I_u$和$I_v$分别是用户$u$和$v$评分的物品集合。

#### 4.1.3 预测评分
对于用户$u$未评分的物品$i$，可以根据其他相似用户的评分来预测：

$\hat{r}_{ui} = \frac{\sum_{v\in S^k_u(i)}sim(u,v)r_{vi}}{\sum_{v\in S^k_u(i)}|sim(u,v)|}$

其中$S^k_u(i)$是与用户$u$最相似的$k$个用户中对物品$i$有评分的用户集合。

### 4.2 简历文本分类
#### 4.2.1 TF-IDF权重
单词$t$在文档$d$中的TF-IDF权重为：

$tfidf_{t,d} = tf_{t,d} \times idf_t$

其中，$tf_{t,d}$是单词频率，$idf_t$是逆文档频率：

$idf_t = \log \frac{N}{n_t}$

$N$为总文档数，$n_t$为包含单词$t$的文档数。

#### 4.2.2 朴素贝叶斯分类
对于简历$d$，其属于类别$c$的后验概率为：

$P(c|d) = \frac{P(c)P(d|c)}{P(d)} \propto P(c)\prod_{i=1}^nP(w_i|c)$

其中，$P(c)$是类别$c$的先验概率，$P(w_i|c)$是类别$c$下单词$w_i$的条件概率，$n$是简历$d$的单词数。

### 4.3 关联规则挖掘
#### 4.3.1 支持度
项集$X$的支持度定义为包含$X$的事务占总事务的比例：

$support(X) = \frac{freq(X)}{N}$

其中，$freq(X)$是包含项集$X$的事务数，$N$为总事务数。

#### 4.3.2 置信度
关联规则$X \Rightarrow Y$的置信度定义为：

$confidence(X \Rightarrow Y) = \frac{support(X \cup Y)}{support(X)}$

表示在包含$X$的事务中，同时包含$Y$的比例。

#### 4.3.3 Apriori算法
Apriori算法基于以下两个原理：
- 所有非空频繁项集的子集也是频繁的
- 如果一个项集是非频繁的，那么它的所有超集也是非频繁的

算法流程如下：
1. 生成长度为1的候选项集$C_1$
2. 扫描事务数据库，计算$C_1$中每个项集的支持度，得到频繁1-项集$L_1$
3. 由$L_{k-1}$生成候选$k$项集$C_k$  
4. 扫描事务数据库，计算$C_k$中每个项集的支持度，得到频繁$k$-项集$L_k$
5. 重复3-4直到无法生成新的频繁项集

### 5.2 爬虫与数据采集
使用Python的requests库和BeautifulSoup库编写爬虫，抓取企业和岗位信息：

```python
import requests
from bs4 import BeautifulSoup

def crawl_company(url):
    r = requests.get(url)
    soup = BeautifulSoup(r.text, 'html.parser')
    name = soup.select('.company-name')[0].text
    industry = soup.select('.company-industry')[0].text
    desc = soup.select('.company-description')[0].text
    return name, industry, desc

def crawl_job(url):
    r = requests.get(url)
    soup = BeautifulSoup(r.text, 'html.parser')
    company = soup.select('.job-company')[0].text
    title = soup.select('.job-title')[0].text
    desc = soup.select('.job-description')[0].text
    requirement = soup.select('.job-requirement')[0].text
    return company, title, desc, requirement
```

### 5.3 简历解析与推荐
使用Python的docx库解析简历，jieba库进行分词，gensim库进行文本相似度计算：

```python
import docx
import jieba
from gensim import corpora, models, similarities

def parse_resume(file):
    doc = docx.Document(file)
    text = []
    for para in doc.paragraphs:
        text.append(para.text)
    return ' '.join(text)

def recommend_job(resume, job_desc):
    resume_words = list(jieba.cut(resume))
    job_words = list(jieba.cut(job_desc))
    dictionary = corpora.Dictionary([resume_words, job_words])
    resume_vec = dictionary.doc2bow(resume_words)
    job_vec = dictionary.doc2bow(job_words)
    similarity = similarities.MatrixSimilarity([job_vec])[resume_vec]
    return similarity[0]
```

import pandas as pd
import matplotlib.pyplot as plt

def analyze_student_data(data):
    df = pd.read_csv(data)
    
    # 专业分布
    major_count = df['major'].value_counts()
    plt.pie(major_count, labels=major_count.index, autopct='%1.1f%%')
    plt.title('专业分布')
    plt.show()
    
    # 实习比例
    intern_ratio = df['internship'].value_counts(normalize=True)
    plt.bar(intern_ratio.index, intern_ratio)
    plt.title('实习比例')
    plt.show()

def analyze_company_data(data):
