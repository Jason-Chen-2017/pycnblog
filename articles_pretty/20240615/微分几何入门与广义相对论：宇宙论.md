# 微分几何入门与广义相对论：宇宙论

## 1. 背景介绍

在现代物理学中，广义相对论是描述宇宙大尺度结构和演化的基础理论。它由爱因斯坦在1915年提出，将引力描述为时空的几何性质。微分几何作为数学的一个分支，为广义相对论提供了语言和工具。本文旨在探讨微分几何与广义相对论的基本概念，以及它们在宇宙论中的应用。

## 2. 核心概念与联系

微分几何研究的是曲面和流形的局部性质，以及它们的微分和积分性质。广义相对论中的基本概念是时空，它可以被视为一个四维的流形。在这个流形上，引力不再是一种力，而是由时空的曲率引起的。物体在时空中的运动遵循测地线，这是微分几何中的一个核心概念。
