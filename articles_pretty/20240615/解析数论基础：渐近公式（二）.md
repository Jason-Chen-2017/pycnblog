## 3.核心算法原理具体操作步骤

大O符号表示法是一种渐近公式，它用来描述函数的增长速度上界。具体来说，如果存在正常数C和自然数N，使得当x>N时，函数f(x)满足f(x)<=C*g(x)，则我们可以写成f(x)=O(g(x))。其中，g(x)是一个已知的函数，称为渐近函数。

大O符号表示法的核心思想是，当自变量趋近于无穷大时，函数的增长速度可以用一个已知的函数来描述。例如，当我们研究一个算法的时间复杂度时，可以用大O符号表示法来描述算法的最坏情况时间复杂度。

```python
import math

def prime_count(x):
    primes = [2]
    for i in range(3, x+1):
        is_prime = True
        for j in primes:
            if j > math.sqrt(i):
                break
            if i % j == 0:
                is_prime = False
                break
        if is_prime:
            primes.append(i)
    return len(primes)

def pi(x):
    return prime_count(x)

def main():
    x = 1000000
    print("pi({}) = {}".format(x, pi(x)))
    print("pi({}) = O({})".format(x, x/math.log(x)))

if __name__ == "__main__":
    main()
```
