## 3. 核心算法原理具体操作步骤

### 3.1 欧几里得距离

欧几里得距离是最常用的距离度量方法之一，它计算两个向量之间的直线距离。对于两个 n 维向量 $x$ 和 $y$，它们的欧几里得距离计算公式如下：

$$
d(x, y) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}
$$

### 3.2 余弦相似度

余弦相似度计算两个向量夹角的余弦值，它更关注向量之间的方向而不是大小。对于两个 n 维向量 $x$ 和 $y$，它们的余弦相似度计算公式如下：

$$
\cos(\theta) = \frac{x \cdot y}{||x|| \cdot ||y||}
$$

### 4.1 欧几里得距离的性质

*   **非负性**：距离总是大于或等于零。
*   **同一性**：当且仅当两个向量相等时，它们的距离为零。
*   **对称性**：$d(x, y) = d(y, x)$。
*   **三角不等式**：$d(x, z) \leq d(x, y) + d(y, z)$。

### 4.2 余弦相似度的性质

*   **取值范围**：余弦相似度的取值范围为 $[-1, 1]$，值越接近 1 表示向量越相似，值越接近 -1 表示向量越不相似。
*   **与向量长度无关**：余弦相似度只关注向量之间的方向，与向量长度无关。


## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 计算欧几里得距离和余弦相似度的示例代码：

```python
import numpy as np

def euclidean_distance(x, y):
    """计算欧几里得距离"""
    return np.sqrt(np.sum((x - y)**2))

def cosine_similarity(x, y):
    """计算余弦相似度"""
    return np.dot(x, y) / (np.linalg.norm(x) * np.linalg.norm(y))

# 示例向量
x = np.array([1, 2, 3])
y = np.array([4, 5, 6])

# 计算距离和相似度
distance = euclidean_distance(x, y)
similarity = cosine_similarity(x, y)

print("欧几里得距离:", distance)
print("余弦相似度:", similarity) 
```
