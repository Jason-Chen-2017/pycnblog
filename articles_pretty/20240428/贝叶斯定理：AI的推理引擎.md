## 2. 核心概念与联系

### 2.1 先验概率 (Prior Probability)

先验概率是指在观察到任何新的证据之前，我们对事件发生概率的初始信念。它通常基于我们过去的经验、知识或直觉。

### 2.2 后验概率 (Posterior Probability)

后验概率是指在观察到新的证据之后，我们对事件发生概率的更新信念。它是根据贝叶斯定理，结合先验概率和新的证据计算得出的。

### 2.3 似然度 (Likelihood)

似然度是指在给定某个假设的情况下，观察到特定证据的概率。它反映了证据对假设的支持程度。

### 2.4 贝叶斯定理公式

贝叶斯定理可以用以下公式表示：

$$P(A|B) = \frac{P(B|A) * P(A)}{P(B)}$$

其中：

* $P(A|B)$ 是后验概率，表示在观察到证据 B 的情况下，事件 A 发生的概率。
* $P(B|A)$ 是似然度，表示在事件 A 发生的条件下，观察到证据 B 的概率。
* $P(A)$ 是先验概率，表示在观察到任何证据之前，事件 A 发生的概率。
* $P(B)$ 是证据 B 发生的概率，它是一个归一化常数，确保后验概率的总和为 1。


## 3. 核心算法原理具体操作步骤

贝叶斯推理的基本步骤如下：

1. **确定先验概率:** 根据现有的知识和经验，确定事件 A 发生的先验概率 $P(A)$。
2. **确定似然度:** 确定在事件 A 发生的条件下，观察到证据 B 的概率 $P(B|A)$。
3. **计算后验概率:** 使用贝叶斯定理公式，计算在观察到证据 B 的情况下，事件 A 发生的概率 $P(A|B)$。
4. **更新信念:** 根据计算出的后验概率，更新我们对事件 A 发生的信念。

### 4.1 朴素贝叶斯分类器

朴素贝叶斯分类器是贝叶斯方法在机器学习中的一个经典应用。它基于一个简单的假设，即每个特征都是条件独立的，这意味着一个特征的值不影响其他特征的值。

假设我们有一个数据集，其中每个样本都由 n 个特征 $x_1, x_2, ..., x_n$ 和一个类别标签 y 组成。朴素贝叶斯分类器的目标是根据给定的特征向量 x，预测其所属的类别 y。

朴素贝叶斯分类器的公式如下：

$$P(y|x) = \frac{P(x|y) * P(y)}{P(x)}$$

其中：

* $P(y|x)$ 是后验概率，表示在给定特征向量 x 的情况下，样本属于类别 y 的概率。
* $P(x|y)$ 是似然度，表示在类别 y 的条件下，观察到特征向量 x 的概率。由于朴素贝叶斯假设特征条件独立，因此可以将似然度分解为每个特征的似然度的乘积：

$$P(x|y) = P(x_1|y) * P(x_2|y) * ... * P(x_n|y)$$

* $P(y)$ 是先验概率，表示类别 y 出现的概率。
* $P(x)$ 是特征向量 x 出现的概率，它是一个归一化常数。
