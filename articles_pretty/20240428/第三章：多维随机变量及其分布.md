## 第三章：多维随机变量及其分布

### 1. 背景介绍

在现实生活中，我们常常会遇到多个随机变量同时出现的情况。例如，研究一个地区的降雨量和气温，分析学生的数学和语文成绩，等等。这些情况下，单个随机变量无法完全描述问题的特征，我们需要引入多维随机变量的概念来进行分析。

### 2. 核心概念与联系

**2.1 多维随机变量**

如果 $X_1, X_2, ..., X_n$ 是定义在同一个样本空间 $\Omega$ 上的 n 个随机变量，则称 $(X_1, X_2, ..., X_n)$ 为 n 维随机变量。

**2.2 联合分布函数**

n 维随机变量 $(X_1, X_2, ..., X_n)$ 的联合分布函数定义为：

$$F(x_1, x_2, ..., x_n) = P(X_1 \le x_1, X_2 \le x_2, ..., X_n \le x_n)$$

**2.3 边缘分布函数**

n 维随机变量 $(X_1, X_2, ..., X_n)$ 的边缘分布函数是指其中某一个随机变量的分布函数。例如，$X_1$ 的边缘分布函数为：

$$F_1(x_1) = P(X_1 \le x_1) = F(x_1, +\infty, ..., +\infty)$$

**2.4 条件分布函数**

n 维随机变量 $(X_1, X_2, ..., X_n)$ 的条件分布函数是指在给定某些随机变量的取值条件下，其他随机变量的分布函数。例如，在 $X_2 = x_2, ..., X_n = x_n$ 的条件下，$X_1$ 的条件分布函数为：

$$F_{X_1|X_2,...,X_n}(x_1|x_2,...,x_n) = P(X_1 \le x_1 | X_2 = x_2, ..., X_n = x_n)$$

### 3. 核心算法原理具体操作步骤

**3.1 求解联合分布函数**

联合分布函数可以通过对联合概率密度函数进行积分得到。

**3.2 求解边缘分布函数**

边缘分布函数可以通过对联合分布函数进行积分得到。

**3.3 求解条件分布函数**

条件分布函数可以通过联合分布函数和边缘分布函数的比值得到。

### 4. 数学模型和公式详细讲解举例说明

**4.1 二维正态分布**

二维正态分布是常用的多维随机变量分布之一，其概率密度函数为：

$$f(x,y) = \frac{1}{2\pi\sigma_x\sigma_y\sqrt{1-\rho^2}}exp\left\{-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_x)^2}{\sigma_x^2}-2\rho\frac{(x-\mu_x)(y-\mu_y)}{\sigma_x\sigma_y}+\frac{(y-\mu_y)^2}{\sigma_y^2}\right]\right\}$$

其中，$\mu_x, \mu_y$ 为均值，$\sigma_x, \sigma_y$ 为标准差，$\rho$ 为相关系数。

**4.2 多项分布**

多项分布是描述 n 个事件发生概率的分布，其概率函数为：

$$P(X_1 = x_1, X_2 = x_2, ..., X_n = x_n) = \frac{n!}{x_1!x_2!...x_n!}p_1^{x_1}p_2^{x_2}...p_n^{x_n}$$

其中，$x_i$ 为事件 $i$ 发生的次数，$p_i$ 为事件 $i$ 发生的概率。
