### 4.1 协同过滤

协同过滤是一种基于用户行为数据的推荐算法，其基本思想是： 

*   **基于用户的协同过滤 (User-based CF):** 找到与目标用户兴趣相似的用户，将这些用户喜欢的商品推荐给目标用户。
*   **基于物品的协同过滤 (Item-based CF):** 找到与目标用户喜欢的商品相似的商品，将这些商品推荐给目标用户。

协同过滤算法的核心是计算用户或物品之间的相似度，常用的相似度计算方法包括：

*   **余弦相似度:**  $$ sim(u,v) = \frac{\vec{u} \cdot \vec{v}}{||\vec{u}||||\vec{v}||} $$
*   **皮尔逊相关系数:** $$ sim(u,v) = \frac{cov(u,v)}{\sigma_u \sigma_v} $$

### 4.2 深度学习

深度学习模型可以通过学习用户和物品的隐含特征，进行更加精准的推荐。常用的深度学习模型包括：

*   **深度神经网络 (DNN):** 可以学习用户和物品的非线性关系。
*   **卷积神经网络 (CNN):** 可以学习图像、文本等数据的局部特征。

## 5. 项目实践：代码实例

以下是一个基于Python的简单协同过滤代码示例：

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(data.pivot_table(index='userId', columns='itemId', values='rating'))

# 为用户推荐商品
def recommend(userId, n_recommendations):
    # 找到与目标用户最相似的K个用户
    similar_users = user_similarity[userId].argsort()[-n_recommendations-1:-1]
    
    # 获取相似用户喜欢的商品
    similar_items = data[data['userId'].isin(similar_users)]['itemId'].unique()
    
    # 推荐给目标用户
    return similar_items

# 推荐5个商品给用户ID为1的用户
recommendations = recommend(1, 5)

# 打印推荐结果
print(recommendations)
```
