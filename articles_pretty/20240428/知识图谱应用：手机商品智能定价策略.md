### 4.1 线性回归模型

线性回归模型是一种常用的价格预测模型，其公式如下：

$$
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n
$$

其中，$y$ 表示手机价格，$x_i$ 表示影响手机价格的因素，$\beta_i$ 表示模型参数。

### 4.2 决策树模型

决策树模型是一种基于树形结构的分类和回归模型，其原理是根据特征值将数据划分成不同的子集，并对每个子集进行预测。

from sklearn.linear_model import LinearRegression

# 读取数据

# 特征选择

X = data[features]

# 模型训练

model.fit(X, y)

# 价格预测
new_data = pd.DataFrame([[‘Apple’, ‘iPhone 14’, 6.1, 6, 128]], columns=features)
predicted_price = model.predict(new_data)

# 打印预测结果
print(predicted_price)

*   接着，创建线性回归模型，并使用训练数据进行训练。
*   最后，利用训练好的模型对新数据进行价格预测，并打印预测结果。
