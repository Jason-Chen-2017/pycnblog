## 4. 数学模型和公式详细讲解举例说明

### 4.1 线性回归

线性回归是一种用于建模因变量和一个或多个自变量之间关系的简单而广泛使用的算法。它试图找到一条最适合数据的直线，可以用以下等式表示：

$$
y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n
$$

其中：

*   $y$ 是因变量
*   $x_1, x_2, ..., x_n$ 是自变量
*   $\beta_0$ 是截距
*   $\beta_1, \beta_2, ..., \beta_n$ 是系数

线性回归模型的系数使用最小二乘法估计，该方法旨在最小化预测值与实际值之间的平方差之和。

### 4.2 逻辑回归

逻辑回归是一种用于分类问题的统计模型。它用于估计事件发生的概率，基于一个或多个自变量。逻辑回归使用 sigmoid 函数将线性回归等式的输出转换为介于 0 和 1 之间的概率。sigmoid 函数的等式为：

$$
p = \frac{1}{1 + e^{-( \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n)}}
$$

其中：

*   $p$ 是事件发生的概率
*   $x_1, x_2, ..., x_n$ 是自变量
*   $\beta_0$ 是截距
*   $\beta_1, \beta_2, ..., \beta_n$ 是系数

逻辑回归模型的系数使用最大似然估计来估计，该方法旨在找到使观察数据的可能性最大化的系数。

### 5.1 使用 Scikit-learn 进行线性回归

以下是如何使用 Python 中的 Scikit-learn 库执行线性回归的示例：

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 生成示例数据
x = np.array([1, 2, 3, 4, 5]).reshape((-1, 1))
y = np.array([2, 4, 5, 4, 5])

# 创建线性回归模型
model = LinearRegression()

# 拟合模型
model.fit(x, y)

# 做出预测
y_pred = model.predict(x)

# 打印系数
print('Coefficients:', model.coef_)
print('Intercept:', model.intercept_)
```

此代码片段首先生成一些示例数据，然后创建一个线性回归模型并使用数据对其进行拟合。然后，该模型用于对输入数据进行预测，并打印模型的系数和截距。
