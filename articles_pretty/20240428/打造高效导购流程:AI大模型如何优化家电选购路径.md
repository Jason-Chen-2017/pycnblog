### 4.1 余弦相似度

余弦相似度用于衡量两个向量之间的相似程度，可以用于计算用户需求和产品特征之间的相似度。公式如下：

$$
similarity(u, v) = \frac{u \cdot v}{||u|| \cdot ||v||}
$$

其中，$u$ 和 $v$ 分别表示用户需求向量和产品特征向量，$||u||$ 和 $||v||$ 分别表示向量的模长。

### 4.2 协同过滤

协同过滤是一种常用的推荐算法，它基于用户的历史行为和偏好，推荐与用户兴趣相似的产品。协同过滤算法可以分为以下两种：

*   **基于用户的协同过滤:** 找到与目标用户兴趣相似的用户，并将这些用户喜欢的产品推荐给目标用户。
*   **基于物品的协同过滤:** 找到与目标用户喜欢的产品相似的产品，并将这些产品推荐给目标用户。

```python
import numpy as np

def cosine_similarity(u, v):
    """
    计算两个向量之间的余弦相似度。
    """
    return np.dot(u, v) / (np.linalg.norm(u) * np.linalg.norm(v))

def recommend_products(user_needs, product_features):
    """
    根据用户需求和产品特征，推荐最相似的产品。
    """
    similarities = []
    for features in product_features:
        similarity = cosine_similarity(user_needs, features)
        similarities.append(similarity)
    
    # 返回相似度最高的三个产品
    top_indices = np.argsort(similarities)[-3:]
    return top_indices
```
