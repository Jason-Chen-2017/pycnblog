## 2. 核心概念与联系

### 2.1 神经网络

深度学习的核心是人工神经网络，它模拟了人脑神经元的结构和功能。神经网络由多个层级的神经元组成，每个神经元接收来自上一层神经元的输入，并通过激活函数进行非线性变换，最终输出到下一层神经元。

### 2.2 深度学习

深度学习是指使用多层神经网络进行机器学习的方法。相比于传统的浅层神经网络，深度学习模型具有更强的表达能力和学习能力，能够从数据中自动提取特征，并进行更复杂的模式识别和决策。

## 3. 核心算法原理

### 3.1 前向传播

前向传播是指输入数据从输入层逐层传递到输出层的过程。在每一层，神经元接收来自上一层的输入，并通过激活函数进行非线性变换，最终输出到下一层。

### 3.2 反向传播

反向传播是指根据模型的输出误差，逐层调整神经网络参数的过程。通过反向传播算法，可以最小化模型的损失函数，从而提升模型的性能。

### 3.3 梯度下降

梯度下降是反向传播算法中常用的优化方法，它通过计算损失函数的梯度，并沿着梯度的反方向调整参数，从而逐步降低损失函数的值。

## 4. 数学模型和公式

### 4.1 激活函数

激活函数是神经网络中非线性变换的关键，常用的激活函数包括Sigmoid函数、Tanh函数和ReLU函数等。

*   **Sigmoid函数**: $$f(x) = \frac{1}{1 + e^{-x}}$$
*   **Tanh函数**: $$f(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$
*   **ReLU函数**: $$f(x) = max(0, x)$$

### 4.2 损失函数

损失函数用于衡量模型的预测值与真实值之间的差距，常用的损失函数包括均方误差和交叉熵等。

*   **均方误差**: $$MSE = \frac{1}{n} \sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$
*   **交叉熵**: $$CE = -\frac{1}{n} \sum_{i=1}^{n}[y_i log(\hat{y}_i) + (1-y_i)log(1-\hat{y}_i)]$$
