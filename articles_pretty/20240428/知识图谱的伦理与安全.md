## 5. 项目实践

### 5.1 代码实例

以下是一个简单的 Python 代码示例，演示如何使用 NetworkX 库构建一个知识图谱：

```python
import networkx as nx

# 创建一个图
G = nx.Graph()

# 添加节点
G.add_node("张三")
G.add_node("北京")
G.add_node("谷歌")

# 添加边
G.add_edge("张三", "北京", relation="居住地")
G.add_edge("张三", "谷歌", relation="工作单位")

# 打印图的信息
print(G.nodes)
print(G.edges)
```

### 5.2 详细解释

* `networkx` 是一个用于创建、操作和研究复杂网络的 Python 库。
* `Graph()` 创建一个空的图对象。
* `add_node()` 向图中添加节点。
* `add_edge()` 向图中添加边，并指定边的关系。
* `nodes` 属性返回图中的所有节点。
* `edges` 属性返回图中的所有边。
