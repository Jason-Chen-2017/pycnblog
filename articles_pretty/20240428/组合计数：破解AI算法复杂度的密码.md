**问题:** 计算从 n 个不同元素中取出 k 个元素的组合数。

**Python 代码:**

```python
def combinations(n, k):
  """
  计算组合数 C(n, k).
  """
  if k == 0 or k == n:
    return 1
  else: 
    return combinations(n-1, k-1) + combinations(n-1, k)
```

**解释:**

该代码使用了递归方法计算组合数。当 k 为 0 或 k 等于 n 时，组合数为 1。否则，根据组合数的递推公式，将问题分解为两个子问题：从 n-1 个元素中取出 k-1 个元素，以及从 n-1 个元素中取出 k 个元素。
