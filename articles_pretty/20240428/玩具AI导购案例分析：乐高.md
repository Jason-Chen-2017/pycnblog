### 4.1 协同过滤算法

协同过滤算法的核心思想是“物以类聚，人以群分”。通过计算用户之间的相似度，可以找到与目标用户兴趣相似的其他用户，并推荐他们购买过的商品。

*   **余弦相似度:**  余弦相似度是常用的用户相似度计算方法之一，其公式如下：

$$
sim(u,v) = \frac{\sum_{i=1}^n r_{ui} \cdot r_{vi}}{\sqrt{\sum_{i=1}^n r_{ui}^2} \cdot \sqrt{\sum_{i=1}^n r_{vi}^2}}
$$

其中， $u$ 和 $v$ 表示两个用户， $r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分。

以下是一个基于Python的简单协同过滤算法示例：

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载用户评分数据
ratings = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
similarity_matrix = cosine_similarity(ratings)

# 找到与目标用户最相似的用户
target_user = 1
similar_users = similarity_matrix[target_user].argsort()[::-1][1:]

# 推荐相似用户购买过的商品
recommendations = []
for user in similar_users:
    user_ratings = ratings[ratings['userId'] == user]
    recommendations.extend(user_ratings['itemId'].tolist())

# 去重并排序推荐结果
recommendations = list(set(recommendations))
recommendations.sort(key=lambda x: recommendations.count(x), reverse=True)

# 输出推荐结果
print(recommendations)
```
