### 4.1 协同过滤算法

协同过滤算法的核心思想是找到与目标用户相似的用户，并推荐相似用户喜欢的商品。

*   **基于用户的协同过滤 (User-based CF):** 计算用户之间的相似度，例如使用余弦相似度或皮尔逊相关系数。
    $$
    sim(u,v) = \frac{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)(r_{vi} - \bar{r}_v)}{\sqrt{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)^2}\sqrt{\sum_{i \in I_{uv}}(r_{vi} - \bar{r}_v)^2}}
    $$
*   **基于物品的协同过滤 (Item-based CF):** 计算物品之间的相似度，例如使用余弦相似度或调整余弦相似度。
    $$
    sim(i,j) = \frac{\sum_{u \in U_{ij}}(r_{ui} - \bar{r}_u)(r_{uj} - \bar{r}_u)}{\sqrt{\sum_{u \in U_{ij}}(r_{ui} - \bar{r}_u)^2}\sqrt{\sum_{u \in U_{ij}}(r_{uj} - \bar{r}_u)^2}}
    $$

### 4.2 深度学习推荐模型

深度学习推荐模型使用深度神经网络学习用户和商品的特征，进行更精准的推荐。

*   **矩阵分解 (MF):** 将用户-商品评分矩阵分解为用户特征矩阵和商品特征矩阵。
*   **深度神经网络 (DNN):** 使用多层神经网络学习用户和商品的非线性特征。

## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的基于用户的协同过滤算法的 Python 代码示例：

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载用户-商品评分数据
ratings = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(ratings.pivot_table(index='userId', columns='itemId', values='rating').fillna(0))

# 预测用户对未评分商品的评分
def predict_rating(user_id, item_id):
    # 找到与目标用户相似的用户
    similar_users = user_similarity[user_id].argsort()[::-1][1:]
    # 计算相似用户的平均评分
    similar_user_ratings = ratings[(ratings['userId'].isin(similar_users)) & (ratings['itemId'] == item_id)]['rating']
    # 返回预测评分
    return similar_user_ratings.mean()

# 预测用户 1 对商品 10 的评分
predicted_rating = predict_rating(1, 10)
print(predicted_rating)
```
