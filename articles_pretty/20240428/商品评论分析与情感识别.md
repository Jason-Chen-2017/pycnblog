## 4. 数学模型和公式详细讲解举例说明

### 4.1 朴素贝叶斯算法

朴素贝叶斯算法是一种基于贝叶斯定理的简单概率分类算法，假设各个特征之间相互独立。其数学模型如下：

$$P(c|x) = \frac{P(x|c)P(c)}{P(x)}$$

其中，$P(c|x)$ 表示给定特征 $x$ 的条件下，样本属于类别 $c$ 的概率；$P(x|c)$ 表示类别 $c$ 中出现特征 $x$ 的概率；$P(c)$ 表示类别 $c$ 的先验概率；$P(x)$ 表示特征 $x$ 的先验概率。

### 4.2 支持向量机(SVM)

支持向量机是一种二分类模型，旨在找到一个超平面，将不同类别的数据点 максимально возможно отделить друг от друга. 其数学模型如下：

$$
\begin{aligned}
&\max_{\alpha} \sum_{i=1}^{n} \alpha_i - \frac{1}{2} \sum_{i=1}^{n} \sum_{j=1}^{n} \alpha_i \alpha_j y_i y_j K(x_i, x_j) \\
&\text{subject to } 0 \leq \alpha_i \leq C, \sum_{i=1}^{n} \alpha_i y_i = 0
\end{aligned}
$$

其中，$\alpha_i$ 是拉格朗日乘子，$y_i$ 是样本标签，$K(x_i, x_j)$ 是核函数，$C$ 是惩罚参数。

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 和 scikit-learn 库进行商品评论情感分析的示例代码：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import Pipeline

# 训练数据
train_data = [
    ("这款手机太棒了，我很喜欢！", "positive"),
    ("这个产品的质量很差，我非常失望。", "negative"),
    ("这个软件的功能很强大，使用起来很方便。", "positive"),
]

# 将训练数据拆分为文本和标签
train_texts, train_labels = zip(*train_data)

# 创建 TF-IDF 特征提取器
vectorizer = TfidfVectorizer()

# 创建朴素贝叶斯分类器
classifier = MultinomialNB()

# 创建管道
pipeline = Pipeline([
    ("vectorizer", vectorizer),
    ("classifier", classifier),
])

# 训练模型
pipeline.fit(train_texts, train_labels)

# 测试数据
test_text = "这个产品的性能很稳定，我很满意。"

# 预测情感
predicted_label = pipeline.predict([test_text])[0]

# 打印预测结果
print(f"预测的情感：{predicted_label}")
```
