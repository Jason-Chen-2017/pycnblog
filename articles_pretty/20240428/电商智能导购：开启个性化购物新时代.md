## 4. 数学模型和公式详细讲解举例说明

### 4.1 用户相似度计算

- **余弦相似度**: 

$$
sim(u,v) = \frac{\vec{u} \cdot \vec{v}}{||\vec{u}|| \cdot ||\vec{v}||}
$$

其中，$\vec{u}$ 和 $\vec{v}$ 分别表示用户 $u$ 和 $v$ 的向量表示，例如购买商品的向量。

### 4.2 商品相似度计算

- **Jaccard相似度**:

$$
sim(i,j) = \frac{|N(i) \cap N(j)|}{|N(i) \cup N(j)|} 
$$

其中，$N(i)$ 和 $N(j)$ 分别表示购买商品 $i$ 和 $j$ 的用户集合。

### 5.1 Python代码示例：协同过滤推荐

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 读取用户行为数据
data = pd.read_csv('user_behavior.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(data)

# 获取当前用户
current_user = 1

# 获取与当前用户最相似的用户
similar_users = user_similarity[current_user].argsort()[::-1][1:]

# 获取相似用户购买过的商品
recommended_items = set()
for user in similar_users:
    recommended_items.update(data.loc[data['user_id'] == user, 'item_id'])

# 推荐商品
print(recommended_items)
```
