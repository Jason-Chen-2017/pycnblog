#### 4.1 协同过滤算法

协同过滤算法的数学模型可以表示为：

$$
r_{ui} = \sum_{v \in N(u)} w_{uv} r_{vi}
$$

其中，$r_{ui}$ 表示用户 $u$ 对物品 $i$ 的评分，$N(u)$ 表示与用户 $u$ 相似的用户集合，$w_{uv}$ 表示用户 $u$ 和用户 $v$ 的相似度，$r_{vi}$ 表示用户 $v$ 对物品 $i$ 的评分。

#### 4.2 基于内容的推荐算法

基于内容的推荐算法的数学模型可以表示为：

$$
sim(i,j) = \frac{\vec{v_i} \cdot \vec{v_j}}{||\vec{v_i}|| \cdot ||\vec{v_j}||}

其中，$sim(i,j)$ 表示物品 $i$ 和物品 $j$ 的相似度，$\vec{v_i}$ 和 $\vec{v_j}$ 分别表示物品 $i$ 和物品 $j$ 的特征向量。

#### 5.1 Python代码示例：协同过滤算法

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(data.pivot_table(index='userId', columns='movieId', values='rating'))

# 预测用户u对物品i的评分
def predict_rating(u, i):
    # 找到与用户u相似的用户
    similar_users = user_similarity[u].argsort()[::-1]
    # 计算预测评分
    predicted_rating = 0
    total_weight = 0
    for v in similar_users:
        if data.loc[(data['userId'] == v) & (data['movieId'] == i), 'rating'].any():
            rating = data.loc[(data['userId'] == v) & (data['movieId'] == i), 'rating'].iloc[0]
            weight = user_similarity[u, v]
            predicted_rating += rating * weight
            total_weight += weight
    if total_weight > 0:
        return predicted_rating / total_weight
    else:
        return 0
```

该代码首先加载用户评分数据，然后计算用户相似度矩阵。最后，定义了一个函数`predict_rating`，用于预测用户对物品的评分。该函数首先找到与目标用户相似的用户，然后根据相似用户的评分和相似度计算预测评分。
