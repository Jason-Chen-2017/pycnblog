### 4.1 Accumulator的数学表示
Accumulator可以用一个三元组 $(v, z, f)$ 来表示，其中：
- $v$ 表示 Accumulator 的当前值；
- $z$ 表示 Accumulator 的初始值；
- $f$ 表示 Accumulator 的聚合函数，用于将新的输入值与当前值进行组合。

例如，对于一个用于计算总和的 Accumulator，可以将其表示为：

$$(\sum_{i=1}^{n} x_i, 0, +)$$

其中，$\sum_{i=1}^{n} x_i$ 表示当前的累加和，$0$ 表示初始值，$+$ 表示聚合函数为加法操作。

### 4.2 聚合函数的数学模型
不同类型的聚合函数具有不同的数学模型，下面以求和、平均值、最大值为例进行说明。

#### 4.2.1 求和函数的数学模型
求和函数用于计算一组数据的总和，其数学模型可以表示为：

$$sum(x_1, x_2, ..., x_n) = \sum_{i=1}^{n} x_i$$

其中，$x_1, x_2, ..., x_n$ 表示输入的数据集合，$\sum$ 表示求和操作。

#### 4.2.2 平均值函数的数学模型
平均值函数用于计算一组数据的算术平均值，其数学模型可以表示为：

$$avg(x_1, x_2, ..., x_n) = \frac{1}{n} \sum_{i=1}^{n} x_i$$

其中，$x_1, x_2, ..., x_n$ 表示输入的数据集合，$\sum$ 表示求和操作，$\frac{1}{n}$ 表示对求和结果进行归一化。

#### 4.2.3 最大值函数的数学模型
最大值函数用于查找一组数据中的最大值，其数学模型可以表示为：

$$max(x_1, x_2, ..., x_n) = \max_{i=1}^{n} x_i$$

其中，$x_1, x_2, ..., x_n$ 表示输入的数据集合，$\max$ 表示取最大值操作。
