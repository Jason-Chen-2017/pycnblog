
## **1. 背景介绍**

**[1.1 数据爆炸时代的到来](./background_1.md)**

随着互联网普及和技术发展，社交媒体平台如Twitter、Facebook、Instagram等积累了海量用户生成的数据。这些数据不仅是个人生活痕迹的记录，更是社会动态的反映，对于科学研究、商业分析以及政策制定具有重要价值。

**[1.2 数据集的重要性与特点](./dataset_importance_1.md)**

- 大规模性：每个用户的日常互动形成复杂的关系网，数据量级可达PB级别。
- 高维度多样性：包括文字、图片、视频等多种类型，且语义丰富多变。
- 实时更新特性：每秒都有新信息产生，需要实时处理能力。

**[3.2 滚动视窗计算](./rolling_window_3.md)**
选取一个固定大小的时间窗口，滑动分析期间内的用户行为特征，适用于发现短期波动和局部相关性。

```python
def windowed_stats(data, window_size):
    stats_list = []
    for i in range(len(data) - window_size + 1):
        stats_list.append(np.mean(data[i:i+window_size]))
    return stats_list
```

$$\text{PCA: } X = WZ \Rightarrow Z_i = \sum_j w_{ij}X_j,$$
其中\(W\)是旋转矩阵，\(Z\)为主成分得分。


## **4. 数学模型和公式详解举例说明**

**[4.1 网络聚类效果评估](./clustering_effectiveness_4.md)**
常用Silhouette系数衡量簇内部紧密程度与外部距离，优化K-means或其他聚类算法的效果：

\[ S(i) = b(i) - a(i), \]
\[ a(i) = \frac{1}{n_b}\sum_{j\in C_b(i)}d(x_i,x_j), \]
\[ b(i) = \min_{C_k\neq C_b(i)}\left(\frac{1}{|C_k|}\sum_{j\in C_k}d(x_i,x_j)\right), \]

其中\(i\)代表第\(i\)个样本，\(C_b(i)\)为其所属簇，\(C_k\)其他所有簇。

**[4.2 强连通分量（Strongly Connected Components, SCCs）](./strongly_connected_components_4.md)**
用深度优先搜索或广度优先搜索求解图中强连通组件，帮助理解社交网络的循环结构：

\[ G=(V,E) \rightarrow DFS(G) \rightarrow stronglyConnectedComponents(V,E) \]

**[4.3 自编码器应用于异常检测](./autoencoder_anomaly_detection_4.md)**
通过比较重建误差与正常数据分布，自编码器可以识别潜在的异常点：

\[ R(X)=f(W,X)-X,\quad where\; f(W,X)\sim p_\theta(X)\]
