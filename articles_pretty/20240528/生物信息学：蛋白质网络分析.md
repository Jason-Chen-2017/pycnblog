### 4.2 中心性度量的数学定义与计算
#### 4.2.1 度中心性的定义与计算公式
$$C_D(v)=\frac{d(v)}{n-1}$$
其中，$d(v)$表示节点$v$的度，$n$为网络中节点的总数。
#### 4.2.2 紧密中心性的定义与计算公式
$$C_C(v)=\frac{n-1}{\sum_{u\neq v}d(u,v)}$$
其中，$d(u,v)$表示节点$u$和$v$之间的最短路径长度。
#### 4.2.3 中介中心性的定义与计算公式
$$C_B(v)=\sum_{s\neq v\neq t}\frac{\sigma_{st}(v)}{\sigma_{st}}$$
其中，$\sigma_{st}$表示节点$s$和$t$之间的最短路径数，$\sigma_{st}(v)$表示经过节点$v$的$s$和$t$之间的最短路径数。

### 4.3 模块化与社团检测的数学模型
#### 4.3.1 模块度的定义与计算公式
$$Q=\frac{1}{2m}\sum_{i,j}(A_{ij}-\frac{k_ik_j}{2m})\delta(c_i,c_j)$$
其中，$A_{ij}$表示节点$i$和$j$之间的边的权重，$k_i$表示节点$i$的度，$m$表示网络中边的总权重，$\delta(c_i,c_j)$表示当节点$i$和$j$属于同一社团时取1，否则取0。

## 5. 项目实践：代码实例与详细解释说明

### 5.1 使用Python构建PPI网络
```python
import networkx as nx

# 从文件中读取PPI数据
with open('ppi_data.txt', 'r') as f:
    edges = [line.strip().split() for line in f]

# 创建无向图
G = nx.Graph()

# 添加边
G.add_edges_from(edges)

# 可视化网络
nx.draw(G, with_labels=True, node_size=500, node_color='lightblue', font_size=12)
```

### 5.2 使用R计算网络中心性度量
```r
library(igraph)

# 从文件中读取PPI网络
ppi_net <- read.graph("ppi_network.gml", format = "gml")

# 计算度中心性
degree_centrality <- degree(ppi_net, mode = "all", normalized = TRUE)

# 计算紧密中心性
closeness_centrality <- closeness(ppi_net, mode = "all", normalized = TRUE)

# 计算中介中心性
betweenness_centrality <- betweenness(ppi_net, normalized = TRUE)

# 输出结果
print(degree_centrality)
print(closeness_centrality)  
print(betweenness_centrality)
```
