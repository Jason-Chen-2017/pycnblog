### 4.1. 余弦相似度

余弦相似度用于衡量两个向量之间的相似程度，其取值范围为 -1 到 1，其中 1 表示完全相似，-1 表示完全不相似。在食品推荐系统中，我们可以将用户的评分向量或食品的特征向量作为输入，使用余弦相似度公式计算它们之间的相似度：

$$
\cos(\theta) = \frac{A \cdot B}{\|A\|\|B\|}
$$

其中，A 和 B 分别表示两个向量，$\theta$ 表示它们之间的夹角，$\cdot$ 表示向量点积，$\|A\|$ 和 $\|B\|$ 分别表示 A 和 B 的模长。

欧几里得距离用于衡量两个点之间的距离，在食品推荐系统中，我们可以将用户的评分向量或食品的特征向量作为输入，使用欧几里得距离公式计算它们之间的距离：

$$

其中，A 和 B 分别表示两个向量，$n$ 表示向量的维度，$A_i$ 和 $B_i$ 分别表示 A 和 B 在第 $i$ 维上的值。

以下是一个使用 Python 和 scikit-learn 库实现协同过滤算法的示例代码：

```python
from sklearn.metrics.pairwise import cosine_similarity

# 用户评分数据
ratings = [[5, 4, 3], [2, 1, 5], [4, 3, 4]]

# 计算用户之间的相似度矩阵
similarity_matrix = cosine_similarity(ratings)

# 找到与目标用户（例如用户 0）最相似的用户
similar_users = similarity_matrix[0]

# 推荐相似用户喜欢的食品
recommendations = []
for i, similarity in enumerate(similar_users):
    if i != 0 and similarity > 0.5:
        recommendations.extend(ratings[i])

print(recommendations)
```
