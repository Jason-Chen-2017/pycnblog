# -图像特征提取：SIFT、HOG

## 1.背景介绍

### 1.1 什么是图像特征提取？

在计算机视觉和图像处理领域中，图像特征提取是一个非常重要的步骤。它旨在从图像中提取出具有代表性和区分性的特征信息,这些特征信息可用于后续的图像分析、识别和匹配等任务。图像特征提取的目标是将图像中的关键信息转换为易于计算和处理的数值形式,同时尽可能保留图像的重要特性。

图像特征可以分为多种类型,例如边缘、角点、斑点、纹理等。不同类型的特征适用于不同的应用场景,需要根据具体任务选择合适的特征提取算法。良好的图像特征应该具备以下几个特性:

1. **独特性(Distinctiveness)**:特征应该具有足够的区分能力,能够有效地区分不同的图像对象。
2. **不变性(Invariance)**:特征应该对图像的几何变换(如旋转、缩放、平移等)和光照变化具有一定的鲁棒性。
3. **高效性(Efficiency)**:特征提取算法应该具有较高的计算效率,以满足实时性要求。
4. **局部性(Locality)**:特征应该具有良好的局部性,能够描述图像的局部结构信息。
