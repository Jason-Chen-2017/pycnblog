### 5.1 知识抽取代码示例 (Python)

```python
import spacy

nlp = spacy.load("en_core_web_sm")

text = "Barack Obama was born in Honolulu."

doc = nlp(text)

for ent in doc.ents:
    print(ent.text, ent.label_)
```

**输出:**

```
Barack Obama PERSON
Honolulu GPE
```

**解释:**

这段代码使用 spaCy 库进行命名实体识别，识别出文本中的两个命名实体：Barack Obama (人物) 和 Honolulu (地名)。

### 5.2 知识表示代码示例 (Python)

from rdflib import Graph, Literal, URIRef

g = Graph()

```

```

**解释:**

这段代码使用 RDFLib 库创建一个 RDF 图，并将一个三元组添加到图中，表示 Bob 的名字是 "Bob"。
