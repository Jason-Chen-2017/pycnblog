### 1.2 相似矩阵与合同矩阵

在矩阵的研究中，相似矩阵和合同矩阵是两个重要的概念，它们描述了矩阵之间某种特殊的等价关系。相似矩阵表示同一个线性变换在不同基下的矩阵表示，而合同矩阵则表示同一个二次型在不同基下的矩阵表示。

## 2. 核心概念与联系

### 2.1 相似矩阵

**定义：** 设 $A$ 和 $B$ 是 $n$ 阶方阵，如果存在可逆矩阵 $P$，使得 $P^{-1}AP = B$，则称 $A$ 和 $B$ 相似，记作 $A \sim B$。

**几何意义：** 相似矩阵表示同一个线性变换在不同基下的矩阵表示。$P$ 可以看作是基变换矩阵，它将 $A$ 所在的基变换到 $B$ 所在的基。

**性质：**

* 相似矩阵具有相同的特征值。
* 相似矩阵具有相同的行列式。
* 相似矩阵具有相同的秩。
* 相似矩阵具有相同的迹。

### 2.2 合同矩阵

**定义：** 设 $A$ 和 $B$ 是 $n$ 阶方阵，如果存在可逆矩阵 $C$，使得 $C^TAC = B$，则称 $A$ 和 $B$ 合同，记作 $A \cong B$。

**几何意义：** 合同矩阵表示同一个二次型在不同基下的矩阵表示。$C$ 可以看作是基变换矩阵，它将 $A$ 所在的基变换到 $B$ 所在的基。

**性质：**

* 合同矩阵具有相同的正负惯性指数。
* 合同矩阵具有相同的秩。

### 4.1 相似矩阵的特征值

设 $A$ 和 $B$ 是相似矩阵，即 $P^{-1}AP = B$，$\lambda$ 是 $A$ 的特征值，则：

$$
\begin{aligned}
AP &= \lambda P \\
P^{-1}AP &= \lambda P^{-1}P \\
B &= \lambda I
\end{aligned}
$$

即 $\lambda$ 也是 $B$ 的特征值。

### 4.2 合同矩阵的正负惯性指数

设 $A$ 是一个实对称矩阵，则存在可逆矩阵 $C$，使得 $C^TAC = D$，其中 $D$ 是一个对角矩阵，其对角线元素为 $A$ 的特征值。$D$ 中正元素的个数称为 $A$ 的正惯性指数，负元素的个数称为 $A$ 的负惯性指数。

```python
import numpy as np

def is_similar(A, B):
  """
  判断两个矩阵是否相似。
  """
  # 计算特征值
  eigenvalues_A = np.linalg.eigvals(A)
  eigenvalues_B = np.linalg.eigvals(B)

  # 比较特征值
  return np.allclose(eigenvalues_A, eigenvalues_B)
```
