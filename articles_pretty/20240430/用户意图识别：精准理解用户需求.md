## 4. 数学模型和公式

### 4.1 朴素贝叶斯

朴素贝叶斯是一种基于贝叶斯定理的分类算法，其基本思想是通过计算文本属于不同意图的概率来进行分类。

$$P(c|d) = \frac{P(d|c)P(c)}{P(d)}$$

其中，$P(c|d)$ 表示文本 $d$ 属于类别 $c$ 的概率，$P(d|c)$ 表示类别 $c$ 中出现文本 $d$ 的概率，$P(c)$ 表示类别 $c$ 的先验概率，$P(d)$ 表示文本 $d$ 的先验概率。

### 4.2 支持向量机

支持向量机是一种二分类模型，其基本思想是找到一个超平面，将不同类别的样本点分开。

$$f(x) = w^Tx + b$$

其中，$w$ 为权重向量，$b$ 为偏置项，$x$ 为样本点。

## 5. 项目实践：代码实例

### 5.1 基于 Python 的用户意图识别示例

```python
# 导入必要的库
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC
from sklearn.pipeline import Pipeline

# 定义训练数据
train_data = [
    ("我想订一张去北京的机票", "订票"),
    ("北京今天的天气怎么样", "天气"),
    ("播放音乐", "音乐"),
]

# 定义特征提取器和分类器
vectorizer = TfidfVectorizer()
classifier = SVC()

# 创建管道
pipeline = Pipeline([
    ("vectorizer", vectorizer),
    ("classifier", classifier),
])

# 训练模型
pipeline.fit([x[0] for x in train_data], [x[1] for x in train_data])

# 进行意图识别
text = "我想去上海旅游"
intent = pipeline.predict([text])[0]

print(f"用户意图: {intent}")
```
