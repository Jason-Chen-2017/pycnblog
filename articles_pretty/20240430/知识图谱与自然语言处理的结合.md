**5.2 实体链接**

以下是一个使用 Python 进行实体链接的示例代码：

```python
from fuzzywuzzy import fuzz

# 定义实体列表
entities = ["苹果公司", "苹果手机", "苹果电脑"]

# 定义文本
text = "我喜欢使用苹果手机"

# 找到最相似的实体
max_similarity = 0
best_match = None
for entity in entities:
    similarity = fuzz.ratio(text, entity)
    if similarity > max_similarity:
        max_similarity = similarity
        best_match = entity

# 输出结果
print("最佳匹配实体:", best_match)
```
