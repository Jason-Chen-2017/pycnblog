4.3 自适应培训推荐引擎
基于个性化学习模型和绩效预测结果,开发了一个基于协同过滤的培训课程推荐引擎,实时为员工推荐最合适的培训内容。

```python
import numpy as np
from scipy.spatial.distance import cosine

def recommend_courses(user_id, course_data, n=5):
    user_profile = course_data[user_id]
    course_similarities = [(course_id, 1-cosine(course_data[course_id], user_profile)) for course_id in course_data]
    course_similarities.sort(key=lambda x: x[1], reverse=True)
    return [course_id for course_id, _ in course_similarities[:n]]
```
