### 4.1 Logistic回归模型

$$
P(Y=1|X) = \frac{1}{1+e^{-(\beta_0 + \beta_1 X_1 + ... + \beta_n X_n)}}
$$

其中，$Y$表示是否购买特定品牌，$X_1, ..., X_n$表示影响购买的因素，如年龄、专业、价格等，$\beta_0, ..., \beta_n$为模型参数。

### 5.1 Python代码示例

```python
import pandas as pd
from sklearn.linear_model import LogisticRegression

# 加载数据
data = pd.read_csv('student_data.csv')

# 划分特征和标签
X = data[['age', 'major', 'income']]
y = data['brand']

# 构建模型

# 预测新数据的购买概率
new_data = [[22, 'finance', 4000]]
probability = model.predict_proba(new_data)[0][1]
print(probability)
```
