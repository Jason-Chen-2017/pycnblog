## 4. 数学模型和公式详细讲解举例说明

协同过滤算法中常用的相似度计算公式如下：

**余弦相似度：**

$$
\cos(\theta) = \frac{\sum_{i=1}^{n}(a_i \times b_i)}{\sqrt{\sum_{i=1}^{n}a_i^2} \times \sqrt{\sum_{i=1}^{n}b_i^2}}
$$

其中，$a_i$ 和 $b_i$ 分别表示用户 $a$ 和用户 $b$ 对商品 $i$ 的评分。

**Jaccard相似度：**

$$
J(A,B) = \frac{|A \cap B|}{|A \cup B|}
$$

其中，$A$ 和 $B$ 分别表示用户 $a$ 和用户 $b$ 购买的商品集合。

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载用户评分数据
ratings = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(ratings)

# 获取目标用户的 ID
target_user_id = 1

# 寻找与目标用户最相似的用户
similar_users = user_similarity[target_user_id].argsort()[::-1][1:]

# 获取相似用户喜欢的产品
recommendations = ratings.loc[similar_users].mean(axis=0).sort_values(ascending=False)

# 输出推荐结果
print(recommendations.head(10))
```
