## 4. 数学模型和公式详细讲解举例说明

### 4.1 余弦相似度

余弦相似度用于计算两个向量之间的夹角余弦值，夹角越小，相似度越高。公式如下：

$$
sim(u, v) = \frac{u \cdot v}{||u|| ||v||}
$$

其中，$u$ 和 $v$ 分别表示用户向量和产品向量，$||u||$ 和 $||v||$ 分别表示向量的模长。

### 4.2 Jaccard 相似度

Jaccard 相似度用于计算两个集合之间的交集和并集的比例，比例越高，相似度越高。公式如下：

$$
sim(A, B) = \frac{|A \cap B|}{|A \cup B|}
$$

其中，$A$ 和 $B$ 分别表示用户喜欢的产品集合和目标产品的属性集合。

import numpy as np

def cosine_similarity(u, v):
    """
    计算两个向量的余弦相似度
    """
    dot_product = np.dot(u, v)
    norm_u = np.linalg.norm(u)
    norm_v = np.linalg.norm(v)
    return dot_product / (norm_u * norm_v)
