以下是一个简单的Python代码示例，演示如何使用线性回归模型预测用户的血压：

```python

from sklearn.linear_model import LinearRegression

# 加载数据
data = pd.read_csv('health_data.csv')

# 划分特征和标签
X = data[['age', 'weight', 'exercise']]
y = data['blood_pressure']

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 预测血压
new_data = [[30, 60, 1]]
predicted_blood_pressure = model.predict(new_data)

# 打印预测结果
print(predicted_blood_pressure)
```
