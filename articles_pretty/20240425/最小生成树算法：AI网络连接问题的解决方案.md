## 3. 核心算法原理具体操作步骤

### 3.1 Prim算法

Prim算法是一种贪心算法，用于寻找最小生成树。算法的具体步骤如下：

1.  从任意一个节点开始，将其加入到生成树中。
2.  找到与生成树相连的边中权重最小的边，将其加入到生成树中。
3.  重复步骤2，直到所有节点都加入到生成树中。

### 3.2 Kruskal算法

Kruskal算法也是一种贪心算法，用于寻找最小生成树。算法的具体步骤如下：

1.  将所有边按照权重从小到大排序。
2.  依次遍历每条边，如果该边连接的两个节点不在同一个连通分量中，则将该边加入到生成树中。
3.  重复步骤2，直到所有节点都加入到生成树中。

```python
def prim(graph):
    # 初始化生成树
    mst = set()
    # 初始化节点集合
    nodes = set(graph.keys())
    # 选择任意一个节点作为起始节点
    start_node = next(iter(nodes))
    # 将起始节点加入到生成树中
    mst.add(start_node)
    # 移除起始节点
    nodes.remove(start_node)
    # 循环直到所有节点都加入到生成树中
    while nodes:
        # 找到与生成树相连的边中权重最小的边
        min_edge = None
        for node in mst:
            for neighbor, weight in graph[node].items():
                if neighbor in nodes and (min_edge is None or weight < min_edge[1]):
                    min_edge = (node, neighbor, weight)
        # 将最小边加入到生成树中
        mst.add(min_edge[1])
        # 移除最小边的目标节点
        nodes.remove(min_edge[1])
    # 返回生成树
    return mst
```
