## 3. 核心算法原理具体操作步骤

### 3.1 需求分析

首先，系统需要收集和分析顾客的健康需求，包括：

* **症状描述:** 顾客描述的症状，例如头痛、发烧等。
* **病史信息:** 顾客的既往病史和过敏史。
* **用药情况:** 顾客正在服用的药物。

### 3.2 药品推荐

根据顾客的需求，系统会进行以下步骤：

1. **症状匹配:** 将顾客描述的症状与知识图谱中的疾病进行匹配，确定可能的疾病。
2. **药品筛选:** 根据疾病和顾客的病史、用药情况等信息，筛选出合适的药品。
3. **排序推荐:** 根据药品的疗效、安全性、价格等因素，对候选药品进行排序，并推荐给顾客。

### 3.3 交互反馈

系统可以与顾客进行多轮对话，进一步了解其需求，并根据反馈调整推荐结果。

## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的Python代码示例，演示如何使用余弦相似度计算症状与疾病的相似度：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 定义症状和疾病
symptom = "头痛发烧"
diseases = ["感冒", "流感", "脑膜炎"]

# 使用TF-IDF将文本转换为向量
vectorizer = TfidfVectorizer()
vectors = vectorizer.fit_transform([symptom] + diseases)

# 计算症状与疾病的相似度
similarities = cosine_similarity(vectors[0], vectors[1:])

# 打印相似度
for disease, similarity in zip(diseases, similarities[0]):
    print(f"{disease}: {similarity:.2f}")
```
