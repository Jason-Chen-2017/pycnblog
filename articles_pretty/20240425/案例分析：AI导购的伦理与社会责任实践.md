### 4.1 协同过滤算法

协同过滤算法是一种常用的推荐算法，它基于用户的历史行为数据，寻找与目标用户相似的用户，并推荐这些相似用户喜欢的商品。

**公式：**

$$
sim(u, v) = \frac{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)(r_{vi} - \bar{r}_v)}{\sqrt{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)^2}\sqrt{\sum_{i \in I_{uv}}(r_{vi} - \bar{r}_v)^2}}
$$

其中，$sim(u, v)$ 表示用户 $u$ 和用户 $v$ 之间的相似度，$I_{uv}$ 表示用户 $u$ 和用户 $v$ 共同评价过的商品集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$\bar{r}_u$ 表示用户 $u$ 的平均评分。

### 4.2 深度学习模型

深度学习模型可以学习用户行为数据的复杂特征，并进行更精准的商品推荐。例如，可以使用循环神经网络 (RNN) 模型学习用户的浏览历史序列，并预测用户下一步可能感兴趣的商品。

```python
import tensorflow as tf

# 定义用户-商品评分矩阵
ratings = tf.constant([
    [4, 3, 2, 1],
    [5, 1, 3, 2],
    [1, 4, 5, 3],
])

# 计算用户相似度矩阵
user_similarity = tf.matmul(ratings, ratings, transpose_b=True)

# 预测用户对未评分商品的评分
predictions = tf.matmul(user_similarity, ratings)

# 打印预测结果
print(predictions)
```
