## 2. 核心概念与联系

### 2.1 机器学习的类型

* **监督学习**：算法从带有标签的训练数据中学习，例如分类和回归问题。
* **无监督学习**：算法从无标签数据中学习，例如聚类和降维问题。
* **强化学习**：算法通过与环境交互并获得奖励来学习。

### 2.2 关键术语

* **训练数据**：用于训练机器学习模型的带标签或无标签的数据集。
* **特征**：数据的可测量属性，例如图像的像素值或文本的单词。
* **标签**：训练数据中每个样本的预期输出，例如图像的类别或文本的情感。
* **模型**：从数据中学习到的模式的表示，用于进行预测或决策。
* **算法**：用于训练模型的一组规则或步骤。


## 3. 核心算法原理

### 3.1 线性回归

线性回归是一种用于建立特征和连续目标变量之间线性关系的算法。它通过找到最佳拟合线来最小化预测值和实际值之间的误差。

### 3.2 逻辑回归

逻辑回归是一种用于分类问题的算法，它将线性回归的输出映射到 0 到 1 之间的概率值，表示样本属于某个类别的可能性。

### 3.3 决策树

决策树是一种基于树形结构进行分类或回归的算法。它通过一系列if-else规则将数据分成不同的子集，直到达到最终的预测结果。

### 3.4 支持向量机 (SVM)

SVM 是一种用于分类和回归的算法，它通过在特征空间中找到一个超平面来最大化不同类别之间的间隔。

### 3.5 K-means 聚类

K-means 是一种无监督学习算法，用于将数据点分组到 K 个簇中，使得簇内数据点之间的距离最小化，而簇间距离最大化。


## 4. 数学模型和公式

### 4.1 线性回归模型

$$
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n
$$

其中，$y$ 是目标变量，$x_i$ 是特征，$\beta_i$ 是模型参数。

### 4.2 逻辑回归模型

$$
P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + ... + \beta_n x_n)}}
$$

其中，$P(y=1|x)$ 表示样本 $x$ 属于类别 1 的概率。

### 4.3 决策树模型

决策树模型使用熵或基尼系数等指标来衡量数据的不纯度，并选择最佳特征和分割点来构建树形结构。

  tf.keras.layers.Flatten(input_shape=(28, 28)),

  tf.keras.layers.Dense(10, activation='softmax')

model.evaluate(x_test, y_test)
