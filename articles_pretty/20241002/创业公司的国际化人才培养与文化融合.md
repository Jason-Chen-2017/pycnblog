- **增强全球竞争力**：文化融合有助于企业更好地理解和适应全球市场，提高全球竞争力。
- **促进创新**：文化融合有助于激发团队的创新思维，促进企业的发展。

# 实现多语言切换
def switch_language(language):
    if language in languages:
        return language
    else:
        return 'zh'

# 实现多语言翻译
def translate(text, language):
    if language == 'zh':
        return text
    elif language == 'en':
        return text + ' (English)'
    elif language == 'fr':
        return text + ' (Français)'

# 实现多语言测试

    translated_text = translate(text, language)
    print(f'Translated text: {translated_text}')

# 测试

test_translation('Hello', 'en')
test_translation('Hello', 'fr')
