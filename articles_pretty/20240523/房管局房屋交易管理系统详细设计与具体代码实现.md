from sqlalchemy import create_engine

from models import House

```python
from flask_login import current_user

@app.route('/house/<int:id>', methods=['GET'])
def house_detail(id):
    house = session.query(House).get(id)
    if current_user.role != 'admin' and current_user.id != house.owner_id:
        abort(403)
    return render_template('house_detail.html', house=house)
```
### 5.3 匹配算法代码

最后，我们来看一下匹配算法的代码。我们使用Python的NumPy库来进行数值计算。以下是一个计算匹配评分的示例代码：

```python
import numpy as np

def compute_score(buyer, seller):
    w = np.array([0.3, 0.4, 0.3])
    x = np.array([buyer.requirement, buyer.price, buyer.time])
    y = np.array([seller.offering, seller.price, seller.time])
    score = np.dot(w, np.abs(x - y))
    return score
```
