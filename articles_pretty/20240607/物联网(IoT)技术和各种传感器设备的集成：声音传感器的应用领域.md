## 5. 项目实践：代码实例和详细解释说明

以下是一个使用声音传感器进行环境监测的项目实践示例：

```python
import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BOARD)

# 设置声音传感器引脚
sound_pin = 7

GPIO.setup(sound_pin, GPIO.IN)

while True:
    # 读取声音传感器的状态
    sound_status = GPIO.input(sound_pin)
    
    if sound_status == GPIO.HIGH:
        # 声音传感器检测到声音信号
        print("Sound detected!")
    else:
        # 声音传感器未检测到声音信号
        print("No sound detected.")
    
    time.sleep(1)
```

以上代码使用树莓派连接声音传感器，实现对环境中声音信号的实时监测。当声音传感器检测到声音信号时，会输出"Sound detected!"，否则输出"No sound detected."。
