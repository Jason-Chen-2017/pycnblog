```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

class Student(Person):
    def __init__(self, name, age, student_id):
        super().__init__(name, age)
        self.student_id = student_id

class Teacher(Person):
    def __init__(self, name, age, course):
        super().__init__(name, age)
        self.course = course

class School:
    def __init__(self, name):
        self.name = name
        self.students = []
        self.teachers = []

    def add_student(self, student):
        self.students.append(student)

    def add_teacher(self, teacher):
        self.teachers.append(teacher)

school = School("Beijing University")
student1 = Student("Alice", 20, "123456")
teacher1 = Teacher("Bob", 30, "Mathematics")

school.add_student(student1)
school.add_teacher(teacher1)
```

在这个代码实例中，我们定义了一个`Person`类，用于表示人员信息。然后我们定义了两个子类`Student`和`Teacher`，分别表示学生和教师信息。最后，我们定义了一个`School`类，用于表示学校信息，并实例化了一个学校对象，并添加了一个学生和一个教师。
