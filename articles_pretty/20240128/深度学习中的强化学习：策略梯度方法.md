2. 从当前状态下选择一个动作。

4. 接收环境的反馈。
5. 更新策略参数。

$$

    return np.random.choice(A, p=np.exp(np.dot(theta, s)))

# 定义策略梯度方法

    for s in S:
        for a in A:
            a_prime = policy(s, theta)

    return grad
