代码实例：

以Selenium为例，下面是一个使用Python编写的简单测试脚本：

```python
from selenium import webdriver

# 初始化WebDriver
driver = webdriver.Chrome()

# 打开网页
driver.get("https://www.example.com")

# 找到输入框并输入内容
input_box = driver.find_element_by_name("username")
input_box.send_keys("admin")

# 找到按钮并点击
button = driver.find_element_by_name("login")
button.click()

# 关闭浏览器
driver.quit()
```

详细解释说明：

1. 首先，通过`selenium.webdriver.Chrome()`初始化WebDriver。
2. 然后，通过`driver.get("https://www.example.com")`打开网页。
3. 接下来，通过`driver.find_element_by_name("username")`找到输入框，并通过`input_box.send_keys("admin")`输入内容。
4. 之后，通过`driver.find_element_by_name("login")`找到按钮，并通过`button.click()`点击按钮。
5. 最后，通过`driver.quit()`关闭浏览器。
