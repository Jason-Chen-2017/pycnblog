# 大模型与AI辅助艺术创作：从模仿到创新

作者：禅与计算机程序设计艺术

## 1. 背景介绍

### 1.1 人工智能与艺术创作

人工智能技术正在以前所未有的速度发展，其应用范围也越来越广泛，艺术创作领域也不例外。AI 辅助艺术创作的概念逐渐从科幻小说走向现实，成为艺术家们探索新形式、新风格的有力工具。

### 1.2 大模型的崛起

近年来，随着深度学习技术的突破，大规模预训练模型（简称“大模型”）应运而生。这些模型拥有数十亿甚至数万亿的参数，能够在海量数据中学习复杂的模式和规律，并生成高质量的文本、图像、音频等内容。
