## 2.核心概念与联系

房屋租赁管理信息系统主要由三个部分组成：租户管理、房源管理和租赁合同管理。这三个部分互相关联，共同构成了系统的核心。

租户管理是指对租户的信息进行管理，包括租户的基本信息、联系方式、租赁历史等。房源管理则是对可租赁的房源进行管理，包括房源的基本信息、价格、状态等。而租赁合同管理则是对租赁合同进行管理，包括合同的创建、修改、到期提醒等。

## 4.项目实践：代码实例和详细解释说明

在Python中，我们可以使用SQLite库来创建和操作数据库。例如，我们可以使用以下代码来创建一个租户数据库：

```python
import sqlite3

# 连接到SQLite数据库
# 如果数据库不存在，那么它将被创建
conn = sqlite3.connect('tenant.db')

# 创建一个Cursor对象
c = conn.cursor()

# 创建tenant表
c.execute('''
    CREATE TABLE tenant(
        id INTEGER PRIMARY KEY,
        name TEXT,
        contact TEXT,
        history TEXT
    )
''')

# 提交事务
conn.commit()

# 关闭连接
conn.close()
```

这段代码首先导入了sqlite3库，然后连接到tenant.db数据库（如果数据库不存在，那么它将被创建），然后创建了一个Cursor对象，然后通过Cursor对象执行了SQL命令来创建tenant表，然后提交了事务，最后关闭了数据库连接。
