## 1. 背景介绍

### 1.1 机器翻译的发展历程

机器翻译，简单来说就是利用计算机自动将一种自然语言转换为另一种自然语言的技术。自上世纪50年代机器翻译概念提出以来，其发展历程大致可以分为三个阶段：

*   **规则 based machine translation (RBMT):**  早期机器翻译系统主要基于语言学家制定的规则，通过分析源语言语法结构，将其转换为目标语言的对应结构。这种方法需要大量的语言学知识和人工编写的规则，难以处理语言的歧义性和复杂性，翻译质量有限。
*   **Statistical machine translation (SMT):** 统计机器翻译利用大型平行语料库，通过统计方法学习两种语言之间的对应关系。这种方法无需人工编写规则，翻译质量比 RBMT 有所提升，但仍然受限于语料库的规模和质量。
*   **Neural machine translation (NMT):** 基于深度学习的机器翻译，利用神经网络模型自动学习语言之间的映射关系。近年来，随着深度学习技术的快速发展，NMT 取得了突破性进展，翻译质量大幅提升，逐渐成为机器翻译的主流方法。
