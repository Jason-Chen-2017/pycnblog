## 4. 数学模型和公式详细讲解举例说明

### 4.1 自回归模型的数学表示
#### 4.1.1 AR(p)模型的数学表示
$$X_t=c+\sum_{i=1}^p \varphi_i X_{t-i}+\varepsilon_t$$
其中，$X_t$为t时刻的观测值，$c$为常数项，$\varphi_i$为自回归系数，$\varepsilon_t$为白噪声。

#### 4.1.2 AR(p)模型的平稳性条件
AR(p)模型的特征方程为：
$$1-\varphi_1z-\varphi_2z^2-...-\varphi_pz^p=0$$
当且仅当上述特征方程的根都位于单位圆外时，AR(p)模型是平稳的。

### 4.2 移动平均模型的数学表示
#### 4.2.1 MA(q)模型的数学表示  
$$X_t=\mu+\varepsilon_t+\sum_{i=1}^q \theta_i \varepsilon_{t-i}$$
其中，$X_t$为t时刻的观测值，$\mu$为常数项，$\theta_i$为移动平均系数，$\varepsilon_t$为白噪声。

#### 4.2.2 MA(q)模型的可逆性条件
MA(q)模型的特征方程为：
$$1+\theta_1z+\theta_2z^2+...+\theta_qz^q=0$$  
当且仅当上述特征方程的根都位于单位圆外时，MA(q)模型是可逆的。

### 4.3 自回归移动平均模型的数学表示
#### 4.3.1 ARMA(p,q)模型的数学表示
$$X_t=c+\sum_{i=1}^p \varphi_i X_{t-i}+\varepsilon_t+\sum_{i=1}^q \theta_i \varepsilon_{t-i}$$
其中，$X_t$为t时刻的观测值，$c$为常数项，$\varphi_i$为自回归系数，$\theta_i$为移动平均系数，$\varepsilon_t$为白噪声。

#### 4.3.2 ARMA(p,q)模型的平稳性与可逆性条件
ARMA(p,q)模型的平稳性条件与AR(p)模型相同，可逆性条件与MA(q)模型相同。即：
- 平稳性条件：AR部分的特征方程的根都位于单位圆外
- 可逆性条件：MA部分的特征方程的根都位于单位圆外

## 5. 项目实践：代码实例和详细解释说明

### 5.1 数据准备
```python
import numpy as np
import pandas as pd
from statsmodels.tsa.arima.model import ARIMA

# 生成随机数据
np.random.seed(123)
data = np.random.normal(0, 1, 1000)
```

### 5.2 AR模型
```python
# 拟合AR(1)模型
model = ARIMA(data, order=(1, 0, 0))
results = model.fit()

print(results.summary())
```
输出结果包含了模型的各项参数估计值、标准误差、置信区间等信息。

### 5.3 MA模型
```python
# 拟合MA(1)模型
model = ARIMA(data, order=(0, 0, 1))  
results = model.fit()

print(results.summary())
```
输出结果包含了模型的各项参数估计值、标准误差、置信区间等信息。

### 5.4 ARMA模型
```python
# 拟合ARMA(1,1)模型
model = ARIMA(data, order=(1, 0, 1))
results = model.fit()

print(results.summary())
```
输出结果包含了模型的各项参数估计值、标准误差、置信区间等信息。

### 5.5 模型预测
```python
# 使用拟合的ARMA(1,1)模型进行预测
forecast = results.forecast(steps=10)
print(forecast)
```
输出结果为未来10个时间步的预测值。
