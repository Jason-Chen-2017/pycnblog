## 4. 数学模型和公式详细讲解举例说明

### 4.1  协同过滤算法

协同过滤算法的核心思想是“人以群分”，即兴趣相似的用户，可能会喜欢相同的书籍。

假设用户 $u$ 对书籍 $i$ 的评分为 $r_{ui}$，则用户 $u$ 和用户 $v$ 之间的相似度可以表示为：

$$
sim(u,v) = \frac{\sum_{i \in I}(r_{ui}-\bar{r_u})(r_{vi}-\bar{r_v})}{\sqrt{\sum_{i \in I}(r_{ui}-\bar{r_u})^2}\sqrt{\sum_{i \in I}(r_{vi}-\bar{r_v})^2}}
$$

其中，$I$ 表示用户 $u$ 和用户 $v$ 都评分过的书籍集合，$\bar{r_u}$ 表示用户 $u$ 的平均评分。

根据用户相似度，可以预测用户 $u$ 对书籍 $j$ 的评分：

$$
\hat{r_{uj}} = \bar{r_u} + \frac{\sum_{v \in U}sim(u,v)(r_{vj}-\bar{r_v})}{\sum_{v \in U}|sim(u,v)|}
$$

其中，$U$ 表示与用户 $u$ 相似的用户集合。

### 4.2  内容推荐算法

内容推荐算法的核心思想是“物以类聚”，即主题相似的书籍，可能会被同一个用户喜欢。

假设书籍 $i$ 的特征向量为 $\vec{x_i}$，用户 $u$ 的兴趣向量为 $\vec{y_u}$，则用户 $u$ 对书籍 $i$ 的兴趣度可以表示为：

$$
interest(u,i) = \vec{x_i} \cdot \vec{y_u}
$$

根据用户兴趣度，可以预测用户 $u$ 对书籍 $j$ 的评分：

$$
\hat{r_{uj}} = \frac{interest(u,j)}{\sum_{i \in I} interest(u,i)}
$$

其中，$I$ 表示所有书籍的集合。

def cosine_similarity(vector1, vector2):
    """
    计算余弦相似度

    参数：
        vector1: 向量1
        vector2: 向量2

    返回值：
        余弦相似度
    """

    dot_product = np.dot(vector1, vector2)
    norm1 = np.linalg.norm(vector1)
    norm2 = np.linalg.norm(vector2)

    if norm1 == 0 or norm2 == 0:
        return 0

    return dot_product / (norm1 * norm2)
