### 4.2 基于强化学习的信号灯控制
利用强化学习算法,如Q-learning、SARSA等,可以实现对交通信号灯的动态优化控制,提高整体交通网络的通行效率。以下是一个基于OpenAI Gym的信号灯控制环境的代码示例:

```python
import gym
import numpy as np
from gym.spaces import Discrete, Box

class TrafficLightEnv(gym.Env):
    def __init__(self):
        self.action_space = Discrete(2)
        self.observation_space = Box(low=0, high=1, shape=(4,), dtype=np.float32)
        self.state = np.array([0.5, 0.5, 0.5, 0.5])
        self.reward = 0

    def step(self, action):
        if action == 0:
            self.state[0] += 0.1
            self.state[1] -= 0.1
        else:
            self.state[0] -= 0.1
            self.state[1] += 0.1
        self.state = np.clip(self.state, 0, 1)
        self.reward = -np.sum(np.abs(self.state - 0.5))
        done = False
        return self.state, self.reward, done, {}

    def reset(self):
        self.state = np.array([0.5, 0.5, 0.5, 0.5])
        self.reward = 0
        return self.state
```

该环境模拟了两个交叉路口的信号灯控制,agent可以通过选择不同的动作(切换信号灯)来最大化整体通行效率。利用强化学习算法,agent可以学习到最优的信号灯控制策略。
