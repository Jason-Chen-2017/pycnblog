## 4. 具体最佳实践：代码实例和详细解释说明
以下是基于BERT模型进行法律文书分析的Python代码示例:

```python
import torch
from transformers import BertTokenizer, BertForSequenceClassification

# 加载预训练的BERT模型和分词器
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
model = BertForSequenceClassification.from_pretrained('bert-base-uncased', num_labels=2)

# 输入法律文书文本
text = "This contract is made on January 1, 2023 between Party A and Party B. Party A agrees to provide the following services..."

# 文本预处理
input_ids = tokenizer.encode(text, return_tensors='pt')

# 模型推理
output = model(input_ids)
logits = output.logits
predicted_class = logits.argmax(-1).item()

if predicted_class == 0:
    print("This is a legal contract.")
else:
    print("This is not a legal contract.")
```

该代码展示了如何利用预训练的BERT模型对法律文书进行分类识别。通过输入法律文书文本,BERT模型可以自动识别出该文书是否为合法的法律合同。这种方法可以大大提高法律文书分析的效率和准确性。
