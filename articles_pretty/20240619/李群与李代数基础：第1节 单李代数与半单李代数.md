## 2. 核心概念与联系

### 李群的定义

李群是一种连续群，其中群的乘法运算在某些情况下可以微分，从而产生一个李代数。李群的元素通常表示为矩阵，而李群的乘法运算对应于矩阵的乘法。

### 李代数的定义

李代数是李群的“微分”版本，它定义在李群的邻域内，捕捉了群的局部性质。李代数通过李括号运算来定义，这个运算满足反身性、交换律和结合律。

### 单李代数与半单李代数

- **单李代数**：如果李代数没有非零的可约子代数，则称为单李代数。
- **半单李代数**：如果李代数没有非零的可约子代数且不为零，则称为半单李代数。

单李代数是李代数分类中的一个基本组成部分，而半单李代数则是在单李代数的基础上进一步细分的一个类别，具有更丰富的结构和理论价值。

```cpp
#include <Eigen/Dense>

// 定义二维旋转群的生成元和李括号运算
Eigen::Matrix2d rotationMatrix(const double& theta) {
    return Eigen::Matrix2d::Identity() * cos(theta) - Eigen::Matrix2d::Identity() * sin(theta);
}

Eigen::Matrix2d skewSymmetric(const Eigen::Matrix2d& matrix) {
    return matrix - matrix.transpose();
}

void computeLieBracket(const Eigen::Matrix2d& A, const Eigen::Matrix2d& B) {
    Eigen::Matrix2d AB = A * B;
    Eigen::Matrix2d BA = B * A;
    Eigen::Matrix2d bracket = AB - BA;
    std::cout << \"Lie Bracket [A,B]:\
\" << bracket << std::endl;
}

int main() {
    double theta1 = M_PI / 4;
    double theta2 = M_PI / 6;
    Eigen::Matrix2d A = rotationMatrix(theta1);
    Eigen::Matrix2d B = rotationMatrix(theta2);
    computeLieBracket(A, B);
    return 0;
}
```
