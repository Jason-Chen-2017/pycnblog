## 2. 核心概念与联系

### 2.1 同伦群πn(Sn)

同伦群πn(Sn)定义了Sn上的n维循环的空间，其中n为非负整数。对于n=0, π0(Sn)总是Z/2Z，表示Sn是否可以被连续映射到一个圈。当n为偶数时，πn(Sn)与整数环Z相关联，而当n为奇数时，πn(Sn)=0。这一现象反映了球面的内在几何结构和连续性的性质。

### 2.2 Hopf同伦分类

Hopf同伦分类给出了πn(Sn)的具体结构，尤其是当n为偶数时。Hopf构造了一个映射，将πn(Sn)与某些特定的群关联起来，揭示了πn(Sn)的结构与其在拓扑群中的地位。这一分类不仅加深了对同伦群的理解，也为后续的拓扑研究提供了宝贵的线索。

    if n % 2 == 0:
        if n == 0:
            return Zmod(2)
        else:
            return ZZ
    else:
        return Zmod(0)

def calculate_pi(sn, n):
    mapping = hopf_mapping(n)
    if mapping == ZZ:
        print(f\"π{n}(S{sn}) is isomorphic to {ZZ}\")

        print(f\"π{n}(S{sn}) is zero\")

        print(f\"π{n}(S{sn}) is {mapping}\")
