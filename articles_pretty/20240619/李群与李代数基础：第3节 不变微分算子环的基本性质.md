## 2. 核心概念与联系

### 李群

李群是由连续变换构成的群，其中群运算（即群的乘法）在群的邻域内保持可微性。李群的概念融合了群论和微积分的精髓，使得我们能够研究群的局部性质，同时保持对群的整体结构的洞察。

### 李代数

李代数是与李群相对应的代数结构，它定义在李群的质点处的切空间上。李代数提供了李群局部行为的代数描述，通过李括号这一操作，揭示了李群的内在结构和性质。

### 不变微分算子环

不变微分算子环是指作用于某个空间上的微分算子组成的环，其中这些算子在李群的作用下保持不变。这一概念在李群理论中尤为重要，因为它关联着群的行动不变性，并且在描述李群的几何结构和动力系统行为时具有关键作用。

## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

- **李群**：设$G$为李群，其李代数记作$\\mathfrak{g}$。对于$G$中的任意元素$g$，定义$Ad_g:\\mathfrak{g}\\to\\mathfrak{g}$为李群作用下的李括号映射，满足$Ad_g[x,y]=[Ad_g(x),Ad_g(y)]$。

### 4.2 公式推导过程

- **不变性定理**：如果微分算子$D$在李群$G$的作用下保持不变，即$\\forall g\\in G, D(g\\cdot f)=g\\cdot D(f)$，则$D$属于某个不变微分算子环。

### 4.3 案例分析与讲解

#### 案例一：旋转群$SO(3)$

- **李群**：$SO(3)$是三维空间中所有正交矩阵组成的群，表示三维空间的旋转。
- **李代数**：$\\mathfrak{so}(3)$是$SO(3)$的李代数，由$3\\times3$反对称矩阵组成。
- **微分算子**：考虑$SO(3)$上的微分算子$D_f$，定义为$D_f(x)=f'(x)$，其中$f$是$SO(3)$上的光滑函数，$f'$是其导数。
- **不变性**：验证$D_f$在$SO(3)$的作用下保持不变，即$\\forall g\\in SO(3), D_f(g\\cdot f)=g\\cdot D_f(f)$。

#### 案例二：李群$SL(n,\\mathbb{R})$

- **李群**：$SL(n,\\mathbb{R})$是实数域上的$n\\times n$行列式为1的所有矩阵组成的群。
- **李代数**：$\\mathfrak{sl}(n,\\mathbb{R})$是$n\\times n$实数矩阵的集合，其中每行和每列的元素之和为零。
- **微分算子**：考虑$SL(n,\\mathbb{R})$上的微分算子$D_f$，定义为$D_f(A)=f'(A)$，其中$f$是$SL(n,\\mathbb{R})$上的光滑函数，$f'$是其导数。
- **不变性**：验证$D_f$在$SL(n,\\mathbb{R})$的作用下保持不变，即$\\forall A\\in SL(n,\\mathbb{R}), D_f(g\\cdot A)=g\\cdot D_f(A)$。

def rotation_matrix(theta):
    \"\"\"生成旋转矩阵\"\"\"
    c, s = np.cos(theta), np.sin(theta)
    return np.array([[c, -s], [s, c]])

def differential_operator(f):
    \"\"\"定义微分算子\"\"\"
    return lambda x: np.gradient(f(x), axis=0)

def check_invariance(f, theta):
    \"\"\"验证微分算子的不变性\"\"\"
    operator = differential_operator(f)
    original = operator(f)
    rotated = operator(lambda x: rotation_matrix(theta) @ x)
    return np.allclose(original, rotated)
