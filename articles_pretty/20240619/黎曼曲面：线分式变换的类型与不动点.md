## 2. 核心概念与联系

### 2.1 黎曼曲面基本概念

黎曼曲面可以被看作是复平面上的多孔性区域，每个孔都对应着一个“手”的把手或洞。在数学上，黎曼曲面是由一组局部定义的复变量函数和连续变换组成的集合。黎曼曲面的局部性质决定了其全局结构和性质，例如奇点、周期点和分支点的存在与否。

### 2.2 线分式变换

线分式变换是一种特殊的复变函数，形式为$f(z) = \\frac{az + b}{cz + d}$，其中$a, b, c, d$是复数，且$ad - bc \
eq 0$。这种变换在复平面上定义了一个映射，它不仅改变了点的位置，还可能改变点的局部性质，如奇点和周期点的性质。

### 2.3 不动点

不动点是线分式变换的一个特殊性质，指的是变换作用于某点时该点保持不变。即满足$f(p) = p$的点$p$称为线分式变换的不动点。不动点的存在与否以及它们的性质对于理解变换的全局行为至关重要。

### 4.2 公式推导过程

不动点的定义为$f(z) = z$。将$f(z)$代入等式$f(z) = \\frac{az + b}{cz + d} = z$，可以得到不动点的方程：

$$z = \\frac{az + b}{cz + d}$$

通过移项和简化，可以得到：

$$cz^2 + dz = az + b$$

$$cz^2 + dz - az - b = 0$$

$$cz^2 + (d-a)z - b = 0$$

这是一个关于$z$的二次方程，可以通过求根公式来解，得到不动点的具体形式。
