## 2. 核心概念与联系

### 2.1 环论概述

环是一个代数结构，由一个非空集合\\( R \\)，以及在其上的两个二元运算：加法（\\( + \\)）和乘法（\\( \\cdot \\)），满足以下性质：

- 结合律：\\( (a + b) + c = a + (b + c) \\)，\\( (a \\cdot b) \\cdot c = a \\cdot (b \\cdot c) \\)
- 单位元：存在一个元素\\( 0 \\)，对于任意\\( a \\in R \\)，有\\( a + 0 = 0 + a = a \\)；存在一个元素\\( 1 \\)，对于任意\\( a \\in R \\)，有\\( a \\cdot 1 = 1 \\cdot a = a \\)
- 分配律：\\( a \\cdot (b + c) = a \\cdot b + a \\cdot c \\) 和 \\( (a + b) \\cdot c = a \\cdot c + b \\cdot c \\)

### 2.2 Brauer群定义

Brauer群，记作\\( Br(R) \\)，是与环\\( R \\)相关的数学结构，它由所有有限维不约化\\( R \\)-代数组成，这些代数的中心为\\( R \\)的商环。这里的“不约化”意味着不存在非平凡的理想使得这个代数成为一个直积环。Brauer群在环的分类和理解方面起着关键作用，尤其是在研究环的结构和性质时。

### 2.3 Brauer群与代数群的联系

Brauer群与代数群之间存在着深刻的联系，主要体现在代数群的分类和研究中。代数群是代数结构和群结构的结合体，通常在复数域或者更一般的局部域上定义。通过Brauer群，可以将代数群的结构信息映射到环论的框架下，进而通过环的性质来分析和理解代数群的性质。这一联系不仅丰富了代数群的理论，也为探索环论和群论的深层联系开辟了新的途径。

\\[ Br(k) := \\{[\\alpha] \\mid \\alpha \\in \\mathscr{A}(k) \\} \\]

其中，\\(\\mathscr{A}(k)\\)是所有有限维\\( k \\)-代数的集合，且\\[ \\alpha \\sim \\beta \\]当且仅当\\( \\alpha/\\beta \\)在\\( k \\)-分割环上是等价的。这里的\\( [\\alpha] \\)表示\\( \\alpha \\)的等价类。
