## 4.项目实践：代码实例和详细解释说明

首先，我们来看一下产品展示页面的代码实例：

```javascript
Page({
  data: {
    productList: []
  },
  onLoad: function() {
    wx.request({
      url: 'https://example.com/api/getProductList',
      method: 'GET',
      success: (res) => {
        this.setData({
          productList: res.data
        });
      }
    });
  }
});
```

在这段代码中，我们首先定义了一个名为`productList`的数组来存储我们的产品列表。然后，在页面加载时，我们发送一个GET请求到我们的服务器来获取产品列表。当请求成功时，我们把返回的数据保存到`productList`中。
