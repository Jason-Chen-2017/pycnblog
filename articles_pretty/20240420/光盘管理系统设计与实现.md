## 3. 核心算法原理和具体操作步骤

### 3.1 算法原理

深度优先搜索是一种用于遍历或搜索树或图的算法。这种算法从树的根（或图的某一指定节点）开始，尽可能深地搜索树的分支。当节点v的所在边都已被探寻过，搜索将回溯到发现节点v的那条边的起始节点。这一过程一直进行到已发现从源节点可达的所有节点为止。如果还存在未被发现的节点，则选择其中一个作为源节点并重复以上过程，整个进程反复进行直到所有节点都被访问为止。

### 3.2 操作步骤

1. 从根节点开始，选择一个未访问的子节点，标记它并访问它。
2. 如果当前节点没有未访问的子节点，则回溯到上一节点。
3. 如果上一节点没有未访问的子节点，则继续回溯，直到找到一个有未访问子节点的节点，然后访问该子节点。
4. 重复以上步骤，直到所有节点都被访问过。

## 4. 数学模型和公式详细讲解举例说明

在深度优先搜索中，我们可以用以下公式来表示搜索过程：

$$
DFS(v):
\begin{cases}
\text{if } v \text{ is visited, return}\\
\text{mark } v \text{ as visited}\\
\text{for each } w \text{ in } neighbors(v) \text{ do DFS(w)}
\end{cases}
$$

在这个公式中，$v$代表当前节点，$neighbors(v)$代表$v$的所有邻居节点，也就是$v$的所有子节点。这个公式表示的是，对于每个节点$v$，如果$v$已经被访问过了，那么就直接返回；否则，标记$v$为已访问，然后对$v$的所有邻居节点（也就是$v$的所有子节点）进行深度优先搜索。
