模型的基本公式如下：

$$
R(u,i) = \sum_{j \in N(i)} Sim(i,j) \cdot R(u,j)
$$

这个公式表示的是用户 $u$ 对物品 $i$ 的预测评分 $R(u,i)$ 等于用户 $u$ 对物品 $j$ 的实际评分 $R(u,j)$ 与物品 $i$ 和 $j$ 的相似度 $Sim(i,j)$ 的乘积之和。

import numpy as np

# Load data

# Calculate similarity
similarity = data.corr()

# Predict rating
def predict_rating(user_id, item_id):
    top_similar = similarity[item_id].sort_values(ascending=False)[1:10]
    top_similar_items = top_similar.index
    top_similar_values = top_similar.values
    user_ratings = data.loc[user_id, top_similar_items]
