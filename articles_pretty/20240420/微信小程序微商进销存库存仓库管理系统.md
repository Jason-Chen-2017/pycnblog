## 4.数学模型和公式详细讲解举例说明

### 4.1 进货总价的计算

进货总价的计算公式为：

$$
进货总价 = 商品数量 * 进货价格
$$

例如，如果进货的商品数量为10，进货价格为20元，那么进货总价为：

$$
进货总价 = 10 * 20 = 200元
$$

### 4.2 销售总额的计算

销售总额的计算公式为：

$$
销售总额 = 商品数量 * 销售价格
$$

例如，如果销售的商品数量为5，销售价格为30元，那么销售总额为：

$$
销售总额 = 5 * 30 = 150元
$$

### 4.3 库存的计算

库存的计算公式为：

$$
库存 = 进货数量 - 销售数量
$$

例如，如果进货数量为10，销售数量为5，那么库存为：

$$
库存 = 10 - 5 = 5
$$

```javascript
  // 查询进货信息
  db.collection('products').get().then(res => {
    let purchaseQuantity = res.data[0].quantity;
    // 查询销售信息
    db.collection('sales').get().then(res => {
      let salesQuantity = res.data[0].quantity;
      // 计算库存

      // 更新库存信息
      db.collection('inventory').add({
        data: {
          inventory: inventory
        }
      });
    });
  });
}
```
