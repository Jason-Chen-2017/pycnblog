```python
INSTALLED_APPS = [
    ...
    'shop',
]
```

接下来，我们需要设计我们的数据模型。在`shop/models.py`中，我们创建`Product`和`Order`两个模型：

```python
from django.db import models
from django.contrib.auth.models import User

class Product(models.Model):
    name = models.CharField(max_length=200)
    price = models.DecimalField(max_digits=5, decimal_places=2)

class Order(models.Model):
    user = models.ForeignKey(User, on_delete=models.CASCADE)
    products = models.ManyToManyField(Product)
```

## 4.数学模型和公式详细讲解举例说明

在网络购物系统中，我们经常需要进行价格的计算。假设我们有一个购物车，其中包含n个商品，每个商品的价格为$p_i$，那么购物车中所有商品的总价格$P$可以用下面的公式来计算：

$$
P = \sum_{i=1}^{n} p_i
$$

这个公式非常简单，但是在实际的开发中，我们还需要考虑到折扣、税费等因素，这就需要我们对公式进行一些修改。

## 4.项目实践：代码实例和详细解释说明

我们已经创建了数据模型，接下来我们需要创建视图和模板。在`shop/views.py`中，我们创建一个用于显示商品列表的视图：

```python
from django.shortcuts import render
from .models import Product

def product_list(request):
    products = Product.objects.all()
    return render(request, 'shop/product_list.html', {'products': products})
```

在`shop/templates/shop/product_list.html`中，我们创建一个用于显示商品列表的模板：

```html
{% for product in products %}
    <div>
        <h2>{{ product.name }}</h2>
        <p>{{ product.price }}</p>
    </div>
{% endfor %}
```

接下来，我们需要在`myshop/urls.py`中添加一个URL模式，将URL和视图关联起来：

```python
from django.urls import path
from shop import views

urlpatterns = [
    path('products/', views.product_list, name='product_list'),
]
```

至此，我们已经完成了商品列表的显示。用户可以在网页上看到所有的商品，每个商品包括名称和价格。
