## 4.数学模型和公式详细讲解举例说明
### 4.1 TF-IDF
TF-IDF是一种常用的特征提取方法，它由两部分组成：TF(Term Frequency,词频)和IDF(Inverse Document Frequency,逆文档频率)。TF表示一个词在文档中出现的频率，IDF表示一个词在所有文档中出现的频率的倒数。

TF的计算公式为：
$$ TF(t, d) = \frac{n_{t,d}}{\sum_{k}n_{k,d}} $$
其中，$n_{t,d}$表示词t在文档d中出现的次数，$\sum_{k}n_{k,d}$表示文档d中所有词出现的次数之和。

IDF的计算公式为：
$$ IDF(t, D) = \log{\frac{|D|}{1+|{d\in D: t\in d}|}} $$
其中，|D|表示文档集D中文档的总数，$|{d\in D: t\in d}|$表示含有词t的文档数。

然后，TF-IDF的计算公式为：
$$ TFIDF(t, d, D) = TF(t, d) \times IDF(t, D) $$
TF-IDF的值越大，表示词t在文档d中的重要性越高。

### 4.2 朴素贝叶斯
朴素贝叶斯是一种基于贝叶斯定理的分类算法，它假设特征之间是独立的。对于一个文档d，其属于类别c的概率可以表示为：
$$ P(c|d) = \frac{P(d|c)P(c)}{P(d)} $$
其中，$P(d|c)$可以通过计算词的条件概率得到，$P(c)$是类别c的先验概率，$P(d)$是文档d的概率。由于对所有的类别来说，$P(d)$都是相同的，因此在分类时，可以忽略$P(d)$，只需要比较$P(d|c)P(c)$的大小。

```python
import pandas as pd
import jieba

from sklearn.naive_bayes import MultinomialNB
from sklearn.metrics import classification_report
```

然后，我们读取网易云音乐的评论数据，并进行预处理。

```python
data = pd.read_csv('comments.csv')
data['content'] = data['content'].apply(lambda x: ' '.join(jieba.cut(x)))
```

接下来，我们使用TF-IDF进行特征提取。

```python
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(data['content'])
```

然后，我们使用朴素贝叶斯进行情感分类。

```python
model = MultinomialNB()
model.fit(X, data['label'])
```

最后，我们可以对新的评论进行情感分析。

```python
new_comment = vectorizer.transform(['这首歌真好听'])
result = model.predict(new_comment)
print('情感分析结果：', result)
```
