## 4.项目实践：代码实例和详细解释说明

以下是一个简单的药品管理系统的代码实例。

```python
class Drug:
    def __init__(self, name, category, manufacturer, purchase_price, sale_price):
        self.name = name
        self.category = category
        self.manufacturer = manufacturer
        self.purchase_price = purchase_price
        self.sale_price = sale_price
        self.stock = 0

    def purchase(self, quantity):
        self.stock += quantity

    def sell(self, quantity):
        if quantity > self.stock:
            print("库存不足")
            return
        self.stock -= quantity

    def show_info(self):
        print(f"药品名称：{self.name}")
        print(f"药品类别：{self.category}")
        print(f"生产厂商：{self.manufacturer}")
        print(f"进货价格：{self.purchase_price}")
        print(f"售价：{self.sale_price}")
        print(f"库存：{self.stock}")
```

在这个代码示例中，我们定义了一个名为`Drug`的类，用于表示药品。这个类有几个属性，如`name`（药品名称）、`category`（药品类别）、`manufacturer`（生产厂商）、`purchase_price`（进货价格）、`sale_price`（售价）和`stock`（库存）。同时，这个类也有几个方法，如`purchase`（购入药品）、`sell`（卖出药品）和`show_info`（显示药品信息）。
