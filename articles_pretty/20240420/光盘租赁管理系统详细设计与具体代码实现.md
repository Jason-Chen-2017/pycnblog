## 4.项目实践：代码实例和详细解释说明

下面我们将使用Java语言和MySQL数据库来实现光盘租赁管理系统。我们首先需要创建相应的数据库表结构，然后实现相应的功能。

### 4.1 数据库表结构

我们需要创建三个表：`光盘`、`客户`和`租赁`。表的结构如下：

```sql
CREATE TABLE 光盘 (
  光盘号 CHAR(10) NOT NULL,
  名称 VARCHAR(100) NOT NULL,
  类型 VARCHAR(20),
  租赁状态 CHAR(1),
  PRIMARY KEY (光盘号)
);

CREATE TABLE 客户 (
  客户号 CHAR(10) NOT NULL,
  姓名 VARCHAR(50) NOT NULL,
  联系方式 VARCHAR(50),
  PRIMARY KEY (客户号)
);

CREATE TABLE 租赁 (
  租赁编号 CHAR(10) NOT NULL,
  光盘号 CHAR(10) NOT NULL,
  客户号 CHAR(10) NOT NULL,
  租赁时间 DATETIME,
  归还时间 DATETIME,
  PRIMARY KEY (租赁编号),
  FOREIGN KEY (光盘号) REFERENCES 光盘(光盘号),
  FOREIGN KEY (客户号) REFERENCES 客户(客户号)
);
```
