### 5.3 创建用户注册和登录视图

在Django中，我们可以使用视图来处理用户的请求。下面是一个简单的用户注册和登录视图的例子：

```python
from django.shortcuts import render, redirect
from .models import User
from django.contrib import messages

def register(request):
    if request.method == 'POST':
        username = request.POST['username']
        password = request.POST['password']
        email = request.POST['email']
        
        user = User(username=username, password=password, email=email)
        user.save()
        
        messages.success(request, '注册成功！')
        return redirect('login')
    else:
        return render(request, 'register.html')

def login(request):
    if request.method == 'POST':
        username = request.POST['username']
        password = request.POST['password']
        
        user = User.objects.filter(username=username, password=password)
        
        if user:
            return redirect('home')
        else:
            messages.error(request, '用户名或密码错误！')
            return redirect('login')
    else:
        return render(request, 'login.html')
```

在注册视图中，我们首先检查请求的方法是否为POST。如果是，说明用户提交了注册表单，我们就获取表单中的数据，创建一个新的用户，并保存到数据库中。如果请求的方法不是POST，说明用户访问了注册页面，我们就返回注册页面。

在登录视图中，我们也首先检查请求的方法是否为POST。如果是，说明用户提交了登录表单，我们就获取表单中的数据，从数据库中查找是否存在对应的用户。如果存在，说明用户名和密码正确，我们就重定向用户到首页。如果不存在，说明用户名或密码错误，我们就返回登录页面，并显示错误信息。
