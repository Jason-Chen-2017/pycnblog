### 4.3 权限管理

在课程管理系统中，我们需要为不同的用户角色设置不同的权限。在Java中，我们可以通过角色和权限的数据表，以及用户、角色和权限的关联表，来实现权限管理。

例如，我们可以创建一个用户表（user）、一个角色表（role）、一个权限表（permission），以及一个用户角色关联表（user_role）和一个角色权限关联表（role_permission）。

```sql
CREATE TABLE user (
    user_id INT PRIMARY KEY,
    username VARCHAR(50),
    password VARCHAR(50)
);

CREATE TABLE role (
    role_id INT PRIMARY KEY,
    role_name VARCHAR(50)
);

CREATE TABLE permission (
    permission_id INT PRIMARY KEY,
    permission_name VARCHAR(50)
);

CREATE TABLE user_role (
    user_id INT,
    role_id INT,
    PRIMARY KEY (user_id, role_id)
);

CREATE TABLE role_permission (
    role_id INT,
    permission_id INT,
    PRIMARY KEY (role_id, permission_id)
);
```

在Java中，我们可以通过用户的ID，查询出用户的所有角色和权限：
