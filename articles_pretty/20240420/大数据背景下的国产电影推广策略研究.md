## 4.数学模型和公式详细讲解举例说明

在数据分析过程中，我们常常需要使用到各种数学模型。例如，在聚类分析中，我们可以使用K-means算法。K-means算法的基本思想是通过迭代，将数据点分配到K个聚类中，使得每个数据点到其所在聚类的中心的距离之和最小。

K-means算法的数学模型可以表示为：

$$
min \sum_{i=1}^{k}\sum_{x \in C_i}||x-\mu_i||^2
$$

其中，$C_i$ 是第 i 个聚类，$\mu_i$ 是第 i 个聚类的中心，$||x-\mu_i||^2$ 是数据点 x 到聚类中心的距离。

## 4.项目实践：代码实例和详细解释说明

接下来，我们以Python为例，展示如何使用K-means算法进行聚类分析。我们使用的数据是某电影的观众年龄和观看次数。

```python
from sklearn.cluster import KMeans
import numpy as np

# 假设我们有以下的数据
age = [25, 26, 28, 29, 30, 31, 35, 36, 40, 42]
times = [1, 2, 1, 3, 2, 1, 4, 3, 2, 1]

# 将数据转换为Numpy数组
X = np.array(list(zip(age, times)))

# 创建KMeans对象
kmeans = KMeans(n_clusters=3)

# 进行聚类
kmeans.fit(X)

# 输出每个数据点的所在聚类
print(kmeans.labels_)
```
