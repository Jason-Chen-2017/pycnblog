$$ O = f(I * K + b) $$

其中，$K$是卷积核，$b$是偏置，$*$表示卷积操作，$f$是激活函数，如ReLU函数。

### 4.2 决策控制的映射

在强化学习中，决策控制的映射可以通过Q-learning算法来实现。Q-learning算法通过学习一个动作价值函数$Q(s, a)$，来评估在状态$s$下执行动作$a$的价值。动作价值函数$Q(s, a)$的更新规则如下：

$$ Q(s, a) \leftarrow Q(s, a) + \alpha [r + \gamma \max_{a'} Q(s', a') - Q(s, a)] $$

其中，$\alpha$是学习率，$r$是即时奖励，$\gamma$是折扣因子，$s'$是状态$s$执行动作$a$后的新状态，$a'$是新状态$s'$下的最佳动作。

```python
import gym

Q = np.zeros((env.observation_space.n, env.action_space.n))
alpha = 0.5
gamma = 0.95

for i_episode in range(1000):
    state = env.reset()
    for t in range(100):
        action = np.argmax(Q[state]) if np.random.uniform(0, 1) > epsilon else env.action_space.sample()
        next_state, reward, done, info = env.step(action)

        state = next_state
        if done:
            break
```
