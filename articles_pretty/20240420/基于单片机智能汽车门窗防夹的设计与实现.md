下面是一个简单的单片机程序实例，实现了车窗防夹功能的基本逻辑。

```c
#include <reg51.h>

#define THRESHOLD 10  //设定阈值

void main() {
    int v;  //车窗速度
    while (1) {
        v = get_speed();  //获取车窗速度
        if (v < THRESHOLD) {  //如果车窗速度小于阈值
            stop_window();  //停止车窗运动
        }
    }
}
```

在这个代码实例中，我们首先定义了一个阈值`THRESHOLD`，然后在主函数中，我们通过一个无限循环，不断获取车窗的速度并进行判断。如果车窗的速度小于阈值，就调用`stop_window()`函数停止车窗的运动，实现了防夹功能。
