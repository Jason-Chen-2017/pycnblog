## 4.项目实践：代码实例和详细解释说明

下面我们来看一下宾馆管理系统的具体代码实现。这里以客房预订功能为例，给出其主要代码和实现思路。

```python
class Room:
    def __init__(self, room_id, room_type):
        self.room_id = room_id
        self.room_type = room_type
        self.status = 'available'

    def book(self):
        if self.status == 'available':
            self.status = 'booked'
            return True
        else:
            return False
```

上面的代码定义了一个`Room`类，表示一个客房。每个客房有一个唯一的`room_id`和`room_type`，以及一个状态`status`。`book`方法可以用来预订客房，如果客房已经被预订，就返回`False`。
