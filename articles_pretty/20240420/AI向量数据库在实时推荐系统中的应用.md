假设我们有一个用户行为数据集$D = \{d_1, d_2, ..., d_n\}$，其中每个$d_i$都是一个用户的行为记录。我们的目标是将每个$d_i$转化为一个向量$v_i$。

向量化的过程可以用下面的公式表示：

$$
v_i = f(d_i)
$$

其中，$f$是向量化函数，它的作用是将用户行为记录$d_i$转化为向量$v_i$。

### 3.2 相似度计算

相似度计算是计算两个向量之间相似度的过程。在推荐系统中，我们需要计算用户行为向量和物品属性向量之间的相似度。相似度的计算方法也有很多种，比如余弦相似度，欧氏距离等。

假设我们有一个用户行为向量$v_u$和一个物品属性向量$v_i$，我们的目标是计算它们之间的相似度$s$。

相似度计算的过程可以用下面的公式表示：

$$
s = g(v_u, v_i)
$$

其中，$g$是相似度计算函数，它的作用是计算用户行为向量$v_u$和物品属性向量$v_i$之间的相似度$s$。

接下来，我们以余弦相似度为例，详细讲解相似度计算的过程。

假设我们有两个向量$v_u = (x_1, x_2, ..., x_n)$和$v_i = (y_1, y_2, ..., y_n)$，它们的余弦相似度$s$可以用下面的公式计算：

$$
s = \frac{\sum_{j=1}^{n} x_j y_j}{\sqrt{\sum_{j=1}^{n} x_j^2}\sqrt{\sum_{j=1}^{n} y_j^2}}
$$

这个公式的分子是$v_u$和$v_i$的点积，分母是$v_u$和$v_i$的模长。这个公式的值在-1到1之间，值越大，表示$v_u$和$v_i$越相似。

```python
def vectorize(d):

# 相似度计算函数
def cosine_similarity(v_u, v_i):
    return np.dot(v_u, v_i) / (np.linalg.norm(v_u) * np.linalg.norm(v_i))
```

最后，我们就可以使用上面的函数来实现实时推荐系统了：
