## 2.核心概念与联系

### 2.1 向量积的定义与性质

向量积是一种特殊的二元运算，它接受两个向量作为输入，输出一个新的向量。向量积的一个重要特性是它的结果是垂直于输入向量构成的平面的。

### 2.2 张量积的定义与性质

张量积（也称为外积或笛卡尔积）是一种将两个向量组合成一个二阶张量的运算。这个新的张量可以被视为一个矩阵，其中的每个元素都是原始向量中对应元素的乘积。

### 2.3 向量积与张量积的联系

向量积和张量积都是矩阵运算的一种，它们都可以将两个或多个矩阵（或向量）组合在一起。然而，它们的结果和使用方式却有很大的不同。

### 5.1 向量积的代码实例

在 Python 中，我们可以使用 Numpy 库中的 `cross` 函数来计算向量积：

```python
import numpy as np

a = np.array([1, 0, 0])
b = np.array([0, 1, 0])

c = np.cross(a, b)

print(c)  # 输出：[0 0 1]
```

### 5.2 张量积的代码实例

同样，我们可以使用 Numpy 库中的 `outer` 函数来计算张量积：

```python
import numpy as np

a = np.array([1, 2])
b = np.array([3, 4])

c = np.outer(a, b)

print(c)  # 输出：[[3 4] [6 8]]
```
