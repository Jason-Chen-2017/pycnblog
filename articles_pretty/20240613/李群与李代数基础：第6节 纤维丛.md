# 李群与李代数基础：第6节 纤维丛

## 1.背景介绍

纤维丛是微分几何和代数几何中的一个重要概念,它将拓扑空间与其上的矢量丛联系起来,为研究流形上的矢量场、微分方程和并行输运等问题提供了强有力的工具。纤维丛理论不仅在数学领域有着广泛的应用,在物理学中也扮演着重要角色,例如在规范场论、相对论和量子力学等领域。

## 2.核心概念与联系

### 2.1 纤维丛的定义

纤维丛由以下四个组成部分构成:

1. **底空间(Base Space)** $B$
2. **总空间(Total Space)** $E$
3. **纤维(Fiber)** $F$
4. **射影(Projection)** $\pi: E \rightarrow B$

其中,底空间 $B$ 和纤维 $F$ 都是拓扑空间,总空间 $E$ 是一个覆盖在底空间 $B$ 上的空间,每一个点 $p \in B$ 对应着一个纤维 $F_p \cong F$,射影 $\pi$ 将总空间 $E$ 上的点映射到底空间 $B$ 上。

纤维丛可以形式化地定义为一个四元组 $(E, \pi, B, F)$,满足以下条件:

1. $\pi: E \rightarrow B$ 是一个连续映射。
2. 对于每个 $p \in B$,存在一个拓扑空间 $F_p$,使得 $\pi^{-1}(p) \cong F_p$。
3. 存在一个开覆盖 $\{U_\alpha\}$ 使得对于每个 $\alpha$,存在一个同胚映射 $\phi_\alpha: \pi^{-1}(U_\alpha) \rightarrow U_\alpha \times F$,使得对于每个 $x \in \pi^{-1}(U_\alpha)$,有 $\pi(x) = \text{pr}_1 \circ \phi_\alpha(x)$,其中 $\text{pr}_1: U_\alpha \times F \rightarrow U_\alpha$ 是投影到第一个因子的映射。

### 2.2 纤维丛的例子

1. **切丛(Tangent Bundle)**: 对于任意流形 $M$,其切丛 $TM$ 是一个纤维丛,其底空间是 $M$,纤维是 $\mathbb{R}^n$ (其中 $n$ 是 $M$ 的维数),总空间是所有切矢量构成的空间。
2. **余切丛(Cotangent Bundle)**: 对于任意流形 $M$,其余切丛 $T^*M$ 是一个纤维丛,其底空间是 $M$,纤维是 $\mathbb{R}^n$,总空间是所有余切矢量构成的空间。
3. **帧丛(Frame Bundle)**: 对于任意流形 $M$,其帧丛 $FM$ 是一个纤维丛,其底空间是 $M$,纤维是 $GL(n, \mathbb{R})$ (其中 $n$ 是 $M$ 的维数),总空间是所有帧构成的空间。

### 2.3 纤维丛的同构

两个纤维丛 $(E_1, \pi_1, B_1, F_1)$ 和 $(E_2, \pi_2, B_2, F_2)$ 被称为同构,如果存在一个同胚映射 $\Phi: E_1 \rightarrow E_2$,使得对于每个 $x \in E_1$,有 $\pi_2 \circ \Phi(x) = \pi_1(x)$,并且对于每个 $p \in B_1$,限制在纤维 $\pi_1^{-1}(p)$ 上的映射 $\Phi|_{\pi_1^{-1}(p)}: \pi_1^{-1}(p) \rightarrow \pi_2^{-1}(\pi_1(p))$ 是一个同构。

## 3.核心算法原理具体操作步骤

### 3.1 纤维丛的局部平凡性

纤维丛的一个重要性质是局部平凡性,即在底空间的每个点附近,纤维丛都可以看作是一个"产品"结构。更精确地说,对于每个 $p \in B$,存在一个包含 $p$ 的开邻域 $U$,使得 $\pi^{-1}(U)$ 同胚于 $U \times F$。这种局部平凡性允许我们在研究纤维丛时将问题分解为局部问题,从而简化了计算和分析。

### 3.2 纤维丛的切丛和余切丛

对于任意纤维丛 $(E, \pi, B, F)$,我们可以构造它的切丛 $TE$ 和余切丛 $T^*E$。切丛 $TE$ 是一个纤维丛,其底空间是 $E$,纤维是 $\mathbb{R}^{n+k}$ (其中 $n$ 是 $B$ 的维数, $k$ 是 $F$ 的维数),总空间是所有切矢量构成的空间。余切丛 $T^*E$ 是一个纤维丛,其底空间是 $E$,纤维是 $\mathbb{R}^{n+k}$,总空间是所有余切矢量构成的空间。

### 3.3 纤维丛的分类

纤维丛可以根据纤维的性质进行分类:

1. **矢量丛(Vector Bundle)**: 如果纤维是一个矢量空间,那么纤维丛被称为矢量丛。
2. **主丛(Principal Bundle)**: 如果纤维是一个李群,并且总空间上的群作用满足某些条件,那么纤维丛被称为主丛。
3. **仿射丛(Affine Bundle)**: 如果纤维是一个仿射空间,那么纤维丛被称为仿射丛。

### 3.4 纤维丛的构造

构造新的纤维丛是纤维丛理论的一个重要方面。一些常见的构造方法包括:

1. **直积构造**: 给定两个纤维丛 $(E_1, \pi_1, B, F_1)$ 和 $(E_2, \pi_2, B, F_2)$,我们可以构造它们的直积纤维丛 $(E_1 \times E_2, \pi_1 \times \pi_2, B, F_1 \times F_2)$。
2. **张量积构造**: 给定两个矢量丛 $(E_1, \pi_1, B, V_1)$ 和 $(E_2, \pi_2, B, V_2)$,我们可以构造它们的张量积矢量丛 $(E_1 \otimes E_2, \pi_1 \otimes \pi_2, B, V_1 \otimes V_2)$。
3. **对合构造**: 给定一个矢量丛 $(E, \pi, B, V)$,我们可以构造它的对合矢量丛 $(E^*, \pi^*, B, V^*)$,其中 $E^*$ 是 $E$ 的对合丛,纤维是 $V^*$。

## 4.数学模型和公式详细讲解举例说明

### 4.1 纤维丛上的切矢量场

设 $(E, \pi, B, F)$ 是一个纤维丛,我们可以定义纤维丛上的切矢量场。一个切矢量场 $X$ 是一个映射,将每个点 $x \in E$ 映射到 $X(x) \in T_xE$,并且满足以下条件:

1. 对于每个 $x \in E$,有 $\pi_*X(x) \in T_{\pi(x)}B$,其中 $\pi_*$ 是 $\pi$ 的导数。
2. 对于每个 $p \in B$,限制在纤维 $\pi^{-1}(p)$ 上的映射 $X|_{\pi^{-1}(p)}: \pi^{-1}(p) \rightarrow T_{\pi^{-1}(p)}E$ 是一个矢量丛同构。

我们可以将切矢量场分解为水平和垂直部分:

$$
X(x) = X_h(x) + X_v(x)
$$

其中 $X_h(x) \in \ker \pi_*$,称为 $X$ 在 $x$ 处的水平部分,而 $X_v(x) \in T_x(\pi^{-1}(\pi(x)))$,称为 $X$ 在 $x$ 处的垂直部分。

### 4.2 纤维丛上的连接

连接是纤维丛理论中的一个重要概念,它为纤维丛上的平行输运提供了一种方式。一个连接 $\nabla$ 是一个映射,将每个切矢量场 $X$ 映射到另一个切矢量场 $\nabla_X$,满足以下条件:

1. $\nabla$ 是 $\mathbb{R}$-线性的,即对于任意实数 $a,b$ 和切矢量场 $X,Y$,有 $\nabla_{aX+bY} = a\nabla_X + b\nabla_Y$。
2. $\nabla$ 满足leibniz规则,即对于任意切矢量场 $X,Y$ 和光滑函数 $f$,有 $\nabla_X(fY) = X(f)Y + f\nabla_XY$。

连接的作用是将切矢量场沿着另一个切矢量场做平行输运。具体来说,如果 $\gamma: [0,1] \rightarrow B$ 是一条曲线,那么对于任意切矢量场 $X$,方程

$$
\nabla_{\dot{\gamma}(t)}X(t) = 0
$$

定义了沿着 $\gamma$ 对 $X$ 的平行输运,其中 $\dot{\gamma}(t)$ 是 $\gamma$ 在 $t$ 处的切矢量。

### 4.3 曲率张量

曲率张量是纤维丛理论中另一个重要概念,它测量了纤维丛上的"扭曲程度"。对于任意连接 $\nabla$,我们可以定义它的曲率张量 $R$,它是一个映射,将每对切矢量场 $X,Y$ 映射到另一个切矢量场 $R(X,Y)$,定义为:

$$
R(X,Y) = \nabla_X\nabla_Y - \nabla_Y\nabla_X - \nabla_{[X,Y]}
$$

其中 $[X,Y]$ 是 $X$ 和 $Y$ 的李括号。曲率张量满足以下性质:

1. $R(X,Y)$ 是 $\mathbb{R}$-线性的,即对于任意实数 $a,b$,有 $R(aX+bY,Z) = aR(X,Z) + bR(Y,Z)$。
2. $R(X,Y)$ 满足斯基姆对称性,即对于任意切矢量场 $X,Y,Z$,有 $R(X,Y)Z + R(Y,Z)X + R(Z,X)Y = 0$。

曲率张量的非平凡性反映了纤维丛的"扭曲程度"。如果曲率张量为零,那么纤维丛是平坦的,否则它就是曲率的。

## 5.项目实践：代码实例和详细解释说明

在这一部分,我们将通过一个具体的例子来说明如何在Python中实现和可视化一个简单的纤维丛。我们将使用NumPy和Matplotlib库来进行数值计算和可视化。

### 5.1 Möbius带

Möbius带是一个经典的拓扑物体,它可以被视为一个纤维丛,其底空间是圆周 $S^1$,纤维是单位区间 $[0,1]$,总空间是一个扭曲的环面。我们将构造一个Möbius带的数值表示,并将其可视化。

```python
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

# 定义Möbius带的参数
n = 100  # 离散点数
r = 1    # 半径

# 生成底空间和纤维的离散点
theta = np.linspace(0, 2 * np.pi, n)
t = np.linspace(0, 1, n)

# 构造Möbius带的总空间
X = np.outer(np.cos(theta), (1 + 0.5 * t) * np.cos(np.pi * t))
Y = np.outer(np.sin(theta), (1 + 0.5 * t) * np.sin(np.pi * t))
Z = np.outer(np.ones(n), 0.5 * t)

# 可视化Möbius带
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z, rstride=1, cstride=1, cmap='viridis')
ax.set_xlabel('X')
ax.set_ylabel('Y')
ax.set_zlabel('Z')
ax.set_title('Möbius Band')
plt.show()
```
