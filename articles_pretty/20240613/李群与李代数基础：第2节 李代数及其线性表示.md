# 李群与李代数基础：第2节 李代数及其线性表示

## 1.背景介绍
### 1.1 李群与李代数的历史渊源
李群和李代数是现代数学和物理学中极其重要的概念,在微分几何、拓扑学、量子力学等领域有着广泛的应用。它们以挪威数学家Sophus Lie的开创性工作命名,他在19世纪70年代引入了连续变换群的概念,并系统地研究了它们的性质。

### 1.2 李代数在数学和物理学中的重要性
李代数作为李群的线性化,在数学上有着极其优雅的结构。它是一个带有李括号运算的向量空间,李括号满足一些特殊的性质如反对称性和雅可比等式。李代数的表示理论在现代数学中占有核心地位,与表示论、不变量论等多个分支有着紧密的联系。

在物理学中,李群和李代数更是无处不在。例如量子力学中的Heisenberg代数、角动量代数,粒子物理中的Poincare群、规范场论中的规范群等等,都是李群和李代数的经典实例。可以说,没有李群与李代数就难以想象现代物理学的发展。

本文将重点介绍李代数的定义和基本性质,以及它在线性表示理论中的重要作用。我们将从李代数的公理化定义出发,讨论它的一些重要结构如Cartan子代数、根空间分解等,并给出若干具体的例子。在此基础上,我们引入李代数表示的概念,讨论它与线性李群表示之间的关系,并给出Lie定理、Engel定理等一些重要结果。最后,我们将简要介绍李代数表示的完全可约性与不可约表示的分类问题。

## 2.核心概念与联系
### 2.1 李代数的定义与例子
#### 2.1.1 李代数的公理化定义
李代数$(L,[,])$是一个带有二元运算$[,]$的向量空间,其中$[,]$称为李括号(Lie bracket),满足以下性质:

1. 双线性性:$\forall x,y,z \in L, a,b \in \mathbb{F}$,有
$$ 
[ax+by,z] = a[x,z]+b[y,z], \quad [z,ax+by] = a[z,x]+b[z,y]
$$
2. 反对称性:$\forall x,y \in L$,有
$$
[x,y] = -[y,x]
$$
3. 雅可比等式:$\forall x,y,z \in L$,有
$$
[x,[y,z]] + [z,[x,y]] + [y,[z,x]] = 0
$$

#### 2.1.2 李代数的具体实例
1. $\mathfrak{gl}(n,\mathbb{F})$:由全体$n$阶方阵构成的向量空间,李括号定义为$[A,B]=AB-BA$。

2. $\mathfrak{sl}(n,\mathbb{F})$:由全体$n$阶迹为零方阵构成的向量空间,李括号同上。

3. 向量场李代数:由全体光滑向量场$\mathfrak{X}(M)$构成的向量空间,李括号定义为向量场的Lie导数:
$$
[X,Y] = \mathcal{L}_X Y
$$

### 2.2 李代数的结构理论初步
#### 2.2.1 李代数的导子与理想
设$L$为李代数,$x \in L$,定义$\mathrm{ad}_x:L \to L$为
$$
\mathrm{ad}_x(y) = [x,y], \quad \forall y \in L
$$
容易验证$\mathrm{ad}_x$是$L$到自身的线性变换,称为$x$的伴随表示。

李代数$L$的子空间$I$称为$L$的理想,如果$\forall x \in L, y \in I$,都有$[x,y] \in I$。

#### 2.2.2 单李代数与半单李代数
如果李代数$L$不含非平凡理想,则称$L$为单李代数。半单李代数是指不含非平凡可解理想的李代数。

#### 2.2.3 Cartan子代数与根空间分解
设$L$为有限维李代数,如果$L$的子代数$H$满足:

1. $H$由半单元素生成($\forall h \in H, \mathrm{ad}_h$可对角化)
2. $H$是极大交换子代数($H$不包含于更大的交换子代数中)

则称$H$为$L$的Cartan子代数。可以证明,在复数域上每个有限维李代数都有Cartan子代数。

固定$L$的一个Cartan子代数$H$,对任意线性泛函$\alpha \in H^*$,定义根空间
$$
L_{\alpha} = \{x \in L | [h,x]=\alpha(h)x, \forall h \in H\}
$$
如果$L_{\alpha} \neq 0$,则称$\alpha$为$L$关于$H$的根(root)。可以证明,除了$\alpha=0$的情形,其余根空间的维数都等于1。

## 3.核心算法原理具体操作步骤
### 3.1 计算李代数的结构常数
设$L$为$n$维李代数,取一组基$\{x_1,\cdots,x_n\}$,则李括号完全由结构常数$c_{ij}^k$决定:
$$
[x_i,x_j] = \sum_{k=1}^n c_{ij}^k x_k
$$
其中结构常数满足反对称性和雅可比等式:
$$
c_{ij}^k = -c_{ji}^k, \quad \sum_{l=1}^n (c_{ij}^l c_{lk}^m + c_{ki}^l c_{lj}^m + c_{jk}^l c_{li}^m) = 0
$$

给定一个具体的李代数,计算结构常数的步骤如下:

1. 选取一组方便计算的基$\{x_1,\cdots,x_n\}$。
2. 对所有的基元素两两计算李括号$[x_i,x_j]$,用基$\{x_1,\cdots,x_n\}$表示结果。
3. 读出结构常数$c_{ij}^k$的值。

### 3.2 求解李代数的根系与权系
设$L$为复半单李代数,$H$为其Cartan子代数,则可以进行如下的计算:

1. 求出$L$关于$H$的所有非零根$\alpha_1,\cdots,\alpha_l$。
2. 在根空间中选取适当的基,使得李括号满足:
$$
[e_{\alpha},e_{\beta}] = \begin{cases}
h_{\alpha}, & \alpha+\beta=0 \\
N_{\alpha,\beta}e_{\alpha+\beta}, & \alpha+\beta \text{为根} \\
0, & \text{其他情形}
\end{cases}
$$
其中$N_{\alpha,\beta}$为某些整数。
3. 定义素根(simple root)为不能表示成其他根的正线性组合的正根。求出全体素根$\alpha_1,\cdots,\alpha_r$。
4. 定义权(weight)为全体素根的线性组合$\lambda = \sum m_i \alpha_i$,其中系数$m_i$为非负整数。求出$L$的全体权。

## 4.数学模型和公式详细讲解举例说明
### 4.1 $\mathfrak{sl}(2,\mathbb{C})$的根系与权系
考虑$\mathfrak{sl}(2,\mathbb{C})$的一组基:
$$
h=\begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}, \quad
e=\begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}, \quad
f=\begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}
$$
其中$\{h\}$生成$\mathfrak{sl}(2,\mathbb{C})$的Cartan子代数$H$。通过计算李括号得到:
$$
[h,e]=2e, \quad [h,f]=-2f, \quad [e,f]=h
$$
因此$\mathfrak{sl}(2,\mathbb{C})$有两个非零根$\pm \alpha$,且$\alpha$即为素根。$\mathfrak{sl}(2,\mathbb{C})$的权全体为:
$$
\Lambda = \{m\alpha | m \in \mathbb{N}\}
$$

### 4.2 Verma模与BGG范畴
设$L$为复半单李代数,$H$为其Cartan子代数,$\lambda \in H^*$为权。定义Verma模$M(\lambda)$为由一个最高权向量$v_{\lambda}$生成的$L$模,满足:
$$
hv_{\lambda} = \lambda(h)v_{\lambda}, \quad ev_{\lambda}=0, \quad \forall h \in H, e \in L_{\alpha}, \alpha \text{为正根}
$$
可以证明,Verma模$M(\lambda)$有如下的性质:

1. $M(\lambda)$有唯一的极大真子模$N(\lambda)$。
2. 商模$L(\lambda)=M(\lambda)/N(\lambda)$是不可约的最高权模,称为不可约Verma模。
3. 任意最高权$L$模同构于某个$L(\lambda)$的直和。

BGG范畴$\mathcal{O}$是由全体最高权$L$模构成的满子范畴。它在表示论中有重要的地位,与投射模、Verma模等概念密切相关。

## 5.项目实践：代码实例和详细解释说明
下面我们用Python实现计算$\mathfrak{sl}(2,\mathbb{C})$的结构常数和Casimir算子:

```python
import sympy as sp

# 定义李代数的生成元
h, e, f = sp.symbols('h e f')

# 定义李括号
def bracket(x, y):
    if x == h and y == e:
        return 2*e
    elif x == h and y == f:
        return -2*f
    elif x == e and y == f:
        return h
    elif x == y:
        return 0
    else:
        return -bracket(y, x)

# 计算结构常数
basis = [h, e, f]
for i in range(3):
    for j in range(3):
        print(f'[{basis[i]},{basis[j]}] = {bracket(basis[i], basis[j])}')

# 计算Casimir算子        
C = h**2 / 2 + e*f + f*e
print(f'Casimir operator: {sp.expand(C)}')
```

输出结果为:
```
[h,h] = 0
[h,e] = 2*e
[h,f] = -2*f
[e,h] = -2*e
[e,e] = 0
[e,f] = h
[f,h] = 2*f
[f,e] = -h
[f,f] = 0
Casimir operator: h**2/2 + e*f + f*e
```

可以看到,代码正确计算出了$\mathfrak{sl}(2,\mathbb{C})$的结构常数和Casimir算子。其中Casimir算子是李代数的中心元素,在表示论中有重要作用。

## 6.实际应用场景
李代数在现代数学和物理学中有着极其广泛的应用,下面列举几个重要的实例:

1. 量子力学:Heisenberg代数、角动量代数等都是李代数,它们刻画了量子系统的对称性。

2. 粒子物理:基本粒子的内禀对称性由规范群描述,它们的无穷小生成元构成李代数,如SU(3)色动力学。

3. 流形上的几何结构:向量场、微分形式等构成无穷维李代数,它们刻画了流形的几何性质。

4. 可积系统:许多可积系统(如KdV方程)的无穷小对称性构成李代数,它们与孤子解、Lax对等概念密切相关。

5. 组合数学:Young图的对称性由对称群的表示刻画,而对称群的泛函就是一类特殊的李代数。

总之,只要涉及连续对称性,李代数就会自然地出现并发挥重要作用。
