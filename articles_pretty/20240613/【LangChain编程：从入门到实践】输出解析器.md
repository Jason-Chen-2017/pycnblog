```
# 定义解析器
def parser(output):
    # 解析输出
    data = {}
    data['name'] = output.split(':')[0]
    data['age'] = int(output.split(':')[1])
    return data

# 定义格式化器
def formatter(data):
    # 将中间格式转换成人类可读的形式
    return 'Name: {}\nAge: {}'.format(data['name'], data['age'])

# 读取程序的输出
output = read()

# 解析输出
data = parser(output)

# 将中间格式转换成人类可读的形式
formatted_output = formatter(data)

# 输出结果
print(formatted_output)
```

在上面的代码中，我们定义了一个解析器和一个格式化器。解析器负责将程序的输出解析成一个字典，格式化器负责将字典转换成人类可读的形式。最后，我们读取程序的输出，使用解析器将其解析成一个字典，然后使用格式化器将字典转换成人类可读的形式，最终输出结果。
