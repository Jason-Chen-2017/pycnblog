### 5.2 问题分析
要计算对偶空间V*的一组基{f1*,f2*,…,fn*}，需要先求出线性函数{f1,f2,…,fn}在基{e1,e2,…,en}下的矩阵表示A。然后，通过求A的逆矩阵A-1，得到{f1*,f2*,…,fn*}在基{e1,e2,…,en}下的矩阵表示B。最后，将B转置得到{f1*,f2*,…,fn*}在基{e1,e2,…,en}下的矩阵表示{f1*,f2*,…,fn*}。

### 5.3 问题解答

2. 在compute_dual_space函数中，首先定义一个变量A，用于存储线性函数{f1,f2,…,fn}在基{e1,e2,…,en}下的矩阵表示。
3. 然后，通过计算A的逆矩阵A-1，得到{f1*,f2*,…,fn*}在基{e1,e2,…,en}下的矩阵表示B。
4. 最后，将B转置得到{f1*,f2*,…,fn*}在基{e1,e2,…,en}下的矩阵表示{f1*,f2*,…,fn*}。

import numpy as np

def compute_dual_space(e1, e2,..., en, f1, f2,..., fn):
    # 定义一个变量A，用于存储线性函数{f1,f2,…,fn}在基{e1,e2,…,en}下的矩阵表示
    A = np.array([[f1(e1), f1(e2),..., f1(en)],
                  [f2(e1), f2(e2),..., f2(en)],
                 ...,
                  [fn(e1), fn(e2),..., fn(en)]])

    # 使用numpy库来计算矩阵的逆
    B = np.linalg.inv(A)

    # 将B转置得到{f1*,f2*,…,fn*}在基{e1,e2,…,en}下的矩阵表示{f1*,f2*,…,fn*}
    f1_star, f2_star,..., fn_star = np.transpose(B)

    return f1_star, f2_star,..., fn_star
