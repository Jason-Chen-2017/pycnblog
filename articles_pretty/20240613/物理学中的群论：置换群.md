### 群的运算

在置换群中，群的运算是置换的复合。例如，对于两个置换P和Q，它们的复合可以表示为PQ。具体地，对于一个元素x，它在PQ下的结果可以表示为P(Q(x))。

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用Python实现置换群的例子：

```python
class Permutation:
    def __init__(self, perm):
        self.perm = perm

    def __call__(self, x):
        return self.perm[x]

    def __mul__(self, other):
        return Permutation([self.perm[other.perm[i]] for i in range(len(self.perm))])

    def __str__(self):
        return str(self.perm)

p = Permutation([1, 2, 0])
q = Permutation([2, 0, 1])
r = p * q
print(r(0)) # 输出2
```

在这个例子中，我们定义了一个Permutation类来表示置换。其中，__init__方法用于初始化置换，__call__方法用于返回置换后的元素，__mul__方法用于实现置换的复合，__str__方法用于输出置换的字符串表示。
