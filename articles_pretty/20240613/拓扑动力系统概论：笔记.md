## 4.数学模型和公式详细讲解举例说明

拓扑动力系统的数学模型可以用微分方程、差分方程、格点模型等形式表示。例如 Lorenz 方程是一个经典的拓扑动力系统模型：

$$
\begin{aligned}
\frac{dx}{dt} &= \sigma(y-x) \\
\frac{dy}{dt} &= x(\rho-z)-y \\
\frac{dz}{dt} &= xy-\beta z
\end{aligned}
$$

其中 $x,y,z$ 分别表示系统的三个状态变量，$\sigma,\rho,\beta$ 是系统的参数。

```python
import numpy as np
from scipy.integrate import odeint
import tadasets
import gudhi

# 定义 Lorenz 方程
def lorenz(xyz, t, sigma, rho, beta):
    x, y, z = xyz
    dxdt = sigma * (y - x)
    dydt = x * (rho - z) - y
    dzdt = x * y - beta * z
    return [dxdt, dydt, dzdt]

# 数值计算
sigma, rho, beta = 10, 28, 8/3
t = np.linspace(0, 100, 10000)
xyz0 = [1, 1, 1]
xyz = odeint(lorenz, xyz0, t, args=(sigma, rho, beta))

# 拓扑分析
rips_complex = gudhi.RipsComplex(points=xyz)
simplex_tree = rips_complex.create_simplex_tree(max_dimension=2)
diag = simplex_tree.persistence()

# 可视化
gudhi.plot_persistence_diagram(diag)
```
