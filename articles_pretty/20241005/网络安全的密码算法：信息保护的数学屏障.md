### 1.4 术语表
#### 1.4.1 核心术语定义

- **密钥**：用于加密和解密信息的密钥。
- **对称加密**：加密和解密使用相同密钥的加密算法。
- **非对称加密**：加密和解密使用不同密钥的加密算法。

- **公钥**：非对称加密算法中用于加密的密钥。
- **私钥**：非对称加密算法中用于解密的密钥。

## 2. 核心概念与联系

### 2.1 对称加密
对称加密算法使用相同的密钥进行加密和解密。常见的对称加密算法包括AES、DES等。

### 2.2 非对称加密
非对称加密算法使用一对密钥，即公钥和私钥。公钥用于加密，私钥用于解密。常见的非对称加密算法包括RSA、ECC等。

### 2.3 哈希函数
哈希函数将任意长度的数据转换为固定长度的摘要。常见的哈希函数包括SHA-256、MD5等。

### 2.4 核心概念联系
- **对称加密**和**非对称加密**：对称加密适用于大量数据的快速加密，而非对称加密适用于密钥的分发和安全通信。
- **哈希函数**：用于生成固定长度的摘要，常用于验证数据的完整性和存储密码。

## 3. 核心算法原理 & 具体操作步骤

### 3.1 对称加密算法原理
对称加密算法的基本原理是使用相同的密钥进行加密和解密。常见的对称加密算法包括AES和DES。

#### 3.1.1 AES算法原理
AES算法是一种分组密码，使用128位、192位或256位密钥进行加密。其基本操作包括：
- **轮函数**：包括字节替换、行移位、列混淆和轮密钥加。
- **密钥扩展**：将密钥扩展为一系列轮密钥。

#### 3.1.2 AES算法伪代码
```plaintext
function AES_encrypt(plaintext, key):
    // 密钥扩展
    key_schedule = key_expansion(key)
    // 初始置换
    state = initial_permutation(plaintext)
    // 10/12/14轮迭代
    for round in 1 to 10/12/14:
        state = round_function(state, key_schedule[round])
    // 最终置换
    ciphertext = final_permutation(state)
    return ciphertext

function AES_decrypt(ciphertext, key):
    // 密钥扩展
    key_schedule = key_expansion(key)
    // 初始置换
    state = initial_permutation(ciphertext)
    // 10/12/14轮逆迭代
    for round in 10/12/14 to 1:
        state = inverse_round_function(state, key_schedule[round])
    // 最终置换
    plaintext = final_permutation(state)
    return plaintext
```

### 3.2 非对称加密算法原理
非对称加密算法使用一对密钥，即公钥和私钥。公钥用于加密，私钥用于解密。常见的非对称加密算法包括RSA和ECC。

#### 3.2.1 RSA算法原理
RSA算法基于大数分解的困难性。其基本原理是：
- **密钥生成**：选择两个大素数p和q，计算n = p * q，选择一个与φ(n)互质的e，计算d = e^(-1) mod φ(n)。
- **加密**：使用公钥(e, n)进行加密。
- **解密**：使用私钥(d, n)进行解密。

#### 3.2.2 RSA算法伪代码
```plaintext
function RSA_encrypt(m, e, n):
    return (m^e) mod n

function RSA_decrypt(c, d, n):
    return (c^d) mod n
```

### 3.3 哈希函数原理
哈希函数将任意长度的数据转换为固定长度的摘要。常见的哈希函数包括SHA-256和MD5。

#### 3.3.1 SHA-256算法原理
SHA-256算法是一种安全哈希算法，其基本原理是：
- **初始化**：使用固定初始值。
- **分组处理**：将输入数据分组处理。
- **压缩函数**：使用压缩函数处理每组数据。
- **输出**：生成固定长度的摘要。

#### 3.3.2 SHA-256算法伪代码
```plaintext
function SHA256(message):
    // 初始化
    h0 = 0x6a09e667

    // 输出
    return (h0, h1, h2, h3, h4, h5, h6, h7)
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 对称加密算法数学模型
对称加密算法的数学模型可以表示为：
\[ E_k(P) = C \]
\[ D_k(C) = P \]
其中，\( E_k \)表示加密函数，\( D_k \)表示解密函数，\( P \)表示明文，\( C \)表示密文，\( k \)表示密钥。

### 4.2 非对称加密算法数学模型
非对称加密算法的数学模型可以表示为：
\[ C = E_{pub}(P) \]
\[ P = D_{priv}(C) \]
其中，\( E_{pub} \)表示公钥加密函数，\( D_{priv} \)表示私钥解密函数，\( P \)表示明文，\( C \)表示密文。

### 4.3 哈希函数数学模型

    cipher = AES.new(key, AES.MODE_EAX)

    return ciphertext, tag, cipher.nonce

    plaintext = cipher.decrypt_and_verify(ciphertext, tag)
