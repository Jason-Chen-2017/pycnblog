$$
H(X) = -\sum_{i=1}^{n} p(x_i) \log p(x_i)

其中，$X$ 是一个随机变量，$p(x_i)$ 是随机变量 $X$ 取值 $x_i$ 的概率。

#### 3.2.3 量子纠缠

假设我们有一个随机变量 $X$，它有三个可能的取值 $x_1$、$x_2$ 和 $x_3$，它们的概率分别为 $p(x_1) = 0.5$、$p(x_2) = 0.3$ 和 $p(x_3) = 0.2$。我们可以计算信息熵为：
$$

```python
import numpy as np

def entropy(p):
    return -np.sum(p * np.log2(p))

# 示例：计算随机变量X的信息熵
p = np.array([0.5, 0.3, 0.2])
print("信息熵：", entropy(p))
```

#### 5.2.2 量子态
```python
import numpy as np

def quantum_state(theta1, theta2):
    psi = (1 / np.sqrt(2)) * (np.exp(1j * theta1) + np.exp(1j * theta2))
    return psi

# 示例：计算在位置x处找到粒子的概率密度
theta1 = np.pi / 4
theta2 = np.pi / 3
psi = quantum_state(theta1, theta2)
probability_density = np.abs(psi) ** 2
print("概率密度：", probability_density)
```
