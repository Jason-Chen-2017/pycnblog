## 3. 核心算法原理 & 具体操作步骤
### 3.1 核心算法原理
我们将通过一个具体的数学问题来探讨数学证明的可读性和可理解性。假设我们要证明一个简单的数学命题：**“对于任意正整数n，n^2 + n + 41是一个素数。”**

### 3.2 具体操作步骤
#### 3.2.1 形式化证明
形式化证明通常使用严格的符号和规则来表达。以下是一个形式化的证明示例：

```latex
\begin{proof}
    \text{假设 } n \text{ 是任意正整数。}
    \text{我们需要证明 } n^2 + n + 41 \text{ 是一个素数。}
    \text{首先，我们考虑 } n = 40 \text{ 的情况：}
    \text{此时，} n^2 + n + 41 = 40^2 + 40 + 41 = 1681 = 41 \times 41.
    \text{因此，当 } n = 40 \text{ 时，} n^2 + n + 41 \text{ 不是一个素数。}
    \text{因此，原命题不成立。}
\end{proof}
```

自然语言证明则更易于理解和阅读。以下是一个自然语言的证明示例：

### 5.2 源代码详细实现和代码解读
以下是一个使用Python实现的数学证明代码示例：

```python
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def check_expression(n):
    expression_value = n**2 + n + 41
    if is_prime(expression_value):
        return True
    return False

def main():
    for n in range(1, 100):
        if not check_expression(n):
            print(f"n = {n} is a counterexample: {n**2 + n + 41} is not a prime number.")
            break
    else:
        print("No counterexample found in the range 1 to 100.")

if __name__ == "__main__":
    main()
```

1. **is_prime函数**：判断一个数是否为素数。
2. **check_expression函数**：检查给定的表达式是否为素数。
3. **main函数**：遍历1到100之间的所有正整数，检查表达式是否为素数。如果找到一个反例，则输出该反例。
