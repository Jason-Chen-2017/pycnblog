## 4. 具体最佳实践：代码实例和详细解释说明

以下是一个基于知识图谱的美妆品牌管理系统的代码示例:

```python
import networkx as nx
import matplotlib.pyplot as plt

# 构建知识图谱
G = nx.Graph()
G.add_node("brand1", type="brand", category="skincare")
G.add_node("product1", type="product", brand="brand1", category="moisturizer")
G.add_node("ingredient1", type="ingredient", products=["product1"])
G.add_node("consumer1", type="consumer", age=25, gender="female")
G.add_edge("brand1", "product1", relation="produces")
G.add_edge("product1", "ingredient1", relation="contains")
G.add_edge("consumer1", "product1", relation="uses")

# 可视化知识图谱
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True)
plt.show()

# 基于知识图谱的消费者画像
consumer_profile = {
    "age": G.nodes["consumer1"]["age"],
    "gender": G.nodes["consumer1"]["gender"],
    "preferred_brands": [n for n, d in G.nodes(data=True) if d["type"] == "brand" and "consumer1" in [c for c,r,p in G.edges(n, data="relation") if r == "uses"]],
    "preferred_products": [n for n, d in G.nodes(data=True) if d["type"] == "product" and "consumer1" in [c for c,r,p in G.edges(n, data="relation") if r == "uses"]]
}
print(consumer_profile)
```

该示例展示了如何使用NetworkX库构建美妆类目知识图谱,并基于图谱数据生成消费者画像。通过可视化知识图谱,企业可以直观地了解品牌、产品、消费者之间的关系。基于消费者画像,企业可以制定更加精准的营销策略。
