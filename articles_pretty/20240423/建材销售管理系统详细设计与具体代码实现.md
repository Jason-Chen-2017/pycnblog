## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的建材销售管理系统的代码实例。这个实例使用Python语言和SQLite数据库。

```python
import sqlite3

# 连接到SQLite数据库
conn = sqlite3.connect('sales.db')

# 创建一个游标对象
cursor = conn.cursor()

# 创建一个表
cursor.execute('''
    CREATE TABLE sales (
        product TEXT,
        quantity INTEGER,
        price REAL
    )
''')

# 插入一些数据
cursor.execute('''
    INSERT INTO sales VALUES ('Brick', 100, 0.5)
''')
cursor.execute('''
    INSERT INTO sales VALUES ('Cement', 200, 1.0)
''')

# 提交事务
conn.commit()

# 查询数据
cursor.execute('''
    SELECT * FROM sales
''')
rows = cursor.fetchall()
for row in rows:
    print(row)

# 关闭连接
conn.close()
```

以上代码首先连接到一个SQLite数据库，然后创建一个表，插入一些数据，查询数据，最后关闭连接。
