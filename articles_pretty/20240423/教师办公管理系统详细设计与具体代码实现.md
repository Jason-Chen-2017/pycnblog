## 4. 数学模型和公式详细讲解举例说明

在本系统中，我们主要使用了基于散列的密码存储方法。散列函数$h$的基本形式为：

$$
h : \{0,1\}^* \rightarrow \{0,1\}^n
$$

这里的$\{0,1\}^*$表示任意长度的二进制字符串，$\{0,1\}^n$表示长度为$n$的二进制字符串。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 用户登录验证代码实例

下面是一个简单的用户登录验证代码实例：

```python
def check_password(hashed_password, user_password):
    return hashed_password == hashlib.md5(user_password.encode()).hexdigest()
```

这段代码中，`hashlib.md5(user_password.encode()).hexdigest()`将用户输入的密码进行散列处理，然后与存储的散列密码进行比较，如果二者相等，则验证通过。

### 5.2 数据查询代码实例

下面是一个简单的数据查询代码实例：

```python
def query_data(sql_query):
    db = MySQLdb.connect("localhost", "testuser", "test123", "TESTDB")
    cursor = db.cursor()
    cursor.execute(sql_query)
    results = cursor.fetchall()
    return results
```

这段代码中，我们首先建立了与数据库的连接，然后通过`cursor.execute(sql_query)`执行SQL查询，最后通过`cursor.fetchall()`获取查询的结果。
