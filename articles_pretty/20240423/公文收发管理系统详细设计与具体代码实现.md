class Document(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()

在这个模型中，`title` 字段表示公文的标题，`content` 字段表示公文的内容，`status` 字段表示公文的状态。

### 4.2 实现公文的收发功能

接下来，我们需要实现公文的收发功能。在Django框架中，我们可以通过定义一个视图函数来实现这个功能。以下是一个简单的公文收发视图函数的定义：

```python
from django.http import HttpResponse
from .models import Document

def send_document(request):
    if request.method == 'POST':
        title = request.POST['title']
        content = request.POST['content']
        document = Document(title=title, content=content, status='sent')
        document.save()
        return HttpResponse('Document sent successfully.')
    else:
        return HttpResponse('Invalid request method.')
```

在这个视图函数中，我们首先检查请求的方法是否为POST。如果是，我们从请求中获取公文的标题和内容，然后创建一个新的公文对象，并将其状态设置为'sent'，最后保存这个公文对象到数据库。如果请求的方法不是POST，我们返回一个错误消息。
