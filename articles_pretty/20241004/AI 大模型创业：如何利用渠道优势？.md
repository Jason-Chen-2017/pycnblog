## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 模型训练过程

#### 4.1.1 损失函数

$$
\mathcal{L}(\mathbf{y}, \hat{\mathbf{y}}) = -\sum_{i=1}^{n} y_i \log(\hat{y}_i)
$$

- **损失函数**：定义损失函数，衡量模型预测与真实值之间的差异。
- **优化算法**：使用梯度下降等优化算法更新模型参数。

### 4.2 模型优化过程

#### 4.2.1 正则化

$$
\mathcal{L}(\mathbf{y}, \hat{\mathbf{y}}) = -\sum_{i=1}^{n} y_i \log(\hat{y}_i) + \lambda \sum_{j=1}^{m} w_j^2
$$

- **正则化项**：通过正则化项防止模型过拟合。
- **超参数调整**：调整正则化参数 $\lambda$，找到最优值。

#### 5.2.2 模型训练

```python
import torch
import torch.nn as nn
import torch.optim as optim

# 定义模型
class MyModel(nn.Module):
    def __init__(self):
        super(MyModel, self).__init__()
        self.fc1 = nn.Linear(10, 64)
        self.fc2 = nn.Linear(64, 32)
        self.fc3 = nn.Linear(32, 1)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = torch.relu(self.fc2(x))
        x = torch.sigmoid(self.fc3(x))
        return x

# 初始化模型
model = MyModel()

# 定义损失函数和优化器
criterion = nn.BCELoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 训练模型
for epoch in range(100):
    optimizer.zero_grad()
    outputs = model(X_train)
    loss = criterion(outputs, y_train)
    loss.backward()
    optimizer.step()
    print(f'Epoch {epoch+1}, Loss: {loss.item()}')
```
