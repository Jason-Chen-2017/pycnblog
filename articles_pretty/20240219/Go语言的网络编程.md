### 4.1 TCP编程

在Go语言中，我们可以使用`net`包来进行TCP编程。以下是一个简单的TCP服务器的示例：

```go
package main

import (
	"net"
	"log"
	"io"
)

func main() {
	listener, err := net.Listen("tcp", "localhost:8000")
	if err != nil {
		log.Fatal(err)
	}

	for {
		conn, err := listener.Accept()
		if err != nil {
			log.Print(err)
			continue
		}
		go handleConn(conn)
	}
}

func handleConn(c net.Conn) {
	defer c.Close()
	io.Copy(c, c)
}
```

在这个示例中，我们首先使用`net.Listen`函数来监听一个TCP端口。然后，我们在一个无限循环中接受新的连接，并为每个连接创建一个新的goroutine来处理。在`handleConn`函数中，我们使用`io.Copy`函数来将接收到的数据回写到连接中，实现了一个简单的回显服务器。

### 4.2 HTTP编程

在Go语言中，我们可以使用`net/http`包来进行HTTP编程。以下是一个简单的HTTP服务器的示例：

```go
package main

import (
	"net/http"
	"fmt"
)

func main() {
	http.HandleFunc("/", handler)
	http.ListenAndServe("localhost:8000", nil)
}

func handler(w http.ResponseWriter, r *http.Request) {
	fmt.Fprintf(w, "URL.Path = %q\n", r.URL.Path)
}
```

在这个示例中，我们首先使用`http.HandleFunc`函数来注册一个处理函数。然后，我们使用`http.ListenAndServe`函数来启动HTTP服务器。在`handler`函数中，我们将请求的URL路径写回到响应中。
