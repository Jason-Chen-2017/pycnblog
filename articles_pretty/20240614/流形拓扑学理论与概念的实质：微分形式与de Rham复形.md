```python
from sympy import symbols, diff, Matrix, simplify

# 定义变量和函数
x, y = symbols('x y')
f = x**2 + y**2

# 计算微分形式
dfdx = diff(f, x)
dfdy = diff(f, y)
omega = Matrix([dfdx, dfdy])

# 计算微分算子
domega = Matrix([[diff(dfdx, y)], [diff(dfdy, x)]])
domega = simplify(domega)

# 输出结果
print('微分形式：', omega)
print('微分算子：', domega)
```

在这个代码实例中，我们首先定义了一个二维流形上的函数f，然后计算了它的微分形式和微分算子。最后，我们输出了计算结果。
