## 4. 具体最佳实践：代码实例和详细解释说明
下面我们以自然语言处理在哲学文献分析中的应用为例,给出具体的代码实现和说明:

```python
import spacy
from collections import Counter

# 加载英语语言模型
nlp = spacy.load("en_core_web_sm")

# 读取哲学文献
with open("philosophical_text.txt", "r") as f:
    text = f.read()

# 文本预处理
doc = nlp(text)
tokens = [token.lemma_ for token in doc if not token.is_stop and not token.is_punct]

# 词频统计
word_counts = Counter(tokens)
top_words = word_counts.most_common(20)

# 关键词提取
keywords = [word for word, count in top_words if count > 10]

# 输出结果
print("Top 20 words:")
for word, count in top_words:
    print(f"{word}: {count}")

print("\nKeywords:")
print(", ".join(keywords))
```

该代码首先使用spaCy加载英语语言模型,然后读取哲学文献文本,进行预处理(去除停用词和标点符号)。接下来统计词频,输出出现频率最高的20个词,并提取出频率大于10的关键词。这些结果可以帮助研究者快速了解文献的主要内容和关键概念。
