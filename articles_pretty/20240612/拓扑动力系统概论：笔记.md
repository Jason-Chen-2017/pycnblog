# 拓扑动力系统概论：笔记

## 1.背景介绍

### 1.1 什么是拓扑动力系统

拓扑动力系统(Topological Dynamical System, TDS)是一个研究连续变换在拓扑空间上动力学行为的数学理论。它将拓扑学和动力系统理论相结合,是一门跨学科的数学分支。拓扑动力系统的研究对象是一个拓扑空间X和一个从X到自身的连续变换f,记作(X, f)。

拓扑动力系统的主要研究内容包括:

- 轨道的性质和分类
- 不变集和极限集的结构
- 周期性和混沌性
- 熵理论和遗传性

### 1.2 拓扑动力系统的应用

拓扑动力系统理论不仅在数学领域有重要意义,而且在物理、生物、经济等许多领域都有广泛的应用。例如:

- 天体运动
- 流体力学
- 神经网络
- 基因调控网络
- 经济周期
- 密码学

## 2.核心概念与联系

### 2.1 拓扑空间

拓扑空间是拓扑动力系统的基础。一个拓扑空间(X, τ)由一个非空集合X和X上的一个拓扑τ(开集组)构成。拓扑τ必须满足以下条件:

1. 空集∅和X本身都属于τ。
2. τ中任意多个开集的并集仍属于τ。
3. τ中任意有限多个开集的交集仍属于τ。

常见的拓扑空间有欧几里得空间、离散拓扑空间、有限补拓扑空间等。

### 2.2 连续变换

连续变换是拓扑动力系统的核心。设(X, τ)是一个拓扑空间,f: X → X是一个自映射,如果对于X中任意开集U,f^-1(U)也是X中的开集,那么我们称f是连续的。

连续变换具有以下性质:

- 保持拓扑不变性
- 可逆性
- 保持极限的性质

### 2.3 轨道和不变集

给定拓扑动力系统(X, f),对于X中任意点x,我们定义x的正向轨道为:

```
O^+(x) = {f^n(x) | n ≥ 0}
```

其中f^n表示f的n次迭代。

负向轨道和全轨道分别定义为:

```
O^-(x) = {f^-n(x) | n ≥ 0}
O(x) = O^+(x) ∪ O^-(x)
```

不变集是指在变换f作用下保持不变的子集,即对于X的子集A,如果f(A) = A,那么A就是(X, f)的不变集。

### 2.4 周期点和周期轨道

如果存在正整数p,使得f^p(x) = x,那么我们称x为f的周期点,最小的这样的p称为x的周期。周期点x的全轨道O(x)称为周期轨道。

### 2.5 极限集

对于x ∈ X,我们定义x的ω-极限集为:

```
ω(x) = ∩_{n≥0} 闭包(O^+(f^n(x)))
```

ω-极限集描述了x正向轨道在"无穷远"处的渐近行为。类似地,我们可以定义α-极限集来描述负向轨道的渐近行为。

### 2.6 混沌性

混沌是动力系统中的一种行为,具有以下特征:

- 对初值的高度敏感依赖性
- 轨道的不可预测性
- 密集的周期轨道

混沌现象在许多实际系统中都可以观察到,如天气、流体、人口动态等。

## 3.核心算法原理具体操作步骤

拓扑动力系统的核心算法主要包括以下几个方面:

### 3.1 轨道分类

对于给定的拓扑动力系统(X, f),我们可以将X中的点按照其轨道的性质进行分类:

1. 周期轨道
2. 遗传轨道(密度轨道)
3. 漂移点(无遗传性)
4. 奇异遍历点(Poisson稳定)

这种分类可以帮助我们深入理解系统的动力学行为。

算法步骤:

1. 计算每个点的正向轨道O^+(x)
2. 检查O^+(x)是否存在周期性
3. 如果存在周期,则将x划分为周期轨道
4. 否则,计算ω(x)的闭包
5. 如果闭包(ω(x)) = X,则x为遗传轨道
6. 如果闭包(ω(x))是一个固定点,则x为漂移点
7. 如果闭包(ω(x))是一个有限集,且满足Poisson稳定条件,则x为奇异遍历点

### 3.2 熵计算

熵是衡量动力系统随机性和混沌程度的一个重要指标。对于拓扑动力系统(X, f),我们定义其熵为:

$$
h(f) = \sup_{U} \lim_{n \to \infty} \frac{1}{n} \log N(U, f^n)
$$

其中U是X的一个开覆盖,N(U, f^n)表示开覆盖U在f^n作用下的分拆数。

算法步骤:

1. 给定开覆盖U
2. 计算U在f^n作用下的分拆数N(U, f^n)
3. 计算极限lim(1/n)log N(U, f^n)
4. 取U的上确界得到h(f)

### 3.3 不变集计算

对于拓扑动力系统(X, f),我们需要计算其不变集的结构。常见的不变集包括:

- 固定点集
- 周期轨道
- 链环
- Solenoid
- ...

算法步骤:

1. 计算f的所有固定点
2. 计算f的所有周期轨道
3. 对于每个周期轨道,计算其不变集(链环或Solenoid)
4. 合并所有不变集

这个算法的复杂度很高,因为需要枚举所有可能的周期轨道。在实践中,我们通常使用数值方法来近似计算不变集。

### 3.4 Conley指数计算

Conley指数是研究动力系统中不变集的一个重要的代数拓扑不变量。对于拓扑动力系统(X, f)的一个isolated不变集S,我们定义其Conley指数为:

$$
CH_*(S) = H_*(N, L)
$$

其中N是S的isolating neighborhood,L是N中f的不动点集。

算法步骤:

1. 找到S的isolating neighborhood N
2. 计算N中f的不动点集L
3. 计算链复形C_*(N, L)
4. 计算C_*(N, L)的同调群H_*(N, L)

Conley指数能够刻画不变集的代数拓扑性质,是研究动力系统的重要工具。

## 4.数学模型和公式详细讲解举例说明

在上一节中,我们已经给出了一些拓扑动力系统理论中的核心公式,如熵公式、Conley指数公式等。现在让我们通过具体例子来深入理解这些公式。

### 4.1 熵公式举例

考虑单位圆周上的旋转变换f:S^1 → S^1,其中f(x) = x + α (mod 2π),α是一个常数。我们来计算这个系统的熵h(f)。

首先,我们选取开覆盖U = {U_1, U_2},其中U_1 = (0, π),U_2 = (π, 2π)。计算U在f^n作用下的分拆数N(U, f^n):

- 当n = 1时,f(U_1) = (α, π+α),f(U_2) = (π+α, 2π+α),因此N(U, f) = 2
- 当n = 2时,f^2(U_1) = (2α, 2π),f^2(U_2) = (0, 2α),因此N(U, f^2) = 2
- 当n ≥ 3时,f^n(U_1)和f^n(U_2)将覆盖整个S^1,因此N(U, f^n) = 1

根据熵公式,我们有:

$$
h(f) = \lim_{n \to \infty} \frac{1}{n} \log N(U, f^n) = 0
$$

这说明了单位圆周上的旋转变换是一个非混沌系统,其熵为0。

### 4.2 Conley指数举例

考虑平面上的双陷阱系统:

$$
\begin{cases}
\dot{x} = x - x^3 - xy^2\\
\dot{y} = -y + xy^2
\end{cases}
$$

这个系统有两个稳定的不动点(±1, 0),以及一个鞍点(0, 0)。我们来计算(-1, 0)的Conley指数。

首先,我们可以找到一个isolating neighborhood N,例如以(-1, 0)为中心的足够小的圆盘。在N中,f的不动点集L只包含(-1, 0)这一个点。

接下来,我们计算链复形C_*(N, L)。由于L只有一个点,因此C_0(N, L) = Z,C_n(N, L) = 0 (n ≠ 0)。

最后,计算同调群:

$$
H_0(N, L) = Z, H_n(N, L) = 0 (n ≠ 0)
$$

因此,(-1, 0)的Conley指数为CH_*(-1, 0) = Z。这说明了(-1, 0)是一个稳定的不动点。

通过这些例子,我们可以更好地理解拓扑动力系统理论中的数学模型和公式。

### 5.1 计算轨道

首先,我们定义一个表示拓扑动力系统的Python类:

```python
class TDS:
    def __init__(self, X, f):
        self.X = X
        self.f = f

    def orbit(self, x, n):
        """计算点x在变换f作用下的n步正向轨道"""
        orbit = [x]
        for i in range(n):
            x = self.f(x)
            orbit.append(x)
        return orbit
```

这个类包含了拓扑空间X和变换f的定义。orbit方法可以计算给定点x的n步正向轨道。

例如,我们可以计算双陷阱系统中(0.1, 0.1)点的轨道:


### 5.2 计算熵

接下来,我们实现熵的计算算法:

```python
def entropy(tds, cover, n, n_max=10):
    """计算拓扑动力系统tds在开覆盖cover下的n步熵"""
    entropy = []
    for k in range(n, n_max+1):
        N = box_counting(tds, cover, k)
        entropy.append(np.log(N) / k)
    return np.mean(entropy)

def box_counting(tds, cover, n):
    """计算开覆盖cover在变换f作用下的n步分拆数"""
    boxes = []
    for U in cover:
        boxes.extend([tds.orbit(x, n) for x in U])
    return len(set(map(tuple, boxes)))
```

entropy函数利用box_counting方法计算给定开覆盖在变换作用下的分拆数,然后根据熵公式计算熵的近似值。

例如,我们可以计算双陷阱系统的熵:

```python

```
