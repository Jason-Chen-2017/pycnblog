## 4. 数学模型和公式详细讲解举例说明

### 4.1 TF-IDF

TF-IDF 是一种用于信息检索的常用算法，用于衡量一个词语在一个文档中的重要程度。

$$
TF-IDF(t, d) = TF(t, d) \times IDF(t)
$$

其中：

*   $TF(t, d)$ 表示词语 $t$ 在文档 $d$ 中出现的频率。
*   $IDF(t)$ 表示词语 $t$ 的逆文档频率，用于衡量词语的普遍程度。

### 4.2 词向量模型

词向量模型将词语表示为高维向量，可以用于计算词语之间的语义相似度。例如，Word2Vec 是一种常用的词向量模型。

### 4.1 使用 Python 实现简单的问答系统

```python
import nltk

# 定义知识库
knowledge_base = {
    "苹果手机": "苹果公司生产的智能手机",
    "华为手机": "华为公司生产的智能手机",
    "小米手机": "小米公司生产的智能手机",
}

# 用户输入
user_input = "我想买一部苹果手机"

# 分词
tokens = nltk.word_tokenize(user_input)

# 关键词匹配
for token in tokens:
    if token in knowledge_base:
        print(knowledge_base[token])
```
