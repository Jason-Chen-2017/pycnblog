矩阵分解的基本思想是将用户-物品评分矩阵 $R$ 分解为两个低维矩阵 $P$ 和 $Q$，其中 $P$ 表示用户的隐含特征，$Q$ 表示物品的隐含特征。分解后的矩阵满足：

$$
R \approx P^T Q
$$

其中，$P$ 是一个 $m \times k$ 的矩阵，$Q$ 是一个 $n \times k$ 的矩阵，$k$ 是隐含特征的维度。

import numpy as np

def matrix_factorization(R, k, steps=5000, alpha=0.0002, beta=0.02):
  """
  矩阵分解算法
  """
  m, n = R.shape
  P = np.random.rand(m, k)
  Q = np.random.rand(n, k)

  for step in range(steps):
    for i in range(m):
      for j in range(n):
        if R[i, j] > 0:
          eij = R[i, j] - np.dot(P[i, :], Q[j, :].T)
          for k in range(k):
            P[i, k] = P[i, k] + alpha * (2 * eij * Q[j, k] - beta * P[i, k])
            Q[j, k] = Q[j, k] + alpha * (2 * eij * P[i, k] - beta * Q[j, k])

  return P, Q
