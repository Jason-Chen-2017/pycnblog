## 5. 项目实践：代码实例和详细解释说明

以下是一个简单的用户画像构建示例代码，使用Python语言实现：

```python
import pandas as pd
from sklearn.cluster import KMeans

# 读取用户数据
data = pd.read_csv('user_data.csv')

# 特征工程
features = ['age', 'gender', 'occupation', 'income']
X = data[features]

# 聚类
kmeans = KMeans(n_clusters=3)
kmeans.fit(X)

# 构建用户画像
data['cluster'] = kmeans.labels_

# 分析不同类别用户的特征
for i in range(3):
    cluster_data = data[data['cluster'] == i]
    print(f'Cluster {i}:')
    print(cluster_data.describe())
```

**代码解释：**

1. 首先使用pandas库读取用户数据。
2. 选择用户特征，并将其存储在X变量中。
3. 使用KMeans聚类算法将用户聚集成3个类别。
4. 将聚类结果存储在data['cluster']列中。
5. 循环遍历每个类别，并打印该类别用户的特征描述。
