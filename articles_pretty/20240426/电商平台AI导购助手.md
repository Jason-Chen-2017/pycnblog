sim(u,v) = \frac{\sum_{i \in I_{uv}}(r_{ui} - \bar{r_u})(r_{vi} - \bar{r_v})}{\sqrt{\sum_{i \in I_{uv}}(r_{ui} - \bar{r_u})^2}\sqrt{\sum_{i \in I_{uv}}(r_{vi} - \bar{r_v})^2}}
$$

其中，$sim(u,v)$ 表示用户 $u$ 和用户 $v$ 的相似度，$I_{uv}$ 表示用户 $u$ 和用户 $v$ 都评价过的商品集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$\bar{r_u}$ 表示用户 $u$ 的平均评分。


### 4.2 基于物品的协同过滤

$$
sim(i,j) = \frac{\sum_{u \in U_{ij}}(r_{ui} - \bar{r_i})(r_{uj} - \bar{r_j})}{\sqrt{\sum_{u \in U_{ij}}(r_{ui} - \bar{r_i})^2}\sqrt{\sum_{u \in U_{ij}}(r_{uj} - \bar{r_j})^2}}
$$

其中，$sim(i,j)$ 表示商品 $i$ 和商品 $j$ 的相似度，$U_{ij}$ 表示同时评价过商品 $i$ 和商品 $j$ 的用户集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$\bar{r_i}$ 表示商品 $i$ 的平均评分。

import numpy as np

def user_similarity(ratings):
    # 计算用户相似度矩阵
    sim = np.dot(ratings, ratings.T)
    norms = np.array([np.sqrt(np.diagonal(sim))])
    return sim / norms / norms.T

def recommend(user_id, ratings, sim):
    # 找到与目标用户最相似的 K 个用户
    k = 5
    sim_scores = list(enumerate(sim[user_id]))
    sim_scores = sorted(sim_scores, key=lambda x: x[1], reverse=True)
    sim_scores = sim_scores[1:k+1]

    # 获取相似用户的商品评分
    neighbor_ratings = ratings[np.array([i[0] for i in sim_scores])]

    # 计算推荐商品的加权平均评分
    user_ratings = ratings[user_id]
    neighbor_sim = np.array([i[1] for i in sim_scores])
    weighted_avg = np.dot(neighbor_ratings.T, neighbor_sim) / neighbor_sim.sum()

    # 推荐评分最高的商品
    recommendations = np.argsort(weighted_avg)[::-1]
    return recommendations
