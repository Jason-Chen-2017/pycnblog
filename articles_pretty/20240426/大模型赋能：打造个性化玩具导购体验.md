### 4. 数学模型和公式详细讲解举例说明

#### 4.1 协同过滤

协同过滤是一种常用的推荐算法，它基于用户之间的相似性进行推荐。例如，如果用户 A 和用户 B 购买了相同的玩具，那么系统可以将用户 A 购买的其他玩具推荐给用户 B。

协同过滤算法的数学模型可以表示为：

$$
\hat{r}_{ui} = \frac{\sum_{v \in N(u)} sim(u,v) \cdot r_{vi}}{\sum_{v \in N(u)} sim(u,v)}
$$

其中，$\hat{r}_{ui}$ 表示用户 $u$ 对物品 $i$ 的预测评分，$N(u)$ 表示与用户 $u$ 相似的用户集合，$sim(u,v)$ 表示用户 $u$ 和用户 $v$ 之间的相似度，$r_{vi}$ 表示用户 $v$ 对物品 $i$ 的评分。

#### 4.2 基于内容的推荐

基于内容的推荐算法根据物品之间的相似性进行推荐。例如，如果玩具 A 和玩具 B 都是积木玩具，那么系统可以将玩具 A 推荐给喜欢玩积木的用户。

基于内容的推荐算法的数学模型可以表示为：

$$
sim(i,j) = \frac{\vec{v}_i \cdot \vec{v}_j}{\|\vec{v}_i\| \cdot \|\vec{v}_j\|}
$$

其中，$sim(i,j)$ 表示物品 $i$ 和物品 $j$ 之间的相似度，$\vec{v}_i$ 和 $\vec{v}_j$ 分别表示物品 $i$ 和物品 $j$ 的特征向量。

### 5. 项目实践：代码实例和详细解释说明

以下是一个基于 Python 的玩具推荐系统代码示例：

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
users = pd.read_csv('users.csv')
toys = pd.read_csv('toys.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(users.drop('user_id', axis=1))

# 计算玩具相似度矩阵
toy_similarity = cosine_similarity(toys.drop('toy_id', axis=1))

# 获取用户购买过的玩具
def get_user_toys(user_id):
    return toys[toys['user_id'] == user_id]['toy_id'].tolist()

# 获取与用户购买过的玩具相似的玩具
def get_similar_toys(user_id, top_n=10):
    user_toys = get_user_toys(user_id)
    similar_toys = []
    for toy_id in user_toys:
        similar_toys.extend(toy_similarity[toy_id].argsort()[-top_n:][::-1])
    return similar_toys

# 为用户推荐玩具
def recommend_toys(user_id, top_n=10):
    similar_users = user_similarity[user_id].argsort()[-top_n:][::-1]
    similar_toys = []
    for similar_user in similar_users:
        similar_toys.extend(get_user_toys(similar_user))
    return list(set(similar_toys))
```
