## 4. 数学模型和公式详细讲解举例说明

### 4.1 TF-IDF算法

TF-IDF算法用于计算词语在文档中的重要程度，常用于文本分类和信息检索。其公式如下：

$$
TF-IDF(t, d) = TF(t, d) \times IDF(t)
$$

其中：

* $TF(t, d)$ 表示词语 $t$ 在文档 $d$ 中出现的频率。
* $IDF(t)$ 表示词语 $t$ 的逆文档频率，用于衡量词语的普遍程度。

### 4.2 协同过滤算法

协同过滤算法是一种常用的推荐算法，其基本思想是根据用户的历史行为数据，找到与目标用户相似的用户，并推荐相似用户喜欢的物品。 

## 5. 项目实践：代码实例和详细解释说明

以下是一个基于Python的药品AI导购系统示例代码：

```python
# 导入必要的库
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv("drugs.csv")

# 文本特征提取
tfidf = TfidfVectorizer()
tfidf_matrix = tfidf.fit_transform(data["description"])

# 计算相似度矩阵
similarity_matrix = cosine_similarity(tfidf_matrix)

# 推荐药品
def recommend_drugs(drug_id, num_recommendations):
    # 获取相似药品的索引
    similar_drugs_indices = similarity_matrix[drug_id].argsort()[::-1][1:num_recommendations+1]
    # 返回相似药品的名称
    return data.iloc[similar_drugs_indices]["name"].tolist()

# 示例
drug_id = 10
recommendations = recommend_drugs(drug_id, 5)
print(recommendations)
```
