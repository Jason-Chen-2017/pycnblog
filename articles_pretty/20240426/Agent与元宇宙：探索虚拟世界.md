## 4. 数学模型和公式详细讲解举例说明

### 4.1 强化学习 Q-Learning 算法

Q-Learning 算法是一种常用的强化学习算法，用于学习 Agent 在不同状态下采取不同行动的价值。

**Q-Learning 公式：**

$$Q(s, a) \leftarrow Q(s, a) + \alpha [r + \gamma \max_{a'} Q(s', a') - Q(s, a)]$$

其中：

*   $Q(s, a)$ 表示在状态 $s$ 下采取行动 $a$ 的价值。
*   $\alpha$ 表示学习率。
*   $r$ 表示奖励。
*   $\gamma$ 表示折扣因子。
*   $s'$ 表示下一个状态。
*   $a'$ 表示下一个行动。
