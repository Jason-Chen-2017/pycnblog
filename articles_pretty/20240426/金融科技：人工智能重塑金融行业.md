## 4. 数学模型和公式详细讲解举例说明

### 4.1 线性回归

线性回归是一种用于预测连续数值型变量的机器学习算法。线性回归模型可以用以下公式表示：

```
y = w_0 + w_1 * x_1 + w_2 * x_2 + ... + w_n * x_n
```

其中，y 是预测值，x_1, x_2, ..., x_n 是输入变量，w_0, w_1, w_2, ..., w_n 是模型参数。

### 4.2 逻辑回归

逻辑回归是一种用于预测二分类变量的机器学习算法。逻辑回归模型可以用以下公式表示：

```
p = 1 / (1 + exp(-z))
```

其中，p 是预测概率，z 是线性组合。

### 4.3 决策树

决策树是一种用于分类和回归的机器学习算法。决策树模型由一系列节点和分支组成，每个节点代表一个判断条件，每个分支代表一个判断结果。
