## 2. 核心概念与联系

### 2.1 知识图谱的构成

知识图谱主要由以下三部分构成：

*   **实体（Entity）**：指现实世界中的事物，例如商品、品牌、用户等。
*   **关系（Relation）**：指实体之间的联系，例如“商品属于品牌”、“用户购买商品”等。
*   **属性（Attribute）**：指实体的特征，例如商品的价格、颜色、尺寸等。

### 2.2 知识图谱的构建

知识图谱的构建主要包括以下步骤：

1.  **数据获取**：从各种数据源中获取相关数据，例如商品信息、用户信息、交易数据等。
2.  **实体识别**：识别数据中的实体，并将其分类。
3.  **关系抽取**：识别实体之间的关系，并将其表示为三元组（实体1，关系，实体2）。
4.  **属性提取**：提取实体的属性，并将其与实体关联。
5.  **知识融合**：将来自不同数据源的知识进行融合，消除冗余和冲突。

import networkx as nx

# 创建知识图谱
graph = nx.Graph()

graph.add_edge("商品A", "商品B", label="相似")

# 获取用户购买过的商品
purchased_items = [n for n in graph.neighbors("用户A")]

# 推荐相似商品
recommended_items = []
for item in purchased_items:
    for similar_item in graph.neighbors(item):
        if similar_item not in purchased_items:
            recommended_items.append(similar_item)

# 打印推荐结果
print("推荐商品：", recommended_items)
