## 5. 项目实践：代码实例

以下是一个简单的AI导购Agent代码示例：

```python
# 导入必要的库
import nltk
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# 定义商品信息列表
products = [
    {"name": "iPhone 14", "category": "手机", "description": "苹果最新款手机"},
    {"name": "MacBook Pro", "category": "笔记本电脑", "description": "苹果高端笔记本电脑"},
    {"name": "AirPods Pro", "category": "耳机", "description": "苹果无线蓝牙耳机"}
]

# 定义用户输入
user_input = "我想买一个手机"

# 使用TF-IDF模型将文本数据转换为向量表示
vectorizer = TfidfVectorizer()
product_vectors = vectorizer.fit_transform([p["description"] for p in products])
user_vector = vectorizer.transform([user_input])

# 计算用户输入与商品描述之间的相似度
similarities = cosine_similarity(user_vector, product_vectors)

# 找到相似度最高的商品
most_similar_product_index = similarities.argmax()
most_similar_product = products[most_similar_product_index]

# 输出推荐结果
print("推荐商品：", most_similar_product["name"])
```
