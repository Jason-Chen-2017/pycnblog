## 4. 数学模型和公式详细讲解举例说明

### 4.1 TF-IDF 算法

TF-IDF 算法用于计算关键词在文档中的重要程度，常用于信息检索和文本分类。

**TF (Term Frequency):** 词频，表示关键词在文档中出现的频率。

**IDF (Inverse Document Frequency):** 逆文档频率，表示关键词在所有文档中出现的稀有程度。

**TF-IDF = TF * IDF**

TF-IDF 值越高，表示关键词在文档中越重要。

### 4.2 协同过滤算法

协同过滤算法用于推荐系统，根据相似用户的行为预测用户偏好。

**基于用户的协同过滤:**  找到与目标用户相似的用户，推荐相似用户喜欢的物品。

**基于物品的协同过滤:**  找到与目标用户喜欢的物品相似的物品，推荐给目标用户。

```python
    "This document is the second document.",

# 将文档转换为 TF-IDF 矩阵

# 打印 TF-IDF 矩阵
print(tfidf_matrix.toarray())
```
