## 4. 数学模型和公式详细讲解举例说明

### 4.1 协同过滤

协同过滤算法可以分为基于用户的协同过滤和基于物品的协同过滤。

*   **基于用户的协同过滤：** 找到与目标用户兴趣相似的用户，并将这些用户喜欢的物品推荐给目标用户。

    $$
    sim(u,v) = \frac{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)(r_{vi} - \bar{r}_v)}{\sqrt{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)^2}\sqrt{\sum_{i \in I_{uv}}(r_{vi} - \bar{r}_v)^2}}
    $$

    其中，$sim(u,v)$ 表示用户 $u$ 和用户 $v$ 之间的相似度，$I_{uv}$ 表示用户 $u$ 和用户 $v$ 都评价过的物品集合，$r_{ui}$ 表示用户 $u$ 对物品 $i$ 的评分，$\bar{r}_u$ 表示用户 $u$ 的平均评分。
*   **基于物品的协同过滤：** 找到与目标物品相似的物品，并将这些物品推荐给喜欢目标物品的用户。

    $$
    sim(i,j) = \frac{\sum_{u \in U_{ij}}(r_{ui} - \bar{r}_i)(r_{uj} - \bar{r}_j)}{\sqrt{\sum_{u \in U_{ij}}(r_{ui} - \bar{r}_i)^2}\sqrt{\sum_{u \in U_{ij}}(r_{uj} - \bar{r}_j)^2}}
    $$

    其中，$sim(i,j)$ 表示物品 $i$ 和物品 $j$ 之间的相似度，$U_{ij}$ 表示同时评价过物品 $i$ 和物品 $j$ 的用户集合，$r_{ui}$ 表示用户 $u$ 对物品 $i$ 的评分，$\bar{r}_i$ 表示物品 $i$ 的平均评分。

### 4.2 基于内容的推荐

基于内容的推荐算法利用物品的特征和用户的画像，计算物品与用户之间的相似度，并推荐相似度最高的物品给用户。

$$
sim(u,i) = \sum_{f \in F} w_f \cdot sim(u_f, i_f)
$$

其中，$sim(u,i)$ 表示用户 $u$ 和物品 $i$ 之间的相似度，$F$ 表示物品的特征集合，$w_f$ 表示特征 $f$ 的权重，$sim(u_f, i_f)$ 表示用户 $u$ 在特征 $f$ 上与物品 $i$ 的相似度。

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 实现基于用户的协同过滤算法的示例代码：

```python
import pandas as pd

def user_similarity(ratings_data):
    # 计算用户之间的相似度矩阵
    similarity_matrix = ratings_data.corr(method='pearson')
    return similarity_matrix

def recommend_items(user_id, similarity_matrix, ratings_data, k=5):
    # 找到与目标用户最相似的 k 个用户
    similar_users = similarity_matrix[user_id].sort_values(ascending=False)[1:k+1]
    # 获取相似用户评价过的物品
    similar_items = ratings_data[ratings_data.index.isin(similar_users.index)]
    # 计算每个物品的推荐分数
    item_scores = similar_items.mean(axis=0)
    # 排序并返回推荐列表
    recommendations = item_scores.sort_values(ascending=False)
    return recommendations

# 读取评分数据
ratings_data = pd.read_csv('ratings.csv', index_col='user_id')

# 计算用户相似度矩阵
similarity_matrix = user_similarity(ratings_data)

# 为用户 1 推荐物品
recommendations = recommend_items(1, similarity_matrix, ratings_data)

print(recommendations)
```
