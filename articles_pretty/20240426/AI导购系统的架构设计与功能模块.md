## 4. 数学模型和公式详细讲解举例说明

以协同过滤算法为例，其核心思想是：

*   **用户相似性**: 具有相似购买行为的用户，可能对相同的商品感兴趣。
*   **商品相似性**: 被相同用户购买的商品，可能具有相似特征。

常用的相似性度量方法包括：

*   **余弦相似度**: 计算两个向量夹角的余弦值，取值范围为 [-1, 1]，值越大表示相似度越高。

$$
sim(u, v) = \frac{\vec{u} \cdot \vec{v}}{||\vec{u}|| \cdot ||\vec{v}||}
$$

*   **皮尔逊相关系数**: 衡量两个变量之间的线性相关程度，取值范围为 [-1, 1]，值越大表示相关性越强。

$$
r_{uv} = \frac{\sum_{i=1}^{n}(u_i - \bar{u})(v_i - \bar{v})}{\sqrt{\sum_{i=1}^{n}(u_i - \bar{u})^2}\sqrt{\sum_{i=1}^{n}(v_i - \bar{v})^2}}
$$

## 5. 项目实践：代码实例和详细解释说明

以下是一个基于 Python 的协同过滤算法示例：

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载用户评分数据
ratings = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(ratings.pivot_table(index='userId', columns='movieId', values='rating'))

# 获取用户的历史评分
user_ratings = ratings[ratings['userId'] == 1]

# 找到与该用户最相似的用户
similar_users = user_similarity[1].argsort()[::-1][1:]

# 获取相似用户评分过的电影，并推荐给当前用户
recommendations = ratings[(ratings['userId'].isin(similar_users)) & (~ratings['movieId'].isin(user_ratings['movieId']))]
recommendations = recommendations.sort_values(by='rating', ascending=False)

# 输出推荐结果
print(recommendations[['movieId', 'rating']].head(10))
```
