### 6.1 项目概述：鸢尾花分类

我们将使用著名的鸢尾花数据集进行分类任务。该数据集包含三种鸢尾花品种（山鸢尾、变色鸢尾和维吉尼亚鸢尾）的萼片和花瓣长度、宽度等特征数据。我们的目标是训练一个机器学习模型，能够根据这些特征预测鸢尾花的品种。

### 6.2 代码实现步骤

#### 6.2.1 数据导入与预处理

首先，我们需要导入必要的库并加载鸢尾花数据集。

```python
import pandas as pd
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

# 加载数据集
iris = load_iris()
data = pd.DataFrame(data=iris.data, columns=iris.feature_names)
target = pd.Series(data=iris.target, name='species')

# 数据集划分
X_train, X_test, y_train, y_test = train_test_split(data, target, test_size=0.2, random_state=42)
```

#### 6.2.2 特征工程

在某些情况下，我们可能需要对特征进行处理，例如：

*   **标准化**: 将特征缩放到相同的尺度，避免某些特征因量纲差异而影响模型性能。
*   **独热编码**: 将类别型特征转换为数值型特征。
*   **特征选择**: 选择对目标变量影响较大的特征，减少模型复杂度。

在本例中，由于鸢尾花数据集的特征已经进行了标准化处理，因此我们无需进行额外的特征工程。

#### 6.2.3 模型选择与训练

根据任务类型和数据特点，选择合适的机器学习模型。在本例中，由于是分类任务，我们可以选择逻辑回归、支持向量机或决策树等模型。

```python
from sklearn.linear_model import LogisticRegression

# 创建模型实例
model = LogisticRegression()

# 模型训练
model.fit(X_train, y_train)
```

#### 6.2.4 模型评估

使用测试集评估模型的性能，例如准确率、精确率、召回率等指标。

```python
from sklearn.metrics import accuracy_score

# 模型预测
y_pred = model.predict(X_test)

# 准确率计算
accuracy = accuracy_score(y_test, y_pred)
print("Accuracy:", accuracy)
```
