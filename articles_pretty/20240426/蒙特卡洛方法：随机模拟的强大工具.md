### 2.1 随机数生成

蒙特卡洛方法的核心是随机数生成。随机数是指在一定范围内均匀分布的数字序列，它不遵循任何规律，具有不可预测性。在计算机中，通常使用伪随机数生成器来生成随机数序列。

### 2.2 概率分布

概率分布描述了随机变量取值的可能性。常见的概率分布包括均匀分布、正态分布、泊松分布等。在蒙特卡洛模拟中，我们需要根据问题的特点选择合适的概率分布来生成随机数。

### 2.3 大数定律与中心极限定理

大数定律指出，随着随机试验次数的增加，随机事件发生的频率会趋近于其概率。中心极限定理则表明，在一定条件下，大量独立随机变量的平均值会趋近于正态分布。这两个定理为蒙特卡洛方法的有效性提供了理论基础。

### 5.1 Python 代码示例：计算圆周率

```python
import random

def monte_carlo_pi(n):
  """
  使用蒙特卡洛方法计算圆周率
  """
  count = 0
  for _ in range(n):
    x = random.uniform(0, 1)
    y = random.uniform(0, 1)
    if x**2 + y**2 <= 1:
      count += 1
  return 4 * count / n

pi = monte_carlo_pi(1000000)
print("圆周率π的估计值：", pi)
```

### 5.2 代码解释

这段代码首先定义了一个 `monte_carlo_pi` 函数，它接受一个参数 `n`，表示随机投掷点的数量。函数内部使用 `random.uniform(0, 1)` 生成随机数，并判断点是否落在圆内。最后，函数返回圆周率π的估计值。
