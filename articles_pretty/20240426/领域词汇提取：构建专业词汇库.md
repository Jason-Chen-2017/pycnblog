## 2. 核心概念与联系

## 3. 核心算法原理

### 3.1 基于统计的方法

* **TF-IDF**: 通过计算词语在文档中的词频和逆文档频率，识别出文档中的关键词。
* **N-gram**: 将文本分割成连续的 n 个词语，统计 n-gram 的出现频率，识别出高频的 n-gram 作为候选词汇。

### 3.2 基于机器学习的方法

* **隐马尔可夫模型 (HMM)**: 利用词性标注信息，识别出名词短语作为候选词汇。
* **条件随机场 (CRF)**: 利用词性、词形、上下文等特征，识别出领域词汇。
* **深度学习**: 利用词向量、循环神经网络 (RNN) 等技术，自动学习文本特征，识别出领域词汇。

## 4. 数学模型和公式

### 4.1 TF-IDF

$$
TF-IDF(t,d) = TF(t,d) \times IDF(t)
$$

其中，$TF(t,d)$ 表示词语 $t$ 在文档 $d$ 中出现的频率，$IDF(t)$ 表示词语 $t$ 的逆文档频率。

### 4.2 N-gram 语言模型

$$
P(w_1, w_2, ..., w_n) = P(w_1) \times P(w_2|w_1) \times ... \times P(w_n|w_{n-1})
$$

其中，$P(w_1, w_2, ..., w_n)$ 表示 n-gram 的概率，$P(w_i|w_{i-1})$ 表示在 $w_{i-1}$ 出现的情况下，$w_i$ 出现的条件概率。

## 5. 项目实践：代码实例

### 5.1 Python 代码示例

```python
from sklearn.feature_extraction.text import TfidfVectorizer

# 创建 TF-IDF 向量化器
vectorizer = TfidfVectorizer()

# 训练模型
vectorizer.fit(corpus)

# 获取特征名称
feature_names = vectorizer.get_feature_names()

# 获取文档的 TF-IDF 特征向量
tfidf_vectors = vectorizer.transform(corpus)

# 打印特征名称和 TF-IDF 值
for i, doc in enumerate(corpus):
    print(f"Document {i}:")
    for j, feature_name in enumerate(feature_names):
        print(f"  {feature_name}: {tfidf_vectors[i, j]}")
```

### 5.2 解释说明

* `TfidfVectorizer` 类用于计算 TF-IDF 特征向量。
* `fit()` 方法用于训练模型，学习词语的 TF-IDF 值。
* `get_feature_names()` 方法用于获取特征名称，即词语列表。
* `transform()` 方法用于将文档转换为 TF-IDF 特征向量。
