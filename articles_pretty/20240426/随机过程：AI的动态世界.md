## 2. 核心概念与联系

### 2.1 随机变量与概率分布

随机变量是数值结果的变量，其值是随机现象的数值结果。概率分布描述了随机变量取不同值的可能性。

### 2.2 随机过程的类型

*   **马尔可夫过程：**未来的状态仅取决于当前状态，而与过去状态无关。
*   **泊松过程：**描述在特定时间段或空间区域内事件发生的频率。
*   **布朗运动：**描述粒子的随机运动，是金融数学和物理学中的重要模型。

## 3. 核心算法原理具体操作步骤

### 3.1 马尔可夫链蒙特卡罗 (MCMC) 方法

MCMC 是一种用于从复杂概率分布中进行采样的算法。它通过构建马尔可夫链来探索目标分布，并生成样本。

**操作步骤：**

1.  定义目标分布。
2.  构建马尔可夫链，使其平稳分布为目标分布。
3.  从任意初始状态开始，根据转移概率进行状态转移。
4.  丢弃初始样本（“burn-in”阶段），并收集后续样本作为目标分布的近似样本。

### 3.2 卡尔曼滤波

卡尔曼滤波是一种用于估计动态系统状态的算法，它结合了测量值和系统模型来生成更准确的估计。

**操作步骤：**

1.  预测：根据系统模型预测下一时刻的状态。
2.  更新：根据测量值和预测值计算最优估计。

## 4. 数学模型和公式详细讲解举例说明

### 4.1 马尔可夫链

马尔可夫链可以用转移矩阵表示，其中每个元素表示从一个状态转移到另一个状态的概率。

$$
P_{ij} = P(X_{t+1} = j | X_t = i)
$$

### 4.2 泊松分布

泊松分布用于描述在特定时间段或空间区域内事件发生的次数。

$$
P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}
$$

其中，$\lambda$ 是事件发生的平均速率。

### 5.1 使用 Python 模拟随机游走

```python
import random

def random_walk(n):
    x = 0
    for _ in range(n):
        step = random.choice([-1, 1])
        x += step
    return x
```

此代码模拟了一个简单的随机游走，其中每次移动的步长为 1 或 -1，概率相等。
