## 4. 数学模型和公式

### 4.1 用户相似度计算

协同过滤算法需要计算用户之间的相似度，常用的方法包括：

*   **余弦相似度:** $$sim(u,v) = \frac{\vec{u} \cdot \vec{v}}{||\vec{u}|| \cdot ||\vec{v}||}$$
*   **皮尔逊相关系数:** $$sim(u,v) = \frac{cov(u,v)}{\sigma_u \cdot \sigma_v}$$

### 4.2 推荐算法

常见的推荐算法包括：

*   **基于用户的协同过滤 (User-based CF):**  $$p_{u,i} = \frac{\sum_{v \in N(u)} sim(u,v) \cdot r_{v,i}}{\sum_{v \in N(u)} sim(u,v)}$$
*   **基于物品的协同过滤 (Item-based CF):** $$p_{u,i} = \sum_{j \in N(i)} sim(i,j) \cdot r_{u,j}$$

```python
def user_based_cf(ratings, user_id, item_id):
    # 计算用户相似度
    similarities = {}
    for other_user_id in ratings:
        if other_user_id != user_id:
            similarities[other_user_id] = cosine_similarity(ratings[user_id], ratings[other_user_id])
    
    # 找到最相似的 K 个用户
    k_nearest_neighbors = sorted(similarities, key=similarities.get, reverse=True)[:K]
    
    # 计算预测评分
    numerator = 0
    denominator = 0
    for neighbor_id in k_nearest_neighbors:
        neighbor_rating = ratings[neighbor_id].get(item_id, 0)
        similarity = similarities[neighbor_id]
        numerator += neighbor_rating * similarity
        denominator += similarity
    
    if denominator == 0:
        return 0
    else:
        return numerator / denominator
```

### 5.2 解释说明

该代码首先计算用户之间的余弦相似度，然后找到与目标用户最相似的 K 个用户，最后根据这些用户的评分计算目标用户对指定商品的预测评分。
