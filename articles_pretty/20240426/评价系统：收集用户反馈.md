## 4. 数学模型和公式详细讲解举例说明

评价系统中使用的数学模型和公式取决于具体的算法和应用场景。以下是一些常见的例子：

### 4.1 余弦相似度

余弦相似度用于衡量两个向量之间的相似程度，例如用户之间的相似性或内容之间的相似性。余弦相似度的公式如下：

$$
\cos(\theta) = \frac{\mathbf{A} \cdot \mathbf{B}}{\|\mathbf{A}\| \|\mathbf{B}\|}
$$

其中，$\mathbf{A}$ 和 $\mathbf{B}$ 是两个向量，$\theta$ 是它们之间的夹角。

### 4.2 TF-IDF

TF-IDF (Term Frequency-Inverse Document Frequency) 是一种用于评估词语在文档中重要性的统计方法。TF-IDF 的公式如下：

$$
\text{TF-IDF}(t, d) = \text{TF}(t, d) \times \text{IDF}(t)
$$

其中，$t$ 是词语，$d$ 是文档，$\text{TF}(t, d)$ 是词语 $t$ 在文档 $d$ 中出现的频率，$\text{IDF}(t)$ 是词语 $t$ 的逆文档频率。

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 编写的简单评价系统示例：

```python
class RatingSystem:
    def __init__(self):
        self.ratings = {}  # 存储用户评价

    def add_rating(self, user, item, rating):
        self.ratings.setdefault(user, {})[item] = rating

    def get_average_rating(self, item):
        ratings = self.ratings.get(item, [])
        if ratings:
            return sum(ratings) / len(ratings)
        else:
            return 0
```

这个示例代码定义了一个 `RatingSystem` 类，它可以存储用户评价并计算平均评分。
