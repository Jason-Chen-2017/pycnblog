### 4.1 余弦相似度

余弦相似度用于计算两个向量之间的夹角余弦值，取值范围为 [-1, 1]。余弦值越接近 1，表示两个向量越相似。

$$
cos(\theta) = \frac{A \cdot B}{||A|| ||B||}
$$

其中，A 和 B 分别表示两个向量，$A \cdot B$ 表示两个向量的点积，$||A||$ 和 $||B||$ 分别表示两个向量的长度。

**例子：**

假设有两个用户 A 和 B，他们的阅读历史如下：

* 用户 A:  [书籍 1, 书籍 2, 书籍 3]
* 用户 B:  [书籍 2, 书籍 3, 书籍 4]

可以使用余弦相似度计算用户 A 和 B 之间的相似度：

$$
cos(\theta) = \frac{2}{\sqrt{3} \sqrt{3}} = \frac{2}{3}
$$

其中，R 表示用户-物品矩阵，P 和 Q 分别表示用户特征矩阵和物品特征矩阵。

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv("books.csv")

# 计算书籍之间的相似度
similarity_matrix = cosine_similarity(data[["genre", "author"]])

# 获取用户感兴趣的书籍
user_id = 1
user_books = data[data["user_id"] == user_id]["book_id"].tolist()

# 找到与用户感兴趣的书籍相似的书籍
similar_books = []
for book_id in user_books:
    similar_book_ids = similarity_matrix[book_id].argsort()[::-1][1:6]
    similar_books.extend(data.iloc[similar_book_ids]["book_id"].tolist())

# 推荐书籍
recommended_books = list(set(similar_books) - set(user_books))

# 打印推荐结果
print("推荐书籍:", recommended_books)
```

**代码解释：**

1. 加载包含书籍信息和用户阅读历史的数据。
2. 使用余弦相似度计算书籍之间的相似度。
3. 获取用户感兴趣的书籍。
4. 找到与用户感兴趣的书籍相似的书籍。
5. 从相似书籍中排除用户已经读过的书籍，得到推荐结果。
