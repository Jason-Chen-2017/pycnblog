### 4.1 协同过滤算法

协同过滤算法的核心思想是，相似用户具有相似的偏好。常用的协同过滤算法包括基于用户的协同过滤和基于物品的协同过滤。

#### 4.1.1 基于用户的协同过滤

基于用户的协同过滤算法首先计算用户之间的相似度，然后根据相似用户的购买记录，预测目标用户可能感兴趣的商品。用户相似度可以通过余弦相似度、皮尔逊相关系数等方法计算。

#### 4.1.2 基于物品的协同过滤

基于物品的协同过滤算法首先计算物品之间的相似度，然后根据目标用户购买过的商品，推荐与之相似的商品。物品相似度可以通过 Jaccard 相似度、余弦相似度等方法计算。

### 4.2 基于内容的推荐算法

基于内容的推荐算法根据用户的兴趣和需求，推荐与用户偏好相似的商品。例如，如果用户经常购买运动类商品，则 AI 导购系统可以推荐其他运动类商品。

import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载用户评分数据
ratings_data = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity_matrix = cosine_similarity(ratings_data.pivot_table(index='user_id', columns='item_id', values='rating'))

# 获取目标用户的相似用户
target_user_id = 1
similar_users = user_similarity_matrix[target_user_id].argsort()[::-1][1:]

# 获取相似用户购买过的商品
similar_user_ratings = ratings_data[ratings_data['user_id'].isin(similar_users)]
recommended_items = similar_user_ratings['item_id'].unique()

# 打印推荐结果
print(f'Recommended items for user {target_user_id}: {recommended_items}')
