## 4. 수학 모델 및 공식 설명과 예시

### 4.1 협업 필터링

협업 필터링은 비슷한 취향을 가진 사용자들의 상품 평가를 기반으로 상품을 추천하는 알고리즘입니다. 협업 필터링에는 사용자 기반 협업 필터링과 상품 기반 협업 필터링이 있습니다.

**사용자 기반 협업 필터링**은 특정 사용자와 비슷한 취향을 가진 다른 사용자들이 좋아하는 상품을 추천합니다.

**상품 기반 협업 필터링**은 특정 상품과 비슷한 특성을 가진 다른 상품을 추천합니다. 

### 4.2 콘텐츠 기반 필터링

콘텐츠 기반 필터링은 사용자가 과거에 좋아했던 상품과 비슷한 특성을 가진 상품을 추천합니다. 예를 들어, 사용자가 과거에 구매한 컴퓨터의 CPU, RAM, 그래픽 카드 등의 사양을 기반으로 비슷한 사양의 컴퓨터를 추천합니다. 

### 4.3 하이브리드 추천

하이브리드 추천은 협업 필터링과 콘텐츠 기반 필터링을 결합한 추천 방식입니다. 두 가지 방식의 장점을 결합하여 더 정확하고 다양한 상품 추천을 제공할 수 있습니다.
