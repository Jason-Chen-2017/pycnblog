## 4. 数学模型和公式

### 4.1 协同过滤

协同过滤算法的数学模型可以表示为：

$$
r_{ui} = \frac{\sum_{v \in N(u)} r_{vi} \cdot sim(u,v)}{\sum_{v \in N(u)} sim(u,v)}
$$

其中，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$N(u)$ 表示与用户 $u$ 相似的用户集合，$sim(u,v)$ 表示用户 $u$ 和用户 $v$ 之间的相似度。

### 4.2 Seq2Seq模型

Seq2Seq模型是一种深度学习模型，可以用于对话生成。其数学模型可以表示为：

$$
P(y_1, y_2, ..., y_T | x_1, x_2, ..., x_N) = \prod_{t=1}^T P(y_t | y_{<t}, x_{1:N})
$$

其中，$x_{1:N}$ 表示输入序列，$y_{1:T}$ 表示输出序列，$P(y_t | y_{<t}, x_{1:N})$ 表示在给定输入序列和前 $t-1$ 个输出的情况下，生成第 $t$ 个输出的概率。

import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(data.pivot_table(index='user_id', columns='item_id', values='rating').fillna(0))

# 为用户推荐商品
def recommend(user_id, num_recommendations):
  # 找到与目标用户最相似的用户
  similar_users = user_similarity[user_id].argsort()[::-1][1:num_recommendations+1]
  # 获取相似用户喜欢的商品
  recommendations = data[data['user_id'].isin(similar_users)]['item_id'].value_counts().index[:num_recommendations]
  return recommendations
