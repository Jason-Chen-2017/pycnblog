### 2.1 符号主义与连接主义

早期的人工智能研究主要受理性主义影响，以符号主义为主导。符号主义将智能视为符号操作的过程，通过逻辑推理和规则演绎来实现智能行为。连接主义则受到经验主义启发，模拟人脑神经网络结构，通过大量数据的训练来学习知识和技能。

### 2.2 知识表示与推理

```python
# 定义规则
rules = [
    ("IF 发烧 AND 咳嗽 THEN 感冒"),
    ("IF 感冒 THEN 建议多喝水"),
]

# 匹配规则
def match_rule(facts):
    for rule in rules:
        condition, action = rule.split(" THEN ")
        if all(fact in facts for fact in condition.split(" AND ")):
            return action
    return None

# 输入事实
facts = ["发烧", "咳嗽"]

# 推理结果
result = match_rule(facts)
print(result)  # 输出：建议多喝水
```
