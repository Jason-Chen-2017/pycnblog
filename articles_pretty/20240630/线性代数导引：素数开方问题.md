## 2. 核心概念与联系
### 2.1  素数
素数是大于1的自然数，除了1和自身外，没有其他因数。例如，2、3、5、7、11等都是素数。

### 2.2  质因数分解
任何大于1的自然数都可以唯一地表示为素数的乘积，这个乘积称为该数的质因数分解。例如，12的质因数分解为2×2×3，即2²×3。

### 2.3  欧拉函数

## 3. 核心算法原理 & 具体操作步骤
### 3.1  算法原理概述
素数开方问题可以分为两个主要方面：素数判定和素因子分解。

* **素数判定:** 判断一个数是否为素数。
* **素因子分解:** 将一个数分解成其所有素因子的乘积。

### 3.2  算法步骤详解
#### 3.2.1  素数判定算法
**试除法:** 
1. 从2开始，依次尝试将n除以每个整数，直到达到n的平方根。
2. 如果n能被某个整数整除，则n不是素数。
3. 如果n不能被任何整数整除，则n是素数。

**Miller-Rabin素性测试:** 
1. 将n-1表示为2^s*d，其中d为奇数。
2. 随机选择一个整数a，满足0<a<n-1。
3. 计算a^d mod n。
4. 如果a^d mod n=1或a^d mod n=n-1，则n可能是素数。
5. 重复步骤3和4，直到满足一定条件。

#### 3.2.2  素因子分解算法
**试除法:** 
1. 从2开始，依次尝试将n除以每个整数，直到达到n的平方根。
2. 如果n能被某个整数整除，则该整数是n的一个素因子。
3. 重复步骤1，直到找到所有素因子。

**Pollard's rho算法:** 
1. 选择一个随机函数f(x)。
2. 随机选择两个整数x0和x1。
3. 计算x0, x1, f(x0), f(x1), f(f(x0)), f(f(x1))等。
4. 如果发现两个数的差值可以被n整除，则该差值是n的一个素因子。

### 3.3  算法优缺点
| 算法 | 优点 | 缺点 |
|---|---|---|
| 试除法 | 简单易懂 | 效率低 |
| Miller-Rabin素性测试 | 效率高 | 概率性算法 |
| Pollard's rho算法 | 效率高 | 概率性算法 |

### 3.4  算法应用领域
素数开方问题在以下领域有着广泛的应用：

* **密码学:** RSA加密算法、Diffie-Hellman密钥交换协议等都依赖于素数分解的困难性。
* **编码理论:** 纠错码的设计和实现需要用到素数的性质。

```
def is_prime(n, a):
    if n == 2:
        return True
    if n % 2 == 0:
        return False
    s = 0
    d = n - 1
    while d % 2 == 0:
        s += 1
        d //= 2
    x = pow(a, d, n)
    if x == 1 or x == n - 1:
        return True
    for _ in range(s - 1):
        x = pow(x, 2, n)
        if x == n - 1:
            return True
    return False
```

```
    x = 2
    y = 2
    d = 1
    while d == 1:
        x = f(x)
        y = f(f(y))
        d = gcd(abs(x - y), n)
    return d
```

```python
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def prime_factorization(n):
    factors = []
    i = 2
    while i * i <= n:
        if n % i:
            i += 1
        else:
            n //= i
            factors.append(i)
    if n > 1:
        factors.append(n)
    return factors

# 测试代码
number = 12
if is_prime(number):
    print(f"{number} is a prime number.")
else:
    print(f"{number} is not a prime number.")
    print(f"Prime factors of {number}: {prime_factorization(number)}")
```
