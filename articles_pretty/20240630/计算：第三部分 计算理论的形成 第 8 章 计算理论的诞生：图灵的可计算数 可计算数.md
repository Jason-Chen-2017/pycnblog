## 2. 核心概念与联系

### 2.1 图灵机

图灵机是一种抽象的计算模型，它由以下部分组成：

* **带子：** 一条无限长的纸带，被分成一个个方格，每个方格可以存储一个符号。
* **读写头：** 一个可以移动的读写头，它可以读取带子上的符号，并写入新的符号。
* **状态寄存器：** 一个存储图灵机当前状态的寄存器。
* **状态转移函数：** 一个函数，它根据当前状态和读写头读取的符号，决定下一个状态和读写头的动作（移动方向和写入符号）。

**图灵机的运行过程如下：**

1. 图灵机从初始状态开始，读写头指向带子的第一个方格。
2. 根据当前状态和读写头读取的符号，状态转移函数决定下一个状态和读写头的动作。
3. 图灵机按照状态转移函数的指示，不断地改变状态和移动读写头，直到到达最终状态。

**图灵机的计算能力非常强大，它可以模拟任何实际的计算机程序。**

class TuringMachine:
    def __init__(self, states, input_symbols, tape_symbols, transition_function, initial_state, final_states):
        self.states = states
        self.input_symbols = input_symbols
        self.tape_symbols = tape_symbols
        self.transition_function = transition_function
        self.initial_state = initial_state
        self.final_states = final_states
        self.tape = ['空格'] * 100
        self.head_position = 0
        self.current_state = initial_state

    def run(self, input_string):
        self.tape[self.head_position:self.head_position + len(input_string)] = list(input_string)
        while self.current_state not in self.final_states:
            current_symbol = self.tape[self.head_position]
            next_state, next_symbol, direction = self.transition_function[self.current_state, current_symbol]
            self.tape[self.head_position] = next_symbol
            if direction == 'L':
                self.head_position -= 1
            elif direction == 'R':
                self.head_position += 1
            self.current_state = next_state
        return ''.join(self.tape)

# 定义图灵机 M

transition_function = {
    ('q0', '3'): ('q1', '空格', 'R'),
    ('q1', '5'): ('q2', '空格', 'R'),
    ('q2', '空格'): ('q3', '8', 'L'),
    ('q3', '空格'): ('q4', '空格', 'R'),
}
initial_state = 'q0'
final_states = {'q4'}

# 创建图灵机对象
turing_machine = TuringMachine(states, input_symbols, tape_symbols, transition_function, initial_state, final_states)

# 运行图灵机

# 输出结果
