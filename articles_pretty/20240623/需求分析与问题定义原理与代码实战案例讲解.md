```java

    private Long id;

    // getters and setters
```

接着，我们需要创建一个商品服务类（ProductService）来提供商品相关的服务。商品服务类包含以下方法：获取所有商品（getAllProducts），获取指定ID的商品（getProductById），创建商品（createProduct），更新商品（updateProduct），删除商品（deleteProduct）。

```java
@Service
public class ProductService {
    @Autowired
    private ProductRepository productRepository;

    public List<Product> getAllProducts() {
        return productRepository.findAll();
    }

    public Product getProductById(Long id) {
        return productRepository.findById(id).orElse(null);
    }

    public Product createProduct(Product product) {
        return productRepository.save(product);
    }

    public Product updateProduct(Product product) {
        return productRepository.save(product);
    }

    public void deleteProduct(Long id) {
        productRepository.deleteById(id);
    }
}
```

最后，我们需要创建一个订单服务类（OrderService）来提供订单相关的服务。订单服务类包含以下方法：获取所有订单（getAllOrders），获取指定ID的订单（getOrderById），创建订单（createOrder），更新订单（updateOrder），删除订单（deleteOrder）。

```java
@Service
public class OrderService {
```
