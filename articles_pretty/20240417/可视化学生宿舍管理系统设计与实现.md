## 4.项目实践：代码实例和详细解释说明

让我们通过一个简单的例子来说明如何实现宿舍管理系统的一些功能。在这个例子中，我们将实现一个查询宿舍信息的功能。

首先，我们需要在数据库中创建一个名为`dormitory`的表，用来存储宿舍的信息。

```sql
CREATE TABLE dormitory (
    id INT PRIMARY KEY,
    number VARCHAR(50),
    capacity INT,
    current_usage INT
);
```

然后，我们可以使用以下SQL语句查询宿舍的使用率：

```sql
SELECT number, current_usage / capacity AS usage_rate
FROM dormitory
```

这个SQL语句会返回每个宿舍的编号和使用率。
