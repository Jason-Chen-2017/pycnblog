## 4. 数学模型和公式详细讲解
4.1 基于机器学习的覆盖预测
假设5G信号强度服从对数正态分布,则其概率密度函数为:
$$f(x) = \frac{1}{x\sigma\sqrt{2\pi}}e^{-\frac{(\ln x-\mu)^2}{2\sigma^2}}$$
其中,$\mu$为对数均值,$\sigma^2$为对数方差。利用训练数据拟合得到$\mu$和$\sigma^2$,即可预测任意位置的5G信号覆盖概率。

4.2 基于排队论的容量分析 
假设5G用户请求服从泊松分布,服务时间服从指数分布,则系统可用M/M/c/K排队论模型描述。根据此模型,可计算出系统中平均用户数$L$和系统阻塞概率$P_b$:
$$L = \frac{\rho c}{c-\rho}(1-P_b)$$
$$P_b = \frac{\left(\frac{\rho^c}{c!}\right)}{\sum_{n=0}^c\frac{\rho^n}{n!}+\frac{\rho^c}{c!(1-\rho/c)}}$$
其中,$\rho=\lambda/\mu$为服务强度,$\lambda$为用户到达率,$\mu$为服务率。

4.3 基于时间同步的时延测量
假设测量节点间时间误差服从正态分布$N(0,\sigma^2)$,则端到端时延$T$可表示为:
$$T = t_2 - t_1 + \Delta t$$
其中,$t_2$和$t_1$分别为发送和接收时间戳,$\Delta t$为时间同步偏差。通过统计分析可得时延分布的均值和方差。

4.4 基于Markov链的可靠性建模
假设5G网络故障过程可用连续时间Markov链描述,状态集合为$S=\{0,1,2,...,n\}$,其中0表示完好状态,其他状态表示不同程度的故障。状态转移概率矩阵为$\mathbf{P}$,故障发生率为$\lambda$,修复率为$\mu$。则网络可用性$A$可计算为:
$$A = \pi_0 = \frac{\mu}{\lambda+\mu}$$
其中,$\pi_0$为稳态下网络完好状态的概率。

def coverage_prediction(x, mu, sigma):

# 5G容量分析  
def capacity_analysis(rho, c, K):
    p = np.zeros(c+1)
    p[0] = 1
    for n in range(1, c+1):
        p[n] = p[n-1] * rho / n
    p_b = sum(p[c:]) / (1 + sum(p))
    L = rho * c / (c - rho) * (1 - p_b)
    return L, p_b

# 5G时延测量
def latency_measurement(t1, t2, sigma):
    delta_t = np.random.normal(0, sigma)
    latency = t2 - t1 + delta_t
    return latency

def reliability_modeling(lambd, mu, t):
    p0 = mu / (lambd + mu)
    p = np.exp(-(lambd + mu) * t)
    return p0 * p
