## 4.数学模型和公式详细讲解举例说明

### 4.1 Dijkstra算法

Dijkstra算法是一种用于寻找最短路径的算法。设G=(V,E)是一个有向图，V是顶点的有限集合，E是边的有限集合。$w_{ij}$表示连接顶点i和j的边的权重。

Dijkstra算法的数学模型可以表示为：

$$
Minimize \ \sum_{(i,j) \in E} w_{ij} x_{ij} \\
s.t. \ \sum_{j \in V} x_{ij} - \sum_{j \in V} x_{ji} = 0\ \forall i \in V \\
x_{ij} \in \{0,1\} \ \forall (i,j) \in E
$$

其中$x_{ij}$表示是否选择边(i,j)。如果选择了边(i,j)，那么$x_{ij}=1$，否则$x_{ij}=0$。

class InventoryManagement {
    private Map<String, Integer> inventory;

    public InventoryManagement() {
        inventory = new HashMap<>();
    }

    public void add(String item, int quantity) {
        inventory.put(item, inventory.getOrDefault(item, 0) + quantity);
    }

    public void reduce(String item, int quantity) {
        if (inventory.containsKey(item)) {
            inventory.put(item, inventory.get(item) - quantity);
        }
    }

    public int getQuantity(String item) {
        return inventory.getOrDefault(item, 0);
    }
}

### 5.2 Dijkstra算法类

以下是Dijkstra算法的Java实现：

```java
class Dijkstra {
    private int[] dist;
    private Set<Integer> settled;
    private PriorityQueue<Node> pq;
    private int V;
    List<List<Node>> adj;

    public Dijkstra(int V) {
        this.V = V;
        dist = new int[V];
        settled = new HashSet<>();
        pq = new PriorityQueue<>(V, new Node());
    }

    public void dijkstra(List<List<Node>> adj, int src) {
        this.adj = adj;

        for (int i = 0; i < V; i++) {
            dist[i] = Integer.MAX_VALUE;
        }

        pq.add(new Node(src, 0));

        dist[src] = 0;
        while (settled.size() != V) {
            int u = pq.remove().node;
            settled.add(u);
            e_Neighbours(u);
        }
    }

    private void e_Neighbours(int u) {
        int edgeDistance = -1;
        int newDistance = -1;

        for (int i = 0; i < adj.get(u).size(); i++) {
            Node v = adj.get(u).get(i);

            if (!settled.contains(v.node)) {
                edgeDistance = v.cost;
                newDistance = dist[u] + edgeDistance;

                if (newDistance < dist[v.node]) {
                    dist[v.node] = newDistance;
                }

                pq.add(new Node(v.node, dist[v.node]));
            }
        }
    }
}

class Node implements Comparator<Node> {
    public int node;
    public int cost;

    public Node() {
    }

    public Node(int node, int cost) {
        this.node = node;
        this.cost = cost;
    }

    @Override
    public int compare(Node node1, Node node2) {
        if (node1.cost < node2.cost) {
            return -1;
        }
        if (node1.cost > node2.cost) {
            return 1;
        }
        return 0;
    }
}
```
