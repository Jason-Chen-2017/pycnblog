### 5.3 优化算法的使用

以下是一个简单的例子，展示了如何使用优化算法来定制和预订旅游服务。

```python
# 导入所需的库
import numpy as np
from scipy.optimize import linprog

# 定义优化问题的参数
c = np.array([-1, -2])  # 目标函数的系数（注意我们需要最大化目标函数，所以这里的系数是负数）
A = np.array([[1, 1], [1, 2]])  # 不等式约束的系数
b = np.array([10, 20])  # 不等式约束的右侧值

# 使用线性规划算法求解优化问题
res = linprog(c, A_ub=A, b_ub=b)

# 输出最优解
print(res.x)
```
