## 4.数学模型和公式详细讲解举例说明

在个人通信簿管理系统中，我们并没有使用到复杂的数学模型和公式。然而，我们仍然需要处理一些基本的数据结构和算法问题。例如，在查询联系人时，我们需要使用到二分查找算法。二分查找算法的基本思想是：首先，将待查找的元素与中间元素进行比较，如果相等，则查找成功；如果大于中间元素，则在中间元素的右边进行查找；如果小于中间元素，则在中间元素的左边进行查找。

二分查找算法的时间复杂度为$O(logn)$，其中$n$为联系人的数量。

### 4.1 数据库操作代码示例

```python
def add_contact(name, phone, email):
    # 添加联系人
    sql = "INSERT INTO contacts (name, phone, email) VALUES (?, ?, ?)"
    db.execute(sql, [name, phone, email])
    db.commit()

def delete_contact(id):
    # 删除联系人
    sql = "DELETE FROM contacts WHERE id = ?"
    db.execute(sql, [id])
    db.commit()

def update_contact(id, name, phone, email):
    # 更新联系人信息
    sql = "UPDATE contacts SET name = ?, phone = ?, email = ? WHERE id = ?"
    db.execute(sql, [name, phone, email, id])
    db.commit()

def query_contact(name):
    # 查询联系人
    sql = "SELECT * FROM contacts WHERE name = ?"
    result = db.execute(sql, [name]).fetchall()
    return result
```
