4. 删除数据：使用哈希函数将键映射到哈希表的一个位置，然后在该位置删除数据。

## 4. 数学模型和公式详细讲解举例说明

在哈希表中，哈希函数的选择非常重要，它直接影响到哈希表的性能。一个好的哈希函数可以将键均匀地分布在哈希表中，从而提高哈希表的性能。

假设我们的哈希表有m个槽，哈希函数h将键k映射到哈希表的一个槽，我们可以使用以下公式来表示这个过程：

$$ h(k) = k \mod m $$

这个公式表示，我们将键k除以m，然后取余数作为哈希值。这个哈希值就是键k在哈希表中的位置。

例如，假设我们的哈希表有10个槽，键k为15，那么哈希值为：

$$ h(15) = 15 \mod 10 = 5 $$

这意味着，键15在哈希表中的位置为5。

```python
class HashTable:
    def __init__(self, m):
        self.table = [None] * m
        self.m = m

    def insert(self, k, v):
        i = k % self.m
        self.table[i] = v

    def search(self, k):
        i = k % self.m
        return self.table[i]

    def delete(self, k):
        i = k % self.m
        self.table[i] = None
```

在这个代码中，我们首先定义了一个HashTable类，它有一个初始化方法和三个操作方法。初始化方法用于创建一个大小为m的哈希表。插入方法用于插入数据，查找方法用于查找数据，删除方法用于删除数据。这三个方法都使用了哈希函数h(k) = k mod m来计算键k在哈希表中的位置。
