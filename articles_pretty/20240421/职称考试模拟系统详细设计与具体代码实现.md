推荐系统通常使用协同过滤（Collaborative Filtering）算法，该算法主要包括基于用户的协同过滤（User-Based Collaborative Filtering）和基于项目的协同过滤（Item-Based Collaborative Filtering）。

基于用户的协同过滤算法的核心思想是：对于一个用户，找到和他有相似兴趣的用户，然后推荐那些用户感兴趣的项目给他。我们可以通过如下的公式计算用户之间的相似度：

$$
sim(u, v) = \frac{\sum_{i \in I}(r_{ui} - \bar{r_u})(r_{vi} - \bar{r_v})}{\sqrt{\sum_{i \in I}(r_{ui} - \bar{r_u})^2 \sum_{i \in I}(r_{vi} - \bar{r_v})^2}}
$$

其中，$r_{ui}$ 是用户 $u$ 对项目 $i$ 的评分，$\bar{r_u}$ 是用户 $u$ 的平均评分，$I$ 是用户 $u$ 和用户 $v$ 都评分的项目集合。

    '''计算矩阵中每个用户之间的相似度，返回一个相似度矩阵'''
    n = len(data)
    # 初始化一个n*n的矩阵
    matrix = np.zeros((n, n))

            # 计算用户i和用户j的相似度

def cos_similarity(v1, v2):
    '''计算两个向量的余弦相似度'''
    sum_x = np.sum(v1 * v1)
    sum_y = np.sum(v2 * v2)
    sum_xy = np.sum(v1 * v2)
    
    return sum_xy / np.sqrt(sum_x * sum_y)
