然后，我们创建一个名为app.py的文件，内容如下：

```python
from flask import Flask, render_template, request
import mysql.connector

app = Flask(__name__)

@app.route('/', methods=['GET', 'POST'])
def index():
    if request.method == 'POST':
        id = request.form.get('id')
        name = request.form.get('name')
        score = request.form.get('score')
        
        conn = mysql.connector.connect(user='root', password='123456', database='test')
        cursor = conn.cursor()
        cursor.execute('INSERT INTO scores (id, name, score) VALUES (%s, %s, %s)', (id, name, score))
        conn.commit()
        cursor.close()
        conn.close()
        
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)
```

这个例子中，我们创建了一个Flask应用，并定义了一个路由。当用户访问这个路由时，如果是POST请求，那么就从请求中获取学生的id、姓名和成绩，然后将这些数据插入到数据库中。如果是GET请求，那么就直接返回一个HTML页面。
