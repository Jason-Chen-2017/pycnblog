### 4.3 前端设计代码示例

我们最后来看一个前端设计的代码示例。这里我们使用React作为前端框架。

```javascript
import React from 'react';
import axios from 'axios';

class App extends React.Component {
    state = {
        users: [],
        resources: [],
        activities: [],
    };

    componentDidMount() {
        axios.get('/api/users').then((response) => {
            this.setState({ users: response.data });
        });

        axios.get('/api/resources').then((response) => {
            this.setState({ resources: response.data });
        });

        axios.get('/api/activities').then((response) => {
            this.setState({ activities: response.data });
        });
    }

    render() {
        return (
            <div>
                <h1>Users</h1>
                {this.state.users.map((user) => (
                    <div key={user.id}>{user.name}</div>
                ))}

                <h1>Resources</h1>
                {this.state.resources.map((resource) => (
                    <div key={resource.id}>{resource.name}</div>
                ))}

                <h1>Activities</h1>
                {this.state.activities.map((activity) => (
                    <div key={activity.id}>{activity.name}</div>
                ))}
            </div>
        );
    }
}

export default App;
```

这个示例中，我们创建了一个React组件`App`。这个组件有一个状态，这个状态包含三个数组：`users`、`resources`和`activities`。在组件挂载后，我们用axios发送三个GET请求，获取所有用户、所有资源和所有活动，然后把这些数据保存到状态中。在渲染函数中，我们把这些数据显示到页面上。
