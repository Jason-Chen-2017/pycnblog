### 4.1 移动平均模型

移动平均模型是一种时间序列预测模型，其基本思想是用过去若干期的平均值作为未来的预测值。模型的公式如下：

$$ \hat{X}_{t} = \frac{1}{n} \sum_{i=1}^{n}{X_{t-i}} $$

其中，$\hat{X}_{t}$ 是第 $t$ 期的预测值，$X_{t-i}$ 是第 $(t-i)$ 期的实际值，$n$ 是平均的期数。

例如，如果我们要预测下一季度的销售量，可以取过去四个季度的销售量的平均值作为预测值。

## 4.项目实践：代码实例和详细解释说明

下面我们以 Python 为例，实现一个简单的移动平均模型。

```python
import numpy as np

def moving_average_forecast(sales, window_size):
    """
    Function to compute moving average forecast
    sales : array-like, sales data
    window_size : int, the size of moving window
    """
    forecast = np.convolve(sales, np.ones(window_size), 'valid') / window_size
    return forecast
```

在这个函数中，我们使用了 numpy 的 `convolve` 函数来计算移动平均。`np.ones(window_size)` 创建了一个长度为 `window_size` 的全 1 数组，与销售数据进行卷积运算，然后除以窗口大小，即得到了移动平均预测值。
