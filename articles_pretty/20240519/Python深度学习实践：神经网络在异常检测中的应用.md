## 4. 数学模型和公式详细讲解举例说明
### 4.1 自编码器的数学模型
#### 4.1.1 编码器
$$z = f(Wx+b)$$
#### 4.1.2 解码器  
$$\hat{x} = g(W'z+b')$$
#### 4.1.3 重构误差
$$L(x,\hat{x}) = \frac{1}{n}\sum_{i=1}^n(x_i-\hat{x}_i)^2$$
### 4.2 变分自编码器的数学模型
#### 4.2.1 编码器
$$q_\phi(z|x) = \mathcal{N}(z|\mu_\phi(x),\sigma^2_\phi(x)I)$$  
#### 4.2.2 解码器
$$p_\theta(x|z) = \mathcal{N}(x|\mu_\theta(z),\sigma^2_\theta(z)I)$$
#### 4.2.3 变分下界
$$\mathcal{L}(\theta,\phi;x) = -D_{KL}(q_\phi(z|x)||p(z)) + \mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)]$$
### 4.3 生成对抗网络的数学模型 
#### 4.3.1 生成器
$$\tilde{x} = G(z), z \sim p(z)$$
#### 4.3.2 判别器
$$D(x) \in (0,1)$$  
#### 4.3.3 对抗损失
$$\min_G \max_D \mathbb{E}_{x \sim p_{data}}[\log D(x)] + \mathbb{E}_{z \sim p(z)}[\log (1-D(G(z)))]$$
