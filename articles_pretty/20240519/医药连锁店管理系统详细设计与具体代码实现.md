## 4.项目实践：代码实例和详细解释说明
在项目实践中，我们以Python为主要编程语言，以下是一段简单的库存管理功能的代码实例：

```python
class Inventory:
    def __init__(self):
        self.items = {}

    def add_item(self, item_name, quantity):
        if item_name in self.items:
            self.items[item_name] += quantity
        else:
            self.items[item_name] = quantity

    def reduce_item(self, item_name, quantity):
        if item_name in self.items and self.items[item_name] >= quantity:
            self.items[item_name] -= quantity
        else:
            print("Error: Not enough inventory for " + item_name)

    def check_item(self, item_name):
        if item_name in self.items:
            return self.items[item_name]
        else:
            return 0
```
