## 4. Detailed Explanation and Examples of Mathematical Models and Formulas

In this section, we will discuss mathematical models and formulas used in online recruitment systems, such as keyword search algorithms, machine learning-based matching algorithms, and interview scheduling algorithms.

### 4.1 Keyword Search Algorithms

Keyword search algorithms can be implemented using techniques such as TF-IDF (Term Frequency-Inverse Document Frequency) and cosine similarity.

#### 4.1.1 TF-IDF

TF-IDF calculates the importance of a word in a document based on its frequency within the document and the overall frequency of the word across all documents.

$$
\\text{TF-IDF} = \\text{TF} \\times \\text{IDF}
$$

Where:

- **TF** (Term Frequency) is the number of times a word appears in a document.
- **IDF** (Inverse Document Frequency) is the logarithm of the total number of documents divided by the number of documents containing the word.

#### 4.1.2 Cosine Similarity

Cosine similarity measures the cosine of the angle between two vectors, representing the similarity between two documents.

$$
\\text{Cosine Similarity} = \\frac{\\mathbf{A} \\cdot \\mathbf{B}}{||\\mathbf{A}|| ||\\mathbf{B}||}
$$

Where:

- **A** and **B** are the vectors representing two documents.
- $\\mathbf{A} \\cdot \\mathbf{B}$ is the dot product of vectors **A** and **B**.
- $||\\mathbf{A}||$ and $||\\mathbf{B}||$ are the magnitudes (lengths) of vectors **A** and **B**.

### 5.1 User Authentication

User authentication is crucial for ensuring the security of the online recruitment system. We will discuss a simple example using Python and Flask.

```python
from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy
from flask_bcrypt import Bcrypt

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///recruitment.db'
db = SQLAlchemy(app)
bcrypt = Bcrypt(app)

users = db.Table('users',
    db.Column('username', db.String(80)),
    db.Column('password', db.String(120))
)

roles = db.Table('roles',
    db.Column('id', db.Integer, primary_key=True),
    db.Column('name', db.String(80))
)

users_roles = db.Table('users_roles',
    db.Column('user_id', db.Integer, db.ForeignKey('users.id')),
    db.Column('role_id', db.Integer, db.ForeignKey('roles.id'))
)

class Role(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80))

class User(UserMixin, db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True)
    password_hash = db.Column(db.String(120))
    roles = db.relationship('Role', secondary=users_roles, backref=db.backref('users', lazy='dynamic'))

@app.route('/register', methods=['POST'])
def register():
    username = request.json['username']
    password = request.json['password']
    role_id = request.json['role_id']

    hashed_password = bcrypt.generate_password_hash(password).decode('utf-8')
    new_user = User(username=username, password_hash=hashed_password, roles=[Role.query.get(role_id)])
    db.session.add(new_user)
    db.session.commit()

    return jsonify({'message': 'User registered successfully'})
```
