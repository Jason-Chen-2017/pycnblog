#### Logistic映射
$$
x_{n+1} = r x_n (1 - x_n)
$$

### 模型构建步骤
1. **初始化参数**：设置初始值和参数。
2. **迭代计算**：通过迭代计算Logistic映射，生成时间序列数据。

### 伪代码
```python
# 初始化参数
r = 3.8  # 参数r
x = 0.5  # 初始值
iterations = 1000  # 迭代次数

# 迭代计算

    x = r * x * (1 - x)
    if i >= 500:  # 跳过初始的500个迭代值
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 混沌吸引子的数学描述
混沌吸引子可以用以下公式描述：
$$
\dot{x} = f(x, y, z)
$$
$$
\dot{y} = g(x, y, z)
$$
$$
\dot{z} = h(x, y, z)
$$

### 例子说明
假设我们有一个简单的混沌系统，其状态变量为$x$、$y$和$z$，状态方程为：
$$
\dot{x} = y
$$
$$
\dot{y} = -x + z
$$
$$
\dot{z} = -0.1z + 0.2y
$$

### 数学模型的构建
我们将使用上述方程构建混沌吸引子。具体步骤如下：
1. **定义状态变量**：$x$、$y$和$z$。
2. **定义状态方程**：$\dot{x} = y$，$\dot{y} = -x + z$，$\dot{z} = -0.1z + 0.2y$。
3. **数值求解**：使用数值方法（如欧拉法或龙格-库塔法）求解上述方程。

### 数学模型的实现
```python
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

# 定义状态方程
def chaotic_system(t, state):
    x, y, z = state
    dxdt = y
    dydt = -x + z
    dzdt = -0.1 * z + 0.2 * y
    return [dxdt, dydt, dzdt]

# 初始条件
initial_state = [1, 1, 1]

# 时间范围
t_span = (0, 100)
t_eval = np.linspace(0, 100, 10000)

# 求解方程
sol = solve_ivp(chaotic_system, t_span, initial_state, t_eval=t_eval)

# 绘制混沌吸引子
fig = plt.figure(figsize=(10, 8))
ax = fig.add_subplot(111, projection='3d')
ax.plot(sol.y[0], sol.y[1], sol.y[2])
ax.set_xlabel('X')
ax.set_ylabel('Y')
ax.set_zlabel('Z')
ax.set_title('Chaos Attractor')
plt.show()
```
