## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 同调群的数学模型
同调群的数学模型可以用链复形和边界算子来描述。链复形是一个序列，每个元素是一个链群，边界算子将链群映射到下一个链群。

#### 数学公式
$$
C_n \xrightarrow{d_n} C_{n-1} \xrightarrow{d_{n-1}} \cdots \xrightarrow{d_1} C_0
$$

### 4.2 上同调的数学模型
上同调的数学模型可以用上同调群来描述。上同调群是链复形的对偶概念。

#### 数学公式
$$
C^n \xrightarrow{d^n} C^{n+1} \xrightarrow{d^{n+1}} \cdots \xrightarrow{d^1} C^0
$$

### 4.3 举例说明
考虑一个简单的链复形：
$$
C_2 \xrightarrow{d_2} C_1 \xrightarrow{d_1} C_0
$$

其中：
$$
C_2 = \mathbb{Z}, \quad C_1 = \mathbb{Z}^2, \quad C_0 = \mathbb{Z}
$$

边界算子为：
$$
d_2: \mathbb{Z} \to \mathbb{Z}^2, \quad d_1: \mathbb{Z}^2 \to \mathbb{Z}
$$

计算同调群：
$$

$$

$$

```python
import numpy as np

def compute_homology_chain_complex(C):
    H = []
    for i in range(len(C)):
        kernel = np.linalg.matrix_rank(C[i])
        image = np.linalg.matrix_rank(C[i+1])
        Hi = kernel - image
        H.append(Hi)
    return H

def compute_cohomology_chain_complex(C):
    H = []
    for i in range(len(C)):
        kernel = np.linalg.matrix_rank(C[i])
        image = np.linalg.matrix_rank(C[i-1])
        Hi = kernel - image
        H.append(Hi)
    return H

# 示例链复形
C = [np.array([[1, 0], [0, 1]]), np.array([[1, 1], [0, 1]]), np.array([[1, 0]])]

homology = compute_homology_chain_complex(C)
cohomology = compute_cohomology_chain_complex(C)

print("Homology groups:", homology)
print("Cohomology groups:", cohomology)
```
