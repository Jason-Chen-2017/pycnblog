function generateFractal(depth, scale, offset):
    if depth == 0:
        return
    else:
        for i in range(scale):
            for j in range(scale):
                x = i / scale - 0.5
                y = j / scale - 0.5
                z = 0
                for k in range(100):
                    z = z * z + (x + offset) + 1j * (y + offset)
                    if abs(z) > 2:
                        break
                plot(x, y, color(k))
```

function generateBezierCurve(points, t):
    if len(points) == 2:
        return (1 - t) * points[0] + t * points[1]
    else:
        return generateBezierCurve([a + (1 - t) * (b - a) for a, b in zip(points[:-1], points[1:])], t)
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 分形数学模型
#### 4.1.1 Mandelbrot集
$$
z_{n+1} = z_n^2 + c
$$
其中，$z_0 = 0$，$c$是复平面上的点。

#### 4.1.2 Julia集
$$
z_{n+1} = z_n^2 + c
$$
其中，$z_0 = z$，$c$是复平面上的点。

### 4.2 贝塞尔曲线数学模型
#### 4.2.1 二次贝塞尔曲线
$$
B(t) = (1-t)^2P_0 + 2t(1-t)P_1 + t^2P_2
$$
其中，$P_0, P_1, P_2$是控制点。

#### 4.2.2 三次贝塞尔曲线
$$
B(t) = (1-t)^3P_0 + 3t(1-t)^2P_1 + 3t^2(1-t)P_2 + t^3P_3
$$
其中，$P_0, P_1, P_2, P_3$是控制点。

### 4.3 傅里叶变换
#### 4.3.1 傅里叶变换公式
$$
F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-i\omega t} dt
$$
其中，$F(\omega)$是频域表示，$f(t)$是时域表示。

#### 4.3.2 逆傅里叶变换
$$
f(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} F(\omega) e^{i\omega t} d\omega
$$

### 5.2 源代码详细实现和代码解读
```python
import numpy as np
import matplotlib.pyplot as plt

def mandelbrot(c, max_iter):
    z = c
    for n in range(max_iter):
        if abs(z) > 2:
            return n
        z = z*z + c
    return max_iter

def mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter):
    r1 = np.linspace(xmin, xmax, width)
    r2 = np.linspace(ymin, ymax, height)
    n3 = np.empty((width, height))
    for i in range(width):
        for j in range(height):
            n3[i, j] = mandelbrot(r1[i] + 1j*r2[j], max_iter)
    return (r1, r2, n3)

def plot_mandelbrot(xmin, xmax, ymin, ymax, width, height, max_iter):
    r1, r2, n3 = mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter)
    plt.imshow(n3.T, extent=[xmin, xmax, ymin, ymax], cmap='hot')
    plt.colorbar()
    plt.show()

plot_mandelbrot(-2.0, 1.0, -1.5, 1.5, 1000, 1000, 256)
```

### 5.3 代码解读与分析
- **mandelbrot**函数：计算复数$c$的Mandelbrot集。
- **mandelbrot_set**函数：生成Mandelbrot集的图像数据。
- **plot_mandelbrot**函数：绘制Mandelbrot集的图像。
