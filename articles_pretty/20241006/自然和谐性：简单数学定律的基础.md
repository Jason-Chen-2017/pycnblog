### 2.4 费马小定理

## 3. 核心算法原理 & 具体操作步骤

### 3.1 递归算法原理
递归算法的基本思想是将问题分解为更小的子问题，然后递归地解决这些子问题。递归算法通常包括两个部分：基本情况和递归情况。

#### 3.1.1 基本情况
基本情况是指可以直接解决的问题，不需要进一步递归。例如，在计算阶乘时，基本情况是n=0或n=1。

#### 3.1.2 递归情况
递归情况是指需要进一步递归解决的问题。例如，在计算阶乘时，递归情况是n>1。

### 3.2 递归算法伪代码
```plaintext
function factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n-1)
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 信息熵
信息熵是衡量信息不确定性的度量，用于描述系统的无序程度。信息熵的公式为：

$$
H(X) = -\sum_{i=1}^{n} p(x_i) \log_2 p(x_i)
$$

其中，$H(X)$ 表示信息熵，$p(x_i)$ 表示事件 $x_i$ 的概率。

### 4.2 费马小定理
费马小定理是数论中的一个重要定理，它指出如果p是一个质数，且a是任意整数，则a的p次方减去a可以被p整除。这个定理可以用于优化某些算法。

### 4.3 举例说明
假设我们有一个质数p=7，我们需要计算2的7次方减去2的结果。根据费马小定理，我们可以直接得出结果：

$$
2^7 - 2 = 128 - 2 = 126
$$

126可以被7整除，验证了费马小定理的正确性。

### 5.1 开发环境搭建

### 5.2 源代码详细实现和代码解读
```python
def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n-1)

# 测试代码
print(factorial(5))  # 输出 120
```

### 5.3 代码解读与分析
- `factorial` 函数接受一个整数 `n` 作为参数。
- 如果 `n` 等于0或1，函数返回1，这是基本情况。
- 否则，函数返回 `n` 乘以 `factorial(n-1)`，这是递归情况。
