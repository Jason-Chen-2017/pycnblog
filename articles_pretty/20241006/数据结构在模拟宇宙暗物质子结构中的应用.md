## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 数学模型
在模拟暗物质子结构时，我们通常使用牛顿引力定律来计算粒子之间的引力作用。牛顿引力定律可以表示为：

$$
F = G \frac{m_1 m_2}{r^2}
$$

其中，$F$ 是引力，$G$ 是引力常数，$m_1$ 和 $m_2$ 是两个粒子的质量，$r$ 是两个粒子之间的距离。

### 4.2 公式详细讲解
在模拟过程中，我们需要计算每个粒子之间的引力作用。假设我们有 $N$ 个粒子，每个粒子的质量为 $m_i$，位置为 $\mathbf{r}_i$，则每个粒子受到的总引力可以表示为：

$$
\mathbf{F}_i = \sum_{j \neq i} G \frac{m_i m_j}{|\mathbf{r}_i - \mathbf{r}_j|^2} \frac{\mathbf{r}_i - \mathbf{r}_j}{|\mathbf{r}_i - \mathbf{r}_j|}
$$

其中，$\mathbf{F}_i$ 是粒子 $i$ 所受的总引力，$\mathbf{r}_i - \mathbf{r}_j$ 是粒子 $i$ 和粒子 $j$ 之间的位移向量。

### 4.3 举例说明
假设我们有 4 个粒子，其质量分别为 $m_1 = 1$, $m_2 = 2$, $m_3 = 3$, $m_4 = 4$，位置分别为 $\mathbf{r}_1 = (0, 0)$, $\mathbf{r}_2 = (1, 0)$, $\mathbf{r}_3 = (0, 1)$, $\mathbf{r}_4 = (1, 1)$。引力常数 $G = 1$。计算粒子 1 所受的总引力：

$$
\mathbf{F}_1 = G \left( \frac{m_1 m_2}{|\mathbf{r}_1 - \mathbf{r}_2|^2} \frac{\mathbf{r}_1 - \mathbf{r}_2}{|\mathbf{r}_1 - \mathbf{r}_2|} + \frac{m_1 m_3}{|\mathbf{r}_1 - \mathbf{r}_3|^2} \frac{\mathbf{r}_1 - \mathbf{r}_3}{|\mathbf{r}_1 - \mathbf{r}_3|} + \frac{m_1 m_4}{|\mathbf{r}_1 - \mathbf{r}_4|^2} \frac{\mathbf{r}_1 - \mathbf{r}_4}{|\mathbf{r}_1 - \mathbf{r}_4|} \right)
$$

代入具体数值：

$$
\mathbf{F}_1 = 1 \left( \frac{1 \cdot 2}{1^2} \frac{(0, 0) - (1, 0)}{1} + \frac{1 \cdot 3}{1^2} \frac{(0, 0) - (0, 1)}{1} + \frac{1 \cdot 4}{2^2} \frac{(0, 0) - (1, 1)}{2} \right)
$$

简化后：

$$
\mathbf{F}_1 = ( -2, 0 ) + ( 0, -3 ) + ( -1, -1 )
$$

最终结果：

$$
\mathbf{F}_1 = ( -3, -4 )
$$

### 5.2 源代码详细实现和代码解读
以下是模拟暗物质子结构的完整代码示例：

```python
import numpy as np
from mpi4py import MPI

# 初始化参数
numParticles = 1000
simulationTime = 100
timeStep = 0.1
G = 1

# 初始化粒子位置和速度
positions = np.random.rand(numParticles, 2)
velocities = np.zeros((numParticles, 2))

# 初始化 MPI
comm = MPI.COMM_WORLD
rank = comm.Get_rank()
size = comm.Get_size()

# 分配粒子到每个进程
localNumParticles = numParticles // size
localPositions = positions[rank * localNumParticles:(rank + 1) * localNumParticles]
localVelocities = velocities[rank * localNumParticles:(rank + 1) * localNumParticles]

# 主循环
for time in range(int(simulationTime / timeStep)):
    # 引力计算
    localForces = np.zeros_like(localPositions)
    for i in range(localNumParticles):
        for j in range(i + 1, localNumParticles):
            r = localPositions[j] - localPositions[i]
            force = G * localVelocities[i] * localVelocities[j] / np.linalg.norm(r)**2
            localForces[i] += force
            localForces[j] -= force
    
    # 并行化
    globalForces = np.zeros_like(positions)
    comm.Allreduce(localForces, globalForces, op=MPI.SUM)
    
    # 运动更新
    for i in range(localNumParticles):
        velocities[rank * localNumParticles + i] += globalForces[i] * timeStep
        positions[rank * localNumParticles + i] += velocities[rank * localNumParticles + i] * timeStep
    
    # 边界条件处理
    # 假设边界条件为周期性边界条件
    positions = np.mod(positions, 1)

# 输出最终结果
print("Final positions:", positions)
```

- **初始化 MPI**：使用 MPI 进行并行计算。
- **分配粒子到每个进程**：将粒子均匀分配到每个进程。
- **主循环**：在每个时间步长中执行引力计算、运动更新和边界条件处理。
