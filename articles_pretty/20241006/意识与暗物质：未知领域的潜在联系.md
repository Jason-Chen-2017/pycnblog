## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 量子态表示
量子态可以用向量表示，如：
$$
|\psi\rangle = \alpha|0\rangle + \beta|1\rangle
$$
其中，$\alpha$和$\beta$是复数，表示量子态的幅度。

### 4.2 量子门操作
量子门操作可以用矩阵表示，如：
$$
U = \begin{pmatrix}
1 & 0 \\
0 & e^{i\theta}
\end{pmatrix}
$$
其中，$\theta$是相位角。

### 4.3 量子态测量
量子态测量可以用投影算子表示，如：
$$
P_0 = |0\rangle\langle0|
$$
$$
P_1 = |1\rangle\langle1|
$$
测量结果的概率可以用投影算子表示，如：
$$
P(\text{测量结果为 } 0) = \langle\psi|P_0|\psi\rangle
$$
$$
P(\text{测量结果为 } 1) = \langle\psi|P_1|\psi\rangle
$$

### 4.4 举例说明
假设有一个量子态：
$$
|\psi\rangle = \frac{1}{\sqrt{2}}|0\rangle + \frac{1}{\sqrt{2}}|1\rangle
$$
通过测量得到的结果为0的概率为：
$$
P(\text{测量结果为 } 0) = \left|\frac{1}{\sqrt{2}}\right|^2 = \frac{1}{2}
$$
通过测量得到的结果为1的概率为：
$$
P(\text{测量结果为 } 1) = \left|\frac{1}{\sqrt{2}}\right|^2 = \frac{1}{2}

```python
from qiskit import QuantumCircuit, execute, Aer
from qiskit.visualization import plot_histogram

# 初始化量子态
def initialize_quantum_state(input_data):
    qc = QuantumCircuit(1)
    qc.h(0)
    return qc

# 量子门操作
def apply_gate(quantum_state, gate):
    qc = quantum_state.copy()
    qc.append(gate, [0])
    return qc

# 测量结果
def measure_quantum_state(quantum_state):
    backend = Aer.get_backend('qasm_simulator')
    result = execute(quantum_state, backend, shots=1000).result()
    counts = result.get_counts()
    return counts

# 主函数
def main():
    input_data = [0]
    quantum_state = initialize_quantum_state(input_data)
    quantum_state = apply_gate(quantum_state, QuantumCircuit(1).rz(0.5, 0))
    output_result = measure_quantum_state(quantum_state)
    print(output_result)

if __name__ == '__main__':
    main()
```
