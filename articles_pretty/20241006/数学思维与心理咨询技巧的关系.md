## 3. 核心算法原理 & 具体操作步骤
### 3.1 逻辑推理算法
逻辑推理算法是数学思维在心理咨询中的基础。通过逻辑推理，我们可以帮助个体识别和纠正认知偏差。以下是一个简单的逻辑推理算法示例：

```python
def identify_cognitive_bias(thoughts):
    # 定义认知偏差类型
    cognitive_biases = {
        "过度概括": "将一次负面事件视为普遍现象",
        "选择性概括": "只关注负面信息，忽略正面信息",
        "灾难化": "将负面事件视为最糟糕的情况",
        "个人化": "将负面事件归咎于自己"
    }
    
    # 分析个体的思维模式
    for thought in thoughts:
        for bias, description in cognitive_biases.items():
            if bias in thought:
                return bias, description
    return None, None
```

### 3.2 心理模型算法

```python
            model["环境"]["工作"].append(info["工作"])
        if "社交" in info:
            model["环境"]["社交"].append(info["社交"])
    
    return model
```
