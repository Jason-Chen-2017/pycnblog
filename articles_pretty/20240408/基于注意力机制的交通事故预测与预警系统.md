## 4. 项目实践：代码实例和详细解释说明

下面给出一个基于PyTorch实现的注意力机制交通事故预测模型的代码示例:

```python
import torch
import torch.nn as nn
import torch.optim as optim

class AttentionModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(AttentionModel, self).__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, batch_first=True)
        self.attention = nn.Linear(hidden_size, 1)
        self.fc = nn.Linear(hidden_size, output_size)
        self.sigmoid = nn.Sigmoid()

    def forward(self, x):
        batch_size, seq_len, _ = x.size()
        h0 = torch.zeros(1, batch_size, self.lstm.hidden_size).to(x.device)
        c0 = torch.zeros(1, batch_size, self.lstm.hidden_size).to(x.device)

        output, (hn, cn) = self.lstm(x, (h0, c0))
        attention_weights = self.sigmoid(self.attention(output))
        weighted_output = output * attention_weights
        pooled = weighted_output.mean(dim=1)
        out = self.fc(pooled)
        return out
```

在这个模型中,我们首先使用LSTM层提取输入数据的时序特征。然后,通过一个全连接层计算注意力权重,并将其应用到LSTM输出上,得到加权平均的特征向量。最后,将这个特征向量输入到一个全连接层,输出事故发生概率。

在训练过程中,我们可以使用交叉熵损失函数,并采用Adam优化器进行参数更新。通过不断优化模型参数,可以提高事故预测的准确性。
