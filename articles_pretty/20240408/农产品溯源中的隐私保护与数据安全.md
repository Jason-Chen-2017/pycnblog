## 4. 项目实践：代码实例和详细解释说明
下面我们来看一个基于区块链的农产品溯源系统的代码实例:

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@openzeppelin/contracts/token/ERC721/ERC721.sol";
import "@openzeppelin/contracts/utils/Counters.sol";

contract AgriTraceability is ERC721 {
    using Counters for Counters.Counter;
    Counters.Counter private _tokenIdCounter;

    struct ProductInfo {
        string productName;
        string farmLocation;
        string productionDate;
        string harvestDate;
        string transportationDetails;
        string storageConditions;
        string retailerInfo;
    }

    mapping(uint256 => ProductInfo) public productInfos;

    constructor() ERC721("AgriTrace", "AGT") {}

    function createProduct(
        string memory _productName,
        string memory _farmLocation,
        string memory _productionDate,
        string memory _harvestDate,
        string memory _transportationDetails,
        string memory _storageConditions,
        string memory _retailerInfo
    ) public returns (uint256) {
        uint256 tokenId = _tokenIdCounter.current();
        _tokenIdCounter.increment();
        _safeMint(msg.sender, tokenId);

        ProductInfo memory newProduct = ProductInfo({
            productName: _productName,
            farmLocation: _farmLocation,
            productionDate: _productionDate,
            harvestDate: _harvestDate,
            transportationDetails: _transportationDetails,
            storageConditions: _storageConditions,
            retailerInfo: _retailerInfo
        });
        productInfos[tokenId] = newProduct;

        return tokenId;
    }

    function getProductInfo(uint256 _tokenId)
        public
        view
        returns (ProductInfo memory)
    {
        return productInfos[_tokenId];
    }
}
```

在这个代码示例中,我们定义了一个名为`AgriTraceability`的智能合约,继承自OpenZeppelin的`ERC721`标准。

`ProductInfo`结构体用于存储农产品的各项信息,包括产品名称、农场位置、生产日期、收获日期、运输细节、存储条件和零售商信息。

`createProduct`函数允许用户创建一个新的农产品,并将其相关信息存储在区块链上。每个农产品都会被赋予一个唯一的`tokenId`,用于后续的查询和追溯。

`getProductInfo`函数则可以根据`tokenId`查询对应农产品的详细信息。

通过这种基于区块链的方式,我们可以确保农产品溯源信息的安全性和可靠性,同时也能够有效地保护参与方的隐私。
