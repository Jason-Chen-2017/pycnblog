首先，我们需要定义一个Questionnaire类，用于设计问卷和收集数据。

```python
class Questionnaire:
    def __init__(self, questions):
        self.questions = questions
        self.responses = []

    def add_response(self, response):
        self.responses.append(response)

    def get_average_scores(self):
        scores = [0] * len(self.questions)
        for response in self.responses:
            for i, score in enumerate(response):
                scores[i] += score
        return [score / len(self.responses) for score in scores]
```

然后，我们需要定义一个Survey类，用于分析数据和呈现结果。

```python
class Survey:
    def __init__(self, questionnaire):
        self.questionnaire = questionnaire

    def get_average_scores(self):
        return self.questionnaire.get_average_scores()

    def print_results(self):
        average_scores = self.get_average_scores()
        for i, score in enumerate(average_scores):
            print(f"Question {i+1}: {score:.2f}")
```

最后，我们可以创建一个问卷，收集数据，分析数据，和呈现结果。

```python

# 收集数据

questionnaire.add_response([5, 5])

# 分析数据和呈现结果

survey.print_results()
```
