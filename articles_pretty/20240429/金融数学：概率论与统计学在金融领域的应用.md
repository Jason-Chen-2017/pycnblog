## 4. 数学模型和公式详细讲解举例说明

### 4.1 几何布朗运动模型

几何布朗运动模型是一种常见的随机过程模型，用于描述资产价格的变动。其数学表达式为：

$$
dS_t = \mu S_t dt + \sigma S_t dW_t
$$

其中：

*   $S_t$ 表示t时刻的资产价格。
*   $\mu$ 表示资产的预期收益率。
*   $\sigma$ 表示资产的波动率。
*   $W_t$ 表示标准布朗运动。

### 4.2 Black-Scholes期权定价模型

Black-Scholes期权定价模型是金融领域最著名的模型之一，用于对欧式期权进行定价。其公式为：

$$
C = S_0 N(d_1) - Ke^{-rT} N(d_2)
$$

其中：

*   $C$ 表示欧式看涨期权的价格。
*   $S_0$ 表示标的资产的当前价格。
*   $K$ 表示期权的行权价格。
*   $r$ 表示无风险利率。
*   $T$ 表示期权的到期时间。
*   $N(x)$ 表示标准正态分布的累积分布函数。
*   $d_1 = \frac{ln(S_0/K) + (r + \sigma^2/2)T}{\sigma \sqrt{T}}$
*   $d_2 = d_1 - \sigma \sqrt{T}$

### 5.1 蒙特卡洛模拟股票价格

```python
import numpy as np

# 设置参数
S0 = 100  # 初始价格
mu = 0.05  # 预期收益率
sigma = 0.2  # 波动率
T = 1  # 时间期限
N = 10000  # 模拟次数

# 生成随机数
dt = T / N
dW = np.random.normal(0, np.sqrt(dt), (N, 1))

# 模拟股票价格路径
S = S0 * np.exp((mu - 0.5 * sigma**2) * dt + sigma * dW).cumprod(axis=0)

# 计算统计特征
mean_price = np.mean(S[-1])
std_price = np.std(S[-1])

print("平均价格:", mean_price)
print("标准差:", std_price)
```
