## 2. 核心概念与联系

### 2.1 意图

意图是指用户希望通过自然语言表达达到的目标或目的。例如，用户说“我想订一张明天从北京到上海的机票”，其意图是预订机票。

### 2.2 实体

实体是指自然语言语句中包含的具有特定意义的词汇或短语，例如人名、地名、时间等。例如，在“我想订一张明天从北京到上海的机票”这句话中，“北京”、“上海”、“明天”都是实体。

### 2.3 意图识别

意图识别是指将用户的自然语言输入映射到预定义的意图集合中的过程。例如，将“我想订一张明天从北京到上海的机票”这句话识别为“订机票”的意图。

### 2.4 实体识别

实体识别是指从自然语言语句中识别出实体的过程。例如，从“我想订一张明天从北京到上海的机票”这句话中识别出“北京”、“上海”、“明天”等实体。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 基于 Python 的意图识别示例

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC

# 加载训练数据
train_data = [
    ("我想订一张机票", "订机票"),
    ("我想查一下天气", "查天气"),
    ("我想听音乐", "听音乐"),
]

# 提取特征
vectorizer = TfidfVectorizer()
X_train = vectorizer.fit_transform([text for text, label in train_data])

# 训练模型
model = SVC()
model.fit(X_train, [label for text, label in train_data])

# 预测意图
test_text = "我想去上海"
X_test = vectorizer.transform([test_text])
predicted_intent = model.predict(X_test)[0]

print(f"预测意图：{predicted_intent}")
```

### 5.2 代码解释

1. 使用 `TfidfVectorizer` 提取文本特征，将文本转化为数值向量。
2. 使用 `SVC` 训练支持向量机模型，对意图进行分类。
3. 使用训练好的模型对新的文本进行预测，输出预测的意图。
