## 4. 数学模型和公式详细讲解举例说明

### 4.1 线性回归

线性回归是一种监督学习算法，它用于预测连续值输出。线性回归模型的数学公式如下：

$$
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n
$$

其中，$y$ 是预测值，$x_1, x_2, ..., x_n$ 是输入变量，$\beta_0, \beta_1, ..., \beta_n$ 是模型参数。

### 4.2 逻辑回归

逻辑回归是一种监督学习算法，它用于预测二元分类输出。逻辑回归模型的数学公式如下：

$$
P(y=1|x) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n)}}
$$

其中，$P(y=1|x)$ 是预测为正例的概率，$x_1, x_2, ..., x_n$ 是输入变量，$\beta_0, \beta_1, ..., \beta_n$ 是模型参数。

```python
import pandas as pd
from sklearn.linear_model import LinearRegression

# 加载数据
data = pd.read_csv('toy_sales.csv')

# 划分特征和标签
X = data[['price', 'advertising']]
y = data['sales']

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 预测新数据
new_data = [[10, 100]]
predicted_sales = model.predict(new_data)

# 打印预测结果
print(predicted_sales)
```

* 首先，加载玩具销售数据，包括价格、广告费用和销量。
* 然后，将数据划分为特征（价格和广告费用）和标签（销量）。
* 创建一个线性回归模型，并使用训练数据对其进行训练。
* 最后，使用训练好的模型预测新数据（价格为10，广告费用为100）的销量。
