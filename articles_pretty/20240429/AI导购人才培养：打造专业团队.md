## 4. 项目实践

### 4.1 代码实例

以下是一个简单的基于协同过滤的推荐算法代码示例：

```python
import pandas as pd
from sklearn.neighbors import NearestNeighbors

# 读取用户评分数据
ratings = pd.read_csv('ratings.csv')

# 创建最近邻模型
model = NearestNeighbors(metric='cosine', algorithm='brute')
model.fit(ratings)

# 获取与指定用户最相似的用户
distances, indices = model.kneighbors(ratings.iloc[0, :].values.reshape(1, -1), n_neighbors=5)

# 推荐相似用户喜欢的商品
recommendations = ratings.iloc[indices[0], :].mean(axis=0).sort_values(ascending=False)[:5]

print(recommendations)
```

### 4.2 解释说明

上述代码首先读取用户评分数据，然后创建了一个最近邻模型，用于计算用户之间的相似度。接着，获取与指定用户最相似的用户，并推荐相似用户喜欢的商品。
