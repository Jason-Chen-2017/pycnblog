### 4.2 标签传播的关键公式
#### 4.2.1 转移概率矩阵的计算
$$
P_{ij} = \frac{w_{ij}}{\sum_{k=1}^{n}w_{ik}}
$$
其中，$w_{ij}$表示节点$i$到节点$j$的边权重，$n$为图中节点的数量。

#### 4.2.2 标签传播的迭代公式
$$
\mathbf{Y}^{(t+1)} = \alpha \mathbf{P} \mathbf{Y}^{(t)} + (1-\alpha) \mathbf{Y}^{(0)}
$$

其中，$\mathbf{Y}^{(t)}$表示第$t$次迭代后的标签矩阵，$\alpha$为平滑因子，$\mathbf{P}$为转移概率矩阵，$\mathbf{Y}^{(0)}$为初始标签矩阵。

#### 4.2.3 收敛性分析
标签传播算法可以证明在一定条件下能够收敛到平稳分布，即满足：

$$
\mathbf{Y}^{*} = \mathbf{P} \mathbf{Y}^{*}
$$

其中，$\mathbf{Y}^{*}$为收敛后的标签矩阵。


#### 5.1.3 初始化标签
```python

```

#### 5.1.4 标签传播迭代过程
```python
def label_propagation(G, labels, max_iter=100, alpha=0.85):
    # 初始化标签矩阵
    Y = np.zeros((len(G.nodes()), 2))
    for i, node in enumerate(G.nodes()):
        if node in labels:
            Y[i, labels[node]] = 1.0
    
    # 计算转移概率矩阵
    P = nx.to_numpy_array(G)
    P = P / P.sum(axis=1, keepdims=True)
    
    # 迭代传播标签
    for _ in range(max_iter):
        Y_prev = Y.copy()
        Y = alpha * np.dot(P, Y) + (1 - alpha) * Y
        if np.allclose(Y, Y_prev):
            break
    
    return Y
```

#### 5.1.5 运行算法并可视化结果
```python
Y = label_propagation(G, labels)
print(Y)

pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True, node_size=500, node_color=Y[:,0], cmap=plt.cm.coolwarm)
plt.show()
```

#### 5.2.1 构建图模型

#### 5.2.2 初始化标签
定义一个字典，键为节点名称，值为对应的初始标签（0或1）。

#### 5.2.3 标签传播迭代过程
定义一个函数`label_propagation`，输入图模型、初始标签、最大迭代次数和平滑因子，输出收敛后的标签矩阵。

函数内部首先初始化标签矩阵，然后计算转移概率矩阵。接着进入迭代过程，每次迭代根据标签传播公式更新标签矩阵，直到达到最大迭代次数或标签矩阵收敛。

#### 5.2.4 运行算法并可视化结果
