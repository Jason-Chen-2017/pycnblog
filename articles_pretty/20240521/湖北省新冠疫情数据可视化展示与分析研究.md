## 4. 数学模型和公式详细讲解举例说明

### 4.1 SIR 模型

SIR 模型是一种经典的传染病动力学模型，它将人群分为三类：易感者 (S)、感染者 (I) 和康复者 (R)。模型假设：

* 易感者与感染者接触后，有一定概率被感染。
* 感染者经过一段时间后会康复，并获得免疫力。

SIR 模型的数学表达式如下：

$$
\begin{aligned}
\frac{dS}{dt} &= -\beta SI \\
\frac{dI}{dt} &= \beta SI - \gamma I \\
\frac{dR}{dt} &= \gamma I
\end{aligned}
$$

其中，$\beta$ 表示传染率，$\gamma$ 表示康复率。

**举例说明:**

假设湖北省初期有 1000 万易感者，100 名感染者，传染率为 0.01，康复率为 0.1。根据 SIR 模型，可以计算出不同时间节点的易感者、感染者和康复者数量。

### 4.2 SEIR 模型

SEIR 模型是在 SIR 模型的基础上，增加了潜伏期 (E) 的概念。模型假设：

* 易感者与感染者接触后，首先进入潜伏期。
* 潜伏期结束后，感染者才会出现症状并具有传染性。

SEIR 模型的数学表达式如下：

$$
\begin{aligned}
\frac{dS}{dt} &= -\beta SI \\
\frac{dE}{dt} &= \beta SI - \sigma E \\
\frac{dI}{dt} &= \sigma E - \gamma I \\
\frac{dR}{dt} &= \gamma I
\end{aligned}
$$

其中，$\sigma$ 表示潜伏期结束的速率。

**举例说明:**

假设湖北省初期有 1000 万易感者，100 名感染者，传染率为 0.01，潜伏期为 7 天，康复率为 0.1。根据 SEIR 模型，可以计算出不同时间节点的易感者、潜伏者、感染者和康复者数量。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 数据获取与处理

```python
import pandas as pd

# 从湖北省卫健委官方网站获取数据
url = "http://wjw.hubei.gov.cn/fbjd/dtyw/index.shtml"
tables = pd.read_html(url)

# 选择包含疫情数据的表格
df = tables[0]

# 数据清洗
df = df.dropna()
df = df.drop_duplicates()

# 数据格式转换
df["日期"] = pd.to_datetime(df["日期"])
df["确诊病例"] = df["确诊病例"].astype(int)
```

### 5.2 可视化展示

```python
import matplotlib.pyplot as plt

# 绘制确诊病例数随时间的变化趋势
plt.plot(df["日期"], df["确诊病例"])
plt.xlabel("日期")
plt.ylabel("确诊病例数")
plt.title("湖北省新冠疫情确诊病例数变化趋势")
plt.show()
```
