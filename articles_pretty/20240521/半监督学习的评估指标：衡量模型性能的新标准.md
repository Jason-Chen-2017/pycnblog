## 1. 背景介绍

### 1.1 机器学习的演变：从监督到半监督

机器学习近年来取得了显著的进展，其应用已渗透到各个领域。从传统的监督学习到无监督学习，再到如今备受关注的半监督学习，机器学习的范式不断演变，以应对日益增长的数据量和复杂性。

监督学习依赖于大量的标注数据，这在许多实际应用中是昂贵且难以获得的。无监督学习则完全不需要标注数据，但其学习到的信息往往较为有限。半监督学习作为一种折衷方案，利用少量标注数据和大量未标注数据进行学习，在许多场景下展现出优于监督学习和无监督学习的性能。

### 1.2 半监督学习的优势和挑战

半监督学习的优势在于：

* **降低标注成本：**  利用未标注数据进行学习，减少了对标注数据的依赖，降低了数据成本。
* **提升模型泛化能力：** 未标注数据可以提供更多关于数据分布的信息，帮助模型更好地泛化到未见数据。
* **适用于数据标注困难的场景：** 在一些领域，例如医学影像分析，数据标注需要专业知识且成本高昂，半监督学习提供了一种有效的解决方案。

然而，半监督学习也面临着一些挑战：

* **模型性能评估：** 由于使用了未标注数据，传统的监督学习评估指标可能不再适用。
* **算法选择和参数调整：**  不同的半监督学习算法对数据分布和任务类型有不同的要求，需要仔细选择和调整参数。
* **未标注数据的质量：**  未标注数据的质量会影响模型的性能，需要进行数据清洗和预处理。

### 1.3  评估指标的重要性

为了有效地评估半监督学习模型的性能，我们需要新的评估指标，这些指标应能够：

* **准确反映模型对未标注数据的利用程度。**
* **与实际应用场景相一致。**
* **易于理解和使用。**

## 2. 核心概念与联系

### 2.1  传统监督学习评估指标

在深入探讨半监督学习评估指标之前，让我们先回顾一下传统的监督学习评估指标：

* **准确率 (Accuracy):**  正确预测的样本数占总样本数的比例。
* **精确率 (Precision):**  预测为正类的样本中真正为正类的比例。
* **召回率 (Recall):** 真正为正类的样本中被正确预测为正类的比例。
* **F1 score:**  精确率和召回率的调和平均值。
* **ROC 曲线和 AUC:**  ROC 曲线描述了模型在不同分类阈值下的真阳性率和假阳性率，AUC 是 ROC 曲线下的面积，代表模型的整体性能。

### 2.2 半监督学习评估指标的特殊性

传统的监督学习评估指标主要关注模型对标注数据的预测能力，而忽略了未标注数据的作用。在半监督学习中，未标注数据对模型的训练和性能提升至关重要，因此需要新的评估指标来衡量模型对未标注数据的利用程度。

### 2.3  半监督学习评估指标的分类

根据评估目标的不同，半监督学习评估指标可以分为以下几类：

* **基于聚类的指标:** 衡量模型对未标注数据进行聚类的效果，例如 Adjusted Rand Index (ARI) 和 Normalized Mutual Information (NMI)。
* **基于一致性的指标:** 衡量模型对未标注数据预测的一致性，例如 Entropy Minimization 和 Mean Teacher。
* **基于迁移学习的指标:** 衡量模型将从标注数据学到的知识迁移到未标注数据的能力，例如 Transductive Support Vector Machine (TSVM) 和 Transductive Graph Learning。
