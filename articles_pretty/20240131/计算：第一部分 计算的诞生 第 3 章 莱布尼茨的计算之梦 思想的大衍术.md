```python

    binary = ''
    while decimal > 0:
        binary = str(decimal % 2) + binary
        decimal //= 2
    return binary
```

```python
def derivative(f, x, h=0.0001):
    return (f(x+h) - f(x)) / h

def integral(f, a, b, n=1000):
    dx = (b - a) / n
    x = a
    sum = 0
    for i in range(n):

```
