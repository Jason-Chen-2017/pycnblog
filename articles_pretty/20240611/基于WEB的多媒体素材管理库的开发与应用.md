以下是一个简单的代码示例，演示了如何实现文件的上传和下载功能：

```python
from django.http import HttpResponse
from django.views.decorators.csrf import csrf_exempt

@csrf_exempt
def upload_file(request):
    if request.method == 'POST':
        # 获取上传的文件
        file = request.FILES['file']
        # 保存文件
        with open('uploads/' + file.name, 'wb+') as f:
            for chunk in file.chunks():
                f.write(chunk)
        return HttpResponse('文件上传成功')
    else:
        return HttpResponse('无效的请求方法')

@csrf_exempt
def download_file(request, file_path):
    try:
        # 打开文件
        with open('uploads/' + file_path, 'rb') as f:
            content = f.read()
        # 设置响应头
        response = HttpResponse(content, content_type='application/octet-stream')
        response['Content-Disposition'] = 'attachment; filename="' + file_path + '"'
        return response
    except FileNotFoundError:
        return HttpResponse('文件未找到')
```

在上述代码中，我们定义了两个视图函数`upload_file`和`download_file`，分别用于实现文件的上传和下载功能。`upload_file`函数接收一个`POST`请求，获取上传的文件，并将文件保存到指定的目录下。`download_file`函数接收一个`GET`请求，根据传入的文件路径下载文件，并设置响应头，使得浏览器能够正确下载文件。
