4.2 数学公式
在线点餐订餐系统需要使用一些数学公式来计算订单的价格、折扣等信息。以下是一些常用的数学公式：
- 订单总价 = 菜品总价 - 折扣金额
- 菜品总价 = 菜品数量 × 菜品价格
- 折扣金额 = 订单总价 × 折扣率

4.3 举例说明
以下是一个具体的例子，说明如何使用数学公式计算订单的价格和折扣：
1. 假设有一个订单，包含以下菜品：
    - 菜品 1：数量 2，价格 10 元
    - 菜品 2：数量 1，价格 20 元
2. 假设该订单享受 8 折优惠，折扣率为 0.8
3. 计算订单的总价：
    - 菜品总价 = 2 × 10 + 1 × 20 = 40 元
    - 订单总价 = 40 × 0.8 = 32 元
4. 计算折扣金额：
    - 折扣金额 = 32 × 0.8 = 25.6 元
5. 最终订单价格 = 32 - 25.6 = 6.4 元

以下是一个简单的在线点餐订餐系统的服务器代码实例，演示了如何实现订单的处理和保存：
```python
from flask import Flask, request, jsonify
import json

app = Flask(__name__)

# 订单处理函数
@app.route('/order', methods=['POST'])
def process_order():
    # 从请求中获取订单信息
    data = request.get_json()

    # 验证订单信息
    if not all(key in data for key in ['customer_name', 'customer_phone', 'order_items']):
        return jsonify({'message': '订单信息不完整'}), 400

    # 处理订单
    order_id = generate_order_id()
    total_price = sum(item['price'] for item in data['order_items'])
    order = {
        'order_id': order_id,
        'customer_name': data['customer_name'],
        'customer_phone': data['customer_phone'],
        'order_items': data['order_items'],
        'total_price': total_price
    }

    # 保存订单到数据库
    with open('orders.json', 'a') as f:
        json.dump(order, f, indent=4)

    # 返回订单处理结果
    return jsonify({'message': '订单处理成功', 'order_id': order_id})

# 生成订单 ID 函数
def generate_order_id():
    import uuid
    return str(uuid.uuid4())

# 启动服务器
if __name__ == '__main__':
    app.run(debug=True)
```
在上述代码中，我们使用 `Flask` 框架实现了一个简单的在线点餐订餐系统的服务器。服务器接收来自微信小程序的订单请求，并将订单信息保存到本地的 `orders.json` 文件中。同时，我们还使用了 `uuid` 模块生成订单 ID，并返回订单处理结果给微信小程序。
