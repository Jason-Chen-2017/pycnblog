### 5.2 代码实例

```langchain
class Calculator {
    function add(x, y) {
        return x + y;
    }

    function sub(x, y) {
        return x - y;
    }

    function mul(x, y) {
        return x * y;
    }

    function div(x, y) {
        if (y != 0) {
            return x / y;
        } else {
            throw new Exception("Division by zero");
        }
    }
}

var calc = new Calculator();
var result = calc.invoke("add", 1, 2);
print(result);
```

然后，我们创建了一个`Calculator`的实例`calc`，并通过`invoke`方法调用了`add`方法，进行了一次加法运算。

最后，我们打印出了运算的结果。
