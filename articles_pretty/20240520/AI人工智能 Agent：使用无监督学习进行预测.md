## 1. 背景介绍

### 1.1 人工智能 Agent 的崛起

近年来，人工智能（AI）取得了显著的进展，特别是在机器学习领域。机器学习算法使计算机能够从数据中学习并做出预测，而无需明确编程。其中一个最令人兴奋的领域是人工智能 Agent 的发展，这些 Agent 能够感知环境、做出决策并采取行动以实现目标。

### 1.2 无监督学习的优势

传统的机器学习方法通常依赖于监督学习，其中算法在标记数据上进行训练，这意味着每个数据点都与一个预定义的类别或值相关联。然而，在许多现实世界的情况下，获取标记数据既昂贵又耗时。无监督学习提供了一种替代方案，它允许算法从未标记的数据中学习，识别数据中的模式和结构。
