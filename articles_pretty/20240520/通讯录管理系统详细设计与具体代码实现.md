## 2. 核心概念与联系

### 2.1  数据结构

通讯录管理系统中的核心数据结构是联系人信息，通常包括姓名、电话号码、电子邮件地址、家庭住址等。这些信息可以存储在数据库、文本文件或其他数据存储介质中。

### 2.2 功能模块

通讯录管理系统通常包括以下功能模块：

* 添加联系人：允许用户添加新的联系人信息。
* 删除联系人：允许用户删除已有的联系人信息。
* 修改联系人：允许用户修改已有的联系人信息。
* 查询联系人：允许用户根据姓名、电话号码或其他条件查询联系人信息。
* 导入/导出联系人：允许用户导入或导出联系人信息，以便与其他系统进行数据交换。

### 2.3  用户界面

通讯录管理系统的用户界面可以是命令行界面、图形用户界面或 Web 界面。用户界面应该简洁易用，方便用户进行各种操作。

### 2.4 系统架构

通讯录管理系统的系统架构可以采用 C/S 架构或 B/S 架构。C/S 架构是指客户端/服务器架构，客户端负责用户界面和数据输入，服务器负责数据存储和处理。B/S 架构是指浏览器/服务器架构，用户通过浏览器访问服务器端的应用程序。


## 3. 核心算法原理具体操作步骤

### 3.1 添加联系人

添加联系人操作的步骤如下：

1. 用户输入联系人信息，包括姓名、电话号码、电子邮件地址等。
2. 系统验证用户输入的信息是否合法。
3. 系统将联系人信息存储到数据库或其他数据存储介质中。

### 3.2 删除联系人

删除联系人操作的步骤如下：

1. 用户选择要删除的联系人。
2. 系统从数据库或其他数据存储介质中删除该联系人信息。

### 3.3 修改联系人

修改联系人操作的步骤如下：

1. 用户选择要修改的联系人。
2. 用户修改联系人信息。
3. 系统验证用户输入的信息是否合法。
4. 系统更新数据库或其他数据存储介质中的联系人信息。

### 3.4 查询联系人

查询联系人操作的步骤如下：

1. 用户输入查询条件，例如姓名、电话号码或其他条件。
2. 系统根据查询条件从数据库或其他数据存储介质中检索匹配的联系人信息。
3. 系统将检索到的联系人信息显示给用户。

### 3.5 导入/导出联系人

导入联系人操作的步骤如下：

1. 用户选择要导入的联系人文件。
2. 系统解析联系人文件，提取联系人信息。
3. 系统将联系人信息存储到数据库或其他数据存储介质中。

导出联系人操作的步骤如下：

1. 用户选择要导出的联系人格式。
2. 系统从数据库或其他数据存储介质中检索联系人信息。
3. 系统将联系人信息转换成指定的格式，并保存到文件中。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 Python 实现

```python
import sqlite3

# 连接数据库
conn = sqlite3.connect('contacts.db')
cursor = conn.cursor()

# 创建联系人表
cursor.execute('''
CREATE TABLE IF NOT EXISTS contacts (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    phone TEXT NOT NULL,
    email TEXT
)
''')

# 添加联系人
def add_contact(name, phone, email):
    cursor.execute('''
    INSERT INTO contacts (name, phone, email)
    VALUES (?, ?, ?)
    ''', (name, phone, email))
    conn.commit()
    print(f"联系人 {name} 添加成功！")

# 删除联系人
def delete_contact(name):
    cursor.execute('''
    DELETE FROM contacts WHERE name = ?
    ''', (name,))
    conn.commit()
    print(f"联系人 {name} 删除成功！")

# 修改联系人
def update_contact(name, phone, email):
    cursor.execute('''
    UPDATE contacts
    SET phone = ?, email = ?
    WHERE name = ?
    ''', (phone, email, name))
    conn.commit()
    print(f"联系人 {name} 修改成功！")

# 查询联系人
def query_contact(name):
    cursor.execute('''
    SELECT * FROM contacts WHERE name = ?
    ''', (name,))
    contact = cursor.fetchone()
    if contact:
        print(f"姓名：{contact[1]}")
        print(f"电话：{contact[2]}")
        print(f"邮箱：{contact[3]}")
    else:
        print(f"未找到联系人 {name}")

# 导入联系人
def import_contacts(file_path):
    with open(file_path, 'r') as f:
        for line in f:
            name, phone, email = line.strip().split(',')
            add_contact(name, phone, email)
    print(f"联系人导入成功！")

# 导出联系人
def export_contacts(file_path):
    cursor.execute('''
    SELECT * FROM contacts
    ''')
    contacts = cursor.fetchall()
    with open(file_path, 'w') as f:
        for contact in contacts:
            f.write(f"{contact[1]},{contact[2]},{contact[3]}\n")
    print(f"联系人导出成功！")

# 测试
add_contact('张三', '13800000000', 'zhangsan@example.com')
delete_contact('张三')
add_contact('李四', '13900000000', 'lisi@example.com')
update_contact('李四', '13911111111', 'lisi@example.net')
query_contact('李四')
import_contacts('contacts.txt')
export_contacts('contacts_export.txt')

# 关闭数据库连接
conn.close()
```

### 5.2 代码解释

* `sqlite3` 模块用于操作 SQLite 数据库。
* `contacts.db` 是数据库文件名。
* `contacts` 表用于存储联系人信息，包括 `id`、`name`、`phone` 和 `email` 列。
* `add_contact()` 函数用于添加联系人。
* `delete_contact()` 函数用于删除联系人。
* `update_contact()` 函数用于修改联系人。
* `query_contact()` 函数用于查询联系人。
* `import_contacts()` 函数用于导入联系人。
* `export_contacts()` 函数用于导出联系人。
