### 4.1 知识图谱构建实践

以下是使用Python实现商品详情页知识图谱构建的代码示例：

```python
import spacy
import networkx as nx

# 加载自然语言处理模型
nlp = spacy.load("en_core_web_sm")

# 商品详情页文本
text = """
This is a sample product description. The product has a 5-inch screen, 64GB storage, and a 12MP camera. It comes with a charger and a user manual.
"""

# 实体识别和关系抽取
doc = nlp(text)
entities = [ent for ent in doc.ents]
relations = [(token.head, token, token.children) for token in doc if token.dep_ in ("attr", "dobj")]

# 构建知识图谱
graph = nx.DiGraph()
graph.add_nodes_from(entities)
for relation in relations:
    head, token, children = relation
    graph.add_edges_from([(head, child) for child in children])

# 可视化知识图谱
nx.draw(graph, with_labels=True)
```
