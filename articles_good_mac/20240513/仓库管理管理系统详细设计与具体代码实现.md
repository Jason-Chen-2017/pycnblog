#### 5.1.1 SKU信息表设计与创建
```sql
CREATE TABLE sku_info (
  id INT PRIMARY KEY AUTO_INCREMENT,
  sku_code VARCHAR(20) NOT NULL,
  sku_name VARCHAR(100) NOT NULL,
  category VARCHAR(50),
  unit VARCHAR(10),
  price DECIMAL(10,2),
  supplier_id INT,
  barcode VARCHAR(50), 
  created_time DATETIME,
  updated_time DATETIME,
  INDEX(sku_code)
);
```

```sql  
CREATE TABLE outbound_order (
  id INT PRIMARY KEY AUTO_INCREMENT,

  qty INT NOT NULL,  
  created_time DATETIME
);  
```
#### 5.3.2 入库上架与出库拣货核心代码
```java
// 创建入库单
String inboundSql = "INSERT INTO inbound_order (order_no, sku_id, warehouse_id, qty, created_time) VALUES (?, ?, ?, ?, NOW())";
```
