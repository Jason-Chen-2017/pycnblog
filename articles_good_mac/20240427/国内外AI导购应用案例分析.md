## 4. 数学模型和公式详细讲解举例说明

### 4.1 余弦相似度

余弦相似度是计算向量之间相似度的一种常用方法。其公式如下：

$$
cos(\theta) = \frac{\vec{A} \cdot \vec{B}}{||\vec{A}|| \cdot ||\vec{B}||}
$$

其中，$\vec{A}$ 和 $\vec{B}$ 表示两个向量，$\theta$ 表示两个向量之间的夹角。余弦相似度的取值范围为 $[-1, 1]$，值越接近 1，表示两个向量越相似。

**举例说明：**

假设有两个用户 A 和 B，他们的购买记录如下：

| 用户 | 商品 1 | 商品 2 | 商品 3 |
|---|---|---|---|
| A | 1 | 0 | 1 |
| B | 1 | 1 | 0 |

将用户的购买记录转换成向量表示：

$$
\vec{A} = (1, 0, 1) \\
\vec{B} = (1, 1, 0)
$$

计算用户 A 和 B 之间的余弦相似度：

$$
cos(\theta) = \frac{(1, 0, 1) \cdot (1, 1, 0)}{\sqrt{1^2 + 0^2 + 1^2} \cdot \sqrt{1^2 + 1^2 + 0^2}} = \frac{1}{\sqrt{2} \cdot \sqrt{2}} = 0.5
$$

因此，用户 A 和 B 之间的相似度为 0.5。

### 5.1 Python 代码示例

以下是一个使用 Python 实现协同过滤算法的代码示例：

```python
import numpy as np

def cosine_similarity(a, b):
  """
  计算两个向量之间的余弦相似度。
  """
  return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

def collaborative_filtering(user_item_matrix, user_id):
  """
  使用协同过滤算法推荐商品。
  """
  # 计算用户相似度
  similarities = []
  for other_user_id in range(user_item_matrix.shape[0]):
    if other_user_id != user_id:
      similarity = cosine_similarity(user_item_matrix[user_id], user_item_matrix[other_user_id])
      similarities.append((other_user_id, similarity))

  # 寻找相似用户
  similarities.sort(key=lambda x: x[1], reverse=True)
  k = 5  # 选择相似度最高的 k 个用户
  similar_users = similarities[:k]

  # 推荐商品
  recommendations = []
  for similar_user_id, _ in similar_users:
    for item_id in range(user_item_matrix.shape[1]):
      if user_item_matrix[user_id, item_id] == 0 and user_item_matrix[similar_user_id, item_id] > 0:
        recommendations.append(item_id)

  return recommendations
```
