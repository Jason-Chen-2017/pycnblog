## 4. 数学模型和公式

### 4.1 TF-IDF

TF-IDF (Term Frequency-Inverse Document Frequency) 是一种常用的文本特征提取方法，用于评估一个词语在文档集合中的重要程度。TF-IDF 的计算公式如下：

$$
\text{TF-IDF}(t, d) = \text{TF}(t, d) \times \text{IDF}(t)
$$

其中：

*   $t$ 表示词语
*   $d$ 表示文档
*   $\text{TF}(t, d)$ 表示词语 $t$ 在文档 $d$ 中出现的频率
*   $\text{IDF}(t)$ 表示词语 $t$ 的逆文档频率，计算公式如下：

$$
\text{IDF}(t) = \log \frac{N}{df(t)}
$$

其中：

*   $N$ 表示文档集合中总的文档数量
*   $df(t)$ 表示包含词语 $t$ 的文档数量

### 4.2 Word2Vec

Word2Vec 是一种词嵌入模型，可以将词语表示为低维向量。Word2Vec 模型可以通过学习词语的上下文信息，将语义相似的词语映射到向量空间中相近的位置。

## 5. 项目实践

### 5.1 代码实例 (Python)

```python
import nltk
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC

# 加载数据
data = [
    ("我想买一部手机", "交易"),
    ("苹果手机怎么样", "信息获取"),
    ("百度官网", "导航"),
]

# 分词
tokenizer = nltk.word_tokenize

# 特征提取
vectorizer = TfidfVectorizer(tokenizer=tokenizer)
X = vectorizer.fit_transform([text for text, label in data])

# 模型训练
model = SVC()
model.fit(X, [label for text, label in data])

# 模型预测
new_text = "华为手机价格"
new_X = vectorizer.transform([new_text])
predicted_label = model.predict(new_X)[0]

print(f"预测意图: {predicted_label}")
```

### 5.2 代码解释

*   **nltk**：自然语言处理工具包，用于分词。
*   **TfidfVectorizer**：TF-IDF 特征提取器。
*   **SVC**：支持向量机分类器。
