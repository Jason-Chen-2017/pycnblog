$$
\cos(\theta) = \frac{\mathbf{A} \cdot \mathbf{B}}{\|\mathbf{A}\| \|\mathbf{B}\|}
$$

其中，$\mathbf{A}$ 和 $\mathbf{B}$ 是两个向量，$\theta$ 是它们之间的夹角。余弦相似度的取值范围为 $[-1, 1]$，值越接近 1 表示两个向量越相似。

### 5.1 使用 Python 实现协同过滤算法

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
data = pd.read_csv('toy_ratings.csv')

# 计算用户之间的相似度矩阵
user_similarity = cosine_similarity(data.T)

# 找到与目标用户兴趣最相似的 K 个用户
target_user = 1
k = 5
similar_users = user_similarity[target_user].argsort()[-k:]

# 推荐这些用户喜欢的玩具给目标用户
recommendations = []
for user in similar_users:
    user_ratings = data.iloc[user]
    recommendations.extend(user_ratings[user_ratings > 0].index.tolist())

# 去重并排序
recommendations = list(set(recommendations))
recommendations.sort(key=lambda x: data.loc[target_user, x], reverse=True)

# 打印推荐结果
print(recommendations)
```

1.  首先，使用 pandas 库加载玩具评分数据。
2.  然后，使用 sklearn 库中的 cosine\_similarity 函数计算用户之间的相似度矩阵。
3.  接下来，找到与目标用户兴趣最相似的 K 个用户。
4.  然后，推荐这些用户喜欢的玩具给目标用户。
