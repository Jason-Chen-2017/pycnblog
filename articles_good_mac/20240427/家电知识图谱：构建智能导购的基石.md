## 4. 数学模型和公式详细讲解举例说明

### 4.1 关系抽取

关系抽取可以使用**依存句法分析**来实现。依存句法分析将句子中的词语视为节点，词语之间的语法关系视为边，从而形成一棵依存句法树。例如，句子“海尔BCD-456WDIEU1冰箱拥有风冷无霜功能”的依存句法树如下：

```
       拥有
      /    \
    冰箱   功能
   /   \      \ 
 海尔  型号   风冷无霜
```

通过分析依存句法树，可以抽取出实体“海尔BCD-456WDIEU1冰箱”和“风冷无霜功能”之间的“拥有”关系。

### 4.2 实体对齐

实体对齐可以使用**Jaccard相似度**来衡量两个实体的相似程度。Jaccard相似度计算公式如下：

$$
Jaccard(A, B) = \frac{|A \cap B|}{|A \cup B|}
$$

其中，A和B分别表示两个实体的属性集合。Jaccard相似度越高，表示两个实体越相似，越有可能表示同一个实体。
