## 4. 数学模型和公式详细讲解举例说明

### 4.1 词袋模型

词袋模型将文本表示为一个向量，向量的每个元素表示一个词语在文本中出现的次数。例如，句子“我想订一张去北京的机票”可以表示为向量 [1, 1, 1, 1, 0, ...], 其中向量的第一、二、三、四個元素分别表示“我”、“想”、“订”、“机票”出现的次数。

### 4.2 TF-IDF

TF-IDF 是一种用于评估词语重要性的方法，它考虑了词语在文档中出现的频率和词语在整个语料库中出现的频率。TF-IDF 的计算公式如下：

$$
tfidf(t, d, D) = tf(t, d) * idf(t, D)
$$

其中，$tf(t, d)$ 表示词语 $t$ 在文档 $d$ 中出现的频率，$idf(t, D)$ 表示词语 $t$ 的逆文档频率。

### 4.3 Word2Vec

Word2Vec 是一种词嵌入模型，它将词语表示为一个低维向量，向量之间的距离反映了词语之间的语义相似度。

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 和 scikit-learn 库实现的基于机器学习的意图识别模型的示例代码：

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC

# 训练数据
train_data = [
    ("我想订一张去北京的机票", "订机票"),
    ("我想订一个酒店", "订酒店"),
    ("明天的天气怎么样", "查天气"),
]

# 特征提取
vectorizer = TfidfVectorizer()
train_features = vectorizer.fit_transform([text for text, label in train_data])

# 模型训练
model = SVC()
model.fit(train_features, [label for text, label in train_data])

# 预测
test_text = "我想知道北京的天气"
test_features = vectorizer.transform([test_text])
predicted_label = model.predict(test_features)[0]

print(f"预测意图: {predicted_label}")
```
