## 2. 核心概念与联系

### 2.1 伯努利分布

伯努利分布是最简单的概率分布之一，用于描述只有两种可能结果的随机试验，例如抛硬币（正面或反面）、产品检验（合格或不合格）等。

- **参数：** $p$，表示成功的概率。
- **概率质量函数：** 
$$
P(X = k) = 
\begin{cases}
p, & \text{if } k = 1 \\
1 - p, & \text{if } k = 0
\end{cases}
$$

### 2.2 二项分布

二项分布描述了在n次独立的伯努利试验中，成功次数的概率分布。例如，投掷10次硬币，出现正面次数的概率分布。

- **参数：** $n$ (试验次数) 和 $p$ (单次试验成功的概率)。
- **概率质量函数：** 
$$
P(X = k) = {n \choose k} p^k (1-p)^{(n-k)}
$$

### 2.3 泊松分布

泊松分布描述了在特定时间或空间范围内，事件发生次数的概率分布。例如，某网站每小时的访问量、某地区每年的地震次数等。

- **参数：** $\lambda$，表示单位时间或空间内事件发生的平均次数。
- **概率质量函数：** 
$$
P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}
$$

### 2.4 三者之间的联系

- 伯努利分布是二项分布和泊松分布的基础。
- 二项分布可以看作是n次独立的伯努利试验的和。
- 当n很大且p很小时，二项分布可以用泊松分布来近似。

## 4. 数学模型和公式详细讲解举例说明

### 4.1 二项分布公式推导

假设进行 $n$ 次独立的伯努利试验，每次试验成功的概率为 $p$，失败的概率为 $q = 1 - p$。设随机变量 $X$ 表示 $n$ 次试验中成功的次数，则 $X$ 的可能取值为 $0, 1, 2, ..., n$。

对于 $X = k$，即 $n$ 次试验中恰好有 $k$ 次成功，其概率为：

- 从 $n$ 次试验中选出 $k$ 次成功的组合数为 ${n \choose k}$。
- $k$ 次成功发生的概率为 $p^k$。
- $n-k$ 次失败发生的概率为 $q^{n-k}$。

因此，$X = k$ 的概率为：

$$
P(X = k) = {n \choose k} p^k q^{n-k}
$$

### 4.2 泊松分布公式推导

泊松分布的推导较为复杂，涉及到极限和级数的概念。简单来说，当二项分布中的 $n$ 很大且 $p$ 很小时，二项分布的概率质量函数可以近似于泊松分布的概率质量函数。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 Python 代码示例

```python
import numpy as np
from scipy.stats import bernoulli, binom, poisson

# 伯努利分布
p = 0.6  # 成功概率
rv = bernoulli(p)
print(rv.pmf(1))  # 计算 P(X=1)

# 二项分布
n = 10  # 试验次数
p = 0.4  # 成功概率
rv = binom(n, p)
print(rv.pmf(5))  # 计算 P(X=5)

# 泊松分布
λ = 2  # 平均发生次数
rv = poisson(λ)
print(rv.pmf(3))  # 计算 P(X=3)
```

### 5.2 代码解释

- `numpy` 和 `scipy.stats` 是常用的科学计算库，提供了各种概率分布的函数。
- `bernoulli`、`binom` 和 `poisson` 分别表示伯努利分布、二项分布和泊松分布。
- `pmf` 函数用于计算概率质量函数，即 $P(X=k)$。
