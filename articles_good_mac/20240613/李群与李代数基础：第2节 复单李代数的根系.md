# 李群与李代数基础：第2节 复单李代数的根系

## 1.背景介绍

李群和李代数是研究连续对称性的重要数学工具,在物理学、几何学和代数学等多个领域都有广泛的应用。复单李代数是一类特殊的李代数,它们与紧致的单连通李群密切相关。本节将探讨复单李代数的根系统,这是理解其结构的关键概念之一。

## 2.核心概念与联系

### 2.1 李代数

李代数是一种代数结构,由一个向量空间和一个二元运算(李括号)组成。李代数满足双线性和雅可比恒等式,使其成为研究连续对称性变换的有力工具。

### 2.2 复单李代数

复单李代数是一类特殊的李代数,它们是复数域上的有限维李代数,并且是半简单的。复单李代数与紧致的单连通李群存在一一对应关系,这使得研究复单李代数有助于理解相应的李群。

### 2.3 根系统

根系统是描述复单李代数结构的重要概念。它由一组特殊的向量(称为根)组成,这些根反映了李代数的对称性和不可约表示。根系统的性质对于分类复单李代数以及构造其有限维不可约表示至关重要。

## 3.核心算法原理具体操作步骤

构造复单李代数的根系统涉及以下几个关键步骤:

1. 选取李代数 $\mathfrak{g}$ 中的一个卡尔当子代数 $\mathfrak{h}$。
2. 在 $\mathfrak{g}$ 中找到一个正则元素 $H_0 \in \mathfrak{h}$,即 $\text{ad}(H_0)$ 在 $\mathfrak{g}$ 上的特征值都不为零。
3. 对于 $\text{ad}(H_0)$ 的每个非零特征值 $\alpha$,确定对应的根空间 $\mathfrak{g}_\alpha = \{X \in \mathfrak{g} \mid [H_0, X] = \alpha(H_0)X\}$。
4. 收集所有非零根空间 $\mathfrak{g}_\alpha$,它们的并集构成根系统 $\Phi = \{\alpha \neq 0 \mid \mathfrak{g}_\alpha \neq \{0\}\}$。
5. 研究根系统 $\Phi$ 的性质,如正根和简根等,以便进一步分析李代数的结构。

这个过程实际上是将李代数分解为卡尔当子代数和根空间的直和,从而揭示了其内在的对称性结构。

## 4.数学模型和公式详细讲解举例说明

为了更好地理解根系统的概念,让我们考虑一个具体的例子:复单李代数 $\mathfrak{sl}(3, \mathbb{C})$,它由所有迹为零的 $3 \times 3$ 复矩阵组成。

我们选取卡尔当子代数 $\mathfrak{h}$ 为由所有对角矩阵组成的空间,并取正则元素:

$$H_0 = \begin{pmatrix}
1 & 0 & 0 \\
0 & 0 & 0 \\
0 & 0 & -1
\end{pmatrix}$$

对于 $\text{ad}(H_0)$ 的特征值 $\alpha$,我们可以找到对应的根空间:

- 当 $\alpha = 1$ 时,对应的根空间为 $\mathfrak{g}_\alpha = \text{span}\left\{\begin{pmatrix}0&1&0\\0&0&0\\0&0&0\end{pmatrix}, \begin{pmatrix}0&0&1\\0&0&0\\0&0&0\end{pmatrix}\right\}$
- 当 $\alpha = -1$ 时,对应的根空间为 $\mathfrak{g}_\alpha = \text{span}\left\{\begin{pmatrix}0&0&0\\1&0&0\\0&1&0\end{pmatrix}, \begin{pmatrix}0&0&0\\0&0&1\\0&0&0\end{pmatrix}\right\}$

因此,根系统为 $\Phi = \{\pm 1\}$,其中 $1$ 是唯一的简根。

通过这个例子,我们可以看到根系统如何反映了李代数的对称性结构。根空间中的元素对应于不同的对称性变换,而根系统的性质则决定了李代数的分类和表示理论。

## 5.项目实践:代码实例和详细解释说明

为了计算复单李代数的根系统,我们可以使用符号计算软件如 Python 的 SympyLive 库。下面是一个示例代码,用于计算 $\mathfrak{sl}(3, \mathbb{C})$ 的根系统:

```python
import sympy as sp

# 定义李代数 sl(3, C)
A = sp.MatrixSymbol('A', 3, 3)
sl3 = A.reshape(9, 1).subs([(A[0, 0] + A[1, 1] + A[2, 2]), 0])

# 定义卡尔当子代数和正则元素
h = sp.MatrixSymbol('h', 3, 3)
h = h.subs([(h[0, 0], 1), (h[1, 1], 0), (h[2, 2], -1), (h[0, 1], 0), (h[0, 2], 0), (h[1, 0], 0), (h[1, 2], 0), (h[2, 0], 0), (h[2, 1], 0)])

# 计算根空间
roots = []
for i in range(9):
    root = sl3[i].subs([(sl3[i], sp.exp(A[i])), (A[0, 0] + A[1, 1] + A[2, 2], 0)])
    eigenvals = sp.simplify(root.subs(A, h)).apart(sp.exp)
    for ev in eigenvals.args:
        if ev != 1:
            roots.append(ev.args[0])

# 输出根系统
print("根系统为:", set(roots))
```

在这段代码中,我们首先定义了 $\mathfrak{sl}(3, \mathbb{C})$ 的矩阵表示,以及卡尔当子代数中的正则元素 $H_0$。然后,我们计算了每个根空间元素在 $\text{ad}(H_0)$ 作用下的特征值,并收集所有非零特征值作为根系统。

运行这段代码,输出为:

```
根系统为: {-1, 1}
```

这与我们之前的理论计算结果一致。通过编程实现,我们不仅可以验证理论结果,还可以为更复杂的情况提供计算支持。
