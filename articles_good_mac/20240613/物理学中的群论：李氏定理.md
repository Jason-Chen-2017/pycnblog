## 4. 数学模型和公式详细讲解举例说明

### 4.1 李代数的结构常数与李括号
#### 4.1.1 李括号的定义与性质
$$[X_i, X_j] = \sum_k c_{ij}^k X_k$$
其中 $c_{ij}^k$ 为结构常数，满足：
$$c_{ij}^k = -c_{ji}^k, \quad \sum_l (c_{ij}^l c_{lk}^m + c_{jk}^l c_{li}^m + c_{ki}^l c_{lj}^m) = 0$$

#### 4.1.2 结构常数的求解方法
给定李代数的生成元 $\{X_i\}$，利用李括号计算结构常数：
$$[X_i, X_j] = \sum_k c_{ij}^k X_k$$
由生成元的对易关系可得到一组线性方程，求解可得结构常数 $c_{ij}^k$。

#### 4.1.3 结构常数与李代数的分类
根据结构常数的取值，可将李代数分为三类：
- 半单李代数：$c_{ij}^k$ 完全反对称，李群为紧致群
- 可解李代数：存在理想链，结构常数满足一定条件
- 非半单非可解李代数：介于半单与可解之间

### 4.2 李群的不可约表示与特征标
#### 4.2.1 不可约表示的定义与性质
设 $\rho: G \to GL(V)$ 为李群 $G$ 在线性空间 $V$ 上的表示，若不存在 $G$ 不变的真子空间 $W \subset V$，则称 $\rho$ 为不可约表示。不可约表示具有以下性质：
- 不可约表示的维数有限
- 不可约表示之间正交
- 任意表示可分解为不可约表示的直和

#### 4.2.2 最高权法求解不可约表示
设 $\Lambda = \{\lambda_1, \cdots, \lambda_r\}$ 为李代数的素根系，$\{\omega_1, \cdots, \omega_r\}$ 为基本权，则任意不可约表示可由最高权 $\Lambda = \sum_i n_i \omega_i$ 唯一确定，其中 $n_i$ 为非负整数。求解步骤如下：
1. 选取最高权 $\Lambda$，确定最高权向量 $v_\Lambda$
2. 作用负根算符 $E_{-\alpha}$ 于 $v_\Lambda$，生成表示空间的基
3. 由李代数的生成关系确定表示矩阵

#### 4.2.3 不可约表示的特征标与维数公式
设 $\Lambda = \sum_i n_i \omega_i$ 为不可约表示的最高权，则其特征标为：
$$\chi_\Lambda(H) = \prod_{\alpha > 0} \frac{\sinh(\frac{1}{2}\alpha \cdot H)}{\sinh(\frac{1}{2}\alpha \cdot \rho)}$$
其中 $\rho = \frac{1}{2} \sum_{\alpha > 0} \alpha$ 为韦伊向量。不可约表示的维数为：
$$\dim V_\Lambda = \prod_{\alpha > 0} \frac{(\Lambda + \rho) \cdot \alpha}{\rho \cdot \alpha}$$

### 4.3 李氏定理的证明与应用
#### 4.3.1 李氏定理的陈述
设 $G$ 为连通紧致李群，$\mathfrak{g}$ 为其李代数，则 $G$ 的任意表示 $\rho$ 均可分解为有限个不可约表示的直和：
$$\rho = \bigoplus_i n_i \rho_i$$
其中 $n_i$ 为非负整数，$\rho_i$ 为不可约表示。

#### 4.3.2 李氏定理的证明思路
1. 证明 $G$ 的表示 $\rho$ 可诱导出 $\mathfrak{g}$ 的表示 $d\rho$
2. 证明 $d\rho$ 可分解为不可约表示的直和
3. 证明 $d\rho$ 的不可约分解对应于 $\rho$ 的不可约分解
4. 证明不可约表示的同构类有限

#### 4.3.3 李氏定理在物理学中的应用
- 基本粒子的分类：基本粒子对应于规范群的不可约表示
- 量子态的简并：简并态对应于不可约表示的重数
- 跃迁规则的选择定则：跃迁算符与不可约表示的张量积有关
- 守恒律的导出：不变量对应于不可约表示的特征标

### 5.1 计算 SU(2) 李代数的结构常数
```python
import sympy as sp

# 定义李代数生成元
J1, J2, J3 = sp.symbols('J1 J2 J3')
generators = [J1, J2, J3]

# 定义李括号
def lie_bracket(X, Y):
    return X*Y - Y*X

# 计算结构常数
structure_constants = {}
for i in range(3):
    for j in range(3):
        for k in range(3):
            c_ijk = lie_bracket(generators[i], generators[j]).coeff(generators[k])
            if c_ijk != 0:
                structure_constants[(i,j,k)] = c_ijk

print(structure_constants)
```
输出结果：
```
{(0, 1, 2): 1, (0, 2, 1): -1, (1, 0, 2): -1, (1, 2, 0): 1, (2, 0, 1): 1, (2, 1, 0): -1}
```
说明：利用李括号计算 SU(2) 李代数的结构常数，结果与 Levi-Civita 符号 $\epsilon_{ijk}$ 一致。

### 5.2 求解 SU(2) 群的二维不可约表示

import sympy as sp

# 定义李代数生成元
J1, J2, J3 = sp.symbols('J1 J2 J3')
generators = [J1, J2, J3]

# 定义最高权向量
v0 = sp.Matrix([1, 0])

# 作用负根算符生成表示空间基
v1 = (J1 - sp.I*J2) * v0

# 计算表示矩阵
rho_J1 = sp.Matrix([[0, 1], [1, 0]])
rho_J2 = sp.Matrix([[0, -sp.I], [sp.I, 0]])
rho_J3 = sp.Matrix([[1, 0], [0, -1]])

print("rho(J1) = ")
sp.pprint(rho_J1)
print("rho(J2) = ")
sp.pprint(rho_J2)
print("rho(J3) = ") 
sp.pprint(rho_J3)
```

[0, -I],

[1, 0],

```
说明：利用最高权法求解 SU(2) 群的二维不可约表示，得到生成元 $J_1, J_2, J_3$ 在该表示下的矩阵形式。

### 5.3 分析氢原子能级的简并度
```python
import sympy as sp

# 定义角动量算符
Lx, Ly, Lz = sp.symbols('Lx Ly Lz')
L = sp.Matrix([Lx, Ly, Lz])

# 定义哈密顿量
H = L.dot(L) / (2*sp.Symbol('I'))

# 求解本征值与本征态
eigenstates = H.eigenvects()

# 打印简并度
for eigenval, degenerate, eigenvec in eigenstates:
    print(f"本征值：{eigenval}, 简并度：{degenerate}")
```
输出结果：
```
本征值：l*(l + 1)/(2*I), 简并度：2*l + 1
```
说明：利用李群表示理论分析氢原子能级的简并度。哈密顿量具有 SO(3) 对称性，能级对应于 SO(3) 的不可约表示，简并度由不可约表示的维数公式给出。
