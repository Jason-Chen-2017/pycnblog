## 2. 核心概念与联系

### 2.1 概率论

概率论是研究随机现象的数学分支，它提供了描述和分析随机事件发生的可能性以及随机变量的分布规律的工具。 

**核心概念:**

* **样本空间:** 所有可能结果的集合。
* **事件:** 样本空间的子集，表示某种结果发生的集合。
* **概率:** 事件发生的可能性大小，通常用 0 到 1 之间的数值表示。
* **随机变量:** 可以取不同数值的变量，其取值具有随机性。
* **概率分布:** 描述随机变量取不同值的概率规律。

### 2.2 统计学

统计学是收集、分析、解释和表达数据的方法论科学，它提供了一套从数据中学习和提取信息的方法。

**核心概念:**

* **总体:** 所研究对象的全体。
* **样本:** 从总体中抽取的一部分个体。
* **统计量:** 用来描述样本特征的量，例如样本均值、样本方差等。
* **参数估计:** 利用样本统计量来估计总体参数。
* **假设检验:** 判断样本数据是否支持某种假设。


### 2.3 概率论与统计学的联系

概率论为统计学提供了理论基础，而统计学则将概率论的理论应用于实际数据的分析和处理。例如，在参数估计中，我们使用概率分布来描述总体参数的可能取值，并利用样本数据来估计参数的真实值。在假设检验中，我们使用概率论来计算检验统计量的分布，并根据分布来判断假设是否成立。


## 3. 核心算法原理具体操作步骤

### 3.1 贝叶斯定理

贝叶斯定理是概率论中的一个重要定理，它描述了在给定观测数据的情况下，如何更新对事件概率的信念。贝叶斯定理的公式如下：

$$
P(A|B) = \frac{P(B|A)P(A)}{P(B)}
$$

其中，$P(A)$ 表示事件 $A$ 发生的先验概率，$P(B|A)$ 表示在事件 $A$ 发生的条件下，事件 $B$ 发生的条件概率，$P(B)$ 表示事件 $B$ 发生的概率，$P(A|B)$ 表示在事件 $B$ 发生的条件下，事件 $A$ 发生的posterior 概率。

**操作步骤:**

1. 确定先验概率 $P(A)$，即在没有观测数据的情况下，对事件 $A$ 发生的信念。
2. 确定似然函数 $P(B|A)$，即在事件 $A$ 发生的条件下，观测数据 $B$ 发生的概率。
3. 计算边缘概率 $P(B)$，即观测数据 $B$ 发生的概率。
4. 使用贝叶斯定理计算posterior 概率 $P(A|B)$，即在观测数据 $B$ 发生的条件下，事件 $A$ 发生的概率。


### 3.2 最大似然估计

最大似然估计 (MLE) 是一种常用的参数估计方法，它通过最大化似然函数来估计总体参数的值。

**操作步骤:**

1. 确定似然函数，即在给定参数值的情况下，观测数据发生的概率。
2. 对似然函数取对数，得到对数似然函数。
3. 对对数似然函数求导，并令导数等于零，得到参数的估计值。


## 4. 数学模型和公式详细讲解举例说明

### 4.1 正态分布

正态分布 (Normal Distribution) 也称为高斯分布 (Gaussian Distribution)，是概率论和统计学中最重要的分布之一。正态分布的概率密度函数如下：

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

其中，$\mu$ 表示均值，$\sigma^2$ 表示方差。

**举例说明:**

假设一个班级的学生身高服从正态分布，均值为 170 厘米，标准差为 5 厘米。我们可以使用正态分布来计算学生身高落在某个范围内的概率。例如，计算学生身高在 165 厘米到 175 厘米之间的概率：

```python
import scipy.stats as stats

mu = 170
sigma = 5
x1 = 165
x2 = 175

probability = stats.norm.cdf(x2, mu, sigma) - stats.norm.cdf(x1, mu, sigma)

print(probability)  # 输出：0.6826894921370859
```


### 4.2 二项分布

二项分布 (Binomial Distribution) 描述了 n 次独立重复试验中，事件 A 发生的次数的概率分布。二项分布的概率质量函数如下：

$$
P(X=k) = {n \choose k} p^k (1-p)^{(n-k)}
$$

其中，$n$ 表示试验次数，$k$ 表示事件 A 发生的次数，$p$ 表示事件 A 在一次试验中发生的概率。

**举例说明:**

假设一枚硬币正面朝上的概率为 0.5，抛掷 10 次硬币，计算正面朝上的次数为 5 次的概率：

```python
import scipy.stats as stats

n = 10
k = 5
p = 0.5

probability = stats.binom.pmf(k, n, p)

print(probability)  # 输出：0.24609375
```
