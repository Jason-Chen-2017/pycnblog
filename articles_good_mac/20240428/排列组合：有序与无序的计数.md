## 2. 核心概念与联系

排列组合是计数问题中两种重要的计数方法，它们的区别在于是否考虑元素的顺序。

### 2.1 排列

排列是指从n个不同元素中取出m个元素（m≤n），按照一定的顺序排成一列的方案数。排列考虑元素的顺序，因此不同的排列顺序代表不同的方案。

例如，从{A, B, C}三个元素中取出2个元素进行排列，可以得到以下6种方案：

* AB
* AC
* BA
* BC
* CA
* CB

### 2.2 组合

组合是指从n个不同元素中取出m个元素（m≤n）的所有方案数，不考虑元素的顺序。组合不考虑元素的顺序，因此相同的元素组成的不同排列顺序被视为同一种方案。

例如，从{A, B, C}三个元素中取出2个元素进行组合，可以得到以下3种方案：

* AB
* AC
* BC

### 2.3 排列与组合的关系

排列与组合之间存在着密切的联系。从n个不同元素中取出m个元素进行排列的方案数，等于从n个不同元素中取出m个元素进行组合的方案数，再乘以m个元素的全排列数。

即：

$$
P_n^m = C_n^m \times m!
$$

其中，$P_n^m$表示从n个不同元素中取出m个元素进行排列的方案数，$C_n^m$表示从n个不同元素中取出m个元素进行组合的方案数，$m!$表示m个元素的全排列数。

def permutation(n, m):
    """

    if m > n:
        return 0
    result = 1
    for i in range(n, n-m, -1):

    return result

def combination(n, m):
    """
    计算组合数
    """
    if m > n:
        return 0
    return permutation(n, m) // permutation(m, m)
