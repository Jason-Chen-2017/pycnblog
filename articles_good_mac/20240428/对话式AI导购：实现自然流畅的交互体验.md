## 4. 数学模型和公式详细讲解举例说明

### 4.1 TF-IDF

TF-IDF是一种用于信息检索和文本挖掘的常用算法，它可以用来衡量一个词语在一个文档中的重要程度。TF-IDF的计算公式如下：

$$
\text{TF-IDF}(t, d) = \text{TF}(t, d) \times \text{IDF}(t)
$$

其中，

*   $t$ 表示词语。
*   $d$ 表示文档。
*   $\text{TF}(t, d)$ 表示词语 $t$ 在文档 $d$ 中出现的频率。
*   $\text{IDF}(t)$ 表示词语 $t$ 的逆文档频率，计算公式如下：

$$
\text{IDF}(t) = \log \frac{N}{df(t)}
$$

其中，

*   $N$ 表示文档总数。
*   $df(t)$ 表示包含词语 $t$ 的文档数量。

### 4.2 Word2Vec

Word2Vec是一种用于词嵌入的算法，它可以将词语表示成向量，并且语义相似的词语在向量空间中距离更近。Word2Vec的训练方法包括：

*   **CBOW模型**: 使用上下文词语预测目标词语。
*   **Skip-gram模型**: 使用目标词语预测上下文词语。

```python
import nltk
from sklearn.feature_extraction.text import TfidfVectorizer

# 定义商品列表
products = [
    {"name": "iPhone 14", "category": "手机", "price": 5999},
    {"name": "MacBook Pro", "category": "笔记本电脑", "price": 12999},
    {"name": "AirPods Pro", "category": "耳机", "price": 1999},
]

# 定义停用词列表
stopwords = nltk.corpus.stopwords.words("english")

# 创建TF-IDF向量化器
vectorizer = TfidfVectorizer(stop_words=stopwords)

# 将商品名称转换为TF-IDF向量
product_vectors = vectorizer.fit_transform([p["name"] for p in products])

# 定义用户意图识别函数
def identify_intent(text):
    # 使用TF-IDF向量化用户输入文本
    text_vector = vectorizer.transform([text])
    # 计算用户输入文本与商品名称的相似度
    similarities = product_vectors.dot(text_vector.T).toarray()
    # 返回相似度最高的商品
    most_similar_product_index = similarities.argmax()
    return products[most_similar_product_index]

# 模拟用户输入
user_input = "我想买一部手机"

# 识别用户意图
recommended_product = identify_intent(user_input)

# 打印推荐的商品
print(f"推荐商品：{recommended_product['name']}")
```
