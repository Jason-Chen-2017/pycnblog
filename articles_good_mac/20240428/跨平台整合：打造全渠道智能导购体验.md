## 4. 数学模型和公式详细讲解举例说明

### 4.1 协同过滤算法

协同过滤算法的核心思想是利用用户之间的相似性来进行推荐。常用的协同过滤算法包括：

* **基于用户的协同过滤 (User-based CF)**: 找到与目标用户兴趣相似的用户，并推荐这些用户喜欢的商品。
* **基于物品的协同过滤 (Item-based CF)**: 找到与目标用户喜欢的商品相似的商品，并推荐给目标用户。

以基于用户的协同过滤为例，其数学模型如下：

$$
sim(u, v) = \frac{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)(r_{vi} - \bar{r}_v)}{\sqrt{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)^2}\sqrt{\sum_{i \in I_{uv}}(r_{vi} - \bar{r}_v)^2}}
$$

其中，$sim(u, v)$ 表示用户 $u$ 和用户 $v$ 之间的相似度，$I_{uv}$ 表示用户 $u$ 和用户 $v$ 共同评价过的商品集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$\bar{r}_u$ 表示用户 $u$ 的平均评分。

### 4.2 内容推荐算法

内容推荐算法的核心思想是根据商品的属性信息和用户画像，推荐与用户兴趣匹配的商品。常用的内容推荐算法包括：

* **基于关键词的推荐**:  根据商品的关键词和用户搜索关键词，推荐与用户搜索意图相匹配的商品。
* **基于主题模型的推荐**: 利用主题模型提取商品和用户兴趣的主题分布，推荐与用户兴趣主题相似的商品。

以基于关键词的推荐为例，其数学模型如下：

$$
sim(u, i) = \sum_{k \in K} w_k \cdot TFIDF(k, u) \cdot TFIDF(k, i)
$$

其中，$sim(u, i)$ 表示用户 $u$ 和商品 $i$ 之间的相似度，$K$ 表示关键词集合，$w_k$ 表示关键词 $k$ 的权重，$TFIDF(k, u)$ 表示关键词 $k$ 在用户 $u$ 的兴趣描述中的 TF-IDF 值，$TFIDF(k, i)$ 表示关键词 $k$ 在商品 $i$ 的描述中的 TF-IDF 值。

```python
import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载数据
ratings = pd.read_csv('ratings.csv')

# 计算用户相似度矩阵
user_similarity = cosine_similarity(ratings.pivot_table(index='userId', columns='movieId', values='rating'))

# 为用户推荐商品
def recommend_movies(userId, num_recommendations):
    # 获取与目标用户最相似的用户
    similar_users = user_similarity[userId].argsort()[::-1][1:num_recommendations+1]
    
    # 获取相似用户喜欢的商品
    similar_user_movies = ratings[ratings['userId'].isin(similar_users)]['movieId'].unique()
    
    # 推荐相似用户喜欢的商品中目标用户未评价过的商品
    return similar_user_movies[~similar_user_movies.isin(ratings[ratings['userId'] == userId]['movieId'])]

# 推荐 5 部电影给用户 1
recommended_movies = recommend_movies(1, 5)
print(recommended_movies)
```
