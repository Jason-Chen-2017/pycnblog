## 5. 项目实践：代码实例和详细解释说明

以下是一个基于Python的智能灌溉控制代码示例：

```python
import time
import RPi.GPIO as GPIO

# 定义传感器和继电器引脚
sensor_pin = 17
relay_pin = 27

# 设置GPIO模式
GPIO.setmode(GPIO.BCM)
GPIO.setup(sensor_pin, GPIO.IN)
GPIO.setup(relay_pin, GPIO.OUT)

# 定义土壤水分阈值
threshold = 500

# 循环检测土壤水分
while True:
    # 读取传感器数值
    moisture_value = GPIO.input(sensor_pin)

    # 判断土壤水分是否低于阈值
    if moisture_value < threshold:
        # 打开继电器进行灌溉
        GPIO.output(relay_pin, GPIO.HIGH)
    else:
        # 关闭继电器停止灌溉
        GPIO.output(relay_pin, GPIO.LOW)

    # 等待一段时间
    time.sleep(1)
```

该代码示例使用树莓派和土壤水分传感器，实现了一个简单的智能灌溉控制系统。当土壤水分低于设定阈值时，系统会自动打开继电器进行灌溉，当土壤水分达到阈值时，系统会自动关闭继电器停止灌溉。
