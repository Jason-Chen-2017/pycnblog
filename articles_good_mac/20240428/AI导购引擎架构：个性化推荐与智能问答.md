## 4. 数学模型和公式详细讲解举例说明

### 4.1 协同过滤

协同过滤算法主要使用相似度度量方法，例如：

*   **余弦相似度**: 衡量用户或商品向量之间的夹角余弦值。
*   **皮尔逊相关系数**: 衡量用户或商品之间的线性相关程度。

例如，计算用户 $u$ 和用户 $v$ 之间的余弦相似度：

$$
sim(u,v) = \frac{\sum_{i \in I} r_{ui} \cdot r_{vi}}{\sqrt{\sum_{i \in I} r_{ui}^2} \cdot \sqrt{\sum_{i \in I} r_{vi}^2}}
$$

其中，$I$ 表示用户 $u$ 和用户 $v$ 都评价过的商品集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分。

### 4.2 基于内容的推荐

基于内容的推荐算法主要使用文本相似度度量方法，例如：

*   **TF-IDF**: 衡量词语在文档中的重要程度。
*   **Word2Vec**: 将词语表示为向量，计算词语之间的语义相似度。

例如，使用 TF-IDF 计算商品 $i$ 和商品 $j$ 之间的相似度：

$$
sim(i,j) = \sum_{t \in T} tfidf(t,i) \cdot tfidf(t,j)
$$

其中，$T$ 表示商品描述文本中的词语集合，$tfidf(t,i)$ 表示词语 $t$ 在商品 $i$ 描述文本中的 TF-IDF 值。

### 5.2 NLP代码示例

以下是一个简单的意图识别代码示例 (Python)：

```python
import nltk
from nltk.classify import NaiveBayesClassifier

# 训练数据
train_data = [
    ('我想买手机', '购买意图'),
    ('这款手机多少钱', '询问价格'),
    ('帮我推荐一款手机', '寻求推荐'),
]

# 特征提取
def extract_features(text):
    tokens = nltk.word_tokenize(text)
    return {
        'contains_buy': '买' in tokens,
        'contains_price': '价格' in tokens or '多少钱' in tokens,
        'contains_recommend': '推荐' in tokens,
    }

# 训练分类器
classifier = NaiveBayesClassifier.train([
    (extract_features(text), label) for text, label in train_data
])

# 预测意图
text = '我想买一部拍照好的手机'
features = extract_features(text)
label = classifier.classify(features)
print(label)  # 输出：购买意图
```
