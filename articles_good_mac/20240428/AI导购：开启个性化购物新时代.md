## 4. 数学模型和公式详细讲解举例说明

### 4.1 余弦相似度

余弦相似度是计算用户之间相似度的一种常用方法。其公式如下：

$$
sim(u,v) = \frac{\sum_{i=1}^n u_i v_i}{\sqrt{\sum_{i=1}^n u_i^2} \sqrt{\sum_{i=1}^n v_i^2}}
$$

其中，$u$ 和 $v$ 分别表示两个用户的特征向量，$n$ 表示特征向量的维度，$u_i$ 和 $v_i$ 分别表示用户 $u$ 和 $v$ 在第 $i$ 个特征上的取值。余弦相似度的取值范围为 $[-1, 1]$，值越大表示用户之间的相似度越高。 

### 4.2 TF-IDF

TF-IDF 是一种用于计算词语重要程度的统计方法。其公式如下：

$$
tfidf(t,d) = tf(t,d) \times idf(t)
$$

其中，$tf(t,d)$ 表示词语 $t$ 在文档 $d$ 中出现的频率，$idf(t)$ 表示词语 $t$ 的逆文档频率。TF-IDF 值越高，表示词语 $t$ 在文档 $d$ 中的重要性越高。

### 5.1 基于协同过滤的电影推荐系统

import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

# 加载电影评分数据
ratings = pd.read_csv('ratings.csv')

# 计算用户之间的相似度矩阵
user_similarity = cosine_similarity(ratings.pivot_table(index='userId', columns='movieId', values='rating').fillna(0))

# 获取目标用户的评分数据
target_user_id = 1
target_user_ratings = ratings[ratings['userId'] == target_user_id]

# 找到与目标用户最相似的用户
similar_users = user_similarity[target_user_id].argsort()[-10:][::-1][1:]

# 获取相似用户评分过的电影
similar_user_movies = ratings[ratings['userId'].isin(similar_users)]['movieId'].unique()

# 推荐相似用户评分过的电影给目标用户
recommendations = similar_user_movies[~similar_user_movies.isin(target_user_ratings['movieId'])]

print(recommendations)
```

from sklearn.feature_extraction.text import TfidfVectorizer

# 加载新闻数据
news = pd.read_csv('news.csv')

vectorizer = TfidfVectorizer()
news_features = vectorizer.fit_transform(news['content'])

# 获取目标用户的兴趣标签
target_user_interests = ['科技', '金融']

# 计算新闻与用户兴趣的相似度
similarity = cosine_similarity(news_features, vectorizer.transform(target_user_interests))

# 推荐与用户兴趣相似的新闻
recommendations = news.iloc[similarity.argsort(axis=0)[-10:][::-1,0]].index.tolist()

print(recommendations)
```
