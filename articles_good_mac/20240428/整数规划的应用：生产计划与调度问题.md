## 2. 核心概念与联系

### 2.1 整数规划的基本概念

整数规划是指决策变量限制为整数的线性规划问题。其数学模型可以表示为：

$$
\begin{aligned}
\text{max/min} \ & c^Tx \\
\text{s.t.} \ & Ax \leq b \\
& x \geq 0 \\
& x_j \in \mathbb{Z}, \ j \in J
\end{aligned}
$$

其中：

*   $c$ 为目标函数系数向量
*   $x$ 为决策变量向量
*   $A$ 为约束条件系数矩阵
*   $b$ 为约束条件右端常数向量
*   $J$ 为整数变量指标集合

### 2.2 生产计划与调度问题中的应用

整数规划在生产计划与调度问题中的应用主要体现在以下方面：

*   **资源分配:**  确定每种资源在不同工序上的分配方案
*   **工序排序:** 确定各工序的加工顺序
*   **时间安排:** 确定每道工序的开始和结束时间
*   **产量计划:** 确定每种产品的生产数量

## 3. 核心算法原理具体操作步骤

### 3.1 分支定界法

分支定界法是求解整数规划问题最常用的算法之一。其基本思想是：

1.  **分支:** 将原问题分解为多个子问题，每个子问题对应一个决策变量取值范围的划分
2.  **定界:** 利用线性规划松弛问题计算子问题的下界或上界，排除不可行解或非最优解
3.  **剪枝:** 根据定界结果，剪掉不包含最优解的子问题
4.  **迭代:** 重复上述步骤，直至找到最优解

### 3.2 割平面法

割平面法通过添加新的约束条件来缩小可行域，从而逼近整数解。其基本思想是：

1.  求解线性规划松弛问题，得到一个分数解
2.  根据分数解构造一个割平面，将分数解排除在可行域之外
3.  将割平面添加到原问题中，形成一个新的线性规划问题
4.  重复上述步骤，直至找到整数解

## 4. 数学模型和公式详细讲解举例说明

### 4.1 单机调度问题

单机调度问题是指将 $n$ 个工件安排在一台机器上加工，每个工件有特定的加工时间和交货期限，目标是最小化总完工时间或最大化机器利用率。其数学模型可以表示为：

$$
\begin{aligned}
\text{min} \ & \sum_{j=1}^n C_j \\
\text{s.t.} \ & C_j \geq p_j + \sum_{i=1}^{j-1} x_{ij} p_i, \ j = 1,2,...,n \\
& \sum_{i=1}^{j-1} x_{ij} + \sum_{k=j+1}^n x_{jk} = 1, \ j = 1,2,...,n \\
& x_{ij} \in \{0,1\}, \ i,j = 1,2,...,n
\end{aligned}
$$

其中：

*   $p_j$ 为工件 $j$ 的加工时间
*   $C_j$ 为工件 $j$ 的完工时间
*   $x_{ij}$ 为 0-1 变量，表示工件 $i$ 在工件 $j$ 之前加工 ($x_{ij}=1$) 或之后加工 ($x_{ij}=0$)

### 4.2 流水车间调度问题

流水车间调度问题是指将 $n$ 个工件安排在 $m$ 台机器上加工，每个工件需按特定顺序在每台机器上加工一次，目标是最小化总完工时间或最大化机器利用率。其数学模型可以表示为：

$$
\begin{aligned}
\text{min} \ & C_{n,m} \\
\text{s.t.} \ & C_{i,j} \geq C_{i-1,j} + p_{i,j}, \ i = 1,2,...,n; j = 1,2,...,m \\
& C_{i,j} \geq C_{i,j-1} + p_{i,j}, \ i = 1,2,...,n; j = 2,3,...,m \\
\end{aligned}
$$

其中：

*   $p_{i,j}$ 为工件 $i$ 在机器 $j$ 上的加工时间
*   $C_{i,j}$ 为工件 $i$ 在机器 $j$ 上的完工时间

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用 Python 和 PuLP 库求解单机调度问题的示例代码：

```python
from pulp import *

# 定义工件数量和加工时间
n = 5
p = [3, 5, 2, 4, 1]

# 创建模型
model = LpProblem("SingleMachineScheduling", LpMinimize)

# 定义决策变量
x = LpVariable.dicts("x", ((i, j) for i in range(n) for j in range(n)), cat="Binary")

# 定义目标函数
model += lpSum(p[j] * x[(i, j)] for i in range(n) for j in range(n))

# 定义约束条件
for j in range(n):
    model += lpSum(x[(i, j)] for i in range(n)) == 1
for i in range(n):
    model += lpSum(x[(i, j)] for j in range(n)) == 1
for i in range(n):
    for j in range(i):
        model += x[(i, j)] + x[(j, i)] <= 1

# 求解模型
model.solve()

# 输出结果
print("Status:", LpStatus[model.status])
for i in range(n):
    for j in range(n):
        if x[(i, j)].varValue == 1:
            print("Job", i+1, "before Job", j+1)
```
