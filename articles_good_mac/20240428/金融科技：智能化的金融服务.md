## 4. 数学模型和公式详细讲解举例说明

### 4.1 线性回归

线性回归（Linear Regression）是一种用于建立变量之间线性关系的统计方法。线性回归模型可以用以下公式表示：

$$
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n + \epsilon
$$

其中：

*   $y$ 是因变量。
*   $x_1, x_2, ..., x_n$ 是自变量。
*   $\beta_0, \beta_1, \beta_2, ..., \beta_n$ 是回归系数。
*   $\epsilon$ 是误差项。

线性回归模型的求解方法通常是使用最小二乘法。

### 4.2 逻辑回归

逻辑回归（Logistic Regression）是一种用于解决二分类问题的统计方法。逻辑回归模型可以用以下公式表示：

$$
P(y=1|x) = \frac{1}{1 + e^{-(b_0 + b_1 x_1 + b_2 x_2 + ... + b_n x_n)}}
$$

其中：

*   $P(y=1|x)$ 是给定自变量 $x$ 的情况下，因变量 $y$ 等于 1 的概率。
*   $x_1, x_2, ..., x_n$ 是自变量。
*   $b_0, b_1, b_2, ..., b_n$ 是回归系数。

逻辑回归模型的求解方法通常是使用最大似然估计法。


## 5. 项目实践：代码实例和详细解释说明

### 5.1 使用 Python 实现线性回归

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 创建样本数据
x = np.array([[1], [2], [3], [4]])
y = np.array([2, 4, 5, 7])

# 创建线性回归模型
model = LinearRegression()

# 训练模型
model.fit(x, y)

# 预测
y_pred = model.predict([[5]])
print(y_pred)  # 输出 [8.]
```

### 5.2 使用 Python 实现逻辑回归

```python
import numpy as np
from sklearn.linear_model import LogisticRegression

# 创建样本数据
x = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([0, 0, 1, 1])

# 创建逻辑回归模型

# 训练模型
model.fit(x, y)

# 预测
y_pred = model.predict([[5, 6]])

```
