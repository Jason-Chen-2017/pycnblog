## 4.数学模型和公式详细讲解举例说明

### 4.1 协同过滤推荐算法

#### 4.1.1 用户-职位评分矩阵
$$
R = 
\begin{bmatrix}
r_{11} & r_{12} & \cdots & r_{1n}\\
r_{21} & r_{22} & \cdots & r_{2n}\\
\vdots & \vdots & \ddots & \vdots\\
r_{m1} & r_{m2} & \cdots & r_{mn}\\
\end{bmatrix}
$$

其中 $r_{ij}$ 表示用户 $i$ 对职位 $j$ 的评分。

#### 4.1.2 用户相似度计算
用余弦相似度计算用户 $u$ 和用户 $v$ 的相似度：

$sim(u,v) = \frac{\sum_{i\in I_{uv}}r_{ui}r_{vi}}{\sqrt{\sum_{i\in I_{u}}r_{ui}^2}\sqrt{\sum_{i\in I_{v}}r_{vi}^2}}$

其中 $I_{uv}$ 是用户 $u$ 和 $v$ 共同评分过的职位集合，$I_u$ 和 $I_v$ 分别是用户 $u$ 和 $v$ 评分过的职位集合。

#### 4.1.3 职位推荐生成
对用户 $u$，推荐得分最高的Top-N个职位。职位 $i$ 的推荐得分计算公式为：

$p_{ui} = \frac{\sum_{v\in S^k(u,i)}sim(u,v)r_{vi}}{\sum_{v\in S^k(u,i)}|sim(u,v)|}$

其中 $S^k(u,i)$ 是与用户 $u$ 最相似的 $k$ 个用户中对职位 $i$ 有评分的用户集合。

### 4.2 基于内容的推荐算法

#### 4.2.1 职位与简历的向量化表示
使用TF-IDF、Word2Vec等方法将职位和简历文本转换为固定维度的稠密向量 $\mathbf{d} \in \mathbb{R}^n$。

#### 4.2.2 相似度计算
计算职位向量 $\mathbf{d}_j$ 与求职者简历向量 $\mathbf{d}_u$ 的相似度，常用的相似度度量有：

- 余弦相似度：$sim(\mathbf{d}_j, \mathbf{d}_u) = \frac{\mathbf{d}_j \cdot \mathbf{d}_u}{||\mathbf{d}_j|| \cdot ||\mathbf{d}_u||}$

- 点积：$sim(\mathbf{d}_j, \mathbf{d}_u) = \mathbf{d}_j \cdot \mathbf{d}_u$

- 欧氏距离：$sim(\mathbf{d}_j, \mathbf{d}_u) = \frac{1}{1+||\mathbf{d}_j - \mathbf{d}_u||_2}$

#### 4.2.3 个性化推荐
根据求职者的历史行为数据（如点击、投递、收藏等）构建用户画像向量 $\mathbf{p}_u$，计算其与职位的相关度得分：

$rel(\mathbf{d}_j|\mathbf{p}_u) = \mathbf{p}_u \cdot \mathbf{d}_j$

结合职位的相似度和相关度，生成个性化推荐列表。

    @Autowired
    private UserService userService;

        // 用户注册

@RestController
@RequestMapping("/api/job")
public class JobController {

    @Autowired
    private JobService jobService;

    @PostMapping
    public Result addJob(@RequestBody Job job) {
        // 发布职位
        jobService.add(job);
        return Result.ok();
    }

    @PutMapping("/{id}")
    public Result updateJob(@PathVariable Long id, @RequestBody Job job) {
        // 更新职位
        job.setId(id);
        jobService.update(job);
        return Result.ok();
    }

    @DeleteMapping("/{id}")
    public Result deleteJob(@PathVariable Long id) {
        // 删除职位
        jobService.delete(id);
        return Result.ok();
    }
}
```

@Service
public class ResumeServiceImpl implements ResumeService {

    @Autowired
    private ResumeMapper resumeMapper;
    
    @Autowired
    private RestHighLevelClient esClient;

    @Override
    public void add(Resume resume) {
        // 保存到数据库
        resumeMapper.insert(resume);
        
        // 构建ES索引
        IndexRequest request = new IndexRequest("resume");
        request.id(resume.getId().toString());
        request.source(JSON.toJSONString(resume), XContentType.JSON);
        esClient.index(request, RequestOptions.DEFAULT);
    }
}
```

#### 5.2.4 智能推荐与人岗匹配

```java
@Service
public class RecommendServiceImpl implements RecommendService {

    @Autowired
    private RestHighLevelClient esClient;

    @Autowired
    private MongoTemplate mongoTemplate;

    @Override
    public List<Job> recommend(Long userId) {
        // 协同过滤生成推荐
        UserRecs userRecs = mongoTemplate.findById(userId, UserRecs.class);
        List<Long> jobIds = userRecs.getRecommends();

        // 从ES检索职位详情
        SearchRequest request = new SearchRequest("job");
        SearchSourceBuilder builder = new SearchSourceBuilder();
        builder.query(QueryBuilders.idsQuery().addIds(jobIds.stream().map(String::valueOf).toArray(String[]::new)));
        request.source(builder);
        SearchResponse response = esClient.search(request, RequestOptions.DEFAULT);
        
        // 结果解析
        List<Job> jobs = new ArrayList<>();
        for (SearchHit hit : response.getHits()) {
            jobs.add(JSON.parseObject(hit.getSourceAsString(), Job.class));
        }
        return jobs;
    }
}
```
