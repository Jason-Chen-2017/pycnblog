### 1.4 术语表
#### 1.4.1 核心术语定义
- **密码学**：研究信息的加密和解密技术。
- **信息安全**：确保信息的机密性、完整性和可用性。
- **公钥加密**：一种加密方法，使用一对密钥进行加密和解密。
- **私钥加密**：一种加密方法，使用一个密钥进行加密，另一个密钥进行解密。
- **哈希函数**：一种将任意长度的数据转换为固定长度输出的函数。
- **对称加密**：使用相同的密钥进行加密和解密。
- **非对称加密**：使用一对密钥进行加密和解密，公钥用于加密，私钥用于解密。

#### 1.4.2 相关概念解释
- **RSA算法**：一种非对称加密算法，基于大数分解问题。
- **Diffie-Hellman密钥交换**：一种安全地交换密钥的方法。
- **椭圆曲线加密**：一种基于椭圆曲线离散对数问题的加密算法。
- **哈希函数**：如SHA-256，用于生成固定长度的哈希值。
- **对称加密算法**：如AES，用于加密和解密数据。

#### 1.4.3 缩略词列表
- **RSA**：Rivest-Shamir-Adleman
- **AES**：Advanced Encryption Standard
- **SHA**：Secure Hash Algorithm
- **Diffie-Hellman**：Diffie-Hellman密钥交换
- **ECC**：Elliptic Curve Cryptography

## 2. 核心概念与联系
### 2.1 数学原理
密码学的核心在于数学原理的应用，主要包括数论、代数、概率论等。这些数学原理为密码学提供了坚实的理论基础。

### 2.2 核心算法
密码学中的核心算法包括对称加密算法、非对称加密算法、哈希函数等。这些算法通过数学原理实现信息的安全传输和存储。

## 3. 核心算法原理 & 具体操作步骤
### 3.1 对称加密算法
对称加密算法使用相同的密钥进行加密和解密。常见的对称加密算法包括AES、DES等。

#### 3.1.1 AES算法原理
AES算法是一种对称加密算法，使用128位、192位或256位密钥进行加密和解密。其核心原理包括：
- **轮函数**：包括字节替换、行移位、列混淆和轮密钥加。
- **密钥扩展**：将密钥扩展为一系列轮密钥。

#### 3.1.2 AES算法伪代码
```plaintext
function AES_encrypt(plaintext, key):
    // 密钥扩展
    key_schedule(key)
    // 初始轮密钥
    round_keys = key_schedule(key)
    // 初始置换
    state = initial_permutation(plaintext)
    // 主轮迭代
    for round in 1 to 10:
        state = sub_bytes(state)
        state = shift_rows(state)
        state = mix_columns(state)
        state = add_round_key(state, round_keys[round])
    // 最后一轮
    state = sub_bytes(state)
    state = shift_rows(state)
    state = add_round_key(state, round_keys[11])
    // 最终置换
    ciphertext = final_permutation(state)
    return ciphertext

function AES_decrypt(ciphertext, key):
    // 密钥扩展
    key_schedule(key)
    // 初始轮密钥
    round_keys = key_schedule(key)
    // 初始置换
    state = initial_permutation(ciphertext)
    // 主轮迭代
    for round in 10 to 1:
        state = inv_shift_rows(state)
        state = inv_sub_bytes(state)
        state = add_round_key(state, round_keys[round])
        if round != 1:
            state = inv_mix_columns(state)
    // 最后一轮
    state = inv_shift_rows(state)
    state = inv_sub_bytes(state)
    state = add_round_key(state, round_keys[1])
    // 最终置换
    plaintext = final_permutation(state)
    return plaintext
```

### 3.2 非对称加密算法
非对称加密算法使用一对密钥进行加密和解密，公钥用于加密，私钥用于解密。常见的非对称加密算法包括RSA、Diffie-Hellman等。

#### 3.2.1 RSA算法原理
RSA算法是一种非对称加密算法，基于大数分解问题。其核心原理包括：
- **密钥生成**：选择两个大素数p和q，计算n = p * q，选择一个与φ(n)互质的e，计算d = e^(-1) mod φ(n)。
- **加密**：使用公钥(e, n)进行加密。
- **解密**：使用私钥(d, n)进行解密。

#### 3.2.2 RSA算法伪代码
```plaintext
function RSA_keygen():
    // 生成两个大素数p和q
    p = generate_prime()
    q = generate_prime()
    // 计算n = p * q
    n = p * q
    // 计算φ(n) = (p-1) * (q-1)
    phi_n = (p-1) * (q-1)
    // 选择一个与φ(n)互质的e
    e = choose_e(phi_n)
    // 计算d = e^(-1) mod φ(n)
    d = modinv(e, phi_n)
    // 返回公钥和私钥
    return (e, n), (d, n)

function RSA_encrypt(plaintext, public_key):
    // 解析公钥
    e, n = public_key
    // 将明文转换为整数
    plaintext_int = convert_to_int(plaintext)
    // 加密
    ciphertext_int = pow(plaintext_int, e, n)
    // 返回密文
    return ciphertext_int

function RSA_decrypt(ciphertext, private_key):
    // 解析私钥
    d, n = private_key
    // 解密
    plaintext_int = pow(ciphertext, d, n)
    // 将整数转换为明文
    plaintext = convert_to_text(plaintext_int)
    return plaintext
```

### 3.3 哈希函数
哈希函数是一种将任意长度的数据转换为固定长度输出的函数。常见的哈希函数包括SHA-256、MD5等。

#### 3.3.1 SHA-256算法原理
SHA-256算法是一种哈希函数，其核心原理包括：
- **初始化**：使用一个初始状态。
- **分组**：将输入数据分成固定长度的块。
- **处理**：对每个块进行处理，包括字节替换、行移位、列混淆和轮密钥加。
- **输出**：生成固定长度的哈希值。

#### 3.3.2 SHA-256算法伪代码
