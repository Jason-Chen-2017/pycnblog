## 3. 核心算法原理 & 具体操作步骤
### 3.1 伪随机数生成
#### 3.1.1 线性同余生成器
线性同余生成器是一种常用的伪随机数生成算法，其原理如下：
$$
X_{n+1} = (aX_n + c) \mod m
$$
其中，$X_n$ 是第 $n$ 个伪随机数，$a$、$c$ 和 $m$ 是常数，$m$ 通常是一个大质数。

#### 3.1.2 伪随机数生成的伪代码
```python
def linear_congruential_generator(seed, a, c, m):
    X = seed
    while True:
        X = (a * X + c) % m
        yield X
```

### 3.2 消息摘要
#### 3.2.1 哈希函数
哈希函数是一种将任意长度的数据转换为固定长度摘要的函数，常见的哈希函数有 MD5、SHA-256 等。

#### 3.2.2 哈希函数的伪代码
```python
def hash_function(data):
    # 假设使用 SHA-256
    import hashlib
    return hashlib.sha256(data).hexdigest()
```

### 3.3 数字签名
#### 3.3.1 RSA 算法
RSA 算法是一种非对称加密算法，用于生成公钥和私钥对。

#### 3.3.2 RSA 算法的伪代码
```python
def rsa_key_generation():
    # 生成大素数 p 和 q
    p = generate_large_prime()
    q = generate_large_prime()
    n = p * q
    phi = (p - 1) * (q - 1)
    e = generate_public_exponent(phi)
    d = generate_private_exponent(e, phi)
    return (n, e), (n, d)

def rsa_encrypt(message, public_key):
    n, e = public_key
    return pow(message, e, n)

def rsa_decrypt(ciphertext, private_key):
    n, d = private_key
    return pow(ciphertext, d, n)
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 伪随机数生成的数学模型
线性同余生成器的数学模型如下：
$$
X_{n+1} = (aX_n + c) \mod m
$$
其中，$X_n$ 是第 $n$ 个伪随机数，$a$、$c$ 和 $m$ 是常数，$m$ 通常是一个大质数。

### 4.2 消息摘要的数学模型
哈希函数的数学模型如下：
$$
H(M) = h(M)
$$
其中，$M$ 是输入数据，$h$ 是哈希函数，$H(M)$ 是输出的固定长度摘要。

### 4.3 数字签名的数学模型
RSA 算法的数学模型如下：
- **公钥**：$(n, e)$
- **私钥**：$(n, d)$
- **加密**：$c = m^e \mod n$
- **解密**：$m = c^d \mod n$

### 4.4 举例说明
#### 4.4.1 伪随机数生成
假设 $a = 1664525$，$c = 1013904223$，$m = 2^{32}$，$X_0 = 1$，生成前 5 个伪随机数：
```python
seed = 1
a = 1664525
c = 1013904223
m = 2**32

for _ in range(5):
    seed = (a * seed + c) % m
    print(seed)
```

#### 4.4.2 消息摘要
假设输入数据为 "Hello, World!"，使用 SHA-256 生成消息摘要：
```python
import hashlib
data = "Hello, World!"
hash_value = hashlib.sha256(data.encode()).hexdigest()
print(hash_value)
```

#### 4.4.3 数字签名
假设使用 RSA 算法生成公钥和私钥对，加密和解密数据：
```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# 生成公钥和私钥
key = RSA.generate(2048)
private_key = key.export_key()
public_key = key.publickey().export_key()

# 加密数据
cipher = PKCS1_OAEP.new(RSA.import_key(public_key))
message = b"Hello, World!"
ciphertext = cipher.encrypt(message)

# 解密数据
cipher = PKCS1_OAEP.new(RSA.import_key(private_key))
plaintext = cipher.decrypt(ciphertext)
print(plaintext.decode())
```
