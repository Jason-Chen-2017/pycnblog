## 3. 核心算法原理 & 具体操作步骤
### 3.1 SIR模型
SIR模型是描述易感、感染和康复三个状态的传播过程的数学模型。模型的基本假设是人群分为三类：易感节点（S）、感染节点（I）和康复节点（R）。

#### 3.1.1 SIR模型的数学描述
$$
\frac{dS}{dt} = -\beta SI
$$
$$
\frac{dI}{dt} = \beta SI - \gamma I
$$
$$
\frac{dR}{dt} = \gamma I
$$

其中，$\beta$ 是传播率，$\gamma$ 是康复率。

#### 3.1.2 SIR模型的具体操作步骤
1. 初始化：设定初始易感节点数 $S(0)$、感染节点数 $I(0)$ 和康复节点数 $R(0)$。
2. 计算传播率 $\beta$ 和康复率 $\gamma$。
3. 使用差分方程进行迭代计算：
   - 计算 $S(t+1)$、$I(t+1)$ 和 $R(t+1)$。
   - 更新时间步长 $t$。
4. 重复步骤3，直到达到终止条件。

### 3.2 SEIR模型
SEIR模型是SIR模型的扩展，增加了潜伏期的概念。模型的基本假设是人群分为四类：易感节点（S）、潜伏节点（E）、感染节点（I）和康复节点（R）。

#### 3.2.1 SEIR模型的数学描述
$$
\frac{dS}{dt} = -\beta SI
$$
$$
\frac{dE}{dt} = \beta SI - \sigma E
$$
$$
\frac{dI}{dt} = \sigma E - \gamma I
$$
$$
\frac{dR}{dt} = \gamma I
$$

其中，$\beta$ 是传播率，$\sigma$ 是潜伏期率，$\gamma$ 是康复率。

#### 3.2.2 SEIR模型的具体操作步骤
1. 初始化：设定初始易感节点数 $S(0)$、潜伏节点数 $E(0)$、感染节点数 $I(0)$ 和康复节点数 $R(0)$。
2. 计算传播率 $\beta$、潜伏期率 $\sigma$ 和康复率 $\gamma$。
3. 使用差分方程进行迭代计算：
   - 计算 $S(t+1)$、$E(t+1)$、$I(t+1)$ 和 $R(t+1)$。
   - 更新时间步长 $t$。
4. 重复步骤3，直到达到终止条件。

### 5.2 源代码详细实现和代码解读
#### 5.2.1 SIR模型的实现
```python
import numpy as np
import matplotlib.pyplot as plt

def sir_model(S0, I0, R0, beta, gamma, t):
    S = [S0]
    I = [I0]
    R = [R0]
    for t in range(1, len(t)):
        S.append(S[t-1] - beta * S[t-1] * I[t-1])
        I.append(I[t-1] + beta * S[t-1] * I[t-1] - gamma * I[t-1])
        R.append(R[t-1] + gamma * I[t-1])
    return S, I, R

S0, I0, R0 = 999, 1, 0
beta, gamma = 0.3, 0.1
t = np.linspace(0, 100, 101)
S, I, R = sir_model(S0, I0, R0, beta, gamma, t)

plt.plot(t, S, label='Susceptible')
plt.plot(t, I, label='Infected')
plt.plot(t, R, label='Recovered')
plt.xlabel('Time')
plt.ylabel('Number of people')
plt.legend()
plt.show()
```
