## 3. 核心算法原理 & 具体操作步骤

### 3.1 二分查找算法
二分查找是一种高效的查找算法，适用于有序数组。其基本思想是通过不断缩小查找范围来定位目标值。

#### 伪代码
```plaintext
function binary_search(arr, target):
    low = 0
    high = length(arr) - 1
    
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    
    return -1
```

### 3.2 快速排序算法
快速排序是一种高效的排序算法，通过分治法将数组分成两个子数组，分别进行排序。

#### 伪代码
```plaintext
function quick_sort(arr):
    if length(arr) <= 1:
        return arr
    
    pivot = arr[0]
    left = []
    right = []
    
    for i in range(1, length(arr)):
        if arr[i] < pivot:
            left.append(arr[i])
        else:
            right.append(arr[i])
    
    return quick_sort(left) + [pivot] + quick_sort(right)
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 二分查找算法的数学模型
二分查找算法的数学模型可以表示为递归函数。设数组长度为 \( n \)，则递归关系为：
$$
T(n) = T\left(\frac{n}{2}\right) + O(1)
$$
其中，\( T(n) \) 表示查找 \( n \) 个元素所需的时间。

### 4.2 快速排序算法的数学模型
快速排序算法的数学模型可以表示为递归函数。设数组长度为 \( n \)，则递归关系为：
$$
T(n) = T(k) + T(n - k - 1) + O(n)
$$
其中，\( k \) 表示划分点，\( T(k) \) 和 \( T(n - k - 1) \) 分别表示左右子数组的排序时间。

### 4.3 举例说明
#### 二分查找
假设有一个有序数组 \( [1, 3, 5, 7, 9, 11, 13, 15] \)，目标值为 7。通过二分查找，我们可以逐步缩小查找范围，最终找到目标值。

#### 快速排序
假设有一个数组 \( [5, 3, 8, 4, 2, 7, 1, 6] \)，通过快速排序，我们可以逐步将数组分成两个子数组，分别进行排序，最终得到有序数组。
