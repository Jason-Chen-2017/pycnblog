### 5.2 基于零知识证明的身份验证系统
```python
from zokrates import ZoKrates

# 定义零知识证明电路
code = """
def main(private field a, field b) -> (field):
  field c = if a * a == b then 1 else 0 fi
  return c
"""

# 编译电路
zok = ZoKrates()
artifacts = zok.compile(code)

# 生成证明
proof = zok.generate_proof(artifacts, a, b)

# 验证证明
result = zok.verify(artifacts, proof)
```
