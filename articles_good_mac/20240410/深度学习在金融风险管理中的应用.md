## 4. 项目实践：代码实例和详细解释说明

### 4.1 基于LSTM的股票价格预测
以下是一个基于LSTM的股票价格预测的Python代码示例:

```python
import numpy as np
import pandas as pd
from keras.models import Sequential
from keras.layers import LSTM, Dense

# 加载数据
data = pd.read_csv('stock_data.csv')
X_train = data['close'][:-10].values.reshape(-1, 1, 1)
y_train = data['close'][1:-9].values.reshape(-1, 1)
X_test = data['close'][-10:].values.reshape(-1, 1, 1)

# 构建LSTM模型
model = Sequential()
model.add(LSTM(64, input_shape=(1, 1)))
model.add(Dense(1))
model.compile(optimizer='adam', loss='mse')

# 训练模型
model.fit(X_train, y_train, epochs=100, batch_size=32, verbose=0)

# 进行预测
y_pred = model.predict(X_test)
print(y_pred)
```

该代码展示了如何使用LSTM模型进行股票价格预测。首先加载历史股票数据,将其划分为训练集和测试集。然后构建一个简单的LSTM模型,包括一个LSTM层和一个全连接层。最后,训练模型并使用测试数据进行预测。

### 4.2 基于自编码器的异常交易检测
以下是一个基于自编码器的异常交易检测的Python代码示例:

```python
import numpy as np
from keras.models import Sequential
from keras.layers import Dense, Dropout, Input
from keras.models import Model

# 加载交易数据
X_train = np.load('train_data.npy')

# 构建自编码器模型
input_dim = X_train.shape[1]
encoding_dim = 32

input_layer = Input(shape=(input_dim,))
encoded = Dense(encoding_dim, activation='relu')(input_layer)
decoded = Dense(input_dim, activation='sigmoid')(encoded)

autoencoder = Model(input_layer, decoded)
autoencoder.compile(optimizer='adam', loss='binary_crossentropy')

# 训练自编码器
autoencoder.fit(X_train, X_train, epochs=100, batch_size=32, shuffle=True, validation_data=(X_test, X_test))

# 检测异常交易
X_new = np.load('new_transaction_data.npy')
reconstruction_error = np.mean(np.power(X_new - autoencoder.predict(X_new), 2), axis=1)
anomalies = reconstruction_error > 0.5
print(anomalies)
```

该代码展示了如何使用自编码器模型进行异常交易检测。首先,构建一个简单的自编码器模型,包括一个编码层和一个解码层。然后,使用历史交易数据训练自编码器模型。最后,利用训练好的模型对新的交易数据进行重构,计算重构误差,并将重构误差超过阈值的交易标记为异常交易。

通过这两个代码示例,读者可以更直观地了解深度学习在金融风险管理中的具体应用。
