### 5.1 数据库设计
首先，我们需要设计数据库。在这个示例中，我们将创建三个表：phones、customers和sales。phones表存储手机信息，customers表存储顾客信息，sales表存储销售信息。

```python
import sqlite3
conn = sqlite3.connect('sales.db')

c = conn.cursor()

c.execute('''
CREATE TABLE phones (
    id INTEGER PRIMARY KEY,
    model TEXT,
    price REAL
)
''')

c.execute('''
CREATE TABLE customers (
    id INTEGER PRIMARY KEY,
    name TEXT,
    email TEXT
)
''')

c.execute('''
CREATE TABLE sales (
    id INTEGER PRIMARY KEY,
    phone_id INTEGER,
    customer_id INTEGER,
    date TEXT,
    FOREIGN KEY(phone_id) REFERENCES phones(id),
    FOREIGN KEY(customer_id) REFERENCES customers(id)
)
''')

conn.commit()
```
