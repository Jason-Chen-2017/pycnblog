### 2.5 推荐系统

#### 5.2.3 知识图谱
```python
import networkx as nx

# 数据收集
data = pd.read_csv('student_behavior.csv')

# 数据预处理
data = data.dropna()  # 去除噪声数据

# 知识图谱构建
G = nx.Graph()
for index, row in data.iterrows():
    G.add_edge(row['learning_time'], row['learning_content'])

# 知识图谱查询
query = 'learning_time'
neighbors = list(nx.neighbors(G, query))  # 查询知识图谱，了解学生的学习需求和能力
```
