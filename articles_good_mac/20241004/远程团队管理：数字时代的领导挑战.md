```python
def track_time(team_members, tasks):
    time_records = {}
    for member in team_members:
        time_records[member] = {}
    for task in tasks:
        for member in task['assignees']:
            if member in time_records:
                if task['id'] in time_records[member]:
                    time_records[member][task['id']] += task['duration']
                else:
                    time_records[member][task['id']] = task['duration']
    return time_records
```

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 沟通效率模型
#### 4.1.1 数学模型
- **沟通效率公式**：$E = \frac{C}{T}$
  - $E$：沟通效率
  - $C$：沟通内容的复杂度
  - $T$：沟通时间

#### 4.1.2 详细讲解
- **沟通内容复杂度**：根据沟通内容的复杂度调整沟通频率和渠道
- **沟通时间**：根据沟通时间调整沟通频率和渠道

#### 4.1.3 举例说明
假设一个项目团队需要讨论一个复杂的技术问题，沟通内容复杂度为5，沟通时间为1小时。根据公式，沟通效率为$E = \frac{5}{1} = 5$。为了提高沟通效率，可以调整沟通频率为每周一次，选择视频会议工具进行沟通。

### 4.2 团队协作效率模型
#### 4.2.1 数学模型
- **团队协作效率公式**：$S = \frac{P}{T}$
  - $S$：团队协作效率
  - $P$：团队成员的生产力
  - $T$：团队成员的协作时间

#### 4.2.2 详细讲解
- **团队成员生产力**：根据团队成员的能力和时间安排调整任务分配
- **团队成员协作时间**：通过项目管理工具实时跟踪任务进度

#### 4.2.3 举例说明
假设一个项目团队有5名成员，每人每天的工作时间为8小时，团队成员的生产力为0.8。根据公式，团队协作效率为$S = \frac{0.8 \times 5}{8} = 0.5$。为了提高团队协作效率，可以调整任务分配，确保每个成员的工作时间充分利用。

### 4.3 项目管理效率模型
#### 4.3.1 数学模型
- **项目管理效率公式**：$M = \frac{P}{T}$
  - $M$：项目管理效率
  - $P$：项目计划的详细程度
  - $T$：项目管理工具的使用频率

#### 4.3.2 详细讲解
- **项目计划详细程度**：根据项目目标和时间表制定详细的项目计划
- **项目管理工具使用频率**：通过时间跟踪工具记录团队成员的工作时间

#### 4.3.3 举例说明
假设一个项目团队需要完成一个复杂的软件开发项目，项目计划的详细程度为0.9，项目管理工具的使用频率为每天一次。根据公式，项目管理效率为$M = \frac{0.9}{1} = 0.9$。为了提高项目管理效率，可以调整项目计划的详细程度，确保每个任务都有详细的计划和时间表。
