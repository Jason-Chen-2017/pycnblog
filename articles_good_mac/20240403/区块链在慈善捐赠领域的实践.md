4.1 智能合约设计
智能合约是区块链技术的核心,它定义了慈善捐赠的规则和逻辑。以下是一个简单的慈善捐赠智能合约的Solidity代码:

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract CharityDonation {
    address payable public charityAddress;
    mapping(address => uint) public donations;

    constructor(address payable _charityAddress) {
        charityAddress = _charityAddress;
    }

    function donate() public payable {
        donations[msg.sender] += msg.value;
        charityAddress.transfer(msg.value);
    }

    function withdraw() public {
        uint amount = donations[msg.sender];
        require(amount > 0, "No donations to withdraw");
        donations[msg.sender] = 0;
        charityAddress.transfer(amount);
    }
}
```

该合约包含以下关键功能:
- 定义了慈善机构的收款地址
- 记录每个捐赠者的捐赠金额
- 实现捐赠和提现的功能

4.2 前端界面实现
前端界面负责与用户交互,调用智能合约完成捐赠和查询等操作。以下是一个基于React的前端代码示例:

```javascript
import React, { useState, useEffect } from 'react';
import Web3 from 'web3';
import CharityDonation from './CharityDonation.json';

function App() {
  const [web3, setWeb3] = useState(null);
  const [contract, setContract] = useState(null);
  const [donationAmount, setDonationAmount] = useState(0);
  const [userDonations, setUserDonations] = useState(0);

  useEffect(() => {
    async function initWeb3() {
      if (window.ethereum) {
        try {
          await window.ethereum.request({ method: 'eth_requestAccounts' });
          const web3Instance = new Web3(window.ethereum);
          setWeb3(web3Instance);

          const contractInstance = new web3Instance.eth.Contract(
            CharityDonation.abi,
            '0x...contract_address...'
          );
          setContract(contractInstance);

          const userDonations = await contractInstance.methods.donations(web3Instance.eth.accounts[0]).call();
          setUserDonations(web3Instance.utils.fromWei(userDonations, 'ether'));
        } catch (error) {
          console.error(error);
        }
      } else {
        console.error('Web3 is not detected.');
      }
    }
    initWeb3();
  }, []);

  const handleDonation = async () => {
    try {
      await contract.methods.donate().send({
        from: web3.eth.accounts[0],
        value: web3.utils.toWei(donationAmount, 'ether')
      });
      const userDonations = await contract.methods.donations(web3.eth.accounts[0]).call();
      setUserDonations(web3.utils.fromWei(userDonations, 'ether'));
    } catch (error) {
      console.error(error);
    }
  };

  return (
    <div>
      <h1>Charity Donation DApp</h1>
      <p>Your total donations: {userDonations} ETH</p>
      <input
        type="number"
        value={donationAmount}
        onChange={(e) => setDonationAmount(e.target.value)}
      />
      <button onClick={handleDonation}>Donate</button>
    </div>
  );
}

export default App;
```

该前端代码实现了以下功能:
- 连接以太坊节点,并实例化智能合约
- 读取用户的捐赠记录
- 提供捐赠输入框和按钮,调用智能合约的donate()函数完成捐赠

通过前端界面和智能合约的配合,用户可以方便地进行慈善捐赠,并实时查看自己的捐赠记录。
