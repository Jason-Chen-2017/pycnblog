## 4.数学模型和公式详细讲解举例说明

### 4.1 李代数的定义

一个李代数 $\mathfrak{g}$ 是一个向量空间，配备一个双线性映射 $[\cdot, \cdot]: \mathfrak{g} \times \mathfrak{g} \to \mathfrak{g}$，满足以下条件：

1. 反对称性：$[X, Y] = -[Y, X]$
2. 雅可比恒等式：$[X, [Y, Z]] + [Y, [Z, X]] + [Z, [X, Y]] = 0$

### 4.2 半单李代数的根系

设 $\mathfrak{h}$ 是 $\mathfrak{g}$ 的卡当子代数，根系 $\Delta$ 定义为：

$$
\Delta = \{\alpha \in \mathfrak{h}^* \mid \mathfrak{g}_\alpha \neq 0\}
$$

其中 $\mathfrak{g}_\alpha = \{X \in \mathfrak{g} \mid [H, X] = \alpha(H)X, \forall H \in \mathfrak{h}\}$。

### 4.3 结构常数的规范化

设 $\{E_\alpha\}$ 是 $\mathfrak{g}$ 的根向量，结构常数 $C_{\alpha, \beta}^\gamma$ 定义为：

$$
[E_\alpha, E_\beta] = \sum_\gamma C_{\alpha, \beta}^\gamma E_\gamma
$$

通过适当的线性变换，可以将 $C_{\alpha, \beta}^\gamma$ 规范化，使其满足特定的对称性条件。

```python
def cartan_subalgebra(matrix):
```

```python
def root_system(cartan_matrix):
    # 计算根系
    roots = []
    for i in range(cartan_matrix.shape[0]):
        for j in range(cartan_matrix.shape[1]):
            if cartan_matrix[i, j] != 0:
                roots.append((i, j))
    return roots

# 示例卡当矩阵
cartan_matrix = np.array([[2, -1], [-1, 2]])
root_system(cartan_matrix)
```
