## 2. 核心概念与联系

### 2.1 量子比特与量子态
量子比特是量子计算的基本单元，它可以处于 0 和 1 的叠加态，即同时处于 0 和 1 的状态。量子态可以用一个向量来表示，例如：

$$
|\psi\rangle = \alpha|0\rangle + \beta|1\rangle
$$

其中，$|0\rangle$ 和 $|1\rangle$ 分别表示量子比特的 0 和 1 状态，$\alpha$ 和 $\beta$ 是复数，表示处于相应状态的概率幅。

### 2.2 量子门与量子线路
量子门是量子计算中的基本操作，它可以对量子比特进行变换，例如：

*   **Hadamard 门 (H 门)**：将量子比特从 $|0\rangle$ 或 $|1\rangle$ 状态变换到叠加态。
*   **CNOT 门**：根据控制比特的状态对目标比特进行操作。
*   **旋转门 (R 门)**：将量子比特绕着 Bloch 球旋转一定的角度。

量子线路是由多个量子门组成的序列，它可以实现复杂的量子计算操作。

### 2.3 强化学习的基本要素
强化学习主要包括以下要素：

*   **智能体 (Agent)**：进行决策并与环境交互的实体。
*   **环境 (Environment)**：智能体所处的外部世界，它会根据智能体的动作产生新的状态和奖励。
*   **状态 (State)**：描述环境当前状况的信息。
*   **动作 (Action)**：智能体可以执行的操作。
*   **奖励 (Reward)**：环境对智能体动作的反馈信号。
*   **策略 (Policy)**：智能体根据当前状态选择动作的规则。

### 2.4 量子强化学习的结合
QRL 将量子计算的概念融入到强化学习中，例如：

*   **量子态表示状态空间**：利用量子比特的叠加性可以同时表示多个状态，从而更高效地探索状态空间。
*   **量子门操作进行状态转移**：利用量子门操作可以实现对量子态的变换，从而模拟智能体的状态转移过程。
*   **量子算法优化策略**：利用量子算法可以更高效地搜索最优策略。
