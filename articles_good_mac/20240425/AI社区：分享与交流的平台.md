## 5. 项目实践

### 5.1 代码实例 

以下是一个使用 Python 实现的简单协同过滤算法的示例代码：

```python
import numpy as np

def cosine_similarity(u, v):
  """
  计算用户 u 和用户 v 之间的余弦相似度。
  """
  return np.dot(u, v) / (np.linalg.norm(u) * np.linalg.norm(v))

def recommend(user_id, ratings):
  """
  为用户 user_id 推荐项目。
  """
  # 计算用户之间的相似度
  similarities = {}
  for other_user_id, other_ratings in ratings.items():
    if other_user_id != user_id:
      similarities[other_user_id] = cosine_similarity(ratings[user_id], other_ratings)

  # 找到最相似的用户
  most_similar_user_id = max(similarities, key=similarities.get)

  # 推荐最相似用户评价过的项目
  recommendations = []
  for item_id, rating in ratings[most_similar_user_id].items():
    if item_id not in ratings[user_id]:
      recommendations.append((item_id, rating))

  return recommendations
```

该代码首先定义了一个 `cosine_similarity` 函数，用于计算两个用户之间的余弦相似度。然后定义了一个 `recommend` 函数，该函数首先计算当前用户与其他用户之间的相似度，然后找到最相似的用户，并推荐最相似用户评价过的项目。
