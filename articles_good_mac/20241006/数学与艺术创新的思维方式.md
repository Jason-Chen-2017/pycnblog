## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 黄金分割
黄金分割是一种特殊的比例关系，其比值约为1.618。这种比例关系在自然界和艺术中广泛存在，如斐波那契数列、贝祖数等。

#### 公式
$$
\phi = \frac{1 + \sqrt{5}}{2}
$$

### 4.2 斐波那契数列
斐波那契数列是一种递归数列，其前两项为0和1，后续每一项为前两项之和。

#### 公式
$$
F_n = F_{n-1} + F_{n-2}
$$

### 4.3 贝祖数
贝祖数是一种特殊的数列，其前两项为1和1，后续每一项为前两项之和减去前一项。

#### 公式
$$
B_n = B_{n-1} + B_{n-2} - B_{n-3}
$$

### 5.2 源代码详细实现和代码解读
#### 代码实现
```python
import numpy as np
import matplotlib.pyplot as plt

def generate_random_texture(width, height, seed):
    random_texture = np.random.RandomState(seed).rand(width, height)
    return random_texture

def golden_ratio_split(length):
    golden_ratio = (1 + np.sqrt(5)) / 2
    split_point = length / golden_ratio
    return split_point

def plot_texture(texture):
    plt.imshow(texture, cmap='gray')
    plt.axis('off')
    plt.show()

# 生成随机纹理
width, height = 500, 500
seed = 42
random_texture = generate_random_texture(width, height, seed)

# 生成黄金分割点
split_point = golden_ratio_split(width)

# 绘制纹理
plot_texture(random_texture)
```

### 5.3 代码解读与分析
- **generate_random_texture**：生成一个随机纹理，参数为宽度、高度和随机种子。
- **golden_ratio_split**：计算黄金分割点，参数为长度。
- **plot_texture**：绘制纹理，使用Matplotlib库。
