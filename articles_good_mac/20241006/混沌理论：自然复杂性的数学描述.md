## 3. 核心算法原理 & 具体操作步骤
### 3.1 Logistic映射
Logistic映射是描述种群增长的混沌行为的离散映射。其数学模型为：
$$
x_{n+1} = r x_n (1 - x_n)
$$
其中，$x_n$表示第$n$代种群数量，$r$是增长率参数。

### 3.2 Henon映射
Henon映射是描述二维混沌系统的离散映射。其数学模型为：
$$
\begin{cases}
x_{n+1} = 1 - a x_n^2 + y_n \\
y_{n+1} = b x_n
\end{cases}
$$
其中，$x_n$和$y_n$表示第$n$代系统的状态，$a$和$b$是参数。

洛伦兹吸引子的数学模型为：
$$
\begin{cases}
\frac{dx}{dt} = \sigma (y - x) \\
\frac{dy}{dt} = x (\rho - z) - y \\
\frac{dz}{dt} = xy - \beta z
\end{cases}
$$
其中，$x$、$y$和$z$表示系统的状态，$\sigma$、$\rho$和$\beta$是参数。

### 5.2 源代码详细实现和代码解读
#### 5.2.1 Logistic映射
```python
import numpy as np
import matplotlib.pyplot as plt

def logistic_map(x0, r, n):
    x = [x0]
    for _ in range(n):
        x_next = r * x[-1] * (1 - x[-1])
        x.append(x_next)
    return x

x0 = 0.1
r = 3.8
n = 1000
x = logistic_map(x0, r, n)

plt.plot(range(n), x, 'b.')
plt.xlabel('迭代次数')
plt.ylabel('种群数量')
plt.title('Logistic映射')
plt.show()
```

#### 5.2.2 Henon映射
```python
import numpy as np
import matplotlib.pyplot as plt

def henon_map(x0, y0, a, b, n):
    x, y = [x0], [y0]
    for _ in range(n):
        x_next = 1 - a * x[-1]**2 + y[-1]
        y_next = b * x[-1]
        x.append(x_next)
        y.append(y_next)
    return x, y

x0 = 0.1
y0 = 0.1
a = 1.4
b = 0.3
n = 1000
x, y = henon_map(x0, y0, a, b, n)

plt.scatter(x, y, s=1, c='b')
plt.xlabel('x')
plt.ylabel('y')
plt.title('Henon映射')
plt.show()
```
