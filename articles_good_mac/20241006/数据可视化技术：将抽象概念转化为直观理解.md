## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 数学模型
数据可视化中的数学模型主要包括：
- **统计模型**：用于描述数据的分布和关系，如正态分布、线性回归等。
- **机器学习模型**：用于从数据中提取有价值的信息和知识，如决策树、支持向量机等。

### 4.2 公式
以下是一些常用的数学公式：
- **正态分布**：$f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$
- **线性回归**：$y = \beta_0 + \beta_1 x + \epsilon$
- **决策树**：$f(x) = \sum_{i=1}^{n} \alpha_i I(x \in R_i)$

### 4.3 详细讲解
#### 4.3.1 正态分布
正态分布是一种常见的统计模型，用于描述数据的分布情况。其公式为：
$$ f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} $$
其中，$\mu$ 是均值，$\sigma$ 是标准差。

#### 4.3.2 线性回归
线性回归是一种常用的统计模型，用于描述数据之间的线性关系。其公式为：
$$ y = \beta_0 + \beta_1 x + \epsilon $$
其中，$\beta_0$ 是截距，$\beta_1$ 是斜率，$\epsilon$ 是误差项。

#### 4.3.3 决策树
决策树是一种常用的机器学习模型，用于从数据中提取有价值的信息和知识。其公式为：
$$ f(x) = \sum_{i=1}^{n} \alpha_i I(x \in R_i) $$
其中，$I(x \in R_i)$ 是指示函数，表示$x$是否属于区域$R_i$。

### 4.4 举例说明
以下是一个简单的例子，用于说明如何使用正态分布和线性回归进行数据可视化：
```python
import numpy as np
import matplotlib.pyplot as plt

# 生成数据
mu, sigma = 0, 0.1
x = np.random.normal(mu, sigma, 1000)
y = 2 * x + np.random.normal(0, 0.1, 1000)

# 绘制数据
plt.scatter(x, y)
plt.xlabel("x")
plt.ylabel("y")
plt.title("数据分布")
plt.show()

# 计算正态分布
mu, sigma = np.mean(x), np.std(x)
x_fit = np.linspace(-3, 3, 100)
y_fit = 1 / (sigma * np.sqrt(2 * np.pi)) * np.exp(-(x_fit - mu)**2 / (2 * sigma**2))

# 绘制正态分布
plt.plot(x_fit, y_fit, 'r--')
plt.xlabel("x")
plt.ylabel("概率密度")
plt.title("正态分布")
plt.show()

# 计算线性回归
slope, intercept = np.polyfit(x, y, 1)
y_fit = slope * x_fit + intercept

# 绘制线性回归
plt.plot(x_fit, y_fit, 'g--')
plt.xlabel("x")
plt.ylabel("y")
plt.title("线性回归")
plt.show()
```
