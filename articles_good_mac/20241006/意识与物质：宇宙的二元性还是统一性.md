## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 量子力学的基本公式
量子力学的基本公式是薛定谔方程，它描述了量子系统的演化过程。
$$
i\hbar \frac{\partial}{\partial t}|\psi(t)\rangle = \hat{H}|\psi(t)\rangle
$$
其中，$|\psi(t)\rangle$ 是量子系统的波函数，$\hat{H}$ 是哈密顿算子，$\hbar$ 是约化普朗克常数。

### 4.2 信息论的基本公式
信息论的基本公式是香农熵，它描述了信息的度量。
$$
H(X) = -\sum_{x \in X} p(x) \log p(x)
$$
其中，$X$ 是随机变量的取值集合，$p(x)$ 是随机变量的分布函数。

### 4.3 举例说明
假设我们有一个量子比特，其初始状态为 $|0\rangle$。我们应用一个单量子比特门操作，将其状态变为 $|1\rangle$。然后我们测量该量子比特，得到的结果为 $|1\rangle$ 的概率为1。

### 5.2 源代码详细实现和代码解读
```python
from qiskit import QuantumCircuit, execute, Aer

# 初始化量子比特
n = 1
qubits = QuantumCircuit(n)

# 应用量子门操作
qubits.x(0)

# 测量量子比特
qubits.measure_all()

# 执行量子电路
backend = Aer.get_backend('qasm_simulator')
job = execute(qubits, backend, shots=1000)
result = job.result()

# 输出结果
counts = result.get_counts(qubits)
print(counts)
```
