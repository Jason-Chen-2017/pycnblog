## 4. 数学模型和公式详细讲解举例说明

### 4.1 线性回归模型

线性回归模型用于分析变量之间的线性关系，例如分析广告投放量与销售额之间的关系。

$$ y = \beta_0 + \beta_1 x + \epsilon $$

其中，$y$ 是因变量（例如销售额），$x$ 是自变量（例如广告投放量），$\beta_0$ 和 $\beta_1$ 是模型参数，$\epsilon$ 是误差项。

### 4.2 时间序列模型

时间序列模型用于分析时间序列数据的趋势和周期性，例如预测未来酒类产品的销售趋势。

$$ y_t = \alpha + \beta t + \gamma sin(2 \pi t/12) + \epsilon_t $$

其中，$y_t$ 是时间 $t$ 时的销售额，$\alpha$、$\beta$、$\gamma$ 是模型参数，$\epsilon_t$ 是误差项。


## 5. 项目实践：代码实例和详细解释说明

import pandas as pd
import matplotlib.pyplot as plt

# 读取数据
data = pd.read_csv('wine_sales.csv')

# 数据清洗
data.dropna(inplace=True)  # 删除缺失值
data = data[data['sales'] > 0]  # 删除销售额为 0 的数据

# 计算核心指标
mean_sales = data['sales'].mean()
std_sales = data['sales'].std()

# 绘制销售额分布直方图
plt.hist(data['sales'], bins=20)
plt.xlabel('Sales')
plt.ylabel('Frequency')
plt.show()

# 线性回归分析
from sklearn.linear_model import LinearRegression

model = LinearRegression()
model.fit(data[['advertising']], data['sales'])

# 预测未来销售额
future_advertising = pd.DataFrame({'advertising': [100, 200, 300]})
predicted_sales = model.predict(future_advertising)

print(predicted_sales)
