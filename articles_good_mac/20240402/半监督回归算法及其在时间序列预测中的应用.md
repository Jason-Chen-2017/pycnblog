## 4. 数学模型和公式详细讲解

半监督回归算法的数学模型可以表示为:

$$ \min_{\theta} \sum_{i=1}^{l} (y_i - f(x_i;\theta))^2 + \lambda \sum_{i=l+1}^{n} (f(x_i;\theta) - \bar{y}_i)^2 $$

其中:
- $\{(x_1, y_1), (x_2, y_2), ..., (x_l, y_l)\}$ 是标注数据集
- $\{x_{l+1}, x_{l+2}, ..., x_n\}$ 是无标注数据集
- $\bar{y}_i$ 是无标注样本 $x_i$ 的伪标签,由初始模型预测得到
- $f(x;\theta)$ 是待优化的回归模型,参数为 $\theta$
- $\lambda$ 是平衡标注数据loss和无标注数据loss的超参数

这个优化问题包含两部分:
1. 最小化标注数据的预测误差
2. 最小化无标注数据的预测伪标签与模型输出的差距

通过交替优化这两部分,半监督回归算法能够学习出一个更加准确的回归模型。具体的优化求解过程涉及到梯度下降、正则化等技术,在此就不再赘述。

    def __init__(self, input_size, hidden_size, output_size):

        x = self.fc1(x)
        x = self.activation(x)

        return x

# 半监督回归算法实现
model = RegressionModel(input_size=X.shape[1], hidden_size=64, output_size=1)

    optimizer.zero_grad()
    output = model(X_labeled)
