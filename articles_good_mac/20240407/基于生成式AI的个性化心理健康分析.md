## 4. 数学模型和公式详细讲解
生成式AI模型通常涉及复杂的数学原理,如概率图模型、信息论等。以变分自编码器(VAE)为例,其核心思想是利用贝叶斯推断,将输入数据$x$映射到潜在变量$z$的分布$q(z|x)$,并通过生成网络学习从$z$到$x$的生成分布$p(x|z)$。其优化目标是最大化证据下界(ELBO):

$$ \mathcal{L}(x) = \mathbb{E}_{q(z|x)}[\log p(x|z)] - \text{KL}[q(z|x)||p(z)] $$

其中,$\text{KL}[\cdot||\cdot]$表示KL散度。通过优化这一目标函数,VAE可以学习数据的潜在表示,并生成与输入数据相似的样本。

import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader
import numpy as np

# 定义VAE模型
class VAE(nn.Module):
    def __init__(self, input_dim, latent_dim):
        super(VAE, self).__init__()
        self.input_dim = input_dim
        self.latent_dim = latent_dim

        # 编码器
        self.encoder = nn.Sequential(
            nn.Linear(input_dim, 512),
            nn.ReLU(),
            nn.Linear(512, 256),
            nn.ReLU(),
            nn.Linear(256, latent_dim * 2)
        )

        # 解码器
        self.decoder = nn.Sequential(
            nn.Linear(latent_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 512),
            nn.ReLU(),
            nn.Linear(512, input_dim),
            nn.Sigmoid()
        )

    def forward(self, x):
        # 编码
        z_params = self.encoder(x)
        mu, log_var = z_params[:, :self.latent_dim], z_params[:, self.latent_dim:]
        # 采样
        z = self.reparameterize(mu, log_var)
        # 解码
        x_recon = self.decoder(z)
        return x_recon, mu, log_var

    def reparameterize(self, mu, log_var):
        std = torch.exp(0.5 * log_var)
        eps = torch.randn_like(std)
        return mu + eps * std
