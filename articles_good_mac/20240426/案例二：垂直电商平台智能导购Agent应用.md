### 4. 数学模型和公式

#### 4.1 协同过滤算法

协同过滤算法基于用户-商品评分矩阵，利用用户之间的相似性或商品之间的相似性进行推荐。

* **基于用户的协同过滤：** $$ sim(u, v) = \frac{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)(r_{vi} - \bar{r}_v)}{\sqrt{\sum_{i \in I_{uv}}(r_{ui} - \bar{r}_u)^2}\sqrt{\sum_{i \in I_{uv}}(r_{vi} - \bar{r}_v)^2}} $$

其中，$sim(u, v)$ 表示用户 $u$ 和用户 $v$ 之间的相似度，$I_{uv}$ 表示用户 $u$ 和用户 $v$ 共同评分过的商品集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$\bar{r}_u$ 表示用户 $u$ 的平均评分。

* **基于商品的协同过滤：** $$ sim(i, j) = \frac{\sum_{u \in U_{ij}}(r_{ui} - \bar{r}_i)(r_{uj} - \bar{r}_j)}{\sqrt{\sum_{u \in U_{ij}}(r_{ui} - \bar{r}_i)^2}\sqrt{\sum_{u \in U_{ij}}(r_{uj} - \bar{r}_j)^2}} $$

其中，$sim(i, j)$ 表示商品 $i$ 和商品 $j$ 之间的相似度，$U_{ij}$ 表示同时评分过商品 $i$ 和商品 $j$ 的用户集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$\bar{r}_i$ 表示商品 $i$ 的平均评分。

#### 4.2 TF-IDF 算法

TF-IDF 算法用于计算词语在文档中的重要程度，常用于文本特征提取和信息检索。

* **词频 (TF)：** $$ tf_{i,j} = \frac{n_{i,j}}{\sum_{k} n_{k,j}} $$

其中，$tf_{i,j}$ 表示词语 $i$ 在文档 $j$ 中的词频，$n_{i,j}$ 表示词语 $i$ 在文档 $j$ 中出现的次数，$\sum_{k} n_{k,j}$ 表示文档 $j$ 中所有词语出现的次数总和。

* **逆文档频率 (IDF)：** $$ idf_i = \log \frac{N}{df_i} $$

其中，$idf_i$ 表示词语 $i$ 的逆文档频率，$N$ 表示文档总数，$df_i$ 表示包含词语 $i$ 的文档数量。

* **TF-IDF：** $$ tfidf_{i,j} = tf_{i,j} \times idf_i $$

其中，$tfidf_{i,j}$ 表示词语 $i$ 在文档 $j$ 中的 TF-IDF 值。

import numpy as np

def cosine_similarity(user1, user2):
  """
  计算两个用户之间的余弦相似度。
  """
  dot_product = np.dot(user1, user2)
  norm_user1 = np.linalg.norm(user1)
  norm_user2 = np.linalg.norm(user2)
  return dot_product / (norm_user1 * norm_user2)

# 用户-商品评分矩阵
ratings = np.array([
  [5, 3, 4, 4, 0],
  [3, 1, 2, 3, 3],
  [4, 3, 4, 3, 5],
  [3, 2, 1, 5, 4],
])

# 计算用户 1 和用户 2 之间的相似度
similarity = cosine_similarity(ratings[0], ratings[1])

# 打印相似度
print(similarity)
