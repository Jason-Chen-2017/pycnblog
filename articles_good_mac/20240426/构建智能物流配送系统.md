## 4. 数学模型和公式详细讲解举例说明

### 4.1 路径规划问题数学模型

路径规划问题可以抽象为图论中的最短路径问题。设图 $G = (V, E)$，其中 $V$ 表示节点集合，$E$ 表示边集合。每个边 $(u, v) \in E$ 都有一个权重 $w(u, v)$，表示从节点 $u$ 到节点 $v$ 的距离或时间成本。路径规划问题就是要找到一条从起点 $s$ 到终点 $t$ 的路径，使得路径上所有边的权重之和最小。

### 4.2 Dijkstra 算法

Dijkstra 算法是一种用于求解单源最短路径问题的贪心算法。算法步骤如下：

1.  初始化：将起点 $s$ 的距离设置为 0，其他节点的距离设置为无穷大。
2.  选择距离起点最近的节点 $u$，并将其标记为已访问。
3.  更新与节点 $u$ 相邻的节点 $v$ 的距离：如果 $dist(u) + w(u, v) < dist(v)$，则将 $dist(v)$ 更新为 $dist(u) + w(u, v)$。
4.  重复步骤 2 和 3，直到所有节点都被访问。

### 4.3 A\* 算法

A\* 算法是一种启发式搜索算法，它在 Dijkstra 算法的基础上，增加了启发式函数 $h(n)$，用于估计节点 $n$ 到终点 $t$ 的距离。算法步骤如下：

1.  初始化：将起点 $s$ 的距离设置为 0，其他节点的距离设置为无穷大，并将起点 $s$ 加入到优先队列中。
2.  从优先队列中取出距离起点最近的节点 $u$，并将其标记为已访问。
3.  如果 $u$ 是终点 $t$，则算法结束。
4.  更新与节点 $u$ 相邻的节点 $v$ 的距离和启发式函数值：
    *   如果 $dist(u) + w(u, v) < dist(v)$，则将 $dist(v)$ 更新为 $dist(u) + w(u, v)$，并将 $v$ 加入到优先队列中。
    *   将 $v$ 的启发式函数值设置为 $h(v)$。
5.  重复步骤 2 到 4，直到找到终点 $t$。

### 5.1 Python 代码实现 Dijkstra 算法

```python
def dijkstra(graph, start):
    dist = {node: float('inf') for node in graph}
    dist[start] = 0
    visited = set()
    queue = [start]
    while queue:
        u = min(queue, key=lambda node: dist[node])
        queue.remove(u)
        visited.add(u)
        for v in graph[u]:
            if v not in visited:
                alt = dist[u] + graph[u][v]
                if alt < dist[v]:
                    dist[v] = alt
                    queue.append(v)
    return dist
```

                came_from[neighbor] = current
                g_score[neighbor] = tentative_g_score
                f_score[neighbor] = g_score[neighbor] + h(neighbor)

                    open_set.add(neighbor)

def reconstruct_path(came_from, current):
    path = [current]
    while current in came_from:
        current = came_from[current]
        path.append(current)
    path.reverse()
    return path
