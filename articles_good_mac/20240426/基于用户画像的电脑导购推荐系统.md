## 2. 核心概念与联系

### 2.1. 用户画像

用户画像是用户信息的标签化描述，它包含了用户的静态信息（例如性别、年龄、职业等）和动态信息（例如浏览记录、购买记录、搜索记录等）。通过对用户画像的分析，可以了解用户的兴趣爱好、消费习惯、生活方式等信息。

### 2.2. 推荐算法

推荐算法是推荐系统的核心，它根据用户的历史行为和用户画像，预测用户对不同商品或服务的兴趣程度，并推荐最符合用户需求的商品或服务。常见的推荐算法包括：

*   **协同过滤算法:** 基于用户的历史行为，找到与目标用户兴趣相似的其他用户，并将这些用户喜欢的商品推荐给目标用户。
*   **内容推荐算法:** 基于商品或服务的属性，找到与用户喜欢的商品或服务相似的其他商品或服务，并将这些商品或服务推荐给用户。
*   **混合推荐算法:** 结合协同过滤算法和内容推荐算法的优势，提供更精准的推荐结果。

### 2.3. 评价指标

推荐系统的评价指标用于衡量推荐结果的质量。常见的评价指标包括：

*   **准确率:** 推荐结果与用户实际需求的匹配程度。
*   **召回率:** 推荐结果覆盖用户实际需求的比例。
*   **F1值:** 准确率和召回率的综合指标。
*   **NDCG:** 考虑推荐结果排序的评价指标。

### 4.1. 协同过滤算法

协同过滤算法是推荐系统中最常用的算法之一。它基于用户的历史行为，找到与目标用户兴趣相似的其他用户，并将这些用户喜欢的商品推荐给目标用户。 

**基于用户的协同过滤算法：**

1.  计算用户之间的相似度。例如，可以使用余弦相似度计算用户之间的相似度： 

$$

$$

其中，$u$ 和 $v$ 表示两个用户，$I_u$ 和 $I_v$ 表示用户 $u$ 和 $v$ 评价过的商品集合，$r_{u,i}$ 表示用户 $u$ 对商品 $i$ 的评分。

2.  找到与目标用户最相似的 k 个用户。
3.  将这 k 个用户喜欢的商品推荐给目标用户。

$$

$$

其中，$i$ 和 $j$ 表示两个商品，$U_i$ 和 $U_j$ 表示评价过商品 $i$ 和 $j$ 的用户集合，$r_{u,i}$ 表示用户 $u$ 对商品 $i$ 的评分。

2.  找到与目标用户喜欢的商品最相似的 k 个商品。

from sklearn.metrics.pairwise import cosine_similarity

# 计算用户之间的相似度
user_similarity = cosine_similarity(ratings)

# 计算商品之间的相似度
item_similarity = cosine_similarity(ratings.T)

# 基于用户的协同过滤推荐
def user_cf(user_id, k):
    # 找到与目标用户最相似的 k 个用户
    sim_users = user_similarity[user_id].argsort()[::-1][1:k+1]
    # 获取这 k 个用户喜欢的商品
    sim_items = ratings.loc[sim_users].mean(axis=0).sort_values(ascending=False)
    # 返回推荐结果
    return sim_items.index[:10]

# 基于物品的协同过滤推荐
def item_cf(item_id, k):

    sim_items = item_similarity[item_id].argsort()[::-1][1:k+1]
    # 获取这 k 个商品的评分
    sim_ratings = ratings.loc[:, sim_items]
    # 计算推荐得分
    sim_scores = sim_ratings.sum(axis=1) / (sim_ratings > 0).sum(axis=1)
    # 返回推荐结果
