## 4. 数学模型和公式详细讲解举例说明

### 4.1 准确率

准确率是指正确预测的样本数占总样本数的比例，公式如下：

$$
Accuracy = \frac{TP + TN}{TP + TN + FP + FN}
$$

其中：

* TP: 真正例，即实际为正例，预测也为正例
* TN: 真负例，即实际为负例，预测也为负例
* FP: 假正例，即实际为负例，预测为正例
* FN: 假负例，即实际为正例，预测为负例

### 4.2 召回率

召回率是指正确预测的正例数占实际正例数的比例，公式如下：

$$
Recall = \frac{TP}{TP + FN}
$$

### 4.3 F1值

F1值是准确率和召回率的调和平均值，公式如下：

$$
F1 = \frac{2 * Precision * Recall}{Precision + Recall}
$$

## 5. 项目实践：代码实例和详细解释说明

以下是一个使用Python计算准确率、召回率和F1值的代码示例：

```python
from sklearn.metrics import accuracy_score, recall_score, f1_score

# 实际标签
y_true = [0, 1, 1, 0, 1]
# 预测标签
y_pred = [0, 1, 0, 0, 1]

# 计算准确率
accuracy = accuracy_score(y_true, y_pred)
# 计算召回率
recall = recall_score(y_true, y_pred)
# 计算F1值
f1 = f1_score(y_true, y_pred)

print("Accuracy:", accuracy)
print("Recall:", recall)
print("F1:", f1)
```
