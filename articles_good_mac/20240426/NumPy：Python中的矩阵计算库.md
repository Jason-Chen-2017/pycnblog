## 2. 核心概念与联系

### 2.1. ndarray：多维数组

ndarray 是 NumPy 的核心数据结构，它表示一个 N 维数组，可以存储相同数据类型的元素。ndarray 支持高效的元素访问、切片、迭代和广播操作，是进行数值计算的基础。

### 2.2. 数据类型

NumPy 支持多种数值数据类型，包括整数、浮点数、复数和布尔值等。用户可以根据需要选择合适的数据类型，以提高计算效率和精度。

### 2.3. 数组形状和维度

ndarray 的形状表示其各个维度的大小，例如 (3, 4) 表示一个 3 行 4 列的二维数组。数组的维度数量称为秩，例如二维数组的秩为 2。

### 2.4. 广播机制

广播机制允许不同形状的数组进行运算，只要它们的形状满足一定的规则。广播机制可以避免不必要的数组复制，提高计算效率。

## 4. 数学模型和公式详细讲解举例说明

### 4.1. 线性方程组求解

NumPy 可以用于求解线性方程组，例如：

$$
Ax = b
$$

其中，A 是系数矩阵，b 是常数向量，x 是未知向量。使用 NumPy 的 `linalg.solve` 函数可以求解 x 的值。

### 4.2. 特征值和特征向量

NumPy 可以计算矩阵的特征值和特征向量，例如：

$$
Av = \lambda v
$$

其中，A 是矩阵，v 是特征向量，λ 是特征值。使用 NumPy 的 `linalg.eig` 函数可以计算 A 的特征值和特征向量。

### 4.3. 奇异值分解

NumPy 可以进行矩阵的奇异值分解 (SVD)，例如：

$$
A = U\Sigma V^T
$$

其中，U 和 V 是正交矩阵，Σ 是对角矩阵。使用 NumPy 的 `linalg.svd` 函数可以进行 SVD。
