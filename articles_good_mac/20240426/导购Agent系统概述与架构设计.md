## 4. 数学模型和公式详细讲解举例说明

### 4.1 协同过滤算法

#### 4.1.1 基于用户的协同过滤

**相似度计算:**

$$
sim(u, v) = \frac{\sum_{i \in I_{uv}} (r_{ui} - \bar{r}_u)(r_{vi} - \bar{r}_v)}{\sqrt{\sum_{i \in I_{uv}} (r_{ui} - \bar{r}_u)^2} \sqrt{\sum_{i \in I_{uv}} (r_{vi} - \bar{r}_v)^2}}
$$

其中，$sim(u, v)$ 表示用户 $u$ 和用户 $v$ 的相似度，$I_{uv}$ 表示用户 $u$ 和用户 $v$ 共同评分的商品集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$\bar{r}_u$ 表示用户 $u$ 的平均评分。

**预测评分:**

$$
\hat{r}_{ui} = \bar{r}_u + \frac{\sum_{v \in N_u(i)} sim(u, v)(r_{vi} - \bar{r}_v)}{\sum_{v \in N_u(i)} |sim(u, v)|}
$$

其中，$\hat{r}_{ui}$ 表示用户 $u$ 对商品 $i$ 的预测评分，$N_u(i)$ 表示与用户 $u$ 最相似的 $k$ 个用户中，对商品 $i$ 有评分的用户集合。

#### 4.1.2 基于物品的协同过滤

**相似度计算:**

$$
sim(i, j) = \frac{\sum_{u \in U_{ij}} (r_{ui} - \bar{r}_i)(r_{uj} - \bar{r}_j)}{\sqrt{\sum_{u \in U_{ij}} (r_{ui} - \bar{r}_i)^2} \sqrt{\sum_{u \in U_{ij}} (r_{uj} - \bar{r}_j)^2}}
$$

其中，$sim(i, j)$ 表示商品 $i$ 和商品 $j$ 的相似度，$U_{ij}$ 表示同时购买了商品 $i$ 和商品 $j$ 的用户集合，$r_{ui}$ 表示用户 $u$ 对商品 $i$ 的评分，$\bar{r}_i$ 表示商品 $i$ 的平均评分。

**预测评分:**

$$
\hat{r}_{ui} = \frac{\sum_{j \in N_i(u)} sim(i, j)r_{uj}}{\sum_{j \in N_i(u)} |sim(i, j)|}
$$

其中，$\hat{r}_{ui}$ 表示用户 $u$ 对商品 $i$ 的预测评分，$N_i(u)$ 表示与商品 $i$ 最相似的 $k$ 个商品中，用户 $u$ 购买过的商品集合。

### 4.2 内容推荐算法

内容推荐算法的数学模型根据具体的算法有所不同，例如，可以使用 TF-IDF 模型计算商品的关键词权重，或使用 Word2Vec 模型计算商品的词向量表示。

### 4.3 深度学习推荐算法

深度学习推荐算法的数学模型通常比较复杂，例如，可以使用循环神经网络 (RNN) 学习用户的行为序列，或使用卷积神经网络 (CNN) 学习商品的图像特征。

# 预测用户评分
def predict_rating(user_id, movie_id):
    # 找到与目标用户最相似的 k 个用户
    similar_users = user_similarity[user_id].argsort()[-k:]
    # 计算预测评分
    weighted_sum = 0
    similarity_sum = 0
    for similar_user in similar_users:
        if movie_id in data[data['userId'] == similar_user]['movieId'].values:
            rating = data[(data['userId'] == similar_user) & (data['movieId'] == movie_id)]['rating'].values[0]
            similarity = user_similarity[user_id, similar_user]
            weighted_sum += rating * similarity
            similarity_sum += similarity
    if similarity_sum == 0:
        return 0
    else:
        return weighted_sum / similarity_sum

# 预测用户 1 对电影 10 的评分
