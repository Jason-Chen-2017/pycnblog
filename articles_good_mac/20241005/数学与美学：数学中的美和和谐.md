## 2. 核心概念与联系
### 2.1 黄金分割
黄金分割是一种数学比例，其比值约为1.618，常用于美学和设计中。黄金分割的比例可以表示为：
$$
\phi = \frac{1 + \sqrt{5}}{2} \approx 1.618
$$
黄金分割在自然界和艺术作品中广泛存在，例如：人体比例、建筑、绘画等。

### 2.2 斐波那契数列
斐波那契数列是一个数列，每个数字是前两个数字之和，可以表示为：
$$
F_n = F_{n-1} + F_{n-2}
$$
其中，$F_0 = 0$，$F_1 = 1$。斐波那契数列在自然界和艺术作品中广泛存在，例如：植物的叶片排列、贝壳的螺旋等。

### 2.3 对称性
对称性是指物体或图案在某种变换下保持不变的性质。对称性在自然界和艺术作品中广泛存在，例如：人体的左右对称、建筑的对称设计等。

### 2.4 黄金角
黄金角约等于137.5°，是自然界中常见的一种角度。黄金角在自然界和艺术作品中广泛存在，例如：植物的叶片排列、贝壳的螺旋等。

### 2.5 贝塞尔曲线
贝塞尔曲线是一种用于描述平滑曲线的数学方法，广泛应用于图形设计和计算机图形学中。贝塞尔曲线可以表示为：
$$
B(t) = \sum_{i=0}^{n} \binom{n}{i} (1-t)^{n-i} t^i P_i
$$
其中，$P_i$是控制点，$t$是参数。

```python
import numpy as np
import matplotlib.pyplot as plt

def bezier_curve(points, t):
    n = len(points) - 1
    if n == 0:
        return points[0]
    else:
        return (1 - t) * bezier_curve(points[:-1], t) + t * bezier_curve(points[1:], t)

def plot_bezier_curve(points):
    t = np.linspace(0, 1, 100)
    curve = [bezier_curve(points, ti) for ti in t]
    x, y = zip(*curve)
    plt.plot(x, y)
    plt.show()

# 测试
points = [(0, 0), (1, 1), (2, 0), (3, 1)]
plot_bezier_curve(points)
```
