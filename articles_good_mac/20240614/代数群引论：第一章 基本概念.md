## 2.核心概念与联系

### 2.1 群

群是一种代数结构，它由一个集合和一个二元运算组成，满足以下四个条件：

1. 封闭性：对于任意的$a,b\in G$，$a*b\in G$。
2. 结合律：对于任意的$a,b,c\in G$，$(a*b)*c=a*(b*c)$。
3. 单位元：存在一个元素$e\in G$，使得对于任意的$a\in G$，$a*e=e*a=a$。
4. 逆元：对于任意的$a\in G$，存在一个元素$a^{-1}\in G$，使得$a*a^{-1}=a^{-1}*a=e$。

其中，$*$表示群的二元运算，$e$表示群的单位元素，$a^{-1}$表示$a$的逆元素。

### 2.2 环

环是一种代数结构，它由一个集合和两个二元运算组成，满足以下四个条件：

1. 封闭性：对于任意的$a,b\in R$，$a+b\in R$，$a*b\in R$。
2. 加法结合律：对于任意的$a,b,c\in R$，$(a+b)+c=a+(b+c)$。
3. 加法交换律：对于任意的$a,b\in R$，$a+b=b+a$。
4. 加法单位元：存在一个元素$0\in R$，使得对于任意的$a\in R$，$a+0=0+a=a$。

其中，$+$表示环的加法运算，$*$表示环的乘法运算，$0$表示环的加法单位元素。

### 2.3 域

域是一种代数结构，它由一个集合和两个二元运算组成，满足以下四个条件：

1. 封闭性：对于任意的$a,b\in F$，$a+b\in F$，$a*b\in F$。
2. 加法结合律：对于任意的$a,b,c\in F$，$(a+b)+c=a+(b+c)$。
3. 加法交换律：对于任意的$a,b\in F$，$a+b=b+a$。
4. 加法单位元：存在一个元素$0\in F$，使得对于任意的$a\in F$，$a+0=0+a=a$。

此外，还满足以下两个条件：

1. 乘法结合律：对于任意的$a,b,c\in F$，$(a*b)*c=a*(b*c)$。
2. 乘法分配律：对于任意的$a,b,c\in F$，$a*(b+c)=a*b+a*c$。

其中，$+$表示域的加法运算，$*$表示域的乘法运算，$0$表示域的加法单位元素。

### 2.4 代数群

代数群是一种同时具有代数结构和群结构的数学对象，它由一个集合和一个二元运算组成，满足以下四个条件：

1. 封闭性：对于任意的$a,b\in G$，$a*b\in G$。

3. 单位元：存在一个元素$e\in G$，使得对于任意的$a\in G$，$a*e=e*a=a$。
4. 逆元：对于任意的$a\in G$，存在一个元素$a^{-1}\in G$，使得$a*a^{-1}=a^{-1}*a=e$。

此外，还满足以下两个条件：

1. 代数结构：代数群的集合上还有一个代数结构，如环、域等。
2. 运算结构：代数群的二元运算还满足代数结构的运算结构，如环的加法运算、乘法运算等。
