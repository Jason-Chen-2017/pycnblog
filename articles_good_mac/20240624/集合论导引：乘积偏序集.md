## 2. 核心概念与联系

偏序集是集合论中的基本概念，它是一种具有偏序关系的集合。偏序关系是一种具有反身性、反对称性和传递性的二元关系。乘积偏序集是由两个或多个偏序集的笛卡尔积构成的，它的偏序关系是由构成它的偏序集的偏序关系决定的。

### 5.2 源代码详细实现

以下是一个简单的Python程序，它实现了乘积偏序集的构造算法：

```python
def product_partial_order(A, B):
    # Compute the Cartesian product of A and B
    product = [(a, b) for a in A for b in B]

    # Define the partial order relation
    def leq(x, y):
        return x[0] <= y[0] and x[1] <= y[1]

    # Construct the product partial order set
    PPO = (product, leq)

    return PPO
```
