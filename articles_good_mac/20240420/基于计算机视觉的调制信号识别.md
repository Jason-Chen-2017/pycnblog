## 4. 数学模型和公式详细讲解

在这一部分，我们将详细介绍用于信号到图像的转换、特征提取和调制方式识别的数学模型和公式。

### 4.1 信号到图像的转换

信号到图像的转换主要使用短时傅里叶变换（STFT）。STFT的公式如下：

$$
STFT(x(t)) = X(t, f) = \int_{-\infty}^{\infty} x(\tau) w(t - \tau) e^{-j2\pi f \tau} d\tau
$$

其中，$x(t)$是信号，$w(t)$是窗函数，$t$是时间，$f$是频率。通过STFT，我们可以得到信号的时频表示，然后将其作为图像来处理。

### 4.2 特征提取

特征提取主要使用卷积神经网络（CNN）。CNN的基本组成部分是卷积层和池化层。卷积层的公式如下：

$$
X_{out}[i, j, k] = \sum_{m=0}^{M-1} \sum_{n=0}^{N-1} \sum_{p=0}^{P-1} W[m, n, p, k] \cdot X_{in}[i+m, j+n, p] + b[k]
$$

其中，$X_{in}$是输入特征图，$W$是卷积核，$b$是偏置，$X_{out}$是输出特征图，$M$，$N$和$P$是卷积核的尺寸。

池化层通常使用最大池化或平均池化。最大池化的公式如下：

$$
X_{out}[i, j, k] = \max_{m=0}^{M-1} \max_{n=0}^{N-1} X_{in}[i \cdot M + m, j \cdot N + n, k]
$$

### 4.3 调制方式的识别

调制方式的识别主要使用全连接神经网络（FCNN）。FCNN的公式如下：

$$
X_{out} = W \cdot X_{in} + b
$$

其中，$X_{in}$是输入特征，$W$是权重，$b$是偏置，$X_{out}$是输出特征，即调制方式的识别结果。

    model = Sequential()
    model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(64, 64, 1)))

    model.add(Dense(10, activation='softmax'))
    return model

# 定义训练过程
def train(x, y, model):
    model.compile(optimizer='adam',
                  loss='sparse_categorical_crossentropy',
                  metrics=['accuracy'])
    model.fit(x, y, epochs=10)

# 定义预测过程
def predict(x, model):
    return np.argmax(model.predict(x), axis=-1)

# 加载数据
x = np.random.normal(size=(1000, 1024))
