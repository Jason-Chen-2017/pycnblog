### 5.1 用户注册和登录
在Flask中，我们可以使用Blueprint来定义用户的注册和登录路由。注册路由接收用户名和密码，检查用户名是否已经存在，如果不存在，则创建一个新的用户，并保存到数据库。登录路由接收用户名和密码，检查用户名和密码是否匹配，如果匹配，则创建一个新的会话，并保存到cookie。

```python
from flask import Blueprint, request, session
from sqlalchemy.orm.exc import NoResultFound
from .models import User, db

auth = Blueprint('auth', __name__)

@auth.route('/register', methods=['POST'])
def register():
    username = request.form.get('username')
    password = request.form.get('password')
    try:
        user = User.query.filter_by(username=username).one()
        return 'Username already exists'
    except NoResultFound:
        user = User(username=username, password=password)
        db.session.add(user)
        db.session.commit()
        return 'User registered successfully'

@auth.route('/login', methods=['POST'])
def login():
    username = request.form.get('username')
    password = request.form.get('password')
    try:
        user = User.query.filter_by(username=username).one()
        if user.password == password:
            session['username'] = username
            return 'User logged in successfully'
        else:
            return 'Incorrect password'
    except NoResultFound:
        return 'User not found'

@auth.route('/logout', methods=['POST'])
def logout():
    session.pop('username', None)
    return 'User logged out successfully'
```

### 5.2 发布和查看招聘信息
发布招聘信息的路由接收职位的名称、工作内容、工作地点、薪资待遇等信息，创建一个新的招聘信息，并保存到数据库。查看招聘信息的路由返回数据库中的所有招聘信息。

```python
from flask import Blueprint, request
from .models import Job, db

job = Blueprint('job', __name__)

@job.route('/post', methods=['POST'])
def post():
    title = request.form.get('title')
    content = request.form.get('content')
    location = request.form.get('location')
    salary = request.form.get('salary')
    job = Job(title=title, content=content, location=location, salary=salary)
    db.session.add(job)
    db.session.commit()
    return 'Job posted successfully'

@job.route('/list', methods=['GET'])
def list():
    jobs = Job.query.all()
    return {'jobs': [job.to_dict() for job in jobs]}
```
