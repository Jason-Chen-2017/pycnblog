## 2. 核心概念与联系

在这一部分，我们将介绍流形、切丛、特征类和Pontrjagin类的核心概念，并探讨它们之间的联系。

### 2.1 流形

流形是一个局部类似于欧几里得空间的拓扑空间。具体来说，一个$n$维流形是一个拓扑空间，其中每一点都有一个邻域同胚于$n$维欧几里得空间。流形的一个重要性质是它的局部平坦性，这使得我们可以在流形上进行微积分运算。

### 2.2 切丛

切丛是流形上每一点的切空间的集合。切空间是流形上每一点的局部线性近似，它是一个向量空间。切丛将这些切空间组织在一起，形成一个新的流形。切丛的一个重要性质是它的向量丛结构，这使得我们可以在切丛上进行线性代数运算。

### 2.3 特征类

特征类是流形上向量丛的拓扑不变量。特征类的一个重要性质是它们在同胚下保持不变，这使得它们成为研究流形拓扑性质的有力工具。常见的特征类包括Stiefel-Whitney类、Chern类和Pontrjagin类。

### 2.4 Pontrjagin类

Pontrjagin类是通过研究流形的切丛来定义的特征类。具体来说，Pontrjagin类是切丛的Chern类的某种组合。Pontrjagin类的一个重要性质是它们在同胚下保持不变，这使得它们成为研究流形拓扑性质的有力工具。

以下是计算Pontrjagin类的Python代码实现：

```python
import sympy as sp

# 定义变量
x, y, z = sp.symbols('x y z')

# 定义切丛的曲率形式

# 计算Chern类
chern_class = sp.det(curvature_form)

# 计算Pontrjagin类
pontrjagin_class = chern_class**2
```
