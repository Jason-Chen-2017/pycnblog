## 4. 数学模型和公式详细讲解举例说明

### 4.1 Logistic 回归

Logistic 回归是一种用于预测二元变量的统计模型。 例如，我们可以使用 Logistic 回归预测消费者是否会购买特定产品。

**公式:**

$$ P(Y=1|X) = \frac{1}{1 + e^{-(b_0 + b_1X_1 + ... + b_nX_n)}} $$

其中：

* $P(Y=1|X)$ 是消费者购买产品的概率。
* $X_1, ..., X_n$ 是影响消费者购买决策的因素。
* $b_0, b_1, ..., b_n$ 是模型参数。

### 4.2 K-means 聚类

K-means 聚类是一种将数据点分组到 k 个集群中的算法。 例如，我们可以使用 K-means 聚类将消费者分为不同的细分市场。

**算法步骤:**

1. 随机选择 k 个数据点作为初始聚类中心。
2. 将每个数据点分配到距离其最近的聚类中心。
3. 重新计算每个聚类的中心点。
4. 重复步骤 2 和 3，直到聚类中心不再变化。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 Python 代码示例

```python
import pandas as pd
from sklearn.linear_model import LogisticRegression
from sklearn.cluster import KMeans

# 加载数据
data = pd.read_csv('consumer_data.csv')

# 准备数据
X = data[['age', 'income', 'purchase_history']]
y = data['purchase']

# 训练 Logistic 回归模型
model = LogisticRegression()
model.fit(X, y)

# 预测新消费者的购买概率
new_data = [[30, 50000, 10]]
probability = model.predict_proba(new_data)[0][1]
print(f"购买概率: {probability:.2f}")

# 使用 K-means 聚类对消费者进行细分
kmeans = KMeans(n_clusters=3)
kmeans.fit(X)

# 获取每个消费者的细分市场标签
labels = kmeans.labels_
print(f"细分市场标签: {labels}")
```

### 5.2 代码解释

* 首先，我们加载消费者数据并准备用于模型训练的数据。
* 然后，我们训练 Logistic 回归模型，并使用该模型预测新消费者的购买概率。
* 最后，我们使用 K-means 聚类将消费者分为不同的细分市场，并获取每个消费者的细分市场标签。
