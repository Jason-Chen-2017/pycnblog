## 4. 数学模型和公式详细讲解举例说明

### 4.1 SIR 模型

SIR 模型是一种经典的传染病模型，它将人群分为易感者（S）、感染者（I）和康复者（R）三类。

$$
\frac{dS}{dt} = -\beta SI
$$

$$
\frac{dI}{dt} = \beta SI - \gamma I
$$

$$
\frac{dR}{dt} = \gamma I
$$

其中：

* $\beta$ 表示传染率
* $\gamma$ 表示康复率

### 4.2 SEIR 模型

SEIR 模型是在 SIR 模型的基础上增加了潜伏期（E），将人群分为易感者（S）、潜伏者（E）、感染者（I）和康复者（R）四类。

$$
\frac{dS}{dt} = -\beta SI
$$

$$
\frac{dE}{dt} = \beta SI - \sigma E
$$

$$
\frac{dI}{dt} = \sigma E - \gamma I
$$

$$
\frac{dR}{dt} = \gamma I
$$

其中：

* $\sigma$ 表示潜伏期结束的速率

### 4.3 举例说明

假设某地区初始有 10000 人，其中 100 人感染，传染率为 0.001，康复率为 0.1，潜伏期为 7 天。使用 SEIR 模型可以模拟疫情的发展趋势，并预测未来一段时间内的感染人数。

## 5. 项目实践：代码实例和详细解释说明

### 5.1 Python 代码示例

```python
import matplotlib.pyplot as plt
import numpy as np

# 设置参数
N = 10000
I0 = 100
R0 = 0
E0 = 0
S0 = N - I0 - R0 - E0
beta = 0.001
gamma = 0.1
sigma = 1/7

# 定义 SEIR 模型
def seir_model(t, y):
    S, E, I, R = y
    dSdt = -beta * S * I / N
    dEdt = beta * S * I / N - sigma * E
    dIdt = sigma * E - gamma * I
    dRdt = gamma * I
    return [dSdt, dEdt, dIdt, dRdt]

# 设置时间跨度
t_span = (0, 100)
t_eval = np.linspace(t_span[0], t_span[1], 1000)

# 求解微分方程
from scipy.integrate import solve_ivp
sol = solve_ivp(seir_model, t_span, [S0, E0, I0, R0], t_eval=t_eval)

# 绘制结果
plt.plot(sol.t, sol.y[0], label='Susceptible')
plt.plot(sol.t, sol.y[1], label='Exposed')
plt.plot(sol.t, sol.y[2], label='Infected')
plt.plot(sol.t, sol.y[3], label='Recovered')
plt.xlabel('Time (days)')
plt.ylabel('Population')
plt.legend()
plt.show()
```

### 5.2 代码解释

* 首先，设置模型参数，包括总人口数、初始感染人数、传染率、康复率、潜伏期等。
* 然后，定义 SEIR 模型的微分方程。
* 接着，设置时间跨度和时间点。
* 使用 `solve_ivp` 函数求解微分方程，得到各个时间点的易感者、潜伏者、感染者和康复者人数。
* 最后，使用 `matplotlib` 库绘制结果，展示疫情的发展趋势。
