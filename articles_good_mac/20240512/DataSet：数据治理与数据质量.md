## 4. 数学模型和公式详细讲解举例说明

### 4.1 数据质量评估指标
   #### 4.1.1 准确性
     ##### 4.1.1.1 公式： $ 准确率 = \frac{正确记录数}{总记录数} $
     ##### 4.1.1.2 例： 100条数据中，90条正确，准确率为90%。
   #### 4.1.2 完整性
     ##### 4.1.2.1 公式： $ 完整率 = \frac{非空记录数}{总记录数} $
     ##### 4.1.2.2 例： 100条数据中，95条非空，完整率为95%。
   #### 4.1.3 一致性
     ##### 4.1.3.1 公式： $ 一致率 = \frac{符合规则的记录数}{总记录数} $
     ##### 4.1.3.2 例： 100条数据中，98条符合规则，一致率为98%。

### 4.2 数据清洗算法
   #### 4.2.1 K-近邻算法
     ##### 4.2.1.1 原理： 找到样本点的k个最近邻，用它们的平均值填充缺失值。
     ##### 4.2.1.2 例： 年龄字段缺失，找到最近的3个邻居的平均年龄填充。
   #### 4.2.2 箱线图法
     ##### 4.2.2.1 原理： 通过上下四分位数和异常值边界，识别异常值。
     ##### 4.2.2.2 例： 销售额字段，通过箱线图识别超出正常范围的异常值。

### 5.1 Python Pandas数据清洗

```python
import pandas as pd

# 读取数据
data = pd.read_csv('data.csv')

# 缺失值处理
data.fillna(method='ffill', inplace=True) # 向前填充

# 异常值处理
data = data[(data['sales'] > 100) & (data['sales'] < 1000)] # 剔除异常值

# 重复值处理
data.drop_duplicates(inplace=True)

# 保存数据
data.to_csv('cleaned_data.csv', index=False)
```
