### 第一章：宗白华的学术地位与《宗白华讲稿》的背景

#### 1.1 宗白华的生平与学术贡献

宗白华（1897-1986），原名宗之瑜，字白华，是中国现代美学和哲学领域的重要人物。他出生于湖北省黄冈市黄州区，自幼受到中国传统文化的熏陶，后赴德国留学，专攻美学和哲学。他在哲学、美学、艺术学等领域均有深厚的研究，被誉为“中国现代美学奠基人之一”。

宗白华在学术领域的贡献主要体现在以下几个方面：

1. **美学研究**：宗白华提出了一系列重要美学观点，如“意境论”、“美感论”、“艺术表现论”等，对后世美学研究产生了深远影响。

2. **哲学研究**：他深入探讨了中西哲学的异同，尤其是对西方哲学的引进和解读，为我国哲学研究开辟了新的道路。

3. **艺术学研究**：宗白华对艺术理论有着深刻的理解，其关于艺术本质、艺术创作和艺术欣赏的理论，对艺术学的发展起到了重要的推动作用。

4. **文化研究**：他关注传统文化与现代文明的冲突与融合，提出了“中西文化融合”的观点，为我国文化研究提供了新的思路。

#### 1.2 《宗白华讲稿》的编纂与出版背景

《宗白华讲稿》是宗白华在20世纪20年代末的多篇课堂讲稿和演讲稿的汇编，包括《美学》、《艺术学》、《形上学》、《孔子形上学《论格物》诸篇》等。这些讲稿和演讲稿原本分散在不同的时间和场合，由其弟子和学生整理成册，并于1981年由中华书局首次出版。

本书的编纂和出版具有以下背景：

1. **学术传承**：宗白华的学术思想在学术界有着广泛的影响，编纂和出版他的讲稿和演讲稿，有助于将他的思想传承下去，对后来的学者进行研究提供了宝贵的资料。

2. **文化反思**：20世纪20年代末，中国正处于动荡变革的时期，文化领域也面临着前所未有的挑战。宗白华的讲稿和演讲稿，以其深刻的思想和独特的视角，为当时的文化反思提供了重要的参考。

3. **学术整理**：宗白华的学术思想体系丰富而独特，对他的讲稿和演讲稿进行整理和出版，有助于系统地展现他的学术思想和学术风格。

综上所述，《宗白华讲稿》不仅是对宗白华学术生涯的一次回顾，也是对中国现代美学和哲学领域的一次梳理。本书的编纂和出版，对于研究者和读者来说，都具有重要的学术价值和思想启迪。

### 第二章：《宗白华讲稿》的内容概述

《宗白华讲稿》是一部集美学、艺术学、哲学于一体的学术著作，涵盖了宗白华在多个领域的独特见解和深入研究。以下是本书各个部分的主要内容概述：

#### 2.1 《美学》：西方美学的系统阐述

《美学》部分是宗白华对西方美学理论的系统阐述，内容涉及美学的基本概念、理论框架和研究方法。宗白华在本书中深入分析了美的本质、美感、审美经验等美学核心问题，并借鉴西方美学理论，结合中国传统文化，提出了一系列富有创新性的美学观点。

主要内容包括：

1. **美的本质**：宗白华认为，美的本质在于和谐与统一，是多样性的统一，是形式的完美与内容的丰富相结合。

2. **美感**：他提出美感是一种超越自我、融入自然的精神体验，是人类心灵的一种高级形态。

3. **审美经验**：宗白华通过对审美经验的深入探讨，揭示了审美活动的过程和特点，强调了审美经验的主观性和个体性。

4. **美学理论**：宗白华在本书中介绍了西方美学理论的发展历程，从古希腊到19世纪，对美学理论进行了系统的梳理和分析。

#### 2.2 《艺术学》：艺术理论的深度解析

《艺术学》部分是宗白华对艺术理论的深入探讨，内容涵盖了艺术本质、艺术创作、艺术欣赏等多个方面。他通过对艺术作品的分析，揭示了艺术创作和欣赏的内在规律，提出了一系列有启发性的艺术理论观点。

主要内容包括：

1. **艺术本质**：宗白华认为，艺术的本质在于表现，是艺术家通过形式表达内心情感和思想的过程。

2. **艺术创作**：他强调艺术创作是一种自由的创造，是艺术家对生活、自然和情感的独特感悟和表达。

3. **艺术欣赏**：宗白华提出，艺术欣赏是一种情感体验和理性思考的融合，是观众与艺术作品之间的心灵交流。

4. **艺术理论**：宗白华在本书中分析了不同艺术形式的特点和创作规律，如绘画、雕塑、音乐、文学等，提出了一系列有指导意义的艺术理论。

#### 2.3 《形上学》：中西哲学的对比研究

《形上学》部分是宗白华对中西哲学的对比研究，内容涉及中西哲学的基本观念、哲学体系的构建以及哲学思想的传承与发展。他通过对中西哲学的深入比较，揭示了两者之间的异同，并提出了一系列富有创见的哲学观点。

主要内容包括：

1. **哲学基本观念**：宗白华分析了中西哲学的基本观念，如“存在”、“理性”、“自由”等，揭示了两者在基本观念上的差异。

2. **哲学体系**：他通过对中西哲学体系的对比，探讨了中西哲学在理论架构和思想方法上的不同。

3. **哲学思想的传承与发展**：宗白华认为，中西哲学各自有着独特的传承与发展路径，两者之间的交流与融合是哲学发展的必然趋势。

4. **哲学思想的应用**：他在本书中提出了哲学思想在现代社会中的应用价值，强调了哲学对人类生活和社会发展的指导作用。

#### 2.4 《孔子形上学《论格物》诸篇》：古今“格物”思想的论辩

《孔子形上学《论格物》诸篇》部分是宗白华对古今“格物”思想的深入探讨，内容涉及《论语》中“格物”观念的解读、古今学者对“格物”思想的诠释以及“格物”观念在现代社会的应用。

主要内容包括：

1. **《论语》中的“格物”观念**：宗白华对《论语》中“格物”观念的解读，揭示了孔子“格物”思想的内涵和特点。

2. **古今学者对“格物”思想的诠释**：他分析了不同时期学者对“格物”观念的解读和诠释，探讨了“格物”思想在中国哲学史上的地位和影响。

3. **“格物”观念的现代解读**：宗白华将“格物”观念与现代科学精神相结合，提出“格物”观念在现代社会中的价值和意义。

4. **“格物”观念的应用**：他在本书中提出了“格物”观念在科学研究、社会发展和个人修养等方面的应用，强调了“格物”观念对现代社会的重要启示。

综上所述，《宗白华讲稿》内容丰富，涵盖了美学、艺术学、哲学等多个领域，通过对西方美学、艺术学的基本概念及研究方法的阐述，中西形上学的比较，以及古今“格物”思想的论辩，展现了宗白华在学术领域的深厚功底和独特见解。

### 第三章：《宗白华讲稿》的主要特点

#### 3.1 融贯中西：跨文化的思想交流

《宗白华讲稿》在内容上融贯中西，展现了跨文化的思想交流与融合。宗白华在研究西方美学、艺术学的同时，始终没有脱离中国传统文化的影响。他通过对中西哲学、美学的对比分析，揭示了两者之间的共通之处和差异，并提出了一系列具有跨文化视角的学术观点。

首先，宗白华在美学研究中，既借鉴了西方美学理论，如形式主义、表现主义等，又融合了中国传统美学思想，如意境、韵味等。他提出的美学观点，既具有西方美学体系的逻辑性和系统性，又富含中国传统文化的人文情怀和审美情趣。

其次，在艺术学研究中，宗白华通过对中西艺术形式的深入比较，分析了绘画、雕塑、音乐、文学等艺术形式的特点和创作规律。他强调了艺术创作中的个性表达和情感抒发，同时也关注艺术作品与观众之间的互动关系。这种跨文化的艺术研究方法，使得他的艺术理论具有更广泛的适用性和深远的影响力。

最后，在哲学研究中，宗白华通过对中西哲学的对比研究，揭示了中西哲学的基本观念、哲学体系的构建以及哲学思想的传承与发展。他提出的中西哲学融合的观点，为跨文化哲学研究提供了新的思路和方法。

《宗白华讲稿》在论述过程中引经据典，展现了深厚的学术底蕴。宗白华在讲稿中广泛引用了中外哲学、美学、文学等领域的经典著作，对各种理论观点进行了深入的分析和阐述。这种引经据典的方法，不仅增强了讲稿的理论深度，也为读者提供了丰富的学术资源。

首先，宗白华在美学研究中，引用了西方美学经典著作，如康德的《判断力批判》、黑格尔的《美学》等，对这些经典著作进行了深入解读，揭示了西方美学理论的发展脉络和核心观点。同时，他还结合了中国传统美学思想，如庄子的《逍遥游》、王夫之的《尚书引义》等，对这些思想进行了现代解读和阐释。

其次，在艺术学研究中，宗白华引用了中外艺术理论经典，如亚里士多德的《诗学》、达·芬奇的《绘画论》等，对这些经典著作进行了详细的解读和评述。同时，他还引用了中国古代艺术理论著作，如苏轼的《东坡志林》、董其昌的《画禅室随笔》等，对这些艺术理论进行了深入的探讨和分析。

最后，在哲学研究中，宗白华引用了西方哲学经典著作，如柏拉图的《理想国》、康德的《纯粹理性批判》等，对这些经典著作进行了系统的解读和批判。同时，他还引用了中国古代哲学著作，如《论语》、《道德经》等，对这些哲学思想进行了现代解读和诠释。

通过这些引经据典的论述，宗白华不仅展现了自己深厚的学术功底，也为读者提供了丰富的学术资源，使得《宗白华讲稿》具有很高的学术价值和思想深度。

《宗白华讲稿》在内容上具有很强的系统性，展现了知识体系的完整性。宗白华在讲稿中不仅对美学、艺术学、哲学等领域的具体问题进行了深入探讨，还对这些领域进行了整体的思考和建构。

其次，在艺术学领域，宗白华通过对绘画、雕塑、音乐、文学等艺术形式的深入研究，揭示了这些艺术形式的特点和创作规律。他提出的艺术理论体系，不仅涵盖了艺术创作的各个环节，还关注了艺术作品与观众之间的互动关系，形成了一套完整的艺术理论体系。

最后，在哲学领域，宗白华通过对中西哲学的对比研究，揭示了中西哲学的基本观念、哲学体系的构建以及哲学思想的传承与发展。他提出的哲学理论体系，既具有西方哲学的逻辑性和系统性，又融合了中国传统哲学的思想精髓，形成了一套具有完整性的哲学理论体系。

通过这些系统性的论述，宗白华不仅展现了自己在美学、艺术学、哲学等领域的深厚功底，也为读者提供了一套完整、系统的学术知识体系。

### 第四章：宗白华在美学、哲学领域的独特贡献

#### 4.1 对美学、哲学的基本概念及其研究方法的贡献

宗白华在美学和哲学领域的研究中，对基本概念及其研究方法有着独特的贡献。他的研究不仅深化了这些领域的理论基础，还为后来的学者提供了重要的参考。

首先，在美学领域，宗白华提出了一系列具有创新意义的基本概念。他强调美的本质在于和谐与统一，认为美是多样性的统一，是形式的完美与内容的丰富相结合。这一观点突破了传统美学对美的单一解读，使人们对美的认识更加全面和深刻。此外，宗白华还提出了美感、审美经验等概念，并对其进行了深入探讨。他认为美感是一种超越自我、融入自然的精神体验，是人类心灵的一种高级形态。审美经验则是审美活动的过程和特点，具有主观性和个体性。这些概念的提出，丰富了美学理论体系，为美学研究提供了新的视角。

其次，在研究方法上，宗白华主张跨学科的视角。他认为美学和哲学的研究应该借鉴其他学科的方法，如心理学、艺术学、文学等。他通过对这些学科方法的综合运用，形成了一套独特的研究方法。这种方法不仅使得他的研究更加深入和全面，也为后来的学者提供了借鉴。例如，他在美学研究中，既借鉴了西方美学理论，如形式主义、表现主义等，又融合了中国传统美学思想，如意境、韵味等。这样的跨学科研究方法，使得他的美学研究具有更高的学术价值和实践意义。

#### 4.2 中西比较研究的开创性意义

宗白华的中西比较研究在美学和哲学领域具有重要的开创性意义。他通过对中西哲学、美学的基本观念、哲学体系的构建以及哲学思想的传承与发展的深入比较，揭示了两者之间的异同，为中西文化的交流与融合提供了重要的理论支持。
