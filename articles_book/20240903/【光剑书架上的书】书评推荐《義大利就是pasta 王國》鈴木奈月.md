1. 適量熱水，加入鹽，將雞蛋打散，加入麵粉攪拌均勻。
2. 揉成麵團，蓋上濕布，饑一餓後，再揉一次。
3. 摺麵，切成適當的大小，搓成球形。
4. 用叉子輕輕壓成扁平形，做成義大利麵。
5. 煮鍋水，待水開後加入義大利麵，煮至麵條變軟，約8-10分鐘。

調味：
- 當麵條煮熟後，可以搭配番茄醬汁、奶油醬汁或火腿、芝士等食材。

#### 2. 義大利燉麵食譜

材料：
- 義大利麵：200克
- 蘿蔔：1個
- 白蘿蔔：1個
- 洋蔥：1個
- 蒜：3瓣
- 番茄醬：200毫升
- 植物油：2大勺
- 鹽：适量
- 糖：适量

步驟：
1. 洋蔥切絲，蒜切末，蘿蔔和白蘿蔔切塊。
2. 鍋中倒入油，加入洋蔥和蒜爆香。
3. 加入蘿蔔和白蘿蔔翻炒至軟爛。
4. 倒入番茄醬，加入適量鹽和糖調味。
5. 加入義大利麵，煮至麵條軟爛，汁液濃稠即可。

#### 3. 義大利奶油麵食譜

材料：
- 義大利麵：200克
- 奶油：50克
- 雞蛋：2個
- 鹽：适量
- 蒜末：1小勺

步驟：
1. 鍋中加水煮沸，加入適量鹽，加入義大利麵煮至軟爛。
2. 鍋中倒入奶油融化，加入雞蛋攪拌均勻。
3. 加入義大利麵，攪拌均勻，直到麵條吸收奶油的鮮美口感。
4. 最後加入蒜末提香即可。
