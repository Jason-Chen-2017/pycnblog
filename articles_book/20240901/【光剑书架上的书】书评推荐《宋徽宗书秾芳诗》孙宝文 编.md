### 宋徽宗书秾芳诗的基本背景

《宋徽宗书秾芳诗》是中国书法史上的瑰宝，它的作者宋徽宗赵佶（1082-1155）是北宋时期的第七位皇帝，同时也是一位杰出的书法家和艺术家。赵佶在位期间，致力于文化和艺术的发展，推行了一系列的改革措施，使得北宋的文学、艺术达到了前所未有的高度。

《宋徽宗书秾芳诗》的创作背景颇为特殊。据记载，此诗是宋徽宗为宫廷花园中的一株花木所题写的诗篇。这首诗名为《秾芳诗》，全文如下：

```
秾芳依翠萼，焕烂一庭中。
零露霑如醉，残霞照似融。
丹青难下笔，造化独留功。
舞蝶迷香径，翩翩逐晚风。
```

这首诗描绘了花朵的美丽和生机，充满了对自然的赞美之情。宋徽宗用瘦金体书写这首诗，既表现了瘦金体的独特魅力，又体现了他对自然之美的深刻感悟。

关于《宋徽宗书秾芳诗》的出版信息，本书属于“中国著名碑帖彩色放大本”系列，由孙宝文编。这一系列书籍底本精良，印制精美，大8开本，为首次彩色放大本。书籍不仅提供了原文，还延请专家进行了释文，方便读者研究学习。这种印制方式使得《宋徽宗书秾芳诗》得以更广泛地传播和欣赏，让更多的书法爱好者能够近距离感受到宋徽宗瘦金体的艺术魅力。

在台北故宫博物院，珍藏着一卷大字瘦金书《秾芳诗》。此卷为绢地，共二十行，每行两个字，共计四十字。其纵二十七点七厘米，横三百四十厘米。卷后还有陈邦彦的观款：“宣和书画超轶千古，此卷以画法作书，脱去笔墨畦径，行间如幽兰丛竹，冷冷作风雨声，真神品也。”这一评价充分体现了《宋徽宗书秾芳诗》的艺术价值和高超技艺。

### 秾芳诗的内容解读

《秾芳诗》是宋徽宗赵佶的一首五言律诗，全文如下：

```
秾芳依翠萼，焕烂一庭中。
零露霑如醉，残霞照似融。
丹青难下笔，造化独留功。
舞蝶迷香径，翩翩逐晚风。
```

这首诗通过对花朵的描绘，展现了自然界的生机与美丽。诗中的“秾芳”指的是盛开的花朵，“翠萼”是指花蕾的绿色部分。首句“秾芳依翠萼，焕烂一庭中”描绘了花朵盛开时的壮丽景象，将读者带入一个繁花似锦的世界。

接下来的“零露霑如醉，残霞照似融”两句，通过描绘露水和晚霞，进一步加强了诗中自然景象的生动感。露水的“霑如醉”形象地表现出露水的湿润和光泽，而“残霞照似融”则通过晚霞的余晖与花朵的互动，营造出一种柔和而充满诗意的氛围。

第三句“丹青难下笔，造化独留功”中的“丹青”指的是绘画，这里用来比喻花朵的美丽难以用画笔完全表达。宋徽宗通过这句话表达了对自然造物之神奇和美丽的敬畏之情，强调了大自然的鬼斧神工。

最后两句“舞蝶迷香径，翩翩逐晚风”以动态的舞蝶和轻柔的晚风，为整首诗增添了生机与活力。舞蝶在花间穿梭，仿佛在追逐晚风，形成了一幅充满活力和生机的画面。这一结尾不仅与开头形成了呼应，也强化了诗中自然景象的生动感和美感。

通过这首诗，宋徽宗不仅表达了对自然之美的赞美，也展现了他对生命和宇宙的深刻感悟。诗中的每一句都充满了细腻的情感和丰富的意象，通过自然景象的描绘，传达了诗人内心的世界和对人生的感悟。
