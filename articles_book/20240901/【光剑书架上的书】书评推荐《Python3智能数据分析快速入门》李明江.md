## 第二章：Python基础语法

Python基础语法是掌握Python编程的关键。本章详细介绍了Python的基础语法，包括编码声明、注释、缩进等，以及常见的数据类型和运算符。

### 2.1 编码声明与注释

编码声明是Python代码中不可或缺的一部分。Python默认使用UTF-8编码，但在某些情况下，需要明确指定编码类型。

#### 2.1.1 编码声明

- **UTF-8编码**：在Python代码的开头添加`# -*- coding: utf-8 -*-`，即可指定代码的UTF-8编码。
- **其他编码**：根据需要，还可以使用`# -*- coding: iso-8859-1 -*-`等编码声明。

#### 2.1.2 注释

注释是解释代码的重要工具。Python中的注释分为单行注释和多行注释。

- **单行注释**：以`#`开头，如`# 这是单行注释`。
- **多行注释**：使用三个单引号`'''`或三个双引号`"""`，如：
  ```python
  '''
  这是多行注释
  '''
  ```

### 2.2 常见数据类型

Python提供了多种数据类型，包括字符串（str）、列表（list）、元组（tuple）、字典（dict）和集合（set）。

#### 2.2.1 字符串（str）

字符串是Python中用于表示文本的数据类型。字符串可以通过单引号（'”）或双引号（”）定义。

- **单引号定义**：`a = 'Hello, World!'`
- **双引号定义**：`b = "Hello, World!"`

#### 2.2.2 列表（list）

列表是Python中用于存储有序集合的数据类型。列表可以通过方括号（[]）定义。

- **创建列表**：`c = [1, 2, 3, 4, 5]`
- **访问元素**：`c[0]`返回列表的第一个元素
- **修改元素**：`c[0] = 0`将列表的第一个元素修改为0

#### 2.2.3 元组（tuple）

元组是Python中用于存储不可变有序集合的数据类型。元组通过圆括号（()）定义。

- **创建元组**：`d = (1, 2, 3, 4, 5)`
- **访问元素**：`d[0]`返回元组的第一个元素
- **修改元素**：由于元组是不可变的，因此不能直接修改元素。

#### 2.2.4 字典（dict）

字典是Python中用于存储键值对的数据类型。字典通过大括号（{ }）定义。

- **创建字典**：`e = {'name': 'Alice', 'age': 25}`
- **访问键值**：`e['name']`返回字典中的'name'键对应的值
- **修改键值**：`e['name'] = 'Bob'`将字典中的'name'键对应的值修改为'Bob'

#### 2.2.5 集合（set）

集合是Python中用于存储无序且不可重复元素的集合数据类型。集合通过花括号（{ }）定义。

- **创建集合**：`f = {1, 2, 3, 4, 5}`
- **访问元素**：集合中的元素无法直接访问，但可以通过成员运算符`in`和`not in`进行判断。
- **修改集合**：集合支持添加（`add()`）和移除（`remove()`）元素。

### 2.3 运算符

Python提供了丰富的运算符，包括算术运算符、逻辑运算符、成员运算符和位运算符。

#### 2.3.1 算术运算符

算术运算符用于对数值进行运算。

- **加法（+）**：`a + b`将两个数值相加
- **减法（-）**：`a - b`将两个数值相减
- **乘法（*）**：`a * b`将两个数值相乘
- **除法（/）**：`a / b`将两个数值相除，返回浮点数
- **取模（%）**：`a % b`返回两个数值相除的余数
- **整数除法（//）**：`a // b`返回两个数值相除的整数部分

#### 2.3.2 逻辑运算符

逻辑运算符用于对布尔值进行运算。

- **与（and）**：`a and b`当两个操作数都为真时，返回真
- **或（or）**：`a or b`当两个操作数中有一个为真时，返回真
- **非（not）**：`not a`当操作数为真时，返回假

#### 2.3.3 成员运算符

成员运算符用于判断元素是否属于集合。

- **in**：`a in b`当元素存在于集合中时，返回真
- **not in**：`a not in b`当元素不存在于集合中时，返回真

#### 2.3.4 位运算符

位运算符用于对整数进行位操作。

- **按位与（&）**：`a & b`对两个整数的相应位进行与操作
- **按位或（|）**：`a | b`对两个整数的相应位进行或操作
- **按位异或（^）**：`a ^ b`对两个整数的相应位进行异或操作
- **左移（<<）**：`a << b`将整数向左移动b位
- **右移（>>）**：`a >> b`将整数向右移动b位

## 第三章：控制语句

控制语句是编程的核心，它允许程序根据不同的条件执行不同的代码块。Python中的控制语句包括条件语句、循环语句和异常处理。本章将详细介绍这些控制语句，并展示如何在实际编程中使用它们。

### 3.1 条件语句

条件语句用于根据条件的真假来执行不同的代码块。Python中的条件语句包括`if`、`elif`和`else`。

#### 3.1.1 单分支条件语句

单分支条件语句是最简单的条件语句形式，它根据条件的真假执行不同的代码块。

```python
if condition:
    # 条件为真时执行的代码块
```

例如，以下代码根据用户输入的数字判断是否为正数：

```python
num = int(input("请输入一个数字："))
if num > 0:
    print("这是一个正数。")
```

#### 3.1.2 双分支条件语句

双分支条件语句在条件为真或假时分别执行不同的代码块。

```python
if condition:
    # 条件为真时执行的代码块
else:
    # 条件为假时执行的代码块
```

例如，以下代码根据用户输入的数字判断是正数还是负数：

```python
num = int(input("请输入一个数字："))
if num > 0:
    print("这是一个正数。")
else:
    print("这是一个负数。")
```

#### 3.1.3 多分支条件语句

多分支条件语句使用`elif`关键字来处理多个条件。

```python
if condition1:
    # condition1 为真时执行的代码块
elif condition2:
    # condition2 为真时执行的代码块
elif condition3:
    # condition3 为真时执行的代码块
else:
    # 以上条件都不为真时执行的代码块
```

例如，以下代码根据用户输入的分数判断成绩等级：

```python
score = int(input("请输入你的分数："))
if score >= 90:
    print("成绩等级为A。")
elif score >= 80:
    print("成绩等级为B。")
elif score >= 70:
    print("成绩等级为C。")
elif score >= 60:
    print("成绩等级为D。")
else:
    print("成绩等级为E。")
```

### 3.2 循环语句

循环语句用于重复执行某个代码块，直到满足特定的条件为止。Python中的循环语句包括`for`和`while`。

#### 3.2.1 `for`循环

`for`循环用于遍历序列（如列表、元组、字典、集合、字符串）中的元素。

```python
for element in sequence:
    # 对每个元素执行的代码块
```

例如，以下代码遍历列表并打印每个元素：

```python
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num)
```

#### 3.2.2 `while`循环

`while`循环用于根据条件重复执行代码块。

```python
while condition:
    # 当条件为真时重复执行的代码块
```

例如，以下代码使用`while`循环计算1到10的数字总和：

```python
total = 0
num = 1
while num <= 10:
    total += num
    num += 1
print("总和为：", total)
```

#### 3.2.3 循环控制语句

循环控制语句用于在循环中提前退出或继续循环。

- `break`语句：用于提前退出循环。
  ```python
  while num <= 10:
      if num == 5:
          break
      print(num)
      num += 1
  ```

- `continue`语句：用于跳过当前循环迭代，继续下一次迭代。
  ```python
  while num <= 10:
      if num == 5:
          continue
      print(num)
      num += 1
  ```

### 3.3 异常处理

异常处理用于处理程序运行过程中可能发生的错误。Python中的异常处理使用`try`、`except`、`else`和`finally`。

#### 3.3.1 基本语法

```python
try:
    # 尝试执行的代码块
except ExceptionType:
    # 捕获异常后执行的代码块
else:
    # 没有异常时执行的代码块（可选）
finally:
    # 无论是否发生异常都执行的代码块（可选）
```

例如，以下代码使用异常处理来尝试执行一个可能引发错误的操作：

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("发生了一个除以零的错误。")
else:
    print("计算结果为：", result)
finally:
    print("异常处理完成。")
```

#### 3.3.2 自定义异常

除了捕获Python内置的异常类型，还可以自定义异常。

```python
class MyError(Exception):
    pass

try:
    if condition:
        raise MyError("这是一个自定义异常。")
except MyError as e:
    print(e)
```

### 4.1 函数

函数是Python中的一个重要组成部分，它允许我们将代码组织成可重用的代码块。通过函数，我们可以将复杂的问题分解为更小、更易于管理的部分。

#### 4.1.1 定义函数

在Python中，定义函数的基本语法如下：

```python
def function_name(parameters):
    # 函数体
    return result
```

例如，以下是一个简单的函数，用于计算两个数的和：

```python
def add(a, b):
```

#### 4.1.2 传递参数

  ```python
  def greet(name, age):
  ```

NumPy数组可以通过多种方式创建，包括使用内置函数、列表转换和手动创建。

  使用`numpy.array()`函数可以将Python列表转换为NumPy数组。

  ```python
  import numpy as np

  arr2 = np.array([[1, 2], [3, 4]])
  ```

  ```python
  arr3 = np.array([[1, 2, 3], [4, 5, 6]])
  ```

- **手动创建数组**


NumPy数组支持传统的索引和切片操作，使得数据访问变得灵活高效。

  ```python
  arr10 = np.array([[1, 2], [3, 4]])

  print(arr10 - arr11)  # 广播减法
  ```

- **统计函数**

#### 8.1.1 数据清洗

数据清洗是处理缺失值、异常值和重复值的过程。

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

iris = load_iris()
X, y = iris.data, iris.target

y = y[~np.isnan(X).any(axis=1)]

# 删除异常值
X = X[~np.isinf(X).any(axis=1)]
y = y[~np.isinf(X).any(axis=1)]

# 删除重复值
X = np.unique(X, axis=0)
y = np.unique(y, axis=0)
```
