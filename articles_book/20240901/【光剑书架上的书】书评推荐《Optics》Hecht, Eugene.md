   - 准备凸透镜和光源，将光源放置在凸透镜前方。
   - 观察光线通过凸透镜后的成像现象，记录成像位置和大小。
   - 分析成像原理，验证凸透镜的成像规律。

3. 激光实验
   - 准备激光器和光屏，将激光器放置在光具座上。
   - 观察激光束在光屏上的传播和反射现象，记录激光束的方向和亮度。
   - 分析激光的特性和应用。

4. 光学传感器实验
   - 准备光敏电阻和光纤传感器，将传感器连接到实验装置上。
   - 观察传感器对光的响应，记录传感器的工作原理和应用。

**附录三：光学实验注意事项**

1. 实验过程中，注意安全，避免激光直射眼睛。
2. 实验器材要按照规定使用，切勿擅自拆卸或损坏。
3. 实验操作要规范，避免因操作不当导致实验失败。
4. 实验数据要真实可靠，切勿伪造实验结果。
