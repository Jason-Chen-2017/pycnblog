   - 本书系统介绍了中国乐器的发展历程，详细描述了各种乐器的起源、演变及其在音乐文化中的地位。黄翔鹏先生是中国音乐史研究的权威，他的著作具有很高的学术价值。

2. **《中国古代音乐文化史》**：作者：杜亚雄
   - 杜亚雄先生对中国古代音乐文化有着深入的研究，本书从历史的角度，详细介绍了中国古代音乐文化的演变和发展，内容丰富，论述详实。

4. **《中国民族音乐》**：作者：张骏祥
   - 张骏祥先生是中国民族音乐研究的专家，本书系统介绍了中国各民族的音乐特点、乐器、歌曲等，图文并茂，适合广大音乐爱好者阅读。

5. **《中国传统乐器文化》**：作者：彭丽媛
   - 本书以中国传统乐器为主题，介绍了各种乐器的制作工艺、演奏技巧、文化内涵等，对了解中国乐器文化提供了全面的资料。

6. **《中国传统音乐资料汇编》**：作者：中国音乐学院编

**附录二：中国乐器简介**

1. **古琴**：又称七弦琴，是中国最古老的乐器之一，具有深厚的历史和文化底蕴。古琴音色悠扬，悠远飘渺，常被视为高雅文化的象征。
2. **琵琶**：起源于中国汉代，是四大古琴之一，音色激昂，节奏感强，常被用来抒发侠义精神。
3. **笛**：起源于春秋时期，音色清脆，悠扬动听，常被视为幽雅与超脱的象征。
4. **筝**：起源于战国时期，音色悠扬，节奏舒缓，常被视为深情与缠绵的象征。
5. **胡琴**：起源于汉代，音色柔和，节奏平缓，常被视为平凡与生活的象征。
