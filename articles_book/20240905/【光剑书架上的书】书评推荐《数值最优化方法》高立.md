#include <pthread.h>
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>

#define N_THREADS 4

void* thread_function(void* arg) {
   long tid;
   tid = (long)arg;
   printf("Thread ID: %ld\n", tid);
   sleep(1);
   return NULL;
}

int main(int argc, char* argv[]) {
   pthread_t threads[N_THREADS];
   int i, rc;

   for (i = 0; i < N_THREADS; i++) {
       rc = pthread_create(&threads[i], NULL, thread_function, (void*)i);
       if (rc) {
           printf("Error:unable to create thread %d\n", rc);
           exit(-1);
       }
   }

   for (i = 0; i < N_THREADS; i++) {
       pthread_join(threads[i], NULL);
   }

   return 0;
}

#### 补充说明

- **线程数量**：本例中，我们创建了4个线程。线程的数量可以根据具体需求进行调整。

- **线程ID**：每个线程都会接收到一个ID参数，用于识别线程。在本例中，线程ID即为线程在数组中的索引。

- **线程函数**：线程函数`thread_function`用于执行线程的特定任务。在本例中，线程函数仅用于打印线程ID并休眠1秒钟。

- **创建线程**：使用`pthread_create`函数创建线程。该函数接受线程ID、线程属性、线程函数和线程参数。

- **等待线程结束**：使用`pthread_join`函数等待所有线程结束。这将确保主线程在所有线程完成执行后才能结束。

#### 注意事项

- **线程安全**：在多线程程序中，应确保共享资源的安全访问，避免数据竞争和死锁等问题。

- **线程栈大小**：默认情况下，线程栈大小较小。如果线程执行的操作需要较大的栈空间，可能需要调整线程栈大小。

- **线程取消**：可以使用`pthread_cancel`函数取消线程的执行。在取消线程时，应确保线程已经释放了所有资源。

- **线程优先级**：在特定平台上，线程的优先级可能影响线程的执行顺序。可以根据需要调整线程的优先级。
