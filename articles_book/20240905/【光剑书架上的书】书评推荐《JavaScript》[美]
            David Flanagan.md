#### 常用JavaScript函数

1. **字符串处理函数**
   - `toUpperCase()`: 将字符串转换为小写。
   - `toLowerCase()`: 将字符串转换为大写。
   - `split()`: 将字符串按指定分隔符分割成数组。
   - `substring()`: 返回字符串的子字符串。
   - `substr()`: 返回字符串的子字符串（基于起始索引和长度）。

2. **数组处理函数**
   - `indexOf()`: 返回指定元素在数组中的索引。
   - `lastIndexOf()`: 返回指定元素在数组中的最后一个索引。
   - `forEach()`: 对数组的每个元素执行一次回调函数。
   - `map()`: 创建一个新数组，其结果是该数组中的每个元素调用一次提供的函数后的返回值。
   - `filter()`: 创建一个新数组，包含通过所提供函数实现的测试的所有元素。

3. **数学处理函数**
   - `parseInt()`: 将字符串解析为整数。
   - `parseFloat()`: 将字符串解析为浮点数。
   - `Math.random()`: 返回一个随机数。
   - `Math.floor()`: 返回小于或等于一个给定数字的最大整数。
   - `Math.ceil()`: 返回大于或等于一个给定数字的最小整数。

4. **日期处理函数**
   - `Date()`: 创建一个日期对象。
   - `Date.parse()`: 解析日期字符串并返回毫秒表示的日期。
   - `Date.UTC()`: 返回日期对象的UTC日期。
   - `Date.now()`: 返回当前时间的毫秒表示。

5. **其他函数**
   - `JSON.parse()`: 将JSON字符串转换为JavaScript对象。
   - `JSON.stringify()`: 将JavaScript对象转换为JSON字符串。

1. **选择器**
   - `$()`: 选择一个或多个DOM元素。
   - `$("#id")`: 选择具有指定ID的元素。

   - `.querySelector()`: 使用CSS选择器选择元素。

2. **DOM操作**
   - `.append()`: 向指定元素追加内容。
   - `.prepend()`: 向指定元素前插入内容。
   - `.remove()`: 删除匹配的元素。
   - `.replaceWith()`: 用新元素替换匹配的元素。

3. **事件处理**
   - `.on()`: 绑定事件监听器。
   - `.off()`: 解绑事件监听器。

4. **动画效果**
