#### **前言：干酪的魅力与本书的目的**

干酪，一种源自欧洲的美食，随着时代的变迁，逐渐风靡全球。它不仅是一种美味的食品，更是一种文化的象征。干酪的魅力在于其多样的风味和独特的口感，让人一试难忘。日本文艺社的《干酪品鉴大全》，正是为了帮助读者深入了解干酪，从而更好地欣赏和享用这一美味而创作的。

本书的主要目的是为读者提供一份全面的干酪品鉴指南。书中不仅介绍了干酪的分类、特点、风味、食用方法等基础知识，还提供了详细的选购、切割、保存方法，以及与葡萄酒、面包等搭配的美食做法。无论你是干酪的初学者，还是资深爱好者，这本书都将为你带来全新的味觉体验。

#### **干酪的分类与特点**

干酪是一种通过发酵、凝乳、成型等工艺制成的食品，其种类繁多，风味各异。根据制作工艺和原料的不同，干酪可以分为以下几类：

1. **硬质干酪**：硬质干酪是干酪中最为常见的一种，其质地坚硬，口感丰富。如帕尔马干酪、切达干酪等。

2. **软质干酪**：软质干酪的质地柔软，口感细腻，适合直接食用或作为蛋糕、沙拉等食物的配料。如布里干酪、卡蒙贝尔干酪等。

3. **半硬质干酪**：半硬质干酪介于硬质干酪和软质干酪之间，既具有硬质干酪的口感，又有软质干酪的细腻。如戈尔贡佐拉干酪、蓝带干酪等。

4. **软质发酵干酪**：软质发酵干酪是通过发酵过程制成的，质地柔软，口感丰富，如卡芒贝尔干酪、蒙特利干酪等。

5. **蓝纹干酪**：蓝纹干酪因其独特的蓝纹而闻名，其风味独特，口感丰富。如罗克福干酪、斯蒂尔顿干酪等。

6. **羊奶酪**：羊奶酪以其浓郁的奶香和独特的口感而受到喜爱。如羊乳酪、羊乳布丁等。

7. **水牛奶酪**：水牛奶酪以其丰富的营养和独特的口感而受到推崇。如水牛芝士、水牛乳酪等。

每种干酪都有其独特的风味和口感，只有亲自品尝，才能领略其中的魅力。

#### **干酪的风味与食用方法**

干酪的风味丰富多样，可以从口感、味道、香气等多个方面进行品鉴。以下是一些常见的干酪风味和食用方法：

1. **口感**：干酪的口感可以从软到硬，从细腻到粗糙。软质干酪口感细腻，适合直接食用；硬质干酪口感粗糙，适合切割或磨碎后加入菜肴。

2. **味道**：干酪的味道可以从甜到咸，从清淡到浓郁。甜味干酪适合作为甜点食用；咸味干酪适合作为开胃菜或配酒食用。

3. **香气**：干酪的香气可以从果香到奶香，从清淡到浓郁。果香干酪适合搭配水果食用；奶香干酪适合搭配面包或葡萄酒食用。

4. **食用方法**：干酪的食用方法多种多样，可以单独食用，也可以作为菜肴的配料。例如，卡芒贝尔干酪可以搭配水果沙拉；帕尔马干酪可以搭配意大利面；切达干酪可以加入汉堡肉饼中。

#### **干酪的选购与保存**

选购干酪时，需要考虑以下几个因素：

1. **产地**：不同产地的干酪风味各异，应根据个人口味选择。

2. **种类**：根据个人喜好选择干酪的种类，如硬质干酪、软质干酪等。

3. **保质期**：选购干酪时，应关注其保质期，选择新鲜的干酪。

4. **包装**：干酪的包装应完好无损，避免购买已破损或受潮的干酪。

保存干酪时，需要注意以下几点：

1. **温度**：干酪应存放在冰箱中，以保持其新鲜度和口感。

2. **湿度**：干酪应存放在干燥的环境中，避免受潮。

3. **切割**：在食用前，应将干酪切割成适当的大小，以避免浪费。

#### **干酪与葡萄酒、面包的搭配**

干酪与葡萄酒、面包的搭配，可以提升干酪的口感和风味。以下是一些常见的搭配建议：

1. **干酪与葡萄酒**：软质干酪搭配轻酒，如卡芒贝尔干酪搭配香槟；硬质干酪搭配红酒，如帕尔马干酪搭配赤霞珠。

2. **干酪与面包**：软质干酪搭配软面包，如卡芒贝尔干酪搭配法式长棍面包；硬质干酪搭配硬面包，如切达干酪搭配意大利面包。

3. **干酪与水果**：果香干酪搭配水果，如戈尔贡佐拉干酪搭配草莓；奶香干酪搭配水果，如蒙特利干酪搭配苹果。

#### **干酪美食的制作**

干酪不仅可以作为配菜或开胃菜，还可以作为主要食材制作美食。以下是一些简单的干酪美食制作方法：

1. **干酪沙拉**：将切达干酪、生菜、西红柿、黄瓜等食材混合，加入沙拉酱拌匀。

2. **干酪汉堡**：将切达干酪加入汉堡肉饼中，烤制至熟透。

3. **干酪意面**：将帕尔马干酪撒在煮好的意大利面上，加入适量的橄榄油和黑胡椒。

4. **干酪烤面包**：将卡芒贝尔干酪铺在面包片上，放入烤箱烤至奶酪融化。

#### **干酪的历史与文化**

干酪的历史可以追溯到公元前5000年的中东地区，随着时间的推移，干酪逐渐传播到世界各地。干酪不仅是食物，更是文化的一部分。在不同国家和地区，干酪有着不同的制作工艺和风味特点。

1. **欧洲干酪**：欧洲是干酪的发源地，拥有丰富的干酪种类和制作工艺。法国、意大利、瑞士等国家的干酪闻名于世。

2. **美国干酪**：美国的干酪产业始于19世纪，经过几百年的发展，形成了独特的干酪风格，如切达干酪、蓝带干酪等。

3. **亚洲干酪**：亚洲地区的干酪制作相对较晚，但随着人们对健康饮食的追求，干酪在亚洲也逐渐受到欢迎。

#### **干酪的未来与发展**

随着人们对健康饮食的关注，干酪作为一种营养丰富、口感独特的食品，在未来有着广阔的发展前景。以下是干酪未来发展的几个方向：

1. **健康干酪**：研发低脂、低盐、高钙等健康干酪，满足消费者的健康需求。

2. **有机干酪**：推广有机干酪，提高干酪的品质和附加值。

3. **个性化干酪**：根据消费者的口味和需求，定制个性化干酪。

4. **科技与干酪**：运用现代科技，提高干酪的产量和品质，降低生产成本。
