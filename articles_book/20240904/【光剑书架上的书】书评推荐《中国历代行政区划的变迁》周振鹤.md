### 第一章 秦代行政区划的设立与郡县制的推行

秦代是中国历史上一个重要的转折点，秦始皇统一六国后，为了巩固统治，实施了一系列政治、军事和文化上的改革，其中最为关键的一项就是行政区划的设立与郡县制的推行。这一改革不仅奠定了中国历代行政区划的基础，也对后世的政治制度产生了深远的影响。

#### 秦代行政区划的设立

公元前221年，秦王嬴政完成统一大业，建立了秦朝，成为中国历史上第一个统一的中央集权国家。为了巩固新成立的政权，秦始皇开始着手进行行政区划的改革。他采纳了丞相李斯的建议，实行“分天下以为三十六郡”的政策。这一政策的主要内容包括：

1. **设立郡县**：秦始皇将全国划分为三十六个郡，每个郡下设若干县。郡县的长官由中央任命，实行中央垂直管理的体制，以确保政令的统一和政权的稳定。

2. **废除分封制**：在之前的诸侯分封制下，地方诸侯拥有较大的独立性和权力，易于形成割据势力。秦始皇废除分封制，改为设立郡县，实现了地方政权的集中统一。

3. **推广法治**：在行政区划的改革过程中，秦始皇还推行了法治政策，强调法律面前人人平等，进一步强化了中央集权。

#### 郡县制的推行

秦始皇的行政区划改革并非一蹴而就，而是经历了长期的酝酿和实施。具体来说，郡县制的推行主要经历了以下几个阶段：

1. **试点推行**：在统一六国后，秦始皇首先在部分地区试点推行郡县制，以验证其可行性和效果。

2. **全面推广**：在试点取得成功的基础上，秦始皇逐渐将郡县制推广到全国范围。公元前221年，三十六郡的设立标志着郡县制在全国范围内的正式推行。

3. **不断调整**：在推行郡县制的过程中，秦始皇也根据实际情况进行了多次调整。例如，随着边疆地区的开发和治理需要，秦始皇增设了一些郡县，以加强中央对边疆的控制。

#### 秦代行政区划改革的影响

秦代行政区划的设立和郡县制的推行，对中国历史产生了深远的影响。具体表现在以下几个方面：

1. **中央集权**：通过设立郡县，实行中央垂直管理，秦始皇大大加强了中央集权，实现了对全国的有效统治。这一体制为后世的中国政治制度奠定了基础。

2. **地方治理**：郡县制的实施，使得地方治理更加规范和高效。郡县长官由中央任命，有利于统一政令，减少地方割据现象。

3. **法律统一**：推广法治，强调法律面前人人平等，有助于社会的稳定和进步。

4. **行政管理**：郡县制为中国的行政管理提供了有效的模式，这一制度在后世被不断沿用和发展，成为中国行政管理的基本框架。

总之，秦代行政区划的设立和郡县制的推行，不仅为秦朝的统治奠定了基础，也为后世的政治制度提供了宝贵的经验。这一变革标志着中国政治制度的重大进步，对中国历史产生了深远的影响。

### 第二章 汉代行政区划的变迁

秦代行政区划的设立为中国历代行政制度奠定了基础，而汉代则在继承和发展的过程中，进一步推动了行政区划的变迁。汉代行政区划的变迁不仅体现了中央集权制度的发展，也反映了社会经济的变革和政治稳定的需求。

#### 汉代行政区划的演变

1. **初期行政区划**：秦统一后，全国设立三十六郡，但到了汉代，这一制度有所调整。西汉初期，全国共有郡国百余，其中郡和国并行设置，形成了“郡国制”的局面。

2. **郡国并行制**：郡国并行制是西汉时期的主要行政区划形式，郡设郡守，国设诸侯王。这种制度在一定程度上保持了地方权力的相对独立性，但也导致了地方割据和诸侯叛乱的问题。

3. **郡国制度的调整**：为了解决诸侯王国问题，汉武帝时期进行了大规模的行政区划调整。他推行了“推恩令”，通过允许诸侯王分割封地给子弟，削弱了诸侯王的权力，加强了中央对地方的控制。

4. **汉武帝的行政区划改革**：汉武帝时期，全国行政区划进一步调整，郡国数量增多，郡县制逐渐取代了郡国并行制，成为主要的行政区划形式。这一时期，郡县逐渐取代了诸侯王国，中央集权得到进一步加强。

#### 汉代行政区划的特点

1. **中央集权的发展**：汉代行政区划的变迁，尤其是汉武帝时期的改革，大大加强了中央集权。通过削弱诸侯王的权力，中央政府实现了对地方更直接的控制。

2. **地方治理的规范化**：郡县制的推行，使得地方治理更加规范。郡县长官由中央任命，实行统一的行政管理，有利于政令的统一和行政效率的提高。

3. **行政区划的灵活性**：汉代在行政区划上具有一定的灵活性，可以根据实际情况进行调整。例如，在边疆地区和少数民族地区，设立了特别行政区，以适应不同的治理需求。

4. **行政区划与经济发展**：汉代行政区划的调整与经济发展密切相关。随着经济的繁荣，人口增加，新的郡县不断设立，以满足社会发展的需求。

#### 汉代行政区划的历史意义

汉代行政区划的变迁，不仅为中国历代行政制度的发展提供了重要经验，也对中国历史产生了深远的影响。具体表现在以下几个方面：

1. **中央集权制度的完善**：汉代通过行政区划的调整，进一步完善了中央集权制度，为中国历史上的统一多民族国家的形成奠定了基础。

2. **地方治理的规范**：郡县制的推行，使得地方治理更加规范和高效，为后世行政制度的发展提供了有益的经验。

3. **社会经济的发展**：行政区划的调整与经济发展紧密相关，通过设立新的郡县，促进了边疆地区和少数民族地区的开发，推动了整个社会经济的繁荣。

4. **政治稳定**：通过加强中央集权和规范地方治理，汉代实现了政治稳定，为后续的统一和发展创造了良好的条件。

总之，汉代行政区划的变迁是中国历史上一个重要的转折点，它不仅为后来的行政制度发展提供了宝贵的经验，也对中国的政治、经济和社会产生了深远的影响。

### 第三章 唐代行政区划的变迁

唐代是中国历史上的一个辉煌时期，不仅经济繁荣、文化昌盛，而且在政治制度方面也进行了许多创新和改革。其中，唐代行政区划的变迁尤为显著，它既反映了当时国家治理的需求，也体现了唐代政治制度的独特特点。

#### 唐代行政区划的基本情况

1. **行政区划的层次**：唐代行政区划分为省、道、府、州、县五级，这种多层次、分级管理的体制在当时是较为先进的。

2. **省和道的设立**：唐朝在地方设立“道”，以加强对地方的管理。道下设府、州、县，形成了省道并行的体制。

3. **节度使制度**：唐代中期以后，随着边疆地区的开发和国防需求的增加，设立了节度使制度。节度使负责一定区域内的军政事务，这种制度在一定程度上加强了中央对边疆的控制，但也为后来的藩镇割据埋下了隐患。

#### 唐代行政区划的特点

1. **中央集权的强化**：唐代初期，中央集权制度得到加强，通过设立“道”和“府”，中央政府对地方的控制更加直接和有效。

2. **行政区划的灵活性**：唐代在行政区划上具有较强的灵活性，可以根据实际情况进行调整。例如，在一些边疆地区和少数民族地区，设立了特殊的行政区划，以适应不同的治理需求。

3. **行政区划与军事关系的紧密联系**：唐代节度使制度的设立，使得行政区划与军事关系紧密相连。节度使不仅负责行政事务，还掌握大量军事权力，这在一定程度上影响了行政区划的稳定和统一。

#### 唐代行政区划的历史意义

唐代行政区划的变迁，不仅反映了当时国家治理的需求，也具有深远的历史意义。具体表现在以下几个方面：

1. **中央集权的巩固**：通过设立“道”和“府”，唐代中央政府对地方的控制更加直接和有效，加强了中央集权。

2. **行政区划制度的创新**：唐代在行政区划上进行了许多创新，如设立节度使制度，这种制度在当时是较为先进的，为后世行政区划的发展提供了借鉴。

3. **地方治理的规范**：唐代通过行政区划的调整，使得地方治理更加规范和高效。府、州、县三级制在当时具有较高的行政效率。

4. **对后世的影响**：唐代行政区划的经验对后世产生了重要影响。许多后世王朝在行政区划的设立和调整上，都借鉴了唐代的经验，如明清时期的省府州县四级制。

### 第四章 宋代行政区划的变迁

宋代是中国历史上一个重要的时期，其行政区划的变迁不仅反映了当时国家治理的需求，也体现了社会经济、政治和文化等多方面的变革。宋代行政区划的特点和变革，对中国历史产生了深远的影响。

#### 宋代行政区划的基本情况

1. **行政区划的层次**：宋代行政区划分为路、府、州、军、监五级。其中，路是最高一级的行政区划，府、州、军、监则在路的辖区内进行管理。

2. **路的设立**：北宋初年，全国设立十五路，后来随着地域的扩展和治理的需要，路数逐渐增加。

3. **府、州、军、监的职能**：府、州、军、监是地方行政机构，分别负责不同的职能。府一般设于省会，是最高行政单位；州是中等行政单位，负责州内的行政管理；军是边疆地区的特殊行政区划，主要负责军事事务；监则是对特定事务进行管理的机构。

#### 宋代行政区划的特点

1. **军事与行政结合**：宋代在行政区划中特别强调军事与行政的结合。府、州、军、监等行政区划不仅负责行政管理，还承担着军事防御的任务。

2. **行政区划的灵活性**：宋代在行政区划上具有较强的灵活性，可以根据实际情况进行调整。例如，在面对边疆战争或治理需要时，可以迅速设立或撤销某些行政区划。

3. **经济与行政结合**：宋代行政区划的调整与经济发展密切相关。在经济发展较为繁荣的地区，往往会设立更多的行政区划，以促进地方经济的发展和管理。

4. **行政区划的多样性**：宋代在行政区划上采用了多种形式，如府、州、军、监等，这种多样性反映了当时国家治理的灵活性和复杂性。

#### 宋代行政区划的变革

1. **路的增多**：随着北宋的统一和疆域的扩展，路的数量逐渐增多，全国一度达到几十路。

2. **行政区划的调整**：宋代中期以后，随着政治和经济的变化，行政区划进行了多次调整。例如，在一些边疆地区和经济发展较快的地区，设立了新的行政区划，以适应新的治理需求。

3. **军政分开**：为了解决军事与行政结合带来的问题，宋代后期开始推行军政分开的政策，逐步减少军区的设置，强化府、州、县的行政管理职能。

### 第七章 近现代中国行政区划的变迁

#### 民国时期的行政区划变迁

民国时期是中国历史上的一个动荡时期，行政区划也经历了多次重大变迁。1912年，中华民国成立后，北洋政府进行了一系列的行政区划调整，旨在实现国家统一和行政效率。主要变迁如下：

1. **中央与地方的权力分配**：民国初期，中央政府试图加强对地方的控制，通过设立“道”和“特别区”来管理省份，以实现中央集权。

2. **省制改革**：1927年，南京国民政府开始推行省制改革，废除道制，全国划分为若干省份，每个省份设省政府，实行省辖制。

3. **县制改革**：民国时期，县制也得到了重视，推行县自治，提高县级政府的行政能力和独立性。

#### 抗日战争时期的行政区划变迁

抗日战争时期，中国面临着严重的民族危机，行政区划也进行了相应的调整，以适应战争环境和国家治理需求。主要变迁如下：

1. **临时行政区划**：由于战乱，许多地区设立了临时行政区划，如行政督察区、战时区等，以便于军事治理和应急管理。

2. **特区设置**：在沦陷区，国民政府设立了若干特区，以维持对沦陷区的控制，同时也在抗日根据地设立了相应的行政区划。

3. **行政效率提升**：为了应对战争需求，国民政府加强了行政区划的调整，提高了行政效率，确保政令的迅速传达和执行。

2. **设立行署**：在新解放的地区，设立了行署，作为省级政府的派出机构，负责行政管理。

3. **省县制改革**：新中国成立后，全国实行省县制，全国划分为若干省，每个省下辖若干县。这一制度在1950年代初期得到全面推行。

新中国成立后，行政区划经历了多次重大调整，以适应国家发展的需要。主要变迁如下：

1. **直辖市和特别行政区设立**：1950年代，北京被设立为直辖市，成为国家的政治中心。1997年，香港回归后设立了香港特别行政区，1999年澳门回归后设立了澳门特别行政区。

2. **省级行政区划调整**：1954年，全国划分为二十四个省、五个自治区、三个直辖市。此后，随着国家的发展，省级行政区划多次调整，如增设自治区、直辖市，调整省的边界等。

3. **县级行政区划调整**：新中国成立后，县级行政区划也经历了多次调整。一方面，增设新县，扩大现有县的辖区；另一方面，撤销一些县，合并相邻县，以提高行政效率和治理能力。

近现代中国行政区划的变迁，不仅反映了政治局势的变动和社会经济的发展，也具有重要的历史意义：

3. **经济发展和社会稳定的保障**：行政区划的调整与经济发展、社会稳定密切相关。通过合理的行政区划设置，为国家的发展提供了制度保障。

2. **行政区划与经济发展的关系**：历史上的行政区划调整与经济发展密切相关。例如，唐代在边疆地区设立节度使制度，以推动边疆地区的开发；宋代通过行政区划的调整，促进了经济的发展和人口的流动。这一经验表明，现代行政区划改革应充分考虑经济发展的需求，优化资源配置，推动经济协调发展。

3. **行政区划与民族政策的结合**：元代和明清时期在行政区划上采取了多样化的策略，充分考虑了民族政策的需求。这种做法有助于促进民族融合和共同发展。现代中国作为一个多民族国家，行政区划改革也应充分考虑民族政策，加强对少数民族地区的支持和帮助。

4. **行政区划的灵活性与稳定性**：历史上的行政区划变迁既体现了灵活性，也保持了相对的稳定性。例如，宋代在行政区划上具有较强的灵活性，可以根据实际情况进行调整；明清时期的省府州县四级制则相对稳定，得到了长期沿用。现代行政区划改革应在保持稳定性的基础上，增强灵活性，以适应社会经济发展的变化。

#### 对现代行政区划改革的启示

基于上述历史行政区划变迁的经验，我们可以为现代行政区划改革提供以下几条具体启示：

1. **加强中央集权**：在当前国家治理现代化的背景下，应继续加强中央集权，确保政令的统一和行政效率。同时，应通过科学的行政区划设置，加强对地方的管理和控制。

2. **优化行政区划设置**：应充分考虑经济发展的需求，优化行政区划设置。例如，针对高新技术产业、生态保护区、自贸区等特殊经济区域，可以设立专门的行政机构，以优化资源配置和产业布局。

3. **注重民族政策**：在行政区划改革中，应充分考虑民族政策，加强对少数民族地区的支持和帮助，促进民族融合和共同发展。这不仅可以维护国家统一，也有利于社会稳定。

4. **增强行政区划的灵活性**：在保持行政区划稳定性的基础上，应增强其灵活性，以适应社会经济发展的变化。例如，在应对自然灾害、突发公共事件时，可以迅速设立或撤销特定的行政区划，提高应急管理的效率。

5. **推进城乡一体化**：随着城镇化进程的加快，应推进城乡一体化发展，设立更大的行政单位，以提高行政效率和公共服务水平。

6. **注重环境保护**：在行政区划改革中，应注重环境保护，通过科学设置行政区划，实现生态保护和区域发展的双赢。

总之，历史行政区划变迁的经验为现代行政区划改革提供了宝贵的借鉴。通过总结和运用这些经验，我们可以推动现代行政区划改革，实现国家治理体系和治理能力现代化，为国家的长期稳定和发展提供制度保障。

### 结论

《中国历代行政区划的变迁》一书通过系统梳理和深入分析中国历史上的行政区划演变，为我们提供了一幅详实而全面的行政区划变迁图。从秦代的郡县制到明清的省府州县四级制，作者周振鹤教授以其深厚的学术功底和独到的视角，揭示了行政区划变迁背后的政治逻辑和社会动因，展现了中国古代社会地理空间格局的形成和发展。本书不仅具有深厚的学术价值，对历史学者和公共管理研究者具有重要参考意义，同时也适合广大历史爱好者的阅读。

行政区划的变迁不仅是政治制度的变革，更是社会经济发展的重要体现。通过分析历史上的行政区划变迁，我们可以更好地理解中国的历史脉络和治理模式，为现代中国的行政区划改革提供宝贵的经验。在推进国家治理体系和治理能力现代化的过程中，如何借鉴历史上的行政区划变迁经验，实现行政区划的科学合理化，是一个值得深入探讨的课题。

《中国历代行政区划的变迁》一书，为我们打开了一扇理解中国历史的窗口，也为我们提供了许多有益的启示。在当前社会快速发展和变革的背景下，希望读者能够从这本书中获得启发，为中国的未来治理提供智慧和力量。
