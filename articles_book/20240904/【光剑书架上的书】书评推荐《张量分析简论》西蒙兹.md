#### **1. 张量的变换性质**

**定理 1.1** 设 \(T\) 为一个张量，其分量为 \(T_{ij}\)。当坐标系变换时，张量的分量将按照以下公式变换：

\[ T_{ij}^{'} = T_{kl} \frac{\partial x^{k}}{\partial x^{'}_{i} } \frac{\partial x^{l}}{\partial x^{'}_{j} } \]

**定理 1.2** 协变张量和反变张量的变换性质：

- 协变张量：\[ T^{i}_{j}^{'} = T^{k}_{l} \frac{\partial x^{k}}{\partial x^{'}_{i} } \frac{\partial x^{'}_{j}}{\partial x^{l} } \]
- 反变张量：\[ T_{ij}^{'} = T_{kl} \frac{\partial x^{'}_{i}}{\partial x^{k} } \frac{\partial x^{'}_{j}}{\partial x^{l} } \]

#### **2. 张量的加法和数乘**

**定理 2.1** 张量的加法和数乘运算满足以下规则：

- 加法：\[ (T + S)_{ij} = T_{ij} + S_{ij} \]
- 数乘：\[ (cT)_{ij} = c(T_{ij}) \]

#### **3. 张量的乘法**

**定理 3.1** 张量乘法的规则：

- 外积：\[ (T \otimes S)_{ijk\ell} = T_{ij} S_{k\ell} \]
- 内积：\[ (T \cdot S)_{ij} = T_{ik} S_{kj} \]

#### **4. 张量的导数**

**定理 4.1** 设 \(T\) 为一个张量，其分量为 \(T_{ij}\)。则张量的导数满足以下规则：

\[ \frac{\partial T_{ij}}{\partial x^{k}} = T_{ik} \frac{\partial x^{k}}{\partial x^{i}} \frac{\partial x^{j}}{\partial x^{k}} \]

#### **5. 张量的合同变换**

**定理 5.1** 设 \(T\) 为一个张量，其合同变换表示为：

\[ T_{ij}^{'} = A_{ik} A_{jl} T_{kl} \]

其中 \(A\) 是一个变换矩阵。合同变换保持张量的行列式不变。

#### **6. 张量的不变量**

**定理 6.1** 张量的不变量包括：

- 行列式：\[ \det(T) = T^{i}_{ij} T_{ji}^{i} \]
- 转置：\[ T_{ij}^{T} = T_{ji} \]
- 逆张量：\[ (T^{-1})_{ij} = \frac{1}{\det(T)} T_{ji} \]
