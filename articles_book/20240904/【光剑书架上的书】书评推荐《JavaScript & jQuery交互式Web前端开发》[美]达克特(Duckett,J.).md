**示例1：变量声明与赋值**
```javascript
// 声明变量并赋值
```

```javascript
greet("李四");
```

#### 2. jQuery基础示例

**示例1：选择器与DOM操作**
```javascript
$(document).ready(function() {
  // 选择并改变文本内容
  $("#title").text("新的标题");

  // 添加新元素
  $("#content").append("<p>这是一段新的内容。</p>");

  // 删除元素
  $("#old-element").remove();
});
```

**示例2：事件处理**
```javascript
$(document).ready(function() {
  // 给按钮添加点击事件
  $("#btn-click").click(function() {
    alert("按钮被点击了！");
  });
```
