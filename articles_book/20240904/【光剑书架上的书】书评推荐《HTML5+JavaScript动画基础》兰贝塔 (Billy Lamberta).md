#### 1. 动画基础

**1.1 动画原理**

动画的基础在于连续展示一系列静态图像，使得观众产生视觉上的连续运动感。这一原理被称为“视觉暂留效应”。《HTML5+JavaScript动画基础》的第一部分深入探讨了动画的基本概念和原理，包括帧率、位移、速度和加速度等。这些基础知识是理解和实现动画的核心。

**1.2 JavaScript基础**

在动画开发中，JavaScript起着至关重要的作用。本书详细介绍了JavaScript的基础知识，包括变量、函数、数组、对象等。通过这些基础，读者可以更好地理解动画的核心算法和实现方法。

**1.3 动画与Web开发**

HTML5和CSS3提供了丰富的动画功能，使得Web开发人员能够创造出各种形式的动画效果。本书详细介绍了如何使用HTML5和CSS3实现基础动画，包括使用CSS关键帧和JavaScript定时器等。通过这些技术，读者可以轻松实现简单的动画效果。

#### 2. 基础动画

**2.1 CSS动画**

CSS动画是Web开发中常用的动画技术之一。本书详细介绍了如何使用CSS关键帧实现动画，包括动画的创建、控制、过渡效果等。读者可以通过学习这些技术，创造出丰富的CSS动画效果。

**2.2 SVG动画**

SVG（可伸缩矢量图形）是一种基于XML的矢量图形格式，广泛应用于Web开发中。本书介绍了如何使用SVG实现动画，包括SVG动画的基本语法、动画属性等。通过这些技术，读者可以创造出动态的矢量图形效果。

**2.3 JavaScript动画**

JavaScript是动画开发的核心技术之一。本书详细介绍了如何使用JavaScript实现基础动画，包括使用JavaScript定时器、动画函数等。通过这些技术，读者可以创造出更加复杂的动画效果。

#### 3. 高级动画

**3.1 缓动与弹动**

缓动与弹动是高级动画中的重要技术。本书介绍了如何使用缓动与弹动技术，使得动画更加自然、流畅。读者可以通过学习这些技术，创造出更加逼真的动画效果。

**3.2 碰撞检测**

碰撞检测是动画开发中常见的技术。本书详细介绍了如何使用碰撞检测技术，检测动画对象之间的碰撞，从而实现更加真实的物理效果。

#### 4. 3D动画

- **CSS**：层叠样式表（Cascading Style Sheets）的缩写，用于控制Web页面的样式和布局。
- **SVG**：可伸缩矢量图形（Scalable Vector Graphics）的缩写，一种基于XML的矢量图形格式。
- **WebGL**：Web图形库（Web Graphics Library）的缩写，一种用于在Web上创建3D图形的技术。
- **Three.js**：一个基于WebGL的3D库，用于简化3D图形的开发。
- **粒子系统**：一种用于创建大量动态物体的技术，常用于模拟烟花、流体等效果。
- **正向运动学**：一种用于模拟物体运动的物理模型，基于速度、加速度等概念。
