**1.3 理解变量、函数和类**

在ActionScript 3.0中，变量、函数和类是编程的基础。下面是关于这些概念的基本介绍：

**变量（Variables）**

变量是用来存储数据的容器。在ActionScript 3.0中，变量可以分为以下几种类型：

- **局部变量（Local Variables）**：在函数内部声明的变量，作用域仅限于该函数。
- **全局变量（Global Variables）**：在函数外部声明的变量，作用域是整个应用程序。
- **类变量（Class Variables）**：在类中声明的变量，用于存储类的公共数据。

以下是一个局部变量和全局变量的例子：

```actionscript
// 全局变量
var appName:String = "Hello World Application";

// 函数内部
function displayAppName():void {
    var message:String = appName;
    trace(message);
}
```

**函数（Functions）**

函数是用于执行特定任务的代码块。在ActionScript 3.0中，函数可以分为以下几种类型：

- **内置函数（Built-in Functions）**：由ActionScript 3.0语言提供，如`trace()`和`Math.random()`。
- **自定义函数（Custom Functions）**：由开发者自定义，用于执行特定的任务。

以下是一个自定义函数的例子：

```actionscript
// 自定义函数，计算两个数的和
function addNumbers(a:Number, b:Number):Number {
    return a + b;
}

// 调用函数
var sum:Number = addNumbers(3, 4);
trace(sum); // 输出：7
```

**类（Classes）**

类是用于定义对象的结构和行为的蓝图。在ActionScript 3.0中，类可以是内置类（如`Object`和`Array`），也可以是自定义类。

以下是一个自定义类的例子：

```actionscript
// 自定义类，表示一个点
class Point {
    public var x:Number;
    public var y:Number;

    public function Point(_x:Number, _y:Number) {
        x = _x;
        y = _y;
    }

    // 成员函数，计算两点之间的距离
    public function distanceTo(otherPoint:Point):Number {
        var dx:Number = x - otherPoint.x;
        var dy:Number = y - otherPoint.y;
        return Math.sqrt(dx * dx + dy * dy);
    }
}

// 创建对象
var point1:Point = new Point(1, 2);
var point2:Point = new Point(4, 6);

// 计算两点之间的距离
var distance:Number = point1.distanceTo(point2);
trace(distance); // 输出：5
```

通过了解这些基础概念，你将为后续的学习打下坚实的基础。在《ActionScript 3.0 Cookbook》中，你将学习到更多关于变量、函数和类的实际应用和高级特性。

#### **二、数据类型和操作**

**2.1 字符串操作**

字符串是ActionScript 3.0中最常用的数据类型之一。在处理文本数据时，字符串操作变得尤为重要。以下是一些常见的字符串操作：

- **拼接（Concatenation）**：将两个或多个字符串连接在一起。

```actionscript
var str1:String = "Hello";
var str2:String = "World";
var result:String = str1 + " " + str2;
trace(result); // 输出：Hello World
```

- **提取子字符串（Substring）**：从字符串中提取一部分字符。

```actionscript
var str:String = "Hello World";
var substr:String = str.substring(6, 11);
trace(substr); // 输出：World
```

- **查找字符串（IndexOf）**：在字符串中查找特定子字符串的位置。

```actionscript
var str:String = "Hello World";
var index:int = str.indexOf("World");
trace(index); // 输出：6
```

- **替换字符串（Replace）**：将字符串中的一个子字符串替换为另一个子字符串。

```actionscript
var str:String = "Hello World";
var newStr:String = str.replace("World", "Universe");
trace(newStr); // 输出：Hello Universe
```

- **分割字符串（Split）**：将字符串分割成多个子字符串，根据指定的分隔符。

```actionscript
var str:String = "Hello,World,Universe";
var arr:Array = str.split(",");
trace(arr); // 输出：[Hello, World, Universe]
```

通过这些字符串操作，你可以轻松地处理文本数据，实现各种文本处理需求。

在ActionScript 3.0中，数值类型包括整数（Integer）和浮点数（Number）。以下是一些常见的数值类型操作：

- **乘法（Multiplication）**：将两个数值相乘。

#### **三、事件处理**

**3.1 理解事件流**

在ActionScript 3.0中，事件处理是应用程序的重要组成部分。事件流（Event Flow）描述了事件在应用程序中的传播和响应过程。事件流可以分为以下三种：

- **冒泡流（Bubble Flow）**：事件从触发点开始，依次向上传播到父元素，直到document对象。
- **捕获流（Capture Flow）**：事件从document对象开始，依次向下传播到触发点，直到目标元素。
- **捕获/冒泡流（Capture/Bubble Flow）**：事件同时经历捕获流和冒泡流。

以下是一个简单的示例，展示了事件流的传播过程：

```actionscript
// 创建一个简单的按钮
var button:Button = new Button();
button.label = "点击我";
button.x = 100;
button.y = 100;

// 添加按钮到舞台
stage.addChild(button);

// 添加事件监听器
button.addEventListener(MouseEvent.CLICK, onClick);

// 事件处理函数
function onClick(event:MouseEvent):void {
    trace("按钮被点击");
}
```

在这个示例中，当用户点击按钮时，事件将从按钮开始，依次向上传播到父元素（如容器），然后到达舞台，最后触发事件监听器。

**3.2 创建自定义事件**

在ActionScript 3.0中，可以使用`Event`类创建自定义事件。自定义事件可以用于应用程序中的各种场景，如通知其他对象数据更新、触发特定操作等。

以下是一个简单的自定义事件示例：

```actionscript
// 定义自定义事件
class MyEvent extends Event {
    public var data:Object;

    public function MyEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, data:Object = null) {
        super(type, bubbles, cancelable);
        this.data = data;
    }
}

// 创建事件监听器
var listener:Function = function(event:MyEvent):void {
    trace("事件监听器触发，数据为：" + event.data);
}

// 分发事件
var event:MyEvent = new MyEvent("myEvent", false, false, "Hello World");
dispatchEvent(event);
```

在这个示例中，我们定义了一个名为`MyEvent`的自定义事件，并在事件监听器中处理事件数据。

**3.3 处理鼠标事件**

在ActionScript 3.0中，鼠标事件是应用程序交互的重要组成部分。以下是一些常见的鼠标事件：

- **点击事件（MouseEvent.CLICK）**：当用户点击鼠标时触发。
- **双击事件（MouseEvent.DOUBLE_CLICK）**：当用户双击鼠标时触发。
- **鼠标按下事件（MouseEvent.MOUSE_DOWN）**：当用户按下鼠标按钮时触发。
- **鼠标释放事件（MouseEvent.MOUSE_UP）**：当用户释放鼠标按钮时触发。
- **鼠标移动事件（MouseEvent.MOUSE_MOVE）**：当用户移动鼠标时触发。

以下是一个简单的鼠标事件示例：

```actionscript
// 创建一个简单的按钮
var button:Button = new Button();
button.label = "点击我";
button.x = 100;
button.y = 100;

// 添加按钮到舞台
stage.addChild(button);

// 添加鼠标事件监听器
button.addEventListener(MouseEvent.CLICK, onClick);
button.addEventListener(MouseEvent.DOUBLE_CLICK, onDoubleClick);
```

**4.3 创建动画**

在ActionScript 3.0中，你可以使用多种方法创建动画，如补间动画、帧动画和事件驱动动画。以下是一些常见的动画技术：

- **补间动画（Tweening）**：使用补间动画可以轻松地创建平滑的动画效果。ActionScript 3.0提供了`Tween`类来实现补间动画。


- **帧动画（Frame-by-Frame Animation）**：帧动画是一种逐帧渲染的动画方法，适用于复杂的动画效果。

```actionscript
for (var i:uint = 1; i <= 10; i++) {
    container.graphics.clear();
    container.graphics.beginBitmapFill(bitmapData, new Matrix());
    container.graphics.drawRect(0, 0, bitmapData.width, bitmapData.height);
    container.graphics.endFill();
    stage.update();
    Thread.sleep(100);
}
```

- **事件驱动动画（Event-Driven Animation）**：事件驱动动画是一种基于事件的动画方法，适用于响应用户输入或其他事件触发的动画。

```actionscript
container.addEventListener(MouseEvent.CLICK, onClick);

function onClick(event:MouseEvent):void {
    container.x += 20;
}
```

以下是一个简单的补间动画示例：


**4.4 应用补间动画**

补间动画是ActionScript 3.0中一种强大的动画工具，它允许开发者创建平滑、流畅的动画效果。以下是一些关于补间动画的要点：

- **缓动函数（Easing Functions）**：缓动函数用于控制动画的速度变化，使动画更加自然。常见的缓动函数包括线性（Linear）、加速（EaseIn）、减速（EaseOut）和加速减速（EaseInOut）。

```actionscript
var myTween:Tween = new Tween(container, "x", Tween.EASE_IN, 100, 200, 2, true);
myTween.play();
```

- **缓动曲线（Easing Curves）**：缓动曲线是缓动函数的图形表示，用于控制动画的加速度。可以使用内置的缓动曲线，也可以自定义缓动曲线。

```actionscript
var easingFunction:Function = EasingFunctions.easeInOutQuad;
var myTween:Tween = new Tween(container, "x", easingFunction, 100, 200, 2, true);
myTween.play();
```

- **序列动画（Sequential Animation）**：序列动画是一种将多个补间动画组合在一起的方法，可以创建复杂的动画效果。

```actionscript
var myTween1:Tween = new Tween(container, "x", Tween.EASE_OUT, 100, 200, 2, true);
var myTween2:Tween = new Tween(container, "y", Tween.EASE_OUT, 100, 200, 2, true);
myTween1.play();
myTween2.play();
```

- **动画循环（Animation Loop）**：动画循环是一种连续播放动画的方法，适用于创建无限循环的动画。

```actionscript
var myLoop:Timeline = new Timeline([myTween1, myTween2]);
myLoop.repeatCount = Timeline.INFINITY;
myLoop.play();
```

通过掌握补间动画的这些要点，你可以创建出丰富多样的动画效果，提升应用程序的交互性和视觉体验。
