《皮凯蒂之后》的核心观点可以概括为以下几点：

2. **经济因素**：经济视角下的不平等研究集中在收入分配机制、财富积累和消费行为等方面。特别是技术进步和人力资本投资，对不平等现象具有显著的影响。

3. **社会结构**：社会视角下的不平等研究关注社会分层、社会流动性和族群差异等方面。社会结构的变化对不平等现象产生了深远的影响。

5. **全球化与制度**：全球化视角下的不平等研究关注全球化如何改变国际经济格局，影响收入分配。制度视角下的不平等研究则探讨不同制度对不平等现象的影响。

### **研究方法**

本书采用的研究方法多样，涵盖了定量分析、历史分析、比较研究和案例研究等。作者们通过分析大量数据、历史事件和国际经验，从不同角度对不平等现象进行了深入探讨。

2. **历史分析**：作者们通过回顾历史事件和社会变迁，分析了历史因素如何影响不平等现象的发展。

3. **比较研究**：书中对比了不同国家、地区和社会制度下的不平等现象，探讨了全球化对不平等的影响。

4. **案例研究**：作者们通过具体案例，如某地区或国家的收入分配改革，分析了政策实践对不平等现象的影响。

书中主要发现可以归纳为以下几点：

1. **不平等现象的普遍性**：不平等现象在全球范围内普遍存在，不同国家、地区和社会制度下都存在收入分配不均的问题。

3. **教育不平等的影响**：教育水平的不平等加剧了社会不平等。高技能劳动者因拥有更高的工资和职业稳定性，而低技能劳动者则面临更大的就业压力和收入差距。

4. **政策的重要性**：政府政策在缓解不平等现象中扮演着重要角色。通过税收政策、社会福利制度和监管政策，政府可以有效地调节收入分配，促进社会公平。

《皮凯蒂之后》为政策制定提供了重要的启示：

1. **教育改革**：提高教育质量，特别是职业教育和技能培训，有助于缩小教育不平等，提高低收入群体的就业能力和收入水平。

2. **税收政策调整**：通过增加对高收入者的税收负担，减少对低收入者的税收负担，可以实现收入再分配，缓解社会不平等。

3. **社会福利制度完善**：建立完善的社会福利制度，如医疗、养老和住房保障等，有助于减少贫困和社会不平等。

4. **监管政策加强**：通过加强市场监管，打击非法收入和财富积累，可以有效遏制不平等现象的进一步恶化。
