### 第二章：DOM操作与事件处理

DOM（Document Object Model）是JavaScript操作HTML文档的核心接口，它将HTML文档映射为一种树状结构，使得开发者可以通过JavaScript来访问和操作文档中的各种元素。事件处理则是JavaScript与用户交互的重要机制，它使得Web页面能够响应用户的操作，如点击、按键等。本章将详细介绍DOM操作和事件处理的相关内容。

#### 2.1 DOM概述

DOM是HTML和XML文档的编程接口，它定义了如何将文档结构表示为一种树形结构，使得开发者可以使用JavaScript来访问和操作文档中的各个部分。DOM树是由节点构成的，每个节点都代表文档中的一个部分，如元素、属性、文本等。

**2.1.1 DOM树的构成**

DOM树通常由以下几种节点组成：

- **文档节点（Document）**：代表整个文档。
- **元素节点（Element）**：代表HTML标签，如`<div>`、`<p>`等。
- **属性节点（Attribute）**：代表元素节点的属性，如`id`、`class`等。
- **文本节点（Text）**：代表元素中的文本内容。
- **注释节点（Comment）**：代表文档中的注释。

**2.1.2 获取DOM节点**

开发者可以通过多种方式获取DOM节点，以下是一些常用的方法：

- `getElementById()`：通过ID获取一个元素节点。
- `getElementsByClassName()`：通过类名获取多个元素节点。
- `getElementsByTagName()`：通过标签名获取多个元素节点。
- `querySelector()`：通过CSS选择器获取第一个匹配的元素节点。
- `querySelectorAll()`：通过CSS选择器获取所有匹配的元素节点。

```javascript
// 获取ID为"myDiv"的元素
var div = document.getElementById("myDiv");

// 获取类名为"myClass"的所有元素
var divs = document.getElementsByClassName("myClass");

// 获取标签名为"p"的所有元素
var paras = document.getElementsByTagName("p");

// 获取第一个匹配CSS选择器的元素
var element = document.querySelector(".myClass");

// 获取所有匹配CSS选择器的元素
var elements = document.querySelectorAll(".myClass");
```

#### 2.2 DOM操作

DOM操作是指通过JavaScript动态地创建、添加、删除和修改文档中的元素。以下是一些常见的DOM操作方法：

**2.2.1 创建节点**

- `createElement()`：创建一个新的元素节点。
- `createTextNode()`：创建一个新的文本节点。

```javascript
// 创建一个新div元素
var div = document.createElement("div");

// 创建一个新文本节点
var text = document.createTextNode("Hello, World!");
```

**2.2.2 添加节点**

- `appendChild()`：将新节点添加到子节点列表的末尾。
- `insertBefore()`：将新节点插入到指定子节点之前。
- `replaceChild()`：用新节点替换指定子节点。

```javascript
// 将新div元素添加到文档中
document.body.appendChild(div);

// 将新文本节点添加到div元素之前
div.insertBefore(text, div.firstChild);

// 用新文本节点替换div元素中的第一个子节点
div.replaceChild(text, div.firstChild);
```

**2.2.3 删除节点**

- `removeChild()`：从子节点列表中删除指定子节点。

```javascript
// 删除div元素
var removed = document.body.removeChild(div);
```

**2.2.4 修改节点**

- `setAttribute()`：设置元素的属性值。
- `innerText`：设置或获取元素内部的文本内容。
- `innerHTML`：设置或获取元素内部的HTML内容。

```javascript
// 设置元素的属性值
div.setAttribute("id", "newId");

// 设置元素内部的文本内容
div.innerText = "New content for div";

// 设置元素内部的HTML内容
div.innerHTML = "<p>New paragraph content</p>";
```

#### 2.3 事件处理

事件处理是JavaScript中与用户交互的关键机制。它使得Web页面能够响应用户的操作，如点击、键盘输入等。以下是一些常见的事件处理方法：

**2.3.1 事件模型**

JavaScript的事件模型基于事件流，包括三个阶段：捕获阶段、目标阶段和冒泡阶段。

- **捕获阶段**：从根节点开始，向下传递事件。
- **目标阶段**：事件到达目标元素。
- **冒泡阶段**：事件从目标元素向上传递，直到根节点。

**2.3.2 事件处理函数**

事件处理函数是当特定事件触发时执行的函数。可以使用`addEventListener()`方法为元素绑定事件处理函数。

```javascript
// 绑定点击事件处理函数
document.getElementById("myButton").addEventListener("click", function() {
  console.log("Button clicked!");
});
```

**2.3.3 事件委托**

事件委托是一种优化事件处理的方法，它通过为父元素绑定事件处理函数来处理子元素的点击事件，从而减少事件处理函数的数量，提高性能。

```javascript
// 使用事件委托处理子元素的点击事件
var ul = document.getElementById("myList");
ul.addEventListener("click", function(event) {
  if (event.target.tagName === "LI") {
    console.log("List item clicked:", event.target.textContent);
  }
});
```

#### 2.4 实例演示

以下是一个简单的DOM操作和事件处理实例，演示如何创建一个新的列表项并动态添加到列表中，并为其绑定点击事件处理函数。

```html
<!DOCTYPE html>
<html>
<head>

      li.textContent = "New list item";
      ul.appendChild(li);
    }

    // 绑定点击事件处理函数
    addButton.addEventListener("click", addListItem);
  </script>
</body>
</html>
```

### 第三章：JavaScript API

JavaScript API是开发者用于扩展JavaScript功能的重要工具。这些API不仅包括由浏览器提供的客户端API，还包括由Node.js提供的服务器端API。本章将详细介绍这些API的概述、使用方法和实际应用。

#### 3.1 BOM（Browser Object Model）

BOM（Browser Object Model）是浏览器对象模型，它提供了一系列可以访问和操作浏览器的API。BOM主要由以下几个部分组成：

**3.1.1 窗口（Window）**

窗口是BOM的核心部分，它代表浏览器窗口。以下是一些常用的窗口API：

- `window.opener`：返回打开当前窗口的原始窗口。
- `window.open()`：打开一个新的浏览器窗口或标签页。
- `window.close()`：关闭当前窗口。

```javascript
// 打开一个新的浏览器窗口
var newWindow = window.open("http://www.example.com", "_blank");

// 关闭当前窗口
window.close();
```

**3.1.2 位置（Location）**

位置对象（Location）用于获取和设置浏览器的URL。以下是一些常用的位置API：

- `location.href`：获取或设置完整的URL。
- `location.hostname`：获取或设置主机名。
- `location.pathname`：获取或设置路径名。
- `location.search`：获取或设置查询字符串。

```javascript
// 获取当前URL
var currentURL = location.href;

// 设置新的URL
location.href = "http://www.example.com/page.html";
```

**3.1.3 历史记录（History）**

历史记录对象（History）用于操作浏览器的历史记录。以下是一些常用的历史记录API：

- `history.back()`：返回上一页。
- `history.forward()`：前进到下一页。
- `history.go()`：前进或返回指定的页面。

```javascript
// 返回上一页
history.back();

// 前进到下一页
history.forward();
```

#### 3.2 AJAX

AJAX（Asynchronous JavaScript and XML）是一种通过JavaScript异步请求数据的技巧，它使得Web应用可以无需刷新页面就更新内容。AJAX的基础原理是通过XMLHttpRequest对象发送HTTP请求，并处理响应。

**3.2.1 AJAX基础**

以下是一个简单的AJAX示例，演示如何使用XMLHttpRequest对象发送GET请求：

```javascript

// 设置请求完成后的回调函数
xhr.onload = function() {
  if (xhr.status >= 200 && xhr.status < 300) {
    // 请求成功，处理响应数据
    var data = JSON.parse(xhr.responseText);
    console.log(data);
  } else {
    // 请求失败，处理错误
    console.error("请求失败：", xhr.status);
  }
};

// 发送HTTP请求
xhr.send();
```

#### 4.1 闭包与作用域

闭包是JavaScript中的一个重要特性，它允许函数访问并保持其定义时的作用域。闭包由两部分组成：函数和创建该函数的环境。环境通常包括外部函数的作用域链。

**4.1.1 闭包的基本概念**

闭包是函数及其作用域的组合。当一个函数在其声明作用域之外被调用时，它仍然能够访问外部函数的变量。这种现象就是闭包。

```javascript
function outer() {
  var outerVar = "I am from outer function";
  function inner() {
    var innerVar = "I am from inner function";
    return outerVar + " and " + innerVar;
  }
  return inner;
}

var closure = outer();
console.log(closure()); // 输出 "I am from outer function and I am from inner function"
```

在上面的示例中，`inner`函数是一个闭包，它能够访问`outer`函数中的`outerVar`变量。

**4.1.2 闭包的应用**

闭包在许多场景中有用，如模块化编程、回调函数、事件处理等。以下是一个模块化编程的示例：

```javascript
function createCounter() {
  var count = 0;
  return function() {
    return count++;
  };
}

var counter = createCounter();
console.log(counter()); // 输出 1
console.log(counter()); // 输出 2
```

在这个示例中，`createCounter`函数返回一个闭包，它能够保持`count`变量的状态。

#### 4.2 函数式编程

函数式编程是一种编程范式，它强调使用不可变数据和纯函数来解决问题。纯函数是没有副作用、输入和输出固定的函数。

**4.2.1 纯函数**

纯函数的特点是：

- 输入和输出明确，无副作用。
- 函数执行的结果只依赖于输入，与外部状态无关。

```javascript
function add(a, b) {
  return a + b;
}

console.log(add(2, 3)); // 输出 5
console.log(add(2, 3)); // 输出 5
```

在上面的示例中，`add`函数是一个纯函数，每次调用时都会返回相同的输出。

**4.2.2 函数组合**

函数组合是一种将多个函数组合成一个新的函数的方法。这有助于提高代码的可读性和可维护性。

```javascript
function compose(f, g) {
  return function(x) {
    return f(g(x));
  };
}

function double(x) {
  return x * 2;
}

function addFive(x) {
  return x + 5;
}

var doubleAndAddFive = compose(double, addFive);
console.log(doubleAndAddFive(10)); // 输出 25
```

在这个示例中，`doubleAndAddFive`函数是`double`和`addFive`函数的组合。

**4.2.3 函数式编程工具**

JavaScript提供了许多函数式编程工具，如`map`、`filter`、`reduce`等。这些工具能够简化数据处理和操作。

```javascript
var numbers = [1, 2, 3, 4, 5];

var doubled = numbers.map(function(x) {
  return x * 2;
});

var even = numbers.filter(function(x) {
  return x % 2 === 0;
});

var sum = numbers.reduce(function(acc, x) {
  return acc + x;
}, 0);

console.log(doubled); // 输出 [2, 4, 6, 8, 10]
console.log(even); // 输出 [2, 4]
console.log(sum); // 输出 15
```

#### 4.3 模块与包

模块化编程是现代JavaScript开发的基础。它有助于将代码拆分成可重用的模块，从而提高代码的可读性和可维护性。

**4.3.1 模块的基本概念**

模块是一种封装代码的方法，它将功能相关的代码组合在一起，并提供一个公共接口。

```javascript
// math.js
export function add(a, b) {
  return a + b;
}

export function subtract(a, b) {
  return a - b;
}

// main.js
import { add, subtract } from "./math.js";

console.log(add(5, 3)); // 输出 8
console.log(subtract(5, 3)); // 输出 2
```

**4.3.2 ES6 Modules**

ES6（ECMAScript 2015）引入了模块化的标准，使得模块化编程变得更加简单和强大。

- **导入（import）**：使用`import`语句导入模块。
- **导出（export）**：使用`export`语句导出模块。


**4.4.3 解构赋值**


**4.4.4 Promise**

Promise是一种用于异步编程的新语法，它使得处理异步操作更加简洁和可靠。

```javascript
function fetchData() {
  return new Promise(function(resolve, reject) {
    // 异步操作

      resolve("Data fetched");
    }, 1000);

  console.log(data); // 输出 "Data fetched"
```
