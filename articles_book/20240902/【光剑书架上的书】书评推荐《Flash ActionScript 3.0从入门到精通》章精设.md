                 

# 【光剑书架上的书】《Flash ActionScript 3.0从入门到精通》章精设 书评推荐语

## 引言

《Flash ActionScript 3.0从入门到精通》是由资深读书人章精设编写的一本全面讲解Flash ActionScript 3.0编程技巧的著作。作为一本面向初学者和中级开发人员的教材，本书系统而深入地讲解了ActionScript 3.0的核心概念和高级应用，从基础语法到复杂的数据类型、从面向对象编程到事件处理，再到服务器端通信，几乎涵盖了Flash ActionScript 3.0的方方面面。

本书的内容安排合理，结构清晰，讲解深入浅出，既有详细的理论阐述，又有大量的实例分析，真正做到了理论与实践相结合。无论是想要深入学习ActionScript 3.0的编程爱好者，还是需要掌握这门技术以应对实际开发工作的专业人士，都能从本书中获得宝贵的知识和经验。

## 详细书评

### 第1章 了解ActionScript3.0

**1.1 ActionScript的历史**

在进入编程的世界之前，首先需要对ActionScript的历史有一个基本的了解。ActionScript 3.0的出现，不仅是Flash技术的重大更新，也是编程语言发展史上的一个里程碑。从ActionScript 1.0到ActionScript 2.0，再到ActionScript 3.0，每一次的升级都带来了显著的性能提升和功能扩展。

**1.2 认识舞台和主时间轴**

舞台（Stage）和主时间轴（Main Timeline）是Flash编程中至关重要的概念。它们分别代表了显示区域和时间线，对于理解Flash动画的基本结构和运行机制具有重要意义。本书通过详细的图解和实例，帮助读者快速掌握这两个概念。

**1.3 了解变量**

变量是编程中的基础，掌握变量的定义、赋值和输出是进入编程世界的第一步。本书详细讲解了各种变量类型、变量命名规则以及变量的作用域和生命周期，让读者对变量有全面的认识。

**1.4 通过属性面板定义变量**

属性面板是Flash编程中常用的工具之一，通过属性面板可以方便地定义和修改变量。本书介绍了如何通过属性面板控制影片剪辑实例和动态文本，并讲解了变量的执行顺序和注意事项。

**1.5 了解动作面板**

动作面板是编写ActionScript代码的主要工具，掌握动作面板的使用对于编写高效的代码至关重要。本书详细讲解了动作面板的组成、代码提示和代码折叠功能，以及如何添加单行注释、多行注释和文档注释。

**1.6 代码注释**

代码注释是编写可维护代码的重要组成部分。本书介绍了单行注释、多行注释和文档注释的使用方法，并讲解了如何编写高质量的代码注释，以提高代码的可读性和可维护性。

**1.7 代码位置**

代码位置的选择直接影响到程序的执行效率和可维护性。本书讲解了外部文件代码和类代码的编写方法，并介绍了如何优化变量定义和代码结构，以提高程序的性能。

### 第2章 简单数据类型

**2.1 数据类型**

数据类型是编程中最基础的知识点之一。本书详细介绍了ActionScript 3.0中的简单数据类型，包括int类型、uint类型、Number类型、Boolean类型和String类型。通过丰富的实例，让读者深入了解这些数据类型的使用方法和特点。

**2.2 int类型和int类**

int类型是ActionScript中最常用的数据类型之一，用于表示整数。本书讲解了int类型的表示方法、取整数的数位以及使用toString方法将整数转换为字符串。

**2.3 uint类型与uint类**

uint类型与int类型类似，用于表示无符号整数。本书详细介绍了RGB颜色和键控代码，并讲解了如何使用uint类型来表示颜色和键值。

**2.4 Number类型与Number类**

Number类型用于表示浮点数，包括整数和分数。本书讲解了浮点数的表示方法、浮点运算中的注意事项以及如何使用Number类进行数值计算。

**2.5 Boolean类型与Boolean类**

Boolean类型用于表示逻辑值，即真或假。本书介绍了如何定义和操作Boolean变量，并讲解了Boolean类的用法。

**2.6 String类型与String类**

String类型用于表示字符串，是编程中常用的数据类型。本书详细介绍了字符串的表示方法、字符串的索引、转义符以及字符串的操作方法。

**2.7 变量名的命名规则**

变量名的命名规则是编写清晰、高效代码的基础。本书讲解了变量名是标识符、同一个变量名不能重复定义、不能用MovieClip类的属性名和方法名、不能使用关键字等命名规则，并介绍了如何编写规范的变量名。

**2.8 变量的更多知识**

变量是编程中最常用的元素之一，本书介绍了变量的默认值、变量定义与代码优化、“*”类型与自动类型转换以及类型转换函数等知识点，帮助读者更深入地理解变量的用法。

### 第3章 复杂数据类型
