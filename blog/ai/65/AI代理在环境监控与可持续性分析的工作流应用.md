### 4.2 公式推导过程

以下以线性回归模型为例，介绍公式推导过程。

假设我们有 $n$ 个样本 $(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)$，其中 $x_i$ 是自变量，$y_i$ 是因变量。线性回归模型的目标是找到最佳拟合线：

$$
y = \beta_0 + \beta_1 x + \epsilon
$$

其中 $\beta_0$ 是截距，$\beta_1$ 是斜率，$\epsilon$ 是误差项。

为了找到最佳拟合线，我们需要最小化误差平方和：

$$
\min_{\beta_0, \beta_1} \sum_{i=1}^n (y_i - \beta_0 - \beta_1 x_i)^2
$$

对上式进行求导，得到：

$$
\frac{\partial}{\partial \beta_0} \sum_{i=1}^n (y_i - \beta_0 - \beta_1 x_i)^2 = 0
$$

$$
\frac{\partial}{\partial \beta_1} \sum_{i=1}^n (y_i - \beta_0 - \beta_1 x_i)^2 = 0
$$

解上述方程组，得到最佳拟合线的参数 $\beta_0$ 和 $\beta_1$。

### 5.2 源代码详细实现

以下是一个使用PyTorch进行空气质量监测的简单示例：

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader, TensorDataset

# 数据准备
data = {
    'PM2.5': [50, 60, 70, 80, 90],
    'PM10': [80, 90, 95, 100, 105],
    '温度': [25, 26, 27, 28, 29],
    '相对湿度': [60, 61, 62, 63, 64],
    'AQI': [100, 150, 200, 250, 300]
}

# 转换为Tensor
PM2_5 = torch.tensor(data['PM2.5'], dtype=torch.float32)
PM10 = torch.tensor(data['PM10'], dtype=torch.float32)
温度 = torch.tensor(data['温度'], dtype=torch.float32)
相对湿度 = torch.tensor(data['相对湿度'], dtype=torch.float32)
AQI = torch.tensor(data['AQI'], dtype=torch.float32)

# 创建数据集和数据加载器
train_dataset = TensorDataset(PM2_5, PM10, 温度, 相对湿度, AQI)
train_loader = DataLoader(train_dataset, batch_size=2, shuffle=True)

# 模型定义
class AQIModel(nn.Module):
    def __init__(self):
        super(AQIModel, self).__init__()
        self.fc1 = nn.Linear(4, 10)
        self.fc2 = nn.Linear(10, 1)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

model = AQIModel().to(device)

# 损失函数和优化器
criterion = nn.MSELoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 训练模型
for epoch in range(100):
    for data in train_loader:
        input_data = data[:4]
        labels = data[4]
        outputs = model(input_data)
        loss = criterion(outputs, labels)
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
    print(f"Epoch {epoch+1}, loss: {loss.item()}")

# 测试模型
test_data = torch.tensor([[90, 105, 29, 64]], dtype=torch.float32)
test_output = model(test_data)
print(f"预测的AQI: {test_output.item()}")

```
