## 4. 数学模型和公式详细讲解举例说明
### 4.1 职位相关性计算模型
#### 4.1.1 基于VSM的职位相似度计算
$$\text{sim}(p_1,p_2) = \frac{\sum_{i=1}^n w_{1i}w_{2i}}{\sqrt{\sum_{i=1}^n w_{1i}^2} \sqrt{\sum_{i=1}^n w_{2i}^2}}$$
其中$p_1$和$p_2$是两个职位，$w_{1i}$和$w_{2i}$分别是第$i$个特征词在两个职位中的权重。

#### 4.1.2 基于主题模型LDA的职位相关性计算
LDA通过以下生成过程来描述文档的主题分布和主题的词分布：
$$P(w_i|d) = \sum_{j=1}^k P(w_i|z_j)P(z_j|d)$$
其中$w_i$为第$i$个词，$z_j$为第$j$个主题，$P(z_j|d)$表示文档$d$的主题分布，$P(w_i|z_j)$表示主题$z_j$的词分布。

### 4.2 简历排序模型
#### 4.2.1 基于多因素加权的简历排序
$$\text{score}(r) = \sum_{i=1}^n \alpha_i f_i(r)$$
其中$r$为一份简历，$f_i$为第$i$个评估因素，$\alpha_i$为相应的权重系数。

#### 4.2.2 基于学习排序LTR的简历排序
以Pointwise方法为例，假设$y$为简历相关性标签，$x$为简历特征向量，排序模型为：
$$\hat{y} = F(x) = \sum_{i=1}^n w_i x_i$$
通过最小化损失函数$\text{Loss}(y, \hat{y})$来求解参数$w$。

### 5.3 后端服务实现
#### 5.3.1 用户注册
```java
@PostMapping("/register")
public Result register(@RequestBody RegisterRequest registerRequest) {
    // 参数校验
    // 用户名、密码非空判断
    // 邮箱格式校验

    // 用户名是否已存在
    if (userService.existsByUsername(registerRequest.getUsername())) {
        return Result.fail("用户名已存在");
    }

    // 密码加密
    String encodePassword = DigestUtils.md5DigestAsHex(registerRequest.getPassword().getBytes());

    // 构造 User 对象
    User user = new User();
    user.setUsername(registerRequest.getUsername());
    user.setPassword(encodePassword);
    user.setEmail(registerRequest.getEmail());
    user.setRole(registerRequest.getRole());
    user.setCreateTime(new Date());

    // 保存用户
    userService.save(user);

    return Result.ok();
}
```

#### 5.3.2 职位检索
```java
@GetMapping("/search")
public Result<PageResult<JobVO>> search(@RequestParam(required = false) String keyword,
                                         @RequestParam(defaultValue = "1") Integer pageNum,
                                         @RequestParam(defaultValue = "10") Integer pageSize) {
    // 构造查询条件
    QueryWrapper<Job> queryWrapper = new QueryWrapper<>();
    queryWrapper.like(StringUtils.isNotBlank(keyword), "title", keyword)
            .or().like(StringUtils.isNotBlank(keyword), "description", keyword)
            .orderByDesc("create_time");

    // 分页查询
    Page<Job> page = new Page<>(pageNum, pageSize);
    Page<Job> jobPage = jobService.page(page, queryWrapper);

    // 结果转换
    List<JobVO> jobVOList = jobPage.getRecords().stream().map(this::convert).collect(Collectors.toList());

    // 封装分页结果
    PageResult<JobVO> pageResult = new PageResult<>(jobVOList, jobPage.getTotal(), jobPage.getSize(), jobPage.getCurrent());

    return Result.ok(pageResult);
}
```
