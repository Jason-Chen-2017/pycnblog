## 2. 核心概念与联系

### 数论基础

- **整数**：正整数、负整数和零的集合，通常表示为$\mathbb{Z}$。
- **质数**：只能被1和自身整除的整数，通常表示为$\mathbb{P}$。
- **同余类**：形如$a \equiv b \ (\text{mod } m)$的等式表示a和b在模m的意义下相等。

### 关联概念

寻找满足特定性质的整数对（x，y）通常涉及以下概念：

- **模运算**：用于定义同余类的基本运算，形式为$a \equiv b \ (\text{mod } m)$，表示a和b在模m的意义下相等。
- **同余方程**：形如ax ≡ b (mod m)的方程，其中x是未知数。
- **最小非负整数类**：在模m的意义下，最小非负整数类是$\overline{0}, \overline{1}, \overline{2}, ..., \overline{m-1}$。

### 5.2 源代码详细实现

```python
import math

def find_integer_pairs(n):
    integer_pairs = []
    for x in range(-int(math.sqrt(n)), int(math.sqrt(n)) + 1):
        y_squared = n - x**2
        y = math.sqrt(y_squared)
        if y.is_integer():
            integer_pairs.append((x, int(y)))
    return integer_pairs

# 示例：查找满足x^2 + y^2 = 100的所有整数对
n = 100
pairs = find_integer_pairs(n)
print(f"Integer pairs that satisfy x^2 + y^2 = {n}: {pairs}")
```

这段代码实现了寻找满足$x^2 + y^2 = n$的所有整数对的功能：

- **函数定义**：`find_integer_pairs(n)`接收一个整数$n$作为参数。
- **循环遍历**：通过循环遍历$x$从$-\sqrt{n}$到$\sqrt{n}$。
- **计算y**：通过计算$y = \sqrt{n - x^2}$来尝试找到满足条件的$y$。
- **检查整数**：使用`math.sqrt()`和`.is_integer()`方法检查$y$是否为整数。
- **结果收集**：如果$y$是整数，则将（x，y）添加到结果列表中。
