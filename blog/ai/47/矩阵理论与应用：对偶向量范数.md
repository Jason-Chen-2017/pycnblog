## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

对偶向量范数的数学模型可以表示为：

$$\|v\|_D = \sqrt{\sum_{i=1}^n a_i^2}$$

其中，$a_i = \langle v, v_i \rangle$是向量$v$与基向量$v_i$的内积。

### 4.2 公式推导过程

假设我们有一组标准正交基$\{v_1, v_2, \dots, v_n\}$，向量$v$可以表示为：

$$v = a_1v_1 + a_2v_2 + \dots + a_nv_n$$

根据内积的定义，我们有：

$$a_i = \langle v, v_i \rangle$$

对$a_i$进行平方，得到：

$$a_i^2 = \langle v, v_i \rangle \langle v, v_i \rangle$$

由于基向量是标准正交的，因此$\langle v_i, v_j \rangle = 0$（对于$i \
eq j$），从而：

$$a_i^2 = \langle v, v_i \rangle^2$$

将$a_i^2$代入对偶向量范数的定义，得到：

$$\|v\|_D = \sqrt{\sum_{i=1}^n a_i^2} = \sqrt{\sum_{i=1}^n \langle v, v_i \rangle^2}$$

### 4.3 案例分析与讲解

假设我们有一组标准正交基$\{v_1, v_2, \dots, v_3\}$，向量$v = [1, 2, 3]^T$。

首先，计算向量$v$与基向量的内积：

$$a_1 = \langle v, v_1 \rangle = \langle [1, 2, 3]^T, [1, 0, 0]^T \rangle = 1$$
$$a_2 = \langle v, v_2 \rangle = \langle [1, 2, 3]^T, [0, 1, 0]^T \rangle = 2$$
$$a_3 = \langle v, v_3 \rangle = \langle [1, 2, 3]^T, [0, 0, 1]^T \rangle = 3$$

然后，计算对偶向量范数：

$$\|v\|_D = \sqrt{a_1^2 + a_2^2 + a_3^2} = \sqrt{1^2 + 2^2 + 3^2} = \sqrt{14} \approx 3.74$$

def dual_norm(v, v_base):
    """
    计算向量v的对偶向量范数。

    参数:
    v -- 输入向量
