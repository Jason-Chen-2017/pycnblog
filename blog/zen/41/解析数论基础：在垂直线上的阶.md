### 2.3 案例分析与讲解

以$a=3$为例，我们考察$f(x) = 3^x$在$x \in \mathbb{Z}$上的行为。观察到$f(1)=3$，$f(2)=9$，$f(3)=27$，$f(4)=81$，等等。不难发现，随着$x$的增加，$f(x)$的值呈现出指数增长的趋势。若考虑模运算，例如求$f(x) \mod n$，则可以观察到周期性模式，即在一定范围内重复出现相同的模值。

    result = 1

    while exponent > 0:

        exponent = exponent >> 1
        base = (base * base) % modulus
    return result
