```python
import mermaid

# 定义知识点
knowledge_units = [
    {"name": "线性回归", "weight": 0.3, "quality": 0.8},
    {"name": "神经网络", "weight": 0.5, "quality": 0.9},
```

3. **渲染知识结构图**：调用 `render` 方法将知识结构图转换为 Mermaid 格式，并打印出来。
