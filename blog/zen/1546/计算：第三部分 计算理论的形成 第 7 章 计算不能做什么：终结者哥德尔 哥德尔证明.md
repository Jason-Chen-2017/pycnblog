## 2. 核心概念与联系

### 2.1 哥德尔不可计算定理

哥德尔不可计算定理可以分为两部分：第一部分是“不完全性定理”，第二部分是“不可判定性定理”。

**不完全性定理**指出，对于任何足够强的形式数学系统，都存在一些命题，既不能被证明也不能被证伪。这意味着，形式系统无法证明所有数学命题。

**不可判定性定理**进一步指出，存在一些命题，不仅不能被证明或证伪，甚至无法确定它们是否可证或可证伪。这意味着，形式系统无法解决所有数学问题。

### 2.2 哥德尔证明的原理

哥德尔证明的核心在于构造两个特殊的命题：C 和 G。

**C 命题**（Consistency）：系统 S 是一致的。

**G 命题**（Undecidable）：系统 S 无法证明 C。

哥德尔通过数学方法，将这两个命题转化为形式系统的内部命题，并证明了它们是相互独立的。这就是说，系统 S 无法同时证明 C 和 G，也无法证明它们的否定。这意味着，系统 S 无法解决所有数学问题。

### 2.3 哥德尔证明的步骤

哥德尔的证明分为以下几个步骤：

1. **构造命题 C 和 G**：将 C 和 G 转化为形式系统的内部命题。
2. **证明 C 和 G 的独立性**：证明系统 S 无法同时证明 C 和 G，也无法证明它们的否定。
3. **推导不完全性定理和不可判定性定理**：利用 C 和 G 的独立性，推导出形式系统 S 的不完全性和不可判定性。
