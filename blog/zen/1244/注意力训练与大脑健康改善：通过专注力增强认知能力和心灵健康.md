$$\Delta L = f(L, I)$$

其中，$\Delta L$ 表示神经元连接强度的变化，$L$ 表示神经元连接的初始强度，$I$ 表示外部输入的强度。

### 4.2 公式推导过程

$$f_i(x) = \frac{1}{1 + e^{-x}}$$

当神经元 $i$ 的输入 $x$ 超过阈值 $T$ 时，神经元会被激活。我们可以将阈值 $T$ 定义为：

$$T = \theta \cdot \sum_{j} w_{ij}$$

其中，$\theta$ 表示神经元的阈值参数，$w_{ij}$ 表示神经元 $i$ 与神经元 $j$ 之间的连接权重。

### 4.3 案例分析与讲解

假设我们有一个神经网络，其中包含三个神经元 $i$、$j$ 和 $k$。初始时，这三个神经元之间的连接权重分别为 $w_{ij} = 0.5$、$w_{ji} = 0.3$ 和 $w_{ki} = 0.4$。外部输入的强度为 $I = 0.8$。

$$\Delta L_{ij} = f(0.5, 0.8) = \frac{1}{1 + e^{-(0.5 \cdot 0.8)}} = 0.632$$

$$\Delta L_{ji} = f(0.3, 0.8) = \frac{1}{1 + e^{-(0.3 \cdot 0.8)}} = 0.556$$

$$\Delta L_{ki} = f(0.4, 0.8) = \frac{1}{1 + e^{-(0.4 \cdot 0.8)}} = 0.574$$

根据计算结果，我们可以得出神经元 $i$、$j$ 和 $k$ 的连接强度分别为：

$$L_{ij} = 0.5 + 0.632 = 1.132$$

$$L_{ji} = 0.3 + 0.556 = 0.856$$

$$L_{ki} = 0.4 + 0.574 = 0.974$$
