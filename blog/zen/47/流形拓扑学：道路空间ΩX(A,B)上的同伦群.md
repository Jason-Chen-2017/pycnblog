## 2. 核心概念与联系

### 2.1 道路空间ΩX(A,B)

道路空间ΩX(A,B)是由所有从区间[0,1]到X的连续映射构成的集合，其中A和B是X中的两个不同点。它是一个拓扑空间，其拓扑结构由以下性质确定：

- 闭集：包含0和1的闭集是ΩX(A,B)的闭集。
- 交集：任意两个ΩX(A,B)的闭集的交集仍然是ΩX(A,B)的闭集。
- 并集：任意有限个ΩX(A,B)的闭集的并集仍然是ΩX(A,B)的闭集。

### 2.2 同伦群

同伦群是同伦论中的一个基本概念，它描述了拓扑空间之间的连续变形下的等价关系。对于一个拓扑空间X，其同伦群定义为：

$$\pi_n(X) = \{ \alpha \mid \alpha: S^n \rightarrow X \text{ 是连续映射} \}$$

其中，S^n是n维球面。对于两个映射α和β，如果存在一个连续映射F: [0,1] × S^n → X，使得F(0,x) = α(x)，F(1,x) = β(x)，并且F(t,x)是α和β之间的同伦，则称α与β同伦。

### 2.3 道路空间ΩX(A,B)上的同伦群

道路空间ΩX(A,B)上的同伦群定义为：

$$\pi_n(ΩX(A,B)) = \{ \alpha \mid \alpha: S^n \rightarrow ΩX(A,B) \text{ 是连续映射} \}$$
