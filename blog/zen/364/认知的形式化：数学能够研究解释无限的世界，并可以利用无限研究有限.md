以下是使用Sympy构建符号逻辑模型的Python代码：

```python
from sympy import symbols, Eq, solve, And

# 定义符号
P1, P2, M1, M2 = symbols('P1 P2 M1 M2')

# 定义符号间的关系
rules = [
    Eq(P1, M1),
    Eq(P2, M2),
    Eq(M1, P1),
    Eq(M2, P2),
    Eq(M1, M2),
    Eq(M2, M1)
]

# 定义推理规则
inference_rules = [
    Eq(P1, M1),
    Eq(P2, M2),
    Eq(M1, P1),
    Eq(M2, P2),
    Eq(M1, M2),
    Eq(M2, M1)
]

# 构建符号逻辑模型
model = And(*rules)

# 验证推理过程
result = solve(model, (P1, P2, M1, M2))

# 输出验证结果
print(result)
```
