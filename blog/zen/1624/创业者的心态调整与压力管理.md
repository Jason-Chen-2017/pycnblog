### 5.2 源代码详细实现

以下是实现心态调整与压力管理的Python代码示例：

```python
import random

# 认知重构函数
def cognitive_reconstruction(depression_degree):
    # 随机选择积极认知
    positive_cognitions = ["目标明确", "充满信心", "乐观面对"]
    depression_degree -= random.choice(positive_cognitions)
    return depression_degree

# 目标设定函数
def goal_setting(depression_degree, goal_num):
    # 实现目标
    achieved_goals = goal_num
    for i in range(goal_num):
        depression_degree -= random.choice(["实现目标", "提高能力", "获得成就感"])
    return depression_degree, achieved_goals

# 压力管理函数
def stress_management(depression_degree, work_time, relax_time):
    # 有效放松
    depression_degree -= (relax_time / work_time) * 100
    return depression_degree

# 主函数
def mental_adjustment(depression_degree, goal_num, work_time, relax_time):
    # 认知重构
    depression_degree = cognitive_reconstruction(depression_degree)
    print("认知重构后：", depression_degree)
    
    # 目标设定
    depression_degree, achieved_goals = goal_setting(depression_degree, goal_num)
    print("目标设定后：", depression_degree)
    print("实现目标数量：", achieved_goals)
    
    # 压力管理
    depression_degree = stress_management(depression_degree, work_time, relax_time)
    print("压力管理后：", depression_degree)
    
    return depression_degree

# 测试
depression_degree = 100
goal_num = 25
work_time = 50
relax_time = 30
depression_degree_final = mental_adjustment(depression_degree, goal_num, work_time, relax_time)
print("最终心态调整度：", depression_degree_final)
```

### 5.3 代码解读与分析

上述代码主要实现了心态调整与压力管理的三个核心步骤：认知重构、目标设定和压力管理。以下是代码的详细解读：

- **认知重构函数**：该函数随机选择一个积极认知，并将其从抑郁程度中减去，以减少抑郁程度。
- **目标设定函数**：该函数随机选择一个实现目标，并将其从抑郁程度中减去，同时记录实现目标数量。这样，抑郁程度会随着实现目标数量的增加而减少。
- **压力管理函数**：该函数通过计算有效放松时间与总工作时间的比例，将其从抑郁程度中减去，以减少抑郁程度。
- **主函数**：该函数依次调用认知重构、目标设定和压力管理函数，最终返回调整后的抑郁程度。
