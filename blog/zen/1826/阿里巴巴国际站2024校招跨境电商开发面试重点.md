### 4.2 公式

以下是几个常见的数学公式：

1. 线性回归公式：
   $$y = ax + b$$
   其中，$y$ 是因变量，$x$ 是自变量，$a$ 和 $b$ 是模型参数。
2. 概率公式：
   $$P(A) = \frac{n(A)}{n(S)}$$
   其中，$P(A)$ 是事件 $A$ 的概率，$n(A)$ 是事件 $A$ 的样本点数，$n(S)$ 是样本空间 $S$ 的样本点数。
3. 动态规划公式：
   $$f(i) = \min_{j=1}^{m} (c_{ij} + f(i-j))$$
   其中，$f(i)$ 是第 $i$ 个子问题的最优解，$c_{ij}$ 是第 $i$ 个子问题和第 $j$ 个子问题之间的成本。

### 4.3 举例说明

假设我们有一个电商系统，需要对商品进行排序，以便用户可以更方便地找到所需商品。我们可以使用快速排序算法来实现这一目标。以下是快速排序算法的Python实现：

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [3, 6, 8, 10, 1, 2, 1]
print(quick_sort(arr))
```

运行结果为 `[1, 1, 2, 3, 6, 8, 10]`，即对原数组进行了排序。
