## 3. 核心算法原理 & 具体操作步骤（Core Algorithm Principles and Specific Operational Steps）

以下是一个典型的算法面试题及其解答：

### 题目：给定一个整数数组，找到最大子序和。

### 解答：

1. 定义两个变量：`max_so_far` 和 `max_ending_here`，分别表示最大子序和和当前子序和。
2. 遍历数组，对于每个元素 `arr[i]`，更新 `max_ending_here` 为 `arr[i]` 和 `max_ending_here + arr[i]` 的最大值。
3. 如果 `max_ending_here` 大于 `max_so_far`，则更新 `max_so_far`。
4. 返回 `max_so_far` 作为最大子序和。

```python
def max_subarray_sum(arr):
    max_so_far = arr[0]
    max_ending_here = arr[0]
    
    for i in range(1, len(arr)):
        max_ending_here = max(arr[i], max_ending_here + arr[i])
        max_so_far = max(max_so_far, max_ending_here)
    
    return max_so_far
```

### 题目：给定一个数列，求其前 n 项和。

### 解答：

1. 定义数列的通项公式 `a[n] = an^2 + bn + c`。
2. 使用求和公式 `S[n] = n/2 * (a[1] + a[n])` 计算前 n 项和。

def sum_of_series(n, a, b, c):
