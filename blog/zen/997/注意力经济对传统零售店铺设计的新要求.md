#### 算法编程题与解析

**1. 计算店铺中顾客的平均移动路径长度。**

**题目描述：** 假设一个店铺的布局是一个正方形网格，每个格子代表一个单位面积。顾客在店铺内随机走动，要求计算顾客的平均移动路径长度。

**答案：** 可以使用深度优先搜索（DFS）算法遍历网格，记录每个格子被访问的次数，然后计算平均移动路径长度。

```python
def average_path_length(grid):
    rows, cols = len(grid), len(grid[0])
    visited = [[False for _ in range(cols)] for _ in range(rows)]
    total_path_length = 0
    num_paths = 0

    def dfs(i, j):
        nonlocal total_path_length, num_paths
        if visited[i][j] or i < 0 or i >= rows or j < 0 or j >= cols:
            return
        visited[i][j] = True
        total_path_length += abs(i - start_i) + abs(j - start_j)
        num_paths += 1
        dfs(i - 1, j)
        dfs(i + 1, j)
        dfs(i, j - 1)
        dfs(i, j + 1)

    start_i, start_j = 0, 0
    dfs(start_i, start_j)
    return total_path_length / num_paths if num_paths > 0 else 0
```

**2. 优化店铺照明系统以节省能源。**

**题目描述：** 假设一个店铺的照明系统由若干盏灯组成，每盏灯都有不同的功率。要求设计一个算法，根据店铺的布局和顾客的移动路径，优化照明系统以节省能源。

**答案：** 可以使用贪心算法，优先关闭功率较低的灯，并根据顾客的移动路径调整灯的状态。

```python
def optimize_lighting(grid, customers):
    rows, cols = len(grid), len(grid[0])
    lights = [(i, j, power) for i in range(rows) for j in range(cols) for power in grid[i][j]]

    def is_customer_near(light, customer):
        return abs(light[0] - customer[0]) + abs(light[1] - customer[1]) <= threshold

    threshold = 1
    total_power = 0

    for light in lights:
        for customer in customers:
            if is_customer_near(light, customer):
                total_power += light[2]
                break

    return total_power
```
