### 4.1 数学模型构建

构建用户需求分析模型需要以下几个数学公式：

1. **用户兴趣度**：$I_i = \frac{B_i}{T_i}$

# 读取用户行为数据
data = pd.read_csv('user_behavior.csv')

# 计算用户兴趣度
data['interest'] = data[['programming_views', 'literature_views']].sum(axis=1) / data['total_views']

# 计算内容深度需求
data['depth'] = data[['programming_comments', 'literature_comments']].sum(axis=1) / data['total_comments']

# 计算互动性需求

# 打印结果
print(data[['interest', 'depth', 'activity']])

- `data['interest'] = data[['programming_views', 'literature_views']].sum(axis=1) / data['total_views']`：计算用户对编程和文学领域的兴趣度。
- `data['depth'] = data[['programming_comments', 'literature_comments']].sum(axis=1) / data['total_comments']`：计算用户对内容深度的需求。
