## 2. 核心概念与联系

### 2.1 微分几何基本概念

微分几何是研究几何对象在局部区域内的性质，如曲线、曲面以及更高维度的超曲面等。其核心概念包括：

- **流形**：一个具有局部欧几里德结构的空间，可以是二维曲面、三维空间或者更高维度的空间。
- **度量**：用来衡量距离、角度和曲率的函数，是流形上的结构元素之一。
- **曲率**：描述空间弯曲程度的量，对于理解空间结构至关重要。

### 2.2 转盘系的概念

转盘系（或称为“转动盘”）是一个特殊的几何结构，可以视为在一定条件下绕着某个轴线旋转的曲面。在微分几何中，转盘系可以被描述为在三维空间中的旋转曲面，其几何性质在物理理论中有广泛应用。转盘系的特点包括：

- **正交性**：在转盘系中，不同轴线上的向量相互正交，意味着它们之间不存在夹角，且可以构成一组标准基。
- **刚性**：转盘系在物理理论中指的是其几何结构不会因外部力的作用而发生变形，保持固有的形状和尺寸。

### 2.3 转盘系与广义相对论

转盘系在广义相对论中的应用主要体现在描述宇宙中的空间结构和物质运动。在广义相对论中，转盘系的概念可以帮助理解时空的几何结构如何受到物质分布的影响，以及物质如何通过引力相互作用。通过引入转盘系的概念，可以更直观地分析黑洞周围的时空弯曲现象、宇宙膨胀的模式，以及宇宙大尺度结构的形成。

```python
import numpy as np
import matplotlib.pyplot as plt

def parametric_disk(r, theta, phi):
    \"\"\"
    参数化函数，用于生成转盘系的三维坐标。
    :param r: 半径
    :param theta: 极角，范围[0, 2π]
    :param phi: 方位角，范围[0, π]
    :return: 转盘系的三维坐标
    \"\"\"
    x = r * np.cos(theta) * np.sin(phi)
    y = r * np.sin(theta) * np.sin(phi)
    z = r * np.cos(phi)
    return np.column_stack((x, y, z))

def plot_disk(r):
    \"\"\"
    绘制转盘系。
    \"\"\"
    theta = np.linspace(0, 2*np.pi, 100)
    phi = np.linspace(0, np.pi, 50)
    theta, phi = np.meshgrid(theta, phi)
    X, Y, Z = parametric_disk(r, theta, phi)
    fig = plt.figure()
    ax = fig.add_subplot(111, projection='3d')
    ax.plot_surface(X, Y, Z)
    ax.set_xlabel('X axis')
    ax.set_ylabel('Y axis')
    ax.set_zlabel('Z axis')
    plt.show()

if __name__ == \"__main__\":
    plot_disk(1)  # 绘制半径为1的转盘系
```
