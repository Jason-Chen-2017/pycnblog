#### 5.2.2 分片上传Controller
```java
@RestController
@RequestMapping("/video")
public class UploadController {

    @Autowired
    private UploadService uploadService;

    @PostMapping("/upload")
    public Result upload(MultipartFile file, String md5, Integer chunk, Integer chunks) throws IOException {
        if (chunk == null || chunks == null) {
            return Result.ofFail(-1, "缺少必要参数");
        }
        String uploadPath = FileUtil.getUploadPath(md5, chunk);
        FileUtil.writeWithNotExist(uploadPath, file.getBytes());
        if (chunk == chunks - 1) {
            uploadService.merge(md5, chunks);
        }
        return Result.ofSuccess();
    }
}
```
