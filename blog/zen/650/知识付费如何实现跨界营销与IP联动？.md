### 5.2 源代码详细实现

下面以知识付费平台为例，给出使用Python进行K-means聚类分析的代码实现。

```python
import pandas as pd
from sklearn.cluster import KMeans
import numpy as np

# 读取用户数据
user_data = pd.read_csv('user_data.csv')

# 提取特征
features = user_data[['age', 'gender', 'study_hours']]

# 初始化K-means模型
kmeans = KMeans(n_clusters=5, random_state=42)

# 进行聚类
kmeans.fit(features)

# 输出聚类结果
labels = kmeans.labels_
print(labels)
```

这段代码实现了K-means聚类分析的基本流程，包括数据读取、特征提取、模型训练和结果输出。通过聚类分析，可以构建用户画像，推荐用户感兴趣的内容和产品。

**特征提取**：

**K-means模型训练**：
- 初始化K-means模型，设置聚类数为5。

**聚类结果输出**：
