### 4.1 数学模型构建

求解多项式方程的数学模型构建主要涉及以下几个方面：

### 4.2 公式推导过程

#### 4.2.1 代入法

代入法的推导过程如下：

1. 将多项式方程写为：
   $$
   P(x) = a_nx^n + a_{n-1}x^{n-1} + \ldots + a_1x + a_0
   $$
2. 选择一个解 $x_0$ 代入方程：
   $$
   P(x_0) = a_nx_0^n + a_{n-1}x_0^{n-1} + \ldots + a_1x_0 + a_0
   $$
3. 验证 $P(x_0) = 0$，则 $x_0$ 是多项式方程的一个解。

#### 4.2.2 因式分解法

因式分解法的推导过程如下：

1. 将多项式方程写为：
   $$

   $$
2. 尝试因式分解多项式：
   $$

3. 展开因式分解后的多项式，与原多项式比较系数，得到解。

#### 4.2.3 牛顿法

牛顿法的推导过程如下：

1. 设 $f(x) = P(x)$，则 $f'(x) = P'(x)$。
2. 选择一个初始解 $x_0$。
3. 计算迭代公式：
   $$
   x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}
   $$
4. 重复步骤3，直到 $x_{k+1}$ 与 $x_k$ 的差值小于某个阈值。

def solve_poly(x, a, b, c):
    """
    求解一元二次方程 ax^2 + bx + c = 0
    """
    discriminant = b**2 - 4*a*c
    if discriminant < 0:
        return "无实数解"
    else:
        x1 = (-b + np.sqrt(discriminant)) / (2*a)
        x2 = (-b - np.sqrt(discriminant)) / (2*a)
        return x1, x2

x, a, b, c = 1, 1, -5, 6
solution = solve_poly(x, a, b, c)
print(solution)
