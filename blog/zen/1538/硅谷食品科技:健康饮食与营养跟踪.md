### 5.2 源代码详细实现

以下是一个简化的Python代码示例，用于实现线性回归模型：

```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression

# 数据集加载
data = pd.read_csv("nutrition_data.csv")

# 特征和目标变量提取
X = data.iloc[:, :-1].values
y = data.iloc[:, -1].values

# 模型训练
model = LinearRegression()
model.fit(X, y)

# 模型预测
predicted_nutrition = model.predict(X)

# 结果分析
np.set_printoptions(precision=2)
print("Predicted nutrition:", predicted_nutrition)
```
