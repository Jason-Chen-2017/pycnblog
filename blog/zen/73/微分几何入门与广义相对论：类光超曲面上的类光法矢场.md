## 2. 核心概念与联系

### 2.1 微分几何简介

微分几何是研究几何结构上的微分性质的数学分支。在物理学中，它被用来描述和分析物理量在空间和时间上的变化。广义相对论中的时空可以用四维黎曼流形来描述，其中黎曼度量给出了距离、角度和曲率的概念。

### 2.2 类光超曲面

在四维时空中，一个类光超曲面指的是那些在任意一点处都存在至少一个类光向量（即速度等于光速的方向）的超曲面。在广义相对论中，这样的超曲面可以用来描述黑洞事件视界、宇宙膨胀前沿等物理现象。

### 2.3 类光法矢场

类光法矢场是在类光超曲面上的法向量场，即垂直于超曲面的向量场。在物理上，类光法矢场可以用来描述超曲面的“外表面”特性，对于理解超曲面的几何结构及其物理行为至关重要。

g = g.simplify()

# 计算拉格朗日度量
L = lambda u, v: np.linalg.norm(x.diff(u)).dot(g).dot(x.diff(u))

# 寻找类光法矢场
# 假设法向量为法向量场的导数
N = Matrix([x.diff(u), x.diff(v)])
N = N.simplify()

# 类光条件：N*N = c^2 * g(N,N)
c_squared = 1
class_light_condition = N.dot(N) - c_squared * g.subs({u: u, v: v}).dot(N)

# 解决类光条件，寻找合适的N
# 这里采用数值方法解方程
from scipy.optimize import minimize
sol = minimize(lambda x: class_light_condition.subs({u: x[0], v: x[1]}), [u, v])
