### 5.2 源代码详细实现

以下是一个简单的Python代码实例，用于创建一个简单的思维导图：

```python
import matplotlib.pyplot as plt
import networkx as nx

# 创建一个图
G = nx.Graph()

# 添加节点和边
G.add_nodes_from(['A', 'B', 'C'])
G.add_edges_from([('A', 'B'), ('B', 'C')])

# 绘制思维导图
pos = nx.spring_layout(G)
nx.draw(G, pos, with_labels=True)

# 显示图形
plt.show()
```
