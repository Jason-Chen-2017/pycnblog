## 2. 核心概念与联系

### 2.1 复分析基础

黎曼曲面理论建立在复分析的基础上，因此，我们首先简要回顾一下复分析的基本概念。

#### 2.1.1 复数

复数是由实数和虚数单位$i$组成的数，可以表示为$a + bi$的形式，其中$a$和$b$是实数，$i^2 = -1$。

#### 2.1.2 复变函数

复变函数是定义在复数域上的函数，通常用$z$表示自变量，$f(z)$表示函数值。复变函数有丰富的性质，如解析性、可微性等。

### 2.2 黎曼曲面定义

黎曼曲面是复多变量函数的等价类，它将复多变量函数的多个值域点映射到同一个点。更具体地说，给定一个复多变量函数$f(z, w)$，它的黎曼曲面是所有满足$f(z, w) = c$的点$(z, w)$的集合。

### 2.3 黎曼曲面性质

黎曼曲面具有以下性质：

- **连通性**：黎曼曲面是一个连通的复流形。
- **紧致性**：黎曼曲面是紧致的复流形。
- **可微性**：黎曼曲面上的函数是可微的。

### 5.2 源代码详细实现

以下是一个简单的Python代码实例，用于绘制解析函数$f(z, w) = z^2 + w^2$的黎曼曲面。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义解析函数
def f(z, w):
    return z**2 + w**2

# 绘制黎曼曲面
def plot_riemann_surface(z, w):
    x, y = z.real, z.imag
    u, v = w.real, w.imag
    u_min, u_max = -2, 2
    v_min, v_max = -2, 2
    z = np.linspace(u_min, u_max, 400)
    w = np.linspace(v_min, v_max, 400)
    X, Y = np.meshgrid(z, w)
    Z = f(X + 1j*Y)
    plt.figure(figsize=(8, 6))
    plt.contourf(X, Y, Z.real, 20, cmap='viridis')
    plt.colorbar()
    plt.show()

# 绘制解析函数的黎曼曲面
plot_riemann_surface(1+1j, 1+1j)
```
