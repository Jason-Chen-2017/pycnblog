## 4.数学模型和公式详细讲解举例说明

接下来重点讲解EOQ模型和ABC分类法所涉及的数学模型和公式。

### 4.1 EOQ模型
EOQ模型的目标是求解使总订货成本和总持有成本之和最小的最优订货批量Q。其数学模型为:

$$TC(Q) = \frac{D}{Q}S + \frac{Q}{2}H$$

其中:
- TC(Q):总成本函数
- D:年需求量
- S:单位订货成本
- H:单位年持有成本
- Q:订货批量

通过对TC(Q)求导并令其等于0,可得最优订货批量:

$$EOQ = \sqrt{\frac{2DS}{H}}$$

举例说明:某物资年需求量为1000件,单位订货成本为100元,单位年持有成本为20元,则其EOQ为:

$$EOQ = \sqrt{\frac{2*1000*100}{20}} = 100$$

### 4.2 ABC分类法
ABC分类法是基于帕累托(20/80)法则,将物资按年度发出金额大小分为三类,并采取不同管理策略的方法。其数学模型为:

$$P_i = \frac{D_iC_i}{\sum_{j=1}^{n}D_jC_j} \times 100\%$$

其中:
- $P_i$:第i种物资的年度发出金额百分比
- $D_i$:第i种物资的年需求量
- $C_i$:第i种物资的单价
- n:物资品种总数

举例说明:某企业有甲乙丙三种物资,年用量和单价分别为:
- 甲:1000件,100元/件
- 乙:500件,50元/件
- 丙:2000件,10元/件

则三种物资的年度发出金额百分比分别为:
- 甲:$P_1 = \frac{1000*100}{1000*100+500*50+2000*10} = 76.9\%$
- 乙:$P_2 = \frac{500*50}{1000*100+500*50+2000*10} = 19.2\%$
- 丙:$P_3 = \frac{2000*10}{1000*100+500*50+2000*10} = 3.9\%$

可见甲属于A类,乙属于B类,丙属于C类。

### 5.1 物资编码
```java
public class Material {
    private String code; //物资编码
    private String name; //物资名称
    private String spec; //规格型号

    //生成物资编码
    public static String generateCode(String category, String subCategory, int seq) {
        return category + subCategory + String.format("%05d", seq);
    }
}
```

`Material`类封装了物资的基本信息,其中`generateCode`方法用于根据物资分类、子分类和序号生成唯一的物资编码。

### 5.2 库存管理
```java
public class Inventory {
    private String materialCode; //物资编码
    private String warehouseCode; //仓库编码
    private BigDecimal quantity; //库存数量

    //入库操作
    public void warehousing(BigDecimal amount) {
        this.quantity = this.quantity.add(amount);
    }

    //出库操作
    public void delivery(BigDecimal amount) {
        if(this.quantity.compareTo(amount) >= 0) {
            this.quantity = this.quantity.subtract(amount);
        } else {
            throw new RuntimeException("库存不足");
        }
    }
}
```

`Inventory`类用于管理具体某种物资在某仓库的库存,提供了入库(`warehousing`)和出库(`delivery`)操作,并进行库存不足检查。

### 5.3 EOQ与ABC分类
```java
public class InventoryOptimizer {
    //EOQ计算
    public static double calculateEOQ(int annualDemand, double orderCost, double holdingCost) {
        return Math.sqrt(2 * annualDemand * orderCost / holdingCost);
    }

    //ABC分类
    public static String classifyABC(Material material, List<Material> materials) {
        double amount = material.getAnnualDemand() * material.getUnitPrice();
        double totalAmount = materials.stream().mapToDouble(m -> m.getAnnualDemand() * m.getUnitPrice()).sum();
        double percentage = amount / totalAmount;

        if(percentage <= 0.8) {
            return "A";
        } else if(percentage <= 0.95) {
            return "B";
        } else {
            return "C";
        }
    }
}
```

`InventoryOptimizer`类实现了EOQ和ABC分类的算法。`calculateEOQ`方法根据年需求量、单位订货成本、单位年持有成本计算经济订货批量。`classifyABC`方法根据物资年度发出金额占比将其划分为A、B、C三类。

## 9.附录：常见问题与解答

### Q1:物资编码的原则是什么?
A1:物资编码应遵循"唯一性、稳定性、扩展性、简洁性"四项原则,通过分类编码、流水号等方式生成。

### Q2:如何判断库存是否合理?
A2:可通过库存周转率、库龄分析等指标评估库存合理性。同时要结合EOQ、ABC分类等方法优化库存。

### Q3:如何提高出入库效率?
A3:主要措施包括:优化作业流程、采用移动终端、条码/RFID技术、WMS系统等。

### Q4:物资调拨的主要方式有哪些?
A4:常见的调拨方式有:总仓到分仓、分仓间横向调拨、同仓库内部调拨等。

### Q5:盘点的作用是什么?
A5:盘点可以帮助企业准确掌握实际库存状况,及时发现和解决存货管理问题,为库存优化提供数据基础。
