以下是一个简单的Python代码，用于定义序列多项式环并进行加法和乘法运算：

```python
import sympy as sp

# 定义符号变量
x = sp.symbols('x')

# 定义序列多项式环
R = sp.PolynomialRing(sp.RationalField(), x)

# 定义两个序列多项式
f = R(x)
g = R(1 + x)

# 计算加法和乘法
f_plus_g = f + g
f_times_g = f * g

# 输出结果
print("f(x) + g(x) =", f_plus_g)
print("f(x) * g(x) =", f_times_g)
```
