### 4.1 数学模型构建

2. **量子门操作**：使用量子门 \(U\) 来操作量子比特，实现量子态的转换。
3. **量子测量**：使用测量算符 \(M\) 来测量量子系统的状态。

### 4.2 公式推导过程

1. **量子态演化**：

   量子态的演化由薛定谔方程描述：

   \[i\hbar \frac{\partial}{\partial t} |\psi\rangle = H |\psi\rangle\]

   其中，\(H\) 是哈密顿量，\(\hbar\) 是约化普朗克常数。

2. **量子门操作**：

   量子门 \(U\) 可以表示为：

   \[U = \sum_{i} a_i |i\rangle\langle i|\]

   其中，\(a_i\) 是复数系数，\(|i\rangle\) 是量子态基。

3. **量子测量**：

   量子测量可以表示为：

   \[P = \sum_{i} |i\rangle\langle i|\]

   其中，\(P\) 是测量算符，\(|i\rangle\) 是测量结果的基。

import qiskit

import numpy as np

qc.h(qc.q[0])

# CNOT 门操作
qc.cx(qc.q[0], qc.q[1])

qc.measure_all()

# 编译量子电路

result = backend.run(qc).result()

# 获取量子态

# 编译模型

# 训练模型
model.fit(state, state, epochs=100)

# 评估模型
loss, accuracy = model.evaluate(state, state)
print(f"Loss: {loss}, Accuracy: {accuracy}")
