### 5.2 源代码详细实现

下面是多重身份管理的源代码实现：

```python
import PyQt5
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
import sys

class IdentityManager:
    def __init__(self):
        self.identities = []
        self.key = get_random_bytes(16)

    def create_identity(self, name):
        cipher = AES.new(self.key, AES.MODE_EAX)
        nonce = cipher.nonce
        tag = cipher.tag
        ciphertext = cipher.encrypt(name.encode('utf-8'))
        self.identities.append({'name': name, 'nonce': nonce, 'tag': tag, 'ciphertext': ciphertext})

    def delete_identity(self, name):
        for identity in self.identities:
            if identity['name'] == name:
                self.identities.remove(identity)
                return True
        return False

    def list_identities(self):
        return self.identities

app = PyQt5.QtWidgets.QApplication(sys.argv)
identity_manager = IdentityManager()

window = PyQt5.QtWidgets.QWidget()
window.setWindowTitle('Identity Manager')
window.setGeometry(100, 100, 400, 300)
```

代码中的关键部分是加密和解密操作。我们使用了AES加密算法，它是一种常用的对称加密算法。在创建身份时，我们将用户输入的身份信息加密并存储。在删除身份时，我们只需要将对应身份信息从列表中移除。在列出身份时，我们返回所有已创建的身份信息。
