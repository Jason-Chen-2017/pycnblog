## 4. 数学模型和公式 & 详细讲解 & 举例说明

### 4.1 数学模型构建

在专利分析与管理中，常用的数学模型包括：

#### 4.1.1 关联规则挖掘模型

假设有专利集 D = {d1, d2, ..., dn}，其中每个专利 di 表示为一个集合 Ci = {ci1, ci2, ..., cim}，表示该专利的关键词。关联规则挖掘的目标是找出满足最小支持度和最小置信度的关键词组合。

支持度（Support）表示关键词组合在专利集 D 中的比例：

$$
Support(A \cap B) = \frac{|D(A \cap B)|}{|D|}
$$

置信度（Confidence）表示在存在关键词 A 的情况下，关键词 B 同时存在的概率：

$$
Confidence(A \rightarrow B) = \frac{|D(A \cap B)|}{|D(A)|}
$$

其中，|D|表示专利集 D 的总数，|D(A)|表示在专利集 D 中包含关键词 A 的专利数，|D(A \cap B)|表示在专利集 D 中同时包含关键词 A 和 B 的专利数。

#### 4.1.2 机器学习模型

在机器学习模型中，常用的模型包括线性回归、逻辑回归、决策树、随机森林和支持向量机等。以下以逻辑回归为例进行说明。

逻辑回归是一种概率型线性模型，用于分类任务。其公式如下：

$$
P(y=1|X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_1 + ... + \beta_nx_n)}}
$$

其中，X 表示特征向量，y 表示标签，$\beta_0, \beta_1, ..., \beta_n$表示模型参数。

### 4.2 公式推导过程

以逻辑回归模型为例，推导过程如下：

假设有 m 个训练样本，每个样本表示为一个特征向量 X = [x1, x2, ..., xn] 和标签 y ∈ {0, 1}。

对于每个样本，计算其预测概率：

$$
\hat{y}_i = P(y=1|X_i) = \frac{1}{1 + e^{-(\beta_0 + \beta_1x_{i1} + ... + \beta_nx_{in})}}
$$

定义损失函数为：

$$
J(\beta) = -\frac{1}{m} \sum_{i=1}^{m} y_i \log(\hat{y}_i) + (1 - y_i) \log(1 - \hat{y}_i)
$$

其中，J(β) 表示模型在训练数据上的损失。

为了求解最优参数 β，需要对 J(β) 求导并令其等于 0：

$$
\frac{\partial J(\beta)}{\partial \beta_j} = -\frac{1}{m} \sum_{i=1}^{m} [y_i - \hat{y}_i] x_{ij} = 0
$$

解得：

$$
\beta_j = \frac{1}{m} \sum_{i=1}^{m} x_{ij} y_i
$$

### 4.3 案例分析与讲解

假设有如下一个简单的专利数据集：

| 专利编号 | 关键词1 | 关键词2 | 标签 |
|--------|------|------|----|
| 1      | A    | B    | 1  |
| 2      | A    | C    | 1  |
| 3      | B    | C    | 1  |
| 4      | A    | D    | 0  |
| 5      | B    | D    | 0  |

#### 4.3.1 关联规则挖掘

利用Apriori算法挖掘关键词组合，设置最小支持度为 0.5，最小置信度为 0.7。

1. 初始集合：{A, B, C, D}
2. 计算支持度：{AB}，{AC}，{AD}，{BC}，{BD}，{CD} 的支持度分别为 0.4、0.4、0.4、0.4、0.4 和 0.4，均大于最小支持度。
3. 计算置信度：{AB} 的置信度为 0.8，大于最小置信度；{AC}，{AD}，{BC}，{BD}，{CD} 的置信度分别为 0.6、0.6、0.6、0.6 和 0.6，均小于最小置信度。

最终得到一个关联规则：{AB} → {C}，置信度为 0.8。

#### 4.3.2 逻辑回归模型

利用训练数据集 {1, 2, 3} 建立逻辑回归模型，参数 β = [0.5, 0.5, 0.5]。

对于样本 1，计算预测概率：

$$
P(y=1|X) = \frac{1}{1 + e^{-(0.5 + 0.5 \cdot 1 + 0.5 \cdot 1)}} = 0.8
$$

由于预测概率大于 0.5，将样本 1 的标签预测为 1。

```python
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import LabelEncoder

# 读取专利数据
data = pd.read_csv('patent_data.csv')

# 数据清洗和预处理
data.drop_duplicates(inplace=True)
data.fillna('', inplace=True)

# 特征提取
data['keyword_list'] = data['abstract'].apply(lambda x: x.split())

# 标签编码
label_encoder = LabelEncoder()
data['label'] = label_encoder.fit_transform(data['label'])

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(data['keyword_list'], data['label'], test_size=0.2, random_state=42)
```

```python
from sklearn.linear_model import LogisticRegression
```
