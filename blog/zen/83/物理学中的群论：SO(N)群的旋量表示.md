## 2. 核心概念与联系

### 2.1 群论的基本概念

群论是研究具有封闭性、结合律和单位元等性质的代数系统的数学分支。一个群G包含一组元素，以及一个二元运算（通常是乘法），满足以下条件：

1. 封闭性：对于群G中的任意两个元素a和b，它们的运算结果c也属于群G。
2. 结合律：对于群G中的任意三个元素a、b和c，有(a*b)*c = a*(b*c)。
3. 单位元：存在一个元素e，使得对于群G中的任意元素a，都有e*a = a*e = a。

### 2.2 SO(N)群的性质

SO(N)群是描述N维空间旋转对称性的群，其元素是N×N正交矩阵，满足行列式为1。SO(N)群具有以下性质：

1. 封闭性：SO(N)群中的任意两个矩阵的乘积仍然属于SO(N)群。
2. 结合律：SO(N)群中的任意三个矩阵的乘积满足结合律。
3. 单位元：单位矩阵I属于SO(N)群，且对于任意SO(N)群中的矩阵A，都有I*A = A*I = A。

### 2.3 旋量的定义和性质

旋量是四维空间中的一种特殊对象，可以表示为四维复向量。旋量具有以下性质：

1. 反对称性：旋量a和b的反交换子[a, b]定义为[a, b] = ab - ba，对于任意两个旋量a和b，[a, b]也是一个旋量，并且[a, b] = -[b, a]。
2. 宇称：旋量具有宇称性质，即旋量乘以一个宇称变换矩阵后，其值不变。

    eigenvalues, eigenvectors = np.linalg.eig(R)
