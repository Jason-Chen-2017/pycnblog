## 2. 核心概念与联系

### 2.1 纤维丛的定义

纤维丛是由一组局部同构构成的集合，这些局部同构将每个局部邻域映射到相同的纤维。一个纤维丛由以下四个元素组成：

- 柔性流形$M$，称为底流形。
- 纤维$F$，称为纤维。
- 拉回映射$\pi: E \rightarrow M$，称为投影映射。
- 局部同构$\phi_p: U_p \rightarrow U_p \times F$，称为纤维丛的局部结构。

### 2.2 纤维丛结构

纤维丛的局部结构$\phi_p: U_p \rightarrow U_p \times F$可以通过以下方式定义：

- 对于底流形$M$上的每个点$p$，存在一个局部邻域$U_p$。
- 对于$U_p$中的每个点$p$，存在一个纤维$f(p) \in F$。
- 纤维丛的局部结构$\phi_p$将$U_p$映射到$U_p \times F$，使得$\pi \circ \phi_p = id_{U_p}$。

### 2.3 纤维丛的基本性质

- **纤维丛的纤维是同构的**：对于纤维丛的每个局部邻域$U_p$，纤维$\phi_p(U_p)$与纤维$F$同构。
- **纤维丛的全局结构**：如果纤维丛的纤维在全局上都是相同的，则纤维丛是平凡的；否则，纤维丛是非平凡的。

#### 4.2.1 纤维丛的局部结构公式

$$\pi \circ \phi_p = id_{U_p}$$

这个公式表明纤维丛的局部结构$\phi_p$将$U_p$映射到$U_p \times F$，使得$\pi$在$\phi_p(U_p)$的作用下等于单位映射$ID_{U_p}$。

#### 4.2.2 纤维丛分解公式

霍奇分解：

$$E = U \times F + \sum_{i=1}^n E_i$$

斯温森分解：

$$E = U \times F + \sum_{i=1}^n \Sigma_i$$

其中，$E_i$和$\Sigma_i$分别表示霍奇分解和斯温森分解中的子纤维丛。

### 5.2 源代码详细实现

```python
import numpy as np
from scipy.linalg import svd

def fiber_bundle_decomposition(E):
    # 对矩阵E进行奇异值分解
    U, s, Vt = svd(E)
    # 计算分解后的纤维
    fiber = U[:, :len(s)]
    base = Vt[:, :len(s)]
    return fiber, base

# 创建一个纤维丛
E = np.random.rand(3, 6)

# 对纤维丛进行分解
fiber, base = fiber_bundle_decomposition(E)

# 输出分解结果
print("纤维：")
print(fiber)
print("\
底流形：")
print(base)
```

### 5.3 代码解读与分析

1. 导入所需的库，包括numpy和scipy。
2. 定义`fiber_bundle_decomposition`函数，该函数接收一个纤维丛E作为输入，并返回分解后的纤维和底流形。
3. 使用scipy库中的`svd`函数对纤维丛E进行奇异值分解。
4. 将奇异值分解后的左奇异向量作为纤维，右奇异向量作为底流形。
5. 创建一个随机的纤维丛E。
6. 对纤维丛E进行分解。
7. 输出分解后的纤维和底流形。
