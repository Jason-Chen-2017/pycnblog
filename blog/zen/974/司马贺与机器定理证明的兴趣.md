### 4.3 案例分析与讲解

以下是一个简单的数学定理的证明过程：

定理：对于任意的自然数 \( n \)，都有 \( 1 + 2 + \ldots + n = \frac{n(n+1)}{2} \)。

证明：

1. 当 \( n = 1 \) 时，显然成立。
2. 假设当 \( n = k \) 时，等式成立，即 \( 1 + 2 + \ldots + k = \frac{k(k+1)}{2} \)。
3. 当 \( n = k+1 \) 时，我们有：
   \[
   1 + 2 + \ldots + k + (k+1) = \frac{k(k+1)}{2} + (k+1) = \frac{(k+1)(k+2)}{2}
   \]
   这证明了当 \( n = k+1 \) 时，等式也成立。

由归纳法可知，对于任意的自然数 \( n \)，都有 \( 1 + 2 + \ldots + n = \frac{n(n+1)}{2} \)。

### 5.2 源代码详细实现

以下是一个简单的Python代码实例，用于证明上述的数学定理：

```python
from sympy import symbols, Eq, simplify

n = symbols('n')
sum_n = Eq(1 + 2 + ... + n, n * (n + 1) / 2)

print(simplify(sum_n))
```

### 5.3 代码解读与分析

这段代码首先导入了Python中的符号数学库Sympy，并定义了一个符号变量 `n`。然后，我们使用 `Eq` 函数创建了一个等式对象 `sum_n`，表示从1加到n的和等于n乘以n加1的一半。最后，我们使用 `simplify` 函数对等式进行化简，并打印出结果。

### 5.4 运行结果展示

运行上述代码，我们将得到以下结果：

```
n*(n + 1)/2
```

这证明了对于任意的自然数 \( n \)，都有 \( 1 + 2 + \ldots + n = \frac{n(n+1)}{2} \)。
