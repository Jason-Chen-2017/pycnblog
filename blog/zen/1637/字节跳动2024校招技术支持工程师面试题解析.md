#### 3.2.1 排序算法

1. 冒泡排序：

   ```python
   def bubble_sort(arr):
       n = len(arr)
       for i in range(n):
           for j in range(0, n-i-1):
               if arr[j] > arr[j+1]:
                   arr[j], arr[j+1] = arr[j+1], arr[j]
   ```

2. 选择排序：

   ```python
   def selection_sort(arr):
       n = len(arr)
       for i in range(n):
           min_idx = i
           for j in range(i+1, n):
               if arr[j] < arr[min_idx]:
                   min_idx = j
           arr[i], arr[min_idx] = arr[min_idx], arr[i]
   ```

3. 插入排序：

   ```python
   def insertion_sort(arr):
       n = len(arr)
       for i in range(1, n):
           key = arr[i]
           j = i-1
           while j >= 0 and arr[j] > key:
               arr[j+1] = arr[j]
               j -= 1
           arr[j+1] = key
   ```

4. 快速排序：

   ```python
   def quick_sort(arr, low, high):
       if low < high:
           pi = partition(arr, low, high)
           quick_sort(arr, low, pi-1)
           quick_sort(arr, pi+1, high)
   ```

#### 3.2.2 查找算法

1. 二分查找：

   ```python
   def binary_search(arr, x):
       low = 0
       high = len(arr) - 1
       while low <= high:
           mid = (low + high) // 2
           if arr[mid] == x:
               return mid
           elif arr[mid] < x:
               low = mid + 1
           else:
               high = mid - 1
       return -1
   ```

2. 线性查找：

   ```python
   def linear_search(arr, x):
       for i in range(len(arr)):
           if arr[i] == x:
               return i
       return -1
   ```

   ```python
   def dfs(graph, v, visited):
       visited[v] = True
       print(v, end=' ')
       for neighbor in graph[v]:
           if not visited[neighbor]:
               dfs(graph, neighbor, visited)
   ```

2. 广度优先搜索（BFS）：

   ```python
   from collections import deque

   def bfs(graph, start):
       visited = [False] * len(graph)
       queue = deque([start])
       visited[start] = True
       while queue:
           v = queue.popleft()
           print(v, end=' ')
           for neighbor in graph[v]:
               if not visited[neighbor]:
                   queue.append(neighbor)
                   visited[neighbor] = True
   ```

#### 3.2.4 动态规划

1. 最长公共子序列（LCS）：

   ```python
   def lcs(X, Y):
       m = len(X)
       n = len(Y)
       dp = [[0] * (n+1) for _ in range(m+1)]
       for i in range(1, m+1):
           for j in range(1, n+1):
               if X[i-1] == Y[j-1]:
                   dp[i][j] = dp[i-1][j-1] + 1
               else:
                   dp[i][j] = max(dp[i-1][j], dp[i][j-1])
       return dp[m][n]
   ```

- 查找算法：二分查找的时间复杂度较低，适用于大数据量的查找；线性查找的时间复杂度较高，适用于小数据量的查找。

- 动态规划：动态规划可以求解最优子结构问题，但需要较大的内存空间。

### 4.2 公式推导过程

以下是一个简单的线性规划公式的推导过程：

目标函数：$max z = c^T x$

约束条件：$Ax \leq b$

松弛变量：$x_r$

引入松弛变量后的线性规划公式：$max z = c^T x + c_r x_r$

约束条件：$Ax + Ax_r \leq b$

目标函数值：$z = c^T x + c_r x_r$

根据目标函数的定义，我们可以得到：

$z = c^T x + c_r x_r = c^T x + c_r (b - Ax)$

$z = c^T x + c_r b - c^T A x$

$z = c^T (x - A^T c_r) + c_r b$

为了使目标函数取得最大值，我们需要使$x - A^T c_r$取得最大值。

由于$x_r$是非负的，我们可以选择$c_r$的值为$-c^T A$，这样$x - A^T c_r$的值就等于$x$的值。

因此，最优解为：$x = A^T c_r + b$

### 4.3 案例分析与讲解

假设我们有一个简单的线性规划问题，要求在满足约束条件的情况下最大化目标函数。

目标函数：$max z = 2x + 3y$

约束条件：$x + y \leq 4$

$x - 2y \geq 1$

$x, y \geq 0$

首先，我们可以将约束条件转化为标准形式：

$x + y - s_1 = 4$

$x - 2y + s_2 = 1$

$s_1, s_2 \geq 0$

然后，我们可以构造拉格朗日函数：

$L(x, y, s_1, s_2) = 2x + 3y + \lambda_1 (4 - x - y) + \lambda_2 (1 - x + 2y)$

对$x, y, s_1, s_2$求偏导数并令其为零，得到：

$\frac{\partial L}{\partial x} = 2 - \lambda_1 - \lambda_2 = 0$

$\frac{\partial L}{\partial y} = 3 - \lambda_1 + 2\lambda_2 = 0$

$\frac{\partial L}{\partial s_1} = -x - y + 4 = 0$

$\frac{\partial L}{\partial s_2} = -x + 2y - 1 = 0$

解上述方程组，得到：

$x = 2$

$y = 1$

$s_1 = 2$

$s_2 = 0$

此时，目标函数取得最大值：$z = 2 \times 2 + 3 \times 1 = 7$

因此，最优解为：$x = 2, y = 1$

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import linprog

# 定义目标函数
c = np.array([2, 3])

# 定义约束条件
A = np.array([[1, 1], [1, -2]])
b = np.array([4, 1])

# 定义变量
x0 = np.zeros(2)

# 求解线性规划问题
result = linprog(c, A_eq=A, b_eq=b, x0=x0)

# 输出结果
print("最优解：x =", result.x)
print("最大值：z =", result.fun)

# 绘制约束条件的图形
x1 = np.linspace(0, 4, 100)
x2 = (4 - x1) / 1
plt.plot(x1, x2, label="约束条件")
plt.scatter(result.x[0], result.x[1], color="red", label="最优解")
plt.xlabel("x1")
plt.ylabel("x2")
plt.legend()
plt.show()
```
