### 5.2 源代码详细实现

以下是一个简单的 Python 代码实例，用于实现物理实体自动化：

```python
import numpy as np
import matplotlib.pyplot as plt

# 状态空间模型参数
A = np.array([[1, 1], [0, 1]])
B = np.array([[1], [0]])
C = np.array([[1, 0]])
D = np.array([[0]])

# 初始状态
x0 = np.array([[0], [1]])

# 控制输入
u = np.array([[1], [0]])

# 模型训练
x = np.zeros((2, 100))
x[:, 0] = x0
for k in range(1, 100):
    x[:, k] = A @ x[:, k - 1] + B @ u

# 预测结果
y = C @ x

# 可视化结果
plt.plot(x[0, :], y)
plt.show()
```

- **预测结果**：使用 C 矩阵，将状态转换为输出。
