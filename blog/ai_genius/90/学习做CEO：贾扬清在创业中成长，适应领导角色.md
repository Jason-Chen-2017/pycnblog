#### 伪代码：领导力评估算法

```
function 领导力评估（团队成员列表）：
    初始化总评分 = 0
    初始化成员数量 = 长度（团队成员列表）

    for 每个团队成员 in 团队成员列表：
        获取团队成员的领导力评分
        总评分 = 总评分 + 领导力评分

    平均领导力评分 = 总评分 / 成员数量
    return 平均领导力评分
```

```python
class TeamMember:
    def __init__(self, name, role, leadership_score):
        self.name = name
        self.role = role
        self.leadership_score = leadership_score

class TeamBuilder:
    def __init__(self):
        self.members = []

    def add_member(self, member):
        self.members.append(member)

    def calculate_average_leadership(self):
        total_score = 0
        for member in self.members:
            total_score += member.leadership_score
        return total_score / len(self.members)

# 创建团队成员对象
member1 = TeamMember("Alice", "Developer", 8)
member2 = TeamMember("Bob", "Designer", 7)
member3 = TeamMember("Charlie", "Tester", 9)

# 创建团队构建对象
team_builder = TeamBuilder()

# 添加团队成员
team_builder.add_member(member1)
team_builder.add_member(member2)
team_builder.add_member(member3)

# 计算平均领导力评分
average_leadership = team_builder.calculate_average_leadership()
print(f"平均领导力评分：{average_leadership}")
```

- `TeamMember` 类定义了团队成员的基本信息，包括姓名、角色和领导力评分。
- `TeamBuilder` 类负责管理团队成员的列表，并提供添加成员和计算平均领导力评分的方法。
- 实例化 `TeamMember` 对象并添加到 `TeamBuilder` 对象的成员列表中。
- 调用 `TeamBuilder` 对象的 `calculate_average_leadership` 方法计算平均领导力评分，并打印结果。
