#### 1.1 自然现象中的数学

自然现象是数学起源的一个重要背景。在人类早期，自然现象如日出日落、潮汐变化、季节更替等对人类的生活产生了深远的影响。为了更好地适应和预测这些自然现象，人类开始探索数量和形状的概念。

首先，从简单的计数开始，人类学会了数数来记录日常生活中的事物。例如，农民种植作物时需要记录种植的植物数量，猎人需要记录狩猎的动物数量。这种简单的计数活动促使人类对数量有了初步的理解。

随着时间的推移，人类开始关注形状和空间的关系。例如，人们在建造房屋和制作工具时，需要考虑几何形状的优化。古埃及人修建金字塔时，使用了精确的几何知识来设计和建造。这些实践活动不仅促进了数学的发展，也为后来的几何学奠定了基础。

此外，自然现象还促使人类探索时间的变化规律。古代文明如巴比伦和印度发展了时间测量系统，用于记录农业活动和宗教仪式。这些时间测量系统为后来的算术和代数的发展提供了重要启示。

总的来说，自然现象中的数学探索不仅反映了人类对自然的认知和适应，也为数学的起源和发展提供了丰富的实践基础。通过观察和解读自然现象，人类逐渐发展出了各种数学工具和原理，为后来的数学理论体系奠定了基础。

### 第1章：数学的起源

#### 1.2 人类对数学的探索

人类对数学的探索是一个漫长的过程，贯穿了从古代到现代的历史。在这个探索过程中，数学从最初对日常生活的简单记录和测量，逐渐发展成为一门系统化的学科，涵盖了广泛的理论和应用。

在古代，数学的发展主要体现在解决实际问题的需求上。例如，古埃及人和巴比伦人发展了算术和几何学，用于测量土地、计算税收、建造建筑物等。古埃及的数学文献《阿梅斯纸草文稿》和巴比伦的《汉谟拉比法典》中，都有关于数学应用的记录。这些古代文明通过解决实际问题，积累了大量的数学知识和经验。

古希腊是数学发展的重要阶段。古希腊数学家如泰勒斯、毕达哥拉斯、欧几里得等，对数学进行了系统的研究和理论化。泰勒斯提出了“一切皆数”的观点，为后来的数学奠定了哲学基础。毕达哥拉斯提出的“毕达哥拉斯定理”在几何学中具有重要地位。欧几里得的《几何原本》则建立了几何学的完整体系，对后世数学的发展产生了深远影响。

中世纪的阿拉伯世界是数学发展的另一个高峰。阿拉伯数学家如花拉子米和阿尔-哈里德希，翻译并传播了古希腊的数学文献，同时他们自己的研究也对数学的发展做出了重要贡献。花拉子米提出了代数的系统化方法，而阿尔-哈里德希则在三角学和天文学中做出了开创性的工作。

近代欧洲的文艺复兴和科学革命，推动了数学的进一步发展。法国数学家笛卡尔引入了坐标几何，将代数和几何结合在一起，为后来的解析几何奠定了基础。英国数学家牛顿和德国数学家莱布尼茨几乎同时发明了微积分，这使得数学在描述自然现象和解决科学问题中发挥了关键作用。

在现代，数学的发展进入了一个多样化、高度专业化的阶段。数学家们不仅在纯数学领域取得了重大突破，如哥德尔的不完备性定理、陈景润的哥德巴赫猜想证明等，还在应用数学领域取得了丰硕成果，如数学在计算机科学、经济学、生物学等领域的应用。

总的来说，人类对数学的探索是一个不断演进的过程。从古代对日常生活的简单记录，到现代高度抽象的数学理论，数学的发展不仅反映了人类对自然和世界的认知，也为科技进步和社会发展提供了强大的工具。

### 第2章：心智与数学

#### 2.1 心智的定义与功能

心智（Mind）是指人类大脑进行思考、感知、理解和记忆的能力。心智不仅仅局限于智力活动，还包括情感、意志和动机等方面。在心理学中，心智通常被视为一种复杂的认知过程，涉及信息处理、决策制定和问题解决等多个方面。

心智的定义和功能可以从不同角度进行理解。首先，心智是一种信息处理系统，它接收外部刺激，通过感知、注意和记忆等过程，对信息进行加工和处理。其次，心智具有目的性，它不仅能够对信息进行识别和分类，还能够根据目标和需求进行问题解决和决策制定。此外，心智还表现出适应性，能够根据环境和情境的变化，调整自己的行为和认知策略。

心智的功能可以分为以下几个主要方面：

1. **感知与注意**：心智能够感知外部世界的刺激，并通过注意力机制选择性地关注某些信息。这种能力使得人类能够有效地与环境互动，获取必要的知识和信息。

2. **记忆与认知**：心智能够将信息存储在记忆中，并通过认知过程对信息进行加工和组织。记忆分为短期记忆和长期记忆，前者负责临时存储信息，后者则负责长期储存和回忆信息。

3. **问题解决与决策**：心智能够识别问题，制定解决方案，并在复杂情境中进行决策。问题解决和决策制定过程通常涉及信息分析、逻辑推理和创造力等认知活动。

4. **情感与意志**：心智不仅涉及理性思考，还包括情感和意志等非理性因素。情感影响人的情绪和行为，意志则涉及个体对目标坚持和行动力的控制。

通过上述功能，心智使得人类能够适应复杂多变的环境，进行有效的学习和生活。因此，心智在人类认知和行为中扮演着核心角色。

### 第3章：数学的核心概念

#### 3.1 数的概念

数（Number）是数学中最基本的概念之一，它用来表示和描述物体或对象的集合。数的概念贯穿于数学的各个分支，如算术、代数、几何和微积分等。数的分类和定义不仅反映了数学的抽象性，也展示了数学的广泛应用。

数的分类可以按照不同的标准进行。首先，按照数的性质，数可以分为自然数、整数、有理数、无理数和复数。自然数是正整数，用于计数和排序。整数包括自然数和它们的相反数，即正整数、零和负整数。有理数是可以表示为两个整数之比的数，包括整数和分数。无理数是不能表示为两个整数之比的数，如π和√2。复数是包含实部和虚部的数，形式为a + bi，其中i是虚数单位，满足i² = -1。

数的定义和性质是数学理论的基础。自然数的定义通常基于集合论，即自然数是满足某些条件的对象集合。整数的定义是基于自然数，通过加入负数得到。有理数的定义是通过分数的形式表示。无理数的定义是基于它们不能表示为分数，必须通过构造或证明来确定。复数的定义则结合了实数和虚数，形成了更为广泛的数系。

数在数学中的应用非常广泛。在算术中，数用于进行各种基本运算，如加法、减法、乘法和除法。在代数中，数用于建立方程和不等式，解决各种实际问题。在几何学中，数用于测量长度、面积和体积。在微积分中，数用于描述变化和极限。此外，数在物理学、工程学、经济学等领域的应用也至关重要。

总的来说，数是数学的核心概念，它不仅反映了数学的抽象性和逻辑性，也为数学在其他领域的应用提供了基础。通过理解数的概念和性质，我们可以更好地理解和应用数学。

#### 3.2 形状的概念

形状（Shape）是几何学中的一个基本概念，用来描述物体的外部轮廓和结构。形状不仅具有视觉上的美感，也在数学和科学中有着重要的应用。形状的分类和定义多种多样，常见的有平面形状和立体形状。

平面形状是指存在于二维平面上的几何图形，如点、线、圆、多边形等。点是没有大小的基本几何元素，线是无限延伸的一维图形，圆是平面上所有到定点距离相等的点的集合。多边形是由直线段连接形成的封闭图形，如三角形、正方形、五边形等。

立体形状是指存在于三维空间中的几何图形，如立方体、球体、圆锥体、圆柱体等。立方体是六个正方形面构成的立体图形，球体是所有到定点距离相等的点的集合，圆锥体是由直线和旋转面形成的立体图形，圆柱体是两个平行圆面和一个侧面围成的立体图形。

形状的定义和分类不仅反映了几何学的抽象性，也展示了几何学的广泛应用。在建筑学中，形状用于设计和建造各种建筑结构。在物理学中，形状用于描述物体的运动和变形。在工程学中，形状用于优化设计，提高效率和质量。在计算机科学中，形状用于图形处理和图像识别。

总的来说，形状是几何学中的核心概念，它不仅反映了几何学的抽象性和逻辑性，也为各个领域提供了重要的数学工具。通过理解和研究形状，我们可以更好地理解和应用几何学。

#### 3.3 空间的概念

空间（Space）是几何学和物理学中的一个基本概念，用来描述物体存在的三维环境。空间的概念不仅涵盖了物理空间的几何性质，还包括时间和动态的维度，形成了更为复杂的空间结构。

在几何学中，空间通常指的是三维欧几里得空间，即我们日常生活中所体验到的空间。欧几里得空间中的点、线、面和体都是空间的基本元素。点是没有大小的几何元素，线是无限延伸的一维图形，面是无限扩展的二维图形，体是存在于三维空间中的三维图形。欧几里得空间的基本性质包括直角、平行线、角度和距离等，这些性质构成了几何学的基础。

物理学中的空间概念则更为广泛和复杂。空间不仅包括几何性质，还包括物理性质，如质量、能量、力和运动。例如，在牛顿力学中，空间被描述为均匀和绝对的参考框架，物体的运动遵循一定的规律。在广义相对论中，空间和时间被视为一个整体，称为时空，物体的运动受到引力的影响，时空的几何性质可以改变。

空间的概念不仅在物理和几何中有着重要应用，还在其他科学领域如计算机科学、生物学和社会学中发挥着重要作用。在计算机科学中，空间用于描述数据结构和算法的复杂性。在生物学中，空间用于研究生物体的形态和分布。在社会学中，空间用于研究人类的行为和社会结构。

总的来说，空间是几何学和物理学中的核心概念，它不仅反映了物质的几何性质和物理规律，也为各个科学领域提供了重要的理论基础。通过理解和研究空间，我们可以更好地理解和应用几何学和物理学。

### 第4章：数学原理与逻辑

#### 4.1 基本逻辑原理

逻辑（Logic）是数学和哲学中的基础学科，它研究推理和论证的有效性。逻辑的基本原理包括命题逻辑、谓词逻辑和证明论。这些原理构成了数学证明和推理的核心框架。

**命题逻辑**是研究命题之间关系和推理规则的逻辑系统。命题是一个具有真假二值判断的陈述句。命题逻辑中的基本概念包括命题、逻辑运算符（如“与”、“或”、“非”）和逻辑等价式。例如，“P 且 Q”表示两个命题P和Q同时为真，“P 或 Q”表示至少有一个命题为真，“非P”表示命题P为假。

**谓词逻辑**是研究带有变量和量词的命题的逻辑系统。谓词逻辑中的命题不仅包含个体，还包括属性和关系。谓词逻辑的基本概念包括谓词、个体、量词（全称量词和存在量词）和推理规则。例如，全称命题“所有的学生都学习数学”可以用谓词逻辑表示为“∀x，学生(x) → 学习数学(x)”，存在命题“有的学生喜欢数学”可以用谓词逻辑表示为“∃x，学生(x) ∧ 喜欢数学(x)”。

**证明论**是研究证明方法和证明结构的逻辑分支。证明论的基本任务是通过逻辑规则和假设，从已知的前提出发，推导出新的结论。证明论中的基本概念包括公理系统、推理规则和证明方法。例如，在自然演绎系统中，可以通过假设、演绎和归纳等方法构建证明。

这些逻辑原理在数学中具有重要作用。命题逻辑提供了对命题形式和推理规则的系统化描述，使得数学推理更加严谨和可靠。谓词逻辑扩展了命题逻辑，使得数学能够处理更复杂的对象和关系。证明论则为数学证明提供了理论框架，保证了数学结论的严格性和有效性。

总的来说，逻辑原理是数学推理和证明的基础，它不仅提高了数学的严谨性，也为数学在其他领域的应用提供了重要的工具。

#### 4.2 数学原理的构建

数学原理的构建是一个复杂而系统的过程，涉及多个层次和步骤。数学原理不仅包括基本的定义和公理，还包括推理规则和证明方法。以下逐步介绍数学原理的构建过程：

**第一步：基本定义**

数学原理的构建首先从基本定义开始。基本定义是对数学对象和概念的最基本描述。例如，在几何学中，定义点、线、面等基本元素。在代数中，定义数、函数、集合等基本概念。这些定义构成了数学理论的基础。

**第二步：公理系统**

在基本定义的基础上，数学原理的构建进一步依赖于公理系统。公理是一系列被假设为真的基本命题，它们不是通过证明得到的，而是作为数学理论的前提条件。例如，欧几里得几何的五个公设构成了该几何学的理论基础。非欧几何则通过改变这些公设，发展出了不同的几何体系。

**第三步：推理规则**

公理系统确定了数学理论的基本框架，而推理规则则用于从已知的前提推导出新的结论。推理规则包括演绎推理、归纳推理和类比推理等。演绎推理是从一般到特殊的推理过程，例如，从公理推导出定理。归纳推理则是从特殊到一般的推理过程，例如，通过观察多个具体实例，推断出一般规律。类比推理是通过比较不同情境之间的相似性，推导出新结论。

**第四步：证明方法**

在构建数学原理的过程中，证明方法是必不可少的。证明方法包括直接证明、反证法、数学归纳法等。直接证明是通过一系列逻辑步骤，从已知前提推导出结论。反证法是假设结论不成立，推导出矛盾，从而证明结论成立。数学归纳法是一种特殊的证明方法，用于证明与自然数相关的命题。

**第五步：理论体系**

通过上述步骤，数学原理逐渐构建出一个完整的理论体系。这个体系不仅包括了基本的定义和公理，还包括了各种定理和公式。例如，在微积分中，通过极限、导数和积分等基本概念，构建了一个完整的理论体系，用于研究函数的性质和变化。

总的来说，数学原理的构建是一个逐步深入、系统化构建的过程。从基本定义到公理系统，再到推理规则和证明方法，每个步骤都是构建数学理论不可或缺的一部分。通过这种系统化的方法，数学能够精确地描述和解决各种问题，为科学和技术的进步提供了强大的工具。

### 第二部分：数学在自然科学中的应用

#### 第5章：数学在物理学中的应用

物理学是研究物质和能量的科学，而数学作为自然语言，为物理学的理论构建、模型建立和实验数据分析提供了强有力的工具。数学在物理学中的应用贯穿于各个分支，包括经典力学、量子力学和相对论等。

**5.1 物理学的数学基础**

物理学的数学基础主要包括微积分、线性代数、微分方程和复变函数等。微积分是研究函数变化率和积分过程的基本工具，广泛应用于描述速度、加速度、力、能量等物理量。线性代数研究线性方程组、向量空间和矩阵理论，为处理多变量物理问题提供了基础。微分方程则用于描述物理系统中的动态变化，如波动方程、热传导方程等。复变函数在电磁学和量子力学中有着重要应用。

**5.2 数学在经典力学中的应用**

经典力学是物理学的基础之一，主要研究质点和刚体的运动规律。牛顿的运动定律是经典力学的核心，其中包含了力的概念和物体运动的数学描述。牛顿定律可以通过数学方程精确地描述质点的运动轨迹。此外，拉格朗日力学和哈密顿力学提供了另一种描述经典力学的数学框架，通过拉格朗日量和哈密顿量，可以处理更复杂的系统。

**5.3 数学在量子力学中的应用**

量子力学是研究微观粒子和量子系统的物理学分支。量子力学的数学描述主要基于波函数和薛定谔方程。波函数是一个复数函数，描述了量子系统的概率分布。薛定谔方程是一个非线性偏微分方程，它描述了波函数随时间的演化。量子力学的数学工具还包括算符、态叠加原理和量子测量的概率解释等。这些工具使得量子力学的预测具有高度精确性和独特性。

**5.4 数学在相对论中的应用**

相对论是描述高速运动和强引力场下物理现象的理论。狭义相对论由爱因斯坦提出，它引入了洛伦兹变换和质能方程，对时间和空间的相对性进行了深刻的阐述。广义相对论则将相对性原理扩展到引力场，通过弯曲时空的概念描述引力现象。广义相对论中的关键方程是爱因斯坦场方程，它将物质能量分布与时空曲率联系起来，提供了引力作用的新视角。

总的来说，数学在物理学中的应用极为广泛且深远。它不仅为物理学理论提供了坚实的数学基础，也使得复杂的物理现象可以通过数学模型进行精确描述和预测。数学与物理学的结合，推动了人类对自然世界的深入理解和科技的发展。

#### 第6章：数学在生物学中的应用

数学在生物学中的应用已经变得不可或缺，它为生物学的研究提供了强有力的工具，帮助科学家更好地理解和解释生物现象。数学模型和数学方法在生物学各个领域都有广泛应用，包括生态学、遗传学和分子生物学。

**6.1 生物学中的数学模型**

数学模型是生物学研究中的重要工具，它通过数学方程和公式来描述生物系统的动态行为和相互作用。在生态学中，数学模型用于研究种群动态、种群间的竞争和食物链。经典的Lotka-Volterra方程描述了捕食者和猎物种群的数量变化，通过数学方法可以预测种群数量的波动和生态平衡。在遗传学中，数学模型用于分析基因频率、遗传变异和进化过程。哈迪-温伯格定律就是一个重要的遗传学数学模型，它描述了在没有自然选择、突变、迁移和随机漂变的情况下，基因频率和等位基因频率的稳定分布。

**6.2 数学在生态学中的应用**

生态学是研究生物与其环境之间相互作用的一门学科，数学在生态学中的应用非常广泛。数学模型可以用来研究生态系统的稳定性、物种多样性、生态网络结构和生态过程。例如，使用稳定性分析可以预测生态系统在受到外界干扰时的响应和恢复能力。种间竞争模型如Holling的两种捕食者-猎物模型，通过数学方法描述了捕食者和猎物种群数量的动态变化。此外，生态网络模型如食物网和生态位模型，通过数学方法可以揭示生态系统的结构和功能特性。

**6.3 数学在遗传学中的应用**

遗传学是研究遗传现象和基因结构的学科，数学在遗传学中的应用极大地促进了基因图谱的构建和基因功能的理解。通过概率论和统计方法，科学家可以分析基因的分布、遗传标记和连锁不平衡。例如，连锁分析是一种常用的遗传分析方法，通过比较不同个体之间基因序列的相似性，可以推断基因在染色体上的位置。此外，基因表达数据分析中常用的方法如微阵列分析和RNA测序，都需要使用复杂的数学模型和算法来处理大量的数据，从而识别出基因的表达模式和相关功能。

**6.4 数学在分子生物学中的应用**

分子生物学是研究生物大分子如DNA、RNA和蛋白质的结构和功能的一门学科，数学在分子生物学中的应用主要体现在基因组学和蛋白质组学中。基因组学是研究基因组的结构和功能的科学，通过数学模型和算法，科学家可以解析基因组序列，识别基因、转录因子结合位点和其他功能元件。例如，生物信息学中的序列比对算法如BLAST和隐马尔可夫模型HMM，用于识别基因和蛋白质家族。蛋白质组学则通过定量分析蛋白质的表达水平，揭示蛋白质的功能和相互作用网络。数学方法如主成分分析（PCA）、聚类分析和网络分析，被广泛应用于蛋白质组数据的分析和解释。

总的来说，数学在生物学中的应用为生物学研究提供了强大的工具和方法，使得科学家能够更深入地理解生物系统的复杂性和多样性。随着数学方法和计算技术的不断发展，数学在生物学中的应用将会更加广泛和深入，为生物科学的进步和生物技术的创新提供支持。

#### 第7章：数学在化学中的应用

数学在化学中的应用是化学理论和实验研究中不可或缺的一部分，它为化学家提供了精确描述和分析化学现象的工具。化学中涉及的许多概念和过程，如分子结构、反应动力学和热力学，都需要数学模型和计算方法来进行定量描述。

**7.1 化学中的数学原理**

化学中的数学原理主要包括概率论、统计热力学、线性代数和微分方程等。概率论在化学中的应用主要体现在量子化学的计算中，用于求解分子的电子结构。统计热力学则用于描述系统的热力学性质，如温度、压力和能量分布。线性代数在量子化学和分子轨道理论中有着重要应用，用于处理电子的线性组合。微分方程则用于描述化学反应的动力学，如反应速率方程和扩散方程。

**7.2 数学在分子生物学中的应用**

分子生物学是研究生物大分子的结构、功能和相互作用的学科。数学在分子生物学中的应用主要体现在基因组学和蛋白质组学中。基因组学通过数学模型和算法分析基因组序列，识别基因、转录因子结合位点和其他功能元件。例如，序列比对算法如BLAST和隐马尔可夫模型HMM用于识别基因和蛋白质家族。蛋白质组学通过定量分析蛋白质的表达水平，揭示蛋白质的功能和相互作用网络。数学方法如主成分分析（PCA）、聚类分析和网络分析，被广泛应用于蛋白质组数据的分析和解释。

数学在药物设计中的应用是化学与生物学交叉领域的亮点。药物设计旨在发现和开发新的药物分子，用于治疗疾病。数学模型和计算方法在药物设计中的多个环节发挥着重要作用。首先，分子对接（molecular docking）是一种基于数学模型的计算方法，用于预测药物分子与靶点蛋白的结合模式。通过分子对接，科学家可以筛选出具有潜在活性的药物分子。其次，量子化学计算如密度泛函理论（DFT）和分子力学（MM）方法，用于优化药物分子的结构，提高其生物活性。此外，机器学习和人工智能技术也被广泛应用于药物设计，用于预测药物分子的性质和活性。

### 第8章：数学在计算机科学中的应用

数学在计算机科学中的应用是不可或缺的，它为算法设计、人工智能和计算机图形学等提供了理论基础和工具。以下从算法设计、人工智能和计算机图形学三个方面详细探讨数学在计算机科学中的应用。

#### 8.1 计算机科学中的数学基础

计算机科学中的数学基础包括离散数学、概率论、线性代数、微积分和图论等。离散数学是研究离散结构的数学分支，如集合、关系、函数和图等，为算法分析和逻辑推理提供了基础。概率论和统计方法是计算机科学中用于分析和理解随机现象的重要工具，尤其在人工智能和数据分析领域有广泛应用。线性代数和微积分则用于处理线性方程组、优化问题和微分方程等，在机器学习和计算机图形学中有重要应用。图论是研究图及其性质的数学分支，广泛应用于网络结构分析和路径优化问题。

#### 8.2 数学在算法设计中的应用

数学在算法设计中的应用主要体现在算法的分析和设计过程中。算法分析是评估算法性能的重要方法，包括时间复杂度和空间复杂度分析。时间复杂度描述算法执行所需的时间，通常用大O符号表示，如O(n)、O(n²)等。空间复杂度描述算法执行所需的空间，用于评估算法的存储需求。算法设计中的数学方法包括分治算法、动态规划、贪心算法和回溯算法等。例如，分治算法通过递归将问题分解为更小的子问题，然后合并子问题的解，适用于许多经典问题如归并排序和快速排序。动态规划通过保存子问题的解，避免重复计算，用于解决最优子结构问题，如背包问题和最短路径问题。贪心算法通过每一步选择当前最优解，逐步构建全局最优解，适用于许多优化问题，如Prim算法和Kruskal算法。回溯算法通过试探和回溯，逐步搜索所有可能的解，适用于组合优化问题，如八皇后问题和旅行商问题。

**示例：快速排序算法**

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    return quick_sort(left) + middle + quick_sort(right)

# 示例使用
arr = [3, 6, 8, 10, 1, 2, 1]
sorted_arr = quick_sort(arr)
print(sorted_arr)
```

**Black-Scholes期权定价模型：**

```latex
C(S_0, t) = S_0N(d_1) - Ke^{-r(T-t)}N(d_2)
```

其中，C是期权价格，S_0是当前股票价格，K是执行价格，r是无风险利率，T是到期时间，N是累积标准正态分布函数。这个模型使得期权交易者和投资者能够根据市场参数计算期权的合理价格，从而进行有效的投资决策。

**示例：t检验在心理实验中的应用**

假设一位心理学家想要研究不同类型的音乐对人的情绪的影响。通过t检验，可以比较不同音乐类型下情绪反应的差异。

```plaintext
t = (M1 - M2) / (s1 / √n1 - s2 / √n2)
```

其中，t是t统计量，M1和M2是两组实验的平均值，s1和s2是两组实验的标准差，n1和n2是两组实验的样本大小。通过t检验，心理学家可以判断两组实验结果的差异是否显著。

  $P(A \cup B) = P(A) + P(B) - P(A \cap B)$（概率加法公式）
  $P(A | B) = \frac{P(A \cap B)}{P(B)}$（条件概率）
  $P(A \cup B) = 1 - P(A' \cap B')$（概率补集公式）

- **线性代数公式：**
  $Ax = b$（线性方程组）
  $det(A) = 0$（行列式为零，矩阵不可逆）
  $A^{-1} = \frac{1}{det(A)} \text{adj}(A)$（逆矩阵）
