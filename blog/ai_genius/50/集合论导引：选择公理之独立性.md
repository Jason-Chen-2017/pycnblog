### 第一部分：集合论基础

#### 1.1 集合论的基本概念

集合论是现代数学的基石，它为数学提供了一个严谨的逻辑框架。集合论的基本概念包括集合的定义与性质、集合的表示方法、集合的基数与运算以及特殊集合。

**集合的定义与性质**

集合是一个由确定的元素组成的整体。这些元素可以是任意的对象，包括数字、字母、图形等。集合中的元素通常用大写字母表示，如 A、B、C 等。

集合的性质包括：

1. **确定性**：集合中的元素必须是确定的，不能含糊不清。
2. **互异性**：集合中的元素必须是互不相同的，不能重复。
3. **无序性**：集合中的元素没有固定的顺序。

**集合的表示方法**

集合的表示方法主要有两种：列举法和描述法。

- **列举法**：将集合中的所有元素一一列举出来，用花括号{}括起来。例如，A = {1, 2, 3} 表示集合 A 包含元素 1、2、3。
- **描述法**：用一句或几句话来描述集合中的元素所满足的条件。例如，B = {x | x 是偶数} 表示集合 B 包含所有偶数。

**集合的运算**

集合的运算包括并集、交集、差集、补集等。

- **并集**：A ∪ B 表示集合 A 和集合 B 的并集，它包含所有属于 A 或属于 B 的元素。例如，A = {1, 2, 3}，B = {2, 3, 4}，则 A ∪ B = {1, 2, 3, 4}。
- **交集**：A ∩ B 表示集合 A 和集合 B 的交集，它包含所有既属于 A 又属于 B 的元素。例如，A = {1, 2, 3}，B = {2, 3, 4}，则 A ∩ B = {2, 3}。
- **差集**：A - B 表示集合 A 和集合 B 的差集，它包含所有属于 A 但不属于 B 的元素。例如，A = {1, 2, 3}，B = {2, 3, 4}，则 A - B = {1}。
- **补集**：A' 表示集合 A 的补集，它包含所有不属于 A 的元素。例如，A = {1, 2, 3}，则 A' = {4, 5, 6, ...}。

**子集与真子集**

- **子集**：如果集合 A 的所有元素都属于集合 B，则称 A 是 B 的子集，记作 A ⊆ B。
- **真子集**：如果集合 A 是集合 B 的子集，但 A 不等于 B，则称 A 是 B 的真子集，记作 A ⊂ B。

#### 1.2 集合的表示方法

集合的表示方法主要有列举法和描述法。

- **列举法**：将集合中的所有元素一一列举出来，用花括号{}括起来。例如，A = {1, 2, 3} 表示集合 A 包含元素 1、2、3。

- **描述法**：用一句或几句话来描述集合中的元素所满足的条件。例如，B = {x | x 是偶数} 表示集合 B 包含所有偶数。

#### 1.3 集合的基数与运算

集合的基数是指集合中元素的数量。基数是一个非负整数，它用来表示集合的大小。

- **基数**：对于有限集合 A，集合 A 的基数记作 |A|，表示集合 A 中元素的数量。对于无限集合 A，集合 A 的基数记作 |A|，表示集合 A 中元素的数量，或者是不能计数无穷大的一个无穷数。

集合的基数运算主要包括并集、交集和差集的基数运算。

- **并集的基数运算**：\[ |A \cup B| = |A| + |B| - |A \cap B| \]
- **交集的基数运算**：\[ |A \cap B| \leq \min(|A|, |B|) \]
- **差集的基数运算**：\[ |A - B| = |A| - |A \cap B| \]

#### 1.4 特殊集合

- **真空集**：真空集（也称为空集）是一个不包含任何元素的集合，记作 ∅。
- **无穷集**：无穷集是一个包含无穷多个元素的集合，例如自然数集合 N、整数集合 Z、有理数集合 Q、实数集合 R 等。
- **有限集合**：有限集合是一个包含有限个元素的集合，例如集合 {1, 2, 3}。
- **无限集合**：无限集合是一个包含无穷多个元素的集合，例如实数集合 R。

#### 1.5 集合的划分与覆盖

- **划分**：一个划分是指将一个集合分成若干个子集，使得这些子集之间互不相交，并且它们的并集等于原集合。
- **覆盖**：一个覆盖是指存在一个集合，它的所有子集都是另一个集合的子集。
