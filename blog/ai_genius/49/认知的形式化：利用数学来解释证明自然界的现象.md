#### 项目实战

**案例：使用Python实现Sigmoid函数**

```python
import math

def sigmoid(x):
    return 1 / (1 + math.exp(-x))

# 测试Sigmoid函数
x = 2
print(sigmoid(x))
```

**代码解读与分析：**

- 导入math模块以使用exp函数。
- 定义sigmoid函数，接受一个输入x。
- 使用数学公式 $f(x) = \frac{1}{1 + e^{-x}}$ 计算输出。
- 测试函数，输出当x为2时的结果。
