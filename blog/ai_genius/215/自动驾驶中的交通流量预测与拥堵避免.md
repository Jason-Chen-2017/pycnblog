### 第2章: 交通流量预测的基础知识

交通流量预测是智能交通系统（Intelligent Transportation Systems, ITS）中的关键组成部分，对于提升交通管理效率和减少拥堵具有重要意义。本章将深入探讨交通流量预测的基础知识，包括交通流量的定义与特征、交通流量预测的数据来源以及常见交通流量预测方法。

#### 2.1 交通流量的定义与特征

交通流量是指单位时间内通过某一交通路段的车辆数量。具体来说，交通流量可以用以下参数进行描述：

- **流量密度**：单位路段长度内车辆的数量。
- **速度**：车辆在路段上的平均行驶速度。
- **流量**：单位时间内通过某一路段的车流量，通常以辆/小时为单位。

交通流量的特征包括：

1. **时空变化性**：交通流量随时间和空间位置的不同而变化，例如高峰时段和周末的流量可能会有显著差异。
2. **非线性**：交通流量与道路容量之间存在非线性关系，当流量低于某一阈值时，增加车辆数会导致速度增加；当流量超过该阈值时，速度反而会下降。
3. **突发性**：突发事件（如交通事故、施工等）会导致交通流量突然变化。

#### 2.2 交通流量预测的数据来源

交通流量预测依赖于多种数据来源，主要包括：

1. **历史流量数据**：通过交通监测设备（如交通摄像头、传感器等）收集的历史数据，可用于训练预测模型。
2. **实时流量数据**：通过实时监测设备获取的当前交通流量数据，可用于动态调整预测模型。
3. **社会经济数据**：如人口、经济、天气等社会经济因素，这些因素会影响交通流量。
4. **交通事件数据**：交通事件（如交通事故、施工等）的信息，可用于预测交通状况的变化。

#### 2.3 常见交通流量预测方法概述

交通流量预测方法可以分为以下几类：

1. **统计方法**：
   - **时间序列分析**：基于历史流量数据，使用统计学方法（如自回归模型、移动平均模型等）预测未来的交通流量。
   - **回归分析**：通过建立流量与其他影响因素（如天气、时间等）的回归模型，预测交通流量。

2. **机器学习方法**：
   - **监督学习**：使用标记的数据集训练预测模型，如线性回归、决策树、支持向量机等。
   - **无监督学习**：不需要标记数据，如聚类分析、主成分分析等。

3. **深度学习方法**：
   - **神经网络**：使用多层感知器（MLP）、卷积神经网络（CNN）等神经网络结构进行流量预测。
   - **递归神经网络（RNN）**：如长短时记忆网络（LSTM）、门控循环单元（GRU）等，适用于时间序列数据的预测。

4. **多模型融合方法**：
   - 将多种方法（如统计方法、机器学习方法等）进行融合，以提高预测精度。

通过以上方法，交通流量预测能够更好地应对复杂的交通状况，为交通管理和自动驾驶提供有力支持。

### 第3章: 拥堵避免的理论与方法

拥堵是现代城市交通中的一大挑战，它不仅降低了道路通行效率，还增加了交通事故发生的风险，对城市经济和社会发展产生了负面影响。因此，研究有效的拥堵避免理论和方法具有重要意义。本章将介绍拥堵的定义、分类、策略及其实际技术实现。

#### 3.1 拥堵的定义与分类

拥堵是指交通流在道路上因为某些原因而减缓或停止，导致车辆排队等待的现象。根据拥堵发生的机理和特点，可以将其分为以下几类：

1. **静态拥堵**：车辆在道路上因交通事件（如事故、施工等）或道路容量不足而停滞，车辆速度降低。
2. **动态拥堵**：车辆因预期到的交通拥堵而提前减速，导致前方车辆减速，从而引发连锁反应，形成拥堵。
3. **混合拥堵**：静态拥堵和动态拥堵的混合形式，通常由于多种因素（如交通事件、道路施工、恶劣天气等）共同作用导致。

#### 3.2 拥堵避免的策略与措施

为了避免拥堵，可以从以下几个方面采取策略和措施：

1. **交通需求管理**：
   - **交通需求预测**：通过预测未来的交通流量，提前采取措施，如调整工作时间、鼓励错峰出行等。
   - **出行引导**：利用交通信息提供系统，向驾驶员提供最优出行路线和出行时间，减少高峰期的交通压力。

2. **交通供给管理**：
   - **道路容量扩充**：通过扩建道路、建设高架桥、隧道等基础设施，增加道路容量。
   - **交通信号优化**：通过智能交通信号控制系统，动态调整交通信号灯的配时，优化交通流量。

3. **交通组织与管理**：
   - **动态交通分配**：通过智能交通系统，实时监测交通状况，动态调整交通流向和流量。
   - **事故管理**：快速响应交通事件，如交通事故、施工等，通过交通监控和通信系统及时通知驾驶员，引导车辆绕行。

4. **公共交通优先**：
   - **公交优先**：通过设置公交专用道、优化公交路线和时间表，提高公共交通的吸引力。
   - **多模式交通**：鼓励使用非机动交通方式，如自行车、步行等，减少私家车出行。

#### 3.3 拥堵避免的技术实现

拥堵避免的技术实现主要包括以下几个方面：

1. **智能交通信号控制**：
   - **自适应交通信号控制系统**：根据实时交通流量和交通事件信息，动态调整交通信号灯的配时，优化交通流量。
   - **协同信号控制**：通过交通监控和通信系统，实现相邻路口的信号灯协同控制，减少交叉口的停车次数和延误。

2. **动态路径规划**：
   - **实时交通信息获取**：利用GPS、交通摄像头、传感器等设备，实时获取交通流量、速度、事故等信息。
   - **动态路径规划算法**：基于实时交通信息，使用A*算法、Dijkstra算法等，计算最优路径，引导车辆避开拥堵路段。

3. **交通事件预警与响应**：
   - **事件检测**：通过视频分析、传感器监测等技术，实时检测交通事件（如事故、施工等）。
   - **事件预警**：通过交通监控和通信系统，及时向驾驶员和交通管理部门发出预警信息。
   - **应急响应**：根据交通事件类型和位置，采取相应的应急措施，如引导车辆绕行、调整交通信号灯等。

通过以上策略与措施，以及智能交通系统的技术支持，可以有效避免拥堵，提高交通通行效率，改善城市交通状况。

### 第4章: 交通流量预测的核心算法原理

交通流量预测是智能交通系统（ITS）中的关键技术之一，它的准确性直接影响到交通管理、路径规划和车辆调度等领域的效率和安全性。本章将深入探讨交通流量预测的核心算法原理，主要包括时间序列分析算法、机器学习算法以及深度学习算法。

#### 4.1 时间序列分析算法

时间序列分析是一种基于历史数据进行分析和预测的方法，适用于交通流量这种具有时间依赖性的数据。常见的时间序列分析算法包括以下几种：

1. **自回归模型（AR）**：
   自回归模型通过历史数据中的滞后项来预测未来的流量。其基本形式为：
   $$ X_t = c + \phi_1 X_{t-1} + \phi_2 X_{t-2} + \ldots + \phi_p X_{t-p} + \epsilon_t $$
   其中，$X_t$ 表示时间 $t$ 的交通流量，$c$ 为常数项，$\phi_1, \phi_2, \ldots, \phi_p$ 为自回归系数，$\epsilon_t$ 为误差项。

2. **移动平均模型（MA）**：
   移动平均模型通过历史数据的移动平均值来预测未来的流量。其基本形式为：
   $$ X_t = c + \theta_1 X_{t-1} + \theta_2 X_{t-2} + \ldots + \theta_q X_{t-q} + \epsilon_t $$
   其中，$c$ 为常数项，$\theta_1, \theta_2, \ldots, \theta_q$ 为移动平均系数，$\epsilon_t$ 为误差项。

3. **自回归移动平均模型（ARMA）**：
   ARMA模型结合了自回归和移动平均模型的特点，其基本形式为：
   $$ X_t = c + \phi_1 X_{t-1} + \phi_2 X_{t-2} + \ldots + \phi_p X_{t-p} + \theta_1 X_{t-1} + \theta_2 X_{t-2} + \ldots + \theta_q X_{t-q} + \epsilon_t $$
   其中，$c$ 为常数项，$\phi_1, \phi_2, \ldots, \phi_p$ 和 $\theta_1, \theta_2, \ldots, \theta_q$ 分别为自回归和移动平均系数，$\epsilon_t$ 为误差项。

4. **自回归积分移动平均模型（ARIMA）**：
   ARIMA模型是ARMA模型的扩展，适用于非平稳时间序列。其基本形式为：
   $$ X_t = c + \phi_1 X_{t-1} + \phi_2 X_{t-2} + \ldots + \phi_p X_{t-p} + \theta_1 (1 - \Phi_1 X_{t-1} - \ldots - \Phi_q X_{t-q}) + \epsilon_t $$
   其中，$c$ 为常数项，$\phi_1, \phi_2, \ldots, \phi_p$ 和 $\theta_1, \theta_2, \ldots, \theta_q$ 分别为自回归和移动平均系数，$\Phi_1, \Phi_2, \ldots, \Phi_q$ 为差分算子。

#### 4.2 机器学习算法

机器学习算法通过学习和分析历史数据来预测未来的交通流量。与时间序列分析算法相比，机器学习算法能够捕捉到数据中的非线性关系和复杂模式。以下是几种常见的机器学习算法：

1. **线性回归**：
   线性回归是一种最简单的机器学习算法，通过建立线性模型来预测交通流量。其基本形式为：
   $$ Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \ldots + \beta_n X_n $$
   其中，$Y$ 表示交通流量，$X_1, X_2, \ldots, X_n$ 表示影响交通流量的特征，$\beta_0, \beta_1, \beta_2, \ldots, \beta_n$ 为回归系数。

2. **支持向量机（SVM）**：
   支持向量机是一种强大的分类和回归算法，通过找到最佳超平面来最大化分类间隔。其基本形式为：
   $$ w \cdot x + b = 0 $$
   其中，$w$ 表示权重向量，$x$ 表示特征向量，$b$ 表示偏置项。

3. **决策树**：
   决策树通过一系列二分类节点来建立决策规则，适用于分类和回归问题。其基本形式为：
   $$ \text{if } x_i \leq v_i \text{ then } \text{left branch} $$
   $$ \text{else } \text{right branch} $$
   其中，$x_i$ 表示特征值，$v_i$ 表示阈值。

4. **随机森林**：
   随机森林是一种基于决策树的集成学习方法，通过构建多个决策树并取平均来提高预测性能。其基本形式为：
   $$ \hat{y} = \sum_{i=1}^{n} w_i \cdot h_i(x) $$
   其中，$w_i$ 表示第 $i$ 棵决策树的权重，$h_i(x)$ 表示第 $i$ 棵决策树对特征 $x$ 的分类结果。

#### 4.3 深度学习算法

深度学习算法通过多层神经网络结构来捕捉复杂的非线性关系。以下是一些常见的深度学习算法：

1. **多层感知器（MLP）**：
   多层感知器是一种前馈神经网络，通过多个隐层来提取特征并预测交通流量。其基本形式为：
   $$ a_{ij} = \sigma(\sum_{k=1}^{n} w_{ik} \cdot x_k + b_j) $$
   其中，$a_{ij}$ 表示隐层第 $j$ 个神经元输出，$\sigma$ 表示激活函数，$w_{ik}$ 和 $b_j$ 分别为权重和偏置项。

2. **卷积神经网络（CNN）**：
   卷积神经网络通过卷积操作来捕捉空间特征，适用于交通流量图像数据的处理。其基本形式为：
   $$ h_{ij} = \sum_{k=1}^{c} w_{ik} \cdot a_{kj-1} + b_j $$
   其中，$h_{ij}$ 表示卷积层第 $j$ 个神经元输出，$a_{kj-1}$ 表示前一卷积层的特征图，$w_{ik}$ 和 $b_j$ 分别为卷积核和偏置项。

3. **递归神经网络（RNN）**：
   递归神经网络通过记忆状态来处理序列数据，适用于时间序列交通流量预测。其基本形式为：
   $$ h_t = \sigma(W_h \cdot [h_{t-1}, x_t] + b_h) $$
   其中，$h_t$ 表示隐状态，$x_t$ 表示输入序列，$W_h$ 和 $b_h$ 分别为权重和偏置项，$\sigma$ 表示激活函数。

4. **长短时记忆网络（LSTM）**：
   长短时记忆网络是RNN的一种改进，通过引入门控机制来解决长期依赖问题。其基本形式为：
   $$ i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i) $$
   $$ f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f) $$
   $$ o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o) $$
   $$ c_t = f_t \odot c_{t-1} + i_t \odot \sigma(W_c \cdot [h_{t-1}, x_t] + b_c) $$
   $$ h_t = o_t \odot \sigma(c_t) $$
   其中，$i_t, f_t, o_t, c_t, h_t$ 分别为输入门、遗忘门、输出门、细胞状态和隐状态，$\odot$ 表示元素乘法，$\sigma$ 表示激活函数，$W_i, W_f, W_o, W_c$ 和 $b_i, b_f, b_o, b_c$ 分别为权重和偏置项。

通过以上时间序列分析算法、机器学习算法和深度学习算法，可以构建有效的交通流量预测模型，为智能交通系统提供有力支持。接下来，我们将在下一章讨论拥堵避免的算法原理。

### 第5章: 拥堵避免的算法原理讲解

拥堵避免是智能交通系统（ITS）中的关键技术之一，其核心目的是通过优化交通流和资源分配来减少交通拥堵，提高道路通行效率和安全性。本章将详细讲解三种主要的拥堵避免算法原理：最优路径规划算法、交通信号控制算法和拥堵检测与预警算法。

#### 5.1 最优路径规划算法

最优路径规划算法旨在为车辆提供一条从起点到终点的最优路径，以避免拥堵和减少行驶时间。以下是一些常见的方法：

1. **Dijkstra算法**：
   Dijkstra算法是一种基于贪心的最短路径算法，其基本思想是从源点开始，逐步扩展到未访问的节点，每次选择当前未访问节点中距离源点最近的节点。其伪代码如下：

```python
function Dijkstra(G, source):
    for each vertex v in G:
        dist[v] = INFINITY
        prev[v] = undefined
    dist[source] = 0
    for each edge (u, v) in G:
        dist[v] = weight(u, v)
    while G is not empty:
        select an unvisited vertex u with the smallest distance
        for each edge (u, v) in G:
            alt = dist[u] + weight(u, v)
            if alt < dist[v]:
                dist[v] = alt
                prev[v] = u
        remove u from G
    return dist[], prev[]
```

2. **A*算法**：

#### 6.1 交通流量预测数学模型

交通流量预测的数学模型主要分为时间序列模型和机器学习模型两大类。以下分别介绍这两种模型的基本概念和数学公式。

1. **时间序列模型**

时间序列模型基于历史交通流量数据，通过分析时间序列数据来预测未来的交通流量。常见的时间序列模型包括自回归模型（AR）、移动平均模型（MA）和自回归移动平均模型（ARMA）。

- **自回归模型（AR）**

自回归模型通过历史数据的滞后项来预测未来的交通流量。其数学公式如下：

$$
X_t = c + \phi_1 X_{t-1} + \phi_2 X_{t-2} + \ldots + \phi_p X_{t-p} + \epsilon_t
$$

其中，$X_t$ 表示时间 $t$ 的交通流量，$c$ 为常数项，$\phi_1, \phi_2, \ldots, \phi_p$ 为自回归系数，$\epsilon_t$ 为误差项。

- **移动平均模型（MA）**

移动平均模型通过历史数据的移动平均值来预测未来的交通流量。其数学公式如下：

$$
X_t = c + \theta_1 X_{t-1} + \theta_2 X_{t-2} + \ldots + \theta_q X_{t-q} + \epsilon_t
$$

其中，$c$ 为常数项，$\theta_1, \theta_2, \ldots, \theta_q$ 为移动平均系数，$\epsilon_t$ 为误差项。

- **自回归移动平均模型（ARMA）**

自回归移动平均模型结合了自回归和移动平均模型的特点，其数学公式如下：

$$
X_t = c + \phi_1 X_{t-1} + \phi_2 X_{t-2} + \ldots + \phi_p X_{t-p} + \theta_1 X_{t-1} + \theta_2 X_{t-2} + \ldots + \theta_q X_{t-q} + \epsilon_t
$$

其中，$c$ 为常数项，$\phi_1, \phi_2, \ldots, \phi_p$ 和 $\theta_1, \theta_2, \ldots, \theta_q$ 分别为自回归和移动平均系数，$\epsilon_t$ 为误差项。

2. **机器学习模型**

机器学习模型通过学习和分析历史数据来预测未来的交通流量。常见的机器学习模型包括线性回归、决策树、支持向量机和神经网络等。

- **线性回归**

线性回归模型通过建立线性关系来预测交通流量。其数学公式如下：

$$
Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \ldots + \beta_n X_n
$$

其中，$Y$ 表示交通流量，$X_1, X_2, \ldots, X_n$ 表示影响交通流量的特征，$\beta_0, \beta_1, \beta_2, \ldots, \beta_n$ 为回归系数。

- **支持向量机（SVM）**

支持向量机模型通过找到最佳超平面来预测交通流量。其数学公式如下：

$$

$$

其中，$w$ 表示权重向量，$x$ 表示特征向量，$b$ 表示偏置项。

#### 9.1 强化学习算法

强化学习（Reinforcement Learning, RL）是一种通过试错和反馈来学习如何实现最佳行为策略的机器学习技术。在交通流量预测中，强化学习算法通过学习历史交通数据和实时反馈，动态调整预测模型，从而提高预测准确性。

1. **基本原理**

强化学习算法包括四个主要组成部分：环境（Environment）、代理（Agent）、状态（State）和动作（Action）。代理通过观察环境状态，选择一个动作，并在执行动作后接收环境反馈，通过不断的迭代，学习最优策略。

2. **应用案例**

- **交通信号控制**：使用强化学习算法优化交通信号灯的切换时间，根据实时交通流量动态调整信号灯配时，提高道路通行效率。
- **路径规划**：自动驾驶车辆通过强化学习算法，根据实时交通状况动态调整行驶路径，避开拥堵路段。

3. **数学模型**

强化学习算法常用的数学模型包括马尔可夫决策过程（MDP）和策略梯度算法。MDP的基本形式为：

$$
\begin{align*}
\pi(a|s) &= P(A=a|S=s) \\
Q(s,a) &= R(s,a,s') + \gamma \max_{a'} Q(s',a')
\end{align*}
$$

其中，$\pi(a|s)$ 表示在状态 $s$ 下选择动作 $a$ 的概率，$Q(s,a)$ 表示在状态 $s$ 下执行动作 $a$ 的预期回报，$R(s,a,s')$ 表示立即回报，$\gamma$ 为折扣因子。

#### 9.2 聚类分析算法

聚类分析（Clustering Analysis）是一种无监督学习方法，用于将数据点按照其相似性进行分组。在交通流量预测中，聚类分析算法可以用于发现交通流量数据的规律，从而提高预测精度。

1. **基本原理**

聚类分析算法基于数据点的相似性度量，将数据点划分为若干个簇。常见的聚类算法包括K-means、层次聚类和DBSCAN等。

- **K-means**：K-means算法通过最小化簇内方差来划分数据点，每个簇的中心点代表该簇的数据特征。
- **层次聚类**：层次聚类算法通过自底向上的合并或自顶向下的分解，构建层次结构，形成多个簇。
- **DBSCAN**：DBSCAN（Density-Based Spatial Clustering of Applications with Noise）算法基于数据点的密度，将高密度区域划分为簇，同时处理噪声点。

2. **应用案例**

- **交通流量模式识别**：使用聚类分析算法识别不同时间段和路段的交通流量模式，为预测模型提供基础数据。
- **交通事件检测**：通过聚类分析算法发现交通流量数据中的异常点，用于检测交通事件。

3. **数学模型**

- **K-means**：

$$
\begin{align*}
\text{Minimize} \quad \sum_{i=1}^{n} \sum_{j=1}^{k} w_{ij} (x_i - \mu_j)^2 \\
\mu_j = \frac{1}{N_j} \sum_{i=1}^{n} w_{ij} x_i
\end{align*}
$$

其中，$x_i$ 表示数据点，$\mu_j$ 表示簇中心点，$w_{ij}$ 表示数据点 $x_i$ 属于簇 $j$ 的权重。

#### 9.3 多尺度交通流量预测方法

多尺度交通流量预测方法结合不同时间尺度和空间尺度的数据，提高预测的准确性和实时性。常见的方法包括空间尺度融合和时间尺度融合。

1. **空间尺度融合**

空间尺度融合方法通过整合不同区域或路段的交通流量数据，提高预测精度。常用的技术包括：

- **空间插值**：使用空间插值方法，如克里金插值（Kriging），将低分辨率数据转换为高分辨率数据，用于交通流量预测。
- **空间相关性分析**：通过分析交通流量数据的空间相关性，建立空间模型，如空间自回归模型（Spatial Autoregressive Model, SAR），进行多尺度融合预测。

2. **时间尺度融合**

时间尺度融合方法通过整合不同时间尺度的数据，提高预测的实时性和准确性。常用的技术包括：

- **时间序列分解**：将时间序列分解为趋势、季节性和随机性成分，分别进行建模和预测，然后进行融合。
- **长短时记忆网络（LSTM）**：LSTM可以同时捕捉时间序列的短期和长期依赖性，适用于多尺度交通流量预测。

3. **应用案例**

- **多尺度交通流量预测**：通过空间尺度融合和时间尺度融合方法，建立多尺度交通流量预测模型，提高预测的精度和实时性。
- **实时交通流量监控**：使用多尺度预测模型，实时监控交通流量变化，提供动态交通管理策略。

通过强化学习算法、聚类分析算法和多尺度交通流量预测方法，可以应对复杂交通环境，提高交通流量预测的准确性，为智能交通系统提供有力支持。

#### 15.2 拥堵避免算法实现

以下是一个简单的拥堵避免算法实现示例，使用A*算法为车辆提供最优行驶路径。

```python
import heapq

def heuristic(a, b):
    # 使用曼哈顿距离作为启发式函数
    return abs(a[0] - b[0]) + abs(a[1] - b[1])

def astar(graph, start, goal):
    # 初始化优先队列和访问过的节点
    open_set = []
    heapq.heappush(open_set, (heuristic(start, goal), start))
    came_from = {}
    g_score = {start: 0}
    
    while open_set:
        # 选择具有最小F值的节点进行扩展
        current = heapq.heappop(open_set)[1]
        
        if current == goal:
            # 达到目标节点，构建路径
            path = []
            while current in came_from:
                path.append(current)
                current = came_from[current]
            path.append(start)
            path.reverse()
            return path
        
        # 扩展当前节点的邻居节点
        for neighbor in graph.neighbors(current):
            tentative_g_score = g_score[current] + graph.cost(current, neighbor)
            
            if tentative_g_score < g_score.get(neighbor, float('inf')):
                came_from[neighbor] = current
                g_score[neighbor] = tentative_g_score
                f_score = tentative_g_score + heuristic(neighbor, goal)
                heapq.heappush(open_set, (f_score, neighbor))
    
    return None

# 定义一个简单的图（用于演示）
graph = {
    'A': ['B', 'C', 'D'],
    'B': ['A', 'E'],
    'C': ['A', 'F'],
    'D': ['A', 'G'],
    'E': ['B', 'H'],
    'F': ['C', 'I'],
    'G': ['D', 'I'],
    'H': ['E', 'I'],
    'I': ['F', 'G', 'H']
}

# 使用A*算法找到从A到I的最短路径
path = astar(graph, 'A', 'I')
print(f'Most optimal path from A to I: {path}')

# 可视化路径
import networkx as nx
G = nx.Graph()

for node, neighbors in graph.items():
    G.add_nodes_from([node])
    G.add_edges_from(zip(node, neighbors))

nx.draw(G, with_labels=True)
plt.show()
```

在这个示例中，我们首先定义了一个简单的图，并使用A*算法找到从A到I的最短路径。算法中使用了曼哈顿距离作为启发式函数，以加速搜索过程。通过NetworkX库，我们可以可视化最短路径。

# 假设已经处理好的交通流量数据存储在traffic_data.npy文件中
traffic_data = np.load('traffic_data.npy')

# 划分训练集和测试集
split = int(0.8 * len(X))
X_train, y_train = X[:split], y[:split]
X_test, y_test = X[split:], y[split:]

# 模型定义
model = Sequential()
model.add(LSTM(units=50, return_sequences=True, input_shape=(X_train.shape[1], 1)))
model.add(LSTM(units=50))
model.add(Dense(1))

# 编译模型
model.compile(optimizer='adam', loss='mse')

# 训练模型
model.fit(X_train, y_train, epochs=100, batch_size=32, validation_data=(X_test, y_test))

# 预测
predictions = model.predict(X_test)

# 评估模型性能
mse = mean_squared_error(y_test, predictions)
print(f'MSE: {mse}')

# 可视化预测结果
plt.figure(figsize=(10, 6))
plt.plot(y_test, label='实际流量')
plt.plot(predictions, label='预测流量')
plt.title('交通流量预测')
plt.xlabel('时间')
plt.ylabel('流量')
