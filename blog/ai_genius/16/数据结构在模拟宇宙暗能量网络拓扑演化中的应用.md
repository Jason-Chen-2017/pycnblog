##### 3.2 图的遍历算法

图的遍历算法是用于访问图中所有节点的算法。常见的遍历算法包括深度优先搜索（DFS）和广度优先搜索（BFS）。

- **深度优先搜索（DFS）**：

  深度优先搜索是一种从起始节点开始，沿着路径一直深入到底，然后回溯到上一个节点继续遍历的算法。DFS的伪代码如下：

  ```plaintext
  function DFS(graph, startNode):
      mark startNode as visited
      for each neighbor in graph[startNode]:
          if neighbor is not visited:
              DFS(graph, neighbor)
  ```

  DFS的时间复杂度为O(V+E)，其中V是节点数，E是边数。

- **广度优先搜索（BFS）**：

  广度优先搜索是一种从起始节点开始，依次访问其所有相邻节点，然后逐层遍历的算法。BFS的伪代码如下：

  ```plaintext
  function BFS(graph, startNode):
      create an empty queue
      mark startNode as visited
      enqueue startNode into the queue
      while queue is not empty:
          dequeue a node from the queue
          visit the node
          for each unvisited neighbor in graph[node]:
              mark neighbor as visited
              enqueue neighbor into the queue
  ```

  BFS的时间复杂度也为O(V+E)。

1. **深度优先搜索（DFS）**：

   ```plaintext
   function DFS(graph, startNode):
       mark startNode as visited
       for each neighbor in graph[startNode]:
           if neighbor is not visited:
               DFS(graph, neighbor)
   ```

2. **广度优先搜索（BFS）**：

   ```plaintext
   function BFS(graph, startNode):
       create an empty queue
       mark startNode as visited
       enqueue startNode into the queue
       while queue is not empty:
           dequeue a node from the queue
           visit the node
           for each unvisited neighbor in graph[node]:
               mark neighbor as visited
               enqueue neighbor into the queue
   ```

3. **最短路径算法（Dijkstra）**：

   ```plaintext
   function Dijkstra(graph, startNode):
       initialize distances with infinity
       distances[startNode] = 0
       for each node in graph:
           if node is not startNode:
               distances[node] = infinity
       priorityQueue = new MinPriorityQueue()
       for each node in graph:
           priorityQueue.enqueue(node, distances[node])
       while not priorityQueue.isEmpty():
           currentNode = priorityQueue.dequeue()
           for each neighbor in graph[currentNode]:
               distance = distances[currentNode] + weight(currentNode, neighbor)
               if distance < distances[neighbor]:
                   distances[neighbor] = distance
                   priorityQueue.enqueue(neighbor, distance)
   ```

4. **并查集（Union-Find）**：

   ```plaintext
   function UnionFind(n):
       parent = [i for i in range(n)]
       rank = [0 for i in range(n)]

       function find(x):
           if parent[x] != x:
               parent[x] = find(parent[x])
           return parent[x]

       function union(x, y):
           rootX = find(x)
           rootY = find(y)
           if rootX != rootY:
               if rank[rootX] > rank[rootY]:
                   parent[rootY] = rootX
               elif rank[rootX] < rank[rootY]:
                   parent[rootX] = rootY
               else:
                   parent[rootY] = rootX
                   rank[rootX] += 1
   ```
