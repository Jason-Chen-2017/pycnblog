```python
import sys

class Formula:
    def __init__(self, expression):
        self.expression = expression

    def __str__(self):
        return self.expression

class FormulaSystem:
    def __init__(self):
        self.formulas = []

    def add_formula(self, formula):
        self.formulas.append(formula)

    def deduce(self):
        for i in range(len(self.formulas) - 1):
            for j in range(i + 1, len(self.formulas)):
                if self.formulas[i].expression.startswith("¬") and \
                   self.formulas[j].expression.startswith("¬") and \
                   self.formulas[j].expression[1:] == self.formulas[i].expression[1:]:
                    self.formulas.remove(self.formulas[j])

    def __str__(self):
        return "\
".join([str(formula) for formula in self.formulas])

if __name__ == "__main__":
    fs = FormulaSystem()
    fs.add_formula(Formula("¬A → B"))
    fs.add_formula(Formula("B → C"))
    fs.add_formula(Formula("¬A"))
    fs.deduce()
    print(fs)
```

### 5.3 代码解读与分析

上述程序定义了两个类：`Formula` 和 `FormulaSystem`。

- `Formula` 类表示一个公式，包含一个表达式属性。
- `FormulaSystem` 类表示一个公式系统，包含一个公式列表属性。
- `add_formula` 方法用于向公式系统中添加公式。
- `deduce` 方法用于进行演绎推理，移除矛盾公式。
- `__str__` 方法用于打印公式系统中的所有公式。

在程序中，我们定义了三个公式：

1. $¬A → B$
2. $B → C$
3. $¬A$

然后，我们调用 `deduce` 方法进行演绎推理。由于第一个公式和第二个公式的结论 $C$ 与第三个公式的假设 $¬A$ 矛盾，因此 `deduce` 方法会移除第二个公式。最终，公式系统中的公式变为：

1. $¬A → B$
3. $¬A$

这就证明了 $B$。
