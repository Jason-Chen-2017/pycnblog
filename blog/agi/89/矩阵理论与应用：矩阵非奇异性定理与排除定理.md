## 2. 核心概念与联系
在介绍矩阵非奇异性定理与排除定理之前，我们需要先了解以下核心概念：

- **矩阵**：一个由数字组成的矩形数组，是线性代数中的基本对象。
- **方阵**：行数和列数相等的矩阵。
- **可逆矩阵**：存在一个矩阵 $A^{-1}$，使得 $AA^{-1} = A^{-1}A = I$，其中 $I$ 为单位矩阵。
- **奇异性**：矩阵不可逆的性质。
- **非奇异性**：矩阵可逆的性质。

矩阵非奇异性定理与排除定理之间的联系如下：

- 矩阵非奇异性定理提供了一种判断矩阵是否可逆的方法。
- 排除定理则用于排除矩阵奇异性，即确保矩阵可逆。

## 3. 核心算法原理与具体操作步骤
### 3.1 算法原理概述

矩阵非奇异性定理：一个方阵 $A$ 可逆的充分必要条件是 $\det(A) \
eq 0$，其中 $\det(A)$ 表示 $A$ 的行列式。

排除定理：对于线性方程组 $Ax = b$，如果 $A$ 是可逆的，则方程组有唯一解 $x$；如果 $A$ 是奇异的，则方程组无解或有无穷多解。

### 3.2 算法步骤详解

判断矩阵 $A$ 非奇异性：

1. 计算矩阵 $A$ 的行列式 $\det(A)$。
2. 如果 $\det(A) \
eq 0$，则 $A$ 可逆，非奇异的；否则，$A$ 不可逆，奇异的。

排除矩阵奇异性：

1. 将矩阵 $A$ 进行行简化，得到行最简形式矩阵 $B$。
2. 如果 $B$ 的主对角线元素均非零，则 $A$ 非奇异的；否则，$A$ 奇异的。

### 3.3 算法优缺点

**矩阵非奇异性定理**：

优点：简单易用，能够直接判断矩阵是否可逆。

缺点：仅适用于方阵，对于非方阵无法判断奇异性。

**排除定理**：

优点：适用于任意矩阵，能够排除矩阵奇异性。

缺点：需要将矩阵进行行简化，计算量较大。

### 3.4 算法应用领域

矩阵非奇异性定理与排除定理在以下领域得到广泛应用：

1. **线性方程组求解**：判断方程组是否有解，并确定解的唯一性。
2. **系统识别**：在系统建模和参数估计过程中，判断系统是否可识别。

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 数学模型构建

本节将介绍矩阵非奇异性定理与排除定理的数学模型及其推导过程。

**4.1.1 矩阵非奇异性定理**

假设 $A$ 是一个 $n \times n$ 的方阵，则 $A$ 可逆的充分必要条件是 $\det(A) \
eq 0$。

证明：

1. 假设 $A$ 可逆，则存在一个可逆矩阵 $B$，使得 $AB = BA = I$。
2. 对 $AB = I$ 进行行列式运算，得到 $\det(A)\det(B) = 1$。
3. 由于 $B$ 可逆，则 $\det(B) \
eq 0$，从而 $\det(A) \
eq 0$。
4. 反之，如果 $\det(A) \
eq 0$，则存在 $B$，使得 $AB = BA = I$，即 $A$ 可逆。

**4.1.2 排除定理**

假设 $A$ 是一个 $m \times n$ 的矩阵，则以下命题成立：

1. 如果 $A$ 是奇异的，则 $rank(A) < \min(m,n)$。
2. 如果 $A$ 是非奇异的，则 $rank(A) = \min(m,n)$。

证明：

1. 假设 $A$ 是奇异的，则 $\det(A) = 0$，从而 $rank(A) < n$。
2. 假设 $A$ 是非奇异的，则 $rank(A) = n$，且 $rank(A^T) = n$。
3. 由于 $rank(A) \leq \min(m,n)$，故 $rank(A) = n$。

import numpy as np

def is_square_matrix(A):
    return A.shape[0] == A.shape[1]

def is_invertible(A):
    if is_square_matrix(A):
        return np.linalg.det(A) != 0
    else:
        return np.linalg.matrix_rank(A) == min(A.shape)

def rank(A):
    if is_square_matrix(A):
        return np.linalg.matrix_rank(A)
    else:
        return min(A.shape)
