#### 题目一：最大公约数的算法原理

**题目：** 请解释欧几里得算法（辗转相除法）的原理，并编写一个 Golang 实现该算法的函数。

**答案：** 欧几里得算法（辗转相除法）是一种用于计算两个整数最大公约数的算法。其原理是通过连续用较小数去除较大数，然后用余数去除前一个除数，直到余数为零。此时，除数即为两个数的最大公约数。

```go
package main

import "fmt"

func gcd(a, b int) int {
    for b != 0 {
        t := b
        b = a % b
        a = t
    }
    return a
}

func main() {
    fmt.Println(gcd(48, 18)) // 输出 6
}
```

**解析：** 在这个例子中，`gcd` 函数使用欧几里得算法计算两个整数 `a` 和 `b` 的最大公约数。通过不断取余并交换变量，直到余数为零，此时 `a` 即为最大公约数。
