#### 1. 利用深度学习模型进行用户行为预测

**题目：** 假设你已经训练了一个深度学习模型，用于预测用户的下一步行为。请编写代码，实现以下功能：

* 加载训练好的模型；
* 输入用户行为数据，预测用户的下一步行为。

**答案：**

```python
import tensorflow as tf

# 加载训练好的模型
model = tf.keras.models.load_model('user_behavior_model.h5')

# 输入用户行为数据
user_behavior_data = [[1, 0, 1], [0, 1, 0], [1, 1, 0]]  # 示例数据

# 预测用户的下一步行为
predictions = model.predict(user_behavior_data)

print(predictions)
```
