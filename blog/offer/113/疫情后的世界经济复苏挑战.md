#### 算法编程题库

**1. 最短路径问题（Dijkstra 算法）**

**题目：** 给定一个加权无向图，找到起点和终点之间的最短路径。

**答案：**

```python
import heapq

def dijkstra(graph, start, end):
    # 初始化距离表
    distances = {node: float('inf') for node in graph}
    distances[start] = 0
    # 初始化优先队列
    priority_queue = [(0, start)]
    # 记录前驱节点
    previous = {node: None for node in graph}
    while priority_queue:
        # 取出最小距离的节点
        current_distance, current_node = heapq.heappop(priority_queue)
        # 如果当前节点是终点，结束循环
        if current_node == end:
            break
        # 遍历当前节点的邻接节点
        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight
            # 如果新的距离更短，更新距离表和前驱节点
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                previous[neighbor] = current_node
                heapq.heappush(priority_queue, (distance, neighbor))
    # 构建最短路径
    path = []
    current = end
    while current is not None:
        path.insert(0, current)
        current = previous[current]
    return path
```

**2. 股票买卖最佳时机 II**

**题目：** 给定一个整数数组 prices ，其中 prices[i] 是第 i 天股市的价格。如果你可以无限次地完成交易，包括交易多次同一种股票，编写一个函数来计算你能够获取的最大利润。

**答案：**

```python
def maxProfit(prices):
    max_profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i - 1]:
            max_profit += prices[i] - prices[i - 1]
    return max_profit
```

**3. 合并两个有序链表**

**题目：** 将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

**答案：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def mergeTwoLists(list1, list2):
    dummy = ListNode()
    curr = dummy
    while list1 and list2:
        if list1.val < list2.val:
            curr.next = list1
            list1 = list1.next
        else:
            curr.next = list2
            list2 = list2.next
        curr = curr.next
    curr.next = list1 or list2
    return dummy.next
```

**11. 动态规划 - 最长公共子序列**

**题目：** 给定两个字符串，找出它们的**最长公共子序列**。

**答案：**

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    return dp[m][n]
```

**12. 广度优先搜索 - 单源最短路径**

**题目：** 给定一个无权图和起点，使用广度优先搜索（BFS）找到从起点到其他各点的最短路径。

**答案：**

```python
from collections import deque

def bfs_shortest_path(graph, start, goal):
    visited = set()
    queue = deque([start])
    visited.add(start)
    steps = {start: 0}
    while queue:
        vertex = queue.popleft()
        for next in graph[vertex]:
            if next not in visited:
                visited.add(next)
                queue.append(next)
                steps[next] = steps[vertex] + 1
    return steps.get(goal, -1)
```

**13. 深度优先搜索 - 递归实现**

**题目：** 给定一个无权图，使用深度优先搜索（DFS）遍历图并输出遍历顺序。

**答案：**

```python
def dfs(graph, node, visited, result):
    visited.add(node)
    result.append(node)
    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited, result)

def dfs_order(graph):
    visited = set()
    result = []
    for node in graph:
        if node not in visited:
            dfs(graph, node, visited, result)
    return result
```
