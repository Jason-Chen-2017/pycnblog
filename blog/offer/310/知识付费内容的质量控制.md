##### 4. 编写一个函数，计算两个字符串的相似度。

**题目：** 编写一个函数 `stringSimilarity(str1, str2 string) float64`，计算两个字符串 `str1` 和 `str2` 的相似度。相似度范围为 [0, 1]，其中 0 表示完全不相似，1 表示完全相似。

**答案：** 下面是一个基于编辑距离的字符串相似度计算函数：

```go
func stringSimilarity(str1, str2 string) float64 {
    m, n := len(str1), len(str2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }
    for i := 1; i <= m; i++ {
        dp[i][0] = i
    }
    for j := 1; j <= n; j++ {
        dp[0][j] = j
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if str1[i-1] == str2[j-1] {
                dp[i][j] = dp[i-1][j-1]
            } else {
                dp[i][j] = min(dp[i-1][j]+1, min(dp[i][j-1]+1, dp[i-1][j-1]+1))
            }
        }
    }
    similarity := 1 - float64(dp[m][n]) / float64(max(m, n))
    return similarity
}

func min(a, b int) int {
    if a < b {
        return a
    }
    return b
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

**解析：** 该函数使用动态规划实现编辑距离，然后计算相似度。编辑距离越小，表示两个字符串相似度越高。
