#### **面试题 1：描述美国 H-1B 工作签证的申请流程。**

**答案：**

1. **找到合适的雇主：** 首先，程序员需要找到一家愿意为其提供工作并愿意为其申请 H-1B 工作签证的雇主。
2. **提交 Petition：** 雇主向美国公民及移民服务局（USCIS）提交 H-1B 工作签证的 Petition。
3. **抽签：** 如果雇主所在的行业和职位符合需求，USCIS 将对所有提交的 Petition 进行随机抽签，抽中者将继续申请流程。
4. **审批：** USCIS 对通过抽签的 Petition 进行审批。
5. **签证：** 如果 Petition 被批准，雇主将为程序员申请签证。
6. **入境：** 程序员携带批准通知和签证前往美国。

**解析：** 本题主要考察应聘者对美国 H-1B 工作签证申请流程的了解。

#### **面试题 2：阐述澳大利亚 482 签证的分类和申请条件。**

**答案：**

澳大利亚 482 签证分为短期签证和长期签证：

1. **短期签证：**
   - 申请条件：申请人需持有澳大利亚雇主的工作邀请，且薪资符合澳大利亚市场薪资水平。
   - 有效期：最长 4 年。

2. **长期签证：**
   - 申请条件：申请人需持有澳大利亚雇主的工作邀请，且符合特定技能要求，如技术移民类职业列表（STSOL）或区域职业列表（ROL）。
   - 有效期：最长 5 年。

**解析：** 本题考察应聘者对澳大利亚 482 签证的分类和申请条件的了解。

#### **面试题 3：解释加拿大技术移民 Express Entry 系统的工作原理。**

**答案：**

2. **评分标准：** Express Entry 采用 Comprehensive Ranking System（CRS）评分标准，根据申请人的年龄、教育背景、语言能力、工作经验等因素进行评分。
3. **邀请申请：** 加拿大移民、难民及公民部（IRCC）会定期根据申请人的评分情况邀请符合要求的申请人提交正式的移民申请。
4. **移民申请：** 被邀请的申请人需要在 60 天内提交完整的移民申请。

**解析：** 本题考察应聘者对加拿大技术移民 Express Entry 系统的理解。

**算法思路：**

1. **输入：** 各个国家程序员的薪资数据列表。
2. **处理：**
   - 计算每个国家的程序员薪资总和。
   - 计算全球程序员薪资总和。
   - 计算全球平均薪资水平。
3. **输出：** 全球程序员的平均薪资水平。

**伪代码：**

```python
def calculate_average_salary(salary_data):
    total_salary = 0
    num_countries = len(salary_data)
    
    for country, salary in salary_data.items():
        total_salary += salary
    
    average_salary = total_salary / num_countries
    return average_salary
```
