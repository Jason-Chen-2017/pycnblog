##### 1. 设计一个投票系统

**题目：** 设计一个投票系统，确保每个用户只能投票一次，且投票结果能够及时更新。

**答案：** 可以使用哈希表和计数器实现一个投票系统。具体步骤如下：

```python
# 存储投票结果的字典
votes = {}

# 投票函数
def vote(candidate):
    if candidate in votes:
        print("你已经投过票了。")
    else:
        votes[candidate] = votes.get(candidate, 0) + 1
        print(f"你为 {candidate} 投了一票。")

# 更新结果函数
def update_results():
    sorted_votes = sorted(votes.items(), key=lambda x: x[1], reverse=True)
    print("投票结果：")
    for candidate, count in sorted_votes:
        print(f"{candidate}: {count} 票")

# 测试
vote("候选人 A")
vote("候选人 B")
update_results()  # 应该显示候选人 B 获胜
```

**解析：** 该投票系统确保每个用户只能投票一次，通过字典存储投票结果，并按票数排序更新结果。

##### 2. 设计一个待办事项列表

**题目：** 设计一个待办事项列表，支持添加、删除和查看待办事项。

**答案：** 可以使用列表和字典实现一个待办事项列表。具体步骤如下：

```python
# 待办事项列表
tasks = []

# 添加待办事项函数
def add_task(task):
    tasks.append(task)
    print(f"已添加待办事项：{task}。")

# 删除待办事项函数
def delete_task(task):
    if task in tasks:
        tasks.remove(task)
        print(f"已删除待办事项：{task}。")
    else:
        print("该待办事项不存在。")

# 查看待办事项函数
def view_tasks():
    print("待办事项列表：")
    for task in tasks:
        print(task)

# 测试
add_task("买牛奶")
add_task("打扫房间")
delete_task("打扫房间")
view_tasks()  # 应该显示买牛奶
```

**解析：** 该待办事项列表支持添加、删除和查看功能，通过列表存储待办事项，便于管理。
