#### 算法编程题库

1. **题目：** 设计一个算法，用于计算一个数列的动态价格策略，以最大化利润。

   **输入：**
   - 一个数列 `prices`，表示每天的价格。
   - 一个整数 `k`，表示每天可以买入和卖出一次。

   **输出：**
   - 最大利润。

   **示例：**
   ```python
   prices = [1, 3, 2, 8, 4, 9]
   k = 2
   max_profit(prices, k) # 输出：11
   ```

   **答案：**
   ```python
   def max_profit(prices, k):
       if k > len(prices) // 2:
           return sum([max(prices[i], prices[i - 1]) for i in range(1, len(prices))]) - prices[0]
       
       dp = [[0] * (k + 1) for _ in range(len(prices))]
       for i in range(1, len(prices)):
           for j in range(1, k + 1):
               max_val = -prices[i]
               for p in range(i):
                   max_val = max(max_val, dp[p][j - 1] - prices[i])
               dp[i][j] = max(dp[i - 1][j], max_val + prices[i])
       
       return dp[-1][-1]
   ```

2. **题目：** 设计一个算法，用于计算给定字符串的文本相似度。

   **输入：**
   - 两个字符串 `text1` 和 `text2`。

   **输出：**
   - 文本相似度分数（0.0到1.0之间）。

   **示例：**
   ```python
   text1 = "AI大模型创业"
   text2 = "AI创业大模型"
   similarity_score(text1, text2) # 输出：0.85
   ```

   **答案：**
   ```python
   from sklearn.metrics.pairwise import cosine_similarity
   from sklearn.feature_extraction.text import TfidfVectorizer
   
   def similarity_score(text1, text2):
       vectorizer = TfidfVectorizer()
       tfidf_matrix = vectorizer.fit_transform([text1, text2])
       return cosine_similarity(tfidf_matrix[0:1], tfidf_matrix[1:2])[0][0]
   ```
