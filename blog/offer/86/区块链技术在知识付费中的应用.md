#### 三、算法编程题库

##### 1. 实现一个简单的区块链结构

**题目：** 编写一个简单的区块链结构，包含区块的基本信息（如：区块编号、时间戳、交易数据、链前哈希值）以及生成新区块的方法。

**答案：**

```python
import hashlib
import time

class Block:
    def __init__(self, index, transactions, timestamp, previous_hash):
        self.index = index
        self.transactions = transactions
        self.timestamp = timestamp
        self.previous_hash = previous_hash
        self.hash = self.compute_hash()

    def compute_hash(self):
        block_string = f"{self.index}{self.transactions}{self.timestamp}{self.previous_hash}"
        return hashlib.sha256(block_string.encode()).hexdigest()

def generate_blockchain():
    blockchain = []
    genesis_block = Block(0, [], time.time(), "0")
    genesis_block.hash = genesis_block.compute_hash()
    blockchain.append(genesis_block)
    return blockchain

def add_block(new_transactions):
    last_block = blockchain[-1]
    new_block = Block(last_block.index + 1, new_transactions, time.time(), last_block.hash)
    new_block.hash = new_block.compute_hash()
    blockchain.append(new_block)

blockchain = generate_blockchain()
add_block(["Transaction 1", "Transaction 2"])
```

**解析：** 该示例实现了一个简单的区块链结构，包含生成创世区块和添加新区块的方法。通过计算每个区块的哈希值，实现了区块链的数据完整性。

##### 2. 实现一个简单的智能合约

**题目：** 编写一个简单的智能合约，实现自动支付功能。当用户支付一定的金额后，合约自动将支付金额转移到指定地址。

**答案：**

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract PaymentContract {
    address public owner;
    mapping(address => uint256) public balances;

    constructor() {
        owner = msg.sender;
    }

    function pay() public payable {
        require(msg.value > 0, "支付金额必须大于 0");
        balances[msg.sender] += msg.value;
    }

    function transfer(address to, uint256 amount) public {
        require(balances[msg.sender] >= amount, "余额不足");
        balances[msg.sender] -= amount;
        balances[to] += amount;
    }
}
```

**解析：** 该示例实现了一个简单的智能合约，包含支付功能和余额转移功能。用户可以通过调用 `pay()` 方法支付金额，然后通过 `transfer()` 方法将支付金额转移到其他地址。
