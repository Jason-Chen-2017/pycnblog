**题目描述：** 编写一个算法，用于自动提取长文本的摘要。

**答案解析：**

* **文本预处理：** 对文本进行分词、去停用词等预处理操作。
* **句子评分：** 使用句子评分算法，如 TextRank、TF-IDF 等，为文本中的每个句子打分。
* **提取摘要：** 根据句子评分，选择前几个高分的句子作为摘要。

**示例代码：**

```python
from nltk.tokenize import sent_tokenize
from collections import defaultdict
from math import log

def sentence_similarity(sentence1, sentence2):
    words1 = set(sentence1.split())
    words2 = set(sentence2.split())
    common_words = len(words1.intersection(words2))
    total_words = len(words1.union(words2))
    return common_words / total_words

def text_rank(text):
    sentences = sent_tokenize(text)
    similarity_matrix = [[0 for _ in range(len(sentences))] for _ in range(len(sentences))]
    for i, s1 in enumerate(sentences):
        for j, s2 in enumerate(sentences):
            if i != j:
                similarity_matrix[i][j] = sentence_similarity(s1, s2)

    # calculate sentence scores
    alpha = 0.85
    k = 1
    iteration = 20
    for _ in range(iteration):
        score = defaultdict(float)
        for i, s in enumerate(sentences):
            score[i] = (1 - alpha) / len(sentences)
            for j in range(len(sentences)):
                if j != i:
                    score[i] += alpha * similarity_matrix[i][j] / len(sentences)

        # normalize scores
        max_score = max(score.values())
        for i in score:
            score[i] = score[i] / max_score

        # update k
        k = max(score.values())

    # select top k sentences as the summary
    summary = [s for s, score in zip(sentences, score) if score >= k]
    return ' '.join(summary)
```
