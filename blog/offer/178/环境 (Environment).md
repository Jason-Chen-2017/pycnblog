## 算法编程题集

### 1. 编写一个程序，实现环境空气质量数据的归一化。

**题目：** 给定一组空气质量数据，实现数据的归一化，将每个数据点映射到 [0, 1] 区间。

**答案：** 

```python
def normalize(data):
    min_val = min(data)
    max_val = max(data)
    normalized_data = [(x - min_val) / (max_val - min_val) for x in data]
    return normalized_data

# 测试
data = [20, 30, 40, 50]
print(normalize(data)) # 输出：[0.0, 0.25, 0.5, 1.0]
```

**解析：** 数据归一化是环境数据分析中的常用方法，可以消除数据量级差异，便于后续分析。

### 2. 编写一个程序，实现基于加权K-means算法的空气质量聚类分析。

**题目：** 基于加权K-means算法，对一组空气质量数据进行聚类分析。

**答案：** 

```python
import numpy as np
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

def weighted_kmeans(data, num_clusters):
    # 计算每个数据点的权重
    weights = np.std(data, axis=1)
    
    # 使用加权K-means算法进行聚类
    kmeans = KMeans(n_clusters=num_clusters, weights=weights)
    kmeans.fit(data)
    
    # 输出聚类结果
    clusters = kmeans.labels_
    centroids = kmeans.cluster_centers_
    
    # 计算轮廓系数评估聚类效果
    silhouette_avg = silhouette_score(data, clusters)
    
    return clusters, centroids, silhouette_avg

# 测试
data = np.array([[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]])
num_clusters = 2
clusters, centroids, silhouette_avg = weighted_kmeans(data, num_clusters)
print("聚类结果：", clusters)
print("聚类中心：", centroids)
print("轮廓系数：", silhouette_avg)
```

**解析：** 加权K-means算法是一种改进的K-means算法，通过对每个数据点赋予不同的权重，可以提高聚类结果的准确性。
