#### 二、算法编程题

1. **最长公共子序列（LCS）**

   **题目描述：** 给定两个字符串，找出它们的最长公共子序列。

   **答案：**
   ```python
   def longest_common_subsequence(s1, s2):
       dp = [[0] * (len(s2) + 1) for _ in range(len(s1) + 1)]

       for i in range(1, len(s1) + 1):
           for j in range(1, len(s2) + 1):
               if s1[i - 1] == s2[j - 1]:
                   dp[i][j] = dp[i - 1][j - 1] + 1
               else:
                   dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

       return dp[-1][-1]

   s1 = "AGGTAB"
   s2 = "GXTXAYB"
   print(longest_common_subsequence(s1, s2))  # 输出：4
   ```

2. **二分查找**

   **题目描述：** 给定一个有序数组和一个目标值，实现二分查找算法，找出目标值在数组中的索引。

   **答案：**
   ```python
   def binary_search(arr, target):
       left, right = 0, len(arr) - 1

       while left <= right:
           mid = (left + right) // 2
           if arr[mid] == target:
               return mid
           elif arr[mid] < target:
               left = mid + 1
           else:
               right = mid - 1

       return -1

   arr = [1, 3, 5, 7, 9, 11]
   target = 7
   print(binary_search(arr, target))  # 输出：3
   ```

3. **寻找两个正序数组的中位数**

   **题目描述：** 给定两个已排序的正序数组，找出它们的第 k 小元素。

   **答案：**
   ```python
   def find_kth_element(arr1, arr2, k):
       if len(arr1) > len(arr2):
           arr1, arr2 = arr2, arr1

       if not arr1:
           return arr2[k - 1]

       if k == 1:
           return min(arr1[0], arr2[0])

       i = min(len(arr1), k // 2)
       j = min(len(arr2), k // 2)

       if arr1[i - 1] > arr2[j - 1]:
           return find_kth_element(arr1, arr2[j + 1:], k - j - 1)
       elif arr1[i - 1] < arr2[j - 1]:
           return find_kth_element(arr1[i + 1:], arr2, k - i - 1)
       else:
           return arr1[i - 1]

   arr1 = [1, 3, 5]
   arr2 = [2, 4, 6]
   k = 3
   print(find_kth_element(arr1, arr2, k))  # 输出：4
   ```
