## 一、数据结构与算法面试题

### 1. 如何在 O(1) 时间复杂度内查找、插入和删除元素？

**题目：** 设计一个数据结构，使其支持在 O(1) 时间复杂度内查找、插入和删除元素。

**答案：** 可以使用哈希表来实现这一功能。哈希表通过哈希函数将关键字转换成地址，从而实现快速查找、插入和删除。

**代码示例：**

```python
class MyHashSet:
    def __init__(self):
        self.hash_set = set()

    def add(self, key: int) -> None:
        self.hash_set.add(key)

    def remove(self, key: int) -> None:
        self.hash_set.discard(key)

    def contains(self, key: int) -> bool:
        return key in self.hash_set
```

### 2. 如何找到链表中的中间节点？

**题目：** 给定一个单链表，如何找到链表的中间节点？

**答案：** 可以使用快慢指针法。快指针每次移动两个节点，慢指针每次移动一个节点。当快指针到达链表末尾时，慢指针指向的就是中间节点。

**代码示例：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def findMiddleNode(head: ListNode) -> ListNode:
    slow = head
    fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow
```

## 二、系统设计面试题

### 3. 如何实现缓存淘汰算法？

**题目：** 设计一种缓存淘汰算法，要求在缓存满了时，按照某种策略淘汰缓存中的数据。

**答案：** 可以实现两种常见的缓存淘汰算法：FIFO（先进先出）和 LRU（最近最少使用）。

**代码示例：**（LRU 算法使用哈希表 + 双向链表实现）

```python
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = OrderedDict()

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.cache.move_to_end(key)
        return self.cache[key]

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.cache.pop(key)
        elif len(self.cache) >= self.capacity:
            self.cache.popitem(last=False)
        self.cache[key] = value
```

## 四、其他领域面试题

### 5. 如何实现一个简单的 HTTP 服务器？

**题目：** 编写一个简单的 HTTP 服务器，实现基本的请求处理功能。

**答案：** 使用 Python 的 `socket` 模块可以轻松实现一个简单的 HTTP 服务器。

**代码示例：**

```python
import socket

def handle_request(client_socket):
    request = client_socket.recv(1024)
    response = "HTTP/1.1 200 OK\n\nHello, World!"
    client_socket.send(response.encode())

server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server_socket.bind(('0.0.0.0', 80))
server_socket.listen(5)

while True:
    client_socket, addr = server_socket.accept()
    handle_request(client_socket)
    client_socket.close()
```

### 6. 如何实现一个加密算法？

**题目：** 使用 Python 实现一个简单的加密算法。

**答案：** 使用 Python 的 `hashlib` 模块可以实现多种加密算法，如 MD5、SHA1、SHA256 等。

**代码示例：**

```python
import hashlib

def encrypt(password: str, algorithm: str = 'sha256') -> str:
    hash_object = hashlib.new(algorithm)
    hash_object.update(password.encode())
    return hash_object.hexdigest()

print(encrypt('my_password', 'sha256'))
```
