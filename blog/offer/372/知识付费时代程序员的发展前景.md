#### 算法编程题库

**5. 请实现一个函数，计算一个数字是否是回文数。**

**题目：** 判断一个整数是否是回文数。回文数是指正读和反读都一样的整数。

**答案：**

```python
def is_palindrome(x: int) -> bool:
    if x < 0:
        return False
    
    reversed_x = 0
    temp = x
    
    while temp > 0:
        reversed_x = reversed_x * 10 + temp % 10
        temp //= 10
    
    return reversed_x == x or reversed_x == -x

# 测试
print(is_palindrome(121))  # 输出 True
print(is_palindrome(-121))  # 输出 False
print(is_palindrome(10))  # 输出 False
```

**解析：** 该算法通过将整数反转并与原整数比较，判断是否为回文数。需要注意的是，负数和10的倍数（除了10本身）不是回文数。

**6. 请实现一个函数，找到未排序数组中的多数元素。**

**题目：** 在一个未排序的数组中，找到出现频率超过数组长度一半的元素。

**答案：**

```python
def majority_element(nums):
    count = 0
    candidate = None
    
    for num in nums:
        if count == 0:
            candidate = num
        count += (1 if num == candidate else -1)
    
    return candidate

# 测试
print(majority_element([3, 2, 3]))  # 输出 3
print(majority_element([2, 2, 1, 1, 1, 2, 2]))  # 输出 2
```

**解析：** 该算法利用摩尔投票算法，通过计数法找到多数元素。算法的时间复杂度为O(n)，空间复杂度为O(1)。

**7. 请实现一个函数，将单链表中的节点按奇偶数位置分割成两个独立的链表。**

**题目：** 给定一个单链表，将其中的奇数位置节点和偶数位置节点分割成两个独立的链表。

**答案：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def odd_even_list(head):
    if not head or not head.next:
        return head
    
    odd, even, even_head = head, head.next, head.next
    
    while even and even.next:
        odd.next = even.next
        odd = odd.next
        even.next = odd.next
        even = even.next
    
    odd.next = even_head
    return head

# 测试
head = ListNode(1, ListNode(2, ListNode(3, ListNode(4, ListNode(5)))))
new_head = odd_even_list(head)
while new_head:
    print(new_head.val, end=" ")
    new_head = new_head.next
# 输出：1 3 5 2 4
```

**解析：** 该算法通过创建两个新的链表，分别存放奇数位置和偶数位置的节点，然后将两个链表连接起来。
