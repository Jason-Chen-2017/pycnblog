##### 1. 算法与数据结构

**题目：** 实现一个函数，判断一个二叉树是否是平衡二叉树。

**答案：** 

```java
public class Solution {
    public boolean isBalanced(TreeNode root) {
        if (root == null) {
            return true;
        }
        int left = depth(root.left);
        int right = depth(root.right);
        if (Math.abs(left - right) > 1) {
            return false;
        }
        return isBalanced(root.left) && isBalanced(root.right);
    }

    private int depth(TreeNode node) {
        if (node == null) {
            return 0;
        }
        return 1 + Math.max(depth(node.left), depth(node.right));
    }
}
```

**解析：** 通过递归计算左右子树的深度，若差值大于1，则不是平衡二叉树。

##### 2. 计算机网络

**题目：** 什么是TCP的三次握手和四次挥手？请简述其过程。

**答案：** 

**三次握手：**
1. 客户端发送SYN包到服务器，并进入SYN_SENT状态。
2. 服务器收到SYN包，发送SYN+ACK包到客户端，并进入SYN_RCVD状态。
3. 客户端收到SYN+ACK包，发送ACK包到服务器，并进入ESTABLISHED状态。
4. 服务器收到ACK包，进入ESTABLISHED状态。

**四次挥手：**
1. 客户端发送FIN包到服务器，并进入FIN_WAIT_1状态。
2. 服务器收到FIN包，发送ACK包到客户端，并进入CLOSE_WAIT状态。
3. 客户端收到ACK包，进入FIN_WAIT_2状态。
4. 服务器发送FIN包到客户端，并进入LAST_ACK状态。
5. 客户端收到FIN包，发送ACK包到服务器，并进入TIME_WAIT状态。
6. 服务器收到ACK包，进入CLOSED状态。

**解析：** 三次握手用于建立TCP连接，四次挥手用于终止TCP连接。

##### 3. 操作系统

**题目：** 什么是进程和线程？它们有什么区别？

**答案：** 

**进程：** 进程是计算机中正在执行的程序的实例，包括程序代码、数据、堆栈等。进程是资源分配和调度的基本单位。

**线程：** 线程是进程中的一条执行路径，共享进程的资源。线程是程序执行的最小单位，是处理器调度的基本单位。

**区别：**
1. 资源：进程拥有独立的内存空间，而线程共享进程的内存空间。
2. 创建和销毁：进程创建和销毁开销较大，线程开销较小。
3. 独立性：进程独立，互不影响；线程之间可以共享数据，但需要注意同步问题。
4. 通信：进程间通信开销较大，线程间通信开销较小。

**解析：** 进程和线程是操作系统中的基本概念，进程是程序的运行实例，线程是进程中的执行路径。

##### 1. 寻找两个正序数组的中位数

**题目：** 给定两个大小分别为 m 和 n 的正序数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。

**答案：**

```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        nums = nums1 + nums2
        nums.sort()
        if len(nums) % 2 == 0:
            return (nums[len(nums) // 2 - 1] + nums[len(nums) // 2]) / 2
        else:
            return nums[len(nums) // 2]
```

**解析：** 将两个数组合并后排序，若数组长度为偶数，返回中间两个数的平均值；若长度为奇数，返回中间数。

##### 2. 两个链表的第一个公共节点

**题目：** 给出两个无向的链表，找出它们的第一个公共节点。

**答案：**

```python
class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        pA, pB = headA, headB
        while pA != pB:
            pA = pA or headB
            pB = pB or headA
        return pA
```

**解析：** 利用双指针，当两个指针相遇时，即为第一个公共节点。

##### 3. 最长公共子序列

**题目：** 给定两个字符串，找出它们的 最长公共子序列 。

**答案：**

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> str:
        m, n = len(text1), len(text2)
        dp = [["" for _ in range(n + 1)] for _ in range(m + 1)]
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i - 1] == text2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1] + text1[i - 1]
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1], key=len)
        return dp[-1][-1]
```

**解析：** 使用动态规划，遍历字符串，若当前字符相同，则累加前一个字符的子序列；否则，取最大长度子序列。
