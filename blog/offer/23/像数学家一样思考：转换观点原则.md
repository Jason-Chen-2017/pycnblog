### 一、数据结构与算法

**1. 颠倒整数**

**题目：** 不使用库函数，将一个整数翻转。

**答案：** 

```python
class Solution:
    def reverse(self, x: int) -> int:
        ans = 0
        while x:
            ans = ans * 10 + x % 10
            x //= 10
        return ans if ans <= 2**31 - 1 else 0
```

**解析：** 将整数除以10并取余，得到个位数，再将这个个位数加到答案上，然后更新x。重复这个过程，直到x变为0。这里使用了整数除法和取余操作，保证了每次循环中答案都正确更新。最后，如果答案大于等于`2**31 - 1`，则返回0，以避免溢出。

**2. 螺旋矩阵**

**题目：** 给定一个 m 行 n 列的二维矩阵，返回一个按顺时针螺旋顺序遍历矩阵的序列。

**答案：**

```python
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        ans = []
        while matrix:
            ans += matrix.pop(0)
            if not matrix or not matrix[0]:
                break
            for row in matrix:
                ans += [row.pop()]
            if not matrix:
                break
            ans += matrix.pop()[::-1]
            if not matrix or not matrix[0]:
                break
            for row in matrix[::-1]:
                ans += [row.pop(0)]
        return ans
```

**解析：** 使用循环遍历矩阵的四个边界，每次遍历完一个边界，就从矩阵中移除相应的行或列。这里使用了pop和pop(0)操作，以及矩阵的切片操作。

### 二、逻辑与推理

**3. 两数之和**

**题目：** 给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**答案：**

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        d = {v: i for i, v in enumerate(nums)}
        for i, v in enumerate(nums):
            j = target - v
            if j in d and d[j] != i:
                return [i, d[j]]
        return []
```

**解析：** 使用哈希表存储数组中的每个元素及其索引。遍历数组，对于每个元素，计算目标值与当前元素的差，如果差在哈希表中，且对应的索引不是当前索引，则找到了两个数。

### 三、数学与几何

**4. 盛水的容器**

**题目：** 给定一个长度为 n 的整数数组 height ，表示一个由矩形体组成的城墙，返回盛水的容器的最大容量。

**答案：**

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        ans, left, right = 0, 0, len(height) - 1
        while left < right:
            ans = max(ans, min(height[left], height[right]) * (right - left))
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        return ans
```

**解析：** 双指针法，left和right分别指向数组两端，计算当前容量，并移动较短的一侧。这样可以保证每次移动都能找到更大的容量。
