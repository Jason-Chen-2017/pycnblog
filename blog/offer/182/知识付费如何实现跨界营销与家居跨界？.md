### 1. 函数是值传递还是引用传递？

**题目：** 在 Golang 中，函数参数传递是值传递还是引用传递？请举例说明。

**答案：** Golang 中所有参数都是值传递。这意味着函数接收的是参数的一份拷贝，对拷贝的修改不会影响原始值。

**解析：** Golang 中函数参数传递采用的是值传递，即传递的是参数的拷贝，而不是原始变量。因此，在函数内部对参数的修改不会影响原始变量。

### 2. 如何安全读写共享变量？

**答案：** 可以使用以下方法安全地读写共享变量：

- 使用互斥锁（Mutex）进行同步操作。
- 使用读写锁（RWMutex）实现读写分离。
- 使用原子操作（Atomic）保证操作原子性。
- 使用通道（Channel）进行数据传递。

### 3. 缓冲、无缓冲 chan 的区别

**题目：** Golang 中，带缓冲和不带缓冲的通道有什么区别？

**答案：** 无缓冲通道（Unbuffered channel）发送操作会阻塞，直到有接收操作准备好接收数据；接收操作会阻塞，直到有发送操作准备好发送数据。带缓冲通道（Buffered channel）发送操作只有在缓冲区满时才会阻塞；接收操作只有在缓冲区为空时才会阻塞。

### 1. 合并两个有序链表

**题目：** 将两个有序链表合并为一个有序链表。

**答案：** 使用递归或迭代方式合并两个有序链表，比较两个链表的头节点值，选择较小的值作为新链表的头节点，然后递归或迭代处理剩余的链表。

```go
func mergeTwoLists(l1 *ListNode, l2 *ListNode) *ListNode {
    if l1 == nil {
        return l2
    }
    if l2 == nil {
        return l1
    }
    if l1.Val < l2.Val {
        l1.Next = mergeTwoLists(l1.Next, l2)
        return l1
    }
    l2.Next = mergeTwoLists(l1, l2.Next)
    return l2
}
```

### 2. 逆波兰表达式求值

**题目：** 计算逆波兰表达式（后缀表达式）的值。

**答案：** 使用栈实现逆波兰表达式的求值，遇到操作数入栈，遇到运算符则弹出栈顶两个操作数进行计算，并将结果入栈。

```go
func evalRPN(tokens []string) int {
    var stack []int
    for _, token := range tokens {
        switch token {
        case "+":
            b := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            a := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            stack = append(stack, a+b)
        case "-":
            b := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            a := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            stack = append(stack, a-b)
        case "*":
            b := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            a := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            stack = append(stack, a*b)
        case "/":
            b := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            a := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            stack = append(stack, a/int(b))
        default:
            stack = append(stack, atoi(token))
        }
    }
    return stack[0]
}
```
