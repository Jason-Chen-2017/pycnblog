**1. 计算课程综合评分**

**题目描述：** 给定一个课程列表，每个课程包含课程名称、课程难度、用户评分等信息，计算每个课程的加权平均分。

**输入：**

```
[
  {"name": "课程1", "difficulty": 3, "rating": [4, 5, 4]},
  {"name": "课程2", "difficulty": 2, "rating": [5, 5, 5]},
  {"name": "课程3", "difficulty": 1, "rating": [3, 4, 3]}
]
```

**输出：**

```
[
  {"name": "课程1", "weighted_average": 4.0},
  {"name": "课程2", "weighted_average": 5.0},
  {"name": "课程3", "weighted_average": 3.7}
]
```

**答案：** 使用 Python 实现：

```python
def calculate_weighted_average(courses):
    result = []
    for course in courses:
        total_rating = sum(course['rating'])
        weighted_average = total_rating / len(course['rating'])
        result.append({
            'name': course['name'],
            'weighted_average': weighted_average
        })
    return result

courses = [
  {"name": "课程1", "difficulty": 3, "rating": [4, 5, 4]},
  {"name": "课程2", "difficulty": 2, "rating": [5, 5, 5]},
  {"name": "课程3", "difficulty": 1, "rating": [3, 4, 3]}
]

print(calculate_weighted_average(courses))
```

**2. 判断用户是否可能购买某个课程**

**题目描述：** 给定一个用户历史购买记录列表和一个课程列表，判断某个用户是否可能购买某个课程。

**输入：**

```
[
  {"user_id": 1, "course_id": 101},
  {"user_id": 1, "course_id": 102},
  {"user_id": 2, "course_id": 103},
  {"user_id": 2, "course_id": 104}
]

[
  {"course_id": 101, "name": "Python 基础"},
  {"course_id": 102, "name": "数据结构"},
  {"course_id": 103, "name": "机器学习"},
  {"course_id": 104, "name": "深度学习"}
]
```

**输出：**

```
[
  {"user_id": 1, "course_id": 103, "may_purchase": True},
  {"user_id": 1, "course_id": 104, "may_purchase": False},
  {"user_id": 2, "course_id": 101, "may_purchase": True},
  {"user_id": 2, "course_id": 102, "may_purchase": True}
]
```

**答案：** 使用 Python 实现：

```python
def predict_purchase(user_purchases, courses):
    result = []
    for purchase in user_purchases:
        user_id = purchase['user_id']
        course_id = purchase['course_id']
        course = next((c for c in courses if c['course_id'] == course_id), None)
        if course:
            result.append({
                'user_id': user_id,
                'course_id': course_id,
                'may_purchase': True
            })
        else:
            result.append({
                'user_id': user_id,
                'course_id': course_id,
                'may_purchase': False
            })
    return result

user_purchases = [
  {"user_id": 1, "course_id": 101},
  {"user_id": 1, "course_id": 102},
  {"user_id": 2, "course_id": 103},
  {"user_id": 2, "course_id": 104}
]

courses = [
  {"course_id": 101, "name": "Python 基础"},
  {"course_id": 102, "name": "数据结构"},
  {"course_id": 103, "name": "机器学习"},
  {"course_id": 104, "name": "深度学习"}
]

print(predict_purchase(user_purchases, courses))
```
