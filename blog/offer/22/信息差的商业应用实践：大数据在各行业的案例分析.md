### 1. 大数据技术在电商行业中的应用

**题目：** 请解释大数据技术在电商行业中的核心应用，并举例说明。

**答案：** 大数据技术在电商行业中的核心应用包括：

- **个性化推荐：** 通过用户行为、浏览记录、购买历史等数据，构建用户画像，实现个性化商品推荐。
- **价格监控：** 对同款商品进行价格监控，分析价格变化趋势，优化价格策略。
- **销售预测：** 基于历史销售数据、季节性因素、促销活动等预测未来销售量，优化库存管理。

**举例：** 以京东为例，其通过用户行为数据构建用户画像，利用机器学习算法进行商品推荐。

**解析：** 京东会收集用户的浏览、购买、评价等行为数据，通过机器学习算法分析用户兴趣和需求，实现个性化推荐。

### 源代码实例

以下是一个简单的 Python 代码示例，用于计算两个正整数的最大公约数（GCD）：

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

# 示例：计算 60 和 48 的最大公约数
result = gcd(60, 48)
print("最大公约数：", result)
```

在这个示例中，我们使用欧几里得算法（也称为辗转相除法）来计算最大公约数。函数 `gcd` 接受两个参数 `a` 和 `b`，然后通过循环迭代计算最大公约数。最后，我们调用 `gcd` 函数并打印结果。输出将是：

```
最大公约数：12
```

这个示例展示了如何使用 Python 编写简单的算法程序，解析了算法的逻辑和执行过程。在实际应用中，大数据处理和算法编程通常会涉及更复杂的数据结构和算法，但基本原理是类似的。通过这些示例，您可以更好地理解大数据技术在各行业的应用场景。
