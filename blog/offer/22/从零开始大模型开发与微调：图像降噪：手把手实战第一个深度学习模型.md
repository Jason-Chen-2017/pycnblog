1. **实现中值滤波**

   **题目：** 实现一个中值滤波器，用于去除图像中的椒盐噪声。

   **答案：** 

   ```python
   import numpy as np

   def median_filter(image, kernel_size):
       padded_image = np.zeros((image.shape[0] + kernel_size[0] - 1, image.shape[1] + kernel_size[1] - 1))
       padded_image[:image.shape[0], :image.shape[1]] = image

       filtered_image = np.zeros_like(image)

       for i in range(image.shape[0]):
           for j in range(image.shape[1]):
               kernel = padded_image[i:i+kernel_size[0], j:j+kernel_size[1]]
               filtered_image[i, j] = np.median(kernel)

       return filtered_image
   ```

2. **实现高斯滤波**

   **题目：** 实现一个高斯滤波器，用于平滑图像。

   **答案：**

   ```python
   import numpy as np

   def gaussian_filter(image, kernel_size, sigma):
       kernel = np.zeros_like(image)
       for i in range(kernel_size):
           for j in range(kernel_size):
               x, y = i - kernel_size // 2, j - kernel_size // 2
               kernel[i, j] = np.exp(-((x**2 + y**2) / (2 * sigma**2))) * np.pi * sigma**2
       kernel /= np.sum(kernel)

       padded_image = np.zeros((image.shape[0] + kernel_size[0] - 1, image.shape[1] + kernel_size[1] - 1))
       padded_image[:image.shape[0], :image.shape[1]] = image

       filtered_image = np.zeros_like(image)

       for i in range(image.shape[0]):
           for j in range(image.shape[1]):
               window = padded_image[i:i+kernel_size[0], j:j+kernel_size[1]]
               filtered_image[i, j] = np.sum(window * kernel)

       return filtered_image
   ```

   图像降噪是一种图像处理技术，旨在从图像中去除噪声，以提高图像质量或提取有用信息。在计算机视觉和深度学习领域，图像降噪技术具有重要意义。例如，在人脸识别、自动驾驶等应用中，清晰、无噪声的图像是准确识别和决策的关键。
