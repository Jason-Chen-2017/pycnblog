##### 1. 贪心算法：最小费用最大流

**题目：** 给定一个有向图，求其最小费用最大流。

**答案：** 可以使用Dijkstra算法求解最小费用最大流。具体步骤如下：

1. 初始化流量为0，费用为0。
2. 对所有顶点进行一遍Dijkstra算法，找出从源点s到汇点t的最小费用路径。
3. 沿着最小费用路径调整流量，直到无法增加流量。
4. 重复步骤2和3，直到最大流达到需求。

**代码示例：**

```python
from heapq import heappop, heappush

def min_cost_max_flow(graph, source, sink):
    flow = 0
    while True:
        dist = {v: float('inf') for v in graph}
        dist[source] = 0
        parent = {v: None for v in graph}
        heap = [(0, source)]
        while heap:
            cost, u = heappop(heap)
            if u == sink:
                break
            for v, (capacity, cost) in graph[u].items():
                if capacity > 0 and dist[v] > dist[u] + cost:
                    dist[v] = dist[u] + cost
                    parent[v] = u
                    heappush(heap, (dist[v], v))
        if not parent[sink]:
            break
        path = [sink]
        while parent[sink]:
            path.append(parent[sink])
            sink = parent[sink]
        for v in path[::-1]:
            u = parent[v]
            flow += min(flow[u][v], graph[u][v][1])
            graph[u][v] = (graph[u][v][0], graph[u][v][1] - flow[u][v])
            graph[v][u] = (graph[v][u][0], graph[v][u][1] + flow[u][v])
    return flow

# 示例图
graph = {
    's': {'a': (3, 2), 'b': (4, 3)},
    'a': {'b': (2, 1), 'c': (3, 3)},
    'b': {'c': (3, 1), 't': (2, 2)},
    'c': {'t': (2, 1)}
}
source = 's'
sink = 't'

# 求解最小费用最大流
max_flow = min_cost_max_flow(graph, source, sink)
print("最小费用最大流：", max_flow)
```

**解析：** 贪心算法在每次迭代中选择最小费用的边进行流量调整，直到无法增加流量为止。该方法在满足可行性条件的前提下，可以得到最优解。

##### 2. 动态规划：最长公共子序列

**题目：** 给定两个字符串，求它们的最长公共子序列。

**答案：** 可以使用动态规划求解最长公共子序列。具体步骤如下：

1. 创建一个二维数组dp，其中dp[i][j]表示字符串s1的前i个字符和字符串s2的前j个字符的最长公共子序列长度。
2. 初始化dp数组的第一行和第一列，其中dp[i][0]=dp[0][j]=0。
3. 遍历字符串s1和s2的每个字符，根据状态转移方程更新dp数组。
4. dp[s1.length][s2.length]即为最长公共子序列的长度。

**代码示例：**

```java
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length();
    int n = text2.length();
    int[][] dp = new int[m + 1][n + 1];

    for (int i = 0; i <= m; i++) {
        for (int j = 0; j <= n; j++) {
            if (i == 0 || j == 0) {
                dp[i][j] = 0;
            } else if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[m][n];
}
```

**解析：** 动态规划通过状态转移方程逐步求解子问题，最终得到最长公共子序列的长度。该方法的时间复杂度为O(mn)，其中m和n分别为两个字符串的长度。
