### 二、算法编程题库

#### 1. 实现K-均值聚类算法

**题目：** 实现K-均值聚类算法，将给定数据集划分为K个簇。

**答案：**

```python
import numpy as np

def k_means(data, K, max_iter):
    centroids = data[np.random.choice(data.shape[0], K, replace=False)]
    for _ in range(max_iter):
        # 计算每个点所属的簇
        distances = np.linalg.norm(data - centroids, axis=1)
        labels = np.argmin(distances, axis=1)
        
        # 更新簇中心
        new_centroids = np.array([data[labels == k].mean(axis=0) for k in range(K)])
        if np.all(centroids == new_centroids):
            break
        centroids = new_centroids
    return centroids, labels
```

#### 3. 实现基于内容的推荐算法

**题目：** 实现基于内容的推荐算法，为用户推荐与当前商品相似的其他商品。

**答案：**

```python
import numpy as np

def content_based_recommendation(item_features, user_history, similarity_threshold):
    similarities = np.dot(item_features, user_history.T)
    recommendations = np.where(similarities >= similarity_threshold)[1]
    return recommendations
```

#### 4. 实现协同过滤推荐算法

**题目：** 实现协同过滤推荐算法，为用户推荐与相似用户喜欢的商品。

**答案：**

```python
import numpy as np

def collaborative_filtering(user_matrix, similarity_threshold, k=5):
    similarities = np.dot(user_matrix, user_matrix.T)
    diagonal = np.diag(similarities)
    np.fill_diagonal(diagonal, 0)
    similarities = (similarities - diagonal) / np.linalg.norm(user_matrix, axis=1)[:, None]
    user_similarity = similarities / np.linalg.norm(similarities, axis=1)[:, None]
    
    recommendations = []
    for user in range(user_matrix.shape[0]):
        user_preferences = user_matrix[user]
        user_preferences[user] = 0
        similar_users = user_similarity[user]
        top_k_indices = np.argsort(similar_users)[-k:]
        top_k_users = user_matrix[top_k_indices]
        recommendations.append(np.dot(top_k_users, user_preferences))
    return recommendations
```
