#### 典型问题/面试题库

**1. 快排中的稳定性分析**

**题目：** 快速排序（Quick Sort）在平均情况下是一个高效的排序算法，但在最坏情况下性能会退化。请分析快排的不稳定性和改进方法。

**答案：**

快速排序在平均情况下的时间复杂度为 \(O(n \log n)\)，但在最坏情况下会退化到 \(O(n^2)\)。这是因为当每次分割都不平衡时，会形成一个链状结构，导致递归深度达到 \(n\)。

改进方法包括：

- 使用随机化的 pivot，减少最坏情况发生的概率。
- 采用三数取中法，选择中间值作为 pivot，避免极端值对性能的影响。
- 使用插入排序（Insertion Sort）处理小规模子数组，以提高整体性能。

**2. Redis 的数据结构**

**题目：** 请简要介绍 Redis 中常用到的数据结构及其优缺点。

**答案：**

Redis 是一个高性能的 key-value 存储系统，其内部使用多种数据结构来支持丰富的功能：

- 字符串（String）：简单但高效的数据结构，适用于存储少量数据。
- 列表（List）：双向链表结构，支持快速插入和删除。
- 集合（Set）：哈希表结构，支持快速添加、删除和判断元素是否存在。
- 有序集合（Sorted Set）：跳表结构，支持快速插入、删除和查找。
- 哈希（Hash）：哈希表结构，适用于存储键值对。

每种数据结构都有其优缺点，适用于不同的场景。例如，字符串适用于存储少量数据，而有序集合适用于需要排序和快速查找的场景。

**3. TCP 三次握手与四次挥手**

**题目：** 请解释 TCP 的三次握手和四次挥手机制，并简要说明其作用。

**答案：**

三次握手用于建立 TCP 连接，四次挥手用于终止 TCP 连接。

- 三次握手：

1. 客户端发送 SYN 包到服务器，并进入 SYN_SENT 状态。
2. 服务器收到 SYN 包后，发送 SYN+ACK 包到客户端，并进入 SYN_RCVD 状态。
3. 客户端收到 SYN+ACK 包后，发送 ACK 包到服务器，并进入 ESTABLISHED 状态。同时，服务器进入 ESTABLISHED 状态。

三次握手的作用是确保双方建立可靠的连接，并交换初始序列号。

- 四次挥手：

1. 客户端发送 FIN 包到服务器，并进入 FIN_WAIT_1 状态。
2. 服务器收到 FIN 包后，发送 ACK 包到客户端，并进入 CLOSE_WAIT 状态。
3. 客户端收到 ACK 包后，进入 FIN_WAIT_2 状态。
4. 服务器发送 FIN 包到客户端，并进入 LAST_ACK 状态。
5. 客户端收到 FIN 包后，发送 ACK 包到服务器，并进入 TIME_WAIT 状态。
6. 服务器收到 ACK 包后，进入 CLOSED 状态。

四次挥手的作用是确保双方完全关闭连接，并清理相关资源。

#### 算法编程题库

**1. 合并两个有序数组**

**题目：** 给定两个有序数组 nums1 和 nums2，将 nums2 合并到 nums1 中，使得 nums1 成为一个有序数组。

**答案：** 

```python
def merge(nums1: List[int], m: int, nums2: List[int], n: int) -> None:
    i, j, k = m - 1, n - 1, m + n - 1
    while i >= 0 and j >= 0:
        if nums1[i] > nums2[j]:
            nums1[k] = nums1[i]
            i -= 1
        else:
            nums1[k] = nums2[j]
            j -= 1
        k -= 1
    while i >= 0:
        nums1[k] = nums1[i]
        i -= 1
        k -= 1
    while j >= 0:
        nums1[k] = nums2[j]
        j -= 1
        k -= 1
```

**解析：** 通过从后向前比较两个数组的元素，将较大的元素依次放入 nums1 的末尾，从而实现合并。

**2. 逆波兰表达式求值**

**题目：** 实现一个函数，计算逆波兰表达式（Postfix Expression）的值。

**答案：**

```python
from collections import deque

def evalRPN(tokens):
    stack = deque()
    for token in tokens:
        if token.isdigit():
            stack.append(int(token))
        else:
            b = stack.pop()
            a = stack.pop()
            if token == '+':
                stack.append(a + b)
            elif token == '-':
                stack.append(a - b)
            elif token == '*':
                stack.append(a * b)
            elif token == '/':
                stack.append(a / b)
    return stack.pop()
```

**解析：** 使用栈实现逆波兰表达式的计算，根据运算符进行相应的计算操作。
