#### 1. 多项式根与系数的关系是什么？

**面试题：** 请简要解释多项式根与系数的关系，并给出一个实际应用的例子。

**答案解析：** 多项式根与系数的关系是指，对于一个n次多项式 \( P(x) = a_nx^n + a_{n-1}x^{n-1} + \cdots + a_1x + a_0 \)，它的每一个根 \( x_i \) 都与系数之间存在如下关系：

- \( x_1 + x_2 + \cdots + x_n = -\frac{a_{n-1}}{a_n} \)
- \( x_1x_2 \cdots x_n = (-1)^n \frac{a_0}{a_n} \)

例如，对于多项式 \( P(x) = x^3 - 6x^2 + 11x - 6 \)，其根与系数的关系为：

- \( x_1 + x_2 + x_3 = 6 \)
- \( x_1x_2x_3 = 6 \)

#### 2. 如何计算多项式的根？

**面试题：** 请描述一种计算多项式根的方法，并给出一个具体实现的例子。

**答案解析：** 计算多项式根的方法有很多，包括因式分解、求根公式、牛顿法等。以下是一个基于求根公式的实现：

```python
import cmath

def find_roots(a, b, c):
    # 计算判别式
    discriminant = b**2 - 4*a*c
    # 计算两个根
    root1 = (-b + cmath.sqrt(discriminant)) / (2 * a)
    root2 = (-b - cmath.sqrt(discriminant)) / (2 * a)
    return root1, root2

# 示例
a = 1
b = -6
c = 11
root1, root2 = find_roots(a, b, c)
print("Roots:", root1, root2)
```

#### 3. 如何确定一个多项式是否有实数根？

**面试题：** 请给出一个判断多项式是否有实数根的方法。

**答案解析：** 一个多项式是否有实数根，可以通过计算其判别式来判断。对于一元二次多项式 \( ax^2 + bx + c = 0 \)，判别式 \( \Delta = b^2 - 4ac \)：

- 如果 \( \Delta > 0 \)，则有两个不相等的实数根。
- 如果 \( \Delta = 0 \)，则有一个实数根。
- 如果 \( \Delta < 0 \)，则没有实数根。

以下是一个简单实现的例子：

```python
def has_real_roots(a, b, c):
    discriminant = b**2 - 4*a*c
    return discriminant >= 0

# 示例
a = 1
b = -5
c = 6
if has_real_roots(a, b, c):
    print("多项式有实数根")
else:
    print("多项式没有实数根")
```

**答案解析：** 判断两个多项式是否有共同根，可以通过以下方法：

1. 将两个多项式相减，得到一个新的多项式。
2. 如果新的多项式有重根，则原两个多项式有共同根。

例如，对于 \( P(x) = x^2 - 5x + 6 \) 和 \( Q(x) = x^2 - 4x + 4 \)，相减得到 \( R(x) = x - 2 \)，显然 \( R(x) \) 有重根 \( x = 2 \)，因此 \( P(x) \) 和 \( Q(x) \) 有共同根。

```python
import cmath

def quadratic_formula(a, b, c):
    discriminant = b**2 - 4*a*c
```

```python
def multiply_polynomials(p1, p2):
    # p1 和 p2 是两个多项式的系数列表
    result = [0] * (len(p1) + len(p2) - 1)
    for i in range(len(p1)):
        for j in range(len(p2)):
            result[i + j] += p1[i] * p2[j]
    return result

# 示例

print("Product:", result)
```

```python
def derivative(p):
    # p 是多项式的系数列表
    result = [0] * (len(p) - 1)
    for i in range(1, len(p)):
        result[i - 1] = i * p[i]
    return result
```
