**题目：** 请给出知识付费产品的用户留存率计算公式。

**答案：** 用户留存率（Retention Rate）是衡量知识付费产品用户保持活跃度的关键指标。其计算公式为：

\[ 用户留存率 = \left( \frac{第n天仍然活跃的用户数}{第n-1天仍然活跃的用户数} \right) \times 100\% \]

**解析：** 该公式反映了第n天仍然活跃的用户数与第n-1天仍然活跃的用户数之间的比例关系。用户留存率越高，说明用户对产品的依赖度和忠诚度越高。

**示例：** 假设第1天有100个用户注册，第2天有80个用户仍然活跃，第3天有60个用户仍然活跃，那么第2天的用户留存率为：

\[ 用户留存率 = \left( \frac{60}{100} \right) \times 100\% = 60\% \]

### 2. 如何定义用户复购率？

**题目：** 请说明知识付费产品的用户复购率的定义。

**答案：** 用户复购率（Repeat Purchase Rate）是指在一定时间内，曾购买过知识付费产品用户再次购买的比例。其计算公式为：

\[ 用户复购率 = \left( \frac{在一定时间内再次购买的用户数}{曾购买过的用户数} \right) \times 100\% \]

**解析：** 用户复购率是衡量知识付费产品用户忠诚度和产品质量的重要指标。高复购率意味着用户对产品满意，具有较高的购买意愿。

**示例：** 假设在一个季度内，有1000个用户曾购买过知识付费产品，其中有200个用户在这个季度再次购买，那么该季度的用户复购率为：

\[ 用户复购率 = \left( \frac{200}{1000} \right) \times 100\% = 20\% \]
