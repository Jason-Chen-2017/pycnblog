#### 1. 最长公共子序列
**题目：** 给定两个字符串，找出它们的最长公共子序列。

**答案：** 使用动态规划方法求解。

```go
func longestCommonSubsequence(s1 string, s2 string) int {
    m, n := len(s1), len(s2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if s1[i-1] == s2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

**解析：** 动态规划通过填充一个二维数组`dp`来记录最长的公共子序列长度，时间复杂度为O(m*n)。

#### 2. 单词搜索
**题目：** 给定一个二维字符网格和一个单词，判断该单词是否存在于网格中。

**答案：** 使用深度优先搜索（DFS）方法。

```go
func exist(board [][]byte, word string) bool {
    m, n := len(board), len(board[0])
    visited := make([][]bool, m)
    for i := range visited {
        visited[i] = make([]bool, n)
    }
    for i := 0; i < m; i++ {
        for j := 0; j < n; j++ {
            if dfs(board, word, i, j, 0, visited) {
                return true
            }
        }
    }
    return false

func dfs(board [][]byte, word string, i, j, k int, visited [][]bool) bool {
    if k == len(word) {
        return true
    }
    if i < 0 || i >= len(board) || j < 0 || j >= len(board[0]) || visited[i][j] || board[i][j] != word[k] {
        return false
    }
    visited[i][j] = true
    if dfs(board, word, i+1, j, k+1, visited) ||
        dfs(board, word, i-1, j, k+1, visited) ||
        dfs(board, word, i, j+1, k+1, visited) ||
        dfs(board, word, i, j-1, k+1, visited) {
        return true
    }
    visited[i][j] = false
    return false
}
```

**解析：** DFS方法通过递归遍历所有可能的路径，检查是否匹配目标单词，避免重复访问。

#### 3. 删除链表的节点
**题目：** 给定一个单链表和一个节点，删除该节点。

**答案：** 通过修改节点值来删除节点。

```go
func deleteNode(node *ListNode) {
    if node == nil || node.Next == nil {
        return
    }
    node.Val = node.Next.Val
    node.Next = node.Next.Next
}
```

**解析：** 直接将当前节点的值设置为下一个节点的值，并将下一个节点指向下下个节点，从而删除当前节点。

### 二、编程题解析

#### 1. 找出数组中的重复元素
**题目：** 给定一个整数数组，找出其中的重复元素。

**答案：** 使用哈希表或排序后线性查找。

```go
func findDuplicates(nums []int) []int {
    m := make(map[int]int)
    var res []int
    for _, num := range nums {
        if m[num] > 0 {
            res = append(res, num)
        }
        m[num]++
    }
    return res
}

func findDuplicatesSorted(nums []int) []int {
    var res []int
    for i := 0; i < len(nums); i++ {
        for nums[i] != i {
            correct := nums[i]
            if nums[i] == nums[correct] {
                res = append(res, nums[i])
                break
            }
            nums[i], nums[correct] = nums[correct], nums[i]
        }
    }
    return res
}
```

**解析：** 第一个函数使用哈希表记录每个元素的计数，第二个函数通过交换元素到正确的位置来找到重复元素。

#### 2. 合并区间
**题目：** 给定一组区间，合并重叠的区间。

**答案：** 对区间排序后，合并重叠的区间。

```go
func merge(intervals [][]int) [][]int {
    var res [][]int
    sort.Slice(intervals, func(i, j int) bool {
        return intervals[i][0] < intervals[j][0]
    })
    for _, interval := range intervals {
        if len(res) == 0 || res[len(res)-1][1] < interval[0] {
            res = append(res, interval)
        } else {
            res[len(res)-1][1] = max(res[len(res)-1][1], interval[1])
        }
    }
    return res
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

**解析：** 首先对区间进行排序，然后合并相邻的区间。

#### 3. 计数质数
**题目：** 计算小于或等于`n`的质数数量。

**答案：** 使用埃拉托斯特尼筛法（Sieve of Eratosthenes）。

```go
func countPrimes(n int) int {
    if n < 2 {
        return 0
    }
    isPrime := make([]bool, n)
    for i := range isPrime {
        isPrime[i] = true
    }
    for i := 2; i < int(math.Sqrt(float64(n))); i++ {
        if isPrime[i] {
            for j := i * i; j < n; j += i {
                isPrime[j] = false
            }
        }
    }
    return sum(isPrime[:n])
}

func sum(nums []int) int {
    result := 0
    for _, num := range nums {
        result += num
    }
    return result
}
```

**解析：** 埃拉托斯特尼筛法通过标记非质数来找出所有质数。
