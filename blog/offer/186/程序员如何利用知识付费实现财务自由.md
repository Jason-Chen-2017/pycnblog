### 1. 算法时间复杂度分析

**题目：** 简述时间复杂度分析的方法，并分析以下代码的时间复杂度。

```python
def find_max(arr):
    max_val = arr[0]
    for i in range(1, len(arr)):
        if arr[i] > max_val:
            max_val = arr[i]
    return max_val
```

**答案：** 时间复杂度分析的方法主要是观察算法中基本操作（如循环、递归）的执行次数与数据规模的关系。对于上述代码，基本操作是 `if` 判断，执行次数为 `n-1`（其中 `n` 为数组长度），所以时间复杂度为 O(n)。

### 2. 数据结构的应用

**题目：** 简述栈和队列的应用场景，并分别给出一个示例。

**答案：** 栈（Stack）适用于后进先出（LIFO）的场景，如递归调用、后退功能等。示例：

```python
class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        return self.items.pop()

# 示例：递归调用
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)
```

队列（Queue）适用于先进先出（FIFO）的场景，如打印任务、队列处理等。示例：

```python
class Queue:
    def __init__(self):
        self.items = []

    def enqueue(self, item):
        self.items.append(item)

    def dequeue(self):
        return self.items.pop(0)

# 示例：打印任务
print_queue = Queue()
print_queue.enqueue("任务1")
print_queue.enqueue("任务2")
print_queue.enqueue("任务3")
while not print_queue.is_empty():
    task = print_queue.dequeue()
    print(task)
```

### 3. 合并两个有序链表

**题目：** 给定两个有序链表，将它们合并为一个有序链表。要求不使用额外空间，时间复杂度为 O(n)。

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_sorted_lists(l1, l2):
    # 你的代码实现
```

**答案：** 

```python
def merge_sorted_lists(l1, l2):
    dummy = ListNode()
    curr = dummy
    while l1 and l2:
        if l1.val < l2.val:
            curr.next = l1
            l1 = l1.next
        else:
            curr.next = l2
            l2 = l2.next
        curr = curr.next
    curr.next = l1 or l2
    return dummy.next
```

**解析：** 这个算法采用两个指针遍历两个链表，比较当前节点值，将较小的值链接到新链表中。当其中一个链表遍历完毕，直接将另一个链表的剩余部分链接到新链表后面。

### 4. 二分查找

**题目：** 实现一个二分查找函数，用于在一个有序数组中查找一个目标值，并返回其索引。如果没有找到，返回 -1。

```python
def binary_search(arr, target):
    # 你的代码实现
```

**答案：**

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

**解析：** 这个算法通过不断缩小区间来查找目标值，时间复杂度为 O(log n)。
