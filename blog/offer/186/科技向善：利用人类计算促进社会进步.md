#### 2. 百度面试题 - 文本相似度计算

**题目：** 编写一个文本相似度计算函数，比较两个文本的相似度。

**答案：**
```python
def jaccard_similarity(text1, text2):
    intersection = set(text1).intersection(set(text2))
    union = set(text1).union(set(text2))
    return len(intersection) / len(union)

text1 = "科技向善"
text2 = "社会进步"
similarity = jaccard_similarity(text1, text2)
print(similarity)
```

**解析：** Jaccard相似度是文本相似度计算的一种常用方法，通过计算两个文本的交集和并集的比例来衡量相似度。

#### 5. 京东面试题 - 搜索引擎中的词频统计

**题目：** 实现一个搜索引擎，统计文档中各个单词的频率。

**答案：**
```python
from collections import Counter

def word_frequency(document):
    words = document.split()
    return Counter(words)

document = "科技向善利用人类计算促进社会进步"
freq = word_frequency(document)
print(freq)
```

**解析：** 词频统计是搜索引擎的基础，通过统计文本中各个单词的出现频率来优化搜索结果。
