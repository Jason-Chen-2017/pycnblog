### 心理健康支持系统的算法编程题库

#### 4. 实现一个情绪分类算法

**题目：** 使用 Python 实现 emotion_classifier 函数，能够根据文本输入将其分类为情绪类别。

**答案：** 可以使用 sklearn 库中的朴素贝叶斯分类器实现情绪分类：

```python
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

def emotion_classifier(text):
    # 加载训练数据和标签
    texts = ['text1', 'text2', ...]
    labels = ['happy', 'sad', ...]

    # 特征提取
    vectorizer = CountVectorizer()
    X = vectorizer.fit_transform(texts)

    # 模型训练
    model = MultinomialNB()
    model.fit(X, labels)

    # 分类
    prediction = model.predict(vectorizer.transform([text]))
    return prediction[0]
```

**解析：** 该算法首先进行文本特征提取，然后使用朴素贝叶斯分类器进行训练，最后根据输入文本进行情绪分类。

#### 5. 实现一个情绪分析 API

**题目：** 使用 Flask 实现一个情绪分析 API，接受用户输入的文本，返回其情绪类别。

**答案：** 可以使用 Flask 框架实现情绪分析 API：

```python
from flask import Flask, request, jsonify
from emotion_classifier import emotion_classifier

app = Flask(__name__)

@app.route('/analyze', methods=['POST'])
def analyze():
    text = request.form['text']
    emotion = emotion_classifier(text)
    return jsonify({'emotion': emotion})

if __name__ == '__main__':
    app.run()
```

**解析：** 该 API 接受 POST 请求，从请求体中获取文本输入，调用情绪分类函数，并将结果以 JSON 格式返回。
