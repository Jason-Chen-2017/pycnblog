### 1. 脑机接口的基本原理是什么？

**题目：** 请简要解释脑机接口（Brain-Computer Interface, BCI）的基本原理。

**答案：** 脑机接口是一种通过直接连接大脑和外部设备来传递信息的界面。其基本原理是利用大脑产生的电信号，通过传感器捕获并转换成计算机可以理解和处理的数据。常见的 BCIs 包括基于脑电图（EEG）、功能性磁共振成像（fMRI）、脑磁图（MEG）等。

**举例：**

```python
# 使用 EEG 捕获脑电信号
from mne import EEG

eeg = EEG()
signal = eeg.record(duration=5) # 记录 5 秒的脑电信号
```

**解析：** 在这个例子中，我们使用 Python 的 MNE 库来捕获脑电信号。通过调用 `record` 方法，可以获取一段时间内的脑电信号数据。
