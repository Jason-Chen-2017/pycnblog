#### 4. 算法编程实例与代码解析

**实例 1：** 实现一个简单的协同过滤推荐系统，根据学生的历史成绩推荐相关课程。

```python
import numpy as np

# 假设学生-课程评分矩阵为 S，其中 S[i][j] 表示学生 i 对课程 j 的评分
S = np.array([[3, 2, 1, 0], [1, 0, 3, 2], [2, 3, 0, 1], [0, 2, 1, 3]])

# 计算学生之间的相似度
def cosine_similarity(rating1, rating2):
    return np.dot(rating1, rating2) / (np.linalg.norm(rating1) * np.linalg.norm(rating2))

def collaborative_filtering(S, user_id, k=2):
    user_rating = S[user_id]
    similar_users = {}
    for i, rating in enumerate(S):
        if i == user_id:
            continue
        similarity = cosine_similarity(user_rating, rating)
        similar_users[i] = similarity
    
    # 根据相似度排序，选择 k 个最相似的用户
    top_k = sorted(similar_users.items(), key=lambda x: x[1], reverse=True)[:k]
    
    # 计算推荐列表
    recommendation = []
    for user, similarity in top_k:
        for j, rating in enumerate(S[user]):
            if S[user_id][j] == 0:
                recommendation.append((j, similarity * rating))
    
    # 按相似度和评分降序排序
    recommendation.sort(key=lambda x: x[1], reverse=True)
    
    return recommendation

# 为学生 2 推荐课程
user_id = 2
recommendation = collaborative_filtering(S, user_id)
print("推荐课程：", recommendation)
```

**解析：**
这个实例使用余弦相似度计算学生之间的相似度，然后根据相似度排序选择最相似的 k 个学生，并计算他们共同喜欢的课程。最后，按照相似度和评分降序排序，输出推荐课程。
