### 2.1 GAN原理

**题目：** 简述生成对抗网络（GAN）的基本原理。

**答案：** 生成对抗网络（GAN）由两个神经网络组成：生成器（Generator）和判别器（Discriminator）。生成器从随机噪声中生成数据，判别器则判断输入数据是真实数据还是生成器生成的数据。训练过程中，生成器和判别器互相博弈，生成器的目标是生成越来越逼真的数据，而判别器的目标是提高对真实数据和生成数据的区分能力。

**代码示例：**

    # 从噪声生成图像

    # 判断图像是否真实

### 2.2 VAE原理

**题目：** 简述变分自编码器（VAE）的基本原理。

**答案：** 变分自编码器（VAE）通过编码器（Encoder）和解码器（Decoder）学习数据的高效表示。编码器将输入数据映射到一个隐含空间中的均值和方差，解码器则根据这些参数生成新的数据。VAE通过最大化数据重排概率来训练，从而实现数据的生成。

**代码示例：**
```python
# VAE框架
import tensorflow as tf
from tensorflow.keras.layers import Input, Dense, Lambda
from tensorflow.keras.models import Model

# 定义编码器和解码器
def vae_encoder(x):
    # 编码器网络
    return tf.keras.layers.Dense(units=20, activation=tf.nn.relu)(x)

def vae_decoder(z):
    # 解码器网络
    return tf.keras.layers.Dense(units=784, activation=tf.nn.sigmoid)(z)

# 定义VAE模型
input_img = Input(shape=(784,))
h = vae_encoder(input_img)
z_mean = Dense(units=20)(h)
z_log_var = Dense(units=20)(h)
z = Lambda ShapiroMF(z_mean, z_log_var)(h)
decoded = vae_decoder(z)

vae = Model(input_img, decoded)
```

### 2.3 图像生成效果评估

**题目：** 如何评估图像生成效果？

**答案：** 图像生成效果的评估通常采用以下指标：

* **生成质量（Image Quality）：** 使用峰值信噪比（PSNR）和结构相似性（SSIM）等指标衡量生成图像的质量。
* **多样性（Diversity）：** 通过计算生成图像之间的相似性来评估多样性。
* **真实感（Realism）：** 使用人类评估或自动评估方法（如Inception Score，IS）来判断生成图像的真实感。
