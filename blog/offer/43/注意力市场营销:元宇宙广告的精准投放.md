**2. 编写一个函数，计算给定广告点击率的置信区间。**

**答案：** 可以使用以下函数：

```python
import math

def confidence_interval@click_rate, confidence_level=0.95):
    n = len(click_rate)
    p_hat = sum(click_rate) / n
    z_score = abs(math.erfInverse((1 + confidence_level / 2) - p_hat))
    margin_of_error = z_score * math.sqrt(p_hat * (1 - p_hat) / n)
    lower_bound = p_hat - margin_of_error
    upper_bound = p_hat + margin_of_error
    return lower_bound, upper_bound

# 示例
click_rate = [10, 20, 30, 40, 50]
confidence_level = 0.95
print(confidence_interval(click_rate, confidence_level))
```

**解析：** 这是一个统计学中的计算题，涉及置信区间的计算。
