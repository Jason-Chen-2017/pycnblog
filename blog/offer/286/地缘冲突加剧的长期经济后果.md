#### 面试题一：经济制裁的影响

**题目：** 经济制裁对于国际贸易和国内经济发展有哪些影响？

**答案解析：**

1. **对国际贸易的影响：**
   - **贸易壁垒增加：** 经济制裁可能导致被制裁国家的贸易壁垒增加，从而降低国际贸易的流动性和效率。
   - **贸易额下降：** 被制裁国家与其他国家的贸易额可能会下降，影响全球贸易总额。
   - **货币汇率波动：** 经济制裁可能导致相关货币汇率波动，从而影响国际贸易的成本和收益。

2. **对国内经济发展的影响：**
   - **投资下降：** 经济制裁可能导致外资对被制裁国家的投资下降，影响国内经济增长。
   - **产业链断裂：** 被制裁国家的关键产业可能受到制裁影响，导致产业链断裂，影响国内产业链的稳定发展。
   - **就业压力增加：** 经济制裁可能导致就业机会减少，增加国内就业压力。

**代码示例：**

```go
package main

import "fmt"

func main() {
    // 假设国内GDP为100亿
    gdp := 100

    // 经济制裁导致的贸易额下降
    tradeDecline := 10
    gdp -= tradeDecline

    // 投资下降
    investmentDecline := 5
    gdp -= investmentDecline

    // 产业链断裂
    industryBreakdown := 3
    gdp -= industryBreakdown

    // 就业压力增加
    employmentPressure := 2
    gdp -= employmentPressure

    fmt.Printf("受经济制裁影响，国内GDP预计下降至：%.2f亿\n", gdp)
}
```
