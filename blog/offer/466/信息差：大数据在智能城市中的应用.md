```python
import numpy as np

kmeans.fit(traffic_data)

else:
    green_time = 45  # 信号灯绿灯时长
```

**解析：** 代码示例中，首先使用KMeans算法对交通流量数据进行聚类，根据聚类结果调整信号灯时长，从而优化交通流量。
