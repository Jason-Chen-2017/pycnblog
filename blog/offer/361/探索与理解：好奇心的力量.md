## 7. 面试题：排序算法

### 7.1. 快速排序算法

**题目：** 请简要描述快速排序算法的原理和步骤，并给出代码实现。

**答案：**

快速排序是一种高效的排序算法，其原理是通过一趟排序将待排序的记录分割成独立的两部分，其中一部分记录的关键字均比另一部分的关键字小，然后分别对这两部分记录继续进行排序，以达到整个序列有序。

**步骤：**

1. 选择一个基准元素（pivot）。
2. 将序列划分为两部分，一部分的元素都比基准元素小，另一部分都比基准元素大。
3. 递归地对两部分序列进行快速排序。

**代码实现：**

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    return quicksort(left) + middle + quicksort(right)

arr = [3, 6, 8, 10, 1, 2, 1]
print(quicksort(arr))
```

**解析：** 在这个实现中，我们选择序列的中间元素作为基准元素。然后使用列表推导式将序列划分为小于、等于和大于基准元素的三个部分，并对这三个部分分别进行快速排序。

### 7.2. 冒泡排序算法

**题目：** 请简要描述冒泡排序算法的原理和步骤，并给出代码实现。

**答案：**

冒泡排序是一种简单的排序算法，其原理是通过重复遍历序列，每次比较相邻的两个元素，如果它们的顺序错误就把它们交换过来。遍历序列的工作重复进行，直到没有再需要交换的元素。

**步骤：**

1. 从头到尾遍历序列，比较相邻的元素，如果顺序错误则交换。
2. 从最后一个元素开始到头遍历序列，重复上述过程，直到没有再需要交换的元素。
3. 重复上述过程，直到整个序列有序。

**代码实现：**

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

arr = [64, 34, 25, 12, 22, 11, 90]
bubble_sort(arr)
print("Sorted array:", arr)
```

**解析：** 在这个实现中，我们使用两个嵌套的循环。外层循环控制遍历的轮数，内层循环执行实际的比较和交换操作。

### 7.3. 插入排序算法

**题目：** 请简要描述插入排序算法的原理和步骤，并给出代码实现。

**答案：**

插入排序是一种简单的排序算法，其原理是通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。

**步骤：**

1. 从第一个元素开始，该元素可以认为已经被排序。
2. 取出下一个元素，在已排序的元素序列中从后向前扫描。
3. 如果该元素（已排序）大于新元素，将该元素移到下一位置。
4. 重复步骤3，直到找到已排序的元素小于或者等于新元素的位置。
5. 将新元素插入到该位置后。
6. 重复步骤2~5。

**代码实现：**

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

arr = [12, 11, 13, 5, 6]
insertion_sort(arr)
print("Sorted array:", arr)
```

**解析：** 在这个实现中，我们使用一个内层循环来搜索已排序序列中适当的位置来插入新元素，同时使用外层循环来遍历未排序序列。

## 8. 算法编程题：查找算法

### 8.1. 二分查找

**题目：** 给定一个排序后的数组，找到目标元素的位置。如果没有找到，返回-1。请实现二分查找算法。

**答案：**

二分查找是一种高效的查找算法，其原理是将序列中间元素与目标元素比较，根据比较结果将序列分成两部分，排除一部分，然后在剩余的序列中继续查找，直到找到目标元素或确定其不存在。

**代码实现：**

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

arr = [1, 3, 5, 7, 9, 11, 13]
target = 7
print(binary_search(arr, target))
```

**解析：** 在这个实现中，我们使用while循环来不断缩小区间，直到找到目标元素或确定其不存在。

### 8.2. 哈希查找

**题目：** 实现一个哈希表，支持插入、删除和查找操作。

**答案：**

哈希查找是通过哈希函数将关键字映射到哈希表中，以实现快速查找。在实现哈希表时，我们通常需要处理哈希冲突，常见的解决方法有链地址法、开放地址法和再散列法。

**代码实现（链地址法）：**

```python
class HashTable:
    def __init__(self):
        self.size = 10
        self.table = [[] for _ in range(self.size)]

    def hash_function(self, key):
        return key % self.size

    def insert(self, key, value):
        hash_key = self.hash_function(key)
        bucket = self.table[hash_key]
        for i, (k, v) in enumerate(bucket):
            if k == key:
                bucket[i] = (key, value)
                return
        bucket.append((key, value))

    def delete(self, key):
        hash_key = self.hash_function(key)
        bucket = self.table[hash_key]
        for i, (k, v) in enumerate(bucket):
            if k == key:
                del bucket[i]
                return
        return -1

    def find(self, key):
        hash_key = self.hash_function(key)
        bucket = self.table[hash_key]
        for k, v in bucket:
            if k == key:
                return v
        return -1

hash_table = HashTable()
hash_table.insert(1, "value1")
hash_table.insert(11, "value11")
print(hash_table.find(11))  # 输出 "value11"
hash_table.delete(11)
print(hash_table.find(11))  # 输出 -1
```

**解析：** 在这个实现中，我们使用链地址法处理哈希冲突。每个哈希桶是一个列表，用于存储映射到该桶的键值对。
