### 典型面试题库

#### 1. 请解释并发和并行之间的区别。

**答案：** 并发（Concurrent）是指在多个任务交替执行时，系统看起来像是同时执行。并行（Parallel）是指多个任务同时执行。并发可以在单核处理器上通过上下文切换实现，而并行通常需要多核处理器或特殊硬件支持。

#### 2. 什么是协程（goroutine）？请简要说明如何使用协程处理并发任务。

**答案：** 协程是 Go 语言中的轻量级线程，它可以在单个线程内并发执行。使用协程可以简化并发编程，通过 `go` 关键字启动新的协程。

```go
func main() {
    go func() {
        // 在新的协程中执行任务
    }()
}
```

#### 3. 什么是内存泄漏？请描述如何检测和避免内存泄漏。

**答案：** 内存泄漏是指程序在运行过程中，不再需要的内存没有被释放，导致内存占用逐渐增加。检测内存泄漏可以使用内存分析工具，如 Valgrind。避免内存泄漏的方法包括合理使用对象生命周期、及时释放不再使用的资源等。

#### 4. 请解释函数式编程和面向对象编程之间的区别。

**答案：** 函数式编程（Functional Programming）是一种编程范式，强调使用函数作为主要构建块，避免使用状态和变量。面向对象编程（Object-Oriented Programming）则是一种编程范式，基于对象和类的概念，通过封装、继承和多态实现代码的重用。

#### 5. 什么是回调函数？请举例说明。

**答案：** 回调函数（Callback Function）是在另一个函数中定义并调用的函数。回调函数允许在执行特定任务时，将控制权交给外部函数。例如，在 JavaScript 中，可以使用回调函数处理事件响应。

```javascript
document.addEventListener("click", function() {
    // 处理点击事件
});
```

### 算法编程题库

#### 1. 斐波那契数列

**题目：** 编写一个函数，使用递归方法计算斐波那契数列的第 n 项。

**答案：**

```python
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```

#### 2. 快速排序

**题目：** 编写一个快速排序函数，对数组进行排序。

**答案：**

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)
```

#### 3. 单词搜索

**题目：** 给定一个二维字符网格和一个单词，编写一个函数来查找网格中单词是否可以横纵相连构成。

**答案：**

```python
def exist(board, word):
    def dfs(i, j, k):
        if k == len(word):
            return True
        if (
            i < 0
            or i >= len(board)
            or j < 0
            or j >= len(board[0])
            or board[i][j] != word[k]
        ):
            return False
        board[i][j] = "#"
        for dx, dy in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
            if dfs(i + dx, j + dy, k + 1):
                return True
        board[i][j] = word[k]
        return False

    for i in range(len(board)):
        for j in range(len(board[0])):
            if dfs(i, j, 0):
                return True
    return False
```
