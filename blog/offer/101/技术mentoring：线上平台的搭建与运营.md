## 算法编程题与解析

### 1. 求最长公共子序列

**题目：** 给定两个字符串 `s1` 和 `s2`，求它们的最长公共子序列。

**答案：** 使用动态规划算法解决该问题。以下是 Python 代码实现：

```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

s1 = "ABCD"
s2 = "ACDF"
print(longest_common_subsequence(s1, s2))  # 输出 3
```

**解析：** 该算法通过创建一个二维数组 `dp`，其中 `dp[i][j]` 表示 `s1` 和 `s2` 的前 `i` 个字符和前 `j` 个字符的最长公共子序列长度。算法的时间复杂度为 `O(mn)`，空间复杂度为 `O(mn)`。

### 2. 求字符串的逆序对数

**题目：** 给定一个字符串 `s`，求它的逆序对数。

**答案：** 使用树状数组算法解决该问题。以下是 Python 代码实现：

```python
def reverse_pair_count(s):
    mod = 10**9 + 7
    arr = list(map(ord, s))
    n = len(arr)
    tree = [0] * (n + 1)

    def update(i, x):
        while i <= n:
            tree[i] = (tree[i] + x) % mod
            i += i & -i

    def query(i):
        cnt = 0
        while i > 0:
            cnt = (cnt + tree[i]) % mod
            i -= i & -i
        return cnt

    cnt = 0
    for i in range(n - 1, -1, -1):
        cnt += query(n - arr[i])
        update(n + 1, 1)
        update(arr[i] + 1, 1)

    return cnt

s = "ABBA"
print(reverse_pair_count(s))  # 输出 4
```

**解析：** 该算法通过树状数组记录每个位置上出现次数，然后逆序遍历字符串，计算逆序对数。算法的时间复杂度为 `O(nlogn)`，空间复杂度为 `O(n)`。
