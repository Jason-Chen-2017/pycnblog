##### 4. 词频统计

**题目：** 编写一个函数，计算一个字符串中各个单词的词频。

**答案：** 

```python
def word_frequency(s):
    words = s.split()
    freq = {}
    for word in words:
        if word in freq:
            freq[word] += 1
        else:
            freq[word] = 1
    return freq

s = "hello world hello"
print(word_frequency(s))
```

**解析：** 该函数使用字典存储词频，遍历字符串中的单词，更新词频。

##### 5. 文本相似度计算

**题目：** 编写一个函数，计算两个文本的相似度。

**答案：** 

```python
from sklearn.metrics.pairwise import cosine_similarity
from sklearn.feature_extraction.text import CountVectorizer

def text_similarity(text1, text2):
    vectorizer = CountVectorizer().fit_transform([text1, text2])
    vectors = vectorizer.toarray()
    return cosine_similarity(vectors[0:1], vectors[1:2])[0][0]

text1 = "我爱北京天安门"
text2 = "天安门上太阳升"
print(text_similarity(text1, text2))
```

**解析：** 该函数使用词袋模型和余弦相似度计算文本相似度。

##### 6. 搜索引擎关键词提取

**题目：** 编写一个函数，提取一个字符串中的关键词。

**答案：**

```python
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

def keyword_extraction(s):
    words = word_tokenize(s)
    stop_words = set(stopwords.words('english'))
    filtered_words = [word for word in words if not word in stop_words]
    return filtered_words

s = "技术进步：不断提升搜索质量"
print(keyword_extraction(s))
```

**解析：** 该函数使用 NLTK 库提取关键词，并去除停用词。
