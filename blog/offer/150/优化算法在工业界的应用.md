## 1. 生产调度问题

### 1.1 题目

某工厂有 N 台机器，每台机器都可以同时加工多个任务。每个任务有特定的加工时间和优先级。如何设计一个调度算法，使得所有任务能在最短的时间内完成？

### 1.2 算法编程题

请编写一个函数 `schedule(tasks []Task)`，其中 `Task` 是一个结构体，包含任务的加工时间和优先级。函数返回一个调度序列，使得所有任务能在最短的时间内完成。

### 1.3 答案解析

我们可以使用贪心算法来解决这个问题。首先将所有任务按照优先级从高到低排序。然后从高优先级开始，依次将任务分配给空闲的机器，直到所有任务都分配完毕。

```go
type Task struct {
    time int
    priority int
}

func schedule(tasks []Task) []int {
    sort.Slice(tasks, func(i, j int) bool {
        return tasks[i].priority > tasks[j].priority
    })

    machines := make([]int, len(tasks))
    for i, task := range tasks {
        machine := findFreeMachine(machines)
        machines[i] = machine
    }

    return machines
}

func findFreeMachine(machines []int) int {
    for i, machine := range machines {
        if machine == 0 {
            return i + 1
        }
    }
    return len(machines) + 1
}
```
