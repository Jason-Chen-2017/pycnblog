#### 1. 使用K-means算法进行生活质量聚类分析。

**题目描述：** 假设你有以下生活质量数据，使用K-means算法对其进行聚类分析，并输出每个簇的中心点和成员。

```python
lifedata = [
    [80, 100, 85, 3],
    [90, 110, 70, 4],
    [85, 95, 80, 2],
    [75, 95, 90, 1],
    [100, 120, 60, 3],
    [95, 105, 70, 4],
    [90, 100, 85, 2],
    [85, 90, 80, 1],
    [80, 85, 75, 3],
    [95, 100, 65, 4]
]
```

**答案解析：** 实现代码如下：

```python
from sklearn.cluster import KMeans
import numpy as np

lifedata = np.array(lifedata)
kmeans = KMeans(n_clusters=3)
kmeans.fit(lifedata)
print("Cluster centers:", kmeans.cluster_centers_)
print("Cluster assignments:", kmeans.labels_)

```
