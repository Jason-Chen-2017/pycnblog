#### 详解解析：

**1. 最长公共子序列（LCS）**

**题目描述：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。

**答案解析：** 可以使用动态规划（DP）方法求解。定义一个二维数组 `dp`，其中 `dp[i][j]` 表示字符串 `str1` 的前 `i` 个字符与字符串 `str2` 的前 `j` 个字符的最长公共子序列的长度。状态转移方程如下：

```python
if str1[i-1] == str2[j-1]:
    dp[i][j] = dp[i-1][j-1] + 1
else:
    dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

**代码示例：**

```python
def longestCommonSubsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```

**2. 最小生成树（MST）**

**题目描述：** 给定一个无向图，求出其中的最小生成树。

**答案解析：** 可以使用Prim算法或Kruskal算法求解。这里以Prim算法为例。Prim算法的基本思想是从一个顶点开始，逐步扩展生成树，直到包含所有顶点。

**代码示例：**

```python
from collections import defaultdict

def prim(graph):
    n = len(graph)
    parent = [-1] * n
    key = [float('inf')] * n
    mst = []

    key[0] = 0
    in_mst = [False] * n

    for _ in range(n):
        u = -1
        min_key = float('inf')
        for i in range(n):
            if not in_mst[i] and key[i] < min_key:
                min_key = key[i]
                u = i
        in_mst[u] = True
        mst.append(u)

        for v in graph[u]:
            if not in_mst[v] and graph[u][v] < key[v]:
                key[v] = graph[u][v]
                parent[v] = u

    return mst
```
