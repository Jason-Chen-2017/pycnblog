
**解析：** 首先对读数数组进行排序，然后根据数组长度判断中位数是单数还是双数，单数返回中间的元素，双数返回中间两个元素的平均值。

**答案：**

```python
def findMedian(distanceReadings):
    distanceReadings.sort()
    length = len(distanceReadings)
    if length % 2 == 0:
        return (distanceReadings[length // 2 - 1] + distanceReadings[length // 2]) / 2
    else:
        return distanceReadings[length // 2]

# 示例
print(findMedian([1, 3, 2])) # 输出 2
print(findMedian([1, 3, 4, 2])) # 输出 2.5
```

```python
def detectAnomalies(sensorData):
```

```python
def predictTemperatureChange(temperatureReadings):
```

```python
def optimizeEnergyConsumption(sensorData):
    n = len(sensorData)
    dp = [0] * (n + 1)
```
