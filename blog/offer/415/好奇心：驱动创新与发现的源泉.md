#### 3.1 阿里巴巴面试题：实现一个函数，判断一个整数是否是素数

**题目：** 实现一个函数 `isPrime(n int) bool`，判断一个整数 `n` 是否是素数。

**答案：** 通过试除法来判断一个整数是否是素数。从 2 开始，依次尝试除以小于等于根号 `n` 的所有整数，如果无法整除，则 `n` 是素数。

**代码实现：**

```go
package main

import (
    "fmt"
    "math"
)

func isPrime(n int) bool {
    if n <= 1 {
        return false
    }
    for i := 2; i <= int(math.Sqrt(float64(n))); i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

func main() {
    fmt.Println(isPrime(17)) // 输出 true
    fmt.Println(isPrime(18)) // 输出 false
}
```

**解析：** 通过好奇心去研究素数的判断方法，我们掌握了试除法这一简单有效的算法，从而能够快速判断一个整数是否是素数。

#### 3.2 百度面试题：最长公共子序列

**题目：** 给定两个字符串 `s1` 和 `s2`，求它们的最长公共子序列。

**答案：** 使用动态规划算法求解。定义一个二维数组 `dp`，其中 `dp[i][j]` 表示 `s1` 的前 `i` 个字符和 `s2` 的前 `j` 个字符的最长公共子序列的长度。

**代码实现：**

```go
package main

import (
    "fmt"
)

func longestCommonSubsequence(s1, s2 string) int {
    dp := make([][]int, len(s1)+1)
    for i := range dp {
        dp[i] = make([]int, len(s2)+1)
        for j := range dp[i] {
            if s1[i-1] == s2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[len(s1)][len(s2)]
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}

func main() {
    fmt.Println(longestCommonSubsequence("abcde", "ace")) // 输出 3
    fmt.Println(longestCommonSubsequence("abc", "abc"))   // 输出 3
}
```

**解析：** 通过好奇心去研究最长公共子序列的算法，我们掌握了动态规划算法，从而能够求解两个字符串的最长公共子序列。

#### 3.3 腾讯面试题：二分查找

**题目：** 给定一个有序数组 `nums` 和一个目标值 `target`，在数组中查找目标值并返回其索引。

**答案：** 使用二分查找算法。在每次迭代中，比较目标值与数组中间元素的值，根据比较结果决定是否继续在左半部分或右半部分查找。

**代码实现：**

```go
package main

import (
    "fmt"
)

func search(nums []int, target int) int {
    left, right := 0, len(nums)-1
    for left <= right {
        mid := left + (right-left)/2
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}

func main() {
    fmt.Println(search([]int{1, 3, 5, 6}, 5)) // 输出 2
    fmt.Println(search([]int{1, 3, 5, 6}, 7)) // 输出 -1
}
```

**解析：** 通过好奇心去研究二分查找算法，我们掌握了如何通过不断缩小区间来提高查找效率，从而能够快速在有序数组中查找目标值。

#### 3.4 字节跳动面试题：最长公共前缀

**题目：** 编写一个函数来查找字符串数组中的最长公共前缀。

**答案：** 使用垂直扫描算法。从字符串的第一个字符开始，逐列比较所有字符串，直到找到不同的字符或到达字符串的末尾。

**代码实现：**

```go
package main

import (
    "fmt"
)

func longestCommonPrefix(strs []string) string {
    if len(strs) == 0 {
        return ""
    }
    for i, v := 0, strs[0][0]; i < len(strs) && i < len(strs[0]); i, v = i+1, strs[0][i] {
        for j := 1; j < len(strs); j++ {
            if i >= len(strs[j]) || strs[j][i] != v {
                return strs[0][:i]
            }
        }
    }
    return strs[0][:i]
}

func main() {
    fmt.Println(longestCommonPrefix([]string{"flower", "flow", "flight"})) // 输出 "fl"
    fmt.Println(longestCommonPrefix([]string{"dog", "racecar", "car"}))    // 输出 ""
}
```

**解析：** 通过好奇心去研究最长公共前缀的算法，我们掌握了如何通过垂直扫描来高效地找出字符串数组中的最长公共前缀。
