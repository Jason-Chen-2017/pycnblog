#### 二、算法编程题部分

##### 1. 字节跳动算法题：如何找出数组中的所有重复元素？

**题目描述：** 给定一个整数数组，请找出数组中的所有重复元素。请编写一个函数，返回包含所有重复元素的数组。

**答案解析：**

1. **使用哈希表**：遍历数组，将每个元素作为键存储在哈希表中，如果键已经存在于哈希表中，则说明该元素是重复的。
2. **排序**：将数组排序，然后遍历排序后的数组，找出相邻重复的元素。

**示例代码：**

```go
// 使用哈希表
func findDuplicates(nums []int) []int {
    m := make(map[int]bool)
    var duplicates []int
    for _, num := range nums {
        if m[num] {
            duplicates = append(duplicates, num)
        } else {
            m[num] = true
        }
    }
    return duplicates
}

// 使用排序
func findDuplicates(nums []int) []int {
    sort.Ints(nums)
    var duplicates []int
    for i := 0; i < len(nums)-1; i++ {
        if nums[i] == nums[i+1] {
            duplicates = append(duplicates, nums[i])
        }
    }
    return duplicates
}
```

##### 2. 京东算法题：如何在一个未排序的数组中找到两个数，它们的和等于一个给定值？

**题目描述：** 给定一个未排序的整数数组和一个目标值，请编写一个函数，找出两个数，它们的和等于目标值。如果找到这样的两个数，请返回它们的索引；如果没有找到，返回空数组。

**答案解析：**

1. **使用哈希表**：遍历数组，对于每个元素，计算目标值与该元素的差，并将其作为键存储在哈希表中。如果差值在哈希表中存在，则找到两个数。
2. **排序+双指针**：将数组排序，然后使用双指针方法，一个指针指向当前元素，另一个指针从当前元素的下一个元素开始，逐步查找符合条件的两个数。

**示例代码：**

```go
// 使用哈希表
func twoSum(nums []int, target int) []int {
    m := make(map[int]int)
    for i, num := range nums {
        complement := target - num
        if j, exists := m[complement]; exists {
            return []int{j, i}
        }
        m[num] = i
    }
    return []int{}
}

// 使用排序+双指针
func twoSum(nums []int, target int) []int {
    sort.Ints(nums)
    left, right := 0, len(nums)-1
    for left < right {
        sum := nums[left] + nums[right]
        if sum == target {
            return []int{left, right}
        } else if sum < target {
            left++
        } else {
            right--
        }
    }
    return []int{}
}
```
