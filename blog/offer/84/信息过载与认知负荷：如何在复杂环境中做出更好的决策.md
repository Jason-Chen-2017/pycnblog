#### 面试题库

##### 1. 如何在有限的时间内做出最优决策？

**答案：** 采用贪心算法，每次选择当前状态下最优的决策，从而保证最终结果是全局最优。

**示例代码：**
```python
def make_decision(states):
    best_actions = []
    for state in states:
        best_action = find_best_action(state)
        best_actions.append(best_action)
    return best_actions
```

##### 2. 如何处理大量数据的决策问题？

**答案：** 利用数据结构优化搜索算法，例如优先队列、并查集等，降低时间复杂度。

**示例代码：**
```python
import heapq

def process_data(data):
    priority_queue = []
    for item in data:
        heapq.heappush(priority_queue, -item)
    result = []
    while priority_queue:
        result.append(heapq.heappop(priority_queue))
    return result
```

##### 3. 如何在不确定性环境中做出更好的决策？

**答案：** 利用概率论和期望值理论，通过计算不同决策结果的期望值，选择期望值最大的决策。

**示例代码：**
```python
def make_decision(possibilities, probabilities):
    best_decision = None
    max_expectation = -1
    for decision, probability in probabilities:
        expectation = decision * probability
        if expectation > max_expectation:
            max_expectation = expectation
            best_decision = decision
    return best_decision
```

#### 算法编程题库

##### 4. 股票买卖的最佳时机（LeetCode 188）

**题目描述：** 给定一个整数数组 prices ，其中 prices[i] 是第 i 天股价。如果可以只完成一笔交易（即同时买入和卖出股票），返回你所能获取的最大利润。否则，返回 0 。

**示例代码：**
```python
def max_profit(prices):
    max_profit = 0
    for i in range(1, len(prices)):
        profit = prices[i] - prices[i - 1]
        max_profit = max(max_profit, profit)
    return max_profit
```

##### 5. 股票买卖的最佳时机 II（LeetCode 123）

**题目描述：** 给定一个整数数组 prices ，其中 prices[i] 是第 i 天的股票价格。请设计一个算法来计算你所能获取的最大利润。你可以完成任意次数的买卖交易，但每次交易中，你都需要先购买后卖出。

**示例代码：**
```python
def max_profit(prices):
    total_profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i - 1]:
            total_profit += prices[i] - prices[i - 1]
    return total_profit
```

##### 6. 股票买卖的最佳时机 III（LeetCode 309）

**题目描述：** 给定一个整数数组 prices ，其中 prices[i] 是第 i 天的股票价格。你最多只能完成两笔交易。返回你所能获取的最大利润。

**示例代码：**
```python
def max_profit(prices):
    first_buy, second_buy = float('-inf'), float('-inf')
    first_sell, second_sell = 0, 0
    for price in prices:
        first_buy = max(first_buy, -price)
        first_sell = max(first_sell, first_buy + price)
        second_buy = max(second_buy, first_sell - price)
        second_sell = max(second_sell, second_buy + price)
    return second_sell
```
