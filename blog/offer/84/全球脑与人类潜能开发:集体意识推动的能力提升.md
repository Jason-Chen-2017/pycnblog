#### 10. 脑波信号处理：降噪算法

**题目：** 设计一个算法来降低脑波信号中的噪声。

**答案解析：** 
可以使用滤波算法，如低通滤波、带阻滤波等，来降低脑波信号中的噪声。以下是一个简单的低通滤波示例：

```python
from scipy.signal import butter, filtfilt

def butter_lowpass_filter(data, cutoff_frequency, sampling_rate):
    nyq = 0.5 * sampling_rate
    b, a = butter(4, cutoff_frequency/nyq, btype='low')
    filtered_data = filtfilt(b, a, data)
    
    return filtered_data
```

def autocorrelation(signal):
    lag = np.arange(0, len(signal))

import scipy.signal as signal

def stft(signal, window_size, hop_size):
