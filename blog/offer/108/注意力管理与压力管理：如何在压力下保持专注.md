**解析：** 一次只处理一个任务可以避免任务切换带来的时间浪费；番茄工作法可以帮助你更好地分配时间和休息；学会说“不”有助于避免承担过多任务，保持专注。

### 算法编程题库

#### 1. 合并两个有序链表

**题目描述：** 将两个有序链表合并为一个新的有序链表并返回。你可以假设 l1 和 l2 已经被排序。

**答案：** 定义一个虚拟头节点，依次比较 l1 和 l2 的节点，将较小的节点添加到新链表中，最后返回新的链表。

```go
func mergeTwoLists(l1 *ListNode, l2 *ListNode) *ListNode {
    dummy := &ListNode{}
    curr := dummy
    for l1 != nil && l2 != nil {
        if l1.Val < l2.Val {
            curr.Next = l1
            l1 = l1.Next
        } else {
            curr.Next = l2
            l2 = l2.Next
        }
        curr = curr.Next
    }
    if l1 != nil {
        curr.Next = l1
    } else if l2 != nil {
        curr.Next = l2
    }
    return dummy.Next
}
```

**解析：** 通过定义虚拟头节点和当前节点，依次比较两个链表的节点，将较小的节点添加到新链表中。最后返回新的链表。

#### 2. 盛水的容器

**题目描述：** 给定一个长度为 n 的整数数组 height ，其中有 n 个容器，容器的高度分别为 height[i] 。假设容器长度分别为 1 和 2 ，返回能盛多少水。

**答案：** 使用双指针法，分别从两端开始遍历，比较左右两端的容器高度，每次移动高度较小的一端。

```go
func maxArea(height []int) int {
    l, r := 0, len(height)-1
    maxArea := 0
    for l < r {
        maxArea = max(maxArea, min(height[l], height[r]))
        if height[l] < height[r] {
            l++
        } else {
            r--
        }
    }
    return maxArea
}
```

**解析：** 定义左右指针，分别指向容器的左右两端。每次比较左右两端的高度，更新最大容积。移动高度较小的一端，直到左右指针相遇。
