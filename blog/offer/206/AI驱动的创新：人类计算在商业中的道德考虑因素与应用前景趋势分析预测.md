```python
import heapq

def prim_algorithm(graph, start):
    mst = []
    visited = set()
    pq = [(0, start)]  # (weight, vertex)

    while pq:
        weight, vertex = heapq.heappop(pq)
        if vertex in visited:
            continue
        visited.add(vertex)
        mst.append((vertex, weight))

        for neighbor, edge_weight in graph[vertex].items():
            if neighbor not in visited:
                heapq.heappush(pq, (edge_weight, neighbor))

    return sum(weight for vertex, weight in mst)

# 示例图（邻接表表示）
```

```python
def coin_change(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0

    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] = min(dp[i], dp[i - coin] + 1)
```

#### 1. KMP算法字符串匹配

**解析：** KMP算法是一种高效的字符串匹配算法，它通过避免不必要的比较，减少时间复杂度。核心思想是构建一个最长公共前后缀数组（LPS），用于指导下一步的匹配过程。

- `compute_lpsArray`: 用于计算LPS数组。
  - `len_x` 和 `len_y` 分别表示模式字符串和LPS数组的长度。
  - 循环遍历模式字符串的每一个字符，根据最长公共前后缀的长度来更新LPS数组。

- `KMPSearch`: 用于实现KMP算法。
  - `M` 和 `N` 分别表示模式字符串和文本字符串的长度。
  - 初始化LPS数组，然后使用两个指针`i`和`j`遍历文本和模式字符串。
  - 当`pat[j] == txt[i]`时，两个指针都向前移动。
  - 如果`j`达到模式字符串的长度，说明找到了一个匹配，记录匹配位置，然后`j`回退到LPS数组的值。

**实例代码：**

#### 2. 深度优先搜索（DFS）图遍历

**解析：** DFS是一种用于遍历图的算法，通过递归方式访问图的每一个顶点。

- `DFS`: 用于实现深度优先搜索。
  - `graph`: 表示图的邻接表。
  - `node`: 当前访问的顶点。
  - `visited`: 记录已访问的顶点。

在递归过程中，首先将当前顶点标记为已访问，然后打印出来。接着，遍历当前顶点的所有邻接点，如果邻接点尚未被访问，则对邻接点进行DFS递归调用。

**实例代码：**
```python
from collections import defaultdict

def DFS(graph, node, visited):
    visited.add(node)
    print(node, end=" ")
    for neighbor in graph[node]:
        if neighbor not in visited:
```

#### 3. 最长公共子序列

**解析：** 最长公共子序列（LCS）问题是一个经典的动态规划问题。通过构建一个二维数组`dp`来记录不同子序列的最长公共子序列长度。

- `longest_common_subsequence`: 用于计算最长公共子序列。
  - `m` 和 `n` 分别表示两个字符串的长度。
  - 初始化`dp`数组，遍历字符串的每一个字符，根据字符是否相同来更新`dp`数组的值。

**实例代码：**
```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[-1][-1]
```

#### 4. Prim算法最小生成树

**解析：** Prim算法是一种用于求解最小生成树的贪心算法。它从某个顶点开始，逐步添加新的边和顶点，直到包含所有顶点。

- `prim_algorithm`: 用于实现Prim算法。
  - `graph`: 表示图的邻接表。
  - `start`: 表示起始顶点。
  - `mst`: 用于记录最小生成树的边。
  - `visited`: 记录已访问的顶点。
  - `pq`: 表示优先队列，用于选择权重最小的边。

在算法执行过程中，从优先队列中选择权重最小的边，如果该边连接的顶点未被访问，则将该顶点加入生成树，并将连接的边加入`mst`。然后更新优先队列，继续选择新的最小权重边。

**实例代码：**
```python
import heapq

def prim_algorithm(graph, start):
    mst = []
    visited = set()
    pq = [(0, start)]  # (weight, vertex)

    while pq:
        weight, vertex = heapq.heappop(pq)
        if vertex in visited:
            continue
        visited.add(vertex)
        mst.append((vertex, weight))

        for neighbor, edge_weight in graph[vertex].items():
            if neighbor not in visited:
                heapq.heappush(pq, (edge_weight, neighbor))

    return sum(weight for vertex, weight in mst)

    4: {3: 9, 5: 10},
    5: {2: 4, 3: 14, 4: 10},
    6: {4: 2},
    7: {0: 8, 1: 11, 6: 1},
    8: {0: 8, 9: 2},
    9: {8: 2, 10: 6},
    10: {9: 6, 5: 10}
}
```

#### 5. 货币兑换问题

**解析：** 货币兑换问题可以使用动态规划来解决。定义一个一维数组`dp`，其中`dp[i]`表示兑换金额为`i`所需的最少硬币数量。

- `coin_change`: 用于计算最少硬币数量。
  - `coins`: 表示硬币的面值数组。
  - `amount`: 表示目标金额。
  - 初始化`dp`数组，遍历每一个面值，更新`dp`数组的值。

在遍历过程中，如果当前面值小于目标金额，则尝试将面值加入兑换组合，更新最少硬币数量。

**实例代码：**
```python
def coin_change(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0

    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] = min(dp[i], dp[i - coin] + 1)

    return dp[amount] if dp[amount] != float('inf') else -1

coins = [1, 2, 5]
amount = 11
print(coin_change(coins, amount))
```
