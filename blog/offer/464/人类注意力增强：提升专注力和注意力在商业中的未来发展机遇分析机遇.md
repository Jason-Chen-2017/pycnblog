- **视频观看时长（Video View Time）：** 用户观看视频的时间长度。
- **用户互动（Comments, Shares, Likes）：** 用户对内容的互动程度。

**举例：** 假设我们想要衡量一个视频平台的用户注意力，可以使用以下代码来计算平均观看时长：

```python
    if total_views == 0:
        return 0
    return sum(video_views) / total_views

```

**解析：** 此代码计算了用户观看视频的平均时长。通过累计所有视频观看时长并除以视频总次数，我们得到了平均值。
