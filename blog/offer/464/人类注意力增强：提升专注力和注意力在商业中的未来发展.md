##### 1. 注意力权重计算

**题目：** 设计一个算法，计算注意力权重。

**答案：** 注意力权重可以通过以下公式计算：

\[ w_i = \frac{e^{r_i}}{\sum_{j=1}^{n} e^{r_j}} \]

其中，\( r_i \) 表示第 \( i \) 个元素的权重，\( n \) 表示元素的总数。

**实例：**

```python
def calculate_attention_weights(scores):
    exp_scores = np.exp(scores)
    sum_exp_scores = np.sum(exp_scores)
    attention_weights = exp_scores / sum_exp_scores
    return attention_weights

# 测试
scores = [0.1, 0.3, 0.2, 0.4]
attention_weights = calculate_attention_weights(scores)
print(attention_weights)
```
