### 二、注意力增强算法编程题库

#### 1. 注意力分散检测算法

**题目：** 编写一个算法，用于检测医务人员在工作过程中是否出现注意力分散现象。

**答案：** 基于信号处理的方法，可以使用以下算法检测注意力分散：

```python
import numpy as np

def attention_detection(signal, threshold=0.1):
    """
    检测注意力分散算法。
    
    :param signal: 信号，表示医务人员在工作过程中的表现，可以是评分、操作时长等。
    :param threshold: 阈值，用于判断注意力分散的标准。
    :return: 注意力分散检测结果，True 表示注意力分散，False 表示未发生注意力分散。
    """
    # 计算平均值
    mean_value = np.mean(signal)
    # 计算标准差
    std_value = np.std(signal)
    # 判断是否超出阈值
    if np.abs(mean_value) > threshold * std_value:
        return True
    else:
        return False

# 示例
signal = np.random.randn(100)
result = attention_detection(signal)
print(result)
```

**解析：** 该算法基于信号处理的原理，通过计算平均值和标准差，判断医务人员在工作过程中的表现是否超出阈值，从而判断是否出现注意力分散现象。

#### 2. 注意力分配优化算法

**题目：** 编写一个算法，用于优化医务人员的注意力分配，使其在工作过程中保持高效。

**答案：** 基于遗传算法的注意力分配优化算法：

```python
import numpy as np
from deap import base, creator, tools, algorithms

# 定义问题参数
N = 10  # 任务数量
max_time = 1000  # 最大工作时间

# 创建遗传算法工具
creator.create("FitnessMax", base.Fitness, weights=(1.0,))
creator.create("Individual", list, fitness=creator.FitnessMax)

# 创建工具
toolbox = base.Toolbox()
toolbox.register("attr_int", np.random.randint, low=0, high=max_time)
toolbox.register("individual", tools.initRepeat, creator.Individual, toolbox.attr_int, N)
toolbox.register("population", tools.initRepeat, list, toolbox.individual)

# 创建适应度函数
def fitness(individual):
    total_time = sum(individual)
    if total_time > max_time:
        return creator.FitnessMax(0)
    else:
        return creator.FitnessMax(1 / (1 + total_time))

toolbox.register("evaluate", fitness)

# 创建遗传算法操作
toolbox.register("select", tools.selTournament, tournsize=3)
toolbox.register("mate", tools.mateUniform)
toolbox.register("mutate", tools.mutUniformInt, low=0, up=max_time, indpb=0.1)
toolbox.register("update", tools.selBest)

# 创建种群
population = toolbox.population(n=50)

# 进行遗传算法迭代
num_generations = 100
for gen in range(num_generations):
    offspring = toolbox.select(population, len(population))
    offspring = toolbox.update(offspring)
    population = offspring
    print("Generation {0}: {1}".format(gen, population[0].fitness.values))

# 输出最佳解
best_individual = tools.selBest(population, 1)[0]
print("Best individual is: {0}".format(best_individual))
```

**解析：** 该算法基于遗传算法，通过优化医务人员的注意力分配，使其在工作过程中保持高效。通过适应度函数评估个体的优劣，并通过交叉、变异等操作进行种群更新。

### 9. 注意力分散检测算法

**题目：** 编写一个简单的算法，用于检测注意力分散现象。

**答案：** 以下是一个简单的注意力分散检测算法示例，基于标准差和均值的比较：

```python
import numpy as np

def detect_attention_diffusion(scores, threshold=0.5):
    """
    检测注意力分散的简单算法。
    
    :param scores: 一组评分，表示个体的注意力水平。
    :param threshold: 阈值，用来判断注意力是否分散。
    :return: True 如果注意力分散，False 如果注意力集中。
    """
    mean_score = np.mean(scores)
    std_score = np.std(scores)
    
    # 如果标准差大于均值的阈值倍数，认为注意力分散
    if std_score > threshold * mean_score:
        return True
    else:
        return False

# 示例数据
scores = np.array([90, 85, 78, 82, 88, 75, 80, 85, 91, 77])

# 检测注意力分散
result = detect_attention_diffusion(scores)
print(f"注意力分散: {result}")
```

**解析：** 该算法通过计算评分的标准差和均值，判断标准差是否超过均值的阈值倍数。如果超过，则认为注意力分散；否则，认为注意力集中。这个简单的算法可以作为一个初步的注意力分散检测工具，实际应用中可能需要更复杂的模型。

### 10. 注意力分配优化算法

**题目：** 请使用遗传算法实现一个简单的注意力分配优化算法。

**答案：** 以下是一个使用遗传算法的简单注意力分配优化算法实现：

```python
import random
import numpy as np

# 问题参数
N_TASKS = 5  # 任务数量
MAX_TIME = 100  # 最大可用时间
P_CROSSOVER = 0.8  # 交叉概率
P_MUTATION = 0.1  # 变异概率

# 适应度函数
def fitness(individual):
    total_time = sum(individual)
    return 1 if total_time <= MAX_TIME else 0

# 遗传算法操作
def crossover(parent1, parent2):
    """
    交叉操作。
    
    :param parent1: 父本1
    :param parent2: 父本2
    :return: 新的个体
    """
    crossover_point = random.randint(1, N_TASKS - 1)
    child = parent1[:crossover_point] + parent2[crossover_point:]
    return child

def mutate(individual):
    """
    变异操作。
    
    :param individual: 个体
    :return: 变异后的个体
    """
    for i in range(N_TASKS):
        if random.random() < P_MUTATION:
            individual[i] = (individual[i] + random.randint(-10, 10)) % MAX_TIME
    return individual

# 初始种群
population = [random.randint(0, MAX_TIME) for _ in range(100)]

# 遗传算法迭代
generations = 100
for _ in range(generations):
    # 适应度评估
    fitness_scores = [fitness(individual) for individual in population]
    
    # 生存选择
    selected = random.choices(population, weights=fitness_scores, k=N_TASKS)
    
    # 交叉和变异
    offspring = [crossover(selected[i], selected[i+1]) for i in range(N_TASKS//2)]
    offspring = [mutate(child) for child in offspring]
    
    # 新种群
    population = offspring

# 输出最佳解
best_individual = max(population, key=fitness)
print(f"最佳个体：{best_individual}, 适应度：{fitness(best_individual)}")
```

**解析：** 该算法使用遗传算法优化注意力分配，目标是最小化超过最大可用时间的任务总和。适应度函数根据个体是否满足时间约束来评估。通过交叉和变异操作，以及生存选择策略，不断迭代优化种群。
