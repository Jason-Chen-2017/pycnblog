**二、算法编程题库**

##### 5. 脑电信号处理
**题目：** 给定一段脑电信号数据，编写代码实现对其降噪处理。

**答案：** 可以使用滤波器（如带通滤波器）对脑电信号进行降噪处理。

```python
import numpy as np
from scipy.signal import butter, filtfilt

def butter_bandpass_filter(data, lowcut, highcut, fs, order=4):
    nyq = 0.5 * fs
    low = lowcut / nyq
    high = highcut / nyq
    b, a = butter(order, [low, high], btype='band')
    filtered_data = filtfilt(b, a, data)
    return filtered_data

# 示例
fs = 1000  # 采样频率
lowcut = 1  # 低截止频率
highcut = 50  # 高截止频率
data = np.random.randn(1000)  # 脑电信号数据

filtered_data = butter_bandpass_filter(data, lowcut, highcut, fs)
```

**解析：** 使用带通滤波器对脑电信号进行降噪处理，可以去除低频噪声和高频噪声，保留有用的信号成分。

##### 6. 心率变异性分析
**题目：** 给定一段心率数据，编写代码实现对其心率变异性（HRV）进行分析。

**答案：** 可以使用小波变换对心率数据进行分解，然后计算各个频带上的功率，从而得到心率变异性。

```python
import numpy as np
from pywt import pyrLEGenda
from scipy.stats import kurtosis

def calculate_hrv(data):
    coeffs = pyrLEGenda(data, 'db4', mode='per')
    freqs = np.fft.rfftfreq(len(data), d=1)
    power = np.abs(coeffs[1])**2
    freqs = freqs[:len(power)]
    power = power[:len(freqs)]

    # 计算低频（LF）和高频（HF）功率
    lf_power = np.sum(power[(freqs >= 0.04) & (freqs <= 0.15)])
    hf_power = np.sum(power[(freqs >= 0.15) & (freqs <= 0.4)])

    # 计算HRV
    hrv = np.sqrt(lf_power / hf_power)

    return hrv

# 示例
data = np.random.randn(1000)  # 心率数据

hrv = calculate_hrv(data)
print("HRV:", hrv)
```

**解析：** 通过计算心率变异性（HRV），可以评估用户的心理压力和放松程度。

##### 7. 肌电信号处理
**题目：** 给定一段肌电信号数据，编写代码实现对其特征提取。

**答案：** 可以使用小波变换对肌电信号进行分解，然后计算各个节段的能量特征。

```python
import numpy as np
from pywt import pyrLEGenda

def extract_features(data):
    coeffs = pyrLEGenda(data, 'db4', mode='per')
    energies = np.abs(coeffs[1])**2
    features = np.mean(energies, axis=1)

    return features

# 示例
data = np.random.randn(1000)  # 肌电信号数据

features = extract_features(data)
print("Features:", features)
```

**解析：** 通过特征提取，可以对肌电信号进行分类和识别，从而评估用户的放松状态。
