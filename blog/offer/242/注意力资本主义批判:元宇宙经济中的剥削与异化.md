### 3. 算法编程题库

#### 3.1 题目 1：设计一个算法，计算用户在元宇宙中的注意力消耗。

**题目描述：** 给定一个用户在元宇宙中的活动记录，记录包括访问时间、访问内容类型（如广告、游戏、购物等），设计一个算法计算用户的注意力消耗。

**输入示例：**
```json
[
  {"time": 10, "type": "ad"},
  {"time": 20, "type": "game"},
  {"time": 30, "type": "shopping"}
]
```

**输出示例：**
```json
{
  "ad": 10,
  "game": 20,
  "shopping": 30
}
```

**解析与答案：**

```go
package main

import (
    "encoding/json"
    "fmt"
)

type Activity struct {
    Time   int    `json:"time"`
    Type   string `json:"type"`
}

func calculateAttentionCost(activities []Activity) (map[string]int, error) {
    attentionCost := make(map[string]int)

    for _, activity := range activities {
        attentionCost[activity.Type] += activity.Time
    }

    return attentionCost, nil
}

func main() {
    activities := []Activity{
        {Time: 10, Type: "ad"},
        {Time: 20, Type: "game"},
        {Time: 30, Type: "shopping"},
    }

    cost, err := calculateAttentionCost(activities)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }

    fmt.Println("Attention Cost:", cost)
}
```

#### 3.2 题目 2：设计一个算法，检测元宇宙平台是否存在注意力剥削行为。

**题目描述：** 给定一个用户在元宇宙平台上的活动记录，记录包括访问时间、访问内容类型（如广告、游戏、购物等），设计一个算法判断平台是否存在注意力剥削行为。

**输入示例：**
```json
[
  {"time": 10, "type": "ad"},
  {"time": 20, "type": "game"},
  {"time": 30, "type": "shopping"}
]
```

**输出示例：**
```json
{
  "isExploited": true,
  "attentionCost": 60
}
```

**解析与答案：**

```go
package main

import (
    "encoding/json"
    "fmt"
)

type Activity struct {
    Time   int    `json:"time"`
    Type   string `json:"type"`
}

func checkExploitation(activities []Activity) (map[string]int, bool) {
    adCost := 0
    gameCost := 0
    shoppingCost := 0

    for _, activity := range activities {
        switch activity.Type {
        case "ad":
            adCost += activity.Time
        case "game":
            gameCost += activity.Time
        case "shopping":
            shoppingCost += activity.Time
        }
    }

    totalCost := adCost + gameCost + shoppingCost

    // 设置一个阈值，如60，表示超过这个阈值的注意力消耗可能被认为是剥削行为
    threshold := 60

    isExploited := totalCost > threshold

    return map[string]int{
        "ad":         adCost,
        "game":       gameCost,
        "shopping":   shoppingCost,
    }, isExploited
}

func main() {
    activities := []Activity{
        {Time: 10, Type: "ad"},
        {Time: 20, Type: "game"},
        {Time: 30, Type: "shopping"},
    }

    cost, isExploited := checkExploitation(activities)
    if isExploited {
        fmt.Println("Attention Cost:", cost)
        fmt.Println("Is Exploited: True")
    } else {
        fmt.Println("Attention Cost:", cost)
        fmt.Println("Is Exploited: False")
    }
}
```

#### 3.3 题目 3：设计一个算法，计算用户在元宇宙中的平均注意力消耗。

**题目描述：** 给定一个用户在元宇宙平台上的活动记录，记录包括访问时间、访问内容类型（如广告、游戏、购物等），设计一个算法计算用户的平均注意力消耗。

**输入示例：**
```json
[
  {"time": 10, "type": "ad"},
  {"time": 20, "type": "game"},
  {"time": 30, "type": "shopping"}
]
```

**输出示例：**
```json
{
  "averageAttentionCost": 20
}
```

**解析与答案：**

```go
package main

import (
    "encoding/json"
    "fmt"
)

type Activity struct {
    Time   int    `json:"time"`
    Type   string `json:"type"`
}

func calculateAverageAttentionCost(activities []Activity) (float64, error) {
    totalTime := 0
    for _, activity := range activities {
        totalTime += activity.Time
    }

    if len(activities) == 0 {
        return 0, fmt.Errorf("no activities to calculate average")
    }

    average := float64(totalTime) / float64(len(activities))
    return average, nil
}

func main() {
    activities := []Activity{
        {Time: 10, Type: "ad"},
        {Time: 20, Type: "game"},
        {Time: 30, Type: "shopping"},
    }

    avgCost, err := calculateAverageAttentionCost(activities)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }

    fmt.Println("Average Attention Cost:", avgCost)
}
```
