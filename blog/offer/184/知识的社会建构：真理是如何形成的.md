#### 1. 找出列表中的最长递增子序列

**题目描述：** 给定一个整数列表，找出并返回它的最长递增子序列的长度。

**示例：** `lengthOfLIS([10, 9, 2, 5, 3, 7, 101, 18])` 返回 4，因为最长递增子序列是 `[2, 3, 7, 101]`。

**解题思路：** 使用动态规划算法，维护一个数组 `dp`，其中 `dp[i]` 表示以 `nums[i]` 结尾的最长递增子序列的长度。

**Python 代码：**

```python
def lengthOfLIS(nums):
    if not nums:
        return 0
    dp = [1] * len(nums)
    for i in range(1, len(nums)):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```

#### 2. 最长公共前缀

**题目描述：** 编写一个函数来查找字符串数组中的最长公共前缀。

**示例：** `longestCommonPrefix(["flower", "flow", "flight"])` 返回 `"fl"`。

**解题思路：** 比较字符串数组中的第一个字符串与其他字符串的前缀，逐步缩小搜索范围。

**Python 代码：**

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s):
            if prefix[i] != s[i]:
                break
            i += 1
        prefix = prefix[:i]
    return prefix
```

#### 3. 合并两个有序链表

**题目描述：** 将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

**示例：** 给定两个有序链表 `l1: 1->2->4` 和 `l2: 1->3->4`，合并后的链表为 `1->1->2->3->4->4`。

**解题思路：** 使用递归或迭代方法，将两个链表中的节点按顺序拼接。

**Python 代码（递归）：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def mergeTwoLists(l1, l2):
    if not l1:
        return l2
    if not l2:
        return l1
    if l1.val < l2.val:
        l1.next = mergeTwoLists(l1.next, l2)
        return l1
    else:
        l2.next = mergeTwoLists(l1, l2.next)
        return l2
```

#### 4. 两数之和

**题目描述：** 给定一个整数数组 `nums` 和一个目标值 `target`，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

**示例：** 给定 `nums = [2, 7, 11, 15]` 和 `target = 9`，因为 `nums[0] + nums[1] = 2 + 7 = 9`，所以返回 `[0, 1]`。

**解题思路：** 使用哈希表存储数组中的元素及其下标，遍历数组并检查目标值是否可以通过哈希表中的元素相加得到。

**Python 代码：**

```python
def twoSum(nums, target):
    hash_map = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hash_map:
            return [hash_map[complement], i]
        hash_map[num] = i
    return []
```

#### 5. 搜索插入位置

**题目描述：** 给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

**示例：** 给定 `nums = [1, 3, 5, 6]` 和 `target = 5`，返回 `2`。给定 `nums = [1, 3, 5, 6]` 和 `target = 2`，返回 `1`。

**解题思路：** 使用二分查找算法找到目标值的位置。

**Python 代码：**

```python
def searchInsert(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return left
```

#### 7. 最长公共子序列

**题目描述：** 给定两个字符串 `text1` 和 `text2`，返回这两个字符串的最长公共子序列的长度。

**示例：** `text1 = "abcde"` 和 `text2 = "ace"`，返回 3。

**解题思路：** 使用动态规划算法，构建一个二维数组 `dp`，其中 `dp[i][j]` 表示 `text1` 的前 `i` 个字符和 `text2` 的前 `j` 个字符的最长公共子序列的长度。

**Python 代码：**

```python
def longestCommonSubsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```

#### 8. 最大子序和

**题目描述：** 给定一个整数数组 `nums` ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**示例：** `nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]`，返回 6。

**解题思路：** 使用动态规划算法，维护一个变量 `max_ending_here` 和 `max_so_far`，分别表示以当前元素结尾的最大子序和以及全局最大子序和。

**Python 代码：**

```python
def maxSubArray(nums):
    max_ending_here = max_so_far = nums[0]
    for num in nums[1:]:
        max_ending_here = max(num, max_ending_here + num)
        max_so_far = max(max_so_far, max_ending_here)
    return max_so_far
```

#### 9. 最小路径和

**题目描述：** 给定一个包含非负整数的二维网格 `grid` ，找出一条从左上角到右下角的最小路径和。

**示例：** `grid = [[1, 3, 1], [1, 5, 1], [4, 2, 1]]`，返回 7。

**解题思路：** 使用动态规划算法，从右下角开始反向遍历网格，更新每个单元格的最小路径和。

**Python 代码：**

```python
def minPathSum(grid):
    m, n = len(grid), len(grid[0])
    for i in range(m - 1, -1, -1):
        for j in range(n - 1, -1, -1):
            if i < m - 1:
                grid[i][j] += min(grid[i + 1][j], grid[i + 1][j + 1])
            if j < n - 1:
                grid[i][j] += min(grid[i][j + 1], grid[i + 1][j + 1])
    return grid[0][0]
```

**题目描述：** 假设你正在爬楼梯。需要 `n` 阶台阶，每次可以爬 1 或 2 个台阶，请问有多少种不同的方法可以爬到楼顶？

**示例：** `n = 3`，返回 3（`[1, 1, 1]`、`[1, 2]`、`[2, 1]`）。

**解题思路：** 使用动态规划算法，维护一个数组 `dp`，其中 `dp[i]` 表示到达第 `i` 阶的方法数。

**Python 代码：**

```python
def climbStairs(n):
    if n == 1:
        return 1
    dp = [0] * (n + 1)
    dp[1], dp[2] = 1, 2
    for i in range(3, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n]
```

#### 11. 合并区间

**题目描述：** 给定一组区间，请合并所有重叠的区间。

**示例：** `intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]`，返回 `[[1, 6], [8, 10], [15, 18]]`。

**解题思路：** 首先，将区间按照左端点排序；然后，遍历区间，合并重叠的区间。

**Python 代码：**

```python
def merge(intervals):
    if not intervals:
        return []
    intervals.sort()
    ans = [intervals[0]]
    for interval in intervals[1:]:
        last = ans[-1]
        if last[1] >= interval[0]:
            ans[-1] = [last[0], max(last[1], interval[1])]
        else:
            ans.append(interval)
    return ans
```

#### 12. 股票买卖最佳时机 II

**题目描述：** 给定一个整数数组 `prices`，其中每个元素代表了某一天的股票价格。设计一个算法能够找到最大的利润。可以在任何时候买入和卖出股票，但是你只能在每次交易的第二天再进行交易。

**示例：** `prices = [7, 1, 5, 3, 6, 4]`，返回 7。

**解题思路：** 遍历价格数组，每次遇到一个比前一个价格高的数，就把这两个价格之间的差值加到结果中。

**Python 代码：**

```python
def maxProfit(prices):
    profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i - 1]:
            profit += prices[i] - prices[i - 1]
    return profit
```

**解题思路：** 从第一个字符串开始，逐个字符与其他字符串的前缀进行比较，逐步缩小公共前缀的范围。

**Python 代码：**

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s):
            if prefix[i] != s[i]:
                break
            i += 1
        prefix = prefix[:i]
    return prefix
```

**Python 代码：**

```python
def hammingWeight(n):
    count = 0
    while n:
        count += n & 1
        n >>= 1
    return count
```

**解题思路：** 使用二分查找算法，在旋转排序数组中查找目标值。

**Python 代码：**

```python
def search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return True
        if nums[left] <= nums[mid]:
            if target >= nums[left] and target < nums[mid]:
                right = mid - 1
            else:
                left = mid + 1
        else:
            if target > nums[mid] and target <= nums[right]:
                left = mid + 1
            else:
                right = mid - 1
    return False
```

**解题思路：** 首先移除牌中的 0，然后判断牌的最大值和最小值之差是否小于 5，且在牌中不存在连续的数字。

**Python 代码：**

```python
def isStraight(nums):
    nums = [x for x in nums if x != 0]
    if not nums:
        return True
    return max(nums) - min(nums) < 5 and len(set(nums)) == len(nums)
```

**解题思路：** 计算数组的中位数，然后遍历数组，统计每个元素与中位数之差的绝对值之和。

**Python 代码：**

```python
def minMoves2(nums):
    nums.sort()
    return sum(abs(x - nums[len(nums) // 2]) for x in nums)
```
