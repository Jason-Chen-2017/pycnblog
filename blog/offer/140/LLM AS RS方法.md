### 3. 算法编程题库

#### 1. 编写一个函数，计算用户和商品之间的相似度。

**题目：** 编写一个函数 `compute_similarity(user, item)`，计算用户 `user` 和商品 `item` 之间的相似度。

**答案：**

```python
def compute_similarity(user, item):
    # 假设 user 和 item 分别为用户和商品的向量表示
    dot_product = np.dot(user, item)
    norm_user = np.linalg.norm(user)
    norm_item = np.linalg.norm(item)
    similarity = dot_product / (norm_user * norm_item)
    return similarity
```

#### 2. 编写一个函数，基于用户-商品交互矩阵，利用矩阵分解技术学习用户和商品的隐含特征。

**题目：** 编写一个函数 `matrix_factorization(R, k, lambda_)$，利用矩阵分解技术学习用户和商品的隐含特征。

**答案：**

```python
import numpy as np

def matrix_factorization(R, k, lambda_):
    n, m = R.shape
    P = np.random.rand(n, k)
    Q = np.random.rand(m, k)
    
    for i in range(1000):
        for i in range(n):
            for j in range(m):
                if R[i][j] > 0:
                    e = R[i][j] - np.dot(P[i], Q[j])
                    P[i] = P[i] + lambda_ * (P[i] - e * Q[j])
                    Q[j] = Q[j] + lambda_ * (Q[j] - e * P[i])
    
    return P, Q
```
