### 算法编程题库

#### 1. 单调栈

**题目：** 给定一个数组，实现一个单调栈，要求：

- 从数组中弹出最大的元素；
- 当栈为空时，返回-1。

**答案：** 使用两个栈实现单调栈，一个栈用于存储元素的值，另一个栈用于存储对应下标。

```python
class MonotonousStack:
    def __init__(self):
        self.stack = []
        self.index_stack = []

    def push(self, x):
        while self.stack and self.stack[-1] <= x:
            self.stack.pop()
            self.index_stack.pop()
        self.stack.append(x)
        self.index_stack.append(len(self.stack) - 1)

    def pop(self):
        if not self.index_stack:
            return -1
        self.stack.pop()
        self.index_stack.pop()
        return self.stack[self.index_stack[-1]]

    def top(self):
        if not self.index_stack:
            return -1
        return self.stack[self.index_stack[-1]]
```

#### 2. 岛屿的最大面积

**题目：** 给定一个由 '1'（陆地）和 '0'（水）组成的的二维网格，计算岛屿的最大面积。

**答案：** 使用深度优先搜索（DFS）算法，遍历网格，计算每个岛屿的面积，并记录最大面积。

```python
def maxAreaOfIsland(grid):
    def dfs(i, j):
        if i < 0 or i >= len(grid) or j < 0 or j >= len(grid[0]) or grid[i][j] == 0:
            return 0
        grid[i][j] = 0
        return 1 + dfs(i - 1, j) + dfs(i + 1, j) + dfs(i, j - 1) + dfs(i, j + 1)

    max_area = 0
    for i in range(len(grid)):
        for j in range(len(grid[0])):
            if grid[i][j] == 1:
                max_area = max(max_area, dfs(i, j))
    return max_area
```

#### 3. 最长公共前缀

**题目：** 编写一个函数来查找字符串数组中的最长公共前缀。

**答案：** 从第一个字符串开始，逐个比较后续字符串，找出最长公共前缀。

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""

    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s):
            if prefix[i] != s[i]:
                break
            i += 1
        prefix = prefix[:i]
    return prefix
```
