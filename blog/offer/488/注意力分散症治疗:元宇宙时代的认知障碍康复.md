#### 二、算法编程题库

**1. 实现一个注意力权重计算算法。**

**题目描述：** 给定一个序列 \(x_1, x_2, ..., x_n\)，实现一个算法计算每个元素的注意力权重。权重计算公式为：\( w_i = \frac{1}{\sum_{j=1}^{n} x_j^2} \)。

**答案：**

```python
def compute_attention_weights(x):
    sum_sq = sum([xi ** 2 for xi in x])
    weights = [1 / sum_sq for xi in x]
    return weights

x = [1, 2, 3, 4, 5]
weights = compute_attention_weights(x)
print(weights)
```
