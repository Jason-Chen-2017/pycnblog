#### 4. 设计一个算法来跟踪用户的注意力状态。

**题目：** 编写一个算法，用于记录和分析用户的注意力状态，输出用户在特定时间段内的专注时长和分心次数。

**答案：**
```python
def track_attention(duration, attention_points):
    """
    计算用户的专注时长和分心次数。
    :param duration: 用户总工作时间（秒）
    :param attention_points: 用户每次注意力集中的时间点（秒）
    :return: (专注时长，分心次数)
    """
    total_attention = 0
    distraction_count = 0

    for point in attention_points:
        if point < duration:
            total_attention += point
        else:
            distraction_count += 1

    return total_attention, distraction_count

# 示例
duration = 1800  # 30分钟
attention_points = [120, 240, 300, 900]  # 每次专注的时间点
print(track_attention(duration, attention_points))
```

#### 5. 实现一个正念练习评分系统。

**题目：** 编写一个程序，根据用户在正念练习中的表现（如专注时长、呼吸均匀度等指标）给予评分。

**答案：**
```python
def mindfulness_score(duration, breath_uniformity, max_score=100):
    """
    给出正念练习的评分。
    :param duration: 练习时长（分钟）
    :param breath_uniformity: 呼吸均匀度（0-1，0表示完全不规则，1表示完全均匀）
    :param max_score: 最大评分，默认为100分
    :return: 练习得分
    """
    base_score = duration * 2  # 假设每分钟基础得分为2分
    breath_bonus = min(breath_uniformity * 10, 100)  # 呼吸均匀度每提升1%，加10分，上限100分

    return min(base_score + breath_bonus, max_score)

# 示例
duration = 30  # 练习时长为30分钟
breath_uniformity = 0.8  # 呼吸均匀度为80%
print(mindfulness_score(duration, breath_uniformity))
```

#### 6. 分析用户注意力波动的算法。

**题目：** 设计一个算法，分析用户在一段特定时间内注意力波动的模式，输出波动次数和波动强度。

**答案：**
```python
def analyze_attention_wave(duration, attention_data):
    """
    分析用户注意力波动。
    :param duration: 用户总工作时间（秒）
    :param attention_data: 用户每次注意力集中的时间点（秒）
    :return: (波动次数，波动强度)
    """
    wave_count = 0
    max_wave_difference = 0

    for i in range(1, len(attention_data)):
        wave_difference = attention_data[i] - attention_data[i - 1]
        wave_count += 1
        if wave_difference > max_wave_difference:
            max_wave_difference = wave_difference

    return wave_count, max_wave_difference

# 示例
duration = 1800  # 30分钟
attention_data = [120, 240, 300, 900]  # 每次专注的时间点
print(analyze_attention_wave(duration, attention_data))
```
