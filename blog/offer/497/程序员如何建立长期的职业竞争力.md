### 高频面试题和算法编程题库

#### 面试题 1：链表反转

**题目描述：** 反转一个单链表。

**答案解析：**

```go
func reverseList(head *ListNode) *ListNode {
    var prev *ListNode = nil
    var curr = head

    for curr != nil {
        nextTemp := curr.Next
        curr.Next = prev
        prev = curr
        curr = nextTemp
    }
    return prev
}
```

#### 面试题 2：二分查找

**题目描述：** 在排序数组中查找一个特定元素的索引。

**答案解析：**

```go
func search(nums []int, target int) int {
    left, right := 0, len(nums) - 1
    for left <= right {
        mid := (left + right) / 2
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}
```

#### 面试题 3：最长公共前缀

**题目描述：** 找出字符串数组中的最长公共前缀。

**答案解析：**

```go
func longestCommonPrefix(strs []string) string {
    if len(strs) == 0 {
        return ""
    }
    prefix := strs[0]
    for i := 1; i < len(strs); i++ {
        for j := 0; j < len(prefix) && j < len(strs[i]); j++ {
            if prefix[j] != strs[i][j] {
                prefix = prefix[:j]
                break
            }
        }
    }
    return prefix
}
```

#### 面试题 4：合并两个有序链表

**题目描述：** 合并两个有序链表并返回新链表。

**答案解析：**

```go
func mergeTwoLists(l1 *ListNode, l2 *ListNode) *ListNode {
    if l1 == nil {
        return l2
    }
    if l2 == nil {
        return l1
    }
    if l1.Val < l2.Val {
        l1.Next = mergeTwoLists(l1.Next, l2)
        return l1
    }
    l2.Next = mergeTwoLists(l1, l2.Next)
    return l2
}
```

#### 面试题 5：两数相加

**题目描述：** 不使用加法运算符，实现两个整数相加。

**答案解析：**

```go
func addTwoNumbers(l1 *ListNode, l2 *ListNode) *ListNode {
    dummy := &ListNode{}
    curr := dummy
    carry := 0

    for l1 != nil || l2 != nil || carry != 0 {
        val1 := 0
        if l1 != nil {
            val1 = l1.Val
            l1 = l1.Next
        }
        val2 := 0
        if l2 != nil {
            val2 = l2.Val
            l2 = l2.Next
        }
        sum := val1 + val2 + carry
        carry = sum / 10
        curr.Next = &ListNode{Val: sum % 10}
        curr = curr.Next
    }

    return dummy.Next
}
```

#### 面试题 6：盛水最多的容器

**题目描述：** 给定一个二维数组（网格），找出能容纳水的最大容器。

**答案解析：**

```go
func maxArea(height []int) int {
    left, right := 0, len(height)-1
    maxArea := 0

    for left < right {
        low := height[left]
        high := height[right]
        maxArea = max(maxArea, (right-left)*min(low, high))
        if low < high {
            left++
        } else {
            right--
        }
    }

    return maxArea
}
```

#### 面试题 7：移动零

**答案解析：**

```go
func moveZeroes(nums []int) {
    index := 0
    for i := 0; i < len(nums); i++ {
        if nums[i] != 0 {
            nums[index] = nums[i]
            index++
        }
    }
    for i := index; i < len(nums); i++ {
        nums[i] = 0
    }
}
```

#### 面试题 8：搜索二维矩阵

**题目描述：** 给定一个排序后的二维矩阵，实现一个搜索算法，找出给定元素的位置。

**答案解析：**

```go
func searchMatrix(matrix [][]int, target int) bool {
    rows := len(matrix)
    cols := len(matrix[0])
    row, col := 0, cols-1

    for row < rows && col >= 0 {
        if matrix[row][col] == target {
            return true
        } else if matrix[row][col] < target {
            row++
        } else {
            col--
        }
    }
    return false
}
```

#### 面试题 9：最长公共子序列

**题目描述：** 给定两个字符串，找出它们的最长公共子序列。

**答案解析：**

```go
func longestCommonSubsequence(text1 string, text2 string) int {
    m, n := len(text1), len(text2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }

    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if text1[i-1] == text2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }

    return dp[m][n]
}
```

#### 面试题 10：最小路径和

**题目描述：** 给定一个包含非负整数的二维网格，找出从左上角到右下角的最小路径和。

**答案解析：**

```go
func minPathSum(grid [][]int) int {
    rows, cols := len(grid), len(grid[0])
    dp := make([][]int, rows)
    for i := range dp {
        dp[i] = make([]int, cols)
    }

    dp[0][0] = grid[0][0]
    for i := 1; i < rows; i++ {
        dp[i][0] = dp[i-1][0] + grid[i][0]
    }
    for j := 1; j < cols; j++ {
        dp[0][j] = dp[0][j-1] + grid[0][j]
    }

    for i := 1; i < rows; i++ {
        for j := 1; j < cols; j++ {
            dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]
        }
    }

    return dp[rows-1][cols-1]
}
```
