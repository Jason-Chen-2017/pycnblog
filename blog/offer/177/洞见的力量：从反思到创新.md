#### 一、算法编程题库

##### 1. 最长公共子序列（LCS）

**题目：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。

**答案：** 使用动态规划求解。

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

# 示例
str1 = "AGGTAB"
str2 = "GXTXAYB"
print(longest_common_subsequence(str1, str2))
```

**解析：** 最长公共子序列问题是一个经典的动态规划问题。通过构建一个二维数组 `dp`，其中 `dp[i][j]` 表示 `str1` 的前 `i` 个字符和 `str2` 的前 `j` 个字符的最长公共子序列长度。最后，返回 `dp[m][n]` 即可得到最长公共子序列的长度。

##### 2. 股票买卖（Best Time to Buy and Sell Stock）

**题目：** 给定一个数组 `prices`，其中 `prices[i]` 是第 `i` 天的股票价格。如果在一个明确的时间范围内只能（无限次地）完成一笔买卖交易，设计一个算法来找到最大利润。

**答案：** 使用贪心算法。

```python
def max_profit(prices):
    max_profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i - 1]:
            max_profit += prices[i] - prices[i - 1]
    return max_profit

# 示例
prices = [7, 1, 5, 3, 6, 4]
print(max_profit(prices))
```

**解析：** 该问题可以使用贪心算法解决。遍历数组 `prices`，只要当前价格比前一天高，就买入股票并卖出，以此类推。最后返回总利润。

#### 二、面试题库

##### 1. 翻转字符串里的单词（Reverse Words in a String）

**题目：** 编写一个函数，实现字符串中单词的翻转。例如，输入字符串 `"the sky is blue"`，输出 `"blue is sky the"`。

**答案：** 使用栈实现。

```python
def reverse_words(s):
    words = []
    word = []
    for c in s:
        if c == ' ':
            words.append(''.join(word))
            word = []
        else:
            word.append(c)
    words.append(''.join(word))
    return ' '.join(words[::-1])

# 示例
s = "the sky is blue"
print(reverse_words(s))
```

**解析：** 该问题可以通过栈实现。遍历字符串 `s`，遇到空格将当前单词压入栈，否则将字符压入当前单词。最后，将栈中单词依次弹出并拼接，即可得到翻转后的字符串。

##### 2. 有效的括号（Valid Parentheses）

**题目：** 给定一个字符串 `s` ，其中只包含 `'('`、`')'`、`'{'`、`'}'`、`'['` 和 `']'` 这些字符，判断字符串是否有效。

**答案：** 使用栈实现。

```python
def isValid(s):
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}
    for c in s:
        if c in mapping:
            top_element = stack.pop() if stack else '#'
            if mapping[c] != top_element:
                return False
        else:
            stack.append(c)
    return not stack

# 示例
s = "()[]{}"
print(isValid(s))
```

**解析：** 该问题可以通过栈实现。遍历字符串 `s`，对于每个字符，如果它是右括号，则将栈顶元素弹出并与该字符进行匹配，否则将字符压入栈。最后，如果栈为空，则字符串有效。
