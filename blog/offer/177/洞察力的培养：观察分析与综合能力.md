#### 二、算法编程题库

##### 1. 写一个算法，找出数组中的最大和次大元素。

**题目描述：** 给定一个整数数组 `nums`，找出数组中的最大和次大元素。假设每个输入只包含一个答案，且保证数组的最小大小为 2。

**Python 代码：**

```python
def find_two_largest(nums):
    max1, max2 = float('-inf'), float('-inf')
    for num in nums:
        if num > max1:
            max2 = max1
            max1 = num
        elif num > max2:
            max2 = num
    return max1, max2

nums = [3, 10, 7, 6, 2]
print(find_two_largest(nums))  # 输出应为 (10, 7)
```

**解析：**
此题考查算法的优化和基础知识，特别是关于变量赋值和循环的理解。使用两个变量 `max1` 和 `max2` 来记录最大和次大元素，通过一次遍历即可得到结果。

##### 2. 实现一个堆排序算法。

**题目描述：** 编写一个函数，使用堆排序算法对一个整数数组进行排序。

**Python 代码：**

```python
def heapify(arr, n, i):
    largest = i
    left = 2 * i + 1
    right = 2 * i + 2

    if left < n and arr[i] < arr[left]:
        largest = left

    if right < n and arr[largest] < arr[right]:
        largest = right

    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

def heap_sort(arr):
    n = len(arr)

    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)

    for i in range(n - 1, 0, -1):
        arr[i], arr[0] = arr[0], arr[i]
        heapify(arr, i, 0)

arr = [12, 11, 13, 5, 6, 7]
heap_sort(arr)
print("Sorted array is:", arr)
```

**解析：**
此题考查数据结构和算法的理解，特别是关于堆（Heap）和排序算法的掌握。堆排序算法包括建立最大堆和逐步提取堆顶元素，直到堆为空。

##### 3. 实现一个查找所有重复元素的函数。

**题目描述：** 给定一个整数数组 `nums`，编写一个函数来找出数组中的所有重复元素。数组中可能包含重复的元素，但至少有一个元素是唯一的。

**Python 代码：**

```python
def find_duplicates(nums):
    duplicates = []
    seen = set()

    for num in nums:
        if num in seen:
            duplicates.append(num)
        else:
            seen.add(num)

    return duplicates

nums = [1, 2, 3, 1, 5, 2, 3, 5]
print(find_duplicates(nums))  # 输出应为 [1, 2, 3, 5]
```

**解析：**
此题考查集合（Set）数据结构的应用，以及基础的循环和条件判断。使用一个集合来记录已经看到的元素，如果发现一个元素已经在集合中，则它是一个重复元素。
