### 4. 算法编程题库

**题目：** 实现一个基于协同过滤的推荐系统。

**答案：** 

```python
import numpy as np

def collaborative_filtering(ratings, k=10):
    # ratings 是一个 N * M 的矩阵，表示 N 个用户对 M 个物品的评分
    # k 是邻居的数量
    
    # 计算用户之间的相似度矩阵
    similarity_matrix = np.dot(ratings, ratings.T) / np.linalg.norm(ratings, axis=1)[:, np.newaxis]
    
    # 选择相似度最高的 k 个邻居
    top_k_neighbors = np.argsort(-similarity_matrix, axis=1)[:, :k]
    
    # 为每个用户计算推荐评分
    recommendations = np.zeros_like(ratings)
    for i in range(ratings.shape[0]):
        neighbor_ratings = ratings[top_k_neighbors[i], :]
        recommendations[i, :] = np.dot(neighbor_ratings, similarity_matrix[i, top_k_neighbors[i]]) / np.sum(similarity_matrix[i, top_k_neighbors[i]])
    
    return recommendations

# 示例
ratings_matrix = np.array([[5, 3, 0, 1], [4, 0, 0, 1], [1, 1, 0, 5], [1, 0, 0, 4], [0, 1, 5, 4]])
recommendations = collaborative_filtering(ratings_matrix)
print(recommendations)
```

**解析：** 上述代码实现了基于用户协同过滤的推荐系统，通过计算用户之间的相似度矩阵，为每个用户生成推荐评分。

### 4. 算法编程题库

**题目：** 实现一个基于协同过滤的推荐系统。

**答案：**

```python
import numpy as np

def collaborative_filtering(ratings, k=10):
    # ratings 是一个 N * M 的矩阵，表示 N 个用户对 M 个物品的评分
    # k 是邻居的数量

    # 计算用户之间的相似度矩阵
    similarity_matrix = np.dot(ratings, ratings.T) / np.linalg.norm(ratings, axis=1)[:, np.newaxis]
    
    # 选择相似度最高的 k 个邻居
    top_k_neighbors = np.argsort(-similarity_matrix, axis=1)[:, :k]
    
    # 为每个用户计算推荐评分
    recommendations = np.zeros_like(ratings)
    for i in range(ratings.shape[0]):
        neighbor_ratings = ratings[top_k_neighbors[i], :]
        recommendations[i, :] = np.dot(neighbor_ratings, similarity_matrix[i, top_k_neighbors[i]]) / np.sum(similarity_matrix[i, top_k_neighbors[i]])
    
    return recommendations

# 示例
ratings_matrix = np.array([[5, 3, 0, 1], [4, 0, 0, 1], [1, 1, 0, 5], [1, 0, 0, 4], [0, 1, 5, 4]])
recommendations = collaborative_filtering(ratings_matrix)
print(recommendations)
```

**解析：** 上述代码实现了基于用户协同过滤的推荐系统，通过计算用户之间的相似度矩阵，为每个用户生成推荐评分。

import numpy as np

def collaborative_filtering(ratings, k=10):
    # ratings 是一个 N * M 的矩阵，表示 N 个用户对 M 个物品的评分
    # k 是邻居的数量

    # 计算用户之间的相似度矩阵
    similarity_matrix = np.dot(ratings, ratings.T) / np.linalg.norm(ratings, axis=1)[:, np.newaxis]
    
    # 选择相似度最高的 k 个邻居
    top_k_neighbors = np.argsort(-similarity_matrix, axis=1)[:, :k]
    
    # 为每个用户计算推荐评分
    recommendations = np.zeros_like(ratings)
    for i in range(ratings.shape[0]):
        neighbor_ratings = ratings[top_k_neighbors[i], :]
        recommendations[i, :] = np.dot(neighbor_ratings, similarity_matrix[i, top_k_neighbors[i]]) / np.sum(similarity_matrix[i, top_k_neighbors[i]])
    
    return recommendations

# 示例

### 4. 算法编程题库

**题目：** 实现一个简单的协同过滤推荐系统。

**答案：** 

```python
import numpy as np

def collaborative_filtering(ratings, k=10):
    # ratings 是一个 N * M 的矩阵，表示 N 个用户对 M 个物品的评分
    # k 是邻居的数量

    # 计算用户之间的相似度矩阵
    similarity_matrix = np.dot(ratings, ratings.T) / np.linalg.norm(ratings, axis=1)[:, np.newaxis]
    
    # 选择相似度最高的 k 个邻居
    top_k_neighbors = np.argsort(-similarity_matrix, axis=1)[:, :k]
    
    # 为每个用户计算推荐评分
    recommendations = np.zeros_like(ratings)
    for i in range(ratings.shape[0]):
        neighbor_ratings = ratings[top_k_neighbors[i], :]
        recommendations[i, :] = np.dot(neighbor_ratings, similarity_matrix[i, top_k_neighbors[i]]) / np.sum(similarity_matrix[i, top_k_neighbors[i]])
    
    return recommendations

# 示例
ratings_matrix = np.array([[5, 3, 0, 1], [4, 0, 0, 1], [1, 1, 0, 5], [1, 0, 0, 4], [0, 1, 5, 4]])
recommendations = collaborative_filtering(ratings_matrix)
print(recommendations)
```

**解析：** 上述代码实现了基于用户协同过滤的推荐系统，通过计算用户之间的相似度矩阵，为每个用户生成推荐评分。
