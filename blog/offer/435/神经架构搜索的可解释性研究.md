**题目：** 编写一个简单的遗传算法，用于搜索最优神经网络结构。

**答案：** 下面是一个使用遗传算法搜索最优神经网络结构的简单示例：

```python
import random
import numpy as np

def generate_initial_population(pop_size, n_layers, n_neurons_per_layer):
    population = []
    for _ in range(pop_size):
        individual = []
        for layer in range(n_layers):
            if layer == 0:
                individual.append(random.randint(1, n_neurons_per_layer))
            else:
                individual.append(random.randint(1, n_neurons_per_layer // 2))
        population.append(individual)
    return population

def fitness_function(individual, X, y):
    # 构建神经网络结构
    layers = [individual[layer] for layer in range(len(individual))]
    model = build_model(layers)
    
    # 训练模型
    model.fit(X, y, epochs=10, batch_size=16, verbose=0)
    
    # 评估模型性能
    score = model.evaluate(X, y, verbose=0)
    return score[1]

def crossover(parent1, parent2):
    # 选择交叉点
    crossover_point = random.randint(1, len(parent1) - 1)
    # 交叉生成新个体
    child = parent1[:crossover_point] + parent2[crossover_point:]
    return child

def mutation(individual):
    # 选择突变点
    mutation_point = random.randint(0, len(individual) - 1)
    individual[mutation_point] = random.randint(1, 10)
    return individual

def main():
    # 参数设置
    pop_size = 100
    n_layers = 3
    n_neurons_per_layer = 10
    n_iterations = 50
    
    # 生成初始种群
    population = generate_initial_population(pop_size, n_layers, n_neurons_per_layer)
    
    for _ in range(n_iterations):
        # 计算适应度
        fitness_scores = [fitness_function(individual, X, y) for individual in population]
        
        # 选择适应度较高的个体作为父代
        selected_parents = random.choices(population, weights=fitness_scores, k=2)
        
        # 交叉生成子代
        child = crossover(selected_parents[0], selected_parents[1])
        
        # 突变子代
        child = mutation(child)
        
        # 替换种群中的最差个体
        worst_individual = min(population, key=lambda x: fitness_function(x, X, y))
        population.remove(worst_individual)
        population.append(child)

    best_individual = None
```

1. 生成初始种群，每个个体表示一种可能的神经网络结构。
2. 定义适应度函数，用于评估神经网络结构的性能。
3. 选择适应度较高的个体作为父代。
4. 通过交叉和突变操作生成子代。
5. 将子代中的最优个体替换种群中的最差个体。
6. 迭代进行，直至达到预设的迭代次数。

通过这个简单的遗传算法，可以自动搜索出性能优异的神经网络结构。
