### 面试题解析

#### 1. 什么是 CAP 定理？

**答案：** CAP 定理是指在一个分布式系统中，一致性（Consistency）、可用性（Availability）和分区容错性（Partition tolerance）三者之间，只能同时满足两项。

**解析：** CAP 定理指出，在分布式系统中，由于网络的不稳定性，无法同时保证一致性、可用性和分区容错性。根据实际需求，可以在这三个特性中选择两项进行优化。

#### 2. 什么是二分查找？

**答案：** 二分查找是一种在有序数组中查找特定元素的算法。算法的基本思想是：每次将查找区间缩小一半，逐步逼近目标元素。

**解析：** 二分查找的时间复杂度为 O(logn)，相比于线性查找的 O(n)，效率显著提高。在实际应用中，二分查找广泛用于数据库、搜索引擎等场景。

#### 3. 请简述 TCP 和 UDP 的区别。

**答案：** TCP（传输控制协议）和 UDP（用户数据报协议）都是传输层协议，但它们在可靠性、速度和传输方式等方面存在差异。

- **可靠性：** TCP 提供可靠的数据传输，确保数据的完整性和顺序；UDP 则不保证数据的完整性和顺序。
- **速度：** TCP 的传输速度相对较慢，但更稳定；UDP 的传输速度较快，但稳定性较差。
- **传输方式：** TCP 是连接导向的，需要在传输前建立连接；UDP 是非连接导向的，无需建立连接。

#### 4. 什么是哈希表？

**答案：** 哈希表是一种数据结构，它通过哈希函数将键映射到数组索引，从而实现快速查找、插入和删除操作。

**解析：** 哈希表在搜索引擎、数据库等场景中具有广泛的应用。通过合理的哈希函数和哈希表实现，可以显著提高数据的访问速度。

### 算法编程题解析

#### 1. 求最大子序和

**题目：** 给定一个整数数组，求出数组中连续子序列的最大和。

**示例：**
```
输入：[1, -2, 3, 4, -5]
输出：6
```

**答案：**
```python
def max_subarray_sum(nums):
    if not nums:
        return 0
    max_sum = nums[0]
    current_sum = nums[0]
    for i in range(1, len(nums)):
        current_sum = max(nums[i], current_sum + nums[i])
        max_sum = max(max_sum, current_sum)
    return max_sum

# 测试
print(max_subarray_sum([1, -2, 3, 4, -5]))  # 输出 6
```

#### 2. 单调栈

**题目：** 使用单调栈实现一个函数，用于计算数组中的每个元素对应的最小值。

**示例：**
```
输入：[3, 4, 2, 1]
输出：[1, 1, 1, 2]
```

**答案：**
```python
def min_stack(nums):
    stack = []
    result = []
    for num in nums:
        if not stack or num < stack[-1]:
            stack.append(num)
            result.append(stack[-1])
        else:
            stack.append(stack[-1])
            result.append(stack[-2])
    return result

# 测试
print(min_stack([3, 4, 2, 1]))  # 输出 [1, 1, 1, 2]
```

#### 3. 拓扑排序

**题目：** 使用拓扑排序实现一个函数，用于判断一个有向无环图（DAG）是否可以完成拓扑排序。

**示例：**
```
输入：[['A', 'B'], ['B', 'C'], ['C', 'D']]
输出：True
```

**答案：**
```python
def canFinish(numCourses, prerequisites):
    indegrees = [0] * numCourses
    graph = [[] for _ in range(numCourses)]
    for course1, course2 in prerequisites:
        graph[course1].append(course2)
        indegrees[course2] += 1
    queue = deque()
    for i, indegree in enumerate(indegrees):
        if indegree == 0:
            queue.append(i)
    while queue:
        course = queue.popleft()
        numCourses -= 1
        for next_course in graph[course]:
            indegrees[next_course] -= 1
            if indegrees[next_course] == 0:
                queue.append(next_course)
    return numCourses == 0

# 测试
print(can_finish(4, [['A', 'B'], ['B', 'C'], ['C', 'D']])))  # 输出 True
```
