#### 1. 最长公共子序列

**题目：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。

**答案：**

使用动态规划求解最长公共子序列问题。

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if str1[i-1] == str2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[m][n]
```

**解析：** 通过构建一个二维数组 `dp`，其中 `dp[i][j]` 表示 `str1` 的前 `i` 个字符和 `str2` 的前 `j` 个字符的最长公共子序列的长度。动态规划的核心思想是利用子问题的最优解来求解原问题。

def intersection(nums1, nums2):

#### 3. 股票买卖

**题目：** 给定一个数组 `prices`，其中 `prices[i]` 是第 `i` 天的股票价格。如果在一个日子里，你最多只能完成一笔交易（即买入和卖出一次股票），请返回你所能获得的最大利润。

**答案：**

使用一次遍历求解最大利润。

```python
def max_profit(prices):
    max_profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i-1]:
            max_profit += prices[i] - prices[i-1]
    return max_profit
```

**解析：** 通过遍历数组，记录当前价格与前一天价格之差，如果差值为正，则累加到 `max_profit` 中。

#### 4. 求最长回文子串

**题目：** 给定一个字符串 `s`，求它的最长回文子串。

**答案：**

使用动态规划求解最长回文子串问题。

```python
def longest_palindromic_substring(s):
    n = len(s)
    dp = [[False] * n for _ in range(n)]
    start, max_len = 0, 1

    for i in range(n):
        dp[i][i] = True

    for i in range(n-1, -1, -1):
        for j in range(i+1, n):
            if s[i] == s[j]:
                if j - i == 1 or dp[i+1][j-1]:
                    dp[i][j] = True
                    if max_len < j - i + 1:
                        start = i
                        max_len = j - i + 1

    return s[start:start+max_len]
```

**解析：** 通过构建一个二维数组 `dp`，其中 `dp[i][j]` 表示字符串 `s` 的第 `i` 个字符到第 `j` 个字符是否为回文。动态规划的核心思想是利用子问题的最优解来求解原问题。

#### 5. 求两个字符串的最小编辑距离

**题目：** 给定两个字符串 `word1` 和 `word2`，求它们之间的最小编辑距离。

**答案：**

使用动态规划求解最小编辑距离问题。

```python
def min_edit_distance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(m+1):
        for j in range(n+1):
            if i == 0:
                dp[i][j] = j
            elif j == 0:
                dp[i][j] = i
            elif word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1

    return dp[m][n]
```

**解析：** 通过构建一个二维数组 `dp`，其中 `dp[i][j]` 表示将字符串 `word1` 的前 `i` 个字符编辑成字符串 `word2` 的前 `j` 个字符所需的最小编辑次数。动态规划的核心思想是利用子问题的最优解来求解原问题。

#### 6. 求二叉树的层序遍历

**题目：** 给定一个二叉树，求它的层序遍历结果。

**答案：**

使用广度优先搜索（BFS）求解二叉树的层序遍历。

```python
from collections import deque

def level_order_traversal(root):
    if not root:
        return []

    queue = deque([root])
    result = []

    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        result.append(level)

    return result
```

**解析：** 通过队列实现广度优先搜索，逐层遍历二叉树，将每层的节点值存储在列表中，最终得到层序遍历结果。

#### 7. 求两个数的最大公约数

**题目：** 给定两个整数 `a` 和 `b`，求它们的最大公约数。

**答案：**

使用辗转相除法求解两个数的最大公约数。

```python
def max_gcd(a, b):
    while b:
        a, b = b, a % b
    return a
```

**解析：** 通过不断用较小数去除较大数，直到余数为 0，此时的较大数即为最大公约数。

#### 9. 求两个数组的交集并集

**题目：** 给定两个整数数组 `nums1` 和 `nums2`，求它们的交集和并集。

**答案：**

使用集合（Set）求解两个数组的交集和并集。

```python
def intersection_and_union(nums1, nums2):
    return list(set(nums1) & set(nums2)), list(set(nums1) | set(nums2))
```

**解析：** 通过集合的交集和并集操作求解交集和并集。

#### 10. 求二叉树的深度

**题目：** 给定一个二叉树，求它的深度。

**答案：**

使用递归求解二叉树的深度。

```python
def tree_depth(root):
    if not root:
        return 0
    return 1 + max(tree_depth(root.left), tree_depth(root.right))
```

**解析：** 通过递归遍历二叉树，计算每个节点的深度，并返回最大深度。
