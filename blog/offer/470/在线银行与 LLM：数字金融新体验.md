### 算法编程题与答案解析

#### 1. 实现一个算法，判断一个字符串是否为回文。

**代码示例：**

```python
def is_palindrome(s):
    return s == s[::-1]

# 测试
print(is_palindrome("racecar"))  # 输出：True
print(is_palindrome("hello"))    # 输出：False
```

**解析：** 通过字符串切片操作，将原字符串与反转后的字符串进行比较，判断是否为回文。

#### 2. 实现一个算法，找出一个整数数组中的重复元素。

**代码示例：**

```python
def find_duplicates(nums):
    return [num for num, count in Counter(nums).items() if count > 1]

# 测试
print(find_duplicates([1, 2, 3, 4, 5, 5, 6]))  # 输出：[5]
```

**解析：** 使用 Python 的 `Counter` 函数对数组中的元素进行计数，然后筛选出出现次数大于 1 的元素。
