```python
group_a = data[data['group'] == 'A']['satisfaction']
```

3. **优化策略：** 基于分析结果制定优化策略。

**实例代码：**
```python
# Python 示例代码
import pandas as pd
from scipy.optimize import linprog

# 数据预处理
data = pd.read_csv('supply_chain_data.csv')

# 建模与分析
objective = [-data['unit_cost'] * data['quantity']]  # 最小化成本
constraints = [[data['capacity_limit']], [data['demand']]]
bounds = [(0, None)] * data.shape[0]

result = linprog(objective, constraints=constraints, bounds=bounds, method='highs')

# 优化策略
optimal_quantity = result.x
data['optimal_quantity'] = optimal_quantity

# 评估效果
from sklearn.metrics import mean_squared_error

mse = mean_squared_error(data['actual_quantity'], optimal_quantity)
print('MSE:', mse)
```
