### 算法编程题 1：编写一个函数，判断一个整数是否为焦虑指数。

**题目描述：** 一个整数被称为焦虑指数，如果该整数是 3 的倍数且末位数字为 1，那么它就是焦虑指数。请编写一个函数，判断一个整数是否为焦虑指数。

**答案：**

```python
def is_anxiety_index(n):
    return n % 3 == 0 and n % 10 == 1

# 测试
print(is_anxiety_index(21))  # True
print(is_anxiety_index(24))  # False
```

**解析：** 通过对整数的取模运算，可以判断一个整数是否为 3 的倍数以及末位数字是否为 1。这两个条件都满足时，该整数就是焦虑指数。

### 算法编程题 2：编写一个函数，计算一组整数的平均焦虑指数。

**题目描述：** 给定一组整数，请编写一个函数，计算这些整数中焦虑指数的平均值。

**答案：**

```python
def average_anxiety_index(numbers):
    anxiety_indices = [n for n in numbers if is_anxiety_index(n)]
    return sum(anxiety_indices) / len(anxiety_indices) if anxiety_indices else 0

# 测试
print(average_anxiety_index([21, 24, 27, 30]))  # 27.0
print(average_anxiety_index([22, 25, 28, 31]))  # 0.0
```

**解析：** 通过列表推导式筛选出焦虑指数，然后计算它们的平均值。如果不存在焦虑指数，返回 0。

### 算法编程题 3：编写一个函数，找出一个数组中最长的焦虑序列。

**题目描述：** 给定一个数组，一个序列被称为焦虑序列，如果序列中的每个数都是焦虑指数。请编写一个函数，找出一个数组中最长的焦虑序列的长度。

**答案：**

```python
def longest_anxiety_sequence(nums):
    longest = 0
    current = 0

    for num in nums:
        if is_anxiety_index(num):
            current += 1
            longest = max(longest, current)
        else:
            current = 0

    return longest

# 测试
print(longest_anxiety_sequence([21, 24, 27, 30, 33]))  # 3
print(longest_anxiety_sequence([22, 25, 28, 31, 34]))  # 0
```

**解析：** 遍历数组，如果当前数是焦虑指数，则增加当前序列长度，更新最长序列长度；如果不是，则重置当前序列长度。

### 算法编程题 4：编写一个函数，找出一个字符串中的焦虑子串。

**题目描述：** 给定一个字符串，一个子串被称为焦虑子串，如果子串中的每个字符都是焦虑字符（即字符的 ASCII 码值是 3 的倍数）。请编写一个函数，找出一个字符串中的最长焦虑子串。

**答案：**

```python
def longest_anxiety_substring(s):
    longest = ""
    current = ""

    for char in s:
        if ord(char) % 3 == 0:
            current += char
            longest = max(longest, current)
        else:
            current = ""

    return longest

# 测试
print(longest_anxiety_substring("abc123def456"))  # "123"
print(longest_anxiety_substring("abc123"))  # "123"
```

**解析：** 遍历字符串，如果当前字符是焦虑字符，则将其添加到当前子串中，更新最长子串；如果不是，则重置当前子串。注意使用 `ord(char)` 来获取字符的 ASCII 码值。

均衡饮食有助于提供身体所需的营养，增强免疫力。建议多吃蔬菜、水果、全谷物、低脂肪蛋白质和健康脂肪。

**示例：** 每个工作日晚上进行 30 分钟快走，周末参加一次 1 小时的游泳活动。

学会放松技巧，如深呼吸、冥想或瑜伽，有助于缓解压力，提高生活质量。

**示例：** 每天晚上睡前进行 10 分钟的深呼吸练习，或在周末尝试一次瑜伽课程。

### 5. 保持社交活动

与家人、朋友和同事保持良好的社交关系，有助于获得情感支持，提高心理健康。

**示例：** 每周安排一次与朋友的聚会，或每月与同事一起参加团队活动。

### 6. 学习时间管理

合理安排时间，提高工作效率，有助于减轻工作压力。

**示例：** 制定每日工作计划，优先处理重要且紧急的任务，合理安排休息时间。

定期体检可以帮助及早发现健康问题，采取预防措施。
