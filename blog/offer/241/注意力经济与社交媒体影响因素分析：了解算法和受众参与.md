### 6. 简单的协同过滤推荐算法

**题目：** 编写一个简单的基于用户评分的协同过滤推荐算法，推荐用户可能喜欢的商品。

**答案：**
以下是一个简单的基于用户评分的协同过滤推荐算法实现：

```python
import numpy as np

def collaborative_filtering(ratings, k=10):
    # 计算平均评分
    mean_ratings = np.mean(ratings, axis=1)

    # 计算用户-用户相似度矩阵
    similarity_matrix = ratings.T.dot(ratings) / np.linalg.norm(ratings, axis=1).dot(np.linalg.norm(ratings, axis=0))
    similarity_matrix = (similarity_matrix + similarity_matrix.T) / 2

    # 计算邻居用户的评分预测
    neighbor_ratings = (similarity_matrix * (ratings - mean_ratings[:, np.newaxis])).sum(axis=0)
    predicted_ratings = neighbor_ratings + mean_ratings

    # 获取用户未评分的商品
    unrated_items = np.where(ratings == 0)[0]

    # 预测未评分商品的评分
    predicted_scores = predicted_ratings[unrated_items]

    # 排序并返回前k个推荐商品
    return np.argsort(predicted_scores[-k:])[::-1]

# 示例数据
ratings = np.array([[5, 3, 0, 1],
                    [4, 0, 0, 1],
                    [1, 5, 0, 0],
                    [0, 4, 5, 0],
                    [0, 2, 3, 4]])

recommendations = collaborative_filtering(ratings, k=2)
print("Recommended items:", recommendations)
```

**解析：**
协同过滤算法的核心思想是找到与目标用户相似的用户群体，然后根据这些用户的评分预测目标用户的评分。该实现中，我们首先计算每个用户的平均评分，然后计算用户之间的相似度矩阵，最后使用这些相似度矩阵预测未评分商品的评分。

### 7. 实现基于内容的推荐算法

**题目：** 编写一个基于内容的推荐算法，根据用户历史喜欢的商品推荐类似的新商品。

**答案：**
以下是一个简单的基于内容的推荐算法实现：

```python
import numpy as np

def content_based_filtering(item_features, user_history, k=10):
    # 计算用户历史喜欢商品的特征均值
    user_history_avg = np.mean(user_history, axis=0)

    # 计算每个商品与用户历史喜欢商品特征的相似度
    similarity_matrix = np.dot(user_history_avg, item_features.T)

    # 获取用户未浏览的商品
    unrated_items = np.where(np.isnan(item_features))[0]

    # 预测未浏览商品的相似度
    predicted_similarity = similarity_matrix[unrated_items]

    # 排序并返回前k个推荐商品
    return np.argsort(-predicted_similarity)[:k]

# 示例数据
user_history = np.array([[1, 0, 1],
                         [0, 1, 0],
                         [1, 1, 1]])

item_features = np.array([[1, 0, 1],
                          [0, 1, 1],
                          [1, 1, 0]])

recommendations = content_based_filtering(item_features, user_history, k=2)
print("Recommended items:", recommendations)
```

**解析：**
基于内容的推荐算法通过计算用户历史喜欢商品的特性，然后寻找具有相似特性的新商品进行推荐。该实现中，我们首先计算用户历史喜欢商品的特性均值，然后计算每个商品与用户历史喜欢商品特性的相似度，最后根据相似度预测未浏览商品的推荐列表。
