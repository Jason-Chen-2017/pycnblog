##### 4. 最长公共子序列（LCS）

**题目：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。

**解析：** 使用动态规划算法，定义一个二维数组 `dp`，其中 `dp[i][j]` 表示 `str1` 的前 `i` 个字符和 `str2` 的前 `j` 个字符的最长公共子序列的长度。

**代码示例：**

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n+1) for _ in range(m+1)]
    
    for i in range(1, m+1):
        for j in range(1, n+1):
            if str1[i-1] == str2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    
    return dp[m][n]
```

##### 6. 深度优先搜索（DFS）

**题目：** 实现一个深度优先搜索算法，用于求解图的节点连通性。

**解析：** 深度优先搜索是一种用于遍历或搜索树或图的算法，它沿着一个分支遍历到该分支的末端，然后沿着下一个分支继续遍历。

**代码示例：**

```python
def dfs(graph, node, visited):
    if node not in visited:
        visited.add(node)
        for neighbor in graph[node]:
            dfs(graph, neighbor, visited)

# 示例使用
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E'],
}
visited = set()
dfs(graph, 'A', visited)
print(visited)
```

##### 10. 单源最短路径算法

**题目：** 给定一个带权重的无向图，实现单源最短路径算法（如 Dijkstra 算法），并求解图中每个顶点到指定源点的最短路径。

**解析：** Dijkstra 算法是一种用于求解单源最短路径的贪心算法。算法的基本思想是维护一个集合 `S`，其中包含已经确定最短路径的顶点，初始时 `S` 只包含源点。每次从剩余的顶点中选择一个距离源点最近的顶点加入 `S`，并更新其他顶点的最短路径。

**代码示例：**

```python
import heapq

def dijkstra(graph, source):
    distances = {vertex: float('infinity') for vertex in graph}
    distances[source] = 0
    priority_queue = [(0, source)]

    while priority_queue:
        current_distance, current_vertex = heapq.heappop(priority_queue)

        if current_distance > distances[current_vertex]:
            continue

        for neighbor, weight in graph[current_vertex].items():
            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(priority_queue, (distance, neighbor))

    return distances

# 示例使用
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1},
}
print(dijkstra(graph, 'A'))
```

##### 11. 暴力解法求最大子序列和

**题目：** 给定一个整数数组 `nums`，找出其中非空子序列的最大和。子序列必须连续。

**解析：** 暴力解法的思路是遍历所有可能的子序列，计算它们的和，并找到最大值。

**代码示例：**

```python
def max_subarray_sum(nums):
    max_sum = float('-infinity')
    n = len(nums)

    for i in range(n):
        for j in range(i, n):
            current_sum = sum(nums[i:j+1])
            max_sum = max(max_sum, current_sum)

    return max_sum

# 示例使用
nums = [1, -2, 3, 4]
print(max_subarray_sum(nums))
```

##### 12. 动态规划解法求最大子序列和

**题目：** 给定一个整数数组 `nums`，找出其中非空子序列的最大和。子序列必须连续。

**解析：** 动态规划解法的思路是维护一个数组 `dp`，其中 `dp[i]` 表示以 `nums[i]` 为结尾的最大子序列和。

**代码示例：**

```python
def max_subarray_sum_dp(nums):
    max_sum = nums[0]
    dp = [num for num in nums]

    for i in range(1, len(nums)):
        dp[i] = max(dp[i-1] + nums[i], nums[i])
        max_sum = max(max_sum, dp[i])

    return max_sum

# 示例使用
nums = [1, -2, 3, 4]
print(max_subarray_sum_dp(nums))
```
