#### 1. 求两个字符串的最长公共子序列

**题目描述：** 给定两个字符串`str1`和`str2`，求它们的最长公共子序列。

**答案：** 使用动态规划算法解决。

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

# 示例
str1 = "ABCD"
str2 = "ACDF"
print(longest_common_subsequence(str1, str2))  # 输出 2
```

**解析：** 使用二维数组`dp`存储子序列的长度，`dp[i][j]`表示`str1`的前`i`个字符和`str2`的前`j`个字符的最长公共子序列长度。根据状态转移方程，计算`dp`数组，最后返回`dp[m][n]`即为结果。

#### 2. 单源最短路径算法

**题目描述：** 给定一个加权无向图和源点`s`，求从`s`到其他所有点的最短路径。

**答案：** 可以使用迪杰斯特拉算法（Dijkstra算法）。

```python
import heapq

def dijkstra(graph, source):
    n = len(graph)
    dist = [float('inf')] * n
    dist[source] = 0
    priority_queue = [(0, source)]

    while priority_queue:
        curr_dist, curr_vertex = heapq.heappop(priority_queue)

        if curr_dist > dist[curr_vertex]:
            continue

        for neighbor, edge_weight in enumerate(graph[curr_vertex]):
            weight = curr_dist + edge_weight

            if weight < dist[neighbor]:
                dist[neighbor] = weight
                heapq.heappush(priority_queue, (weight, neighbor))

    return dist

# 示例
graph = [
    [0, 4, 0, 0, 0],
    [4, 0, 8, 0, 2],
    [0, 8, 0, 1, 0],
    [0, 0, 1, 0, 7],
    [0, 2, 0, 7, 0]
]
print(dijkstra(graph, 0))  # 输出 [0, 4, 5, 3, 9]
```

**解析：** 使用优先队列（小根堆）维护当前未访问的节点及其距离。初始化源点距离为0，其余节点距离为无穷大。循环遍历优先队列，更新未访问节点的最短路径距离，并继续处理未访问节点。最后返回最短路径距离数组。
