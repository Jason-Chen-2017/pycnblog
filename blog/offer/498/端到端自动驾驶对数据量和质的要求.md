**答案：** 自动驾驶数据预处理主要包括以下步骤：

##### 1. 数据归一化

**题目：** 编写一个Python函数，实现数据归一化。

**答案：** 

```python
import numpy as np

def normalize_data(data):
    return (data - np.mean(data)) / np.std(data)
```

**解析：** 数据归一化有助于提高模型训练效果，通过将特征值缩放到相同的范围，避免某些特征对模型影响过大。

##### 2. 数据增强

**题目：** 编写一个Python函数，实现图像旋转的数据增强。

**答案：** 

```python
import cv2
import numpy as np

def rotate_image(image, angle):
    (h, w) = image.shape[:2]
    center = (w / 2, h / 2)
    M = cv2.getRotationMatrix2D(center, angle, 1.0)
    rotated = cv2.warpAffine(image, M, (w, h))
    return rotated
```

**解析：** 数据增强有助于提高模型的泛化能力，通过旋转图像等操作，增加数据多样性。
