3. **影响力大：** 一个好的口碑可以迅速提升品牌知名度和影响力。

## 算法编程题库

### 6. 编写一个算法，计算社交网络中两个用户之间的影响力距离。

**答案：**

```python
from collections import deque

def influence_distance(graph, user1, user2):
    distances = {node: float('inf') for node in graph}
    distances[user1] = 0
    
    queue = deque([user1])
    while queue:
        current = queue.popleft()
        for neighbor in graph[current]:
            if distances[neighbor] == float('inf'):
                distances[neighbor] = distances[current] + 1
                queue.append(neighbor)
    
    return distances[user2]

# 示例
graph = {
    'A': ['B', 'C'],
    'B': ['D'],
    'C': ['D', 'E'],
    'D': ['E'],
    'E': []
}
user1 = 'A'
user2 = 'E'
print(influence_distance(graph, user1, user2))  # 输出 3
```

### 7. 编写一个算法，找出社交网络中的核心用户。

**答案：**

```python
from collections import Counter

def core_users(graph):
    degree = Counter()
    for node in graph:
        degree[node] = len(graph[node])
    return [node for node, count in degree.items() if count > 1]

# 示例
graph = {
    'A': ['B', 'C'],
    'B': ['D'],
    'C': ['D', 'E'],
    'D': ['E'],
    'E': []
}
print(core_users(graph))  # 输出 ['B', 'C', 'D']
```
