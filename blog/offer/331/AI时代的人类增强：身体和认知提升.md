#### 算法编程题库

##### 4. 设计一个算法，计算两个DNA序列之间的编辑距离

**题目：** 给定两个DNA序列，设计一个算法计算它们之间的编辑距离。

**答案：** 

以下是一个使用动态规划的Python代码实现：

```python
def edit_distance(seq1, seq2):
    m, n = len(seq1), len(seq2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(m+1):
        for j in range(n+1):
            if i == 0:
                dp[i][j] = j
            elif j == 0:
                dp[i][j] = i
            elif seq1[i-1] == seq2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])

    return dp[m][n]

# 测试
seq1 = "ACTG"
seq2 = "ACG"
print(edit_distance(seq1, seq2)) # 输出 1
```

**解析：** 该算法使用二维数组 `dp` 来存储子问题的解，其中 `dp[i][j]` 表示 `seq1[0...i]` 和 `seq2[0...j]` 之间的编辑距离。算法的时间复杂度为 `O(m*n)`，其中 `m` 和 `n` 分别为两个序列的长度。

##### 5. 设计一个算法，判断一个人体动作序列是否为平衡序列

**题目：** 给定一个人体动作序列，设计一个算法判断该序列是否为平衡序列。平衡序列的定义是：每个动作都有相应的反动作，且反动作为最后一个动作。

**答案：**

以下是一个使用栈的Python代码实现：

```python
def is_balanced(sequences):
    stack = []
    for seq in sequences:
        if seq not in ["A", "B", "C", "D"]:
            return False
        if seq in ["A", "B"]:
            stack.append(seq)
        elif seq in ["C", "D"]:
            if not stack or stack[-1] != "A" or stack[-1] != "B":
                return False
            stack.pop()
    return not stack

# 测试
sequences = ["A", "B", "C", "D", "A", "B", "C"]
print(is_balanced(sequences)) # 输出 True
```

**解析：** 该算法使用栈来跟踪动作序列。对于每个动作，如果它是 "A" 或 "B"，则将其入栈；如果它是 "C" 或 "D"，则检查栈顶元素是否为 "A" 或 "B"，如果不是，则返回 False。最后，如果栈为空，则说明序列平衡，返回 True。
