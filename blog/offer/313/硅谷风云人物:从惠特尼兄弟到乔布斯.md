#### 算法编程题库

**1. 给定一个整数数组，找到第一个只出现一次的元素。**

**输入：** `[2, 2, 3, 3, 4]`

**输出：** `4`

**答案解析：**

我们可以使用哈希表来解决这个问题。遍历数组，将每个元素的值作为键，其出现的次数作为值存储在哈希表中。然后再次遍历数组，检查每个元素的值在哈希表中的次数是否为1。以下是实现代码：

```python
def firstUniqInt(arr):
    hash_map = {}
    for num in arr:
        if num in hash_map:
            hash_map[num] += 1
        else:
            hash_map[num] = 1
    for num in arr:
        if hash_map[num] == 1:
            return num
    return -1
```

**2. 设计一个算法来查找字符串数组中的最长公共前缀。**

**输入：** `["flower", "flow", "flight"]`

**输出：** `"fl"`

**答案解析：**

我们可以通过比较字符串的字符来查找最长公共前缀。从第一个字符串开始，逐个比较后续字符串的相应位置字符，直到找到第一个不相匹配的字符。以下是实现代码：

```python
def longestCommonPrefix(strs):
    if not strs:
        return ""
    prefix = strs[0]
    for s in strs[1:]:
        i = 0
        while i < len(prefix) and i < len(s):
            if prefix[i] != s[i]:
                break
            i += 1
        prefix = prefix[:i]
    return prefix
```

**3. 给定一个整数数组，实现一个算法来找出数组中的下一个更大数字。**

**输入：** `[2, 1, 5, 7, 3, 0]`

**输出：** `[7, 0, 5, 0, 0, 0]`

**答案解析：**

我们可以使用单调栈来实现这个算法。遍历数组，使用栈存储尚未处理的元素。对于每个元素，从栈顶弹出元素，直到找到比当前元素更大的值或栈为空。如果找到比当前元素更大的值，则将当前元素的值更新为该值的索引，否则更新为0。以下是实现代码：

```python
def nextGreaterElements(nums):
    n = len(nums)
    ans = [0] * n
    stk = []
    for i in range(2 * n - 1):
        while stk and nums[stk[-1]] < nums[i % n]:
            ans[stk.pop()] = nums[i % n]
        stk.append(i % n)
    return ans
```
