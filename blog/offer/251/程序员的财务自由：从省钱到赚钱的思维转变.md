#### 三、算法编程题库及答案解析

**1. 股票买卖时机选择**

**题目：** 给定一个数组 prices ，其中 prices[i] 是第 i 天的股票价格。如果可以只完成最多两笔交易，请返回你最多可以从中获取多少利润。

**答案：** 动态规划思路，定义一个二维数组 dp ，其中 dp[i][0] 表示第 i 天选择不持有股票的最大利润，dp[i][1] 表示第 i 天选择持有股票的最大利润。状态转移方程如下：

```python
dp[i][0] = max(dp[i-1][0], dp[i-1][1] + prices[i])
dp[i][1] = max(dp[i-1][1], -prices[i])
```

初始条件：dp[0][0] = 0，dp[0][1] = -prices[0]。

**2. 股票买卖的最佳时机**

**题目：** 给定一个数组 prices ，其中 prices[i] 是第 i 天的股票价格。如果只能完成一笔交易，请返回你能够获取的最大利润。如果不能获取任何利润，返回 0 。

**答案：** 暴力解法：枚举第 i 天买入，第 j 天卖出，计算利润并取最大值。时间复杂度 O(n^2)。

```python
max_profit = max(prices[j] - prices[i] for i in range(n) for j in range(i+1, n)]
```

优化解法：使用变量 low 记录最小买入价格，变量 max_profit 记录最大利润。遍历数组，更新 low 和 max_profit 的值。

```python
low = float('inf')
max_profit = 0
for price in prices:
    low = min(low, price)
    max_profit = max(max_profit, price - low)
return max_profit
```

**3. 最佳买卖股票时机含冷冻期**

**题目：** 给定一个数组 prices ，其中 prices[i] 是第 i 天的股票价格。如果你最多只能完成一笔交易且需遵守以下规则：

- 卖出股票的前一天你必须要买进它。
- 你不能同时参与多笔交易。
- 你卖出股票后，你无法当天再买入股票（即冷冻期为 1 天）。

请返回你最多可以获取的利润。

**答案：** 动态规划思路，定义一个三维数组 dp ，其中 dp[i][0] 表示第 i 天选择不持有股票的最大利润，dp[i][1] 表示第 i 天选择持有股票的最大利润，dp[i][2] 表示第 i 天处于冷冻期的最大利润。状态转移方程如下：

```python
dp[i][0] = max(dp[i-1][0], dp[i-1][2])
dp[i][1] = max(dp[i-1][1], dp[i-1][2] - prices[i])
dp[i][2] = dp[i-1][1]
```

初始条件：dp[0][0] = 0，dp[0][1] = -prices[0]，dp[0][2] = 0。

**4. 现在能赚多少**

**题目：** 给定一个数组 prices ，其中 prices[i] 是第 i 天的股票价格。如果你可以在第 i 天买入股票并持有到第 j 天，你将获得 prices[j] - prices[i] 的利润。返回你可以从数组中获取的最大利润。

**答案：** 动态规划思路，定义一个一维数组 dp ，其中 dp[i] 表示前 i 天的最大利润。状态转移方程如下：

```python
dp[i] = max(dp[i-1], prices[i] - prices[i-max_days])
```

初始条件：dp[0] = 0。

**5. 股票的最大利润 II**

**题目：** 给定一个数组 prices ，其中 prices[i] 是第 i 天的股票价格。你只能无限次地完成一组买卖操作。在买卖操作中，你买入股票后，必须在同一天内卖出。设计一个算法来计算你所能获取的最大利润。

**答案：** 动态规划思路，定义一个一维数组 dp ，其中 dp[i] 表示前 i 天的最大利润。状态转移方程如下：
