##### 6. 编写一个简单的反序列化程序。

**题目：** 给定一个序列化后的二进制文件，编写程序将其反序列化为原始数据结构。

**答案：**

```python
import pickle

def deserialize(file_path):
    with open(file_path, 'rb') as f:
        data = pickle.load(f)
    return data

file_path = 'data.bin'
original_data = deserialize(file_path)
print(original_data)
```

**解析：** 该题考察对Python中的pickle模块的理解和应用，实现数据的反序列化。

##### 7. 编写一个排序算法。

**题目：** 编写一个快速排序算法，并分析其时间复杂度。

**答案：**

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

arr = [3, 6, 8, 10, 1, 2, 1]
sorted_arr = quicksort(arr)
print(sorted_arr)
```

**解析：** 该题考察对排序算法的理解和实现能力，快速排序算法的平均时间复杂度为O(nlogn)。
