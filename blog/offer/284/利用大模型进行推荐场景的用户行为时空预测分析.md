#### 25. 使用变换器（Transformer）进行文本分类

**题目：** 编写一个使用变换器（Transformer）进行文本分类的代码示例。

**答案：** 

```python
import tensorflow as tf
from tensorflow.keras.models import Model
from tensorflow.keras.layers import Embedding, Dense, Input, Flatten, GlobalAveragePooling1D

# 假设我们有文本数据 X，以及对应的标签 y
X = np.array(["I love this book", "This is a great movie", "I hate this food"]) # 文本数据
y = np.array([1, 0, 2]) # 对应的标签

# 定义输入层
input_text = Input(shape=(None,), dtype='int32')

# 创建嵌入层
embedding = Embedding(input_dim=10000, output_dim=16)(input_text)

# 创建变换器层
transformer = tf.keras.layers.Conv1D(filters=128, kernel_size=5, activation='relu')(embedding)
transformer = tf.keras.layers.GlobalAveragePooling1D()(transformer)

# 创建输出层
output = Dense(units=3, activation='softmax')(transformer)

# 创建模型
model = Model(inputs=[input_text], outputs=[output])
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# 训练模型
model.fit(X, y, epochs=100, batch_size=16)

# 使用模型进行预测
predictions = model.predict(X)
print(predictions)
```

**解析：** 此代码示例展示了如何使用TensorFlow和Keras库构建一个简单的Transformer模型进行文本分类。通过定义输入层、嵌入层、变换器层和输出层，编译模型并使用categorical_crossentropy损失函数进行训练，最后使用训练好的模型进行预测。
