1. **最长公共子序列**

**题目：** 给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。

**答案：** 使用动态规划算法，具体步骤如下：

- **定义状态：** `dp[i][j]` 表示 `str1` 的前 `i` 个字符和 `str2` 的前 `j` 个字符的最长公共子序列长度。
- **状态转移方程：**
  - 如果 `str1[i - 1] == str2[j - 1]`，则 `dp[i][j] = dp[i - 1][j - 1] + 1`；
  - 如果 `str1[i - 1] != str2[j - 1]`，则 `dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])`。
- **边界条件：** `dp[0][j] = dp[i][0] = 0`。

**代码示例：**

```python
# Python 代码示例
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```

2. **二分查找**

**题目：** 在一个有序数组中，查找一个给定的元素，并返回其索引。

**答案：** 使用二分查找算法，具体步骤如下：

- **初始化：** 设置左边界 `left` 为 0，右边界 `right` 为数组长度减 1。
- **循环条件：** 当 `left <= right` 时，执行以下步骤：
  - 计算中间索引 `mid = (left + right) // 2`；
  - 如果 `nums[mid] == target`，返回 `mid`；
  - 如果 `nums[mid] < target`，更新 `left = mid + 1`；
  - 如果 `nums[mid] > target`，更新 `right = mid - 1`。
- **返回结果：** 如果找到目标元素，返回其索引；否则，返回 -1。

**代码示例：**

```python
# Python 代码示例
def binary_search(nums, target):
    left, right = 0, len(nums) - 1

    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1
```

3. **深度优先搜索**

**题目：** 实现一个深度优先搜索（DFS）算法，用于求解连通图中的所有路径。

**答案：** 使用递归方法实现 DFS 算法，具体步骤如下：

- **初始化：** 创建一个空列表 `paths` 保存所有路径。
- **递归过程：**
  - 对于当前节点 `node`，将 `node` 添加到当前路径 `path`；
  - 如果 `node` 是目标节点，将当前路径 `path` 添加到 `paths`；
  - 对于 `node` 的每个未访问的邻居 `neighbor`，执行以下步骤：
    - 将 `neighbor` 添加到当前路径 `path`；
    - 递归调用 `find_paths`；
    - 从当前路径 `path` 中删除 `neighbor`。

**代码示例：**

```python
# Python 代码示例
def find_paths(graph, start, end):
    paths = []
    path = [start]

    def dfs(node):
        if node == end:
            paths.append(path[:])
        else:
            for neighbor in graph[node]:
                if neighbor not in path:
                    path.append(neighbor)
                    dfs(neighbor)
                    path.pop()

    dfs(start)
    return paths
```

4. **广度优先搜索**

**题目：** 实现一个广度优先搜索（BFS）算法，用于求解连通图中的最短路径。

**答案：** 使用队列实现 BFS 算法，具体步骤如下：

- **初始化：** 创建一个队列 `queue`，将起始节点 `(start, 0)` 加入队列，其中 `0` 表示起始节点的距离为 0。
- **循环条件：** 当 `queue` 不为空时，执行以下步骤：
  - 从队列中取出队首元素 `(node, dist)`；
  - 对于 `node` 的每个未访问的邻居 `neighbor`，执行以下步骤：
    - 将 `(neighbor, dist + 1)` 加入队列；
    - 记录 `(neighbor, dist + 1)` 为 `neighbor` 的最短路径；
    - 标记 `neighbor` 为已访问；
- **返回结果：** 如果找到目标节点，返回其最短路径；否则，返回 None。

**代码示例：**

```python
# Python 代码示例
from collections import deque

def breadth_first_search(graph, start, end):
    queue = deque([(start, 0)])
    visited = set()
    parent = {start: None}

    while queue:
        node, dist = queue.popleft()
        if node == end:
            path = []
            while node is not None:
                path.append(node)
                node = parent[node]
            return path[::-1]
        for neighbor in graph[node]:
            if neighbor not in visited:
                queue.append((neighbor, dist + 1))
                parent[neighbor] = node
                visited.add(neighbor)

    return None
```
