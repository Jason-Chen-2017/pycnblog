### 7. 如何分析宏观经济数据？

**题目：** 编写一个算法，用于分析宏观经济数据，包括GDP增长率、通货膨胀率、失业率等指标，并给出综合评价。

**答案：** 宏观经济数据分析算法设计如下：

```python
# Python 伪代码

def economic_analysis(data):
    # GDP增长率分析
    gdp_growth_rate = calculate_growth_rate(data['GDP'])

    # 通货膨胀率分析
    inflation_rate = calculate_growth_rate(data['CPI'])

    # 失业率分析
    unemployment_rate = data['Unemployment']

    # 综合评价
    evaluation = {}
    if gdp_growth_rate > 3:
        evaluation['GDP'] = '增长稳定'
    elif gdp_growth_rate > 0:
        evaluation['GDP'] = '增长缓慢'
    else:
        evaluation['GDP'] = '负增长'

    if inflation_rate < 2:
        evaluation['CPI'] = '稳定'
    elif inflation_rate < 5:
        evaluation['CPI'] = '温和上涨'
    else:
        evaluation['CPI'] = '过热'

    if unemployment_rate < 5:
        evaluation['Unemployment'] = '充分就业'
    elif unemployment_rate < 10:
        evaluation['Unemployment'] = '就业压力较大'
    else:
        evaluation['Unemployment'] = '就业市场严峻'

    return evaluation

def calculate_growth_rate(current, previous):
    # 计算增长率
    return (current - previous) / previous
```

**解析：** 宏观经济数据分析是衡量国家经济状况的重要手段。上述算法通过对GDP增长率、通货膨胀率和失业率等指标的分析，给出宏观经济状况的综合评价。这有助于决策者了解经济形势，制定相应的政策。
