```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

# 示例
arr = [3, 6, 8, 10, 1, 2, 1]
print(quick_sort(arr))  # 输出 [1, 1, 2, 3, 6, 8, 10]
```

**题目：** 设计一个缓存系统。

**答案：**

```python
class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = {}
        self.order = []

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.order.remove(key)
        self.order.append(key)
        return self.cache[key]

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.order.remove(key)
        elif len(self.cache) >= self.capacity:
            key_to_remove = self.order.pop(0)
            del self.cache[key_to_remove]
        self.cache[key] = value
        self.order.append(key)

# 示例
lru_cache = LRUCache(2)
lru_cache.put(1, 1)
lru_cache.put(2, 2)
print(lru_cache.get(1))  # 输出 1
lru_cache.put(3, 3)
print(lru_cache.get(2))  # 输出 -1 (因为 2 被移除)
```

**解析：** 这个实现使用哈希表和双向链表来模拟 LRU（Least Recently Used）缓存策略，保证缓存的大小不超过给定容量。

### 题目：实现一个函数，找出字符串中的所有唯一单词

#### 题目描述

给定一个字符串，请你找出其中的所有唯一单词，并按照字典序排列输出。

#### 输入

字符串（字符串中可能包含空格、标点符号、数字等）

#### 输出

按照字典序排列的所有唯一单词列表

#### 例子

输入："Hello, world!"

输出：["Hello", "world"]

### 解题思路

为了解决这个问题，我们可以采用以下步骤：

1. **清洗字符串**：将字符串中的标点符号和数字去除，并将字符串转换成小写，以便于统一处理。
2. **分割字符串**：使用空格将字符串分割成单词列表。
3. **去重**：使用集合（Set）数据结构来存储所有唯一的单词。
4. **排序**：将集合转换为列表并排序，得到最终结果。

### 详细步骤

#### 步骤 1：清洗字符串

我们可以使用正则表达式来去除字符串中的标点符号和数字。以下是清洗字符串的代码：

```python
import re

def clean_string(s):
    return re.sub(r'[^a-zA-Z\s]', '', s).lower()
```

这段代码中，`re.sub(r'[^a-zA-Z\s]', '', s)` 用于替换字符串中的所有非字母和非空格字符，`lower()` 方法用于将字符串转换成小写。

#### 步骤 2：分割字符串

使用空格将字符串分割成单词列表，可以使用 Python 的 `split()` 方法：

```python
def split_string(s):
    return s.split()
```

#### 步骤 3：去重

使用集合（Set）数据结构来存储所有唯一的单词：

```python
def unique_words(words):
    return set(words)
```

#### 步骤 4：排序

将集合转换为列表并排序：

```python
def sort_words(unique_words):
    return sorted(unique_words)
```

### 综合代码示例

将上述步骤整合在一起，得到完整的函数实现：

```python
def find_unique_words(s):
    cleaned_string = clean_string(s)
    words = split_string(cleaned_string)
    unique_words = unique_words(words)
    sorted_words = sort_words(unique_words)
    return sorted_words

# 示例

```

### 复杂逻辑解析

在上述步骤中，我们使用了正则表达式和集合数据结构，这些是解决此类问题的核心工具。正则表达式用于字符串的清洗，集合用于去重。以下是这两个工具的详细解释：

#### 正则表达式解析

- `[^a-zA-Z\s]`：这个模式表示匹配所有非字母和非空格的字符。`^` 表示取反，`a-zA-Z` 表示所有字母，`\s` 表示空格。
- `re.sub()`：这个函数用于替换字符串中的匹配项。第一个参数是匹配模式，第二个参数是替换的字符，第三个参数是被替换的字符串。

#### 集合数据结构解析

- `set()`：创建一个无序不重复的集合。在去重操作中，集合非常适合，因为它不允许重复的元素。
- `sorted()`：将集合转换为列表后，使用这个函数进行排序。这个函数会根据元素的自然顺序进行排序。

  - **答：** 不影响。我们在清洗字符串时已经将所有字母转换为小写，因此大小写字母被视为相同的。

- **问：如果输入的字符串为空，函数会返回什么？**
  - **答：** 函数会返回一个空列表 `[]`。

以下是解决题目“找出字符串中的所有唯一单词”的完整源代码实例：

```python
import re

def find_unique_words(s):
    """
    找出字符串中的所有唯一单词，并按照字典序排序输出。

    :param s: 输入的字符串
    :return: 按字典序排序的唯一单词列表
    """
    
    # 清洗字符串，去除标点符号和数字，并将字符串转换为小写
    cleaned_string = re.sub(r'[^a-zA-Z\s]', '', s).lower()
    
    # 分割字符串为单词列表
    words = cleaned_string.split()
    
    # 去除重复单词，得到唯一单词的集合
    unique_words = set(words)
    
    # 将集合转换为列表并排序
    sorted_words = sorted(unique_words)
    
    return sorted_words

# 示例字符串
sample_string = "Hello, world! Hello, Python!"

# 调用函数并打印结果
print(find_unique_words(sample_string))  # 输出 ['hello', 'python', 'world']
```

2. **函数定义**：`find_unique_words` 函数接收一个字符串参数 `s`，并返回一个列表，其中包含字符串中的所有唯一单词。

3. **清洗字符串**：使用 `re.sub(r'[^a-zA-Z\s]', '', s).lower()` 清洗字符串。这里 `re.sub` 函数将字符串中的非字母和非空格字符替换为空字符，`lower()` 方法确保所有字母都转换为小写。

4. **分割字符串**：使用 `split()` 方法将清洗后的字符串分割成单词列表。

5. **去重**：使用 `set(words)` 创建一个集合，自动去除重复的单词。

6. **排序**：使用 `sorted(unique_words)` 将集合转换为列表，并按照字典序排序。

7. **返回结果**：函数最后返回排序后的单词列表。
