**1. 编写一个函数，计算元宇宙时间银行中用户的虚拟货币总额。**

**题目描述：** 给定一个用户的时间银行记录列表，其中每个用户都有一个对应的虚拟货币金额。编写一个函数，计算所有用户的虚拟货币总额。

**输入：** 
```go
users := []map[string]int{
    {"user1": 100},
    {"user2": 200},
    {"user3": 300},
}
```

**输出：**
```go
total := 600
```

**答案：**
```go
package main

import "fmt"

func getTotal(users []map[string]int) int {
    total := 0
    for _, user := range users {
        total += user["user1"]
    }
    return total
}

func main() {
    users := []map[string]int{
        {"user1": 100},
        {"user2": 200},
        {"user3": 300},
    }
    total := getTotal(users)
    fmt.Println("Total:", total)
}
```

**解析：** 该函数遍历用户列表，将每个用户的虚拟货币金额累加到总金额中，最后返回总金额。
