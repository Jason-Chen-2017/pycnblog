##### 4. 编写一个函数，实现性别比例统计。

**题目描述：** 给定一个包含用户性别信息的列表，编写一个函数，统计列表中男性和女性的比例。

**输入：** 

```python
users = [
    {"name": "Alice", "gender": "F"},
    {"name": "Bob", "gender": "M"},
    {"name": "Charlie", "gender": "M"},
    {"name": "Diana", "gender": "F"},
    {"name": "Evan", "gender": "M"},
]
```

**输出：**

```python
{'male': 3, 'female': 2}
```

**答案：**

```python
def gender_ratio(users):
    gender_count = {'male': 0, 'female': 0}
    for user in users:
        if user['gender'] == 'M':
            gender_count['male'] += 1
        elif user['gender'] == 'F':
            gender_count['female'] += 1
    return gender_count

users = [
    {"name": "Alice", "gender": "F"},
    {"name": "Bob", "gender": "M"},
    {"name": "Charlie", "gender": "M"},
    {"name": "Diana", "gender": "F"},
    {"name": "Evan", "gender": "M"},
]
print(gender_ratio(users))
```

**解析：** 这个函数通过遍历用户列表，统计男性和女性的数量，并返回一个包含性别比例的字典。这种方法简单直接，易于理解。

##### 5. 编写一个函数，检测性别歧视言论。

**题目描述：** 给定一个包含文本的列表，编写一个函数，检测其中是否存在性别歧视言论。

**输入：**

```python
texts = [
    "All women should be nurses.",
    "Men are better drivers than women.",
    "All engineers should be male.",
    "Women are great at multitasking.",
]
```

**输出：**

```python
['All women should be nurses.', 'Men are better drivers than women.', 'All engineers should be male.']
```

**答案：**

```python
def detect_sexism(texts):
    sexist_phrases = [
        "All women should be", "All men should be", "Women are not", "Men are not",
        "Only women can", "Only men can", "Women are better at", "Men are better at",
    ]
    sexist_texts = []
    for text in texts:
        for phrase in sexist_phrases:
            if phrase in text:
                sexist_texts.append(text)
                break
    return sexist_texts

texts = [
    "All women should be nurses.",
    "Men are better drivers than women.",
    "All engineers should be male.",
    "Women are great at multitasking.",
]
print(detect_sexism(texts))
```

**解析：** 这个函数通过检查文本中是否存在一系列可能表现出性别歧视的短语，来确定文本中是否存在性别歧视言论。这种方法简单有效，但需要注意可能存在的误报和漏报。

##### 6. 编写一个函数，对性别歧视言论进行评分。

**题目描述：** 给定一个包含文本的列表，编写一个函数，对每个文本中的性别歧视言论进行评分。

**输入：**

```python
texts = [
    "All women should be nurses.",
    "Men are better drivers than women.",
    "All engineers should be male.",
    "Women are great at multitasking.",
]
```

**输出：**

```python
[
    {'text': "All women should be nurses.", 'score': 3},
    {'text': "Men are better drivers than women.", 'score': 4},
    {'text': "All engineers should be male.", 'score': 4},
    {'text': "Women are great at multitasking.", 'score': 1},
]
```

**答案：**

```python
def rate_sexism(texts):
    rating_phrases = {
        "All women should be": 3,
        "All men should be": 3,
        "Women are not": 4,
        "Men are not": 4,
        "Only women can": 3,
        "Only men can": 3,
        "Women are better at": 3,
        "Men are better at": 3,
    }
    results = []
    for text in texts:
        score = 0
        for phrase, rating in rating_phrases.items():
            if phrase in text:
                score = rating
                break
        results.append({'text': text, 'score': score})
    return results

texts = [
    "All women should be nurses.",
    "Men are better drivers than women.",
    "All engineers should be male.",
    "Women are great at multitasking.",
]
print(rate_sexism(texts))
```

**解析：** 这个函数根据文本中包含的特定短语，对性别歧视言论进行评分。评分越高，表示性别歧视言论的程度越严重。这种方法提供了一个简单的评分机制，但需要根据实际情况进行调整和完善。
