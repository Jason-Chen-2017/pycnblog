### 算法编程题库

#### 1. 题目：根据历史销售数据和未来市场需求预测，设计一个库存管理算法。

**答案：**

```python
def inventory_management(sales_data, future_demand):
    # 历史销售数据的均值
    average_sales = sum(sales_data) / len(sales_data)
    
    # 未来需求的倍数
    demand_multiplier = future_demand / average_sales
    
    # 库存量
    inventory = int(demand_multiplier * average_sales * 1.5)  # 安全库存
    
    return inventory

# 示例
sales_data = [100, 150, 200, 250, 300]
future_demand = 350
print("预测库存量：", inventory_management(sales_data, future_demand))
```

**解析：** 该算法通过计算历史销售数据的均值和未来需求的倍数，来确定一个合适的库存量。安全库存量是根据历史销售数据均值和未来需求倍数来计算的，以应对不确定性。

#### 2. 题目：设计一个运输路线优化算法，给定起点、终点和多个中间点，求出最优的运输路线。

**答案：**

```python
import heapq

def optimal_route(start, end, points):
    # Dijkstra 算法
    distances = {start: 0}
    queue = [(0, start)]
    while queue:
        current_distance, current_point = heapq.heappop(queue)
        if current_point == end:
            return current_distance
        for neighbor, weight in points[current_point].items():
            distance = current_distance + weight
            if neighbor not in distances or distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(queue, (distance, neighbor))
    return None

# 示例
start = 'A'
end = 'E'
points = {
    'A': {'B': 10, 'C': 20},
    'B': {'D': 5},
    'C': {'D': 15},
    'D': {'E': 20},
    'E': {}
}
print("最优运输路线长度：", optimal_route(start, end, points))
```

**解析：** 该算法使用 Dijkstra 算法来求解给定起点、终点和多个中间点的最优运输路线。算法的核心思想是使用优先队列来维护当前已知的距离，并逐步扩展到未访问的节点，直到找到终点。
