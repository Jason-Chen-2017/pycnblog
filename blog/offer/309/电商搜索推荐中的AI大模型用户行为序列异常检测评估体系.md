#### 1. 如何评估用户行为序列的异常检测效果？

**题目：** 在电商搜索推荐中，如何评估用户行为序列异常检测的效果？

**答案：**

* **准确率（Accuracy）：** 检测结果中正确分类的样本数占总样本数的比例。
* **召回率（Recall）：** 对于所有实际为异常的用户行为，检测出的异常行为数占实际异常行为总数的比例。
* **精确率（Precision）：** 检测结果中正确分类的异常行为数占检测出的异常行为总数的比例。
* **F1值（F1-score）：** 准确率和召回率的调和平均值，用于平衡这两个指标。

**代码示例：**

```python
from sklearn.metrics import accuracy_score, recall_score, precision_score, f1_score

# 假设y_true为实际标签，y_pred为预测结果
y_true = [0, 1, 0, 0, 1, 1]
y_pred = [0, 0, 1, 1, 1, 1]

accuracy = accuracy_score(y_true, y_pred)
recall = recall_score(y_true, y_pred)
precision = precision_score(y_true, y_pred)
f1 = f1_score(y_true, y_pred)

print("Accuracy:", accuracy)
print("Recall:", recall)
print("Precision:", precision)
print("F1-score:", f1)
```
