#### 算法编程题库

**4. 请用 Python 实现 K-均值聚类算法。**

**答案：** K-均值聚类算法是一种基于距离度量的聚类方法。以下是一个简单的 Python 实现：

```python
import numpy as np

def k_means(data, k, max_iterations):
    # 初始化聚类中心
    centroids = data[np.random.choice(data.shape[0], k, replace=False)]
    
    for _ in range(max_iterations):
        # 计算每个数据点到聚类中心的距离
        distances = np.linalg.norm(data - centroids, axis=1)
        
        # 分配数据点给最近的聚类中心
        labels = np.argmin(distances, axis=1)
        
        # 更新聚类中心
        new_centroids = np.array([data[labels == i].mean(axis=0) for i in range(k)])
        
        # 判断聚类中心是否收敛
        if np.linalg.norm(new_centroids - centroids) < 1e-5:
            break

        centroids = new_centroids
    
    return centroids, labels
```

**5. 请用 Python 实现 SVM 分类算法。**

**答案：** SVM（支持向量机）是一种二分类模型，以下是一个简单的 Python 实现：

```python
import numpy as np
from numpy.linalg import inv

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def svm_train(X, y, C):
    # 初始化权重和偏置
    w = np.zeros(X.shape[1])
    b = 0
    
    # 循环迭代
    for _ in range(1000):
        # 预测标签
        y_pred = sigmoid(np.dot(X, w) + b)
        
        # 计算损失函数
        loss = -y * np.log(y_pred) - (1 - y) * np.log(1 - y_pred)
        
        # 计算梯度
        dw = np.dot(X.T, (y - y_pred)) + C * w
        db = np.sum(y - y_pred)
        
        # 更新权重和偏置
        w -= 0.01 * dw
        b -= 0.01 * db
    
    return w, b
```

**6. 请用 Python 实现 K-最近邻（K-NN）分类算法。**

**答案：** K-最近邻是一种基于实例的学习方法，以下是一个简单的 Python 实现：

```python
import numpy as np

def k_nn(X_train, y_train, X_test, k):
    # 计算测试数据点到训练数据的距离
    distances = np.linalg.norm(X_test - X_train, axis=1)
    
    # 获取最近的 k 个邻居及其标签
    neighbors = np.argpartition(distances, k)[:k]
    neighbor_labels = y_train[neighbors]
    
    # 计算邻居标签的多数表决结果
    most_common = np.argmax(np.bincount(neighbor_labels))
    
    return most_common
```
