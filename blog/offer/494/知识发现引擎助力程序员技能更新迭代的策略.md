### 3. 如何通过知识发现引擎实现动态更新技术资讯？

**答案：**

```python
import requests

def get_tech_news():
    url = "https://api.example.com/tech_news"
    response = requests.get(url)
    if response.status_code == 200:
        news_list = response.json()
        return news_list
    else:
        return []

# 测试获取技术资讯
tech_news = get_tech_news()
for news in tech_news:
    print("标题：", news["title"])
    print("摘要：", news["summary"])
    print()
```
