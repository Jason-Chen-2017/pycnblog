### 12. 如何在GitHub上建立个人博客？

**题目：** 程序员如何在GitHub上建立个人博客？

**答案：**

要在GitHub上建立个人博客，程序员可以按照以下步骤操作：

1. **创建GitHub仓库：** 在GitHub上创建一个新仓库，用于存储博客内容和源代码。
2. **配置博客框架：** 选择一个适合的博客框架，如Hexo、Jekyll等，并在GitHub仓库中配置框架。
3. **编写文章：** 使用Markdown格式编写博客文章，并将其存储在仓库的特定目录中。
4. **部署博客：** 使用GitHub Pages功能将博客部署到GitHub上，使其对外可见。

**举例：** 你可以使用Hexo框架在GitHub上建立博客，按照以下步骤操作：

1. 安装Hexo：在本地电脑上安装Hexo，执行命令 `npm install -g hexo-cli`。
2. 创建仓库：在GitHub上创建一个名为 `yourname.github.io` 的仓库。
3. 配置Hexo：执行命令 `hexo init`，初始化Hexo项目。
4. 编写文章：在项目目录下创建Markdown文件，编写博客内容。
5. 部署博客：执行命令 `hexo generate` 和 `hexo deploy`，将博客部署到GitHub Pages。

**解析：** GitHub Pages是一种免费的静态站点托管服务，通过将博客框架和GitHub仓库结合起来，程序员可以在GitHub上建立个人博客，展示自己的技术文章和作品。
