#### 二、算法编程题库

##### 1. 最长公共子序列

**题目：** 给定两个字符串 `s1` 和 `s2`，找出它们的最长公共子序列。

**答案：** 最长公共子序列（Longest Common Subsequence，LCS）是一个经典算法问题。可以使用动态规划方法求解。

```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

s1 = "ABCD"
s2 = "ACDF"
print(longest_common_subsequence(s1, s2))  # 输出 3
```

**解析：** 通过动态规划，我们可以计算出 `s1` 和 `s2` 的最长公共子序列长度。在实现中，我们使用一个二维数组 `dp` 来存储中间结果，其中 `dp[i][j]` 表示 `s1` 的前 `i` 个字符和 `s2` 的前 `j` 个字符的最长公共子序列长度。

##### 2. 判断二叉树是否对称

**题目：** 给定一个二叉树，判断它是否对称。

**答案：** 判断二叉树是否对称可以通过递归实现。首先判断根节点的左右子树是否相同，然后递归判断左右子树的左右子树是否相同，以此类推。

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def isSymmetric(root):
    if root is None:
        return True
    return isMirror(root.left, root.right)

def isMirror(left, right):
    if left is None and right is None:
        return True
    if left is None or right is None:
        return False
    if left.val != right.val:
        return False
    return isMirror(left.left, right.right) and isMirror(left.right, right.left)

# 构建对称二叉树
root = TreeNode(1)
root.left = TreeNode(2)
root.right = TreeNode(2)
root.left.left = TreeNode(3)
root.left.right = TreeNode(4)
root.right.left = TreeNode(4)
root.right.right = TreeNode(3)

print(isSymmetric(root))  # 输出 True
```

**解析：** 在实现中，我们定义一个辅助函数 `isMirror` 来判断两个子树是否镜像对称。我们首先判断根节点的左右子树是否相同，然后递归判断左右子树的左右子树是否相同，以此类推。

##### 3. 合并区间

**题目：** 给定一个区间列表，合并所有重叠的区间。

**答案：** 合并区间可以通过排序和合并的方法实现。首先将区间按照起点排序，然后遍历区间，合并重叠的区间。

```python
def merge(intervals):
    if not intervals:
        return []

    intervals.sort(key=lambda x: x[0])
    result = [intervals[0]]

    for interval in intervals[1:]:
        last = result[-1]
        if interval[0] <= last[1]:
            last[1] = max(last[1], interval[1])
        else:
            result.append(interval)

    return result

intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]
print(merge(intervals))  # 输出 [[1, 6], [8, 10], [15, 18]]
```

**解析：** 在实现中，我们首先将区间按照起点排序，然后遍历区间，合并重叠的区间。当遇到一个区间与上一个区间重叠时，我们将它们的终点取最大值；否则，将当前区间添加到结果列表中。
