### 3. 算法编程题库

**题目1：编写一个Python函数，实现搜索排序算法。**

```python
def search_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(i+1, n):
            if arr[i] > arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr

arr = [64, 25, 12, 22, 11]
sorted_arr = search_sort(arr)
print(sorted_arr)
```

**答案解析：** 该函数使用冒泡排序算法对数组进行排序。

**题目2：编写一个Python函数，实现二分查找算法。**

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

arr = [64, 25, 12, 22, 11]
target = 25
result = binary_search(arr, target)
if result != -1:
    print("元素在数组中的索引是：", result)
else:
    print("元素不在数组中。")
```

**答案解析：** 该函数使用二分查找算法在有序数组中查找目标元素。
