#### 2. 亚轨道飞行器和轨道飞行器有什么区别？

**题目：** 亚轨道飞行器和轨道飞行器有什么区别？

**答案：** 亚轨道飞行器和轨道飞行器的区别主要在于它们的飞行轨迹：

1. **亚轨道飞行器：** 飞行轨迹没有完全进入轨道，不会绕地球运行，飞行高度一般在100公里以内。
2. **轨道飞行器：** 飞行轨迹进入地球轨道，能够绕地球运行，通常飞行高度在100公里以上。

### 二、算法编程题库

#### 1. 计算太空飞行器的轨道周期

**题目：** 已知太空飞行器的轨道半径为 r，请编写一个函数计算其轨道周期 T。

**答案：** 根据开普勒第三定律，轨道周期 T 与轨道半径 r 的关系为 \( T = 2\pi \sqrt{\frac{r^3}{GM}} \)，其中 G 为万有引力常数，M 为地球质量。

```python
import math

def calculate_orbit_period(r):
    G = 6.674 * 10**-11
    M = 5.972 * 10**24
    T = 2 * math.pi * math.sqrt(r**3 / (G * M))
    return T
```

#### 2. 模拟太空飞船在月球表面的着陆过程

**题目：** 编写一个函数模拟太空飞船在月球表面的着陆过程。假设飞船的初始速度为 v，月球表面的重力加速度为 g，请计算飞船着陆所需的时间 t。

**答案：** 飞船在月球表面的着陆过程可以看作是自由落体运动，着陆时间 t 可以通过公式 \( t = \frac{v}{g} \) 计算。

```python
def calculate_landing_time(v, g):
    t = v / g
    return t
```

#### 3. 计算太空旅游的成本

**题目：** 编写一个函数计算太空旅游的成本。假设太空飞行的费用为 f1，住宿费用为 f2，其他费用为 f3，请计算总费用 total_cost。

**答案：** 太空旅游的总费用为各部分费用的总和。

```python
def calculate_total_cost(f1, f2, f3):
    total_cost = f1 + f2 + f3
    return total_cost
```

#### 4. 预测未来太空旅游的需求量

**题目：** 编写一个函数预测未来太空旅游的需求量。假设当前需求量为 n，年增长率为 r，请计算 n 年后的需求量。

**答案：** 需求量 n 的增长可以看作是指数增长，预测公式为 \( n_{future} = n * (1 + r)^n \)。

```python
def predict_future_demand(n, r, n_years):
    n_future = n * (1 + r)**n_years
    return n_future
```
