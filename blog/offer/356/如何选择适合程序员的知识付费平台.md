### 2.1  计算字符串的最大单词数

**题目描述：** 给定一个字符串，编写一个函数来计算该字符串中的最大单词数。

**输入：** 字符串 s（1 <= s.length <= 1e4）

**输出：** 最大单词数

**示例：**
```plaintext
输入："hello"
输出：1
输入："hello world"
输出：2
```

**答案：**

```python
def max_word_count(s: str) -> int:
    words = s.split()
    word_count = Counter(words)
    return max(word_count.values())

# 示例
print(max_word_count("hello"))  # 输出：1
print(max_word_count("hello world"))  # 输出：2
```

**解析：** 使用 Python 的 `split()` 方法将字符串分割成单词，然后使用 `Counter` 计算每个单词的计数。最后，返回最大计数值。

### 2.4  计算字符串的编辑距离

**题目描述：** 给定两个字符串，编写一个函数来计算它们之间的编辑距离。

**输入：** 两个字符串 s1 和 s2（1 <= |s1|, |s2| <= 1e4）

**输出：** 编辑距离

**示例：**
```plaintext
输入："horse", "ros"
输出：3
输入："intention", "execution"
输出：5
```

**答案：**

```python
def min_edit_distance(s1: str, s2: str) -> int:
    dp = [[0] * (len(s2) + 1) for _ in range(len(s1) + 1)]

    for i in range(len(s1) + 1):
        for j in range(len(s2) + 1):
            if i == 0:
                dp[i][j] = j
            elif j == 0:
                dp[i][j] = i
            elif s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]
            else:
                dp[i][j] = 1 + min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1])

    return dp[len(s1)][len(s2)]

# 示例
print(min_edit_distance("horse", "ros"))  # 输出：3
print(min_edit_distance("intention", "execution"))  # 输出：5
```

**解析：** 使用动态规划计算两个字符串的编辑距离。定义一个二维数组 `dp`，其中 `dp[i][j]` 表示字符串 `s1` 的前 `i` 个字符和字符串 `s2` 的前 `j` 个字符之间的编辑距离。

### 2.5  检查字符串是否是数字

```python
import re

def is_number(s: str) -> bool:
    pattern = re.compile(r"^-?\d+(\.\d+)?$")
    return bool(pattern.match(s))

# 示例
print(is_number("123"))  # 输出：True
print(is_number("123.456"))  # 输出：True
print(is_number("hello"))  # 输出：False
```

**解析：** 使用正则表达式检查字符串是否匹配数字模式。正则表达式 `^-?\d+(\.\d+)?$` 匹配整数或浮点数。

### 2.6  删除字符串中的所有相邻重复字符

**题目描述：** 给定一个字符串，编写一个函数来删除字符串中的所有相邻重复字符，只保留一个字符。

**输入：** 字符串 s（1 <= s.length <= 1e4）

**输出：** 删除相邻重复字符后的字符串

**示例：**
```plaintext
输入："abbaca"
输出："ca"
输入："abbccdd"
输出："abcd"
```

**答案：**

```python
def remove_adjacent_duplicates(s: str) -> str:
    stack = []
    for char in s:
        if stack and char == stack[-1]:
            stack.pop()
        else:
            stack.append(char)
    return ''.join(stack)

# 示例
print(remove_adjacent_duplicates("abbaca"))  # 输出："ca"
print(remove_adjacent_duplicates("abbccdd"))  # 输出："abcd"
```

**解析：** 使用栈实现，遍历字符串，如果当前字符与栈顶字符相同，则弹出栈顶字符；否则，将当前字符压入栈中。最后，将栈中的字符拼接成字符串返回。

```plaintext
输出：False
```

**答案：**

```python
def is_alphabetically_ordered(s: str) -> bool:
    return all(s[i] <= s[i + 1] for i in range(len(s) - 1))

# 示例

print(is_alphabetically_ordered("abcc"))  # 输出：False
```

**解析：** 使用嵌套列表解析表达式，检查字符串中每个字符是否小于等于下一个字符。如果所有字符都满足这个条件，返回 True。

```plaintext
输入："2022-13-01"
输出：False
```

**答案：**

```python
def is_date_format(s: str) -> bool:
    pattern = re.compile(r"^\d{4}-\d{2}-\d{2}$")
    if not pattern.match(s):
        return False
    year, month, day = map(int, s.split('-'))
    return 1 <= month <= 12 and 1 <= day <= 31

# 示例
print(is_date_format("2022-12-31"))  # 输出：True
print(is_date_format("2022-13-01"))  # 输出：False
```

**解析：** 首先使用正则表达式检查字符串是否匹配日期的格式，然后解析日期的年、月、日，并检查月份是否在 1 到 12 之间，日期是否在 1 到 31 之间。
