#### 1. 最长公共子序列

**题目：** 字节跳动面试题：给定两个字符串，求它们的最长公共子序列。

**答案解析：**
我们可以使用动态规划的方法求解。定义一个二维数组 dp，其中 dp[i][j] 表示字符串 s1 的前 i 个字符与字符串 s2 的前 j 个字符的最长公共子序列长度。

```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[m][n]
```

#### 3. 单调栈

**题目：** 腾讯面试题：给定一个数组 prices，其中 prices[i] 是第 i 天的股票价格。如果一笔交易需要连续两天完成，则第一次购买股票和第二次出售股票必须发生在连续的两天，请返回获得的最大利润。

**答案解析：**
我们可以使用单调栈的方法来解决这个问题。首先，我们需要一个栈来存储股票价格的索引。遍历数组 prices，对于每个元素 prices[i]，我们需要找到离它最近的一个左边较小值和右边较小值，分别表示买入和卖出的最佳时机。

```python
def max_profit(prices):
    if not prices:
        return 0

    stack = []
    max_profit = 0
    for i, price in enumerate(prices):
        while stack and prices[stack[-1]] >= price:
            stack.pop()
        if stack:
            max_profit = max(max_profit, price - prices[stack[-1]])
        stack.append(i)

    for i in range(len(prices)-1, -1, -1):
        while stack and prices[stack[-1]] <= prices[i]:
            stack.pop()
        if stack:
            max_profit = max(max_profit, prices[i] - prices[stack[-1]])
        stack.append(i)

    return max_profit
```

#### 6. 搜索旋转排序数组

**题目：** 给定一个旋转排序的数组 nums ，和一个目标值 target ，请编写一个函数查找 target 是否存在于数组中。如果存在返回 true ，否则返回 false。

**算法：** 二分查找

```python
def search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return True
        if nums[left] <= nums[mid]:
            if nums[left] <= target < nums[mid]:
                right = mid - 1
            else:
                left = mid + 1
        else:
            if nums[mid] < target <= nums[right]:
                left = mid + 1
            else:
                right = mid - 1

    return False
```

#### 7. 最小栈

**题目：** 设计一个支持 push ，pop ，top 操作的栈，同时还要支持获取最小元素的值。push、pop 和 top 操作的时间复杂度都是 O(1) ，并且只使用常数空间。

**算法：** 双栈实现

```python
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = []

    def push(self, val: int) -> None:
        self.stack.append(val)
        if not self.min_stack or val < self.min_stack[-1]:
            self.min_stack.append(val)

    def pop(self) -> None:
        if self.stack.pop() == self.min_stack[-1]:
            self.min_stack.pop()

    def top(self) -> int:
        return self.stack[-1]

    def getMin(self) -> int:
        return self.min_stack[-1]
```

#### 8. 链表中的节点每 k 个一组翻转

**题目：** 给定一个链表，每 k 个节点一组进行翻转，并返回翻转后的链表。k 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 k 的整数倍，那么请将最后不足 k 个的节点保持原有顺序。

**算法：** 递归

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverseKGroup(head: ListNode, k: int) -> ListNode:
    if not head:
        return head

    dummy = ListNode(0)
    dummy.next = head
    prev = dummy
    while True:
        kth = prev
        count = 0
        while kth and count < k:
            kth = kth.next
            count += 1
        if count == k:
            next_group = kth
            prev.next, kth, prev = kth, prev.next, head
            head.next, head = None, reverse(head)
            head = next_group
        else:
            break

    return dummy.next
```

**题目：** 将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

**算法：** 递归

class ListNode:

        self.next = next

def mergeTwoLists(l1, l2):
    if l1 is None:
        return l2
    if l2 is None:
        return l1
    if l1.val < l2.val:
        l1.next = mergeTwoLists(l1.next, l2)
        return l1
    else:
        l2.next = mergeTwoLists(l1, l2.next)
        return l2

#### 10. 最长回文子串

**题目：** 给你一个字符串 s，找到最长的回文子串。

**算法：** 动态规划

```python
def longestPalindrome(s: str) -> str:
    n = len(s)
    dp = [[False] * n for _ in range(n)]
    start, max_len = 0, 1

    for i in range(n):
        dp[i][i] = True

    for i in range(n - 1, -1, -1):
        for j in range(i + 1, n):
            if s[i] == s[j]:
                if j - i == 0:
                    dp[i][j] = True
                elif j - i == 1:
                    dp[i][j] = True
                else:
                    dp[i][j] = dp[i + 1][j - 1]
                if dp[i][j]:
                    start = i
                    max_len = j - i + 1

    return s[start: start + max_len]
```

   - 本书全面介绍了深度学习的基本概念、技术和应用，适合对深度学习感兴趣的读者。

2. **《设计模式：可复用面向对象软件的基础》** by Erich Gamma, Richard Helm, Ralph Johnson, and John Vlissides
   - [在线资源](https://www.amazon.com/Design-Patterns-Objects-Introducing-Design/dp/0201633612)
   - 介绍了面向对象设计中的经典模式，对提升编程技能和设计能力有很大帮助。

3. **《Effective Java》** by Joshua Bloch
   - [在线资源](https://www.oracle.com/java/technologies/javase/effective-java.html)
   - 针对Java编程语言的使用，提供了78个建议，有助于写出更加高效、可读性更好的代码。

4. **《代码大全》** by Martin Fowler
   - [在线资源](https://www.amazon.com/Code-Complete-Practical-Techniques-Construction/dp/013419516X)
   - 系统讲解了编写高质量代码的方法和技巧，适合编程新手和资深开发者。

5. **《从零开始学深度学习》** by 李沐
   - [在线资源](https://d2l.ai/d2l-zh/)
   - 是一本适合初学者的深度学习入门书籍，内容深入浅出，便于理解和实践。

6. **《算法导论》** by Al Aho, John E. Hopcroft, and Jeffrey D. Ullman
   - [在线资源](https://www.amazon.com/Introduction-Algorithms-Second-Edition-Classical/dp/0201010413)
   - 详细介绍了算法的基本概念、设计和分析方法，是算法学习者的经典教材。

7. **《人工智能：一种现代的方法》** by Stuart J. Russell and Peter Norvig
   - [在线资源](https://www.amazon.com/Artificial-Intelligence-A-Modern-Approach/dp/0136042597)
   - 全面介绍了人工智能的基本概念、技术和应用，适合对人工智能感兴趣的读者。
