**答案解析：** 实现一个基于集体智慧的在线购物推荐系统，需要综合运用用户行为分析、协同过滤算法和机器学习等方法。以下是一个简单的基于物品的协同过滤算法的实现：

```python
import numpy as np

def collaborative_filtering(ratings, k=5):
    # 计算相似度矩阵
    similarity_matrix = np.dot(ratings, ratings.T) / np.linalg.norm(ratings, axis=1)[:, np.newaxis]

    # 对相似度矩阵进行 k 近邻排序
    sorted_indices = np.argsort(-similarity_matrix)

    # 计算推荐列表
    recommendations = []
    for user_id in range(len(ratings)):
        neighbors = sorted_indices[user_id, :k]
        neighbor_ratings = ratings[neighbors]
        item_ratings = ratings[user_id]
        recommendation_score = np.dot(neighbor_ratings, item_ratings) / np.linalg.norm(neighbor_ratings)
        recommendations.append(recommendation_score)

    return recommendations
```

**源代码实例：** 以上代码实现了一个简单的基于物品的协同过滤算法，通过计算用户之间的相似度，为用户生成个性化推荐列表。
