### 1. 使用协同过滤算法预测用户偏好

**题目：** 假设有一个包含用户和物品的评分矩阵，编写一个函数，使用协同过滤算法预测给定用户对未知物品的评分。

**答案：** 使用矩阵分解（例如Singular Value Decomposition, SVD）进行协同过滤预测。

**解析：**

```python
import numpy as np

def collaborative_filter(R, user_id, item_id, k=10, alpha=0.01, num_iterations=100):
    # R是评分矩阵，user_id和item_id是待预测的用户和物品索引
    # k是分解矩阵的维度
    # alpha是正则化参数
    # num_iterations是迭代次数

    # 初始化用户和物品的矩阵
    U = np.random.rand(num_iterations, R.shape[0], k)
    V = np.random.rand(num_iterations, R.shape[1], k)

    for i in range(num_iterations):
        # 计算预测评分
        predictions = U[i].dot(V[i].T)
        
        # 计算误差
        error = R - predictions
        
        # 更新用户和物品的矩阵
        U[i+1] = U[i] + alpha * (error * V[i].T - alpha * U[i])
        V[i+1] = V[i] + alpha * (error.T * U[i] - alpha * V[i])

    # 返回最终预测的评分
    return predictions[user_id, item_id]

# 示例评分矩阵
R = np.array([[5, 3, 0, 1],
              [4, 0, 0, 1],
              [1, 1, 0, 5],
              [1, 0, 0, 4],
              [2, 5, 0, 1]])

# 预测用户2对未知物品的评分
predicted_rating = collaborative_filter(R, 2, 3)
print(predicted_rating)
```

### 2. 使用大语言模型提取文本特征

**题目：** 使用大语言模型（例如BERT）提取一段文本的特征向量。

**答案：** 使用预训练的大语言模型，通过传递文本数据到模型，获取文本的嵌入向量。

**解析：**

```python
from transformers import BertTokenizer, BertModel

def extract_features(text, model_name='bert-base-uncased'):
    # 加载预训练的BERT模型和分词器
    tokenizer = BertTokenizer.from_pretrained(model_name)
    model = BertModel.from_pretrained(model_name)

    # 对文本进行分词并添加特殊 tokens
    inputs = tokenizer(text, return_tensors='pt', padding=True, truncation=True)

    # 通过模型获取文本的嵌入向量
    with torch.no_grad():
        outputs = model(**inputs)

    # 取最后一个隐藏层的平均值作为文本特征
    features = outputs.last_hidden_state.mean(dim=1)

    return features

# 示例文本
text = "This is an example sentence for BERT."

# 提取文本特征
features = extract_features(text)
print(features.shape)
```
