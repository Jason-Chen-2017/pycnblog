#### 5. 如何用 Python 编写一个跨部门沟通协调的算法？

**题目：** 请用 Python 编写一个简单的跨部门沟通协调算法，解决以下问题：

- 有 n 个部门，每个部门有一个负责人。
- 需要计算所有部门负责人的总沟通成本，即所有负责人之间的沟通次数之和。

**答案：** 

```python
def calculate_communication_cost(n, department_heads):
    # 初始化沟通次数之和
    communication_cost = 0

    # 遍历每个部门的负责人，计算与其他负责人的沟通次数
    for i in range(n):
        for j in range(i + 1, n):
            communication_cost += 1

    return communication_cost

# 测试数据
n = 5
department_heads = [1, 2, 3, 4, 5]

# 计算沟通成本
result = calculate_communication_cost(n, department_heads)
print("沟通成本：", result)
```

**解析：** 这个算法通过嵌套循环遍历所有负责人，计算每对负责人之间的沟通次数，然后将所有沟通次数累加得到总沟通成本。这里假设每个负责人都需要与其他负责人沟通，因此沟通次数等于 n * (n - 1) / 2。
