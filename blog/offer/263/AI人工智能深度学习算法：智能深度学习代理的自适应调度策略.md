以下是一个基于 Python 实现的自适应调度策略的示例：

```python
import random
import time

class Task:
    def __init__(self, id, priority, execution_time):
        self.id = id
        self.priority = priority
        self.execution_time = execution_time

    def execute(self):
        print(f"Executing task {self.id} with priority {self.priority}")
        time.sleep(self.execution_time)

def schedule_tasks(tasks):
    while tasks:
        # 按优先级排序
        sorted_tasks = sorted(tasks, key=lambda x: x.priority, reverse=True)

        # 执行优先级最高的任务
        current_task = sorted_tasks[0]
        current_task.execute()

        # 更新任务状态
        current_task.priority = random.randint(1, 10)

        # 从任务列表中删除已执行的任务
        tasks.remove(current_task)

def main():
    # 创建任务
    tasks = [Task(id=i, priority=random.randint(1, 10), execution_time=random.randint(1, 5)) for i in range(10)]

    # 调度任务
    schedule_tasks(tasks)

if __name__ == "__main__":
    main()
```

**源代码实例解析：**

- **Task 类**：表示一个任务，包括任务 ID、优先级和执行时间。
- **schedule_tasks 函数**：实现自适应调度策略。首先按优先级排序任务，然后执行优先级最高的任务。在任务执行后，更新任务优先级并从任务列表中删除已执行的任务。
- **main 函数**：创建任务并调用调度函数进行任务调度。
