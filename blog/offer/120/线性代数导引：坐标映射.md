#### 矩阵乘法算法的实现

**问题：** 实现矩阵乘法算法。

**答案解析：** 矩阵乘法是指两个矩阵相乘得到一个新的矩阵。矩阵乘法满足以下规则：

- 矩阵 A（m×n）和矩阵 B（n×p）可以相乘，结果矩阵 C（m×p）。
- 结果矩阵 C 的第 i 行第 j 列元素 C_ij 是矩阵 A 的第 i 行与矩阵 B 的第 j 列对应元素的内积。

以下是一个 Python 实现矩阵乘法的源代码示例：

```python
def matrix_multiply(A, B):
    m, n = len(A), len(B[0])
    p = len(B)
    C = [[0 for _ in range(p)] for _ in range(m)]
    for i in range(m):
        for j in range(p):
            for k in range(n):
                C[i][j] += A[i][k] * B[k][j]
    return C
```

**示例：** 假设矩阵 A 为：

```
A = [
    [1, 2],
    [3, 4]
]
```

矩阵 B 为：

```
B = [
    [5, 6],
    [7, 8]
]
```

使用矩阵乘法算法计算 C = A * B：

```python
C = matrix_multiply(A, B)
print(C)
```

输出结果为：

```
[
    [19, 22],
    [43, 50]
]
```
