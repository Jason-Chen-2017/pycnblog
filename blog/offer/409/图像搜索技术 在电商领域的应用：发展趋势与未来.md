**题目：** 简要解释图像搜索技术的基本原理。

**答案：** 图像搜索技术的基本原理通常包括以下几个步骤：

1. **图像特征提取：** 使用特征提取算法（如 SIFT、SURF、ORB 等）从输入图像中提取出具有独特性的特征点。
2. **特征向量计算：** 将提取出的特征点转换成特征向量，用于表示图像。
3. **特征匹配：** 将新图像的特征向量与数据库中的特征向量进行匹配，找到相似度最高的图像。
4. **结果展示：** 根据匹配结果，展示相似图像供用户查看。

def image_similarity(img1, img2):
    """
    计算两幅图像的余弦相似度。
    """
    # 将图像转换为特征向量
    vec1 = extract_features(img1)
    vec2 = extract_features(img2)
    
    # 计算两向量的余弦相似度
    similarity = np.dot(vec1, vec2) / (np.linalg.norm(vec1) * np.linalg.norm(vec2))
    return similarity

    """
