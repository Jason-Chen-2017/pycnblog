#### 面试题库

**1. 什么是单例模式？请描述其在编程中的典型应用场景。**

**答案：** 单例模式是一种设计模式，确保一个类仅有一个实例，并提供一个全局访问点。它常用于以下场景：

- **数据库连接池**：确保只有一个数据库连接实例，避免频繁地创建和销毁连接。
- **系统日志**：确保系统日志只有一个实例，避免多个进程同时写日志导致数据混乱。
- **线程池**：确保线程池只有一个实例，避免创建过多的线程造成资源浪费。

**解析：** 可以通过编写一个私有构造函数和使用静态变量来实现单例模式。例如：

```java
public class Singleton {
    private static Singleton instance;
    private Singleton() {}
    
    public static Singleton getInstance() {
        if (instance == null) {
            instance = new Singleton();
        }
        return instance;
    }
}
```

**2. 请解释递归和迭代在算法中的区别。**

**答案：** 递归和迭代是两种常用的算法实现方法，其主要区别在于执行过程的不同：

- **递归**：递归是一种方法调用自身的算法，每次调用都会创建一个新的栈帧，直到达到递归的终止条件。递归适合解决具有递归结构的问题，例如阶乘、递归搜索等。
- **迭代**：迭代是一种循环结构，通过重复执行某些操作来解决问题。迭代适合解决可以转化为循环结构的问题，例如遍历数组、计算斐波那契数列等。

**解析：** 递归和迭代各有优缺点，递归易于理解和实现，但可能会造成栈溢出；迭代适用于大数据集，但可能需要更多的内存空间。

**3. 请解释深度优先搜索（DFS）和广度优先搜索（BFS）的工作原理。**

**答案：** 深度优先搜索和广度优先搜索是两种常用的图搜索算法：

- **深度优先搜索（DFS）**：从起始节点开始，尽可能深地搜索图的分支。DFS适用于解决连通性问题、最小生成树等。
- **广度优先搜索（BFS）**：从起始节点开始，逐层地搜索图的节点。BFS适用于解决最短路径问题、图的最小环等。

**解析：** DFS和DFS的工作原理可以通过以下示例进行说明：

```python
# 深度优先搜索
def dfs(graph, node, visited):
    visited.add(node)
    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)

# 广度优先搜索
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    while queue:
        node = queue.popleft()
        if node not in visited:
            visited.add(node)
            for neighbor in graph[node]:
                if neighbor not in visited:
                    queue.append(neighbor)
```

#### 算法编程题库

**1. 给定一个整数数组，实现一个函数，找出数组中的最大元素。**

**答案：** 可以通过遍历数组，使用一个变量来记录最大元素，时间复杂度为 O(n)。

```python
def find_max(arr):
    max_val = arr[0]
    for num in arr:
        if num > max_val:
            max_val = num
    return max_val
```

**2. 给定一个字符串，实现一个函数，判断字符串是否为回文。**

**答案：** 可以通过比较字符串的首尾字符，逐渐向中间移动，判断是否为回文，时间复杂度为 O(n)。

```python
def is_palindrome(s):
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1
    return True
```
