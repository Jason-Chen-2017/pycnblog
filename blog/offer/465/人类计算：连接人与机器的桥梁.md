2. **阿里巴巴 02. 最长公共前缀**

3. **百度 03. 股票买卖**
   - **题目描述：** 给定一个整数数组 `prices`，其中 `prices[i]` 是第 `i` 天股票的价格。如果某人想在第 `i` 天买入股票并能在第 `j` 天（当 `j > i`）卖出股票，他可以获取的最大利润为 `prices[j] - prices[i]`。实现一个函数，计算并返回能够获得的最大利润。
   - **答案解析：** 动态规划，记录前一天的最大利润和当前最大利润。

4. **腾讯 04. 整数拆分**
   - **题目描述：** 给定一个正整数 `n`，将其拆分为若干个 1 到 9 的正整数之和，使得到的和最小。求最小的和。
   - **答案解析：** 动态规划，记录前 i 个数的和，求最小和。

5. **字节跳动 05. 合并两个有序链表**

   - **答案解析：** 遍历两个链表，比较当前节点的值，选择较小的值作为新链表的下一节点。

6. **拼多多 06. 盛最多水的容器**

#### 算法编程题库

1. **阿里巴巴 11. 二进制中 1 的个数**
   - **题目描述：** 编写一个函数，输入一个无符号整数，返回其二进制表示中 1 的个数。
   - **代码实例：**

   ```java
   public int hammingWeight(int n) {
       int count = 0;
       while (n != 0) {
           count += n & 1;
           n >>= 1;
       }
       return count;
   }
   ```

2. **百度 12. 最长公共子序列**
   - **题目描述：** 给定两个字符串 text1 和 text2，返回这两个字符串的最长公共子序列的长度。
   - **代码实例：**

   ```python
   def longestCommonSubsequence(text1, text2):
       m, n = len(text1), len(text2)
       dp = [[0] * (n + 1) for _ in range(m + 1)]
       for i in range(1, m + 1):
           for j in range(1, n + 1):
               if text1[i - 1] == text2[j - 1]:
                   dp[i][j] = dp[i - 1][j - 1] + 1
               else:
                   dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
       return dp[m][n]
   ```

3. **腾讯 13. 搜索旋转排序数组**
   - **题目描述：** 假设按照升序排序的数组在预先未知的某个点上进行了旋转。请找出并返回数组中的目标元素。
   - **代码实例：**

   ```javascript
   function search(nums, target) {
       let left = 0, right = nums.length - 1;
       while (left <= right) {
           mid = Math.floor((left + right) / 2);
           if (nums[mid] == target) {
               return mid;
           }
           if (nums[left] <= nums[mid]) {
               if (target >= nums[left] && target < nums[mid]) {
                   right = mid - 1;
               } else {
                   left = mid + 1;
               }
           } else {
               if (target > nums[right] && target <= nums[left]) {
                   right = mid - 1;
               } else {
                   left = mid + 1;
               }
           }
       }
       return -1;
   }
   ```

4. **字节跳动 14. 最长连续序列**
   - **题目描述：** 给定一个未排序的整数数组，找出最长连续序列的长度。
   - **代码实例：**

   ```csharp
   public int longestConsecutive(int[] nums) {
       if (nums.Length == 0) {
           return 0;
       }
       var set = new HashSet<int>(nums);
       var ans = 1;
       foreach (var num in set) {
           if (!set.Contains(num - 1)) {
               var currentNum = num;
               var currentLength = 1;
               while (set.Contains(currentNum + 1)) {
                   currentNum++;
                   currentLength++;
               }
               ans = Math.Max(ans, currentLength);
           }
       }
       return ans;
   }
   ```

5. **拼多多 15. 合并两个有序链表**

   - **代码实例：**

   ```python
   class ListNode:
       def __init__(self, val=0, next=None):
           self.val = val
           self.next = next

   def mergeTwoLists(l1, l2):
       if l1 is None:
           return l2
       if l2 is None:
           return l1
       if l1.val < l2.val:
           result = l1
           l1 = l1.next
       else:
           result = l2
           l2 = l2.next
       current = result
       while l1 is not None and l2 is not None:
           if l1.val < l2.val:
               current.next = l1
               l1 = l1.next
           else:
               current.next = l2
               l2 = l2.next
           current = current.next
       if l1 is not None:
           current.next = l1
       if l2 is not None:
           current.next = l2
       return result
   ```

6. **京东 16. 最小栈**
   - **题目描述：** 实现一个具有最小栈功能的栈，可以在 O(1) 时间获取栈的最小元素。
   - **代码实例：**

   ```java
   class MinStack {
       Stack<Integer> stack;
       Stack<Integer> minStack;

       public MinStack() {
           stack = new Stack<>();
           minStack = new Stack<>();
       }

       public void push(int x) {
           stack.push(x);
           if (minStack.isEmpty() || x <= minStack.peek()) {
               minStack.push(x);
           }
       }

       public void pop() {
           if (stack.pop() == minStack.peek()) {
               minStack.pop();
           }
       }

       public int top() {
           return stack.peek();
       }

       public int getMin() {
           return minStack.peek();
       }
   }
   ```

7. **美团 17. 翻转单词顺序**
   - **题目描述：** 给定一个字符串，逐个翻转字符串中的所有单词。
   - **代码实例：**

   ```javascript
   function reverseWords(s) {
       let words = s.split(' ');
       let result = [];
       for (let i = 0; i < words.length; i++) {
           result.push(words[i].split('').reverse().join(''));
       }
       return result.reverse().join(' ');
   }
   ```

9. **滴滴出行 19. 最长公共前缀**
   - **题目描述：** 编写一个函数来查找字符串数组中的最长公共前缀。
   - **代码实例：**

   ```php
   function longestCommonPrefix($strs) {
       if (empty($strs)) {
           return "";
       }
       $prefix = $strs[0];
       for ($i = 1; $i < count($strs); $i++) {
           while (strpos($strs[$i], $prefix) !== 0) {
               $prefix = substr($prefix, 0, -1);
               if (empty($prefix)) {
                   return "";
               }
           }
       }
       return $prefix;
   }
   ```

10. **小红书 20. 等差数列求和**
    - **题目描述：** 给定一个整数数组 `nums`，返回数组中所有奇数长度子数组的和。
    - **代码实例：**

    ```c++
    int sumOddLengthSubarrays(vector<int>& nums) {
        int totalSum = 0;
        int n = nums.size();
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n - i; j++) {
                if ((j + 1) % 2 == 1) {
                    int subarraySum = 0;
                    for (int k = 0; k < j + 1; k++) {
                        subarraySum += nums[i + k];
                    }
                    totalSum += subarraySum;
                }
            }
        }
        return totalSum;
    }
    ```
