### 3.1 自注意力（Self-Attention）实现
**题目：** 请实现一个简单的自注意力层，并解释其原理。

**答案：** 自注意力层是一种注意力机制，用于计算输入序列中每个元素与其余元素之间的关联性。以下是一个简单的自注意力层实现：

```python
import tensorflow as tf

def self_attention(inputs, hidden_size):
    # 计算输入序列的嵌入表示
    embeddings = tf.keras.layers.Dense(hidden_size)(inputs)

    # 计算自注意力权重
    attention_weights = tf.keras.layers.Dense(1, activation='sigmoid')(embeddings)

    # 加权求和
    outputs = tf.reduce_sum(embeddings * attention_weights, axis=1)

    return outputs
```

### 3.2 多头注意力（Multi-Head Attention）实现
**题目：** 请实现一个简单的多头注意力层，并解释其原理。

**答案：** 多头注意力是一种扩展自注意力机制的方法，通过将输入序列拆分为多个子序列，并分别计算注意力权重。以下是一个简单的多头注意力层实现：

```python
import tensorflow as tf

def multi_head_attention(inputs, hidden_size, num_heads):
    # 计算输入序列的嵌入表示
    embeddings = tf.keras.layers.Dense(hidden_size)(inputs)

    # 计算自注意力权重
    attention_weights = tf.keras.layers.Dense(num_heads, activation='sigmoid')(embeddings)

    # 拆分输入序列为多个子序列
    inputs_split = tf.split(inputs, num_heads, axis=1)

    # 计算每个子序列的注意力权重
    attention_heads = [tf.reduce_sum(attention_weights[i] * inputs_split[i], axis=1) for i in range(num_heads)]

    # 将多个子序列的注意力结果拼接起来
    outputs = tf.concat(attention_heads, axis=1)

    return outputs
```
