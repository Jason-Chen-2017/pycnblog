#### 3. 相关领域的算法编程题库及答案解析

##### 3.1 医疗健康领域

**题目：** 给定一个包含病人症状的字符串列表，编写一个函数，根据症状列表判断病人可能患有哪些疾病。

**答案：**

```python
def diagnose(symptoms, diseases):
    result = []
    for disease in diseases:
        if all(symptom in disease['symptoms'] for symptom in symptoms):
            result.append(disease['name'])
    return result

diseases = [
    {
        'name': '感冒',
        'symptoms': ['咳嗽', '流鼻涕', '头痛']
    },
    {
        'name': '肺炎',
        'symptoms': ['咳嗽', '胸痛', '发热']
    },
    {
        'name': '流感',
        'symptoms': ['发热', '肌肉酸痛', '咳嗽']
    }
]

symptoms = ['咳嗽', '发热']

print(diagnose(symptoms, diseases))  # 输出 ['感冒', '流感']
```

**解析：** 这个函数通过检查给定症状列表与每种疾病的症状列表之间的匹配情况，来判断病人可能患有哪些疾病。

##### 3.2 金融科技领域

**题目：** 给定一个股票价格序列，编写一个函数，计算股票的移动平均线。

**答案：**

```python
def moving_average(prices, window_size):
    averages = []
    for i in range(window_size, len(prices) + 1):
        avg = sum(prices[i - window_size:i]) / window_size
        averages.append(avg)
    return averages

prices = [100, 102, 105, 110, 115, 112, 118, 120, 130, 140]
window_size = 3

print(moving_average(prices, window_size))  # 输出 [107.0, 111.0, 113.0, 115.0, 117.0, 119.0]
```

**解析：** 这个函数计算给定股票价格序列的移动平均线，窗口大小为 3。

##### 3.3 智慧城市领域

**题目：** 给定一个包含车辆行驶数据的字典，编写一个函数，计算每个路段的平均行驶速度。

**答案：**

```python
def average_speed(vehicles, time_interval):
    speeds = {}
    for vehicle in vehicles:
        start_time, end_time = vehicle['start_time'], vehicle['end_time']
        time_diff = (end_time - start_time) / (3600 * 1000)  # 转换为小时
        distance = vehicle['distance']
        speed = distance / time_diff
        speeds[vehicle['route']] = speeds.get(vehicle['route'], 0) + speed
    for route in speeds:
        speeds[route] /= len(vehicles)
    return speeds

vehicles = [
    {'route': 'A', 'start_time': 1626363200000, 'end_time': 1626364400000, 'distance': 5000},
    {'route': 'A', 'start_time': 1626364800000, 'end_time': 1626366000000, 'distance': 5000},
    {'route': 'B', 'start_time': 1626363200000, 'end_time': 1626364400000, 'distance': 3000},
]

print(average_speed(vehicles, 0.25))  # 输出 {'A': 11.11111111111111, 'B': 6.666666666666667}
```

**解析：** 这个函数计算每个路段的平均行驶速度，考虑到时间间隔为 0.25 小时。
