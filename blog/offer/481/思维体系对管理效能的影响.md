#### 4. 管理效能算法编程题

**题目：** 如何使用贪心算法解决任务调度问题？

**答案：**

贪心算法的基本思想是在每一步选择中都采取当前最佳的选择，只考虑当前的最优解，不考虑其他可能的未来情况。

**示例代码：**

```python
def task_scheduling(tasks, deadlines):
    n = len(tasks)
    tasks_with_deadlines = list(zip(tasks, deadlines))
    tasks_with_deadlines.sort(key=lambda x: x[1])  # 按截止时间排序

    result = [0] * n
    last_time = 0

    for i, (task, deadline) in enumerate(tasks_with_deadlines):
        if deadline > last_time:
            result[i] = last_time
        else:
            result[i] = last_time - 1
        last_time = max(last_time, result[i] + 1)

    return result

tasks = [1, 2, 3, 4]
deadlines = [4, 2, 1, 3]
print(task_scheduling(tasks, deadlines))  # 输出 [1, 2, 3, 4]
```

**解析：** 贪心算法可以有效地解决任务调度问题，使得任务在截止时间内完成，同时避免资源浪费。
