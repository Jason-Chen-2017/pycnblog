4. **评估：** 通过评估模型对集体潜意识强度的预测准确性，来衡量集体潜意识的强度。

**源代码实例：**

```python
import nltk
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.cluster import KMeans

# 假设已收集了一组文本数据

# 特征提取
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

# K-means 聚类
kmeans = KMeans(n_clusters=5)
kmeans.fit(X)

# 预测集体潜意识强度
def predict_strength(text):
    features = vectorizer.transform([text])
    cluster = kmeans.predict(features)[0]
    # 使用聚类中心距离作为强度指标
    strength = np.linalg.norm(kmeans.cluster_centers_[cluster] - features)
    return strength

# 示例
strength = predict_strength("全球脑与集体潜意识的研究具有重要意义。")
print("集体潜意识强度：", strength)
```
