### 7. 元宇宙中虚拟法庭的算法编程题库

**题目：** 编写一个算法，用于计算元宇宙中虚拟法庭的审理时间。

**输入：** 
- 案件数量（n）：一个正整数，表示每天提交的案件中，预计需要审理的案件数量。
- 法官数量（m）：一个正整数，表示虚拟法庭中可用的法官数量。
- 案件平均审理时间（t）：一个正整数，表示每个案件平均需要审理的时间（以小时为单位）。

**输出：** 
- 审理时间（hours）：一个正整数，表示虚拟法庭能够完成所有案件所需的预计时间（以小时为单位）。

**示例：** 
```
输入：n = 50, m = 3, t = 4
输出：审理时间 = 50
```
**解析：** 

- 假设每个法官每天可以审理 `t` 个案件，那么 `m` 个法官每天总共可以审理的案件数量为 `m * t`。
- 如果每天提交的案件中，预计需要审理的案件数量 `n` 小于等于 `m * t`，则虚拟法庭可以在一天内完成所有案件。
- 如果每天提交的案件中，预计需要审理的案件数量 `n` 大于 `m * t`，则需要的天数为 `ceil(n / (m * t))`，其中 `ceil` 函数用于向上取整。

**答案：** 

```python
import math

def calculate_trials_time(n, m, t):
    trials_time = math.ceil(n / (m * t))
    return trials_time

# 示例
n = 50
m = 3
t = 4
print(calculate_trials_time(n, m, t))  # 输出：审理时间 = 13
```
