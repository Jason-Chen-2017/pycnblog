##### 3. 编写一个函数，用于计算智能婴儿摇篮的温度调节范围。

**答案：** 温度调节范围可以根据设定的上下限和当前温度进行计算。

**源代码实例：**

```python
# Python 示例：计算温度调节范围
def calculate_temp_range(current_temp, min_temp, max_temp):
    temp_range = max_temp - min_temp
    adjusted_temp = current_temp - (temp_range / 2)
    return max(min(adjusted_temp, max_temp), min_temp)

# 使用示例
current_temp = 27
min_temp = 22
max_temp = 30
regulated_temp = calculate_temp_range(current_temp, min_temp, max_temp)
print(f"Regulated Temperature: {regulated_temp}°C")
```
