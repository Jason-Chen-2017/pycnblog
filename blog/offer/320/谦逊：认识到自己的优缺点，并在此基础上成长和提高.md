##### 1. 如何在团队合作中保持谦逊？

**面试题：** 请谈谈在团队合作中如何保持谦逊？

**答案：**

1. **倾听与尊重：** 在团队中，始终保持倾听的心态，尊重他人的意见和建议，无论对方的职位高低或经验多少。
2. **愿意接受批评：** 谦逊的心态体现在愿意接受他人的批评和建议，将其视为成长的机会。
3. **分享荣誉：** 在取得成绩时，将功劳归功于整个团队，而非个人。
4. **不断学习：** 谦逊的人会时刻保持学习的心态，不断充实自己，提升团队整体的竞争力。

**算法编程题：** 请实现一个函数，用于判断一个字符串是否为“谦逊字符串”。一个字符串被认为是“谦逊字符串”，当且仅当它满足以下条件：
- 字符串中所有的单词都由小写字母组成。
- 字符串中的单词之间用单个空格分隔。
- 字符串中没有前导或尾随空格。
- 字符串中至少有两个单词。

**代码示例：**

```python
def is_humble(s: str) -> bool:
    words = s.strip().split()
    return len(words) >= 2 and all(word.islower() for word in words)

# 测试
print(is_humble("i am humble")) # True
print(is_humble("i Am HUMBLE")) # False
print(is_humble("humble")) # False
```

##### 2. 如何在职业发展中保持谦逊？

2. **虚心请教：** 在遇到问题时，不吝啬向他人请教，学习他人的经验和观点。
3. **反思与总结：** 定期反思自己的工作和行为，总结经验教训，找出不足之处，不断改进。
4. **不断学习：** 职业发展中，保持谦逊的心态，不断学习新知识、新技能，跟上行业发展。

**算法编程题：** 请实现一个函数，用于找出字符串中的最长谦逊子字符串。谦逊子字符串定义为：一个子字符串，其中所有单词的长度都大于等于2，且按照字典序从小到大排列。

**代码示例：**

```python
def longest_humble_substring(s: str) -> str:
    words = s.split()
    max_length = 0
    start = 0
    for i in range(1, len(words)):
        if len(words[i]) >= 2 and (i == 1 or words[i] > words[i - 1]):
            if i - start + 1 > max_length:
                max_length = i - start + 1
                end = i
        else:
            start = i
    return ' '.join(words[start:end + 1])

# 测试
print(longest_humble_substring("this is a humble string")) # "is a"
print(longest_humble_substring("humble this is a string")) # "humble this is a"
```

**算法编程题：** 请实现一个函数，用于找出字符串中的最长谦逊子字符串。谦逊子字符串定义为：一个子字符串，其中所有单词的长度都大于等于2，且按照字典序从小到大排列。

**代码示例：**

```python
def longest_humble_substring(s: str) -> str:
    words = s.split()
    max_length = 0
    start = 0
    for i in range(1, len(words)):
        if len(words[i]) >= 2 and (i == 1 or words[i] > words[i - 1]):
            if i - start + 1 > max_length:
                max_length = i - start + 1
                end = i
        else:
            start = i
    return ' '.join(words[start:end + 1])

# 测试
```

```python
def is_humble(s: str) -> bool:

    return len(words) >= 2 and all(word.islower() for word in words)
```
