**答案：**

```go
func calculatePositivity(words []string) int {
    positivityScores := map[string]int{
        "快乐": 10,
        "兴奋": 8,
        "满足": 6,
        "幸福": 9,
        "焦虑": -7,
        "悲伤": -5,
        "愤怒": -8,
    }

    totalPositivity := 0
    for _, word := range words {
        totalPositivity += positivityScores[word]
    }

    return totalPositivity
}
```

**解析：** 此题考察编程实现情绪词语积极程度的计算。答案定义了一个哈希表，用于存储情绪词语及其积极程度得分。通过遍历输入的词语数组，计算总积极程度并返回。
