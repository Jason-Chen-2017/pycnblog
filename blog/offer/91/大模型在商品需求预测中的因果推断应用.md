#### 算法编程题库

##### 1. 使用 Python 实现线性回归模型进行商品需求预测。

**题目：** 编写一个 Python 函数，使用线性回归模型预测商品需求。输入为商品历史销售数据（时间序列），输出为预测的商品需求。

**代码示例：**

```python
import numpy as np
from sklearn.linear_model import LinearRegression

def predict_demand(history_sales):
    # 数据预处理
    X = np.array([i for i, _ in enumerate(history_sales)])
    y = np.array(history_sales)

    # 模型训练
    model = LinearRegression()
    model.fit(X, y)

    # 预测
    prediction = model.predict(np.array([len(history_sales)]))
    return prediction

# 示例数据
history_sales = [10, 15, 12, 18, 20, 22, 25]

# 预测
predicted_demand = predict_demand(history_sales)
print(predicted_demand)
```

##### 2. 使用 TensorFlow 实现卷积神经网络（CNN）进行商品需求预测。

**题目：** 编写一个 TensorFlow 函数，使用卷积神经网络（CNN）预测商品需求。输入为商品历史销售数据（时间序列），输出为预测的商品需求。

**代码示例：**

```python
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Conv1D

def build_cnn_model(input_shape):
    model = Sequential([
        Conv1D(filters=64, kernel_size=3, activation='relu', input_shape=input_shape),
        Dense(1)
    ])
    model.compile(optimizer='adam', loss='mse')
    return model

def predict_demand_cnn(history_sales):
    # 数据预处理
    X = np.array(history_sales).reshape(-1, 1)
    
    # 模型训练
    model = build_cnn_model(input_shape=(X.shape[1], X.shape[0]))
    model.fit(X, X, epochs=10, batch_size=1, verbose=0)

    # 预测
    predicted_demand = model.predict(np.array([history_sales[-1]]))
    return predicted_demand

# 示例数据
history_sales = [10, 15, 12, 18, 20, 22, 25]

# 预测
predicted_demand = predict_demand_cnn(history_sales)
print(predicted_demand)
```
