2. **如何进行文本分类？**
   **答案解析：**
   - 使用LLM生成文本特征向量。
   - 使用特征向量输入分类模型进行分类。

   ```python
   import openai
   import numpy as np
   from sklearn.model_selection import train_test_split
   from sklearn.metrics import accuracy_score

   openai.api_key = "your_api_key"

   def generate_features(texts, model="text-embedding-ada-002"):
       embeddings = openai.Embedding.create(input=texts, model=model)["data"]
       return np.array([emb["embedding"] for emb in embeddings])

   # 假设已有文本数据和标签
   texts = ["这是一条积极的评论", "这是一条消极的评论"]
   labels = [1, 0]  # 1表示积极，0表示消极

   # 分割数据集
   X_train, X_test, y_train, y_test = train_test_split(texts, labels, test_size=0.2, random_state=42)

   # 生成特征向量
   X_train_embedding = generate_features(X_train)
   X_test_embedding = generate_features(X_test)

   # 训练分类模型（如支持向量机）
   from sklearn.svm import SVC

   classifier = SVC(kernel="linear")
   classifier.fit(X_train_embedding, y_train)

   # 测试模型
   predictions = classifier.predict(X_test_embedding)
   accuracy = accuracy_score(y_test, predictions)
   print("测试准确率：", accuracy)
   ```

   以上代码展示了如何使用OpenAI的LLM模型生成文本特征向量，并使用特征向量进行文本分类。请注意，根据实际需求，可能需要调整模型类型、参数设置等。
