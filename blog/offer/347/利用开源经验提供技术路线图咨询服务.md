5. **题目示例：**

   ```markdown
   # 题目：二分查找

   **题目描述：** 在一个有序数组中查找一个目标值，并返回其索引。如果目标值不存在于数组中，返回 -1。

   **输入描述：** 
   ```
   arr = [1, 3, 5, 7, 9]
   target = 3
   ```

   **输出描述：** 
   ```
   索引：1
   ```

   **解决方案：**
   ```go
   func binarySearch(arr []int, target int) int {
       left, right := 0, len(arr)-1
       for left <= right {
           mid := (left + right) / 2
           if arr[mid] == target {
               return mid
           } else if arr[mid] < target {
               left = mid + 1
           } else {
               right = mid - 1
           }
       }
       return -1
   }
   ```

   **解析：** 这个示例展示了如何实现二分查找算法。通过不断缩小区间，二分查找可以在 O(log n) 时间内找到目标值。
   ```

**解析：** 通过构建一个丰富、分类清晰的算法编程题库，咨询师可以快速地为不同客户提供针对性的技术挑战，帮助他们巩固基础知识，提升解决实际问题的能力。
