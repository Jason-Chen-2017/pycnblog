### 算法编程题库

#### 1. 编写一个基于协同过滤算法的推荐系统。

**题目：** 编写一个基于协同过滤算法的推荐系统，实现对用户历史购买数据的推荐。

**答案：** 

```python
import numpy as np
from collections import defaultdict

# 假设用户-物品评分矩阵为R，用户数目为m，物品数目为n
R = np.array([
    [5, 3, 0, 1],
    [0, 0, 0, 2],
    [1, 1, 0, 0],
    [3, 0, 2, 0]
])

m, n = R.shape

# 计算用户之间的相似度矩阵
def calculate_similarity(R):
    sim = np.zeros((m, m))
    for i in range(m):
        for j in range(m):
            if i == j:
                sim[i][j] = 0
            else:
                sim[i][j] = np.dot(R[i], R[j]) / np.sqrt(np.dot(R[i], R[i]) * np.dot(R[j], R[j]))
    return sim

sim = calculate_similarity(R)

# 根据相似度矩阵和用户评分矩阵计算预测评分
def predict_scores(sim, R):
    pred = np.zeros((m, n))
    for i in range(m):
        for j in range(n):
            pred[i][j] = np.dot(sim[i], R[j])
    return pred

pred = predict_scores(sim, R)

# 根据预测评分矩阵生成推荐列表
def generate_recommendations(pred, R, top_n=3):
    rec = []
    for i in range(m):
        user_pred = pred[i]
        user_rating = R[i]
        sim_indices = np.argsort(user_pred)[-top_n:]
        for index in sim_indices:
            if user_rating[index] == 0:
                rec.append((index, user_pred[index]))
                break
    return rec

recommendations = generate_recommendations(pred, R)
print("Recommended items for each user:", recommendations)
```

**解析：** 这个例子中，我们首先计算用户之间的相似度矩阵，然后使用相似度矩阵和用户评分矩阵计算预测评分，最后根据预测评分矩阵生成推荐列表。

#### 2. 编写一个基于决策树的分类器。

**题目：** 编写一个基于决策树的分类器，用于预测用户是否会在一个月内购买商品。

**答案：**

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
import matplotlib.pyplot as plt

# 加载鸢尾花数据集
iris = load_iris()
X, y = iris.data, iris.target

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 构建决策树分类器
clf = DecisionTreeClassifier()
clf.fit(X_train, y_train)

# 预测测试集
y_pred = clf.predict(X_test)

# 评估分类器性能
from sklearn.metrics import accuracy_score
accuracy = accuracy_score(y_test, y_pred)
print("Accuracy:", accuracy)

# 可视化决策树
from sklearn.tree import plot_tree
plt.figure(figsize=(12, 8))
plot_tree(clf, filled=True, feature_names=iris.feature_names, class_names=iris.target_names)
plt.show()
```

**解析：** 这个例子中，我们首先加载鸢尾花数据集，然后划分训练集和测试集。接着，我们使用训练集构建决策树分类器，并使用测试集评估分类器的性能。最后，我们将决策树可视化。
