#### 一、面试题库

##### 1. 什么是单例模式？请举例说明。

**答案：** 单例模式是一种设计模式，确保一个类仅有一个实例，并提供一个访问它的全局访问点。

**举例：**

```java
public class Singleton {
    private static Singleton instance;

    private Singleton() {}

    public static Singleton getInstance() {
        if (instance == null) {
            instance = new Singleton();
        }
        return instance;
    }
}
```

**解析：** 这个示例中的 `Singleton` 类通过私有构造函数和静态的 `getInstance()` 方法来确保类只有一个实例。如果实例尚未创建，则创建它；否则，返回现有实例。

##### 2. 请解释并发和多线程的区别。

**答案：** 并发是指在多个事件或任务之间交替执行的能力，而多线程是指在程序中同时运行多个线程。

**解析：** 并发通常涉及操作系统层面的调度，而多线程通常涉及程序层面的设计。多线程是一种实现并发的方式，但并发也可以通过其他方法（如异步编程）实现。

##### 3. 什么是内存泄漏？如何检测和避免？

**答案：** 内存泄漏是指程序在分配内存后，无法释放不再使用的内存，导致内存逐渐消耗殆尽。

**检测和避免：**

- **检测：** 使用内存分析工具，如 Valgrind 或 Leaks。
- **避免：** 及时释放不再使用的对象或资源，避免使用全局变量，合理使用引用计数或垃圾回收机制。

##### 4. 请解释 MVC 设计模式。

**答案：** MVC（Model-View-Controller）是一种设计模式，用于将应用程序分为三个组件：模型（Model）、视图（View）和控制器（Controller）。

**解析：** 模型负责数据存储和业务逻辑；视图负责显示数据；控制器负责处理用户输入和协调模型和视图。

#### 二、算法编程题库

##### 5. 编写一个算法，找出数组中的重复元素。

**答案：**

```python
def find_duplicates(nums):
    seen = set()
    duplicates = []
    for num in nums:
        if num in seen:
            duplicates.append(num)
        else:
            seen.add(num)
    return duplicates

# 示例
nums = [1, 2, 3, 4, 5, 5, 6]
print(find_duplicates(nums))  # 输出 [5]
```

##### 6. 编写一个算法，实现快速排序。

**答案：**

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

# 示例
arr = [3, 6, 8, 10, 1, 2, 1]
print(quick_sort(arr))  # 输出 [1, 1, 2, 3, 6, 8, 10]
```

##### 7. 编写一个算法，实现链表反转。

**答案：**

```java
class ListNode {
    int val;
    ListNode next;
    ListNode(int x) { val = x; }
}

public ListNode reverseList(ListNode head) {
    ListNode prev = null;
    ListNode curr = head;
    while (curr != null) {
        ListNode nextTemp = curr.next;
        curr.next = prev;
        prev = curr;
        curr = nextTemp;
    }
    return prev;
}
```

**解析：** 这个示例中的 `reverseList` 函数通过迭代反转链表。
