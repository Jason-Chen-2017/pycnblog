#### 二、算法编程题库

**1. 给定一个字符串，输出字符串中所有重复的子串。**

**答案：** 

```python
def find_duplicates(s):
    duplicates = []
    n = len(s)
    for i in range(n):
        for j in range(i + 1, n):
            if s[i:j+1] in s[j+1:]:
                duplicates.append(s[i:j+1])
    return duplicates
```

**2. 给定一个整数数组，输出数组中所有重复的元素。**

**答案：**

```python
def find_duplicates(nums):
    duplicates = []
    visited = set()
    for num in nums:
        if num in visited:
            duplicates.append(num)
        else:
            visited.add(num)
    return duplicates
```

**3. 给定一个字符串，输出字符串中所有有效的子串。**

**答案：**

```python
def find_valid_substrings(s):
    valid_substrings = []
    n = len(s)
    for i in range(n):
        for j in range(i + 1, n + 1):
            valid_substrings.append(s[i:j])
    return valid_substrings
```

**4. 给定一个整数数组，输出数组中所有缺失的元素。**

**答案：**

```python
def find_missing_elements(nums):
    missing_elements = []
    visited = set()
    for num in nums:
        visited.add(num)
    for i in range(1, max(nums) + 1):
        if i not in visited:
            missing_elements.append(i)
    return missing_elements
```
