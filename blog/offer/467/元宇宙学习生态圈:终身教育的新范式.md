### 1. 实现一个简单的聊天室系统

**题目描述：** 编写一个简单的聊天室系统，支持用户发送消息、接收消息、查看历史消息等功能。

**答案示例（Python）：**

```python
import threading

class ChatRoom:
    def __init__(self):
        self.messages = []

    def send_message(self, user, message):
        self.messages.append((user, message))
        print(f"{user}: {message}")

    def receive_message(self, user):
        for u, m in self.messages:
            if u != user:
                print(f"{u}: {m}")

def user_thread(room, user):
    while True:
        message = input(f"{user}, input your message:")
        room.send_message(user, message)

def main():
    room = ChatRoom()
    users = ["Alice", "Bob", "Charlie"]

    for user in users:
        t = threading.Thread(target=user_thread, args=(room, user))
        t.start()

if __name__ == "__main__":
    main()
```

**解析：** 这个示例使用 Python 的多线程实现了一个简单的聊天室系统。每个用户输入的消息会被发送到聊天室，其他用户可以接收并显示这些消息。
