#### 1. 区块链数据结构实现

**题目：** 实现一个简单的区块链数据结构，并添加功能以添加新的区块和验证区块链的完整性。

**答案：**

以下是一个简单的区块链数据结构的实现，包括添加新的区块和验证区块链的完整性的功能：

```python
import hashlib
import json
from time import time

class Block:
    def __init__(self, index, transactions, timestamp, previous_hash):
        self.index = index
        self.transactions = transactions
        self.timestamp = timestamp
        self.previous_hash = previous_hash
        self.hash = self.compute_hash()

    def compute_hash(self):
        block_string = json.dumps(self.__dict__, sort_keys=True)
        return hashlib.sha256(block_string.encode()).hexdigest()

class Blockchain:
    def __init__(self):
        self.unconfirmed_transactions = []
        self.chain = []
        self.create_genesis_block()

    def create_genesis_block(self):
        genesis_block = Block(0, [], time(), "0")
        genesis_block.hash = genesis_block.compute_hash()
        self.chain.append(genesis_block)

    def add_new_transaction(self, transaction):
        self.unconfirmed_transactions.append(transaction)

    def mine(self):
        if not self.unconfirmed_transactions:
            return False

        last_block = self.chain[-1]
        new_block = Block(index=last_block.index + 1,
                          transactions=self.unconfirmed_transactions,
                          timestamp=time(),
                          previous_hash=last_block.hash)
        new_block.hash = new_block.compute_hash()
        self.chain.append(new_block)
        self.unconfirmed_transactions = []
        return new_block.index

    def is_chain_valid(self):
        for i in range(1, len(self.chain)):
            current = self.chain[i]
            previous = self.chain[i - 1]
            if current.hash != current.compute_hash():
                return False
            if current.previous_hash != previous.hash:
                return False
        return True

# 示例使用
blockchain = Blockchain()
blockchain.add_new_transaction("Transaction 1")
blockchain.add_new_transaction("Transaction 2")
blockchain.mine()
print(blockchain.is_chain_valid())
```

**解析：** 这个实现中，`Block` 类定义了区块链中的区块，包含索引、交易、时间戳、前一个哈希和当前哈希。`Blockchain` 类包含未确认的交易、区块链本身以及添加新交易、挖掘新区块和验证区块链完整性的功能。

#### 2. 智能合约实现

**题目：** 实现一个简单的智能合约，用于管理数字艺术品的所有权转移。

**答案：**

以下是一个简单的智能合约实现的示例，用于管理数字艺术品的所有权转移：

```python
from blockchain import Blockchain

class Artwork:
    def __init__(self, id, title, artist):
        self.id = id
        self.title = title
        self.artist = artist
        self.current_owner = artist
        self.history = []

    def transfer_ownership(self, new_owner):
        self.history.append({"from": self.current_owner, "to": new_owner})
        self.current_owner = new_owner

class SmartContract:
    def __init__(self, blockchain):
        self.blockchain = blockchain

    def execute_transfer(self, artwork_id, new_owner):
        artwork = self.find_artwork(artwork_id)
        if artwork:
            artwork.transfer_ownership(new_owner)
            transaction = {
                "artwork_id": artwork_id,
                "new_owner": new_owner
            }
            self.blockchain.add_new_transaction(transaction)
            self.blockchain.mine()
            return True
        return False

    def find_artwork(self, artwork_id):
        for block in self.blockchain.chain:
            for transaction in block.transactions:
                if transaction["artwork_id"] == artwork_id:
                    return Artwork(transaction["artwork_id"],
                                   transaction["title"],
                                   transaction["artist"])
        return None

# 示例使用
blockchain = Blockchain()
artwork = Artwork("1", "数字画作", "艺术家A")
smart_contract = SmartContract(blockchain)
smart_contract.execute_transfer("1", "新所有者B")
```

**解析：** 这个实现中，`Artwork` 类定义了数字艺术品的基本属性和所有权转移的方法。`SmartContract` 类包含了执行所有权转移和查找艺术品的功能。该合约将交易添加到区块链中，并在成功转移所有权后挖掘新区块。
