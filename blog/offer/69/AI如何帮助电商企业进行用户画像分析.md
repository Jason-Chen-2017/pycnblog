### 14. 算法编程题：使用K-means算法进行用户画像聚类

**题目：** 使用K-means算法进行用户画像聚类，编写相应的Python代码。

**答案：** 下面是一个使用K-means算法进行用户画像聚类的Python代码示例。

```python
import numpy as np
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

# 假设我们有一个2D的用户画像数据集
data = np.array([[1, 2], [1, 4], [1, 0],
                  [10, 2], [10, 4], [10, 0]])

# 使用KMeans类创建模型，设置聚类数量为2
kmeans = KMeans(n_clusters=2, random_state=0).fit(data)

# 输出聚类中心
print("聚类中心：", kmeans.cluster_centers_)

# 输出每个样本的聚类结果
print("每个样本的聚类结果：", kmeans.labels_)

# 绘制聚类结果
plt.scatter(data[:, 0], data[:, 1], c=kmeans.labels_, s=100, cmap='viridis')
plt.scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1], s=300, c='red', label='Centroids')
plt.title("K-means Clustering")
plt.xlabel("Feature 1")
plt.ylabel("Feature 2")
plt.legend()
plt.show()
```

**解析：**

在这个示例中，我们首先导入必要的库，然后创建一个简单的2D用户画像数据集。使用`KMeans`类创建模型，并设置聚类数量为2。通过`fit`方法训练模型，并输出聚类中心和每个样本的聚类结果。最后，使用`matplotlib`库绘制聚类结果，以可视化聚类效果。

1. **用户-项目评分矩阵构建**：收集用户对项目的评分数据，构建用户-项目评分矩阵。

**解析：**

协同过滤算法可以分为基于用户和基于项目的协同过滤。基于用户的协同过滤通过计算用户之间的相似度，推荐与目标用户相似的其他用户喜欢的项目；基于项目的协同过滤通过计算项目之间的相似度，推荐与目标项目相似的其他项目。在实际应用中，可以根据业务需求和数据特点选择合适的协同过滤算法。

### 16. 算法编程题：实现基于用户的协同过滤推荐系统

**题目：** 编写一个简单的基于用户的协同过滤推荐系统，使用余弦相似度计算用户之间的相似度，并生成推荐列表。

**答案：** 下面是一个简单的基于用户的协同过滤推荐系统的Python代码示例。

```python
import numpy as np

def cosine_similarity(user_ratings):
    # 计算用户之间的余弦相似度矩阵
    similarity_matrix = np.dot(user_ratings, user_ratings.T) / (np.linalg.norm(user_ratings, axis=1) * np.linalg.norm(user_ratings, axis=0))
    return similarity_matrix

def collaborative_filtering(user_ratings, similarity_matrix, item_ratings, k=5):
    # 计算相似用户对项目的评分预测
    predicted_ratings = np.dot(similarity_matrix, item_ratings) / np.sum(np.abs(similarity_matrix), axis=1)
    
    # 排序并取前k个推荐项目
    recommended_items = np.argsort(predicted_ratings)[:-k-1:-1]
    return recommended_items

# 假设我们有一个用户-项目评分矩阵
user_ratings = np.array([[5, 3, 0, 1],
                         [4, 0, 0, 1],
                         [1, 1, 0, 5],
                         [1, 0, 4, 3]])

# 构建相似度矩阵
similarity_matrix = cosine_similarity(user_ratings)

# 假设我们有一个项目-评分矩阵
item_ratings = np.array([[1, 2, 3, 4],
                         [5, 4, 3, 2],
                         [0, 1, 2, 0],
                         [4, 3, 2, 1]])

# 生成推荐列表
recommended_items = collaborative_filtering(user_ratings, similarity_matrix, item_ratings, k=2)
print("推荐项目索引：", recommended_items)
```
