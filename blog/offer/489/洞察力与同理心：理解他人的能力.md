#### **1. 相同点的子集数量**

**题目描述：** 给定一个整数数组 `nums`，返回相同点和的子集数量。相同点是指两个数在数组中的下标之和相同。

**示例：**  
输入：`nums = [1, 2, 3, 4]`  
输出：`2`  
解释：子集 `[1, 3]` 和 `[3, 1]` 的相同点都是 `4`。

**答案解析：**  
解决这个问题，我们可以使用哈希表来记录每个点的下标之和及其出现的次数。然后遍历数组，对于每个数，我们计算其与其它数的下标之和，并查找哈希表中是否存在相同的和。如果存在，则将次数加1。

**代码示例：**

```python
def count_subsets_with_same_sum(nums):
    counts = defaultdict(int)
    for i, x in enumerate(nums):
        counts[x + i] += 1

    return sum(counts[v] * (counts[v] - 1) // 2 for v in counts.values())

# 测试
nums = [1, 2, 3, 4]
print(count_subsets_with_same_sum(nums))  # 输出: 2
```

#### **2. 最长公共子序列**

**题目描述：** 给定两个字符串 `str1` 和 `str2`，返回它们的最长公共子序列的长度。

**示例：**  
输入：`str1 = "ABCD"，str2 = "ACDF"`  
输出：`2`  
解释：最长公共子序列是 `"AC"`。

**答案解析：**  
我们可以使用动态规划来解决这个问题。定义一个二维数组 `dp`，其中 `dp[i][j]` 表示 `str1` 的前 `i` 个字符和 `str2` 的前 `j` 个字符的最长公共子序列长度。

**代码示例：**

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

# 测试
str1 = "ABCD"
str2 = "ACDF"
print(longest_common_subsequence(str1, str2))  # 输出: 2
```

#### **3. 有效的括号序列**

**题目描述：** 给定一个字符串 `s`，判断它是否是一个有效的括号序列。

**示例：**  
输入：`s = "()"`  
输出：`True`  
输入：`s = "()[]"`  
输出：`True`  
输入：`s = "(]"`  
输出：`False`

**答案解析：**  
我们可以使用一个栈来解决这个问题。遍历字符串，对于每个字符：

- 如果是左括号，将其入栈。
- 如果是右括号，检查栈顶元素是否为与之匹配的左括号，如果是，则弹出栈顶元素，否则返回 `False`。
- 遍历结束后，如果栈为空，则返回 `True`，否则返回 `False`。

**代码示例：**

```python
def is_valid_parentheses(s):
    stack = []
    parentheses_map = {')': '(', ']': '[', '}': '{'}

    for char in s:
        if char in parentheses_map.values():
            stack.append(char)
        elif char in parentheses_map and stack and stack[-1] == parentheses_map[char]:
            stack.pop()
        else:
            return False

    return not stack

# 测试
s = "()[]"
print(is_valid_parentheses(s))  # 输出: True
s = "(]"
print(is_valid_parentheses(s))  # 输出: False
```

#### **4. 合并区间**

**题目描述：** 给定一个区间列表，合并所有重叠的区间。

**示例：**  
输入：`intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]`  
输出：`[[1, 6], [8, 10], [15, 18]]`  
输入：`intervals = [[1, 4], [4, 5]]`  
输出：`[[1, 5]]`

**答案解析：**  
首先，我们将区间按照起点排序。然后遍历排序后的区间列表，对于当前区间，如果它与下一个区间重叠，则合并它们。否则，保持当前区间不变，继续处理下一个区间。

**代码示例：**

```python
def merge_intervals(intervals):
    intervals.sort(key=lambda x: x[0])
    merged = []

    for interval in intervals:
        if not merged or merged[-1][1] < interval[0]:
            merged.append(interval)
        else:
            merged[-1][1] = max(merged[-1][1], interval[1])

    return merged

# 测试
intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]
print(merge_intervals(intervals))  # 输出: [[1, 6], [8, 10], [15, 18]]
intervals = [[1, 4], [4, 5]]
print(merge_intervals(intervals))  # 输出: [[1, 5]]
```
