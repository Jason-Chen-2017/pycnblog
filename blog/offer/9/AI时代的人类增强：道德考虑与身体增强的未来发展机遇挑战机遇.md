#### 二、算法编程题库及解析

##### 1. 寻找最长公共子序列

**题目：** 给定两个字符串，找出它们的最长公共子序列。

**答案：** 采用动态规划算法，使用二维数组记录子问题的最优解。

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if str1[i - 1] == str2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]

# 示例
str1 = "ABCD"
str2 = "ACDF"
print(longest_common_subsequence(str1, str2))  # 输出 2
```

**解析：** 该算法的时间复杂度为 O(mn)，其中 m 和 n 分别为两个字符串的长度。

##### 2. 最小生成树算法

**题目：** 使用Prim算法求解给定无向图的最小生成树。

**答案：** Prim算法是一种贪心算法，从任意一个顶点开始，逐步增加新顶点和边，直到所有顶点都被包含在生成树中。

```python
import heapq

def prim_algorithm(edges, n):
    # edges: (权重，顶点)
    # n: 顶点数量
    visited = [False] * n
    mst = []
    total_weight = 0

    heapq.heapify(edges)
    while edges:
        weight, v = heapq.heappop(edges)
        if not visited[v]:
            total_weight += weight
            mst.append((v, weight))
            visited[v] = True
            for neighbor, w in edges:
                if not visited[neighbor]:
                    heapq.heappush(edges, (w, neighbor))

    return mst, total_weight

# 示例
edges = [(2, 0), (2, 1), (3, 2), (4, 3), (5, 4)]
n = 5
print(prim_algorithm(edges, n))
```

**解析：** 该算法的时间复杂度为 O(ElogE)，其中 E 为边的数量。
