#### 6.2 算法编程题

**6.2.1 题目：** 如何使用 Golang 实现一个简单的并发请求处理器，支持并发请求的处理和响应。

**答案：** 使用 Golang 的 goroutine 和 channel 实现一个简单的并发请求处理器，代码示例如下：

```go
package main

import (
    "fmt"
    "time"
)

func processRequest(reqID int, reqChan chan<- int) {
    // 处理请求的逻辑
    time.Sleep(2 * time.Second)
    fmt.Printf("Processed request %d\n", reqID)

    // 发送处理结果到 channel
    reqChan <- reqID
}

func main() {
    // 创建一个容量为 5 的 channel
    reqChan := make(chan int, 5)

    // 创建多个 goroutine 处理请求
    for i := 0; i < 10; i++ {
        go processRequest(i, reqChan)
    }

    // 从 channel 接收处理结果
    for i := 0; i < 10; i++ {
        reqID := <-reqChan
        fmt.Printf("Received processed request %d\n", reqID)
    }
}
```

**解析：** 该程序创建了一个并发请求处理器，其中 `processRequest` 函数模拟请求处理过程，`main` 函数创建多个 goroutine 发送请求，并从 channel 接收处理结果。

**6.2.2 题目：** 如何在 Golang 中实现一个简单的分布式锁？

**答案：** 在 Golang 中，可以使用 channel 实现一个简单的分布式锁，代码示例如下：

```go
package main

import (
    "fmt"
    "sync"
    "time"
)

func distributedLock(lockChan chan bool) {
    // 获取锁
    lockChan <- true

    // 处理逻辑
    time.Sleep(2 * time.Second)
    fmt.Println("Lock acquired and processing")

    // 释放锁
    <-lockChan
}

func main() {
    // 创建一个 channel 作为锁
    lockChan := make(chan bool)

    // 创建多个 goroutine 获取锁
    for i := 0; i < 5; i++ {
        go distributedLock(lockChan)
    }

    // 等待所有 goroutine 结束
    time.Sleep(10 * time.Second)
}
```

**解析：** 该程序使用一个 channel `lockChan` 作为锁，每个 goroutine 在获取锁时发送 `true`，在释放锁时接收 `true`。这样可以确保同一时间只有一个 goroutine 获取到锁。
