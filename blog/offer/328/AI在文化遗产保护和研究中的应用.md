### 2. 利用Python实现基于K-means算法的图像聚类

**题目：** 请用Python实现基于K-means算法的图像聚类，并简要介绍算法过程。

**答案：** 

```python
from sklearn.cluster import KMeans
import numpy as np
import cv2

# 读取图像
image = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)

# 将图像转换为向量
image_vector = image.reshape(-1, 1)

# 初始化K-means算法
kmeans = KMeans(n_clusters=5)

# 训练K-means算法
kmeans.fit(image_vector)

# 获取聚类结果
labels = kmeans.predict(image_vector)

# 为每个像素点分配颜色
output_image = np.zeros_like(image)
output_image[labels == 0] = 255

# 显示结果
cv2.imshow('K-means Clustering', output_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

**解析：** 该算法首先读取图像，将其转换为向量。然后初始化K-means算法，并使用图像向量训练算法。最后，利用聚类结果为每个像素点分配颜色，实现图像聚类。
