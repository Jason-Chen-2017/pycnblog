### 面试题库

#### 1. 集合的基本操作

**题目：** 请描述集合的并集、交集和补集操作，并给出相应的代码实现。

**答案：**

并集操作：将两个集合中的所有元素合并，去除重复元素。

```python
def union(set1, set2):
    return set1.union(set2)

# 示例
set1 = {1, 2, 3}
set2 = {3, 4, 5}
print(union(set1, set2))  # 输出 {1, 2, 3, 4, 5}
```

交集操作：返回两个集合共有的元素。

```python
def intersection(set1, set2):
    return set1.intersection(set2)

# 示例
print(intersection(set1, set2))  # 输出 {3}
```

补集操作：返回属于第一个集合但不属于第二个集合的元素。

```python
def difference(set1, set2):
    return set1.difference(set2)

# 示例
print(difference(set1, set2))  # 输出 {1, 2}
```

#### 2. 集合的等价关系和划分

**题目：** 请解释等价关系和划分的概念，并给出一个示例。

**答案：**

等价关系：一个集合上的关系，如果满足自反性、对称性和传递性，则称为等价关系。

示例：在整数集合上定义一个等价关系，使得两个整数等价当且仅当它们的差是 3 的倍数。

```python
def is_equivalent(a, b):
    return (a - b) % 3 == 0

# 示例
print(is_equivalent(3, 6))  # 输出 True
```

划分：将集合划分为若干不相交的子集，使得每个子集内的元素之间是等价的。

示例：根据等价关系，将整数集合划分为三个子集。

```python
def partition(set1):
    partition = [[] for _ in range(3)]
    for x in set1:
        partition[x % 3].append(x)
    return partition

# 示例
print(partition({1, 2, 3, 4, 5, 6}))  # 输出 [[1, 4], [2, 5], [3, 6]]
```

#### 3. 集合的基数和势

**题目：** 请解释集合的基数和势的概念，并给出一个示例。

**答案：**

基数：集合中元素的数量。

示例：计算集合 {1, 2, 3, 4, 5} 的基数。

```python
def cardinality(set1):
    return len(set1)

# 示例
print(cardinality({1, 2, 3, 4, 5}))  # 输出 5
```

势：集合的所有子集的个数。

示例：计算集合 {1, 2, 3} 的势。

```python
def power_set(set1):
    n = len(set1)
    power_set_size = 2**n
    return [[] for _ in range(power_set_size)]

# 示例
print(len(power_set({1, 2, 3})))  # 输出 8
```

### 算法编程题库

#### 1. 单调栈

**题目：** 实现一个单调栈，用于解决数组中的元素的最大值问题。

**答案：**

```python
class MonotonicStack:
    def __init__(self):
        self.stack = []

    def push(self, x):
        while self.stack and x >= self.stack[-1]:
            self.stack.pop()
        self.stack.append(x)

    def pop(self):
        return self.stack.pop()

    def top(self):
        return self.stack[-1]

    def isEmpty(self):
        return not self.stack

# 示例
stack = MonotonicStack()
stack.push(1)
stack.push(2)
stack.push(3)
stack.push(2)
stack.push(1)
print(stack.top())  # 输出 3
```

#### 2. 等价类划分

**题目：** 给定一个包含数字的数组，将数组划分为若干等价类，使得每个类中的元素之和相等。

**答案：**

```python
def partition_equal_set(nums):
    total_sum = sum(nums)
    if total_sum % 2 != 0:
        return []
    target = total_sum // 2
    dp = [[False] * (target + 1) for _ in range(len(nums) + 1)]
    dp[0][0] = True
    for i in range(1, len(nums) + 1):
        for j in range(target + 1):
            if j == 0:
                dp[i][j] = True
                continue
            dp[i][j] = dp[i - 1][j]
            if nums[i - 1] <= j:
                dp[i][j] = dp[i][j] or dp[i - 1][j - nums[i - 1]]
    if not dp[len(nums)][target]:
        return []
    result = []
    for i in range(len(nums), 0, -1):
        if dp[i][target]:
            result.append(nums[i - 1])
            target -= nums[i - 1]
    return result

# 示例
nums = [1, 5, 11, 5]
print(partition_equal_set(nums))  # 输出 [1, 5, 5]
```

#### 3. 背包问题

**题目：** 给定一个容量为 `W` 的背包和一组物品，每个物品有一个重量和价值，求解如何选择物品使得背包的总价值最大。

**答案：**

```python
def knapsack(W, weights, values, n):
    dp = [[0] * (W + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(1, W + 1):
            if weights[i - 1] <= w:
                dp[i][w] = max(dp[i - 1][w], dp[i - 1][w - weights[i - 1]] + values[i - 1])
            else:
                dp[i][w] = dp[i - 1][w]
    return dp[n][W]

# 示例
W = 50
weights = [10, 20, 30]
values = [60, 100, 120]
n = len(values)
print(knapsack(W, weights, values, n))  # 输出 220
```
