def classify_workers(workers):
    # 假设技能水平越高，得分越高
    high_end = []
    low_end = []

    for worker in workers:
        if worker['skill_score'] > 80:
            high_end.append(worker)
        else:
            low_end.append(worker)

    return high_end, low_end

workers = [
    {'name': '张三', 'skill_score': 90, 'employment_status': '稳定'},
    {'name': '李四', 'skill_score': 70, 'employment_status': '不稳定'},
    {'name': '王五', 'skill_score': 85, 'employment_status': '稳定'},
    {'name': '赵六', 'skill_score': 60, 'employment_status': '不稳定'}
]

high_end, low_end = classify_workers(workers)

print("高端群体：", high_end)
print("低端群体：", low_end)

import matplotlib.pyplot as plt

def analyze_employment_trend(workers):

    # 绘制就业变化趋势图

    plt.show()

**题目：** 请解释高端与低端纺织工人在劳动力市场中的变化，并分析其背后的原因。

1. **自动化和智能化技术的应用**：随着自动化生产设备和智能化系统的普及，低端纺织工人的需求逐渐减少。这些技术能够替代重复性、低技能的劳动，使得低端纺织工人面临就业压力。

2. **技能要求的提高**：高端纺织工人需要具备较高的技能水平，包括机械操作、设备维护、质量控制等。随着行业的升级和消费者对产品质量的要求提高，高端纺织工人的需求增加。

3. **教育背景的重要性**：教育水平的提高成为高端纺织工人就业的重要门槛。许多纺织企业开始偏好具备高等教育背景的求职者，以适应现代化生产的需求。

**原因分析：**

- **技术进步**：自动化和智能化的技术进步显著提高了纺织生产效率，减少了低端纺织工人的就业机会。同时，高端纺织工人需要具备更高的技术水平，以操作和维护先进的设备。

- **全球竞争**：全球化使得纺织行业竞争加剧，企业为了降低成本和提高竞争力，更倾向于采用自动化和智能化技术，从而减少对低端纺织工人的需求。

- **消费者需求**：随着消费者对产品质量和环保要求的提高，高端纺织产品的需求增加，从而带动了高端纺织工人的就业增长。
