#### 二、算法编程题库

**1. 请实现一个基于 K-均值算法的聚类函数。**

**算法描述：** K-均值算法是一种基于距离度量的聚类算法，其目标是把数据分为 K 个簇，使得每个簇的内部距离之和最小。

**输入参数：** 数据集 X（二维数组，每行代表一个数据点），簇数 K。

**输出参数：** 簇中心坐标数组 C（二维数组，每行代表一个簇中心坐标）。

**代码示例：**

```python
import numpy as np

def kmeans(X, K):
    # 初始化簇中心
    C = X[np.random.choice(X.shape[0], K, replace=False)]
    
    while True:
        # 计算每个数据点与簇中心的距离
        distances = np.linalg.norm(X - C, axis=1)
        
        # 分配数据点到最近的簇中心
        labels = np.argmin(distances, axis=1)
        
        # 重新计算簇中心
        new_C = np.array([X[labels == k].mean(axis=0) for k in range(K)])
        
        # 判断收敛条件
        if np.linalg.norm(new_C - C) < 1e-6:
            break

        C = new_C
    
    return C

# 示例数据
X = np.array([[1, 2], [1, 4], [1, 0],
              [10, 2], [10, 4], [10, 0]])

# 聚类结果
C = kmeans(X, 2)
print(C)
```

**2. 请实现一个基于支持向量机（SVM）的分类函数。**

**算法描述：** 支持向量机是一种监督学习算法，主要用于二分类问题。它的目标是找到一个最优的超平面，将不同类别的数据点分开。

**输入参数：** 数据集 X（二维数组，每行代表一个数据点），标签 Y（一维数组，每个元素代表数据点的类别）。

**输出参数：** 分类结果 predictions（一维数组，每个元素代表数据点的预测类别）。

**代码示例：**

```python
import numpy as np
from sklearn.svm import SVC

def svm_classifier(X, Y):
    # 初始化 SVM 模型
    model = SVC(kernel='linear')
    
    # 训练模型
    model.fit(X, Y)
    
    # 预测分类结果
    predictions = model.predict(X)
    
    return predictions

# 示例数据
X = np.array([[1, 2], [1, 4], [1, 0],
              [10, 2], [10, 4], [10, 0]])
Y = np.array([0, 0, 0, 1, 1, 1])

# 分类结果
predictions = svm_classifier(X, Y)
print(predictions)
```

**3. 请实现一个基于决策树（Decision Tree）的回归函数。**

**算法描述：** 决策树是一种树形结构，每个节点代表一个特征，每个分支代表特征取值的不同情况，叶节点代表预测结果。

**输入参数：** 数据集 X（二维数组，每行代表一个数据点），标签 Y（一维数组，每个元素代表数据点的真实值）。

**输出参数：** 回归结果 predictions（一维数组，每个元素代表数据点的预测值）。

**代码示例：**

```python
import numpy as np
from sklearn.tree import DecisionTreeRegressor

def decision_tree_regressor(X, Y):
    # 初始化决策树模型
    model = DecisionTreeRegressor()
    
    # 训练模型
    model.fit(X, Y)
    
    # 预测回归结果
    predictions = model.predict(X)
    
    return predictions

# 示例数据
X = np.array([[1, 2], [1, 4], [1, 0],
              [10, 2], [10, 4], [10, 0]])
Y = np.array([1, 2, 0, 11, 12, 10])

# 回归结果
predictions = decision_tree_regressor(X, Y)
print(predictions)
```
