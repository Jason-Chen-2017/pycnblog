##### 三、算法编程题库及解析

1. **最大子序列和**
   - **题目：** 给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个数），返回其最大和。
   - **答案：** Python 示例代码：
     ```python
     def maxSubArray(nums):
         max_so_far = float("-inf")
         curr_max = 0
         for num in nums:
             curr_max = max(num, curr_max + num)
             max_so_far = max(max_so_far, curr_max)
         return max_so_far
     ```
     - **解析：** 该算法使用贪心策略，从左到右遍历数组，计算当前子数组的和，如果当前子数组的和小于0，则重置为0。遍历结束后，返回最大子数组的和。

2. **二分查找**
   - **题目：** 实现一个二分查找算法，在有序数组中查找目标值。
   - **答案：** Java 示例代码：
     ```java
     public int search(int[] nums, int target) {
         int left = 0, right = nums.length - 1;
         while (left <= right) {
             int mid = left + (right - left) / 2;
             if (nums[mid] == target) {
                 return mid;
             } else if (nums[mid] < target) {
                 left = mid + 1;
             } else {
                 right = mid - 1;
             }
         }
         return -1;
     }
     ```
     - **解析：** 该算法通过不断缩小区间，实现二分查找。如果中间值小于目标值，则在右半边继续查找；如果中间值大于目标值，则在左半边继续查找；如果中间值等于目标值，则返回索引。
