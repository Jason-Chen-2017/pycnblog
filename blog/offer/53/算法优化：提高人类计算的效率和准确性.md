### 1. 如何在 O(1) 时间内删除链表中的节点？

**题目：** 给定一个单链表的头节点 head，请实现一个函数，在 O(1) 时间内删除链表中的节点。

**答案：** 

```python
def deleteNode(head, node):
    node.val = head.val
    node.next = head.next
    head = head.next
    return head
```

**解析：** 利用“巧妙的覆盖法”，将待删除节点的下一个节点的值赋给待删除节点，然后将待删除节点的指针指向下一个节点，即可删除待删除节点。

### 2. 如何找到数组中的第 k 个最大元素？

**题目：** 给定一个整数数组 nums 和一个整数 k，请找到数组中的第 k 个最大元素。

**答案：**

```python
def findKthLargest(nums, k):
    n = len(nums)
    low, high = 0, n - 1
    while low < high:
        pivot = partition(nums, low, high)
        if pivot == k - 1:
            return nums[pivot]
        elif pivot < k - 1:
            low = pivot + 1
        else:
            high = pivot - 1
    return nums[low]
```

**解析：** 使用快速选择算法，通过不断调整分区点，找到第 k 个最大元素。

### 3. 如何求解二叉树的直径？

**题目：** 给定一棵二叉树，请找到二叉树中任意两个节点之间的最大距离。

**答案：**

```java
class Solution {
    private int max = 0;

    public int diameterOfBinaryTree(TreeNode root) {
        dfs(root);
        return max;
    }

    private int dfs(TreeNode node) {
        if (node == null) {
            return 0;
        }
        int left = dfs(node.left);
        int right = dfs(node.right);
        max = Math.max(max, left + right);
        return 1 + Math.max(left, right);
    }
}
```

**解析：** 利用后序遍历，计算每个节点的左右子树高度，并更新直径的最大值。

### 1. 求解最大子序和

**题目：** 给定一个整数数组 nums，求解该数组中的最大子序和。

**答案：**

```python
def maxSubArray(nums):
    if not nums:
        return 0
    max_sum = nums[0]
    curr_sum = nums[0]
    for i in range(1, len(nums)):
        curr_sum = max(nums[i], curr_sum + nums[i])
        max_sum = max(max_sum, curr_sum)
    return max_sum
```

**解析：** 利用动态规划，维护当前子序列和的最大值，更新最大子序和。

### 2. 求解旋转数组中的最小值

**题目：** 给定一个旋转排序的数组，求解数组中的最小值。

**答案：**

```java
public int findMin(int[] nums) {
    int low = 0, high = nums.length - 1;
    while (low < high) {
        int mid = low + (high - low) / 2;
        if (nums[mid] > nums[high]) {
            low = mid + 1;
        } else {
            high = mid;
        }
    }
    return nums[low];
}
```

**解析：** 利用二分查找，找到旋转点的位置，返回旋转后的最小值。

### 3. 求解有效的括号序列

**题目：** 给定一个字符串，判断其是否为有效的括号序列。

**答案：**

```python
def isValid(s):
    stack = []
    for c in s:
        if c == '(' or c == '[' or c == '{':
            stack.append(c)
        elif c == ')' and len(stack) > 0 and stack[-1] == '(':
            stack.pop()
        elif c == ']' and len(stack) > 0 and stack[-1] == '[':
            stack.pop()
        elif c == '}' and len(stack) > 0 and stack[-1] == '{':
            stack.pop()
        else:
            return False
    return len(stack) == 0
```

**解析：** 利用栈实现，遍历字符串，遇到左括号入栈，遇到右括号出栈，最后判断栈是否为空。
