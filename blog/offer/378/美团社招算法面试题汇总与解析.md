### 2. 数据结构与算法

数据结构与算法是计算机科学的基础，也是算法面试的重点。以下是一些常见的面试题目及解析。

#### 2.1. 题目：数组与链表

**题目描述：** 给定一个整数数组，实现一个函数，将数组中的元素旋转 k 个位置。

**解析：** 可以使用原地旋转方法，分为两步进行：

1. **循环右移 k 次：** 将数组最右边的 k 个元素移动到最左边。
2. **循环左移 n-k 次：** 将剩余的 n-k 个元素移动到原来 k 个元素的位置。

**示例代码：**

```python
def rotateArray(nums, k):
    n = len(nums)
    k %= n  # 处理 k > n 的情况
    reverse(nums, 0, n - 1)
    reverse(nums, 0, k - 1)
    reverse(nums, k, n - 1)

def reverse(nums, start, end):
    while start < end:
        nums[start], nums[end] = nums[end], nums[start]
        start += 1
        end -= 1
```

#### 2.2. 题目：二分查找

**题目描述：** 给定一个排序后的整数数组，实现一个函数，查找目标值并返回其索引。

**解析：** 使用二分查找算法，分为以下步骤：

1. **初始化：** 设置两个指针 `low` 和 `high`，分别指向数组的第一个和最后一个元素。
2. **循环：** 当 `low <= high` 时，计算中间索引 `mid`。
3. **比较：** 比较目标值与中间元素的值：
   - 如果目标值等于中间元素，返回 `mid`。
   - 如果目标值小于中间元素，将 `high` 更新为 `mid - 1`。
   - 如果目标值大于中间元素，将 `low` 更新为 `mid + 1`。

**示例代码：**

```python
def binarySearch(nums, target):
    low, high = 0, len(nums) - 1
    while low <= high:
        mid = (low + high) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

### 3. 数学与逻辑

数学与逻辑题在算法面试中占据重要地位，以下是一些典型题目及解析。

#### 3.1. 题目：最大子序和

**题目描述：** 给定一个整数数组，找出连续子数组的最大和。

**解析：** 可以使用动态规划或贪心算法：

1. **动态规划：** 维护一个数组 `dp`，其中 `dp[i]` 表示以 `i` 为结尾的子数组的最大和。
2. **贪心算法：** 维护一个变量 `max_sum`，每次遍历更新 `max_sum` 为当前元素和 `max_sum` 的最大值。

**示例代码：**

```python
def maxSubArray(nums):
    max_sum = nums[0]
    for num in nums[1:]:
        max_sum = max(max_sum + num, num)
    return max_sum
```

#### 3.2. 题目：排列组合

**题目描述：** 给定两个整数 `n` 和 `k`，求组合数 `C(n, k)`。

**解析：** 使用阶乘计算组合数：

1. 计算 `n! / (k! * (n-k)!)`。
2. 使用递归或循环计算阶乘。

**示例代码：**

```python
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

def combination(n, k):
    return factorial(n) // (factorial(k) * factorial(n-k))
```

### 4. 编程实践

编程实践题主要考察面试者的编程能力和解决问题的能力，以下是一些典型题目及解析。

#### 4.1. 题目：字符串匹配

**题目描述：** 给定一个文本字符串和一个模式字符串，实现一个函数，找出模式在文本中出现的所有位置。

**解析：** 可以使用 KMP 算法：

1. **构建部分匹配表：** 根据模式字符串构建部分匹配表。
2. **匹配：** 使用部分匹配表进行匹配，跳过已经匹配的部分。

**示例代码：**

```python
def KMP_match(s, p):
    n, m = len(s), len(p)
    lps = [0] * m
    build_lps(p, m, lps)
    i, j = 0, 0
    while i < n:
        if p[j] == s[i]:
            i += 1
            j += 1
        if j == m:
            return i - j
        elif i < n and p[j] != s[i]:
            if j != 0:
                j = lps[j - 1]
            else:
                i += 1
    return -1

def build_lps(p, m, lps):
    length = 0
    lps[0] = 0
    i = 1
    while i < m:
        if p[i] == p[length]:
            length += 1
            lps[i] = length
            i += 1
        else:
            if length != 0:
                length = lps[length - 1]
            else:
                lps[i] = 0
                i += 1
```

#### 4.2. 题目：排序算法

**题目描述：** 给定一个整数数组，实现几种常见的排序算法，如冒泡排序、快速排序、归并排序等。

**解析：** 各类排序算法的基本思想如下：

1. **冒泡排序：** 依次比较相邻元素的大小，若逆序则交换，重复直到整个数组有序。
2. **快速排序：** 选择一个基准元素，将数组分为两部分，然后递归对两部分进行排序。
3. **归并排序：** 将数组分为两部分，分别递归排序，然后合并两个有序数组。

**示例代码：**

```python
def bubble_sort(nums):
    n = len(nums)
    for i in range(n):
        for j in range(0, n-i-1):
            if nums[j] > nums[j+1]:
                nums[j], nums[j+1] = nums[j+1], nums[j]

def quick_sort(nums, low, high):
    if low < high:
        pi = partition(nums, low, high)
        quick_sort(nums, low, pi-1)
        quick_sort(nums, pi+1, high)

def partition(nums, low, high):
    pivot = nums[high]
    i = low - 1
    for j in range(low, high):
        if nums[j] < pivot:
            i += 1
            nums[i], nums[j] = nums[j], nums[i]
    nums[i+1], nums[high] = nums[high], nums[i+1]
    return i + 1

def merge_sort(nums):
    if len(nums) > 1:
        mid = len(nums) // 2
        left = nums[:mid]
        right = nums[mid:]
        merge_sort(left)
        merge_sort(right)
        i = j = k = 0
        while i < len(left) and j < len(right):
            if left[i] < right[j]:
                nums[k] = left[i]
                i += 1
            else:
                nums[k] = right[j]
                j += 1
            k += 1
        while i < len(left):
            nums[k] = left[i]
            i += 1
            k += 1
        while j < len(right):
            nums[k] = right[j]
            j += 1
            k += 1
```
