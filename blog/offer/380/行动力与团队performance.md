##### 1. 面包队列问题（LeetCode 1146）

**题目：** 设计一个算法，监控队列中的面包制作过程。面包店源源不断地添加新鲜面包到队列中，并按照先进先出的原则进行出售。如果队列中出现新的面包，那么该面包后面就不会再添加面包。实现以下操作：

- `void insert(int bread):` 在队列末尾插入一个新的面包。
- `int remove():` 删除队列最前面的面包并返回它的种类。
- `int count():` 返回队列中的面包数量。

**答案：** 使用双端队列实现，将队列模拟成先进先出，同时使用一个变量记录队列的长度。

```python
from collections import deque

class BreadthFirstSearch:
    def __init__(self):
        self.queue = deque()
        self.bread_count = 0

    def insert(self, bread: int) -> None:
        if not self.queue or self.queue[-1] != bread:
            self.queue.append(bread)
            self.bread_count += 1

    def remove(self) -> int:
        if self.queue:
            self.bread_count -= 1
            return self.queue.popleft()
        return -1

    def count(self) -> int:
        return self.bread_count
```

##### 2. 排队进站问题（LeetCode 1114）

**题目：** 在地铁系统中，乘客按照他们到达站点的顺序排队等待上车。地铁可以同时容纳多个乘客，但一次只能让一个乘客下车。设计一个算法，使乘客进站和下车的操作能够高效进行。

**答案：** 使用两个队列，一个用于等待进站的乘客，另一个用于等待下车的乘客。

```python
class地铁站：
    def __init__(self):
        self.waiting_in = deque()
        self.waiting_out = deque()

    def board(self, t: int) -> int:
        self.waiting_in.append(t)
        return 0

    def dispatch(self) -> int:
        if not self.waiting_out:
            if not self.waiting_in:
                return -1
            self.waiting_out.append(self.waiting_in.popleft())
        return self.waiting_out.pop()
```

##### 3. 爬楼梯问题（LeetCode 70）

**题目：** 一个楼梯有n阶台阶，每次可以爬1阶或2阶。问有多少种不同的方法可以爬到楼顶？

**答案：** 使用动态规划，保存每一步的方法数。

```python
def climbStairs(n: int) -> int:
    if n < 2:
        return n
    a, b = 0, 1
    for i in range(2, n + 1):
        c = a + b
        a = b
        b = c
    return b
```
