### 1. 序数集合的表示方法

**题目：** 序数集合可以用哪些表示方法？请举例说明。

**答案：** 序数集合可以使用自然数表示法、序数记数法、集合表示法等。

**解析：**
- **自然数表示法**：直接使用自然数表示序数，例如，{ω, ω+1, ω*2, ...}。
- **序数记数法**：使用序数记数法表示序数，例如，ω表示无穷大序数，ω+1表示比ω大1的序数。
- **集合表示法**：将序数集合视为一个集合，例如，{{ω}, {ω, ω+1}, {ω, ω+1, ω*2}, ...}。

**示例：**
```markdown
ω = {0, 1, 2, 3, ...}  # 无穷大序数
ω+1 = {ω, ω+1, ω*2, ...}  # 比ω大1的序数
```

### 2. 序数函数的性质

**题目：** 序数函数有哪些常见的性质？请举例说明。

**答案：** 序数函数具有以下常见性质：

- **单射**：对于任意两个不同的序数a和b，如果序数函数f是单射，则f(a) ≠ f(b)。
- **满射**：对于任意一个序数b，存在至少一个序数a，使得f(a) = b。
- **双射**：如果序数函数既是单射又是满射，则它是双射。
- **连续性**：序数函数通常具有连续性，即如果a < b，则f(a) ≤ f(b)。

**解析：**
- **单射性**：保证序数函数不会将不同的序数映射到相同的值。
- **满射性**：保证序数函数能够覆盖所有的序数值。
- **双射性**：保证序数函数既是单射又是满射，即每个序数值都有且只有一个对应的序数函数值。
- **连续性**：保证序数函数在序数集合上的连续性。

**示例：**
```markdown
f(ω) = ω+1  # 单射性
f(ω) = ω    # 满射性
f(ω) = ω    # 双射性
f(ω) = ω    # 连续性
```

**题目：** 如何构造一个序数函数？请举例说明。

- **集合映射法**：利用集合之间的映射关系构造序数函数。
- **递归构造法**：利用递归方法构造序数函数。

**解析：**
- **直接定义法**：直接指定序数函数的映射规则，例如，f(a) = a+1。
- **集合映射法**：利用两个集合之间的映射关系构造序数函数，例如，将ω映射到ω+1。
- **递归构造法**：通过递归方法构造序数函数，例如，定义f(0) = ω，f(n+1) = f(n)+1。

**示例：**
```markdown
f(a) = a+1  # 直接定义法
f(ω) = {ω, ω+1, ω*2, ...}  # 集合映射法
```
