## 4. 数学模型和公式详细讲解举例说明
### 4.1 资料相似度计算
#### 4.1.1 TF-IDF算法原理
$$ w_{i,j} = tf_{i,j} \times \log{\frac{N}{df_i}} $$
其中，$w_{i,j}$ 表示词项 $t_i$ 在文档 $d_j$ 中的权重，$tf_{i,j}$ 表示词项 $t_i$ 在文档 $d_j$ 中出现的频率，$df_i$ 表示包含词项 $t_i$ 的文档数，$N$ 表示文档总数。
#### 4.1.2 余弦相似度计算
$$ \cos(\theta) = \frac{\mathbf{A} \cdot \mathbf{B}}{\|\mathbf{A}\| \|\mathbf{B}\|} = \frac{\sum_{i=1}^n A_i B_i}{\sqrt{\sum_{i=1}^n A_i^2} \sqrt{\sum_{i=1}^n B_i^2}} $$
其中，$\mathbf{A}$ 和 $\mathbf{B}$ 是两个 $n$ 维向量，$\theta$ 是它们之间的夹角，$\cos(\theta)$ 表示它们的余弦相似度。

### 5.2 后端接口开发
#### 5.2.1 公告列表接口
```python
@app.route('/api/notices')
def get_notices():
    notices = Notice.query.order_by(Notice.publish_time.desc()).all()
    return jsonify([notice.to_dict() for notice in notices])
```
#### 5.2.2 公告详情接口
```python
@app.route('/api/notices/<int:notice_id>')
def get_notice(notice_id):
    notice = Notice.query.get(notice_id)
    if notice:
        return jsonify(notice.to_dict())
    else:
        return jsonify({'error': 'Notice not found'}), 404
```
### 5.3 数据库操作
#### 5.3.1 公告模型定义
```python
class Notice(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(100), nullable=False)
    content = db.Column(db.Text, nullable=False)
    publish_time = db.Column(db.DateTime, default=datetime.utcnow)
    author_id = db.Column(db.Integer, db.ForeignKey('user.id'))

    def __repr__(self):
        return f'<Notice {self.title}>'

    def to_dict(self):
        return {
            'id': self.id,
            'title': self.title,
            'content': self.content,
            'publishTime': self.publish_time.strftime('%Y-%m-%d %H:%M:%S'),
            'author': self.author.username
        }
```
#### 5.3.2 资料模型定义

```python
class Document(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(100), nullable=False)
    file_path = db.Column(db.String(200), unique=True, nullable=False)
    upload_time = db.Column(db.DateTime, default=datetime.utcnow)
    uploader_id = db.Column(db.Integer, db.ForeignKey('user.id'))

    def __repr__(self):
        return f'<Document {self.title}>'

    def to_dict(self):
        return {
            'id': self.id,
            'title': self.title,
            'filePath': self.file_path,
            'uploadTime': self.upload_time.strftime('%Y-%m-%d %H:%M:%S'),
            'uploader': self.uploader.username
        }
```
