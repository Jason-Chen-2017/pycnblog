## 2. 核心概念与联系
在 ReAct 框架中，我们引入了一些核心概念，例如模型、组件和管道。模型是大语言模型的核心，它由一系列层组成，每个层都有自己的参数和功能。组件是模型的基本组成部分，它可以是一个神经元、一个池化层或一个全连接层等。管道是将多个组件连接在一起形成的一个序列，它可以用于处理输入数据、生成输出数据等。

模型、组件和管道之间的联系是通过连接和参数共享来实现的。连接是指将一个组件的输出连接到另一个组件的输入，参数共享是指将一个组件的参数应用到多个组件上。通过这种方式，我们可以构建复杂的大语言模型，并通过管道来处理输入数据和生成输出数据。
